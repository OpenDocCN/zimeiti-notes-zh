# 微信视频号运营从0到1全攻略：P26：11.如何填写邀请函 📝

![](img/92c77e6fe3f2de940e9af8651dbe09de_1.png)

在本节课中，我们将学习如何正确填写视频号创作者计划的邀请函。这是加入计划、获取官方支持的关键一步，我们将一步步拆解每个填写项，确保信息准确无误。

上一节我们介绍了如何注册视频号，本节中我们来看看如何填写后续的邀请函。

![](img/92c77e6fe3f2de940e9af8651dbe09de_3.png)

![](img/92c77e6fe3f2de940e9af8651dbe09de_4.png)

## 第一步：查找视频号账号信息

首先，你需要找到自己视频号的昵称和粉丝数量。以下是具体操作步骤。

![](img/92c77e6fe3f2de940e9af8651dbe09de_6.png)

1.  **打开微信应用**：在手机上找到并点击微信图标，启动微信应用。

![](img/92c77e6fe3f2de940e9af8651dbe09de_7.png)

2.  **进入“发现”页面**：在微信底部导航栏中，点击“发现”选项。

3.  **找到“视频号”入口**：在“发现”页面中，向上滑动，找到并点击“视频号”入口（图标为黄色蝴蝶形状）。

4.  **访问个人主页**：进入视频号页面后，点击右上角的个人头像图标，进入个人主页。

5.  **查看账号信息**：在个人主页中向下滑动，找到“我的视频号”区域。这里会明确显示你的**账号昵称**和**粉丝数量**。例如，显示为“天道酬勤，39917个粉丝”。

![](img/92c77e6fe3f2de940e9af8651dbe09de_9.png)

![](img/92c77e6fe3f2de940e9af8651dbe09de_10.png)

## 第二步：填写邀请函各项内容

![](img/92c77e6fe3f2de940e9af8651dbe09de_11.png)

找到账号信息后，即可开始填写邀请函。以下是需要填写的核心项目。

1.  **视频号昵称**：填写你在上一步中查看到的账号昵称，例如“天道酬勤”。
2.  **粉丝数量**：填写你当前视频号的粉丝总数，例如“39917”。
3.  **活动邀请人**：根据你的加入渠道，填写对应的邀请人姓名，例如“张良”或“张良老师”。
4.  **实名信息**：此处必须填写你本人的真实姓名，例如“周建国”。
5.  **结算手机号**：填写你用于接收验证码和后续结算的手机号码，格式为连续数字，例如“13812345678”。

![](img/92c77e6fe3f2de940e9af8651dbe09de_13.png)

![](img/92c77e6fe3f2de940e9af8651dbe09de_14.png)

## 第三步：提交与确认

所有信息填写完毕后，进行最终提交。

![](img/92c77e6fe3f2de940e9af8651dbe09de_16.png)

1.  **点击提交**：检查所有信息无误后，点击页面上的蓝色“提交”按钮。
2.  **二次确认**：系统会弹出“确定要提交吗？”的确认框，务必点击“确认”完成提交。成功提交后，页面会提示“已成功加入创作者分成计划”。

![](img/92c77e6fe3f2de940e9af8651dbe09de_18.png)

![](img/92c77e6fe3f2de940e9af8651dbe09de_20.png)

## 课程总结

![](img/92c77e6fe3f2de940e9af8651dbe09de_22.png)

本节课中我们一起学习了填写视频号创作者计划邀请函的完整流程。关键点在于准确找到并填写自己的**账号昵称**、**粉丝数**、**实名信息**和**手机号**。确保每一步操作正确，最后不要忘记点击确认提交，这样才能成功加入计划，为后续的视频创作和收益获取打下基础。
