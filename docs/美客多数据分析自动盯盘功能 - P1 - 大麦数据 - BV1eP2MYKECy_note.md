# 美客多数据分析：P1：自动盯盘功能详解 🚀

在本节课中，我们将学习美客多（Mercado Libre）数据分析工具“大麦数据”的自动盯盘功能。该功能能帮助卖家自动监控商品数据变化，并在达到预设条件时发送提醒，从而提升运营效率。

## 功能概述

自动盯盘功能旨在帮助卖家实现自动化监控与实时提醒。卖家预先设置好监控条件，系统将持续监测相关数据。一旦数据变化达到触发阈值，系统会立即通过消息通知卖家，便于卖家进行实时决策。

上一节我们介绍了自动盯盘的核心价值，本节中我们来看看具体的使用方法。

## 使用方法详解

![](img/e213ff4ad2a17bd7a84b15866a8b9dbf_1.png)

以下是使用自动盯盘功能的完整步骤。

### 1. 设置盯盘规则

首先，需要创建具体的监控规则。

![](img/e213ff4ad2a17bd7a84b15866a8b9dbf_3.png)

1.  访问大麦数据官网，找到并点击“自动盯盘”功能，然后选择“新增规则”。
2.  为规则设置一个易于识别的名称。
3.  使用微信关注“大麦数据”公众号，以便接收后续提醒。
4.  设置具体的提醒条件，主要包括价格、销量、库存的日涨跌幅。例如，可以设定当**价格日跌幅 > 10%** 时触发提醒。
5.  完成设置后，点击确认保存规则。

### 2. 新增盯盘SKU

规则设置完成后，即可添加需要监控的具体商品。

![](img/e213ff4ad2a17bd7a84b15866a8b9dbf_5.png)

1.  进入“盯盘管理”页面，点击“新增”按钮。
2.  在弹出的界面中，选择上一步已创建好的规则模板。
3.  输入或选择需要被监控的产品SKU。
4.  同一个监控规则下可以重复添加多个产品SKU。
5.  设置完毕后，点击确认完成添加。

### 3. 查看推送消息

完成以上设置后，系统将开始自动监控。

用户需确保已关注“大麦数据”公众号。系统会根据已设置的绿色（生效）规则，自动将产品的变动信息推送至该公众号。卖家只需留意公众号消息即可获取实时监控报告。

![](img/e213ff4ad2a17bd7a84b15866a8b9dbf_7.png)

### 4. 盯盘管理

卖家可以在“盯盘管理”页面进行全面的监控信息查询与管理。

卖家可以根据预设的规则名称、产品SKU等条件筛选和查询监控信息。此外，还可以通过“推送记录”查看所有历史的产品变动提醒，便于进行数据复盘与分析。

## 总结

本节课中，我们一起学习了“大麦数据”自动盯盘功能的完整使用流程。从**设置监控规则**、**添加监控商品**，到**接收公众号提醒**及**在管理后台进行查询**，该功能实现了对商品价格、销量、库存等关键指标的自动化监控，是拉美地区美客多卖家进行专业选品与运营的实用工具。
