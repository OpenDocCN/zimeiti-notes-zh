以下是主页各设置项的优化要点：

![](img/cc83798e6467c0cc5ffdddca72d4eb86_36.png)

![](img/cc83798e6467c0cc5ffdddca72d4eb86_38.png)

*   **头像**：头像需要具有代表性。例如，如果你是分享美妆内容，可以使用品牌代言人、公司内有颜值的员工肖像，或你自己的肖像。核心原则是**头像需要与你的内容领域强相关**。
*   **昵称**：昵称应让人一眼就能看出你的账号是做什么的。例如，“穿搭日记”、“美妆小课堂”等。
*   **简介**：简介必须说明你的身份。有些企业或个人可能未在小红书开通店铺，需要将流量引导至其他平台进行转化。此时可以在简介中标注，但**必须注意遵守小红书社区公约**，避免使用违规词汇。

## 兴趣类目选择策略

![](img/cc83798e6467c0cc5ffdddca72d4eb86_48.png)

![](img/cc83798e6467c0cc5ffdddca72d4eb86_50.png)

![](img/cc83798e6467c0cc5ffdddca72d4eb86_52.png)

在我们初次使用账号时，系统会提示选择感兴趣的类目。虽然小红书允许选择四个类目，但对于新账号，建议采取以下策略：

![](img/cc83798e6467c0cc5ffdddca72d4eb86_54.png)

![](img/cc83798e6467c0cc5ffdddca72d4eb86_56.png)

![](img/cc83798e6467c0cc5ffdddca72d4eb86_58.png)

**核心策略 = 重点运营第一类目 + 其余三类目辅助 + 四类目间需有关联**

![](img/cc83798e6467c0cc5ffdddca72d4eb86_60.png)

![](img/cc83798e6467c0cc5ffdddca72d4eb86_62.png)

![](img/cc83798e6467c0cc5ffdddca72d4eb86_64.png)

以下是具体操作指南：

![](img/cc83798e6467c0cc5ffdddca72d4eb86_66.png)

![](img/cc83798e6467c0cc5ffdddca72d4eb86_68.png)

![](img/cc83798e6467c0cc5ffdddca72d4eb86_70.png)

*   **聚焦核心**：如果你是首次运营小红书账号，应重点深耕你选择的**第一个类目**。
*   **关联辅助**：其余三个类目作为辅助选择，并且这四种类目之间最好存在自然关联。例如，主做“男士穿搭”（时尚类），可以关联选择“搞笑”、“情感”、“运动”。因为关注时尚的男性用户也可能对这些内容感兴趣。
*   **避免牵强**：避免选择关联性弱的类目。例如，“男士穿搭”与“科技数码”的关联性就不如前面几项强，并非所有男性用户都同时喜欢这两者。
*   **明确主次**：你需要明确哪个是账号的**主要类目**。例如，将“服装搭配”作为账号初始化或冷启动阶段的**核心内容定位**。这意味着你初期创作的笔记内容都应围绕这个核心领域展开。
    *   **个人账号**：可参考上述策略，选择你擅长且有关联的领域。
    *   **企业账号**：必须**基于企业自身的产品和业务类型**来选择核心类目。

![](img/cc83798e6467c0cc5ffdddca72d4eb86_72.png)

![](img/cc83798e6467c0cc5ffdddca72d4eb86_74.png)

![](img/cc83798e6467c0cc5ffdddca72d4eb86_76.png)
