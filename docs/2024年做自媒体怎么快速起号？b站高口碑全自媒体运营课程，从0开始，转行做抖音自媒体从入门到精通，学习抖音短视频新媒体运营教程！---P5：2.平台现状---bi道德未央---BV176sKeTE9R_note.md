# Python入门：01：变量与数据类型 📘

在本节课中，我们将要学习Python编程中最基础也是最重要的两个概念：变量和数据类型。理解它们是如何存储和表示信息，是编写任何程序的第一步。

## 什么是变量？ 🏷️

变量是计算机内存中用于存储数据的“盒子”或“标签”。你可以把数据放进去，也可以随时取出来或更换它。

在Python中，创建一个变量并给它赋值非常简单，只需要使用等号 `=`。等号左边是变量名，右边是你想存储的值。

以下是创建一个变量的基本语法：
```python
变量名 = 值
```

例如：
```python
name = "小明"
age = 20
```

## 变量的命名规则 📝

上一节我们介绍了如何创建变量，本节中我们来看看如何给变量起一个合适的名字。变量名不能随意乱取，需要遵循一些规则。

以下是Python中变量命名的核心规则：
*   变量名只能包含字母、数字和下划线（`_`）。
*   变量名不能以数字开头。
*   变量名是区分大小写的（例如，`age` 和 `Age` 是两个不同的变量）。
*   不能使用Python的保留关键字（如 `if`、`for`、`while` 等）作为变量名。

良好的命名习惯能让代码更易读，例如使用 `user_name` 而不是 `un`。

## 基本数据类型 🔢

变量可以存储不同类型的数据。Python有一些内置的基本数据类型，最常见的有以下几种。

以下是三种最基础的数据类型介绍：

1.  **整数**
    *   表示没有小数部分的数字，可以是正数或负数。
    *   代码示例：`count = 10`， `temperature = -5`

2.  **浮点数**
    *   表示带有小数点的数字。
    *   代码示例：`price = 19.99`， `pi = 3.14159`

3.  **字符串**
    *   表示文本信息，需要用单引号 `‘ ’` 或双引号 `“ ”` 括起来。
    *   代码示例：`greeting = “你好，世界！”`

## 使用 `type()` 函数检查类型 🔍

有时你可能不确定一个变量是什么类型。Python提供了一个内置函数 `type()` 来帮助我们查看。

其使用**公式**为：
```python
type(变量名)
```

这个函数会返回变量的数据类型。例如，执行 `print(type(age))` 会输出 `<class ‘int’>`，表明 `age` 是整数类型。

## 一个简单的实践示例 💻

让我们把上面学到的知识组合起来，写几行简单的代码。

```python
# 定义几个变量
student_name = “张三”
student_score = 95.5
years_of_study = 3

# 打印这些变量
print(“学生姓名：”, student_name)
print(“考试分数：”, student_score)
print(“学习年限：”, years_of_study)

# 检查变量的类型
print(“name的类型是：”, type(student_name))
print(“score的类型是：”, type(student_score))
```

运行这段代码，你会看到变量的值被打印出来，同时也能看到它们各自的数据类型。

本节课中我们一起学习了Python的变量与数据类型。我们知道了变量是存储数据的容器，并且学会了如何命名变量。我们还认识了三种基本数据类型：用于整数的`int`，用于小数的`float`，以及用于文本的`str`。最后，我们掌握了使用`type()`函数来检查任何变量的数据类型。这些概念是构建所有Python程序的基石，请务必理解透彻。
