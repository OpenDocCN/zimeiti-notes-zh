# 小红书运营教程：P43：小红书0基础开店-店铺注册 🛒

![](img/43c83ba30fb57c8e8f4c68af98c3bee1_1.png)

在本节课中，我们将学习如何从零开始完成小红书店铺的注册流程。这是开启小红书电商之旅的第一步，我们将详细介绍每一步的操作和注意事项。

---

上一节我们介绍了开店前的准备工作，本节中我们来看看如何具体完成店铺的注册。

首先，您需要一部智能手机。使用个人手机或工作手机均可。请确保在手机上已经下载并安装了小红书App。

以下是具体的注册步骤：

1.  **登录账号**：打开小红书App，使用您准备好的手机号登录账号。
2.  **进入个人中心**：登录后，点击App右下角的“我”图标，进入个人中心页面。
3.  **找到创作中心**：在个人中心页面的右上角或左上角，点击“三条杠”菜单图标。在弹出的菜单中，选择“创作中心”。
4.  **进入更多服务**：在“创作中心”页面，找到并点击“更多服务”选项。
5.  **选择开通店铺**：在“更多服务”页面中，找到第三排的“开通店铺”选项（通常是一个独立的方框），点击进入。
6.  **开始认证**：系统可能会展示一个推广界面，直接找到并点击“立即认证”按钮，即可开始店铺注册流程。

完成上述步骤后，系统将引导您进入正式的店铺注册流程。

---

在注册过程中，有几个关键选项需要您特别注意。

**选择店铺类型**：系统会提供多种店铺类型，例如个人店、企业店、旗舰店等。对于新手，强烈建议选择 **`个人店`**。个人店门槛低，适合用来熟悉整个电商操作流程，避免初期投入过大导致压力或失去兴趣。

**填写店铺信息**：按照页面提示，如实填写店铺名称、经营类目等信息。请注意，这些信息后期可以修改，但建议一次性填写准确，以避免不必要的麻烦。

**提交审核材料**：根据要求，上传您提前准备好的身份证正反面照片等材料。提交后，等待平台审核。

---

提交所有信息后，店铺就进入审核阶段。小红书的审核速度相对其他平台可能稍慢。

审核反馈时间通常在 **`5分钟到1天`** 之间，大部分情况下会在几小时内完成。请耐心等待，审核通过后，您的店铺就正式注册成功了。

店铺注册成功后，还有一项重要的准备工作：下载 **小红书客服工作台** App。这个独立的App用于处理买家咨询、订单售后等沟通事宜，因为在小红书主App内无法直接进行即时聊天。请务必提前下载安装，以备开店后使用。
