# 剪映教程：P14：剪映核心剪辑功能——视频比例与背景设置 📐

在本节课中，我们将学习剪映的核心剪辑功能之一：如何调整视频比例以及设置视频背景。这是视频剪辑的基础操作，正确的设置能有效提升视频的观感和平台推荐效果。

在抖音、快手等短视频平台，我们看到的视频绝大多数都是竖屏格式。因此，了解如何将视频调整为适合这些平台的格式至关重要。即使你拍摄的是横屏视频，也可以通过本节课的方法轻松转换为竖屏并美化背景。

![](img/a67f49f3f91e288adbdee58f24b6b586_1.png)

## 认识视频比例

视频比例决定了画面的宽高形状。对于短视频平台，主要有两种常用比例：
*   **竖屏比例**：`9:16`。这是抖音、快手等平台最主流、推荐使用的格式。
*   **横屏比例**：`16:9`。传统视频格式，适用于Vlog等场景，但在短视频平台直接发布可能影响推荐效果。

视频分辨率通常选择`1080P`，帧率可选`60帧`，以获得清晰的播放效果。剪映也支持导出更高质量的`2K`或`4K`视频。

上一节我们介绍了视频比例的基本概念，本节中我们来看看如何在剪映中进行具体操作。

## 调整视频比例

![](img/a67f49f3f91e288adbdee58f24b6b586_3.png)

以下是调整视频比例的具体步骤。

1.  **导入素材**：打开剪映，点击“开始创作”，导入你需要编辑的视频素材。
    
    ![](img/a67f49f3f91e288adbdee58f24b6b586_3.png)

![](img/a67f49f3f91e288adbdee58f24b6b586_5.png)

2.  **找到比例功能**：导入素材后，在底部一级菜单栏向右滑动，找到并点击“比例”选项。
    
    ![](img/a67f49f3f91e288adbdee58f24b6b586_5.png)

3.  **选择目标比例**：在弹出的比例选项中，选择`9:16`。此时，横屏视频画面会居中显示，上下两侧会出现黑色区域（即“黑边”）。

完成以上步骤，视频的基础比例就调整好了。接下来，我们将处理上下黑边，让视频背景更美观。

![](img/a67f49f3f91e288adbdee58f24b6b586_7.png)

## 设置视频背景

调整比例后产生的黑边显得单调，我们可以通过“背景”功能进行美化。背景设置提供了三种主要方式。

![](img/a67f49f3f91e288adbdee58f24b6b586_9.png)

![](img/a67f49f3f91e288adbdee58f24b6b586_11.png)

以下是背景设置的三种方法。

![](img/a67f49f3f91e288adbdee58f24b6b586_13.png)

*   **画布颜色**：为背景填充纯色。你可以选择灰色、红色等各种颜色，让背景与视频主题更搭配。
    
    ![](img/a67f49f3f91e288adbdee58f24b6b586_7.png)

![](img/a67f49f3f91e288adbdee58f24b6b586_15.png)

*   **画布样式**：使用系统内置的图片或图案作为背景。这能为视频增添更多创意和风格化效果。
    
    ![](img/a67f49f3f91e288adbdee58f24b6b586_9.png)
    ![](img/a67f49f3f91e288adbdee58f24b6b586_11.png)
    ![](img/a67f49f3f91e288adbdee58f24b6b586_13.png)

*   **画布模糊**：将视频画面本身进行模糊处理，然后填充到背景中。这种效果常见于一些影视剪辑，能使主体更突出，画面更有层次感。通常选择第二或第三档模糊强度即可。
    
    ![](img/a67f49f3f91e288adbdee58f24b6b586_15.png)
    ![](img/a67f49f3f91e288adbdee58f24b6b586_17.png)

![](img/a67f49f3f91e288adbdee58f24b6b586_17.png)

## 内容与格式建议

了解操作后，如何选择比例和背景呢？这里有一些实用建议。

![](img/a67f49f3f91e288adbdee58f24b6b586_19.png)

*   **情景剧、美食、旅行等内容**：建议直接拍摄或制作为`9:16`竖版视频，这样在平台的推荐效果通常更好。
*   **Vlog等内容**：可以拍摄`16:9`横版视频，然后在剪辑时通过上述方法转换为竖版，并利用上下黑边区域添加背景色、标题或核心关键词。

![](img/a67f49f3f91e288adbdee58f24b6b586_19.png)

本节课中我们一起学习了剪映中视频比例与背景设置的核心功能。我们主要围绕短视频平台主流的`9:16`竖版格式展开，讲解了如何将横屏视频调整为竖屏，以及如何通过“画布颜色”、“画布样式”和“画布模糊”三种方法美化视频背景，从而让视频内容更丰富、更具吸引力，提升视频的推荐潜力。
