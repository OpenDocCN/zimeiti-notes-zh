# 亚马逊店铺注册：P1：公司资质要求详解

在本节课中，我们将详细讲解注册亚马逊店铺时，对公司主体的具体要求、注意事项以及常见误区。无论你是跨境电商新手还是正在考虑注册公司，本节内容都将为你提供清晰的指引。

## 🏢 公司性质与基本要求

首先需要明确，注册亚马逊店铺必须使用公司主体。对于公司的性质，亚马逊没有特殊限制。

*   无论你是**有限公司**还是**股份公司**，均可用于注册。
*   无论公司所属行业是**金融**、**自媒体**还是**教育培训**，也都符合要求。

核心要求是：该公司必须是合法存续的有限公司，且在政府官方网站上无任何异常记录。

**公式：** 合规公司 = 有限公司 + 无工商/税务异常

只要你的公司状态正常，且法人能够配合注册流程，该公司就具备注册亚马逊店铺的基本资格。

## 📝 无关紧要的公司信息

上一节我们介绍了公司的基本性质要求，本节中我们来看看哪些公司信息是亚马逊不关注的。

对于公司的具体信息，亚马逊持开放态度，以下几项均不影响店铺注册：

以下是亚马逊不审查的公司维度：

1.  **经营范围**：公司营业执照上的经营范围与实际在亚马逊销售的商品无需一致。例如，经营范围是“金融咨询”的公司，同样可以销售服装。
2.  **注册资本**：注册资本金额大小无关，无论是1万元还是1000万元。
3.  **注册地址**：注册地址所在的城市或区域不受限制，无论是在深圳等大城市还是在农村。

**核心概念**：在亚马逊平台，国内经营所需的行业资质与经营范围匹配规则**不适用**。

但是，有一个至关重要的时间点需要注意：自2023年11月起，**个体工商户**将无法再注册新的亚马逊店铺。现有店铺不受影响，但新注册必须使用有限公司。

## 🏷️ 品牌与公司的关系

了解了公司本身的要求后，我们来看看品牌。在亚马逊上销售商品，品牌是必不可少的。

品牌通常需要注册在运营亚马逊店铺的**公司名下**。关于品牌类目，亚马逊同样没有限制。

**流程说明**：
1.  拥有一个已注册的商标（品牌）。
2.  将该品牌在亚马逊后台成功完成**品牌备案**。
3.  备案成功后，你可以销售任何类目的商品（需符合平台规则），并享受亚马逊的品牌保护政策。

因此，无需纠结公司性质或经营范围是否与品牌类目匹配。

## ⚠️ 关键注意事项与建议

前面我们梳理了公司的硬性要求和无关信息，本节将重点介绍注册过程中必须注意的关键事项，这些细节直接影响注册成功率和后续运营。

以下是注册公司时必须优先考虑的几点：

1.  **法人可配合**：公司的法人必须能够随时配合进行实名验证、拍照等审核流程。购买他人公司资料存在极大风险，一旦法人无法配合，店铺可能无法注册或后续被封。
2.  **股权结构简单**：公司的股权结构应尽可能简单。
    *   **最优选择**：自然人独资（法人100%持股）。
    *   **次优选择**：若存在其他股东，其持股比例最好低于15%。
    *   **原因**：若股东持股超过一定比例（通常认为15%以上），在注册亚马逊时，该股东也可能需要接受身份审核。股东越多，审核越复杂，通过难度增加。
    *   **建议**：与合伙人可通过私下协议约定收益，而非直接体现在公司股权上，以简化注册流程。
3.  **法人年龄建议**：不建议使用年龄超过60岁或刚满18岁的法人。
    *   **超过60岁**：在审核过程中可能触发更复杂的验证，通过率可能降低。
    *   **刚满18岁**：新成立的成人身份，在亚马逊风控系统中可能被视为高风险，更容易触发严格审核（如二审）或导致注册直接失败。

此外，每位自然人的身份（作为法人或主要股东）在亚马逊系统中是一种“资源”。如果一个身份已用于某个亚马逊店铺，则该身份通常无法再用于注册另一个全新的店铺。

## 📚 本节课总结

在本节课中，我们一起学习了注册亚马逊店铺对公司主体的全面要求：

*   **核心要求**：必须使用**状态正常的有限公司**，且法人能配合审核。个体工商户自2023年11月起不可用于新注册。
*   **无关信息**：公司的**经营范围、注册资本、注册地址**均不影响注册资格。

*   **关键注意**：务必确保**法人可配合**；优化**股权结构**，建议自然人独资；谨慎选择**法人年龄**，避免使用年龄过大或过小的资源。
