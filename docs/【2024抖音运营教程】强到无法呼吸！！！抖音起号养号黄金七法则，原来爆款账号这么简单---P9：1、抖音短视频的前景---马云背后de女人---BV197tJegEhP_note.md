# 抖音运营教程：P9：1、抖音短视频的前景 📈

![](img/a848c642adc2853121395b54adf20659_1.png)

在本节课中，我们将要学习抖音短视频平台的发展前景，分析其市场潜力与用户基础，并探讨为何选择抖音作为内容创作平台。

![](img/a848c642adc2853121395b54adf20659_3.png)

了解抖音短视频的前景，需要先通过一张图片来了解。

![](img/a848c642adc2853121395b54adf20659_1.png)

这张图片是抖音创作者大会的一个数据报告。报告显示，有超过2200万人在抖音的合计收入超过了417亿元。抖音希望把这个数字翻一翻，让创作者的收入达到800亿。

![](img/a848c642adc2853121395b54adf20659_3.png)

这是巨大的流量扶持。同时，需要明确第一个知识点：抖音上面有两类人。第一类人叫做玩家，玩家是单纯玩抖音，没想过从抖音上赚到钱。另外一类人是创作者，即想通过制作抖音内容来获取收益。搞清楚这两类人很重要，后续还会提到。

那么，为什么我们选择抖音短视频呢？这么多短视频平台，例如快手、火山短视频，为什么非得做抖音？这就像开实体店铺选位置，一定会找一个流量好、适合开店的地方。做短视频也一样，一定要选一个流量大的平台。平台自身流量小，就无法分给创作者，创作者就赚不着钱。

抖音每天的活跃用户达到了8亿以上。一个月的活跃用户更是达到了40亿人次以上。这是巨大的流量。

流量大，竞争压力会不会很大？不会。创作者只有约5000万。创作者并非必须是网红达人。例如，邻居家的老太太拿手机拍自己做的饭传到抖音上，这个老太太也算创作者。所以，90%的竞争压力都来自于普通用户。

接着，做抖音短视频需不需要买电脑或摄像机？不需要。只需要一部智能手机和一个手机号，就可以注册抖音账号。抖音倡导全民能够通过短视频的形式分享和记录自己的生活。

第三个问题，有什么样的风口值得我们去做？首先，抖音计划帮助1000名优质的艺术创作者涨粉超过100万。有的同学说不会创作。不用担心，正是因为不会，才需要学习。虚心请教厉害的人不丢人。不以年龄定高低，不以穷富断今后。希望学员是谦虚好学的。

市场大，竞争足，行业还很容易产出内容。现在是大数据时代。大数据时代人人无隐私。你喜欢看什么，抖音就推荐什么。以至于用户刷到的内容会非常片面。

有的同学是宝妈，不知道拍什么，可以拍宝宝或做的饭。有的同学开实体店，想引流，可以拍产品。有的同学是普通上班族，想分享生活，也可以拍。一个月40亿的用户群体，只要内容不触犯法律道德红线，不是低俗的东西，总有一部分人会喜欢。

为什么老师说一定会有人喜欢你，你却只有几十个粉丝呢？是因为喜欢你的这一部分人还没有刷到过你的作品。所以，对于新手来说，不要把涨粉放成第一个目标。第一个目标是把流量和播放量涨起来，粉丝自然就来了。

本节课中我们一起学习了抖音短视频的广阔前景、庞大的用户基础以及低门槛的创作环境。我们明确了创作者与玩家的区别，理解了选择抖音的核心原因在于其巨大的流量池。同时，我们认识到在起步阶段，应将提升内容播放量作为首要目标，粉丝增长会随之而来。保持谦虚学习的心态，是开始抖音创作之旅的关键。
