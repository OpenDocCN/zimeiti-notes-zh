# 抖音运营教程：P7：1、抖音短视频的前景 📈

![](img/c290169506248d21cc739415c6e98676_1.png)

在本节课中，我们将要学习抖音短视频平台的发展前景，分析其市场潜力与机遇，并明确作为创作者入局的可行性。

## 平台数据与收入前景

![](img/c290169506248d21cc739415c6e98676_3.png)

上一节我们介绍了课程背景，本节中我们来看看抖音平台的具体数据。通过一张来自抖音创作者大会的数据报告图片可以了解到，有超过2200万人在抖音的合计收入超过了417亿元。

![](img/c290169506248d21cc739415c6e98676_1.png)

并且，抖音平台希望将这个数字翻一番，目标是让创作者的总收入达到800亿元。

![](img/c290169506248d21cc739415c6e98676_3.png)

这体现了平台巨大的流量扶持力度。

## 抖音上的两类用户

以下是抖音平台上的主要用户分类：
*   **玩家**：这类用户单纯在抖音上娱乐，没有计划通过平台赚钱。
*   **创作者**：这类用户（包括本课程的学员）旨在通过制作和发布抖音内容来获取收益。

## 为什么选择抖音平台

了解了用户分类后，我们来看看选择抖音的核心原因。如同开设实体店需要选择人流量大的地段，做短视频也应选择流量大的平台。抖音的日活跃用户超过8亿，月活跃用户更是达到40亿人次以上，流量基础非常庞大。

流量大是否意味着竞争激烈？并非如此。平台上的创作者数量约为5000万。这里的“创作者”定义广泛，并非特指网红达人。例如，一位邻居老太太用手机拍摄自己做饭的视频并上传，她也算创作者。因此，大部分竞争压力实际上来源于普通用户。

## 入局门槛与内容方向

明确了平台优势，接下来分析入局的实际条件。做抖音短视频无需购买专业电脑或摄像机，只需要一部智能手机和一个手机号即可注册账号。抖音倡导全民通过短视频分享和记录生活，门槛很低。

那么，当前有哪些值得关注的内容风口呢？抖音平台推出了专项计划，旨在帮助1000名优质的艺术创作者实现粉丝增长超过100万。

对于担心自己不会创作的学员，无需焦虑。学习的意义就在于从零开始。市场空间巨大，竞争相对温和，行业内容产出容易。当前是大数据时代，平台会根据用户的喜好推荐内容。

以下是针对不同身份学员的内容方向建议：
*   **宝妈**：可以拍摄宝宝日常或烹饪过程。
*   **实体店主**：可以拍摄产品进行引流。
*   **普通上班族**：可以分享个人生活点滴。

在月活跃用户高达40亿的平台上，只要内容不触犯法律和道德底线，不是低俗内容，总会有一部分受众喜欢。之所以有些创作者粉丝量少，是因为潜在粉丝尚未刷到其作品。

因此，对于新手而言，首要目标不应急于涨粉，而应是**提升视频的播放量和流量**。流量增长后，粉丝自然会随之而来。

本节课中我们一起学习了抖音平台的巨大流量潜力、清晰的用户分类、较低的入局门槛以及多样化的内容创作方向。核心在于理解：在庞大的用户基数下，坚持产出合规内容并优先提升流量，是新手起步的关键。
