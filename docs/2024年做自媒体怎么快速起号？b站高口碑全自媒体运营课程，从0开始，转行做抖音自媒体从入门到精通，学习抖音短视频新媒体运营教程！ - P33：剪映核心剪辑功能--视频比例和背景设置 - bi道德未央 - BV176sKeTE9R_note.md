# 自媒体运营课程：P33：剪映核心剪辑功能--视频比例和背景设置 🎬

在本节课中，我们将要学习剪映软件中一个非常基础但至关重要的功能：如何调整视频的比例以及设置视频的背景。这对于制作符合抖音、快手等平台要求的短视频至关重要。

## 概述：视频比例的重要性

![](img/b46d4b5b3c0e13a65ea5957cf412fb9f_1.png)

在抖音和快手等短视频平台，我们观察到一个普遍现象：超过90%的视频都是竖屏播放的。虽然也存在横屏视频，但数量较少。因此，在发布视频时，平台也推荐使用竖屏格式。了解这一需求后，我们在使用剪映进行视频剪辑时，就需要将视频比例调整为竖屏格式。

视频的画幅比例和背景调整属于剪辑的基础操作。如果设置错误，可能会影响视频的推荐和播放效果。

## 视频格式的选择建议

以下是针对不同内容类型的视频格式建议：

![](img/b46d4b5b3c0e13a65ea5957cf412fb9f_3.png)

![](img/b46d4b5b3c0e13a65ea5957cf412fb9f_4.png)

*   **VLOG视频**：建议拍摄横版视频（16:9）。在剪辑时，可以为上下黑边添加背景色，并在此区域添加标题或核心关键词。
*   **情景剧、美食、旅行类视频**：建议直接拍摄竖版视频（9:16）。这种格式在平台上的推荐效果可能更好。

![](img/b46d4b5b3c0e13a65ea5957cf412fb9f_6.png)

关于视频参数，常见的分辨率为 **1080P 60帧**。如果拥有更高清的素材，如2K或4K，剪映在导出视频时也支持选择这些更高清的格式。

上一节我们介绍了视频比例的基础概念和选择建议，本节中我们来看看如何在剪映软件中进行实际操作。

![](img/b46d4b5b3c0e13a65ea5957cf412fb9f_8.png)

## 实战操作：调整比例与设置背景

接下来，我们进入剪映软件，详细讲解如何设置视频比例并进行背景调整。

**第一步：导入素材并调整比例**

![](img/b46d4b5b3c0e13a65ea5957cf412fb9f_10.png)

1.  打开剪映，点击“开始创作”，导入一段横版视频素材。
2.  预览素材后，我们需要将其变为竖版。在底部一级菜单栏向右滑动，找到“比例”选项。
3.  点击“比例”，选择 **9:16**。此时，视频画面会变为竖版，上下出现黑色区域（即“黑边”）。

**第二步：设置视频背景**

![](img/b46d4b5b3c0e13a65ea5957cf412fb9f_12.png)

![](img/b46d4b5b3c0e13a65ea5957cf412fb9f_14.png)

调整比例后，上下黑边默认为黑色。为了使视频更美观，我们可以为背景进行设置。

![](img/b46d4b5b3c0e13a65ea5957cf412fb9f_16.png)

![](img/b46d4b5b3c0e13a65ea5957cf412fb9f_18.png)

返回到上一级菜单，点击底部的“背景”选项。这里提供了三种背景设置方式：

以下是三种背景设置方式的详细介绍：

![](img/b46d4b5b3c0e13a65ea5957cf412fb9f_20.png)

*   **画布颜色**：为背景设置纯色。你可以选择灰色、红色或其他任何颜色。
*   **画布样式**：为背景设置预设的图片或图案。剪映内置了多种样式可供选择，如渐变、纹理或创意海报背景。
*   **画布模糊**：将视频画面进行模糊处理，并填充到背景中。这种效果常见于视频主体清晰、背景模糊的样式。模糊强度通常有多个等级可选（例如从轻微模糊到深度模糊），一般选择中间等级即可达到较好效果。

选择“画布模糊”的第二个或第三个等级，然后应用。这样，我们就成功将一个横版视频转换为竖版，并将单调的黑色背景替换为具有模糊效果的动态背景，使视频整体观感更加丰富和专业。

## 总结

![](img/b46d4b5b3c0e13a65ea5957cf412fb9f_22.png)

本节课中我们一起学习了剪映的核心剪辑功能——视频比例与背景设置。

我们主要围绕抖音主流的 **9:16** 竖版视频展开。核心操作是：如果原始素材是横版（16:9），首先需要将其比例调整为竖版（9:16）。随后，对调整后产生的上下黑边区域，通过设置背景色、背景样式或背景模糊来进行美化处理。正确的比例和用心的背景设计，可以有效提升视频的视觉吸引力和平台推荐潜力。
