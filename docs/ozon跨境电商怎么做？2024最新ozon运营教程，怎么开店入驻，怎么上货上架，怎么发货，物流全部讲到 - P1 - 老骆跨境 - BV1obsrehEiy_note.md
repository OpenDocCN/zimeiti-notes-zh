# Ozon跨境电商运营教程：P1：开店入驻准备篇 🛒

在本节课中，我们将要学习如何在俄罗斯电商平台Ozon上开设店铺。课程将详细介绍开店前必须准备的五项核心资料，确保您能顺利完成第一步。

上一节我们介绍了课程的整体框架，本节中我们来看看开店的具体准备工作。想在Ozon上成功开店，首先需要准备好以下五项资料。

以下是开店所需的五项核心资料清单：

1.  **企业营业执照**
    *   必须是有限公司的企业营业执照，个体工商户的营业执照无法注册。
    *   在注册前，需通过“国家企业信用信息公示系统”官网查询营业执照状态。
    *   状态必须为“开业”或“存续”等正常状态。若为“异常”、“吊销”等状态，则无法注册成功。
    *   对于新注册的营业执照，建议等待一周后再进行店铺注册。

2.  **手机号**
    *   使用您本人常用的手机号即可。

3.  **邮箱号**
    *   可以使用QQ邮箱、163邮箱、126邮箱等常见邮箱。

![](img/492efceb516af2620bee66bc737da290_12.png)

![](img/492efceb516af2620bee66bc737da290_13.png)

![](img/492efceb516af2620bee66bc737da290_14.png)

![](img/492efceb516af2620bee66bc737da290_15.png)

4.  **海外收款账户**
    *   Ozon目前仅支持绑定两个主流的跨境收款平台：**PingPong** 和 **连连（LianLian）**。
    *   注册这两个平台账户时，**必须完成企业认证**，而非个人实名认证，以确保后续能成功绑定Ozon店铺。

5.  **法人身份证照片**
    *   在注册过程中，需要提交店铺所属公司法人的身份证照片。

![](img/492efceb516af2620bee66bc737da290_17.png)

由于Ozon的注册步骤较多，填写信息时务必仔细，一旦信息填错很容易导致注册失败。

本节课中我们一起学习了开设Ozon店铺前必须准备的五项关键资料：企业营业执照、手机号、邮箱号、海外收款账户以及法人身份证照片。确保这些资料齐全且符合要求，是成功开店的第一步。下一节，我们将进入教程的第二课，详细讲解具体的注册流程。
