# 剪映手机版教学：P28：7. 电影感镂空字体片头制作教程 🎬

在本节课中，我们将学习如何使用剪映手机版制作一个具有电影感的镂空字体片头。这个效果常用于视频开场，能瞬间提升视频的质感。我们将通过使用**蒙版**和**关键帧**功能，一步步实现文字从上到下逐渐显现的动画效果。

## 概述
我们将分两个主要步骤完成这个片头：首先制作一个纯文字视频，然后在一个新项目中利用画中画、混合模式、蒙版和关键帧动画，合成最终的电影感镂空效果。

## 教程步骤

### 第一步：制作文字素材
首先，我们需要创建一个作为素材的文字视频。

1.  导入一段纯黑色背景到剪映项目中。
2.  在黑色背景上添加你想要的文字。建议将文字位置调整得稍微靠下一些。
3.  将这个带有文字的片段导出为一个视频文件备用。

上一节我们创建了基础的文字素材，本节中我们将在新项目中利用它制作动画效果。

### 第二步：合成动画效果
接下来，我们新建一个项目，开始制作核心的镂空动画。

以下是制作文字显现动画的步骤：
1.  通过“画中画”功能，导入第一步中导出的文字视频。
2.  调整画中画视频的位置，使其与主画面匹配。
3.  选中画中画图层，点击“混合模式”，选择 **“变暗”** 模式。这样，文字部分将变为镂空效果。
4.  将画中画视频末尾与主视频对齐，分割并删除多余的部分。

现在，文字已经以镂空形式呈现，接下来我们让它动起来。

以下是添加文字扫入动画的步骤：
1.  选中画中画图层，在视频开始处添加一个**关键帧**。
2.  点击工具栏中的“蒙版”，选择 **线性蒙版**。
3.  在第一个关键帧处，将蒙版的黄线拖动到画面顶部，完全盖住文字。
4.  将时间线移动到视频的末尾，再次点击“蒙版”。此时会自动生成第二个关键帧。
5.  在末尾的关键帧处，将蒙版的黄线向下拖动，直到文字完全显示出来。这样就创建了一个文字从上向下逐渐出现的动画。

文字动画完成后，我们需要添加一个黑色背景的扫入效果来增强电影感。

以下是添加背景扫入效果的步骤：
1.  再次通过“画中画”，导入一段纯黑色背景视频。
2.  将黑色背景放大至全屏，并将其时长调整到与下方画中画对齐。
3.  在黑色背景图层的开头添加一个**关键帧**。
4.  点击“蒙版”，同样选择 **线性蒙版**。
5.  在第一个关键帧处，将蒙版的红线拖动到画面底部，完全盖住黑色背景。
6.  将时间线移动到该图层的末尾，再次点击“蒙版”。
7.  在末尾的关键帧处，将蒙版的红线向上拖动，直到黑色背景完全消失，其最终位置与上方文字动画的蒙版线位置接近即可。

## 总结
本节课中，我们一起学习了如何制作电影感镂空字体片头。我们首先创建了文字素材，然后通过**画中画**、**混合模式（变暗）**、**线性蒙版**和**关键帧**的配合，实现了文字与背景依次扫入的动画效果。掌握这个技巧后，你可以轻松为你的视频制作出专业且吸引人的开场。
