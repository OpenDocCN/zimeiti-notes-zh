# 亚马逊选品：初识商机雷达 - P1

在本节课中，我们将要学习如何使用树挖商机智选平台的“AI商机雷达”功能。这是一个强大的工具，能帮助亚马逊卖家发现潜在的市场机会。为了让您更好地掌握，我们将分集讲解。本节是第一部分，主要介绍该功能的界面布局和基础操作。

## 进入AI商机雷达模块

我们从平台左侧的导航栏点击“AI商机雷达”进入该模块。模块加载时，界面会展示许多默认数据。我们暂时忽略这些数据，将注意力集中在界面顶部。

## 了解质选策略

界面上方，数挖质选平台提供了七种不同的质选策略用于筛选商机。

*   **“全部”策略**：在此模式下，平台会展示当前所有的商机，数据量可能非常大。您可以通过选择品类、排序等方式进行二次筛选。后续我们将学习如何通过这些筛选条件定位您需要的数据。

通过点击不同的策略标签，我们可以切换计算引擎的模式。同时，我们可以滚动鼠标滑轮来浏览商机结果数据。

## 选择数据展示模式

平台支持三种数据展示模式，您可以根据个人需要或喜好随意切换。

以下是三种模式的介绍：

*   **卡片模式**：以卡片形式展示，信息较为直观。
*   **精简模式**：展示的信息较为简洁。
*   **列表模式**：以列表形式展示，通常能容纳更多数据条目。

## 使用排序功能筛选结果

当我们切换到不同的质选策略时，平台会计算出相应的商机。结果数据量可能很多，干扰我们的查询。此时，我们可以利用排序条件进行快速筛选。

排序功能支持多个条件。尝试使用排序后，结果会变得更容易查看。

## 其他策略与平台访问

其他各种质选策略在操作上与前述策略一致。除了背后的计算逻辑不同，操作方法均相同。

目前，数蛙自选平台对全球亚马逊平台买家开放，支持免费注册和试用。欢迎各位朋友随时进入平台体验。

本节课中，我们一起学习了AI商机雷达的基本界面、七种质选策略、三种数据展示模式以及如何使用排序功能来初步筛选商机。
