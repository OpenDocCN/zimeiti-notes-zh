# 视频剪辑入门：P24：音频素材设置（上）

在本节课中，我们将学习视频剪辑中音频素材的基础操作，包括如何导入、添加、调整时长以及设置淡入淡出效果。音频是视频的灵魂，恰当的处理能让视频更具感染力。

一个完整的视频通常由画面和音频两部分组成。视频中的音频可以是视频原声、后期录制的旁白，也可以是特殊的音效或背景音乐。音频是视频不可或缺的组成部分。原本普通的视频画面，只要配上符合的背景音乐，便会更加深入人心。

上一节我们介绍了视频素材的处理，本节中我们来看看如何操作音频素材。

## 导入本地音频素材

![](img/cf286dc1f58b4d95a3dde95fe466d281_1.png)

如何进行导入本地的音频素材呢？大家可以在素材库中选择“导入”功能。

以下是操作步骤：
*   在这里可以找到整理好的音频文件。
*   常见的音频文件格式有 **WAV** 以及 **mp3**。
*   选中文件后，可以选择“打开”或直接双击进行导入。
*   也可以和视频文件一起选中，同时导入。

![](img/cf286dc1f58b4d95a3dde95fe466d281_1.png)

选择打开后，音频和视频就被导入进了素材库中。

## 添加音频至时间轴

那么如何添加至时间轴呢？这个操作和添加视频素材是相似的。

我们可以直接将它进行拖拽。首先将视频拖拽到时间轴，音频也是同样的道理。可以将音频文件直接拖进时间轴，你会发现它会自动建立一个新轨道，这个轨道就是音频轨道。

## 调整音频素材时长

接下来，我们看看如何调整音频素材的时长。大家可以看到，如果视频画面播放完毕之后，音频文件还在继续，画面会以黑屏的方式显示，但音乐仍在播放。

![](img/cf286dc1f58b4d95a3dde95fe466d281_3.png)

这个时候我们需要对当前音频素材的时长进行调整。大家可以选中当前的音频，拖拽它的尾部，直接拖拽到视频结束的位置，就可以调整音频至合适的时长。

![](img/cf286dc1f58b4d95a3dde95fe466d281_4.png)

也可以选择另一种方法：将播放头移动到合适的位置，选择“分割”功能。单击之后，就将当前的音频进行了切割。然后选中不需要的音频片段，按住删除键或 `Delete` 键进行删除。

![](img/cf286dc1f58b4d95a3dde95fe466d281_3.png)

![](img/cf286dc1f58b4d95a3dde95fe466d281_4.png)

当然，音频轨道和视频、图片轨道的操作是相似的，也可以对它进行其他操作，比如分割。

## 复制和删除音频素材

删除音频素材很容易理解，直接选中当前的音频，按删除键即可。

如何进行复制呢？大家可以看到，我们选中当前的音频之后，可以按住 `Ctrl + C` 进行复制。然后将播放头放置到合适的位置，按住 `Ctrl + V` 进行粘贴。

也可以选中当前音频素材，复制后，在音频轨道后面的任意区域单击鼠标右键，找到“粘贴”选项进行粘贴。

## 调整音频音量

选中当前的音频，在右上角的参数调节面板中能够找到“音量”选项。选择当前的滑块可以进行音量调整。

滑块往左是降低音量，往右是增加音量。降低音量时显示负值（如 `-10 dB`），增加时显示正值（如 `+10 dB`）。单位是 **dB**。可以移动滑块来调整，也可以直接输入数值，或使用上下按钮进行微调。

## 设置音频淡入淡出效果

对于一些没有前奏和尾声的音乐，直接播放会显得比较突兀。那么如何设置淡入以及淡出效果呢？这能让音频与画面的配合更自然，或在切换音乐时过渡更平滑。

选中导入的音频素材，在参数面板中能够找到“淡入时长”以及“淡出时长”选项。

我们可以移动滑块来调整时长，单位是秒。调整后，音频轨道的波形图首尾会以弧度的方式呈现。例如，设置淡入时长为3秒，意味着音乐会在3秒内从无声逐渐增大到正常音量。淡出时长的调整同理，它会让音频在结尾处逐渐降低音量直至无声。

本节课中我们一起学习了音频素材的基础设置，包括导入、添加、调整时长、复制删除、音量控制以及淡入淡出效果。掌握这些操作，是让视频音画和谐的第一步。
