在本节课中，我们将学习如何对苹果手机进行“抹机”操作。这是运营TikTok跨境电商店铺前的重要准备工作，目的是将手机设置为目标市场（如美国）的环境，包括语言、时区和定位等。我们将以一部苹果6S手机为例，进行详细的步骤演示。

## 抹机流程详解

首先，确保你的手机是一部新手机或已恢复出厂设置的手机，内部没有插入SIM卡，并且已连接到一个可用的Wi-Fi网络。

以下是抹机的具体步骤：

1.  **进入设置并选择还原**
    打开手机，进入“设置”应用。向下滑动屏幕，找到并点击“通用”选项。在通用菜单中，继续向下滑动，找到“还原”并点击。

2.  **抹除所有内容与设置**
    在还原菜单中，选择第二个选项“抹掉所有内容和设置”。系统会弹出确认窗口，点击“抹掉iPhone”以继续。此操作将使手机重启并进入初始化界面，整个过程大约需要1到2分钟，请耐心等待。

3.  **选择语言与地区**
    手机重启后，会进入初始化设置界面。按下Home键（圆形按钮）开始设置。首先选择语言，如果目标市场是英美地区，请选择“English”。接着选择地区，滑动到列表底部，选择“United States”（美国）。

4.  **连接网络并跳过Apple ID**
    选择地区后，点击蓝色的“下一步”按钮。系统会要求连接Wi-Fi网络，请连接你的个人热点或可用Wi-Fi。连接成功后，继续点击“下一步”。当系统提示登录Apple ID时，选择“忘记密码或没有Apple ID？”，然后点击“稍后在设置中设置”。后续所有涉及Apple ID的步骤，均选择“不使用”或“稍后设置”。

5.  **同意协议并关闭非必要服务**
    阅读并同意苹果的服务条款。在接下来的设置中，关于定位服务、Siri、数据分析等功能，均选择“禁用”或“不共享”。对于屏幕外观等个性化设置，使用默认选项即可，一路点击“继续”或“下一步”。

6.  **完成初步设置并进入系统**
    完成上述步骤后，手机会进入主界面。但抹机操作尚未完全结束，我们还需要进行关键的系统设置。

上一节我们完成了手机的初始化设置，本节中我们来看看进入系统后必须调整的关键配置。

## 关键系统设置

进入系统主界面后，我们需要手动调整时区和确认隐私设置，以确保环境符合要求。

以下是必须检查的设置项：

1.  **设置时区**
    打开“设置”应用，找到并点击“通用” -> “日期与时间”。关闭“自动设置”选项，然后在时区中搜索“Los Angeles”或“LAX”（洛杉矶），将其时区设置为美国太平洋时间。

2.  **确认隐私设置**
    返回设置主菜单，找到并点击“隐私与安全性”。首先检查“定位服务”，确保其已关闭。然后向下滑动，检查“跟踪”选项（部分机型有），如果存在，请关闭“允许App请求跟踪”的开关。

3.  **允许应用下载**
    完成设置后，你可能会发现主屏幕上的原生应用（如App Store）显示为等待下载状态。点击任意一个这样的应用图标，系统会弹出网络访问请求，选择“允许使用无线局域网与蜂窝数据”。之后，所有预装应用将开始自动下载。
