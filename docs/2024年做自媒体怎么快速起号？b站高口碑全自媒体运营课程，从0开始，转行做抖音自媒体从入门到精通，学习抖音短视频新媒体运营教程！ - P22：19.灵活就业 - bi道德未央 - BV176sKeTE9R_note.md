# 自媒体运营课程：P22：19.灵活就业 🚀

在本节课中，我们将探讨自媒体作为一种灵活就业选择的可能性，分析其适合人群、行业前景以及相关的职业发展路径。

上一节我们介绍了自媒体运营的多种变现方式，本节中我们来看看如何将自媒体技能转化为一份灵活的职业。

## 自媒体作为灵活就业的选择

自媒体可以作为一项灵活的就业选择。这种选择比较适合现阶段临近大学毕业、即将进入社会但不确定哪个行业有发展前景的人群。它也适合那些本职工作已没有良好升值或涨薪空间、希望转行进入互联网行业的人群。现在，自媒体的教学以及专业的自媒体运营师，其应用行业非常广泛。

## 自媒体运营的行业应用

![](img/abdc2c2650d93cf531f391b6797de783_3.png)

如今，每个行业都会运用到“宣传”。无论你是从事餐饮、传媒还是教育行业，都一定会涉及到宣传，即发布广告。现在我们已经很少能在大街上看到发传单、发小卡片的人了。他们并非不做宣传，而是将阵地转移到了互联网平台。这正是未来自媒体运营师的工作范围之一。

## 全媒体运营师的职业化

![](img/abdc2c2650d93cf531f391b6797de783_5.png)

目前，国家已将“全媒体运营师”纳入国家职业分类。该职业于去年2月份被纳入，是一个合法的职业。包括在Boss直聘等招聘平台上，该职位给出的底薪也非常可观。

以下是招聘平台上零经验自媒体运营岗位的薪资示例：
*   5000~8000元
*   5000~7000元
*   7000~8000元

## 团队与职业机会介绍

如果你未来真的考虑转行进入互联网行业，可以考虑加入我们的团队。我们公司现阶段位于江西省赣州市的恒客产业园，是受政府园区邀请入驻办公的。

以下是公司主要部门的介绍：

*   **业务支持中心**：这是我们的前端部门，为后端所有部门提供支持。
*   **课程研发中心**：该部门由我主要负责。无论是平台更新还是课程更新，全部由我全权把控。
*   **流量中心**：现阶段主要负责为我们内部的学员对接官方流量。此外，也负责对接如影视宣传推广、抖音直播切片、挂载小黄车等千万级网红视频的合作。
*   **VIP服务中心**：这是学员加入后的后端扶持部门。当学员遇到素材不会找或视频不知如何制作等问题时，我们的老师会通过语音、视频或远程操控的方式提供帮助。我们后端的VIP老师都非常专业。

## 公司文化与工作环境

我们公司也比较注重员工的内在成长与工作体验。除了日常工作，我们还会组织读书会等活动。公司也配备了咖啡厅、影音室和健身房。我们希望所有伙伴都能在工作中感到开心，并快乐地挣钱。

如果你未来真的考虑转行，可以来找我。但需要说明的是，我们希望你确实是具备相应技能和知识储备的。因为我要对自己的团队和未来的学员负责。
