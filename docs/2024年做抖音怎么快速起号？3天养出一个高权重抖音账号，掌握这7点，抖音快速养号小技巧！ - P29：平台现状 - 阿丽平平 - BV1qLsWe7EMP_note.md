# 自媒体入门：P29：平台现状 📊

在本节课中，我们将要学习自媒体行业的平台现状，分析其可行性、政策支持、平台激励以及市场潜力，帮助你判断是否值得进入这个领域。

上一节我们了解了自媒体的基本概念，本节中我们来看看这个行业所处的平台环境究竟如何。

## 平台现状分析

![](img/b1d7214351977643d97098779c12270c_1.png)

现阶段，整个社会氛围比较浮躁，许多人都渴望快速赚钱。但需要注意的是，过去有人采取非法手段牟利，甚至因此遭受损失，这类行为是国家明令禁止的，对我们没有任何好处。

与之相反，**自媒体行业是国家目前明确支持的领域**。国家已将支持文件下达到各个地方政府。

![](img/b1d7214351977643d97098779c12270c_3.png)

![](img/b1d7214351977643d97098779c12270c_5.png)

以下是相关政策文件的例证：

*   **荆州市六项措施**：该文件由人社局（国家单位）于今年6月2日发布，明确指出要“加大政策宣传力度，有力支持自媒体以及新媒体行业的蓬勃发展”。
*   **广西壮族自治区文件**：同样发布了支持新媒体行业发展的政策性文件。

这些文件都强调了“坚持正确导向，内容为王”。这意味着，未来作为自媒体创作者，我们创作的**文章和视频内容**，将是获得收益的核心与关键。

既然国家政策已经明确支持，那么自媒体平台也积极响应号召，投入资金激励创作者。

![](img/b1d7214351977643d97098779c12270c_7.png)

## 平台激励措施

接下来，我们通过一个具体案例来了解平台提供的激励。

![](img/b1d7214351977643d97098779c12270c_9.png)

以下是西瓜视频（字节跳动旗下平台）宣布的激励计划关键信息：
*   **激励金额**：未来一年，额外拿出**20亿元**人民币补贴中视频创作人。
*   **性质**：创始人强调，这是**真金白银的20亿元**，不包含任何商单、直播和电商收入，属于纯补贴。
*   **模式**：平台将积极探索“保底+分成”的模式，助力创作人职业化，并且20亿只是保底，上不封顶。

这20亿元补贴，正是我们未来通过创作视频可以参与分享的收益来源之一。

除了政策与资金支持，自媒体行业目前还有一个显著优势：庞大的用户基础。

## 市场潜力与用户规模

现阶段自媒体行业的优势在于其庞大的用户数量。根据2022年《中国互联网络发展状况统计报告》（中国互联网络信息中心发布），我国网民规模已达**9.4亿**，接近10亿。

这9.4亿人，就是潜在的视频观众。他们拥有手机和视频平台，每天都会消费视频内容。

有如此多的观众，那么创作者有多少呢？根据相关市场调查：
*   截止去年，我国全职从事自媒体的人约有**370万**。
*   兼职从事自媒体的人约有**600万**。
*   两者相加，创作者总数约为**970万**，我们可近似看作**1000万**。

这就形成了一个简单的比例：**大约1000万创作者对应9.4亿观众**，相当于每100个看视频的人中，只有大约1个人在生产视频。一个人生产内容的速度，很难满足一百个人消费内容的需求。

这种供需关系的不平衡，正是国家支持自媒体行业发展的重要原因之一。从我们观看媒介的演变——从黑白电视到彩色电视，再到如今能连接WIFI的智能电视——可以看出，互联网视频内容的分发渠道和受众规模都在持续扩大，未来的发展空间非常广阔。

![](img/b1d7214351977643d97098779c12270c_11.png)

本节课中我们一起学习了自媒体行业的平台现状。我们了解到，该行业受到国家政策的有力支持，各大平台也投入真金白银进行激励，并且拥有近10亿用户的庞大市场，而创作者数量相对较少，存在巨大的发展潜力和机会。因此，从平台环境来看，自媒体是一个值得考虑和进入的领域。
