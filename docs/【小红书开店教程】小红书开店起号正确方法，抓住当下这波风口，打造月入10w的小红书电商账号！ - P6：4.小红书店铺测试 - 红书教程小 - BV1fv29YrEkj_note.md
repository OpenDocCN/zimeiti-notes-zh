# 小红书开店教程：P6：4.小红书店铺账号测试 🧪

![](img/66b5732ac72902208290b3b8eae3bc17_1.png)

在本节课中，我们将要学习如何为你的小红书店铺测试账号权重。这是开店前至关重要的一步，能帮你筛选出有潜力的账号，避免后续投入精力却因账号本身问题而失败。

![](img/66b5732ac72902208290b3b8eae3bc17_3.png)

上一节我们介绍了开店的基础准备，本节中我们来看看如何测试账号。

![](img/66b5732ac72902208290b3b8eae3bc17_7.png)

小红书店铺账号与其他平台不同，并非所有新创建的账号都具备基础权重。无论是使用新手机号、新身份证注册，还是使用3年以上的老手机号，都存在一定概率（约5%-10%）注册到没有权重的账号。如果账号本身没有权重，无论投入多少精力运营店铺，效果都可能不理想。因此，在正式开店前，必须对账号进行测试。

## 测试核心原理与流程

账号测试的核心是：**在未开通店铺前，模拟正常发布笔记，观察其自然流量数据**。一个有权重的账号，其新发布的笔记会获得平台的基础推荐流量。

以下是具体的操作步骤：

1.  **准备测试账号**：准备2-3个待测试的账号即可。无需过多，这足以覆盖并筛选出有权重的账号。
2.  **发布测试笔记**：使用每个账号发布一篇**原创内容**。内容形式可以是视频或图文，主题不限（例如分享好物、生活技巧等）。
3.  **观察数据指标**：发布笔记后，重点观察“小眼睛”（即笔记浏览量）数据。**关键观察期是发布后的3天内**。

## 测试成功的关键标准

判断账号是否有基础权重的数据标准如下：
*   **短期流量**：笔记发布后约2分钟内，“小眼睛”数量应达到20以上（此为平台强推流量，仅作初步参考）。
*   **持续流量**：在接下来的3天观察期内，笔记**每日**的“小眼睛”增长应超过50。
*   **总流量**：3天后，该篇笔记的**总“小眼睛”数量应超过150**。达到此标准，即可认为该账号具备基础权重，适合用于开店。

## 必须注意的四大要点 ⚠️

为确保测试结果准确，避免因操作不当导致误判，必须严格遵守以下规则：

1.  **内容必须原创**：测试笔记**必须是原创内容**。如果发布视频，时长**必须超过15秒**（建议20秒左右）。直接复制他人内容会导致测试无效，即使账号有权重，后续发布非原创商品也可能没有流量。
2.  **禁止同设备ID测试**：多个测试账号**不能在同一部手机或同一个电脑浏览器环境下切换登录操作**。平台会识别设备ID，同设备操作会导致账号关联，均被判定为无权重。应使用不同的手机设备进行测试。
3.  **必须使用手机流量**：测试期间，操作手机**必须使用手机移动数据网络**，**绝对不能连接Wi-Fi**。
4.  **避免共用Wi-Fi历史**：如果几部测试手机曾经连接过同一个Wi-Fi，需要将所有手机断开该Wi-Fi，并使用流量操作**至少一周后**，再进行测试。否则，平台可能通过IP地址关联账号，影响权重判定。

只要遵循以上步骤和要点，花费3-5天时间，你就能有效筛选出适合开店的优质账号，为后续的店铺运营打下坚实基础。
