# 视频号运营教程：P20：3. 简单的书单原创视频制作教程

## 概述
在本节课中，我们将学习如何使用剪映工具，从零开始制作一个简单的书单推荐视频。整个过程将分为几个清晰的步骤，包括素材选择、文案添加、样式调整、画中画应用以及音频处理，确保即使是初学者也能轻松掌握。

---

## 素材准备与导入
上一节我们介绍了视频号的基础概念，本节中我们来看看如何具体制作一个书单视频。首先，我们需要准备视频素材。

1.  打开剪映APP，点击“开始创作”按钮。
2.  进入编辑界面后，点击底部工具栏中的“素材库”。
3.  在素材库顶部的搜索框（放大镜图标）内输入关键词“书单”，然后点击搜索。
4.  在搜索结果中，选择你喜欢的书单背景素材，点击素材右上角的选框，然后点击右下角的“添加”按钮，将其导入到时间轴。

**核心操作**：`点击[开始创作] -> 进入[素材库] -> 搜索“书单” -> 选择并添加素材`

---

## 添加与编辑文案
素材导入后，接下来我们需要为视频添加核心的文字内容。

1.  首先，选中主视频轨道，点击底部工具栏的“关闭原声”按钮。
2.  接着，在底部主工具栏中找到并点击“文本”选项。
3.  在文本工具中，选择“新建文本”。
4.  将你事先准备好的书单推荐文案粘贴到文本输入框中。

以下是文案的样式调整步骤：

1.  点击输入框上的“字体”，在字体列表中根据喜好选择，例如选择“书法”分类下的“位杯体”。
2.  点击“样式”，可以更改文字颜色，例如选择黄色或白色。
3.  继续在“样式”中点击“排列”，向下滑动找到“行间距”，拖动滑块将数值调整为 **4**。
4.  调整完成后，点击右下角的对勾（✓）确认。
5.  在预览窗口中，用双指可以旋转和缩放文本，将其拖动到画面中合适的位置。
6.  最后，在时间轴上选中这段文本，拖动其结尾的白色边框，使其长度与主视频对齐。

---

## 创建视频标题
为了让视频主题更明确，我们还需要添加一个标题。

1.  再次点击底部工具栏的“文本”，选择“新建文本”。
2.  输入你的视频标题文字。
3.  点击“样式”，为标题选择一个醒目的颜色，例如黄色。
4.  在预览窗口中，拖动标题到合适位置，并用双指适当放大。
5.  点击对勾（✓）确认，然后在时间轴上拖动标题片段的白色边框，使其长度与主视频对齐。

---

## 添加画中画素材
为了丰富视频内容，我们可以通过画中画功能添加一段人物或场景视频。

1.  点击底部工具栏的“画中画”，选择“新增画中画”。
2.  从相册中选择一段你准备好的视频素材，点击“添加”。
3.  选中这段新增的画中画素材，在底部二级工具栏中向后滑动，找到并点击“蒙版”。
4.  在蒙版选项中选择“圆形”蒙版。
5.  在预览窗口中，用双指调整圆形蒙版的大小和位置，使其框住你想突出显示的部分（如人物面部）。
6.  拖动蒙版下方的羽化箭头，适当增加边缘羽化效果，使融合更自然，然后点击对勾（✓）确认。
7.  在预览窗口中，双指缩放并拖动画中画素材，将其调整到合适的大小和位置（例如左上角）。
8.  由于画中画素材可能带有原声，需要选中它，点击底部工具栏的“音量”，将滑块拖到最左侧（数值为 **0**），然后点击对勾（✓）确认。
9.  将时间线移动到主视频的结尾处，选中画中画素材，点击“分割”，然后删除后半段多余的部分，使其与主视频等长。

**核心概念**：`蒙版` 是一种控制图层显示区域的工具，`圆形蒙版` 意味着只显示圆形区域内的画面。

---

## 添加文字动画与背景音乐
动态的文字和合适的音乐能极大提升视频的观赏性。

1.  **添加文字动画**：在时间轴上选中之前添加的主文案文本，在底部工具栏中找到“动画”。选择“入场动画”，然后在列表中选择一个动画效果，例如“打字机”。将动画时长滑块拖到最右侧以延长效果时间，点击对勾（✓）确认。
2.  **添加背景音乐**：
    *   点击底部工具栏的“音频”，选择“音乐”。
    *   在音乐库的搜索框中输入你想要的歌曲名，例如“老男孩”，然后点击搜索。
    *   从搜索结果中选择一个版本，点击右侧的“使用”按钮。
    *   在时间轴上，将音频轨道向前拖动，使其开头与视频开头对齐。
    *   将时间线移动到视频的结尾处，选中音频轨道，点击“分割”，然后删除视频结尾之后多余的音频部分。

---

## 总结
本节课中，我们一起学习了制作一个简易书单视频的全过程。从在素材库中寻找背景、添加并美化文案和标题，到使用画中画和蒙版功能丰富画面，最后为文字添加动画并配上背景音乐。掌握这些基础操作后，你就可以灵活运用剪映，制作出属于自己的原创书单视频了。
