在本节课中，我们将学习如何利用AI工具全自动创作短视频。这种方法无需手动操作，即可生成符合平台要求的优质内容，帮助账号快速涨粉与变现。

## 概述

平台青睐的短视频作品通常具备两个核心要素：**优秀的文案**和**高度相关且精彩的画面**。基于此，我们将介绍一套结合三个AI工具的自动化创作流程。

## 核心工具介绍

以下是实现全自动创作所需的三个核心工具。

### 1. 文案创作：ChatGPT

ChatGPT是一款强大的人工智能文案创作工具。它不仅能生成高度原创的文本，其内容甚至能体现出类似人类的情感和思维，这对于创作吸引人的故事文案至关重要。

**使用方法示例**：
```python
# 向ChatGPT提问的示例
prompt = "请帮我写一个约1000字的中国历史故事。"
# 可以进一步细化要求，例如：
prompt = "请帮我写一个约1000字、关于湖南地区的冷门民间故事。"
```
如果对初次生成的文案不满意，可以追加条件让其优化，例如：“请在故事中加入更多人物，同时保持总字数不变。”

### 2. 文案转配音：AI配音工具

获得优质文案后，下一步是将其转换为语音。我们可以使用如“配音神器”这类AI配音工具。

**操作流程**：
1.  打开配音工具，点击“制作配音”。
2.  将ChatGPT生成的文案粘贴进去。
3.  在“更多主播”中选择喜欢的音色风格。
4.  生成并保存配音文件。

![](img/3b51b6b1e2a1e6b170ea8229acc29c53_6.png)

### 3. 画面匹配：三三搜帧

最后一步是为配音匹配高质量、高相关度的动态视频画面。这里推荐使用“三三搜帧”工具。它不仅能根据关键词搜索无版权视频素材，更具备“AI合成视频”功能。

![](img/3b51b6b1e2a1e6b170ea8229acc29c53_8.png)

**核心操作**：
1.  在工具首页点击“合成视频”。
2.  导入上一步得到的配音文件。
3.  系统会自动根据台词，为每一句匹配多个相符的视频画面。
4.  用户确认或选择画面后，点击“生成视频”即可完成创作。

## 完整流程演示

上一节我们介绍了三个核心工具，本节我们来看一个完整的操作演示。

![](img/3b51b6b1e2a1e6b170ea8229acc29c53_10.png)

1.  **生成文案**：使用ChatGPT，输入“我需要一份1000字左右的中国历史故事”，获得原创文案。
2.  **生成配音**：将文案复制到配音工具中，选择主播音色并生成配音文件。
3.  **合成视频**：打开三三搜帧的“AI合成视频”功能，导入配音文件，系统自动匹配画面，最终生成成片。

![](img/3b51b6b1e2a1e6b170ea8229acc29c53_12.png)

通过以上三步，一个由AI全自动创作的、包含优质文案、专业配音和高度吻合画面的短视频就制作完成了。
