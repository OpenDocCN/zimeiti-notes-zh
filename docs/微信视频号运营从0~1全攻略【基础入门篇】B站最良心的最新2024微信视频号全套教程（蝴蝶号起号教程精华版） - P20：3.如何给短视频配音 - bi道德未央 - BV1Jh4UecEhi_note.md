# 微信视频号运营从0~1全攻略：P20：如何给短视频配音 🎙️

在本节课中，我们将学习如何为短视频进行配音。当您觉得原视频的配音效果不佳时，可以将其替换为自己的配音。我们将介绍配音工具的选择、具体操作步骤以及如何将新配音应用到视频中。

## 配音工具介绍

上一节我们介绍了视频处理的基本概念，本节中我们来看看配音工具。配音工具种类繁多，例如“配音神器”就是一个功能强大的选择。它不仅能配音，还集成了去水印、文案提取等多种实用功能。

![](img/d64bb77881207cc2ac905c5f7a83be58_1.png)

以下是“配音神器”小程序的主要功能点：
*   **配音**：输入文案脚本即可生成语音。
*   **去水印**：可以去除视频中的水印。
*   **文案提取**：从视频中提取文字内容。
*   **修改视频MD5**：修改视频文件的唯一标识码，有助于增加视频的原创度。

> **注意**：仅修改MD5值并不能完全规避平台审核，后续仍需进行视频混剪处理。混剪方法将在后续课程中详细讲解。

## 如何搜索与使用工具

![](img/d64bb77881207cc2ac905c5f7a83be58_3.png)

了解了核心工具后，我们来看看如何寻找和使用它们。无论是去水印、配音还是修改MD5，都可以在微信中直接搜索对应的小程序。

![](img/d64bb77881207cc2ac905c5f7a83be58_5.png)

操作步骤如下：
1.  打开微信，点击顶部的“搜索”框。
2.  输入关键词，如“去水印”、“修改MD5”或“配音”。
3.  在搜索结果中选择“小程序”分类，即可找到大量相关工具。

推荐使用“配音神器”或“青抖”。若某个工具暂时无法使用，可随时更换其他同类小程序。“配音神器”的大部分功能（如去水印、文案提取）是免费的，仅配音功能需要消耗金币。

![](img/d64bb77881207cc2ac905c5f7a83be58_7.png)

## 免费使用与金币获取

为了持续免费使用配音功能，我们需要了解如何获取金币。“配音神器”提供了每日签到领取金币的福利。

以下是金币获取的规律：
*   首次签到可能获得2个金币。
*   连续签到，后续可获得5个、8个、10个甚至更多金币。
*   生成一段配音通常消耗3个金币。

通过每日签到积累金币，可以满足基本的配音需求。如果需求量较大，也可以考虑开通会员。

## 制作配音的详细步骤

现在，我们进入核心环节，学习如何制作一段配音。整个过程在“配音神器”小程序中完成，非常简单直观。

具体操作流程如下：
1.  打开“配音神器”小程序，点击“配音”功能。
2.  清除默认文字，将您准备好的文案脚本粘贴或输入到文本框中。
3.  在声音库中选择喜欢的音色（如女声、男声、方言等）。
4.  点击“**合成配音**”按钮，系统会开始生成语音。
5.  生成完成后，选择“**保存配音视频到相册**”或“**保存字幕视频到相册**”。前者是纯音频文件，后者是带有字幕画面的视频文件。

![](img/d64bb77881207cc2ac905c5f7a83be58_9.png)

## 在剪映中替换视频原声

配音制作完成后，下一步就是将其应用到目标视频中。我们将在剪映App中完成音频替换。

![](img/d64bb77881207cc2ac905c5f7a83be58_11.png)

替换原声的操作方法如下：
1.  打开剪映，导入需要处理的视频素材。
2.  选中视频轨道，点击“**关闭原声**”图标。
3.  点击底部工具栏的“**音频**”选项。
4.  选择“**提取音乐**”，从手机相册中导入刚刚保存好的配音音频文件。
5.  导入后，视频的原声即被新的配音替换。您可以调整音频位置，使其与画面同步。

> **提示**：以上演示中视频画面与文案内容并不相关，仅为展示操作流程。在实际制作时，请确保配音内容与视频主题匹配。
