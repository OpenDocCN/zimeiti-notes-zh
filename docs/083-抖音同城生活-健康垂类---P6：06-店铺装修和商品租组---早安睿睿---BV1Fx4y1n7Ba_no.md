# 抖音同城生活-健康垂类0到1运营：P6：店铺装修和商品组品 🏪

![](img/cce67e02237277105e8d90b5070bacd1_0.png)

![](img/cce67e02237277105e8d90b5070bacd1_2.png)

在本节课中，我们将要学习抖音生活服务商家如何通过**店铺装修**和**商品组品**来提升用户信任感与购买转化率。我们将理解其重要性，并掌握具体的操作步骤。

![](img/cce67e02237277105e8d90b5070bacd1_4.png)

---

![](img/cce67e02237277105e8d90b5070bacd1_6.png)

许多商家朋友会问，为什么需要做店铺装修，还需要给不同的商品做套餐组合，难道不是把商品挂到抖音就可以了吗。

![](img/cce67e02237277105e8d90b5070bacd1_8.png)

为什么要做这么复杂呢？其实在抖音生活服务上做生意，和线下开店是一样的。大家可以想想我们为什么要装修实体门店。

![](img/cce67e02237277105e8d90b5070bacd1_10.png)

我们给店里布置不同的装潢风格，为什么要设置不同价位的商品，还会时常安排实惠的套餐。不就是为了让客户进入门店后，能通过我们的装修快速建立信任，喜欢用餐环境。

![](img/cce67e02237277105e8d90b5070bacd1_12.png)

看到店里的菜单和招牌爆品，看到不同商品的组合后，比单品更优惠的价格吸引客户下单吗。

![](img/cce67e02237277105e8d90b5070bacd1_14.png)

![](img/cce67e02237277105e8d90b5070bacd1_16.png)

现在我们来跟随抖音用户的脚步，看看用户在每一个阶段的心理变化，您就更能理解了。

![](img/cce67e02237277105e8d90b5070bacd1_18.png)

比如用户被一条短视频吸引，关注到我们的店铺，很喜欢视频中的店铺风格，通过点击短视频上的POI进入我们的店铺。

![](img/cce67e02237277105e8d90b5070bacd1_20.png)

却没看到什么内容。这时的用户心里会是：这是虚假宣传吗？怎么店铺连一张介绍图片都没有？也没有说能不能停车？我晚上09：30去会不会关门啊？

![](img/cce67e02237277105e8d90b5070bacd1_22.png)

算了，还是看看别家吧。

![](img/cce67e02237277105e8d90b5070bacd1_24.png)

所以如果没做好店铺信息的完善和装修，我们辛辛苦苦吸引过来、想要下单的用户就很容易因不信任而流失。可见商家朋友们在店铺装修环节一定要重视，按照我们的操作指引来建设内容。

用丰满的店铺页面与用户建立信任。

![](img/cce67e02237277105e8d90b5070bacd1_26.png)

![](img/cce67e02237277105e8d90b5070bacd1_28.png)

然后用户才会开始查看，有哪些商品是他感兴趣的。这时如果他只看到一个商品链接，可能会直接因为怕踩雷而选择放弃。所以店铺装修是第一步，做好商品的组合、上线满足更多人需求的团品是更重要的一步。

让不同的商品发挥不同的作用，才能更好地吸引用户下单。店铺装修和商品组货是开启经营必完成的两步。

![](img/cce67e02237277105e8d90b5070bacd1_30.png)

其实操作并不难。首先来回答大家一个问题：门店装修有什么用？

![](img/cce67e02237277105e8d90b5070bacd1_32.png)

通过门店装修可以配置门店主图、官方相册、发布公告，以提升用户对于门店的好感度，有利于促进团购订单成交。

![](img/cce67e02237277105e8d90b5070bacd1_34.png)

以下是具体的操作步骤：

![](img/cce67e02237277105e8d90b5070bacd1_36.png)

选择抖音来客手机端登录，进入抖音来客首页。

![](img/cce67e02237277105e8d90b5070bacd1_38.png)

**第一步**，点击“门店装修”，进入装修页。

**第二步**，填写基础信息。选择服务设施，例如：免费开放、景区讲解、导游服务、无需取票、可带宠物、免费停车、可停车、行李寄存、吸烟区、无障碍设施、轮椅租赁等。根据商户实际情况选择。

![](img/cce67e02237277105e8d90b5070bacd1_40.png)

![](img/cce67e02237277105e8d90b5070bacd1_42.png)

**第三步**，基础装修。店铺入口图可上传六张单张图片，大小不超过五兆的图片作为店面门面。

![](img/cce67e02237277105e8d90b5070bacd1_44.png)

官方相册上传相关图片，根据图片选择相册名称。上传的照片会显示到主页上的地点相册中外景小图片。建议上传尺寸为 **1000×750** 大小的图片。

**第四步**，公告管理。可以选择打开公告内容，然后配置公告。上传一张不超过五兆、比例在 **16:9** 的公告照片，选择公告类型，并且写上公告标题以及公告详情（注意字数的限制），最后选择有效期时间。

![](img/cce67e02237277105e8d90b5070bacd1_46.png)

**第五步**，提交审核。审核提交后可以在抖音来客首页查看审核结果，一般一至两个工作日会反馈审核结果。如果审核不通过，可以根据提示进行修改后再次提交上传。

![](img/cce67e02237277105e8d90b5070bacd1_48.png)

![](img/cce67e02237277105e8d90b5070bacd1_49.png)

本节课中，我们一起学习了店铺装修与商品组品的重要性。我们明白了，一个装修完善、信息丰富的店铺页面是建立用户信任的基础，而合理的商品组合则是促进下单的关键。通过抖音来客App，按照五个步骤即可完成店铺的基础装修，为后续的经营打下坚实基础。
