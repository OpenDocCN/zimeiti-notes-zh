# 抖音运营教程：P38：如何衡量选题质量

## 📝 概述
在本节课中，我们将学习如何判断一个抖音视频选题是否优秀。我们将介绍五个核心标准，帮助你从众多选题中筛选出最具潜力的那一个，从而提升视频的吸引力和传播效果。

---

## 🔍 选题衡量标准
上一节我们介绍了如何寻找和拓展选题。本节中，我们来看看如何用具体的标准去衡量这些选题的好坏。一个好的选题往往能同时满足多个标准。

以下是衡量选题质量的五个核心标准：

![](img/10feb7116e83f864a86b4833855f1886_1.png)

1.  **低成本**
    *   **含义**：指用户无需付出过多时间、金钱或精力，就能达到视频所承诺的效果。
    *   **例子**：“玩手机躺着也能减肥”之所以吸引人，是因为它承诺了“减肥”这个高价值结果，但过程却是“躺着玩手机”这种极低的行为成本。相比之下，“适合婚礼的50首歌曲”因数量过多（成本高）而降低了吸引力。

2.  **戳中痛点**
    *   **含义**：选题直接击中了用户某个强烈、迫切的需求或烦恼。
    *   **例子**：用户已经看中了一款口红（如李佳琦推荐的999色号），但觉得价格太贵。此时，一个“该色号五折优惠”的选题就精准地戳中了“价格过高”这个痛点。

3.  **场景熟悉**
    *   **含义**：选题基于用户日常生活中常见、高频率出现的场景或经历，容易引发共鸣。
    *   **例子**：“你有没有在电影院里碰到这样一类人？”或“放假回家妈妈总唠叨的几句话”。这类选题利用共同记忆，让用户迅速产生“我也遇到过”的熟悉感。

4.  **需求高频**
    *   **含义**：选题涉及的需求是用户经常遇到的，而非一次性或小众需求。
    *   **例子**：“厨房收纳神器”、“新手化妆流程”、“国庆旅游好物清单”。这些都属于很多人会反复遇到的问题或常规性需求。

5.  **人群精准**
    *   **含义**：选题明确针对某一特定人群，内容能深深吸引该群体。
    *   **例子**：“12种女生干练穿搭方法”精准定位职场女性；“订婚这些流程你知道吗”精准定位适婚情侣。

---

## 🛠️ 标准应用与筛选流程
我们经常使用的核心标准是 **低成本** 和 **戳中痛点**。一个选题如果能同时满足这两点，往往就成功了一大半。

在实际操作中，你可以遵循以下流程：

1.  **生成选题**：使用九宫格法等工具， brainstorm 出大量选题。
2.  **套用标准**：将每个选题代入上述五个标准中进行衡量。
3.  **进行筛选**：一个选题通常只需符合其中两到三点，即可被视为优质选题，值得进一步拍摄。
4.  **接受删减**：并非所有生成的选题都是好选题，筛选和删减是必要步骤。
5.  **优化调整**：如果觉得所有选题都不够好，可以回到文案或场景本身进行优化和调整。

## 📌 总结
本节课我们一起学习了衡量抖音选题质量的五个关键标准：**低成本**、**戳中痛点**、**场景熟悉**、**需求高频** 和 **人群精准**。掌握这些标准后，你就能像拥有一个“选题过滤器”，从海量灵感中快速识别出最具爆款潜力的方向，让你的内容创作事半功倍。
