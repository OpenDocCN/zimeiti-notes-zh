# 微信视频号运营教程：P9：7.账号定位与素材获取

## 概述
在本节课中，我们将学习微信视频号账号运营的两个核心环节：账号定位的初期处理与原创视频素材的获取方法。课程将详细解答是否需要隐藏旧作品、如何快速获取初始粉丝，并介绍一个获取高质量剪辑素材的途径。

## 账号初期处理与粉丝积累

上一节我们介绍了账号定位的基础概念，本节中我们来看看账号运营初期的具体操作。

关于发布频率，素材无需着急，坚持一天发布一条内容即可。

我们的首要目标是积累100个粉丝，之后再解决素材问题。请按步骤逐个解决。

## 原创视频素材问题解决方案

解决了粉丝目标后，接下来我们解决原创视频素材的问题。

因为很多同学提到没有像示例中那样优质的拍摄素材。这一点可以理解，示例素材也是由我提供给那位同学的。既然可以提供给他，同样也可以提供给大家。

![](img/1ef4d86d0aeab7b2c28d5e4997f16bb5_5.png)

![](img/1ef4d86d0aeab7b2c28d5e4997f16bb5_6.png)

我了解多数同学目前的情况：不会拍摄、不会剪辑、不会制作视频。如果你也是这种情况，请确认。

![](img/1ef4d86d0aeab7b2c28d5e4997f16bb5_7.png)

针对这个普遍问题，今天来到课堂的所有同学，都将获得一项重磅福利。

以下是福利的具体内容：
*   每位同学都可以从我这里领取30个视频素材。

或许有同学担心素材是否充足。无需担心，我向大家展示我的独家素材库。这个素材库前后投入接近200万元。其中许多素材已经是剪辑好的成品。

例如，在“搞笑娱乐”类目下，我的专业剪辑师和选材师制作的素材质量很高。我们可以感受一下。

（此处插入视频示例描述：一段剪辑好的搞笑短视频，包含音乐和趣味画面。）

![](img/1ef4d86d0aeab7b2c28d5e4997f16bb5_9.png)

这些素材都是已剪辑完成，直接提供给各位的。内容涵盖搞笑娱乐、生活妙招等多种类型。

关于素材储备，我的公司有2000多名员工，其中剪辑人员就有500人，占公司四分之一。每位成熟剪辑师的底薪为8000元，每月在素材剪辑上的工资支出就超过100万元。因此素材储备充足，且无需担心重复，因为链接一旦被领取就会失效。

## 素材领取方式

那么，如何进行领取呢？我的要求很简单。

以下是领取素材需要满足的三个条件：
1.  在公屏上打出“想”字。
2.  在课堂中积极互动。
3.  完整听完课程。我们的后台可以统计听课时长与互动次数。

![](img/1ef4d86d0aeab7b2c28d5e4997f16bb5_11.png)

满足条件后，请按以下步骤操作：
*   找到向你发送听课链接的助理老师。
*   并向助理老师说明：“是张老师让我来领取的。”

这样说明的原因很简单：助理老师与大家非亲非故，没有义务直接提供素材。但提及我的名号则不同，因为我是给他们发工资的人。在外行事，需要讲究方式方法。
