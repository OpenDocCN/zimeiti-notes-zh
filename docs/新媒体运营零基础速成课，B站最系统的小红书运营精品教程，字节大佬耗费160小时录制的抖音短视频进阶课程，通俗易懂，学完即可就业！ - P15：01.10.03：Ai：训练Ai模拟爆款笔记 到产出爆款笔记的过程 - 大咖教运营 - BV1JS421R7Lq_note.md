# 新媒体运营零基础速成课：P15：01.10.03：AI：从分析爆款笔记到产出爆款笔记 🧠

在本节课中，我们将学习如何利用AI工具（如ChatGPT）深入分析平台上的爆款内容，并基于分析结果，指导AI生成具备爆款潜质的新笔记。整个过程将分为两个核心步骤：**分析学习**与**模仿创作**。

上一节我们介绍了通过“喂资料”的方式，让AI理解特定平台的文案风格。虽然以旅游攻略为例的最终产出可能不理想，但方法是正确的。关键在于让AI透彻分析你提供的优质内容，从而学习到该平台的文案写作方式和风格。

本节中，我们来看看如何更进一步：让AI分析具体的爆款笔记，提炼其成功要素，并最终指导AI创作出新的内容。

## 第一步：明确分析目标与指令

![](img/652e366869dd905e6723435be77a1f0a_11.png)

![](img/652e366869dd905e6723435be77a1f0a_13.png)

![](img/652e366869dd905e6723435be77a1f0a_15.png)

![](img/652e366869dd905e6723435be77a1f0a_16.png)

在开始分析前，我们需要给AI下达清晰、具体的指令。分析的目标应聚焦于以下几个核心点：

以下是需要AI重点分析的几个方面：

1.  **爆款因素**：分析内容为何能成为爆款，例如是否引发了情感共鸣、提供了实用价值、讲述了逆袭故事等。
2.  **文案结构**：
    *   **开篇**：用什么方式（如提问、制造焦虑、引用名人）吸引用户继续阅读。
    *   **中间**：提供了哪些内容来戳中用户痛点，锁住用户停留时间。
    *   **结尾**：如何设置“钩子”（如鼓励互动、引导私信、提供福利）促使用户产生下一步行为。
3.  **关键词策略**：分析笔记中是否刻意植入了特定关键词。如果有，需分析其**出现频率**、**分布方式**以及**与中文分词法的结合情况**。

![](img/652e366869dd905e6723435be77a1f0a_18.png)

![](img/652e366869dd905e6723435be77a1f0a_20.png)

给AI的指令可以这样组织：
```text
我将提供一篇小红书平台上的爆款笔记文案，请你分析：
1. 它的爆款因素有哪些？
2. 它的整体结构：开篇如何吸引用户？中间如何戳中痛点？结尾如何引导互动？
3. 分析文案是否结合中文分词法设计了关键词。如果有，请指出关键词是什么、出现频率如何。
```

## 第二步：选择与分析爆款案例

![](img/652e366869dd905e6723435be77a1f0a_32.png)

![](img/652e366869dd905e6723435be77a1f0a_34.png)

我们以“考研”领域为例，选取几篇高互动数据（如高点赞、收藏）的笔记作为分析样本。

将选中的爆款笔记全文复制，并逐一提交给AI，要求其根据上述指令进行分析。AI会给出类似以下的反馈：

![](img/652e366869dd905e6723435be77a1f0a_36.png)

![](img/652e366869dd905e6723435be77a1f0a_38.png)

![](img/652e366869dd905e6723435be77a1f0a_40.png)

*   **爆款因素**：情感共鸣、提供明确的备考计划、分享逆袭故事、给出针对性建议、使用表情符号增加亲和力、结尾传递正能量等。
*   **文案结构**：开篇描述备考焦虑引发共鸣 → 中间分享个人经验与详细规划 → 结尾鼓励并引导互动。
*   **关键词分析**：多数笔记没有非常刻意地设计分词关键词，词汇以直接、常用的备考术语为主。

## 第三步：整合爆款要素并生成大纲

![](img/652e366869dd905e6723435be77a1f0a_59.png)

![](img/652e366869dd905e6723435be77a1f0a_61.png)

![](img/652e366869dd905e6723435be77a1f0a_62.png)

在分析了多篇爆款笔记后，我们需要让AI对分析结果进行去重和整合，提炼出该领域爆款内容的共性要素。

可以给AI如下指令：
```text
请将你前面分析过的所有考研爆款笔记的爆款因素总结整合成一个清晰的列表。
```

AI会生成一个整合后的列表，例如：情感共鸣、逆袭故事、明确的时间规划、各科备考干货、资源技巧推荐、正能量鼓励、使用表情符号等。

接下来，基于这些提炼出的爆款要素，指导AI生成新笔记的创作大纲。

![](img/652e366869dd905e6723435be77a1f0a_64.png)

指令如下：
```text
请结合你总结的爆款因素，为我规划一篇关于考研的小红书笔记大纲。大纲需要细分到具体的小节点，在我确认后再进行文案创作。
```

![](img/652e366869dd905e6723435be77a1f0a_66.png)

AI会提供一个大纲，例如：
1.  开篇：用引人入胜的话语或问题吸引注意，引入逆袭小故事。
2.  主题介绍：简要说明笔记核心内容。
3.  专业选择建议。
4.  详细备考时间规划（分政治、英语、数学、专业课）。
5.  时间管理与心态调整方法。
6.  结尾：鼓励互动，引导关注或私信。

---

## 第四步：根据大纲生成完整文案

大纲确认后，即可令AI根据大纲和爆款要素生成完整的笔记文案。此时指令需更加具体。

![](img/652e366869dd905e6723435be77a1f0a_68.png)

![](img/652e366869dd905e6723435be77a1f0a_70.png)

![](img/652e366869dd905e6723435be77a1f0a_71.png)

![](img/652e366869dd905e6723435be77a1f0a_73.png)

指令示例：
```text
请按照你提供的大纲，撰写具体的文案。
要求：
1. 文案结构需严格遵循爆款笔记的要素和结构。
2. 需包含一个简单的“逆袭”小故事来引入读者情绪。
3. 需包含适量的emoji表情。
4. 需尝试结合中文分词法，自然地融入“考研”、“备考计划”等关键词。
```

AI将生成一篇包含故事引入、干货分享、情感鼓励和互动引导的完整笔记。生成的文案可能已具备基础的可发布质量，但对于更高要求的运营，可以在此基础上提出更细致的修改指令（如调整语气、丰富细节、优化排版）进行迭代优化。

## 进阶提示：使用定制化AI助手

为了获得更稳定、更符合个人风格的结果，可以考虑使用如**ChatGPT的GPTs**等功能创建定制化助手。

你可以：
*   上传多篇符合你要求的优秀笔记作为学习资料。
*   在助手配置中详细说明你对标题、开头、结构、表情符号使用频率、关键词植入方式等的偏好。
*   这样创建的私人AI助手能更一致地理解并输出你想要的文案风格，减少反复修正指令的麻烦。
