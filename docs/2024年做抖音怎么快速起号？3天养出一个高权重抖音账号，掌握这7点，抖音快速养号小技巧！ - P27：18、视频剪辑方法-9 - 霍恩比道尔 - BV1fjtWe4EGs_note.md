我们使用**剪映**进行剪辑。请确保下载的是黑色图标版本。剪映是抖音官方出品的剪辑工具，支持手机版和电脑版。我们先讲解手机版的操作。

![](img/96261fe2d2c70512872fd2aaa8e3f5bb_41.png)

![](img/96261fe2d2c70512872fd2aaa8e3f5bb_43.png)

以下是开始剪辑的步骤：

![](img/96261fe2d2c70512872fd2aaa8e3f5bb_45.png)

![](img/96261fe2d2c70512872fd2aaa8e3f5bb_47.png)

![](img/96261fe2d2c70512872fd2aaa8e3f5bb_49.png)

![](img/96261fe2d2c70512872fd2aaa8e3f5bb_51.png)

1.  **导入素材**：打开剪映App，点击“开始创作”。选择你需要剪辑的视频片段，点击“添加”。如果视频由多个片段组成，建议先剪辑好第一个片段，再点击轨道上的“+”号导入下一个片段，依次处理，避免混乱。
2.  **调整构图与比例**：选中视频片段，向右滑动工具栏，点击“编辑”。首先点击“旋转”，将画面调整到正确的方向。如果需要裁剪画面比例，点击“裁剪”，可以选择1:1、4:3、16:9等常用比例。调整完毕后，点击对勾确认。
3.  **剪辑视频内容**：播放视频，找到不需要的部分（如开头沉默、结尾无关动作）。将时间轴（白色竖线）移动到需要分割的位置，选中视频片段，点击“分割”。然后选中需要删除的片段，点击“删除”。重复“选中 -> 分割 -> 选中 -> 删除”的步骤，即可流畅地剪掉多余部分。
4.  **添加背景音乐**：点击底部工具栏的“音频”。如果你有预先准备好的配音文件（如从淘金阁导出），可以点击“提取音乐”导入。如果没有，则点击“添加音乐”，在曲库中根据分类（如美食、美妆）选择或搜索合适的音乐，试听后点击“使用”。
5.  **剪辑与调整音乐**：如果音乐过长，处理方法与剪辑视频相同：将时间轴移到目标位置，选中音乐轨道，点击“分割”，然后删除多余部分。选中音乐轨道，点击“音量”，可以调整背景音乐的音量大小，使其不掩盖人声。
6.  **添加与编辑字幕**：点击底部工具栏的“文字”，然后选择“识别字幕”。建议选择“仅视频”，开始识别。系统会自动为视频中的人声生成字幕。识别完成后，检查字幕是否有错别字，双击字幕即可进行修改。你还可以调整字幕的字体、颜色和样式，使其与视频风格更搭配。
7.  **导出高清视频**：所有编辑完成后，点击右上角的“导出”。在导出设置中，为了获得最佳抖音播放效果，请将**帧率设置为30**，**分辨率拉至最高**（如1080p或4K）。这样可以最大程度保证视频清晰度，避免被压缩。导出过程中请勿锁屏或切换程序，耐心等待完成。导出后，可直接分享至抖音。

![](img/96261fe2d2c70512872fd2aaa8e3f5bb_55.png)

视频剪辑的核心操作可以总结为以下流程：
```
导入素材 -> 调整画面(旋转/裁剪) -> 分割删除冗余片段 -> 添加并调整背景音乐 -> 智能识别并编辑字幕 -> 高清设置(帧率30，分辨率最高)导出
```

这个软件叫做**爱提词**。它的作用是让你在拍摄时无需背诵，可以照着提词器朗读。



![](img/96261fe2d2c70512872fd2aaa8e3f5bb_73.png)

![](img/96261fe2d2c70512872fd2aaa8e3f5bb_75.png)

![](img/96261fe2d2c70512872fd2aaa8e3f5bb_77.png)

以下是使用爱提词的方法：

![](img/96261fe2d2c70512872fd2aaa8e3f5bb_81.png)

![](img/96261fe2d2c70512872fd2aaa8e3f5bb_83.png)

![](img/96261fe2d2c70512872fd2aaa8e3f5bb_85.png)

![](img/96261fe2d2c70512872fd2aaa8e3f5bb_87.png)

1.  **创建提词稿**：打开爱提词App，点击右下角的“+”号。输入标题（如“情感语录”），然后将你的台词内容粘贴或输入到正文框中，点击保存。
2.  **使用提词器拍摄**：保存后，点击“开始提词”，选择“录制模式”。此时手机屏幕会像提词器一样滚动显示你的台词，你可以一边看一边录制。再也不用担心忘词。
3.  **个性化设置**：点击屏幕中央的“设置”图标，可以调整台词文字的**字体大小和滚动速度**，方便近视的同学观看。点击左下角的“美颜”图标，可以开启美颜功能，适度调整瘦脸、大眼、磨皮等参数，让拍摄效果更自然美观，避免原相机的生硬感。
4.  **获取灵感**：App内还设有“爆款文案”板块，涵盖了职场、情感、生活、科普等多个领域的文案素材，可以为你的内容创作提供灵感。
