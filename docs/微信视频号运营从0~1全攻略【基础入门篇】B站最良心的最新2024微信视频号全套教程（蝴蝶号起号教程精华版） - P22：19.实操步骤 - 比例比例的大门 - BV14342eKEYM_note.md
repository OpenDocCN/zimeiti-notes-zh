# 微信视频号运营从0~1全攻略：P22：19.实操步骤 - 开通与上架商品

在本节课中，我们将学习微信视频号开通商品橱窗权限以及上架商品的具体操作步骤。整个过程清晰简单，即使是初学者也能轻松跟上。

上一节我们介绍了视频号的基础概念，本节中我们来看看如何实际操作，迈出视频号带货的第一步。

## 第一步：打开微信
操作的第一步是启动微信应用。这是所有后续操作的基础。

![](img/e363c4b480f84e7232b6c9cce759b097_1.png)

![](img/e363c4b480f84e7232b6c9cce759b097_2.png)

## 第二步：进入“发现”页
在微信主界面，点击底部的 **“发现”** 选项卡。

## 第三步：找到视频号入口
在“发现”页面中，将视线向上移动。您会看到 **“视频号”** 的入口，其图标是一个黄色的蝴蝶标志。点击该入口进入视频号。

![](img/e363c4b480f84e7232b6c9cce759b097_4.png)

![](img/e363c4b480f84e7232b6c9cce759b097_6.png)

![](img/e363c4b480f84e7232b6c9cce759b097_1.png)
![](img/e363c4b480f84e7232b6c9cce759b097_2.png)

## 第四步：进入个人中心
进入视频号后，点击页面右上角的人形头像图标，进入个人中心。

![](img/e363c4b480f84e7232b6c9cce759b097_8.png)

![](img/e363c4b480f84e7232b6c9cce759b097_4.png)
![](img/e363c4b480f84e7232b6c9cce759b097_6.png)

![](img/e363c4b480f84e7232b6c9cce759b097_9.png)

## 第五步：进入创作者中心
在个人中心页面，向下滑动，直到页面最底部。找到并点击 **“创作者中心”**。如果您的账号是新注册的且未发布过作品，此选项可能不会显示，需要先发布一条视频。

## 第六步：找到带货中心
进入创作者中心后，在页面中找到 **“带货中心”** 的选项，其图标类似一个小卖部。点击进入。

![](img/e363c4b480f84e7232b6c9cce759b097_8.png)
![](img/e363c4b480f84e7232b6c9cce759b097_9.png)

接下来，系统会根据您的账号状态呈现不同界面。以下是两种情况的处理方式。

![](img/e363c4b480f84e7232b6c9cce759b097_11.png)

### 情况一：开通橱窗权限（未开通用户）
如果您尚未开通商品橱窗功能，点击“带货中心”后会跳转到开通权限的界面。

以下是开通权限的具体步骤：
1.  **选择带货身份**：页面会要求您选择带货身份。通常有三个选项：“我是商家”、“我是达人”、“我是商家且已有视频号”。
2.  **身份选择建议**：对于大多数个人创作者，建议选择 **“我是达人”**。即使您是厂家，也建议先用这个身份开通，因为其产品库是共享的。一个身份证可以认证两个账号，第二个账号可以选择“我是商家”。
3.  **缴纳保证金**：选择“我是达人”后，会跳转到保证金缴纳界面。您需要点击 **“去交纳”**，通过微信支付100元人民币的保证金即可完成开通。

![](img/e363c4b480f84e7232b6c9cce759b097_11.png)
![](img/e363c4b480f84e7232b6c9cce759b097_13.png)

![](img/e363c4b480f84e7232b6c9cce759b097_13.png)

### 情况二：上架商品（已开通用户）
如果您已经开通了橱窗权限，点击“带货中心”后会直接进入选品和商品上架的管理界面。

整个操作流程的核心路径可以总结为以下代码逻辑：
```plaintext
微信 -> 发现 -> 视频号 -> 个人中心（头像）-> 创作者中心 -> 带货中心
```
进入“带货中心”后，系统会自动判断：
-   **if** 未开通权限：跳转至 **开通流程**（选择身份->缴纳保证金）。
-   **else** 已开通权限：进入 **商品上架管理后台**。

![](img/e363c4b480f84e7232b6c9cce759b097_15.png)

本节课中我们一起学习了从打开微信到成功开通视频号商品橱窗权限的全过程。关键点在于找到正确的操作路径，并根据系统提示完成身份选择与保证金缴纳。开通后，您就可以在带货中心开始选择并上架商品，为视频号带货做好准备。
