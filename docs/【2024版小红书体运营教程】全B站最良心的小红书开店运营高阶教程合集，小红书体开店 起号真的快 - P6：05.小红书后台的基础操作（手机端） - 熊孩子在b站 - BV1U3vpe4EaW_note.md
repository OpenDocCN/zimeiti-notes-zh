# 小红书开店运营教程：P6：05.小红书后台的基础操作（手机端） 📱

![](img/53f04c27a976d844be4ea7302cf0f46d_0.png)

## 概述
在本节课中，我们将学习如何使用手机端的小红书千帆系统来设置和管理店铺后台。内容涵盖从登录系统、客服管理、地址设置到商品发布和营销工具的基础操作。

---

## 系统登录与首页概览
上一节我们介绍了开店前的准备，本节中我们来看看如何操作手机端的后台系统。

首先，打开手机应用商店，搜索并下载“小红书千帆”应用。这是官方的小红书商家后台管理系统。

登录后，您会看到系统首页。对于一个新店铺，页面可能比较简洁。首页通常包含“消息”中心和“全部工具”等入口。

---

## 客服与消息中心
在首页点击“消息”图标，即可进入客服与消息中心。

以下是消息中心的主要功能：
*   **客服接待**：当买家咨询店铺商品或服务时，所有消息会在这里显示，您可以在此进行回复。
*   **系统通知**：平台发送的各类通知，例如店铺违规提醒、活动公告等，都会在此处集中展示。
*   **客服配置**：您可以在此管理和配置店铺的客服设置。

---

## 店铺初始化与工具
返回首页，您会看到“初始化”引导和“全部工具”选项。这是进行店铺基础设置和日常管理的核心区域。

点击“全部工具”，会展开一个功能列表。

以下是“全部工具”内的核心功能模块：
*   **内容管理**：`发布作品`、`笔记管理`。
*   **商品管理**：`发布商品`、`管理商品`。
*   **订单管理**：`全部订单`、`售后订单`。
*   **数据分析**：`数据中心`。
*   **店铺设置**：`发布类目`、`品牌类目`、`店铺装修`、`群管理`。

---

## 基础设置：发货地址与物流模板
店铺正常运营的第一步是完善基础信息。在“成长”任务或“全部工具”中，找到并设置“发货地址”。

**地址填写步骤**：
1.  进入地址管理页面。
2.  填写联系人（可填自己）、所在地区及详细地址。
3.  填写有效的手机号码。
4.  **关键点**：在“买家留言”或备注栏，**必须填写“退货前请联系客服”**。如果此处空白，买家退货申请可能被平台自动通过，造成损失。
5.  保存地址。

地址设置完成后，需要配置“物流模板”。

**物流模板设置**：
物流模板用于定义不同地区的运费政策。系统有默认模板，但建议根据实际情况调整。
*   **代码示例（策略逻辑）**：
    ```python
    if 地区 in ["新疆", "西藏", "青海"]:
        运费策略 = "不配送"  # 或设置为“买家承担高额运费”
    else:
        运费策略 = "包邮"  # 或设置为“标准运费”
    ```
*   **操作**：在物流设置中，您可以创建新模板，例如设置全国大部分地区包邮，而对新疆、西藏等偏远地区设置不配送或到付运费。

---

## 营销工具：优惠券设置
基础设置完成后，可以利用营销工具吸引顾客。优惠券是常用的促销手段。

在“全部工具”或“营销”板块找到“优惠券”或“活动报名”入口。

以下是可创建的优惠券类型：
*   **店铺通用券**：全店商品均可使用的优惠券。
*   **指定商品券**：仅限特定商品使用的优惠券。
*   **店铺新人券**：针对首次访问店铺用户发放的优惠券。
您可以查看和管理“正在生效”、“已失效”的优惠券活动。

---

## 店铺认证与保证金
为了保证交易安全和平台合规，需要完成店铺认证并缴纳保证金。

![](img/53f04c27a976d844be4ea7302cf0f46d_2.png)

**操作流程**：
1.  在店铺设置中找到“认证信息”，根据店铺类型（个人店、企业店等）提交资料。
2.  缴纳保证金。金额根据您经营的商品类目而定，不同类目保证金不同。
3.  **重要步骤**：务必开通“微信支付”和“支付宝支付”收款功能。按照页面指引，绑定您的身份证和银行卡，买家才能通过这两种方式付款。
