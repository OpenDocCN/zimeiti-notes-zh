# 新媒体运营：P10：小红书平台笔记审核规则 📝

在本节课中，我们将要学习小红书平台笔记的流量推荐与分发机制。我们将详细拆解一篇笔记从发布到推荐给用户的完整流程，并重点解析其中的审核规则与账号权重概念。

上一节我们介绍了小红书平台的基本运营逻辑，本节中我们来看看笔记发布后的具体审核流程。

## 笔记审核的三个阶段

小红书笔记发布后，会依次经历三个审核阶段：**机器审核**、**去重检测**和**用户匹配**。大多数无问题的笔记在机器审核阶段即可通过。

### 1. 机器审核

笔记首先会被上传至审核系统。该系统依据预设规则进行初步判断。

*   **内容安全**：检测笔记中是否包含**敏感词**或**敏感图片**等违规内容。
*   **基础质量**：评估笔记的**字数**、**广告词出现频率**、**图片清晰度**以及文字内容的可识别性。

如果笔记触发严重违规规则，将被直接封禁。如果内容存在疑似违规或机器难以判断的情况，则会被标记并转入**人工审核**阶段。若笔记完全符合规则，则进入下一阶段。

## 账号权重的影响

![](img/8afb812644e29472f0476e2e5be57419_57.png)

![](img/8afb812644e29472f0476e2e5be57419_59.png)

在机器审核阶段，系统除了判断违规，还会为笔记打一个“基础质量分”。这个分数与发布账号的**权重**密切相关。账号权重是一个隐形分值，代表了平台系统对账号质量的综合评价。

![](img/8afb812644e29472f0476e2e5be57419_61.png)

![](img/8afb812644e29472f0476e2e5be57419_63.png)

![](img/8afb812644e29472f0476e2e5be57419_65.png)

我们可以用一个简单的模型来理解账号权重：

![](img/8afb812644e29472f0476e2e5be57419_67.png)

![](img/8afb812644e29472f0476e2e5be57419_69.png)

![](img/8afb812644e29472f0476e2e5be57419_71.png)

```
假设账号权重分值范围是 0 到 10。
```

![](img/8afb812644e29472f0476e2e5be57419_73.png)

![](img/8afb812644e29472f0476e2e5be57419_75.png)

*   **新注册账号**：权重接近 **0**。此时发布笔记，初始推荐流量池很小（可能仅几十个曝光）。因为系统尚未对账号建立信任标签，笔记无法精准推送给潜在兴趣用户，互动数据难以提升，导致笔记很难突破初始流量池。
*   **成熟垂直账号**：通过持续发布**优质、垂直领域**的内容（如专注彩妆），账号在该领域的权重会逐渐提升至 **2、3、4 或更高**。高权重账号发布笔记时，会获得更大的初始推荐流量池和更精准的用户推送。

![](img/8afb812644e29472f0476e2e5be57419_77.png)

![](img/8afb812644e29472f0476e2e5be57419_79.png)

![](img/8afb812644e29472f0476e2e5be57419_81.png)

提升权重的关键在于账号的**长期正向行为**：避免违规、保持内容垂直度、稳定更新周期等。需要特别注意的是，“权重”是便于理解的概念，平台并无公开的权重分数据。

![](img/8afb812644e29472f0476e2e5be57419_83.png)

## 关于人工审核的补充说明

人工审核主要对机器标记的**违规或疑似违规**内容进行二次复核。但由于内容量巨大、审核成本高，并非所有笔记都会进入此环节。

以下是需要了解的重点：

*   **高敏感领域**：在政治、色情擦边等法律严禁的领域，人工审核的介入力度会非常大。
*   **红线规则**：若笔记内容触发平台的机器审核红线，**大概率会直接封禁**，且可能无人工程序复核。
*   **申诉渠道**：如果创作者认为自己的笔记被误判，可以通过官方渠道进行申诉。但对于新账号，建议**尽量避免任何违规**，因为早期违规可能对账号产生隐形的负面影响。

本节课中我们一起学习了小红书笔记审核的三个核心阶段（机器审核、去重检测、用户匹配），并理解了隐形但至关重要的**账号权重**概念。掌握这些规则，是让你的笔记顺利进入推荐池、获得流量的第一步。
