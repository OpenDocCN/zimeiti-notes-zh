小红书运营教程：P3：第2节：小红书平台检测规则详解 🔍

在本节课中，我们将要学习小红书平台的核心检测规则。了解这些规则是安全、有效运营账号的基础，能帮助我们避免违规，确保内容顺利发布和传播。

上一节我们介绍了账号定位，本节中我们来看看平台的具体监管机制。小红书的检测系统主要覆盖以下几个关键方面。

以下是小红书重点检测的四个内容区域：
1.  **个人资料、笔记内容、私信及评论**：这些位置不得出现微信号、微信二维码、代购、水印、推广淘宝店铺等营销词汇。
2.  **手机设备**：平台会检测登录设备。建议同一台设备最多只登录三个账号，超出可能引发风险。
3.  **笔记内容**：内容中不得诱导用户，例如通过“关注领奖”、“点赞抽奖”等方式吸引互动。这是平台明令禁止的诱导行为。
4.  **内容原创度**：小红书对原创内容的保护程度较高。切勿将同一篇文章在多个账号上重复发布。原则是：一篇文章，一个账号。

除了上述通用规则，另一个需要特别注意的环节是敏感词检测。平台对某些特定领域的词汇管控严格。

![](img/720be2755549d4f28e9cfee7bc02e798_1.png)

以下是常见的敏感词类别示例，在创作时应尽量避免触及：
*   手机卡
*   金融
*   电子烟
*   代购
*   医疗用品

那么，如果在撰写内容时不慎加入了未知的敏感词，该如何提前发现并修改呢？这时就需要借助外部工具进行检测。

![](img/720be2755549d4f28e9cfee7bc02e798_3.png)

![](img/720be2755549d4f28e9cfee7bc02e798_5.png)

以下是两个可用于检测小红书敏感词的网站工具，使用流程基本一致：
1.  **网站A（无需注册）**：
    *   打开网站，将写好的文案粘贴到输入框中。
    *   点击“立即检测”按钮。
    *   系统会高亮或列出文案中存在的敏感词（例如“代购”），方便我们进行修改。
2.  **网站B（需注册）**：
    *   注册登录后，其使用方式与网站A类似。
    *   将文案粘贴后检测，网站会返回过滤结果，指出敏感词。

![](img/720be2755549d4f28e9cfee7bc02e798_7.png)

通常，使用第一个网站进行检测已经足够。我们只需在发布前，将文章内容粘贴到这些检测网站中，就能提前识别并规避风险，确保内容合规。
