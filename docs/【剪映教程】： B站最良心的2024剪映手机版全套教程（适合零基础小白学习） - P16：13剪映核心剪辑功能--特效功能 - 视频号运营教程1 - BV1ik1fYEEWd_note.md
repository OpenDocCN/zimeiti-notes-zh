# 剪映教程：P16：剪映核心剪辑功能——特效功能 🎬

![](img/e00755c7561576062ceac8b1483c7231_1.png)

在本节课中，我们将要学习剪映App中一个非常有趣且强大的功能——特效。特效功能借鉴了短视频平台中流行的道具元素，通过在视频中添加各种动态效果，可以让你的视频变得更加生动、有趣，并大幅提升视频的整体氛围和观感。

上一节我们介绍了剪映的其他核心功能，本节中我们来看看如何为视频添加和运用特效。

## 特效功能概述

![](img/e00755c7561576062ceac8b1483c7231_3.png)

剪映的特效功能内置了丰富的特效素材库。通过在视频中植入合适的特效元素，可以极大地提升视频的画面感和氛围。这在制作变装类、反转类或美妆类视频时尤其有用。例如，曾经非常流行的“变装甩头发”视频，就大量运用了特效来制造惊艳的转场和视觉效果。

## 特效功能实操演练

![](img/e00755c7561576062ceac8b1483c7231_8.png)

接下来，我们通过一个具体的案例，来演示如何为视频添加特效。我们将模仿一个包含“模糊变清晰”、“背景氛围”和“动感抖动”特效的短视频。

![](img/e00755c7561576062ceac8b1483c7231_10.png)

![](img/e00755c7561576062ceac8b1483c7231_12.png)

以下是操作步骤：

1.  **导入素材**：打开剪映App，点击“开始创作”，选择你想要编辑的视频素材，将其导入到项目中。
2.  **添加第一个特效（模糊变清晰）**：
    *   将时间轴移动到视频开头。
    *   点击底部工具栏中的 **`特效`** 按钮。
    *   在特效面板中，选择 **`基础`** 分类。
    *   找到并点击 **`变清晰`** 特效，即可为视频开头添加一个从模糊到清晰的效果。
3.  **添加第二个特效（背景氛围）**：
    *   将时间轴移动到人物跳舞的片段。
    *   再次点击底部 **`特效`** 按钮。
    *   选择 **`梦幻`** 或其他合适的分类（例如案例中的“妖企”特效）。
    *   点击你想要添加的背景氛围特效（如“邀请”），该特效就会应用到当前时间点。
4.  **添加第三个特效（动感抖动）**：
    *   将时间轴移动到“甩头发”的瞬间。
    *   点击 **`特效`** -> **`新增特效`**。
    *   选择 **`动感`** 分类。
    *   添加如 **`横纹故障`** 或 **`抖动`** 这类特效，以增强动作的冲击力。
5.  **调整与叠加**：
    *   添加每个特效后，你可以在时间轴上拖动其左右边缘，以调整特效持续的时长。
    *   特效功能支持**叠加使用**，你可以通过多次点击“新增特效”，在同一段视频上添加多个不同的效果，创造出更复杂的视觉体验。
6.  **完善作品**：最后，别忘了为视频添加合适的背景音乐，音画结合会让整体效果更加出色。

## 特效素材库简介

![](img/e00755c7561576062ceac8b1483c7231_16.png)

点击剪映的一级菜单中的 **`特效`**，即可进入丰富的特效素材库。库中特效被分为多个类别，例如：

*   **基础**：包含`开幕`、`闭幕`、`变清晰`、`变模糊`、`电影感`等常用效果。
*   **梦幻**：提供各种氛围光效和唯美背景。
*   **动感**：包含`抖动`、`闪白`、`故障`等能增强节奏感的特效。
*   **复古**：提供`录像机`、`噪点`、`方格`等怀旧风格效果。

在创作时，可以根据视频内容和风格大胆尝试不同的特效组合。
