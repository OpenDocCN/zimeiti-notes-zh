# 拼多多运营实操教程：P14：如何设置运费模板 🚚

在本节课中，我们将学习拼多多店铺运营中一个非常实用的功能——运费模板的设置。运费模板能帮助我们高效地管理不同商品的运费规则，是店铺物流设置的基础。

## 什么是运费模板？

![](img/d69cf4279d90bbe67b063df64fc39d0e_2.png)

上一节我们介绍了店铺的基础设置，本节中我们来看看如何管理物流费用。运费模板是为同一批商品设置统一运费规则的模板。当你修改模板中的配送地区和运费时，所有关联该模板的商品运费信息会同步更新。

运费模板支持为不同地区设置不同的配送规则和运费。

## 设置路径在哪里？

![](img/d69cf4279d90bbe67b063df64fc39d0e_9.png)

![](img/d69cf4279d90bbe67b063df64fc39d0e_10.png)

以下是进入运费模板设置页面的具体路径。

1.  登录拼多多商家后台。
2.  在左侧导航栏找到并点击“发货管理”。
3.  在“发货管理”页面，点击“物流工具”。
4.  进入后，即可看到“运费模板”选项，点击“新建运费模板”即可开始设置。

## 运费模板设置详解

运费模板的设置主要包含三个部分：模板基本信息、包邮配送区域以及买家付邮费区域。接下来，我们将分步进行详细操作演示。

### 第一步：填写模板基本信息

首先，我们需要填写模板的基本信息。

*   **模板名称**：可自定义填写，用于区分不同的运费模板，例如“普通商品模板”或“测试模板”。
*   **发货地址**：填写真实的商品发货地址。若为一件代发，则填写供货商的发货地址。地址不实可能导致物流投诉。
*   **模板类型**：选择“自定义模板”。

### 第二步：设置包邮配送区域

包邮区域是指买家在此地区下单无需支付运费的地区。

*   **常规设置**：通常勾选大部分内地省份。对于新疆、西藏、香港、澳门、台湾等偏远地区，由于快递成本较高，一般默认不勾选（即不包邮）。
*   **特殊情况**：若您的快递合作支持且利润空间允许，可自行勾选这些地区。若与顺丰合作，可勾选“顺丰包邮”选项。

### 第三步：设置买家付邮费区域

对于不包邮的地区，您可以设置让买家承担运费。

*   **指定区域运费**：针对上一步未勾选的地区（如新疆），可以设置运费计费规则。
    *   **计费方式**：商品较轻时选择“按件数”；商品较重时选择“按重量”。
    *   **运费公式**：`首件运费 + (增加件数 * 续件运费)`。例如：首件10元，每增加一件加8元。
*   **指定条件包邮**：可设置优惠条件，例如“满5件包邮”或“满100元包邮”，以刺激消费。
*   **不配送区域**：对于完全无法发货的地区（如某些海外地区），可在此直接选择“不配送”。

完成以上所有设置后，点击“提交”按钮，运费模板即创建成功。

## 如何应用运费模板？

![](img/d69cf4279d90bbe67b063df64fc39d0e_13.png)

![](img/d69cf4279d90bbe67b063df64fc39d0e_14.png)

模板创建好后，需要在发布商品时进行关联应用。

在商品发布页面填写完其他信息后，在物流设置部分找到“运费模板”选项。点击“选择运费模板”，在列表中找到您刚刚创建好的模板并选中它即可。
