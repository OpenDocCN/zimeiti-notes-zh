在本节课中，我们将学习如何训练AI（如ChatGPT）来撰写符合小红书平台风格的文案。上一节我们介绍了如何让AI生成爆款标题，本节我们将重点探讨如何让AI理解并创作出符合平台调性的完整内容。

![](img/ed4ea56e51dce864310ff03c8e5ea605_1.png)

## 核心思路与准备工作

要让AI写出符合特定平台风格的内容，关键在于“投喂”资料。AI需要通过学习大量该平台的爆款文案，来理解其独特的语言风格、常用词汇、语气和内容结构。

![](img/ed4ea56e51dce864310ff03c8e5ea605_3.png)

以下是训练AI的几个核心步骤：

1.  **明确指令**：首先，你需要清晰地告诉AI你的任务。例如，让它分析你提供的文案，并学习其中的风格、语气和平台专用语。
2.  **提供高质量案例**：选择目标平台上具有代表性的爆款文案作为学习材料。这些文案应能体现平台的独特“文化”，例如小红书的“种草”语气、亲切的称呼（如“宝子”）、活泼的用词等。
3.  **获取反馈与迭代**：要求AI在每次学习后给出分析反馈，确保它真正理解了材料。然后，基于它的学习成果，让它进行创作，并根据结果调整你提供的指令或材料。

![](img/ed4ea56e51dce864310ff03c8e5ea605_5.png)

### 第一步：下达初始指令

![](img/ed4ea56e51dce864310ff03c8e5ea605_16.png)

首先，我们需要给AI一个明确的角色和任务。打开ChatGPT，输入以下指令：

![](img/ed4ea56e51dce864310ff03c8e5ea605_18.png)

![](img/ed4ea56e51dce864310ff03c8e5ea605_20.png)

```
接下来，我会给你提供一些小红书上的爆款笔记文案。你需要根据这些文案，分析小红书平台的文案撰写风格、特定的平台用语、常用词汇、语气等特点。每当我提供一篇文案后，请告诉我你已经学习完毕，并给出你对这篇文案在风格、语气、类型等方面的分析。
```

这个指令设定了AI的学习目标，并要求它给出反馈，便于我们监控学习过程。

![](img/ed4ea56e51dce864310ff03c8e5ea605_22.png)

### 第二步：投喂学习材料

![](img/ed4ea56e51dce864310ff03c8e5ea605_24.png)

接下来，我们需要寻找并输入高质量的小红书爆款文案。请注意，材料的选择至关重要。

**以下是选择材料时的注意事项：**

*   **避免通用内容**：例如单纯的“第一天、第二天、第三天”行程列表，这类内容缺乏平台特色，AI难以从中学习到风格。
*   **寻找特色文案**：应选择那些带有强烈个人分享色彩、情感表达（如惊喜、吐槽）、平台特色用语（如“绝了！”“踩雷”“安利”）的笔记。
*   **涵盖多种角度**：可以混合提供攻略、避坑指南、好物分享等不同类型的文案，让AI学习更全面的表达方式。

![](img/ed4ea56e51dce864310ff03c8e5ea605_26.png)

例如，我们可以输入一篇带有个人情感和避坑建议的旅游文案（需清理掉特殊字符和无关信息）：

![](img/ed4ea56e51dce864310ff03c8e5ea605_28.png)

![](img/ed4ea56e51dce864310ff03c8e5ea605_30.png)

![](img/ed4ea56e51dce864310ff03c8e5ea605_32.png)

```
标题：张家界旅游血泪教训！看完这篇帮你省下1000块！
文案：刚和闺蜜从张家界回来，真的是一把辛酸泪…宝子们，这些坑你们千万别踩！①门票一定要提前在网上订…②山上的食物又贵又难吃，建议自备干粮…虽然过程有点曲折，但风景真的绝美！附上我们的行程供参考…
```

![](img/ed4ea56e51dce864310ff03c8e5ea605_33.png)

![](img/ed4ea56e51dce864310ff03c8e5ea605_35.png)

![](img/ed4ea56e51dce864310ff03c8e5ea605_37.png)

AI在接收后，会给出类似这样的分析反馈：“已学习。这篇文案采用第一人称分享经历，增强了真实感。标题使用‘血泪教训’、‘省下1000块’等词汇制造焦虑和吸引力。文中使用‘宝子们’等亲切称呼，语气是分享与告诫并存，符合小红书常见的‘种草’与‘避坑’类笔记风格。”

![](img/ed4ea56e51dce864310ff03c8e5ea605_39.png)

![](img/ed4ea56e51dce864310ff03c8e5ea605_41.png)

重复此过程，为AI提供3-5篇不同风格但都属精品的文案进行学习。

![](img/ed4ea56e51dce864310ff03c8e5ea605_43.png)

### 第三步：让AI进行创作

在AI学习了足够多的样本后，即可发出创作指令。指令需要具体，并包含你的所有要求。

![](img/ed4ea56e51dce864310ff03c8e5ea605_45.png)

![](img/ed4ea56e51dce864310ff03c8e5ea605_47.png)

![](img/ed4ea56e51dce864310ff03c8e5ea605_49.png)

```
请结合你刚才的学习，帮我撰写一篇具有爆款潜质的“张家界5日游攻略”小红书笔记。要求如下：
1. 文案需具有小红书平台特有的亲切、分享语气。
2. 合理使用emoji表情符号。
3. 内容结构清晰，需包含行程安排、必看景点推荐、避坑建议和个人体验分享。
4. 同时，为这篇笔记生成10个符合小红书旅游赛道风格的爆款标题。
```

### 第四步：评估与优化结果

![](img/ed4ea56e51dce864310ff03c8e5ea605_55.png)

![](img/ed4ea56e51dce864310ff03c8e5ea605_57.png)

AI生成的初稿可能不尽如人意。例如，它可能只列出了景点名称，而缺乏具体的路线规划、推荐理由和真实的避坑点。

![](img/ed4ea56e51dce864310ff03c8e5ea605_59.png)

![](img/ed4ea56e51dce864310ff03c8e5ea605_61.png)

![](img/ed4ea56e51dce864310ff03c8e5ea605_63.png)

**如果结果不理想，问题通常出在以下方面：**

![](img/ed4ea56e51dce864310ff03c8e5ea605_65.png)

![](img/ed4ea56e51dce864310ff03c8e5ea605_67.png)

1.  **学习材料不足或质量不高**：提供的文案本身风格不鲜明，或数量不够。
2.  **创作指令不够具体**：没有明确要求AI写出“路线规划”、“推荐理由”和“避坑点”。
3.  **模型限制**：免费版的ChatGPT 3.5理解能力弱于4.0等更高级的模型。

![](img/ed4ea56e51dce864310ff03c8e5ea605_69.png)

![](img/ed4ea56e51dce864310ff03c8e5ea605_71.png)

**优化方法：**

![](img/ed4ea56e51dce864310ff03c8e5ea605_73.png)

*   **细化指令**：可以分步引导AI。例如，先让它列出张家界所有热门景点，再针对每个景点撰写推荐理由和注意事项，最后整合成一篇攻略。
*   **提供更专业的资料**：从百度、知乎等平台搜集更详细的景点介绍、网友评价作为补充材料投喂给AI。
*   **使用高级模型**：如果条件允许，使用ChatGPT 4.0等模型会获得质量更高的输出。
*   **新建对话并结合标题技巧**：关于生成标题，可以开启一个新的对话，直接应用上一节课所教的爆款标题生成指令和关键词库，这样生成的标题会更精准。

![](img/ed4ea56e51dce864310ff03c8e5ea605_75.png)

## 举一反三：应用于其他领域

![](img/ed4ea56e51dce864310ff03c8e5ea605_77.png)

![](img/ed4ea56e51dce864310ff03c8e5ea605_79.png)

本方法不仅适用于旅游攻略，对于任何垂直领域都有效。

![](img/ed4ea56e51dce864310ff03c8e5ea605_81.png)

![](img/ed4ea56e51dce864310ff03c8e5ea605_83.png)

![](img/ed4ea56e51dce864310ff03c8e5ea605_85.png)

**例如，如果你想做一个知识分享类账号（如考研、健身）：**

![](img/ed4ea56e51dce864310ff03c8e5ea605_87.png)

![](img/ed4ea56e51dce864310ff03c8e5ea605_89.png)

1.  **搜集资料**：找到该领域下小红书上的爆款笔记（如“考研逆袭计划”、“健身小白必看”）。
2.  **训练AI**：将这些笔记喂给AI，让它学习知识类博主的专业感、干货密度以及鼓励读者的语气。
3.  **创作内容**：让AI基于某个主题（如“如何高效背单词”）创作笔记，它就能模仿出“分点论述”、“数据支撑”、“结尾加油打气”的风格。

![](img/ed4ea56e51dce864310ff03c8e5ea605_91.png)

![](img/ed4ea56e51dce864310ff03c8e5ea605_93.png)

关键在于，你提供给AI的学习材料，决定了它能产出什么风格的内容。材料越优质、越有代表性，AI的产出就越接近你的期望。
