# 新媒体运营：P11：01.9：AI辅助账号定位工具使用教程 🧠

在本节课中，我们将学习如何利用AI人工智能模型工具来辅助完成小红书账号的定位。上一节我们介绍了账号定位的四个思考方向，本节我们将具体演示一个实用的AI工具，帮助你解决定位难题。

## 回顾：账号定位的核心

上一节我们介绍了账号定位的四个思考方向：
1.  **我是谁**：明确账号主体。
2.  **我能提供什么价值**：明确内容对用户的意义。
3.  **账号的差异化**：找到与同类账号的不同之处。
4.  **内容的可持续性**：确保有持续输出的能力。

同时，我们强调了领域定位的三个关键点：**自身擅长/了解**、**用户需求**以及**分析对标账号**。其中，用户是核心，内容必须为用户创造价值。分析对标账号的成功内容，是了解用户喜好的有效途径。

## 引入AI定位辅助工具

尽管定位逻辑已很清晰，但部分同学可能仍感到困惑。为此，本节推荐一个基于ChatGPT等AI模型开发的小红书账号辅助工具。这个工具能智能分析爆款笔记的构成要素，并进行“爆款复刻”，即围绕核心爆款因子生成新文案。它还能进行敏感词检测，避免内容被限流，以及根据你的输入生成符合小红书风格的文案。

**请注意**：该工具的输出结果可作为重要参考，但不应作为决策的唯一依据。
