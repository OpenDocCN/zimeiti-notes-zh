# 小红书运营教程：P6：小红书笔记不被收录的原因深度解析

在本节课中，我们将深入探讨小红书笔记发布后未被平台收录的常见原因。理解这些原因对于确保您的内容能够正常曝光和传播至关重要。

上一节我们介绍了笔记发布的基本流程，本节中我们来看看哪些因素会导致笔记无法被系统正常收录。

## 笔记不被收录的五大原因

以下是导致小红书笔记不被收录的五个主要原因，您可以逐一对照检查。

1.  **违规推广**
    笔记内容包含硬性广告，例如直接提及品牌名称或进行明显的商品推销。平台倾向于推荐真实、分享性质的内容，而非纯粹的商业广告。

2.  **敏感词检测**
    笔记或账号本身存在违规行为。即使笔记初期被收录，若系统后续检测到内容中含有敏感词汇，该笔记仍会被移除或限制展示。**敏感词**包括但不限于政治类、破坏民族团结、颠覆国家政权等法律法规明令禁止的词汇及其谐音。

3.  **内容违规**
    此原因涵盖图片和文字两方面的违规。
    *   **图片违规**：建议不要使用带有其他平台水印或二维码的图片。此类跨平台引流方式已不符合小红书当前的运营规则。
    *   **文字违规**：内容中不得出现违规词、敏感词、不文明用语或诱导性信息。我们的目标是通过合法营销获取合理收入，务必遵守平台规定与国家法律法规。

4.  **发布频率异常**
    短时间内发送文章频率过高、过快，可能导致账号被系统判定为异常。系统会认为这是机器人在滥发内容，从而不收录相关笔记。

5.  **设备被拉黑**
    若您使用的手机设备曾有过严重违规行为（如封号），即使更换了新账号，该设备也可能被平台记录并拉黑。这是因为APP会读取设备的硬件参数（如IMEI码、手机号等）。一旦设备被标记，后续在该设备上登录的任何账号都可能受到影响。

本节课中我们一起学习了导致小红书笔记不被收录的五类核心原因：违规推广、敏感词检测、内容违规、发布频率异常以及设备被拉黑。在发布笔记前，仔细检查这些方面，可以有效提高笔记被正常收录和推荐的概率。
