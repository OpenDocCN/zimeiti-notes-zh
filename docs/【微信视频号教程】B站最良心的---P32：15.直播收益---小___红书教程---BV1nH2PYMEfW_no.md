# 微信视频号运营教程：P32：15.直播收益 💰

在本节课中，我们将学习如何查看和理解微信视频号的直播收益数据。通过分析实时数据大屏，你可以清晰地掌握直播间的成交金额变化，这对于评估直播效果和优化运营策略至关重要。

---

上一节我们介绍了直播的基本设置，本节中我们来看看直播的核心成果之一：收益数据。

![](img/2e0ba3c0906676009bf794e4a4473ebd_1.png)

视频号的直播后台提供了一个实时的数据大屏。讲师在直播中向观众展示了这个界面。界面上的“累计成交金额”是核心指标，它显示了从直播开始到当前时刻的总收入。

**核心数据公式：**
`累计成交金额 = 直播期间所有订单的实收金额总和`

讲师现场演示了数据的实时性。他多次刷新页面，金额从“2959254”元变动为“2959750”元。这证明了数据是动态更新的，真实反映了直播期间产生的收益。

为了证明这是真人实时直播，而非录播或无人直播，讲师重新打开了摄像头，并与观众进行了实时互动，确认了当前日期和时间。
