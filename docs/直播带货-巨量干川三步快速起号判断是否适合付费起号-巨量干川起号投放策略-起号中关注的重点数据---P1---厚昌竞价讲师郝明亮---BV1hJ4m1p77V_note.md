# 直播带货：P1：巨量千川三步快速起号与投放策略 📊

在本节课中，我们将学习如何利用巨量千川进行直播带货的快速起号。课程将分为三个核心步骤：首先判断你的品类是否适合付费起号，然后制定具体的投放策略，最后通过数据分析进行优化。我们将深入探讨不同直播形式和赛道的特点，并提供清晰的策略框架与数据关注点，帮助初学者建立系统的付费起号认知。

## 什么是起号？🎯

上一节我们介绍了课程的整体框架，本节中我们来看看“起号”的核心概念。起号是指一个新的抖音号或直播间，通过一系列操作让系统识别并推荐给潜在用户的过程。

起号与不起号的区别在于，起号能让系统了解哪些人喜欢你的直播间，以及你的直播间能创造多少价值。这对应两个核心问题：
1.  **哪些人喜欢你的直播间**：这体现在用户点击进入、停留、互动、加粉丝团等行为上。
2.  **你的直播间能创造多少价值**：这体现在用户送礼、购买等付费行为上。

![](img/5ee751a87c3dfb83994b85f34fa5f2db_5.png)

![](img/5ee751a87c3dfb83994b85f34fa5f2db_7.png)

起号的本质是让更多人在你的直播间完成“进入-停留-互动-付费”这一系列动作。起号成功的标志是直播间能够稳定地拥有一批在线观众，无论人数是几十还是几百，并能产生稳定的成交。

![](img/5ee751a87c3dfb83994b85f34fa5f2db_9.png)

## 起号方法概览 🛠️

![](img/5ee751a87c3dfb83994b85f34fa5f2db_11.png)

![](img/5ee751a87c3dfb83994b85f34fa5f2db_13.png)

市面上流传着多种起号方法，例如“七天螺旋起号”、“清仓起号”、“鱼塘起号”等。无论哪种方法，其本质都是通过某种方式获取访客并让他们留下来甚至付费。

![](img/5ee751a87c3dfb83994b85f34fa5f2db_15.png)

![](img/5ee751a87c3dfb83994b85f34fa5f2db_17.png)

直播间的流量主要来源于以下几个渠道：
*   付费流量
*   自然推荐流量
*   粉丝关注流量
*   短视频流量
*   搜索流量

![](img/5ee751a87c3dfb83994b85f34fa5f2db_19.png)

**核心公式：起号成功 = 获取访客 + 促进停留/互动 + 引导付费**

![](img/5ee751a87c3dfb83994b85f34fa5f2db_21.png)

你可以选择将其中一种流量来源做到极致来起号。例如，专注于运营技巧和话术，把自然流量的停留互动做到极致；或者，专注于付费投放，通过巨量千川快速引入精准流量。本节课将重点讲解**付费起号**的策略。

## 巨量千川三步快速起号法 🚀

![](img/5ee751a87c3dfb83994b85f34fa5f2db_23.png)

![](img/5ee751a87c3dfb83994b85f34fa5f2db_25.png)

我们已经了解了起号的基本概念，接下来进入核心部分：巨量千川三步快速起号法。这三步是一个完整的闭环：**判断 -> 策略 -> 优化**。

![](img/5ee751a87c3dfb83994b85f34fa5f2db_27.png)

1.  **第一步：判断品类是否适合付费起号**。如果品类本身不适合，强行付费可能无法起号或导致亏损。
2.  **第二步：制定起号投放策略**。确定适合付费起号后，需要规划具体的投放方式和方法。
3.  **第三步：分析数据并优化**。投放后要密切关注数据，根据反馈调整优化后续动作。

遵循这三步，可以系统性地进行付费起号，减少试错成本。

### 第一步：判断是否适合付费起号

判断需要从两个维度交叉分析：**直播方式**和**赛道选择**。

#### 1. 直播方式

![](img/5ee751a87c3dfb83994b85f34fa5f2db_29.png)

当前主流的直播方式有两种：

*   **过款型直播**：直播间内有引流品、福利品、正价款等多种商品组合。通过憋单、秒杀、福袋等运营套路和话术渲染氛围，主要目的是**提升用户停留和互动**。这类直播间的**自然推荐流量是主要来源和成交来源**，短视频流量做补充，付费流量仅起辅助作用。
    *   **适用场景**：主播表现力强、运营技巧完善、团队配合好、商品利润率较低但受众较广的品类。

![](img/5ee751a87c3dfb83994b85f34fa5f2db_31.png)

*   **平播型直播**：通常采用绿幕背景，循环讲解固定话术，直播间氛围相对平淡，客单价较为平均。这类直播难以依靠内容吸引和留住用户，因此**付费流量是其主要的流量和成交来源**，需配合短视频和自然流量。
    *   **适用场景**：单品或垂直品类售卖、商品利润率高（建议毛利率50%以上）。

**结论**：平播型直播通常**必须依赖付费起号**；而过款型直播可以自然流量起号，付费仅作为辅助和加速手段，不能过度依赖。

#### 2. 赛道选择

不同商品类目的特性决定了其付费起号的适用性和策略。

以下是几个典型赛道的分析：

![](img/5ee751a87c3dfb83994b85f34fa5f2db_33.png)

![](img/5ee751a87c3dfb83994b85f34fa5f2db_35.png)

![](img/5ee751a87c3dfb83994b85f34fa5f2db_37.png)

*   **服饰鞋帽**：商品要求高，成本透明，退货率高，季节性强。赛道成熟，有大量成功案例。
    *   **投放建议**：建议以自然流量起号，付费投放目的为**协调自然流量**，用金钱换时间，快速帮助账号进入下一个流量池，但付费占比不宜过高。

![](img/5ee751a87c3dfb83994b85f34fa5f2db_39.png)

*   **个护家清**：需区分低利润品和高利润品。
    *   **低利润品**：以短视频流量为主，配合少量付费加速拓量。
    *   **高利润品**：以付费流量为主，优化投放，配合直播间循环话术进行平播。

*   **美容护肤**：用户需求量大且持续，复购率高。品牌与白牌两极分化。
    *   **投放建议**：付费流量是主力（可占70-80%），投放效果高度依赖视频素材质量。不过审是常见难点，达人种草类素材效果更佳。

*   **食品饮料**：需区分高客单价和低客单价。
    *   **高客单价**：通过产品组合（高SKU）突出价值，自然流量难有起色，**付费作为主力流量**。
    *   **低客单价**：以自然流量为主，难以支撑过高广告成本，付费主要起**协同作用**。

*   **3C电器**：同样需区分客单价。
    *   **高客单价**：多为品牌货，对付费依赖大，**付费是主力流量**。
    *   **低客单价**：多为功能性“黑科技”产品，需要**通过短视频流量种草，直播收割**，付费协同自然与视频流量。

#### 起号方法四象限

综合以上两个维度，我们可以得出一个清晰的“起号方法四象限”决策图：

```
| 直播方式 | 高利润 (毛利率>50%)               | 低利润 (毛利率<50%)               |
|----------|-----------------------------------|-----------------------------------|
| 平播型   | **象限A：付费为主/纯付费**        | **象限C：短视频+自然+付费共振**  |
|          | 例：高客单美妆、大家电、黑科技产品 | 例：个护家清、图书、低客单食品    |
| 过款型   | **象限B：自然为主，付费辅助**     | **象限D：自然流量为主**          |
|          | 例：高端女装、珠宝、翡翠          | 例：普通女装、零食                |
```

**使用方法**：确定你的直播间形式（平播/过款）和商品利润水平（高/低），找到对应的象限，即可明确你的核心流量来源和起号策略方向。

### 第二步：千川起号投放策略

根据第一步的判断，如果你的品类适合付费起号，并确定了付费流量的角色（主力或辅助），接下来就需要制定具体的投放策略。策略主要分为两大类。

#### 策略一：纯付费起号（付费作为主力流量）

![](img/5ee751a87c3dfb83994b85f34fa5f2db_41.png)

![](img/5ee751a87c3dfb83994b85f34fa5f2db_43.png)

此策略适用于**平播型+高利润**品类（象限A），付费流量目标占比可达60%-80%。

**核心策略路径如下：**

1.  **阶段一：精准流量打标签**
    *   **目标**：重点关注直播间停留、互动数据，保持稳定的在线人数（如20-50人），不追求极高在线。
    *   **投放方式**：
        *   `小店随心推` + `达人定向`（对标达人）或 `基础定向`（仅限年龄、性别、地区）。
        *   `极速推广` + `达人定向` 或 `基础定向`（预算最少300元）。
        *   `专业推广` + `徕卡定向`：行为兴趣只选“电商互动行为”，关键词只加核心产品词，兴趣标签加产品词或相关账号。
    *   **优化目标**：一半计划投**浅层目标**（进入直播间、商品点击），用于打标签和优化直播间互动；另一半计划投**深层目标**（成交），即使跑不动也要开着。
    *   **创意**：根据资源选择直投直播间或短视频引流，比例可参考7:3。

![](img/5ee751a87c3dfb83994b85f34fa5f2db_45.png)

2.  **阶段二：测试维度，寻找标杆**
    *   **时机**：当成交计划能稳定跑出，直播间有稳定在线和成交时。
    *   **目标**：重点关注成交和ROI数据，找出最优的投放组合。
    *   **投放方式**：重点测试`成交`和`ROI`优化目标。测试不同计划类型（极速控成本/放量、专业控成本/放量/ROI），找出跑量和ROI综合最优的作为主力。
    *   **预算分配**：99%预算给到成交和ROI目标。避免全部通投，应定期投放一些`徕卡定向`或`相似达人定向`的计划（即使ROI较低），为直播间引入新鲜血液，防止流量越跑越窄。

![](img/5ee751a87c3dfb83994b85f34fa5f2db_47.png)

![](img/5ee751a87c3dfb83994b85f34fa5f2db_49.png)

![](img/5ee751a87c3dfb83994b85f34fa5f2db_51.png)

3.  **阶段三：裂变组合，拓展放量**
    *   **目标**：根据测试出的最优维度（定向、创意、出价方式）进行裂变投放，降低测试成本，提高准确度。
    *   **直播间动作**：采用5分钟循环话术（留人、讲解、逼单、催单），配合付费计划节奏。

**关键提示**：对于纯付费直播间，如果直投效果不佳，**必须重点打造爆款视频**。视频素材的质量往往决定成败，需做好评论区维护。

#### 策略二：付费辅助起号（协调自然流量）

此策略适用于**过款型+高利润**（象限B）或**平播/过款型+低利润**（象限C/D）且希望自然流量占主导的品类。付费流量占比建议控制在20%以下，最高不超过30%，以免压制自然流量。

**核心策略路径如下：**

1.  **轻量投放，补流稳场**
    *   **目标**：以直播间自然流量为主，付费在开播极速流量过后进行补流，帮助场观进入第二、第三个高峰，或在流量下滑时快速补量，避免因在线人数下滑导致流量层级跌落。
    *   **投放方式**：优先使用`小店随心推`的`放量投放`，投**浅层目标**（进入直播间）。定向可用达人相似或基础年龄性别。预算建议100元/小时分段投放。可配合少量成交目标计划。
    *   **适用阶段**：起号初期或日常稳场。

2.  **深浅结合，突破层级**
    *   **时机**：当自然流量的场观、在线、GMV遇到瓶颈时。
    *   **目标**：通过付费流量提高场观和GPM，突破到更高流量层级。
    *   **投放方式**：使用`系统推荐`或`智能定向`，让自然流量模型带动付费流量。优先控成本，少量预算可尝试放量。创意根据测试决定。
    *   **优化目标**：90%深层目标（成交/ROI），10%浅层目标（补人气）。

![](img/5ee751a87c3dfb83994b85f34fa5f2db_53.png)

3.  **成交计划，提升销量**
    *   **目标**：付费作为销量的补充。
    *   **投放方式**：深度转化目标为主，严格控制付费流量占比（<20%）。定向仍以系统推荐为主，如需破圈可结合徕卡定向。

![](img/5ee751a87c3dfb83994b85f34fa5f2db_55.png)

![](img/5ee751a87c3dfb83994b85f34fa5f2db_57.png)

![](img/5ee751a87c3dfb83994b85f34fa5f2db_59.png)

![](img/5ee751a87c3dfb83994b85f34fa5f2db_61.png)

**直播间动作（针对过款型）**：
*   **开播**：用低价引流品A配合浅层付费目标，拉高初始人气。
*   **过程中**：切换高价利润款B，用深层付费目标（宽泛定向）促进成交。
*   **流量下滑时**：再次切换回引流品A，并配合浅层付费目标快速补人。
*   **核心**：做好AB款切换，利用付费加快精准人群进入，提升整体GPM，冲击更高流量池。

### 第三步：起号中关注的重点数据 📈

![](img/5ee751a87c3dfb83994b85f34fa5f2db_75.png)

![](img/5ee751a87c3dfb83994b85f34fa5f2db_77.png)

制定并执行了投放策略后，最后一步是通过数据监控优化效果。在起号阶段，应重点关注以下数据：

![](img/5ee751a87c3dfb83994b85f34fa5f2db_79.png)

1.  **场观（观看人次）**：直播间的总曝光进入人数，反映流量规模。
2.  **成交金额（GMV）**：最核心的结果指标。
3.  **千次观看成交金额（GPM）**：衡量流量转化效率的核心指标，公式为 `(成交金额 / 观看人次) * 1000`。
4.  **在线人数**：实时反映直播间热度。
5.  **人均观看时长**：反映内容吸引力和停留能力。
6.  **成交密度**：指单位时间内的成交订单数。高成交密度能有效拉动自然流量推荐。

![](img/5ee751a87c3dfb83994b85f34fa5f2db_81.png)

**起号期数据监控心法**：
*   **螺旋递增**：追求每一场直播的**场观**和**成交金额**都比上一场有所提升。这是整体数据增长的前提，其他如停留、互动等数据尽力而为即可。
*   **开源节流**：先解决“有流量”的问题（开源），再优化“流量成本和质量”的问题（节流）。如果深层目标投不出去，就从浅层目标开始，先让流量进来，再逐步优化和转化。
*   **当GMV增长缓慢时**，应回头优化直播间的**互动和停留**数据，获取更多自然流量曝光，从而带动销量。

## 总结 🎓

![](img/5ee751a87c3dfb83994b85f34fa5f2db_83.png)

![](img/5ee751a87c3dfb83994b85f34fa5f2db_85.png)

本节课中我们一起学习了巨量千川付费起号的完整体系。我们首先明确了起号的定义与目标，然后深入讲解了**三步快速起号法**：

1.  **判断**：利用“直播方式”与“赛道选择”两个维度，并通过“起号方法四象限”模型，科学判断你的品类是否适合付费起号，以及付费应扮演何种角色（主力或辅助）。
2.  **策略**：针对“纯付费主力”和“付费辅助协调”两种场景，分别给出了清晰的阶段性投放策略、投放方式选择（随心推/极速/专业）和优化目标搭配（浅层/深层）。
3.  **优化**：明确了起号期应核心关注**场观**和**成交金额**的螺旋式增长，遵循“开源节流”原则，并通过监控GPM、在线人数、成交密度等数据来持续优化直播间表现。

![](img/5ee751a87c3dfb83994b85f34fa5f2db_87.png)

记住，成功的起号是系统性的工程，需要将品类判断、投放策略和数据分析三者有机结合，才能实现快速、高效、低成本地启动直播间。
