抖音运营教程：P12：基础篇丨爆款文案套路

在本节课中，我们将学习一个适用于各行各业的爆款文案创作套路。这个套路能有效提升视频的停留、完播和互动数据，是内容创作的核心技能之一。

![](img/bbdb59179b1a6a4606449d469c4c9ba6_0.png)

在内容为王的时代，运营者必须具备脚本创作能力。而提升创作能力的第一步，是理解什么是好文案。好文案的核心只有一点：**要让观众认为内容与自己有关**。

如果内容无法引起观众的共鸣，观众自然不会停留观看。

那么，具体应该如何操作呢？下面介绍一个万能公式。

![](img/bbdb59179b1a6a4606449d469c4c9ba6_2.png)

以下是爆款文案的万能结构公式：

**现象 + 危害 + 原因 + 解决办法**

这个公式可以拆解为四个步骤：
1.  **现象**：描述一个普遍存在的问题或场景，筛选目标用户，让观众感觉“与我有关”，从而愿意停留。
2.  **危害**：阐述该问题带来的负面影响或痛苦，放大观众的焦虑或共鸣情绪，防止其在5秒内划走。
3.  **原因**：分析问题产生的原因，引导观众继续观看以寻求答案，从而提高视频的完播率。
4.  **解决办法**：给出具体的解决方案或建议，满足观众需求，并鼓励点赞、收藏等互动行为，提升互动率。

为了帮助你更好地理解，我们来看两个具体案例。

以下是服装行业的应用案例：

> “梨形身材的女孩太难买裤子了（**现象**）。挑错了样式显得腿又粗又短（**危害**）。但梨形身材是天生的，很难改变（**原因**）。而这三种裤子不仅显瘦，还显腿长，穿上第二天就被小哥哥搭讪了，姐妹们赶快收藏起来吧（**解决办法**+**互动引导**）。”

这段文案完美契合了公式。开头锁定目标客户，随后逐步引导观众看完视频。基于这种信任感，后续的粉丝变现也会更容易。

以下是运营行业的应用案例：

> “每天用心制作短视频，但发完只有100播放（**现象**）。继续下去只是浪费时间，徒增焦虑（**危害**）。这跟你没有系统的运营底层逻辑有关（**原因**）。现在去看完我的前三条视频，不花一分钱也能学会运营知识，还不快快行动起来（**解决办法**+**行动号召**）。”

只要用好这个套路，套用到任何行业都行之有效。这类文案通常不足30秒，拍摄制作起来也很轻松，尤其适合进行矩阵式账号运营，能极大提升内容生产效率。

![](img/bbdb59179b1a6a4606449d469c4c9ba6_4.png)

以上我们已经举了两个例子进行说明。至于举一反三，就需要你自己来实践了。

你可以根据自己所在行业的特性，运用这个公式进行创作练习，并将你的思考过程分享出来。

![](img/bbdb59179b1a6a4606449d469c4c9ba6_6.png)

本节课我们一起学习了爆款文案的万能公式：**现象 + 危害 + 原因 + 解决办法**。掌握这个结构，能帮助你快速创作出吸引目标用户、提升视频关键数据的高质量文案。
