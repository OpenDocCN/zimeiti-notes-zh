## 平台对原创内容的支持

上一节我们介绍了发布作品的基础，本节中我们来看看平台对原创内容的扶持政策。平台本身支持原创，并设有创作者激励计划，为原创内容提供流量倾斜。

以下是创作者激励计划的具体内容：
*   **流量扶持**：平台计划让1000万原创作者获得流量支持。
*   **收入激励**：让百万优质创作者获得收入。
*   **零粉阶段**：平台会赠送2500到5万点的流量券，用于推广作品。
*   **千粉阶段**：可加入重点作者扶持计划，优质视频可申报专项奖金。
*   **万粉及百万粉阶段**：可申报的专项奖金更多。

因此，原创内容能获得平台一级一级的流量池推进。当然，原创需要个人具备拍摄、编写台本等技术能力。

## 伪原创的完整操作流程

![](img/0714517e6039c82d4be5360e372be0cb_6.png)

当然，搬运并非简单下载后直接发布。平台支持原创，直接搬运带水印的作品可能导致账号违规。因此，我们需要进行“伪原创”处理。

以下是伪原创的完整操作流程，我们将一步步进行说明。

### 第一步：选品与定位

首先，你需要确定账号的定位和要售卖的商品。确定好要带的货之后，再去找相关的视频内容。

![](img/0714517e6039c82d4be5360e372be0cb_8.png)

在视频号App内，你可以通过以下路径进入选品中心：
1.  点击右上角的个人头像。
2.  进入个人页面后，找到并点击“创作者中心”。
3.  在创作者中心内，可以浏览和选择商品。

你也可以去其他电商平台（如阿里巴巴1688、拼多多等）寻找灵感。例如：
*   如果你想做音乐类账号，可以考虑售卖车载U盘或承接音乐广告。
*   如果你想做影视剧类账号，可以考虑承接广告或通过内容吸引粉丝。

**公式：账号定位 = 目标商品 + 内容类型**

### 第二步：寻找灵感与素材

确定定位后，需要寻找创作灵感和素材来源。你可以利用一些数据平台。

以下是寻找灵感的途径：
*   **视频号排行榜**：在数据平台（如有望数据）上查看视频号各类目（如影视、音乐）的排行榜，分析优质账号的内容和优点。
*   **跨平台寻找**：例如，在抖音发现一个优质的宠物类账号。**关键步骤**：在决定搬运前，务必先在视频号搜索，确认该账号的内容是否已被大量搬运过，避免内容重复。

### 第三步：下载与去水印

![](img/0714517e6039c82d4be5360e372be0cb_10.png)

![](img/0714517e6039c82d4be5360e372be0cb_11.png)

找到合适的素材账号并确认可搬运后，不要直接下载发布。第一步是去除原视频的水印。

你可以使用各类去水印工具或小程序，将原视频链接输入，获取无水印的视频文件。

**代码逻辑示意（伪代码）**：
```
输入： 带水印的视频链接
过程： 调用去水印工具接口
输出： 无水印的视频文件
```

### 第四步：二次剪辑与创作

获得无水印素材后，需要进行二次剪辑，使其成为你的“新”作品。

以下是二次剪辑的常见手法：
*   **剪辑重组**：调整视频顺序，拆分或合并片段。
*   **改变格式**：修改视频尺寸、速度（加速/慢放）、添加滤镜或调色。
*   **叠加元素**：添加新的背景音乐、字幕、贴纸、画中画或你自己的语音解说。
*   **重构内容**：对视频内容进行深度解读、添加点评或混剪多个素材。

经过这些步骤，原素材就变成了一个具有你个人风格的“伪原创”作品，可以大大降低被平台识别为重复搬运的风险。
