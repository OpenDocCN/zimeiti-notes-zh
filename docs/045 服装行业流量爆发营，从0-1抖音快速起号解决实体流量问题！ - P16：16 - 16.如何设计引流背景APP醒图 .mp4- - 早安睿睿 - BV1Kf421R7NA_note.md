在本节课中，我们将学习如何使用“醒图”这款手机应用，从零开始设计一张精美的短视频头图。即使你是设计新手，也能通过本教程掌握核心技巧，解决实体店的流量问题。

上一节我们介绍了引流的重要性，本节中我们来看看如何通过视觉设计来吸引用户。

## 核心设计思路

设计头图前，需要明确三个核心要点。

**1. 突出主题**
头图的核心是传递价值。你需要先想清楚要突出什么信息。例如，你的价值主张可能是“关注我，解决小个子穿搭难题”。设计时必须确保这句话清晰醒目，避免使用过于花哨的背景或难以辨认的字体，导致主题被弱化。

![](img/1b641c3347727d3f420ff63b71fa8fdb_2.png)

**2. 遵守尺寸要求**
上传头图时，平台有严格的尺寸规定。务必按照官方要求的尺寸进行设计，以确保图片显示正常。

![](img/1b641c3347727d3f420ff63b71fa8fdb_4.png)

![](img/1b641c3347727d3f420ff63b71fa8fdb_6.png)

**3. 使用正确比例**
头图有特定的比例要求，不应使用1:1的正方形比例。具体比例设定，我们将在后续由设计师详细演示。

![](img/1b641c3347727d3f420ff63b71fa8fdb_8.png)

## 使用醒图APP设计头图

![](img/1b641c3347727d3f420ff63b71fa8fdb_16.png)

首先，在应用商店下载并打开免费的“醒图”APP。我们将采用一个进阶方法开始设计。

![](img/1b641c3347727d3f420ff63b71fa8fdb_18.png)

![](img/1b641c3347727d3f420ff63b71fa8fdb_19.png)

![](img/1b641c3347727d3f420ff63b71fa8fdb_21.png)

**第一步：创建画布**
1.  点击左上角的彩色圆环图标（“添加画布”）。
2.  选择一个喜欢的颜色作为背景，例如红色。
3.  在比例选项中，选择最右侧的 **16:9**。
4.  点击创建，即可得到一个空白画布。

![](img/1b641c3347727d3f420ff63b71fa8fdb_23.png)

![](img/1b641c3347727d3f420ff63b71fa8fdb_25.png)

**第二步：添加与编辑文字**
在画布上，我们可以开始添加核心文案。
1.  点击底部工具栏的 **“文字”** 按钮。
2.  在输入框内双击，即可开始打字，例如输入“公主，帮你越来越美”。
3.  点击确定，文字便添加到画布上。
4.  你可以使用APP提供的各种字体和样式模板来美化文字，并调整其大小和位置。

![](img/1b641c3347727d3f420ff63b71fa8fdb_27.png)

![](img/1b641c3347727d3f420ff63b71fa8fdb_29.png)

以下是添加文字的基础操作示意：
```plaintext
[点击“文字”按钮] -> [输入文案] -> [选择样式/调整大小位置]
```

![](img/1b641c3347727d3f420ff63b71fa8fdb_31.png)

**第三步：添加与抠图（进阶）**
在文字基础上，可以添加图片使头图更生动。
1.  点击底部工具栏最右侧的 **“导入图片”**。
2.  从相册选择一张人物照片导入。
3.  将图片拖动到合适的位置。
4.  若想让人物从背景中分离，可以使用“智能抠图”功能：
    *   点击选中已导入的图片。
    *   点击底部出现的 **“抠图”** 按钮。
    *   选择 **“智能抠图”**，APP会自动识别人物轮廓。
    *   点击确定，即可得到抠出的人物图像，将其放置在文字周围。

![](img/1b641c3347727d3f420ff63b71fa8fdb_33.png)

![](img/1b641c3347727d3f420ff63b71fa8fdb_35.png)

![](img/1b641c3347727d3f420ff63b71fa8fdb_37.png)

**第四步：设计引流款头图**
我们还可以设计更具吸引力和暗示性的头图来引导用户。
1.  删除或修改原有文字。
2.  重新添加文字，例如输入“帮粉丝”。
3.  点击输入法中的表情符号，添加一个礼物或爱心表情（例如英文“free”可能关联出的礼物图标）。
4.  这暗示了“为粉丝提供福利”，你可以结合自己的业务自由发挥文案。
5.  再次为文字更换一个醒目的字体，完成设计。

![](img/1b641c3347727d3f420ff63b71fa8fdb_39.png)

本节课中我们一起学习了使用醒图APP设计短视频头图的完整流程。从明确“突出主题”的设计思路，到创建16:9画布、添加编辑文字，再到导入图片并使用智能抠图进行合成，最后还掌握了设计引流款头图的小技巧。记住，清晰传达价值是头图设计的首要目标。
