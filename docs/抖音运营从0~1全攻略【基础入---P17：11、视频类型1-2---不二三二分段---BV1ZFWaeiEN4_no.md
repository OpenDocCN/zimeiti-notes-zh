# 抖音运营从0~1全攻略：P17：11、视频类型1-2 - 人设与内容构建 🎬

在本节课中，我们将要学习抖音运营中一个至关重要的中间环节——**人设的建立**，并探讨如何围绕人设来构建视频内容。我们将区分两种主要的人设类型，并详细介绍一种基础且高效的内容形式：口播。

上一节我们解决了文案问题，本节中我们来看看内容创作的下一个关键步骤。许多教程会直接谈论涨粉与变现，但在这两者之间，还有一个不可或缺的环节：**建立你的人设**。人设的目的并非为了炫耀，而是为了在众多同类创作者中让你脱颖而出。例如，有十个人都做装修内容，如果其中一位创作者（比如“小王同学”）树立了一个幽默搞笑的人设，观众就更容易记住他，而忽略其他九位。当观众因为喜欢你这个人而关注你时，涨粉和后续的变现便水到渠成。

## 人设的两种类型：强与弱

首先，我们来区分两种核心人设类型：**强人设**与**弱人设**。理解这一区别后，你就能快速判断任何视频所采用的人设策略。

**强人设**的核心是**以创作者个人为切入点**。内容与创作者自身密切相关，通常涉及分享个人的行业经验、专业知识、生活日常或兴趣爱好。这类内容大多需要**真人出镜**，因为核心是展示“你”这个人。

以下是强人设常见的切入点领域：

*   **行业经验**：装修、办公、房产、汽车、餐饮、婚介、法律咨询、电商等。
*   **兴趣爱好**：美食、旅行、手工、宠物、星座占卜、美妆护肤、服装穿搭、游戏、摄影等。

**弱人设**则与创作者本人**关系不大**，核心目的是纯粹地把账号内容做好。创作者本人并非内容的焦点或记忆点。

以下是弱人设的典型例子：

*   **情感语录账号**：仅发布文字配音乐的视频，无真人出镜或配音。
*   **影视解说账号**：虽然有人声解说，但焦点是电影内容，观众关注的是解说而非解说者本人。

理解强弱人设的区别，是选择你后续内容方向的基础。

## 内容构建：从人设到视频

明确了人设，我们就可以开始构建视频内容。一个核心原则是：**只要内容与你（强人设）或你的账号定位（弱人设）相关，都可以作为视频素材**。下面，我们首先介绍一种非常适合强人设、且操作简单的视频形式：**口播**。

### 口播：用讲述构建内容

口播，简而言之，就是**对着镜头讲话**。它是一种通过语言直接传递信息、故事或情感的内容形式。

口播主要分为两大类，服务于不同目的：

1.  **讲故事型口播**：分享客户故事、从业经历或人生感悟。故事本身不要求完全真实，可以从文案平台寻找有趣素材。目的是让观众觉得**有意思、有共鸣**。
2.  **教知识型口播**：分享育儿、金融、养生、装修、房产、汽车等领域的专业知识。目的是让观众在几十秒内获得**收获感**，觉得学到了东西。

以下是关于口播的几个常见问题解答：

*   **Q：口才不好能做口播吗？**
    *   **A：可以。** 口播是**提前准备稿子**再录制，并非临场发挥。大部分口播者的流畅表达都源于充分准备。
*   **Q：普通话不好能做口播吗？**
    *   **A：可以，有时甚至更有优势。** 如果你的目标受众是同城或有地域属性，使用**方言**会显得更亲切、有趣。同时，可以通过添加字幕确保所有观众都能理解内容。
*   **Q：口播如何拍摄？**
    *   **A：非常简单。** 基本模式是：**找好文案 -> 准备稿子 -> 用手机支架固定手机 -> 对着镜头录制**。这是一种后期很容易起步和操作的内容形式。

本节课中我们一起学习了抖音运营中承上启下的关键——**人设的建立**。我们明确了**强人设**（以个人为中心）与**弱人设**（以内容为中心）的区别，并深入探讨了适合强人设起步的**口播**这种内容形式。记住，清晰的人设是吸引精准粉丝的第一步，而口播则是将你的人设和知识传递给观众的最直接工具之一。
