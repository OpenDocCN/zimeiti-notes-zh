# 新媒体运营：P29：11：小红书平台用户标签及流量入口 📊

![](img/7001f5d721e655dce26f44e324a49841_1.png)

在本节课中，我们将学习小红书平台如何为用户打上标签，以及平台主要的流量分发入口。理解这些机制，有助于我们分析笔记如何被系统推荐给目标用户，从而优化运营策略。

上一节我们介绍了系统如何判定内容标签，本节中我们来看看平台如何判定用户标签。这个过程能帮助我们理解笔记是如何被推荐给用户的，并可以从底层逻辑反推，如何让笔记被精准用户看到。

## 用户标签判定机制

系统为用户打标签的主要依据是用户的**浏览行为**。用户注册时选择的兴趣范围参考价值不大，仅作为初期内容方向的引导，无法进入系统深层的用户标签池。

系统根据以下用户行为来判定其兴趣分类：

以下是判定用户标签的核心行为指标：

1.  **点击率**
    *   **定义**：系统在发现页向用户推送多种内容（如美妆、家居、母婴等），用户点击某一类内容的频率。
    *   **作用**：系统通过统计用户点击不同类别内容的概率，初步判断其兴趣。例如，用户点击10条内容中，美妆类3次，家居类2次，母婴类0次，系统会判定该用户对母婴内容不感兴趣，并持续推送更多美妆内容进行试探。

2.  **停留时长**
    *   **定义**：用户点开一篇笔记后观看的持续时间。
    *   **作用**：即使同属美妆领域，若用户快速划走某些笔记，说明内容不吸引他。系统会继续推荐该领域下其他细分内容（如上妆、护肤）。如果用户对偶然推荐的家居类笔记停留很久，系统可能判断其有装修或购买家居用品的潜在需求。

3.  **页面浏览量（PV）**
    *   **定义**：用户浏览页面的深度，例如不仅看了单篇笔记，还进入了博主主页并查看了多篇笔记。
    *   **作用**：PV值能反映用户对账号内容的兴趣深度。提升用户在你主页的浏览深度，对账号健康度和评分有积极帮助。

4.  **互动行为**
    *   **定义**：包括点赞、收藏、评论、关注等用户操作。
    *   **作用**：这些数据直接提升笔记的互动率。数据越高，笔记进入更大推荐流量池的几率越高，同时也能带动账号内其他笔记的曝光，并有助于粉丝增长和后续变现。

## 平台流量分发入口

了解用户标签后，我们来看小红书主要的流量来自哪里。平台主要有四个流量分发入口。

以下是小红书平台的四个核心流量入口：

*   **发现页**
    *   **内容构成**：
        1.  已关注达人发布的内容。
        2.  根据用户标签（点击率、停留时长等）匹配的推荐内容。
        3.  部分平台为丰富内容多样性而推荐的发散性内容。
    *   **重要性**：这是笔记和账号获取流量的最主要来源。

*   **关注页**
    *   **内容构成**：纯粹展示用户所关注博主发布的最新笔记。
    *   **特点**：流量精准但相对有限，取决于粉丝的主动查看习惯。

*   **附近页**
    *   **内容构成**：根据手机定位，推送同城或附近地区的相关内容。
    *   **适用内容**：具有强地域特征的需求，如美食、本地旅游、线下店推广（美容院、景点等）。做线下实体店推广需重点关注此入口。

*   **购物页**
    *   **内容构成**：小红书电商生态的入口，直接匹配购物需求。
    *   **现状**：部分用户尚未形成在此直接购物的强习惯，更多人仍倾向于先搜索相关种草笔记。此入口是平台商业变现的重要方向。
