# 微信视频号运营教程：P24：7. 账号定位与素材获取 🎯

在本节课中，我们将要学习微信视频号运营中两个关键环节：账号定位的后续处理，以及如何解决原创视频素材短缺的问题。我们将一步步拆解，确保新手能够清晰掌握。

上一节我们介绍了账号定位的重要性，本节中我们来看看如何为已经定位的账号进行“清洁”并获取启动素材。

## 账号初始化：隐藏旧作品

有同学询问是否需要隐藏账号之前的作品。答案是肯定的。在确定新的账号定位后，隐藏与当前定位不符的旧作品，有助于平台和观众更清晰地认识你的账号内容方向，这对账号的长期发展是必要的。

![](img/f21140b7c31b57385e332eef9b16895d_1.png)

操作很简单，进入视频号后台管理页面即可设置。素材准备无需着急，保持日更一条视频的节奏即可。

## 解决原创视频素材问题 🎬

![](img/f21140b7c31b57385e332eef9b16895d_5.png)

![](img/f21140b7c31b57385e332eef9b16895d_6.png)

接下来解决原创视频素材的问题。很多同学面临“不会拍摄、不会剪辑、没有专业素材”的困境。以下是针对这一普遍情况的解决方案。

![](img/f21140b7c31b57385e332eef9b16895d_7.png)

为了帮助初学者快速起步，本课程提供一项福利：每位学员都可以领取30个已剪辑好的视频素材。

这些素材来自一个投入重金打造的独家素材库，涵盖了多种类型，例如搞笑娱乐、生活妙招等。每个视频都经过专业剪辑师精心制作，直接可用。

以下是素材示例的简要描述：
*   一段节奏轻快、带有搞笑元素的短视频。
*   视频包含人物互动与音乐卡点，符合当下流行趋势。

领取这些素材后，你可以直接使用，无需担心版权或重复问题，因为每个领取链接都是唯一的。

## 如何领取素材？

![](img/f21140b7c31b57385e332eef9b16895d_9.png)

领取素材需要满足几个简单的条件。以下是具体步骤和要求：

1.  **课堂互动**：在直播公屏上打出指定关键词（如“想”字）。
2.  **积极听课**：保证课程听课时长，并与老师积极互动。
3.  **联系助理**：课程结束后，联系向你发送听课链接的助理老师。
4.  **表明身份**：向助理老师说明是“张良老师课程学员”，以便顺利领取福利。

请注意礼貌沟通。直接、生硬地索要素材可能无法获得帮助。提及课程老师名号是确认学员身份的有效方式。
