# 抖音运营教程：P11：扶持加权与降权限流

在本节课中，我们将学习抖音平台对视频内容的扶持与限制规则。理解哪些内容会获得流量扶持，哪些行为会导致降权限流，是成功运营账号的基础。

上一节我们介绍了账号权重的分类，本节中我们来看看平台具体的扶持与限流标准。

![](img/b32ed670de2a3db7a93043884946d9cb_1.png)

## 🎬 获得扶持加权的内容

平台会优先推荐符合以下标准的视频内容：

![](img/b32ed670de2a3db7a93043884946d9cb_3.png)

*   **视频格式**：竖版（9:16）的移动端原生视频。
*   **画面质量**：画面平稳，作品清晰。
*   **声音质量**：无过多杂音或噪音。

![](img/b32ed670de2a3db7a93043884946d9cb_1.png)

## 🚫 导致降权限流的内容

以下是会被平台降权限流的内容类型，发布时需特别注意。

![](img/b32ed670de2a3db7a93043884946d9cb_3.png)

以下是具体的不允许行为列表：

1.  **违规信息**：内容涉及吸烟、暴力、金钱诱导等。
2.  **外部引流**：视频中出现微信、微博、淘宝等外部平台联系方式。
3.  **商业标识**：长时间出现商品Logo、手机号。二维码出现时间**最长不要超过3秒**，但审核严格时，即使出现1秒也可能被驳回。
4.  **硬性广告**：进行明显的营销广告、推销或引流卖货行为。

![](img/b32ed670de2a3db7a93043884946d9cb_5.png)

## ⚠️ 特别注意：好物推荐类内容

对于希望通过分享好物来带货的账号，操作不当极易导致商品链接（小黄车）被下架。以下是两个关键点：

*   **避免直接标价**：在视频画面中直接标注价格（如“九块九”），属于明显的销售行为，商品链接通常无法通过审核。
*   **必须配有口播**：单纯展示产品、拆箱或使用，而没有口头介绍，会被判定为低质营销内容。

**正确的做法是**：在展示产品时，配合简单的口播介绍。例如：
> “我儿子玩这个玩具，四个小时都没停下来。”

通过口播将视频内容转化为真实的体验分享，而非单纯的广告，这样更容易通过审核。
