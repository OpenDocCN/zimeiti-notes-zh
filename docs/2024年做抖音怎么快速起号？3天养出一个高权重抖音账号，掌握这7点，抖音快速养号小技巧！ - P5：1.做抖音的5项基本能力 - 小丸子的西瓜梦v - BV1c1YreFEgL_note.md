# 抖音运营入门：P5：1. 做抖音的5项基本能力 📚

在本节课中，我们将要学习成为一名抖音创作者所必备的五项基本能力。掌握这些核心技能，是制作出优质内容、成功运营账号的基础。

## 概述

首先，我们来了解做抖音创作必备的五项基本能力。

## 五项基本能力详解

### 1. 拍摄剪辑能力 🎬

最基本的技能是如何用手机拍出画面流畅、表达清晰的视频素材。这个能力是需要具备的。其次，是手机剪映或者电脑版剪映的使用，也就是剪辑的能力。如何剪出让用户能看下去，并且觉得有价值的作品，也是需要经过不断练习才可以的。

### 2. 内容策划能力 💡

内容策划其实就是你的作品构思。比如，你想要通过什么样类型的内容来传递你想要表达的信息。或者，你每一个镜头以及在视频中说的每一句话，它的作用是什么，如何进行串联起来，形成一个优质的作品。这些内容是需要不断花时间揣摩和练习，才能找到感觉的。

### 3. 镜头表现力 🎭

如果是真人出镜，如何在镜头前轻松地表达自己，而且还要把自己想要表达出来的情绪以及感觉演绎到位。这个也是需要我们个人不断去练习的。

### 4. 细分行业的知识储备 📖

如果你提供的是知识类的内容，如何让用户关注你，就需要你有相对比较专业的细分行业知识，才能获得粉丝的芳心。

### 5. 流量变现技能 💰

![](img/ea0250f723ae725dbb20eaf84298b140_1.png)

不管你是通过什么方式来变现的，当系统给你的作品推送大量流量的时候，你能不能将流量快速变现。这个需要你提前就规划好变现的路径。

如果你本身就有成熟的商业模式或赚钱方法，承接抖音流量会比较容易。如果你是通过短视频或者直播带货的方式，那么还要熟悉直播电商的整体流程，比如选品能力以及卖点提炼、直播能力和话术技巧等。如果你是通过星图广告，那么你需要努力做好个人人设或者个人IP。当然，如果你仅仅是靠流量来获取平台的广告分成，那么就需要有不断创作爆款视频的能力，以及获取平台更多流量的能力。

## 运营前的五项准备

上一节我们介绍了五项核心能力，本节中我们来看看，在正式开始前，我们还需要准备哪些方面。

以下是做好抖音需要准备的五个方面：

1.  **拍摄剪辑设备**：拍摄设备相对比较简单一点。一般我们自己使用的手机，只要支持 `1080P 30帧` 就可以满足需求。不过最好可以用苹果12以上的，或者是国产旗舰版或者次旗舰版型号的手机。当然，如果你的内容方向对画面质量要求较高，建议你还是用相机拍摄。剪辑设备，手机当然是可以的，不过如果对电脑操作熟练，用电脑剪辑视频效率会更高效一点。
2.  **有充足的时间去做选题**：做抖音短视频并不是一朝一夕的事情。我们看到别人很多爆款视频，你首先要确定自己有没有足够多的时间去发掘选题、拍摄剪辑等。
3.  **平台规则的熟悉**：平台的规则是我们每个人都需要遵守的，都需要在策划的时候就需要规避掉。因此做之前需要把平台规则好好看一遍。
4.  **看过足够多不同内容类型的作品**：在抖音中创作出优秀的内容，是需要作者具备网感的。基本的网络热梗、金句、热门视频、热门BGM是需要我们花时间去熟悉、理解和整理收集的。而且这些内容也在实时更新，需要我们不断地去看，才能跟上创作的步伐。
5.  **熟练内容创作的流程**：科学的创作流程，能够大大提升我们起号以及做出爆款视频的几率。因此，如何更专业地创作短视频，需要我们不断地去学习。

## 需要关注的五个核心方面

最后我们还要认识到，其实做抖音并没有那么复杂。我们创作的作品发布到平台上，在互联网中我们就是一个博主而已。这个过程并没有想象的，或者是有些人说的那么繁琐。唯一有难度的就是，需要我们持续地创作出优质的内容作品。

因此，我们关注以下五个核心方面即可：

1.  **热门视频、BGM和对标账号**：这些内容我们只要花时间去刷、去找，就可以找到很多、了解很多。慢慢地对抖音创作就会有一个正确而且细致的认知。说白了就是要见得足够多。
2.  **内容价值和垂直度**：内容价值通俗点讲，就是让刷到你的用户觉得你的内容要么有意思，要么有知识，要么有感觉，要么没有见过、没体验过，要么就是自己内心想的被你说出来了，产生了情感共鸣。当然，除了内容价值之外，账号是不能乱发作品的。也就是如果你发有用的知识，就一直发这类的；如果你发别人有感觉的，就一直发这类的。
3.  **画面、文案和内容呈现形式**：在抖音中是靠推荐页的竖版视频吸引粉丝的。那么视频中的元素无非就是画面、声音和文字。所不同的是，这些元素经过不同的组合，形成了不同的内容形式，传递出来的信息和感觉就不一样了。所以，我们不管是平时刷视频还是创作视频，所谓的内容质量就是这些元素细节在不断优化。
4.  **账号人设和个人IP塑造**：我们要变现，做人设是比较容易的方法。人设也就是你在这个账号中，人物角色的设定是什么。经过长时间的沉淀积累起粉丝粘性之后，也就是你的粉丝信任你，这样就形成了个人的IP。IP形成之后，不管是通过广告还是带货来变现，都是比较容易的事情了。
5.  **完播率、互动率和粉丝画像**：我们在运营账号的时候，需要关注的数据就是作品的完播率。也就是多少人刷到你的作品，有多少人看完。这个数据比较重要，会被反馈到抖音系统中。如果数据比较好，平台给你的流量就会更多。当然还有互动率，也就是点赞、评论、转发、收藏、关注等用户行为。另外，我们还要去查看观看我们视频的用户是什么样的人，是不是我们想要吸引的人。

## 总结

本节课中，我们一起学习了成为一名抖音创作者必备的五项基本能力，了解了运营前需要做的五项准备，并明确了在账号运营中需要持续关注的五个核心方面。这些是构建成功抖音账号的基石。
