*   **客户行业**：你的客户从事什么行业？例如，传统行业（工厂、餐饮、线下服装）的客户与互联网行业的客户，其活跃平台和信息获取习惯截然不同。你需要分析目标行业人群最常使用的软件和平台。
*   **用户基础**：你的用户中包含新手（小白）吗？如果包含，你的内容和产品门槛不能过高，否则会吓退他们。如果针对的是有基础的进阶用户，内容则需要更加专业和有深度。
*   **客户年龄段**：不同年龄段的用户聚集在不同的平台。例如，哔哩哔哩（B站）以年轻二次元用户为主；小红书女性用户占比较高，且用户偏年轻化；抖音和微信视频号则覆盖了更广泛的年龄层，尤其是30-50岁的用户。选择与客户年龄段匹配的平台至关重要。
*   **客户性别**：男性与女性用户的兴趣点和活跃平台有显著差异。例如，男性用户可能更关注科技、数码、机械、汽车等内容；女性用户则可能对情感、美妆、穿搭、生活分享等内容更感兴趣。这直接影响你内容创作的方向和渠道选择。

## 第二步：确定产品力 💪

**核心概念**：**信息茧房** 与 **时间成本**
现代用户大多通过“被动接收”的方式获取信息（如平台推送），这容易形成“信息茧房”——用户只看到系统推荐或他人展示的信息。同时，用户的“时间成本”很高，他们很少会主动、花时间去横向对比所有信息。因此，只要你的产品在用户首次接触时，能展现出足够的吸引力和价值感，就可能促成交易。你必须突出并放大产品的价值。

*   **短视频**：以抖音为核心，可同步至小红书、西瓜视频等。适合快节奏、强吸引力的内容。
*   **长视频**：以哔哩哔哩（B站）为核心。适合知识干货、系统教学、深度评测等时长较长的内容。
*   **图文/文字**：适合小红书、今日头条、百家号、搜狐号、微信公众号、知乎等平台。知乎尤其适合专业问答和深度文章。
*   **传统电商/商务信息**：对于机械、实体商品等，可能需要考虑阿里巴巴（1688）、百度爱采购、中国供应商等传统B2B平台。

**核心代码**（平台选择逻辑）：
```plaintext
if 用户群体 == “年轻学生”:
    重点渠道 = [“哔哩哔哩”， “小红书”， “抖音”]
elif 用户群体 == “中年男性”:
    重点渠道 = [“抖音”， “微信视频号”， “今日头条”]
elif 用户群体 == “年轻女性”:
    重点渠道 = [“小红书”， “抖音”]
# ... 以此类推
```
