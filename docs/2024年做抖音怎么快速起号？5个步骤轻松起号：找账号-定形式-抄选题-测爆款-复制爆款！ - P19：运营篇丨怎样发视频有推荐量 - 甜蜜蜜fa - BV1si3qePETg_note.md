# 抖音运营：P19：如何创作高推荐量视频

在本节课中，我们将学习一个将抽象内容价值转化为具体短视频语言的实用公式。这个公式能帮助你系统性地优化视频结构，从而提升视频的推荐量。

![](img/bec563fb3e3723c904ac7cecfae93b4f_1.png)

上一节我们讨论了优质内容的重要性，但“优质”一词过于抽象。本节中，我们来看看如何通过一个具体的公式，将“好内容”拆解为可执行的创作步骤。

这个公式被称为 **“爆炒虎头、什锦猪肚、红烧豹尾”**。它源自一句商业格言：客户购买的并非产品本身，而是产品带来的感觉。该公式将视频分为三个关键部分进行设计。

## 爆炒虎头：打造精彩的开头

“爆炒虎头”意指视频开头的五秒必须像虎头一样精彩夺目，迅速抓住观众注意力。

以下是两个具体的应用方法：

*   **展现具体表现，而非直接陈述**：避免平铺直叙的介绍。例如，不要说“给大家推荐一部高分末日电影”，而可以这样开头：“那一天，上海感受到了被吞噬的恐惧。夜空中出现了一张血盆大口，钢筋森林都被它吸进嘴里。”
*   **点明观众痛点，而非单纯分享**：避免自我视角的分享。例如，不要说“我给大家分享三个上海的景点”，而可以这样开头：“有人说来上海旅游没什么好玩的，那可能是你的打开方式不太对。作为10年的老沪漂，我想送你今天这份上海旅行攻略。”

## 什锦猪肚：构建饱满的内容主体

![](img/bec563fb3e3723c904ac7cecfae93b4f_3.png)

“什锦猪肚”是指视频的主体内容要像猪肚一样饱满丰富，节奏紧凑，信息量足。

内容需要画面切换流畅，并符合情绪递进的逻辑。例如，在分析市场动态时，可以这样衔接：“然而，过了一天，股市又普遍回血。那问题就来了，接下来这场仗，对全球经济到底会有什么样的影响呢？” 核心要求是内容充实，杜绝任何节奏拖沓或华而不实的信息。

![](img/bec563fb3e3723c904ac7cecfae93b4f_5.png)

## 红烧豹尾：设计有力的视频结尾

“红烧豹尾”是指结尾要像豹尾一样简短有力且意味深长，能够升华主题或促进互动。

以下是两种有效的结尾方式：

![](img/bec563fb3e3723c904ac7cecfae93b4f_7.png)

*   **使用反问句，提升互动数据**：例如，在美食视频结尾可以问：“屏幕前的你看完会不会食欲大增呢？反正我能多干两碗米饭。”
*   **升华主题，从小事件上升到大格局**：例如，在讲述个人经历后总结：“吞下了委屈，喂大了格局。改变不了别人，就改变自己。”

理解了“爆炒虎头、什锦猪肚、红烧豹尾”这个公式后，请重新审视你的内容：开头是否缺乏细节？内容是否节奏拖沓，让人看不下去？结尾是否只是简单的“我们下期再见”？这通常是投放“抖加”效果不佳的核心原因。问题不在于平台，而在于内容本身尚未达到被广泛推荐的标准。

本节课中，我们一起学习了“爆炒虎头、什锦猪肚、红烧豹尾”这一内容创作公式。通过打造精彩开头、充实内容主体、设计有力结尾，你可以系统性地提升视频质量，从而获得更好的平台推荐量。
