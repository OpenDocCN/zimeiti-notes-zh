## 笔记被推荐的核心逻辑

笔记发布后，平台会根据一系列数据指标决定是否将其推荐给更多用户。这个过程可以理解为“流量升级”。例如，一个新账号的笔记初始可能只有约1000次展示（曝光），如何让它升级到2000-5000次甚至更高层级的展示量？这取决于以下两个核心指标：

### 1. 打开率 (Click-Through Rate)
打开率是决定笔记能否进入下一级推荐池的首要指标。

当用户打开小红书App，首页推荐流会一次性展示约4-6篇笔记。平台会一次性测试约20篇同类型笔记。你的笔记就在这20篇中进行初始测试。

**核心公式：**
`打开率 = 笔记点击量 / 笔记展示量`

平台要求笔记的**平均打开率需达到11%以上**，才能获得进入下一级流量池的资格。
*   举例：如果平台给你的初始展示量是1000次，那么你需要获得至少110次点击（小眼睛👀）。
*   如果达到11%，平台可能会将展示量翻倍至2000-5000次。
*   如果低于11%，笔记的推荐通常就会停止，只有初始的一波流量。

**结论：** 封面图和标题的吸引力至关重要，它们直接决定了打开率。

### 2. 互动率 (Engagement Rate)
当笔记通过打开率测试进入更大流量池后，互动率就成为下一次升级的关键。

![](img/f98fb15ce683c117f625f41543d6a76d_9.png)

![](img/f98fb15ce683c117f625f41543d6a76d_11.png)

**核心公式：**
`互动率 = (点赞+收藏+评论+分享) / 笔记点击量`

![](img/f98fb15ce683c117f625f41543d6a76d_13.png)

![](img/f98fb15ce683c117f625f41543d6a76d_15.png)

平台要求笔记的**平均互动率需达到3%以上**，才能继续向更高层级的流量池推荐。
*   举例：假设你的笔记有110个点击，那么至少需要约4个互动（评论、点赞等）。
*   互动率越高，笔记获得的后续推荐量就越大。

![](img/f98fb15ce683c117f625f41543d6a76d_17.png)

**流量升级路径总结：**
1.  **第一级：** 考核**打开率**（≥11%），通过则进入更大流量池。
2.  **第二级：** 考核**互动率**（≥3%），通过则可能进入万级甚至十万级展示池。
3.  对于新账号，通常能升级2-3次；对于高权重老账号，起点更高，升级潜力更大。
4.  更高层级的流量（如百万级）通常需要结合内容质量和商业推广。

![](img/f98fb15ce683c117f625f41543d6a76d_19.png)

除了打开率和互动率，用户在你笔记页内的所有行为都有权重评分，例如：
*   **点赞**
*   **收藏**
*   **评论**
*   **分享**
*   **关注博主**

这些行为都能为笔记增加“热度权重”，但前提是**打开率和互动率这两个基础指标必须首先达标**。

## 必须避免的运营“坑”🚫

了解推荐逻辑后，我们还需要规避一些常见的违规操作，以免账号被降权或限流。以下是新手必须注意的要点：

![](img/f98fb15ce683c117f625f41543d6a76d_21.png)

![](img/f98fb15ce683c117f625f41543d6a76d_23.png)

*   **新号不要立刻发笔记：** 新注册的账号需要“养号”，即先模拟正常用户浏览、互动几天，让平台识别为真实用户。立即发笔记会消耗初始权重，导致后续流量不佳。
*   **违规必被降权：** 任何违规行为（如导流、抄袭、发布违禁内容）都会导致笔记推荐减少、搜索排名下降，甚至账号被限流。一次严重违规可能导致流量下降90%并持续一段时间，恢复难度极大。
*   **标签切忌反复修改：** 账号标签、简介等确定后，不要频繁改动。尤其是在账号已有稳定流量后修改核心标签，极易引发限流。
*   **一机一号一卡：** 尽量避免一台手机登录多个小红书账号，尤其是不问领域的账号。最好做到“一机、一卡、一号”，用手机流量操作更安全。电脑端登录也建议不超过3个账号。
*   **谨慎处理推广与引流：** 不要在笔记或评论区直接留联系方式、二维码或明显导流到其他平台的信息。私信引流也需非常谨慎，容易被系统判定为营销行为。
*   **谨慎处理留言私信：** 对于用户的留言和私信，要积极合规互动，避免产生投诉或纠纷。
