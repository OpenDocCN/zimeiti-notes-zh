## 如何搜索与分析潜在爆款
以下是使用爆款参谋进行搜索和分析的基本步骤。

1.  输入你计划销售的商品关键词（例如“服装”、“百货”）。
2.  点击搜索按钮。
3.  工具会展示一系列相关商品及其数据。

点击搜索后，我们可以直接透视其他卖家的数据。例如，可以看到某些商品的新增播放量数据：2.9万、5.9万、20.1万、17.9万。

## 区分“爆款”与“大爆款”
理解播放量和实际销量的区别后，我们来看看如何区分普通爆款和真正的大爆款。

![](img/56723394dfe915173656f9f8b62e7886_10.png)

我们点开一个商品查看其销售数据。如图所示，该商品在6天时间内卖出了56.3万单。

![](img/56723394dfe915173656f9f8b62e7886_12.png)

**计算公式：**
`日均销量 = 总销量 / 销售天数`
`日均销量 = 56.3万单 / 6天 ≈ 9.4万单/天`

这意味着平均每天能卖出约9万单。即使每单只赚1元钱，日利润也能达到9万元。这已经是一个不错的爆款。

但我们的目标是寻找“大爆款”。接下来，我们继续向下寻找。

![](img/56723394dfe915173656f9f8b62e7886_9.png)

![](img/56723394dfe915173656f9f8b62e7886_10.png)

![](img/56723394dfe915173656f9f8b62e7886_14.png)

我们发现另一款商品，其新增播放量为9.7万。点击查看其销售详情。

![](img/56723394dfe915173656f9f8b62e7886_12.png)

数据显示，该商品在6天时间内卖出了398万单。

**计算公式：**
`日均销量 = 398万单 / 6天 ≈ 66.3万单/天`

这意味着日均销量超过66万单。这才是真正的“大爆款”，其盈利潜力远大于普通爆款。

## 利用数据透视把握时机
找到大爆款后，关键在于把握跟进的时机。爆款参谋还能帮助我们分析商品的上架时间。

![](img/56723394dfe915173656f9f8b62e7886_16.png)

我们查看该大爆款商品的发布时间。如图所示，发布日期是2024年8月2日晚上9点46分。而当前时间是2024年8月9日晚上9点21分。

![](img/56723394dfe915173656f9f8b62e7886_14.png)

这意味着我们不仅透视了它的销量（398万单），还知道了它的上架时间。如果我们能在8月4日，即原商品发布后不久迅速跟进发布同类内容，就能极大可能抓住这波流量红利。
