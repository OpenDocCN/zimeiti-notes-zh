# AI短视频创作教程：P11：8.影视推广平台 🎬

在本节课中，我们将要学习如何将AI生成的视频发布到抖音、快手等平台，并通过“影视推广”任务来结算收益。我们将详细介绍影视推广的概念、操作流程以及各平台的收益计算规则。

上一节我们介绍了如何将视频分发到多个自媒体平台获取播放量收益。本节中我们来看看，如果只想专注于抖音、快手等熟悉平台，如何同样实现变现。

很多同学问到，视频如果发到抖音或快手，该如何结算收益。需要明确的是，将视频发到除抖音、快手外的30多个自媒体平台，可以依靠视频播放量直接结算收益。

![](img/469cff0965163d25d140ae7edf4ff7fa_1.png)

![](img/469cff0965163d25d140ae7edf4ff7fa_5.png)

但是，抖音和快手这两个平台，视频发上去主要作用是增长粉丝，通常不能直接靠播放量获得平台收益。

---

![](img/469cff0965163d25d140ae7edf4ff7fa_7.png)

那么，有没有方法可以在抖音、快手上既涨粉，又能通过播放量结算收益呢？答案是肯定的。

以下是通过“影视推广”平台实现这一目标的方法。

![](img/469cff0965163d25d140ae7edf4ff7fa_5.png)

*   **可发布平台**：视频可以发布在抖音、快手、小红书以及微信视频号这四个平台。
*   **核心途径**：通过“影视推广”平台接取任务，完成任务后结算奖金。

影视推广是指电影、电视剧、综艺或短剧制作方，为了宣传作品，委托创作者制作并发布推广视频。创作者根据视频带来的播放量或互动数据获得奖金。

---

以下是影视推广任务的具体操作流程。

1.  **选择任务**：在影视推广平台上有大量任务，例如爱奇艺、搜狐等平台提供的剧集推广任务。任务列表会明确显示剧集名称和奖励规则。
    ![](img/469cff0965163d25d140ae7edf4ff7fa_7.png)
2.  **制作与发布视频**：领取任务后，使用提供的影视素材制作短视频，并发布到指定的平台（如抖音、小红书等）。
3.  **提交任务回填**：发布后，需要在平台提交“任务回填链接”。需要填写以下信息：
    *   个人微信号、账号昵称、发布平台的ID。
    *   所推广的剧集名称。
    *   所发布视频的作品链接。
    *   视频发布时间。
    *   视频播放量或点赞量的截图。
4.  **结算收益**：平台审核回填信息无误后，会根据任务规则结算并发放奖金。

影视推广的收益通常比单纯依靠平台播放量分成的收益更高。

---

了解了操作流程后，我们来看看不同平台的收益计算规则。影视推广的收益计算主要分为两类：按播放量结算和按点赞量结算。

**按播放量结算（适用于抖音、视频号、小红书）**：
*   **抖音**：每1万播放量，奖金为 **50元**。
*   **微信视频号**：每1万播放量，奖金为 **10元**。（因用户基数大，单价较低）
*   **小红书**：每1000播放量，奖金为 **30元**。（单价最高）

**公式示例（抖音）**：`收益 = (播放量 / 10000) * 50元`

**按点赞量结算（适用于抖音+快手组合任务）**：
此类任务无粉丝数量要求，按单作品的点赞数计算。
*   **抖音**：5000赞奖励100元，1万赞奖励200元。
*   **快手**：1万赞奖励180元。

**公式示例**：`收益 = 达成点赞档位对应的固定奖金`

此外，部分短剧推广任务（如搜狐短剧）的奖金单价也相对较高。

本节课中我们一起学习了影视推广的变现模式。总结来说，通过接入影视推广平台，创作者可以在抖音、快手等平台发布视频，不仅能够吸引粉丝，还能通过完成具体的推广任务，根据视频的播放量或点赞量获得可观的奖金收益。这是一种对账号粉丝量要求低、收益明确的短视频变现方法。
