# 商业分析技能课：P2-3-6：电子支付市场生态与玩家竞争力分析 🏦

![](img/f5040b708cdd877f2989a1b609feacf4_0.png)

在本节课中，我们将学习电子支付市场的当前生态格局，并分析不同市场参与者的核心竞争力。课程将从行业发展历程、体系架构、生态圈构成、盈利模式、驱动因素以及未来路径等多个维度进行拆解。

## 行业发展历程 📜

上一节我们介绍了课程的整体框架，本节中我们来看看电子支付行业在我国的具体发展路径。我国金融科技的发展始于1997年，最初由O2O和电子支付驱动。

在萌芽阶段，电子支付主要依托网上银行。传统金融机构将原有金融产品及服务互联网化。例如，招商银行率先推出“一网通”。随后，行业逐渐向无卡支付演进，并最终发展到当前的聚合支付模式，各大银行开始推动以二维码为主的聚合支付。

2003年是一个关键节点，支付宝上线。支付宝借鉴了国外的PayPal模式，但在移动端用户体验上更具优势。目前，PayPal在美国市场正逐渐被Square取代，后者凭借更佳的用户体验、更低的手续费率以及与电商的深度绑定而胜出。

在我国，支付方式的选择高度依赖于其绑定的场景。目前最大的两家支付公司——支付宝和微信支付，就分别对接了电商和社交两大核心场景。

2013年余额宝的上线引爆了电子支付行业。虽然第一批第三方支付牌照在2011年就已发放，但直到余额宝出现，行业才迎来爆发式增长。同年，腾讯推出微信支付。

2013年至2016年进入爆发期，主要的金融科技公司引领了大量业务创新，创造了虚拟信用卡、电子钱包、移动支付等新模式，这些均围绕第三方支付展开。

2016年后，金融科技进入科技高度引领的阶段。人工智能、大数据、生物识别、云计算等技术全面渗透金融行业，不仅影响了电子支付，也重塑了整个金融业的风控、清算与结算系统，带来了效率的极大提升和新的盈利空间。同时，用户信息与数据安全也成为需要关注的问题。

电子支付作为金融科技中起步最早的模块，不仅是当前创新的基础，也为未来的模式创新与技术进步提供了底层架构。支付掌握着货币的流通节奏、流量与方向，是理解后续大数据征信、智能投顾等领域的前提。

## 体系架构与生态圈 🏗️

了解了发展历程后，我们进一步剖析电子支付的整体体系架构。我国电子支付体系现已领先全球，其架构可分为四大模块：支付工具、支付系统、支付服务组织和支付体系监管。

**支付工具**包括实物载体（如POS机、NFC手机）和虚拟载体（如支付宝、微信支付等APP）。POS机市场竞争激烈，因为它能带来稳定的交易分润和宝贵的消费数据。

**支付系统**分为面向机构的金融专用支付系统和面向大众的现代零售支付系统，我们日常使用的属于后者。

**支付服务组织**包括中国人民银行（监管机构）、中国银联（清算机构）、商业银行以及第三方支付机构等，共同为支付流程提供服务。

**支付体系监管**框架则由法律法规、央行监管文件及自律协会等构成。

目前，市场研究的焦点主要集中在第三方支付机构和商业银行。尽管银联试图抢占市场份额，但由于技术和产品丰富度不及阿里、腾讯，且难以切入具体场景，其市场份额相对较小。

接下来，我们看看电子支付的生态圈全貌。

以下是电子支付生态圈的主要层级与参与者：

*   **顶层：监督管理机构**。包括中国人民银行、银保监会、支付清算协会等。
*   **核心：支付流程**。连接用户与商户的现金流。
    *   **线下支付**：通过银行卡收单、扫码支付、预付卡等方式完成。
    *   **线上支付**：通过互联网支付和移动支付，提供跨行支付及金融衍生服务。
*   **中介与服务方**：
    *   **央行与银联**：提供支付系统与跨行资金清算。
    *   **商业银行**：提供银行卡、支票等支付工具。
    *   **第三方支付机构**：提供支付解决方案，可能涉及POS收单、软硬件提供。
    *   **通信运营商与手机厂商**：提供网络与设备支持。

在众多参与者中，网络支付（尤其是互联网支付和移动支付）是当前的主流，市场份额大且用户体验好。对于初学者而言，无需了解所有参与者，应重点关注顶级玩家，即**支付宝（阿里系）** 和**财付通/微信支付（腾讯系）**。

例如，Apple Pay虽经大力推广，但因未能深度绑定高频场景（如支付宝的电商、微信的社交），且手机市场份额下降，在中国市场发展有限。支付宝和微信支付则通过积分体系、社交红包等功能建立了极高的用户粘性。

## 盈利模式与驱动因素 💰

在认识了生态圈的主要玩家后，我们来分析他们的盈利模式。以下是产业链中不同参与者的主要收入来源：

*   **银联（清算机构）**：主要收取清算费，例如在银行卡收单业务中分成约10%。
*   **商业银行（发卡行）**：作为发卡方收取手续费，同时从电子银行转账等业务中收费。分成比例通常在70%到20%之间浮动。
*   **第三方支付机构**：
    *   向电商平台或商户收取交易佣金。
    *   获取用户沉淀资金的利息收入。
    *   收取接入费、交易手续费（约20%）、技术服务费等。

我国支付行业的盈利结构经历过改革。2016年“96费改”前，普遍采用“721”固定分成模式（发卡行70%，收单机构20%，清算机构10%）。“96费改”后，定价更精细化，区分了消费与商户类型，并设置了金额上限，整体费率规模下降，市场竞争也愈加激烈。各银行会结合自身特色提供差异化费率优惠。

接下来，我们探讨驱动电子支付行业发展的三个核心因素：

1.  **人口基数与市场渗透**：国内人口红利基本消失，支付公司转向提高现有用户渗透率，并通过并购整合、出海国际化（如进军东南亚、东欧市场）寻求增长。
2.  **互联网产业发展**：互联网与移动端的普及是电子支付发展的载体。没有成熟的线上商业形态，电子支付难以独立壮大。
3.  **监管政策**：金融是国之命脉，全球各国对支付业务都实行牌照管理，并对备付金有严格要求。中国是目前备付金要求最高的市场之一，美国则相对市场化。

## 发展路径与未来展望 🚀

分析了现状与驱动因素后，本节我们展望电子支付的未来发展方向。其根本发展路径在于获取用户的逻辑不同：

*   **“账户到用户”模式**：传统玩家（如商业银行、纯支付机构）的路径。先让用户建立金融账户，再围绕账户运营（如提升资产规模AUM、销售产品）拓展用户。这是产品导向的思维。
    *   `AUM (Asset Under Management) = 账户管理的总资产余额`
*   **“用户到账户”模式**：新兴玩家的路径。先通过高频场景（如电商、社交、生活服务）获取高粘性用户，再引导其绑定金融账户。这是用户和场景导向的思维。
    *   例如：支付宝源于淘宝的电商担保需求；美团支付源于其生活服务场景，并借此衍生出“美团小贷”。

我更看好“用户到账户”的模式，因为它基于用户体验和真实场景，更符合互联网发展趋势。而**综合型玩家**（同时拥有用户和账户）实力最强，如网商银行和微众银行。它们持有银行牌照，支付平台完善，目前聚焦小微金融，未来试点期结束后，有望快速进入零售市场，推广支付、理财、借贷等全面服务。

最后，介绍一个前沿创新方向：**数字货币与电子支付的结合**。

例如，创业公司**Mononicco**发行以数字货币结算的借记卡和贷记卡。用户存入资金后，系统通过区块链网络全球询价，以最优汇率兑换数字货币进行支付，剩余资产同时享受存款利息和数字货币的升值潜力。

这与持有传统外汇资产不同，数字货币支付能在全球范围内动态选择最有利的汇率。据测算，每笔约200美元的交易可节省10美元。这种模式打破了传统跨境支付的监管壁垒，未来可能重塑全球支付体系。

本节课中，我们一起学习了电子支付市场的完整图景。我们回顾了行业从银行触网到移动聚合支付的发展历程；剖析了包含工具、系统、组织与监管的四大体系架构；描绘了以用户、商户、中介与监管为核心的多层生态圈；分析了银联、银行与第三方支付的不同盈利模式；总结了人口、互联网产业与监管三大驱动因素；并探讨了“账户到用户”与“用户到账户”两种核心发展路径，以及数字货币支付这一未来创新方向。理解这些，是进一步分析金融科技领域其他业务的基础。
