# AI新媒体运营：P15：01.10.03：训练AI从分析到产出爆款笔记 📝

## 概述
在本节课中，我们将学习如何训练AI（以ChatGPT为例）分析爆款笔记，并最终生成具备爆款潜质的新笔记。我们将通过具体的步骤，让AI理解平台风格、内容结构和爆款要素，从而辅助我们高效地产出优质内容。

![](img/f72e72f5f6e183377c47230efde010ed_1.png)

上一节我们介绍了通过“喂资料”的方式，让AI学习特定平台的用语和风格。虽然上一节以旅游攻略为例的成果可能不尽理想，但方法是正确的。关键在于，如果你能向AI提供你所在赛道的优质内容，AI就能通过分析和总结，学习到该平台文案的写作方式和风格。

![](img/f72e72f5f6e183377c47230efde010ed_5.png)

本节中，我们来看看如何深化这个过程，即让AI分析具体的爆款内容，并基于分析结果来生成新的笔记。

## 第一步：明确分析指令与要点
要让AI有效分析，我们需要给出清晰、具体的指令。核心是让AI拆解爆款内容的成功要素。

![](img/f72e72f5f6e183377c47230efde010ed_9.png)

![](img/f72e72f5f6e183377c47230efde010ed_11.png)

![](img/f72e72f5f6e183377c47230efde010ed_13.png)

以下是需要AI重点分析的几个方面：
1.  **爆款因素**：分析内容为何能成为爆款，例如是否引发了情感共鸣、提供了实用价值等。
2.  **整体结构**：分析文案的叙事逻辑。
    *   **开篇**：用什么方式（如提问、制造焦虑、分享故事）吸引用户继续阅读。
    *   **中间**：提供了哪些内容来戳中用户痛点，满足用户需求，从而提升用户停留时长。
    *   **结尾**：设置了什么“钩子”（如鼓励互动、提供联系方式）来引导用户私信、关注或评论。
3.  **关键词策略**：分析笔记中是否刻意植入了特定关键词。
    *   如果有，关键词是什么？
    *   关键词的出现频率和分布情况如何？
    *   是如何结合中文分词法来设计关键词的？

![](img/f72e72f5f6e183377c47230efde010ed_15.png)

![](img/f72e72f5f6e183377c47230efde010ed_17.png)

## 第二步：选择并输入爆款内容
我们需要为AI提供真实、高质量的爆款笔记作为分析样本。本例中，我们选择“考研”领域的几篇高点赞笔记。

![](img/f72e72f5f6e183377c47230efde010ed_19.png)

![](img/f72e72f5f6e183377c47230efde010ed_21.png)

操作流程如下：
1.  在小红书等平台找到目标领域的爆款笔记（例如点赞量高的笔记）。
2.  将笔记的文案内容完整复制下来。
3.  在ChatGPT中新建一个对话，输入我们准备好的分析指令，并将复制的文案内容粘贴给AI。

![](img/f72e72f5f6e183377c47230efde010ed_23.png)

**示例指令代码**：
```
你是一个小红书爆款笔记分析师。我将提供一篇小红书上的考研相关爆款笔记文案，请你分析：
1. 它的爆款因素有哪些？
2. 它的整体结构：开篇如何吸引用户？中间如何戳中痛点？结尾如何引导互动？
3. 它是否设计了关键词？如果有，是什么关键词？出现频率如何？
```
随后，将找到的爆款文案粘贴在指令下方。

## 第三步：解析AI的分析结果
AI会对我们提供的每篇笔记进行分析并给出反馈。我们需要审阅这些反馈，了解AI识别出的爆款模式。

![](img/f72e72f5f6e183377c47230efde010ed_33.png)

![](img/f72e72f5f6e183377c47230efde010ed_35.png)

![](img/f72e72f5f6e183377c47230efde010ed_36.png)

![](img/f72e72f5f6e183377c47230efde010ed_38.png)

![](img/f72e72f5f6e183377c47230efde010ed_40.png)

例如，AI可能总结出以下常见爆款因素：
*   **情感共鸣**：开篇描述焦虑、担忧等普遍情绪，让读者产生“这就是我”的感觉。
*   **逆袭故事**：分享个人从低谷到成功的经历，增加可信度和激励感。
*   **实用干货**：提供明确的备考计划、时间节点、具体学习方法和资源推荐。
*   **结构清晰**：分点论述，逻辑层次分明，方便读者获取信息。
*   **亲和力表达**：使用表情符号（emoji）和口语化语言，拉近与读者的距离。
*   **正能量鼓励**：结尾给予读者信心和鼓励，营造积极氛围。
*   **引导互动**：明确鼓励读者提问、私信或关注。

![](img/f72e72f5f6e183377c47230efde010ed_44.png)

## 第四步：整合爆款要素并生成大纲
在向AI投喂多篇爆款笔记并获取分析后，我们需要让AI对这些分析进行整合与去重，提炼出一套通用的爆款要素清单。

![](img/f72e72f5f6e183377c47230efde010ed_54.png)

![](img/f72e72f5f6e183377c47230efde010ed_56.png)

**操作指令**：
```
请将你之前分析过的所有关于考研的爆款笔记因素总结一下，整合成一个完整的爆款要素列表。
```

![](img/f72e72f5f6e183377c47230efde010ed_58.png)

获得要素列表后，即可指令AI根据这些要素，为我们规划一篇新笔记的大纲。

**操作指令**：
```
现在，请结合你总结的爆款笔记因素，为我规划一篇关于考研的小红书笔记大纲。大纲需要结构完整，并尽可能融入你总结的爆款要素。每个大纲节点下，请再细分出几个要点。
```

![](img/f72e72f5f6e183377c47230efde010ed_60.png)

AI生成的大纲可能包含：吸引人的标题、共鸣式开篇、逆袭故事引入、分科目备考攻略、时间管理建议、心态调整方法、结尾互动引导等部分。

![](img/f72e72f5f6e183377c47230efde010ed_62.png)

## 第五步：根据大纲生成完整文案
大纲确认无误后，我们就可以让AI根据大纲和爆款要素，生成完整的笔记文案了。此时，指令需要更加具体。

**操作指令**：
```
请根据你刚才提供的大纲，撰写一篇完整的小红书考研笔记文案。
要求：
1. 结构严格遵循爆款笔记的格式：开篇引发共鸣，中间提供干货，结尾引导互动。
2. 在开篇部分，融入一个简单的“逆袭”小故事来引入情绪。
3. 文案中要自然地使用emoji表情。
4. 尝试在文案中融入“考研”、“备考规划”等关键词。
5. 语言风格亲切、有感染力，就像一位成功的学长学姐在分享经验。
```

![](img/f72e72f5f6e183377c47230efde010ed_64.png)

AI会根据指令生成一篇包含故事、干货、鼓励和互动引导的完整笔记。这篇文案已经具备了爆款笔记的基本框架和要素，可以作为我们内容创作的优质初稿。

![](img/f72e72f5f6e183377c47230efde010ed_66.png)

## 总结
本节课我们一起学习了利用AI辅助产出爆款笔记的完整流程。我们首先教会AI如何通过分析现有爆款来识别成功要素，包括**结构分析**（开篇、中段、结尾）和**策略分析**（情感共鸣、干货价值、关键词等）。然后，我们引导AI将这些要素整合，并**生成具备爆款潜质的内容大纲和完整文案**。

![](img/f72e72f5f6e183377c47230efde010ed_68.png)

![](img/f72e72f5f6e183377c47230efde010ed_70.png)

这个过程极大地提升了内容创作的效率和质量，尤其适合需要持续产出垂直领域内容的运营者。记住，AI是我们的强大助手，最终的优化和调整仍需结合你的人性化判断和平台的具体规则。通过不断“训练”和“调教”，你将能打造出更懂你需求的专属内容创作助手。
