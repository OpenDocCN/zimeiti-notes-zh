### 1. 关键词匹配

![](img/e55248c3ce77ddbb7dea2aea1d323f2f_12.png)

在撰写小红书笔记的标题、正文，或制作图片与视频时，内容是否清晰、准确地匹配了你账号的垂直定位，并体现了相关的关键词，这一点非常重要。

![](img/e55248c3ce77ddbb7dea2aea1d323f2f_14.png)

![](img/e55248c3ce77ddbb7dea2aea1d323f2f_16.png)

**核心概念**：笔记内容需与账号定位及目标关键词高度相关。
**公式**：`流量相关性 ≈ 关键词匹配度 × 内容垂直度`

### 2. 账号权重（活跃度）

![](img/e55248c3ce77ddbb7dea2aea1d323f2f_24.png)

虽然有些人认为账号不存在“权重”这一说法，但实质上，权重指的是你账号的活跃度。请记住：任何一个平台都喜欢活跃的用户。

![](img/e55248c3ce77ddbb7dea2aea1d323f2f_26.png)

![](img/e55248c3ce77ddbb7dea2aea1d323f2f_28.png)

因此，保持有规律的内容更新与发布，积极参与平台互动，有利于提升账号的整体表现效果。

![](img/e55248c3ce77ddbb7dea2aea1d323f2f_30.png)

**核心概念**：通过规律性活跃行为提升账号在平台系统中的“权重”。
**代码描述**：`账号权重 = f(发布频率, 互动行为, 内容质量)`

### 3. 付费投放（薯条）

![](img/e55248c3ce77ddbb7dea2aea1d323f2f_38.png)

小红书为笔记提供了一个名为“薯条”的付费推广渠道。通过付费，可以在一定程度上提升单篇笔记的流量、点赞、评论等数据效果。

![](img/e55248c3ce77ddbb7dea2aea1d323f2f_40.png)

![](img/e55248c3ce77ddbb7dea2aea1d323f2f_42.png)

以下是使用“薯条”功能的具体操作步骤：
1.  打开小红书应用。
2.  进入你想要推广的笔记浏览界面。
3.  点击右上角的“...”三个点图标。
4.  在弹出的菜单底部，即可找到“薯条”推广功能。

![](img/e55248c3ce77ddbb7dea2aea1d323f2f_44.png)
