# 拼多多运营：P28：拼多多店铺子账号设置教程 🛠️

在本节课中，我们将学习如何在拼多多店铺后台设置和管理子账号。子账号功能对于团队分工协作至关重要，它能帮助您为不同职责的员工分配相应的后台操作权限。

## 什么是子账号？ 👥

子账号是由主账号创建的，用于对店铺进行分权管理的后台账户。例如，您可以为运营、客服等不同岗位的员工创建独立的子账号，让他们分管店铺的不同板块。运营可以查看店铺数据，客服则专注于处理客户咨询。

## 子账号的角色与权限 🔐

上一节我们介绍了子账号的概念，本节中我们来看看子账号的具体角色分类及其对应的权限。在拼多多的角色管理中，主要包含以下几类：

以下是主要的子账号角色及其权限说明：

*   **管理员**：权限仅次于主账号，可以操作店铺内绝大部分功能。但无法进行某些核心操作，例如：
    *   修改商品主体信息。
    *   执行店铺资金出入账操作。
    *   公式表示为：`管理员权限 ≈ 总权限 - {核心财务权限， 核心主体修改权限}`

*   **运营**：拥有商品上新、活动提报、订单管理以及客服咨询等权限。

*   **客服**：可以查看客服工作状态、设置离线消息、接待客户咨询。

*   **物流客服**：权限最低，仅能查看发货管理模块，处理与物流相关的问题。

## 如何新增子账号 ➕

了解了角色权限后，接下来我们学习创建子账号的具体步骤。请注意，拼多多平台规定，一个手机号码只能绑定一个子账号，且该手机号不能已在任何拼多多店铺注册过。

以下是新增子账号的关键步骤和注意事项：

1.  **登录后台**：使用店铺主账号登录拼多多商家后台。
2.  **进入管理页面**：在后台左下角找到 **“店铺管理”**，点击其中的 **“子账号管理”**。
3.  **填写账号信息**：点击 **“新增子账号”**，进入创建页面。
    *   **手机号**：输入一个未在拼多多注册过的手机号码。
    *   **账号名**：**这是需要牢记的登录凭证**。您需要手动设置一个账户名称（例如“运营小李”），登录时使用此账号名，而非系统生成的一串数字。
    *   代码示例：登录账号应为 `运营小李`，而非类似 `PDD155XXXX` 的数字ID。
4.  **短信验证**：创建过程需要使用主账号或管理员绑定的手机号接收并输入短信验证码。

## 如何管理与删除子账号 ⚙️

创建子账号后，您可能需要根据人员变动或职责调整来修改其权限或删除账号。

以下是管理与删除子账号的操作方法：

*   **修改权限/角色**：只有主账号或管理员可以执行此操作。在“子账号管理”页面，找到对应账号，点击 **“编辑权限”**。您可以在角色列表中选择新角色，或在权限列表中勾选/取消具体权限项。
*   **删除子账号**：同样由主账号或管理员操作。在“账号管理”列表中找到目标子账号，直接点击 **“删除”** 即可。

## 后台操作演示 🖥️

现在，让我们进入后台进行实际操作。请跟随以下步骤：

1.  打开拼多多商家后台。
2.  在左侧导航栏找到并点击 **“店铺管理”**。
3.  在展开的菜单中点击 **“子账号管理”**，系统可能会要求进行短信验证。
4.  验证通过后，您将看到 **“角色管理”** 和 **“账号管理”** 两个主要板块。
5.  在“账号管理”中，您可以查看所有子账号的账户名、备注、角色及绑定状态，右侧提供“编辑”和“删除”按钮。
6.  如需调整角色权限模板，可进入 **“角色管理”** 进行设置。
