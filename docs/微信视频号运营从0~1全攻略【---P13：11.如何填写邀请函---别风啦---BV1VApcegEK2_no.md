# 微信视频号运营从0~1全攻略：P13：如何填写邀请函 📝

在本节课中，我们将学习如何正确填写视频号创作者计划的邀请函。这是加入计划、开启运营的关键一步，我们将一步步拆解每个填写项，确保信息准确无误。

![](img/f0dab7e36a0ba74ff1e01bbe5d741a43_1.png)

上一节我们介绍了如何注册视频号，本节中我们来看看如何完成后续的邀请函填写。

![](img/f0dab7e36a0ba74ff1e01bbe5d741a43_3.png)

![](img/f0dab7e36a0ba74ff1e01bbe5d741a43_4.png)

## 第一步：找到你的视频号账号信息

首先，你需要找到自己视频号的昵称和粉丝数量。以下是具体操作步骤。

以下是找到账号信息的详细步骤：

![](img/f0dab7e36a0ba74ff1e01bbe5d741a43_6.png)

![](img/f0dab7e36a0ba74ff1e01bbe5d741a43_7.png)

1.  **打开微信**：在手机上启动微信应用。
2.  **进入“发现”页**：点击微信底部的“发现”选项卡。
3.  **找到“视频号”入口**：在“发现”页面中，向上滑动，找到并点击“视频号”入口（图标为黄色蝴蝶形状）。
4.  **进入个人主页**：点击视频号页面右上角的**人头像图标**，进入个人中心。
5.  **查看账号信息**：在个人中心页面下滑，找到“我的视频号”区域。这里会显示你的**账号昵称**和**粉丝数量**。

例如，你的信息可能显示为：`天道酬勤，39917个粉丝`。那么，账号昵称就是“天道酬勤”，粉丝量是“17”。

## 第二步：填写邀请函各项内容

找到账号信息后，就可以开始填写邀请函了。邀请函通常包含以下几个核心部分。

以下是需要填写的项目及说明：

![](img/f0dab7e36a0ba74ff1e01bbe5d741a43_9.png)

![](img/f0dab7e36a0ba74ff1e01bbe5d741a43_10.png)

![](img/f0dab7e36a0ba74ff1e01bbe5d741a43_11.png)

*   **视频号账号昵称**：填写你在上一步中查看到的昵称，例如 `天道酬勤`。
*   **视频号粉丝量**：填写你查看到的粉丝数量，例如 `17`。
*   **活动邀请人**：根据你的加入渠道填写。例如，如果通过“张良老师”的渠道加入，则填写 `张良` 或 `张良老师`。
*   **实名信息**：此处必须填写你本人的**真实姓名**，例如 `周建国`。
*   **结算手机号**：填写你用于接收验证码和后续结算的**手机号码**，例如 `13812345678`。注意号码中间不要有空格。

填写完所有信息后，仔细检查一遍。确认无误后，点击页面上的 **“提交”** 按钮。系统可能会弹出“确定要提交吗？”的确认框，点击 **“确认”** 即可完成提交。

成功提交后，页面通常会显示“成功加入创作者分成计划”之类的提示。

![](img/f0dab7e36a0ba74ff1e01bbe5d741a43_13.png)

![](img/f0dab7e36a0ba74ff1e01bbe5d741a43_14.png)

## 第三步：获取与发布素材（补充说明）

提交邀请函后，就可以开始准备发布内容了。作为示例，导师会为学员提供已处理好的视频素材。

![](img/f0dab7e36a0ba74ff1e01bbe5d741a43_16.png)

![](img/f0dab7e36a0ba74ff1e01bbe5d741a43_18.png)

例如，导师为学员“周建国”提供了三个影视剪辑素材，并编号为1、2、3，方便其按顺序发布。这些素材来源于拥有正规版权的素材库，涵盖了影视、动漫等多种类型，例如《还珠格格》、《我不是药神》等，确保内容发布的合规性。
