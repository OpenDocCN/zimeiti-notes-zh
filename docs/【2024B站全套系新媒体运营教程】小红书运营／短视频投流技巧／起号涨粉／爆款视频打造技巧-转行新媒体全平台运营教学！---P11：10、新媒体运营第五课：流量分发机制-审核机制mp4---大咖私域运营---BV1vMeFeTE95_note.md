## 内容审核机制 🔍

上一节我们介绍了内容定位，本节中我们来看看内容发布后的第一道关卡：审核机制。所有平台的内容审核主要分为两大块：**机器审核**和**人工审核**。绝大部分内容的前期审核由机器完成。

### 机器审核

![](img/b218f335f62b66c71e84a44ab434a665_15.png)

![](img/b218f335f62b66c71e84a44ab434a665_17.png)

笔记或视频发布后，首先由机器审核内容是否合规。合规性主要包含以下两点：

![](img/b218f335f62b66c71e84a44ab434a665_19.png)

1.  **敏感词检测**：系统会检测你的图文、口播文案或页面中是否包含平台禁止的敏感词汇。例如违反广告法的词汇、法律禁止的内容、政治敏感话题或平台不允许的“黑五类”赛道内容。如果存在敏感词，内容可能被直接限流、审核不通过或打回。
2.  **营销广告检测**：平台（尤其是基于兴趣分发的短视频或基于种草分享的图文平台）不鼓励直接、生硬的广告内容。如果笔记中直接宣传产品、服务、价格优势等，容易被判定为营销广告，导致前期流量受损。因此，初期应注重分享价值，而非直接推广。

![](img/b218f335f62b66c71e84a44ab434a665_21.png)

![](img/b218f335f62b66c71e84a44ab434a665_23.png)

以下是机器审核的核心流程伪代码描述：
```python
def machine_review(content):
    if contains_sensitive_keywords(content):
        return "限流或打回"
    elif is_hard_advertisement(content):
        return "判定为营销广告，限流"
    else:
        return "进入下一步：去重检测"
```

### 去重检测

平台拥有自己的内容数据库。你的内容发布后，系统会提取其特征，并与数据库中的历史内容进行比对。

![](img/b218f335f62b66c71e84a44ab434a665_31.png)

*   如果发现高度或完全重合，内容会被判定为**抄袭**。
*   结果通常是**打回**或**限流**。

因此，创作时应组合他人内容的优秀价值点，但必须进行调整、优化或用自身语言重新表述，避免完全照搬。利用AI工具辅助修改也是一种方法，但需注意合理使用。

![](img/b218f335f62b66c71e84a44ab434a665_33.png)

![](img/b218f335f62b66c71e84a44ab434a665_35.png)

### 人工审核

![](img/b218f335f62b66c71e84a44ab434a665_37.png)

人工审核主要在两种情况下介入：

![](img/b218f335f62b66c71e84a44ab434a665_39.png)

![](img/b218f335f62b66c71e84a44ab434a665_41.png)

![](img/b218f335f62b66c71e84a44ab434a665_43.png)

1.  **机器审核存疑**：当机器检测到内容可能存在敏感或违规风险时，会拉长审核时间，并将内容提交给人工审核员进行最终判断。一旦人工确认违规，流量将受到严重影响。通常，机器审核未通过的内容，在人工审核阶段也很难通过。
2.  **日常人工排查**：平台会有审核人员定期对站内已有内容进行抽查，检查是否有违规或“擦边”行为。这可能导致之前已发布的内容在后期被提示违规或打回。

![](img/b218f335f62b66c71e84a44ab434a665_45.png)

![](img/b218f335f62b66c71e84a44ab434a665_47.png)

![](img/b218f335f62b66c71e84a44ab434a665_49.png)

人工审核的存在是因为任何平台的自动审核规则都存在漏洞。有些内容可能通过特殊方式绕过机器审核，但会在人工排查或用户举报环节暴露风险。

![](img/b218f335f62b66c71e84a44ab434a665_51.png)

## 流量分发机制 🚀

内容审核通过后，便进入流量分发阶段。系统会决定将内容分发给哪些用户。这个过程主要分为两步：**特征提取与分类**和**数据测试**。

![](img/b218f335f62b66c71e84a44ab434a665_53.png)

![](img/b218f335f62b66c71e84a44ab434a665_55.png)

### 第一步：特征提取与初步分类

系统会分析你的笔记内容，为其打上初步标签，以确定分发的目标人群。主要通过以下三个维度：

![](img/b218f335f62b66c71e84a44ab434a665_57.png)

1.  **关键词分析**：系统通过语义分析、中文分词等技术，从你的标题、正文、语音识别字幕、封面及图片文案中提取核心主题关键词。
2.  **Tag标签**：发布时手动添加的话题或标签为系统提供参考，但并非决定性因素，主要用于辅助判断。
3.  **账号历史数据**：系统会参考该账号以往发布内容的数据表现。如果历史内容垂直度高（例如一直发布护肤类内容），且获得过相关兴趣用户的积极反馈，那么新内容在该领域的分类权重会相应提高。

![](img/b218f335f62b66c71e84a44ab434a665_59.png)

![](img/b218f335f62b66c71e84a44ab434a665_61.png)

![](img/b218f335f62b66c71e84a44ab434a665_63.png)

**这正是我们强调要做“垂直账号”的原因**。账号垂直度越高，系统对你的内容分类越精准，越容易将内容分发给真正感兴趣的用户，从而获得更好的初始数据反馈。

![](img/b218f335f62b66c71e84a44ab434a665_65.png)

![](img/b218f335f62b66c71e84a44ab434a665_67.png)

![](img/b218f335f62b66c71e84a44ab434a665_69.png)

我们可以用一个简单的公式来理解初步分类的权重计算：
`初步分类得分 = f(关键词匹配度， Tag标签相关度， 账号历史垂直度)`
得分越高，内容被分发给精准人群的概率越大。

### 第二步：数据测试与内容质量评估

![](img/b218f335f62b66c71e84a44ab434a665_71.png)

完成初步分类后，系统会将内容分发给一小部分测试用户，并根据他们的行为反馈来判断内容质量和更精确的分类。核心评估指标如下：

![](img/b218f335f62b66c71e84a44ab434a665_73.png)

![](img/b218f335f62b66c71e84a44ab434a665_75.png)

以下是评估内容质量的四个关键用户行为维度：

![](img/b218f335f62b66c71e84a44ab434a665_77.png)

![](img/b218f335f62b66c71e84a44ab434a665_79.png)

1.  **点击率 (CTR)**：`点击率 = 点击次数 / 曝光次数`。它反映了封面和标题的吸引力。高点击率意味着内容在第一时间抓住了用户眼球。
2.  **停留时长/完播率**：在视频平台体现为平均播放时长，在图文平台体现为用户浏览笔记的时长（包括阅读、翻页、查看评论等）。时长越长，说明内容越有吸引力。
3.  **浏览深度 (PV)**：指用户在一次访问中浏览的页面数量。例如，用户从推荐页点击你的笔记，又进入你的主页看了其他笔记。浏览页面越多，说明用户对你的账号越感兴趣，为账号带来的数据增益越高。
4.  **互动行为**：包括点赞、收藏、评论、分享、关注等。这些行为直接反映了用户对内容的认可度和参与度。

![](img/b218f335f62b66c71e84a44ab434a665_81.png)

![](img/b218f335f62b66c71e84a44ab434a665_83.png)

![](img/b218f335f62b66c71e84a44ab434a665_85.png)

![](img/b218f335f62b66c71e84a44ab434a665_87.png)

系统会综合这些维度的数据，给你的内容质量打分。数据表现越好，内容就越容易被判定为“优质”，从而进入下一轮更大范围的推荐流量池。

*   **发现页/首页**：这是最大的公域流量入口。平台根据用户标签和实时热度，混合推荐各种内容。用户在此浏览多基于兴趣探索。
*   **关注页**：用户在此查看已关注博主的新内容。这部分流量目的性非常强，用户粘性高。
*   **同城页**：满足用户基于地理位置的需求，如寻找本地美食、住宿、娱乐等。
*   **搜索页**：用户有明确意图时使用，例如搜索具体问题或产品。搜索流量精准，转化潜力大。
*   **购物/商城入口**：用户带有明确的购物需求，是电商转化的重要场景。

## 总结 📝

![](img/b218f335f62b66c71e84a44ab434a665_113.png)

![](img/b218f335f62b66c71e84a44ab434a665_115.png)

本节课我们一起学习了新媒体平台流量分发与审核的核心机制。

1.  **审核是门槛**：内容需先通过**机器审核**（敏感词、营销广告、去重检测）和可能的**人工审核**，确保合规后方能进入分发流程。
2.  **分发是算法游戏**：系统通过**提取特征**（关键词、标签、历史数据）对内容进行初步分类，然后进行**数据测试**，依据**点击率、停留时长、浏览深度、互动行为**等指标评估内容质量，决定是否给予更多流量。
3.  **优化需有的放矢**：创作者应规避审核风险，保持内容原创，并围绕分发算法的评估维度（如制作吸引人的封面标题提升点击率，创作有价值的内容提升停留和互动）来优化内容。
4.  **流量入口多元化**：内容可能出现在**发现页、关注页、搜索页**等不同入口，对应着用户不同的意图和需求。

![](img/b218f335f62b66c71e84a44ab434a665_117.png)

![](img/b218f335f62b66c71e84a44ab434a665_119.png)

理解这套机制，能帮助你有策略地创作和优化内容，从而在新媒体运营中更有效地获取流量。
