# 自媒体运营课程：P36：剪映核心剪辑功能--贴纸 🎬

在本节课中，我们将要学习剪映软件中的一个核心功能——**贴纸**。贴纸是提升视频画面表现力和趣味性的重要工具，广泛应用于VLOG、美食、旅拍等多种视频类型中。

上一节我们介绍了其他剪辑功能，本节中我们来看看如何利用贴纸为视频增色。

![](img/5400eb84c2e6de3a306482c3abcdcd63_1.png)

## 贴纸功能概述 🎨

贴纸最初常用于照片修图，如今在视频制作中同样扮演着重要角色。在视频中添加合适的贴纸，能有效提升画面的整体质量，并帮助创作者更清晰地传递信息。

在剪映软件中，内置了强大的贴纸添加功能。其素材库提供了海量的贴纸元素，包括平台热门款式。使用这些贴纸可以显著增强视频的视觉效果。

![](img/5400eb84c2e6de3a306482c3abcdcd63_3.png)

例如，在制作换装或变身类视频时，贴纸能极大地增强画面冲击力。对于日常VLOG类视频，添加贴纸则能提升视频的形象感和趣味性。

![](img/5400eb84c2e6de3a306482c3abcdcd63_5.png)

## 贴纸功能实操指南 ✂️

![](img/5400eb84c2e6de3a306482c3abcdcd63_6.png)

以下是贴纸功能的具体操作步骤。

1.  **新建项目与导入素材**
    首先，在剪映中新建一个项目，并导入需要编辑的视频素材。

2.  **进入贴纸菜单**
    在软件主界面的一级菜单左下方，找到并点击 **“添加贴纸”** 按钮。

3.  **选择与添加贴纸**
    点击后，会进入贴纸素材库。库中包含多种类别的贴纸，例如：
    *   表情类贴纸（如“寂寞表情”）
    *   装饰类贴纸（如太阳、彩虹、蝴蝶、花朵）
    *   文字类贴纸
    *   边框类贴纸
    你可以根据视频题材和风格，浏览并选择适合的贴纸，点击即可添加到视频轨道上。

4.  **设置贴纸属性**
    添加贴纸后，可以进行以下调整：
    *   **时长设置**：拖动贴纸轨道两端的边缘，可以调整其显示时长。你可以让贴纸贯穿整个视频，也可以让它只在特定片段出现。
    *   **动画设置**：选中贴纸，在编辑菜单中可以为它添加入场、出场等动画效果，让贴纸的出现方式更生动。

![](img/5400eb84c2e6de3a306482c3abcdcd63_8.png)

5.  **预览与调整**
    完成设置后，预览视频效果。根据预览情况，可以返回调整贴纸的位置、时长或动画，直到满意为止。

## 课程总结 📝

本节课中我们一起学习了剪映的贴纸功能。主要要点如下：
*   贴纸能快速提升视频的画面氛围，让内容更加生动有趣。
*   操作路径是：点击 **“添加贴纸”** -> 从素材库选择 -> 添加到时间轴。
*   添加后，关键操作是设置贴纸的 **显示时长** 和 **动画特效**。

![](img/5400eb84c2e6de3a306482c3abcdcd63_10.png)

掌握贴纸的灵活运用，能为你的视频创作增添更多创意和表现力。
