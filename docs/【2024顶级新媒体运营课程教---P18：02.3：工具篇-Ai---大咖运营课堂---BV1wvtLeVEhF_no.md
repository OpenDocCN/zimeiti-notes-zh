## 使用方法一：导入本地大纲生成

![](img/c3e0bd56fbf43423d6516a1c9244874d_1.png)

![](img/c3e0bd56fbf43423d6516a1c9244874d_3.png)

我通常使用“导入本地大纲”的方式来生成PPT，这种方法对内容的把控度更高。

首先，我们需要准备一份详细的大纲。可以使用ChatGPT等AI助手来辅助生成。例如，我们可以向ChatGPT提问：“帮我撰写一份‘装修行业在小红书平台的运营推广计划’的PPT大纲，并完善每一个小节点的具体内容信息，保持好结构和格式排版。”

AI生成的大纲可能在某些细节上不够完善，但我们可以以此为基础进行修改和填充。

准备好大纲文档后，即可开始操作：

1.  在工具界面选择 **`导入本地大纲`** 功能。
2.  上传准备好的大纲文档文件。
3.  工具会快速梳理文档的大纲结构。
4.  从丰富的模板库中，挑选一个适合主题（如装修行业）的风格模板。
5.  点击 **`生成PPT`**。

生成完成后，如果发现某些页面内容缺失或排版不理想，可以在编辑界面进行调整：

*   在 **`大纲编辑`** 区域，可以补充或修改章节的详细内容。
*   单击任一页面，右侧会出现 **`文字设置`**、**`形状设置`**、**`背景设置`** 等编辑选项。
*   如果对页面布局不满意，可以单击页面，使用 **`一键更换`** 功能选择其他布局模板。
*   整个PPT的模板也可以随时通过 **`模板替换`** 功能进行整体更换，响应速度很快。

**核心操作流程总结为代码形式如下：**
```plaintext
1. 准备详细大纲文档
2. 选择“导入本地大纲”
3. 上传文档 -> 系统解析结构
4. 选择主题模板
5. 点击“生成PPT”
6. 进入编辑界面进行微调（文字/布局/模板替换）
```

## 使用方法二：AI智能生成

除了导入大纲，该工具也支持直接通过主题智能生成。

1.  在工具界面选择 **`AI智能生成`** 功能。
2.  在输入框中填写详细的PPT主题，例如：“装修行业在小红书平台的运营推广计划”。
3.  点击生成，工具会先自动构思并生成大纲。
4.  大纲确认后，挑选一个合适的模板。
5.  点击 **`生成PPT`**，工具将根据主题自动生成包括详细文案在内的完整PPT内容。

生成完成后，同样可以进入编辑页面，对所有字体、样式、模板进行调整和替换，也可以编辑大纲的具体内容。
