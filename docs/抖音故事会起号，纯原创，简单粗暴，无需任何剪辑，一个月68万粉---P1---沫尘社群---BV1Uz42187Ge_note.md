# 抖音故事会起号教程：P1：纯原创图文视频制作方法 🎬

在本节课中，我们将要学习一种在抖音上快速起“故事会”账号的方法。该方法无需复杂剪辑，通过制作纯原创的图文视频，即可实现快速涨粉。我们将从分析对标账号开始，逐步讲解素材准备、视频制作到最终发布的完整流程。

## 对标账号分析 🔍

上一节我们介绍了课程目标，本节中我们来看看具体的操作方法。首先，我们需要理解成功的对标账号是如何制作的。

![](img/547cf55c4f5da2baa1ebe43abcd190fc_1.png)

观察对标账号可以发现，其所有视频都由几张静态图片构成。例如，一个典型的视频标题是“53岁的婆婆再婚，儿媳随礼8万块钱，但是看到新郎以后”。这类视频的核心是几张图片搭配一段背景音乐。我们重点研究的账号，其技术核心在于“卡图文”技术。

![](img/547cf55c4f5da2baa1ebe43abcd190fc_3.png)

## 素材准备步骤 📥

![](img/547cf55c4f5da2baa1ebe43abcd190fc_5.png)

了解了对标账号的模式后，接下来我们需要准备制作视频所需的素材。以下是具体步骤：

![](img/547cf55c4f5da2baa1ebe43abcd190fc_7.png)

1.  **收藏或搜索音频**：进入对标账号的视频，将其使用的背景音乐收藏起来。或者，也可以根据故事主题去其他平台搜索合适的音频。
2.  **搜索并保存图片**：根据视频文案（例如“老年人结婚”），去百度等搜索引擎搜索相关图片。保存2到3张符合故事内容的图片备用。

![](img/547cf55c4f5da2baa1ebe43abcd190fc_9.png)

![](img/547cf55c4f5da2baa1ebe43abcd190fc_11.png)

![](img/547cf55c4f5da2baa1ebe43abcd190fc_1.png)

## 使用剪映制作图文视频 ✂️

![](img/547cf55c4f5da2baa1ebe43abcd190fc_13.png)

准备好素材后，我们进入视频制作环节。这里使用“剪映”App将图片转换为视频格式。

![](img/547cf55c4f5da2baa1ebe43abcd190fc_15.png)

1.  **打开剪映并导入图片**：打开剪映App，点击“开始创作”。导入之前保存好的2-3张图片。
    ![](img/547cf55c4f5da2baa1ebe43abcd190fc_3.png)
2.  **编辑图文内容**：每张图片默认时长为3秒。根据对标账号的文案（如“再婚的婆婆”），为图片添加文字标题。
    ![](img/547cf55c4f5da2baa1ebe43abcd190fc_5.png)
3.  **调整比例与导出**：将视频比例调整为9:16（竖屏）。调整文字大小和位置，使其清晰美观。编辑完成后，为视频选择一个封面并导出。
    ![](img/547cf55c4f5da2baa1ebe43abcd190fc_7.png)
    ![](img/547cf55c4f5da2baa1ebe43abcd190fc_9.png)
4.  **制作第二个图文视频**：返回剪辑界面，将第一张图片替换为另一张已保存的图片，制作第二个图文视频并导出。至此，我们得到了两个独立的图文视频。
    ![](img/547cf55c4f5da2baa1ebe43abcd190fc_11.png)

## 获取并使用抖音模板 🎞️

![](img/547cf55c4f5da2baa1ebe43abcd190fc_17.png)

![](img/547cf55c4f5da2baa1ebe43abcd190fc_19.png)

为了使视频更具吸引力，我们需要使用一个抖音热门模板。以下是操作方法：

![](img/547cf55c4f5da2baa1ebe43abcd190fc_21.png)

1.  **搜索模板**：在对标账号中，观察其使用的模板（例如“变回小时候的样子”）。在抖音的搜索栏中搜索该模板名称。
    ![](img/547cf55c4f5da2baa1ebe43abcd190fc_13.png)
2.  **使用模板制作视频**：选择一个使用量高的模板（如400多万使用量的）。点击“剪同款”，随意上传一张图片，生成模板视频并“导出无水印”保存。
    ![](img/547cf55c4f5da2baa1ebe43abcd190fc_15.png)
    ![](img/547cf55c4f5da2baa1ebe43abcd190fc_17.png)

![](img/547cf55c4f5da2baa1ebe43abcd190fc_22.png)

![](img/547cf55c4f5da2baa1ebe43abcd190fc_24.png)

## 在抖音中完成最终合成与发布 📱

现在，我们拥有了所有组件：两个图文视频和一个模板视频。最后一步是在抖音App内完成合成并发布。

1.  **使用对标音频**：打开抖音，找到对标账号的视频。点击其右下角的“碟片”图标，选择“拍同款”。此时，拍摄界面已经加载了该视频的音频。
    ![](img/547cf55c4f5da2baa1ebe43abcd190fc_21.png)
    ![](img/547cf55c4f5da2baa1ebe43abcd190fc_22.png)
2.  **上传并排列素材**：在拍摄界面，先随意选择2张图片。然后点击“上传”，将之前制作好的两个图文视频和模板视频按顺序上传。
    ![](img/547cf55c4f5da2baa1ebe43abcd190fc_24.png)
3.  **关键操作：“卡模板”**：
    *   上传后，删掉最开始随意选的那2张图片。
    *   点击“下一步”，此时会看到“剪同款”按钮出现。
    *   点击返回，删掉模板视频。
    *   检查音频是否还在（播放试听）。确认后再次点击“下一步”。
    *   **对于安卓手机**：若直接发布不显示模板，可将模板视频调整到素材首位，点击“下一步”出现模板选项后，再返回删掉它。或者，将模板视频的时长裁剪到最短（如1秒），保留其模板属性。
    *   最终，确保发布页面只保留两个图文视频，并带有热门模板的样式和背景音乐。
4.  **发布视频**：编辑好标题等描述信息后，直接发布即可。

## 总结与后续学习 📚

本节课中我们一起学习了抖音故事会账号的起号方法。其核心流程可以概括为：**分析对标 -> 准备音频图片 -> 用剪映制作图文视频 -> 获取抖音模板 -> 在抖音内完成“卡图文”合成与发布**。

这是一个简单有效的原创内容制作玩法。更多关于涨粉、搬运等运营技术，将在我们的社群及公众号“沫尘创业网”中持续更新。
