# 抖音同城生活-健康垂类0到1运营：P25：眼科团购组品攻略 👁️

![](img/f2208a90c660784742837b3d26c491ff_0.png)

在本节课中，我们将学习如何为眼科诊所或医院在抖音本地生活平台上，从零开始规划并上架有效的团购产品组合。我们将从官方基础产品出发，探讨如何结合自身优势进行差异化竞争，并拓展至电商领域，实现更全面的用户触达与转化。

上一节我们介绍了眼科团购的整体框架，本节中我们来看看具体的产品组合策略。

## 官方基础产品：青少年视力检查

抖音官方为眼科类目开放团购后，最明确且基础的可上架产品是**青少年视力检查**。商家可以在设备、专家服务以及价格等多个维度进行设置，从而组合出不同的团购套餐。

## 差异化竞争策略：结合门店优势

以下是基于经验总结的两种差异化组品思路：

1.  **与关联业务捆绑**：许多眼科诊所附设有眼镜店。可以将视力检查团购与眼镜店的配镜代金券等产品进行捆绑销售，形成联动效应。
2.  **聚焦高价值产品**：避免在普通树脂镜片等低价产品上进行价格竞争。作为专业机构，应在**离焦镜**等高客单价项目上做文章。离焦镜是目前控制青少年近视的主流品类之一。需要思考如何将这类产品与OK镜等线下服务组合，并设计相应的私域流量转化路径，这里存在多种运营玩法。

## 拓展边界：结合内容与电商

第三种策略已超出纯粹本地生活的范畴，但可以与门店账号的内容运营紧密结合，即上架视光类电商产品。

以下是可考虑的电商产品示例：
*   反转拍等知识付费内容。
*   儿童钙片。
*   护眼药膏。
*   护眼学习灯。
*   叶黄素等营养补充剂。

愿意购买此类产品的家长，无疑是眼科机构的潜在精准用户。因此，视光类目相比其他消费医疗类目更具灵活性，能够融合抖音电商与抖音本地生活，进行多元化产品组合。

![](img/f2208a90c660784742837b3d26c491ff_2.png)

本节课中我们一起学习了眼科团购的三大组品方向：从官方的视力检查入手，通过捆绑销售和聚焦高客单价项目实现差异化，最后拓展至视光电商产品以实现更精准的用户筛选与价值延伸。视光类目的运营可以巧妙结合平台的内容、电商与本地生活功能，创造出丰富的玩法。
