# 抖音运营入门：P9：1、抖音短视频的前景

![](img/d3336d65ee4b304a9b33687e4fc40e1c_1.png)

在本节课中，我们将要学习抖音短视频平台的发展前景与核心优势，帮助你理解为何选择抖音作为内容创作的起点。

![](img/d3336d65ee4b304a9b33687e4fc40e1c_3.png)

了解抖音短视频的前景，首先需要通过一张图片来建立直观认识。

![](img/d3336d65ee4b304a9b33687e4fc40e1c_1.png)

这张图片来自抖音创作者大会的数据报告。报告显示，有超过2200万人在抖音的合计收入超过了417亿元。抖音希望将这个数字翻一番，让创作者的总收入达到800亿元。

![](img/d3336d65ee4b304a9b33687e4fc40e1c_3.png)

这体现了平台巨大的流量扶持力度。同时，这里需要明确第一个知识点：抖音用户主要分为两类。第一类是“玩家”，他们单纯使用抖音娱乐，没有明确的赚钱目标。第二类是“创作者”，即像我们这样希望通过制作内容来获取收益的人。明确这两类人的区别很重要，后续内容中还会提及。

那么，在众多短视频平台中，我们为何选择抖音？这类似于开设实体店铺。选择店址时，你一定会寻找流量大、位置佳的地方。做短视频同理，必须选择一个流量庞大的平台。如果平台自身流量不足，就无法分配流量给创作者，我们也难以赚到钱。抖音的日活跃用户超过8亿，月活跃用户更是达到了40亿人次。这是一个极其庞大的流量池。

流量巨大，竞争压力是否也会很大？答案是不会。平台上的创作者数量约为5000万。这里的“创作者”定义广泛，并非特指网红达人。例如，邻居家的老太太用手机拍摄自己做饭的视频并上传到抖音，她也算创作者。因此，你90%的竞争压力，实际上来自于与你相似的普通用户。

接着，做抖音短视频是否需要购买专业设备？答案是不需要。你只需要一部智能手机和一个手机号，就可以注册抖音账号。抖音倡导全民通过短视频分享和记录生活，门槛很低。

那么，当前有哪些值得关注的风口？首先是“抖音计划”。该计划旨在帮助1000名优质的艺术创作者实现粉丝增长超过100万。可能有同学担心自己不会创作。这无需忧虑，正因为我们有所欠缺，才需要学习。老师曾指导过许多学员，包括年长者。他们秉持的态度是：虚心向比自己厉害的人请教并不丢人。只要对方在自己不懂的领域有专长，而自己又想学习，对方就是老师。这种谦虚好学的态度，能让人少走很多弯路。

市场空间巨大，竞争环境相对宽松，行业内容产出也相对容易。我们正处于大数据时代。在这个时代，平台会根据你的喜好推荐内容。例如，你喜欢看帅哥，刷到的就多是相关内容；你喜欢看美女，看到的也多是相关视频。这可能导致我们接触的内容比较片面。

对于内容创作，许多新手存在困惑。以下是几种常见情况及其可行性分析：
*   **宝妈**：可以拍摄宝宝或自己做饭的日常。
*   **实体店主**：可以拍摄产品，用于引流。
*   **普通上班族**：可以分享日常生活。

为什么这些内容都有机会？因为面对每月40亿的庞大用户群体，只要你的内容不触犯法律和道德底线，不是低俗内容，总会有一部分人喜欢。你之所以可能只有几十个粉丝，是因为喜欢你的人还没有刷到过你的作品。

因此，对于新手小白而言，虽然涨粉心情迫切，但不要把涨粉作为首要目标。我们的第一个目标应该是：**先把流量（播放量）做起来**。播放量提升后，粉丝自然会增长。

本节课中，我们一起学习了抖音平台的巨大潜力、用户分类、低门槛优势以及新手起步的核心策略。关键在于选择这个流量高地，并以提升内容流量为首要目标。
