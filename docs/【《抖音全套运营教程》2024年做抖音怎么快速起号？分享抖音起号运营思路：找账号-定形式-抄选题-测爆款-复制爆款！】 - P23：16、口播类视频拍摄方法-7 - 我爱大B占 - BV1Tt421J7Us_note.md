# 抖音全套运营教程：P23：16、口播类视频拍摄方法-7 🎬

在本节课中，我们将要学习口播类视频的具体拍摄方法。我们将通过一个成功案例，详细拆解拍摄时需要注意的构图、内容定位等核心要点，帮助你掌握制作高质量口播视频的技巧。

---

上一节我们介绍了视频拍摄的通用注意事项，本节中我们来看看如何将这些要点应用到具体的口播视频拍摄中。

![](img/5a9a3f2265af34c907570461db7e7760_1.png)

口播视频的拍摄方法具有高度通用性。无论你从事金融、装修、养生还是情感领域，其核心拍摄逻辑都是一致的。同样，记录日常生活、拍摄段子或美食等内容，其拍摄方法也基本相同。因此，掌握以下两种类型的拍摄思路，几乎就能覆盖所有场景。

首先，我们来看第一个类型：口播视频。我将以我们的一位学员“种小草”为例进行讲解。

![](img/5a9a3f2265af34c907570461db7e7760_1.png)

你可以搜索到她的账号。她是一位40岁、带着三个孩子的宝妈，专注于情感口播。她的一条视频播放量超过8000万，并凭借该视频涨粉30万。为什么她的一条视频就能取得如此效果，而我们涨粉却很难？

关键在于她吸引了高度精准的受众群体。看完她的视频，你就能立刻明白她的目标观众是谁。例如，如果我的目标是20多岁的学生群体，那么我拍摄的所有内容都围绕学生感兴趣的话题，这样我的涨粉速度就会非常快，且粉丝非常精准。

我们先从构图说起。抖音是**9:16**的竖屏平台。但关键在于视频内容本身是采用横屏还是竖屏构图。你看她的视频，采用的是**4:3**的横屏构图。

**核心要点**：无论你做哪种类型的视频，构图比例必须统一。如果你选择横屏，就全部使用横屏；如果选择竖屏，就全部使用竖屏。保持稳定，这是需要注意的细节。

接着，人物主体应占据画面中央约**70%**的面积。不要怼脸拍摄，那样会让人观感不适。确保画面的左、右、上方留有适当空白，这样看起来画面会更清晰、舒适。

如果你觉得自己外貌并不出众，是否还能做口播？答案是肯定的。抖音的成功并非完全依赖颜值。口播的核心在于传达内容和知识。你只需要让他人觉得你清晰、体面即可。如果对自己外貌不自信，可以开启美颜滤镜。在这个时代，合理使用美颜功能，能让每个人都以更好的状态出镜。请放心使用，只要不过于夸张即可。

接下来，我将带你观看她这条涨粉30万的视频。我的要求是：看完后请直接回答，这个视频想要吸引的是哪个特定群体？

![](img/5a9a3f2265af34c907570461db7e7760_3.png)

![](img/5a9a3f2265af34c907570461db7e7760_3.png)

（视频内容示例：“跟婆婆在一起住是什么样的感受？我跟我婆婆在一起住了16年了，平时磕磕碰碰有点小不愉快，那都是正常的……我最大的感受就是这十多年以来，我娘家人很少在我家住上几天……”）

有同学提到“婆媳关系”。但这具体指向哪个群体呢？婆媳关系是一种关系，那么谁对此最有感触？男性群体对此的感触通常没有女性深刻。而未结婚的十几岁、二十几岁的小姑娘，也很难有切身感受。因此，最能产生共鸣的，是**已婚的女性群体**。她吸引的正是这个精准的受众。我们稍后可以查看数据来验证。

另外，请注意她是否使用了美颜？答案是肯定的。所以，如果你觉得自己不上镜，可以适当开启美颜，让自己看起来更体面。她的构图也是**4:3**的横屏。

有同学问：“老师，做口播是不是必须把稿子背下来？我记性不好怎么办？” 你不需要背稿。在剪辑环节，我会推荐一个好工具，让你可以边看提示边录制，这比背诵简单得多。

![](img/5a9a3f2265af34c907570461db7e7760_5.png)

做口播，另一个关键是声音必须清晰。因此，我推荐你购买一个手机收声麦克风。这东西很便宜，大约十几块钱，在淘宝、京东等平台都能买到。它能显著提升你视频的音频质量。

以下是拍摄口播视频的关键步骤总结：

![](img/5a9a3f2265af34c907570461db7e7760_5.png)

1.  **确定构图比例**：统一使用**9:16**竖屏中的**4:3**横屏或全竖屏构图。
2.  **安排人物位置**：人物居中，约占画面**70%**，四周留白。
3.  **运用辅助工具**：合理使用美颜功能提升画面观感；使用**手机收声麦**保证音频清晰。
4.  **明确内容定位**：内容必须针对特定群体（如案例中的**已婚女性**），以引发深度共鸣。
5.  **准备讲稿方式**：无需背诵，可借助提词工具进行录制。
