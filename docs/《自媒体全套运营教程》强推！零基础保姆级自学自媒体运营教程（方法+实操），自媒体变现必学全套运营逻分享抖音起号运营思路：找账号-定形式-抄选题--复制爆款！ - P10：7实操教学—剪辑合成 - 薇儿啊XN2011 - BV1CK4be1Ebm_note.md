# 自媒体全套运营教程：P10：7实操教学—剪辑合成 🎬

在本节课中，我们将学习如何将翻译好的字幕与视频素材进行剪辑合成，并添加动态水印，最终完成一个可供发布的视频作品。

---

上一节我们介绍了如何为视频添加字幕，本节中我们来看看如何将字幕与视频素材结合，并进行最终的剪辑与包装。

视频与字幕素材已准备就绪。首先，我们需要检查翻译字幕的效果。播放视频，观察字幕与画面的匹配度及可读性。

以下是检查字幕效果的步骤：
1.  播放视频，确认字幕时间轴基本准确。
2.  阅读中文字幕，确保其含义在画面配合下易于理解。
3.  对于不影响理解的冗余英文部分，可直接删除，保留核心中文信息。

我们的核心目标是：**字幕无需逐字精确，但需确保观众能结合画面理解视频大意**。中文阅读能力很强，简洁达意的字幕配合画面已足够。

确认字幕后，即可将其导出。我们仅需中文字幕文件。

![](img/5886e85fbb404dafe87b45247d025b70_1.png)

以下是导出字幕的步骤：
1.  在字幕工具中，选择“仅导出中文字幕”选项。
2.  将字幕文件保存至本地设备。

![](img/5886e85fbb404dafe87b45247d025b70_3.png)

![](img/5886e85fbb404dafe87b45247d025b70_5.png)

---

字幕导出后，下一步是使用剪辑软件将字幕与视频合成。我们将使用“剪映”专业版进行操作。

**注意**：请确保你使用的是**剪映3.0或更高版本**，旧版本可能不支持字幕导入功能。你可以在软件内检查版本并进行更新。

以下是导入素材与字幕的步骤：
1.  打开剪映专业版，创建新项目。
2.  点击“导入”或直接将之前下载好的视频素材拖入素材库。
3.  将视频素材拖到下方的时间轨道上。
4.  在界面左侧功能栏中找到“文本”选项。
5.  在“文本”栏目最下方，点击“本地字幕导入”。
6.  选择刚刚导出的中文字幕文件（通常是 `.srt` 或 `.ass` 格式），将其导入到时间轨道中。

导入后，播放视频预览效果。你可能会发现字幕的样式（如大小、字体）需要调整。

以下是调整字幕样式的步骤：
1.  在时间轨道上选中字幕片段。
2.  在右侧的编辑面板中，调整“字体”、“字号”、“颜色”等参数。
3.  剪映提供了丰富的“花字”预设，你可以选择喜欢的样式。
4.  如果字幕断句不合理，可以在右侧面板的文本框中，根据语义手动调整换行。

调整至字幕清晰美观、观看舒适即可。

![](img/5886e85fbb404dafe87b45247d025b70_7.png)

---

视频与字幕合成后，我们还需要增加一道防搬运的工序——添加水印。静态水印容易被去除，因此推荐制作动态水印。

动态水印的原理是让水印（如你的账号名）在视频播放过程中持续移动。这可以通过在剪辑软件中为水印的位置添加**关键帧动画**来实现。

以下是添加动态水印的步骤：
1.  在剪映的“文本”工具中，添加一个默认文本或选择一个花字。
2.  在文本框中输入你的账号名称，例如“安雅看综艺”。
3.  将文本拖到视频预览画面的左上角，并调整大小和透明度（建议降低透明度，避免过于遮挡画面）。
4.  将时间轨道上的这个文本长度拉至与视频时长一致。
5.  在视频起始位置，选中文本，在右侧编辑面板的“位置”参数旁，点击菱形图标**添加一个关键帧**。这记录了水印的起始位置。
6.  将播放指针移动到视频中间某个时间点，然后在预览窗口中直接将水印拖动到右下角。软件会自动在此刻**生成一个新的关键帧**。
7.  再将播放指针移动到视频末尾，把水印从右下角拖回左上角，**生成第三个关键帧**。
8.  播放预览，水印便会按照你设定的路径（左上→右下→左上）平滑移动。

通过设置动态水印，他人搬运你的视频时将难以彻底去除水印，从而有效保护你的创作。

有同学询问软件名称，再次强调，本教程使用的剪辑软件是**剪映**，它在手机和电脑上均可使用。
