# 小红书电商运营课：P18：第5课：客服的聊天流程 💬

## 概述
在本节课中，我们将学习小红书电商运营中客服聊天的完整流程。我们将重点了解自主客服与官方客服的区别，掌握客服回复的核心规则与指标，并学习如何在电脑端和手机端进行有效的客服沟通。

## 客服模式选择：自主 vs. 官方
上一节我们介绍了客服工作的重要性，本节中我们来看看具体如何选择客服模式。

小红书为商家提供了两种客服模式：官方代聊服务和自主客服。

**官方代聊服务**由平台提供，但需要收费。虽然能节省时间，但成本较高。

**自主客服**是指商家自己或员工进行聊天回复。这是更推荐的选择，原因在于其成本效益。设置路径为：进入小红书后台 -> 点击“客服” -> 进入“客服团队”页面。在该页面，可以将运营模式设置为“自主客服”。

不建议使用官方客服的主要原因是其收费性质。对于大多数新手和小型商家而言，自主客服完全可以满足需求。

## 自主客服的核心规则与指标
选择了自主客服后，理解并遵守平台的规则至关重要。以下是两个最重要的考核指标：

1.  **24小时人工回复率**：此指标要求商家在用户咨询后的24小时内，至少进行一次**人工回复**。平台规定回复率需大于80%。
    *   **公式**：`24小时人工回复率 = (24小时内已人工回复的会话数 / 所有需回复的会话总数) * 100% > 80%`
    *   **核心概念**：“人工回复”指手动输入文字或发送表情，系统自动回复不计入。此规则非常宽松，例如顾客早上咨询，晚上回复即可。
    *   若此指标不达标，可能会影响店铺笔记和商品的曝光。

2.  **3分钟人工回复率**：此指标衡量商家在3分钟内对用户新消息进行人工回复的速度。它主要影响店铺的“咨询体验”评分。
    *   **公式**：`3分钟人工回复率 = (3分钟内已人工回复的会话数 / 所有新消息会话总数) * 100%`
    *   该指标对店铺整体影响相对较小，但快速回复能提升顾客体验。

## 客服后台设置与操作
了解了核心规则后，我们进入后台看看如何进行具体设置和操作。

以下是客服后台的一些实用功能：

![](img/e64b93ffca0816c35db1c4d95b87e357_7.png)

*   **欢迎语与助手**：可以设置自动欢迎语和关键词自动回复。例如，当用户消息中包含“发货”关键词时，系统可自动回复预设的物流信息。
*   **客服数据**：在此查看每日的聊天记录、回复率等数据。
*   **网页端工作台**：在电脑上，通过后台的“客服”->“客服工作台”进行聊天。所有与顾客的“会话”都会在此列表显示。

需要区分“人工回复”与“系统自动回复”。只有商家手动发送的内容才被计入考核指标。

## 移动端客服：小红书商家版APP
如果无法随时使用电脑，手机端管理同样方便。这需要借助“小红书商家版”APP。

在手机端处理客服的步骤如下：
1.  下载并登录“小红书商家版”APP。
2.  在APP首页，点击底部的“消息”入口，即可进入客服聊天窗口。
3.  在此与顾客进行实时沟通。

**关键建议**：为了确保及时回复，请务必开启APP的消息推送通知，并允许声音提醒。这样即使在外出时，也能第一时间收到咨询消息并处理。

![](img/e64b93ffca0816c35db1c4d95b87e357_9.png)

此外，商家版APP还集成了订单管理（待发货）、售后工单处理等功能，是移动端运营的核心工具。

## 总结
本节课我们一起学习了小红书电商的客服聊天全流程。我们明确了**自主客服**是更经济实用的选择，掌握了**24小时人工回复率**和**3分钟回复率**这两个核心考核指标。我们熟悉了如何在电脑后台进行客服设置与沟通，并学会了通过**小红书商家版APP**在移动端高效处理客户咨询。记住，及时、有效的人工回复是提升店铺服务质量和转化率的关键。
