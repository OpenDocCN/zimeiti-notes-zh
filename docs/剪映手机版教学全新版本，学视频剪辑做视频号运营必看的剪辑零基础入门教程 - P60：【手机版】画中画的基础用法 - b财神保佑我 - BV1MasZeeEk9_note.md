# 剪映手机版教学：P60：画中画的基础用法 🎬

在本节课中，我们将要学习剪映手机版中“画中画”功能的基础用法。画中画是视频剪辑中一项非常核心的技术，它允许你在主视频之上叠加额外的视频或图片图层，从而实现丰富的视觉效果。我们将从如何添加画中画开始，逐步介绍其层级管理、切换方法以及几种常见的应用场景。

![](img/0e236769a21cd50c6da3c3d7c1f376b1_1.png)

## 什么是画中画？🔍

画中画功能允许你在一个视频画面上叠加另一个视频或图像。主轨道上的视频位于最底层，而通过“画中画”添加的图层则叠加在其上方。

操作步骤如下：
1.  导入主视频后，在底部工具栏中找到并点击 **`画中画`**。
2.  点击 **`新增画中画`**。
3.  从相册中选择要叠加的视频或图片，点击 **`添加`**。

此时，新添加的素材就会以图层形式出现在主视频画面之上。

## 画中画的层级管理 📊

上一节我们介绍了如何添加画中画，本节中我们来看看如何管理多个画中画图层的上下顺序。

剪映最多支持添加6个画中画图层。当多个图层叠加时，上方的图层会遮挡下方的图层。如果需要调整某个图层的显示顺序，可以使用“层级”功能。

以下是调整层级的具体步骤：
*   **选中**需要调整的画中画图层。
*   在底部工具栏中找到并点击 **`层级`**。
*   在弹出界面中，**长按并拖动**该图层的图标，即可调整其前后位置。将其拖到最前面即为“置顶”，它将显示在所有图层之上。

**核心概念**：主轨道视频永远处于最底层，无法通过层级调整改变。层级功能仅用于调整主轨道上方的所有画中画图层之间的遮挡关系。

![](img/0e236769a21cd50c6da3c3d7c1f376b1_4.png)

## 主轨道与画中画的切换 🔄

除了新增，我们还可以将主轨道上的视频片段直接转换为画中画，反之亦然。

以下是切换方法：
*   **主轨切画中画**：选中主轨道上的一段视频，在底部工具栏中找到并点击 **`切画中画`**。该片段会自动被移动到画中画轨道，并可以自由调整位置和大小。
*   **画中画切主轨**：选中一个画中画图层，在底部工具栏中找到并点击 **`切主轨`**。该图层会移动到主视频轨道，成为背景视频。

![](img/0e236769a21cd50c6da3c3d7c1f376b1_6.png)

## 画中画的应用场景 🛠️

掌握了基础操作后，我们来看看画中画功能在实际剪辑中的几种常见用法。

### 1. 添加电影遮幅

通过添加黑色画中画，可以快速为视频营造出宽荧幕的电影感。

操作流程：
1.  点击 **`新增画中画`**，在“素材库”中选择 **`黑场`** 视频并添加。
2.  将黑色画中画拖动到视频上方，覆盖住一部分画面。
3.  选中该黑色画中画，点击 **`复制`**，并将复制出的图层拖动到视频下方对齐。
4.  调整两个黑色图层的位置，使其形成上下黑边，电影遮幅效果即完成。

**提示**：此时缩放主视频，不会影响画中画黑色遮幅的大小。

![](img/0e236769a21cd50c6da3c3d7c1f376b1_6.png)

### 2. 制作圆形头像/小窗

这种效果常见于游戏直播或解说视频中，将人物头像以小窗形式展示在角落。

操作流程：
1.  **新增画中画**，导入人物视频。
2.  选中该画中画，在工具栏中找到 **`蒙版`**，选择 **`圆形`** 蒙版。
3.  调整圆形蒙版的大小和位置，框选出人物头部，点击确认。
4.  将圆形头像缩小，并移动到画面左上角或右上角。

**进阶美化**：可以为头像添加一个白色边框。
1.  再次 **`新增画中画`**，添加一个 **`白场`** 视频。
2.  同样为白场视频添加 **`圆形蒙版`**，并调整得比头像蒙版稍大一圈。
3.  通过 **`层级`** 功能，将白色圆形图层调整到人物头像图层的下方，即可形成白边效果。

![](img/0e236769a21cd50c6da3c3d7c1f376b1_8.png)

### 3. 制作底部字幕背景

画中画可以用来创建纯净的字幕背景条，使文字更清晰。

操作流程：
1.  **新增画中画**，添加一个 **`黑场`** 视频。
2.  将其放大并拖动到视频画面底部。
3.  此时，可以返回主菜单，使用 **`文本`** 功能在黑色背景条上添加字幕，文字会非常醒目。

![](img/0e236769a21cd50c6da3c3d7c1f376b1_8.png)

![](img/0e236769a21cd50c6da3c3d7c1f376b1_10.png)

### 4. 制作三分屏效果

![](img/0e236769a21cd50c6da3c3d7c1f376b1_12.png)

三分屏是一种流行的视频排版方式，可以同时展示多个画面。

操作流程：
1.  导入一段主视频，将画面比例设置为 **`9:16`**（竖屏）。
2.  选中该视频，点击 **`切画中画`**，将其转为画中画图层。
3.  再次导入同一段视频到主轨道（或复制主轨道视频）。
4.  选中画中画图层，点击 **`复制`**，得到两个相同的画中画。
5.  调整三个视频图层（一个主轨，两个画中画）的位置和大小，将它们均匀地排列在竖屏画面中。

![](img/0e236769a21cd50c6da3c3d7c1f376b1_10.png)
![](img/0e236769a21cd50c6da3c3d7c1f376b1_12.png)

### 5. 替换边框模板中的内容

我们经常能看到带有固定边框的综艺感视频，利用画中画可以轻松替换边框内的内容。

操作流程：
1.  **新增画中画**，导入一个带有透明区域的“边框”素材。
2.  再次 **`新增画中画`**，导入你想放在边框内的视频。
3.  调整视频的大小和位置，使其恰好嵌入边框的透明区域中。

![](img/0e236769a21cd50c6da3c3d7c1f376b1_14.png)

**进阶：遮盖原模板文字**
如果边框模板上带有不需要的文字，可以：
1.  新增一个 **`白场`** 视频画中画。
2.  为其添加 **`矩形蒙版`**，调整蒙版大小以恰好覆盖原文字。
3.  将白色矩形放置于原文字上方进行遮盖。
4.  最后，使用 **`文本`** 工具在遮盖处添加自己的文字。

![](img/0e236769a21cd50c6da3c3d7c1f376b1_14.png)
![](img/0e236769a21cd50c6da3c3d7c1f376b1_16.png)
