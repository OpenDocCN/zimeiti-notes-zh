# 微信视频号教程：P28：短视频变声与配音的两种方法 🎬

在本节课中，我们将学习在混剪视频时，如何对视频的声音进行去重处理。我们将介绍两种实用方法：一种是在剪映App内直接进行变声处理；另一种是使用配音神器提取文案并生成全新的配音。这两种方法都能有效帮助你的视频通过平台审核。

## 概述

进行视频混剪时，直接使用原视频的声音可能导致内容重复，影响推荐。对声音进行去重处理是必要步骤。本节将详细讲解两种实现声音去重的操作流程。

![](img/f32091fa3b707f8f621113b0dd95fa03_1.png)

## 方法一：在剪映App内直接变声

上一节我们介绍了视频混剪的基本概念，本节中我们来看看第一种声音去重方法——使用剪映自带的变声功能。此方法操作简单快捷，能直接改变原视频的人声音色。

操作步骤如下：

1.  将需要处理的视频导入剪映。
2.  选中视频轨道，在编辑工具栏中向左滑动，找到并点击 **“声音效果”** 选项。
3.  在变声效果列表中，选择你喜欢的效果，例如“圣诞老人”、“台湾小哥”或“男生”等。软件提供多种免费选项，足够日常使用。
4.  部分变声效果支持参数微调，你可以根据预览效果进行调节。

通过以上步骤，原视频的人声会发生改变，从而实现声音去重。但需要注意的是，这种方法**只改变了人声，视频原有的背景音乐保持不变**。

## 方法二：使用配音神器生成全新配音

如果希望更彻底地改变音频，包括替换背景音乐，我们可以采用第二种方法。即先提取原视频文案，修改后使用配音软件生成全新的语音，最后替换原视频音频。

![](img/f32091fa3b707f8f621113b0dd95fa03_3.png)

以下是具体操作流程：

1.  **提取文案**：打开配音神器App，导入目标视频，使用其文案提取功能获取视频中的文字内容。
2.  **修改文案**：对提取出的文案进行适当修改。这是增强去重效果的关键一步。
    *   **修改原则**：保持原意，但可以替换部分词语或调整句式。例如，将“老人”改为“大人”，将“怎么办呢”改为“咋办呢”。
    *   **注意事项**：尽量避免增加或删减过多字数，微调即可。
3.  **生成新配音**：将修改后的文案复制到配音神器的制作界面。
    *   选择一个你喜欢的声音角色（如“男生”）。
    *   点击 **“合成配音”** 并导出音频文件。
4.  **替换音频**：打开剪映，导入原视频。
    *   关闭视频的 **“原声”**。
    *   点击 **“添加音频”** -> **“提取音乐”**，导入刚刚生成的新配音文件。
    *   此时，视频的音频已完全被新配音替换。你还可以进一步添加新的背景音乐，并调整音量平衡。

![](img/f32091fa3b707f8f621113b0dd95fa03_5.png)

这种方法实现了人声和背景音乐的全面更新，去重效果更为显著。

![](img/f32091fa3b707f8f621113b0dd95fa03_7.png)

## 总结

本节课我们一起学习了为混剪视频进行声音去重的两种核心方法。
*   **剪映变声**：操作便捷，能快速改变人声音色，适合快速处理。
*   **配音神器重制**：步骤稍多，但能彻底替换音频，去重效果更佳，并允许你自定义背景音乐。

在实际操作中，你可以根据视频素材和效率需求灵活选择或组合使用这两种方法。掌握它们，能让你的混剪视频更具独特性，更好地抓住流量风口。
