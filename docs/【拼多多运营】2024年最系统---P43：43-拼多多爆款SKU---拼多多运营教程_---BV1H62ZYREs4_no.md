## 什么是SKU？

![](img/8ba552da9031f91e39023500d07f5a69_4.png)

![](img/8ba552da9031f91e39023500d07f5a69_5.png)

SKU（Stock Keeping Unit）即库存量单位，指产品的具体规格选项。例如，一件衣服有“红色-小码”、“蓝色-中码”等不同组合，每一个组合就是一个独立的SKU。

![](img/8ba552da9031f91e39023500d07f5a69_6.png)

## SKU布局的核心作用

![](img/8ba552da9031f91e39023500d07f5a69_8.png)

一个优秀的SKU布局能为店铺带来多重益处。

以下是SKU布局的四个主要作用：

1.  **提升产品权重与曝光**：SKU的名称和属性会被系统收录。即使产品标题中未包含某些关键词（如特定颜色），只要SKU中设置了，该产品依然能被相应关键词搜索到，从而增加曝光机会。
2.  **灵活应对市场变化**：通过拆分和组合产品属性，可以创造出新的SKU。例如，将不同颜色的上衣与裤子进行组合，形成新的套装SKU。这能创造出更多具有价格竞争力的选项，提升商品的市场适应能力。
3.  **优化用户购物体验**：清晰、合理的SKU设置能帮助消费者快速找到所需规格，减少决策过程中的困惑，从而提高购物效率。
4.  **引导消费者达成购买**：通过策略性的SKU布局，可以突出展示你希望主推或利润更高的规格，有效引导消费者选择，促进转化并获取更佳利润。

## 爆款SKU布局策略

理解了SKU的重要性后，我们来看两种实操性极强的布局策略。

### 策略一：价值定价法

价值定价法是基于消费者对产品价值的感知来设定价格，而非单纯依据成本。其核心是认识到不同消费者有不同的价格敏感度和支付意愿，并通过提供不同规格的产品来满足各层次需求。

以下是价值定价法的操作逻辑，我们以手机为例进行说明：

*   **基础款（锚点）**：设置一个配置适中、价格具有吸引力的SKU。例如：`12G+256G` 版本，定价 **3999元**。此款作为价格“锚点”，让消费者觉得产品“物有所值”。
*   **主推款（性价比款）**：设置一个配置更高、价格增幅合理的SKU。例如：`16G+512G` 版本，定价 **4299元**。与基础款相比，消费者只需多付300元就能获得显著升级，感知“性价比极高”，此款常作为利润和销量的主力。
*   **形象款（高价款）**：设置一个顶配SKU。例如：`16G+1T` 版本，定价 **4699元**。此款并非为了大量销售，而是与基础款共同“衬托”主推款的超高性价比，促使消费者快速决策购买中间档位。

**公式总结**：`基础款（建立价值锚点）` + `主推款（凸显性价比）` + `形象款（拉高价值感知）` = 引导消费者选择目标SKU。

### 策略二：价格指引法

价格指引法是通过营销手段（如促销标签、限时优惠）影响消费者的价格感知，加速其购买决策。

我们继续使用手机的例子，但加入营销元素：

*   **常规价格体系**：
    *   `12G+256G`: 3999元
    *   `16G+512G`: 4499元
    *   `16G+1T`: 4899元
*   **施加价格指引**：对其中部分SKU添加营销标识。
    *   `16G+512G`（新品惠）: **4299元**
    *   `16G+1T`（限时折扣）: **4699元**

当消费者浏览时，会注意到带有“新品惠”、“限时折扣”标签的SKU价格比常规价更低，从而产生“机不可失”的紧迫感和划算心理，极大促进了对这两个SKU的购买转化。

**代码式描述**：
```plaintext
if SKU.标签 in [“新品惠”, “限时折扣”]:
    显示价格 = 原价 - 优惠金额
    消费者感知价值 = 高
    购买转化概率 = 提升
```

## 总结与后续

本节课中我们一起学习了爆款SKU布局的核心价值和两种关键策略：**价值定价法**与**价格指引法**。有效的SKU布局不仅能提升搜索曝光，更能从心理和营销层面引导消费者，最终实现转化率和客单价的双重增长。
