# 剪映手机版教学：P23：2.【手机剪辑】穿越手机效果

在本节课中，我们将学习如何使用剪映手机版制作一个“手机穿越”的创意视频效果。这个效果能让视频画面仿佛从手机屏幕中穿越而出，非常适合用于制作吸引眼球的短视频内容。

## 概述

我们将通过导入素材、使用画中画、色度抠图以及添加特效和滤镜等一系列操作，来完成这个效果的制作。整个过程步骤清晰，即使是剪辑新手也能轻松跟上。

## 详细步骤

以下是制作“手机穿越”效果的具体操作流程。

**第一步：导入与裁剪素材**

打开剪映应用，首先导入一段你准备好的主体视频。接着，点击“画中画”功能，导入我们提供的“手机穿越”素材。你可能需要对这个素材进行裁剪，以匹配你的视频尺寸和构图需求。

**第二步：进行色度抠图**

选中画中画轨道上的手机穿越素材，在编辑菜单中选择“色度抠图”功能。将取色器移动到素材的绿色背景上并点击，以选取需要抠除的颜色。然后，将“强度”和“阴影”两个参数都调整到100，确保绿色背景被完全、干净地移除。

**第三步：添加音频与定位**

点击“音频”选项，为你的视频导入一段背景音乐。将时间轴上的指针拖动到音乐的高潮部分，我们将在这里设置视觉特效，让画面效果与音乐节奏同步。

**第四步：添加视觉特效**

在指针定位好的时间点，点击“特效”按钮。从特效库中选择“积粉”特效并应用。接着，点击“滤镜”按钮，为视频选择一个名为“激光”的滤镜，以增强画面的视觉冲击力。

## 效果预览

完成以上所有步骤后，点击播放按钮，即可预览最终的“手机穿越”视频效果。

## 总结

本节课我们一起学习了“手机穿越”效果的制作方法。我们掌握了如何利用**画中画**导入素材，使用**色度抠图**功能移除背景，并通过添加**特效**和**滤镜**来提升视频的观赏性。希望你能运用这个技巧，创作出更多有趣的视频作品。
