# AI工具应用：P17：02.2：工具篇-AI生成PPT 🚀

在本节课中，我们将学习如何使用一款强大的AI工具来快速生成PPT。这款工具能根据你提供的主题或现有文档，自动梳理大纲并生成详细的演示文稿内容，极大地提升工作效率。

## 工具简介与模式

上一节我们介绍了新媒体运营的各种工具，本节中我们来看看这款专为PPT设计服务的AI工具。该工具主要提供两种使用模式。

以下是两种核心的使用模式：
1.  **主题生成模式**：输入一个主题，工具会结合类似ChatGPT的生成式AI能力，自动创作出完整的PPT全文。
2.  **文档生成模式**：上传一篇现有的文档（如TXT、PDF），AI助手会自动总结文档大纲，并基于此大纲生成详细的PPT内容。

## 编辑与优化生成结果

生成PPT只是第一步，我们还可以对其进行美化和调整，使其更符合个人需求。

工具的核心优势在于它能基于大纲的每个小节点，自动填充详细内容。这与许多其他AI PPT工具不同，那些工具通常需要用户提供全部完整内容。

在编辑界面，我们可以进行多项调整。

![](img/97bddefed6cb04a1d538caa9b61426bb_13.png)

首先，可以更换PPT的设计模板。虽然当前模板库的选择可能有限，但切换非常迅速。

![](img/97bddefed6cb04a1d538caa9b61426bb_15.png)

![](img/97bddefed6cb04a1d538caa9b61426bb_15.png)

其次，在右侧的编辑面板中，我们可以对文字、形状等元素进行细致的修改和替换。
