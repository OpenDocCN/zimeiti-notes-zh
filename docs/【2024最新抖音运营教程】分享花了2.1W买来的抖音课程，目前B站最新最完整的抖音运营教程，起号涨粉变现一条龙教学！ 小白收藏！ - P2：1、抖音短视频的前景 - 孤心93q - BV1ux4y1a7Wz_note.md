# 抖音运营教程：P2：1、抖音短视频的前景 📈

![](img/52e535b9fc976f7e95e215594b1ff0b2_1.png)

在本节课中，我们将要学习抖音短视频平台的发展前景，分析其市场潜力与用户生态，并探讨为何它适合新手创作者起步。

要了解抖音短视频的前景，首先需要通过一张图片来认识。

![](img/52e535b9fc976f7e95e215594b1ff0b2_3.png)

![](img/52e535b9fc976f7e95e215594b1ff0b2_1.png)

这张图片是抖音创作者大会的一份数据报告。报告显示，有超过2200万人在抖音的合计收入超过了417亿元。抖音希望将这个数字翻一番，让创作者的总收入达到800亿元。

![](img/52e535b9fc976f7e95e215594b1ff0b2_3.png)

这体现了平台巨大的流量扶持力度。同时，这里介绍第一个知识点：抖音用户主要分为两类。第一类是**玩家**，他们单纯地使用抖音娱乐，没有考虑通过平台赚钱。第二类是**创作者**，也就是本课程的学员，你们的目标是自己制作内容来获取收益。这个概念稍后还会提到。

那么，为什么我们要选择抖音短视频呢？市面上有快手、火山等多个短视频平台。这就像开实体店选址，你一定会选择流量大、位置好的地方。做短视频同样需要选择一个流量庞大的平台。如果平台自身流量不足，就无法分配给我们，我们也难以赚到钱。

抖音的日活跃用户超过8亿，月活跃用户更是达到了40亿人次。这是一个极其庞大的流量池。

流量大是否意味着竞争压力也大呢？并非如此。抖音的创作者数量约为5000万。这里的“创作者”定义广泛，并非特指网红达人。例如，邻居家的老太太用手机拍摄自己做饭的视频上传到抖音，她也算创作者。因此，你90%的竞争压力，实际上来自于像邻里乡亲这样的普通用户。

接着，做抖音短视频需要购买电脑或摄像机吗？答案是不需要。你只需要一部智能手机和一个手机号，就可以注册抖音账号。抖音倡导全民通过短视频分享和记录生活，门槛很低。

那么，当前有哪些值得关注的风口呢？首先是“抖音计划”。该计划表示，抖音将帮助1000名优质的艺术创作者实现粉丝增长超过100万。

有的同学可能会说：“老师，我不会创作。” 这不用担心。如果你什么都会，就不需要来听课了。正是因为大家是新手，才需要学习。老师曾指导过许多学员，包括年纪较大的长辈。他们秉持的态度是：虚心请教比自己厉害的人并不丢人。只要你在某个我不懂的领域比我有本事，而我又想学习，你就是我的老师。虚心请教能让人少走很多弯路。

因此，不以年龄定高低，不以贫富论资历。老师希望学员都是谦虚好学的。如果你认为自己已经全会了，那么这门课可能不适合你。

市场庞大、竞争环境相对宽松、内容创作门槛低，这些都是抖音的优势。我们现在处于大数据时代。在这个时代，系统会根据你的喜好推荐内容。你喜欢看帅哥，刷到的就是有腹肌的帅哥；你喜欢看美女，刷到的就是长腿美女。这导致我们每个人看到的内容都比较片面。

以下是不同人群可以创作的内容方向举例：
*   **宝妈**：可以拍摄宝宝的生活日常或自己做饭的过程。
*   **实体店主**：可以拍摄自己的产品，用于引流。
*   **普通上班族**：可以分享自己的生活点滴。

面对每月40亿的活跃用户，只要你创作的内容不触犯法律和道德底线，不是低俗内容，总会有一部分人喜欢上你。一定会有人喜欢你的。

那为什么你觉得自己的作品只有几十个粉丝呢？这是因为喜欢你内容的那部分人，还没有刷到过你的作品。他们无法看到，自然无法成为你的粉丝。

所以，对于新手小白来说，大家都迫切想涨粉的心情可以理解。但说句实在话，不要把“涨粉”作为第一个目标。我们的第一个目标应该是：**先把流量和播放量做起来**。当你的内容被更多人看到时，粉丝自然就会增长。

本节课中，我们一起学习了抖音平台的巨大流量前景、用户分类（玩家与创作者）、低门槛的创作条件以及新手起步的核心策略。记住，在起步阶段，核心是提升内容曝光（播放量），粉丝增长会随之而来。
