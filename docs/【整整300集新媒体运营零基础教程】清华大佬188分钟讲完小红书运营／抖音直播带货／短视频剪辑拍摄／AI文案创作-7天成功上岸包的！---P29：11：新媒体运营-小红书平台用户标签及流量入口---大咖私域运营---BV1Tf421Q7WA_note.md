# 新媒体运营零基础教程：P29：小红书平台用户标签及流量入口 📊

在本节课中，我们将深入探讨小红书平台如何为用户打上标签，以及笔记是如何通过不同的流量入口被分发给用户的。理解这些底层逻辑，将帮助我们更好地运营账号，让内容精准触达目标受众。

上一节我们介绍了小红书如何判定内容标签，本节中我们来看看平台如何判定用户标签。理解这个过程，可以帮助我们分析和了解自己的笔记是如何被系统推荐给用户的。我们可以从背后的逻辑反推，如何让笔记被我们的精准用户看到。

## 用户标签判定逻辑 🔍

![](img/b15eec95e547e70a10d66e28b29a2714_11.png)

![](img/b15eec95e547e70a10d66e28b29a2714_13.png)

小红书系统为用户打标签，主要依据是用户的**浏览行为**。用户刚注册时选择的兴趣范围，参考价值不大，它只是平台做的初步内容方向引导和用户筛选，还无法进入系统最深层的用户标签池。

### 1. 点击率
**点击率**是指系统在发现页给用户推送多个内容（如明星、美妆、家居、母婴、装修等）后，用户点击哪一类内容更多。系统通过算法分析用户的点击偏好。

![](img/b15eec95e547e70a10d66e28b29a2714_25.png)

![](img/b15eec95e547e70a10d66e28b29a2714_26.png)

![](img/b15eec95e547e70a10d66e28b29a2714_28.png)

![](img/b15eec95e547e70a10d66e28b29a2714_30.png)

![](img/b15eec95e547e70a10d66e28b29a2714_32.png)

例如，用户点击10个内容，其中美妆类3次，家居类2次，母婴类1次或0次。系统会判定该用户对母婴内容不感兴趣（可能不是宝妈），并对美妆类内容感兴趣。随后，系统会持续推送更多美妆领域内容进行试探。

![](img/b15eec95e547e70a10d66e28b29a2714_34.png)

![](img/b15eec95e547e70a10d66e28b29a2714_35.png)

通过点击率的逻辑，系统能为用户打上**强关联**和**弱关联**的标签。

![](img/b15eec95e547e70a10d66e28b29a2714_37.png)

![](img/b15eec95e547e70a10d66e28b29a2714_38.png)

![](img/b15eec95e547e70a10d66e28b29a2714_39.png)

![](img/b15eec95e547e70a10d66e28b29a2714_40.png)

为了帮助理解，我们可以想象一个“标签云图”。假设一个用户的标签云图中，“美妆”这个词最大，其他如“母婴”、“带娃”、“装修”等词较小，就代表该用户是美妆领域的重度爱好者。系统后续会主要为其推送美妆相关内容，减少或不推送其不感兴趣的内容，从而使用户获得满足感和足够的停留时长。

![](img/b15eec95e547e70a10d66e28b29a2714_41.png)

![](img/b15eec95e547e70a10d66e28b29a2714_42.png)

![](img/b15eec95e547e70a10d66e28b29a2714_43.png)

![](img/b15eec95e547e70a10d66e28b29a2714_44.png)

![](img/b15eec95e547e70a10d66e28b29a2714_45.png)

![](img/b15eec95e547e70a10d66e28b29a2714_46.png)

![](img/b15eec95e547e70a10d66e28b29a2714_47.png)

这就是系统给用户打标签希望达成的目的：连接**平台**、**用户**和**博主**。作为博主，我们需要平衡自己发布的内容和账号，思考其能否为用户和平台提供价值。平台也需要为用户提供价值，三者形成串联关系。

![](img/b15eec95e547e70a10d66e28b29a2714_49.png)

### 2. 停留时长
**停留时长**是另一个重要的用户行为分析维度。

![](img/b15eec95e547e70a10d66e28b29a2714_50.png)

![](img/b15eec95e547e70a10d66e28b29a2714_51.png)

![](img/b15eec95e547e70a10d66e28b29a2714_52.png)

![](img/b15eec95e547e70a10d66e28b29a2714_53.png)

![](img/b15eec95e547e70a10d66e28b29a2714_54.png)

![](img/b15eec95e547e70a10d66e28b29a2714_55.png)

例如，用户喜欢美妆内容，但很多笔记点开就划走了，说明这些具体笔记不是他喜欢的。系统会持续推送美妆领域内其他细分内容（如上粉、护肤等）来观察用户行为。

![](img/b15eec95e547e70a10d66e28b29a2714_57.png)

![](img/b15eec95e547e70a10d66e28b29a2714_58.png)

![](img/b15eec95e547e70a10d66e28b29a2714_59.png)

![](img/b15eec95e547e70a10d66e28b29a2714_60.png)

反之，如果用户今天点开一个平常很少点击的家居类内容，并且停留很久，仔细翻看图片和评论，系统就可能判定该用户有装修需求或近期想购买相关商品（如盘子），从而考虑为其推送更多相关笔记或商品。

### 3. 页面浏览量 (PV)
**PV** 可以理解为用户浏览页面的深度，这也有一定的分值。

![](img/b15eec95e547e70a10d66e28b29a2714_62.png)

![](img/b15eec95e547e70a10d66e28b29a2714_63.png)

![](img/b15eec95e547e70a10d66e28b29a2714_64.png)

![](img/b15eec95e547e70a10d66e28b29a2714_65.png)

![](img/b15eec95e547e70a10d66e28b29a2714_66.png)

![](img/b15eec95e547e70a10d66e28b29a2714_67.png)

![](img/b15eec95e547e70a10d66e28b29a2714_68.png)

例如，用户刷到一篇装修笔记，点进去后不仅看了这篇笔记，还进入了博主主页，并浏览了该博主的第二、第三、第四篇笔记。这就是用户浏览深度的体现。

![](img/b15eec95e547e70a10d66e28b29a2714_69.png)

![](img/b15eec95e547e70a10d66e28b29a2714_70.png)

如果你的账号主页内容非常精致，用户进来后就有可能提升在你页面的浏览数。这个数值对账号健康度有很大帮助。浏览深度足够时，用户就可能产生**关注**行为。关注是一个永久提升账号权重的行为，粉丝越多，账号质量越高。即使在粉丝基数不大时，提升主页浏览深度也能为账号带来非常健康的评分。

![](img/b15eec95e547e70a10d66e28b29a2714_72.png)

![](img/b15eec95e547e70a10d66e28b29a2714_73.png)

![](img/b15eec95e547e70a10d66e28b29a2714_74.png)

![](img/b15eec95e547e70a10d66e28b29a2714_75.png)

### 4. 互动行为
互动行为包括**点赞、收藏、评论、关注**。这些数据都是我们在做每一篇笔记时需要努力提升的。

![](img/b15eec95e547e70a10d66e28b29a2714_76.png)

![](img/b15eec95e547e70a10d66e28b29a2714_77.png)

![](img/b15eec95e547e70a10d66e28b29a2714_78.png)

![](img/b15eec95e547e70a10d66e28b29a2714_79.png)

![](img/b15eec95e547e70a10d66e28b29a2714_80.png)

![](img/b15eec95e547e70a10d66e28b29a2714_81.png)

*   点赞、收藏数据高，笔记获得更多推荐流量的几率更高。
*   数据好的笔记可能带动账号内其他笔记的流量攀升。
*   这些互动行为能直接提升粉丝数，并为后续接单、变现打下基础。

因此，提升互动行为是我们运营小红书一直努力的目标。互动行为能够提升笔记进入下一层推荐池的流量值。

### 1. 发现页
发现页是流量分发的核心入口，主要包含以下几类内容：
*   **关注达人内容**：你关注过的博主发布的内容。
*   **兴趣匹配内容**：系统根据你的点击率、浏览行为等用户标签，判定你可能喜欢的内容。
*   **发散性内容**：系统也会推荐一些你可能不感兴趣或没刷到过的内容，以增加平台的多样性和探索性。

![](img/b15eec95e547e70a10d66e28b29a2714_97.png)

![](img/b15eec95e547e70a10d66e28b29a2714_98.png)

![](img/b15eec95e547e70a10d66e28b29a2714_102.png)

平台需要满足用户多样化的需求（如看攻略、教程、趣事分享），因此发现页的内容不会过于单一。对于博主而言，后期能获得的主要流量也来源于发现页。

### 2. 关注页
关注页的内容非常简单，就是你**所有关注博主发布的内容**。虽然很多用户可能直接去刷发现页或从关注列表里找人，但关注页仍然是粉丝查看其喜爱博主动态的直接入口。

![](img/b15eec95e547e70a10d66e28b29a2714_113.png)

### 3. 附近页
附近页的功能类似于早期微信的“附近的人”。它会根据用户的手机定位，推送**附近及同城相关的内容**。

![](img/b15eec95e547e70a10d66e28b29a2714_119.png)

![](img/b15eec95e547e70a10d66e28b29a2714_120.png)

![](img/b15eec95e547e70a10d66e28b29a2714_121.png)

这个板块主要以美食、当地旅游等强地域特征的需求内容为主。如果你是做线下实体店推广（如美容院、美食店、景点），那么在运营和投放信息流广告时，就需要特别注意对附近人群的推送。

### 4. 购物页
购物页是小红书向电商方向发展的重要入口，主要匹配用户的购物需求。

![](img/b15eec95e547e70a10d66e28b29a2714_130.png)

![](img/b15eec95e547e70a10d66e28b29a2714_132.png)

目前，部分用户在小红书上还未形成强烈的购物习惯，可能更倾向于在传统电商平台或抖音购物。小红书上的购物行为，目前更多与直播带货相关，直接进入商城搜索购买的比例相对较低。许多用户仍习惯通过搜索笔记来“种草”。不过，笔记中广告出现频率过高也是当前用户吐槽较多的问题。

本节课中，我们一起学习了小红书平台判定用户标签的核心逻辑与四大流量分发入口。

![](img/b15eec95e547e70a10d66e28b29a2714_145.png)

*   **用户标签**：系统主要依据用户的**点击率、停留时长、页面浏览深度(PV)和互动行为（点赞、收藏等）** 来为用户打上兴趣标签，从而实现内容的精准推荐。
*   **流量入口**：笔记主要通过**发现页、关注页、附近页和购物页**四个入口分发给用户。其中，**发现页**是流量的主要来源，而**附近页**对于本地生活类账号尤为重要。
