## 账号类型解析

上一节我们介绍了课程背景，本节中我们来看看小红书平台上的账号类型。小红书账号主要分为三类，理解它们是制定推广策略的基础。

以下是三种核心账号类型及其特点：

1.  **企业号**：主要作用是树立官方品牌形象。企业号拥有官方认证标志，可以展示品牌名称、地址等信息，并享有官方特权，便于开展业务。
2.  **个人号**：这是目前大多数用户运营的账号类型。个人号的特点是**灵活性高**，可以是品牌员工或消费者身份，更容易塑造个人品牌特色，也适合品牌相关人物出镜。
3.  **引流号**：这类账号在之前的课程中被称为“炮灰号”。引流号**数量可以多个**，作用通常是辅助推广，用完即可舍弃，或在紧急情况下作为备用小号使用。这类账号通常没有精细的个性规划。

## 企业号矩阵运营建议

了解了账号分类后，我们来看看如何有效利用它们。一般的品牌推广主要通过企业号、个人号或引流号发布笔记进行引流。我们今天的核心建议是：构建并运营好你的企业号矩阵。

![](img/dabc40377fdf058a494fffb9f5c55426_1.png)

以下是打造企业号矩阵的四点具体建议：

![](img/dabc40377fdf058a494fffb9f5c55426_3.png)

![](img/dabc40377fdf058a494fffb9f5c55426_5.png)

1.  **积极参与品牌号活动**：例如，如果你是婚纱、服装或美妆类企业号，可以组织试用、团购等活动，并利用多个小号积极参与，增加活动热度。
2.  **个人号展示品牌属性**：在个人号的笔记内容中，应多次暗示品牌的定位。例如，品牌目标用户是城市年轻人还是职场新人，需要在内容中清晰体现。
3.  **控制个人号营销比例**：个人号发布品牌产品内容的频率不宜过高。一个较好的比例是 **`营销内容 : 种草/科普内容 = 3 : 7`**。即30%用于营销，70%用于提供价值的种草或科普。
4.  **利用引流号进行评论引流**：企业号在发布内容后，可以安排引流号在评论区进行互动或引导，实现更自然的流量转化。

## 官方广告投放介绍

![](img/dabc40377fdf058a494fffb9f5c55426_9.png)

![](img/dabc40377fdf058a494fffb9f5c55426_11.png)

除了运营自有账号矩阵，另一种高效的推广方式是借助平台力量。接下来，我们看看小红书官方的广告投放系统。

![](img/dabc40377fdf058a494fffb9f5c55426_13.png)

小红书官方广告是一种竞价广告，需要在拥有**企业号**资质后才能申请投放。申请流程可在小红书官网的“业务合作” -> “推广合作”板块中找到。

官方广告主要分为两大类：

1.  **信息流广告**：当用户浏览小红书首页信息时，系统会不定期间隔插入广告内容。
2.  **搜索广告**：当用户使用关键词搜索时（例如搜索“帽子”），系统会根据算法匹配，在搜索结果中展示相关的广告内容（例如推荐搭配的“大衣”）。

其中，搜索广告是典型的**关键词竞价**模式，价高者可以获得更靠前的广告位。目前信息流广告位常见于搜索结果页的第3、第13、第23等位置。

![](img/dabc40377fdf058a494fffb9f5c55426_15.png)

关于广告投放的更多常见问题与详细规则，可以在小红书广告平台的官方帮助文档中查询。需要注意的是，官方广告投放需要一定的资质审核费用，且后续竞价成本可能较高。在资金允许的情况下，与官方合作投放广告能直接获得平台推荐的精准流量。
