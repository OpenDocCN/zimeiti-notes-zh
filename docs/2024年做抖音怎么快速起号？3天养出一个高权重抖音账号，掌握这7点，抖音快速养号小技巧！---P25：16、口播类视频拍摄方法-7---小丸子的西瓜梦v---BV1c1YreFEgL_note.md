# 抖音运营教程：P25：口播类视频拍摄方法

## 概述
在本节课中，我们将学习口播类视频的具体拍摄方法。掌握正确的拍摄技巧，可以有效提升视频质量，帮助你吸引目标观众，实现快速涨粉。

## 拍摄的核心要点
拍摄环节需要注意以下几点：**构图比例**、**主题突出**、**事物清晰**、**光线色彩**和**内容连贯**。这些要点是制作优质视频的基础。

---

## 视频类型分类
拍摄方法主要分为两大类：**口播类型**和**日常生活记录类型**。这两种方法几乎涵盖了所有视频的拍摄方式。上一节我们介绍了拍摄的核心要点，本节中我们来看看第一种类型——口播视频的具体拍摄方法。

---

## 口播视频拍摄详解
首先，我们以一位学员“种小草”为例进行分析。她是一位40岁带三个娃的宝妈，从事情感口播，曾创下单条视频播放量破8000万、涨粉30万的成绩。

### 成功的关键：精准吸引目标群体
她的成功在于视频内容能精准吸引**针对性群体**。例如，如果你的目标观众是20多岁的学生，那么所有内容都应围绕学生感兴趣的话题展开，这样涨粉会更快、更精准。

以下是口播视频拍摄的具体步骤和注意事项：

![](img/2900ac11884333203f135cb77481318a_3.png)

### 1. 构图与画幅比例
抖音是**9:16**的竖屏平台。拍摄时，你需要决定采用横屏还是竖屏构图。案例中的视频采用了**4:3**的横屏构图。

**关键点**：保持画幅统一。如果你选择横屏，那么所有视频都应采用横屏；如果选择竖屏，则全部使用竖屏。保持稳定统一的风格很重要。

### 2. 人物位置与画面留白
人物主体应占据画面中央约**70%** 的面积。避免将脸部怼满屏幕拍摄。

**正确做法**：确保画面左、右、上方留有适当的空白区域。这样能使画面看起来更清晰、舒适。

### 3. 出镜形象与设备
*   **关于外貌**：做口播不要求外貌特别出众。口播的核心是**传达内容与知识**。只要出镜形象清晰、体面即可。
*   **使用美颜**：如果对自己的外貌不自信，可以开启美颜滤镜。适当使用美颜能让形象更佳，但注意不要过度夸张。
*   **关于背景**：口播可以在任何地点拍摄，室内或户外均可。关键是确保背景整洁、美观，避免杂乱难看。
*   **关于收声**：口播视频要求声音清晰。建议购买一个**手机收声麦克风**（价格便宜，约十几元），这能显著提升音频质量。

![](img/2900ac11884333203f135cb77481318a_5.png)

### 4. 内容与台词
*   **台词记忆**：如果担心记不住台词，无需死记硬背。可以在剪辑环节，采用“边读边录”的方式，后期再进行剪辑处理。
*   **内容共鸣**：视频内容需要引发目标群体的共鸣。以“种小草”的视频为例，其主题“与婆婆同住的感受”精准地吸引了**已婚女性群体**，因为她们对婆媳关系有最深切的体会。

## 总结
本节课我们一起学习了口播类视频的拍摄方法。核心在于：**选择统一的画幅比例**、**合理安排人物构图与留白**、**运用设备提升音画质量**，以及**制作能引发特定目标群体共鸣的内容**。掌握这些技巧，你也能拍出吸引人的优质口播视频。
