# 新媒体运营：P16：02.1：工具篇-全平台短视频文案一键提取 📝

在本节课中，我们将学习一个非常实用的运营工具，它能帮助我们从抖音、小红书、哔哩哔哩等多个平台的短视频中，一键提取出文案内容。掌握这个工具，可以极大地提升我们收集素材和创作内容的效率。

## 工具介绍与优势

上一节我们介绍了文案提取的需求，本节中我们来看看这个工具的具体优势。传统提取文案的方法通常比较繁琐，例如使用“轻斗”小程序，或者先将视频下载下来，再导入“剪映”等剪辑软件进行识别。这些方法往往只适用于单一平台。

相比之下，本节课介绍的工具具备显著优势：它支持**在线操作**，无需下载视频。你只需将目标短视频的链接复制到工具网页中，即可一键提取出文案。该工具支持包括抖音、快手、小红书、哔哩哔哩在内的多个主流平台。

以下是该工具的核心操作流程：
1.  复制目标短视频的分享链接。
2.  将链接粘贴到工具指定的输入框中。
3.  点击“提取文案”按钮。
4.  稍等片刻，即可获得提取出的纯文本文案。

## 扩展应用：提取哔哩哔哩视频文案

接下来，我们测试该工具在哔哩哔哩平台上的效果，以验证其跨平台能力。

我们准备一个B站视频，复制其链接（通常以 `BV` 号开头）。同样地，将链接粘贴到工具输入框中，点击提取。

![](img/288b301d0eab7b5845c2a6788bd9101e_5.png)

提取完成后，工具准确地输出了B站视频的文案，例如：“新手做直播电商，怎么才能打造出直播爆品？在直播销售的不同阶段……”。这证明了该工具在不同平台上的文案识别准确性都很高。
