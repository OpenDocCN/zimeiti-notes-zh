# 微信视频号运营教程：P21：7.账号定位

在本节课中，我们将要学习账号定位阶段的两个核心问题：如何处理旧作品以及如何解决原创视频素材来源。我们将提供清晰、直接的解决方案，帮助你顺利迈出视频号运营的第一步。

## 旧作品处理与发布节奏

上一节我们介绍了账号定位的重要性，本节中我们来看看定位明确后，如何处理账号的既有内容。

首先，需要将账号之前发布的作品隐藏。这是为了确保账号内容风格统一，不影响新的定位。

![](img/a7400d19c3a27896d992dcad1d1a80c0_1.png)

其次，发布新内容时需注意节奏。素材无需着急准备过多，保持一天发布一条视频的频率即可。

## 分步解决核心问题

![](img/a7400d19c3a27896d992dcad1d1a80c0_3.png)
