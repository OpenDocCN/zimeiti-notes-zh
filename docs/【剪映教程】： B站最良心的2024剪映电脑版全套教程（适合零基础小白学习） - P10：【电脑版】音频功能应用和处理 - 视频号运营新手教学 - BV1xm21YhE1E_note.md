# 剪映PC教程：P10：音频功能应用与卡点视频制作

![](img/b5aba4c2047f5be135da53489a202dcd_0.png)

在本节课中，我们将学习剪映PC版音频功能模块的具体应用，并通过一个完整的卡点视频案例，掌握如何利用这些功能高效地制作出富有节奏感的短视频。

## 概述：音频功能模块详解

上一节我们介绍了剪映的基础界面，本节中我们来看看音频功能的具体应用。选中音频轨道后，下方会出现五个功能模块，它们构成了剪映强大的音频处理体系。

### 音乐素材库

![](img/b5aba4c2047f5be135da53489a202dcd_2.png)

第一个模块是音乐素材库。它内置了丰富的音乐资源，无需用户自行下载。音乐库进行了细致的分类，方便用户快速查找。

![](img/b5aba4c2047f5be135da53489a202dcd_4.png)

以下是主要的音乐分类：
*   **抖音热门**：包含当前流行的热门音乐。
*   **浪漫/旅行/Vlog**：针对不同视频主题（如浪漫氛围、旅行记录、日常Vlog）精选的音乐。
*   **卡点音乐**：专门为制作节奏感强的卡点视频准备的音乐集合。
*   **纯音乐/环保/搞笑/混剪**：覆盖了多种风格和用途的音乐类型。

使用音乐时，只需点击下载，然后将其拖拽到时间线即可。

### 音效库

![](img/b5aba4c2047f5be135da53489a202dcd_12.png)

![](img/b5aba4c2047f5be135da53489a202dcd_13.png)

![](img/b5aba4c2047f5be135da53489a202dcd_14.png)

第二个模块是音效库。音效能为视频画面增添生动感和趣味性，是视频包装的重要元素。

音效库同样有详细的分类：
*   **笑声/综艺**：常用于营造轻松、搞笑的氛围。
*   **机械/打斗**：适用于科技感或动作类视频。
*   **人声转场/BGM**：包含常用的过渡音效和背景音。
*   **动物叫声**：如小猫、布谷鸟等动物的声音效果。

### 音频提取

![](img/b5aba4c2047f5be135da53489a202dcd_16.png)

![](img/b5aba4c2047f5be135da53489a202dcd_17.png)

第三个功能是音频提取。此功能可以快速从已有视频文件中分离出音频轨道。

操作方法是：导入一个视频文件，软件会自动将其音频提取为一个独立的素材，用户可以将其拖到时间线上使用。代码逻辑可以理解为 `extract_audio(video_file)`。

### 抖音收藏

第四个模块是抖音收藏。此功能需要用户登录自己的抖音账号。

![](img/b5aba4c2047f5be135da53489a202dcd_19.png)

登录后，你在抖音APP中收藏的音乐会自动同步到剪映PC版的“抖音收藏”列表中，方便在制作视频时直接调用。

### 链接下载

![](img/b5aba4c2047f5be135da53489a202dcd_21.png)

第五个功能是链接下载。如果你在抖音上发现了喜欢的音乐但无法收藏，可以复制该视频的分享链接。

将链接粘贴到指定输入框，剪映会自动解析并下载其中的音乐，存入音乐库供你使用。

## 实战：制作卡点视频案例

了解了音频功能后，我们将运用这些知识，结合其他工具，制作一个完整的卡点视频。

### 第一步：准备素材与音乐

![](img/b5aba4c2047f5be135da53489a202dcd_23.png)

首先，将案例所需的所有图片素材导入媒体池。接着，导入背景音乐。我们可以使用“音频提取”功能，从一个已完成的作品视频中提取音乐，并拖拽到时间线上。

### 第二步：使用自动踩点功能

卡点视频的核心是让画面切换与音乐鼓点同步。剪映的“自动踩点”功能可以辅助我们快速定位鼓点。

![](img/b5aba4c2047f5be135da53489a202dcd_25.png)

操作步骤如下：
1.  选中时间线上的音频素材。
2.  在工具栏找到 **`自动踩点`** 按钮（图标类似小红旗）。
3.  点击后选择节拍模式（如“节拍2”），软件会自动在音频波形上生成标记点。

这些标记点就是软件识别的鼓点位置，为后续的画面剪辑提供了精准的参考。

### 第三步：制作开场动画

1.  **添加文字模板**：在“文本”->“文字模板”中，选择一个合适的模板（如“怦然心动”），拖到时间线开头，并调整其长度与第一个音乐段落匹配。修改文字内容为“You know what I think?”。
2.  **添加图片与构图**：导入第一张图片素材，在“画面”设置中将比例裁剪为 **`3:4`**。接着，将整个项目的画幅比例也设置为 **`3:4`**。
3.  **应用滤镜**：为图片添加“复古胶片”分类下的“美高”滤镜。
4.  **使用蒙版制作分屏效果**：
    *   为图片添加一个“矩形”蒙版。
    *   复制多层图片，并分别调整每层蒙版的位置和大小，营造出多层错落有致的分屏效果。
    *   让每一层图片在不同的鼓点标记处依次出现。
5.  **组合文字与画面**：将开场文字层置于画面中央，调整其出现时间，使其与分屏动画的节奏配合。

![](img/b5aba4c2047f5be135da53489a202dcd_27.png)

### 第四步：拼接节奏画面

![](img/b5aba4c2047f5be135da53489a202dcd_29.png)

1.  导入多张图片素材，依次拖到时间线上。
2.  将每张图片的时长调整为一个或两个鼓点间隔。
3.  为所有图片统一应用“美高”滤镜，并确保每张图片的画幅比例均为 **`3:4`**。
4.  根据音乐节奏，排列图片顺序，形成快速的画面切换。

![](img/b5aba4c2047f5be135da53489a202dcd_31.png)

![](img/b5aba4c2047f5be135da53489a202dcd_33.png)

### 第五步：制作彩色分屏转场

![](img/b5aba4c2047f5be135da53489a202dcd_35.png)

1.  导入新的人物图片，裁剪为 **`3:4`**。
2.  使用“矩形”蒙版，制作竖向排列的多个分屏。
3.  为每个分屏图层添加“负片偏移”特效。
4.  选中特效，在参数面板中点击调色板图标，为每个分屏更改不同的颜色。
5.  在对应的分屏内添加“YOU”、“ARE”、“DOPE”等文字。

### 第六步：制作特效结尾画面

1.  导入人物特写图片。
2.  复制一层，对上层图片使用“矩形”蒙版，框出人物主体。
3.  为上层图片添加“负片偏移”和“黑白胶片”特效，制作出人物彩色、背景黑白的对比效果。
4.  后续画面可采用类似蒙版方法，制作上下分屏、不同色调的效果。

### 第七步：添加结尾字幕

1.  在视频结尾部分，使用文本工具添加“LIKE REALLY DOPE”字样。
2.  可以复制文字层，调整大小、位置和颜色，制作错落有致的文字排版效果。
3.  为文字添加合适的“花字”样式或描边，使其更突出。

![](img/b5aba4c2047f5be135da53489a202dcd_37.png)

### 第八步：导出视频

![](img/b5aba4c2047f5be135da53489a202dcd_39.png)

所有效果制作完成后，即可导出视频。

![](img/b5aba4c2047f5be135da53489a202dcd_41.png)

1.  点击右上角 **`导出`** 按钮。
2.  设置导出参数：
    *   **标题**：自定义文件名称。
    *   **导出至**：选择保存路径。
    *   **分辨率**：选择 **`1920x1080`** 或更高。
    *   **码率**：选择“自定义”和“VBR”模式。
    *   **编码**：H.264。
    *   **格式**：MP4。
    *   **帧率**：30fps。
3.  点击 **`导出`** 按钮，等待渲染完成。
