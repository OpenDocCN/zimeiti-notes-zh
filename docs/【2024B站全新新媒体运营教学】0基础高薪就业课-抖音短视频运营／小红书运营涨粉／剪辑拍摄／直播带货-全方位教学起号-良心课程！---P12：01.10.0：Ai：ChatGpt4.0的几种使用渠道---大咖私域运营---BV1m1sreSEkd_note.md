# 新媒体运营AI工具指南：P12：01.10.0：ChatGPT 4.0的几种使用渠道 🤖

## 概述
在本节课中，我们将学习如何获取和使用当前最强大的AI对话模型——ChatGPT 4.0。对于新媒体运营人员而言，掌握AI工具可以极大地提升工作效率，例如撰写脚本、制作PPT和生成报告。我们将介绍几种体验和使用ChatGPT 4.0的渠道，包括免费和付费选项，并分析各自的优缺点。

## 微软Edge浏览器：Copilot 🌐
上一节我们概述了AI对运营工作的重要性，本节中我们首先来看看一个集成在操作系统中的便捷工具。

![](img/e261688b85e0e991ecce6b3355396cb0_1.png)

![](img/e261688b85e0e991ecce6b3355396cb0_3.png)

微软基于ChatGPT 4.0模型，在Edge浏览器中内置了名为“Copilot”的AI助手功能。它在一定程度上具备了4.0模型的智能水平，适合日常办公使用。

使用Copilot有两个前提条件：
1.  需要登录微软账号（可使用Outlook邮箱注册）。
2.  需要能够访问外部网络的工具（通常称为“梯子”）。

Copilot提供三种对话模式：
*   **更有创造力**：AI的回答更具开放性和想象力。
*   **平衡**：介于创造力和精确性之间，适合一般用途。
*   **更精确**：回答更为严谨和准确，适合专业内容需求。

例如，我们可以使用“更有创造力”模式，输入指令：`请帮我写一个关于装修的短视频脚本，时长控制在3分钟左右。` AI便会生成相应的脚本文案。

此外，Copilot的优势在于可以分析你当前浏览器打开的网页内容，提供相关帮助。

**然而，Copilot也存在一些缺点：**
1.  **精准度稍逊**：相较于官方ChatGPT 4.0，经过微软调校的Copilot在某些回答上可能不够精准。
2.  **连续对话能力受限**：在当前对话中持续提问多次后，可能需要开启新对话窗口。
3.  **记忆功能有所缺失**：在利用历史对话优化后续回答方面能力较弱，且缺乏插件功能。

对于大多数运营工作，Copilot的功能已足够使用。如果你的Edge浏览器没有Copilot图标，可以尝试下载最新的Edge Dev（开发者）测试版本，该版本通常包含此功能。

## 第三方平台与共享账号 🔗
上一节我们介绍了系统集成的解决方案，本节中我们来看看一些由开发者搭建的第三方平台，它们提供了访问ChatGPT的另一种途径。

以下是几个可用的第三方平台示例：

**1. 灵感（示例平台A）**
该平台集成了多种AI模型，包括GPT-3.5、GPT-4.0以及国内的一些大模型。
*   **优点**：模型多样；新用户可在一定时间内免费使用GPT-4.0；无需特殊网络工具。
*   **缺点**：响应速度可能较慢。
*   **费用**：提供免费额度，后续有包月等付费选项。

**2. Vee（示例平台B）**
该平台提供GPT-3.5和GPT-4.0模型。
*   **优点**：响应速度快；提供初始免费使用额度。
*   **缺点**：免费额度通常仅适用于GPT-3.5；需要特殊网络工具。
*   **费用**：按Token（使用量）计费，提供不同的套餐。

**3. ChatGPT共享账号平台**
这是一个共享付费ChatGPT账号的网站。平台上列出了许多他人分享的账号，状态由颜色标识：
*   **绿色**：账号空闲。
*   **橙色**：账号近期被使用过。
*   **红色**：账号可能正在被使用或使用频繁。
*   **金色**：通常是可用的GPT-4.0付费账号。

![](img/e261688b85e0e991ecce6b3355396cb0_5.png)

![](img/e261688b85e0e991ecce6b3355396cb0_7.png)

通过这类平台，你可以临时使用共享的GPT-4.0账号，但稳定性和可用性无法保证，且同样需要特殊网络工具。

**1. 百度文心一言**
百度的AI对话模型，提供基础版和付费的“文心一言4.0”版本。在撰写文案、获取灵感等基础运营任务上可以满足需求。

**2. 字节跳动豆包AI**
字节跳动推出的AI助手，同样在脚本生成、内容建议等方面表现不错。

**3. 其他模型**
如阿里的通义千问、昆仑万维的天工大模型等，也都是可选的国内AI工具。
