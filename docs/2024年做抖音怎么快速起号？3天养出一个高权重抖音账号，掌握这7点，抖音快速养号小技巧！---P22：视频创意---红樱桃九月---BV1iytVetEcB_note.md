## 视频创意的来源

![](img/0b6e4464d251f506d9ebb2ad7b14510e_9.png)

视频创意的核心是确定选题。你可以使用上节课提到的九宫格选题法等方法来构思，但首先需要确定一个大的主题方向，例如是讲述励志故事还是产品故事。

当你缺乏灵感或希望蹭热点时，可以借助一些创意工具网站。以下是寻找创意的具体方法。

## 关于二次创作与搬运的边界

在利用现有素材进行创作时，一个关键问题是：二次创作到什么程度才不算搬运？

![](img/0b6e4464d251f506d9ebb2ad7b14510e_23.png)

实际上，平台对于“搬运”的判定并没有一个绝对统一的技术标准，它通常结合了机器审核和人工审核。因此，判定存在一定的主观性和不确定性。

为了避免被判定为搬运，我们在处理素材时应尽量做到以下几点：

![](img/0b6e4464d251f506d9ebb2ad7b14510e_25.png)

1.  **保留核心，增加原创**：保留原素材的核心信息或精彩片段，但加入你自己的观点、解说或新的叙事角度。
2.  **进行深度加工**：对素材进行剪辑、调速、添加滤镜、背景音乐和新的字幕。
3.  **改变视频形式**：可以尝试对视频进行镜像翻转、画幅调整等操作。
4.  **彻底清除第三方标识**：务必去除素材中原本带有的其他平台水印、电视台台标、广告商标等任何可能被视为营销推广的信息。

**重要提示**：随着平台规则日益严格，直接简单处理现有视频的风险越来越高。因此，更推荐将以素材为基础的“二次创作”作为灵感来源和内容补充，而非主要的内容生产方式。
