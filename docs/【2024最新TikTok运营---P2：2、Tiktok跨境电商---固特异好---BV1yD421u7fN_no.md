## 核心差异一：货品选择 🛒

“货”是未做过跨境电商的朋友极易踩入的雷区。在确定售卖什么货品之前，必须首先了解平台的禁售规则和物流限制。

以下是选品前必须完成的两个关键步骤：

![](img/76f7b849389e27477a5f431e8779b230_1.png)

1.  **熟读平台规则**：明确哪些商品是平台禁止售卖的。避免辛苦选品后，发现商品根本无法运出海外。
2.  **研究物流规则**：搞清楚哪些商品属于禁运品，哪些商品的运费成本极高。同时，必须规划好退货处理方案。例如，服装类商品通常无法承担退回国内的运费，因此需要将预估的退货率计入成本。

**成本计算公式示例**：
`总成本 = 商品成本 + 物流成本 + (退货率 * 商品单价)`

## 如何科学选品 🔍

选品策略直接影响直播成败。如果盲目跟风售卖当前爆品，很容易陷入困境。因为每个产品都有其生命周期：发展期、增长期、平稳期、衰退期。

我的建议是，尽量避免选择正处于猛烈增长期或平稳期的爆品。你可以在TikTok联盟后台的“热销板块”中，向后翻阅二三十页，从中筛选那些上升趋势较快的产品。这类产品已被部分用户验证了需求，且竞争相对较小。

当然，并非完全不能利用爆品热度。关键在于不能售卖一模一样的产品，而是要进行创新。

以下是两种有效的“蹭热度”方法：

1.  **重组SKU（库存量单位）**：通过产品组合提升竞争力与客单价。
    *   **示例**：当螺蛳粉在国内热卖时，单纯比拼口味难以胜出。有商家将螺蛳粉与煮锅、碗具捆绑销售，提高了产品吸引力。
    *   **优势**：跨境物流成本固定，售卖两件组合产品的利润通常高于售卖一件单品。利润往往隐藏在巧妙的SKU设计中。
    *   **代码/公式表示**：
        ```python
        # 假设单品利润
        profit_single = item_price - item_cost - shipping_cost
        # 组合产品利润（假设组合节省了部分包装或物流成本）
        profit_bundle = bundle_price - (item_cost * 2) - shipping_cost_bundle
        # 目标：profit_bundle > profit_single * 2
        ```

## 直播团队的人员配置 👥

在不同发展阶段，直播团队的配置需要动态调整。以下是基于直播间在线人数的配置建议：

![](img/76f7b849389e27477a5f431e8779b230_7.png)

*   **新手期/在线人数 < 50人**：配置1名主播和1名运营。
*   **成长期/在线人数 50~500人**：增加1名场控，形成“1主播+1场控+1运营”的配置。
*   **成熟期/在线人数 > 500人**：可以扩展至“4主播+2场控+1运营”的配置。实际上，采用“2主播+1场控”的配置，已足以高效滚动运作两个直播间。

对于主播的选择，有两个优先考虑因素：
1.  能够使用英语进行流利交流。
2.  对目标市场（如英国）的文化有一定了解。具备这些条件的主播会更具优势。
