# 抖音运营入门：P20：新手如何做好内容定位 📝

在本节课中，我们将学习一个高效的内容选题公式，帮助新手解决“不知道发什么”的难题，从而更精准地吸引目标用户，为账号快速打上标签。

## 选题的重要性

![](img/ee8b62cbb862016fa622c71f3422b68f_1.png)

不会选题是许多新手创作者的共同困扰。这直接导致内容产出困难，更新频率无法保证。

会找选题的创作者，能将更多时间投入视频制作。通过提高发布频率（例如每天三条），仅靠数量积累就能形成竞争优势，甚至无需依赖付费推广。更重要的是，善于选题的人能更精准地捕捉用户痛点，吸引精准粉丝观看。

这有助于账号快速获得平台推荐，打上垂直标签。那么，如何解决选题难题呢？

## 核心选题公式

![](img/ee8b62cbb862016fa622c71f3422b68f_3.png)

当然有解决方法。这是一套百万粉丝博主常用的选题技巧。为了方便理解，我们将其总结为一个公式：

**内容定位 + 用户痛点 + 借势热点**

用好这个公式，你的内容方向就不会跑偏。

![](img/ee8b62cbb862016fa622c71f3422b68f_5.png)

## 公式应用详解

以下是公式的三个组成部分：

1.  **内容定位**：确定你的账号所属领域或行业。你需要挖掘该行业的核心关键词。例如，知识博主领域的关键词可能包括：播放量、粉丝量、定位、选题等。这些关键词应是客观的行业术语，而非主观描述。
2.  **用户痛点**：在你的领域内，寻找目标用户最关心、最困扰的问题。例如，在知识分享领域，用户痛点可能是：如何便捷高效地学习、有什么实用技巧、如何输出内容等。
3.  **借势热点**：关注并利用近期的社会热点、节日、流行话题或明星事件。

将以上三个部分的关键词进行任意排列组合，就能生成大量潜在的选题方向。

## 行业应用案例

上一节我们介绍了公式的构成，本节中我们来看看如何在不同行业里应用它。

**案例一：知识博主**
*   **内容定位**：知识分享、运营技巧。
*   **用户痛点**：便捷、高效、技巧教学、内容输出。
*   **借势热点**：结合平台新规则、热门讨论话题。
*   **组合选题示例**：“利用平台新算法，3个技巧让播放量翻倍”、“高效内容输出法，日更3条不头疼”。

![](img/ee8b62cbb862016fa622c71f3422b68f_7.png)

**案例二：美妆行业**
*   **内容定位**：美白、除皱、口红。
*   **用户痛点**：平价好用、明星同款、避坑扫雷。
*   **借势热点**：明星妆容、特定节日（如七夕）。
*   **组合选题示例**：“迪丽热巴同款美白面膜推荐”、“七夕千万别送女朋友这3支口红（避雷指南）”。
