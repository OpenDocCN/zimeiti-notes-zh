# 拼多多运营实操教程：P40：五步玩转场景推广（一）

在本节课中，我们将要学习拼多多场景推广的基础知识，包括其核心概念、定向人群、广告展示位置以及后台设置入口。这是系统掌握场景推广玩法的第一步。

上一节我们介绍了拼多多运营的基础知识，本节中我们来看看一个重要的推广工具——场景推广。很多新手商家刚开始接触拼多多，对场景推广的玩法不太清楚。本节将总结场景推广的核心操作经验。

## 认识场景推广

首先，我们需要了解场景推广是什么。场景推广的核心在于“溢价”和“曝光渠道”，即我们如何出价以及广告会展示在哪些位置。

### 基础出价与人群定向

在场景推广中，如果只进行普通的出价，这种方式称为 **通投**。但更精细的操作需要对特定人群进行溢价。以下是可供选择的五类定向人群：

1.  **访客重定向**：面向曾访问过您店铺或商品的用户。
2.  **相似商品定向**：面向对与您商品相似的产品感兴趣的用户。
3.  **叶子类目定向**：面向对您商品所在最细分类目感兴趣的用户。
4.  **相似店铺定向**：面向关注与您店铺类似的店铺的用户。
5.  **兴趣点定向**：面向具有特定购物兴趣标签的用户。

在店铺后台，您可以按以下路径找到并设置这些定向：
`店铺后台 -> 推广工具 -> 推广计划 -> 场景推广 -> 新建计划 -> 选择“商品推广”及“自定义”模式 -> 添加商品后设置人群溢价。`

### 资源位选择

除了人群，我们还需要选择广告希望展示的具体位置，即资源位。主要有三大可溢价资源位：

![](img/74e9454f68423dfd089bba14fbf90225_6.png)

1.  **商品类目页**：当用户浏览商品分类列表时出现的广告位。
2.  **商品详情页**：在其它商品详情页面底部或侧边的推荐位。
3.  **营销活动页**：如多多果园、边逛边赚等拼多多平台活动页面内的广告位。

![](img/74e9454f68423dfd089bba14fbf90225_8.png)

在后台设置中，您可以在创建计划时的“资源位”部分看到并进行选择。

![](img/74e9454f68423dfd089bba14fbf90225_6.png)
![](img/74e9454f68423dfd089bba14fbf90225_8.png)

### 基础流量包

此外，系统提供一个 **基础流量包**。这是一个覆盖全网优质流量的智能推广包。操作建议是直接开启它，作为流量的基础保障。

以上便是操作场景推广时需要了解的溢价设置与曝光渠道。

## 广告展示位置详解

接下来，我们具体分析场景推广的广告会展示在哪些页面。主要有以下五个方面：

1.  **商品类目页**：广告以“1+6N”的形式穿插在商品列表中（即1个广告位后跟随6个自然位，循环出现）。
2.  **营销活动页**：例如“多多果园”、“边逛边赚”等活动页面。
3.  **拼多多公众号及推文**：在拼多多官方微信公众号及推送的文章中展示。
4.  **商品详情页推荐**：在其它商品的详情页底部，以“为你推荐”等形式出现，排名多为“3+6N”形式。也包括“精选推广”等位置。
5.  **现金签到页**：在拼多多APP的现金签到页面，前50个广告位通常属于场景推广。
