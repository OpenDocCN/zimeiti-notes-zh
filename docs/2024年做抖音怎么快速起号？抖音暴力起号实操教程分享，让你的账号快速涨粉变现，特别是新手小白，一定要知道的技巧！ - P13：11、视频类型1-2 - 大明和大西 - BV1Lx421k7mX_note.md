# 抖音运营实战：P13：11、视频类型1-2

在本节课中，我们将要学习抖音账号运营中一个至关重要的环节：**人设的建立**。我们将深入探讨强人设与弱人设的区别，并详细介绍一种核心的视频内容形式——**口播**。掌握这些概念，将帮助你明确账号方向，创作出更具吸引力的内容。

上一节我们解决了文案问题，本节中我们来看看内容创作的下一个关键环节。从涨粉到变现并非一条直线，中间还有一个核心环节：**建立人设**。

人设的目的并非为了炫耀，而是为了在众多同类账号中脱颖而出。例如，有十个人都做装修内容，如果其中一位（比如小王同学）建立了幽默搞笑的人设，观众就更容易记住他，从而忽略其他九位。当用户因为喜欢你这个人而关注你时，他就成为了你的粉丝。这就是人设的价值。

我们来学习这个知识点。人设主要分为两种：**强人设**与**弱人设**。区分清楚后，你就能快速判断任何视频的人设类型。

首先，**强人设**是指以创作者**个人**为切入点的内容。它必须与创作者自身密切相关。

以下是强人设常见的切入点：

*   **个人职业与经验**：分享所在行业的专业知识与经验。例如：装修、办公、房产、汽车、餐饮、婚介、法律咨询、电商等领域。
*   **个人兴趣与生活**：分享个人的兴趣爱好与日常生活。例如：美食、旅行、手工、宠物、星座占卜、美妆、护肤、服装穿搭、游戏、摄影等。

强人设的核心是分享与自己相关的工作、生活和爱好，并且**大部分需要真人出镜**。

与之相对的是**弱人设**。弱人设内容与创作者本人关系不大，核心目的是运营好账号本身。

以下是弱人设的例子：

*   **情感语录账号**：发布配乐的情感文字图片或视频，无真人出镜或配音。
*   **影视解说账号**：虽然有人声解说电影，但焦点是电影内容，而非解说者本人。

区分强、弱人设是选择内容方向的基础。接下来，我们看看如何将“与自己相关”的内容转化为视频。

内容创作环节中，任何与你个人相关的话题都可以作为视频素材。我们首先讲解第一种广泛适用的形式：**讲故事**。

讲故事可以分享客户故事、从业经历或人生感悟。故事本身不要求完全真实，可以从文案平台寻找有趣的故事进行分享。而讲故事最直接的方式，就是通过**口播**。

口播是一种对着镜头讲述内容的视频形式。它拥有丰富的分类，适用于多个领域。

以下是常见的口播类型：

*   **故事型口播**：讲述各类故事，目的是让观众觉得有趣、有共鸣。
*   **知识型口播**：分享实用知识，例如育儿、金融、养生、装修、房产、汽车等领域的科普。目的是让观众在几十秒内获得“收获感”，觉得学到了东西。

无论你身处哪个领域，只要你有知识或故事可分享，都可以尝试口播。它的拍摄门槛较低，只需准备好文案，用手机架好就能录制。

针对常见的顾虑，这里做出解答：

*   **口才不好能做口播吗？**  
    **可以**。口播是依据提前准备好的稿子进行录制，并非临场发挥。很多口播者的流畅表达都源于充分准备。
*   **普通话不好能做口播吗？**  
    **可以**。如果你的目标客户是同城或有地域属性，使用方言可能效果更佳，能增加亲切感和趣味性。同时，可以通过添加字幕来确保信息传递。

口播是一种非常高效且易于上手的内容形式，值得重点学习和实践。在后续课程中，我们会详细教授口播视频的具体拍摄方法。

本节课中我们一起学习了人设的建立，明确了强人设与弱人设的区别，并深入了解了**口播**这种核心视频形式的特点、价值与拍摄要点。理解并应用这些知识，能为你的抖音账号打下坚实的内容基础。
