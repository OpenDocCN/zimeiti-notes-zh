# 小红书运营：P29：28、小红书笔记引流：平台流量结构 📊

![](img/88c720e2af6b882e8fafc92823ab938a_1.png)

在本节课中，我们将要学习小红书平台的流量结构。了解流量从哪里来，以及不同流量渠道的特点，是制定有效运营策略的基础。我们将重点分析发现页、关注页和搜索页这三个核心流量来源，并探讨它们对笔记引流效果的影响。

上一节我们介绍了笔记制作的相关内容，本节中我们来看看笔记发布后，平台是如何分配流量的。

## 什么是流量结构

流量结构指的是小红书平台整体的流量分布和来源。理解流量结构，能帮助我们明确什么样的笔记，投放到什么样的渠道，能获得什么样的效果。简单来说，就是了解平台的“交通地图”，知道用户从哪里来，我们该到哪里去吸引他们。

小红书App的整体流量结构主要分为三个核心页面。

以下是三个核心流量页面：

1.  **发现页**：App打开后的第一个主页面。
2.  **关注页**：显示用户已关注账号内容的页面。
3.  **搜索页**：通过放大镜图标进入的搜索页面。

此外，App中还存在一个“本地”页面，它会根据用户地理位置显示内容。这个页面主要用于特定场景，例如寻找本地美食、探店或同城服务。对于大多数希望将商品卖向全国的商家而言，**应谨慎使用本地页面进行引流**，因为它会打乱账号的标签，导致内容被推荐给非目标人群（例如，卖食品的笔记可能被推送给只看本地美女视频的用户）。

因此，我们笔记引流需要聚焦的**核心流量结构就是三个点：发现页、关注页和搜索页**。

## 发现页：最主要的流量来源

发现页是整个小红书**最重要**的流量来源，其流量约占平台总流量的 **70%**。它也是平台进行流量层级测试的主阵地。一篇笔记最终能获得多大规模的曝光，很大程度上取决于它在发现页获得的用户反馈（如点赞、收藏、评论）。

![](img/88c720e2af6b882e8fafc92823ab938a_3.png)

平台会根据你的账号权重，为每篇新笔记提供一个初始曝光量（例如1000或2000）。这**初始曝光量的100%都会投放到发现页**。笔记后续能否进入更大流量池，完全取决于在发现页的初始表现。因此，运营好发现页是小红书起号的关键。

## 搜索页：转化率最高的流量来源

搜索页的流量占比约为 **20%**，是仅次于发现页的第二大流量来源，并且是小红书平台**增长最快**的流量板块。

搜索流量的用户意图非常明确，他们通常是带着具体的需求或问题来寻找解决方案或商品。因此，**搜索页的成交转化率极高，估计占小红书总成交量的70%以上**。

但是，搜索流量并非新手初期可以轻易获取的。它依赖于笔记中包含有效的**关键词**，以及商品或账号本身积累的**权重**（如历史成交、评价等）。这部分流量是我们在账号运营中后期需要重点攻克和维护的。

## 关注页：最精准但量小的流量来源

关注页的流量占比最小，大约只有 **10%**，甚至可能更低（约7%）。虽然量小，但这部分流量是**最精准**的。关注你的用户都是对你的内容真正感兴趣的人，投其所好进行引流或推广会相对容易。

然而，有一个关键点需要注意：来自关注页的流量**不计入笔记的基础权重计算**。也就是说，即使你有1万个粉丝，笔记发布后全部粉丝都来阅读，这篇笔记也未必能“爆”。平台判断笔记是否优质，主要依据它在**发现页**的公域流量池里的表现。关注页的阅读量更像是一种“私域”流量，对撬动平台更大的公域流量帮助有限。

## 运营策略与顺序

基于以上分析，我们可以得出清晰的运营顺序和策略重点：

1.  **前期主攻发现页**：集中精力创作优质内容，提升笔记在发现页的互动数据，从而获得平台更多的公域流量推荐。
2.  **中期布局搜索页**：在笔记中合理布局行业关键词，优化商品和账号权重，逐步从搜索流量中获取高意向用户。
3.  **后期维护关注页**：将公域流量转化为私域粉丝，通过关注页与核心用户保持互动，进行深度运营和转化。

这个顺序，就是我们打开小红书流量密码的钥匙。

本节课中我们一起学习了小红书平台的三大流量结构：**发现页（70%）、搜索页（20%）和关注页（<10%）**。我们明确了发现页是流量的起点和主战场，搜索页是高转化率的宝藏地，而关注页则是核心用户的沉淀池。理解并运用好这个结构，能为你的小红书运营打下坚实的基础。
