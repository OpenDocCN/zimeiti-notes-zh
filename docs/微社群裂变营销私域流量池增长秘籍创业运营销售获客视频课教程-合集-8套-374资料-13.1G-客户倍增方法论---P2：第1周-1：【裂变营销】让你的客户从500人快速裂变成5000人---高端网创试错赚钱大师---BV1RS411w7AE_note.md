在讲解这部分之前，我们需要先理解两种营销模型。这样才能明白为什么我们要做裂变式营销。
第一个观念是：没有裂变，等于坐吃山空。常见的营销模型有两种。
第一种是“关注 -> 兴趣 -> 渴望 -> 记忆 -> 购买”的漏斗式模型。它从很多人关注开始，逐渐向下递减，最终只有少数人完成购买。
第二种是“关注 -> 感兴趣 -> 购买 -> 分享”的沙漏型模型。

沙漏型模型的好处在于，它有一个关键的“分享”流程。如果你的产品或服务能让每个客户都为你发一条朋友圈，你还会愁客源吗？基于对这两种模型的观察，我们主张你应该更努力去打造一个沙漏型的营销模型。

在营销模型中，有一个非常关键的概念叫做ROI，即投入产出比。只有当产出大于投入时，你才是盈利的。在裂变营销中，我提出一个概念叫做“裂变比”。它指的是一个老客户或用户能带来多少个新用户。

假设你的商业模式中，裂变比大于1，会发生神奇的事情。例如，你的每个客户在一定的周期内（比如一个季度）平均会为你带来两个新客户，那么裂变比就是1:2。这意味着，如果你目前有1000个客户，三个月后，这1000个客户会为你带来2000个新客户，加上原有的1000个，总客户数就变成了3000个。新增的这2000个客户如果继续保持1:2的裂变比，客户数量就会持续裂变增长。

**公式：总客户数增长 = 初始客户数 × (1 + 裂变比)^周期数**

如果你的裂变比大于1，那么你的整个商业模式就非常容易实现倍增。这就是我们从两种营销模型中推导出的不同商业结果。以我们自己为例，线下课每新增100个学员，其中90%以上都是老学员转介绍的。老学员越多，为我们转介绍的人就越多，招生就越容易。这就是裂变的秘密，倍增的秘诀。

所以，我们的目标就是要打造一种裂变比大于1的营销模式。我们整个课程的设计，都将围绕如何突破并实现裂变比大于1这个目标，从各个关键节点和流程为你提供帮助。

接下来，我们看看裂变营销有哪些常见的误区。常见的误区有四个。
第一，狭义地理解裂变，以为裂变只是搞活动。
第二，认为裂变来的粉丝难以转化，只能吸引低价用户。
第三，裂变初期效果不理想，导致很多人很快放弃。
第四，误以为裂变就是分销或拉人头。

在这里，我希望和大家建立一个共同的认识：裂变是什么？
我对裂变的定义有两个维度。
从狭义上讲，裂变是指通过任务宝、进群宝、分销宝等各种工具，在微信生态下进行的一次曝光活动。
从广义上讲，任何能实现“人传人”性质的营销策略、营销活动或商业模式，都可以称之为裂变。关键点在于“人传人”。

以前讲“人传人”可能很多人不太理解，但现在因为疫情，大家都明白“人传人”会带来多么可怕的传播速度。反过来想，如果能让你的产品和服务像病毒一样传播到全世界，这对商业来说是一种非常理想的状态。

![](img/c88e4a088a5621f8b2bc9fe6919716b9_14.png)

第三，裂变有两个大的前提。如果你的客户不具备这两个前提，再多的方法也没用。
第一个前提是：用户分享我们的内容给他的朋友，有助于他帮助身边的人。我提出一个概念，叫做“蚂蚁效应”。想象一只蚂蚁在路边看到一块糖果，它会立刻回到蚁窝告诉同伴，然后一起把糖果搬回来。这就是蚂蚁效应。

判断一个项目或产品能否做裂变，首先要看人群。可以从两个维度看。
第一，线下是否有聚堆的空间。即，使用产品的人是否在学校、办公室、家里、宿舍、聚会活动、商场等线下聚堆场景中使用产品。如果是在这种场景中使用，就有概率能够裂变。
第二，线上是否有聚堆的社群。比如家长群、妈妈群、学生群、白领群、家人亲戚群、闺蜜群、同学群、战友群等。这些都是能够帮助我们引发裂变的良好群体。

第二个前提是：有助于提升他的社交形象。简单来说，他把你的内容或产品发在朋友圈，能够提升他的品味，让他更有面子。
如果符合“正面且可以公开”这一条，裂变效果最好。
“正面但不好公开，可以私下讨论”的类型也可以裂变，但范围较小。
比较难的是“负面且私密”的类型，比如贷款、私密用品等，相对难以裂变。
“公开的负面”类型则更加难以裂变。

所以，正常来说，第二个前提是：目标用户分享我们的内容到朋友圈，有助于提升他的社交形象。简单说，就是用户发你的东西到朋友圈会不会提高他的逼格？如果不会，我们可能需要做一些特殊的包装来提升这种“逼格感”。

以上是关于裂变营销的第一部分内容。

接下来是第二部分：裂变的玩法。快速掌握让客户倍增的N种裂变新玩法。

![](img/c88e4a088a5621f8b2bc9fe6919716b9_12.png)

首先，我们把裂变分为两个阶段：成交前和成交后。
成交前，我们称之为“流量裂变”。它包括通过活动裂变、技术裂变、拼团裂变等方式。在用户成为我们的客户之前，我们统一称之为“流量”。
成交后，我们称之为“客户裂变”。它包括口碑传播、利益相关传播以及产品自带传播等。
简单概括就是：在裂变中成交，在成交中裂变，在成交后再裂变。这是我们进行裂变营销需要把握的几个关键环节。

那么，裂变具体有哪些玩法呢？这就是第二部分要讲的内容：裂变的N种玩法。找对传播引爆点，策划出N种裂变玩法。

![](img/c88e4a088a5621f8b2bc9fe6919716b9_14.png)

首先，你应该看过类似的海报。这些活动仅凭一张海报就能轻松裂变获取数万粉丝。
例如，我们做的“私域流量营销实战”送书活动，累计吸引了5万粉丝。
“12招掌握微信引流与成交”活动，累计吸引了4万粉丝。
我们的一个学员模仿送书活动，做了“1元试吃猕猴桃”活动，一次涨粉4000，最终有2000位伙伴进行试吃。
另一个做连锁培训和咨询的学员，模仿“12招掌握微信引流与成交”的裂变活动，一次活动涨粉5万。我们自己做一次大概涨粉五六千，但他的学员基数大（线下会议有1万多人），所以涨粉量达到了5万。这些都是掌握了裂变思维、活动策划和执行后产生的效果。

刚才我们把裂变分成了成交前的流量裂变和成交后的客户裂变。它们各自又有三种不同的形式和活动。下面我为大家一一拆解。这些要点在我们接下来的课程中会有更详细的讲解，现在先带大家浏览一遍，了解常见形式是什么样子。

首先，我们来看成交前的流量裂变。
**1. 活动裂变**
在朋友圈可能会看到这几种海报：
*   **任务宝裂变**：扫码后进入公众号，公众号提醒你转发海报并邀请朋友关注，以获得奖品。它依托于公众号。
*   **进群宝裂变**：扫码后进入一个微信群，群内有机器人提醒你转发并截图到群内审核。它依托于微信群。
*   **分销宝裂变**：扫码后进入一个H5网页，提示你购买，右上角有分享海报赚取奖励的入口。常见于朋友圈刷屏的分销活动。

我们第二周的课程会教大家如何设置这样的裂变活动。

**2. 技术裂变**
在朋友圈偶尔会看到刷屏级的证件照、名片生成等活动。这其实就是简单的H5技术实现的裂变活动，用户填写信息后生成个性化海报，满足了炫耀、好玩的心理，从而引发传播。例如生成结婚证、答案测试题（答题后需关注公众号看答案）、好友助力猜红包（朋友点击助力，积累到一定金额可提现）等。

![](img/c88e4a088a5621f8b2bc9fe6919716b9_18.png)

**3. 拼团裂变**
以拼多多为代表，做实物拼团效果最好。我们也模仿做了课程拼团，原价39元，发起拼团仅需9.9元，能触发客户一起拼团。还有内购拼团（分享给朋友获得内购价）、好友砍价（朋友砍价越多，价格越优惠）等形式。

以上三种就是我们常见的成交前流量裂变玩法：活动裂变、技术裂变和拼团裂变。每种裂变会触发不同的效果和人群，我们会在后续课程详细讲解。

接下来是第二个阶段：成交后的客户裂变。
**1. 口碑传播**
例如，我们线下课的朋友圈发布率高达80%，即每100人参课，超过80人会发朋友圈，传播效果非常好。另一个例子是“高危学堂”（科学创业培训机构），其96%的新学员来自朋友介绍。大众点评、小红书等平台也容易引发口碑传播。

**2. 利益相关**
我们按利益从低到高排序，分别是：代言人、分销员、合伙人、经销商、代理商、加盟商等。其特点是都与利益相关，利益越高，传播的动力就越强。这是我们常见的利益驱动型裂变模式。

![](img/c88e4a088a5621f8b2bc9fe6919716b9_16.png)

**3. 产品传染**
指产品本身的设计能促使用户主动分享。
*   **虚拟荣誉**：如得到APP，学习或做笔记多会获得设计精美的虚拟勋章，促使用户分享。
*   **有趣包装**：如“斑马精酿”啤酒，买满6瓶可将照片定制在瓶身上，收到后用户很可能发朋友圈。
*   **附属型会员卡**：办一张会员卡，可让身边多人共享同等权益和积分。
*   **精美物料**：我们线下课精心设计的笔记本和教材，是学员朋友圈发布率最高的物料。

所以，成交后的客户裂变有三种：口碑传播、利益相关和产品传染。

好的，第一部分我们讲了为什么要做裂变模式，客户倍增的秘诀在于裂变比，目标是打造裂变比大于1的营销模式。第二部分我们介绍了裂变的两个阶段：成交前的三种流量裂变玩法和成交后的三种客户裂变玩法。

现在，我们用一个具体案例来拆解，如何帮助一家企业玩转成交前和成交后两种裂变。

![](img/c88e4a088a5621f8b2bc9fe6919716b9_18.png)

这个品牌叫做“智工教育”，是专注广西公务员考试辅导的培训机构。在广西，每年新增的公务员中，每三个就有一个来自智工教育，它是广西区域内非常厉害的辅导品牌。

它是怎么玩转裂变的呢？首先看它的客户群是谁。考公务员的人一般是谁？常见的公务员考试辅导产品形式是什么？
考公务员通常有两个条件：学历条件和年龄条件。客户群中有相当一部分是大学生。但这个行业成交难的点在于，辅导服务普遍售价较高，在3-4万元左右。

我们做的第一件事是打造一个“杀手级”产品。针对广西大学生，我们推出了“980无限学”产品。
*   **含义**：只需先交980元，即可学习5年。考过之后再付尾款，没考过则无需付尾款。
*   **缓冲期**：考过后，还有6个月时间筹钱付尾款。
*   **奖学金激励**：越早通过，奖学金越高。例如第一年通过可获得1万元奖学金，实际尾款只需付19800元；第二年通过奖学金5500元；第三年以后无奖学金。
*   **转介绍激励**：同时设有转介绍奖励机制。
*   **产品包装**：980元约等于广西大学生两周生活费，考上公务员等于拿到“金饭碗”。因此打出口号：“用两周的生活费，换一辈子的金饭碗”。

设计好产品后，需要寻找目标客户群。这里引入一个客户分层图表，将客户由远到近分为4个层次：
1.  **潜在客户**：符合成交条件，但尚未产生需求。（例如：所有广西大学生）
2.  **目标客户**：已经产生需求，但尚未购买。（例如：已报考公务员考试的广西大学生）
3.  **购买客户**：已经成为客户，但尚未推荐他人。（例如：选择智工教育的广西大学生）
4.  **铁杆客户**：已经成为客户，且有推荐行为。

我们的目标是不断培养更多的铁杆客户。

那么，我们是如何一步步策划裂变活动的呢？
**针对潜在客户和目标客户（成交前裂变）：**
1.  **国庆抽奖活动**：针对广西大学生，设置三个奖品（广西周边双人游、报销回家往返车票、视频平台会员）。活动规则：转发海报到朋友圈即可获得抽奖机会。为精准筛选，中奖兑换需出示广西大学生学生证。活动效果很好，吸引了几千潜在客户。
2.  **请看电影活动**：国庆期间，结合热点，举办“请你看《我和我的祖国》”活动，形式同上，奖品换成电影票，兑换同样需验证学生证，进一步扩大传播并筛选精准人群。
3.  **针对警察学院专项活动**：警察学院学生是必须考公务员的精准目标群体。我们针对他们设计专属奖品，进行裂变活动，成功吸引该学院大量学生关注公众号并添加课程顾问。

**针对购买客户和铁杆客户（成交后裂变）：**
1.  **转介绍减免学费**：我们发现很多在政府单位工作的人是“在职不在编”，他们非常渴望考编。我们设计规则：已考上公务员的学员（需付尾款），如果介绍其“在职不在编”的同事来报名，可以获得相应的尾款减免。这极大地激发了已成功学员的转介绍动力。
2.  **“状元晚宴”口碑传播**：我们观察到，学员考上公务员对家庭来说是大事，如同古代中状元。因此，我们将普通的“庆功宴”升级为“状元晚宴”，邀请考上公务员的学员及其亲朋好友参加。学员的父母会非常乐意拍照发朋友圈分享喜悦，这形成了极佳的口碑传播效果。

通过将成交前后分成两个裂变阶段，并针对不同层级的客户（潜在、目标、购买、铁杆）设计不同的营销裂变活动，我们可以有效扩大裂变半径，同时实现更精准的营销。

希望这个案例对你有启发。我强烈建议你针对自己的业务，梳理出潜在客户、目标客户、购买客户和铁杆客户，进行分类。这将极大地有助于后续策划裂变活动。

精准是成交的前提。大部分潜在客户无法成交，都是因为不够精准。所以我们要做好客户分层，做对裂变营销。
作业提交有参考格式。你需要复制潜在客户、目标客户、购买客户、铁杆客户这几个分类，然后进行有效区分。
我们做这个区分的目的，是针对不同的客户层级来策划对应的裂变营销，这样才能有的放矢。
购买客户和铁杆客户比较容易区分。潜在客户和目标客户的区分可能需要多思考。潜在客户需要满足一些条件，例如：
*   **地域性**：如果你的产品有地域限制，潜在客户就是该地区的用户。
*   **身份角色**：是否有特定身份限制（如大学生、宝妈）。
*   **职业**：特定职业人群。
*   **场所**：他们会出现在哪里？（例如，厦门伴手礼品牌“苏小唐”的潜在客户主要是去厦门旅游的游客）
*   **身体状况/体型等**：如健身、医疗产品。
*   **间接客户**：使用者与购买者可能不是同一人。（如老板为员工报名，父母为孩子报名，孩子为父母购买产品）

第二，你的目标客户，即正在比较、准备购买的客户群有哪些，也请你做好梳理。
这个作业非常重要，因为我们后续的裂变营销策划都是围绕这些客户群展开的。如果一开始客户群没找对，后面的方向可能会走偏。希望大家认真思考并提交作业。我们的助教团队会给予反馈，帮助你对准目标客户。
