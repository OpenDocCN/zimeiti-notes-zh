# 拼多多运营实操教程：P37：拼多多开店-OCPX系列（一）

## 概述
在本节课中，我们将要学习拼多多场景推广中的一种重要工具——OCPX。我们将从基础定义开始，了解它是什么、在哪里找到它，以及它的核心运作原理和两个关键阶段。通过本课的学习，你将能对OCPX有一个清晰的认识。

## OCPX基础认识
上一节我们介绍了拼多多运营的整体框架，本节中我们来看看一个具体的推广工具：OCPX。OCPX本质上是拼多多场景推广中的一种出价方式。

当你在拼多多商家后台创建推广计划时，可以在“多多场景”的“新建计划”环节看到它。系统会提供两种推广方案供选择，其中一种就是OCPX出价。

## OCPX的核心特点
OCPX的付费方式与我们日常的推广有所不同。它按照点击进行计费，但出价逻辑围绕“预期成交”展开。

![](img/2b433b7aebddbca8d18b64a012d2688a_1.png)

![](img/2b433b7aebddbca8d18b64a012d2688a_2.png)

**核心付费逻辑**：你设定一个愿意为获取一个订单所支付的最高推广费用。例如，如果你愿意花10元获得一个订单，那么就将“预期成交出价”设置为10元。系统会基于这个价格和商品历史数据，自动优化并触达转化率更高的人群。这种方式能非常直接地帮助你控制预期的投入产出比。

## OCPX运作的两个阶段
OCPX计划的运作分为两个明确的阶段，每个阶段有不同的目标和操作重点。

以下是两个阶段的详细说明：

1.  **第一阶段：数据累积期**
    *   **目标**：快速积累初始数据，让系统学习你的商品和受众。
    *   **操作**：建议直接采用系统给出的“建议出价”。此阶段消耗可能较高，且投产比（ROI）可能不理想，但无需过分关注。重点是让流量跑起来，为进入第二阶段打下数据基础。
    *   **关键**：不要频繁调整出价，以免影响数据累积的稳定性。计划开启时间越长，累积的权重通常越高。

2.  **第二阶段：优化提升期**
    *   **目标**：基于第一阶段的数据，优化出价，提升投产比（ROI）。
    *   **操作**：当计划状态显示进入“第二周期”后，便可以开始逐步优化出价。你可以根据曝光、点击、转化等数据反馈，对出价进行微调，以提高投资回报率。
    *   **注意**：优化过程也应循序渐进，避免大幅频繁改动。

在后台的推广计划列表中，你可以清晰看到每个OCPX计划当前所处的阶段，从而决定是继续累积数据还是开始优化。

## 总结与后续
本节课中我们一起学习了OCPX的基础知识。我们了解到OCPX是拼多多场景推广中一种以目标转化为导向的智能出价工具，其运作分为**数据累积**和**优化提升**两个核心阶段。

如果经过一段时间的投放（例如约14天），发现OCPX的投产效果仍不理想，后期可以考虑切换回场景推广的“自定义”模式。在自定义模式下，可以重点投放“访客重定向”等精准人群，尝试拉高投产比。
