# TikTok运营教程：P1：使用AI技术制作图文视频

在本节课中，我们将学习如何利用AI工具，快速制作并发布一个符合TikTok平台特性的图文视频。整个过程将涵盖从寻找对标账号、生成图片素材、处理文案到最终视频合成的完整步骤。

## 寻找对标账号与内容分析

上一节我们介绍了课程目标，本节中我们来看看如何寻找并分析优秀的对标账号。

首先，在电脑上登录TikTok软件，浏览平台内容。针对当前流行的素材进行搜索，例如近期较火的“暗黑心理学”。在搜索栏中输入“暗黑心理学”进行搜索。

搜索时，应重点查看“账号”分类，而非“综合”或“视频”。目的是发现专注于该领域的账号。例如，一个拥有27万粉丝的英文账号是理想的对标对象，应优先选择英文账号，避免选择使用阿拉伯语等小语种的账号。

![](img/cc355a746659cfed2b7a9f078cd8345c_1.png)

点开目标账号，分析其视频内容。该账号的视频多采用图文结合的形式，部分置顶视频播放量超过百万。观察其近期视频，虽然并非每个都爆火，但整体视频质量尚可。例如，一个视频拥有2万多播放量，这属于可接受的范围。

这种图文视频的核心在于：**吸引人的画面**与**引发共鸣的文案**。如果视频播放量很低（例如仅5个播放），通常意味着其文案或内容吸引力不足。

![](img/cc355a746659cfed2b7a9f078cd8345c_3.png)

## 拆解与准备素材

![](img/cc355a746659cfed2b7a9f078cd8345c_5.png)

在分析了优秀案例后，我们需要拆解其制作方法并准备自己的素材。

![](img/cc355a746659cfed2b7a9f078cd8345c_7.png)

对标账号的每个图文视频都包含多张图片，每张图片上都有一个标题。这些标题的作用是吸引用户注意力，增加视频停留时间。

要制作类似的视频，我们需要同类风格的图片。这些图片明显是使用AI工具生成的。生成方式主要有两种：
1.  使用 **Midjourney** 这类专业AI绘画工具。
2.  使用 **ChatGPT** 的图片生成功能。

本教程将使用第二种更简便的方法。打开ChatGPT，输入生成图片的指令。指令需要明确描述图片主题和比例。

![](img/cc355a746659cfed2b7a9f078cd8345c_9.png)

**示例指令：**
```
请生成一张符合“暗黑心理学”主题的人物图片，人物看起来凶狠，难以接近。图片比例为9:16。
```

生成第一张图片后，将其下载。重复此过程，生成5到6张风格统一的图片。

![](img/cc355a746659cfed2b7a9f078cd8345c_11.png)

## 视频制作与文案处理

![](img/cc355a746659cfed2b7a9f078cd8345c_13.png)

准备好图片素材后，接下来进入视频编辑和文案处理阶段。

![](img/cc355a746659cfed2b7a9f078cd8345c_15.png)

打开剪映软件，导入所有生成的图片。首先在剪映中确认视频比例设置为 **9:16**，以适应TikTok的竖屏格式。

![](img/cc355a746659cfed2b7a9f078cd8345c_17.png)

接下来处理文案。回到TikTok对标账号的视频，将其中的英文文案提取出来。可以使用手机的截图翻译功能（如微信翻译）直接提取文字。

![](img/cc355a746659cfed2b7a9f078cd8345c_19.png)

获得英文原文后，使用谷歌翻译等工具将其翻译成中文，理解其含义。例如，一句文案可能翻译为：“尊重那些尊重你的人，不尊重那些不尊重你的人”。

然后，进行文案“转译”以规避重复：
1.  将中文翻译结果，用不同的表达方式重新组织。
2.  将重新组织后的中文，再次翻译回英文。
3.  确保最终得到的英文文案与原文意思相近，但表述不完全相同。

以下是处理文案的核心思路，可以用伪代码表示：
```
原英文文案 -> 翻译为中文 -> 用不同句式改写中文 -> 将改写后的中文翻译回英文 -> 得到新英文文案
```

![](img/cc355a746659cfed2b7a9f078cd8345c_21.png)

在剪映中，选择“添加文本”功能，将处理好的新英文文案输入。可以调整文字的字体、大小、颜色和位置。将文字图层的时长拖动至与对应图片的时长一致。为每一张图片重复此操作，添加相应的文案。

## 导出与发布

![](img/cc355a746659cfed2b7a9f078cd8345c_23.png)

视频内容编辑完成后，进行最后的导出与发布设置。

![](img/cc355a746659cfed2b7a9f078cd8345c_25.png)

在剪映中完成所有图文编辑后，进行导出。注意，**导出时选择“视频”格式，但不添加任何背景音乐**。音乐应在TikTok发布时，使用其内置的音乐库添加，这样更符合平台生态，也可能获得更多流量。

![](img/cc355a746659cfed2b7a9f078cd8345c_27.png)

![](img/cc355a746659cfed2b7a9f078cd8345c_29.png)

导出视频后，即可在TikTok上发布。发布时，从TikTok的音乐库中选择一个与视频氛围匹配的热门音乐。

![](img/cc355a746659cfed2b7a9f078cd8345c_31.png)

最后，补充两个常见问题：
1.  **为什么选择“暗黑心理学”？** 因为它是当前的热门话题，流量潜力大。你也可以寻找其他热门领域。
2.  **如何获取AI工具？** 如果需要ChatGPT或Midjourney等AI工具的使用资源，可以通过特定渠道获取。

本节课中我们一起学习了利用AI工具高效制作TikTok图文视频的全流程。关键步骤包括：寻找并分析英文对标账号、使用ChatGPT生成符合主题的9:16比例图片、通过“翻译-改写-再翻译”的方式处理文案以避免重复、使用剪映合成视频，最后在TikTok发布时添加平台音乐。掌握这个方法，你可以快速批量生产内容，尝试在热门领域起号。
