## 封面尺寸 📏

![](img/da2a243182c1f778f1992dd112ed785e_10.png)

封面尺寸是设计的基础。对于图文类内容，小红书官方推荐了三种尺寸比例：竖屏 **3:4**、正方形 **1:1** 和横屏 **4:3**。

![](img/da2a243182c1f778f1992dd112ed785e_12.png)

然而，在实际操作中，为了获得最佳的用户体验和平台推荐效果，**商品笔记的封面比例必须选择 3:4**。小红书是一个以竖屏浏览为主的娱乐放松平台，用户习惯上下滑动。如果使用横屏（4:3）比例，用户需要将手机横过来观看，这会极大降低他们的浏览兴趣，导致点击率下降。因此，我们制作小红书的所有图片比例都应统一为 **3:4**，其他比例不予考虑。

## 封面设计原则 ✨

以下是设计封面时需要遵循的核心原则：

1.  **图片清晰度高**：封面图片必须清晰。这意味着需要使用高清的手机或相机拍摄原创图片，避免使用模糊的截图或经过多次转存的低质量图片。清晰的图片如同高清电视画面，能带来更好的视觉体验，有助于提升笔记质量。
2.  **风格保持一致**：你所发布的所有笔记封面，其文案模板和图片风格应保持统一。这有助于小红书系统快速识别你的账号内容，进行更精准的内容分发和人群匹配。统一的风格也能减少系统审核和推荐的延迟。
3.  **重点突出**：封面需要有一个明确的视觉焦点，并能结合文案激发用户的点击欲望。无论是展示产品细节、人物形象还是使用效果，都要确保核心卖点一目了然。一个无法吸引点击的封面，后续内容再好也难以获得曝光。

## 封面类型 🖼️

封面可以根据内容和目标人群分为多种类型，选择合适的类型能事半功倍。以下是常见的几种封面类型及其适用场景：

*   **高颜值单图**：指画面精美、主体突出的单张图片，可有模特或无模特。适用于美妆教程、家居装修、穿搭分享、产品介绍等类目。
*   **图片拼贴**：将多张图片（如抠图）拼接组合在一起，并可在上面添加文字。适用于旅游攻略、美食探店等类目，适合网络博主使用。
*   **海报大片**：具有设计感，类似宣传海报，通常包含主题和副标题。适用于旅游、生活方式、知识学习类笔记。
*   **对比图**：通过使用前/后、传统/网红等对比形式来展示产品效果。非常适用于口红、美甲、饰品、服装等实用性商品。
*   **纯文字封面**：以精心设计的文字排版作为封面主体。适用于知识分享、干货教程类内容，通过强调标题来吸引目标读者。

请根据你选择的商品类目和目标人群，选择最合适的封面类型。

## 视频封面 🎬

对于商品笔记，使用视频封面通常比纯图文封面效果更好。视频封面制作相对简单，关键在于视频本身的质量。

视频封面无需额外添加复杂的文字或图案引导。你需要确保拍摄的**原创视频内容清晰、亮度足够、效果突出**。如果一次拍不好，可以多拍几遍选择最佳版本。视频封面特别适合产品介绍、穿搭展示等需要动态呈现的内容。

![](img/da2a243182c1f778f1992dd112ed785e_14.png)

视频的尺寸比例同样推荐使用竖屏 **3:4**。虽然平台支持其他比例，但为保持最佳浏览体验，应统一使用 **3:4** 的竖版视频。

## 注意事项 ⚠️

最后，在制作封面时还需注意以下几点：

1.  **确保图片尺寸适当**：避免使用过大或过度压缩的图片，以免影响加载速度和显示效果。
2.  **保持标题风格一致**：封面上的标题（无论是图片上的文案还是文字标题）其风格和位置应基于账号内容定位，并保持统一。不要频繁变换样式。
3.  **满足像素要求**：封面图的推荐像素尺寸为 **800 x 1200**，像素密度（DPI）建议在 **150-300** 之间，以保证清晰度。
