# 抖音运营实战：P16：6.视频剪辑实操（下）

![](img/363bd42c055ab0543086affcdbfb470c_0.png)

在本节课中，我们将继续学习视频剪辑的进阶操作，重点解决非标准比例素材的处理、转场的添加以及字幕的自动识别。这些技巧能帮助你更高效地利用素材，提升视频的专业度和观看体验。

上一节我们介绍了素材导入与基础剪辑，本节中我们来看看如何处理非标准比例的素材。

## 调整视频比例与背景

在寻找素材时，你找到的视频可能并非标准的16:9横屏比例。例如，当前这个视频就不是16:9横屏。如果你想使用它，可以按照以下步骤调整。

首先，点击右下角的 **比例** 选项。在弹出的菜单中，选择 **16:9** 这一横屏比例。

选择比例后，观察视频画面。如果视频主体（如人物）在画面中位置固定，但画面本身不是16:9，我们就需要进一步裁剪。

![](img/363bd42c055ab0543086affcdbfb470c_2.png)

点击视频轨道左侧的 **剪辑** 选项。

![](img/363bd42c055ab0543086affcdbfb470c_2.png)

在剪辑菜单中，选择 **编辑**，然后进入 **裁剪** 功能。裁剪时，请务必裁掉视频自带的原有字幕，并仔细操作，避免留下过多的黑色边框，以免影响最终画面效果。裁剪完成后，点击右下角的对勾确认。

![](img/363bd42c055ab0543086affcdbfb470c_4.png)

![](img/363bd42c055ab0543086affcdbfb470c_4.png)

裁剪后，视频画面会放大，变得更清晰，但它可能仍然不是完整的16:9横屏。此时，需要为视频添加背景。

找到并点击右下角的 **背景** 选项。

选择右侧的 **画布模糊** 功能，并选用第一个最清晰的模糊样式。这样，视频就拥有了一个完整的16:9横屏背景。记住这个操作步骤：对于非16:9的素材，通过“裁剪”配合“画布模糊”背景来达成横屏效果。最后，点击对勾确认。

## 为视频添加转场

调整好比例后，下一步是为视频添加转场效果。转场是连接不同视频片段的过渡内容，例如在盘点类视频中，两个不同内容之间插入的搞笑片段。

添加转场的目的，是让用户明确感知到一个视频片段已经结束，另一个全新的片段即将开始。如果缺少转场，用户可能会因画面突然切换而感到困惑，甚至退出视频，这会严重影响视频的完播率，而完播率是影响视频推荐和收益的关键因素。

以下是添加转场的步骤：

找到并点击轨道右侧的 **加号**。

![](img/363bd42c055ab0543086affcdbfb470c_6.png)

![](img/363bd42c055ab0543086affcdbfb470c_6.png)

在素材库中，选择 **热门** 分类，里面提供了大量常用的转场素材。

![](img/363bd42c055ab0543086affcdbfb470c_8.png)

![](img/363bd42c055ab0543086affcdbfb470c_8.png)

![](img/363bd42c055ab0543086affcdbfb470c_10.png)

![](img/363bd42c055ab0543086affcdbfb470c_10.png)

你可以看到各种熟悉的转场效果。平台上的许多转场都源自于此，可以放心使用。

选择转场时需要注意：时长不宜过长。避免使用6秒、7秒甚至更长的转场。转场的作用是过渡，观众通常没有耐心看完冗长的过渡内容。**转场时长保持在3到5秒最为合适**。

选择一个约3秒时长的转场素材，点击添加。

![](img/363bd42c055ab0543086affcdbfb470c_12.png)

![](img/363bd42c055ab0543086affcdbfb470c_12.png)

添加转场后，同样需要确保其比例与主视频一致，即16:9横屏。选中转场片段，点击右下角的 **背景**，选择 **画布模糊** 并应用第一个清晰样式即可。

## 自动识别与添加字幕

转场添加完毕后，下一步是为视频添加字幕。最简便的方法是使用软件的自动识别功能。

点击左下角的 **文字** 选项。

在文字菜单中，选择 **新建文本** 右侧的 **识别字幕** 功能。

点击 **开始匹配**，软件会自动识别视频中的人物语音，并将其转换为文字字幕。这个过程非常方便。

字幕识别完成后，通常只需要调整其中一个即可。选中字幕条，拖动右下角的方框将其放大。然后，将整个字幕条拖动到视频画面的最下方位置，这样观看体验更舒适。调整好一个字幕后，后续的字幕会自动应用相同的样式和位置。

![](img/363bd42c055ab0543086affcdbfb470c_14.png)

至此，视频的背景添加、比例调整、转场设置以及字幕识别与添加就全部完成了。

## 总结

![](img/363bd42c055ab0543086affcdbfb470c_16.png)

本节课中我们一起学习了视频剪辑的三大核心进阶操作：如何处理非标准比例素材并填充背景、如何添加合适的转场以提升视频流畅度，以及如何利用自动识别功能快速添加字幕。掌握这些技巧，你就能更自由地运用各类素材，制作出完成度高、观感专业的视频内容。整个流程并不复杂，按照步骤操作即可轻松完成。
