1.  **角色**
    角色是指你在特定赛道中呈现的专业身份。你选择什么赛道、输出什么内容，就需要让别人知道你在这个领域具备专业性。
    *   **公式**：`人设角色 = 赛道专家身份`
    *   **示例**：
        *   **装修博主**：展示真实装修过程、成果对比与避坑经验。
        *   **母婴赛道**：可以是分享亲身经验的“宝妈”，也可以是提供专业知识的“护理专家”。
        *   **情感赛道**：可以塑造为“心理学顾问”或分享自身蜕变故事的“成长型人设”。

2.  **内容**
    内容需要与你选择的赛道高度垂直且持续。这关系到账号的垂直度和平台标签，直接影响账号成长和流量分发。
    *   **核心要求**：内容吸引用户、提供价值、具备平台特色。
    *   **关键点**：保持内容垂直。如果内容杂乱（例如第一篇讲教育，第二篇讲宠物），会混淆用户和平台算法，导致推荐效果下降。

3.  **形式**
    形式是指内容的具体呈现方式，主要包括以下三个方面：
    *   **产出场景**：确定拍摄地点（如室内、室外、车内），可参考对标账号。
    *   **声音呈现**：决定是本人出镜讲解，还是使用配音。
    *   **内容形式**：选择剧情、干货分享或生活记录等类型。不同形式各有优劣，例如干货型需要扎实的文案，生活记录型需要丰富的素材。

4.  **记忆点** 🧠
    记忆点是让用户记住你的关键，需要通过固定的设计环节反复强化。
    *   **视频开头前置**：固定动作、特效、开场白。这些元素在视频前几秒出现，能形成强烈的重复冲击，加深用户印象。
    *   **贯穿视频全程**：统一的背景音乐（BGM）。
    *   **贯穿内容全局**：统一出现的物品、固定风格的服装或背景布置。
