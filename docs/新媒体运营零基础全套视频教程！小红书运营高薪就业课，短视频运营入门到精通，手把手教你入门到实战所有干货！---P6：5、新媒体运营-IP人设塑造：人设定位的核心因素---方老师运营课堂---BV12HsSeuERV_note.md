1.  **角色**
    角色说白了，就是你在这个赛道中以什么身份出现。你做什么赛道，准备往什么方向输出内容，就需要让别人知道你提供的内容具有一定专业性。
    *   **例如**：如果你是一个装修博主，你需要告诉别人你确实有房子在装修或已装修完成，并记录和分享整个过程遇到的坑和经验。
    *   **例如**：在母婴赛道，你通常可以扮演“宝妈”角色。但根据内容方向，你也可以扮演“护理专家”的角色，不一定非要是宝妈本人。
    *   **例如**：在情感赛道，你可以是“心理学顾问”或“情感大师”，分享情感、成长、励志话题。
    *   **例如**：你也可以打造“成长型”人设，分享自己过去在家庭、创业、职场中遇到的问题和应对方法，以及如何走出失败的经历。
    角色其实很简单，你做什么赛道方向的内容，准备提供什么价值，就把自己塑造成该方向的专家身份即可。

2.  **内容**
    内容这一点很简单。我们讲内容定位时提到过，你的内容要参照几个点：是否吸引用户、是否有价值、在平台上是否有特色等。只要满足这些点，基本上问题不大。
    在人设定位这一块，内容的关键在于**垂直和持续**。你做什么赛道，就持续提供与该赛道相关的内容。这涉及到账号的垂直度和标签问题。
    > **核心公式**：**账号垂直度 = 持续输出同一赛道内容**
    如果你的内容方向杂乱，比如第一篇讲儿童教育，第二篇讲宠物，那么平台无法给你打上精准标签，喜欢A内容的用户可能对B内容不感兴趣，这会直接影响视频的完播、互动等数据，进而影响平台的推荐流量。因此，一定要记住：你做什么赛道，就提供什么内容。

3.  **形式**
    形式指的是内容呈现的具体方式。你需要考虑以下几个层面：
    *   **产出场景**：你的内容是在固定的室内场景拍摄，还是在不同的室外场景切换？例如，在上班路上、车内等。拍摄场景可以提前确定，并参考对标账号的呈现方式。
    *   **声音呈现**：你是自己出镜讲解，还是不想出镜，使用配音？现在很多账号采用不出镜只配音的形式，技术上用“剪映”等工具就能轻松实现。你需要评估自己的声音表现力，能否流畅地完成讲解。
    *   **内容形式**：你的内容是以**剧情**、**干货分享**还是**生活记录**的形式呈现？
        *   **剧情型**：创作难度相对较高。
        *   **干货型**：需要准备丰富的文案脚本。
        *   **生活记录型**：需要大量的日常拍摄素材。
    每种形式各有优劣势，需要根据自身情况选择。

4.  **记忆点**
    记忆点至关重要，它决定了用户如何记住你。记忆点是通过固定设计的环节来强化的。
    以下是常见的记忆点设计：
    *   **视频前几秒的固定动作、特效或开场白**（需前置呈现）。
    *   **统一的BGM（背景音乐）**（贯穿视频全程）。
    *   **统一出现的某个物品**（如特定的摆件、台灯）。
    *   **统一风格的服装**（如农村题材的质朴服装、职场题材的职业装）。
    **记忆点的设计逻辑**：
    *   **前置强化**：像固定动作、特效、开场白这类元素，一定要放在视频开头。当用户刷到你多个视频时，每次开头都看到相同的元素，记忆会不断被加深，效果比放在视频中间或随机出现要强烈得多。
    *   **全局贯穿**：像BGM、特定物品、服装风格等，则贯穿整个视频或系列内容，形成一种整体的氛围和识别度。
    记忆点需要反复刻入用户心智，让他们一看到某个瞬间，就能立刻联想到你和你的内容。
