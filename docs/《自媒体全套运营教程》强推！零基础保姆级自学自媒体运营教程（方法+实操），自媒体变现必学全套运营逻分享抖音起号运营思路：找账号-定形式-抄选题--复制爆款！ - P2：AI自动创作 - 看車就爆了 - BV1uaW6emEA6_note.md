---

上一节我们介绍了自媒体运营的基本思路。本节中，我们来看看如何利用AI工具自动化创作中视频内容。

平台青睐的中视频作品需要满足两个核心条件：**文案足够优秀**，以及**画面高度相关且精彩**。基于此，我们只需借助三个工具即可实现全自动创作。

以下是实现AI自动创作所需的三个核心工具：

1.  **文案创作工具：ChatGPT**
    这款人工智能工具拥有强大的文案创作能力，能生成高度原创且富有情感的文本。其输出不像机器生成，更接近人类思维。对于中视频计划，这是一款神器。
    例如，你可以输入以下指令来获取文案：
    ```
    请为我创作一个约1000字的中国历史故事。
    ```
    你可以不断追加条件来细化要求，例如：
    ```
    在刚才的故事中加入更多人物，并保持字数不变。
    ```

2.  **AI配音工具：例如“配音神器”**
    获得优质文案后，下一步是将其转化为语音。这一步相对简单，使用AI配音工具即可完成。
    操作流程是：将文案粘贴进工具，选择喜欢的主播音色，然后生成并保存配音文件。

3.  **视频素材匹配工具：例如“三三搜帧”**
    最后一步是为配音匹配高质量、高相关度的动态视频画面。这个工具能根据文案含义，智能搜索无版权的相关视频素材。
    其核心功能“AI合成视频”可以自动完成匹配：导入配音文件，工具会根据台词自动推荐画面，最终一键生成视频。

---

接下来，我为大家演示具体的操作步骤。建议点赞收藏，内容非常干货。

首先是使用ChatGPT生成文案。以历史故事为例，你可以提出具体需求。你提的问题越精细，得到的答案就越符合预期。如果对初版文案不满意，可以追加条件进行优化。

![](img/fcb379358e6539bcea1d98ee80080552_5.png)

获得高度原创的文案后，我们进入第二步：配音。

打开配音工具，粘贴文案，选择合适的主播音色进行合成，最后保存生成的配音文件。

最后一步是匹配画面并合成视频。

我们使用“三三搜帧”工具。进入其“AI合成视频”功能，导入上一步得到的配音文件。系统会自动根据每一句台词，匹配多个相关的视频画面供你选择。确认画面后，点击“生成视频”，即可得到一段高质量的原创中视频。

![](img/fcb379358e6539bcea1d98ee80080552_7.png)
