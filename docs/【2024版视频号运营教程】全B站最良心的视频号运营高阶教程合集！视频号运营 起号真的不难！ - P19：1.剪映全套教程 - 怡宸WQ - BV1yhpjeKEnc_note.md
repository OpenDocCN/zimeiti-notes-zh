# 视频号运营教程：P19：1. 如何自学剪映剪辑 🎬

在本节课中，我们将学习如何从零开始，利用官方和平台上的免费资源，高效地自学剪映软件，掌握视频剪辑的基础技能。

---

## 课程概述

![](img/0b3a8b9b72ca5d36b9dda258a71bab72_1.png)

短视频普及多年，许多人已具备一定剪辑基础。如果你从未使用过剪映，本教程将指导你如何通过官方渠道和优质博主，系统地学习这款功能强大的免费剪辑工具。

![](img/0b3a8b9b72ca5d36b9dda258a71bab72_3.png)

## 获取与认识剪映

![](img/0b3a8b9b72ca5d36b9dda258a71bab72_5.png)

首先，你需要在手机应用商店下载“剪映”App。这是抖音官方的剪辑软件，功能强大且大部分功能免费，足以满足日常剪辑需求。其他软件如“必剪”等，其操作与剪映大同小异。

## 通过官方课堂系统学习

![](img/0b3a8b9b72ca5d36b9dda258a71bab72_7.png)

打开剪映App后，自学的最佳途径是使用其内置的官方教程。

![](img/0b3a8b9b72ca5d36b9dda258a71bab72_9.png)

以下是具体操作步骤：

![](img/0b3a8b9b72ca5d36b9dda258a71bab72_11.png)

1.  **进入创作者课堂**：在剪映首页最下方，找到并点击“创作者课堂”入口。
2.  **搜索官方账号**：在课堂页面的搜索框中，输入“剪映小助手”并进行搜索。
3.  **关注官方账号**：在搜索结果中，选择“用户”标签页，第一个结果便是“剪映官方”账号，请点击关注。
4.  **学习免费课程**：进入官方账号主页，点击“课程”栏目。这里推荐学习《剪映功能大全》和《新手常见问题》两个系列课程。前者包含87节课，后者包含10节课。

![](img/0b3a8b9b72ca5d36b9dda258a71bab72_13.png)

**核心提示**：这些课程每节时长很短，多为几十秒到两分钟。完整学完所有课程大约仅需两小时，即可让你从零基础达到熟练操作的水平。务必先系统学习全部基础功能，再进行实操。

![](img/0b3a8b9b72ca5d36b9dda258a71bab72_15.png)

## 通过抖音博主补充学习

![](img/0b3a8b9b72ca5d36b9dda258a71bab72_17.png)

除了官方课程，抖音平台上也有许多优质博主提供精炼的剪辑教程。

以下是推荐的搜索与学习方法：

![](img/0b3a8b9b72ca5d36b9dda258a71bab72_19.png)

1.  **搜索特定博主**：打开抖音，在搜索框中输入抖音号 **“妮妮6868618”**，搜索并关注这位博主。
2.  **学习系列教程**：该博主的视频通常较短，但内容系统，涵盖了从第一节到第十节的完整入门教程。例如，一个4分钟的视频就能讲清一个核心功能。
3.  **拓展搜索**：你也可以在抖音直接搜索“零基础学剪辑”等关键词，会发现大量教学账号。观看其中几个高赞视频，便能快速掌握剪映的核心操作。

**核心提示**：结合官方教程的系统性和博主教程的实用性，可以高效掌握技能。我们日常剪辑所用的基本都是这些基础功能，完全足够。

![](img/0b3a8b9b72ca5d36b9dda258a71bab72_21.png)
