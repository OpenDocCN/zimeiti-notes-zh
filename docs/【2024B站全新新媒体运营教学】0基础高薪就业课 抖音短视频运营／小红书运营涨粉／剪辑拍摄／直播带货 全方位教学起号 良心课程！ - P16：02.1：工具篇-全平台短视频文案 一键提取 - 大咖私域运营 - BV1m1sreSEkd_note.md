## 传统文案提取方法的局限

上一节我们介绍了新媒体运营的基础概念，本节中我们来看看如何高效获取文案素材。以往，当我们看到优秀的短视频并想借鉴其文案时，通常需要一些繁琐的步骤。

以下是几种传统的方法：
*   **使用小程序**：例如“轻抖”等工具，但可能仅限于特定平台。
*   **下载后提取**：先将视频下载到本地，再导入“剪映”等剪辑软件中识别字幕。
*   **手动记录**：最原始的方法，即边听边手动打字记录。

这些方法在单一平台（如抖音）或许可行，但当我们需要处理来自**小红书、视频号、哔哩哔哩、快手**等多个平台的视频时，过程就会变得复杂且耗时。

### 案例一：提取抖音短视频文案

首先，我们尝试提取一个抖音视频的文案。
1.  在抖音中找到一个目标视频（例如，搜索“情感语录”类视频）。
2.  复制该视频的分享链接。
3.  打开工具网页，将链接粘贴到指定输入框。
4.  点击“提取文案”按钮。

![](img/0713f5c9bdd3959c74ad7b8f700629e4_8.png)

稍等片刻（通常不超过2分钟），工具就会输出完整的文案。例如，可能得到如下结果：

### 案例二：提取哔哩哔哩（B站）视频文案

2.  将链接粘贴到工具输入框。
3.  点击“提取文案”。
