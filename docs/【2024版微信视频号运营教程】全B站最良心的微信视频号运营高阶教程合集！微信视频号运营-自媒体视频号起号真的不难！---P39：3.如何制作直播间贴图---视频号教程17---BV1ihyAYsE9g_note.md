# 微信视频号运营教程：P39：3. 如何制作直播间贴图 🎨

在本节课中，我们将学习如何为微信视频号直播间制作一个透明背景的贴图。这种贴图常用于展示促销信息、商品卖点或直播间主题，能有效吸引观众注意力并传递关键信息。

## 概述

许多直播间会使用文字贴图来突出信息，例如“好物推荐 老国货 黄氏 9块91盒”。本节教程将详细介绍如何使用一款免费工具“微麦水印相机”（原微商水印相机）来制作这种贴图，并成功添加到直播间画面中。

## 工具准备与选择

首先，你需要在手机应用商店搜索并下载“微麦水印相机”应用。请注意，制作贴图无需开通会员，使用免费功能即可。

## 制作透明背景贴图

以下是使用“微麦水印相机”制作贴图的详细步骤。

### 1. 进入水印模板库

打开应用后，找到“水印库”或类似入口。点击“暂不开通”跳过会员提示，然后选择“更多水印”或“水印模板”。

### 2. 选择文字模板

在水印模板列表中，找到“横竖文字”分类。选择第一个或任意一个你喜欢的纯文字模板作为基础。

### 3. 编辑贴图文字

点击选中的模板，进入编辑界面。你可以将模板中的示例文字替换成你自己的直播间信息。
例如，你可以输入：
```
好物推荐
老国货品牌
9块91盒
19块93盒
29块95盒
```
你可以模仿其他成功直播间的文案进行修改。

### 4. 自定义样式

接下来，你可以对贴图的样式进行个性化调整。
*   **文字颜色**：点击颜色选项，可以更改为彩色、粉色或其他你喜欢的颜色。
*   **字体**：应用提供了多种字体可供选择。
*   **背景设置（关键步骤）**：找到“背景色”设置选项。**必须将背景设置为透明**。通常，你需要选择一个带有斜杠“/”图标的选项，或者将背景色的不透明度数值调整为0。这一步至关重要，能确保贴图在直播间里只显示文字，而没有底色方块。

![](img/6450b1709dbf8c401939cc83d6d58cae_1.png)

### 5. 生成并保存贴图

完成所有编辑后，点击“生成”或“保存”按钮。确保将制作好的图片保存到你的手机相册中。

![](img/6450b1709dbf8c401939cc83d6d58cae_3.png)

## 在直播间添加贴图

![](img/6450b1709dbf8c401939cc83d6d58cae_5.png)

贴图制作完成后，我们来看看如何将其添加到微信视频号的直播间。

### 1. 开启直播

![](img/6450b1709dbf8c401939cc83d6d58cae_7.png)

在视频号中点击“发起直播”。

![](img/6450b1709dbf8c401939cc83d6d58cae_9.png)

### 2. 添加贴图

在直播准备界面，找到添加“贴图”、“画面”或“图片”的按钮（图标可能是一个笑脸或图片标志）。选择从“手机相册”添加图片。

### 3. 选择并调整贴图

![](img/6450b1709dbf8c401939cc83d6d58cae_11.png)

从相册中选择你刚刚制作并保存好的贴图。贴图会出现在直播画面中，你可以用手指拖动来调整其位置，或使用双指缩放来调整大小，将其放置在合适的位置。

![](img/6450b1709dbf8c401939cc83d6d58cae_12.png)

### 4. 后期修改

如果添加后觉得文字大小或样式不合适，你可以返回“微麦水印相机”重新编辑贴图。在编辑界面，你可以调整**字体大小**、**字间距**和**行距**等参数，然后重新生成并保存，再次添加到直播间即可。
