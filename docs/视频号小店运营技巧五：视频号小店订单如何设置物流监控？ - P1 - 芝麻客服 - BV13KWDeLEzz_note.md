# 视频号小店运营技巧：P1：如何设置物流监控？ 📦

在本节课中，我们将学习如何为视频号小店设置订单物流监控。掌握此功能，可以帮助你及时处理物流异常，减少客户投诉与平台罚款，从而更高效地管理店铺。

上一节我们介绍了物流监控的重要性，本节中我们来看看具体的设置方法。

## 物流监控的必要性

视频号小店官方后台不具备物流监控功能。当订单发货超时、物流状态异常或派件出现问题时，你需要借助第三方工具来第一时间获取预警信息。

## 使用第三方工具：芝麻视频号助手

你需要使用视频号小店第三方管理工具——芝麻视频号助手来完成物流监控的设置。

以下是具体的操作步骤：

1.  **授权接入**：在芝麻视频号助手后台，授权并接入你的视频号小店。
2.  **开通功能**：开通“物流监控”功能。
3.  **选择店铺**：选择你需要监控的具体视频号小店。
4.  **自动监控**：系统将自动监控已发货订单的物流状态。
5.  **手动上传**：你也可以自行上传需要监控的物流单号。
