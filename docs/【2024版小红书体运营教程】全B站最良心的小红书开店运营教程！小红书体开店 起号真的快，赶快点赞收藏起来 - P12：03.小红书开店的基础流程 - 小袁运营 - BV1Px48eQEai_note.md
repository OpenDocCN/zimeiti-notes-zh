# 小红书开店运营教程：P12：03.小红书开店的基础流程 🛒

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_1.png)

在本节课中，我们将学习如何在小红书平台开通店铺。课程将详细介绍个人店、个体店等不同类型店铺的开通流程、所需材料、注意事项以及相关费用规则，帮助初学者快速上手。

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_3.png)

## 店铺类型与基础要求

上一节我们介绍了小红书运营的宏观概念，本节中我们来看看具体的开店流程。首先，你需要了解小红书支持的主要店铺类型及其基础要求。

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_5.png)

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_7.png)

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_9.png)

以下是三种常见店铺类型及其核心要求：

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_11.png)

*   **个人店**：这是最基础的店铺类型。开通需要**身份证**和**银行卡**，且必须为同一人持有。
*   **个体店**：适用于个体工商户。开通需要**营业执照**和**运营人的身份证**。
*   **企业店/旗舰店**：适用于公司企业。开通涉及企业资质，并可能需处理税务问题。

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_13.png)

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_15.png)

**核心提示**：用于收款的银行卡持有人必须与店铺认证的身份证持有人一致，这是为了避免后续的经济纠纷。如果使用他人身份信息开店，务必选择熟悉可靠的人。

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_17.png)

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_19.png)

## 个人店开通详细步骤

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_21.png)

了解了基础要求后，我们以最常用的个人店为例，详细讲解开通步骤。这个过程非常简单，可以全程在手机APP上完成。

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_23.png)

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_25.png)

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_27.png)

以下是具体的操作步骤：

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_29.png)

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_31.png)

1.  **搜索入口**：打开小红书APP，在顶部搜索栏输入“**开店**”并搜索。
2.  **进入官方页面**：在搜索结果中找到“小红书电商官方平台”，点击“**开店**”按钮。
3.  **选择店铺类型**：点击“立即开店”，在店铺类型中选择“**个人**”。
4.  **填写信息**：根据页面提示，上传身份证正反面照片，填写个人信息。部分非必填信息（如邀请码）可以跳过。
5.  **完成开通**：信息填写无误后，点击“立即开店”。通常情况下，店铺可以快速开通。

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_33.png)

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_35.png)

**关键点说明**：
*   **店铺名称与类目**：店铺名称可以暂时随意填写，后期支持修改（每7天可改一次）。经营类目也可以先任意选择一个，后续再调整。
*   **保证金**：开店过程中可以**跳过保证金缴纳步骤**，先完成店铺开通。保证金金额约为1000元，具体根据类目而定，可在后续经营中补缴。

## 重要规则与费用说明

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_37.png)

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_39.png)

成功开通店铺只是第一步，理解平台的运营规则和费用结构至关重要。本节将解释保证金和技术服务费这两个核心财务概念。

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_41.png)

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_43.png)

*   **保证金**：开店后可暂不缴纳，但若要持续经营或上架特定商品，则需补缴。个人店保证金通常在**1000元左右**，按类目浮动。
    > **公式**：`应缴保证金 ≈ 基础保证金（如1000元） + 类目附加保证金`

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_45.png)

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_47.png)

*   **平台技术服务费（扣点）**：平台对成交订单收取一定比例的费用。
    *   **免扣点条件**：一个自然月内，订单结算金额（扣除退款后）**不超过1万元**，则免收技术服务费。
    *   **收费规则**：结算金额超过1万元的部分，平台按 **5%** 的比例收取技术服务费。
    > **公式**：`技术服务费 = (当月结算金额 - 10000元) * 5%` （仅当结算金额 > 10000元时计算）
    > **示例**：若某月结算金额为2万元，则技术服务费为 `(20000 - 10000) * 5% = 500元`。

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_49.png)

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_51.png)

## 特殊类目与个体店补充

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_53.png)

除了个人店，你可能对经营特定商品或开通个体店感兴趣。本节补充这些内容，并提醒一些特殊类目的准入要求。

**个体店开通**：需要准备**小红书实名账号**和**营业执照**（法人最好与账号实名一致）。可通过“小红书千帆”（商家版APP）或官网进行开通。

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_55.png)

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_57.png)

**特殊类目要求**：部分商品类目需要额外的资质证明，新手需特别注意。
以下是几个常见特殊类目的要求：

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_59.png)

*   **食品类**（如酒水、零食、水果）：需提供**营业执照**和**食品经营许可证**，部分需要品牌授权。
*   **美妆类**：需提供**化妆品备案凭证**。
*   **家电/3C数码类**：需提供**3C认证证书**。

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_61.png)

> **建议**：如果不熟悉这些行业或无法提供相关资质，初期应避免选择这些类目。平台所有类目的具体要求，均可在开店后台的“类目管理”中查询。

![](img/d05f5fb5c1bd7d0ee3d4e6172fbc30d0_63.png)
