上一节我们介绍了开店的基础流程，本节中我们来看看店铺的具体分类和资金要求。

## 店铺类型详解

小红书平台上的店铺类型主要分为以下几种，每种类型对应不同的申请主体和经营权限。

以下是各类店铺的具体说明：

1.  **个人店**
    *   **申请主体**：个人用户。适合刚毕业或想尝试小规模经营的个体，类似于“线上摆摊”。
    *   **经营特点**：可经营限定类目下的零星、小额商品，如首饰、服装、手机壳等，属于“杂货铺”模式。
    *   **入驻要求**：仅需个人身份证，非常简单。
    *   **店铺命名格式**：系统自动生成，格式为“专业号名称 + 店名”。
    *   **保证金**：涉及基础保证金和浮动保证金，二者取金额高者缴纳。个人店的基础保证金通常为1000元。

2.  **个体工商户店**
    *   与个人店的主要区别在于**入驻要求**：需要提供**营业执照**。对品牌授权无要求。
    *   其他方面，如经营特点和保证金规则，与个人店基本一致。

![](img/10e62769643c236f4f790f93ce03e050_7.png)

![](img/10e62769643c236f4f790f93ce03e050_9.png)

3.  **普通企业店**
    *   **申请主体**：企业。需要持有企业的营业执照。
    *   其与后续更高级店铺的核心区别同样在于**入驻要求**的复杂程度。通常，入驻要求越简单，店铺初始权重可能越低；要求越复杂，权重可能越高。对于新手，建议从要求简单的店铺类型开始测试。

4.  **专卖店**
    *   **入驻要求**：需要营业执照，并且是品牌方（持有自有品牌或非独占授权品牌）。即销售自己品牌或已获得销售授权的品牌商品。

![](img/10e62769643c236f4f790f93ce03e050_11.png)

5.  **旗舰店**
    *   **入驻要求**：需要营业执照，并且必须持有**品牌授权书**（销售他人品牌商品）。

![](img/10e62769643c236f4f790f93ce03e050_13.png)

6.  **卖场型旗舰店**
    *   **经营特点**：需具备零售资质，并可经营多个品牌。但在小红书平台，此类店铺较少，因为平台以女性用户为主，流量特性与大型综合卖场不完全匹配。这反而为中小卖家提供了更友好的竞争环境。

了解完店铺类型后，接下来我们需要关注一个重要的实操环节：保证金。

## 保证金规则说明

![](img/10e62769643c236f4f790f93ce03e050_15.png)

保证金的设置是为了保障平台交易安全。小红书保证金主要包括两部分：**基础保证金**和**浮动保证金**。

![](img/10e62769643c236f4f790f93ce03e050_17.png)

### 基础保证金与类目保证金

![](img/10e62769643c236f4f790f93ce03e050_19.png)

![](img/10e62769643c236f4f790f93ce03e050_21.png)

*   **基础保证金**：通常为 **`1000元`**。这是开通店铺功能的基础。
*   **类目保证金**：部分特殊商品类目（如普通企业店的特殊类目、跨境类目等）可能需要额外缴纳固定金额的保证金。具体金额需参考平台官方发布的类目保证金表。

![](img/10e62769643c236f4f790f93ce03e050_23.png)

### 浮动保证金（核心规则）

![](img/10e62769643c236f4f790f93ce03e050_25.png)

浮动保证金是根据店铺的月成交金额动态计算的，这是需要重点理解的规则。

以下是浮动保证金的计算阶梯：

*   **月成交金额 ≤ 5万元**：浮动保证金为 **`0元`**。（但需已缴纳1000元基础保证金）
*   **5万元 < 月成交金额 ≤ 10万元**：需保证金 **`3000元`**。
*   **10万元 < 月成交金额 ≤ 50万元**：需保证金 **`5000元`**。
*   **月成交金额 > 50万元**：需保证金 **`20000元`**。

**重要提示**：
平台在商家提现时会校验保证金账户总额是否充足。如果店铺月成交额已超过5万元，但保证金总额（基础+浮动）未达到3000元，则**无法进行提现**。务必定期关注店铺成交额和保证金状态，及时补足差额。

公式表示如下：
```
所需保证金 = max(基础保证金， 类目保证金， 浮动保证金)
```
其中，浮动保证金根据上述月成交额阶梯确定。
