## 商品准备的核心规划

小红书开店的前期规划包含三个步骤：**商品准备**、**基础数据准备**和**操作人员准备**。只有将这三者都准备好，后续发布的笔记才能有效引流并带来转化。缺乏这些基础数据，即使发布了产品和笔记，也难以产生任何转化量。

## 商品数量的科学配置

在小红书创建店铺并上架商品时，商品的基础数量有特定的要求。虽然理论上可以上架任意数量的商品，但数量会直接影响店铺的动销率、转化率、账号活跃度及权重。因此，我们需要找到一个对账号提升最有效的商品数量范围。

![](img/efbdd74fe46cd676e5b2491568daf904_8.png)

以下是小红书店铺商品数量的配置公式：

**店铺商品总数 = 主产品 + 副产品 + 补量产品**

具体的数量配置建议如下：

*   **整体店铺（中后期发展）**：商品总数建议在 **7~15个** 之间。
    *   包含 **1件主产品**、**2-4件副产品** 和 **5-10件补量产品**。
*   **单店模式（前期启动）**：商品总数建议为 **7个**。
    *   包含 **1件主产品**、**1-2件副产品** 和 **4-5件补量产品**。

以下是关于各类产品的定义与作用：

*   **主产品**：你计划主推、希望打造成爆款的核心商品。
*   **副产品**：作为搭配或次要推荐，能卖则卖，不作为主要销售目标。
*   **补量产品**：主要用于“凑数”，丰富店铺陈列，提升店铺专业感和用户信任度，本身不追求销量。

## 商品数量的动态调整策略

商品配置并非一成不变，需要随着店铺发展动态调整。

**前期启动**：无论是单店还是多店模式，都从 **7件商品** 开始（例如：1主、1副、5补）。

**发展中后期**：当你成功推出第二个爆款（即新的主产品）时，需要对商品结构进行调整。
*   增加新的主产品。
*   副产品数量可相应调整。
*   补量产品数量需增加，以保持总商品数在建议范围内（如增至10-12个）。
*   公式示例：`2件主产品 + 2件副产品 + 8件补量产品 ≈ 12件商品`

**持续发展**：当推出第三、第四个爆款时，逐步用新的主产品替换部分补量产品，始终将店铺总商品数维持在 **7-15件** 的黄金区间内。

## 商品数量配置的重要性

严格控制商品数量范围至关重要，原因如下：

1.  **维护店铺健康指标**：商品过多会导致动销率低下，影响店铺权重和转化率，可能引发数据崩盘。
2.  **建立用户信任**：商品过少（低于7件）会让用户觉得店铺不专业，像是“二道贩子”，难以建立信任感。保持7-15件的商品数量，能营造出店铺的专业性和诚意。

因此，必须把握的核心原则是：**店铺商品数量不能低于7件，也不要超过15件**。在这个范围内，最有利于维护账号活跃度、把握成交数据并实现权重的层层升级。
