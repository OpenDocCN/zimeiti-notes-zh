在本节课中，我们将深入学习小红书运营规则的第二部分，重点解析“被推核心逻辑”与“推广规则”。理解这些规则是避免违规、获取平台额外流量支持的关键。

上一节我们介绍了小红书的基础规则，本节中我们来看看更具体的流量分配逻辑。

## 被推核心逻辑

“被推核心逻辑”指的是小红书系统为笔记提供额外推广和权重加成的核心规则。简单来说，就是平台在何种情况下会主动给你的笔记增加曝光。

以下是获得平台额外流量加成的四种主要途径：

![](img/629a5deb74ebcbc441508b2eb8f9d66f_5.png)

1.  **与小红书存在利益相关的笔记**：指品牌方或相关权威机构进行认证或合作的笔记。这类账号会获得基础权重加成。
    *   **公式**：`账号基础权重 = 普通账号权重 + 品牌/认证加成`
    *   例如，普通新号笔记的基础展现可能在100-300次，而获得认证的账号，初始展现可能达到2000-6000次。加成幅度取决于品牌影响力和合作深度。

![](img/629a5deb74ebcbc441508b2eb8f9d66f_7.png)

2.  **付费推广**：使用小红书平台内的付费推广工具（如薯条、信息流广告）。付费行为本身会带来权重加成。
    *   **注意**：付费推广竞争激烈，且商品展示位有限（约每10-20篇笔记出现一个商品位），因此**不建议运营前期使用**。它更适合中后期店铺突破流量瓶颈时使用。

3.  **积极参与官方活动**：参加小红书官方发起的各类话题活动、品牌合作活动等。官方会对参与优质活动的账号给予流量扶持。

4.  **晒单与抽奖**：参与或发起符合平台规范的晒单、抽奖类互动。这类内容容易引发用户参与，从而获得系统推荐。

**核心要点**：被推逻辑是平台给予的“额外奖励”，而非流量来源的根本。你的内容质量和基础运营仍是核心。

了解了平台偏好的推广逻辑后，我们来看看在具体发布和推广时需要遵守哪些规则。

## 推广规则详解

推广规则主要规范我们在小红书发布和运营内容时的具体行为，确保不触犯平台红线。

以下是推广中必须注意的三个关键点：

1.  **发布须知**：内容必须符合《小红书社区规范》。
    *   **核心禁令**：**避开需要长期使用才能体现功效的产品宣传**。例如，“XX产品一个月瘦身XX斤”这类承诺具体时效的功能性宣传，极易被判定为虚假宣传并导致限流。
    *   **正确做法**：可以分享“一周健康饮食打卡”这类记录个人体验的笔记，但避免将其与产品的即时功效强绑定进行商品推广。

2.  **注意事项**：主要指禁止对笔记数据进行人工干预或造假。
    *   **禁止行为**：刷粉丝、刷点赞、刷收藏、刷搜索点击等任何形式的数据造假。
    *   **原因分析**：小红书是“内容为王”的平台，初始权重已很高。人为刷数据不仅效果有限（`人工刷量权重 < 自然流量权重`），且极易被系统检测，导致笔记限流甚至账号处罚。应将精力集中在创作优质内容上。
    *   **发布频率建议**：
        *   推广单品的账号，**每天最多发布2篇**相关笔记。
        *   整体账号，**每天建议发布2篇，最多不超过5篇**精品笔记。
        *   发布过多（尤其是超过5篇）会导致账号权重分散，反而影响核心笔记的流量。

3.  **违规处罚**：了解平台的处罚机制。
    *   **管家警告**：收到系统或人工“管家”的警告后，笔记曝光会被严重限制（例如从2000曝光降至200以下）。
    *   **数据异常监测**：账号的互动数据（点赞、收藏、粉丝增长）若在**月度周期内波动超过20%**，会触发系统监测。如果是自然爆文导致的增长则无碍；但若是异常陡增，则会被核查是否为刷量，并可能面临处罚。
    *   **技巧**：将数据增长控制在平稳、合理的范围内（月波动尽量低于20%），更容易被系统视为正常运营。

本节课中我们一起学习了小红书“被推”流量的核心逻辑以及必须遵守的推广规则。总结来说，**获得额外加成的关键是“官方认可”与“优质参与”**，而**安全运营的底线是“内容真实”与“数据纯净”**。
