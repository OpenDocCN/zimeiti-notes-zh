## 什么是新品标签？🏷️

![](img/d4ca669ef9c055931dfdafe81c3d3d01_4.png)

上一节我们了解了新品曝光的重要性，本节中我们来看看其核心载体——新品标签。

新品标签是平台对原创且款式新颖的商品，在满足内部考核标准后自动授予的标识。打标后的商品能获得更多推荐曝光和展示机会。

![](img/d4ca669ef9c055931dfdafe81c3d3d01_6.png)

新品标签的优势如下：

![](img/d4ca669ef9c055931dfdafe81c3d3d01_8.png)

![](img/d4ca669ef9c055931dfdafe81c3d3d01_9.png)

*   **增加曝光**：在所有端口获得更多展示机会。
*   **提升访客**：为店铺带来显著的访客增长。
*   **提高活动通过率**：提报活动时，通过率会大大增加。
*   **获得特定展示**：报名日常活动后，可在新品专属区域得到曝光。
*   **提升店铺健康度**：增加店铺活跃度与用户粘性，拉升复购率。

![](img/d4ca669ef9c055931dfdafe81c3d3d01_10.png)

![](img/d4ca669ef9c055931dfdafe81c3d3d01_11.png)

**核心提示**：部分类目可能没有显性的“新品标签”，但所有类目的新品都享有平台的新品加权推荐。因此，无论是否有标签，都应重视新品运营。

3.  **对商家**：为新商家提供快速成长的机会，并帮助所有商家测试市场反应，以便及时调整产品与营销策略。

## 如何获取新品推荐加权？🎯

明确了新品推荐的价值与逻辑后，本节我们将聚焦于最实际的操作环节：如何获取新品加权。

![](img/d4ca669ef9c055931dfdafe81c3d3d01_30.png)

### 店铺端要求

![](img/d4ca669ef9c055931dfdafe81c3d3d01_35.png)

店铺需满足以下基础条件：
*   **店铺评分**：近90天描述相符/服务态度/物流服务三项DSR评分，需超过同行30%以上。
*   **店铺领航员**：整体表现需为良好或以上。

![](img/d4ca669ef9c055931dfdafe81c3d3d01_37.png)

![](img/d4ca669ef9c055931dfdafe81c3d3d01_38.png)

### 商品端要求

商品本身需满足以下条件：
*   **款式新颖**：商品为新品且款式新颖。
*   **信息完整**：商品信息（属性、标题等）须完整详细。
*   **图片美观**：主图需清晰美观。

## 如何保证图片原创性？🖼️

既然图片是获取新品推荐的关键，那么如何确保图片的原创性呢？以下是两种主要方法。

![](img/d4ca669ef9c055931dfdafe81c3d3d01_51.png)

![](img/d4ca669ef9c055931dfdafe81c3d3d01_52.png)

![](img/d4ca669ef9c055931dfdafe81c3d3d01_53.png)

![](img/d4ca669ef9c055931dfdafe81c3d3d01_54.png)

### 方法一：自行拍摄
这是最直接有效的方式，能最大程度保证图片的唯一性。

![](img/d4ca669ef9c055931dfdafe81c3d3d01_55.png)

![](img/d4ca669ef9c055931dfdafe81c3d3d01_56.png)

![](img/d4ca669ef9c055931dfdafe81c3d3d01_57.png)

![](img/d4ca669ef9c055931dfdafe81c3d3d01_58.png)

### 方法二：制作“伪原创”图片
若不具备拍摄条件，可通过后期软件修改图片像素信息，制作“伪原创”图片以提升系统判定为原创的概率。

以下是使用Adobe Photoshop进行“伪原创”处理的基本步骤：

1.  **导入图片**：将供应商提供的图片导入PS。
2.  **叠加图层**：在图片上层添加一个颜色构成复杂的图片或纹理图层。
3.  **调整透明度**：降低上层图层的不透明度，直至不影响底层商品图片的视觉观感。
4.  **导出图片**：合并图层并导出为新图片文件。

![](img/d4ca669ef9c055931dfdafe81c3d3d01_60.png)

![](img/d4ca669ef9c055931dfdafe81c3d3d01_61.png)

![](img/d4ca669ef9c055931dfdafe81c3d3d01_62.png)

**原理简述**：此操作改变了图片的底层像素分布信息（`像素矩阵`），从而影响系统的同款识别判定。虽然不能完全规避识别，但能有效增加被判定为“原创”的权重。

![](img/d4ca669ef9c055931dfdafe81c3d3d01_63.png)

![](img/d4ca669ef9c055931dfdafe81c3d3d01_64.png)

![](img/d4ca669ef9c055931dfdafe81c3d3d01_65.png)

**代码/公式示意**：
原始图片像素矩阵可能表示为 `P_original`，经过叠加纹理图层 `T` 并调整透明度 `α` 后，新图片像素矩阵 `P_new` 可近似表示为：
`P_new ≈ (1 - α) * P_original + α * T`
（其中 `α` 为一个较小的值，例如0.05）
