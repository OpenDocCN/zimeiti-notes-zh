# 新媒体运营：P17：02.2：工具篇-AI生成PPT 🚀

在本节课中，我们将学习如何使用一款高效的AI工具来辅助生成PPT。这款工具能根据你提供的主题或现有文档，自动梳理大纲并生成详细的演示文稿内容，极大地提升内容创作效率。

上一节我们介绍了新媒体运营的多种工具，本节中我们来看看如何利用AI技术快速制作专业的PPT。

## 工具核心功能概述

这款AI生成PPT工具主要提供两种核心工作模式：

1.  **主题生成模式**：输入一个主题，工具会结合生成式AI（如ChatGPT）自动创作全文并生成PPT。
2.  **文档转换模式**：上传现有的文档（如TXT、PDF），AI助手会自动总结文档大纲，并基于此大纲生成PPT内容。

我试用过许多AI生成PPT的工具，此工具在基于大纲生成详细内容方面表现尤为出色。

## PPT的编辑与优化

生成PPT后，我们可以对其进行进一步的美化和调整。

在编辑界面右侧，提供了“更换模板”的选项。虽然当前内置的模板数量不算非常多，但切换速度非常迅速。我们可以选择一个合适的模板进行应用。

![](img/fec7af1b15cb350b43bb1090e8e2dfb7_13.png)

此工具一个显著的优点是：它能够分析你提供的内容，并将其智能归类到不同的大纲节点下。许多其他AI工具需要用户提供完整的文稿才能生成PPT，而这工具仅需大纲或基础文档就能完善具体内容，这才是AI在PPT生成领域的核心价值。

除了更换模板，我们还可以在右侧对每一页的**文字**、**形状**等元素进行自定义调整和替换。

![](img/fec7af1b15cb350b43bb1090e8e2dfb7_15.png)

![](img/fec7af1b15cb350b43bb1090e8e2dfb7_15.png)

![](img/fec7af1b15cb350b43bb1090e8e2dfb7_17.png)

如果对在线编辑不满意，我们也可以将生成好的PPT下载到本地，使用PowerPoint或Keynote等专业软件进行深度调整。
