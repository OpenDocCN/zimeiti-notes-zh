# 视频号创作分成计划：P5：3. 视频号与快手抖音的区别 🆚

在本节课中，我们将详细拆解视频号与抖音、快手这两大主流短视频平台的核心区别。我们将从用户数据、创作者生态、平台扶持和未来机遇等多个维度进行对比，帮助你理解为何当前是入局视频号的绝佳时机。

上一节我们介绍了视频号的基本情况，本节中我们来看看它与抖音、快手的具体差异。

## 平台核心数据对比 📊

我们既然将视频号与抖音、快手对标，那么它具体有哪些优势？我们为何不选择抖音、快手，而选择视频号平台？我们从以下三个关键数据点进行分析。

第一个是日活跃用户数（DNU）。视频号目前的日活用户为 **5亿**，抖音为 **6.5亿**，快手为 **3.2亿**。抖音达到这个数据量，从2016年开始，整整花了5-6年时间。而视频号仅用一年半的时间，就远超快手、直追抖音，其发展速度非常快。

## 创作者生态分析 👥

![](img/96d82db078d88889fe0b080d831277c5_2.png)

接下来，我们看一下创作者数量，这个指标非常重要。以下是三个平台的创作者数量对比：

*   **视频号**：创作者数量相对较少。
*   **抖音 & 快手**：创作者数量接近 **2亿**。

抖音和快手的创作者数量庞大，意味着竞争异常激烈。如果你没有独特的技术或显著优势，很难从中脱颖而出。因此，视频号目前竞争压力相对较小。

![](img/96d82db078d88889fe0b080d831277c5_1.png)

![](img/96d82db078d88889fe0b080d831277c5_2.png)

上图清晰地展示了平台间的对比。此外，视频号目前的商业化程度较低，但未来必然向商业化程度高的方向发展。根据腾讯内部相关消息，**视频号计划在2022年下半年正式开启商业化进程**。这表明平台对创作者的扶持力度非常大。

## 平台流量扶持实例 🚀

如果你经常刷视频号，可能会发现一个现象：一些曾经在抖音或其他平台火过的段子，在视频号上再次发布后，又能火一遍，甚至播放量更高。这就是平台流量扶持的体现。

以下是我学员的实际案例，同样的“缝纫小技巧”生活妙招内容：

![](img/96d82db078d88889fe0b080d831277c5_4.png)

![](img/96d82db078d88889fe0b080d831277c5_5.png)

*   **抖音平台数据**（同一天发布）：点赞497，评论11，收藏255，转发88。
*   **视频号平台数据**（同一天发布）：播放量3万+，收藏7.8万，点赞5.9万，评论475。

![](img/96d82db078d88889fe0b080d831277c5_7.png)

可以看到，**完全相同**的作品，在视频号上获得了远超抖音的流量和互动数据，这直观地证明了视频号当前巨大的流量扶持红利。

![](img/96d82db078d88889fe0b080d831277c5_8.png)

## 为何选择现在入局视频号？🎯

![](img/96d82db078d88889fe0b080d831277c5_9.png)

![](img/96d82db078d88889fe0b080d831277c5_10.png)

基于以上分析，我们现在选择做视频号，主要有以下几个原因：

![](img/96d82db078d88889fe0b080d831277c5_11.png)

第一个原因，正如刚才所说，平台有明确的流量扶持政策。更重要的是，官方已经发布了“创作者激励计划”。

![](img/96d82db078d88889fe0b080d831277c5_7.png)

![](img/96d82db078d88889fe0b080d831277c5_8.png)

![](img/96d82db078d88889fe0b080d831277c5_9.png)

![](img/96d82db078d88889fe0b080d831277c5_13.png)

![](img/96d82db078d88889fe0b080d831277c5_10.png)

![](img/96d82db078d88889fe0b080d831277c5_11.png)

官方推出的“三有”扶持政策，计划在未来一年内，通过**流量扶持、专项奖励和全生命周期成长权益体系**，让平台创作者实现“有流量、有收入、有成长”。这对于零基础的新手创作者来说，是极大的机遇。

同时，视频号具备先天的流量优势。

![](img/96d82db078d88889fe0b080d831277c5_13.png)

它依托于微信生态，**天生自带约12亿的潜在用户流量**。而抖音和快手是从零开始积累的。这相当于视频号“生下来就自带流量加成Buff”。

此外，抖音和快手的发展已可见天花板，内容模式趋于固化，想要突破很难。而视频号即将在下半年进入商业化模式，这意味着将会有大量资本涌入。目前，视频号上少有顶级流量账号，几十、几百播放量的内容也很常见，这反而意味着**现在起号、做号更加容易**。

一旦你的账号做起来，掌握了流量密码，自然就能与品牌、资本合作，通过推广广告赚取红利。这就是当下的时代机遇。

## 总结与展望 💎

本节课中，我们一起学习了视频号与抖音、快手的核心区别。

现在入局视频号，就相当于**5年前入局抖音**。回想早期通过抖音短视频赚钱的人非常多。例如2018年双十一，薇娅单场带货销售额达2.67亿，单场直播利润超过200万；李佳琦单场利润超过300万。他们正是踩中了时代的红利。
