# 小红书电商运营：第37课：手机端基础设置 📱

在本节课中，我们将学习如何通过手机端的小红书商家版App，完成店铺的基础设置。这些设置是店铺正常运营的基石，包括物流、资金和营销工具的初步配置。

上一节我们介绍了店铺注册与开通，本节中我们来看看如何通过手机进行后续的基础配置。

---

## 一、物流设置 🚚

物流设置是确保订单能够正常发货与退货的关键。以下是需要完成的三个主要步骤。

### 1. 地址库管理

首先，我们需要设置两个关键地址：发货地址和售后地址。

*   **发货地址**：填写你实际发货所在的地址。
*   **售后地址**：通常设置为一个统一的地址，用于处理退货退款。

**操作流程如下：**

1.  进入商家版App，点击「消息」->「接待」进入客服页面。
2.  找到「基础设置」入口。
3.  在「地址库管理」中，点击「新增地址」。
4.  填写你的实际发货地址（例如：浙江省杭州市钱塘区下沙街道XX路XX号）和手机号，保存。
5.  将该地址「设为默认发货地址」。
6.  再次「新增地址」，填写一个统一的售后地址（例如：中国台湾，详细地区联系客服），并使用相同的手机号。
7.  将该地址「设为默认售后地址」。

### 2. 物流模板

物流模板用于关联你的发货地址。

**操作流程如下：**

1.  在物流设置中找到「物流模板」。
2.  点击「新建模板」，名称可随意填写以便自己识别。
3.  在「发货地址」选项中，选择你刚才设置的默认发货地址。
4.  保存即可，其他选项暂无需调整。

### 3. 运费模板

运费模板决定了向买家收取的运费规则。

**操作流程如下：**

1.  在物流设置中找到「运费模板」。
2.  通常可以选择「非偏远地区包邮」模板。这意味着大部分地区包邮，但新疆、西藏等偏远地区不配送或需额外付费。
3.  你可以根据自己产品的利润和发货成本，对这个模板进行微调。

至此，物流相关的核心设置就完成了。

## 二、账户资金与保证金 💰

完成物流设置后，我们需要关注店铺的资金账户。核心是保证金的缴纳，它关系到支付功能的开通和店铺权重。

以下是关于资金设置的要点列表：

*   **保证金缴纳**：保证金是平台要求缴纳的诚信担保金。你可以选择立即缴纳或稍后缴纳，但某些功能（如运费险）需缴纳后才能开通。
*   **支付方式绑定**：
    *   **支付宝**：即使未开通商家支付宝账号，买家仍可使用支付宝付款。
    *   **微信支付**：**必须**开通商家微信支付账号后，买家才能使用微信付款。
    *   **银行卡**：需要绑定你的收款银行卡。
*   **开通顺序**：请注意，支付账号（支付宝/微信支付）的开通和「运费险」功能的开通，通常都**需要先缴纳保证金**。这些操作目前主要在电脑端完成。

## 三、店铺类型与营销工具 🛍️

最后，我们来设置店铺的经营类目和初步的营销工具。

### 1. 店铺经营类型

这决定了你可以销售的商品类目。

**操作要点：**
在设置中找到「店铺类型」或「经营类目」选项。为了获得最大的选品灵活性，建议将提供的所有经营类型**全部勾选**。然后填写对应的资质地址等信息即可。

### 2. 优惠券设置

优惠券是吸引顾客下单的有效营销工具。如果主要使用手机操作，建议优先设置优惠券。

**操作流程与建议：**

1.  在「营销中心」找到「优惠券」创建入口。
2.  你可以创建固定面额的优惠券，也可以设置阶梯式优惠券（如满100减10，满200减25）。
3.  **重要建议**：如果你**仅使用手机操作**，建议只设置优惠券，**暂时不要设置商品折扣**。因为商品折扣功能通常在电脑端操作更为方便，两者同时设置可能导致计算冲突。

**关于上架价格的补充说明：**
视频中提到的“两倍加5上架”是一种定价策略，其公式为：
```
上架标价 = (成本价 × 2) + 5
```
例如，成本价30元的商品，上架标价设为 `(30 * 2) + 5 = 65` 元。后续再通过折扣或优惠券降低实际售价，给顾客营造优惠感。
