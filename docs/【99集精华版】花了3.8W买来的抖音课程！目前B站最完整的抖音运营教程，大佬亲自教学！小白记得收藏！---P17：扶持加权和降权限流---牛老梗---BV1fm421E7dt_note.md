# 抖音运营教程：P17：扶持加权和降权限流

在本节课中，我们将要学习抖音平台对视频内容的扶持与限制规则。我们将了解哪些类型的视频会获得平台的流量扶持，哪些内容则可能导致账号被降权限流。掌握这些规则，是避免踩坑、让作品获得更好曝光的基础。

上一节我们介绍了账号权重的分类，本节中我们来看看具体影响权重的视频内容规范。

![](img/c9d4c177427bd1ab4b6785ac892ef6ab_1.png)

## 获得扶持加权的内容特征 🚀

![](img/c9d4c177427bd1ab4b6785ac892ef6ab_3.png)

平台会对符合特定标准的视频内容给予流量扶持。以下是获得扶持加权的核心要求：

*   **视频格式**：必须为**竖版移动端原生视频**，即**9:16**的格式。这是抖音平台推荐的标准视频格式。
*   **画面质量**：画面需保持平稳，作品清晰度高。
*   **音频质量**：视频中无过多的杂音或噪音。

![](img/c9d4c177427bd1ab4b6785ac892ef6ab_5.png)

这些是获得基础流量推荐的关键技术指标。

## 导致降权限流的内容禁区 ⚠️

发布以下类型的内容，极有可能导致视频被限流，甚至账号被处罚。以下是需要严格避免的内容清单：

*   **违规信息**：内容涉及吸烟、暴力、炫富（金钱）等不良导向的信息。
*   **外部引流**：视频中出现或引导用户前往微信、微博、淘宝等外部平台联系方式。
*   **硬性广告**：长时间、明显地展示商品Logo、手机号、二维码等营销信息。请注意，二维码的出现时间**最长不应超过3秒**，且在审核严格时，即使出现1秒也可能导致视频不通过。
*   **营销行为**：进行明显的广告推销、引流卖货等行为。

特别需要注意第15点：**无口播的拆箱或好物推荐**。这是许多新手容易犯错的地方。

### 关于“无口播好物推荐”的详细说明

许多同学在操作好物推荐类账号时，会遇到小黄车（购物车）被下架的情况。主要问题出在视频的呈现方式上。

**错误示范1：直接标价**
在视频画面中直接写上“9.9元”等价格信息，这属于赤裸裸的销售行为，平台通常不会通过。

**错误示范2：纯展示无解说**
视频仅展示产品（如玩具、日用品），全程没有口播介绍，只是搭配背景音乐。这种内容容易被系统判定为低质或硬广。

**正确做法：结合口播进行软性推荐**
即使不做复杂的剧情，也**必须加上口播解说**。你可以简单地介绍产品特点或使用感受。
例如：
> “我儿子玩这个玩具，整整四个小时都没停下来。”
> “这款纸巾沾水后也不会破，可以反复使用好几次。”

通过口播将单纯的“展示”转化为“体验分享”，这样既能传达产品信息，又符合平台对优质内容的要求。切记，不要不说话直接展示产品。

![](img/c9d4c177427bd1ab4b6785ac892ef6ab_7.png)

至于直接口播广告、念联系方式等更露骨的卖货方式，在抖音平台是行不通的，更适合传统电商场景。

![](img/c9d4c177427bd1ab4b6785ac892ef6ab_8.png)

本节课中我们一起学习了抖音平台对内容的扶持规则与禁忌。核心在于，要获得流量，需使用**9:16**的竖版高清视频；要避免限流，则需规避违规信息、硬性广告和**无口播的纯产品展示**。理解并应用这些规则，是账号健康运营的第一步。
