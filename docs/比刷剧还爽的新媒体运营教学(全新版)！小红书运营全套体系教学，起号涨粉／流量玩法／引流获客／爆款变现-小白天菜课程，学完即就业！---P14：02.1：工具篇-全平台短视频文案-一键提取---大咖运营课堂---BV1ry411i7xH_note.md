# 新媒体运营教学：P14：02.1：工具篇-全平台短视频文案一键提取 📝

在本节课中，我们将学习一个非常实用的运营工具，它能帮助我们从多个平台的短视频中一键提取文案，从而极大地提高内容创作和复盘的效率。

上一节我们介绍了新媒体运营的基础概念，本节中我们来看看一个能提升工作效率的具体工具。

## 工具介绍与优势

以往，当我们想获取一个短视频的文案时，可能需要使用“轻抖”小程序，或者先将视频下载下来，再导入“剪映”等软件中进行语音识别和文案提取。这种方法不仅步骤繁琐，而且难以应对多平台（如抖音、小红书、B站、视频号、快手等）的需求。

今天分享的工具解决了这个问题。它是一个在线工具，你只需将目标短视频的链接复制到工具网页中，即可一键提取出该视频的文案。这种方法省去了下载视频的步骤，支持多个主流平台，操作简单快捷。

以下是该工具的核心优势列表：
*   **全平台支持**：可处理抖音、快手、小红书、B站、视频号等平台的视频链接。
*   **在线操作**：无需下载安装任何软件，在网页中即可完成。
*   **高效准确**：提取速度快，文案识别准确率高。
*   **功能丰富**：提取后支持一键复制和格式化排版，便于后续编辑。

## 工具使用演示

接下来，我们通过实际案例来演示这个工具的使用方法。

![](img/fa02a838117ab07e5fec1b3f043b04da_1.png)

### 提取抖音视频文案

![](img/fa02a838117ab07e5fec1b3f043b04da_3.png)

首先，我们打开抖音，找到一个想要提取文案的视频。例如，我们搜索“情感语录”相关视频。

![](img/fa02a838117ab07e5fec1b3f043b04da_5.png)

![](img/fa02a838117ab07e5fec1b3f043b04da_1.png)

![](img/fa02a838117ab07e5fec1b3f043b04da_7.png)

找到一个合适的视频后，复制其分享链接。

![](img/fa02a838117ab07e5fec1b3f043b04da_9.png)

![](img/fa02a838117ab07e5fec1b3f043b04da_3.png)

然后，我们打开该在线工具页面，将复制的链接粘贴到指定输入框中。

![](img/fa02a838117ab07e5fec1b3f043b04da_5.png)

点击“提取文案”按钮。稍等片刻，系统就会自动识别并显示出视频的完整文案。

![](img/fa02a838117ab07e5fec1b3f043b04da_7.png)

提取出的文案内容示例为：
> 听完你就释怀了，村上春树说，我动了离开你的念头，不是因为你不好，也不是因为不爱了……

我们可以点击“一键复制”直接使用文案，也可以使用“格式化”功能，让文案按照字幕的段落形式清晰排列，方便阅读和修改。

![](img/fa02a838117ab07e5fec1b3f043b04da_9.png)

此外，该工具也支持上传本地视频文件来提取文案。

### 提取B站视频文案

该工具同样适用于B站（哔哩哔哩）。我们找到一个B站视频，复制其链接地址（通常以 `BV` 号开头）。

![](img/fa02a838117ab07e5fec1b3f043b04da_11.png)

将链接粘贴到工具中并点击提取。很快，B站视频的文案也被成功提取出来，例如：
> 新手做直播电商，怎么才能打造出直播爆品？在直播销售的不同阶段……

实践证明，该工具对于B站视频的文案提取也具有很高的准确性。

## 应用场景与价值

这个工具在运营工作中具有广泛的应用场景。提取出的文案可以用于：
*   **内容复盘**：分析爆款视频的文案结构和话术。
*   **二次创作**：在获取原文案的基础上进行修改、调整，用于自己的口播视频或图文内容。
*   **素材积累**：快速建立自己的文案素材库。

它集成了多个单一平台工具的功能，一个工具就能满足大部分需求，能为我们节省大量寻找、下载和转录的时间。
