# 新媒体运营：P32：02.1：工具篇-全平台短视频文案一键提取 📝

在本节课中，我们将学习一个非常实用的运营工具，它能帮助我们从抖音、小红书、B站等多个平台的短视频中，一键提取出完整的文案内容。掌握这个工具，将极大提升你的内容创作效率。

---

上一节我们介绍了文案提取的需求，本节中我们来看看如何用一个在线工具解决多平台文案提取的难题。

以往，提取短视频文案可能需要下载视频，再借助剪映或“轻抖”等工具进行识别。这种方法步骤繁琐，且难以覆盖所有平台。

现在，有一个在线工具可以简化这个过程。你只需复制短视频的链接，粘贴到工具中，即可在线一键提取出文案。它支持抖音、快手、小红书、B站、视频号等多个主流平台。

以下是该工具的核心操作流程：

1.  **复制链接**：在目标平台（如抖音）找到你想提取文案的短视频，复制其分享链接。
2.  **粘贴提取**：打开该在线工具网页，将复制的链接粘贴到指定输入框，点击“提取文案”按钮。
3.  **获取结果**：稍等片刻，工具会自动分析视频并生成完整的文案文本。
4.  **复制与格式化**：你可以直接复制提取出的文案，也可以使用“一键格式化”功能，让文案以更清晰的分段形式呈现。
