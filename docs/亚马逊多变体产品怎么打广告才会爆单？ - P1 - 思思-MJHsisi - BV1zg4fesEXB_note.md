亚马逊广告投放：P1：多变体产品广告策略详解

在本节课中，我们将学习如何为亚马逊上的多变体产品制定有效的广告策略，以实现销量增长。多变体产品（如不同颜色、尺寸的同一商品）的广告投放有其特殊性，需要系统性的规划。

上一节我们介绍了课程主题，本节中我们来看看多变体产品的核心广告结构。

对于多变体产品，建议采用“**父体广告 + 子体广告**”的组合结构。父体广告用于测试和引流，子体广告用于精准转化和优化。

以下是具体的广告活动搭建步骤：

![](img/02e6633c300bd9ff008a9f3de2f0f683_1.png)

1.  **创建父体商品定位广告活动**：此活动以父体ASIN为定位目标，目的是测试整个产品系列中哪些子体表现更佳。
2.  **创建子体自动广告活动**：为每个表现优异的子体单独开设自动广告活动，用于收集广泛的关键词和受众数据。
3.  **创建子体手动广告活动**：基于自动广告收集的数据，为每个子体创建手动关键词广告，进行精准投放和预算分配。

接下来，我们将深入探讨第一步：父体广告活动的设置。

父体广告活动的主要目标是进行市场测试，发现潜力子体。其预算设置应遵循“**测试期总预算 = 测试子体数量 × 单子体测试预算**”的原则。

例如，若一个父体下有10个子体，你计划为每个子体分配10美元的测试预算，那么该父体广告活动的总预算应设置为100美元。此阶段建议使用商品定位，并选择“同类商品”或“关联商品”作为定位方式。

在通过父体广告识别出表现优秀的子体后，下一步就是为这些“明星子体”建立独立的广告活动。

![](img/02e6633c300bd9ff008a9f3de2f0f683_3.png)

为潜力子体开设独立的自动广告活动，目的是让系统更深入地学习该特定产品的相关搜索词和受众。每个子体的自动广告应设置独立的预算，例如每天5-10美元，以便清晰追踪数据。

当子体自动广告运行一段时间并积累足够数据后，我们就可以进入更精细化的投放阶段。

基于自动广告报告中的高绩效搜索词，为每个子体创建手动关键词广告活动。在此类活动中，你可以：
*   将表现最好的关键词放入**精准匹配**组，确保流量高度相关。
*   将表现一般或相关的词放入**词组匹配**或**广泛匹配**组，用于拓展流量。
*   对不同广告组或关键词设置不同的竞价，将预算向转化率更高的方向倾斜。

最后，我们来总结一下整个流程的关键点。

本节课中我们一起学习了多变体产品的广告投放体系。核心路径是：**通过父体广告测试选款 → 为优胜子体开设自动广告收集数据 → 基于数据搭建精准的手动关键词广告**。这套方法旨在系统化地管理多变体广告，实现预算的有效分配和广告效率的最大化。
