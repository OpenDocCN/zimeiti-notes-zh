# 微信视频号运营从0~1全攻略：P35：直播时发布草稿箱视频与引流技巧

![](img/8f4b935a2d9e429ec1cb8d158bb11362_0.png)

在本节课中，我们将学习在微信视频号直播时，如何发布预先准备好的草稿箱视频，以及如何通过分享直播间来吸引更多观众。这些技巧对于提升直播间的初始人气和互动至关重要。

## 直播中发布草稿箱视频

![](img/8f4b935a2d9e429ec1cb8d158bb11362_2.png)

上一节我们介绍了直播前的准备工作，本节中我们来看看如何在直播开始后，通过发布视频来吸引观众。

![](img/8f4b935a2d9e429ec1cb8d158bb11362_4.png)

当直播开始后，如果发现直播间观众较少，可以按照以下步骤操作。

以下是具体的操作步骤：

![](img/8f4b935a2d9e429ec1cb8d158bb11362_6.png)

1.  点击直播间界面右上角的**三个点**菜单按钮。
2.  在弹出的菜单中，选择**最小化直播间**的选项。
3.  此时，返回视频号的主页界面。
4.  在主页中，找到并点击进入**草稿箱**。
5.  选择你**提前准备好的视频**，点击**发表**按钮。

![](img/8f4b935a2d9e429ec1cb8d158bb11362_8.png)

> 注意：在发表时，系统可能会提示“正在直播中，请稍后”，但视频**依然可以成功发布**。请勿直接在直播界面内点击“发表视频”按钮，该路径在直播中通常无法使用。

## 直播间的分享与引流

![](img/8f4b935a2d9e429ec1cb8d158bb11362_10.png)

成功发布视频可以吸引一部分观众，但主动分享直播间是更直接的引流方式。接下来，我们看看如何有效地将直播间分享出去。

![](img/8f4b935a2d9e429ec1cb8d158bb11362_12.png)

以下是几种有效的分享方法：

*   **分享至私域流量**：将直播间链接分享到你的**微信朋友圈**，或直接**转发给微信好友**。
*   **利用社群引流**：提前准备一些微信群（如业主群、兴趣群、粉丝群等），在直播时将链接分享到这些群中。
*   **使用小号辅助**：如果你不希望用主号在某些群内分享，可以**使用微信小号**加入相关群组并进行分享。

> 官方对于私域引流有流量扶持。例如，如果你通过私域（微信群、朋友圈）为直播间引入了10位观众，平台通常会从公域再为你推荐约10位观众。

![](img/8f4b935a2d9e429ec1cb8d158bb11362_14.png)

## 直播表现自查

如果执行了以上所有操作后，直播间依然无人或观众停留时间很短，就需要审视自身的直播表现。

以下是需要自查的几个方面：

*   **直播话术**：你的开场白和互动话术是否吸引人？这需要通过反复练习来提升。
*   **个人状态**：你是否精神饱满、充满热情？观众进入一个死气沉沉的直播间通常会立刻离开。
*   **内容与产品**：在优化外部引流的同时，也要持续打磨直播内容和选品。

## 总结

本节课中我们一起学习了两个核心的直播运营技巧。

首先，我们掌握了在直播中发布草稿箱视频的方法，其关键路径是：**最小化直播间 → 进入视频号主页 → 从草稿箱发表视频**。

其次，我们了解了如何通过**分享至私域和社群**来为直播间引流，并知晓了官方对此的流量扶持政策。

最后，我们认识到，当外部引流效果不佳时，应回归到对自身**直播话术、状态和内容**的优化上。将这些方法结合运用，能有效提升直播间的初始人气和整体效果。
