# 视频号运营教程：P24：直播时如何发布草稿箱视频并提升人气

![](img/a2e1bc37375b3437f8a3c82a8a140dac_0.png)

在本节课中，我们将学习一个非常实用的直播技巧：如何在直播过程中，将提前准备好的草稿箱视频发布出去，以吸引更多观众进入直播间。同时，我们也会探讨一些提升直播间初始人气的有效方法。

![](img/a2e1bc37375b3437f8a3c82a8a140dac_2.png)

## 直播中发布草稿箱视频的步骤

![](img/a2e1bc37375b3437f8a3c82a8a140dac_4.png)

上一节我们介绍了直播前的准备工作，本节中我们来看看如何在直播进行时，巧妙地利用草稿箱内容为直播间引流。

![](img/a2e1bc37375b3437f8a3c82a8a140dac_6.png)

当直播开始后，如果发现直播间观众较少，可以按照以下步骤操作：

![](img/a2e1bc37375b3437f8a3c82a8a140dac_8.png)

1.  **最小化直播间**：点击直播间右上角的三个点（`...`）菜单，选择将直播间最小化。
2.  **进入视频号主页**：返回手机主界面，找到并打开你的视频号个人主页。
3.  **发布草稿箱视频**：在主页中，找到你提前准备好的草稿箱视频，点击“发表”。

![](img/a2e1bc37375b3437f8a3c82a8a140dac_10.png)

**核心操作**可以概括为：`最小化直播间 -> 进入个人主页 -> 发布草稿箱视频`。

![](img/a2e1bc37375b3437f8a3c82a8a140dac_12.png)

需要注意的是，在直播过程中，如果直接点击底部导航栏的“发表”按钮来发布新视频，系统会提示“正在直播中，请稍后”并无法成功。但通过上述“绕道”个人主页的方法，视频可以正常发布出去。

## 提升直播间人气的辅助方法

![](img/a2e1bc37375b3437f8a3c82a8a140dac_14.png)

学会了发布视频的技巧后，我们还需要主动为直播间引入初始流量。以下是几种行之有效的方法：

1.  **分享直播间**：利用直播界面自带的分享功能，将直播间链接分享到你的微信朋友圈，或直接转发给好友。
2.  **利用社群导流**：提前准备一些微信群（如业主群、兴趣群、粉丝群等），在直播时将链接分享到这些群里。如果不想打扰自己的主要社交圈，可以使用小号进行操作。
3.  **利用平台扶持**：视频号官方有流量扶持机制。简单来说，如果你能从自己的私域（如微信群、朋友圈）引导一定数量的用户进入直播间，平台会相应地为你从公域推荐更多观众。其**核心逻辑**可以理解为：`私域导入流量 ≈ 公域推荐流量`。

## 直播表现与持续优化

如果尝试了以上所有方法后，直播间人气依然不高，或者观众进入后很快离开，那么可能需要从自身直播表现上寻找原因。

直播是一项需要练习的技能。请检查自己的直播话术是否吸引人，直播状态是否充满激情。如果主播表现得死气沉沉，观众很可能会立刻退出。持续反思和优化自己的直播内容与状态，是提升直播效果的根本。

本节课中我们一起学习了两个关键技能：第一，掌握了在直播中通过个人主页发布草稿箱视频的具体操作流程；第二，了解了通过分享和社群导流为直播间引入初始人气的方法。记住，成功的直播是充分准备与积极运营共同作用的结果。
