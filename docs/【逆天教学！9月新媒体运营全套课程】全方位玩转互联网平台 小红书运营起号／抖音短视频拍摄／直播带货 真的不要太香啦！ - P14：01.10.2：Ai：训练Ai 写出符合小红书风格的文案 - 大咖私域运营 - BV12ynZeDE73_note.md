# 新媒体运营：P14：01.10.2：训练AI写出符合小红书风格的文案 ✍️🤖

在本节课中，我们将学习如何训练AI（如ChatGPT）来撰写符合小红书平台风格的文案。上一节我们介绍了如何让AI生成爆款标题，本节我们将重点探讨如何让AI理解并创作出符合平台调性的完整内容。

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_1.png)

## 概述：训练AI的核心思路

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_3.png)

让AI写出符合特定平台风格的内容，关键在于“投喂”和“训练”。你需要提供该平台的真实案例，让AI从中学习文案的风格、语气、常用词汇和内容结构。这个方法不仅适用于小红书，也适用于抖音、B站等其他拥有独特社区文化的平台。

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_5.png)

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_14.png)

要让AI了解你需要的风格，你必须提供明确的指令和具体的案例。你需要告诉AI，你提供的是小红书平台的爆款笔记文案，并要求它分析这些文案的撰写风格、平台专用词汇、语气等特点。

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_16.png)

以下是给AI的初始指令示例：

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_18.png)

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_20.png)

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_22.png)

```
接下来我会给你提供一些小红书上面的爆款笔记文案。你需要根据这些文案，分析小红书上面的文案撰写风格、特定的平台用语、词汇、语气等。
每一次提供文案后，如果你理解了，请告诉我你学习到了什么，并给出你对这篇文案的风格、语气、类型等相关分析。
```

这个指令明确了AI的任务：学习你提供的资料，并给出反馈，确保它确实在分析和记忆。

## 第二步：筛选与投喂高质量文案

训练效果的好坏，很大程度上取决于你“投喂”的文案质量。你需要寻找具有鲜明小红书风格的文案作为学习材料。

以下是筛选文案时需要注意的几个要点：

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_29.png)

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_31.png)

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_32.png)

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_33.png)

*   **人称与视角**：观察文案是以第一人称分享（“我”的经历），还是以第二人称建议（“你”可以这样做）的方式撰写。小红书笔记多以个人真实经历分享为主。
*   **语气与用词**：注意文案中是否使用了平台特色用语（如“宝子”、“集美”、“绝绝子”、“踩雷”、“安利”等），以及是否带有少女感、亲切感或吐槽语气。
*   **内容结构**：分析文案是单纯的清单罗列（如第一天、第二天去哪），还是包含了个人体验、情感共鸣、避坑建议等更有深度的内容。后者才是训练的重点。
*   **多样性**：尽量提供不同题材（如旅游攻略、美妆种草、知识分享、探店打卡）和不同语气（如兴奋分享、真诚推荐、幽默吐槽）的文案，让AI学习更全面。

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_35.png)

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_37.png)

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_39.png)

**实操演示**：
1.  找到一篇小红书爆款笔记（例如：“张家界旅游血泪教训！看完这篇不踩坑！”）。
2.  将标题和正文内容清晰地发送给AI。
3.  AI会进行分析并反馈，例如：“本文案以个人游玩经历为主线，增加了真实感和亲近感。标题使用‘血泪教训’和‘不踩坑’等词汇，强调了经验分享和价值。语气亲切，像朋友间的建议。”

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_41.png)

持续投喂多篇不同类型的高质量文案，AI的“知识库”就会不断丰富。

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_44.png)

## 第三步：基于学习成果生成内容

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_48.png)

当AI学习了足够多的案例后，你就可以要求它基于所学知识进行创作。

你可以使用如下指令：

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_51.png)

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_52.png)

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_53.png)

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_54.png)

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_56.png)

```
请结合你的学习，帮我写一篇具有爆款潜质的[你的主题，例如：张家界5日游攻略]小红书笔记。内容需涵盖丰富的小红书风格元素，同时配上10个符合小红书[你的赛道，例如：旅游赛道]的爆款标题。
```

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_57.png)

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_59.png)

**生成结果分析**：
AI生成的初稿可能不尽如人意。例如，一篇旅游攻略可能只罗列了景点名称，而缺乏具体的路线规划、个人体验、避坑建议等有血有肉的内容。这说明训练可能还存在不足。

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_61.png)

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_62.png)

**问题与优化**：
*   **素材局限性**：如果投喂的文案本身深度不够，AI生成的内容也会流于表面。需要寻找更优质、更具分析性的文案。
*   **指令细化**：可以要求AI先列出某个主题下的关键点（如张家界所有景点），再对每个点进行详细规划和优缺点分析，最后整合成文。
*   **模型差异**：高级模型（如GPT-4）在理解复杂指令和生成高质量内容方面通常优于基础模型（如GPT-3.5）。

## 关键技巧与注意事项

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_68.png)

![](img/1bf6774d9d46b244b9beec6e6f1e56ed_70.png)

1.  **分场景对话**：对于不同的任务（如写标题、写正文），建议开启新的对话窗口，并输入针对该任务的专用指令库，避免之前对话的干扰。
2.  **持续训练**：AI训练不是一蹴而就的。建立一个属于你自己的高质量文案库，并持续投喂，AI会越来越懂你的要求。
3.  **结合专用指令**：生成标题时，最好结合上一节课提到的“爆款标题指令公式”和本赛道的关键词库，这样生成的标题会更精准。
4.  **人工润色**：AI是强大的辅助工具，但生成的内容仍需你进行审核、修改和润色，使其更符合你的个人风格和具体需求。

## 总结

本节课我们一起学习了训练AI撰写平台风格文案的全流程。核心步骤是：**明确指令 -> 投喂高质量案例 -> 基于学习成果生成 -> 优化与润色**。关键在于通过提供精准的案例，让AI从一个“通才”转变为理解你所在领域和平台调性的“专家”。记住，AI是你的助手，熟练运用这些方法，将能极大提升你在小红书等平台的内容创作效率和质量。
