# 抖音运营教程：P17：如何寻找文案-1 📝

在本节课中，我们将要学习如何为抖音视频寻找和准备优质的文案。这是优化作品、吸引粉丝的关键第一步。我们将介绍几个实用的文案工具网站，并讲解如何利用它们解决文案创作和配音问题。

---

上一节我们讨论了优化作品的重要性，本节中我们来看看如何具体解决文案来源的问题。很多初学者不会写文案，这没有关系，我们可以借助一些现成的工具和资源。

以下是五个实用的文案资源网站，它们都是网页版，无需下载APP，在手机或电脑浏览器中搜索即可使用。

1.  淘金阁
2.  （其他四个网站名称在视频中未具体列出，但方法通用）

---

我们以“淘金阁”为例进行详细讲解。你可以通过其网址直接访问，避免搜索时遇到过多广告。

![](img/51ecd27709d956700919aad166b55f97_1.png)

进入淘金阁主页后，你会看到它按类别收录了海量文案素材。

![](img/51ecd27709d956700919aad166b55f97_1.png)

类别包括财经、军事、动漫、科技、美食、美文、星座、体育、旅游、育儿、汽车、社会、娱乐、时尚、房产、国际、宠物等，非常全面。

![](img/51ecd27709d956700919aad166b55f97_3.png)

例如，我们点击“情感”类别。

![](img/51ecd27709d956700919aad166b55f97_3.png)

页面会展示大量情感类话题文案，并且通常有多页内容可供浏览。这五个平台提供的文案资源，足以满足日常创作需求。

![](img/51ecd27709d956700919aad166b55f97_5.png)

当你面对众多文案不知如何选择时，可以套用“一个群体 + 一个核心话题 + 疑问”的公式来筛选你觉得合适的内容。

![](img/51ecd27709d956700919aad166b55f97_5.png)

使用这个方法，你基本不会再缺文案。

![](img/51ecd27709d956700919aad166b55f97_7.png)

---

有的同学可能会发现，现有分类里没有自己想要的内容。这时有两种解决方法。

![](img/51ecd27709d956700919aad166b55f97_9.png)

有条件的同学可以开通会员，使用顶部的搜索功能。不想开通会员的同学，可以留意页面下方的“输入多个关键词”搜索框。

例如，我们搜索“宝妈”这个关键词。

![](img/51ecd27709d956700919aad166b55f97_7.png)

![](img/51ecd27709d956700919aad166b55f97_11.png)

点击“开始抓取”后，系统会列出所有包含“宝妈”相关内容的文案。这个方法在手机和电脑上操作完全一致，非常方便。

![](img/51ecd27709d956700919aad166b55f97_13.png)

![](img/51ecd27709d956700919aad166b55f97_9.png)

解决了文案来源，我们再来解决另一个常见问题：担心自己声音不好听，不想亲自配音。

淘金阁还提供了一个强大的免费工具：在线配音生成。点击页面右上角的“小工具”图标。

![](img/51ecd27709d956700919aad166b55f97_15.png)

![](img/51ecd27709d956700919aad166b55f97_11.png)

![](img/51ecd27709d956700919aad166b55f97_17.png)

在工具列表中，选择第四个“配音在线生成”功能。

![](img/51ecd27709d956700919aad166b55f97_19.png)

![](img/51ecd27709d956700919aad166b55f97_13.png)

这个功能完全免费，且配音种类极其丰富。

以下是部分配音类型示例：
*   **女声类型**：亲和女生、温柔女生、甜美女生、自然女声、卖场广播等。
*   **直播配音**：亲切女声、活力女声。
*   **通用场景**：标准男声、标准女生。
*   **特色配音**：湖南口音、童声、英文、四川话、粤语、东北话、台湾话、外语，以及情感解说、影视解说、娱乐解说等精品男配。

![](img/51ecd27709d956700919aad166b55f97_15.png)

使用方法很简单：
1.  在左侧“配音文本”框中输入或粘贴你的文案。
    ![](img/51ecd27709d956700919aad166b55f97_17.png)
2.  在右侧选择你喜欢的声音类型，例如选择“东北话”。
    ![](img/51ecd27709d956700919aad166b55f97_19.png)
3.  调整音量、语速和语调参数。
    ![](img/51ecd27709d956700919aad166b55f97_21.png)
4.  点击“立即生成”。
    ![](img/51ecd27709d956700919aad166b55f97_23.png)

![](img/51ecd27709d956700919aad166b55f97_21.png)

在手机上使用时，可以截图然后用微信扫码登录，即可免费使用所有配音功能。

![](img/51ecd27709d956700919aad166b55f97_23.png)

![](img/51ecd27709d956700919aad166b55f97_25.png)

生成后，在“配音生成列表”中可以直接下载音频文件到手机。

![](img/51ecd27709d956700919aad166b55f97_25.png)

![](img/51ecd27709d956700919aad166b55f97_27.png)

下载的音频可以在后续剪辑环节直接使用。具体的剪辑方法我们会在后面的课程中讲解。

![](img/51ecd27709d956700919aad166b55f97_27.png)
