# 抖音运营入门：P13：3、如何完善个人资料 🧑‍💼

在本节课中，我们将要学习如何完善抖音的个人资料。一个专业且吸引人的个人主页是吸引粉丝、建立信任的第一步。我们将从头像、昵称、个人简介到主页联系方式等细节，逐一拆解，帮助你打造一个符合平台规则且能有效涨粉的个人资料。

## 头像选择

上一节我们介绍了个人资料的重要性，本节中我们来看看如何选择头像。头像应清晰、有辨识度，并能体现你的账号定位。

以下是头像选择的几种常见方案：
*   **有实体店**：使用实体店铺的照片。
*   **无实体店**：使用个人清晰的自拍照或生活照。
*   **宝妈类型**：使用你和宝宝的合照。
*   **宠物类型**：使用宠物的照片。

## 昵称设置

设置好头像后，一个关键的环节是起一个好名字。昵称是用户对你的第一印象，直接影响账号的辨识度和搜索。

**核心原则**：**不要使用自己的真实姓名**（如“小月”、“兰亭”）。抖音平台会将其判定为普通用户而非创作者，从而影响流量推荐。同时，暴露个人信息也无意义。

**命名规则**：
1.  **字数**：控制在五个字以内。
2.  **字符**：避免使用繁体字和特殊符号（如 ❤️）。这不利于用户记忆和搜索。

**命名方法**：
以下是三种有效的命名公式，你可以选择其一：

*   **公式一：昵称 + 领域定位**
    *   `小月说游戏`
    *   `小月小厨房`
    *   `小月聊房`
    *   `月姐穿搭`

*   **公式二：昵称 + 职业身份**
    *   `宝妈小月`
    *   `律师小月`
    *   `摄影师小月`

*   **公式三：昵称 + 所在地区**
    *   `赣州月姐`（适合三农、本地生活类）
    *   `小月在北漂`

## 个人简介撰写

![](img/b4e3ed38273cadd676e9c6d83257f204_1.png)

昵称确定后，我们需要用简短的文字进一步介绍自己。个人简介是说服用户关注你的关键。

**核心要素**：个人简介应围绕以下三点展开：
1.  **我是谁**：例如，“中年大叔依然保持母婴初衷”。
2.  **我是做什么的**：例如，“从事母婴行业13年，做母婴商贸8年有余”。
3.  **关注我能得到什么**：例如，“分享育儿经验”。

![](img/b4e3ed38273cadd676e9c6d83257f204_3.png)

你可以根据自己的性格（幽默、可爱等）来调整写作风格，使其更生动活泼。

## 主页联系方式注意事项

完善了基本资料，最后要注意主页的“联系方式”区域。这是一个容易违规的地方。

**重要规则**：**不要在主页擅自挂载微信或手机号等站外联系方式**。这是跨平台引流行为，违反抖音规则。

**例外情况**：
*   已开通**蓝V企业号**的实体店商家可以挂载联系电话。
*   粉丝达到**1万**并成功入驻**星图平台**后，可按平台规则挂载联系方式（关于星图变现，后续课程会讲解）。

如果已经违规挂载，请立即删除。
