# 抖音运营教程：P11：产品与用户兴趣标签 🧩

在本节课中，我们将学习一种名为“九宫格选题法”的实用技巧。该方法通过结合产品属性与用户兴趣标签，帮助创作者系统性地挖掘和生成源源不断的视频选题，解决“不知道拍什么”的难题。

上一节我们介绍了选题的重要性，本节中我们来看看如何具体操作。

## 核心方法：九宫格选题法

该方法的核心是将一个核心概念（产品或用户）置于中心，围绕它联想出八个相关标签，然后通过“连连看”的方式组合这些标签，从而生成创意选题。

## 第一步：分析产品标签

我们以健身软件 **Keep** 为例进行演示。首先，将产品本身放在九宫格的中心位置。

以下是围绕 **Keep** 这个产品，我们可以联想到的八个特点或属性：

1.  **线上健身软件**：产品的核心形态。
2.  **专业健身教练**：产品提供的核心服务之一。
3.  **饮食搭配建议**：产品提供的增值服务。
4.  **减肥攻略**：产品解决的主要用户需求之一。
5.  **健身干货知识**：产品提供的教育内容。
6.  **科学运动方法**：产品的专业性体现。
7.  **打卡运动制度**：产品的用户激励和社交机制。
8.  **一二线城市用户为主**：产品的主要用户画像特征。

完成这一步后，产品相关的标签就填充完毕了。

## 第二步：分析用户兴趣标签

接下来，我们分析产品的用户。将核心用户（例如“健身爱好者”）放在另一个九宫格的中心。

以下是围绕“健身爱好者”这个用户群体，我们可以联想到的八个兴趣标签：

![](img/353c15146dc56d719c937afef3e3ef0d_1.png)

1.  **健身房**：线下健身的场所。
2.  **健身装备**：如运动鞋、紧身衣等。
3.  **轻脂美食/减肥餐**：与健身目标相关的饮食兴趣。
4.  **旅行**：作为热爱生活的一种表现。
5.  **星座**：常见的泛娱乐兴趣。
6.  **明星**：大众娱乐的关注点。
7.  **鸡汤文**：情感激励类内容。
8.  **时尚穿搭**：与外形、身材管理相关的兴趣。

通过这一步，我们将用户群从“健身爱好者”扩展到了更广泛的“生活热爱者”，为选题提供了更丰富的交叉点。

![](img/353c15146dc56d719c937afef3e3ef0d_1.png)

## 第三步：进行“连连看”组合选题

现在，我们拥有了两个填满的九宫格：一个产品标签格，一个用户兴趣标签格。接下来就是创造性地将两边的标签进行组合，生成具体的视频选题。

以下是几个组合示例：

*   **组合1：饮食搭配 + 旅行**
    *   **生成的选题**：“外出旅行时，如何享受美食又能控制体重？”
*   **组合2：运动方法 + 明星**
    *   **生成的选题**：“赵丽颖都在练的5个高效瘦身动作，你一定要学！”
*   **组合3：科学 + 健身房**
    *   **生成的选题**：“用最科学的方法，让你在健身房的每一分钟都高效。”

通过这种“连连看”的方式，理论上可以组合出无数个选题方向。当自己灵感枯竭时，可以借助搜索引擎，输入组合出的关键词（如“健身 科学”），查看相关的长尾词或热门话题，进一步激发灵感。

## 如何判断选题的好坏？

生成了大量选题后，我们需要进行筛选。一个好的选题通常需要具备以下几点：

1.  **符合逻辑**：标签之间的连接要自然、合理，不能生搬硬套。
2.  **吸引目标用户**：选题必须能戳中产品核心用户（健身爱好者）的兴趣点。
3.  **具有可操作性**：选题要能具体落地成可拍摄的视频内容。

例如，“在健身房如何科学地交友”这个选题，虽然结合了“健身房”和社交（可视为泛兴趣），但其与核心产品“科学健身”的关联度较弱，可能不是最优选。

## 总结

![](img/353c15146dc56d719c937afef3e3ef0d_3.png)

本节课中我们一起学习了“九宫格选题法”。我们首先分别梳理了**产品自身的属性标签**和**目标用户的兴趣标签**，然后将它们填入两个九宫格。最后，通过**跨格子的“连连看”组合**，我们能够系统性地碰撞出大量潜在的视频选题，并建立自己的选题素材库。这个方法能有效解决内容枯竭的问题，让你的账号创作思路源源不断。
