# 抖音运营入门：P6：扶持加权与降权限流

在本节课中，我们将学习抖音平台对内容的扶持与限制规则。了解哪些内容能获得流量推荐，哪些内容会导致账号被降权或限流，是成功运营抖音账号的基础。

![](img/6c7e123bf3173deb76bd7d014629a0e4_1.png)

上一节我们介绍了账号权重的概念，本节中我们来看看具体哪些行为会影响账号的权重。

## 🚀 获得扶持加权的内容

![](img/6c7e123bf3173deb76bd7d014629a0e4_3.png)

平台会优先推荐符合以下标准的优质内容：

![](img/6c7e123bf3173deb76bd7d014629a0e4_5.png)

*   **视频格式**：竖版（9:16）的移动端原生视频。
*   **画面质量**：画面平稳，作品清晰。
*   **音频质量**：无过多杂音或噪音。

这些是平台鼓励的基础创作规范，遵循它们有助于视频进入推荐流量池。

## ⚠️ 导致降权限流的内容

以下是平台明确禁止或限制的内容类型，发布此类内容将面临审核不通过、限流甚至处罚的风险。

以下是具体会被降权限流的内容列表：

1.  **违规信息**：视频内容涉及吸烟、暴力、过度宣扬金钱等不良信息。
2.  **外部引流**：在视频或文案中留下微信、微博、淘宝等外部联系方式。
3.  **商业标识**：长时间、明显地展示商品Logo、手机号或二维码。二维码的出现时间建议不超过3秒，但审核严格时，即使出现1秒也可能被驳回。
4.  **硬性广告**：进行明显的营销广告、推销或引流卖货行为。

特别需要注意“无口播拆箱或好物推荐”类内容。如果只是单纯展示产品、拆箱而不做任何介绍，很容易被判定为营销广告。

**案例分析**：
*   **错误示范1**：视频中直接出现“9.9元”等价格信息，意图直接卖货。这属于明显的硬广，小黄车功能通常无法通过审核。
*   **错误示范2**：仅用背景音乐展示玩具，没有任何口头介绍。这种“默片”式推荐也容易被系统识别为低质营销内容。
*   **正确做法**：即使不做复杂剧情，也应加入简单口播。例如：“我儿子玩这个玩具，专注地玩了四个小时。” 通过分享真实使用体验来软化广告属性，这样更容易通过审核。

5.  **直接口播广告**：在视频中直接口播联系方式或广告语。这种方式在抖音行不通，更适合传统电商平台。

本节课中我们一起学习了抖音平台的“红线”与“绿灯”。核心在于理解平台鼓励**原生、优质、有价值的内容分享**，反对**生硬、直接、干扰用户体验的营销行为**。掌握这些规则，能帮助你在创作时有效规避风险，让账号获得更好的成长空间。
