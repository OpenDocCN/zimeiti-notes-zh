## 店铺类型总览

小红书的店铺体系主要分为以下几种类型：个体店、个体工商户店、普通企业店、专卖店、旗舰店以及卖场型旗舰店。这些类型主要根据申请主体和资质要求进行区分。

以下是各类店铺的详细说明：

*   **个体店**
    *   **定义**：申请主体为个人用户。类似于现实中的“摆地摊”，适合个人创业者、刚毕业的学生或想尝试副业的人士。
    *   **经营特点**：可经营限定类目下的零星、小额商品，如首饰、服装、手机壳等。可销售多个品牌，类似“杂货铺”。
    *   **入驻要求**：非常简单，仅需个人身份证。
    *   **店铺命名**：格式为“专业号名称 + 自定店名”，系统会自动匹配。
    *   **保证金**：涉及**类目保证金**和**浮动保证金**，二者取高。基础保证金通常为1000元。

*   **个体工商户店**
    *   **定义**：与个体店的主要区别在于申请主体需持有**营业执照**。
    *   **经营特点**：与个体店类似。
    *   **入驻要求**：需要个体工商户营业执照，无需品牌授权。
    *   **其他**：保证金规则等与个体店基本一致。

![](img/d768a530fc987afbc3410951a2c0f51e_6.png)

*   **普通企业店**
    *   **定义**：申请主体为企业。
    *   **经营特点**：可经营企业自有产品或授权产品。
    *   **入驻要求**：需持有企业的营业执照。
    *   **提示**：对于运营新手，初期建议从个人店或个体店开始测试，无需投入过高成本。

![](img/d768a530fc987afbc3410951a2c0f51e_8.png)

*   **专卖店**
    *   **定义**：经营单一品牌（自有或授权）的店铺。
    *   **入驻要求**：需营业执照及品牌认可文件（自有品牌需商标注册证，授权品牌需品牌授权书）。

*   **旗舰店**
    *   **定义**：通常指品牌官方旗舰店，是品牌形象的代表。
    *   **入驻要求**：需营业执照及品牌授权书（若为品牌方直接开设，则为自有品牌证明）。

![](img/d768a530fc987afbc3410951a2c0f51e_10.png)

![](img/d768a530fc987afbc3410951a2c0f51e_12.png)

*   **卖场型旗舰店**
    *   **定义**：具有零售资质，可开设并经营多个品牌旗舰店的店铺。
    *   **现状**：此类店铺在小红书平台较少见，因为小红书以女性用户为主，流量特性与大型综合卖场不完全匹配。这反而为中小卖家提供了更友好的竞争环境。

## 保证金规则详解 💰

了解保证金规则对店铺资金管理至关重要。保证金主要分为**类目保证金**和**浮动保证金**。

### 类目保证金

![](img/d768a530fc987afbc3410951a2c0f51e_14.png)

类目保证金根据经营的商品类目固定收取。以下是部分类目示例（具体金额以平台最新规则为准）：

![](img/d768a530fc987afbc3410951a2c0f51e_16.png)

*   **普通企业类目**：基础保证金。
*   **特殊类目**：如食品、化妆品等，可能有更高保证金要求。
*   **跨境类目**：经营进口商品需缴纳相应保证金。

### 浮动保证金

浮动保证金是根据店铺的月成交金额动态计算的，这是需要重点关注的规则。

其计算逻辑如下：
`浮动保证金 = f(月成交金额)`

具体阶梯如下：

*   **月成交金额 ≤ 5万元**：浮动保证金为 **0元**（但需先缴纳基础保证金）。
*   **5万元 < 月成交金额 ≤ 10万元**：需补足保证金至 **3000元**。
*   **10万元 < 月成交金额 ≤ 50万元**：需补足保证金至 **5000元**。
*   **月成交金额 > 50万元**：需补足保证金至 **20000元**。

**重要提醒**：平台会在提现时校验保证金账户余额是否充足。如果店铺月成交额超过5万元，但保证金总额未达到对应阶梯要求，则无法成功提现。务必定期检查并确保保证金充足。
