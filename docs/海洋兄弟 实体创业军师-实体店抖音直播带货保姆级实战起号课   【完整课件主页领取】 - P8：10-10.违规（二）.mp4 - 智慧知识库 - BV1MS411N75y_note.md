## 直播行为雷区

![](img/7783422d9d71f1886c8e2fef85e288d6_15.png)

直播违规分为三个等级：**一级违规**、**二级违规**、**三级违规**。其中，一级违规最为严重，处罚也最严厉。

![](img/7783422d9d71f1886c8e2fef85e288d6_17.png)

![](img/7783422d9d71f1886c8e2fef85e288d6_19.png)

以下是直播间中容易被忽略的几个一级违规行为：

![](img/7783422d9d71f1886c8e2fef85e288d6_21.png)

![](img/7783422d9d71f1886c8e2fef85e288d6_23.png)

1.  **人证不符**
    *   含义：指抖音账号的实名认证身份证信息与直播出镜人不是同一人。
    *   后果：属于一级违规，平台有权封禁账号。虽然目前可能未普遍处罚，但一旦被举报或平台查实，封号风险极高。
    *   解决方案：尽量本人出镜直播。若本人不便，可考虑认证**企业号**。

![](img/7783422d9d71f1886c8e2fef85e288d6_25.png)

![](img/7783422d9d71f1886c8e2fef85e288d6_27.png)

2.  **大面积裸露纹身**
    *   说明：并非禁止展示纹身，而是禁止大面积裸露。具体尺度由人工审核判定。
    *   建议：通常小于12厘米的纹身问题不大。若纹身面积较大，直播时应穿着长袖衣物等进行遮挡。

![](img/7783422d9d71f1886c8e2fef85e288d6_29.png)

![](img/7783422d9d71f1886c8e2fef85e288d6_31.png)

3.  **利益诱导**
    *   含义：在直播中鼓动、诱导观众刷礼物。
    *   现状：平台对此类行为的监管日趋严格，需绝对避免。

![](img/7783422d9d71f1886c8e2fef85e288d6_33.png)

![](img/7783422d9d71f1886c8e2fef85e288d6_35.png)

4.  **穿着暴露**
    *   说明：指通过暴露的着装打擦边球，特别是结合某些舞蹈动作时，易被平台判定为存在挑逗行为。此行为绝对禁止。

![](img/7783422d9d71f1886c8e2fef85e288d6_37.png)

![](img/7783422d9d71f1886c8e2fef85e288d6_39.png)

5.  **直播间内抽烟、喝酒**
    *   规则：直播过程中展示抽烟、喝酒行为是不允许的。一旦被系统检测到，很可能导致直播中断或账号处罚。

![](img/7783422d9d71f1886c8e2fef85e288d6_41.png)

![](img/7783422d9d71f1886c8e2fef85e288d6_43.png)

6.  **挂假人（虚假人气）**
    *   说明：通过技术手段为直播间制造虚假在线人数。
    *   后果：一旦被平台检测到，将受到处罚。且对于卖货直播间而言，挂假人对实际销售并无帮助。
    *   正确做法：应专注于用心打造 **人（主播）、货（产品）、场（直播间场景）** 的核心竞争力。

![](img/7783422d9d71f1886c8e2fef85e288d6_45.png)

了解了行为上的禁区后，接下来我们看看在直播话术上需要注意哪些问题。

![](img/7783422d9d71f1886c8e2fef85e288d6_47.png)

## 直播话术雷区

![](img/7783422d9d71f1886c8e2fef85e288d6_49.png)

![](img/7783422d9d71f1886c8e2fef85e288d6_51.png)

![](img/7783422d9d71f1886c8e2fef85e288d6_53.png)

在直播或发布视频时，一旦提及某些敏感词汇，系统可能立即检测并触发处罚。

![](img/7783422d9d71f1886c8e2fef85e288d6_55.png)

![](img/7783422d9d71f1886c8e2fef85e288d6_57.png)

以下是常见的话术雷区：

![](img/7783422d9d71f1886c8e2fef85e288d6_59.png)

![](img/7783422d9d71f1886c8e2fef85e288d6_61.png)

1.  **政治敏感言论**
    *   规则：在任何情况下都绝对禁止讨论政治敏感话题。无论在抖音的哪个功能板块，触碰此红线都会导致严重的封号后果。

![](img/7783422d9d71f1886c8e2fef85e288d6_63.png)

![](img/7783422d9d71f1886c8e2fef85e288d6_65.png)

2.  **过度“喊关注”**
    *   现状：抖音平台对主播过度、频繁地引导用户点击关注的监管越来越严格，需注意话术的合理性。

![](img/7783422d9d71f1886c8e2fef85e288d6_67.png)

3.  **不文明用语**
    *   说明：包括脏话、侮辱性词汇（如“智障”等口头禅）。
    *   风险：此类用语容易被直播间观众举报，举报成功率较高，可能导致直播被中断或收到官方警告。

![](img/7783422d9d71f1886c8e2fef85e288d6_69.png)

![](img/7783422d9d71f1886c8e2fef85e288d6_71.png)

4.  **夸大与虚假宣传**
    *   违规示例：使用“全民免单”、“恭喜获奖”、“抢疯了”、“再不抢就没了”等词汇，易被判定为夸大宣传。
    *   后果：可能引发禁播或停播处罚。

![](img/7783422d9d71f1886c8e2fef85e288d6_73.png)

5.  **行业禁用词**
    *   规则：每个行业都有特定的宣传禁区，必须在取得相应资质后才能使用相关词汇。
    *   示例：
        *   **食品行业**：在未认证的情况下，不能说“绿色食品”、“无公害”；不能将普通食品宣传为具有药物功效。
        *   **服装行业**：在未提供检测证明时，不能说“100%纯棉”。
    *   公式：`宣传用语 ≤ 已有资质/证明的范围`

6.  **站外引流词（如微信、赚钱）**
    *   规则：可以适当提及“微信”、“赚钱”、“淘宝”等词汇本身（例如，讲解如何赚钱、提及可分享至微信朋友圈）。
    *   红线：**严禁**以此为手段，将用户引导至抖音站外进行交易或联系。平台的核心规则是禁止站外引流。
    *   代码逻辑：
        ```plaintext
        if (话术意图 == 引导用户至站外) {
            处罚；
        } else if (话术仅为正常提及词汇) {
            通常安全；
        }
        ```

本节课中我们一起学习了抖音直播中的行为与话术雷区。核心在于，**人证不符、诱导互动、虚假宣传、站外引流**是高压线，务必规避；而在话术上，需做到文明、真实、合规，不触碰政治与行业禁词。牢记这些规则，才能保障直播间的长期安全稳定运营。
