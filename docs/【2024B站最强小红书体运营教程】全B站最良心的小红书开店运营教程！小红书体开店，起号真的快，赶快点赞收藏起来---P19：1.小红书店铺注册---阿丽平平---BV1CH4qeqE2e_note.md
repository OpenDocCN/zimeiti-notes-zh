# 小红书开店运营教程：P19：小红书店铺注册 📝

![](img/bdeda6e6de696e4f01c95436ef43014d_1.png)

在本节课中，我们将学习如何从零开始完成小红书店铺的注册流程。这是开启小红书电商之旅的第一步，我们将详细介绍每一步操作、需要注意的选项以及注册后的准备工作。

---

上一节我们介绍了开店前的准备工作，本节中我们来看看如何具体完成店铺的注册。

首先，你需要使用一部手机进行操作。如果条件允许，建议使用专门的工作手机。在手机上下载并安装小红书App。

安装完成后，请按以下步骤操作：

以下是账号登录与入口寻找的步骤：
1.  使用手机号登录你的小红书账号。
2.  登录后，点击App右下角的“我”，进入个人中心。
3.  在个人中心页面的右上角或左上角，找到并点击“三条杠”图标（菜单按钮）。
4.  在弹出的菜单中，选择“创作中心”。
5.  进入创作中心后，点击“更多服务”。
6.  在更多服务的页面中，找到并点击“开通店铺”的选项（通常位于第三排的方框内）。

点击“开通店铺”后，系统会跳转至一个引导页面。你只需找到并点击“立即认证”按钮即可开始注册流程，其他广告或推荐内容可以忽略。

接下来是关键的认证与信息填写环节。

点击“立即认证”后，请按照系统提示的步骤逐步操作。以下是核心步骤与注意事项：

以下是认证与信息填写步骤：
*   **提交材料**：上传你提前准备好的**身份证正反面**照片。
*   **选择店铺类型**：系统会提供个人店、企业店、旗舰店等类型。**对于新手，强烈建议选择“个人店”**。个人店门槛低，适合用于熟悉整个电商操作流程，避免初期投入过大而失去信心。
*   **填写店铺信息**：按照要求填写店铺名称、简介等信息。这些信息后期可以修改，但建议尽量填写准确，避免频繁更改。

信息填写完毕并提交后，就进入了审核等待阶段。

提交所有资料后，点击完成认证。小红书店铺的注册就基本完成了。平台审核需要一定时间，通常在**5分钟到1天左右**会有结果，请耐心等待。

店铺注册成功后，还有一项重要的准备工作。

当店铺审核通过后，为了处理未来的订单咨询与售后，你还需要在手机上下载一个独立的App：**小红书客服工作台**。店铺的聊天沟通功能主要在这个工作台中完成，而非主App内。
