# 新媒体运营AI实战：P15：01.10.03：训练AI从分析到产出爆款笔记 📝

在本节课中，我们将学习如何训练AI（以ChatGPT为例）分析爆款笔记，并最终生成具备爆款潜质的新内容。整个过程分为两步：**让AI学习爆款笔记的规律**，以及**指导AI应用这些规律进行创作**。

上一节我们介绍了通过“喂资料”的方式，让AI理解特定平台的文案风格。虽然上一节的旅游攻略案例效果不理想，但方法是正确的。即通过提供目标领域的优质内容，让AI分析并学习其写作风格和平台用语。

本节中我们来看看如何深化这一过程，具体聚焦于分析“爆款”内容，并让AI学会产出类似的高质量笔记。

## 第一步：指导AI分析爆款笔记 🔍

![](img/5855f07aae63fae16212e15f25d959b1_9.png)

![](img/5855f07aae63fae16212e15f25d959b1_11.png)

![](img/5855f07aae63fae16212e15f25d959b1_13.png)

要让AI学会创作，首先需要让它理解什么是“好”的内容。我们需要明确地指导AI从以下几个维度分析我们提供的爆款笔记：

![](img/5855f07aae63fae16212e15f25d959b1_15.png)

![](img/5855f07aae63fae16212e15f25d959b1_16.png)

以下是要求AI进行分析的几个核心指令点：

1.  **分析爆款因素**：找出内容之所以能成为爆款的关键点（例如：情感共鸣、实用干货、逆袭故事等）。
2.  **解构文案整体结构**：
    *   **开篇**：用什么方式（如提问、制造焦虑、引用名人）吸引用户。
    *   **中间**：用什么内容（如痛点分析、解决方案、详细规划）留住用户，提升停留时长。
    *   **结尾**：用什么“钩子”（如鼓励互动、提供福利、引导关注）促使用户行动。
3.  **分析关键词策略**：检查笔记是否刻意植入了特定关键词，并分析其出现频率和分布方式。

![](img/5855f07aae63fae16212e15f25d959b1_18.png)

**核心操作指令示例（给AI的提示词）**：
```text
你是一个新媒体运营专家。我将提供一篇小红书上的爆款笔记文案，请你分析：
1. 它的爆款因素有哪些？
2. 它的整体结构：开篇如何吸引用户？中间如何戳中痛点？结尾如何引导互动？
3. 它是否设计了关键词？如果有，是什么关键词？出现频率如何？
```

## 第二步：实战演练——以“考研”领域为例 🎯

我们以“考研”领域的爆款笔记为例，进行实战演练。

![](img/5855f07aae63fae16212e15f25d959b1_26.png)

![](img/5855f07aae63fae16212e15f25d959b1_28.png)

### 1. 寻找并输入样本
首先，我们需要在小红书等平台寻找多篇（建议3-5篇）点赞数据高的“考研”相关爆款笔记，将文案内容复制下来。

![](img/5855f07aae63fae16212e15f25d959b1_30.png)

![](img/5855f07aae63fae16212e15f25d959b1_32.png)

![](img/5855f07aae63fae16212e15f25d959b1_33.png)

### 2. 让AI逐篇分析
将每一篇笔记的文案，连同上述分析指令，逐一提交给AI（如ChatGPT 3.5）。AI会返回类似以下的分析结果：

![](img/5855f07aae63fae16212e15f25d959b1_35.png)

![](img/5855f07aae63fae16212e15f25d959b1_37.png)

*   **爆款因素**：情感共鸣（如备考焦虑）、逆袭故事、提供明确备考计划、时间节点清晰、推荐实用资源、正能量鼓励。
*   **文案结构**：开篇用自身焦虑经历引发共鸣 → 中间分享详细科目备考方法和时间规划 → 结尾鼓励并引导私信提问。
*   **关键词分析**：多数笔记没有严格设计关键词，常用词汇如“考研”、“备考”、“政治”、“英语”等自然出现。

![](img/5855f07aae63fae16212e15f25d959b1_39.png)

![](img/5855f07aae63fae16212e15f25d959b1_41.png)

![](img/5855f07aae63fae16212e15f25d959b1_43.png)

**注意**：AI有时会“遗忘”之前的指令或分析过的内容。如果发现它跑题，需要重新强调核心指令，或提醒它“回顾之前分析过的X篇笔记”。

![](img/5855f07aae63fae16212e15f25d959b1_44.png)

![](img/5855f07aae63fae16212e15f25d959b1_45.png)

### 3. 整合爆款要素
在AI分析完所有样本后，下达整合指令，让它去重并总结出该领域爆款笔记的共性要素。

![](img/5855f07aae63fae16212e15f25d959b1_50.png)

![](img/5855f07aae63fae16212e15f25d959b1_52.png)

![](img/5855f07aae63fae16212e15f25d959b1_54.png)

![](img/5855f07aae63fae16212e15f25d959b1_55.png)

**整合指令示例**：
```text
请将你前面分析过的所有考研爆款笔记的爆款因素，总结整合成一个完整的列表。
```

得到的整合列表可能包含：情感共鸣、逆袭故事、明确计划、针对性建议、时间管理、资源推荐、正能量结尾等。

## 第三步：让AI根据所学知识创作 📄

分析完成后，即可引导AI运用总结出的规律进行创作。

![](img/5855f07aae63fae16212e15f25d959b1_61.png)

### 1. 生成内容大纲
首先，让AI根据爆款要素，生成一篇新笔记的详细大纲。

![](img/5855f07aae63fae16212e15f25d959b1_63.png)

**大纲生成指令示例**：
```text
我要写一篇关于考研的小红书笔记。请结合你刚才总结的爆款因素，为我规划一个具备爆款潜质的笔记大纲。大纲需要细分到每个部分的具体要点。
```

AI生成的大纲可能包括：吸引人的开篇、主题介绍、分科目备考规划、时间管理技巧、心态调整方法、互动引导等部分。

### 2. 根据大纲生成完整文案
确认大纲无误后，指令AI根据大纲和爆款要素，填充具体内容，生成完整文案。

**文案生成指令示例**：
```text
请按照你提供的大纲，撰写完整的笔记文案。要求：
1. 结构严格遵循爆款笔记的要素。
2. 开篇融入一个简短的“逆袭”小故事引发共鸣。
3. 使用适量的emoji表情。
4. 尝试在文中自然融入“考研”、“备考规划”等关键词。
```

![](img/5855f07aae63fae16212e15f25d959b1_65.png)

![](img/5855f07aae63fae16212e15f25d959b1_66.png)

### 3. 评估与优化
AI生成的初稿可能“勉强可用”，但通常有优化空间。例如，故事不够生动、语言略显生硬、关键词植入不够自然等。我们可以：
*   要求AI调整语气，使其更亲切。
*   指定某部分需要更详细或更简洁。
*   使用更高级的模型（如GPT-4）或定制化的AI助手（如GPTs），通过上传更多风格样本和详细指令，获得质量更优、风格更稳定的输出。

## 总结与进阶提示 💡

本节课中我们一起学习了利用AI辅助内容创作的核心流程：**分析 → 总结 → 创作**。

1.  **精准分析**：通过明确的指令，让AI从爆款因素、文案结构、关键词策略等维度解构优质内容。
2.  **有效学习**：提供多个样本，并让AI整合共性，形成可复用的爆款公式。
3.  **引导创作**：基于总结出的公式，先规划大纲，再生成文案，使产出内容更具爆款潜质。

![](img/5855f07aae63fae16212e15f25d959b1_68.png)

**重要提示**：与AI对话的**措辞非常关键**。例如，“使用丰富的emoji”和“使用适量的emoji”会导致完全不同的结果。需要不断调整指令，使AI准确理解你的意图。

对于追求更高效、更稳定质量输出的运营者，可以探索：
*   使用更强大的模型（如GPT-4）。
*   创建自定义的**GPTs助手**，通过上传专属风格指南、排版范例、表情库等文件，打造一个深度理解你个人风格的私人创作助手。
