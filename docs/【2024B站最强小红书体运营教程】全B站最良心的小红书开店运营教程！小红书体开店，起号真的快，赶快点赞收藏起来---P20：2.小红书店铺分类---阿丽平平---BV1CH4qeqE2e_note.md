# 小红书开店运营教程：P20：2.小红书店铺分类 🏪

![](img/35063191a451d8d539810388de89626f_1.png)

在本节课中，我们将要学习小红书平台上的店铺分类体系。了解不同类型的店铺及其特点，是开店前的重要准备工作，有助于你根据自身情况选择最合适的店铺类型。

![](img/35063191a451d8d539810388de89626f_3.png)

上一节我们介绍了开店的基础概念，本节中我们来看看小红书平台具体有哪些店铺类型。

![](img/35063191a451d8d539810388de89626f_5.png)

## 店铺类型概述

小红书平台支持多种店铺类型，以满足不同经营主体的需求。了解整个店铺体系，有助于我们在不同阶段选择合适的操作方式。

以下是小红书主要的店铺类型列表：

*   **个体店**
*   **个体工商户店**
*   **普通企业店**
*   **专卖店**
*   **旗舰店**
*   **卖场型旗舰店**

接下来，我们将逐一解析每种店铺的特点。

## 各类店铺详解

### 个体店

个体店类似于现实中的个人摆摊。申请主体为个人用户，适合刚毕业或想尝试小规模经营的创业者。

![](img/35063191a451d8d539810388de89626f_7.png)

*   **经营特点**：可经营限定的、零星小额商品，例如首饰、服装、包包、手机壳等。类似于“杂货铺”，可销售多个品牌商品。
*   **入驻要求**：非常简单，仅需个人身份证。
*   **店铺命名格式**：系统自动生成，格式为“专业号昵称 + 店名”。
*   **保证金**：涉及**基础保证金**和**浮动保证金**，二者取高值。个人店的基础保证金通常为1000元。

![](img/35063191a451d8d539810388de89626f_9.png)

### 个体工商户店

个体工商户店与个体店的主要区别在于入驻要求。

*   **核心区别**：需要提供**营业执照**，对品牌授权无要求。
*   **其他方面**：如经营特点、保证金规则等，与个体店基本一致。

![](img/35063191a451d8d539810388de89626f_11.png)

### 普通企业店

![](img/35063191a451d8d539810388de89626f_13.png)

普通企业店及其他更高级的店铺（如专卖店、旗舰店），主要区别在于入驻要求的复杂程度。

> **重要提示**：店铺入驻要求越简单，其平台权重可能越低；要求越复杂，权重可能越高。但对于新手，建议从简单的个人店开始测试，无需初期投入过多。

*   **入驻要求**：申请主体必须为企业，需持有**企业营业执照**。

### 专卖店、旗舰店与卖场型旗舰店

这几种店铺类型主要是名称和授权要求的区别。

![](img/35063191a451d8d539810388de89626f_15.png)

*   **专卖店**：需营业执照和品牌认可（自有品牌或非独立品牌均可）。即销售自己品牌的产品。
*   **旗舰店**：需营业执照和**品牌授权书**（销售他人品牌的产品）。
*   **卖场型旗舰店**：要求具备零售资质，并可经营多个品牌。但在小红书平台较少见，因为平台流量以女性用户为主，与大型卖场型店铺的匹配度不高。这反而为中小卖家提供了更宽松的竞争环境。

了解了店铺类型后，我们来看看与之相关的关键规则：保证金制度。

![](img/35063191a451d8d539810388de89626f_17.png)

## 保证金规则详解 💰

![](img/35063191a451d8d539810388de89626f_19.png)

保证金分为**基础保证金**和**浮动保证金**，店铺需缴纳的金额为两者中的较高值。

![](img/35063191a451d8d539810388de89626f_21.png)

![](img/35063191a451d8d539810388de89626f_23.png)

### 基础保证金（类目保证金）

基础保证金与商品类目相关，不同类目金额固定。例如：
*   普通企业店部分类目：`1000元`
*   特殊类目或跨境类目：`金额可能更高（如2000元、5000元等）`

![](img/35063191a451d8d539810388de89626f_25.png)

具体类目保证金金额可查阅平台官方资料。

### 浮动保证金

浮动保证金是**根据店铺成交金额动态计算**的，这是需要重点理解的规则。

浮动保证金的计算方式如下：

*   **月成交金额 ≤ 5万元**：浮动保证金为 `0元`。（但需已缴纳基础保证金，如1000元，否则无法提现）
*   **5万元 < 月成交金额 ≤ 10万元**：浮动保证金为 `3000元`。
*   **10万元 < 月成交金额 ≤ 50万元**：浮动保证金为 `5000元`。
*   **月成交金额 > 50万元**：浮动保证金为 `20000元`。

**核心规则**：平台在提现时会检查保证金账户总额是否充足。如果店铺月成交额超过5万元，但保证金总额（基础+浮动）未达到对应标准（如3000元），则**账户内的资金将无法提现**。

**公式表示**：
`应缴保证金 = MAX(基础类目保证金， 当前浮动保证金)`
`可提现条件：账户保证金余额 ≥ 应缴保证金`

![](img/35063191a451d8d539810388de89626f_27.png)

因此，务必随时关注店铺成交额和保证金状态，确保资金可正常提取。

![](img/35063191a451d8d539810388de89626f_29.png)
