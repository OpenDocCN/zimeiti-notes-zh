# 抖音运营教程：P10：产品与用户兴趣标签

在本节课中，我们将学习如何运用“九宫格选题法”，结合产品属性与用户兴趣标签，为抖音账号创作源源不断的优质选题。我们将以健身软件Keep为例，详细拆解这一方法。

上一节我们介绍了选题的重要性，本节中我们来看看如何系统地生成选题。

## 产品标签分析

首先，我们需要分析产品本身的特点。以Keep为例，我们将产品名称放在中心，围绕它联想其核心属性。

以下是Keep可能具备的产品标签：

1.  **线上健身软件**：产品的核心形态。
2.  **专业健身教练**：产品提供的核心服务之一。
3.  **饮食搭配建议**：产品提供的增值内容。
4.  **减肥/塑形攻略**：产品解决的主要用户需求。
5.  **健身干货知识**：产品的教育属性。
6.  **科学运动方法**：产品倡导的理念。
7.  **打卡运动制度**：产品的用户激励体系。
8.  **一二线城市用户为主**：产品的主要用户画像。

完成产品标签填充后，我们暂时将其放在一边。

## 用户兴趣标签分析

接下来，我们分析目标用户的兴趣。使用Keep的用户可能是健身爱好者或想要减肥塑形的人。我们将这个用户群体放在中心，开始联想他们可能还关心什么。

以下是健身爱好者可能感兴趣的标签：

1.  **健身房**：线下健身的场所。
2.  **健身装备**：如运动鞋、紧身衣等。
3.  **轻脂美食/减肥餐**：与健康饮食相关。
4.  **旅行**：热爱生活的一种表现。
5.  **星座/明星**：普遍的娱乐兴趣点。
6.  **心灵鸡汤**：激励内容。
7.  **时尚穿搭**：与外形管理相关。
8.  **生活技巧**：提升生活品质的内容。

通过联想，我们将用户群从“健身爱好者”扩展到了“热爱生活者”，这有助于我们挖掘更广泛的选题方向。

## 运用“连连看”生成选题

现在，我们手头有两组标签：产品标签和用户兴趣标签。生成选题的关键在于将这两组标签进行创意连接。

以下是具体的操作方法：

1.  **随机连接**：从产品标签和用户兴趣标签中各选一个进行组合。
2.  **构思文案**：基于这个组合，构思一句完整的视频标题或文案。

例如：
*   **连接“饮食搭配”和“旅行”**：可以构思选题为《外出旅行时，如何享受美食又不长胖？》。
*   **连接“运动方法”和“明星”**：可以构思选题为《赵丽颖都在练的3个高效瘦身动作，赶紧学起来！》。
*   **连接“科学（运动方法）”和“健身房”**：可以构思选题为《在健身房如何科学训练，避免受伤？》。

如果自己联想遇到困难，可以借助搜索引擎。输入组合关键词，如“健身 科学”，参考搜索联想词来激发灵感。

## 筛选与优化选题

通过“连连看”可以产生大量选题思路，但并非所有都是好选题。我们需要进行筛选。

一个好的选题通常需要满足以下几点：
*   **符合逻辑**：产品与兴趣点的连接要自然、合理。
*   **吸引目标用户**：标题能戳中用户痛点或兴趣点。
*   **具备可操作性**：你自己有能力基于这个选题制作出视频。

![](img/5c5b92602d6752b89758a4a472eac882_3.png)

本节课中我们一起学习了“九宫格选题法”。我们首先分别梳理了产品自身的属性标签和目标用户的兴趣标签，然后通过将这两组标签进行创意组合（连连看），来批量生成视频选题灵感。这个方法能有效解决“不知道拍什么”的难题，帮助你建立持续的选题素材库。记住，生成选题后，还需根据逻辑性和吸引力进行筛选，才能找到最适合你账号的优质内容方向。
