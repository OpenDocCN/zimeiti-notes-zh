# 抖音运营教程：P11：3、如何完善个人资料 🧑‍💼

在本节课中，我们将要学习如何完善抖音的个人资料。一个专业且吸引人的个人主页是吸引粉丝、建立信任的第一步。我们将从头像、昵称、个人介绍以及主页联系方式等几个方面，详细讲解如何设置。

## 头像设置

上一节我们介绍了个人资料的重要性，本节中我们来看看头像的选择。头像应清晰、有辨识度，并能体现你的账号定位。

以下是头像选择的几种建议：
*   **有实体店**：可以使用实体店铺的照片。
*   **无实体店**：可以使用个人自拍照或生活照。
*   **宝妈群体**：可以使用与宝宝的合照。
*   **宠物类型账号**：可以使用宠物的照片。

## 昵称设置

设置好头像后，一个关键的步骤是起一个好记且专业的昵称。请注意，**不要使用自己的真实姓名**作为抖音昵称。例如，如果你的名字是“小月”，直接使用“小月”作为昵称是不合适的。抖音平台可能会将此判定为普通用户而非内容创作者，从而影响流量推荐。同时，这也暴露了个人隐私。

那么，如何起一个合适的昵称呢？请遵循以下原则：
*   **字数**：控制在五个字以内。
*   **字符**：避免使用繁体字和特殊符号（如 ❤️、✨ 等），以确保用户易于记忆和搜索。

以下是三种有效的昵称命名方法，你可以选择其一：

1.  **名字 + 定位**：`小月说游戏`、`小月小厨房`、`小月聊房`、`小月说车`、`小月穿搭`
2.  **名字 + 职业**：`宝妈小月`、`律师小月`、`摄影师小月`、`造型师小月`
3.  **名字 + 地区**：`赣州月姐`、`小月在北漂`、`小月在海外`

## 个人介绍撰写

昵称设置完成后，我们需要撰写个人介绍。个人介绍是让用户快速了解你的窗口，应围绕三个核心要素展开。

![](img/c67e011108aaa1f4ef39662761562221_1.png)

请参考以下三点来撰写你的个人介绍：
1.  **我是谁**：例如，“中年大叔依然保持母婴初衷”。
2.  **我是做什么的**：例如，“从事母婴行业13年，做母婴商贸8年有余”。
3.  **关注我能得到什么**：例如，“分享育儿经验”。

你可以根据自己的性格（幽默、可爱等）来调整介绍的风格，使其更生动活泼。

![](img/c67e011108aaa1f4ef39662761562221_3.png)

## 主页联系方式注意事项

在完善资料时，有一个常见的误区需要注意：**不要在个人主页上随意挂载微信或电话号码等联系方式**。

这是因为：
*   **平台规则**：抖音与微信（腾讯）属于不同公司，在抖音主页放置其他平台联系方式属于违规引流行为。
*   **效果有限**：在粉丝量较少时，挂载联系方式效果甚微。

**例外情况**：
*   已认证的**蓝V企业号**可以挂载联系电话。
*   当粉丝达到 **1万** 并成功入驻 **星图平台** 后，可以按规定挂载联系方式。关于星图平台，我们将在后续的变现环节详细讲解。

如果已经挂载了联系方式，请尽快将其删除。
