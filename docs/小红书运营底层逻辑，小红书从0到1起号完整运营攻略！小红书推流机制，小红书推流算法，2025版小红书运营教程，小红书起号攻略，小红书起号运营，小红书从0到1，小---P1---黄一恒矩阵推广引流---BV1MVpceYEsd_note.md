## 小红书推流机制

上一节我们介绍了课程的整体框架，本节中我们来看看小红书平台是如何决定笔记流量的。理解推流机制是做好运营的第一步。

小红书是一个“点击为王，互动为后”的平台。这意味着：
*   **点击率**决定笔记观看量的**下限**。点击率低，初始曝光量就低。
*   **互动量**决定笔记观看量的**上限**。互动数据好，笔记才有机会进入更大的流量池。

### 点击率的核心：封面设计
点击率主要由笔记封面决定。封面质量直接关系到笔记能否获得初始曝光。许多账号内容优质但流量不佳，问题往往出在封面上。

### 互动量的算法：CES评分
平台通过一套用户反馈算法（CES评分）来衡量笔记的互动质量，并据此决定是否推荐至更高流量池。

以下是CES评分的计算方式：
*   **点赞** = 1分
*   **收藏** = 2分
*   **评论** = 4分
*   **转发** = 4分
*   **关注** = 8分

笔记发布后获得的互动行为会累加CES总分，分数越高，进入更大流量池的机会越大。

### 内容标签与分发
平台通过打标签来识别和分发内容。

*   **大标签**：平台首页分类（如美食、穿搭、露营），目前约有37个。
*   **小标签**：在大分类下，根据用户兴趣细分出的标签（如“露营”下的“露营装备”、“露营地点”）。

内容的分发遵循“信息标签”和“重定向”机制：
*   **信息标签分发**：系统根据用户近期的点击、互动行为，推荐相关标签的内容。
*   **重定向机制**：用户与某个账号产生过互动（点赞、评论等），后续更容易刷到该账号的新内容。

### 流量层级与流量池
账号的**流量层级**由最近30天的观看量、互动量数据决定。层级越高，新笔记获得的初始曝光量通常越大。

小红书的推荐系统存在多个**流量池**（例如：300-500 → 1000-2000 → 5000-8000…）。笔记会根据CES评分，决定是停留在当前池还是跃升到下一个池。

## 小红书起号原理

了解了流量如何产生后，本节我们来看看一个新账号是如何被平台“认识”并开始获得流量的。起号过程可分为三个阶段。

### 第一阶段：观察期
新账号注册后，系统会进入**观察期**。此时系统不确定账号用途和质量，会限制流量以观察行为，防止违规。此阶段流量低属正常现象。

**操作建议**：发布内容需“脱敏”，避免营销信息，多发干货、利他型内容，帮助账号平稳度过观察期。

### 第二阶段：识别期
度过观察期后，进入**识别期**。系统需要识别你的内容类型，以决定推送给什么样的人群。匹配成功即为精准流量，匹配失败则为泛流量。

**操作方法**：通过发布该领域用户**高频搜索**的内容，帮助系统快速识别账号定位。例如，做“露营”账号，就多发与“露营”搜索结果中热门选题相关的内容。

### 第三阶段：入池期
当笔记发布后前几天流量很低，随后突然显著增长（如从500涨到2000），即表示内容**入池**，进入了平台的推荐流量池。

**加速入池技巧**：投放“薯条”（小红书官方推广工具）。薯条会触发人工审核，能帮助账号快速跳过观察期，并提高初始曝光量。
*   **投放策略**：分两次投放。第一次投“点赞收藏”，提升CES评分；24小时后第二次投“浏览量”，拉动流量增长。

## 小红书起号方法

明确了起号的原理和阶段，本节我们来看看两种具体、可操作的起号方法。

### 方法一：搜索起号法
这是最简单有效的方法，核心是**利用搜索行为让系统快速识别账号定位**。小红书的搜索流量占比很高，利用好搜索是关键。

**操作步骤**：
1.  确定你的领域关键词（如“净水器”）。
2.  在小红书搜索框输入该关键词。
3.  将搜索结果页的“下拉推荐词”和“热门笔记选题”作为你的内容选题。

以下是搜索起号法的六个注意细节：
1.  优先选择收藏数大于点赞数的选题，这类内容更容易获得搜索排名。
2.  借鉴他人内容时，修改幅度要超过30%-50%，避免被判搬运。
3.  标题必须包含核心关键词，标题权重高于正文。
4.  封面可参考高点击率同行，已验证的封面能保障点击率下限。
5.  如果是视频笔记，需在正文中复制一部分视频文案，方便系统识别。
6.  初期内容避免营销广告，确保能通过薯条审核。

### 方法二：骗评论起号法（互动撬动法）
这是一种通过**刻意设计高互动内容**来快速提升CES评分，从而撬动流量的方法。因为“评论”在CES评分中价值4分，权重很高。

**核心思路**：设计低门槛的互动问题，诱导用户评论。

**两种常用形式**：
1.  **极限二选一**：让用户在A和B之间做选择（如“两款扫地机器人，你选哪个？”）。用户只需评论“1”或“2”，互动成本极低。
2.  **开放式问答题**：提出简单易答的问题（如“说出你最喜欢的城市”）。同样能有效激发用户评论欲望。

## 小红书内容运营

掌握了起号方法后，本节我们探讨账号稳定运营阶段的内容策略。关键在于平衡内容的广度与深度。

### 内容类型：流量型 vs 营销型
我们需要发布两种类型的内容：

*   **流量型内容（宽口内容）**：
    *   **特点**：选题宽泛，潜在受众广，流量大，但转化率相对较低。
    *   **示例**（装修赛道）：《2024年最流行的装修风格》。适合吸引行业小白和潜在客户。

*   **营销型内容（窄口内容）**：
    *   **特点**：选题精准、细分，流量较小，但转化率高。
    *   **示例**（装修赛道）：《200平大平层，厨房如何装修？》。直接吸引有明确需求的高意向客户。

### 内容策略：浅深结合
最有效的策略是将两者结合，搭建内容矩阵。

*   **“浅”内容（流量型）**：负责**引流**，扩大流量池，触达更多潜在用户。
*   **“深”内容（营销型）**：负责**转化**，筛选并服务高意向客户，实现变现。

**策略应用**：
*   **只浅不深**：有流量，但订单少。
*   **只深不浅**：依赖有限的老客户，难以扩大规模。
*   **浅深结合**：既能持续获取新流量，又能高效转化客户，实现良性循环。
