# 爱嘉客服🤖AI+客服：P1：企业微信外部群AI智能客服机器人配置教程 🚀

在本节课中，我们将学习如何为企业微信外部群配置爱嘉AI智能客服机器人，实现自动识别与回复功能。

## 概述
本教程将引导你完成爱嘉AI智能客服机器人在企业微信外部群中的基础设置与工作流程演示。我们将从平台选择开始，逐步展示机器人如何响应客户提问。

![](img/2a0b7dfa78fd71e44beb4420235b47de_1.png)

## 教程步骤

### 第一步：选择平台并启动
首先，在爱嘉客服平台选择你需要配置的服务，然后点击“开始”按钮以启动设置流程。

![](img/2a0b7dfa78fd71e44beb4420235b47de_1.png)

### 第二步：理解角色与工作流程
配置完成后，系统内将存在两个核心角色：
*   **个人微信**：代表发起提问的客户。
*   **企业微信**：代表我们的**AI智能客服机器人**。

其自动化工作流程的核心逻辑如下：
1.  **AI智能客服机器人**识别到有客户在外部群聊中“@”提及它。
2.  机器人将识别到的问题内容发送至后台的**AI模型**进行处理。
3.  等待**AI模型**生成回复内容。
4.  机器人将收到的AI回复自动转发回企业微信外部群聊中，完成应答。

以下是该流程的简单代码表示：
```python
if 消息中提及(@AI客服机器人):
    问题 = 提取消息内容()
    AI回复 = 调用AI模型(问题)
    发送消息到群聊(AI回复)
```

### 第三步：触发条件验证
需要明确的是，该AI客服机器人的响应具有特定触发条件。如果客户在外部群聊中发言但**没有“@”提及AI智能客服机器人**，机器人将不会主动回答任何问题。

![](img/2a0b7dfa78fd71e44beb4420235b47de_3.png)

为了验证这一机制，我们可以再次尝试在群聊中“@”AI智能客服机器人，观察它是否会做出响应。

## 总结
本节课中，我们一起学习了爱嘉AI智能客服机器人的基本配置。关键点在于：机器人仅在客户**“@”提及它**时才会被触发，随后自动完成“提问->AI处理->回复”的全流程。这确保了客服响应的精准性与自动化效率。
