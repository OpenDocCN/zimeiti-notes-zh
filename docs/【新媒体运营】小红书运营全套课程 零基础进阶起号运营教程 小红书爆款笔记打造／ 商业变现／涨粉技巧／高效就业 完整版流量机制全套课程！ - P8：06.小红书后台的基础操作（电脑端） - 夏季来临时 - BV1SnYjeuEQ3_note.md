# 小红书运营全套课程：P8：06.小红书后台的基础操作（电脑端） 💻

![](img/c7522150f14d6f5ce1dcd7f8a1175d40_1.png)

在本节课中，我们将学习如何通过电脑端操作小红书商家后台，重点掌握保证金、支付方式、运费保和地址设置这四个核心功能。这些是店铺正常运营和实现交易的基础。

![](img/c7522150f14d6f5ce1dcd7f8a1175d40_3.png)

上一节我们介绍了手机端的操作，本节中我们来看看电脑端如何设置这些关键功能。

## 保证金管理 💰

保证金是开店时需要向平台缴纳的一笔诚信押金。前期可以暂不缴纳，但若想长期经营或开通更多支付方式，则建议缴纳。

![](img/c7522150f14d6f5ce1dcd7f8a1175d40_5.png)

*   **金额**：个人店保证金通常在**1000元**左右，具体金额因经营类目而异。企业店保证金在2000元至数万元不等。
*   **缴纳与退还**：保证金在店铺关闭后可申请退还。平台会扣除保证金以处理店铺违规或未履行的售后责任（如未按约定退款），正常经营则不会扣款。
*   **缴纳入口**：登录**小红书千帆系统** -> 点击“**资金**” -> 进入“**店铺保证金管理**”。
*   **影响**：未缴纳保证金**无法开通微信支付**功能。缴纳后，可同时支持支付宝和微信支付。

![](img/c7522150f14d6f5ce1dcd7f8a1175d40_7.png)

![](img/c7522150f14d6f5ce1dcd7f8a1175d40_9.png)

以下是查询和缴纳保证金的步骤：
1.  进入“店铺保证金管理”页面。
2.  选择您店铺的实际经营类目，系统会显示应缴金额。
3.  点击“充值”或“补缴”完成支付。

![](img/c7522150f14d6f5ce1dcd7f8a1175d40_11.png)

## 支付方式开通 💳

开通支付方式是让顾客能够付款的关键。主要包括支付宝和微信支付两种。

![](img/c7522150f14d6f5ce1dcd7f8a1175d40_13.png)

*   **开通位置**：同样在**小红书千帆系统**的“**资金**”模块下。
*   **开通流程**：
    *   **支付宝**：通常可即时开通并绑定。
    *   **微信支付**：需先缴纳保证金，次日方可开通设置。提现金额将进入店铺注册法人的账户。
*   **所需材料**：根据提示绑定相应的支付账户。个体店/企业店需准备好**营业执照**等信息。

![](img/c7522150f14d6f5ce1dcd7f8a1175d40_15.png)

以下是开通支付方式的核心步骤：
1.  在“资金”模块找到“支付设置”。
2.  分别点击“支付宝”和“微信支付”的开通按钮。
3.  按照页面指引，完成账户绑定与信息填写。

![](img/c7522150f14d6f5ce1dcd7f8a1175d40_17.png)

## 运费保设置 🛡️

运费保是一项平台服务，旨在降低消费者的下单顾虑，可能提升订单转化率。

*   **作用**：开通后，若消费者符合条件退货退款，**系统将自动补贴运费**，无需卖家承担。这类似于淘宝的“运费险”。
*   **好处**：打消顾客对运费的顾虑，提升下单意愿，同时降低因运费产生的交易纠纷。
*   **开通位置**：在后台的“**售后**”管理板块中，找到“**运费保**”服务。

![](img/c7522150f14d6f5ce1dcd7f8a1175d40_19.png)

![](img/c7522150f14d6f5ce1dcd7f8a1175d40_21.png)

开通运费保的建议操作如下：
1.  进入“售后管理”或类似板块。
2.  查找“运费保”或“退货保障”服务。
3.  阅读条款后，点击“开通全店运费保服务”。

![](img/c7522150f14d6f5ce1dcd7f8a1175d40_23.png)

## 地址与物流设置 📦

![](img/c7522150f14d6f5ce1dcd7f8a1175d40_25.png)

正确设置发货和退货地址，是保障物流顺畅、减少售后问题的关键。

![](img/c7522150f14d6f5ce1dcd7f8a1175d40_27.png)

![](img/c7522150f14d6f5ce1dcd7f8a1175d40_29.png)

*   **设置入口**：在“**订单**”或“**物流工具**”板块中，找到“**地址管理**”或“**新建地址**”。
*   **发货地址**：有货源的商家填写真实仓库地址。无货源（一件代发）的商家，可同步上游供应商的地址信息。
*   **退货地址设置技巧**：
    *   建议填写一个**偏远地区地址**（如新疆），并在地址后备注“**退货前请联系客服**”。
    *   这样做的目的是当顾客发起退货时，能主动与您沟通，方便您引导其将货物退至正确的供应商地址，避免错退。
*   **物流模板**：可根据商品类型和发货地，设置不同的运费模板（如包邮、按地区收费等）。

以下是地址设置的核心要点：
1.  在“物流工具”中新建发货地址和退货地址。
2.  对于退货地址，采用“偏远地区地址 + 联系客服”的策略。
3.  根据运营模式（有货源/无货源），填写或同步正确的地址信息。

![](img/c7522150f14d6f5ce1dcd7f8a1175d40_31.png)

本节课中我们一起学习了小红书电脑端后台的四个基础操作：保证金的查询与缴纳、支付方式的开通、运费保服务的设置，以及发货退货地址的配置。掌握这些设置，是店铺能够正常接收订单、完成交易并保障售后顺畅的基础。请务必根据您的实际情况，逐一完成这些后台配置。
