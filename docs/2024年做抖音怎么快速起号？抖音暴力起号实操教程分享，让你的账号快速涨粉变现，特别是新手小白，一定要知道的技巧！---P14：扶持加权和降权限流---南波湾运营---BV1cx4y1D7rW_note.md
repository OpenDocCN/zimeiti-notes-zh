# 抖音运营实战：P14：扶持加权与降权限流

在本节课中，我们将学习抖音平台对视频内容的两种核心处理机制：扶持加权与降权限流。理解这些规则，是确保你的账号健康、快速成长并避免违规风险的基础。

上一节我们介绍了账号权重的分类，本节中我们来看看哪些内容会受到平台的鼓励，哪些内容则可能被限制。

以下是获得扶持加权的关键视频制作要求：
*   **视频格式**：必须为竖版移动端原生视频，即 **9:16** 的格式。
*   **画面质量**：画面需保持平稳，作品清晰度高。

## 会被降权限流的内容

与扶持加权相反，如果你的视频内容涉及以下违规点，则可能被系统降权、限制推荐，甚至审核不通过。

以下是明确会导致降权限流的内容类型：
1.  **违规信息**：视频中出现吸烟、暴力、炫富（过度展示金钱）等内容。
2.  **外部引流**：出现或引导至微信、微博、淘宝等站外联系方式。长时间出现商品Logo、手机号、二维码（二维码出现时间建议不超过3秒，但审核严格时，即使短时间出现也可能被驳回）。
3.  **硬性广告**：进行明显的营销广告、推销行为，直接引流卖货。

## 特别注意：好物推荐类内容的误区

对于做好物推荐、带货类账号的同学，需要特别注意一个常见误区：**无口播的拆箱或产品展示**。

许多同学希望通过分享好物（如玩具、日用品）来挂小黄车带货，但视频发布后小黄车常被下架。主要原因如下：

以下是两个导致失败的典型案例及正确做法：
*   **案例一（错误）**：视频画面中直接出现“**9.9元**”等价格信息。这属于直接的营销行为，容易被判定为硬广。
*   **案例二（错误）**：全程无任何口播，仅用背景音乐展示玩具玩法。缺乏内容阐述，被系统判定为低质或纯广告内容。
*   **正确做法**：应采用“**剧情类软广告**”或加入口播介绍。例如，展示纸巾时，可以口播说：“这款纸巾沾水后依然不易破，可以反复使用。” 或者展示玩具时，口播说：“我儿子玩这个玩具，专注了四个小时。” 通过增加真人使用体验和描述，将广告转化为内容分享。

简单来说，**不要不说话，直接展示产品**。为你的推荐增加价值描述和真实体验，是合规带货的关键。

## 内容总结与资料说明

本节课中我们一起学习了抖音平台的核心内容规则：
1.  平台会扶持**格式规范（9:16）、画质清晰、音质干净**的优质原生视频。
2.  涉及**违规信息、外部引流和硬性广告**的内容会被降权限流。
3.  对于**好物推荐类**视频，务必避免无口播的纯产品展示，应通过口播或剧情融入产品卖点和体验。
