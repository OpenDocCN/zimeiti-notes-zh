# 新媒体运营：01.8：小红书平台笔记审核规则 📝

在本节课中，我们将要学习小红书平台笔记的流量推荐与分发机制。我们将详细拆解一篇笔记从发布到最终推荐给用户的完整流程，并解释其中的核心规则与概念。

## 笔记审核与分发流程概述

![](img/e02755f5bd89d722df04b3f6888ad1c0_1.png)

小红书与许多新媒体平台一样，采用“千人千面”的内容展现形式，即根据用户的特定标签推送特定内容。本节中，我们来看看一篇笔记发布后，需要经历哪些审核与分发阶段。

![](img/e02755f5bd89d722df04b3f6888ad1c0_3.png)

### 第一阶段：机器审核

笔记上传后，首先进入机器审核系统。该系统依据预设的基本规则进行初步筛查。

以下是机器审核的主要检查项：

![](img/e02755f5bd89d722df04b3f6888ad1c0_5.png)

![](img/e02755f5bd89d722df04b3f6888ad1c0_7.png)

*   **敏感词与敏感图片**：系统会检测文本和图片中是否存在违规内容。
*   **笔记基础质量**：包括笔记字数、广告词出现频率、图片清晰度以及图片中文字与物体的可识别性。

![](img/e02755f5bd89d722df04b3f6888ad1c0_9.png)

![](img/e02755f5bd89d722df04b3f6888ad1c0_11.png)

如果笔记在机器审核阶段发现明确违规内容，可能会被直接封禁。如果系统无法准确判断（即“疑似违规”），笔记会被标记并转入下一阶段——人工审核。若笔记完全通过机器审核，则进入下一个环节。

![](img/e02755f5bd89d722df04b3f6888ad1c0_13.png)

![](img/e02755f5bd89d722df04b3f6888ad1c0_15.png)

### 第二阶段：去重检测

![](img/e02755f5bd89d722df04b3f6888ad1c0_17.png)

![](img/e02755f5bd89d722df04b3f6888ad1c0_19.png)

![](img/e02755f5bd89d722df04b3f6888ad1c0_21.png)

通过机器审核的笔记，会进入去重检测阶段。系统会将你的笔记与平台数据库中的现有内容进行比对，计算相似度。

![](img/e02755f5bd89d722df04b3f6888ad1c0_23.png)

![](img/e02755f5bd89d722df04b3f6888ad1c0_25.png)

去重检测主要比对以下内容：

![](img/e02755f5bd89d722df04b3f6888ad1c0_27.png)

![](img/e02755f5bd89d722df04b3f6888ad1c0_29.png)

*   **图片相似度**：是否为直接抄袭或复制的图片。
*   **文案重复率**：文案内容、段落在数据库中的重复比例。

![](img/e02755f5bd89d722df04b3f6888ad1c0_31.png)

![](img/e02755f5bd89d722df04b3f6888ad1c0_33.png)

![](img/e02755f5bd89d722df04b3f6888ad1c0_35.png)

如果笔记被检测为高度重复（例如原封不动抄袭），则会被封禁。如果对图片（如裁剪、加滤镜）和文案（如调整段落、扩充内容）进行了修改，可能降低相似度，从而通过此关。

![](img/e02755f5bd89d722df04b3f6888ad1c0_37.png)

![](img/e02755f5bd89d722df04b3f6888ad1c0_39.png)

### 第三阶段：用户匹配与权重影响

![](img/e02755f5bd89d722df04b3f6888ad1c0_41.png)

![](img/e02755f5bd89d722df04b3f6888ad1c0_43.png)

通过前两轮检测的笔记，系统会分析其内容特征，以便匹配给可能感兴趣的用户。然而，笔记能获得多少初始曝光，很大程度上受**账号权重**影响。

![](img/e02755f5bd89d722df04b3f6888ad1c0_45.png)

![](img/e02755f5bd89d722df04b3f6888ad1c0_47.png)

![](img/e02755f5bd89d722df04b3f6888ad1c0_49.png)

账号权重是一个系统对账号质量的综合评分，它并非公开数据，但真实存在。我们可以用一个简单的模型来理解：

![](img/e02755f5bd89d722df04b3f6888ad1c0_51.png)

![](img/e02755f5bd89d722df04b3f6888ad1c0_53.png)

**账号权重模型（假设）**：`权重分值 ∈ [0, 10]`

![](img/e02755f5bd89d722df04b3f6888ad1c0_55.png)

![](img/e02755f5bd89d722df04b3f6888ad1c0_57.png)

*   **新号阶段（权重≈0）**：新注册的账号初始权重很低。此阶段笔记的初始推荐量通常很少（例如几十个浏览）。因为系统尚未给账号打上稳定标签，笔记无法精准推送给目标人群，互动数据难以提升，导致流量池突破困难。
*   **成熟账号阶段（权重提升）**：当账号持续发布垂直领域内容（如彩妆），且无违规记录，系统会逐渐提升其在该领域的权重分值。权重越高，笔记获得的初始曝光基数通常越大。

![](img/e02755f5bd89d722df04b3f6888ad1c0_59.png)

![](img/e02755f5bd89d722df04b3f6888ad1c0_61.png)

**核心要点**：在新号阶段（低权重时期），重点应放在“养号”上，即通过持续发布垂直内容、保持互动、严格避免违规（如广告、违禁词）来逐步提升账号权重，而不是期望单篇笔记爆火。

![](img/e02755f5bd89d722df04b3f6888ad1c0_63.png)

![](img/e02755f5bd89d722df04b3f6888ad1c0_65.png)

### 第四阶段：人工审核

![](img/e02755f5bd89d722df04b3f6888ad1c0_67.png)

![](img/e02755f5bd89d722df04b3f6888ad1c0_69.png)

人工审核主要对机器标记为“疑似违规”的内容进行二次复核。但由于内容量巨大，成本高昂，并非所有笔记都会进入此环节。

![](img/e02755f5bd89d722df04b3f6888ad1c0_71.png)

![](img/e02755f5bd89d722df04b3f6888ad1c0_73.png)

人工审核的重点领域包括：

![](img/e02755f5bd89d722df04b3f6888ad1c0_75.png)

![](img/e02755f5bd89d722df04b3f6888ad1c0_77.png)

*   政治相关
*   色情擦边
*   其他法律严禁内容

![](img/e02755f5bd89d722df04b3f6888ad1c0_79.png)

![](img/e02755f5bd89d722df04b3f6888ad1c0_81.png)

目前，平台算法已较为完善，触发明确红线的笔记大多由机器直接处理，人工复核比例减少。如果笔记被误判，创作者可以通过申诉渠道反馈，但新号阶段应尽量避免任何违规，以免对账号造成隐性限流等负面影响。

## 总结

本节课我们一起学习了小红书笔记的审核与分发机制。关键流程总结如下：

1.  **机器审核**：筛查敏感词、图片及基础质量。
2.  **去重检测**：比对内容相似度，打击抄袭。
3.  **权重影响**：账号权重（受垂直度、合规性影响）决定笔记的初始曝光量级。
4.  **人工审核**：对疑似违规内容进行补充复核。

理解这套机制，有助于我们在运营小红书时，更规范地发布内容，并通过长期稳定的优质运营来提升账号健康度与内容曝光机会。
