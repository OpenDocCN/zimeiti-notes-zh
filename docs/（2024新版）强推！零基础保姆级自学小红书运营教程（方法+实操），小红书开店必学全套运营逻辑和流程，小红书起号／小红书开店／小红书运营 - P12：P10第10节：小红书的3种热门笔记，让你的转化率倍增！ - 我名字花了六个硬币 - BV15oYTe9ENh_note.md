# 小红书运营教程：P12：小红书的3种热门笔记类型

在本节课中，我们将详细讲解小红书的三种热门笔记类型。掌握这些笔记的写作方法，可以有效提升内容的转化率。课程内容详实，建议在学习时做好笔记。

## 教育类笔记 📚

上一节我们介绍了笔记类型的重要性，本节中我们首先来看看教育类笔记。这类笔记的核心是传授知识或技能，适合将产品自然地植入内容中。

教育类笔记适合的行业包括：美妆、瘦身、美食、P图软件、美容、摄影等。在撰写时，可以将产品作为教学工具或材料进行植入。

以下是教育类笔记封面的三种常见构成方式：

*   **对比图**：展示使用产品前后的效果对比。例如，口红涂在手上与上唇后的颜色对比，并标明色号。
*   **效果图**：直接展示使用产品后的最终效果。例如，展示某款眼影上妆后的整体妆容效果。
*   **步骤图**：通过步骤分解展示过程。例如，“五步告别圆肩驼背”教程中，可能会展示并使用到辅助产品（如背背佳）。

接下来，我们看看教育类笔记的正文应如何撰写。正文应遵循教学逻辑，通常包含以下五个部分：

1.  **使用工具或材料**：列出所需的**产品**和辅助工具。
2.  **操作步骤**：详细说明具体的操作流程。
3.  **注意事项**：提示操作过程中需要避免的问题或关键点。
4.  **产品安利**：在讲解过程中，自然地推荐你觉得好用的产品。**注意**：介绍产品的文字篇幅**不要超过整体文章的30%**，以免被判定为硬广。
5.  **常见问题解答**：针对操作中可能出现的**失败案例**（例如“为什么贴不上”）进行解答，这比展示成功更能吸引读者。

## 合集类笔记 📦

了解了以教学为核心的教育类笔记后，我们来看看适用范围更广的合集类笔记。这类笔记通过集合多个相关物品进行推荐，几乎适用于所有行业。

合集类笔记的核心是针对特定场景、人群或需求，推荐一系列产品。例如：旅行必备清单、夏季穿搭指南、健身期推荐食品等。

在撰写合集类笔记时，一个常见问题是角度单一。为了避免这个问题，可以从以下七个不同角度切入创作：

1.  **针对人群**：例如，《学生党开学彩妆攻略》、《上班族代步小电动车推荐》。
2.  **针对成本**：例如，《100元内平价护肤品》、《5块钱网红早餐包》。
3.  **针对效果**：例如，《让男朋友合不拢嘴的夏季穿搭》、《瘦身必备零食推荐》。
4.  **针对地区**：例如，《香港平价美食推荐》、《南京本地网红店打卡》。
5.  **针对场景**：例如，《旅游必备清单》、《居家清洁好物指南》。
6.  **针对产品属性**：例如，《微电流美容仪测评》、《某某色号口红合集》。
7.  **针对品牌**：例如，《体察内部员工隐藏菜单》、《星巴克夏季新品测评》。

合集类笔记的第二个常见问题是推广嫌疑过重。解决方法如下：
*   **均匀分布产品篇幅**：不要只着重介绍一两款产品。
*   **客观评价**：同时说明产品的**优点**和**缺点**，增加可信度。

合集类笔记的封面主要有两种形式：
*   **产品拼图**：使用美图秀秀、黄油相机等APP将多个产品图片拼接在一起。
*   **产品摆拍**：将推荐的所有物品（如旅行箱内的物品）整齐地摆放在一起拍摄。

总结一下，合集类笔记的标题应从展现的角度切入，正文则应包含**推荐理由**和**产品特点说明**。如果配图已有详细文字说明，正文可简单提及产品名称，这有助于平台搜索引擎抓取关键词标签。

## 视频类笔记 🎥

最后，我们来看看当前平台流量扶持力度最大的视频类笔记。如果条件允许，非常推荐大家尝试制作这类内容。

制作视频类笔记，首先需要准备相应的工具。
*   **拍摄工具**：手机、专业相机、三脚架、补光灯。预算充足可考虑简易摄影棚。
*   **剪辑工具**：手机APP（如剪映），或电脑专业软件（如 **Adobe Premiere Pro**）。

关于视频制作，有以下几点需要注意：
*   **视频时长**：小红书默认视频时长为**1分钟**。发布一定数量且质量达标的笔记后，平台会自动开放**5分钟**权限。若想发布超过10分钟的视频，需主动向官方邮件申请。
*   **视频尺寸**：推荐两种比例。
    *   **3:4（竖屏）**：大尺寸，视觉冲击力强。
    *   **4:3（横屏）**：适合展示化妆品等需要精致呈现的产品。

视频内容方面，可以结合自身行业，制作教程、测评、私人分享、生活VLOG等。一个有效的学习方法是：多研究抖音、快手、小红书上同行业**成功博主**的视频，总结他们的**时长控制、构图、选品和脚本结构**（如开头多少秒吸引人，中间如何展开，结尾如何收尾），模仿其成功经验。
