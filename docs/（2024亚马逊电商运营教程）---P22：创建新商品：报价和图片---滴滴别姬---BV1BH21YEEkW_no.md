# 亚马逊电商运营教程：P22：创建新商品：报价和图片

![](img/c4b705637a2e5f00cd5fef0754ba215d_0.png)

在本节课中，我们将学习如何填写商品创建页面的“报价”和“图片”部分。这是将商品成功上架的关键步骤，涉及定价、库存管理和视觉展示。

上一节我们介绍了商品的基本信息填写，本节中我们来看看如何设置报价和上传商品图片。

## 报价设置

报价部分决定了商品的销售价格、库存和物流信息。

以下是报价部分需要填写的关键字段：

*   **您的价格**：这是您希望商品在亚马逊上销售的实际价格。
*   **市场价**：这是商品的市场参考价或原价，用于与您的价格对比，显示折扣。例如，您可以设置市场价为 `100` 美元。
*   **处理时间**：指买家下单后，您需要多少天来准备并发货。通常以天数计算，例如 `1-2` 天。
*   **补货日期**：根据您的库存情况，预估商品售罄后需要多久能补货上架。
*   **礼品选项**：此功能通常仅对**专业销售计划**卖家开放。个人卖家无需选择，即使选择也无效。
*   **订单商品最大数量**：限制单个买家在一次订单中购买此商品的最大件数，例如 `10` 件。
*   **开始销售时间**：设置商品在亚马逊上架并开始销售的具体日期和时间。

## 合规信息与图片上传

![](img/c4b705637a2e5f00cd5fef0754ba215d_3.png)

完成报价设置后，我们继续处理其他必要信息。

### 合规信息

![](img/c4b705637a2e5f00cd5fef0754ba215d_5.png)

对于大多数普通商品（如服装、家居用品），如果没有特殊成分（如电池、液体、磁性材料），**合规信息部分通常可以留空不填**。此部分主要针对航空运输受限的特殊商品。

### 上传商品图片

![](img/c4b705637a2e5f00cd5fef0754ba215d_7.png)

商品图片是吸引买家、促进转化的核心要素。亚马逊要求至少上传一张主图。

![](img/c4b705637a2e5f00cd5fef0754ba215d_9.png)

以下是上传图片的步骤和注意事项：

1.  点击“选择文件”或拖拽区域，从您的电脑中选择图片文件。
2.  第一张上传的图片将自动被设为主图。主图必须符合亚马逊的要求（纯白背景、展示完整商品等）。
3.  您可以继续添加更多图片，从不同角度展示商品细节、功能或使用场景。

![](img/c4b705637a2e5f00cd5fef0754ba215d_11.png)

**注意**：关于商品变体（如不同颜色、尺寸），每个独立的变体都需要一个唯一的UPC码。例如，如果一个商品有2种颜色和2种尺寸，那么总共需要 `2 * 2 = 4` 个UPC码。

![](img/c4b705637a2e5f00cd5fef0754ba215d_13.png)
