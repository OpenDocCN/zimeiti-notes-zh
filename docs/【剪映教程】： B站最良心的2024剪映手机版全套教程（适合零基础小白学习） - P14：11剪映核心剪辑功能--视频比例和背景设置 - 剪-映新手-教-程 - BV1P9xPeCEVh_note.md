# 剪映教程：P14：视频比例与背景设置 🎬

在本节课中，我们将学习剪映的核心剪辑功能之一：如何调整视频比例以及设置视频背景。这是视频剪辑的基础操作，正确的设置能有效提升视频的观感和平台推荐效果。

视频的画幅比例和背景调整是剪辑的基础。如果设置错误，会影响视频的推荐和播放效果。在抖音、快手等平台，绝大多数视频都采用竖屏格式。因此，了解如何将视频调整为合适的比例并美化背景至关重要。

上一节我们介绍了基础的剪辑界面，本节中我们来看看如何具体调整视频的显示比例和背景。

## 视频比例的选择

![](img/b2a1d6667ff8f390c0c4dc74c7875c01_1.png)

在抖音或快手发布视频时，推荐以竖屏为主。如果你拍摄的是横屏视频，也无需担心，可以通过后期调整将其转换为竖屏格式。

以下是常见的视频比例选择建议：
*   **竖版视频（9:16）**：适用于拍摄情景剧、美食或旅行等内容，在手机端观看体验更佳，推荐效果更好。
*   **横版视频（16:9）**：适用于拍摄Vlog等内容。可以保留上下黑边，并添加背景色或标题。

视频分辨率通常选择 **1080P** 和 **60帧**。剪映也支持导出更清晰的 **2K** 或 **4K** 分辨率视频。

## 实战操作：调整比例与背景

接下来，我们进入软件，详细讲解设置步骤。

首先，新建一个项目并导入一段横版视频素材。

![](img/b2a1d6667ff8f390c0c4dc74c7875c01_3.png)

![](img/b2a1d6667ff8f390c0c4dc74c7875c01_1.png)

导入后预览，可以看到这是一个典型的横版（16:9）视频。

![](img/b2a1d6667ff8f390c0c4dc74c7875c01_3.png)

![](img/b2a1d6667ff8f390c0c4dc74c7875c01_5.png)

### 第一步：调整视频比例

我们需要将其调整为竖版。在一级菜单栏向右滑动，底部可以找到“比例”和“背景”选项。

![](img/b2a1d6667ff8f390c0c4dc74c7875c01_5.png)

点击“比例”，然后选择 **9:16**。视频画面会立即变为竖版，上下出现黑色边框。

### 第二步：设置视频背景

![](img/b2a1d6667ff8f390c0c4dc74c7875c01_7.png)

黑色的背景显得单调，我们可以对其进行美化。返回到上一级菜单，点击“背景”选项。

点击后会出现三个子选项：画布颜色、画布样式、画布模糊。

以下是每个选项的功能介绍：

![](img/b2a1d6667ff8f390c0c4dc74c7875c01_9.png)

**1. 画布颜色**
此功能可为视频上下黑边添加纯色背景。你可以选择灰色、红色或其他任何颜色。

![](img/b2a1d6667ff8f390c0c4dc74c7875c01_11.png)

![](img/b2a1d6667ff8f390c0c4dc74c7875c01_7.png)

![](img/b2a1d6667ff8f390c0c4dc74c7875c01_13.png)

**2. 画布样式**
除了纯色，还可以为背景添加图案或图片模板。剪映内置了多种样式可供选择。

![](img/b2a1d6667ff8f390c0c4dc74c7875c01_15.png)

![](img/b2a1d6667ff8f390c0c4dc74c7875c01_9.png)
![](img/b2a1d6667ff8f390c0c4dc74c7875c01_11.png)
![](img/b2a1d6667ff8f390c0c4dc74c7875c01_13.png)

**3. 画布模糊**
这个功能可以创建一种常见的效果：中间视频清晰播放，而上下背景是模糊的视频画面本身。它提供四种模糊强度。

![](img/b2a1d6667ff8f390c0c4dc74c7875c01_15.png)

![](img/b2a1d6667ff8f390c0c4dc74c7875c01_17.png)

通常选择第二或第三档强度即可。选择后点击“应用到全部”。

![](img/b2a1d6667ff8f390c0c4dc74c7875c01_17.png)

至此，我们成功将一个横版（16:9）视频转换为竖版（9:16），并将单调的黑色背景替换为美观的模糊画布背景。

![](img/b2a1d6667ff8f390c0c4dc74c7875c01_19.png)

![](img/b2a1d6667ff8f390c0c4dc74c7875c01_19.png)

## 课程总结

本节课我们一起学习了剪映中视频比例与背景设置的核心功能。

我们主要围绕抖音等平台主流的 **9:16 竖版视频**进行讲解。核心操作是：先将横版视频调整为竖版比例，然后利用“背景”功能，对产生的上下黑边进行美化。可以通过添加**纯色**、**样式图案**或**模糊效果**来丰富画面，从而提升视频的视觉吸引力和推荐潜力。
