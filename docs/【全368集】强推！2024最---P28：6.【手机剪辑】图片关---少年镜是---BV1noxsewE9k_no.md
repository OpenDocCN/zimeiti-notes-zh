# 手机剪辑教程：P28：图片关键帧音乐卡点 🎬

在本节课中，我们将学习如何使用关键帧功能，为图片剪辑添加动态效果，并使其精准地卡在音乐节拍上。这是一种让视频更具节奏感和视觉冲击力的常用技巧。

## 概述
本节教程将分步讲解如何利用关键帧和动画，实现图片随音乐抖动的卡点效果。我们将从素材导入开始，逐步完成动画添加、音乐踩点、关键帧设置以及效果调整。

## 详细步骤

上一节我们介绍了素材准备的基础，本节中我们来看看具体的操作流程。

### 第一步：导入素材与设置时长
首先，在剪映应用中导入两张照片。将第一张照片的时长设置为**10.3秒**，第二张照片的时长设置为**3.6秒**。

### 第二步：添加动画效果
以下是添加动画的具体操作。

1.  选中第一张照片，点击“动画”选项。
2.  为前半段添加“入场动画”中的“轻微抖动3”效果。
3.  为后半段添加“组合动画”中的“入晃动”效果。

### 第三步：设置转场与音乐
完成动画设置后，我们需要连接片段并添加背景音乐。

1.  点击两张照片之间的小白块（转场按钮），添加“推进”转场效果。
2.  导入从抖音收藏好的音乐。
3.  点击音乐轨道，选择“踩点”功能，然后点击“自动踩点”，让软件自动生成节拍点（小黄点）。

### 第四步：为第一张照片添加关键帧抖动
现在，我们让第一张照片的动作跟随音乐节拍。以下是关键帧操作的核心步骤。

1.  选中第一张照片轨道。
2.  将时间线移动到第一个小黄点位置，在此处添加一个关键帧。
3.  将时间线稍微移动到这个小黄点的左右两侧，各添加一个关键帧。
4.  最后，将时间线移回这个小黄点上，用手指按住预览窗口中的照片，将其轻微向下拉动。此时，一个新的关键帧会自动生成，从而创建出向下抖动的效果。
5.  对后面所有的小黄点重复此操作，使照片在每个节拍处都产生抖动。

### 第五步：添加白色闪白效果
为了使转场更炫酷，我们可以在节拍点添加白色闪白效果。

1.  在第二个小黄点位置，点击“画中画”，新增一个画中画轨道。
2.  在素材库中导入一张白色色卡，将其放大至铺满全屏，并使其末尾对齐第三个小黄点。
3.  在白色色卡的开头处打上一个关键帧。
4.  将时间线移动到色卡的尾端，将“不透明度”参数调整为零。这样会形成一个白色渐隐的效果。
5.  后续需要闪白的位置，重复此操作添加白色色卡并设置关键帧。

### 第六步：处理第二张照片
第二张照片的处理方法与第一张照片类似。

1.  选中第二张照片轨道。
2.  在音乐的所有小黄点位置，为其打上关键帧，并创建抖动效果。
3.  添加白色闪白效果的方法与第五步完全相同。

## 总结
本节课中我们一起学习了如何制作关键帧音乐卡点视频。核心流程包括：导入并设置素材时长、添加动画与转场、为音乐自动踩点、通过关键帧制作图片抖动效果，以及利用白色色卡制作闪白过渡。掌握这些步骤，你就能轻松为视频添加富有节奏感的动态效果了。
