# 小红书运营教程：P2：小红书无货源电商入门指南

## 概述
在本节课中，我们将学习如何从零开始，在不囤积任何货源的情况下，于小红书平台开设并运营一家店铺。课程将涵盖基础概念、准备工作以及核心运营思路，旨在帮助新手小白快速理解并上手。

## 讲师介绍
上一节我们介绍了课程主题，本节中我们来认识一下授课讲师。

彤彤老师拥有十多年的电商运营经验：
*   2011年开始从事淘宝运营，曾同时是4家顶级店铺的店主。
*   2015年运营过亚马逊平台。
*   2018至2019年期间，曾作为幕后运营操盘手与抖音平台合作。
*   2023年，通过小红书矩阵运营模式实现电商变现，业绩达到千万级别，是该玩法在小红书平台的早期成功实践者。

彤彤老师最擅长的领域是帮助零基础的初学者，在一周内于小红书平台成功出单，并在一个月内打造出爆款单品实现变现。
