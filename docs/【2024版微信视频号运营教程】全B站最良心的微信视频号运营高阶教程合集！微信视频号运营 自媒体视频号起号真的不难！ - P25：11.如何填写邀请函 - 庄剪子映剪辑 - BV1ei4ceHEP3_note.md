# 微信视频号运营教程：P25：11.如何填写邀请函 📝

在本节课中，我们将学习如何正确填写视频号创作者计划的邀请函。我们将一步步演示如何找到自己的视频号账号信息，并准确无误地完成邀请函的每一项填写，确保顺利加入计划。

---

![](img/c20c8974bdfbda2da0b5b6bf5cecac47_1.png)

上一节我们介绍了视频号的基础概念，本节中我们来看看如何完成一份关键的申请文件——邀请函的填写。整个过程分为几个清晰的步骤。

以下是填写邀请函的具体步骤：

**第一步：打开微信**
首先，在您的手机上找到并打开微信应用。



![](img/c20c8974bdfbda2da0b5b6bf5cecac47_1.png)


![](img/c20c8974bdfbda2da0b5b6bf5cecac47_3.png)

![](img/c20c8974bdfbda2da0b5b6bf5cecac47_4.png)

**第二步：进入“发现”页面**
打开微信后，点击底部导航栏的“发现”选项。



![](img/c20c8974bdfbda2da0b5b6bf5cecac47_3.png)

![](img/c20c8974bdfbda2da0b5b6bf5cecac47_4.png)


**第三步：找到“视频号”入口**
在“发现”页面中，向上滑动屏幕，找到并点击“视频号”入口。该入口有文字标识，图标样式类似一只黄色的蝴蝶。



![](img/c20c8974bdfbda2da0b5b6bf5cecac47_6.png)

![](img/c20c8974bdfbda2da0b5b6bf5cecac47_6.png)

![](img/c20c8974bdfbda2da0b5b6bf5cecac47_7.png)

![](img/c20c8974bdfbda2da0b5b6bf5cecac47_7.png)


**第四步：进入个人中心**
进入视频号页面后，点击右上角的个人头像图标，进入个人中心。

**第五步：查看账号信息**
在个人中心页面，向下滑动，找到“我的视频号”区域。这里会显示您的**账号昵称**和**粉丝数量**。例如，显示为“天道酬勤，39917个粉丝”。在邀请函中，账号昵称填写“天道酬勤”，粉丝数量填写“39917”。

---

在准确找到并记录了账号信息后，接下来我们继续填写邀请函的其他项目。

以下是剩余项目的填写说明：

**第六步：填写邀请人**
由于您是通过特定渠道加入，请在“活动邀请人”一栏填写 **“张良”** 或 **“张良老师”**。

**第七步：填写实名信息**
在“实名信息”一栏，请填写您本人的真实姓名。例如，如果您的名字是“周建国”，则填写 **“周建国”**。

**第八步：填写结算手机号**
在“结算手机号”一栏，填写您用于接收验证码和后续结算的手机号码，格式为连续的11位数字，例如 **`13812345678`**。注意，号码中间不能有空格。

![](img/c20c8974bdfbda2da0b5b6bf5cecac47_9.png)

![](img/c20c8974bdfbda2da0b5b6bf5cecac47_10.png)

**第九步：提交申请**
所有信息填写完毕后，仔细检查一遍。确认无误后，点击页面底部的蓝色“提交”按钮。系统会弹出“确定要提交吗？”的确认框，请点击“确认”以完成提交。



![](img/c20c8974bdfbda2da0b5b6bf5cecac47_11.png)

![](img/c20c8974bdfbda2da0b5b6bf5cecac47_9.png)

![](img/c20c8974bdfbda2da0b5b6bf5cecac47_10.png)

![](img/c20c8974bdfbda2da0b5b6bf5cecac47_11.png)


提交成功后，页面会提示“已成功加入我们的创作者分成计划”。

本节课中我们一起学习了如何填写视频号创作者计划的邀请函。关键点在于通过微信逐步找到自己的视频号账号与粉丝数，并准确填写昵称、粉丝量、邀请人、实名信息和手机号。最后，务必确认所有信息无误后再提交。完成申请后，即可利用提供的正版素材库开始您的视频号运营之旅。
