# 抖音全套运营教程：P2：素材网站 📚

![](img/8a53c832f582ed06eb0df1e734fb2a2e_0.png)

![](img/8a53c832f582ed06eb0df1e734fb2a2e_2.png)

在本节课中，我们将学习如何通过“复制粘贴”的方法，利用国外视频素材在国内自媒体平台（如B站、今日头条）创作内容并获取收益。课程的核心是介绍一系列实用的素材网站，帮助你高效完成文案、视频、图片等素材的搜集与创作。

![](img/8a53c832f582ed06eb0df1e734fb2a2e_4.png)

## 概述

![](img/8a53c832f582ed06eb0df1e734fb2a2e_6.png)

本教程将分享一种通过搬运和二次创作国外热门视频，在国内平台实现内容变现的方法。该方法的核心步骤是：寻找国外高热度视频、提取文案并翻译、下载视频素材、使用剪辑软件一键生成，最后发布到国内平台获取播放量收益。成功的关键在于坚持执行。接下来，我们将详细介绍支撑这一流程所需的各类素材网站。

![](img/8a53c832f582ed06eb0df1e734fb2a2e_8.png)

## 核心操作方法

![](img/8a53c832f582ed06eb0df1e734fb2a2e_10.png)

上一节概述了整体思路，本节中我们来看看具体的操作公式。其核心流程可以概括为以下代码块：

![](img/8a53c832f582ed06eb0df1e734fb2a2e_12.png)

```plaintext
1. 寻找素材：在国外视频平台（如YouTube）找到高点赞、高播放量的视频。
2. 处理文案：提取视频文案 -> 翻译成中文。
3. 处理视频：下载原视频 -> 使用手机剪辑软件的“一键生成”功能进行剪辑。
4. 发布变现：将成品视频发布到B站、今日头条等有播放量收益的平台。
```

![](img/8a53c832f582ed06eb0df1e734fb2a2e_14.png)

根据平台规则，**1万播放量收益约40元**。持续产出内容，累积播放量可以获得可观收入。

![](img/8a53c832f582ed06eb0df1e734fb2a2e_16.png)

## 素材网站大全

![](img/8a53c832f582ed06eb0df1e734fb2a2e_18.png)

支撑上述操作，需要各类工具网站。以下是按类别整理的实用网站列表，每个类别前会有简短介绍。

![](img/8a53c832f582ed06eb0df1e734fb2a2e_20.png)

### 1. 文案类网站

以下是寻找文案灵感和学习案例的网站。

*   **文案狗**：这是一个导航网站，汇集了众多图片和文案素材相关的网站链接。
*   **TOPYS**：该网站提供大量创意文案、视频及图片的拍摄创意，是激发题目和文案灵感的优质来源。
*   **梅花网**：汇聚了互联网上传播较好的各类精彩广告营销案例，每个案例附有分析，可以学习营销传播知识并运用到自己的作品中。
*   **广告门**：这是一个分析优秀广告案例的网站。其“灵感日历”功能非常有用，可以提前查看热点事件日期，便于提前策划内容、把握热点。

### 2. 运营类网站

以下是学习内容运营和追踪热点的网站。

*   **新榜**：最有价值的功能是追踪平台热门内容，了解当下热点，帮助你有针对性地创作，实现“蹭热点”以获得更好流量。
*   **运营派**：这是一个专注于教学运营知识（内容运营、用户运营等）的网站，重点教你如何运营作品，让好作品发挥最大价值。

### 3. 视频素材与灵感类网站

以下是寻找视频灵感、素材和模板的网站。

![](img/8a53c832f582ed06eb0df1e734fb2a2e_22.png)

*   **新片场**：内含大量剪辑精良的视频，可提供创作灵感。同时提供许多可用的视频模板、音乐、图片等素材供下载。
*   **开眼**：这是一个APP，内含众多剪辑精细、拍摄富有创意的视频，是学习拍摄手法和寻找灵感的好去处。

![](img/8a53c832f582ed06eb0df1e734fb2a2e_24.png)

### 4. 图片设计与排版类网站

以下是进行图片设计、排版和处理的工具。

![](img/8a53c832f582ed06eb0df1e734fb2a2e_26.png)

*   **Canva**：提供海量模板，可用于设计封面、拼图等，提供多种字体和图片模板，操作简单，能满足自媒体图片制作需求，也有APP版本。
*   **黄油相机**：其字体风格丰富，可以在制作封面时设计出好看的字体，也适合制作帖子图片。
*   **PixArt**：这是一个图片处理功能较多的网站。其“在一张图片上添加另一张图片”的简易功能操作方便。
*   **美图秀秀**：修饰人像的常用工具，各种美颜参数设置能有效优化图片。
*   **VSCO**：一款以滤镜见长的相机APP，可以为照片添加各种风格的滤镜。

### 5. 图库类网站

![](img/8a53c832f582ed06eb0df1e734fb2a2e_28.png)

以下是寻找高质量图片素材的网站。

![](img/8a53c832f582ed06eb0df1e734fb2a2e_30.png)

*   **Pexels**：提供各种风格（风景、人物、创意等）的高清图片，均可免费下载。
*   **堆糖**：图片风格偏向小清新，也包含一些搞笑有创意的表情包，图片清晰度高。
*   **StockSnap**：分类清晰的高清图片网站，可根据心情、地点或场景便捷查找图片。
*   **Unsplash**：高质量图片网站，可以找到许多富有意境的图片。
*   **FoodiesFeed**：专注于美食的图片网站，展示大量拍摄精美的美食图片，可为美食博主提供拍摄灵感和借鉴。

![](img/8a53c832f582ed06eb0df1e734fb2a2e_32.png)
