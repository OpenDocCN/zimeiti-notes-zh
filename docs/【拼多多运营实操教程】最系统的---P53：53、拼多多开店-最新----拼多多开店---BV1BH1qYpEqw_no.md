## 概述
在本节课中，我们将系统学习拼多多搜索推广（直通车）的核心玩法——14天拖价法。该方法旨在通过一系列精细化操作，在短时间内提升关键词质量分，从而以极低的点击成本获取大量精准流量，最终实现店铺流量破万的目标。课程将从测试、上分、拖价到增流四个步骤，详细拆解每个环节的操作要点与底层逻辑。

## 一、 开车前的准备工作与问题答疑

![](img/de2e4cea0600a459d7f7e1592c1aeebd_3.png)

![](img/de2e4cea0600a459d7f7e1592c1aeebd_4.png)

在正式进入课程核心内容前，我们先解决一些常见的基础问题，并明确开车前必须完成的准备工作。

### 1.1 常见问题解答
以下是针对直播中观众提出问题的集中解答：

*   **标题怎么写？**
    拼多多标题有固定格式：**开头黄金词 + 中间属性词/规格词 + 结尾促销词**。
    *   **黄金词**：核心引流词，可通过“推广中心”->“推广工具”->“搜索词分析”查找。选择时需综合考量点击率、转化率和竞争强度。
    *   **属性词/规格词**：描述产品特性（如服装的款式）或规格（如食品的克重、数量）。
    *   **促销词**：如“包邮”、“拍一发二”，利用价格敏感度提升点击。

*   **标品（如食品、电器）怎么做？**
    标品运营核心是 **价格** 和 **销量**。
    1.  将价格压到有竞争力。
    2.  前期通过活动或多多进宝快速起量，积累基础销量和评价。
    3.  销量起来后，更容易获取类目流量和活动流量。

![](img/de2e4cea0600a459d7f7e1592c1aeebd_6.png)

![](img/de2e4cea0600a459d7f7e1592c1aeebd_7.png)

![](img/de2e4cea0600a459d7f7e1592c1aeebd_9.png)

*   **新店没有流量怎么办？**
    新店启动有标准流程：
    1.  **选品**：选择有潜力的产品。
    2.  **基础优化**：做好主图、标题、详情页（SU）。
    3.  **基础销量与评价**：完成至少50个有效评价，使店铺被平台收录，进入新品流量池。
    4.  **推广**：前期可考虑多多进宝，后期结合搜索/场景推广。

*   **“改销量”有用吗？**
    改销量主要作用是 **提升转化率**（让消费者觉得产品热卖），对搜索权重提升没有直接帮助。属于“锦上添花”，而非“雪中送炭”。

![](img/de2e4cea0600a459d7f7e1592c1aeebd_11.png)

*   **哪些类目适合个人卖家？**
    建议选择竞争较小、有货源或特色优势的类目，例如：
    *   廉价化妆品（如口红）、功效型产品（如祛痘）。
    *   生鲜花卉（如有产地优势）。
    *   活体宠物（如观赏鱼、龟粮）。
    *   小众特色产品。

### 1.2 开车前的必备准备
在开启搜索推广计划前，必须完成以下两点：
1.  **基础销量**：产品需有10-30个基础销量，确保其进入新品流量池，系统才会分配测试流量。
2.  **明确目的**：初期开车的核心目的是 **测试**——测试主图点击率、关键词效果和人群精准度。

## 二、 第一步：测试阶段
测试阶段的目标是花钱获取数据，找出表现最好的主图、关键词和人群，为期约3-7天。

以下是创建测试计划的具体操作步骤：

![](img/de2e4cea0600a459d7f7e1592c1aeebd_13.png)

![](img/de2e4cea0600a459d7f7e1592c1aeebd_14.png)

1.  **创建计划**
    *   进入推广中心，新建搜索推广计划。
    *   推广方案选择 **“自定义”**。

2.  **设置日限额与时间折扣**
    *   **日限额**：先设置为100元。
    *   **时间折扣**：
        *   0:00 - 8:00：设置为 **50%**。切勿选择“不投放”，以保证计划连续性，避免掉分。
        *   8:00 - 24:00：设置为 **100%**。

3.  **添加商品与关键词**
    *   选择要测试的商品。
    *   **关闭** “智能词包”和“流量拓展包”，它们带来的流量不精准，会浪费预算。
    *   从系统推荐词中，手动添加与产品 **高度相关** 的关键词（约5-10个）。
    *   出价统一设为 **市场平均价的120%**。
    *   **精确匹配溢价暂时关闭**。

4.  **设置人群**
    *   为了全面测试，将所有可选人群标签 **全部开启**。
    *   人群溢价统一设置为 **10%**。
    *   地域暂不设置（全选）。

5.  **设置创意（创意图）**
    *   **关闭“智能创意”**，因为它无法判断具体哪张图点击率高。
    *   选择 **“静态创意”**，上传4张不同的创意图。
    *   创意图应使用不同的文案和视觉风格（如全景图、白底图、细节图），以测试点击率。

![](img/de2e4cea0600a459d7f7e1592c1aeebd_17.png)

## 三、 第二步：上分阶段
在获得测试数据后，我们进入上分阶段。目标是快速将关键词质量分提升至9-10分，为后续拖价打下基础。核心是 **理解排名规则并保持排名**。

### 3.1 理解排名与质量分逻辑
*   **综合排名公式**：`综合排名 = 关键词出价 × 质量分`
*   要想以较低出价获得高排名，就必须提升 **质量分**。
*   质量分受以下因素影响：
    1.  **关键词相关性**：关键词需与产品标题、属性高度相关。
    2.  **类目相关性**：关键词与产品发布类目一致。
    3.  **推广商品质量**：主要看 **点击率**、**销量** 和 **销售额**。

### 3.2 上分核心操作：卡排名
*   **目的**：让产品出现在消费者容易看到的位置。数据显示，绝大多数消费者只会浏览前10-15个商品（即前2-3页）。
*   **操作**：
    1.  分析测试数据，筛选出点击率和转化率较好的关键词。
    2.  新建一个推广计划（不再使用测试计划）。
    3.  将这些关键词添加到新计划中，出价参考测试阶段的数据，目标是将关键词排名 **卡在首页（前4-6名）或至少前10名**。
        *   **标品**：建议卡位 **前3名**。
        *   **非标品（服饰鞋包）**：卡位 **7-10名** 即可。
    4.  每天观察排名（每小时刷新），通过微调出价来保持目标排名。每次调价幅度控制在 **5%-10%**。
        *   排名在20名后：加价10%。
        *   排名在11-15名：加价5-8%。

通过3-7天的“卡排名”操作，在保证高点击率和高销量的情况下，关键词质量分将稳步上升至9-10分。

## 四、 第三步：拖价阶段（权重叠加法）
当质量分稳定在高位后，即可进行拖价。这里介绍一种高效的“权重叠加法”，而非单纯降低出价。

### 4.1 核心原理
通过人为干预，在短时间内大幅提升计划的 **点击率** 和 **点击量**，从而快速拉高整体计划权重。权重提升后，系统允许你用更低的出价维持之前的排名。

![](img/de2e4cea0600a459d7f7e1592c1aeebd_30.png)

### 4.2 具体操作步骤（周期约3天）
1.  **选词**：在搜索词分析中，寻找与产品相关但 **竞争强度极低**、**市场平均出价低** 的长尾词（例如“开春连衣裙”）。添加5个左右这样的词到新计划中。
2.  **配合工具**：通过人为操作（如老客户、资源），对这些关键词进行点击。
3.  **执行任务**：
    *   设置较低的日限额（如100元）。
    *   让点击集中在这些冷门词上，将计划的 **整体点击率做到30%以上**（越高越好，无需刻意控制）。
    *   确保计划 **总点击量每天达到80个以上**（若5个词，则每个词约点击16次）。
    *   点击时间建议在 **每天上午10点** 左右进行。
4.  **暂停计划**：日限额花费完毕后，让计划自动暂停。
5.  **观察与迭代**：重复此过程2-3天。随着计划权重飙升，你会发现即使出价很低（如0.1-0.3元），也能获得曝光和点击。
6.  **引入核心词**：2-3天后，将你真正要推广的核心关键词添加到这个高权重的计划中。这些词添加进来后，初始质量分就会很高，从而能够以远低于市场均价的花费获取大量流量。

**注意**：此方法利用了平台规则，旨在快速建立计划权重。操作时需分散点击，模拟真实用户行为。

## 五、 第四步：增流阶段
完成拖价后，你已经拥有了一个权重极高、点击成本极低的推广计划。

1.  **扩大关键词**：在计划中，逐步添加更多相关的关键词。由于计划权重高，这些新词也能以较低成本快速获得展现。
2.  **放大流量**：在投入产出比（ROI）可控的情况下，逐步 **提高日限额**，让系统获取更多流量。
3.  **优化投产**：持续监控数据，淘汰无转化或投产低的关键词，优化人群溢价，实现流量和利润的最大化。

通过以上四个步骤的循环与优化，可以在14天左右，以极低的推广成本，为店铺带来破万甚至数万的精准搜索流量。

## 总结
本节课我们一起学习了拼多多搜索推广的“14天拖价法”完整流程：
1.  **测试**：明确目的，搭建计划，找出优质素材、关键词和人群。
2.  **上分**：理解排名规则，通过“卡排名”提升点击率与销量，从而拉升关键词质量分。
3.  **拖价**：利用“权重叠加法”，通过短期人为提升点击率和点击量，快速提高计划权重，实现低价获客。
4.  **增流**：在低成本的基上，拓展关键词，放大日限额，实现流量规模化增长。
