## 音乐卡点剪辑实操

![](img/6b7529fa6d8df42623050e18d61ebb4a_21.png)

接下来我们来实操一下，像卡点这种剪辑具体怎么做。

![](img/6b7529fa6d8df42623050e18d61ebb4a_23.png)

我们通过录屏来讲解步骤。

![](img/6b7529fa6d8df42623050e18d61ebb4a_25.png)

1.  **导入音乐**：在剪映中导入你选择的音乐，并将其放入轨道。
    ```plaintext
    操作：打开剪映 -> 点击“音频” -> 选择“音乐”并导入。
    ```
2.  **自动踩点**：在音频轨道上选择“踩点”功能，软件会自动读取音乐的节奏节点，并以黄色点的形式标记出来。
    ```plaintext
    操作：选中音频轨道 -> 点击下方工具栏的“踩点” -> 选择“自动踩点”。
    ```
3.  **匹配画面**：导入你想使用的图片或视频片段，将它们一一对准时间轴上的黄色节奏点进行放置。
    ```plaintext
    操作：导入素材 -> 将素材片段拖动到时间轴，使其开头或关键动作与黄点对齐。
    ```
4.  **预览导出**：全部匹配完成后，预览视频效果，确认无误后即可导出。

这种方式相对简单，在各个赛道都适用，大家可以实操实践一下。
