# 拼多多新手教程：P1：产品定价策略详解 💰

在本节课中，我们将学习拼多多新手如何为产品科学定价。定价是电商运营的关键一步，直接影响利润和后续活动参与。我们将通过一个核心公式，帮助您计算出合理的售价，避免亏损。

## 定价核心公式解析

上一节我们概述了定价的重要性，本节中我们来看看具体的定价方法。核心在于一个公式，它能帮您预留出所有必要费用。

定价公式为：
**最终拼单价 = (期望售价 + 优惠券金额) / 0.7**

例如，假设您的产品：
*   期望售价（含利润）为 **20** 元。
*   计划设置 **5** 元优惠券。
*   平台有“7折不计价”活动需预留空间。

计算过程如下：
`(20 + 5) / 0.7 = 35.7`
因此，您应将商品拼单价定为 **35.7** 元。

## 公式组成部分详解

以下是公式中每个部分的详细解释：

*   **期望售价**：这不是成本，而是您希望卖出并包含利润的价格。例如成本10元，您想卖20元，这里就填20。
*   **优惠券金额**：为您计划在未来设置的优惠券（如关注券）预留费用。这部分钱需要从定价中提前留出。
*   **除以0.7**：这是因为拼多多平台有“7折不计价”的大促活动。将价格除以0.7，相当于预留出约30%的降价空间，确保您未来能以活动价参与促销而不亏损。

## 新店过渡策略：新客立减券

对于刚开店、尚未缴纳保证金的商家，无法立即参加“7折不计价”等活动。此时可以采用一个过渡策略。

您可以在后台营销工具中，设置“新客立减券”，将价格从35.7元降低到20元左右进行销售。

> **重要提示**：“新客立减券”不计入商品历史最低价，而其他如普通立减券、优惠券会计入。

## 关键概念：历史最低价与活动价

理解“历史最低价”至关重要，它直接关系到您能否参加平台活动。

*   **历史最低价定义**：指该产品过去30天内的最低成交价格。
*   **其影响**：系统会根据您的历史最低价，给出“活动建议价”。您必须按照这个建议价销售，才能成功报名活动。
*   **错误后果**：如果前期因设置错误的优惠券导致历史最低价被拉得太低，那么系统给出的活动价也会很低。您按此价格销售可能会亏损，若不按此价格则无法参加活动，从而错失流量。

本节课中，我们一起学习了拼多多的产品定价策略。首先，我们掌握了包含利润、优惠券及活动空间的**定价公式**。接着，了解了新店可使用**新客立减券**作为过渡。最后，我们明确了**历史最低价**的概念及其对参与平台活动的决定性影响。请牢记，科学定价是盈利的基础，切勿盲目设置价格或优惠券。
