# 小红书运营：P22：视频剪辑课1-1 🎬

在本节课中，我们将学习如何从抖音获取视频素材，并通过剪辑处理，使其成为适合在小红书发布的原创内容。整个过程将分为素材获取、基础剪辑、片段重组与混剪几个核心步骤，确保即使是零基础的学员也能轻松上手。

---

## 素材获取与准备

上一节我们介绍了课程概述，本节中我们来看看如何获取并准备原始视频素材。

首先，你需要确定你的产品。例如，如果你销售手机壳，就需要去抖音搜索同款产品视频作为素材来源。

以下是获取素材的具体步骤：

1.  **打开抖音，搜索你的产品关键词**（例如“手机壳”）。
2.  **使用筛选功能**：点击搜索结果页面的“小漏斗”图标。
3.  **设置筛选条件**：选择“最多点赞”和“一周内”。这能帮你找到短期内热度高、质量好的视频。半年内的高赞视频同样可以使用。
4.  **浏览并选择素材**：寻找转场较多、画面丰富的视频，这样便于后续剪辑。例如，找到一个展示手机壳多角度、有动作变化的视频。
5.  **下载去水印视频**：不要直接使用抖音的“保存视频”功能，这会带有平台水印。正确方法是复制视频链接，然后使用专门的去水印小程序（如蜗牛pro、青岛、龙猫等），粘贴链接并“一键去水印”，将无水印视频保存到手机相册。

## 基础剪辑：调整比例与分割

获取素材后，我们进入剪辑环节。首先需要使用剪映App进行基础处理。

打开剪映，点击“开始创作”，导入刚刚下载好的视频。

**第一步是调整视频比例以适应小红书平台。** 小红书推荐比例为3:4，而抖音视频通常是9:16。

以下是调整比例的步骤：

1.  在剪映底部工具栏，向右滑动找到“比例”选项，点击并选择 **3:4**。
2.  选中视频轨道，在底部工具栏找到“编辑”或直接点击“裁剪”。
3.  在裁剪界面，将比例设置为 **3:4**。此时画面会被裁切，你需要手动调整画面位置，确保产品主体在画面中央且完整。
4.  裁剪后，视频周围可能出现黑边。使用双指在预览画面中将视频放大，直至填满整个3:4的画布。

**第二步是分离音频并分割视频片段。** 这是为了后续重新编排做准备。

1.  选中视频轨道，在底部工具栏找到“音频分离”，将视频原声分离为独立的音频轨道。
2.  点击视频轨道左侧的“关闭原声”图标，将视频本身的静音。
3.  **分割视频**：观察视频内容，在动作变化或镜头转场处进行分割。例如，从“拿起手机壳”到“放下手机壳”是一个分割点；“盖上盖子”到“开始扭动”是另一个分割点。
    *   **小技巧**：在时间轴上用双指向外拉伸，可以让时间刻度变细，便于更精准地定位和切割。

## 进阶处理：片段重组与混剪

完成基础分割后，我们可以通过两种方式让视频变得“与众不同”。

### 方法一：打乱原片段顺序

这是最简单的混剪方法，即改变原视频片段的播放顺序。

1.  注意保留视频开头的“黄金3秒”片段不动，因为这是吸引用户的关键。
2.  选中分割后的其他片段，长按并拖动，将它们的前后顺序随机打乱。例如，把第二个片段移到第四段的位置，把第五段移到第二段的位置。
3.  通过打乱顺序，你就得到了一条与原视频结构不同的新视频。

### 方法二：替换与混入其他素材

这是更高级的混剪方法，能让视频内容更加丰富。

1.  回到抖音，找到你下载素材的那个原作者主页。
2.  查看他是否发布了同一产品的其他视频。通常一个账号会有多个类似视频。
3.  选择另一个不同的视频，用同样的方法（复制链接、去水印）下载下来。
4.  回到剪映中你正在编辑的项目。
5.  选中你想替换的某个片段（例如第三个片段），在底部工具栏向左滑动，找到“替换”功能。
6.  从手机相册中选择你刚刚下载的第二个视频，并从中挑选一个合适的片段进行替换。
7.  **关键步骤**：替换进来的新片段比例可能仍是9:16。你需要**重复“基础剪辑”中的步骤**：选中该片段 -> 点击“编辑” -> “裁剪” -> 选择 **3:4** 比例 -> 调整画面 -> 放大至满屏。
8.  可以替换2-5个片段，具体数量根据视频总时长决定。
9.  替换完成后，务必**将新片段的音量降至0**，以免多个背景音混杂。选中替换的片段，点击“音量”，将滑块拉到最左。

## 去重与最终调整

最后，我们需要对视频进行一些微调，以进一步降低被平台识别为搬运的风险。

**重点处理视频的开头部分（第一帧或第一个片段）：**

1.  **镜像翻转**：选中开头片段，在“编辑”中找到“镜像”功能并点击。这会使画面左右反转。
2.  **缩放与位移**：稍微放大或缩小画面，或上下左右轻微移动画面位置。
3.  **旋转角度**：在“编辑”的旋转功能中，将画面向左或向右旋转1-2度。这个变化肉眼难以察觉，但能有效改变视频的底层数据。

完成以上所有步骤后，预览整条视频，检查画面衔接是否流畅、比例是否正确、是否有杂音。确认无误后，即可导出视频，准备在小红书发布。

本节课中我们一起学习了小红书视频剪辑的第一种核心方法：**单素材混剪**。我们掌握了从抖音获取无水印素材、在剪映中调整3:4比例、分割音频与视频、打乱片段顺序，以及通过替换素材和进行镜像、缩放、旋转等操作进行深度去重的完整流程。记住，处理开头画面和统一视频比例是关键步骤。下节课我们将学习如何将多条不同素材混合剪辑成一条新视频。
