## 一、完全复制

完全复制是指对从小红书或其他平台获取的完整笔记内容，进行三次小修改。其目的是提升笔记的权重属性，并降低被小红书系统判定为重复内容的概率。

**核心公式**：`原始笔记` + `三次小修改` = `降低重复率，提升基础权重`

以下是具体操作步骤：

1.  **视频修改**：
    *   使用剪映等剪辑软件，将完整视频分割成三段，然后重新组合发布。
    *   此操作会留下剪辑痕迹，能有效排除90%-95%的重复数据量。系统通过识别视频的秒和帧来判断重复，分割操作能有效“欺骗”系统。

2.  **图片修改**：
    *   使用任何图片编辑软件进行修改。
    *   例如，截图后使用，或在图片不显眼处添加一两个小黑点/白点。
    *   同样，经过2-3次操作后，可排除约95%的竞争对手。

3.  **文字修改**：
    *   复制文字后，修改字体、大小、显示形式（如简体转繁体），或调整排版（横放、斜放）。
    *   文字修改的效果通常比视频和图片更好，能让系统判定为接近全新的内容。

**适用场景**：完全复制操作简单快捷，适合新手或不熟悉小红书操作的用户，主要用于引流。由于同类产品笔记众多，其本身权重不高，直接复制不修改效果甚微。

## 二、主体复制

主体复制与完全复制不同，它需要更多的创造性工作。其核心是复制产品的整体框架，但舍弃原文案，由自己重新编写产品宣传、折扣信息或标题。

![](img/00a996d96b1d4df7de3dbcb06f99af10_13.png)

![](img/00a996d96b1d4df7de3dbcb06f99af10_15.png)

**核心概念**：`复制产品框架` + `自创文案` = `保留结构，注入新意`

*   操作方法是：将别人的图片和文案框架借鉴过来，然后进行“手术式”修改。保留有用的部分，替换掉无用的部分，并加入自己产品的特色内容。
*   如果想蹭原笔记的流量，更适合用“完全复制”。如果只想借鉴其文案格式或视频录制方法，则使用“主体复制”。
*   **主要适用于图片和文字笔记**。因为视频笔记通常是一套完整的系统，较难进行主体层面的替换。小红书平台上，图文笔记的占比超过85%。

## 三、格式复制

格式复制是另一种精细化操作方式。它保留原笔记精彩的文案和推广格式，但更换产品的主图或图片，使其与自己的产品相匹配。

**核心概念**：`保留精彩文案与格式` + `更换产品图片` = `借力优质文案，展示自家产品`

![](img/00a996d96b1d4df7de3dbcb06f99af10_17.png)

*   当发现一篇笔记的文案非常出色，但图片质量一般或与自家产品不符时，可采用此方法。
*   操作关键是：替换掉所有图片，并将文案中的核心关键词（如产品名称）替换为自己的产品信息。

## 总结与过渡

本节课我们一起学习了小红书笔记的三种内容修改方式：**完全复制**、**主体复制**和**格式复制**。无论采用哪种方式，核心原则都是**必须对复制来的内容进行修改**，以避免被系统判定为高度重复内容，从而失去基础权重。

这种修改的必要性源于小红书早期存在的大量“店群”模式。这些店群经营者会海量上架产品并发布笔记，导致平台存在巨量的重复和低质内容。系统为了维持内容质量，会屏蔽这些重复内容。通过我们的修改操作，可以有效避开这90%-95%的无效竞争池，为自己争取到有效的曝光空间。
