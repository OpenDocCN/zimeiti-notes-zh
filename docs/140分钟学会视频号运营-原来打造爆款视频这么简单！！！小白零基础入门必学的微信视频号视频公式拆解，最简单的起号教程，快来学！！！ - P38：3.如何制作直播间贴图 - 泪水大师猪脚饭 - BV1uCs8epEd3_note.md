# 视频号运营：P38：如何制作直播间贴图 🎨

在本节课中，我们将学习如何为视频号直播间制作一个透明的文字贴图。这种贴图常用于展示商品信息、促销活动或直播间标题，能有效吸引观众注意力并传递关键信息。

## 概述

许多直播间会使用文字贴图来突出显示信息，例如“好物推荐 老国货 9.9元1盒”。本节教程将详细拆解制作这种透明背景贴图的完整流程，从工具选择到最终在直播间应用，每一步都会进行说明。

## 制作步骤详解

上一节我们准备好了工具，本节中我们来看看具体的制作步骤。整个过程可以分为选择模板、编辑内容、设置透明背景和保存导出几个环节。

以下是使用“微麦水印相机”制作贴图的具体步骤：

![](img/0f792f0d3d90bee7640b147234357a4a_1.png)

1.  **打开应用并找到模板**：启动App后，在主界面找到“水印库”或类似入口。点击进入后，选择“更多水印”或“水印模板”。
2.  **选择文字模板**：在模板库中，寻找“横竖文字”类别的模板。例如，可以选择一个简单的横向排列文字模板作为基础。
3.  **编辑文字内容**：点击模板，进入编辑界面。将示例文字替换为你需要的直播间信息。例如：
    *   `好物推荐`
    *   `老国货品牌`
    *   `9.9元1盒`
    *   `19.9元3盒`
    *   `29.9元5盒`
4.  **自定义样式**：你可以调整文字的颜色、字体，使其更符合直播间的视觉风格。
5.  **关键设置：透明背景**：找到“背景色”或“背景”设置选项。**必须将背景设置为完全透明**。通常，需要选择一个带有斜杠（/）标识的“无背景”或“透明”选项，并将背景透明度参数调整至0。
6.  **生成并保存图片**：编辑完成后，点击“生成”或“保存”按钮。将制作好的透明贴图保存到你的手机相册中。

## 在直播间应用贴图

![](img/0f792f0d3d90bee7640b147234357a4a_5.png)

制作好贴图后，我们需要将其添加到直播画面中。这个步骤在视频号的直播发起界面完成。

![](img/0f792f0d3d90bee7640b147234357a4a_7.png)

以下是添加贴图到直播间的操作流程：

![](img/0f792f0d3d90bee7640b147234357a4a_9.png)

1.  **发起直播**：在视频号中点击“发起直播”。
2.  **打开贴图功能**：在直播准备界面，找到“画面特效”、“贴纸”或“图片”添加功能（不同版本位置可能略有不同）。
3.  **选择本地图片**：从功能列表中选择“手机相册”或“从相册选择”。
4.  **导入并调整**：从相册中选中你刚才制作好的透明贴图。贴图会出现在直播画面上，你可以用手指拖动来调整其位置，双指缩放来调整大小，直到满意为止。
