# 抖音运营教程：P5：3.账号定位 🎯

![](img/fcca669a0a4b30fea1f6eec2f7a26a10_0.png)

在本节课中，我们将学习抖音账号运营的核心环节——账号定位。一个清晰、精准的定位是账号快速起号、获得高权重的基础。我们将通过反思三个核心问题来构建你的账号定位框架。

![](img/fcca669a0a4b30fea1f6eec2f7a26a10_2.png)

---

上一节我们介绍了赛道选择的重要性，本节中我们来看看如何具体进行账号定位。定位的本质是向平台和用户清晰地回答三个问题：你是谁，你是干什么的，以及为什么要关注你。

![](img/fcca669a0a4b30fea1f6eec2f7a26a10_0.png)

首先，我们需要确定账号的类型。这不仅仅是选择一个大的赛道（一级垂类），更要深入思考其下的细分领域（二级垂类）。

例如，参加运动会时，你选择“田径”这个大项目（一级垂类）。如果你最擅长且具有优势的是“百米赛跑”，那么“百米赛跑”就是你的二级垂类。同理，通过定位账号的细分类目，可以确定账号的垂直类型，从而在庞大的流量池中更精准地吸引目标用户。

![](img/fcca669a0a4b30fea1f6eec2f7a26a10_2.png)

在这张图中可以看到，在抖音，只要定位足够清晰，任何行业都有机会。因为抖音流量巨大，一旦做起来，增长和裂变速度会非常快。

其次，我们要确定账号主体是谁。账号主体是账号的拥有者，也是最终呈现给粉丝的主人公。这需要让粉丝明确知道他们是在看谁的作品。

在抖音商业类账号中，账号主体主要分为两类：
*   **商家账号**：主体是公司。内容需偏向商业化，同时要通过内容不断拉近与粉丝的距离，避免过于严肃。这样既能建立信任感，也能带来好感，更符合抖音的娱乐属性。
*   **达人账号**：主体是个人。首要任务是解决信任问题。可以通过IP打造的方式，为账号树立一个积极、正面的人设。

然后，我们需要明确这个账号是做什么的。最直接的方式是通过抖音的官方认证来表明身份。

*   如果是个人达人，可以申请**黄V认证**。
*   如果是公司，可以申请**蓝V认证**。

认证后，粉丝可以清晰地看到认证标签，从而明白你的职业和账号的大致方向。同时，认证也能获得平台背书，增加信任感。

在未认证之前，也可以通过账号的基础设置向粉丝传达你是做什么的。以下是进行业务关联统一设置的关键点：
*   头像
*   背景图
*   个人简介
*   视频封面

最后，我们必须花时间思考一个根本问题：**你将为粉丝提供什么样的价值？** 在抖音，粉丝对博主的价值需求主要分为以下几类：

**1. 知识价值**
抖音中的知识一般分为几种类型：
*   **认知型**：如人生思考、思维方式、读书分享、行业内幕、成功经验、个人提升等。
*   **干货型**：如技能提升、经验总结、操作步骤、冷知识等。
*   **常识普及型**：如生活、健康、产品、科学、历史、地区等常识的解读和普及。

这类账号提供的价值公式是：**认同感 = 三观认同 + 观点认同 + 减少知识盲区的获得感**，最终转化为对作者的认同。

**2. 情绪价值**
情绪是影响观众的重要因素。作品中容易引发共鸣的情绪类型包括：
*   **开心愉悦**：如搞笑、夸张表情感染类内容。
*   **人物关系共鸣**：如情侣、婆媳、夫妻、家庭、上下级、师生等关系话题。
*   **情绪波动**：由争议性话题、剧情反转、内容反差、反常识开场等激发的情绪。
*   此外，怀旧、伤感、震惊、好奇、颠覆三观等也是用户容易产生的情绪。

这类账号提供的价值核心是**共鸣**，即利用表情、文字、画面、声音、肢体动作等元素，将情绪准确到位地传递给观众，直接感染他们。

**3. 感觉价值**
博主通过短视频的画面、文字和声音传递信息。这类价值包括：
*   **视觉享受**：画面美感、构图、治愈感、视觉冲击力，以及对作者人格魅力的仰慕感。
*   **听觉享受**：视频声音带来的节奏感、心跳加速或听觉愉悦。

常见于颜值、萌宠、美景、才艺、明星、文化等领域。这类账号提供的价值是**共情**，即因为表情、文字、画面、声音、肢体动作等的自然表现，让观众产生联想，并在情绪、认知等意识层面与作者产生连接，从而间接引发共鸣。

**4. 体验价值**
抖音让用户看到了不一样的生活方式。体验价值在于让用户：
*   了解自己到不了的地方。
*   体验未曾体验过的事物。
*   感受不曾体会过的感觉。
*   观看他人做自己想做却做不到的事。

例如流量很大的美食账号（享用/制作体验）、旅行、运动、角色体验等。这类账号主要满足粉丝的**好奇心**，带来紧张、激动、感动等体验和获得感。

**5. 期待价值**
当用户被你的视频内容、观看体验或对作者本人产生好奇时，他们会在看完后意犹未尽，从而点击关注，**期待你的下一次更新**。许多博主会利用这一点，不会在一个视频中呈现所有内容，以制造悬念。

本节课中我们一起学习了抖音账号定位的完整框架。你需要依次确定：**账号的细分类型**、**明确的主体身份**、**清晰的业务内容**，并最终思考你能为粉丝提供的核心**价值**（知识、情绪、感觉、体验或期待）。一个精准的定位是账号获得高权重、实现快速起号的基石。下一节，我们将探讨如何基于定位进行内容规划。
