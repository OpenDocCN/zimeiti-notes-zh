# 剪映手机版教学：P49：入门练习：分屏蒙版卡点效果

在本节课中，我们将学习如何利用剪映的蒙版功能，制作一个分屏卡点视频。教程将分为三个核心部分：**使用矩形蒙版创建分屏效果**、**根据音乐节奏让分屏逐格出现**，以及**利用动画制作梦幻闪烁效果**。通过本课的学习，你将掌握蒙版与音频卡点结合的基础技巧。

## 课程概述

我们将通过一个实战案例，学习制作分屏蒙版卡点视频。主要流程包括：导入素材与音乐、进行音频踩点、使用文字辅助对齐蒙版、复制并调整多个蒙版图层以实现分屏，最后为分屏片段和梦幻效果片段添加动画。核心在于理解蒙版的位置调整与音频节奏的匹配。

## 第一步：素材准备与音频踩点

首先，我们需要准备视频素材和背景音乐，并对音乐进行节奏分析，这是制作卡点视频的基础。

1.  打开剪映，点击“开始创作”，导入小女孩的视频素材。
2.  点击“添加音频”，选择“提取音乐”，从手机中导入准备好的背景音乐。
3.  选中音频轨道，点击“节拍”，选择“自动踩点”。软件会自动分析音乐节奏并生成卡点标记。本例中音乐有8个明显的节奏点，我们将视频对应地分为8段。

> **提示**：制作卡点视频的第一步始终是处理音乐节奏，确保后续视频剪辑能与节拍同步。

## 第二步：创建分屏蒙版布局

上一节我们完成了音乐踩点，本节中我们来看看如何利用蒙版精确划分屏幕。这里会使用一个文字技巧来辅助我们对齐。

1.  返回主菜单，点击“文本”，新建一个文本。
2.  在文本框内输入字母 **`I`**。由于我们需要8个分屏，因此输入 **9个 `I`**（分屏数+1）。公式为：**`文本“I”的数量 = 所需分屏数量 + 1`**。
3.  点击“样式”，在“排列”选项中调整“字间距”，将9个 `I` 均匀拉开，直至两端贴近屏幕边缘。这样，`I` 与 `I` 之间的空隙就形成了8个等宽的参考区域。
4.  选中视频素材，点击“蒙版”，选择“矩形蒙版”。
5.  先将矩形蒙版拉长至覆盖整个屏幕高度，然后缩窄其宽度，使其与一个 `I` 的宽度大致对齐，并微调位置，让蒙版两侧与 `I` 之间留有细小、均匀的缝隙。
6.  完成第一个蒙版调整后，选中该视频片段，点击“复制”。我们需要复制出总共 **8个** 带有相同蒙版的视频片段。
7.  将复制出来的片段依次“切画中画”，排列在下方轨道，形成8个完全相同的图层。

> **提示**：先统一调整好一个蒙版的粗细，再通过复制生成其他图层，之后只需移动蒙版位置，无需重复调整大小，这样可以保证所有分屏缝隙宽度一致。

## 第三步：调整蒙版位置与卡点出现

现在我们已经有了8个分屏图层，接下来需要将它们的蒙版移动到正确位置，并根据音乐卡点让它们依次出现。

1.  从第二个画中画图层开始，依次选中每个片段，点击“蒙版”，然后**仅移动**矩形蒙版的位置，使其分别对齐第2到第8个参考空隙。操作完成后，屏幕被均匀分为8个部分。
2.  删除之前用于辅助对齐的文本图层。
3.  根据音频轨道上的8个卡点标记，我们来安排每个分屏的出现顺序。原则是：**分屏的出现顺序不要相邻**，可以一左一右或一上一下交错出现，以增强动感。
    *   例如，主轨道（第一个分屏）在第一个卡点出现。
    *   然后，在第二个卡点，让画中画中靠右的一个分屏出现（将其片段起始端裁剪并对齐第二个卡点）。
    *   在第三个卡点，让画中画中靠左的一个分屏出现。
    *   依此类推，直到所有8个分屏在8个卡点内全部出现完毕。
4.  播放检查，确保每个分屏的出现都精准对应一个音乐节拍，并且所有黑色缝隙均匀。

## 第四步：制作梦幻闪烁效果

当所有分屏出现后，接下来我们制作中间段落的梦幻闪烁效果。这里我们将利用“画中画”和“动画”功能来实现。

1.  在最后一个分屏出现后的音乐节奏点（通常是连续的“咚咚”声开始处），将主轨道及所有画中画片段在此位置进行“分割”，并删除后面多余的部分。
2.  再次导入小女孩的同一段或另一段视频素材，将其“切画中画”并拖放到上一组片段结束的位置。
3.  放大时间线，根据音乐节奏，每隔两个节奏点（或约0.2秒）对这个新的画中画片段进行一次“分割”，将其切成多个短片段。
4.  为这些短片段交替设置动画：
    *   选中第一个短片段，点击“动画”，在“入场动画”中选择“渐显”，时长拉满至0.2秒。
    *   选中第二个短片段，点击“动画”，在“出场动画”中选择“渐隐”，时长拉满至0.2秒。
    *   后续片段重复此模式：**“渐显” -> “渐隐” -> “渐显” -> “渐隐”...**
5.  播放预览，即可看到人物随着节奏产生一种朦胧、闪烁的梦幻视觉效果。

## 课程总结

本节课中我们一起学习了分屏蒙版卡点效果的制作。
1.  **核心技巧**是使用**矩形蒙版**分割画面，并通过**复制图层**和**移动蒙版位置**来快速创建均匀分屏。
2.  **关键步骤**是将分屏的出现时间与音乐的**自动踩点**节拍对齐，实现卡点效果。
3.  **进阶效果**通过**分割片段**并交替添加**“渐显”** 和**“渐隐”** 动画，制作出梦幻闪烁的转场。

通过这个练习，你不仅掌握了蒙版的基础应用，也学会了如何将视觉变化与音频节奏紧密结合，这是制作动感短视频的重要技能。课后请多加练习，尝试调整分屏数量、出现顺序和动画时长，创造出属于你自己的卡点视频。
