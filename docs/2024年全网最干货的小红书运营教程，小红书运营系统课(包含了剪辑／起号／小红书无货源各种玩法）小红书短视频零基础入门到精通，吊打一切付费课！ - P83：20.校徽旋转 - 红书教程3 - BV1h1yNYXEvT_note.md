# 小红书运营系统课：P83：20.校徽旋转

## 概述
在本节课中，我们将学习如何使用剪映制作一个校徽旋转叠加在人物上的短视频效果。这个技巧常用于制作纪念视频或个性化内容，操作简单，效果出众。

---

上一节我们介绍了基础的画中画功能，本节中我们来看看如何结合关键帧与混合模式，制作动态叠加效果。

首先，在剪映中导入两张素材图片：一张人物照片和一张校徽图片。

将人物图片的时长调整到合适的秒数。接着选中校徽图片，点击“切换画中画”功能，将其转换为画中画图层，并与下方的人物图片时间轴对齐。

以下是后续的核心操作步骤：

1.  再次选中主轨道上的人物图片，点击“复制”。
2.  选中复制得到的人物图片片段，再次点击“切换画中画”，将其也转为画中画图层，并确保它与前两个图层在时间轴上起始位置对齐。
3.  选中这个新的人物画中画图层，点击工具栏中的“智能抠像”功能，将人物从背景中分离出来。
4.  现在，选中校徽所在的画中画图层，点击“混合模式”，并选择“叠加”模式，使校徽能与下方图像融合。

完成基础叠加后，我们为校徽添加旋转动画。这需要用到关键帧功能。

在校徽图层的**开始位置**，点击添加关键帧的按钮。然后，将时间线移动到图层的**结尾位置**，再次点击添加关键帧。

在结尾处的关键帧状态上，手动将校徽图片**旋转360度**。这样，剪映会自动在起始和结尾两个关键帧之间生成平滑的旋转动画。

---

## 总结
本节课我们一起学习了如何利用画中画、智能抠像、混合模式叠加以及关键帧旋转，制作出校徽环绕人物的动态视频效果。掌握这些基础动画组合，你可以创作出更多个性化的视觉内容。
