# 新媒体运营课程：P29：11：小红书平台用户标签及流量入口 📊

![](img/3d43609c1c4fcdbdac700c94c188aff7_1.png)

在本节课中，我们将学习小红书平台如何为用户打上标签，以及平台内主要的流量入口有哪些。理解这些机制，有助于我们分析笔记如何被系统推荐给目标用户，从而优化运营策略。

上一节我们介绍了系统如何判定内容标签，本节中我们来看看平台如何判定用户标签。这个过程能帮助我们反向推导，如何让笔记被精准用户看到。

## 用户标签判定机制

系统主要依据用户的**浏览行为**来判定其标签。用户注册时选择的兴趣范围参考价值不大，仅作为初期内容引导，无法进入系统深层的用户标签池。

以下是系统判定用户兴趣分类的主要行为依据：

### 1. 点击率
**点击率**指系统在发现页向用户推送多个内容（如美妆、家居、母婴等）后，用户点击各类内容的概率。例如，用户点击了10条内容，其中美妆类3次，家居类2次，母婴类0次。系统会判定该用户对美妆更感兴趣，对母婴不感兴趣，并持续推送更多美妆内容进行试探。

### 2. 停留时长
**停留时长**指用户在单篇笔记页面的观看时间。例如，用户虽然常点击美妆内容，但多数快速划走。此时，系统会尝试推送美妆下的细分领域（如上妆、护肤）或其他类别（如家居）。如果用户对某篇家居笔记停留很久，仔细阅读图片和评论，系统可能判定其有装修或购买家居用品的需求。

### 3. 页面浏览量
**页面浏览量**可以理解为用户浏览的深度。例如，用户不仅看了某篇装修笔记，还点进博主主页，并浏览了第二、第三篇笔记。提升用户在自己主页的浏览深度，对账号健康度评分有积极帮助。

### 4. 互动行为
互动行为包括**点赞、收藏、评论、关注**。这些数据直接提升笔记的互动率，能帮助笔记进入更大的推荐流量池，同时也有助于账号涨粉和后续变现。

通过以上行为，系统会为用户生成一个标签云图。例如，一个用户的“美妆”标签最大，代表他是美妆内容的偏好者，系统便会主要推送相关内容。平台、用户与博主三者形成关联：博主提供有价值的内容给用户，平台则负责将合适的内容匹配给用户。

## 平台流量入口解析

了解用户标签后，我们来看看小红书主要的流量分发入口。平台目前主要有四个流量入口。

以下是各入口的详细说明：

### 1. 发现页
发现页是笔记获取流量的主要来源。其内容构成包括：
*   **已关注的达人**发布的内容。
*   根据用户**历史点击率**等行为标签，系统推荐的“可能喜欢”的内容。
*   少量**发散性内容**，用于增加平台内容的多样性，探索用户潜在兴趣。

### 2. 关注页
关注页的内容非常简单，仅展示用户**已关注的所有博主**发布的笔记。

### 3. 附近页
附近页基于**手机定位**，推送同城或附近的内容，尤其适合具有强地域特征的需求，如美食、旅游、线下店推广等。做本地实体店推广的运营者，需要重点关注此入口。

### 4. 购物页
购物页是小红书向电商化发展的重要入口，直接匹配用户的购物需求。目前，用户习惯仍在培养中，许多人仍倾向于通过搜索笔记来“种草”。
