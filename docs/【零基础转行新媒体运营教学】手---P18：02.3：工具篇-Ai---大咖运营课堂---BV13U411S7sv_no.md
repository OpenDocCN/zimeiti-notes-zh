## 两种核心生成模式
该工具提供两种主要的内容生成方式，以下是具体介绍：

**1. AI智能生成**
此模式由工具根据用户输入的主题，自动构思大纲并填充内容。
*   **操作方式**：在指定区域输入PPT主题，点击生成。
*   **特点**：适合快速起稿，但生成内容的精确度和细节深度可能需进一步调整。

**2. 导入本地大纲**
此模式需要用户预先准备好详细的文档大纲，工具基于此结构生成PPT。
*   **操作方式**：将排版好的文档内容粘贴或导入到工具中。
*   **特点**：生成结果更可控，排版更贴近预期，是推荐的主要使用方式。

## 实战演练：从大纲到PPT
我们将通过一个“装修行业在小红书平台的运营推广计划”的案例，演示如何使用“导入本地大纲”模式。

**第一步：准备详细内容大纲**
首先，我们需要一份结构清晰、内容详实的大纲文档。可以使用AI助手（如GPT-4）辅助生成和润色。
```markdown
# 装修行业小红书运营推广计划
## 一、市场与平台分析
### 1.1 小红书平台特点
- 用户以年轻女性为主，消费决策受内容影响大。
- 内容形式以图文、短视频笔记为核心。
### 1.2 装修行业内容趋势
- 当前流行“奶油风”、“原木风”等风格分享。
- 用户关注装修避坑、软装搭配、预算控制等实用技巧。
## 二、目标用户画像
### 2.1 基本属性
- 年龄：25-35岁。
- 性别：女性占比70%以上。
- 城市：以一二线城市新房业主、旧房改造业主为主。
...
```
**关键提示**：大纲层级（如`#`， `##`， `###`）和段落格式会影响工具对内容结构的解析，建议提前整理好。

**第二步：导入大纲并生成PPT**
1.  在工具中选择“导入本地大纲”功能。
2.  将准备好的大纲内容粘贴到输入框。
3.  工具会快速解析文档结构，形成预览大纲。
4.  从丰富的模板库中，选择一个符合“装修行业”风格的模板。
5.  点击“生成PPT”，等待片刻即可完成初稿。

**第三步：编辑与优化**
生成初稿后，可以在工具的编辑界面进行精细化调整：
*   **更换单页版式**：点击页面，可从右侧提供的多种版式中一键替换。
*   **整体更换模板**：使用“模板替换”功能，为整个PPT快速更换另一套设计风格。
*   **编辑内容与样式**：在右侧面板中，可以修改文字、形状、背景等所有元素的样式。
*   **完善大纲内容**：如果发现某些页面内容缺失，可以返回“大纲编辑”视图，在对应节点下补充详细文字，系统会同步更新。
