# 新媒体运营课程：P29：11：小红书平台用户标签及流量入口 📊

在本节课中，我们将要学习小红书平台如何为用户打上标签，以及平台主要的流量入口有哪些。理解这些机制，能帮助我们分析笔记如何被推荐给用户，并指导我们创作出更易被目标用户看到的内容。

上一节我们介绍了小红书如何判定内容标签，本节中我们来看看平台如何判定用户标签。这个过程能帮助我们分析和理解自己的笔记是如何被系统推荐给用户的，并能从背后的逻辑反推，如何让笔记被我们的精准用户看到。

## 用户标签的判定逻辑

![](img/dbb09d31594ec1af706f9c6412c787d4_3.png)

小红书系统为用户打标签，主要依据用户的**浏览行为**。用户刚注册时选择的兴趣范围，参考价值不大，它仅作为平台对用户的内容方向引导，进行初步筛选，但无法进入系统最深层的用户标签池。

系统判定用户的兴趣分类，完全基于其浏览行为。以下是系统分析用户行为的主要维度：

### 1. 点击率
**点击率**是指系统在“发现页”向用户推送多个不同领域（如明星、美妆、家居、母婴、装修）的内容后，用户点击各类内容的频率。

![](img/dbb09d31594ec1af706f9c6412c787d4_11.png)

![](img/dbb09d31594ec1af706f9c6412c787d4_13.png)

![](img/dbb09d31594ec1af706f9c6412c787d4_15.png)

![](img/dbb09d31594ec1af706f9c6412c787d4_17.png)

例如，用户点击了10条内容，其中美妆类3次，家居类2次，母婴类1次或0次。系统通过算法可以判定：该用户对美妆内容更感兴趣，对母婴内容可能不感兴趣（例如，用户可能不是一位宝妈）。

![](img/dbb09d31594ec1af706f9c6412c787d4_19.png)

基于点击率，系统会持续推送更多美妆领域的内容来试探用户。通过用户对不同内容的点击偏好，系统为用户打上**强关联**和**弱关联**的标签。

![](img/dbb09d31594ec1af706f9c6412c787d4_21.png)

![](img/dbb09d31594ec1af706f9c6412c787d4_22.png)

为了便于理解，我们可以想象一个标签云图。假设平台为用户打上了“美妆”、“美甲”、“美发”、“母婴”、“带娃”、“装修”等标签。如果“美妆”这个标签的权重最大，就代表该用户是美妆领域的深度爱好者。系统后续便会主要为其推送美妆相关内容，减少或不推送其不感兴趣的内容（如母婴），从而提升用户的获得感与停留时长。这就是系统为用户打标签希望达成的目的。

### 2. 停留时长
**停留时长**是另一个关键的分析行为。即使用户喜欢美妆内容，但如果他点开美妆笔记后很快划走，说明这些具体内容并非其喜好。系统会继续推送美妆领域内其他细分内容（如上妆、护肤等）来观察用户反应。

![](img/dbb09d31594ec1af706f9c6412c787d4_29.png)

![](img/dbb09d31594ec1af706f9c6412c787d4_31.png)

反之，如果用户对平时点击较少的家居类内容，某次点开后停留了很久，仔细浏览图片和评论，系统就可能判定该用户有潜在的装修或购买家居用品（如盘子）的需求，从而考虑为其推送更多相关笔记或商品。

### 3. 页面浏览量
**页面浏览量**可以理解为用户浏览页面的深度。例如，用户点开一篇装修笔记后，不仅看了这篇笔记，还进入了博主主页，并连续浏览了该博主的第二、第三、第四篇笔记。这个浏览的页面数就是PV值。

![](img/dbb09d31594ec1af706f9c6412c787d4_33.png)

![](img/dbb09d31594ec1af706f9c6412c787d4_34.png)

![](img/dbb09d31594ec1af706f9c6412c787d4_35.png)

如果博主主页内容做得非常精致，就能有效提升用户在其页面的浏览深度。这个数值对账号健康度有非常大的帮助。当浏览深度足够时，用户很可能会产生**关注**行为。关注是永久性的加分项，能提升账号质量。即使在粉丝基数不大时，提升主页的浏览深度也能为账号带来非常健康的评分。

### 4. 互动行为
互动行为包括**点赞、收藏、评论、关注**。这些数据是我们在创作每篇笔记时都需要努力去提升的指标。

![](img/dbb09d31594ec1af706f9c6412c787d4_37.png)

![](img/dbb09d31594ec1af706f9c6412c787d4_38.png)

![](img/dbb09d31594ec1af706f9c6412c787d4_39.png)

![](img/dbb09d31594ec1af706f9c6412c787d4_40.png)

*   点赞、收藏数据高，笔记获得下一轮推荐流量的几率就更高。
*   数据优秀的笔记不仅能带动账号内其他笔记的流量攀升，还能提升粉丝数，为后续接单、变现打下基础。

因此，提升互动行为是我们运营小红书一直努力的目标。

## 平台的流量分发入口

了解用户标签后，我们来看看小红书主要的流量分发入口。目前小红书平台主要有四个流量入口：发现页、关注页、附近页和购物页。

![](img/dbb09d31594ec1af706f9c6412c787d4_47.png)

以下是各入口的详细介绍：

*   **发现页**：这是笔记和账号后期获得流量的主要来源。发现页呈现的内容主要包括：
    1.  用户已关注的达人发布的内容。
    2.  系统根据用户标签（点击率、停留时长等）判断其可能喜欢的内容。
    3.  系统也会推荐一些用户可能不感兴趣或从未刷过的内容，以增加平台内容的多样性，满足用户（如看攻略、教程、趣事分享）的多样化需求。

![](img/dbb09d31594ec1af706f9c6412c787d4_49.png)

![](img/dbb09d31594ec1af706f9c6412c787d4_50.png)

![](img/dbb09d31594ec1af706f9c6412c787d4_52.png)

![](img/dbb09d31594ec1af706f9c6412c787d4_54.png)

*   **关注页**：这里集中展示用户所有已关注博主发布的内容。虽然用户直接刷关注页的频率可能不如发现页高，但对于关注了大量博主的用户，这里是查找强关联博主内容的直接入口。

![](img/dbb09d31594ec1af706f9c6412c787d4_56.png)

*   **附近页**：此入口基于手机定位，向用户推送附近或同城的相关内容。内容通常以**美食、本地旅游、线下服务**等强地域特征的需求为主。如果你是线下实体店（如美容院、餐厅、景点）的运营者，想通过小红书推广，就需要特别注意“附近”人群的推送，在做信息流广告时可侧重此板块。

![](img/dbb09d31594ec1af706f9c6412c787d4_58.png)

*   **购物页**：这是小红书向电商方向发展的重要入口，主要匹配用户的购物需求。目前，部分用户尚未养成在小红书直接购物的强习惯，更多人仍倾向于搜索笔记获取推荐。但购物入口是小红书变现和直播带货的关键板块。
