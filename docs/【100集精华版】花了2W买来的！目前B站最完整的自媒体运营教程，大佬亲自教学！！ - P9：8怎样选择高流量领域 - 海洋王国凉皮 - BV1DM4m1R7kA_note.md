# 自媒体运营教程：P9：8：如何选择高流量领域 🎯

在本节课中，我们将要学习如何选择一个高流量的自媒体领域。这是获得平台推荐流量、提升收益的关键一步。我们将探讨平台推流逻辑、内容禁忌，并提供实用的领域选择工具。

---

上一节我们介绍了平台推流的重要性，本节中我们来看看具体如何选择高流量的领域。没有流量，意味着没有观众观看你的内容，也就无法获得收入。因此，选对平台正在重点推广的领域至关重要。

其次，你必须规避平台的禁止内容。例如，涉及抽烟、喝酒、纹身或穿着暴露等大尺度内容，都是平台所不允许的。如果你的视频流量不高，可以从以下两方面检查：
1.  你所在的领域是否是平台当前正在推流或举办活动的领域。
2.  你的素材中是否包含上述违规内容。一旦视频因违规内容被限流，其传播效果将大打折扣。

选对大类领域后，还需要注意领域的垂直度。为了让平台更精准地识别你的内容并推荐给目标用户，你需要进一步聚焦。

**平台领域细分表示例（以美食领域为例）**

![](img/66daff73958fd4e5e70d3bc6debb9e17_17.png)

很多人选择美食领域后，内容过于分散，例如同时做探店、试吃、文化、配料盘点等多个细分方向。这种做法并不好。

![](img/66daff73958fd4e5e70d3bc6debb9e17_19.png)

以下是美食领域的细分方向，建议最多选择其中1-2个进行深耕：
*   探店
*   试吃
*   文化
*   配料盘点

如果你在一个大领域下做了四五个细分方向，平台算法将难以判断你的核心内容，从而无法为你精确推荐流量和匹配粉丝，最终影响收益。因此，明确你的细分领域和目标人群至关重要。
