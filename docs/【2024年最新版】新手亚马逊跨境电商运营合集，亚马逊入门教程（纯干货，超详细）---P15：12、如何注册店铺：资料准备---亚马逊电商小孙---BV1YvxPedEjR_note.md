# 亚马逊跨境电商运营合集：P15：12、如何注册店铺：资料准备 📋

![](img/9085e2bb43a859290321f9ee92515de5_0.png)

在本节课中，我们将要学习注册亚马逊店铺前必须准备的各项资料。了解并备齐这些资料是成功开店的第一步。

上一节我们介绍了如何选择亚马逊站点，本节中我们来看看注册店铺需要准备哪些具体的材料。

## 资料清单概述

注册亚马逊店铺需要准备六项核心资料。以下是每一项资料的详细说明与注意事项。

### 1. 营业执照彩色扫描件

第一份需要准备的资料是营业执照的彩色扫描件，即营业执照的照片。这份执照可以是企业性质的，也可以是个体工商户性质的。您可以在当地的工商局免费办理。

*   **办理建议**：如果您没有营业执照，可以考虑借用家人或朋友的执照。例如，开超市或服装店的执照通常可以使用。如果无法借用，也可以寻找代办机构或联系推荐您学习的助理老师协助办理，费用通常在几百到一千元人民币左右。
*   **执照要求**：
    *   **注册资本与年限**：没有硬性要求。无论是1元还是1000万注册资本，无论是新注册的还是已注册多年的执照，均可使用。
    *   **经营范围**：需要带有销售性质即可，例如零售、批发等。**但请注意，经营范围中切勿包含“进出口贸易”**。如果添加此项，您将需要额外办理复杂的进出口手续，属于画蛇添足。
    *   **与销售产品的关系**：执照的经营范围与您计划在亚马逊上销售的产品**不需要一致**。例如，使用超市的执照注册店铺后，销售服装是完全允许的。

### 2. 法人身份证正反面照片

第二份需要指明的资料是营业执照上法人的身份证正反面照片。

**核心要求**：营业执照和这份法人身份证**必须是同一个人的**。这是唯一一组必须关联的资料。

### 3. 付款信用卡

第三份需要准备的资料是用于付款的信用卡。它的唯一作用是支付亚马逊的月租费。

*   **卡片类型**：需要使用全币种信用卡，首选带有 **Visa** 或 **Mastercard** 标识的卡片。
*   **办理与使用**：国内各大银行（如工商银行、浦发银行、建设银行等）均可办理。如果没有信用卡，可以借用他人的，或花费约100元购买一张虚拟卡，每月按时存入39.99美元月租即可。
*   **作用说明**：此卡主要用于店铺初期扣缴月租。当店铺产生营业额后，亚马逊会优先从您的店铺余额中扣款，因此这张信用卡的实际作用有限。

### 4. 海外收款账户

![](img/9085e2bb43a859290321f9ee92515de5_2.png)

![](img/9085e2bb43a859290321f9ee92515de5_3.png)

第四份需要准备的资料是海外收款账户。它的功能类似于国际版的支付宝。

![](img/9085e2bb43a859290321f9ee92515de5_4.png)

*   **功能类比**：当您在淘宝开店赚取收入后，需要通过支付宝提现到储蓄卡。同样，在亚马逊赚取的收入需要通过这个海外收款账户提现到您的国内银行账户。
*   **获取方式**：您可以自行在线注册此类第三方收款服务（如Payoneer、PingPong等），过程类似于注册支付宝，只需绑定您的身份证和银行卡即可，通常只需几分钟。

### 5. 电话号码

第五个需要准备的资料是一个电话号码，使用国内的手机号码即可。

### 6. 电子邮箱

第六个需要准备的资料是一个电子邮箱地址。可以使用163邮箱、126邮箱或QQ邮箱等，其中**163邮箱**是个人比较推荐的选择。

本节课中我们一起学习了注册亚马逊店铺必须准备的六项核心资料：营业执照、法人身份证、付款信用卡、海外收款账户、电话号码和电子邮箱。请务必确保营业执照与法人身份证为同一人，并注意避免在执照经营范围中添加“进出口贸易”。准备好这些材料后，您就可以着手进行店铺注册了。
