# 视频号运营教程：P20：5. 如何制作引导下单的视频 📝

![](img/be633d4d0babd3abe8d3ef7d39318186_1.png)

在本节课中，我们将学习如何在视频号中制作引导用户下单的教程视频。这种视频能有效引导观众完成购买，是提升转化率的关键工具。我们将介绍三种主要的引导方式，并重点讲解如何通过录屏和剪辑制作一个清晰、直观的购买引导视频。

![](img/be633d4d0babd3abe8d3ef7d39318186_2.png)

![](img/be633d4d0babd3abe8d3ef7d39318186_4.png)

## 引导下单的三种方式

![](img/be633d4d0babd3abe8d3ef7d39318186_6.png)

上一节我们介绍了引导下单的重要性，本节中我们来看看具体有哪些方法可以实现。主要有以下三种途径：

![](img/be633d4d0babd3abe8d3ef7d39318186_8.png)

1.  **个人简介引导**：将引导话术直接填写在视频号的个人简介中。用户点开你的主页即可看到。
2.  **视频评论区引导**：在发布视频后，自己在评论区发布引导话术。话术模板可以参考常见格式，发布后若不满意可以删除重新编辑。
3.  **置顶引导视频**：制作一个专门的教程视频，并置顶在你的主页。这是最直观、最有效的方式，能主动教会用户如何购买。

![](img/be633d4d0babd3abe8d3ef7d39318186_10.png)

![](img/be633d4d0babd3abe8d3ef7d39318186_12.png)

## 制作置顶引导视频

![](img/be633d4d0babd3abe8d3ef7d39318186_14.png)

了解了基本方式后，我们重点学习如何制作第三种，即置顶的引导视频。这类视频通常通过录屏展示操作步骤，并添加箭头和文字说明。

![](img/be633d4d0babd3abe8d3ef7d39318186_16.png)

以下是制作引导视频的核心步骤：

![](img/be633d4d0babd3abe8d3ef7d39318186_18.png)

![](img/be633d4d0babd3abe8d3ef7d39318186_20.png)

**第一步：录制操作过程**
*   在手机上，打开你想要引导用户购买的商品视频。
*   将该视频转发给你的任意一个微信好友。
*   从聊天窗口中点开这个视频，开始手机录屏功能。**关键点**：必须从聊天窗口点开录制，这样才能录下观众视角的界面。

**第二步：使用剪映剪辑视频**
*   将录制好的视频导入剪映APP。
*   在需要指引操作的位置（如头像、按钮），通过“贴纸”功能添加箭头。可以搜索“箭头”，选择样式并调整位置和方向。
*   通过“文字”功能添加说明。可以使用文字模板，调整样式、颜色和大小，并将其与箭头对齐，确保指引清晰。
*   如果视频过长，可以裁剪掉不必要的部分，只保留关键操作步骤。

![](img/be633d4d0babd3abe8d3ef7d39318186_22.png)

![](img/be633d4d0babd3abe8d3ef7d39318186_23.png)

**第三步：完善指引内容**
*   按照操作流程，在后续步骤继续添加箭头和文字贴纸进行说明。例如，指引用户“点击商品橱窗”、“搜索关键词”等。
*   确保所有指引元素在时间轴上与视频画面同步，且布局整齐。

![](img/be633d4d0babd3abe8d3ef7d39318186_25.png)

![](img/be633d4d0babd3abe8d3ef7d39318186_27.png)

制作完成后，将这个视频发布到你的视频号，并设置为置顶。用户访问你的主页时，首先就能看到这个购买教程。
