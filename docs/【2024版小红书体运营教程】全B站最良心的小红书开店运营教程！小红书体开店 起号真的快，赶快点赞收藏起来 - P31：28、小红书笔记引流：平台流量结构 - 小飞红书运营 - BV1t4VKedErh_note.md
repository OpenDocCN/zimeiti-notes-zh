# 小红书运营教程：P31：28、小红书笔记引流：平台流量结构 📊

![](img/2f93386bb83378c8a4afdd65df44e9d7_1.png)

在本节课中，我们将要学习小红书平台的核心流量结构。了解流量从哪里来，以及不同流量入口的特点，是制定有效运营策略的基础。掌握这些知识，能帮助我们明确笔记的投放方向，从而更高效地获取曝光和转化。

上一节我们介绍了笔记制作的基础，本节中我们来看看笔记发布后，会进入怎样的流量世界。

## 什么是平台流量结构

流量结构指的是小红书平台整体流量的分布与来源。理解流量结构，是为了让我们明白：不同类型的笔记，投放到平台的不同位置，会获得怎样的效果。

小红书的整体流量结构主要分为三个部分。

## 三大核心流量入口

![](img/2f93386bb83378c8a4afdd65df44e9d7_3.png)

以下是小红书流量的三个主要来源页面，它们共同构成了笔记曝光的基础。

1.  **发现页**
    *   这是小红书**最重要**的流量来源，约占平台总流量的 **70%**。
    *   它是平台进行**流量层级测试的主阵地**。一篇笔记最终能获得多大规模的曝光，主要取决于它在发现页获得的用户反馈（如点赞、收藏、评论）。
    *   新账号发布笔记后，平台根据账号权重给予的初始曝光（例如1000或2000次），几乎**100%** 会分配在发现页。

2.  **搜索页**
    *   搜索页的流量占比约为 **20%**，是仅次于发现页的流量入口。
    *   这是目前小红书流量**增长最快**的板块，也承载了平台约 **70%** 的直接成交转化。
    *   用户通过搜索框主动寻找商品或内容，意图明确，转化率高。但这部分流量需要笔记拥有良好的关键词布局、商品权重和用户评价积累，新账号初期较难获取。

3.  **关注页**
    *   关注页的流量占比最小，大约在 **7%-10%** 左右。
    *   虽然量小，但流量**最精准**。这部分用户是已经关注你的粉丝，兴趣匹配度高，易于引导和转化。
    *   需要注意的是，关注页的浏览数据**不计入笔记的基础权重计算**。即使有大量粉丝阅读，若笔记在发现页的反馈不佳，依然难以获得平台的大规模推荐。

## 关于“本地”页面的特别说明

在APP中，除了上述三个主页面，你还会看到“本地”页面。对此，需要特别了解：

*   **主要用途**：该页面主要基于地理位置推荐内容，适合用于**素材搜集**，观察本地用户喜好。
*   **运营建议**：对于大多数希望将商品卖向全国的商家，**不建议**主要依靠“本地”页面进行引流。地域性过强的标签可能会打乱账号的整体标签，影响平台对你的内容进行全国范围的推荐。
*   **适用场景**：仅适用于特定类型的博主或商家，例如本地美食探店、同城线下服务（如手作饰品送货上门）等。

本节课中我们一起学习了小红书平台的三大流量入口：**发现页**、**搜索页**和**关注页**。记住这个核心公式：**平台总流量 ≈ 70% 发现页 + 20% 搜索页 + 10% 关注页**。对于新账号，运营重心应放在**发现页**，通过优质内容提升初始反馈；同时布局**搜索页**关键词，为未来转化蓄力；并稳步积累**关注页**的精准粉丝。
