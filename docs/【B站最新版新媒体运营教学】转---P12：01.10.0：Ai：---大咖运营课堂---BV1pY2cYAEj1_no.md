# 新媒体运营：P12：01.10.0：ChatGPT 4.0的几种使用渠道 🚀

在本节课中，我们将学习如何获取和使用目前最强大的AI对话模型——ChatGPT 4.0。对于新媒体运营人员而言，掌握AI工具可以极大地提升工作效率，例如撰写脚本、制作PPT和生成报告。本节课将详细介绍几种体验和使用ChatGPT 4.0的渠道，包括免费和付费选项，并分析各自的优缺点。

## 概述：为什么运营需要AI？

在之前的课程中我们提到，当前时代最需要接触AI智能对话模型的两个行业是程序员和新媒体运营。程序员可以利用AI编写代码和测试程序。而对于运营人员，AI可以协助完成短视频脚本创作、PPT制作、表格处理和报告撰写等任务。掌握AI工具，意味着你拥有了一位私人的知识专家，无需具备特定领域的深厚专业知识也能开展工作。

## 渠道一：微软Edge浏览器（Copilot）💻

首先，我们来看看通过微软Edge浏览器体验AI功能的方法。这是目前对普通用户较为友好的一种方式。

微软在其Edge浏览器中集成了名为“Copilot”的功能，它基于ChatGPT技术打造，专为Windows系统用户设计。在功能智能程度上，它与4.0模型有一定匹配度。

使用Copilot有两个前提条件：
1.  登录微软账号（可使用Outlook邮箱注册）。
2.  需要“梯子”（VPN）才能正常访问该服务。

Copilot通常集成在浏览器右侧工具栏。更新至Win11最新版本后，也可能在系统任务栏直接出现图标。其界面提供“更有创造力”、“平衡”和“精确”三种模式，以适应不同的使用场景。

以下是Copilot的一个简单应用示例，我们让它生成一个短视频脚本：
```
用户输入：请帮我写一个关于装修的短视频脚本，时长控制在3分钟左右。
AI回复（示例）：开场画面显示一个简陋的房间，墙壁脏污，地板破损，家具陈旧。旁白：“你是否厌倦了住在这样的房间里？”...
```
此外，Copilot还能基于你当前打开的网页内容进行分析，这在其工具栏图标点开后可以实现。

然而，Copilot也存在一些缺点：
1.  **精准度稍逊**：虽基于4.0模型，但经过微软调教和限制后，其回答的精准度与官方ChatGPT 4.0相比略有差距。
2.  **连续对话能力受限**：在当前对话中持续提问的次数有限，达到一定限制后会提示开启新对话窗口。
3.  **记忆功能有所缺失**：在读取历史对话记录和基于上下文优化后续回答方面，能力不如官方版本完整，且缺乏插件功能。

对于大部分日常运营工作，Copilot的功能已足够使用。如果你的Edge浏览器没有Copilot图标，可以尝试下载最新版的Edge Dev（测试版）或Beta版。你可以在Edge设置中检查更新，或访问微软官网的Edge Insider频道下载测试版。

## 渠道二：第三方集成平台 🌐

上一节我们介绍了系统集成的方案，本节中我们来看看一些由开发者搭建的第三方平台。这些平台通过调用ChatGPT的API，为国内用户提供了访问渠道。

以下是几个可以体验GPT-4的第三方平台：

*   **灵感AI**：该平台集成了包括GPT-3.5、GPT-4、GPT-4联网版等多种模型，以及国内如讯飞星火、文心一言等模型。新注册用户可在7天内免费无限次使用GPT-4（但可能有次数限制，如13次）。其缺点是响应速度可能较慢。付费套餐例如19.9元包月。
*   **WEE Chat**：该平台提供GPT-3.5和GPT-4模型，响应速度较快。新用户通常会获得一定量的免费Token（计算单位），但免费Token通常仅适用于GPT-3.5模型。尝试使用GPT-4可能需要付费。它还具有语音回答功能。费用示例为5美元约100万文字Token。使用此平台需要“梯子”。
*   **ChatGPT账号共享平台**：这是一个共享官方ChatGPT账号的网站。平台上账号以颜色标记状态：绿色（空闲）、橙色（近期使用过）、红色（正在使用）。带有“金光”标识的账号为付费的Plus会员账号，可使用GPT-4。你可以尝试使用这些共享账号登录官方ChatGPT界面。但免费GPT-4账号非常稀缺，且可能有使用次数或时间限制（如官方限制每3小时约30条消息）。使用此平台同样需要“梯子”。

上面推荐的渠道大多需要“梯子”，对于运营人员，接触各种资源是必备能力。本节我们介绍一些在国内可直接访问、无需“梯子”的AI大模型，它们并非ChatGPT，但也能提供辅助。

![](img/f1836a5e4ac3e46c3af2efa326d8e141_19.png)

![](img/f1836a5e4ac3e46c3af2efa326d8e141_21.png)

以下是几个国内的AI大模型平台：

*   **百度文心一言**：百度的AI对话产品，有免费版和付费的“文心一言4.0”版。在基础内容生成、获取灵感方面能满足一般运营需求。你可以尝试其付费版以体验更强大的功能。
*   **字节跳动豆包AI**：字节跳动推出的AI助手。同样可以完成脚本撰写等任务，并能提供景别、画面内容、台词等细节。
*   **其他模型**：例如阿里巴巴的**通义千问**、**天宫大模型**等，也都是国内可供选择的AI工具。

你可以从这些平台中选择一个用起来顺手的，将其培养成适合你工作流程的AI助手，从而提升运营效率。

## 官方ChatGPT 4.0体验对比 🏆

![](img/f1836a5e4ac3e46c3af2efa326d8e141_23.png)

![](img/f1836a5e4ac3e46c3af2efa326d8e141_25.png)

最后，我们来直观感受一下官方ChatGPT 4.0的能力。我们使用同样的指令进行测试：
```
用户输入：请帮我写一份关于装修的短视频脚本，文案要求镜头丰富，语言吸引人。
AI回复（示例）：标题：《变形计：从旧居到梦想家园》，时长：一分钟。场景1（0-10秒）：缓慢推进的镜头展示一间陈旧、有些破旧的居室。旁白：“每一个旧居都藏着一个改造的梦想。”...
```
通过对比可以发现，官方GPT-4在脚本的结构性、场景设计的细腻度和语言的感染力上，通常表现更为出色。国内大模型与GPT-4的差距受多种因素影响。对于运营工作，我们的原则是：哪个工具能更好提升效率，就使用哪个。

## 总结与展望

本节课中，我们一起学习了三种体验和使用ChatGPT 4.0的主要渠道：
1.  **微软Edge Copilot**：系统集成，方便但能力有阉割。
2.  **第三方平台**：如灵感AI、WEE Chat和共享账号，提供了更多选择，但需要注意速度、费用和访问条件。
3.  **国内AI大模型**：如文心一言、豆包等，访问便捷，是重要的备选方案。
