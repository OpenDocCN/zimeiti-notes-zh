## 使用方法一：导入本地大纲 📄

我通常更倾向于使用“导入本地大纲”的方式来生成PPT。这种方法需要提前准备好结构清晰、内容详实的大纲文档。

以下是操作步骤：

1.  **准备大纲内容**：首先，使用ChatGPT等AI工具辅助撰写详细的大纲。例如，我们可以输入指令：“帮我撰写一份‘装修行业在小红书平台的运营推广计划’的PPT大纲，并完善每一个小节点的具体内容信息，保持好结构和格式排版。” 将生成的详细内容保存为文档。
2.  **导入大纲文件**：在工具界面选择“导入本地大纲”功能，上传准备好的大纲文档。工具会快速解析文档结构。
3.  **选择设计模板**：从丰富的模板库中，挑选一个符合行业风格（如装修行业）的模板。
4.  **生成并编辑PPT**：点击“生成PPT”。生成后，若发现部分内容未完全加载，可进入“大纲编辑”界面进行补充。在编辑界面，可以轻松调整文字、形状、背景，并一键更换单页的排版样式。

## 使用方法二：AI智能生成 🧠

除了导入大纲，该工具也支持直接输入主题，由AI自动生成大纲和PPT内容。

以下是操作步骤：

1.  **输入核心主题**：在工具界面选择“AI智能生成”功能，在输入框中直接填写PPT主题，例如：“装修行业在小红书平台的运营推广计划”。
2.  **AI生成大纲与内容**：点击生成，AI会先自动构思并生成详细的大纲结构。
3.  **选择模板并生成**：大纲生成后，同样从模板库中选择合适的风格，然后点击“生成PPT”。AI会基于主题自动填充每一页的详细文案。
4.  **进入编辑优化**：生成完成后，进入编辑页面。在此可以调整字体、样式，更换整体模板，或对AI自动生成的内容进行微调，直至满意。

## 使用技巧与总结 📝

本节课中我们一起学习了第二款AI生成PPT工具的核心功能。要想充分发挥其效能，请注意以下关键点：
*   **内容准备是关键**：无论是自己撰写还是借助AI，提供给工具的**大纲内容越详细、格式越清晰**，生成的PPT质量就越高。建议将节点内容分段排版，以帮助AI更好地理解结构。
