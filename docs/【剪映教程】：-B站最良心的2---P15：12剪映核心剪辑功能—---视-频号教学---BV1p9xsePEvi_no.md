# 剪映教程：P15：视频过渡转场 🎬

在本节课中，我们将学习剪映的核心剪辑功能之一——视频过渡转场。通过学习如何为视频镜头之间添加转场效果，可以让你的视频衔接更自然、视觉效果更炫酷。

![](img/715a03b2f658d13d329738a6697558dc_1.png)

视频过渡转场是指两个视频镜头之间，为了更好衔接而进行的视频处理技巧或剪辑方法。在短视频制作中，巧妙运用转场技巧可以制作出吸引人的卡点视频或高逼格的运镜视频。

![](img/715a03b2f658d13d329738a6697558dc_3.png)

上一节我们介绍了视频剪辑的基础操作，本节中我们来看看如何具体应用转场功能来提升视频效果。

![](img/715a03b2f658d13d329738a6697558dc_5.png)

## 准备工作 🛠️

![](img/715a03b2f658d13d329738a6697558dc_7.png)

![](img/715a03b2f658d13d329738a6697558dc_9.png)

首先，我们需要创建一个新项目并导入素材，以制作一个卡点视频为例进行练习。

![](img/715a03b2f658d13d329738a6697558dc_10.png)

1.  开始创作，导入几张照片作为素材。
2.  为视频添加一个节奏感强的背景音乐。在`音频`中选择`抖音收藏`，使用收藏好的音乐。
3.  试听音乐，确认其具有明显的节奏点（如“动次打次”），这非常适合制作卡点视频。

![](img/715a03b2f658d13d329738a6697558dc_12.png)

![](img/715a03b2f658d13d329738a6697558dc_13.png)

![](img/715a03b2f658d13d329738a6697558dc_15.png)

## 音乐踩点 🎵

![](img/715a03b2f658d13d329738a6697558dc_16.png)

![](img/715a03b2f658d13d329738a6697558dc_17.png)

![](img/715a03b2f658d13d329738a6697558dc_19.png)

卡点视频的关键是让画面切换与音乐节拍同步。接下来我们需要为音乐进行踩点。

![](img/715a03b2f658d13d329738a6697558dc_21.png)

![](img/715a03b2f658d13d329738a6697558dc_23.png)

![](img/715a03b2f658d13d329738a6697558dc_25.png)

1.  点击音频轨道。
2.  在底部工具栏中选择`踩点`。
3.  打开`自动踩点`，选择`节拍2`，它会生成更密集的节拍点，适合制作炫酷的卡点效果。
4.  点击对号确认，音频轨道上会出现一系列小黄点，这些就是节拍标记。

![](img/715a03b2f658d13d329738a6697558dc_27.png)

![](img/715a03b2f658d13d329738a6697558dc_29.png)

## 调整素材时长 ⏱️

![](img/715a03b2f658d13d329738a6697558dc_31.png)

![](img/715a03b2f658d13d329738a6697558dc_33.png)

为了让每个照片的展示时间与音乐节拍对齐，我们需要调整它们的时长。

![](img/715a03b2f658d13d329738a6697558dc_35.png)

![](img/715a03b2f658d13d329738a6697558dc_37.png)

![](img/715a03b2f658d13d329738a6697558dc_38.png)

1.  选中第一张照片，拖动其右边缘，使其结束时间与第一个节拍点对齐。
2.  按照此方法，依次调整后续每张照片的时长，让它们的起点和终点都落在节拍点上。

![](img/715a03b2f658d13d329738a6697558dc_40.png)

![](img/715a03b2f658d13d329738a6697558dc_42.png)

![](img/715a03b2f658d13d329738a6697558dc_43.png)

## 添加转场效果 ✨

![](img/715a03b2f658d13d329738a6697558dc_45.png)

![](img/715a03b2f658d13d329738a6697558dc_46.png)

![](img/715a03b2f658d13d329738a6697558dc_48.png)

![](img/715a03b2f658d13d329738a6697558dc_49.png)

这是本节课的核心操作。在两个镜头之间添加转场，可以消除生硬的切换感。

![](img/715a03b2f658d13d329738a6697558dc_51.png)

![](img/715a03b2f658d13d329738a6697558dc_52.png)

![](img/715a03b2f658d13d329738a6697558dc_54.png)

1.  放大时间轴，以便更清晰地看到两个素材之间的衔接点（显示为一个小方块）。
2.  点击两个素材之间的小方块，进入转场选择界面。
3.  剪映提供了丰富的转场效果库，以下是主要类别：
    *   **基础转场**：如淡入淡出、闪黑。
    *   **运镜转场**：如推进、拉远、旋转，能模拟摄像机运动效果。
    *   **幻灯片**：如百叶窗、风车等2D切换效果。
    *   **特效转场**：如炫光、模糊等。
    *   **遮罩转场**：使用图形进行过渡。
4.  本例中，我们选择`运镜转场`，依次为每个衔接点添加不同的效果（如“拉远”、“推进”等）。

![](img/715a03b2f658d13d329738a6697558dc_56.png)

![](img/715a03b2f658d13d329738a6697558dc_57.png)

![](img/715a03b2f658d13d329738a6697558dc_59.png)

## 优化画面与添加特效 🌈

![](img/715a03b2f658d13d329738a6697558dc_62.png)

![](img/715a03b2f658d13d329738a6697558dc_63.png)

![](img/715a03b2f658d13d329738a6697558dc_64.png)

![](img/715a03b2f658d13d329738a6697558dc_65.png)

添加转场后，我们还可以进一步优化视频的整体观感。

![](img/715a03b2f658d13d329738a6697558dc_66.png)

![](img/715a03b2f658d13d329738a6697558dc_67.png)

![](img/715a03b2f658d13d329738a6697558dc_69.png)

![](img/715a03b2f658d13d329738a6697558dc_70.png)

1.  **调整画幅比例**：如果导入的图片比例不一致，可以选中图片，在`编辑`中调整其大小和位置，或为整个项目设置统一比例（如9:16）。
2.  **添加特效**：为了让视频更绚丽，可以在`特效`菜单中添加一些动态特效（如“变清晰”、“抖动”等），并调整其作用时长和位置。

![](img/715a03b2f658d13d329738a6697558dc_72.png)

![](img/715a03b2f658d13d329738a6697558dc_74.png)

## 效果预览与总结 📝

![](img/715a03b2f658d13d329738a6697558dc_76.png)

完成所有调整后，预览整个视频。对比添加转场前后，视频的节奏感和视觉流畅度得到了显著提升。

![](img/715a03b2f658d13d329738a6697558dc_78.png)

本节课中我们一起学习了剪映的视频过渡转场功能。我们了解到，通过点击视频轨道上素材之间的小方块，可以调用丰富的转场效果库，包括基础、运镜、幻灯片等多种类型。合理运用这些转场，能够极大地提升视频制作的视觉效果，对于制作卡点类视频尤其有帮助。
