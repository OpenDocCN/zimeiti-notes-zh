# 新媒体运营零基础入门：P15：01.10.03：AI：从分析爆款笔记到产出爆款笔记 📝

在本节课中，我们将学习如何利用AI工具（如ChatGPT）分析平台上的爆款内容，并基于分析结果，指导AI生成具备爆款潜质的新笔记。我们将以小红书考研领域的笔记为例，演示从分析到产出的完整流程。

上一节我们介绍了通过“喂资料”的方式，让AI学习特定平台的文案风格和用语习惯。虽然上一节的旅游攻略案例效果不理想，但方法是正确的。本节中，我们来看看如何深化这一过程，通过分析具体的爆款笔记，提炼出可复制的成功要素，并最终生成新内容。

## 分析爆款笔记的核心要点

![](img/51ff14a0cf82bfb5a657a9f4104b1989_3.png)

![](img/51ff14a0cf82bfb5a657a9f4104b1989_5.png)

要让AI学习爆款内容，我们需要引导它从多个维度进行分析。以下是几个关键的分析点：

![](img/51ff14a0cf82bfb5a657a9f4104b1989_7.png)

![](img/51ff14a0cf82bfb5a657a9f4104b1989_8.png)

![](img/51ff14a0cf82bfb5a657a9f4104b1989_10.png)

![](img/51ff14a0cf82bfb5a657a9f4104b1989_12.png)

1.  **爆款因素分析**：分析内容为何能成为爆款，例如是否引发了情感共鸣、提供了实用价值、讲述了逆袭故事等。
2.  **文案结构分析**：拆解文案的框架。
    *   **开篇**：用什么方式（如提问、制造焦虑、引用名人）吸引用户继续阅读。
    *   **中间**：提供了哪些内容来戳中用户痛点，满足用户需求，从而提升用户停留时长。
    *   **结尾**：设置了什么“钩子”（如鼓励互动、提供联系方式）来引导用户私信、关注或评论。
3.  **关键词策略分析**：分析笔记中是否刻意植入了特定关键词，以及这些关键词的出现频率和分布方式。

*   **爆款因素**：

    *   结尾：用正能量话语鼓励读者，并暗示可提供进一步帮助。

## 整合分析结果，提炼爆款公式

在AI分析了多篇笔记后，我们需要让它对分析结果进行整合与去重，提炼出该领域爆款内容的共性。

![](img/51ff14a0cf82bfb5a657a9f4104b1989_53.png)

![](img/51ff14a0cf82bfb5a657a9f4104b1989_55.png)

![](img/51ff14a0cf82bfb5a657a9f4104b1989_57.png)

我们可以给AI如下指令：
```
请将你前面分析过的共4篇考研爆款笔记的爆款因素总结一下，整合成一个清晰的列表。
```

![](img/51ff14a0cf82bfb5a657a9f4104b1989_60.png)

AI可能会总结出如下常见爆款因素：
*   情感共鸣（焦虑、担忧、期待）
*   逆袭/个人成长故事
*   明确、详细的备考计划与时间节点
*   针对各科目的具体方法和资源推荐
*   时间管理与效率提升建议
*   使用Emoji表情增加亲和力
*   结尾的正能量鼓励与互动引导

**注意**：AI有时会“遗忘”或遗漏之前分析过的某些点（如“逆袭故事”），这时需要我们通过对话提醒它回顾所有分析过的内容。

## 指导AI生成新的爆款笔记大纲

有了爆款因素列表，我们就可以指导AI进行创作了。首先，让它根据这些因素生成一个内容大纲。

![](img/51ff14a0cf82bfb5a657a9f4104b1989_62.png)

指令示例：
```
现在，我需要写一篇关于考研的小红书笔记。请结合你刚才总结的爆款因素，为我规划一个具备爆款潜质的笔记大纲。大纲需要细分到具体的小节点，例如开篇具体怎么写，中间分哪几个部分等。在我确认大纲没问题后，再请你生成完整文案。
```

AI生成的大纲可能如下：
1.  **引人入胜的开头**
    *   用问题或共鸣性话语吸引注意。
    *   简要介绍笔记主题（考研备考）。
2.  **个人逆袭故事引入**
    *   简短讲述自己从迷茫到成功的经历，引发共鸣。
3.  **核心干货分享**
    *   专业选择建议。
    *   全年备考时间规划（分月/分阶段）。
    *   政治、英语、数学、专业课分科备考策略。
    *   时间管理与效率提升技巧。
4.  **心态调整与鼓励**
    *   分享克服焦虑的方法。
    *   给予读者正能量鼓励。
5.  **互动引导结尾**
    *   鼓励读者提问、评论。
    *   引导私信或关注。

## 根据大纲生成完整文案

![](img/51ff14a0cf82bfb5a657a9f4104b1989_64.png)

![](img/51ff14a0cf82bfb5a657a9f4104b1989_65.png)

![](img/51ff14a0cf82bfb5a657a9f4104b1989_67.png)

大纲确认后，即可让AI生成完整文案。我们需要给出更细致的指令，将爆款因素具体化。

最终生成指令示例：
```
请根据上面确认的大纲，撰写完整的小红书笔记文案。要求如下：
1. 文案结构严格遵循爆款笔记的结构（开篇吸引、中间干货、结尾引导）。
2. 需要包含一个简单的个人逆袭小故事来引入情绪。
3. 文中需要自然地融入一定数量的Emoji表情。
4. 尝试结合中文分词法的思路，将核心关键词（如“考研全流程”、“考研政治”）自然地融入文案中。
```

AI会根据指令生成一篇初稿。生成的文案在专业度和结构上通常已具备基础，但可能在某些地方显得生硬或“AI感”过重。对于普通运营需求，此初稿经简单调整即可使用。若追求更高品质，可以在此基础上提出更具体的修改要求，或使用更强大的模型（如GPT-4）进行迭代优化。

**进阶提示**：利用ChatGPT的“自定义GPT”（GPTs）功能，可以创建一个专属的“小红书文案助手”。你可以上传平台规范、优质文案范例、常用Emoji库等文件供其学习，从而生成更符合你特定风格和要求的文案。

## 课程总结

本节课中，我们一起学习了利用AI辅助内容创作的核心流程：
1.  **指令引导**：向AI清晰说明需要分析的维度（爆款因素、结构、关键词）。
2.  **样本分析**：提供多个爆款笔记样本，让AI进行学习与分析。
3.  **总结提炼**：让AI整合分析结果，提炼出可复制的爆款公式。
4.  **大纲规划**：基于爆款公式，指导AI生成新内容的大纲。
5.  **文案生成与优化**：根据大纲和详细指令生成初稿，并可根据需要进行优化。

这个方法不仅适用于小红书考研领域，可以迁移到任何你希望深耕的垂直领域。通过持续地“分析-学习-生成”，你将能高效地产出更符合平台调性、更具爆款潜力的内容。在后续课程中，我们将继续探讨AI在抖音等不同平台内容创作中的应用技巧。
