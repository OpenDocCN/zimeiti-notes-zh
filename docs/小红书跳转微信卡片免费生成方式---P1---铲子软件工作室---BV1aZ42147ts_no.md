# 小红书私域运营教程：P1：跳转微信卡片生成与自动回复设置 🚀

在本节课中，我们将学习如何生成小红书跳转至微信的卡片，并设置自动回复功能，以实现私域流量的初步引导。

---

上一节我们概述了课程目标，本节中我们来具体看看如何生成跳转卡片。

这种小红书跳转微信的卡片是通过特定工具后台配置生成的。

![](img/6de54fb683545916790d362ae0eca2ba_1.png)

以下是生成跳转卡片的具体操作步骤：

1.  首先，你需要登录到相关工具的后台。
2.  在后台找到“账号管理”功能，点击“新建账号”。
3.  输入你的小红书账号和密码。
4.  你还需要准备一张用于跳转的微信卡片。
5.  点击“确定”，你的账号就成功绑定到后台了。

![](img/6de54fb683545916790d362ae0eca2ba_3.png)

---

在成功绑定账号后，接下来我们需要配置自动回复规则。

点击“管理设置”或“回复设置”，进入自动回复配置页面。

以下是设置自动回复信息的步骤：

*   **文本回复**：你可以设置纯文字信息自动回复给用户。
*   **笔记回复**：你可以设置自动回复一条指定的笔记链接。
*   **卡片回复**：你可以设置自动回复我们之前准备好的微信跳转卡片。

设置完毕后，点击“确定”保存配置。

---

![](img/6de54fb683545916790d362ae0eca2ba_5.png)

最后一步是启动我们设置好的规则。

回到账号管理列表，找到已绑定的账号，点击“启动”按钮。

点击启动后，你设置的自动回复功能就正式生效了。

![](img/6de54fb683545916790d362ae0eca2ba_7.png)

---

本节课中，我们一起学习了从绑定小红书账号、配置微信跳转卡片到设置并启动自动回复的完整流程。这是将小红书公域流量引导至微信私域的关键第一步。
