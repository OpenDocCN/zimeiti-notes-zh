1.  **角色**
    *   **定义**：指你在特定内容赛道中扮演的专业身份。你需要让用户相信你提供的内容具有专业性。
    *   **公式**：`人设角色 = 赛道方向 + 专家身份`
    *   **示例**：
        *   **好物分享/测评赛道**：扮演亲身试用、客观评价的“测评官”角色。
        *   **装修赛道**：扮演正在装修或已装修完毕，分享全过程经验与避坑指南的“屋主”或“设计师”角色。
        *   **母婴赛道**：可以是分享亲身经历的“宝妈”，也可以是提供专业知识的“育儿顾问”或“护理专家”。
        *   **情感赛道**：可以是提供建议的“心理学顾问”，也可以是分享自身成长故事的“成长型博主”（例如，分享女性在职场、家庭、创业中的经历与感悟）。

2.  **内容**
    *   **定义**：指你持续提供的、与所选赛道强相关的具体信息。这决定了账号的垂直度和平台标签。
    *   **核心要求**：内容需吸引用户、提供价值，并具有一定特色。
    *   **关键原则**：**保持内容垂直**。平台算法会根据你的内容标签进行流量分发。如果内容杂乱（例如，第一篇讲教育，第二篇讲宠物），会混淆用户画像，导致数据反馈差，影响推荐效果。
    *   **代码逻辑**：
        ```python
        if 内容_赛道 == 账号_定位_赛道:
            推荐_精准度 = 高
            账号_标签 = 清晰
        else:
            用户_兴趣 = 可能降低
            平台_推荐流量 = 可能减少
        ```

3.  **形式**
    *   **定义**：指内容呈现的具体方式，包括产出场景、出镜/声音形式、内容体裁等。
    *   **构成要素**：
        *   **产出场景**：室内、室外、车内、特定工作场景等。可参考对标账号，并结合自身条件确定。
        *   **呈现形式**：本人出镜口播、不出镜仅用配音（可用`剪映`等工具实现）、多人对话情景等。需考虑个人表达能力和连贯性。
        *   **内容体裁**：剧情故事型、干货经验分享型、生活记录Vlog型等。不同体裁各有优劣：
            *   **干货型**：需精心准备文案脚本。
            *   **生活记录型**：需积累大量拍摄素材。
            *   **剧情型**：创作难度相对较高。

4.  **记忆点**
    *   **定义**：指那些能让用户快速记住并识别你的固定元素。这是强化人设的关键。
    *   **常见记忆点设计**：
        *   **视频开头前置元素**（用于制造强烈初始印象）：
            *   固定的开场动作。
            *   固定的特效。
            *   固定的开场白。
        *   **贯穿视频全程的元素**（用于持续强化印象）：
            *   统一的背景音乐（BGM）。
            *   统一出现的标志性物品。
            *   统一风格的服装。

本节课中，我们一起学习了IP人设定位的四大核心因素：**角色**、**内容**、**形式**和**记忆点**。
*   **角色**是你专业身份的声明。
*   **内容**需要垂直且持续，以打牢账号标签。
*   **形式**决定了你如何呈现内容，需根据自身情况选择。
*   **记忆点**则是通过设计固定的视觉、听觉元素，让用户牢牢记住你的关键。
