# 小红书运营教程：P28：小红书账号类型与官方广告解析

在本节课中，我们将学习小红书账号的三种主要类型，并对比分析自建账号运营与投放官方广告这两种推广方式的区别与适用场景。通过理解这些核心概念，你可以为自己的品牌或产品选择更合适的运营策略。

## 账号类型解析

上一节我们介绍了账号定位的重要性，本节中我们来看看小红书平台具体有哪些账号类型。了解不同类型的账号及其用途，是制定有效运营策略的基础。

小红书账号主要可以分为以下三类：

**1. 企业号**
企业号的主要作用是树立官方品牌形象。
*   拥有官方认证标志。
*   享有官方特权，例如展示品牌名称、地址等信息。
*   便于开展与品牌相关的官方业务。

**2. 个人号**
目前大部分用户运营的是小红书个人号。
*   运营方式**灵活**。
*   身份可以是品牌员工、产品消费者等。
*   更容易塑造个人品牌特色。
*   适合品牌相关人物或员工出镜进行内容创作。

**3. 引流号**
引流号在之前的课程中也有提及。
*   数量可以是多个。
*   俗称“炮灰号”，可用后即弃，或作为不重要的小号打理。
*   在紧急情况下可作为备用小号使用。
*   通常没有精细的个性与内容规划。

综上所述，常见的品牌推广渠道主要是通过企业号、个人号或引流号自行发布笔记进行引流。

## 企业号矩阵运营建议

在了解了账号类型后，我们来看看如何协同使用这些账号。以下是关于打造企业号矩阵的一些实操建议：

**1. 积极参与品牌号活动**
个人号应积极参与企业号发起的活动。
*   例如，婚纱、服装、美妆类企业号发起的试用或团购活动。
*   可以使用多个小号分别参与，增加活动热度与覆盖面。

![](img/8e75b0095feba62f897a870228de637c_2.png)

**2. 个人号展示品牌属性**
个人号应在内容中巧妙暗示品牌定位。
*   明确品牌面向的群体，例如“城市年轻人”或“职场新人”。
*   在笔记内容、行文风格、视觉呈现上持续传递这一属性。

![](img/8e75b0095feba62f897a870228de637c_5.png)

**3. 合理规划内容发布比例**
个人号发布品牌产品内容需注意频率。
*   建议遵循 **30% 营销内容 + 70% 种草/科普内容** 的比例。
*   这种比例能维持账号的健康度，避免用户因广告过多而产生反感。

![](img/8e75b0095feba62f897a870228de637c_7.png)

![](img/8e75b0095feba62f897a870228de637c_9.png)

**4. 使用引流号进行评论引流**
当企业号需要进行引流时，更推荐使用引流号。
*   在企业号发布的笔记评论区，通过引流号进行互动和引导。
*   这种方式比直接在企业号评论中放置广告链接更为安全和自然。

## 官方广告（CPC竞价广告）介绍

![](img/8e75b0095feba62f897a870228de637c_11.png)

除了运营自建账号矩阵，另一种重要的推广方式是借助平台力量。接下来，我们将介绍小红书的官方广告系统。

小红书官方广告即 **CPC（Cost Per Click）竞价广告**。以下是其核心信息与申请流程：

**申请前提与流程**
投放官方广告有一个必要前提：**必须先认证企业号**。
1.  访问小红书官网，进入“业务合作”板块。
2.  找到“推广合作”入口。
3.  按照指引提交企业资质并支付审核费用。
4.  认证通过后，普通账号即升级为官方企业号，获得广告投放资格。

**广告主要类型**
官方信息流广告主要分为两大类：
1.  **信息流广告**：用户浏览小红书首页信息时，不定时出现的原生广告。
2.  **搜索广告**：当用户使用关键词搜索时，出现的相关广告。其匹配逻辑由平台算法决定，例如用户搜索“考研资料”，系统可能推荐补充能量的商品。

**竞价广告原理**
搜索竞价广告的代表是百度，小红书借鉴了其模式。
*   核心是**针对关键词进行出价竞争**。
*   公式可简化为：**广告排名 = f(出价， 质量度)**。出价不同，获得的广告位排名也不同。
*   常见的广告位有第3位、第13位、第23位等。

![](img/8e75b0095feba62f897a870228de637c_13.png)

**注意事项与建议**
*   官方提供了详细的常见问题解答（FAQ），可在申请页面查看。
*   官方广告投放费用较高。
*   **建议**：在资金允许的情况下，可以考虑与官方合作投放竞价广告，因为它能直接获得由平台官方推荐的精准流量。
