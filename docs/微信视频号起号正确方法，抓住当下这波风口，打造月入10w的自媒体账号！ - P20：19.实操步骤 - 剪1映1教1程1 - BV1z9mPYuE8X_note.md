# 微信视频号运营教程：P20：19.实操步骤 - 开通与上架商品

在本节课中，我们将学习如何在微信视频号中开通商品橱窗功能并上架商品。这是通过视频号进行带货变现的关键一步，操作流程简单明了。

上一节我们介绍了视频号的基础概念，本节中我们来看看如何具体开通带货功能。

## 第一步：打开微信
首先，在您的手机上打开微信应用。

## 第二步：进入“发现”页
打开微信后，点击底部导航栏的“发现”选项。

![](img/4679f800f51bcb1e2b36457aeb89772c_1.png)

## 第三步：找到视频号入口
在“发现”页面中，将视线向上移动，找到并点击“视频号”入口。该入口图标为黄色，形似蝴蝶。

![](img/4679f800f51bcb1e2b36457aeb89772c_2.png)

![](img/4679f800f51bcb1e2b36457aeb89772c_1.png)
![](img/4679f800f51bcb1e2b36457aeb89772c_2.png)

## 第四步：进入个人中心
进入视频号页面后，点击右上角的人形头像图标，进入个人中心。

![](img/4679f800f51bcb1e2b36457aeb89772c_4.png)

![](img/4679f800f51bcb1e2b36457aeb89772c_6.png)

![](img/4679f800f51bcb1e2b36457aeb89772c_4.png)
![](img/4679f800f51bcb1e2b36457aeb89772c_6.png)

## 第五步：进入创作者中心
在个人中心页面，向下滑动屏幕，在最底部找到并点击“创作者中心”。请注意，新注册的账号需要先发布一条作品才会显示此入口。

## 第六步：找到带货中心
进入创作者中心后，在页面中找到“带货中心”的选项，其图标类似一个小卖部。

![](img/4679f800f51bcb1e2b36457aeb89772c_8.png)

![](img/4679f800f51bcb1e2b36457aeb89772c_8.png)
![](img/4679f800f51bcb1e2b36457aeb89772c_9.png)

![](img/4679f800f51bcb1e2b36457aeb89772c_9.png)

点击“带货中心”后，系统会根据您的账号状态进行跳转。如果尚未开通橱窗权限，则会进入开通流程；如果已开通，则会进入商品上架界面。

## 开通橱窗权限操作
如果您是首次开通，点击“带货中心”后会进入身份选择界面。

以下是需要选择的身份类型：
*   **我是商家**：适用于拥有自己产品的账号。
*   **我是达人**：适用于推荐他人产品的账号。
*   **我是商家授权号**：适用于获得商家授权的账号。

**操作建议**：对于绝大多数初学者，建议选择 **`我是达人`**。因为达人身份可以销售平台提供的精选联盟商品，无需自己拥有产品。一个身份证可以注册两个账号，第二个账号可以选择其他身份。

![](img/4679f800f51bcb1e2b36457aeb89772c_11.png)

![](img/4679f800f51bcb1e2b36457aeb89772c_11.png)

选择“我是达人”后，会进入保证金交纳界面。

**核心操作**：您需要点击 **`去交纳`** 按钮，通过微信支付交纳100元人民币的保证金。完成支付后，即成功开通视频号带货权限。

![](img/4679f800f51bcb1e2b36457aeb89772c_13.png)

## 商品上架操作
成功开通橱窗权限后，再次点击“带货中心”，即可进入商品选品和上架界面。

![](img/4679f800f51bcb1e2b36457aeb89772c_13.png)

此时，您可以从视频号平台的商品库（精选联盟）中选择想要推广的商品，将其添加到自己的橱窗中。

![](img/4679f800f51bcb1e2b36457aeb89772c_15.png)

**流程总结**：整个操作的核心逻辑是一个条件判断，用伪代码表示如下：
```python
if 用户未开通橱窗权限:
    显示身份选择与保证金交纳流程
    用户完成开通
else: # 用户已开通权限
    显示商品选品库与上架界面
    用户选择并上架商品
```

![](img/4679f800f51bcb1e2b36457aeb89772c_15.png)

本节课中我们一起学习了开通微信视频号商品橱窗和上架商品的具体步骤。关键点在于：通过“创作者中心”进入“带货中心”，根据引导选择“达人”身份并交纳保证金即可开通权限。开通后，即可在商品库中选品上架，开始您的视频号带货之旅。整个过程无需粉丝门槛，操作简便。
