# 视频号运营教程：P18：4.老号重启

## 概述
在本节课中，我们将学习如何评估和处理一个已有的视频号（即“老号”），判断其是否适合用于带货运营，并针对不同违规情况提供具体的解决方案。

## 账号状态检查
上一节我们介绍了课程主题，本节中我们来看看如何检查老号的状态。首先，您需要确认账号是否处于正常可用状态。

请按照以下步骤操作：
1.  打开微信，进入“发现”页，点击“视频号”。
2.  点击右上角的人形图标，进入个人中心。
3.  点击“创作者服务中心”。
4.  在此页面，您需要检查两个关键信息：
    *   **账号状态**：显示“正常”为佳。
    *   **加热工具**：点击“加热工具”，进入“我的”页面，查看账户状态。

以下是两种正常的账号状态示例：
*   **状态一**：`账号状态`显示“正常”，且`加热工具`内的“我的账户”状态也显示“正常”。
*   **状态二**：`账号状态`显示“正常”，`加热工具`内“视频可加热”，但“商品不可加热”。这是因为粉丝数未达到1000，属于正常情况，不影响账号基础功能。

## 账号异常处理方案
如果您的账号状态异常，则需要根据违规的严重程度采取不同措施。以下是具体的判断与处理方法。

### 情况一：重度违规账号
重度违规通常指账号被平台禁止私信功能180天。此类账号不适合继续用于运营。

**特征判断**：
在“创作者服务中心”的`账号状态`处，会显示为红色警告，例如“账号被禁止私信180天”。

**处理方案**：
根据账号是否开通商品橱窗以及是否有过成交记录，选择以下两种方式之一：

1.  **直接注销账号**（适用于**未开通橱窗**或**已开通橱窗但无历史成交**的账号）：
    *   操作路径：视频号个人中心 -> 点击右上角`...` -> `账号管理` -> `注销账号`。
    *   注销后，账号通常会在7天后变为空号（显示为`*`号）。

2.  **更换视频号管理员**（适用于**已开通橱窗且有历史成交记录**的账号）：
    *   由于有交易记录的账号注销流程复杂、耗时较长（可能超过一个月），更推荐使用“更换管理员”功能。
    *   操作路径：视频号个人中心 -> 点击右上角`...` -> `账号管理` -> `更换管理员`。
    *   按照系统指引，将账号的管理员权限转移给一个新的微信账号即可。此方法比重度违规账号的注销流程更快捷。

### 情况二：轻度违规账号
轻度违规通常指发布的内容被系统识别出存在问题，例如视频带有其他平台的水印或Logo。

**特征判断**：
在视频号的`私信`中，会收到系统通知，指出具体哪条视频违规。

**处理方案**：
1.  根据系统通知，找到对应的违规视频。
2.  将该视频**删除**。
3.  删除后，确保“创作者服务中心”内的`账号状态`和`加热工具`状态均显示为“正常”。

处理完成后，该账号即可视为正常账号，其后续的“养号”步骤与新号相同。

## 总结
本节课我们一起学习了老号重启的完整流程。核心步骤是**先检查账号状态**，区分正常、轻度违规和重度违规三种情况。对于正常和已修复的轻度违规账号，可按新号方式继续运营；对于重度违规账号，则需根据是否开通橱窗和有无成交记录，选择**注销账号**或**更换管理员**。确保账号健康是成功运营的第一步。
