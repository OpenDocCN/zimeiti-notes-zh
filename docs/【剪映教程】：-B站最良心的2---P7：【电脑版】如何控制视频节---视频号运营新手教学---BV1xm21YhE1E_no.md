# 剪映教程：P7：如何控制视频节奏

在本节课中，我们将学习如何使用剪映电脑版中的“变速”功能来控制视频的节奏。通过调整视频的播放速度，我们可以创造出快慢交替的视觉效果，从而更好地匹配音乐节奏或营造特定的情绪氛围。

## 概述：理解视频速度

上一节我们介绍了视频剪辑的基础操作，本节中我们来看看如何调整视频的播放速度。视频的正常播放速度是1.0倍速。通过“变速”功能，我们可以让视频变快或变慢，从而改变其节奏和时长。

## 常规变速

常规变速是最基础的调速方式，它可以整体改变视频的播放速度。

*   **速度公式**：`速度倍率 = 1.0` 为正常速度。
*   **操作逻辑**：
    *   倍率 **低于 1.0**（如 0.5）：视频速度变慢，总时长变长。
    *   倍率 **高于 1.0**（如 2.0）：视频速度变快，总时长变短。

以下是操作步骤：
1.  在“效果空间”面板中找到并点击“变速”功能。
2.  选择“常规变速”。
3.  拖动“倍速”滑块。低于1.0为慢放，高于1.0为快放。
4.  调整后，视频轨道上的片段长度会相应变化。

## 曲线变速

上一节我们介绍了整体调速，本节中我们来看看更高级的“曲线变速”。它允许视频在一段时长内进行动态的速度变化，非常适合配合音乐卡点。

曲线变速提供了多种预设模式，例如“蒙太奇”、“英雄时刻”、“子弹时刻”等。每个预设都对应一条速度变化曲线。

**理解速度曲线**：
在曲线编辑界面中，纵向坐标代表速度倍率。
*   **水平中线（1.0X）**：代表正常播放速度。
*   **曲线上扬**：代表加速，最高可达10倍速。
*   **曲线下沉**：代表减速，最低可至0.1倍速。

因此，波峰最高的点代表速度最快，波谷最低的点代表速度最慢。

![](img/d0e4fa7272f6954200487fee760983bf_2.png)

### 自定义曲线变速

如果你想自由控制速度变化，可以使用“自定义”功能。

以下是自定义曲线的操作方法：
1.  在“曲线变速”中选择“自定义”。
2.  视频预览条上会出现可编辑的节点。
3.  向上拖动节点可加速，向下拖动节点可减速。
4.  你可以在线上任意位置点击以添加新的控制点，从而创建复杂的速度变化曲线。

## 智能补帧

当我们大幅度降速（制作慢动作）时，视频可能会因为帧数不足而出现卡顿。这时就需要使用“智能补帧”功能来生成中间帧，使慢动作画面更流畅。

剪映提供两种补帧算法：
*   **帧融合**：处理速度快，但效果一般。
*   **光流法**：处理速度较慢，但生成的中间帧更准确、画面更流畅。推荐使用此方法。

![](img/d0e4fa7272f6954200487fee760983bf_4.png)

**代码示例（概念说明）**：
```python
# 假设原视频为25帧/秒 (25fps)
# 减速到0.1倍速后，相当于每秒只需要2.5帧来表现原内容
# 智能补帧会通过算法，在原有的2.5帧之间“计算”并插入新的帧，使其补足到25帧/秒播放，从而流畅。
original_fps = 25
slow_motion_speed = 0.1
# 减速后理论所需的帧间隔变大，可能导致卡顿
theoretical_frame_interval = 1 / (original_fps * slow_motion_speed)
# 补帧后，系统会尝试填充帧，使播放保持流畅
interpolated_fps = original_fps  # 通过补帧，输出仍为25fps的流畅视频
```

## 实战案例：结合音乐卡点变速

现在，我们将把学到的知识应用到一个实战案例中，制作一个视频节奏跟随音乐变化的效果。

**操作流程**：
1.  **准备素材**：将视频素材和背景音乐拖入时间轴。
2.  **音乐踩点**：使用剪映的“自动踩点”功能（如“节拍2”），为音乐标记节奏点。
3.  **添加速度关键帧**：在视频上启用“曲线变速”->“自定义”。
4.  **对齐节奏**：在音乐重音或人声突出的节奏点位置，在速度曲线上添加控制点。
5.  **调整曲线**：将节奏点对应的速度控制点向上拖动，制造加速效果；在节奏间隙将控制点拉回正常速度。
6.  **微调与补帧**：反复预览，微调控制点的位置和速度值，使视觉加速与听觉节奏完美同步。如果慢放部分卡顿，记得勾选“智能补帧”并选择“光流法”。
7.  **完成**：播放整个序列，检查效果。
