## 什么是机器学习？

机器学习是人工智能的一个分支。它的目标是让计算机系统能够从数据中学习并改进，而无需进行明确的编程。机器学习模型通过分析数据中的模式来做出预测或决策。

上一节我们介绍了机器学习的定义，本节中我们来看看它通常如何分类。

## 机器学习的类型

机器学习主要可以分为三大类，以下是它们的简要介绍：

*   **监督学习**：模型从带有标签的训练数据中学习。每个训练样本都包含输入数据和对应的正确答案（标签）。模型的目标是学习一个从输入到输出的映射函数。例如，根据房屋面积预测房价。
*   **无监督学习**：模型在没有标签的数据中寻找内在结构或模式。常见的任务包括聚类（将数据分组）和降维（减少数据特征数量）。
*   **强化学习**：模型（智能体）通过与环境互动来学习。它根据行动获得的奖励或惩罚来调整策略，以最大化累积奖励。例如，训练一个玩游戏的AI。

了解了基本分类后，我们将聚焦于最基础的监督学习算法——线性回归。

## 线性回归：一个简单的预测模型

线性回归用于建立自变量（特征）和因变量（目标）之间的线性关系。它假设输出是输入特征的加权和，再加上一个常数项（偏差）。

我们可以用以下**公式**来描述一个简单的线性回归模型：

`y_pred = w * x + b`

其中：
*   `x` 是输入特征。
*   `y_pred` 是模型预测的输出。
*   `w` 是权重（斜率）。
*   `b` 是偏差（截距）。

模型的目标是找到一组最佳的参数 `w` 和 `b`，使得预测值 `y_pred` 尽可能接近真实值 `y_true`。

那么，如何衡量“接近”程度，并找到这组最佳参数呢？这就需要引入损失函数和优化方法。

## 训练模型：损失与优化

为了找到最佳的 `w` 和 `b`，我们需要一个标准来衡量模型预测的好坏。这个标准就是损失函数。

对于线性回归，常用的损失函数是均方误差。其**公式**如下：

`MSE = (1/n) * Σ(y_true - y_pred)²`

这个公式计算了所有样本预测值与真实值之差的平方的平均值。MSE越小，说明模型预测得越准确。

我们的目标就是最小化这个损失函数。梯度下降法是一种常用的优化算法。它通过计算损失函数关于参数（`w` 和 `b`）的梯度（导数），并沿着梯度相反的方向更新参数，从而逐步降低损失值。

参数更新的**公式**可以简化为：

`w = w - learning_rate * ∂(MSE)/∂w`
`b = b - learning_rate * ∂(MSE)/∂b`

其中，`learning_rate`（学习率）控制着每次更新的步长。

让我们通过一个完整的代码示例，将以上所有概念串联起来。

## 动手实践：代码示例

以下是一个使用Python和NumPy实现简单线性回归的示例。我们将用梯度下降法来训练模型。

```python
import numpy as np

# 1. 生成一些模拟数据
np.random.seed(42)
X = 2 * np.random.rand(100, 1)  # 特征
y = 4 + 3 * X + np.random.randn(100, 1)  # 真实值，带有一些噪声

# 2. 初始化模型参数
w = np.random.randn(1)
b = np.random.randn(1)

# 3. 设置超参数
learning_rate = 0.01
epochs = 1000  # 训练轮数

# 4. 梯度下降训练
for epoch in range(epochs):
    # 前向传播：计算预测值
    y_pred = w * X + b

    # 计算损失（均方误差）
    mse = np.mean((y_pred - y) ** 2)

    # 反向传播：计算梯度
    dw = (2 / len(X)) * np.dot(X.T, (y_pred - y))
    db = (2 / len(X)) * np.sum(y_pred - y)

    # 更新参数
    w = w - learning_rate * dw
    b = b - learning_rate * db

    # 每100轮打印一次损失
    if epoch % 100 == 0:
        print(f"Epoch {epoch}, Loss: {mse:.4f}")

print(f"\n训练完成！最终参数： w = {w[0]:.4f}, b = {b[0]:.4f}")
```
