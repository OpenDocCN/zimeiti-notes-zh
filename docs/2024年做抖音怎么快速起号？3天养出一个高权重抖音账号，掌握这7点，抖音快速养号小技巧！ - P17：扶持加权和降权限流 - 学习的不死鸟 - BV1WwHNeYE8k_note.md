# 抖音运营教程：P17：扶持加权与降权限流

在本节课中，我们将学习抖音平台的内容审核规则，了解哪些内容会获得平台的扶持与加权，哪些内容则可能导致账号被降权甚至限流。掌握这些规则是账号健康运营的基础。

上一节我们介绍了账号权重的分类，本节中我们来看看具体影响权重的发布内容。

## 🎬 获得扶持加权的内容

![](img/0bf372bf4834b20fc057d20d44fe7657_1.png)

平台会优先推荐符合以下标准的视频内容：

*   **竖版原生视频**：采用移动端原生的 **9:16** 竖版格式。
*   **画面稳定**：拍摄时画面平稳，无过度晃动。
*   **画质清晰**：作品分辨率高，确保观看体验。
*   **音质干净**：无过多杂音或背景噪音。

![](img/0bf372bf4834b20fc057d20d44fe7657_3.png)

![](img/0bf372bf4834b20fc057d20d44fe7657_1.png)

以上是获得平台流量扶持的基本要求。接下来，我们看看哪些是平台禁止或限制的内容。

## 🚫 导致降权限流的内容

![](img/0bf372bf4834b20fc057d20d44fe7657_5.png)

发布以下内容将面临被降权、限流甚至处罚的风险：

以下是涉及违规信息的内容：
*   吸烟、暴力、炫富（展示大量金钱）等不良信息。

![](img/0bf372bf4834b20fc057d20d44fe7657_3.png)

以下是涉及不当引流的内容：
*   留下微信、微博、淘宝等外部联系方式。
*   视频中长时间出现商品Logo、手机号或二维码（二维码出现时间建议不超过3秒，但审核严格时，即使出现1秒也可能被驳回）。

![](img/0bf372bf4834b20fc057d20d44fe7657_5.png)

以下是涉及硬性广告的行为：
*   明显的营销、广告、推销行为。
*   直接的引流卖货。

**特别需要注意的一点是第15条：无口播的拆箱或好物推荐。** 许多同学在做好物推荐账号时容易触犯此规则。

例如，有同学发布玩具或日用品的推荐视频并挂上小黄车，但视频发布后小黄车立即被下架。原因通常有以下两种：

1.  **直接标价**：在视频画面中直接写上“9.9元”等价格信息，这属于直接的销售行为。
2.  **缺乏口播介绍**：视频仅展示产品拆箱或使用过程，全程无任何语音介绍，只配了背景音乐。

**正确的做法是**：为视频添加口播解说。即使只是简单的介绍，例如：“我儿子玩这个玩具玩了四个小时都没停”或“这款纸巾沾水也不会破”，都能有效避免被系统判定为纯广告内容。**切忌**单纯沉默地展示产品。

至于直接口播广告、念联系方式等更明显的违规行为，相信同学们不会轻易尝试，这在抖音平台是行不通的。

本节课中，我们一起学习了抖音平台的内容规则。核心要点是：发布**竖版、清晰、稳定的原生视频**能获得扶持；而涉及**违规信息、不当引流和硬性广告**的内容会导致降权限流，尤其要注意**好物推荐类视频必须配有口播解说**。理解并遵守这些规则，是账号获得健康流量的第一步。
