# 小红书运营教程：P27：小红书暴力玩法揭秘！吸粉推广篇 🔍

在本节课中，我们将要学习小红书账号矩阵的构建与官方广告投放的区别与策略。我们将理清自建账号推广与官方广告投放的关系，并详细介绍如何有效利用这两种方式来提升品牌曝光和引流效果。

## 账号类型解析

上一节我们介绍了小红书的整体运营逻辑，本节中我们来看看平台上的账号类型。小红书的账号主要可以分为三类。

以下是三类账号的具体说明：

*   **企业号**：主要作用是树立官方品牌形象。企业号拥有官方认证标志，可以展示品牌名称、地址等信息，并享有官方特权，便于开展业务。
*   **个人号**：这是目前大多数用户运营的账号类型。个人号的特点是**灵活性高**，可以是品牌员工或消费者身份。它便于塑造个人品牌特色，也适合品牌相关人物出镜。
*   **引流号**：这类账号在之前的课程中被称为“炮灰号”。引流号可以创建多个，其作用通常是辅助引流，用完即可舍弃，或在紧急情况下作为备用小号使用。这类账号通常没有精细的个性规划。

## 企业号矩阵运营建议

了解了账号分类后，我们来看看如何协同使用它们。品牌推广通常通过企业号、个人号或引流号发布笔记来实现。本节将重点介绍打造企业号矩阵的一些核心建议。

以下是构建有效账号矩阵的四点建议：

1.  **积极参与品牌活动**：品牌的企业号发起活动（如试用、团购）时，应使用多个小号积极参与，以增加活动热度。
2.  **个人号展示品牌属性**：个人号应在笔记内容中，通过场景、话题等方式，巧妙暗示品牌的定位（例如，面向城市青年或职场新人）。
3.  **控制营销内容比例**：个人号发布品牌产品内容的频率不宜过高。建议遵循 **“三七分”原则**，即30%的内容用于营销，70%的内容用于种草或科普，以维持账号健康度。
4.  **利用引流号进行评论引流**：企业号若需直接引流，可搭配使用引流号，通过在其笔记下方进行评论互动的方式，来实现引流目的。

## 官方广告投放介绍

![](img/1640434da72f8b5ebdb49d87d9476d39_5.png)

除了运营自有的账号矩阵，另一种重要的推广方式是借助平台力量。接下来，我们将探讨小红书的官方广告系统。

![](img/1640434da72f8b5ebdb49d87d9476d39_7.png)

![](img/1640434da72f8b5ebdb49d87d9476d39_9.png)

小红书官方广告主要指 **“小红书CBC广告”**，即竞价广告。其申请与使用流程可在小红书官方网站的“业务合作” -> “推广合作”板块找到。但请注意，投放官方广告有一个前提：**必须先认证企业号**。

![](img/1640434da72f8b5ebdb49d87d9476d39_11.png)

企业号的入驻流程同样在官网完成，需要提交相关资质并支付审核认证费用。认证成功后，普通账号即可升级为官方企业号，从而获得广告投放资格。

![](img/1640434da72f8b5ebdb49d87d9476d39_13.png)

官方广告主要分为两大类：

*   **信息流广告**：当用户浏览小红书首页信息时，系统会不定期间隔插入相关的广告内容。
*   **搜索广告**：当用户使用关键词进行搜索时，系统会根据算法匹配并展示相关的广告。例如，用户搜索“帽子”，可能看到搭配推荐的“大衣”广告；搜索“考研资料”，可能看到“营养补充品”广告。

搜索竞价广告的核心原理是关键词竞争，其模式与百度搜索广告类似。广告主针对特定关键词出价，出价不同，获得的广告排名位置也不同。

目前，信息流广告位主要出现在以下三个位置：

1.  第3位
2.  第13位
3.  第23位

![](img/1640434da72f8b5ebdb49d87d9476d39_15.png)

关于广告投放的更多常见问题，可以在小红书广告平台的官方问答页面找到详细解答。需要注意的是，官方广告投放的费用相对较高。因此，在资金允许的情况下，建议考虑与官方合作投放竞价广告，因为它能直接获得由平台算法推荐的精准流量。
