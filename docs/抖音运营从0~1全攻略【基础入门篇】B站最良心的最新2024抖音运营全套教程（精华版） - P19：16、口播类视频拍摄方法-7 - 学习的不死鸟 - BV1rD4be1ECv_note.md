# 抖音运营从0~1全攻略：P19：口播类视频拍摄方法

## 概述
在本节课中，我们将学习口播类视频的具体拍摄方法。我们将通过一个成功案例，详细拆解其构图、内容定位、拍摄技巧等核心要素，帮助你掌握制作高质量口播视频的关键步骤。

---

## 拍摄的核心要点
拍摄环节需要注意以下几点：**构图比例**、**主题突出**、**事物清晰**、**光线色彩**以及**内容连贯**。这些要点看似抽象，接下来我们将通过具体案例进行讲解。

视频拍摄主要分为两大类：**口播类型**和**日常生活类型**。无论是金融、装修、养生还是情感口播，其拍摄方法本质相同。同样，无论是拍摄段子、记录生活还是制作美食，其拍摄方法也基本一致。本节课，我们将重点讲解**口播类型**的拍摄方法。

## 案例解析：一条涨粉30万的口播视频
首先，我们分析一个学员的成功案例。这位名为“众小草”的创作者，是一位40岁带着三个孩子的宝妈。她的一条情感口播视频播放量超过8000万，并借此涨粉30万。

为什么她的视频能取得如此巨大的成功？核心在于**精准吸引了目标群体**。看完她的视频，你就能立刻明白她吸引的是哪类人群。例如，如果我的内容针对20多岁的学生党，那么我拍摄的所有内容都应围绕学生的兴趣点，这样涨粉才会又快又精准。

### 1. 构图与画幅
抖音平台采用 **9:16** 的竖屏比例。关键在于视频内容本身是选择横屏还是竖屏。以“众小草”的视频为例，她采用的是 **4:3** 的横屏构图。

**重要提示**：无论制作哪种类型的视频，请保持画幅统一。如果选择横屏，则全部视频都拍横屏；如果选择竖屏，则全部拍竖屏。保持稳定性是需要注意的细节。

在构图上，人物主体应位于画面中央，占据约70%的空间。避免将脸部怼得过近拍摄，那样会让观众感到不适。确保画面左、右、上方留有适当空白，这样会使画面看起来更清晰、舒适。

### 2. 出镜形象与设备
如果你认为自己外形不够出众，是否还能做口播？答案是肯定的。抖音的成功并非完全依赖颜值。口播视频的核心在于**传递内容和知识**，只要让观众觉得你清晰、体面即可。

![](img/d9a80535a587a8baeae1193b838fbbdc_3.png)

对于自觉外形不自信的创作者，可以开启美颜滤镜。在这个时代，合理使用美颜功能能有效提升形象，只需注意不要过度夸张即可。

此外，口播视频最重要的是**声音清晰**。为此，推荐大家购买一个便宜的**手机收声麦克风**（价格通常在十几元左右），这在淘宝、京东等平台都能买到，能极大提升音频质量，对口播创作者非常方便。

### 3. 内容与目标群体定位
现在，我们来观看“众小草”这条涨粉30万的视频片段，并思考：这个视频想要吸引哪个群体的眼球？

> **视频文案示例**：
> 跟婆婆在一起住是什么样的感受？我跟我婆婆在一起住了16年了，平时磕磕碰碰有点小不愉快，那都是正常的。买了衣服不敢光明正大拿回家，新衣服都是放几天再穿，也都不是事儿。我最大的感受就是这十多年以来，我娘家人很少在我家住上几天，很少。这对于我内心来说，其实是一个很大的缺失。

有同学可能认为是“婆媳关系”。但这是一种关系，具体是哪个群体呢？实际上，对婆媳关系感触最深的，绝非男性群体，而是**已婚女性群体**。十几岁、二十几岁的未婚女性很难对此产生深切共鸣。只有已婚女性，才能真切体会并与此视频内容产生共鸣。因此，这条视频精准吸引的是**已婚女性群体**。

### 4. 拍摄场景与准备
有同学问：口播视频是否必须在户外拍摄？能否在家拍摄？答案是可以。拍摄地点不限，关键在于**背景要整洁美观**，避免过于杂乱难看。

还有同学担心：做口播是否需要背下所有台词？如果记忆力不好，背不下来怎么办？解决办法很简单：**不需要背诵**。在剪辑环节，我们可以借助提词器等工具，实现边看边录，这会让拍摄过程简单得多。

## 总结
本节课我们一起学习了口播类视频的拍摄方法。我们通过“众小草”的案例，重点剖析了：
1.  **构图与画幅**的统一性（如采用稳定的4:3横屏）。
2.  **精准定位目标群体**（如针对已婚女性）是内容成功的关键。
3.  **出镜形象**可通过适度美颜和清晰收声（使用手机麦克风）来优化。
4.  **拍摄场景**灵活，注重背景美观。
5.  **台词准备**可借助工具，无需死记硬背。

掌握这些基础要点，你就能迈出制作优质口播视频的第一步。下一节，我们将继续探讨日常生活类视频的拍摄技巧。
