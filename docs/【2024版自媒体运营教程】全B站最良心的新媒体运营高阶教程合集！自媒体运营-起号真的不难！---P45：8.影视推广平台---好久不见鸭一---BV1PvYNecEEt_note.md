# 自媒体运营教程：P45：8.影视推广平台 🎬

![](img/7d86a29464e4e53b437f4a065538d6af_1.png)

在本节课中，我们将要学习如何通过“影视推广平台”来为你的视频内容获取收益。我们将详细解析影视推广的概念、操作流程以及不同平台的收益结算规则，帮助你理解如何将视频发布到抖音、快手等平台并实现变现。

![](img/7d86a29464e4e53b437f4a065538d6af_3.png)

---

上一节我们介绍了视频的多平台分发策略，本节中我们来看看一个特殊的变现渠道——影视推广平台。很多同学会问，如果想把视频发布到抖音或快手，该如何结算收益呢？这里需要明确一个关键点。

![](img/7d86a29464e4e53b437f4a065538d6af_5.png)

如之前所讲，视频剪辑完成后可以分发到30多个自媒体平台。除抖音和快手外，其他平台通常可以依靠视频本身的播放量来结算收益。

但是，抖音和快手这两个平台有所不同。就像之前提到的，视频发布到这两个平台，主要的作用可能是增长粉丝。这是需要强调的一点。

![](img/7d86a29464e4e53b437f4a065538d6af_7.png)

有没有同学只想专注于抖音和快手这两个熟悉的平台呢？如果你想同时实现涨粉和通过播放量结算收益，答案是肯定的。接下来介绍的方法就能满足这个需求。

刚才讲到的所有视频，可以发布到以下四个平台：抖音、快手、小红书以及微信视频号。这四个平台大家应该都很熟悉，是我们日常会使用的。小红书可能更多被女性或年轻群体如95后使用。微信视频号则内置于我们人手一个的微信中，是一个可以发布视频的小程序。

在这几个平台发布视频后，我们通过“影视推广平台”来接取任务，并通过完成任务来结算奖金。可以看一下，平台提供的素材非常多。

以下是任务类型的举例，包括红字标注的部分：
*   **爱奇艺的任务**
*   **搜狐的任务**
*   各种影视剧的名称

那么，影视推广具体是什么意思呢？影视推广指的是，每一部电视剧、电影或综艺节目都需要进行宣传。现在，许多广告商、明星或网红，由于疫情等原因无法进行线下宣传，便会利用影视推广平台。我们制作视频帮助他们推广，就可以根据视频带来的播放量来结算奖金。

对于每一个视频任务，我们完成后需要提交一个“任务回填链接”进行报名。我们需要在链接中填写自己的微信号、账号昵称、抖音或快手的ID，并清楚说明做的是哪一部剧。此外，还需要粘贴作品发布链接、注明发布时间，并上传视频播放量的截图。回填链接填写完毕后，平台就会为我们结算收益。

这就是影视推广的完整流程。值得一提的是，影视推广的收益通常比单纯依靠平台播放量分成的收益要高一些。

现在来看一下具体的收益规则。短剧推广是当前非常流行的一种形式。许多传媒公司和网红会自制短剧，他们无法独自完成广告宣传，便会委托我们制作视频并发布到指定平台。

以下是短剧推广的可发布平台及收益计算方式：
*   **可发布平台**：抖音、小红书、视频号（这三个平台均可发布）。
*   **收益计算**：根据播放量要求结算收益，**对账号粉丝数没有要求**。你有没有粉丝与任务结算无关。
*   **结算标准**：取一部剧中，**播放量最高的三个作品，其播放量总和**作为结算依据。

那么这个总和具体如何计算收益呢？不同平台的单价不同：

*   **抖音平台**：每1万播放量，奖金为 **50元**。
    *   （对比：之前提到的头条号单价约为每1万播放量34元，抖音的收益明显更高，因为抖音日活跃用户接近10亿，流量庞大。）
*   **微信视频号**：每1万播放量，奖金为 **10元**。
    *   （单价较低的原因是微信用户基数大，每个人都有可能观看视频号。）
*   **小红书平台**：每1000播放量，奖金为 **30元**。
    *   （这是单价最高的平台，收益非常可观。）

除此之外，视频的播放量在结算后如果继续增长，每增加1万播放量通常还会有额外的奖金。

还有一些其他类型的任务，例如搜狐短剧推广。搜狐视频曾购买大量版权，但部分剧集知名度可能不高，评分或播放量有限，所以也会发布推广任务。

对于抖音和快手双平台发布的任务，其结算方式有所不同。以下是按点赞数结算的规则（同样没有粉丝要求）：
*   **抖音平台**：单个作品获得5000赞，奖金为 **100元**；获得1万赞，奖金为 **200元**。
*   **快手平台**：单个作品获得1万赞，奖金为 **180元**。
    *   （在按点赞结算的模式中，快手的单价相对较高。）

本节课中我们一起学习了影视推广平台的运作模式。我们了解了影视推广的定义，掌握了从接取任务、制作发布到提交回填、结算收益的全流程。关键点在于，通过这个平台，即使在抖音、快手这类不以播放量直接分成的平台，我们也能通过完成推广任务来获得可观收益。记住不同平台（抖音、视频号、小红书）的播放量结算单价差异，以及按点赞结算的规则，这能帮助你更好地选择任务和规划发布策略。
