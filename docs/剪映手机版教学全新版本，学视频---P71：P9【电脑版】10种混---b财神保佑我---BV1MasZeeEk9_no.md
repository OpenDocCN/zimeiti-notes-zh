# 剪映手机版教学：P9：【电脑版】10种混合模式 🎬

![](img/296cf3937176f9917328c4c0b090916a_0.png)

在本节课中，我们将学习剪映电脑版中的“混合模式”功能。这是制作高级视觉效果，例如水墨晕开转场的关键工具。我们将通过一个具体案例，系统地了解混合模式的分类、原理和应用方法。

## 概述：什么是混合模式？ 🤔

混合模式决定了当两个图层叠加时，它们的像素颜色如何相互作用并产生新的视觉效果。它不是一个单一的效果，而是一组不同的计算规则。

在剪映中，混合模式位于功能面板中，默认状态为“正常”，表示上下图层互不影响。除此之外，还有10种特殊的混合模式可供选择。

## 混合模式的三大分组 📊

为了便于理解和记忆，我们可以将这10种混合模式分为三组：**去亮组**、**去暗组**和**对比组**。理解分组逻辑比死记每个模式更重要。

### 去亮组：保留暗部，去除亮部 🌑

上一节我们了解了混合模式的基本概念，本节中我们来看看第一组：去亮组。其核心作用是：**去除上方图层中偏亮的部分，保留偏暗的部分**。

**公式/逻辑**：最终画面 = 保留两个图层中更暗的像素。

以下是去亮组包含的混合模式：
*   变暗
*   正片叠底
*   颜色加深
*   线性加深

**应用示例**：当你想让一个白色背景的Logo（亮部为白，暗部为图形）叠加到视频上时，使用“变暗”或“正片叠底”可以瞬间去除白色背景，只留下Logo图形。

### 去暗组：保留亮部，去除暗部 🌕

了解了如何去除亮部后，我们自然要学习其反向操作：去暗组。这组模式的效果与去亮组相反：**去除上方图层中偏暗的部分，保留偏亮的部分**。

**公式/逻辑**：最终画面 = 保留两个图层中更亮的像素。

以下是去暗组包含的混合模式：
*   变亮
*   滤色（绿色）
*   颜色减淡

**应用示例**：这正是我们制作水墨晕开效果的关键。水墨素材通常是黑色背景上有白色水墨。使用“滤色”模式可以去除黑色背景（暗部），只留下白色水墨（亮部）与下方视频融合。

### 对比组：增强明暗对比 ⚖️

最后，我们来看相对复杂一些的对比组。这组模式会综合评估上下图层的亮度信息，其核心效果是：**去除中性灰，同时让暗部更暗、亮部更亮，从而增强整体对比度**。

**公式/逻辑**：最终画面 = 根据混合层亮度，强化下方图层的明暗对比（混合层亮则提亮，混合层暗则压暗）。

以下是对比组包含的混合模式：
*   叠加
*   柔光
*   强光

![](img/296cf3937176f9917328c4c0b090916a_2.png)

**应用示例**：常用于添加纹理、质感或调整色调。例如，叠加一个偏暗的纹理图层并使用“叠加”模式，可以为视频增添复古、做旧的质感；叠加一个偏亮的图层则可能产生柔光或过曝的梦幻效果。

## 实战演练：制作水墨晕开转场 🖌️

学习了理论知识后，现在我们来动手实践，制作课程开头展示的水墨晕开效果。这个案例完美运用了“去暗组”的原理。

1.  **准备素材**：在时间线上放置好你的主视频。
2.  **添加水墨素材**：点击“素材库”，搜索“水墨”或“水墨转场”，选择一个你喜欢的水墨素材（通常是黑底白墨），将其拖到主视频轨道上方。
3.  **应用混合模式**：选中水墨素材，在功能面板中找到“混合模式”，选择“滤色”（绿色）。此时，黑色背景被去除，白色水墨自然地与下方视频融合，形成晕开效果。
4.  **调整与叠加**：你可以复制多个水墨素材，错落放置，并分别应用“滤色”模式，以创建更丰富、层次感更强的转场效果。

## 学习技巧与总结 📝

本节课我们一起学习了剪映中混合模式的分类与应用。最后，为大家总结一个快速选择混合模式的小技巧：

*   **混合层黑色多**（如黑底白字、黑底光效）：优先使用 **去暗组**（滤色、变亮）。
*   **混合层白色多**（如白底Logo、白底图案）：优先使用 **去亮组**（变暗、正片叠底）。
*   **混合层为纯色或纹理**：可以尝试 **对比组**（叠加、柔光）来增添质感。

记住这个口诀，并在实践中多加尝试，你就能熟练掌握混合模式，为你的视频创作增添更多视觉魅力。我们下节课再见！
