# 抖音运营从0到1全攻略：P30：5.视频剪辑找素材 🎬

在本节课中，我们将要学习视频剪辑类内容创作的第一步：如何高效、合规地寻找素材。这是制作出优质视频的基础，也是避免平台违规的关键。

## 视频剪辑的类型

![](img/9ccbdfb37c5c38b49946b7933ad794c9_1.png)

视频剪辑类内容涵盖多种类型。以下是常见的几种：
*   **电影剪辑**：对电影内容进行二次创作。
*   **电视剧剪辑**：对电视剧内容进行剪辑。
*   **综艺剪辑**：对综艺节目内容进行剪辑。

上一节我们介绍了视频剪辑的常见类型，本节中我们来看看如何为这些剪辑寻找素材。

![](img/9ccbdfb37c5c38b49946b7933ad794c9_3.png)

## 寻找素材的三大渠道

以下是寻找视频素材的三个主要渠道。

### 1. 自媒体平台

所有视频平台都是潜在的素材库。这里有一份包含30多个自媒体平台的列表，这些平台都可以作为素材来源。

![](img/9ccbdfb37c5c38b49946b7933ad794c9_1.png)

![](img/9ccbdfb37c5c38b49946b7933ad794c9_3.png)

### 2. 专业影视与素材网站

第二个渠道是专业的影视或素材网站。例如，豆瓣电影这类网站。我自己整理了近50个无版权要求的影视网站，以及60多个涵盖更广的素材网站。

这些网站提供的素材类型非常丰富，包括：
*   **影视类**
*   **动漫类**
*   **综艺类**
*   **搞笑类**
*   **旅游、图片、音效、体育、美食等**

### 3. 影视推广平台

![](img/9ccbdfb37c5c38b49946b7933ad794c9_5.png)

第三个渠道是影视推广平台。在抖音、快手等平台，普通视频可能主要用来涨粉，但发布这些平台提供的影视素材，有时可以依靠播放量直接获得奖金。这个渠道后续会详细讲解。

![](img/9ccbdfb37c5c38b49946b7933ad794c9_7.png)

了解了素材来源后，我们来看看具体如何操作。

## 如何具体搜索素材

![](img/9ccbdfb37c5c38b49946b7933ad794c9_9.png)

我们将以在“西瓜视频”上搜索为例，演示具体步骤。

![](img/9ccbdfb37c5c38b49946b7933ad794c9_11.png)

1.  打开百度，搜索“西瓜视频”并进入其网页。
    ![](img/9ccbdfb37c5c38b49946b7933ad794c9_5.png)
    ![](img/9ccbdfb37c5c38b49946b7933ad794c9_7.png)

2.  确定搜索关键词。例如，如果你想做综艺或娱乐类内容，可以搜索相关人物名称，如“大杨哥”、“小杨哥”。
    ![](img/9ccbdfb37c5c38b49946b7933ad794c9_9.png)

3.  点击搜索，即可找到大量相关视频素材。
    ![](img/9ccbdfb37c5c38b49946b7933ad794c9_11.png)

再举一个例子，搜索自带流量的明星，如“沈腾”，也能找到丰富素材。
![](img/9ccbdfb37c5c38b49946b7933ad794c9_13.png)

## 关键注意事项：避免“搬运”

在找素材过程中，有一个至关重要的原则：**不要直接下载整个视频并发布**。这种做法就是“搬运”，目前平台审核非常严格，95%的搬运内容都无法通过。

正确的做法是：
**从大量视频中，提取出符合自己视频主题的片段，每个片段的时长控制在10秒左右。**

![](img/9ccbdfb37c5c38b49946b7933ad794c9_13.png)

这样处理后的素材，在平台审核时更容易通过，不易违规。

## 从找素材到开始剪辑

找到素材后，下一步就是进行视频剪辑。最常用的软件是**剪映**。根据反馈，我们将以手机版剪映为例进行后续操作。

开始剪辑前，首先要**确定视频主题**。例如，我们决定制作一个主题为 **“沈腾搞笑名场面”** 的视频。那么，所有选取的素材片段都必须包含沈腾本人，并围绕“搞笑”这一核心。
