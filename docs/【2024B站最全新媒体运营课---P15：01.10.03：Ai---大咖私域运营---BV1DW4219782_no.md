# 新媒体运营课程：P15：01.10.03：AI：训练AI模拟爆款笔记到产出爆款笔记的过程 🧠

在本节课中，我们将学习如何利用AI工具（如ChatGPT）分析爆款笔记，并基于分析结果生成具备爆款潜质的新内容。我们将从提供资料、分析爆款因素，到最终生成完整文案，一步步拆解这个过程。

上一节我们介绍了通过“喂资料”的方式让AI理解特定平台的文案风格。虽然上一节的旅游攻略案例效果不理想，但方法是正确的。本节中，我们来看看如何通过分析具体的爆款内容，让AI学习并模仿其成功要素。

## 分析爆款笔记的核心要点

要让AI学习爆款笔记，我们需要引导它分析几个关键方面。以下是需要AI重点分析的内容：

![](img/bfdd18b49a3c5deb29b5071b57d1e951_1.png)

1.  **爆款因素分析**：分析内容为何能成为爆款，例如是否引发了情感共鸣、提供了实用价值或讲述了逆袭故事。
2.  **内容结构分析**：拆解文案的整体结构。
    *   **开篇**：用什么方式（如提问、制造焦虑、引用名人）吸引用户继续阅读。
    *   **中间**：提供了哪些解决用户痛点的内容，如何维持用户的阅读兴趣和停留时长。
    *   **结尾**：设置了什么“钩子”（如鼓励互动、引导私信、提供福利）来促使用户行动。
3.  **关键词分析**：检查笔记中是否刻意植入了特定关键词。
    *   如果有，分析是哪个关键词，以及它在全文中的**出现频率**和**分布方式**。

## 实战演练：以考研领域为例

我们将以小红书平台的考研领域爆款笔记为例，演示整个操作流程。

### 第一步：提供爆款笔记原文

![](img/bfdd18b49a3c5deb29b5071b57d1e951_7.png)

![](img/bfdd18b49a3c5deb29b5071b57d1e951_9.png)

首先，我们需要找到目标领域的爆款笔记，并将其原文提供给AI。例如，我们找到一篇点赞量3000+的考研笔记。

![](img/bfdd18b49a3c5deb29b5071b57d1e951_11.png)

```
（此处插入爆款笔记原文，例如：“阅读一篇错三个政治选题，两眼抹黑...”）
```

### 第二步：向AI下达分析指令

接下来，在AI工具中（本例使用ChatGPT 3.5）输入明确的指令，要求它根据上述要点进行分析。

![](img/bfdd18b49a3c5deb29b5071b57d1e951_13.png)

![](img/bfdd18b49a3c5deb29b5071b57d1e951_15.png)

**指令示例**：
```
我给你提供一篇小红书平台上的考研爆款笔记文案，请你分析：
1. 它的爆款因素有哪些？（例如：情感共鸣、实用建议等）
2. 分析文案的整体结构：开篇如何吸引用户？中间如何戳中用户痛点？结尾如何引导用户互动或关注？
3. 分析整篇文案是否结合中文分词法设计了关键词。如果有，请指出关键词是什么，以及它的出现频率。
```

![](img/bfdd18b49a3c5deb29b5071b57d1e951_17.png)

将找到的爆款笔记原文粘贴在指令下方，发送给AI。

![](img/bfdd18b49a3c5deb29b5071b57d1e951_19.png)

### 第三步：解读AI的分析结果

AI会返回类似以下的分析报告：

*   **爆款因素**：情感共鸣（开篇制造焦虑）、逆袭故事分享、提供具体备考建议、使用表情符号增加亲和力、结尾给予正能量鼓励。
*   **文案结构**：
    *   开篇：用“焦虑”情绪吸引考生群体。
    *   中间：分享个人备考经验和各科目具体方法。
    *   结尾：鼓励读者并暗示可提供进一步帮助。
*   **关键词分析**：本篇笔记没有明显刻意设计的关键词，词汇均为常用词。

![](img/bfdd18b49a3c5deb29b5071b57d1e951_21.png)

![](img/bfdd18b49a3c5deb29b5071b57d1e951_23.png)

为了获得更全面的分析，我们可以重复第一到第三步，多提供几篇不同风格的爆款笔记给AI学习。

### 第四步：整合爆款因素并生成大纲

在AI分析了多篇笔记后，我们需要让它总结出共通的爆款因素。

**指令示例**：
```
请将你前面分析过的几篇考研爆款笔记的爆款因素总结一下，去重后整合成一个完整的列表。
```

AI可能会总结出如下列表：
*   情感共鸣与逆袭故事引入
*   提供明确、阶段性的备考计划
*   给出针对各科目的具体建议和资源推荐
*   强调时间管理和效率提升
*   使用表情符号和亲切语气
*   结尾进行正能量鼓励并引导互动

得到总结后，即可指令AI根据这些爆款因素，生成一篇新笔记的大纲。

**指令示例**：
```
请结合你总结的爆款笔记因素，为我规划一篇关于考研的小红书笔记大纲。大纲需要细分到每个部分的要点。
```

### 第五步：根据大纲生成完整文案

最后，我们让AI根据确认好的大纲，生成完整的文案。

**指令示例**：
```
请按照你提供的大纲，撰写一篇完整的小红书文案。要求：
1. 结构严格遵循爆款笔记因素。
2. 包含一个简短的“逆袭”小故事引入。
3. 使用适量的emoji表情。
4. 尝试在文中自然融入“考研”、“备考”等关键词。
```

AI将生成一篇融合了多种爆款元素的考研备考笔记初稿。这篇初稿已经具备了基本框架和吸引力，运营人员可以在此基础上进行优化和调整，使其更贴合个人风格或具体需求。

## 总结与进阶提示

本节课中我们一起学习了利用AI分析爆款笔记并生成新内容的完整流程：**提供样本 -> 分析要素 -> 总结规律 -> 生成大纲 -> 产出文案**。

![](img/bfdd18b49a3c5deb29b5071b57d1e951_25.png)

**核心提示**：
1.  **指令需明确**：给AI的指令越具体，分析结果和产出内容就越精准。
2.  **样本要多样**：提供多篇爆款笔记进行分析，能让AI学习到更全面的爆款模式。
3.  **结果需优化**：AI生成的初稿是很好的起点，但通常需要人工进行润色、调整和个性化补充，才能达到最佳效果。
4.  **工具进阶**：使用更高级的模型（如GPT-4）或利用自定义GPT（GPTs）功能，上传专属的风格指南和文案样本，可以打造出更懂你需求的私人AI助手，从而生成质量更高、更符合要求的文案。

![](img/bfdd18b49a3c5deb29b5071b57d1e951_27.png)

通过这个方法，你可以系统化地拆解任何领域的爆款内容，并快速批量化生产具备爆款潜质的笔记草稿，极大提升内容创作效率。
