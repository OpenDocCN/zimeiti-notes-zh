# 拼多多运营：P1：单链接SKU拼单价设置教程 🛒

![](img/8fe8308af4c52e7bc8f3b50db14f3eba_1.png)

在本节课中，我们将要学习如何为拼多多商品链接中的单个SKU合理设置拼单价。核心内容包括理解“卡拼单价”的概念，以及针对不同客单价商品的具体定价策略和计算公式。

![](img/8fe8308af4c52e7bc8f3b50db14f3eba_3.png)

## 理解“卡拼单价”概念

![](img/8fe8308af4c52e7bc8f3b50db14f3eba_5.png)

![](img/8fe8308af4c52e7bc8f3b50db14f3eba_7.png)

上一节我们介绍了课程主题，本节中我们来看看核心概念“卡拼单价”。

![](img/8fe8308af4c52e7bc8f3b50db14f3eba_9.png)

“卡拼单价”是一种定价策略，主要适用于低客单价商品。其核心操作是设置一个较低的拼单价，但搭配一个较高的“外露价”。

![](img/8fe8308af4c52e7bc8f3b50db14f3eba_11.png)

*   **外露价**：消费者在商品列表页或详情页第一眼看到的价格。
*   **拼单价**：用户实际拼单时需要支付的价格，也是系统记录链接价格的核心依据。

![](img/8fe8308af4c52e7bc8f3b50db14f3eba_13.png)

通过这种方式，链接在系统内保持低价，有利于获得流量和参与活动，而较高的外露价可以塑造商品价值感，并为后续的营销折扣（如优惠券、限时折扣）预留空间。

![](img/8fe8308af4c52e7bc8f3b50db14f3eba_15.png)

## 低客单价商品的定价策略

![](img/8fe8308af4c52e7bc8f3b50db14f3eba_16.png)

对于低客单价且不计划频繁参与平台大促活动的商品，建议直接采用“卡拼单价”策略。

![](img/8fe8308af4c52e7bc8f3b50db14f3eba_18.png)

操作思路很简单：将你希望系统记录的低价设置为SKU的拼单价，同时设置一个更高的外露展示价。

## 通用定价公式与“虚高”策略

![](img/8fe8308af4c52e7bc8f3b50db14f3eba_20.png)

如果你的商品不属于上述低客单类型，或者你需要一个更通用的定价方法，则可以采用“价格虚高”策略。这需要借助一个公式来计算初始定价。

![](img/8fe8308af4c52e7bc8f3b50db14f3eba_22.png)

以下是计算商品初始“虚高”定价的公式：

![](img/8fe8308af4c52e7bc8f3b50db14f3eba_24.png)

**初始定价 = (目标售价 + 不计入历史最低价的券金额) / 折扣系数**

![](img/8fe8308af4c52e7bc8f3b50db14f3eba_26.png)

让我们来分解这个公式中的各个部分：

![](img/8fe8308af4c52e7bc8f3b50db14f3eba_28.png)

*   **目标售价**：你最终希望卖给消费者的实际价格。
*   **不计入历史最低价的券**：使用某些特定优惠券，其优惠金额不会拉低系统记录的商品历史最低价。这为后续调价保留了灵活性。
*   **折扣系数**：你计划通过“限时限量购”等工具设置的折扣力度，通常为0.7至0.95之间（即7折到95折）。

![](img/8fe8308af4c52e7bc8f3b50db14f3eba_29.png)

你需要选择合适的优惠券。以下是几种常见的不计入历史最低价的优惠券类型：

![](img/8fe8308af4c52e7bc8f3b50db14f3eba_31.png)

*   新客立减券
*   商品关注券
*   直播专享券
*   店铺关注券

![](img/8fe8308af4c52e7bc8f3b50db14f3eba_33.png)

其中，**新客立减券**设置后立即生效，使用最为方便，是常用的选择。

![](img/8fe8308af4c52e7bc8f3b50db14f3eba_35.png)

**举例说明**：假设你希望商品最终实付价为100元，你计划提供一张10元的“新客立减券”，并打算通过“限时限量购”打9折。那么你的初始定价应为：
`(100 + 10) / 0.9 ≈ 122.22元`
你需要将商品价格设置为122.22元，然后配置10元新客立减券和9折的限时限量折扣，最终消费者支付的价格约为100元。

![](img/8fe8308af4c52e7bc8f3b50db14f3eba_37.png)

## 总结

本节课中我们一起学习了拼多多单链接SKU的拼单价设置方法。

我们首先了解了“卡拼单价”策略，它通过区分外露价和拼单价来平衡流量与利润。接着，我们学习了适用于大多数情况的“价格虚高”通用公式，该公式通过结合目标售价、不计价优惠券和折扣系数来科学设定初始价格。最后，我们列举了几种关键的不计入历史最低价的优惠券类型。

![](img/8fe8308af4c52e7bc8f3b50db14f3eba_39.png)

掌握这些方法，可以帮助你更合理地为商品定价，在吸引流量和保证利润之间找到平衡点。
