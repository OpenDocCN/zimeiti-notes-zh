## 内容审核机制 🔍

上一节我们介绍了内容定位，本节中我们来看看内容发布后的第一道关卡——审核机制。所有平台的内容审核主要分为两大块：**机器审核**和**人工审核**。绝大部分内容的前期审核由机器完成。

![](img/376b604454678fe5999c7f32e812c669_11.png)

![](img/376b604454678fe5999c7f32e812c669_13.png)

### 机器审核

![](img/376b604454678fe5999c7f32e812c669_15.png)

内容发布后，首先由机器审核其合规性。合规性主要包含以下两点：

![](img/376b604454678fe5999c7f32e812c669_17.png)

![](img/376b604454678fe5999c7f32e812c669_19.png)

![](img/376b604454678fe5999c7f32e812c669_21.png)

1.  **敏感词检测**
    *   系统会检测内容（图文、口播文案、页面）是否包含平台禁止的敏感词汇。
    *   **结果**：若存在敏感词，内容可能被直接限流、审核不通过或打回。敏感词范围包括违反广告法、法律法规及平台特定规则（如禁止的“黑五类”赛道）的词汇。

![](img/376b604454678fe5999c7f32e812c669_23.png)

![](img/376b604454678fe5999c7f32e812c669_25.png)

2.  **营销广告检测**
    *   系统会判断内容是否为硬性营销广告。平台鼓励基于兴趣分发（短视频）或种草分享（图文）的内容，直接推广产品或服务容易被判定为营销广告。
    *   **影响**：在未进行广告投放或未认证企业号（蓝V）的情况下，发布营销广告内容会对账号初期流量造成损害。

![](img/376b604454678fe5999c7f32e812c669_27.png)

3.  **查重检测**
    *   平台会将新内容与数据库中的历史内容进行特征比对。
    *   **公式**：`内容相似度 = 比对(新内容特征， 历史内容特征)`
    *   **结果**：若发现高度或完全重合，则判定为抄袭，可能导致内容被打回或限流。因此，借鉴他人内容时需进行优化调整，可使用AI工具辅助，但不可直接照搬。

### 人工审核

![](img/376b604454678fe5999c7f32e812c669_29.png)

人工审核主要在两种情况下介入：

![](img/376b604454678fe5999c7f32e812c669_31.png)

![](img/376b604454678fe5999c7f32e812c669_33.png)

1.  **机器审核存疑时**：当机器检测到内容可能存在敏感或违规风险时，会延长审核时间，并将内容提交给人工审核员进行最终判断。通常，被转入人工审核的内容通过率较低。
2.  **日常内容排查**：平台会安排审核人员定期对站内已发布的内容进行抽查。这可能导致之前正常的内容在后期因被排查出问题而被打回或提示违规。此外，用户举报也会触发人工审核。

![](img/376b604454678fe5999c7f32e812c669_35.png)

![](img/376b604454678fe5999c7f32e812c669_37.png)

![](img/376b604454678fe5999c7f32e812c669_39.png)

![](img/376b604454678fe5999c7f32e812c669_41.png)

**过渡**：了解审核机制是确保内容能进入分发池的前提。审核通过后，内容将进入流量分发的核心环节。

## 流量分发机制 🚀

内容通过审核后，系统会如何决定将它分发给哪些用户呢？这个过程主要分为两步：**特征提取**与**数据测试**。

![](img/376b604454678fe5999c7f32e812c669_49.png)

### 第一步：特征提取与初步分类

![](img/376b604454678fe5999c7f32e812c669_51.png)

![](img/376b604454678fe5999c7f32e812c669_53.png)

![](img/376b604454678fe5999c7f32e812c669_55.png)

系统会分析内容特征，为其打上初步标签，以确定测试分发的方向。主要通过以下方式：

*   **关键词分析**：系统通过语义分析、中文分词等技术，从标题、文案、语音识别文本、画面文字等提取核心主题关键词。
    *   **代码示例（概念）**：`关键词列表 = 提取(标题文本, 正文文本, 语音转文本, 图片OCR文本)`
*   **标签参考**：创作者手动添加的话题标签（Tag）为系统提供参考，但并非唯一分类依据。
*   **账号历史数据**：系统会参考该账号以往发布内容的数据表现（如哪些用户互动过），来判断新内容可能的分类。这强调了**账号垂直运营**的重要性。

![](img/376b604454678fe5999c7f32e812c669_57.png)

![](img/376b604454678fe5999c7f32e812c669_59.png)

![](img/376b604454678fe5999c7f32e812c669_61.png)

系统会综合以上信息，给内容一个初步的分类评分（例如0.1分/满分10分），但这个分数在初始阶段通常不高。

![](img/376b604454678fe5999c7f32e812c669_63.png)

![](img/376b604454678fe5999c7f32e812c669_65.png)

### 第二步：数据测试与内容评估

系统会将内容分发给一小部分用户进行测试，并根据他们的行为反馈来评估内容质量，决定是否给予更多流量。核心评估维度如下：

![](img/376b604454678fe5999c7f32e812c669_67.png)

![](img/376b604454678fe5999c7f32e812c669_69.png)

以下是评估内容质量的四个关键用户行为维度：

![](img/376b604454678fe5999c7f32e812c669_71.png)

![](img/376b604454678fe5999c7f32e812c669_73.png)

1.  **点击率**：衡量曝光转化为点击的效率。
    *   **公式**：`点击率 = (点击量 / 曝光量) * 100%`
2.  **停留时长/完播率**：衡量内容吸引力。在短视频平台体现为平均播放时长，在图文平台体现为页面浏览时长、翻页、查看评论等深度交互行为。
3.  **页面浏览深度**：衡量用户对账号的整体兴趣。例如，用户从笔记点击进入主页，并查看了其他内容，这会产生多个页面浏览，对账号整体有增益。
4.  **互动行为**：包括点赞、收藏、评论、分享、关注等。用户完成其中任意几项，都能为内容带来数据增益。

![](img/376b604454678fe5999c7f32e812c669_75.png)

![](img/376b604454678fe5999c7f32e812c669_77.png)

![](img/376b604454678fe5999c7f32e812c669_79.png)

![](img/376b604454678fe5999c7f32e812c669_81.png)

**过渡**：理解了平台如何评估内容，我们还需要知道流量主要从哪些入口来。这将帮助我们有针对性地优化内容。

![](img/376b604454678fe5999c7f32e812c669_99.png)

以下是平台内几个核心的流量来源渠道：

![](img/376b604454678fe5999c7f32e812c669_103.png)

1.  **发现页**：平台首页，根据用户标签和实时热度进行个性化内容推荐，是最大的公域流量池。
2.  **关注页**：用户查看已关注博主更新的内容，用户目的性强，流量精准。
3.  **同城页**：基于地理位置推荐内容，适合本地生活、探店等需求。
4.  **搜索入口**：用户主动表达明确需求（如“XX攻略”、“附近美食”）的渠道，关键词优化至关重要。
5.  **购物/电商入口**：用户带有明确购物意图，是直接进行商品推广和转化的关键场景。

## 总结 📝

本节课我们一起学习了新媒体平台的流量分发与审核机制。我们了解到：
1.  内容发布后需通过**机器审核**（敏感词、营销广告、查重）和可能触发的**人工审核**。
2.  审核通过的内容会进入分发系统，经历**特征提取**和**数据测试**两个阶段。
3.  系统通过**点击率、停留时长、浏览深度、互动行为**等数据来评估内容质量，决定后续流量分配。
4.  流量主要来源于**发现页、关注页、同城页、搜索入口和电商入口**等不同场景。

![](img/376b604454678fe5999c7f32e812c669_109.png)

![](img/376b604454678fe5999c7f32e812c669_111.png)

![](img/376b604454678fe5999c7f32e812c669_113.png)

掌握这套机制，能帮助你在创作和发布内容时更有针对性，从而有效提升内容曝光和互动效果。
