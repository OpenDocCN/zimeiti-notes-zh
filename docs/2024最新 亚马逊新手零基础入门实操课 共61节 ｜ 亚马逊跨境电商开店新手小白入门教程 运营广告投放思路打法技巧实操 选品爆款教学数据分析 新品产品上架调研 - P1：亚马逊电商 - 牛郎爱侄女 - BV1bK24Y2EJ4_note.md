# 亚马逊电商入门：P1：零成本启动与选品基础

## 概述
在本节课中，我们将要学习如何以零成本或低成本的方式启动亚马逊店铺，并掌握基础的选品思路。这是一种适合资金有限的初学者的入门玩法。

## 账号注册与站点选择
上一节概述了本课程的目标，本节中我们来看看如何迈出第一步：注册账号并选择销售站点。

首先，你需要注册一个亚马逊卖家账号。具体的注册条件可以参考下图（此处为描述，实际教程中应插入图片）。操作流程是：通过百度搜索“亚马逊全球开店”，进入官网后，找到右上角的“前往站点注册”并选择“北美站”进行注册。

选择北美站的原因是它包含了美国、加拿大和墨西哥三个市场。开通北美站后，你可以同时在三个国家销售，市场容量较大。对于新手而言，从北美站开始操作是一个比较稳妥的建议。

## 核心选品思路与操作
完成了账号注册，接下来就是最关键的一步：寻找可以销售的产品。对于新手，产品来源可能是个难题。

以下是推荐的选品参考网站：
*   **1688**
*   **中国制造网**
*   **义乌购**

这些网站上通常有“跨境专供”的资源。选品的核心原则是寻找**国内外差价较大**的产品。例如，一款在国内很火的“大肚杯”，在国内的拿货成本大约是10元人民币。

我们可以通过亚马逊将其销售到美国，售价可以定为**19.9美元**（约合人民币100多元）。扣除所有成本后，一单的利润可以达到80多元人民币。

这个模式的优势在于，你**不需要囤货**，也**不需要大量押资金**。你只需要在上述货源网站上寻找支持“一件代发”的供应商。

## 上架与订单处理流程
找到了有潜力的产品后，下一步就是将其上架到你的亚马逊店铺。

具体的操作流程很简单：
1.  从货源网站复制产品的图片、标题和详情页信息。
2.  将这些信息进行适当编辑和加价后，上传到你注册好的亚马逊店铺后台。

当有海外顾客下单后，你只需在货源网站进行“一件代发”操作，由供应商直接发货给顾客。通过这种方式，单个产品每月稳定获得3000多美元（约合人民币2万多元）的收入是可行的。

## 总结
本节课中我们一起学习了亚马逊零基础启动的核心路径。我们首先完成了账号注册与站点选择，然后掌握了通过1688等平台进行“一件代发”选品的基本思路，最后了解了复制产品信息上架和处理订单的简单流程。这个方法的门槛在于执行力，只要坚持操作，后期可以获得比较稳定的收益。
