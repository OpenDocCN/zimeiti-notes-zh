## 什么是自媒体？

那么，首先让我们来了解一下到底什么是自媒体。

用通俗的话来讲，自媒体就是：今天我们打开手机，在百度上搜索一个自媒体平台，然后去该平台注册一个属于自己的账号。在我们自己的这个账号上，发布我们自己创作的文章或视频。当有人观看时，我们就可以依靠播放量来结算收益。这就叫做**自媒体**。

其核心流程可以用一个简单的公式表示：

![](img/5331adb678e7c1fc3c93aa58ee50cd9b_1.png)

**自媒体 = 注册平台账号 + 发布原创内容（文章/视频） + 获取播放量/阅读量 → 获得收益**

![](img/5331adb678e7c1fc3c93aa58ee50cd9b_2.png)
