## 手机选购指南

首先需要明确，我们将使用苹果手机来模拟海外用户环境，以正常下载和运营TikTok。因此，手机的选购是基础且关键的一步。

![](img/8f4951c0daf7cbb26d1d036851a4723b_3.png)

以下是选购苹果手机时需要关注的核心要点：

1.  **型号选择**
    *   推荐基础型号为 **iPhone 6S**。此型号性能足以运行TikTok，且二手价格经济实惠。
    *   更高型号如iPhone 7、8、X、11、12、13等也完全适用。
    *   核心原则是：无需购买最新最贵的型号，一部二手iPhone 6S即可满足起步需求。

2.  **内存要求**
    *   手机内存**最低要求为32GB**。如果内存低于此容量，在后期运行TikTok时可能会出现卡顿。
    *   选择64GB或128GB内存的手机会有更流畅的体验。

3.  **购买渠道**
    *   常见渠道包括：淘宝、京东、闲鱼等电商平台。
    *   在购买前，必须向商家确认两个关键问题：
        *   手机能否**恢复出厂设置（抹机）**？不能抹机的手机无法使用。
        *   手机内是否已**退出原用户的Apple ID**？如果手机绑定了他人的ID且不知道密码，将无法完成抹机操作。
    *   如果找不到可信的购买渠道，可以向课程老师寻求推荐。

## 核心准备工作预览

完成手机选购后，后续还需要进行一系列设置，主要包括：
*   对国行手机进行**抹机**，以模拟海外环境。
*   注册用于TikTok的**邮箱**。
*   注册**海外苹果ID**，用于下载海外版App。
*   搭建稳定的**网络环境**。
*   进行手机**伪装环境检测**，确保设置无误。
*   最终完成**TikTok账号注册**并熟悉其界面。
