## 第四步：访问个人中心

进入视频号后，需要访问个人管理界面。

**操作：** 点击屏幕右上角的个人头像图标。

![](img/444cd45e940258f04cdb63ed6e772f06_16.png)

---

## 第五步：进入创作中心

在个人中心页面，可以找到管理创作功能的核心区域。

**操作：** 将页面滑动至最底部，找到并点击“创作中心”。如果未显示，发布一条视频后即可出现。

## 第六步：进入带货中心

![](img/444cd45e940258f04cdb63ed6e772f06_20.png)

创作中心是管理所有创作功能的地方，接下来我们进入商品带货相关设置。

![](img/444cd45e940258f04cdb63ed6e772f06_22.png)

**操作：** 在“创作中心”页面内，找到并点击“带货中心”。

![](img/444cd45e940258f04cdb63ed6e772f06_24.png)

> **核心概念提示：** 如果点击“带货中心”后提示需要开通权限，系统会引导你完成**达人认证**等步骤，通常需要提交资料并可能涉及费用。

---

## 第七步：使用选品中心添加商品

成功开通带货权限后，就可以开始为你的橱窗添加商品了。

**操作：** 在“带货中心”页面，找到并点击“选品中心”或“去选品”。

以下是选品和上架的具体步骤：

![](img/444cd45e940258f04cdb63ed6e772f06_26.png)

1.  **浏览商品**：在选品中心页面，系统会展示大量可供带货的商品。
2.  **选择商品**：浏览并找到你想推荐的商品，例如一个“菜板”。
3.  **添加至橱窗**：点击该商品下方的“加橱窗”按钮。

**核心操作代码描述：**
```python
# 模拟商品上架操作
if 用户.已开通带货权限:
    商品列表 = 选品中心.获取推荐商品()
    for 商品 in 商品列表:
        if 用户.点击(商品.加橱窗按钮):
            商品.加入用户橱窗()
            print("商品已成功添加至橱窗！")
```

完成添加后，需要确认商品是否已成功上架。

**操作：** 添加商品后，返回你的视频号个人主页，检查“商品橱窗”中是否已显示刚添加的商品。
