# 抖音自媒体运营从0~1全攻略：P13：11、视频类型1-2

在本节课中，我们将要学习短视频创作中一个至关重要的概念：**人设**。我们将深入探讨强人设与弱人设的区别，并介绍一种核心的视频制作形式——口播。理解这些概念，是规划你账号内容方向、吸引精准粉丝的第一步。

## 人设：连接涨粉与变现的关键环节

上一节我们解决了文案问题，本节中我们来看看下一个关键环节：人设。很多课程会直接教你涨粉或变现，但从涨粉到变现并非一条直线，中间还有一个关键环节，那就是**建立你的人设**。

建立人设的目的不是为了炫耀，而是为了在众多同类创作者中脱颖而出。例如，有十个人都在做装修内容，如果其中一位创作者（比如小王同学）树立了一个鲜明有趣的人设，观众就更容易记住他，而忽略其他九位。当观众因为喜欢你这个人而关注你时，他们就成为了你的粉丝。这就是人设的价值。

接下来，我们来学习这个核心知识点。人设主要分为两种：**强人设**和**弱人设**。区分清楚这两种人设后，你就能快速判断任何视频的类型。

### 强人设：以“你”为核心

强人设是指内容以创作者**个人**为切入点，与创作者自身密切相关。这类内容通常围绕你的专业、工作或兴趣爱好展开。

以下是强人设常见的切入点领域：

*   **专业/行业经验**：例如装修、办公、房产、汽车、餐饮、婚介、法律咨询、电商等。
*   **个人兴趣爱好**：例如美食、旅行、手工、宠物、星座占卜、美妆、护肤、服装穿搭、游戏、摄影等。

### 弱人设：以“内容”为核心

与强人设相对的是弱人设。弱人设的内容与创作者本人关系不大，主要目的是为了把账号内容做好。

以下是弱人设的典型例子：

*   **情感语录类**：发布情感文案配音乐，通常无真人出镜或配音。
*   **影视解说类**：虽然有人声解说电影，但重点在电影内容本身，与解说者个人特质关联较弱。

弱人设的关键在于内容本身的价值，而非塑造创作者的个人形象。

理解了强弱人设的区别，我们才能更好地规划自己的内容方向。接下来，我们将聚焦于一种非常适合强人设、且操作简单的视频形式。

## 口播：用讲述创造价值的视频形式

在内容创作环节，任何与你个人相关的主题都可以制作成视频。我们先从第一种广泛适用的形式讲起：**讲故事**。

讲故事可以分享客户故事、从业经历或人生感悟。故事本身不要求完全真实，可以从文案平台寻找有趣的故事进行分享。而讲故事最直接的方式，就是通过**口播**。

口播，即面对镜头进行讲述的视频形式。它有很多分类，适用于不同领域。

以下是常见的口播类型：

*   **故事型口播**：如情感口播、讲述奇闻轶事等，目的是让观众觉得有趣、有共鸣。
*   **知识型口播**：如育儿、金融、养生、装修、房产、汽车等领域的知识分享，目的是让观众在几十秒内获得“收获感”，觉得学到了东西。

口播形式非常适合法律、保险、金融、装修等需要专业知识输出的领域。它的优势在于**拍摄简单**：只需用手机架好，准备好文案即可录制。

针对常见的顾虑，这里有两个关键解答：

1.  **口才不好能做口播吗？**
    *   可以。口播使用的是**提前准备好的稿子**，并非临场发挥。你看到的口播达人，大多也是依据脚本进行录制。
    *   **公式**：`口播视频 = 提前准备的文案 + 面对镜头的稳定讲述`

2.  **普通话不好能做口播吗？**
    *   可以。如果你的目标客户是同城或有地域属性，使用**方言**可能效果更好，能增加亲切感和趣味性。
    *   同时，你可以为视频配上字幕，确保所有观众都能理解内容。

口播是一种非常实用且易于起步的视频形式，在后续课程中我们会详细讲解具体的拍摄技巧。
