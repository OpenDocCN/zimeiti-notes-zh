# 小红书运营教程：P27：11：小红书平台用户标签及流量入口 📊

![](img/a1370d34dcaf8726cc31f4c9deab1968_1.png)

在本节课中，我们将深入探讨小红书平台如何判定用户标签，以及平台的主要流量入口有哪些。理解这些底层逻辑，有助于我们分析自己的账号定位，并让笔记更精准地触达目标用户。

上一节我们介绍了系统如何判定内容标签，本节中我们来看看平台如何判定用户标签。这个过程能帮助我们反向推导，理解笔记是如何被系统推荐给用户的，从而找到让笔记被精准用户看到的方法。

## 用户标签判定逻辑 🔍

系统为用户打标签的主要依据是**用户浏览行为**。用户注册时选择的兴趣范围参考价值不大，仅作为初期内容方向的引导，无法进入系统深层的用户标签池。系统判定用户兴趣分类的核心依据是动态的浏览行为。

以下是系统分析用户行为并打标签的几个关键维度：

*   **点击率**：系统在发现页向用户推送多种内容（如美妆、家居、母婴等）。用户点击哪类内容更多，系统就能计算出其偏好。例如，用户点击10条内容，其中美妆类3次，家居类2次，母婴类0次，系统会判定该用户对母婴内容不感兴趣，并持续推送更多美妆内容进行试探。通过点击率，系统能为用户打上**强关联**和**弱关联**的标签。

*   **停留时长**：用户对某篇笔记的浏览时间长短是重要的分析依据。例如，用户常点美妆内容但很快划走，说明当前推荐不精准。若某天系统推荐了一条家居内容，用户不仅点击，还长时间浏览图片、查看评论，系统则会判断该用户可能有装修或购买家居用品的需求，后续会相应调整推荐内容。

*   **页面浏览量（PV）**：这指用户浏览的页面深度。例如，用户看完一篇笔记后，又进入博主主页浏览了其他多篇笔记。提升用户在你主页的浏览深度，对账号健康度有显著的正面影响。

*   **互动行为**：包括点赞、收藏、评论、关注。这些数据直接关系到笔记能否进入更大的流量推荐池，也是账号接单、变现的基础。互动数据好的笔记，不仅能获得更多推荐，还可能带动账号内其他笔记的流量。

## 平台流量分发入口 🚪

了解用户标签后，我们来看小红书主要的流量分发入口。平台目前主要有四个流量入口：发现页、关注页、附近页和购物页。

以下是各流量入口的详细说明：

*   **发现页**：这是笔记获取流量的主要来源。其内容包含：
    1.  已关注达人发布的内容。
    2.  根据用户标签（点击率、停留时长等）匹配的个性化推荐内容。
    3.  部分平台为了内容多样化而推荐的、用户可能感兴趣的新内容。

*   **关注页**：这里集中展示所有已关注博主发布的最新笔记。用户可以直接在此查看强关联博主的内容。

*   **附近页**：基于手机定位，推送同城或附近区域的内容。此入口强地域特征，适合美食、本地旅游、线下店推广（如美容院、餐厅）等内容。做线下实体店推广的博主，需重点关注此入口。

*   **购物页**：这是小红书向电商发展的重要入口，直接匹配用户的购物需求。用户可在此搜索商品、进入商城或观看带货直播。但当前用户更习惯通过搜索笔记来“种草”，直接使用购物页的习惯仍在培养中。
