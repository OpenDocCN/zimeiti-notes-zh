# 《自媒体全套运营教程》：P7：4 怎样选择平台 🧭

![](img/bd813da4b5f082a9e4ba20c2db543ac7_1.png)

![](img/bd813da4b5f082a9e4ba20c2db543ac7_3.png)

在本节课中，我们将要学习如何从众多自媒体平台中进行选择，特别是针对新手，我们将探讨从小平台起步的优势、具体操作方法以及一个高效的入门技巧。

![](img/bd813da4b5f082a9e4ba20c2db543ac7_5.png)

上一节我们介绍了账号定位与内容形式，本节中我们来看看如何选择适合的平台。

现在我们知道自媒体的平台非常多。那么我们该怎么样去选择去做呢。

![](img/bd813da4b5f082a9e4ba20c2db543ac7_7.png)

![](img/bd813da4b5f082a9e4ba20c2db543ac7_9.png)

首先，我给出一个结论：建议所有新手先做小平台，再做大平台。一开始做大平台不是不行，只是起步会艰难一点点。

![](img/bd813da4b5f082a9e4ba20c2db543ac7_11.png)

![](img/bd813da4b5f082a9e4ba20c2db543ac7_13.png)

因此我建议大家先做小平台再做大平台。为什么呢？前期建议大家先选择小平台练习技巧，收获思路。就像今天你刚刚下载了王者荣耀准备去打，但刚开始时，你不知道自己打上单、中单还是法师更合适。所以你可以先选择一个小平台去练习。等你后期知道自己擅长什么了，就可以在后期选择大平台去做一文多发，以提高收益。

![](img/bd813da4b5f082a9e4ba20c2db543ac7_15.png)

那么在前面新手阶段，小平台有哪些呢？我刚刚也跟大家说了，新手攒经验攒技能，先选择小平台去做。

以下是可供选择的小平台列表：
*   趣头条
*   汇头条
*   快船号
*   搜狐号

![](img/bd813da4b5f082a9e4ba20c2db543ac7_17.png)

这些都属于小平台。那么小平台很明显它的缺点是什么？**单价不高，基本上是5~15块钱**，而且它上面的日活跃用户相对比较少。

![](img/bd813da4b5f082a9e4ba20c2db543ac7_19.png)

![](img/bd813da4b5f082a9e4ba20c2db543ac7_21.png)

那么小平台的优点是什么呢？你可以在上面做伪原创、做搬运，这样没有任何新手期。你完全可以很自由地去探索自己到底适合做哪个领域。基于做搬运、做伪原创没有问题的话，你在上面没有任何新手期，只要坚持发视频、发内容，基本上月入2000到4000。我认为这是一个比较简单的事情，不难做到。

![](img/bd813da4b5f082a9e4ba20c2db543ac7_23.png)

![](img/bd813da4b5f082a9e4ba20c2db543ac7_25.png)

那么零基础零粉丝的朋友们，应该怎么样去上手操作呢？其实我认为是很简单的。在这边我先教大家去做一个伪原创，做一个搬运，直接教你5分钟搞定视频创作。

![](img/bd813da4b5f082a9e4ba20c2db543ac7_27.png)

![](img/bd813da4b5f082a9e4ba20c2db543ac7_29.png)

因为如果说你现在不会做视频采集，也不知道该怎么样去做素材采集，又或者你觉得在国内搬运的重复度太高，那么我认为大家不妨去看一下海外的巨量优质资源，把目光放长远一点。

在这里我找了一个B站的视频给大家看一下。

> （视频内容描述：一个海外荒岛求生综艺节目，被搬运到B站，视频本身没有复杂剪辑，只是添加了中文字幕。）

大家可以看到这个是在B站上面的一个荒岛求生视频。很明显它是属于一个海外综艺，搬运到了国内。你可以清楚地看到这个视频没有经过很多剪辑，它就是原视频画面、原视频原声。它搬运到国内后，唯一做的改变就是添加上了中文字幕。但是大家可以看到，它在B站上有232.9万的播放量。

我们来算一下232.9万是多少钱。这一条视频，232.9万乘以B站的单价（大约15元/万播放），可以计算得出：
```
收益 ≈ 232.9 * 15 ≈ 3493.5 元
```
这条视频搬运到国内哔哩哔哩网站后，能够获取到的收益大约是3490元。你可以发现，他不需要做复杂的视频剪辑，也没有复杂的采集素材，只是从海外的原声原画搬运而来。

![](img/bd813da4b5f082a9e4ba20c2db543ac7_31.png)

本节课中我们一起学习了如何为自媒体运营选择平台。核心策略是**新手先从小平台起步**，其优点是**门槛低、无新手期、适合练习技巧和探索方向**。我们还了解了一个高效的入门方法：**从海外平台寻找优质资源进行搬运和伪原创（如添加字幕）**，这可以快速产生收益并积累经验。掌握这个思路，能帮助你在自媒体道路上更顺利地迈出第一步。
