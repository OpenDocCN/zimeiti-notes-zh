# 数据科学入门：P1：数据质量初步评估

在本节课中，我们将学习如何对一个新获得的数据集进行初步的质量评估。这是数据科学工作流程中至关重要的一步，它帮助我们理解数据的可靠性与可用性，为后续的分析与建模打下坚实基础。

## 数据质量的重要性

上一节我们介绍了课程目标，本节中我们来看看为什么数据质量如此关键。高质量的数据是得出准确、可靠结论的前提。如果数据本身存在问题，无论后续使用多么复杂的模型，其结果都可能产生误导。

## 核心评估维度

评估数据质量可以从多个维度进行。以下是几个最核心的评估方面：

1.  **准确性**：数据是否真实、正确地反映了现实世界的情况。例如，年龄字段中不应出现负数。
2.  **完整性**：数据是否存在缺失值。缺失值过多会影响分析。在Python的pandas库中，可以用 `df.isnull().sum()` 来快速检查。
3.  **一致性**：数据在不同部分或不同时间点是否遵循统一的格式和标准。例如，日期格式应统一为“YYYY-MM-DD”。
4.  **时效性**：数据是否在有效时间范围内，对于分析当前问题是否仍然相关。
5.  **唯一性**：数据集中不应存在不必要的重复记录。可以使用 `df.duplicated().sum()` 来检查重复行。

## 初步评估实践步骤

了解了核心维度后，我们来看看如何进行一次快速而有效的初步评估。以下是具体操作步骤：

*   **步骤一：数据概览**。使用 `df.head()`、`df.info()` 和 `df.describe()` 查看数据的前几行、结构及数值型字段的统计摘要。
*   **步骤二：检查缺失**。计算每个字段的缺失值比例，公式为：**缺失比例 = (缺失值数量 / 总行数) * 100%**。通常，缺失比例过高的字段需要考虑是否剔除。
*   **步骤三：查找异常**。观察数值型字段的最小值、最大值和分位数，识别可能的异常值（如超出合理范围的年龄或收入）。
*   **步骤四：验证一致性**。检查分类字段的唯一值，确保没有拼写错误或格式不一致的情况，例如“Male”和“male”应统一。

## 总结

本节课中我们一起学习了数据质量初步评估的基本概念与方法。我们首先明确了数据质量对于分析结果的决定性作用，然后介绍了准确性、完整性、一致性、时效性和唯一性这五个核心评估维度。最后，我们通过一个简单的四步流程，演示了如何对数据集进行快速检查。记住，“垃圾进，垃圾出”，在开始任何复杂分析之前，花时间了解并评估你的数据永远是值得的。
