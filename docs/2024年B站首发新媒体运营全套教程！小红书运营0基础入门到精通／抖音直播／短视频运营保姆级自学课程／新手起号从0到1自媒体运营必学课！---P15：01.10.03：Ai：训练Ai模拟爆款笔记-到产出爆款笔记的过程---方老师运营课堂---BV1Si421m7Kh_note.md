# AI新媒体运营教程：P15：01.10.03：训练AI从分析到产出爆款笔记 📝

## 概述
在本节课中，我们将学习如何训练AI（以ChatGPT为例）分析爆款笔记，并基于分析结果生成具备爆款潜质的新笔记。我们将通过一个考研领域的实例，完整演示从“喂资料”到“出文案”的全过程。

上一节我们介绍了通过“喂资料”让AI理解特定平台风格和用语的方法。虽然上一节的旅游攻略案例效果不理想，但方法是正确的。本节中，我们来看看如何让AI分析爆款内容，并学习其成功要素。

## 第一步：明确分析要点
在让AI分析爆款内容前，我们需要明确几个关键的分析维度。以下是需要AI重点关注的几个方面：

1.  **爆款因素分析**：分析内容为何能成为爆款，例如是否引发了情感共鸣、提供了实用价值等。
2.  **内容结构拆解**：分析文案的整体结构。
    *   **开篇**：用什么方式吸引用户继续阅读。
    *   **中间**：提供了哪些内容来戳中用户痛点，提升用户停留时长。
    *   **结尾**：设置了什么“钩子”来引导用户互动（如私信、关注）。
3.  **关键词策略分析**：分析笔记中是否植入了特定关键词。
    *   如果有，关键词是什么？
    *   关键词的出现频率和分布结构是怎样的？

## 第二步：收集并输入爆款文案
我们需要找到目标领域（本例为“考研”）的爆款笔记文案，并将其提供给AI进行分析。

![](img/215a25051296455d6c911f427e6c8807_13.png)

![](img/215a25051296455d6c911f427e6c8807_15.png)

操作指令示例：
```
我给你提供一些关于小红书平台上的考研相关爆款笔记文案，你需要分析这些文案的爆款因素，例如：
1. 文案的整体结构（开篇如何吸引用户，中间如何戳中痛点，结尾如何引导互动）。
2. 分析每篇笔记是否有结合中文分词法设计关键词。如果有，请告诉我是哪个关键词以及它的出现频率。
```

随后，将复制好的爆款文案内容粘贴给AI。

## 第三步：审阅AI的初步分析
AI会返回对每篇文案的分析结果。我们需要审阅其分析是否到位，例如是否准确识别了“情感共鸣”、“逆袭故事”、“详细规划”等爆款因素，以及是否正确判断了关键词的使用情况。

**注意**：AI（特别是3.5版本）可能在分析多篇内容后“忘记”最初的任务要求。如果发现分析跑偏，需要重新发送或强调最初的指令，将其“拉回”正轨。

![](img/215a25051296455d6c911f427e6c8807_23.png)

![](img/215a25051296455d6c911f427e6c8807_25.png)

## 第四步：整合爆款因素并生成大纲
在AI分析了数篇爆款笔记后，我们需要让它总结出共通的爆款因素。

操作指令示例：
```
请将上面分析过的几篇考研笔记的爆款因素总结一下，整合成一个去重后的列表。
```

获得爆款因素列表（如：情感共鸣、明确备考计划、时间节点清晰、资源技巧建议、正能量鼓励等）后，即可指令AI根据这些因素创作新笔记的大纲。

操作指令示例：
```
我要写一篇关于考研的小红书笔记，请结合你前面分析的爆款笔记因素，帮我梳理一个具备爆款因素的笔记大纲。每个大纲节点需要进一步细分。在我确认大纲没问题后，再生成完整文案。
```

## 第五步：根据大纲生成完整文案
对AI生成的大纲满意后，即可令其生成完整文案。此时需给出更具体的写作要求。

操作指令示例：
```
请按照上述大纲，写出具体的文案。要求如下：
1. 文案结构需严格遵循爆款笔记因素和结构。
2. 需要包含一个简单的“逆袭”小故事来引入读者情绪。
3. 需要包含适量的emoji表情。
4. 文案中需以中文分词法的方式融入“考研”等关键词。
```

## 第六步：优化与进阶思路
AI生成的初稿可能已具备可用性，但通常还有优化空间。我们可以：
*   手动调整语言，使其更精炼、更生动。
*   使用更强大的模型（如GPT-4）获得更优效果。
*   利用ChatGPT的“自定义GPT”（GPTs）功能，创建专属助手。你可以上传平台风格指南、常用表情包、优秀文案范例等文件供其学习，从而生成更符合你个人或品牌调性的内容。

## 总结
本节课我们一起学习了训练AI产出爆款笔记的完整流程：**明确分析维度 -> 输入爆款文案 -> 整合爆款因素 -> 生成内容大纲 -> 输出完整文案 -> 优化与进阶**。通过这个方法，你可以系统化地利用AI学习任何领域的爆款逻辑，并快速生成符合平台调性的优质内容草稿，极大地提升新媒体运营的效率。
