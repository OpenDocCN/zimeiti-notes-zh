# 亚马逊运营教程：P1：使用表格批量拆分商品变体

在本节课中，我们将学习如何通过亚马逊后台的“批量上传商品”功能，规范、高效地拆分商品变体。此方法利用电子表格进行操作，能有效避免手动操作的混乱与错误。

上一节我们介绍了拆分变体的重要性，本节中我们来看看如何借助官方工具实现标准化操作。

## 📋 第一步：下载库存文件模板
首先，登录亚马逊卖家后台。在“目录”菜单下，找到并点击“批量上传商品”。进入页面后，点击“下载库存文件”。

以下是具体操作步骤：
*   在下载页面，将电子表格语言切换为中文。
*   在搜索框中，输入你需要拆分变体的商品所属类目节点名称并进行搜索。
*   找到准确类目后，点击选择。鼠标向下滑动页面。
*   选择你想要发布商品的目标商城（例如亚马逊美国站）。
*   确认所有信息无误后，点击“生成电子表格”按钮进行下载。

## 📝 第二步：填写电子表格
下载完成后，打开生成的电子表格文件。点击底部名为 **`template`** 的工作表。

你需要先大致浏览表格的结构。核心填写规则很简单：
*   **表格的第一行用于填写父体（Parent）信息。**
*   **表格的第二行及之后的行，用于填写各个子体（Child，即变体）信息。**

请根据你的商品实际情况，参照图片中的示例和表格列标题的说明，填写所有必要信息，如SKU、商品关系类型、变体主题等。

## ⬆️ 第三步：上传并检查状态
填写好电子表格并保存后，回到亚马逊后台的“批量上传商品”页面。选择“上传您的电子表格”标签页，将刚才填写好的表格文件上传。

系统底部会出现上传进度条。等待上传完成后，点击“上传文件”按钮以提交处理。

最后，点击“监控上传状态”或“电子表格上传状态”进入相应页面。检查本次上传的处理状态。如果系统报告任何错误，请根据提示信息对表格进行修改，并重新上传直至成功。

本节课中我们一起学习了利用亚马逊官方电子表格来拆分商品变体的完整流程。从下载模板、规范填写到上传检查，这个方法能帮助你更有条理地管理商品变体关系。祝你销售顺利，天天报单！😊
