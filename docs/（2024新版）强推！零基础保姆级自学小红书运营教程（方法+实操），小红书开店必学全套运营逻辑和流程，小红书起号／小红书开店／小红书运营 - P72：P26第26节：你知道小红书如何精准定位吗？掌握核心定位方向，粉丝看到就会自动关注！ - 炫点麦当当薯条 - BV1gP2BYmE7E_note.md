# 小红书运营教程：P26：小红书账号类型与官方广告解析 🎯

在本节课中，我们将要学习小红书运营中的两个核心问题：不同类型的账号应如何选择与使用，以及官方广告投放的基本逻辑。通过理清这两者的关系，你可以为自己的品牌或业务制定更精准的运营策略。

## 小红书账号的三种类型

首先，我们介绍一下小红书的账号主要可以分为哪三类。

以下是三种主要的账号类型及其特点：

1.  **企业号**
    *   主要作用是树立官方品牌形象。
    *   拥有官方认证标志和特权，例如展示品牌名称、地址等。
    *   便于开展官方业务。

2.  **个人号**
    *   目前大多数用户运营的账号类型。
    *   特点是非常灵活，可以是品牌员工或消费者身份。
    *   更容易塑造个人品牌特色，适合品牌相关人物出镜。

3.  **引流号**
    *   在之前的课程中提及过，俗称“炮灰号”。
    *   数量可以是多个，作用相对次要，可以随时使用或舍弃。
    *   通常没有详细的个性规划，用于应对紧急情况或辅助引流。

一般来说，品牌的推广渠道主要通过企业号、个人号或引流号自行发布笔记来实现引流。

## 打造企业号矩阵的建议

上一节我们介绍了账号类型，本节中我们来看看如何协同使用这些账号，打造一个有效的运营矩阵。

以下是打造企业号矩阵的四点建议：

*   **积极参与品牌号活动**：例如，婚纱、服装美妆等企业号发起的试用或团购活动，可以用小号逐个参与。
*   **个人号展示品牌属性**：在个人号的笔记内容中，应多次暗示品牌的定位，例如目标用户是城市年轻人还是职场年轻人。
*   **个人号合理发布产品**：个人号发布品牌产品的频率不宜过高。建议采用 **3:7** 的比例，即30%的内容用于营销，70%的内容用于种草或科普。
*   **利用引流号进行评论引流**：当企业号需要引流时，最好使用引流号在其笔记评论区进行相关引导。

## 小红书官方广告（CPC竞价广告）

![](img/b0c4ec400fa8e429dba893a9ea4d2b3e_3.png)

除了运营自有账号矩阵，另一种重要的推广方式是小红书的官方广告。

![](img/b0c4ec400fa8e429dba893a9ea4d2b3e_5.png)

![](img/b0c4ec400fa8e429dba893a9ea4d2b3e_7.png)

官方广告即小红书的CPC（按点击付费）竞价广告。其申请和使用流程可在小红书官网的“业务合作” -> “推广合作”中找到。但请注意，投放官方广告有一个前提：**必须先拥有一个认证的企业号**。

![](img/b0c4ec400fa8e429dba893a9ea4d2b3e_9.png)

企业号的注册流程同样在官网完成，需要提交相关资质并支付审核认证费用。认证成功后，普通账号即可升级为官方企业号，从而获得投放信息流广告的资格。

## 官方广告的两种主要形式

![](img/b0c4ec400fa8e429dba893a9ea4d2b3e_11.png)

拥有了企业号之后，我们可以投放的官方广告主要分为两大类。

以下是两种主要的广告形式：

1.  **信息流广告**
    *   用户浏览小红书首页信息时，不定时出现在信息流中的广告。

2.  **搜索广告**
    *   当用户使用关键词搜索时（例如搜索“帽子”），出现的相关广告。系统算法可能会根据搜索词推荐搭配商品（如大衣）或相关品类商品（如搜索“考研资料”可能推荐营养品）。

其中，搜索竞价广告最具代表性，其原理与百度关键词竞价类似。通过对不同关键词出价，竞争不同的广告排名位置。

![](img/b0c4ec400fa8e429dba893a9ea4d2b3e_13.png)

目前，信息流广告位主要出现在以下三个位置：**第3位、第13位和第23位**。

关于广告投放的更多常见问题，可以在官方平台的问答区找到详细解答。需要注意的是，官方广告投放的费用相对较高。因此，在资金允许的情况下，可以考虑与官方合作投放竞价广告，因为它能直接获得由平台官方推荐的精准流量。
