## 一、课程回顾与核心逻辑

上一节我们介绍了投流的底层逻辑。本节中，我们来看看如何将其应用于实战。

付费投流并非一成不变，它需要根据直播间的状态和进度进行动态调整与优化。理解底层逻辑是后续所有优化动作的重要基础，因为它能帮助你在流量变化中做出正确决策。

付费起号的核心优势在于速度。相比自然流和私域，付费投流能最快地加速流量进入和账号学习。对于新号而言，它缺乏人群标签，自然流启动慢且不稳定，容易陷入“死亡曲线”。付费投流的目的，就是**打破账号的初始流速，快速建立人群模型**。

## 二、人货场：投流前的必备验证

在开始任何投流动作前，必须首先验证“人货场”（即主播、产品、场景）的承接能力。未经验证的人货场直接投流，大概率会失败。

以下是验证人货场承接能力的标准流程：

![](img/5276f3aa841928855ee5c31aced9c07b_7.png)

![](img/5276f3aa841928855ee5c31aced9c07b_9.png)

1.  **投放测试计划**：为新直播间创建一条500元预算的“商店”目标计划，设置长时间、低出价。
2.  **观察数据**：直播4小时以上，观察下播前20分钟的数据。
3.  **判断标准**：如果直播间的在线人数曲线（黄线）能稳定高于进入直播间的流量曲线（蓝线），则说明主播具备留住用户和互动的能力，人货场初步合格。
4.  **后续动作**：若无法拉出在线，则需优先调整人货场和排品，而非盲目加大投流。

![](img/5276f3aa841928855ee5c31aced9c07b_11.png)

![](img/5276f3aa841928855ee5c31aced9c07b_13.png)

**核心思想**：先把产品变得好卖，再把产品卖好。

## 三、付费暴力起号三步法 🎯

![](img/5276f3aa841928855ee5c31aced9c07b_15.png)

![](img/5276f3aa841928855ee5c31aced9c07b_17.png)

![](img/5276f3aa841928855ee5c31aced9c07b_19.png)

验证好人货场后，即可开始付费起号。一套成熟的打法通常分三场进行，若人货场能力强，甚至可在一天内完成。

![](img/5276f3aa841928855ee5c31aced9c07b_21.png)

### 第一场：硬核学习，打破冷启动

![](img/5276f3aa841928855ee5c31aced9c07b_23.png)

![](img/5276f3aa841928855ee5c31aced9c07b_25.png)

![](img/5276f3aa841928855ee5c31aced9c07b_27.png)

本场目标是利用付费流量，让系统快速学习直播间的人群行为。

以下是第一场的具体投放策略：

*   **开播动作**：直接预约投放“商店”和“涨粉”目标计划。
*   **计划设置**：时长2小时。定向设置：卡住性别、年龄，可尝试圈选对标账号或兴趣标签。若不确定，可通投让系统学习。出价方式：智能出价。
*   **私域配合**：若有私域流量，可同时引导进场，辅助拉高在线和互动。
*   **开播后20-30分钟**：补充投放“互动”目标计划。设置同上（性别年龄卡住，其他智能出价）。
*   **开播后1小时**：补充投放“订单成交”目标计划，时长2小时，智能出价。此计划可在1小时后手动关闭。
*   **关键动作**：2小时付费计划结束后**不要下播**，继续用自然流播1-2小时，让系统基于已学习的行为交集，尝试拉高自然流量，破开初始流速。

**底层逻辑**：系统学习用户行为有顺序：**停留 → 关注/商品点击 → 互动 → 成交**。我们按顺序投“商店”（点击）、 “涨粉”（关注）、“互动”，就是加速这些精准行为人群的进入和学习。成交则能最快速地打正人群模型。

### 第二场：精准收缩，巩固模型

经过第一场，账号已初步学习。本场目标是减少泛流量冲击，用更精准的计划辅助自然流。

以下是第二场的投放调整：

![](img/5276f3aa841928855ee5c31aced9c07b_29.png)

![](img/5276f3aa841928855ee5c31aced9c07b_31.png)

![](img/5276f3aa841928855ee5c31aced9c07b_33.png)

*   **开播动作**：**不要预约投放**。先开播，承接第一波自然流量，做好停留和互动。
*   **开播后20-30分钟**：开始投放“涨粉”、“商店”、“互动”目标计划。
*   **计划设置**：全部采用“控成本出价”，出价设为平台建议的最低值。时长3-4小时，2小时后可视情况关闭。定向应尽量圈选核心目标客户（参考第一场跑得好的计划）。
*   **开播后30分钟**：补充“订单成交”计划，设置同上（最低出价，3-4小时）。
*   **计划优化**：关掉第一场数据差的计划，重点投放“商品曝光率”和“商品点击率”数据好的目标。
*   **关键动作**：同样，付费计划结束后继续播1-2小时，用自然流破流速。

![](img/5276f3aa841928855ee5c31aced9c07b_35.png)

### 第三场：微付费稳号，进入稳定期

![](img/5276f3aa841928855ee5c31aced9c07b_37.png)

![](img/5276f3aa841928855ee5c31aced9c07b_39.png)

前两场成功后，账号应已打上标签。本场进入微付费阶段，辅助自然流稳定放大。

以下是第三场的策略：

*   **流量预期**：开播可能会有极速流量，需用福利品或话术做好承接。
*   **投放策略**：进入“微付费”阶段。挑选前两场数据最好的定向（目标和兴趣），以最低出价，设置24小时长时间计划。
*   **核心目标**：让付费流量以低流速持续为直播间补充精准流量，主要依靠自然流跑出最终结果。
*   **起号成功标志**：人群精准、流速稳定、转化率稳定。这标志账号已度过冷启动，进入可运营的稳定状态。

**代码示例（计划设置思路）**：
```python
# 第三场微付费计划设置逻辑
if 前两场数据好的定向目标 exists:
    计划.目标 = 数据好的定向目标
    计划.出价 = 系统建议最低出价
    计划.时长 = 24小时
else:
    计划.目标 = 通投学习
    计划.出价 = 保守出价
    计划.时长 = 6-8小时 # 观察后再调整
```

## 四、付费投流直接赚钱策略 💰

此策略适用于**高利润、高客单价**的直播间，核心是追求广告投入产出比（ROI）。

以下是高客单直播间投流赚钱的核心要点：

*   **投放节奏**：可以预约投放。开播即用精准计划硬砸，采用控成本出价。
*   **计划优化**：
    1.  每半小时可复制优秀计划进行叠投。
    2.  **重点观察计划漏斗数据**（商品曝光率、点击率、下单率），若漏斗转化率远低于直播间整体转化率，则关停。
    3.  ROI（投产比）不是唯一标准，但通投计划必须能赚钱，否则无意义。
    4.  哪个计划跑偏（数据差）就关掉哪个。
*   **辅助技巧**：
    1.  保持高在线率，有助于更多计划跑动。
    2.  一个计划只针对1-2个兴趣标签，便于测试和优化。
    3.  使用网页版或另一台设备（非直播手机）调广告，方便操作。
*   **与起号投流的区别**：低客单直播间投流核心是**撬动自然流**，而高客单直播间可以直接追求**付费流本身的盈利**。

## 五、关键补充与答疑 🔧

在本节中，我们汇总一些常见问题的解答和重要补充。

*   **私域与预约**：如果拥有私域流量或通过短视频积累了预约用户，将其与付费投流结合，起号速度和成功率将极大提升。因为这部分流量精准且能快速产生成交，帮助系统快速学习。
*   **新品类兴趣选择**：若新品不知如何圈选兴趣标签，前期可通投学习，或通过“对标账号”来间接定位人群。更精细的测兴趣方法将在后续课程详解。
*   **投流是精细活**：投流界面看似简单，但背后是目标、定向、出价、时长组合的复杂测试与优化过程，需要耐心记录数据并持续迭代。
