# 小红书电商教程：P3：小红书店铺注册指南 📝

![](img/9e5315059c9179c7751e26252af22a1e_1.png)

在本节课中，我们将学习如何从零开始，完成小红书个人店铺的注册流程。这是开启小红书电商之旅的第一步，流程虽不复杂，但有几个关键选项需要注意。

上一节我们介绍了开店前的准备工作，本节中我们来看看如何具体操作，完成店铺的实名认证与开通。

## 第一步：下载APP与登录账号

首先，你需要使用手机进行操作。建议使用个人常用手机或工作手机。

操作步骤如下：
1.  在手机应用商店搜索并下载 **小红书APP**。
2.  打开APP，使用准备好的手机号登录账号。
3.  登录后，点击右下角的 **“我”**，进入个人中心。

## 第二步：找到店铺开通入口

登录账号后，需要找到隐藏较深的店铺开通入口。其路径不像其他电商平台那样直接。

以下是进入路径：
1.  在“我”页面，点击右上角或左上角的 **“三条杠”** 图标。
2.  在弹出的菜单中，选择 **“创作中心”**。
3.  进入创作中心后，点击 **“更多服务”**。
4.  在更多服务页面，找到第三排左右的 **“开通店铺”** 选项（通常是一个独立方框），点击进入。

## 第三步：提交资料与选择店铺类型

点击“开通店铺”后，系统会引导你进行认证。以下是核心步骤与注意事项。

**核心注意事项：店铺类型选择**
对于零基础新手，强烈建议选择 **`个人店铺`**。这是用于熟悉平台操作和流程的最佳起点，试错成本低，有助于建立信心。

具体操作流程：
1.  在开通界面，点击 **“立即认证”**。
2.  按照系统提示，上传提前准备好的 **身份证正反面照片**。
3.  在店铺类型选择时，勾选 **“个人店”**。
4.  随后填写店铺基本信息。这些信息后期可以修改，但建议尽量填写准确，避免频繁更改。

## 第四步：完成认证与等待审核

资料提交后，便进入了平台审核阶段。

1.  确认所有信息无误后，提交认证申请。
2.  小红书系统的审核反馈时间约为 **5分钟至1天**，通常比主流电商平台稍慢，需要耐心等待。
3.  审核通过后，你的小红书店铺就正式注册成功了。

## 第五步：下载必备工具——客服工作台

店铺注册成功并非终点，还有一个关键工具需要准备。

为了处理订单和与买家沟通，你必须下载 **小红书商家版APP**（即客服工作台）。常规小红书APP无法实现即时聊天客服功能，所有售后咨询都需在此独立平台完成。

**操作指令：**
请务必在手机应用商店搜索 **“小红书商家版”** 并下载安装。
