# 拼多多运营教程：第1章：万能投产公式与调车策略 🚀

在本节课中，我们将要学习拼多多广告投放中的一个核心概念——“万能投产公式”，并掌握如何根据此公式调整广告出价以实现利润最大化。课程将涵盖公式的计算方法、应用场景以及针对不同广告数据的调车策略。

## 概述

拼多多广告投放没有绝对的“万能公式”。通常，预算低时采用低出价，预算充足时则可采用不限额高出价。然而，存在一个基于利润和签收率计算理想投产比的公式，可以帮助我们找到最赚钱的广告出价点。

## 核心投产比计算公式

上一节我们提到了寻找最赚钱的出价点，本节中我们来看看具体的计算方法。其核心公式如下：

**理想投产比 = (售价 / 利润) / 签收率 × 1.4**

让我们通过一个例子来理解这个公式：
*   **假设**：商品成本为10元，售价为20元，因此单件利润为10元。
*   **假设**：退货率为20%，则商品签收率为 1 - 20% = 80%。
*   **计算步骤**：
    1.  计算售价与利润的比值：20 / 10 = 2
    2.  代入公式计算：(2 / 80%) × 1.4 = 2.5 × 1.4 = 3.5

根据计算，当广告的投产比达到3.5时，这辆“车”（广告计划）处于最赚钱的状态。

## 广告数据诊断与调车策略

掌握了理想投产比后，我们需要根据广告的实际表现数据进行调整。以下是常见的四种数据场景及其对应的优化策略：

当广告数据出现以下情况时，可以参照以下策略进行优化。

1.  **曝光高，点击率低**
    *   **问题诊断**：广告展示了多次，但用户点击意愿低。
    *   **优化方向**：问题通常出在主图或标题上，需要优化主图吸引力和标题文案。

2.  **点击率高，曝光量少**
    *   **问题诊断**：用户愿意点击，但看到广告的总人数太少。
    *   **优化方向**：这往往是广告出价过低导致的，需要适当提高出价以获取更多曝光。

![](img/38673f8d3c314a95b26b7eafb0e937da_1.png)

3.  **曝光量低，转化率高**
    *   **问题诊断**：看到广告的人不多，但看到的人购买意愿很强。
    *   **优化方向**：这表示产品具有竞争力，应果断拉高出价，争取更多曝光来放大成交。

4.  **点击率和曝光量不增长，花费持续上涨**
    *   **问题诊断**：广告花费在增加，但效果没有提升。
    *   **优化方向**：应立即降低出价，避免无效消耗，并检查广告定向或竞争环境是否发生变化。
