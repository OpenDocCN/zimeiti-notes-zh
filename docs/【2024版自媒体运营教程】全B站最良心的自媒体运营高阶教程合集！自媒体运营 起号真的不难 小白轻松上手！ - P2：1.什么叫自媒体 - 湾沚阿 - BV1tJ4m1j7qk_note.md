上一节我们介绍了课程的整体框架，本节中我们来看看“自媒体”这个概念到底是什么。

自媒体，用最通俗的话来解释，就是个人或小团队利用互联网平台进行内容创作和传播的行为。具体来说，它包含以下几个关键步骤：

以下是构成自媒体行为的核心环节：

![](img/940c3c0ba4da7b9fc42dbfeef2ecb7f6_2.png)

1.  **注册专属账号**：在百度等搜索引擎上搜索“自媒体平台”，选择一个平台并注册一个属于你自己的账号。这个账号需要完成**实名认证**。
2.  **满足年龄门槛**：目前主流平台要求账号注册者必须**年满18周岁**。这是一个基本前提条件。
3.  **发布原创内容**：在你注册好的账号里，发布**由你自己创作**的文章或视频。这里需要特别强调“原创”的重要性。
4.  **核心原则：拒绝搬运**：自媒体赚钱**不是**简单地将别人的视频下载下来，再上传到自己的账号。平台有严格的审核机制，搬运行为通常无法获得收益，甚至会导致违规。**必须**是经过你自己动手处理、加工或完全原创的内容。

只有同时满足了以上所有条件——拥有实名认证的个人账号、发布自己创作的内容——这一系列行为才构成了我们所说的“自媒体”。
