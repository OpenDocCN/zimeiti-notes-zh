## 渠道一：微软 Edge 浏览器 Copilot 🌐
首先，我们来看看一个集成在系统中的便捷工具。我优先推荐的是微软Edge浏览器中的Copilot功能。它是微软基于ChatGPT技术，为Windows系统用户打造的内置AI助手。

### 使用前提与设置
使用Copilot有两个前提条件：
1.  登录微软账号（可使用Outlook邮箱注册）。
2.  需要准备“梯子”（VPN工具）以访问相关服务。

### 功能界面与模式
成功连接后，Copilot界面会出现在浏览器侧边栏或系统任务栏。它提供三种响应模式：
*   **更有创造力**：生成更开放、创新的回答。
*   **平衡**：在精确与创造力之间取得平衡。
*   **更精确**：提供更严谨、准确的回答。

对于非专业内容，建议在“更有创造力”和“平衡”模式间选择。若输出内容不符合要求，可尝试“更精确”模式。

**示例：生成短视频脚本**
我们可以直接向Copilot提出需求：
```
请帮我写一个关于装修的短视频脚本，时长控制在3分钟左右。
```
AI将开始生成包含场景、画面和旁白的详细脚本。

此外，Copilot还能分析你当前打开的网页内容，提供总结或解答，这在其侧边工具栏中实现。

### 优点与局限性
虽然Copilot基于GPT-4模型，功能强大且能执行一些简单的系统命令，但它也存在一些缺点：
1.  **精准度稍逊**：相较于官方ChatGPT-4.0，经过微软调校的Copilot在某些回答上可能不够精准。
2.  **连续对话能力受限**：在当前对话中持续提问多次后，可能需要开启新对话窗口。
3.  **记忆功能有所缺失**：在利用历史对话优化后续回答方面能力较弱，且缺乏插件功能。

尽管如此，对于大多数日常运营任务，Copilot已经完全够用。如果你的Edge浏览器没有Copilot图标，可以尝试下载最新的Edge Dev（开发测试版）。

## 渠道二：第三方集成平台与共享账号 🔌
上一节我们介绍了系统集成的方案，本节中我们来看看一些由开发者搭建的第三方平台。这些平台通过调用ChatGPT的API，让国内用户也能便捷访问。

以下是几个可用性较高的平台：

### 1. 灵感AI
这是一个集成了多种AI模型的平台。
*   **优点**：
    *   模型丰富，包括GPT-3.5、GPT-4.0（含联网版）、文心一言、通义千问等。
    *   GPT-4.0提供一定数量的免费使用额度。
    *   无需“梯子”即可访问。
*   **缺点**：响应速度可能较慢。
*   **费用**：提供免费试用，后续有包月等付费选项。

### 2. Wee（示例名称）
这是一个专注于提供GPT模型的平台。
*   **优点**：
    *   响应速度很快。
    *   支持GPT-3.5和GPT-4.0。
    *   具备语音回答功能。
*   **缺点**：
    *   免费额度（Token）有限，且通常仅适用于GPT-3.5。
    *   需要“梯子”访问。
    *   连续对话能力一般。

![](img/d64fcb7defcc2c27710ac5b155263fc8_5.png)

![](img/d64fcb7defcc2c27710ac5b155263fc8_7.png)

### 3. ChatGPT共享账号平台
这是一个提供共享账号信息的网站。
*   **优点**：
    *   可以免费使用大量共享的GPT-3.5账号。
    *   偶尔能找到可用的GPT-4.0付费共享账号（标注为“金光Plus”）。
*   **缺点**：
    *   账号状态不稳定，可能被封禁或达到使用上限。
    *   需要“梯子”访问。
    *   GPT-4.0账号稀缺且有限制（如每3小时约30条消息）。

![](img/d64fcb7defcc2c27710ac5b155263fc8_9.png)

**核心概念：Token**
在AI对话中，Token是计费和使用的基本单位，可以理解为单词或字词的碎片。例如：
```
“Hello, world!” 可能被拆分为 [“Hello”, “,”, “ world”, “!”] 4个Token。
```
平台提供的免费额度通常指一定数量的Token。

## 渠道三：国内AI大模型替代方案 🇨🇳
如果无法使用前述需要“梯子”的渠道，国内多家科技公司也推出了优秀的AI大模型，它们是无需特殊网络环境的选择。

![](img/d64fcb7defcc2c27710ac5b155263fc8_19.png)

以下是几个主要的国内AI模型：

### 1. 百度文心一言
百度的AI对话产品。
*   **体验**：基本功能完备，能完成脚本撰写等任务。响应速度较快。
*   **现状**：提供免费版和付费的“4.0”版本，可根据需要尝试。

### 2. 字节跳动豆包AI
字节跳动推出的AI助手。
*   **体验**：能生成结构清晰的短视频脚本，包含景别、画面、台词等要素，实用性较强。

### 3. 其他模型
此外，还有**阿里云的通义千问**、**昆仑万维的天工AI**等，都是可供选择的国内替代方案。

![](img/d64fcb7defcc2c27710ac5b155263fc8_21.png)

**对比测试**
我们以相同的指令测试不同模型：
```
指令：请帮我写一个关于装修的短视频脚本文案，要求镜头丰富，语言吸引人。
```
*   **国内模型（如文心一言、豆包）**：能提供结构完整、包含基本要素的脚本，满足一般需求。
*   **ChatGPT-4.0**：生成的脚本在场景设计、叙事节奏和语言感染力上往往更细腻、更具创意，体现出差异。

![](img/d64fcb7defcc2c27710ac5b155263fc8_23.png)

选择国内模型时，建议你亲自体验，挑选一个用起来最顺手、最能提升工作效率的工具。

## 总结
本节课中我们一起学习了体验和使用ChatGPT 4.0的多种渠道。

我们首先介绍了**微软Edge浏览器的Copilot**，它是一个方便但略有功能限制的系统集成方案。接着，探索了**第三方平台和共享账号**，这些渠道可能提供免费或低成本的GPT-4.0访问，但稳定性和速度各异。最后，我们了解了**国内AI大模型**，如文心一言和豆包AI，它们访问便捷，是可靠的备选方案。

![](img/d64fcb7defcc2c27710ac5b155263fc8_25.png)

对于新媒体运营而言，核心原则是**利用工具提升效率**。建议在条件允许的情况下，优先尝试功能更强大的GPT-4.0。如果访问受限，国内的AI模型也足以胜任大部分辅助工作。请根据你的实际网络环境和工作需求，选择最适合你的工具，并加以熟练运用。
