# 直播起号实战：09：新人如何拉人 🎯

在本节课中，我们将学习新人主播在开播初期如何有效吸引并留住观众，即“拉人”的核心方法。我们将摒弃僵化的“话术”思维，专注于通过精准的内容和状态，在关键的几秒钟内抓住目标人群的注意力。

---

上一节我们探讨了直播的整体框架，本节中我们来看看新人开播时最关键的“拉人”环节具体该如何操作。

我强调一点：我非常反对新人开播时照读所谓的“首播话术”。原因在于，这会禁锢你的思维。许多人不知道开播说什么，根本问题就在于认为必须背诵特定的话术或大纲。

实际上，新号首播并不要求你表现得多么完美。这场直播是给账号打标签的重中之重，但很多人并未意识到。我后续会有专门课程讲解。首播重要吗？重要。但每场直播都同样重要。对于依赖自然流量的知识分享或经验分享主播而言，只要有一场数据表现不佳，下一场的数据就可能变差。平台的奖励机制是：连续七天数据优秀，你会获得一波优质的流量扶持；反之，如果其中某天数据不好，则会拉低整体平均值。

让我们继续聚焦“新人如何拉人”。

**绝对不要在直播间照读首播话术。** 你需要明白，直播间留住观众的关键不在开播后的8到10分钟，而在于最开始的几秒钟。这就像短视频的“五秒完播率”一样，观众划走直播间和划走视频的决策时间相同。这里涉及“曝光进入率”等数据，所有直播数据都环环相扣，我会在专门讲曝光技术的课程中详述。

对于新人主播，筹备首播时通常会做一个非常规范化的开场，例如：“欢迎所有家人们来到我的直播间”。但往往话没说完，人就走了。那么，正确的开场应该是什么？

*   **状态**和**场景**是直播内容搭建的一部分。
*   同时，你需要像做短视频一样，在开头设置**爆点**。

以下是直播开场的核心要素，它们共同构成“黄金开场”：

*   **强前置**
*   **黄金五秒完播率**
*   **钩子话术**
*   **钩子画面**
*   **钩子槽点**
*   **钩子热度**

新人开播会经过平台的测试流量。开播后第一句话，**必须直击你对标人群的痛点，或抛出你独特的观点**。

以知识分享类主播为例。目标人群的普遍痛点是“个位数直播间播不起来”。那么，你可以这样说：

> “直播间刚刚打开，所有正在直播的朋友，如果你的直播间也是个位数，请留在我的直播间。你看一看我今天如何突破个位数。我必定能破，因为这已经不是我的第一个账号了。”

接下来，你需要密集地抛出几个与你目标人群相关的问题或痛点。

我以自己为例进行拆解。我抛出的点是：“你们纠结于个位数的真实原因，可能是账号根本不推流。” 不推流的原因有多种：账号先天被风控、后天有软违规等。我会接着说：

> “如果你的直播间是个位数，别总怀疑是自己能力或话术不行。反过来想想，是不是平台根本不给你推流？一个没有水源的池塘，怎么可能留住鱼？没有推流，自然没有停留。所以，你要先测试账号是否健康。”

有些账号确实不健康，会被平台风控，从而拿不到流量。这套逻辑是真实有效的，你可以借鉴使用。

这个开场的核心，决定了你账号未来成长的起点。我课程中讲过账号的四个阶段，而这是起号阶段最重要的环节。

你需要抛出3到4个这样的痛点。如果记不住，讲1到2个也可以。这与照读话术完全不同。你可以观察，所有能留住人的直播间，不管是通过画面、声音还是状态，其内容核心都在于用几句话或几个动作抓住观众。例如，有些主播的核心就是“三句话，五个动作”。

我的方法是：**戳中对标人群的痛点，从而留住并转化他们**。如果你无法留住和转化观众，后续运营就会出问题。

你不需要完全模仿我，但可以借鉴这个思路，并结合自身风格，挖掘出一套属于自己的独特方式。这就是你个人的直播风格，我后续课程会分享如何树立个人风格。

所以，新人拉人的核心在于：**不是8到10分钟，而是开播最初的几秒钟决定了观众的走留**。你必须把筹备首播的精力，全部投入到打磨这个“黄金开场”上。这是所有做自然流直播起号的关键。

这个原理是通用的。即使是带货直播间，前期也是依靠密集的话术钩子来吸引人。因为平台机制是相通的。

理解这一点，你就能明白我为何反对照读话术。**在你完成账号启动期后，进入成长期时，适当使用话术没有问题，因为那时你已有基础流量。但在启动期，你必须把所有能力聚焦于此：找到一个目标人群的最大痛点，并用一种方式告诉他们——留在这里，你有价值，我这里有你需要的答案。**

最后，有一个实操要点：**直播间是一对多的场景，你的目标是照顾好所有观众。** 因此，从开场拉满状态时起，你甚至可以不用紧盯屏幕上的进人信息或评论。

不要看这里（指在线人数），也不要看这里（指评论）。突然有人说你“吹牛”，或者你发现没人进来，都会极大影响你本就不稳定的直播状态，让你瞬间慌乱。这是百分之百的新人主播会犯的错误。

你的目光应该聚焦在哪里？假设这是你的手机直播画面：

```plaintext
[摄像头]
```
你的视线应固定在摄像头附近，大约在额头反射线的位置。我起号时有时也不看屏幕，只管说我的内容。

同时，你可以在话术中设计互动环节，例如：

> “如果你认可主播说的话，就留在直播间浅听几分钟。我会用我的真心和分享，让你感受到价值……”

这就是新人首播拉流量、拉人的核心。我已经讲得非常细致，几乎是手把手教学。你们可以在此基础上整理出自己的风格。通常，经过几场直播，你就能摸索出自己熟悉的风格，因为你会逐渐发现说什么内容观众反应最强烈。

---

本节课中，我们一起学习了新人主播拉人的核心方法：摒弃固定话术，在开播黄金几秒内，通过直击目标人群痛点、抛出独特观点并保持良好状态来吸引观众。关键在于打磨属于你自己的“黄金开场”，并在一对多的直播场景中稳住心态，聚焦内容输出。这是账号成功启动的第一步。
