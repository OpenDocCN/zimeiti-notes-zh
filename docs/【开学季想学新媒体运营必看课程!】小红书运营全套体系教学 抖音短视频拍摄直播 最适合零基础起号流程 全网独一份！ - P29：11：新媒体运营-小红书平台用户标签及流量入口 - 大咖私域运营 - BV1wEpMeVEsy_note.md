# 新媒体运营：P29：11：小红书平台用户标签及流量入口 📊

![](img/8e7edd2c2ea3037238216578e07837c7_1.png)

在本节课中，我们将深入探讨小红书平台如何为用户打上标签，以及平台有哪些主要的流量入口。理解这些机制，能帮助我们更好地分析自己的账号和笔记，从而让内容更精准地触达目标用户。

上一节我们介绍了小红书系统如何判定内容标签，本节中我们来看看平台是如何判定用户标签的。这个过程能帮助我们反向推导，理解我们的笔记是如何被系统推荐给用户的，以及如何让笔记被我们的精准用户看到。

## 用户标签判定机制 🔍

系统为用户打标签的主要依据是用户的**浏览行为**。用户刚注册时选择的兴趣范围，参考价值不大，它仅作为平台对用户内容方向的初步引导，无法进入系统深层的用户标签池。

系统根据用户的一系列浏览行为来判定其兴趣分类。以下是几个核心的判定维度：

以下是判定用户标签的几个核心行为指标：

1.  **点击率**
    *   **公式**：`点击率 = (点击某类内容的次数) / (系统推送的该类内容总次数)`
    *   系统在发现页向用户推送多种类型的内容（如美妆、家居、母婴）。用户点击哪一类内容更多，系统就能计算出其偏好。例如，用户点击了10条推送，其中美妆类点了3次，家居类2次，母婴类0次，系统会判定该用户对美妆更感兴趣，并持续推送更多美妆内容进行试探。

2.  **停留时长**
    *   即用户在某篇笔记页面的观看时长。即使同属美妆领域，如果用户点开笔记后很快划走，说明内容不吸引他。系统会尝试推送美妆下其他细分内容（如上妆、护肤）或其他领域内容（如家居），并观察用户的停留时长变化。例如，一个通常不看家居的用户，某天点开一篇家居笔记并停留很久，甚至仔细阅读评论，系统可能判断他有装修或购买家居用品的潜在需求。

3.  **页面浏览量**
    *   指用户浏览页面的深度。例如，用户不仅看了一篇笔记，还点进博主主页，并连续看了第二、第三篇笔记。这个数值（PV）对账号健康度有积极影响。在粉丝基数较小时，提升用户在你主页的浏览深度，能为账号带来正向评分。

4.  **互动行为**
    *   包括点赞、收藏、评论、关注。这些数据直接提升单篇笔记的质量和推荐几率。数据好的笔记更容易进入更大的流量池，同时也能带动账号内其他笔记的曝光，并吸引粉丝关注。

## 平台流量分发入口 🚪

了解用户标签后，我们来看看小红书主要的流量分发入口。作为用户你可能经常使用，但需要分析其区别。

小红书平台主要有四个流量入口：

以下是小红书平台的四个主要流量入口及其特点：

1.  **发现页**
    *   这是最主要的流量来源。内容包含：
        *   你已关注的达人发布的内容。
        *   系统根据你的**用户标签**（点击率、停留时长等）推荐的、你可能感兴趣的内容。
        *   系统为增加平台内容多样性而推荐的一些你可能未接触过的内容。

2.  **关注页**
    *   这里只展示你**已关注的所有博主**发布的最新笔记。用户通常直接从这里查找强关联的博主内容。

3.  **附近页**
    *   根据手机定位，推送同城或附近的内容。此板块以具有强地域特征的内容为主，如美食、本地旅游、线下店铺等。对于运营线下实体店（如美容院、餐厅）的账号，在投放信息流广告时，应重点关注附近人群的推送。

4.  **购物页**
    *   这是小红书向电商化发展的重要入口，直接匹配用户的购物需求。用户可在此搜索商品、进入商城或观看带货直播。但目前许多用户仍习惯通过搜索笔记来获取购物建议。

本节课中我们一起学习了小红书平台判定用户标签的核心逻辑（点击率、停留时长、页面浏览量、互动行为），以及四大流量入口（发现页、关注页、附近页、购物页）的特点与价值。理解这些底层机制，能帮助我们从平台和用户的角度出发，更有效地规划内容与运营策略，让笔记获得更精准的推荐。
