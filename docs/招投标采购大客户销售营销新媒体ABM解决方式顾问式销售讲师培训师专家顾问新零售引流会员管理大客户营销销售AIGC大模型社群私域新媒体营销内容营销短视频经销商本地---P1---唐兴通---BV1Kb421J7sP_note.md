# 销售心理学：P1：消除渴求感，建立自信销售

在本节课中，我们将要学习一个在销售工作中至关重要但常被忽视的心理技巧：如何消除自身的“渴求感”。这对于一线销售人员，无论是面向企业客户还是个人消费者，都同样重要。

## 概述：渴求感的负面影响

对于一线的销售人员，无论是面向企业还是个人消费者，都存在一个被大家忽略的问题：你必须想办法消除你的渴求感。

你要记住一句话：你个人表现得越迫切，你的客户就越感到害怕。你不应该去寻求客户的认可，因为那是一种渴望的表现。

## 渴求感如何损害销售关系

上一节我们提到了渴求感的存在，本节中我们来看看它具体如何破坏销售过程。

那种渴望的迫切行为，实际上会暴露你自己的弱点。它会引发对方的恐惧和不确定感。

客户会因此觉得你是否有威胁，是否隐瞒了很多信息。这最终会导致你的客户选择逃避。这背后其实是人类原始的心理防御机制在起作用，客户会根据你传递的信息做出判断。

对方表现得越迫切，那么你的感受就会越糟糕。

## 解决方案：找回自信与镇定

因此，在这样的关键节点中，你需要找回自信，找回镇定。

我们可以用一个核心公式来总结这个关系：
**渴求感 = 你的软弱弱点**

## 核心行动准则

以下是你在营销、销售以及日常沟通中需要贯彻的第一条行动准则：

*   第一条就是要打消掉你的渴求感。

## 总结

本节课中，我们一起学习了“渴求感”在销售中的危害。关键在于认识到，过度的迫切会暴露弱点、引发客户疑虑，从而阻碍成交。成功的销售始于自信与镇定的姿态，消除渴求感是建立平等、信任的客户关系的第一步。
