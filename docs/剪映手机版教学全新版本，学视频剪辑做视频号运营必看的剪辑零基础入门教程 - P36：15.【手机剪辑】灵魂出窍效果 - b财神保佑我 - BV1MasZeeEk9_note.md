# 剪映手机版教学：P36：15. 手机剪辑灵魂出窍效果

在本节课中，我们将学习如何使用剪映手机版制作一个流行的“灵魂出窍”视觉效果。这个效果常用于音乐卡点或营造奇幻氛围，操作简单，适合初学者掌握。

## 🎬 效果制作步骤详解

上一节我们介绍了动画的基础应用，本节中我们来看看如何组合多个功能实现创意效果。

以下是制作“灵魂出窍”效果的核心步骤列表。

1.  **调整视频时长**：首先，在剪映中导入视频，根据背景音乐的节拍，裁剪并调整视频的时长。
2.  **添加主体动画**：选中视频片段，点击下方工具栏的 **“动画”** 按钮。选择 **“组合动画”** 分类，为其添加 **“回弹伸缩”** 效果，并将动画时长拉满。
3.  **复制并创建画中画**：为视频添加滤镜后，将视频复制一份。选中复制出的片段，在工具栏中找到并点击 **“切画中画”**，将其移动到与主视频对齐的轨道。
4.  **设置混合模式**：选中画中画图层，点击 **“混合模式”**，将其更改为 **“绿色”**。
5.  **添加关键帧控制透明度**：在画中画图层的第二个黄点（或音乐节拍点）处，**添加一个关键帧**。接着，在视频结尾处，将画中画视频**放大**。然后，回到第一个关键帧正前方的位置，**将不透明度参数调整为0**。最后，在结尾关键帧处，也**将不透明度调整为0**。这样，画中画图层会呈现淡入淡出并放大的效果。
6.  **添加特效增强效果**：点击 **“特效”** 按钮，在 **“热门”** 分类中找到 **“荧光线描”** 效果，并将其强度参数调整为 **50** 左右。将特效的作用对象选择为 **“画中画”** 图层，并将其时长对齐视频的后两个节拍。后面的片段可同理重复此操作。

本节课中我们一起学习了如何利用剪映的动画、画中画、关键帧和特效功能，分步骤组合创造出“灵魂出窍”的视觉动画。关键在于通过**关键帧**控制画中画图层的**透明度**与**缩放**，再结合**荧光线描**特效增强视觉冲击力。掌握这个流程后，你可以尝试调整参数，创造出属于自己的独特效果。
