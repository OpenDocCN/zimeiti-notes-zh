# 微信视频号运营教程：P12：11.如何填写邀请函

在本节课中，我们将学习如何正确填写视频号创作者计划的邀请函。这是加入计划、获取官方资源支持的关键一步。我们将一步步拆解填写流程，确保信息准确无误。

上一节我们介绍了如何注册视频号，本节中我们来看看如何填写后续的邀请函，以完成创作者计划的加入。

![](img/6c3fe689f3b817b6dc6bea038d846450_1.png)

## 第一步：打开微信
首先，您需要打开手机上的微信应用。



![](img/6c3fe689f3b817b6dc6bea038d846450_1.png)


![](img/6c3fe689f3b817b6dc6bea038d846450_3.png)

## 第二步：进入“发现”页
打开微信后，点击底部导航栏的“发现”选项。



![](img/6c3fe689f3b817b6dc6bea038d846450_4.png)

![](img/6c3fe689f3b817b6dc6bea038d846450_3.png)

![](img/6c3fe689f3b817b6dc6bea038d846450_6.png)

![](img/6c3fe689f3b817b6dc6bea038d846450_4.png)


## 第三步：找到并进入“视频号”
在“发现”页面中，向上滑动，找到并点击“视频号”入口。图标通常是一个黄色的蝴蝶形状。



![](img/6c3fe689f3b817b6dc6bea038d846450_6.png)


![](img/6c3fe689f3b817b6dc6bea038d846450_8.png)

![](img/6c3fe689f3b817b6dc6bea038d846450_9.png)

## 第四步：访问个人主页
进入视频号后，点击页面右上角的个人头像图标，进入个人主页。



![](img/6c3fe689f3b817b6dc6bea038d846450_8.png)

![](img/6c3fe689f3b817b6dc6bea038d846450_9.png)


在个人主页中，向下滑动，找到“我的视频号”区域。这里会显示您的**账号昵称**和**粉丝数量**。
例如，昵称可能是“天道酬勤”，粉丝数显示为“39917”。在填写邀请函时，账号昵称就填写“天道酬勤”，粉丝数量填写“39917”。

## 第五步：填写邀请函信息
找到邀请函填写页面后，按照以下顺序逐一填写信息。

以下是需要填写的五项核心信息及填写说明：

1.  **视频号账号昵称**：填写您在上一步中查看到的账号昵称，例如 `天道酬勤`。
2.  **视频号粉丝数量**：填写您查看到的粉丝数，例如 `39917`。
3.  **活动邀请人**：由于是通过特定渠道加入，此处填写邀请人姓名，例如 `张良` 或 `张良老师`。
4.  **实名信息**：此处必须填写您微信支付绑定的**真实姓名**，例如 `周建国`。
5.  **结算手机号**：填写用于接收验证码和后续结算的本人手机号，格式为连续数字，例如 `13812345678`。

所有信息填写完毕后，仔细检查一遍，然后点击页面上的蓝色“提交”按钮。系统可能会弹出“确定要提交吗？”的确认框，点击“确认”即可完成提交。成功后，页面会提示“已成功加入创作者分成计划”。

## 第六步：获取与发布素材
提交成功后，作为学员，您可以从导师的素材库中获得已剪辑好、拥有版权的视频素材用于发布。

例如，素材库中可能包含各类影视动漫素材，如《还珠格格》、《我不是药神》、《大话西游》等，这些均已被处理并确保版权可用。

本节课中我们一起学习了填写视频号创作者计划邀请函的完整流程，从查找账号信息到填写五项关键内容，最后确认提交。同时，也了解了成功加入后如何获取正版素材进行发布。请严格按照步骤操作，确保信息准确，即可顺利开启您的视频号创作之旅。
