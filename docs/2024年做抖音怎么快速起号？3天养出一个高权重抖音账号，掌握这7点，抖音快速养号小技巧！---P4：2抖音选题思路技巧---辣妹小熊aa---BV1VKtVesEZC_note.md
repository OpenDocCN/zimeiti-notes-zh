# 抖音运营指南：P4：抖音选题思路技巧 🎯

在本节课中，我们将学习一套高效的抖音选题公式。这套方法能帮助你快速确定内容方向，解决“不知道发什么”的难题，从而更精准地吸引目标用户，为账号打上正确标签。

---

![](img/92b9618c362edad0800363a9e68928c8_1.png)

上一节我们介绍了账号养成的整体框架，本节中我们来看看如何具体构思每一条视频的选题。选题是内容创作的第一步，也是决定视频能否获得流量的关键。

会找选题的人，能将更多时间投入视频制作。每天发布三条视频，一个月就能积累90条内容。依靠数量优势就能在竞争中脱颖而出，甚至无需依赖付费推广。此外，擅长选题的人能更精准地定位用户痛点，也更容易吸引精准粉丝观看，从而帮助账号快速打上标签。

那么，如何解决选题困难的问题呢？当然有方法。这是一套百万粉丝博主常用的选题技巧。为了方便理解，我们将其总结为一个公式：

![](img/92b9618c362edad0800363a9e68928c8_3.png)

**选题公式 = 内容定位 + 用户痛点 + 借势热点**

用好这个公式，你的内容方向就不会跑偏。

以下是应用该公式的具体步骤：

![](img/92b9618c362edad0800363a9e68928c8_5.png)

首先，确定你的**内容定位**。例如，如果你想成为一名知识博主，就需要挖掘行业内的关键词，如“播放量”、“粉丝量”、“定位”、“选题”等。这些关键词应客观、具体，不包含主观描述。

接着，在上述基础上寻找**用户痛点**。用户的痛点通常围绕他们的需求和困扰，例如“便捷”、“高效”、“技巧教学”、“内容输出”等。

最后，结合近期的**热点**进行创作。将定位、痛点和热点进行任意组合，就能生成大量选题。

例如，临近春节时，可以创作如下选题：
*   “如何用拜年话术提升视频播放量”
*   “春节必火的三大选题内容”
*   “解析‘张同学’快速涨粉的三个秘诀”

为了加深理解，我们再将这个公式套用到另一个行业试试看。以美妆行业为例：

*   **内容定位**：美白、除皱、口红。
*   **用户痛点**：平价好用、明星同款、避坑扫雷。
*   **借势热点**：结合明星动态或节日。

进行排列组合后，可以得到如下选题：
*   “迪丽热巴同款美白面膜推荐”
*   “七夕千万别送女朋友这几种口红”

![](img/92b9618c362edad0800363a9e68928c8_7.png)

以上我们已经举了两个行业的例子进行“举一反二”。至于“举一反三”的任务就交给你了。你可以根据自己所在行业的特性，运用这个公式进行思考，并将你的思考过程分享出来。

本节课中，我们一起学习了抖音选题的核心公式：**内容定位 + 用户痛点 + 借势热点**。掌握这个公式，能帮助你系统化地生成大量优质选题，节省思考时间，更精准地触达目标观众，从而加速账号的成长。
