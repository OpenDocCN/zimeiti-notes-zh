# 抖音运营实战：P19：视频剪辑与提词工具使用教程 🎬

在本节课中，我们将学习如何使用剪映App进行视频剪辑，以及如何利用爱提词App辅助口播视频的拍摄。课程将详细讲解从导入素材、调整画面、剪辑内容、添加音乐字幕到最终导出的完整流程，并介绍一款能提升口播拍摄效率的实用工具。

## 剪辑软件选择与导入

![](img/d70d2657ddc81d257b60425265b89f8d_1.png)

上一节我们介绍了视频制作的前期准备，本节中我们来看看具体的剪辑操作。我们使用**剪映**进行剪辑，这是抖音官方的剪辑工具，有手机版和电脑版。以下从手机版开始讲解。

首先，打开剪映App，点击“开始创作”按钮。

![](img/d70d2657ddc81d257b60425265b89f8d_3.png)

![](img/d70d2657ddc81d257b60425265b89f8d_1.png)

![](img/d70d2657ddc81d257b60425265b89f8d_5.png)

接着，从相册中选择需要剪辑的视频素材，点击“添加”将其导入到剪辑轨道中。如果您的视频由多个片段组成，建议先导入并剪辑好第一个片段，再点击轨道尾部的“+”号添加后续片段，这样可以保持工程清晰有序。

![](img/d70d2657ddc81d257b60425265b89f8d_3.png)
![](img/d70d2657ddc81d257b60425265b89f8d_5.png)

## 视频画面基础调整

![](img/d70d2657ddc81d257b60425265b89f8d_7.png)

视频导入后，第一步是检查并调整画面构图。以下是调整画面比例和方向的步骤。

![](img/d70d2657ddc81d257b60425265b89f8d_9.png)

首先，选中轨道上的视频片段。然后向右滑动底部工具栏，找到并点击“编辑”选项（图标通常是一个方块）。

![](img/d70d2657ddc81d257b60425265b89f8d_11.png)

![](img/d70d2657ddc81d257b60425265b89f8d_7.png)

在编辑菜单中，点击“旋转”功能，将画面调整到正确的方向。

![](img/d70d2657ddc81d257b60425265b89f8d_9.png)

如果需要裁剪画面比例，可以点击“裁剪”功能。这里提供了多种比例选项，如 **1:1**、**4:3**、**16:9** 等，您可以根据抖音的展示需求或个人喜好进行选择。

![](img/d70d2657ddc81d257b60425265b89f8d_13.png)

![](img/d70d2657ddc81d257b60425265b89f8d_11.png)

![](img/d70d2657ddc81d257b60425265b89f8d_15.png)

## 视频内容剪辑

调整好画面后，接下来需要对视频内容进行修剪，去掉多余或无效的部分。剪辑的核心操作是“分割”与“删除”。

![](img/d70d2657ddc81d257b60425265b89f8d_17.png)

播放视频，找到需要剪掉的起始位置（例如开头沉默的部分）。将时间轴（白色竖线）移动到这个位置。

![](img/d70d2657ddc81d257b60425265b89f8d_19.png)

选中视频片段，点击底部工具栏的“分割”按钮。视频会被从此处切成两段。

![](img/d70d2657ddc81d257b60425265b89f8d_21.png)

![](img/d70d2657ddc81d257b60425265b89f8d_13.png)

选中需要删除的前一段视频，点击“删除”按钮即可将其移除。重复此操作，可以删除中间或结尾不需要的部分。

![](img/d70d2657ddc81d257b60425265b89f8d_23.png)

**操作总结公式为：`选中片段 -> 定位分割点 -> 点击分割 -> 选中多余段落 -> 点击删除`。**

![](img/d70d2657ddc81d257b60425265b89f8d_25.png)

![](img/d70d2657ddc81d257b60425265b89f8d_15.png)

![](img/d70d2657ddc81d257b60425265b89f8d_27.png)

## 添加与编辑背景音乐

![](img/d70d2657ddc81d257b60425265b89f8d_29.png)

视频剪辑完成后，需要添加背景音乐。以下是添加和调整音乐的方法。

![](img/d70d2657ddc81d257b60425265b89f8d_31.png)

点击底部工具栏的“音频”按钮。如果您已有下载好的音乐文件，可以点击“提取音乐”，从手机视频中提取音频。

![](img/d70d2657ddc81d257b60425265b89f8d_33.png)

![](img/d70d2657ddc81d257b60425265b89f8d_17.png)

如果没有，则点击“添加音乐”。剪映内置了丰富的音乐库，您可以通过分类查找或直接搜索来选择合适的音乐。

![](img/d70d2657ddc81d257b60425265b89f8d_35.png)

![](img/d70d2657ddc81d257b60425265b89f8d_19.png)
![](img/d70d2657ddc81d257b60425265b89f8d_21.png)

![](img/d70d2657ddc81d257b60425265b89f8d_37.png)

点击音乐可以预览，确认后点击“使用”即可添加到轨道。

![](img/d70d2657ddc81d257b60425265b89f8d_39.png)

![](img/d70d2657ddc81d257b60425265b89f8d_23.png)
![](img/d70d2657ddc81d257b60425265b89f8d_25.png)

如果音乐过长，其剪辑方法与视频相同：`选中音频 -> 分割 -> 删除`多余部分。

![](img/d70d2657ddc81d257b60425265b89f8d_41.png)

![](img/d70d2657ddc81d257b60425265b89f8d_27.png)
![](img/d70d2657ddc81d257b60425265b89f8d_29.png)

您还可以调整音乐音量。选中音频轨道，点击“音量”按钮，拖动滑块即可调节。

![](img/d70d2657ddc81d257b60425265b89f8d_43.png)

![](img/d70d2657ddc81d257b60425265b89f8d_31.png)
![](img/d70d2657ddc81d257b60425265b89f8d_33.png)

![](img/d70d2657ddc81d257b60425265b89f8d_45.png)

## 自动识别与添加字幕

添加字幕能让视频更专业、更易懂。剪映提供了高效的自动识别功能。

![](img/d70d2657ddc81d257b60425265b89f8d_47.png)

点击底部工具栏的“文字”按钮，然后选择“识别字幕”。

![](img/d70d2657ddc81d257b60425265b89f8d_35.png)

![](img/d70d2657ddc81d257b60425265b89f8d_49.png)

在弹出的菜单中，选择“仅识别视频原声”，这样可以避免将背景音乐的歌词也识别为字幕。点击“开始识别”。

![](img/d70d2657ddc81d257b60425265b89f8d_37.png)

![](img/d70d2657ddc81d257b60425265b89f8d_51.png)

识别完成后，字幕会自动添加到视频中。您可以双击任意字幕条修改错别字，也可以在右侧菜单中调整字体、颜色、样式等。

![](img/d70d2657ddc81d257b60425265b89f8d_53.png)

![](img/d70d2657ddc81d257b60425265b89f8d_39.png)
![](img/d70d2657ddc81d257b60425265b89f8d_41.png)
![](img/d70d2657ddc81d257b60425265b89f8d_43.png)

## 导出高清视频

![](img/d70d2657ddc81d257b60425265b89f8d_55.png)

所有剪辑工作完成后，在导出前需要进行关键设置以保证视频清晰度。

![](img/d70d2657ddc81d257b60425265b89f8d_57.png)

点击右上角的“导出”按钮。在导出设置页面，请调整两个参数：
1.  **分辨率**：建议拉至最高（如1080P），数值越高视频越清晰。
2.  **帧率**：设置为 **30 fps**，这对抖音平台最为合适。

![](img/d70d2657ddc81d257b60425265b89f8d_45.png)
![](img/d70d2657ddc81d257b60425265b89f8d_47.png)

设置完成后，点击“导出”按钮。导出过程中请勿锁屏或切换应用程序，耐心等待完成即可。视频导出后，可以直接分享到抖音。

![](img/d70d2657ddc81d257b60425265b89f8d_49.png)
![](img/d70d2657ddc81d257b60425265b89f8d_51.png)
![](img/d70d2657ddc81d257b60425265b89f8d_53.png)

![](img/d70d2657ddc81d257b60425265b89f8d_59.png)

## 口播提词工具推荐

![](img/d70d2657ddc81d257b60425265b89f8d_61.png)

对于需要拍摄口播视频的用户，背诵台词是一大难题。接下来我们介绍一款能解决这个问题的工具——**爱提词**。

这是一款提词器软件，可以帮助您在不背诵的情况下流畅完成口播。打开爱提词App，点击右下角的“+”号添加新台词。

![](img/d70d2657ddc81d257b60425265b89f8d_55.png)
![](img/d70d2657ddc81d257b60425265b89f8d_57.png)

![](img/d70d2657ddc81d257b60425265b89f8d_63.png)

输入台词标题和正文（可直接粘贴复制好的文案），然后保存。

![](img/d70d2657ddc81d257b60425265b89f8d_59.png)

![](img/d70d2657ddc81d257b60425265b89f8d_65.png)

保存后，点击“开始提词”并选择“录制模式”。此时台词会以悬浮窗形式显示在屏幕上，您就可以看着台词进行录制了。

![](img/d70d2657ddc81d257b60425265b89f8d_67.png)

![](img/d70d2657ddc81d257b60425265b89f8d_61.png)

在提词界面，点击中部可以进行设置，例如调整字体大小，方便近视用户观看。

![](img/d70d2657ddc81d257b60425265b89f8d_69.png)

![](img/d70d2657ddc81d257b60425265b89f8d_63.png)

![](img/d70d2657ddc81d257b60425265b89f8d_71.png)

该软件还内置了拍摄功能。点击左下角的“美颜”按钮，可以开启美颜效果，如瘦脸、大眼、磨皮等，让拍摄效果更自然美观。
