## 概述
在本节课中，我们将要学习抖音运营中一个至关重要的核心概念：**用户思维**。我们将探讨如何从传统的销售思维转向用户思维，理解抖音作为娱乐平台的本质，并学习如何通过内容编排和标题设计来吸引用户，从而实现有效的引流和转化。

## 从销售思维到用户思维
上一节我们介绍了商业定位的重要性。本节中，我们来看看如何将定位转化为有效的内容。首先，我们需要理解两种不同的思维方式。

![](img/fa7af63a152db82b5457e55ac86a5ac9_30.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_32.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_34.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_36.png)

如果以纯粹的销售思维去做抖音，我们的做法可能是直接宣传产品。例如：
*   我是医美机构，我能让你变美。
*   我的医生特别专业权威。
*   我的书原价59元，今天七夕特惠降10元。

![](img/fa7af63a152db82b5457e55ac86a5ac9_38.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_40.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_42.png)

这种直接推销的方式在**电商平台（如淘宝）** 上可能有效，因为用户打开淘宝时，本身就带有明确的购物需求和消费预期。

![](img/fa7af63a152db82b5457e55ac86a5ac9_44.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_46.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_48.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_50.png)

然而，在**抖音**上，这种方式往往行不通。因为抖音本质上是一个**娱乐平台**。用户打开抖音的主要目的是为了放松和寻找乐趣。当用户处于娱乐心态时，生硬的推销会让他们感到厌烦，即使优惠再大，也可能因为“与我无关”而被直接划走。

![](img/fa7af63a152db82b5457e55ac86a5ac9_52.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_54.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_56.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_58.png)

因此，我们必须转换思维，从“我想卖什么”转向“**用户想看什么**”。这就是**用户思维**的核心。

### 内容编排：把“利益”前置
如果你要制作一个视频，讲解几种打光技巧（如测光、底光、蝴蝶光、轮廓光），你会如何安排内容顺序？

![](img/fa7af63a152db82b5457e55ac86a5ac9_84.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_86.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_88.png)

一个没有用户思维的顺序可能是随机的，想到哪个讲哪个：
1.  测光怎么打——让视频更清晰。
2.  底光怎么打——让画面更恐怖。
3.  蝴蝶光怎么打——让人物更美丽。

![](img/fa7af63a152db82b5457e55ac86a5ac9_90.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_92.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_94.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_96.png)

但如果我们运用用户思维，就需要思考：哪种效果对用户最有吸引力？最能给他们带来“利益获得感”？

![](img/fa7af63a152db82b5457e55ac86a5ac9_98.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_100.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_102.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_104.png)

显然，“让女生更漂亮”比“让画面更恐怖”对大多数用户更具吸引力。因此，我们应该将最具吸引力的点放在视频最前面：
1.  **蝴蝶光怎么打——让人物轮廓更美，更上镜。**
2.  测光怎么打——让视频更清晰。
3.  底光怎么打——营造特殊氛围。

![](img/fa7af63a152db82b5457e55ac86a5ac9_106.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_108.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_110.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_112.png)

**核心公式：内容价值 = 用户感知利益 / 用户付出成本（注意力、时间）**

![](img/fa7af63a152db82b5457e55ac86a5ac9_114.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_116.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_118.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_120.png)

通过把高价值、高利益点的内容前置，能在视频开头瞬间抓住用户注意力，提高完播率。

### 标题设计：回答“关我什么事”
标题是吸引用户点击的第一道门。一个只有“销售思维”的标题可能是：“卷心菜教你三种打光方式”。

![](img/fa7af63a152db82b5457e55ac86a5ac9_130.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_132.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_134.png)

但用户看到后的第一反应可能是：“打光方式？关我什么事？”

![](img/fa7af63a152db82b5457e55ac86a5ac9_136.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_138.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_140.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_142.png)

运用用户思维，我们需要在标题中直接点明用户能获得的利益或结果。例如，可以修改为：
*   **“一招打光技巧，让你在视频里颜值翻倍！”**
*   **“想把女朋友拍得更美？这个摄影师必会的打光神技快学起来！”**

![](img/fa7af63a152db82b5457e55ac86a5ac9_144.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_146.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_148.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_150.png)

这样的标题直接关联了用户的潜在需求（变美、拍好照），给出了明确的利益承诺，点击率自然会更高。

## 核心要点总结
本节课中我们一起学习了抖音内容创作的基石——用户思维。以下是需要掌握的核心要点：

![](img/fa7af63a152db82b5457e55ac86a5ac9_166.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_168.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_170.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_172.png)

1.  **平台本质认知**：抖音是娱乐平台，不是电商平台。用户的核心需求是消遣，而非购物。
2.  **思维模式转换**：放弃硬推销的“销售思维”，转向思考“用户喜欢什么”的**用户思维**。
3.  **内容创作核心**：在策划视频的每一秒时，都要预判用户的反应。将最能给用户带来**利益获得感**的内容放在最前面。
4.  **标题设计关键**：标题必须回答用户的疑问“这关我什么事？”，直接点明他能获得的具体好处或解决的具体问题。

![](img/fa7af63a152db82b5457e55ac86a5ac9_174.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_176.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_178.png)

![](img/fa7af63a152db82b5457e55ac86a5ac9_180.png)

记住这个核心循环：**洞察用户需求 -> 创作对应内容（利益前置）-> 通过标题强化利益点 -> 实现吸引和转化**。始终站在用户的角度去思考，你的内容才能真正在抖音上获得关注。
