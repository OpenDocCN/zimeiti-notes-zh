# 微信视频号运营从0~1全攻略：P23：11.如何填写邀请函 📝

在本节课中，我们将学习如何正确填写视频号创作者计划的邀请函。这是加入计划、获取官方支持的关键一步，我们将一步步拆解每个填写项，确保信息准确无误。

## 概述

![](img/1b1824508caa17cc68b98f96ce045741_1.png)

填写邀请函是加入视频号创作者计划的必要流程。本节教程将详细指导你如何找到自己的视频号信息，并逐一填写邀请函中的各项内容，包括账号昵称、粉丝量、邀请人、实名信息及结算手机号。

## 查找视频号账号信息

上一节我们介绍了如何注册视频号，本节中我们来看看如何找到自己的账号昵称和粉丝数量，这是填写邀请函的第一步。

![](img/1b1824508caa17cc68b98f96ce045741_3.png)

![](img/1b1824508caa17cc68b98f96ce045741_4.png)

以下是查找账号信息的详细步骤：

1.  **打开微信应用**
    在手机上找到并打开微信应用。

2.  **进入“发现”页面**
    在微信底部导航栏中，点击“发现”选项。

3.  **进入“视频号”入口**
    在“发现”页面中，向上滑动，找到并点击“视频号”入口（图标为黄色蝴蝶形状）。

    ![](img/1b1824508caa17cc68b98f96ce045741_3.png)
    ![](img/1b1824508caa17cc68b98f96ce045741_4.png)

![](img/1b1824508caa17cc68b98f96ce045741_6.png)

![](img/1b1824508caa17cc68b98f96ce045741_7.png)

4.  **点击个人头像**
    进入视频号页面后，点击右上角的个人头像图标。

    ![](img/1b1824508caa17cc68b98f96ce045741_6.png)

5.  **查看“我的视频号”**
    在打开的页面中向下滑动，找到“我的视频号”栏目。这里会清晰显示你的**账号昵称**和**粉丝数量**。

    ![](img/1b1824508caa17cc68b98f96ce045741_7.png)

    例如，显示为“天道酬勤，39917个粉丝”，则：
    *   **账号昵称**：`天道酬勤`
    *   **粉丝数量**：`39917`

## 填写邀请函各项内容

成功找到账号信息后，我们就可以开始填写邀请函了。邀请函通常包含以下几个核心部分，需要逐一准确填写。

以下是邀请函的具体填写指南：

1.  **视频号账号昵称**
    填写你在上一步中查看到的账号昵称，例如：`天道酬勤`。

2.  **粉丝数量**
    填写你在上一步中查看到的粉丝总数，例如：`39917`。

3.  **活动邀请人**
    根据你的加入渠道，填写指定的邀请人姓名。例如，通过本教程渠道加入，则填写：`张良` 或 `张良老师`。

![](img/1b1824508caa17cc68b98f96ce045741_9.png)

![](img/1b1824508caa17cc68b98f96ce045741_10.png)

4.  **实名信息**
    此处必须填写你本人的真实姓名，与身份证信息一致。例如：`周建国`。

![](img/1b1824508caa17cc68b98f96ce045741_11.png)

5.  **结算手机号**
    填写你本人用于接收验证码和后续结算的**手机号码**，格式需正确无误，例如：`13812345678`。

    ![](img/1b1824508caa17cc68b98f96ce045741_9.png)
    ![](img/1b1824508caa17cc68b98f96ce045741_10.png)

## 提交与确认

所有信息填写完毕后，最后一步是提交并确认。

1.  **点击提交**
    检查所有信息无误后，点击页面上的蓝色“提交”按钮。

![](img/1b1824508caa17cc68b98f96ce045741_13.png)

![](img/1b1824508caa17cc68b98f96ce045741_14.png)

2.  **确认提交**
    系统会弹出“确定要提交吗？”的确认框，务必点击“确认”完成提交。

    ![](img/1b1824508caa17cc68b98f96ce045741_11.png)

3.  **提交成功**
    成功提交后，页面会显示“已成功加入创作者分成计划”的提示。
