## 1. 如何申请加入品牌库 🏷️

![](img/03a01d02b8fe5dbd61e51ea24b7e6ec9_3.png)

![](img/03a01d02b8fe5dbd61e51ea24b7e6ec9_4.png)

在编辑商品时，若发现品牌属性下拉选项中没有自己的品牌，可以申请加入品牌库。

**操作路径：**
在商品编辑页面的“品牌属性”栏下方，找到并点击 **“点击申请”** 按钮。

**申请流程：**
1.  根据实际情况填写品牌名称。
2.  填写品牌的注册证编号。
3.  选择品牌所属的类目。
4.  完成填写后提交申请。

申请成功后，在后续编辑商品时即可直接选择并使用自己的品牌。新发布商品时，系统也会默认关联到对应的品牌。

## 2. 如何查看商品发布进度 📊

发布商品后，需要了解其审核状态（如发布中、已驳回、发布成功）。

![](img/03a01d02b8fe5dbd61e51ea24b7e6ec9_6.png)

**操作路径：**
登录拼多多商家管理后台，依次点击 **“商品管理” -> “商品列表” -> “上传记录”**。

**状态说明：**
*   **发布成功**：商品已成功上架。
*   **发布中**：商品正在审核或发布流程中。
*   **已驳回**：商品因不符合平台规范被驳回。

![](img/03a01d02b8fe5dbd61e51ea24b7e6ec9_8.png)

若商品被驳回，需在“上传记录”中查看具体原因，并按平台要求修改后重新提交上传。

## 3. 如何管理草稿箱与回收站中的商品 📁

![](img/03a01d02b8fe5dbd61e51ea24b7e6ec9_10.png)

在编辑商品时，若因故无法立即完成上架，可保存为草稿；误删除的商品可在回收站找回。

**操作路径：**
在商家后台的 **“商品管理 -> 商品列表”** 页面右上角，可找到 **“草稿箱”** 和 **“回收站”** 入口。

**功能说明：**
*   **草稿箱**：存放未完成发布的商品草稿，可随时打开继续编辑并上架。
*   **回收站**：存放已删除的商品。若商品误删或找不到，可在此处找回并重新发布上架。

![](img/03a01d02b8fe5dbd61e51ea24b7e6ec9_12.png)

![](img/03a01d02b8fe5dbd61e51ea24b7e6ec9_13.png)

## 4. 如何处理平台要求的商品寄样 📦

![](img/03a01d02b8fe5dbd61e51ea24b7e6ec9_14.png)

部分类目商品或提报活动时，平台可能要求商家寄送样品。

**操作路径：**
在商家后台，依次点击 **“店铺营销 -> 竞价活动 -> 竞价助手”**，页面内可找到 **“寄样管理”**。

![](img/03a01d02b8fe5dbd61e51ea24b7e6ec9_16.png)

**重要提示：**
目前平台要求的商品寄样**基本不支持退回**，请在寄样前知悉此规则。若确需寄样，请按上述路径完成操作。

## 补充说明与操作技巧 💡

![](img/03a01d02b8fe5dbd61e51ea24b7e6ec9_21.png)

除了以上四个核心问题，还有两个实用的操作技巧需要了解。

### 资源位商品的下架流程

参与平台活动的商品会上架到特定“资源位”，这类商品不能像普通商品一样随意下架。

**下架流程：**
若需下架资源位商品，必须点击 **“申请下架”** 并提交官方客服审核。审核通常在24小时内完成，**审核通过后方可下架**，若未通过则暂时无法下架。

![](img/03a01d02b8fe5dbd61e51ea24b7e6ec9_23.png)

![](img/03a01d02b8fe5dbd61e51ea24b7e6ec9_24.png)

### 快速查找店铺商品的方法

当店铺商品数量较多时，可以使用筛选功能快速定位目标商品。

**操作方法：**
在“商品列表”页面，您可以通过以下方式筛选：
1.  在搜索框中输入 **商品ID** 或 **商品标题关键词**。
2.  利用列表顶部的排序功能，如按 **“上架时间”** 或 **“累计销量”** 排序，快速找到商品。
