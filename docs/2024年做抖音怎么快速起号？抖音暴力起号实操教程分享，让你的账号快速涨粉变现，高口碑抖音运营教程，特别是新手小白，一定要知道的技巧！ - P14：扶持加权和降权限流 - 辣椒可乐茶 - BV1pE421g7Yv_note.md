# 抖音运营教程：P14：扶持加权与降权限流 📊

在本节课中，我们将学习抖音平台对视频内容的两种核心处理机制：扶持加权与降权限流。理解这些规则，是确保你的账号获得流量推荐、避免违规处罚的关键。

上一节我们介绍了账号权重的分类，本节中我们来看看哪些内容会受到平台的扶持，哪些内容又会导致降权限流。

## 扶持加权的内容特征 ✅

![](img/6178f248838a1ab1abed0f4ea48faad7_1.png)

平台会对符合以下特征的优质内容进行流量扶持和权重加成。

![](img/6178f248838a1ab1abed0f4ea48faad7_3.png)

以下是能够获得扶持加权的内容标准：

*   **视频格式**：必须为竖版移动端原生视频，即 **9:16** 的格式。
*   **画面质量**：画面平稳，作品清晰度高。
*   **音频质量**：无过多的杂音和噪音。

![](img/6178f248838a1ab1abed0f4ea48faad7_1.png)

![](img/6178f248838a1ab1abed0f4ea48faad7_5.png)

这些标准相对简单明了。平台希望通过这些规则，鼓励创作者生产更适合手机端观看的高质量内容。

## 降权限流的内容禁区 🚫

与扶持相对，如果你的内容涉及以下禁区，则很可能被平台降权限流，甚至审核驳回。

以下是会导致降权限流的具体内容类型：

1.  **违规信息**：内容涉及吸烟、暴力、炫富（金钱）等不良导向。
2.  **外部引流**：视频中出现或引导用户至微信、微博、淘宝等站外联系方式。
3.  **商业标识**：长时间出现商品Logo、手机号、二维码等。二维码的出现时间**最长不要超过3秒**，但在审核严格时，即使出现1秒也可能被驳回。
4.  **硬性广告**：进行明显的营销广告、推销或引流卖货行为。

![](img/6178f248838a1ab1abed0f4ea48faad7_3.png)
![](img/6178f248838a1ab1abed0f4ea48faad7_5.png)

特别需要注意第15点：**无口播的拆箱或好物推荐**。这是许多新手容易踩坑的地方。

### 好物推荐类内容的注意事项

许多同学在运营好物推荐账号，希望通过分享好物来带货。但经常遇到视频小黄车被下架的情况，原因通常如下：

*   **场景一：直接标价**
    有同学在视频上方直接写上“九块九”。这属于直接涉及金钱交易的硬广行为，小黄车必然无法通过。
*   **场景二：纯展示无口播**
    有同学仅用背景音乐展示玩具，全程无任何口播介绍。这种“哑剧”式的拆箱展示风险极高。

**正确的做法是采用“剧情类软广告”或加入口播介绍。** 例如，介绍一款纸巾时，你可以说：“这款纸巾韧性很好，沾水后也不易破，可以反复使用。” 通过真实的使用体验进行介绍，而非单纯展示。对于玩具，可以口播：“我儿子玩这个玩具，专注地玩了四个小时。” 这样简单的口播就能有效降低被判定为硬广的风险。

![](img/6178f248838a1ab1abed0f4ea48faad7_7.png)
![](img/6178f248838a1ab1abed0f4ea48faad7_8.png)

![](img/6178f248838a1ab1abed0f4ea48faad7_7.png)

**核心要点**：千万不要不说话，直接展示产品、拆箱或推荐，这非常危险。其他如直接口播广告、念联系方式等更明显的违规行为，在抖音平台是行不通的。

![](img/6178f248838a1ab1abed0f4ea48faad7_8.png)

本节课中我们一起学习了抖音平台的扶持规则与限流禁区。掌握“**做什么能获得流量**”和“**做什么会失去流量**”同样重要。请务必在创作时避开上述禁区，并按照优质内容的标准来制作视频，这样才能让你的账号健康、快速地成长。关于内容优化的更多细节，我们将在后续课程中深入探讨。
