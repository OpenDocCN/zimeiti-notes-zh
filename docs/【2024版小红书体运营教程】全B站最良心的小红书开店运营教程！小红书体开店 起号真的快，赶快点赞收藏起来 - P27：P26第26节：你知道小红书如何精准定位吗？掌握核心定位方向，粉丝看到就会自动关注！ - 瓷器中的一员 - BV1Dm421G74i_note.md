# 小红书运营教程：P27：小红书账号类型与官方广告投放解析

在本节课中，我们将学习小红书运营中的两个核心问题：不同类型的账号应如何运用，以及如何利用小红书官方的广告系统进行推广。我们将理清自建账号运营与官方广告投放之间的关系，帮助你制定更有效的运营策略。

## 账号类型解析

上一节我们探讨了账号定位，本节中我们来看看小红书平台上的账号具体分为哪几类，以及它们各自的特点和用途。

小红书账号主要分为三类：企业号、个人号和引流号。

以下是三类账号的详细说明：

*   **企业号**：主要用于树立官方品牌形象。企业号拥有官方认证标志，可以展示品牌名称、地址等信息，并享有平台赋予的特权，便于开展官方业务。
*   **个人号**：这是目前大多数用户运营的账号类型。个人号运营灵活，可以扮演品牌员工、产品消费者等多种角色。它有助于塑造个人品牌特色，也适合品牌相关人物或员工出镜进行内容创作。
*   **引流号**：这类账号在之前的课程中也有提及，俗称“炮灰号”。引流号可以创建多个，其作用相对次要，通常用于测试内容、参与活动或在紧急情况下作为备用小号使用，一般没有精细的个性规划。

通常，品牌的推广和引流工作，会通过企业号、个人号或引流号发布笔记来完成。

## 企业号矩阵运营建议

了解了账号分类后，我们来看看如何协同使用这些账号，特别是如何围绕企业号构建一个有效的运营矩阵。

以下是打造企业号矩阵的三点建议：

![](img/8227d674c308235a336274ed2e1f85a4_1.png)

![](img/8227d674c308235a336274ed2e1f85a4_3.png)

1.  **积极参与品牌活动**：对于企业号发起的试用、团购等活动，可以动用多个个人号或引流号去参与，增加活动热度与真实感。
2.  **个人号展示品牌属性**：在个人号的内容中，应多次暗示品牌的定位。例如，如果品牌定位于城市年轻人，那么个人号的内容风格和话题也应围绕这一群体展开。
3.  **控制营销内容比例**：个人号在发布品牌产品相关内容时，需注意频率。建议遵循 **“三七分”** 原则，即30%的内容用于产品营销，70%的内容用于种草分享或知识科普，这样的内容比例更为健康。
4.  **巧用引流号评论**：当企业号需要引导用户时，可以安排引流号在其笔记评论区进行互动或引导，这是一种比较自然的引流方式。

## 官方广告投放介绍

![](img/8227d674c308235a336274ed2e1f85a4_7.png)

除了运营自有账号矩阵，另一种重要的推广方式是借助小红书平台官方的广告系统。接下来，我们将介绍小红书官方广告的基本形式。

![](img/8227d674c308235a336274ed2e1f85a4_9.png)

![](img/8227d674c308235a336274ed2e1f85a4_11.png)

小红书官方广告主要指 **小红书CPC广告**，即按点击付费的竞价广告。其主要分为两大类：

![](img/8227d674c308235a336274ed2e1f85a4_13.png)

*   **信息流广告**：当用户浏览小红书首页推荐内容时，系统会不定期间插入展示的广告。其核心是 **`广告展示在信息流中`**。
*   **搜索广告**：当用户通过关键词搜索时，在搜索结果中出现的广告。其核心是 **`广告匹配用户搜索意图`**。例如，用户搜索“考研资料”，系统可能推荐相关的学习工具或营养品。这与平台的推荐算法密切相关。

搜索竞价广告的代表是百度，小红书借鉴了其模式，通过对关键词出价来竞争不同的广告排名位置。目前，信息流广告位常见的有第3位、第13位、第23位等。

## 如何申请官方广告

如果你想尝试官方广告投放，需要先完成企业号认证，然后通过官方渠道申请。

以下是申请官方广告的简要步骤：

1.  **认证企业号**：访问小红书官网，在“业务合作”中找到企业号入驻流程。按照指引提交相关资质并支付审核费用，即可将普通账号认证为企业号。
2.  **申请广告投放**：拥有企业号后，在官网的“业务合作”板块找到“推广合作”入口，即可进入广告投放系统，设置信息流或搜索广告。

![](img/8227d674c308235a336274ed2e1f85a4_15.png)

关于广告投放的常见问题，官方帮助中心有详细解答。需要注意的是，官方广告投放需要一定的资金预算，费用相对较高。

## 总结

本节课中我们一起学习了小红书运营的两大板块：账号矩阵运营与官方广告投放。我们明确了**企业号**、**个人号**和**引流号**的不同角色与协同方法，并介绍了 **小红书CPC竞价广告** 的两种主要形式：**信息流广告** 与 **搜索广告**。对于资金充足的品牌，建议可以尝试官方广告投放以获取更直接的平台流量。在实际运营中，将自有账号的内容运营与官方广告投放相结合，往往能取得更好的效果。
