# 抖音运营指南：P13：短视频文案高效获取方法

在本节课中，我们将学习如何高效获取短视频文案，解决创作初期“没东西可写”的困境。我们将介绍一种通过借鉴已有爆款内容来快速启动账号的方法，并探讨其后续的进阶方向。

---

群里一位刚做短视频一个月的伙伴决定放弃，原因是每天需要撰写800字的文案稿件，却收获甚少观看量，感到心力交瘁。

他询问是否存在一种输出方式，能让自己在独立运营时，既能保持作品数量又能保证质量。方法确实存在，但手段本身存在争议，通常不推荐使用。然而，确实有一位大V依靠此方法积累了上千万粉丝。

![](img/162c9685c42edd6adcda6ee992ae102e_3.png)

该方法的核心可以用一句话概括：**将任何图文平台上出现过的“10万+”爆款内容，用短视频的形式重新演绎一遍。**

其原理在于，一篇能达到“10万+”阅读量的文章，意味着它已至少获得了十万人的认同，是属于能引发广泛情绪共鸣的高质量内容。

例如，一篇讲述“拼多多砍价”经历的文章引发了普通用户的强烈共鸣和愤怒情绪，让人觉得终于有人替自己发声。如果你想蹭这个热点，但自己又无法深入阐述，便可使用此方法。

具体操作是，前往公众号、知乎等平台搜索相关话题，找到那些高赞、高阅读量的爆款文章或回答。然后，你无需原创，只需将这些文字内容，用自己的语言进行口播录制，转化为短视频即可。

这个思路适用于任何你感兴趣的领域。假设你认为健身减肥是一个有潜力的赛道，想销售轻卡零食或代餐产品，但自身却是门外汉。

此时，你同样可以运用此方法。持续寻找并转化该领域的爆款文案，通过视频进行讲述。坚持三个月，你看得多了，讲得多了，粉丝自然会积累起来，你也会逐渐成为该领域的“专家”。

![](img/162c9685c42edd6adcda6ee992ae102e_5.png)

这是第一个阶段：**通过“转换输出”的方式帮助自己快速启动账号**。因为曾经火爆过的内容，其话题性和共鸣点已经过市场验证，有很大概率再次引爆流量。

然而，进入第二个阶段后，就不能再单纯依赖这种方法了。如果长期被观众发现你没有自己的观点，始终在复述他人的内容，将对账号的信誉和长期发展造成损害。
