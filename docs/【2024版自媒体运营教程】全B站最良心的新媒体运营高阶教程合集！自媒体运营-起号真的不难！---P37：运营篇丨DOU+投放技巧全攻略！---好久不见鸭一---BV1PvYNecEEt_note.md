## 第一阶段：1000粉丝以内

![](img/50e158996592f2bbd6f05775f0eb905a_2.png)

上一节我们介绍了课程的整体框架，本节中我们来看看账号初期的投放策略。

在账号粉丝数低于1000时，建议采用“混搭投放”策略，即在视频获得自然流量后，再使用DOU+进行助推。

![](img/50e158996592f2bbd6f05775f0eb905a_4.png)

以下是具体的操作步骤：

1.  **第一步：智能推荐投放**
    *   首先，选择一条视频，在自然流量跑完后，为其投放100元的DOU+。
    *   投放目标选择“点赞评论”。
    *   这一步的目的是让系统为你的视频数据“美化”，即通过付费获得更多初始互动。数据表现更好的视频，能吸引更多用户停留观看。

![](img/50e158996592f2bbd6f05775f0eb905a_6.png)

2.  **第二步：达人相似投放**
    *   完成第一步后，为同一条视频再投放100元DOU+。
    *   这次选择“达人相似”投放，并手动选择5个对标账号。
    *   选择对标账号的标准是：**最好选择短期内粉丝刚增长到1万左右的账号**。这类账号的粉丝粘性相对不强，对新号内容接受度更高，因此对新号更友好。

## 第二阶段：3000粉丝以内

![](img/50e158996592f2bbd6f05775f0eb905a_8.png)

了解了初期的策略后，当粉丝增长到3000以内时，我们的目标应转向获取更精准的粉丝。

此阶段采用“定向投放”策略。在视频自然流量结束后，直接投放200元的“达人相似”。对标账号的选择标准与第一阶段相同。

但操作不止于此，你需要根据账号定位，添加更精细的定向条件：

*   **公式：精准投放 = 达人相似 + 受众画像（年龄/性别/地域）**
*   例如，如果你是家居好物账号，主要受众可能是**30-40岁的女性**。
*   如果你是同城号，则需要额外添加**地域**限制。

这样操作能让吸引的粉丝更加精准，有助于平台为你的账号快速打上标签，从而加速向1万粉丝冲击。

## 第三阶段：1万粉丝以上

当粉丝突破1万后，账号通常已打上稳定标签，会获得一个基础的稳定播放量（几千到几万不等）。此时，投放的核心目标发生变化。

![](img/50e158996592f2bbd6f05775f0eb905a_12.png)

此阶段建议采用“智能投放”。在视频自然流量跑完后，从表现最好的视频中选出一条，投放200元，目标选择“点赞评论”。

为什么不继续投“达人相似”？因为账号已有稳定基本盘，破播放不再是核心问题。此时投放的意义在于**冲击热门，通过热门事件来增粉**，而不是继续“洗”同行粉丝。

因此，本阶段的建议是：**可投可不投，应更专注于打磨作品质量**。

**代码式决策逻辑：**
```python
if 作品质量极高 and 自然流量未达预期:
    投放DOU+进行“补考”
else:
    继续优化内容
```
如果某条作品你自认为质量上乘、有望热门，但自然流量结果未达预期，可以尝试投放DOU+，给它一个“补考”的机会，或许能助推其上热门。

## 总结

![](img/50e158996592f2bbd6f05775f0eb905a_14.png)

本节课中我们一起学习了DOU+投放的三个核心阶段：
1.  **千粉以内**：混搭投放（智能推荐+达人相似），目的是美化数据，友好起号。
2.  **三千粉以内**：定向投放（达人相似+精准定向），目的是加速打标签，获取精准粉丝。
3.  **万粉以上**：智能投放，目的是助推优质内容冲击热门，实现爆发式增长。

记住，投放是放大器，核心永远是优质的内容。根据不同阶段的目标，灵活运用上述策略，才能让DOU+发挥最大价值。
