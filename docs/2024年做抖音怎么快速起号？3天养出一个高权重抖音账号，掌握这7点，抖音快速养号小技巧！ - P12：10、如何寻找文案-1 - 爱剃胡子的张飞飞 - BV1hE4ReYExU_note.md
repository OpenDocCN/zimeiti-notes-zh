## 免费配音功能

![](img/66dbb6928702a5d2893c51ac31b46bae_17.png)

对于担心自己声音不好听的同学，淘金阁还提供了免费的在线配音功能。

![](img/66dbb6928702a5d2893c51ac31b46bae_19.png)

点击页面右上角的“小工具”图标，然后选择“配音在线生成”功能。

```
（小工具与配音功能入口图片）
```

这个功能完全免费，且提供多种音色选择，包括各类男声、女声、方言、童声，甚至外语和影视解说风格。

![](img/66dbb6928702a5d2893c51ac31b46bae_21.png)

```
（配音音色选择列表图片）
```

![](img/66dbb6928702a5d2893c51ac31b46bae_23.png)

使用方法很简单：
1.  在左侧“配音文本”框内输入或粘贴你的文案。
2.  在右侧选择喜欢的音色（如“东北话”）。
3.  调整音量、语速、语调等参数。
4.  点击“立即生成”。

```
（配音参数设置与生成界面图片）
```

![](img/66dbb6928702a5d2893c51ac31b46bae_25.png)

生成后，在“配音生成列表”中即可下载音频文件到手机，供后续剪辑使用。

