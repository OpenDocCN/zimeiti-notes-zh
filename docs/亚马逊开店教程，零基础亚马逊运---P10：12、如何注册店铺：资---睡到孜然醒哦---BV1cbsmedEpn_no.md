# 亚马逊开店教程：P10：12、如何注册店铺：资料准备 📋

![](img/2393417216bf55e0a0cffded86b8fe32_0.png)

在本节课中，我们将学习注册亚马逊店铺前需要准备的所有资料。了解并备齐这些资料是成功开店的第一步。

上一节我们介绍了如何选择亚马逊站点，本节中我们来看看注册店铺需要准备哪些具体材料。

## 资料清单概述

注册亚马逊店铺需要准备六项核心资料。以下是每一项资料的详细说明和要求。

### 1. 营业执照彩色扫描件

第一份需要准备的资料是营业执照的彩色扫描件，即营业执照的照片。

*   **执照类型**：可以是企业性质的执照，也可以是个体工商户性质的执照。
*   **办理方式**：可在当地工商局免费办理。
*   **获取途径**：若无执照，可借用家人或朋友的执照。例如，开超市或服装店的执照均可借用。若实在没有，也可寻找代办机构或联系课程助理老师协助办理，费用通常在几百到一千元左右。
*   **执照要求**：
    *   **注册资本与年限**：无硬性要求。注册资本是1元还是100万元均可，执照是刚注册的还是已注册多年的也没有关系。
    *   **经营范围**：只需带有销售性质即可，例如“零售”、“批发”等。
    *   **重要提醒**：**切勿**在经营范围中添加“进出口贸易”。添加此项后，必须去相关部门办理进出口手续，属于画蛇添足。
    *   **经营范围与销售产品**：执照的经营范围与你在亚马逊店铺中实际销售的产品**不需要一致**。例如，使用超市的执照注册的店铺，完全可以销售服装。

### 2. 法人身份证正反面照片

第二份需要指明的资料是法人身份证的正反面照片。

*   **核心关联**：**营业执照和对应的法人身份证必须是同一个人的**。
*   **人员要求**：除了上述两份资料必须一致外，后续其他资料可以与法人不是同一个人。例如，你借用朋友的营业执照和身份证注册，其余资料（如信用卡、收款账户）绑定你自己的信息是完全可行的。

### 3. 付款信用卡

第三份需要准备的资料是用于付款的信用卡。

*   **唯一作用**：此信用卡的唯一作用是**支付亚马逊的月租**（目前为39.99美元/月）。
*   **卡片要求**：需要**全币种信用卡**，首选带有 **Visa** 或 **Mastercard** 标识的卡片。
    *   `卡片示例：卡面上有 Visa 或 Mastercard 图标`
*   **办理银行**：国内各大银行（如工商银行、农业银行、建设银行、浦发银行、广发银行、中信银行等）均可办理。
*   **解决方案**：若没有信用卡，可以借用他人的，或花费约100元购买一张虚拟信用卡。只需每月向虚拟卡中充值足够扣缴月租的金额即可。
*   **扣款逻辑**：首次扣租会从该信用卡扣除。当店铺产生营业额后，后续月租会优先从店铺账户余额中扣除，不会持续扣信用卡。

### 4. 海外收款账户

第四份需要准备的资料是海外收款账户。

![](img/2393417216bf55e0a0cffded86b8fe32_2.png)

*   **功能类比**：它相当于**国际版的支付宝**。
*   **作用**：用于将亚马逊店铺的销售所得提现到你的国内银行储蓄卡中。正如淘宝收入通过支付宝提现，亚马逊收入则通过此账户提现。
*   **获取方式**：可通过服务商提供的链接免费注册，流程类似于注册支付宝，只需绑定个人身份证和国内银行卡，几分钟即可完成。

### 5. 电话号码

第五个需要准备的资料是电话号码。

*   **要求**：使用**国内的手机号码**即可。

### 6. 邮箱号码

第六个需要准备的资料是邮箱号码。

*   **要求**：可以使用163邮箱、126邮箱或QQ邮箱等。
*   **个人推荐**：建议使用**163邮箱**。
