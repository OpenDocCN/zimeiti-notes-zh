## 概述
在本节课中，我们将要学习小红书店铺运营中至关重要的“人群定位”环节。上一节我们介绍了店铺定位的基础，本节中我们来看看如何精准地定位你的目标客户群体，这是实现高效引流和转化的关键。

## 人群定位详解
小红书的店铺人群定位与账号定位有相似之处，但更侧重于店铺整体的目标客户画像。理解并做好人群定位，是后续所有店铺操作的基础。

### 1. 个人属性
用户的个人属性是定位的起点，包括性别、年龄范围和喜欢的内容。这与之前选择店铺类型时的逻辑相似。

以下是个人属性的核心要素：
*   **性别**：根据产品目标客户确定。
*   **年龄范围**：根据产品特性，通常聚焦2-3个年龄段。
*   **喜欢的内容**：用户在小红书上的兴趣偏好。

**核心关系**：`店铺人群属性` 需与 `账号内容属性` 相匹配。若两者脱节，引流的用户可能与你的产品不契合，影响转化效果。

### 2. 消费能力
用户的消费能力指其在小红书平台上的平均月度消费水平。平台会通过用户的购买历史、浏览商品的价格区间等行为数据，隐性地评估其消费能力。

**定价策略**：`产品定价` 应瞄准目标人群的 `消费能力区间`。定价合理，才能有效吸引具备相应购买力的用户。

### 3. 购买类型
小红书系统会分析用户的购买行为，并通常为其推荐最多三种类型的商品。大类目可包括日用品、生活用品、户外用品、家具、玩具等。

**平台逻辑**：由于“千人千面”的推荐机制，用户近期浏览和购买什么，系统就会持续推荐相关内容。这种标签通常需要数天才能形成或改变。

### 4. 购买地域
系统会根据用户的收货地址地域信息，分析不同地区的消费偏好（例如，北上广深可能对服装、食品有特定偏好）。

**地域匹配**：`店铺产品属性` 若符合某一地域（如北京）用户的普遍偏好，系统在推荐时，将该店铺展示给该地域用户的 `概率` 就会增加。公式可理解为：`推荐概率 = f(产品属性， 地域用户偏好)`。

![](img/807d1b204380bcf1e7b1c807087b6212_8.png)

### 5. 购买习惯
用户的购买习惯也有不同模式，例如“货比三家”型（长时间浏览比较后购买）。系统根据用户的搜索和浏览兴趣进行判断。

![](img/807d1b204380bcf1e7b1c807087b6212_10.png)

**注意事项**：如果用户在你的店铺浏览却最终在同行店铺成交，系统后续可能会更多推荐那家同行店铺。这是一种潜在的流量竞争影响，需要留意。

![](img/807d1b204380bcf1e7b1c807087b6212_12.png)

## 定位策略与应用总结
本节课中我们一起学习了人群定位的五个维度：个人属性、消费能力、购买类型、购买地域和购买习惯。

以下是人群定位的综合应用路径：
1.  **店铺定位**：确定经营方向。
2.  **选品**：选择具体类目和产品。
3.  **定价**：初步设定价格区间。
4.  **人群定位**：综合运用上述五个维度，刻画目标客户。

**给新手的建议**：前期若以熟悉平台节奏为主，可暂不深入进行精准人群限定（尤其是后三点）。因为精准限定虽会提升流量精准度，但也会导致初始曝光量减少。你可以先完成前几步，待店铺运营稳定后，再通过“价格定位”等策略，反向优化和精确人群标签，以实现长期的数据提升。
