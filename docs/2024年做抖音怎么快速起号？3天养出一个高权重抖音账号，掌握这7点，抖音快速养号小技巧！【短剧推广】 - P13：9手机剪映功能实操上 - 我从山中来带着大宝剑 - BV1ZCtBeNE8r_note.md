## 视频去水印 🚫

![](img/0bf01511aedb33d69caa7225ed2585eb_19.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_21.png)

上一节我们介绍了剪映的基本功能，本节中我们来看看如何去除视频中的水印。发布到任何平台的作品都不能带有其他平台的标识水印，否则会影响推荐。

![](img/0bf01511aedb33d69caa7225ed2585eb_27.png)

去除水印主要有两种方法：

![](img/0bf01511aedb33d69caa7225ed2585eb_31.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_35.png)

### 方法一：解析法（使用小程序）

![](img/0bf01511aedb33d69caa7225ed2585eb_39.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_41.png)

这种方法通过技术手段直接去除平台水印，不会影响视频画面。

以下是使用“皮皮去水印”小程序的操作步骤：
1.  **复制视频链接**：在抖音等App中，点击视频分享按钮，找到并点击“复制链接”。
2.  **打开小程序**：在微信中下拉聊天界面，搜索并打开“皮皮去水印”小程序。
3.  **解析下载**：将复制的链接粘贴到小程序输入框，点击“去水印”，解析成功后下载视频即可。

![](img/0bf01511aedb33d69caa7225ed2585eb_43.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_45.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_47.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_49.png)

**注意**：皮皮去水印主要支持抖音、快手、B站等平台的**短视频**水印去除。对于爱奇艺、腾讯视频等长视频平台的水印，或视频创作者自行添加的水印，此方法无效。

![](img/0bf01511aedb33d69caa7225ed2585eb_51.png)

### 方法二：物理法（使用剪映）

![](img/0bf01511aedb33d69caa7225ed2585eb_53.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_55.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_57.png)

当无法用解析法去除水印时，可使用裁剪或缩放功能，但这会影响画面。

![](img/0bf01511aedb33d69caa7225ed2585eb_59.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_61.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_63.png)

**在剪映中操作**：
1.  **导入视频**：将带水印的视频导入剪映。
2.  **裁剪去水印**：
    *   点击视频轨道，选择底部工具栏的`剪辑` -> `编辑` -> `裁剪`。
    *   拖动裁剪框，将水印部分排除在框外，点击对勾确认。
3.  **缩放去水印**：
    *   选中视频轨道，画面四周会出现红框。
    *   双指捏合放大画面，使水印移出画面区域；或缩小画面以舍弃边缘部分。

![](img/0bf01511aedb33d69caa7225ed2585eb_67.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_71.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_73.png)

语音片头是指在视频开头，用画外音介绍主题的简短片段，通常搭配黑色背景和文字。

![](img/0bf01511aedb33d69caa7225ed2585eb_103.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_105.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_107.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_109.png)

以下是制作语音片头的完整流程：
1.  **准备黑底素材**：在剪映点击`开始创作` -> `素材库`，在“黑白场”分类中选择第二个“黑底”素材，下载并添加到项目。
2.  **调整视频比例**：在底部工具栏找到`比例`，选择`9:16`（抖音竖屏比例）。
3.  **输入文案并生成语音**：
    *   点击底部`文字` -> `新建文本`，输入你的片头文案（如“大年三十的早晨”）。
    *   输入完成后，点击右侧`文本朗读`。
    *   在弹出界面选择你喜欢的声音（如“小姐姐”），点击对勾，系统会自动生成语音音频。

一个简单的语音片头就制作完成了。你可以播放预览效果。

## 制作动态片尾 🎬

![](img/0bf01511aedb33d69caa7225ed2585eb_127.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_129.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_131.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_133.png)

一个专业的片尾可以引导观众关注，提升账号形象。我们将制作一个常见的“圆圈聚焦头像”式片尾。

![](img/0bf01511aedb33d69caa7225ed2585eb_135.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_137.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_139.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_141.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_143.png)

**准备材料**：
*   片尾圆圈素材（需向指导老师索取）。
*   一张个人头像或图片。

![](img/0bf01511aedb33d69caa7225ed2585eb_145.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_147.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_149.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_151.png)

**详细操作步骤**：
1.  **导入圆圈素材**：将片尾素材视频导入剪映。
2.  **第一次裁剪（留白圈）**：
    *   点击素材轨道，选择`剪辑` -> `编辑` -> `裁剪`。
    *   调整裁剪框，只留下中间的**白色圆圈**部分，确认。
3.  **第一次画中画（添加头像）**：
    *   返回一级工具栏，点击`画中画` -> `新增画中画`，导入你的头像图片。
    *   选中头像轨道，在工具栏找到`混合模式`，选择`变暗`。
    *   此时头像会出现在白圈内，调整头像的大小和位置使其匹配。
4.  **第二次画中画（添加黑圈）**：
    *   再次点击`新增画中画`，第二次导入同一个片尾圆圈素材。
    *   适当放大该素材，使其与底层画面边缘对齐。
    *   选中该轨道，点击`剪辑` -> `编辑` -> `裁剪`，这次只留下**黑色圆圈**部分，确认。
5.  **混合模式对齐**：
    *   选中第二次导入的素材轨道，点击`混合模式`，选择`绿色`。此时黑白两个圆圈应对齐，形成头像被圈出的效果。
6.  **添加引导文案**：
    *   点击`文字` -> `新建文本`，输入“谢谢关注”等文案。
    *   同样可以使用`文本朗读`功能为文案配音。

![](img/0bf01511aedb33d69caa7225ed2585eb_153.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_155.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_157.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_159.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_161.png)

> **操作核心**：本片尾的关键在于两次使用`画中画`和`混合模式`（变暗、绿色）的配合，以及对同一素材进行两次不同区域的`裁剪`。

![](img/0bf01511aedb33d69caa7225ed2585eb_165.png)

## 设置重点重播 🔁

重点重播是指将视频中的精彩片段重复播放一次，常用于强调搞笑或关键情节，在综艺节目中很常见。

![](img/0bf01511aedb33d69caa7225ed2585eb_171.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_173.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_175.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_177.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_179.png)

**实现步骤如下**：
1.  **分割精彩片段**：导入视频后，找到需要重播的片段。点击`剪辑` -> `分割`，在该片段的开始和结束处各切一刀，将其独立分割出来。
2.  **复制片段**：点击被分割出来的片段，在底部工具栏找到`复制`，这样会得到前后两段相同的视频。
3.  **处理复制的片段**：选中后面复制出来的那段视频，进行差异化处理：
    *   **添加滤镜**：点击`滤镜`，在“风格化”分类中选择“牛皮纸”或其他黑白滤镜，使重播片段颜色不同。
    *   **调整变速**：点击`变速`，将速度调慢（如0.6倍），实现慢放效果。如果原片段有声音，请勾选`声音变调`以保持音调自然。
4.  **预览效果**：播放视频，正常播放后会接一段经过滤镜和慢放处理的重复片段，重点得以突出。

![](img/0bf01511aedb33d69caa7225ed2585eb_181.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_183.png)

## 添加转场特效 ✨

转场特效用于连接两个不同的视频片段，使场景切换更自然，不生硬。对于二次剪辑创作者，合理添加转场也是提升视频原创度的手段之一。

![](img/0bf01511aedb33d69caa7225ed2585eb_185.png)

**重要特性**：转场特效只能添加在视频的**分割点**处。

![](img/0bf01511aedb33d69caa7225ed2585eb_187.png)

![](img/0bf01511aedb33d69caa7225ed2585eb_189.png)

**添加方法**：
1.  在时间轴上，找到两个视频片段之间的**白色小方块**（分割标志）。
2.  直接点击这个白色小方块，即可打开转场特效库。
3.  选择你喜欢的转场效果（如“叠化”、“闪光灯”等）。
4.  在下方调整转场的持续时间（通常建议调短，如0.1秒至0.5秒），点击对勾确认。

![](img/0bf01511aedb33d69caa7225ed2585eb_191.png)

> **应用建议**：为了不影响观看体验，二次剪辑视频建议使用最自然的`叠化`转场，并将时长调到最短。夸张的转场仅用于特定风格需求。

![](img/0bf01511aedb33d69caa7225ed2585eb_193.png)
