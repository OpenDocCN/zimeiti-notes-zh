# 美客多选品教程：方法12：基于品牌上新情况选品 📈

在本节课中，我们将学习一种适合供应链能力较强卖家的选品方法。该方法的核心是监控竞争对手品牌的新品发布情况，并快速跟进类似产品，从而利用品牌效应和价格优势抢占市场。

![](img/64a8c1cfae9322fdbfa31a7afb7902ff_0.png)

## 方法概述与适用场景

上一节我们介绍了多种选品思路，本节中我们来看看如何基于品牌动态进行选品。这种方法特别适合供应链响应速度快的卖家。其逻辑是：跟随有影响力的品牌上新，推出类似产品，通过设定略低的价格形成竞争力，从而快速切入市场。

**核心公式**：**快速跟进 + 价格优势 = 市场机会**

在操作中，必须注意一个关键前提：确保目标商品的卖家数量大于1。这能有效减少因版权、专利等问题导致的跟卖限制和侵权风险。

## 操作步骤详解

那么，如何具体查看品牌卖家的上新情况呢？以下是详细的操作流程。

![](img/64a8c1cfae9322fdbfa31a7afb7902ff_2.png)

进入“大麦数据”平台的“商品全类目商品”页面。

以下是具体的筛选与跟踪步骤：
1.  在搜索框中输入你想要关注的竞争对手品牌名称。
2.  设置“跟买数”筛选条件，要求大于1且小于3。这一步是为了找到已有一定关注度但竞争尚未白热化的商品。
3.  利用平台的“竞品跟踪”功能，持续关注该品牌的新品发布动态。

![](img/64a8c1cfae9322fdbfa31a7afb7902ff_3.png)

通过以上步骤，你可以系统性地监控竞品动向，为快速跟进选品提供数据支持。

![](img/64a8c1cfae9322fdbfa31a7afb7902ff_4.png)
