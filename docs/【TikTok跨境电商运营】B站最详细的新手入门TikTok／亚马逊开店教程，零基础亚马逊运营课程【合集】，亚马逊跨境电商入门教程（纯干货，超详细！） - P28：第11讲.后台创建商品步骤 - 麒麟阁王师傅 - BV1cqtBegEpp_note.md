# TikTok跨境电商运营：P28：第11讲.后台创建商品步骤

在本节课中，我们将学习如何在亚马逊卖家后台手动创建商品。我们将详细介绍创建过程中的必填项、常见问题以及如何设置商品变体，确保您能够成功上架第一个产品。

上一节我们介绍了商品上架前的准备工作，本节中我们来看看具体的后台操作步骤。

## 创建商品核心信息

首先，登录卖家后台，将页面语言切换为中文以便操作。在“库存”菜单下，选择“添加新商品”。

以下是创建商品时需要填写的核心信息：

1.  **商品编码**：选择 **UPC** 或 **EAN** 等编码类型，并填入您购买的有效编码。
2.  **商品名称**：格式通常为 **品牌名 + 产品核心名称**。例如：`Anker PowerCore 10000mAh 移动电源`。
3.  **品牌**：如果尚未完成品牌备案，可以选择“无品牌”或填写通用品牌标识，如 **N/A**、**N/B** 或 **Generic**。
4.  **制造商**：这里可以填写您计划注册的品牌名称。这是一个实用技巧：在品牌备案完成前，先在此处填入已通过核名、计划注册的商标名，未来修改品牌时会更容易。
5.  **商品属性**：根据产品实际情况填写，如“是否有保质期”通常选“否”，“包装数量”填`1`，“重量”按实际克数填写。不相关的属性（如“热敏性”）可直接忽略。

## 处理品牌与报价设置

关于品牌设置，需要特别注意其历史变化。过去修改品牌相对容易，但现在即使联系客服并提供使用证据，过程也较为复杂。因此，提前在“制造商”栏填写计划品牌名是推荐做法。

接下来是报价部分：

![](img/d598ec3a8f4a073504e0aa301303ec3e_1.png)

1.  **价格与数量**：设置您的商品价格（如`10`美元）和初始库存数量（如`10`个）。
2.  **配送渠道**：初期建议选择“卖家自行配送”。产品测试稳定后，可再手动转换为“亚马逊配送”（FBA）。这仅是个人操作习惯，您也可以直接选择FBA。

![](img/d598ec3a8f4a073504e0aa301303ec3e_3.png)

## 上传商品图片

![](img/d598ec3a8f4a073504e0aa301303ec3e_5.png)

![](img/d598ec3a8f4a073504e0aa301303ec3e_6.png)

![](img/d598ec3a8f4a073504e0aa301303ec3e_8.png)

图片是影响转化率的关键因素。通常需要上传至少1张主图和8张辅图，共**9张图片**。

![](img/d598ec3a8f4a073504e0aa301303ec3e_9.png)

![](img/d598ec3a8f4a073504e0aa301303ec3e_10.png)

![](img/d598ec3a8f4a073504e0aa301303ec3e_11.png)

如果您的商品有变体（如不同颜色），需要特别注意：系统会为每个变体自动生成一个展示小图。为了保持所有变体列表页面的美观与统一，建议在创建变体时，为每个子体单独上传格式统一的主图。

## 解决常见报错

在填写过程中，可能会遇到报错提示。例如，系统提示“使用的UPC与尝试提供的产品不匹配”。这通常**不是UPC码本身的问题**，而是**品牌栏填写有误**导致的。

**解决方法**：将品牌栏从“通用”修改为 **N/A** 或 **N/B** 等选项，通常即可解决此报错，继续完成保存。

## 创建商品变体

如果您的商品有不同颜色、尺寸等变体，需要在创建时进行设置。

![](img/d598ec3a8f4a073504e0aa301303ec3e_13.png)

以下是创建变体的步骤：

![](img/d598ec3a8f4a073504e0aa301303ec3e_14.png)

![](img/d598ec3a8f4a073504e0aa301303ec3e_15.png)

1.  在“重要信息”部分，选择“是，此商品有变体”。
2.  根据竞争对手或产品特性，选择变体主题，如“颜色-尺寸”。
3.  在出现的表格中，为每个变体组合（如“黑色-XL”、“灰色-L”）填写详细信息。
4.  **卖家SKU**：为父体和每个子体设置清晰的SKU以便管理。例如：
    *   父体SKU：`Lashes-Set01`
    *   子体SKU：`Lashes-Set01-Black-XL`
5.  为每个子体单独填写UPC、价格、数量等信息，确保所有必填项完整。

![](img/d598ec3a8f4a073504e0aa301303ec3e_17.png)

## 商品上架与后续优化

商品信息填写完毕后，点击“保存并完成”。商品可能不会立即上架，而是进入“草稿”或“禁止显示”状态。

![](img/d598ec3a8f4a073504e0aa301303ec3e_19.png)

您需要在“库存 > 管理库存”或“修复商品”页面找到该商品，根据提示补全信息，常见问题包括：

![](img/d598ec3a8f4a073504e0aa301303ec3e_21.png)

![](img/d598ec3a8f4a073504e0aa301303ec3e_22.png)

*   **长描述缺失**：补充商品要点和描述。
*   **图片问题**：上传符合要求的主图。
*   **商品质量信息不完整**：在“商品信息控制面板”中，亚马逊会给出优化建议（如补充材质、保养说明等）。补全这些信息有助于提升商品曝光和转化率。

![](img/d598ec3a8f4a073504e0aa301303ec3e_24.png)

当所有问题修复后，商品状态变为“在售”，即表示上架成功。

![](img/d598ec3a8f4a073504e0aa301303ec3e_26.png)

本节课中我们一起学习了在亚马逊后台手动创建商品的完整流程，包括填写核心信息、设置报价、上传图片、处理报错、创建变体以及完成上架后的优化。掌握这些步骤是开启亚马逊销售的第一步。
