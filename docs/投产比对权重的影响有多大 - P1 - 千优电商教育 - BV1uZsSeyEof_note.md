# 电商运营入门：P1：投产比与链接权重的关系 🔍

在本节课中，我们将要学习一个常见的认知误区：付费推广中的投产比（ROI）是否直接影响商品链接的权重。我们将澄清这个概念，并了解平台真正关注的指标是什么。

## 核心概念澄清

当我们进行付费推广时，实际跑出的投产比越高，是否意味着链接权重越高？答案是否定的。

昨天有一位朋友问我，能否在直通车里补一些单。我询问他的目的，他回答说是为了让投产比更高一些，他认为这样权重就会提高。这个观点是错误的。

## 投产比与平台的关系

首先，你的投产比高，可能代表你实现了盈利。然而，**投产比的高低与平台没有任何直接关系**。平台并不关心你的投产比具体是多少。

平台关心的是哪些指标呢？平台真正关心的是你的**点击率**、**转化率**和**销售额**。因为这些指标直接与平台的业绩和生态健康相关。唯独投产比，是商家自身的盈亏核算，与平台无关。

因此，你必须理解：**投产比并不能决定你的链接权重**。

## 本节课总结

本节课中，我们一起学习了投产比与链接权重的真实关系。我们明确了高投产比不代表高权重，平台算法关注的核心是**点击率**、**转化率**和**销售额**等直接影响平台生态的指标。理解这一点，有助于我们更精准地优化付费推广策略，避免在错误的指标上浪费精力。
