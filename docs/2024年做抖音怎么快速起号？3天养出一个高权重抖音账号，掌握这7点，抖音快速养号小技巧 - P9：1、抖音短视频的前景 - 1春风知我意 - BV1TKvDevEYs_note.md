# 抖音运营入门：P9：1：抖音短视频的前景 📈

![](img/6aecf9003f837c69afb2b6d6ecde5243_1.png)

在本节课中，我们将要学习抖音短视频平台的发展前景，分析其市场潜力与用户生态，并探讨为何它适合新手创作者起步。

![](img/6aecf9003f837c69afb2b6d6ecde5243_3.png)

了解抖音短视频的前景，需要先通过一张图片来认识其市场规模。

![](img/6aecf9003f837c69afb2b6d6ecde5243_1.png)

这张图片来自抖音创作者大会的数据报告。报告显示，有超过2200万人在抖音的合计收入超过了417亿元。抖音希望将这个数字翻一番，让创作者的总收入达到800亿元。

![](img/6aecf9003f837c69afb2b6d6ecde5243_3.png)

这代表了巨大的流量扶持。这里引入第一个知识点：抖音用户主要分为两类。第一类是**玩家**，他们单纯地使用抖音娱乐，没有考虑从中赚钱。第二类是**创作者**，即像本课程学员这样，希望通过制作抖音内容来获取收益的人。明确这两类身份对后续理解平台逻辑很重要。

那么，在众多短视频平台中，我们为何选择抖音？这类似于开设实体店铺选址。做短视频一定要选择一个流量大的平台。如果平台自身流量不足，就无法分配流量给创作者，创作者也就难以赚钱。

抖音的流量数据非常庞大。其日活跃用户超过8亿，月活跃用户更是达到了40亿人次以上。这是一个巨大的流量池。

流量大是否意味着竞争压力也大？并非如此。抖音的创作者数量约为5000万。这里的“创作者”定义广泛，并非特指网红达人。例如，一位邻居老太太用手机拍摄自己做饭的视频并上传到抖音，她也算创作者。因此，90%的竞争压力实际上来自于身边的普通用户。

接着，做抖音短视频是否需要昂贵的设备？答案是不需要。只需要一部智能手机和一个手机号，就可以注册抖音账号。抖音倡导全民通过短视频分享和记录生活，门槛很低。

那么，当前有哪些值得关注的风口？抖音官方曾提出计划，将帮助1000名优质的艺术创作者实现粉丝增长超过100万。有的学员可能会担心自己不会创作。这无需忧虑，正是因为不会，才需要学习。虚心向在该领域有经验的人请教并不可耻，这能帮助自己少走弯路。保持谦虚好学的态度至关重要。

市场空间巨大，竞争环境相对宽松，且内容创作容易。我们正处于大数据时代。平台算法会根据用户的喜好推荐内容。例如，喜欢看帅哥的用户，刷到的内容会多为有腹肌的帅哥；喜欢看美女的用户，则会看到更多长腿美女。这导致个人看到的内容可能比较片面。

对于内容方向感到迷茫的学员，无论是宝妈想分享宝宝日常、实体店主想展示产品引流，还是普通上班族想记录生活，都可以进行创作。在月活跃用户高达40亿的平台上，只要内容不触犯法律和道德底线，不是低俗内容，总会有一部分人喜欢你的作品。

为什么有的创作者觉得没人喜欢自己，粉丝只有几十个？这是因为喜欢你内容的那部分人，还没有通过算法刷到你的作品。他们无法看到，自然无法成为粉丝。

因此，对于新手小白而言，心情迫切想涨粉可以理解，但不要把涨粉作为首要目标。第一个目标应该是**提升视频的播放量**。先把流量做起来，粉丝自然会随之增长。

本节课中我们一起学习了抖音平台的巨大前景、用户分类、流量优势、低门槛特性以及对于新手而言的核心策略：优先关注播放量而非粉丝数。理解这些基础，是开启抖音运营的第一步。
