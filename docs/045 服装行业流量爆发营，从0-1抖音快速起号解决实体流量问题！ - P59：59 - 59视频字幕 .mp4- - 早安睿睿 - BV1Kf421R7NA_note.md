# 服装行业流量爆发营：045：剪映字幕与花字添加教程 🎬

![](img/78203549db8799afe516028aebbd27dc_0.png)

在本节课中，我们将学习如何使用剪映App为视频添加人物字幕和动态花字效果。字幕是提升视频信息传达和观看体验的重要元素，尤其适用于口播、剧情或带有背景音乐的视频。

## 新建文本与基础设置

上一节我们介绍了课程概述，本节中我们来看看如何创建和设置基础字幕。

![](img/78203549db8799afe516028aebbd27dc_2.png)

首先，在剪映中导入一段素材。点击底部工具栏的 **“文本”** 选项。文本菜单包含五个功能：**新建文本**、**文字模板**、**识别字幕**、**识别歌曲**和**添加贴纸**。我们先从第一个“新建文本”开始。

点击“新建文本”并确定后，屏幕上会出现一个文本框。点击“编辑”即可对字幕内容进行详细设置。

以下是文本编辑的主要功能：

*   **字体**：此处可以修改字幕的字体。App内置了多种字体，您需要根据视频风格选择合适的字体。
*   **样式**：第一行是App预设的文字样式模板，可直接点击应用。第二行第一个按钮用于设置**文字颜色**，第二个是**描边**，第三个是**背景**（为文字添加底色），第四个是**阴影**（可为阴影设置不同颜色和参数）。
*   **排列**：此功能用于调整文字的排版、大小和对齐方式。
*   **粗斜体**：此选项可以为文字应用**加粗**或*倾斜*效果。
*   **花字**：点击此处可将普通文字转换为风格化的综艺花字。剪映提供了丰富的花字样式供您选择。
*   **动画**：此功能能为静态文字添加动态效果，让字幕更生动。这在剧情类视频或需要表达特定情绪时非常有用。
*   **文本朗诵**：此功能可将文字转换为语音（需联网使用）。您可以选择不同的音色，确定后App会自动生成一段音频。

## 识别字幕与歌词

了解了手动创建字幕后，我们来看看如何利用剪映的智能识别功能快速生成字幕。

“识别字幕”和“识别歌词”功能本质相同，都是通过识别语音来自动转换成字幕。我们以“识别歌词”为例进行演示。

首先，在时间轴上导入一段包含人声或歌声的音频素材。

![](img/78203549db8799afe516028aebbd27dc_4.png)

接着，点击底部“文本”选项，然后选择“识别歌词”，点击“开始识别”。请注意，此功能同样需要联网。

![](img/78203549db8799afe516028aebbd27dc_6.png)

识别完成后，字幕会自动出现在时间轴上。您可以批量或单独调整这些字幕的位置和样式。

若要批量修改所有字幕的样式（如花字），在编辑界面勾选 **“应用到所有歌词”**，然后更改样式，所有字幕将统一更新。

如果只想修改某一句字幕，则取消勾选“应用到所有歌词”，再进行编辑，这样更改就只应用于当前选中的字幕。

## 总结

![](img/78203549db8799afe516028aebbd27dc_8.png)

本节课中，我们一起学习了在剪映中添加和设置字幕的完整流程。您掌握了如何手动**新建文本**并进行字体、样式、动画等详细设置，也学会了使用**识别歌词/字幕**功能来高效生成字幕，并对其进行批量或单独编辑。合理运用这些功能，能让您的视频字幕更专业、更吸引人。
