# 抖音运营实战：P11：扶持加权与降权限流

在本节课中，我们将学习抖音平台对视频内容的两种核心处理机制：扶持加权与降权限流。理解这些规则是避免踩坑、让账号获得更多推荐流量的关键。

![](img/616ec08c7806047386125c4daf1862fc_1.png)

上一节我们介绍了账号权重的分类，本节中我们来看看哪些内容会受到平台的扶持，哪些内容则可能导致降权或限流。

![](img/616ec08c7806047386125c4daf1862fc_3.png)

## 扶持加权的内容标准

平台会给予符合以下标准的视频更多流量扶持：

![](img/616ec08c7806047386125c4daf1862fc_5.png)

以下是获得扶持加权的视频需要满足的具体条件：

*   **视频格式**：必须为**竖版（9:16）** 的移动端原生视频。
*   **画面质量**：画面平稳，作品清晰度高。
*   **音频质量**：无过多杂音和噪音。

这些标准旨在鼓励创作者生产适合手机端观看的高质量内容。相关详细说明已整理为资料页，可供课后查阅。

## 降权限流的内容禁区

反之，发布以下内容将极大可能触发平台的降权或限流机制。

以下是明确会导致降权限流的内容类型列表：

1.  **违规信息**：内容涉及吸烟、暴力、炫富（金钱）等不良导向。
2.  **外部引流**：出现微信、微博、淘宝等站外联系方式。长时间展示商品Logo、手机号、二维码（即使出现，建议不超过3秒，但审核严格时仍可能被驳回）。
3.  **硬性广告**：进行明显的营销广告、推销或引流卖货行为。
4.  **无口播的好物推荐**：这是新手极易犯的错误。仅展示拆箱过程或产品，没有口播介绍。例如：
    *   **错误示范**：视频中只出现“9.9元”的价格标签，或全程无声展示玩具玩法。
    *   **正确做法**：应配合简单口播介绍使用体验，例如：“我儿子玩这个玩具，专注了四个小时都没停。” 通过口播将内容转化为分享体验，而非单纯卖货。
5.  **口播广告**：在口播中直接念出联系方式或广告语。

请注意，直接、生硬的广告行为在抖音是行不通的。平台鼓励的是通过优质内容进行“软性”种草。

本节课中我们一起学习了抖音平台扶持与限流的核心规则。关键在于制作**竖屏、高清、无杂音**的原生视频，并避免**硬广、引流和无口播的纯产品展示**。掌握这些原则，能为你的账号打下健康的基础。
