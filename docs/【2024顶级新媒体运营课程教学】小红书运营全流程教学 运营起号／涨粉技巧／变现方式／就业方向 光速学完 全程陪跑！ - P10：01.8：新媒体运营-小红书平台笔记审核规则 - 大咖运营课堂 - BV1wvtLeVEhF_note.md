# 小红书运营全流程教学：P10：01.8：小红书平台笔记审核规则 📝

在本节课中，我们将要学习小红书平台的核心机制之一：笔记的流量推荐与分发流程。理解这套规则，是做好小红书运营的基础。

小红书与许多新媒体平台一样，采用“千人千面”的内容展现形式，即根据用户的特定标签推送特定内容。本节我们将详细拆解一篇笔记从发布到推荐给用户的完整流程。

## 笔记审核的三个阶段

笔记发布后，首先会进入平台的审核系统。这个过程主要分为三个阶段。

### 第一阶段：机器审核

笔记上传后，首先会进入机器审核系统。该系统会根据预设的基本规则进行初步判断。

![](img/6df2c24ba54aac7043813f4c0f3fd847_1.png)

![](img/6df2c24ba54aac7043813f4c0f3fd847_3.png)

以下是机器审核的主要检查项：

![](img/6df2c24ba54aac7043813f4c0f3fd847_5.png)

![](img/6df2c24ba54aac7043813f4c0f3fd847_7.png)

![](img/6df2c24ba54aac7043813f4c0f3fd847_9.png)

*   **敏感内容检测**：识别笔记中的敏感词汇、敏感图片等。
*   **初步质量评估**：检测笔记的基础质量，例如字数、广告词出现频率、图片清晰度、文字可识别性等。

如果笔记存在明确违规内容，可能会被直接封禁。如果机器无法准确判断（即“疑似违规”），笔记会被标记并转入下一阶段。

通过机器初步审核的笔记，会进入查重检测阶段。系统会将你的笔记与平台数据库内的现有内容进行比对。

![](img/6df2c24ba54aac7043813f4c0f3fd847_27.png)

![](img/6df2c24ba54aac7043813f4c0f3fd847_29.png)

查重检测主要比对以下几个方面：

*   **图片相似度**：图片是否为直接抄袭或简单处理（如裁剪、加滤镜）。
*   **文案重复率**：文案内容、段落在数据库中的重复比例。

*   **成本与比例**：由于内容量巨大且人工成本高，并非所有笔记都会进入人工审核，该阶段占比相对较小。
*   **重点领域**：在政治、色情擦边、法律严禁等特殊领域，人工审核的力度会非常大。
*   **红线机制**：触及平台机器审核红线的笔记，目前大多会直接处理，人工复核较少。如果认为被误判，创作者可以通过申诉渠道反馈。

**特别提醒**：对于新账号，前期应尽量避免任何违规行为。即使是一两次警告或限流，也可能对账号产生隐性的负面影响。

## 关于账号“权重”的深入理解

![](img/6df2c24ba54aac7043813f4c0f3fd847_73.png)

![](img/6df2c24ba54aac7043813f4c0f3fd847_75.png)

在机器审核阶段，系统除了判断违规，还会评估笔记的“基础质量分”。这引出了一个重要概念：账号权重。虽然平台没有公开的“权重分”说法，但其推荐机制确实会综合评价账号的长期表现。

![](img/6df2c24ba54aac7043813f4c0f3fd847_77.png)

![](img/6df2c24ba54aac7043813f4c0f3fd847_79.png)

![](img/6df2c24ba54aac7043813f4c0f3fd847_81.png)

我们可以用一个从 **0 到 10** 的假设模型来理解账号权重：

![](img/6df2c24ba54aac7043813f4c0f3fd847_83.png)

*   **新号阶段（权重 ≈ 0）**：新注册的账号初始权重很低。此阶段的核心任务是“养号”，通过持续、垂直的优质内容输出和规范的账号行为（如定期更新、避免违规），来逐步积累系统信任。
*   **成长与稳定阶段（权重提升）**：当账号持续发布优质内容，并被系统打上清晰领域标签（如“彩妆”）后，账号在该领域的权重会提升。高权重账号发布的笔记，通常会获得更高的初始推荐流量池。

**权重的影响示例**：
一个新号（权重≈0）发布的笔记，初始推荐量可能只有几十。即使内容优质，也因为账号缺乏标签而难以精准推送给感兴趣的用户，导致互动数据不佳，从而无法进入更大的流量池。而一个高权重账号（权重=5）发布的同等质量笔记，起始推荐量可能更高，且推送更精准，更容易获得好的互动数据，形成正向循环。

**核心要点**：在账号权重低的时期，不要期望获得大量流量。应专注于通过**垂直领域的内容**和**规范的账号行为**来稳步提升账号在系统内的综合评价。
