# 拼多多运营：P15：拼多多企业店铺入驻流程 📝

在本节课中，我们将学习拼多多企业店铺的完整入驻流程。我们将了解所需的材料、不同类型店铺的区别、特殊行业的要求，以及如何在后台上传相关资质。

## 概述

上一节我们介绍了拼多多开店的基础知识，本节中我们来看看如何以企业身份入驻拼多多。企业店铺相比个人店铺拥有更多功能和权限，但入驻流程和所需材料也更为复杂。我们将分步骤详细讲解。

## 一、企业入驻所需核心材料

首先，我们需要了解开设企业店铺必须准备的基础材料。

![](img/1bc2d21d05c59f7f01b267e1864261df_1.png)

以下是所有企业店铺都需要的两项核心材料：
1.  **企业法定代表人身份证**：需要提供身份证原件的正反面照片。
2.  **店铺管理人身份证**：同样需要正反面照片。店铺管理人可以与法定代表人是同一人。

## 二、不同类型企业店铺的资质要求

拼多多的企业店铺主要分为四种类型：旗舰店、专卖店、专营店和普通企业店。它们所需的资质有所不同。

![](img/1bc2d21d05c59f7f01b267e1864261df_3.png)

以下是各类店铺的详细材料清单：

*   **旗舰店**
    *   **自有品牌旗舰店**：需要企业三证、质检报告、商标注册证、对公账户开户证明。**不需要**品牌授权书。
    *   **授权品牌旗舰店**：需要企业三证、质检报告、商标注册证、品牌授权书、对公账户开户证明。
    *   **卖场型品牌旗舰店**：需要企业三证、质检报告、商标注册证、品牌授权书、对公账户开户证明。

*   **专卖店**
    *   需要企业三证、质检报告、商标注册证、品牌授权书、对公账户开户证明。

*   **专营店**
    *   需要企业三证、质检报告、商标注册证、品牌授权书、对公账户开户证明。

![](img/1bc2d21d05c59f7f01b267e1864261df_5.png)

![](img/1bc2d21d05c59f7f01b267e1864261df_6.png)

*   **普通企业店**
    *   仅需要企业三证和对公账户开户证明。

![](img/1bc2d21d05c59f7f01b267e1864261df_7.png)

![](img/1bc2d21d05c59f7f01b267e1864261df_8.png)

> **核心概念解释**：
> *   **企业三证**：通常指营业执照、组织机构代码证、税务登记证。现在多证合一，通常只需提供**营业执照**即可。
> *   **对公账户开户证明**：由银行开具的企业基本存款账户信息表或开户许可证。

## 三、开店数量限制

了解材料后，我们还需知道一个企业主体能开多少家店。拼多多对企业开店数量有如下限制：

*   同一个企业营业执照下，最多可以开设 **5家** 普通企业店。
*   同一个企业营业执照下，旗舰店、专卖店、专营店这三类品牌店，加起来总共可以开设 **5家**。
*   因此，一个企业主体在拼多多平台最多合计可开设 **10家** 店铺。

## 四、特殊行业的额外要求

对于某些特殊经营类目，拼多多有额外的资质备案要求。以下是两个常见例子：

1.  **书籍类目**
    *   **个人/个体店**：需要个体工商户营业执照和《出版物经营许可证》。
    *   **企业店**：需要企业营业执照和对应的《出版物经营许可证》。
    *   入驻后，需在店铺后台提交资质进行备案。

2.  **虚拟商品类目**（如手机充值、游戏点卡）
    *   需要提交更多特定资质，例如与运营商或平台方的合作协议、虚拟业务备案证明等。
    *   具体所需材料可在拼多多商家后台的入驻指引中详细查询。

## 五、店铺创建与资质上传实操

准备好所有材料后，我们就可以开始创建店铺并上传资质了。

### 1. 店铺命名规则

在创建店铺时，需要根据店铺类型遵循相应的命名格式：

*   **普通企业店**：使用企业商号，例如“巨皇”。
*   **专营店**：`企业商号 + 类目 + 专营店`，例如“巨皇女装专营店”。
*   **专卖店**：`品牌名 + 企业商号 + 专卖店`，例如“雀巢巨皇专卖店”。
*   **旗舰店**：`品牌名 + (类目) + 旗舰店`，例如“巨皇旗舰店”或“巨皇女装旗舰店”。

![](img/1bc2d21d05c59f7f01b267e1864261df_10.png)

### 2. 资质上传路径

![](img/1bc2d21d05c59f7f01b267e1864261df_11.png)

不同材料的提交时机和位置不同：

*   **企业三证与开户证明**：在**注册店铺**的流程中直接填写和上传。
*   **商标、质检报告、授权书等**：在店铺注册成功后，进入**商家后台**上传。
    *   操作路径：登录商家后台 -> 进入【店铺】-> 点击【店铺信息】-> 找到【品牌资质】或【商标信息】-> 点击“新增”进行上传。
*   **行业经营许可证**（如食品经营许可证、出版物许可证）：同样在商家后台的【店铺信息】部分，找到【行业许可证】或相关入口进行提交。

![](img/1bc2d21d05c59f7f01b267e1864261df_13.png)

### 3. 店铺升级

![](img/1bc2d21d05c59f7f01b267e1864261df_14.png)

如果你已经拥有个人店铺，可以将其升级为企业店铺。

操作路径：商家后台 -> 【店铺】-> 【店铺信息】-> 找到【店铺升级】选项。按照提示填写升级申请书，并上传新的企业法人证明及营业执照等材料即可。
