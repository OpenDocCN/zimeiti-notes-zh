# 抖音运营教程：P14：11、视频类型1-2

在本节课中，我们将要学习抖音账号运营中一个至关重要的环节：**人设的建立**。我们将深入探讨强人设与弱人设的区别，并详细介绍一种核心的视频内容形式——**口播**。掌握这些概念，将帮助你更清晰地规划账号内容，吸引目标粉丝。

上一节我们解决了文案问题，本节中我们来看看内容创作的下一个关键环节。从涨粉到变现并非一条直线，中间有一个核心环节：**建立人设**。建立人设的目的不是为了炫耀，而是为了在众多同类账号中脱颖而出。例如，有十个做装修的账号，如果你的账号人设鲜明有趣，用户就更容易记住你，进而喜欢你、关注你，最终成为你的粉丝。

这就是人设的作用。接下来，我们正式讲解这个知识点，请边听边记。

人设主要分为两种类型：**强人设**和**弱人设**。理解这两种人设后，你就能轻松判断任何视频的人设类型。

首先，我们来解释什么是强人设。

![](img/606d195fb711563a7fc758e82c904856_1.png)

**强人设**是指以创作者**个人**为切入点的内容。它必须与创作者自身密切相关。以下是强人设常见的切入点：

![](img/606d195fb711563a7fc758e82c904856_3.png)

*   **个人经验与职业**：分享你所在行业的专业知识或经验。例如：装修、办公设计、房产、汽车、餐饮、婚介、法律咨询、电商等领域。
*   **个人兴趣与爱好**：分享你的生活爱好。例如：美食、旅行、手工、宠物、星座占卜、美妆护肤、服装穿搭、游戏、摄影等。

![](img/606d195fb711563a7fc758e82c904856_1.png)

![](img/606d195fb711563a7fc758e82c904856_3.png)

强人设的核心是分享与你个人相关的工作、生活和爱好。因此，大部分强人设账号会采用**真人出镜**的形式。

与之相对的是**弱人设**。

**弱人设**是指内容与创作者本人关系不大，主要目的是为了运营好账号本身。例如：
*   **情感语录账号**：发布情感文案配音乐，无真人出镜或配音。
*   **影视解说账号**：虽然有人声解说电影，但重点在电影内容，而非解说者本人。

弱人设的内容核心是题材本身，而非塑造创作者的个人形象。

区分清楚强人设和弱人设，是规划账号内容的基础。接下来，我们看看如何基于人设来创作视频内容。

内容创作环节中，任何与你个人相关的话题都可以作为视频素材。我们首先介绍一种非常实用且高效的内容形式：**讲故事**。

讲故事，即分享客户故事、从业经历或人生感悟。故事本身不要求完全真实，可以从文案平台寻找有趣的故事进行分享。而讲故事最常用的表现形式就是**口播**。

口播是指面对镜头进行讲述的视频形式。它有很多分类，适用于不同领域：
*   **故事型口播**：目的是让观众觉得有趣、有共鸣。
*   **知识型口播**：分享育儿、金融、养生、装修、房产、汽车等领域的知识，目的是让观众在几十秒内获得“收获感”，觉得学到了东西。

口播是一种非常适合新手起步的视频形式。它的优势很明显：
1.  **适用范围广**：法律、保险、金融、装修、汽车等知识科普领域都适合做口播。
2.  **拍摄简单**：只需一部手机和准备好的文案即可拍摄。
3.  **对口才要求不高**：口播是提前准备好稿子再录制，并非临场发挥。
4.  **不限制语言**：普通话不标准也可以做。如果你的目标客户是同城，使用方言可能更亲切、更有趣。观众听不懂可以看字幕。

本节课中我们一起学习了抖音账号运营中“人设”的概念，明确了**强人设**与**弱人设**的区别。同时，我们详细介绍了一种核心的视频内容形式——**口播**，了解了它的类型、优势及拍摄要点。建立清晰的人设并选择合适的表达形式，是账号吸引精准粉丝、实现长远发展的关键一步。
