# 实用ChatGPT亚马逊运营指令：P1：雪球站外推广

![](img/e58a609c077e7a3ba7830cdb736bd790_1.png)

在本节课中，我们将学习如何利用ChatGPT来提升亚马逊运营的效率，特别是通过一系列具体指令来辅助完成标题撰写、五点描述优化、关键词挖掘和竞品分析等工作。

上一节我们介绍了ChatGPT在亚马逊运营中的潜力，本节中我们来看看一系列可直接使用的具体指令。

以下是12条实用的ChatGPT亚马逊运营指令：

![](img/e58a609c077e7a3ba7830cdb736bd790_3.png)

1.  **指令：** 请为[你的产品名称]生成10个亚马逊产品标题，要求包含核心关键词[你的核心关键词]，并符合亚马逊SEO最佳实践。
2.  **指令：** 请优化以下五点描述，使其更具说服力并包含关键词[关键词1]、[关键词2]：[你的五点描述原文]。
3.  **指令：** 请基于核心关键词“[核心关键词]”，为我扩展20个相关的长尾关键词。
4.  **指令：** 请分析以下竞品ASIN的价格和促销策略，并总结其定价规律：[竞品ASIN列表]。
5.  **指令：** 请为产品[产品名称]撰写一份详细的产品描述，突出其[卖点1]、[卖点2]和[卖点3]。
6.  **指令：** 请将以下产品功能列表转化为吸引消费者的好处描述：[功能列表]。
7.  **指令：** 请为我的亚马逊店铺[店铺名称]构思5个品牌故事的角度。
8.  **指令：** 请根据客户评论“[代表性评论内容]”，生成3条回复模板，一条用于感谢好评，一条用于处理中评，一条用于应对差评。
9.  **指令：** 请为[产品类别]的亚马逊新品，制定一个为期4周的站外推广内容计划大纲。
10. **指令：** 请检查以下A+页面文案的语法和流畅度，并提出修改建议：[你的A+文案]。
11. **指令：** 请扮演一位对[产品类别]感兴趣的潜在买家，对我的产品详情页提出5个可能存在的疑问。
12. **指令：** 请分析当前季节/节日（例如：圣诞节），为[产品名称]提供3个捆绑销售或促销活动的创意。

本节课中我们一起学习了12条针对亚马逊运营的实用ChatGPT指令。这些指令覆盖了从Listing优化、关键词研究到竞品分析和客户沟通等多个核心工作环节，能有效帮助运营者提升工作效率与内容质量。
