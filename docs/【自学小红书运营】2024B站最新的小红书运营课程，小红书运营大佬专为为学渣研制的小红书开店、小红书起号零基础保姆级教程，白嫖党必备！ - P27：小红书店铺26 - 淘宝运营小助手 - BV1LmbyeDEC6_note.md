## 概述

![](img/77c000eea437d7c16ba8626dcbe5288e_7.png)

本节课我们将重点探讨小红书开店前的**商品准备**环节。这是店铺启动的第一步，直接关系到后续的流量获取与销售转化。我们将详细拆解商品的数量搭配策略与发布技巧。

## 商品准备

商品准备是店铺规划的第一步，主要包含两个关键点：产品基础数量要求和商品发布技巧。合理的商品布局能为店铺打下坚实的基础。

### 1. 产品基础数量要求

整个店铺的商品数量应控制在7到15个之间。这个数量范围便于管理，并能维持较好的店铺动销率。具体搭配根据店铺类型（单店或多店）有所不同。

以下是产品类型的定义与搭配策略：

*   **主产品**：店铺内销量最高、作为主要推广对象的核心商品。
*   **副产品**：有一定销量，用于辅助主产品、丰富店铺品类的商品。
*   **补量产品**：无需追求成交，主要用于填充店铺、提升店铺整体丰富度的商品。

**对于多店操作**，每个店铺准备7件产品即可。搭配方案如下：
*   **方案A**：1件主产品 + 2件副产品 + 4件补量产品。
*   **方案B**：1件主产品 + 1件副产品 + 5件补量产品。

**对于单店操作**，商品总数可逐步增加至15件。初始阶段先上架7件产品，后续随着新主产品的推出，动态调整结构：
1.  **初期**：1主 + 2副 + 4补量（共7件）。
2.  **推出第二款爆款时**：下架1件副产品或补量产品，上架新的主产品。结构变为：2主 + 1副 + 剩余为补量产品。
3.  **推出第三款爆款时**：结构可调整为：3主 + 0副 + 其余为补量产品。
4.  **当成功推出三款主产品后**，店铺数据已趋稳定，剩余的补量产品可逐步尝试“转正”为有销量的商品。

**核心原则**：店铺总商品数不超过15件，并始终保持“主产品带动，副产品辅助，补量产品填充”的动态平衡。

### 2. 商品发布技巧

商品发布顺序和时间点对系统收录和初始权重有重要影响。错误的发布顺序可能导致前期推广效果大打折扣。

以下是具体的发布步骤与注意事项：

*   **第一步：系统录入**。开店后，首先上传1到2件**副产品**。此时不要进行任何推广，目的是让小红书平台系统识别并收录你的店铺与商品，获得一个基础排名。
*   **第二步：上架主产品**。在副产品上传、店铺正常运行约一天后，开始上传**主产品**。主产品需选择在**流量高峰期**上架（例如晚上18-20点或中午11-13点），上架后等待30分钟至1小时，待系统充分收录后再开始推广。
*   **第三步：控制节奏**。每天上架的新商品数量不宜超过3件。如果时间充裕，每天上架1-2件是更稳妥的选择。
*   **第四步：统一高峰期**。所有主产品的上架和后续主要推广活动，应固定在同一流量高峰期进行，以集中权重，提升搜索排名效果。副产品则可以在任意时间段上架。

**错误做法示范**：第一天就直接上架主产品并立即大力推广。此时系统尚未对店铺建立完整认知，推广效果会大打折扣。

## 总结

本节课我们一起学习了小红书店铺前期规划中的商品准备环节。我们明确了店铺商品应控制在**7-15件**，并掌握了**主产品、副产品、补量产品**的动态搭配策略。同时，我们学习了关键的发布技巧：先上副产品让**系统录入**，再于**固定高峰期**上架主产品并进行推广。做好这些准备，将为后续的店铺运营打下坚实的数据基础。
