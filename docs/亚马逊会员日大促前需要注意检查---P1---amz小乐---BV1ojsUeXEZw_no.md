# 亚马逊运营：P1：会员日大促前检查清单

在本节课中，我们将学习亚马逊会员日大促前必须检查的几个关键事项，以确保促销活动顺利进行并最大化销售效果。

上一节我们介绍了课程主题，本节中我们来看看具体的检查清单。

## 🛒 一、促销状态检查

![](img/a5b3d002326208c2a0a6370c8ddae2e6_1.png)

首先，需要检查所有促销活动的状态。卖家应筛选出需要注意或利用的活动，并根据系统提示进行正确修改，确保促销计划设置成功。

以下是促销状态检查的具体步骤：

![](img/a5b3d002326208c2a0a6370c8ddae2e6_2.png)

1.  登录卖家后台，进入“广告活动管理”或“促销”模块。
2.  使用筛选功能，找出状态为“需注意”或“未开始”的促销活动。
3.  根据每个活动旁边的提示信息，点击“编辑”按钮，修改不正确的设置。

![](img/a5b3d002326208c2a0a6370c8ddae2e6_1.png)

![](img/a5b3d002326208c2a0a6370c8ddae2e6_2.png)

**重要提示**：点击“查看详情”并根据提示原因编辑促销设置。需要注意的是，任何更改编辑后，系统通常需要**两个小时**来更新状态。

## 📦 二、确保库存充足

促销状态确认无误后，库存准备是下一个核心环节。巧妇难为无米之炊，必须按照亚马逊FBA的要求，提前安排仓储和库存，确保货物按时或提前入仓。

以下是库存准备的关键点：

*   **提前备货**：根据销售预测，提前将足量库存发送至亚马逊运营中心。
*   **检查库存水平**：在大促前反复确认在售商品的FBA库存数量，避免断货。
*   **关注入库进度**：在卖家后台跟踪货件状态，确保在大促前完成接收和上架。

核心目标是确保在会员日大促当天有充足的库存来进行销售。

## ✍️ 三、优化商品详情页（Listing）

除了以上与促销直接相关的设置外，大促前Listing的优化也非常重要。商品页面是吸引和转化顾客的关键。

卖家精灵最新版本推出的“五点描述检测”功能，可以帮助卖家自动检查并标注出不符合亚马逊最新政策要求的书写规范，以便于及时改正。

该功能的使用方式如下：
```python
# 概念性代码：检测五点描述
def check_bullet_points(listing):
    for bullet in listing.bullets:
        if length(bullet.text) > 规定字符数:
            highlight_for_review(bullet) # 高亮提示需要修改
        if contains_invalid_symbols(bullet.text):
            highlight_for_review(bullet) # 高亮提示特殊符号问题
```
我们可以看到检测报告中会清晰标注出每个要点的字符总数，以及需要更改的提示（例如字符超限、包含不允许的特殊符号等）。

**提示**：需要免费试用卖家精灵或订阅卖家精灵插件的用户，可以通过视频评论区提供的链接获取。
