# 小红书运营教程：P21：19.导入的素材注意事项 📂

在本节课中，我们将学习导入素材到剪辑软件时一个至关重要的注意事项：妥善管理原始文件。理解并遵循这一点，可以避免未来剪辑工作中出现媒体丢失的麻烦。

![](img/731e00578773f05181cf8cdafb5aa626_1.png)

上一节我们介绍了素材导入的基本操作，本节中我们来看看导入后如何管理这些原始文件，确保工程稳定。

## 核心原则：保留原始文件

导入到素材库的文件，必须在电脑本地硬盘中保存好。例如，如果你从“当前文件夹”中选择了素材进行导入，就必须确保这个“当前文件夹”及其中的素材文件被妥善保存。

![](img/731e00578773f05181cf8cdafb5aa626_3.png)

如果原始文件未被保存好，将会导致严重问题。

## 问题演示：媒体丢失

![](img/731e00578773f05181cf8cdafb5aa626_5.png)

以下是一个典型的问题场景：
1.  我们选择了一个“小猫”视频文件，将其导入到素材库中。
2.  接着，我们将这个“小猫”素材拖拽到时间轴进行剪辑。
3.  假设你花费了很长时间完成了视频剪辑。
4.  之后，你不小心在电脑本地删除了原始的“小猫”视频文件。
5.  当你再次打开这个剪辑工程时，软件会提示“媒体丢失”。在素材库或时间轴中，该文件将显示为异常状态，无法正常使用。

## 最佳实践：文件管理指南

为了避免上述问题，你必须建立良好的文件管理习惯。以下是具体的操作建议：

*   **专用文件夹**：为每个剪辑项目建立独立的文件夹，将所有用到的素材、音乐、工程文件都存放在里面。
*   **合理分区**：项目文件夹尽量不要放在系统C盘，建议放在D盘、E盘等其他盘符，防止系统重装或清理时被误删。
*   **定期备份**：重要的工程和素材文件，应定期备份到移动硬盘或云盘。
*   **不要移动/删除源文件**：在剪辑项目完全结束并导出成片之前，切勿移动或删除导入软件的任何原始素材文件。

良好的文件管理是高效、无压力剪辑的基础。它确保你能随时重新打开和修改旧工程，应对内容调整或客户修改的需求。

本节课中我们一起学习了导入素材后的核心注意事项：**务必在本地妥善保存原始文件**。通过演示，我们看到了删除源文件会导致的“媒体丢失”后果，并给出了建立专用文件夹、合理分区和定期备份等具体管理方案。记住，管理好文件，就是管理好你的剪辑工作流。
