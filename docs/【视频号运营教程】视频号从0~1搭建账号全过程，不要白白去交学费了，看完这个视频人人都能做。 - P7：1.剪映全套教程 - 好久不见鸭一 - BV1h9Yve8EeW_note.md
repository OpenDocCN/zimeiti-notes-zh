# 视频号运营教程：P7：剪映自学指南 📚

在本节课中，我们将学习如何从零开始自学剪映，掌握视频剪辑的基础功能。通过官方课程和优质博主资源，你可以在短时间内学会使用这款强大的免费工具。

## 课程概述

短视频普及多年，许多人已具备一定剪辑基础。如果你从未使用过剪映，本教程将指导你如何高效自学。剪映是抖音官方的剪辑软件，功能强大且大部分免费，足以满足日常创作需求。

## 第一步：获取与认识剪映

首先，你需要在手机应用商店下载“剪映”App。它是目前使用最广泛的剪辑工具之一，与“必剪”等软件功能相似。

![](img/56e59e4a2f5e49c6faa1d27e83b7324b_3.png)

**核心操作**：在应用商店搜索并下载“剪映”。

---

![](img/56e59e4a2f5e49c6faa1d27e83b7324b_5.png)

## 第二步：访问官方免费课程

打开剪映App后，首页最下方有一个“创作者课堂”入口。点击进入后，在搜索框中输入“剪映小助手”进行搜索。

在搜索结果中，选择“用户”标签页。排在第一位的通常是“剪映官方”账号，请关注这个账号。

![](img/56e59e4a2f5e49c6faa1d27e83b7324b_7.png)

关注后，进入该官方账号的主页。你会看到“模板”、“课程”、“素材”等选项。请点击进入“课程”板块。

![](img/56e59e4a2f5e49c6faa1d27e83b7324b_9.png)

---

## 第三步：学习核心免费课程

![](img/56e59e4a2f5e49c6faa1d27e83b7324b_11.png)

在官方课程列表中，请重点学习以下两个免费系列：
1.  **剪映功能大全**：共87节课。
2.  **新手常见问题**：共10节课。

这些课程每节时长很短，多为几十秒到两分钟左右。完整学习全部课程大约需要两个小时，足以让你从零基础达到熟练操作的水平。

![](img/56e59e4a2f5e49c6faa1d27e83b7324b_13.png)

**学习建议**：如果你完全没有剪辑基础，务必先系统学完这些基础功能课程，再进行实际操作练习。

![](img/56e59e4a2f5e49c6faa1d27e83b7324b_15.png)

此外，在抖音搜索“零基础学剪辑”等关键词，也能找到大量教学账号。观看几个高质量视频，就能快速掌握剪映的核心功能。
