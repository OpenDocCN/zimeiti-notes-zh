# 小红书运营教程：P3：2. 小红书平台检测规则详解 📋

在本节课中，我们将要学习小红书平台的核心检测规则。了解这些规则是避免账号违规、确保内容安全发布的基础。我们将从多个维度分析小红书的审查机制，并提供实用的工具和方法来辅助运营。

## 平台内容检测范围

小红书系统会重点检测以下几个方面的内容，任何违规都可能导致笔记被限流或账号受处罚。

以下是平台重点审查的四个区域：

1.  **个人资料、笔记内容、私信及评论**：这些位置**不得含有**微信号、微信二维码、“代购”字样、第三方平台水印，以及推广淘宝等店铺的营销词汇。
2.  **允许的引流方式**：平台对**邮箱**的管制较松，允许在资料中填写。此外，可以使用“微博”或其谐音词进行间接引流，再通过话术引导用户至微信。
3.  **设备登录限制**：平台会检测手机设备。建议**同一台设备最多只登录三个账号**，超出此数量可能引发风险。
4.  **禁止诱导用户行为**：笔记内容**不得诱导**用户进行关注领奖、点赞抽奖等互动。这类行为是小红书明令禁止的。

## 内容原创与发布规范

上一节我们介绍了平台检测的具体范围，本节中我们来看看关于内容本身的核心规则。

内容的原创度在小红书平台受保护程度很高。因此，在推广时需遵守“一稿一发”原则。

以下是必须注意的发布规范：

*   **禁止重复发布**：不要将同一篇文章在多个小红书账号上发布。一篇文章只用一个账号发布即可。
*   **保护原创**：直接搬运或抄袭他人内容极易被系统识别并处罚。

## 敏感词检测与处理

除了上述规则，运营中最常遇到的问题是敏感词检测。敏感词包括但不限于“手机卡”、“金融”、“电子烟”、“代购”、“医疗用品”等领域的词汇。撰写内容时应尽量避免触及这些词汇。

那么，如果在写作时不慎加入了敏感词该怎么办？这时就需要借助敏感词检测工具。

![](img/57c3499c659e8fe832e0d78c0d9c34e2_3.png)

以下是两个实用的敏感词检测网站使用方法：

1.  **网站A（免登录）**：
    *   将写好的笔记内容粘贴到网站的输入框中。
    *   点击“立即检测”按钮。
    *   系统会高亮或列出内容中包含的所有小红书敏感词，例如“代购”。
    *   根据检测结果修改内容即可。

    

2.  **网站B（需注册）**：
    *   使用方法与网站A类似，注册登录后即可使用。
    *   将内容粘贴后，网站会提供过滤后的结果，并列出发现的违禁词。

**操作核心**：在发布任何笔记前，先将文案内容通过上述检测网站筛查一遍，可以有效规避因敏感词导致的违规风险。
