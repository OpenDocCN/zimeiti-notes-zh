# 抖音运营：P19：制造情绪冲突的文案角度

在本节课中，我们将要学习如何通过制造情绪冲突来提升抖音文案的吸引力。我们将探讨为什么同样的选题，不同的表达方式会产生截然不同的效果，并介绍十种实用的文案角度模板，帮助你快速创作出能引发用户共鸣、促进行动的内容。

## 选题与表达方式的重要性

上一节我们介绍了选题的重要性。选好题之后，我们才能进行脚本结构设计和拍摄。本节中我们来看看，同样的核心信息，用不同的方式表达，效果会完全不同。

例如，鼓励用户去健身：
*   **普通表达**：“来健身房，变成更美的自己。” 这句话像一句无关痛痒的广告词，类似“开沃尔沃，更安全”。
*   **冲突表达**：“如果你不去健身房，你就失去了变美的机会。” 这种方式利用了“失去心理”和“恐吓心理”，能在用户心中制造一种紧迫感。

再比如，推广面膜产品：
*   **普通表达**：“敷面膜，永远保持20岁。” 这也是一句干巴巴的广告语。
*   **冲突表达**：“不坚持敷面膜，你会长出难以消除的鱼尾纹、色斑。” 通过描绘具体的负面后果，更能引起用户的重视。

这两种表达方式给用户的心理暗示是不同的。这就像电商促销：
*   **单纯优惠**：“今天购买打五折。”
*   **叠加刺激**：“五折优惠仅剩最后两个名额！” 后者制造的紧迫感会促使你立刻下单。

因此，关键在于制造情绪的冲突。同样的选题，用不同的心理角度去阐述，给用户造成的情感冲击是完全不同的。

## 制造情绪冲突的十种文案角度

以下是制造情绪冲突的十种经典文案角度模板。请注意，前几种模板效果相对普通，但某些行业仍可使用；而“恐惧心理”和“失去心理”是最常使用且效果显著的，但也不宜过度频繁，以免引起用户反感。

1.  **直接承诺法**
    *   **公式**：`做[A]，就能得到[B]。`
    *   **示例**：“来健身房，变成更美的自己。”

2.  **理想身份法**
    *   **公式**：`拥有[产品/服务]，你就是[理想身份]。`
    *   **示例**：“戴上这块表，你就是精英人士。”

3.  **场景体验法**
    *   **公式**：`使用[产品/服务]，体验[美好场景]。`
    *   **示例**：“喝下这杯咖啡，享受清晨的宁静时光。”

4.  **未来收益法**
    *   **公式**：`现在投入[A]，未来收获[B]。`
    *   **示例**：“每天学习一小时，一年后成为行业专家。”

5.  **规避损失法（常用）**
    *   **公式**：`如果不做[A]，你将失去[B]。`
    *   **示例**：“错过这次促销，你就得多花一倍的钱。”

6.  **恐惧心理法（常用）**
    *   **公式**：`如果不做[A]，你将面临[具体的负面后果]。`
    *   **示例**：“不科学健身，会导致腰椎间盘突出。”

![](img/369de75d48d61a442cbb9063629564fe_1.png)

7.  **稀缺心理法**
    *   **公式**：`[机会/产品]仅剩[X]个/仅限[X]时间。`
    *   **示例**：“限量款口红，最后50支！”

8.  **从众心理法**
    *   **公式**：`已经有[X]万人选择了[A]。`
    *   **示例**：“全网爆款，1000万宝妈都在用。”

9.  **权威背书法**
    *   **公式**：`[权威机构/人物]推荐/使用[A]。`
    *   **示例**：“国家营养学会指定产品。”

10. **挑战认知法**
    *   **公式**：`你以为[A]，实际上[B]。`
    *   **示例**：“你以为健身很痛苦？其实每天只需10分钟。”

学会这些角度后，你可以将它们灵活应用到任何视频的文案创作中。

## 实践工具：九宫格选题法

![](img/369de75d48d61a442cbb9063629564fe_3.png)

掌握了制造冲突的角度后，我们还需要系统化的选题方法。这里要记录并学会的是“九宫格选题法”。

![](img/369de75d48d61a442cbb9063629564fe_5.png)

**操作步骤**：
1.  在九宫格中央写下你的**核心产品**或**服务**。
2.  在周围的八个格子中，分别填入：
    *   目标用户的**基础属性**（如年龄、性别）。
    *   用户的**深层需求与痛点**。
    *   产品的**核心优势与特点**。
    *   不同的**使用场景**。
    *   相关的**热门话题**。
    *   可能引发的**情感共鸣点**。
    *   对标的**竞争对手**。
    *   希望达成的**转化目标**。

你可以将任何产品或用户画像代入这个九宫格，然后充分延伸每个维度的内容。最后，利用“连连看”的方式，将不同格子里的元素进行组合，就能碰撞出大量的优质选题。例如：`[年轻宝妈] + [痛点：孩子挑食] + [产品：营养辅食机] = “孩子挑食不吃菜？三步做出让他抢着吃的彩虹营养粥。”`
