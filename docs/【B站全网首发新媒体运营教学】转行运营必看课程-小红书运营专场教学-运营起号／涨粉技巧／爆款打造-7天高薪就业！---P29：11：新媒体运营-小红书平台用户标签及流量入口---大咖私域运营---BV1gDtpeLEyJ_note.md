# 新媒体运营：P29：小红书平台用户标签及流量入口 🏷️➡️🚪

![](img/6dcb0d12a9b846c34cd73b1f2538b769_1.png)

在本节课中，我们将要学习小红书平台如何为用户打上内容标签，以及平台主要的流量入口有哪些。理解这些机制，有助于我们分析笔记如何被系统推荐给精准用户，从而优化运营策略。

上一节我们介绍了系统如何判定内容标签，本节中我们来看看平台如何判定用户标签。这个过程能帮助我们理解笔记是如何被推荐给用户的，并能从底层逻辑反推，如何让笔记被更多精准用户看到。

## 用户标签判定机制

系统为用户打标签的主要依据是用户的**浏览行为**。用户注册时选择的兴趣范围参考价值不大，它仅作为初期内容方向的引导，无法进入系统深层的用户标签池。

系统根据以下几种用户行为来判定其兴趣分类：

以下是判定用户标签的核心行为指标：

1.  **点击率**
    *   **公式**：`点击率 = (点击某类内容的次数) / (系统推送的该类内容总次数)`
    *   系统在发现页向用户推送多种内容（如美妆、家居、母婴）。用户点击哪类内容更多，系统就会判定他对该类内容更感兴趣，并持续推送相关内容进行试探，从而打上强关联或弱关联的标签。

2.  **停留时长**
    *   用户点开笔记后的观看时长。如果用户对某类内容（如美妆）普遍停留时间短，但对偶然推送的另一类内容（如家居）停留很久，系统会判断用户可能产生了新的兴趣点（如近期有装修或购买家居用品的需求）。

3.  **页面浏览量（PV）**
    *   用户浏览页面的深度。例如，用户不仅看了一篇笔记，还进入了博主主页，并连续看了多篇笔记。提升用户在自己主页的浏览深度，对账号健康度有积极影响。

4.  **互动行为**
    *   包括点赞、收藏、评论、关注。这些数据直接提升笔记的互动率，有助于笔记进入更大的推荐流量池，并可能带动账号内其他笔记的流量。

## 平台流量分发入口

了解用户标签后，我们来看笔记会被分发给用户的哪些入口。小红书主要有四个流量分发入口。

以下是小红书平台的四个主要流量入口：

*   **发现页**
    *   **核心逻辑**：`推荐内容 = 已关注达人内容 + 兴趣匹配内容 + 发散性内容`
    *   这是最主要的流量来源。包含你关注的达人内容、系统根据你标签推荐的兴趣内容，以及少量平台为了内容多样化而推送的探索性内容。

*   **关注页**
    *   这里集中展示所有你已关注的博主发布的最新笔记。

*   **附近页**
    *   根据手机定位，推送同城或附近地区的笔记。以美食、旅游、本地生活等强地域性内容为主。适合线下实体店推广。

*   **购物页**
    *   小红书电商和变现的主要入口。直接匹配用户的购物需求，但用户习惯仍在培养中。

本节课中我们一起学习了小红书平台判定用户标签的四大行为依据（点击率、停留时长、PV、互动行为），以及笔记分发的四大流量入口（发现页、关注页、附近页、购物页）。理解这些底层逻辑，能帮助我们更有针对性地创作内容和运营账号，从而提升笔记曝光率与粉丝精准度。
