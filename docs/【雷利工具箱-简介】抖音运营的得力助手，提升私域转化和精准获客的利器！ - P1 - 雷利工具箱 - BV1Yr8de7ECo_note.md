# 雷利工具箱教程：P1：软件简介与核心功能概览 🧰

在本节课中，我们将一起了解“雷利抖音工具箱”这款软件。这是一款旨在辅助抖音运营、提升私域转化与精准获客效率的工具。请注意，本软件仅供学习交流与测试使用，用于违法操作者后果自负。

## 软件功能演示

以下是软件核心操作流程的演示：

*   🎼 发送特定指令（如“666”）以激活工具箱。
*   🎼 等待工具箱响应指令。
*   🎼 成功发送互动卡片，用户可点击跳转。
*   🎼 支持长按识别二维码进行操作。
*   🎼 选择需要采集的账号并设置采集信息。
*   🎼 添加目标达人以便关注或采集。
*   🎼 开启监控开关，启动自动化任务。
*   🎼 系统自动采集账号的新增粉丝。
*   🎼 支持手动采集账号的历史粉丝数据。
*   🎼 完成粉丝采集任务。
*   🎼 展示已采集粉丝的详细信息。
*   🎼 支持生成多通道二维码进行引流。
*   🎼 点击粉丝头像可直接进入其抖音主页。
*   🎼 在抖音内复制用户信息。
*   🎼 对已处理的粉丝进行标记。
*   🎼 提供多种条件（如性别、地区）筛选粉丝。
*   🎼 支持将采集的数据导出到本地文件。

除了粉丝管理，软件也支持视频评论的采集与管理：

*   🎼 支持通过视频链接、达人主页链接或关键词搜索来添加目标视频。
*   🎼 可部分选择或全选视频进行操作。
*   🎼 手动采集和自动采集均主要针对视频的新增评论。
*   🎼 在批量操作中可选择采集视频的历史评论。
*   🎼 完成评论采集任务。
*   🎼 点击达人昵称可直接进入其主页。
*   🎼 点击评论者昵称可查看其详情。
*   🎼 提供私信、@等多种方式触达评论用户。
*   🎼 对已处理的评论进行标记。
*   🎼 支持对评论进行多种条件筛选。
*   🎼 支持将多条数据导出。
*   🎼 支持数据变化的实时导出。
*   🎼 当数据发生变化时，软件会发送通知。
*   🎼 通知内容支持多种变量，使信息更丰富。

## 软件核心特点介绍

![](img/a4a40aeff75cf958b005ee50deae9a78_1.png)

上一节我们通过演示片段了解了软件的基本操作。本节中，我们来详细看看雷利抖音工具箱的几个核心特点。

1.  **跨平台支持**：软件支持 **Windows** 和 **macOS** 双平台，满足不同用户的需求。演示视频中，第一个片段在macOS上录制，后续片段在Windows上录制。

2.  **安全可靠**：软件的插件基于**抖音网页端官方接口**开发，这降低了因软件本身导致账号被封的风险，能在一定程度上确保账号安全。

3.  **灵活便捷**：软件不绑定特定抖音账号或设备机器码。用户可以随时切换登录的抖音账号，也可以自由更换运行设备。

4.  **提升效率**：软件支持**自动回复私信**、**群发消息**、**监控评论**等自动化功能，能有效减少重复性手动操作，显著提高运营效率。

5.  **多账号管理**：软件支持**多开**操作，非常适合拥有多个抖音账号的运营者同时管理。

6.  **持续运行**：软件可以部署到**云服务器**上，实现7x24小时不间断运行，确保自动化服务不会中断。演示中第一个片段就是通过远程连接云服务器上的软件录制的。
