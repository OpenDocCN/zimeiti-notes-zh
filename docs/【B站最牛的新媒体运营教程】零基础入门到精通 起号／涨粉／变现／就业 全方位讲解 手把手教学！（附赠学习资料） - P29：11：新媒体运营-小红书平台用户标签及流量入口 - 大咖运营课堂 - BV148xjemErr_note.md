## 用户标签判定逻辑

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_9.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_11.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_13.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_15.png)

小红书系统为用户打标签，主要依据用户的**浏览行为**。用户注册时选择的兴趣范围，仅作为初期内容方向的引导，参考价值不大，无法进入系统深层的用户标签池。

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_17.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_19.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_21.png)

系统根据用户的具体行为来判定其兴趣分类。以下是判定用户标签的几个核心行为指标：

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_23.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_25.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_27.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_29.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_31.png)

以下是判定用户标签的几个核心行为指标：

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_33.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_35.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_37.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_39.png)

1.  **点击率**
    *   **定义**：系统在发现页向用户推送多个领域的内容（如明星、美妆、家居），用户点击哪一类内容更多，系统就能计算出其偏好。
    *   **作用**：通过点击概率，系统为用户打上**强关联**或**弱关联**的标签。例如，用户频繁点击美妆内容，系统会判定其为美妆兴趣用户，并持续推送相关内容。

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_41.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_43.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_45.png)

2.  **停留时长**
    *   **定义**：用户点开一篇笔记后观看的持续时间。
    *   **作用**：停留时长能更精细地判断用户喜好。例如，用户虽常看美妆，但对某类美妆笔记快速划走，说明兴趣不高。若用户偶然点开一篇家居笔记并停留很久，系统可能判定其有装修或购买家居用品的潜在需求。

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_47.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_49.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_51.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_53.png)

3.  **页面浏览量（PV）**
    *   **定义**：用户浏览页面的深度，例如不仅看了单篇笔记，还进入了博主主页，并连续观看了多篇笔记。
    *   **作用**：PV值能提升账号的健康度评分。用户在你的主页浏览越深入，说明内容吸引力越强，这对账号成长有积极帮助。

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_55.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_57.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_59.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_61.png)

4.  **互动行为**
    *   **定义**：包括点赞、收藏、评论、关注等用户主动操作。
    *   **公式**：`互动数据 = 点赞数 + 收藏数 + 评论数 + 关注数`
    *   **作用**：互动数据直接决定笔记能否进入更大的**推荐流量池**。数据越高，笔记获得曝光的几率越大，同时也能带动账号内其他笔记的流量，并吸引粉丝关注。

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_63.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_65.png)

通过以上行为，系统会为用户构建一个动态的“兴趣云图”。其中占比较大的标签（如美妆），就是用户的核心兴趣标签，系统会据此进行主要的内容推荐。

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_67.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_69.png)

了解用户标签后，我们来看小红书主要的流量分发入口，即用户从哪里看到我们的内容。

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_79.png)

小红书平台主要有四个流量入口：

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_87.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_89.png)

以下是小红书平台的四个主要流量入口：

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_91.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_93.png)

1.  **发现页**
    *   **内容来源**：
        *   用户已关注的达人发布的内容。
        *   根据用户标签（点击率、停留时长等）匹配的**兴趣内容**。
        *   平台为了内容多样化而推荐的**发散性内容**（用户可能感兴趣的新领域）。
    *   **核心价值**：这是博主获取流量的**主要来源**。我们的笔记和账号能否获得大量曝光，关键在于能否进入发现页的推荐流。

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_95.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_97.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_99.png)

2.  **关注页**
    *   **内容来源**：纯粹展示用户**已关注的所有达人**发布的最新笔记。
    *   **特点**：流量相对稳定但有限，依赖于已有的粉丝基础。

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_101.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_103.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_105.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_107.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_109.png)

3.  **附近页**
    *   **内容来源**：根据手机定位，推送**附近及同城**用户发布的内容。
    *   **适用内容**：具有强地域特征的需求，如美食、本地旅游、线下店推广（美容院、餐厅等）。
    *   **提示**：如果你是线下实体店，希望通过小红书引流，需要重点关注此入口，并在投放信息流广告时注意定位设置。

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_111.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_113.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_115.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_117.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_119.png)

4.  **购物页**
    *   **内容来源**：小红书电商板块，直接匹配购物需求。
    *   **平台方向**：这是小红书向电商发展的重要入口，涉及直播带货、商品笔记等内容。
    *   **现状**：部分用户尚未形成在此直接购物的强习惯，更多是通过搜索笔记“种草”。

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_121.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_123.png)

## 博主、平台与用户的关系

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_125.png)

作为内容创作者（博主），我们需要平衡自身内容的价值：
*   **对用户**：提供有价值、感兴趣的内容。
*   **对平台**：贡献优质内容，帮助平台留住用户，提升活跃度。

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_127.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_129.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_131.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_133.png)

平台则作为桥梁，通过算法将博主的内容精准分发给可能感兴趣的用户，满足其需求。这三者形成了一个相互依存、共同成长的串联关系。

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_135.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_137.png)

![](img/5ae7e76eb8eb5ad3602ca647a3b8acc3_139.png)

本节课中我们一起学习了小红书平台判定用户标签的四大行为依据（点击率、停留时长、PV、互动行为），以及四大核心流量入口（发现页、关注页、附近页、购物页）。理解这些底层逻辑，能帮助我们更有效地规划内容，让笔记更精准地触达目标用户，实现账号的成长与变现。
