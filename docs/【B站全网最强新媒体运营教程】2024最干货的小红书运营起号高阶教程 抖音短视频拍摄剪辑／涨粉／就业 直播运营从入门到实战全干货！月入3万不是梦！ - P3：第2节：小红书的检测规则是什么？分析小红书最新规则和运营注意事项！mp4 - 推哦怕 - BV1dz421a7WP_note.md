# 小红书运营教程：P3：第2节：小红书平台检测规则详解 🔍

在本节课中，我们将详细解析小红书的平台检测规则。了解这些规则是账号安全运营、避免违规限流的基础。我们将从检测范围、设备管理、内容规范到原创度与敏感词排查，系统地学习如何合规地在小红书平台进行内容创作与运营。

## 平台检测的核心范围

小红书平台主要对以下几个方面的内容进行检测，以确保社区环境的健康与合规。

以下是平台重点检测的四个区域：

1.  **个人资料、笔记内容、私信以及评论**。在这些区域，不得含有微信号码、微信二维码、代购、水印，以及推广淘宝店铺等营销词汇。
2.  **手机设备**。官方建议，同一台设备最多只登录三个账号，超出此数量可能引发风险。
3.  **笔记内容**。内容中不得诱导用户，例如使用“关注领奖”、“点赞抽奖”等话术，这类行为被平台明令禁止。
4.  **内容原创度**。小红书对原创内容的保护程度较高。一篇笔记最好只由一个账号发布，避免同一内容在多个账号重复发布。

需要注意的是，平台在个人资料部分管理相对宽松，官方允许填写邮箱。运营者也可以使用微博ID或谐音词进行间接引流，再通过话术引导至微信。关于具体的引流技巧，我们将在后续课程中详细讲解。

## 敏感词检测与规避方法

除了上述通用规则，小红书还有一套严格的敏感词检测机制。这是内容审核的关键环节，直接关系到笔记能否成功发布。

敏感词通常涉及一些特定领域，例如手机卡、金融、电子烟、代购、医疗用品等。在创作时，应尽量避免触碰这些领域。

那么，如果在写作时不慎加入了未知的敏感词，该如何处理呢？这时就需要借助敏感词检测工具。下面介绍两个实用的检测网站及其使用方法。

**第一个网站：在线检测工具**

这个工具使用简便，无需注册。你可以将写好的笔记内容粘贴到检测框中，进行快速筛查。

例如，我们输入一段包含“代购”一词的文本。

```
这是一篇关于海外购物的分享，其中提到了代购相关信息。
```

点击“立即检测”后，工具会高亮显示“代购”为敏感词。这样，你就能在发布前及时修改或替换该词汇。
