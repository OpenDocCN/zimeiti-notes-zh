# 视频号创作分成计划保姆级教程：P14：炫酷三分屏卡点效果制作 🎬

在本节课中，我们将学习如何使用剪映制作一个炫酷的三分屏卡点视频效果。这个效果能让你的视频节奏感更强，视觉上更具冲击力。我们将从导入素材开始，逐步完成音频踩点、照片对齐、动画与蒙版设置，最终实现动态的三分屏切换。

---

## 第一步：导入素材与音频踩点 🎼

![](img/c37d8b36306a5d2102c12edda29f65cf_0.png)

首先，打开剪映App，点击“开始创作”，导入你准备好的照片素材。接着，为视频添加背景音乐。

以下是音频踩点的具体步骤：
1.  选中已添加的音频轨道。
2.  点击工具栏中的“踩点”功能。
3.  选择“自动踩点”中的“踩节拍Ⅱ”，让软件自动识别音乐节拍。
4.  点击对勾确认。

![](img/c37d8b36306a5d2102c12edda29f65cf_2.png)

完成踩点后，时间轴上会出现一系列黄色标记点，这些就是音乐的节拍点。

上一节我们完成了音频的准备工作，本节中我们来看看如何将照片与节拍对齐。

![](img/c37d8b36306a5d2102c12edda29f65cf_3.png)

## 第二步：调整照片时长与位置

接下来，我们需要将照片的时长与音频节拍对齐，这是实现卡点效果的基础。

操作步骤如下：
1.  选中主轨道上的第一张照片。
2.  拖动照片的右侧边缘，将其时长拉长，使其结尾与时间轴上的**第三个黄色节拍点**对齐。

这样，第一张照片就占据了前三个节拍的时间。

在将照片与节拍对齐后，我们需要为它添加动态效果，让它“动起来”。

## 第三步：添加入场动画与蒙版

![](img/c37d8b36306a5d2102c12edda29f65cf_5.png)

为了让画面出现时有动感，我们需要为照片添加一个入场动画，并用蒙版将其分割。

以下是具体操作：
1.  选中主轨道的照片，点击工具栏中的“动画”。
2.  选择“入场动画”，在列表中找到“动感放大”效果。
3.  将动画时长设置为 **`0.6秒`**，然后点击对勾确认。
4.  返回上一级工具栏，点击“蒙版”选项。
5.  选择“镜面蒙版”，屏幕上会出现两条可调整的线。
6.  将蒙版**缩小**并**向上移动**，使其占据画面上方约三分之一的位置。
7.  点击对勾确认设置。

现在，主轨道的照片已经处理好了。为了创建三分屏效果，我们需要复制并处理更多的照片层。

## 第四步：复制并创建画中画轨道

三分屏效果需要三个画面同时出现，我们将通过复制和“画中画”功能来实现。

请按以下步骤操作：
1.  选中主轨道上已设置好的照片，连续**复制6份**。
2.  选中复制出来的第一张照片，点击工具栏中的“切画中画”。
3.  将这张切下来的画中画照片（画一轨道），拖动其开始位置，使其与时间轴上的**第二个黄色节拍点**对齐。

现在，我们有了两个轨道的照片。接下来处理画一轨道的蒙版。

![](img/c37d8b36306a5d2102c12edda29f65cf_7.png)

## 第五步：调整画中画蒙版位置

我们需要调整画一轨道照片的蒙版位置，使其出现在屏幕中部，与主轨道照片形成上下排列。

操作如下：
1.  选中画一轨道的照片，点击“蒙版”。
2.  将镜面蒙版**向下移动**，并适当**放大**一些。
3.  调整蒙版大小和位置，使其与主轨道照片的蒙版区域之间**保留一条细边**。
4.  点击对勾确认。

为了完成三分屏，我们需要创建第三个画面轨道。

## 第六步：创建第三个画面轨道

我们将通过复制画一轨道的照片来快速创建第三个轨道。

以下是步骤：
1.  选中画一轨道的照片，再次**复制6份**。
2.  从这6份复制的照片中，将最后一张拖动到**画二轨道**上（即第三个轨道）。
3.  将画二轨道上这张照片的开始位置，拖动到比画一轨道照片**错后一个节拍点**的位置。

![](img/c37d8b36306a5d2102c12edda29f65cf_9.png)

现在，三个轨道都已就位，但画二轨道的蒙版还需要调整。

## 第七步：调整画二轨道蒙版

我们需要将画二轨道的蒙版移动到屏幕下方，完成最终的三分屏布局。

![](img/c37d8b36306a5d2102c12edda29f65cf_11.png)

请按以下步骤操作：
1.  选中画二轨道的照片，点击“蒙版”。
2.  将镜面蒙版**继续向下移动**，并调整其大小。
3.  确保它与画一轨道的蒙版区域之间也**保留一条细边**。
4.  点击对勾确认。
5.  选中画二轨道这张已调整好的照片，**复制5份**。
6.  将所有轨道（主、画一、画二）上照片的结束位置，统一拖动至与**音频的结尾**对齐。

至此，三分屏的骨架和动态效果已经搭建完成。最后一步是替换照片内容。

## 第八步：替换所有照片素材

为了让视频内容更丰富，我们需要将后续节拍点上的照片全部替换成新的图片。

![](img/c37d8b36306a5d2102c12edda29f65cf_13.png)

最终操作如下：
1.  将**三个轨道上**位于第二个节拍位置的第二张照片，同时选中并“替换”成另一张新照片。
2.  后续的每一个节拍点，都按照此方法，将三个轨道对应的照片依次替换成新的图片。

---

本节课中我们一起学习了制作炫酷三分屏卡点视频的全过程。我们首先通过**音频踩点**确定节奏，然后利用**镜面蒙版**分割画面，再通过**复制**和**切画中画**功能构建三个平行的画面轨道，并通过错开入场时间形成递进动画。最后，通过**批量替换**照片完成内容填充。掌握这个技巧，你可以轻松为自己视频号的视频增加专业级的节奏感和视觉吸引力。
