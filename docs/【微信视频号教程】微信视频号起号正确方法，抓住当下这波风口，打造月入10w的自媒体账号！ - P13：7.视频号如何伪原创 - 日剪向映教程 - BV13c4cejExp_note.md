## 平台原创扶持政策

上一节我们介绍了发布作品的基础，本节中我们来看看平台对原创内容的支持。平台本身支持原创，并有创作者激励计划，对原创内容提供流量倾斜。

以下是创作者激励计划的具体内容：
*   **流量扶持**：平台计划让1000万原创作者获得流量。
*   **收入激励**：计划让百万优质创作者获得收入。
*   **零粉阶段**：平台会赠送2500到5万点的流量券，相当于直接推送观众。
*   **千粉阶段**：可加入重点作者扶持计划，优质视频可申报专项奖金。
*   **万粉阶段**：可继续申报专项奖金，百万粉丝级别则有更多权益。

因此，原创内容能获得平台一级一级的流量池推进。当然，原创需要个人具备拍摄、编写台本等技术能力。

当然，搬运有一套流程。你可以先找到相关平台为自己寻找灵感。

以下是可用于寻找灵感的平台：
*   西瓜数据
*   有望数据
*   微信指数

例如，在视频号的排行榜中，可以查看各类目下表现优秀的账号。

如果你想做影视剧类账号，可以查看影视剧排行榜，点进去学习借鉴优秀账号的内容。音乐类账号同理。这就是为自己寻找灵感、观察易火内容的方法。有望数据就是这样一个工具。

接下来，我们详细讲解伪原创的流程。首先，你需要进行选品。

**流程公式**：`确定售卖商品` -> `寻找相关视频` -> `去水印与二次剪辑` -> `发布`

先确定你要卖什么东西。确定好要带的货之后，再去找相关的视频内容。

在视频号APP中，点击右下角的“我”进入个人主页。在页面中找到并点击“创作者中心”。

## 伪原创操作流程

在搬运时，可以先在其他平台（如抖音）寻找感觉不错的账号。当你看到一个不错的账号时，还需要检查一下他的内容是否已经在视频号被人搬运过。如果已经有人搬运，你再发就重复了。

假设我发现目标账号的内容在视频号还没有人搬运，那我就确定以这个账号为素材来源。

找到目标作品后，不是直接下载发布。首先需要给它去水印。

![](img/6f5a420013d735b06aecf7d2749d48d9_13.png)

记住，平台支持原创。如果你直接搬运带有他人水印的作品，账号可能会被处罚。因此，伪原创需要进行“伪”和“创”的处理。

**操作代码逻辑**：
```plaintext
if 发现优质视频源:
    if 该视频在视频号未被搬运:
        执行 去水印操作
        执行 二次剪辑操作 (如裁剪、变速、添加滤镜/字幕/背景音乐)
        发布视频
    else:
        寻找其他视频源
```

去水印后，还需要进行二次剪辑，例如裁剪画面、调整播放速度、添加滤镜、字幕或新的背景音乐等，让视频看起来像是经过自己处理的。
