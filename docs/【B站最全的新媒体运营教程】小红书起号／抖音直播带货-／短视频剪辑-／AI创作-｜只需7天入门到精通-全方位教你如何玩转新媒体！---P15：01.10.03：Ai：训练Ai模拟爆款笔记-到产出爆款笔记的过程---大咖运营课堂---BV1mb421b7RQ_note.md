# 新媒体运营：P15：01.10.03：AI：从分析爆款笔记到生成爆款笔记 🧠

在本节课中，我们将学习如何利用AI工具（如ChatGPT）分析平台上的爆款内容，并基于分析结果，指导AI生成具备爆款潜质的新笔记。整个过程分为两步：**分析学习**与**模仿创作**。

上一节我们介绍了通过“喂资料”的方式让AI理解特定平台的风格。虽然上一节的旅游攻略案例效果不理想，但方法是正确的。本节中，我们来看看如何更具体地分析爆款笔记，并让AI学习其成功要素。

## 第一步：分析爆款笔记 📊

![](img/76aa1391f2c506e9fc0aa3aec201244b_3.png)

![](img/76aa1391f2c506e9fc0aa3aec201244b_5.png)

我们的目标是让AI深入理解一篇笔记为何能成为爆款。需要分析以下几个核心点：

![](img/76aa1391f2c506e9fc0aa3aec201244b_7.png)

![](img/76aa1391f2c506e9fc0aa3aec201244b_9.png)

![](img/76aa1391f2c506e9fc0aa3aec201244b_11.png)

以下是需要AI重点分析的几个方面：

![](img/76aa1391f2c506e9fc0aa3aec201244b_12.png)

![](img/76aa1391f2c506e9fc0aa3aec201244b_14.png)

![](img/76aa1391f2c506e9fc0aa3aec201244b_16.png)

1.  **爆款因素**：找出内容中可能促使其火爆的关键点（例如：情感共鸣、实用干货、逆袭故事等）。
2.  **整体结构**：分析文案的叙事框架。
    *   **开篇**：用什么方式（如提问、制造焦虑、引用名人）吸引用户继续阅读。
    *   **中间**：提供了哪些内容来戳中用户痛点，提升用户停留时长。
    *   **结尾**：如何设置“钩子”（如鼓励互动、引导私信、提供福利）促使用户行动。
3.  **关键词策略**：分析笔记中是否刻意植入了特定关键词，以及这些关键词的出现频率和分布方式。

![](img/76aa1391f2c506e9fc0aa3aec201244b_18.png)

### 操作演示：以考研领域为例

我们选取一篇小红书平台上点赞数较高的考研笔记作为分析样本。

![](img/76aa1391f2c506e9fc0aa3aec201244b_20.png)

![](img/76aa1391f2c506e9fc0aa3aec201244b_22.png)

首先，我们需要给AI下达清晰的分析指令。指令模板如下：

![](img/76aa1391f2c506e9fc0aa3aec201244b_24.png)

![](img/76aa1391f2c506e9fc0aa3aec201244b_26.png)

```text
你是一个新媒体运营专家。我将提供一篇小红书上的爆款笔记文案，请你分析：
1. 它的爆款因素有哪些？
2. 分析它的整体结构：开篇如何吸引用户？中间如何戳中痛点？结尾如何引导互动？
3. 分析该笔记是否设计了关键词。如果有，指出是哪个关键词，并说明其出现频率。
```

![](img/76aa1391f2c506e9fc0aa3aec201244b_28.png)

接着，将找到的爆款笔记全文复制给AI。AI经过分析后，可能会给出如下反馈：

![](img/76aa1391f2c506e9fc0aa3aec201244b_30.png)

![](img/76aa1391f2c506e9fc0aa3aec201244b_32.png)

![](img/76aa1391f2c506e9fc0aa3aec201244b_34.png)

*   **爆款因素**：情感共鸣（开头营造焦虑）、逆袭故事、提供详细备考计划、使用表情符号增加亲和力、结尾正能量鼓励。
*   **整体结构**：开篇用“备考焦虑”引发共鸣 → 中间分享个人经验和具体方法 → 结尾鼓励并引导提问。
*   **关键词分析**：本篇笔记未发现明显刻意设计的关键词。

## 第二步：整合规律并生成大纲 📝

![](img/76aa1391f2c506e9fc0aa3aec201244b_44.png)

![](img/76aa1391f2c506e9fc0aa3aec201244b_46.png)

![](img/76aa1391f2c506e9fc0aa3aec201244b_48.png)

![](img/76aa1391f2c506e9fc0aa3aec201244b_49.png)

在分析了多篇爆款笔记后，我们需要让AI总结出共性规律。

![](img/76aa1391f2c506e9fc0aa3aec201244b_51.png)

以下是给AI的指令：

![](img/76aa1391f2c506e9fc0aa3aec201244b_53.png)

![](img/76aa1391f2c506e9fc0aa3aec201244b_55.png)

![](img/76aa1391f2c506e9fc0aa3aec201244b_57.png)

```text
请将你之前分析过的几篇考研爆款笔记的爆款因素总结一下，去重后整合成一个清晰的列表。
```

![](img/76aa1391f2c506e9fc0aa3aec201244b_59.png)

![](img/76aa1391f2c506e9fc0aa3aec201244b_60.png)

![](img/76aa1391f2c506e9fc0aa3aec201244b_62.png)

![](img/76aa1391f2c506e9fc0aa3aec201244b_64.png)

AI可能会总结出如下爆款因素列表：
*   情感共鸣（焦虑、逆袭）
*   明确的备考计划与时间节点
*   针对性的实用建议与资源推荐
*   使用表情符号增强感染力
*   正能量鼓励与互动引导

![](img/76aa1391f2c506e9fc0aa3aec201244b_65.png)

![](img/76aa1391f2c506e9fc0aa3aec201244b_67.png)

![](img/76aa1391f2c506e9fc0aa3aec201244b_68.png)

掌握了这些规律后，我们就可以指导AI进行创作了。首先，让它根据这些爆款因素，规划一篇新笔记的框架。

以下是生成大纲的指令：

```text
请结合你总结的爆款笔记因素，为我规划一篇关于考研的小红书笔记大纲。大纲需要细分到具体的小节点。
```

![](img/76aa1391f2c506e9fc0aa3aec201244b_70.png)

AI生成的大纲可能包含：
1.  开篇：用引人入胜的话语或问题吸引注意。
2.  引入主题：简要说明笔记内容。
3.  逆袭故事：分享个人经历，引发共鸣。
4.  专业选择建议。
5.  分科备考规划（政治、英语、专业课）。
6.  时间管理与心态调整。
7.  结尾：鼓励互动，引导关注或私信。

![](img/76aa1391f2c506e9fc0aa3aec201244b_72.png)

## 第三步：根据大纲生成完整文案 ✍️

![](img/76aa1391f2c506e9fc0aa3aec201244b_74.png)

大纲确认无误后，即可命令AI生成完整的笔记文案。此时需要给出更细致的写作要求。

以下是生成完整文案的指令：

```text
请严格按照上述大纲和爆款笔记因素，撰写完整的文案。
要求：
1. 文案结构需符合爆款规律。
2. 包含一个简短的“逆袭小故事”引入情绪。
3. 自然地融入一定量的emoji表情。
4. 尝试以“中文分词法”的思路，融入“考研”等相关关键词。
```

AI根据指令生成的文案，开头可能是这样的：
> “当你曾经迷茫，觉得时间不够用，我理解那种感觉…但我也亲身经历了逆袭的过程。让我告诉你，只要下定决心，考研绝非遥不可及！💪”

![](img/76aa1391f2c506e9fc0aa3aec201244b_76.png)

![](img/76aa1391f2c506e9fc0aa3aec201244b_77.png)

![](img/76aa1391f2c506e9fc0aa3aec201244b_79.png)

正文部分则会按照大纲，填充具体的备考计划、时间节点和实用建议，并在文中穿插表情符号和关键词。

**请注意**：与AI对话时的措辞会影响产出。例如，“包含丰富的emoji”和“包含emoji”会导致表情符号的数量不同。需要不断调整指令以获得最佳效果。

## 总结与进阶 🚀

本节课中，我们一起学习了利用AI辅助内容创作的核心流程：

1.  **分析学习**：提供爆款笔记，让AI分析其**爆款因素**、**结构**和**关键词策略**。
2.  **总结规律**：让AI整合多篇笔记的分析结果，提炼共性爆款元素。
3.  **模仿创作**：基于总结的规律，先让AI生成内容**大纲**，再根据大纲写出**完整文案**。

通过这个流程，即使是新手，也能借助AI快速产出符合平台调性、具备爆款潜力的内容草稿。当然，AI生成的初稿可能仍需人工优化和润色，以使其更精炼、更丰富。

![](img/76aa1391f2c506e9fc0aa3aec201244b_81.png)

**进阶提示**：使用更强大的模型（如GPT-4）或创建自定义的**GPTs**助手，通过上传你整理的优秀文案范例、专属表情包列表等资料，可以打造出更懂你需求的私人AI创作助手，从而进一步提升内容质量和创作效率。
