# 抖音运营入门：P7：抖音短视频的前景 📈

![](img/8ae3124326d6b658c3e6bb5b85551141_1.png)

在本节课中，我们将要学习抖音短视频平台的发展前景，分析其市场潜力与用户生态，并探讨普通人如何抓住其中的机会。

![](img/8ae3124326d6b658c3e6bb5b85551141_3.png)

了解抖音短视频的前景，需要先通过一张图片来认识。

![](img/8ae3124326d6b658c3e6bb5b85551141_1.png)

这张图片是抖音创作者大会的一份数据报告。报告显示，有超过2200万人在抖音的合计收入超过了417亿元。抖音希望将这个数字翻一番，让创作者的总收入达到800亿元。

![](img/8ae3124326d6b658c3e6bb5b85551141_3.png)

这代表了巨大的流量扶持。这里介绍第一个知识点：抖音平台上有两类人。第一类人叫做“玩家”，他们单纯地玩抖音，没想过从平台上赚钱。另一类人，可以称作“创作者”，他们希望通过制作内容来获取收益。区分这两类人对后续理解很重要。

那么，为什么我们要选择抖音短视频？在众多短视频平台中，为何专注于抖音？这类似于开实体店选址。做短视频一定要选择一个流量大的平台。平台自身流量不足，就无法分配流量给创作者，创作者也就难以赚钱。

抖音每日的活跃用户超过8亿，月活跃用户更是达到了40亿人次。这是一个极其庞大的流量池。流量大，竞争压力是否也会很大？答案是不会。平台上的创作者大约只有5000万。创作者的定义并非必须是网红或达人。例如，邻居家的老太太用手机拍摄自己做的饭菜并上传到抖音，她也算创作者。因此，90%的竞争压力实际上来自于身边的普通人。

接着，做抖音短视频是否需要购买电脑或摄像机？答案是不需要。只需要一部智能手机和一个手机号，就可以注册抖音账号。抖音倡导全民通过短视频的形式分享和记录生活，门槛很低。

那么，当前有什么样的风口值得我们去尝试？首先，抖音计划扶持1000名优质的艺术创作者，帮助他们涨粉超过100万。有的同学可能会说，自己不会创作。这不用担心，正是因为不会，才需要学习。虚心请教专业人士并不丢人。在你不懂的领域，对方就是你的老师，这能让你少走很多弯路。

市场空间巨大，内容产出也相对容易。现在是大数据时代，平台会根据用户的喜好推荐内容。你喜欢看什么，抖音就会给你推荐什么。因此，无论你的身份是宝妈、实体店主还是普通上班族，你分享的内容（如宝宝日常、产品展示、生活记录）总会有一部分人喜欢。之所以你的粉丝不多，可能是因为喜欢你的人还没有刷到你的作品。

所以，对于新手来说，首要目标不应该是急于涨粉，而应该是**先把流量和播放量做起来**。播放量提升后，粉丝自然会增长。

本节课中我们一起学习了抖音平台的巨大流量前景、用户分类（玩家与创作者）、低门槛的参与条件以及新手运营的核心策略：优先提升内容流量。理解这些是成功开启抖音运营之旅的第一步。
