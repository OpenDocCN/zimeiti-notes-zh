# 抖音运营实战：P6：人物设定

![](img/6ef2303ab999379a8ef014217152607f_0.png)

## 概述
在本节课中，我们将要学习抖音账号运营中一个至关重要的环节——**人物设定**。我们将探讨如何像设计游戏角色一样，为你或你的账号策划一个真实、立体且富有吸引力的人格化形象，从而解决粉丝信任、促进传播并增强变现能力。

上一节我们介绍了账号定位的基础，本节中我们来看看如何为你的账号注入灵魂，即**打造人设**。

![](img/6ef2303ab999379a8ef014217152607f_8.png)

在抖音中策划人设，就像在游戏中创建角色一样，需要提前进行设定。这包括称呼、职业、技能点等元素。

![](img/6ef2303ab999379a8ef014217152607f_10.png)

但与游戏角色不同的是，我们在抖音中设定的是**人生角色**，因此需要具备更高的真实感。

![](img/6ef2303ab999379a8ef014217152607f_12.png)

无论是短视频还是直播，一个成功的IP能够有效解决粉丝的信任问题。

![](img/6ef2303ab999379a8ef014217152607f_14.png)

它还能起到裂变传播的作用，并具备很强的变现潜力。

![](img/6ef2303ab999379a8ef014217152607f_16.png)

因此，我们运营账号，无论是创作内容还是策划选题，其核心目的之一就是在获取流量的同时**打造IP**。

![](img/6ef2303ab999379a8ef014217152607f_18.png)

打造IP的核心，是在特定的场景或情境中，完成**账号人格化**的过程。

这个过程需要注入情感、塑造故事、并贴上具有辨识度的标签。账号人格化，就是我们通常所说的**账号人设**。

对于账号人设的策划，我们可以从以下几个方面着手。以下是具体策划维度：

1.  **名字/昵称**：一个IP首先需要一个称呼，这是其传播的名片。它就像品牌的商标一样重要。需要明确这个IP叫什么。
2.  **职业/身份**：人格化必然具备身份。这是用户认识这个IP的联想入口，例如：`老师`、`教授`、`律师`。
3.  **社会称谓**：一个恰当的称呼能快速拉近与用户的距离，产生移情效果。例如：`XX哥`、`XX大爷`、`XX姥姥`。
4.  **形象/第一印象**：从用户视角塑造IP形象，使其更容易占据用户心智。例如：`大叔形象`、`铁憨憨`、`甜美可爱`。
5.  **性格/爱好**：一个具有鲜明特点的IP，记忆点强且更真实。过于完美的形象反而容易脱离大众。例如：`暴躁`、`娘炮`、`爱吃爱喝酒`。
6.  **穿着/社会符号**：借助社会普遍认知和刻板印象，快速在用户心智中定位IP。例如：`开豪车`代表成功人士，`打扮朴素`可能代表老师。

以上内容策划完成后，就可以结合你自身的产品或服务优势，将其落实到**账号搭建**中，即抖音账号的“五件套”：头像、昵称、背景图、简介、视频封面。

除了对账号进行人格化策划，还需要通过选题和内容策划，为你的人设提炼出**记忆点**，即**超级符号**。

超级符号的提炼可以从以下几个维度进行。以下是具体提炼方向：

*   **画风（视觉）**：如独特的场景、道具、服饰风格。
*   **听觉**：如标志性的BGM、特效音、配音声线特点。
*   **文案**：能够反映优势且容易记忆的提炼词，例如：`XX界天花板`、`简单粗暴`。
*   **表情/情绪**：如标志性的笑、惊恐、无奈等表情包式情绪传递。
*   **原特色**：如方言口音、标志性口头禅（触发音）、网络语、独特的说话节奏（慢条斯理/抑扬顿挫）。
*   **动作/习惯**：如习惯性的撩头发、扶眼镜等肢体动作。

超级符号提炼好之后，在你进行短视频创作时，就可以持续地在视频中重复呈现这些元素，不断强化用户记忆。

---

## 总结
本节课中，我们一起学习了抖音账号**人物设定**的核心方法与步骤。我们了解到，人设的本质是**账号人格化**，需要从名称、身份、形象、性格等多维度进行策划，并将其融入账号搭建。更重要的是，要通过提炼视觉、听觉、文案、行为等方面的**超级符号**，在内容中不断强化，从而打造出一个真实、独特、令人难忘的IP形象，为账号的长期发展与变现奠定坚实基础。
