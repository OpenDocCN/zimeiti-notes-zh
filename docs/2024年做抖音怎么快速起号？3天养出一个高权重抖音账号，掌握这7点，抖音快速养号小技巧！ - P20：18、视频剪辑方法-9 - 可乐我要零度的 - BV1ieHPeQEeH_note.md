## 基础剪辑步骤

![](img/963e701ec55a0906fdbb209f9e63a08e_21.png)

![](img/963e701ec55a0906fdbb209f9e63a08e_23.png)

以下是视频剪辑的核心步骤，我们将逐一拆解。

![](img/963e701ec55a0906fdbb209f9e63a08e_25.png)

### 1. 调整画面构图与比例
导入视频后，首先检查画面构图。如果画面方向或比例不正确，需要进行调整。

![](img/963e701ec55a0906fdbb209f9e63a08e_27.png)

![](img/963e701ec55a0906fdbb209f9e63a08e_29.png)

*   **操作**：选中视频轨道，向右滑动工具栏，点击“编辑”按钮（方块图标）。
*   **旋转**：点击左侧“旋转”功能，将画面调整到合适的方向。
*   **裁剪**：点击“裁剪”功能，可以选择 **1:1**、**4:3**、**16:9** 等比例，或进行自由裁剪。

![](img/963e701ec55a0906fdbb209f9e63a08e_31.png)

![](img/963e701ec55a0906fdbb209f9e63a08e_9.png)

![](img/963e701ec55a0906fdbb209f9e63a08e_33.png)

![](img/963e701ec55a0906fdbb209f9e63a08e_35.png)

调整完毕后，返回主编辑界面。

![](img/963e701ec55a0906fdbb209f9e63a08e_37.png)

### 2. 分割与删除无用片段
播放视频，剪掉开头、结尾或中间不必要的沉默、失误等部分。

![](img/963e701ec55a0906fdbb209f9e63a08e_39.png)

*   **操作**：将时间轴（白色竖线）移动到需要切割的位置，选中视频片段，点击工具栏中的“分割”按钮。视频会被切成两段。
*   **删除**：选中需要删除的片段，点击“删除”按钮即可。
*   **口诀**：`选中 -> 分割 -> 选中 -> 删除`。重复此流程修剪所有冗余部分。

![](img/963e701ec55a0906fdbb209f9e63a08e_41.png)

![](img/963e701ec55a0906fdbb209f9e63a08e_43.png)

![](img/963e701ec55a0906fdbb209f9e63a08e_15.png)

![](img/963e701ec55a0906fdbb209f9e63a08e_45.png)

![](img/963e701ec55a0906fdbb209f9e63a08e_47.png)

### 3. 添加与编辑背景音乐
视频剪辑好后，需要添加合适的背景音乐（BGM）。

![](img/963e701ec55a0906fdbb209f9e63a08e_49.png)

*   **添加音乐**：点击底部“音频”按钮。
    *   如果你有已下载的配音文件（如从淘金阁获取），可点击“提取音乐”导入。
    *   否则，点击“添加音乐”，在曲库中按分类（美食、美妆、情感等）选择，或直接搜索歌曲。点击歌曲可预览，选定后点击“使用”。
*   **剪辑音乐**：如果音乐过长，处理方法与剪辑视频相同：`选中音乐轨道 -> 分割 -> 删除`多余部分。
*   **调整音量**：选中音乐轨道，点击“音量”滑块，可以调整背景音乐的音量大小，避免盖过人声。

![](img/963e701ec55a0906fdbb209f9e63a08e_51.png)

![](img/963e701ec55a0906fdbb209f9e63a08e_27.png)

![](img/963e701ec55a0906fdbb209f9e63a08e_53.png)

### 4. 自动识别与添加字幕
手动添加字幕效率低下，剪映的自动识别功能可以快速完成。

![](img/963e701ec55a0906fdbb209f9e63a08e_55.png)

*   **操作**：点击底部“文字”按钮，选择“识别字幕”。
*   **设置**：在弹出窗口中，选择“仅视频”，以避免识别到背景音乐里的歌词。
*   **修改**：识别完成后，字幕会自动添加到时间轴。双击字幕条可以修改错别字。你还可以调整字幕的字体、颜色、样式等，使其与视频风格更匹配。

![](img/963e701ec55a0906fdbb209f9e63a08e_57.png)

![](img/963e701ec55a0906fdbb209f9e63a08e_37.png)

![](img/963e701ec55a0906fdbb209f9e63a08e_59.png)

至此，一个包含画面、音乐、字幕的完整视频就基本制作完成了。

![](img/963e701ec55a0906fdbb209f9e63a08e_61.png)

![](img/963e701ec55a0906fdbb209f9e63a08e_63.png)

### 5. 导出高清视频
在导出前，需要进行关键设置以保证视频清晰度。

*   **操作**：点击右上角“导出”按钮。
*   **参数设置**：在导出设置中，将“帧率”调整为 **30**，将“分辨率”拉到最高（如1080p或4K）。这是保证抖音视频清晰度的关键设置。
    *   **公式**：`导出清晰度 = 高分辨率 + 30帧率`
*   **注意事项**：导出过程中请勿锁屏或切换App，耐心等待导出完成。完成后，可以直接通过剪映分享到抖音。

## 进阶工具推荐：爱提词App 🎤

![](img/963e701ec55a0906fdbb209f9e63a08e_71.png)

对于需要出镜进行口播讲解的创作者，背诵稿子是一大难题。接下来，我们介绍一个能提升口播效率的实用工具。

![](img/963e701ec55a0906fdbb209f9e63a08e_73.png)

爱提词App可以帮助你实现“提词拍摄”，无需背诵。

![](img/963e701ec55a0906fdbb209f9e63a08e_75.png)

*   **创建台词**：打开App，点击右下角“+”号，新建一个台词稿。你可以直接输入或粘贴复制好的文案，并设置一个标题（如“情感语录”），然后保存。

![](img/963e701ec55a0906fdbb209f9e63a08e_77.png)

![](img/963e701ec55a0906fdbb209f9e63a08e_79.png)

![](img/963e701ec55a0906fdbb209f9e63a08e_67.png)

![](img/963e701ec55a0906fdbb209f9e63a08e_81.png)

*   **提词拍摄**：保存后，点击“开始提词”，选择“录制模式”。此时，台词会以悬浮窗形式滚动显示在屏幕上，你可以看着台词直接录制，无需死记硬背。
*   **个性化设置**：
    *   **字体大小**：点击屏幕中央的“设置”，可以调整台词字体大小，方便阅读。
    *   **美颜功能**：点击左下角的“美颜”按钮，可以开启瘦脸、大眼、磨皮、美白等效果，让拍摄画面更美观自然，且比一些美颜相机的效果更真实。
*   **灵感来源**：App内还提供“爆款文案”板块，涵盖职场、情感、生活等多个领域，可以作为内容创作的参考。
