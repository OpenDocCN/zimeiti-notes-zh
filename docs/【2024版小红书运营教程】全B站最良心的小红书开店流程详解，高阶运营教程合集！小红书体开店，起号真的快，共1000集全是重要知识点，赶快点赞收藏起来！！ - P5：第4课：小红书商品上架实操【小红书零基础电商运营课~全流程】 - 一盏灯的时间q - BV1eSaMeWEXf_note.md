# 小红书电商运营课：P5：第4课：小红书商品上架实操 🛒

在本节课中，我们将学习如何将商品上架到小红书店铺后台。这是进行销售的基础步骤，只有完成商品上架，消费者才能看到并购买你的产品。

## 准备工作：电脑与软件

![](img/28c42ecdfe238bb8465bee1662da971b_1.png)

上一节我们介绍了开店的基础流程，本节中我们来看看商品上架的具体操作。首先，进行商品上架操作，**必须使用电脑**。手机端虽然也能操作，但过程繁琐，效率极低，不利于项目的正式开展。电脑配置要求不高，只要能流畅打开网页即可。

其次，我们需要订购一个商品上架软件。手动上传商品图片效率低下，而使用专业软件可以实现一键复制搬运，极大提升效率。

![](img/28c42ecdfe238bb8465bee1662da971b_3.png)

以下是几款推荐的上架软件，功能类似，任选其一即可：
*   摸鱼助手
*   搞定商品
*   蚂蚁搬家
*   达卡宝

![](img/28c42ecdfe238bb8465bee1662da971b_5.png)

![](img/28c42ecdfe238bb8465bee1662da971b_6.png)

本节课将以目前提供**12周免费试用**的“达卡宝”软件为例进行演示。

## 软件订购与设置

![](img/28c42ecdfe238bb8465bee1662da971b_8.png)

现在，我们开始进入实操环节，学习如何订购软件并进行基础设置。

![](img/28c42ecdfe238bb8465bee1662da971b_10.png)

1.  **进入服务市场**
    登录小红书商家后台，点击顶部导航栏的“服务”，然后选择“服务市场”。

2.  **订购软件**
    在服务市场中搜索或找到“一键搬家”类目。选择“达卡宝”软件，点击“前往获取”并完成授权。接着，点击“免费试用商品搬家”，以**0元价格订购90天的试用服务**，并完成协议签署。

![](img/28c42ecdfe238bb8465bee1662da971b_12.png)

3.  **配置搬家设置**
    订购成功后，进入达卡宝的搬家界面。在开始搬运商品前，**务必先点击“搬家设置”进行配置**，以确保所有商品按统一规则上架。
    以下是关键的设置项：
    *   **运费模板**：选择“非偏远地区包邮模板”，避免向新疆、西藏等地区收取高额运费。
    *   **价格设置**：通常将“市场价”设置为进货价的3倍（例如，进货价100元，则设市场价为300元），“售卖价”设置为进货价的2倍（即200元）。公式表示为：`售卖价 = 进货价 * 2`。
    *   **发货时间**：设置为“48小时发货”。
    *   **售后服务**：务必勾选“支持7天无理由退货”。
    完成所有设置后，点击“确定”保存。

![](img/28c42ecdfe238bb8465bee1662da971b_14.png)

![](img/28c42ecdfe238bb8465bee1662da971b_15.png)

## 商品搬运与上架实操

配置好软件后，我们就可以开始从货源网站搬运商品了。最常见的货源网站是**拼多多**和**1688**。

![](img/28c42ecdfe238bb8465bee1662da971b_17.png)

![](img/28c42ecdfe238bb8465bee1662da971b_18.png)

1.  **复制商品链接**
    在拼多多等货源网站，找到你想销售的商品，复制其商品详情页的网址链接。

![](img/28c42ecdfe238bb8465bee1662da971b_19.png)

2.  **解决风控问题（如遇提示）**
    近期拼多多升级，可能需要安装浏览器插件才能采集。如果遇到提示，请按照指引下载并安装对应的采集插件（如使用360极速浏览器）。

![](img/28c42ecdfe238bb8465bee1662da971b_21.png)

![](img/28c42ecdfe238bb8465bee1662da971b_22.png)

3.  **采集与搬运**
    安装插件后，刷新拼多多商品页面，点击出现的“采集该商品”按钮。然后返回达卡宝的“复制记录”页面，查看商品搬运状态。

![](img/28c42ecdfe238bb8465bee1662da971b_23.png)

![](img/28c42ecdfe238bb8465bee1662da971b_24.png)

4.  **检查与上架**
    搬运完成后，在小红书商家后台的“商品列表”中找到刚搬运的商品。**必须点击“编辑”进行检查**，核心是核对“商品类目”是否准确。如果类目错误，需手动修正。
    确认商品信息无误后，点击“上架全部SKU”，该商品即可在店铺前台展示销售。

![](img/28c42ecdfe238bb8465bee1662da971b_26.png)

## 操作建议与总结

最后，我们来总结一下商品上架过程中的关键点和建议。

![](img/28c42ecdfe238bb8465bee1662da971b_28.png)

为了提高效率，请注意以下建议：
*   **批量操作**：如果一天需要上架多个商品，建议提前收集好所有商品链接，在软件中一次性添加并搬运，节省等待时间。
*   **仔细核对**：每次搬运后，务必进入商品列表检查类目等关键信息是否正确。

![](img/28c42ecdfe238bb8465bee1662da971b_30.png)

本节课中我们一起学习了小红书商品上架的完整流程。我们了解到使用电脑和上架软件是提升效率的关键，掌握了从订购软件、配置搬家规则，到从拼多多等平台搬运商品、检查并最终上架销售的全部步骤。记住，上架后仔细核对商品信息是保证店铺正常运营的重要一环。
