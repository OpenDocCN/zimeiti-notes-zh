# 抖音运营教程：P3：2 抖音选题思路技巧 🎯

## 概述
在本节课中，我们将学习一套高效的抖音选题公式。这套公式能帮助你快速确定视频内容方向，解决“不知道发什么”的难题，从而更精准地吸引目标用户。

---

## 选题的重要性
你现在做抖音时遇到的困扰，往往源于选题不当。不知道明天该发什么内容，会消耗大量时间和精力。

会找选题的人，能把更多时间投入到视频制作中。即使每天发布三条视频，一个月也能积累九十条内容。凭借数量优势就能在竞争中脱颖而出，无需额外技巧。此外，擅长选题的人能更精准地找到用户痛点，更容易吸引精准粉丝观看。

从而快速为账号打上标签。

## 解决方案：万能选题公式
有什么办法解决选题难题吗？当然有。这是一套百万粉丝博主常用的选题技巧。为了方便理解，我将其总结为一个公式。

**核心公式：选题 = 内容定位 + 用户痛点 + 借势热点**

用好这个公式。

![](img/7aa3cc142858dff6792da3832e868e75_5.png)

![](img/7aa3cc142858dff6792da3832e868e75_5.png)

你的内容方向一定不会跑偏。

## 公式应用详解
上一节我们介绍了核心公式，本节中我们来看看如何具体应用。

例如，你想和我一样成为一名知识博主，就需要挖掘行业内的关键词。像“播放量”、“粉丝量”、“定位”、“选题”等，都是关键词。这些词本身没有主观描述。然后，在这个基础上寻找用户的痛点。

例如，“便捷”、“高效”、“技巧教学”、“内容输出”等。最后，结合近期的热点事件，将这些元素进行任意组合即可。

眼下春节即将到来，你可以创作以下选题：
*   如何用拜年的方式，打造春节必火的内容。
*   张同学快速涨粉的三个秘诀。

## 跨行业应用案例
我们已经用知识博主行业演示了公式，现在将其套用在其他行业试试。

以美妆行业为例：
*   **内容定位**：美白、除皱、口红。
*   **用户痛点**：平价好用、明星同款、避坑扫雷。
*   **借势热点**：结合近期热门事件或人物。

进行排列组合后，可以产生如下选题：
*   迪丽热巴同款美白面膜推荐。
*   七夕千万别送女朋友的口红。
