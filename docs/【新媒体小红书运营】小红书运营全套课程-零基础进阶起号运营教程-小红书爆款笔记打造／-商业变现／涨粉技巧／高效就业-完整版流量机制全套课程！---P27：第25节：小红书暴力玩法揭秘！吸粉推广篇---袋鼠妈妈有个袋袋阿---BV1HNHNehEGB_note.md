# 小红书运营：P27：小红书暴力玩法揭秘！吸粉推广篇 🔍

在本节课中，我们将要学习小红书运营中的两种核心推广方式：自建账号矩阵与官方广告投放。我们将理清它们的关系，并详细介绍各自的运作模式与适用场景。

## 账号类型解析

上一节我们介绍了课程背景，本节中我们来看看小红书平台上的账号类型。了解账号分类是制定推广策略的基础。

小红书账号主要分为三类：

*   **企业号**：主要用于树立官方品牌形象。其核心功能包括展示官方认证标志、品牌名称与地址，并享有平台赋予的特权，便于开展商业活动。
*   **个人号**：这是大多数用户运营的账号类型。其特点是**灵活性高**，运营者可以是品牌员工或真实消费者。个人号便于塑造个人品牌特色，也适合品牌相关人物出镜。
*   **引流号**：这类账号在之前的课程中被称为“炮灰号”。其作用通常是辅助引流，可以**批量创建**，用于参与活动或评论互动，对账号本身的质量与规划要求不高。

## 企业号矩阵运营建议

了解了账号分类后，我们聚焦于如何系统化地运用它们。品牌推广通常结合企业号、个人号与引流号发布笔记进行引流。以下是打造企业号矩阵的具体建议：

*   **积极参与品牌活动**：品牌企业号发起的试用、团购等活动，可以动用多个小号（引流号或个人号）参与，增加活动热度与真实感。
*   **强化个人号品牌属性**：运营个人号时，需在笔记内容中**多次暗示**品牌的定位与调性，例如明确目标用户是城市青年还是职场新人。
*   **控制营销内容比例**：个人号发布品牌产品相关内容时，需注意频率。建议遵循 **“三七分”原则**，即30%的内容用于直接营销，70%的内容用于种草分享或知识科普。
*   **巧用引流号评论**：企业号进行引流时，可以配合使用引流号在其笔记评论区进行引导或补充说明，以实现更自然的转化。

## 官方广告投放详解

除了自建账号矩阵，另一种高效的推广方式是借助平台力量。本节我们来详细看看小红书官方的广告投放系统。

![](img/f4f116982c57cc81f9ae3e6cc7d127d7_1.png)

![](img/f4f116982c57cc81f9ae3e6cc7d127d7_3.png)

小红书官方广告主要指 **“小红书CBC广告”**，即竞价广告。广告主通过竞价购买广告位，以获得流量推荐。

![](img/f4f116982c57cc81f9ae3e6cc7d127d7_5.png)

### 广告申请流程

![](img/f4f116982c57cc81f9ae3e6cc7d127d7_7.png)

投放官方广告有一个前提：**必须拥有一个已认证的企业号**。

![](img/f4f116982c57cc81f9ae3e6cc7d127d7_9.png)

![](img/f4f116982c57cc81f9ae3e6cc7d127d7_11.png)

以下是申请与注册的关键步骤：

![](img/f4f116982c57cc81f9ae3e6cc7d127d7_13.png)

1.  **注册企业号**：访问小红书官网，在“业务合作”板块找到“企业号入驻”流程。按照指引提交相关资质文件并支付审核费用，通过后普通账号即可升级为官方认证的企业号。
2.  **申请广告投放**：拥有企业号后，在官网的“业务合作”中进入“推广合作”板块，即可按照流程申请开通广告投放权限。

### 广告类型与展示位

官方广告主要分为两大类：

*   **信息流广告**：当用户浏览小红书首页信息时，系统会**不定期**在信息流中插入广告。这种广告形式与原生内容混合，相对自然。
*   **搜索广告**：当用户使用关键词进行搜索时，系统会根据算法在搜索结果中展示相关的广告。例如，用户搜索“考研资料”，可能会看到能量补充食品的广告。其核心机制是**关键词竞价**，价高者获得的广告排位更靠前。

目前，信息流广告常见的固定展示位有：**第3位、第13位、第23位**。广告主可以针对这些位置进行竞价投放。

### 注意事项

![](img/f4f116982c57cc81f9ae3e6cc7d127d7_15.png)

官方广告投放涉及费用，且**成本通常较高**。平台官网提供了详细的常见问题解答（FAQ），建议在投放前仔细阅读。在资金允许的情况下，与官方合作投放竞价广告是获取精准流量的有效途径。
