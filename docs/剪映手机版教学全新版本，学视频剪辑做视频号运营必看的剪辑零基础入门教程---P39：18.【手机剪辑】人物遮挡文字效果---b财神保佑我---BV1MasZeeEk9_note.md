# 剪映手机版教学：P39：18. 人物遮挡文字效果 🎬

在本节课中，我们将学习如何使用剪映手机版制作一个“人物遮挡文字”的动态效果。这种效果能让文字仿佛被前景人物遮挡，随着人物移动而逐渐显现，常用于视频开场或转场，能有效提升视频的视觉吸引力。

---

## 一、制作文字边框素材 🖼️

上一节我们介绍了本节课的目标，本节中我们来看看如何制作核心的文字边框素材。

![](img/e6af0c5d4c464ee7874baf599af9fa63_1.png)

首先，在剪映中导入一段纯黑色背景。

点击底部工具栏的 **“文本”** 按钮，选择 **“新建文本”**。

![](img/e6af0c5d4c464ee7874baf599af9fa63_1.png)

在输入框中输入你想要的文字。接着，为文字选择一个合适的字体，并在“样式”中选择一个醒目的颜色（例如白色）。使用双指在屏幕上 pinch 手势，将文字放大到合适的大小。

接下来，我们需要为文字制作一个边框。以下是具体步骤：

1.  **添加白色底图**：点击底部工具栏的 **“画中画”**，选择 **“新增画中画”**，从相册导入一张纯白色的图片。
2.  **裁剪底图**：选中白色图片图层，点击 **“编辑”** -> **“裁剪”**。将图片裁剪成一个与文字大小相近的长方形比例。
3.  **对齐与调整**：将裁剪后的白色图片调整大小和位置，使其完全覆盖在文字上方。
4.  **应用蒙版**：选中白色图片图层，点击 **“蒙版”**，选择 **“矩形”** 蒙版。
5.  **反转蒙版**：点击蒙版参数中的 **“反转”** 按钮。此时，白色区域会变成边框，中间变为透明。
6.  **调整边框粗细**：拖动矩形蒙版边缘的箭头，向内收缩，直到白色边框达到你想要的粗细。可以上下左右微调，使其均匀。

至此，一个带有边框的文字素材就制作完成了。

最后，分别将文字图层和白色边框图层的时长都调整至大约 **6秒**。点击右上角 **“导出”** 按钮，将这段素材保存到你的手机相册备用。

---

## 二、合成最终效果 ✨

上一节我们准备好了文字边框素材，本节中我们将其与人物视频结合，完成最终效果。

在剪映中 **“新建一个项目”**，导入你准备好的、带有人物运动的视频素材作为主轨道。

以下是合成步骤：

![](img/e6af0c5d4c464ee7874baf599af9fa63_3.png)

1.  **导入文字素材**：点击 **“画中画”** -> **“新增画中画”**，导入刚才保存好的文字边框素材。
2.  **应用混合模式**：选中文字素材图层，点击 **“混合模式”**，选择 **“绿色”**。这样就去除了文字的黑色背景，只保留白色边框部分。

![](img/e6af0c5d4c464ee7874baf599af9fa63_3.png)

3.  **添加动画**：保持文字素材图层选中状态，点击 **“动画”** -> **“入场动画”**。在列表中找到并选择 **“镜像翻转”**，为文字添加一个旋转出现的动态效果。
4.  **处理人物图层**：选中主轨道的人物视频，点击 **“复制”**。然后选中复制出来的片段，点击 **“切画中画”**，将其拖到下方画中画轨道，并与主轨道的开头对齐。
5.  **智能抠像**：选中切下来的画中画人物图层，点击 **“智能抠像”**。软件会自动识别人物主体，去除背景。此时，这个人物图层就会遮挡住下方的文字，形成“人物在前，文字在后”的遮挡关系。
6.  **添加装饰**：你可以根据喜好，点击 **“文本”** 或 **“贴纸”**，添加一些辅助文字或装饰性元素，丰富画面。
7.  **添加背景音乐**：点击 **“音频”** -> **“音乐”**，选择一首合适的歌曲导入，并调整音量。

调整各图层的时长和位置，确保动画衔接流畅。点击播放预览，一个动态的人物遮挡文字效果就制作完成了。

---

## 总结 📝

本节课中我们一起学习了“人物遮挡文字”效果的制作方法。关键分为两步：**第一步**是利用蒙版功能制作一个镂空的文字边框素材并导出；**第二步**是在新项目中通过画中画、混合模式（绿色）、入场动画（镜像翻转）以及智能抠像功能的组合应用，将人物与文字素材合成，创造出人物移动时逐渐显现文字的视觉动态效果。掌握这个技巧，能为你的视频增添专业的转场和开场效果。
