# 新媒体全平台运营课程：P14：01.10.2：训练AI写出符合平台风格的文案 📝

在本节课中，我们将要学习如何训练AI（如ChatGPT）来撰写符合特定平台（如小红书）风格的文案。上一节我们介绍了如何让AI生成爆款标题，本节中我们来看看如何让AI理解并创作出符合平台调性的完整内容。

## 概述

AI能够生成大量内容，但要让其产出符合特定平台风格和用户喜好的文案，需要进行针对性的“训练”。核心在于向AI提供高质量的范例，让它学习该平台的文案风格、常用词汇、语气和内容结构。

## 训练AI理解平台风格

要让AI写出符合平台风格的文案，关键在于提供“学习材料”。你需要让AI分析该平台的爆款内容，从中提取风格特征。

以下是训练AI的核心步骤：

![](img/6d7fd88a36fc08bfa74d8c8701419f97_9.png)

![](img/6d7fd88a36fc08bfa74d8c8701419f97_11.png)

1.  **明确指令**：首先，你需要清晰地告诉AI它的任务。例如，你可以输入：“接下来我会给你提供一些小红书上的爆款笔记文案。你需要根据这些文案，分析小红书的文案撰写风格，包括特定的平台用语、词汇、语气等。”
2.  **提供范例**：向AI发送精选的、具有代表性的平台爆款文案。这些文案应能体现该平台的独特风格（如亲切分享、使用“宝子”等昵称、带有特定表情符号等）。
3.  **要求反馈**：在每次提供文案后，要求AI给出反馈，确认它已理解并分析出该文案的风格特点。这能确保学习过程有效。例如，指令中可以加上：“请在每次接收文案后告诉我你学习到了什么，并给出你对这篇文案风格、语气、类型的分析。”
4.  **持续“投喂”**：训练不是一次性的。在同一个对话中持续提供多样化的文案范例（如不同题材：旅游攻略、产品种草、避坑指南），能帮助AI建立更全面、深入的理解。AI会记忆对话上下文，从而积累关于该平台风格的知识库。

**核心指令示例（代码块）**：
```text
角色：你现在是一名精通小红书平台文案风格的助理。
任务：我将为你提供多篇小红书爆款笔记。请你分析并学习这些文案的以下特点：
1. 人称使用（如第一人称“我”的分享视角）
2. 常用语气词和平台特色词汇（如“宝子”、“绝绝子”、“踩雷”）
3. 文案结构（如痛点引入-亲身经历-总结建议）
4. 整体风格（如亲切、活泼、实用性强）
请在分析每篇文案后，用一句话总结你学到的风格要点。
```
## 实践演练：生成小红书风格旅游攻略

在让AI学习了足够多的范例后，就可以指令它进行创作。但创作效果取决于你提供的范例质量和指令的清晰度。

例如，在训练AI学习了一些旅游类笔记后，你可以发出创作指令：
```text
请结合你刚才学习到的小红书文案风格，为我撰写一篇具有爆款潜质的“张家界5日游攻略”小红书笔记。要求：
1. 使用第一人称分享视角，语气亲切。
2. 结构清晰，包含行程安排、必去景点推荐、避坑建议和美食分享。
3. 自然地融入小红书常用词汇和表情符号。
4. 同时，为这篇笔记生成10个符合小红书旅游赛道风格的爆款标题。
```

**要点分析**：
*   **指令需具体**：比起“写一篇攻略”，“写一篇包含行程、避坑建议和美食的攻略”指令更明确，产出更符合预期。
*   **范例质量决定产出质量**：如果提供给AI的范例多是简单的行程列表，它学到的就是罗列信息，难以产出有深度、带个人体验和避坑建议的“爆款”文案。
*   **分步优化**：对于复杂内容（如详细旅游攻略），可以分步指令AI。例如，先让它列出张家界所有重要景点，再针对每个景点生成介绍和注意事项，最后组合成一篇完整攻略。
*   **模型差异**：更高级的AI模型（如GPT-4）在理解复杂指令、学习风格和创造性方面通常表现更优。

## 关于标题生成的补充

在创作完整文案时，也可以要求AI一并生成标题。但为了获得更精准、高质量的标题，最佳实践是：
1.  开启一个**新的对话**。
2.  使用上一节课中提供的、针对标题优化的专用指令（例如，结合热点词、关键词、句式模板的指令）。
3.  在这个新对话中，直接指令AI生成所需数量的标题。这样可以避免受到当前对话中其他内容指令的干扰，让AI更专注于标题创作任务。

## 总结

本节课中我们一起学习了如何训练AI写出符合平台风格的文案。核心方法是：通过提供明确的指令和高质量的爆款文案范例，让AI学习特定平台的文案风格、用语和结构。训练是一个持续的过程，范例越丰富、越典型，AI的产出就越贴近平台调性。记住，清晰的指令和优质的“学习材料”是成功的关键。
