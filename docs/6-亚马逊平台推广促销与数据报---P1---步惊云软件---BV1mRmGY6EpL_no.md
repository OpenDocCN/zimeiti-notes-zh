# 亚马逊平台推广促销与数据报告：P1：引流转化与数据分析 📊

![](img/5d646c609b50850d5f3528636202c9dd_1.png)

在本节课中，我们将学习亚马逊平台的核心推广促销工具以及如何利用数据报告来优化店铺运营。卖家可以通过这些工具为店铺引流、提升转化率，并通过数据分析制定更有效的运营策略。

## A+页面：丰富产品详情页 🖼️

![](img/5d646c609b50850d5f3528636202c9dd_3.png)

首先来了解什么是A+页面。它是一个可以丰富商品详情页的工具。通过它，你可以使用额外的图片和文本，进一步完善商品的描述部分。

从网购者的视角来看，一款图片精美、文案布局工整、产品功能讲解细致的详情页能够快速吸引买家的目光，从而为店铺获取流量。而我们从前面的商品上架过程可知，亚马逊的产品长描述只能以纯文本方式介绍，不能添加图片，因此这部分描述通常显得单调，不易引人关注。

![](img/5d646c609b50850d5f3528636202c9dd_5.png)

如果使用A+页面进行完善，就能将以往无人关注的长描述区域，转变为展示品牌和产品优势的黄金位置。创建时，平台会提供几个不同的模板样式，你可以根据需要任选一个进行编辑排版。

可以在其中自定义编写信息，充分体现品牌、产品细节和公司理念等。A+页面设置完成后会非常美观。根据实际数据表明，带有A+页面的产品转化率比普通产品高出40%以上。

![](img/5d646c609b50850d5f3528636202c9dd_7.png)

但是要强调的是，并非所有卖家都有资格使用A+页面功能，只有品牌店铺才能够享有该项权限。实际操作中，需要卖家预先申请到品牌商标后，再将品牌在平台中进行登记与审核。在模拟平台中，需要大家在操作前完成“品牌旗舰店”下的“亚马逊品牌注册”流程。具体可参考流程说明指导，按照页面引导逐步完成店铺品牌的登记工作。完成以后，你就可以操作A+页面了。

![](img/5d646c609b50850d5f3528636202c9dd_8.png)

## 促销打折工具：提升引流转化率 🛒

![](img/5d646c609b50850d5f3528636202c9dd_10.png)

![](img/5d646c609b50850d5f3528636202c9dd_11.png)

上一节我们介绍了从完善产品详情页的角度进行店铺引流。此外，我们还可以配合一些促销打折工具来帮助提升引流后的转化率。

![](img/5d646c609b50850d5f3528636202c9dd_13.png)

以下是几种主要的促销工具：

![](img/5d646c609b50850d5f3528636202c9dd_15.png)

### 1. 购买折扣
购买折扣是亚马逊免费促销方式的一种，在“管理促销”模块下操作。它的功能是给产品做满减及折扣活动，即我们常听到的“满X减Y”这类常规促销方式。

在现实中，卖家一般会在新品刚推出、着力打造热销或爆款，或想将商品作为样品赠送给买家以获取好评等情况下使用此工具。

![](img/5d646c609b50850d5f3528636202c9dd_17.png)

### 2. 优惠券
优惠券的作用在于，设置了优惠券的商品会拥有独特亮眼的标识，能够快速吸引消费者的目光。同时，它对卖家的门槛要求比较低。在实际中，只要店铺评分不低于3.5星就可以使用，而且费用可控。

![](img/5d646c609b50850d5f3528636202c9dd_19.png)

因此，在模拟平台中对卖家无相关资质要求，与上面的购买折扣一样，任何卖家都可以操作该项功能。优惠券活动同样适用于刚上架不久、没有评价的商品，可以大大提升产品的点击率和转化率。

### 3. Prime专享折扣
亚马逊的Prime是一种VIP收费会员制度，类似于国内京东的Plus会员。购物者需要缴纳一定的会员费，即可享受亚马逊购物的增值服务。以美亚为例，目前美国亚马逊Prime会员的年费为139美金。

![](img/5d646c609b50850d5f3528636202c9dd_21.png)

参加Prime会员服务的消费者可以获得包括0元包邮、提前参加闪购、免费试听音乐，更能拿到会员专属折扣等多重增值服务。因此，Prime专享折扣是面向会员买家的折扣活动。

带有Prime专享折扣标志的商品会显示折扣价格，正常价格会被划掉，但这种价格普通买家看不到。其优势在于通过向会员买家提供优惠，促进下单，提升产品销量，并能帮助店铺快速积累一批忠实客户群体。在长期优化运营下，能吸引更多消费会员关注店铺。

此类活动最主要的要求在于，只有FBA配送状态下的商品才能设置会员折扣，而且折扣必须比实际价格优惠至少10%。此外，其他要求如店铺评分、卖家评分等，在实际平台中也需要达到一定条件。

![](img/5d646c609b50850d5f3528636202c9dd_23.png)

### 4. 秒杀活动
秒杀活动一般分为三种：LD（每日秒杀）、BD（7天秒杀）和DOTD秒杀。模拟平台中提供的是第一类LD秒杀。因为在实际中，另外两类秒杀活动一般卖家很难申请到，需要满足一定条件后通过官方招商经理或平台特邀才有相关权限。

![](img/5d646c609b50850d5f3528636202c9dd_25.png)

LD秒杀是一种限时促销优惠，活动时间通常为4到12小时（根据站点不同略有差异），具体活动时间由亚马逊官方排期决定。卖家每次参加秒杀活动都需要支付相应的秒杀费用，美国站的收费标准是每次150美金。

参加秒杀活动的商品需要采用FBA配送，而且折扣至少要达到8折。实际中，对于店铺评分和商品评价也有要求。

对于参加此类活动的卖家，建议首先明确自己的产品是否适合或满足参加秒杀活动的条件。秒杀活动并非免费，模拟平台中的LD秒杀，卖家每次参加都需缴纳150美金，费用较高。因此，可以选择一些客单价相对较高的店铺主打商品，价格太低廉的商品不太适合做秒杀。如果在现实中，还要保证产品质量，否则有了订单和流量后，如果出现差评、退货等问题，将得不偿失。

## 广告推广：付费引流工具 🔍

![](img/5d646c609b50850d5f3528636202c9dd_27.png)

最后要介绍的是亚马逊的广告推广工具，在平台中的“广告活动管理”中可以创建。它是平台推出来给卖家的一款付费引流工具。

![](img/5d646c609b50850d5f3528636202c9dd_29.png)

当购物者在前台通过搜索关键词并显示相关的产品结果时，部分产品会带有“Sponsored”标识，那就表示它们是亚马逊的广告产品。

广告类型有商品推广、品牌推广和展示型推广。目前，模拟平台支持前两种推广：
*   **商品推广**：根据卖家投放的商品关键词展示的站内广告。买家使用相关关键词搜索后，就能查看到相应的推广产品。
*   **品牌推广**：让卖家在广告中推广自己的品牌头条、Logo或至多三项品牌产品。这类广告会将买家引导到品牌商店首页或特定的着陆页。

收费方式均为点击扣费（CPC），平台会根据商家设定的竞价，在买家每点击一次后扣除相应的费用。

![](img/5d646c609b50850d5f3528636202c9dd_31.png)

投放时，可以任选自动投放还是手动投放：
*   如果选择**自动投放**，选定相应的推广商品后，不需要卖家设定关键词。亚马逊会根据商品信息中嵌入的关键词信息以及产品类目，自动匹配关键词。
*   如果选择**手动投放**，则需要卖家设定好相应的关键词，平台会根据关键词的匹配程度来决定是否展示你的广告。这种方式下，卖家还可以建立自己的否定关键词词库，即在特定关键词下不展示广告，从而避免浪费无关的广告费。

## 数据报告与资金管理 📈

以上就是亚马逊平台的几种推广和促销工具。建议大家在运营过程中，根据商品的销售情况来搭配各种活动。活动的设置原则是“短时多次”，即各类活动并非只需做一次，应该频繁进行。每次活动的时间可以设置得短一些，要根据商品销量判断活动效果，效果不佳的商品应及时调整或替换。

在“数据报告”模块下的“亚马逊销售指导”中，可以查看店铺所有商品变体在过去30天和14天的销售情况。或者，你也可以根据商品编号到“订单”模块中查看各个产品的销量情况。

在“业务报告”模块下，你还可以定位观察某一天针对FBA或自配送商品的销售数量和销售额，从而判断商品的销售走势，为店铺制定合适的运营策略。

另外，在“付款”模块下，可以查询店铺所有订单的入账情况。在“结算一览”中，显示的是店铺当前尚未提现的订单入账总资金。点击“请求付款”，可以将资金提现。

建议大家在运营过程中，每隔一段时间就将累积的资金提现出来。提现是提现到你所绑定的第三方收款账户中，只有提现出来才会算入你的公司总资产，否则不会计算。

![](img/5d646c609b50850d5f3528636202c9dd_33.png)

![](img/5d646c609b50850d5f3528636202c9dd_34.png)

在“所有结算”里会记录每一笔订单的收款情况。“付款”则统计卖家每一次提现金额的记录。
