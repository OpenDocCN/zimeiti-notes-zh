# 抖音运营教程：P17：11、视频类型1-2 📚

在本节课中，我们将学习抖音账号运营中关于“人设”的核心概念，并深入探讨两种主要的人设类型：强人设与弱人设。理解并选择适合自己的人设，是连接内容创作与粉丝增长、最终实现变现的关键一环。

上一节我们解决了文案问题，本节中我们来看看内容创作的下一个关键环节：建立人设。许多教程直接谈论涨粉与变现，但在这两者之间，还有一个至关重要的步骤——**立人设**。人设的目的并非为了炫耀，而是为了在众多同类创作者中脱颖而出。例如，有十位装修领域的创作者，如果其中一位（比如“小王同学”）建立了鲜明有趣的人设，观众就更容易记住他，从而忽略其他九位。当观众喜欢你、记住你时，他们自然就成为你的粉丝。

## 人设的两种类型：强与弱 🎭

现在我们来讲解这个知识点，请边听边记。人设主要分为两种：**强人设**和**弱人设**。区分清楚这两种人设后，你就能轻松判断任何视频所属的类型。

### 什么是强人设？

**强人设**的核心是以创作者**个人**为切入点，内容与创作者自身密切相关。

以下是强人设常见的切入点：

*   **个人经验与职业**：分享你所在行业的专业知识或经验。例如：装修、办公设备、房产、汽车、餐饮、婚介、法律咨询、电商等领域。
*   **个人兴趣与爱好**：分享你的兴趣爱好与生活方式。例如：美食、旅行、手工、宠物、星座占卜、美妆、护肤、服装穿搭、游戏、摄影等。

强人设的内容通常是分享个人的工作、生活或爱好，并且为了增强真实性与亲和力，**大部分强人设账号会选择真人出镜**。

### 什么是弱人设？

与强人设相对的是**弱人设**。弱人设的内容与创作者**个人**关联不大，核心目的是为了运营好账号本身。

以下是弱人设的典型例子：

*   **情感语录类账号**：发布配乐的情感文字图片或视频，无真人出镜或配音。
*   **影视解说类账号**：虽然有人声解说电影，但观众关注的是电影内容和解说本身，而非解说者个人。

**弱人设公式**：**内容主题** ≠ **个人**（核心是内容素材本身，如文案、影视、音乐等）

弱人设的创作更侧重于内容素材的整合与呈现，而非个人形象的塑造。

## 从人设到内容：讲故事与口播 🎬

理解了人设分类，我们来看看如何基于人设创作内容。只要内容与你个人相关，都可以作为视频素材。我们首先讲解一种常见且有效的内容形式：**讲故事**。

讲故事可以分享客户故事、从业经历或人生感悟。故事本身不要求完全真实，可以从文案平台寻找有趣的故事进行分享。而讲故事最常用的形式就是**口播**。

口播即用嘴巴讲述，它拥有丰富的分类，能满足不同人设的需求：

*   **讲故事型口播**：目的是让观众觉得有趣、有共鸣。
*   **知识分享型口播**：例如育儿、金融、养生、装修、房产、汽车等领域的知识科普。目的是让观众在几十秒内获得“收获感”，觉得学到了东西。

**口播代码示例（伪代码）**：
```
if (人设 == “强人设” && 领域 in [“知识分享”, “经验故事”]) {
    内容形式 = “口播”;
    拍摄方式 = “准备稿子 -> 面对镜头录制”;
}
```

### 关于口播的常见问题解答

以下是针对口播形式的几个关键解答：

*   **口才不好能做口播吗？**  
    可以。口播是**提前准备好稿子**再录制，并非临场发挥。大部分口播者的流畅表达都源于充分准备。
*   **普通话不好能做口播吗？**  
    可以。如果你的目标客户是同城或有地域属性，使用**方言**可能效果更好，能增加亲切感和趣味性。同时，可以配上字幕确保信息传达。
*   **口播如何拍摄？**  
    非常简单。用手机架好，准备好文案，直接录制即可。这是一种易于起步的内容形式。

本节课中我们一起学习了抖音账号运营中“人设”的重要性，并详细区分了**强人设**与**弱人设**。强人设以个人为核心，利于建立独特形象；弱人设以内容为核心，专注于账号运营。同时，我们介绍了基于人设进行内容创作的一种高效方式——**口播**，并解答了关于口播的常见疑虑。理解这些概念，将帮助你更清晰地规划自己的抖音内容方向。
