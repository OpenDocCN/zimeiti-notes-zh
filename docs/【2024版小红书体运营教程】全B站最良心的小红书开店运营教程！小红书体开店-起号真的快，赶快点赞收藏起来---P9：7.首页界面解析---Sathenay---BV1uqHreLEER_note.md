# 剪映专业版教程：P9：首页界面解析 🏠

在本节课中，我们将学习剪映专业版软件启动后的首页界面。我们将了解如何创建新项目、导入素材以及管理已有的剪辑草稿。

---

安装剪映专业版后，双击桌面快捷方式即可打开软件。首先看到的是首页界面。本节重点解析这个界面。在首页界面中，可以创建新的视频剪辑项目，方法是单击“开始创作”。

同时，也可以对已有的剪辑项目进行重命名、删除等基本操作。

![](img/bbc5e40eefedf7f2141cdfbfac23e1a3_1.png)

## 创建与管理剪辑项目

如何创建与管理剪辑项目呢？选择“开始创作”并单击，即可打开视频编辑界面。

![](img/bbc5e40eefedf7f2141cdfbfac23e1a3_3.png)

![](img/bbc5e40eefedf7f2141cdfbfac23e1a3_1.png)

![](img/bbc5e40eefedf7f2141cdfbfac23e1a3_4.png)

在编辑界面中，可以选择导入素材。单击“导入”按钮，会打开选择媒体资源的对话框。

![](img/bbc5e40eefedf7f2141cdfbfac23e1a3_3.png)
![](img/bbc5e40eefedf7f2141cdfbfac23e1a3_4.png)

![](img/bbc5e40eefedf7f2141cdfbfac23e1a3_6.png)

在对话框中，选择当前想要编辑的素材文件，然后点击“打开”。

![](img/bbc5e40eefedf7f2141cdfbfac23e1a3_6.png)

![](img/bbc5e40eefedf7f2141cdfbfac23e1a3_8.png)

所选素材随即被导入到软件的素材库中。此时，可以直接将素材从素材库拖拽到下方的时间轴上。

![](img/bbc5e40eefedf7f2141cdfbfac23e1a3_8.png)

![](img/bbc5e40eefedf7f2141cdfbfac23e1a3_10.png)

这样就完成了素材的调用，可以开始进行剪辑。

## 返回首页与项目管理

上一节我们介绍了如何导入素材，本节我们来看看如何返回首页并管理项目。如何返回首页界面呢？可以在顶部菜单栏中单击，选择“返回到首页”选项。

![](img/bbc5e40eefedf7f2141cdfbfac23e1a3_10.png)

也可以直接单击编辑界面右上角的关闭按钮（叉号），同样可以返回首页界面。

回到首页界面后，可以看到刚刚创建的剪辑项目，已被存放到“剪辑草稿箱”中。每个项目右侧有一个由三个小点组成的按钮，单击它可以对当前项目进行操作。

以下是可执行的基本操作列表：

*   **上传**：登录抖音账户后，可将项目上传至云空间。在另一台电脑登录相同账户后，可以下载并继续剪辑。
*   **重命名**：单击可修改项目名称。建议保持良好习惯，对剪辑好的视频进行重命名，方便日后查找和修改。
*   **复制草稿**：单击后会创建当前项目的一个副本。
*   **删除**：如果不再需要当前项目，可以单击删除。也可以使用键盘上的**空格键**或**删除键**来执行删除操作。

---

本节课中，我们一起学习了剪映专业版首页界面的核心功能。我们掌握了如何通过“开始创作”新建项目、如何导入并调用素材，以及如何返回首页对项目进行上传、重命名、复制和删除等管理操作。熟悉首页界面是高效使用剪映进行视频剪辑的第一步。
