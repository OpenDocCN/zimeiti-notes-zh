# 《自媒体全套运营教程》：P19：动漫分屏效果制作教程 🎬

![](img/fc6f37472bfc76503b203e76e37a934a_0.png)

![](img/fc6f37472bfc76503b203e76e37a934a_2.png)

在本节课中，我们将学习如何使用手机剪辑软件制作一个动漫分屏效果。这个效果常用于展示多部动漫作品，通过动态的入场和出场动画，让画面富有节奏感和视觉冲击力。我们将从导入素材开始，逐步讲解裁剪、画中画、动画设置等核心操作。

![](img/fc6f37472bfc76503b203e76e37a934a_4.png)

## 概述
本节教程将详细拆解一个动漫分屏视频的制作过程。我们将使用四张动漫海报作为素材，通过调整比例、应用画中画功能以及设置入场和出场动画，最终实现海报依次滑入、折叠退出的动态效果。

![](img/fc6f37472bfc76503b203e76e37a934a_6.png)

## 素材准备与裁剪
首先，我们需要准备视频背景和四张动漫海报图片。以下是操作步骤。

![](img/fc6f37472bfc76503b203e76e37a934a_8.png)

![](img/fc6f37472bfc76503b203e76e37a934a_10.png)

1.  点击“开始创作”，导入准备好的视频背景素材。
2.  点击“添加”，将四张动漫海报图片（例如：海贼王、火影忍者、鬼灭之刃、咒术回战）一并导入项目。

![](img/fc6f37472bfc76503b203e76e37a934a_12.png)

![](img/fc6f37472bfc76503b203e76e37a934a_14.png)

![](img/fc6f37472bfc76503b203e76e37a934a_16.png)

接下来，我们需要统一所有海报图片的比例。

![](img/fc6f37472bfc76503b203e76e37a934a_17.png)

1.  选中第一张海报，点击底部工具栏的“编辑”。
2.  选择“裁剪”功能。
3.  在比例选项中选择 **5:8**。
4.  调整图片的显示区域，然后点击确认。
5.  重复以上步骤，将剩余三张海报的比例都裁剪为 **5:8**。

## 应用画中画与层级调整
上一节我们完成了素材的裁剪，本节中我们来看看如何将它们叠加到视频背景上。

我们需要使用“画中画”功能将图片置于视频上方，并通过“层级”调整它们的上下顺序。

1.  在时间轴上选中一张海报，在底部工具栏找到并点击“切画中画”。
2.  重复此操作，将四张海报全部切入画中画轨道。
3.  点击任意画中画图层，在工具栏找到“层级”按钮。
4.  在层级面板中，通过拖拽调整四张图片的上下顺序。例如，将海贼王拖到最前，火影忍者第二，鬼灭之刃第三，咒术回战第四。

## 设置入场动画与错位出场
图片顺序调整好后，我们将为它们添加动态效果。以下是设置入场动画的步骤。

1.  选中“海贼王”画中画图层，点击工具栏的“动画”。
2.  选择“入场动画”，在列表中找到“向下滑动”效果。
3.  将动画时长设置为大约 **1秒**。
4.  为“火影忍者”、“鬼灭之刃”、“咒术回战”三个图层重复以上操作，均添加“向下滑动”的入场动画，时长统一为1秒。

为了让四张图片依次出现，我们需要错开它们的出现时间。

1.  将时间轴放大，便于精细操作。
2.  将“火影忍者”图层的起始位置，拖到时间轴上第 **10帧** 处。
3.  将“鬼灭之刃”图层的起始位置，拖到第 **20帧** 处。
4.  将“咒术回战”图层的起始位置，拖到第 **1秒** 处。

## 设置出场动画与时间交错
图片依次入场后，接下来让它们以折叠的方式依次离场。以下是具体方法。

首先为后三张图片添加出场动画。

1.  选中“火影忍者”图层，点击“动画”，选择“出场动画”。
2.  找到“向左滑动”效果，将时长设置为 **0.5秒**。
3.  为“鬼灭之刃”和“咒术回战”图层重复此操作，均添加时长为0.5秒的“向左滑动”出场动画。

为了让离场更有层次感，我们需要错开它们的出场动画开始时间。

1.  选中“咒术回战”图层，进入其“出场动画”设置。
2.  将动画开始的触发点提前，例如设置为 **0.8秒**（即入场结束后0.8秒开始向左滑出）。
3.  选中“鬼灭之刃”图层，将其出场动画触发点设置为 **0.6秒**。
4.  选中“火影忍者”图层，将其出场动画触发点设置为 **0.4秒**。
5.  “海贼王”图层无需出场动画，它将作为背景持续显示。

## 复制图层与延长展示
现在，我们需要让每张海报在滑出前，能作为主画面展示一段时间。以下是实现方法。

我们将通过复制画中画图层并调整来实现。

1.  选中“火影忍者”图层，点击“复制”。
2.  将复制出来的新图层，拖到时间轴后方，紧接在“海贼王”图层展示结束的位置。
3.  将新图层的位置移动到画面最左侧。
4.  选中这个新图层，点击“动画”，将其“出场动画”设置为“无”。
5.  回到前方的“海贼王”图层，将其时长拉长，确保在“火影忍者”新图层滑入的过程中，“海贼王”画面依然存在。
6.  延长“火影忍者”新图层的时长，直至“鬼灭之刃”即将出现的位置。

重复以上复制和延长的逻辑，处理“鬼灭之刃”和“咒术回战”图层，使每个动漫海报都能依次成为主画面并展示一段时间。

## 最终微调与总结
基本效果完成后，我们可以进行一些微调使视频更完美。例如，调整背景视频的位置，让主体人物在画中画出现时能居于画面视觉中心。

1.  在时间轴上选中背景视频，在需要调整的位置使用“分割”工具。
2.  选中分割后的视频片段，在预览窗口中拖动，将其向右侧移动，使人物居中。
3.  检查所有图层的时长，确保与背景视频对齐。

本节课中我们一起学习了动漫分屏效果的制作。大家需要重点掌握的核心操作包括：使用**画中画**功能叠加图层、通过**层级**管理上下关系、运用**入场动画**和**出场动画**创造动态效果，以及对图片进行**编辑->裁剪**来统一比例为 **5:8**。掌握这些技巧，你就能制作出属于自己的动态分屏视频了。
