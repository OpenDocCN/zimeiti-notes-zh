# 抖音运营教程：P26：18、视频剪辑方法-9 ✂️

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_1.png)

在本节课中，我们将学习如何使用手机版剪映App进行视频剪辑，并掌握从导入素材到导出成片的完整流程。同时，我们也会介绍一个对“口播”类视频非常有帮助的提词工具。

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_3.png)

## 剪辑软件准备与导入素材

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_5.png)

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_7.png)

上一节我们介绍了视频策划的思路，本节中我们来看看如何将素材剪辑成片。首先，你需要下载正确的剪辑软件。

请确保下载的是**剪映**（黑色图标版本），这是抖音官方的剪辑工具，有手机版和电脑版。本节课我们以手机版为例进行讲解。

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_9.png)

以下是开始剪辑的第一步操作：

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_11.png)

1.  打开手机版剪映App，点击首页的 **`开始创作`** 按钮。
2.  从相册中选择你想要剪辑的视频片段，点击 **`添加`** 将其导入项目。
3.  如果你的视频由多个片段（如开头、中间、结尾）组成，建议先单独剪辑好第一个片段。完成后，点击轨道右侧的 **`+`** 号，继续导入并剪辑下一个片段。这样可以保持工程清晰，不易混乱。

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_13.png)

## 基础剪辑：调整与裁剪

导入素材后，我们首先需要检查并调整视频的基础构图和比例。

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_15.png)

点一下时间轨道上的视频片段，下方工具栏会滑出编辑选项。向右滑动工具栏，找到并点击 **`编辑`** 按钮（图标通常是一个方框）。

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_17.png)

以下是调整视频的步骤：

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_19.png)

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_21.png)

1.  **旋转**：点击 **`旋转`** 按钮，将视频画面调整到正确的方向。
2.  **裁剪**：点击 **`裁剪`** 按钮，你可以选择不同的画面比例，如 **`9:16`** (抖音竖屏)、**`1:1`**、**`16:9`** 等，也可以手动拖动框线自由裁剪。调整完成后点击确认。

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_23.png)

基础画面调整好后，我们就可以开始修剪视频内容了。

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_25.png)

## 核心剪辑：分割与删除

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_27.png)

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_29.png)

剪辑的核心操作是“分割”与“删除”，用于去掉视频中多余、无效的部分。

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_31.png)

播放视频，找到需要剪掉的起始位置（例如开头沉默的部分）。将时间线（那条白色的竖线）移动到这个位置。

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_33.png)

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_35.png)

以下是修剪视频的操作流程：

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_37.png)

1.  **分割**：选中视频片段，点击工具栏中的 **`分割`** 按钮。视频会在此处被切成两段。
2.  **删除**：选中你想要删除的那一段视频（如开头的空白段），点击 **`删除`** 按钮。
3.  重复上述操作，去除中间不必要的部分和结尾的冗余画面。

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_39.png)

**操作口诀**：`选中 -> 分割 -> 选中 -> 删除`。掌握这个流程，你就能流畅地修剪视频。

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_41.png)

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_43.png)

## 添加与编辑背景音乐

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_45.png)

视频内容修剪完毕后，接下来需要添加背景音乐（BGM）来增强氛围。

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_47.png)

点击底部工具栏的 **`音频`** 按钮，进入音乐添加界面。

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_49.png)

以下是添加和编辑音乐的几种方法：

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_51.png)

1.  **提取音乐**：如果你已经用其他工具（如“淘金阁”）生成了配音文件，可以点击 **`提取音乐`**，直接从手机视频中提取音轨。
2.  **添加音乐**：点击 **`音乐`**，你可以使用剪映自带的音乐库。库内音乐按“美食”、“vlog”、“流行”等分类，方便查找。点击歌曲旁边的播放图标可以预览，选择后点击 **`使用`** 即可添加。
3.  **剪辑音乐**：如果添加的音乐过长，处理方法与剪辑视频相同：将时间线移到音乐结尾处，`选中音乐轨道 -> 点击分割 -> 选中多余部分 -> 点击删除`。
4.  **调整音量**：选中音乐轨道，点击 **`音量`** 按钮，可以拖动滑块调整背景音乐的音量大小，避免掩盖人声。

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_53.png)

## 快速添加与修改字幕

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_55.png)

为视频添加字幕可以提升观看体验，尤其是对于口播类内容。剪映提供了非常便捷的自动识别功能。

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_57.png)

点击底部工具栏的 **`文本`** 按钮，然后选择 **`识别字幕`**。

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_59.png)

以下是字幕处理的步骤：

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_61.png)

1.  在识别字幕界面，选择 **`仅视频`**（如果你不希望识别出背景音乐的歌词），然后点击开始识别。
2.  系统会自动识别视频中的语音并生成字幕条，效率非常高。
3.  识别完成后，检查字幕是否有错别字。**双击**字幕条即可进入编辑模式，进行修改。
4.  你还可以选中字幕，在工具栏中修改**字体**、**样式**（如颜色、描边）等，让字幕更美观。例如，可以根据衣服颜色搭配字幕颜色。

## 导出设置与发布

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_63.png)

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_65.png)

完成所有剪辑后，在导出前有一个关键设置，它决定了视频的清晰度。

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_67.png)

点击右上角的 **`导出`** 按钮。在导出设置页面，请进行以下调整：

*   **帧率 (FPS)**：设置为 **`30`**。这个帧率对于抖音平台最为合适。
*   **分辨率**：尽可能选择最高的选项（如1080p或4K）。分辨率越高，视频越清晰。

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_69.png)

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_71.png)

**导出口诀**：`帧率30，分辨率拉满`。设置好后，点击导出。导出过程中请勿锁屏或切换App，耐心等待完成即可。导出后，可以直接通过剪映分享到抖音。

## 辅助工具推荐：口播提词器

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_73.png)

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_75.png)

对于需要出镜讲话的“口播”类视频，背诵台词是一大难题。这里推荐一个实用的辅助工具：“爱提词”App。

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_77.png)

这款App可以让你在拍摄时看着提词器朗读，无需背诵。

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_79.png)

以下是“爱提词”的基本使用方法：

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_81.png)

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_83.png)

1.  下载并打开“爱提词”App，点击右下角的 **`+`** 号。
2.  输入标题（如“情感语录”），然后将你的台词稿复制或输入到正文区，点击保存。
3.  保存后，点击 **`开始提词`**，选择 **`录制模式`**。此时手机屏幕会像提词器一样滚动显示你的台词，你可以对着它直接朗读拍摄。
4.  在提词界面，点击中间的设置按钮，可以**调整字体大小**，方便近视的同学观看。
5.  该App也内置了拍摄功能。点击左下角的 **`美颜`** 按钮，可以适度调整瘦脸、大眼、磨皮等参数，让拍摄效果更自然美观，避免原相机的生硬或第三方美颜相机的过度失真。
6.  此外，App内还有一个“爆款文案”区，提供了职场、情感、生活等多个领域的文案参考，可以为你提供创作灵感。

![](img/1bd4fe6ad6807ffbf3f26beeb5ebe133_85.png)
