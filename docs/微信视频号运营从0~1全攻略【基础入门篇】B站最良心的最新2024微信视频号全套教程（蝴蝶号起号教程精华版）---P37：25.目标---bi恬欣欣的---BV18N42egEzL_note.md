拥有时间和执行力是基础，但你还必须有一个清晰的目标。请将你的目标（例如负债金额）明确地写下来。人都有赚钱的欲望，但很多时候这种欲望不够强烈。

人不能给自己留退路。因为留有退路，你会错过许多时代机遇。

以下是历史上的一些机遇实例：
*   **84年改革开放**：是机遇，但很多人因有退路而未下海经商。
*   **房地产浪潮**：是机遇，但很多人因已有房贷或觉得麻烦而未参与。
*   **淘宝兴起**：是机遇，但很多人最初认为网上购物是骗局。
*   **抖音红利**：是机遇，但很多人又觉得自己学不会。

![](img/939da09461854904eb5b14d7c2d05d40_6.png)

中国大约每五到十年就会出现一次重大机遇，这就是所谓的“风口”或“红利期”。现在，视频号就是当下的风口。你必须决定是否要把握住它。

因此，不要再为自己寻找任何借口。当你跌入低谷、已无退路时，反而能看清什么才是最重要的。我希望大家能把握住这次机会。
