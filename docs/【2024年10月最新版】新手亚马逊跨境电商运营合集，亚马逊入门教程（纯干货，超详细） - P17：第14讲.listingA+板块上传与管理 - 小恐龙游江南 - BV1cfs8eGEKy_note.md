## A+内容的基本要求与站点差异

![](img/9fc1a9b8550e11dfa560a6d3624b62f7_9.png)

首先需要明确一点，不同站点的A+内容使用权限不同。
*   **日本站**：无需品牌备案即可使用A+内容功能。
*   **美国站与欧洲站**：必须完成品牌备案后，才能使用A+内容功能。这是亚马逊对日本站卖家的一个扶持政策。

## A+内容的两种形式

![](img/9fc1a9b8550e11dfa560a6d3624b62f7_11.png)

在卖家后台点击“创建A+页面”，可以看到两种主要形式。

![](img/9fc1a9b8550e11dfa560a6d3624b62f7_12.png)

![](img/9fc1a9b8550e11dfa560a6d3624b62f7_13.png)

以下是两种形式的介绍：
1.  **品牌故事**：主要用于展示品牌背景、理念和故事，强化品牌形象。
2.  **增强型商品描述**：主要用于详细展示产品功能、细节、使用场景和卖点，是大多数卖家更常使用的形式。

![](img/9fc1a9b8550e11dfa560a6d3624b62f7_15.png)

## 品牌故事板块详解

![](img/9fc1a9b8550e11dfa560a6d3624b62f7_17.png)

品牌故事板块的展示形式通常如下：
*   **顶部背景图**：可以放置品牌模特或核心产品的大图。
*   **品牌故事叙述**：阐述品牌的创立初衷、理念或使命。
*   **产品与场景展示**：通过多张图片展示产品细节、使用效果和适用场景。
*   **品牌价值主张**：说明品牌优势以及能为客户提供的独特价值。

这个板块能有效提升客户对品牌的整体认知和好感度，结合产品主图与A+图文描述，对品牌打造和提升转化率是很有力的手段。

![](img/9fc1a9b8550e11dfa560a6d3624b62f7_19.png)

![](img/9fc1a9b8550e11dfa560a6d3624b62f7_21.png)

在创建品牌故事时，可以添加以下模块：
*   **关于我们模块**：用于填写品牌介绍。
*   **四图模块**：用于并排展示多张图片。
*   **单图模块**：用于突出展示一张核心图片。
*   **品牌卡片模块**：用于总结品牌关键信息。

## 增强型商品描述（基础板块）实操

对于不以品牌建设为主要目的的卖家，增强型商品描述（基础板块）更为常用。该板块提供了多种内容模块组合。

![](img/9fc1a9b8550e11dfa560a6d3624b62f7_27.png)

以下是创建增强型描述的基本步骤：
1.  **选择模块**：从后台提供的多种布局模块中选择，例如“Logo和三个图片”模块。
2.  **上传内容**：在模块中插入设计好的图片并添加对应文字说明。
3.  **填写图片关键词**：在上传图片时，有一个“图片关键词”字段，最多可填写100个字符。虽然其是否被搜索引擎抓取未被官方明确证实，但建议在此处埋入相关关键词。
4.  **关联商品**：完成内容编辑后，需要将创建的A+页面关联到具体的ASIN（商品编码）。
5.  **预览与提交**：提交前务必预览效果，确认无误后提交审核。通常审核需要一段时间。

具体操作代码逻辑可简化为：
```plaintext
1. 进入卖家后台 -> 广告 -> A+内容管理器
2. 点击“创建A+内容” -> 选择“增强型商品描述”
3. 选择内容模块 -> 上传图片/填写文本/埋入关键词
4. 保存并关联目标商品ASIN
5. 预览 -> 提交审核
```
