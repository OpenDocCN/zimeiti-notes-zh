1.  **角色**
    *   角色即你在特定赛道中扮演的专业身份。你需要让用户相信，你提供的内容具有专业性。
    *   **公式：角色 = 赛道方向 + 专家身份**
    *   例如：
        *   做好物分享或测评，你就是“产品测评官”。
        *   做装修博主，你就是“亲身经历装修的屋主”或“室内设计师”。
        *   做母婴赛道，你可以是“分享经验的宝妈”，也可以是“专业的育儿护理专家”。
        *   做情感赛道，你可以是“心理学顾问”或“分享个人成长经历的过来人”（成长型人设）。

2.  **内容**
    *   内容必须与你设定的角色和赛道高度垂直、持续一致。这关系到账号的垂直度和平台标签，直接影响流量分发。
    *   **核心原则：内容垂直，持续输出赛道相关价值。**
    *   如果内容混乱（例如第一篇讲教育，第二篇讲宠物），会扰乱用户预期和平台算法，导致数据下滑，影响推荐效果。

3.  **形式**
    *   形式指内容的具体呈现方式，主要包括以下三点：
        *   **产出场景**：是在固定的室内场景，还是多变的室外场景？需要提前规划。
        *   **声音呈现**：是自己出镜讲解，还是使用配音？需考虑个人表达能力和连贯性。
        *   **内容形式**：是剧情演绎、干货分享，还是生活记录？不同形式各有优劣：
            *   **剧情型**：创作难度较高。
            *   **干货型**：需要精心准备文案脚本。
            *   **生活记录型**：需要积累大量拍摄素材。

4.  **记忆点** 🧠
    *   记忆点是让用户快速记住并识别你的关键。它需要通过精心设计的固定元素来反复强化。
    *   **记忆点植入清单**：
        *   **视频前几秒的固定动作/特效/开场白**：这些元素需要前置，在视频开头就形成强烈冲击，加深印象。
        *   **统一的BGM（背景音乐）**：贯穿视频全程的标志性音乐，能让人一听到就联想到你。
        *   **统一出现的物品或服装风格**：贯穿内容全局的视觉元素，如特定的摆设、道具或着装风格。

    设计记忆点时，要思考其特征：**固定动作、特效、开场白**这类元素因其“前置性”，能在用户每次观看时第一时间强化印象；而**BGM和统一物品**则通过“全局贯穿”来持续塑造整体氛围和辨识度。

本节课中我们一起学习了IP人设定位的四大核心因素：**角色、内容、形式和记忆点**。明确你在赛道中的专业角色，输出垂直且持续的内容，选择适合的呈现形式，并精心设计强化辨识度的记忆点，是构建一个成功人设的基础。下一节，我们将探讨如何通过账号的配套设置来进一步丰满和支撑这个人设。
