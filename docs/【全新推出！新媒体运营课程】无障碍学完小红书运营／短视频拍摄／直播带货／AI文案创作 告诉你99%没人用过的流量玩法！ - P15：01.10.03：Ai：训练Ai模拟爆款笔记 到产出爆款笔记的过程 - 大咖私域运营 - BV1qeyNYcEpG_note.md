# AI新媒体运营课程：P15：01.10.03：训练AI模拟爆款笔记到产出爆款笔记的过程 🚀

## 概述
在本节课中，我们将学习如何训练AI（如ChatGPT）分析和学习爆款笔记的要素，并最终指导它生成具备爆款潜力的新笔记。我们将从提供爆款内容给AI分析开始，到整合分析结果，最后生成完整文案。

![](img/4ba02acc37bbda1d5261f4238b019eec_1.png)

上一节我们介绍了通过喂资料的方式，让AI理解特定平台的用语和风格。虽然以旅游攻略为例的最终内容可能不理想，但方法是正确的。本节中，我们来看看如何通过分析爆款内容，让AI学习其成功秘诀。

## 第一步：让AI分析爆款内容
要让AI学会创作爆款笔记，首先需要让它理解什么是“爆款”。我们需要提供真实的爆款内容，并引导AI从几个关键维度进行分析。

以下是分析爆款内容时需要关注的几个核心点：
1.  **爆款因素**：分析内容中哪些点可能导致了它的高热度（例如：情感共鸣、实用建议、逆袭故事）。
2.  **内容结构**：拆解文案的框架。
    *   **开篇**：用什么方式（如提问、制造焦虑、引用名人）吸引用户继续阅读。
    *   **中间**：提供了哪些内容来戳中用户痛点，提升用户停留时长。
    *   **结尾**：设置了什么“钩子”（如鼓励互动、提供联系方式）来促使用户行动。
3.  **关键词策略**：分析笔记中是否刻意植入了特定关键词，以及这些关键词的出现频率和分布方式。

## 第二步：实践操作与指令设计
我们以“考研”领域的笔记为例进行实际操作。首先，需要在小红书等平台找到点赞量高的爆款笔记，复制其文案内容。

接下来，我们需要设计清晰的指令给AI（如ChatGPT-3.5）。指令的核心是要求它根据我们提供的文案，分析上述几个维度。

![](img/4ba02acc37bbda1d5261f4238b019eec_3.png)

**示例指令代码如下：**
```
我给你提供一些小红书平台上关于“考研”的爆款笔记文案。
你需要分析这些文案的爆款因素，比如：
1. 文案的整体结构：开篇用什么吸引用户？中间用什么内容戳中用户痛点？结尾用什么方式让用户形成互动、私信或关注等行为？
2. 分析我给你的每一篇笔记的关键词：根据整篇内容，分析该笔记是否有结合中文分词法设计关键词。如果没有，请告诉我“没有关键词”；如果有，告诉我是哪个关键词，以及该关键词的出现频率。
```
将找到的爆款文案粘贴在指令下方，发送给AI。AI会返回初步分析结果，例如指出文案运用了“情感共鸣”、“提供明确备考计划”、“时间节点清晰”等因素，并判断关键词使用情况。

![](img/4ba02acc37bbda1d5261f4238b019eec_5.png)

## 第三步：整合分析与优化指令
AI在连续分析多篇内容后可能会遗忘最初的要求，或者分析不够全面。这时，我们需要通过对话引导它进行整合与回顾。

我们可以使用如下指令让AI总结所有分析过的爆款因素：
```
请把你上面分析过的关于“考研”的几篇笔记的爆款因素总结一下，整合成一个去重后的列表。
```
如果AI遗漏了某些重点（如“逆袭故事”、“使用表情符号”），我们可以提醒它：“请重新回顾一下你之前分析过的X篇笔记，确保总结全面。” 这个过程是为了让AI自己提炼出一套关于“考研爆款笔记”的通用要素库。

## 第四步：生成爆款笔记大纲与文案
当AI掌握了爆款要素后，就可以指导它进行创作。首先，让它根据学习到的因素生成一个笔记大纲。

**生成大纲的指令代码如下：**
```
我要写一篇关于“考研”的小红书笔记。请结合你前面分析的爆款笔记因素，帮我梳理一个具备爆款潜力的笔记大纲。每个大纲节点需要进一步细分为更小的要点。在我确认大纲没问题后，再生成完整文案。
```
AI可能会给出类似的大纲：开篇引入 -> 专业选择建议 -> 分科（政治、英语、数学、专业课）备考规划 -> 时间管理与心态调整 -> 结尾互动引导。

大纲确认后，即可发出最终生成指令。为了更贴近平台风格，可以在指令中加入细节要求。

**生成文案的指令代码如下：**
```
请按照上述大纲，写出具体的文案。文案结构要严格遵循爆款笔记因素，并且需要：
1. 提供一个简单的“逆袭”小故事引入，引发读者共鸣。
2. 包含适量的emoji表情。
3. 尝试以“中文分词法”的方式，自然融入“考研”、“政治备考”等关键词。
```
根据这个指令，AI会生成一篇包含个人经历、详细规划、鼓励话语和互动引导的完整笔记。虽然对于资深运营来说可能还需优化，但对于一般需求，其结构完整性和专业性已基本达标。

## 总结
本节课我们一起学习了利用AI辅助创作爆款笔记的全过程。关键步骤包括：**提供爆款样本 -> 设计指令引导AI分析结构、因素与关键词 -> 整合分析结果 -> 基于AI总结的要素生成大纲与文案**。通过这种方法，即使是不擅长文案的运营者，也能借助AI快速产出符合平台调性、具备爆款潜力的内容。记住，与AI对话时，指令的清晰度和细节要求至关重要，这直接影响最终产出质量。使用更强大的模型（如GPT-4）或利用“自定义GPT”功能喂入专属风格资料，将能获得更惊艳的效果。
