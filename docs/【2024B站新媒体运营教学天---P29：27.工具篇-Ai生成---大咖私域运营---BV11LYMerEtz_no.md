# 新媒体运营全攻略：P29：27.工具篇-AI生成PPT 🚀

在本节课中，我们将学习一款非常实用的AI工具，它能帮助我们根据主题或现有文档，快速生成结构完整、内容详实的PPT演示文稿。

## 工具概述与核心功能

上一节我们介绍了多种新媒体运营工具，本节中我们来看看如何利用AI提升PPT制作效率。这款工具的核心功能在于其智能的内容理解和生成能力。

它主要提供两种生成模式：
1.  **主题生成模式**：输入一个主题，工具会像ChatGPT一样，自动生成完整的PPT内容。
2.  **文档生成模式**：上传一篇文档（如TXT、PDF），AI助手会自动总结大纲，并基于大纲生成详细的PPT内容。

其核心优势在于，它不仅能格式化已有内容，更能基于**大纲节点自动拓展和生成详细解说**，这比单纯将现有内容排版成幻灯片更有价值。

## 编辑与优化生成结果

![](img/18691df0ff8195e89c094872366f6f68_17.png)

PPT内容生成完毕后，我们可以进入编辑界面进行调整和美化。

如果对默认模板不满意，可以点击右侧的“更换模板”功能。虽然当前提供的模板数量有限，但切换非常迅速。



![](img/18691df0ff8195e89c094872366f6f68_15.png)



更重要的是，我们可以在右侧编辑面板中，对每一页的文字、形状等元素进行精细化调整。



![](img/18691df0ff8195e89c094872366f6f68_21.png)



![](img/18691df0ff8195e89c094872366f6f68_19.png)

如果希望更自由地设计，也可以将生成好的PPT下载到本地，使用PowerPoint或Keynote等专业软件进行深度编辑。

![](img/18691df0ff8195e89c094872366f6f68_21.png)
