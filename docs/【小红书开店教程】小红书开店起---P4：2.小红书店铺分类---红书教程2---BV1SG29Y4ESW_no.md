在本节课中，我们将要学习小红书电商平台上的店铺分类体系。了解不同类型的店铺及其要求，是开启小红书电商之旅的第一步，能帮助你根据自身情况选择最合适的店铺类型。

上一节我们介绍了开店的基础准备，本节中我们来看看小红书平台具体有哪些店铺类型。

## 店铺类型概述

小红书的店铺体系主要分为以下几类：**个人店**、**个体工商户店**、**普通企业店**、**专卖店**、**旗舰店**以及**卖场型旗舰店**。这些店铺类型主要根据申请主体和资质要求进行区分。

## 各类店铺详解

![](img/d5a47f1cbccc6de526c515bb323d007a_6.png)

以下是各类店铺的具体介绍：

![](img/d5a47f1cbccc6de526c515bb323d007a_8.png)

### 1. 个人店
*   **申请主体**：个人用户。适合刚毕业、辞职或想尝试小规模经营的创业者，类似于“线上摆地摊”。
*   **经营特点**：可经营限定类目下的零星、小额商品，如首饰、服装、手机壳等。相当于一个“杂货铺”，可销售多个品牌商品。
*   **入驻要求**：非常简单，仅需个人身份证。
*   **店铺命名**：格式为“专业号名称 + 自选店名”，系统会自动匹配。
*   **保证金**：包含**类目保证金**和**浮动保证金**。个人店的起始保证金通常为 **1000元**。

![](img/d5a47f1cbccc6de526c515bb323d007a_10.png)

![](img/d5a47f1cbccc6de526c515bb323d007a_12.png)

### 2. 个体工商户店
*   **与个人店的区别**：主要在于**入驻要求**。除了身份证，还需要提供**个体工商户营业执照**。
*   **其他方面**：如经营特点、保证金规则等，与个人店基本一致。

### 3. 普通企业店
*   **申请主体**：企业。需要持有**企业营业执照**。
*   **特点**：适合拥有自己公司或产品的卖家。平台对不同店铺的权重分配不同，通常资质要求越复杂，店铺权重可能越高。但对于新手，建议从要求简单的店铺开始测试。

### 4. 专卖店 / 旗舰店 / 卖场型旗舰店
这几类店铺均以企业为主体，核心区别在于品牌资质：
*   **专卖店**：需营业执照及**品牌认可**（销售自有品牌或非独占授权品牌）。
*   **旗舰店**：需营业执照及**品牌授权书**（销售他人品牌商品）。
*   **卖场型旗舰店**：需零售资质，并可经营多个品牌。但在小红书平台较为少见，因为平台以女性用户为主，流量特性与大型卖场店不完全匹配，这反而为中小卖家提供了发展空间。

## 保证金规则详解

![](img/d5a47f1cbccc6de526c515bb323d007a_16.png)

![](img/d5a47f1cbccc6de526c515bb323d007a_17.png)

![](img/d5a47f1cbccc6de526c515bb323d007a_19.png)

了解保证金规则对资金管理至关重要。保证金主要包括两部分：

![](img/d5a47f1cbccc6de526c515bb323d007a_20.png)

![](img/d5a47f1cbccc6de526c515bb323d007a_22.png)

### 1. 类目保证金
根据经营的商品类目不同，平台会设定相应的固定保证金。例如：
```
基础保证金：1000元
普通企业类目：XXX元
特殊类目（如食品、化妆品）：XXX元
跨境类目：XXX元
```
（具体金额需参考平台官方最新类目保证金表。）

### 2. 浮动保证金 ⚠️
这是根据店铺月成交额动态计算的保证金，**必须特别关注**，它直接影响你是否能顺利提现。

浮动保证金计算规则如下：
*   月成交额 **≤ 5万元**：无需额外浮动保证金（但需已缴纳基础保证金）。
*   5万元 **<** 月成交额 **≤ 10万元**：需补足保证金至 **3000元**。
*   10万元 **<** 月成交额 **≤ 50万元**：需补足保证金至 **5000元**。
*   月成交额 **> 50万元**：需补足保证金至 **20000元**。

**关键点**：平台会在提现时校验保证金账户总额。若账户余额不足，则无法提现。例如，若你只缴纳了1000元保证金，当月成交额达6万元，则因为保证金不足3000元，你将无法提现。
