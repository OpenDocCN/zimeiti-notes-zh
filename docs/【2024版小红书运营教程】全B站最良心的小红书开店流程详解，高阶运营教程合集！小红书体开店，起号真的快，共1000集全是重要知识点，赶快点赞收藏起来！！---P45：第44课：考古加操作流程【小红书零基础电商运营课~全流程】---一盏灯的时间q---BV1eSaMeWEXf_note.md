# 小红书电商运营课：P45：考古加操作流程详解 🛠️

在本节课中，我们将学习如何使用“考古加”工具进行选品和寻找素材。这是一个高效的工具，能帮助我们从抖音等平台发现近期热销商品，并获取对应的视频素材，为我们在小红书开店和发布笔记提供支持。

## 工具介绍与访问

首先，考古加是一个需要付费使用的网页工具。你可以通过百度搜索“考古加”找到其官网。该工具需要开通会员才能使用全部功能，费用约为每月200多元。

## 选品流程：如何寻找潜力商品

上一节我们介绍了工具的基本情况，本节中我们来看看如何用它来寻找有潜力的商品。选品是运营的第一步，关键在于找到近期有热度且适合在小红书操作的产品。

![](img/cc5cc52b8946b2af3ab38a3d8eac3a80_1.png)

![](img/cc5cc52b8946b2af3ab38a3d8eac3a80_2.png)

以下是使用考古加进行选品的基本步骤：

![](img/cc5cc52b8946b2af3ab38a3d8eac3a80_4.png)

![](img/cc5cc52b8946b2af3ab38a3d8eac3a80_6.png)

1.  **输入关键词**：在搜索框输入你想操作的类目关键词。关键词可以来自各平台的热榜，或是你计划深耕的细分领域。
2.  **选择搜索类型**：主要使用“视频为主”或“商品卡为主”两种模式进行筛选。
    *   **视频为主**：筛选出主要通过短视频带货的商品。这适合我们进行视频混剪。
    *   **商品卡为主**：筛选出主要依靠商城自然流量出单的商品。这类商品不依赖直播或短视频，说明其产品本身吸引力强，自然权重高，同样具有参考价值。
3.  **分析销量趋势**：将“新增销量”的筛选条件设置为“7日”。观察商品的销量曲线。
    *   **理想曲线**：呈现**持续稳定上涨**的趋势，表明产品热度健康。
    *   **需谨慎的曲线**：突然**暴涨后暴跌**，可能是付费投流的结果，热度可能不持久；或呈现**持续下滑**趋势，说明产品热度已过。
4.  **筛选细分产品**：为了找到更精准的细分市场，关键词应尽可能具体。例如，不做宽泛的“女装”，而是搜索“毛衣开衫”、“冬季大衣”等应季且具体的产品。

![](img/cc5cc52b8946b2af3ab38a3d8eac3a80_7.png)

![](img/cc5cc52b8946b2af3ab38a3d8eac3a80_9.png)

![](img/cc5cc52b8946b2af3ab38a3d8eac3a80_11.png)

**核心逻辑**：我们倾向于操作“近期热销”的产品。因为从抖音搬运到小红书存在时间差，操作近期爆款可以提高我们“截流”的成功率。

## 素材寻找：如何获取视频与图片

![](img/cc5cc52b8946b2af3ab38a3d8eac3a80_17.png)

![](img/cc5cc52b8946b2af3ab38a3d8eac3a80_18.png)

找到潜力商品后，下一步就是为它寻找可用于制作笔记的素材。我们主要目标是获取视频素材进行混剪，以降低违规风险。

![](img/cc5cc52b8946b2af3ab38a3d8eac3a80_20.png)

![](img/cc5cc52b8946b2af3ab38a3d8eac3a80_22.png)

以下是获取素材的方法：

![](img/cc5cc52b8946b2af3ab38a3d8eac3a80_24.png)

1.  **在商品页寻找关联素材**：点击感兴趣的商品，进入详情页。在“关联视频”区域，可以查看所有推广该商品的短视频。
2.  **评估素材可用性**：
    *   **优质视频素材**：真人实拍、多角度展示产品的短视频是首选，适合用于混剪。
    *   **图片素材**：高质量的图片（尤其是多色展示图）可以用作笔记封面，或通过工具将多张图片合成为视频。
    *   **需规避的素材**：带有夸张大头贴、漫画特效过多的视频，或质量低劣的图片，不建议使用。
3.  **重要提醒**：直接搬运单一图片或视频很容易被平台识别为违规。因此，**前期务必通过“多视频素材混剪”** 的方式来制作内容。

![](img/cc5cc52b8946b2af3ab38a3d8eac3a80_26.png)

## 进阶技巧：直接搜索视频与发现对标账号

除了通过商品找素材，考古加还提供了更直接的视频搜索功能，并能帮助我们找到对标账号。

![](img/cc5cc52b8946b2af3ab38a3d8eac3a80_28.png)

![](img/cc5cc52b8946b2af3ab38a3d8eac3a80_29.png)

以下是两个进阶的素材寻找路径：

1.  **在“电商视频”中搜索**：直接搜索产品关键词（如“宽松毛衣”），然后选择“电商视频”标签。这里会列出所有挂载了购物车的相关短视频。你可以直接筛选出展示产品的视频片段，效率很高。
2.  **在“内容视频”中挖掘账号**：同样搜索关键词后，选择“内容视频”标签。点击一个挂载了商品的视频，进入其详情页。这里不仅可以看同款视频，更重要的是可以**点击进入发布者的抖音主页**。
    *   **发现对标账号**：该账号下其他穿搭类视频，只要风格和品类与你目标一致，其展示的**所有产品**都可以作为你的备选品和素材来源。
    *   **获取系列素材**：如果一个账号专门推广某一类产品（如各种开衫），你就能获得大量风格统一的视频素材，非常适合进行系统化的混剪操作。

## 课程总结

![](img/cc5cc52b8946b2af3ab38a3d8eac3a80_36.png)

本节课中我们一起学习了考古加工具的核心操作流程。

*   **选品**：我们通过**关键词搜索**，并利用 **“视频为主”/“商品卡为主”** 筛选和**分析7日销量曲线**，来定位近期有潜力的热销商品。
*   **找素材**：我们学会了在商品页获取**关联视频**，并评估素材质量。核心原则是优先使用视频进行**混剪**，避免直接搬运。
*   **进阶应用**：我们掌握了通过 **“电商视频”** 直接找带货视频，以及通过 **“内容视频”** 深入挖掘**对标账号**以获得系列化素材的方法。

![](img/cc5cc52b8946b2af3ab38a3d8eac3a80_38.png)

本质上，考古加是一个高效的筛选器，帮助我们快速从抖音平台海量内容中，找到那些已经被市场验证、近期有销量的“准爆款”商品及其配套素材，从而提升我们在小红书运营的效率和成功率。
