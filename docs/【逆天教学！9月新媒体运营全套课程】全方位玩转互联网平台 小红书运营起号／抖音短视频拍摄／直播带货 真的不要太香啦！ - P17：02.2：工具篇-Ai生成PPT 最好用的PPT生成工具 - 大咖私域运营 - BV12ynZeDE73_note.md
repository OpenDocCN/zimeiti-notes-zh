# 新媒体运营课程：P17：02.2：工具篇-AI生成PPT 🚀

在本节课中，我们将学习如何使用一款高效的AI工具来辅助生成PPT。这款工具能根据你提供的主题或现有文档，自动梳理大纲并生成详细的演示文稿内容，极大地提升内容创作效率。

上一节我们介绍了新媒体运营的多种工具，本节中我们来看看如何利用AI技术快速制作专业PPT。

## 工具核心功能

![](img/99928606735e158017e94bdead00a306_1.png)

这款AI工具主要提供两种生成模式：

![](img/99928606735e158017e94bdead00a306_3.png)

1.  **主题生成模式**：你只需输入一个核心主题，工具便会利用其生成式AI能力，自动创作出完整的PPT全文。
2.  **文档生成模式**：你可以上传已有的文档（如TXT、PDF文件），AI助手会自动总结文档大纲，并基于此大纲生成对应的PPT内容。

![](img/99928606735e158017e94bdead00a306_4.png)

![](img/99928606735e158017e94bdead00a306_5.png)

我测试过主题生成模式，其生成的内容非常详尽，有时甚至可能达到六七十页，虽然可能存在部分重复，但作为初稿或灵感来源极具价值。

## 实战演示：基于文档生成PPT

![](img/99928606735e158017e94bdead00a306_7.png)

为了更直观地展示，我们将通过上传一篇准备好的文章进行演示。

![](img/99928606735e158017e94bdead00a306_9.png)

以下是操作步骤：

![](img/99928606735e158017e94bdead00a306_11.png)

1.  准备好你的文档。这里我们使用一篇关于“新媒体运营流量与创作标签”的文章。
    
    ![](img/99928606735e158017e94bdead00a306_1.png)
    
2.  在工具界面点击“继续”，然后上传文件。
    
    ![](img/99928606735e158017e94bdead00a306_3.png)
    ![](img/99928606735e158017e94bdead00a306_4.png)
    ![](img/99928606735e158017e94bdead00a306_5.png)
    
3.  工具会快速解读文档，并在下方列出分析后的大纲。例如，它可能总结出“新媒体运营的本质”、“平台运营的角度”、“账号运营的策略”等章节，分析通常非常细致。
    
4.  大纲确认后，点击“开始制作”。
    
    ![](img/99928606735e158017e94bdead00a306_7.png)
    
5.  工具进入生成页面，基于对内容的理解和大纲，自动生成PPT的主体内容。这个过程通常很快，但有时可能因网络环境或平台负载需要稍作等待。
    
    ![](img/99928606735e158017e94bdead00a306_9.png)
    
6.  生成完毕后，即可查看和编辑PPT。
    
    ![](img/99928606735e158017e94bdead00a306_13.png)
    

![](img/99928606735e158017e94bdead00a306_13.png)

## PPT的编辑与优化

生成初稿后，我们可以进一步优化PPT，使其更符合个人需求。

![](img/99928606735e158017e94bdead00a306_15.png)

以下是可进行的调整：

*   **更换模板**：工具右侧提供“更换模板”选项，虽然当前模板库数量有限，但切换速度很快。你可以选择一个更符合内容风格的模板。
    
    ![](img/99928606735e158017e94bdead00a306_15.png)
    
*   **调整内容**：在编辑界面，你可以直接修改任何页面的文字、形状等元素。工具的核心优势在于它能基于大纲节点生成详细内容，这比手动填充所有幻灯片要高效得多。
    
    ![](img/99928606735e158017e94bdead00a306_17.png)
    
*   **本地化编辑**：你也可以在调整好大致模板和内容后，将PPT下载到本地，使用PowerPoint或Keynote等软件进行更精细的排版和设计。
