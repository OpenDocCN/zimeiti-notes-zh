# 抖音全套运营教程：P18：扶持加权与降权限流 📊

在本节课中，我们将学习抖音平台对视频内容的审核与流量分配机制。具体来说，我们将了解哪些内容会获得平台的“扶持加权”，从而获得更多推荐流量；以及哪些内容会导致“降权限流”，甚至无法通过审核。理解这些规则是成功运营抖音账号的基础。

上一节我们介绍了账号权重的分类，本节中我们来看看具体影响权重的视频内容标准。

## 一、获得扶持加权的内容 ✅

![](img/ea66641a5a1a1963a74f76eb0a2d73b9_5.png)

平台会优先推荐符合以下技术标准和内容规范的原生视频。

以下是获得流量扶持的关键要点：
*   **视频格式**：必须为**竖版（9:16）** 的移动端原生视频。这是抖音的标准格式。
*   **画面质量**：画面需保持平稳，作品清晰度高。
*   **音频质量**：视频中无过多的杂音或噪音。

这些是基础要求，平台算法会优先识别和推荐符合这些标准的视频。

## 二、导致降权限流的内容 ❌

发布以下内容将导致视频被限流、审核不通过，甚至账号受到处罚。

以下是常见的违规内容类型：
1.  **不良信息**：视频内容涉及吸烟、暴力、炫富（过度展示金钱）等信息。
2.  **外部引流**：在视频中留下或展示微信、微博、淘宝等外部平台的联系方式。长时间出现商品Logo、手机号、二维码也属于违规行为。
    *   **关于二维码的补充**：二维码并非绝对禁止出现，但展示时间**最长不要超过3秒**。在某些审核严格时期，即使出现1秒也可能导致视频被驳回。
3.  **硬性广告**：进行明显的营销广告、推销或引流卖货行为。
4.  **无口播的好物推荐**：这是需要特别注意的一点。如果视频内容是拆箱或推荐好物，但**没有口播介绍**，只是单纯展示产品，极易被判定为广告而限流或下架购物车。
    *   **案例说明**：例如，推荐一款玩具，如果视频中只有背景音乐，没有任何口头介绍（如“我儿子玩了这个玩具，专注地玩了四个小时”），则风险很高。添加简单的口播描述可以有效降低违规风险。
    *   **另一个错误案例**：在视频画面中直接标注“9.9元”等价格信息，属于直接的销售行为，通常无法通过审核。
5.  **口播广告与联系方式**：在口播中直接念出广告语或联系方式，这在抖音是不被允许的。

请注意，单纯展示产品、无口播的拆箱或推荐行为风险很高。在抖音，需要通过内容化的方式（如加入场景、故事或口播介绍）来软性传递产品信息，而非直接进行淘宝式的商品展示。

本节课中我们一起学习了抖音平台关于“扶持加权”与“降权限流”的核心规则。关键在于制作符合平台标准的原生竖版视频，并避免发布违规内容。特别是对于希望带货的账号，需掌握“软性推荐”而非“硬性广告”的技巧。理解并应用这些规则，是账号健康运营、获得稳定流量的前提。
