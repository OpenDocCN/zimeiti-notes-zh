# 抖音运营教程：P24：制造情绪冲突的文案角度

在本节课中，我们将要学习如何通过制造情绪冲突来提升抖音文案的吸引力。我们将探讨不同的文案表达方式如何影响用户心理，并介绍一个实用的“九宫格选题法”来辅助创作。

---

选题是内容创作的基础。选完题之后，我们才能进行脚本结构设计和拍摄工作。同样的一句话，用不同的方式说出来，其表达效果截然不同。

例如，如果我们想鼓励一个人去健身瘦身。直接说“来健身房，变成更美的自己”，这句话显得有些无关痛痒。它就像一句普通的广告语，例如“开沃尔沃，更安全”，难以引发强烈共鸣。

但如果换一种说法，利用“失去心理”或“恐吓心理”告诉用户：“如果你不去健身房，你就失去了变美的机会”。这会在用户心中形成一种紧迫感。

销售面膜也是同理。告诉用户“敷面膜，永远保持20岁”，是一句干巴巴的广告词，类似“弹弹弹，弹走鱼尾纹”。如果换成：“如果不尽早护肤敷面膜，你就会长出难以去除的鱼尾纹、色斑”，用户可能会更加重视。

以上例子说明，同一个选题，用不同的话术或文案表达，给用户的心理暗示是不同的。

这就像双十一抢购。商家说“今天购买打五折”，这是一个优惠信息。如果再加上一个刺激：“五折优惠仅剩最后两个名额”，用户就会立刻下单。这两种刺激带来的紧迫感完全不同。

主播带货也运用了相同逻辑。恰当的表述可以让你“卖断货”。

现在回想一下，如果你的文案是“如何科学健身”，这同样是一句无关痛痒的话。用户可能知道，但缺乏去实操的动力。它就像“敷面膜永远保持20岁”一样平淡。

如果你告诉他：“如果不科学健身，你会造成什么后果？长期久坐劳累，可能导致腰间盘突出等问题”。虽然举例可能有些夸张，但道理是相通的。

同一个选题，用不同的方式和心理角度阐述，给用户造成的情感冲突和情绪波动是不同的。因此，我们的核心目标是：**制造情绪的冲突**。

![](img/738b61306efd50055f6e2a6ba0f6ce78_1.png)

制造情绪冲突有一些模板可以使用。前几种模板效果可能一般，但在某些行业仍可应用。而“恐惧心理”和“失去心理”是经常使用的有效方法，但也不宜过于频繁，以免给用户带来不良体验。

大家可以适当记录这些方法，学会后可以应用到你的任何视频创作中。例如，健身领域可以塑造“水蛇腰”，并强调“一时的身体疼痛，可能成为终身的疼痛”。

优秀的广告公司（如奥美公司）正是以出色的广告文案而闻名。他们的创作思路值得借鉴。

接下来，我们暂时结束选题技巧的讨论。大家需要重点学习和记录的是下面的“九宫格选题法”。

![](img/738b61306efd50055f6e2a6ba0f6ce78_1.png)

![](img/738b61306efd50055f6e2a6ba0f6ce78_3.png)

**九宫格选题法**是一个实用的工具。以下是使用步骤：

1.  在九宫格中央写下你的核心产品或主题。
2.  在周围的八个格子中，分别填入与核心主题相关的不同用户画像、使用场景、痛点或需求。

你可以将任何不同的产品和用户代入这个框架，然后进行延伸拓展。

![](img/738b61306efd50055f6e2a6ba0f6ce78_5.png)

延伸之后，利用“连连看”的方式，将不同格子里的元素进行组合，从而生成一个具体的选题。

![](img/738b61306efd50055f6e2a6ba0f6ce78_3.png)

例如：`核心产品（健身）` + `用户画像（办公室白领）` + `痛点（腰酸背痛）` = 《办公室白领，每天10分钟，告别腰酸背痛》。

![](img/738b61306efd50055f6e2a6ba0f6ce78_5.png)

![](img/738b61306efd50055f6e2a6ba0f6ce78_7.png)

通过这种方式，我们可以系统化地产生大量创意选题。

![](img/738b61306efd50055f6e2a6ba0f6ce78_7.png)

本节课中，我们一起学习了制造情绪冲突对于抖音文案的重要性。通过对比“直接陈述”与“利用恐惧/失去心理”的不同效果，我们理解了情绪如何影响用户决策。同时，我们掌握了一个名为“九宫格选题法”的实用工具，它可以帮助我们系统地拓展思路，生成更具吸引力的视频选题。记住，好的文案是引发共鸣和行动的关键。
