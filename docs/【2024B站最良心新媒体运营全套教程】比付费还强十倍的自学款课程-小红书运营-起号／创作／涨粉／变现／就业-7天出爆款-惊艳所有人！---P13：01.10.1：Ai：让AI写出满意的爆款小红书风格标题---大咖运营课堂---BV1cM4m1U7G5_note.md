# 小红书运营：P13：01.10.1：🤖 让AI写出满意的爆款小红书风格标题

在本节课中，我们将要学习如何利用AI工具，特别是ChatGPT，来生成符合小红书平台风格的爆款标题。这是提升笔记点击率、优化运营效率的关键技能。

## 概述：AI与内容运营的结合

当前，人工智能在各领域的应用已非常广泛。对于运营工作而言，最核心的就是内容产出。AI工具能够帮助我们高效地产出文案、设计脚本、植入关键词以及撰写标题。

然而，许多人在使用AI时，常遇到沟通障碍，即AI无法完全理解我们的具体需求。本节课程将解决这个问题，教你如何通过精准的指令，让AI写出真正符合小红书风格的标题。

## 如何优化AI指令

![](img/7c978d933b8436d656745eba610b4a39_30.png)

![](img/7c978d933b8436d656745eba610b4a39_32.png)

![](img/7c978d933b8436d656745eba610b4a39_34.png)

核心方法在于：**为AI提供具体、明确的“爆款关键词”和风格要求**。这相当于给AI一个清晰的创作框架。

![](img/7c978d933b8436d656745eba610b4a39_35.png)

![](img/7c978d933b8436d656745eba610b4a39_37.png)

![](img/7c978d933b8436d656745eba610b4a39_39.png)

一个优化的指令结构应包含以下要素：
1.  **明确任务：** 写一篇关于[主题]的小红书笔记。
2.  **提供关键词：** 根据以下爆款关键词。
3.  **具体输出要求：** 取10个爆款标题，20字以内。
4.  **风格要求：** 加上小红书风格的Emoji表情。

![](img/7c978d933b8436d656745eba610b4a39_41.png)

![](img/7c978d933b8436d656745eba610b4a39_43.png)

![](img/7c978d933b8436d656745eba610b4a39_44.png)

**优化后的指令公式：**
```
我要写一篇关于[你的主题]的小红书笔记。请根据以下爆款关键词：[插入关键词1、关键词2、关键词3...]，为我生成10个爆款标题。要求：每个标题20个字以内，并且加上符合小红书风格的Emoji表情。
```
