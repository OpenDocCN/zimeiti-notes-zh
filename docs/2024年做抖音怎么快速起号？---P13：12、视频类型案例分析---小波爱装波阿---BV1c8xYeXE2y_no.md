## 案例分析：强人设 vs. 弱人设

![](img/25cde4a7c79825599089166a22681f34_21.png)

![](img/25cde4a7c79825599089166a22681f34_23.png)

这是一个典型的**弱人设**视频。

![](img/25cde4a7c79825599089166a22681f34_25.png)

**判断依据如下：**
1.  **内容与讲述者无关**：视频讨论的是“穷人”和“富人”的普遍现象，但并未与视频发布者本人的经历、身份或观点建立直接、强烈的联系。
2.  **形式与内容割裂**：视频画面（如治愈系画面）与所讲述的严肃社会议题（阶层跨越）是两码事，没有形成统一的人格形象。

![](img/25cde4a7c79825599089166a22681f34_27.png)

![](img/25cde4a7c79825599089166a22681f34_29.png)

那么，为什么这个弱人设的视频能火呢？原因在于它实现了 **“双向吸引”**。

![](img/25cde4a7c79825599089166a22681f34_31.png)

![](img/25cde4a7c79825599089166a22681f34_33.png)

以下是该视频吸引观众的核心点：

![](img/25cde4a7c79825599089166a22681f34_35.png)

*   **吸引点一：形式吸引**。一部分观众被视频的**画面**（如治愈强迫症的动画或场景）所吸引，从而停留观看。
*   **吸引点二：内容共鸣**。另一部分观众被视频的**观点**所打动，认为其说出了自己的心声，产生了强烈的共鸣和认同感。

![](img/25cde4a7c79825599089166a22681f34_37.png)

![](img/25cde4a7c79825599089166a22681f34_39.png)

这种“画面吸引一批人，观点吸引另一批人”的策略，大大拓宽了视频的受众面，提升了完播率和互动数据。

## 方法论：如何应用“双向吸引”策略

![](img/25cde4a7c79825599089166a22681f34_47.png)

![](img/25cde4a7c79825599089166a22681f34_49.png)

理解了案例的成功逻辑后，我们可以将其应用到自己的内容创作中。核心方法是：**为你的主体内容（产品、生活、专业）搭配一个能引发广泛共鸣的“外壳”或“故事”**。

![](img/25cde4a7c79825599089166a22681f34_51.png)

![](img/25cde4a7c79825599089166a22681f34_53.png)

以下是针对不同创作者的具体应用举例：

*   **宝妈创作者**：如果单纯拍摄宝宝日常或做饭过程吸引力不足，可以尝试为视频**配上一个搞笑的家庭趣事或育儿心得**的旁白。
    *   **公式**：`日常画面 + 共鸣故事/观点 = 更吸引人的内容`
*   **实体店/工厂主**：如果单纯展示产品或车间无人问津，可以尝试讲述**你的从业故事、创业经历或个人感悟**。
    *   **代码逻辑**：
        ```python
        if 内容 == “单纯产品展示”:
            吸引力 = “低”
        elif 内容 == “产品展示 + 个人故事/行业洞察”:
            吸引力 = “高”
        ```
    这样既能吸引对你故事感兴趣的人，又能让他们在看故事的同时了解你的产品。

前期通过精准的标题吸引初始流量，后期配合这种“双向吸引”的内容技巧，就能有效提升视频的传播效果。许多视频完播率低，问题往往就在于缺少这样的小技巧。
