对大家来说，掌握规则只有好处没有坏处。那么，我们来具体了解什么是小红书的规则。

## 小红书规则的四大要点

小红书的规则内容非常丰富，我将其归纳为四个核心要点。大家不要看只有四点，每一点都包含大量具体内容和细节。其涵盖的页面和知识量，远超之前介绍的基础部分，因此我们必须理解清楚。

以下是小红书规则的四个核心组成部分：

![](img/1cb88caa3d4af8968ae6525395636bc2_19.png)

1.  **推荐规则**
2.  **审核规则**
3.  **社区公约**
4.  **商业规范**

由于课程时间关系，本节课我们重点讲解第一个，也是最重要的部分：**推荐规则**。

## 详解推荐规则

![](img/1cb88caa3d4af8968ae6525395636bc2_21.png)

![](img/1cb88caa3d4af8968ae6525395636bc2_23.png)

![](img/1cb88caa3d4af8968ae6525395636bc2_25.png)

![](img/1cb88caa3d4af8968ae6525395636bc2_27.png)

推荐规则是指小红书系统如何决定将你的内容展示给哪些用户，以及给予多少初始流量。无论你是发布笔记、售卖商品还是希望引流，如果小红书不给你流量，一切努力都将白费。系统会通过分析你的账号信息、发布内容等数据，为你的账号打上标签，并据此进行推荐。

推荐规则主要围绕三个维度来评估账号和内容，以下是这三个维度的具体说明：

![](img/1cb88caa3d4af8968ae6525395636bc2_29.png)

*   **用户行为**
    *   **禁止行为**：所有内容不得含有明显的营销词汇。具体包括：**微信号、二维码、代购、水印、某某店铺**。例如，你不能在笔记中直接宣传自己的店铺或引导用户添加微信。
    *   **允许行为**：个人资料中可以留邮箱；可以使用谐音或话术间接引导（但需非常谨慎，每日私聊引导次数极有限）；可以提及微博。

*   **资讯/商品**
    *   **禁止行为**：不能轻易提供医疗和投资建议，这涉及平台严格管控的领域，容易引发诈骗风险。例如，推荐具体医院、股票、房产等都不被允许。
    *   **重要限制**：同一台电脑设备上，最多同时登录**3个**小红书账号。如需管理更多账号，需要使用模拟器（如雷电模拟器）并进行IP隔离等技术处理，且仍需遵守数量限制，否则易被系统检测。

![](img/1cb88caa3d4af8968ae6525395636bc2_31.png)

*   **图文/视频（内容本身）**
    *   **禁止行为**：不得在内容中诱导用户互动，例如“关注领奖”、“抽奖”、“点击领取资料”等。平台对此类引导行为审查严格。
    *   **核心要求**：内容原创度越高，前期获得的基础流量扶持就越大。直接搬运他人的图片和文案（抄袭）会导致账号权重降低。原创度低（如50%）的内容虽有一定流量，但远不如高原创内容。这是新人起步时需要特别注意的。

---

本节课中我们一起学习了小红书运营的第一块基石——**推荐规则**。我们了解了系统通过**用户行为、资讯商品、图文视频**三个维度来评判账号，并明确了哪些行为是禁止的（如留微信号、发二维码、提供医疗投资建议、诱导互动、抄袭），哪些是需要注意的（如设备登录限制）。理解这些规则，能帮助我们在合规的前提下，更有效地规划内容与运营策略。
