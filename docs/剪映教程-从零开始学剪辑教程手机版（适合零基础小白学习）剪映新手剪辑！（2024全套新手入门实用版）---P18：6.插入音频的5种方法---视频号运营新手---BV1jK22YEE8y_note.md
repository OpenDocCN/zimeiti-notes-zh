# 剪映教程：P18：6.插入音频的5种方法 🎵

在本节课中，我们将学习如何在剪映中添加和编辑音频，并制作一个简单的卡点视频。掌握这些方法，能让你的视频更具感染力和节奏感。

## 概述 📋

![](img/3435b9ddb0a1ccc8d669666df1bed04f_1.png)

音频是视频的灵魂。本节将详细介绍在剪映中添加音频的五种主要方法，包括使用内置音乐库、收藏音乐、导入外部音乐、提取视频背景音以及录音。之后，我们还将学习音频的基本剪辑技巧，并运用这些知识制作一个卡点视频案例。

## 添加音频的五种方法

上一节我们了解了音频的重要性，本节中我们来看看具体如何为视频添加声音。剪映提供了多种便捷的音频添加途径。

以下是添加音频的五种主要方法：

1.  **音乐库**：点击底部工具栏的“音频”，进入二级工具栏后选择“音乐”。这里提供了系统推荐的丰富音乐资源。使用前需点击下载，下载完成后才可点击“使用”。
    ![](img/3435b9ddb0a1ccc8d669666df1bed04f_1.png)

2.  **我的收藏**：在音乐库中，“我的收藏”保存了你在剪映音乐库内收藏的音乐。

3.  **抖音收藏**：登录账号后，可以同步你在抖音App中收藏的音乐。

4.  **导入音乐**：此功能包含三个子选项：
    *   **链接下载**：可复制其他音乐平台（如网易云音乐、抖音）的歌曲链接，粘贴后下载使用。请注意版权问题，尽量使用无版权或已获授权的音乐。
    *   **提取音乐**：可以从你手机本地的视频文件中提取其背景音乐。
    *   **本地音乐**：通过iTunes同步到手机的音乐文件，适合已购买正版歌曲的用户。

5.  **录音**：点击“录音”功能，按住红色按钮即可开始录制你的旁白或音效，松手结束，点击对号确认，录音会出现在时间轴上。

## 音频的剪辑与调整

学会了如何添加音频后，接下来我们需要掌握如何对音频进行精细的剪辑和调整，使其更贴合视频内容。

选中时间轴上的音频片段，底部会出现音频编辑工具栏。以下是核心的剪辑功能：

*   **音量**：点击“音量”，可以调整音频大小，默认音量为`100`，最高支持`200`。
*   **淡化**：点击“淡化”，可以为音频添加**淡入**和**淡出**效果，让音频的开始和结束过渡更自然，公式可理解为：`起始音量从0渐强` 和 `结束音量渐弱至0`。
*   **分割**：点击“分割”，可以将一段音频在播放头位置切割成两段，便于独立编辑或删除。
*   **踩点**：点击“踩点”，可以为音频添加标记点。这是制作卡点视频的关键功能，我们将在案例中详细讲解。
*   **删除**：点击“删除”，可以移除选中的音频片段。
*   **变速**：点击“变速”，可以对音频进行加速或减速处理。
*   **复制**：点击“复制”，可以复制一份当前选中的音频片段。

## 案例：制作卡点视频 🎬

掌握了音频的添加与剪辑基础，现在让我们通过一个实际案例，将这些技巧综合运用起来，制作一个流行的卡点视频。

1.  **导入素材与音乐**：点击“开始创作”，导入你的图片或视频素材。然后点击“音频”->“音乐”，为视频添加一首节奏感较强的音乐。
2.  **进行踩点操作**：选中添加的音乐，点击底部工具栏的“踩点”。剪映提供了两种手动方式和一种自动方式：
    *   手动点击“添加点”进行标记。
    *   移动光标到点的位置可删除该点。
    *   **自动踩点**：对于新手，更推荐使用“自动踩点”功能。系统提供`踩节拍Ⅰ`、`踩节拍Ⅱ`和`踩旋律`三种模式，你可以逐一试听，选择节奏最清晰的一种。例如，选择`踩节拍Ⅱ`后，时间轴上会自动生成黄色的节奏点。
3.  **对齐素材与节奏点**：接下来，调整每段素材的时长，使其结尾恰好对齐一个节奏点。拖动素材的尾部时，它会自动“吸附”到附近的节奏点上，操作非常方便。重复此步骤，让所有素材都与节奏点对齐。
4.  **预览与完成**：对齐所有素材后，点击预览播放。一个节奏感鲜明的简单卡点视频就制作完成了。
