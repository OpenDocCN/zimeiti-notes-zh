## 视频剪辑全流程

![](img/08db0f28b83dabe159604656f99256c2_15.png)

以下是使用剪映剪辑一条视频的详细步骤。

![](img/08db0f28b83dabe159604656f99256c2_17.png)

![](img/08db0f28b83dabe159604656f99256c2_19.png)

### 1. 导入素材
打开剪映App，点击“开始创作”按钮。从手机相册中选择你需要剪辑的视频片段，点击“添加”将其导入到剪辑轨道中。

![](img/08db0f28b83dabe159604656f99256c2_21.png)

![](img/08db0f28b83dabe159604656f99256c2_23.png)

如果你有多个片段（例如生活Vlog的开头、中间、结尾），建议先导入并剪辑好第一条，然后点击轨道尾部的“+”号，继续添加第二条片段。这样可以保持工程清晰，避免混乱。

![](img/08db0f28b83dabe159604656f99256c2_25.png)

### 2. 调整构图与比例
导入视频后，首先检查画面构图。如果视频方向不对，可以进行旋转调整。

![](img/08db0f28b83dabe159604656f99256c2_27.png)

![](img/08db0f28b83dabe159604656f99256c2_29.png)

![](img/08db0f28b83dabe159604656f99256c2_31.png)

**操作步骤**：
1.  点击轨道上的视频片段。
2.  在底部工具栏中向右滑动，找到并点击“编辑”按钮（图标通常是一个方框）。
3.  点击“旋转”功能，将画面调整到正确的方向。
4.  如果需要裁剪画面比例，可以点击“裁剪”功能。这里提供了多种比例选项，如 **1:1**、**4:3**、**16:9** 等，选择适合抖音的竖屏比例（如9:16）即可。

![](img/08db0f28b83dabe159604656f99256c2_33.png)

![](img/08db0f28b83dabe159604656f99256c2_35.png)

### 3. 剪掉多余片段
播放视频，找到不需要的部分（如开场沉默、结尾冗余动作）。使用“分割”和“删除”功能进行修剪。

![](img/08db0f28b83dabe159604656f99256c2_37.png)

**核心操作公式**：
`选中视频片段 -> 移动时间线到切割点 -> 点击“分割” -> 选中要删除的片段 -> 点击“删除”`

![](img/08db0f28b83dabe159604656f99256c2_39.png)

![](img/08db0f28b83dabe159604656f99256c2_41.png)

![](img/08db0f28b83dabe159604656f99256c2_43.png)

重复此操作，剪掉所有不必要的开头、中间和结尾部分。

![](img/08db0f28b83dabe159604656f99256c2_45.png)

### 4. 添加背景音乐
视频内容修剪完毕后，为其添加背景音乐。

![](img/08db0f28b83dabe159604656f99256c2_47.png)

![](img/08db0f28b83dabe159604656f99256c2_49.png)

**操作步骤**：
1.  点击底部工具栏的“音频”。
2.  如果你已有下载好的配音文件（如从其他工具生成），可以点击“提取音乐”，从视频中提取音频。
3.  如果没有，则点击“添加音乐”。剪映内置了丰富的音乐库，你可以按分类（美食、美妆、情感等）查找，或直接搜索歌曲。点击音乐可以预览，选定后点击“使用”即可添加。
4.  **剪辑音乐**：如果音乐过长，处理方法与剪辑视频相同：`选中音乐轨道 -> 分割 -> 删除`多余部分。
5.  **调整音量**：选中音乐轨道，点击“音量”，可以调大或调小背景音乐的音量，避免掩盖人声。

![](img/08db0f28b83dabe159604656f99256c2_51.png)

### 5. 自动添加字幕
手动添加字幕效率低下。剪映的“识别字幕”功能可以快速自动生成。

![](img/08db0f28b83dabe159604656f99256c2_53.png)

**操作步骤**：
1.  点击底部工具栏的“文本”。
2.  点击“识别字幕”。
3.  在弹出窗口中，选择“仅视频”（避免识别到背景音乐的歌词），然后点击“开始识别”。
4.  系统会自动生成字幕条。检查是否有错别字，双击字幕条即可进行修改。
5.  你还可以统一或单独修改字幕的字体、颜色、样式等，使其与视频风格更搭配。

![](img/08db0f28b83dabe159604656f99256c2_55.png)

![](img/08db0f28b83dabe159604656f99256c2_57.png)

### 6. 导出高清视频
所有剪辑完成后，在导出前需要进行关键设置，以保证视频清晰度不被压缩。

![](img/08db0f28b83dabe159604656f99256c2_59.png)

![](img/08db0f28b83dabe159604656f99256c2_61.png)

**导出设置代码**：
```
帧率 (Frame Rate): 30
分辨率 (Resolution): 1080p 或更高 (尽可能拉满)
```

1.  点击右上角的“导出”按钮。
2.  在导出设置中，将“帧率”调整为 **30**，将“分辨率”调到最高（如1080p或4K）。
3.  点击“导出”按钮。**在导出过程中，请勿锁屏或切换手机应用**，耐心等待导出完成。
4.  导出后，可以直接通过剪映分享到抖音。

![](img/08db0f28b83dabe159604656f99256c2_63.png)

![](img/08db0f28b83dabe159604656f99256c2_65.png)

## 口播视频提词神器

![](img/08db0f28b83dabe159604656f99256c2_67.png)

对于需要录制口播视频的同学，背诵稿子是一大难题。接下来我们介绍一个能极大提升效率的工具。

![](img/08db0f28b83dabe159604656f99256c2_69.png)

![](img/08db0f28b83dabe159604656f99256c2_71.png)

这个工具叫“爱提词”，它可以帮助你在录制时看着台词念，无需背诵。

![](img/08db0f28b83dabe159604656f99256c2_73.png)

**以下是爱提词App的核心使用方法：**

![](img/08db0f28b83dabe159604656f99256c2_75.png)

![](img/08db0f28b83dabe159604656f99256c2_77.png)

1.  **创建提词板**：打开App，点击右下角“+”号，添加新文稿。输入或粘贴你的口播文案，并设置一个标题（如“情感语录”），然后保存。
2.  **开始提词录制**：打开刚才创建的文稿，点击“开始提词”并选择“录制模式”。你的手机屏幕就会变成一个提词器，文案会以滚动形式出现，你可以一边看一边录制。
3.  **自定义设置**：点击屏幕中央的“设置”图标，可以调整字体大小、滚动速度等，方便阅读。
4.  **使用内置美颜**：在提词录制界面，左下角有“美颜”功能。你可以在这里调整瘦脸、大眼、磨皮、美白等参数，让拍摄效果更自然美观，优于手机原相机。
5.  **寻找灵感**：该App内还设有“爆款文案”板块，分类整理了职场、情感、教育等多个领域的文案，可以作为你创作内容的参考。

![](img/08db0f28b83dabe159604656f99256c2_79.png)

![](img/08db0f28b83dabe159604656f99256c2_81.png)

## 课程总结

![](img/08db0f28b83dabe159604656f99256c2_83.png)

![](img/08db0f28b83dabe159604656f99256c2_85.png)

本节课中我们一起学习了视频剪辑的完整流程：
1.  使用**剪映**导入并裁剪视频素材。
2.  通过`分割`和`删除`功能修剪视频。
3.  添加并编辑背景音乐。
4.  利用`识别字幕`功能快速生成和修改字幕。
5.  按照 **帧率30**、**分辨率最高** 的设置导出高清视频。
6.  介绍了使用**爱提词**App解决口播视频的背稿和拍摄问题。

![](img/08db0f28b83dabe159604656f99256c2_87.png)

掌握这些基础而重要的技巧，你就能独立完成一个结构完整、视听体验良好的短视频制作了。请务必动手练习，将每一步操作都实践一遍。
