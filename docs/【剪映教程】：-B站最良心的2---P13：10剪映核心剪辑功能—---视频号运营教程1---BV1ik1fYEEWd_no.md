# 剪映核心剪辑功能教程：P13：色度抠图功能 🎬

在本节课中，我们将要学习剪映APP中一项强大且新近添加的核心剪辑功能——色度抠图。这项功能特别适合制作特效类视频，它能够将绿幕（或其他纯色背景）素材中的主体完美地抠取出来，并合成到其他视频中，过去这通常需要借助PR或AE等专业软件才能完成。

## 功能概述与原理

上一节我们介绍了剪映的基础剪辑功能，本节中我们来看看色度抠图。色度抠图的核心原理是识别并去除视频素材中的特定颜色（通常是绿色或蓝色），从而将前景物体分离出来。其核心操作可以概括为以下步骤：

1.  **取色**：使用取色器选取需要去除的背景颜色。
2.  **强度调整**：通过调整强度滑块，控制颜色去除的程度。
3.  **边缘优化**：使用阴影或边缘羽化功能，让合成效果更自然。

## 实战演示：合成飞机飞过天空

了解了基本原理后，我们通过一个具体案例来学习如何操作。我们将把一段绿幕飞机视频合成到一段天空背景视频中，制作出飞机飞过的效果。

![](img/fd87f20d6dc7cb9fca3047c56464adb3_3.png)

以下是详细的操作步骤：

![](img/fd87f20d6dc7cb9fca3047c56464adb3_5.png)

![](img/fd87f20d6dc7cb9fca3047c56464adb3_7.png)

1.  **导入主视频素材**：首先，在剪映中导入作为背景的天空视频。
2.  **添加绿幕素材**：点击底部工具栏的 **`画中画`** 功能，导入包含飞机的绿幕视频素材。此时，绿幕素材会叠加在背景视频之上。
3.  **启用色度抠图**：在时间轴上选中绿幕素材，在底部工具栏中找到 **`色度抠图`** 选项（通常位于蒙版功能旁边）。
4.  **取色与抠除**：
    *   点击取色器（一个圆形图标），将其移动到画面中的绿色背景上并点击，以选取要去除的颜色。
    *   随后，拖动 **`强度`** 滑块。随着强度值增加，绿色背景会逐渐被去除。调整至飞机轮廓清晰、绿色背景完全消失为宜。
5.  **优化边缘**：如果抠图后飞机边缘有残留的绿色杂边，可以适当调整 **`阴影`** 滑块，这能使合成边缘更柔和、更自然。
6.  **调整位置与剪辑**：
    *   拖动画面中的飞机，将其调整到合适的位置（例如，从画面右侧飞入）。
    *   使用 **`编辑`** -> **`裁剪`** 功能，裁掉绿幕素材中不需要的部分（如水印）。
    *   在时间轴上裁剪掉视频首尾多余的部分，使飞机飞行动画与背景视频时长匹配。
7.  **添加动画（可选）**：为了让飞机出现更自然，可以为其添加一个 **`渐显`** 或 **`向右滑动`** 的入场动画。

![](img/fd87f20d6dc7cb9fca3047c56464adb3_9.png)

完成以上步骤后，预览视频，你就能看到飞机在天空飞过的合成效果了。

![](img/fd87f20d6dc7cb9fca3047c56464adb3_10.png)

## 应用场景与总结

本节课中我们一起学习了色度抠图功能。这项功能极大地简化了特效视频的制作流程，让用户能在手机端轻松实现过去需要专业软件才能完成的绿幕合成。

![](img/fd87f20d6dc7cb9fca3047c56464adb3_12.png)

总结来说，色度抠图功能的核心用途是：**方便地将绿幕素材中的特效主体抠出，并通过画中画功能进行合成**。这广泛应用于抖音、B站等平台的短视频创作中，用于制作各种有趣、炫酷的特效内容。
