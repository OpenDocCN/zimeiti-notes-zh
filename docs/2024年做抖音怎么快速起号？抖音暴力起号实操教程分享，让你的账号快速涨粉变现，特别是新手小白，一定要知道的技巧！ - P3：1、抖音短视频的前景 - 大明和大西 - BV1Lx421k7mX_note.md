# 抖音运营入门：P3：1、抖音短视频的前景

![](img/dcc3f41598980c0686347be981fcbc3f_1.png)

在本节课中，我们将要学习抖音短视频平台的发展前景与核心优势，帮助你理解为何选择抖音作为内容创作的起点。

![](img/dcc3f41598980c0686347be981fcbc3f_3.png)

了解抖音短视频的前景，首先需要通过一张图片来认识。

![](img/dcc3f41598980c0686347be981fcbc3f_1.png)

这张图片来自抖音创作者大会的数据报告。报告显示，有超过2200万人在抖音的合计收入超过了417亿元。抖音希望将这个数字翻一番，让创作者的总收入达到800亿元。

![](img/dcc3f41598980c0686347be981fcbc3f_3.png)

这体现了平台巨大的流量扶持力度。接下来介绍第一个核心知识点：抖音用户主要分为两类。第一类是**玩家**，他们单纯在抖音上娱乐，没有考虑通过平台赚钱。第二类是**创作者**，即像我们这样希望通过制作内容来获取收益的人。明确这两类人的区别很重要，后续内容中还会提及。

那么，在众多短视频平台中，我们为什么选择抖音呢？这类似于开设实体店铺。选择店面位置时，你肯定会找一个流量大、适合经营的地点。做短视频同理，必须选择一个流量庞大的平台。如果平台自身流量不足，就无法分给创作者，我们也难以赚到钱。抖音的日活跃用户超过8亿，月活跃用户更是达到了40亿人次。这是一个极其庞大的流量池。

流量巨大，竞争压力是否也会很大？答案是不会。平台上的**创作者**数量约为5000万。这里的“创作者”并非专指网红达人。例如，邻居家的老太太用手机拍摄自己做的饭菜并上传到抖音，她也算创作者。因此，你90%的竞争压力实际上来自于身边的普通用户。

接着，做抖音短视频是否需要购买专业设备？答案是不需要。你只需要一部智能手机和一个手机号，就可以注册抖音账号。抖音倡导全民通过短视频分享和记录生活，门槛很低。

那么，当前有哪些值得关注的风口呢？首先是“抖音计划”。该计划表示，抖音将帮助1000名优质的艺术创作者实现粉丝增长超过100万。有的同学可能会担心自己不会创作。这完全不必担心，正因为我们有所欠缺，才需要学习。虚心向在该领域有经验的人请教并不可耻，这能让我们少走很多弯路。保持谦虚好学的态度至关重要。

市场空间巨大，内容产出也相对容易。我们正处于大数据时代。系统会根据你的喜好推荐内容，例如，喜欢看帅哥就会刷到腹肌帅哥，喜欢看美女就会刷到长腿美女。这导致我们每个人看到的内容都相对片面。因此，无论你是宝妈、实体店主还是普通上班族，都可以找到自己的创作方向：宝妈可以分享宝宝和美食，实体店主可以展示产品，上班族可以记录生活。在月活40亿的用户群体中，只要你的内容不触犯法律和道德底线，不是低俗内容，总会有一部分人喜欢上你。

有的同学可能会问，既然老师说一定会有人喜欢，为什么我的粉丝只有几十个？这是因为喜欢你的人还没有刷到过你的作品。对于新手小白而言，心情迫切可以理解，但不要把涨粉作为首要目标。我们的第一个目标应该是**提升流量和播放量**。播放量涨起来之后，粉丝自然就会增长。

本节课中我们一起学习了抖音平台的巨大潜力、用户分类、低门槛优势以及新手起步的核心策略。记住，在起步阶段，专注于提升内容曝光和播放量是关键的第一步。
