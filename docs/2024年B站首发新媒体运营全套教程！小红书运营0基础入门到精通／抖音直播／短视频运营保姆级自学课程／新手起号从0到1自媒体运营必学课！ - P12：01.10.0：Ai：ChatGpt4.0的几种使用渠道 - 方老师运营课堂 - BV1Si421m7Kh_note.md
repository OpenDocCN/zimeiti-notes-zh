### AI对运营与开发的重要性
在之前的课程中，我们提到，在当前时代，有两个行业最需要接触和使用AI智能对话模型：程序员和新媒体运营。

![](img/e1112d5dada261f154094f2504a1f39c_3.png)

对于程序员，AI可以帮助编写代码、程序和测试Bug。其核心应用可以表示为：
```python
# 示例：请求AI协助编写代码
prompt = “请用Python写一个快速排序算法”
```

对于新媒体运营，AI可以协助撰写短视频脚本、制作PPT、生成表格和编写报告等。这意味着，掌握AI工具后，运营人员无需具备特定行业的深厚专业知识，也能借助AI完成相关工作。AI可以成为你的私人知识专家。

### 渠道一：微软Edge浏览器（Copilot）
上一节我们介绍了AI对运营工作的价值，本节我们来看看第一个推荐的使用渠道：微软Edge浏览器内置的Copilot功能。

Copilot是微软基于ChatGPT技术，为Windows系统用户打造的功能。它基于GPT-4模型开发，在智能程度上与GPT-4相匹配，是普通用户可便捷使用的渠道之一。

使用Copilot有两个前提条件：
1.  登录微软账号（可使用Outlook邮箱注册）。
2.  需要使用“梯子”（VPN）才能访问该服务。

Copilot通常集成在浏览器侧边栏或系统任务栏。它提供三种对话模式：
*   **更有创造力**：AI的回答更具开放性和创造性。
*   **平衡**：介于创造力和精确性之间，适合一般用途。
*   **更精确**：回答更为严谨和准确，适合专业内容。

例如，我们可以用“更有创造力”模式请求生成一个短视频脚本：
```
请帮我写一个关于装修的短视频脚本，时长控制在3分钟左右。
```
Copilot的优势在于可以分析你当前打开的网页内容。但其缺点也很明显：
1.  **精准度稍逊**：虽基于GPT-4，但经过微软调教和限制，在某些方面不如官方ChatGPT精准。
2.  **连续对话能力受限**：无法在同一对话窗口中进行过多次的持续提问。
3.  **记忆功能有所缺失**：在利用历史对话优化后续回答方面能力较弱，且缺乏插件功能。

如果您的Edge浏览器没有Copilot图标，可以尝试下载最新的Edge Dev（测试版）。您可以在微软官网的Edge下载页面，找到“Dev频道”版本进行下载。或者，在浏览器设置中检查并更新到最新版本。

### 渠道二：第三方集成平台
除了官方和浏览器集成渠道，我们还可以借助第三方开发者搭建的平台使用GPT-4。以下是几个可用选项：

**1. 灵感AI**
这是一个集成了多种AI模型的平台，包括GPT-3.5、GPT-4、GPT-4联网版以及国内主流模型。

该平台的优势如下：
*   **模型丰富**：集成市面上多数主流AI模型。
*   **GPT-4可免费试用**：新注册用户在一定时间内有免费使用额度。
*   **无需“梯子”**：国内网络可直接访问。

但其主要缺点是**响应速度较慢**，生成内容可能需要等待数分钟。平台也提供付费套餐，如19.9元包月。

**2. Wee AI**
这个平台同样提供GPT-3.5和GPT-4模型。

其优势在于：
*   **响应速度快**：达到正常使用水准。
*   **具备语音功能**：支持语音回答。

![](img/e1112d5dada261f154094f2504a1f39c_5.png)

![](img/e1112d5dada261f154094f2504a1f39c_7.png)

但需要注意：
*   **需要“梯子”**。
*   **免费额度有限**：初始赠送的Token（计算单位）通常仅适用于GPT-3.5模型。
*   **计费方式需留意**：Token消耗较快，需关注使用情况。

![](img/e1112d5dada261f154094f2504a1f39c_9.png)

![](img/e1112d5dada261f154094f2504a1f39c_11.png)

**3. ChatGPT共享账号平台**
这是一个由网络社区维护的共享账号平台，提供他人分享的ChatGPT账号。

![](img/e1112d5dada261f154094f2504a1f39c_13.png)

![](img/e1112d5dada261f154094f2504a1f39c_15.png)

平台使用颜色标识账号状态：
*   **绿色**：空闲，可用。
*   **橙色**：近期被使用过。
*   **红色**：正在被使用或使用频繁。
*   **金色边框**：代表付费的GPT-4 Plus账号，可用但稀缺。

![](img/e1112d5dada261f154094f2504a1f39c_17.png)

![](img/e1112d5dada261f154094f2504a1f39c_19.png)

使用方法是找到绿色或金色的账号，点击进入并使用。需要注意的是，免费的GPT-3.5账号较多，但GPT-4 Plus账号由于官方对话次数限制（约每3小时30条消息），非常抢手且可能已达上限。此方法同样**需要“梯子”**。

### 渠道三：国内AI大模型
如果无法使用“梯子”，我们还可以选择国内公司开发的AI大模型。它们无需特殊网络条件，虽然并非ChatGPT，但在许多场景下也能提供有效协助。

以下是几个主流选择：

**1. 百度文心一言**
百度的AI对话模型，提供基础版和付费的“文心一言4.0”版本。在撰写文案、获取灵感等基础运营任务上表现足够。你可以尝试其免费版本，或开通付费版体验更强大的功能。

![](img/e1112d5dada261f154094f2504a1f39c_21.png)

![](img/e1112d5dada261f154094f2504a1f39c_23.png)

**2. 字节跳动豆包AI**
字节跳动出品的AI助手。我们用它生成同样的装修脚本，其输出结构清晰，包含了景别、画面内容、台词、时长等要素，对于视频脚本创作有直接帮助。

**3. 其他可选模型**
此外，还有**阿里巴巴的通义千问**、**昆仑万维的天工大模型**等。你可以都尝试一下，选择一个用起来最顺手、最能提升你工作效率的模型进行深入使用。

### 官方GPT-4体验对比
最后，我们对比一下官方ChatGPT 4.0的生成效果。使用同样的请求：
```
请帮我写一份关于装修的短视频脚本文案，要求镜头丰富，语言吸引人。
```
官方GPT-4生成的脚本在场景设计、镜头语言和叙事节奏上，通常展现出更强的创意性和细腻度。这体现了其当前的技术领先性。选择工具时，我们应秉持“效率优先”原则，哪种工具能更好提升工作效果，就选择哪种。

## 总结
本节课我们一起学习了体验和使用ChatGPT 4.0的多种渠道。我们介绍了三大类方法：
1.  **微软Edge Copilot**：方便但略有限制，需“梯子”。
2.  **第三方平台**：如灵感AI、Wee AI和共享账号，提供了免费或低成本体验GPT-4的途径，但各有使用条件和限制。
3.  **国内AI大模型**：如文心一言、豆包等，无需“梯子”，易于获取，是可靠的备选方案。
