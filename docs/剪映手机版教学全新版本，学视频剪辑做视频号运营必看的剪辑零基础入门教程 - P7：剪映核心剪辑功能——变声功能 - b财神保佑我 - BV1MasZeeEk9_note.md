# 剪映手机版教学：P7：核心剪辑功能——变声功能 🎤

在本节课中，我们将要学习剪映App的核心剪辑功能之一——变声功能。这个功能可以帮助你改变视频中的人声，无论是为了优化音质、创造趣味效果，还是模拟多人对话场景，都非常实用。

---

上一节我们介绍了其他剪辑工具，本节中我们来看看如何为视频添加变声效果。很多初学者在制作短视频时，可能会觉得自己的原声不够理想，或者希望为视频增加一些创意元素。变声功能正是解决这些问题的好帮手。

## 功能简介与用途

变声功能允许你改变视频或音频轨道中的人声音色。其核心应用场景主要包括：
*   **优化原声**：解决对自己声音不满意的问题。
*   **创造趣味效果**：制作搞笑、玩梗类视频，例如将男声变为女声、萝莉音或怪物音效。
*   **模拟多人对话**：通过为同一段视频素材应用不同的变声效果，可以实现一人分饰多角，模拟出对话场景。

## 操作步骤详解

以下是使用变声功能的完整步骤：

1.  **导入并选中素材**：打开剪映App，导入一段包含人声的视频素材。在时间轴上点击选中该素材。

    ![](img/030508b739053a4aee887a23f1277e9f_1.png)

2.  **找到变声功能**：选中素材后，底部会弹出二级编辑菜单。在菜单中找到并点击 **`变声`** 功能按钮。

    ![](img/030508b739053a4aee887a23f1277e9f_3.png)

3.  **选择变声效果**：点击后，界面会显示多种预设的变声效果，例如“大叔”、“萝莉”、“女生”、“男生”、“怪物”等。你可以点击不同的效果进行试听。
    *   例如，选择“女生”效果，原男声会变为女声。
    *   再选择“萝莉”效果，声音会变得卡通化。

    ![](img/030508b739053a4aee887a23f1277e9f_5.png)

![](img/030508b739053a4aee887a23f1277e9f_5.png)

4.  **确认与应用**：试听满意后，该变声效果会自动应用到当前选中的视频片段上。你可以通过播放来预览最终效果。

## 功能总结与技巧

变声功能不仅解决了声音不理想的尴尬，更能极大地丰富视频的创作可能性。通过巧妙地运用变声，你可以轻松实现一人演绎家庭对话、情侣聊天等多种互动场景，这大大降低了新手用户制作复杂音频内容的门槛，有效提升了视频的趣味性和专业质感。
