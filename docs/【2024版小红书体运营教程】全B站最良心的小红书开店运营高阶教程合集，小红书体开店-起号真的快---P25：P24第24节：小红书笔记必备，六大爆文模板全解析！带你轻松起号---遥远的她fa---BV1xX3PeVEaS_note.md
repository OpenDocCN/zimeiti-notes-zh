# 小红书运营教程：P25：小红书笔记必备爆文模板全解析

在本节课中，我们将学习小红书笔记的两种核心爆文模板：专业教程类与视频笔记类。掌握这些模板，能帮助你在特定场景下，更有效地组织内容，提升笔记被推荐至首页的几率。

## 专业教程类模板解析

上一节我们探讨了笔记的基础结构，本节中我们来看看如何撰写步骤清晰的专业教程类笔记。这类模板适用于有明确操作流程的领域。

专业教程类模板适用于美妆、瘦身、美食、软件操作、美容、摄影等具有步骤性的内容领域。

以下是撰写专业教程类笔记的具体建议：

*   **首图类型**：建议采用**对比效果图**或**步骤分解图**。
    *   **对比效果**：例如展示使用产品前后的对比，或自家产品与别家产品的效果对比。
    *   **步骤分解**：例如摄影教程，首图可依次展示：第一步购买配件、第二步组装设备、第三步选择场景、第四步拍摄成片。

*   **标题要点**：标题应突出内容的特别之处或核心价值。关于标题的具体起法，可以参考我们之前的课程内容。

*   **正文结构**：正文内容需要条理清晰，建议包含所用工具、材料及详细步骤。
    *   避免内容混杂，应分点叙述。例如，先阐明核心要点，再分1、2、3、4小点详细说明，每个小点可配图辅助。
    *   可以在正文开头部分，单独列出“准备工作”，说明需要哪些工具或材料。

## 视频笔记类模板解析

了解了图文教程的写法后，我们再来看看当下非常流行的视频笔记应如何制作。视频笔记的核心在于快速吸引并留住观众。

视频笔记的制作有两个基本要素需要注意：

*   **视频时长**：初期建议**1分钟**以内。获得1分钟权限后，可根据内容需要，制作**5分钟**或更长的视频。
*   **视频比例**：必须采用**3:4**的竖屏比例。其公式为 `高度 : 宽度 = 3 : 4`，这与传统的4:3横屏比例不同。

现在很多视频面临的问题是开头几秒无法吸引观众，导致被直接划走。那么，如何制作有吸引力的视频呢？

关键在于 **“重点前置”** ，即将最精彩、最核心的内容放在视频开头。我们可以参考“毒舌电影”的叙事手法。

例如，在介绍一部电影时，不应从抒情或背景铺垫开始，而应直接切入剧情最高潮或最有趣的片段。毒舌电影的视频开头可能是：“他已经饿了两天，身无分文，想吃霸王餐……”。仅用前五秒钟，就抛出了故事中最具冲突性的场景，瞬间抓住观众好奇心，之后才逐步介绍电影背景和详细信息。

我们可以将这种方法应用到产品推广中。对比以下两种开头方式：

*   **平淡开头**：“第一步，清洁面部；第二步，涂抹我们的产品……”
*   **重点前置开头**：“这是使用我家产品的效果，这是使用别家产品的效果。为何差距如此之大？本期视频告诉你答案。”

显然，第二种方式通过开场展示强烈对比并提出核心问题，能更有效地吸引观众继续观看。**重点前置**的技巧，对于优化作品结构和提升叙事吸引力有极大帮助。

---

本节课中我们一起学习了小红书的两大爆文笔记模板。**专业教程类模板**强调步骤清晰、对比直观和结构分明；**视频笔记类模板**则核心在于遵守格式（3:4竖屏），并运用**重点前置**的原则，在开头几秒内用最精彩的内容抓住观众注意力。掌握这两种模板，能帮助你更高效地创作出优质笔记。
