# 小红书运营起号正确方法：P7：05.小红书后台的基础操作（手机端） 📱

![](img/42ae3addb6fbef16cd4489e417718c09_0.png)

在本节课中，我们将学习如何使用手机端的小红书商家后台——千帆系统，完成店铺的基础设置。这是开启店铺运营的第一步，内容简单明了，适合初学者上手。

上一节我们介绍了开店前的准备，本节中我们来看看如何通过手机进行后台的初始配置。

## 系统入口与首页概览

首先，需要在手机应用商店搜索并下载“小红书千帆”应用。这是商家管理店铺的官方后台工具。

打开应用后，你会看到首页。对于一个新店，首页可能显示“初始化”状态。页面主要包含“消息”和“全部工具”等核心入口。

## 消息中心与客服管理

“消息”中心至关重要。点击进入后，你可以看到所有客服接待通知。当有用户咨询店铺商品或服务时，你需要在这里进行回复。此外，店铺的各类通知，如违规提醒，也会在此处显示。

## 核心工具与初始设置

“全部工具”是后台功能的集合中心。点击进入后，可以看到发布作品、管理商品、处理订单、查看数据等多项功能。

店铺运营的第一步是完成基础信息设置。在“成长”或相关区域，系统会引导你完成一系列初始化任务。

以下是完成初始设置的关键步骤列表：

1.  **填写发货地址**：这是开店后的首要操作。你需要设置一个默认的发货地址。即使你做一件代发，也需要填写一个有效地址。填写时，联系人、所在地区和详细地址需按实填写，手机号码必须准确。**特别注意**：“买家留言”处**不能**填写“请联系客服”，必须填写具体地址，否则会影响退货流程，平台审核也可能不通过。
2.  **设置运费模板**：在物流板块，你可以创建运费模板。例如，你可以设置新疆、西藏等偏远地区不配送，或让买家承担邮费。系统有默认模板（如“全国包邮”），但建议根据你的商品情况自定义。公式可以理解为：`运费策略 = 基础地区(包邮) + 特殊地区(不配送/付费)`。
3.  **配置支付方式**：为确保顾客能顺利下单，务必在支付设置中，按照流程开通“微信支付”和“支付宝支付”。这通常需要绑定你的身份证和银行卡信息。
4.  **缴纳店铺保证金**：若计划长期运营，需要根据经营类目缴纳相应的保证金。不同类目的保证金金额不同，这是平台对商家的一种履约保障。
5.  **创建营销优惠券**：在“营销”工具中，你可以创建各种优惠券来吸引顾客。例如：
    *   店铺通用优惠券
    *   指定商品优惠券
    *   店铺新人券
    你可以查看正在生效、已失效的优惠券，并参与平台推广活动。

## 总结

![](img/42ae3addb6fbef16cd4489e417718c09_2.png)

本节课中我们一起学习了小红书千帆后台（手机端）的基础操作。核心步骤包括：下载应用、关注消息中心、通过“全部工具”完成发货地址填写、运费模板设置、支付方式开通、保证金缴纳以及优惠券创建。这些是店铺能够正常接待客户、完成交易的基础，请务必认真完成。多操作、多摸索，这些设置并没有太大难度。
