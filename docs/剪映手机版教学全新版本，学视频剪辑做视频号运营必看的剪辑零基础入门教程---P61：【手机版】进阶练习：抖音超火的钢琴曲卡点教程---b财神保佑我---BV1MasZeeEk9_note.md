# 剪映手机版教学：P61：进阶练习：抖音钢琴曲卡点教程 🎹

在本节课中，我们将学习如何制作抖音上非常流行的钢琴曲卡点视频。这个效果的核心是运用蒙版功能，将视频画面分割成类似钢琴琴键的样式，并让这些“琴键”随着音乐的节奏依次出现。

## 概述
我们将通过一个实战案例，学习如何利用蒙版分割画面，并结合音乐的节拍点，制作出富有节奏感的卡点视频。整个过程会涉及素材导入、音乐卡点、蒙版绘制与对齐，以及素材替换等关键步骤。

## 准备工作
上一节我们介绍了蒙版的基础应用，本节中我们来看看如何将其用于复杂的节奏卡点。

首先，我们需要准备制作素材。
*   三张比例与尺寸完全一致的图片（例如：森林、沙漠、冰川）。
*   一段节奏清晰的背景音乐。

### 第一步：导入素材与音乐
我们点击“开始创作”，导入第一张图片（例如森林图片）。
接着，我们需要为视频添加背景音乐。点击“音频”，选择“提取音乐”，从手机中导入准备好的音乐文件。

![](img/f736f386a72b00eaf0d6cd9cea3ce587_16.png)

![](img/f736f386a72b00eaf0d6cd9cea3ce587_18.png)

### 第二步：为音乐添加卡点
音乐是卡点视频的灵魂，准确标记节拍点至关重要。
选中音频轨道，点击工具栏中的“节拍”。
在节拍设置界面，通过手动点击“添加点”的方式，根据音乐鼓点标记出所有节奏位置。本例中，我们标记了7个节拍点。

### 第三步：使用参考线分割画面
为了将画面均匀分割成7份，我们需要借助文本工具创建视觉参考线。
点击“文本”，新建一个文本。
在输入框中，连续输入8个大写英文字母 **`I`**（7个间隙需要8个参考点）。
进入“样式”中的“排列”选项，调整字间距，让这8个“I”均匀分布并撑满整个画面。完成后，画面被均匀分成了7个等份区域。

### 第四步：应用并复制矩形蒙版
现在，我们开始使用蒙版来切割画面。
选中主视频轨道，点击“蒙版”，选择“矩形”蒙版。
调整矩形蒙版的大小和位置，使其宽度与一个参考间隙相同，并覆盖最左侧的第一个区域。确保蒙版边缘与参考线对齐，以留下均匀的黑色缝隙。
确认蒙版位置后，勾选确认。
接下来，我们需要复制出多个相同的图层。选中这个已添加蒙版的视频，点击“复制”。
然后选中复制出的片段，点击“切画中画”，将其拖到与主视频开头对齐。
重复“复制”和“切画中画”操作，直到我们总共拥有7个视频图层（1个主轨道+6个画中画轨道）。

### 第五步：调整各图层蒙版位置
这是关键步骤，需要耐心调整，使所有“琴键”排列整齐。
从下往上，依次选中每一个视频图层。
为每个图层点击“蒙版”，然后单独移动其矩形蒙版的位置，让它们从左到右依次排列，共同拼合成完整的画面。确保每个蒙版之间的黑色缝隙粗细一致。
所有蒙版调整完毕后，可以删除之前创建的文本参考线。

### 第六步：根据卡点替换素材
现在，我们将让不同的“琴键”随着音乐节奏，变化成不同的图片。
从第二个节拍点开始，我们计划让某些“琴键”变成沙漠图片。例如，我们想让最左侧的第一个“琴键”在第一个节拍点变化。
选中对应位置的视频图层（最左侧的那个），将时间线移动到第一个节拍点位置。
点击“分割”，将该图层从此处切开。
选中分割出来的后半段视频，点击工具栏中的“替换”。
**重要提示**：选择替换为“沙漠”图片。务必确保替换的图片与原图（森林）的比例和尺寸完全相同，否则会导致画面错位或出现黑边。
按照设计好的节奏顺序，重复“分割”和“替换”操作，将其它图层的相应片段替换成“沙漠”或“冰川”图片。例如，可以让“琴键”从左到右、再从右到左依次变化。
最后，将所有视频图层的时长统一拖拽至与音乐结尾对齐。

## 核心原理与技巧总结
本节课中我们一起学习了钢琴曲卡点视频的制作方法。

其核心原理是：**利用蒙版分割画面 + 依据音乐节拍点替换分割后的画面内容**。
关键技巧在于：
1.  **音乐卡点是基础**：必须准确标记音乐的节奏点。
2.  **参考线保证精度**：使用文本工具辅助，能确保蒙版分割均匀。
3.  **素材规格需统一**：所有替换的图片或视频必须保持比例和分辨率一致，这是使用“替换”功能不穿帮的前提。
4.  **灵活运用原理**：掌握此原理后，你可以创造竖排、斜排或任何形状的蒙版组合，并适配各种类型的音乐，制作属于自己的创意卡点视频。
