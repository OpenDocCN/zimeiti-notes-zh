# 抖音全套运营教程：P22：13.原创解说文案

## 概述
在本节课中，我们将要学习如何创作抖音解说视频的文案。我们将探讨三种主要的文案来源方式，并重点讲解如何撰写“说人话”的纯原创文案，这是提升视频收益的关键。

---

很多同学表示不想使用“一搜猫”素材库。当然有其他方式。我自己拥有一个纯原创的解说文库。

这个文库中的所有文案都是我原创的内容。我可以将这个文库提供给你们使用。这个文库没有使用限制，内容纯属原创。我和我的团队成员会持续更新这个文库。每篇文案大约有两三千字。

你们只需要根据我的文案，去寻找对应的视频素材进行合成即可。这是一种比较方便的操作方式。

除了上述方式，也有同学表示既不想用“一搜猫”的素材，也不想用我的文库。他们询问是否可以自己撰写纯原创文案。

确实有同学向我表达过想自己写纯原创文案的想法。我非常欣赏和支持这种有底气的同学。

![](img/5048acef8ec7d30b2aa861926b531d56_1.png)

如果你想自己撰写纯原创文案，我支持你。因为纯原创文案相比我们找来的素材，其视频播放收益单价通常会高出两倍。这是一个显著的优势。

我本人开设了一门专门的文案课程。

![](img/5048acef8ec7d30b2aa861926b531d56_1.png)

无论是我自己还是我的团队成员撰写文案，其方法和技巧都源自这门课程。很多同学害怕自己写文案，觉得文案创作是一项非常专业的工作。

![](img/5048acef8ec7d30b2aa861926b531d56_3.png)

他们认为文案需要像写作文一样专业。其实并非如此。对于害怕写文案的同学，请记住这句话：**解说文案的核心是“说人话”**。

“说人话”的意思就是使用大白话，即日常交流的口语。

![](img/5048acef8ec7d30b2aa861926b531d56_3.png)

举个例子。当我们向父母或朋友推荐一部电视剧，希望他们也去看时，我们通常不会采用正式的报告形式。

我们不会这样说：“妈妈，我今天看了一部电视剧，我觉得很好看。这部电视剧名叫《XXX》，由XX导演执导，于X年X月上映。男主角叫A，女主角叫B。第一集讲述了……第二集讲述了……”

我们通常会这样说：“妈妈，我今天看了一部电视剧，特别好看！男主角超级帅，女主角也特别美，他们俩好般配。剧情是这样的：两个人相爱十年准备结婚，结果男的婚前出轨了。女的知道后去捉奸，当场扇了男的一巴掌说‘离婚吧，这婚我不结了！’”

这就是我们俗称的“口水话”。解说文案不需要像写作文那样讲究起承转合。如果解说像写官方简介一样枯燥，谁还会来看你的视频呢？观众不如直接去看原片。

一个典型的例子是《甄嬛传》。这部剧很好看，但如果我们有时间看第一集，没时间看第二集，到第三集可能就跟不上剧情了。当时，很多做《甄嬛传》影视解说的人都赚到了钱。原剧中有不少文言文对白，并非所有观众都能完全理解。而解说者用大白话讲清楚这一集发生了什么事，观众就能明白。这就是价值。

在我的团队中，做视频解说的人是最多的。

为什么选择做解说的人最多？就是因为影视解说或其他类型的解说视频，是**最赚钱的领域之一**。你们可以看到，做解说的人确实很多。

这里有纪录片解说、综艺解说、电视剧解说、动物解说、科普解说、历史解说、动漫解说等等，种类非常多。

大家都选择做解说，原因在于解说并不难，且收益可观。这里也分享一个方向：如果你们现在想做解说视频，影视推广是一个选择，做影视解说可以起到版权澄清的作用。

如果不想做影视领域，

## 总结
本节课我们一起学习了抖音解说文案的创作。我们了解了三种文案来源：使用现有素材库、利用原创文案库以及自己撰写纯原创文案。我们重点强调了**纯原创文案在提升收益方面的重要性**，并掌握了其核心要领——**用“说人话”的方式，以口语化的大白话将故事讲清楚**。最后，我们还探讨了动物解说这一潜力巨大的细分领域。记住，优秀的解说文案不在于辞藻华丽，而在于清晰、生动地传递信息。
