# 亚马逊跨境电商入门到精通：P15：3.3、了解亚马逊卖家中心 🏪

在本节课中，我们将详细介绍亚马逊卖家中心的核心功能与界面布局。卖家中心是您管理店铺、处理订单、优化商品和开展营销活动的主要操作平台。掌握其使用方法，是成功运营亚马逊店铺的基础。

---

## 登录与界面概览

首先，使用您注册时填写的电子邮件ID和密码，登录亚马逊卖家中心。

登录后，您将看到顶部有一系列功能选项卡。通过这些选项卡，您可以浏览和使用卖家中心的所有功能。

---

## 核心功能模块详解

上一节我们介绍了登录与界面概览，本节中我们来详细看看每个核心功能模块的具体作用。

### 目录管理

以下是**Catalog（目录）** 选项卡的主要功能：
*   **添加商品**：在此处创建新的商品列表。
*   **管理草稿**：查看和编辑尚未完成的商品信息草稿。
*   **查看销售申请状态**：跟踪您提交的、需要亚马逊审批才能销售的商品或类别的申请进度。

### 库存管理

在**Inventory（库存）** 选项卡上，您可以管理所有库存信息。
*   **管理库存**：查看所有在售商品的库存状况。
*   **添加商品**：支持单个添加或通过文件批量上传商品。
*   **库存报告**：生成各类库存报告，帮助您更好地管理商品目录。
*   **全球销售**：了解如何将商品销售到亚马逊的其他国际站点。

### 定价策略

**Pricing（定价）** 选项卡帮助您监控和调整商品价格。
*   **定价状况**：查看商品定价是否存在问题。
*   **定价仪表板**：总览所有商品的定价表现。
*   **定价警报**：设置价格提醒，以便及时应对市场变化。
*   **自动定价**：创建定价规则，让系统根据您设定的条件自动调整价格。

### 订单与客户服务

**Orders（订单）** 选项卡集中处理所有客户订单。
*   **管理订单**：查看已收到的订单详情。
*   **处理退货**：处理客户的退货请求。
*   **订单报告**：下载订单数据报告，用于财务对账或业务分析。

### 广告与促销

**Advertising（广告）** 选项卡帮助您推广商品。
*   **广告活动管理**：创建和管理商品推广活动。
*   **Sponsored Products（竞价商品）**：这是亚马逊的按点击付费广告计划，公式可表示为 `广告费用 = 点击次数 × 每次点击出价`。
*   **促销工具**：使用优惠券、折扣等促销方式吸引顾客。
*   **A+内容管理器**：品牌所有者可以在此为商品页面添加图文并茂的增强描述。
*   **店铺构建器**：使用亚马逊的自助工具创建或编辑您的品牌旗舰店。

### 计划与报告

**Programs（计划）** 选项卡汇集了亚马逊的各项卖家计划。
*   **FBA（亚马逊配送）**：在此注册和管理亚马逊物流服务。

**Reports（报告）** 选项卡提供各类业务数据。
*   **下载报告**：获取付款报告、业务报告和税务报告。

### 账户绩效与支持

**Performance（绩效）** 选项卡反映了您账户的健康状况。
*   **账户状况**：总览账户绩效指标。
*   **客户反馈**：查看买家评价和任何未解决的索赔。
*   **客户之声**：识别客户集中反馈的商品问题，以便改进。

**卖家大学**：我们强烈建议您访问此板块，观看教学视频以提升运营技能。

### 工具与服务

**Apps and Services（应用和服务）** 选项卡提供扩展工具。
*   **第三方应用**：集成可自动化日常任务（如定价、图片处理）的应用程序。
*   **服务提供商网络**：查找亚马逊认证的服务商，获取专业服务以促进业务增长。

### 其他实用功能

*   **B2B**：为批量采购的企业买家设置企业折扣价。
*   **搜索栏**：在页面右上角，可快速搜索您需要的功能或信息。
*   **Messages（消息）**：用于与买家沟通。
*   **Help（帮助）**：获取亚马逊官方的客服支持。
*   **Settings（设置）**：管理账户的所有设置，包括付款信息、发货设置、退货政策、税务信息和用户权限等。
*   **Navigation（导航）**：快速访问卖家论坛、账户专属建议、亚马逊新闻和待办任务列表。
