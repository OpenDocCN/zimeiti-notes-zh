# 抖音运营新手起号：P23：抖音沙雕动画玩法 🎬

在本节课中，我们将学习一种被称为“沙雕动画”的视频制作方法。这种方法操作简单，无需复杂技术或原创剧本，仅用一部手机即可完成，非常适合新手入门。我们将从软件操作到内容创作，一步步拆解其制作流程。

## 制作工具与基础设置 🛠️

上一节我们介绍了沙雕动画的概念，本节中我们来看看具体的制作工具和初始设置。

![](img/3a76da38a23fb8fb910cd3a28245ce15_1.png)

制作沙雕动画，你只需要一部手机和一个特定软件。在软件中，首先需要进行基础设置以确保视频质量。

以下是设置视频参数的具体步骤：
*   打开制作软件。
*   选择合适的视频比例。例如，抖音平台常用 **9:16**，西瓜视频常用 **16:9**。
*   将分辨率设置为 **1080P**，这能使视频更清晰。
*   帧率保持默认设置即可。



![](img/3a76da38a23fb8fb910cd3a28245ce15_1.png)


![](img/3a76da38a23fb8fb910cd3a28245ce15_3.png)

## 选择场景与角色 🏞️👤

完成基础设置后，接下来需要为动画搭建舞台和选择演员，即场景与角色。

![](img/3a76da38a23fb8fb910cd3a28245ce15_5.png)

软件的场景库通常按主题分类，方便你快速找到所需背景。

以下是选择场景与角色的方法：
*   在“场景”板块中，选择一个主题，例如“古代”。
*   向右滑动，可以浏览该主题下的所有具体场景。
*   “角色”的选择方式类似，分类中有大量模型可供选择，包括沙雕风格、武侠风格、动漫角色等，非常丰富。

## 动画制作与合成 🎞️

当场景和角色都准备好后，我们就可以开始让角色动起来，并合成完整的动画了。

其操作逻辑与手机剪映类似，非常直观易懂。

以下是制作动画的基本流程：
1.  点击已添加的角色。
2.  选择“添加动作”。
3.  点击“播放”预览，人物就会执行动作。
4.  如果不喜欢当前动作，可以随时更换。
5.  可以继续添加更多角色来丰富剧情。
6.  最后，导入背景音乐或配音音频，即可合成一段完整的动画视频。



## 高效配音与内容创作策略 🎤

动画制作本身并不复杂，但内容是吸引流量的关键。本节我们来看看如何高效解决配音和剧本问题。

![](img/3a76da38a23fb8fb910cd3a28245ce15_9.png)

这里介绍一个“懒人方法”——拍同款。你无需自己编写搞笑段子和配音，也能创作出有流量的作品。

以下是采用“拍同款”方法的具体好处：
*   **原创度高**：使用他人音频但搭配自己制作的动画画面，作品原创度高，不易被平台限制推荐，有利于获得收益。
*   **蹭取流量**：你可以借助原声发布者（通常是大博主）的流量。许多用户喜欢点击原声，观看其他人的创意视频，这能为你带来额外曝光。
*   **提升效率**：省去了构思剧本和录制配音的时间，让你能更专注于动画制作。效率提升后，甚至可以同时运营多个账号，获取更多流量和收益。
