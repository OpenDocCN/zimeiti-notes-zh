# AI短视频创作：P3：小说推文项目实操教程 🚀

在本节课中，我们将学习一种无需实拍、无需撰写原创文案的抖音短视频创作方法——小说推文项目。这是一种通过推广小说内容来获取收益的玩法，操作简单，适合新手入门。我们将从项目原理、操作步骤到收益计算，进行系统性的讲解。

## 项目原理与优势 💡

上一节我们介绍了AI生成视频的多种形式，本节中我们来看看一个具体的落地项目：小说推文。

这种视频无需真人实拍出镜，也无需创作原创文案。文案直接复制自小说中引人入胜的段落。同时，它也不需要投放“抖加”进行推广，操作难度非常低。可以说，这是最适合新手尝试的项目之一。

那么，这种视频如何变现？难道仅靠“中视频计划”就能一条视频收入两三千元吗？答案是否定的。

除了参与“中视频计划”获取播放量收益外，这类视频最关键的一点是带有**CPA（按行动付费）**属性。虽然视频画面可能是解压内容，但文案一直在朗读某部小说的片段。只要有人通过该视频去搜索小说的完整版，那么视频作者每为小说平台成功推荐一位读者，就能获得约十元左右的佣金。这项收益不亚于“中视频计划”。

如果你查看这类视频的评论区，通常会发现作者置顶了一条引导用户寻找小说完整版的评论。玄机就在这条置顶评论里。他让你搜索的“关键词”，可能并非小说原名，而是平台与作者约定的特定**推广关键词**。用户通过此关键词在平台上找到小说并成为用户后，即算作者成功为平台推荐了一名用户。这个关键词就是作者与平台之间的约定。

目前平台上已有很多人在操作此项目，并且成绩斐然。这就是**小说推文项目**的核心原理：通过在抖音发布作品，为小说平台推荐用户，从而获取佣金。我们圈子内就有人操作此项目，达到了月入数万元的水平。

## 项目操作步骤详解 🛠️

了解了原理和优势后，接下来我们看看具体如何操作。要操作这个项目，主要分为两大步骤。

### 第一步：获取平台授权

首先，你需要成为“推荐官”，获得小说平台的授权。这意味着你需要拿到每部小说对应的推广关键词。

关于如何获取授权，考虑到平台规则，无法在视频中直接说明。我将在教程结尾部分告诉大家具体的操作方法。

### 第二步：制作视频内容

![](img/a8a1bece62e9d9190fd1607f749e3ca1_1.png)

获得授权后，最关键的一步就是视频制作。这个过程其实非常简单。

**1. 准备文案与配音**
文案无需自己撰写，直接从小说中选取比较经典、能引发好奇心的段落。复制大约1000字左右的内容。
将这段文案粘贴到**AI配音工具**中。
选择一个听起来舒适的主播声音，然后直接导出配音音频文件。

**代码示例：获取音频文件**
```python
# 伪代码：示意将文本提交给AI配音API并导出文件
audio_file = text_to_speech(小说文案, voice=‘选定主播音色’)
export_audio(audio_file, ‘output.mp3’)
```

至此，我们便拥有了视频的配音文件。

**2. 准备与处理视频素材**
接下来，我们需要寻找一些解压类视频素材，就像课程开头展示的例子一样。
**请注意**：解压视频不要只使用一个，而是选取多个进行重新组合剪辑。
将解压视频的原声调至最小或关闭。

**3. 合成最终视频**
最后，使用剪映等剪辑软件，将处理好的解压视频素材与生成的配音音频进行合成，然后导出成品视频。

**公式：视频制作流程**
`最终视频 = 处理后的解压视频素材 + AI生成的配音音频`

这样的视频生产速度极快，基本上十分钟以内就能完成一个。

## 收益分析与总结 📈

在掌握了操作方法后，我们来算一笔账，看看这个项目的收益潜力。

根据我的了解，小说推文项目在抖音上大约每1万次播放，能产生2到5个订单。每单收益大约在十元左右。

![](img/a8a1bece62e9d9190fd1607f749e3ca1_3.png)

假设你在抖音上运营十个账号，每个账号每天发布三条视频。即使每条视频只获得抖音基础的500次播放，那么每日总播放量也达到了：
`10个号 * 3条/号 * 500播放/条 = 15,000播放`

这1.5万播放量足以产生5单以上的收益。事实上，我们圈内很多小伙伴一人就操作二三十个账号，这毫不夸张。据观察，基本上每发布100条作品，其中肯定会产生一条爆款视频。一旦出现爆款，播放量将非常可观，仅这一条视频就有可能带来数千元的收益。
