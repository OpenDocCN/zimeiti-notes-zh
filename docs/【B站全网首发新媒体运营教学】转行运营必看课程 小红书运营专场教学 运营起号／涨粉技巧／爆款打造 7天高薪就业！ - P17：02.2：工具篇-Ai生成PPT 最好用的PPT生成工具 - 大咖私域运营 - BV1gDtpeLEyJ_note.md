# 新媒体运营课程：P17：02.2：工具篇-AI生成PPT 🚀

在本节课中，我们将学习如何使用一款高效的AI工具来自动生成PPT。这款工具能根据你提供的主题或现有文档，自动梳理大纲并生成详细的演示文稿内容，极大地提升工作效率。

上一节我们介绍了新媒体运营的基础工具，本节中我们来看看如何利用AI技术快速制作专业PPT。

## 工具核心功能

这款AI生成PPT工具主要提供两种创作模式：

1.  **主题生成模式**：你只需输入一个核心主题，工具会结合类似ChatGPT的生成式AI能力，自动创作出完整的PPT全文。
2.  **文档转换模式**：你可以上传已有的文档（如TXT、PDF文件），AI助手会自动总结文档大纲，并基于此大纲生成详细的PPT内容。

我测试过许多AI生成PPT的工具，这款在基于大纲生成详细内容方面表现尤为出色。

## 实战演示：从文档到PPT

![](img/2cce0e707031e2f270175741cbf7be53_1.png)

以下是使用“文档转换模式”的具体操作步骤。

![](img/2cce0e707031e2f270175741cbf7be53_3.png)

首先，我准备了一篇关于“新媒体运营流量与创作标签”的文章作为示例素材。

![](img/2cce0e707031e2f270175741cbf7be53_1.png)

我们将基于这篇文章来测试工具的PPT生成能力。

1.  **上传文档**：在工具界面点击“继续”，上传准备好的文件。工具会开始解读文档内容。

![](img/2cce0e707031e2f270175741cbf7be53_5.png)

![](img/2cce0e707031e2f270175741cbf7be53_3.png)

![](img/2cce0e707031e2f270175741cbf7be53_7.png)

2.  **生成大纲**：文档解析完成后，工具会在下方自动列出分析得到的内容大纲。例如，我的文章被梳理为以下几个部分：
    *   新媒体运营的本质
    *   平台运营的角度
    *   账号运营的策略
    *   用户行为的分析与应用
    *   创作激励与流量分配
    *   数据分析在运营中的作用
    *   运营案例的解析
    可以看到，AI分析得非常细致。

3.  **开始制作**：大纲确认后，点击“开始制作”，工具将进入PPT生成页面。

![](img/2cce0e707031e2f270175741cbf7be53_5.png)

4.  **等待生成**：系统会基于对文档的理解和大纲结构，自动生成每一页PPT的详细内容。生成速度通常很快，但可能受网络环境和使用频率影响。

![](img/2cce0e707031e2f270175741cbf7be53_7.png)

## 编辑与优化

![](img/2cce0e707031e2f270175741cbf7be53_13.png)

PPT内容生成完毕后，我们可以进行进一步的调整和美化。

1.  **查看与编辑**：生成完成后，点击“查看和编辑”进入PPT编辑界面。如果生成过程中出现卡顿或显示问题，可以尝试返回后重新进入。

![](img/2cce0e707031e2f270175741cbf7be53_11.png)

2.  **更换模板**：工具右侧提供“更换模板”功能。虽然当前内置的模板数量不算非常多，但切换速度非常快。你可以选择一个与内容风格匹配的模板。

![](img/2cce0e707031e2f270175741cbf7be53_13.png)

3.  **调整内容**：在编辑界面，你可以自由调整任何元素。右侧工具栏提供了文字、形状等元素的编辑和替换选项。

![](img/2cce0e707031e2f270175741cbf7be53_15.png)

![](img/2cce0e707031e2f270175741cbf7be53_17.png)

4.  **本地化精修**：最关键的优势在于，工具能基于每个小节的大纲节点生成对应的详细内容。完成初步的模板选择和内容生成后，你可以将PPT下载到本地，再用PowerPoint或Keynote进行深度调整和美化。

![](img/2cce0e707031e2f270175741cbf7be53_15.png)
