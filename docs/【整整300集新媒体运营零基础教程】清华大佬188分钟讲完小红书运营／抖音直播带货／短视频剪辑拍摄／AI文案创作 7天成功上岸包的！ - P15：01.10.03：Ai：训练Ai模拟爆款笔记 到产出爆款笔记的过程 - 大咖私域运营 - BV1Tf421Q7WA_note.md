# AI新媒体运营教程：P15：01.10.03：训练AI从分析到产出爆款笔记 📝

## 概述
在本节课中，我们将学习如何训练AI工具（如ChatGPT）分析爆款笔记，并基于分析结果生成具备爆款潜质的新内容。我们将重点讲解分析维度、指令技巧以及从大纲到成文的完整流程。

上一节我们介绍了通过“喂资料”的方式让AI理解特定平台的文案风格。虽然上一节以旅游攻略为例的成果可能不理想，但方法是正确的。关键在于为AI提供你所在赛道的优质内容，让它分析并学习其中的写作模式和风格。

本节中，我们来看看如何深化这一过程，即让AI分析具体的爆款内容，并学习其成功要素，最终用于产出新的爆款笔记。

![](img/c3d9384f996c11704160f2c5193a4e20_5.png)

## 第一步：明确分析维度 🎯
在向AI提供爆款笔记前，需要明确我们希望AI分析哪些关键点。以下是核心的分析维度：

![](img/c3d9384f996c11704160f2c5193a4e20_7.png)

![](img/c3d9384f996c11704160f2c5193a4e20_8.png)

![](img/c3d9384f996c11704160f2c5193a4e20_10.png)

以下是需要AI重点分析的几个方面：
1.  **爆款因素**：分析内容为何能成为爆款，例如是否击中用户痛点、引发情感共鸣等。
2.  **内容结构**：拆解文案的框架。
    *   **开篇**：用什么方式（如提问、制造焦虑、讲述故事）吸引用户继续阅读。
    *   **中间**：提供了哪些具体内容来满足用户需求、解决痛点，以提升用户停留时长。
    *   **结尾**：设置了什么“钩子”（如鼓励互动、引导私信、提供福利）来促进用户行动。
3.  **关键词策略**：分析笔记中是否刻意植入了特定关键词。
    *   如果有，关键词是什么？
    *   关键词的出现频率和分布方式如何？

## 第二步：收集与提供爆款样本 📄
我们需要找到目标领域（例如“考研”）的真实爆款笔记作为分析样本。

![](img/c3d9384f996c11704160f2c5193a4e20_12.png)

![](img/c3d9384f996c11704160f2c5193a4e20_14.png)

操作步骤如下：
1.  在小红书等平台搜索目标关键词（如“考研”），筛选点赞量较高的笔记。
2.  复制笔记的完整文案内容。
3.  在AI工具中开启新对话，输入清晰的指令。

![](img/c3d9384f996c11704160f2c5193a4e20_16.png)

**初始指令示例**：
```
你是一位小红书爆款文案分析师。我将为你提供几篇小红书平台上关于“考研”的爆款笔记文案。
请你完成以下分析：
1. 分析每篇笔记的爆款因素。
2. 分析每篇笔记的整体结构：开篇如何吸引用户，中间如何戳中用户痛点，结尾如何引导用户互动（如私信、关注）。
3. 分析每篇笔记是否运用了中文分词法刻意设计关键词。如果有，请指出关键词及其出现频率。
```
随后，将复制好的文案逐一提供给AI进行分析。

## 第三步：整合分析结果与优化指令 🔄
AI对每篇笔记进行分析后，其反馈可能分散且标准不一。我们需要引导它对所有分析结果进行整合与去重。

![](img/c3d9384f996c11704160f2c5193a4e20_18.png)

**整合指令示例**：
```
请将你刚才分析的4篇关于“考研”的爆款笔记的所有爆款因素总结归纳，形成一个统一的、去重后的爆款因素列表。
```

![](img/c3d9384f996c11704160f2c5193a4e20_20.png)

![](img/c3d9384f996c11704160f2c5193a4e20_22.png)

有时AI可能会“忘记”之前的部分指令或分析内容，导致总结不全。此时，需要耐心地通过对话引导它“回忆”或重新明确要求，这是使用此类AI工具时的常见操作。

## 第四步：基于爆款因素生成新内容大纲 📝
获得整合后的爆款因素列表后，即可指导AI根据这些因素创作新笔记。首先，从生成大纲开始。

**生成大纲指令示例**：
```
现在，我需要你撰写一篇关于“考研”的小红书笔记。
请结合你刚才总结的爆款因素，为我梳理一个具备爆款潜质的笔记大纲。
要求：大纲需要细分到具体的小节点。待我确认大纲无误后，再请你生成完整文案。
```
AI会根据指令生成一个包含开篇、核心内容模块、结尾引导等部分的结构化大纲。

## 第五步：根据大纲撰写完整文案 ✍️
确认大纲后，即可发出最终指令，让AI产出完整文案。指令需具体，包含对风格、元素和关键词的明确要求。

**撰写文案指令示例**：
```
请按照你提供的大纲，撰写一篇完整的小红书文案。
要求：
1. 文案结构需严格遵循爆款笔记的因素和结构。
2. 开篇需用一个简单的“逆袭”小故事引入，引发读者共鸣。
3. 文中需适当使用Emoji表情符号。
4. 需运用中文分词法的思路，自然融入“考研”、“备考规划”、“政治冲刺”等关键词。
```
AI将生成一篇融合了情感故事、实用建议、明确关键词和互动引导的笔记初稿。这份初稿对于一般运营需求已具备可用性，后续可根据个人审美和深度理解进行优化调整。

## 总结
本节课我们一起学习了利用AI辅助内容创作的核心流程：
1.  **明确分析维度**：从爆款因素、内容结构、关键词策略三个角度拆解内容。
2.  **提供样本与指令**：收集真实爆款内容，通过清晰指令引导AI分析。
3.  **整合与总结**：将AI的分散分析整合成统一的爆款要素清单。
4.  **从大纲到成文**：基于爆款要素生成内容大纲，并据此撰写融合了特定要求（如故事、表情、关键词）的完整文案。
