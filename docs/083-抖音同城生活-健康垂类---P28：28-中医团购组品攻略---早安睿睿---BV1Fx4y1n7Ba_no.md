# 抖音同城生活-健康垂类0到1运营：28：中医团购组品攻略 🧑‍⚕️

![](img/313cdb3ca337bfc8cf375b51df7fc4e8_0.png)

在本节课程中，我们将学习如何在抖音本地生活平台上，为中医类目设计并上架有效的团购产品组合。我们将探讨如何利用现有规则，通过巧妙的品类组合与上架策略，实现用户引流与转化。

上一节我们介绍了平台的基本规则，本节中我们来看看针对中医这一特殊垂类的具体组品方法。

中医类目在抖音本地生活平台中较为特殊，它并未被直接纳入“消费医疗”板块。因此，我们需要采用特定的策略来操作。目前在杭州地区，已有成熟的玩法。核心策略是通过“推拿”这个类目进行切入。因为推拿项目在报备白名单后即可正常上架团购。我们利用推拿团购吸引用户付费到店，随后在店内进行服务转化，例如推荐办理会员卡、开具药方或进行针灸等治疗。

关于如何具体实现“强行上架”，前面课程已有详细说明。这里简要回顾关键步骤：你需要在现有中医馆地址的附近、旁边或楼上，分割出一块独立的地址。然后以此地址去认领一个包含“推拿”服务、但不包含“诊疗”或“医疗”资质的普通推拿馆营业执照。接着，使用该营业执照在“抖音来客”平台正常认领门店并提交团购，即可通过审核。这是实现中医服务上架的核心干货。

接下来，我们详细拆解三种有效的产品组合策略。

以下是第一种核心策略：**推拿引流组合**。
*   **核心操作**：申请仅含“推拿”服务的营业执照，以此资质上架推拿团购。
*   **转化路径**：用户购买推拿团购到店后，在服务过程中转化为更高价值的诊疗服务（如开方、针灸）。

除了推拿，艾灸也是一个可行的切入点。抖音本地生活平台目前不允许上架如拔罐、针灸这类涉及破皮的项目，但艾灸是被允许的。因此，艾灸可以作为一个独立且合规的品项进行上架。

以下是第二种产品组合策略：**推拿与艾灸组合**。
*   **组合目的**：推拿服务人力成本较高，而艾灸的边际成本相对较低。将两者组合打包，可以提升整体客单价与利润空间。
*   **操作方式**：设计包含“推拿+艾灸”的套餐，或分别设置引流品与利润品。

第三种策略是针对特定需求用户进行精准引流。平台目前无法直接上架“正骨”项目，但我们可以从具体部位入手。

以下是第三种引流策略：**专项部位理疗券**。
*   **操作方法**：上架如“肩颈调理”、“膝关节护理”等特定部位的理疗或体验券。这些项目不直接宣称“正骨”，但能吸引有相应部位不适的目标用户到店。
*   **转化逻辑**：这些用户到店后，其真实需求往往是治疗而不仅是放松。此时，向其推荐正骨、针灸等治疗服务的转化效果会非常好。

本节课中我们一起学习了中医类目在抖音本地生活的三种核心组品攻略：通过推拿资质合规上架并店内转化；利用艾灸与推拿组合提升利润；以及设计专项部位理疗券精准吸引治疗需求用户。掌握这些方法，可以帮助你在平台规则内，有效开展中医服务的线上运营。
