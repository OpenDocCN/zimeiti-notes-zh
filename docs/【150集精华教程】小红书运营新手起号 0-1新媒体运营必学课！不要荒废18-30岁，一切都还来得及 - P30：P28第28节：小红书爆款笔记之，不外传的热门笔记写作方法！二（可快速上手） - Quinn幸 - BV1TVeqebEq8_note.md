## 一、确立笔记内容的五个步骤

很多人在写小红书内容时，首先不知道如何确立主题和方向。以下是确立内容的五个核心步骤。

### 1. 确定内容领域
首先，你需要明确自己账号运营的领域。例如，是美妆、护肤、母婴还是其他类别。
如果无法确定领域，会导致账号垂直度降低。账号垂直度降低会引发一系列问题，例如官方减少推荐、粉丝流失。这会形成一个恶性循环：粉丝越来越不喜欢你的内容，你的内容就越来越不会被官方推荐。

### 2. 梳理行业关键词
确定领域后，需要梳理与该领域相关的关键词。
在之前的课程中，已经教大家使用百度工具、爱站网、5118、百度指数等关键词工具进行查询。本节课不再赘述。根据你的行业，可以梳理出许多关键词及其热度排行。用好这些关键词排行，能极大提升内容效果。

### 4. 筛选与整理内容
搜集好同行案例后，就可以对这些案例的内容进行筛选。筛选完成后，将其整理成自己的内容，然后进行发布。
这里我们用“千瓜”来演示一下。在千瓜左侧栏目中，可以看到许多功能，包括热词排行榜、热词搜索榜、笔记收录查询等。
今天主要讲解“笔记收录查询”功能。例如，发布笔记后，你可能担心笔记是否被官方收录，就可以利用千瓜的这个功能进行查询。
热门内容分析和热词排行榜等功能在此不展开，主要介绍“达人搜索”。
达人搜索可以按昵称搜索，或按笔记内容搜索。例如搜索“美妆”（此功能需付费），我们按昵称搜索，效果类似。
按昵称搜索会显示相关达人的属性，例如有做彩妆、护肤、母婴、美式饮品等。还包括服饰穿搭等分类属性。大家可以根据自己的数据和需求进行筛选。

## 二、其他注意事项
以下是撰写笔记标题时需要注意的其他事项。

1.  **控制标题字数**：标题字数尽量控制在 **12~18个字** 之内。
2.  **使用符号美化**：标题中要多带一些符号（如❗️🔥💯）进行美化。因为小红书女性用户居多，使用美化符号能很好地吸引她们的注意力。
3.  **加入具体数据**：可以在标题中加入一些具体数据，以增加真实感。例如上一节课截图中的“百万”这样的数字，或其它详细数字。这样看起来会更可信。因此，在拟定标题文字时，可以加入数字。
