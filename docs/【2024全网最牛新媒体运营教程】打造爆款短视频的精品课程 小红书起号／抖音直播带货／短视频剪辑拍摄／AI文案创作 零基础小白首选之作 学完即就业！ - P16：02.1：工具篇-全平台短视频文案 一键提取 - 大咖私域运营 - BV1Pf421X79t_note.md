# 新媒体运营教程：P16：02.1：工具篇-全平台短视频文案一键提取 📝

在本节课中，我们将学习一个非常实用的运营工具，它能帮助我们从各大短视频平台一键提取视频文案。掌握这个工具，可以极大提升内容分析和创作的效率。

## 工具介绍与背景

上一节我们介绍了文案创作的基础，本节中我们来看看如何高效获取参考文案。在运营工作中，我们常会遇到优秀的短视频，希望将其文案内容提取出来进行分析或二次创作。

传统方法存在一些不便：
*   使用“轻抖”等小程序提取，但通常仅限于特定平台。
*   先将视频下载到本地，再导入“剪映”等软件中提取字幕。这种方法步骤繁琐，且跨平台操作不便。

面对抖音、小红书、视频号、哔哩哔哩、快手等多个平台，需要一个更通用的解决方案。

## 实战演示：提取抖音文案

让我们通过一个实际例子来演示如何使用。首先，在抖音找到一个想要提取文案的视频，例如一段“情感语录”视频。

操作步骤如下：
1.  在抖音APP中复制该短视频的分享链接。
2.  打开文案提取工具的网页界面。
3.  将复制的链接粘贴到工具的输入框中。
4.  点击“提取文案”按钮。

稍等片刻，工具就会完成处理。提取出的文案会以纯文本形式展示，例如：
> “听完你就释怀了。村上春树说，我动了离开你的念头，不是因为你不好，也不是因为不爱了……”

此时，我们可以一键复制全部文案，或使用“格式化”功能，让文案按句段排列得更整齐，便于后续使用。

## 实战演示：提取哔哩哔哩文案

该工具同样适用于哔哩哔哩（B站）平台。我们换一个B站视频链接进行测试。

操作步骤与上述一致：
1.  复制B站视频的链接（通常以 `BV` 号开头）。
2.  将链接粘贴到工具输入框。
3.  点击“提取文案”。

提取过程同样迅速，并能准确输出视频的文案内容，例如：
> “新手做直播电商，怎么才能打造出直播爆品？在直播销售的不同阶段……”

这证明了该工具跨平台提取的准确性和高效性。
