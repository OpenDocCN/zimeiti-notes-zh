# 视频号运营：P33：简单的书单原创视频制作教程

## 概述
在本节课中，我们将学习如何使用剪映工具，一步步制作一个简单的书单原创视频。整个过程将涵盖素材选择、文案添加、字体样式调整、画中画应用、音频添加等核心步骤，适合零基础的新手学习。

---

### 素材导入与基础设置
上一节我们了解了书单视频的概念，本节中我们来看看如何开始制作。首先，打开剪映APP，点击“开始创作”按钮。

进入创作界面后，您会看到“素材库”选项。点击进入素材库。

在素材库顶部的搜索框（放大镜图标）内，输入关键词“书单”并进行搜索。

搜索结果会显示大量书单背景素材。您可以根据喜好选择。例如，选中一个喜欢的素材后，点击右上角的复选框，然后点击右下角的“添加”按钮，将其导入到时间线。

素材添加后，第一步是关闭视频原声。点击时间线上素材左侧的“关闭原声”图标即可。

### 添加与编辑文案
接下来，我们需要为书单添加文字内容。在底部主工具栏中找到并点击“文本”选项。

在文本工具中，选择“新建文本”。

将您事先准备好的文案粘贴到文本输入框中。您可以使用 `Ctrl+V`（Windows）或 `Cmd+V`（Mac）进行粘贴。

以下是文本样式调整的步骤：

1.  **调整字体**：点击“字体”选项，从列表中选择喜欢的字体。例如，选择“书法”分类下的“位杯体”。
2.  **调整颜色**：点击“样式”选项，在“颜色”中选择您喜欢的文字颜色，例如黄色或白色。
3.  **调整排版**：仍在“样式”选项中，找到“排列”。向下滑动，找到“行间距”选项。拖动滑块，将数值调整为 **4**。

完成上述设置后，点击右下角的对勾（✓）确认。

在预览屏幕上，用双指按住文本层，可以旋转角度并拖动到合适的位置。

最后，在时间线上选中这段文本，拖动其末尾的白色边框，使其长度与主视频素材的时长对齐。

### 添加视频标题
现在，我们来为视频添加一个标题。再次返回主工具栏，点击“文本” -> “新建文本”。

输入您的标题文字。

参照编辑正文文案的方法，通过“样式”选项调整标题的颜色和大小。例如，将颜色设置为黄色。

在预览屏幕上，双指按住标题可以放大，并拖动到合适的位置。

点击对勾（✓）确认后，同样在时间线上拖动标题片段的白色边框，使其与主视频时长对齐。

### 使用画中画增添内容
为了使视频更丰富，我们可以添加一个画中画视频。返回主工具栏，点击“画中画”选项，然后选择“新增画中画”。

从相册中选择一段您准备好的视频素材，点击“添加”。

素材导入后，在时间线上选中它，在底部二级工具栏中向后滑动，找到并点击“蒙版”工具。

选择“圆形”蒙版。屏幕上会出现一个圆形蒙版区域。

1.  用双指在屏幕上捏合，可以放大或缩小圆形蒙版。
2.  拖动蒙版中心，将其对准您想突出显示的人物或物体。
3.  拖动蒙版参数中下方的羽化箭头，适当增加羽化值，使边缘过渡更自然。

设置完成后，点击对勾（✓）。

接着，在预览屏幕上，双指按住画中画视频，可以缩放其大小并调整角度，然后将其拖动到左上角等合适位置。

如果画中画素材带有声音，需要将其静音。选中该素材，在二级工具栏中找到“音量”，将滑块拖到最左侧（数值为 **0**），然后点击对勾（✓）。

最后，将时间线移动到主视频的结尾处，选中画中画素材，点击“分割”，然后删除后半部分多余的内容，使其与主视频同时结束。

### 为文案添加动画
返回主工具栏，在时间线上选中您之前添加的正文文案片段。

在底部二级工具栏中向后滑动，找到“动画”选项并点击。

我们为文字添加一个入场动画。在“入场动画”列表中，选择“打字机”效果。

将动画时长滑块拖动到最右侧，以设置较慢的动画速度，然后点击对勾（✓）确认。

### 添加背景音乐
视频制作的最后一步是添加背景音乐。返回主工具栏，点击“音频” -> “音乐”。

在音乐库的搜索框中，输入您想要的歌曲名，例如“老男孩”，然后点击搜索。

从搜索结果中选择一个版本，点击右侧的“使用”按钮。

音乐添加后，在时间线上按住音频片段，向前拖动，使其开头与视频开头对齐。

将时间线移动到视频的结尾处，选中音频片段，点击“分割”，然后删除视频结尾之后多余的音频部分。

### 总结
本节课中，我们一起学习了制作一个简单书单原创视频的全流程。从**导入书单背景素材**开始，到**添加并格式化文案与标题**，再到运用**画中画**和**蒙版**功能丰富画面，接着为文字添加**入场动画**，最后**匹配并裁剪背景音乐**。掌握这些基础操作，您就能快速制作出属于自己的书单视频。
