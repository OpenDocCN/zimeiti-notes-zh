## 店铺类型概述

小红书平台提供了多种店铺类型，以满足不同经营主体的需求。从个人卖家到品牌企业，都能找到对应的开店方式。了解整个体系，有助于我们在不同阶段做出合适的操作选择。

小红书的店铺体系主要分为以下几种：**个体店**、**个体工商户店**、**普通企业店**、**专卖店**、**旗舰店**以及**卖场型旗舰店**。其中，卖场型旗舰店本质上是品牌经销商店铺。

## 各类店铺详解

上一节我们介绍了店铺的整体分类，本节中我们来详细看看每一种店铺的具体情况。

### 1. 个体店

个体店类似于线下的“摆地摊”，是个人用户开设的店铺。适合刚毕业或想尝试小规模创业的个人。

*   **申请主体**：个人用户。
*   **经营特点**：经营限定类目下的零星、小额商品，例如首饰、服装、包包、手机壳等。可以销售多个品牌，类似于“杂货铺”。
*   **入驻要求**：非常简单，仅需个人身份证。
*   **店铺命名格式**：系统自动生成，格式为“专业号昵称 + 店名”。
*   **保证金**：涉及内部保证金和浮动保证金，二者取高值。个人店的起始保证金通常为 **1000元**。

### 2. 个体工商户店

![](img/dbb0ab9f4c5aee2af9b98eaf21b8f5c9_7.png)

个体工商户店与个体店的主要区别在于入驻要求，它需要经营者拥有营业执照。

![](img/dbb0ab9f4c5aee2af9b98eaf21b8f5c9_9.png)

*   **与个体店的区别**：主要在于入驻时需要提供**营业执照**。对品牌授权没有要求。
*   **其他方面**：如经营特点、保证金规则等，与个体店基本一致。

### 3. 普通企业店

普通企业店面向拥有正规公司的经营主体。其与后续店铺类型的核心区别都体现在入驻要求上。

![](img/dbb0ab9f4c5aee2af9b98eaf21b8f5c9_11.png)

![](img/dbb0ab9f4c5aee2af9b98eaf21b8f5c9_13.png)

*   **入驻要求**：申请主体必须为企业，需要持有**企业营业执照**。
*   **权重说明**：通常，入驻要求越简单的店铺类型，平台赋予的初始权重可能越低；要求越复杂，权重可能越高。但对于新手，建议从简单的个人店开始测试，无需初期投入过多。

### 4. 专卖店、旗舰店与卖场型旗舰店

这三种店铺类型主要面向品牌方或拥有品牌授权的经营者。

*   **专卖店**：
    *   **入驻要求**：需要**营业执照**和**品牌资质**（自有品牌或非自有品牌均可）。核心是销售特定品牌的产品。
*   **旗舰店**：
    *   **入驻要求**：必须持有**营业执照**和**品牌授权书**（销售他人品牌时需提供）。
*   **卖场型旗舰店**：
    *   **经营特点**：拥有零售资质，并可经营多个品牌的旗舰店。但此类店铺在小红书平台较少见，因为小红书以女性用户为主，流量特性与大型卖场店不完全匹配。这反而为中小卖家提供了更友好的竞争环境。

## 保证金规则详解

![](img/dbb0ab9f4c5aee2af9b98eaf21b8f5c9_15.png)

了解店铺类型后，我们需要重点关注开店涉及的保证金规则，这直接关系到资金提现。

保证金分为**内部保证金**和**浮动保证金**，平台会取两者中的较高值作为应缴金额。

![](img/dbb0ab9f4c5aee2af9b98eaf21b8f5c9_17.png)

![](img/dbb0ab9f4c5aee2af9b98eaf21b8f5c9_19.png)

### 内部保证金

![](img/dbb0ab9f4c5aee2af9b98eaf21b8f5c9_21.png)

内部保证金主要与店铺类型和经营类目相关，金额相对固定。以下是普通企业店部分特殊类目的保证金示例（仅供参考，具体以平台最新规则为准）：

![](img/dbb0ab9f4c5aee2af9b98eaf21b8f5c9_23.png)

```
普通企业店 - 内部保证金示例（部分类目）：
*   美妆护肤 -> 20,000元
*   珠宝首饰 -> 50,000元
*   ...（其他类目）
```

![](img/dbb0ab9f4c5aee2af9b98eaf21b8f5c9_25.png)

由于类目较多且规则可能更新，建议在平台官方资料库查询最新标准。

### 浮动保证金（重点）

浮动保证金是根据店铺的成交金额动态计算的，这是我们需要理解的核心。

其计算逻辑如下：
1.  所有店铺都需先缴纳**1000元**的基础保证金。
2.  平台会根据店铺过去30天的成交金额（GMV）区间，要求对应额度的浮动保证金。
3.  **只有当保证金账户总金额（基础+浮动）满足要求时，才能进行提现。**

浮动保证金阶梯示例如下：

```
月成交金额 (X)        浮动保证金
X ≤ 50,000元          0元
50,000元 < X ≤ 100,000元  3,000元
100,000元 < X ≤ 500,000元 5,000元
X > 500,000元          20,000元
```

**重要提示**：例如，你只缴纳了1000元基础保证金，当某月成交额达到6万元时，系统会要求保证金总额达到4000元（1000基础+3000浮动）。若账户不足，则该月所有货款均无法提现。务必定期检查保证金是否充足。
