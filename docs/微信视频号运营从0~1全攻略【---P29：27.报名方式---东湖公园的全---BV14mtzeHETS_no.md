# 微信视频号运营从0~1全攻略：P29：27.报名方式

在本节课中，我们将要学习如何通过官方渠道报名参加张老师的视频号运营课程。报名流程涉及识别官方联系人、完成支付以及确认名额等关键步骤。

## 官方报名渠道说明

上一节我们介绍了课程的价值，本节中我们来看看如何获取课程名额。获取名额的唯一方式是联系指定的官方报名老师。

![](img/17c5a0d8bfee9053fda50c3d761319b5_1.png)

**核心概念：官方报名老师 = 陈丽老师**

![](img/17c5a0d8bfee9053fda50c3d761319b5_2.png)

这是唯一的报名通道。只有通过陈丽老师，才能成功加入张老师的课程群并获得相关资源。

![](img/17c5a0d8bfee9053fda50c3d761319b5_3.png)

## 报名与支付流程

![](img/17c5a0d8bfee9053fda50c3d761319b5_5.png)

![](img/17c5a0d8bfee9053fda50c3d761319b5_6.png)

成功添加官方老师后，需要立即完成报名和支付操作。以下是具体步骤：

![](img/17c5a0d8bfee9053fda50c3d761319b5_7.png)

![](img/17c5a0d8bfee9053fda50c3d761319b5_8.png)

**第一步：联系官方老师并说明支付方式**
添加陈丽老师的企业微信后，需第一时间告知你计划使用的支付方式。微信转账是速度最快的方式。

![](img/17c5a0d8bfee9053fda50c3d761319b5_9.png)

![](img/17c5a0d8bfee9053fda50c3d761319b5_10.png)

**第二步：完成课程费用支付**
课程费用为 **2880** 元。支付方式支持微信转账或信用卡支付。这是官方的缴费渠道。

![](img/17c5a0d8bfee9053fda50c3d761319b5_11.png)

![](img/17c5a0d8bfee9053fda50c3d761319b5_12.png)

**第三步：提交支付凭证**
支付完成后，**必须**将支付成功的截图发送给官方报名老师（陈丽老师）进行确认。名额的确认以收到支付截图为准。

**第四步：确认名额并等待开课**
向陈丽老师反馈“张老师，我已报名”以确认名额。名额确认后，今天完成资源对接，明天课程即可正式开始。

## 如何添加官方报名老师

不会添加联系人的同学，可以按照以下步骤操作。此操作教学视频也已录制，可供反复观看。

![](img/17c5a0d8bfee9053fda50c3d761319b5_14.png)

![](img/17c5a0d8bfee9053fda50c3d761319b5_15.png)

以下是添加官方报名老师陈丽老师的具体步骤：

![](img/17c5a0d8bfee9053fda50c3d761319b5_16.png)

1.  **点开**屏幕上显示的**二维码图片**。
2.  **长按**二维码。
3.  在弹出的菜单中，点击“**打开对方的企业微信**”。
4.  进入企业微信名片页后，点击“**添加到通讯录**”。
5.  点击“**发送**”好友申请。
6.  **等待**报名老师通过你的好友申请。

![](img/17c5a0d8bfee9053fda50c3d761319b5_17.png)

> **提示**：请务必认准官方报名老师——陈丽老师，这是加入张老师课程群的唯一途径。

## 核心步骤总结

本节课中我们一起学习了完整的课程报名流程。最后，我们来总结一下成功报名的几个核心步骤：

1.  添加唯一官方报名老师 **陈丽老师** 的企业微信。
2.  向老师说明支付方式，并完成 **2880元** 的费用支付。
3.  将 **支付成功截图** 发送给陈丽老师以锁定名额。
4.  向老师确认报名，随后等待资源对接与课程开启。

![](img/17c5a0d8bfee9053fda50c3d761319b5_19.png)

请严格按照此流程操作，以确保顺利报名。
