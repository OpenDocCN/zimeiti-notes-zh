## 登录商家后台

![](img/110b01671d7d0a67a8f60619ee0dcd09_11.png)

店铺申请开通成功后，需要使用专门的商家后台进行管理。这个后台叫做“小红书千帆”，是商家管理店铺的工具。

![](img/110b01671d7d0a67a8f60619ee0dcd09_13.png)

以下是登录步骤：
*   **苹果手机**：在App Store搜索“小红书千帆”并下载。
*   **安卓手机**：如果搜索不到“千帆”，可以尝试搜索“小红书商家版本”，目前两者功能一致。

![](img/110b01671d7d0a67a8f60619ee0dcd09_15.png)

请使用与小红书账号绑定的手机号登录“千帆”后台。

## 完成初始化设置

![](img/110b01671d7d0a67a8f60619ee0dcd09_23.png)

登录后，在首页会看到“完全初始化设置”或“开始经营交易”的提示。点击进入后，需要设置几个关键项目。

![](img/110b01671d7d0a67a8f60619ee0dcd09_25.png)

需要设置的核心项目包括：
1.  **物流信息**：设置发货和退货地址。
2.  **经营类目与公示信息**：填写经营信息并选择经营类型。
3.  **保证金**：根据平台规则缴纳。
4.  **微信进件**：绑定微信支付（非必填，可后续操作）。

![](img/110b01671d7d0a67a8f60619ee0dcd09_27.png)

我们建议先完成前两项必填设置，后两项可以等店铺有收入后再处理。

## 设置物流信息

![](img/110b01671d7d0a67a8f60619ee0dcd09_37.png)

![](img/110b01671d7d0a67a8f60619ee0dcd09_39.png)

上一节我们提到了初始化设置，本节中我们来看看如何具体设置物流信息。点击“物流信息”下的“去设置”，主要任务是创建物流模板和填写地址。

![](img/110b01671d7d0a67a8f60619ee0dcd09_41.png)

物流模板设置步骤如下：
1.  **模板名称**：填写一个便于自己识别的名称，例如“通用模板”或具体产品名。
2.  **贸易类型**：选择“一般贸易”。
3.  **发货/退货地址**：
    *   **单品店铺**：直接填写供应链提供的地址。
    *   **多品店铺（无货源模式）**：为防止被平台稽查，建议使用中转仓地址，或暂时填写自家地址。避免频繁更换多个不同地址。
4.  填写完毕后，点击“保存”。

![](img/110b01671d7d0a67a8f60619ee0dcd09_43.png)

关于中转仓的详细使用方法，我们会在后续课程中单独讲解。

## 设置经营类目与公示信息

![](img/110b01671d7d0a67a8f60619ee0dcd09_51.png)

![](img/110b01671d7d0a67a8f60619ee0dcd09_53.png)

物流信息设置完成后，接下来需要完善经营类目与公示信息。点击对应项目的“去设置”。

![](img/110b01671d7d0a67a8f60619ee0dcd09_55.png)

需要填写的内容包括：
*   **联系方式**：填写自己常用的手机号并接收验证码。
*   **公示经营类型**：**务必勾选“从事零星小额交易活动”**。
*   **承诺书**：勾选同意本店合法合规经营的承诺。
*   勾选承诺后，提交按钮会变红，点击“提交”即可。

![](img/110b01671d7d0a67a8f60619ee0dcd09_57.png)

![](img/110b01671d7d0a67a8f60619ee0dcd09_59.png)

保存成功后，店铺的基础设置就完成了，可以开始发布商品。

## 设置运费险

![](img/110b01671d7d0a67a8f60619ee0dcd09_67.png)

![](img/110b01671d7d0a67a8f60619ee0dcd09_69.png)

运费险可以提升买家购物信心，但不是强制设置。你可以根据供应链是否提供此服务，以及自身利润空间来决定是否开通。

![](img/110b01671d7d0a67a8f60619ee0dcd09_71.png)

![](img/110b01671d7d0a67a8f60619ee0dcd09_73.png)

设置路径如下：
1.  在“千帆”后台找到“全部工具”。
2.  在工具列表中找到“运费保”或类似选项。
3.  点击进入后，选择“同意并开通全店服务”。

![](img/110b01671d7d0a67a8f60619ee0dcd09_75.png)

![](img/110b01671d7d0a67a8f60619ee0dcd09_77.png)

开通时请注意：
*   系统会显示预估服务费，例如 **`每笔订单0.59元`**。
*   这笔费用会在发生退货理赔时扣除。
*   如果店铺退货率过高，平台可能会上调这笔费用。

![](img/110b01671d7d0a67a8f60619ee0dcd09_79.png)

## 出单后补填或修改物流信息

![](img/110b01671d7d0a67a8f60619ee0dcd09_84.png)

![](img/110b01671d7d0a67a8f60619ee0dcd09_86.png)

如果出单后发现之前设置的物流信息有误，或者需要新增地址，可以按以下步骤操作。

![](img/110b01671d7d0a67a8f60619ee0dcd09_88.png)

![](img/110b01671d7d0a67a8f60619ee0dcd09_90.png)

在“千帆”后台修改物流信息的流程：
1.  进入“物流设置” -> “地址管理”。
2.  点击“添加新地址”，将正确的发货/退货地址填写并保存。
3.  保存后，**必须进入“物流模板”**进行关联。
4.  在物流模板中，可以修改模板名称（例如“连衣裙供应商A”），并选择上一步添加的新地址作为发货地址。
5.  保存模板后，在发布商品时，就可以在物流选项中选择对应的模板。

![](img/110b01671d7d0a67a8f60619ee0dcd09_92.png)

![](img/110b01671d7d0a67a8f60619ee0dcd09_94.png)

**重要提醒**：对于发货和退货地址单一的店铺，可以直接在地址管理中将常用地址设置为“默认发货地址”和“默认退货地址”。平台会稽查无货源店铺，其中一个特征就是发货/退货地址过多且杂乱。因此，地址应尽量统一。

![](img/110b01671d7d0a67a8f60619ee0dcd09_96.png)

## 更换经营类目

![](img/110b01671d7d0a67a8f60619ee0dcd09_102.png)

![](img/110b01671d7d0a67a8f60619ee0dcd09_104.png)

如果你需要更换所售商品的类目（例如从冬季服装转卖夏季服装），可以按照以下方法操作。

![](img/110b01671d7d0a67a8f60619ee0dcd09_106.png)

更换类目的步骤：
1.  **清理内容**：将小红书账号上关于旧类目的商品笔记**删除或隐藏**。
2.  **修改后台**：在“千帆”后台的“物流设置”中，根据需要添加新的供应商地址并创建新的物流模板。
3.  之后即可按照新类目上架商品并发布笔记。

![](img/110b01671d7d0a67a8f60619ee0dcd09_108.png)

![](img/110b01671d7d0a67a8f60619ee0dcd09_110.png)
