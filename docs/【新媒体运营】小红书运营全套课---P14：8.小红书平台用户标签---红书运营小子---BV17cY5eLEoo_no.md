## 用户标签的判定逻辑

![](img/078a71281aa8e4858a1ee6802f33ac80_9.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_10.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_12.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_14.png)

平台为用户打标签的主要依据是用户的**浏览行为**。用户注册时选择的兴趣范围，仅作为初期内容方向的引导，参考价值不大，无法进入系统深层的用户标签池。

![](img/078a71281aa8e4858a1ee6802f33ac80_16.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_18.png)

系统根据用户的一系列浏览行为来判定其兴趣分类。以下是判定用户标签的核心行为指标：

![](img/078a71281aa8e4858a1ee6802f33ac80_20.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_22.png)

### 1. 点击率
点击率是指系统在发现页向用户推送多个内容后，用户点击不同类型内容的概率。例如，系统推送了明星、美妆、家居、母婴等内容，用户点击美妆类内容三次，家居类两次，母婴类一次或未点击。系统据此判定用户对母婴内容不感兴趣，可能不是宝妈，而对美妆内容感兴趣，后续会推送更多美妆内容进行试探。

![](img/078a71281aa8e4858a1ee6802f33ac80_24.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_26.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_28.png)

**公式示例**：
```
用户点击率 = (点击某类内容的次数 / 推送的该类内容总数) * 100%
```

![](img/078a71281aa8e4858a1ee6802f33ac80_30.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_32.png)

通过点击率，系统能为用户打上强关联或弱关联的标签。

![](img/078a71281aa8e4858a1ee6802f33ac80_34.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_36.png)

### 2. 停留时长
停留时长指用户点开一篇笔记后观看的时长。例如，用户通常快速划走美妆笔记，但某天点开一篇家居笔记后停留很久，仔细阅读图片和评论。系统会据此推测用户可能有装修或购买家居用品的需求，从而调整推送策略。

![](img/078a71281aa8e4858a1ee6802f33ac80_38.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_40.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_42.png)

### 3. 页面浏览量（PV）
PV指用户浏览页面的深度。例如，用户点开一篇装修笔记后，还进入博主主页浏览了其他多篇笔记。这提升了用户在页面的浏览数，对账号健康度评分有积极影响。在粉丝基数较小时，提升主页浏览深度能为账号带来非常健康的评分。

![](img/078a71281aa8e4858a1ee6802f33ac80_44.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_46.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_48.png)

### 4. 互动行为
互动行为包括点赞、收藏、评论和关注。这些数据直接影响笔记能否进入更大的推荐流量池，并带动账号其他笔记的流量和粉丝增长。这是我们运营小红书一直努力提升的目标。

![](img/078a71281aa8e4858a1ee6802f33ac80_50.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_52.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_54.png)

**代码示例（互动行为对推荐的影响逻辑）**：
```python
def calculate_note_score(likes, collects, comments, follows):
    # 假设各项互动有不同的权重
    score = likes * 0.3 + collects * 0.4 + comments * 0.2 + follows * 0.5
    return score

![](img/078a71281aa8e4858a1ee6802f33ac80_56.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_58.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_60.png)

# 笔记分数越高，进入下一级流量池的几率越大
note_score = calculate_note_score(100, 50, 30, 10)
```

## 小红书流量分发入口

![](img/078a71281aa8e4858a1ee6802f33ac80_66.png)

了解用户标签后，我们来看笔记通过哪些入口分发给用户。小红书主要有四个流量分发入口：

![](img/078a71281aa8e4858a1ee6802f33ac80_72.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_74.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_76.png)

### 1. 发现页
发现页是用户获取内容的主要入口，其推荐逻辑混合了多种因素：
*   **关注达人内容**：用户已关注的博主发布的新笔记。
*   **兴趣内容匹配**：根据用户标签（点击率、停留时长等）推荐的、系统认为用户可能喜欢的内容。
*   **发散性内容匹配**：系统也会推荐一些用户可能不感兴趣或未接触过的内容，以增加平台内容的多样性，满足用户潜在的多方位需求。

![](img/078a71281aa8e4858a1ee6802f33ac80_78.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_80.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_82.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_84.png)

对于大多数博主而言，后期能获得的主要流量就来源于发现页。

### 2. 关注页
关注页的内容相对纯粹，只展示用户所关注的所有博主发布的笔记。用户通常从这里直接查看强关注对象的更新。

![](img/078a71281aa8e4858a1ee6802f33ac80_86.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_88.png)

### 3. 附近页
附近页根据手机定位，向用户推送同城或附近地区的相关内容。此入口强地域特征，适合美食、本地旅游、线下店推广（如美容院、餐厅）等内容。若运营线下实体店，需特别注意附近人群的推送策略。

![](img/078a71281aa8e4858a1ee6802f33ac80_90.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_92.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_94.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_95.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_97.png)

### 4. 购物页
购物页是小红书向电商方向发展的重要入口，主要满足用户的购物需求，包含商品笔记、直播带货和商城。目前，许多用户仍习惯通过搜索笔记来“种草”，而非直接进入购物页搜索商品。

![](img/078a71281aa8e4858a1ee6802f33ac80_99.png)

## 总结

本节课中我们一起学习了小红书平台判定用户标签的核心逻辑与四大流量分发入口。

![](img/078a71281aa8e4858a1ee6802f33ac80_101.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_103.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_105.png)

*   **用户标签** 主要由**点击率、停留时长、页面浏览深度（PV）和互动行为**决定，系统据此描绘用户画像，实现精准推荐。
*   **流量入口** 主要包括**发现页、关注页、附近页和购物页**。其中，**发现页**是笔记获取流量的主战场，其推荐机制综合了用户标签与平台内容多样性需求。

![](img/078a71281aa8e4858a1ee6802f33ac80_107.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_109.png)

![](img/078a71281aa8e4858a1ee6802f33ac80_111.png)

理解“用户-平台-博主”三方的关联（平台通过算法服务用户，博主创作内容满足用户并为平台提供价值），是做好小红书运营的基础。通过优化内容以提升用户的点击、停留、浏览和互动数据，我们的笔记才能更有效地被系统识别，并推荐给精准用户，从而实现涨粉与变现的目标。
