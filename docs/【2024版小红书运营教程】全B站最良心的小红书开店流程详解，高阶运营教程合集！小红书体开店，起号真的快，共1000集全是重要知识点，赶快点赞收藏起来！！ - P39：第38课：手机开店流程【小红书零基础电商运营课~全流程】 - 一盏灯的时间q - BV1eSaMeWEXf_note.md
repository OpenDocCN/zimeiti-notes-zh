# 小红书电商运营课：P39：手机开店流程

## 概述
在本节课中，我们将学习如何通过手机应用开通一个小红书店铺。课程将详细演示从进入入口到完成入驻的全过程，并解释其中的关键注意事项，例如保证金政策与店铺类型的选择。

## 课程内容

上一节我们介绍了小红书电商的基本概念，本节中我们来看看如何具体操作，在手机上完成店铺的开通。

很多用户不知道如何开通小红书店铺，这里直接进行演示。开通店铺不需要预先缴纳保证金，这一点需要注意。因为缴纳保证金通常是为了开通特定功能。

小红书对保证金的限制并不严格，前期无需缴纳。前期如果需要开通运费险、支付宝支付、微信支付以及提现功能，才需要缴纳保证金。

因此，前期注册店铺可以直接操作，无需先缴纳保证金。

这是小红书应用的主页面。注册店铺的方法是：点击左上角的三条横线图标，进入菜单后选择“创作中心”。

进入创作中心后，页面下方有“更多服务”选项。

以下是找到开店入口的步骤：
1.  在“更多服务”中，找到并点击“开通店铺”。
2.  点击后即可进入开通店铺的流程。

还有另一个入口路径：从“我”的页面进入“账号与安全”设置。

在“账号与安全”中下滑找到“升级专业号”选项。升级专业号是更全面的方法。

以下是升级专业号并开店的步骤：
1.  在升级专业号页面，选择第一个选项，立即申请。
2.  按要求上传身份证信息，提交专业号升级申请。
3.  专业号升级成功后，再去点击“立即入驻店铺”。

为什么推荐先升级专业号？因为有时直接点击“立即开店”，可能会遇到违规提示或违反社区规定等情况。

先认真升级专业号后再开通店铺，流程更齐全，基本上不会出错。

入驻店铺时，在选择店铺类型环节需要注意。

以下是入驻时的关键选择：
*   **店铺类型**：务必选择“普通商品”。
*   **信息上传**：根据提示对应上传身份证等信息即可。

提交申请后，开店速度因店铺类型而异：
*   **个人店**：通常是秒开。
*   **个体店**：需要等待约三个工作日审核，这点需要注意。

## 总结
本节课我们一起学习了通过手机开通小红书店铺的完整流程。核心要点包括：前期无需缴纳保证金、通过“创作中心”或“升级专业号”两个入口进入、推荐先升级专业号以确保流程顺利、入驻时选择“普通商品”类型，并了解了个人店与个体店在审核时间上的区别。
