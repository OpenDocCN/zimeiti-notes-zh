# 剪映入门教程：P25：剪辑篇丨基础界面全解析 🎬

在本节课中，我们将学习剪映App的基础界面和核心操作。通过一个简单的案例，你将了解从创建项目到导出的完整流程，并掌握各个功能区的用途。

## 概述

我们将从打开剪映开始，逐步介绍主界面布局、素材导入、基础工具（如文本、音频、剪辑）的使用，以及画中画、特效、导出等关键功能。目标是让你对剪映的核心界面有一个清晰、全面的认识。

## 详细教程

打开剪映App。主界面有两个关键部分：顶部的“开始创作”按钮用于新建项目；下方的“草稿箱”用于保存所有已完成或未完成的作品。

接下来，我们通过一个简单案例来了解基础界面。点击“开始创作”进入编辑页面。

编辑页面上方是素材选择区。这里需注意两个标签：“视频”标签下导入视频文件，“照片”标签下导入图片文件。不要在“视频”标签中寻找照片。

我们可以使用剪映自带的“素材库”。素材库内有丰富的视频和图片素材可供下载使用。

本节将使用一个黑色背景素材进行讲解。点击添加该素材后，界面中央是“预览窗口”。

首先需要设置视频比例。初始比例通常是横屏16:9。但抖音支持竖屏视频，因此我们需要更改比例。

在底部工具栏中找到“比例”选项，选择竖屏9:16。预览窗口中的画面会随之变为竖版。

完成基础设置后，应习惯性点击左上角箭头“返回”主工具栏。主工具栏包含多项核心功能。

以下是主工具栏各功能的简要说明：

*   **文本**：点击“文本”可以为视频添加字幕或标题。
*   **音频**：点击“音频”可以为视频添加背景音乐或音效。
*   **剪辑**：选中一段素材后，“剪辑”功能可对该段素材进行详细编辑。
*   **贴纸**：贴纸功能类似于动态贴图，可以为视频添加各种装饰性元素。我们点开查看，里面有多种贴纸可供选择。使用后记得返回主工具栏。
*   **画中画**：此功能可在主视频上叠加另一段视频素材。例如，当前已有一段黑色背景素材，点击“新增画中画”添加另一段素材（如一段风景视频）。新增的素材会覆盖在主素材之上。通过双指在屏幕上向外滑动可以放大画中画素材。此时，画中画素材会遮挡住下方的黑色背景。
*   **特效**：特效功能可以为视频添加各种视觉效果，如模糊、闪光等。
*   **滤镜**：滤镜功能可以一键调整视频的整体色调和风格，类似于照片美颜中的滤镜效果。里面预设了多种配色方案。
*   **背景**：背景指视频画布的颜色或样式。当我们使用竖屏（9:16）比例时，视频两侧可能出现黑边，背景功能可以更改这两侧区域的颜色或图案。更改后如果没立即看到效果，可能是因为画中画素材覆盖了背景。向后拖动时间轴，或暂时隐藏上层素材，即可看到背景变化。
*   **导出**：作品完成后，点击右上角“导出”按钮即可保存视频到手机。导出时需选择参数，通常选择“1080P”和“60帧”可以获得较高清晰度和流畅度。

上一节我们介绍了工具栏的静态功能，本节中我们来看看对时间线上素材的动态编辑。

编辑区下方是时间线轨道。选中一段素材后，可以对它进行基础剪辑操作。

以下是两个最常用的剪辑操作：

*   **分割**：如果一段素材过长，需要删除中间部分。可以将时间线指针移动到想要切割的位置，点击“分割”工具，素材会被切成两段。选中不需要的片段，点击“删除”（垃圾桶图标）即可移除。
*   **拖动裁剪**：更快捷的方法是直接拖动素材两端的白色边框，向内拖动即可裁剪掉头尾部分。

许多新手可能会遇到轨道上的素材“消失”或“合并”的情况。有两种方法可以找回：

1.  点击时间线轨道左侧的“折叠/展开”小圆圈，可以展开或收起所有轨道。
2.  如果素材是通过“画中画”添加的，点击底部工具栏的“画中画”选项，该轨道会自动显示出来。

如果想要全屏预览视频效果，可以点击预览窗口右下角的“放大镜”图标进入全屏模式，再次点击则退出。

## 总结

本节课中我们一起学习了剪映的基础界面和核心操作。我们了解了如何创建项目、导入素材、设置视频比例，并详细讲解了主工具栏中文本、音频、画中画、特效、滤镜等关键功能。同时，也掌握了分割素材、导出视频以及管理多轨道的基本方法。熟悉这个界面是进行后续复杂剪辑的第一步。
