# 剪映教程：P11：剪映核心剪辑功能讲解--音频功能 🎵

![](img/44c8a8b512914787bc7ef0416940c618_1.png)

在本节课中，我们将要学习剪映APP中一个至关重要的功能——音频功能。一条完整的短视频不仅需要精彩的画面，也离不开合适的背景音乐或音效来烘托氛围。本节课程将详细介绍如何为你的视频添加和处理音频。

上一节我们介绍了视频剪辑中的分割功能，本节中我们来看看如何利用音频功能提升视频的质感。

## 概述：音频功能的重要性

![](img/44c8a8b512914787bc7ef0416940c618_3.png)

我们知道，一条吸引人的爆款视频，一定离不开一个热门的BGM（背景音乐）。在剪映中，我们可以通过音频功能为视频添加背景音乐或音效，还可以通过提取音乐和抖音收藏的方式直接选取热门BGM，最后还能为视频录制旁白。这些操作能大大提升视频的听觉质量和整体氛围。

![](img/44c8a8b512914787bc7ef0416940c618_4.png)

## 音频功能界面介绍

音频功能位于剪映APP的一级菜单栏。添加视频素材后，点击底部菜单栏的“音频”按钮，即可进入音频操作界面。

![](img/44c8a8b512914787bc7ef0416940c618_6.png)

![](img/44c8a8b512914787bc7ef0416940c618_8.png)

以下是音频功能包含的五个核心子功能：

*   **添加音乐**：从剪映内置音乐库选择背景音乐。
*   **添加音效**：为视频添加各种短小的效果音。
*   **提取音乐**：从手机相册的视频文件中提取音频。
*   **抖音收藏**：使用在抖音APP中收藏过的音乐。
*   **录音**：为视频录制旁白或解说。

![](img/44c8a8b512914787bc7ef0416940c618_10.png)

![](img/44c8a8b512914787bc7ef0416940c618_11.png)

接下来，我们将通过实际操作，逐一讲解这五个功能。

![](img/44c8a8b512914787bc7ef0416940c618_13.png)

## 实操演练：添加与处理音频

我们首先导入一段示例视频素材。预览原素材可以发现，它只有环境音（如风声），缺乏明确的主题氛围。我们可以通过添加音频来强化视频想传达的思想。

### 1. 添加音乐

第一种方法是在视频轨道下方点击“添加音频”按钮，然后选择“添加音乐”。

点击后，会跳转到剪映的音乐素材库。这个曲库与抖音热门音乐是打通的，并按照“卡点”、“流行”、“旅行”、“美食”等类别进行了划分，同时也推荐当前热门音乐。

例如，我们可以选择“夏天的风”这首音乐，点击“使用”按钮。

音乐添加后，需要调整其位置。选中音频轨道，按住并向前拖动，使其与视频开头对齐。

预览效果时，如果原视频声音干扰了背景音乐，可以点击视频轨道上的“关闭原声”按钮。

如果音乐长度超过视频，我们需要进行裁剪。定位到视频结尾处，选中音频轨道，点击底部“分割”按钮，然后将多余的部分删除。

为了让音乐的开始和结束更自然，我们可以使用“淡化”功能。选中音频轨道，点击“淡化”，分别设置“淡入时长”和“淡出时长”，例如各设置为1秒。

![](img/44c8a8b512914787bc7ef0416940c618_15.png)

此外，我们还可以调整音乐的音量。由于平台压缩可能导致音量变小，建议将导入的音乐音量适当调大。

![](img/44c8a8b512914787bc7ef0416940c618_17.png)

**相关操作代码示意：**
*   分割音频：`选中轨道 -> 点击“分割”`
*   淡化处理：`选中轨道 -> 点击“淡化” -> 设置时长`
*   调整音量：`选中轨道 -> 点击“音量” -> 拖动滑块`

### 2. 添加音效

![](img/44c8a8b512914787bc7ef0416940c618_24.png)

![](img/44c8a8b512914787bc7ef0416940c618_25.png)

![](img/44c8a8b512914787bc7ef0416940c618_26.png)

音效是指一些特殊的短音频，常用于Vlog、搞笑视频等场景，用以增强某个动作或场景的效果。

![](img/44c8a8b512914787bc7ef0416940c618_27.png)

![](img/44c8a8b512914787bc7ef0416940c618_28.png)

![](img/44c8a8b512914787bc7ef0416940c618_29.png)

在“音频”菜单下选择“音效”，即可打开音效素材库。库中包含游戏、魔法、美食、动物、环境等多种类型的音效，可以根据视频内容直接选取使用。

![](img/44c8a8b512914787bc7ef0416940c618_30.png)

![](img/44c8a8b512914787bc7ef0416940c618_31.png)

![](img/44c8a8b512914787bc7ef0416940c618_32.png)

### 3. 提取音乐

此功能允许你将手机相册中某个视频文件的音频单独提取出来，并用到当前剪辑中。

在“音频”菜单下选择“提取音乐”，然后从相册中选择一个带有音频的视频文件。导入后，该视频的音频会被提取为一个独立的轨道，你可以像处理普通音乐一样对其进行调整。

![](img/44c8a8b512914787bc7ef0416940c618_34.png)

![](img/44c8a8b512914787bc7ef0416940c618_36.png)

### 4. 使用抖音收藏的音乐

如果你想使用抖音上的热门音乐拍同款，可以先将该音乐收藏。

![](img/44c8a8b512914787bc7ef0416940c618_38.png)

在抖音APP中，找到目标视频，点击右下角的“碟片”图标，然后点击“收藏”。收藏后，回到剪映的“音频”界面，选择“抖音收藏”，就能看到并使用你收藏过的音乐了。

![](img/44c8a8b512914787bc7ef0416940c618_40.png)

### 5. 录音（添加旁白）

![](img/44c8a8b512914787bc7ef0416940c618_42.png)

如果你需要为视频添加后期解说或旁白，可以使用录音功能。

在“音频”菜单下选择“录音”，按住红色录音按钮开始录制。录制完成后，录音文件会以音频轨道的形式出现在时间轴上。你可以对其位置、音量进行调整，使其与背景音乐和画面配合。

## 总结

本节课我们一起学习了剪映的核心剪辑功能——音频功能。通过本课，你掌握了以下技能：

![](img/44c8a8b512914787bc7ef0416940c618_44.png)

1.  **添加音乐**：从剪映曲库选择合适BGM，并进行对齐、裁剪、淡化和音量调整。
2.  **添加音效**：使用丰富的音效素材库为视频增添趣味和临场感。
3.  **提取音乐**：将本地视频中的音频提取出来复用。
4.  **抖音收藏**：便捷地使用在抖音上收藏的热门音乐。
5.  **录音**：为视频录制自定义的旁白或解说。

综合利用这些音频功能，你可以快速为视频配上背景音乐和音效，甚至通过录制旁白来明确视频主题，从而显著提升视频的完整度和专业感。
