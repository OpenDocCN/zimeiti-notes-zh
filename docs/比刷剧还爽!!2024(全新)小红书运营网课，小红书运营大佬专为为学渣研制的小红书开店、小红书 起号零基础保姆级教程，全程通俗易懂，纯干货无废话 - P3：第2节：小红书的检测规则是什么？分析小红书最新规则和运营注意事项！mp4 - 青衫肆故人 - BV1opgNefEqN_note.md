## 平台检测的四大核心方面

上一节我们介绍了小红书运营的基础概念，本节中我们来看看平台具体会从哪些方面进行审核。小红书的检测系统主要围绕以下四个维度展开。

以下是平台重点检测的四个具体方面：

1.  **账号与内容信息**：平台会检测个人资料、笔记内容、私信及评论。这些地方不得含有微信号、微信二维码、代购、水印、推广淘宝店铺等营销词汇。
2.  **设备与登录环境**：平台会检测手机设备。建议同一台设备最多只登录三个账号，超出可能引发风险。
3.  **用户互动行为**：笔记内容不得诱导用户。例如，“关注领奖”、“点赞抽奖”等属于小红书明令禁止的诱导行为。
4.  **内容原创度**：平台对原创内容的保护程度很高。切勿将同一篇文章在多个账号上发布，应遵循“一篇文章一个账号”的原则。

## 敏感词检测与规避方法

除了上述通用规则，内容中的敏感词是触发审核的常见原因。因此，掌握敏感词的检测与规避至关重要。

以下是需要特别注意的敏感词类别及检测方法：

*   **高风险领域**：例如“手机卡”、“金融”、“电子烟”、“代购”、“医疗用品”等领域的词汇，在创作时应尽量避免触及。
