# 抖音运营实战：P12：2.自媒体平台现状 📊

![](img/f8954c14d213b9fc37b2b1cf2e06282d_0.png)

在本节课中，我们将要学习自媒体行业的现状，分析其是否值得进入，并了解国家政策与平台支持的具体情况。这对于判断行业前景和制定个人策略至关重要。

上一节我们介绍了自媒体的基本概念，本节中我们来看看这个行业当前的发展环境与可行性。

![](img/f8954c14d213b9fc37b2b1cf2e06282d_2.png)

很多人对进入自媒体行业存在疑问，不确定这个行业是否真的能做，或者对自己有什么好处。近年来，由于各种原因，许多人希望寻找副业或转行。在这个过程中，部分人抱有暴富心态，甚至采取了不合法的手段，最终导致严重后果。这些行业是国家明令禁止的，对个人没有任何好处。

![](img/f8954c14d213b9fc37b2b1cf2e06282d_4.png)

现阶段，自媒体行业属于国家支持的范畴。国家已将支持文件下达到各个地方政府。

![](img/f8954c14d213b9fc37b2b1cf2e06282d_6.png)

以下是地方政府支持政策的示例：

![](img/f8954c14d213b9fc37b2b1cf2e06282d_8.png)

![](img/f8954c14d213b9fc37b2b1cf2e06282d_10.png)

![](img/f8954c14d213b9fc37b2b1cf2e06282d_11.png)

*   **荆州市六项措施**：该文件由荆州市人力资源和社会保障局于去年6月发布，明确指出要“加大政策宣传力度，有力支持自媒体以及新媒体行业的蓬勃发展”。
*   **广西壮族自治区政策**：同样发布了关于支持新媒体行业发展的政策性文件。
*   **其他地区文件**：多个地区都发布了相应的政策性文件，文件中强调了要坚持正确导向。

![](img/f8954c14d213b9fc37b2b1cf2e06282d_13.png)

这些文件表明，我们可以放心地从事自媒体行业。但必须强调一点，国内自媒体平台监管严格。

以下是严格禁止的内容类型：

![](img/f8954c14d213b9fc37b2b1cf2e06282d_15.png)

*   涉及“黄、赌、毒”的内容。
*   任何形式的擦边球内容。

例如，某些海外平台上的擦边内容绝不能发布到国内平台。这一点必须首先明确。

有了国家政策支持，行业就有了基础保障。常言道，跟着国家脚步走，一定能赚到钱。在政策号召下，自媒体平台也积极响应，为创作者提供了实质支持。

平台通过提供创作基金、补贴等方式支持视频创作者。例如，西瓜视频曾宣布未来一年将额外拿出20亿元补贴中视频创作人。

这20亿元有两个关键点：

![](img/f8954c14d213b9fc37b2b1cf2e06282d_17.png)

1.  **这是真金白银的纯补贴**，不包含任何商单、直播和电商的收入。
2.  **20亿元只是保底，上不封顶**。

![](img/f8954c14d213b9fc37b2b1cf2e06282d_19.png)

这说明了什么？它说明自媒体创作者并非只有带货一条变现路径。通过创作视频或文章，依靠纯播放量同样可以变现。这20亿元就是创作者未来可能获得的收益之一。

除了平台补贴，自媒体行业本身拥有庞大的流量基础。根据相关市场调查报告，现阶段中国网民规模已达约9.4亿人。从学生到各年龄段成年人，都在各类平台上观看视频。

与庞大的观看群体相比，内容创作者的数量相对较少。数据显示，我国全职从事自媒体的人约有370万，兼职约600万，合计近1000万人。

我们可以做一个简单的对比：**观看人数 / 创作人数 ≈ 94**。这相当于约100个看视频的人，对应1个做视频的人。一个人创作视频的速度，很难满足一百个人观看视频的需求。这种供需关系的不平衡，正是国家支持自媒体行业发展的重要原因之一。

很多同学担心行业是否已经饱和。根据行业趋势分析，在大数据时代背景下，**自媒体行业远远没有饱和**。“远远没有”这个词明确指出了行业在未来很长一段时间内，依然有巨大的发展空间和机会。无论是作为兼职还是全职，自媒体在未来十年内都是一个值得考虑的创收行业。

![](img/f8954c14d213b9fc37b2b1cf2e06282d_21.png)

本节课中我们一起学习了自媒体行业的现状。我们了解到该行业受到国家政策明确支持，且各大平台提供了真金白银的补贴激励。同时，庞大的用户基数与相对较少的创作者数量形成了供需缺口，表明行业远未饱和，发展前景广阔。当然，从业必须严格遵守平台规范，远离违规内容。理解这些现状，是迈出成功第一步的关键。
