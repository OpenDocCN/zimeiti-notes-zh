# 自媒体运营课程：P25：剪辑核心功能——滤镜调色 🎨

在本节课中，我们将学习如何使用剪映的滤镜与调色功能，快速提升视频画面的色彩表现力与整体质感。

很多人在拍摄视频时会遇到色彩不够鲜亮、画面显得单调的问题。实际上，许多视觉效果出色的视频并非拍摄难度高，而是在后期剪辑中巧妙地运用了滤镜和调色。本节课将指导你如何借助剪映的滤镜和调色功能，让你的视频也变得色彩鲜明、富有吸引力。

## 为何需要调色？

快速提升视频质感的关键在于调色。在传统视频剪辑中，可以使用如 **Adobe Premiere Pro (Pr)** 的调色面板或插件，甚至专业的 **DaVinci Resolve (达芬奇)** 软件。但对于初学者而言，这些工具可能过于复杂。

剪映同样提供了便捷的调色功能，主要分为两部分：
*   **滤镜**：这是预设好的色彩模板，直接套用即可快速改变视频风格。
*   **调色**：这是对视频色彩进行个性化调整的功能。例如，你可以单独让天空变得更蓝，或让草地变得更绿。

接下来，我们将进入实战环节，详细讲解如何使用这两项功能。

## 实战：滤镜与调色应用

![](img/c6c8cfa0927e298612818d6a1a03b7ee_3.png)

![](img/c6c8cfa0927e298612818d6a1a03b7ee_4.png)

上一节我们介绍了调色的基本概念，本节中我们来看看具体的操作步骤。

首先，在剪映中“开始创作”，导入一段需要调整的素材。观察原素材，你可能会发现树木不够绿，天空不够蓝。我们的调色目标就是让天空更蓝，植物更绿，使画面色彩更鲜亮。

了解需求后，我们开始调整。

### 第一步：使用滤镜快速美化

以下是使用滤镜的步骤：

1.  在剪辑界面，点击右下角的 **“滤镜”** 按钮。
2.  在滤镜库中选择一个喜欢的模板，例如 **“鲜亮”**。
3.  应用后，可以立即看到视频色彩发生显著变化。你可以尝试不同滤镜，观察效果。

以“鲜亮”滤镜为例，对比原视频和应用后的视频，变化非常明显。滤镜能快速统一视频风格。

### 第二步：进行个性化调色

如果对滤镜效果仍有不满意的地方，可以进行更精细的个性化调整。

以下是使用调节功能的步骤：

1.  点击底部工具栏的 **“调节”**（或“新增调节”）选项。
2.  这里提供了多个可调整的参数：
    *   **亮度**：控制画面明暗。
    *   **对比度**：增强明暗差异，使画面更立体。
    *   **饱和度**：控制色彩鲜艳程度。**公式：饱和度 ↑ = 色彩更鲜艳**。
    *   **锐化**：让画面细节更清晰。
    *   **高光**：调整画面亮部区域的明暗。
    *   **阴影**：调整画面暗部区域的明暗。
    *   **色温**：调整画面冷暖色调。
    *   **色调**：调整画面色彩倾向。

3.  根据素材进行微调。例如：
    *   亮度已足够，可略微降低。
    *   提升对比度和饱和度，让色彩更鲜明、丰富。
    *   适当增加锐化，让画面更细腻。
    *   微调高光、色温和色调，使画面色彩更柔和、自然。

调整完成后，再次对比原视频与调色后的视频。你会发现，天空变得更蓝，树木也更绿了，整体观感大幅提升。
