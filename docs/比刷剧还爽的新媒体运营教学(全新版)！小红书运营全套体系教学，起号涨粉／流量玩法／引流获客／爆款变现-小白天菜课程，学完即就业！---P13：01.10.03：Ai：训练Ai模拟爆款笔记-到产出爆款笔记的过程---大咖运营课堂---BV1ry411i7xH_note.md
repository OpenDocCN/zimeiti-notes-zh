# 新媒体运营教学：P13：01.10.03：AI训练与爆款笔记产出 🚀

在本节课中，我们将学习如何训练AI（如ChatGPT）分析爆款笔记，并最终生成具备爆款潜质的新笔记。整个过程分为两个核心步骤：首先，让AI学习并总结爆款内容的成功要素；其次，指导AI基于这些要素创作新的内容。

上一节我们介绍了通过“喂资料”的方式，让AI理解特定平台的文案风格。虽然上一节的旅游攻略案例效果不理想，但方法是正确的。本节中，我们来看看如何深化这一过程，通过分析具体的爆款笔记，让AI掌握创作爆款内容的关键。

## 第一步：指导AI分析爆款笔记 📊

我们的目标是让AI成为我们的“内容分析师”。为此，我们需要向AI提供具体的爆款笔记，并明确指令它分析以下几个核心维度：

![](img/01767a5e85f4d73f8033be5ebd9246dd_7.png)

![](img/01767a5e85f4d73f8033be5ebd9246dd_9.png)

![](img/01767a5e85f4d73f8033be5ebd9246dd_11.png)

![](img/01767a5e85f4d73f8033be5ebd9246dd_13.png)

以下是需要AI分析的具体要点列表：
*   **爆款因素**：分析内容为何能成为爆款（例如：情感共鸣、实用价值、逆袭故事等）。
*   **整体结构**：
    *   **开篇**：用什么方式（如提问、痛点描述、名人引言）吸引用户继续阅读。
    *   **中间**：提供了哪些具体内容来满足用户需求或解决痛点，以提升用户停留时长。
    *   **结尾**：设置了什么“钩子”（如鼓励互动、提供福利、引导私信）来促进用户行动（关注、私信、评论）。
*   **关键词策略**：分析笔记中是否植入了特定关键词，以及关键词的出现频率和分布方式。

## 实战演练：以考研领域为例 📚

![](img/01767a5e85f4d73f8033be5ebd9246dd_21.png)

![](img/01767a5e85f4d73f8033be5ebd9246dd_23.png)

我们选取一篇小红书平台上点赞数较高的考研笔记作为分析样本。

![](img/01767a5e85f4d73f8033be5ebd9246dd_25.png)

![](img/01767a5e85f4d73f8033be5ebd9246dd_27.png)

![](img/01767a5e85f4d73f8033be5ebd9246dd_29.png)

首先，我们向ChatGPT（此处以3.5版本为例）发出清晰的指令：
```
“我给你提供一篇小红书平台上的考研爆款笔记文案，你需要分析：
1. 文案的爆款因素。
2. 文案的整体结构（开篇如何吸引用户，中间如何戳中痛点，结尾如何引导互动）。
3. 分析该笔记是否结合了关键词策略。如果有，指出关键词及其出现频率。”
```
然后，将复制的爆款笔记文案粘贴给AI。

![](img/01767a5e85f4d73f8033be5ebd9246dd_31.png)

![](img/01767a5e85f4d73f8033be5ebd9246dd_33.png)

![](img/01767a5e85f4d73f8033be5ebd9246dd_35.png)

AI返回的分析示例如下：
*   **爆款因素**：情感共鸣（开篇描述备考焦虑）、逆袭故事、提供实用的备考方法与时间规划、使用表情符号增加亲和力、结尾传递正能量与鼓励。
*   **整体结构**：开篇用焦虑场景引发共鸣 → 中间分享个人逆袭经验和分科备考建议 → 结尾鼓励读者并暗示可提供进一步帮助。
*   **关键词分析**：本篇笔记未发现明显的关键词策略，多为直接、常用的词汇。

我们重复此过程，为AI提供多篇（建议3-4篇）同领域的爆款笔记进行分析，让它积累更丰富的认知。有时AI可能会“忘记”最初的分析要求，需要我们在对话中重新强调指令，将其“拉回”正轨。

## 第二步：整合要素并生成新笔记 ✍️

在AI分析了多篇样本后，我们需要让它对学习到的爆款要素进行整合与去重。

我们向AI发出指令：
```
“请将你前面分析过的几篇考研爆款笔记的爆款因素总结一下，整合成一个清晰的列表。”
```
AI总结的爆款因素列表可能包括：情感共鸣、逆袭故事、明确的备考计划、针对性建议、具体资源推荐、时间节点管理、正能量鼓励、使用表情符号等。

接下来，我们进入创作阶段。首先，让AI根据总结的爆款要素，规划一篇新笔记的框架。

![](img/01767a5e85f4d73f8033be5ebd9246dd_41.png)

指令如下：
```
“我要写一篇关于考研的小红书笔记。请结合你前面总结的爆款笔记因素，帮我梳理一个具备爆款潜质的笔记大纲。每个大纲节点需要进一步细分出要点。”
```
AI生成的大纲可能包括：吸引人的标题与开篇、专业选择分析、各科目（政治/英语/数学/专业课）备考规划、时间管理与心态调整、结尾互动引导等部分。

大纲确认后，我们给出最终生成指令：
```
“请按照你提供的大纲，撰写具体的文案。要求：
1. 文案结构严格遵循爆款笔记因素和结构。
2. 包含一个简单的‘逆袭小故事’引入，引发读者情绪。
3. 融入适量的emoji表情。
4. 尝试结合关键词策略（如‘考研’、‘备考规划’等）。”
```
AI根据指令生成的文案，在结构、情感点和实用性上通常已具备基础框架，能够满足一般运营需求。例如，开篇可能以“我理解那种时间不够用的焦虑…”引入，中间提供分阶段备考建议，结尾以“加油，有任何问题可以找我…”引导互动。

## 优化与进阶 🚀

对于生成的内容，我们可以进一步优化：
*   **内容调整**：根据自身经验，对AI生成的文案进行润色，使其更简洁、有力或更符合个人风格。
*   **工具升级**：使用更强大的模型（如GPT-4）或利用“自定义GPT”（GPTs）功能，上传更多专属风格样本和表情库，打造更懂你的私人AI助手，从而获得更高质量、更稳定的输出。
