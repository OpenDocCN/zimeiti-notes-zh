# 新媒体运营全套教程：P17：02.2：工具篇-AI生成PPT 🚀

## 概述
在本节课中，我们将学习如何使用一款高效的AI工具来生成PPT。这款工具能够根据你提供的主题或现有文档，自动梳理大纲并生成详细的PPT内容，极大地提升内容创作效率。

## 工具介绍与模式
上一节我们介绍了新媒体运营的各种策略，本节中我们来看看如何利用AI工具辅助内容呈现。这款AI生成PPT工具主要提供两种工作模式。

以下是两种核心使用模式：
1.  **主题生成模式**：输入一个主题，工具会结合类似ChatGPT的生成式AI能力，自动创作全文并生成PPT。
2.  **文档转换模式**：上传一篇现有的文档（如TXT、PDF），AI助手会自动总结文档大纲，并基于此大纲生成PPT内容。

## 编辑与优化生成结果
等待内容生成完毕后，我们可以进入编辑界面查看和调整PPT。这是将AI产出转化为最终成果的关键步骤。

工具界面右侧提供了丰富的编辑选项，允许用户进行细节调整。

![](img/2f327e97466e263c7e24d4dd05e9d940_7.png)

以下是主要的编辑与优化功能：
*   **更换模板**：虽然当前内置模板数量有限，但切换速度很快，未来可能会增加。
*   **调整内容**：可以直接在生成好的PPT页面上修改文字、替换形状等。
*   **本地深度编辑**：可以将PPT下载到本地，使用PowerPoint或Keynote等软件进行更精细的调整。

![](img/2f327e97466e263c7e24d4dd05e9d940_9.png)
