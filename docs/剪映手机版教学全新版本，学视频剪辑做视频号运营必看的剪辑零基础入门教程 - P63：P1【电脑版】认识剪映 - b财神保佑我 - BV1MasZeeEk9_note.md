## 软件下载与安装

首先，我们需要下载剪映专业版。

![](img/5d90b9607156ab3a2ad06832e44f652e_9.png)

1.  打开浏览器，搜索“剪映”。
2.  找到并进入剪映的官方网站。
3.  在官网上选择“专业版”进行下载。

下载完成后，找到下载的应用程序文件（通常为 `.exe` 文件），双击开始安装。安装过程中，如果电脑安全软件弹出提示，请选择“允许”或“继续”。安装完成后，点击“立即体验”即可启动剪映软件。

## 主界面功能详解

![](img/5d90b9607156ab3a2ad06832e44f652e_11.png)

启动剪映后，我们看到的是软件的主界面。上一节我们完成了安装，本节中我们来详细了解一下主界面的各个部分。

### 左侧个人面板

界面左侧是个人功能区域。

![](img/5d90b9607156ab3a2ad06832e44f652e_13.png)

*   **账号登录**：如果你有抖音账号，可以在此登录。登录后，你在抖音APP中收藏的音乐可以同步到电脑版剪映中使用。
*   **热门模板**：这里展示了抖音上流行的各种视频模板。如果你缺乏创作灵感，可以直接使用这些模板进行快速制作。
*   **云空间**：这是你的个人云端存储空间。你可以将本地电脑上的音乐、视频、图片等素材拖拽上传至此。这样，你在其他电脑上登录同一账号时，也能使用这些素材。上传方法很简单：**直接将文件拖拽到“云空间”区域即可**。
*   **小组云空间**：功能与个人云空间类似，但用于团队协作。团队成员可以在此共享素材和工程文件。
*   **热门活动**：这里会展示抖音官方的热门活动。参与这些活动可能获得更多的流量推荐。

### 中央创作区域

界面中央是核心的创作入口和项目管理区。

*   **开始创作**：点击中间的“+”号，即可进入视频剪辑界面，开始创作新项目。
*   **草稿箱**：你所有保存过的剪辑项目（在剪映中称为“草稿”）都会自动陈列在这里。你可以随时点击任一草稿继续编辑，无需手动寻找工程文件。**在剪映中，“项目”等同于“草稿”**。
*   **导入工程**：这是一个强大的功能，支持导入Adobe Premiere (PR) 的工程文件。将鼠标悬停在该按钮上可以看到提示：**支持导入PR工程，支持属性包含基础剪辑、音频、视频基础属性、文本内容**。这意味着你可以在PR中完成初步剪辑，然后导入剪映来添加丰富的特效、转场和调色。不过需要注意的是，PR中的关键帧动画和部分转场效果在导入后可能无法被识别。

### 顶部智能工具

界面顶部提供了四个基于AI技术的智能创作工具，能极大提升效率。

*   **创作脚本**：这个功能可以帮助你制作专业的分镜头脚本。你可以创建项目，填写大纲、镜头描述等内容，规划好视频的每一个画面，完成后可直接导入剪辑面板。
*   **一起拍**：这是一个线上实时协作功能。你可以创建一个房间，邀请好友加入，双方可以开启摄像头和麦克风，实现远程合拍视频或同步观影。
*   **智能比例**：此功能可以智能地将横版视频重新构图，适配手机竖屏（如9:16）或其他比例。你只需导入视频，选择目标比例，软件会自动跟踪画面主体（如人物）并调整构图，然后导出或导入草稿箱进一步编辑。
*   **文字成片**：这是非常强大的AI功能。它可以根据你输入的关键词**智能生成文案**，并进一步根据文案**自动匹配素材、配音、背景音乐和字幕，生成一个完整的视频**。例如，输入产品名称和特点（如“酸奶，口感好，价格便宜”），AI就能创作出营销文案并配好视频，非常适合快速制作信息流广告或内容视频。
