# 抖音全套运营教程：P6：人物设定 🎭

在本节课中，我们将要学习抖音账号运营中一个至关重要的环节——**人物设定**（简称“人设”）。我们将探讨人设的重要性、如何系统性地策划人设，以及如何通过“超级符号”强化人设，让账号在众多内容中脱颖而出。

![](img/9569866655b98813325267753bfc5829_0.png)

## 人设策划的核心概念

![](img/9569866655b98813325267753bfc5829_2.png)

上一节我们介绍了账号定位与选题，本节中我们来看看如何为账号注入灵魂——即打造人物设定。

![](img/9569866655b98813325267753bfc5829_4.png)

在抖音中策划人设，就像在游戏中创建角色一样，需要提前对人物的各个方面进行设定。例如，角色的**称呼**、**职业**、**技能**等。

![](img/9569866655b98813325267753bfc5829_6.png)

但与游戏角色不同的是，我们在抖音中设定的是**人生角色**，因此需要更具真实性和可信度。

## 人设的重要性

![](img/9569866655b98813325267753bfc5829_8.png)

一个成功的人设IP，无论是在短视频还是直播中，都能解决一个核心问题：**建立粉丝信任**。

![](img/9569866655b98813325267753bfc5829_10.png)

它不仅能促进内容的裂变传播，还具备强大的商业变现潜力。

![](img/9569866655b98813325267753bfc5829_12.png)

因此，我们运营账号，无论是创作内容还是策划选题，其核心目的都是在获取流量的同时**打造IP**。

打造IP的本质，是在特定的场景或情境中，完成**账号人格化**的过程。

![](img/9569866655b98813325267753bfc5829_14.png)

## 如何系统策划人设

![](img/9569866655b98813325267753bfc5829_16.png)

账号人格化，即我们通常所说的“人设”。对于人设的策划，我们可以从以下几个维度系统性地着手。

![](img/9569866655b98813325267753bfc5829_18.png)

以下是策划人设的六个关键方面：

1.  **名字/昵称**
    *   一个IP首先需要一个称呼，这是IP传播的名片。它就像一个品牌的名称或商标，至关重要。

2.  **职业/职务**
    *   人格化必然需要身份。职业是用户认识这个IP的联想入口，例如老师、律师、博士等。

3.  **称呼/社会关系**
    *   一个恰当的称呼能快速拉近与用户的距离，产生情感共鸣。例如“XX哥”、“XX姐”、“XX姥姥”。

4.  **形象/第一印象**
    *   从用户视角塑造IP形象，使其更容易占据用户心智。例如大叔形象、甜美可爱、铁憨憨等。

5.  **性格/爱好**
    *   一个具有鲜明特点的IP，记忆点强且更真实。过于完美的形象反而容易脱离大众。例如暴躁、吃货、爱喝酒等。

6.  **穿着打扮/财富地位**
    *   借助社会普遍认知和刻板印象，快速在用户心中为IP定位。例如开豪车代表成功，打扮朴素可能代表老师。

## 从人设到账号搭建

以上内容策划完成后，就可以结合你自身的产品或服务优势，将其落实到**抖音账号的五件套**中：**头像、昵称、背景图、简介、视频封面**。

## 提炼人设的“超级符号”

除了系统策划，还需要通过内容和选题，为人设提炼出独特的**记忆点**，即“超级符号”。

超级符号可以从以下几个方向提炼：

*   **画风（视觉）**：独特的场景、道具、服饰。
*   **听觉**：标志性的BGM、特效音、配音声线。
*   **文案**：一句反应核心优势、易于记忆的提炼词，例如“简单粗暴”。
*   **表情/情绪**：标志性的笑容、惊恐、无奈等表情包式情绪传递。
*   **原特色**：方言、口音、独特的说话节奏（如慢条斯理、抑扬顿挫）。
*   **动作/习惯**：习惯性的肢体动作，例如撩头发、扶眼镜。

超级符号提炼好后，在进行短视频创作时，就应**持续、反复地在视频中呈现**这些元素，不断强化用户记忆。

## 课程总结

本节课中，我们一起学习了抖音账号运营的“人物设定”。我们明白了人设是账号人格化的核心，能够建立信任、促进传播与变现。我们掌握了从六个维度系统策划人设的方法，并学会了如何提炼和运用“超级符号”来强化人设记忆点。记住，一个鲜明、真实、一致的人设，是你在抖音海洋中打造个人品牌、连接目标用户的最有力武器。
