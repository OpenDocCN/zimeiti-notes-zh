# 抖音运营教程：P24：扶持加权与降权限流

在本节课中，我们将学习抖音平台对于视频内容的扶持与限制规则。了解哪些内容能获得流量扶持，哪些行为会导致账号被降权限流，是运营抖音账号的基础。

![](img/cbeb97c7769e649a809180b9dd454906_1.png)

上一节我们介绍了账号权重的分类，本节中我们来看看具体哪些内容会影响账号的权重。

![](img/cbeb97c7769e649a809180b9dd454906_3.png)

## 🎬 获得扶持加权的内容

平台会优先推荐符合以下标准的视频内容，这有助于提升账号权重和视频曝光。

![](img/cbeb97c7769e649a809180b9dd454906_5.png)

以下是获得流量扶持的视频特征：

*   **视频格式**：竖版（9:16）的移动端原生视频。这是抖音推荐的原生格式。
*   **画面质量**：画面平稳，作品清晰度高。
*   **声音质量**：无过多杂音和噪音。

这些是平台推荐的基础标准，符合这些条件的视频更容易进入推荐流量池。

## ⚠️ 导致降权限流的内容

反之，发布以下类型的内容，可能会导致账号被降权、视频被限流，甚至审核不通过。

以下是常见的违规内容类型：

1.  **不良信息**：视频内容涉及吸烟、暴力、过度宣扬金钱等不良或敏感信息。
2.  **外部引流**：在视频或文案中留下微信、微博、淘宝等外部联系方式。长时间出现商品Logo、手机号、二维码也属于违规行为。
    *   **注意**：二维码并非绝对禁止，但出现时间最好不要超过3秒。有时审核严格，即使出现1秒也可能导致视频无法通过。
3.  **硬性广告**：进行明显的营销广告、推销或引流卖货行为。
4.  **无口播的好物推荐**：这是需要特别注意的一点。如果你在制作好物推荐或拆箱类视频（例如推荐玩具、日用品等），并计划挂小黄车带货，**务必添加口播解说**。
    *   **错误示范**：视频中仅展示产品，或直接标注价格（如“9.9元”），没有任何口头介绍。这会被系统判定为纯广告，小黄车可能无法通过审核。
    *   **正确做法**：在展示产品时，配合口播进行简单介绍或使用体验分享。例如：“我儿子玩这个玩具玩了四个小时都没停。” 这样以分享的形式融入广告，更容易通过审核。
5.  **口播广告与联系方式**：在口播中直接念出广告语或联系方式。这种方式在抖音行不通，更适合传统电商平台。

本节课中我们一起学习了抖音平台对内容的扶持与限制规则。核心在于：**制作高质量的原生竖版视频能获得扶持；而发布硬广、引流信息或无解说带货视频则会导致限流**。理解并规避这些风险点，是账号健康运营的前提。
