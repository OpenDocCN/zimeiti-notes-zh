# 视频号运营教程：P26：如何制作直播间贴图 🎨

在本节课中，我们将学习如何为视频号直播间制作一个透明的文字贴图。这种贴图常用于展示商品信息或活动标语，能有效吸引观众注意并传递关键信息。

## 概述
许多直播间会使用文字贴图来突出产品信息，例如“好物推荐 老国货黄氏霜 9块91盒”。本节教程将详细讲解如何使用“微麦水印相机”App，从零开始制作一个背景透明的自定义贴图，并成功添加到你的直播间画面中。

## 工具准备与基础操作
首先，我们需要准备制作工具。请在你的手机应用商店搜索并下载“微麦水印相机”App。这款应用提供了制作文字水印的功能，我们使用其免费功能即可完成贴图制作。

以下是下载和进入核心功能区的步骤：
1.  打开手机应用商店。
2.  搜索“微麦水印相机”并下载安装。
3.  打开App，在主界面找到“水印库”选项。
4.  点击“暂不开通”跳过会员提示，然后选择“更多水印”。

## 创建与编辑贴图内容
上一节我们准备好了工具，本节中我们来看看如何创建和编辑具体的贴图内容。在“水印模板”中，我们需要找到合适的模板并输入自定义文字。

进入模板库后，请按以下步骤操作：
1.  在模板分类中找到“横竖文字”类目，并选择第一个基础模板。
2.  点击模板中的文字区域，开始编辑内容。你可以模仿其他直播间的文案，例如输入“好物推荐 老国货品牌 9块91盒 19块9三盒 29块9五盒”。
3.  编辑完成后，你可以对贴图进行美化。应用允许你更改文字颜色、字体样式。

以下是可调整的样式选项：
*   **文字颜色**：可以更改为彩色、粉色或其他任何你喜欢的颜色。
*   **字体样式**：在字体设置中，可以选择不同的字体风格。

![](img/8cf3e5d0f31146107430b6d6acbbc74b_1.png)

## 设置透明背景并保存
内容编辑完成后，最关键的一步是设置透明背景。如果背景不透明，贴图会遮挡直播画面，影响观感。

![](img/8cf3e5d0f31146107430b6d6acbbc74b_3.png)

![](img/8cf3e5d0f31146107430b6d6acbbc74b_5.png)

请严格按照以下步骤设置透明背景：
1.  找到“背景色”设置选项。
2.  在背景色选择面板中，务必选中带有斜杠标识的“无背景”或“透明”选项。这表示背景完全透明。
3.  确保背景的所有参数（如不透明度）都调整为0。
4.  确认设置无误后，点击“生成”或“保存”按钮。

![](img/8cf3e5d0f31146107430b6d6acbbc74b_7.png)

操作的关键代码逻辑可以理解为：`background-color: transparent;` 或 `opacity: 0;`。这确保了最终生成的图片只有文字，没有底色。

![](img/8cf3e5d0f31146107430b6d6acbbc74b_9.png)

## 在直播间添加贴图
贴图制作并保存到手机相册后，接下来我们将其添加到直播间。这个步骤需要在视频号的直播准备界面中完成。

![](img/8cf3e5d0f31146107430b6d6acbbc74b_11.png)

以下是添加贴图到直播间的具体流程：
1.  在视频号中点击“发起直播”。
2.  进入直播预览画面后，找到添加贴图或挂件的功能按钮（通常是一个“+”或图片图标）。
3.  点击后选择“手机相册”或“从相册选择”。
4.  从相册中选中你刚刚制作并保存好的透明背景贴图。
5.  贴图会出现在直播画面上，此时你可以用手指拖动来调整其位置，双指缩放来调整其大小。

![](img/8cf3e5d0f31146107430b6d6acbbc74b_12.png)

如果添加后觉得贴图文字大小不合适，你可以返回“微麦水印相机”App，重新编辑该贴图，调整文字大小、行距等参数，再次生成并覆盖保存即可。

![](img/8cf3e5d0f31146107430b6d6acbbc74b_14.png)

## 总结
本节课中我们一起学习了制作直播间透明贴图的完整流程。我们首先介绍了所需的工具“微麦水印相机”，然后逐步讲解了如何创建文字内容、编辑样式，并**重点强调了将背景色设置为透明**的关键步骤，最后演示了如何将制作好的贴图添加到视频号直播间。掌握这个技能，你就能轻松为自己的直播间制作个性化的提示标签了。
