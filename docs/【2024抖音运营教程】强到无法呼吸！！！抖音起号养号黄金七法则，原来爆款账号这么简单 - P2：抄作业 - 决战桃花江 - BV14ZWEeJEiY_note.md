# 抖音运营教程：P2：抄作业 🧑‍🏫

在本节课中，我们将学习一种通过“抄作业”方式参与中视频计划的实用方法。该方法操作简单，适合新手，核心在于借鉴爆款内容并进行二次创作，以实现流量与收益。

![](img/0d01898d2fa95917de387bb25bc9f3dc_1.png)

## 概述
我们将分步骤拆解“抄作业”全流程，从寻找爆款素材到最终发布视频。关键在于理解并执行，而非复杂技巧。

---

上一节我们介绍了中视频计划的潜力，本节中我们来看看如何通过“抄作业”具体操作。

### 第一步：寻找爆款选题与素材
首先，需要确定一个容易引发共鸣、具备爆款潜力的视频赛道。温情故事类内容因其强烈的情绪感染力，是理想的选择。

以下是具体操作步骤：
1.  打开抖音，在搜索框输入“温情故事”。
2.  点击顶部“视频”板块。
3.  使用筛选功能，选择“最多点赞”。
4.  页面将展示大量高点赞量的爆款温情视频，可作为参考素材。

![](img/0d01898d2fa95917de387bb25bc9f3dc_3.png)

![](img/0d01898d2fa95917de387bb25bc9f3dc_3.png)

### 第二步：提取视频文案
找到目标视频后，下一步是获取其文案内容进行再创作。

以下是具体操作步骤：
1.  进入选定的爆款视频播放页。
2.  点击右下角“分享”按钮。
3.  在弹出的菜单中找到并点击“复制链接”。
4.  将复制的链接粘贴到文案提取工具中，获取视频的原始文案。

**核心提示**：提取出的文案不可直接使用。必须通读全文，理解后用自己的语言重新讲述一遍。这是避免违规风险的关键。

![](img/0d01898d2fa95917de387bb25bc9f3dc_5.png)

### 第三步：利用工具快速成片
获得自己重新组织的文案后，可以借助剪辑工具快速生成视频，极大降低制作门槛。

操作流程如下：
1.  打开剪映APP，将处理好的文案导入。
2.  使用“一键成片”功能。
3.  工具会自动根据文案匹配画面素材并生成配音。
4.  检查并微调后，一个完整的视频即制作完成。

### 第四步：通过正确渠道发布
视频制作完成后，需通过中视频计划入口发布，才能获得播放收益。

发布步骤如下：
1.  在抖音顶部搜索“中视频计划”。
2.  进入“中视频伙伴计划”官方入口。
3.  通过该入口发布制作好的视频。
4.  发布时务必勾选“同步到西瓜视频和今日头条”。

**收益公式**：**总收益 = 抖音播放收益 + 西瓜视频播放收益 + 今日头条播放收益**

只要视频有播放量，就能在这三个平台同时产生收益。

![](img/0d01898d2fa95917de387bb25bc9f3dc_5.png)

## 方法核心总结
本节课我们一起学习了“抄作业”四步法。该方法看似简单，但核心在于巧妙借力：**爆款文案 + 爆款选题 = 高播放概率**。通过提取爆款视频的文案内核，并用自己的语言进行二次创作，既降低了创作难度，又提高了内容成为爆款的潜力。
