# 小红书运营指南：P28：小红书广告投放策略解析

在本节课中，我们将学习小红书账号的类型划分，并深入探讨两种主要的推广方式：自建账号矩阵运营与官方付费广告投放。我们将理清两者的关系与适用场景，帮助你根据自身情况做出合适的选择。

## 账号类型解析

上一节我们介绍了小红书运营的基础，本节中我们来看看小红书平台上的账号主要分为哪几类。理解账号类型是制定推广策略的第一步。

小红书账号主要分为以下三类：

1.  **企业号**：主要用于树立官方品牌形象。拥有官方认证标志，可以展示品牌名称、地址等信息，并享有平台赋予的特权，便于开展商业活动。
2.  **个人号**：这是大多数用户使用的账号类型，其特点是**灵活性高**。账号主体可以是品牌员工、产品消费者或任何个人，便于塑造个人品牌特色，适合以真人出镜等方式进行内容创作。
3.  **引流号**：这类账号在之前的课程中被称为“炮灰号”。其作用通常是辅助引流，账号本身不重要，可以随意打理或在紧急情况下作为备用小号使用，一般没有长期、精细的运营规划。

通常，我们看到的产品推广内容，大多由企业号、个人号或引流号通过发布笔记来完成。

## 企业号矩阵运营建议

了解了账号分类后，如果我们选择自建账号进行推广，如何有效利用不同账号构建运营矩阵呢？以下是打造企业号矩阵的一些具体建议。

![](img/81d8af04608fee00d2f72391f70b6ff2_1.png)

*   **积极参与品牌活动**：例如，如果你是婚纱、服装或美妆类目的企业号，可以组织试用、团购等活动，并利用多个小号积极参与这些活动，增加互动与曝光。
*   **强化个人号品牌属性**：在个人号的内容中，应多次暗示品牌的定位。例如，品牌目标用户是都市年轻人还是职场新人，需要在笔记内容中清晰体现。
*   **控制产品发布频率**：个人号发布品牌产品内容的频率不宜过高。一个较好的比例是 **营销内容 : 种草/科普内容 = 3 : 7**。即30%的内容用于直接营销，70%的内容用于分享体验或知识科普。
*   **利用引流号进行评论引流**：当企业号发布内容后，可以使用引流号在评论区进行互动或引导，以实现更隐蔽的引流效果。

## 官方付费广告（CPC）介绍

![](img/81d8af04608fee00d2f72391f70b6ff2_7.png)

除了自建账号矩阵，另一种重要的推广方式是借助小红书平台的力量。本节我们将介绍小红书官方的付费广告系统。

![](img/81d8af04608fee00d2f72391f70b6ff2_9.png)

![](img/81d8af04608fee00d2f72391f70b6ff2_11.png)

小红书官方广告主要指 **CPC（Cost Per Click）竞价广告**，即按点击付费的广告模式。其申请入口位于小红书官网的“业务合作” - “推广合作”板块。但请注意，开通广告投放功能有一个前提：**必须先认证企业号**。

![](img/81d8af04608fee00d2f72391f70b6ff2_13.png)

企业号认证流程同样在官网完成，需要提交相关资质文件并支付审核费用，通过后普通账号即可升级为官方认证的企业号。

## 官方广告类型与投放

拥有企业号后，便可以投放信息流广告。官方广告主要分为两大类：

1.  **信息流广告**：当用户浏览小红书首页推荐流时，系统会不定期间插入广告内容。
2.  **搜索广告**：当用户使用关键词进行搜索时，出现的与关键词相关的广告。其原理类似于百度的竞价排名，通过对关键词出价来竞争不同的广告展示位。

目前，信息流广告常见的展示位置主要是搜索结果页的第3位、第13位和第23位。关于广告投放的更多细节和常见问题，可以在小红书广告平台的官方文档中查阅。

![](img/81d8af04608fee00d2f72391f70b6ff2_15.png)

需要留意的是，官方广告投放通常需要一定的资金预算。因此，在资金允许的情况下，选择官方竞价广告可以获得由平台直接推荐的精准流量。
