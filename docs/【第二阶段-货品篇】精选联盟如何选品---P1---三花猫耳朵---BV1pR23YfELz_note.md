## 精选联盟如何选品 - P1

🎼 大家好，欢迎来到“从0到1成为金牌直播运营”的第二阶段货品篇。这一阶段内容至关重要。抖音直播间通常遵循“货带人”的逻辑，除非是超级大IP或达人能用个人影响力带货。对于大多数直播间而言，货品是直播间成立的前提。

在选择主播时，一个重要原则是“人货匹配”。主播的作用是为货品加分，助力货品销售。因此，货品是直播间的核心单元。

接下来，我们梳理本阶段的大纲。第一阶段，讲解精选联盟如何选品。无论您是否有自有供应链，这部分内容都值得关注。有供应链的老板可以通过精选联盟寻找爆品为直播间引流；没有供应链的老板则能借此获取货源。

第二阶段，讲解直播间货品排列应遵循的规则。第三阶段，讲解如何测款。打造爆款前需先测款，找出有潜力的产品。测款主要有两种形式：直播间测款和短视频测款。我们将梳理如何通过短视频有效测款。

后续课程还会讲解中小商家如何高效选品，以及低客单价与高客单价产品在直播间如何有效过品。同时，分析竞品对标账号也极为重要。知己知彼，百战不殆。我们需要持续关注竞品直播间，先模仿，再超越。因此，课程将讲解如何分析对标账号。

在抖音做直播必须懂得营销。最后四讲，我们将讲解四种货品的营销策略。

现在，开始第一讲：精选联盟应该如何选品。

### 开通带货权限
在通过精选联盟选品前，必须先开通带货权限。本节将手把手带您完成操作。请准备好手机跟随操作，若来不及可截图保存以便后续操作。

开通商品权限需满足以下要求：
1.  完成实名认证。一个身份证只能实名认证一个账号，若您的抖音号已认证，需使用其他身份证。
2.  缴纳500元保证金。费用不高，流程简单。
3.  发布10个以上公开作品。
4.  账号粉丝量达到1000以上。

满足要求后，您将获得以下权限：
*   拥有个人主页的商品橱窗功能。
*   支持通过精选联盟带货。
*   可在直播和短视频中直接挂载小黄车带货。

整个开通流程约需30-40分钟。

以下是具体操作流程：
1.  打开抖音，点击右下角【我】。
2.  点击右上角【≡】图标，进入【创作者服务中心】。
3.  在服务中心内找到并点击【商品橱窗】。
4.  点击【成为带货达人】。
5.  根据提示完成带货权限申请的各项要求。
6.  申请通过后，开通收款账户。

至此，权限开通流程结束。

### 精选联盟选品与上架流程
上一节我们介绍了如何开通带货权限，本节我们来看看如何在精选联盟中选品并上架到直播间。

首先，进入选品广场：
1.  在个人主页或创作者服务中心点击【商品橱窗】。
2.  在商品橱窗页面点击【选品广场】。选品广场汇集了精选联盟的所有商品。

以下是选品操作步骤：
*   若已确定经营类目，可直接进入相应类目浏览商品。
*   若未确定类目，可多浏览不同赛道，查看各品类商品的销量和佣金状况，以确定适合的类目。
*   选择心仪商品后，点击【添加到橱窗】。

此时，商品已添加至您的橱窗。接下来，需将其上架到直播间。

直播上架流程如下：
1.  开播前，点击底部【+】号，选择【开直播】。
2.  在直播准备界面，找到并点击【商品】（小黄车图标）。
3.  在商品管理页面，将橱窗内的商品添加到本次直播的小黄车中。
4.  操作完成后，界面会显示“直播间上架成功”。

此外，您还可以设置“讲解回放”功能。该功能可在直播中录制您对特定商品的讲解片段。例如，当您正在讲解1号链接，却有用户想回顾2号链接的讲解时，您可以引导用户观看该商品的讲解回放。设置路径通常在直播设置中。
