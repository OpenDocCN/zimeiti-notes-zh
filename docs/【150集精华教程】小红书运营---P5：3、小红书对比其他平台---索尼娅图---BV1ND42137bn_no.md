# 小红书运营新手起号：P5：小红书对比其他平台 📊

## 概述
在本节课中，我们将学习小红书平台与其他主流平台（如抖音）的核心区别。我们将通过对比分析，理解小红书的平台特性、内容创作逻辑以及为何它对新入局者更具机会。

---

上一节我们介绍了小红书平台的基本概况，本节中我们来看看小红书与其他平台，特别是抖音，在运营逻辑和创作要求上的具体差异。

刚才讲到了抖音的玩法。说实话，不太建议新手做抖音。以我们团队为例，去年做抖音时有200多人，年初缩减到80人左右。原因是抖音的投产比越来越低。如果不投钱，很难获得回报。抖音现在限流很严重，这是市场饱和导致的。太多人做橱窗带货，导致新号或作品质量不高、非原创的内容，一旦挂车就会被限流。

但是小红书恰恰相反。在小红书，你不挂车反而可能没流量。挂了车，平台会直接给你推送流量。这就是核心区别，因为平台需要依靠你为其创造收入。

![](img/87918b9f5549bb2a04c7ccd878d5c501_1.png)

目前了解小红书的人相对较少。今天听课的很多同学可能也不了解。在大街上随便问亲戚朋友，很多人不知道这个平台。一个平台知道的人越少，机会就越大。

相较于其他平台，做小红书更简单一些。我们可以通过素材对比来理解。

以下是抖音与小红书的内容创作对比：

*   **抖音案例**：一个销售完美日记卧蚕笔的视频。视频中模特非常漂亮，表演夸张。旁白是：“天呐这哪里是在画卧蚕，简直是在徒手捏卧蚕，一笔两笔，三笔四笔，无论多少笔都不会手重，不会画脏，再用另一头的提亮轻轻画几下，卧蚕就蓬起来了。”
*   **抖音逻辑**：短视频平台需要在两三秒内用美女吸引用户注意力。如果第一眼看到普通女孩，用户可能直接划走。但这个视频在抖音一个月只卖了500多单。原因是，消费者看到漂亮模特使用产品，会被吸引，但可能没有下单冲动。他们会认为：“你本来就好看，用完产品好看是应该的。但我这么普通，用完能有这个效果吗？”
*   **抖音成本**：要制作一个类似质量的视频，需要找模特、找代拍，单个视频成本在**800~1000元**。

接下来，我们看看小红书平台。

![](img/87918b9f5549bb2a04c7ccd878d5c501_3.png)

小红书平台的性质更像是**高端的买家秀平台**。

以下是小红书平台的优秀案例：

*   **小红书案例**：同样是销售卧蚕笔，这个笔记一个月卖了3000多单。图中的女孩长相普通。
*   **小红书逻辑**：在小红书，放特别漂亮的美女图，点击率可能还不如普通女孩的图。因为大部分用户是想变美、想改变的普通人。她们看到普通女孩的“使用前”照片，第一感觉是真实。看到“使用后”的变化，会产生兴趣，认为“我用了可能也有这个效果”。
*   **小红书创作**：只需要几张对比图片，加上一个吸引人的标题和一段真实的产品描述，就能达到很好的带货效果。公式可以概括为：**优质图片 + 好标题 + 真实描述 = 带货效果**。即使不会视频剪辑，也可以操作。

另外，小红书平台目前很缺创作者，因此竞争相对较小。

## 总结
本节课中我们一起学习了小红书与抖音等平台的核心差异。关键点在于：抖音流量饱和、投入高、对内容质量和原创性要求苛刻，且限流严重；而小红书鼓励带货、内容创作门槛较低（以真实图片和文字为主）、用户更青睐真实普通的“买家秀”，且目前竞争环境相对宽松，为新入局者提供了更多机会。理解这些差异，是做好小红书运营的第一步。
