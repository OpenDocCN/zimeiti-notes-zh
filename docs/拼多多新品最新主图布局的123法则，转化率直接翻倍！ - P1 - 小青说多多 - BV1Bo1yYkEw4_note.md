# 拼多多运营教程：P1：新品主图布局123法则 🚀

在本节课中，我们将学习拼多多平台上新品主图布局的核心法则。这套“123法则”旨在通过优化主图内容，直接提升商品的点击率与转化率。

## 概述

在拼多多平台，约90%的买家不会查看商品详情页。他们通常浏览主图、查看价格、阅读评价或行家心得，如有疑问则咨询客服，无疑问便直接下单。因此，无论详情页做得多出色，都难以大幅提升转化率。提升转化率最直接有效的方法，就是优化主图布局。

上一节我们概述了主图的重要性，本节中我们来看看具体的布局法则。

## 主图布局123法则详解

我分享的“123法则”是指将买家关心的核心卖点，系统化地布局在五张主图（特别是第二和第五张）上的方法。以下是具体的布局步骤：

**第一张主图：展示核心功能**
这张图用于吸引点击，需要清晰展示产品最主要的功能或特征。

**第二张主图：突出产品优势**
这张图用于建立初步好感。需要展示与同类产品相比，本产品的独特优势，让买家进行“优势对比”选择。

**第三张主图：呈现使用利益**
这张图用于激发购买欲望。需要展示使用产品能获得的好处，例如功效、效果或具体的使用场景。

**第四张主图：提供信任证据**
这张图用于打消买家顾虑。需要提供让买家相信你的证据，例如售后保障、问题解决方案或品质认证。

通过以上四个步骤，可以系统性地解决买家从兴趣到信任的决策过程，从而有效提高转化率。

## 总结

本节课我们一起学习了拼多多新品主图的“123布局法则”。关键在于：第一张图吸引点击，第二张图突出优势，第三张图呈现利益，第四张图提供信任证据。系统化地布局这五张主图，可以有效打消买家顾虑，显著提升商品的转化效果。
