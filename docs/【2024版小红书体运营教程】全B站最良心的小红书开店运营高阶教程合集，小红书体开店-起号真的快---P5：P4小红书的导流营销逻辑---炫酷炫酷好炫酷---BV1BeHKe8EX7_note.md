## 平台核心角色与内容

![](img/69a3f3a12740b3caee10dab6b3631031_30.png)

![](img/69a3f3a12740b3caee10dab6b3631031_32.png)

小红书平台上的用户主要分为两类：个人用户（常被称为“小红薯”）和企业用户。他们发布的内容统称为“笔记”。

![](img/69a3f3a12740b3caee10dab6b3631031_38.png)

![](img/69a3f3a12740b3caee10dab6b3631031_40.png)

![](img/69a3f3a12740b3caee10dab6b3631031_42.png)

![](img/69a3f3a12740b3caee10dab6b3631031_44.png)

**笔记** 是小红书内容的基本单位。平台鼓励用户持续创作和分享购物体验与商品使用心得，这种记录和分享的形式，因此被命名为“笔记”。

## 个人账号的导流路径

![](img/69a3f3a12740b3caee10dab6b3631031_54.png)

![](img/69a3f3a12740b3caee10dab6b3631031_56.png)

个人小红薯通过分享优质笔记，可以触发以下两种主要的流量转化路径：

![](img/69a3f3a12740b3caee10dab6b3631031_62.png)

![](img/69a3f3a12740b3caee10dab6b3631031_64.png)

![](img/69a3f3a12740b3caee10dab6b3631031_66.png)

![](img/69a3f3a12740b3caee10dab6b3631031_68.png)

以下是个人账号实现导流与变现的两种方式：

![](img/69a3f3a12740b3caee10dab6b3631031_70.png)

![](img/69a3f3a12740b3caee10dab6b3631031_72.png)

![](img/69a3f3a12740b3caee10dab6b3631031_74.png)

![](img/69a3f3a12740b3caee10dab6b3631031_76.png)

1.  **主动分享，引导消费**：用户购买或体验某商品后，若感觉满意，可撰写笔记分享其优点（如：显白、面料舒适）。其他用户看到笔记后产生兴趣，会主动搜索该商品或店铺，从而为商家带来流量与潜在销售。其路径可概括为：
    `个人笔记 -> 吸引兴趣用户 -> 引导至商品/店铺 -> 产生销售`

![](img/69a3f3a12740b3caee10dab6b3631031_78.png)

![](img/69a3f3a12740b3caee10dab6b3631031_80.png)

![](img/69a3f3a12740b3caee10dab6b3631031_82.png)

![](img/69a3f3a12740b3caee10dab6b3631031_84.png)

2.  **优质创作，吸引合作**：当个人账号在某一垂直领域（如美妆、穿搭）持续产出高质量笔记，积累起影响力后，可能会被相关领域的商家注意到。商家会主动寻求合作，进行产品推广，从而使个人创作者获得**变现**机会。

![](img/69a3f3a12740b3caee10dab6b3631031_88.png)

## 企业账号的导流路径

![](img/69a3f3a12740b3caee10dab6b3631031_94.png)

![](img/69a3f3a12740b3caee10dab6b3631031_96.png)

![](img/69a3f3a12740b3caee10dab6b3631031_98.png)

![](img/69a3f3a12740b3caee10dab6b3631031_100.png)

企业类小红书账号的核心目标是直接促进自身商品的销售。

![](img/69a3f3a12740b3caee10dab6b3631031_102.png)

![](img/69a3f3a12740b3caee10dab6b3631031_104.png)

![](img/69a3f3a12740b3caee10dab6b3631031_106.png)

![](img/69a3f3a12740b3caee10dab6b3631031_108.png)

企业通过官方账号发布笔记，主动向目标消费者展示产品的利好信息与核心价值点，激发用户的兴趣与购买欲望，从而将平台公域流量转化为进入自家店铺或商城的私域流量，最终达成销售。其核心路径为：
`企业笔记（释放产品价值）-> 吸引目标消费者 -> 导流至企业店铺/商城 -> 产生销售`

![](img/69a3f3a12740b3caee10dab6b3631031_110.png)

除了上述带有明确商业目的的账号，平台上也存在纯粹用于记录生活、分享心得的个人账号，或将其当作自媒体经营的账号。

![](img/69a3f3a12740b3caee10dab6b3631031_124.png)

![](img/69a3f3a12740b3caee10dab6b3631031_126.png)

![](img/69a3f3a12740b3caee10dab6b3631031_128.png)

然而，从运营和商业角度出发，小红书平台最根本的价值在于：**无论是个人还是企业，都能通过创作优质内容获取流量，并将流量进行合理转化，最终实现变现与营收**。这才是我们学习和运用小红书导流逻辑的终极目标。

![](img/69a3f3a12740b3caee10dab6b3631031_132.png)

![](img/69a3f3a12740b3caee10dab6b3631031_134.png)

![](img/69a3f3a12740b3caee10dab6b3631031_136.png)

![](img/69a3f3a12740b3caee10dab6b3631031_138.png)

本节课中我们一起学习了小红书导流营销的核心逻辑。我们明确了“笔记”是内容载体，并梳理了个人账号与企业账号不同的流量转化路径。理解“**内容创造流量，流量实现转化**”这一根本原则，是后续所有运营动作的基石。
