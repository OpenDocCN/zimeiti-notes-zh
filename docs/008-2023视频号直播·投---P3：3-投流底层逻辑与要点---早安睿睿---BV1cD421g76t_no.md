## 投流的商业底层逻辑

![](img/6522aa279d5719bd75634edcc732b1bd_21.png)

上一节我们介绍了课程背景，本节中我们来看看投流在整个商业闭环中的位置。

![](img/6522aa279d5719bd75634edcc732b1bd_23.png)

任何商业项目都遵循一个基本流程：**外部引流拓客 -> 建立信任 -> 刺激销售 -> 达成成交 -> 客户留存与复购 -> 裂变带来新流量**。

![](img/6522aa279d5719bd75634edcc732b1bd_25.png)

![](img/6522aa279d5719bd75634edcc732b1bd_27.png)

![](img/6522aa279d5719bd75634edcc732b1bd_29.png)

![](img/6522aa279d5719bd75634edcc732b1bd_31.png)

![](img/6522aa279d5719bd75634edcc732b1bd_33.png)

直播间本质上也是这个逻辑：
1.  从公域获取流量（自然流/付费流）。
2.  在直播间通过互动、停留建立初步联系。
3.  促成成交转化。
4.  通过关注、加粉丝团实现留存。
5.  系统召回或用户主动复购，形成稳定的人群模型和推流。

这一切的基础都围绕着 **人、货、场** 来打造高效的闭环盈利模型。投流是“锦上添花”，而非“雪中送炭”，它建立在稳定的人货场基础之上。

![](img/6522aa279d5719bd75634edcc732b1bd_35.png)

## 直播间人群学习的底层逻辑

![](img/6522aa279d5719bd75634edcc732b1bd_37.png)

![](img/6522aa279d5719bd75634edcc732b1bd_39.png)

理解了商业闭环，我们再来看看系统是如何“认识”你的直播间的。这是投流策略制定的核心依据。

![](img/6522aa279d5719bd75634edcc732b1bd_41.png)

![](img/6522aa279d5719bd75634edcc732b1bd_43.png)

直播间的系统推流机制基于对用户**行为交集**的学习。系统会追踪进入直播间的用户产生的多种行为数据：

*   **行为数据**：停留、互动、转粉、加粉丝团。
*   **电商数据**：商品曝光、商品点击、商品成交。

系统会寻找同时产生了多种关键行为的用户。例如，一个用户既**停留**了较长时间，又参与了**互动**，还**点击**了商品，最终**成交**。这部分用户的行为**交集**，就定义了你直播间“最精准的经营人群”。

**公式表示**：
`最精准人群 = 停留 ∩ 互动 ∩ 商品点击 ∩ 成交` （这是一个行为交集的概念）

系统会持续寻找与这部分“交集人群”画像相似的新用户，并推荐到你的直播间，这就是“人群螺旋学习”的底层框架。

![](img/6522aa279d5719bd75634edcc732b1bd_45.png)

![](img/6522aa279d5719bd75634edcc732b1bd_47.png)

## 投流目标的本质：是学习，不是创造

![](img/6522aa279d5719bd75634edcc732b1bd_49.png)

基于上述逻辑，我们必须澄清一个关键误区：**投流的本质是加速系统学习已有行为的人群，而不是凭空创造该行为。**

![](img/6522aa279d5719bd75634edcc732b1bd_51.png)

![](img/6522aa279d5719bd75634edcc732b1bd_53.png)

![](img/6522aa279d5719bd75634edcc732b1bd_55.png)

![](img/6522aa279d5719bd75634edcc732b1bd_57.png)

例如：
*   **投“互动”目标**：是让系统去学习那些**已经在你的直播间产生了互动行为**的用户画像，然后找来更多类似的人。而不是你投了“互动”，机器就能强迫进来的人互动。
*   **投“成交”目标**：是学习**已经成交的用户**画像。

机器无法影响人的主观行为。用户是否互动、是否成交，完全取决于你直播间的**人货场**吸引力。因此，投流是在优质人货场的基础上，付费让系统更快、更准地找到你的目标客户。

## 直投直播间的付费逻辑

![](img/6522aa279d5719bd75634edcc732b1bd_65.png)

![](img/6522aa279d5719bd75634edcc732b1bd_67.png)

现在，我们来看看如何将底层逻辑应用于实战。首先从最直接的“直投直播间”开始。

![](img/6522aa279d5719bd75634edcc732b1bd_71.png)

![](img/6522aa279d5719bd75634edcc732b1bd_73.png)

![](img/6522aa279d5719bd75634edcc732b1bd_75.png)

![](img/6522aa279d5719bd75634edcc732b1bd_77.png)

直投直播间的核心逻辑是：**付费加速系统寻找更多的“行为交集人群”**。

![](img/6522aa279d5719bd75634edcc732b1bd_79.png)

![](img/6522aa279d5719bd75634edcc732b1bd_81.png)

![](img/6522aa279d5719bd75634edcc732b1bd_83.png)

不同的投放目标，对应学习不同行为标签的人群：
*   **投观众/停留**：学习长时间停留的用户。
*   **投互动**：学习参与互动的用户。
*   **投转粉**：学习关注账号的用户。
*   **投商品点击**：学习点击商品链接的用户。
*   **投成交**：学习最终下单成交的用户。

**选择策略**：
*   **前期**：优先选择数据量充足、易于学习的目标，如**商品点击**、**互动**。因为“成交”数据量可能太小，系统难以学习。
*   **后期/成熟期**：当直播间模型稳定、成交数据充足后，可以重点投**成交**目标，追求最高精度。
*   **特殊场景**：对于憋单时间长、停留数据突出的直播间，投**观众/停留**目标可能效果更好。

![](img/6522aa279d5719bd75634edcc732b1bd_85.png)

**重要认知**：
1.  **扣费方式**：付费流量按**展现（曝光）扣费**，而非按达成目标（如成交几单）扣费。
2.  **学习过程**：计划初期主要学习“目标行为人群”，后期会与直播间的实际“行为交集”结果融合，越跑越准。
3.  **计划跑不动或跑飞**：
    *   **消耗不动**：可能人群圈定过窄或出价过低，竞争不过同行。需逐步抬高出价。
    *   **计划跑飞（穿透率骤升）**：直播间长期无法从投来的人群中学习出有效转化，系统放弃精准探索，导入大量泛流量试图重新学习。

## 短视频投放的增效逻辑

除了直投，短视频是另一个重要的流量入口。它的优势在于改变了用户的决策路径。

![](img/6522aa279d5719bd75634edcc732b1bd_93.png)

![](img/6522aa279d5719bd75634edcc732b1bd_95.png)

![](img/6522aa279d5719bd75634edcc732b1bd_96.png)

![](img/6522aa279d5719bd75634edcc732b1bd_98.png)

用户的经典消费决策链为：**诱发需求 -> 收集信息 -> 选择评估 -> 下单成交 -> 售后评价**。

![](img/6522aa279d5719bd75634edcc732b1bd_100.png)

*   **直投直播间**：用户进入后，你需要直播间内完成“诱发需求、收集信息、选择评估”等多个步骤，才能推动成交。
*   **短视频引流**：优质的短视频可以在用户进入直播间**前**，就完成“诱发需求”和“收集信息”甚至部分“选择评估”的工作。用户进入直播间的目的性更强，**决策成本更低**，因此转化率通常更高。

**短视频制作与投放要点**：
以下是高效利用短视频引流的几个关键动作：
*   **内容方向**：不必执着原创，可模仿对标同行爆款视频。内容必须与售卖商品**垂直相关**，完成消费决策链的前置步骤。
*   **创作方法**：音频可直接借鉴，画面可重新拍摄。以数量对抗概率，找到数据好的视频模板就反复拍。
*   **数据标准**：完播率25%-30%以上、时长不低于10秒的视频，可视为有潜力的“好视频”。
*   **发布与互动**：开播前1-2小时发布。短视频用户进入直播间后，应多引导互动，形成“短视频内容”与“直播间氛围”的双频共振，放大引流效果。

![](img/6522aa279d5719bd75634edcc732b1bd_102.png)

## 投流实战要点精讲

掌握了底层逻辑和两种投放方式后，我们汇总一些实战中的核心要点。

![](img/6522aa279d5719bd75634edcc732b1bd_104.png)

### 1. 投流与自然流的关系
付费流量与自然流量共享**行为交集学习**机制。因此：
*   **自然流承接好**，付费流更容易学习精准，形成正向循环。
*   **自然流模型偏**，付费流也会难以跑正。此时可通过拉时长、高出价“矫正”计划等方式尝试拉回。
*   **马太效应**：数据好的场次应加大投放，让好的更好；数据差的场次需谨慎调整。

### 2. 出价策略与调整
出价是竞价逻辑，代表抢流量的意愿。
*   **初期/测计划**：可按系统建议的**最低出价**起投。
*   **竞争调整**：计划消耗慢时，可**阶梯式小幅上调出价**。
*   **智能出价**：仅建议在投**观众/停留**目标时使用“放量加热”。其他目标慎用。
*   **出价高低的核心**：最终取决于你直播间**人货场的承接转化能力**。转化率越高，低出价越可能跑出好数据。

### 3. 投放时长与流速控制
投放时长影响流量速率。
*   **测计划/求稳**：建议用**12或24小时**的长时长计划，让流量缓慢进入，便于观察和模型学习，避免泛流量冲击。
*   **控流速**：日常主要通过**调整投放时长**来控制流速快慢。调预算也可能影响流速，但可能导致人群变泛。

![](img/6522aa279d5719bd75634edcc732b1bd_106.png)

![](img/6522aa279d5719bd75634edcc732b1bd_108.png)

### 4. 人群定向：对标与生态链
*   **投自己（老粉回流）**：强化直播间人群模型稳定性，适用于有复购的直播间。
*   **投同行（抢客户）**：核心思想是“你比对方强，就去抢”。要投**开过直播**的对标账号。
*   **投人群（生态链思维）**：客户消费不是孤立的。思考你产品的**前置需求**和**关联需求**。例如，卖卫浴的可以投“装修设计”、“瓷砖”等相关账号的人群。这能拓展精准人群池。

![](img/6522aa279d5719bd75634edcc732b1bd_110.png)

![](img/6522aa279d5719bd75634edcc732b1bd_112.png)

![](img/6522aa279d5719bd75634edcc732b1bd_114.png)

### 5. 效果评估与成本核算
*   **核心看漏斗**：评估计划精准度，主要看**曝光-点击-成交**这个漏斗的转化率。
*   **ROI计算要全面**：
    *   **看间接成交**：关注“7天下单金额”，很多用户是多次触达后才下单。
    *   **看自然流放大**：付费流会撬动额外的自然流，这部分价值应计入总收益。
    *   **算综合成本**：将**投流占比 + 退货退款率 + 其他运营成本**一并计算，确保最终利润率健康。
*   **关于赔付**：精细化运营中不刻意“卡赔付”（如投互动却不拉互动）。引导用户完成目标行为，有助于直播间整体学习，长期价值远大于赔付优惠。

本节课我们一起学习了视频号投流的底层逻辑与核心要点：
1.  **底层逻辑**：投流是基于直播间**用户行为交集学习**的付费加速器，本质是“学习”而非“创造”。
2.  **两大入口**：**直投直播间**加速寻找精准人群；**短视频引流**通过前置消费决策步骤，提升转化效率。
3.  **核心关系**：付费流与自然流相互影响，**强大的人货场是投流成功的基石**。
4.  **实战要点**：包括出价策略、时长控制、人群定向的生态链思维，以及全面评估ROI的方法。
