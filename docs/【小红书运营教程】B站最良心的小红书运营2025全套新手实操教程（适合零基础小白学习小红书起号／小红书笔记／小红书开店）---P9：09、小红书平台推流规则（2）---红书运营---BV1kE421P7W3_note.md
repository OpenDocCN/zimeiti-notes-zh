# 小红书运营教程：P9：09、小红书平台推流规则（2）🔍

![](img/7eb62dd6e04a434daf5e0b037a27b485_1.png)

## 概述
在本节课中，我们将深入学习小红书平台流量规则的另一个核心部分——搜索流量。我们将详细拆解搜索流量的构成、排名机制以及如何通过优化笔记内容来提升在搜索结果中的曝光机会。

![](img/7eb62dd6e04a434daf5e0b037a27b485_3.png)

---

上一节我们介绍了小红书“发现页”的流量逻辑，本节中我们来看看平台另一个重要的流量入口：搜索流量。

![](img/7eb62dd6e04a434daf5e0b037a27b485_1.png)

搜索流量是指用户通过输入关键词进行搜索后，系统根据关键词匹配和排序规则，将相关笔记展示给用户的流量。这与发现页由系统随机推荐的流量有本质区别。

![](img/7eb62dd6e04a434daf5e0b037a27b485_3.png)

### 搜索流量的构成
搜索流量主要由两部分构成：

1.  **站内排名流量（约65%）**：这部分流量取决于笔记在特定关键词搜索结果中的排名。排名越高，获得的曝光机会越大。
2.  **系统推荐流量（约35%）**：这是平台分配给新发布笔记的固定流量。只要笔记内容与用户搜索的关键词具有相关性，就有机会获得这部分初始曝光。

因此，运营小红书时需明白，搜索流量不仅来自激烈的排名竞争，新笔记也有固定的机会获得初始曝光。

### 搜索流量的运行机制
平台默认的搜索流量分配规则是：65%的流量分配给排名靠前的笔记，35%的流量用于推荐新笔记。

这意味着，即使一个新账号什么都不做，只要笔记内容匹配了用户的搜索关键词，就有35%的概率出现在搜索结果中，获得初始曝光。

然而，绝大部分（65%）的流量分配，依赖于账号权重和关键词权重排名。点击率、完播率等数据表现优异的笔记会优先展示。

**热门搜索**榜单中的流量，几乎全部属于站内排名流量，与新笔记关系不大，主要由排名极高的头部笔记占据。

所以，发布新笔记时不要认为搜索流量与自己无关。虽然竞争激烈，但仍有35%的系统推荐机会。笔记的初始表现（点击率、完播率、互动率）将决定是否能进入那65%的排名流量池，获得更多曝光。

### 如何获取搜索流量
以下是获取搜索流量的几个关键点：

*   **精准匹配关键词**：在笔记中补充准确、相关的联想关键词，有助于系统识别并匹配搜索意图。
*   **布局大流量词汇**：即使竞争激烈，布局热门大词也有35%的概率获得系统推荐的初始流量。
*   **关注热门趋势词**：不仅要搜大词，还要关注与自身产品/内容属性相关的近期热门关键词。盲目追逐与内容无关的热词是无效的。
*   **提升笔记权重**：权重越高，排名越靠前。排名直接影响在65%流量池中的曝光份额。排名第一可能占据该流量池50%的展示机会，而排名靠后可能只有1%-5%。

### 影响搜索排名的核心因素
搜索排名主要依据笔记的互动数据和质量指标：

*   **核心公式**：**排名权重 ≈ 点赞数 + 收藏数 + 评论数 + 完播率 + 转发率**
*   单篇笔记的互动数据（点赞、收藏、评论）越多，在该领域的权重越高，排名越靠前。
*   完播率、转发率等深度互动指标越高，笔记权重也越高。
*   这种权重是流动且具有累积效应的。如果账号产生了爆款笔记，系统会默认提升该账号后续发布笔记的基础权重。

**简单来说**：爆款笔记能带动账号整体权重。拥有爆款笔记后，日常笔记的基础流量也会比之前提升约三分之一。爆款笔记越多，后续笔记获得的初始曝光和观看量就越大，形成良性循环。反之，如果笔记数据始终无法提升，账号可能长期在低流量区间徘徊。

### 触发平台额外推流的核心逻辑
当笔记数据达到一定标准后，平台系统会将其推送给更多用户。这个推流逻辑主要看两个指标：

![](img/7eb62dd6e04a434daf5e0b037a27b485_5.png)

1.  **打开率（点击率）**：在搜索结果中的平均打开率需**高于11%**。
    *   **定义**：打开率 = （笔记点击量 / 笔记曝光量） * 100%
    *   **示例**：如果笔记获得了2000次基础曝光，那么至少需要220次点击（小眼睛数量）才算达标。

![](img/7eb62dd6e04a434daf5e0b037a27b485_5.png)

2.  **互动率**：平均互动率需达到**3%** 以上。
    *   **定义**：互动率 = （（点赞数+收藏数+评论数） / 笔记点击量） * 100%
    *   **示例**：如果有100个人点击了笔记，那么需要至少有3条以上的互动（点赞、收藏或评论）。

**两项指标必须同时达标**，平台才会进行额外推流。仅互动率高但打开率不达标，或仅打开率高但互动率不达标，都无法触发系统的推广机制。

---

## 总结
本节课我们一起深入学习了小红书搜索流量的核心规则：
1.  搜索流量由**65%的排名流量**和**35%的系统推荐流量**构成。
2.  新笔记通过**精准匹配关键词**有机会获得初始曝光。
3.  搜索排名取决于笔记的**权重**，而权重由点赞、收藏、评论、完播率等数据综合决定。
4.  爆款笔记能提升**账号整体权重**，形成良性循环。
5.  触发平台额外推流需要同时满足**打开率 > 11%** 和 **互动率 > 3%** 两个核心条件。

理解并应用这些规则，是提升小红书笔记搜索曝光和获取流量的关键。
