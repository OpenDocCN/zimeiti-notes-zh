# 微信视频号教程：P6：视频号保证金、带货权限与直播权限解读 🎬

在本节课中，我们将详细解读视频号的带货身份、各类带货权限（视频带货、橱窗带货、直播带货）的具体含义与开通条件，以及直播权限的最新规则。这些是开始视频号电商运营前必须了解的基础知识。

## 带货身份与经营资质

上一节我们介绍了视频号的入口，本节我们来看看如何查看和设置你的带货身份。

在创作者中心板块，点击右上角的个人头像，即可查看你的带货身份。身份主要分为两种：**达人**和**商家**。你可以根据自身情况选择，这个身份后期也可以修改，无需担心选错。

经营资质部分通常无需特别关注。

## 带货权限详解

了解身份后，我们来看看具体的带货权限。以下是三种主要的带货方式及其区别。

### 1. 视频带货
视频带货指的是在发布的视频中挂载购物车，类似于抖音的“小黄车”功能。用户观看视频时，可以看到一个橙色的购物车图标，点击即可跳转购买。

**开通条件**：
*   视频必须为**原创**。
*   账号需拥有**1000个有效粉丝**。

对于进行内容搬运的账号，通常很难开通此权限。即便开通，一旦出现违规，此权限也可能被永久关闭。不过，权限关闭后，仍可通过橱窗进行卖货。

![](img/fefdd395a295466efa4ed456538956d7_1.png)

### 2. 橱窗带货
橱窗带货是直接在个人主页展示商品橱窗，供粉丝浏览和购买。这是一种基础且稳定的带货方式，对粉丝数通常没有硬性要求（具体以平台最新规则为准），是新手起步的常用选择。

![](img/fefdd395a295466efa4ed456538956d7_3.png)

![](img/fefdd395a295466efa4ed456538956d7_4.png)

### 3. 直播带货
直播带货是在直播过程中销售商品。这是目前转化效率较高的方式，但开通条件也更为严格。

![](img/fefdd395a295466efa4ed456538956d7_5.png)

![](img/fefdd395a295466efa4ed456538956d7_7.png)

**开通条件**：
*   **身份实名限制**：根据当前规则，一个人虽然可以实名5个微信，但只能为其**中2个视频号**进行实名认证。
*   **粉丝数要求**：需要在直播间挂载商品，必须满足**1000个有效粉丝**的要求。未满足此条件，直播间无法添加商品，但依然可以开启直播（只是不能卖货）。

![](img/fefdd395a295466efa4ed456538956d7_9.png)

如果无法满足1000粉丝的要求，另一个解决方案是开通**视频号小店**。开通小店后，即使粉丝数不足，也可以在直播间挂载商品。开通小店需要营业执照，具体流程我们将在后续课程中详细讲解。

![](img/fefdd395a295466efa4ed456538956d7_11.png)

## 直播权限实操与验证

![](img/fefdd395a295466efa4ed456538956d7_13.png)

为了让大家更清晰地理解，我们来实际验证一下直播权限。

![](img/fefdd395a295466efa4ed456538956d7_15.png)

对于一个已满足粉丝条件的账号，发起直播后，可以顺利点击“商品”按钮添加购物车，然后开始直播。

![](img/fefdd395a295466efa4ed456538956d7_17.png)

对于一个未满足粉丝条件的账号，发起直播后，点击“商品”按钮时会收到系统提示：“**视频号有效关注1000人才可以进行直播带货**”。这意味着该直播间无法添加任何商品。

## 总结

本节课我们一起学习了视频号电商功能的核心权限。
*   **带货身份**分为达人和商家，可后期修改。
*   **视频带货**门槛较高，需千粉和原创，不稳定。
*   **橱窗带货**是基础且稳定的带货方式。
*   **直播带货**效率高，但受实名认证数量（目前限2个）和粉丝数（需千粉）限制。若粉丝数不足，可通过开通**视频号小店**来解锁直播卖货权限。

![](img/fefdd395a295466efa4ed456538956d7_19.png)

理解这些规则，能帮助你更好地规划视频号的运营和变现路径。下节课，我们将开始讲解如何具体操作。
