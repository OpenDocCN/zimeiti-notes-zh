# 抖音运营：P19：平台思维：理解算法，持续创作

![](img/cb89f66f2e25718317d9e7feb12091b7_1.png)

在本节课中，我们将学习一个对短视频创作者至关重要的思维方式——**平台思维**。掌握这种思维，能帮助你理解抖音等平台的底层逻辑，从而制定更有效的运营策略，实现账号的持续成长。

很多人做短视频难以成功，是因为缺乏一种关键的思维方式。这种思维方式就是**平台思维**。一旦你拥有了这种意识，你做任何平台相关的事情，失败的概率都会大大降低。

![](img/cb89f66f2e25718317d9e7feb12091b7_3.png)

想象一下，如果你是平台方，你的目标是什么？你的目标是让创作者源源不断地为你生产内容。你可能会想，那当然是要给创作者收益，有钱大家才愿意干活。这个想法没错。

但更准确地说，平台需要建立一套**激励机制**。我们可以把平台想象成一个牌桌，我们创作的作品就是我们手中的牌。每次发布作品，都像是摸到一张新牌。虽然之前拿到的牌可能不好，但好在每次都有拿到新牌的机会。这就是我们要理解的第一个核心要点：**不确定性**。

**不确定性**可以极大地激发一个人的热情，刺激他心中“好赌”的欲望，促使他持续不断地创作下去。如果平台规定，每天只要发作品就固定给10块钱，流量平均分配，那么你将看不到优质作品，所有人都会变得麻木。

![](img/cb89f66f2e25718317d9e7feb12091b7_5.png)

尽管这样每天都能“捡到钱”。

但平台很快就会因为内容同质化和缺乏活力而消亡。反过来，如果平台提供**不确定的奖励机制**，只要创作者发布了一个关键作品，就可能流量爆炸、收益倍增。这种玩法非常刺激，会让你舍不得离开这个“牌桌”，因为你永远不知道下一个作品会不会爆火。

每当你快要放弃的时候，平台算法可能突然给你的某个作品推一波流量。大家的内容和质量可能差不多，但就是给你的流量更多，让你瞬间“满血复活”，重燃创作热情。当你不会离开牌桌时，平台又会用同样的方法去刺激其他创作者。

这样，每个创作者都会拼尽全力去创作。你会认为是自己的努力战胜了别人，是自己的内容更优质。其实不然，第一名和第十一名的内容差距可能并没有那么大，因为“好坏”本身并没有绝对明确的界定标准。

![](img/cb89f66f2e25718317d9e7feb12091b7_7.png)

平台可以让你火，也可以让他火。这才是平台真正的核心规则：**用最小的代价，激发创作者最大的创作动力**。

![](img/cb89f66f2e25718317d9e7feb12091b7_9.png)

明白了这个道理，我们再来看，想要做好短视频应该怎么办？答案是：**持续更新**。

因为平台必须给你奖励（流量），否则你就会流失，这是平台不愿意看到的。所以，当你更新得越多，你离下一次获得算法奖励就越近。即使是同样质量的内容，平台也可能会优先给你流量，让你保持活跃。

![](img/cb89f66f2e25718317d9e7feb12091b7_11.png)

就视频表现力而言，我一个用简陋PPT做的视频都有人看，你难道还不如我吗？可为什么还有这么多人放弃呢？因为他们目光太短浅，功利心太重。功利心太重的人生往往是一场灾难。如果你无法克服自身的惰性，甚至没有足够的耐心等到视频结尾（意指等到爆款出现），那么你做任何事情都难以成功，他们只是刚好选择了做短视频而已。

盲目追逐利益、内心浮躁消耗功力。你越是急于获利，越是不想吃亏，最后反而会失去更多机会。因此，要向有结果的人学习。

**本节课总结**：
本节课我们一起学习了**平台思维**的核心。我们了解到，平台通过**不确定性**和**激励机制**来驱动创作者持续生产内容。作为创作者，最关键的行动就是**持续更新**，保持耐心，克服短视和功利心，这样才能在平台的规则下，最大化获得流量奖励和成功的机会。记住，平台就像一个牌桌，坚持留在桌上，你才有摸到“好牌”（爆款）的可能。
