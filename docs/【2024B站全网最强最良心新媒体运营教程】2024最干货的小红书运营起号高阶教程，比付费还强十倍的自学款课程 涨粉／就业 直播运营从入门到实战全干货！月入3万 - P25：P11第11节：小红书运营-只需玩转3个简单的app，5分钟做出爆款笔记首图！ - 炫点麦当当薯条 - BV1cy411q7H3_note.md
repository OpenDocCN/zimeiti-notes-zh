## 拼图功能详解
上一节我们介绍了核心工具，本节中我们来看看如何使用美图秀秀的拼图功能。

拼图功能适用于需要展示多张图片的笔记场景，例如电影推荐、旅游分享、穿搭好物或美食合集。

以下是使用美图秀秀拼图的步骤与建议：
*   **操作路径**：打开美图秀秀 -> 选择“拼图”功能 -> 导入多张图片。
*   **模板选择**：**仅使用“模板”选项**，避免使用“海报”、“自由”或“拼接”模式。依靠模板的自动布局和特效，可以快速生成符合小红书风格的封面。
*   **适用场景**：以下是四种适合采用拼图形式的封面类型：
    1.  **同类产品不同款式**：例如，展示同一品类（如帽子）的不同设计（遮阳帽、防水帽等）。
    2.  **效果对比图**：例如，使用美白产品在不同阶段的效果对比。
    3.  **产品图+效果图**：展示产品本身与使用后的效果对比。
    4.  **影视剧推荐**：拼接剧中的经典场景或对话截图。

## 图片加文字技巧
学会了拼图，接下来我们学习如何为图片添加清晰有力的文字。

图片中的文字应尽量放大、突出，确保用户一眼就能抓住笔记核心。文字颜色需与背景形成鲜明对比（例如，浅色背景用深色字，深色背景用浅色字），以增强视觉吸引力。

以下是图片加文字的排版要点：
*   **文字精炼**：封面图上通常只放核心标题。
*   **对比强烈**：确保文字颜色与背景色有足够反差。
*   **排版清晰**：避免使用过度倾斜或带有复杂特效的字体，保证易读性。
*   **背景干净**：选择简洁、干净的背景，更利于突出文字和产品。
*   **字体统一**：单张图片使用的字体样式最好不超过三种，并以黑白等基础色为主，显得专业。
*   **年轻化风格**：针对大学生、年轻白领等群体，可以尝试使用“文字+气泡+色块”的组合，让封面更活泼。

> **提示**：以上所有图文排版功能，均可使用美图秀秀完成。

## 抠图功能应用
除了图文排版，抠图也是制作专业封面的重要技能。本节我们来看看天天P图的使用。

抠图功能常用于需要将产品主体从原背景中分离出来的场景，以便进行自由组合和排版。

以下是三类适合使用抠图的笔记场景：
1.  **穿搭测评**：将整套穿搭（上衣、裤子、包等）的实物图抠出，并列排版。
2.  **干货科普**：例如，将不同种类的美容仪产品图抠出，并标注名称和特点。
3.  **横向测评**：将同类型多个产品（如所有热门美容仪）抠图后集中展示在一张图上。

![](img/7260c61b4832b451f6e33e9252d68c9f_3.png)

### 抠图注意事项
并非所有图片都适合抠图，请遵循以下原则：
1.  **主体明确**：适合抠取细节清晰的小物件（如化妆品、首饰），而非汽车、房屋等大场景。
2.  **背景干净**：主体与背景颜色反差越大，抠图越容易、越精准。纯色背景（白、绿、蓝等）是最佳选择。
3.  **次数限制**：该App有每日免费抠图次数限制。用完后，可通过卸载重装App来重置（但需注意数据丢失风险）。

### 抠图操作步骤
以下是使用天天P图进行抠图的简易流程：
1.  打开天天P图，选择“魔法抠图”功能。
2.  从图库中选择一个喜欢的背景模板。
3.  导入你想要抠图的照片。
4.  使用画笔涂抹需要保留的主体部分，软件会自动识别边缘。
5.  稍作调整，确认后即可得到抠好的图片，保存使用。
