# 剪映教程：P8：时间线详解 🎬

![](img/0b38230c1b45792bf68116833bda9f4a_0.png)

在本节课中，我们将要学习剪映电脑版中最重要的面板之一——时间线。我们将了解如何导入和管理素材、认识时间线的各个功能按钮，并掌握基本的剪辑操作。

上一节我们介绍了素材库、特效、贴纸和滤镜面板。本节中，我们来看看剪辑工作的核心——时间线面板。

## 时间线基础

时间线是剪辑视频最重要的面板。如果不使用时间线，即使将素材导入剪映，也无法进行剪辑。

首先，导入一个素材到素材库。通过拖拽的方式，可以将素材直接拖到时间线上。视频素材会出现在时间线上，你可以通过拉动滑块或按**空格键**来预览和播放视频。

## 轨道逻辑

初始状态下，时间线只有一个主轨道。如果需要导入多个素材，可以将新素材拖到主轨道素材的后面，或者直接放在空白区域。

*   **放在主轨道后面**：素材会接在主轨道现有素材之后。
*   **放在空白区域**：时间线会自动创建一个新的副轨道来放置素材。

副轨道上的内容会覆盖主轨道上的内容，这是一种叠加关系。如果想调整显示顺序，只需将素材拖到空白区域，即可生成新的轨道来改变层级。

## 音频轨道

音频素材的导入方式与视频相同。将音频拖入素材箱，再拖到时间线上。音频轨道会自动创建在视频轨道的下方。导入多条音频轨道的方法相同，放在空白处即可创建新轨道。

## 轨道控制按钮

以下是轨道左侧控制按钮的功能介绍：

*   **轨道标识**：视频轨道图标是带播放键的正方形（类似TV），音频轨道图标是带音乐符号的圆形。
*   **锁定轨道**：图标像一把锁。锁定后，无法移动该轨道上的任何素材。常用于保护已确定位置的主轨道内容，防止误操作。
*   **隐藏轨道**：眼睛图标。点击会隐藏整个轨道上的所有素材，用于查看被上层素材覆盖的画面。**注意**：隐藏的是整个轨道，而非单个素材。
*   **单独隐藏素材**：如需仅隐藏某个素材，可右键点击该素材，选择 **`停用片段`**。其快捷键是 **`Shift + 1`**。
*   **关闭原声**：对于未分离音视频的素材，点击此按钮可关闭视频自带的音频。另一种方法是右键素材，选择 **`分离音频`**，然后单独删除音频轨道。
*   **编辑封面**：可以为视频素材指定在时间线上显示的缩略图。点击 **`编辑封面`**，从视频中选择一帧画面，并可添加文字，完成后点击 **`完成设置`**。

## 时间线工具栏

![](img/0b38230c1b45792bf68116833bda9f4a_10.png)

认识完轨道基础，我们来看看时间线上方的工具栏。

### 基础编辑工具

![](img/0b38230c1b45792bf68116833bda9f4a_12.png)

*   **选择工具**：默认工具，用于选择和拖动素材。
*   **撤销/恢复**：**`Ctrl + Z`** 撤销上一步操作，**`Ctrl + Shift + Z`** 恢复被撤销的操作。也可点击工具栏按钮。
*   **分割工具**：用于切割素材。
    *   **方式一**：将滑块移到切割点，点击 **`分割`** 按钮或按快捷键 **`Ctrl + B`**。
    *   **方式二**：选择“分割”工具（图标像剃刀），鼠标会变成刀片形状，可在任意位置点击切割。
*   **向左/向右裁剪**：将滑块移到指定位置。
    *   **向左裁剪**：删除滑块左侧的所有部分。
    *   **向右裁剪**：删除滑块右侧的所有部分。
    与分割工具相比，裁剪工具自动删除了不需要的部分，步骤更简洁。
*   **删除工具**：选中素材后（素材框变白），点击此按钮或按 **`Delete`** 键即可删除。
*   **定格**：在滑块位置截取当前画面，生成一张静态图片（默认约3秒），常用于制作画面暂停效果。
*   **倒放**：将选中的视频片段反向播放。
*   **镜像**：将画面进行水平翻转。
*   **旋转**：点击按钮可90度旋转画面，也可手动输入任意角度。
*   **裁剪**：对画面进行区域裁剪，只保留选定部分。
*   **智能剪口播**：自动分析语音素材，识别并剪掉“嗯”、“啊”等语气词。适用于课程、访谈等内容的快速粗剪。

![](img/0b38230c1b45792bf68116833bda9f4a_14.png)

### 其他实用工具

*   **录音**：点击后弹出录音窗口，可录制旁白。录制完成后，音频会自动添加到时间线。
*   **关闭主轨道吸附**：默认开启时，主轨道素材会自动吸附到时间线起始点（0秒处）。关闭后，主轨道素材可自由移动位置。
*   **打开自动吸附**：开启后，当移动素材的头尾接近另一素材的头尾时，会出现绿色提示线，松开鼠标即可精准拼接，避免出现黑帧或重叠。
*   **关闭联动**：用于建立素材间的关联。例如，选中一个贴纸和主视频，点击“联动”，两者之间会出现连接线。之后移动主视频，贴纸会跟随移动。再次点击可取消联动。
*   **打开预览轴**：开启后，时间线上会出现一条黄色预览线。不移动白色滑块，直接拖动黄色线即可快速预览整条时间线的内容。在预览点单击，白色滑块会跳至该位置。
*   **缩放时间线**：拖动滑块或使用 **`+`**（放大）、**`-`**（缩小）快捷键，可以调整时间线显示的精细度，便于进行帧级别的精细剪辑。

## 右键菜单功能

在时间线素材上右键单击，会弹出更多实用功能。

*   **停用片段**：快捷键 **`Shift + 1`**，用于单独隐藏素材。
*   **创建组合/解除组合**：选中多个素材（如视频、贴纸、文字），右键选择 **`创建组合`**，可以将它们编组，方便整体移动和编辑。选择 **`解除组合`** 可解散编组。
*   **新建复合片段/解除复合片段**：选中多个素材（特别是效果层），右键选择 **`新建复合片段`**，可以将它们打包成一个整体，简化时间线视图，但效果不变。**`解除复合片段`** 可还原。
*   **设定时间区域**：如果只想导出视频的某一部分，有两种方法：
    1.  右键目标片段，选择 **`时间区域`** -> **`以片段选定区域`**。
    2.  使用快捷键 **`I`** 设定入点，**`O`** 设定出点。设定后，时间线上会高亮显示选定区域，导出时仅导出该部分。
