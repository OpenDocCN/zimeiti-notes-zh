## 核心四步法：推拉摇移

陈列拍摄遵循一套标准流程，即“推、拉、摇、移”。掌握这四种基本运镜方式，可以让你轻松拍出专业感。

以下是每一步的具体操作方法和要点：

1.  **推**
    *   将想要重点呈现的一杆陈列放置在画面中央。
    *   在画面两侧安排一些前景，使核心陈列更加突出。

2.  **拉**
    *   运镜方向与“推”相反，旨在展现从局部到全景的过渡。
    *   可以从一个精美的陈列细节（如一件有设计感的衣服）开始。
    *   镜头平稳缓慢地向后移动，逐渐展现出整个店铺的全景。

3.  **摇**
    *   这是一种环绕或水平转动的拍摄方式。
    *   可以全方位地展示某一区域的陈列。
    *   一种高级用法是：在镜头前设置一个虚化的前景（如走过的人影或植物），焦点对准远处的陈列。
    *   然后镜头缓慢向左或向右平移，使观众的注意力始终聚焦在清晰的远景陈列上。

4.  **移**
    *   指摄像机本身进行水平移动。
    *   常用于跟随展示或浏览式的拍摄，让观众有身临其境的逛店感。

## 拍摄前的关键准备

在正式拍摄之前，明确的拍摄目的至关重要。这决定了你运镜的方式和重点。

你需要提前思考并确定：本次拍摄的核心是展示**衣服的细节质感**，还是展示**整体的陈列氛围与空间感**。想清楚这一点，你的镜头才会有重点。
