# 小红书无货源电商教程：P26：小红书账号类型与官方广告解析 🎯

在本节课中，我们将学习小红书账号的三种主要类型，并对比分析自建账号运营与投放官方广告这两种推广方式的区别与联系。我们将理清它们之间的关系，帮助你根据自身情况选择最合适的运营策略。

## 账号类型解析

上一节我们介绍了小红书的平台特性，本节中我们来看看平台上的账号主要分为哪几类。理解账号类型是进行精准运营和推广的基础。

小红书账号主要可以分为以下三类：

1.  **企业号**
    企业号的主要作用是树立官方品牌形象。它拥有官方认证标志，并能行使一些官方特权，例如展示品牌名称、品牌地址等。这有助于更便捷地开展相关业务。

2.  **个人号**
    目前大部分用户运营的是小红书个人号。个人号的特点是灵活性高，运营者可以是品牌员工，也可以是品牌消费者。这使得账号能承载更多个人品牌特色，品牌相关人物或员工出镜也显得更为自然。

3.  **引流号**
    引流号在之前的课程中也有提及，俗称“炮灰号”。这类账号可以创建多个，其作用相对次要，通常用于特定引流目的后便可舍弃，或在紧急情况下作为备用小号使用。引流号一般没有精细的个性规划。

通常，我们看到的品牌推广或引流渠道，主要是通过企业号、个人号或引流号自行发布笔记来实现。

## 企业号矩阵运营建议

了解了账号分类后，我们来看看如何有效运营企业号及其关联账号，构建一个高效的推广矩阵。

![](img/ae6db2f62d983bb4fe10644adf0da5b6_1.png)

![](img/ae6db2f62d983bb4fe10644adf0da5b6_3.png)

以下是打造企业号矩阵的一些具体建议：

![](img/ae6db2f62d983bb4fe10644adf0da5b6_5.png)

*   **积极参与品牌活动**：品牌企业号发起的活动，如试用、团购等，应鼓励使用多个小号积极参与，增加活动热度与真实性。
*   **强化个人号品牌属性**：在个人号的笔记内容中，应多次暗示品牌的定位属性。例如，品牌定位是都市青年还是职场新人，需要在内容中清晰体现。
*   **合理规划产品发布频率**：个人号发布品牌产品笔记的频率不宜过高。建议采用 **“三七分”原则**，即30%的内容用于营销推广，70%的内容用于种草分享或知识科普，这样的内容比例更为健康。
*   **巧用引流号进行评论引流**：当企业号需要直接引导用户时，最好使用引流号在笔记评论区进行互动和引流，这样更为自然且不易引起用户反感。

## 官方广告（CPC竞价广告）介绍

![](img/ae6db2f62d983bb4fe10644adf0da5b6_9.png)

![](img/ae6db2f62d983bb4fe10644adf0da5b6_11.png)

除了运营自建账号矩阵，另一种重要的推广方式是投放小红书官方广告。接下来，我们将详细介绍这种付费推广模式。

![](img/ae6db2f62d983bb4fe10644adf0da5b6_13.png)

小红书官方广告即CPC（按点击付费）竞价广告。其展现形式主要有两种：

1.  **信息流广告**：用户浏览小红书首页信息时，系统会在信息流中不定期插入相关的广告内容。
2.  **搜索广告**：当用户使用关键词进行搜索时，与关键词匹配的广告会出现在搜索结果中。其原理类似于百度的关键词竞价排名，通过对不同关键词出价，竞争不同的广告展示位。

目前，信息流广告常见的展示位置主要是搜索结果页的第3位、第13位和第23位。

## 如何申请官方广告

如果你想尝试官方广告推广，需要先完成企业号认证。以下是关键步骤：

1.  **认证企业号**：访问小红书官网，在“业务合作”中找到“推广合作”入口。投放广告的前提是拥有一个已认证的企业号。你需要按照官方流程提交相关资质并缴纳审核费用，通过后普通账号即可升级为官方认证的企业号。
2.  **进入广告平台**：成功认证企业号后，即可通过官网的“推广合作”入口，进入广告投放平台，创建和管理你的广告计划。
3.  **参考官方指南**：关于信息流广告投放的常见问题，如计费方式、优化技巧等，平台官方帮助中心提供了详细的问答指南，建议在投放前仔细阅读。

![](img/ae6db2f62d983bb4fe10644adf0da5b6_15.png)

需要注意的是，官方广告投放涉及一定的费用，且企业号认证也有相应成本。因此，建议在资金允许的情况下考虑与官方合作，因为它能直接通过平台算法获取精准流量。
