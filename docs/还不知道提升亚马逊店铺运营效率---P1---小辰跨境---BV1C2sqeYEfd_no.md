# 亚马逊ERP系统教程：P1：跟卖功能详解 🚀

在本节课中，我们将要学习如何使用亚马逊自发货ERP系统的核心功能——跟卖。我们将详细介绍从创建跟卖、查询产品到批量采集和上架的完整操作流程，帮助您提升店铺运营效率。

## 概述

跟卖功能允许卖家通过多种方式快速找到并上架亚马逊平台上的现有商品。本节将系统性地讲解该功能的各个操作模块。

## 跟卖功能模块详解

上一节我们介绍了课程概述，本节中我们来看看跟卖功能包含的几个核心操作模块。

### 1. 创建跟卖

创建跟卖主要通过输入商品的**ASIN**或**UPC**码来查询并定位目标产品。
*   **操作方式**：在其他渠道找到商品ASIN后，在系统内输入进行查询。

### 2. 子体查询

此功能用于查询商品的变体信息。
*   **操作方式**：通过输入父体**ASIN**，系统将列出所有相关的子体ASIN，供您选择具体跟卖哪一个。

### 3. 跟卖产品管理

在此模块中，您可以对选定的商品进行批量操作。
*   **操作方式**：您可以执行**批量任务**、**批量上架**以及设置**最大库存数量**。所有修改完成后，可直接一键上架。

## 商品采集功能

在了解了基本的跟卖操作后，我们来看看系统另一个强大的功能——商品采集。许多用户主要通过此功能来使用我们的系统。

### 采集方式

系统提供五种灵活的采集方式，帮助您获取商品信息。
以下是五种采集方式的列表：
*   **关键词采集**
*   **店铺链接采集**
*   **商品列表采集**
*   **类目采集**
*   **ASIN采集**

### 采集结果展示

采集到的商品会在“采集列表”中集中展示，信息非常全面。
采集列表将显示以下关键信息：
*   ASIN
*   商品标题
*   品牌
*   商品状态
*   卖家配送方式
*   跟卖卖家数量
*   评论数量
*   上下架时间
*   销量排名

### 采集操作指南

![](img/5e3e5410feea61d4c62dc3f4d0ad7b60_1.png)

对于测试用户或新用户，在采集过程中需要注意分类与关键词的对应关系。
具体操作步骤如下：
1.  **选择分类**：确保选择的商品分类与输入的关键词相匹配。例如，采集服装类商品，就应输入与服装、鞋靴、珠宝等相关的关键词。
2.  **设置参数**：您可以限制采集的页数范围（例如1-10页、1-100页）。
3.  **筛选条件（可选）**：价格区间和最低评分等筛选条件可暂时不设置。
4.  **开始采集**：点击“确定”开始采集任务。

![](img/5e3e5410feea61d4c62dc3f4d0ad7b60_1.png)

### 处理采集结果

采集任务启动后，需要按步骤处理。
1.  **执行任务**：在采集列表中找到目标商品，点击“执行任务”。
2.  **获取详情**：系统第一步会自动获取商品的详细数据。
3.  **等待加载**：由于采集的是实时数据，商品详情不会瞬间全部显示。每个商品的详情获取大约需要 **2到3秒** 的时间，请耐心等待列表逐步加载完成。
