# TikTok跨境电商运营：P24：亚马逊店铺注册流程实操 🛒

在本节课中，我们将学习亚马逊店铺注册的完整模拟流程。通过详细的步骤演示，你将了解从准备资料到完成视频认证的每一个环节，为实际注册店铺打下坚实基础。

---

上一节我们介绍了注册前的准备工作，本节中我们来看看具体的注册模拟操作。

![](img/20d13958514a5a5e28b1493046475679_1.png)

![](img/20d13958514a5a5e28b1493046475679_3.png)

首先，我们需要访问亚马逊店铺注册模拟器。在搜索引擎中搜索“亚马逊店铺注册模拟”即可找到。本次演示以注册美国站为例，其他站点（如欧洲站、日本站）的注册流程基本一致。目前注册美国站后，其他12个站点也会同步开通，但处于未激活状态，后续在后台激活即可。



![](img/20d13958514a5a5e28b1493046475679_1.png)



![](img/20d13958514a5a5e28b1493046475679_5.png)

以下是开始模拟注册的步骤：
1.  选择“新卖家模拟注册”。
2.  注意提示：**不要使用真实信息填写**，以免与未来或现有账号产生关联。
3.  建议首次注册者先观看注册讲解视频。



![](img/20d13958514a5a5e28b1493046475679_3.png)



我们直接开始模拟注册过程。页面提供了虚拟资料用于模拟，以及相关的知识点和帮助内容。

![](img/20d13958514a5a5e28b1493046475679_7.png)

以下是填写账户信息的步骤：
1.  **姓名**：填写虚拟法人姓名，例如“田丽琴”。
2.  **邮箱地址**：点击“账号资料”，使用提供的虚拟邮箱填写。
3.  **密码**：随机设置一个密码，例如 `123`。
4.  点击“下一步”。



![](img/20d13958514a5a5e28b1493046475679_9.png)

![](img/20d13958514a5a5e28b1493046475679_5.png)



![](img/20d13958514a5a5e28b1493046475679_11.png)

系统会向邮箱发送验证码。注意查看网页右下角的弹窗或登录邮箱后台获取验证码，例如 `231416`，然后进行验证。



![](img/20d13958514a5a5e28b1493046475679_13.png)

![](img/20d13958514a5a5e28b1493046475679_7.png)



验证邮箱后，需要绑定手机号。

以下是绑定手机号的步骤：
1.  选择“中国大陆”，填写虚拟注册手机号。
2.  点击“发送验证码”。
3.  查看手机短信，复制验证码并填写。



![](img/20d13958514a5a5e28b1493046475679_9.png)



完成上述步骤后，账户创建成功，进入正式注册的第一步。

以下是填写公司信息的步骤：
1.  **营业执照国家**：选择“中国”。
2.  **主体类型**：选择“私有企业”。
3.  **公司名称**：填写中文及拼音，例如“杭州博辰网络科技有限公司”。注意拼音全部小写且无空格。
4.  确认信息无误后，点击“同意并继续”。



![](img/20d13958514a5a5e28b1493046475679_11.png)



接下来需要填写公司的详细资料。

![](img/20d13958514a5a5e28b1493046475679_15.png)

以下是填写公司详细资料的步骤：
1.  **公司注册编号**：填写营业执照上的“统一社会信用代码”，例如 `9111010872******5P`。务必仔细核对。
2.  **公司地址**：填写营业执照上的注册地址，包括省、市、区、街道和具体门牌号。
3.  **邮编**：通过查询工具确认地址对应的邮编，例如 `310016`。
4.  仔细核对所有信息，因为提交后可能无法返回修改。



![](img/20d13958514a5a5e28b1493046475679_17.png)

![](img/20d13958514a5a5e28b1493046475679_13.png)



信息填写完毕后，进行手机号验证。此处可使用之前绑定的手机号，也可以添加备用手机号用于店铺二次验证。发送并填写验证码即可。



![](img/20d13958514a5a5e28b1493046475679_15.png)



完成公司信息验证后，进入法人信息填写环节。

以下是填写法人信息的步骤：
1.  **姓名**：填写法人姓名，如“田丽琴”。
2.  **国籍与出生地**：选择“中国”。
3.  **出生日期**：按身份证信息填写，例如 `1997年5月23日`。
4.  **身份证号码**：填写完整的身份证号。
5.  **身份证有效期**：填写截止日期，例如 `2037年2月22日`。
6.  **身份证显示名称**：再次核对姓名，确保无错别字。



![](img/20d13958514a5a5e28b1493046475679_17.png)



接下来填写法人的居住地址。这里需要注意：**居住地址最好填写法人身份证地址或信用卡账单地址**。这样做的原因是，如果账号触发审核（如二审或QIC审核），提供与此地址一致的账单（如信用卡账单）有助于提高审核通过率。对于欧洲站，建议直接填写信用卡账单地址；对于美国站或日本站，此方法也有约50%的通过概率。填写完毕后，可以选择使用此地址。



![](img/20d13958514a5a5e28b1493046475679_19.png)



手机号码可默认使用已绑定的号码。接下来是关于“企业受益所有人”和“企业法人代表”的勾选。

以下是勾选企业关系的注意事项：
*   如果公司是**个人独资**，可以全部勾选。
*   如果公司有**其他股东**，则只能勾选“是企业的法人代表”。
*   如果有**持股超过30%的股东**，需要取消勾选后，在亚马逊要求的额外栏目中添加该股东信息。**注意**：若添加的股东是其他店铺的法人，可能导致店铺关联。
*   务必如实填写，点击“确定”并保存。



![](img/20d13958514a5a5e28b1493046475679_21.png)



接下来进入收款账户信息填写环节。注册前需准备好身份证、营业执照、信用卡和收款账户（如连连支付）。

以下是填写收款账户信息的步骤：
1.  **金融机构**：选择你使用的收款服务商，例如“连连支付”。
2.  **持有人姓名**：填写法人姓名。
3.  **识别代码**：登录收款账户后台查找9位数识别代码，或联系客服获取。
4.  系统验证收款账户时，点击“继续”进入下一步。



![](img/20d13958514a5a5e28b1493046475679_23.png)



现在需要填写信用卡信息。请使用支持外币扣款的信用卡（如Visa或Mastercard）。

以下是填写信用卡信息的步骤：
1.  **信用卡资料**：填写卡号、持卡人姓名、有效期（如 `2024年9月`）。
2.  **收费地址**：**务必更改为信用卡账单地址**。点击“添加新地址”重新填写。
3.  地址添加成功后，点击“下一页”。



![](img/20d13958514a5a5e28b1493046475679_25.png)



接着设置店铺名称。店铺名称在店铺注册成功后可以修改，因此可以先拟定一个，例如使用法人姓名。确认名称可用后，点击“下一步”。

关于UPC编码和品牌问题：
*   **是否拥有UPC编码**：可选择“是”。
*   **是否为制造者或品牌所有者**：如果是工厂或已进行品牌备案，选择“是”；否则，可选择“否”或“其中一部分”。本例选择“否”，然后点击“下一步”。



![](img/20d13958514a5a5e28b1493046475679_19.png)

![](img/20d13958514a5a5e28b1493046475679_27.png)



现在进入身份验证文件上传环节。需要提前准备好高清的营业执照和身份证正反面照片。

![](img/20d13958514a5a5e28b1493046475679_21.png)

以下是上传身份文件的步骤：
1.  点击“上传”按钮。
2.  **务必确保**图片清晰，且四个边角完整露出。
3.  依次上传身份证正面、身份证背面和营业执照。
4.  上传后，仔细核对之前填写的所有信息。
5.  确认无误后，点击“提交”。



![](img/20d13958514a5a5e28b1493046475679_29.png)



提交资料后，系统会要求进行视频通话认证。

以下是预约和准备视频认证的步骤：
1.  点击“下一步”预约时间。选择周一到周五方便的时间段（显示为当地时间）。
2.  预约成功后（例如周五下午1:30），进入最后一步。
3.  **重要**：保存当前页面网址、注册邮箱和密码。
4.  视频认证前，确保摄像头和麦克风可用。
5.  认证时，需由法人本人携带营业执照原件参与。
6.  视频通话通常持续5-15分钟，问题较简单，例如确认是否本人、询问身份证号、计划销售的产品等。



![](img/20d13958514a5a5e28b1493046475679_31.png)



点击“接受”后，保存好网址。在预约时间前半小时，使用具备摄像头和麦克风的电脑或手机登录。

**关键注意事项**：
*   如果使用手机进行视频通话，**务必使用手机4G/5G流量，不要连接Wi-Fi**，以免因网络环境导致店铺关联风险。
*   视频通话结束后，等待亚马逊的成功通知即可。
