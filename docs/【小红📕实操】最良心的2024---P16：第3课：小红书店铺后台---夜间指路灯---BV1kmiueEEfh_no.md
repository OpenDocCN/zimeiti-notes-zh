# 小红书电商运营课：P16：第3课：小红书店铺后台的基础设置 🛠️

在本节课中，我们将学习如何完成小红书店铺后台的基础设置。这些设置是店铺正式运营和销售商品前的必要步骤，包括保证金、支付方式、运费险、物流设置和店铺公示等关键环节。完成这些设置后，你的店铺就具备了基本的运营条件。

上一节我们介绍了如何开通小红书店铺，本节中我们来看看开通后必须完成的后台基础设置。

## 1. 保证金 💰

保证金是开店的基础要求之一。对于个人店铺，保证金金额通常为**1000元**。这笔费用在开店初期可以选择不缴纳，待店铺开始盈利后再补交。保证金支持随时退还，关闭店铺即可申请退款，没有额外损失。

以下是关于保证金的详细说明：
*   **金额**：个人店铺的保证金基本为**1000元**。绝大多数商品类目都适用此标准。
*   **缴纳时机**：初期资金紧张时可不交，不影响店铺基本功能。建议在有能力时尽早缴纳。
*   **查看与规则**：进入商家后台，点击“资金”->“保证金”页面可查看具体金额。后台“课堂”中搜索“保证金”，可以查阅《店铺保证金平台服务费介绍》等官方规则文档。

## 2. 支付方式 💳

支付方式关系到货款结算，需要绑定收款账户。主要包括支付宝和微信支付两种。

以下是支付方式设置的要点：
*   **必须绑定**：支付宝和微信支付账户都需要开通。
*   **开通条件**：不缴纳保证金时，**微信支付账户无法开通**，但这基本不影响顾客下单转化。支付宝账户可以正常开通。
*   **审核时间**：支付宝认证通常当天可审核通过。微信支付需等待**第二天**审核。
*   **账户一致性**：绑定的支付宝、微信以及银行卡的**法人姓名**，必须与开店认证的**身份证信息完全一致**。

## 3. 运费险 📦

开通运费险可以降低消费者的购物顾虑，从而提升转化率，尤其对于特定商品类目效果显著。

以下是关于运费险的说明：
*   **建议开通**：特别是对于**服装、鞋包**等需要试穿、容易产生退换货需求的商品类目，开通运费险对转化率提升很有必要。
*   **成本与收益**：每成交一单，系统会扣除约**0.59元**作为保费。当顾客退货时，平台会向顾客补贴约**7元**的退货运费，消除了顾客的运费顾虑。
*   **开通路径**：进入商家后台，点击“交易”->“运费保障”即可进行开通设置。

## 4. 物流设置 🚚

物流设置包括地址库管理、物流模板和运费模板，是确保订单正常发货和退货的关键。

![](img/a83193e8b4c8ab52f4d253d1f0f1d40d_3.png)

以下是物流设置的具体步骤和技巧：

**地址库管理**
地址库需设置发货地址和退货地址。
*   **发货地址**：建议填写自己的真实地址和手机号。若本人所在地属偏远地区，可填写如杭州、上海、广州等快递发达的一线城市地址，这样前台会显示从该地发货，给顾客收货快的印象。
*   **退货地址**：这是重要技巧。由于小红书有时会系统默认同意退货申请，而你的供应商（如上家）可能有很多个，他们的地址各不相同。因此，**默认退货地址不能填写真实供应商地址**。
    *   **技巧**：将默认退货地址设置为**港澳台地区**（如台湾、香港），并在详细地址栏填写“**已咨询客服获取**”。联系电话仍留自己的。这样设置不违规，且能有效避免商品被误退至无法处理的地址。
    *   **多地址退货功能**：小红书新增了“多地址退货”功能。你可以为每个商品链接单独设置对应的供应商退货地址。例如，为“A款童装”链接关联其上家的真实退货地址。这样，该商品产生的所有退货都会自动使用这个地址，管理起来非常方便。

![](img/a83193e8b4c8ab52f4d253d1f0f1d40d_7.png)

**运费模板**
系统提供“全国包邮”和“非偏远地区包邮”两种默认模板。
*   **必须选择**：上架商品时，务必选择 **“默认非偏远地区包邮模板”**。
*   **原因**：该模板会自动将新疆、西藏、青海、宁夏、海南、内蒙古等偏远地区设置为不包邮。这样可以避免因向这些地区包邮而产生的额外运费损失，因为你的上家（如拼多多）通常也不对这些地区包邮。

**物流模板与店铺运费策略**
*   **物流模板**：通常无需特别设置，使用系统默认即可，主要确认发货地址。
*   **店铺运费策略**：如果店铺商品全部包邮，则无需设置此项。

## 5. 店铺公示 📢

![](img/a83193e8b4c8ab52f4d253d1f0f1d40d_11.png)

店铺公示是商品上架前的最后一步。如果不完成公示，系统会阻止你发布商品。

以下是店铺公示的填写要求：
*   **填写真实信息**：根据提示，如实填写联系人、手机号、邮箱和详细地址。这些信息用于平台在紧急情况下联系你。
*   **经营类型**：可以全选所有选项，或至少勾选前三个，这对店铺运营没有实质影响。
*   **完成操作**：填写完毕后，点击“保存公示”即可。

![](img/a83193e8b4c8ab52f4d253d1f0f1d40d_13.png)

本节课中我们一起学习了小红书店铺后台的五大基础设置：**保证金、支付方式、运费险、物流设置和店铺公示**。完成这些步骤后，你的店铺就完成了开业准备，接下来就可以进入选品、发布笔记和销售商品的运营阶段了。请务必按照教程仔细操作，特别是物流设置中的地址技巧，能为后续运营避免很多麻烦。
