# 小红书电商运营教程：P13：小红书店铺类型详解 🏪

![](img/0c3b01f3a1a34da27f8d9870c10a09a3_0.png)

在本节课中，我们将系统学习小红书电商的店铺类型。了解不同类型的店铺及其特点，是开启小红书电商之旅的第一步。

## 概述

小红书平台上的店铺主要分为两大类：**商家店**和**官方自营店**。商家店是品牌或商家自行入驻开设的店铺，而官方自营店则由小红书平台直接运营。接下来，我们将逐一拆解这些店铺类型。

## 商家店类型详解

商家店是大多数品牌和商家入驻小红书时选择的店铺形式，它根据品牌资质和经营模式，进一步细分为以下几种。

以下是商家店的具体分类：

1.  **官方旗舰店**
    *   这是权重最高的店铺类型，通常只授予被小红书平台认定的、具有较高知名度的品牌。

2.  **旗舰店**
    *   同样需要品牌具备一定的知名度，其权重仅次于官方旗舰店。

3.  **品牌店**
    *   这是最普遍、最常见的商家店类型。普通商家入驻小红书，默认开通的就是品牌店，其名称通常为“某某某品牌店”。

4.  **专卖店**
    *   与品牌店类似，也是商家常见的店铺形态之一。

5.  **集合店**
    *   这类店铺允许在一个店铺内销售多个不同品牌的产品，类似于天猫的“专营店”。**但需要注意的是，目前小红书平台已暂停新集合店的入驻**，未来是否会重新开放需关注平台政策。

上一节我们介绍了商家店的五种具体类型，本节中我们来看看不同类型店铺之间的关系与升级路径。

## 店铺权重与升级机制

不同类型的店铺在平台内享有不同的权重，这可能会影响到商品和笔记在搜索和推荐中的排名。其权重关系可表示为：

**店铺权重排序：官方旗舰店 > 旗舰店 > 品牌店/专卖店/集合店**

这意味着，在其他条件相同的情况下，官方旗舰店的商品可能获得更多的曝光机会。但请注意，店铺权重只是影响因素之一，并非绝对。

对于已经开通品牌店的商家，满足一定条件后可以申请升级为旗舰店或官方旗舰店。**升级的核心条件通常是店铺的营业额**。平台会根据不同商品类目设定不同的营业额标准，例如：
*   小类目商品可能月销售额达到几十万即可申请。
*   大类目商品则可能要求更高的销售额门槛。

## 官方自营店：小红书福利社

除了商家自行开设的店铺，小红书平台也像“京东自营”一样，拥有自己的官方自营渠道——**小红书福利社**。

商家可以将自己的产品以供货的形式入驻福利社。通常，小红书运营团队会主动寻找在平台上销量好、热度高的产品，邀请其品牌入驻。合作模式是：商家作为供货商，以较低的供货价（例如约为市场价的6折）将商品供给小红书福利社，由福利社作为经销商进行销售和运营。

## 总结

![](img/0c3b01f3a1a34da27f8d9870c10a09a3_2.png)

本节课中，我们一起学习了小红书电商的店铺体系。我们了解到店铺主要分为**商家店**和**官方自营店**。商家店包含官方旗舰店、旗舰店、品牌店、专卖店和集合店（已暂停入驻），它们之间存在权重差异并可依据营业额进行升级。而**小红书福利社**作为平台自营渠道，为优质商品提供了另一种销售路径。理解这些店铺类型，有助于你根据自身品牌定位和发展阶段，做出最合适的开店选择。
