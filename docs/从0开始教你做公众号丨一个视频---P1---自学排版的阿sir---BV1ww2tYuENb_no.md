# 从0开始做公众号：P1：详解公众号白名单功能 🔐

在本节课中，我们将要学习微信公众号的“白名单”功能。我们将了解什么是原创保护，如何为单篇或全部原创文章设置转载权限，以及如何正确转载他人的原创内容。掌握这些知识，能有效保护你的原创作品，也能在遵守规则的前提下进行内容分享。

---

上一节我们介绍了公众号的基本概念，本节中我们来看看如何管理文章的转载权限，即“白名单”功能。

![](img/76a6b588f30498169840755fb8ff03d6_1.png)

首先，你需要准备一篇自己原创的推文。在编辑器中发布时，可以点击“申请原创”来为文章添加原创保护。这就是为什么直接转发他人的原创文章会受到限制。

申请原创时，系统主要识别文字内容。这意味着，即使你修改了文章的排版样式，平台依然能通过文字识别出这是原创作品。

![](img/76a6b588f30498169840755fb8ff03d6_3.png)

申请原创后，需要填写作者名称。这个名称可以自由发挥，但有字数限制，最多只能填写**8个字符**。

![](img/76a6b588f30498169840755fb8ff03d6_4.png)

接下来是核心的“白名单”设置。在这里，你可以输入其他公众号的名称，授权该公众号转载你这篇原创文章。这与普通的“分享”有本质区别。



设置白名单时，需要配置具体的转载权限。主要有两个选项：
*   **可修改文章**：允许转载方对文章内容进行编辑和修改。
*   **可不显示来源**：允许转载方发布时不显示“本文转载自XXX”的来源提示。

**注意**：务必取消“快捷转载”选项。如果开启，意味着所有公众号都可以不受限制地转载你的文章，白名单就失去了意义。



设置完成后，即可发表文章。成功声明原创的文章标题旁会有一个“原创”标识。此时，只有被你加入白名单的公众号才能转载这篇文章。

![](img/76a6b588f30498169840755fb8ff03d6_6.png)

为了更清晰地理解，我们回到公众号的转载界面。在转载设置中，被添加的公众号会出现在列表里，并拥有你设定的权限。

![](img/76a6b588f30498169840755fb8ff03d6_7.png)

![](img/76a6b588f30498169840755fb8ff03d6_8.png)

如果原创作者在设置白名单时，**没有**勾选“可不显示来源”的权限，那么转载方发布后，文章顶部会自动出现来源声明。点击该声明会跳转到原创公众号的主页。如果勾选了“可不显示来源”，则转载后的文章看起来就像自己原创的一样，不会有来源提示。





![](img/76a6b588f30498169840755fb8ff03d6_9.png)

---

![](img/76a6b588f30498169840755fb8ff03d6_10.png)

以上我们讲解的是针对**单篇**原创文章设置白名单。但如果你希望某个公众号能转载你**所有**的原创文章，难道需要每发布一篇就设置一次吗？这样显然非常麻烦。

当然有更方便的方法。以下是设置全局白名单的步骤：
1.  登录公众号后台，在左侧工具栏找到并点击“原创”功能。
2.  在页面中找到“全局可转载账号”设置项。
3.  添加你希望授权的公众号ID或名称即可。

完成此设置后，被添加的公众号就获得了转载你所有原创文章的长期权限。



因此，当你想转载他人的原创文章时，正确的做法是主动联系原作者（通过文章留言或私信），请求对方将你的公众号加入其文章或账号的**白名单**中。

本节课中我们一起学习了微信公众号的原创保护与白名单机制。我们了解了如何为自己的原创文章设置单篇或全局转载权限，也明白了转载他人作品前需获得授权。合理使用这些功能，既能保护自己的创作成果，也能在尊重版权的基础上进行内容交流。
