    *   **描述**：张老师声称其投入近200万元建立的素材库，承诺永久免费使用。

2.  **软件工具包**
    *   **描述**：一套市场价值宣称达12800元的软件工具包。
    *   **包含**：做图软件、拍摄软件、剪辑软件、配音软件各一套。

除了礼包，张老师还承诺提供以下支持：

以下是承诺提供的额外支持与资源：

![](img/5d359430a0bda79d5622dda6a59d761e_28.png)

![](img/5d359430a0bda79d5622dda6a59d761e_29.png)

![](img/5d359430a0bda79d5622dda6a59d761e_30.png)

*   **私人联系方式与靠山**：提供张老师的私人联系方式，并承诺成为学员在互联网上的“靠山”。
*   **法律援助案例**：以“吴春兰吴大姐”为例，声称通过发律师函等方式帮助学员追回在其他平台受骗的损失。
*   **玩法资源对接**：承诺打包提供视频号各种玩法的教学资源，包括创作分成、视频带货、无人直播、书单号、情感IP、短剧等。
*   **不限制玩法与账号**：承诺不限制学员操作的账号数量和玩法，并展示了“吴淑良”等学员操作多个账号的案例。
*   **签订合同与一次收费**：承诺签订合同，一次性收费，无二次收费，并在国家税务局备案。
*   **五大保障**：
    1.  保证包教会。
    2.  保证所有账号百分之百赚钱，否则亲自操刀。
    3.  承诺三天内让学员见到收益（以8月9日至12日为例）。
    4.  7天无理由退款。
    5.  提供“私人定制保姆式教学”计划（例如第一周安排创作分成等）。
