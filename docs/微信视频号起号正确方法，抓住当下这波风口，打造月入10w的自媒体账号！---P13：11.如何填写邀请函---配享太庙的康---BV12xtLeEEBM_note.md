# 微信视频号运营教程：P13：11.如何填写邀请函 📝

在本节课中，我们将学习如何正确填写视频号创作者计划的邀请函。这是加入官方计划、获取收益权限的关键一步。我们将一步步拆解填写流程，确保信息准确无误。

![](img/ee67b93f59af70385d622a774be8798e_1.png)

---

## 第一步：找到你的视频号账号信息 🔍

![](img/ee67b93f59af70385d622a774be8798e_3.png)

![](img/ee67b93f59af70385d622a774be8798e_4.png)

上一节我们介绍了如何注册视频号，本节中我们来看看如何找到账号昵称和粉丝数等关键信息。这是填写邀请函的基础。

以下是具体操作步骤：

![](img/ee67b93f59af70385d622a774be8798e_6.png)

1.  **打开微信应用**。在手机主屏幕找到微信图标并点击打开。

![](img/ee67b93f59af70385d622a774be8798e_8.png)

2.  **点击“发现”页面**。在微信底部导航栏中，找到并点击“发现”选项。

![](img/ee67b93f59af70385d622a774be8798e_9.png)

3.  **进入“视频号”**。在“发现”页面中，向上滑动，找到“视频号”入口（图标为黄色蝴蝶形状），点击进入。

4.  **点击个人头像**。进入视频号页面后，点击右上角的个人头像图标。

5.  **查看账号信息**。在新页面中向下滑动，找到“我的视频号”栏目。这里会清晰显示你的**账号昵称**和**粉丝数量**。例如：`天道酬勤，39917个粉丝`。

## 第二步：填写邀请函表格 📄

找到账号信息后，我们就可以开始填写邀请函了。请确保每一步都仔细核对。

以下是邀请函各栏目的填写说明：

![](img/ee67b93f59af70385d622a774be8798e_11.png)

![](img/ee67b93f59af70385d622a774be8798e_12.png)

![](img/ee67b93f59af70385d622a774be8798e_13.png)

1.  **视频号账号昵称**：填写你在上一步中查看到的账号昵称，例如：`天道酬勤`。

2.  **粉丝数量**：填写你在上一步中查看到的粉丝数，例如：`39917`。

3.  **活动邀请人**：根据你的邀请渠道填写。例如，如果通过“张良老师”的渠道加入，则填写`张良`或`张良老师`。

4.  **实名信息**：此处必须填写你本人的真实姓名，与身份证信息一致。例如：`周建国`。

![](img/ee67b93f59af70385d622a774be8798e_15.png)

![](img/ee67b93f59af70385d622a774be8798e_16.png)

5.  **结算手机号**：填写你用于接收验证码和后续收益结算的手机号码。格式为连续数字，例如：`13812345678`。

## 第三步：提交与确认 ✅

![](img/ee67b93f59af70385d622a774be8798e_18.png)

所有信息填写完毕后，最后一步是提交。

![](img/ee67b93f59af70385d622a774be8798e_20.png)

![](img/ee67b93f59af70385d622a774be8798e_22.png)

请点击页面底部的**蓝色提交按钮**。系统会弹出“确定要提交吗？”的确认框，务必点击**确认**。提交成功后，页面会显示“已成功加入创作者分成计划”的提示。

---

![](img/ee67b93f59af70385d622a774be8798e_24.png)

本节课中我们一起学习了填写视频号邀请函的完整流程。关键点在于**准确找到并填写个人账号信息**，并确保**实名信息与手机号正确无误**。完成提交后，你就正式加入了创作者计划，可以开始通过发布内容获得收益了。
