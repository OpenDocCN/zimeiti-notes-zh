## 景别的判断与运用

上一节我们介绍了课程的整体框架，本节中我们来看看视频拍摄的第一个核心概念：景别。景别是视频画面的基础，指拍摄主体在画面中所占的比例和范围。学会判断景别是快速入门的第一个关键点。

你需要了解并能够区分以下几种基本景别：
*   **远景**：展示广阔的环境和空间。
*   **全景**：展现主体的全貌及其所处环境。
*   **中景**：拍摄人物膝盖以上或物体的主要部分。
*   **近景**：拍摄人物胸部以上或物体的局部细节。
*   **特写**：聚焦于人物的面部表情或物体的关键细节。
*   **大特写**：极度放大某个局部，如眼睛或产品logo。

理解景别后，一个高效的学习方法是“对标拆解”。以下是具体操作步骤：

1.  **寻找对标**：根据你的行业（如医美、健康），找到拍摄风格你欣赏的账号或视频。
2.  **截图分析**：将视频中不同场景的画面截图。
3.  **标注景别**：在每张截图旁标注它属于哪种景别（如：店内环境 - 全景；产品展示 - 特写）。
4.  **复刻拍摄**：按照你标注的景别顺序和构图，一比一地模仿拍摄。这个过程的核心就是 **`模仿与复刻`**，先做到形似。

## 公式化运镜技巧

了解了静态的景别后，我们来看看让画面动起来的技巧：运镜。对于非专业出身的拍摄者，系统学习复杂的运镜理论成本较高。一个更实用的方法是掌握几种“公式化”的运镜模板，直接套用。

例如，一个常见的运镜组合可以描述为以下流程：
**`镜头稍仰 -> 平稳前推至主体中心 -> 环绕主体 -> 匀速后拉远离`**

你可以通过关注一些专业的抖音教学号来免费学习更多这样的模板。只需搜索相关关键词，就能找到大量将运镜步骤拆解展示的视频教程。照着这些模板反复练习，就能快速上手。

对于健康、医疗类商家而言，视频的核心要求是 **`清晰`** 和 **`稳定`**，对运镜的创意和复杂度要求通常低于吃喝玩乐类目。因此，掌握几种基础、稳定的运镜公式足以满足70分的拍摄需求。

如果你的团队对画面质感有更高要求，可以借助一些软硬件工具来提升。这里推荐一个组合方案：

1.  **专业拍摄APP**：在苹果手机上下载如 **ProMovie** 这类付费专业拍摄软件（价格约数十元）。这类APP能提供更手动、更专业的参数控制（如快门速度、ISO、白平衡）。
2.  **手机稳定器**：配合一个物理稳定器使用，可以极大消除手持拍摄的抖动，保证画面流畅平稳。

结合 **`专业APP + 物理稳定器`** 这个组合，你用手机拍摄出的视频质感，完全可以媲美专业单反相机的基础拍摄效果。
