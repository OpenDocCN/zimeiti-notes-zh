# 新媒体运营教程：P25：多个视频融合起来 🎬

![](img/1405177b26d2f3f162a84af9d739fe6f_1.png)

在本节课中，我们将学习如何将手机中的多段视频素材，通过关键帧和混合模式功能，无缝衔接并融合成一个流畅的视频。这种方法适用于制作转场效果，能让你的视频更具专业感。

![](img/1405177b26d2f3f162a84af9d739fe6f_3.png)

## 概述

![](img/1405177b26d2f3f162a84af9d739fe6f_5.png)

![](img/1405177b26d2f3f162a84af9d739fe6f_7.png)

上一节我们介绍了视频剪辑的基础操作，本节中我们来看看如何实现多个视频的融合转场。核心在于使用**关键帧**控制动画，以及**混合模式**调整图层透明度，从而实现平滑过渡。

## 操作步骤详解

以下是实现多视频融合的具体步骤，请跟随指引逐步操作。

1.  **导入视频素材**
    首先打开剪辑软件，点击“开始创作”。从手机相册中勾选三段提前准备好的视频，点击右下角的“添加”按钮将它们导入时间轴。导入后，第一步是关闭所有视频片段的原声。

2.  **处理第二段视频（画中画与关键帧）**
    接着，我们需要将第二段视频转换为画中画。在时间轴上选中第二段视频，在下方工具栏向左滑动，找到并点击“切画中画”功能。长按这段新生成的画中画素材，将其向前拖拽，使其开始时间对齐主轨道视频的**2秒**处。

    然后，点击这段画中画素材，在**2秒**的位置为其添加一个**关键帧**。添加后，在下方工具栏中找到“混合模式”并点击，将其不透明度（进度条）调整为**0**，然后点击对勾确认。

    接下来，将时间轴移动到**4秒**的位置，再次点击画中画素材，添加第二个**关键帧**。然后再次打开“混合模式”，将不透明度（进度条）调整为**100**，点击对勾确认。这样，第二段视频就在2到4秒之间完成了从透明到完全显示的淡入效果。

3.  **处理第三段视频（画中画与关键帧）**
    现在来处理第三段视频。在时间轴上选中第三段视频，同样使用“切画中画”功能将其转为画中画。长按这段素材，将其开始时间拖拽到与主轨道**4秒**对齐的位置。

    在**4秒**处，点击这段画中画素材，添加一个**关键帧**。然后打开“混合模式”，将其不透明度调整为**0**，点击对勾。

    之后，将时间轴移动到**6秒**的位置，点击素材添加第二个**关键帧**，并再次打开“混合模式”，将不透明度调整为**100**。这样，第三段视频就在4到6秒之间完成了淡入。

4.  **调整素材时长与对齐**
    完成关键帧设置后，需要调整各段画中画素材的时长，使其与主轨道视频的结尾对齐。分别长按第二段和第三段画中画素材的末端，向后拖拽或向前收缩，确保它们的结束时间与主视频一致。

5.  **添加背景音乐**
    视频画面融合完成后，我们来添加背景音乐。点击下方工具栏的“返回”按钮两次，回到主菜单，找到并点击“音频”，选择“音乐”。在音乐库中，你可以选择喜欢的曲目，例如在“舒缓”分类下选择第二个音乐，点击“使用”。

    音乐添加后，将时间轴移动到视频的结尾处，选中音频轨道，点击“分割”，将多余的音乐部分删除，使音乐与视频时长匹配。

6.  **预览与导出**
    所有步骤完成后，你可以点击播放按钮预览整个视频效果。确认无误后，即可导出成品。

## 总结

![](img/1405177b26d2f3f162a84af9d739fe6f_9.png)

本节课中我们一起学习了利用**关键帧**和**混合模式**实现多视频融合的技巧。关键步骤可以总结为：**导入素材 -> 切画中画 -> 在起始时间点打关键帧并设混合模式为0 -> 在结束时间点打关键帧并设混合模式为100**。通过重复这一流程，你可以为任意多段视频添加平滑的淡入淡出转场，从而制作出无缝衔接的融合视频。
