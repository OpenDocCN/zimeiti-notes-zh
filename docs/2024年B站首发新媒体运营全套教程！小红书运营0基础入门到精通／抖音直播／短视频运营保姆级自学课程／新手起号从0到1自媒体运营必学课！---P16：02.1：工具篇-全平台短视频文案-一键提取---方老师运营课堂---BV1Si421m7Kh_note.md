# 新媒体运营全套教程：P16：02.1：工具篇-全平台短视频文案一键提取 📝

在本节课中，我们将学习一个非常实用的运营工具，它能帮助我们从抖音、小红书、B站等多个平台的短视频中，一键提取出完整的文案内容。掌握这个工具，可以极大地提升我们收集素材和进行内容再创作的效率。

## 工具介绍与背景

上一节我们介绍了多种内容创作工具，本节中我们来看看一个专门用于提取文案的高效工具。以往，当我们看到优秀的短视频时，想要获取其文案，通常需要比较繁琐的步骤。

以下是几种传统方法的示例：
*   **使用小程序**：例如“轻斗”等工具。
*   **本地软件处理**：先将视频下载到本地，再导入“剪映”等剪辑软件中提取字幕。

这些方法往往只能针对单一平台（如抖音），且步骤较多。如果我们需要处理来自小红书、视频号、哔哩哔哩、快手等多个平台的视频，操作就会更加复杂。

今天分享的这个工具，可以完美解决上述问题。它支持**在线操作**，你只需将目标短视频的链接复制到工具网页中，即可一键提取出该视频的文案。

### 提取抖音短视频文案

首先，我们尝试提取一个抖音短视频的文案。在抖音中找到一个合适的视频，例如搜索“情感语录”。



![](img/58be7315d20168bed6acd06b910ed760_9.png)

我们选择一个时长合适的视频，例如这个1分43秒的内容。复制该视频的分享链接。



![](img/58be7315d20168bed6acd06b910ed760_11.png)

回到工具界面，将复制的链接粘贴到输入框中。然后，点击“提取文案”按钮。



系统会显示“提取中”，这个过程通常非常快，只需几秒钟。提取完成后，视频的完整文案就会显示在下方区域。



如图所示，文案“听完你就释怀了，村上春树说，我动了离开你的念头...”已经被成功提取出来。

提取出的文案，我们可以进行两项便捷操作：
1.  **一键复制**：直接复制全部文本。
2.  **一键格式化**：点击后，文案会按照视频字幕的段落格式进行自动排版，更便于阅读和修改。

此外，该工具也支持**上传本地视频文件**来提取文案，功能非常全面。

### 提取哔哩哔哩（B站）视频文案

上面我们演示了抖音平台，现在我们来测试一下它对B站视频的支持情况。我们提前准备了一个B站视频的链接。



将B站视频的链接（通常以`BV`号开头）复制到工具输入框中，再次点击“提取文案”。



提取过程同样迅速。可以看到，B站视频的文案“新手做直播电商，怎么才能打造出直播爆品...”也被准确无误地提取了出来。

## 工具的应用价值与总结

本节课中我们一起学习了这款全平台短视频文案提取工具的使用方法。它的**核心优势**在于：
*   **多平台支持**：涵盖抖音、快手、小红书、B站、视频号等主流平台。
*   **在线便捷操作**：无需下载视频，只需一个链接。
*   **高效准确**：提取速度快，文案还原度非常高。

这个工具在运营工作中能提供巨大的帮助。无论是用于分析竞品文案结构、收集热门话题素材，还是提取文案后进行修改和再创作（例如用于自己的口播视频），它都能节省我们大量的时间和精力。对于需要处理多平台内容的新媒体从业者而言，这无疑是一个强大的效率利器。
