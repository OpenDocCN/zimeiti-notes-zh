# 抖音运营教程：P10：产品与用户兴趣标签

在本节课中，我们将学习一种实用的选题方法——九宫格选题法。我们将以健身软件Keep为例，详细拆解如何通过分析产品标签和用户兴趣标签，来生成源源不断的视频创作灵感。

上一节我们介绍了选题的重要性，本节中我们来看看如何具体操作。

## 产品标签分析

首先，我们需要分析产品本身的属性与特点。将核心产品（例如Keep）置于中心，围绕它联想并填充八个相关的产品特点标签。

以下是分析Keep产品特点的步骤：

1.  **产品形态**：它是一款线上健身软件。
2.  **核心资源**：软件内配备专业的健身教练。
3.  **内容服务**：提供饮食搭配建议与减肥攻略。
4.  **知识价值**：分享健身干货与科学的运动方法。
5.  **用户机制**：设有运动打卡制度，鼓励持续使用。
6.  **市场定位**：主要针对一二线城市用户开发。

> 产品标签的分析逻辑类似于分析平台用户画像。例如，抖音用户多集中于一、二线城市，而快手用户则更多分布在三、四线城市。这类数据可以在百度指数或艾瑞网等平台查询到。

完成产品标签填充后，我们暂时将其放在一边。

## 用户兴趣标签分析

接下来，我们分析目标用户的兴趣。将核心用户（例如“健身爱好者”或“想要减肥的人”）置于另一个九宫格的中心，围绕他们展开兴趣联想。

![](img/7ec501c3e49800fc63eae662619ecdaf_1.png)

以下是分析用户兴趣标签的步骤：

1.  **线下场景**：他们可能对“健身房”感兴趣。
2.  **装备需求**：他们可能关注“健身装备”，如紧身衣、运动鞋。
3.  **饮食相关**：他们可能对“减肥餐”、“轻脂美食”感兴趣。
4.  **生活方式**：作为热爱生活的人，他们可能还喜欢“旅行”。
5.  **泛娱乐**：兴趣可能延伸至“星座”、“明星”或“鸡汤文”。

通过这种方式，我们可以将用户群体从“健身爱好者”扩展为“生活热爱者”，从而覆盖更广泛的兴趣点，为内容创作提供更多可能性。

## 生成选题：连连看组合法

现在，我们手头有两个填满的九宫格：一个是产品标签，一个是用户兴趣标签。接下来，就是生成具体选题的关键步骤。

我们将采用“连连看”的方法，从两个九宫格中分别选取一个标签进行组合，碰撞出创意火花。

以下是几个组合示例：

*   **组合1**：产品标签“饮食搭配” + 用户标签“旅行”
    *   生成选题示例：`外出旅行时，如何享受美食又能干吃不胖？`
*   **组合2**：产品标签“运动方法” + 用户标签“明星”
    *   生成选题示例：`赵丽颖都在练的5个健身方法，你一定要学起来！`
*   **组合3**：产品标签“科学（方法）” + 用户标签“健身房”
    *   生成选题示例：`自律成就自由！用最科学的方法带你高效使用健身房。`

如果组合后思路枯竭，可以借助搜索引擎。例如，搜索“健身 科学”等关键词，查看联想出的长句，它们常常能激发新的选题灵感。

## 方法总结与优势

本节课中我们一起学习了九宫格选题法的完整流程。

这种方法的核心在于**连接产品属性与用户兴趣**。通过系统的标签联想和随机的组合碰撞，我们可以轻松衍生出数十甚至上百个选题方向，从而建立庞大的内容素材库，彻底解决“不知道拍什么”的难题。

记住，好的选题往往诞生于不同领域元素的创新结合之中。
