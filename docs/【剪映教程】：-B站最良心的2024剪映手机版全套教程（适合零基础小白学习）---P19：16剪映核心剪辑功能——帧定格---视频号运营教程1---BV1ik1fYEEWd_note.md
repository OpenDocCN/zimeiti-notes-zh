# 剪映教程：P19：剪映核心剪辑功能——帧定格 🎬

在本节课中，我们将要学习剪映的核心剪辑功能之一——**定格功能**。该功能可以将视频的某一瞬间画面静止，常用于制作卡点特效视频，为作品增添视觉冲击力。

## 概述：什么是定格功能？

定格功能指的是在动态的视频画面中，突然停止在某个特定的镜头画面上。在制作特效视频，尤其是卡点类视频时，这个功能非常实用。它能帮助我们在剪映中快速制作出画面绚丽、节奏感强的视频效果。

上一节我们介绍了基础的剪辑操作，本节中我们来看看如何具体使用“定格”来创造独特的视觉效果。

## 案例展示与理解

![](img/587ddfbe747c40f25d46d9630e79b58f_3.png)

讲了这么多理论，可能很多初学者对定格功能的具体表现还不甚清晰。我们可以通过案例来直观理解。

以下是几个使用了定格功能的视频效果描述：
*   **万有引力卡点视频**：这是抖音上曾流行的一种视频。在每个镜头切换的最后一帧，画面会突然静止，形成“拍照”般的酷炫效果，这正是运用了定格功能。
*   **静态转场效果**：在一个动态镜头（如行驶的火车）的末尾，画面变为静态图像，并可能叠加滤镜（如黑白褪色、牛皮纸质感），然后切换到下一个场景。

接下来，我们将进入实操环节，学习如何在剪映中实现这种效果。

## 实操步骤：如何添加定格效果

我们将通过一个简单的例子，一步步学习定格功能的使用方法。

1.  **导入素材**：打开手机剪映APP，点击顶部“创作”按钮，导入一段视频素材。

    ![](img/587ddfbe747c40f25d46d9630e79b58f_1.png)

2.  **选择定格点**：在时间轴上选中需要添加定格效果的视频片段。通常，我们会选择某个动作的结尾或转场前一帧。

3.  **应用定格功能**：点击底部二级工具栏，找到并点击 **“定格”** 按钮。

    ![](img/587ddfbe747c40f25d46d9630e79b58f_3.png)

4.  **编辑定格画面**：点击“定格”后，软件会自动截取选中时间点的画面，并生成一段时长为 **3秒** 的静态图片片段。你可以对这个定格片段进行进一步编辑。

    以下是你可以对定格片段进行的操作：
    *   **调整时长**：拖动定格片段两端的白色选框，可以自由缩短或延长其持续时间，不限于默认的3秒。
    *   **添加特效**：为定格画面添加“特效”（如模糊、震动）或“滤镜”（如牛皮纸、褪色），增强视觉效果。
    *   **进行裁剪或删除**：像处理普通视频一样，可以对定格片段进行分割、删除等操作。

    ![](img/587ddfbe747c40f25d46d9630e79b58f_5.png)

5.  **预览效果**：编辑完成后，点击预览播放，查看动态视频突然变为静态画面的卡点效果。

## 功能要点总结

使用“定格”功能后，剪映会自动为选中的那一帧视频画面生成一段静态视频。这个功能的核心价值在于帮助我们轻松制作出节奏感强的**卡点视频**。

![](img/587ddfbe747c40f25d46d9630e79b58f_5.png)

关键操作可以总结为以下代码逻辑：
```plaintext
1. 选中视频片段 -> 2. 点击“定格”按钮 -> 3. 编辑生成的静态片段（调时长、加特效）
```
你可以通过拖动时间轴上的选框，自由设置这个静态画面的持续时间，非常灵活。
