## 素材导入与基础调整 🎬

![](img/c9963d814ce835f042c7a0fe209b966a_11.png)

![](img/c9963d814ce835f042c7a0fe209b966a_13.png)

首先，打开手机版剪映App，点击“开始创作”按钮。

以下是导入素材的步骤：
1.  从相册中选择需要剪辑的视频片段。
2.  点击“添加”按钮，将视频导入到剪辑轨道中。
3.  如果视频由多个片段组成，建议先导入并剪辑完第一个片段，再点击轨道上的“+”号添加后续片段，这样可以保持工程清晰，避免混乱。

![](img/c9963d814ce835f042c7a0fe209b966a_15.png)

导入后，第一步是检查并调整视频的构图与比例。

![](img/c9963d814ce835f042c7a0fe209b966a_17.png)

以下是调整视频的步骤：
1.  点击轨道上的视频片段，向右滑动底部工具栏。
2.  找到并点击“编辑”按钮（图标通常是一个方块）。
3.  点击“旋转”功能，将视频画面调整到正确的方向。
4.  如需裁剪画面比例，可点击“裁剪”功能，选择如1:1、4:3、9:16等预设比例，或手动调整。

![](img/c9963d814ce835f042c7a0fe209b966a_19.png)

![](img/c9963d814ce835f042c7a0fe209b966a_21.png)

## 视频剪辑与修剪 ✂️

![](img/c9963d814ce835f042c7a0fe209b966a_23.png)

![](img/c9963d814ce835f042c7a0fe209b966a_25.png)

调整好画面后，接下来需要修剪掉视频中无用的部分，例如开头长时间的沉默或结尾不必要的动作。

![](img/c9963d814ce835f042c7a0fe209b966a_27.png)

![](img/c9963d814ce835f042c7a0fe209b966a_29.png)

核心剪辑操作遵循一个简单的公式：**选中 → 分割 → 选中 → 删除**。

![](img/c9963d814ce835f042c7a0fe209b966a_31.png)

![](img/c9963d814ce835f042c7a0fe209b966a_33.png)

以下是修剪视频的步骤：
1.  播放视频，找到需要剪掉的起始位置，将时间线（白色竖线）移动到此。
2.  选中视频片段，点击底部工具栏的“分割”按钮。视频会被切成两段。
3.  选中需要删除的那一段视频，点击“删除”按钮。
4.  重复此操作，修剪掉所有不需要的片段，使视频内容紧凑流畅。

![](img/c9963d814ce835f042c7a0fe209b966a_35.png)

## 添加与编辑背景音乐 🎵

![](img/c9963d814ce835f042c7a0fe209b966a_37.png)

![](img/c9963d814ce835f042c7a0fe209b966a_39.png)

视频内容修剪完毕后，需要为其添加合适的背景音乐以增强氛围。

![](img/c9963d814ce835f042c7a0fe209b966a_41.png)

以下是添加音乐的步骤：
1.  点击底部一级工具栏的“音频”按钮。
2.  如果你已有下载好的配音或音乐文件，可以点击“提取音乐”，从手机视频中提取音频。
3.  若没有现成音乐，可点击“音乐”，进入剪映自带的音乐库。库内音乐按“美食”、“vlog”、“流行”等场景分类，可以预览后选择。
4.  选中喜欢的音乐，点击“使用”即可添加到轨道。

![](img/c9963d814ce835f042c7a0fe209b966a_43.png)

![](img/c9963d814ce835f042c7a0fe209b966a_45.png)

添加后，音乐可能过长，需要修剪以匹配视频长度。

![](img/c9963d814ce835f042c7a0fe209b966a_47.png)

编辑音乐的方法与剪辑视频类似：
1.  选中音频轨道，将时间线移到需要剪掉的位置。
2.  点击“分割”，然后选中多余部分并“删除”。
3.  可以选中音频轨道，点击“音量”按钮，调整音乐声音的大小，使其不掩盖人声。

![](img/c9963d814ce835f042c7a0fe209b966a_49.png)

![](img/c9963d814ce835f042c7a0fe209b966a_51.png)

## 自动识别与添加字幕 📝

![](img/c9963d814ce835f042c7a0fe209b966a_53.png)

为视频添加字幕能极大提升观看体验。剪映提供了高效的自动识别功能，无需手动打字。

![](img/c9963d814ce835f042c7a0fe209b966a_55.png)

以下是添加字幕的步骤：
1.  点击底部一级工具栏的“文本”按钮。
2.  选择“识别字幕”功能。
3.  在弹出窗口中，选择“仅视频”（避免识别到背景音乐里的歌词），然后点击“开始识别”。
4.  系统会自动生成字幕并匹配到时间线上。

![](img/c9963d814ce835f042c7a0fe209b966a_57.png)

识别完成后，还需对字幕进行优化。

![](img/c9963d814ce835f042c7a0fe209b966a_59.png)

![](img/c9963d814ce835f042c7a0fe209b966a_61.png)

以下是优化字幕的步骤：
1.  检查是否有错别字，双击字幕条即可进行修改。
2.  可以统一调整字幕的样式，如字体、颜色、大小等，使其与视频风格更搭配。

## 视频导出与清晰度设置 ⚙️

![](img/c9963d814ce835f042c7a0fe209b966a_63.png)

![](img/c9963d814ce835f042c7a0fe209b966a_65.png)

所有剪辑工作完成后，即可导出视频。在导出前，需要进行关键设置以保证视频清晰度。

![](img/c9963d814ce835f042c7a0fe209b966a_67.png)

导出设置的核心参数是：
*   **帧率 (FPS)**：设置为 **30**。这是抖音平台的推荐帧率，能保证视频流畅。
*   **分辨率**：尽可能选择最高选项（如1080p或4K）。分辨率越高，视频越清晰。

![](img/c9963d814ce835f042c7a0fe209b966a_69.png)

以下是导出视频的步骤：
1.  点击右上角的“导出”按钮。
2.  在导出设置中，将帧率调整为30，分辨率拉至最高。
3.  点击“导出”按钮，在导出过程中请勿锁屏或切换应用，耐心等待完成。
4.  导出后，可以直接通过剪映分享到抖音平台。
