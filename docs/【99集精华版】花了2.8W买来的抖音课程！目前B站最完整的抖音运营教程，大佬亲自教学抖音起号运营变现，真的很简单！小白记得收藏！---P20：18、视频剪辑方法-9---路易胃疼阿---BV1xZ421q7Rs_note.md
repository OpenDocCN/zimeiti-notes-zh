## 剪辑软件选择与素材导入

上一节我们介绍了视频策划的思路，本节中我们来看看如何将素材变成成品。剪辑环节采用直接实操的方式进行，请务必跟随操作并做好记录。

![](img/74d0ab234f385751f3b487effb476075_7.png)

我们使用“剪映”App进行剪辑，这是抖音官方的剪辑工具，请确保下载的是黑色图标版本。它提供手机版和电脑版，本教程以手机版为例进行讲解。

![](img/74d0ab234f385751f3b487effb476075_9.png)

![](img/74d0ab234f385751f3b487effb476075_11.png)

操作步骤如下：

1.  打开剪映App，点击“开始创作”按钮。
2.  从相册中选择需要剪辑的视频素材，点击“添加”将其导入工程。

如果你的视频由多个生活片段（如开头、中间、结尾）组成，建议先导入第一条进行剪辑。完成第一条的剪辑后，再点击轨道上的“+”号，导入第二条素材。这样逐条导入和剪辑，可以避免混乱。

## 画面调整与基础剪辑

![](img/74d0ab234f385751f3b487effb476075_19.png)

导入素材后，我们首先检查并调整画面的构图与比例。

![](img/74d0ab234f385751f3b487effb476075_21.png)

![](img/74d0ab234f385751f3b487effb476075_23.png)

**第一步：调整画面方向与比例**
选中视频轨道，向右滑动底部工具栏，找到并点击“编辑”图标（一个方框形状）。
*   点击“旋转”功能，将画面调整到合适的正向角度。
*   如需裁剪画面比例，可点击“裁剪”功能，选择如1:1、4:3、16:9等预设比例，或手动调整。

![](img/74d0ab234f385751f3b487effb476075_25.png)

调整完毕后，点击返回键回到主编辑界面。

![](img/74d0ab234f385751f3b487effb476075_27.png)

**第二步：剪辑冗余片段**
播放视频，检查是否有长时间静音或无意义的片段，这些部分需要剪掉。
剪辑的核心操作是“分割”与“删除”：
*   将时间轴（白色竖线）移动到需要裁剪的位置。
*   选中视频轨道，点击底部工具栏的“分割”按钮，视频即被切成两段。
*   选中需要删除的片段，点击“删除”按钮。

![](img/74d0ab234f385751f3b487effb476075_29.png)

![](img/74d0ab234f385751f3b487effb476075_31.png)

重复此操作，可去除视频开头、中间或结尾所有不需要的部分。记住操作口诀：**`选中 -> 分割 -> 选中 -> 删除`**。

![](img/74d0ab234f385751f3b487effb476075_33.png)

## 添加与编辑背景音乐

![](img/74d0ab234f385751f3b487effb476075_37.png)

![](img/74d0ab234f385751f3b487effb476075_39.png)

视频画面剪辑完成后，接下来为其添加背景音乐。

![](img/74d0ab234f385751f3b487effb476075_41.png)

点击底部工具栏的“音频”按钮。
*   **方法一（提取音乐）**：如果你已使用其他软件（如淘金阁）为视频配好音，可点击“提取音乐”，直接导入视频文件中的音频。
*   **方法二（添加音乐）**：点击“添加音乐”，剪映内置了丰富的音乐库。你可以通过分类（如美食、美妆、萌宠）查找，或直接搜索歌曲名称。点击歌曲旁边的播放图标可进行预览，选定后点击“使用”即可添加。

![](img/74d0ab234f385751f3b487effb476075_43.png)

**编辑音乐**：
*   **剪辑长度**：如果音乐过长，操作与剪辑视频相同。将时间轴移至目标位置，选中音乐轨道，点击“分割”，再删除多余部分。
*   **调整音量**：选中音乐轨道，点击左侧出现的“音量”按钮，拖动滑块即可调整音乐声音大小。

## 自动添加与修改字幕

为视频添加字幕可以提升观看体验。剪映提供了高效的自动识别功能。

![](img/74d0ab234f385751f3b487effb476075_49.png)

![](img/74d0ab234f385751f3b487effb476075_51.png)

点击底部工具栏的“文本”按钮。
1.  选择“识别字幕”功能。
2.  在弹出窗口中，建议选择“仅识别视频原声”，以避免同时识别背景音乐里的歌词造成干扰。
3.  点击“开始识别”，系统会自动为视频中的语音生成字幕。

![](img/74d0ab234f385751f3b487effb476075_53.png)

识别完成后，字幕会出现在视频的相应位置。你需要进行以下检查与优化：
*   **修改错别字**：双击字幕条，即可对识别错误的文字进行修改。
*   **调整样式**：选中字幕条，可以使用底部工具栏的“样式”、“花字”、“动画”等功能，更改字体、颜色、大小等，使其与视频风格更匹配。

## 导出设置与发布

所有剪辑工作完成后，在导出前需进行关键设置，以保证视频清晰度。

![](img/74d0ab234f385751f3b487effb476075_59.png)

点击右上角的“导出”按钮。
在导出设置界面，请调整两个参数：
*   **帧率 (FPS)**：设置为 **`30`**。这是抖音平台最合适的帧率。
*   **分辨率**：尽可能选择最高选项（如1080p）。分辨率越高，视频越清晰。

![](img/74d0ab234f385751f3b487effb476075_61.png)

**核心导出公式**：**`帧率 = 30`， `分辨率 = 最高`**。

设置完成后，点击“导出”。**请注意：导出过程中请勿锁屏或切换至其他应用程序**，以免中断导出进程。导出完毕后，可以直接通过剪映的“分享到抖音”按钮进行发布。

## 口播提词神器推荐

![](img/74d0ab234f385751f3b487effb476075_67.png)

对于需要录制口播视频的同学，背诵稿子可能是一大难题。接下来推荐一款能提升效率的提词器软件。

![](img/74d0ab234f385751f3b487effb476075_69.png)

![](img/74d0ab234f385751f3b487effb476075_71.png)

这款软件名为 **“爱提词”**。它的核心功能是让你在录制时能看到台词，无需背诵。

![](img/74d0ab234f385751f3b487effb476075_73.png)

**使用方法如下**：
1.  打开App，点击右下角的“+”号。
2.  输入或粘贴你的口播文案，并设置一个标题（如“情感语录”），然后保存。
3.  在文案列表中点击该条目，选择“开始提词”或“录制模式”。
4.  此时手机屏幕会悬浮显示你的台词，你可以一边看一边录制。你可以在设置中调整**字体大小**和滚动速度。
5.  该App还内置了**美颜功能**（如瘦脸、大眼、磨皮），拍摄效果比原相机更美观自然，且不会过度失真。
6.  此外，App内还提供了“爆款文案”版块，涵盖职场、情感、生活等多个领域，可供你寻找灵感和参考。
