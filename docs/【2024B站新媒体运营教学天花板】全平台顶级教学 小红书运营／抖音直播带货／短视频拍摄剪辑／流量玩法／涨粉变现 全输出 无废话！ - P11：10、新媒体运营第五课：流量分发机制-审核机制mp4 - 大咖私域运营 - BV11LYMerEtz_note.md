在本节课中，我们将深入探讨新媒体平台的内容流量分发机制。具体来说，我们将了解一个视频或笔记从发布到最终获得流量的完整过程，包括内容审核、特征提取、数据测试以及流量分发等核心环节。

## 内容审核机制 🔍

上一节我们介绍了内容定位，本节中我们来看看内容发布后的第一道关卡：审核。所有平台的内容审核主要分为两大块：**机器审核**和**人工审核**。绝大部分内容的前期审核工作由机器完成。

![](img/3d2c598b60f139b34589e280565697b9_13.png)

![](img/3d2c598b60f139b34589e280565697b9_15.png)

### 机器审核

![](img/3d2c598b60f139b34589e280565697b9_17.png)

笔记或视频发布后，首先由机器审核内容是否合规。合规性检查主要包括以下两点：

![](img/3d2c598b60f139b34589e280565697b9_19.png)

![](img/3d2c598b60f139b34589e280565697b9_21.png)

1.  **敏感词检测**：系统会检查你的图文、口播文案或整个内容页面是否包含平台禁止的敏感词汇。敏感词范围广泛，包括违反广告法的词汇、法律禁止的词汇、政治敏感内容以及平台明确禁止的“黑五类”赛道内容。如果存在敏感词，内容可能被直接限流、审核不通过甚至打回。
    *   **核心概念示例（伪代码）**:
        ```python
        if content.contains(sensitive_words_list):
            handle_violation() # 处理违规，如限流或打回
        ```

![](img/3d2c598b60f139b34589e280565697b9_23.png)

![](img/3d2c598b60f139b34589e280565697b9_25.png)

![](img/3d2c598b60f139b34589e280565697b9_27.png)

2.  **营销广告检测**：平台（尤其是基于兴趣分发的短视频平台和基于种草分享的图文平台）不鼓励直接的硬广内容。如果笔记中直接宣传产品、服务、价格优势等，容易被判定为营销广告，导致前期不投流的情况下流量受损。因此，初期内容应以分享和价值提供为主。

在通过以上两点检测后，系统会进行**去重检测**。

![](img/3d2c598b60f139b34589e280565697b9_29.png)

**去重检测**是指平台将你的内容特征与其数据库中的历史内容进行比对。如果发现高度或完全重合，则判定为抄袭。抄袭的后果通常是限流或打回。因此，创作时应借鉴他人优秀价值点并进行调整、优化或使用AI工具改写，避免直接复制。

### 人工审核

![](img/3d2c598b60f139b34589e280565697b9_31.png)

![](img/3d2c598b60f139b34589e280565697b9_33.png)

人工审核主要在两种情况下介入：

![](img/3d2c598b60f139b34589e280565697b9_35.png)

![](img/3d2c598b60f139b34589e280565697b9_37.png)

![](img/3d2c598b60f139b34589e280565697b9_39.png)

1.  **机器审核存疑时**：当机器审核发现内容可能存在敏感词或违规风险时，会拉长审核时间，并将内容提交给人工审核员进行最终判断。通常，被提交到人工审核的内容通过率较低。
2.  **日常内容排查**：平台会有审核人员定期主动巡查站内内容。你可能遇到之前发布正常的笔记，在后期被判定违规的情况，这可能是由于用户举报或人工排查时发现了问题。

![](img/3d2c598b60f139b34589e280565697b9_41.png)

![](img/3d2c598b60f139b34589e280565697b9_43.png)

![](img/3d2c598b60f139b34589e280565697b9_45.png)

![](img/3d2c598b60f139b34589e280565697b9_47.png)

**为什么需要人工审核？** 因为任何平台的系统审核规则都存在漏洞。一些内容可能通过特殊方式绕过机器审核，这就需要后期的人工筛查或用户举报来弥补。了解这一点，运营人员可以在规则范围内探索一些可行的内容策略。

![](img/3d2c598b60f139b34589e280565697b9_49.png)

## 内容分发流程 🚀

内容审核通过后，下一步就进入面向用户的分发阶段。这个过程决定了你的内容会被推送给谁。

![](img/3d2c598b60f139b34589e280565697b9_51.png)

### 第一步：特征提取与初步分类

系统首先会分析你的笔记内容，提取特征，并尝试进行初步分类，以确定测试推送的人群。分类依据主要来自三个方面：

![](img/3d2c598b60f139b34589e280565697b9_53.png)

![](img/3d2c598b60f139b34589e280565697b9_55.png)

![](img/3d2c598b60f139b34589e280565697b9_57.png)

1.  **关键词分析**：系统通过语义分析、中文分词等技术，从你的标题、文案、语音识别出的字幕、封面及内容图中的文字提取核心主题关键词。
2.  **Tag标签**：发布时手动添加的话题或标签为系统提供参考价值，但并非唯一判定标准。
3.  **账号历史数据**：系统会参考你账号以往发布内容的数据。如果历史内容垂直度高（例如都是护肤主题），且获得了相关兴趣用户的互动，那么新内容在分类时会更倾向于该垂直领域。这就是为什么强调**账号垂直**的重要性。

![](img/3d2c598b60f139b34589e280565697b9_59.png)

![](img/3d2c598b60f139b34589e280565697b9_61.png)

### 第二步：数据测试与内容评估

完成初步分类后，系统会将内容分发给一小部分用户进行数据测试。通过这部分用户的行为反馈，系统判断内容质量并决定后续流量分配。

![](img/3d2c598b60f139b34589e280565697b9_63.png)

![](img/3d2c598b60f139b34589e280565697b9_65.png)

评估内容质量主要通过以下四个层面的用户行为数据：

![](img/3d2c598b60f139b34589e280565697b9_67.png)

![](img/3d2c598b60f139b34589e280565697b9_69.png)

![](img/3d2c598b60f139b34589e280565697b9_71.png)

以下是衡量内容质量的核心用户行为指标：

![](img/3d2c598b60f139b34589e280565697b9_73.png)

![](img/3d2c598b60f139b34589e280565697b9_75.png)

![](img/3d2c598b60f139b34589e280565697b9_77.png)

*   **点击率 (CTR)**：`点击率 = (点击用户数 / 曝光用户数) * 100%`。它反映了内容封面和标题的吸引力。
*   **停留时长/平均播放时长**：用户观看内容的时间长度。对于图文笔记，还包括用户翻阅页面、查看评论等行为的综合停留时间。
*   **页面浏览深度 (PV)**：用户从点击内容到浏览主页、查看其他笔记等所访问的页面数量。浏览页面越多，说明用户对账号越感兴趣，为账号带来的数据增益越高。
*   **互动行为**：包括点赞、收藏、评论、分享、关注等。用户完成其中任何一项，都对内容有积极的加权作用。

![](img/3d2c598b60f139b34589e280565697b9_79.png)

![](img/3d2c598b60f139b34589e280565697b9_81.png)

![](img/3d2c598b60f139b34589e280565697b9_83.png)

![](img/3d2c598b60f139b34589e280565697b9_85.png)

系统通过分析这些数据，综合评估内容的优劣。因此，在内容创作前期，就需要思考如何优化以提升这些关键数据。

![](img/3d2c598b60f139b34589e280565697b9_87.png)

![](img/3d2c598b60f139b34589e280565697b9_89.png)

![](img/3d2c598b60f139b34589e280565697b9_91.png)

## 主要流量入口 🚪

![](img/3d2c598b60f139b34589e280565697b9_95.png)

![](img/3d2c598b60f139b34589e280565697b9_97.png)

了解内容如何分发后，我们还需要知道用户从哪里看到内容。以某图文笔记平台为例，主要流量入口如下：

![](img/3d2c598b60f139b34589e280565697b9_99.png)

以下是平台主要的几个流量来源：

![](img/3d2c598b60f139b34589e280565697b9_101.png)

*   **发现页/首页**：平台根据用户标签和站内热门内容进行混合推荐的页面，是最大的公域流量池。用户在此基于兴趣浏览。
*   **关注页**：用户在此查看已关注博主的新内容，目的性非常强，属于私域流量。
*   **同城**：用户有明确本地需求（如找美食、住宿）时会使用的入口，需求精准。
*   **搜索入口**：用户主动表达明确需求（如搜索“xx教程”、“xx测评”）的入口，流量意图明确。
*   **购物入口**：用户带有明确购物意图的入口，是电商转化的主要场景。

本节课中，我们一起学习了新媒体平台流量分发与审核的核心机制。我们了解到，内容发布后需先通过**机器审核**（敏感词、营销广告、去重检测）和可能的**人工审核**。审核通过后，系统会进行**特征提取与初步分类**，然后将内容进行小范围的**数据测试**，依据**点击率、停留时长、页面浏览深度和互动行为**等数据评估内容质量，最终决定是否将其推向更大的**流量入口**（如发现页、关注页、搜索等）。理解这套机制，能帮助我们有针对性地优化内容，提升流量获取效率。
