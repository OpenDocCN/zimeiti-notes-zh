# 拼多多直通车教程：P1：成交出价与投产比的核心逻辑 🚀

在本节课中，我们将学习拼多多直通车的一个关键高级技巧：为什么新计划必须优先开启“成交出价”模式，而非直接使用“投产比”模式。理解这两种模式的扣费逻辑差异，是避免广告浪费、成功测试产品的第一步。

## 核心原则：先测后投

对于新手而言，启动直通车广告时，首要步骤是选择“成交出价”模式，而非直接设置“投产比”。这样做的目的并非因为先开成交出价再转投产比的效果更优，而是为了利用成交出价模式来为你的产品进行有效测试。

## 两种扣费逻辑详解

上一节我们介绍了优先使用成交出价的原则，本节中我们来详细看看成交出价与投产比在扣费逻辑上的根本区别。

### 成交出价模式

在成交出价模式下，你设定的出价就是每次成交的固定广告花费。

*   **公式**：`单笔成交广告花费 = 成交出价`
*   **示例**：若你设置成交出价为10元，则无论用户购买的是低价引流款（如2元）还是高利润款（如50元），系统每达成一笔订单，都固定扣除10元广告费。

这种模式的优点是广告花费稳定，因此系统分配的广告排名和流量资源位也相对稳定。

### 投产比模式

在投产比模式下，广告花费与订单的实际成交金额直接挂钩，是浮动扣费。

*   **公式**：`单笔成交广告花费 = 订单成交金额 / 投产比`
*   **示例**：若你设置投产比为2，当用户购买50元的高价规格时，广告花费为 `50 / 2 = 25元`；当用户购买2元的低价规格时，广告花费则仅为 `2 / 2 = 1元`。

这种模式会导致广告花费随订单金额剧烈波动，进而严重影响流量质量。

## 流量质量与模式选择的关系

理解了扣费逻辑后，我们来看看它如何直接影响广告效果。在拼多多直通车体系中，一个核心规则是：**广告出价越高，获得的排名就越高，流量的质量和数量也越好**，从而带来更高的点击率和转化率。

以下是两种模式在不同订单情况下的表现对比：

*   **情况一：用户购买高价规格**
    *   **成交出价**：固定扣费（如10元），排名稳定，流量优质。
    *   **投产比**：扣费高昂（如25元），排名很高，流量大但成本也高。

*   **情况二：用户购买低价规格**
    *   **成交出价**：固定扣费（如10元），排名稳定，但可能因商品利润低而亏损。
    *   **投产比**：扣费极低（如1元），排名非常低，只能获得少量劣质流量或几乎没有流量。

## 实操步骤：测试与优化

基于以上逻辑，正确的操作流程不是盲目投放，而是先进行测试。

**第一步：用成交出价模式测试产品**
通过开启成交出价计划，观察数据，判断你的产品用户更倾向于购买低价规格还是高价规格。

**第二步：根据测试结果决策**
测试后，你将面临两种情况，需要采取不同策略：
1.  **情况A：用户主要购买低价规格**
    *   这表明如果直接使用投产比模式，广告排名会极低，无法获取有效流量。此类产品很难通过广告盈利。
    *   **应对策略**：你需要优化商品页面的规格描述与布局，引导用户转向购买高客单价规格。在产品完成优化前，不宜大力推广。

2.  **情况B：用户主要购买高价规格**
    *   这表明你的链接拥有健康的高客单成交倾向。
    *   **应对策略**：此时，将这个链接的广告计划从“成交出价”转为“投产比”模式，才有机会实现稳定出单和盈利。

## 总结

本节课中我们一起学习了拼多多直通车“先成交出价，后投产比”的核心技巧。关键点在于：**成交出价模式用于固定成本测试产品真实的客单价结构；只有测试确认用户主要购买高客单规格后，转为投产比模式才有盈利可能**。请务必记住这个逻辑，它能帮助你在投放初期避免重大损失，并找到真正有潜力的推广产品。
