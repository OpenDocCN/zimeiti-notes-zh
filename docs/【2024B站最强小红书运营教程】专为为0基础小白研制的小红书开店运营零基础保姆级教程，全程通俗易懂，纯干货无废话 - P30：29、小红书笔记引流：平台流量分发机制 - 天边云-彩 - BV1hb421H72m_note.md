# 小红书运营教程：P30：小红书笔记引流：平台流量分发机制 📊

![](img/cc8afc10c1b769dd52230e92dcdc104f_1.png)

在本节课中，我们将要学习小红书平台的核心流量分发机制。了解这套算法，是提升笔记曝光和排名的关键。我们将拆解其评分系统、展示规则和流量推送逻辑，帮助你从底层理解如何优化内容以获得更多流量。

## 概述：流量分发的核心逻辑

小红书平台的流量分发并非随机，而是基于一套综合评分系统。这套系统决定了你的笔记在用户信息流中的排名和展示次数。理解它，是做好小红书运营的第一步。

## 基础评分系统：S1CCES

上一节我们介绍了流量分发的概念，本节中我们来看看其核心的计算系统。小红书通过一套名为 **S1CCES** 的评分系统来评估笔记质量，并据此进行排名。S1CCES代表的是用户对笔记的几种关键交互行为。

以下是构成S1CCES评分系统的具体行为及其对应分值：

*   **点赞**：1分
*   **收藏**：1分
*   **转发**：4分
*   **评论**：10分
*   **关注**：8分

当一个用户完成从点赞到关注的全部行为时，该笔记就能从这一个用户身上获得 **18分** 的综合评分。系统会累计所有用户的交互分数，作为笔记排名的重要依据。

## 展示与排名机制

了解了评分系统后，我们来看看笔记是如何被展示和排序的。小红书的展示页面与抖音等单条下滑模式不同，它采用分页刷新的形式。

*   小红书信息流每页展示 **20篇** 笔记，用户需要下滑才能加载下一页（即第21-40名）。
*   这20个位置中存在少量付费推广笔记（类似“直通车”）。
*   系统的目标就是让你的笔记排名进入当前页面的 **前20名**，尤其是前几名，以获得最大曝光。

你的笔记排名由 **CES综合评分** 决定。评分越高，在当前20篇笔记中的排名就越靠前。

## 流量推送的阶梯流程

排名决定了展示位置，而流量推送则是一个动态、分阶段的过程。一篇笔记发布后，并不会一次性获得所有流量。

笔记发布后，系统会先给予一个**基础曝光量**（例如1000次）。这个初始流量会分几波进行推送，例如第一波400，第二波300等。能否获得后续更大流量的关键，在于第一波流量中的数据表现。

以下是影响能否进入下一波推流的核心数据指标：

1.  **点击率**：用户是否被你的封面和标题吸引并点击。这是所有后续互动的前提。
2.  **互动率**：用户点击后，是否产生点赞、收藏、评论等S1CCES行为。
3.  **完播率**：对于视频笔记，用户是否完整观看了内容。
4.  **互动值**：账号作者与评论用户之间的回复互动频率和质量。

如果笔记在第一波推广中点击率和互动率等数据达标，系统就会在下一波给予更多流量扶持，形成**阶梯式推荐**。反之，如果数据不佳，后续流量就会减少甚至停止，笔记的生命周期可能只有短短几天。

## 算法核心与公平性

无论平台的数据指标如何微调，其核心算法始终围绕一点：**考察笔记带来的粉丝交互行为**。互动率直接反映了内容质量。

即使是一个粉丝数不多的新账号，只要单篇笔记内容优质，能引发高质互动并获得高S1CCES评分，同样有机会被系统推荐给更多用户。

小红书平台在这方面相对公平。通常，一篇自主创作的笔记，最基础的展示量也能达到 **150-200次**（即“零层级”曝光）。我们要做的，就是通过优化内容，在这个基础上不断撬动系统的阶梯推荐机制。

## 总结

![](img/cc8afc10c1b769dd52230e92dcdc104f_3.png)

本节课中我们一起学习了小红书的流量分发机制。我们了解到，系统通过 **S1CCES评分系统**（点赞、收藏、转发、评论、关注）来评估笔记，并据此进行分页排名。流量的推送是一个基于**点击率、互动率、完播率和互动值**等数据的阶梯式过程。算法的核心始终是**用户交互行为**。掌握这些原理，就能有针对性地优化封面、标题和内容，提升笔记的排名与曝光，从而获得更多流量。
