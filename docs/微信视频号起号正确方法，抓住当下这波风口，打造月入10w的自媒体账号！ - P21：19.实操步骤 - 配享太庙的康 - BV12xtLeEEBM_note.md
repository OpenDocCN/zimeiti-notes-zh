# 微信视频号运营教程：P21：19.实操步骤 - 开通与上架商品 🛒

在本节课中，我们将学习微信视频号开通商品橱窗权限以及上架商品的具体操作步骤。整个过程简单直接，即使是初学者也能轻松跟上。

上一节我们介绍了视频号的基础概念，本节中我们来看看如何实际操作，迈出视频号带货的第一步。

## 第一步：打开微信
首先，在您的手机上找到并打开微信应用。

![](img/876a50db8acd5b4c02e5e119e7f7185c_1.png)

## 第二步：进入“发现”页
打开微信后，点击底部导航栏的 **“发现”** 选项。

![](img/876a50db8acd5b4c02e5e119e7f7185c_2.png)

## 第三步：找到视频号入口
在“发现”页面中，将视线向上移动。您会看到 **“视频号”** 的入口，其图标是一个黄色的蝴蝶形状。点击此处进入视频号。

![](img/876a50db8acd5b4c02e5e119e7f7185c_1.png)
![](img/876a50db8acd5b4c02e5e119e7f7185c_2.png)

![](img/876a50db8acd5b4c02e5e119e7f7185c_4.png)

![](img/876a50db8acd5b4c02e5e119e7f7185c_6.png)

## 第四步：进入个人中心
进入视频号后，点击页面右上角的 **人形头像** 图标，进入个人中心。

![](img/876a50db8acd5b4c02e5e119e7f7185c_4.png)
![](img/876a50db8acd5b4c02e5e119e7f7185c_6.png)

## 第五步：进入创作者中心
在个人中心页面，向下滑动屏幕，直到最底部。找到并点击 **“创作者中心”**。

![](img/876a50db8acd5b4c02e5e119e7f7185c_8.png)

![](img/876a50db8acd5b4c02e5e119e7f7185c_9.png)

> **注意**：如果您是新注册的账号且未发布过作品，可能暂时看不到此入口。发布一条视频后，该入口便会显示。

## 第六步：找到带货中心
进入创作者中心后，您会看到数据面板等功能。在其中找到 **“带货中心”** 的入口（图标类似一个小卖部），并点击进入。

![](img/876a50db8acd5b4c02e5e119e7f7185c_8.png)
![](img/876a50db8acd5b4c02e5e119e7f7185c_9.png)

接下来，我们将分为两种情况讲解：**开通橱窗权限** 和 **上架商品**。

### 情况一：开通商品橱窗权限
如果您是首次使用，点击“带货中心”后会跳转到开通权限的界面。

![](img/876a50db8acd5b4c02e5e119e7f7185c_11.png)

以下是开通权限的具体步骤：
1.  **选择带货身份**：系统会要求您选择身份。请务必选择 **“我是达人”** 这个选项。
    > **核心提示**：即使您是厂家或拥有自有品牌，初期也建议先选择“达人”身份，以便使用平台的选品中心。一个身份证可以注册两个账号，后续可用第二个账号选择“商家”身份。
    ![](img/876a50db8acd5b4c02e5e119e7f7185c_11.png)

2.  **交纳保证金**：选择身份后，会跳转到交纳保证金的页面。您需要点击 **“去交纳”**，系统将通过微信支付扣除 **100元** 的保证金。
    ![](img/876a50db8acd5b4c02e5e119e7f7185c_13.png)

完成以上两步，您的商品橱窗权限即开通成功。

### 情况二：上架商品
如果您已经开通了橱窗权限，再次点击“带货中心”将直接进入商品管理界面，您可以在此进行选品和上架操作。

![](img/876a50db8acd5b4c02e5e119e7f7185c_13.png)

整个操作流程的路径可以总结为以下代码逻辑：
```plaintext
打开微信 -> 点击“发现” -> 进入“视频号” -> 点击右上角头像 -> 进入“创作者中心” -> 点击“带货中心”
```
- 若未开通：选择“我是达人” -> 交纳100元保证金 -> 开通成功。
- 若已开通：直接进入选品上架界面。

本节课中我们一起学习了从零开始开通微信视频号商品橱窗的完整流程。关键点在于通过“创作者中心”进入“带货中心”，并根据提示完成身份选择与保证金交纳。操作本身并不复杂，掌握后您就具备了在视频号进行电商带货的基础条件。
