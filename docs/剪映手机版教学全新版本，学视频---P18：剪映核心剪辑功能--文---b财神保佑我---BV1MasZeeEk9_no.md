# 剪映手机版教学：P18：剪映核心剪辑功能--文本功能 📝

在本节课中，我们将要学习剪映的核心功能之一：文本功能。一个完整的视频除了画面和音频，文本或字幕也是传递信息、突出主题的关键元素。通过添加文本，我们可以让观众更快速、更准确地理解视频内容。

上一节我们介绍了音频功能，本节中我们来看看如何为视频添加和美化文字。

![](img/8c5d1eae9def322e5bfa94089bf93acd_1.png)

## 文本功能概述

在剪映中，我们可以自由地为视频添加字幕。添加的字幕可以单独设置样式，包括字体、字号、颜色、间距以及动画效果。

剪映还提供了两个高效的功能：**字幕自动识别**和**歌词识别**。当视频配有画外音或歌曲时，可以使用这两个功能一键快速生成字幕，极大地降低了操作难度和时间成本。

此外，文本功能中还可以添加有趣的贴纸来丰富画面。

关于文本功能的基本介绍就到这里，接下来我们进入功能实操环节。

## 功能实操演练

文本功能位于一级菜单栏。点击底部菜单栏的 **`文字`** 按钮，即可看到四个主要选项：`新建文本`、`识别字幕`、`识别歌词`和`贴纸`。

我们将通过以下步骤详细学习每个功能。

### 1. 新建文本（手动添加字幕）

手动添加字幕是最基础的方式。以下是操作步骤：

1.  在剪辑界面点击底部菜单栏的 **`文字`**。
2.  选择 **`新建文本`**。
3.  在输入框中输入想要的文字内容，例如“剪映视频剪辑教程”。
4.  输入完成后，时间轴上会新增一个文字轨道，屏幕上会显示输入的文字。

文字添加后，我们可以对其样式进行个性化设置。点击选中文字轨道，底部会出现编辑菜单：

![](img/8c5d1eae9def322e5bfa94089bf93acd_3.png)

*   **样式**：可以设置文字的**字体**、**描边**、**标签**、**阴影**、**字间距**和对齐方式。
    *   例如，选择“新青年体”字体，并为其添加一个**红色描边**。
*   **花字**：抖音官方提供的各种特色字体，适用于种草、搞笑等风格的视频。
*   **气泡**：提供对话形式的文字气泡样式，常见于Vlog或采访视频。
*   **动画**：可以为文字添加**入场动画**、**出场动画**，让文字动起来，使视频更生动。
    *   例如，为文字添加一个“螺旋上升”的入场动画，并设置动画持续时间。

设置完成后，点击预览即可查看效果。

### 2. 识别字幕（为语音自动添加字幕）

![](img/8c5d1eae9def322e5bfa94089bf93acd_5.png)

![](img/8c5d1eae9def322e5bfa94089bf93acd_6.png)

如果视频本身带有语音解说或对话，可以使用“识别字幕”功能自动生成字幕。

1.  确保视频轨道上已添加包含人声的音频。
2.  点击 **`文字`** -> **`识别字幕`**。
3.  点击 **`开始识别`**。
4.  系统会自动识别音频中的语音，并生成对应的文字轨道。

识别完成后，可以像编辑“新建文本”一样，对自动生成的字幕进行样式和位置的调整。

![](img/8c5d1eae9def322e5bfa94089bf93acd_8.png)

### 3. 识别歌词（为歌曲自动添加歌词）

![](img/8c5d1eae9def322e5bfa94089bf93acd_10.png)

![](img/8c5d1eae9def322e5bfa94089bf93acd_12.png)

如果视频添加了带有歌词的歌曲，可以使用“识别歌词”功能。

![](img/8c5d1eae9def322e5bfa94089bf93acd_14.png)

1.  为视频添加一首歌曲（例如，从“抖音收藏”中添加）。
2.  点击 **`文字`** -> **`识别歌词`**。
3.  点击 **`开始识别`**。
4.  系统会自动识别歌曲，并将歌词匹配到时间轴上。

识别出的歌词同样可以进行美化。例如，调整歌词位置、更换字体、添加描边和动画效果，使其更符合视频的整体风格。

### 4. 贴纸

除了文字，在`文字`菜单中还可以选择`贴纸`，为视频添加各种有趣的图形元素，增加视频的趣味性和表现力。

## 课程总结

![](img/8c5d1eae9def322e5bfa94089bf93acd_19.png)

本节课中我们一起学习了剪映的文本功能。我们掌握了如何通过 **`新建文本`** 手动添加字幕，以及如何利用 **`识别字幕`** 和 **`识别歌词`** 功能高效地为语音和音乐自动生成字幕。同时，我们也学会了如何对字幕的字体、颜色、动画等进行个性化设置，并通过添加贴纸来丰富画面。

通过为视频添加精美的文本，可以大大提升视频内容的信息传达效率和整体质量。文本功能是视频剪辑中不可或缺的一环，希望大家能多加练习，灵活运用。
