# 视频号运营：P31：如何扒取直播间话术

在本节课中，我们将学习如何高效地扒取其他成功直播间的话术。掌握这个方法，你可以快速复制并优化他人的直播脚本，为自己的直播带货打下基础。

## 概述：为何需要学习话术扒取

上一节我们介绍了视频内容创作，本节中我们来看看如何优化直播环节的核心——话术。直播话术直接影响转化率。直接借鉴已被市场验证的成功话术，是快速入门和提升效率的有效方法。你可以复制那些卖得好、且你能低成本复制的产品直播间的话术。

## 工具准备：飞书妙计

我们需要使用一个工具来实现话术录制与转写：**飞书妙计**。它是飞书内置的一个功能，可以实时录制音频并转写成文字。

![](img/56f0b1a998d724c1ccdd43a875ef991f_1.png)

![](img/56f0b1a998d724c1ccdd43a875ef991f_3.png)

以下是使用飞书妙计的具体步骤：

1.  **下载并打开飞书App**。
2.  点击底部导航栏的 **“工作台”**。
3.  在工作台页面中，找到并点击 **“飞书妙计”** 应用。

![](img/56f0b1a998d724c1ccdd43a875ef991f_1.png)

## 操作流程：录制与导出话术

找到工具后，我们就可以开始录制目标直播间的话术了。

1.  **开始录制**：在飞书妙计界面，点击右下角的 **语音按钮**，即可开始录音。
2.  **打开目标直播间**：切换到微信，打开你想学习和模仿的直播间。飞书妙计会在后台持续录制手机发出的所有声音，包括直播间的音频。
3.  **录制时长**：建议录制**15分钟**左右。这个时长通常足以覆盖主播介绍产品、互动和逼单的关键环节。
4.  **结束并处理**：录制完成后，在飞书妙计中点击“完成”。系统会自动将音频转写成文字，这个过程可能需要几分钟。

![](img/56f0b1a998d724c1ccdd43a875ef991f_3.png)

## 话术整理：形成自己的脚本

录制转写完成后，下一步是将原始文本整理成可供自己使用的直播脚本。

1.  **导出文本**：在转写完成的文稿页面，点击 **“分享”** 按钮。选择 **“可编辑”** 或 **“可阅读”** 模式，然后复制生成的链接。
2.  **在微信中打开**：将复制的链接发送到微信（例如文件传输助手或自己的聊天框），然后在微信中点击打开。此时你就能看到完整的转写文字。
3.  **复制与提炼**：复制全部文字内容。你需要仔细阅读，提炼出其中的产品介绍逻辑、互动问答和促单话术。
4.  **编辑成稿**：将提炼出的核心话术，重新编辑成适合你自己语言风格的脚本。可以写在纸上，或保存在手机备忘录中。
5.  **开播使用**：直播时，如果忘记台词，可以对照这份整理好的话术进行讲述，确保直播流程顺畅。
