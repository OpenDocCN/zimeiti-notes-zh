# AI短视频创作教程：P10：7.视频剪辑实操（下）🎬

在本节课中，我们将继续学习视频剪辑的实操部分，重点讲解如何为多个视频片段添加转场效果，以及如何为视频添加字幕。这些步骤对于提升视频的流畅度和专业性至关重要。

上一节我们介绍了视频比例调整与背景设置，本节中我们来看看如何衔接不同的视频片段。

## 添加视频转场

![](img/751b6cf32e3c71b1f31a8360edb740d1_1.png)

我们的视频由多个约10秒的片段组成，并非一个完整的原视频。片段之间必须有衔接，因此需要添加转场效果。

![](img/751b6cf32e3c71b1f31a8360edb740d1_3.png)

转场的作用是让用户明确感知到每个片段内容是不同的。如果不添加转场，观众切换到下一个片段时可能会感到突兀，甚至退出视频查看。这会直接导致视频**完播率**下降，而完播率会影响视频的单价和最终收益。因此，添加转场是必要步骤。

![](img/751b6cf32e3c71b1f31a8360edb740d1_5.png)

以下是添加转场的具体操作：

![](img/751b6cf32e3c71b1f31a8360edb740d1_7.png)

![](img/751b6cf32e3c71b1f31a8360edb740d1_9.png)

1.  在视频轨道右侧，找到一个**加号（+）** 按钮。点击它。
2.  在弹出的界面中，选择**“素材库”**。
3.  在素材库的“热门”分类中，可以找到大量转场素材，例如各种动态效果。这些转场素材在众多视频中都很常见，可以放心使用。

![](img/751b6cf32e3c71b1f31a8360edb740d1_11.png)

在选择转场时，请注意一点：不要选择时长过长的转场，例如16秒的素材。观众通常没有耐心看完不重要的内容。尽量选择时长在**3到5秒**之间的转场，其中**3秒**时长的最为合适。

![](img/751b6cf32e3c71b1f31a8360edb740d1_13.png)

选择好转场素材并添加后，还需要确保其视频比例与主视频保持一致，即**16:9**的横屏比例。

添加好转场视频后，在编辑界面右下角可以找到**“画布模糊”**选项。点击它，并选择第一个“清晰的画面”效果。这样，转场视频的背景就会自动适配为16:9的横屏。最后，记得点击右下角的**对勾（✓）** 确认操作。

![](img/751b6cf32e3c71b1f31a8360edb740d1_15.png)

## 为视频添加字幕

![](img/751b6cf32e3c71b1f31a8360edb740d1_17.png)

完成转场设置后，下一步是为视频添加文案，也就是字幕。

以下是添加字幕的具体步骤：

![](img/751b6cf32e3c71b1f31a8360edb740d1_19.png)

1.  找到手机剪辑界面左下角的**“文字”**选项，点击它。
2.  在文字功能中，选择**“识别字幕”**。
3.  点击**“开始匹配”**。系统会自动识别视频或音频中人物所说的话，并生成对应的字幕文本。

![](img/751b6cf32e3c71b1f31a8360edb740d1_21.png)

字幕生成后，通常需要稍作调整以达到最佳视觉效果。

1.  首先，调整字幕大小。拖动字幕文本框右下角，向右拖动即可放大字幕。
2.  然后，调整字幕位置。将放大后的字幕文本框整体拖动到视频画面最下方的位置，这样观看体验最舒适。

调整好第一段字幕后，后续的字幕都会自动应用相同的样式和位置，无需重复调整，非常方便。

![](img/751b6cf32e3c71b1f31a8360edb740d1_23.png)

## 核心要点总结

本节课中我们一起学习了视频剪辑后半部分的核心操作。

![](img/751b6cf32e3c71b1f31a8360edb740d1_25.png)

*   **转场的作用**：衔接不同片段，提升观看流畅度，保障**完播率**。
*   **转场的选择**：从素材库“热门”中选取，时长控制在**3~5秒**。
*   **比例统一**：转场素材也需通过“画布模糊”功能调整为**16:9**横屏。
*   **字幕添加**：使用“识别字幕”功能自动生成，并调整大小和位置至视频底部。

![](img/751b6cf32e3c71b1f31a8360edb740d1_27.png)

请严格按照上述步骤操作。在寻找素材时，也务必注意不要使用过长的视频，以免在平台审核时无法通过。整个流程并不复杂，关键在于细心和遵循规范。
