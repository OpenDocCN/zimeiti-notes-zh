# 小红书电商运营课：P24：工具篇：手机剪映使用教程 📱✂️

在本节课中，我们将学习如何使用手机版剪映APP进行视频剪辑。这是制作小红书笔记视频的基础工具，掌握它能帮助你快速完成视频的二次创作与优化。

## 认识剪映APP界面

打开手机上的剪映APP，可以看到主界面提供了多种功能。

以下是几个核心功能的简介：
*   **一键成片**：导入多张图片，软件会自动生成一个带模板的视频。
*   **图文成片**：将图文内容转换为视频，这可以作为一种二次创作的方式。

我们最常用的是“一键成片”功能。接下来，我们将重点讲解如何进行具体的视频剪辑操作。

## 视频剪辑基础步骤

上一节我们认识了剪映的界面，本节中我们来看看如何对一个视频进行基础的剪辑。

**步骤1：导入素材并关闭原声**
点击“开始创作”，选择你需要剪辑的视频（无论是搬运的或一键合成的视频），然后点击“添加”。视频导入后，首先需要关闭其原始声音。点击轨道上的视频片段，在下方工具栏找到并点击“音量”图标，将音量调整至0。

**步骤2：进行基础混剪**
基础混剪主要包括调整视频长度和播放速度。
*   **裁剪（掐头去尾）**：选中视频片段，拖动首尾的白色边框，可以裁剪掉不需要的部分。
*   **变速**：点击下方工具栏的“变速”功能，可以调整视频的播放速度，轻微修改即可。

完成以上操作，视频就完成了一次简单的修改。

## 添加文字、贴纸与音乐

除了基础的剪辑，我们还可以为视频添加元素使其更丰富。

**添加文字与贴纸**
点击下方工具栏的“文字”或“贴纸”选项，可以为视频添加相应的元素。添加后，可以拖动调整其显示时长和位置。

以下是关于添加装饰元素的建议：
*   添加与否取决于你笔记的风格和对标账号的做法。
*   如果你的对标账号没有添加这些，你也不必多此一举。
*   追求简洁明了的风格往往效果更好。

**添加背景音乐**
由于我们关闭了原声，现在需要添加新的背景音乐。点击下方工具栏的“音频”，选择“音乐”，从曲库中挑选合适的音乐点击“使用”即可。

添加音乐后，务必对音乐进行裁剪。选中音频轨道，点击“分割”，删除多余的部分，使音乐长度与视频匹配。这是因为不裁剪可能导致视频结尾留有冗长的无声片段或突兀的结尾。

## 导出与注意事项

完成所有编辑后，点击右上角的“导出”按钮即可生成最终视频。

在操作过程中，请牢记以下核心要点：
*   确保视频流畅、简洁、有吸引力。
*   视频的前三秒和封面至关重要，它们决定了用户是否会停留观看。
*   一切编辑应以对标账号和自身风格为参考，避免不必要的复杂化。

本节课中我们一起学习了使用手机剪映进行视频剪辑的全流程，从导入素材、基础剪辑到添加元素和最终导出。掌握这些基础操作，你就能为小红书笔记制作出合格的视频内容了。
