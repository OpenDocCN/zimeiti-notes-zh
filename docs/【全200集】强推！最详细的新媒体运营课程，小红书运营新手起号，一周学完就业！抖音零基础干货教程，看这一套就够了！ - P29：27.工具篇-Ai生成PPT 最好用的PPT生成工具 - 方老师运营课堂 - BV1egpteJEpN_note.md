# 新媒体运营课程：P29：工具篇-AI生成PPT工具使用教程 📄

在本节课中，我们将学习如何使用一款AI工具，根据主题或现有文档自动生成PPT。这个工具能自动梳理内容大纲，并基于大纲生成详细的PPT页面。

## 工具简介与核心功能

上一节我们介绍了多种新媒体运营工具，本节中我们来看看这款AI生成PPT工具。该工具主要提供两种生成模式。

以下是两种核心工作模式：
1.  **主题生成模式**：输入一个主题，工具会结合类似ChatGPT的生成能力，自动创建完整的PPT内容。
2.  **文档生成模式**：上传一篇已有的文档（如TXT、PDF），AI助手会自动总结文档大纲，并基于此大纲生成PPT内容。

## 生成后编辑与优化

PPT内容生成完毕后，我们可以进入编辑界面进行调整和美化。

点击“查看和编辑”进入编辑模式。在这里，你可以看到AI生成的完整PPT内容。



![](img/d7130a51aab6fda40000288f653b3658_11.png)

工具的一个关键优势在于，它能将文档内容智能地归类到不同的大纲节点下，而不仅仅是简单地将所有文本堆砌到幻灯片上。

在编辑界面右侧，提供了多种调整选项。

以下是主要的编辑功能：
*   **更换模板**：工具提供多种PPT模板可供切换，选择更符合内容风格的样式。
*   **调整文字**：修改幻灯片上的文字内容、字体和大小。
*   **替换形状**：更改图表、图形等元素的样式。



![](img/d7130a51aab6fda40000288f653b3658_19.png)

![](img/d7130a51aab6fda40000288f653b3658_13.png)

如果对内置模板不满意，一个高效的工作流是：先使用工具生成内容和大体版式，然后下载到本地（如PowerPoint或Keynote中）进行深度调整和美化。
