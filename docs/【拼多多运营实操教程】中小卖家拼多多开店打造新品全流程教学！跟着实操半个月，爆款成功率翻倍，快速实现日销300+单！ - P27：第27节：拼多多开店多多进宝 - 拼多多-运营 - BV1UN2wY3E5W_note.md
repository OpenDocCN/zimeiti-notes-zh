# 拼多多运营实操教程：P27：拼多多开店多多进宝 💎

在本节课中，我们将要学习拼多多平台的一个重要推广工具——多多进宝。我们将了解它的核心作用、运作原理以及如何在后台上手操作，帮助你利用这个工具为店铺快速积累销量和流量。

## 什么是多多进宝？

上一节我们介绍了店铺的基础设置，本节中我们来看看一个关键的推广渠道。多多进宝是拼多多平台的一种营销工具。商家可以自主为商品设置佣金比例，激励推广者（即“多多客”）帮助分享和推广商品链接，从而实现以佣金换销量的目的。

## 多多进宝的核心作用

以下是多多进宝能为店铺带来的五个主要好处：

1.  **快速累积基础销量与权重**：通过推广产生初始销量，提升商品在平台内的基础权重。
2.  **累积搜索排名权重**：销量增长有助于提高商品在自然搜索中的排名。
3.  **快速打造爆款**：借助推广者的力量，可以加速潜力商品的曝光与成交，形成爆款。
4.  **扩宽流量渠道**：推广不仅限于站内，还能通过微信等社交平台进行站外引流，获取全方位流量。
5.  **为全店引流，提升知名度**：成功的推广可以吸引用户进入店铺，关注店铺，从而提升店铺整体知名度和用户粘性。

## 多多进宝的运作原理与规则

![](img/445e5234ba590b3ff2b50ff3dfbd1a7e_1.png)

了解了它的作用后，我们来看看它是如何具体运作的。其核心逻辑是 **“商家设置佣金 -> 推手推广 -> 成交后结算佣金”**。

![](img/445e5234ba590b3ff2b50ff3dfbd1a7e_2.png)

*   **佣金计算**：佣金基于商品的最终成交金额计算。公式为：`推手佣金 = 商品最终成交金额 × 设置佣金比例`。
    *   例如，商品售价100元，佣金比例设为10%，若无优惠券，成交一单推手获佣金 `100 × 10% = 10元`。
    *   若消费者使用了50元优惠券，则最终成交金额为50元，推手佣金为 `50 × 10% = 5元`。
*   **结算周期**：佣金每14天结算一次。消费者确认收货14天后，推手方可提现。
*   **退款规则**：
    *   确认收货14天内发生退款，佣金将退回，商家无需支付。
    *   确认收货14天后发生退款，佣金不再退回，由商家承担全部退款金额及已支付的佣金。

## 如何开通与设置多多进宝？

接下来，我们进入实操环节，学习如何在后台开通和设置多多进宝。

![](img/445e5234ba590b3ff2b50ff3dfbd1a7e_8.png)

1.  **进入后台**：登录拼多多商家后台，在页面底部找到“推广”相关入口，点击进入“多多进宝”首页。
2.  **开通并设置**：首次使用需先开通。一般建议初始佣金比例设置为10%。
3.  **添加推广商品**：开通后，点击“新建推广”或类似按钮，添加你想要推广的商品链接。
4.  **设置优惠券**：**请注意**，价格高于10元的商品必须设置优惠券才能推广。如果暂无可用优惠券，需在推广时同步创建新的店铺优惠券。
5.  **完成设置**：填写好佣金比例、优惠券等信息后，即可提交并开始推广。
