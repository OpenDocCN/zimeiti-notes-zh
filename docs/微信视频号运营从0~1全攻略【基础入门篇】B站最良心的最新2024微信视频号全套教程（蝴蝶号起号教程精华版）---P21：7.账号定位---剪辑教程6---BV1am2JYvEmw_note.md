# 微信视频号运营从0~1全攻略：P21：7.账号定位 - 剪辑教程6

在本节课中，我们将要学习如何解决视频号运营初期遇到的两个核心难题：如何快速获取初始粉丝，以及如何获得高质量的原创视频素材。课程将提供具体的解决方案和操作步骤。

## 解决初始粉丝与素材问题

![](img/19a575f4ece2e6df2a5380e65f8f7ea5_1.png)

上一节我们介绍了账号定位的基础，本节中我们来看看如何为账号启动打下坚实基础。启动新账号时，许多同学面临两个主要问题：粉丝数量少和缺乏原创素材。

![](img/19a575f4ece2e6df2a5380e65f8f7ea5_3.png)

首先，需要将账号之前的作品隐藏。这是启动新账号的标准操作。

其次，发布内容不要急于求成。建议保持**一天发布一条**视频的节奏。

我们的目标是先解决100个粉丝的问题，再解决素材问题。应按照顺序，逐个解决。

## 获取原创视频素材

接下来解决原创视频的问题。许多同学反馈没有专业的拍摄素材，也不会拍摄和剪辑。

以下是同学们普遍面临的情况：
*   不会拍摄视频。
*   不会剪辑视频。
*   不会制作完整的视频内容。

针对这些情况，本课程提供一项福利：每位同学都可以领取30个已经剪辑好的视频素材。

![](img/19a575f4ece2e6df2a5380e65f8f7ea5_9.png)

这些素材来自一个独家素材库，该库投入了大量成本进行建设。库内素材种类丰富，例如包含搞笑娱乐类视频，这些视频均由专业团队完成剪辑和选材。

领取素材需要满足三个条件：
1.  在公屏上打出“想”字。
2.  在课堂上积极互动。
3.  完整听完课程（后台会统计听课时长与互动次数）。

满足条件后，需联系发放听课链接的助理老师进行领取。领取时，请说明是“张良老师”推荐的，这是获得素材的关键。
