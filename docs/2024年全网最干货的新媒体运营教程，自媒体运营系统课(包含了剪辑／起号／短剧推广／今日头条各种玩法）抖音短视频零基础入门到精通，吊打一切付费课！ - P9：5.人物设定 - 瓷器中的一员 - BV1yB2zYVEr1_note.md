# 新媒体运营系统课：P9：人物设定 🎭

在本节课中，我们将要学习抖音账号运营中一个至关重要的环节——人物设定。一个清晰、真实且富有魅力的人设，是账号吸引粉丝、建立信任并实现商业价值的基础。

## 什么是人设？

在抖音中建立人设，类似于在游戏中创建角色，需要预先设定角色的称呼、职业、技能等属性。但与游戏角色不同，抖音中的人设是真实的人生角色，因此需要更强的真实性。无论是短视频还是直播，一个成功的人设IP能够解决粉丝的信任问题，促进裂变传播，并具备强大的变现能力。

我们运营账号，无论是创作内容还是策划选题，其核心目的都是在获取流量的同时，打造一个成功的IP。

## 如何打造IP？🧩

![](img/5c9a8ddd5d981d9c76c1ba687e71935a_3.png)

打造IP的核心，是在特定场景或情境中，完成账号的人格化过程。这个过程需要注入情感、塑造故事，并贴上具有辨识度的符号标签。

![](img/5c9a8ddd5d981d9c76c1ba687e71935a_4.png)

上一节我们介绍了人设和IP的概念，本节中我们来看看如何具体策划一个账号的人格化设定。

![](img/5c9a8ddd5d981d9c76c1ba687e71935a_5.png)

![](img/5c9a8ddd5d981d9c76c1ba687e71935a_6.png)

对于账号人设的策划，我们可以从以下几个维度着手。以下是六个核心策划方向：

![](img/5c9a8ddd5d981d9c76c1ba687e71935a_7.png)

1.  **名字/昵称**
    一个IP首先需要一个称呼，这是IP传播的名片。它就像品牌的名称或商标一样重要。

![](img/5c9a8ddd5d981d9c76c1ba687e71935a_8.png)

![](img/5c9a8ddd5d981d9c76c1ba687e71935a_9.png)

2.  **职业/身份**
    人格化必然要具备身份，这是用户认识这个IP的联想入口。例如：老师、教授、博士、律师等。

![](img/5c9a8ddd5d981d9c76c1ba687e71935a_10.png)

3.  **社会称谓**
    一个IP有了亲切的称谓，能快速拉近与用户的距离，起到情感共鸣的作用。例如：XX哥、XX大爷、XX姥姥。

4.  **形象印象**
    从用户视角塑造IP形象，使其更容易占据用户心智。例如：大叔形象、萌萌哒、铁憨憨、甜美可爱等。

5.  **性格与特点**
    一个具有鲜明特点的IP，除了记忆点强，也更具真实性。过于完美的形象容易脱离大众。例如：暴躁、爱吃、爱喝酒等。

6.  **穿着与象征**
    借助社会普遍认知和刻板印象，可以快速在用户心智中为IP定位。例如：有钱代表会赚钱、开豪车代表成功人士、朴素打扮代表老师。

## 从策划到落地 🚀

以上内容策划好之后，就可以结合你自身的产品或服务优势，将其落实到账号搭建中，即抖音账号的“五件套”：头像、昵称、背景图、简介和视频封面。

除了对账号进行人格化策划，还需要通过持续的选题和内容策划，为你的人设提炼出独特的记忆点，也就是通常所说的“超级符号”。

## 如何提炼“超级符号”？🔍

“超级符号”的提炼，可以从以下几个感官维度着手。以下是五个关键的提炼方向：

*   **画风（视觉）**：如独特的场景、道具、服饰等。
*   **听觉**：如标志性的BGM、特效音、配音声线特点等。
*   **文案**：能够反映优势且易于记忆的提炼词，如“简单粗暴”。
*   **情绪与表情**：如标志性的笑、惊恐、无奈等表情或情绪传递方式。
*   **原特色与动作**：如方言口音、习惯性的肢体动作（撩头发、扶眼镜）等。

“超级符号”提炼好之后，在你进行短视频创作时，就可以不断在视频中重复呈现这些元素，起到强化用户记忆的作用。
