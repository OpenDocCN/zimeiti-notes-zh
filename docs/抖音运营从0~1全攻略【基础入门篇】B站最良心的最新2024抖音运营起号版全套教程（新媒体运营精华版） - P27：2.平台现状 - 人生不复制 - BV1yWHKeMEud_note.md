# 抖音运营从0到1全攻略：P27：2.平台现状 📊

## 概述
在本节课中，我们将要学习自媒体行业的平台现状。我们将分析国家政策、平台补贴、用户规模等关键信息，以判断当前是否适合进入自媒体领域。

---

上一节我们了解了自媒体的基本概念，本节中我们来看看这个行业的平台现状如何，以及它是否值得去做。

首先，我们需要认识到现阶段社会氛围较为浮躁，许多人都想快速赚钱。过去，有人曾采取非法手段牟利，甚至因此遭受损失，例如前往缅甸等地的危险事件。这些行为已被国家明令禁止，我们绝不能参与，因为它们毫无益处。

现阶段，自媒体行业属于国家明确支持的范畴。国家已将支持文件下达到各个地方政府。

以下是荆州市出台的“六项措施”文件示例，该文件由人社局于今年6月2日发布。

如图所示，各地都发布了针对新媒体的政策性文件。文件强调要坚持正确导向，内容为王。这意味着，未来作为自媒体创作者，我们创作的**文章和视频内容**，将是获得收益的核心途径。

![](img/cb0ce2061757894146f5825e5c0142bb_7.png)

听到这里，你认为自媒体行业可以进入吗？答案是肯定的，因为国家已经通过政府文件明确表示了支持。

除了政策支持，自媒体平台也响应国家号召，投入资金鼓励视频创作。那么，平台具体投入了多少资金呢？我们通过一个视频来了解。



![](img/cb0ce2061757894146f5825e5c0142bb_7.png)



![](img/cb0ce2061757894146f5825e5c0142bb_9.png)

视频中提到，西瓜视频决定在未来一年额外拿出**20亿元**，用于补贴中视频创作人。创始人强调了两点：第一，这是**真金白银的20亿元纯补贴**，不包含任何商单、直播和电商收入；第二，20亿元只是保底，上不封顶。



![](img/cb0ce2061757894146f5825e5c0142bb_9.png)



从这个视频中，我们了解到平台为视频创作提供了巨额资金支持——20亿元。这20亿元是纯补贴，意味着创作者通过制作视频，就有机会从这部分资金中获得收益。

除此之外，现阶段自媒体行业还有一个显著优势：庞大的用户基数。用户数量接近**9.4亿**。

根据2022年中国研究生网站发布的自媒体行业市场调查与发展前景分析报告（发布于今年9月9日），我国现阶段网民规模已达**9.4亿人**，接近10亿。这9.4亿人就是潜在的视频观众，他们拥有手机和视频平台，每天都会刷视频。

有如此多的观众，那么创作者有多少呢？报告显示，截至去年，我国全职从事自媒体的人数为**370万**，兼职人数为**600万**，合计约**970万**。我们取个整数，算作**1000万**。

这意味着，大约有**1000万**人在制作视频，而有**9.4亿**人在观看视频。这相当于每**100个人**中，只有**1个人**在生产内容。试想一下，一个人制作视频的速度，能否满足一百个人观看的需求？答案显然是不能的。

这正体现了行业巨大的内容需求缺口，也是国家支持自媒体行业发展的重要原因。我国未来的发展趋势必然深度融入互联网。

我们可以从生活变迁中看出端倪：我们小时候看的是黑白电视，长大后看彩电，而现在家里的电视甚至可以连接WIFI。未来，我们制作的视频完全有可能出现在千家万户的电视屏幕上。由此可以想象，其用户规模将是何等庞大。

## 总结
本节课中，我们一起学习了自媒体行业的平台现状。我们了解到：第一，国家已出台政策明确支持自媒体发展；第二，主流平台投入巨额真金白银（如20亿元）补贴创作者；第三，行业拥有近10亿的庞大用户基础，而创作者数量相对较少，存在巨大的市场机会。这些现状表明，自媒体是一个政策支持、有资金激励且需求旺盛的领域。
