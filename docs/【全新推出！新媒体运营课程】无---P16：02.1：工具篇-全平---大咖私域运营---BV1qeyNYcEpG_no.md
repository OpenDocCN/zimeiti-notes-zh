# 新媒体运营课程：P16：02.1：工具篇-全平台短视频文案一键提取 📝

在本节课中，我们将学习一个非常实用的运营工具，它能帮助我们从多个平台的短视频中一键提取文案，从而极大地提升内容创作和素材整理的效率。

上一节我们介绍了运营工作的基本概念，本节中我们来看看如何利用工具高效获取文案素材。以往，当我们看到优秀的短视频时，想要借鉴或分析其文案，通常需要先将视频下载下来，再借助剪映或“青抖”等小程序进行提取。这种方法不仅步骤繁琐，而且难以覆盖小红书、视频号、哔哩哔哩、快手等多个平台。

今天分享的工具则能完美解决这个问题。它支持在线操作，只需将目标短视频的链接粘贴到工具中，即可一键提取出文案内容。

![](img/ba461956fd9f89592c1da4db26f31ab9_3.png)

![](img/ba461956fd9f89592c1da4db26f31ab9_4.png)

以下是该工具的核心操作步骤：

1.  **复制链接**：在抖音、哔哩哔哩、小红书等平台找到目标短视频，复制其分享链接。
2.  **粘贴提取**：将复制的链接粘贴到工具的指定输入框中。
3.  **获取文案**：点击“提取文案”按钮，工具会自动处理并生成纯文本文案。
4.  **复制与格式化**：提取出的文案可以一键复制。工具还提供“格式化”功能，能将文案按对话或字幕段落进行清晰排版。

此外，该工具也支持上传本地视频文件来提取其中的文案。

为了让大家更直观地理解，我们来看两个具体示例。

**示例一：提取抖音短视频文案**
我们以一个情感语录类抖音视频为例。将其链接粘贴到工具中并点击提取，很快就能得到如下文案：
```
听完你就释怀了。村上春树说，我动了离开你的念头，不是因为你不好，也不是因为不爱了。
```
提取出的文案准确无误，并且可以方便地复制使用。

**示例二：提取哔哩哔哩视频文案**
我们尝试提取一个B站关于直播电商的教学视频。同样地，粘贴链接并提取，工具成功输出了视频的文案内容：
```
新手做直播电商怎么才能打造出直播爆品在直播销售的不同阶段...
```
这证明了该工具跨平台提取的准确性和高效性。

这个工具在运营工作中用途广泛。无论是进行竞品分析、获取灵感，还是提取文案后进行修改、调整，用于自己的口播视频创作，都能节省大量时间和精力。它集成了多个平台的功能，一个工具就能替代许多其他单一用途的工具。
