# AI新媒体运营实战：P15：01.10.03：利用AI分析爆款笔记并生成内容 📝

## 概述
在本节课中，我们将学习如何利用AI工具（如ChatGPT）分析小红书等平台的爆款笔记，并基于分析结果，指导AI生成具备爆款潜质的新内容。整个过程将分为两个核心步骤：**分析学习**与**模仿创作**。

---

## 从分析到创作：核心步骤解析

![](img/cfb71742d21e7a057e7ee9ec0e07c1d0_1.png)

上一节我们介绍了通过“喂资料”的方式，让AI学习特定平台的文案风格。虽然上一节以旅游攻略为例的成果可能不理想，但方法是正确的。关键在于，你需要为AI提供你所在赛道的优质内容，让它进行分析和学习。

![](img/cfb71742d21e7a057e7ee9ec0e07c1d0_3.png)

![](img/cfb71742d21e7a057e7ee9ec0e07c1d0_5.png)

本节中，我们来看看如何更深入地分析爆款内容，并让AI基于这些分析进行创作。以下是需要让AI重点分析的几个方面：

### 爆款内容分析要点
以下是需要引导AI分析的几个核心维度：

![](img/cfb71742d21e7a057e7ee9ec0e07c1d0_7.png)

1.  **爆款因素**：分析内容为何能成为爆款，例如是否引发了情感共鸣、提供了实用价值、讲述了逆袭故事等。
2.  **内容结构**：
    *   **开篇**：用什么方式（如提问、制造焦虑、引用名人）吸引用户继续阅读。
    *   **中间**：提供了哪些内容来戳中用户痛点，锁住用户注意力，提升停留时长。
    *   **结尾**：设置了什么“钩子”（如鼓励互动、提供福利、引导私信）来促进用户行动（关注、私信、评论）。
3.  **关键词策略**：分析笔记中是否刻意植入了特定关键词。如果有，需要分析：
    *   关键词是什么。
    *   关键词的出现频率和分布。
    *   是如何结合中文分词法进行铺设的。

## 实战演练：以“考研”领域为例

![](img/cfb71742d21e7a057e7ee9ec0e07c1d0_19.png)

![](img/cfb71742d21e7a057e7ee9ec0e07c1d0_21.png)

我们以小红书上的“考研”领域爆款笔记为例，进行完整演示。

![](img/cfb71742d21e7a057e7ee9ec0e07c1d0_23.png)

![](img/cfb71742d21e7a057e7ee9ec0e07c1d0_25.png)

### 第一步：提供素材并下达分析指令
首先，我们需要找到目标领域的爆款笔记文案，并将其提供给AI。接着，给AI下达清晰的分析指令。

![](img/cfb71742d21e7a057e7ee9ec0e07c1d0_27.png)

**给AI的指令示例：**
```
我给你提供几篇小红书平台上关于“考研”的爆款笔记文案。请你分析：
1.  每篇文案的爆款因素。
2.  文案的整体结构：开篇如何吸引用户？中间如何戳中痛点？结尾如何引导互动？
3.  分析文案是否设计了关键词。如果有，请指出关键词、出现频率及铺设方式。
```

![](img/cfb71742d21e7a057e7ee9ec0e07c1d0_29.png)

![](img/cfb71742d21e7a057e7ee9ec0e07c1d0_31.png)

**操作过程：**
1.  在平台上（如小红书）搜索“考研”，筛选出点赞数据较高的笔记。
2.  复制笔记的完整文案内容。
3.  在ChatGPT中新建对话，粘贴指令和文案。

![](img/cfb71742d21e7a057e7ee9ec0e07c1d0_33.png)

![](img/cfb71742d21e7a057e7ee9ec0e07c1d0_35.png)

**AI分析结果示例（摘要）：**
*   **爆款因素**：情感共鸣（描述备考焦虑）、逆袭故事、提供明确备考计划与实用建议、使用表情符号增加亲和力、结尾传递正能量。
*   **结构分析**：开篇用焦虑情绪吸引→中间分享个人经验和详细方法→结尾鼓励并引导互动。
*   **关键词分析**：多数笔记没有明显刻意设计的关键词，词汇以常用词为主。

> **提示**：可以连续提供多篇（如3-4篇）同领域爆款笔记给AI分析，让它学习更全面的模式。

### 第二步：整合分析结果，提炼爆款公式
在AI分析了多篇笔记后，我们需要让它对分析结果进行整合和去重，总结出该领域的通用爆款要素。

![](img/cfb71742d21e7a057e7ee9ec0e07c1d0_37.png)

![](img/cfb71742d21e7a057e7ee9ec0e07c1d0_39.png)

**给AI的指令：**
```
请将你刚才分析的几篇关于“考研”的爆款笔记因素总结一下，整合成一个清晰的列表。
```

**AI生成的爆款因素列表可能包括：**
1.  情感共鸣与痛点切入。
2.  个人逆袭故事经历分享。
3.  提供明确、阶段性的备考计划。
4.  给出具体学科（政治、英语、专业课）的针对性建议和资源。
5.  强调时间管理和效率提升。
6.  使用表情符号（emoji）增强感染力。
7.  结尾进行正能量鼓励并引导互动（提问、私信）。

### 第三步：基于爆款公式，生成内容大纲
现在，我们可以命令AI根据总结出的爆款因素，为我们规划一篇新笔记的框架。

**给AI的指令：**
```
我需要写一篇关于“考研”的小红书笔记。请结合你刚才总结的爆款因素，为我规划一个具备爆款潜质的笔记大纲。大纲需要细分到具体的小节点。
```

**AI生成的大纲示例：**
1.  **开篇：吸引注意**
    *   用引人入胜的话语或问题开头。
    *   简要介绍笔记主题（考研备考）。
2.  **核心内容：干货分享**
    *   专业选择建议。
    *   全年时间规划表（分阶段）。
    *   政治/英语/数学/专业课备考方法。
    *   时间管理与心态调整技巧。
3.  **结尾：引导互动**
    *   鼓励读者，传递信心。
    *   引导读者提问或私信。

![](img/cfb71742d21e7a057e7ee9ec0e07c1d0_41.png)

### 第四步：根据大纲，生成完整文案
最后，我们让AI根据确定好的大纲和爆款要求，生成完整的笔记文案。

![](img/cfb71742d21e7a057e7ee9ec0e07c1d0_43.png)

**给AI的指令：**
```
请按照上面确定的大纲，撰写具体的文案。要求如下：
1.  文案结构严格遵循爆款笔记因素。
2.  开头融入一个简单的“逆袭小故事”来引发共鸣。
3.  文中适量使用emoji表情。
4.  尝试以自然的方式融入“考研规划”、“政治备考”等关键词。
```

**AI生成的文案示例（节选）：**
> “曾经我也在深夜刷题到崩溃，觉得考研上岸是个遥不可及的梦……但通过合理的规划，我最终成功逆袭！把我的经验分享给你，相信你也可以！📚
>
> **🔑 核心备考规划：**
> *   **阶段一（现在-6月）：** 基础夯实，主攻英语单词和专业课课本。
> *   **阶段二（7-9月）：** 强化练习，政治选择题+英语真题第一轮。
> *   **阶段三（10-12月）：** 冲刺模拟，全科真题模拟+错题复盘。
>
> **💡 政治备考提分关键：** 抓住选择题，尤其是多选题！……
>
> 别怕时间不够，只要现在开始行动并坚持，明年上岸的就是你！有任何**考研规划**问题，欢迎评论区问我~ 💪”

## 技巧总结与优化方向

本节课我们一起学习了利用AI分析爆款笔记并生成内容的完整流程。

1.  **流程回顾**：**提供爆款素材 → AI分析（因素、结构、关键词）→ 整合爆款公式 → 生成内容大纲 → 产出完整文案**。
2.  **指令技巧**：给AI的指令需具体、清晰。对于“适量使用emoji”这类模糊要求，AI的理解可能不一致，需要在多次交互中调整措辞。
3.  **模型差异**：使用更强大的模型（如GPT-4）或定制化AI助手（如GPTs），可以获得更高质量、更符合个性化要求的文案。你可以为定制助手提供专属的排版风格、常用表情包库等资料，使其输出更精准。
4.  **人工优化**：AI生成的文案是很好的初稿和灵感来源。对于专业运营人员，仍需在此基础上进行润色、优化逻辑、丰富细节，使其更具个人特色和传播力。
