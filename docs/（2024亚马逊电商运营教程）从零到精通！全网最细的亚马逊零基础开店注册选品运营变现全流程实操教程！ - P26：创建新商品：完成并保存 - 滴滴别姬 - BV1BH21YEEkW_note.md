# 亚马逊电商运营教程：P26：创建新商品：完成并保存

在本节课中，我们将学习如何填写商品发布页面的剩余关键信息，包括搜索关键词、商品尺寸与重量、变体主题等，并最终完成商品的保存与发布。我们将使用翻译工具辅助填写，并确保所有信息的准确性。

![](img/4d3481d51556bf84a4402315d146895d_1.png)

上一节我们介绍了商品图片和描述的设置，本节中我们来看看如何完善商品的后台属性信息。

## 填写搜索关键词 🔍

![](img/4d3481d51556bf84a4402315d146895d_3.png)

搜索关键词是买家通过搜索找到你商品的重要途径。以下是添加关键词的步骤：

![](img/4d3481d51556bf84a4402315d146895d_5.png)

1.  将之前通过查词软件找到的热门关键词，填入“搜索关键词”字段中。
2.  确保关键词准确、相关，以提高商品被搜索到的概率。

## 填写商品重量与尺寸 📏

准确填写商品的物理属性有助于亚马逊计算运费和提供准确的商品信息。

1.  **发货重量**：根据商品实际情况填写。例如，若商品重700克，则填写 `0.7`，单位选择“千克”。
2.  **商品尺寸**：需要填写胸围、颈围/肩宽、袖长等具体尺寸。
    *   **胸围**：例如 `112` 厘米。
    *   **颈围/肩宽**：例如 `45` 厘米。
    *   **袖长**：根据实际测量结果填写。
    *   **注意**：如果商品有多个尺码，在尺寸字段中填写最小尺码的尺寸即可。

## 理解变体主题 🧩

变体主题用于管理同一商品的不同款式，如颜色或尺寸。

*   **核心概念**：变体商品共享大部分商品信息（如标题、描述），仅在特定属性（如颜色、尺寸）上不同。
*   **重要原则**：创建变体时，必须保持“主题”一致。例如，同为T恤的不同颜色，应属于同一主题。不能将不同主题的商品（如T恤和裤子）创建为变体。

## 其他信息与合规性 📋

并非所有字段都必须填写。

1.  尽量填写所有能填写的字段，使商品信息更完整。
2.  对于明显不相关的部分（如销售电子产品时才需填写的“合规信息”），可以留空。

## 使用浏览器翻译工具 🌐

为了更准确地填写英文后台信息，可以使用浏览器的翻译功能。

以下是操作建议：

1.  **谷歌浏览器**：右键点击页面，选择“翻译成中文”。可能需要逐个页面翻译。
2.  **360安全浏览器**：通常能自动或一键翻译整个页面，方便对照中文含义填写英文字段。
3.  **工作流程建议**：使用**谷歌浏览器**进行商品发布操作（响应速度较快），同时打开**360安全浏览器**的翻译页面作为对照参考，确保信息填写正确。

## 最终保存与发布 ✅

检查所有已填写的信息无误后，点击“保存并完成”按钮，提交商品信息。

系统可能需要一些时间处理，请耐心等待发布完成。
