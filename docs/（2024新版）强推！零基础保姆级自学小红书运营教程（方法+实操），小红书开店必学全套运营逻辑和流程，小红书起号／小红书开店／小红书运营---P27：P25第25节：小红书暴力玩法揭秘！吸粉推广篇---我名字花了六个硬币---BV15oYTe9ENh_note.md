# 小红书运营教程：P27：小红书暴力玩法揭秘！吸粉推广篇 🔍

在本节课中，我们将要学习小红书推广的两种核心方式：自建账号矩阵推广与官方付费广告投放。我们将理清两者的关系、适用场景与具体操作，帮助你根据自身情况选择最合适的推广路径。

上一节我们介绍了账号的基础运营，本节中我们来看看如何通过不同策略实现高效推广。

## 账号类型解析：自建推广的基石

小红书的账号主要分为三类，理解它们是构建推广策略的基础。

以下是三种主要账号类型及其特点：

1.  **企业号**
    *   主要作用是树立官方品牌形象。
    *   拥有官方认证标识，可展示品牌名称、地址等信息。
    *   享有官方特权，便于开展相关业务。

2.  **个人号**
    *   目前大多数用户运营的账号类型。
    *   特点是**灵活性高**，可以是品牌员工或消费者身份。
    *   易于塑造个人品牌特色，品牌相关人物出镜更自然。

3.  **引流号**
    *   在之前课程中提及，俗称“炮灰号”。
    *   数量可以是多个，作用相对次要，可随意打理。
    *   常用于临时应急或作为小号使用，通常无长期个性规划。

通常，品牌推广会通过企业号、个人号或引流号自行发布笔记进行引流。

## 企业号矩阵运营建议 🧩

![](img/de5ffb9989335277eb967a5d78b0ba82_3.png)

了解了账号类型后，我们来看看如何协同运营它们，构建一个有效的推广矩阵。

![](img/de5ffb9989335277eb967a5d78b0ba82_5.png)

以下是打造企业号矩阵的四点具体建议：

![](img/de5ffb9989335277eb967a5d78b0ba82_7.png)

*   **积极参与品牌活动**：企业号发起的活动（如试用、团购），应使用小号积极参与。
*   **个人号展示品牌属性**：在个人号笔记中，应多次暗示品牌定位（例如，面向城市青年或职场新人）。
*   **控制营销内容比例**：个人号发布品牌产品的内容频率不宜过高。建议遵循 **“三七分”原则**：`30%` 内容用于营销，`70%` 内容用于种草或科普。
*   **利用引流号进行评论引流**：企业号发布内容后，可使用引流号在评论区进行引导，实现间接引流。

## 官方付费广告：小红书CBC竞价广告 💰

![](img/de5ffb9989335277eb967a5d78b0ba82_13.png)

除了自建账号推广，另一种高效方式是投放小红书官方的付费广告，即CBC竞价广告。

官方广告的申请与使用流程，可在小红书官网的“业务合作” -> “推广合作”板块找到。投放前提是**必须先拥有一个认证的企业号**。

企业号认证流程同样在官网完成，需提交相关资质并支付审核费用。

认证成功后，即可投放信息流广告，主要分为两大类：

1.  **信息流广告**：用户浏览小红书首页时，不定时出现在信息流中的广告。
2.  **搜索广告**：当用户搜索特定关键词时出现的广告。其原理是**关键词竞价排名**，通过对关键词出价竞争不同的广告位。

目前信息流广告位主要出现在搜索结果页的第3位、第13位和第23位。

![](img/de5ffb9989335277eb967a5d78b0ba82_15.png)

关于广告投放的常见问题，官网有详细问答可供查阅。需要注意的是，官方广告投放费用较高。在资金允许的情况下，建议考虑与官方合作投放竞价广告，因为它能直接获得平台推荐的精准流量。
