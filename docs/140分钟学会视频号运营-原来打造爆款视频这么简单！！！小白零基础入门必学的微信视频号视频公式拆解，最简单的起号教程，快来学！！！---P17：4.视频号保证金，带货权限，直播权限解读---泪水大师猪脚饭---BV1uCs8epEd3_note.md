# 视频号运营：P17：4.视频号保证金、带货权限与直播权限解读 🔍

在本节课中，我们将详细解读视频号的带货身份、视频带货、橱窗带货以及直播带货的权限要求与区别。了解这些规则是开启视频号电商变现的基础。

## 带货身份与经营资质

上一节我们介绍了视频号的基础设置，本节中我们来看看如何查看和设置带货身份。

在创作者中心板块，点击右上角的个人头像，即可查看你的带货身份。身份主要分为两种：**达人**和**商家**。这个身份后期可以修改，无需担心初次选择错误。

经营资质部分通常无需过多关注。

## 带货权限详解

了解身份后，我们来看看具体的带货权限。权限主要分为三个板块：视频带货、橱窗带货和直播带货。

以下是三种带货方式的具体说明：

*   **视频带货**
    *   **含义**：指在发布的视频中挂载购物车，类似抖音的“小黄车”。用户观看视频时，屏幕会出现橙色购物车图标，点击即可购买商品。
    *   **开通条件**：需要满足两个核心条件。
        1.  粉丝数 ≥ **1000**（且为有效粉丝）。
        2.  视频内容必须为**原创**。
    *   **注意事项**：此权限审核严格。对于搬运内容的账号，几乎无法开通。即便开通后，一旦发生违规，权限也会被收回。权限被封后，仍可通过橱窗卖货。

*   **橱窗带货**
    *   **含义**：在视频号主页展示商品橱窗，用户点击进入后可浏览和购买商品。这是最基础、最常用的带货方式，对粉丝数无硬性要求。
    *   **优势**：即使没有视频带货和直播带货权限，也可以通过精心运营橱窗实现销售。

![](img/4d24a97278f0775d42ccc7046e89d573_5.png)

*   **直播带货**
    *   **含义**：在直播过程中，向观众展示并销售商品。
    *   **开通条件**：目前开通直播带货功能，需满足以下任一条件。
        1.  粉丝数 ≥ **1000**。
        2.  账号成功开通**视频号小店**。
    *   **重要规则**：根据平台最新规定，一个身份证最多只能实名认证**2个**视频号。如果已认证满额，则无法再为新号开通直播带货。

## 直播权限实操演示

![](img/4d24a97278f0775d42ccc7046e89d573_11.png)

为了让大家更清晰地理解直播带货的权限，这里进行简要说明。

![](img/4d24a97278f0775d42ccc7046e89d573_13.png)

对于已满足粉丝要求的账号，在直播界面可以顺利添加商品并开播。流程为：`发起直播 -> 点击“商品” -> 添加商品 -> 开始直播`。

![](img/4d24a97278f0775d42ccc7046e89d573_15.png)

![](img/4d24a97278f0775d42ccc7046e89d573_17.png)

对于未满足粉丝要求的账号，尝试在直播中添加商品时，系统会提示“**视频号有效关注1000人才可以进行直播带货**”。这意味着，在粉丝数达标前，直播间内无法挂载商品进行销售。

因此，对于急于开展直播带货但粉丝数不足的创作者，解决方案是开通**视频号小店**。开通小店后，即使粉丝不足1000，也可以在直播间挂商品。关于如何办理营业执照及开通视频号小店的具体流程，我们将在后续课程中详细讲解。

本节课中我们一起学习了视频号的三种带货权限及其开通条件。核心在于：**视频带货**要求高且易被封；**橱窗带货**是基础变现手段；**直播带货**则可通过积累粉丝或开通小店来实现。理解这些规则，能帮助你更有效地规划视频号的运营与变现路径。
