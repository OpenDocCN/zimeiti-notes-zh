# 亚马逊跨境电商入门教程：P7：04、对跨境的态度 🏛️

![](img/659c4f132795753130e87abd33b6f40f_1.png)

在本节课中，我们将要学习国家对于跨境电商行业的态度与相关政策，理解为何当前是进入该领域的有利时机。

## 国家态度：支持出口

上一节我们介绍了跨境电商的基本概念，本节中我们来看看国家层面对此的态度。一个核心问题是：出口和进口这两种贸易形式，哪一种对国家的经济发展更有利？

答案是出口。回顾历史，在我国制造业尚不发达的年代，我们大量依赖进口，例如“洋火”、“洋车”。进口意味着将资金支付给国外，购买外国产品。而出口则是将我国的产品销售到海外，为国家赚取外汇收入。因此，出口贸易对国家经济更为有利。

## 鼓励政策：无票免税

为了鼓励普通民众积极参与到出口贸易中，将事业做大做强，国家推出了一项关键扶持政策：**无票免税政策**。

![](img/659c4f132795753130e87abd33b6f40f_3.png)

以下是关于该政策的详细说明：

![](img/659c4f132795753130e87abd33b6f40f_5.png)

*   **政策名称**：跨境电商零售出口无票免税政策。
*   **政策来源**：该政策由国务院总理李克强提出，并通过央视新闻联播等官方渠道发布，具有高度的权威性和真实性。
*   **核心内容**：该政策规定，对跨境电商零售出口实施税收优惠。简单来说，就是通过跨境电商平台将中国商品销售给海外消费者，所获得的收入目前无需向国家缴纳相关税款。

## 政策意义与行业现状

![](img/659c4f132795753130e87abd33b6f40f_7.png)

这项政策的意义非常明确。在国内，绝大多数行业，如开超市、服装店或网店，当收入达到一定标准时都需要依法纳税。然而，现阶段从事亚马逊等跨境电商零售出口，则可以享受“无票免税”的优惠。

这清晰地表明了国家对于发展跨境电商行业的态度：**现阶段是大力鼓励和支持的**。这为个人创业者和小微企业进入该领域提供了宝贵的政策窗口期。
