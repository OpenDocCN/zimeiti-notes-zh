# 剪映教程：P19：剪映核心剪辑功能——帧定格 🎬

在本节课中，我们将要学习剪映的核心剪辑功能之一——**定格功能**。这个功能可以将视频的动态画面瞬间静止，常用于制作卡点视频或强调某个精彩瞬间，为你的作品增添独特的视觉效果。

## 概述

![](img/31b2fb2047a31c9f71641216251ec45d_1.png)

定格功能指的是在视频播放过程中，将某一帧画面静止并持续一段时间。这个功能在制作特效视频，特别是节奏感强的卡点视频时非常有用。它能让动态画面突然“暂停”，创造出类似拍照或时间静止的炫酷效果。

## 功能原理与案例展示

![](img/31b2fb2047a31c9f71641216251ec45d_3.png)

上一节我们介绍了定格功能的基本概念，本节中我们来看看它的具体表现。为了便于理解，我们可以参考一个案例：在抖音上曾流行一时的“万有引力”卡点视频。该视频在每次镜头切换的最后一帧都使用了定格功能，画面会静止片刻，然后配合音效切换到下一个镜头，视觉效果非常出色。

## 实操步骤：如何添加定格

理论讲完，接下来我们进入实际操作环节，看看如何在剪映中添加定格效果。

1.  **导入素材**：首先，打开手机剪映APP，点击顶部“创作”按钮，导入你想要编辑的视频素材。

    

2.  **定位与分割**：导入视频后，播放预览，找到你想要添加定格效果的时间点。例如，我们想在一个镜头结束时进行定格。可以使用分割工具（`分割`）将视频在所需位置剪开。

3.  **应用定格功能**：选中你想要定格的那个视频片段。在底部出现的二级工具栏中，找到并点击 **`定格`** 功能。

    

    点击后，剪映会自动提取选中片段**最后一帧**的画面，并生成一段默认时长为3秒的静态图片片段。

4.  **编辑定格画面**：生成的这个静态片段（定格画面）可以像普通图片或视频一样进行编辑。以下是你可以进行的操作：
    *   **调整时长**：拖动定格片段两端的白色选框，可以自由缩短或延长其静止的持续时间。
    *   **添加特效**：为定格画面单独添加滤镜（如黑白、褪色）、特效（如模糊、震动）等，以增强视觉效果。
    *   **进行裁剪或调色**：对定格画面进行进一步的画面处理。

    

5.  **预览与导出**：编辑完成后，预览整个视频。你会看到动态画面在指定点变为静态，持续你设定的时间后，再继续播放或转场，从而完成卡点效果。确认无误后即可导出视频。

![](img/31b2fb2047a31c9f71641216251ec45d_5.png)

## 核心要点总结

本节课中我们一起学习了剪映的定格功能。让我们总结一下关键点：

*   **功能本质**：`定格` 功能能将视频的某一帧画面提取出来，生成一段静态图像视频。
*   **默认效果**：点击`定格`后，会自动生成一段 **3秒钟** 的静态画面。
*   **高度可定制**：生成的定格片段时长可以自由调整，并且支持添加滤镜、特效等二次编辑。
*   **主要用途**：该功能是制作**卡点视频**、强调瞬间、创造时间静止感等特效的核心工具之一。
