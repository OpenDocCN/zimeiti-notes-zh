# 抖音全套运营教程：P13：11、视频类型1-2 🎬

在本节课中，我们将要学习抖音账号运营中一个至关重要的环节：**建立人设**。我们将深入探讨人设的概念、分类，并重点讲解一种基于人设的核心视频形式——口播视频的制作思路。

上一节我们解决了文案问题，本节中我们来看看内容创作的下一个关键环节。许多课程会直接谈论涨粉与变现，但在这两者之间，还有一个不可或缺的步骤：**建立人设**。人设的目的并非为了炫耀，而是为了在众多同类账号中脱颖而出。例如，有十个人都做装修内容，如果其中一位“小王同学”树立了一个幽默搞笑的人设，观众就更容易记住他，从而忽略其他九位。当观众喜欢你、认可你时，自然就会成为你的粉丝。

为了让大家清晰理解，我们来区分两种主要的人设类型：**强人设**与**弱人设**。理解这一区分后，你将能快速判断任何视频的人设类型。

## 强人设与弱人设

**强人设**的核心是**以创作者个人为切入点**。内容与创作者自身密切相关，可以是分享个人的行业经验、专业技能或兴趣爱好。

以下是强人设常见的切入点领域：

![](img/9b0138a10bca40791a6826ff31c67d23_1.png)

*   **行业经验**：装修、办公、房产、汽车、餐饮、婚介、法律咨询、电商等。
*   **兴趣爱好**：美食、旅行、手工、宠物、星座占卜、美妆护肤、服装穿搭、游戏、摄影等。

强人设视频通常需要**真人出镜**，通过分享工作、生活或爱好来塑造鲜明的个人形象。

**弱人设**则与创作者本人关系不大，核心目的是**运营好账号本身**。内容不依赖于个人特质。

以下是弱人设的典型例子：

*   **情感语录类**：搭配音乐的纯文字视频，无真人出镜或配音。
*   **影视解说类**：虽有解说声音，但焦点是电影内容，而非解说者本人。

区分清楚人设类型后，我们才能更好地规划内容。接下来，我们看看基于人设，具体可以创作哪些视频内容。

## 核心视频形式：口播

只要内容与你个人相关，都可以转化为视频。我们首先讲解一种非常高效且通用的形式：**讲故事**。

讲故事可以分享客户故事、从业经历或人生感悟。故事本身不要求绝对真实，可以从文案平台寻找有趣素材进行分享。而讲故事最直接的方式，就是**口播**。

口播即用嘴巴讲述，它拥有丰富的分类，适用于不同领域：

*   **故事型口播**：如情感口播、养生故事口播，目的是让观众觉得有趣、有共鸣。
*   **知识型口播**：如育儿、金融财经、装修、房产汽车知识口播，目的是在几十秒内让观众获得“收获感”，觉得学到了东西。

无论你身处哪个领域，只要你有知识或故事可分享，都可以尝试口播。它的优势在于**制作简单**：一部手机就能完成拍摄，关键在于准备好文案。

关于口播，初学者常有以下疑问：

*   **口才不好能做吗？**  
    **可以**。口播是依据提前准备好的稿子进行录制，并非临场发挥。很多口播者的流畅表达都源于充分准备。
*   **普通话不标准能做吗？**  
    **可以，有时甚至更有优势**。如果你的目标客户在同城，使用方言可能更显亲切，效果更好。同时，你可以通过添加字幕来确保信息传递，观众“听不懂也能看懂”。

本节课中我们一起学习了抖音账号运营中“建立人设”的核心价值，明确了**强人设**与**弱人设**的区别，并深入介绍了基于人设的一种高效视频形式——**口播**。记住，口播的关键在于文案准备，它门槛低、易上手，是内容创作的利器。在接下来的课程中，我们将具体学习如何拍摄口播视频。
