# 新媒体运营入门课程：P16：02.1：工具篇-全平台短视频文案一键提取 📝

在本节课中，我们将学习一个非常实用的运营工具，它能帮助我们从各大短视频平台一键提取视频文案，从而极大地提高内容创作和素材整理的效率。

上一节我们介绍了文案的重要性，本节中我们来看看如何高效地获取文案素材。以往，当我们看到优秀的短视频时，想要获取其文案内容，通常需要借助一些较为原始的方法。

以下是几种传统提取文案的方法：
*   使用“轻抖”等小程序进行提取。
*   先将视频下载到本地，再导入“剪映”等剪辑软件中提取字幕。

这些方法虽然可行，但存在局限性。例如，它们可能只适用于特定平台（如抖音），而面对小红书、视频号、哔哩哔哩、快手等多个平台时，操作会变得繁琐。今天分享的工具可以完美解决这个问题。

下面，我们通过实际操作来演示其使用方法。

### 提取抖音视频文案

首先，我们尝试提取一个抖音短视频的文案。
1.  在抖音中找到一个目标视频（例如，搜索“情感语录”）。
2.  复制该视频的分享链接。
3.  将链接粘贴到工具的输入框中。
4.  点击“提取文案”按钮。

![](img/f8959271cc2f6201d0b7f10e4d68dbe2_3.png)
![](img/f8959271cc2f6201d0b7f10e4d68dbe2_4.png)

提取过程非常迅速。完成后，工具会准确显示视频的文案内容，例如：“听完你就释怀了，村上春树说，我动了离开你的念头，不是因为你不好，也不是因为不爱了……”。

提取出的文案支持一键复制。此外，工具还提供“格式化”功能，能将文案按照字幕的段落格式进行自动排版，使其更清晰易读。该工具也支持上传本地视频文件来提取文案。

### 提取哔哩哔哩（B站）视频文案

该工具同样适用于哔哩哔哩平台。我们用一个B站视频链接进行测试。
1.  复制B站视频的链接（通常以 `BV` 号开头）。
2.  将链接粘贴到工具中。
3.  再次点击“提取文案”。

提取成功后，我们得到了准确的文案，例如：“新手做直播电商，怎么才能打造出直播爆品？在直播销售的不同阶段……”。这证明了工具跨平台提取的准确性和高效性。
