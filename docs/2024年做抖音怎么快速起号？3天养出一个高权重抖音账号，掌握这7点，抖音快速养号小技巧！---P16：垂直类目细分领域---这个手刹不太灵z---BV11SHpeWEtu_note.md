# 抖音运营教程：P16：垂直类目细分领域 🎯

在本节课中，我们将要学习抖音账号垂直度的分级概念。理解垂直类目的细分，能帮助你更精准地定位内容，吸引目标观众。

上一节我们介绍了内容标签的重要性，本节中我们来看看如何将“垂直”这个概念进行更细致的划分。

我们经常说做账号要做垂直。许多初学者可能会简单地认为，选择“美食”或“游戏”这样的大类目一直做下去，就是垂直。这个理解没错，但这仅仅属于最基础的垂直级别。

以下是垂直类目的分级体系，我们以“游戏”领域为例进行说明：

1.  **一级垂直领域**
    *   这是最宽泛的分类。例如，你决定做“游戏”内容。在这个级别下，你可以涵盖手游、网游、电竞、线下游戏等所有与游戏相关的内容。
    *   **公式表示**：`垂直领域 = “游戏”`

2.  **二级垂直领域**
    *   这是一级领域下的具体分支。在“游戏”这个大类下，你可以选择专注于“手游”、“电竞”或“网游”中的一个。
    *   **公式表示**：`垂直领域 = “游戏” -> “手游”`

3.  **三级垂直领域**
    *   这是对二级领域的进一步细化。例如，在“手游”这个分支下，你可以选择专门制作关于“王者荣耀”或“和平精英”（吃鸡）的内容。
    *   **公式表示**：`垂直领域 = “游戏” -> “手游” -> “王者荣耀”`

4.  **四级垂直领域（及更细）**
    *   垂直度可以无限细分下去，这通常能打造出极具个人特色的账号。例如：
        *   在“王者荣耀”下，只玩“安琪拉”这个英雄。
        *   在“和平精英”下，只玩“雨林”这张地图，或只制作“剧情模式”的内容。
    *   **代码描述**：
        ```python
        # 这是一个极度垂直的账号定位示例
        vertical_field = “游戏 -> 手游 -> 王者荣耀 -> 英雄：安琪拉”
        ```

所以，垂直类目的细分领域有一级、二级、三级甚至更多级别。大领域和小领域都可以作为你的选择，这完全取决于你自己的兴趣和规划。
