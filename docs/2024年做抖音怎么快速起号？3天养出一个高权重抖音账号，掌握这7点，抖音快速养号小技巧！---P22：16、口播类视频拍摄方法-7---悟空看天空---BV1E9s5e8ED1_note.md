# 抖音运营教程：P22：口播类视频拍摄方法

在本节课中，我们将学习口播类视频的具体拍摄方法。掌握正确的拍摄技巧，可以有效提升视频的观感与专业性，从而更好地吸引目标观众。

上一节我们介绍了视频拍摄的通用注意事项，本节中我们来看看如何将这些要点应用到具体的口播视频拍摄中。

## 口播视频拍摄核心要点

![](img/cd4545b0acd69acbd75a43595d9b64d5_1.png)

口播视频的拍摄方法具有通用性，无论内容主题是金融、装修、养生还是情感，其拍摄逻辑都是一致的。以下是拍摄口播视频时需要关注的几个核心方面：

1.  **构图比例**
2.  **主题突出**
3.  **事物清晰**
4.  **光线色彩**
5.  **内容连贯**

为了让这些要点更易于理解，我们将通过一个具体案例进行分析。

## 案例分析：学员“种小草”

我们以学员“种小草”（一位40岁带三个娃的宝妈）的情感口播账号为例。她的一条关于婆媳关系的视频播放量超过8000万，并借此涨粉30万。成功的关键在于视频内容精准吸引了特定群体。

### 构图与画面设置

首先分析其视频的构图。抖音是**9:16**的竖屏平台，但视频内容可以选择横屏或竖屏拍摄。

*   **比例选择**：“种小草”的视频采用了**4:3**的横屏构图。需要注意的是，一旦选定横屏或竖屏，应保持风格统一，不要随意切换。
*   **人物位置**：人物主体应占据画面中央约**70%** 的面积。避免将脸部怼满屏幕，应在画面的左、右、上方适当留白，使画面看起来清晰、舒适。
*   **背景与美观**：如果对自己外貌不自信，可以开启美颜滤镜。口播视频的核心是传递内容和知识，只要画面清晰、人物体面即可。背景也应整洁美观。

### 内容与目标群体定位

![](img/cd4545b0acd69acbd75a43595d9b64d5_3.png)

现在，我们结合案例视频内容，分析其如何吸引目标群体。

视频文案节选：
> “跟婆婆在一起住是什么样的感受？我跟我婆婆在一起住了16年了……我最大的感受就是这十多年以来，我娘家人很少在我家住上几天……这对于我内心来说其实是一个很大的缺失。”

这段内容讨论的是婆媳关系与家庭生活。那么，它最能引起谁的共鸣？

*   对婆媳关系感触最深的，通常是**已婚女性群体**。
*   未婚或年轻的女性对此感受不深，男性群体的共鸣感也相对较弱。

因此，这条视频精准吸引的是**已婚女性群体**。这启示我们，拍摄前需明确内容想吸引哪个群体，并围绕该群体的兴趣点进行创作。

### 拍摄与收音技巧

对于口播拍摄中的常见问题，这里提供两个实用解决方案：

1.  **记不住台词怎么办？**
    无需背诵。可以在剪辑环节使用提词器软件，实现边看边录，大大降低拍摄难度。公式表示为：**拍摄 = 看提词器 + 自然讲述**。

![](img/cd4545b0acd69acbd75a43595d9b64d5_5.png)

2.  **如何保证声音清晰？**
    声音清晰是口播视频的关键。建议购买一个**手机收声麦克风**（价格通常在十几元）。这能有效减少环境噪音，提升录音质量。代码式建议为：`装备 += 手机收声麦`。

## 总结

本节课我们一起学习了口播类视频的拍摄方法。关键点包括：选择并固定**4:3横屏**或**9:16竖屏**构图；确保人物居中并留有适当空白；利用美颜和背景提升画面质感；最重要的是，内容要精准针对像**已婚女性**这样的特定群体。同时，利用提词器解决忘词问题，并配备收声麦保证音频清晰。掌握这些技巧，你也能拍出更专业、更具吸引力的口播视频。
