# 小红书运营教程：P2：小红书的检测规则与运营注意事项 🔍

在本节课中，我们将详细解析小红书的平台检测规则。了解这些规则是避免账号违规、确保内容安全发布的基础。我们将从四个核心方面进行讲解，并介绍一个实用的工具来辅助你的运营工作。

## 平台检测的四个核心方面

小红书的检测系统主要针对以下四个方面进行审查，运营者必须严格遵守。

以下是具体的检测内容列表：

1.  **个人资料、笔记内容、私信及评论**：这些位置不得包含微信号、微信二维码、代购、水印、推广淘宝店铺等明显的营销词汇。
2.  **手机设备**：同一台设备最多建议只登录三个小红书账号，过多账号可能引发风险提示。
3.  **笔记内容规范**：笔记中不得出现诱导用户的行为，例如“关注领奖”、“点赞抽奖”等。
4.  **内容原创度**：小红书对原创内容保护程度高。严禁将同一篇文章在多个账号上重复发布。

## 敏感词检测与实用工具

除了上述通用规则，小红书还通过敏感词库对内容进行过滤。涉及手机卡、金融、电子烟、医疗用品等领域的词汇需要格外谨慎。

那么，在撰写内容时，如何确保没有误用敏感词呢？这时我们可以借助外部工具进行检测。

以下是两个推荐的敏感词检测网站及其使用方法：

*   **网站一（无需注册）**：
    1.  将写好的笔记内容粘贴到网站的输入框中。
    2.  点击“立即检测”按钮。
    3.  系统会高亮标记出内容中存在的敏感词，例如“代购”。你可以据此进行修改。

    2.  粘贴内容并检测后，网站会给出过滤结果。
