# 抖音运营教程：P18：口播类视频拍摄方法

## 概述
在本节课中，我们将学习口播类视频的具体拍摄方法。我们将通过一个成功案例，详细拆解拍摄时需要注意的构图、内容定位、设备使用等核心要点，帮助你掌握制作高质量口播视频的技巧。

上一节我们介绍了视频拍摄的通用注意事项，本节中我们来看看如何将这些要点应用到具体的视频类型中。口播类视频是抖音上非常常见且有效的内容形式，无论你的领域是金融、装修、养生还是情感，其拍摄方法都是相通的。

### 案例引入与内容定位
首先，我们分析一个成功案例。这位名为“种小草”的博主，是一位40岁带三个孩子的宝妈，她的一条关于婆媳关系的情感口播视频播放量超过8000万，并借此涨粉30万。

**她的成功关键在于精准的内容定位。** 她的视频内容专门吸引有特定共鸣的群体。例如，如果我的目标群体是20多岁的学生，那么我所有的内容都应围绕学生感兴趣的话题展开，这样涨粉才会又快又精准。

以下是分析其视频吸引力的方法：
*   **观察视频主题**：她的视频主题是“与婆婆同住的感受”，这直接关联到**婆媳关系**。
*   **定位核心群体**：对婆媳关系感触最深的，并非男性或未婚年轻女性，而是**已婚女性群体**。这个群体能与视频内容产生深度共鸣，因此是她视频吸引的主要对象。

### 拍摄构图与画面设置
明确了内容为谁而做之后，我们来看如何通过拍摄呈现它。以下是口播视频拍摄的核心画面要求：

**1. 比例统一**
抖音是**9:16**的竖屏平台。你可以在其中选择横屏或竖屏构图，但必须保持统一。案例中的博主选择了**4:3**的横屏构图。请确保你的所有视频保持一致的画面比例。

![](img/4691d0168ab3333f28e7e70a2dad766d_3.png)

**2. 主体突出**
人物应位于画面中间，占据约**70%** 的面积。避免怼脸拍摄，要在画面的左、右、上方适当留白，使画面看起来清晰、舒适。

**3. 形象管理**
口播视频的核心是传递内容和知识，对颜值的要求并非绝对。你只需要做到形象清晰、体面即可。如果对自己外貌不自信，可以开启美颜滤镜，但注意不要过度夸张。

**4. 背景选择**
拍摄地点没有限制，室内或户外均可。关键在于背景要整洁、美观，避免杂乱无章影响观感。

### 提词与收音技巧
解决了“拍什么”和“怎么拍”的问题后，我们来看看如何解决拍摄过程中的实际困难。

**1. 使用提词器**
如果你担心记不住台词，完全不需要死记硬背。在剪辑环节，你可以使用提词器APP，实现边看边读边录制，这能极大降低拍摄难度。

![](img/4691d0168ab3333f28e7e70a2dad766d_5.png)

**2. 保证收音质量**
口播视频中，声音清晰度至关重要。为了保证音质，建议你购买一个**手机收声麦克风**。这种设备价格便宜（约十几元），在淘宝、京东等平台都能买到，能有效减少环境噪音，让你说话的声音更清晰。
