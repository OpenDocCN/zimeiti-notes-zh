# 剪映PC版教程：P11：字幕库详解 🎬

![](img/dd62aadb06bb8983e7f8e949710d7642_0.png)

在本节课中，我们将学习剪映PC版中“文本”功能模块的完整使用方法。字幕是视频包装的重要组成部分，能够有效提升视频的质感和信息传达效率。我们将从基础文本编辑开始，逐步深入到智能字幕、歌词识别等高效工具的应用。

## 文本基础：自定义文字添加与样式调整

上一节我们介绍了视频剪辑的基本流程，本节中我们来看看如何为视频添加和美化文字。

首先，在菜单栏选择第三个选项“文本”。打开后，下方会显示多个功能模块。第一个是“默认”文本，即自定义文本。你可以在此输入任意文字。

**操作步骤**：
1.  点击“默认”下的“+”号或直接将“默认”文本拖拽到时间轴轨道上。
2.  文本会添加到当前播放指示器的位置。
3.  在右侧的“基础”面板中，可以编辑文字内容。

选中轨道上的文本后，右侧面板会提供丰富的调整选项：

*   **字体与字号**：在“字体”下拉菜单中可以选择系统内所有字体，通过滑块或直接输入数值调整字号大小。
*   **样式**：可使用 **`B`**（加粗）、**`U`**（下划线）、**`I`**（倾斜）按钮改变文字样式。
*   **颜色**：点击颜色块，可以从预设色板中选择，或使用取色器自定义颜色。
*   **间距**：
    *   **字间距**：调整字符之间的水平距离。公式可理解为：`视觉宽度 = 字符数 × (字宽 + 字间距)`。
    *   **行间距**：当文字有多行时，调整行与行之间的垂直距离。
*   **对齐**：此处的对齐方式指多行文字之间的对齐（如左对齐、居中、右对齐）。
*   **位置**：在“位置”选项中，可以设置文字在整个视频画面中的对齐方式（如靠左、居中、靠右），或直接使用鼠标拖拽调整。
*   **不透明度**：调整文字的透明程度。
*   **高级样式**：可以为文字添加描边、背景框、发光、阴影等效果，并调整这些效果的强度、大小和颜色。

## 花字与气泡：使用预设模板快速美化

如果觉得手动调整文字样式复杂，可以使用预设的“花字”和“气泡”模板。

“花字”是已设计好颜色、描边、阴影等效果的文字样式模板。应用方法如下：
1.  选中时间轴上的文本。
2.  在右侧面板切换到“花字”标签页。
3.  浏览并点击喜欢的样式，即可一键应用。

“气泡”则是预设的对话框样式，常用于人物对话或漫画风格的视频中。使用方法与“花字”相同。

## 文字动画：让字幕动起来

静态文字可以添加动画，使其入场、出场或在画面中循环运动，增强视觉效果。

以下是添加动画的步骤：
1.  选中文本，在右侧面板切换到“动画”标签页。
2.  **入场动画**：控制文字如何出现。选择一个效果（如“旋入”、“渐显”），并调整动画时长以控制速度快慢。
3.  **出场动画**：控制文字如何消失。同样选择效果并调整时长。
4.  **循环动画**：让文字在显示期间持续运动（如跳动、闪烁）。此动画可与入场、出场动画叠加。

**关键提示**：入场、出场、循环动画可以分别独立添加和设置，为文字创造丰富的动态效果。

## 文字模板：一键生成炫酷片头与标题

“文字模板”功能提供了更复杂、更完整的文字动画组合，非常适合制作片头、片尾或视频中的标题包装。

使用方法极其简单：
1.  在“文本”菜单下的“文字模板”区，浏览分类（如标题、片头、节日等）。
2.  将喜欢的模板直接拖拽到时间轴轨道上。
3.  选中轨道上的模板，在右侧“文本”编辑区，修改模板中的文字内容为你需要的文案。
4.  你仍然可以调整模板的整体大小、位置、字体等属性。

与手动制作相比，文字模板能极大提升效率和专业度。

## 智能工具：高效生成字幕与歌词

上一部分我们学习了如何手动创建和美化文字，接下来看看剪映如何智能地帮助我们生成字幕。

### 智能字幕

为已有配音或对话的视频添加字幕，无需手动输入。
1.  将带有配音的视频素材导入时间轴。
2.  点击“文本” -> “智能字幕”。
3.  点击“开始识别”，软件会自动分析音频并生成同步的字幕文本。
4.  识别完成后，所有字幕会以“字幕组”形式出现在时间轴。你可以统一修改它们的“花字”样式，或单独调整某一句的位置和动画。

### 文稿匹配

如果你已有完整的视频文案（脚本），可以使用此功能实现字幕的精准匹配。
1.  将配音音频导入时间轴。
2.  点击“文本” -> “文稿匹配”。
3.  将准备好的文案粘贴进输入框。
4.  点击“开始匹配”，软件会根据音频节奏，将文案自动分段并生成时间轴字幕。这能有效避免智能识别可能产生的错别字。

### 识别歌词

为音乐视频添加滚动歌词。
1.  将歌曲音频导入时间轴。
2.  点击“文本” -> “识别歌词”。
3.  点击“开始识别”，软件会自动识别音频中的歌词并生成字幕。
4.  **注意**：需检查识别结果是否有错别字，并手动修正。

### 本地字幕（SRT文件导入）

![](img/dd62aadb06bb8983e7f8e949710d7642_9.png)

支持导入外部制作好的字幕文件（如 `.srt`、`.lrc` 格式），便于团队协作或使用专业工具制作的字幕。
1.  点击“文本” -> “本地字幕”。
2.  点击“导入”，选择你的字幕文件。
3.  将导入的字幕文件拖拽到时间轴上，对齐音频。
4.  导入后，可以像处理普通字幕一样，统一修改样式或添加动画。
