# AI短视频创作教程：P5：2.平台现状 📊

## 概述
在本节课中，我们将要学习自媒体行业的平台现状。我们将分析国家政策、平台支持力度以及市场供需关系，以判断当前是否适合进入自媒体领域。

---

上一节我们介绍了自媒体的基本概念，本节中我们来看看这个行业的平台现状如何，以及它是否值得去做。

首先，我们需要了解社会背景。现阶段社会氛围比较浮躁，许多人都想赚钱。过去，有些人采取了不合法的手段，例如前往缅甸等地，最终导致了严重的后果。这类行为已被国家明令禁止，我们不能去做，因为它对我们没有任何好处。

现阶段，自媒体行业是属于国家支持的领域。国家已经将支持文件下达到了各个地方政府。

以下是相关政策文件的示例：

该文件指出，要加大政策宣传力度，有力支持自媒体以及新媒体行业的蓬勃发展。你们也可以查看自己所在地是否有类似政策。



![](img/25167866ae96a772bb589c10c76d5dbb_3.png)



如图所示，各地都发布了支持新媒体的政策性文件。文件中强调要坚持正确导向，内容为王。这意味着，未来作为自媒体创作者，我们创作的文章和视频，才是结算收益最重要的方式和途径。

听到这里，你认为自媒体行业可以来做吗？答案是肯定的，因为国家已经通过政府文件明确表示支持。

![](img/25167866ae96a772bb589c10c76d5dbb_7.png)

除了国家政策支持，自媒体平台也需要响应号召，出资鼓励视频创作。那么平台出多少钱呢？



![](img/25167866ae96a772bb589c10c76d5dbb_7.png)



从视频中我们了解到，平台决定额外拿出**20亿元**补贴中视频创作人。这20亿元是真金白银的纯补贴，不包含任何商单、直播和电商的收入。平台还会积极探索保底加分成的模式。这20亿元只是保底，上不封顶。



![](img/25167866ae96a772bb589c10c76d5dbb_9.png)

![](img/25167866ae96a772bb589c10c76d5dbb_9.png)



这意味着，我们未来通过创作视频，就有机会从这20亿元中获得收益。

除此之外，现阶段自媒体行业还有一个巨大优势：用户数量非常庞大。

根据2022年的一份行业报告，我国现阶段中国网民规模约为**9.4亿人**，接近10亿。这9.4亿人就是潜在的视频观众，他们拥有手机和视频平台，每天都会刷视频。

有如此多的观众，做视频的人有多少呢？报告显示，截止去年，我国全职从事自媒体的人数是**370万**，兼职人数是**600万**，合计约**970万**人。我们可以近似看作有**1000万**人在制作视频。

这就形成了一个简单的供需关系：
`观众数量 (9.4亿) / 创作者数量 (0.1亿) ≈ 94 : 1`

这相当于每100个看视频的人中，只有大约1个人在做视频。那么，一个人制作的视频，足够100个人看吗？一个人制作视频的速度，能赶上100个人观看视频的速度吗？答案显然是不够的。

这正是国家支持自媒体行业发展的原因。我们国家未来的发展趋势必然在互联网行业。从我们小时候看的黑白电视，到长大后的彩电，再到如今可以连接WIFI的智能电视，未来我们制作的视频，甚至可以在自家电视上被看到。你可以思考一下，这个用户基数有多么庞大。

## 总结
本节课中我们一起学习了自媒体行业的平台现状。我们了解到，该行业受到国家政策明确支持，各大平台也投入巨额资金进行补贴。同时，市场呈现出观众数量（约9.4亿）远大于创作者数量（约1000万）的蓝海状态，供需关系极不平衡，存在巨大的市场机会。因此，从政策、资金和市场三个维度看，现阶段进入自媒体行业是可行的。
