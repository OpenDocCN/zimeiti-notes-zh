# 视频号运营教程：P41：如何扒取直播间话术 🎤

在本节课中，我们将学习一个实用的技巧：如何高效地获取并借鉴其他成功直播间的话术。这对于希望快速掌握直播带货技巧的初学者来说，是一个低成本且高效的学习方法。

---

## 核心方法与工具

上一节我们介绍了模仿成功直播间的重要性，本节中我们来看看具体如何操作。核心是使用一个名为“飞书妙计”的工具来录制并转写直播内容。

![](img/6590e5cbe8ffba04be0f88f011163d03_1.png)

你需要准备的工具如下：
*   **飞书App**：下载并安装。
*   **目标直播间**：一个你希望学习和模仿的、销售情况良好的直播间。

![](img/6590e5cbe8ffba04be0f88f011163d03_3.png)

---

## 操作步骤详解

以下是使用飞书妙计录制并提取话术的完整流程。

1.  **打开飞书妙计**
    打开飞书App，进入“工作台”，找到并点击“飞书妙记”功能。

2.  **开始录制直播**
    点击飞书妙计界面右下角的**语音按钮**，即可开始录音。此时，打开你想要学习的直播间，该工具会实时录制直播间内的所有声音。

    ![](img/6590e5cbe8ffba04be0f88f011163d03_1.png)

3.  **获取并导出文本**
    录制大约15分钟后，点击“完成”结束。系统会自动将录音转换为文字。转换完成后，点击进入详情页，选择“分享”，创建一个“可编辑”的链接。

    ![](img/6590e5cbe8ffba04be0f88f011163d03_3.png)

4.  **整理与提炼话术**
    将上一步获得的分享链接复制到微信中打开，即可看到完整的文字记录。你可以复制这些文本，然后进行编辑和提炼，形成适合你自己风格和产品的直播话术。

    ![](img/6590e5cbe8ffba04be0f88f011163d03_5.png)

5.  **应用话术开播**
    将整理好的话术写在纸上或存入手机备忘录。当你开始直播时，如果忘记内容，可以对照这些话术进行讲解。
