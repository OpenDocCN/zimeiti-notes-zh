# 视频号运营：P1：商家与达人合作全流程指南

在本节课中，我们将学习视频号商家如何邀请达人合作、达人如何接受邀请并管理合作商品，以及如何设置定向佣金。整个过程涉及商家后台操作与达人手机端操作，我们将分步详细讲解。

---

## 商家发起合作邀请

商家需要先在后台向目标达人发出合作邀请，这是开启合作的第一步。

![](img/2770a3129201a1f54b093d40bb9b9a07_0.png)

以下是商家在视频号后台发起邀请的步骤：
1.  打开微信，点击底部导航栏的 **“发现”**。
2.  进入 **“视频号”** 板块。
3.  点击页面右上角的个人头像图标。
4.  在打开的菜单中选择 **“创作者中心”**。
5.  进入 **“带货中心”**。

在“带货中心”内，商家可以找到邀请达人合作的相关功能入口，完成对达人的邀请。

---

## 达人接受合作邀请

上一节我们介绍了商家如何发起邀请，本节中我们来看看达人如何接受邀请。达人主要在手机端进行操作。

达人接受合作的具体流程如下：
1.  达人在手机端进入视频号 **“创作者中心”**。
2.  找到并点击 **“合作管理”** 功能。
3.  在“合作管理”页面中，可以查看到商家发来的 **“合作申请”**。
4.  点击进入申请详情，选择 **“同意合作”**。
5.  同意后，可以 **“查看合作商品”**，了解商家希望推广的商品清单。

---

## 商家设置合作商品与定向佣金

达人同意合作后，合作并未完全建立。商家需要在后台为已同意的达人具体设置可推广的商品及佣金比例。

此阶段的核心操作是设置 **定向佣金**，其逻辑可以表示为：
**定向佣金 = 为特定达人设置的专属商品佣金比例**
商家在后台的商品管理中，可以为已建立合作关系的达人单独配置某款商品的佣金，这通常高于通用佣金以激励推广。

---

## 达人将商品加入橱窗

商家设置好合作商品和佣金后，达人便可以在自己的橱窗中添加这些商品，开始进行推广。

![](img/2770a3129201a1f54b093d40bb9b9a07_2.png)

以下是达人将合作商品添加至橱窗的步骤：
1.  在手机端 **“合作管理”** 页面，点击 **“合作中”** 的状态。
2.  找到并点击 **“查看合作商品”**。
3.  页面将展示所有已建立合作关系的商品列表。
4.  点击商品旁的 **“加入橱窗”** 按钮，即可完成添加。

---

本节课中我们一起学习了视频号从商家邀请达人、达人接受合作、商家设置定向佣金到达人最终上架商品的完整闭环流程。掌握这些步骤，是顺利进行视频号带货合作的基础。
