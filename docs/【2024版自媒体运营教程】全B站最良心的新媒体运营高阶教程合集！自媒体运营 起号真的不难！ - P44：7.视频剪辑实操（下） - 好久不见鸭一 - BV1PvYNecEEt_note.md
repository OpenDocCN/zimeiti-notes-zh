# 自媒体运营教程：P44：视频剪辑实操（下）

在本节课中，我们将继续学习视频剪辑的后续关键步骤，包括如何为视频片段添加转场、调整转场素材的比例，以及如何为视频添加和调整字幕。掌握这些技巧能让你的视频衔接更自然，观看体验更流畅，从而有效提升视频的完播率。

上一节我们介绍了视频素材的导入与基础比例调整，本节中我们来看看如何让多个视频片段平滑过渡。

## 添加视频转场

![](img/7505f04fedd69dfa2bfd66c45f48897f_3.png)

每个视频通常由多个短片段组成，片段之间需要有衔接。添加转场是为了让用户明确感知到内容片段的切换。如果不添加转场，观众在切换到不同内容的片段时可能会感到突兀，甚至退出视频查看，这将直接导致**视频完播率**下降，从而影响收益。

![](img/7505f04fedd69dfa2bfd66c45f48897f_5.png)

![](img/7505f04fedd69dfa2bfd66c45f48897f_7.png)

**完播率**是评估视频观看完成度的重要指标，公式可简单理解为：
`完播率 = 完整观看视频的用户数 / 视频的总播放次数`

![](img/7505f04fedd69dfa2bfd66c45f48897f_9.png)

高完播率通常意味着内容吸引人，有助于平台推荐和收益提升。

![](img/7505f04fedd69dfa2bfd66c45f48897f_11.png)

以下是添加转场的具体步骤：

![](img/7505f04fedd69dfa2bfd66c45f48897f_13.png)

1.  在视频编辑轨道右侧，找到并点击“加号”按钮。
2.  在弹出的菜单中选择“素材库”。
3.  在素材库的“热门”分类中，可以找到大量转场素材，例如各种动态效果或搞笑片段。
4.  选择转场素材时，注意时长不宜过长，尽量选择**3~5秒**的转场，以保证节奏紧凑。避免使用长达十几秒的转场。
5.  选中合适的转场素材（例如一个3秒的片段），点击添加。

![](img/7505f04fedd69dfa2bfd66c45f48897f_15.png)

## 统一转场素材比例

![](img/7505f04fedd69dfa2bfd66c45f48897f_17.png)

添加转场素材后，必须确保其画面比例与主视频保持一致（例如16:9的横屏比例），以维持视觉统一。

以下是调整比例的步骤：

![](img/7505f04fedd69dfa2bfd66c45f48897f_19.png)

1.  选中已添加的转场素材片段。
2.  在编辑菜单中找到“画布模糊”或“背景”选项。
3.  选择“画布模糊”下的第一个清晰模式，系统会自动将素材背景适配为16:9的横屏比例。
4.  最后，点击右下角的“对勾”确认修改。

![](img/7505f04fedd69dfa2bfd66c45f48897f_21.png)

## 为视频添加字幕

字幕能显著提升视频的可看性和信息传达效率，尤其对于没有字幕就无法理解内容的观众。我们可以利用软件的自动识别功能快速添加字幕。

![](img/7505f04fedd69dfa2bfd66c45f48897f_23.png)

以下是添加和调整字幕的步骤：

![](img/7505f04fedd69dfa2bfd66c45f48897f_25.png)

1.  点击手机编辑界面左下角的“文字”选项。
2.  选择“新建文本”，然后点击右侧的“识别字幕”功能。
3.  点击“开始匹配”，软件会自动识别视频或音频中的人物对话，并生成字幕文本。
4.  字幕生成后，需要进行基础调整以优化观看体验：
    *   **放大字幕**：拖动字幕文本框右下角，向右拖动以放大字体。
    *   **调整位置**：将放大后的字幕文本框拖动到视频画面的最下方，这是最符合观看习惯的位置。
5.  调整好第一段字幕后，后续字幕通常会同步应用相同的样式和位置，无需重复调整。

![](img/7505f04fedd69dfa2bfd66c45f48897f_27.png)

本节课中我们一起学习了视频剪辑中三个核心的后期处理技巧：**添加转场以平滑衔接片段**、**统一转场素材的比例**以及**利用识别功能快速添加与调整字幕**。记住，选择简短的转场、保持所有元素比例一致、添加清晰的字幕，是制作出专业、流畅视频的关键。严格按照这些步骤操作，你的视频将更容易通过平台审核，并获得更好的观看数据。
