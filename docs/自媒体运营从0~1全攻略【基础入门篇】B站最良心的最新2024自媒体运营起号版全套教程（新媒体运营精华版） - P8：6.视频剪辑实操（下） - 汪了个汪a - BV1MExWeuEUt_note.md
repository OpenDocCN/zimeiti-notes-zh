# 自媒体运营从0~1全攻略：P8：视频剪辑实操（下） 🎬

![](img/e80daa9efd2e91a4df3e0b71fc107009_0.png)

在本节课中，我们将继续学习视频剪辑的核心操作，重点掌握如何将非标准比例的素材调整为16:9横屏、如何添加转场效果以及如何自动识别并添加字幕。这些技巧能有效提升视频的专业度和观看体验。

上一节我们介绍了素材导入与基础剪辑，本节中我们来看看如何对素材进行精细化处理。

## 调整视频比例至16:9

在寻找素材时，并非所有视频都是标准的16:9横屏比例。如果遇到想使用但比例不符的素材，可以按以下步骤调整。

首先，在剪辑界面右下角找到并点击 **“比例”** 选项。

![](img/e80daa9efd2e91a4df3e0b71fc107009_0.png)

![](img/e80daa9efd2e91a4df3e0b71fc107009_2.png)

在弹出的菜单中，选择 **16:9** 的横屏比例。选择后，观察视频画面。如果视频主体在调整比例后没有移动或充满画面（例如视频主体较小，周围有大量黑边），则需要进行裁剪。

接下来，在视频轨道左侧找到并点击 **“剪辑”** 选项。

![](img/e80daa9efd2e91a4df3e0b71fc107009_2.png)

在打开的菜单中，选择 **“编辑”**，然后进入 **“裁剪”** 功能。裁剪时需注意：移除视频自带的字幕，并仔细裁剪掉多余的黑色边框，避免保留过多黑边影响最终观感。裁剪完成后，点击右下角的对勾确认。

![](img/e80daa9efd2e91a4df3e0b71fc107009_4.png)

此时，视频主体被放大，但可能仍非完整的16:9画面。最后一步，找到右下角的 **“背景”** 选项。

![](img/e80daa9efd2e91a4df3e0b71fc107009_4.png)

选择 **“画布模糊”**，并应用第一个最清晰的模糊样式。这样，视频背景会被模糊填充，从而形成一个完整的16:9横屏画面。再次点击对勾确认所有设置。

## 为视频添加转场

调整好比例后，下一步是为视频片段之间添加转场。转场的作用是明确提示观众一个片段结束、另一个不同的片段开始，这有助于维持观看节奏，提升视频完播率。

转场通常用于连接不同场景、人物或内容的视频片段。如果不添加转场，观众可能会感到突兀，甚至退出重看，这会严重影响视频的完播率数据。

以下是添加转场的步骤：

1.  在两个视频片段之间，点击轨道上的 **“+”** 加号按钮。
2.  在弹出的素材库中，选择 **“转场”** 或进入 **“热门”** 分类，里面有丰富的转场效果可供选择。

![](img/e80daa9efd2e91a4df3e0b71fc107009_6.png)
![](img/e80daa9efd2e91a4df3e0b71fc107009_7.png)
![](img/e80daa9efd2e91a4df3e0b71fc107009_8.png)

![](img/e80daa9efd2e91a4df3e0b71fc107009_6.png)

选择转场时需注意：**避免使用过长的转场**（如6秒、7秒甚至更长的）。观众的耐心有限，转场应保持简短。**3到5秒的转场时长最为合适**。

![](img/e80daa9efd2e91a4df3e0b71fc107009_7.png)

![](img/e80daa9efd2e91a4df3e0b71fc107009_8.png)

选择好一个约3秒的转场效果，点击添加即可。

![](img/e80daa9efd2e91a4df3e0b71fc107009_10.png)

添加后，同样需要确保转场素材的比例为16:9。选中转场片段，在右下角点击 **“背景”** -> **“画布模糊”**，选择第一个清晰样式进行调整。

## 识别与添加字幕

![](img/e80daa9efd2e91a4df3e0b71fc107009_10.png)

视频剪辑的最后关键一步是添加字幕。清晰的字幕能提升视频的可访问性和专业度。

添加字幕最简单的方法是使用软件的自动识别功能。

首先，在界面左下角找到并点击 **“文字”** 选项。

![](img/e80daa9efd2e91a4df3e0b71fc107009_12.png)

然后点击 **“新建文本”**，在右侧功能栏中找到 **“识别字幕”** 按钮。点击 **“开始匹配”**，软件便会自动识别视频中的语音并生成字幕条。

字幕识别完成后，只需调整其中一个字幕条的样式，后续字幕会自动统一。调整方法如下：
1.  选中一个字幕条，拖动右下角的方框将其**放大**到合适尺寸。
2.  将字幕条**拖动**到视频画面底部合适的位置，以确保观看舒适。

调整好一个字幕后，所有字幕都会应用相同的样式和位置。

![](img/e80daa9efd2e91a4df3e0b71fc107009_12.png)

本节课中我们一起学习了视频剪辑的三个核心进阶技巧：将任意素材调整为16:9横屏、添加转场效果以提升观看流畅度，以及利用自动识别功能快速添加字幕。掌握这些操作，你就能制作出比例规范、过渡自然、信息传达清晰的视频内容。整个过程并不复杂，关键在于耐心练习和灵活运用。
