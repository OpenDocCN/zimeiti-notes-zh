# 视频号运营教程：P37：二创书单制作方法二

## 概述
在本节课中，我们将学习制作二创书单视频的第二种方法。我们将通过一个具体案例，详细讲解如何利用现有视频素材，通过替换画面、重新配音等操作，制作出符合平台要求的原创内容。整个过程将涉及视频下载、剪辑、配音和最终合成。

![](img/b068ec0f28d426151088d24ba71e7ac7_1.png)

---

## 素材寻找与初步处理
上一节我们介绍了二创的基本思路，本节中我们来看看具体的操作步骤。首先，我们需要找到合适的原始视频素材。

![](img/b068ec0f28d426151088d24ba71e7ac7_3.png)

1.  在抖音搜索目标书籍名称，例如“走进大学城”。
2.  在搜索结果中选择“商品”标签，并按“销量”排序。
3.  在相关达人的视频作品中，寻找可用的素材。例如，一个关于“汕头大学”的1分58秒视频，其文案旨在激发孩子的大学梦。

![](img/b068ec0f28d426151088d24ba71e7ac7_5.png)

![](img/b068ec0f28d426151088d24ba71e7ac7_7.png)

对于找到的视频，如果其前后部分声音不一致，我们可以选择只使用前半部分，或处理整个视频。

![](img/b068ec0f28d426151088d24ba71e7ac7_9.png)

![](img/b068ec0f28d426151088d24ba71e7ac7_11.png)

**核心操作**：复制视频链接，使用去水印工具去除水印。

---

## 视频剪辑与文案提取
获得去水印的视频后，我们进入剪辑环节，目的是提取出我们需要的音频文案。

![](img/b068ec0f28d426151088d24ba71e7ac7_13.png)

1.  将视频导入剪映。
2.  在音频发生变化的时间点（例如53秒处）进行分割，删除不需要的后半部分，保留前半部分用于提取文案。
3.  将处理好的视频片段导出。

接下来，我们需要从这段视频中提取出文字文案。

**核心操作**：使用“配音神器”等工具的“视频提取文案”功能。
*   打开配音神器，选择通过相册视频提取文案。
*   导入上一步导出的视频片段，系统会自动识别并生成文字稿。

---

## 音频重新生成与画面替换
提取文案后，我们便可以生成新的、属于自己的音频，并开始替换原始视频画面，这是二创的核心。

以下是音频生成步骤：
1.  在配音神器中选择一个适合的声音模型。
2.  将提取的文案导入，生成新的配音音频并导出。

以下是画面替换步骤：
1.  回到剪映，打开原始视频。在需要替换的画面起点（如53秒处）分割。
2.  删除原视频中想要替换的片段（例如一段3秒的“毕业典礼”画面）。
3.  在抖音搜索符合当前文案意境的素材（如搜索“高校学生毕业”），下载并去水印。
4.  将新素材导入剪映，截取所需时长（如3秒），替换到原片段位置。
5.  在新旧片段之间添加转场效果（如叠化）。
6.  **关键步骤**：务必关闭新增片段的原声，将音量调整为0。

我们可以重复此过程，替换多个片段。例如：
*   将“打高尔夫”的画面替换为搜索到的“美女打高尔夫”视频。
*   将“宿舍单间”的画面替换为搜索到的“精装修室内”视频。
替换时，注意新画面与原文案意境、格调保持一致。

---

## 音画对齐与细节优化
替换完画面后，我们需要将新生成的配音与剪辑后的视频进行合成与对齐。

1.  关闭原视频的所有原声。
2.  导入之前用配音神器生成的新配音音频。
3.  检查音频时长与视频时长是否匹配。
    *   **音频过长**：可以适当加快视频播放速度（常规变速），或补充一些相关画面片段。
    *   **音频过短**：可以适当放慢视频速度，或删减部分冗余画面。
4.  确保音画基本同步后，删除多余部分。

对于视频中保留的、带有原声的片段（如人物采访片段），可以保留其声音，但**建议替换掉画面**，并使用其他符合语境的素材，避免同一人物面部特写出现时间过长，减少潜在版权风险。

---

## 最终润色与导出
所有元素组合完毕后，进行最后的调整，让视频更完整。

1.  **添加背景音乐**：新生成的配音没有背景乐，需在剪映中添加合适的背景音乐，调节音量使其不干扰配音。
2.  **生成字幕**：使用剪映的“识别字幕”功能，为新的配音生成字幕。此举可以覆盖掉原视频可能残留的旧字幕。
3.  **检查水印**：确保所有使用的素材均无第三方平台水印。如有，可通过去水印工具处理，或通过放大画面等方式将其裁剪出画面之外。
4.  **导出设置**：导出视频时，选择分辨率为1080P，帧率为60帧，码率选择推荐值，然后导出。

**最终成果**：经过以上步骤，你便得到了一个由自己重新配音、组合画面的二创书单视频。

## 总结
本节课我们一起学习了二创书单视频制作的全流程。核心路径是：**寻找优质素材 -> 提取文案 -> 重新配音 -> 替换画面 -> 音画对齐 -> 后期润色**。关键在于通过替换关键画面和声音，改变视频的“指纹”，使其成为平台认可的原创内容。初学者前期操作可能较慢，但熟练后，制作一个视频的效率将大幅提升。请记住，剪辑技能的核心在于“熟能生巧”，多加练习是成功的唯一捷径。
