# 视频号运营：P4：第二节：解剖推荐机制 🔍

在本节课中，我们将深入剖析视频号的核心推荐机制。理解这套机制是制作爆款视频、成功运营账号的基础。它与抖音等平台的逻辑有本质区别，掌握其独特之处至关重要。

上一节我们介绍了视频号的平台属性，本节中我们来看看它的核心推荐逻辑是如何运作的。

## 推荐机制的核心：社交链启动

![](img/75ed46749febc2d12191ec2afc0a619e_3.png)

![](img/75ed46749febc2d12191ec2afc0a619e_4.png)

视频号的推荐机制与抖音不同，其核心是基于社交链进行“强启动”。

**社交链**是指：当用户A点赞了某个视频，这个行为会通知给A的所有好友，并且该视频有很大概率会出现在A的好友的推荐流中。

这与抖音有显著区别。在抖音，用户可以关闭“朋友在看”功能，点赞行为相对私密，不会主动分发给好友。但在视频号，点赞是一个公开的社交行为，是流量扩散的起点。

![](img/75ed46749febc2d12191ec2afc0a619e_6.png)

## 流量扩散模型：从私域到公域

我们可以将视频号的流量扩散过程理解为三个板块：

![](img/75ed46749febc2d12191ec2afc0a619e_8.png)

1.  **初始用户冷启动**：内容首先发布给创作者的粉丝（初始用户）。
2.  **社交圈辐射**：初始用户中认可内容的用户会进行**点赞、转发**。一旦点赞，其所有好友（一环社交圈）都会看到该视频。
3.  **进入公域流量池**：如果内容在一环、二环（好友的好友）等社交链中获得了足够高的互动数据（达到某个**阈值**），平台会判定其为优质内容，将其推入更大的**公域流量池**，让更多陌生人看到。

![](img/75ed46749febc2d12191ec2afc0a619e_9.png)

**阈值**是一个关键概念。它指内容进入公域流量池所需达到的互动数据标准，不同领域阈值不同。例如，一个知识博主的阈值可能是 **200次点赞，辐射2000次观看**。

因此，你在视频号中看到的视频主要来自两类：
*   **社交推荐**：你好友点赞过的视频。
*   **公域推荐**：平台根据算法直接推荐的视频（你的好友未点赞）。

这两者紧密关联，社交推荐是打开公域推荐大门的钥匙。

## 流量分级：一级与二级传播

为了更直观地理解，我们可以将流量分为一级流量和二级流量。

*   **一级流量**：你的粉丝。发布内容后，首先推送给这部分用户。
*   **二级流量**：你粉丝的好友。当粉丝点赞后，视频会推荐给他的好友（二级流量）。二级流量的用户若继续点赞，则会继续辐射到三级流量（好友的好友），如此蔓延。

**公式表示**：`内容传播 = 一级流量辐射 + 二级（及多级）流量传播`

## 如何触发推荐：用户反馈是关键

要想内容进入爆款流量池，关键在于驱动用户产生正向反馈。平台主要依据以下用户反馈数据做判断：

![](img/75ed46749febc2d12191ec2afc0a619e_11.png)

以下是几个核心的用户反馈指标：
1.  **点赞/转发率**：权重最高。点赞直接触发社交链扩散。
2.  **完播率**：用户观看视频的完整程度。
3.  **互动率**：主要指评论率和关注率。

其中，**高点赞和高转发是冲击阈值、进入公域池的核心**。

## 驱动点赞的秘诀：以“情”动人

那么，如何制作出能获得高点赞的内容？根据运营经验，一个非常有效的方法是：**动情促进高互动，更易催生爆款**。

“情”包含多种，能有效击破用户心理防线，促使其点赞：
*   **感动之情**：如公益、亲情、爱情等感人故事。
*   **骄傲之情**：如国家成就、民族自豪感相关内容。
*   **共鸣之情**：如引发特定群体（如女性、父母、职场人）深度共鸣的内容。

这也解释了为何视频号头部爆款中，**情感类账号占比极高**（约40%-50%）。情感内容是驱动社交互动最强大的引擎之一。

## 平台的流量倾斜策略

![](img/75ed46749febc2d12191ec2afc0a619e_14.png)

了解平台对哪些账号有流量扶持，有助于我们定位自身。流量倾斜大致分为三级：

以下是平台流量倾斜的三个级别：
1.  **一级倾斜（最多）**：头部大号（尤其是情感等领域）。平台会直接给予大量公域流量扶持。
2.  **二级倾斜（其次）**：MCN机构旗下的账号。平台会提供一定的初始流量包或倾斜，根据内容反馈决定是否追加。
3.  **三级倾斜（较少）**：素人白号。这类账号没有先天流量优势，必须依靠**高互动、高反馈**的内容数据，才能撬动平台推荐，进入更大的流量池。

## 运营起号的核心策略

基于以上机制，一个核心问题浮现：对于没有初始粉丝（冷启动困难）的新号，该如何运营？

正确的策略是：**从公域平台获取初始流量**。

![](img/75ed46749febc2d12191ec2afc0a619e_16.png)

具体操作：不要只局限于视频号。可以同步在抖音、小红书等公域平台运营，将其他平台的粉丝引导至视频号，作为你的**初始种子用户**。拥有这批初始用户后，再专注于在视频号内制作能引发他们高互动（点赞、转发）的内容，从而触发社交链扩散，并最终冲击阈值，获得视频号平台本身的公域流量推荐。

![](img/75ed46749febc2d12191ec2afc0a619e_17.png)

![](img/75ed46749febc2d12191ec2afc0a619e_18.png)

本节课中我们一起学习了视频号独特的推荐机制。其核心是**基于社交链的冷启动**，通过**点赞触发社交辐射**，凭借**高互动数据冲击阈值**，最终进入**公域流量池**。理解并利用好“以情动人”的内容法则和“从公域引流转私域”的起号策略，是做好视频号运营的关键。下一节，我们将探讨如何具体规划你的视频号内容。
