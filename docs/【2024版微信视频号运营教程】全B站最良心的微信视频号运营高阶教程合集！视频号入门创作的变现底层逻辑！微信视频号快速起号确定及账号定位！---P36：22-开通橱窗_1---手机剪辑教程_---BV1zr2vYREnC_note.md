# 微信视频号运营教程：P36：开通橱窗与添加商品

## 概述
在本节课中，我们将学习如何为微信视频号开通商品橱窗功能，并完成首次直播认证与商品上架。这是实现视频号变现的关键步骤。

上一节我们掌握了视频剪辑与账号养号的方法，本节中我们来看看如何将流量转化为实际收入。

---

## 开通商品橱窗权限
首先，我们需要在视频号中开通带货权限。

1.  打开微信，进入“视频号”页面。
2.  点击右上角的“人像”图标，进入个人中心。
3.  找到并点击“创作者服务中心”。
4.  在服务中心内，找到“带货权限”选项并点击进入。

以下是开通橱窗的具体操作流程：
*   **缴纳保证金**：进入“带货权限”页面后，你会看到“待缴纳”状态。点击此处，系统会引导你完成保证金的缴纳。
*   **身份认证**：在缴纳保证金前，需要进行身份认证。请务必选择“个人”身份，并提前准备好身份证信息以便填写。
*   **提交审核**：填写并提交信息后，状态会变为“待审核”。审核时间通常为几小时到一天不等。
*   **完成开通**：审核通过后，返回此页面点击“充值缴纳”，完成保证金支付，橱窗功能即开通。

## 完成首次直播认证
在开始添加商品前，有一个至关重要的准备工作：完成首次直播的实名与人脸认证。

如果未完成此认证，即使开通了橱窗也无法开播。因此，请务必提前操作。

操作步骤如下：
1.  在视频号个人主页，点击“发起直播”。
2.  首次点击时，系统会要求你完成“实名认证”和“人脸认证”。
3.  请确保认证者本人在场，因为人脸认证需实时进行。
4.  认证成功后，再次点击“发起直播”将直接进入开播界面，无需重复认证。

**核心提示**：`务必在计划开播前，提前完成此次认证`，避免因认证人不在场而影响直播计划。

## 为橱窗添加商品
开通权限并完成认证后，即可为橱窗添加商品。通常，你可以从合作的商家或团队获取商品链接。

添加商品的方法如下：
1.  获取商品专属的“橱窗码”。此码通常由供应商提供。
2.  打开微信的“扫一扫”功能。
3.  扫描该“橱窗码”。
4.  根据页面提示（可能需再次确认身份信息），即可将商品成功添加至你的视频号橱窗。

## 如何寻找货源与商品
初学者常问：商品链接和“橱窗码”从哪里获取？

常见的渠道是加入相关的电商或带货社群。在这些群聊中，商家会直接发布带样品的商品链接、推广话术和专属的橱窗码。

**操作建议**：你可以联系你的课程老师或同行，请求加入可靠的货源对接群。在群内，你可以直接扫码获取商品，丰富自己的橱窗。

## 总结
本节课中我们一起学习了视频号变现的启动流程。关键步骤有三点：第一，在创作者服务中心开通橱窗并缴纳保证金；第二，提前完成首次直播的人脸认证，扫清开播障碍；第三，通过扫描供应商提供的“橱窗码”为店铺添加商品。完成这些准备后，你的视频号就具备了通过直播和短视频带货变现的基础能力。
