# 小红书运营教程：P15：小红书运营配套软件分享 📦

在本节课中，我们将学习小红书运营过程中常用的一些配套软件和工具。这些工具能帮助你更高效地收集素材、查询敏感词、分析数据，从而提升运营效率。

上一节我们介绍了运营的核心策略，本节中我们来看看有哪些实用的工具可以辅助我们的工作。

## 官方资料与实用工具

首先，课程提供了一份**官方资料包**，其中包含小红书平台提供的基础运营技巧。官方资料主要阐述平台的基本规则和推荐机制，但对于一些深度的、基于实际运营经验的技巧可能不会涉及。因此，这份资料更适合新手运营者进行初步学习和了解。

除了官方资料，掌握自查自纠的能力也很重要。以下是课程整理和筛选出的一些核心实用工具，这些工具会持续维护和更新。

1.  **敏感词查询工具**
    *   用于检测笔记内容中是否包含平台禁止或限制的词汇，帮助避免违规。

![](img/cb929b9d24715b6f1816ab20745a6b56_5.png)

2.  **数据统计与分析APP**
    *   用于追踪账号数据（如粉丝增长、笔记互动量）、分析热门话题和竞品账号。

3.  **图片与素材收集工具**
    *   帮助高效收集和管理来自小红书、微博等平台的优质图片和文案素材。

4.  **图片处理工具（如莱卡风格）**
    *   用于对图片进行调色、加滤镜、排版等处理，提升笔记的视觉吸引力。

![](img/cb929b9d24715b6f1816ab20745a6b56_3.png)

## 工具获取与使用说明

这些工具资源会持续更新。课程录制于特定时间（如6月），但后续发现更好用的工具也会补充进来。由于视频内容无法实时更新，建议学员可以定期（例如每月）向课程提供方咨询是否有新的工具更新。

![](img/cb929b9d24715b6f1816ab20745a6b56_5.png)

所有工具将免费提供给本课程的学员。具体工具的获取地址和使用方法已整理在配套资料中。

其中，**图片收集工具**非常实用。当你在小红书、微博等平台看到喜欢的文字或图片素材，但不知来源或素材带有水印时，可以使用该工具提供的方法进行高效收集和处理。
