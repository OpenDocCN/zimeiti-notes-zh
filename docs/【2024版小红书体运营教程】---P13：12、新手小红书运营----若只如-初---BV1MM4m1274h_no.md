## 笔记被推荐的核心逻辑

笔记发布后，平台会给予一个基础的展示量。能否获得更多流量，取决于笔记能否通过平台的“数据考核”。其核心逻辑分为两个关键阶段：**打开率**和**互动率**。

### 第一阶段：打开率考核

打开率是笔记获得更多流量的第一道门槛。

当用户打开小红书App，首页“发现”页会展示4到6篇笔记。用户向下滑动，平台会一次性加载约20篇笔记，然后刷新。

*   **考核标准**：你的笔记需要在这批20篇同类型笔记中，**打开率（点击率）达到平均11%以上**。
*   **计算方式**：`打开率 = 笔记阅读量（小眼睛数） / 笔记展示量`。
*   **运作机制**：例如，平台给你的笔记1000次基础展示。如果阅读量超过110（即1000 * 11%），系统会判定笔记有吸引力，从而进入下一级流量池，展示量可能跃升至2000-5000。如果打开率不达标，推荐通常就会停止。

**账号权重越高，获得的基础展示量也越高。但无论基础量多少，打开率都是升级的第一关键。**

### 第二阶段：互动率考核

![](img/8943b9020fc016bb8f47db14a7688b06_9.png)

通过打开率考核后，笔记进入更大的流量池。此时，**互动率**成为能否继续升级的决定因素。

![](img/8943b9020fc016bb8f47db14a7688b06_11.png)

![](img/8943b9020fc016bb8f47db14a7688b06_13.png)

![](img/8943b9020fc016bb8f47db14a7688b06_15.png)

*   **考核标准**：笔记的**平均互动率需达到3%以上**。
*   **计算方式**：`互动率 = （点赞+收藏+评论+转发等互动数） / 笔记阅读量`。
*   **运作机制**：假设笔记有2000次阅读，互动数需要超过60（即2000 * 3%），才有机会进入万级甚至十万级的展示流量池。

**流量升级是阶梯式的**：新号通常有机会从第一级升至第三级；老号因基础权重高，可能从更高级别起步，并有潜力升至更高层级。达到顶级流量（如第六、七级）通常需要内容极其优质，并配合商业推广。

### 用户行为的权重评分

除了打开率和互动率，用户的每一个具体行为都对笔记的“评分”有影响。

以下是平台评估笔记质量的核心互动维度及其权重（示意）：
```
综合评分 = f(点赞权重, 收藏权重, 评论权重, 关注权重, 分享权重)
```
**注意**：这些互动权重建立在打开率和互动率达标的基础上。如果基础数据不达标，互动权重再高也难以获得大量推荐。

## 必须避免的运营“坑”

了解推荐逻辑后，我们还需要规避违规行为，以免账号被降权。以下是新手运营中常见的错误：

![](img/8943b9020fc016bb8f47db14a7688b06_21.png)

![](img/8943b9020fc016bb8f47db14a7688b06_23.png)

*   **新号切勿立即发笔记**：新注册账号需要“养号”，即先浏览、互动，让平台识别为正常用户。立即发笔记会浪费初始权重，导致后续流量低迷。
*   **违规必然导致降权**：任何违规（如引流、抄袭、违禁内容）都会导致笔记推荐和搜索排名大幅下降，恢复难度极大。
*   **标签切忌反复修改**：账号、店铺或人群标签一旦确定，频繁修改会打乱平台对你的识别，容易导致限流。
*   **避免一机多号**：一台设备尽量只登录一个账号，最多不超过三个，过多可能被判定为营销号。
*   **谨慎处理推广与引流**：在笔记或私信中直接发布联系方式、二维码等引流信息，是高风险行为。
*   **谨慎处理用户互动**：对留言、私信中的违规或敏感内容要妥善处理，避免牵连账号。
