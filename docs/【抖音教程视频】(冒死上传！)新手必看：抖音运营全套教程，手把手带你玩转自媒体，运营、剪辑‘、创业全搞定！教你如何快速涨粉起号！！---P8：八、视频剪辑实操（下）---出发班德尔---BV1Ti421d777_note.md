## 调整非标准比例素材

![](img/3ad91d77c9e663a390ed3fb72b96ca1c_5.png)

在寻找素材的过程中，你找到的视频素材可能并非标准的16:9横屏比例。例如，我们当前使用的这个视频就不是16:9横屏。如果你想使用这类素材，就需要手动进行调整。

以下是调整非标准比例素材的步骤：

1.  **调整基础比例**：在剪辑界面右下角找到“比例”选项，点击后选择“16:9”横屏模式。
2.  **检查视频适配情况**：选择比例后，观察视频画面。如果视频主体（如人物）在画面中静止不动，导致画面留白过多，就需要进一步裁剪。
3.  **进入裁剪编辑**：点击视频轨道左侧的“剪辑”选项，在弹出的菜单中选择“编辑”，然后选择“裁剪”功能。
4.  **精细裁剪画面**：在裁剪界面中，将视频自带的字幕和多余的黑边全部裁掉。操作时需仔细，避免留下过多黑色边框，否则会影响最终画面的美观度。裁剪完成后，点击右下角的对勾确认。
5.  **填充背景**：裁剪后视频可能仍非标准16:9。此时，在右下角找到“背景”选项，选择“画布模糊”，并应用第一个最清晰的模糊效果。这样，视频就拥有了完整的16:9横屏背景。

通过以上步骤，我们成功将一个非标准比例的视频素材，调整为了适合平台发布的16:9横屏格式。

## 为视频添加转场

![](img/3ad91d77c9e663a390ed3fb72b96ca1c_7.png)

调整好视频比例后，下一步是为视频片段之间添加转场。转场是连接不同视频内容的过渡片段，例如在盘点类视频中，两个内容之间常会插入一个搞笑的短片段。

![](img/3ad91d77c9e663a390ed3fb72b96ca1c_9.png)

![](img/3ad91d77c9e663a390ed3fb72b96ca1c_11.png)

添加转场的目的在于，明确告知用户一个视频段落已经结束，即将开始另一个全新的内容。如果不添加转场，用户可能会对内容的突然切换感到困惑，甚至退出视频重新观看，这会严重降低视频的完播率，从而影响视频的推荐和收益。

以下是添加转场的具体方法：

![](img/3ad91d77c9e663a390ed3fb72b96ca1c_13.png)

1.  **找到转场素材**：点击轨道右侧的“+”号，进入“素材库”，在“热门”分类中可以找到大量现成的转场素材。
2.  **选择合适的转场**：选择你喜欢的转场样式。需要注意的是，转场时长不宜过长，6秒、7秒甚至更长的转场会消耗用户耐心。**转场时长应保持在3到5秒之间**最为合适。选择一个约3秒的转场素材，将其添加到视频轨道中。
3.  **统一转场比例**：添加的转场素材也需要保持16:9的横屏比例。选中转场片段，在右下角同样使用“画布模糊”功能，选择第一个清晰的模糊效果，使其与主视频比例一致。

![](img/3ad91d77c9e663a390ed3fb72b96ca1c_15.png)

## 自动识别与添加字幕

![](img/3ad91d77c9e663a390ed3fb72b96ca1c_17.png)

转场添加完毕后，最后一步是为视频添加字幕。手动输入字幕非常耗时，我们可以利用软件的自动识别功能。

以下是识别和添加字幕的步骤：

![](img/3ad91d77c9e663a390ed3fb72b96ca1c_19.png)

1.  **启动字幕识别**：点击剪辑界面左下角的“文字”选项，然后选择右侧的“识别字幕”功能，点击“开始匹配”。软件会自动分析视频中的语音，并将其转换为文字字幕。
2.  **调整字幕样式**：识别完成后，只需调整其中一条字幕的样式即可，后续字幕会自动应用相同样式。通常的操作是：选中一条字幕，拖动右下角的方框将其放大，然后将整条字幕条拖动到视频画面的底部位置。这样调整后，所有字幕都会统一为放大并位于底部的样式，观感更加舒适。

![](img/3ad91d77c9e663a390ed3fb72b96ca1c_21.png)

本节课中我们一起学习了如何处理非标准比例的视频素材、如何添加有效的视频转场以及如何快速为视频添加字幕。这些操作看似复杂，但按步骤进行其实非常简单。记住核心流程：**调整比例 -> 添加转场 -> 识别字幕**。按照这个模式操作，你就能制作出完成度高、观感专业的视频内容。
