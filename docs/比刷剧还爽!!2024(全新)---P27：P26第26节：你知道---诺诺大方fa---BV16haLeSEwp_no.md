# 小红书运营：P27：自建账号与官方广告投放详解

## 概述
在本节课中，我们将探讨小红书运营中的一个核心问题：是自建账号进行内容引流更好，还是与官方合作投放广告更有效。我们将理清这两者的关系，并详细介绍小红书的账号类型、企业号矩阵的打造方法，以及官方广告的投放流程与类型。

## 小红书账号类型解析
上一节我们介绍了账号定位，本节中我们来看看小红书平台上有哪些具体的账号类型。了解不同类型的账号及其用途，是制定运营策略的基础。

小红书账号主要分为以下三类：

1.  **企业号**
    *   主要作用是树立官方品牌形象。
    *   拥有官方认证标识，可展示品牌名称、地址等信息。
    *   享有官方特权，便于开展相关业务。

2.  **个人号**
    *   目前大多数用户运营的账号类型，特点是**灵活性高**。
    *   运营者可以是品牌员工、产品消费者等角色。
    *   易于塑造个人品牌特色，适合员工或相关人物出镜。

3.  **引流号**
    *   在之前的课程中提过，俗称“炮灰号”。
    *   数量可以是多个，作用相对次要，无需精细运营。
    *   主要用于特定引流目的，或在紧急情况下作为备用小号使用。

通常，品牌的推广渠道依赖于企业号、个人号或引流号自行发布笔记进行引流。

## 打造企业号矩阵的建议
了解了账号分类后，我们来看看如何协同使用这些账号，构建一个有效的运营矩阵。以下是打造企业号矩阵的具体建议：

1.  **积极参与品牌号活动**
    *   使用多个小号参与企业号发起的试用、团购等活动，增加活动热度与真实性。

![](img/cbee0d000ac5d1b42172fac2dec3d555_1.png)

2.  **个人号展示品牌属性**
    *   在个人号的笔记内容中，应多次暗示品牌的定位属性（例如，目标用户是都市青年还是职场新人）。

![](img/cbee0d000ac5d1b42172fac2dec3d555_3.png)

![](img/cbee0d000ac5d1b42172fac2dec3d555_5.png)

3.  **控制个人号营销内容比例**
    *   个人号发布品牌产品内容的频率不宜过高。建议遵循 **“三七分”原则**：`30%` 的内容用于营销推广，`70%` 的内容用于种草分享或知识科普。

4.  **利用引流号进行评论引流**
    *   企业号发布内容后，可使用引流号在评论区进行互动或引导，实现更自然的流量转化。

## 官方广告投放：小红书CBC广告
除了自建账号矩阵，另一种重要的推广方式是投放小红书官方广告。下面我们来详细了解这种付费推广模式。

![](img/cbee0d000ac5d1b42172fac2dec3d555_11.png)

小红书官方广告即 **CBC（Cost Per Click）竞价广告**。其申请入口位于小红书官网的“业务合作” - “推广合作”板块。但请注意，投放官方广告有一个前提：**必须先认证企业号**。

![](img/cbee0d000ac5d1b42172fac2dec3d555_13.png)

企业号认证流程同样在官网完成，需提交相关资质并支付审核费用，通过后普通账号即可升级为官方认证账号。

### 官方广告的主要类型
拥有企业号后，可以投放的官方广告主要分为两大类：

1.  **信息流广告**
    *   当用户浏览小红书首页信息时，系统会不定期间隔插入广告内容。

2.  **搜索广告**
    *   当用户使用关键词进行搜索时，系统会根据算法匹配并展示相关的广告内容。例如，用户搜索“考研资料”，可能会看到能量补充类商品的广告。

搜索竞价广告的核心原理是 **关键词竞价排名**，即广告主对特定关键词出价，出价越高，广告获得的展示位通常越好。目前，信息流广告常见的展示位序是第3位、第13位和第23位。

![](img/cbee0d000ac5d1b42172fac2dec3d555_15.png)

关于广告投放的更多常见问题，可以在小红书官方平台的问答页面找到详细解答。

![](img/cbee0d000ac5d1b42172fac2dec3d555_17.png)

**请注意**：官方广告投放的费用相对较高。在资金允许的情况下，与官方合作投放竞价广告是获取精准推荐流量的有效方式。

![](img/cbee0d000ac5d1b42172fac2dec3d555_19.png)
