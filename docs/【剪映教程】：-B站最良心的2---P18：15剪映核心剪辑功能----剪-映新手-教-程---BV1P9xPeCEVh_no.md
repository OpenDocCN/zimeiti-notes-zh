# 剪映核心剪辑功能：P18：文本功能详解 📝

在本节课中，我们将要学习剪映的核心功能之一——文本功能。一个完整的视频不仅包含画面和背景音乐，文本或字幕也是传递信息、突出主题的关键元素。通过添加文本，可以更准确、更快速地让观众理解视频内容。

![](img/d75563cc7d188412f2341e5eaf34b33d_1.png)

上一节我们介绍了音频功能，本节中我们来看看如何为视频添加和设置文本。

## 文本功能概述

在剪映中，我们可以自由地为视频添加字幕。添加的字幕可以单独设置样式，包括字体、字号、颜色、间距以及动画效果。此外，剪映还提供了两个高效的功能：**自动识别字幕**和**识别歌词**。这两个功能可以基于视频中的语音或音频，一键快速生成字幕，极大地降低了操作难度和时间成本。文本功能模块中还包含添加贴纸的选项。

## 功能实操演练

文本功能位于一级菜单栏。点击底部菜单栏的“文字”按钮，即可进入文本功能界面，你会看到四个主要选项：**新建文本**、**识别字幕**、**识别歌词**和**贴纸**。

以下是每个功能的详细操作步骤。

### 1. 新建文本（手动添加）

这是最基础的添加字幕方式。

1.  在剪辑界面，点击底部菜单栏的 **`文字`** 按钮。
2.  选择 **`新建文本`**。
3.  在输入框中键入你想添加的文字，例如“剪映视频剪辑教程”。
4.  输入完成后，时间轴上会出现一个对应的文本轨道。

![](img/d75563cc7d188412f2341e5eaf34b33d_3.png)

添加文本后，可以对其样式进行个性化设置。点击选中文本轨道，在底部菜单中可以进行以下调整：

*   **样式**：可以更改字体、设置描边、添加标签、阴影，调整字间距和对齐方式等。
    *   例如，选择“新青年体”字体。
    *   添加红色描边，并调整描边粗细。
*   **花字**：可以直接应用抖音官方提供的各种特色艺术字体。
*   **气泡**：可以为文字添加对话气泡样式，常用于采访或Vlog。
*   **动画**：可以为文字添加入场、出场或循环动画，让文字更生动。
    *   例如，选择“螺旋上升”作为入场动画，并设置动画时长。

![](img/d75563cc7d188412f2341e5eaf34b33d_5.png)

![](img/d75563cc7d188412f2341e5eaf34b33d_6.png)

### 2. 识别字幕（语音转文字）

如果你的视频包含人声对话或旁白，可以使用此功能自动生成字幕。

![](img/d75563cc7d188412f2341e5eaf34b33d_8.png)

1.  确保视频轨道上已添加包含人声的音频。
2.  点击 **`文字`** -> **`识别字幕`**。
3.  点击 **`开始识别`**。
4.  剪映会自动分析音频，并将识别出的文字以字幕形式添加到视频中。识别完成后，可以像编辑“新建文本”一样，对这些自动生成的字幕进行样式修改。

![](img/d75563cc7d188412f2341e5eaf34b33d_10.png)

### 3. 识别歌词（音频转歌词）

![](img/d75563cc7d188412f2341e5eaf34b33d_12.png)

![](img/d75563cc7d188412f2341e5eaf34b33d_14.png)

如果你为视频添加了带歌词的歌曲，可以使用此功能快速生成歌词字幕。

1.  为视频添加一首带歌词的歌曲（如从抖音收藏添加）。
2.  点击 **`文字`** -> **`识别歌词`**。
3.  点击 **`开始识别`**。
4.  系统会自动识别歌曲，并将歌词同步到视频时间轴上。生成后，同样可以调整歌词的字体、位置、颜色，并添加动画效果以提升观感。

### 4. 贴纸

![](img/d75563cc7d188412f2341e5eaf34b33d_16.png)

![](img/d75563cc7d188412f2341e5eaf34b33d_17.png)

除了文字，你还可以在“文字”功能模块中找到“贴纸”选项，为视频添加各种有趣的图形元素，丰富画面内容。

![](img/d75563cc7d188412f2341e5eaf34b33d_19.png)

## 总结

![](img/d75563cc7d188412f2341e5eaf34b33d_21.png)

本节课中我们一起学习了剪映的文本功能。我们掌握了如何通过**新建文本**手动添加字幕，以及如何利用**识别字幕**和**识别歌词**功能高效地自动生成字幕。同时，我们也学会了如何对字幕的字体、颜色、动画等样式进行个性化设置，并了解了添加贴纸的方法。合理运用文本功能，可以显著提升视频的信息传达效率和整体内容质量。
