# 视频号运营：P12：2.视频来源处理

在本节课中，我们将学习视频号运营中获取和处理素材的两个核心技能：如何为视频去除水印，以及如何提取视频中的文案。这些是进行视频搬运和二次创作的基础步骤。

上一节我们介绍了视频素材的来源，本节中我们来看看如何对这些素材进行初步处理，使其符合我们的使用要求。

![](img/3f5fb16c0b274967bd0def04448dc87d_1.png)

## 🧹 视频去水印

搬运视频时，直接下载的视频通常带有平台水印。我们需要使用工具去除这些水印。以下是详细的操作步骤。

**操作流程公式：** `复制视频链接` -> `打开去水印工具` -> `粘贴链接` -> `保存无水印视频`

1.  **在抖音找到目标视频**：打开抖音，搜索或找到你想使用的视频。例如，搜索“擦丝器”。
2.  **复制视频链接**：点击视频分享按钮（箭头图标），选择“复制链接”。
3.  **使用去水印工具**：打开微信，在搜索框输入“去水印”，会出现许多相关小程序。例如“青抖去水印”。
4.  **处理并保存**：进入小程序后，点击“去水印”功能，粘贴刚才复制的链接，然后点击“一键去水印”。根据提示（如观看一段广告）后，即可将无水印视频保存到手机相册。

> **注意**：有些视频可能关闭了下载或复制链接功能，如果遇到这种情况，请更换其他视频。

## 📝 提取视频文案

除了视频画面，视频中的文案（旁白、字幕）也是重要的创作素材。我们可以轻松地将文案提取出来。

![](img/3f5fb16c0b274967bd0def04448dc87d_5.png)

**操作流程公式：** `复制视频链接` -> `打开文案提取工具` -> `粘贴链接` -> `复制提取出的文案`

![](img/3f5fb16c0b274967bd0def04448dc87d_6.png)

以下是推荐的工具和步骤。

**推荐工具：配音神器Pro**
这款工具在提取文案和去水印时通常无需观看广告，体验更佳。

1.  **获取工具**：在微信中搜索“配音神器Pro”小程序并打开。
2.  **提取文案**：在工具首页找到“文案提取”功能，将抖音视频的链接粘贴进去，点击确定。
3.  **复制文案**：稍等片刻，工具就会将视频中的语音识别成文字并展示出来，直接复制即可使用。

> **提示**：配音神器Pro不仅支持从抖音、快手等平台提取文案，也支持提取手机相册中本地视频的文案。
