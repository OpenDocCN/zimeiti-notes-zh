# 抖音运营教程：P16：11、视频类型1-2 - 人设与口播

在本节课中，我们将要学习抖音账号运营中至关重要的两个概念：**人设**的建立与**口播**视频的制作。理解并应用这些知识，是连接内容创作与粉丝增长、实现变现的关键环节。

上一节我们解决了文案问题，本节中我们来看看如何通过建立人设和选择视频类型来吸引并留住观众。

## 人设：让你的账号被记住

从单纯追求涨粉到最终实现变现，并非一条直线。中间有一个核心环节，就是建立你的**人设**。建立人设的目的不是为了炫耀，而是为了在众多同类账号中脱颖而出。例如，有十个人都做装修内容，如果其中一位建立了幽默搞笑的人设，观众就更容易记住他，从而忽略其他九位。当观众喜欢你这个人时，自然就会成为你的粉丝。

人设主要分为两种：**强人设**与**弱人设**。区分清楚这两种人设，有助于你更好地定位自己的账号。

![](img/77b5837aa897f2569475a31e42ce8470_1.png)

以下是两种人设的具体说明：

*   **强人设**
    *   **核心**：以**你个人**为切入点，内容与你自己密切相关。
    *   **内容方向**：
        *   分享**行业经验**（如装修、办公、房产、汽车、餐饮、婚介、法律咨询、电商等）。
        *   分享**个人兴趣爱好**（如美食、旅行、手工、宠物、星座、美妆、护肤、服装穿搭、游戏、摄影等）。
    *   **形式特点**：通常需要**真人出镜**，通过分享工作、生活或爱好来塑造鲜明的个人形象。

![](img/77b5837aa897f2569475a31e42ce8470_3.png)

*   **弱人设**
    *   **核心**：内容与创作者本人**关系不大**，主要目的是运营好账号本身。
    *   **内容举例**：
        *   **情感语录类**：配乐视频加上文字，无真人出镜或配音。
        *   **影视解说类**：虽有解说声音，但核心是电影内容，与解说者个人特质关联弱。
    *   **形式特点**：不依赖真人形象，重点在于内容主题的呈现。

清晰地区分强弱人设，是规划账号内容的第一步。

## 内容创作：从“讲故事”开始

明确了人设方向后，任何与你个人相关的内容都可以成为视频素材。我们首先从最基础且有效的一种形式讲起：**讲故事**。

讲故事可以分享客户故事、从业经历或人生感悟。故事本身不要求完全真实，可以从文案平台寻找有趣的内容进行改编分享。而讲故事最直接的方式，就是通过**口播**。

### 口播视频详解

口播，即面对镜头进行讲述。它是一种应用广泛且易于上手的视频形式。

以下是常见的口播分类：

*   **故事型口播**：如情感口播、养生故事、育儿经历等。目的是让观众觉得内容有趣、有共鸣。
*   **知识型口播**：如房产、汽车、金融、财经、装修、法律等领域的知识分享。目的是在几十秒内给观众带来**收获感**，让他们觉得学到了东西。

对于想做口播的同学，需要明确以下几个关键点：

1.  **领域广泛**：法律、保险、金融、装修、汽车等知识科普类内容都适合做口播。
2.  **拍摄简单**：只需准备好手机和文案，即可开始拍摄。
3.  **口才要求**：口播依赖的是**提前准备好的稿子**，而非临场发挥。口才并非决定性因素。
4.  **普通话问题**：普通话不标准也可以做口播。如果你的目标客户在同城，使用**方言**可能更具亲和力和趣味性，同时可以通过添加字幕确保信息传达。

口播是一种可以快速启动并持续运营的内容形式。

本节课中我们一起学习了抖音账号运营的核心环节：**建立人设**与制作**口播视频**。我们明确了强人设与弱人设的区别，并详细介绍了口播视频的类型、优势及实操要点。掌握这些基础，是打造具有辨识度和吸引力的抖音账号的重要一步。
