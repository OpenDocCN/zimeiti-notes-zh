# 小红书运营新手教程：P10：小红书规则（1）📚

## 什么是小红书规则？🤔

![](img/9fda1c5ef6b97786dfb013e01a460531_5.png)

![](img/9fda1c5ef6b97786dfb013e01a460531_7.png)

小红书的规则，是平台为了维护社区环境、规范用户行为而设立的一系列准则。它决定了你的笔记能否被系统推荐、账号是否安全。如果不了解这些规则，即使内容优质，也可能无法获得流量，甚至导致账号受限。

## 小红书规则的四大要点

![](img/9fda1c5ef6b97786dfb013e01a460531_13.png)

![](img/9fda1c5ef6b97786dfb013e01a460531_15.png)

小红书规则内容繁多，但可以归纳为四个核心要点。以下是这四个要点的简要介绍，本节课我们将重点讲解第一个要点：**推荐规则**。

![](img/9fda1c5ef6b97786dfb013e01a460531_17.png)

1.  **推荐规则**：系统如何决定将你的内容展示给哪些用户。
2.  **审核规则**：平台对内容合规性的基本要求。
3.  **社区公约**：用户行为的基本准则与价值观。
4.  **商业规范**：涉及广告、营销、交易等商业行为的特殊规定。

## 核心要点一：推荐规则详解

上一节我们介绍了规则的四个要点，本节中我们来看看最重要的**推荐规则**。这是决定你笔记曝光量的核心机制。

推荐规则是指小红书系统通过分析你的账号信息、发布内容及用户互动数据，来判定账号属性，并据此决定向哪些用户推荐你的内容、给予多少初始流量曝光的过程。理解并优化这个过程中的“权重”，是做好数据的关键。

推荐规则主要围绕以下三个方面展开：

### 1. 用户行为规范

![](img/9fda1c5ef6b97786dfb013e01a460531_19.png)

用户行为规范明确了在平台上**允许**和**禁止**的互动方式。以下是需要特别注意的禁止行为列表，违反这些规定将严重影响账号权重。

*   **严禁行为（绝对不能做）**：
    *   **包含营销词汇**：笔记中不得直接出现“微信号”、“二维码”、“代购”、“水印”、“某某店铺”等明显引流或广告词汇。
    *   **直接导流**：试图将小红书用户引导至微信、QQ群、微博等外部平台进行交易或沉淀，在笔记正文或评论区直接操作风险极高。

*   **允许行为（可以谨慎操作）**：
    *   **使用谐音或话术引导**：例如，将“微信”写作“V心”或“薇信”，在文案中通过巧妙的话术暗示。
    *   **利用邮箱或微博**：个人资料中可以留邮箱，或引导至微博（相对宽松）。
    *   **小号互动导流**：通过运营小号，在主号的笔记评论区以“官方”口吻置顶评论并@小号，间接引导用户关注小号（小号可承载更多商业信息）。这是一种常见的规避策略，但需注意频率和方式。

**核心技巧**：对于“某某店铺”这类信息，避免在笔记中直接宣传。可以采用“小号互动法”，即用主号在评论区置顶一条看似官方的评论，如“更多款式可以看@XXX（你的小号）”，来实现间接导流。

![](img/9fda1c5ef6b97786dfb013e01a460531_23.png)

### 2. 内容安全与责任（自序商品）

![](img/9fda1c5ef6b97786dfb013e01a460531_25.png)

![](img/9fda1c5ef6b97786dfb013e01a460531_27.png)

这部分规范主要涉及内容的法律和社会责任风险。平台对此管控严格，一旦触犯，后果严重。

*   **严禁行为（高风险领域）**：
    *   **提供医疗建议**：例如，“这个病去XX医院看很好”、“吃XX药有效”。这涉及专业医疗 advice，风险极高。
    *   **提供投资建议**：例如，“这只股票会涨”、“那个楼盘买了肯定赚”。这涉及金融 advice，同样属于敏感禁区。
    *   **账号登录限制**：在**电脑端**操作时，同一设备（即使使用模拟器分身）建议最多同时登录**3个**账号，超过此数量易被系统检测为批量营销行为，导致账号风险。操作多账号时，建议使用“雷电模拟器”等工具进行环境隔离，并注意更换IP。

**重要提示**：医疗和投资是两大红线，无论粉丝多么信任你，都绝对不能在笔记或互动中给出具体建议，这不仅是规则问题，更可能涉及法律责任。

![](img/9fda1c5ef6b97786dfb013e01a460531_29.png)

### 3. 笔记发布规范（图文/视频）

这部分规范直接关系到你发布的每一篇笔记能否通过审核并获得推荐。

![](img/9fda1c5ef6b97786dfb013e01a460531_31.png)

*   **严禁行为（影响推荐的行为）**：
    *   **诱导互动**：例如，“关注我领取资料”、“评论抽奖送礼物”。平台严格打击以利益诱导关注、点赞、评论的行为。
    *   **低原创度/抄袭**：直接复制粘贴他人的图片和文案，账号权重会变得极低，甚至被“废掉”。系统鼓励高原创度的内容。

*   **允许与建议（提升权重的行为）**：
    *   **高原创内容**：原创度越高，前期获得的基础流量扶持就越大。即使是50%原创（如借用网络图片但配以自己撰写的深度文案），也能获得一定流量。
    *   **合规的店铺活动**：如果想做促销，可以将活动信息放在**商品详情页**或**店铺主页**，而不是直接在笔记标题或正文中做“关注领奖”式的引导。虽然这样搜索权重较低，但是合规的做法。

**给新手的建议**：对于新人来说，从零开始做高原创内容可能难度较大。一个折中的起步方法是：寻找其他平台的优质图片，但务必自己重新撰写文案，形成“伪原创”。这样起步难度（约3级）远低于完全抄袭（账号作废）或纯原创（约10级），是相对安全且有效的冷启动策略。

## 课程总结

![](img/9fda1c5ef6b97786dfb013e01a460531_33.png)

本节课中，我们一起学习了小红书运营的核心——**平台规则**，并重点剖析了 **“推荐规则”** 的三大组成部分：
1.  **用户行为规范**：要避免直接营销词汇，学会用谐音、话术和小号互动等技巧合规引流。
2.  **内容安全责任**：坚决不触碰医疗和投资建议的红线，注意电脑端登录账号的数量限制。
3.  **笔记发布规范**：杜绝诱导互动和抄袭，坚持发布高原创度内容以获得系统推荐。

![](img/9fda1c5ef6b97786dfb013e01a460531_35.png)

![](img/9fda1c5ef6b97786dfb013e01a460531_37.png)

理解这些规则是避免踩坑的第一步。下节课，我们将深入探讨 **“被推荐的核心逻辑”** ，即系统如何判断一篇笔记的好坏，并决定分配多少流量。这将帮助我们从“了解规则”进阶到“利用规则”提升数据。
