# 小红书开店教程：P4：小红书店铺分类 📚

![](img/2c46ed5bcab37b136035f4a5fb0514b2_1.png)

在本节课中，我们将要学习小红书电商平台上的店铺分类体系。了解不同类型的店铺及其要求，是开设店铺、规划运营策略的第一步。我们将详细解析个人店、企业店等各类店铺的特点、入驻要求以及关键的保证金规则。

## 店铺类型体系概述

![](img/2c46ed5bcab37b136035f4a5fb0514b2_3.png)

![](img/2c46ed5bcab37b136035f4a5fb0514b2_4.png)

小红书平台为不同经营主体提供了多样化的店铺类型。了解整个体系有助于我们根据自身情况选择合适的店铺类型，并明确在不同阶段应如何操作。

以下是小红书主要的店铺类型列表：

*   **个体店**
*   **个体工商户店**
*   **普通企业店**
*   **专卖店**
*   **旗舰店**
*   **卖场型旗舰店**

这些类型本质上对应了从个人到品牌方的不同经营主体。接下来，我们将逐一进行解析。

## 各类店铺详解

上一节我们概述了小红书的店铺体系，本节中我们来看看每种店铺的具体定义和要求。

### 个体店

**个体店**的申请主体是个人用户。这类似于我们在线下摆地摊，适合刚毕业、辞职或想尝试小规模经营的创业者。

其经营特点是限定经营零星、小额的商品，例如首饰、服装、包包、手机壳等。它可以销售多个品牌，像一个杂货铺。

![](img/2c46ed5bcab37b136035f4a5fb0514b2_6.png)

入驻要求非常简单，仅需个人身份证即可。

店铺命名格式由“专业号名称”加上系统分配的“店名”组成。

![](img/2c46ed5bcab37b136035f4a5fb0514b2_8.png)

关于保证金，个体店涉及**类目保证金**和**浮动保证金**，二者取高值。基础保证金通常为1000元。如果保证金要求超过1000元，则可能对应个体工商户店或更高级的店铺类型。具体的保证金等级划分将在课程最后详细说明。

### 个体工商户店

![](img/2c46ed5bcab37b136035f4a5fb0514b2_10.png)

**个体工商户店**与个人店的主要区别在于入驻要求。

![](img/2c46ed5bcab37b136035f4a5fb0514b2_12.png)

它需要申请人提供**营业执照**，但对品牌授权没有需求。除了多出营业执照这一要求，其他方面与个体店基本一致。

### 普通企业店及更高级店铺

从普通企业店开始，包括后面的专卖店、旗舰店等，它们与前述店铺的核心区别都体现在**入驻要求**上。

不同类型的店铺，其入驻资质要求不同。通常，要求越简单的店铺，平台赋予的权重可能越低；要求越复杂的店铺，权重则可能越高。

对于刚入门的新手，建议从要求最简单的个人店开始测试，验证产品在小红书平台的潜力，无需初期投入过多。

以下是这些店铺的入驻要求简述：

![](img/2c46ed5bcab37b136035f4a5fb0514b2_14.png)

*   **普通企业店**：申请主体必须为企业，需持有**企业营业执照**。
*   **专卖店**：需提供**营业执照**和**品牌认可**（自有品牌或非独占授权品牌均可）。这意味着只要拥有在国家注册认证的品牌，即可开设专卖店销售自己的产品。
*   **旗舰店**：需提供**营业执照**和**品牌授权书**。如果自身没有品牌，但获得了其他品牌的授权，也可以开设旗舰店。
*   **卖场型旗舰店**：具体要求可能随平台规则调整，其经营特点是需具备零售资质，并开设经营多个品牌的旗舰店。不过，由于小红书用户以女性为主，流量特性与大型卖场店不完全对标，因此这类店铺在平台上较少，这反而为中小卖家提供了更宽松的竞争环境。

## 保证金规则详解

![](img/2c46ed5bcab37b136035f4a5fb0514b2_16.png)

![](img/2c46ed5bcab37b136035f4a5fb0514b2_17.png)

在了解了各类店铺后，我们需要重点关注运营成本中的关键一项：保证金规则。它主要分为类目保证金和浮动保证金两部分。

![](img/2c46ed5bcab37b136035f4a5fb0514b2_19.png)

### 类目保证金

![](img/2c46ed5bcab37b136035f4a5fb0514b2_20.png)

类目保证金根据经营的商品类目固定收取。以下是常见的保证金类目划分示例（具体金额以平台最新规则为准）：

![](img/2c46ed5bcab37b136035f4a5fb0514b2_22.png)

*   **普通企业类目**：基础保证金，例如 1000元。
*   **特殊类目**：如食品、化妆品等，可能需要更高的保证金，例如 2000元 或 5000元。
*   **跨境类目**：经营进口商品可能需要单独的保证金，例如 10000元。

由于类目规则较为固定且可通过官方渠道查询，此处不再展开详细列表。建议在开店前根据所选类目查询平台最新的保证金标准。

### 浮动保证金

**浮动保证金**是运营中需要动态管理的部分，非常重要，大家一定要理解。

浮动保证金根据店铺的成交金额动态计算。计算公式和阶梯大致如下：

```
如果 月成交金额 < 5万元，则浮动保证金为 0元（但需先缴纳基础保证金）。
如果 5万元 ≤ 月成交金额 < 10万元，则需补足保证金至 3000元。
如果 10万元 ≤ 月成交金额 < 50万元，则需补足保证金至 5000元。
如果 月成交金额 ≥ 50万元，则需补足保证金至 20000元。
```

**核心机制**：平台会在提现时检查保证金账户总额是否满足当前成交额对应的要求。如果账户余额不足，将无法提现。

例如，若只缴纳了1000元基础保证金，当某月成交额达到6万元时，由于未满足3000元保证金的要求，该月所有货款将无法提现。因此，务必随时关注店铺成交额，确保保证金充足，避免资金冻结。
