# 小红书运营：P19：8. 小红书平台用户标签及流量入口 📊

![](img/a13a90f31b448991f126a5a0f4aef0bf_1.png)

在本节课中，我们将要学习小红书平台如何为用户打上标签，以及平台主要的流量入口有哪些。理解这些底层逻辑，能帮助我们更好地创作内容，让笔记被更多精准用户看到。

上一节我们介绍了系统如何判定内容标签，本节中我们来看看平台如何判定用户标签。这个过程能帮助我们分析和理解自己的标签，以及笔记是如何被系统推荐给用户的。我们可以从背后的逻辑反推，如何让笔记被我们的精准用户看到。

## 用户标签的判定逻辑

系统为用户打标签的主要依据是用户的**浏览行为**。用户刚注册时选择的兴趣范围，参考价值不大。它只是平台在初期做的内容方向引导，只能进行初步的用户筛选，还无法进入系统最深层的用户标签池。

系统判定用户兴趣分类，是根据用户持续的浏览行为来决定的。以下是系统分析用户行为的主要维度：

![](img/a13a90f31b448991f126a5a0f4aef0bf_3.png)

以下是判定用户标签的几个关键行为指标：

1.  **点击率**
    *   **公式**：`点击率 = (点击某类内容的次数) / (系统推送的该类内容总次数)`
    *   系统在发现页为用户推送多种内容（如美妆、家居、母婴）。用户点击哪一类内容更多，系统就能计算出用户的点击概率。例如，用户点击了10条内容，其中美妆类3次，家居类2次，母婴类0次，系统会判定该用户对母婴内容不感兴趣，可能不是宝妈，并对美妆内容更感兴趣。

2.  **停留时长**
    *   用户点开笔记后观看的持续时间。如果用户经常快速划走美妆类笔记，但某天对一个家居类笔记停留了很久，仔细阅读图片和评论，系统可能会判断用户产生了新的兴趣（如装修需求或想购买某款家居产品），从而调整推荐内容。

3.  **页面浏览量（PV）**
    *   用户浏览页面的深度。例如，用户不仅看了一篇笔记，还点进博主主页，并连续看了第二、第三篇笔记。提升用户在你主页的浏览深度，能为账号健康度带来非常积极的评分。

4.  **互动行为**
    *   包括点赞、收藏、评论、关注。这些数据直接提升笔记的互动率。互动数据高，笔记进入更大推荐流量池的几率就更高，同时也能带动账号内其他笔记的流量，并增加粉丝数。这是我们运营小红书一直要努力提升的目标。

**核心关系**：博主创作有价值的内容 → 平台将内容匹配给带有相应标签的用户 → 用户获得感兴趣的内容，平台获得留存。三者形成一个价值闭环。

## 平台流量分发入口

了解用户标签后，我们来看看笔记可能被展示的流量入口。小红书平台主要有四个流量分发入口：

以下是四个主要的流量入口及其特点：

*   **发现页**
    *   这是笔记流量的主要来源。包含三部分内容：
        1.  已关注的达人发布的新内容。
        2.  系统根据你的**用户标签**（点击率、停留时长等）推荐的、你可能感兴趣的内容。
        3.  系统为了保持内容多样性，偶尔推荐的、你可能未接触过的内容。

*   **关注页**
    *   这里集中展示所有你关注的博主发布的笔记。用户通常从这里直接查看核心关注者的动态。

*   **附近页**
    *   根据手机定位，推送同城或附近的内容。主要以美食、本地旅游、线下店铺等强地域特征的内容为主。如果你是线下实体店（如美容院、餐厅），需要重点运营这个入口，在做信息流广告时也可侧重附近人群。

![](img/a13a90f31b448991f126a5a0f4aef0bf_5.png)

*   **购物页**
    *   小红书向电商发展的重要入口，直接匹配用户的购物需求。目前用户习惯仍在培养中，许多购物行为仍始于笔记搜索和种草。
