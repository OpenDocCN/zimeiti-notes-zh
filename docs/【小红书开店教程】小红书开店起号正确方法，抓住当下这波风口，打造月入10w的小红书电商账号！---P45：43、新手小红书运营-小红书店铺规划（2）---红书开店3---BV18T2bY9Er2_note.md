在本节课中，我们将要学习小红书店铺运营的中期与后期规划。我们将了解中期需要关注的各项排名机制，以及后期发展的两个关键方向，帮助你构建完整的店铺运营思路。

![](img/c6a428d61a81f42a2ddcb88a035d10ee_7.png)

![](img/c6a428d61a81f42a2ddcb88a035d10ee_9.png)

上一节我们介绍了店铺规划的前期准备，本节中我们来看看店铺运营进入中期后，我们应该如何操作。

中期规划的重点内容非常多，其核心目标是提升店铺在平台内的各项排名，从而获取更多的基础曝光流量。中期规划主要涉及以下几个排名系统。

![](img/c6a428d61a81f42a2ddcb88a035d10ee_11.png)

以下是中期规划的核心排名系统：

![](img/c6a428d61a81f42a2ddcb88a035d10ee_13.png)

1.  **综合排名分类**：这是平台默认的商品排序方式，根据店铺的综合因素对商品进行排名。其考核维度多达16项，包括：
    *   商品人气
    *   商品销量
    *   店铺信誉
    *   商品价格
    *   新店与新品扶持
    *   商品橱窗优先级
    *   商品下架时间
    *   标题关键词优化
    *   商品转化率
    *   商品收藏率
    *   顾客回头率
    *   商品好评率
    *   累计交易数量
    *   商品描述与主图质量
    *   店铺整体交易情况
    *   个性化搜索匹配度

    在这16项中的排名越高，你能从平台分配的20%基础流量中获取的份额就越大。

![](img/c6a428d61a81f42a2ddcb88a035d10ee_15.png)

![](img/c6a428d61a81f42a2ddcb88a035d10ee_17.png)

2.  **价格排名**：根据商品价格进行排序。系统会考量价格的竞争力、真实性、稳定性以及与销量的平衡关系。同价格的商品，其排名会受到关键词匹配度和商品类目的影响。公式可以理解为：
    `价格排名 = f(价格竞争力， 价格真实性， 价格稳定性， 价格与销量平衡， 关键词匹配度， 类目相关性)`

![](img/c6a428d61a81f42a2ddcb88a035d10ee_19.png)

3.  **销量排名**：根据商品的销售数量进行排序。高销量通常意味着高人气。但销量高却排名靠后，可能是由于商品类目不精准导致的。系统还会参考销量的增长趋势、类目相关性和商品评价。

4.  **关键词排名**：用户搜索不同关键词时，商品的展示顺序不同。例如，“T恤”、“男士T恤”、“女士T恤”是不同的搜索词，对应着不同的排名列表。你需要优化商品标题和属性，使其在目标关键词下获得更好排名。

5.  **个性化搜索**：平台根据用户的浏览、购买习惯等属性，结合店铺和商品的标签，进行个性化推荐。个性化搜索做得越好，吸引到的用户就越精准。

![](img/c6a428d61a81f42a2ddcb88a035d10ee_21.png)

![](img/c6a428d61a81f42a2ddcb88a035d10ee_23.png)

6.  **动态与静态PK**：这部分涉及更复杂的运营策略，通常在小红书店铺运营达到较高层级（第四、第五层级）后才需要重点考虑，本入门教程暂不展开。

![](img/c6a428d61a81f42a2ddcb88a035d10ee_25.png)

由于中期规划的每个部分都包含大量细节，本节课旨在让你了解整体框架。如果你对其中任何一点特别感兴趣，可以截图保存，并搜索相关专题进行深入学习。

了解了中期规划后，我们再来看看后期的发展方向。后期规划主要围绕“发展”展开，通常有两个关键节点供你选择。

以下是后期规划的两个发展方向：

![](img/c6a428d61a81f42a2ddcb88a035d10ee_27.png)

*   **方向一：加大力度投入**。如果你认为店铺仍有巨大潜力，可以在此阶段进行深度运营，为冲击更高层级做准备。这需要你做好以下五方面工作：
    1.  数据分析：深入分析店铺各项数据指标。
    2.  店铺分析：诊断店铺整体健康状况。
    3.  数据掌控：建立数据监控与预警机制。
    4.  店铺优化：持续优化商品、页面和流程。
    5.  瓶颈突破：尝试付费推广等方式，突破增长天花板，从“中层用户”向“大用户”发展。

*   **方向二：开设新店**。如果你觉得当前店铺或产品已触及天花板，不想冒险加大投入，则可以总结经验，开设新店。具体步骤为：
    1.  吸取第一个店铺的运营经验和教训。
    2.  经营类似商品，并沿用已验证成功的运营模式。
    3.  在新店运营顺畅后，再根据资金和信心，决定是否对其“加大力度投入”。

本节课中我们一起学习了小红书店铺的中期与后期规划。中期规划的核心是理解并优化**综合排名**、**价格排名**、**销量排名**、**关键词排名**和**个性化搜索**，以提升店铺的基础流量获取能力。后期规划则为你指明了两个发展方向：**加大投入突破瓶颈**或**总结经验开设新店**。这些内容构成了店铺运营的完整思路框架。
