# 新媒体运营：P29：11：小红书平台用户标签及流量入口 📊

![](img/4eb69cd2fab0e0852fe8613410ce99a3_1.png)

在本节课中，我们将学习小红书平台如何为用户打上标签，以及平台有哪些主要的流量分发入口。理解这些机制，有助于我们分析笔记如何被推荐给用户，并反推出让内容被精准用户看到的方法。

上一节我们介绍了系统如何判定内容标签，本节中我们来看看平台如何判定用户标签。这个过程能帮助我们了解笔记是如何被系统推荐给用户的，并能从背后的逻辑反推，如何让笔记被我们的精准用户看到。

## 用户标签判定机制 🔍

系统为用户打标签的主要依据是用户的**浏览行为**。用户刚注册时选择的兴趣范围，参考价值不大，它仅作为平台对用户的内容方向引导，只能进行初步筛选，无法进入系统最深层的用户标签池。

系统判定用户兴趣分类的核心依据是用户的浏览行为。以下是系统分析用户行为的主要维度：

以下是判定用户标签的几个关键行为指标：

1.  **点击率**
    *   **定义**：系统在发现页向用户推送多个内容（如美妆、家居、母婴等），用户点击了哪些内容、点击频率如何。
    *   **作用**：系统通过计算用户对不同类别内容的点击概率，来打上强关联或弱关联的标签。例如，用户点击10条内容，其中美妆类3次，家居类2次，母婴类0次，系统会判定该用户对美妆更感兴趣，对母婴不感兴趣，并持续推送更多美妆内容进行试探。

2.  **停留时长**
    *   **定义**：用户点开一篇笔记后观看的持续时间。
    *   **作用**：用于判断用户对内容的真实喜好程度。例如，用户常看美妆内容但划走得快，说明推送的笔记不精准。若某天系统推送了一条家居内容，用户不仅点了，还停留很久、仔细阅读，系统则会判断用户可能有装修或购买家居用品的需求，从而调整推送策略。

3.  **PV（页面浏览量）**
    *   **定义**：用户浏览页面的深度，例如不仅看了单篇笔记，还进入了博主主页并查看了多篇笔记。
    *   **作用**：PV值高，说明用户对账号内容兴趣浓厚。这能提升账号的健康度评分。对于新账号而言，提升主页内容的浏览深度，对账号成长有非常大的帮助。

4.  **互动行为**
    *   **定义**：包括点赞、收藏、评论、关注等用户主动操作。
    *   **作用**：每一项数据都直接影响笔记的传播。数据好能提升笔记进入更大推荐流量池的几率，带动账号其他笔记的曝光，并有助于涨粉和后续变现。

简单来说，系统通过持续观察用户的这些行为，像绘制一幅兴趣云图一样，为用户打上大小、强弱不同的标签（如**美妆**、**母婴**、**装修**），其中最大的标签即代表用户的核心兴趣，系统会据此主要推送相关领域内容。

## 平台流量分发入口 🚪

了解用户标签后，我们来看看小红书平台主要的流量分发入口。作为用户你可能经常使用，但未必分析过其区别。

小红书目前主要有四个流量分发入口：

以下是四个核心流量入口及其特点：

1.  **发现页**
    *   **内容构成**：
        *   已关注达人发布的内容。
        *   根据用户标签（点击率、兴趣等）匹配的、系统认为你可能喜欢的内容。
        *   部分系统试探性推荐的、多样化的新内容，以保持平台内容生态的丰富性。
    *   **重要性**：对于博主而言，后期能获得的主要流量就来源于发现页。

2.  **关注页**
    *   **内容构成**：纯粹展示用户所关注的所有博主发布的最新笔记。
    *   **特点**：流量相对稳定但有限，是粉丝与博主保持联系的主要阵地。

3.  **附近页**
    *   **内容构成**：根据手机定位，推送附近及同城的相关内容。
    *   **适用内容**：具有强地域特征的需求，如本地美食、旅游、线下店（美容院、餐厅等）推广。
    *   **提示**：若运营目标是推广线下实体店，在规划内容或投放信息流广告时，需重点关注附近人群的推送。

4.  **购物页**
    *   **定位**：小红书向电商平台发展的重要入口，专注于满足用户的购物需求。
    *   **现状**：部分用户尚未形成在此直接购物的强习惯，更多人仍倾向于先搜索相关笔记（种草）再做决策。该入口与笔记种草生态紧密相关。

本节课中我们一起学习了小红书平台判定用户标签的四大行为依据（点击率、停留时长、PV、互动行为），以及四个核心的流量分发入口（发现页、关注页、附近页、购物页）。理解这些底层逻辑，能帮助我们更有效地规划内容，让笔记更精准地触达目标用户，并为账号的长期增长和变现打下坚实基础。
