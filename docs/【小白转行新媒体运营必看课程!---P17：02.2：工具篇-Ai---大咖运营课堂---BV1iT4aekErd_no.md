# 新媒体运营课程：P17：02.2：工具篇-AI生成PPT 🚀

在本节课中，我们将要学习一个非常实用的AI工具，它能帮助我们根据主题或现有文档，自动生成结构清晰、内容详实的PPT演示文稿。

## 工具简介与核心功能

上一节我们介绍了多种运营工具，本节中我们来看看如何利用AI提升内容制作效率。这个工具的核心功能是根据输入自动生成PPT。它主要支持两种模式：

1.  **主题生成模式**：你提供一个主题，工具会像ChatGPT一样，生成完整的PPT内容。
2.  **文档生成模式**：你上传一篇文档（如TXT、PDF），AI助手会先自动总结出大纲，再基于大纲生成详细的PPT内容。

在我使用过的众多AI生成PPT工具中，这个工具的表现非常出色。它的文档生成模式尤其好用，能深入理解内容并归纳出逻辑大纲。

## 编辑与优化生成结果

生成完毕后，我们可以进入编辑界面查看和调整PPT。这个工具最突出的优点是：它能基于我们提供的大纲节点，自动生成对应的详细阐述内容。

以下是该工具的几个关键编辑功能：

*   **更换模板**：虽然当前提供的模板数量有限，但切换非常迅速。我们可以选择一个合适的模板作为基础。
*   **内容调整**：在右侧编辑栏，我们可以自由调整文字、替换形状等元素。
*   **本地深度编辑**：最便捷的工作流是，先在工具内选好模板并生成内容大纲，然后下载到本地（如PowerPoint或Keynote）进行细致的排版和内容优化。
