# 剪映手机版教学：P8：剪映核心剪辑功能——常用素材片段 📚

在本节课中，我们将要学习剪映APP中一个非常实用的功能——**素材库**。这个功能内置了大量现成的视频片段，能帮助初学者快速找到并应用素材，从而节省时间，提升视频制作效率。

上一节我们介绍了其他核心剪辑功能，本节中我们来看看如何利用剪映自带的素材库。

## 素材库的位置与作用

许多新手在制作视频时，常常为寻找合适的素材而苦恼。剪映APP内置的**素材库**功能，完美解决了这个问题。它提供了大量如黑屏、白屏、雪花故障等常用视频片段，特别适合用于制作沙雕搞笑类段子或日常Vlog。

![](img/147d12278689970da0f8b6a47bc92081_1.png)

**素材库的位置**如下：
1.  打开剪映APP，点击顶部“**开始创作**”按钮。
2.  在导入素材的界面，左侧是“照片视频”（即你手机相册中的内容）。
3.  在右侧，你可以找到“**素材库**”选项。

![](img/147d12278689970da0f8b6a47bc92081_1.png)

![](img/147d12278689970da0f8b6a47bc92081_3.png)

在这里添加的视频素材，就是剪映自带的资源，可以直接调用。

剪映的素材库内容非常丰富，并且进行了分类，方便用户查找。

*   **搞笑片段**：包含网络流行的搞笑视频接头，能快速为视频增加趣味性。

## 如何使用素材库

使用素材库的方法非常简单，其核心操作可以概括为以下流程：

```
打开剪映 -> 开始创作 -> 点击“素材库” -> 浏览或搜索所需片段 -> 点击“+”号添加至时间轴
```

你可以多浏览、预览不同的分类，尝试将喜欢的素材添加到时间轴进行调试。这大大减少了从外部寻找素材的时间成本。
