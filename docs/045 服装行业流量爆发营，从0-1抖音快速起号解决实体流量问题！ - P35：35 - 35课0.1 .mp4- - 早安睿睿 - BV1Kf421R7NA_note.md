在本节课中，我们将要学习如何将上节课设计的脚本，通过“分镜”这一专业手法，转化为具体、可执行的拍摄计划。分镜是确保视频内容完整、情绪到位、画面流畅的关键一步。

上一节我们介绍了如何根据故事发展阶段来设计脚本。本节中我们来看看如何通过分镜，将脚本中的每个阶段和情节，具体化为一个个独立的镜头。

如果未经专业训练，可能不了解“分镜”的含义。分镜指的是在拍摄前，将整个视频内容分解为一个个独立的镜头，并有计划、有规划地进行录制。

许多初学者在录制前，并不清楚自己要拍什么，也不知道哪些部分是必须拍摄的。这会导致后期剪辑时，发现遗漏了关键镜头，或者缺少必要的远景、特写、运镜等画面。

例如，在上一个分析的形象改造视频中，从头到尾都没有出现形象设计师本人的镜头。但对于打造个人IP而言，我们需要被观众看见，人物应该出现在镜头前。相比而言，那个视频就遗漏了关于“改造者”的镜头，尽管它有很多“被改造者”的镜头。

因此，在拍摄前，必须通过分镜策划，将重要的画面规划为单个镜头进行录制，这样才能避免遗漏。

接下来，我们通过一段优秀博主的视频素材，来具体分析分镜设计的要点。这是一位名叫“门叔”的摄影博主，虽然内容不是形象改造，但改造类视频的鼻祖可追溯到摄影和美发行业，因为他们非常擅长做前后对比。门叔就是在摄影行业中，将前后对比做得非常出色的博主。

![](img/f6b8597ff876dc81b228e697bf3153ee_3.png)

以下是该视频的分镜设计分析，这些设计值得我们学习：

**第一个分镜：女孩奔跑而来**
这个镜头设计了一个女孩从远处奔跑靠近的画面。拍摄时就知道需要专门拍摄这样一个“由远及近”的运镜镜头。

**第二个分镜：门叔的抬头反应**
这个镜头是从上往下拍摄门叔，他原本在看书，听到女孩呼喊后抬头看向她。这个分镜用于展示对话中另一方的反应。

**第三个分镜：经典的“回头”细节**
当女孩说完“分手啦”后，设计了一个她转过头、头发飘扬的镜头。这个细节分镜极具表现力，象征着她要决绝地离开这段回忆。如果没有这个回头镜头，视频的情绪表达就不够完整。

这里尤其要提醒大家，在形象改造视频中，可以多设计一些“细节分镜”。细节分镜与故事的情绪有极大关联，我们将在下一节详细讲解其重要性。

**第四个分镜：为“变装”做衔接**
女孩回头后，头发飞扬的素颜镜头，与后面变装完成、化妆后同样头发飞扬、眼含泪水的悲伤镜头完美衔接。通过相同的姿势和动作，利用特效切换，实现了前后情节的流畅对比与衔接。

这种设计使得上一个情节与下一个情节的过渡更加自然，也让观众能清晰看到人物在特定时刻的反应。

综上所述，分镜设计就是用具体的镜头语言，来承接脚本的每个阶段。每个阶段该用大景别还是小景别、远景还是近景、固定镜头还是运动镜头，都需要在分镜中提前规划好。

这就是我们进行形象改造视频创作的第二个核心方法：**确定好你的分镜镜头**。

本节课中，我们一起学习了分镜的概念、重要性，并通过实例拆解了优秀分镜的设计思路。掌握分镜设计，能确保你的脚本被完整、生动地呈现出来，避免拍摄遗漏，并增强视频的情绪感染力与流畅度。
