# 抖音运营入门：P3：1、抖音短视频的前景

![](img/8bbf427dadec98e11053f1dae0d776c2_1.png)

在本节课中，我们将要学习抖音短视频平台的发展前景，了解其市场规模、用户构成以及为何它适合新手创作者起步。

![](img/8bbf427dadec98e11053f1dae0d776c2_3.png)

要了解抖音短视频的前景，我们可以先通过一张图片来获得直观认识。

![](img/8bbf427dadec98e11053f1dae0d776c2_1.png)

这张图片来自抖音创作者大会的数据报告。报告显示，有超过2200万人在抖音的合计收入超过了417亿元。并且，抖音希望将这个数字翻一番，让创作者的总收入达到800亿元。

![](img/8bbf427dadec98e11053f1dae0d776c2_3.png)

这体现了平台巨大的流量扶持力度。同时，这里需要明确第一个知识点：抖音上主要存在两类人。第一类是“玩家”，他们单纯地使用抖音娱乐，没有从中赚钱的打算。第二类是“创作者”，也就是我们课程的目标学员，你们希望通过制作抖音内容来获取收益。区分这两类人对后续理解很重要。

那么，为什么在众多短视频平台中，我们要选择抖音呢？这类似于开实体店选址。做实体店生意时，你肯定会选择一个流量大、位置好的地方。做短视频同理，必须选择一个流量庞大的平台。如果平台自身流量不足，就无法分配流量给我们，我们也难以赚到钱。

抖音的流量数据非常可观：其日活跃用户超过8亿，月活跃用户更是达到了40亿人次。这是一个极其庞大的流量池。

面对如此大的流量，竞争压力是否会很大？答案是不会。因为平台上的“创作者”数量约为5000万。这里的“创作者”定义广泛，并非特指网红达人。例如，你邻居家的老太太用手机拍下自己做的饭并上传到抖音，她也算是一位创作者。因此，你90%的竞争压力，实际上来自于这些普通的“邻里乡亲”。

接着，做抖音短视频需要很高的硬件门槛吗？答案是不需要。你只需要一部智能手机和一个手机号，就可以注册抖音账号。抖音倡导全民通过短视频的形式分享和记录生活，入门非常简单。

那么，当前有什么具体的风口值得我们关注呢？首先是“抖音计划”。该计划表明，抖音将帮助1000名优质的艺术创作者实现粉丝增长超过100万。有的同学可能会担心自己不会创作。这完全不用担心，如果你什么都会，就不需要来听课了。正是因为我们是新手，才需要系统地学习。老师曾指导过许多学员，包括年长者。他们虚心请教的态度值得学习：**“虚心请教厉害的人不丢人”**。在你不懂的领域，比你懂的人就是你的老师。保持谦虚好学的态度，能让你少走很多弯路。

当前市场空间巨大，内容创作门槛低，行业很容易产出内容。我们正处于大数据时代。在这个时代，平台会根据你的喜好推荐内容。例如，你喜欢看帅哥，刷到的就多是有腹肌的帅哥；你喜欢看美女，刷到的就多是长腿美女。这导致我们每个人看到的内容圈层相对固定。

因此，无论你是什么身份，都有创作的可能：
*   如果你是宝妈，可以拍摄宝宝或自己做的饭菜。
*   如果你是实体店主，可以拍摄产品进行引流。
*   如果你是普通上班族，可以分享日常生活。

月活跃用户高达40亿，只要你创作的内容不触犯法律和道德底线，不是低俗内容，总会有一部分人喜欢你的风格。你可能会问，既然总有人喜欢，为什么我的粉丝只有几十个？这是因为“喜欢你的人”还没有刷到过你的作品。

所以，对于新手小白而言，虽然涨粉心情迫切，但请不要把“涨粉”作为第一个目标。我们的首要目标是：**先把流量（播放量）做起来**。当你的视频播放量提升后，粉丝自然就会增长。

本节课中，我们一起学习了抖音平台的巨大前景、用户分类、低门槛特性以及新手起步的核心思路。关键在于选择这个流量巨大的平台，明确自己“创作者”的身份，并采取先做流量、后自然增粉的务实策略。
