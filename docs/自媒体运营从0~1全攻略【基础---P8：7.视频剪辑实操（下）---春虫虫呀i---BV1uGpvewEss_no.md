## 添加视频转场

![](img/00f73334e19b9dffad8a7ae8acbe8b66_3.png)

每个视频通常由多个片段组成，每个片段时长大约在10秒左右。这些片段之间需要有自然的衔接，因此必须添加转场效果。转场的作用是明确告知用户，当前片段内容已切换，避免用户在观看时因内容突然变化而产生困惑，甚至退出视频。视频的完播率会直接影响平台的推荐权重和收益。

**完播率影响公式**：`收益权重 ∝ 完播率`

![](img/00f73334e19b9dffad8a7ae8acbe8b66_5.png)

以下是添加转场的具体步骤：

![](img/00f73334e19b9dffad8a7ae8acbe8b66_6.png)

![](img/00f73334e19b9dffad8a7ae8acbe8b66_7.png)

1.  在视频编辑轨道右侧，找到并点击“加号”按钮。
2.  在弹出的菜单中选择“素材库”。
3.  在“热门”分类中，选择时长在3到5秒之间的转场素材。避免使用过长的转场（如16秒），以免影响观看耐心。
4.  选择好转场素材（例如一个3秒的片段）后，点击添加。

![](img/00f73334e19b9dffad8a7ae8acbe8b66_8.png)

![](img/00f73334e19b9dffad8a7ae8acbe8b66_9.png)

转场素材添加后，必须确保其画面比例与主视频保持一致，即**16:9**的横屏比例。

以下是调整转场素材比例的步骤：

1.  选中已添加的转场片段。
2.  在编辑菜单中找到“画布模糊”或“背景”选项。
3.  选择“画布模糊”下的第一个清晰样式，将背景填充为16:9的横屏。
4.  点击右下角的“对勾”确认设置。

![](img/00f73334e19b9dffad8a7ae8acbe8b66_11.png)

![](img/00f73334e19b9dffad8a7ae8acbe8b66_12.png)

## 添加视频字幕

字幕能显著提升视频的信息传达效率和观看体验，尤其适用于在无声环境或需要理解对话的场景下观看。

以下是添加字幕的具体步骤：

![](img/00f73334e19b9dffad8a7ae8acbe8b66_14.png)

![](img/00f73334e19b9dffad8a7ae8acbe8b66_15.png)

1.  点击手机剪辑界面左下角的“文字”选项。
2.  选择“新建文本”，然后点击右侧的“识别字幕”功能。
3.  点击“开始匹配”，软件会自动识别视频或音频中的人物对话，并生成对应的字幕文本。
4.  对生成的字幕进行微调：拖动字幕文本框右下角以放大字体，然后将整个字幕条拖动到视频画面底部最合适的位置。调整好第一句字幕后，后续字幕通常会同步该样式。

## 核心要点总结

本节课中我们一起学习了视频剪辑后半部分的核心操作：

![](img/00f73334e19b9dffad8a7ae8acbe8b66_17.png)

![](img/00f73334e19b9dffad8a7ae8acbe8b66_18.png)

1.  **添加转场**：在视频片段间插入3-5秒的转场素材，以平滑过渡，提升完播率。务必保持转场素材比例为**16:9**。
2.  **添加字幕**：使用“识别字幕”功能自动生成字幕，并调整其大小和位置至视频底部，以优化观看体验。

![](img/00f73334e19b9dffad8a7ae8acbe8b66_19.png)

![](img/00f73334e19b9dffad8a7ae8acbe8b66_20.png)

记住，剪辑视频并不难，关键在于严格遵循步骤：保持正确的比例、添加必要的转场和字幕。同时，在选择素材时，应避免使用过长的视频片段，以确保内容紧凑，并顺利通过平台审核。按照这些步骤操作，你就能制作出流畅、专业的视频内容。
