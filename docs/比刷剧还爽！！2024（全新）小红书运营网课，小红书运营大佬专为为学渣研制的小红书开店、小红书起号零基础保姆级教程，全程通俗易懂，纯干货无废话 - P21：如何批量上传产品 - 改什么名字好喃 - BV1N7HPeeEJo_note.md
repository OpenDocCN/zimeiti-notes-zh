# 小红书运营：P21：如何批量上传产品

## 概述
在本节课中，我们将学习如何在小红书商家后台批量上传商品。这对于希望扩大店铺商品数量的运营者来说，是一个提高效率的重要方法。我们将从登录后台开始，逐步讲解使用“商品搬家”工具从其他平台（如拼多多）批量抓取并上传商品的完整流程。

## 登录商家管理后台
上一节我们介绍了基础的店铺运营概念，本节中我们来看看如何进入操作后台。批量上传商品需要在“小红书商家管理后台”进行，你可以通过电脑或手机登录。

以下是两种登录方式的具体步骤：

*   **电脑登录**：在电脑浏览器中搜索“小红书商家管理后台”，找到并进入官方页面进行登录。
*   **手机登录**：在手机小红书APP内，点击加号搜索“小红书商家管理后台”，进入相关服务页面，通常需要通过扫描二维码完成登录。

两种方式进入的后台功能完全一致，只是界面大小不同。为便于演示，后续操作将以电脑端为例。

## 使用“商品搬家”工具
成功登录后台后，我们就可以开始批量上传商品了。核心工具是后台内置的“商品搬家”功能，它能帮助我们将其他平台的商品信息快速迁移到小红书。

操作路径如下：
1.  在后台左侧菜单栏找到并点击 **`商品`**。
2.  在商品管理页面，进入 **`商品列表`**。
3.  在商品列表中，找到并使用 **`商品搬家`** 工具。

点击后，系统会跳转至“小红书服务市场”。在该市场中，有多个服务商提供搬家工具，我们可以选择其中一个，例如“无忧商品”。

## 抓取与复制商品
进入选定的搬家工具后，下一步就是从源平台抓取商品。这个过程主要是复制商品链接。

具体操作步骤如下：

1.  在工具页面点击 **`复制商品`** 或类似按钮。
2.  选择商品来源平台，例如 **`拼多多`**。
3.  前往拼多多，选择你想要上传的商品（建议初期选择3-10个，以防部分商品匹配失败）。
4.  复制选中商品的商品链接。
    *   **电脑操作**：直接复制链接，并粘贴到搬家工具指定的输入框中。
    *   **手机操作**：复制链接后，在浏览器中打开商家后台页面进行粘贴。
5.  在搬家工具中，确认设置（如“非偏远地区包邮”），然后执行抓取。系统会显示抓取成功或失败的商品数量。

## 编辑并提交商品信息
商品信息抓取成功后，并非直接上架，我们需要在后台对商品详情进行核对与编辑，确保符合小红书规范。

以下是需要检查和编辑的关键信息项：

*   **基础信息**：检查系统自动填充的商品标题、材质等是否正确，不完整的部分可以补充。
*   **库存与售价**：根据你的情况，修改库存数量（如将 `120` 改为 `999`）和销售价格（如将 `167` 改为 `200`）。
*   **主图与视频**：系统抓取的主图可能带有其他平台logo，建议替换或删除。你可以使用工具提供的其他图片，或自行上传更优质的图片和视频。
*   **服务信息**：这是关键步骤，务必设置正确的物流模板、发货时间（建议选择 **`24小时`** 内发货）、售后政策（如 **`七天无理由退货`** ）等。

所有信息确认无误后，点击 **`提交商品`** 即可。商品通常会在3小时内完成审核上架。

## 总结
本节课我们一起学习了小红书批量上传商品的完整流程。我们首先掌握了电脑和手机登录商家后台的方法，然后重点讲解了利用“商品搬家”工具从拼多多等平台抓取商品链接、在后台编辑商品信息并最终提交上架的操作步骤。请注意，批量上传时可能会有商品审核失败，需要更换商品链接重试。如果觉得此方法操作复杂，你仍然可以使用之前课程中学到的手机端单件商品上架方式。
