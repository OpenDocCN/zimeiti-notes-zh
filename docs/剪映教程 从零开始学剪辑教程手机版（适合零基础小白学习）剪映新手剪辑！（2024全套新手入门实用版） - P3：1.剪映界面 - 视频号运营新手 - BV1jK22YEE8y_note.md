# 剪映教程：P3：1. 剪映界面介绍 📱

在本节课中，我们将要学习剪映手机版的核心界面布局。了解每个区域的功能是开始剪辑的第一步，这能帮助你快速找到所需工具，高效完成视频创作。

## 启动与主界面

![](img/10598e488300c3ba93ef38f6f0884e48_2.png)

首先，在手机应用商店下载并打开剪映应用。启动后，你将看到主界面。



![](img/10598e488300c3ba93ef38f6f0884e48_3.png)

![](img/10598e488300c3ba93ef38f6f0884e48_1.png)

![](img/10598e488300c3ba93ef38f6f0884e48_2.png)

![](img/10598e488300c3ba93ef38f6f0884e48_3.png)

要开始创作，你需要先添加素材。点击“开始创作”，可以从相册中选择视频或图片导入。在我的示例文件夹中只有视频素材。点击“添加”后，素材会进入编辑界面。



![](img/10598e488300c3ba93ef38f6f0884e48_5.png)

## 预览区与时间指示器

![](img/10598e488300c3ba93ef38f6f0884e48_6.png)

添加素材后，整个编辑界面分为几个主要部分。屏幕上半部分的大区域是**预览框**，用于实时预览视频效果。

预览框下方显示时间信息。左侧数字（如 `00:00`）是**时间轴**当前所在的时间点，它由一条竖杠（播放指示器）在时间线上移动来指示。右侧数字（如 `00:17`）代表当前选中素材的完整时长。



![](img/10598e488300c3ba93ef38f6f0884e48_5.png)

![](img/10598e488300c3ba93ef38f6f0884e48_8.png)

![](img/10598e488300c3ba93ef38f6f0884e48_6.png)

![](img/10598e488300c3ba93ef38f6f0884e48_10.png)

预览框下方有一排控制按钮。从左到右依次是：
*   **播放/暂停**按钮。
*   **关键帧**按钮（用于制作动画，后续课程会详细讲解）。
*   **撤销**按钮（可撤回上一步操作）。
*   **恢复**按钮（恢复已撤销的操作）。
*   **全屏预览**按钮（可全屏查看视频，再次点击退出）。

## 时间线与轨道

![](img/10598e488300c3ba93ef38f6f0884e48_13.png)

预览区下方是**时间线**区域，这是剪辑的核心操作区。



![](img/10598e488300c3ba93ef38f6f0884e48_14.png)

![](img/10598e488300c3ba93ef38f6f0884e48_12.png)

![](img/10598e488300c3ba93ef38f6f0884e48_13.png)

![](img/10598e488300c3ba93ef38f6f0884e48_14.png)

时间线由多条平行的“轨道”组成。最上方通常是**视频/图片素材轨道**。下方可以添加**音频轨道**、**文字轨道**等。每条轨道都独立承载一种类型的元素，你可以通过拖动来调整它们在时间线上的位置和时长。

![](img/10598e488300c3ba93ef38f6f0884e48_16.png)

## 一级与二级菜单栏

时间线底部是功能菜单栏。当没有选中任何素材时，这里显示的是**一级菜单**，包含最基础的功能入口。



![](img/10598e488300c3ba93ef38f6f0884e48_16.png)

![](img/10598e488300c3ba93ef38f6f0884e48_18.png)

当你点击时间线上的视频素材，菜单栏会切换为针对视频编辑的**二级菜单**，提供分割、变速、滤镜等详细工具。同理，点击“音频”、“文字”或“贴纸”按钮，也会进入相应的二级菜单界面。

## 导出与设置

视频编辑完成后，点击右上角的红色“导出”按钮即可生成视频。

在导出前，可以点击导出按钮上方的分辨率设置（如 `1080P`）进行调整。以下是关键参数：
*   **分辨率**：决定视频清晰度。常见选项有 `480P`、`720P`、`1080P` 等。数值越大，清晰度越高，文件也越大。
    *   公式表示清晰度排序：`480P < 720P < 1080P < 2K < 4K`
*   **帧率 (fps)**：决定视频流畅度。常见选项有 `24`、`25`、`30`、`60` 帧/秒。默认 `30fps` 适用于大多数自媒体平台（部分平台如百度系推荐 `25fps`）。
