## 政策背景与概述

从9月份开始，拼多多平台持续推出对商家的利好政策。早期的政策，例如售后申诉不限次数和部分场景退还技术服务费，被部分商家认为诚意不足。

然而，自9月24日起实施的新政策，展现了平台更大的诚意。核心变化包括：所有退款订单的基础技术服务费将自动退回，以及大幅下调商家提现的金额门槛。

## 核心政策详解

上一节我们介绍了政策背景，本节中我们来看看两项具体政策的细节。

### 1. 技术服务费自动退回

此项政策规定，自9月24日起，所有产生退款的订单，其基础技术服务费将自动退还给商家。这意味着，无论退款原因属于何种场景，商家都无需再手动申诉，平台系统会自动完成费用返还。

**核心公式**：`最终支付技术服务费 = 原始技术服务费 - 退款订单技术服务费`

### 2. 提现金额门槛下调

拼多多同时下调了商家提现的金额门槛。原有的提现门槛为100万元，现已下调至1元。此举极大地提升了商家资金使用的灵活性和周转效率。

**核心变化**：`提现门槛：1,000,000元 → 1元`
