## 概述

![](img/1cd570bf48b0302310d6d47e6fe1f669_4.png)

A+内容，也称为图文版品牌描述，是亚马逊为品牌卖家提供的高级功能。它允许卖家在商品详情页使用丰富的图文和视频来展示品牌故事与产品细节，从而提升品牌形象、增强消费者信任并提高转化率。

![](img/1cd570bf48b0302310d6d47e6fe1f669_6.png)

![](img/1cd570bf48b0302310d6d47e6fe1f669_7.png)

上一节我们介绍了Listing的其他基础板块，本节中我们来看看如何利用A+板块为产品页面增色。

## A+板块的站点要求差异

首先需要特别注意，不同亚马逊站点对使用A+功能的要求不同。

*   **日本站**：无需完成品牌备案，即可创建和使用A+内容。这是亚马逊对日本站卖家的一项扶持政策。
*   **美国站与欧洲站**：卖家必须完成**品牌备案**后，才具备创建A+内容的资格。

## A+内容的两种形式

![](img/1cd570bf48b0302310d6d47e6fe1f669_12.png)

![](img/1cd570bf48b0302310d6d47e6fe1f669_13.png)

在卖家后台点击“创建A+页面”后，你会看到两种主要形式：品牌故事和增强型商品描述。

![](img/1cd570bf48b0302310d6d47e6fe1f669_15.png)

### 1. 品牌故事

品牌故事主要用于展示品牌形象。其典型布局如下：

1.  **顶部背景图**：通常使用品牌模特或核心产品的大图作为背景。
2.  **品牌故事叙述**：阐述品牌的创立理念、使命或价值观。
3.  **产品与场景展示**：通过多张图片展示产品细节、使用效果和适用场景。
4.  **品牌价值重申**：最后再次强调品牌的优势与承诺。

![](img/1cd570bf48b0302310d6d47e6fe1f669_17.png)

这种形式通过营造整体观感，能有效提升品牌认知度和客户信任。

### 2. 增强型商品描述

对于大多数不以品牌建设为首要目的卖家，增强型商品描述是更常用、更基础的功能。它提供多种模块化布局来展示产品。

![](img/1cd570bf48b0302310d6d47e6fe1f669_19.png)

以下是可用的部分模块类型：

![](img/1cd570bf48b0302310d6d47e6fe1f669_21.png)

![](img/1cd570bf48b0302310d6d47e6fe1f669_22.png)

*   **Logo与标题模块**：展示品牌Logo和简短介绍。
*   **图片与文字模块**：左侧图片配合右侧文字说明，或反之。
*   **多图陈列模块**：例如并排展示三张或四张产品图片。
*   **单张大图模块**：用于突出某个核心卖点或场景。

![](img/1cd570bf48b0302310d6d47e6fe1f669_24.png)

![](img/1cd570bf48b0302310d6d47e6fe1f669_25.png)

## 创建与提交A+页面的全流程

接下来，我们一步步演示如何创建一个增强型商品描述并提交。

![](img/1cd570bf48b0302310d6d47e6fe1f669_27.png)

1.  **选择模块并编辑**：从“增强型商品描述”中选择一个模板，例如“三张图片与文字”模块。
2.  **上传图片并添加关键词**：为每个图片上传位置选择对应的产品图片。在图片的`alt text`（替代文本）或“关键词”字段中，可以填入相关的搜索词。虽然其是否被搜索引擎抓取未被官方确认，但建议在所有可埋词的地方都进行填写。字符限制通常为100个字符。
    *   **代码示例（概念性）**：
        ```html
        <img src="product-image.jpg" alt="无线蓝牙耳机 降噪 运动 2024新款">
        ```
3.  **填写文字内容**：在模块的文本区域，详细描述图片所展示的产品特性或优势。
4.  **关联商品**：为这个A+内容模板命名（例如“A+_蓝色款_15英寸”），然后选择要应用此A+页面的具体商品（ASIN）。
5.  **预览与检查**：利用“预览”功能查看A+页面在前端的实际显示效果。
6.  **提交审核**：预览无误后，点击“下一步”，在商品目录中搜索并选中目标ASIN，点击“应用”，最后“提交”以供亚马逊审核。审核通常需要几个工作日。

## 关于A+内容布局的建议

一个优秀的A+页面布局应逻辑清晰，引导消费者深入了解产品。通常的流程是：先展示核心卖点和应用场景，再分解产品细节和功能，最后通过对比或承诺增强购买信心。我们将在后续关于Listing优化的章节中，具体分析优秀的布局案例。
