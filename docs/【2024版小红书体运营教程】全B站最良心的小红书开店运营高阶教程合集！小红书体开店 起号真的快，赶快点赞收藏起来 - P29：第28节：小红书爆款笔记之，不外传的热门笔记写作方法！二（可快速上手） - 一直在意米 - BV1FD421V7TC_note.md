上一节我们介绍了笔记写作的基础框架，本节中我们来看看如何具体确立和优化笔记内容。

## 内容确立的五个步骤

以下是确立小红书笔记内容的五个核心步骤。

1.  **确定领域**
    首先，你需要明确账号运营的领域。例如，是美妆、护肤、母婴还是其他类别。在之前的课程中已多次强调，不确定领域会导致账号垂直度降低。账号垂直度降低会引发一系列问题，例如官方减少推荐、粉丝流失，并形成恶性循环：粉丝越不喜欢你的内容，官方就越不推荐。

2.  **梳理关键词**
    确定领域后，需要梳理与该领域相关的关键词。之前的课程中，我们介绍了使用百度工具、爱站网、5118、百度指数等工具进行关键词查询的方法。本节不再赘述。根据你的行业，可以梳理出关键词排行列表。用好这些关键词能极大提升内容效果。

3.  **搜集同行案例**
    接下来，可以搜集同行的优秀案例作为参考。有人可能会问，在小红书内搜索无法看到同行的具体数据，该怎么办？解决方法很简单，我们可以借助第三方数据分析工具。

![](img/ed2b3659237cec3ce37559e0300f5e84_3.png)

4.  **筛选与整理内容**
    搜集好同行案例后，需要对这些案例内容进行筛选。分析其优点和可借鉴之处，然后整理并转化为自己的原创内容。

![](img/ed2b3659237cec3ce37559e0300f5e84_5.png)

5.  **发布内容**
    完成内容整理与创作后，即可进行发布。

## 工具推荐：牵瓜数据

为了高效地搜集和分析同行数据，我们推荐使用“牵瓜数据”工具。该工具已稳定运行较长时间，非常实用。

牵瓜数据提供了多项小红书数据分析功能，其优势包括：
*   分析匹配优质达人。
*   追踪热门数据。
*   查看品牌方投放情况。
*   提供各类榜单。

那么，具体如何使用牵瓜数据来筛选内容呢？我们来详细说明。

牵瓜数据网站左侧栏目提供了多种功能，包括热词排行榜、热词搜索榜、笔记收录查询等。本节课主要讲解“笔记收录查询”功能。发布笔记后，若担心笔记是否被官方收录，可利用此功能查询。

此外，“达人搜索”功能也极为有用。可以按昵称或笔记内容进行搜索。例如，搜索“美妆”领域（部分高级功能需付费）。按昵称搜索后，结果会显示达人的分类属性，如彩妆、护肤、母婴、美式饮品、服饰穿搭等。你可以根据自己的需求进行筛选。

结果页面还会显示达人属性，如是否为官方号、品牌号或头部达人，并提供丰富的数据，如达人对比、排名等。牵瓜数据网站的相关服务，如有需要，可在淘宝上寻找开通渠道，费用通常比较划算。

点击任意达人的详情页，可以查看其发布的数据、笔记分类、笔记详情及具体类型。例如，可以看到其笔记评论的“词云”，即高频词汇。善用此工具，能帮助我们更好地掌握小红书平台的数据参数。

利用工具完成案例搜集后，下一步就是内容的整理与创作。

## 内容整理与注意事项

关于内容整理，在之前的课程中已经讲过，主要采用“总-分-总”的结构来撰写。这个结构并不难掌握，如果觉得困难，建议回顾之前的课程内容。本节不再重复这一点。

最后，我们说一下其他注意事项。

以下是撰写笔记标题时的三个关键点。

1.  **控制字数**：标题字数尽量控制在 **12到18个字** 之间。
2.  **添加符号美化**：标题中可多带一些符号（如`❗`、`✨`、`🔥`）进行美化。因为小红书女性用户居多，经过美化的标题更能吸引她们的注意力。
3.  **加入具体数据**：在标题中加入具体数据可以增加真实感。例如，像“**百万**”、“**3天瘦5斤**”这类具体数字，能让人感觉更可信。因此，在拟定标题时，可以适当加入数字。
