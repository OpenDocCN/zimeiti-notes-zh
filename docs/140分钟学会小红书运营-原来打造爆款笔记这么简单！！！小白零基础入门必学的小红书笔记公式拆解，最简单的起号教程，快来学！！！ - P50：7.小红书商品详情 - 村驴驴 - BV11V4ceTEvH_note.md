上一节我们介绍了店铺的基础设置，本节中我们来看看商品详情页的具体构成与制作要点。商品详情页是用户了解产品的核心窗口，其制作质量直接影响购买决策。

## 1. 主图设计 📸

商品详情的第一张主图至关重要，它决定了用户是否愿意继续浏览。后续四张图片的重要性相对较低。

主图设计的核心是**第一张产品主图**。它需要在三秒内抓住用户眼球，展示产品最吸引人的一面。如果主图缺乏吸引力，店铺将难以起步。

以下是主图设计的几个关键点：

*   **根据产品类型调整策略**：
    *   **批发/多SKU商品**：文案可以稍多，用于说明产品组合或优势。
    *   **单品类商品（如服装、饰品、家具）**：文案应精简，甚至不加文字。重点是通过高清图片展示产品的最佳角度和外观。
*   **设计原则**：
    *   选择**高清、具有代表性**的图片，展示商品整体外观和细节。
    *   如需添加文字描述产品优点，文字面积**不应超过屏幕的1/5**。
    *   主图整体应保持**整洁、干净**，避免满屏文字。

**核心公式**：**吸引力 = 高清图片 + 精准展示 + (极简文案)**

## 2. 标题与副标题 🔤

标题和副标题虽然都是标题，但作用截然不同。

*   **产品标题**：
    *   **作用**：用于用户搜索和平台关键词匹配，能带来约30%的搜索流量。
    *   **制作方法**：应包含1-2个**主关键词**，其余用**长尾词**补充。确保60个字符内无重复字眼，保持精细化。
    *   **示例代码（伪代码）**：`标题 = 主关键词 + 属性词 + 长尾词`
*   **副标题**：
    *   **作用**：无搜索权重，主要用于**详细描述产品特点**，提升用户信任感和成交率。
    *   **制作方法**：根据产品特性自由编辑，突出卖点。

## 3. 商品描述 📝

详细、准确的商品描述是基础，也关乎店铺安全。

商品描述必须详细说明商品的**规格、材质、颜色、尺寸**等信息，确保顾客能全面了解。描述与实物不符可能导致系统判定为虚假宣传，引发投诉并导致商品下架。

## 4. 主图SKU与商品视频 🎬

这部分内容相对直观。

*   **主图SKU**：指商品内的具体选项，如颜色、款式、尺码、价格等分类。根据自身产品情况编辑即可。
*   **商品视频**：
    *   **时长建议**：控制在**15秒左右**。视频过大可能因加载缓慢导致用户流失。
    *   **内容**：动态展示产品亮点。

## 5. 详情页制作 🖼️

详情页是主图的延伸和细化，用于多角度展示商品。

详情页应展示产品的**细节、特征和优点**。图片数量建议在**9到12张**左右。需注意整体浏览体验：

**整体浏览时长（视频+图片）建议控制在40秒左右**。详情页图片较多时，商品视频就应缩短，以平衡体验。

## 6. 售后服务说明 🛡️

售后服务说明是详情页的必备项，通常放在最后一张详情页。

一个清晰的售后服务说明（如发货时间、快递方式、7天无理由退换货规则等）能**增强用户安全感，提高成交率**，同时也是发生纠纷时保护卖家的凭证。提前写明规则，可以有效减少后续的投诉和差评。
