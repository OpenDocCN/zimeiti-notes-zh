# 自媒体运营入门：P2：1. 什么是自媒体 🎬

![](img/3a265eadd7ad5da923f8164f4d98606b_0.png)

在本节课中，我们将要学习自媒体的核心定义，并了解其基本运作模式。这是开启自媒体之路的第一步。

---

上一节我们介绍了课程的整体框架，本节中我们来看看“自媒体”这个概念本身。

到底什么是自媒体呢？用最直白的话来讲，自媒体就是**个人在互联网平台上，通过实名认证的账号，发布自己原创或加工处理过的内容（如文章、视频）**。

以下是构成自媒体的几个关键要素：

![](img/3a265eadd7ad5da923f8164f4d98606b_2.png)

1.  **个人账号**：你需要在一个自媒体平台（如B站、抖音、微信公众号等）上，注册一个属于自己的账号。这个账号需要**实名认证**。
2.  **年龄门槛**：通常，注册和运营自媒体账号有年龄限制，主要平台要求用户**年满18周岁**。
3.  **原创内容**：你必须在自己的账号里发布**自己制作或加工的内容**。简单地将别人的视频搬运到自己的账号上，是无法获得收益的。核心公式是：**自媒体收益 ≠ 搬运内容**。
4.  **发布与传播**：将制作好的文章或视频通过你的账号发布出去，让平台上的其他用户看到。

这个过程——**注册实名账号并发布原创内容**——就是自媒体最基本的形态。

本节课中我们一起学习了自媒体的基本定义。它并非遥不可及的技术，而是每个人在满足平台规则（如实名、成年）后，都可以通过创作和分享内容来参与的领域。理解这一点，是后续学习如何选择平台、创作内容和实现变现的基础。下一节，我们将探讨自媒体行业的现状与前景。
