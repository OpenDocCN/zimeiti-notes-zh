# 新媒体运营：P29：11：小红书平台用户标签及流量入口 📊

在本节课中，我们将要学习小红书平台如何为用户打上标签，以及平台主要的流量入口有哪些。理解这些机制，可以帮助我们分析笔记是如何被系统推荐给用户的，并反推出让笔记被精准用户看到的方法。

上一节我们介绍了小红书如何判定内容标签，本节中我们来看看平台如何判定用户标签。

## 用户标签判定机制

![](img/cc9b3d9fe7ad92d1141591b6688e652a_9.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_11.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_12.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_13.png)

小红书系统为用户打标签的主要依据是用户的**浏览行为**。用户注册时选择的兴趣范围参考价值不大，它仅作为初期内容方向的引导和初步筛选，无法进入系统最深层的用户标签池。

![](img/cc9b3d9fe7ad92d1141591b6688e652a_15.png)

系统根据用户的一系列浏览行为来判定其兴趣分类。以下是主要的判定维度：

![](img/cc9b3d9fe7ad92d1141591b6688e652a_16.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_17.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_18.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_19.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_20.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_21.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_22.png)

### 1. 点击率
点击率是指系统在发现页向用户推送多个内容后，用户点击不同类型内容的频率。例如，系统可能同时推送明星、美妆、家居、母婴等内容。通过分析用户点击哪些内容更多、哪些根本不点，系统可以计算出用户的兴趣偏好。

![](img/cc9b3d9fe7ad92d1141591b6688e652a_24.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_26.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_28.png)

**公式示例**：`用户点击率 = (点击某类内容的次数 / 总推送内容数) * 100%`

![](img/cc9b3d9fe7ad92d1141591b6688e652a_30.png)

如果用户点击10条内容中，美妆类点了3次，家居类点了2次，母婴类未点击，系统会判定该用户对美妆更感兴趣，并持续推送更多美妆内容进行试探。

![](img/cc9b3d9fe7ad92d1141591b6688e652a_32.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_33.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_34.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_35.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_36.png)

### 2. 停留时长
停留时长指用户点开一篇笔记后观看的时长。即使同属一个大类（如美妆），如果用户点开后很快划走，说明该细分内容（如上粉）不吸引他。系统会尝试推送该大类下的其他细分内容（如护肤），通过观察停留时长来判断用户喜好。

![](img/cc9b3d9fe7ad92d1141591b6688e652a_38.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_39.png)

如果用户对平时很少点击的领域（如家居）某篇笔记停留很久，甚至仔细阅读图片和评论，系统可能判定他有相关潜在需求（如装修或购买特定商品），并考虑推送更多相关内容。

![](img/cc9b3d9fe7ad92d1141591b6688e652a_40.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_41.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_42.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_43.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_44.png)

### 3. 页面浏览量（PV）
PV指用户浏览页面的深度。例如，用户点开一篇装修笔记后，还进入了博主主页，并连续观看了第二、第三、第四篇笔记。这代表用户浏览深度高。

![](img/cc9b3d9fe7ad92d1141591b6688e652a_46.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_47.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_48.png)

对于博主而言，将主页内容做得精致，可以提升用户的PV。这个数值对账号健康度评分有很大帮助。浏览深度足够时，用户可能产生**关注**行为，这能永久提升账号权重。

![](img/cc9b3d9fe7ad92d1141591b6688e652a_50.png)

### 4. 互动行为
互动行为包括点赞、收藏、评论、关注。这些数据直接关系到笔记能否进入更大的推荐流量池。

![](img/cc9b3d9fe7ad92d1141591b6688e652a_51.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_52.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_53.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_54.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_55.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_56.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_57.png)

**核心逻辑**：互动数据高的笔记，获得推荐的机会更大，同时可能带动账号内其他笔记的曝光，并提升粉丝数。这是我们运营小红书需要持续努力提升的目标。

![](img/cc9b3d9fe7ad92d1141591b6688e652a_58.png)

## 平台流量分发入口

了解用户标签后，我们来看小红书平台主要的流量分发入口。目前小红书有四个主要流量入口：发现页、关注页、附近页和购物页。

以下是各入口的详细介绍：

![](img/cc9b3d9fe7ad92d1141591b6688e652a_72.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_73.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_74.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_75.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_76.png)

### 1. 发现页
发现页是用户获取流量的主要来源。它呈现的内容包括：
*   **已关注的达人**发布的新内容。
*   根据用户**历史点击率和浏览行为**（用户标签）匹配的、系统认为用户可能感兴趣的内容。
*   少量用户未接触过的**发散性内容**，以增加平台内容的多样性，满足用户探索新事物的需求。

![](img/cc9b3d9fe7ad92d1141591b6688e652a_77.png)

### 2. 关注页
关注页的内容相对简单，只展示用户**已关注的所有博主**发布的笔记。当用户关注列表很长时，可以在这里集中查看。

![](img/cc9b3d9fe7ad92d1141591b6688e652a_79.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_80.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_81.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_82.png)

### 3. 附近页
附近页的内容基于用户的**手机定位**，推送附近或同城的相关内容。这个入口以具有强地域特征的需求为主，例如：
*   美食探店
*   本地旅游攻略
*   线下门店推广（如美容院、景点）

![](img/cc9b3d9fe7ad92d1141591b6688e652a_83.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_85.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_87.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_89.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_91.png)

**运营提示**：如果你是线下实体店商家，在做小红书信息流广告时，应重点关注对附近人群的推送。

![](img/cc9b3d9fe7ad92d1141591b6688e652a_93.png)

### 4. 购物页
购物页是小红书向电商方向发展的重要入口，直接匹配用户的购物需求。目前，部分用户尚未养成在小红书商城直接购物的强习惯，更多人仍倾向于先搜索相关笔记了解产品。笔记中植入的广告（软广）也主要服务于最终的购物转化。

本节课中我们一起学习了小红书平台用户标签的判定机制与四大流量入口。

![](img/cc9b3d9fe7ad92d1141591b6688e652a_110.png)

*   **用户标签**：系统通过分析用户的**点击率、停留时长、页面浏览量（PV）和互动行为**等浏览行为，为用户打上兴趣标签，从而实现内容的精准推荐。
*   **流量入口**：平台的流量主要来自**发现页、关注页、附近页和购物页**。其中，**发现页**是笔记获取推荐流量的核心战场，其推荐逻辑紧密依赖于用户标签；**附近页**对线下本地商家尤为重要；**购物页**则是平台商业变现的关键路径。

![](img/cc9b3d9fe7ad92d1141591b6688e652a_111.png)

![](img/cc9b3d9fe7ad92d1141591b6688e652a_112.png)

理解“用户-平台-博主”三者之间通过标签和流量分发形成的关联，是做好小红书运营的基础。作为博主，需要创作既能满足平台内容多样性要求，又能为目标用户提供价值的内容，从而在系统中获得更好的曝光。
