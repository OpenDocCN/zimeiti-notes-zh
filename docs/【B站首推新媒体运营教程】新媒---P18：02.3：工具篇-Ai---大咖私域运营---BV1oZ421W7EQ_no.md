# 小红书运营全平台教程：P18：02.3：工具篇-AI生成PPT 🛠️

## 概述
在本节课中，我们将学习第二款强大的AI生成PPT工具。上一节我们介绍了第一款AI生成PPT工具，本节中我们来看看另一款互补的工具，它拥有更快的速度和更丰富的模板库。

## 工具核心优势
这款工具主要有两个优势。第一是生成速度快且稳定。第二是模板数量非常庞大，提供了丰富的样式选择，便于我们调整细节。它与之前分享的工具可以相辅相成，共同提升制作PPT的效率。

## 两种生成方式
该工具提供了两种核心的PPT生成方式。

以下是两种方式的具体说明：
1.  **AI智能生成**：直接输入主题，由AI自动构思大纲并生成内容。
2.  **导入本地大纲**：上传或粘贴已准备好的详细大纲文档，由AI基于内容生成PPT。

我个人更常使用“导入本地大纲”的方式，因为可控性更强。

## 实践操作：导入本地大纲
接下来，我们通过一个实例来演示如何使用“导入本地大纲”功能。

![](img/1d558a10a13a75256cc19f8f869098be_3.png)

![](img/1d558a10a13a75256cc19f8f869098be_5.png)

![](img/1d558a10a13a75256cc19f8f869098be_6.png)

首先，我们需要准备一份详细的大纲。我们可以借助ChatGPT等工具来辅助生成。例如，我们可以向ChatGPT提问：“请为我撰写一份‘装修行业在小红书平台的运营推广计划’的详细大纲，并完善每一个小节点的具体内容信息，保持好结构和格式排版。”

获得大纲文本后，我们进入PPT生成工具的操作界面。

以下是操作步骤：
1.  选择“导入本地大纲”功能。
2.  将准备好的大纲文本粘贴进去。工具会快速解析文档结构。
3.  在丰富的模板库中，选择一个符合行业风格（例如装修行业）的模板。
4.  点击“生成PPT”，工具便会基于大纲内容和所选模板自动生成幻灯片。

生成完成后，我们进入编辑界面。如果发现某些页面的内容有缺失或排版不理想，可以进行手动调整。

编辑界面提供了完整的在线编辑功能。

以下是主要的编辑功能：
*   **文字/形状/背景设置**：在右侧面板调整页面元素的样式。
*   **页面布局更换**：单击页面，可以选择其他预设的布局模板一键更换。
*   **整体模板替换**：在“模板替换”功能中，可以为整个PPT快速更换另一套设计风格，响应速度很快。

**核心要点**：要想获得更好的生成效果，关键在于提供**结构清晰、内容详尽**的大纲文档。有时调整一下大纲的格式（例如合理换行），能帮助AI更好地理解内容结构。

## 实践操作：AI智能生成
现在，我们来尝试第二种“AI智能生成”方式。

操作流程非常简单：
1.  在工具中选择“AI智能生成”功能。
2.  在输入框中直接键入主题，例如“装修行业在小红书平台的运营推广计划”。
3.  点击生成，AI会先自动构思大纲，然后基于大纲生成完整的PPT文案和设计。
4.  生成完毕后，同样可以进入编辑界面，对所有内容和样式进行二次调整和优化。

这种方式适合快速搭建一个PPT初稿，后续再结合具体需求进行深度编辑。
