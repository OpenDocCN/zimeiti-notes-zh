## 项目原理与案例展示

上一节我们介绍了课程目标，本节中我们来看看这个项目的核心逻辑与成功案例。

视频号的“创作者分成计划”允许创作者通过发布视频获得广告收益。平台会在视频评论区展示广告，用户观看或点击广告，创作者即可获得分成。

一位学员通过此方法，在十几天内实现了日均约1500元的收益，单日最高收益达3547元。其成功关键在于选择了“祝福类”赛道，内容深受目标用户喜爱，作品爆款率极高。

![](img/2b3914b97a287deb5f9c8110b7dd91b6_6.png)

以下是该赛道的一个典型视频案例：
视频内容为对子女的祈福，例如“愿我的孩子健康平安、前程似锦”。此类视频数据表现优异，一个案例获得了5.3万点赞、4.6万转发和2万多评论。

数据优异的原因是，视频号的主要用户群体为中老年人。当他们刷到为儿女祈福的内容时，很容易产生共鸣，从而进行点赞、评论和转发至家族群，这极大地提升了视频的互动与传播。

## 两种核心变现方式

理解了项目原理后，我们来看看具体的盈利方法。视频号祝福类账号主要有两种变现途径。

**第一种是创作者分成计划。**
其核心公式为：
`收益 = 视频广告曝光量 × 单价 + 广告点击量 × 单价`
平台会在你的视频评论区自动植入广告（如图中“蓝图汽车”广告）。用户只要看到或点击该广告，你就能获得相应的佣金分成。这是实现稳定被动收入的基础。

![](img/2b3914b97a287deb5f9c8110b7dd91b6_12.png)

![](img/2b3914b97a287deb5f9c8110b7dd91b6_13.png)

**第二种是商品橱窗带货。**
中老年用户在为子女、孙辈购物时消费意愿较强。开通商品橱窗（需缴纳保证金）后，可以在主页展示相关商品。你可以在视频爆火后，通过评论区引导用户前往橱窗购买，从而赚取销售佣金。

## 开通分成计划的条件与步骤

![](img/2b3914b97a287deb5f9c8110b7dd91b6_15.png)

明确了变现方式，下一步是开通权限。开通“创作者分成计划”需要满足两个前提条件。

![](img/2b3914b97a287deb5f9c8110b7dd91b6_17.png)

以下是两个必须满足的条件：
1.  **内容规范**：发布内容必须符合平台规定，无违规、搬运、低俗或敏感信息。
2.  **粉丝数量**：拥有100个以上的有效关注者（即真实粉丝）。

![](img/2b3914b97a287deb5f9c8110b7dd91b6_19.png)

满足以上条件后，即可按以下步骤申请开通：

![](img/2b3914b97a287deb5f9c8110b7dd91b6_21.png)

以下是开通“创作者分成计划”的具体步骤：
1.  打开微信视频号，点击右上角个人头像。
2.  进入“创作者中心”。
3.  在“创作者服务”区域找到并点击“更多”选项。
4.  在列表中找到“视频变现任务”并点击进入。
5.  在申请界面，确认满足上述两项条件后，点击申请开通。

如果界面没有申请入口，则说明账号尚未满足条件，需要先通过发布内容积累粉丝。
