# 亚马逊开店教程：P16：12、如何注册店铺：资料准备 📋

![](img/436ea802ee249145e5bb8abd2d7ee312_0.png)

在本节课中，我们将要学习注册亚马逊店铺前必须准备的各项资料。了解并备齐这些资料，是成功开启亚马逊卖家之旅的第一步。

上一节我们介绍了如何选择亚马逊站点，本节中我们来看看注册店铺具体需要准备哪些材料。

## 资料清单概述

注册亚马逊店铺需要准备六项核心资料。以下是每一项资料的详细说明与注意事项。

### 1. 营业执照彩色扫描件

第一份需要准备的资料是**营业执照的彩色扫描件**，即营业执照的照片。

*   **执照类型**：可以是企业性质的执照，也可以是个体工商户性质的执照。
*   **办理方式**：可在当地工商局免费办理。若没有执照，可借用家人或朋友的执照，或寻找代办机构协助办理，费用通常在几百至一千元人民币左右。
*   **执照要求**：
    *   **注册资本与年限**：无硬性要求。无论是1元还是1000万注册资本，无论是新办执照还是已注册多年的执照，均可使用。
    *   **经营范围**：需带有销售性质即可（例如：百货销售、服装零售）。**特别注意**：**切勿**在经营范围中添加“进出口贸易”字样。添加后，将必须办理复杂的进出口手续，属于画蛇添足。
    *   **与销售产品的关系**：执照的经营范围与你在亚马逊店铺中实际销售的产品**不需要一致**。例如，使用超市的营业执照，同样可以销售服装。

### 2. 法人身份证正反面照片

第二份需要指明的资料是**法人身份证的正反面照片**。

*   **关联性要求**：此处的“法人”指营业执照上的法定代表人。**营业执照与法人身份证必须是同一个人的**。
*   **与其他资料的关系**：除了营业执照和法人身份证必须一致外，后续其他资料可以不是同一个人。例如，你可以使用朋友的营业执照和身份证注册，但将其他支付、收款账户绑定为你自己的。

### 3. 付款信用卡 💳

第三份需要准备的资料是用于支付亚马逊月租的**付款信用卡**。

*   **唯一作用**：此信用卡的唯一用途是支付亚马逊店铺的月租（目前为39.99美元/月）。
*   **卡片类型**：需要使用**全币种信用卡**，首选带有 **Visa** 或 **Mastercard** 标识的卡片。国内各大银行（如工商银行、建设银行、招商银行等）均可办理。
*   **备选方案**：如果没有信用卡，可以借用他人的，或花费约100元购买一张虚拟信用卡，每月按时存入足额月租即可。
*   **扣款逻辑**：该信用卡主要在店铺无销售额时扣除月租。一旦店铺产生营业额，亚马逊会优先从店铺余额中扣除月租，而非信用卡。

### 4. 海外收款账户 🌐

第四份需要准备的资料是**海外收款账户**。

![](img/436ea802ee249145e5bb8abd2d7ee312_2.png)

*   **核心功能**：此账户用于接收亚马逊店铺的销售货款，并将其提现至你的国内银行储蓄卡。其功能类似于**国际版的支付宝**。
*   **获取方式**：可以通过第三方支付服务商（如Payoneer、PingPong、万里汇等）在线免费注册。注册过程简单快捷，只需绑定个人身份证和国内收款银行卡即可。
*   **重要性**：这是你将亚马逊销售收入转回国内的必要通道。

### 5. 电话号码 📞

第五份需要准备的资料是**电话号码**。

*   **要求**：一个**国内的手机号码**即可，用于接收验证码和官方通知。

### 6. 邮箱地址 📧

第六份需要准备的资料是**邮箱地址**。

*   **要求**：可以使用163邮箱、126邮箱、QQ邮箱等。**个人推荐使用163邮箱**，其在国内外邮件通讯中稳定性较好。
