# 剪映手机版教学：P10：画中画功能详解 🎬

在本节课中，我们将要学习剪映的核心剪辑功能之一——画中画。这个功能允许你将多个视频或图片叠加在同一个时间线上播放，从而创造出内容更丰富、更有趣的视频效果。

## 画中画功能概述

![](img/6f26e784b827725aef95134882db11c4_1.png)

上一节我们介绍了剪映的其他核心功能，本节中我们来看看**画中画**。我们经常看到一些视频中有多个画面同时播放，但这些画面可能并不在同一时间线上。这种效果就是通过画中画功能实现的。

在剪映中，许多特效视频也借用了画中画功能。其原理是将两个或多个视频素材叠加在同一个时间轨道上同步播放。

![](img/6f26e784b827725aef95134882db11c4_3.png)

**核心操作**可以概括为：`主轨道视频 + 新增画中画轨道 = 多画面叠加效果`。

![](img/6f26e784b827725aef95134882db11c4_5.png)

使用画中画的一个重要特点是能让视频内容变得更加丰富和有趣。

![](img/6f26e784b827725aef95134882db11c4_7.png)

## 画中画实操教程

![](img/6f26e784b827725aef95134882db11c4_9.png)

关于画中画的介绍就到这里，接下来我们带大家进行画中画的实际操作。

首先，打开剪映软件，点击“开始创作”，导入一张照片作为背景素材。

![](img/6f26e784b827725aef95134882db11c4_11.png)

![](img/6f26e784b827725aef95134882db11c4_12.png)

导入照片素材后，我们先对照片进行简单的调整。点击底部工具栏的“比例”，将其设置为 **9:16**，以适应手机竖屏观看。

![](img/6f26e784b827725aef95134882db11c4_14.png)

调整好比例后，我们拉长照片的显示时长，以便后续操作。

![](img/6f26e784b827725aef95134882db11c4_16.png)

然后，我们回到一级菜单，选择“画中画”功能，点击“新增画中画”，导入另一个视频素材。这里我们选择一个星空特效视频作为例子。

![](img/6f26e784b827725aef95134882db11c4_17.png)

![](img/6f26e784b827725aef95134882db11c4_18.png)

导入星空素材后，我们发现前面有多余的部分。这时需要将多余的部分删除。

以下是删除多余片段的具体步骤：
1.  将时间线光标移动到需要保留的起始位置。
2.  点击底部工具栏的“分割”按钮。
3.  选中分割出来的前段多余部分，点击“删除”。
4.  将保留的后段视频素材，拖动到时间线的前端。

现在，我们看到了两个视频画面在同一轨道上叠加的效果。像后期会讲到的三连屏或多屏互动效果，也是通过画中画实现的。

我们想将星空特效叠加在背景照片上，但当前星空素材带有黑色背景。我们需要将其抠除。

处理方法是使用画中画的“混合模式”功能。

![](img/6f26e784b827725aef95134882db11c4_20.png)

以下是使用混合模式抠除背景的步骤：
1.  选中画中画轨道（星空素材）。
2.  在底部工具栏中找到并点击“混合模式”。
3.  在混合模式选项中，选择“绿色”或“滤色”模式。
4.  此时，黑色背景就被去掉了。

接着，我们可以调整星空特效的位置和大小，将其放置在背景照片的合适位置。调整完成后，预览一下效果。

这样，我们就通过添加画中画的形式，为一张静态照片制作出了动态的星空特效。

## 课程总结

本节课中我们一起学习了剪映中的画中画功能。

在剪映中，点击画中画功能，可以直接在主视频轨道上新增另一个视频轨道，实现多轨道视频的同步运行。这让单一的视频画面可以变成两个甚至多个画面的同步播放，从而极大地丰富了视频内容。

理解和掌握画中画功能，对我们后期制作特效视频有很大帮助。就像本节课的案例操作，我们通过画中画实现了一个星空特效。未来在学习三连屏、人影分身等技术时，都会运用到画中画功能。
