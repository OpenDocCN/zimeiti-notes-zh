## 讲品前的铺垫

互动环节结束后，不能立即上架产品开始销售。即使活动力度大、价格低，观众也可能不会立刻买单。关键在于，观众需要知道产品是否好，以及自己是否需要。因此，讲品环节至关重要，其核心在于如何有效地讲解产品。

## 讲品的三大核心话术

讲解产品不仅仅是罗列优点。一套优秀的话术需要从多个维度打动观众。以下是构建讲品话术的三个核心部分。

### 1. 卖点话术：突出产品优势

卖点话术的核心是阐述产品自身的优势，即“我的产品哪里好”。这是讲品的基础。

![](img/4d3268bf9be666c8c92c302252f0f1c5_8.png)

卖点可以涵盖多个方面，其核心公式可以概括为：
**产品卖点 = 工艺/材质 + 配方/成分 + 原产地/供应链 + 性价比/服务**

以下是常见的卖点方向：
*   **工艺与材质**：产品的制作工艺、所使用的核心材料。
*   **成分配方**：对于食品、护肤品、保健品等，其核心成分与科学配比。
*   **原产地与供应链**：产品的源头、发货速度、物流保障（如冷链）。
*   **性价比与对比**：相较于同类产品的价格优势、更高的性价比。

不同类型的产品，侧重点不同：
*   **服饰类**：需重点讲解版型、尺码、面料。
*   **食品类**：需强调材质、安全性（可展示检验报告）、适宜人群。
*   **保健品类**：需清楚说明具体功效和使用方法。

然而，仅靠罗列卖点并不足以促成销售。产品好，不代表观众需要。

### 2. 痛点话术：激发用户需求

痛点话术的核心是挖掘并关联用户需求，解决“观众为什么需要”的问题。只有当产品与用户的具体问题（痛点）相关联时，购买意愿才会被激发。

痛点话术的逻辑是：
**用户需求 = 明确目标人群 + 定位具体问题 + 提供解决方案**

我们可以通过一个例子来理解：
*   **卖点**：这款油烟机吸力强、静音。
*   **痛点话术**：“直播间的妈妈们，你们有没有这种困扰？每天做完饭，身上、头发上全是油烟味，家人回家都说味道重。我们这款油烟机，通过独特的XX设计，不仅能强力吸油烟，更能确保您在烹饪时身上不沾染任何异味，彻底解决您的烦恼。”

从“吸力强”到“解决身上油烟味”，后者直接击中了目标人群（家庭主妇）的深层痛点，从而让产品变得“与我有关”。

构建痛点话术的步骤：
1.  **明确产品卖点**。
2.  **思考卖点能解决什么具体问题**。
3.  **定位遇到这些问题的人群**，并在话术中与他们对话。

### 3. 场景话术：营造使用体验

场景话术的核心是将产品的功效和使用效果，通过生动的语言描绘出来，让观众产生身临其境的“感觉”，为理性的卖点和痛点注入感性的价值。

![](img/4d3268bf9be666c8c92c302252f0f1c5_10.png)

![](img/4d3268bf9be666c8c92c302252f0f1c5_12.png)

场景话术的关键在于调动“五感”进行描述：
**生动场景 = 视觉描述 + 听觉描述 + 嗅觉描述 + 味觉描述 + 触觉描述**

举例说明：
*   **基础表述**：这款牙膏能让口气清新。
*   **场景话术**：“使用这款牙膏后，你会感觉每一次呼吸，都像漫步在清新的柠檬果园里，口气自然持久清香。” 这通过嗅觉联想，创造了美好的使用体验。

![](img/4d3268bf9be666c8c92c302252f0f1c5_14.png)

另一个例子（服装）：
*   **基础表述**：这条裤子显腿长。
*   **场景话术**：“穿上这条裤子，面料顺滑贴身，走起路来裤腿线条利落流畅。明天约会或上班，别人一眼看去就会觉得你整个人特别挺拔、有精神。” 这结合了触觉、视觉和心理感受。

通过场景化的描述，硬性的产品参数变成了可感知的、令人向往的体验，极大地增强了话术的感染力。

## 话术的综合运用与总结

优秀的讲品话术是**卖点话术**、**痛点话术**和**场景话术**三者的有机结合。硬性的优势陈述、对需求的精准挖掘，以及柔软感性的体验描绘，共同作用，才能层层递进，真正打动观众，促成购买决策。这正是普通讲解与优秀讲解的关键区别所在。

![](img/4d3268bf9be666c8c92c302252f0f1c5_16.png)
