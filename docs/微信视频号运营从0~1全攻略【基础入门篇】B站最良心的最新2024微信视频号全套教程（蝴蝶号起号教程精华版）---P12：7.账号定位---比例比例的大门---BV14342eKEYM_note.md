# 微信视频号运营从0~1全攻略：P12：7.账号定位 🎯

在本节课中，我们将要学习微信视频号运营中关于账号定位的两个核心问题：如何处理旧作品以及如何获取原创视频素材。本节内容旨在为初学者提供清晰、可执行的解决方案。

## 旧作品处理与发布频率

上一节我们介绍了账号定位的重要性，本节中我们来看看如何为账号的初始阶段做准备。

![](img/009bacc118a53c0d1cc5e13eadc4e56e_1.png)

首先，需要将账号之前发布的作品隐藏。这是为了确保账号以全新的定位面向观众。

![](img/009bacc118a53c0d1cc5e13eadc4e56e_3.png)

关于发布频率，初期应保持每天发布一条视频的节奏。素材准备无需操之过急，按此节奏稳定发布即可。

## 初期目标：获取100个粉丝

在解决素材问题之前，我们优先设定一个明确的初期目标：为账号积累100个粉丝。完成这个目标后，我们再集中精力解决视频素材的来源问题。

## 解决原创视频素材问题

![](img/009bacc118a53c0d1cc5e13eadc4e56e_5.png)

![](img/009bacc118a53c0d1cc5e13eadc4e56e_6.png)

![](img/009bacc118a53c0d1cc5e13eadc4e56e_7.png)

解决了粉丝基础目标后，接下来我们重点解决原创视频素材的来源问题。许多初学者面临不会拍摄、不会剪辑的困境。

以下是初学者常见的三种情况：
*   不会拍摄视频。
*   不会剪辑视频。
*   不会制作完整的视频内容。

针对这些普遍情况，我们提供了一个解决方案：每位学员可以领取30个已经剪辑制作完成的视频素材。这些素材来源于一个投入大量资金建立的独家素材库，涵盖了多种类型。

例如，素材库中包含“搞笑娱乐”类的高质量剪辑视频。这些视频由专业团队制作，内容精良，可以直接使用。

## 如何领取素材

![](img/009bacc118a53c0d1cc5e13eadc4e56e_9.png)

领取素材需要满足几个简单的条件。

以下是领取素材的三个步骤：
1.  在互动公屏上发送指定关键词。
2.  在课程中保持积极互动。
3.  完整听完课程（后台会统计听课时长与互动数据）。

满足条件后，即可联系发放课程链接的助理老师领取素材。请注意沟通方式，表明是遵循课程老师的指导前来领取。
