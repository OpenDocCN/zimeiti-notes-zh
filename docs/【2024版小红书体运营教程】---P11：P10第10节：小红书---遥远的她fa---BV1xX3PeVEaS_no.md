# 小红书运营教程：P11：三种热门笔记类型详解

在本节课中，我们将详细讲解小红书的三种热门笔记类型。掌握这些笔记的写作方法，可以有效提升内容的转化率。课程内容详实，建议在学习时做好笔记。

## 教育类笔记 📚

上一节我们介绍了笔记类型的重要性，本节中我们首先来看看教育类笔记。这种笔记的核心是传授知识或技能，适合将产品自然地植入内容中。

教育类笔记适合的行业包括：美妆、瘦身、美食、P图软件、美容、摄影等。在撰写时，可以将产品作为教学工具或案例融入。

教育类笔记的封面首图通常有三种构成方式：

以下是三种常见的首图类型：
*   **对比图**：展示使用产品前后的效果对比。例如，口红涂在手上与上唇后的颜色对比。
*   **效果图**：直接展示使用产品后的最终效果。例如，眼影上妆后的整体妆容效果。
*   **步骤图**：展示达成某个目标的具体操作步骤。例如，“五步告别圆肩驼背”教程中，可能包含辅助产品如矫正背带。

接下来，我们看看教育类笔记的正文应如何撰写。正文的目的是系统性地指导用户，通常可以遵循一个固定模板。

以下是教育类笔记正文的五步结构：
1.  **使用工具或材料**：列出所需的**产品**和辅助工具。
2.  **操作步骤**：详细说明具体的操作流程。
3.  **注意事项**：提示操作过程中需要避免的问题或关键点。
4.  **产品安利**：顺势推荐在步骤中使用的、你觉得好用的产品。**注意**：介绍产品的篇幅建议不超过全文的30%，以免被判定为硬广。
5.  **常见问题解答**：针对用户操作时可能遇到的**失败情况**进行解答，例如“为什么贴不上？”、“为什么效果不好？”。这类解答往往更实用，更能吸引读者。

## 合集类笔记 📦

了解了教育类笔记后，我们来看看适用范围更广的合集类笔记。这种笔记通过汇总推荐来满足用户多样化的需求。

合集类笔记几乎适用于所有行业。其核心思路是针对不同场景、人群或条件，推荐一系列相关产品或解决方案。例如：旅行必备清单、夏季穿搭指南、健身期推荐食品等。

撰写合集类笔记时，常见问题是角度单一。为了避免这个问题，我们可以从多个维度切入创作。

以下是撰写合集类笔记的七种不同角度：
1.  **针对人群**：例如，《学生党开学彩妆攻略》、《上班族代步电动车推荐》。
2.  **针对成本**：例如，《100元内平价护肤品》、《5元网红早餐包》。
3.  **针对效果**：例如，《让男友合不拢嘴的夏季穿搭》、《瘦身必备零食》。
4.  **针对地区**：例如，《香港平价美食推荐》、《南京本地网红店》。
5.  **针对场景**：例如，《旅游必备清单》、《居家清洁好物指南》。
6.  **针对产品属性**：例如，《微电流美容仪测评》、《某某色号口红合集》。
7.  **针对品牌**：例如，《体察内部员工隐藏菜单》、《星巴克夏季新品测评》。

解决了角度问题，我们还需要注意避免合集笔记推广嫌疑过重的问题。解决方案是均匀分布对各产品的介绍篇幅，并客观说明其**优点**和**缺点**，这样内容会显得更真实可信。

合集类笔记的封面首图主要有两种形式：
*   **产品拼图**：使用美图秀秀、黄油相机等APP，将多个产品图片拼接成一张图。
*   **产品摆拍图**：将推荐的所有产品实物，整齐地摆放在一个场景中进行拍摄。

我们来总结一下合集类笔记的要点。**标题**应从上述某个具体角度切入。**正文**一般应包含推荐理由和产品特点说明，并注意客观评价。如果配图已有详细文字说明，正文可简要提及产品名称，这有助于平台搜索引擎抓取关键词标签。

## 视频类笔记 🎥

最后，我们来看看目前平台非常支持且流量红利较大的视频类笔记。如果条件允许，建议尝试制作这类内容。

视频类笔记的创作主要涉及拍摄和剪辑两部分。

以下是视频制作所需的基本工具：
*   **拍摄工具**：手机、专业相机、三脚架、补光灯。预算允许可购置简易摄影棚。
*   **剪辑工具**：手机APP（如剪映），或电脑专业软件（如 **Adobe Premiere Pro**）。

关于视频规格，需要注意以下几点：
*   **视频时长**：小红书普通笔记视频时长限1分钟。持续发布优质视频，达到一定标准后，可向官方申请开通5分钟乃至更长的视频权限。
*   **视频尺寸**：推荐两种比例。
    *   **3:4（竖屏）**：大尺寸，视觉冲击力强，适合大多数内容。
    *   **4:3（横屏）**：画面更紧凑，适合展示化妆品等精致类产品。

视频内容方面，可以结合所在行业，制作教程、测评、好物分享、生活VLOG等。最有效的方法是**分析与模仿**：找到行业内成功的视频博主，总结他们的**视频脚本结构**（如开头如何吸引人、中间如何展示、结尾如何引导）、选品、构图和节奏，提炼出通用的成功方法。

本节课中我们一起学习了小红书的三种核心笔记类型：**教育类笔记**、**合集类笔记**和**视频类笔记**。掌握每种类型的写作技巧、结构要点和注意事项，能够帮助你创作出更受欢迎、转化效果更好的内容。请结合自身行业特点，灵活运用这些方法。
