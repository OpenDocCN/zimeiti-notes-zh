# 微信视频号运营从0~1全攻略：P2：视频号分成计划与快速起号教程

在本节课中，我们将要学习如何快速开通微信视频号的“分成计划”并获取收益。课程将涵盖分成计划的准入条件、快速过原创的实操方法、收益计算逻辑以及两个高潜力赛道的具体操作流程。

## 概述：什么是视频号分成计划？

视频号分成计划是微信视频号平台为创作者提供的一种变现方式。当创作者发布原创视频并开启“原创声明”后，平台会在视频评论区展示广告。广告产生的曝光与点击收入，将与创作者分成。

![](img/afdb33ca1f3288f3ece1150d9e7156aa_3.png)

上一节我们介绍了视频号的基础概念，本节中我们来看看如何快速加入分成计划并开始盈利。

## 开通分成计划的条件

想要开通视频号分成计划，你需要满足以下两个平台设定的条件：

以下是具体的条件列表：
1.  **粉丝数达到100以上**。
2.  **内容符合平台规范**，即视频需为优质原创，且画风精美。

## 快速开通分成计划的实操方法

目前，单纯拍摄风景类视频已很难通过原创审核。经过实测，以下方法能帮助新号在3天内快速开通分成计划。

以下是具体的操作步骤：
1.  **内容形式**：注册新号后，直接进行“怼脸拍”（人物正面出镜讲解）。
2.  **视频要求**：每条视频时长需超过9秒。
3.  **发布频率**：每天发布5条此类视频。
4.  **申请时机**：连续发布两天（共10条视频）后，在第三天即可找到分成计划入口申请加入。

**关键点**：申请通过后，发布视频时**务必勾选“原创声明”**。若不开启此功能，即使视频获得千万播放，也无法获得任何收益。

## 分成计划的收益如何计算？

平台未公布精确的计算公式，但根据运营经验，收益主要与广告点击率相关。

一个基本的收益估算范围是：
**1万播放量 ≈ 1~3元人民币**

收益高低受多重因素影响，核心公式可理解为：
**最终收益 = f(视频质量， 播放量， 广告点击率， 广告行业单价)**

因此，制作视频时，要想办法提升评论区的互动热度，从而增加广告的曝光和点击机会。

## 推荐赛道（一）：人性语录类

在众多内容领域中，人性语录类是目前非常推荐入局的赛道。它与常见的祝福类不同，通过精辟的话语揭露人性本质，能吸引大量30岁以上、粉丝质量高的用户群体，收益潜力可观。

该赛道的完整制作流程，包括文案获取、配音、视频制作等，已整理成标准化步骤。按照教程操作，即可制作出高热度的视频。

**运营要点**：
*   **坚持发布**：开通原创后，需保持更新。案列显示，收益从初期几元增长到数百元，通常需要7-10天的坚持。
*   **控制频率**：每个账号每天发布3-5条视频为宜，**切忌超过5条**，否则可能影响账号权重。

## 进阶玩法：电视剧解说搬运

除了原创内容，还有一种高效的玩法是搬运并处理电视剧解说视频。这类视频天然具有高传播性和互动性。

以下是核心操作思路：
1.  **素材获取**：从其他视频平台（如某瓜、某音）搜索并下载热门的电视剧解说视频。
2.  **批量处理**：使用专业的去重软件对下载的视频进行自动化处理，以通过平台的原创审核。
3.  **发布变现**：将处理后的视频发布到视频号，依靠分成计划变现。

该玩法的核心在于利用软件工具提升“搬运”和“去重”的效率。相关的软件和详细教程已打包整理。
