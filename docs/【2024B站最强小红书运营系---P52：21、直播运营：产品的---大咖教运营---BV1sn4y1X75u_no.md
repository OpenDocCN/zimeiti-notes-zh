在本节课中，我们将学习直播带货中产品的核心分类及其各自的作用。理解不同产品的角色，是构建高效直播策略的基础。

## 从线下门店理解“货场人” 🏪

直播带货的本质依然是卖货，只是场景从线下转移到了线上。我们可以通过对比线下开店的过程来理解“货场人”的逻辑。

![](img/881fbfbfc5883d54f2d674c153133f4e_7.png)

![](img/881fbfbfc5883d54f2d674c153133f4e_9.png)

以下是线下开店的三个核心步骤：

![](img/881fbfbfc5883d54f2d674c153133f4e_11.png)

![](img/881fbfbfc5883d54f2d674c153133f4e_13.png)

1.  **有货**：这是卖货的前提。你需要有货源，无论是自有工厂、供应链还是拿货渠道。
2.  **选址与装修（场）**：选址决定了人流量（即线上流量/曝光）。装修则要契合产品风格，目的是吸引目标顾客进店，并让产品看起来更好。
3.  **招人（人）**：需要店员来销售、导购、管理或策划营销方案，目的是把货卖得更好。

![](img/881fbfbfc5883d54f2d674c153133f4e_15.png)

线上直播的逻辑与此完全一致。区别主要在于流量来源：线下靠选址和品牌，线上则需遵循平台的流量规则。因此，直播带货的首要任务同样是解决“货”的问题。

## 产品的核心分类与作用 🎯

![](img/881fbfbfc5883d54f2d674c153133f4e_17.png)

![](img/881fbfbfc5883d54f2d674c153133f4e_19.png)

你的产品直接决定了会吸引什么样的人群。因此，直播前必须对产品进行科学分类。通常，一个健康的直播间产品结构应包含以下三类：

以下是三类核心产品及其作用：

*   **主推品（引流款）**
    *   **作用**：如同线下门店的“招牌菜”，用于稳定直播间基础数据、拉动流水和建立品牌认知。
    *   **特点**：**高性价比、高认知度**。利润通常偏低，但能吸引最广泛的人群，是讲解时间最长的产品。
    *   **公式**：`主推品 = 高性价比 + 高认知度 → 吸引广泛人群，稳住数据`

![](img/881fbfbfc5883d54f2d674c153133f4e_21.png)

*   **利润品（利润款）**
    *   **作用**：为直播间创造核心利润。针对对价格不敏感、更看重设计、质感或品牌的小众人群。
    *   **特点**：**利润高，但性价比相对不高**。因此出单量不会很大，但每单利润可观。
    *   **公式**：`利润品 = 高利润 + 独特价值（设计/质感） → 吸引小众人群，创造利润`

*   **福利品（钩子款）**
    *   **作用**：在切换产品（尤其是从主推品转向利润品）时，用于稳住在线人数，防止观众流失，甚至能拉高人气。
    *   **特点**：**绝对低价，甚至微亏**。性价比极高，不愁卖，因此需要**严格控制发放数量**，避免过度亏损。通常讲解时间很短。
    *   **公式**：`福利品 = 极限低价 + 限量 → 留住观众，平稳过渡，拉升数据`

![](img/881fbfbfc5883d54f2d674c153133f4e_23.png)

## 产品组合与排品策略 🔄

![](img/881fbfbfc5883d54f2d674c153133f4e_25.png)

![](img/881fbfbfc5883d54f2d674c153133f4e_27.png)

![](img/881fbfbfc5883d54f2d674c153133f4e_28.png)

理解了每类产品的作用后，关键在于如何将它们组合起来。当从主推品转向利润品时，由于目标人群不同，直播间容易掉人。此时，就需要用福利品进行衔接：通过极具吸引力的福利稳住观众，再顺势推出利润品。

![](img/881fbfbfc5883d54f2d674c153133f4e_30.png)

因此，直播前必须根据自身产品线，明确选出哪款是**主推品**、哪款是**利润品**、哪款作为**福利品**，并设计好它们的讲解顺序和策略。

![](img/881fbfbfc5883d54f2d674c153133f4e_32.png)

![](img/881fbfbfc5883d54f2d674c153133f4e_34.png)

![](img/881fbfbfc5883d54f2d674c153133f4e_36.png)

本节课中我们一起学习了直播产品的三大分类：**主推品**、**利润品**和**福利品**。记住，一个成功的直播间离不开这三类产品的科学组合与排布。下一节，我们将探讨如何构建一个吸引人的“场”，即直播间场景。
