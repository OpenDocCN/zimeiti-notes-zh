# 自媒体运营课程：P30：剪映核心剪辑功能讲解--音频功能 🎵

![](img/13eb95e341796a88140da3d592d9dec0_1.png)

在本节课中，我们将学习剪映App中音频功能的使用方法。一条完整的短视频不仅需要精彩的画面，也离不开合适的背景音乐和音效。音频功能能帮助我们为视频添加音乐、音效，甚至录制画外音，从而提升视频的整体氛围和观感。

上一节我们介绍了视频剪辑中的分割功能，本节中我们来看看如何为视频添加和处理音频。

![](img/13eb95e341796a88140da3d592d9dec0_3.png)

## 音频功能概述

![](img/13eb95e341796a88140da3d592d9dec0_5.png)

![](img/13eb95e341796a88140da3d592d9dec0_6.png)

![](img/13eb95e341796a88140da3d592d9dec0_7.png)

我们知道，一条吸引人的爆款视频离不开一个热门的背景音乐。在剪映中，我们可以通过音频功能为视频添加背景音乐或音效，还可以通过“提取音乐”和“抖音收藏”直接选用抖音平台上的热门BGM。最后，我们还能为视频配音录音。这些操作能显著提升视频的配音质量和效果，优化视频的画面氛围，让用户的观看体验更好。

音频功能位于剪映的一级菜单栏。添加视频素材后，在底部菜单栏点击“音频”，即可进入音频操作界面。

![](img/13eb95e341796a88140da3d592d9dec0_9.png)

![](img/13eb95e341796a88140da3d592d9dec0_11.png)

该界面包含以下五个核心功能：
*   **添加音乐**：从剪映音乐库选择背景音乐。
*   **音效**：为视频添加各种短促的音效。
*   **提取音乐**：从手机本地视频中提取音频。
*   **抖音收藏**：使用在抖音App中收藏的音乐。
*   **录音**：为视频录制画外音。

接下来，我们将通过实际操作，逐一详细讲解这五个功能。

## 实操演示：添加与处理音频

![](img/13eb95e341796a88140da3d592d9dec0_17.png)

我们首先导入一段视频素材。预览原素材可以发现，视频只有环境风声，缺乏明确的主题氛围。我们可以通过添加音频来提升视频的感染力。

### 1. 添加音乐

第一种方法是在视频轨道下方点击“添加音频”按钮。点击后，会进入音频操作界面。

我们首先看“添加音乐”功能。点击后，会跳转到剪映的音乐素材库。这个曲库与抖音的热门音乐是打通的，并按类别（如卡点、流行、旅行、美食、VLOG等）和热门推荐进行了划分。

例如，我们选择“夏天的风”这首音乐，点击“使用”按钮。音乐就会被添加到视频轨道下方的音频轨道上。

添加后，我们需要进行一些基本处理：
1.  **对齐轨道**：长按音频轨道，将其拖动至与视频开头对齐。
2.  **关闭视频原声**：点击视频轨道，选择“关闭原声”，避免原声干扰。
3.  **裁剪多余音频**：将时间线移动到视频结尾处，选中音频轨道，点击底部“分割”工具，将多余的音频部分分割并删除。
4.  **设置淡入淡出**：为了让音乐的开始和结束更自然，选中音频轨道，点击“淡化”选项。我们可以调整“淡入时长”和“淡出时长”，例如分别设置为1秒和2秒。

完成以上步骤后，再次预览视频，音乐过渡就会变得柔和自然。

在音乐轨道上，我们还可以进行其他调整：
*   **音量调整**：点击“音量”滑块，可以增大或减小背景音乐的音量。由于平台压缩可能导致音量变小，建议适当调大。
*   **踩点**：此功能用于制作卡点视频，后期课程会详细讲解。
*   **变速**：可以调整音频的播放速度，范围在0.2倍到4倍之间。

### 3. 提取音乐

![](img/13eb95e341796a88140da3d592d9dec0_31.png)

![](img/13eb95e341796a88140da3d592d9dec0_32.png)

![](img/13eb95e341796a88140da3d592d9dec0_33.png)

“提取音乐”功能允许我们将手机相册中某个视频的音频单独提取出来，并插入到当前剪辑的视频中。

![](img/13eb95e341796a88140da3d592d9dec0_34.png)

![](img/13eb95e341796a88140da3d592d9dec0_35.png)

![](img/13eb95e341796a88140da3d592d9dec0_36.png)

![](img/13eb95e341796a88140da3d592d9dec0_37.png)

操作步骤如下：点击“提取音乐”，从手机相册中选择一个带有音频的视频文件。导入后，该视频的音频会被提取为一个新的音频轨道，添加到当前项目中。之后，我们可以像处理普通音频一样对其进行调整。

![](img/13eb95e341796a88140da3d592d9dec0_38.png)

### 4. 抖音收藏

如果我们想在抖音上拍同款热门音乐视频，可以使用此功能。

![](img/13eb95e341796a88140da3d592d9dec0_40.png)

![](img/13eb95e341796a88140da3d592d9dec0_42.png)

操作步骤如下：
1.  在抖音App中，找到想要使用的音乐视频，点击右下角的“碟片”图标。
2.  点击顶部的“收藏”按钮，将该音乐收藏到自己的抖音账号中。
3.  回到剪映，在音频功能界面点击“抖音收藏”。
4.  在列表中找到并选中刚才收藏的音乐，点击“使用”即可导入。

### 5. 录音

![](img/13eb95e341796a88140da3d592d9dec0_44.png)

“录音”功能可以让我们为视频后期添加画外音或台词解说。

![](img/13eb95e341796a88140da3d592d9dec0_46.png)

![](img/13eb95e341796a88140da3d592d9dec0_48.png)

操作步骤如下：点击“录音”按钮，按住红色录音区域开始说话（类似微信语音）。录制完成后，生成的录音会以音频轨道的形式出现在时间线上。我们可以调整其音量大小，并移动到视频的合适位置。

例如，我们可以用录音功能为视频添加一段文案朗读，同时将背景音乐的音量调低，使文案内容更加突出。

## 课程总结 🎬

本节课我们一起学习了剪映核心剪辑功能中的音频功能。通过音频功能，我们可以快速为视频添加背景音乐和音效。

![](img/13eb95e341796a88140da3d592d9dec0_50.png)

总结其核心操作：
*   在剪映App中，我们可以直接从其音乐曲库添加抖音热门音乐。
*   可以使用丰富的音效素材库来增强视频表现力。
*   可以通过“提取音乐”和“抖音收藏”功能，灵活使用本地或抖音上的音频资源。
*   最后，还能通过“录音”功能为视频配置画外音。

![](img/13eb95e341796a88140da3d592d9dec0_52.png)
