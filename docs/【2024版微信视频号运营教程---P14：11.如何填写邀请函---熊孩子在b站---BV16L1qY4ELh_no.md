# 微信视频号运营教程：P14：11.如何填写邀请函 📝

![](img/c336683f5ca347183db52898d2e16888_1.png)

在本节课中，我们将学习如何正确填写视频号创作者计划的邀请函。这是一个非常关键的步骤，关系到您能否成功加入计划并开始运营。整个过程简单明了，我们将一步步进行拆解。

## 概述
填写邀请函需要准确提供您的视频号账号信息和个人信息。本节教程将手把手教您如何找到这些信息，并正确填写到指定位置。

![](img/c336683f5ca347183db52898d2e16888_3.png)

![](img/c336683f5ca347183db52898d2e16888_4.png)

## 教程步骤

### 第一步：打开微信
首先，您需要打开手机上的微信应用。这一步是后续所有操作的基础。

![](img/c336683f5ca347183db52898d2e16888_6.png)

![](img/c336683f5ca347183db52898d2e16888_7.png)

### 第二步：进入“发现”页面
打开微信后，在底部导航栏找到并点击“发现”选项卡。



### 第三步：找到“视频号”入口
进入“发现”页面后，向上滑动屏幕。您会看到“视频号”的入口，它可能显示为文字“视频号”，也可能显示为一个黄色的蝴蝶图标。点击此处进入视频号。



### 第四步：进入个人主页
进入视频号界面后，请点击屏幕右上角的个人头像图标。这将进入您的个人主页。



### 第五步：查看账号信息
在个人主页中，向下滑动屏幕。您会看到“我的视频号”栏目，这里清晰地展示了您的**账号昵称**和**粉丝数量**。
例如，显示为：`天道酬勤，39917个粉丝`。
*   **账号昵称**：`天道酬勤`
*   **粉丝数量**：`39917`



### 填写邀请函
找到以上信息后，我们就可以开始填写邀请函了。以下是需要填写的核心项目：

![](img/c336683f5ca347183db52898d2e16888_9.png)

![](img/c336683f5ca347183db52898d2e16888_10.png)

![](img/c336683f5ca347183db52898d2e16888_11.png)

1.  **视频号账号昵称**：填写您在上一步中查看到的账号昵称，例如 `天道酬勤`。
2.  **粉丝数量**：填写您查看到的粉丝数，例如 `39917`。
3.  **活动邀请人**：根据您加入计划的渠道，填写指定的邀请人姓名，例如 `张良` 或 `张良老师`。
4.  **实名信息**：填写您本人的真实姓名，例如 `周建国`。
5.  **结算手机号**：填写您用于接收验证码和后续结算的手机号码，例如 `13812345678`。请注意，号码中间不要有空格。

所有信息填写完毕后，请仔细检查一遍。

![](img/c336683f5ca347183db52898d2e16888_13.png)

### 第六步：提交申请
确认所有信息无误后，点击页面上的蓝色“提交”按钮。系统可能会弹出确认提示，点击“确认”即可完成提交。

![](img/c336683f5ca347183db52898d2e16888_14.png)

成功提交后，页面会显示“已成功加入我们的创作者分成计划”之类的提示。



![](img/c336683f5ca347183db52898d2e16888_16.png)

## 后续操作与素材支持
提交成功后，就可以开始发布内容了。为了帮助您快速起步，导师通常会提供一些已处理好的视频素材。例如，导师可能会将素材按顺序编号（如`1、2、3`）后发给您，您只需直接发布即可。

![](img/c336683f5ca347183db52898d2e16888_18.png)

![](img/c336683f5ca347183db52898d2e16888_20.png)

这些素材来源于拥有正规版权的素材库，涵盖了影视、动漫等多种类型，例如《还珠格格》、《我不是药神》、《大话西游》等经典作品，确保您可以安全合规地进行创作。



![](img/c336683f5ca347183db52898d2e16888_22.png)

## 总结
本节课我们一起学习了填写视频号创作者计划邀请函的完整流程。关键点在于：**打开微信 -> 进入发现页 -> 找到视频号 -> 查看个人主页获取昵称和粉丝数 -> 准确填写五项信息 -> 最终提交**。请确保每一步都操作正确，信息填写真实准确，这样才能顺利开启您的视频号运营之旅。
