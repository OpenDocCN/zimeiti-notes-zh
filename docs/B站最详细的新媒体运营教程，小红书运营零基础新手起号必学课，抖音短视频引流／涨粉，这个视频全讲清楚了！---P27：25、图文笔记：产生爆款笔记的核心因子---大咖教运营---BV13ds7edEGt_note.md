## 1. 笔记收录 📥

![](img/89ddafbefac16b10581bc8fd13cd2714_28.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_30.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_32.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_34.png)

笔记收录是笔记获得平台推荐的基础。它指的是笔记发布后，能否通过搜索完整标题被找到。如果搜索不到，说明笔记可能未被收录，这会影响后续的流量分发。

![](img/89ddafbefac16b10581bc8fd13cd2714_36.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_38.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_40.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_42.png)

当前主流平台（如小红书、抖音）普遍采用“千人千面”的分发算法。笔记发布后，平台会评估其原创度、质量等，并先推送给一部分“公域流量”进行测试。

![](img/89ddafbefac16b10581bc8fd13cd2714_44.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_46.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_48.png)

**笔记收录的判定流程可以简化为：**
```
笔记发布 → 平台质量评估 → 进入初级流量池测试 → 数据表现良好 → 正式收录（可被搜索到）→ 可能参与关键词排名
```
如果笔记在初级流量池表现不佳，则可能停止推荐，也不会被收录。

![](img/89ddafbefac16b10581bc8fd13cd2714_50.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_52.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_54.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_56.png)

影响笔记收录的主要有以下几点：

![](img/89ddafbefac16b10581bc8fd13cd2714_58.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_60.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_62.png)

以下是影响笔记收录的具体因素：

![](img/89ddafbefac16b10581bc8fd13cd2714_64.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_66.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_68.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_70.png)

*   **原创度：** 平台会检测笔记内容与其他内容的重复度。拥有独特观点、高价值的原创内容更容易被收录。
*   **规避违规词汇：** 小红书平台审核严格。笔记中若包含平台禁止的词汇（如涉及玄学、伪科学、封建迷信等），即使审核通过，也可能被限流或仅自己可见。
*   **初始数据反馈：** 笔记进入初级流量池后，用户的即时互动数据是关键。
    *   **点击率：** 公式可理解为 `点击率 = 点击量 / 曝光量`。低点击率说明封面或标题吸引力不足。
    *   **停留时长：** 用户在笔记页面的平均停留时间。停留时间短（如仅3-5秒）通常意味着内容不吸引人；停留30秒以上则说明内容有吸引力。
    *   **互动率：** 包括点赞、收藏、评论、转发、关注等行为。积极的互动数据会向平台发出“内容优质”的信号。

![](img/89ddafbefac16b10581bc8fd13cd2714_74.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_76.png)

## 2. 笔记权重分 ⚖️

![](img/89ddafbefac16b10581bc8fd13cd2714_78.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_80.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_82.png)

笔记权重分是平台对单篇笔记质量的综合评分，它决定了笔记能获得多少初始流量。

![](img/89ddafbefac16b10581bc8fd13cd2714_84.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_86.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_88.png)

笔记权重分主要受两方面影响：

![](img/89ddafbefac16b10581bc8fd13cd2714_90.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_92.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_94.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_96.png)

以下是影响笔记权重分的两个核心：

![](img/89ddafbefac16b10581bc8fd13cd2714_98.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_100.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_102.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_104.png)

*   **账号权重：** 账号的长期运营表现（如历史内容质量、粉丝互动、违规记录）会影响其发布笔记的初始权重分。这也是为什么粉丝多、互动好的账号，新笔记起步流量通常更高。
*   **实时数据反馈：** 如上节所述，笔记发布后的点击率、停留时长、互动率等数据会动态影响其权重分。数据越好，权重分越高，获得的推荐流量就越多。

![](img/89ddafbefac16b10581bc8fd13cd2714_106.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_108.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_110.png)

**简单来说：** `笔记初始权重分 = f(账号历史权重， 笔记即时数据)`。运营好账号是基础，做好每篇笔记的数据是提升的关键。

![](img/89ddafbefac16b10581bc8fd13cd2714_112.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_114.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_116.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_118.png)

## 4. 需求匹配度 🎯

![](img/89ddafbefac16b10581bc8fd13cd2714_157.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_159.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_161.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_163.png)

需求匹配度指笔记内容与目标用户当前最迫切需求的吻合程度。匹配度越高，用户点击和深度互动的意愿就越强。

![](img/89ddafbefac16b10581bc8fd13cd2714_165.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_167.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_169.png)

提升需求匹配度，可以从以下两方面入手：

![](img/89ddafbefac16b10581bc8fd13cd2714_171.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_173.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_175.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_177.png)

以下是实现高需求匹配的两个方法：

![](img/89ddafbefac16b10581bc8fd13cd2714_179.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_181.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_183.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_185.png)

*   **精准选题：** 基于关键词调研，优先创作用户搜索量最大、最关心的痛点话题。例如，在装修领域，“小户型装修避坑指南”比“装修风格大全”更能匹配特定用户的即时需求。
*   **内容直给，减少废话：** 避免在文案中使用“我花了很久准备”、“一定要看到最后”等铺垫性话语。直接在封面、标题和正文开头就给出用户最关心的干货信息，快速满足其需求。

![](img/89ddafbefac16b10581bc8fd13cd2714_187.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_189.png)

![](img/89ddafbefac16b10581bc8fd13cd2714_191.png)

本节课中我们一起学习了影响小红书笔记成为爆款的四个核心因子：**笔记收录**是基础门槛，**笔记权重分**决定了起跑线，**封面设计**影响了点击率，而**需求匹配度**则决定了用户的深度参与。将这些因素系统性地运用到你的笔记创作中，才能有效提升爆款概率。
