在本节课中，我们将学习小红书商品笔记制作中至关重要的一环——封面设计。封面是笔记的第一印象，直接决定了笔记的点击率和曝光效果。我们将系统地讲解封面的尺寸、设计原则、类型、视频封面以及相关注意事项，帮助你打造高吸引力的商品笔记。

### 封面尺寸 📐
首先了解一下封面的尺寸。

对于图文类内容，小红书官方推荐的尺寸比例有竖屏 **3:4**、正方形 **1:1** 和横屏 **4:3** 三种样式。

商品笔记一定要选择 **3:4** 的比例，包括其他所有笔记，尽量不要用 **1:1** 或 **4:3** 的横屏方式。小红书是放松心情的娱乐平台。用户点进横屏笔记后需要翻转手机，观看兴趣会减少50%。就像在抖音看到横屏视频，评论区常出现“这么多年了，还没人敢让我把手机横着放”这类话。因此，做小红书时，所有图片比例都应选择 **3:4**，其他比例不考虑。

### 封面设计原则 🎯
接下来是封面设计原则。封面设计原则包含以下几点：

以下是三个核心的封面设计原则：
1.  **图片清晰度高**：照片或截图必须使用高清设备拍摄。模糊的图片效果差，就像看电视时有标清、高清、超清选项。越高清的图片，在小红书上效果越好。
2.  **风格一致**：所发的所有笔记，其文案模板和图片风格要保持一致，只是内容不同。风格一致有助于系统快速识别和分发内容，从而匹配更精准的人群。前提是店铺人群画像已做好。统一的风格也能让系统反应更快，避免因识别延迟错过最佳曝光时间。
3.  **突出重点**：确保图片有视觉焦点，能激发用户的点击欲望。这取决于图片的拍摄效果和文案描述的配合。就像刷小红书时，被好看的女装、有趣的文案吸引而点击，重点突出是关键。封面设计不好，后续努力都是白费。

### 封面类型 🖼️
为什么封面还要分类型？封面类型对应不同的人群和内容特点。

以下是五种主要的封面类型及其适用场景：
1.  **高颜值单图**：指画面清晰、美观的单张图片，可有模特也可无模特。比较适合美妆教程、家居装修、穿搭、产品介绍等类目的笔记。
2.  **图片拼贴/抠图拼贴**：将多张图片抠出后拼接组合在一起，可在拼图上压字。比较适合旅游攻略、美食等类目，网络博主常用此方式做宣传。
3.  **海报大片**：具有海报质感的图片，包含主题、副标题等元素。比较适合旅游、生活方式、知识学习类笔记，是一种宣传方式。
4.  **对比图**：通过使用前后或不同产品间的对比来展示效果。比较适合口红、指甲油、饰品、服装等实用性产品，能直观展示商品优势。
5.  **纯文字封面**：以文字为主的封面，强调标题和核心信息。比较适合知识分享、干货教程类内容，能直接传递硬核信息。

封面类型需根据自己选择的类目和产品进行选择，效果各不相同。

### 视频封面 🎥
视频封面之前提到过，它更适合商品笔记。

视频封面无需额外添加引导性文字或图案，只需将整个视频从头到尾完整拍下来。但第一印象的亮度一定要足够，视频整体效果要好。可以多拍几遍选择最好的。如果觉得麻烦，不如直接使用图文封面。

视频封面比较适用于产品介绍和穿搭展示类内容。视频拍好后，只需保证格式正确即可，无需过多修改，保持原创视频的真实性。

视频内容的尺寸比例依然是 **3:4**（竖版）。小红书虽然提供 **4:3**、**16:9**、全屏等选项，但封面尺寸通常只有横板和竖版，我们一般选择竖版 **3:4**，其他不考虑。

---

### 注意事项 ⚠️
最后是封面设计的注意事项。

以下是三个关键的注意事项：
1.  **确保图片尺寸适当**：避免图片过大导致过度压缩，影响清晰度。虽然小红书后台目前还能承受，但过大的图片仍可能造成轻微加载延迟。
2.  **保持标题风格与位置一致**：标题在图片或文字上的样式（如横条或竖条）应基于账号内容确定，并保持所有笔记风格统一。不要前后样式不一致，让用户感到困惑。
3.  **满足像素要求**：封面图的像素要求为 **800 x 320** 像素，像素密度应在 **150 到 300 PPI** 之间。
