# 微信视频号运营从0~1全攻略：P38：4.如何扒别人直播间话术

在本节课中，我们将学习一个实用的技巧：如何获取并借鉴其他成功直播间的话术。这对于想要快速上手直播带货的新手来说，是一个高效的学习方法。

## 概述：为什么要学习直播间话术

上一节我们介绍了视频内容的创作，本节中我们来看看直播的核心——话术。优秀的话术能有效提升直播间转化率。当你发现某个直播间销量很好，并且其产品你可以低成本复制时，学习他的话术就成为关键。例如，季节性产品（如年货）的话术具有时效性，掌握方法后，你可以随时获取并优化使用。

## 核心工具介绍

![](img/0b88b5d4afa6fbcef1a5cb2093726f12_1.png)

我们需要使用一个工具来实现话术录制与转写：**飞书妙计**。它是飞书办公套件中的一个功能，可以实时录音并转写成文字。

![](img/0b88b5d4afa6fbcef1a5cb2093726f12_3.png)

以下是使用飞书妙计的具体步骤：

1.  **下载并打开飞书App**。
2.  在应用底部找到并点击 **“工作台”**。
3.  在工作台页面中，搜索并找到 **“飞书妙计”** 功能。

## 操作步骤详解

找到工具后，就可以开始录制目标直播间的话术了。

以下是具体的操作流程：

1.  在飞书妙计界面，点击右下角的 **录音按钮**，开始录音。
2.  切换到微信，打开你想要学习和模仿的**目标直播间**。
3.  让直播间声音播放，飞书妙计会**实时录制并转写**所有的语音内容。
4.  建议录制**约15分钟**，这足以覆盖主播介绍产品、互动和逼单的关键话术。
5.  录制完成后，在飞书妙计中点击 **“完成”**。

## 话术导出与整理

录制转写完成后，我们需要将文字内容导出以便整理和学习。

以下是导出与整理的步骤：

1.  在飞书妙计中打开录制好的文件。
2.  点击 **“分享”** 按钮，选择生成链接。建议选择 **“可编辑”** 模式，方便后续修改。
3.  **复制**生成的链接。
4.  将链接发送到**微信的文件传输助手或任意聊天窗口**并打开。
5.  打开后，页面会显示整理好的**文字稿**，你可以全选并复制所有内容。

获取文字稿后，你需要进行提炼和改编，形成适合自己的话术脚本。可以将最终的话术写在纸上或存入手机备忘录，在直播时如果忘记，可以随时查看参考。

## 总结

![](img/0b88b5d4afa6fbcef1a5cb2093726f12_5.png)

本节课我们一起学习了如何利用“飞书妙计”工具来获取他人直播间的优秀话术。关键步骤是：**录制 -> 转写 -> 导出 -> 提炼改编**。掌握这个方法，你就能快速分析竞争对手的直播策略，并吸收其精华，形成自己的直播脚本，从而更自信地开启你的直播带货之旅。
