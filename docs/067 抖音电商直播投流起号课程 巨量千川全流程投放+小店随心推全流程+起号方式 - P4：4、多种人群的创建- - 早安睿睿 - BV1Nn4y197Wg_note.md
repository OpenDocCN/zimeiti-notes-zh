## 核心原则与案例背景

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_52.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_54.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_56.png)

在开始具体搭建前，需要明确一个核心原则：**莱卡投放的关键在于精准度和垂直度，而非关键词数量的多少。** 投放10个精准词的效果远优于投放100个不相关的词。核心公式可以概括为：

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_58.png)

**有效莱卡 = 垂直类目 + 精准产品词 + 关联产品词**

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_62.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_64.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_66.png)

### 组合逻辑与搭建方法

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_88.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_90.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_92.png)

这种组合的核心是 **“行为兴趣双垂直”**。即行为定向和兴趣定向都围绕最核心的类目和产品词展开。

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_94.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_96.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_98.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_100.png)

以下是高精准组合的两种具体搭建方式：

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_102.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_104.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_106.png)

1.  **方式一：行为兴趣逻辑一致**
    *   **行为定向**：主类目（如“食品生鲜”）+ 精准产品词（如“牛肉”、“牛排”）+ 关联产品词（如“牛筋”、“牛肚”）。
    *   **兴趣定向**：采用与行为定向完全相同的逻辑和关键词进行搭建。
    *   **代码示例（逻辑）**：
        ```python
        # 行为定向
        行为关键词 = [“食品生鲜”, “牛肉”, “牛排”, “牛筋”, “牛肚”]
        # 兴趣定向
        兴趣关键词 = [“食品生鲜”, “牛肉”, “牛排”, “牛筋”, “牛肚”]
        ```

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_108.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_110.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_112.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_114.png)

2.  **方式二：行为主导，兴趣仅放主类目**
    *   **行为定向**：主类目（如“食品生鲜”）+ 精准产品词（如“牛肉”、“牛排”）+ 关联产品词。
    *   **兴趣定向**：仅放置一个最核心的主类目（如“食品生鲜”或“美食”）。
    *   **代码示例（逻辑）**：
        ```python
        # 行为定向
        行为关键词 = [“食品生鲜”, “牛肉”, “牛排”, “牛筋”]
        # 兴趣定向
        兴趣关键词 = [“食品生鲜”] # 仅保留最核心大类
        ```

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_116.png)

**过渡**：高精准组合能快速为直播间打上标签，但覆盖人群相对较窄。当我们需要探索更多潜在客户，或产品具有爆品潜力时，可以采用另一种思路。

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_118.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_120.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_122.png)

## 2. 高潜组合 🚀

### 组合逻辑与搭建方法

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_138.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_140.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_142.png)

高潜组合的核心是 **“兴趣主导”**，将主要人群压力放在兴趣定向，行为定向则做简化。

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_144.png)

以下是高潜组合的两种具体搭建方式：

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_146.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_148.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_150.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_152.png)

1.  **方式一：行为仅放产品词，兴趣全量覆盖**
    *   **行为定向**：**不放主类目**，只放精准产品词（如“牛肉”、“牛排”）和少量关联词。
    *   **兴趣定向**：主类目 + 精准产品词 + 关联产品词（全面覆盖）。
    *   **代码示例（逻辑）**：
        ```python
        # 行为定向 (简化)
        行为关键词 = [“牛肉”, “牛排”]
        # 兴趣定向 (全面)
        兴趣关键词 = [“食品生鲜”, “牛肉”, “牛排”, “海鲜”, “鸡肉”]
        ```

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_154.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_156.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_158.png)

2.  **方式二：行为仅放主类目，兴趣全量覆盖**
    *   **行为定向**：仅放置一个主类目（如“食品生鲜”），不放任何产品关键词。
    *   **兴趣定向**：主类目 + 精准产品词 + 关联产品词（全面覆盖）。
    *   **代码示例（逻辑）**：
        ```python
        # 行为定向 (极简)
        行为关键词 = [“食品生鲜”]
        # 兴趣定向 (全面)
        兴趣关键词 = [“食品生鲜”, “牛肉”, “牛排”, “海鲜”, “调味料”]
        ```

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_160.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_162.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_164.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_166.png)

**过渡**：无论是高精准还是高潜组合，都保持了较高的垂直性。但对于一些受众较窄的小众品类，或者账号进入稳定期后需要扩大流量池时，我们就需要“扩人群”组合。

## 3. 扩人群组合 📈

### 组合逻辑与搭建方法

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_194.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_196.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_198.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_200.png)

这种组合下，行为定向保持垂直精准，兴趣定向则加入更多相关大类目，以触达更广泛的相关兴趣人群。

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_202.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_204.png)

以下是扩人群组合的两种具体搭建方式：

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_206.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_208.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_210.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_212.png)

1.  **方式一：行为垂直，兴趣拓宽**
    *   **行为定向**：主类目 + 精准产品词 + 关联产品词（垂直搭建）。
    *   **兴趣定向**：在垂直词的基础上，增加相关大类目。例如，卖牛肉的账号，兴趣定向可以加入“男装”（购买者多为男性）、“生活家电”、“日用百货”等。
    *   **代码示例（逻辑）**：
        ```python
        # 行为定向 (垂直)
        行为关键词 = [“食品生鲜”, “牛肉”, “牛排”]
        # 兴趣定向 (拓宽)
        兴趣关键词 = [“食品生鲜”, “牛肉”, “男装”, “生活家电”, “日用百货”]
        ```

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_214.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_216.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_218.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_220.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_222.png)

2.  **方式二：行为简化，兴趣大幅拓宽**
    *   **行为定向**：参考高潜组合的方式一或方式二，进行简化（只放产品词或只放主类目）。
    *   **兴趣定向**：大幅拓宽，加入多个相关或潜在人群聚集的大类目。

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_224.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_226.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_228.png)

**过渡**：以上六种组合方式（高精准2种、高潜2种、扩人群2种）已能覆盖绝大多数起号与投放场景。对于极少数的特殊情况，还有一种非常规方法——半通投。

## 4. 半通投（谨慎使用）⚠️

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_234.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_236.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_238.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_240.png)

半通投是一种激进的人群探索方式，**仅建议用于视频带货或账号非常成熟、希望突破流量瓶颈时测试使用**，直播间直投慎用。

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_242.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_244.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_246.png)

### 操作方法

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_248.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_250.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_252.png)

1.  在行为定向和兴趣定向中，**全选所有一级类目**。
2.  将与产品**完全无关**的类目手动排除。例如，卖女装的账号可以排除“金融”、“交通”、“户外运动”等。
3.  核心公式为：**半通投 = 全选类目 - 无关类目**

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_254.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_256.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_258.png)

这种方法相当于在大的流量池里，通过排除法过滤掉明显不相关的人群，让系统去探索剩余人群中的潜在客户，成功率不稳定。

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_260.png)

![](img/d10ed898bd0c5b2e29bbf93e1de4c606_262.png)

本节课中我们一起学习了巨量千川莱卡定向的多种人群组合创建方法。我们从最稳妥的**高精准组合**开始，学习了如何锁定核心人群；接着探讨了侧重挖掘潜在客户的**高潜组合**；然后讲解了用于拓宽流量边界的**扩人群组合**；最后了解了需谨慎使用的**半通投**测试方法。
