# TikTok跨境电商运营教程：P25：24.快速挖掘TikTok大卖产品 💡

在本节课中，我们将学习如何在TikTok上快速挖掘具有高潜力的“大卖”产品。我们将详细分析选择产品时必须考虑的六个核心要点，并介绍一些实用的国内货源平台，帮助初学者迈出选品的第一步。

上一节我们介绍了选品的重要性，本节中我们来看看具体如何筛选出有潜力的产品。选择一款合适的产品是成功的基础，需要综合评估多个维度。

以下是选择大卖产品时必须符合的六个核心要点：

**1. 市场容量大**
市场容量及销量是首要考虑因素。量变会产生质变。如果产品市场需求不大、销量有限，则很难销售成功。虽然不排除有新产品能脱颖而出，但概率很小，通常需要大量达人推广或广告投入，否则产品需要长时间沉淀才能起量。

**2. 具备未来趋势**
需要看好产品后期的潜在发展能力。如果预测产品在未来几个月甚至几年内销量会下滑或无人购买，则应避免选择。例如，评估一款杯子在未来两年的销售潜力。

**3. 利润率要高**
我们不能做亏损的生意。利润要高，利润率也要高。例如，卖出一件产品最好能有五六十甚至七八十的收益。通常我们选品时，利润率应力求达到成本的3到5倍。高客单价产品在销售时用户会进行更多对比，因此定价要适中，不能过高或过低，但核心是保证不亏损。

**4. 竞争程度小**
竞争小主要体现在卖家垄断程度低。即该产品目前的卖家较少，头部品牌或商家尚未介入。这样你可以用大量账号推广，抢占市场份额，甚至形成垄断。如果一款产品已有众多卖家和品牌方在推广，新卖家很难跻身其中。

**5. 退换货率低**
涉及尺寸、尺码、颜色多样的产品（如衣服、包包）不建议销售，因为退换货率较高。海外用户购买后若发现尺码不符，通常会要求退换，这将导致你需要承担额外的物流成本并面临退款风险。最好销售款式、颜色单一的产品，让用户“想换也没得换”。

**6. 入门门槛低**
门槛低主要包含以下三个方面：
*   **体积小**：方便运输，降低物流成本，也易于打包。
*   **重量轻**：同样是出于控制物流成本考虑。
*   **投入成本低**：对于新手而言，前期应选择采购价低的产品。例如，国内采购价仅几元人民币的产品，在海外可以销售19或29美金，赚取高额差价。这有助于新手起步，之后可以再过渡到高客单价产品。
