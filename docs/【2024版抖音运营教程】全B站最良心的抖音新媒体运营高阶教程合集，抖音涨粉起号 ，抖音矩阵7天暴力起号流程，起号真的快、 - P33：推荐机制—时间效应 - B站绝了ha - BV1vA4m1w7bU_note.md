# 抖音运营教程：P33：推荐机制—时间效应 📅

![](img/fd60b4a7e4e0cefaf11eb17080831934_1.png)

在本节课中，我们将要学习抖音推荐机制中的第三个重要概念——“时间效应”。我们将了解它的定义、运作原理，以及它与快手等其他平台推荐机制的区别。

上一节我们介绍了基础推荐和叠加推荐，本节中我们来看看“时间效应”。

## 什么是时间效应？

时间效应是抖音推荐机制的一部分。它指的是一个视频在发布后，可能不会立即获得大量推荐或播放量，但在未来某个时间点（例如一周或一个月后），其播放量可能会突然上涨。

## 时间效应如何运作？

为了理解时间效应，我们需要了解抖音的视频库系统。

*   **视频库的存在**：抖音会将所有用户发布的视频存储在自己的视频库中。
*   **初次推荐**：视频发布后，会根据其初始表现（如内容质量、是否蹭热点）进入相应的流量池。
*   **二次推荐的机会**：即使视频初次表现不佳，它依然存在于视频库中。当未来出现新的热点或话题时，抖音的算法会重新扫描视频库，寻找与当前热点相匹配的“库存视频”，并对其进行二次推荐。

以下是时间效应运作流程的简化描述：
1.  用户发布视频。
2.  视频进入抖音视频库。
3.  视频根据初始数据获得初次推荐。
4.  未来出现相关热点。
5.  算法从视频库中匹配到该视频。
6.  视频获得基于新热点的二次推荐，播放量上涨。

**举例说明**：假设你发布了一个关于明星A的视频，当时并未引起关注。一段时间后，明星A因为某事件（如发布新作品、出现新闻）热度飙升。此时，抖音算法可能会从视频库中找到你之前发布的那个相关视频，并再次推荐给对明星A感兴趣的用户，从而使该视频的播放量增加。

## 时间效应的来源与价值

这个概念最初源自贴吧的“挖坟”或“拉坟”行为，即让旧帖子重新获得关注。抖音将其算法化：通过识别热点，重新挖掘视频库中的旧内容进行推荐。

因此，用心制作每一个视频非常重要。即使它没有立即“火起来”，只要内容优质且具有潜在关联性，未来仍有很大机会通过时间效应被再次推荐。

## 与快手机制的对比

需要强调的是，时间效应是抖音平台的特色机制。另一个主流短视频平台——快手，采用的是不同的策略。

快手的推荐机制被称为 **“择新去旧”机制**。
*   **核心规则**：快手倾向于推荐新发布的视频。一个视频在发布一周或一个月后，通常就不再会获得系统推荐，无论之后是否出现相关热点。
*   **与抖音的区别**：这是抖音（**时间效应**）和快手（**择新去旧**）在推荐机制上一个非常重大的区别。抖音给旧视频提供了“复活”的机会，而快手更专注于推送新鲜内容。
