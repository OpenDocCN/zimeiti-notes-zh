## 概述

上一节我们介绍了小红书的基本规则框架，本节中我们来看看笔记被系统推荐的具体逻辑和操作红线。

## 一、被推核心逻辑 🔍
被推核心逻辑指的是小红书系统决定是否以及如何推广一篇笔记的内部规则。简单来说，就是系统根据哪些因素来决定额外给我们笔记流量。

以下是影响笔记被推荐的四个核心因素：

1.  **利益相关认证**：与小红书存在利益关联的品牌方或权威第三方认证的账号，会获得基础权重加成。
    *   **公式**：`账号基础权重 = 普通账号权重 + 认证附加权重`
    *   例如，普通账号笔记基础展现可能在1000-3000，而有认证的账号可能达到2000-6000。

2.  **付费推广**：使用小红书的付费推广工具（如薯条、信息流广告）可以为笔记增加权重。但**不建议新手在前期操作**，因为成本高、竞争激烈，更适合中后期突破瓶颈时使用。

![](img/d74f9044e850d24bbe19956aac4188cc_5.png)

3.  **即请私用**：参与小红书官方发起的品牌活动或任务。官方对你的账号和内容有信心，才会给予流量扶持。

![](img/d74f9044e850d24bbe19956aac4188cc_7.png)

4.  **晒单抽奖**：参与官方或品牌方的晒单、抽奖类活动。这同样是获得官方认可和额外曝光的一种方式。

**核心理解**：以上四点是为笔记流量提供的“额外加成”。对于大多数新手而言，重点应放在做好内容和遵守社区规则上。

## 二、推广规则与注意事项 ⚠️
了解被推逻辑后，我们需要掌握发布推广内容时的具体规则。以下是发布笔记时必须注意的三个要点。

### （一）发布须知
发布的内容必须符合《小红书社区行为规范》。系统会自动检测，一旦触犯规则，笔记将无法获得展示或直接被限制。

需要特别注意：**避开需要长期使用才能体现功效的产品宣传**。例如，“使用一个月见效”的减肥药、护肤品等。这类涉及“虚假宣传”或“功效承诺”的内容极易被限流。

### （二）注意事项
以下是操作中必须规避的雷区：

*   **禁止数据造假**：严禁刷粉丝、刷点赞、刷阅读量等任何形式的人工数据干预。小红书以“内容为王”，系统给予的初始曝光权重已足够高。人为造假不仅效果甚微，且极易被系统识别，导致账号被处罚。
    *   **代码逻辑**：
        ```python
        if 检测到数据异常波动(如点赞率飙升):
            系统标记账号为“可疑”
            if 确认为人工刷量:
                执行处罚(限流、降权、封号)
        ```

*   **控制发布频率**：不要连续发布多篇推广笔记。
    *   **建议**：一个账号每天发布1-2篇精品笔记即可。单品推广每日最多2篇，整个账号每日最多5篇。发布过多会导致权重分散，甚至影响前几篇优质笔记的流量。

### （三）违规处罚
如果违反规则，将面临以下处罚：

1.  **官方警告**：收到站内信或系统通知警告，笔记会被**限制曝光**（例如从2000曝光降至200）。
2.  **数据异常监测**：账号数据（如粉丝数、互动量）月度环比增长若**超过20%**，会触发系统重点监测。如果被判定为异常增长（非自然爆款），将面临处罚。因此，稳定、自然的增长是关键。

*   **被推逻辑**：主要受**利益认证、付费推广、官方活动**等因素影响，它们提供的是流量加成。
