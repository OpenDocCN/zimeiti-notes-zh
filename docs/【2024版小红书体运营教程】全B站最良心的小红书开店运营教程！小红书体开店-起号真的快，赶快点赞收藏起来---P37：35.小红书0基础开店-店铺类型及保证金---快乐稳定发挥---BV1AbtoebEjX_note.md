# 小红书开店运营教程：P37：35. 小红书0基础开店-店铺类型及保证金 💰

在本节课中，我们将要学习小红书电商平台上的店铺类型体系以及相关的保证金规则。了解这些基础知识，有助于你根据自身情况选择合适的店铺类型，并明确资金准备要求。

![](img/c0a9e96fa28ce3f2cf7193b5fb0e03ae_3.png)

上一节我们介绍了开店的基础流程，本节中我们来看看小红书平台具体有哪些店铺类型。

## 店铺类型概述

小红书平台为不同经营主体提供了多种店铺类型，主要可分为以下几类。以下是所有店铺类型的列表：

*   **个人店**
*   **个体工商户店**
*   **普通企业店**
*   **专卖店**
*   **旗舰店**
*   **卖场型旗舰店**

接下来，我们将逐一详细解析每种店铺的特点和要求。

## 各类店铺详解

### 1. 个人店

个人店适用于个人用户，类似于线下摆地摊的经营模式。适合刚毕业或想尝试小规模创业的用户。

![](img/c0a9e96fa28ce3f2cf7193b5fb0e03ae_5.png)

*   **经营特点**：可经营限定类目下的零星、小额商品，例如首饰、服装、手机壳等。类似于“杂货铺”，可销售多个品牌商品。
*   **入驻要求**：非常简单，仅需个人身份证即可。
*   **店铺命名格式**：系统自动生成，格式为“专业号名称 + 店名”。
*   **保证金**：涉及基础保证金和浮动保证金，二者取金额高者缴纳。个人店的基础保证金通常为1000元。

![](img/c0a9e96fa28ce3f2cf7193b5fb0e03ae_7.png)

### 2. 个体工商户店

个体工商户店与个人店的主要区别在于入驻时需要提供营业执照。

*   **与个人店的区别**：仅入驻要求不同，需提供个体工商户营业执照。对品牌授权无要求。
*   **其他方面**：如经营特点、保证金规则等，与个人店基本一致。

![](img/c0a9e96fa28ce3f2cf7193b5fb0e03ae_9.png)

![](img/c0a9e96fa28ce3f2cf7193b5fb0e03ae_11.png)

### 3. 普通企业店

普通企业店要求申请主体为企业。店铺权重通常与入驻要求的复杂程度相关。

*   **入驻要求**：申请主体必须为企业，需持有企业营业执照。
*   **经营建议**：对于新手，初期可从个人店开始测试产品和市场潜力，无需一开始就投入过高成本申请复杂店铺。

### 4. 专卖店、旗舰店与卖场型旗舰店

这三种店铺类型主要面向品牌方或具有品牌授权的经营者。

*   **专卖店**：需营业执照及品牌资质（自有品牌或非独占授权品牌）。
*   **旗舰店**：需营业执照及品牌授权书（销售他人品牌商品时需获得授权）。
*   **卖场型旗舰店**：要求具有零售资质，并可经营多个品牌。但在小红书平台较为少见，因为平台用户以女性为主，与大型卖场的商品品类匹配度不高。这反而为中小卖家提供了竞争压力相对较小的生存空间。

![](img/c0a9e96fa28ce3f2cf7193b5fb0e03ae_13.png)

以上便是小红书主要店铺类型的区分。了解类型后，我们还需要关注一个关键环节：保证金。

## 保证金规则详解

![](img/c0a9e96fa28ce3f2cf7193b5fb0e03ae_19.png)

保证金分为**基础保证金**、**类目保证金**和**浮动保证金**。其中，**浮动保证金**是需要重点理解的部分。

### 基础与类目保证金

*   **基础保证金**：通常为1000元，是开店的基础资金门槛。
*   **类目保证金**：根据经营的特殊类目（如普通企业店特殊类目、跨境类目）不同，可能会有额外的固定保证金要求。具体金额需参考平台官方发布的类目保证金表。

### 浮动保证金（核心规则）

浮动保证金根据店铺的月成交金额动态调整，旨在保障交易安全。规则如下：

以下是浮动保证金的计算阶梯：

*   **月成交金额 ≤ 5万元**：浮动保证金为 **0元**。（但需已缴纳1000元基础保证金）
*   **5万元 < 月成交金额 ≤ 10万元**：浮动保证金为 **3000元**。
*   **10万元 < 月成交金额 ≤ 50万元**：浮动保证金为 **5000元**。
*   **月成交金额 > 50万元**：浮动保证金为 **20000元**。

![](img/c0a9e96fa28ce3f2cf7193b5fb0e03ae_21.png)

**重要提示**：
平台在商家提现时会校验保证金账户总额（基础+类目+浮动）是否充足。如果账户余额低于当前成交额对应的浮动保证金要求，将无法成功提现。例如，若只缴纳了1000元基础保证金，当店铺月成交额超过5万元时，由于浮动保证金要求变为3000元而账户不足，将无法提现。因此，务必随时关注保证金是否充足。
