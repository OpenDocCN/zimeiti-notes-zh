# 小红书手机端自动化工具-【薯摸鱼】-批量取消关注：P1：基础介绍与准备工作

在本节课中，我们将要学习如何使用一款名为“薯摸鱼”的工具，来实现小红书手机端的批量取消关注操作。我们将从工具的基础介绍和必要的准备工作开始。

## 概述

![](img/3a13826170be1ec88baed449a91c4350_0.png)

“薯摸鱼”是一款针对小红书手机端设计的自动化工具，其主要功能之一是帮助用户批量取消关注。通过模拟用户手动操作，它可以高效地处理大量重复性任务，节省用户的时间和精力。

上一节我们介绍了课程的整体目标，本节中我们来看看使用“薯摸鱼”工具前需要做哪些准备工作。

## 准备工作

![](img/3a13826170be1ec88baed449a91c4350_1.png)

为了确保“薯摸鱼”工具能够顺利运行并执行批量取消关注的操作，你需要提前完成以下几项准备工作。

以下是完成准备工作所需的步骤列表：

1.  **安装必要的软件环境**：确保你的电脑上已安装Python运行环境。这是运行“薯摸鱼”脚本的基础。
    ```bash
    # 检查Python是否安装
    python --version
    ```

![](img/3a13826170be1ec88baed449a91c4350_2.png)

2.  **获取“薯摸鱼”工具**：你需要下载“薯摸鱼”工具的脚本或安装包。请从可靠的来源获取，例如项目官方页面。

3.  **准备安卓设备与连接**：你需要一部安卓手机，并开启USB调试模式。然后通过USB数据线将手机连接到电脑。
    *   在手机“设置”中找到“开发者选项”。
    *   开启“USB调试”。

4.  **安装ADB工具**：在电脑上安装Android Debug Bridge (ADB) 工具，用于与连接的安卓设备通信。
    ```bash
    # 示例：在命令行中检查ADB是否可用
    adb devices
    ```

![](img/3a13826170be1ec88baed449a91c4350_3.png)

5.  **登录小红书账号**：在已连接的安卓手机上，提前打开并登录你的小红书账号，并确保停留在“我的关注”列表页面附近。

完成以上步骤后，你的基础运行环境就已搭建完毕。接下来，我们就可以进入工具的具体配置与使用阶段了。

## 总结

本节课中我们一起学习了“薯摸鱼”工具的用途，并详细列出了使用前必须完成的五项准备工作，包括安装Python、获取工具、连接安卓设备、安装ADB以及登录账号。这些步骤是后续自动化操作能够成功执行的基础。下一节，我们将开始学习如何配置和运行“薯摸鱼”工具来进行批量取消关注。
