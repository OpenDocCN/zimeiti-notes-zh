# 抖音运营教程：P47：推荐机制—时间效应 ⏰

![](img/cb9663f86b8a78337c455195568926af_1.png)

在本节课中，我们将要学习抖音推荐机制中的第三个重要概念——时间效应。我们将了解什么是时间效应，它如何运作，以及它与其他平台推荐机制的区别。

---

上一节我们介绍了基础推荐和叠加推荐，本节中我们来看看推荐机制中的另一个关键因素：时间效应。

什么是时间效应？我举个例子。你可能遇到过这种情况：发布一个视频后，当天或第二天推荐量很少，播放量不高。但一周甚至一个月后，这个视频的播放量却开始上涨，并能持续一段时间。这就是时间效应在起作用。

为了理解时间效应，我们需要了解抖音的视频库机制。抖音会将所有用户发布的视频存储在自己的视频库中。当你发布视频后，你的作品就存在于这个庞大的视频库里。

如果你的视频在发布初期没有蹭到热点，内容也不算特别优质，它可能就会暂时“沉睡”在视频库中。然而，当未来某个热点事件出现时，抖音的算法会从视频库中寻找与当前热点相匹配的原有库存视频，并进行二次推荐。

例如，假设你发布了一个关于明星A的视频，当时没有火。但一段时间后，明星A因为某事件热度飙升。此时，抖音算法可能会从视频库中找出你之前发布的那个相关视频，让它重新进入推荐流程，从而获得新的播放量。

这种时间效应机制，最初源于贴吧的“挖坟”或“拉坟”概念。其核心是平台通过算法监测热点，并重新从历史视频库中提取合适的内容，将其推入流量池进行再次推荐。

因此，如果你用心制作视频并遵循了正确的步骤，你的作品将有更大几率在未来的某个时间点，因为契合新的热点而获得二次推荐。

这里需要强调一点，时间效应机制是抖音平台特有的。同为短视频平台，快手的推荐机制与此有显著不同。

快手的推荐机制被称为“择新去旧”机制。该机制的核心是平台主要推荐新发布的视频。一个视频发布一周甚至一个月后，系统基本不会再推荐它，无论之后出现任何与之相关的热点。这是抖音与快手在推荐机制上一个重要的区别。

**核心机制对比：**
*   **抖音**：`推荐机制包含“时间效应”，历史视频可能因契合新热点而被二次推荐。`
*   **快手**：`推荐机制为“择新去旧”，基本不推荐旧视频。`

本节课中我们一起学习了抖音推荐机制的“时间效应”。我们明白了你的视频即使发布初期数据不佳，也可能在未来因契合热点而获得二次推荐机会。同时，我们也了解到这与快手“择新去旧”的机制存在根本不同。理解这些机制，有助于我们更全面地看待视频的长期价值，并保持内容创作的动力。
