# 剪映PC版教程：P8：【电脑版】音频功能应用和处理 🎵

![](img/9f13a17787cb6593e5c0d62ba6306ebb_0.png)

在本节课中，我们将学习剪映PC版中音频功能的应用和处理。课程将详细介绍音频面板的各个模块，并通过一个完整的卡点视频案例，演示如何利用这些功能高效地制作节奏感强的视频。

## 音频功能模块详解

上一节我们介绍了剪映的基础界面，本节中我们来看看音频功能的具体模块。选中时间线上的音频素材后，下方会出现五个主要模块。

### 1. 音乐素材库 🎶

![](img/9f13a17787cb6593e5c0d62ba6306ebb_2.png)

![](img/9f13a17787cb6593e5c0d62ba6306ebb_3.png)

音乐素材库相当于一个内置的音乐库，提供了丰富的音乐资源，无需自行下载。

![](img/9f13a17787cb6593e5c0d62ba6306ebb_5.png)

以下是音乐素材库的主要分类：
*   **抖音热门**：包含当前流行的热门音乐。
*   **浪漫/旅行/Vlog**：按场景和风格分类，方便查找。
*   **卡点音乐**：专门为制作卡点视频准备的节奏感强的音乐。
*   **纯音乐/环保/搞笑等**：分类非常细致，涵盖多种用途。

![](img/9f13a17787cb6593e5c0d62ba6306ebb_7.png)

![](img/9f13a17787cb6593e5c0d62ba6306ebb_8.png)

![](img/9f13a17787cb6593e5c0d62ba6306ebb_9.png)

![](img/9f13a17787cb6593e5c0d62ba6306ebb_10.png)

![](img/9f13a17787cb6593e5c0d62ba6306ebb_11.png)

使用方法：点击喜欢的音乐进行下载，然后直接拖拽到时间线上即可使用。

### 2. 音效库 🔊

![](img/9f13a17787cb6593e5c0d62ba6306ebb_13.png)

![](img/9f13a17787cb6593e5c0d62ba6306ebb_14.png)

![](img/9f13a17787cb6593e5c0d62ba6306ebb_15.png)

音效库为视频添加各种声音效果，能让画面更加生动。

以下是音效库的部分分类：
*   **笑声/综艺**：增加娱乐效果。
*   **机械/BGM**：用于特定场景的背景音。
*   **人声转场/打斗**：增强转场或动作戏的冲击力。
*   **动物叫声**：如小猫、布谷鸟叫声等。

剪映集成了大量音效素材，熟悉并善用它们可以极大提升视频制作的效率和质量。

![](img/9f13a17787cb6593e5c0d62ba6306ebb_17.png)

![](img/9f13a17787cb6593e5c0d62ba6306ebb_18.png)

### 3. 音频提取 🎼

![](img/9f13a17787cb6593e5c0d62ba6306ebb_20.png)

音频提取功能可以从视频文件中直接分离出音频。

![](img/9f13a17787cb6593e5c0d62ba6306ebb_21.png)

操作方法是：导入一个包含音频的视频文件，软件会自动将其音频提取出来，生成一个独立的音频素材，供你直接使用。

### 4. 抖音收藏

此功能需要登录你的抖音账号。

![](img/9f13a17787cb6593e5c0d62ba6306ebb_23.png)

登录后，你在抖音APP上收藏的音乐会自动同步到剪映PC版的“抖音收藏”列表中，方便在剪辑时直接调用。

![](img/9f13a17787cb6593e5c0d62ba6306ebb_25.png)

### 5. 链接下载

如果你在抖音上看到喜欢的视频音乐但无法收藏，可以复制该视频的分享链接。

将链接粘贴到“链接下载”的输入框中，软件会自动解析并下载其中的音乐，保存到你的音乐库中。

## 实战案例：制作卡点视频 🎬

了解了音频功能后，本节我们将通过一个具体的卡点视频案例，来实践如何综合运用这些工具。

### 第一步：导入素材与音乐

![](img/9f13a17787cb6593e5c0d62ba6306ebb_27.png)

![](img/9f13a17787cb6593e5c0d62ba6306ebb_29.png)

首先，将案例所需的所有图片素材导入到媒体池。
接着，导入背景音乐。我们可以使用“音频提取”功能，从一个已有的示例视频中提取音乐，并将其拖拽到时间线上。

### 第二步：使用自动踩点功能

![](img/9f13a17787cb6593e5c0d62ba6306ebb_31.png)

![](img/9f13a17787cb6593e5c0d62ba6306ebb_32.png)

卡点视频的关键是让画面切换对准音乐的鼓点。对于节奏感不强的用户，可以使用“自动踩点”功能。

1.  选中时间线上的音频素材。
2.  在工具栏找到 **`自动踩点`** 按钮（图标类似小红旗）。
3.  点击后选择节拍模式（如“节拍2”），软件会自动在音频波形上生成密集的鼓点标记。

这些标记就是画面需要切换的位置，极大简化了手动卡点的过程。

### 第三步：制作开场动画

![](img/9f13a17787cb6593e5c0d62ba6306ebb_34.png)

1.  **添加文字模板**：在“文本”->“文字模板”中，找到一个合适的英文开场模板（如“怦然心动”），拖到时间线开头，并调整其长度与音乐起始部分对齐。修改文字内容为“You know what I think?”。
2.  **添加图片与蒙版**：
    *   导入第一张图片（如花朵），在“画面”设置中将比例调整为 **`3:4`**。
    *   将项目比例也设置为 **`3:4`**，以消除黑边。
    *   为图片添加“复古胶片”分类下的“美高”滤镜。
    *   使用 **`蒙版`** 功能中的“矩形”蒙版，将图片分割成上下平行的三条，并错开它们的出现时间，使其依次在鼓点出现。
3.  **添加中心文字**：在第三条图片出现时，添加文字模板“夏日”系列中的一款，修改内容为“I think”，放置在画面中央。可以适当降低中间图片层的不透明度，使文字更清晰。

![](img/9f13a17787cb6593e5c0d62ba6306ebb_36.png)

### 第四步：快速图片卡点

![](img/9f13a17787cb6593e5c0d62ba6306ebb_38.png)

1.  导入多张图片素材（如小猫、城市风景等），每张图片都调整为 **`3:4`** 比例。
2.  将这些图片依次拖到时间线上，每张图片的时长对准一个音频鼓点标记。
3.  为这些图片统一拖入“美高”滤镜，保证色调一致。

![](img/9f13a17787cb6593e5c0d62ba6306ebb_39.png)

![](img/9f13a17787cb6593e5c0d62ba6306ebb_41.png)

### 第五步：制作彩色边框效果

1.  导入一张竖构图图片（如脚部特写），调整比例为 **`3:4`**。
2.  使用“矩形”蒙版，将图片分割成并排的竖条，复制多层来实现。
3.  为每一层图片添加“画面特效”中的 **`负片偏移`** 效果。
4.  选中“负片偏移”特效，在基础参数中点击调色板，为每一层更改不同的颜色。
5.  在彩色条之间添加文字“You”、“Are”、“Dope”。

### 第六步：制作人物聚焦效果

1.  导入一张人物图片，调整比例。
2.  复制一层，为上层添加“矩形”蒙版，框选出人物主体。
3.  为上层添加“负片偏移”特效并调整颜色，形成人物彩色、背景原图的效果。

### 第七步：制作分屏对比效果

![](img/9f13a17787cb6593e5c0d62ba6306ebb_43.png)

1.  导入一张图片，调整比例。
2.  复制一层，分别为上下两层添加“矩形”蒙版，将画面分成上下两部分。
3.  为上层同时添加“负片偏移”和“黑白胶片”滤镜，形成上下风格对比的效果。

![](img/9f13a17787cb6593e5c0d62ba6306ebb_45.png)

### 第八步：结尾制作与导出

![](img/9f13a17787cb6593e5c0d62ba6306ebb_47.png)

![](img/9f13a17787cb6593e5c0d62ba6306ebb_49.png)

1.  添加最后几张图片作为结尾，并确保色调统一。
2.  在音乐结束部分，添加结尾文字“Like really dope”，可以使用花字并调整描边颜色。
3.  完成剪辑后，点击 **`导出`** 按钮。
4.  设置导出参数：格式为 **`MP4`**，编码为 **`H.264`**，分辨率根据需要选择，帧率可选 **`30`** 或 **`25`**，然后选择保存路径，点击导出即可。

![](img/9f13a17787cb6593e5c0d62ba6306ebb_51.png)
