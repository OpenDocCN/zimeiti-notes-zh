# 新媒体运营：P11：小红书AI人工智能模型辅助定位工具分享 🧠

## 概述
在本节课中，我们将学习如何利用AI人工智能工具来辅助完成小红书账号的定位工作。上一节我们介绍了账号定位的四个思考方向，本节中我们来看看，当自己难以确定方向时，如何借助智能工具进行分析和参考。

## 定位回顾与工具引入
上一节我们探讨了账号定位的四个核心方向：**我是谁**、**我能提供什么价值**、**账号的差异化**以及**内容的可持续性**。同时，我们也强调了领域定位的三个关键点：**自身擅长或了解**、**以用户需求为核心**以及**分析对标账号**。

然而，即便掌握了这些方法，部分同学可能仍对自身的内容方向感到迷茫。为此，本节将介绍一个基于ChatGPT等AI模型开发的辅助工具。这个工具本质上是一个智能聊天模型，它能帮助我们分析和复刻同行的爆款笔记，检测敏感词，甚至根据选题生成具备小红书风格的文案。

以下是该工具的几个核心功能：

*   **爆款笔记分析**：工具能分析出笔记文案中的爆款因素。
*   **爆款复刻（伪原创）**：提取爆款笔记的核心要素，围绕这些要素生成新的创作文案。
*   **敏感词检测**：检测文案中的敏感词，避免因限流导致阅读量异常。
*   **文案生成**：根据提供的选题方向，自动生成符合平台风格的文案。

## AI工具实操：人设定位演示
我们将重点演示如何使用该工具的“人设定位”功能。你只需要按照提示，输入关于自己的几个维度的信息，工具便会生成相应的定位建议。

以下是需要填写的几个信息维度：

1.  **兴趣爱好**
2.  **工作和生活经历**
3.  **专业知识技能**
4.  **可投入的时间**
5.  **目标和期望**

### 操作示例
我们以一个虚构的案例进行演示，输入以下信息：

*   **兴趣爱好**：打游戏、看直播、唱歌。
*   **工作和生活经历**：做过运营、参加过线上游戏比赛、曾带领100人运营团队。
*   **专业知识技能**：新媒体运营、小红书起号、数学能力好。
*   **可投入时间**：每天3小时。
*   **目标期望**：打造个人IP。

## 工具使用建议与总结
分享这个工具的目的是为了帮助那些在定位环节遇到困难的同学。请注意，工具给出的结果是基于你输入的信息，因此输入的信息应尽可能聚焦于你真实了解或有意深耕的垂直领域。

例如，如果你对箱包领域有研究，就应该输入与之相关的兴趣爱好（如收藏包包）、经历（如参观过国际箱包展）和技能（如鉴别包包价值与市场趋势）。针对不同的擅长领域多进行几次分析，从生成的多个结果中筛选出你认为最可行、最适合自己的方向进行深度尝试。
