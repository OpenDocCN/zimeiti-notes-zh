# 新媒体运营：P11：01.9：小红书AI人工智能模型辅助定位工具分享 🧠

在本节课中，我们将学习如何利用AI工具辅助解决小红书账号定位的难题。上一节我们介绍了账号定位的四个思考方向，本节中我们来看看，当自己仍不确定方向时，一个实用的AI工具如何帮助我们进行分析和启发。

## 定位回顾与难点

上一节我们探讨了账号定位的四个核心方向：**我是谁**、**我能提供什么价值**、**账号的差异化**以及**内容的可持续性**。同时，我们也强调了领域定位的三个关键点：**自身擅长或可深入了解**、**以用户价值为核心**以及**分析对标账号**。

然而，即便掌握了这些理论，部分同学可能仍对“具体该做什么内容”感到困惑。这种不确定性是正常的，尤其是对于初学者。

## AI辅助定位工具介绍

![](img/b409d6ac43c36052751441437b1c47f8_1.png)

如果你确实感到迷茫，这里推荐一个非常实用的工具。这是一个基于**ChatGPT**等AI智能模型接口开发的小红书账号辅助工具。

这个工具本质上是一个智能聊天模型，它能帮助我们完成以下核心任务：
*   **分析爆款笔记**：它能解析一篇笔记的文案，识别出其中可能成为爆款的关键因素。
*   **爆款复刻（伪原创）**：工具可以基于分析出的爆款因子，生成新的文案框架，帮助我们进行围绕这些成功元素的再创作。
*   **敏感词检测**：发布内容前进行检测，避免因敏感词导致限流，这是现代新媒体运营的必备步骤。
*   **生成小红书风格文案**：根据你提供的选题方向，自动生成符合平台风格的文案。

本节课，我们将重点演示该工具的“**人设定位**”功能。

## 工具实操演示：生成人设定位

以下是使用“人设定位”功能的具体步骤。你需要向工具提供关于你自己的几个维度的信息。

首先，我们复制工具提供的提示点，并填入相应信息：

1.  **兴趣爱好**：例如，打游戏、看直播、唱歌。
2.  **工作和生活经历**：例如，做过运营、参加过线上游戏比赛、带领过百人团队。
3.  **专业知识技能**：例如，新媒体运营、小红书起号、数学能力。
4.  **可投入时间**：例如，每天3小时。
5.  **目标与期望**：例如，打造个人IP。

将以上信息提交给AI工具后，它会生成一份定位分析报告。例如，针对上述杂乱的信息，工具可能给出“**游戏解说员**”、“**游戏攻略作者**”或“**唱歌达人**”等建议，并附上匹配度、竞争难度及差异化建议。

## 工具使用建议与总结

分享这个工具的目的是，当你在定位环节遇到巨大障碍、经过长时间思考仍无头绪时，可以借助它获得启发。

**请注意**：你不能像演示中那样随意填写杂乱信息。正确的做法是，基于你潜意识中已有倾向的领域进行描述。例如，如果你对包包领域有了解，就应详细填写与之相关的兴趣爱好、经历和技能。工具会根据这些垂直信息生成更具参考价值的报告。

你可以针对多个潜在领域分别进行分析，然后从生成的多个选项中，筛选出那些你认为**竞争难度适中**且**自身更有能力执行**的方向进行深度尝试。
