# 小红书运营教程：P41：39、小红书主体定位（2）- 人群定位详解 🎯

![](img/c30ad768a4c5a701087b1c2f4cb631d6_1.png)

在本节课中，我们将要学习小红书店铺运营中一个至关重要的环节——人群定位。人群定位决定了你的店铺内容会推送给谁，是提升转化率的基础。上一节我们介绍了主体定位的概览，本节中我们将深入探讨如何精准定义你的目标用户群体。

![](img/c30ad768a4c5a701087b1c2f4cb631d6_3.png)

![](img/c30ad768a4c5a701087b1c2f4cb631d6_4.png)

人群定位与账号定位逻辑相似，但更侧重于店铺的商业转化。不要低估小红书的搜索推荐体系，理解人群定位是为后续店铺操作所做的必要技术准备。

![](img/c30ad768a4c5a701087b1c2f4cb631d6_6.png)

## 用户个人属性 👤

首先，我们从用户个人属性开始。这包括性别、年龄范围和喜欢的内容。这与选择店铺类型时的考量是相通的。

以下是个人属性的核心构成：
*   **性别**：根据产品目标客户确定。
*   **年龄范围**：根据产品定位，选择比账号定位更精准的年龄段，以缩小目标人群范围。
*   **喜欢的内容**：需与产品特性相匹配。

**核心要点**：运营店铺时，需将账号属性与店铺属性相结合。若只做账号定位而忽略店铺人群定位，引流的用户可能并不精准，影响转化。选择方式与之前相同，但此处是以店铺产品为核心来反推人群。

## 用户消费能力 💰

接下来，我们分析用户的消费能力。小红书系统会根据用户的购买行为评估其消费能力。

消费能力的计算公式可以理解为：
`用户月均消费能力 ≈ 用户年总消费额 / 12`

系统通过分析用户购买的产品类型、价格及搜索行为，来判断其消费层级。因此，在为产品定价时，必须参考目标人群的消费能力，才能有效吸引他们。

## 用户购买类型 🛍️

系统分析用户购买行为后，通常只会给用户推荐**最多三种类型**的产品。

常见的产品大类目包括：
*   日用品
*   生活用品
*   户外用品
*   家具玩具

在自然搜索展示时，基本只显示三个产品类型，且根据用户最近的购买行为进行区分。这是小红书“千人千面”推荐机制的体现。用户经常浏览什么内容，系统就会持续推荐相关内容，大约2-3天即可定型。若要改变，可能需要4天左右才能切换到另一属性。

## 用户购买地域 🌍

购买地域指系统根据物流信息判断的用户常驻地区。

例如，系统会分析北京、上海、广州、深圳等城市的用户偏好购买服装、食品还是其他商品。如果你的店铺产品属性恰好符合北京用户的喜好，那么小红书系统就倾向于将你的店铺内容更多地推荐给在北京浏览的用户。

**运作逻辑**：系统将对应目标的产品，分享给对应目标地域的人群。

## 用户购买习惯 🔄

最后是用户购买习惯，例如“货比三家”型（长时间浏览后购买）。

系统根据个人用户的搜索和兴趣偏好进行判断。用户兴趣搜索越多，相应产品的推广就越广。但需要注意的是，如果用户在你的店铺浏览后，最终去竞争对手的店铺成交，那么系统后续推广竞争对手店铺属性的权重就会高于你的店铺。

![](img/c30ad768a4c5a701087b1c2f4cb631d6_10.png)

**注意事项**：过去存在通过批量浏览同行店铺后再到自家店铺成交，以抢夺流量的竞争手段。虽然现在不完全适用，但这种用户行为仍会对系统推荐产生潜在影响，需要留意。

![](img/c30ad768a4c5a701087b1c2f4cb631d6_12.png)

## 综合应用与前期策略 ⚙️

总结来说，店铺运营的完整路径是：**店铺定位 -> 选品类产品 -> 初步定价 -> 保持产品时效性 -> 人群定位**。

对于人群定位中的**个人属性、消费能力、购买类型、购买地域、购买习惯**这五点，系统会自动进行匹配。

**给初学者的建议**：
*   **如果你想做大**：需要深入理解并应用这五点，使推广人群非常精准。
*   **如果你是新手，想先熟悉节奏**：可以暂时只做前面的店铺定位和选品，不必急于应用全部人群定位策略。

**原因**：过早应用精准人群定位，虽然人群更精准，但系统给予的初始曝光量可能会减少（例如从2000-3000展示量降至1000左右）。在前期需要积累基础流量时，可以暂缓使用精准人群定位，优先保证曝光。

价格定位是人群定位的延伸，它会影响购买类型、地域和习惯。若前期不想操作复杂的人群定位，可自行定价，这并不影响起步。但需知，应用精准定位后，前期流量可能会减少近一半，不过后续的数据提升和转化效果通常会更好。
