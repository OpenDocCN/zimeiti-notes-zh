# 抖音小说推文项目：P1：小说推文玩法全解析 🚀

在本节课中，我们将要学习一个无需实拍、无需原创文案的抖音变现项目——小说推文。我们将详细拆解其运作原理、操作步骤和收益模式，帮助初学者快速理解并上手。

## 项目概述与原理

上一节我们介绍了课程主题，本节中我们来看看这个项目的核心逻辑。

![](img/b501750875c8cb6c8d5ac5972968b747_1.png)

这个玩法是近期非常流行的小说推文项目。它不需要你亲自拍摄视频或撰写文案，操作难度小，非常适合新手入门。

![](img/b501750875c8cb6c8d5ac5972968b747_3.png)

其变现原理包含两部分：
1.  **中视频计划收益**：通过视频播放量获得平台分成。
2.  **CPA推广佣金**：这是更主要的收益来源。视频文案朗读某部小说的精彩片段，引发观众好奇。观众若想阅读完整版，需根据作者提供的特定关键词去小说平台搜索。每成功引导一位用户，推广者即可获得约**10元**的佣金。

![](img/b501750875c8cb6c8d5ac5972968b747_5.png)

### 第一步：成为推荐官并获取授权

![](img/b501750875c8cb6c8d5ac5972968b747_13.png)

要操作此项目，首先需要成为小说平台的“推荐官”，获得推广授权。授权后，你会拿到每部小说对应的**特定推广关键词**。这个关键词是作者与平台之间的约定，是结算佣金的依据。

![](img/b501750875c8cb6c8d5ac5972968b747_15.png)

> 关于如何具体获取授权，我们将在课程结尾部分统一说明。

### 第二步：制作并发布视频

![](img/b501750875c8cb6c8d5ac5972968b747_19.png)

获得授权后，核心工作就是制作视频。这个过程非常简单高效。

![](img/b501750875c8cb6c8d5ac5972968b747_21.png)

以下是视频制作的具体步骤：

![](img/b501750875c8cb6c8d5ac5972968b747_23.png)

1.  **准备文案**：无需自己创作。直接从小说中复制一段约**1000字**、能引发好奇心的精彩段落。
2.  **生成配音**：将复制好的文案粘贴到AI配音工具中，选择一个合适的主播声音，导出音频文件。
    *代码示例（概念）：* `audio_file = AI_Voice_Generator(text=novel_excerpt, voice="主播音色")`
3.  **准备画面**：寻找解压类视频素材（如史莱姆、切肥皂等）。注意，不要只使用一个素材，而是将多个解压视频剪辑组合在一起。
4.  **合成视频**：使用剪映等剪辑软件，导入配音音频和组合好的解压视频素材。将解压视频的原声调至最低，确保配音清晰。最后导出成品视频。

![](img/b501750875c8cb6c8d5ac5972968b747_25.png)

整个视频制作流程非常快，**10分钟以内**即可完成一个。

![](img/b501750875c8cb6c8d5ac5972968b747_27.png)

![](img/b501750875c8cb6c8d5ac5972968b747_29.png)

## 收益测算与潜力

完成视频制作后，我们来算一笔账，看看这个项目的收益潜力。

根据了解，小说推文项目的转化率大致为：抖音上每**1万次播放**，可以产生**2到5单**转化。每单佣金约为**10元**。

假设你运营**10个**抖音账号，每个账号每天发布**3条**视频。即使每条视频只有抖音基础的**500次**播放，每日总播放量也能达到 **10 * 3 * 500 = 15,000次**。这至少能带来5单以上的收益。

事实上，很多操作者一人管理二三十个账号。据观察，大约每发布**100条作品**，就会出现一条爆款视频。一旦出现爆款，播放量会急剧上升，仅一条视频就可能带来数千元的收益。
