# 亚马逊卖家教程：P1：亚马逊雅典娜与阿波罗试点计划详解

在本节课中，我们将要学习亚马逊近期推出的两项新试点计划——“雅典娜计划”与“阿波罗计划”。我们将详细解析它们的具体内容、参与条件、潜在利弊，并帮助你判断是否值得参与。

## 概述

在销售旺季来临前，亚马逊推出了两项新的试点计划：雅典娜计划与阿波罗计划。卖家参与这些计划，有机会获得FBA配送费用折扣或免除低库存水平费。然而，这两项计划目前均采用邀请制，且参与后可能面临库存销售限制。本节课程将为你全面拆解这两项计划。

## 计划详情与规则

上一节我们介绍了计划的基本情况，本节中我们来看看两项计划的具体规则与核心要求。

### 雅典娜计划

雅典娜计划的核心福利是**免除低库存水平费**。

*   **福利公式**：`低库存水平费 = 0`（在活动期间内）
*   **活动时间**：2024年9月9日至2024年11月15日。
*   **重要限制**：参与该计划后，你现有的FBA库存**只能在亚马逊规定的特定区域内进行销售和配送**。

### 阿波罗计划

阿波罗计划主要针对低价商品，提供**FBA配送费用折扣**。

*   **目标商品**：售价低于25美元的商品。
*   **福利代码**：享受`FBA费用折扣`。
*   **活动时间**：2024年9月15日至2024年11月15日。
*   **重要限制**：系统将优先使用**同一区域内的库存**来配送买家订单。这意味着你的产品可能仅对该区域的买家可见和可售。

## 参与条件与风险权衡

了解具体规则后，我们需要明确如何参与以及必须考虑的风险。

以下是参与前必须了解的关键点：

1.  **邀请制**：两项计划均为邀请制。只有部分符合亚马逊资格的卖家会收到邀请开通的邮件。
2.  **自愿参与**：收到邀请后，卖家可自行决定是否参加。
3.  **核心风险**：所有加入计划的卖家都需要承担**FBA库存被限制在特定区域销售**的后果。这可能会影响你的整体销量和库存周转。

在决定是否参与前，请务必权衡自身的运营情况。你需要评估区域销售限制可能带来的潜在销量损失，与所能节省的费用（低库存费或配送费）相比，是否利大于弊。

## 总结

![](img/2057015267d2569a9fea7c11efe24d7c_1.png)

本节课中我们一起学习了亚马逊的雅典娜与阿波罗试点计划。总结如下：
*   **雅典娜计划**：在指定期间免除低库存水平费，但库存会被限制在特定区域销售。
*   **阿波罗计划**：为低价商品提供FBA费用折扣，同样面临库存区域销售限制。
*   **共同点**：均为邀请制、自愿参与，核心代价是接受库存的区域销售限制。

对于卖家而言，是否参与应基于详细的利弊分析。如果你的业务能承受特定区域的销售限制，且节省的费用显著，则可以考虑参与。反之，则应谨慎抉择。
