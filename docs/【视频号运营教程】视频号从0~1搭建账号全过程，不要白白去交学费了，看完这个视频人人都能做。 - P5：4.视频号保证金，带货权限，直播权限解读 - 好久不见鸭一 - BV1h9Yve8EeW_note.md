# 视频号运营教程：P5：视频号保证金、带货权限与直播权限解读 🎬

在本节课中，我们将详细解读视频号的带货身份、带货权限以及直播权限的具体规则。了解这些是开通相关功能、避免违规操作的基础。

## 带货身份与经营资质

上一节我们介绍了视频号的基础设置，本节中我们来看看如何查看和设置带货身份。

在创作者中心板块，点击右上角的个人头像，即可查看你的带货身份。身份主要分为两种：**达人**和**商家**。你可以根据自身情况进行选择。这个身份后期可以修改，因此无需担心初次选择错误。

关于经营资质，目前个人用户通常无需额外关注。

## 带货权限详解

了解身份后，我们来看看具体的带货权限。带货权限主要分为三个板块：视频带货、橱窗带货和直播带货。

以下是三种带货方式的详细说明：

*   **视频带货**
    *   **含义**：指在发布的视频中挂载商品链接，类似于抖音的“小黄车”。用户观看视频时，屏幕上会出现橙色购物车图标，点击即可查看和购买商品。
    *   **开通条件**：要求较高，通常需要**1000粉丝**且账号为**原创内容**。搬运内容的账号很难开通。
    *   **注意事项**：该权限非常严格，一旦发生违规，权限会被直接封禁。被封后，视频无法挂车，但橱窗功能仍可使用。因此，对于初学者，不必强求开通此功能。

![](img/88b0405644d65fd00a8185a05cc22438_1.png)

*   **橱窗带货**
    *   **含义**：在视频号主页展示商品橱窗，用户可以通过点击橱窗浏览和购买商品。这是最基础、最稳定的带货方式。
    *   **优势**：开通门槛相对较低，且不受视频带货权限封禁的影响。是新手起步的核心带货方式。

![](img/88b0405644d65fd00a8185a05cc22438_3.png)

![](img/88b0405644d65fd00a8185a05cc22438_4.png)

![](img/88b0405644d65fd00a8185a05cc22438_5.png)

*   **直播带货**
    *   **含义**：在直播过程中销售商品。这是互动性强、转化率高的带货方式。
    *   **开通条件**：这是本节课的重点，涉及两个核心限制。

![](img/88b0405644d65fd00a8185a05cc22438_7.png)

## 直播带货权限的核心规则

![](img/88b0405644d65fd00a8185a05cc22438_9.png)

接下来，我们重点剖析直播带货权限的具体要求。

![](img/88b0405644d65fd00a8185a05cc22438_11.png)

![](img/88b0405644d65fd00a8185a05cc22438_13.png)

**1. 实名认证限制**
根据平台最新规则，一个自然人虽然可以实名认证5个微信，但只能使用其中**两个**进行视频号实名认证。这意味着，你最多只能拥有两个实名认证的视频号用于直播。

![](img/88b0405644d65fd00a8185a05cc22438_15.png)

![](img/88b0405644d65fd00a8185a05cc22438_17.png)

**2. 粉丝数或店铺要求**
在直播间挂载商品销售，必须满足以下两个条件之一：
*   **条件A**：视频号拥有**1000个有效粉丝**。
*   **条件B**：开通**视频号小店**（需要营业执照）。

![](img/88b0405644d65fd00a8185a05cc22438_19.png)

如果以上条件均不满足，虽然可以开启直播，但**无法在直播间添加任何商品**进行销售。此时，只能通过主页橱窗进行带货。

![](img/88b0405644d65fd00a8185a05cc22438_21.png)

为了更直观地理解，以下是开通直播带货功能的逻辑判断：
```plaintext
IF (粉丝数 >= 1000) OR (已开通视频号小店 == TRUE) THEN
    可以“直播带货”（直播间可挂商品）
ELSE
    可以“直播”，但不可“带货”（直播间不可挂商品）
END IF
```

## 课程总结

![](img/88b0405644d65fd00a8185a05cc22438_27.png)

本节课我们一起学习了视频号带货与直播的核心权限规则。

![](img/88b0405644d65fd00a8185a05cc22438_29.png)

我们首先区分了**达人**与**商家**两种带货身份。然后详细解读了**视频带货**、**橱窗带货**和**直播带货**三种权限，其中视频带货门槛高、易被封，橱窗带货最稳定。

最后，我们重点掌握了**直播带货**的两大关键条件：**实名认证数量限制（最多2个）** 以及 **“千粉”或“开店”的二选一门槛**。对于新手，在粉丝不足时，通过办理营业执照开通视频号小店，是快速获得直播带货权限的有效途径。

![](img/88b0405644d65fd00a8185a05cc22438_31.png)

记住这些规则，能帮助你在运营中避免踩坑，更高效地规划你的视频号成长路径。关于如何办理营业执照及开通视频号小店的具体流程，我们将在后续课程中详细讲解。
