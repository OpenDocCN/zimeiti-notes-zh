# 短视频运营教程：P25：产品与用户兴趣标签

在本节课中，我们将学习一种实用的选题方法——九宫格选题法。该方法通过分析产品属性和用户兴趣，帮助创作者找到源源不断的视频创作灵感。我们将以健身软件Keep为例，详细拆解操作步骤。

上一节我们探讨了选题的重要性，本节中我们来看看如何具体生成选题。

## 产品标签分析

首先，我们需要分析产品本身的特点。以Keep为例，我们将产品名称放在九宫格的中心，然后围绕它联想产品的核心属性和功能。

以下是围绕Keep可以联想到的产品标签：

1.  **线上健身软件**：这是产品的核心定位。
2.  **专业健身教练**：产品提供专业指导资源。
3.  **饮食搭配建议**：产品包含营养学相关内容。
4.  **减肥/塑形攻略**：产品的主要服务目标之一。
5.  **健身干货知识**：产品提供教育性内容。
6.  **科学运动方法**：强调其内容的权威性和有效性。
7.  **打卡运动制度**：产品的用户激励和社区功能。
8.  **一二线城市用户为主**：产品的核心用户画像，这属于产品属性。例如，`抖音用户分布`与`快手用户分布`在不同平台的数据报告中可以查到差异。

完成产品标签填充后，我们暂时将其放在一边。

## 用户兴趣标签分析

接下来，我们分析目标用户的兴趣。使用Keep的用户主要是健身爱好者或想要减肥、塑形的人。我们将这个用户群体放在另一个九宫格的中心。

然后，我们开始联想这个群体可能还对哪些事物感兴趣。思路可以放开，从核心兴趣扩展到更广泛的生活领域。

以下是针对“健身爱好者”可以扩展的用户兴趣标签：

1.  **健身房**：线下健身场所。
2.  **健身装备**：如运动服、运动鞋等。
3.  **轻脂美食/减肥餐**：与健康饮食相关。
4.  **旅行**：热爱生活的一种表现。
5.  **星座/明星**：普遍的娱乐兴趣点。
6.  **鸡汤文**：情感激励类内容。
7.  **时尚穿搭**：与外在形象管理相关。
8.  **生活热爱者**：将用户画像从“健身爱好者”升级到更广泛的“热爱生活的人”。

通过以上步骤，我们得到了两个填满的九宫格：一个基于产品，一个基于用户兴趣。

## 生成选题：连连看组合法

现在，我们有了两组关键词。如何将它们变成具体的视频选题呢？方法是进行“连连看”组合。

**操作公式**：`产品标签` + `用户兴趣标签` = `潜在视频选题`

我们从产品标签和用户兴趣标签中各选一个进行组合，尝试构思一句完整的视频标题或文案。

以下是几个组合示例：

*   **组合1：饮食搭配 + 旅行**
    *   潜在选题：`外出旅行时，如何享受美食又能控制体重？`
*   **组合2：运动方法 + 明星**
    *   潜在选题：`赵丽颖都在练的5个高效健身动作，你一定要学！`
*   **组合3：科学 + 健身房**
    *   潜在选题：`揭秘健身房最科学的训练流程，别再瞎练了！`

通过这种交叉组合，理论上可以产生大量的选题思路。这有助于你建立丰富的选题素材库，解决“不知道拍什么”的难题。

## 优化选题与工具辅助

然而，并非所有组合产生的想法都足够好。例如，“在健身房如何科学撩妹”这个方向就可能偏离主题。

当你缺乏灵感时，可以借助搜索工具。例如，将组合出的关键词（如“健身 科学”）输入搜索引擎，查看相关的搜索联想和现有内容，这能为你提供句式参考和灵感补充。

**本节课中我们一起学习了九宫格选题法**。其核心步骤是：首先，分别围绕产品和目标用户填充关键词；然后，通过将两类关键词进行“连连看”式组合，来激发创作灵感；最后，可以借助搜索工具对初步想法进行优化和拓展。掌握这个方法，能帮助你系统性地挖掘视频选题，保持内容更新的持续性与多样性。
