1.  **角色**
    角色即你在特定赛道中的专业身份。你选择什么赛道、输出什么方向的内容，就需要塑造与之对应的专业角色。例如：
    *   **好物分享/测评赛道**：你可以是“资深测评官”或“生活好物挖掘机”。
    *   **装修赛道**：你可以是“正在装修的业主”，通过记录过程、分享避坑经验来建立信任。
    *   **母婴赛道**：常见的角色是“宝妈”，但也可以是“育儿护理专家”或“儿童营养师”。
    *   **情感赛道**：你可以是“情感顾问”、“心理学导师”，也可以是分享自身经历的“成长型人设”，例如记录30岁女性如何应对家庭、职场与创业挑战。

    核心公式为：**人设角色 = 赛道方向 + 专业身份**。

2.  **内容**
    内容是人设的载体。你提供的内容必须与你设定的角色高度一致，并持续深耕垂直领域。这关系到账号标签的建立和平台流量的精准分发。
    如果内容杂乱无章（例如今天讲育儿，明天发宠物），会打乱账号标签，导致推送不精准，影响数据表现和流量推荐。因此，**内容垂直度**是强化人设的关键。

3.  **形式**
    形式是指内容呈现的具体方式，主要包括以下几点：
    *   **产出场景**：是在固定的室内场景，还是多变的室外、车内等场景进行拍摄？
    *   **声音呈现**：是本人出镜讲解，还是使用配音？这取决于你能否流畅地进行口头表达。
    *   **内容形式**：是**剧情演绎**、**干货分享**，还是**生活记录**？每种形式各有优劣：剧情类创作难度高但易传播；干货类需要扎实的文案脚本；记录类则需要丰富的日常素材。

4.  **记忆点**
    记忆点是让用户记住你的关键“钩子”。它需要被刻意设计并反复强化。有效的记忆点通常包括：
    *   **视频开头前置**：固定的开场动作、特效或口头禅。这些元素在视频前几秒出现，能快速建立用户认知。
    *   **贯穿全程**：统一的背景音乐（BGM）、始终出现的标志性物品或特定风格的服装。这些元素在整个视频中持续存在，加深用户印象。

    设计记忆点的核心策略是：**将强视觉/听觉冲击元素（如动作、口号）置于视频开头；将氛围营造元素（如BGM、视觉风格）贯穿内容始终**。
