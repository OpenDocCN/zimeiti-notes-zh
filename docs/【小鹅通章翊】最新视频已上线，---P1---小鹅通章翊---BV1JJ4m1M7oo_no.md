# 编程入门：1：变量与基础运算

在本节课中，我们将要学习编程中最基础的两个概念：变量和基础运算。它们是构建所有复杂程序的基石。

## 概述

变量是用于存储信息的容器。基础运算则允许我们对这些信息进行处理。理解这两者是如何协同工作的，是学习编程的第一步。

## 变量的定义与赋值

上一节我们概述了变量的作用，本节中我们来看看如何具体地创建和使用一个变量。

在编程中，我们可以给一个存储空间起一个名字，并向其中放入数据。这个过程称为“赋值”。例如，在Python语言中，我们可以这样写：

```python
my_number = 10
my_name = “小明”
```

第一行代码创建了一个名为 `my_number` 的变量，并将数值 `10` 存储其中。第二行代码创建了一个名为 `my_name` 的变量，并将文本“小明”存储其中。

以下是变量命名的几个基本规则：
*   变量名只能包含字母、数字和下划线。
*   变量名不能以数字开头。
*   变量名是区分大小写的（例如 `age` 和 `Age` 是两个不同的变量）。

## 基础数学运算

创建了变量之后，我们就可以对其中存储的数值进行各种计算。这就像使用计算器一样简单。

编程支持我们熟悉的加、减、乘、除等运算。它们的写法与数学中略有不同。

以下是几种常见的数学运算符及其用法：
*   **加法**：使用 `+` 符号。例如 `5 + 3` 的结果是 `8`。
*   **减法**：使用 `-` 符号。例如 `5 - 3` 的结果是 `2`。
*   **乘法**：使用 `*` 符号。例如 `5 * 3` 的结果是 `15`。
*   **除法**：使用 `/` 符号。例如 `6 / 3` 的结果是 `2.0`。

我们可以将运算结果直接打印出来，或者存储到一个新的变量中，以便后续使用。

```python
a = 5
b = 3
sum_result = a + b  # 将 a+b 的结果 8 存入 sum_result 变量
print(sum_result)   # 在屏幕上输出 8
```

## 变量与运算的结合

现在，让我们把变量和运算结合起来。这是让程序变得动态和有用的关键。

我们可以用变量参与运算，也可以将运算的结果赋值给变量。这样，当变量的值发生变化时，运算结果也会自动更新。

例如，我们可以计算一个长方形的面积：

```python
length = 10
width = 5
area = length * width
print(area)  # 输出 50
```

如果我们将 `length` 的值改为 `20`，那么 `area` 的值在计算后就会自动变为 `100`，而不需要修改计算面积的公式本身。这体现了使用变量的优势。

## 总结

本节课中我们一起学习了编程的基础：变量和基础运算。

我们了解了变量是一个命名的存储空间，用于保存数据，并通过赋值语句来存放值。我们也学习了如何使用 `+`、`-`、`*`、`/` 这些基本运算符进行数学计算。最重要的是，我们看到了如何将变量与运算结合，通过改变变量的值来动态地得到不同的计算结果。

掌握这些概念，你就已经迈出了编写有用程序的第一步。
