# 剪映电脑版全教程：P4：剪映做短视频的优势 🎬

![](img/fb2b6ca56fed235910827060ad972110_1.png)

在本节课中，我们将要学习剪映这款软件在短视频制作领域的核心优势。了解这些优势，能帮助我们理解为何选择剪映作为剪辑工具，以及它如何帮助我们更高效地创作内容。

上一节我们介绍了课程的整体框架，本节中我们来看看剪映相较于其他软件，具体有哪些过人之处。

## 优势一：简单好用 ✨

剪映的操作界面和功能设计对零基础用户非常友好。其剪辑逻辑直观，无需复杂的专业知识，用户可以快速上手并掌握基本操作。

## 优势二：完全免费 💰

剪映是一款完全免费的剪辑软件。用户无需开通会员或为素材、特效等功能付费。这与早期一些以“免费”为噱头，但核心功能需要付费的剪辑应用形成了鲜明对比。

## 优势三：功能强大 ⚙️

尽管剪映目前版本号不高，但其功能持续更新，已非常强大。它几乎能满足在抖音等平台制作短视频的所有需求。

以下是其核心功能模块：
*   **文本**：添加字幕和标题。
*   **音频**：处理背景音乐和音效。
*   **贴纸**：增加趣味图形元素。
*   **特效**：应用视觉特效。
*   **色度抠图**：通过`画中画`功能，利用`色度抠图`技术去除绿色背景，实现特效视频合成。其操作类似于代码中的函数调用：`apply_chroma_key(绿幕素材, 背景)`。

## 优势四：海量素材 📚

剪映内置了丰富的素材库，极大地方便了用户创作。

其素材优势主要体现在两点：
1.  **无缝连接抖音热门素材**：可以直接使用抖音热门的视频模板和背景音乐（BGM），快速制作同款热门视频。
2.  **自带竖屏素材库**：软件本身提供了大量适用于手机竖屏观看的素材，用户无需费力寻找，对新手尤其友好。

## 优势五：官方推荐与流量扶持 🚀

剪映与抖音同属字节跳动公司。使用剪映制作并发布到抖音的视频，会获得平台的流量推荐和扶持。

![](img/fb2b6ca56fed235910827060ad972110_3.png)

这解释了为何许多抖音视频下方带有“剪映制作”的标识。对于希望在抖音平台发展的创作者而言，剪映是一个能获得官方流量加持的重要渠道。
