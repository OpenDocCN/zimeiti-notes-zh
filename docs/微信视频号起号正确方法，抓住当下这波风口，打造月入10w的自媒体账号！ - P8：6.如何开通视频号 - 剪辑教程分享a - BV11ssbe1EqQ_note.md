# 微信视频号运营教程：P8：6.如何开通视频号

在本节课中，我们将学习如何正确地为微信视频号账号进行定位，解决播放量低、涨粉慢的问题，并了解如何通过创作者分成计划获得收益。我们将重点讲解账号垂直与活跃度这两个核心概念。

## 账号播放量低的原因分析

上一节我们提到了账号运营的常见问题，本节中我们来看看具体原因。许多同学的视频号播放量只有几十或几百，涨粉速度缓慢。造成这种情况的主要原因有两个：**账号不垂直**和**账号不活跃**。

![](img/a84617d28f092a6ece15a288f12fb1de_3.png)

## 核心概念一：账号垂直

![](img/a84617d28f092a6ece15a288f12fb1de_4.png)

首先，我们来深入理解什么是“账号垂直”。简单来说，垂直就是指专一。你需要确定一个内容方向，并持续发布该类型的视频。

以下是账号垂直的具体解释：
*   **定义**：你计划做什么类型的内容，就只发该类型的视频。
*   **举例**：如果你想做美食账号，就只发美食视频；想做服装账号，就只发服装相关视频；想做搞笑账号，就只发搞笑类视频。
*   **错误示范**：不能将视频号当作朋友圈来发。例如，今天发美食，明天发搞笑，后天发风景，这会导致账号定位混乱，系统无法识别你的受众，从而影响推荐和涨粉。
*   **正确案例**：一个账号如果全部发布与“冰雪”相关的搞笑或生活视频，那么这个账号在“冰雪”和“搞笑”领域就是垂直的。

**公式**：**账号垂直 = 持续发布同一领域的内容**

## 核心概念二：账号活跃

理解了垂直的重要性后，我们再来看看第二个关键点：账号活跃度。你可以将运营视频号想象成经营一家线下实体店。

![](img/a84617d28f092a6ece15a288f12fb1de_8.png)

以下是保持账号活跃的要点：
*   **类比**：在互联网上赚钱，相当于开了一家线上店铺。想要有生意，就必须每天开门营业。
*   **要求**：必须保持定期更新，不能“三天打鱼，两天晒网”。心情好就发，心情不好就不发，这种态度无法做好账号。
*   **心态**：请明确自己运营账号的目的。如果是为了学习赚钱，就需要保持专注和坚持。

![](img/a84617d28f092a6ece15a288f12fb1de_6.png)

## 定位总结与收益关联

我们已经介绍了账号垂直和活跃这两个核心要点。将这两点结合起来，就是清晰的账号定位。在线下创业需要选择行业（如餐饮、服装），在线上做视频号同样需要选择并专注于一个内容领域。

![](img/a84617d28f092a6ece15a288f12fb1de_10.png)

![](img/a84617d28f092a6ece15a288f12fb1de_11.png)

按照上述方法理清账号定位、保持内容垂直和更新活跃后，你的账号基础就打好了。此时，可以参与平台的**创作者分成计划**。

![](img/a84617d28f092a6ece15a288f12fb1de_13.png)

关于该计划的收益：
*   单条视频播放量稳定在5000以上时，通常可以获得收益。
*   例如，单条视频达到5000播放量，预计可获得约50元收益。虽然单次金额不大，但积少成多。
*   该计划可能有保底机制，为创作者提供基本保障。
