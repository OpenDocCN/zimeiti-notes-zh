# 小红书运营高阶教程：P30：小红书爆款笔记写作方法（二）

在本节课中，我们将学习如何确立小红书笔记的核心内容。我们将分步骤讲解从确定领域到最终成稿的完整流程，并介绍一个实用的数据分析工具来辅助创作。

上一节我们介绍了爆款笔记的基础框架，本节中我们来看看如何具体确立笔记的内容方向。

## 内容确立的五个步骤

![](img/adaa6bd5ae071c96d01235bba7304686_1.png)

![](img/adaa6bd5ae071c96d01235bba7304686_3.png)

以下是确立小红书笔记内容的五个关键步骤。

![](img/adaa6bd5ae071c96d01235bba7304686_5.png)

![](img/adaa6bd5ae071c96d01235bba7304686_7.png)

1.  **确定内容领域**
    首先，需要明确你的账号专注于哪个领域，例如美妆、护肤、母婴或穿搭。明确的领域定位至关重要。若领域不明确，会导致账号垂直度降低。账号垂直度降低会引发一系列问题，例如平台减少推荐、粉丝流失。这会形成恶性循环：粉丝越不喜欢你的内容，平台就越不推荐你的内容。

![](img/adaa6bd5ae071c96d01235bba7304686_9.png)

2.  **梳理行业关键词**
    确定领域后，需要梳理与该领域相关的核心关键词。在之前的课程中，我们介绍了使用百度工具、爱站网、5118、百度指数等工具进行关键词查询的方法。本节课不再赘述。根据你的行业，可以梳理出许多关键词及其热度排行。用好这些关键词排行，能极大提升内容效果。

3.  **搜集同行案例**
    第三步是搜集和分析同行的优秀案例。有人可能会问，在小红书内搜索无法看到同行的详细数据，该怎么办？解决方法其实很简单。

    今天我们推荐一个工具：**牵瓜数据**。这是一个长期使用且非常好用的数据分析平台。

    ![](img/adaa6bd5ae071c96d01235bba7304686_1.png)

    该平台的优势在于提供小红书的多维度分析，例如匹配优质达人、追踪热门数据、分析品牌投放等。

    ![](img/adaa6bd5ae071c96d01235bba7304686_3.png)
    ![](img/adaa6bd5ae071c96d01235bba7304686_5.png)
    ![](img/adaa6bd5ae071c96d01235bba7304686_7.png)

    平台内还包含各类榜单。下面我们详细说明如何使用牵瓜数据。

4.  **筛选与整合内容**
    搜集好同行案例后，即可对这些案例内容进行筛选。筛选完成后，将优质内容整合成自己的笔记素材，然后进行发布。

    我们以牵瓜数据为例进行演示。在平台左侧栏目中，可以看到许多功能，包括热词排行榜、笔记收录查询等。本节课主要讲解“笔记收录查询”功能。发布笔记后，若担心笔记未被平台收录，即可使用此功能查询。

    对于寻找同行案例，主要使用“达人搜索”功能。可以按昵称或笔记内容搜索，例如搜索“美妆”（部分功能需付费）。按昵称搜索后，会出现许多达人，其分类属性包括彩妆、护肤、母婴、美式饮品、服饰穿搭等。大家可根据自身需求进行筛选。

    ![](img/adaa6bd5ae071c96d01235bba7304686_9.png)

    可以看到达人的属性，如是否为官方号、品牌号或头部达人。平台提供了大量数据，包括达人对比榜单。排名靠前的达人数据具有参考价值。如果需要开通此网站的服务，可以在淘宝上寻找开通渠道，费用通常比较划算。

    使用此功能时，点击任意达人的详情页，即可查看其发布的数据、笔记分类、笔记详情。

    ![](img/adaa6bd5ae071c96d01235bba7304686_11.png)

    甚至可以看到其笔记的评论词云图，即高频出现的词汇。用好这个工具，能有效帮助我们掌握小红书的数据参数。

5.  **整理与撰写内容**
    最后一步是整理并撰写自己的内容。在之前的课程中已讲过，主要采用“总-分-总”的结构来组织内容。这个结构并不难掌握，如果觉得困难，建议回顾之前的课程内容。本节课不再重复这一点。

![](img/adaa6bd5ae071c96d01235bba7304686_11.png)

## 其他注意事项

以下是关于笔记标题和内容的其他几点重要注意事项。

1.  **控制标题字数**：标题字数尽量控制在 **12到18个字** 之间。
2.  **使用符号美化**：标题中可多使用 **“！”、“~”、“【】”** 等符号进行美化。因为小红书女性用户居多，美化后的标题更能吸引她们的注意力。
3.  **加入具体数据**：在标题中加入具体数据可以增加真实感。例如，“**百万** 收藏”、“**3天** 见效”等详细数字，会让人感觉更可信。因此，在拟定标题时，可以酌情加入数字。

本节课中我们一起学习了确立小红书笔记内容的五个步骤：确定领域、梳理关键词、借助“牵瓜数据”搜集同行案例、筛选整合内容、以及最终撰写。同时，我们还掌握了优化标题的三个技巧：控制字数、符号美化和加入数据。运用这些方法，可以更高效地创作出具有潜力的爆款笔记。
