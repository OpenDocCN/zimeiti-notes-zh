# 小红书运营：P29：28、小红书笔记引流：平台流量结构 📊

![](img/2a6fd571ccd1ca5d0e3c3dbb235f956b_1.png)

在本节课中，我们将要学习小红书平台的整体流量结构。了解流量从哪里来，以及不同流量入口的特点，是制定有效运营策略的基础。我们将重点拆解发现页、关注页和搜索页这三大核心流量来源，帮助你明确笔记的投放方向。

## 概述

本节内容旨在解析小红书平台的流量构成。我们将探讨笔记发布后，流量主要来自哪些页面，以及每个页面的流量占比和特性。理解这些结构，有助于我们更精准地进行内容布局和引流操作。

## 平台流量结构解析

小红书笔记的整体流量结构主要分为三个部分：**发现页**、**关注页**和**搜索页**。打开小红书APP，主页面通常显示“发现”和“关注”，旁边的放大镜图标则代表“搜索”。

关于“本地”页面，它并非核心流量入口，主要用于地域性内容展示。例如，在北京、上海、广州等地，它会显示附近用户发布的内容。这个页面适合用于寻找本地化素材，了解当地用户的喜好。但需注意，除非你是专注于本地美食、户外等领域的博主，否则应尽量避免过度依赖地域性引流。因为地域性内容容易打乱账号的整体标签，其流量推荐逻辑是基于地理位置临近、发布时间和点击率，不利于面向全国市场的账号成长。

因此，笔记引流应聚焦于以下三个核心页面。

### 1. 发现页：流量主阵地 🔥

发现页是整个小红书**最重要**的流量来源，约占平台总流量的 **70%**。它也是平台进行流量层级测试的主要场所。

一篇笔记能获得多大规模的曝光，很大程度上取决于它在发现页获得的用户反馈（如点赞、收藏、评论）。账号权重决定了笔记发布后的初始曝光量（例如1000或2000），而这部分初始流量几乎全部投放在发现页。

**公式表示：**
`笔记初始曝光量 ≈ 账号权重 × 基础系数`（该曝光主要进入发现页）

![](img/2a6fd571ccd1ca5d0e3c3dbb235f956b_3.png)

这意味着，对于新账号或新笔记，运营重心必须放在如何提升在发现页的表现上。



### 2. 搜索页：精准流量与增长引擎 🔍

搜索页的流量占比仅次于发现页，约为 **20%**，并且是小红书流量增长最快的板块。

虽然小红书的月活用户量巨大且增长趋于稳定，但**搜索商品的用户量正在持续增加**。搜索流量的用户意图非常明确，因此转化率极高。据统计，小红书平台内直接成交的订单，有约 **70%** 来源于搜索页。

**核心逻辑：**
用户通过搜索关键词寻找商品或解决方案 → 笔记匹配关键词并展示 → 高意向用户点击 → 促成转化。

然而，搜索页的流量并非新手可以轻易获取。它依赖于笔记的**关键词布局**、商品的**权重**（如销量、评价）以及账号的**整体信誉**。这部分流量是后期需要重点攻坚的方向。



### 3. 关注页：高粘性私域流量 ❤️

关注页的流量占比最小，大约只有 **10%**，甚至更低（约7%）。但这部分流量价值最高，因为用户是已经关注你的粉丝，兴趣匹配度极高。

只要用户关注了你，你发布的新笔记就会出现在他们的关注流中。这部分用户粘性强，适合进行深度互动、新品推广或引导至其他私域平台（如微信群）。

但有一个关键点需要注意：来自关注页的流量**不计入笔记的基础权重计算**。也就是说，即使你有1万粉丝，笔记发布后全部粉丝都阅读了，这篇笔记也未必能获得平台的额外推荐流量（即“爆”）。它主要起到维护粉丝关系、稳定基础互动的作用。

**运营顺序建议：**
前期主攻**发现页**，积累数据和权重；中期优化**搜索页**，获取精准转化流量；长期维护**关注页**，沉淀忠实用户。

## 总结

本节课我们一起学习了小红书平台的三大流量结构：
1.  **发现页（占70%）**：流量核心，决定笔记的初始爆发潜力。
2.  **搜索页（占20%）**：精准流量源头，是成交转化的关键，依赖后期优化。
3.  **关注页（占约10%）**：高价值私域流量，用于维护粉丝，但不直接影响推荐权重。

理解这个结构，就能明确不同阶段的运营重点：先做好发现页内容，争取平台推荐；再优化搜索关键词，收割精准用户；同时用心经营关注者，构建自己的流量池。
