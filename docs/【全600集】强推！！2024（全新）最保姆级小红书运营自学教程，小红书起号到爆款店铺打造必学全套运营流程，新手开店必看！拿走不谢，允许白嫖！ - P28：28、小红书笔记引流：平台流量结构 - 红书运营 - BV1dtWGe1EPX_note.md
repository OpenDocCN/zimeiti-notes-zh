# 小红书运营自学教程：P28：小红书笔记引流：平台流量结构 📊

![](img/2809437c1fc244e896496a850adffc4f_1.png)

在本节课中，我们将要学习小红书平台的流量结构。了解流量从哪里来，以及不同流量渠道的特点，是制定有效运营策略的基础。我们将重点分析发现页、关注页和搜索页这三大流量来源，并探讨如何根据它们的特点来优化我们的笔记发布策略。

## 平台流量结构概述

上一节我们介绍了账号权重等基础概念，本节中我们来看看小红书笔记引流的整体流量结构。流量结构指的是小红书平台内流量的整体分布和来源。理解流量结构，能帮助我们了解不同类型的笔记投放到不同位置后，可能获得的效果。

小红书笔记的整体流量结构主要分为三个部分。

以下是三个核心流量入口：

1.  **发现页**
2.  **关注页**
3.  **搜索页**

打开小红书App，主页面通常显示“发现”和“关注”两个标签页，旁边还有一个放大镜图标代表“搜索”页。这就是三个最主要的流量入口。此外，还有一个“本地”页面，但它主要用于特定场景，并非核心的通用流量入口。

“本地”页面会根据用户地理位置展示内容。它适合用于寻找地域性素材，例如了解本地人喜欢什么特色。但需要注意的是，除非你是专注于本地美食、户外等领域的博主，否则应尽量避免在笔记中过度强调地域内容。因为通过地域搜索进来的流量可能会打乱你账号的整体标签，导致推送不精准。例如，你卖首饰，本地推送可能吸引来看本地新闻或美女的用户，而非你的目标客户。如果你的目标是面向全国市场，就更应谨慎使用本地流量。

因此，笔记引流的整体流量结构核心就是三个点：发现页、关注页和搜索页。

## 核心流量入口详解

### 发现页：最主要的流量来源 🚀

发现页是整个小红书最重要的流量来源，约占平台总流量的 **70%**。它也是平台进行流量层级测试的主阵地。一篇笔记最终能获得多大规模的曝光，很大程度上取决于它在发现页获得的用户反馈（如点赞、收藏、评论）。

![](img/2809437c1fc244e896496a850adffc4f_3.png)

之前课程提到，账号权重决定了笔记的基础曝光量。一篇新笔记发布后，根据账号权重，平台可能会给予例如1000或2000的基础曝光。这**70%** 的基础曝光量，几乎全部会投放在发现页。新账号的笔记，初期流量基本都来自发现页。剩下的30%流量，则分布在关注页和搜索页，这些流量目标更明确，用户意图更强。

![](img/2809437c1fc244e896496a850adffc4f_3.png)

### 搜索页：转化率最高的流量 🔍

搜索页的流量占比仅次于发现页，约为 **20%**。这是小红书流量增长最快的板块。小红书的月活跃用户已非常庞大，新增用户速度会放缓，但平台内搜索商品的用户比例正在持续增加。

搜索页流量的核心特点是**高转化率**。虽然其整体流量占比为20%，但它可能贡献了平台超过 **70%** 的直接成交额。因为用户主动搜索时，购买意图非常明确。相比之下，发现页和关注页的用户更多是在浏览内容，产生兴趣后才可能成交。

对于新账号而言，初期很难直接获取搜索页的流量。因为搜索流量依赖于商品笔记的权重，这需要积累关键词排名、成交、转化、评价等综合数据。这部分能力需要在账号运营中后期，通过店铺和商品的整体优化来提升。

### 关注页：最精准但量小的流量 ❤️

关注页的流量占比最小，通常不到 **10%**，大约在7%左右。虽然量小，但这部分用户最为精准。因为他们已经关注了你的账号，对你的内容有兴趣。投其所好，在这部分用户中进行引流或成交会相对容易。

关注页是用户主页左侧的页面。用户关注你之后，你发布的笔记就会出现在他们的关注流中。需要注意的是，来自关注页的流量**不影响笔记的基础权重和流量层级数据**。例如，即使你有1万粉丝，笔记发布后全部粉丝都阅读了，但如果这篇笔记在发现页的基础曝光只有2000且互动率低，它依然不会成为爆款。关注页的流量是“锦上添花”，而非“雪中送炭”。运营初期，重点仍应放在发现页和搜索页上。

## 总结与行动顺序

本节课中我们一起学习了小红书平台的三大流量结构：发现页、搜索页和关注页。

我们来总结一下它们的核心特点和运营初期的行动顺序：

1.  **主攻发现页**：作为最主要的流量来源（**70%**），是新手启动和测试内容的主战场。目标是获得好的用户反馈，进入更大的流量池。
2.  **布局搜索页**：作为高转化率的流量来源（**20%**），需要在账号有一定基础后，通过优化商品标题、关键词等来逐步获取。
3.  **维护关注页**：作为最精准的私域流量（**<10%**），用于维护核心用户，提升粉丝粘性和复购，但不作为前期冲量的主要依赖。

因此，我们的运营顺序应该是：优先做好发现页内容，积累权重和数据；同时布局搜索关键词，为后期转化做准备；并用心维护关注你的用户。这就是掌握小红书流量密码的钥匙。
