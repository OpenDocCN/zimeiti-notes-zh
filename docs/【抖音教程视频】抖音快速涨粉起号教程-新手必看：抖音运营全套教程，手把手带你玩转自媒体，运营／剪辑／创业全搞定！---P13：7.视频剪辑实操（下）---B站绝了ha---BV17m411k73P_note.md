# 抖音运营教程：P13：7.视频剪辑实操（下）

在本节课中，我们将继续学习视频剪辑的核心步骤，重点掌握如何为视频片段添加转场效果，以及如何为视频添加和调整字幕。这些操作对于提升视频的流畅度和专业度至关重要。

上一节我们介绍了视频比例调整与背景设置，本节中我们来看看如何让多个视频片段之间衔接得更自然。

## 添加视频转场

每个视频通常由多个片段组成。片段之间需要有明确的衔接，否则观众在观看时可能会感到突兀，甚至退出视频，这会严重影响视频的**完播率**，进而影响收益。

**完播率**是衡量视频质量的关键指标，公式可简单理解为：
`完播率 = 完整观看视频的用户数 / 观看视频的总用户数`

![](img/162d59626237eb79621535076f7867f5_1.png)

因此，我们必须为视频添加转场。以下是添加转场的具体步骤：

1.  在视频轨道右侧，找到并点击“加号”按钮。
2.  在弹出的界面中，选择“素材库”。
3.  在“热门”分类下，可以找到大量常用的转场素材，例如搞笑的片段或动态效果。
4.  选择转场时，注意时长。过长的转场会让观众失去耐心，建议选择**3~5秒**的转场素材，例如一个3秒的搞笑片段。
5.  点击选中合适的转场素材，将其添加到视频轨道中。

## 添加与调整字幕

![](img/162d59626237eb79621535076f7867f5_17.png)

视频内容制作完成后，下一步是为视频添加字幕（文案），这能极大提升视频的可看性和信息传达效率。

以下是添加字幕的操作流程：

1.  点击手机剪辑界面左下角的“文字”选项。
2.  选择“新建文本”旁边的“识别字幕”功能。
3.  点击“开始匹配”，软件会自动识别视频中的语音并生成字幕文本。

![](img/162d59626237eb79621535076f7867f5_19.png)

![](img/162d59626237eb79621535076f7867f5_19.png)

![](img/162d59626237eb79621535076f7867f5_21.png)

字幕生成后，通常需要调整样式和位置以达到最佳视觉效果：

![](img/162d59626237eb79621535076f7867f5_21.png)

1.  **放大字幕**：拖动字幕文本框右下角，向外拉伸即可放大字体。
2.  **调整位置**：将放大后的字幕文本框拖动到视频画面的最下方，这是最符合观看习惯的位置。

完成第一个字幕的调整后，后续识别的字幕会自动应用相同的样式和位置，无需重复操作。

![](img/162d59626237eb79621535076f7867f5_23.png)

## 核心要点与总结

![](img/162d59626237eb79621535076f7867f5_23.png)

本节课中我们一起学习了视频剪辑后半部分的核心操作。

首先，我们明确了添加**转场**的重要性，它能够平滑连接不同视频片段，提升观看体验，是保障**完播率**的关键手段。操作上，我们从素材库中选择3-5秒的转场素材，并确保其比例与主视频一致。

其次，我们掌握了添加**字幕**的方法。利用“识别字幕”功能可以快速生成字幕，随后通过简单的拖动调整其大小和位置，使视频显得更加专业。

![](img/162d59626237eb79621535076f7867f5_25.png)

最后，请记住制作视频必须严格遵循步骤：选择合适的素材（避免过长）、调整统一比例、添加必要转场、配置清晰字幕。整个流程并不复杂，但每一步都关系到视频最终能否通过平台审核并获得良好推荐。
