# 抖音运营实战：P23：11、视频类型1-2 - 人设与口播详解 📚

在本节课中，我们将要学习抖音账号运营中至关重要的两个概念：**人设**的建立与**口播**视频的制作。我们将深入探讨如何通过塑造鲜明的人设来吸引用户，并详细讲解口播这种高效内容形式的具体操作方法。

上一节我们解决了文案问题，本节中我们来看看内容创作的下一个核心环节。很多课程会直接教你涨粉或变现，但从吸引粉丝到实现变现并非一条直线，中间还有一个关键环节：**建立人设**。

建立人设的目的并非为了炫耀，而是为了在众多同类创作者中脱颖而出。例如，有十个人都做装修内容，如果其中一位创作者（比如小王）塑造了一个幽默搞笑的人设，用户就更容易记住他，而忽略其他九位。当用户因为喜欢你这个人而关注你时，他就成为了你的粉丝。这就是人设的价值。

接下来，我们来学习这个知识点，请边听边记。人设主要分为两种类型：**强人设**与**弱人设**。区分清楚这两种人设后，你就能轻松判断任何视频的人设类型。

![](img/22b3f1f7625e9dbf450acf819e5d43ea_1.png)

首先，第一种是**强人设**。强人设是指内容以创作者**个人**为切入点，与创作者自身密切相关。以下是强人设常见的切入点：

![](img/22b3f1f7625e9dbf450acf819e5d43ea_3.png)

*   **个人经验与职业**：分享你的行业经验，例如装修、办公、房产、汽车、餐饮、婚介、法律咨询、电商等领域。
*   **个人兴趣与生活**：分享你的兴趣爱好与生活，例如美食、旅行、手工、宠物、星座占卜、美妆、美容护肤、服装穿搭、游戏、摄影等。

强人设的内容核心是分享与你个人相关的工作、生活和爱好，并且大部分情况下需要**真人出镜**，因为内容与“你”这个人直接绑定。

那么，与之相对的是**弱人设**。弱人设是指内容与创作者自身关系不大，主要目的是为了运营好账号本身。以下是弱人设的例子：

*   **情感语录账号**：发布配乐的情感文字图片或视频，无真人出镜或配音。
*   **影视解说账号**：虽然有人声解说电影，但核心是电影内容，与解说者本人特质关联较弱。

弱人设的核心是内容本身（如文案、影视素材）吸引人，而非创作者的个人魅力。

在清晰区分了强弱人设之后，我们才能更好地规划内容。接下来，我们聚焦于一种非常适合强人设、且操作简便的内容形式：**讲故事**与**口播**。

只要是与你个人相关的经历或知识，都可以作为视频素材。我们先从“讲故事”开始。你可以分享客户故事、从业经历或人生感悟。故事本身不要求100%真实，可以从文案平台寻找有趣的故事进行分享。

而讲故事最直接的方式，就是用嘴巴讲出来，这被称为**口播**。口播视频有多种分类，适用于不同领域：

*   **故事型口播**：如讲述情感、人生经历的故事，目的是让观众觉得有趣、有共鸣。
*   **知识型口播**：如分享育儿、金融、养生、装修、房产、汽车等领域的知识，目的是让观众在几十秒内获得“收获感”，觉得学到了东西。

口播形式非常适合有专业知识或想树立行业形象的创作者，例如法律、保险、金融、装修、汽车等领域的从业者。它的优势在于**拍摄简单**：只需准备好文案，用手机支架固定手机即可录制。

针对常见的顾虑，这里有两个关键解答：
1.  **口才不好能做口播吗？**  
    **可以**。口播是依据提前准备好的稿子进行录制，并非临场发挥。大部分口播者的流畅表达都源于充分准备。
2.  **普通话不好能做口播吗？**  
    **可以**。如果你的目标客户是同城或有地域属性，使用方言可能比普通话更亲切、更有趣，效果更好。同时，你可以通过添加字幕来确保所有观众都能理解内容。

本节课中我们一起学习了抖音账号运营中“人设”的重要性及其两种类型（强人设与弱人设），并深入探讨了“口播”这一高效内容形式的定义、分类与实操要点。记住，清晰的人设是连接内容与粉丝的桥梁，而口播则是表达个人专业与故事的有力工具。在接下来的课程中，我们将继续学习其他视频类型与拍摄技巧。
