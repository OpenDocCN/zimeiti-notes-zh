在本节课中，我们将要学习什么是“主题IP”，以及如何为自己的抖音账号策划一个清晰、有吸引力的主题。主题IP是账号内容的核心框架，能有效提升粉丝粘性和内容辨识度。

![](img/fdd65addd898314ba0865987d0819ed7_4.png)

## 什么是主题IP？

![](img/fdd65addd898314ba0865987d0819ed7_6.png)

上一节我们介绍了内容定位，本节中我们来看看构成内容吸引力的关键——主题IP。简单来说，主题IP就是贯穿你一系列视频的、具有辨识度的核心主题或系列名称。

为了让大家更好地理解，我们接下来再看两个视频案例。



![](img/fdd65addd898314ba0865987d0819ed7_0.png)

![](img/fdd65addd898314ba0865987d0819ed7_8.png)

**案例一：家庭生活类**
视频标题为“挑战惹老婆生气的第21天”。内容为博主与怀孕的妻子嬉戏打闹的日常。

![](img/fdd65addd898314ba0865987d0819ed7_10.png)

**案例二：情感教学类**
视频标题为“小姨恋爱课”。内容为以“小姨”人设，在一块黑板前讲解当女生说“我没有你想象的那么好”时，应如何巧妙回复（答案是回复“明白了”并后撤）。



这两个视频我们都看过了。



你们有没有发现它们有一个比较共同或相似的特点？

首先，我们来看“挑战惹老婆生气的第21天”这个视频。如果你不看标题文字，只看他和他老婆嬉戏打闹的内容，这似乎只是一个普通的日常视频，镜头和画面也没有特别之处。但是，他加上“挑战惹老婆生气的第21天”这个标题后，效果就不同了。这个标题给了你一个主题，它引导着你去想：这是第21天，是不是还有第20天？是不是还有更有趣的？这个没趣没关系，我想看看他第20天、第19天是怎么惹老婆生气的。这就是他的一个主题。也就是说，他现在有21天，接下来还可能有22天、23天、24天。

同理，之前比较火的“阿拉蕾的装修日记”也是如此。博主围绕“在北京买小院并装修”这件事，衍生出了无数期作品。这就是他的主题。

我们案例二中的“小姨恋爱课”也是一样的道理。



![](img/fdd65addd898314ba0865987d0819ed7_12.png)

“小姨恋爱课”这个策划更绝。她大概发布了20多条视频，涨粉非常厉害。因为她的人设和整个账号策划得非常好。

首先，她的主题就是“小姨恋爱课”。其次，这个主题对目标用户很有用：有些直男不太会追女生；有些人对感情话题特别感兴趣；有些人则对视频中的问题有共鸣，觉得“这个问题我好像遇到过”。她一举三得。而且，这类视频拍起来也非常简单，一块小黑板就可以拍很多期。

那么我们现在主要关注的就是“主题IP”。我们会发现，没有明确主题IP的账号（比如单纯唱歌的账号），用户可能觉得歌好听点个赞就走了，不会关心他昨天唱了什么、明天要唱什么，无法形成持续的吸引力。因此，主题IP的存在非常重要。



## 如何策划主题IP？

那么主题IP我们怎么去取呢？这里也给到大家方法。大家在明确了商业定位（比如选择做医美行业）后，不要着急开拍。内容定位里还有很多东西需要填充，主题IP就是第一个需要填充的关键要素。

![](img/fdd65addd898314ba0865987d0819ed7_14.png)

以下是策划主题IP的方法与原则，核心公式可以概括为：

**主题IP = 前缀 + 垂直领域**

可能单看概念有些抽象，接下来我们通过具体例子来解释，大家就明白了。



以下是几种常见且有效的主题IP命名公式，每种公式都配有实例：

**1. 名字 + 垂直领域**
*   **公式**：`博主名 + 的 + 领域内容`
*   **例子**：某某的北漂日记（第02集）、老韩摄影教学（第12期）、猪猪小姐姐的撩汉指南之XXX。

**2. 人群 + 垂直领域**
*   **公式**：`目标人群 + 的 + 领域内容`
*   **例子**：别人家男朋友的拍照技巧（第三期）、宝妈育儿小技巧之XXX、新手开店避坑指南。

**3. 地域 + 垂直领域**
*   **公式**：`地点/范围 + 的 + 领域内容`
*   **例子**：西藏自驾之旅（第50天）、环球旅居日本站、广州美食之火锅篇。

**4. 场景 + 垂直领域**
*   **公式**：`特定场景 + 的 + 领域内容`
*   **例子**：办公室迷惑行为之为什么总有人脚臭、儿童房风格装修指南、婚礼现场的创意小心思。



这些就是构思主题IP的一些实用方法。大家可以记住“前缀+垂直领域”这个核心公式，未来在为账号策划内容主题时，方向就会明确很多。

## 总结

![](img/fdd65addd898314ba0865987d0819ed7_16.png)

本节课中我们一起学习了“主题IP”的重要性与策划方法。主题IP是你内容系列的灵魂，它像一根线，能把零散的视频珍珠串成一条有价值的项链，吸引用户持续关注。记住，在开始拍摄前，先运用“前缀+垂直领域”的公式，为你的账号找到一个清晰、可持续、有吸引力的核心主题。
