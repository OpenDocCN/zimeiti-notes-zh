# 小红书运营实战：P28：小红书账号类型与官方广告投放指南

在本节课中，我们将学习小红书账号的三种主要类型，并对比分析自建账号运营与官方广告投放两种推广方式的区别与联系。课程将帮助你理清思路，为你的品牌或业务选择最合适的推广路径。

## 账号类型解析

上一节我们介绍了小红书运营的基础概念，本节中我们来看看小红书平台上的账号具体分为哪几类。理解账号类型是进行精准运营的第一步。

小红书账号主要分为以下三类：

1.  **企业号**：核心作用是树立官方品牌形象。企业号拥有官方认证标识，并享有平台赋予的特权，例如展示品牌名称、品牌地址等。这有助于品牌更规范、更权威地开展业务。
2.  **个人号**：这是目前大多数用户运营的账号类型。个人号的特点是**灵活性高**。运营者可以是品牌员工，也可以是品牌消费者。个人号能更好地塑造个人品牌特色，由品牌相关人物或员工出镜也显得更为自然。
3.  **引流号**：这类账号在之前的课程中也有提及，俗称“炮灰号”。引流号可以创建多个，其作用通常较为单一，例如用于特定活动引流，用后即可弃置，或在紧急情况下作为备用小号使用。这类账号通常没有精细的个性规划。

综上所述，目前常见的品牌推广方式，主要是通过企业号、个人号或引流号自行发布笔记进行引流。

## 企业号矩阵运营建议

了解了账号的基本分类后，我们来看看如何协同使用这些账号，构建一个有效的推广矩阵。以下是打造企业号矩阵的一些具体建议：

*   **积极参与品牌活动**：品牌的企业号发起活动（如试用、团购）时，可以动用多个小号（个人号或引流号）去积极参与，增加活动热度与真实感。
*   **强化个人号品牌属性**：运营个人号时，应在笔记内容中多次暗示品牌的定位。例如，如果品牌定位是“都市职场年轻人”，那么个人号的内容风格、话题选择都应围绕这一属性展开。
*   **控制产品发布频率**：个人号在发布品牌产品相关内容时，需注意频率。一个较好的比例是 **营销内容：非营销内容 = 3：7**。即30%的内容用于直接推广，70%的内容用于种草分享、知识科普等，以维持账号的可信度与吸引力。
*   **巧用引流号评论**：当企业号发布内容需要引流时，可以安排引流号在评论区进行互动或引导，这是一种相对隐蔽且有效的辅助引流方式。

## 官方广告投放介绍

![](img/056eb629bb2d18605cb139a9b8dbab95_5.png)

除了自建账号矩阵进行运营推广，小红书平台也提供了官方的广告投放渠道。接下来，我们将介绍另一种重要的推广方式：小红书官方广告。

![](img/056eb629bb2d18605cb139a9b8dbab95_7.png)

![](img/056eb629bb2d18605cb139a9b8dbab95_9.png)

小红书官方广告主要指 **小红书CPC广告**，即按点击付费的竞价广告。广告主通过竞价购买关键词或用户兴趣标签，让自己的广告内容获得更靠前的展示位置。

![](img/056eb629bb2d18605cb139a9b8dbab95_11.png)

官方广告的申请与使用流程，可以在小红书官方网站的“业务合作”板块中找到“推广合作”入口。需要注意的是，申请投放官方广告有一个前提：**必须先认证一个企业号**。

![](img/056eb629bb2d18605cb139a9b8dbab95_13.png)

企业号的入驻流程同样在官网有明确指引，通常需要提交相关资质文件并支付审核认证费用。认证成功后，普通账号即可升级为官方认可的企业号。

拥有企业号后，即可申请投放信息流广告。官方广告主要分为两大类：

1.  **信息流广告**：当用户浏览小红书首页推荐流时，系统会不定期间隔插入广告内容。这种广告形式与原生内容混合，曝光量较大。
2.  **搜索广告**：当用户使用关键词进行搜索时，与关键词匹配的广告会出现在搜索结果页的特定位置。例如，用户搜索“帽子”，系统可能展示与之搭配的“大衣”或“外套”的广告。广告的展示排名与出价和内容质量相关。

目前，信息流广告的常见展示位序有第3位、第13位、第23位等。关于广告投放的更多细节与常见问题，可以在小红书广告平台的官方帮助文档中查阅。

需要指出的是，官方广告投放通常需要一定的资金预算。在资金允许的情况下，与官方合作投放竞价广告是获取精准流量的有效方式，因为其流量直接由平台算法进行推荐和匹配。
