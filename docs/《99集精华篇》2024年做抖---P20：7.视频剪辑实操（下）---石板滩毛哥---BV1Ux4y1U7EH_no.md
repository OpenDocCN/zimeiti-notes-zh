# 抖音暴力起号实操教程：P20：7.视频剪辑实操（下） 🎬

在本节课中，我们将继续学习视频剪辑的核心步骤，重点掌握如何为视频片段添加转场以及如何添加字幕。这些操作对于提升视频的流畅度和专业性至关重要。

上一节我们介绍了视频比例和背景的设置，本节中我们来看看如何让多个视频片段之间衔接得更自然。

![](img/769b46b37e9c9d59958c8e588446511f_1.png)

## 添加转场

![](img/769b46b37e9c9d59958c8e588446511f_3.png)

我们的视频通常由多个片段组成，每个片段时长约10秒。这些片段之间需要有自然的衔接，因此必须添加转场效果。

![](img/769b46b37e9c9d59958c8e588446511f_5.png)

![](img/769b46b37e9c9d59958c8e588446511f_7.png)

转场的作用是明确告知用户，当前内容已切换到另一个不同的片段。如果不添加转场，观众在观看时可能会感到突兀，甚至退出视频确认自己是否点错了内容。一旦用户退出，视频的**完播率**就会直线下降，而完播率直接影响视频的推荐量和最终收益。

![](img/769b46b37e9c9d59958c8e588446511f_9.png)

![](img/769b46b37e9c9d59958c8e588446511f_11.png)

以下是添加转场的具体步骤：

![](img/769b46b37e9c9d59958c8e588446511f_13.png)

1.  在视频编辑轨道上，找到并点击右侧的 **“+”** 号按钮。
2.  在弹出的菜单中，选择 **“素材库”**。
3.  在素材库的 **“热门”** 分类中，可以找到大量转场素材，例如搞笑片段或动态效果。
4.  选择转场时，注意时长。避免选择过长的转场（如16秒），观众通常没有耐心看完。建议选择 **3~5秒** 的转场素材，3秒最为合适。
5.  选中合适的转场后，点击添加。

![](img/769b46b37e9c9d59958c8e588446511f_15.png)

转场素材添加后，同样需要调整其比例，确保与主视频保持一致（16:9）。选中转场片段，在编辑选项中找到 **“画布模糊”** 功能，选择第一个清晰的模糊背景，即可将其适配为横屏比例。最后，记得点击右下角的对勾确认。

![](img/769b46b37e9c9d59958c8e588446511f_17.png)

## 添加字幕（文案）

视频内容剪辑并衔接好后，下一步是为视频添加字幕，方便观众理解。

![](img/769b46b37e9c9d59958c8e588446511f_19.png)

添加字幕可以大幅提升视频的观看体验和专业性。以下是添加字幕的操作流程：

![](img/769b46b37e9c9d59958c8e588446511f_21.png)

1.  在编辑界面左下角，找到并点击 **“文字”** 选项。
2.  点击 **“新建文本”** 右侧的 **“识别字幕”** 功能。
3.  点击 **“开始匹配”**，软件会自动识别视频中的语音，并生成对应的字幕文本。
4.  字幕生成后，可以进行调整。拖动字幕文本框右下角，可以放大字幕。
5.  将放大后的字幕文本框拖动到视频画面的最下方，这是最舒适观看位置。调整好第一句字幕后，后续字幕会自动应用相同的样式。

## 核心要点总结

![](img/769b46b37e9c9d59958c8e588446511f_23.png)

本节课中我们一起学习了视频剪辑后半部分的核心操作：

![](img/769b46b37e9c9d59958c8e588446511f_25.png)

1.  **添加转场**：在视频片段间插入3-5秒的转场素材，使用 **素材库 -> 热门** 中的资源，并确保其比例为 **16:9**。
2.  **添加字幕**：使用 **文字 -> 识别字幕** 功能自动生成字幕，并调整其大小和位置至视频底部。

![](img/769b46b37e9c9d59958c8e588446511f_27.png)

请严格按照上述步骤操作。在寻找素材时，务必注意时长，过长的素材可能导致平台审核不通过。整个流程并不复杂，但每一步都关系到视频的最终质量和传播效果。
