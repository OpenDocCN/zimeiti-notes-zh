# 自媒体运营课程：P31：剪映核心剪辑功能——蒙版功能 🎬

在本节课中，我们将要学习剪映软件中的一个核心剪辑功能——**蒙版**。蒙版是一种遮罩功能，常用于隐藏视频中的特定部分或创造独特的转场效果，能让你的视频看起来更加精致和专业。

上一节我们介绍了剪映的基础操作，本节中我们来看看如何运用蒙版功能来提升视频的视觉效果。

## 什么是蒙版？

蒙版是视频剪辑中常用的**遮罩**或**隐藏**功能。它的核心作用是**选择性显示或隐藏视频画面的某一部分**。例如，当你想遮挡视频中的某些元素不让观众看到时，就可以使用蒙版功能将其隐藏起来。

在剪映中，我们正是通过蒙版功能来实现各种创意效果的。

## 如何在剪映中使用蒙版

![](img/02cceccf78072404b54554afa9cf9690_16.png)

接下来，我们实际操作一下，展示如何在视频中添加和使用蒙版功能。

首先，点击“开始创作”，导入几张图片素材。

![](img/02cceccf78072404b54554afa9cf9690_18.png)

导入后，先预览一下默认的转场效果。此时视频是平铺直叙的静态转场，没有特殊效果。

为了添加类似案例中的蒙版效果，我们需要进行以下操作：

![](img/02cceccf78072404b54554afa9cf9690_20.png)

1.  **选中素材**：在时间轴上选中第一张图片。
2.  **找到蒙版选项**：在底部二级菜单中，找到并点击“**蒙版**”按钮。
3.  **选择蒙版类型**：系统会显示多种蒙版形状供你选择。

![](img/02cceccf78072404b54554afa9cf9690_22.png)

以下是几种常见的蒙版类型及其效果：

![](img/02cceccf78072404b54554afa9cf9690_24.png)

*   **线性蒙版**：这是最基础的蒙版。它会以一条直线为界，隐藏画面的一部分。你可以拖动控制线来调整显示与隐藏的区域范围。例如，向上拖动得越多，被遮盖（隐藏）的部分就越多。之前流行的“遇见当年”类视频就大量使用了线性蒙版来擦除中间多余的部分。
    *   **效果描述**：`画面被一条直线分割，一侧显示，另一侧隐藏。`
*   **镜像蒙版**：它以中心线为轴，向两侧同时进行羽化过渡。你可以拖动箭头调整羽化的强度和范围，使画面融合更自然。
    *   **效果描述**：`画面沿中心线对称地、柔和地向两侧渐隐。`
*   **图形蒙版（如圆形、星形）**：原理相同，让画面只在特定图形区域内显示。通过调整图形的大小、位置和羽化值，可以创造出圆形、心形、五角星形等创意转场。
    *   **操作代码示意**：`选中素材 -> 点击“蒙版” -> 选择“星形” -> 调整位置与羽化`

![](img/02cceccf78072404b54554afa9cf9690_26.png)

添加蒙版后，视频画面已经发生了变化。但案例中的效果是动态的，这需要结合其他功能。

## 制作动态蒙版转场效果

如果我们想制作案例中那种具有动感的双重人影或图形转场，可以结合“画中画”和“动画”功能。

以下是制作步骤：

![](img/02cceccf78072404b54554afa9cf9690_31.png)

![](img/02cceccf78072404b54554afa9cf9690_33.png)

1.  **复制图层**：选中已添加蒙版的素材，点击“复制”。然后通过“画中画”功能，将复制的图层导入并叠加在原视频上方。
2.  **为画中画添加蒙版**：选中画中画图层，再次进入“蒙版”功能。这次可以选择不同的形状，例如“星形”，并调整其位置和羽化值。
3.  **添加动画**：为了让效果动起来，可以为画中画图层添加入场动画。例如，在“动画”选项中选择“向右甩入”。
4.  **重复操作**：对其余素材重复以上步骤，为每个画中画图层设置不同的图形蒙版和动画。

![](img/02cceccf78072404b54554afa9cf9690_35.png)

完成以上操作后预览视频，就能看到具有动态重叠效果的绚丽转场了。这正是运用蒙版功能制作转场特效的典型方法。

![](img/02cceccf78072404b54554afa9cf9690_37.png)
