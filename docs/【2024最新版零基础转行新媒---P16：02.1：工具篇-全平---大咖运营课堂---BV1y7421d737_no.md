# 新媒体运营教程：P16：02.1：工具篇-全平台短视频文案一键提取 📝

在本节课中，我们将学习一个非常实用的运营工具，它能帮助我们从各大短视频平台一键提取视频文案，从而极大地提高内容创作和素材整理的效率。

上一节我们介绍了运营的基础概念，本节中我们来看看如何利用工具高效获取文案素材。在运营工作中，我们常常会看到优秀的短视频，并希望借鉴或分析其文案内容。传统方法可能需要下载视频再用剪辑软件提取，过程繁琐且不适用于所有平台。

## 工具介绍与优势 🛠️

今天分享的工具是一个在线网页工具，它支持多个主流平台。你只需将目标短视频的链接粘贴到工具中，即可在线一键提取出完整的文案内容。

![](img/2ebeb57c5d780412045885a6417a3c35_1.png)

该工具的核心优势在于：
*   **全平台支持**：可处理抖音、快手、小红书、B站、视频号等平台的视频链接。
*   **在线操作**：无需下载视频或安装额外软件，直接在网页完成。
*   **高效准确**：提取速度快，文案识别准确率高。
*   **附加功能**：提供一键复制和格式化排版功能，使文案更易于阅读和修改。

## 操作演示：提取抖音视频文案 📱

![](img/2ebeb57c5d780412045885a6417a3c35_7.png)

以下是使用该工具提取抖音视频文案的具体步骤。

![](img/2ebeb57c5d780412045885a6417a3c35_9.png)

1.  **打开工具网页**：访问工具的在线界面。
2.  **复制视频链接**：在抖音APP中找到目标视频，点击分享按钮复制链接。
3.  **粘贴并提取**：将复制的链接粘贴到工具输入框中，点击“提取文案”按钮。
4.  **获取结果**：稍等片刻，工具界面便会显示出识别出的完整文案。

![](img/2ebeb57c5d780412045885a6417a3c35_11.png)

例如，提取一个情感语录视频的文案，结果可能如下：
> 听完你就释怀了，村上春树说，我动了离开你的念头。不是因为你不好，也不是因为不爱了……

![](img/2ebeb57c5d780412045885a6417a3c35_13.png)

提取出的文案可以一键复制，或使用“格式化”功能，使其按照字幕段落的形式清晰排列。

## 操作演示：提取B站视频文案 🖥️

该工具同样适用于B站（哔哩哔哩）等中长视频平台。

1.  **复制B站视频链接**：在B站播放页复制浏览器地址栏中的链接。
2.  **执行提取操作**：将链接粘贴到工具中，点击提取。
3.  **查看文案**：工具同样能快速准确地识别出视频的旁白或字幕文案。

例如，一个关于直播电商的教学视频，可能提取出如下文案：
> 新手做直播电商，怎么才能打造出直播爆品？在直播销售的不同阶段……
