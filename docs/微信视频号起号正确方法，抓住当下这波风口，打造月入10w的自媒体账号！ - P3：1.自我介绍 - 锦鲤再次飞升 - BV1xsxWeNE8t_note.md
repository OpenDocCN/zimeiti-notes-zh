## 主流短视频平台分析

![](img/9bd9c2cb6412219f1b3374859ad88082_13.png)

![](img/9bd9c2cb6412219f1b3374859ad88082_14.png)

当前，最具代表性的短视频平台主要有三个：**抖音**、**快手**和**微信视频号**。

这三个平台都能赚钱。但对于新手而言，我强烈建议只选择其中一个平台深入钻研。原因很简单，还没学会走，就不要想着跑。

那么，作为一个新手，你认为在哪个平台最有可能赚到第一桶金？
*   认为在**抖音**能赚钱的，请打数字1。
*   认为在**快手**能赚钱的，请打数字2。
*   认为在**视频号**能赚钱的，请打数字3。
*   不知道如何选择的，请打数字4。

如果你打了“123”或“1234”，那么你在互联网上很可能一分钱都赚不到。因为你什么都想做，最终往往什么都抓不住。学习，首先要学的是思维方式。

## 为何视频号是新手的最佳选择

![](img/9bd9c2cb6412219f1b3374859ad88082_16.png)

![](img/9bd9c2cb6412219f1b3374859ad88082_17.png)

![](img/9bd9c2cb6412219f1b3374859ad88082_18.png)

我不再卖关子，直接给出结论：**新手现阶段千万不要再做抖音，而必须抓住视频号这个风口。**

![](img/9bd9c2cb6412219f1b3374859ad88082_19.png)

**为什么不建议做抖音？**
抖音的商业化已经非常严重。你会发现，刷10个视频，可能有7-8个都在带货，甚至刷着刷着就会弹出广告。抖音自2017年发展至今已8年，有无数人涌入其中赚钱，竞争异常激烈。

**为什么不建议做快手？**
快手比抖音出现得更早，其竞争激烈程度同样很高，对新手并不友好。

**为什么必须选择视频号？**
视频号拥有独特的优势，即“人无我有”。对于新手而言，视频号正处于红利期，是必须抓住的风口。
