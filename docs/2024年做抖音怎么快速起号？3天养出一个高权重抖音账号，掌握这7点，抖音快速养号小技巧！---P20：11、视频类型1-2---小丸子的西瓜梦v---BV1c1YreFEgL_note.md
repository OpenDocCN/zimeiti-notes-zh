# 抖音运营：P20：11、视频类型1-2 - 人设与口播 📚

在本节课中，我们将学习抖音账号运营中至关重要的两个概念：**人设**的建立与**口播**视频的制作。我们将深入探讨如何通过塑造人设来让用户记住你，并详细介绍口播这种高效的内容形式。

---

上一节我们解决了文案问题，本节中我们来看看内容创作的核心环节。从涨粉到变现并非一条直线，中间有一个关键环节：**建立人设**。

建立人设的目的并非为了炫耀，而是为了在众多同类创作者中脱颖而出。例如，有十个人都做装修内容，如果其中一位创作者塑造了一个幽默搞笑的人设，用户就更容易记住他，而忽略其他九位。当用户喜欢上你这个人时，他们自然就成为你的粉丝。

接下来，我们来学习这个知识点。人设主要分为两种：**强人设**与**弱人设**。理解这两种人设后，你就能轻松判断任何视频的人设类型。

**强人设**是指内容以创作者**个人**为切入点，与创作者自身密切相关。这包括分享个人行业经验、专业知识或个人兴趣爱好。

![](img/8d18d142880bdbed5246d1f377dc7179_1.png)

以下是强人设常见的切入点领域：

![](img/8d18d142880bdbed5246d1f377dc7179_3.png)

*   **行业经验类**：装修、办公、房产、汽车、餐饮、婚介、法律咨询、电商等。
*   **兴趣爱好类**：美食、旅行、手工、宠物、星座占卜、美妆、护肤、服装穿搭、游戏、摄影等。

![](img/8d18d142880bdbed5246d1f377dc7179_1.png)

![](img/8d18d142880bdbed5246d1f377dc7179_3.png)

强人设的核心是分享自己的生活、工作或爱好，并且大部分需要**真人出镜**，因为内容与“你”这个人直接相关。

**弱人设**则与创作者自身关系不大，主要目的是运营好账号本身。例如，情感语录视频（配音乐和文字，无真人出镜）或影视解说视频（虽有配音，但核心是电影内容，与解说者个人无关），都属于弱人设。

区分清楚人设类型后，我们才能更好地规划内容。接下来，我们看看如何将与你个人相关的内容转化为视频。

内容创作的形式多种多样，我们先从**讲故事**开始。你可以分享客户故事、从业经历或人生感悟。故事不要求完全真实，可以从文案平台寻找有趣的故事进行分享。

讲故事最直接的方式就是用嘴巴讲，这被称为**口播**。口播涵盖多种领域，形式多样。

以下是常见的口播分类：

*   **故事型口播**：讲述有趣的故事，目的是吸引听众，让人觉得有意思。
*   **知识型口播**：分享育儿、金融、养生、装修、房产、汽车等领域的专业知识，目的是让观众在几十秒内获得“收获感”，觉得学到了东西。

口播是一种适用范围极广的视频形式。无论你从事法律、保险、金融，还是想科普装修、汽车、买房知识，都可以采用口播。它的优势在于拍摄简单：一部手机、一个支架即可。

对于常见问题，解答如下：
*   **口才不好能做口播吗？** 可以。口播是**提前准备好稿子**再录制，并非临场发挥。
*   **普通话不好能做口播吗？** 可以。如果你的目标客户是同城或有地域属性，使用**方言**可能效果更好，显得亲切或有趣。同时，可以配上字幕确保观众理解。

口播是一种非常实用且易于上手的内容形式，在后续课程中，我们会详细教学如何拍摄。
