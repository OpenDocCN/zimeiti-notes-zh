# 抖音运营教程：P33：扶持加权与降权限流规则详解 🎯

在本节课中，我们将学习抖音平台关于内容扶持加权与降权限流的核心规则。理解这些规则对于优化视频内容、提升账号权重至关重要。

上一节我们介绍了账号权重的分类，本节中我们来看看哪些内容会受到平台的扶持，哪些内容则可能导致降权或限流。

![](img/8bc638020df173c533d2af622fe0a4e8_1.png)

## 扶持加权的内容标准 ✅

平台会优先扶持符合以下技术标准和内容规范的原生视频。

![](img/8bc638020df173c533d2af622fe0a4e8_3.png)

以下是获得扶持加权的关键要求列表：

*   **视频格式**：必须为竖版移动端原生视频，即 **9:16** 的格式。这是抖音平台的标准视频格式。
*   **画面质量**：要求画面平稳，作品清晰度高。
*   **音频质量**：视频中不应有过多的杂音或噪音。

![](img/8bc638020df173c533d2af622fe0a4e8_5.png)

这些标准较为基础，平台旨在鼓励创作高质量的原生内容。相关细节会整理成资料页供课后查阅。

## 降权限流的内容禁区 🚫

另一部分需要重点注意的是会被平台降权或限流的内容。发布以下类型的内容将非常危险。

以下是会导致降权限流的主要内容类型列表：

*   **违规信息**：视频内容涉及吸烟、暴力或过度宣扬金钱等信息。
*   **外部引流**：在视频中留下微信、微博、淘宝等外部平台的联系方式。长时间出现商品Logo、手机号或二维码也属于违规行为。
    > 注：二维码并非绝对禁止，但出现时间最长不要超过三秒。有时审核严格，即使出现一秒也可能被驳回。
*   **硬性广告**：进行明显的营销广告、推销或引流卖货行为。

## 重点案例：好物推荐类内容的误区 📦

在“降权限流”的规则中，需要特别注意“无口播拆箱或好物推荐”这一条。这对于做好物推荐或带货账号的同学是一个常见陷阱。

以下是两个具体的错误案例及其解析：

*   **案例一：直接标价**。有同学在视频画面中直接写上“九块九”等价格信息。这种直接涉及金钱交易的赤裸裸卖货行为，会导致小黄车功能无法通过审核。
*   **案例二：纯展示无口播**。有同学拍摄玩具拆箱或产品展示视频，但全程没有一句口播解说，仅使用背景音乐。这种纯展示内容也容易被系统判定为低质营销内容，导致小黄车被下架。

**正确的做法是采用“口播+软性推荐”的形式**。例如，在展示产品时，可以自然地口播介绍：“我儿子玩这个玩具，四个小时都没停下来。” 这样既介绍了产品优点，又避免了硬广嫌疑，更容易通过审核。

简单来说，**切忌不说话直接展示产品**。其他如直接口播广告、念联系方式等硬广形式，在抖音平台是行不通的。

本节课中我们一起学习了抖音平台内容扶持与限流的规则。核心在于：**制作高清、竖版的原生视频可获得扶持；而涉及违规信息、外部引流和硬性广告的内容则会受到限制**。特别是对于带货账号，务必通过口播进行软性介绍，避免无声拆箱或直接标价。掌握这些规则，能帮助你更安全、有效地进行内容创作与运营。
