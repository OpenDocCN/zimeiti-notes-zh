# 小红书运营：P5：小红书上架流程实操 🛒

在本节课中，我们将学习如何将其他电商平台的商品，通过一键搬家软件上架到自己的小红书店铺。整个过程涉及软件配置、商品信息审核与优化等关键步骤。

---

![](img/8ae80a7580d3e3ea195cf96fbe8db21c_1.png)

上一节我们介绍了选品策略，本节中我们来看看具体的上架操作流程。首先需要理解两个核心概念：**SPU** 和 **SKU**。

![](img/8ae80a7580d3e3ea195cf96fbe8db21c_3.png)

![](img/8ae80a7580d3e3ea195cf96fbe8db21c_5.png)

*   **SPU** 可以理解为一个商品款型，即一个独立的商品链接。
*   **SKU** 是库存量单位，指一个款型下由不同属性（如颜色、尺码）组成的最小销售单元。例如，一件衣服有红、白两种颜色，每种颜色有S、M、L三个尺码，那么这个款（SPU）下就有 `2 * 3 = 6` 个SKU。

![](img/8ae80a7580d3e3ea195cf96fbe8db21c_7.png)

小红书平台对店铺每日上传的SKU总数有限制，上限为 **5000个**。对于SKU较多的非标品类目，通常一天能上传两三百个款。

上架流程需要使用一键搬家软件。目前常用的有“摸鱼助手”和“欢乐逛”等，它们功能大同小异。下面以“欢乐逛”为例进行演示。

以下是进入服务市场并启用软件的步骤：
1.  登录小红书商家后台，点击“服务”。
2.  进入“服务市场”。
3.  在市场中搜索“一键搬家”相关服务，找到“欢乐逛”。
4.  点击软件，通常有免费试用期（例如一周），点击“免费试用”或“订购”进行启用。

启用后，在我的服务中找到“欢乐逛”并点击使用。软件主界面通常有“复制商品”或类似功能。

![](img/8ae80a7580d3e3ea195cf96fbe8db21c_9.png)

以下是配置搬家软件的关键设置项：
*   **商品链接**：粘贴你要搬运的其他平台（如拼多多）商品链接。
*   **抓取选项**：勾选“专用通道”、“抓取SKU”等，以确保信息完整。
*   **标题**：可选择“使用原标题”。
*   **价格**：建议设置为原价的 **2倍** 上传。对于低客单价商品（十几元），加价倍数可以更高（如3-5倍），以保证利润。
*   **图片处理**：“第一章主图不处理”。**详情图建议不过滤**，搬运后需人工检查详情页中是否带有原平台（如拼多多）的广告或Logo，如有则必须删除或涂抹，否则会违规。
*   **运费模板**：选择你已设置好的“非偏远包邮”模板。
*   **物流方案**：选择你已建立的物流方案。
*   **发货时间**：建议设置为“三天内发货”，这样有更充裕的时间处理订单，即使从其他平台（如淘宝）调货也来得及。
*   **上架状态**：先选择“**否**”（即不上架），待人工审核完毕后再手动上架。
*   **保存配置**：配置完成后，点击“保存当前配置模板”，方便后续批量使用。

![](img/8ae80a7580d3e3ea195cf96fbe8db21c_11.png)

![](img/8ae80a7580d3e3ea195cf96fbe8db21c_13.png)

配置完成后，点击“开始上货”。软件抓取商品后，需进行关键的人工审核与优化。

首先，检查并修正商品类目。软件可能匹配不准确，需手动选择正确的小红书类目（例如，将“裤子”归到“服饰鞋包 > 童装 > 儿童裤子”下）。正确选择后，可点击“记住并应用到相应分类”，方便后续同类商品自动归类。

![](img/8ae80a7580d3e3ea195cf96fbe8db21c_15.png)

然后，进入商品编辑页面，进行详细检查：
1.  **标题**：检查是否有违规的“极限词”（如最、第一）或“品牌词”。
2.  **主图**：检查是否有其他平台的Logo或水印。
3.  **详情页**：**必须逐张图片检查**，删除或涂抹掉所有来自原平台的广告、网址、品牌信息。
4.  **商品属性**：核对软件填写的属性（如季节、风格）是否准确，根据实际情况调整（例如，加绒裤子应将“季节”属性改为“冬季”）。

![](img/8ae80a7580d3e3ea195cf96fbe8db21c_17.png)

![](img/8ae80a7580d3e3ea195cf96fbe8db21c_19.png)

所有信息确认无误后，提交修改。在商品管理列表中，勾选已审核的商品，进行“批量上架”操作。看到“上架成功”提示，即表示整个流程完成。

---

![](img/8ae80a7580d3e3ea195cf96fbe8db21c_21.png)

本节课中我们一起学习了小红书无货源电商的上架全流程。核心要点包括：理解SPU/SKU概念、使用一键搬家软件、合理设置加价与发货时间、以及**最重要的人工审核环节**——务必仔细检查标题、主图、详情页，去除任何原平台信息，确保符合小红书规范。熟练此流程，是高效、合规运营店铺的基础。
