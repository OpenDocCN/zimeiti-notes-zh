# 亚马逊运营：P1：高效追踪站外促销订单的实用技巧

在本节课中，我们将学习一个快速、精准地追踪亚马逊站外推广活动订单效果的方法。通过掌握这个技巧，你可以清晰了解每次站外促销带来的实际销量，从而更有效地评估推广效果。

## 核心操作步骤

上一节我们概述了追踪站外订单的重要性，本节中我们来看看具体的操作流程。以下是追踪站外促销订单的核心步骤。

1.  登录亚马逊卖家后台，进入“广告”模块。
2.  在广告菜单中，找到并点击“管理促销”。
3.  在促销列表里，定位到你想要查询的**站外促销活动名称**或**追踪编码**。
4.  在促销信息表格的**倒数第二列**，你可以直接看到该站外活动产生的订单数量。

## 查询具体订单详情

掌握了订单总数后，你可能会想进一步分析是哪些订单。以下是查询具体订单编号的方法。

1.  在“管理订单”页面，使用日期筛选器，选择站外促销活动进行的日期范围。
2.  在筛选条件中，找到“促销ID”筛选选项。
3.  输入对应的站外促销活动**追踪编码**进行筛选。
4.  页面将列出所有通过该站外促销活动产生的具体订单，你可以查看其订单号等详细信息。

本节课中我们一起学习了如何通过亚马逊后台的“管理促销”功能快速查询站外订单总数，以及如何在“管理订单”中通过筛选“促销ID”来定位具体订单。这个方法能极大提升你分析站外推广效果的效率。
