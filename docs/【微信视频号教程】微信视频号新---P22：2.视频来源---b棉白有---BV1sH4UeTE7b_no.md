# 微信视频号教程：P22：视频来源处理

## 概述
在本节课中，我们将学习如何为视频号带货准备素材，核心内容包括搬运短视频、去除视频水印以及提取视频文案。这些是进行视频混剪和二次创作的基础步骤。

---

## 搬运与去水印操作

![](img/cd01f02d886bd61096a508fcd2bf2c7c_1.png)

上一节我们介绍了视频号带货的基本思路。本节中，我们来看看如何获取并处理原始视频素材。

操作的第一步是找到目标视频并获取其链接。我们以抖音平台为例。

以下是具体操作步骤：
1.  打开抖音，搜索并找到你想要的视频。
2.  点击视频分享箭头，选择“复制链接”。

获取链接后，我们需要使用工具去除视频自带的水印（如抖音Logo）。微信小程序中有许多此类工具。

以下是使用“青抖去水印”小程序的步骤：
1.  在微信搜索“青抖去水印”并打开小程序。
2.  点击“去水印”功能。
3.  粘贴刚才复制的视频链接。
4.  点击“一键去水印”。
5.  根据提示观看广告后，即可将去水印的视频保存到手机相册。

**核心操作公式**：`复制视频链接 -> 粘贴至去水印工具 -> 保存无水印视频`

![](img/cd01f02d886bd61096a508fcd2bf2c7c_3.png)

直接保存平台视频通常会带有水印。使用去水印工具可以解决这个问题。如果某个视频无法复制链接或保存，只需更换其他视频即可。

## 更便捷的工具推荐

![](img/cd01f02d886bd61096a508fcd2bf2c7c_5.png)

上面我们使用了需要观看广告的工具。接下来，我们介绍一个更便捷的去水印工具：“配音神器Pro”。

![](img/cd01f02d886bd61096a508fcd2bf2c7c_6.png)

该工具的优势在于去水印时无需观看广告。你可以通过截图识别小程序码的方式找到它。

以下是使用“配音神器Pro”去水印的步骤：
1.  识别小程序码，进入“配音神器Pro”。
2.  同样使用“去水印”功能。
3.  粘贴视频链接。
4.  点击“一键去水印”后，可直接保存视频。

**代码示例（操作逻辑）**：
```plaintext
if 使用“青抖”:
    去水印后需观看广告
elif 使用“配音神器Pro”:
    去水印后可直接保存
```

## 视频文案提取方法

处理完视频画面后，我们常常还需要视频中的文案（台词/脚本）用于创作。配音神器Pro同样具备文案提取功能。

以下是提取视频文案的步骤：
1.  在“配音神器Pro”中找到“文案提取”功能。
2.  选择“短视频链接”并粘贴目标视频链接。
3.  点击确定，稍作等待，系统就会自动提取出视频的全部文案。
4.  复制提取出的文案即可使用。

![](img/cd01f02d886bd61096a508fcd2bf2c7c_8.png)

这个功能不仅支持抖音、快手等平台的视频链接，也支持提取你手机相册中本地视频的文案。

**核心概念**：文案提取功能将视频中的**音频信息转化为文字**，方便你直接复制使用或进行二次编辑。

![](img/cd01f02d886bd61096a508fcd2bf2c7c_10.png)

## 总结
本节课中，我们一起学习了为视频号准备素材的三个核心技能：
1.  **搬运视频**：从平台复制视频链接。
2.  **去除水印**：使用“青抖去水印”或“配音神器Pro”等工具，获得干净的视频素材。
3.  **提取文案**：利用“配音神器Pro”的文案提取功能，快速获取视频的文字内容。

![](img/cd01f02d886bd61096a508fcd2bf2c7c_12.png)

掌握这些方法，你就能高效地获取和处理原始素材，为接下来的视频混剪与创作打下坚实基础。
