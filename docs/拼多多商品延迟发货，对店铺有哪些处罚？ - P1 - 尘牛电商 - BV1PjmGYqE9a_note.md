# 拼多多运营指南：P1：延迟发货的处罚规则与规避方法

在本节课中，我们将要学习拼多多平台对于商家延迟发货行为的处罚规则。了解这些规则，可以帮助你有效避免违规，保障店铺的正常运营。

## 什么是延迟发货？

如果商家没有在规定的发货时间内完成发货，即没有在规定时间内将成交订单的物流单号上传至拼多多后台，这些订单就会被平台识别为延迟发货订单。

## 各类商品发货时间规定

以下是平台对不同类型商品的发货时间要求，建议截图保存以备查阅。

![](img/5813c4e642ad5d3dbd038b97dea815e8_1.png)

*   普通商品：通常需在**48小时**内发货。
*   特殊活动商品（如大促期间）：发货时间可能缩短，需关注平台具体公告。
*   预售商品：需在商家自行设置的预售发货时间内完成发货。

![](img/5813c4e642ad5d3dbd038b97dea815e8_2.png)

## 延迟发货的处罚措施

![](img/5813c4e642ad5d3dbd038b97dea815e8_3.png)

上一节我们明确了延迟发货的定义，本节中我们来看看平台对此的具体处罚方法。平台主要根据延迟时长，设定了三种层级的处罚。

![](img/5813c4e642ad5d3dbd038b97dea815e8_4.png)

以下是三种主要的处罚方式：

![](img/5813c4e642ad5d3dbd038b97dea815e8_5.png)

1.  **延迟48小时内**：平台将从商家账户**扣除赔付金**，并向买家赔付该笔订单商品实际成交金额的 **`1%`**。
2.  **延迟超过48小时**：平台将向买家赔付该笔订单商品实际成交金额的 **`5%`**。
3.  **缺货处罚**：这是最严重的情况。若商家无法发货或虚假发货，平台将判定为缺货，需向买家赔付该笔订单商品实际成交金额的 **`10%`** 或以上。

![](img/5813c4e642ad5d3dbd038b97dea815e8_6.png)

## 如何规避处罚？

![](img/5813c4e642ad5d3dbd038b97dea815e8_7.png)

了解了严厉的处罚规则后，掌握正确的规避方法至关重要。最有效的策略是提前规划库存与发货能力。

![](img/5813c4e642ad5d3dbd038b97dea815e8_8.png)

如果商品在短时间内确实无法正常发货，务必为商品**设置预售**。通过设置合理的预售发货时间，可以避免因缺货或产能不足导致的延迟发货违规。

![](img/5813c4e642ad5d3dbd038b97dea815e8_9.png)

设置预售功能的具体操作方法，可以参考我们之前的教学视频。

本节课中我们一起学习了拼多多平台对延迟发货的判定标准与处罚规则。核心要点是：务必在规定时间内上传物流单号，延迟将面临 `1%`、`5%` 乃至 `10%` 的订单金额赔付。当预见发货困难时，主动设置预售是合规且有效的解决方案。
