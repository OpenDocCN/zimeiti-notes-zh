## 第一部分：旺季产品分级推广与泛用性调整技巧

上一节我们概述了课程内容，本节中我们来看看如何对产品进行科学分级。

产品通常分为两个大维度：新品（在新品期内的产品）和非新品（已过新品期的产品）。我们需要根据产品所处的不同阶段和状态，分别制定相应的营销推广策略。

### 新品五阶段复盘法

新品五阶段复盘是基于产品生命周期策略中的新品期所做的阶段性复盘工作。这里涉及三个核心概念：

![](img/2feaa13f745f61071987ef6befbeec53_7.png)

1.  **五阶段**：指产品从起步到逐渐进入成长期、成熟期或衰退期的五个演化过程。
2.  **三状态**：指产品根据推广重心划分的三种状态：
    *   **主推产品**：运营推广的重心。
    *   **平推产品**：表现尚可，给予合适预算的产品。
    *   **停清产品**：表现不佳，需考虑停售清货的产品。
3.  **十指标**：用于判断产品当前阶段和状态的一系列数据指标。

![](img/2feaa13f745f61071987ef6befbeec53_9.png)

### 产品生命周期与阶段判断

产品的生命周期通常包括起步期、成长期、成熟期和衰退期。不同阶段，产品的获客成本、销售额、利润和竞争力都有差异。

以下是判断产品所处生命周期的两个案例：

*   **案例一（运营动作维度）**：
    *   **新品期**：产品刚上架，文案图片已上传但未优化，评论数少于15个。
    *   **成长期**：星级提升至4.3星以上，评论数大于15个，图片文案经过优化。
    *   **衰退期**：产品被判定为停清产品。
*   **案例二（广告数据维度）**：
    *   **新品期**：广告数据不稳定，出单和花费波动大。
    *   **成熟期（稳定出单区）**：广告表现趋于稳定。

![](img/2feaa13f745f61071987ef6befbeec53_11.png)

### 新品五阶段分析法的核心

主推产品通常会完整经历新品五阶段，并在结束后仍处于成长期。平推产品可能在五阶段结束时进入成熟期。停清产品则可能在不到两个阶段时就因数据糟糕而被判定。

判断产品状态需要依据十个关键指标，例如阶段日均订单量、真实订单占比、Review评分与数量、广告ACOS、转化率等。

以下是一个参考标准（适用于客单价30美金以下的垂类产品），用于判断各阶段产品状态：

| 阶段 | 主推产品标准 | 平推产品标准 | 停清产品标准 |
| :--- | :--- | :--- | :--- |
| **阶段一** | 阶段日均量 ≥ C级 | 阶段日均量 < C级 | 数据表现极差 |
| **阶段三** | 阶段日均量 ≥ B级，真实订单占比 ≥ 50% | 阶段日均量 = C级，真实订单占比 ≈ 30% | 多项指标不达标 |
| **判断原则** | 大部分指标符合主推标准即可判定 | - | - |

**示例**：某产品在第三阶段，日均量达A级，但真实订单占比未达标。此时运营重点应是分析原因并提升真实订单占比。

![](img/2feaa13f745f61071987ef6befbeec53_13.png)

### 非新品产品的分级

![](img/2feaa13f745f61071987ef6befbeec53_15.png)

对于已过新品期的产品，可以从两个维度进行分级：

![](img/2feaa13f745f61071987ef6befbeec53_17.png)

1.  **数据表现**：核心关注 **ACOS（广告销售成本）** 和 **点击转化率（CVR）**。
2.  **市场目标**：根据希望达到的市场占有率或排名来划线。

![](img/2feaa13f745f61071987ef6befbeec53_19.png)

分级建议标准如下：

*   **主推产品**：点击率与转化率高于类目平均值和中位数；或在垂类类目中排名靠前，占据细分市场主要份额。
*   **平推产品**：点击率 > 0.4% 但未达主推标准，或转化率 > 5% 但未达主推标准；产品表现稳定在B/C级。
*   **停清产品**：长期点击率 < 0.4% 且转化率 < 5%；产品表现差，几乎不出单。

![](img/2feaa13f745f61071987ef6befbeec53_21.png)

### 不同状态产品的推广动作

根据产品状态，推广动作应有不同：

*   **主推产品**：保持或增加推广力度和广告预算。若广告投入产出比不理想但自然单量好，可适当控制广告预算。
*   **平推产品**：
    1.  按平推力度推广。
    2.  每周广告预算不高于平推上限。
    3.  总推广花费占比不高于季度上限。
*   **停清产品**：
    1.  停止所有推广动作。
    2.  使用停售清货操作。若通过促销清货，需严格核算成本，确保能收回货品与物流成本。
    3.  严格控制预算，避免浪费。

## 第二部分：旺季广告投放布局与优化技巧

上一节我们介绍了产品分级的方法，本节中我们来看看在旺季如何针对不同等级的产品进行广告布局。

### 旺季广告投放的核心思路

*   **主推产品**：目标是获取更多流量。可提高竞价和预算，以在旺季脱颖而出。
*   **平推产品**：目标是提高转化率。应采取高转化的精准投放动作，对核心热词谨慎投放。可适当增加预算和竞价，但必须密切监控表现。
*   **停清产品**：通常配合大额折扣促销。目标是借助旺季流量清理库存，至少收回货品和物流成本。

### 泛用性广告调整策略

1.  **利用预算规则**：
    *   **基于时间**：在流量高峰期自动增加预算。
    *   **基于业绩**：当ACOS或转化率表现好时，自动增加预算。
    *   *注意：该工具目前只能增加预算，不能减少预算。*
2.  **提前做好预算规划**：
    *   根据历史数据分配和销售目标，推算各广告活动的预算。
    *   主推产品可按时间增加预算，平推产品可按转化率指标设置预算规则。
3.  **优化广告定向**：确保广告投放在高转化的定向方向上。
4.  **使用批量操作面板**：对表现好但预算活跃时间不足的广告活动，批量提高预算。旺季结束后再批量调整回来。

![](img/2feaa13f745f61071987ef6befbeec53_27.png)

![](img/2feaa13f745f61071987ef6befbeec53_28.png)

### Prime Day旺季广告常见错误

1.  **预算不足**：优质广告活动因预算不足错失订单。
2.  **盲目投放高流量低关联大词**：造成预算浪费。可通过设置广告组合和活动的层级预算上限来规避。
3.  **未及时调整竞价**：竞争对手提高竞价时，自身竞价不变，导致流量流失。
4.  **广告结构单一**：只创建一种广告活动，或多个产品共用一个活动。应组合多种广告活动以实现不同目标。

## 第三部分：主推产品的霸屏式广告布局

![](img/2feaa13f745f61071987ef6befbeec53_33.png)

本节我们将深入探讨主推产品在旺季的霸屏式广告布局策略。

![](img/2feaa13f745f61071987ef6befbeec53_35.png)

在旺季，主推产品的广告布局至关重要，其策略重心应是通过提升竞价、调整预算和竞价策略来吸引更多潜在客户流量。

### 站内广告增流途径

在亚马逊站内，可以通过多种方式增加流量，以下是四个有效途径：

1.  商品推广（关键词定向、商品定向）
2.  品牌推广
3.  品牌旗舰店
4.  帖子（Posts）与视频

霸屏式广告的核心是通过全面覆盖站内免费和付费流量位，在搜索结果页和商品详情页获得最佳展示位置。

### 搜索结果页霸屏布局

在搜索结果页，主要通过以下广告位布局：

![](img/2feaa13f745f61071987ef6befbeec53_37.png)

1.  **商品推广（关键词定向）**：
    *   使用**广告位竞价策略**，争夺搜索结果首页首位的黄金展示位。
    *   **准备工作**：进行关键词调研、整理与分类。可以使用“西柚找词”等工具进行竞品ASIN反查、关键词对比分析。
    *   **关键词分类**：利用亚马逊ABA工具，按搜索排名划分关键词类型（如：5万名内为核心热词，5-20万名为中等大词，20万名外为长尾词）。并通过搜索框验证关键词与产品的相关度（高/中/低）。
2.  **品牌推广（关键词定向）**：
    *   位于商品推广广告位上方。
    *   通过自定义图片或视频展示产品独特卖点，吸引点击，提升转化。

![](img/2feaa13f745f61071987ef6befbeec53_39.png)

![](img/2feaa13f745f61071987ef6befbeec53_40.png)

### 品牌旗舰店与视频准备

*   **品牌旗舰店**：设计需突出促销信息、产品特色，引导关注和转化。可设计专题促销页面或应季主题页面，并善用内置的促销小工具。
*   **品牌推广视频**：遵循亚马逊规范，注意 **“5秒完播原则”** ，在前5秒内迅速吸引消费者并展示核心卖点。

![](img/2feaa13f745f61071987ef6befbeec53_42.png)

### 商品详情页霸屏布局

商品详情页的广告展示位包括：

![](img/2feaa13f745f61071987ef6befbeec53_44.png)

![](img/2feaa13f745f61071987ef6befbeec53_45.png)

1.  五点描述下方的**展示型推广（SD）广告位**。
2.  下方的2-3行**商品推广（商品定向）广告位**。
3.  再往下的**品牌推广（商品级）广告位**。
4.  免费的**帖子（Posts）板块**和**关联视频板块**。

![](img/2feaa13f745f61071987ef6befbeec53_47.png)

**布局策略**：
*   **商品定向广告**：对竞品进行分类（替代品、互补品、优势/劣势竞品），针对不同竞品采取不同投放策略。例如，对高关联核心热词上的劣势竞品，可提高竞价抢夺曝光。
*   **品牌推广商品广告**：选择品牌旗舰店子页面作为落地页，为更多产品创造展示机会。
*   **展示型推广（SD）**：旺季前可做受众定向提升认知；节后可做浏览/购买再营销定向，稳定流量。
*   **免费流量-Posts**：类似社交媒体，可发布带场景图和行动号召文案的帖子，关联多个ASIN，引导至商品页或品牌旗舰店。表现优秀的帖子可一键投放广告。

### 竞价调整思路

主推产品在旺季应及时调高竞价，并设置为“提高和降低”动态竞价策略。可尝试使用**广告位竞价**，将产品打到搜索结果页首页的黄金位置。

## 第四部分：平推与停清产品的旺季广告策略

上一节我们详细讲解了主推产品的广告布局，本节我们来看看平推产品和停清产品在旺季如何利用流量。

### 平推产品的广告布局核心

![](img/2feaa13f745f61071987ef6befbeec53_49.png)

平推产品虽非重点，但量大可产生质变。其广告核心是进行**精准度优化投放**，让花费获得更好回报。

**优化建议**：
1.  **调整定向目标**：将重心放在自身有优势的高转化定向上。
2.  **提高投放精准度**：借助品牌分析工具（如商机探测器、ABA）验证并收集精准定向目标。
3.  **善用否定策略**：在广泛匹配、词组匹配等广告中，否定掉不相关或表现差的词/ASIN，控制广告方向。
4.  **关注转化率**：遵循“转化提升十一法”排查问题，顺序为：投放精准度 → 购物车归属 → 商品详情页优化（标题、图片、价格、评分、五点、A+页面等）。

**竞价与预算调整**：
*   **竞价**：建议设置为**固定竞价**，由运营团队根据过去和预期表现手动调整，以节省不必要花费。
*   **预算**：利用亚马逊的预算调整工具，通过设置指标筛选出有问题的广告活动，进行批量预算调整或暂停。

### 停清产品的清货策略

对于需要清货的产品，可以借助旺季流量进行清理，而非仅依赖站内批量清货（回收价格低）或站外服务商。

![](img/2feaa13f745f61071987ef6befbeec53_51.png)

![](img/2feaa13f745f61071987ef6befbeec53_52.png)

**操作思路**：
1.  整理需清货的ASIN。
2.  在品牌旗舰店创建“大额折扣促销”子页面进行引流，将其作为**引流款**，为主推/平推产品带来流量。
3.  结合广告推广，但**预算要适当**，并做好规划。
4.  流量选择上，可投一些**便宜的常规流量**作为定向目标。

**库存清理节奏参考**：
*   **正常销售期**：按自身节奏投放广告，定期分析库存和广告报告。
*   **周转过长期**：采取“促销+广告”或“降价+广告”组合拳，明确广告目标（曝光、点击或转化）。
*   **产生长期仓储费期**：考虑提交批量清货。**退回或弃置**是速度最快但亏损最大的方式，应尽量避免。

本节课中我们一起学习了亚马逊旺季产品推广的核心框架。

**第一部分**，我们掌握了如何通过“新品五阶段复盘法”和“十指标”对产品进行科学分级（主推、平推、停清），并针对不同状态制定了差异化的推广策略与预算控制方法。

**第二部分**，我们深入探讨了旺季广告投放的布局。对于**主推产品**，重点是霸屏式布局，争夺搜索结果页和商品详情页的优质广告位，并利用品牌旗舰店、帖子等免费流量。对于**平推产品**，核心是优化精准度与转化率。对于**停清产品**，则是借助旺季流量和促销进行成本可控的清货。
