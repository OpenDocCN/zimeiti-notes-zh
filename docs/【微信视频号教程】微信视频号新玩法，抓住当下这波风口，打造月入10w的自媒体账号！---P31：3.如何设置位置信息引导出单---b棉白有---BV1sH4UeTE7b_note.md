# 微信视频号运营：P31：如何设置位置信息引导出单

在本节课中，我们将学习一个提升视频号带货转化率的关键技巧：如何通过设置位置信息，有效引导用户下单购买。

上一节我们介绍了视频内容创作的核心要点，本节中我们来看看如何利用发布设置中的细节功能来促进销售。

## 位置信息的作用

![](img/a7b7dca9678a3418cc413d1ab361d8c3_0.png)

在视频号发布视频时，填写位置信息不仅是一个简单的标注。对于带货账号，尤其是销售本地产品或服务的账号，这是一个非常重要的引流和转化工具。正确设置位置信息，可以直接将观看兴趣转化为购买行动。

## 设置位置信息的核心步骤

以下是设置位置信息引导出单的具体操作流程。

![](img/a7b7dca9678a3418cc413d1ab361d8c3_1.png)

1.  发布视频时，在编辑页面找到并点击“你在哪里”或“添加位置”选项。
2.  在位置搜索框中，**不要直接输入你的店铺或公司名称**。相反，应输入你所售卖的具体产品名称，例如“潮汕牛肉丸”。
3.  从搜索结果中，选择一个与你的产品相关、且看起来是“店铺”类型的位置点（通常带有商家标签）。如果搜索结果中没有理想的，可以尝试搜索产品所在的品类或市场名称。
4.  选择该位置后，它就会显示在你的视频发布信息中。

## 为何要这样操作？

![](img/a7b7dca9678a3418cc413d1ab361d8c3_2.png)

这样设置的核心目的是**降低用户的决策成本**。当用户对你的产品产生兴趣时，他们通常会思考“哪里可以买到”。如果你的视频附带的位置信息直接就是产品名称或相关店铺，就相当于明确地告诉用户：“点击这里，就能找到购买渠道。”

其引导逻辑可以简化为一个公式：
**用户兴趣 → 看到产品关联位置 → 点击位置查看详情 → 进入店铺或商品页 → 完成购买**

相比之下，如果位置只填写了普通的城市或街区名，或者是你公司的官方名称（用户可能不熟悉），就中断了这个引导链条，用户需要额外搜索，流失率会大大增加。

![](img/a7b7dca9678a3418cc413d1ab361d8c3_3.png)

## 实际效果与总结

通过这种方式设置的位置信息，会成为一个直接的“购买入口”。感兴趣的用户点击该位置后，可能会跳转到相关的商家页面、团购页面或商品列表，从而极大地提高了从观看到下单的转化效率。

本节课中我们一起学习了如何巧妙设置视频号的位置信息来引导出单。关键点在于：将位置设置为产品名称或相关品类，而非泛泛的地名或公司名，从而为用户提供最短的购买路径。掌握这个细节技巧，能有效提升视频号的带货能力。
