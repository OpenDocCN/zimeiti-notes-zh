## 分割功能详解

分割功能是视频剪辑中最常用到的处理手段之一。简单来说，它的作用是将一段完整的视频素材，在剪辑过程中截取出需要保留的部分，并将不需要或效果不佳的画面裁切删除。

![](img/bd44a3acdd2eb59b672a08906d0a7633_1.png)

![](img/bd44a3acdd2eb59b672a08906d0a7633_2.png)

**核心操作逻辑**可以概括为：`定位时间线 -> 执行分割 -> 删除多余片段`。通过分割工具，我们可以直接将视频裁切成两段或多段，然后删除不需要的部分。

![](img/bd44a3acdd2eb59b672a08906d0a7633_4.png)

为了理解操作环境，我们需要先认识剪映的剪辑界面。界面主要分为三个部分：
*   **预览窗口**：位于上方，用于预览视频效果。
*   **时间轴**：位于中间，是进行剪辑操作的主要区域。所有视频剪辑软件都具备时间轴。
*   **剪辑菜单**：位于底部，包含了分割、音频、文本等所有剪辑功能。我们对视频进行的一系列操作都在这里选取。

![](img/bd44a3acdd2eb59b672a08906d0a7633_6.png)

![](img/bd44a3acdd2eb59b672a08906d0a7633_7.png)

时间轴上有一条白色的竖线，称为**时间线**。它是视频剪辑中的定位点。例如，如果我们想在视频的第2秒进行分割，就需要先将时间线移动到第2秒的位置。

![](img/bd44a3acdd2eb59b672a08906d0a7633_9.png)

## 实操演示：如何使用分割功能

我们将通过一个实例，一步步演示如何使用分割功能截取视频片段。

**第一步：导入素材**
1.  启动剪映APP，默认进入“剪辑”界面。
2.  点击顶部“**开始创作**”按钮。
3.  从相册中选择要剪辑的视频素材，导入到项目库中。

**第二步：预览与规划**
导入素材后，可以点击预览窗口的播放按钮，完整观看一遍视频，确定想要保留的片段（例如视频中间的某个精彩镜头）和需要删除的部分。

**第三步：放大时间轴进行精细操作**
为了更精确地定位分割点，我们需要放大时间轴。操作方法是用两根手指在时间轴区域**向外滑动**（放大），这样可以更清楚地看到每一帧画面。

**第四步：定位并分割**
1.  拖动时间轴上的白色时间线，将其移动到计划分割的位置（即两个镜头的衔接点）。
2.  点击时间轴上的视频片段，当其周围出现白色框线时，表示已被选中。
3.  在底部一级菜单中，找到并点击“**分割**”功能按钮。
4.  点击后，视频在时间线位置被一分为二，会出现一个白色小方框作为标记。

**第五步：删除多余片段**
1.  点击选中需要删除的前半部分视频片段。
2.  点击右下角的“**删除**”按钮，该片段即被移除。
3.  重复以上“定位->分割->删除”的步骤，找到下一个剪辑点，将需要保留片段之后的多余部分也删除掉。

![](img/bd44a3acdd2eb59b672a08906d0a7633_12.png)

**第六步：微调与导出**
1.  截取完成后，再次播放预览，检查片段开头和结尾是否精准。如有需要，可对首尾进行进一步的分割微调。
2.  确认无误后，点击右上角的“**导出**”按钮。
3.  在导出设置中，通常选择 **1080P** 分辨率和 **30** 帧率，然后点击导出即可完成视频剪辑。

![](img/bd44a3acdd2eb59b672a08906d0a7633_14.png)

## 总结

本节课中，我们一起学习了剪映的核心剪辑功能——分割功能。分割是视频剪辑中运用最多、最广的一种手段。通过`定位、分割、删除`这一流程，我们可以方便地从长视频素材中提炼出需要的精彩片段。

在剪映APP中，直接使用分割功能即可实现视频裁切，操作非常便捷。关键在于操作时准确地将时间线定位到预想的剪辑点。掌握好这个基础功能，就为我们后续拼接镜头、组合素材打下了坚实的基础。
