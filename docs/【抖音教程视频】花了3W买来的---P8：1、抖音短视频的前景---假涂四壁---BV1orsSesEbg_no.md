# 抖音运营入门：P8：1、抖音短视频的前景 📈

![](img/9addb03c4a8daf36f7d836ad7c31ebea_1.png)

在本节课中，我们将要学习抖音短视频平台的发展前景，分析其市场潜力、用户构成以及新手入门的核心优势。通过理解这些基础，你将能更好地定位自己在平台上的角色与机会。

![](img/9addb03c4a8daf36f7d836ad7c31ebea_3.png)

了解抖音短视频的前景，需要先通过一张图片来认识。

![](img/9addb03c4a8daf36f7d836ad7c31ebea_1.png)

这张图片是抖音创作者大会的一份数据报告。报告显示，有超过2200万人在抖音的合计收入超过了417亿元。抖音希望将这个数字翻一番，让创作者的总收入达到800亿元。

![](img/9addb03c4a8daf36f7d836ad7c31ebea_3.png)

这体现了平台巨大的流量扶持力度。同时，这里引入第一个知识点：抖音用户主要分为两类。第一类是**玩家**，他们单纯地使用抖音娱乐，没有考虑从中赚钱。第二类是**创作者**，即像本课程学员这样，希望通过制作抖音内容来获取收益的人。明确这两类人的区别很重要，后续内容还会提及。

那么，在众多短视频平台中，我们为什么选择抖音呢？这类似于开设实体店选址。做短视频一定要选择一个流量大的平台。如果平台自身流量不足，就无法分配流量给创作者，创作者也就难以赚钱。抖音的日活跃用户超过8亿，月活跃用户更是达到了40亿人次。这是一个极其庞大的流量池。

流量巨大，竞争压力是否也会很大？答案是不会。平台上的创作者大约只有5000万。这里的“创作者”定义广泛，并非特指网红达人。例如，邻居家的老太太用手机拍摄自己做的饭菜并上传到抖音，她也算创作者。因此，你90%的竞争压力，实际上来自于身边的普通用户。

接下来，做抖音短视频需要购买电脑或摄像机吗？答案是**不需要**。你只需要一部智能手机和一个手机号，就可以注册抖音账号。抖音倡导全民通过短视频的形式分享和记录生活，入门门槛很低。

那么，当前有哪些值得把握的风口呢？首先，抖音计划实施“抖音将帮助1000名优质的艺术创作者涨粉超过100万”的策略。有的同学可能会担心自己不会创作。这无需忧虑，正因为我们有所欠缺，才需要学习。虚心向比自己懂行的人请教并不丢人，这能让我们少走很多弯路。

市场空间巨大，竞争环境相对宽松，行业内容产出也相对容易。当前是大数据时代，平台会根据用户的喜好推荐内容。你喜欢看什么，抖音就会给你推荐什么。因此，无论你是宝妈想分享宝宝日常和美食，实体店主想展示产品引流，还是普通上班族想记录生活，都可以找到自己的内容方向。每月40亿的用户群体中，只要你的内容不触犯法律和道德底线，不是低俗内容，总会有一部分人喜欢上你。

为什么老师说一定会有人喜欢你，而你却只有几十个粉丝呢？这是因为喜欢你的人可能还没有刷到过你的作品。所以，对于新手来说，心情迫切可以理解，但不要把涨粉作为首要目标。我们的第一个目标应该是**先把流量和播放量做起来**。播放量上涨后，粉丝自然会增长。

本节课中我们一起学习了抖音平台的巨大前景、用户分类、低门槛优势以及新手起步的核心策略。关键在于理解平台流量逻辑，明确创作者身份，并优先专注于提升内容流量，而非单纯追求粉丝数量。
