商家经常有疑问：抖音上什么商品卖得好？是否可以把其他平台的商品照搬过来？

团购商品的设置是在抖音做生意非常关键的一步。本节将讲解如何做好基础的团购组品。

这个问题其实很好回答，只需要转换角度，站在顾客的角度思考。顾客在抖音上看到商品后，是如何一步步被吸引并决定下单的呢？首先，商品要有卖点，才能吸引顾客点进来。

所以首先要思考产品的卖点是什么，是应季商品、折扣力度大，还是有特色菜。其次，商品标题要写清楚适用条件，让顾客一眼就看到重点信息。顾客点进商品页面后，最想了解的是这个团购商品适合几个人、有没有时间限制。

例如：**情人节专享情侣双人和牛自助晚餐**。

接着，需要与其他平台有差异点。顾客看完商品信息后，可能会去其他平台比价，在比价过程中可能被其他商家吸引而流失。

因此需要尽量避免让顾客比较，促使其快速下单。这里介绍几种方法：
*   **独家**：该商品或套餐只在抖音上团购。
*   **加料**：同样价格的套餐，增加一些商品。
*   **差异**：对套餐里的商品进行调整，变成一个新的套餐。

那么具体怎么组品，有哪些好的攻略呢？这里给大家一条建议的公式：

**商品数 + 优惠力度 + 丰富度 + 差异性**

以下是具体的操作建议：

*   **商品数**：建议设置两个以上的团购商品。可以包含引流品（低价或折扣大）、利润品、代金券等。这样顾客进入抖音门店后，才能进行挑选。
*   **优惠力度**：参考市面上同类商品或本店在其他平台的折扣力度，设置一个有竞争力的优惠，以更快吸引用户下单。
*   **丰富度**：团购套餐里的商品需要足够丰富，让顾客有物超所值的感觉。建议包含推荐菜、特色菜等有吸引力的商品。套餐类型可分为二人、四人、多人等多种形式。
*   **差异性**：区别于其他平台的套餐组合，避免顾客在比价过程中流失。

上一节我们介绍了基础的组品思路和公式，本节中我们来看看如何根据不同行业的特点，进行更精细化的商品组合。

大家初步了解了如何发掘商品卖点、优化标题和图片信息、突出平台优势并上架商品。但是，单一的商品类型很难满足海量用户的多样化需求，质量一般的商品转化率也很低。想要拥有好的经营结果，必须在保证商品数量的同时，保证商品的品类覆盖和质量。

**商品数量**指一个门店上线了多少个商品。用户需求是多样的。例如，工作日上班族用餐时间有限，更关注快速，对品质和菜品丰富度要求相对较低，用餐人数一般为一至两人。

所以，在工作用餐场景下，主要的二级品类为快餐，用餐人数主要为单人餐或双人餐。同时，为了保证较好的用户体验，一般建议套餐包含招牌菜，且要有一定的价格竞争力（即价格不高于同类型商品的一般定价）。套餐搭配较为固定。为了满足对菜品有强诉求的用户，建议至少设置一张代金券。

不同于餐饮行业的多样化，酒旅和到店综合行业的需求更加集中。所以上架商品的目的是覆盖最高频的使用场景。例如：
*   **酒店住宿**：大床房、双床房两个房型可覆盖超过90%的需求，且往往会有特价房。
*   **游玩品类**：所有人通用的门票（单人票）消费比例会远超其他品类。
*   **其他到综品类**：如美发行业，一定要包含核心服务（剪发、烫染发等）。

以下是针对不同行业的具体组品建议表格：

| 行业大类 | 细分品类 | 建议商品数量 | 商品类型建议 | 核心要点 |
| :--- | :--- | :--- | :--- | :--- |
| **餐饮** | 火锅、烧烤、烤肉、中餐 | ≥ 4个 | 2-3人餐、4-6人餐、100元代金券 | 每个类型套餐至少有一个包含门店招牌菜，价格满足建议售价要求。 |
| | 必火菜（如小龙虾） | ≥ 4个 | 2-3人餐、代金券 | （不要求4-6人餐） |
| | 西式快餐、面包甜点、茶饮 | ≥ 4个 | 单人餐、2-3人餐、50或100元代金券 | 每个套餐至少有一个门店特色菜。 |
| | 中式快餐 | ≥ 4个 | 2-3人餐、代金券 | （初步要求2-3人餐） |
| | 自助餐 | ≥ 4个 | 单人餐、多人餐 | 需包含单人餐，符合常规用餐需求。 |
| **酒旅** | 酒店住宿 | ≥ 4个 | 单房型（大/双床）、房型+早餐/门票组合 | 需包含基础房型和组合产品。 |
| | 游玩 | ≥ 2个 | 单人成人票、多人票（一大一小/两大一小/情侣票） | 覆盖个人和家庭/团体需求。 |
| **到店综合** | 洗浴、足疗、按摩 | ≥ 4个 | 多种服务套餐 | **最关键**：商品中需包含60-70分钟时长的套餐。 |
| | 丽人美发 | ≥ 2个 | 剪发套餐、次卡 | 必须包含一个剪发套餐。 |
| | 丽人美甲美睫 | ≥ 2个 | 美甲/美睫套餐、次卡 | 必须包含一个美甲或美睫套餐。 |
| | 亲子 | ≥ 2个 | 门票套餐（一大一小/两大一小/多人票）、次卡 | 设置次卡是很好的营销方式。 |
