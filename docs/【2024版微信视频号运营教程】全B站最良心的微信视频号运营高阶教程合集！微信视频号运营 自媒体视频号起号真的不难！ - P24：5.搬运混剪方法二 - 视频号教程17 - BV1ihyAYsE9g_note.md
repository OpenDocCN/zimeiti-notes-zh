# 微信视频号运营教程：P24：5. 搬运混剪方法二

![](img/1bd460eb9a858745b607243a5c9bb14f_1.png)

## 概述
在本节课中，我们将学习一种具体的视频混剪方法。我们将通过一系列步骤，将一个现有的视频素材进行深度修改和二次创作，使其成为一个全新的、符合平台原创要求的视频。整个过程将使用剪映和秒剪两款工具。

![](img/1bd460eb9a858745b607243a5c9bb14f_3.png)

![](img/1bd460eb9a858745b607243a5c9bb14f_5.png)

## 准备工作
在开始混剪之前，需要准备一些素材。你需要提前拍摄至少4段自己的实拍视频，内容可以是生活日常、自拍、家人或身边风景。这些素材将在后续步骤中作为画中画使用。

## 第一步：修改视频MD5值
首先，我们需要处理原始视频素材。为了降低平台对重复内容的识别率，第一步是修改视频文件的MD5值。

![](img/1bd460eb9a858745b607243a5c9bb14f_7.png)

**操作流程：**
1.  打开“配音神器”或其他具有修改MD5功能的工具。
2.  导入已去除水印的原始视频。
3.  执行“修改MD5”操作。

![](img/1bd460eb9a858745b607243a5c9bb14f_9.png)

**核心概念：**
MD5是一种文件“指纹”。修改它可以使系统认为这是一个全新的文件，有助于通过平台查重。

## 第二步：基础剪辑与掐头去尾
修改完MD5后，我们进入剪辑环节。基础操作是去除视频开头和结尾的片段。

**操作流程：**
1.  打开“剪映”，导入修改后的视频。
2.  将时间轴移动到视频起始处，使用“分割”工具，删除最前面的一小段（约0.5-1秒）。
3.  同样，将时间轴移动到视频末尾，分割并删除最后的一小段。

这个步骤可以去除原视频中可能存在的标识性开头或结尾。

![](img/1bd460eb9a858745b607243a5c9bb14f_11.png)

## 第三步：遮盖原字幕并生成新字幕
原视频通常带有字幕。为了体现原创性，我们需要遮盖原字幕并生成新的字幕。

**操作流程：**
1.  选中视频轨道，在二级菜单中找到“蒙版”功能。
2.  选择“矩形蒙版”。
3.  调整蒙版的高度，使其覆盖住原视频的白色字幕区域，并适当拉长宽度。
4.  将蒙版向下拖动，完全盖住字幕。
5.  点击“反转”按钮。
6.  点击对勾确认。
7.  返回主菜单，点击“背景”，选择“画布模糊”，为被遮盖的区域添加模糊背景。
8.  使用“识别字幕”功能，为视频生成全新的字幕。
9.  可以调整新字幕的位置、字体、颜色和动画效果（如“打字机”动画）。

![](img/1bd460eb9a858745b607243a5c9bb14f_13.png)

## 第四步：修改视频音频（变声）
改变视频的声音是混剪中非常有效的一步。我们可以将原声替换为不同的音色。

![](img/1bd460eb9a858745b607243a5c9bb14f_15.png)

**操作流程：**
1.  选中视频轨道，在菜单中找到“声音效果”。
2.  在变声效果库中选择一个合适的效果，例如“台湾小哥”。
3.  应用后，视频的原声就会发生变化。

你也可以使用“配音神器”等工具生成全新的配音音频，然后导入剪映进行替换。

## 第五步：调节视频画面参数
通过调整视频的各项画面参数，可以改变其整体色调和观感。

![](img/1bd460eb9a858745b607243a5c9bb14f_17.png)

**操作流程：**
1.  选中视频轨道，点击“调节”功能。
2.  对亮度、对比度、饱和度、光感、锐化等参数进行微调。
3.  每个参数的调整幅度建议在 **±5** 以内，避免画面失真。
4.  如果一次调节效果不明显，可以点击“新增调节”图层，进行多次叠加调整。

![](img/1bd460eb9a858745b607243a5c9bb14f_19.png)

## 第六步：调整视频镜像与角度
对视频进行镜像翻转或微调角度，可以直观地改变画面。

**操作流程：**
1.  选中视频轨道，点击“编辑”。
2.  点击“镜像”按钮，对画面进行水平翻转。
3.  点击“裁剪”，可以轻微旋转视频角度（如1度）。
    *   **注意**：如果旋转影响了新生成的字幕，则应撤销裁剪操作。

![](img/1bd460eb9a858745b607243a5c9bb14f_21.png)

![](img/1bd460eb9a858745b607243a5c9bb14f_23.png)

## 第七步：添加滤镜
为视频添加滤镜可以统一或改变其色调风格。

**操作流程：**
1.  点击“滤镜”功能。
2.  根据视频内容选择合适的滤镜分类（如“美食”）。
3.  可以添加多个滤镜，并将每个滤镜的强度条拉满，或调整到合适强度。
4.  确保滤镜效果覆盖整个视频时长。

## 第八步：添加特效
特效能为视频增加动态元素。根据内容选择“画面特效”或“人物特效”。

**操作流程：**
1.  点击“特效”功能。
2.  如果视频中有人脸，可选用“人物特效”；否则选用“画面特效”。
3.  可以添加多个特效。
4.  选中每个特效轨道，可以“调整参数”，如降低不透明度、调节速度，使其效果不那么突兀，自然融入视频。
5.  确保所有特效的时长与主视频对齐。

## 第九步：添加贴纸
在视频的边角位置添加动态贴纸，能增加原创元素和趣味性。

**操作流程：**
1.  点击“贴纸”功能。
2.  在视频的四个角或上下左右位置，至少添加4个贴纸。
3.  尽量选择动态贴纸。
4.  调整每个贴纸的大小和位置。
5.  将所有贴纸的时长拉至与主视频一致。

![](img/1bd460eb9a858745b607243a5c9bb14f_25.png)

## 第十步：添加画中画（关键步骤）
这是融入个人实拍素材、体现原创性的核心步骤。使用之前准备的4段实拍视频。

![](img/1bd460eb9a858745b607243a5c9bb14f_27.png)

**操作流程：**
1.  点击“画中画”，新增画中画轨道，导入第一段实拍视频。
2.  将画中画视频拖动到主视频的某个角落。
3.  如果画中画视频时长不足，使用“变速”功能（常规变速）将其拉长；如果过长，则分割删除多余部分。
4.  将画中画的“音量”调到 **1或2**，确保不影响主视频声音。
5.  点击“混合模式”，选择“正常”、“变暗”或“绿色”等模式，并将透明度调到 **10** 左右，营造若隐若现的效果。
6.  对另外3段实拍视频重复上述1-5步操作，分别放置在主视频的其他角落。

![](img/1bd460eb9a858745b607243a5c9bb14f_29.png)

![](img/1bd460eb9a858745b607243a5c9bb14f_30.png)

## 第十一步：添加动态Logo
为自己的账号创建一个动态的文字Logo，增强品牌识别度。

![](img/1bd460eb9a858745b607243a5c9bb14f_32.png)

![](img/1bd460eb9a858745b607243a5c9bb14f_34.png)

**操作流程：**
1.  点击“文本”，新建文本，输入你的账号名（如“狼团联盟笔记”）。
2.  将文本时长拉满。
3.  调整文本的样式、颜色和大小，并将其放置在视频起始位置。
4.  在文本轨道的起始处，添加一个“关键帧”。
5.  将时间轴向后稍微移动，然后拖动文本到另一个位置，系统会自动生成移动动画。
6.  重复步骤5，多次移动文本位置，使其在视频播放期间持续动态移动。

## 第十二步：导出与二次处理
完成所有剪辑后，进行最终导出，并使用官方工具进行最后处理。

![](img/1bd460eb9a858745b607243a5c9bb14f_36.png)

**操作流程：**
1.  点击“导出”，将“帧率”设置为 **60** fps，然后导出视频。
2.  打开微信视频号官方剪辑工具“秒剪”。
3.  导入刚刚导出的视频。
4.  为视频选择一个合适的“模板”（如“治愈”）。
5.  将模板自带的音乐关闭，打开自己视频的原声。
6.  可以在秒剪中再次添加一些滤镜或贴纸进行微调。
7.  完成后，可以直接通过秒剪发布到视频号，或保存到相册后手动发布。

![](img/1bd460eb9a858745b607243a5c9bb14f_38.png)

**核心提示：**
*   以上步骤顺序可以灵活调整。
*   混剪的目的是让系统识别出你的视频与原素材不同，因此步骤越多、修改越全面，效果越好。
*   最终视频的清晰度可能略有损失，但这有助于“去重”，且不影响流量获取。

![](img/1bd460eb9a858745b607243a5c9bb14f_40.png)

## 总结
本节课我们一起学习了搬运混剪的第二种详细方法。我们从修改MD5值开始，逐步完成了掐头去尾、替换字幕和音频、调节画面参数、添加镜像、滤镜、特效、贴纸、画中画以及动态Logo等一系列操作，最后使用秒剪进行模板化处理并发布。这套方法的核心逻辑是通过多层级的修改和添加个人原创元素，降低视频的重复率，使其更符合平台的原创要求。无论你进行哪种类型的视频创作，都可以遵循这个混剪流程。
