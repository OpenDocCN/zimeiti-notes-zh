# Ozon跨境电商选品运营教程：P1：爆款产品筛选方法

## 概述
在本节课中，我们将学习如何在Ozon平台上筛选出具备爆款潜力的产品。我们将通过一个具体的宠物用品类目案例，演示从数据采集到关键指标筛选的完整流程，帮助你快速定位高潜力商品。

## 教程正文

### 第一步：采集热门商品数据
首先，我们需要获取市场数据作为分析基础。打开Ozon卖家后台，找到“热门商品”分析工具。我们以“宠物用品”类目为例，使用该工具将这个类目下的所有热门商品数据采集下来。

完成采集后，我们会得到一个包含多个字段的数据表格，例如商品名称、销量、加购率、动销率等。这份表格是我们后续筛选工作的核心。

### 第二步：筛选关键指标
在获得数据表格后，我们通过筛选关键指标来缩小潜力产品的范围。以下是两个核心筛选步骤：

**1. 筛选加购率**
加购率反映了消费者对商品的兴趣程度。我们首先在表格中筛选出加购率在**7%**以上的产品。这一步可以初步过滤掉市场关注度较低的商品。

**2. 筛选动销率**
动销率衡量了商品的实际销售转化能力。在第一步筛选的结果基础上，我们进一步筛选出动销率在**15%**以上的产品。经过这两轮筛选，最终留下的产品通常同时具备高市场兴趣和强销售转化能力，即具备爆款潜力。

### 第三步：获取与分析结果
完成上述筛选后，表格中剩余的产品就是我们的初步选品结果。你可以对这些产品进行进一步的分析，例如研究其价格、评价、竞争情况等，以做出最终的上架决策。

此外，用于采集数据的工具可以通过回复“选品”来获取。

## 总结
本节课我们一起学习了Ozon平台爆款产品的筛选方法。核心流程是：**采集热门商品数据 -> 筛选高加购率（>7%）产品 -> 筛选高动销率（>15%）产品**。通过这套数据驱动的筛选方法，你可以更高效地从海量商品中识别出潜在爆款，为后续运营打下坚实基础。
