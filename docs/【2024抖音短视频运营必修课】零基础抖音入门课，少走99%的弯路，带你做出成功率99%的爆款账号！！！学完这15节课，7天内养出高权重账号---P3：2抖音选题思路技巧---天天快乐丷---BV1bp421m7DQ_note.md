# 抖音短视频运营必修课：P3：2 抖音选题思路技巧 🎯

在本节课中，我们将学习一套高效的抖音选题公式。这套公式能帮助你快速确定视频方向，解决“不知道发什么”的难题，从而更精准地吸引目标用户。

你现在为抖音流的泪，都是当初定选题时脑子里进的水。你是否为想选题想到头秃，不知道明天该发布什么内容？今天教你一套公式，一分钟帮你解决这个苦恼。

会找选题的人，能有更多时间专注于视频制作。每天发布三条视频，一个月就是九十条。依靠堆量都能形成竞争优势，根本无需偷偷加量。而且，擅长选题的人能更精准地找到用户痛点，也更容易吸引精准粉丝驻足观看，从而快速为账号打上标签。

![](img/baeeb144e98f3215b156044d0b08bf1d_5.png)

那么，有什么办法可以解决选题难题呢？当然有。这是一套百万粉丝博主常用的选题技巧。为了方便理解，我将其总结成了一个公式：**内容定位 + 用户痛点 + 借势热点**。用好这个公式，你的内容方向一定不会跑偏。

例如，如果你想和我一样做一个知识博主，就需要去挖掘行业内的关键词，例如：播放量、粉丝量、定位、选题等。这些都是客观关键词，不包含主观描述。然后，在这个基础上寻找用户的痛点，例如：便捷、高效、技巧教学、内容输出等。最后，去蹭近期的热点，将这些元素进行任意组合即可。眼下春节将至，你可以创作诸如“如何用拜年话术做内容”、“春节必火的选题”、“张同学快速涨粉的三个秘诀”等主题。

我们再把这个公式套用到其他行业试试。以美妆行业为例，内容定位可以是：美白、除皱、口红。用户痛点则是：平价好用、明星同款、避坑扫雷。最后，借势热点进行排列组合，例如：“迪丽热巴同款美白面膜推荐”、“七夕千万别送女朋友的口红”等。

以上已经举了两个例子进行说明，至于举一反三，就交给你了。你可以根据自己所在行业的特性，运用这个公式进行思考和实践。

本节课我们一起学习了抖音选题的核心公式：**内容定位 + 用户痛点 + 借势热点**。通过拆解行业关键词、洞察用户需求并结合时事热点，你可以系统性地生成大量优质选题，从而提升内容创作效率与账号吸引力。
