# 小红书运营教程：P12：小红书商单模式与图文笔记制作详解

在本节课中，我们将要学习小红书平台的核心变现模式之一——商单合作，并详细拆解如何制作一篇合格的商单图文笔记。我们将通过分析完美日记的案例，理解“蒲公英计划”的运作逻辑，并掌握从接单到内容创作的全流程。

## 完美日记的“农村包围城市”策略

上一节我们介绍了小红书平台的基础定位，本节中我们来看看一个成功的商业案例。

完美日记是一个国产品牌。它最初的销量不高，知名度也不高。品牌方希望提升知名度和销量，需要打广告。传统方式是投入几十万甚至上百万在电视剧中植入广告或请明星代言。

![](img/2cb26697a0a0cb02a35ef92e0f2fc19b_1.png)

但完美日记完美地运用了“农村包围城市”的商业策略。与其花费巨资进行传统广告投放，不如将预算拆分，寻找自媒体人和博主进行广告推广。这样既节省了费用，又能达到铺天盖地的宣传效果。

## 小红书的“蒲公英计划”与商单逻辑

![](img/2cb26697a0a0cb02a35ef92e0f2fc19b_3.png)

![](img/2cb26697a0a0cb02a35ef92e0f2fc19b_4.png)

小红书官方的定义是一个购物分享平台。用户是带着购物欲望和消费目的而来的。在这种平台上进行产品宣传，能直接对准目标用户。

因此，小红书官方推出了“蒲公英计划”。这个计划就像将种子撒向各地，本质上是品牌与博主之间的商业订单合作。

以下是关于“蒲公英计划”和商单的几个核心要点：

*   **低粉丝门槛**：参与“蒲公英计划”接商单，对粉丝量要求不高。例如，案例中的博主仅有1000多个粉丝，其视频报价可达700元。
*   **结算方式简单**：平台不单纯看笔记的浏览和点赞数据。只要博主发布了规定的内容并挂上商品链接，佣金就会直接打入账户。
*   **商单资源丰富**：小红书作为购物分享平台，拥有大量品牌方资源。博主通常面临的是接不完的商单，而非接不到。案例中千粉博主的待接商单数量可达65个。

![](img/2cb26697a0a0cb02a35ef92e0f2fc19b_6.png)

我们来计算一下潜在的收益：假设每个商单报价为350元，每天只接1单，一个月发布30条商单笔记。那么，仅商单收入即可达到 **`350元/单 * 30单 = 10500元`**。这还未计算通过笔记销售产品可能带来的额外佣金收入。

## 如何制作商单图文笔记

理解了商单模式后，我们来看看如何制作一篇合格的商单图文笔记。一篇图文笔记通常由几张图片和下方的文案（产品使用心得）组成，并需要挂上商品链接。

制作图文笔记有两个重点：首图和文案。

### 1. 首图：决定点击率的关键

![](img/2cb26697a0a0cb02a35ef92e0f2fc19b_10.png)

首图是笔记的第一张图片，它直接决定了笔记的点击率。因此，设计一张吸引眼球的首图至关重要。

![](img/2cb26697a0a0cb02a35ef92e0f2fc19b_11.png)

对于首图素材问题，可以提供海量的原创素材库以及免费的图片制作软件，帮助学员解决图片来源和设计难题。

### 2. 文案：传达产品心得的桥梁

文案是分享产品使用体验的核心部分。许多初学者在撰写文案时感到困难。

针对文案撰写，课程将提供具体的写作方法和模板，帮助即使不会写文案的学员也能快速产出合格的内容。
