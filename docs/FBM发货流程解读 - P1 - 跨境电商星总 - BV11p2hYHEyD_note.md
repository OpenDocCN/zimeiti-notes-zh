# 跨境电商FBM发货教程：P1：FBM发货核心流程解读

在本节课中，我们将要学习跨境电商FBM（Fulfillment by Merchant，卖家自发货）模式的核心发货流程。我们将重点了解从打包到将包裹交给物流商的完整步骤，并厘清清关、贴标等关键环节的责任归属。

## 流程概述

FBM模式意味着卖家需要自行处理从国内到国外买家的整个物流过程。其核心流程可以概括为：**卖家打包 -> 贴国际面单并处理清关 -> 将包裹交给货代发出**。

上一节我们概述了FBM的整体概念，本节中我们来看看具体的操作步骤。

## 详细步骤解析

以下是FBM发货的标准操作流程，每个环节都至关重要。

1.  **产品打包**
    卖家需要自行将售出的产品妥善打包，准备好发出。

2.  **制作并粘贴面单**
    打包完成后，卖家需要制作并粘贴国际物流面单。**在制作这份面单的同时，必须同步完成清关所需的资料准备工作**。这是确保包裹顺利通关的关键。

3.  **交付货代**
    当面单粘贴妥当后，卖家将包裹交付给合作的货代（国际物流服务商）。货代会负责后续的运输环节，将产品发送至国外买家手中。

以上就是FBM发货的标准流程。接下来，我们将探讨一种特殊情况。

## 一件代发的操作差异

如果你采用一件代发模式，流程会有所不同。在这种情况下，你的产品会直接采购到货代仓库。

货代会替你完成二次打包、清关以及贴标等工作。**这里的关键在于：你需要提前与货代沟通确认，确保他们在代为贴标时，会一并处理好清关事宜。**

本节课中我们一起学习了FBM模式的标准发货流程，包括打包、贴单、清关和交付货代等核心步骤，同时也了解了一件代发模式下的操作差异。理解这些基础流程，是顺利开展跨境电商FBM业务的第一步。
