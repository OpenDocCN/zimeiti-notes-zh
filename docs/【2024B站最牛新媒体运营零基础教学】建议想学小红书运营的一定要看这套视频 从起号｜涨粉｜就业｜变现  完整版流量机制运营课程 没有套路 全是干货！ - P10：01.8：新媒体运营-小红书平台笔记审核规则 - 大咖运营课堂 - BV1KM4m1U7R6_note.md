## 笔记审核与分发流程 🔄

上一节我们介绍了小红书运营的基本概念，本节中我们来看看笔记发布后的具体审核流程。小红书的笔记推荐机制基于“千人千面”原则，即根据用户的特定标签推送特定内容。笔记发布后，会经历以下主要阶段。

### 第一阶段：机器审核

笔记上传后，首先进入审核系统。系统会根据预设的基本规则进行初步审核。

以下是机器审核的主要检查项：
*   **敏感内容**：检测笔记中是否存在敏感词或敏感图片。
*   **基础质量**：评估笔记的字数、广告词出现频率、图片清晰度以及文字可识别性。
*   **初步判定**：若无问题，笔记通过；若确定违规，笔记可能被直接封禁；若疑似违规或机器难以判断，则标记并转入人工审核阶段。

![](img/3053c23f1f276406dc71e245e13104ac_12.png)

### 第二阶段：去重检测

![](img/3053c23f1f276406dc71e245e13104ac_14.png)

![](img/3053c23f1f276406dc71e245e13104ac_16.png)

通过机器审核的笔记，会进入去重检测阶段。系统会将你的笔记与平台数据库中的内容进行比对，检查相似度。

![](img/3053c23f1f276406dc71e245e13104ac_18.png)

![](img/3053c23f1f276406dc71e245e13104ac_20.png)

去重检测的核心逻辑可以用以下伪代码表示：
```python
if 笔记相似度 == 100%:
    判定为抄袭，直接封禁
else:
    进入下一阶段
```
这意味着，如果你的笔记图片或文案与他人高度重复，将被识别。通过裁剪图片、添加滤镜、调整文案结构等方式，可以降低被检测出的概率。

![](img/3053c23f1f276406dc71e245e13104ac_22.png)

### 第三阶段：人工审核

![](img/3053c23f1f276406dc71e245e13104ac_24.png)

人工审核主要针对机器标记的“疑似违规”内容进行二次复核。目前，由于内容量巨大且成本高，进入此阶段的笔记比例较小。

以下是人工审核的重点关注领域：
*   政治类内容
*   色情擦边内容
*   其他法律严禁的内容

**注意**：当前系统算法已较为完善，触发明确红线的笔记通常由机器直接处理，人工复核较少。若认为被误判，可通过申诉渠道反馈，但新号应尽量避免违规，以免对账号产生隐形影响。

## 账号权重与流量分发 ⚖️

![](img/3053c23f1f276406dc71e245e13104ac_26.png)

在了解了审核流程后，我们需要理解另一个影响流量分发的核心因素：账号权重。系统会根据账号的综合表现赋予一个隐形的“权重分”，它直接影响笔记的初始推荐量。

![](img/3053c23f1f276406dc71e245e13104ac_28.png)

![](img/3053c23f1f276406dc71e245e13104ac_30.png)

账号权重的概念可以简化为一个从0到10的分数模型：
*   **新号阶段（权重≈0）**：新注册账号权重很低，笔记初始推荐流量有限（如30-40次展示）。此阶段需通过持续发布垂直内容、保持活跃且不违规来“养号”，提升权重。
*   **成熟账号（权重>0）**：当账号持续产出优质内容并被打上领域标签后，权重会提升。高权重账号发布的笔记，能获得更大的初始流量池和更精准的推荐。

![](img/3053c23f1f276406dc71e245e13104ac_32.png)

![](img/3053c23f1f276406dc71e245e13104ac_34.png)

**核心要点**：权重分是系统内部的综合评价，用户不可见。新号初期流量不佳是正常现象，重点在于通过合规、优质、垂直的运营行为逐步积累权重，而非追求单篇笔记的爆发。
