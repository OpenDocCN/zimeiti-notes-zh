# 小红书运营教程：P6：小红书笔记不被收录的原因分析 📝

在本节课中，我们将探讨小红书笔记发布后不被平台收录的常见原因。理解这些原因有助于优化发布策略，确保内容有效触达目标受众。

上一节我们介绍了笔记发布的基本流程，本节中我们来看看哪些因素可能导致笔记无法被正常收录。

## 笔记不被收录的五点原因

平台不收录笔记通常由多种因素导致。以下是五个主要原因的详细分析。

### 1. 存在违规推广行为
笔记内容若包含硬性广告推广，可能直接导致不被收录。这主要指在笔记中直接、生硬地推广商品或品牌。

*   **包含明确的品牌名称或商品信息**。
*   **以广告文案形式直接引导购买**。

### 2. 触犯敏感词检测规则
平台设有敏感词检测机制。笔记或账号若存在违规，即使暂时被收录，后续也可能被系统剔除或限流。

*   **笔记正文或评论中包含平台禁止的敏感词汇**。
*   **账号历史存在违规记录，影响新笔记的初始收录**。

### 3. 内容本身违规（图片与文字）
内容违规是导致笔记不被收录或后续被限流的直接原因，涵盖图片和文字两个方面。

*   **图片违规**：带有其他平台的水印、二维码等引流信息。公式表示为：`违规图片 = 外部水印 + 引流二维码`。
*   **文字违规**：包含不文明词汇、诱导性信息、政治敏感内容、破坏民族团结的言论等。核心原则是避免谈论法律法规明令禁止的话题。

### 4. 文章发送频率异常
短时间内高频次发布笔记，可能触发平台的风控机制，使账号行为被判定为异常。

*   **系统可能将高频发布账号误判为营销机器人**。
*   **账号状态异常会直接影响新笔记的收录**。

### 5. 发布设备已被平台拉黑
平台能够识别并记录发布设备的硬件信息。若该设备曾有过严重违规行为，即使更换账号，新发布的笔记也可能受到牵连。

*   **设备识别依据**：手机IMEI码、网络基带信息等硬件参数。
*   **后果**：设备被标记后，其上登录的任何账号发布内容都可能受到限制。
