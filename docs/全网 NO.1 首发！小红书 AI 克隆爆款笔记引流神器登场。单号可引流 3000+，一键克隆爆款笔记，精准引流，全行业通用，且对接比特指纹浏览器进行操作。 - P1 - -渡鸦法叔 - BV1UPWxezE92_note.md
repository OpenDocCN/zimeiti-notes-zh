## 🔐 账号管理

上一节我们介绍了软件的基本情况，本节中我们来看看如何进行账号管理。账号管理是使用所有功能的基础。

首先，点击“手动登录账号”来存储账号的Cookie（简称CK）。

**操作步骤如下：**
1.  点击“手动登录”后，软件会自动弹出浏览器窗口。
2.  在打开的页面中，使用手机扫码登录你的小红书账号。
3.  登录完成后，点击“确定”，然后关闭浏览器窗口。
4.  此时，系统会成功提取并存储该账号的CK。

除了自动登录，你也可以手动创建浏览器窗口。

**手动创建窗口的步骤如下：**
1.  在界面中找到“新建创建窗口”选项。
2.  点击后，可以手动创建一个新的浏览器窗口。
3.  在这个手动创建的窗口中，你需要自行配置IP地址，并可以使用“一键随机生成指纹”功能。
4.  你可以在此处填入购买的代理IP（例如SK5代理IP），然后点击“确定”。
5.  创建完成后，请记住该窗口的序列号。之后，你可以在发布笔记或克隆功能中，通过选择“窗口类型”来使用它。

**关于窗口使用的注意事项：**
*   手动创建的窗口，需要你手动打开并登录小红书“创作者中心”，以完成CK的登录。
*   两种登录方式各有用途：第一种是填写CK后系统自动登录；第二种是手动登录后，后续操作可以直接选择已登录的浏览器窗口，无需重复新建。
*   每天请注意检查“窗口数”，如果总数或打开数已满，将无法发布新笔记。若已满，可以购买相关套餐进行扩容。
*   点击“更新信息”按钮，可以查看窗口的名称、备注以及对应的平台账号信息。

## 📝 克隆笔记功能

![](img/8a11b029478d9a16c5347cf5a95c300f_3.png)

掌握了账号管理后，我们进入核心功能之一：克隆笔记。此功能可以快速复制爆款笔记。

克隆功能现在无需手动填写CK。你只需要复制目标作品的链接地址。

**操作步骤如下：**
1.  将作品地址粘贴到指定输入框。
2.  点击“获取”按钮，系统会自动抓取笔记内容。
3.  你可以清理掉多余的标签，然后即可准备发布。

**发布配置说明：**
*   **启动线程**：一个线程对应一个账号。启动两个线程，就会同时用两个账号发布。
*   **间隔时间**：设置账号发布之间的时间间隔，单位是毫秒（例如2000代表2秒）。
*   **配置IP**：如果选择全自动模式，可以在这里配置IP。如果已在比特指纹浏览器界面手动配置过IP，则此处无需重复配置。建议首次使用时让系统自动配置，后续可选择“浏览器窗口”方式。
*   **选择窗口**：在发布前，需要选择已登录账号的浏览器窗口（例如窗口1和窗口5）。请确保这些窗口内的账号已提前手动登录。
*   **账号数量**：选择的账号数量需与启动的线程数匹配。
*   **发布模式**：选择“MD5模式”，然后点击“开始发布”。

**重要提示**：发布账号必须绑定手机号，未绑定手机号的账号会发布失败。

## 🤖 自动化发布笔记

克隆功能适用于复制单一笔记，而自动化发布功能则能实现多元素组合的批量发布，功能更加强大。

在自动化发布区域，你可以设置标题库、图片库、内容库和标签库，并进行排列组合发布。

**以下是各库的使用方法：**

1.  **标题库/图片库/内容库**：你可以导入多篇笔记的标题、图片和内容。系统允许你为每次发布选择不同的组合。
    *   **例如**：你可以选择笔记A的标题，搭配笔记B的图片和笔记C的内容，组合成一篇新的去重笔记。
2.  **标签库**：可以自定义添加标签。注意，每个标签需用**英文逗号**隔开。
    *   **例如**：`创业, 大学生创业, 球鞋` 这代表三个独立的标签。

**发布配置的核心规则：**
*   **线程与账号匹配**：`启动线程数` 必须小于或等于 `可用账号数`，并且最好是整除关系。
*   **公式**： `启动线程数` ≤ `可用账号数`，且 `可用账号数` % `启动线程数` == 0 （理想情况）。
*   **举例说明**：
    *   如果你有4个账号，启动2个线程是允许的（发完前2个，会轮换到后2个）。
    *   如果你只有1个账号，却启动3个线程，则不允许，因为会有线程无账号可用。
*   **操作流程**：配置好标题、图片、内容、标签后，设置发布间隔（如2000毫秒），选择已登录的浏览器窗口，然后点击开始发布。

系统将自动按照你的配置组合内容，并利用选定的账号进行发布。你可以在“笔记管理”板块查看发布状态（如“审核中”）。
