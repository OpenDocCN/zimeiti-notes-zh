# 抖音运营高阶教程：P58：账户简介

![](img/df80e7dbcdec5deddc3c51a8b200c341_0.png)

![](img/df80e7dbcdec5deddc3c51a8b200c341_2.png)

在本节课中，我们将学习如何设置一个专业且有效的抖音账户简介。简介是昵称的延伸和补充，是向用户清晰传达你是谁、提供什么价值的关键位置。

![](img/df80e7dbcdec5deddc3c51a8b200c341_4.png)

上一节我们介绍了昵称的设置，本节中我们来看看如何撰写简介。

![](img/df80e7dbcdec5deddc3c51a8b200c341_6.png)

## 简介的核心作用

![](img/df80e7dbcdec5deddc3c51a8b200c341_8.png)

![](img/df80e7dbcdec5deddc3c51a8b200c341_10.png)

简介的核心作用是将昵称进行细分和补充。它需要直接、明确地告诉用户你的身份和你能提供的价值。

![](img/df80e7dbcdec5deddc3c51a8b200c341_12.png)

例如，一个昵称为“装修老白”的账号，其简介应围绕“装修”和“老白”这两个核心关键词展开，进行具体说明。

![](img/df80e7dbcdec5deddc3c51a8b200c341_14.png)

## 简介撰写的三个核心要点

![](img/df80e7dbcdec5deddc3c51a8b200c341_16.png)

以下是撰写简介时需要包含的三个核心要素：

![](img/df80e7dbcdec5deddc3c51a8b200c341_18.png)

1.  **价值体现**
    明确告诉用户关注你能获得什么好处。例如：“帮你装修少踩坑”、“教你变身小仙女”、“实现你的婚礼梦想”。

2.  **人设强化**
    在简介中再次强调你的人设，加深用户印象。例如：“在装修行业摸爬滚打10年的老白”、“专注变美攻略的帅大叔”。

3.  **联系方式（谨慎使用）**
    对于希望引流的账号，可以在简介中留下联系方式，但需特别注意平台规则。

## 关于联系方式的特别注意事项

![](img/df80e7dbcdec5deddc3c51a8b200c341_20.png)

![](img/df80e7dbcdec5deddc3c51a8b200c341_22.png)

在简介和私信中处理联系方式时，必须遵守平台规则，避免账号被处罚。

![](img/df80e7dbcdec5deddc3c51a8b200c341_24.png)

![](img/df80e7dbcdec5deddc3c51a8b200c341_26.png)

以下是需要特别注意的几点：

![](img/df80e7dbcdec5deddc3c51a8b200c341_28.png)

*   **粉丝数低于1万时**：不建议在简介中直接留下联系方式。
*   **私信沟通禁忌**：在回复用户私信时，**绝对不要**直接出现“微信”、“QQ”等敏感词。可以使用“加我号就行”、“看简介”等暗示性语言进行引导。
*   **违规后果**：直接提及“加我微信”等行为可能导致账号被“评级”或限制，影响正常运营。

![](img/df80e7dbcdec5deddc3c51a8b200c341_30.png)

> **核心规则公式**：`粉丝数 < 10000` 时，简介不留联系方式；私信沟通禁用“微信”、“QQ”等直接引流词汇。

![](img/df80e7dbcdec5deddc3c51a8b200c341_32.png)

## 头图尺寸指南

![](img/df80e7dbcdec5deddc3c51a8b200c341_34.png)

![](img/df80e7dbcdec5deddc3c51a8b200c341_36.png)

在完善了昵称和简介后，我们来看看账户的另一个视觉元素——头图。头图是位于个人主页顶部的横幅图片，常用于引导用户关注。

![](img/df80e7dbcdec5deddc3c51a8b200c341_38.png)

首先，了解头图的尺寸要求至关重要。

*   **官方上传尺寸**：`1125像素 × 633像素`
*   **实际显示的核心安全尺寸**：`900像素 × 400像素`

虽然系统要求上传1125×633的图片，但实际展示给用户时，边缘部分会被裁剪，核心内容区域约为900×400。因此，在设计头图时，必须将关键信息和视觉焦点集中在这个安全区域内，避免重要内容被裁切。

本节课中我们一起学习了抖音账户简介的撰写方法与注意事项。我们明确了简介是昵称的细分，需要清晰体现**价值**、强化**人设**，并谨慎处理**联系方式**。同时，我们了解了头图设计的核心尺寸要求。合理设置这些资料，能让你的账号定位更清晰，更有效地吸引目标用户。
