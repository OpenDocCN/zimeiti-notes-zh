# 抖音运营教程：P9：垂直类目的细分领域 🎯

在本节课中，我们将深入探讨“垂直类目”这一核心概念，并学习如何将其细分为不同级别。理解这一点，对于打造精准、有吸引力的抖音账号至关重要。

上一节我们介绍了内容标签的重要性，本节中我们来看看如何将“垂直”这个概念进行更细致的划分。

我们常说做账号要“垂直”，但垂直本身也有不同的深度和级别。简单地选择一个大的方向（如美食、游戏）只是第一步。

以下是垂直类目的分级体系，我们以“游戏”领域为例进行说明：

*   **一级垂直领域**：这是最宽泛的分类。例如，你决定做“游戏”内容。在这个级别，你可以涵盖手游、网游、电竞等所有游戏类型。
    *   **公式表示**：`垂直领域级别1 = 游戏`

*   **二级垂直领域**：在一级分类下进行更具体的划分。例如，在“游戏”这个一级分类下，你选择专注于“手游”。
    *   **公式表示**：`垂直领域级别2 = 游戏 -> 手游`

*   **三级垂直领域**：在二级分类的基础上，进一步聚焦到具体的游戏产品。例如，在“手游”这个二级分类下，你选择只做“王者荣耀”或“和平精英”的内容。
    *   **公式表示**：`垂直领域级别3 = 游戏 -> 手游 -> 王者荣耀`

垂直的划分甚至可以继续深入，达到四级或更细。例如：

*   在“王者荣耀”下，只制作关于“安琪拉”这个英雄的教学或精彩集锦。
*   在“和平精英”下，只发布“雨林地图”的攻略，或只创作游戏内的“剧情类”短视频。

大领域和小领域都可以作为你的选择，这完全取决于你自己的兴趣和规划。理解垂直细分，能帮助你吸引更精准、更忠实的粉丝群体。
