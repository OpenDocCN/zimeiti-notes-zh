# 视频号运营教程1000集：P14：8.短视频变声与配音的两种方法

在本节课中，我们将学习为短视频进行声音驱重的两种核心方法。无论是通过剪辑软件直接变声，还是借助工具重新生成配音，都能有效提升视频的原创度，避免内容重复。掌握这些技巧对于视频混剪至关重要。

上一节我们介绍了视频剪辑的基础操作，本节中我们来看看如何具体处理视频中的声音，以实现有效的驱重。

## 方法一：在剪映中直接变声

![](img/00ffe87aa33b53ec22c8444240a9f723_1.png)

第一种方法是直接在剪映App内对视频原声进行变声处理。这种方法操作快捷，能改变人声的音色。

操作步骤如下：

1.  将视频素材导入剪映。
2.  选中时间轴上的视频片段。
3.  在屏幕下方滑动工具栏，找到并点击 **“声音效果”**。
4.  在变声效果列表中，选择你喜欢的效果，例如“大叔”、“台湾小哥”等。
5.  部分效果支持调节参数，你可以进一步微调以达到理想效果。

通过这种方式，视频中的人声发生了变化，但背景音乐保持不变。在进行视频混剪时，建议都进行此类处理以增加原创性。

## 方法二：使用配音神器重新生成配音

![](img/00ffe87aa33b53ec22c8444240a9f723_3.png)

接下来我们看看第二种方法。如果希望彻底改变声音，包括背景音效，可以使用配音工具提取文案并生成全新配音。

![](img/00ffe87aa33b53ec22c8444240a9f723_5.png)

以下是具体操作流程：

1.  **提取视频文案**：打开配音神器App，导入目标视频，使用“文案提取”功能获取视频中的文字内容。
2.  **修改文案**：对提取出的文案进行适当修改。这是驱重的关键一步。
    *   **修改原则**：替换近义词、调整语序、改变句式，但需保持原意通顺。
    *   **示例**：将原文“老人”改为“大人”，“怎么办呢？”改为“咋办呢？”，“水果皮就下来了”改为“水果皮就削掉了”。
3.  **生成新配音**：将修改后的文案复制到配音神器的“制作配音”页面。
    *   选择一个你喜欢的声音角色（如“男生”）。
    *   点击“合成配音”。
4.  **导出并使用新配音**：
    *   将合成好的配音导出并下载到手机。
    *   重新打开剪映，导入原视频，**关闭原声**。
    *   点击“添加音频” -> “提取音乐”，选择刚刚下载好的新配音文件导入。

![](img/00ffe87aa33b53ec22c8444240a9f723_7.png)

至此，视频拥有了全新的配音。此时，你还可以为视频添加新的背景音乐，并调整音量平衡。最后，可以运用之前学过的混剪方法，对画面进行进一步处理。

本节课中我们一起学习了为短视频实现声音驱重的两种实用方法：在剪映内直接变声，以及通过配音神器提取文案并生成全新配音。掌握这两种方法，能有效提升你混剪视频的原创度和独特性。
