# 微信视频号教程：P23：二创书单制作方法一 📚

在本节课中，我们将学习如何为书单类视频进行二次创作。这是一种通过替换、重组现有视频素材来制作原创内容的方法，比直接混剪的原创度更高，非常适合新手入门。

---

## 概述

剪辑是制作任何类型视频的必备技能。对于新手而言，初期的不熟悉感是正常的，通过练习十几个视频，熟悉每个功能，就能快速掌握。本节课程将重点讲解书单类视频的二次创作方法，这是一种稳定且容易出单的视频类型。

---

## 书单视频的底层逻辑

上一节我们介绍了混剪的基本概念，本节中我们来看看书单类视频的创作逻辑。

书单账号的运营逻辑是通过一段影视剧片段、故事或道理阐述，最终引出一本书的推荐，从而激发观众的购买欲望。这种形式比直接展示产品功能的“硬广”更自然，转化效果也更好。

## 如何寻找素材与对标账号

![](img/ee53e7779b67b04e9b5edb9db4f1761e_5.png)

以下是寻找素材和确定带货书单的步骤：

1.  **确定书单**：可以直接搜索一些热门书籍作为素材来源。例如：
    *   孩子为自己读书
    *   墨菲定律漫画版
    *   中国通史
    *   走遍中国
    *   告别吼叫的养育话术

2.  **搜索对标账号**：在抖音等平台直接搜索上述书名，进入“商品”页面并按“销量”排序。销量高的商品下方通常有“达人秀”板块，里面聚集了大量正在带货该书的优质视频账号。

![](img/ee53e7779b67b04e9b5edb9db4f1761e_7.png)

![](img/ee53e7779b67b04e9b5edb9db4f1761e_9.png)

3.  **分析账号逻辑**：多刷这些对标账号的视频，理解他们如何组织素材、讲述故事并引出书籍。你会发现，这些视频的素材大多来源于网络，通过二次创作组合而成。

## 二次创作实操步骤

理解了逻辑后，我们进入具体的操作环节。以下是二次创作一个书单视频的完整流程：

![](img/ee53e7779b67b04e9b5edb9db4f1761e_17.png)

1.  **获取原视频与文案**：
    *   在对标账号中找到点赞量高的视频，复制链接。
    *   使用去水印工具（如“配音神器”）保存视频到相册。
    *   使用文案提取工具，将视频的配音文案提取出来。

2.  **生成新配音**：
    *   检查并修正提取出的文案错别字。
    *   使用配音软件（如剪映内的配音功能或第三方工具），选择合适的声音将文案重新合成配音。
    *   如果觉得某段语速过快，可以添加停顿。例如，添加 `1.5` 秒的停顿。
    *   试听并导出新的配音文件。

3.  **在剪映中处理原视频**：
    *   导入去水印后的原视频。
    *   关闭视频原声。
    *   使用**蒙版**功能遮盖原视频的字幕。具体操作：选中视频片段 -> 点击“蒙版” -> 选择“矩形”调整覆盖字幕区域 -> 点击“反转” -> 设置“画布模糊”作为背景。

![](img/ee53e7779b67b04e9b5edb9db4f1761e_19.png)

4.  **替换视频素材片段**：
    *   **思路**：将原视频中的各个场景片段，用其他类似素材替换。
    *   **方法一（替换）**：分割出想要替换的片段，点击“替换”，从“素材库”搜索关键词（如“老师讲课”）选择新素材。
    *   **方法二（删除并新增）**：删除原片段，在相同时间点导入自己找到的新素材。确保时长大致相同。
    *   **素材来源**：
        *   剪映自带“素材库”。
        *   直接在抖音搜索相关场景关键词（如“女生台上培训”），保存合适的无水印片段使用。
    *   **关键操作**：务必关闭新增素材的原声，并调整画面至全屏。

![](img/ee53e7779b67b04e9b5edb9db4f1761e_21.png)

![](img/ee53e7779b67b04e9b5edb9db4f1761e_23.png)

5.  **组合新视频并添加音频**：
    *   将所有替换好的片段按顺序排列。
    *   点击“添加音频” -> “提取音乐”，导入第2步中生成的新配音文件。
    *   根据画面内容，微调音频片段的位置（分割、移动），使音画同步。

6.  **添加背景音乐与最终字幕**：
    *   添加合适的背景音乐，调低音量，避免掩盖配音。
    *   点击“文本” -> “识别字幕”，为新的配音生成字幕。
    *   将新字幕调整到屏幕中合适的位置（通常底部），并为字幕选择美观的样式，点击“应用到全部”。

7.  **导出视频**：
    *   检查无误后，点击“导出”，将帧率设置为 `60` 帧，以获得更流畅的画面效果。

## 总结

![](img/ee53e7779b67b04e9b5edb9db4f1761e_31.png)

本节课我们一起学习了书单视频二次创作的全流程。核心在于**理解“素材替换与重组”的创作思路**，通过获取对标视频、生成新配音、替换视觉素材、添加新字幕等步骤，将已有的视频素材重新组合成原创度更高的新作品。

![](img/ee53e7779b67b04e9b5edb9db4f1761e_33.png)

![](img/ee53e7779b67b04e9b5edb9db4f1761e_35.png)

记住公式：**二次创作 = 原有文案/逻辑 + 替换的视觉素材 + 新的配音/字幕**。这种方法比简单混剪更有效，重在多练习、多实操。现在，你可以尝试使用这个方法，制作你的第一个二创书单视频了。
