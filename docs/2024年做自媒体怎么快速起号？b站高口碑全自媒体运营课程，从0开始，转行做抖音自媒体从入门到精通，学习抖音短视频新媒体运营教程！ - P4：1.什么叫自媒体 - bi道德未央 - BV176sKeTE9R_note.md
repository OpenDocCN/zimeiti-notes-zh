## 什么是自媒体？

上一节我们介绍了课程的整体框架，本节中我们来看看最核心的概念：到底什么是自媒体？

![](img/69cfbe6f86f1d20fe55019eaaaa21285_5.png)

我用最直白的话来解释：
**自媒体** 就是：**个人** 通过 **互联网平台** 发布 **原创内容**，并以此获取 **流量** 与 **收益** 的媒体形式。

![](img/69cfbe6f86f1d20fe55019eaaaa21285_7.png)

我们可以将其拆解为一个简单的操作流程：

![](img/69cfbe6f86f1d20fe55019eaaaa21285_9.png)

1.  打开手机或电脑。
2.  搜索并进入一个自媒体平台（如：`百家号`、`头条号`、`B站`、`抖音`等）。
3.  在该平台注册一个属于你自己的账号。
4.  在你的账号上发布自己创作的文章或视频。
5.  内容获得观看（阅读），你便可以依靠这些流量来结算收益。

这个过程，就叫做运营自媒体。其核心逻辑可以概括为：
`个人创作 -> 平台发布 -> 获取流量 -> 实现变现`
