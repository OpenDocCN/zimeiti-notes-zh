## 笔记被推荐的核心逻辑 🚀

笔记发布后，平台会根据一系列数据指标决定是否给予更多流量。这个过程可以理解为“流量升级”。核心逻辑主要围绕两个关键数据指标：**打开率**和**互动率**。

### 1. 打开率 (Click-Through Rate)

打开率是决定笔记能否进入下一级流量池的首要指标。

*   **定义**：当笔记出现在用户“发现页”时，被用户点击查看的比例。
*   **计算方式**：`打开率 = (笔记点击量 / 笔记展示量) * 100%`
*   **核心阈值**：平台的平均打开率门槛约为 **11%**。
*   **运作机制**：平台会将你的笔记与同批次（约20条）相似笔记进行对比。如果你的笔记打开率超过11%，系统会判定其有吸引力，从而将其推入更高层级的流量池，获得更多展示（例如从1000展示升至2000-5000展示）。若低于此阈值，则通常只有一波基础推荐。

### 2. 互动率 (Engagement Rate)

互动率是笔记在通过第一级筛选后，能否继续获得更高推荐的关键。

![](img/743ed1c75c7b871c7c790b12f8fe4ef5_9.png)

*   **定义**：用户点击进入笔记后，进行点赞、收藏、评论、转发等互动行为的比例。
*   **计算方式**：`互动率 = (互动总量 / 笔记点击量) * 100%`
*   **核心阈值**：平均互动率门槛约为 **3%**。
*   **运作机制**：当笔记因高打开率进入更大流量池后，系统会考核其互动率。如果互动率持续超过3%，笔记将有机会被推荐至更高层级的流量池（例如从数千展示升至数万甚至十万级展示）。

![](img/743ed1c75c7b871c7c790b12f8fe4ef5_11.png)

![](img/743ed1c75c7b871c7c790b12f8fe4ef5_13.png)

![](img/743ed1c75c7b871c7c790b12f8fe4ef5_15.png)

**总结升级路径**：笔记发布 → 考核**打开率**（>11%）→ 进入更大流量池 → 考核**互动率**（>3%）→ 进入更高层级流量池。新账号通常有2-3次升级机会，而高权重老账号的起点更高，升级潜力更大。

![](img/743ed1c75c7b871c7c790b12f8fe4ef5_17.png)

除了打开率和互动率，平台还有一个综合评分系统，用于评估笔记的长期价值。

![](img/743ed1c75c7b871c7c790b12f8fe4ef5_19.png)

以下是影响综合评分的主要互动行为及其权重（由高到低）：
1.  **分享**：用户将笔记分享给站外好友或平台内。
2.  **收藏**：用户认为笔记有价值，便于日后查看。
3.  **评论**：用户产生深度交流，提升笔记活跃度。
4.  **点赞**：用户表达对笔记的基本认可。
5.  **关注**：用户因笔记内容关注账号，属于长期价值。

**请注意**：综合评分建立在打开率和互动率达标的基础上。如果基础数据不达标，即使有少量点赞收藏，笔记也难以获得持续推荐。

## 必须避免的运营“坑” ⚠️

了解推荐逻辑后，我们还需要规避一些常见错误，以防账号被降权或限流。

![](img/743ed1c75c7b871c7c790b12f8fe4ef5_21.png)

以下是新手运营小红书时必须注意的禁忌事项：
1.  **新号勿立刻发笔记**：新注册账号需要“养号”，模拟真实用户行为（浏览、互动）一段时间，建立基础画像后再发布内容。立即发笔记会消耗平台初始信任度，导致后续流量低迷。
2.  **违规必降权**：任何违规行为（如导流、抄袭、虚假宣传）都会导致账号被降权，笔记推荐和搜索排名大幅下降，恢复难度极大。
3.  **标签忌反复修改**：账号、店铺或笔记的标签一旦确定，并积累了一定流量后，切勿频繁修改。大幅修改标签会导致系统重新识别，可能引发限流。
4.  **一机登录多账号**：尽量避免一台手机频繁切换登录多个小红书账号，尤其是营销号。建议遵循“一机一卡一账号”原则，降低关联风险。
5.  **谨慎处理引流内容**：在笔记或私信中直接留联系方式、引导至其他平台需非常谨慎，极易被判为营销导流而违规。
6.  **妥善处理留言私信**：及时、友善地回复评论和私信，但避免出现敏感词或违规承诺。对于恶意评论，建议隐藏而非激烈对骂。
