# 小红书运营教程：P36：34.卡点视频 🎬

![](img/523e2780a42518c6e0de864c46533704_1.png)

在本节课中，我们将学习如何制作一个音乐卡点视频。这是一种节奏感强、画面切换与音乐节拍精准同步的视频形式，能有效提升内容的吸引力和专业感。我们将从导入素材开始，逐步讲解如何添加音乐、设置卡点、调整画面以及添加转场和特效，最终完成一个完整的卡点视频。

![](img/523e2780a42518c6e0de864c46533704_3.png)

## 导入与整理素材

上一节我们介绍了视频制作的基础概念，本节中我们来看看如何开始一个卡点视频项目。首先需要将准备好的素材导入到编辑软件中。

![](img/523e2780a42518c6e0de864c46533704_5.png)

在素材库面板中，单击“导入”按钮。

![](img/523e2780a42518c6e0de864c46533704_7.png)

![](img/523e2780a42518c6e0de864c46533704_1.png)

![](img/523e2780a42518c6e0de864c46533704_9.png)

选择所有需要使用的照片素材，然后单击“打开”。

![](img/523e2780a42518c6e0de864c46533704_11.png)

![](img/523e2780a42518c6e0de864c46533704_3.png)

将这些照片全部拖拽到下方的时间轴中。

![](img/523e2780a42518c6e0de864c46533704_13.png)

![](img/523e2780a42518c6e0de864c46533704_5.png)

以下是快速选中多个素材的两种方法：
*   **框选法**：在素材库的空白处单击并拖拽鼠标，形成一个矩形选区，矩形内的素材会被全部选中。
*   **Shift键连选法**：单击第一个素材，然后按住 `Shift` 键，再单击最后一个素材，这之间的所有连续素材都会被选中。

![](img/523e2780a42518c6e0de864c46533704_15.png)

选中全部素材后，将其拖入时间轴。

![](img/523e2780a42518c6e0de864c46533704_13.png)

![](img/523e2780a42518c6e0de864c46533704_17.png)

## 调整视频比例与画面

素材导入后，我们需要调整视频的整体比例以适应发布平台（如小红书的9:16竖屏比例）。

单击比例设置，选择 **9:16**。此时，照片可能无法完美适配新比例。

![](img/523e2780a42518c6e0de864c46533704_15.png)

我们需要逐一调整每个画面的大小和位置，使其充满画面且构图合适。你可以预览并选择效果最好的调整方案。

## 添加音乐与设置卡点

画面准备就绪后，接下来就是卡点视频的核心——添加音乐并设置节拍点。

在顶部工具栏中选择“音频”，然后进入“音乐素材”库。为了制作卡点视频，我们可以直接在“卡点”分类中挑选节奏感强的音乐。试听后，将选中的音乐拖拽到时间轴的音频轨道上。

![](img/523e2780a42518c6e0de864c46533704_17.png)

![](img/523e2780a42518c6e0de864c46533704_19.png)

选中音频轨道，在上方工具栏中会出现卡点工具。我们主要使用“自动卡点”功能。

![](img/523e2780a42518c6e0de864c46533704_21.png)

单击“自动卡点”，软件会提供选项，如“踩节拍I”或“踩节拍II”。它们代表不同的节拍密度。
*   **`踩节拍I`**：节拍点较稀疏。
*   **`踩节拍II`**：节拍点更密集。

选择后，音频轨道下方会出现一系列黄色小点，这就是软件自动识别的音乐节拍位置，也是我们画面切换的参考点。

![](img/523e2780a42518c6e0de864c46533704_19.png)

![](img/523e2780a42518c6e0de864c46533704_23.png)

你可以使用 `Ctrl + 鼠标滚轮` 来放大时间轴，更清晰地查看这些卡点标记。

![](img/523e2780a42518c6e0de864c46533704_25.png)

## 对齐素材与剪辑

![](img/523e2780a42518c6e0de864c46533704_27.png)

现在，我们将图片的切换点与音乐的卡点对齐。

![](img/523e2780a42518c6e0de864c46533704_29.png)

将时间轴上的每张图片拖拽，使其结束的边缘与黄色的卡点标记对齐。拖拽时，素材会自动吸附到卡点位置，方便精确操作。

![](img/523e2780a42518c6e0de864c46533704_31.png)

![](img/523e2780a42518c6e0de864c46533704_25.png)

![](img/523e2780a42518c6e0de864c46533704_33.png)

确保所有图片都按此规则对齐。

![](img/523e2780a42518c6e0de864c46533704_29.png)

![](img/523e2780a42518c6e0de864c46533704_35.png)

图片对齐完成后，音乐可能有多余的部分。选中音频，在合适的位置使用“分割”工具（快捷键通常是 `Ctrl+B` 或 `Cmd+B`）切断，然后删除后面的部分。

## 添加转场与特效

![](img/523e2780a42518c6e0de864c46533704_37.png)

![](img/523e2780a42518c6e0de864c46533704_39.png)

目前，图片之间的切换可能显得生硬。为了让过渡更流畅，我们需要添加转场效果。

在顶部工具栏选择“转场”，从库中挑选喜欢的转场效果（例如“眩光”）。将其拖拽到时间轴上两张图片之间的位置即可添加。

![](img/523e2780a42518c6e0de864c46533704_41.png)

![](img/523e2780a42518c6e0de864c46533704_37.png)

![](img/523e2780a42518c6e0de864c46533704_43.png)

为了增加视频的趣味性和视觉冲击力，我们还可以添加一些特效。

选择“特效” -> “人物特效”，从中选择一个适合的效果（如“电光描边”），将其拖拽到时间轴的视频轨道上。

![](img/523e2780a42518c6e0de864c46533704_45.png)

![](img/523e2780a42518c6e0de864c46533704_45.png)

![](img/523e2780a42518c6e0de864c46533704_47.png)

## 预览与完成

![](img/523e2780a42518c6e0de864c46533704_49.png)

至此，一个简单的音乐卡点视频就制作完成了。点击播放预览整体效果，检查画面切换是否与音乐节拍吻合，转场和特效是否自然。

![](img/523e2780a42518c6e0de864c46533704_47.png)

![](img/523e2780a42518c6e0de864c46533704_51.png)

本节课中我们一起学习了制作音乐卡点视频的全流程。核心在于利用软件的 **`自动卡点`** 功能快速定位音乐节拍，并将画面切换点与之精确对齐。通过调整比例、添加转场和特效，可以让视频更加专业和生动。课后请多加练习，熟练掌握这些操作。
