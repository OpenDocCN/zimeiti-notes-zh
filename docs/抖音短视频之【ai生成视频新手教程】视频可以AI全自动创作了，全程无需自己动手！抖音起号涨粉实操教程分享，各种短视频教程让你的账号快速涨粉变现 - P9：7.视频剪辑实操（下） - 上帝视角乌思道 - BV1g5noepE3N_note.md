# AI短视频创作教程：P9：7.视频剪辑实操（下）

在本节课中，我们将继续学习视频剪辑的实操部分，重点讲解如何为拼接的视频片段添加转场效果，以及如何为视频添加字幕。这些步骤对于提升视频的流畅度和观看体验至关重要。

上一节我们介绍了如何调整视频比例和背景，本节中我们来看看如何让多个视频片段之间衔接得更自然。

![](img/0ab44bb2d70145caaf31f45a9297fb16_1.png)

## 添加转场效果

![](img/0ab44bb2d70145caaf31f45a9297fb16_3.png)

我们剪辑的视频通常由多个片段组成，每个片段时长约十秒。这些片段之间需要有自然的衔接，因此必须添加转场。

![](img/0ab44bb2d70145caaf31f45a9297fb16_5.png)

转场的作用是让用户明确感知到视频内容进入了另一个不同的片段。如果不添加转场，观众在切换到下一个片段时可能会感到突兀，甚至退出视频查看详情。这种行为会直接导致视频的**完播率**下降，而完播率是影响视频收益和推荐权重的重要指标。

![](img/0ab44bb2d70145caaf31f45a9297fb16_7.png)

以下是添加转场的具体步骤：

![](img/0ab44bb2d70145caaf31f45a9297fb16_9.png)

1.  在视频编辑轨道右侧，找到一个**加号 (+)** 按钮。点击它。
2.  在弹出的菜单中，选择**“素材库”**。
3.  在素材库的“热门”分类中，可以找到大量转场素材，例如搞笑的动态画面或特效。
4.  选择转场时，请注意时长。避免选择过长的转场（如16秒），观众通常没有耐心看完。建议选择**3~5秒**时长的转场素材。
5.  选中合适的转场（例如一个3秒的素材），点击添加。

![](img/0ab44bb2d70145caaf31f45a9297fb16_11.png)

![](img/0ab44bb2d70145caaf31f45a9297fb16_13.png)

转场素材添加后，同样需要调整其比例以匹配主视频。

1.  选中已添加的转场片段，在编辑界面找到**“画布模糊”**或背景设置选项。
2.  选择第一个清晰的背景样式，确保转场画面也变为**16:9**的横屏比例。
3.  最后，点击右下角的**对勾 (✓)** 确认设置。

![](img/0ab44bb2d70145caaf31f45a9297fb16_15.png)

![](img/0ab44bb2d70145caaf31f45a9297fb16_17.png)

至此，视频的比例调整、背景统一以及转场添加就完成了。记住，每一步都需严格按照要求操作，以确保最终成片的质量。

## 添加视频字幕

视频剪辑的下一步是为内容添加字幕。字幕能提升视频的可理解性，尤其在不方便外放的场景下。

![](img/0ab44bb2d70145caaf31f45a9297fb16_19.png)

以下是添加字幕的具体步骤：

![](img/0ab44bb2d70145caaf31f45a9297fb16_21.png)

1.  在手机编辑界面左下角，找到并点击**“文字”**选项。
2.  点击**“新建文本”**，在右侧选择**“识别字幕”**功能。
3.  点击**“开始匹配”**，软件会自动识别视频中的语音，并生成对应的文字字幕。
4.  字幕生成后，可以进行样式调整。拖动字幕文本框**右下角**的控制点，可以放大字幕。
5.  将放大后的字幕文本框，拖动到视频画面最下方的位置，这样观看最舒适。
6.  调整好第一句字幕后，后续字幕通常会**自动应用**相同的样式和位置，非常方便。

## 核心要点与总结

![](img/0ab44bb2d70145caaf31f45a9297fb16_23.png)

本节课中我们一起学习了视频剪辑后半部分的核心操作。

![](img/0ab44bb2d70145caaf31f45a9297fb16_25.png)

*   **转场的作用**：衔接不同片段，明确内容转换，提升完播率。公式表示为：**完播率 ↑ → 视频收益与推荐权重 ↑**。
*   **转场的选择**：从素材库“热门”中选取，时长控制在**3~5秒**。
*   **字幕的添加**：使用“识别字幕”功能自动生成，并统一调整样式和位置。

![](img/0ab44bb2d70145caaf31f45a9297fb16_27.png)

整个视频制作流程要求严谨：从寻找合适时长的素材，到统一画面比例，再到添加转场和字幕，每一步都需认真对待。平台会对视频内容进行审核，规范的操作能有效提高审核通过率。只要按照步骤操作，视频制作并不困难。
