# 视频号运营教程：P38：如何制作直播间贴图 🎨

在本节课中，我们将学习如何为直播间制作一个透明的文字贴图。这种贴图常用于展示商品信息或活动标语，能有效吸引观众注意并传递关键信息。

---

上一节我们介绍了直播间的基本设置，本节中我们来看看如何制作一个实用的直播间贴图。

制作这种贴图，我们可以使用一款名为“微麦水印相机”的手机应用。这款应用操作简单，无需开通会员即可完成基础制作。

以下是具体的操作步骤：

1.  **下载应用**：在手机应用商店搜索并下载“微麦水印相机”。
2.  **选择模板**：打开应用，找到“水印库”或“更多水印”入口。在水印模板中，选择“横竖文字”类别的模板。
3.  **编辑文字内容**：点击选中的模板，进入编辑页面。将模板中的示例文字替换为你需要展示的信息，例如“好物推荐”、“老国货品牌”以及具体的价格组合。
4.  **自定义样式**：你可以调整文字的颜色、字体，使其更符合直播间的风格。
5.  **关键设置：背景透明**：找到“背景色”设置选项。**必须将背景设置为完全透明**。通常应用内会有一个带斜杠的图标或“无背景”选项，选中它，确保背景不显示任何颜色。
6.  **生成并保存**：编辑完成后，点击“生成”或“保存”按钮，将制作好的透明贴图保存到手机相册。

---

现在我们已经制作好了贴图，接下来看看如何在直播中添加它。

1.  **发起直播**：在视频号或其他直播平台内，点击“发起直播”。
2.  **添加贴图**：在直播设置或画面调整界面中，找到“贴图”、“图片”或“画面叠加”功能。
3.  **选择图片**：从手机相册中选择你刚刚制作并保存好的透明贴图。
4.  **调整位置与大小**：将贴图添加到直播画面后，你可以拖动它来调整位置，并通过捏合手势来缩放大小，将其放置在合适且不遮挡主要内容的位置。

本节课中我们一起学习了直播间透明贴图的完整制作与添加流程。总结起来，核心步骤是：使用“微麦水印相机”应用选择模板、编辑文字、**将背景设置为透明**，最后在直播中添加并调整贴图。掌握这个技巧，能让你的直播间信息展示更加专业和醒目。
