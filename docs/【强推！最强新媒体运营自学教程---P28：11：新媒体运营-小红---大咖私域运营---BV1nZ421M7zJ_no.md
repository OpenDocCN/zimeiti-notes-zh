# 新媒体运营：P28：11：小红书平台用户标签及流量入口 📊

![](img/1cf663b11ca254af33d574d0b29b0d8a_1.png)

在本节课中，我们将深入探讨小红书平台如何为用户打上标签，以及笔记是如何通过不同的流量入口被推荐给用户的。理解这些底层逻辑，将帮助你更好地规划内容，使其触达更精准的受众。

上一节我们介绍了平台如何判定内容标签，本节中我们来看看平台如何判定用户标签。这个过程能帮助我们反向推导，理解自己的笔记是如何被系统推荐给目标用户的。

## 用户标签的判定逻辑 🔍

小红书系统为用户打标签，主要依据是用户的**浏览行为**。用户注册时选择的兴趣范围，仅作为初期内容方向的引导，参考价值不大，无法进入系统深层的用户标签池。

系统通过分析用户的一系列互动行为来判定其兴趣分类。以下是主要的判定维度：

### 1. 点击率
**点击率**是指系统在发现页向用户推送多个不同类别的内容（如明星、美妆、家居、母婴等）后，用户点击各类内容的频率。例如，如果用户点击了10条内容，其中美妆类3次，家居类2次，母婴类0次，系统就会判定该用户对母婴内容不感兴趣，并可能持续推送更多美妆内容进行试探。通过点击率，系统能为用户打上**强关联**或**弱关联**的标签。

### 2. 停留时长
**停留时长**指用户点开一篇笔记后观看的时长。即使同属美妆领域，如果用户对某篇笔记很快划走，说明内容不吸引他。系统会继续推荐该领域下的其他细分内容（如上妆、护肤）进行测试。反之，如果用户对一篇平常很少点击的家居类笔记停留很久，甚至仔细阅读评论，系统就可能判断他有装修或购买家居用品的潜在需求。

### 3. 页面浏览量（PV）
**PV** 可以理解为用户浏览的页面深度。例如，用户不仅看了一篇笔记，还进入了博主主页，并连续看了第二、第三篇笔记。提升用户在你主页的浏览深度，能有效增加账号的健康度评分，这对新账号尤其有益。

### 4. 互动行为
互动行为包括**点赞、收藏、评论、关注**。这些数据直接关系到笔记能否进入更大的推荐流量池，也是账号接单、变现能力的重要参考。高的互动数据不仅能提升单篇笔记的流量，还可能带动账号内其他笔记的曝光。

## 平台的流量分发入口 🚪

理解了用户标签，我们再来看看小红书主要的流量分发入口。平台主要通过以下四个页面将内容分发给用户：

### 1. 发现页
发现页是流量最主要的来源。其内容构成如下：
*   **关注达人内容**：你已关注的博主发布的新笔记。
*   **兴趣内容匹配**：系统根据你的点击率、停留时长等行为标签，认为你可能喜欢的内容。
*   **发散性内容匹配**：系统为了保持平台内容多样化，偶尔会推荐一些你可能未接触过但潜在感兴趣的内容。

### 2. 关注页
关注页的内容相对单一，仅展示你**已关注博主**发布的笔记。虽然用户直接刷关注页的频率可能不高，但它是一个稳定的粉丝触达渠道。

### 3. 附近页
附近页基于**手机定位**，推送同城或附近的相关内容，如本地美食、旅游、线下店铺等。对于经营线下实体店（如美容院、餐厅）的账号，在运营或投放信息流广告时，需要重点关注这个入口。

### 4. 购物页
购物页是小红书向电商化发展的重要入口，主要满足用户的购物需求。目前，用户更习惯通过搜索笔记来“种草”，直接在商城购物的习惯仍在培养中。这个入口与直播带货、商品笔记等内容紧密相关。
