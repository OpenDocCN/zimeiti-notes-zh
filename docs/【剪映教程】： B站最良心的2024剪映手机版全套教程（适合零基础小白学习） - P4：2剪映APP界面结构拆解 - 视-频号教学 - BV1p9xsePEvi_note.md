# 剪映教程：P4：剪映APP界面结构拆解 📱

![](img/59ce72b1f5fc9c9a3af9b088107c5938_1.png)

在本节课中，我们将要学习剪映APP的界面结构。了解软件的各个功能模块及其具体含义，是开启视频剪辑之路的第一步。

## 概述

剪映APP的界面设计非常简洁，操作简单。它主要包含四个核心模块：**剪辑**、**剪同款**、**消息**和**我的**。熟悉这四个模块是我们正确使用剪映APP的基础。

## 界面四大模块详解

![](img/59ce72b1f5fc9c9a3af9b088107c5938_3.png)

以下是剪映APP首页的四个主要功能模块介绍。

### 1. 剪辑 ✂️
启动剪映APP后，首先进入的首页就是“剪辑”页面。如果我们要剪辑视频，需要点击顶部的 **`开始创作`** 按钮，创建新的短视频项目，并将素材导入其中。
在首页界面中，我们还可以看到之前剪辑过的项目草稿。

![](img/59ce72b1f5fc9c9a3af9b088107c5938_5.png)

![](img/59ce72b1f5fc9c9a3af9b088107c5938_6.png)

![](img/59ce72b1f5fc9c9a3af9b088107c5938_7.png)

### 2. 剪同款 🎬
“剪同款”功能内置了大量短视频模板。对于新手来说，如果想制作抖音上的爆款视频但技术有限，可以使用此功能。
我们只需选择喜欢的模板，按照要求填入自己的视频素材，即可快速生成同款视频。模板按类别划分，如卡点视频、毕业季、儿童节等。

![](img/59ce72b1f5fc9c9a3af9b088107c5938_9.png)

### 3. 消息 💬
剪映APP与抖音账号互通。授权登录抖音账号后，我们可以在剪映的“消息”模块中，同步查看抖音的官方通知、点赞和评论等信息。

### 4. 我的 👤
“我的”是个人信息页面。成功授权抖音账号登录后，可以在此查看个人账号信息，如头像、名称、抖音号、粉丝量、点赞量等，并可直接跳转到抖音账号主页。

![](img/59ce72b1f5fc9c9a3af9b088107c5938_11.png)

![](img/59ce72b1f5fc9c9a3af9b088107c5938_13.png)

## 软件界面实操演示

上一节我们介绍了四大模块的功能，本节中我们来看看软件的实际操作界面。

![](img/59ce72b1f5fc9c9a3af9b088107c5938_15.png)

### 剪同款功能演示
启动剪映APP，默认进入“剪同款”页面。这里展示了官方推荐的大量模板，例如热门、卡点、Vlog等类别。
1.  选择一个模板（如“旅行Vlog”）进行预览。
2.  点击右下角的 **`剪同款`** 按钮。
3.  根据提示导入所需数量的素材片段。
4.  导入后，可以对视频素材进行简单编辑，如替换、裁剪。
5.  也可以编辑模板中的文字内容。
6.  预览效果无误后，点击右上角 **`导出`**。建议将分辨率设置为 **`1080P`**，然后等待渲染完成即可。

![](img/59ce72b1f5fc9c9a3af9b088107c5938_17.png)

### 剪辑功能界面
回到首页，点击“剪辑”模块顶部的 **`开始创作`**，导入素材并添加到项目，即可进入剪辑主界面。
剪辑界面主要分为三个部分：
1.  **视频预览窗口**：位于界面顶部，用于预览视频效果。
2.  **视频轨道（时间轴）**：位于界面中部。视频是按时间顺序连续播放的画面，时间轴就是控制这些画面顺序的轨道。其中有一条白色的 **`时间线`**，它是进行裁剪、添加转场等操作时的基准位置。
3.  **底部工具栏**：提供各种剪辑功能。一级工具栏包括：**`剪辑`**、**`音频`**、**`文字`**、**`画中画`**、**`特效`**、**`滤镜`**、**`比例`**、**`调节`** 等。我们需要先选中轨道上的素材，然后使用这些工具进行编辑。

![](img/59ce72b1f5fc9c9a3af9b088107c5938_19.png)

![](img/59ce72b1f5fc9c9a3af9b088107c5938_21.png)

### 消息与我的模块
*   **消息**：在此可以查看来自抖音的互动通知。
*   **我的**：在此页面可以管理个人账号信息，并查看在剪映中点赞和收藏过的视频模板。

## 总结

![](img/59ce72b1f5fc9c9a3af9b088107c5938_29.png)

本节课中，我们一起学习了剪映APP的界面结构，它主要由 **剪辑**、**剪同款**、**消息** 和 **我的** 四个模块组成。对于内容创作者而言，**剪辑**和**剪同款**是两个最核心的功能。熟悉这两个功能，是我们快速上手剪映APP、为后续深入学习打下良好基础的前提。
剪映APP是一款简洁易用的剪辑工具，非常适合新手和短视频创作者使用。
