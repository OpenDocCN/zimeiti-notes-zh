在本节课中，我们将学习如何深入分析用户，从四个维度精准定位你的目标受众。这是创作出具有强共鸣感和高传播力内容的关键一步。

## 为什么要研究目标受众？

从用户视角来看，我们在刷短视频时，常会遇到一些感觉“完全就是为我定制”的内容。这类视频能引发大量“你在我身上装了摄像头吗”的评论，因为它们具有很强的沟通感和对话感，仿佛在直接与观众对话。这种共情能力，往往源于对特定目标人群生活的细致观察。

从创作者角度来看，如果不清楚用户是谁，内容创作就容易思维枯竭。缺乏用户思维的内容，往往停留在“作品思维”层面，只想着输出自己想说的。这样产出的作品可能像生硬的广告，缺乏沟通感和代入感，或者过于抽象，难以引发共情，数据自然不会理想。

有人可能会问，目标受众太具体，会不会导致受众面变窄？恰恰相反，指向性强的沟通会让你更加聚焦，能量更有穿透力，反而能吸引更多喜欢你这种鲜明特点的观看者。

从平台算法的角度来看，在去中心化的算法推荐机制中，每个用户都会接收到与自己观看标签最相关的内容。从市场传播的角度来看，任何具有商业目标的品牌，都需要通过内容与用户沟通以实现影响力最大化。这背后离不开市场调研、消费者洞察、数据分析等前期工作。许多广告公司和品牌企业都有专门部门和预算用于用户研究。由此可见，了解用户至关重要。

## 如何收集目标受众信息？

以下是两种收集目标受众信息的主要方法。

![](img/f884235ae82e573b0def860252964015_4.png)

### 方法一：参考对标账号

在星图、抖大大、飞瓜等短视频数据平台，搜索你想要参考的对标账号。你可以查看他们粉丝的人群构成，例如：
*   **性别**比例
*   **年龄**分布
*   **地区**分布
*   **手机型号**比例

这些数据在平台中通常都能直接看到。

另一个直接且有效的方法是，亲自查看对标账号视频下方的评论区，尤其是排名靠前的活跃用户。
*   观察他们的留言观点和态度，可以大致判断其身份、年龄和性别。
*   点击进入他们的个人主页，查看他们点赞或喜欢了哪些其他视频。

![](img/f884235ae82e573b0def860252964015_6.png)

这个方法看似简单甚至有些“笨拙”，却是了解用户视频观看习惯最直接的方式。建议你花些时间做这个功课，收获会很大。参考对标账号能让你对自己要做的内容方向有更具体的参照。

### 方法二：基于业务目标主动调研

你可以利用金数据、问卷星等在线调研工具，设计相关问题，主动了解目标受众的态度和生活形态。或者，直接采访现实中的参考对象——那些最符合你定位的目标受众。面对面的采访不仅能获取信息，他们或许还能为你提供更多创作灵感。

## 案例分析：不同账号的用户画像

为了让你有更直观的认识，我们一起来看看数据平台上几个头部账号的用户画像（以几个基本因素为例）。

**搞笑类账号前三名分析：**

1.  **毛毛姐**：女性用户占比超过75%，18-24岁用户最多。地区上，广东、贵州、四川、浙江、江苏排名前五，有一定南方地域偏好。这与他以“闺蜜”视角沟通、内容多反映女性生活洞察的风格高度吻合。
2.  **祝晓晗**：男性用户占比73.48%，25-30岁用户最多。地区排名靠前的有广东、江苏、山东、河南、浙江，南北省份均有。其内容也以男性视角的沟通为主，与“老丈人说车”一脉相承。
3.  **陈翔六点半**：男性用户占多数（67.12%），18-30岁用户占比高。广东、江苏、浙江、河南、四川是前五省份。其多年坚持的剧情式内容，覆盖的用户圈层范围很广。

**种草类账号前三名分析：**

1.  **兔兔的臭猪猪**（情侣Vlog种草）：女性用户居多，18-24岁用户占一半。

## 用户分析的四个核心维度

看完其他账号的分析，相信你对用户分析已经有了清晰的认识。当我们自己着手操作时，可以从以下四个维度深入了解用户。

![](img/f884235ae82e573b0def860252964015_10.png)

### 1. 人群构成
你的内容是给谁看的？从你的定位标签出发，人群定位的不同，直接决定了沟通的语言、场景和人物对话方式。公式可以概括为：
**目标受众 = 特定人群（如：初入职场的女性）**

![](img/f884235ae82e573b0def860252964015_11.png)

### 2. 生活状态
这些人生活状态是什么样的？可以从“外貌形态、时间、地点、人物、事件、物品”等角度切入。通过这一点，你可以找到具体的选题方向，建立自己的选题库。

### 3. 媒体习惯
他们平时通过什么媒体渠道获取信息？例如，有人喜欢看电影、刷小红书和豆瓣，有人喜欢在知乎发表见解，有人则只看抖音和朋友圈。不同的媒体习惯也反映了不同的生活态度和形态。

### 4. 心理洞察
他们的心理需求是什么？可以参考马斯洛需求层次理论，从**生理需求、安全需求、社交需求、尊重需求到自我实现需求**，针对特定人群将需求细化。

## 如何应用：建立选题库与设计沟通方式

![](img/f884235ae82e573b0def860252964015_13.png)

![](img/f884235ae82e573b0def860252964015_14.png)

将以上四个维度收集的信息细化，并与你的业务场景结合，你会发现无需苦思冥想，就能列出很多选题方向。

我们可以把信息整理成表格，建立一个系统的选题库，并设计出与用户的沟通方式。

以“职场定位账号”为例，我们可以基于受众定位的各个维度去细化方向：

*   **人群构成细化**：初入职场的女性，22-28岁，居住在一线或二线城市。
*   **生活状态细化**：
    *   **时间**：面试、入职、上班、回家。
    *   **地点**：办公室、工位、会议室。
    *   **人物**：老板、同事。
    *   **事件**：年会、团建、吵架。
    *   **物品**：电脑、手机。
*   **媒体习惯细化**：喜欢看情感类和搞笑类内容，喜欢追星，常上微博和刷抖音。
*   **心理洞察细化**：
    *   **自我实现**：有职业理想，追求个人成长。
    *   **尊重需求**：希望获得领导认可、同事夸赞。
    *   **社交需求**：喜欢结交办公室朋友，参加兴趣小组。
    *   **安全需求**：追求内心安全感。
    *   **生理需求**：关注身心健康。

通过这样细化，一个鲜活的目标用户形象就跃然纸上。你还能继续向下挖掘，得出更具体的内容话题，例如：
*   办公室女生的友谊
*   入职1年、3年、5年的区别
*   在大城市上班和过年回家的“我”
*   面试中的尴尬瞬间

这只是简单举例，是不是一下子就有画面感了？想内容也没那么难了。剩下的就交给你来完成，按照这个表格的维度丰富细化你的目标受众，看看能得出多少选题方向。

除了内容创作，另一个重要应用是**设计视频中称呼用户的方式**，这能让你的视频更有指向性和对话感。简单的例子：
*   如果你是**女闺蜜**人设，面向**女生**，可以称呼“姐妹们”。
*   如果你是**大哥**人设，面向**男生**，可以称呼“老铁”或“兄弟”。
你还可以设计更个性化的称呼，例如用自己名字结合粉丝特点，创造一个专属昵称。
