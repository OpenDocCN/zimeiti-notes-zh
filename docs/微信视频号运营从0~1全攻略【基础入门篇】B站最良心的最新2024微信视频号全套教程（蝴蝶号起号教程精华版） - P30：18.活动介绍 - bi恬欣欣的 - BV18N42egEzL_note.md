# 微信视频号运营从0~1全攻略：P30：18.活动介绍 🎬

在本节课中，我们将详细介绍一个由官方发起的视频号商家扶持活动。你将了解到活动的背景、核心负责人、以及参与活动能获得的巨大支持，包括官方提供的视频素材、流量扶持和现金奖励。最后，我们会简要说明参与活动的实操步骤。

## 活动背景与理念

为促进2024年中国行业的新发展，并为视频化带货电商奠定技术基础，特推出商家联盟活动。本次活动由短视频教研部负责人张良全程对接。本次活动以公平公正、同行共赢为理念，特色展开。

## 活动负责人与独家权益

张老师作为本次活动的全国总负责人。你只有在张老师这里才能对接本次活动，其他地方对接不到。这就是张老师的实力和背景。

有同学担心发了视频卖不出去。只要你参加活动，每个账号给到你90万的流量扶持。你无需再担心流量问题。
