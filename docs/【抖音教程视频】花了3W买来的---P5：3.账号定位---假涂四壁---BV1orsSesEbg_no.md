# 抖音运营教程：P5：3.账号定位 🎯

![](img/29659877c7af3d5660d063f80689e16d_0.png)

在本节课中，我们将要学习抖音账号定位的核心方法。账号定位是运营的基石，它决定了内容方向、吸引的粉丝群体以及最终的商业价值。一个清晰的定位能让你的账号在众多内容中脱颖而出。

![](img/29659877c7af3d5660d063f80689e16d_2.png)

## 你是谁？—— 确定账号类型与主体

上一节我们介绍了赛道选择，本节中我们来看看如何进一步细化账号类型并明确账号主体。

首先，我们要确定接下来要做的账号类型。除了之前章节提到的大类赛道（即抖音定义的一级垂类），我们还需思考在一级垂类下细分的具体赛道是什么。

**公式：账号垂直类型 = 一级垂类 + 二级垂类**

例如，你选择“运动健身”作为一级垂类，而你最擅长且具有优势的是“瑜伽教学”，那么“瑜伽教学”就是你的二级垂类。通过定位细分类目，可以确定账号的垂直类型。

其次，我们需要确定账号主体是谁。账号主体是账号的拥有人，也是最终呈现给粉丝的主人公。在抖音商业类账号中，主体主要分为两类：

以下是两种主要的账号主体类型：

*   **商家账号**：主体是公司或品牌。内容需偏向商业化，并应通过内容拉近与粉丝的距离，避免过于严肃。这既能建立信任感，也能带来好感，符合抖音的娱乐属性。
*   **达人账号**：主体是个人。首要任务是解决信任问题，可以通过IP打造的方式，为账号树立一个积极、正面的人设。

## 你是做什么的？—— 明确内容与身份

明确了“你是谁”之后，接下来需要清晰地告诉粉丝“你是做什么的”。

最直接的方式是通过抖音的官方认证来表明身份。认证后，粉丝可以清晰地看到你的认证标签，从而了解你的职业和账号内容方向。

以下是两种主要的认证方式：

*   **黄V认证**：适用于个人达人，是个人专业能力的背书。
*   **蓝V认证**：适用于企业或品牌，是官方身份的背书。

在未进行官方认证之前，也可以通过账号的基础设置向粉丝传达你的业务信息。

以下是可以通过设置传达信息的几个关键部分：

*   **头像**
*   **背景图**
*   **个人简介**
*   **视频封面**

对这些部分进行与业务关联的统一设计，可以有效传达账号定位。

## 为什么要关注你？—— 提供核心价值

最后，也是至关重要的一步，是思考你将为粉丝提供什么样的独特价值。这是用户关注你的根本理由。

在抖音中，粉丝对博主的价值需求主要可以分为以下几类。以下是五种核心价值类型：

1.  **知识价值**：为用户提供认知提升、干货技能或常识普及。
    *   **认知型**：如人生思考、行业经验分享。
    *   **干货型**：如具体技能教学、步骤总结。
    *   **常识型**：如生活、健康、科学常识解读。
    *   **核心价值**：**认同感**（观点认同、减少知识盲区的获得感）。

2.  **情绪价值**：通过内容引发观众的情感共鸣或波动。
    *   **常见类型**：开心愉悦、关系共鸣（如情侣、家庭）、情绪波动（如争议、反转）、怀旧、好奇等。
    *   **核心价值**：**共鸣感**（情绪表达准确到位，直接感染观众）。

3.  **感觉价值**：通过视频的视听元素为用户带来感官享受或仰慕感。
    *   **传递方式**：画面美感、构图、声音节奏、个人魅力等。
    *   **常见领域**：颜值、萌宠、美景、才艺、文化等。
    *   **核心价值**：**共情力**（引发联想，在意识层面与作者产生连接）。

4.  **体验价值**：让用户看到、感受到未曾经历的生活方式或事物。
    *   **常见内容**：美食享用/制作、旅行见闻、运动挑战、角色体验等。
    *   **核心价值**：**满足好奇心与获得感**（体验新奇、紧张、感动等情绪）。

5.  **期待价值**：通过内容设计让用户对下一次更新产生强烈期待。
    *   **实现方法**：内容留有悬念、系列化更新、塑造独特的个人魅力。
    *   **核心价值**：**引导关注**（让用户产生“意犹未尽，期待下次”的感觉）。

---

本节课中我们一起学习了抖音账号定位的三个核心问题：“你是谁”、“你是做什么的”以及“为什么要关注你”。通过确定账号的垂直类型与主体、明确内容方向与身份、并聚焦于为粉丝提供至少一种核心价值（知识、情绪、感觉、体验、期待），你可以构建一个清晰、有吸引力且可持续运营的账号基础。清晰的定位是后续内容创作、粉丝增长和商业变现的导航图。
