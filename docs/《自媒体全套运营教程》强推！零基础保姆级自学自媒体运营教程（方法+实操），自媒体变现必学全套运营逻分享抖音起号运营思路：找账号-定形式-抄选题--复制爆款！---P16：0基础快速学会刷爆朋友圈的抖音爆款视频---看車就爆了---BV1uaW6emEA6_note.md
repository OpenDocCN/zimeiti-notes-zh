### 第二小节：剪映APP界面结构拆解

![](img/34aa90cef915718dbe1e529280053584_11.png)

认识了剪映APP的基本概念后，我们需要深入了解其操作界面。学习剪辑，必须了解APP的整体结构、模块划分及各模块功能。

![](img/34aa90cef915718dbe1e529280053584_7.png)

总的来说，剪映APP的界面结构比较简单，一共划分为四个部分。

以下是四个核心模块的详细介绍：

1.  **剪辑**：这是所有视频剪辑操作的核心区域，也是我们整个课程的重点。
    
    ![](img/34aa90cef915718dbe1e529280053584_9.png)

2.  **剪同款**：这个功能为新手提供了极大便利。剪映将市面上流行的爆款视频做成了视频模板。用户只需填充自己的视频素材，即可一键快速生成同款爆款视频。其操作逻辑可简化为：**选择模板 -> 替换素材 -> 导出视频**。
    
    ![](img/34aa90cef915718dbe1e529280053584_11.png)

3.  **消息**：剪映是抖音官方推出的剪辑工具，与抖音深度关联。登录并授权抖音账号后，可以在此查看抖音的官方消息、收到的点赞、粉丝留言等。
    
    ![](img/34aa90cef915718dbe1e529280053584_13.png)

![](img/34aa90cef915718dbe1e529280053584_13.png)

4.  **我的**：展示个人信息账号。将抖音号与剪映号关联后，抖音账号的相关信息也会在此展示。
    
    ![](img/34aa90cef915718dbe1e529280053584_15.png)
