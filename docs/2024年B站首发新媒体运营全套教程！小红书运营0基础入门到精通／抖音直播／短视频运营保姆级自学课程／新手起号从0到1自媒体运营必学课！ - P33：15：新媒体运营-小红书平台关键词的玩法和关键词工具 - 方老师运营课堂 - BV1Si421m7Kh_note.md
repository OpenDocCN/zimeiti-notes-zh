## 核心概念：核心关键词与长尾关键词

![](img/2e876181978b08910227215569c01655_9.png)

上一节我们介绍了内容策划的基础，本节中我们来看看关键词的分类。理解核心关键词与长尾关键词的区别是进行有效关键词布局的第一步。

![](img/2e876181978b08910227215569c01655_11.png)

**核心关键词** 是指能概括一个行业或产品大类的通用词汇。例如：
*   `女装`
*   `奶粉`
*   `压缩机`

![](img/2e876181978b08910227215569c01655_13.png)

这些词搜索量大，但竞争通常也非常激烈。

**长尾关键词** 则是围绕核心关键词拓展出来的、更具体、更长的短语。其公式可以表示为：
`长尾关键词 = 核心关键词 + 修饰词/需求点`
例如，围绕核心词“女装”可以拓展出：
*   `女装图片`
*   `女装店推荐`
*   `女装穿搭技巧`

长尾关键词虽然单个搜索量较小，但更精准，能覆盖用户更具体、更小众的需求，转化率往往更高。

## 关键词拓展逻辑：从核心词到内容选题

![](img/2e876181978b08910227215569c01655_32.png)

![](img/2e876181978b08910227215569c01655_34.png)

通过工具找到核心词和相关需求点后，我们如何系统地拓展出具体的内容选题？以下是关键词拓展的完整逻辑链条。

关键词拓展是一个层层递进的过程：
1.  **确定核心词**：例如 `眉笔`。
2.  **发现一级长尾词**：通过工具发现用户关心 `颜色`、`色号`、`防水` 等。
3.  **拓展二级长尾词**：针对“颜色”进一步拓展，得到 `眉笔颜色怎么选`、`眉笔什么颜色好看` 等具体问题。
4.  **形成内容选题**：这些具体的“问题”或“短语”就是绝佳的内容选题。例如，你可以创作一篇题为“新手必看：眉笔颜色怎么选？”的笔记。

通过这个逻辑，你可以为一个核心产品衍生出数十甚至上百个具体的内容方向，有效解决“不知道发什么”的难题。

## 小红书关键词实战玩法：堆砌与识别

![](img/2e876181978b08910227215569c01655_38.png)

![](img/2e876181978b08910227215569c01655_40.png)

![](img/2e876181978b08910227215569c01655_42.png)

掌握了关键词的寻找和拓展方法后，我们来看一个在小红书平台上具体的、带有技巧性的实战玩法。

在小红书，系统（Spider）通过识别笔记中的关键词来理解和推荐内容。因此，有些人会使用“关键词堆砌”的技巧来强化系统对笔记主题的认知。

![](img/2e876181978b08910227215569c01655_44.png)

**具体操作**：
在笔记正文结束后、标签（#）之前，重复添加与核心主题相关的关键词。例如，一篇关于“瑞幸咖啡”的笔记，可能会在文末添加：
```
瑞幸拿铁 瑞幸咖啡 瑞幸点单攻略 瑞幸隐藏菜单 瑞幸新品 瑞幸 瑞幸 瑞幸...
```
**原理**：
这增加了核心关键词（如“瑞幸”）在全文中的“词频”。当小红书系统的Spider抓取和分析笔记时，会判断该笔记与“瑞幸”的相关度极高，从而可能在用户搜索相关词时获得更好的排名。

![](img/2e876181978b08910227215569c01655_46.png)

**高级历史玩法（仅供参考）**：
在早期网站SEO中，甚至有通过将关键词字体颜色设置为与背景色相同（如白色），使其对用户不可见但能被Spider抓取的方法。不过，小红书目前不支持自定义字体颜色，且过度堆砌关键词可能影响用户体验，需谨慎使用。

**核心要点**：
你的标题、正文和标签中的关键词，主要是给平台的 **Spider（网络爬虫）** 识别的，用于帮助系统理解内容并匹配给搜索用户。
