# 短视频起号·差异化定位课：38：发作品前的准备（1） 📝

在本节课中，我们将学习在抖音发布第一条作品前需要完成的两项核心准备工作。理解并执行这些步骤，有助于平台更快地识别你的账号属性，为后续内容获得精准推荐打下基础。

上一节我们介绍了账号定位与内容规划的整体框架，本节中我们来看看具体执行的第一步——发布前的准备工作。

---

## 准备工作一：养号 🧑‍💻

我们可以将发布作品类比为入职一家新公司。你的账号对于抖音平台而言是全新的，就像新员工对于公司一样。养号的目的，就是让系统（如同公司的同事）更快地“认识”并“了解”你。

养号的核心逻辑是：**系统为账号打标签的过程，类似于新员工被同事认知的过程**。系统需要判断你的账号内容属性和兴趣标签，这就像同事们在猜测新员工的性格、喜好与工作能力。

养号主要有两个作用：
1.  **提升账号活跃度**：向平台证明这是一个真实、活跃的用户账号。
2.  **初见标签方向**：通过你的互动行为，初步向系统暗示你未来内容的赛道方向。

这个过程类似于入职前与HR沟通、提前加入工作群，是必要的铺垫，但并非起号的绝对核心。然而，它仍然是一个必须完成的步骤。

以下是养号的具体操作方法：
*   **时间**：在新账号注册后，发布第一条作品前的 **3到5天**。
*   **行为**：模仿真实用户的习惯使用抖音。
*   **时长**：每天至少使用 **30分钟到1小时**。
*   **关键操作**：重点与你计划切入的赛道内容（例如美食、亲子）进行互动。看到相关优质作品，尽量完整观看（高完播率），并积极点赞、评论、关注。这能帮助系统初步界定你的兴趣方向。

---

## 准备工作二：提前储备选题 📚

在正式“上岗”（发布作品）前，你需要像准备入职材料一样，提前准备好初期的内容“弹药”。这能保证你在起号阶段保持稳定的内容输出，不会因临时找不到选题而中断。

![](img/4eefc34d1dbea878b4510cb643c26862_1.png)

通常，我们会建议提前筛选并确定 **7到14天** 的选题内容量。关于选题方向，之前课程已介绍过三个切入点：**热点**、**用户需求点** 和 **争议点**。同时，务必多研究对标账号内容下的评论区，从中寻找灵感。

今天，我们新增一个更简单、客观的选题方法，这得益于抖音“热点宝”小程序的更新。它能提供“吸粉量”这一关键数据，帮助我们更精准地判断哪些内容能有效吸引目标粉丝。

以下是利用“热点宝”寻找选题的步骤：
1.  打开抖音“热点宝”小程序。（具体使用方法可参考讲师主页相关视频）
2.  确定你的内容赛道（例如：亲子）。
3.  在热点宝内，可以查看对标账号每条作品在特定周期（如30天）内的 **吸粉量**。
4.  按“吸粉量”排序，找到那些吸粉效果最好的作品。
5.  分析这些高吸粉作品的选题、文案结构和内容切入点。例如，一条题为“亲子关系出问题，孩子能力难提升”的作品吸粉5000+，这证明该话题是强烈的需求点或争议点，值得参考和再创作。

本节课中我们一起学习了发布作品前的两项关键准备：**养号**与**储备选题**。养号是为了让系统初步认识你的账号；储备选题则是为确保起号阶段内容不断更。做好这两点，就如同新员工做好了入职预热和工作计划，为接下来的“正式工作”——内容发布与运营，奠定了扎实的基础。
