# 抖音运营教程：P9：文案素材与视频制作 🎬

![](img/8b2e7c23fbc267866fbef60a03184731_1.png)

![](img/8b2e7c23fbc267866fbef60a03184731_3.png)

在本节课中，我们将学习如何为视频创意寻找文案素材，并了解一种将文字素材转化为视频的创作方法。课程将介绍一个实用的工具，并讲解从文案到视频的制作思路。

![](img/8b2e7c23fbc267866fbef60a03184731_5.png)

![](img/8b2e7c23fbc267866fbef60a03184731_7.png)

上一节我们介绍了视频创意的确定方法，本节中我们来看看如何为已确定的创意寻找合适的文案内容。

![](img/8b2e7c23fbc267866fbef60a03184731_9.png)

![](img/8b2e7c23fbc267866fbef60a03184731_11.png)

视频创意确定后，需要搜索相应的文案。直接搬运他人视频素材并进行处理是一种方式。若希望根据特定创意搜索文案，并以另一种形式自主创作视频，则需要寻找文字素材。

![](img/8b2e7c23fbc267866fbef60a03184731_13.png)

![](img/8b2e7c23fbc267866fbef60a03184731_15.png)

文案素材方面，推荐使用一个名为“淘金阁”的工具网站。

![](img/8b2e7c23fbc267866fbef60a03184731_17.png)

![](img/8b2e7c23fbc267866fbef60a03184731_19.png)

以下是“淘金阁”网站的基本介绍与使用方法：

![](img/8b2e7c23fbc267866fbef60a03184731_21.png)

![](img/8b2e7c23fbc267866fbef60a03184731_23.png)

*   **网站性质**：淘金阁是一个自媒体全网搜索引擎。
*   **素材来源**：它汇集了多个内容平台的图文素材，包括今日头条、百家号、企鹅号、一点资讯、QQ看点、凤凰网、网易资讯、搜狐网、东方网、简书、快资讯等。其内部拥有庞大的文案素材库。
*   **使用方法**：
    1.  在搜索框中输入目标关键词（例如“星座”）。
    2.  按下回车键进行搜索。
    3.  点击“开始抓取”按钮。
    4.  页面下方会展示抓取到的所有相关文章列表。

![](img/8b2e7c23fbc267866fbef60a03184731_25.png)

![](img/8b2e7c23fbc267866fbef60a03184731_27.png)

![](img/8b2e7c23fbc267866fbef60a03184731_29.png)

请注意，这里获取的是**文章素材**，而非视频素材。将别人的文章转化为自己的视频内容，与直接搬运他人视频有本质区别，不属于搬运范畴。

![](img/8b2e7c23fbc267866fbef60a03184731_31.png)

![](img/8b2e7c23fbc267866fbef60a03184731_33.png)

例如，搜索“星座”后，可以从结果中选择一篇文章，如《分手后也能挽回最放不下前任的三大星座》。点击进入即可查看完整的文章内容，其中关于巨蟹座、双鱼座等的描述就是可用的文案。这些文案可以直接通过复制粘贴的方式提取，供后续创作使用。

![](img/8b2e7c23fbc267866fbef60a03184731_35.png)

![](img/8b2e7c23fbc267866fbef60a03184731_37.png)

![](img/8b2e7c23fbc267866fbef60a03184731_39.png)

找到文案后，接下来需要解决视频制作的问题。下面介绍一种视频制作形式。

视频制作可以采用真人拍摄、音乐影视混剪等多种形式。音乐影视混剪可以利用特定网站获取素材，或采用录屏方式。之前课程分享过的台词混剪网站、以及如“mvcat”这类可下载电影的网站，都可以作为影视混剪的素材来源。

![](img/8b2e7c23fbc267866fbef60a03184731_41.png)

本节课中我们一起学习了如何利用“淘金阁”工具搜索文案素材，并理解了将图文素材转化为视频内容的创作逻辑。关键在于为既定创意找到文字基础，并通过不同的视频形式（如混剪）将其呈现出来，这是一种有效的原创视频制作思路。
