## 渠道一：Microsoft Edge浏览器与Copilot 💻

首先，我们推荐使用微软Edge浏览器内置的Copilot功能。这是微软基于ChatGPT 4.0模型打造的AI助手，集成在Windows系统中，为普通用户提供了便捷的使用方式。

### 使用前提
使用Copilot有两个前提条件：
1.  登录微软账号（可使用Outlook邮箱注册）。
2.  具备网络访问工具（通常称为“梯子”），以访问相关服务。

### 功能与界面
Copilot提供了三种对话模式：
*   **更有创造力**：AI的回答更具开放性和创意。
*   **平衡**：介于精确和创造力之间的模式，适合一般性任务。
*   **更精确**：提供更严谨、准确的回答。

例如，我们可以使用“更有创造力”模式，让它生成一个短视频脚本。输入指令：
```
请帮我写一个关于装修的短视频脚本，时长控制在3分钟左右。
```
AI便会开始生成包含场景、镜头、台词等元素的完整脚本。

### 优势与局限
**优势**：
*   集成在浏览器侧边栏，可基于当前打开的网页内容进行分析。
*   可执行一些简单的系统命令（如设置暗黑模式、打开记事本）。

**局限**：
1.  **精准度稍逊**：相较于官方ChatGPT 4.0，Copilot在回答的精准度上略有差距。
2.  **连续对话能力受限**：在当前对话中持续提问的次数有限，达到限制后需开启新对话。
3.  **记忆功能有所缺失**：在读取和分析历史对话记录方面能力较弱，影响回答的连贯性和深度优化。

对于大多数日常运营工作，Copilot的功能已足够使用。如果你的Edge浏览器没有Copilot图标，可以尝试下载最新版的Edge Dev（开发者）或Beta（测试）版本。

以下是几个可用的第三方平台介绍：

*   **平台A（灵感）**：
    *   **特点**：集成了多种AI模型，包括GPT-3.5、GPT-4.0（含联网版）、国内主流大模型（如讯飞星火、文心一言等）。

## 渠道四：国内AI大模型 🇨🇳

如果你没有网络访问条件，可以考虑使用国内公司开发的AI大模型，它们无需特殊工具即可直接访问。

以下是几个主流的国内AI大模型：

![](img/4cdeb4b2fcca34e33508c3fa62e608f0_19.png)

![](img/4cdeb4b2fcca34e33508c3fa62e608f0_20.png)

*   **文心一言（百度）**：功能全面，能满足基本的文案创作、脚本生成等需求。提供付费的“4.0”版本，能力有所提升。
*   **豆包（字节跳动）**：同样可以生成结构清晰的脚本，例如包含景别、画面内容、台词、时长的分镜脚本。
*   **通义千问（阿里巴巴）**、**天工大模型**等也是可用的选择。

这些国内模型在基础文案创作、提供灵感方面完全够用。你可以选择一款用起来最顺手、最能提升你工作效率的进行深度使用。

为了对比，我们最后看一下官方ChatGPT 4.0生成装修脚本的示例，其回答在场景构建、语言感染力和创意方面通常更胜一筹：
```
请帮我写一份关于装修的短视频脚本文案，要求镜头丰富，语言吸引人。
```
生成的脚本可能会以“变形记：从旧居到梦想家园”为主题，运用“缓慢推进镜头展示破旧居室”与“快速剪辑展示改造过程”形成鲜明对比，富有电影感和故事性。

## 总结

本节课中，我们一起学习了体验和使用ChatGPT 4.0的四种主要渠道：
1.  **Microsoft Edge Copilot**：系统集成，方便，但能力有阉割。
2.  **第三方集成平台**：选择多，可能有免费额度，但速度和稳定性各异。
3.  **共享账号平台**：可能免费使用GPT-4.0，但极不稳定，风险高。
4.  **国内AI大模型**：无需特殊访问，易用，但在复杂任务和创意深度上可能与顶尖模型存在差距。

![](img/4cdeb4b2fcca34e33508c3fa62e608f0_22.png)

对于新媒体运营而言，本着“提升效率”的原则，建议在条件允许的情况下，优先尝试使用能力更强的GPT-4.0模型。后续课程我们将继续深入探讨AI与新媒体运营结合的具体技巧和方法。
