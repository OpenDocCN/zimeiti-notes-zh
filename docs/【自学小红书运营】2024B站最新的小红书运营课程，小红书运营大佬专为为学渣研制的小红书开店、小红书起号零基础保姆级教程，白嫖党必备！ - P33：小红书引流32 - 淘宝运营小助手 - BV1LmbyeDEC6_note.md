## 自我分析与账号规划 📝

上一节我们介绍了小红书运营的整体框架，本节中我们来看看如何从自身出发进行规划。自我分析是账号规划的起点，你需要审视个人信息是否能满足博主需求。

以下是个人信息组织的关键要素：
*   **身高、体重、身材**
*   **爱好**
*   **职业方向**
*   **利用情况**

![](img/dbed1326d590e9579a4ff75496aa2d02_5.png)

这些信息是给自己看的，用于判断是否适合做博主。不满足要求的信息可以屏蔽，满足要求的则可以放入个人简介中。这类似于交朋友，先介绍自己的基本情况，以吸引同类型的人群。例如，美妆、化妆或美食博主需要完美地介绍自己。个人情况越真实，获得的认可度就越高。

## 个人账号内容定位 🎬

同时，自我分析也是给自己做定位。个人账号的内容规划涉及IP定位、粉丝定位、内容关键词和选题。

如果你准备做网络博主，需要选择一个核心题目或宣传方向。在一个社区里选择一个类目，做完后再向其他地方扩展。不能选择一个社区后，又想参与所有类目，那样无法成功。

因此，个人账号内容首先要为自己设定一个人设。

以下是设定人设的举例：
*   **一线城市90后或95后独居女性**
*   **每日生活分享**
*   **职场日常**
*   **下班后生活**

在小红书上，虽然可以自由发布内容，但需要根据个人情况了解自己的IP定位。设定好人设和每日分享内容后，需要确定粉丝定位。你需要把自己打造成目标粉丝的同类型人群，成为他们的“领头羊”。通过这种模式，为他们介绍吃喝玩乐或其他内容都会非常方便。这完全取决于你的人设如何设定。

## 网络博主类型选择 🚀

![](img/dbed1326d590e9579a4ff75496aa2d02_21.png)

最后，我们来探讨网络博主的具体类型。既然准备在小红书上做网络博主，就需要选择内容呈现方式：是做直播博主还是文案博主。

### 直播博主 📺

如果你选择做直播博主，那么会涉及以下四个要点。

以下是直播博主需关注的四个核心点：
1.  **直播内容**
2.  **直播时间**
3.  **内容准备**
4.  **直播场景**

直播内容涵盖各行各业，只要设定好目标，基本上都能找到输出方向。

以下是直播内容的可选方向举例：
*   **教育科普**
*   **生活分享**
*   **时尚穿搭**
*   **访谈**
*   **电商**
*   **心理咨询**
*   **文化**

做博主就是不限制，放大自己所擅长的范围，去吸引对应的人群。博主的核心是讲述自己熟悉的专业领域，吸引不懂或了解不深的人，一起交流，教他们如何提高生活品质。否则就只能算作引流。

![](img/dbed1326d590e9579a4ff75496aa2d02_23.png)

直播时间可以个人自行安排。小红书没有固定的直播时间，基本上可以随时开播。但若想成为优秀的主播，需要将每天直播视为固定工作时间。不能今天早上播，下午播，明天后天不播，大后天再播。需要有固定的直播时长和频率，具体取决于个人想如何转化。

内容准备方面，前期做直播博主，需要准备好最少两小时、三个内容的分享。否则直播间前期可能没有人。如果没有足够的影响力和吸引力，即使有粉丝，也需要在粉丝数起来后再开播，效果会更好，也不那么尴尬，无需准备过多内容。如果一开始就想通过直播宣传自己，则需要准备两到三套直播话术，直播时间约一到两小时，才能慢慢积累人气。否则，时间不够，内容重复，或直播间无人互动，别人甚至不会点进来。

直播场景涉及背景、前景和环境的三位一体选择。具体设置可根据个人条件进行。

### 文案博主 ✍️

文案博主相比直播博主简单得多。直播博主需要个人在某方面有很好的素质，而文案博主如果对某些内容不熟，可以直接查资料，然后以文字形式做成文案笔记发布。

这里主要需注意笔记输出内容和文案排版。

![](img/dbed1326d590e9579a4ff75496aa2d02_25.png)

笔记输出内容应选择**4/5的实体输出，留1/5的猜测内容**。不要把答案全部写在笔记里，留出部分猜测内容，可以用小号引导评论，提升账号活跃度。活跃度高，观看的人才会多。

![](img/dbed1326d590e9579a4ff75496aa2d02_27.png)

文案排版方面，主图选择**3~5张**图片进行排版。猜测内容放在第一位吸引用户，第二位的内容则是对第一位的解说。因为笔记封面和关键词是最重要的。通过这种方式吸引用户后，在第二、三、四页进行内容延伸，第五页则预告下一个笔记的主题内容。通过这种循环，吸引更多用户关注你，了解你下一步的更新动态。用户甚至可能会在评论区催更，类似于连载小说。具体如何排版，看个人安排。

笔记发布时间与商品笔记类似，根据自己选择的内容自行发布。

![](img/dbed1326d590e9579a4ff75496aa2d02_29.png)

### 读书种草类博主 📚

![](img/dbed1326d590e9579a4ff75496aa2d02_31.png)

最后介绍读书种草类博主。这类博主是纯分享型，例如分享一本书、晒书种草以及读书感想。这是在小红书上起号最简单的方式之一，但变现难度较大，因为本身可能没有过硬的技术。这种方式比较适合新手和小白操作。如果想往这个方向发展，可以深入研究。
