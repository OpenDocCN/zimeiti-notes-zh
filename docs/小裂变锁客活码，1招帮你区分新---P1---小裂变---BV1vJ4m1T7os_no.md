# 微信营销：01：使用小裂变锁客活码区分新老客户

在本节课中，我们将学习如何使用小裂变的“锁客活码”功能，解决线下拓客活动中礼品被老客户重复领取的问题，并实现对新客户的精准引流与对老客户的二次营销。

## 概述

线下门店常通过发放礼品来吸引新客户。但若不加以区分，宝贵的促销资源可能被老客户重复领取，导致拓客成本上升且效果不佳。小裂变的锁客活码功能能智能识别扫码客户的身份，并根据其新老状态采取不同策略，从而实现精准营销。

## 核心功能与操作步骤

锁客活码的核心逻辑是一个条件判断流程，其工作方式可以用以下伪代码表示：

```
if (客户扫码) {
    if (客户是首次扫码) {
        展示领奖页面; // 识别为新客户
    } else {
        展示优惠券页面; // 识别为老客户
    }
}
```

以下是实现这一功能的具体操作步骤。

首先，您需要访问小裂变平台。在后台找到并创建“锁客活码”活动。

接下来，进行关键的活动规则配置。您需要设置两个核心页面。

*   **新客户页面**：配置新客户扫码后看到的界面，通常用于引导其领取礼品或完成注册。
*   **老客户页面**：配置老客户扫码后看到的界面，可设置为赠送折扣券，引导其再次消费。

最后，完成设置后，将生成的活码下载并打印出来，放置于店门口或宣传物料上，供客户扫描。

## 策略优势与总结

通过以上步骤，我们建立了一个自动化的客户筛选与触达机制。当新客户扫码时，他们能顺利领取礼品，成为您的潜客。而当老客户扫码时，系统会智能地推送一张优惠券，既能避免礼品资源的浪费，又能有效刺激老客户复购，实现一举两得。

本节课中，我们一起学习了小裂变锁客活码的应用。它通过一个简单的扫码动作，智能区分新老客户，并执行不同的营销策略。这不仅保护了您的拓客预算，使其更精准地用于吸引新用户，同时也为维护老客户关系、提升复购率提供了有效工具。如需体验此功能，可联系小裂变官方客服获取免费试用。
