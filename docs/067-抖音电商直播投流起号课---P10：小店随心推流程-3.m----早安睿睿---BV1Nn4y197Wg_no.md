理解了优化目标后，下一个关键决策是选择“吸引观众类型”。这个选择决定了你的广告会展示给哪一类人群。每个优化目标（如进入、点击、下单等）都可以搭配三种观众类型：**系统智能推荐**、**自定义**和**达人相似粉丝**。这意味着，仅一个优化目标就能衍生出多条不同的计划。

### 1. 系统智能推荐 🤖

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_62.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_64.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_66.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_68.png)

系统智能推荐是平台根据你账号的历史数据和模型，自动为你寻找潜在兴趣用户。

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_70.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_72.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_74.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_76.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_78.png)

*   **核心逻辑**：系统会推送“精准且泛”的人群。例如，一个卖羽绒服的账号，系统会推荐给近期有观看女装、羽绒服等行为的用户，但年龄范围可能从18岁到80岁都有。
*   **适用情况**：
    *   **老号（有标签）**：强烈推荐使用。系统能高效地找到数量和质量都更优的人群，这也是千川“通投”逻辑的基础。
    *   **新号（无标签）**：**禁止使用**。因为系统没有数据参考，会推送杂乱无章的人群，不利于打标签。
*   **人群速度**：来人速度最快，能快速拉升直播间在线人数。

### 2. 自定义 🎯

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_90.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_92.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_94.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_96.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_98.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_100.png)

自定义允许你手动设置目标人群的年龄、性别、地域等基础属性。

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_102.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_104.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_106.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_108.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_110.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_112.png)

*   **核心逻辑**：进行精准的人工筛选，圈定你认为最符合产品客群特征的人群。
*   **适用情况**：
    *   **新号起号期**：**必须使用**。这是为直播间打上精准标签的核心手段。例如，卖25-35岁女性羽绒服，就精确设置“女性，25-30岁”。
    *   **老号追求极致精准**：当你的产品受众非常垂直（小众品类）时，即使老号也可以使用自定义来进一步收窄人群，提升精准度。
*   **人群速度**：来人速度比系统智能推荐慢，但人群更精准。

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_114.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_116.png)

### 3. 达人相似粉丝 👥

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_122.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_124.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_126.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_128.png)

将广告展示给指定达人的粉丝，或与这些粉丝兴趣相似的人群。

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_130.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_132.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_134.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_136.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_138.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_140.png)

*   **核心逻辑**：借力成熟达人的粉丝模型，找到具有相似购物兴趣的人群。
*   **适用策略**：
    *   **达人选择技巧**：
        *   避免选择粉丝量过大的头部达人，你的人货场可能竞争不过。
        *   优先选择近期起号、粉丝量适中、且与你在同一品类或关联品类（如卖羽绒服，可选卖羊绒衫、鞋靴的达人）的账号。
        *   关联品类（如美妆、配饰）的达人粉丝也可能有服装购买需求，可以尝试。
    *   **投放策略**：
        *   **新号**：可作为“备用计划”。将大部分预算（如500/800元）用于自定义计划打标签，同时用小部分预算（如200-300元）测试达人计划。能跑动则好，跑不动则预算会退回，风险可控。
        *   **老号**：模型稳定后，投放下单、成交等深层目标时效果很好，能进一步提升GMV。

### 1. 直接加热直播间 🔴

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_166.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_168.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_174.png)

*   **逻辑**：直接将你的直播间曝光给目标用户，用户点击封面即可进入。
*   **优点**：流量路径短，来人速度极快。
*   **适用场景**：**大众品类、成熟直播间**。依靠直播间内的人、货、场直接承接和转化流量。

### 2. 通过视频加热直播间 🎬

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_186.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_188.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_190.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_192.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_194.png)

*   **逻辑**：先将优质短视频推给目标用户，用户被视频内容吸引后，点击进入直播间。
*   **优点**：流量过滤了一层，通过视频内容吸引来的用户意向更强，人群更精准。
*   **适用场景**：**小众品类、高客单价商品、或需要极精准流量的阶段**。用视频内容筛选出高兴趣用户。

## 投放时长设置 ⏱️

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_206.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_208.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_214.png)

投放时长决定了预算的消耗速度，直接影响直播间的在线曲线。

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_216.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_218.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_220.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_222.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_224.png)

*   **短时长（如0.5小时）**：预算集中消耗，能在短时间内快速拉高在线人数，适合憋单、营造热氛围。
*   **长时长（如2小时或整场）**：预算均匀消耗，在线人数更平稳，适合平稳过款、日常销售。
*   **公式理解**：`计划预算 / 投放时长 = 平均每小时消耗速度`。你可以通过调整这个“速度”来控制流量进场的节奏。

本节课我们一起学习了小店随心推投放的最后一个拼图。我们来总结一下核心要点：

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_248.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_250.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_252.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_254.png)

![](img/3b0c912dede8c0b2ac4b22b37e58eeb6_256.png)

1.  **观众类型是计划的“方向盘”**：**系统智能推荐**求快，**自定义**求准，**达人相似粉丝**借力。新号以自定义为主，老号可大胆使用系统推荐。
2.  **加热方式是“过滤器”**：**直投直播间**要速度，**视频加热**要精度。根据产品属性和直播阶段灵活选择。
3.  **投放时长是“油门”**：控制预算消耗节奏，短时长拉高在线，长时长稳定流速。
4.  **计划没有统一模板**：理解了底层逻辑后，你可以自由组合。例如：
    *   老号拉流量：`进入直播间` + `系统智能推荐` + `直投` + `短时长`。
    *   新号打标签：`商品点击` + `自定义（精准年龄性别）` + `视频加热` + `合理时长`。
    *   老号促成交：`下单` + `系统智能推荐/达人粉丝` + `直投`。

现在，你已经掌握了从优化目标到观众类型，再到加热方式的完整决策链。关键在于理解每个选项背后的逻辑，并根据你账号的实际情况（新号/老号）和直播目标（拉流/打标/成交）进行灵活搭配。下一节，我们将进入实战，演示如何一步步搭建这些组合计划。
