# 剪映PC版教学：P4：【电脑版】字幕库 📚

![](img/45453ce7529daceb230f870ad87c96e4_0.png)

在本节课中，我们将要学习剪映PC版中“文本”功能模块的全面使用方法。我们将从最基础的自定义文本开始，逐步深入到花字、文字模板、智能字幕以及本地字幕的导入，帮助你掌握为视频添加和美化文字的各种技巧。

## 文本基础：自定义与样式调整 🎨

上一节我们介绍了视频剪辑的基本流程，本节中我们来看看如何为视频添加文字包装。文字是提升视频质感和信息传达的重要工具。

在剪映中，添加文字非常方便。首先，点击菜单栏的第三个选项“文本”。

打开“文本”后，你会看到多个子模块。第一个是“默认”文本，这相当于自定义文本功能。你可以在此输入任何文字。

**操作步骤**：
1.  点击“默认”文本右侧的“+”号，将其添加到时间线轨道上。
2.  文本会出现在播放器预览窗口中。
3.  双击预览窗口中的文字，即可进行编辑，例如输入“眼睛”。

添加文本后，选中轨道上的文本片段，右侧会出现详细的编辑面板。以下是可调整的主要参数：

*   **字体与字号**：在“字体”下拉菜单中可以选择系统内所有字体，通过滑块上下滚动浏览。在“字号”处可以调整文字大小。
*   **样式按钮**：`B` 代表加粗，`U` 代表下划线，`I` 代表倾斜。
*   **颜色**：点击颜色块，可以从预设色板中选择颜色，也可以使用取色器自定义颜色。
*   **字间距**：调整字符之间的水平距离。公式可理解为：`视觉紧凑度 = 基础间距 + 字间距调整值`。
*   **行间距**：当文字有多行时，调整行与行之间的垂直距离。
*   **对齐方式**：控制多行文字的对齐，如左对齐、居中对齐、右对齐。
*   **样式**：为文字添加预设的描边、背景、阴影等效果。
*   **位置与缩放**：可以手动拖动文字改变位置，也可以在面板中调整“缩放”数值，或使用“等比缩放”锁定宽高比。
*   **对齐（画面）**：此处的对齐指的是文字相对于整个视频画面的位置，如画面居左、画面居中。
*   **不透明度**：调整文字的透明程度。
*   **高级样式**：可以进一步为文字添加“描边”、“背景”、“发光”、“阴影”等效果，并自定义其颜色和强度。

## 花字与气泡：快速应用精美样式 ✨

在掌握了手动调整文字样式后，我们来看看如何快速应用精美的预设样式。“花字”和“气泡”功能可以让你一键获得专业效果。

“花字”功能提供了大量设计好的文字样式模板。如果你觉得自己调整样式不够美观，可以直接使用这些模板。

**使用方法**：
1.  在“文本”模块下，点击“花字”选项卡。
2.  浏览并选择你喜欢的花字样式。
3.  点击样式右侧的“+”号，或直接将其拖动到时间线轨道上。
4.  双击预览窗口中的文字，修改为你需要的文案。

“气泡”功能则是专门用于模拟对话气泡的样式，常见于漫画、对话场景或一些活泼的视频中。使用方法与“花字”类似，选择合适的“气泡”样式应用即可。

## 文字动画：让文字动起来 🎬

静态的文字有时显得单调，为文字添加动画能极大地增强视觉吸引力。剪映提供了丰富的入场、出场和循环动画效果。

选中轨道上的文字片段，在右侧编辑面板中找到“动画”选项。这里有三个子类别：

*   **入场动画**：文字出现时的动画效果。
*   **出场动画**：文字消失时的动画效果。
*   **循环动画**：文字在持续显示期间循环播放的动画效果。

**应用动画**：
1.  选择“入场”、“出场”或“循环”选项卡。
2.  从列表中选择一个动画效果（如“旋入”、“缩小”、“跳动”）。
3.  动画效果会立即应用。你还可以通过调整下方的“时长”滑块，来控制动画播放的快慢。

你可以为同一段文字分别添加入场和出场动画，让它的出现和消失都充满动感。

## 文字模板：一键生成片头与标题 🚀

如果你想更快速地生成复杂的文字效果，如片头、标题或动态标签，“文字模板”库是你的最佳选择。它包含了大量已设计好动画和版式的完整模板。

以下是使用文字模板的步骤：
1.  在“文本”模块下，点击“文字模板”选项卡。
2.  浏览分类（如标题、片头、字幕、流行等），找到喜欢的模板。
3.  将模板拖动到时间线轨道上。
4.  选中轨道上的模板，在右侧编辑面板中，找到“文本”编辑区。模板中的每一段文字都可以单独修改。
5.  修改文字内容，如将“怦然心动”改为你的视频标题。
6.  你仍然可以调整模板的整体位置、缩放和字体样式。

文字模板省去了手动组合动画和样式的繁琐步骤，极大地提升了包装效率。

## 智能字幕：自动生成与匹配 🗣️

对于有旁白、对话或采访的视频，添加字幕是一项重要但耗时的工作。剪映的“智能字幕”功能可以自动将语音转换为文字字幕。

**识别字幕**：
1.  将带有配音的视频素材导入轨道。
2.  点击“文本” -> “智能字幕”。
3.  点击“开始识别”按钮。
4.  剪映会自动分析音频，生成字幕片段并匹配到时间线上。识别完成后，你可以批量修改所有字幕的字体、颜色、大小等样式。

![](img/45453ce7529daceb230f870ad87c96e4_6.png)

**文稿匹配**：
如果你已有完整的文案稿件，使用“文稿匹配”功能可以更精准地生成字幕，避免语音识别可能产生的错误。
1.  将配音音频放入轨道。
2.  点击“文本” -> “智能字幕” -> “文稿匹配”。
3.  将准备好的文案粘贴到输入框中。
4.  点击“开始匹配”按钮，系统会根据音频节奏，将文案自动匹配为字幕。

![](img/45453ce7529daceb230f870ad87c96e4_7.png)

![](img/45453ce7529daceb230f870ad87c96e4_8.png)

**识别歌词**：
为音乐视频添加歌词同样简单。
1.  将音乐素材导入轨道。
2.  点击“文本” -> “识别歌词”。
3.  点击“开始识别”按钮，系统会自动生成歌词字幕。请注意检查是否有识别错误，并手动修正。

![](img/45453ce7529daceb230f870ad87c96e4_10.png)

![](img/45453ce7529daceb230f870ad87c96e4_11.png)

## 本地字幕：协作与高效导入 🤝

在团队协作或使用专业字幕工具时，常常会生成独立的字幕文件（如 `.srt`, `.lrc` 格式）。剪映支持直接导入这些本地字幕文件。

![](img/45453ce7529daceb230f870ad87c96e4_13.png)

**导入SRT字幕**：
1.  点击“文本” -> “本地字幕”。
2.  点击“导入”按钮，选择你的 `.srt` 或 `.lrc` 字幕文件。
3.  字幕文件会作为素材出现在“媒体”库中。
4.  将其拖动到时间线轨道上，对齐音频起点即可。
5.  导入后，你仍然可以像处理普通字幕一样，批量修改它的样式或位置。

![](img/45453ce7529daceb230f870ad87c96e4_14.png)

这种方式非常适合剪辑师与字幕员分工协作的工作流程，能显著提升效率。

![](img/45453ce7529daceb230f870ad87c96e4_16.png)
