# 欧众平台选品实操教程：P1：挖掘同行潜力与热销产品

在本节课中，我们将学习一个实用的欧众平台选品方法。通过这个方法，我们可以快速挖掘出同行店铺中的热销产品和潜力产品，非常适合精品店模式的卖家。

## 🛒 第一步：定位同行店铺

首先，我们需要找到一个同行的店铺作为分析目标。这个店铺可以是您所在品类中表现优秀或风格相似的店铺。

## 🔗 第二步：使用工具分析数据

上一节我们确定了分析目标，本节中我们来看看如何获取其销售数据。将目标店铺的链接复制到指定的分析工具中。

该工具能够抓取并展示该店铺所有商品的详细销售数据，包括：
*   **销售额**
*   **销量**
*   **评价数量** 等关键信息

## 📈 第三步：筛选与上架产品

获取数据后，接下来是筛选和行动。我们可以将同行店铺中销量表现良好的产品，直接上架到我们自己的店铺中。

以下是筛选产品时需要特别关注的两个要点：
*   **近期销量高**：这表明产品有稳定的市场需求。
*   **尚无评价或评价较少**：这类产品通常是店铺正在主推的**潜力爆款**，竞争可能相对较小，机会更大。
