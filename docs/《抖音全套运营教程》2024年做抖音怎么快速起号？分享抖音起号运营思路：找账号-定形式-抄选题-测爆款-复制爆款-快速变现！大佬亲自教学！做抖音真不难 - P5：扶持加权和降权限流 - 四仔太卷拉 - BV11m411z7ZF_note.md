# 抖音全套运营教程：P5：扶持加权与降权限流

在本节课中，我们将学习抖音平台对视频内容的两种核心处理机制：扶持加权与降权限流。理解这些规则，能帮助你有效提升视频曝光，并避免因违规导致流量受限。

上一节我们介绍了账号权重分类，本节中我们来看看哪些内容会受到平台的鼓励，哪些则会受到限制。

![](img/c0b30bdad740b46c1203d8c60770cbc9_1.png)

## 🚀 扶持加权的内容标准

![](img/c0b30bdad740b46c1203d8c60770cbc9_3.png)

平台会对符合特定技术标准的优质原创内容给予流量扶持。以下是获得扶持加权的关键要求：

![](img/c0b30bdad740b46c1203d8c60770cbc9_5.png)

*   **视频格式**：必须为竖版移动端原生视频，即 **9:16** 的格式。
*   **画面质量**：画面需保持平稳，作品清晰度高。
*   **音频质量**：无过多的杂音或噪音。

![](img/c0b30bdad740b46c1203d8c60770cbc9_1.png)

这些标准旨在提升用户的观看体验。符合上述要求的视频，更有可能进入更大的流量池，获得更多推荐。

## ⚠️ 会导致降权限流的内容

相反，如果视频内容触及平台规则红线，则会被降权限流，甚至无法通过审核。以下是需要严格避免的内容类型：

1.  **违规信息**：内容涉及吸烟、暴力、炫富（金钱）等不良导向。
    ![](img/c0b30bdad740b46c1203d8c60770cbc9_3.png)
2.  **外部引流**：视频中出现或引导用户至微信、微博、淘宝等站外联系方式。长时间展示商品Logo、手机号、二维码也属于违规行为。**注意**：二维码并非绝对禁止，但展示时间**最长不要超过3秒**，且在实际审核中可能更为严格。
    ![](img/c0b30bdad740b46c1203d8c60770cbc9_5.png)
3.  **硬性广告**：进行明显的营销广告、推销或引流卖货行为。
4.  **无口播的好物推荐**：这是许多初学者容易踩坑的地方。如果视频是拆箱或好物推荐类内容，**切忌全程无口播、只展示产品**。

    例如，有同学拍摄玩具或日用品，想通过视频带货，但小黄车总被下架。常见问题有两种：
    *   **直接标价**：在视频中直接写上“九块九”等价格信息，属于硬性销售。
    *   **沉默展示**：仅用背景音乐展示产品，没有任何口头介绍。平台会判定其为低质或纯广告内容。

    正确的做法是加入简单的口播介绍，例如：“我儿子玩这个玩具玩了四个小时都没停”或“这款纸巾沾水也不会破”。通过情景化或功能性的描述，将推荐软化，更容易通过审核。

5.  **口播广告**：在口播中直接念出联系方式或广告语。

![](img/c0b30bdad740b46c1203d8c60770cbc9_7.png)
![](img/c0b30bdad740b46c1203d8c60770cbc9_8.png)

**总结**：在抖音，硬广思维行不通。你需要将营销意图包裹在有价值、有观看性的内容之中。

本节课中，我们一起学习了抖音平台扶持内容的技术标准与导致降权限流的具体违规项。核心在于：制作**9:16**格式的优质原生视频，同时避免任何形式的硬性推广和不良信息。掌握这些规则，是账号健康运营、获取流量的基础。
