# 抖音自媒体运营从0~1全攻略：P12：10、如何寻找文案-1 📝

在本节课中，我们将要学习如何解决短视频创作中的核心难题之一——文案创作。我们将介绍几个实用的在线文案工具，并详细讲解如何利用它们高效地获取和生成优质文案，甚至解决配音问题。

上一节我们讨论了优化作品以提升吸粉能力的重要性。本节中，我们来看看如何具体优化作品的第一步：解决文案问题。

很多初学者不会写文案，这没有关系。以下是五个可以直接在网页端使用的文案平台，无需下载APP，在手机或电脑浏览器中搜索即可使用。

*   淘金阁
*   （其他四个平台名称，根据视频内容补充）

我以“淘金阁”为例进行讲解。这是该网站的网址，如果你直接搜索时广告太多，可以直接输入网址访问。

![](img/5a0ae126f88c48352db38393bda2a494_1.png)

![](img/5a0ae126f88c48352db38393bda2a494_1.png)

网站主页提供了非常多的内容分类，包括财经、军事、动漫、科技、美食、美文、星座、体育、旅游、育儿、汽车、社会、娱乐、时尚、房产、国际、宠物等。

我们可以任意点开一个分类，例如“情感”。页面中会列出大量该类别下的现成话题文案，并且通常有多页内容可供选择。

![](img/5a0ae126f88c48352db38393bda2a494_3.png)

![](img/5a0ae126f88c48352db38393bda2a494_3.png)

之前提到的“一个群体加一个核心话题加疑问”的文案公式，可以在这里套用筛选。你觉得合适的文案就可以直接采用。

![](img/5a0ae126f88c48352db38393bda2a494_5.png)

如果现有分类中没有你想要的特定内容，可以使用关键词搜索功能。例如，在搜索框输入“宝妈”，然后点击“开始抓取”。

![](img/5a0ae126f88c48352db38393bda2a494_5.png)

![](img/5a0ae126f88c48352db38393bda2a494_7.png)

搜索结果中会高亮显示所有包含“宝妈”的文案，这通常足以满足创作需求。手机和电脑的操作方法完全相同。

![](img/5a0ae126f88c48352db38393bda2a494_7.png)

![](img/5a0ae126f88c48352db38393bda2a494_9.png)

接下来，针对担心自己声音不好听的同学，平台还提供了免费的配音生成工具。点击网站右上角的“小工具”，然后选择第四个“配音在线生成”功能。

![](img/5a0ae126f88c48352db38393bda2a494_9.png)

![](img/5a0ae126f88c48352db38393bda2a494_11.png)

这个功能完全免费，且配音种类非常丰富。

![](img/5a0ae126f88c48352db38393bda2a494_11.png)

![](img/5a0ae126f88c48352db38393bda2a494_13.png)

![](img/5a0ae126f88c48352db38393bda2a494_13.png)

以下是主要的配音类型：

*   **亲和女生、温柔女生、甜美女生、自然女声**等
*   **直播配音**如亲切女声、活力女声
*   **通用场景**如标准男声、标准女声
*   **特色配音**如湖南口音、童声
*   **外语配音**如英文
*   **方言配音**如四川话、粤语、东北话、台湾话
*   **精品男配**如情感解说、影视解说、娱乐解说

![](img/5a0ae126f88c48352db38393bda2a494_15.png)

![](img/5a0ae126f88c48352db38393bda2a494_15.png)

使用方法很简单。在左侧“配音文本”区域输入或粘贴你的文案。

![](img/5a0ae126f88c48352db38393bda2a494_17.png)

![](img/5a0ae126f88c48352db38393bda2a494_17.png)

例如输入：“欢迎来到我的课堂，今天我们来学习如何寻找文案。”

![](img/5a0ae126f88c48352db38393bda2a494_19.png)

![](img/5a0ae126f88c48352db38393bda2a494_19.png)

接着，在右侧选择你喜欢的声音类型，例如选择“东北话”。你还可以调整音量大小、语速快慢和语调高低。

![](img/5a0ae126f88c48352db38393bda2a494_21.png)

调整完毕后，点击“立即生成”。在手机上使用时，可以扫码登录后免费使用。

![](img/5a0ae126f88c48352db38393bda2a494_21.png)

![](img/5a0ae126f88c48352db38393bda2a494_23.png)

![](img/5a0ae126f88c48352db38393bda2a494_23.png)

生成后，在“配音生成列表”中，可以直接下载音频文件到手机。

![](img/5a0ae126f88c48352db38393bda2a494_25.png)

![](img/5a0ae126f88c48352db38393bda2a494_25.png)

下载的音频可以在后续的剪辑环节直接使用。这样，你不仅解决了文案来源问题，连配音问题也一并解决了。

![](img/5a0ae126f88c48352db38393bda2a494_27.png)

![](img/5a0ae126f88c48352db38393bda2a494_27.png)

本节课中我们一起学习了如何利用在线工具平台寻找和生成短视频文案。我们掌握了通过分类浏览和关键词搜索获取文案的方法，并学会了使用免费的在线配音工具为文案添加高质量的声音，从而有效降低了内容创作的门槛。
