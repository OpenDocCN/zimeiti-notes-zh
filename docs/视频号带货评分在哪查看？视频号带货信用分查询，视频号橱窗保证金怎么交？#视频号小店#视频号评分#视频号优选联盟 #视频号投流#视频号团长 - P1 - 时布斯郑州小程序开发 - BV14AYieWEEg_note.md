# 视频号运营指南：P1：带货评分、信用分与保证金查询教程

在本节课中，我们将学习如何在视频号中查看关键的运营数据，包括**带货评分**、**达人信用分**以及**橱窗保证金**的查询方法。这些数据是评估账号健康度和带货权限的重要指标。

![](img/7a68726fa2b327144e88b95d1d291bd2_0.png)

上一节我们介绍了课程目标，本节中我们来看看具体的操作步骤。

以下是查看带货评分与相关数据的详细步骤列表：

1.  打开微信，点击底部导航栏的 **“发现”** 选项卡。
2.  进入 **“发现”** 页面后，点击 **“视频号”** 入口。
3.  在视频号页面，点击右上角的 **个人头像图标**。
4.  在弹出的页面中，找到并点击 **“创作者中心”**。
5.  进入创作者中心后，点击 **“带货中心”** 功能模块。
6.  在带货中心页面右上角，您可以看到三个关键数据：
    *   **带货评分**：例如显示为 **4.46**，这是评估您带货能力的核心指标。
    *   **达人信用分**：例如显示为 **12**，这反映了您的账号信用状况。
    *   **橱窗保证金**：显示为“宝子”，此处应为您已缴纳的保证金金额或状态提示。

通过以上步骤，您可以清晰地查看到自己的带货评分及相关信用信息。

![](img/7a68726fa2b327144e88b95d1d291bd2_2.png)

本节课中我们一起学习了如何逐步进入视频号后台，并定位到带货评分、信用分和保证金信息的查看位置。掌握这些数据的查询方法是进行视频号带货运营的基础。
