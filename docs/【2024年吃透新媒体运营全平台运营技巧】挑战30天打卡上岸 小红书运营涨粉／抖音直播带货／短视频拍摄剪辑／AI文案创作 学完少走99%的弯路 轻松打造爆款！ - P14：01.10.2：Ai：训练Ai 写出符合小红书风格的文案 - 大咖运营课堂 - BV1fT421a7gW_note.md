![](img/e90365af090e5ac16b4824ec55ae2e3d_1.png)

在本节课中，我们将学习如何训练AI（如ChatGPT）来撰写符合特定平台（如小红书）风格和爆款结构的内容文案。上一节我们探讨了如何让AI生成爆款标题，本节我们将重点转向内容文案的创作。

## 核心思路：让AI学习平台风格

![](img/e90365af090e5ac16b4824ec55ae2e3d_3.png)

大部分AI能够生成内容，但关键在于如何让它写出符合特定平台风格和用户喜好的文案。要实现这一点，我们需要让AI“学习”该平台的独特风格。核心方法是为AI提供该平台的爆款文案作为学习材料，让它从中分析并掌握文案的风格、语气和专用词汇。

**核心公式**：`平台风格 = 爆款案例 + 风格分析 + 关键词/语气提炼`

### 第一步：给AI下达明确指令

![](img/e90365af090e5ac16b4824ec55ae2e3d_33.png)

![](img/e90365af090e5ac16b4824ec55ae2e3d_35.png)

![](img/e90365af090e5ac16b4824ec55ae2e3d_40.png)

首先，你需要明确告诉AI它的任务。指令应清晰说明：你将提供平台爆款文案，它需要分析这些文案的撰写风格、平台专用词汇、语气等特点，并在每次学习后给出反馈。

![](img/e90365af090e5ac16b4824ec55ae2e3d_42.png)

![](img/e90365af090e5ac16b4824ec55ae2e3d_44.png)

![](img/e90365af090e5ac16b4824ec55ae2e3d_46.png)

![](img/e90365af090e5ac16b4824ec55ae2e3d_48.png)

**示例指令代码**：
```
接下来，我会为你提供一些小红书平台的爆款笔记文案。
你需要根据这些文案，分析小红书的文案撰写风格、特定的平台用语、词汇、语气等。
每次收到文案后，请告诉我你已经理解，并给出对该文案风格、语气、类型等方面的分析。
```

### 第二步：筛选并投喂学习材料

![](img/e90365af090e5ac16b4824ec55ae2e3d_58.png)

![](img/e90365af090e5ac16b4824ec55ae2e3d_60.png)

![](img/e90365af090e5ac16b4824ec55ae2e3d_64.png)

选择学习材料至关重要。应选择具有鲜明平台风格、而非通用信息的文案。

![](img/e90365af090e5ac16b4824ec55ae2e3d_66.png)

![](img/e90365af090e5ac16b4824ec55ae2e3d_68.png)

**选择材料的注意事项**：
*   **避免通用内容**：例如单纯的“第一天、第二天、第三天”旅游路线清单，这类内容缺乏风格特征，AI难以学到精髓。
*   **寻找风格化文案**：应选择以第一人称分享、带有强烈个人情感（如惊喜、吐槽）、使用平台特色用语（如“宝子”、“姐妹”、“绝绝子”）的笔记。
*   **覆盖多种类型**：尽可能提供不同风格（如种草、避坑、经验分享）的文案，让AI的学习更全面。

### 第三步：让AI进行内容创作

![](img/e90365af090e5ac16b4824ec55ae2e3d_78.png)

![](img/e90365af090e5ac16b4824ec55ae2e3d_80.png)

![](img/e90365af090e5ac16b4824ec55ae2e3d_82.png)

在投喂足够的学习材料后，即可指示AI进行创作。指令需结合你的具体需求和学习目标。

![](img/e90365af090e5ac16b4824ec55ae2e3d_84.png)

![](img/e90365af090e5ac16b4824ec55ae2e3d_86.png)

**示例创作指令**：
```
请结合你刚才的学习，帮我撰写一篇具有爆款潜质的“张家界5日游攻略”小红书笔记。
要求：涵盖小红书风格的Emoji表情，并同时生成10个符合小红书旅游赛道风格的爆款标题。
```

我们以“张家界旅游攻略”为例进行了演示。在投喂了几篇风格各异的笔记后，我们指示AI进行创作。

![](img/e90365af090e5ac16b4824ec55ae2e3d_98.png)

![](img/e90365af090e5ac16b4824ec55ae2e3d_100.png)

**初始结果分析**：
AI生成的初稿在景点描述上偏向诗化语言（如“仙境探索”、“梦幻之旅”），但缺乏实用性的细节和“避坑”建议，这与小红书上用户喜爱的“真实亲历分享”风格尚有差距。

![](img/e90365af090e5ac16b4824ec55ae2e3d_102.png)

![](img/e90365af090e5ac16b4824ec55ae2e3d_104.png)

![](img/e90365af090e5ac16b4824ec55ae2e3d_106.png)

**问题根源**：
1.  **学习材料不足**：提供的样本中，深度分析类、避坑吐槽类内容较少，导致AI对“实用干货”风格学习不够。
2.  **指令可更具体**：创作指令可以更细化，例如要求其按“景点亮点-避坑指南-行程建议”的结构来写。
3.  **模型差异**：使用更高级的模型（如GPT-4）通常比基础模型（如GPT-3.5）在理解复杂指令和生成高质量内容上表现更佳。

**优化方案**：
*   **分步创作**：先让AI列出张家界所有主要景点及特点，再针对每个景点生成“推荐理由”和“避坑提示”，最后组合成文。
*   **结合标题技巧**：将上一节课的爆款标题生成指令，在新的对话中与内容创作指令结合使用，能产生更符合平台调性的标题。

## 核心要点总结

![](img/e90365af090e5ac16b4824ec55ae2e3d_118.png)

![](img/e90365af090e5ac16b4824ec55ae2e3d_120.png)

本节课我们一起学习了训练AI撰写平台化文案的核心方法。

![](img/e90365af090e5ac16b4824ec55ae2e3d_122.png)

**关键总结**：
1.  **目的明确**：训练AI的核心是让它理解并模仿特定平台的**行文风格和用户语境**，而不仅仅是生成通顺的文字。
2.  **材料是关键**：**投喂的学习材料质量直接决定AI的输出质量**。应精心挑选具有代表性的、风格鲜明的爆款文案。
3.  **指令需迭代**：根据AI的初次输出结果，调整和细化你的指令，这是一个不断优化和沟通的过程。
4.  **分场景使用**：对于复杂任务（如写攻略），可拆解步骤；对于风格化任务（如写标题），可创建新对话结合专用指令。

![](img/e90365af090e5ac16b4824ec55ae2e3d_124.png)

通过本节学习，你掌握了让AI从“通用内容生成器”转变为“平台专属文案助手”的基本方法。下一节课，我们将继续深入，探讨如何利用AI解决更多新媒体运营中的具体难题。
