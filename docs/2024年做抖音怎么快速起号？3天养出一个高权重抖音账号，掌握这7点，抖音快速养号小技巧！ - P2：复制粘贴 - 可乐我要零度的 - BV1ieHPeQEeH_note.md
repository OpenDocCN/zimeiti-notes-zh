# 剪映教程：P2：复制粘贴

![](img/4b5c8a5e09ef4f1d47704461731296a9_0.png)

## 概述
在本节课中，我们将学习一种利用剪映快速制作视频并获取收益的实用方法。这种方法特别适合新手，无需粉丝基础或带货经验，通过简单的“复制粘贴”和二次创作流程，即可生成原创视频内容。

---

## 核心步骤详解

上一节我们介绍了利用剪映进行创作的基本思路，本节中我们来看看具体的操作流程。整个过程可以分解为以下几个关键步骤。

### 1. 寻找并获取素材
首先，我们需要找到合适的视频素材作为创作基础。

以下是具体操作方法：
*   打开抖音，在搜索框中输入“情感故事”。
*   在搜索结果页面选择“视频”标签。
*   点击“筛选”按钮，设置条件为“最多点赞”、“一周内”、“时长1-5分钟”。
*   从筛选结果中，选择一个点赞量较高的视频。
*   进入该视频播放页，点击右下角的“分享”按钮，选择“复制链接”。

### 2. 提取文案与生成视频
获取链接后，下一步是提取其中的文案并利用剪映的智能功能生成初步视频。

操作流程如下：
*   打开一个文案提取工具（如轻抖等），将复制的链接粘贴进去。
*   点击“提取文案”或类似功能，然后“复制”提取出的全部文字。
*   打开手机剪映APP，点击“图文成片”功能。
*   在“正文”框中粘贴刚刚复制的文案，“标题”可留空。
*   点击“完成编辑”，在下方选择“智能匹配素材”。
*   最后，点击“生成视频”，等待系统自动生成一个带有画面和配音的初版视频。

### 3. 进行二次创作
系统生成的视频需要经过二次处理，以增加原创性，避免平台判定为搬运。

以下是二次创作的关键点：
*   在剪映中“开始创作”，导入上一步生成好的视频和一个提前准备好的“解压视频”素材（如风景、动态背景等）。
*   选中生成的原视频，点击“音频分离”，将视频画面和背景音乐/配音分离。
*   删除分离出来的原始视频画面，只保留其音频轨道和新增的“解压视频”画面。
*   此时，核心操作是利用剪映的各种工具对视频进行深度处理：
    *   **添加文本**：为视频添加字幕、标题或花字。
    *   **应用转场**：在不同片段间添加转场效果。
    *   **使用特效**：为视频添加滤镜、动画等特效。
    *   **调整音频**：可以调整配音音量，或添加新的背景音乐。
*   经过以上处理，视频就变成了一个**原创度较高**的新作品。

### 4. 发布与获取收益
视频制作完成后，需要通过正确的渠道发布才能实现收益。

最终发布步骤如下：
*   点击右上角的“导出”按钮，将制作好的视频保存到手机。
*   打开抖音，通过“中视频伙伴计划”的入口发布该视频。
*   视频产生播放量后，通常第二天就能在相关平台看到收益数据。

## 总结
本节课我们一起学习了通过“复制粘贴”素材并进行智能匹配与二次创作来快速制作视频的全流程。关键在于利用工具提取文案，借助剪映的“图文成片”功能快速生成初稿，再通过添加素材、文本、特效等手段进行深度二次创作，最终将其转化为原创内容并通过合适渠道发布以获得收益。这个方法降低了视频创作的门槛，适合新手入门实践。
