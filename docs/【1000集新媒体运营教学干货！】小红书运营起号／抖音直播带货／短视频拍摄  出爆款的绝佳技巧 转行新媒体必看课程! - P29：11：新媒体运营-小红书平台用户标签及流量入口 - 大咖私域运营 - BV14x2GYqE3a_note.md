# 新媒体运营：P29：11：小红书平台用户标签及流量入口 📊

![](img/6a11315968ecb8b6568ccbec852a12c4_1.png)

在本节课中，我们将深入探讨小红书平台如何为用户打上标签，以及平台内主要的流量入口有哪些。理解这些机制，有助于我们分析笔记如何被系统推荐给精准用户，从而优化运营策略。

上一节我们介绍了系统如何判定内容标签，本节中我们来看看平台如何判定用户标签。这个过程能帮助我们反向推导，如何让笔记被我们的精准用户看到。

## 用户标签判定机制 🔍

系统为用户打标签的主要依据是用户的**浏览行为**。用户注册时选择的兴趣范围参考价值不大，它仅作为初期内容方向的引导，无法进入系统深层的用户标签池。

系统根据用户的浏览行为来判定其兴趣分类。以下是几个关键的行为指标：

### 点击率

**点击率**是指系统在发现页向用户推送多个内容（如明星、美妆、家居、母婴等）后，用户点击各类内容的概率。例如，用户点击了10条内容，其中美妆类3次，家居类2次，母婴类0次。系统会判定该用户对母婴内容不感兴趣，可能不是宝妈，并对美妆内容更感兴趣。系统将持续推送更多美妆内容进行试探，通过点击率逻辑为用户打上**强关联**或**弱关联**的标签。

### 停留时长

**停留时长**是另一个重要的分析行为。例如，用户喜欢美妆内容，但点开某些美妆笔记后很快划走，说明这些笔记并非其偏好。系统会尝试推送美妆领域内其他细分内容（如上妆、护肤）或其他领域内容（如家居）来观察用户行为。如果用户对一条家居笔记停留很久，仔细阅读图片和评论，系统可能判定该用户有装修或购买家居用品（如盘子）的需求，从而调整后续推荐。

### 页面浏览量

**页面浏览量**可以理解为用户浏览页面的深度。例如，用户点开一篇装修笔记后，还进入了博主主页，并浏览了第二、第三、第四篇笔记。这个数值（PV）对账号健康度有非常大的帮助。提升用户在你主页的浏览深度，能为账号带来积极的评分。

### 互动行为

以下是主要的用户互动行为，每一项数据都值得我们努力提升：
*   **点赞**
*   **收藏**
*   **评论**
*   **关注**

互动数据高的笔记，进入更高层级推荐流量池的几率更大，也有助于带动账号内其他笔记的流量，并提升粉丝数。这是我们运营小红书一直努力的目标。

## 平台流量分发入口 🚪

小红书平台主要有四个流量分发入口。作为用户可能经常使用，但未必分析过其区别。

### 发现页

发现页是笔记和账号后期流量的主要来源。其呈现的内容主要包括：
*   **关注达人**：用户已关注的博主发布的内容。
*   **兴趣内容匹配**：系统根据用户标签（点击率、停留时长等）判断其可能喜欢的内容。
*   **发散性内容匹配**：系统也会推荐一些用户可能不感兴趣或未刷到过的内容，以增加平台内容的多样性，满足用户多样化的需求（如看攻略、教程、趣事分享）。

### 关注页

关注页的内容相对简单：
*   **关注达人笔记**：仅展示用户所关注博主发布的内容。用户通常从这里直接查找强关注的博主。

### 附近页

附近页的功能类似于早期微信的“附近的人”：
*   **基于地理位置推送**：根据手机定位，推送附近及同城的相关内容。
*   **内容特征**：主要以美食、当地旅游等强地域特征的需求内容为主。
*   **适用场景**：适合线下实体店（如美容院、餐厅、景点）的推广。在做小红书信息流广告时，可重点关注附近人群的推送。

### 购物页

购物页是小红书向电商方向发展的重要入口：
*   **匹配购物需求**：直接满足用户的购物搜索和浏览需求。
*   **现状**：部分用户尚未在小红书形成强购物习惯，更多人仍倾向于在传统电商平台或抖音购物。用户吐槽笔记广告出现频率过高的问题也与此相关。
