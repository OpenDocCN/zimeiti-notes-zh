# 小红书电商运营课：P9：如何在抖店选品 🛍️

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_1.png)

在本节课中，我们将学习如何利用抖音小店（抖店）的数据，为小红书店铺进行高效选品。我们将重点介绍一种低成本、无需依赖昂贵数据软件的实操方法，帮助你筛选出具有爆款潜力的商品。

## 为什么要在抖店选品？ 🤔

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_3.png)

上一节我们介绍了选品的重要性，本节中我们来看看为什么选择抖店作为参考平台。互联网上的爆款商品通常具有共性，尤其是在新媒体平台之间。抖音和小红书同属新媒体平台，抖音拥有成熟的电商体系（即抖店）。因此，在抖音电商中销售表现良好的商品，在小红书平台上也有很大概率会畅销。这就是我们选择在抖店进行选品的主要逻辑。

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_5.png)

## 低成本选品方法：利用抖音橱窗 🛒

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_6.png)

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_7.png)

市面上有许多选品数据软件，但价格昂贵。为了坚持轻资产、低成本创业的原则，我们将使用抖音橱窗功能进行选品，这是一种免费且有效的方法。

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_9.png)

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_10.png)

进行此操作，你需要一个已开通商品橱窗功能的抖音账号。开通橱窗通常需要缴纳**500元保证金**（此保证金后续可退还）。如果你自己没有，也可以借用已开通此功能的他人账号。

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_12.png)

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_13.png)

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_15.png)

以下是利用抖音橱窗选品的具体操作步骤。

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_17.png)

### 第一步：进入选品广场

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_19.png)

1.  打开抖音APP，进入个人主页，点击“商品橱窗”。
2.  在橱窗首页，找到并点击“选品广场”入口。

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_21.png)

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_23.png)

### 第二步：筛选潜力商品

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_25.png)

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_27.png)

进入选品广场后，平台会默认展示近期热卖商品。我们可以利用以下几个榜单进行初步筛选：
*   **爆款热卖榜**：当前销量极高的商品。
*   **搜索热度榜**：用户搜索频繁的商品。
*   **搜索趋势榜**：搜索量正在快速上升的商品。

在开始具体筛选前，我们需要明确几个核心选品原则，以确保选出的商品更容易成为爆款。

## 核心选品原则 📝

以下是筛选商品时必须考虑的四个关键原则：

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_33.png)

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_34.png)

1.  **趋势向上**：商品的整体销量或搜索量曲线应呈现**上升趋势**，而非下降或平稳。这代表其正处于市场需求增长期。
2.  **款式普适**：商品款式不应过于挑人。过于独特或挑身材的款式会导致**退货率升高**，影响利润。
3.  **符合当季**：优先选择应季商品。例如，春季选择春装，夏季选择防晒用品。当季商品转化率更高。
4.  **销量适中**：目标商品的月销量建议在 **1000件至5000件** 之间。这个区间的商品已经过市场验证，且竞争尚未达到白热化，有操作空间。若找不到完全符合的，可适当放宽标准。

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_36.png)

## 实操案例：筛选女童服装 👗

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_37.png)

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_38.png)

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_39.png)

现在，让我们以“亲子童装”类目为例，进行一场完整的选品实操。

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_40.png)

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_41.png)

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_43.png)

1.  **搜索与初选**：在选品广场搜索“童装”，选择“亲子童装”细分品类。我们从榜单中选取一款“女童格子套装”作为示例。
2.  **应用原则筛选**：
    *   **查看趋势**：进入商品详情页，观察其销售趋势曲线。忽略短期波动，判断**整体曲线是否向上**。我们选择的案例商品趋势明显向上。
    *   **评估款式**：商品为经典格子套装，款式大方，不挑人，符合**普适性原则**。
    *   **检查季节**：春季是童装上新季，该商品为**当季春装**。
    *   **核对销量**：该商品月销1000多件，落在我们建议的 **1000-5000件** 区间内。
3.  **确认选品**：该商品符合全部四项核心原则，可被确定为潜力选品。

## 货源寻找与差价计算 💰

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_47.png)

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_48.png)

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_50.png)

确定潜力商品后，下一步是寻找低价货源并计算利润空间。

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_52.png)

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_53.png)

1.  **截图保存**：在抖店商品详情页，截取商品主图。
2.  **以图搜货**：打开拼多多（多多）APP，使用“图片搜索”功能，上传刚才的截图。
3.  **筛选货源**：在搜索结果中，寻找同款或类似款商品。优先选择：
    *   销量高（如上万件）且带有较多真实评价（如几百个）的商品，这代表货源稳定。
    *   价格更低的商品。
4.  **计算差价**：对比抖店售价与拼多多货源价。
    *   例如，抖店售价 **39元**，拼多多找到同款货源价 **28元**，则单件差价（毛利润）为：
        `39 - 28 = 11元`
    *   另一个案例中，抖店售价 **99元** 的服装，在拼多多找到类似款货源价约 **50元**，则单件差价约为：
        `99 - 50 = 49元`

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_55.png)

**注意**：服装等非标品利润空间较大，在小红书定价时可适当提高，无需完全对标抖店价格。

## 总结 📚

![](img/68e7d262e22ed70bc8fac2ecf137e1d3_62.png)

本节课中，我们一起学习了如何为小红书店铺在抖音小店进行选品。我们首先理解了跨平台选品的逻辑，然后掌握了通过抖音橱窗进行低成本选品的具体步骤。核心在于应用 **趋势向上、款式普适、符合当季、销量适中** 四项原则筛选潜力商品，最后通过拼多多等平台寻找低价货源并计算利润。这套方法能帮助你高效地找到具有爆款潜力的商品，为小红书店铺的运营打下坚实基础。
