# 抖音运营教程：P9：策划内容定位 🎯

在本节课中，我们将要学习如何为你的抖音账号策划一个清晰、具体且有竞争力的内容定位。仅仅选择一个热门主题是不够的，我们还需要深入规划身份、场景、道具、态度等核心要素，才能让你的内容在众多同质化账号中脱颖而出。

---

上一节我们介绍了如何构思主题IP，本节中我们来看看如何将这个主题深化为具体、可执行的内容定位。

很多同学可能会担心：我选择的主题（例如“摄影教学”、“美食探店”）已经有很多人在做了，内容似乎都差不多，我现在再做还能火起来吗？这个问题的关键就在于，除了主题IP之外，我们还需要构思更具体的核心内容。

例如，选择了“老韩摄影教学”这个主题后，还需要确定：具体教什么？是拍人像还是风景？用手机教还是相机教？这些更细致的规划，决定了你的内容是否有独特性。

因此，一个完整的内容定位是一环扣一环的。在确定主题IP之后，我们需要进一步规划以下要素：

以下是构成内容定位的几个核心维度，我们需要逐一进行构思：

*   **身份**：你以什么角色出现？是专业摄影师、幽默的吃货，还是知心姐姐？
*   **场景**：你的内容发生在什么环境？是咖啡馆、厨房、工作室，还是户外？
*   **道具**：你会使用哪些标志性的物品？例如特定的服装、器材或小道具。
*   **动作**：你的标志性行为或流程是什么？
*   **机位**：你习惯用什么角度和镜头语言来拍摄？
*   **态度**：你传达的核心情绪或价值观是什么？是专业、暴躁、温暖还是搞笑？

这些要素越丰富、越具体，你的内容定位就越明确，后续的内容产出也会越容易。

为了让大家更好地理解，我们来看一个成功案例：“仙女酵母”这个账号。

![](img/87a37222471b3400c1788e736e904532_9.png)

这个账号的段子内容本身并不新颖，甚至有些简单。但它获得了巨大的成功，拥有超过1500万粉丝。其成功的关键就在于独特的内容策划。

以下是“仙女酵母”账号的内容定位拆解：

*   **身份**：设定为一位住在古堡里的“公主”或“仙女教母”。
*   **场景**：选择欧式古典风格的室内场景。
*   **道具**：华丽的复古长裙、头饰、宫廷式家具。
*   **动作**：接听现代人打来的“求助电话”，并以其古典身份进行幽默回应。
*   **态度**：形成一种“古典身份”与“现代烦恼”之间的反差萌和幽默感。

正是这种在**身份、场景、道具**上的极致化、风格化打造，使得即使内容（段子）本身普通，也能凭借强烈的形式感和记忆点突出重围。这部分工作，本质上属于**人物IP的打造**。

本节课中我们一起学习了如何深化内容定位。我们明白了，不能只停留在选择一个热门主题，而需要通过系统规划**身份、场景、道具、动作、机位、态度**这六个维度，来构建内容的独特性和竞争力。通过“仙女酵母”的案例，我们看到，即使内容内核相似，出色的形式策划也能打造出爆款账号。记住，定位越具体，你的个人特色就越鲜明，也越容易吸引和留住粉丝。
