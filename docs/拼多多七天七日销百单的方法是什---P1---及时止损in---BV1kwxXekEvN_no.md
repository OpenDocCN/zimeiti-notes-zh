# 拼多多运营教程：P1：七日百单核心四步法

在本节课中，我们将学习拼多多平台上一个旨在实现“7天日销百单”目标的核心运营方法。该方法围绕四个基础设置展开，并配合精准的直通车投放策略。

## 🎯 第一步：虚高定价

虚高定价是后续所有促销活动的基础。其操作非常简单：将商品的原价设定为正常计划售价的两倍或以上。例如，若你计划最终售价为50元，则初始定价应设置为100元或更高。

## 🎯 第二步：设置保价比

上一节我们设置了虚高价格，本节中我们来看看如何通过“保价比”规则，让这个高价看起来合理。核心方法是改变商品描述中的计量单位或使用形容词进行修饰。

以下是具体操作示例：
*   将“1500克”描述为“1.5000克”。
*   使用“家庭装”、“豪华版”、“加量款”等词汇进行修饰。

## 🎯 第三步：创建限时优惠券

设置好价格基础后，我们需要通过强有力的促销工具来刺激购买。限时限量优惠券是关键。

具体操作是：在商家后台直接创建一张“限时限量购”优惠券。成功设置后，商品链接会自动显示“限时促销”横幅，极大地提升点击与转化意愿。

## 🎯 第四步：配置低价SKU

仅靠优惠券还不够，我们还需要在商品页面内营造强烈的低价感知。配置低价SKU（商品属性）是实现这一目标的有效手段。

操作方法是：在商品编辑页面，设置一个价格极低的SKU选项（例如1元试用装）。该SKU的价格会以醒目横幅的形式展示在商品主图区域，吸引用户点击进入。

## 🚀 进阶：稳定日销的直通车策略

完成以上四个基础设置只是第一步。想要稳定达成日销百单的目标，还需要配合“8倍直通车”策略来稳定投产比（ROI）。

首先，你需要计算保本投产比，公式为：
**保本投产比 = 售价 / 利润**

前期，我们将直通车的目标投产比设置为这个保本值，先让计划跑起来。运行两天数据稳定后，需要计算更精确的净投产比。

接下来，将退款费率和平台技术服务费计入成本，此时计算出的投产比即为**净投产比**。以此为目标继续跑3天。

最后，在第七天时，将净投产比乘以1.3，得到**最终目标投产比**。
**最终投产比 = 净投产比 × 1.3**

直通车只有开到最终投产比，才是真正意义上的盈利。
