## 一、剪映是什么？ 🤔

剪映是由抖音官方推出的一款手机视频编辑工具，可用于手机短视频的剪辑、制作和发布。它拥有全面且简洁的剪辑操作功能，支持**变速**、**滤镜**、**转场**等多种剪辑效果，并且自带抖音官方的丰富音乐素材库。

说起剪映，就不得不提它的开发公司——深圳脸萌科技有限公司。脸萌科技此前已被字节跳动（即抖音母公司）收购。为了更好地推动抖音及头条系在短视频赛道的发展，抖音于去年3月份正式推出了剪映App。

![](img/5e19ab8f60f0e5f9ba14d0bbd611a4a8_3.png)

在短视频领域，快手早期也以视频剪辑工具起家，并随后推出了自己的剪辑工具“快影”。抖音推出剪映，既是为了应对市场竞争，也是为了更好地辅助平台自身的内容生态发展。了解这一点，有助于我们明白为什么在抖音上，许多标注“由剪映发布”的视频能获得更好的流量推荐，这对于我们未来在抖音平台深耕发展有极大帮助。

如果您的手机上尚未下载剪映，建议在课程结束后立即前往应用商店下载。我们后续的所有视频教程都将围绕这款App展开。

以下是关于“剪映是什么”的小结：

*   **官方出品**：剪映是由抖音官方推出的剪辑App。
*   **完全免费**：与许多需要开通会员的剪辑软件（如快剪辑、爱剪辑、Videoleap、巧影等）不同，剪映是一款完全免费的工具。
*   **简单易用**：它的操作界面和功能设计对新手非常友好，剪辑完成后可以一键分享至抖音平台。

![](img/5e19ab8f60f0e5f9ba14d0bbd611a4a8_5.png)

## 二、剪映做短视频的优势有哪些？ ✨

我们为什么要重点学习剪映？以下是总结出的五点核心优势：

1.  **简单好用**：剪映的操作界面和逻辑非常直观，对零基础用户极其友好，一学就会。
2.  **完全免费**：无需开通会员或为素材、特效付费，所有功能均可免费使用。
3.  **功能强大**：尽管当前版本仍在更新中，但剪映已具备文本、音频、贴纸、特效、色度抠图（可处理绿幕素材）等强大功能，足以满足抖音短视频制作的各类需求。
4.  **海量素材**：App内提供了大量抖音热门的音视频素材库和竖屏素材，用户可以便捷地取用，快速制作同款热门视频，无需自行费力寻找。
5.  **官方推荐**：剪映与抖音同属字节跳动旗下。通过剪映剪辑并发布的视频，在抖音平台通常会获得一定的流量推荐和扶持。这也是许多爆款视频下方会标注“剪映制作”的原因。

以下是使用剪映制作短视频的五大优势小结：
**优势 = 简单好用 + 完全免费 + 功能强大 + 海量素材 + 官方推荐**

了解了剪映的强大优势后，我们接下来看看，具体是哪些人群最能从中受益。

## 三、哪些人适合用剪映做短视频？ 🎯

剪映是一款入门级视频剪辑神器，虽然对新手友好，但并非适用于所有人。它特别适合以下三类人群：

1.  **喜欢拍摄的人群**：在移动互联网时代，许多人习惯用手机记录生活。相比需要高性能电脑和专业技术的PR、AE等软件，剪映能让用户随时随地快速完成视频剪辑，极大提升效率。
2.  **愿意分享的人群**：抖音等短视频平台的核心是分享。如果你乐于分享日常生活、点滴感悟或故事，那么剪映能帮助你轻松地将内容记录下来并制作成视频发布。
3.  **短视频新手/小白**：对于想要进入短视频领域或从事自媒体工作的初学者来说，剪映操作简单、上手快速，不需要复杂的剪辑技术，是理想的起步工具。

以下是适合使用剪映的三类人群小结：
**目标用户 ∈ {喜欢拍摄的人群， 愿意分享的人群， 短视频新手}**
