## 笔记不被收录的五大原因

以下是导致小红书笔记不被收录的五个核心原因，我们将逐一进行解析。

### 1. 存在违规推广行为
笔记内容若包含明显的硬性广告，例如直接提及品牌名称、产品型号或进行露骨的销售引导，平台会判定为违规推广，从而不予收录。平台鼓励的是**原生、真实、有价值**的分享。

### 2. 含有平台敏感词
您的笔记或账号历史中若存在违规内容，即使笔记被暂时收录，系统在审核后也可能将其剔除。关键在于避免使用敏感词汇。

**核心概念**：平台会通过算法检测文本中的**敏感词**。例如：
```python
# 伪代码示例：敏感词检测逻辑
if note_text contains (敏感词列表):
    note_status = "不被收录"
```

内容违规主要分为图片和文字两部分：
*   **图片违规**：避免使用带有其他平台水印或二维码的图片，这类跨平台引流行为已被小红书明令禁止。
*   **文字违规**：严禁出现不文明词汇、诱导性语言以及政治类、破坏社会稳定的敏感词。请注意，一些通过谐音、缩写规避检测的词汇同样属于高风险范畴。

**原则**：我们旨在通过合法合规的营销获取收入，务必远离法律法规和平台规则明令禁止的内容。

### 3. 发布频率异常
短时间内高频次发布笔记，可能导致账号行为异常，被系统误判为**机器人或营销号**，从而触发限制，造成笔记不被收录。

### 4. 账号设备已被拉黑
这是许多用户容易忽略的一点。平台能够识别您使用的硬件设备。

**核心概念**：当您安装并登录小红书App时，应用有权读取您设备的某些硬件参数（如IMEI码、序列号等）。如果该设备曾有过严重违规记录，即使您卸载重装App或更换新账号，平台依然能识别并限制该设备发布的内容。

**公式**：`设备黑名单判定 = f(设备硬件参数， 历史违规记录)`

### 5. 内容质量过低或非原创
除了上述四点，内容本身质量过差、完全抄袭他人或毫无信息价值，也可能无法通过系统的初步审核，导致不被收录。
