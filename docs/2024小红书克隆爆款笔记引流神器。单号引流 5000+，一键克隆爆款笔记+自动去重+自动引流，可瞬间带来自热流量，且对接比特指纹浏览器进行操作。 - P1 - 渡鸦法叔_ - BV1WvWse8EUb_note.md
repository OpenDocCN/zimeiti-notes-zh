## 账号管理板块

软件主要分为两个板块：账号管理和自热功能。本节我们先介绍账号管理部分。

### 手动登录账号与CK获取

以下是手动登录账号并获取CK的步骤：

1.  点击“手动登录账号”。该功能会存储账号的Cookie，简称为CK。
2.  点击后，软件会自动弹出浏览器窗口。
3.  在弹出页面中，使用手机扫码登录小红书账号。
4.  登录完成后，点击“确定”，然后关闭浏览器窗口。
5.  此时，软件界面会成功提取并显示该账号的CK。

### 手动创建与管理浏览器窗口

除了自动登录，你也可以手动创建和管理浏览器窗口。

1.  点击“新建创建窗口”，手动创建一个浏览器实例。
2.  在该窗口中，可以手动配置账号的IP地址，并使用“一键随机生成指纹”功能。
3.  可以在此处填入购买的代理IP（例如SK5代理IP），然后点击“确定”。
4.  **重要**：记住你创建的窗口序列号。在后续发布笔记或克隆功能中，需要选择对应的窗口类型。

**请注意**：如果是手动创建的窗口，你需要手动打开它，并访问“创作者中心”登录账号CK。系统提供了两种登录方式：
*   方式一：在“手动登录账号”处填写信息，系统自动填入并登录。
*   方式二：对于已手动创建并登录过的窗口，后续使用时直接选择该浏览器窗口即可，无需重复登录。

### 窗口状态与信息更新

你需要定期管理浏览器窗口的状态。

1.  每天可以点击“刷新”查看窗口状态。
2.  如果“总数”或“打开数”已满，则无法发布新笔记。若已满，可以购买套餐扩容。
3.  点击“更新信息”，软件会提取并显示窗口的名称、备注、对应的平台账号等信息。

## 克隆笔记功能

上一节我们介绍了账号管理，本节我们来看看如何克隆爆款笔记。此功能已简化，无需手动提取CK。

以下是克隆笔记的操作流程：

1.  复制你想要克隆的爆款笔记作品地址。
2.  将地址粘贴到软件指定位置，点击“获取”。
3.  软件会自动获取笔记的标题、图片、内容。你可以手动删除多余的标签。
4.  配置发布参数：
    *   **启动线程数**：一个线程对应一个账号。例如，启动两个线程，则会同时使用两个账号发布。
    *   **间隔时间**：设置线程间发布间隔，例如2000毫秒（2秒）。
    *   **配置IP**：首次使用可让软件自动配置IP。若已在比特指纹浏览器中手动配置过IP，则此处选择“用浏览器”方式。
5.  选择发布窗口：从列表中选择已登录账号的浏览器窗口（例如窗口1和窗口5）。
6.  选择账号数量，需与启动线程数匹配。
7.  选择去重模式（如MD5模式），点击“开始发布”。

软件将自动打开选定的浏览器窗口并发布克隆的笔记。**请注意**：账号必须绑定手机号，否则发布过程会卡住。

## 自动发布原创笔记功能

克隆功能适合快速复制，而自动发布功能则能实现更灵活的原创内容组合发布。本节将重点讲解此功能。

你可以通过建立标题库、图片库、内容库和标签库，让软件自动排列组合，生成并发布去重后的笔记。

以下是配置自动发布的步骤：

1.  **准备素材库**：
    *   **标题库**：填入多个笔记标题。
    *   **图片库**：添加多张图片。你可以设置单张为一组，或多张为一组。
    *   **内容库**：填入多段笔记正文内容。
    *   **标签库**：填入多个标签，以英文逗号分隔。例如：`创业,大学生创业,球带`。每个空格分隔的短语视为一个标签组。
2.  **组合发布内容**：在软件界面，分别为一次发布任务选择标题、图片、内容和标签。例如，选择标题A、图片B、内容C和标签D，软件会将这些元素组合成一篇新笔记。
3.  **配置发布参数**：
    *   **关键规则**：`启动线程数`必须与`账号数量`成倍数关系。例如，有4个账号，可以启动2个线程（发完两个自动轮换到下两个），但不能启动3个线程。
    *   设置发布间隔（如2000毫秒）。
    *   选择已登录账号的浏览器窗口。
4.  点击“开始发布”，软件将自动执行发布任务。

发布后，你可以在“笔记管理”中查看状态（如“审核中”）。
