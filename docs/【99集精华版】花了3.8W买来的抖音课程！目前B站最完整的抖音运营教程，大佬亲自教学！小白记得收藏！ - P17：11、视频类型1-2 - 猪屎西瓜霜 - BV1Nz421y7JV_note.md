# 抖音运营教程：P17：11、视频类型1-2：人设与口播类型详解 🎬

在本节课中，我们将要学习抖音内容创作中两个核心概念：**人设**的建立与区分，以及**口播**这种视频类型的具体应用。理解这些是规划账号方向和制作有效内容的基础。

上一节我们解决了文案问题，本节中我们来看看内容创作的下一个关键环节。从涨粉到变现并非一条直线，中间有一个重要环节，即建立你的**人设**。

**人设**的作用并非为了炫耀，而是为了在众多同类创作者中脱颖而出。例如，有十个人都做装修内容，如果其中一位建立了幽默搞笑的人设，观众就更容易记住他，从而为后续的涨粉和变现奠定基础。

![](img/9b9b5a29ce88dce082f88850ec778743_1.png)

为了帮助大家理解和应用，我们需要先区分两种主要的人设类型：**强人设**与**弱人设**。

![](img/9b9b5a29ce88dce082f88850ec778743_3.png)

以下是两种人设的具体定义和特点：

*   **强人设**
    *   **定义**：以创作者**个人**为切入点的内容定位。
    *   **核心公式**：**内容主题 = 个人（经验/兴趣/生活）**
    *   **特点**：
        1.  内容与创作者自身高度相关，如分享行业经验（装修、法律、电商）、专业技能或兴趣爱好（美食、旅行、美妆）。
        2.  通常采用**真人出镜**的形式，以增强真实感和个人魅力。
    *   **示例**：一位厨师分享烹饪技巧；一位旅行博主记录旅途见闻。

*   **弱人设**
    *   **定义**：内容核心与创作者**个人**关联度较低，更侧重于内容本身。
    *   **核心公式**：**内容主题 ≠ 个人（或关联度很弱）**
    *   **特点**：
        1.  账号重心在于提供某一类内容，而非塑造个人形象。
        2.  通常不露脸或仅以声音、文字形式出现。
    *   **示例**：发布情感语录配音乐的视频账号；进行电影解说的账号（解说声音为工具，焦点是电影）。

区分清楚人设类型后，我们才能更好地规划内容。接下来，我们看看一种非常实用且与强人设紧密相关的视频形式：**口播**。

**口播**是一种通过语言讲述来传递信息的视频形式，是内容创作的重要工具。它的核心是“用嘴巴讲”，可以是讲故事，也可以是传授知识。

以下是口播的两种主要方向和拍摄要点：

*   **口播的内容方向**
    1.  **讲故事型口播**：分享客户故事、从业经历或人生感悟。目的是让观众觉得有趣、有共鸣。故事本身不要求完全真实，可以从文案平台寻找素材。
    2.  **知识分享型口播**：传授特定领域的知识，如育儿、金融、养生、装修、房产等。目的是在几十秒内让观众获得“收获感”，觉得学到了东西。

*   **关于口播的常见问题与解答**
    *   **问：口才不好能做口播吗？**
        *   **答：可以。** 口播是**提前准备好稿子**再录制，并非临场发挥。所有优秀的口播者都依赖于充分的准备。
    *   **问：普通话不标准能做口播吗？**
        *   **答：可以，有时效果更好。** 对于想做同城流量的实体店主，使用方言可能更亲切。有趣的方言本身也是亮点，且可以通过**配字幕**解决理解问题。
    *   **问：口播怎么拍？**
        *   **答：非常简单。** 用手机架好就能拍摄。关键在于找到或撰写好的文案稿。

本节课中我们一起学习了如何通过建立鲜明的**人设**（强人设/弱人设）来让账号被记住，并深入探讨了**口播**这一高效的内容形式及其应用技巧。理解这些概念，能帮助你更清晰地定位账号，并开始制作你的第一批视频内容。
