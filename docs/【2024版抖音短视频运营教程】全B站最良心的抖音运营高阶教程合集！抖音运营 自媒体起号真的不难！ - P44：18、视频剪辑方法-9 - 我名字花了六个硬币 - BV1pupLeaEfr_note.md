上一节我们介绍了视频剪辑前的准备工作，本节中我们来看看如何使用剪映App进行具体的剪辑操作。

![](img/d925e9014ebb2db826b6460aab83f3cb_9.png)

![](img/d925e9014ebb2db826b6460aab83f3cb_11.png)

![](img/d925e9014ebb2db826b6460aab83f3cb_13.png)

首先，请确保你下载的是**剪映（黑色图标版本）**。这款软件由抖音官方推出，有手机版和电脑版，本节我们先讲解手机版的操作。

![](img/d925e9014ebb2db826b6460aab83f3cb_15.png)

**第一步：导入与初步调整**

![](img/d925e9014ebb2db826b6460aab83f3cb_17.png)

打开剪映App，点击“开始创作”，然后选择你需要剪辑的视频素材并“添加”。

![](img/d925e9014ebb2db826b6460aab83f3cb_19.png)

如果你拍摄了多个生活片段（如开头、中间、结尾），建议**一次只导入一条视频进行剪辑**，剪好后再通过点击“+”号导入下一条，这样操作清晰，不易混乱。

![](img/d925e9014ebb2db826b6460aab83f3cb_21.png)

![](img/d925e9014ebb2db826b6460aab83f3cb_23.png)

添加视频后，第一步是检查并调整画面构图与比例。
1.  点击轨道上的视频片段。
2.  在底部工具栏中向右滑动，找到并点击“编辑”。
3.  在编辑菜单中，点击“旋转”将画面调整到合适的方向。
4.  如需裁剪画面比例，可点击“裁剪”，选择如1:1、4:3、16:9等比例，或手动调整。

![](img/d925e9014ebb2db826b6460aab83f3cb_25.png)

![](img/d925e9014ebb2db826b6460aab83f3cb_27.png)

调整完毕后，点击对勾确认，回到主编辑界面。

![](img/d925e9014ebb2db826b6460aab83f3cb_29.png)

![](img/d925e9014ebb2db826b6460aab83f3cb_31.png)

![](img/d925e9014ebb2db826b6460aab83f3cb_33.png)

**第二步：剪辑视频片段**

![](img/d925e9014ebb2db826b6460aab83f3cb_35.png)

![](img/d925e9014ebb2db826b6460aab83f3cb_37.png)

接下来，我们需要剪掉视频中无用的部分（如开场沉默、结尾冗余动作）。
1.  播放视频，找到需要裁剪的起始位置，将时间轴（白色竖线）移动到此。
2.  点击选中视频片段，在底部工具栏点击“分割”。视频即被分为两段。
3.  选中需要删除的片段（如前一段或后一段），点击“删除”。

![](img/d925e9014ebb2db826b6460aab83f3cb_39.png)

**核心操作可总结为：选中 -> 分割 -> 选中 -> 删除**。重复此步骤，即可流畅地去除所有不需要的片段。

![](img/d925e9014ebb2db826b6460aab83f3cb_41.png)

![](img/d925e9014ebb2db826b6460aab83f3cb_43.png)

**第三步：添加与编辑背景音乐**

![](img/d925e9014ebb2db826b6460aab83f3cb_45.png)

![](img/d925e9014ebb2db826b6460aab83f3cb_47.png)

视频剪辑好后，需要添加背景音乐。
1.  点击底部工具栏的“音频”。
2.  如果你已有准备好的配音文件（如从其他软件生成），可以点击“提取音乐”直接导入。
3.  如果没有，则点击“添加音乐”。App内提供了丰富的音乐库，你可以按分类（美食、美妆、萌宠等）查找，或直接搜索。点击歌曲旁边的播放按钮可以预览，选定后点击“使用”。

![](img/d925e9014ebb2db826b6460aab83f3cb_49.png)

![](img/d925e9014ebb2db826b6460aab83f3cb_51.png)

如果添加的音乐过长，剪辑方法与视频相同：将时间轴移到音乐轨道的目标位置，**选中音乐 -> 点击“分割” -> 选中多余部分 -> 点击“删除”**。

![](img/d925e9014ebb2db826b6460aab83f3cb_53.png)

你还可以调整音乐音量：选中音乐轨道，点击“音量”，拖动滑块即可调节。

![](img/d925e9014ebb2db826b6460aab83f3cb_55.png)

**第四步：添加与修改字幕**

![](img/d925e9014ebb2db826b6460aab83f3cb_57.png)

![](img/d925e9014ebb2db826b6460aab83f3cb_59.png)

为视频添加字幕可以提升观看体验，剪映的“识别字幕”功能能极大提升效率。
1.  点击底部工具栏的“文本”。
2.  选择“识别字幕”。
3.  在弹出的窗口中，建议选择“仅视频”，以避免识别到背景音乐里的歌词。然后点击“开始识别”。

![](img/d925e9014ebb2db826b6460aab83f3cb_61.png)

![](img/d925e9014ebb2db826b6460aab83f3cb_63.png)

系统会自动为视频中的人声生成字幕。识别完成后：
*   你可以拖动字幕条调整其出现的时间位置。
*   双击字幕条，可以修改错别字。
*   点击“编辑”，可以更改字体、颜色、样式等，让字幕更贴合视频风格。

**第五步：导出高清视频**

![](img/d925e9014ebb2db826b6460aab83f3cb_65.png)

![](img/d925e9014ebb2db826b6460aab83f3cb_67.png)

在导出前，需要进行关键设置以保证视频清晰度。
1.  点击右上角的“导出”按钮。
2.  在导出设置中，调整参数：
    *   **帧率 (FPS)**：设置为 **30**。这是抖音平台最合适的帧率。
    *   **分辨率**：尽可能选择最高选项（如1080p或4K）。分辨率越高，视频越清晰。
3.  设置完成后，点击“导出”。**导出过程中请勿锁屏或切换程序**，耐心等待导出完成。
4.  导出后，你可以直接通过剪映分享到抖音。

接下来，我们为制作“口播”类视频的同学推荐一个实用工具。

**口播提词工具推荐：爱提词**

![](img/d925e9014ebb2db826b6460aab83f3cb_83.png)

![](img/d925e9014ebb2db826b6460aab83f3cb_85.png)

对于需要面对镜头讲话的视频，背诵台词是一大难题。“爱提词”这款App可以完美解决这个问题。

![](img/d925e9014ebb2db826b6460aab83f3cb_87.png)

![](img/d925e9014ebb2db826b6460aab83f3cb_89.png)

以下是其主要功能和使用方法：
*   **添加台词**：打开App，点击右下角“+”号，将你的台词文本粘贴或输入进去，保存即可。
*   **提词拍摄**：保存后，点击“开始提词”，选择“录制模式”。镜头前就会悬浮显示你的台词，你可以一边看一边讲，无需背诵。
*   **自定义设置**：点击屏幕中央的“设置”按钮，可以调整台词字体的大小、滚动速度等，方便近视的同学观看。
*   **内置美颜**：在拍摄界面，左下角有“美颜”功能。你可以调整瘦脸、大眼、磨皮等效果，让拍摄效果更自然美观，不同于一些滤镜过重的美颜相机。
*   **文案灵感**：App内还提供“爆款文案”板块，涵盖职场、情感、教育、生活等多个领域，可以作为你创作内容的参考。

![](img/d925e9014ebb2db826b6460aab83f3cb_91.png)

将这个工具与剪映结合使用，能让你更高效、更专业地完成口播类视频的制作。
