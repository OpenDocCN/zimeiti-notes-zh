# 视频号运营教程：P21：7. 账号定位与素材获取 🎯

在本节课中，我们将要学习视频号账号定位后的两个关键执行步骤：如何处理旧作品以及如何获取高质量的原创视频素材。这两个步骤是账号启动初期必须解决的实际问题。

上一节我们介绍了账号定位的核心思路，本节中我们来看看如何为定位好的账号准备内容。

## 处理旧作品与发布频率 📝

有同学询问是否需要隐藏账号之前的作品。答案是肯定的，需要隐藏。这有助于保持账号内容与全新定位的一致性，给新粉丝留下专业的印象。

![](img/3772df919aeecad1fc2558d6a6bdaa51_1.png)

素材准备无需着急。发布频率应保持为**一天发一条**。稳定的更新频率比一次性发布大量内容更重要。

因此，我们的首要目标是先解决获取**100个粉丝**的基础问题，然后再系统性地解决素材问题。请按顺序逐个解决。

![](img/3772df919aeecad1fc2558d6a6bdaa51_3.png)

接下来，我们解决原创视频素材的问题。许多同学面临没有专业素材、不会拍摄、不会剪辑的困境。

*   不会剪辑视频。
*   不擅长制作视频内容。

如果你属于以上情况，这很正常。针对这一痛点，我们提供了一个重磅福利。
