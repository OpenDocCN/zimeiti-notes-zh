# 新媒体运营系统课：P17：剪映新功能 - AI视频制作

## 概述
在本节课中，我们将学习剪映软件的一项全新AI功能。这项功能可以帮助我们快速生成视频文案，并克隆自己的声音，从而实现用AI代替真人出镜制作视频。这对于不擅长拍摄、剪辑、口播或文案写作的初学者来说，是一个非常实用的工具。

### 第一步：导入视频素材
首先，打开剪映APP，点击“开始创作”。从相册中选择一段视频素材（例如一段花海视频），勾选后点击右下角的“高清添加”按钮。视频导入后，建议先关闭原声，为后续添加AI生成的音频做准备。

![](img/b464cc96591d72b0206762e7334653a8_11.png)

操作路径如下：
```
剪映 → 开始创作 → 选择视频 → 高清添加 → 关闭原声
```

### 第二步：使用“智能文案”生成脚本
接下来，我们需要为视频添加文案。以下是具体操作步骤。

在下方工具栏中找到并点击“文本”工具。在文本工具中，选择“智能文案”功能。该功能提供了多种文案类型，如讲解文案、营销文案等。

我们可以根据视频内容输入关键词。例如，视频背景是花海，我们就在输入框中键入“花海风景”。输入后，点击右侧的生成箭头，系统会快速生成一段匹配的文案。如果对生成的文案不满意，可以点击“下一个”重新生成。

找到满意的文案后，点击“确认”，并选择“添加至轨道”，文案就会出现在视频轨道上。

### 第三步：克隆个人音色进行配音
文案准备就绪后，下一步是为其配上声音。以下是克隆个人音色的步骤。

选中轨道上的文案片段，在下方工具栏中向左滑动，找到并点击“文本朗读”功能。在音色分类中，选择“我的”标签页下的“克隆音色”，点击加号开始创建。

系统会提示录制一段示例语音。请点击录制按钮，清晰朗读屏幕给出的示例句子（例如：“猫头鹰的耳朵是不对称的…”）。录制完成后点击暂停，系统将开始生成您的专属克隆音色。

生成成功后，可以先试听效果。如果满意，点击“保存音色”，然后点击右上角的对勾确认。之后，您就可以在“文本朗读”中选择这个克隆音色为文案配音了。

### 第四步：完成与导出
完成以上步骤后，一段由AI生成文案并搭配克隆人声的视频就制作完成了。这个流程解决了不会写稿、不会配音的难题。

最后，检查视频整体效果，进行必要的微调后，即可导出成品。

本节课我们一起学习了剪映的AI视频制作新功能。我们掌握了从导入素材、利用“智能文案”生成脚本，到克隆个人音色进行配音的完整流程。这项功能极大地降低了视频制作的门槛，让每个人都能更轻松地开始创作。
