# 抖音运营教程：P11：产品与用户兴趣标签

在本节课中，我们将学习如何运用“九宫格选题法”，结合产品属性与用户兴趣标签，为抖音账号（以健身软件Keep为例）挖掘源源不断的优质选题。掌握这个方法，可以有效解决“不知道拍什么”的创作瓶颈。

上一节我们探讨了选题的重要性，本节中我们来看看如何系统性地生成具体选题。

## 产品标签分析

首先，我们需要分析产品本身的核心特点。以Keep健身软件为例，将其置于九宫格中心，围绕它联想其产品属性。

以下是Keep可能具备的产品标签：

1.  **线上健身软件**：产品的核心形态。
2.  **专业健身教练**：平台提供的核心资源。
3.  **饮食搭配建议**：与健身配套的服务内容。
4.  **减肥/塑形攻略**：产品解决的主要用户需求。
5.  **健身干货知识**：提供的教育性内容。
6.  **科学运动方法**：产品的专业性和可信度支撑。
7.  **打卡运动制度**：产品的互动与激励设计。
8.  **一二线城市用户为主**：产品的主要用户画像，可通过百度指数、艾瑞网等工具查证。

将以上与产品直接相关的关键词填入九宫格后，这部分工作就完成了。

## 用户兴趣标签分析

接下来，我们分析使用该产品的用户可能还对哪些内容感兴趣。将核心用户（如“健身爱好者”或“减肥人群”）置于另一个九宫格的中心，进行发散联想。

以下是针对该用户群体可能延伸出的兴趣标签：

1.  **健身房**：线下健身场景的延伸。
2.  **健身装备**：如运动服饰、器材等相关产品。
3.  **轻脂美食/减肥餐**：与健康饮食强相关的内容。
4.  **旅行**：作为热爱生活的一种表现。
5.  **星座/明星**：大众化的娱乐兴趣点。
6.  **心灵鸡汤**：激励型内容，与“自律”“坚持”等健身精神契合。
7.  **时尚穿搭**：身材管理后的外在展示需求。
8.  **生活热爱者**：将用户画像从“健身者”扩大到更广泛的积极生活群体。

通过这种方式，我们完成了两个九宫格的填充：一个基于产品，一个基于用户兴趣。

## 运用“连连看”生成选题

填满九宫格后，关键在于如何将这些标签转化为具体的视频选题。方法是进行“连连看”，将产品标签与用户兴趣标签进行两两组合，碰撞出创意。

以下是几个组合示例及产生的选题思路：

*   **组合1：饮食搭配 + 旅行**
    *   **选题思路**：如何在外出旅行时享受美食又不发胖？

*   **组合2：运动方法 + 明星**
    *   **选题思路**：赵丽颖都在练的5个高效瘦身动作。

*   **组合3：科学 + 健身房**
    *   **选题思路**：在健身房必须知道的3个科学原理，避免无效锻炼。

当自己联想遇到瓶颈时，可以借助搜索引擎。例如，搜索“科学 健身”等关键词组合，查看相关的长尾词或热门话题，从中获取灵感。

![](img/8c38980a976c5999c59a100f2a53c296_3.png)

通过这种交叉联想的方法，理论上可以衍生出大量的选题方向，并以此建立你的视频素材库。

## 如何判断选题质量

然而，并非所有组合产生的选题都是好选题。一个好的选题通常需要具备以下特点：

*   **符合逻辑**：标签之间的连接要自然合理，不能生搬硬套。
*   **吸引目标用户**：选题需同时触及产品功能点和用户的真实兴趣或痛点。
*   **具备可操作性**：选题需要能够被拍摄成具体、直观的视频内容。

例如，“在健身房如何科学撩妹”这个选题，虽然结合了“健身房”和“社交”兴趣点，但与健身软件的核心价值关联度较弱，可能不是最优质的选择。

本节课中我们一起学习了“九宫格选题法”。我们首先分别梳理了产品自身的属性标签和核心用户的兴趣标签，然后通过将这两类标签进行创造性组合（“连连看”），来批量生成视频选题创意。这个方法能系统化地解决内容灵感枯竭的问题，帮助你的账号持续产出既贴合产品、又吸引用户的优质内容。
