# 小红书运营教程：P28：小红书爆款笔记写作方法（二）

在本节课中，我们将学习如何确立小红书笔记的内容框架。我们将分两个主要部分讲解：首先是内容确立的五个步骤，其次是一些关键的注意事项。通过本课，你将掌握一套可快速上手的笔记内容规划方法。

上一节我们介绍了爆款笔记的基础概念，本节中我们来看看如何系统地构建笔记内容。

## 内容确立的五个步骤

很多人在撰写小红书内容时，首先面临的问题是如何确立内容方向。以下是确立标题和笔记核心营销内容的五个步骤。

### 1. 确定运营领域
首先，你需要明确自己的运营领域。例如，你选择的是美妆、护肤品还是母婴类产品。在之前的课程中已多次强调，确定领域至关重要。

如果无法确定领域，会导致账号垂直度降低。账号垂直度降低会引发一系列问题，例如平台减少推荐、粉丝流失。

这会形成一个恶性循环：粉丝越来越不喜欢你的内容，你的内容就越来越难获得平台推荐。

### 2. 梳理行业关键词
确定领域后，需要梳理与该行业相关的关键词。在之前的课程中，已经教授了使用百度工具、爱站网、5118、百度指数等关键词工具进行查询的方法。

本节课不再赘述此部分。根据你的行业，可以梳理出许多关键词及其热度排行。

![](img/8308f7acd460534d556a38d616ddbe31_1.png)

**核心操作**：利用关键词工具查询并整理出行业关键词排行列表。用好这些关键词能极大提升内容效果。

![](img/8308f7acd460534d556a38d616ddbe31_3.png)

### 3. 搜集同行案例
第三步是搜集同行的优秀案例。有人可能会问，在小红书内搜索时看不到同行的具体数据，该怎么办？

![](img/8308f7acd460534d556a38d616ddbe31_5.png)

解决方法其实很简单。今天推荐一个工具：**牵瓜数据**。这是一个长期使用且非常好用的数据分析平台。

![](img/8308f7acd460534d556a38d616ddbe31_7.png)

该数据平台的优势在于，它提供小红书的数据分析功能，例如可以分析并匹配优质达人、追踪热门数据、查看品牌方投放情况等。

![](img/8308f7acd460534d556a38d616ddbe31_9.png)

平台内还包含各类榜单。那么，具体如何利用牵瓜数据筛选内容呢？下面我们来详细说明。

### 4. 筛选与整理内容
搜集好同行案例后，就可以对这些案例进行内容筛选。筛选完成后，将内容整理成自己的版本，然后进行发布。

这里我们用牵瓜数据来演示。在平台左侧栏目中，可以看到许多功能，包括热词排行榜、热词搜索榜、笔记收录查询等。

本节课主要讲解“笔记收录查询”功能。例如，发布笔记后，你可能担心笔记是否被官方收录，就可以利用此功能进行查询。

关于热门内容和热词排行榜，此处不再展示。另一个主要功能是“达人搜索”。

达人搜索可以按昵称或笔记内容进行搜索。例如搜索“美妆”（此功能需付费），或按昵称搜索，效果类似。

按昵称搜索会显示相关达人的属性，例如做彩妆、护肤、母婴、美式饮品或服饰穿搭等。这些分类属性可根据自身需求和数据进行筛选。

平台还会显示达人属性，如是否为官方号、品牌号或头部达人。可以看到许多相关数据，包括达人对比、排名靠前的达人等。

如果需要使用此网站，可根据实际需求在淘宝上寻找开通渠道，费用通常比较划算。

拥有此功能后，例如点击某个达人的详情页，就能看到他发布的数据、笔记分类、笔记详情以及笔记的详细类型。

![](img/8308f7acd460534d556a38d616ddbe31_11.png)

例如，可以看到笔记评论的词云图，这些词语的出现频率。用好这个工具，能有效帮助我们掌握小红书的数据参数。

### 5. 整理成文与发布
说完了牵瓜数据的使用，最后我们说一下内容整理方面。在之前的课程中已经讲过，主要采用“总-分-总”的结构来撰写内容。

这种内容结构并不难写。如果觉得困难，建议回顾之前的课程内容。本节课不再重复这一点。

## 其他注意事项

以下是撰写笔记时需要注意的其他事项。

1.  **控制标题字数**：标题字数尽量控制在 **12到18个字** 之内。
2.  **使用符号美化**：标题中应多带一些符号（如✨、❤️、🔥）进行美化。众所周知，小红书女性用户居多，使用美化符号能更好地吸引她们的注意力。
3.  **加入具体数据**：可以在标题中加入具体数据以增加真实感。例如，像上节课截图中使用的“百万”这类数字，或其它详细数字，会让人感觉更可信。因此，在拟定标题文字时，可以加入数字。

本节课中，我们一起学习了确立小红书笔记内容的五个步骤：确定领域、梳理关键词、搜集同行案例、筛选整理内容以及最终成文发布。同时，也掌握了标题撰写的三个关键注意事项：控制字数、美化符号和加入数据。运用这些方法，你可以更有条理地规划并撰写出更具吸引力的笔记内容。
