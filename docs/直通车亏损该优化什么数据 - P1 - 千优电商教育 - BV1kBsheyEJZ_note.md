# 直通车优化指南：P1：亏损时优先优化的核心数据 💡

在本节课中，我们将学习当直通车广告持续亏损时，应该优先从哪个数据维度入手进行优化。我们将深入理解销售额与投产比的核心公式，并明确转化率在其中的关键作用。

## 核心公式解析

上一段我们提到了直通车亏损的普遍现象。要解决这个问题，首先需要理解几个核心的电商公式。这些公式是分析问题的基石。

销售额、转化率、客单价和投产比之间存在明确的数学关系。理解这些关系是进行有效优化的第一步。

以下是定义这些关系的核心公式：

*   **销售额公式**：`销售额 = 转化率 × 订单量 × 客单价`
*   **投产比公式**：`投产比 = 销售额 / 花费`

## 为何优先优化转化率

在理解了基础公式后，我们来看看哪个因素对扭转亏损最为关键。当直通车持续亏损时，最应该优先考虑优化的是**转化率**。

转化率的提升能直接且显著地影响销售额和订单量。我们可以通过一个简单的例子来说明。

![](img/03af77fc659c4375b90b34226eed91a3_1.png)

假设你每天在直通车上的花费固定为100元。当你的转化率为1%时，可能每天产生1到2个订单。如果你成功将转化率优化至2%，在同等花费下，你的订单量将实现翻倍。订单量的增加直接带来了销售额的增长，从而有助于提升投产比，扭转亏损局面。

## 如何优化转化率

我们已经明确了优化转化率的重要性，那么具体应该从哪些方面入手呢？许多商家知道优化基础销量和评价，但之后便无从下手。实际上，下一步的关键在于**SKU转化的引导**。

SKU的设置对于引导用户下单至关重要。有效的SKU策略不仅能提升转化率，还能连带提高客单价。

以下是两个优化SKU引导的具体方向：

![](img/03af77fc659c4375b90b34226eed91a3_3.png)

1.  **设置合理的SKU**：通过清晰的规格、属性或套餐组合，降低用户的选择成本，直接引导其下单。
2.  **利用SKU提升客单价**：如果你店铺的商品以低客单价为主，可以尝试在SKU中添加几个高客单价选项。这不仅能满足不同用户的需求，还能有效引导部分用户选择更高价值的商品，从而同时提升转化率和客单价。

![](img/03af77fc659c4375b90b34226eed91a3_4.png)

本节课中我们一起学习了直通车亏损时的核心优化思路。我们首先通过公式 `销售额 = 转化率 × 订单量 × 客单价` 和 `投产比 = 销售额 / 花费` 明确了各数据之间的关系。接着，我们指出**转化率**是亏损时应优先优化的关键，因为它能直接倍增订单与销售额。最后，我们探讨了超越基础销量评价的优化方法，即通过**优化SKU设置与引导**来提升转化率与客单价。记住，优化是一个持续的过程，从最关键的数据入手才能事半功倍。
