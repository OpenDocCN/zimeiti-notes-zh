## 概述

在本节课中，我们将深入探讨巨量千川的实操投放逻辑。我们将从投放的核心思路讲起，明确不同阶段的计划组合策略，并详细讲解新老账号如何起量、如何筛选和优化计划。无论你是新手还是有经验的投手，本节课的内容都将帮助你理清思路，掌握高效投放的方法。

## 投放逻辑：精准与泛流量的动态平衡

上一节我们详细讲解了专业版页面的搭建。本节中，我们来看看投放的核心逻辑。很多账户投了不少钱却起不了量，往往是因为计划组合出了问题。

投放的核心逻辑在于**精准流量**与**泛流量**的动态搭配与转换。

*   **精准流量计划**：例如徕卡（行为兴趣）、达人定向。这类计划人群精准，但放量能力可能有限。
*   **泛流量计划**：例如**极速智能推荐**、**专业系统智能推荐**、**成本稳投**等。这类计划依赖系统模型，在账户有数据积累后是高效的跑量计划。

具体的投放阶段策略如下：

1.  **起号初期（新号/数据积累期）**
    *   **策略**：**大量精准计划 + 少量泛流量计划**。
    *   **比例参考**：约70%-80%的预算用于精准计划（徕卡、达人），20%-30%用于测试泛流量计划（如极速智能推荐）。
    *   **目的**：通过精准人群进入直播间形成初始成交画像，为系统积累数据，同时测试泛流量计划的潜力。

2.  **账号稳定期（有稳定成交数据后）**
    *   **策略**：**大量泛流量计划 + 少量精准计划**。
    *   **计划类型**：主力投放**极速智能推荐**、**成本稳投（ROI/成交）**、**专业系统智能推荐**等。
    *   **目的**：利用系统模型，放大流量规模，获取最优投产。此时精准计划（如徕卡）主要用于校正或辅助拉新。

**核心公式**：
> **起号期：精准流量(70-80%) + 泛流量(20-30%)**
> **稳定期：泛流量(主力) + 精准流量(辅助)**

简单来说，前期靠“人找货”（定向精准人群），后期靠“货找人”（系统推荐泛流量）。通投（不限任何定向）在前期慎用，容易跑偏人群；稳定后可作为跑量计划的补充进行测试。

## 新号/老号通用起量方法：赛马机制

理解了投放逻辑后，一个常见的问题是：新号到底该怎么投？老号又该怎么优化？其实，无论新老账号，起量的核心方法是一致的：**搭建多类型计划进行“赛马”**。

以下是经过大量账户验证的高潜力计划类型，你可以将它们视为“种子计划”：

1.  极速版 - 达人定向
2.  极速版 - 基础定向（地域/性别/年龄）
3.  专业版 - 徕卡（行为兴趣）
4.  专业版 - 达人定向
5.  专业版 - 系统智能推荐
6.  专业版 - 人群包（可作为备选或后期测试）

**操作步骤**：

1.  **搭建计划**：为新账号（或需要重新起量的老账号）同时搭建以上6种类型的计划。老账号可以额外加入 **成本稳投（成交/ROI）** 计划。
2.  **统一出价**：所有计划采用控成本出价，并设置相同或相近的出价，确保公平竞争。
3.  **放量测试**：让这些计划同时跑量，测试周期建议为3-7天。
4.  **数据分析与筛选**：周期结束后，在千川后台按“消耗”从高到低排序。观察消耗高、投产也好的计划属于哪几种类型。
5.  **确定主力计划**：筛选出的优势计划类型，就是你当前账户应该主力投放的方向。

**这个方法的本质是**：市场上有潜力的计划类型就这些，我们通过“赛马”让账户自己告诉我们，它更适合哪一种。这比盲目猜测或只投单一类型计划要科学高效得多。

## 计划优化与调整：复制与微调

当我们通过赛马机制找到了主力计划后，接下来的工作就是对这些“好计划”进行优化和放大。

**核心操作：复制与变量测试**。

找到一条跑量好、投产高的计划（例如：一条“极速智能推荐-成交”计划），我们可以通过复制它并修改一个变量来探索更多可能性：

*   **变量1：优化目标**：复制原计划，将优化目标从“成交”改为“下单”或“ROI”。
    *   `原计划：极速智能推荐 -> 成交`
    *   `新计划A：极速智能推荐 -> 下单`
    *   `新计划B：极速智能推荐 -> ROI`
*   **变量2：投放方式**：在“控成本”和“放量”之间切换测试。
*   **变量3：出价/ROI目标值**：小幅调整出价（如±10%）或ROI目标值，观察跑量速度和投产变化。
*   **变量4：创意分类/标签**：微调创意分类，看是否能有更好的流量匹配。
*   **变量5：创意形式**：如果是直投直播间，可以复制计划后更换投放的短视频素材进行测试。

**计划观察与关停标准**：

*   **如何判断计划好坏**？
    *   **高点击率 + 高转化率**：优质计划，全力投放。
    *   **高点击率 + 低转化率**：可能流量较泛，观察千次展示费用是否过低，可尝试复制调整人群或关停。
    *   **低点击率 + 高转化率**：流量精准但吸引力可能不足，给予一定预算观察，随着成交数据积累，点击率可能回升。
*   **何时关停计划**？
    *   建立一个简单的关停标准：**消耗金额达到客单价的1/5到1/3仍无转化时，考虑关停**。
    *   *例如*：客单价200元，计划消耗40-70元仍无成交，就可以暂停观察。关停后等待20分钟，如果ROI无回升，则放弃该计划。
    *   高客单价（如3000元）产品，关停阈值相应提高（如500-700元）。

**出价与ROI目标调整技巧**：

*   **控成本出价计划**：计划消耗过快时，每次以5%-10%的幅度降价。如果消耗速度极快，为防止预算秒空，可直接关停计划，大幅降价（如30%-50%）后再开启。
*   **ROI目标计划**：原理与出价相反。ROI目标值设置得越低（如设置8，期望系统跑出10），计划跑量越快；设置越高（如设置12），跑量越慢。根据跑量速度需求，调整ROI目标值。

## 总结

本节课我们一起学习了巨量千川从理论到实操的核心投放逻辑：

1.  **投放逻辑**：明确了“前期精准为主，后期泛流为主”的动态平衡策略，这是制定一切投放计划的基础。
2.  **起量方法**：掌握了通过搭建多类型“种子计划”进行赛马，让数据告诉我们账户最适合的投放方向，这是解决“不知从何投起”的关键。
3.  **优化技巧**：学会了如何对优质计划进行复制和变量测试（优化目标、出价等），以及如何根据点击率、转化率和消耗情况判断计划的去留，并给出了具体的关停参考标准。

记住，千川投放是“锦上添花”，核心永远是“人货场”。但掌握正确的投放逻辑和方法，能确保你买的流量更精准、更高效。建议有经验的投手反复理解投放逻辑部分，而新手朋友可以按照“赛马”步骤一步步操作，逐步优化。
