1.  **角色**
    角色取决于你选择的赛道和内容方向。你需要让别人知道你提供的内容具有专业性。
    *   **公式：角色 = 赛道专家身份**
    *   **示例**：
        *   **装修博主**：展示自己真实的房子、装修过程、避坑经验和最终成果对比。
        *   **母婴赛道**：可以是分享亲身经历的**宝妈**，也可以是提供专业知识的**育儿护理专家**。
        *   **情感赛道**：可以是提供建议的**心理学顾问**，也可以是分享自身蜕变故事的**成长型人设**（例如：一位分享职场、创业心路历程的女性）。

2.  **内容**
    内容需要遵循我们之前讲过的定位原则：吸引用户、提供价值、具有平台特色。在人设定位中，内容必须与你选择的赛道高度相关。
    *   **核心要求**：内容垂直、持续。
    *   **原因**：这关系到账号的垂直度和标签。平台会根据你的内容标签进行流量分发。如果内容杂乱（例如第一篇讲教育，第二篇讲宠物），会干扰算法判断，导致推荐不精准，从而影响流量效果。
    *   **行动指南**：确定赛道后，持续提供该领域的相关内容。

3.  **形式**
    形式指的是你呈现内容的具体方式，它由多个维度构成。
    *   **构成维度**：
        *   **产出场景**：室内固定场景、室外多变场景、车内、特定地点等。
        *   **声音呈现**：本人原声出镜、使用配音（可通过`剪映`等工具实现）、多人对话配音等。
        *   **内容类型**：剧情演绎型、干货分享型、生活记录型等。每种类型各有优劣（如干货型需强文案，记录型需多素材）。

4.  **记忆点** 🧠
    记忆点是让用户记住你的关键。它需要通过固定的设计，反复刻入用户心智。
    *   **常见记忆点设计**：
        *   **视频开头前置**：固定的开场动作、特效、口头禅。这些元素在视频前几秒出现，能形成强烈的重复冲击，加深印象。
        *   **贯穿全程**：统一的背景音乐（BGM）、始终出现的标志性物品、风格一致的服装等。这些元素在整个视频中持续存在，强化整体氛围和识别度。
    *   **设计逻辑**：将强化瞬时印象的元素（如动作、口号）放在开头；将塑造整体氛围的元素（如BGM、视觉风格）贯穿全程。
