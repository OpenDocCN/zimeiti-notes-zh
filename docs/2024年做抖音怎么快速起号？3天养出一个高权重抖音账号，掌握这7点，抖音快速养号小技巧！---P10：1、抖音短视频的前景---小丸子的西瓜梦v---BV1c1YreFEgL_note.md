# 抖音运营入门：P10：1、抖音短视频的前景

在本节课中，我们将要学习抖音短视频平台的发展前景，了解其市场规模、用户构成以及我们作为创作者的机会所在。这将帮助我们理解为何选择抖音作为内容创作的起点。

## 平台规模与创作者收入

![](img/e3d9036c4250c8097e587ec688dc23b6_3.png)

上一节我们了解了课程主题，本节中我们来看看抖音平台的具体数据。通过一张图片可以直观了解抖音的潜力。

![](img/e3d9036c4250c8097e587ec688dc23b6_1.png)

这张图片源自抖音创作者大会的数据报告。报告显示，有超过2200万人在抖音获得的总收入超过了417亿元。抖音平台希望将这个数字翻倍，目标是让创作者的总收入达到800亿元。

## 抖音的用户分类

了解平台规模后，我们需要认识平台上的两类主要用户。抖音用户主要分为两类。

以下是具体的分类说明：
*   **玩家**：这类用户单纯在抖音上娱乐，没有计划通过抖音赚钱。
*   **创作者**：这类用户主动制作并发布抖音内容，旨在获取收益。本课程的学员都属于创作者范畴。

## 选择抖音的原因

明确了自身定位后，我们来看看为何选择抖音而非其他短视频平台。选择平台类似于实体店铺选址，需要寻找流量大、适合自身发展的环境。

抖音平台具有以下优势：
1.  **流量巨大**：抖音每日活跃用户超过8亿，月活跃用户更是达到40亿人次。这为内容传播提供了广阔的土壤。
2.  **竞争压力相对较小**：平台上的创作者数量约为5000万。创作者的定义广泛，任何发布原创内容的用户都算创作者，因此大部分竞争来自普通用户。
3.  **入门门槛低**：无需专业设备。只需要一部智能手机和一个手机号即可注册抖音账号。抖音倡导全民通过短视频分享生活。

## 当前风口与创作方向

了解了平台优势，我们来看看当前有哪些机会。抖音平台推出了“抖音计划”，旨在扶持优质创作者。

对于缺乏创作经验的新手，无需过度担忧。学习是从不会到会的过程。保持谦虚好学的态度是快速成长的关键。

在大数据时代，平台会根据用户喜好推荐内容。这意味着无论你分享什么领域的内容，只要内容合法合规、积极向上，总会吸引到喜欢它的观众。

以下是适合新手尝试的创作方向举例：
*   **宝妈**：可以分享育儿日常或家庭美食。
*   **实体店主**：可以展示产品特色或店铺日常，用于线下引流。
*   **上班族**：可以记录日常生活或职场心得。

## 初期目标设定

最后，我们来设定合理的初期目标。许多新手急于涨粉，但更有效的策略是优先提升内容流量。

核心逻辑是：**播放量 → 曝光度 → 潜在粉丝关注**。因此，应将首要目标定为提高视频的播放量，粉丝增长会随之而来。

本节课中我们一起学习了抖音短视频的广阔前景、平台用户分类、选择抖音的理由、当前的创作风口以及合理的初期目标设定。理解这些基础，能帮助我们在抖音创作道路上建立正确的认知和起点。
