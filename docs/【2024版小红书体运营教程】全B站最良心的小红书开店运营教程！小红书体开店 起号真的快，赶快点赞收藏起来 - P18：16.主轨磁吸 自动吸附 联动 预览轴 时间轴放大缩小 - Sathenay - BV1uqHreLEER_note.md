# 小红书运营教程：P18：16. 主轨磁吸、自动吸附、联动、预览轴与时间轴缩放 📏

![](img/3066a5042964ea1ef746635247cb0239_0.png)

在本节课中，我们将学习视频剪辑软件时间轴界面上的几个核心辅助功能：主轨磁吸、自动吸附、联动、预览轴以及时间轴的缩放操作。这些功能能极大提升剪辑的效率和精确度。

时间轴右上方有**主轨磁吸**、**自动吸附**、**联动**、**预览轴**的开关按钮，以及对当前时间轴视图进行**放大或缩小**的控制。在默认情况下，前四个功能通常都是开启状态。将它们关闭后，可以更清晰地理解各自的作用。

## 主轨磁吸功能 🧲

上一节我们介绍了功能面板的位置，本节中我们来看看第一个功能：主轨磁吸。首先需要明确什么是主轨道。当我们进入编辑界面后，拖入时间轴的第一个素材轨道就是主轨道，它会默认显示封面。其他轨道则没有封面显示。

![](img/3066a5042964ea1ef746635247cb0239_2.png)

![](img/3066a5042964ea1ef746635247cb0239_4.png)

主轨磁吸功能，就是让素材在靠近主轨道时，像被磁铁吸住一样自动对齐。

以下是操作演示：
*   关闭主轨磁吸时，可以任意拖拽素材，起始位置不会自动对齐。
*   打开主轨磁吸后，拖拽素材时，其起始端会**自动吸附对齐**到主轨道素材的边缘或时间点。

![](img/3066a5042964ea1ef746635247cb0239_6.png)

![](img/3066a5042964ea1ef746635247cb0239_8.png)

## 自动吸附功能 🔗

![](img/3066a5042964ea1ef746635247cb0239_10.png)

接下来我们了解自动吸附功能。这个功能主要作用于同一轨道或不同轨道上素材之间的对齐。

![](img/3066a5042964ea1ef746635247cb0239_12.png)

以下是操作步骤：
1.  选中一个素材并将其分割。
2.  将分割后的部分拖拽到其他位置（例如上方轨道）。
3.  当**自动吸附关闭**时，将素材拖回原位置需要手动精确对齐，比较困难。
4.  当**自动吸附开启**后，将素材拖近目标位置时，它会**自动吸附并精准对齐**到其他素材的边缘。

## 联动功能 ⛓️

![](img/3066a5042964ea1ef746635247cb0239_14.png)

现在，我们来看看联动功能。联动功能用于建立不同轨道上素材之间的链接关系，让它们可以同步移动。

![](img/3066a5042964ea1ef746635247cb0239_16.png)

以下是操作步骤：
1.  在主轨道上添加一个文字或其他类型的轨道。
2.  将主轨道上的某个素材分割。
3.  当**联动关闭**时，可以单独拖拽主轨道上的素材片段，文字轨道不会随之移动。
4.  当**联动开启**后，拖拽主轨道上被链接的素材片段时，**与之关联的文字轨道会同步移动**，保持相对位置不变。

![](img/3066a5042964ea1ef746635247cb0239_18.png)

![](img/3066a5042964ea1ef746635247cb0239_20.png)

## 预览轴功能 👁️

![](img/3066a5042964ea1ef746635247cb0239_22.png)

预览轴功能可以方便我们快速浏览视频内容，而无需拖动播放头。

![](img/3066a5042964ea1ef746635247cb0239_24.png)

以下是功能说明：
*   当**预览轴关闭**时，要预览视频画面，必须移动时间轴上的播放头，才能在播放面板中查看。
*   当**预览轴开启**后，只需将鼠标指针在时间轴区域**水平移动**，指针处会出现一条黄色竖线。同时，播放面板会**实时显示**鼠标指针所在时间点的视频画面，实现快速预览。

![](img/3066a5042964ea1ef746635247cb0239_26.png)

![](img/3066a5042964ea1ef746635247cb0239_28.png)

建议在日常剪辑时，将以上四项功能保持默认开启状态，这能极大提升工作效率。如果遇到操作不顺手的情况，可以检查是否误关了这些功能。

![](img/3066a5042964ea1ef746635247cb0239_30.png)

## 时间轴缩放功能 🔍

![](img/3066a5042964ea1ef746635247cb0239_32.png)

最后，我们来学习时间轴的缩放操作。这个功能非常常用，可以让我们更精细地查看和编辑时间轴。

![](img/3066a5042964ea1ef746635247cb0239_34.png)

![](img/3066a5042964ea1ef746635247cb0239_36.png)

以下是操作方法：
*   可以点击时间轴右下角的“+”和“-”按钮，或者拖动滑块来放大或缩小时间轴视图。
*   更常用的方法是使用快捷键：**按住 `Ctrl` 键并滚动鼠标滚轮**，即可快速缩放时间轴视图。放大视图有助于更精确地定位时间点和帧。

![](img/3066a5042964ea1ef746635247cb0239_38.png)

## 总结 📝

本节课中我们一起学习了时间轴上的五个重要辅助功能：
1.  **主轨磁吸**：使素材自动对齐主轨道。
2.  **自动吸附**：使素材之间自动精准对齐。
3.  **联动**：让不同轨道的素材链接并同步移动。
4.  **预览轴**：通过鼠标移动实时预览视频画面。
5.  **时间轴缩放**：通过按钮或 `Ctrl` + 鼠标滚轮 放大缩小视图，便于精细编辑。

![](img/3066a5042964ea1ef746635247cb0239_40.png)

![](img/3066a5042964ea1ef746635247cb0239_41.png)

熟练掌握这些功能，能让你的视频剪辑过程更加流畅和高效。
