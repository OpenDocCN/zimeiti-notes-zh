# 视频号创作分成计划：P13：12.客户反馈 - 短视频变现教程 📈

![](img/5cae11f3777040a35584097c666f5ddf_1.png)

在本节课中，我们将通过一个真实学员案例，详细拆解如何通过坚持发布视频，在视频号创作分成计划中稳定获得收益。我们将重点关注执行策略、收益计算以及导师的陪跑方法。

---

上一节我们介绍了视频发布的基本策略，本节中我们来看看如何通过具体的执行和反馈来获得稳定收益。

有同学提出疑问，为什么别人发一条视频，而我们需要发三条。

![](img/5cae11f3777040a35584097c666f5ddf_1.png)

一条视频的收益是80元，三条视频的收益是240元。多挣160元是更优的选择。

关键在于通过增加发布数量来提升总收益。计算公式为：
**总收益 = 单条视频收益 × 发布条数**

我们的学员周建国先生63岁。对于年纪较大的学员，一遍学会的可能性较低。

教学方法的核心是“勤能补拙”。别人一天发一遍，我们可以一天发三遍。一天多练两遍，一周就是十四遍，半个月是三十遍，一个月就是六十遍。通过大量重复练习，任何人都能学会。

![](img/5cae11f3777040a35584097c666f5ddf_3.png)

我采取全程陪跑的教学方式。学员发布视频后，我会及时检查。做对了给予鼓励，做错了立即指导调整，以加深印象。

我要求学员不要与他人比较，只与过去的自己比较。例如，第一次发布花了20分钟，第二次只花了19分59秒，这就是进步。

他发布视频后，我告诉他第二天就会有收益。到了第二天早上8点47分，他发来消息确认收到了240元。

![](img/5cae11f3777040a35584097c666f5ddf_3.png)

![](img/5cae11f3777040a35584097c666f5ddf_5.png)

我通常在早上8点半前送儿子去幼儿园，周末会带他跑步。我在早上8点47分能及时回复学员，这种责任感源于我的军人背景和多重身份。

对于父母，我是子女；对于妻子，我是丈夫；对于孩子，我是父亲；对于公司，我是老板；对于学员，我是老师。做好每一个身份该做的事，就是我所理解的责任。

当时他发完视频后，我让他把收益截图发给我查看。

收益确实是240元。这是因为发布了三条视频，每条80元，所以总收益为 `80 * 3 = 240` 元。

周叔在提现后，第一时间将钱从视频号提到了微信。他告诉我，因为过去有投资无法提现的经历，所以只有钱真正到账才安心。我向他保证我们平台的可靠性。

我鼓励他坚持发布，肯定能拿到保底的80元收益。随后，我将第二天的三个素材（编号4、5、6）发给了他。

我提醒他，视频不要连续发布，最好分在中午11点、下午4点、晚上8点30分这三个流量高峰期发布。这样不仅能拿保底收益，还能冲击更高的流量提成。

平台给予的补贴政策并非永久不变。目前处于红利期，有保底收益。但我们需要未雨绸缪，在保底收益存在时，积极准备，提升内容以获取提成。例如，去年一个身份证可绑定5个视频号，现在只能绑定2个，这说明规则在变化。
