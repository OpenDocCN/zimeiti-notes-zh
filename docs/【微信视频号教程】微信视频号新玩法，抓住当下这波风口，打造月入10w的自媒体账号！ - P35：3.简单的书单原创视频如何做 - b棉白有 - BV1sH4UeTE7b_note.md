# 微信视频号教程：P35：简单的书单原创视频制作方法

## 概述
在本节课中，我们将学习如何使用剪映工具，制作一个简单的书单原创视频。整个过程分为几个清晰的步骤，从导入素材到添加文字、音乐和动画，即使是初学者也能轻松跟上。

---

### 1. 导入书单背景素材
首先，打开剪映App，点击“开始创作”。在创作界面，点击“素材库”。在素材库页面的搜索框（放大镜图标）中输入“书单”并搜索。从搜索结果中选择一个你喜欢的书单背景素材，点击右上角选中，然后点击右下角的“添加”按钮，将其导入到时间轴。导入后，记得点击视频轨道左侧的“关闭原声”按钮，关闭背景素材自带的音效。

### 2. 添加并设置文案文本
接下来，我们需要为视频添加文字内容。在下方主工具栏中找到并点击“文本”，然后选择“新建文本”。将你事先准备好的文案粘贴到文本输入框中。

以下是设置文本样式的步骤：
*   **选择字体**：点击字体选项，在列表中选择你喜欢的字体，例如在“书法”分类中选择“位杯体”。
*   **调整颜色**：点击“样式”，在颜色选项中为文本选择一个颜色，例如黄色或白色。
*   **调整排版**：继续在“样式”中找到“排列”，向下滑动找到“行间距”。按住滑块将其调整到数值“4”左右，使行距更舒适。
*   **确认并放置**：点击对勾确认设置。然后，用双指在屏幕上捏合或拖动，调整文本的大小和角度，并将其拖放到画面中合适的位置。
*   **调整时长**：在时间轴上选中这段文本，按住其末尾的白色边框向右拖动，使其时长与主视频背景的时长完全对齐。

### 3. 添加并设置标题文本
现在，我们来为视频添加一个标题。再次返回主工具栏，点击“文本” -> “新建文本”。输入你的标题文字。

以下是设置标题的步骤：
*   **设置颜色**：点击“样式”，为标题选择一个醒目的颜色，例如黄色。
*   **调整位置与大小**：将标题拖放到合适的位置（如顶部），并用双指将其放大到合适的尺寸。
*   **确认并调整时长**：点击对勾确认。同样，在时间轴上拖动标题片段的白色边框，使其时长与主视频对齐。

### 4. 添加画中画人物素材
为了让视频更生动，我们可以添加一个人物素材。返回主工具栏，点击“画中画”，然后选择“新增画中画”。从相册中选择一段你准备好的视频素材（例如人物讲话的视频）并添加。

以下是处理画中画素材的步骤：
*   **应用圆形蒙版**：选中画中画素材，在下方二级工具栏中向后滑动，找到并点击“蒙版”。选择“圆形”蒙版。
*   **调整蒙版**：屏幕上会出现一个圆形蒙版。用双指将其放大，并向上拖动，使人物头部位于圆形中央。然后，拖动下方的羽化箭头，为圆形边缘添加一些柔和的羽化效果。点击对勾确认。
*   **调整位置与大小**：用双指缩小画中画素材，并调整其角度，然后将其拖放到画面左上角或其他合适位置。
*   **关闭原声与裁剪**：由于添加的视频可能有声音，需要选中它，在二级工具栏点击“音量”，将音量滑块拖到最左侧（0），然后点击对勾。最后，将时间线移动到主视频的结尾处，选中画中画素材，点击“分割”，删除结尾多余的部分。

### 5. 为文案添加动画效果
接下来，我们为主要的文案文本添加一个入场动画，让它出现得更生动。在时间轴上选中之前添加的文案文本片段，在二级工具栏中找到并点击“动画”。在“入场动画”列表中，选择一个动画效果，例如“打字机”。将动画时长滑块拖到最右侧，让动画速度与视频匹配，然后点击对勾确认。

### 6. 添加并裁剪背景音乐
最后一步是为视频配上背景音乐。返回主工具栏，点击“音频”，然后选择“音乐”。在音乐库的搜索框中输入你想要的歌曲名，例如“老男孩”，点击搜索。从结果中选择一个版本，点击右侧的“使用”按钮添加。

以下是处理音乐的步骤：
*   **对齐音频**：在时间轴上，按住新添加的音频片段，将其向前拖动，使其开头与视频开头对齐。
*   **裁剪多余部分**：将时间线移动到视频的结尾处。选中音频片段，点击“分割”，然后删除视频结尾之后多余的音频部分。

---

## 总结
本节课我们一起学习了制作一个简单书单视频的完整流程。我们掌握了如何导入背景素材、添加并美化文案与标题、使用画中画和蒙版功能添加人物素材、为文字添加动画，以及最后为视频匹配合适的背景音乐。按照这些步骤，你就能快速制作出自己的原创书单视频了。
