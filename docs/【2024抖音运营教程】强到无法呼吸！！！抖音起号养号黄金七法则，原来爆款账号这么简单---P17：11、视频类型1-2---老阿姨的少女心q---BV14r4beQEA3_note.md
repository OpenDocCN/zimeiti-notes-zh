# 抖音运营教程：P17：11、视频类型1-2

在本节课中，我们将要学习抖音账号运营中一个至关重要的概念——**人设**。我们将深入探讨强人设与弱人设的区别，并详细介绍一种核心的视频制作形式：**口播**。理解这些概念，将帮助你明确账号方向，创作出更具吸引力的内容。

上一节我们解决了文案问题，本节中我们来看看内容创作的下一个关键环节。很多课程会直接教你涨粉或变现，但从涨粉到变现并非一条直线，中间还有一个核心环节：**建立人设**。

建立人设的目的并非为了炫耀，而是为了在众多同类账号中脱颖而出。例如，有十个人都做装修内容，如果其中一位（比如小王同学）树立了一个幽默搞笑的人设，观众就更容易记住他，而忽略其他九位。当观众喜欢你、认可你时，他们自然就成为你的粉丝。这就是人设的价值。

接下来，我们来学习这个知识点。请边听边记。人设主要分为两种：**强人设**和**弱人设**。区分清楚后，你就能快速判断任何视频的人设类型。

首先，我们来了解什么是强人设。

**强人设**的核心是以创作者**个人**为切入点，内容与创作者自身密切相关。这包括分享个人的行业经验、专业技能或兴趣爱好。

![](img/508fb9d783bb5a6a2305ed1b874b3223_1.png)

以下是强人设常见的切入点领域：

![](img/508fb9d783bb5a6a2305ed1b874b3223_3.png)

*   **行业经验**：装修、办公、房产、汽车、餐饮、婚介、法律咨询、电商等。
*   **兴趣爱好**：美食、旅行、手工、宠物、星座占卜、美妆、美容护肤、服装穿搭、游戏、摄影等。

![](img/508fb9d783bb5a6a2305ed1b874b3223_1.png)
![](img/508fb9d783bb5a6a2305ed1b874b3223_3.png)

强人设的内容是分享自己的生活、工作或爱好，并且**大部分需要真人出镜**，因为内容与“你”这个人直接相关。

理解了强人设，与之相对应的就是弱人设。

**弱人设**是指内容与创作者本人**没有直接关系**，核心目的是运营好账号本身。例如，纯粹发布情感语录配音乐的视频（无真人出镜、无配音），或者影视解说类视频（虽有解说声音，但焦点是电影内容而非解说者本人），都属于弱人设。

区分清楚强人设和弱人设是后续内容规划的基础。现在，我们来看看如何基于人设来创作视频内容。

只要内容与你个人相关，都可以作为视频素材。我们首先介绍一种非常实用且高效的内容形式：**讲故事**。

讲故事可以分享客户故事、从业经历或人生感悟。故事本身不要求100%真实，可以从文案平台寻找有趣的故事进行分享。而讲故事最常用的形式，就是**口播**。

口播即用嘴巴讲述，它有很多分类。以下是常见的口播类型：

*   **故事型口播**：如情感口播、养生故事口播。目的是让观众觉得有趣、有共鸣。
*   **知识型口播**：如育儿、金融财经、装修、房产汽车知识口播。目的是在几十秒内让观众获得“收获感”，觉得学到了东西。

口播适用于众多领域，如法律、保险、金融、装修知识科普等。它的优势在于**拍摄简单**（一部手机即可），且**依赖提前准备的文案**，而非临场口才。

针对常见问题，这里有两个关键解答：

1.  **口才不好能做口播吗？**
    *   **可以**。口播使用的是提前准备好的稿子，对着镜头录制即可，并非即兴发挥。
2.  **普通话不好能做口播吗？**
    *   **可以**。如果你的目标客户是同城或有地域属性，使用方言可能效果更佳，显得亲切、有趣。同时，可以配上字幕确保观众理解。

口播是一种非常适合新手起步的内容形式，我们将在后续课程中详细讲解其拍摄技巧。

本节课中我们一起学习了抖音账号的**人设**概念，明确了**强人设**（以个人为中心）与**弱人设**（以内容为中心）的区别。同时，我们重点介绍了基于强人设的一种核心视频形式——**口播**，了解了它的类型、优势及适用场景。掌握这些基础知识，将为你后续的内容创作打下坚实的根基。
