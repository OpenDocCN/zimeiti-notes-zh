# 亚马逊ERP系统教程：P1：核心功能详解与操作指南

在本节课中，我们将详细学习亚马逊全功能ERP系统的核心模块与操作流程。我们将从店铺授权开始，逐步了解产品发布、订单处理、采购管理、物流发货及客服工具等关键功能，帮助你全面掌握系统的使用方法。

上一期我们介绍了ERP的两种模式，本节中我们来看看系统的具体功能模块如何运作。

## 店铺授权与管理

首先，你需要在系统的“会员中心”找到“店铺管理”模块，在此授权并绑定你的亚马逊店铺和邮箱。店铺授权成功后，你便可以在系统内直接操作产品发布、查看店铺订单。

## 产品与采购管理

对于从事无货源模式的卖家，系统提供了采购集成功能。

以下是采购管理的核心步骤：
1.  在“采购功能”中绑定你的1688账号。
2.  当店铺产生订单时，系统会自动同步订单信息。
3.  你可以通过系统直接前往1688采购所需产品。
4.  采购单号和国内物流单号将自动同步回系统。

## 订单处理与发货

所有出单产品都会在“订单功能”的“订单管理”列表中显示。你可以在此按店铺筛选订单，并进行发货等操作。

发货前需要打包产品，你可以选择使用系统中转仓或自行打包。

以下是发货操作流程：
1.  在包裹信息中填写采购获得的国内物流单号。
2.  选择物流平台，其他信息按需填写。
3.  填写完毕后，点击“发货”按钮。
4.  在“订单打印”模块，可以打印发货产品的国际面单。

## 物流与客服追踪

已发货订单的物流信息，可以在“订单管理”的“已发货”列表中找到。通过点击对应订单的“查看物流信息”按钮，即可跟踪国际物流单号。

![](img/5267e2d5cd00459eb130d43a39c949db_1.png)

客服功能集中于“工具”模块下的“邮件信息”中。你可以在此查看店铺所有邮件的收发记录，并进行及时回复。

![](img/5267e2d5cd00459eb130d43a39c949db_2.png)

## 财务与辅助工具

除了核心的进销存功能，系统还提供以下高级功能：
*   **财务汇总**：用于查看订单成本、利润及店铺绩效数据。
*   **利润试算**：在发布产品前预估利润。
*   **物流商管理**：如果你选择自行打包，可以在“物流认证”中添加你合作的物流服务商。

系统还内置了多种辅助运营工具：
*   **抢购物车工具**：如定时上下架、防跟卖监控、智能调价等。
*   **运营辅助工具**：包括自动虚拟发货、超卖监控、防御同行恶意竞争等功能。

## 团队与系统定制

在“会员功能”的“子账号角色”中，你可以进行团队权限管理。

以下是设置子账号的步骤：
1.  首先，在“角色管理”中为不同职位分配相应的系统功能权限。
2.  然后，在“子账号管理”中添加成员账号，并为其分配已设置好的角色。

![](img/5267e2d5cd00459eb130d43a39c949db_4.png)

此外，系统支持深度定制，你可以一同定制专属的品牌名称、LOGO，并进行独立部署。所有功能全开放无限制，采用一次性付费、终身使用的模式，并享受终身免费系统更新。

本节课中，我们一起学习了亚马逊ERP系统从店铺授权、产品采购、订单处理到物流发货、财务查看及团队管理的完整操作流程。掌握这些功能，能有效提升你的店铺运营效率与自动化水平。
