# 小红书运营实操教程：P8：4.小红书的劣势 📉

在本节课中，我们将要学习小红书平台的一个关键方面——其劣势。虽然小红书的优势很诱人，但凡事都有两面性。了解其劣势，特别是对新手在选品上的高要求，能帮助我们更好地规划运营策略。

上一节我们介绍了小红书的优势，本节中我们来看看它的主要挑战。对于新手而言，小红书的劣势在于对选品的要求非常高。今天选择的产品，必须符合目标用户的胃口，才能更高效地将产品销售出去。

关于选品这一点，老师我最擅长的就是爆款选品的打造。因此，接下来我会教给大家，新手同学如何选择一个能直接出单、直接赚钱的爆款产品。所以，选品这个点大家不用担心，我会教给你们。

我在1688上帮他拿货的成本价是38元。下面我们来算一笔账。

**利润计算公式如下：**
```
毛利润 = 销售价格 - 拿货成本价
净利润 = 毛利润 - 运费及其他成本
```
*   销售价格：99元
*   拿货成本：38元
*   毛利润：99 - 38 = 61元
*   假设运费为10元
*   **净利润：61 - 10 = 51元**

这意味着，刘宇同学每卖出一套家居服，净利润可以达到51元。

![](img/2cb6188e0d3a6cc5d8e1b8f79a5d9fa8_17.png)

那么他的销售情况如何呢？我们来看一下数据。

![](img/2cb6188e0d3a6cc5d8e1b8f79a5d9fa8_17.png)

到目前为止，这款家居服已经卖了817个订单。

**总利润计算：**
```
总净利润 = 单件净利润 × 总订单数
总净利润 = 51元 × 817 = 41,667元
```
仅凭这一个单品，他在两个月内就实现了41,667元的纯利润。刘宇和大多数同学一样，是零基础的纯新手。

他的店铺不止这一个单品，后期会上架十几个甚至二十几个单品。可以想象，他的收益能够直接翻倍。我们再看一下他店铺的整体数据，他的店铺名叫“爱老虎油”。在10月份，该店铺的销售额达到了95,458元。

**月利润估算：**
```
月净利润 ≈ 总销售额 × 净利润率
```
我们按最低40%的净利润率估算：
```
月净利润 ≈ 95,458元 × 40% = 38,183元
```
这意味着他在10月份净赚了约3.8万元。

最初他只是想每月挣两三千元，结果现在小红书店铺已从兼职转为全职运营，每月赚两三万元没有问题。所以，我们现在看小红书店铺，其实只要有一部手机就能够做起来。

以下是小红书店铺模式的核心优势总结：
*   **无需囤货压货**：采用一件代发模式。
*   **零启动资金**：可以零成本开店赚钱。
*   **门槛低**：无需像开公司或实体店那样投入大量资金。

就像我的学生一样，开一个店，每月挣几千到几万元是完全可行的。
