# 自媒体运营：P8：视频剪辑实操（下） 🎬

在本节课中，我们将继续学习视频剪辑的后续关键步骤，包括如何为视频片段添加转场、如何添加字幕，以及这些操作对视频数据的影响。掌握这些技巧能让你的视频更专业、更流畅，从而提升观众的观看体验和视频的完播率。

上一节我们介绍了视频素材的导入与基础比例调整，本节中我们来看看如何让多个视频片段之间衔接得更自然。

## 添加转场：提升视频流畅度

![](img/01d023735eab96055e1b2c205917185c_1.png)

视频通常由多个短片段组成，每个片段大约十秒钟。这些片段之间需要有自然的衔接，因此必须添加转场效果。

![](img/01d023735eab96055e1b2c205917185c_3.png)

转场的作用是让用户明确感知到视频内容进入了另一个不同的片段。如果不添加转场，观众在切换到下一个画面时可能会感到突兀，甚至产生疑惑，怀疑自己是否点错了视频。一旦观众产生退出视频查看的念头，**视频的完播率就会直线下降**。而完播率直接影响视频的推荐权重和最终收益，因此添加转场至关重要。

![](img/01d023735eab96055e1b2c205917185c_5.png)

以下是添加转场的具体步骤：

![](img/01d023735eab96055e1b2c205917185c_7.png)

1.  在视频编辑轨道上，找到右侧的 **“+”** 号按钮并点击。
2.  在弹出的菜单中，选择 **“素材库”**。
3.  在素材库的“热门”分类中，可以找到大量现成的转场素材，例如各种动态切换效果。目前网络上许多视频使用的转场都来源于此。
4.  选择转场时，注意时长。避免选择过长的转场（如16秒），因为观众没有耐心观看无关紧要的内容。**尽量选择3~5秒时长的转场素材**，3秒左右最为合适。
5.  选中合适的转场素材，点击添加即可插入到视频轨道中。

![](img/01d023735eab96055e1b2c205917185c_9.png)

添加好转场后，还需要确保其画面比例与主视频保持一致。

![](img/01d023735eab96055e1b2c205917185c_11.png)

## 统一转场画面比例

![](img/01d023735eab96055e1b2c205917185c_13.png)

添加的转场视频也必须符合 **16:9** 的横屏比例要求。操作非常简单：

![](img/01d023735eab96055e1b2c205917185c_15.png)

1.  选中轨道上的转场片段。
2.  在编辑界面找到 **“画布模糊”** 或类似的功能选项。
3.  通常选择第一个“清晰画面”或“模糊背景”效果，系统会自动将转场背景处理成符合16:9比例的横屏。
4.  最后，点击右下角的对勾确认操作。

![](img/01d023735eab96055e1b2c205917185c_17.png)

至此，视频的比例调整和转场添加就完成了。记住，每一步都要严格按照要求操作，尤其是素材时长和比例，这关系到平台能否顺利审核通过你的视频。

接下来，我们要为视频添加灵魂——字幕。

## 添加字幕：提升信息传达效率

![](img/01d023735eab96055e1b2c205917185c_19.png)

字幕能帮助观众更好地理解视频内容，尤其在观看环境嘈杂或不便外放时显得尤为重要。添加字幕的步骤如下：

![](img/01d023735eab96055e1b2c205917185c_21.png)

1.  在编辑界面左下方找到 **“文字”** 选项并点击。
2.  进入文字功能后，选择 **“识别字幕”**。这个功能可以自动识别视频或音频中的人物对话，并生成对应的文字字幕。
3.  点击“开始匹配”，系统便会自动生成字幕轨道。

生成字幕后，通常需要调整其样式和位置以达到最佳观看效果：

1.  **调整大小**：选中字幕文本框，拖动其右下角可以放大或缩小字体。
2.  **调整位置**：将放大后的字幕文本框拖动到视频画面的最下方，这是最符合观看习惯的位置。
3.  调整好第一句字幕后，后续的字幕通常会**自动应用相同的样式和位置**，非常方便。

![](img/01d023735eab96055e1b2c205917185c_23.png)

## 核心要点回顾与总结

本节课中我们一起学习了视频剪辑后半部分的核心操作：

![](img/01d023735eab96055e1b2c205917185c_25.png)

1.  **添加转场的必要性**：转场能明确区分视频片段，提升观看流畅度，避免观众因内容跳跃而产生退出行为，从而**保护视频的完播率**。
2.  **转场的添加与调整**：通过素材库添加3-5秒的转场，并使用“画布模糊”功能统一其比例为 **16:9**。
3.  **字幕的添加与优化**：利用“识别字幕”功能自动生成字幕，并调整其大小和位置至视频底部，以提升信息传达效率。

![](img/01d023735eab96055e1b2c205917185c_27.png)

整个视频制作流程并不复杂，但需要你严格遵循步骤：寻找合适时长的素材、统一画面比例、添加转场、配上字幕。坚持这样操作，你就能制作出符合平台要求、观感专业的视频内容。
