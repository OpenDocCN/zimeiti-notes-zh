# 小红书运营教程：P31：小红书笔记引流：平台流量分发机制 🚀

![](img/fb6eae64980836ac232960ccbd8e97d9_1.png)

在本节课中，我们将要学习小红书平台的核心流量分发机制。了解这套算法，是提升笔记曝光和排名的关键。

## 概述

小红书平台的流量分发并非随机，而是基于一套复杂的算法系统。这套系统通过分析用户与笔记的多种交互行为，对内容进行评分和排序，从而决定其展示的优先级和流量规模。

## 流量分发核心机制：S1CCES

小红书整体的流量分发机制，可以简称为 **S1CCES** 系统。这代表了点赞、收藏、转发、评论和关注这五种用户行为，它们都拥有独立的评分计算体系。

## 基础评分与排名逻辑

上一节我们介绍了S1CCES系统，本节中我们来看看它是如何具体计算并影响笔记排名的。

小红书的信息流页面一次展示20篇笔记，刷新后更换下一批（第21-40名）。笔记在这20个位置中的排名，由综合评分决定。这个评分基于单篇笔记获得的用户互动数据，具体分值如下：

*   **点赞**：+1分
*   **收藏**：+1分
*   **转发**：+4分
*   **评论**：+10分
*   **关注**：+8分

**公式：单用户最高贡献分 = 1 + 1 + 4 + 10 + 8 = 24分**

当一个用户完成从点击到关注的全部行为，理论上可为该笔记贡献24分。系统通过计算笔记获得的平均综合分数，来决定其在同批次笔记中的排名位置。

## 流量分发的阶梯流程

理解了评分机制后，我们来看看平台是如何根据这个分数来分配流量的。

一篇笔记发布后，系统会为其打上标签，并推送给具有相关兴趣的用户。新笔记通常会获得一个基础曝光量（例如1000次）。平台的推流是分批次进行的，而非一次性完成。

以下是流量分发的关键步骤和考核指标：

1.  **点击率**：这是流量分发的第一道门槛。用户是否被你的封面和标题吸引并点击进入，决定了后续所有互动行为能否发生。点击率通常需要达到一定标准（如13%-20%以上），才能获得下一波流量扶持。
2.  **互动率与CES评分**：用户点击进入后产生的点赞、收藏、评论、转发、关注等行为，将累积为CES综合评分。
3.  **完播率**：用户是否完整观看了你的视频或认真阅读了图文笔记。
4.  **互动值**：创作者与评论用户之间的回复互动频率和质量。

系统会根据首轮曝光的数据（主要是点击率和初始CES评分），决定是否给予笔记第二轮、第三轮更大规模的推荐。如果数据达标，流量会像滚雪球一样增长；如果未达标，推荐量则会减少甚至停止。

## 核心算法原则

无论表面的数据指标如何变化，小红书流量分发的核心算法原则始终是：**考察笔记带来的粉丝交互行为**。

用户与笔记、用户与创作者之间的互动深度和质量，直接反映了内容的价值。即使是粉丝数不多的新账号，只要单篇笔记的互动数据优秀，也能凭借高评分被系统推荐给更多潜在用户，形成阶梯式的曝光增长。

## 总结

本节课中我们一起学习了小红书平台的流量分发机制。总结如下：

1.  机制核心是 **S1CCES** 综合评分系统，通过量化用户互动行为进行排名。
2.  流量分发是 **分批次、阶梯式** 的，后续流量取决于前一轮的点击率、互动率等数据表现。
3.  算法的根本目的是筛选出能 **引发高质量用户交互** 的内容。
4.  新账号发布原创内容，通常能获得150-200的基础曝光量，此为冷启动流量。
