# 小红书运营系统课：P77：14. 色度抠图去绿幕 🎬

在本节课中，我们将学习视频剪辑中一个非常实用的技巧——**色度抠图**。通过这个功能，我们可以轻松去除视频中的纯色背景（如绿幕），并将其替换为我们想要的画面，是制作创意短视频的必备技能。

---

![](img/b0181791667d3de554441f7699a1d5db_1.png)

上一节我们介绍了视频剪辑的基础操作，本节中我们来看看如何使用“色度抠图”功能去除绿幕背景。

**色度抠图**是一种基于颜色信息来分离前景与背景的技术。其核心原理是选取画面中的特定颜色（如绿色），并将其设置为透明，从而露出下方的视频图层。在剪映中，这一过程通过调整`强度`和`阴影`两个参数来实现。

以下是使用剪映进行色度抠图的具体步骤：

1.  **导入与对齐素材**
    *   首先，打开剪映，导入一段作为背景的主视频。
    *   点击`画中画`功能，新增一个画中画图层，并从素材库导入一段带有绿幕背景的素材。
    *   选中画中画素材，使用`分割`工具剪掉多余部分，并将其与主视频的时长对齐。

2.  **调整画面方向**
    *   保持画中画素材为选中状态，点击`编辑`，选择`镜像`功能，可以水平翻转画面，调整到您想要的方向。

3.  **初步抠取绿色**
    *   返回上一级菜单，点击`色度抠图`。
    *   画面中央会出现一个`取色器`（圆形光标）。移动取色器，使其对准绿幕的绿色部分。
    *   点击`强度`，向右拖动滑块增加数值，观察绿色背景是否被去除。此时，您可能会发现物体边缘仍残留绿色边框。

4.  **优化边缘残留**
    *   退出色度抠图界面，点击`调节`选项。
    *   在调节工具中找到`HSL`（色相、饱和度、亮度）工具。
    *   点击`绿色`通道，将其`饱和度`滑块拖到最左侧（数值为`0`）。此操作会降低画面中所有绿色的饱和度，有助于消除边缘杂色。
    *   再次进入`色度抠图`功能，移动取色器选取绿色，调整`强度`数值（例如`27`）。
    *   接着调整`阴影`数值（例如`71`），这可以为抠像后的物体边缘添加一些半透明效果，使其与背景融合得更自然。

5.  **最终调整**
    *   点击确定，完成抠图。
    *   最后，在预览窗口中拖动已抠像的画中画素材，将其调整到主视频画面中您想要的位置。

这样，一个完整的色度抠图去绿幕效果就制作完成了。

本节课中我们一起学习了**色度抠图**功能的核心用法。关键操作流程可总结为：**导入对齐 → 初步抠色 → 使用HSL工具去杂色 → 精细调整强度与阴影 → 确定位置**。掌握这个技巧，您就能轻松合成各种创意场景，大幅提升短视频的视觉效果。
