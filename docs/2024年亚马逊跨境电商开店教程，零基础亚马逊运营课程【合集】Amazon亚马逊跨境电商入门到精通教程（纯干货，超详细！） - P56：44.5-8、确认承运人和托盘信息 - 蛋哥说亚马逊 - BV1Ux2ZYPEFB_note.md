亚马逊跨境电商运营：P56：确认承运人和托盘信息

在本节课中，我们将学习“发货至亚马逊”工作流程的第四步：确认承运人和托盘信息。此步骤仅在您选择按托盘运输时出现。完成此步骤后，您将准备好将货物交给承运人并发往亚马逊运营中心。

![](img/67327a8d8cb0272e7acc54abcc32088f_1.png)

### 使用亚马逊合作承运人

![](img/67327a8d8cb0272e7acc54abcc32088f_5.png)

如果您选择通过亚马逊合作承运人发货，可以在“发货至亚马逊”工作流程内直接购买货件标签、安排取件并享受折扣运费。

以下是使用合作承运人的具体步骤：

1.  **输入运输就绪日期**：这是承运人可以前来取件的日期。提供准确的日期有助于亚马逊提前规划，在配送网络中接收您的库存，避免延误。
2.  **添加承运人联系人信息**：如果是首次发货，需要添加在取件地点负责对接承运人的联系人信息。如果之前已发过货，可以选择已有的联系人。
3.  **提供托盘信息**：将箱子装入托盘，并为工作流程中的每个货件提供托盘信息。系统会根据您在第一步提供的箱内物品信息预填此部分，请仔细核对。确认无误后点击“确认”以查看预估运费。如有错误，请编辑详细信息。
4.  **了解托盘信息字段**：
    *   **货物等级**：一种标准化分类系统，用于确定货件的计费重量和风险等级。等级范围从50到500不等，可以手动调整或使用亚马逊的默认预估值。
    *   **运费申报价值**：选填字段，有助于确定库存价值。如果留空，系统将默认按“货件重量 × 1.50美元”计算。
    *   **托盘重量与尺寸**：输入每个托盘的重量、高度、是否堆叠以及各种配置的托盘数量。
5.  **确认并继续**：确认所有货件的托盘信息后，接受承运人费用，然后点击“确认承运人和托盘信息”。至此，步骤四完成，可以进入下一步：打印托盘标签并安排取件。

### 使用非合作承运人

![](img/67327a8d8cb0272e7acc54abcc32088f_7.png)

![](img/67327a8d8cb0272e7acc54abcc32088f_8.png)

![](img/67327a8d8cb0272e7acc54abcc32088f_9.png)

如果您决定自行选择亚马逊合作承运人之外的物流公司进行配送，请按以下流程操作。

以下是使用非合作承运人的具体步骤：

1.  **选择非合作承运人**：在承运人选择部分，点选“非合作承运人”。
2.  **确认托盘数量并打印标签**：将箱子装入托盘，确认每个货件的托盘数量。点击“打印”，为每个托盘获取4份亚马逊物流托盘ID标签。这些标签包含运营中心信息，用于货件到达验证。请在托盘的四个侧面各贴上一个标签，以便叉车司机识别。
3.  **选择承运人**：选择您将用来配送至亚马逊的承运人。如果不确定或会使用多家，请选择“其他”。
4.  **标记发货并录入提单号**：将托盘交给承运人后，返回卖家平台，选择“全部标记为已发货”，以表示您的库存已启运。承运人会为每批货件提供一份提单（BOL）。您需要在“追踪货件”页面下的“追踪事件”选项卡中，为每个货件输入对应的BOL编号。

本节课中我们一起学习了“发货至亚马逊”流程中确认承运人和托盘信息的完整步骤。关键点在于区分合作与非合作承运人的不同操作路径，并准确填写托盘信息与运输单据，以确保货件顺利入仓。
