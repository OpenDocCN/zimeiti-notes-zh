# 小红书电商运营教程：P11：小红书店铺类型详解 🏪

![](img/19bae7a4a6cd802e9799a1bd6c1e9dd2_0.png)

在本节课中，我们将系统学习小红书电商平台上的店铺类型。了解不同类型的店铺及其特点，是规划开店策略、明确运营方向的重要基础。

## 店铺类型概述

小红书平台上的店铺主要分为两大类：商家入驻的店铺和平台自营的店铺。其中，商家店又可细分为多种类型，每种类型在入驻条件、权重和功能上有所不同。

## 商家店铺类型详解

上一节我们概述了店铺的分类，本节中我们来详细看看商家可以入驻的几种店铺类型及其特点。

以下是小红书商家店的主要类型：

1.  **官方旗舰店**：品牌知名度最高的一类店铺，由品牌方直接运营或独家授权运营。平台赋予的权重最高。
2.  **旗舰店**：同样需要较高的品牌知名度，可能是品牌方或获得一级授权的代理商运营。权重仅次于官方旗舰店。
3.  **品牌店**：这是普通商家入驻小红书时最常见的店铺类型。名称通常为“某某某品牌店”。
4.  **专卖店**：经营单个品牌产品的店铺，也是商家常见的开店形式。
5.  **集合店**：此类店铺可以经营多个品牌的产品，类似于天猫的“专营店”。**但目前小红书已暂停集合店的新入驻**，未来是否会重新开放需关注平台政策。

## 店铺权重与升级机制

了解了店铺类型后，我们需要知道它们之间的层级关系以及如何提升自己的店铺等级。

这些店铺类型存在明确的权重排序，公式可表示为：
**店铺权重：官方旗舰店 > 旗舰店 > 品牌店/专卖店/集合店**

更高的店铺权重意味着在商品搜索、笔记推荐等场景中可能获得一定的初始优势，但这并非绝对性的流量保证。

对于已开店的商家，存在升级路径。品牌店或专卖店在满足一定条件后，可以申请升级为旗舰店或官方旗舰店。核心升级条件通常是**店铺的营业额**。具体标准因商品类目而异，例如：
-   小类目可能月销售额达到数十万即可申请。
-   大类目的营业额要求则会更高。

平台会根据内部标准评估品牌的“知名度”，但营业额是关键的量化指标。

## 平台自营店铺：小红书福利社

除了商家入驻的店铺，小红书平台自身也运营着自营业务，这与京东自营模式类似。

小红书的自营店铺称为 **“小红书福利社”** 。商家可以以供货商的身份将商品入驻福利社，此时小红书扮演经销商的角色。入驻福利社通常意味着商品在小红书已具备一定的销售热度，平台采销人员会主动联系表现优异的商家。合作模式通常是商家以较低折扣（例如出厂价或批发价）向小红书供货，由小红书福利社负责销售和发货。

**本节课总结**：我们一起学习了小红书电商的店铺体系。主要分为**商家店**（包括官方旗舰店、旗舰店、品牌店、专卖店和已暂停入驻的集合店）和**平台自营店**（小红书福利社）。不同店铺类型权重不同，且商家店可通过提升营业额实现升级。理解这些类型有助于你根据自身品牌定位和发展阶段，选择最合适的开店或运营策略。
