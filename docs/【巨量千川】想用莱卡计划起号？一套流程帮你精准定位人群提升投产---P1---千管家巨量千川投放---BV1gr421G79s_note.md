## 投放方式设置

![](img/cf6ba1bffd1d981846e03bb0451b6800_6.png)

上一节我们介绍了徕卡计划的核心优势，本节中我们来看看如何搭建计划。首先从投放方式开始。

通常情况下，我们推荐选择**控成本投放**，非必要时不选择放量投放。优化目标建议选择**直播间成交**或**商品购买**，因为我们的核心目的是为账号打标签和建立模型。而支付ROI目标更适用于后期，用于收割已有成交计划累积的精准人群。

对于直播投放的优化周期，选择**直接成交**选项的ROI通常更高，不建议选择“7日总成交”。虽然“7日总成交”跑量较大，但ROI相对较低，建议在账号未起量（“没亮”）的阶段再考虑使用。

## 预算与出价策略

了解了投放目标后，接下来是预算与出价的设置，这直接关系到计划的跑量和成本控制。

![](img/cf6ba1bffd1d981846e03bb0451b6800_12.png)

日预算建议设置在**3000元以上**，但注意不要直接拉满预算上限。具体出价需根据您产品的利润实际情况来定。

![](img/cf6ba1bffd1d981846e03bb0451b6800_14.png)

**核心要点**：出价必须填写到小数点后两位，不要填整数。出价即使只差几分或几毛，对曝光量的影响也可能非常巨大。

![](img/cf6ba1bffd1d981846e03bb0451b6800_16.png)

公式示例：`出价 = 预期成本 ± 微调值`

![](img/cf6ba1bffd1d981846e03bb0451b6800_18.png)

## 定向方式详解

![](img/cf6ba1bffd1d981846e03bb0451b6800_20.png)

预算设置完成后，定向是徕卡计划精准触达目标人群的关键。我们将分步拆解。

![](img/cf6ba1bffd1d981846e03bb0451b6800_22.png)

以下是基础定向的必选项，如果遗漏可能导致人群跑偏：
*   **性别**
*   **年龄**
*   **排除幸运地区**

![](img/cf6ba1bffd1d981846e03bb0451b6800_24.png)

接下来是徕卡定向的核心设置。选择**近30天的电商互动行为**作为定向依据。

徕卡定向中，类目词和关键词的选择没有固定公式，需要通过测试来筛选数据表现好的词。其底层逻辑是**排除逻辑**，即排除与产品弱相关、点击率低的人群，其他相关词都可以尝试加入。

![](img/cf6ba1bffd1d981846e03bb0451b6800_26.png)

因此，我们不应只关注选择了多少个关键词，更要关注**覆盖的人群规模**，避免因定向过窄导致计划无法跑量。

![](img/cf6ba1bffd1d981846e03bb0451b6800_28.png)

关于具体选词的技巧和策略，可以回顾我们之前的详细讲解视频。

![](img/cf6ba1bffd1d981846e03bb0451b6800_30.png)

至于其他定向选项：
*   一般不建议同时选择**达人定向**和**徕卡定向**。如果必须使用达人定向，建议选择整个达人分类，而非单个达人。
*   网络环境选择**WIFI和5G**，其他选项（如设备、平台等）可选择“不限”。
*   **智能放量**功能在起号阶段基本可以不用。

![](img/cf6ba1bffd1d981846e03bb0451b6800_32.png)

## 创意设置技巧

![](img/cf6ba1bffd1d981846e03bb0451b6800_34.png)

定向决定了把广告展示给谁，而创意则决定了广告的吸引力。良好的创意设置能有效提升点击率和转化率。

![](img/cf6ba1bffd1d981846e03bb0451b6800_36.png)

选择**自定义创意**或**程序化创意**均可。但需注意，一条计划最多可添加三个视频和三条文案。

![](img/cf6ba1bffd1d981846e03bb0451b6800_38.png)

**优化建议**：在计划开始跑量后，建议调整为**只放一个视频配合三条文案**进行单独投放，以便更清晰地分析数据。

在创意分类方面，初期测试时可以多尝试与产品相关的周边类目，然后选择数据表现最好的一个类目进行持续投放。

![](img/cf6ba1bffd1d981846e03bb0451b6800_40.png)

创意标签必须写满**20个**。可以直接从行为兴趣关键词中寻找，包括：
*   产品词
*   品牌词
*   品类词
*   功能词
*   功效词

![](img/cf6ba1bffd1d981846e03bb0451b6800_42.png)

代码示例（选词思路）：
```python
# 创意标签关键词来源
tags = ["产品核心词", "品牌名", "产品大类", "核心功能1", "核心功能2", "使用场景", "解决痛点", ...]
# 确保填满20个
assert len(tags) == 20
```
