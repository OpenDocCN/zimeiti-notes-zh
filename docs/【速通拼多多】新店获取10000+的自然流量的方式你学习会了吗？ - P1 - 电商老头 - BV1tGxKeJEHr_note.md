# 拼多多运营教程：第1章：解锁自然流量核心入口 🚀

在本节课中，我们将学习如何为拼多多新店铺打开关键的免费自然流量入口。这些设置是提升店铺访客量的基础，操作简单，效果显著。

许多新店铺的流量会卡在很低的水平，这通常是因为一些重要的自然流量入口没有开启。只要正确打开这些入口，获得可观的访客量是完全可能的。

上一节我们提到了流量入口的重要性，本节中我们来看看具体有哪些入口以及如何设置。

以下是五个关键的免费流量入口及其开启方法：

1.  **流量续航活动**
    *   此活动在商家后台通常无法直接找到。
    *   无需审核与比价，能带来大量曝光。
    *   **设置方法**：在营销活动界面，将网址中的活动ID参数修改为 **21967** 即可进入报名页面。

2.  **省钱月卡**
    *   参与此活动可直接获得平台流量扶持。
    *   **设置方法**：同样在营销活动界面，将网址中的活动ID参数修改为 **15591** 进行报名。

3.  **超级满减**
    *   这是平台重要的促销活动入口，参与即享流量曝光。
    *   **设置方法**：在营销活动界面，将网址中的活动ID参数修改为 **21824** 进行报名。

4.  **三组图与首单直降**
    *   **三组图**：在商品管理的“图文素材”中上传，可额外获得展示机会。
    *   **首单直降**：在“营销工具”中设置，能有效吸引新客户。
    *   这两项设置完成后，平台会相应分配流量。

5.  **多多视频**
    *   通过买家端发布与商品相关的视频内容，可以获得更高的流量推荐。
    *   需要主动在买家版App中操作发布。
