# AI短视频创作：P11：8.影视推广平台 🎬

![](img/cf96a005a4ccee837e4067448f0831ba_1.png)

在本节课中，我们将要学习如何通过“影视推广平台”为你的短视频结算收益。我们将详细讲解不同平台的收益规则、任务参与方式以及如何最大化你的收入。

---

![](img/cf96a005a4ccee837e4067448f0831ba_3.png)

很多同学问到，如果想把视频发布到抖音或快手平台，应该如何结算收益。这些视频发布到其他平台，比如之前提到的30多个自媒体平台，可以依靠视频播放量直接结算收益。

但是，抖音和快手这两个平台有所不同。就像之前强调的，视频发布到这两个平台，主要作用是增长粉丝。

![](img/cf96a005a4ccee837e4067448f0831ba_5.png)

如果有同学只想专注于抖音和快手这两个熟悉的平台，并且不仅想涨粉，还想依靠播放量结算收益，那么接下来的内容非常重要。

---

![](img/cf96a005a4ccee837e4067448f0831ba_7.png)

视频可以发布到抖音、快手、小红书以及微信视频号这四个平台。这些平台大家都很熟悉。小红书用户以女性和年轻群体如95后为主。微信视频号则内置于我们每个人都在使用的微信中。

发布到这些平台后，我们需要通过“影视推广平台”来接取任务，并通过完成任务来结算奖金。平台上的素材非常丰富。

以下是影视推广的核心概念：

*   **影视推广**：指为电视剧、电影或综艺节目进行线上宣传。由于线下宣传受限，片方会通过此类平台发布任务，创作者制作推广视频并发布，即可根据视频播放量获得奖金。

参与推广需要按步骤提交任务。

以下是参与影视推广任务的基本流程：

1.  **报名任务**：在影视推广平台选择要推广的剧集任务进行报名。
2.  **填写信息**：提交任务时，需要填写自己的微信号、账号昵称、发布平台的ID（如抖音ID）、所推广的剧集名称。
3.  **提交作品链接**：粘贴你发布的视频作品链接，注明发布时间。
4.  **上传凭证**：上传视频播放量的截图作为凭证。
5.  **结算收益**：平台审核通过后，会根据规则为你结算收益。

影视推广的收益通常比单纯依靠平台播放量分成的收益更高。

现在，我们来具体看看不同平台的收益计算规则。

以下是各平台收益计算方式：

*   **抖音**：按**播放量**结算。每1万播放量奖金为50元。计算公式为：`收益 = (播放量 / 10000) * 50元`。
*   **微信视频号**：按**播放量**结算。每1万播放量奖金为10元。计算公式为：`收益 = (播放量 / 10000) * 10元`。
*   **小红书**：按**播放量**结算。每1000播放量奖金为30元，单价最高。计算公式为：`收益 = (播放量 / 1000) * 30元`。
*   **抖音+快手（针对部分短剧）**：按**点赞数**结算，无粉丝要求。例如：抖音单作品5000赞奖励100元，1万赞奖励200元；快手1万赞奖励180元。

收益通常取一部剧推广期内，你所有作品中**播放量最高的三个视频的数据总和**进行计算。部分短剧任务可能要求抖音和快手两个平台都发布。

除了上述主流平台，像搜狐视频等平台也会购买一些短剧版权并发布推广任务。虽然其剧集知名度可能不高，但提供的奖金单价非常有竞争力。

本节课中，我们一起学习了如何利用影视推广平台为短视频创作变现。关键点在于：理解影视推广的逻辑，掌握在抖音、视频号、小红书等平台通过接任务赚取奖金的方法，并熟知各平台按播放量或点赞数结算的不同规则。通过参与这些任务，你可以在涨粉的同时，获得可观的额外收益。
