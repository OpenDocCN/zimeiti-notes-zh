# 自媒体运营教程：P26：手机剪辑特效卡点视频 🎬

在本节课中，我们将学习如何使用手机剪辑软件制作一个带有特效和音乐卡点的炫酷视频。整个过程涉及素材导入、特效添加、音频踩点以及图层混合等核心操作。

## 导入与基础设置 📁

首先，我们需要准备素材并进行基础的时间线设置。

1.  导入一张选定的图片到剪辑软件的时间线中。
2.  将这张图片的时长拉长至 **9.7秒**，作为我们视频的主体画面。

![](img/8a44d52185b73f2e73e4664783526ab3_1.png)

## 添加画面特效 ✨

上一节我们导入了主画面，本节中我们来看看如何为其添加第一个视觉效果。

![](img/8a44d52185b73f2e73e4664783526ab3_1.png)

在视频的开头位置，点击添加“画面特效”。
在特效列表中，找到“基础”分类，并选择其中的“变清新2”特效。

## 添加与编辑背景音乐 🎵

画面有了初步效果后，接下来需要搭配音乐。音乐是卡点视频的灵魂，正确的处理能让视频节奏感更强。

![](img/8a44d52185b73f2e73e4664783526ab3_3.png)

以下是添加和编辑音频的步骤：
1.  点击“音频”按钮，准备添加背景音乐。
2.  从“抖音收藏”或其他音乐库中选择一首适合的音乐。
3.  将音频轨道上多出视频时长的部分分割并删除，使音乐与视频时长一致。
4.  选中音乐轨道，点击“踩点”功能。
5.  选择“自动踩点”中的“节拍2”选项，时间线上会出现代表节奏点的小黄点。

![](img/8a44d52185b73f2e73e4664783526ab3_3.png)

## 应用滤镜与特效组合 🎞️

音乐节奏点已经标记好，现在我们可以根据节奏点来添加视觉变化，增强冲击力。

1.  找到音频轨道上大约 **4秒后** 的一个小黄点（节奏点）。
2.  在这个时间点，为视频画面添加一个滤镜：在“黑白”分类中选择“江浙沪”滤镜。
3.  将这个滤镜轨道分割，并删除前半部分，使滤镜只从4秒后的节奏点开始生效。
4.  在当前时间点（节奏点），再添加两个画面特效：“光”分类里的“发光”和“动感”分类里的“闪白”。
5.  将“闪白”特效的持续时间缩短，以匹配瞬间闪过的效果。

## 叠加花瓣素材 🌸

核心的卡点效果已经完成，为了使画面更丰富，我们可以在上层叠加一些动态素材。

1.  点击“画中画”功能，导入准备好的“花瓣”素材视频。
2.  将花瓣素材放大，使其与底层照片画面大小对齐。
3.  选中花瓣素材轨道，点击“混合模式”选项。
4.  在混合模式中选择“绿色”，这样花瓣的黑色背景会被去除，只留下亮色的花瓣叠加在画面上。

## 最终效果预览 👀

所有元素添加完毕后，点击播放预览整个视频效果，检查音乐卡点与视觉变化的同步性。

---

本节课中我们一起学习了制作特效卡点视频的全流程。关键步骤包括：设置主画面时长、添加基础特效、利用音乐的自动踩点功能标记节奏、在节奏点上组合使用滤镜和特效，以及通过画中画和混合模式叠加动态素材。掌握这些操作，你就能制作出节奏感强、视觉效果突出的短视频。
