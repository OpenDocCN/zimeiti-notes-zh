# 抖音运营入门：P12：3、如何完善个人资料 🧑‍💻

在本节课中，我们将要学习如何完善抖音账号的个人资料。一个专业且吸引人的个人资料是账号获得高权重、吸引精准粉丝的基础。本节将详细讲解头像、昵称和个人简介的设置方法与核心原则。

上一节我们介绍了养号的基本概念，本节中我们来看看如何通过完善个人资料来提升账号的专业度与吸引力。

## 头像设置

头像应清晰、有辨识度，并与你的账号定位相关。

以下是选择头像的具体建议：
*   **有实体店**：使用实体店铺的照片。
*   **无实体店**：使用个人清晰的自拍照或生活照。
*   **宝妈群体**：可以使用与宝宝的合照。
*   **宠物类型**：使用宠物的可爱照片。

## 昵称设置

昵称是用户对你的第一印象，至关重要。**切勿使用自己的真实姓名**（如：小月、兰亭），这会被平台判定为普通玩家而非内容创作者，从而影响流量推荐。

一个合格的昵称需要遵循以下规则：
*   **字数**：控制在五个字以内。
*   **字符**：避免使用繁体字和特殊符号（如：❤️），以降低用户记忆和搜索难度。

那么，如何起一个好昵称呢？核心方法是“名字+定位/职业/地区”。

以下是三种有效的命名公式：
1.  **名字 + 定位**
    *   示例：`小月说游戏`、`小月小厨房`、`小月聊房`、`小月说车`、`月姐穿搭`
2.  **名字 + 职业**
    *   示例：`宝妈小月`、`律师小月`、`摄影师小月`、`造型师小月`
3.  **名字 + 地区**
    *   示例：`赣州月姐`、`小月在北漂`、`小月在海外`

你可以根据自身情况，从以上三种模式中选择一种进行组合。

## 个人简介撰写

![](img/bceb6cf7457f7492be28d0789a8c15a8_1.png)

个人简介位于主页，是展示自我、说明价值的关键区域。撰写时应围绕以下三个核心要素展开：

以下是个人简介的三要素结构：
1.  **我是谁**：简短地介绍自己的身份或特点。（例如：中年大叔依然保持母婴初衷）
2.  **我是做什么的**：清晰地说明你的领域或职业。（例如：从事母婴行业13年，做母婴商贸8年有余）
3.  **关注我能得到什么**：明确你能为粉丝提供的价值。（例如：分享育儿经验）

![](img/bceb6cf7457f7492be28d0789a8c15a8_3.png)

你可以根据自己的性格（幽默、可爱等）来调整表述风格，使其更生动活泼。

## 重要注意事项：联系方式

**在个人资料中，请勿随意添加微信或电话号码等联系方式。**

这是一个常见的违规行为，原因如下：
*   **平台规则**：抖音与微信（腾讯）属于不同公司，在抖音主页放置其他App的联系方式属于违规引流，意图将抖音流量导出，平台不鼓励此类行为。
*   **效果有限**：在粉丝量较少时，挂联系方式带来的实际添加人数很少，意义不大。

**例外情况**：
*   已认证的**蓝V企业号**可以挂联系电话。
*   当粉丝达到**1万**并成功入驻抖音的**星图平台**后，可按平台规则添加联系方式。关于星图平台，我们将在后续的变现环节详细讲解。

如果已经添加了联系方式，请课后将其删除，以避免违规风险。
