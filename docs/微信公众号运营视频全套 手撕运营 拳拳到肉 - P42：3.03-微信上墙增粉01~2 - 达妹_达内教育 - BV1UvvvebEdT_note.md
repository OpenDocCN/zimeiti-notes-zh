## 后台功能模块解析

上一节我们介绍了活动创建的基本流程，本节中我们来看看活动创建后，后台各个核心功能模块的具体作用与区别。

![](img/2f2d5594a84482caad5b4d8277d1f16f_12.png)

![](img/2f2d5594a84482caad5b4d8277d1f16f_14.png)

以下是后台主要功能模块的说明：

![](img/2f2d5594a84482caad5b4d8277d1f16f_16.png)

![](img/2f2d5594a84482caad5b4d8277d1f16f_18.png)

![](img/2f2d5594a84482caad5b4d8277d1f16f_20.png)

*   **功能设置**：此模块用于完成活动所有核心功能的配置，例如消息上墙、抽奖等规则的设定。
*   **屏幕设计**：此模块用于设计最终展示给用户看的大屏幕外观，包括背景、标题、LOGO等视觉元素。
*   **查看大屏幕**：这是一个**只读的URL地址**，用于在前台（如活动现场的大屏幕）展示活动界面。用户只能观看，无法在此界面进行操作。此地址可以在任何有网络的地方打开。
*   **屏幕控制台**：这是一个**用于后台管理的二维码或链接**。活动管理人员用手机扫描此二维码后，可以在手机上控制活动流程，例如切换屏幕上显示的二维码、启动或停止抽奖环节等。
*   **消息审核**：此模块用于管理用户发送的上墙消息。您可以设置为“先审后发”，即每条消息需经审核才能显示；或设置为“先发后审”，对不合格的消息进行拒绝或删除操作。
*   **手机参与方式**：这是提供给用户的**参与入口**，包含一个链接和二维码。用户通过此入口可以发送消息、参与抽奖等互动，其发送的内容将实时展示在“查看大屏幕”的地址上。

![](img/2f2d5594a84482caad5b4d8277d1f16f_22.png)

简单来说，**“功能设置”** 是配置规则，**“屏幕设计”** 是美化外观，**“查看大屏幕”** 是观众看到的画面，**“屏幕控制台”** 是管理员的遥控器，**“手机参与方式”** 是用户的参与门票。理解这几者的区别，是顺畅运营一场互动活动的基础。

![](img/2f2d5594a84482caad5b4d8277d1f16f_24.png)

## 屏幕设计详解

![](img/2f2d5594a84482caad5b4d8277d1f16f_26.png)

了解了后台各模块的功能后，接下来我们具体学习如何设计一个符合品牌调性的活动大屏幕。

![](img/2f2d5594a84482caad5b4d8277d1f16f_28.png)

进入“屏幕设计”模块，您可以对两大方面进行调整：主题背景和屏幕内容效果。

![](img/2f2d5594a84482caad5b4d8277d1f16f_30.png)

以下是屏幕设计的具体操作步骤：

![](img/2f2d5594a84482caad5b4d8277d1f16f_32.png)

1.  **更换主题背景**：平台提供了年会、节日、商务等多种风格的预设主题。您可以直接选择应用，快速改变整体视觉风格。
2.  **上传自定义背景**：为了更贴合品牌，您可以上传自行设计的背景图（例如带有公司Logo的年会背景），完全替换默认主题。
3.  **修改屏幕内容**：
    *   **替换LOGO水印**：将“嗨现场”的默认水印替换为您自己公司的LOGO图片（建议尺寸为100x100像素）。
    *   **调整活动标题**：可以修改标题文字、字体大小和颜色，使其更醒目。
    *   **自定义底部版权信息**：可以删除平台默认信息，替换为例如“本活动由[您的公司名]主办”等自定义文本或图片。
4.  **关键设置——二维码**：屏幕上显示的二维码默认是平台的参与二维码。**若以实现增粉为目的，必须将此二维码更换为您自己公众号的二维码**。这样用户扫码后，会先关注您的公众号，再通过公众号内的菜单或自动回复引导至活动页面。

![](img/2f2d5594a84482caad5b4d8277d1f16f_33.png)

![](img/2f2d5594a84482caad5b4d8277d1f16f_34.png)

![](img/2f2d5594a84482caad5b4d8277d1f16f_36.png)

屏幕设计没有固定技巧，核心原则是**按照自身品牌形象和活动需求，完成前台展示效果的自定义**，给用户留下专业、统一的品牌印象。

![](img/2f2d5594a84482caad5b4d8277d1f16f_38.png)

![](img/2f2d5594a84482caad5b4d8277d1f16f_39.png)

## 二维码设置与增粉关键

在屏幕设计的各项操作中，二维码的设置是直接影响增粉效果的核心环节，需要单独强调。

![](img/2f2d5594a84482caad5b4d8277d1f16f_43.png)

![](img/2f2d5594a84482caad5b4d8277d1f16f_45.png)

**关键点在于：** 如果想要通过活动增加公众号粉丝，必须将大屏幕上展示的**默认参与二维码，替换成您自己公众号的二维码**。

![](img/2f2d5594a84482caad5b4d8277d1f16f_46.png)

其逻辑路径是：
`用户扫描屏幕二维码 -> 关注您的公众号 -> 通过公众号内的指引进入活动页面参与互动`

![](img/2f2d5594a84482caad5b4d8277d1f16f_48.png)

![](img/2f2d5594a84482caad5b4d8277d1f16f_49.png)

![](img/2f2d5594a84482caad5b4d8277d1f16f_51.png)

这样做的好处是，所有参与活动的用户都会沉淀为您的公众号粉丝，实现了活动互动与粉丝增长的双重目标。仅仅使用平台默认二维码，用户将直接进入活动页面，无法为您的公众号带来新增关注。

![](img/2f2d5594a84482caad5b4d8277d1f16f_53.png)

![](img/2f2d5594a84482caad5b4d8277d1f16f_54.png)
