## 渠道一：微软 Edge 浏览器 Copilot 🌐

第一个推荐的渠道是微软 Edge 浏览器内置的 Copilot 功能。它是微软基于 ChatGPT 4.0 模型为 Windows 用户打造的功能，智能程度与 4.0 模型相匹配。

使用此功能有两个前提条件：
1.  需要登录微软账号（可使用 Outlook 邮箱注册）。
2.  需要准备“梯子”（VPN）以访问服务。

Copilot 提供了三种对话模式：
*   **更有创造力**：AI 的回答更具开放性和创造性。
*   **平衡**：介于精确和创造力之间的模式，适合一般用途。
*   **更精确**：提供更严谨、准确的回答。

例如，我们可以使用“更有创造力”模式，输入指令 `请帮我写一个关于装修的短视频脚本，时长控制在3分钟左右` 来生成内容。

此外，浏览器版本的 Copilot 还有一个优势：它可以基于你当前打开的网页内容进行分析。

然而，Edge Copilot 也存在一些缺点：
1.  **精准度稍逊**：虽然是基于 4.0 模型，但经过微软调教后，在某些方面的回答可能不如官方 ChatGPT 4.0 精准。
2.  **连续对话能力受限**：在当前对话中持续提问的次数有限，达到限制后需要开启新对话窗口。
3.  **记忆功能有所缺失**：在利用历史对话优化后续回答方面能力较弱，且缺乏插件功能。

如果您的 Edge 浏览器没有 Copilot 图标，可以尝试下载最新版的 Edge Dev（测试版）。您可以在 Edge 官网的“Dev 频道”找到下载链接，或在浏览器设置的“关于”中检查更新。

*   **灵感 AI**：该平台集成了多种 AI 模型，包括 GPT-3.5、GPT-4.0（包括联网版）以及国内主流模型。新用户注册后可获得 **7 天免费无限次使用 GPT-4.0** 的额度（每日有次数限制）。其优点是模型多且 GPT-4.0 可免费体验，缺点是**响应速度可能较慢**，且后续需要付费（如 19.9 元/月）。**无需梯子**。
*   **Vee AI**：该平台提供 GPT-3.5 和 GPT-4.0 模型。新用户注册会赠送一定量的免费 token（计算单位），可用于 GPT-3.5。其优点是**响应速度快**，接近官方体验，且具备语音功能。缺点是免费额度有限，使用 GPT-4.0 需要付费（约 5 美元/100万 token），并且**需要梯子**。
*   **ChatGPT 账号共享平台**：这是一个由网络社区维护的共享账号网站。页面上会列出许多他人分享的账号，并用颜色标记状态（绿色为空闲，橙色/红色为使用中，金色边框代表付费的 GPT-4.0 账号）。其优点是**可以免费尝试使用 GPT-3.5 和 GPT-4.0**（尤其是找到可用的金色 Plus 账号时）。缺点是账号状态不稳定，可用性无法保证，并且**需要梯子**。

如果您没有使用“梯子”的条件，也可以考虑使用国内公司开发的 AI 大模型。它们无需特殊网络环境即可访问。

![](img/5c58a0e4d1ace9224e67173f380d9827_17.png)

![](img/5c58a0e4d1ace9224e67173f380d9827_18.png)

以下是几个主流的国内 AI 大模型：

*   **百度文心一言**：百度推出的 AI 对话模型。有免费版和付费的“文心一言 4.0”版本。在生成短视频脚本、提供灵感等基础运营辅助工作上表现足够。
*   **字节跳动豆包 AI**：字节跳动推出的 AI 助手。同样可以用于生成文案、脚本等，响应速度较快。
*   **阿里通义千问**：阿里巴巴推出的 AI 模型。
*   **昆仑万维天工 AI**：另一个可选的国内 AI 大模型。

这些国内模型在通用任务上可以满足基本需求，但在涉及复杂、专业的领域时，其能力与 ChatGPT 4.0 相比可能仍有差距。建议您根据自身工作需求，选择一款用起来顺手的工具进行熟练使用。

## 官方 ChatGPT 4.0 体验示例 ✨

为了直观对比，我们可以看一下官方 ChatGPT 4.0 在相同指令下的表现。输入指令 `请帮我写一份关于装修的短视频脚本，文案要求镜头丰富，语言吸引人`。

它生成的脚本结构清晰，富有场景感，例如开场：“**缓慢推进的镜头展示一间陈旧、有些破旧的居室。旁白：每一个旧居都藏着一个改造的梦想。**” 这种在镜头语言和叙事节奏上的把控，通常更细腻和具有感染力。
