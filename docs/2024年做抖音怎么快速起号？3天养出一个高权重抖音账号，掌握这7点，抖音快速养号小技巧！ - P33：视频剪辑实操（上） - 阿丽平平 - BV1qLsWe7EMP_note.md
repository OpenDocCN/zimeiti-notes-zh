# 抖音运营与视频剪辑：P33：视频剪辑实操（上） 🎬

在本节课中，我们将学习如何使用剪映App进行视频剪辑的基础操作。我们将从导入素材开始，逐步学习如何添加背景、制作标题、添加配音，以及调整视频比例，为后续的精细剪辑打下基础。

![](img/7550981a4a6511928097c01c7507a564_1.png)

## 导入素材与创建项目

![](img/7550981a4a6511928097c01c7507a564_3.png)

上一节我们介绍了视频剪辑前的准备工作，本节中我们来看看如何开始一个剪辑项目。

![](img/7550981a4a6511928097c01c7507a564_5.png)

首先，在手机上找到并打开剪映App。手机端和电脑端的剪映图标是一致的。

![](img/7550981a4a6511928097c01c7507a564_7.png)

![](img/7550981a4a6511928097c01c7507a564_1.png)

打开软件后，你会看到主界面。点击中央的“+”号按钮，进入“开始创作”界面。

![](img/7550981a4a6511928097c01c7507a564_9.png)

![](img/7550981a4a6511928097c01c7507a564_3.png)

![](img/7550981a4a6511928097c01c7507a564_11.png)

接着，从手机相册中选择一个预先准备好的视频素材，点击添加，将其导入到项目中。

![](img/7550981a4a6511928097c01c7507a564_13.png)

![](img/7550981a4a6511928097c01c7507a564_5.png)

![](img/7550981a4a6511928097c01c7507a564_15.png)

素材添加成功后，它会出现在下方的时间轨道上。

![](img/7550981a4a6511928097c01c7507a564_17.png)

![](img/7550981a4a6511928097c01c7507a564_7.png)

## 添加视频背景

我们能够找到视频素材，但视频的主题背景通常需要自己制作。以下是添加背景的步骤。

![](img/7550981a4a6511928097c01c7507a564_19.png)

在视频轨道的右侧，找到一个“+”号按钮，点击它。

![](img/7550981a4a6511928097c01c7507a564_21.png)

![](img/7550981a4a6511928097c01c7507a564_11.png)

然后，在素材添加页面，找到“素材库”选项。这是剪映软件自带的免费素材库。

![](img/7550981a4a6511928097c01c7507a564_23.png)

![](img/7550981a4a6511928097c01c7507a564_13.png)

点击进入“素材库”，在“热门”分类下可以看到许多背景素材。

![](img/7550981a4a6511928097c01c7507a564_15.png)

![](img/7550981a4a6511928097c01c7507a564_17.png)

选择背景时，建议使用简单明了、能快速传达视频主题的样式。例如，可以选择一个纯黑色的背景。选中后，点击“添加”按钮。

![](img/7550981a4a6511928097c01c7507a564_19.png)

背景素材会作为新的轨道添加到原视频上方。

![](img/7550981a4a6511928097c01c7507a564_25.png)

![](img/7550981a4a6511928097c01c7507a564_21.png)

## 制作视频标题

![](img/7550981a4a6511928097c01c7507a564_27.png)

有了背景，接下来需要添加视频的主题标题。以下是具体操作方法。

点击屏幕左下角的“文字”选项，然后选择“新建文本”。

![](img/7550981a4a6511928097c01c7507a564_29.png)

![](img/7550981a4a6511928097c01c7507a564_23.png)

此时会出现一个文本框。在框中输入视频的主题。主题必须明确，且与视频素材内容强相关。例如，如果计划制作关于“沈腾”的搞笑视频，那么标题应包含“沈腾”，并且所有素材中都必须有他出现。

![](img/7550981a4a6511928097c01c7507a564_31.png)

输入标题，例如：“沈腾综艺爆笑名场面”。区分影视和综艺素材，有助于用户理解。

为了让标题更吸引人，我们不直接使用默认白色文字。点击右下角的“文字模板”。

![](img/7550981a4a6511928097c01c7507a564_33.png)

![](img/7550981a4a6511928097c01c7507a564_25.png)

在模板库中，选择一款免费且合适的样式。选中后，点击右上角的“√”确认。

![](img/7550981a4a6511928097c01c7507a564_27.png)

接着，手动拖动文本框右下角，将标题放大到合适尺寸，使其更清晰。

![](img/7550981a4a6511928097c01c7507a564_29.png)

## 为标题添加配音

![](img/7550981a4a6511928097c01c7507a564_35.png)

视频不能没有声音。以下是两种为标题添加配音的方法。

如果你会配音，可以使用手机录音机录制自己的声音并导入。如果不会，可以使用剪映的“文本朗读”功能。

![](img/7550981a4a6511928097c01c7507a564_37.png)

选中文字轨道，点击右下角的“文本朗读”选项。

![](img/7550981a4a6511928097c01c7507a564_31.png)

软件提供了多种音色，包括方言、萝莉音、播音腔等。目前，使用方言配音更具特色和吸引力。

![](img/7550981a4a6511928097c01c7507a564_33.png)

试听并选择一个喜欢的声音，例如“广西话”。选择后，点击“√”，软件会自动生成音频文件。

![](img/7550981a4a6511928097c01c7507a564_35.png)

生成后，可以播放试听：“沈腾综艺爆笑名场面”。最后，选中黑色背景轨道，手动缩短其长度，使其与配音时长同步。

## 调整视频比例

视频比例是平台审核的重要标准。以下是调整视频至正确比例的方法。

![](img/7550981a4a6511928097c01c7507a564_39.png)

首先确认素材中是否包含核心人物（如沈腾），这是使用该素材的前提。

![](img/7550981a4a6511928097c01c7507a564_37.png)

**核心平台比例要求：**
*   **抖音：** `9:16` （竖屏）
*   **其他多数平台（如B站、西瓜视频）：** `16:9` （横屏）

![](img/7550981a4a6511928097c01c7507a564_41.png)

如果当前视频不是`16:9`横屏，需要进行调整。

![](img/7550981a4a6511928097c01c7507a564_43.png)

**方法一：直接调整比例**
点击下方工具栏的“比例”选项，选择“16:9”。如果视频画面随之移动适配，则此方法有效。

**方法二：手动裁剪（适用于画面不动的素材）**
如果选择比例后画面不动，出现黑边，则需要手动裁剪。
1.  选中视频轨道，点击“剪辑”。
2.  选择“编辑”，再点击“裁剪”。
    ![](img/7550981a4a6511928097c01c7507a564_39.png)
3.  在裁剪界面，**移除视频自带的原字幕**，并适当裁剪边框，避免后续留下过多黑边。
    ![](img/7550981a4a6511928097c01c7507a564_41.png)
4.  裁剪完成后点击“√”。
    ![](img/7550981a4a6511928097c01c7507a564_43.png)

**方法三：设置模糊背景**
裁剪后若仍有黑边，点击“背景”选项，选择“画布模糊”，应用第一个模糊样式。这样黑边会被模糊的背景填充，视频整体呈现为完整的`16:9`横屏。
![](img/7550981a4a6511928097c01c7507a564_45.png)

![](img/7550981a4a6511928097c01c7507a564_45.png)
