## 跨站点同步商品 🌐

上一节我们介绍了店铺后台的基础操作，本节中我们来看看如何将已在美国站点销售的产品，同步到其他国家的亚马逊站点进行销售。这能帮助我们拓宽销售渠道，并可能共享产品评价。

跨站点同步商品主要有三种方法：

1.  **亚马逊物流远程配送计划**：适用于将美国站商品同步至加拿大或墨西哥站。
2.  **通过模板上传**：适用于将商品同步至欧洲、日本等其他站点。
3.  **开通泛欧计划**：适用于在欧洲多个国家站点间同步库存。

### 方法一：亚马逊物流远程配送计划

这种方法适合已有美国站产品，并希望同步到加拿大或墨西哥站的卖家。其优势在于可以共享库存和产品评价。

操作步骤如下：

1.  登录美国站后台，依次点击 **库存 > 亚马逊物流远程配送**。
2.  首次进入需同意“亚马逊物流远程配送”方案，授权亚马逊将你的ASIN配送到加拿大和墨西哥。
3.  同意后，有两种情况：
    *   **若由亚马逊主动邀请**：可直接一键同步商品。
    *   **若手动操作**：需在商品编辑页面手动添加报价。
        1.  进入“管理库存”，找到目标产品，点击右侧的“编辑”。
        2.  在“报价”栏目最下方，手动填写加拿大和墨西哥站点的价格。
        3.  保存后，商品即会在对应站点上线，并同步库存。

**核心概念**：ASIN（Amazon Standard Identification Number）是亚马逊商品的唯一标识码。同步商品时，**必须使用原站点的ASIN**，这样亚马逊才能识别并关联商品，实现评价共享。

### 方法二：通过模板上传

如果你想将美国站的产品同步到欧洲、日本等更远的站点，可以使用批量上传模板。

操作步骤如下：

1.  在目标站点（如英国站）后台，通过“批量上传商品”下载模板。
2.  填写模板时，关键字段需与原站点一致：
    *   `SKU`：填写原站点的SKU。
    *   `Brand`：填写品牌名。
    *   `Product ID` 及 `Product ID Type`：填写原站点的**ASIN**。
3.  对于有父子变体的商品，在模板中正确选择变体关系。
4.  在“Update Delete”列选择 **Update**，因为该产品在新站点是首次创建。
5.  上传模板，等待处理完成。

**注意事项**：此方法主要用于同步商品信息（Listing），库存通常需要单独发往目标站点的仓库。

### 方法三：开通泛欧计划

泛欧计划允许你将库存放置在欧洲6国（如德国、法国、意大利等）中的任一国家，由亚马逊自动完成跨国配送。

操作步骤如下：

1.  在欧洲站后台页面顶部，搜索“泛欧计划”。
2.  点击进入并完成注册。通常也可联系招商经理协助开通。
3.  注册后，你只需将库存发往一个有增值税税号的欧洲国家仓库（如意大利），亚马逊即可将该库存配送至其他参与计划的欧洲国家。

**重要提示**：使用泛欧计划前，需在目标销售国注册有效的增值税税号。

## 虚拟捆绑商品 📦

了解了跨站点同步后，我们来看看如何通过“虚拟捆绑商品”功能，将互补或同类商品组合销售，以提升客单价。

虚拟捆绑商品允许你将2-3个独立的ASIN组合成一个新的销售链接，客户下单后，亚马逊会分别从库存中拣货并发货。

**前提条件**：通常需要完成**品牌备案**。

创建步骤如下：

1.  在后台导航栏找到 **品牌 > 虚拟捆绑商品**。
2.  点击“创建新的虚拟捆绑商品”。
3.  在页面中：
    *   添加要捆绑在一起的商品ASIN。
    *   为这个捆绑组合设置独立的**主图**、**标题**、**SKU**和**价格**。
    *   填写商品描述和卖点。
4.  创建完成后，这个新的捆绑商品将作为一个独立的Listing出现在前台。

![](img/5ced11d90ae801570c76b68df029661a_5.png)

![](img/5ced11d90ae801570c76b68df029661a_6.png)

**优势**：可以营销互补产品（如键盘+键盘垫），并创建更具吸引力的主图展示套装。

## 危险品审核 ⚠️

在创建商品时，有时会触发危险品审核。如果你的产品不含电池、液体、粉末或磁铁等，很可能是亚马逊误判。

处理步骤如下：

1.  在后台搜索“危险品审核”并进入相关页面。
2.  根据产品情况，上传相应的文件：
    *   **若为误判**：选择“不含有化学品的商品豁免表”。
    *   **若含普通电池**：选择“电池驱动商品豁免表”。
3.  下载豁免表模板，根据产品实际情况填写（通常大部分选项都选“No”）。
4.  填写产品ASIN、名称等信息，并上传产品图片或文件。
5.  提交后，等待人工审核，通常需要3-7天。

**核心要点**：只要产品确实不属于危险品，最终都能通过审核，只需耐心等待。

## 类目审核 📋

某些特定类目（如美妆、玩具、母婴用品）在上架前需要经过亚马逊审核，提交相关认证（如FDA、CPC）。

如何判断一个类目是否需要审核？有两种直接的方法：

以下是两种检测方法：

*   **方法一：跟卖检测**
    1.  在前台找到你想销售的同类产品（竞品）。
    2.  复制其ASIN，在你店铺后台的“添加商品”页面粘贴。
    3.  点击“销售此商品”。如果页面提示“需要获得批准”或要求提交认证文件，则该类目/品牌可能需要审核。

*   **方法二：自建Listing检测**
    1.  手动创建一个目标类目的新产品Listing（可使用竞品的图片和标题稍作修改）。
    2.  尝试上传。如果Listing成功创建并处于在售状态一两天后无异常，则通常该类目无需预审。
    3.  如果上传过程中或上传后被下架，并要求提供证明，则触发了类目审核。

**应对审核的小技巧**：如果确信是亚马逊误判（例如，将假睫毛误判为接触皮肤的化妆品），可以在提交审核材料时，上传清晰的产品实物图，并在备注中详细说明产品属性及误判原因，请求重新审查。

## 总结与回顾 🎯

本节课中我们一起学习了创建亚马逊产品的几种进阶方法：

1.  **跨站点同步商品**：利用远程配送计划、模板上传和泛欧计划，可以将一个站点的成功产品快速复制到其他站点，共享评价，拓展市场。
2.  **创建虚拟捆绑商品**：通过品牌功能将互补商品捆绑销售，能有效提高订单价值。
3.  **应对危险品审核**：对于误判，通过提交豁免表即可解决，关键是如实填写信息。
4.  **判断与应对类目审核**：通过跟卖或自建Listing的方式可以快速测试类目审核要求。遇到审核时，根据要求准备相应认证文件，或对误判情况进行申诉。

![](img/5ced11d90ae801570c76b68df029661a_8.png)

掌握这些方法，能让你在亚马逊运营中更加灵活高效，并妥善处理后台遇到的各种商品上架问题。
