# 微信视频号运营教程：P23：20.步骤教学

在本节课中，我们将学习如何从零开始，在微信视频号中开通并设置商品橱窗功能，并了解一个成功案例的运营过程。整个过程将分为清晰的步骤进行讲解。

![](img/2aee75895ab19e96a94af37bede89eaa_1.png)

![](img/2aee75895ab19e96a94af37bede89eaa_2.png)

## 概述

本节教程将详细演示开通视频号商品橱窗的具体操作流程。我们将从打开微信开始，一步步引导你完成权限开通、选品和上架的全过程。最后，我们会通过一个真实案例，展示遵循正确方法后可能取得的成果。

![](img/2aee75895ab19e96a94af37bede89eaa_4.png)

## 开通与设置商品橱窗

![](img/2aee75895ab19e96a94af37bede89eaa_5.png)

上一节我们介绍了视频号带货的基本概念，本节中我们来看看具体的操作步骤。以下是开通商品橱窗功能的详细流程。

![](img/2aee75895ab19e96a94af37bede89eaa_7.png)

**第一步：打开微信**
首先，在手机上启动微信应用程序。



![](img/2aee75895ab19e96a94af37bede89eaa_8.png)

![](img/2aee75895ab19e96a94af37bede89eaa_1.png)

![](img/2aee75895ab19e96a94af37bede89eaa_2.png)



![](img/2aee75895ab19e96a94af37bede89eaa_10.png)

**第二步：进入发现页**
在微信底部导航栏中，点击“发现”选项。



![](img/2aee75895ab19e96a94af37bede89eaa_11.png)

**第三步：进入视频号**
在“发现”页面中，找到并点击“视频号”入口（通常是一个黄色的图标）。



![](img/2aee75895ab19e96a94af37bede89eaa_4.png)

![](img/2aee75895ab19e96a94af37bede89eaa_5.png)



**第四步：进入个人中心**
进入视频号后，点击页面右上角的个人头像图标。



![](img/2aee75895ab19e96a94af37bede89eaa_7.png)

![](img/2aee75895ab19e96a94af37bede89eaa_13.png)

![](img/2aee75895ab19e96a94af37bede89eaa_8.png)



![](img/2aee75895ab19e96a94af37bede89eaa_14.png)

**第五步：进入创作中心**
在个人中心页面底部，找到并点击“创作中心”。如果未显示该选项，通常需要先发布一条视频来激活。

**第六步：进入带货中心**
在创作中心页面内，找到“带货中心”并点击进入。



![](img/2aee75895ab19e96a94af37bede89eaa_10.png)

![](img/2aee75895ab19e96a94af37bede89eaa_16.png)

![](img/2aee75895ab19e96a94af37bede89eaa_11.png)



如果尚未开通橱窗权限，系统会引导你完成开通流程，可能需要选择身份（如“达人”）并缴纳相关费用。

## 选品与上架商品

完成权限开通后，下一步是选择商品并上架到你的橱窗。以下是具体的操作步骤。

**第七步：进入选品中心**
在“带货中心”页面，找到并点击“选品中心”。



![](img/2aee75895ab19e96a94af37bede89eaa_18.png)

![](img/2aee75895ab19e96a94af37bede89eaa_13.png)

![](img/2aee75895ab19e96a94af37bede89eaa_20.png)

![](img/2aee75895ab19e96a94af37bede89eaa_14.png)



![](img/2aee75895ab19e96a94af37bede89eaa_22.png)

**第八步：添加商品到橱窗**
在选品中心浏览商品，例如“菜板”。找到心仪的商品后，点击其下方的“加橱窗”按钮。点击后，该商品就会成功添加到你的视频号商品橱窗中。



![](img/2aee75895ab19e96a94af37bede89eaa_24.png)

![](img/2aee75895ab19e96a94af37bede89eaa_16.png)



操作本身并不复杂。个人运营时感到进展缓慢，往往不是因为不够努力，而是可能走了一些弯路。有经验的人指导可以避免这些坑，从而提升效率。

## 总结

![](img/2aee75895ab19e96a94af37bede89eaa_33.png)

本节课中我们一起学习了微信视频号商品橱窗从开通、选品到上架的完整操作流程。关键步骤可以概括为：**打开微信 -> 发现页 -> 视频号 -> 个人中心 -> 创作中心 -> 带货中心 -> 选品中心 -> 加橱窗**。同时，我们通过一个真实案例看到，在正确的指导和策略下，即使是新手也能在短时间内取得显著的销售成果。掌握清晰的操作步骤和运营方法是成功的关键。
