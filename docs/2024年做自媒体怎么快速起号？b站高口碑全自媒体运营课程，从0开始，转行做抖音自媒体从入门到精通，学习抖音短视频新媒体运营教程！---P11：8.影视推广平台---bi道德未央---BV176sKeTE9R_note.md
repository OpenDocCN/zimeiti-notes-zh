# 自媒体运营课程：P11：8.影视推广平台 🎬

![](img/5829b660c65335013f49bcbe9625c89f_1.png)

在本节课中，我们将要学习如何通过“影视推广平台”来为你的视频内容获取收益。我们将详细解释影视推广的概念、操作流程以及在不同平台上的收益结算方式。

![](img/5829b660c65335013f49bcbe9625c89f_3.png)

---

上一节我们介绍了视频的多平台分发策略，本节中我们来看看一个特殊的变现渠道——影视推广平台。很多同学会问，如果想把视频发布到抖音或快手，该如何结算收益呢？

![](img/5829b660c65335013f49bcbe9625c89f_5.png)

与之前提到的30多个靠播放量直接结算收益的自媒体平台不同，抖音和快手这两个平台，视频发布后主要的作用是**增长粉丝**。如果你不仅想涨粉，同时也希望通过播放量获得直接收益，那么影视推广平台就是一个关键途径。

![](img/5829b660c65335013f49bcbe9625c89f_7.png)

以下是影视推广的核心操作平台：
*   **抖音**
*   **快手**
*   **小红书**
*   **微信视频号**

这些平台都是我们日常会使用到的。影视推广，简单来说，就是为电影、电视剧、综艺或短剧进行线上宣传。制作方或广告商会发布推广任务，创作者接取任务后，制作相关视频并发布到指定平台，最终根据视频的播放量或点赞数来结算奖金。

---

了解了基本概念后，我们来看看具体的操作流程。整个过程是通过完成“任务”来获得收益的。

以下是影视推广任务的基本操作步骤：
1.  **选择任务**：在影视推广平台上，有大量来自爱奇艺、搜狐等平台的任务，每个任务对应具体的影视剧或短剧。
2.  **制作视频**：根据任务要求，使用提供的素材制作推广视频。
3.  **发布视频**：将制作好的视频发布到任务指定的平台（如抖音、小红书、视频号）。
4.  **提交回填链接**：这是结算收益的关键步骤。你需要到任务页面提交一个“回填链接”，其中需填写：
    *   你的微信号、账号昵称或平台ID。
    *   你所做的具体电视剧/短剧名称。
    *   你发布的作品链接。
    *   作品发布时间。
    *   作品播放量/点赞数的截图。
5.  **结算收益**：平台审核你提交的信息后，会根据规则为你结算奖金。

---

明确了操作步骤，接下来我们重点看看在不同平台上的收益计算规则。影视推广的收益通常比单纯靠平台播放量分成的收益更高，并且对账号粉丝数没有要求。

以下是各平台影视推广任务的收益计算方式（以一部剧的最高播放量/点赞作品结算为例）：

**按播放量结算（适用于抖音、视频号、小红书）：**
*   **抖音**：每1万播放量，奖金为 **50元**。
*   **微信视频号**：每1万播放量，奖金为 **10元**。（因用户基数大，单价较低）
*   **小红书**：每1000播放量，奖金为 **30元**。（单价最高）

**按点赞数结算（适用于抖音+快手组合任务）：**
*   **抖音**：单作品5000赞奖金100元，1万赞奖金200元。
*   **快手**：单作品1万赞奖金180元。
*   **注意**：此类任务通常要求将视频同时发布在抖音和快手两个平台。

此外，像搜狐短剧等任务，虽然平台知名度可能不如前者，但提供的单价往往非常有竞争力。

本节课中我们一起学习了影视推广这一重要的自媒体变现方式。我们明确了影视推广是通过接取任务为影视内容做宣传，并掌握了在抖音、快手、小红书、视频号等平台发布任务视频并结算收益的具体流程与规则。记住，关键在于**接取任务**和**正确提交回填链接**，这能让你在涨粉的同时，获得可观的额外收入。
