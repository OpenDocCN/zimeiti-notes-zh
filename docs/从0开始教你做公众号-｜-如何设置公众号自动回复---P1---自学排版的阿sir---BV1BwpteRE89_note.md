# 从0开始做公众号：P1：如何设置公众号自动回复 🎯

在本节课中，我们将学习微信公众号后台“自动回复”功能的设置方法。自动回复能帮助你在无法及时回复粉丝时，自动处理留言，提升互动效率。我们将详细介绍三种自动回复类型及其具体配置步骤。

上一节我们介绍了课程概述，本节中我们来看看如何进入自动回复设置界面。

我们打开公众号后台，在左侧工具栏下滑找到“自动回复”选项。自动回复功能指当粉丝在公众号后台发送私信时，系统可自动回复预设内容。

---

## 1. 被关注回复 👋

被关注回复是用户首次关注公众号时，自动弹出的一条欢迎消息。

以下是设置被关注回复的步骤：
*   点击“被关注回复”。
*   在编辑框中输入欢迎语，或添加图片、音频、视频等内容。
*   **注意**：只能设置一条消息，无法同时发送多种类型内容。
*   编辑完成后，点击“保存”即可。

设置完成后，当有新用户关注，便会收到这条自动回复。

## 2. 收到消息回复 💬

收到消息回复指无论粉丝发送任何私信内容，都会触发的一条通用回复。

这条回复通常可以设置为“感谢您的留言，我们会尽快处理”等提示语。大家可根据自身需求灵活设置。

## 3. 关键词回复 🔑

关键词回复功能最为灵活，可根据粉丝发送的特定关键词，回复不同的内容。

点击“添加回复”开始设置。首先需要输入“规则名称”，这只是一个便于你识别和管理不同回复规则的代号。

**核心概念解析：全匹配与半匹配**
*   **全匹配**：粉丝发送的消息必须与设置的关键词**完全一致**，不能多字、少字或错字。公式可表示为：`用户消息 == 设定关键词`
*   **半匹配**：只要粉丝发送的消息**包含**设定的关键词，即可触发回复。公式可表示为：`设定关键词 in 用户消息`

例如，设置关键词“**十周年会员免费领**”并选择“全匹配”，则用户必须完整无误地发送这整句话才能触发回复。

以下是设置关键词回复内容的步骤：
*   在“回复内容”区域，可以选择回复文字、图片、图文消息等。
*   若要回复已发布的文章，点击“回复以发表内容”，然后选择对应的推文即可。
*   最后点击“保存”。
