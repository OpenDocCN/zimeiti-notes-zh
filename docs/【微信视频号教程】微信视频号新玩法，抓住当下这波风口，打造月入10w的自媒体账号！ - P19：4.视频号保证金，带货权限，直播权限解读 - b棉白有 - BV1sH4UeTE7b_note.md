# 微信视频号教程：P19：视频号保证金、带货权限与直播权限解读 🎬

在本节课中，我们将详细解读微信视频号的保证金、带货权限与直播权限。了解这些规则是开启视频号电商变现的基础，能帮助你明确账号的运营方向与合规要求。

## 带货身份与经营资质

上一节我们介绍了视频号的基础设置，本节中我们来看看账号的带货身份。在创作者中心板块，点击右上角的个人头像，即可查看你的带货身份。

你的带货身份主要分为两种：
*   **达人**
*   **商家**

你可以根据自身情况选择身份。如果选择错误，后期也可以进行修改，这一点无需担心。

关于“经营资质”部分，目前个人用户通常无需特别关注。

## 带货权限详解

了解身份后，我们来看看具体的带货权限。在相关页面中，你会看到三个带货权限板块。

以下是三种带货权限的具体说明：

1.  **视频带货**
    *   **含义**：指在发布的短视频中挂载商品链接，类似于抖音的“小黄车”。用户观看视频时，屏幕上会出现一个橙色的购物车图标，点击即可查看和购买商品。
    *   **开通条件**：此功能要求较高。账号需要满足 **1000个有效粉丝**，且发布的视频必须为**原创内容**。
    *   **注意事项**：对于做内容搬运的账号，开通此权限较为困难。即便开通，一旦发生违规，此权限也可能被永久关闭。权限关闭后，你仍可通过橱窗进行卖货。

2.  **橱窗带货**
    *   **含义**：在视频号主页展示商品橱窗，用户点击进入后可浏览和购买所有上架的商品。这是最基础、最常用的带货方式。
    *   **优势**：开通门槛相对较低，且稳定性较高，是搬运类账号或粉丝量不足账号的主要变现途径。

![](img/aad5f8bcd3ee2c247d2b5ecf8f03c79e_3.png)

3.  **直播带货**
    *   **含义**：在视频号直播过程中，向观众展示和销售商品。
    *   **开通条件**：这是本节课的重点。目前开通直播带货权限主要有两种途径：
        *   **途径一**：账号拥有 **1000个有效粉丝**。满足此条件后，即可在直播间挂载商品。
        *   **途径二**：开通**视频号小店**。开通小店后，即使粉丝数不足1000，也可以在直播间挂载商品。

## 直播权限与实名认证规则

![](img/aad5f8bcd3ee2c247d2b5ecf8f03c79e_7.png)

接下来，我们重点讲解直播带货权限的细节和当前的实名认证规则。

![](img/aad5f8bcd3ee2c247d2b5ecf8f03c79e_9.png)

**关于实名认证**：
根据平台最新规则，虽然一个身份证仍可实名5个微信，但只能用于其中**2个视频号**的实名认证。这意味着，你最多只能拥有2个完成实名认证的视频号。

![](img/aad5f8bcd3ee2c247d2b5ecf8f03c79e_11.png)

**关于直播权限**：
如果你尝试发起直播并添加商品，可能会遇到以下提示：
`为达到直播条件，视频号有效关注1000人才可以进行直播带货。`

![](img/aad5f8bcd3ee2c247d2b5ecf8f03c79e_13.png)

这明确表示，在不满足1000粉丝且未开通视频号小店的情况下，你的直播间无法挂载商品。但请注意，**橱窗带货功能依然可以使用**。

![](img/aad5f8bcd3ee2c247d2b5ecf8f03c79e_15.png)

因此，对于粉丝量不足的新账号，若想进行直播带货，**开通视频号小店**是目前最可行的解决方案。开通小店需要准备营业执照，具体的开店流程和营业执照办理方法，我们将在后续课程中详细讲解。

![](img/aad5f8bcd3ee2c247d2b5ecf8f03c79e_17.png)

## 总结

本节课我们一起学习了视频号的带货身份、三种带货权限以及直播权限的核心规则。

关键点总结如下：
*   带货身份分为**达人**和**商家**，可后期修改。
*   **视频带货**门槛高（需1000原创粉），**橱窗带货**是基础且稳定的方式。
*   **直播带货**需满足 **1000粉丝** 或开通 **视频号小店**。
*   个人目前最多只能实名认证 **2个** 视频号。

![](img/aad5f8bcd3ee2c247d2b5ecf8f03c79e_19.png)

理解这些规则，能帮助你更好地规划账号的成长路径和变现策略。下节课，我们将开始深入讲解如何开通视频号小店。
