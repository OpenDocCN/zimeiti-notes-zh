# 自媒体运营课程：P9：视频剪辑实操（上）🎬

在本节课中，我们将学习如何使用剪映App进行视频剪辑的基础操作，包括如何导入素材、制作视频主题背景、添加文字与配音，以及调整视频比例以适应不同平台的要求。我们将从打开软件开始，一步步完成一个视频片头的制作。

上一节我们添加了背景，本节中我们来看看如何添加主题文字。主题添加方法如下：点击左下角的“文字”选项，然后点击最左边的“新建文本”。文本框随即出现。

![](img/2dcbdff730f7e5f0576fc31f689c23d9_23.png)

现在需要输入视频的主题。我们之前提到，今天要制作的视频主题人物是沈腾。这一点必须记住：如果主题人物明确，所找的视频素材中必须包含他。如果素材中没有，则无法使用。主题人物是沈腾。所有素材中必须有他出现。如果没有，需要重新寻找素材。这一点要记住。

![](img/2dcbdff730f7e5f0576fc31f689c23d9_25.png)

首先输入主题人物“沈腾”。沈腾有自己的综艺场面和影视场面。如果素材与影视相关，就输入“影视”；如果与综艺相关，就输入“综艺”。我们需要区分开来。输入“沈腾的综艺爆笑名场面”。主题名称确定后，可以看到文字是白色的，看起来不太美观。人们通常更喜欢色彩鲜艳、抓人眼球的东西。

方言音选择完成后，直接点击右下角的对勾。软件会自动下载这个音频。

![](img/2dcbdff730f7e5f0576fc31f689c23d9_35.png)

音频下载完成后，从头听一下效果：“沈腾综艺爆笑名场面”。现在声音已经有了。接着手动选择黑色的背景，直接点击一下即可。点击完成后，手动将其缩短一些，使其能够音画同步。主题背景制作完成后，紧接着是重要的部分：处理视频素材本身。

![](img/2dcbdff730f7e5f0576fc31f689c23d9_37.png)

![](img/2dcbdff730f7e5f0576fc31f689c23d9_37.png)

首先检查素材中是否有沈腾这个人。如果有，这个素材就能使用。我们之前讲到，如果素材中没有沈腾，就用不了。这一点要记住。

在讲解过程中，我会边讲边说明细节。首先第一点：我们要制作的视频必须是 **16:9** 的横屏。这是所有平台的硬性标准（抖音除外）。如果不是这个比例，平台审核不会通过。一定要记住这个比例。

**核心比例公式：**
*   通用平台（如B站）：`16:9` （横屏）
*   抖音平台：`9:16` （竖屏）

除了抖音需要9:16的竖屏外，其他平台都是16:9的横屏。这两个比例要记住并区分开，制作视频时要选择正确的比例。

现在来看看这个比例该如何调整。就像我们这个视频素材一样，它明显不是一个16:9的横屏，我们该如何调整呢？首先介绍最简单的一种方式：找到手机下方的“比例”选项，直接点击它。

现在可以看到有一个“16:9”的横屏选项，位置已圈出。找到后直接点击。然后观察素材视频是否随之调整。如果视频画面动了，就可以用这种方式调整比例。

但如果像这个视频一样，点击比例后画面一动不动，又该如何手动调整呢？这样可以增加我们使用素材的几率，让更多素材可用。

手动调整方法如下：首先选择左下角的“剪辑”选项，找到“剪辑”后，选择里面的“编辑”选项。

![](img/2dcbdff730f7e5f0576fc31f689c23d9_39.png)

直接选择“编辑”，然后对画面进行“裁剪”。在裁剪过程中，强调一点：所有视频原有的字幕都不要保留，直接裁掉。我们稍后可以自己添加字幕。在裁剪过程中，包括边框也少留一点。

![](img/2dcbdff730f7e5f0576fc31f689c23d9_39.png)

![](img/2dcbdff730f7e5f0576fc31f689c23d9_41.png)

边框留多了会出现黑色边框，影响美观。少留一点即可。画面裁剪完成后，直接点击右下角的对勾。

![](img/2dcbdff730f7e5f0576fc31f689c23d9_43.png)

![](img/2dcbdff730f7e5f0576fc31f689c23d9_41.png)

现在来看一下，视频比例是否调整了？视频明显放大了，但它仍然不是一个16:9的横屏。如果是的话，就不会有黑色背景了。接下来该如何调整呢？找到手机右下角的“背景”选项。

![](img/2dcbdff730f7e5f0576fc31f689c23d9_43.png)

在这里可以看到“背景”，直接选择它。然后右下角有一个“画布模糊”的选项。

![](img/2dcbdff730f7e5f0576fc31f689c23d9_45.png)

直接选择“画布模糊”，我们调整到第一个清晰的画面。现在再看一下，这是否就是一个16:9的横屏了？

本节课中我们一起学习了使用剪映进行视频剪辑的初步操作：从导入素材、利用素材库制作主题背景，到添加文字、套用模板并生成配音。最后，我们重点讲解了如何将视频比例调整为符合各大平台要求的16:9横屏格式，包括使用比例工具和手动裁剪结合画布模糊的方法。这些是视频剪辑最基础且关键的步骤。
