## 核心概念：核心关键词与长尾关键词
上一节我们介绍了内容创作的基础，本节中我们来看看内容被用户发现的关键——关键词。关键词与我们的行业和产品息息相关。

首先，我们需要理解两个核心概念：**核心关键词**和**长尾关键词**。

![](img/2d50c8be2e4632f5d4db032d88bfcae5_13.png)

*   **核心关键词**：指行业或产品的通用名称，是用户搜索时最直接、最宽泛的词汇。
    *   **公式**：`核心关键词 = 产品/行业通用名`
    *   **例子**：`女装`、`奶粉`、`压缩机`

![](img/2d50c8be2e4632f5d4db032d88bfcae5_15.png)

*   **长尾关键词**：指由核心关键词拓展出来的、更具体、更长的搜索短语。它通常反映了用户更精准的意图或需求。
    *   **公式**：`长尾关键词 = 核心关键词 + 修饰词/需求词`
    *   **例子**：`女装图片`、`女装穿搭`、`女装店`

![](img/2d50c8be2e4632f5d4db032d88bfcae5_17.png)

![](img/2d50c8be2e4632f5d4db032d88bfcae5_19.png)

理解这两个概念，是掌握平台搜索优化（SEO）的基础，尤其在抖音和小红书这类注重搜索流量的平台中至关重要。

2.  **分析衍生需求**：通过工具查看围绕核心词的高频长尾词，如`眉笔颜色`。

## 小红书平台的关键词特殊玩法
在众多平台中，小红书对关键词的依赖尤为明显。接下来，我们探讨一种在小红书笔记中使用的关键词策略。

![](img/2d50c8be2e4632f5d4db032d88bfcae5_37.png)

一篇标准的小红书笔记包含三个要素：**标题**、**文案**和**标签**。

![](img/2d50c8be2e4632f5d4db032d88bfcae5_39.png)

![](img/2d50c8be2e4632f5d4db032d88bfcae5_41.png)

目前，部分运营者会采用一种“关键词堆砌”的玩法。具体操作是：在笔记正文结束后、标签之前，额外添加一段与核心关键词相关、但看似无意义的文字。

![](img/2d50c8be2e4632f5d4db032d88bfcae5_43.png)

**代码示例（模拟堆砌行为）：**
```
#瑞幸咖啡 #咖啡推荐 （这里是正常标签）

瑞幸咖啡 瑞幸拿铁 瑞幸新品 瑞幸点单攻略 瑞幸优惠 瑞幸咖啡好喝吗 瑞幸咖啡推荐 瑞幸咖啡热量
```

![](img/2d50c8be2e4632f5d4db032d88bfcae5_45.png)

**玩法原理：**
1.  平台通过名为 **Spider**（网络爬虫）的程序来抓取和识别笔记内容。
2.  通过在文案中多次重复核心关键词（如“瑞幸咖啡”），可以提高该关键词在笔记中的“词频”。
3.  系统会判断这篇笔记与“瑞幸咖啡”的相关性极高，从而当用户搜索该词时，增加笔记被推荐的概率。

**高级玩法（历史SEO手段）：**
在传统网站SEO中，还有更隐蔽的方法，即通过改变字体颜色（如设为白色），使堆砌的关键词对用户不可见，但依然能被Spider抓取。不过，小红书目前不支持用户自定义字体颜色，此方法不适用。

**请注意：**
*   这种堆砌行为对用户体验不友好，平台可能不鼓励。
*   其效果因产品类目而异，对于知识分享、工具类内容可能有一定作用，但不建议作为主要手段。
*   核心仍应专注于创作优质、自然包含关键词的内容。
