# 自媒体运营课程：P38：剪映核心剪辑功能——帧定格 🎬

在本节课中，我们将要学习剪映中的一个核心剪辑功能——**定格**。我们将了解它的定义、应用场景，并通过一个完整的案例演示其具体操作步骤。

## 概述

定格功能指的是在动态的视频画面中，突然停止在某一帧画面上，形成一个静止的图像。这个功能常用于制作卡点视频或需要突出某一瞬间的特效视频，能让视频的节奏感和视觉冲击力更强。

上一节我们介绍了视频剪辑的基础概念，本节中我们来看看如何利用“定格”功能来增强视频的表现力。

![](img/e3a90851fb035650ceb678680f760774_1.png)

## 定格功能详解

定格功能的核心作用，是将视频的某一帧画面提取出来，生成一段静态的图片片段。这个片段可以独立进行编辑，如调整时长、添加滤镜或特效。

在剪映中，操作定格功能的**代码**逻辑可以概括为：
`选中视频片段 -> 点击“定格”按钮 -> 生成静态画面片段`

![](img/e3a90851fb035650ceb678680f760774_3.png)

## 功能应用案例

为了帮助理解，我们来看一个具体的应用案例。以下是制作一个流行卡点视频的步骤，其中就使用了定格功能。

1.  **导入素材**：在剪映中点击“创作”，导入准备好的视频素材。
2.  **定位时间点**：播放视频，找到你想要进行“定格”的那个瞬间。
3.  **分割视频**：将时间轴移动到该瞬间，点击“分割”工具，将视频在此处切开。
4.  **应用定格**：选中分割点前的视频片段，在底部工具栏中找到并点击“**定格**”功能。
5.  **编辑定格画面**：系统会自动生成一个时长为3秒的静态画面。你可以拖动其边缘调整时长，或为其单独添加滤镜、特效等。

通过以上步骤，你就能在动态视频中插入一个静止的瞬间，从而创造出节奏变化。

## 实操演示

![](img/e3a90851fb035650ceb678680f760774_5.png)

现在，我们通过一个完整的例子来演示如何操作。

首先，我们导入一段拍摄天空的视频。原视频是动态的。

![](img/e3a90851fb035650ceb678680f760774_1.png)

我们希望视频在某个镜头结束时，画面定格并变色，以匹配音乐卡点。

1.  将时间轴移动到视频结束前的一瞬间，使用“分割”工具。
2.  选中分割出来的前一段视频，在二级菜单中点击“**定格**”。
3.  此时，轨道上会生成一个3秒钟的静态画面（源自原视频的最后一帧）。
4.  选中这个定格画面，为其添加一个“牛皮纸”或“褪色”滤镜，改变其颜色和质感。
5.  预览效果，可以看到视频从动态流畅播放，突然变为一张有风格的静态照片，持续几秒后结束或转入下一个镜头。

![](img/e3a90851fb035760ceb678680f760774_7.png)

这个效果就是典型的“定格”应用，它让视频的转场更具创意和视觉停顿感。

![](img/e3a90851fb035650ceb678680f760774_7.png)

## 功能要点总结

以下是关于定格功能需要记住的几个关键点：

*   **自动生成**：点击“定格”后，会自动基于当前选中帧生成一个静态片段。
*   **默认时长**：生成的静态画面默认时长为3秒，但这个时长是可以自由拖拽调整的。
*   **独立编辑**：生成的定格片段可以像普通图片或视频一样，被单独添加动画、滤镜、特效等。
*   **应用场景**：主要用于卡点视频、强调某一瞬间、制作回忆相册效果或创意转场。
