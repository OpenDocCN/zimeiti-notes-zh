# 新媒体运营：P10：小红书平台笔记审核规则 📝

![](img/5bfea1dbaab6642b5854fd8eef138095_0.png)

在本节课中，我们将要学习小红书平台的核心机制之一：笔记的审核与流量分发规则。了解这套规则，是确保你的内容能够顺利发布并获得推荐流量的基础。

![](img/5bfea1dbaab6642b5854fd8eef138095_2.png)

上一节我们介绍了小红书平台的基本概况，本节中我们来看看笔记从发布到推荐给用户的具体流程。

## 笔记审核的三个阶段 🔍

小红书平台在笔记发布后，会经过三个主要的审核阶段。大多数没有问题的笔记，在机器审核阶段即可通过。

### 1. 机器审核

笔记首先会被上传至审核系统。该系统会根据预设的基本规则进行初步筛查。

以下是机器审核的主要规则：

*   **敏感词检测**：系统会识别笔记文案中的违禁词汇。
*   **敏感图片检测**：系统会分析图片内容，识别违规或低质图片。
*   **初步质量判断**：系统会评估笔记的基础质量，例如文案字数、广告词出现频率、图片清晰度以及图片内文字/物体的可识别性。

如果笔记在此阶段被判定为**严重违规**，可能会被直接封禁。如果系统**无法准确判断**（即疑似违规），则会标记该笔记并转入下一阶段。

### 2. 去重检测

![](img/5bfea1dbaab6642b5854fd8eef138095_4.png)

通过机器审核的笔记，会进入去重检测阶段。系统会将你的笔记与平台数据库中的现有内容进行比对。

![](img/5bfea1dbaab6642b5854fd8eef138095_6.png)

去重检测主要比对以下方面：

*   **图片相似度**：检查图片是否为直接抄袭或复制他人。
*   **文案重复率**：计算文案内容、段落在数据库中的重复比例。
*   **整体相似度**：综合评估笔记的整体原创性。

如果笔记被检测为**高度重复或抄袭**，将被直接处理。创作者可以通过调整图片（如裁剪、加滤镜）、重组或扩写文案段落等方式，降低被系统识别为重复的风险。

### 3. 人工审核

此阶段主要对机器标记的“疑似违规”内容进行二次复核。由于内容量巨大且人工成本高，并非所有笔记都会进入此环节。

人工审核在特定领域力度较强：

*   **政治相关**内容。
*   **色情擦边**内容。
*   **法律严禁**的其他内容。

目前，小红书的算法已较为完善，机器误判概率较低。若笔记触发红线，**通常会被直接处理，较少进行人工复核**。如果创作者认为自己的笔记被误判，可以通过平台申诉渠道进行反馈。但请注意，**新号前期应尽量避免任何违规**，因为违规记录可能对账号产生隐形的负面影响。

## 账号权重与流量分发 ⚖️

在审核通过后，系统会分析笔记特征并将其匹配给可能感兴趣的用户。这个过程与你的**账号权重**密切相关。

![](img/5bfea1dbaab6642b5854fd8eef138095_8.png)

账号权重是系统根据账号的综合表现（非公开数据）给出的一个隐形的“分数”，它直接影响笔记获得的初始流量池大小。

我们可以将权重简单理解为从0到10的分值：

*   **新号（低权重期）**：新注册的账号权重接近0。此阶段笔记的初始推荐量通常较低（如几十个阅读）。因为系统尚未给账号打上精准标签，笔记无法被有效推送给潜在兴趣用户，导致互动数据难以提升。
*   **成熟号（高权重期）**：当账号持续发布垂直、优质内容后，系统会在相应领域给予更高的权重。高权重账号发布的笔记，通常能获得更大的初始流量池和更精准的推荐。

**核心建议**：在新号阶段（低权重期），重点应放在“养号”和建立垂直度上，避免违规，通过稳定的优质内容积累来逐步提升账号权重，而不是追求单篇笔记的爆款。

## 总结与回顾 📚

本节课中我们一起学习了小红书笔记审核与分发的核心机制。

1.  **审核流程**：笔记需依次通过**机器审核**、**去重检测**，部分疑似违规内容会进入**人工审核**。
2.  **核心规则**：机器审核关注**敏感词、敏感图片和基础质量**；去重检测打击**抄袭和高度重复内容**。
3.  **账号权重**：这是一个影响笔记初始流量的隐形系统评分。**新号权重低，需通过持续发布垂直优质内容来提升**。
4.  **重要原则**：新号运营初期，**严格遵守平台规则、避免违规**是重中之重，这能为账号的长远发展打下良好基础。

![](img/5bfea1dbaab6642b5854fd8eef138095_10.png)

理解并遵循这些规则，是让你的小红书笔记获得曝光、吸引粉丝的第一步。
