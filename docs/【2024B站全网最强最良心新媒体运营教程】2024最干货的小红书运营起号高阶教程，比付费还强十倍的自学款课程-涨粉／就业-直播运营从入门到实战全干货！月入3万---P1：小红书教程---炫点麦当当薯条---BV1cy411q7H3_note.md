## 第一步：开店 🏪

上一节我们介绍了项目的整体逻辑，本节中我们来看看如何迈出第一步——开店。

开店流程本身并不复杂。首先，打开小红书APP，点击左上角的菜单图标。

![](img/7d5be4b6fd4a90f71aff04a230191b06_5.png)

接着，选择“创作中心”，然后点击“更多服务”选项，找到并选择“店铺”，最后点击“立即入住”即可。按照官方要求提交资料完成入住。有人将小红书开店包装成项目出售，并且有不少人购买，这很奇特。

以下是小红书店铺的两种主要形式：

*   **个人店**：分为两种。
    *   **个人店**：仅需身份证和**1000元保证金**即可开通。
    *   **个体工商户店**：除身份证外，还需一个营业执照。
*   **企业店**：分为三种（普通企业店、专卖店、旗舰店）。均需要营业执照，并需缴纳**600元年费**。

![](img/7d5be4b6fd4a90f71aff04a230191b06_7.png)

对于初学者，直接选择**个人店**即可。实测表明，平台并未对企业店有流量倾斜。唯一的区别是，个人店只能开一家，而一份营业执照可以开三个店。如果一个店都没做好，开再多店也无用。

![](img/7d5be4b6fd4a90f71aff04a230191b06_7.png)

小红书电商不适合以“店群”形式操作，因为其电商自然流量占比非常低。铺再多的店和货也用处不大。所以，任何鼓吹“小红书店群”的说法都不可信。小红书要做的是**账号矩阵**，而非店铺矩阵，这一点将在流量部分详细说明。

开店成本主要包括以下几项：
1.  **1000元保证金**（不做可退）。
2.  **平台扣点**，约为营业额的 **5%**。
3.  **支付渠道费用**，每单约 **0.7%**。
4.  **自有流动资金**。初期可采用无货源模式，资金要求不高，准备2000-3000元完全足够。

## 第二步：选品 📦

开完店后，下一步就是决定卖什么，即选品。选品其实很简单，核心逻辑是：**在其他平台上的爆款，大概率在小红书上也能爆**。

目前推荐通过以下三个渠道进行选品：

1.  **某东、某宝**：寻找符合小红书平台调性的爆款产品。例如，有电商朋友将其某宝店铺的爆款搬到小红书，第一篇笔记就出单，第二篇笔记单量破百。
2.  **某音精选联盟**：查看某音精选联盟的排行榜，了解现阶段哪些商品卖得好。只要产品符合小红书用户调性，直接搬运测试即可。
3.  **某多多销量排行榜**：选择适合小红书的类目排行榜，从中挑选爆品搬运到小红书。

整个选品的核心思想是：**利用市场来帮助我们选品，而不是自己瞎想**。我们要借助大卖家的选品能力，弥补自身不足。只需判断排行榜上的爆品是否可能与小红书用户匹配，如果匹配，就搬过来测试。这相当于借助别人的能力为自己选品，是最快捷、最稳妥的方式，成功率远高于盲目测试。

## 第三步：获取流量 📈

产品上架后，就进入最重要的环节——获取流量。小红书是种草平台，店铺九成以上的流量都来自发布的笔记。这就是为什么之前强调不要做店群，而要做**账号矩阵**，尽可能多地发布笔记来获取流量。

其通路是：**用户通过笔记 → 对产品产生兴趣 → 进入店铺完成购买**。堆砌一堆店铺的效果，远不如多发几篇优质笔记。

因此，**如何在小红书上发布笔记，是电商成功的关键**。小红书用户群体认知较高，直接发布硬广效果很差，无法获得信任，更谈不上转化。发笔记时，一定要从**素人（用户）角度**切入，做种草内容，而不是官方视角。

![](img/7d5be4b6fd4a90f71aff04a230191b06_9.png)

![](img/7d5be4b6fd4a90f71aff04a230191b06_9.png)

制作有效的种草笔记其实很简单：去各大平台（如某音、某宝、某东）的商品评论区和买家秀中，寻找好看的素材直接使用。文案则基于买家评价进行修改。这些评价是用户的真实使用体验，天然适合用作种草笔记，且取之不尽。

![](img/7d5be4b6fd4a90f71aff04a230191b06_11.png)

![](img/7d5be4b6fd4a90f71aff04a230191b06_11.png)
