# 视频号运营：P30：如何制作直播间贴图 🎨

在本节课中，我们将学习如何为视频号直播间制作一个透明的信息贴图。这种贴图常用于展示商品信息、促销活动或直播间主题，能有效吸引观众注意并传递关键信息。

上一节我们介绍了直播间的整体设置，本节中我们来看看如何制作一个专业的直播间贴图。

## 工具准备

制作贴图需要使用一款名为“微麦水印相机”的手机应用。该应用原名“微商水印相机”，可在手机应用商店搜索下载。使用其免费功能即可完成制作。

## 制作步骤详解

以下是使用“微麦水印相机”制作透明贴图的具体步骤。

1.  **打开应用并进入水印模板库**
    打开“微麦水印相机”应用，在主界面找到并点击“水印库”。在弹出的会员开通提示上，选择“暂不开通”。接着，点击“更多水印”进入“水印模板”页面。

2.  **选择模板并编辑文字**
    在模板库中，找到“横竖文字”分类，选择第一个或任意喜欢的模板。点击模板进入编辑界面，将示例文字替换为你需要的内容。例如，可以编辑为：
    `好物推荐`
    `老国货品牌`
    `9.9元/1盒`
    `19.9元/3盒`
    `29.9元/5盒`

3.  **自定义样式**
    你可以自由调整文字的**颜色**和**字体**，使其更符合直播间的视觉风格。点击对应选项进行选择即可。

![](img/d7eef3abac064aeb4f54ceb99d551e18_1.png)

4.  **关键设置：背景透明化**
    这是制作透明贴图的核心步骤。找到“背景色”设置选项，**必须选择“无背景”或带有斜杠标识的透明背景**。确保背景色的不透明度调整为0。这一步的目的是让贴图背景完全透明，使其在直播间中只显示文字，而不会遮挡直播画面。

![](img/d7eef3abac064aeb4f54ceb99d551e18_3.png)

5.  **生成并保存图片**
    所有设置完成后，点击“生成”按钮。系统会处理图片，请务必点击“保存到相册”，将制作好的透明PNG格式贴图保存到手机中。

![](img/d7eef3abac064aeb4f54ceb99d551e18_5.png)

## 在直播间添加贴图

![](img/d7eef3abac064aeb4f54ceb99d551e18_7.png)

贴图制作完成后，需要在直播时将其添加到画面中。

![](img/d7eef3abac064aeb4f54ceb99d551e18_9.png)

1.  在视频号中发起直播。
2.  进入直播画面后，找到并点击“贴图”或“添加图片”功能按钮（图标通常为“🖼️”）。
3.  选择“从手机相册选择”。
4.  从相册中选取你刚刚制作并保存好的透明贴图。
5.  贴图会出现在直播画面上，你可以用手指拖动来调整其位置，并用双指缩放来调整其大小，直到满意为止。

## 贴图修改与优化

![](img/d7eef3abac064aeb4f54ceb99d551e18_11.png)

![](img/d7eef3abac064aeb4f54ceb99d551e18_12.png)

如果觉得贴图文字大小或样式不合适，可以随时返回“微麦水印相机”进行修改。

*   重新打开之前保存的项目或使用相同模板新建。
*   在编辑界面，你可以调整**字体大小**、**字间距**和**行距**等参数。
*   修改完毕后，再次生成并保存新图片，然后在直播间中替换旧的贴图即可。

![](img/d7eef3abac064aeb4f54ceb99d551e18_14.png)

本节课中我们一起学习了如何利用“微麦水印相机”制作透明直播间贴图。核心流程可总结为：**选择模板 → 编辑文字 → 设置透明背景 → 保存图片 → 直播时添加**。掌握这个简单的技能，就能让你的直播间看起来更专业、信息更清晰，从而提升观众的观看体验。
