# 抖音运营从0~1全攻略：P14：11、视频类型1-2

在本节课中，我们将要学习抖音内容创作中一个至关重要的环节——**人设**的建立，并深入探讨第一种核心视频类型：**口播**。理解人设是连接内容创作与粉丝增长、商业变现的关键桥梁。

## 人设：从被记住到被喜欢

上一节我们解决了文案问题，本节中我们来看看下一个环节：内容。从涨粉到变现并非一条直线，中间还有一个关键环节，即建立你的**人设**。

建立人设的目的不是为了炫耀，而是为了在众多同类创作者中脱颖而出。例如，有十个人都做装修内容，如果其中一位创作者建立了鲜明有趣的人设，就更容易被观众记住。当观众喜欢上你这个人时，他们自然就成为你的粉丝。这就是人设的作用。

人设主要分为两种：**强人设**与**弱人设**。区分这两种人设后，你就能快速判断任何视频的类型。

### 强人设

强人设是指以创作者**个人**为切入点的内容。这类内容与创作者自身密切相关。

以下是强人设的常见切入点：

*   **个人职业与经验**：分享所在行业的专业知识与经验。例如：装修、办公、房产、汽车、餐饮、婚介、法律咨询、电商等领域。
*   **个人兴趣与爱好**：分享自己的生活爱好。例如：美食、旅行、手工、宠物、占卜、美妆、护肤、服装穿搭、游戏、摄影等领域。

强人设内容通常分享创作者的工作、生活或爱好，并且大部分需要**真人出镜**，因为内容与“你”这个人直接相关。

### 弱人设

![](img/21f57cd13493a877f28b03959702181e_2.png)

弱人设是指内容与创作者**自身关系不大**，主要目的是为了运营好账号。

以下是弱人设的常见例子：

*   **情感语录类**：发布情感文案配乐视频，无真人出镜或配音。
*   **影视解说类**：虽然有人声解说电影，但重点在电影内容，与解说者本人特质关联弱。

区分清楚人设的强弱，是规划后续内容方向的基础。

## 内容创作方向：从讲故事开始

明确了人设概念后，我们来看具体的内容创作方向。只要内容与你个人相关，都可以作为视频素材。首先，我们从最基础且有效的一种类型讲起：**讲故事**。

你可以分享客户故事、从业经历或人生感悟。故事本身不要求完全真实，可以从文案平台寻找有趣的故事进行分享。

讲故事最常见的形式就是**口播**。口播即用嘴巴讲述，面对镜头进行表达。

### 口播的主要类型与价值

口播有多种分类，核心价值在于提供情绪共鸣或知识收获。

以下是常见的口播类型：

*   **讲故事型口播**：例如情感、养生、育儿等领域的故事分享。目的是让观众觉得内容有趣、有共鸣。
*   **教知识型口播**：例如金融、财经、装修、房产、汽车等领域的知识科普。目的是让观众在几十秒内获得“收获感”，觉得学到了东西。

### 关于口播的常见问题解答

许多同学对口播有顾虑，以下是针对常见问题的解答：

*   **口才不好能做口播吗？**  
    **可以**。口播是依据提前准备好的稿子进行录制，并非临场发挥。大部分口播者的流畅表达都源于充分准备。
*   **普通话不好能做口播吗？**  
    **可以**。如果你的目标受众是同城用户，使用方言可能效果更好，能增加亲切感和趣味性。同时，可以通过添加字幕确保信息传达。
*   **口播难拍吗？**  
    **不难**。口播拍摄门槛低，一部手机固定好位置即可开始。关键在于准备好优质的文案脚本。

本节课中我们一起学习了抖音运营中“人设”的概念及其重要性，并详细介绍了第一种视频类型——口播。我们明确了强人设与弱人设的区别，并了解了口播的多种类型与拍摄要点。理解这些基础概念，是迈出内容创作第一步的关键。
