# 机器学习入门：P1：核心概念与线性回归

在本节课中，我们将学习机器学习的基础概念，并通过一个简单的线性回归例子来理解其工作原理。我们将从核心术语开始，逐步构建一个预测模型。

---

## 什么是机器学习？🤖

机器学习是人工智能的一个分支。它让计算机能够从数据中学习规律，而无需被明确编程。

上一节我们介绍了机器学习的定义，本节中我们来看看它的核心目标。机器学习的核心目标是**从数据中学习一个模型**。这个模型可以用于预测新数据或做出决策。

---

## 核心概念解析 🧠

理解机器学习，需要掌握以下几个基本概念。

以下是机器学习中几个关键术语的解释：

*   **特征**：描述数据的属性或变量。例如，预测房价时，房屋面积、卧室数量就是特征。
*   **标签**：我们想要预测的目标值。在房价例子中，房价本身就是标签。
*   **训练数据**：用于“教导”或训练模型的数据集，包含特征和对应的标签。
*   **模型**：从训练数据中学到的函数或规则。它建立了从特征到标签的映射关系。
*   **预测**：将新的、未见过的特征数据输入模型，得到估计的标签值。

---

## 一个简单的例子：线性回归 📈

为了将抽象概念具体化，我们来看一个最基础的机器学习算法——线性回归。

上一节我们了解了机器学习的组成部分，本节中我们来看看它们如何在一个具体算法中协同工作。线性回归试图找到特征和标签之间最佳的线性关系。

假设我们想根据房屋面积预测房价。我们认为两者存在线性关系，即：

**房价 = w × 面积 + b**

在这个公式中：
*   `面积` 是特征。
*   `房价` 是标签。
*   `w` 是权重，代表面积对房价的影响程度。
*   `b` 是偏置，代表基础房价。

我们的目标是找到最合适的 `w` 和 `b`，使得这条直线能最好地拟合我们所有的训练数据点。

---

## 如何“学习”？🔄

模型是如何自动找到最佳的 `w` 和 `b` 的呢？这个过程称为“训练”。

以下是训练一个线性回归模型的基本步骤：

1.  **初始化参数**：随机设置 `w` 和 `b` 的初始值（例如，`w=0`, `b=0`）。
2.  **做出预测**：使用当前的 `w` 和 `b` 以及训练数据中的特征，计算预测的房价。
    ```python
    # 示例代码
    预测房价 = w * 房屋面积 + b
    ```
3.  **计算损失**：比较预测房价和真实的标签（房价），计算所有数据点的平均误差。常用的损失函数是均方误差。
    ```python
    # 示例代码：均方误差
    损失 = (1 / 数据量) * sum((真实房价 - 预测房价) ** 2)
    ```
4.  **优化参数**：通过梯度下降等优化算法，根据损失值调整 `w` 和 `b`，目标是让损失最小化。
5.  **重复迭代**：重复步骤2到4，直到损失不再显著下降或达到预设的迭代次数。此时得到的 `w` 和 `b` 就是模型学习到的参数。

本节课中我们一起学习了机器学习的基础。我们首先定义了机器学习，并解释了特征、标签、模型等核心概念。接着，我们以线性回归为例，详细阐述了模型如何通过初始化参数、计算预测、评估损失和迭代优化来完成“学习”过程，最终得到一个可用于预测的线性方程。

理解这个简单的流程，是进入更复杂机器学习世界的重要第一步。
