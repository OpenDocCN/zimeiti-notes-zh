# 小红书运营教程：P9：05.小红书后台的基础操作（手机端）📱

![](img/6104e3b08574a6fda1c34b3ca07eed9e_0.png)

在本节课中，我们将学习如何使用手机端的小红书商家后台——千帆系统。我们将从下载应用开始，逐步介绍客服管理、店铺信息设置、商品发布、营销工具以及支付与保证金等核心功能，帮助你快速上手店铺的基础操作。

## 系统下载与登录

首先，我们需要在手机应用商店搜索并下载“小红书千帆”应用。这是小红书官方的商家后台管理系统。

下载完成后，打开应用并登录你的店铺账号。登录成功后，你会看到后台的主页界面。

## 客服与消息中心

上一节我们介绍了如何进入系统，本节中我们来看看如何管理客户咨询。

在主页上，你可以找到“消息”入口。点击进入后，这里是你的客服接待中心。所有买家关于店铺的咨询、通知以及平台发送的违规提醒都会在这里显示。你可以在此直接回复买家的问题。

以下是消息中心的主要功能：
*   **客服接待**：实时回复买家的咨询。
*   **通知查看**：接收平台关于订单、违规等各类通知。
*   **客服配置**：可以进行相关的客服设置。

## 店铺初始化与工具概览

了解客服功能后，我们需要对店铺进行基础设置。回到首页，你会看到“去初始化”的提示和“全部工具”的入口。

“去初始化”引导你完成店铺的基础信息填写。而“全部工具”则集成了店铺运营的所有核心功能。

点击“全部工具”，你将看到以下功能模块：
*   **发布作品/商品**：用于发布新的商品或营销笔记。
*   **管理商品/订单**：管理已上架的商品和处理全部订单。
*   **数据中心**：查看店铺的经营数据。
*   **店铺装修**：设置店铺的首页展示。
*   **营销工具**：创建优惠券等活动。

## 核心设置：发货地址与运费模板

完成基础信息填写是运营的第一步，其中最关键的是设置发货地址和运费规则。

你需要首先填写准确的发货地址。路径通常在“成长”任务或“地址管理”中。即使你做一件代发，也需要填写一个有效的地址。

**地址填写公式**：`联系人 + 所在地区 + 详细地址 + 手机号码`

**重要提示**：在“留给买家留言”部分，**不要**填写“联系客服”。必须填写具体地址，否则平台审核可能无法通过。

地址设置完成后，需要配置运费模板。你可以创建不同的模板以适应不同的物流策略。

以下是设置运费模板的示例逻辑（伪代码）：
```plaintext
如果 地区 属于 [新疆, 西藏]：
    设置 为 “不配送”
否则 如果 地区 属于 偏远地区：
    设置 为 “买家承担运费”
否则：
    设置 为 “全国包邮”
```
系统有默认模板，但建议根据你的商品利润和物流成本进行自定义设置，例如为偏远地区设置运费或选择不配送。

## 营销工具：优惠券设置

店铺设置好后，我们可以利用营销工具吸引顾客。优惠券是一个非常重要的促销手段。

在“营销”或“优惠券”板块，你可以创建多种类型的优惠券来促进销售。

以下是你可以创建的优惠券类型：
*   **店铺通用券**：全店商品均可使用。
*   **指定商品券**：仅限特定商品使用。
*   **店铺新人券**：针对首次进店用户发放。

创建后，你可以在后台管理所有优惠券的状态，如“生效中”、“已失效”等。

## 支付开通与保证金缴纳

最后，为了确保店铺能正常收款并长期运营，必须完成支付方式的绑定和保证金的缴纳。

![](img/6104e3b08574a6fda1c34b3ca07eed9e_2.png)

在“支付”或“账户”设置中，按照指引绑定你的微信支付和支付宝。这样买家才能通过这两种方式付款。

同时，根据你经营的类目不同，需要缴纳相应金额的**保证金**。保证金是平台对商家的一种履约担保，建议打算长期经营的商家及时缴纳。支付保证金后，店铺功能会更完整。
