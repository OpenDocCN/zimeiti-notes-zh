# 抖音运营教程：P47：视频创意与素材获取

![](img/4e98a849ae386fda61196acfe4869c35_1.png)

在本节课中，我们将学习如何为视频寻找创意和获取素材。我们将不再讨论脚本套路，而是专注于当你不自己拍摄素材时，如何制作视频。课程将介绍一个实用的工具来帮助你确定选题，并讨论素材处理的相关注意事项。

## 视频创意的来源

![](img/4e98a849ae386fda61196acfe4869c35_3.png)

无论是搬运视频还是原创视频，第一步都是内容策划，这包括选题和素材收集。本节中，我们来看看如何找到视频的创意，也就是确定选题。

![](img/4e98a849ae386fda61196acfe4869c35_5.png)

![](img/4e98a849ae386fda61196acfe4869c35_7.png)

选题法可以帮助我们确定方向，例如上节课介绍的九宫格选题法。但首先需要确定一个大的主题，例如是讲述励志故事还是产品故事。

![](img/4e98a849ae386fda61196acfe4869c35_9.png)

当你自己没有想法，或者想要蹭热点时，可以从外部获取创意。以下是一个推荐的创意网站。

## 创意工具：5118网站

5118网站是一个热词搜索与大数据分析网站，可以帮助我们挖掘热门话题和用户需求。

![](img/4e98a849ae386fda61196acfe4869c35_11.png)

1.  **访问网站**：在浏览器中搜索并打开5118网站。其首页包含多个功能模块，对于新媒体运营，我们主要使用“需求图谱”和“素材搜索”功能。

![](img/4e98a849ae386fda61196acfe4869c35_13.png)

2.  **使用需求图谱**：当你不确定具体写什么，只有一个大方向（例如“励志”）时，可以在搜索框输入关键词。网站会生成一个“需求图谱”，展示与该词关联的高频词汇和用户需求。例如，搜索“励志”后，系统会显示“励志故事”、“励志名言”等词频较高的关联词，这些都可以作为具体的选题方向。

![](img/4e98a849ae386fda61196acfe4869c35_15.png)

![](img/4e98a849ae386fda61196acfe4869c35_17.png)

3.  **拓展思维**：网站还提供放射图、关系脑图、疑问词脑图等可视化工具，帮助你从一个核心词联想出更多相关的高频词汇和具体需求，从而拓展创作思路。

4.  **查看数据**：页面下方会显示相关主题的视频数量等数据（部分高级数据导出功能可能需要会员）。你可以通过这些信息判断某个方向的受欢迎程度。

![](img/4e98a849ae386fda61196acfe4869c35_19.png)

5.  **素材搜索**：除了创意，该网站的“素材搜索”功能还可以在全网搜集与关键词相关的文章等内容，为你的视频文案提供素材参考。它甚至提供“智能原创”和“原创度检测”功能，帮助避免直接的搬运。

![](img/4e98a849ae386fda61196acfe4869c35_21.png)

**核心工具公式**：`视频创意 = 5118网站(关键词) -> 需求图谱分析 -> 高频关联词/用户需求`

这个网站是确定视频大主题和关联创意的有效工具。确定方向后，你可以去其他免费素材网站寻找对应的视频或图片素材。

![](img/4e98a849ae386fda61196acfe4869c35_23.png)

## 关于素材处理与搬运的注意事项

上一节我们介绍了如何获取创意，本节中我们来看看处理现有素材时需要注意的关键问题，特别是“二次创作”的界限。

![](img/4e98a849ae386fda61196acfe4869c35_25.png)

“二次创作到何种程度才不算搬运？”这个问题没有绝对的标准答案。平台的审核机制结合了机器审核和人工审核，存在一定的主观性和不确定性。

以下是一些重要的注意事项：

1.  **审核的不确定性**：同样的视频内容，在不同账号发布，可能产生不同的审核结果。这通常与流量达到一定量级后人工审核的介入有关。
2.  **核心处理原则**：尽量保留原素材的核心内容，但加入自己的观点、解说或文案。对视频进行深度处理，例如镜像翻转、调整播放速度、重新剪辑叙事顺序、更换背景音乐、添加自己的字幕和贴纸等。
3.  **规避风险**：目的是让机器审核难以识别出直接的搬运行为。需要特别注意清除原素材中的第三方广告商标、水印等明显营销信息，否则可能被判定为营销推广。
4.  **长远建议**：随着平台规则日益严格，不建议直接对现有视频进行简单处理后发布。更可持续的方法是，利用创意工具确定方向，然后寻找多源素材进行整合与深度再创作。

![](img/4e98a849ae386fda61196acfe4869c35_27.png)

![](img/4e98a849ae386fda61196acfe4869c35_29.png)

本节课中，我们一起学习了如何利用5118这类工具为视频寻找创意选题，并了解了处理外部素材、避免搬运风险的核心原则。掌握这些方法，可以帮助你在不拍摄原创素材的情况下，也能高效地策划和制作出符合平台要求的视频内容。
