# 剪映教程：P7：剪映核心剪辑功能——变声功能 🎤

在本节课中，我们将要学习剪映App中的一个核心功能——变声功能。这个功能可以帮助你改变视频中的人声，无论是为了优化音质、创造趣味效果，还是模拟多人对话场景，都非常实用。

很多初学者在制作短视频时，可能会觉得自己的原声不够理想。此外，在制作包含多人对话的视频时，也需要对声音进行处理。剪映的变声功能可以轻松将男声变为女声、女声变为男声，或者转换成大叔、萝莉等特色音效，常见于搞笑或玩梗类视频中，用以实现多人对话或增强娱乐效果。

上一节我们介绍了其他剪辑功能，本节中我们来看看如何具体使用变声功能。

## 实操步骤

![](img/4fcc3f5aee15ff0e43770ff24d31355f_1.png)

以下是使用剪映变声功能的详细步骤。

![](img/4fcc3f5aee15ff0e43770ff24d31355f_3.png)

1.  **导入视频素材**
    打开剪映App，导入一段需要处理的视频素材。

    ![](img/4fcc3f5aee15ff0e43770ff24d31355f_1.png)

2.  **选中素材并找到变声功能**
    在时间轴上选中该视频素材，底部会出现二级工具栏。在工具栏中找到并点击 **“变声”** 功能。

    ![](img/4fcc3f5aee15ff0e43770ff24d31355f_3.png)

3.  **选择变声效果**
    点击后，界面会提供多种预设的变声效果，例如“大叔”、“萝莉”、“女生”、“男生”、“怪物”等。你可以点击不同的效果进行试听，实时预览变声后的结果。

4.  **应用并确认**
    选择你喜欢的效果后，该效果会自动应用到视频的音频上。你可以再次播放视频，确认变声效果是否符合预期。

    ![](img/4fcc3f5aee15ff0e43770ff24d31355f_5.png)

## 功能应用与价值

![](img/4fcc3f5aee15ff0e43770ff24d31355f_5.png)

简单来说，变声功能的核心操作是：`选中视频 -> 点击底部“变声” -> 选择效果`。

掌握变声功能，不仅能解决创作者对原声音质不满意的尴尬，更能为视频创作打开新思路。你可以利用它，让一个人声演绎出多个角色，例如同时扮演爸爸和妈妈、儿子和妈妈，或者模拟情侣间的对话。这极大地降低了制作门槛，方便新手用户提升视频的音频趣味性和互动效果。

本节课中我们一起学习了剪映的变声功能。我们了解了它的应用场景，并通过一步步的操作演示，学会了如何为视频素材添加并切换不同的变声效果。利用好这个功能，可以有效增强你视频的表现力和娱乐性。
