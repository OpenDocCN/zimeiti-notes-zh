# 抖音运营孵化：P14：11、视频类型1-2 - 人设与内容构建 🎬

在本节课中，我们将要学习抖音账号运营中一个至关重要的环节：如何构建视频内容。我们将深入探讨“人设”的概念，区分强人设与弱人设，并介绍一种核心的视频内容形式——口播。掌握这些知识，将帮助你明确账号方向，创作出更具吸引力的视频。

上一节我们解决了文案问题，本节中我们来看看内容构建的下一个关键环节。从涨粉到变现并非一条直线，中间有一个不可或缺的环节，那就是建立你的“人设”。人设不是为了炫耀，而是为了在众多同类账号中脱颖而出。例如，有十个人都做装修内容，但其中一位小王同学通过建立搞笑的人设，就能让观众迅速记住他，从而在后续积累粉丝、获得喜爱。

## 理解人设：强与弱

首先，我们来学习这个核心知识点。人设主要分为两种类型：强人设和弱人设。区分清楚这两种人设，你就能快速判断任何视频的类型。

**强人设**是指以创作者个人为切入点的内容。它必须与创作者自身密切相关。

![](img/c8432297e08ca881b671457b355ef08f_1.png)

以下是强人设常见的切入点：

*   **个人经验与专业**：分享你的行业经验，例如装修、办公、房产、汽车、餐饮、婚介、法律咨询、电商等领域的专业知识。
*   **个人兴趣与生活**：分享你的兴趣爱好与日常生活，例如美食、旅行、手工、宠物、星座占卜、美妆护肤、服装穿搭、游戏、摄影等。

![](img/c8432297e08ca881b671457b355ef08f_3.png)

强人设视频大部分需要真人出镜，因为内容与创作者本人直接相关。

**弱人设**则与创作者个人关系不大，核心目的是运营好账号本身。

以下是弱人设的典型例子：

*   **情感语录**：配乐视频加上文字语录，无真人出镜或配音。
*   **影视解说**：虽然有人声解说电影，但焦点是电影内容，而非解说者本人。

## 内容构建：从故事到口播

理解了人设，我们就可以探讨具体的内容形式。只要内容与你个人相关，都可以作为视频素材。我们首先介绍一种非常有效且通用的形式：讲故事，也就是**口播**。

口播即用嘴巴讲述内容。它涵盖多种领域，目的是通过分享吸引观众。

以下是常见的口播分类：

*   **故事型口播**：分享客户故事、从业经历或人生感悟。目的是让内容听起来有趣、有共鸣。
*   **知识型口播**：分享育儿、金融、养生、装修、房产、汽车等领域的专业知识。目的是让观众在几十秒内获得“收获感”，觉得学到了东西。

口播适合绝大多数领域，如法律、保险、金融、装修知识科普等。它的拍摄门槛较低。

以下是关于口播拍摄的几点说明：

1.  **拍摄简便**：只需用手机固定拍摄即可。
2.  **依赖文案**：关键在于提前从文案平台准备好讲述的稿子。
3.  **不要求口才**：口播是依据稿子录制，并非临场发挥。
4.  **不要求普通话**：对于想吸引同城流量或有实体店的创作者，使用方言可能更亲切、更有趣。观众可以通过字幕理解内容。

本节课中我们一起学习了构建抖音视频内容的基础。我们明确了“人设”的重要性，学会了区分**强人设**与**弱人设**，并详细了解了**口播**这种核心视频形式及其优势。记住，清晰的人设是连接内容与粉丝的桥梁，而口播是实践强人设、分享价值的有效工具。在接下来的课程中，我们将继续学习其他内容类型和具体的拍摄方法。
