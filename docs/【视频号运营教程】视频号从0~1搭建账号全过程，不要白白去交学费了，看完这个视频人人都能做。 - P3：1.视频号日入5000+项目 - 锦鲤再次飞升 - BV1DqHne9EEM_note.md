## 概述

在本节课中，我们将学习视频号从零到一的完整搭建过程。视频号是微信旗下的短视频平台。如果你错过了淘宝、微博、公众号、微商、自媒体、抖音、快手和社群电商等利用信息差赚钱的机会，那么视频号很可能是当前时代，普通人进入短视频行业赚钱的最后一个风口。

许多人知道这个机会但不敢尝试，原因是不了解这个行业是否容易操作。本节课将剖析视频号行业，并分享一些行业内的实战经验。

以下是本节课的三个核心板块内容安排。
