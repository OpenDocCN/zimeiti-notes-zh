# 小红书运营教程：P30：小红书笔记引流：平台流量分发机制 🚀

![](img/464ef457d1324c5ed7e6317b9e6a68cc_1.png)

在本节课中，我们将要学习小红书平台的核心流量分发机制。了解这套机制是提升笔记曝光和排名的关键。我们将拆解其算法逻辑，并解释如何通过优化内容来获得更多流量。

---

上一节我们介绍了账号定位，本节中我们来看看小红书如何决定将你的笔记展示给哪些用户。这套机制的核心是一个名为“CES”的综合评分系统。

## 基础评分：CES系统

小红书通过一套综合评分系统（CES）来计算笔记的初始排名。这套系统基于用户的交互行为进行打分。

以下是CES评分系统的具体构成：

*   **点赞**：每次点赞计 **1** 分。
*   **收藏**：每次收藏计 **1** 分。
*   **转发**：每次转发计 **4** 分。
*   **评论**：每次评论计 **10** 分。
*   **关注**：每次通过笔记关注账号计 **8** 分。

当一个用户完成从点击到关注的全部行为时，单次最高可为该笔记贡献 **1+1+4+10+8 = 24** 分。系统会累计这些分数，并以此作为笔记在初始流量池中排名的主要依据。

## 流量分发与排名逻辑

理解了基础评分，我们来看看小红书如何利用这个分数进行流量分发。平台的展示逻辑与抖音等单条推荐不同。

小红书信息流页面一次展示约20篇笔记。这20篇笔记有一个从第1名到第20名的内部排名。用户下滑后，会刷新出下一页（第21-40名），以此类推。我们的核心目标是让笔记进入前20名，并尽可能排在靠前的位置。

笔记发布后，系统会为其打上标签，并推送给具有相应兴趣标签的用户。每篇笔记都会获得一个基础的初始曝光量（例如1000次）。这个初始流量会分批次进行推送，比如第一波400，第二波300等。

系统会根据每一波流量的数据表现，决定是否给予下一波、更大规模的流量推荐。这里就引出了除了CES之外，几个至关重要的前置指标。

## 核心流量阀门：关键指标

CES评分决定了排名，但获得评分的前提是用户要点进你的笔记。因此，以下几个指标是触发后续流量的关键阀门。

以下是影响流量分发的几个关键前置指标：

1.  **点击率**：这是流量的第一道门。点击率取决于笔记的封面和标题是否吸引人。只有用户点击进入，后续的互动和CES评分才有可能发生。
2.  **互动率**：指用户与笔记（及作者）产生评论、回复等双向交流的比例。高互动率直接向系统表明笔记内容质量高、能引发讨论。
3.  **完播率**：对于视频笔记，用户观看完内容的比例非常重要。高完播率说明内容有吸引力。
4.  **互动值**：这是一个综合衡量用户与作者之间互动深度的指标。

**流程可以概括为：优质封面/标题 → 高点击率 → 用户进入 → 高完播/互动率 → 触发点赞、收藏等CES行为 → CES总分提升 → 笔记排名上升 → 获得系统下一轮更大的流量推荐。**

## 机制总结与公平性

无论平台的数据指标如何微调，其核心算法始终是考察笔记带来的**粉丝交互行为**。即使是粉丝数不多的新账号，只要单篇笔记的点击率、互动率和CES评分足够优秀，系统也会将其推荐给更多用户，形成“阶梯式”的流量放大。

平台的基础流量分配相对公平。通常，一篇自主创作的笔记，在最基础的流量层级（零层级或一层级）也能获得150-200次的基础展示。我们要做的，就是用优质内容撬动这个初始流量，通过良好的数据表现进入更高级别的流量池。

---

本节课中我们一起学习了小红书的流量分发核心机制。我们了解到，系统通过 **CES评分系统（点赞、收藏、转发、评论、关注）** 来给笔记排名，而排名的前提是拥有优秀的**点击率、互动率和完播率**。简单来说，想提升排名和曝光，就必须从封面标题开始，一步步做好内容，引导用户完成深度互动。理解这套机制，是做好小红书运营的基础。
