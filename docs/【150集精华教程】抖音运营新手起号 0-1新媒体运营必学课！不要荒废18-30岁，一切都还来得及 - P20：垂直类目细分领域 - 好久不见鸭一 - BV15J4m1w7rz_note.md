# 抖音运营新手起号：P20：垂直类目细分领域 🎯

在本节课中，我们将深入探讨账号垂直度的概念，并学习如何通过细分领域来精准定位你的内容方向。理解垂直类目的分级，是打造高粘性粉丝账号的关键一步。

---

上一节我们介绍了内容标签的重要性，本节中我们来看看如何将“垂直”这个概念进行更细致的划分。

我们经常强调做账号要垂直。很多同学会简单地认为，选择“美食”或“游戏”这样的大类目一直做下去，就是垂直。这个理解没有错，但这仅仅是最基础的层级。

实际上，账号的垂直度是有级别划分的。从宽泛到具体，我们可以将其分为一级、二级、三级甚至更细的垂直领域。理解这些层级，能帮助你找到更精准、竞争更小的内容赛道。

以下是垂直类目的分级示例，我们以“游戏”领域为例进行说明：

*   **一级垂直领域**：最宽泛的大类。
    *   **示例**：`游戏`
    *   **说明**：在这个层级，你可以制作任何与游戏相关的内容，如手游、网游、电竞、线下游戏等。

*   **二级垂直领域**：在一级领域下的具体分类。
    *   **示例**：`游戏 -> 手游`
    *   **说明**：选择“手游”后，你的内容可以涵盖所有手机游戏，例如《和平精英》（吃鸡）、《王者荣耀》、《第五人格》、《我的世界》等。

*   **三级垂直领域**：在二级领域下的进一步细分，通常聚焦于某个具体的产品或主题。
    *   **示例**：`游戏 -> 手游 -> 王者荣耀`
    *   **说明**：此时，你的账号内容将全部围绕《王者荣耀》这一款游戏展开。

三级垂直领域还可以继续向下细分，达到极致的专注度。例如：
*   **模式细分**：`游戏 -> 手游 -> 王者荣耀 -> 只玩“5v5竞技”模式`
*   **角色/英雄细分**：`游戏 -> 手游 -> 王者荣耀 -> 只玩“安琪拉”这个英雄`
*   **风格细分**：`游戏 -> 手游 -> 和平精英 -> 只制作“剧情类”游戏视频`

通过这样的层层细分，你的账号定位会变得无比清晰，吸引来的粉丝也极为精准，粘性会非常高。大领域或小领域都可以作为你的选择，这完全取决于你自己的兴趣和规划。
