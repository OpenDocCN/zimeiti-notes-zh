# 抖音运营教程：P2：如何快速加入中视频计划 🚀

在本节课中，我们将学习如何通过官方渠道，快速完成中视频计划的1.7万播放量门槛。这是一种安全、高效的方法，能帮助你避免踩坑，顺利开启自媒体变现之路。

上一节我们介绍了课程的整体框架，本节中我们来看看实现目标的具体路径。

## 核心方法概述 📋

当前唯一能快速、安全加入中视频计划的方法，是通过完成今日头条和西瓜视频官方发布的新手任务，从而获得官方奖励的流量包。这些流量可以直接用于作品推广，帮助你轻松达到1.7万播放量的审核要求。

市面上可能存在其他声称能快速通过的方法，但大部分存在风险。按照那些方法操作，可能导致审核无法通过，或者即便通过后也会被平台追责。为了避免大家踩坑，接下来介绍经过验证的官方路径。

## 第一步：西瓜视频任务（获取1万流量） 🍉

![](img/582fc98b2dd04d1c2873be19c4ab0e0e_1.png)

以下是获取第一个流量包的具体步骤：

1.  打开西瓜视频客户端。
2.  点击右下角的“我的”页面。
3.  在页面中找到并进入“创作训练营”。
4.  点击“去学习”入口（新账号会显示此入口）。
5.  认真学习官方提供的课程。
6.  完成并提交课程作业。

**核心操作**：`提交合格的官方课程作业`。

只要认真完成作业，通常都能合格。任务完成后，你将获得1万个流量包，该流量包可直接用于作品投放。

## 第二步：今日头条任务（获取约6000流量） 📰

完成西瓜视频任务后，我们需要切换平台以获取更多流量。以下是具体操作：

1.  打开今日头条客户端。
2.  同样点击右下角“我的”。
3.  进入“创作中心”。
4.  找到“新作者成长攻略”活动海报并点击进入。
5.  完成页面内提供的各项新手涨粉任务。

**核心操作**：`完成“新作者成长攻略”中的所有新手任务`。

![](img/582fc98b2dd04d1c2873be19c4ab0e0e_3.png)

这些任务完成后，总共可获得约6000的流量包奖励，同样可用于作品推广。

## 流量计算与剩余目标 🧮

现在，我们来计算一下通过以上两个官方任务获得的总流量：

*   **西瓜视频任务奖励**：10,000 流量包
*   **今日头条任务奖励**：约 6,000 流量包
*   **已获得总流量**：约 16,000 播放量

中视频计划的审核要求是1.7万播放量。因此，在获得约1.6万流量包后，你只需再依靠自然流量获得约1000播放量即可。这可以通过随意发布几个作品轻松实现。

公式表示为：
`剩余播放量需求 = 17,000 - (10,000 + 6,000) ≈ 1,000`
