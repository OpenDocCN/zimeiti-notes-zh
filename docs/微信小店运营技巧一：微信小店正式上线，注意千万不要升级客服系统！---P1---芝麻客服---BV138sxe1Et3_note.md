# 微信小店运营技巧 🛍️：P1：微信小店升级关键注意事项

在本节课中，我们将学习微信小店正式上线后的一个重要运营技巧：如何正确进行店铺升级，并避免因误操作导致客服系统功能受限。

微信已于8月25日正式上线“微信小店”。对于已经拥有视频号小店的商家，现在可以免费将店铺升级为微信小店。升级操作本身很简单：登录视频号小店后台，点击右上角的“升级为微信小店”按钮，然后按照提示流程完成即可。

然而，这个升级过程包含两个独立部分：**店铺升级**和**客服系统升级**。正确处理这两者的关系，是保证您后续运营顺畅的关键。

上一节我们介绍了升级的基本流程，本节中我们来看看升级时的核心注意事项，特别是客服系统的处理。

以下是升级时可能遇到的两种情况及其影响：

![](img/befaa2c5f139f65411c3c8f82e651c2c_1.png)

1.  **情况一：视频号小店已接入微信客服**
    *   完成店铺升级后，**不要主动点击升级“小店客服”**。
    *   只要不升级客服系统，您原先使用的“芝麻视频号助手”的所有功能将不受任何影响。

![](img/befaa2c5f139f65411c3c8f82e651c2c_2.png)

![](img/befaa2c5f139f65411c3c8f82e651c2c_3.png)

2.  **情况二：视频号小店未接入微信客服**
    *   完成店铺升级后，系统**会自动将您的客服同步升级为官方的“小店客服”**。
    *   一旦升级为小店客服，“芝麻”系统将无法正常获取客服消息，导致其所有客服相关功能失效。

![](img/befaa2c5f139f65411c3c8f82e651c2c_4.png)

因此，在升级小店前，**务必提前联系“芝麻客服”的人工客服进行确认**，以避免不必要的功能损失。

![](img/befaa2c5f139f65411c3c8f82e651c2c_5.png)

此外，目前微信官方推出的“小店客服”系统功能较为基础，主要支持：
*   绑定客服员工
*   查看订单
*   发送商品
*   编辑快捷回复

相比之下，“芝麻视频号助手”提供了更强大的运营功能，例如：
*   设置多店铺消息聚合回复
*   多种客服接待规则与自动回复
*   在客服侧边栏直接编辑订单价格、地址、备注
*   快速处理售后订单
*   小额打款等功能

为了确保您能继续正常使用“芝麻客服系统”的丰富功能，再次强调：在升级微信小店前，请务必联系我们的客服人员。

本节课中我们一起学习了微信小店升级的核心注意事项。关键在于区分店铺升级与客服升级，并谨慎处理客服系统部分，以保留原有高效工具的完整功能。
