# 《抖音全套运营教程》：P18：16、口播类视频拍摄方法-7 🎬

在本节课中，我们将要学习口播类视频的具体拍摄方法。我们将通过一个成功案例，详细拆解拍摄时需要注意的构图、内容定位等核心要素，帮助你掌握制作高质量口播视频的技巧。

上一节我们介绍了视频拍摄的通用注意事项，本节中我们来看看如何将这些要点应用到具体的口播视频拍摄中。

## 一、拍摄核心要点

![](img/6780feb8c57f966bcdf6b9f446d615e0_1.png)

拍摄口播视频需要注意以下几点：**构图比例**、**主题突出**、**事物清晰**、**光线色彩**和**内容连贯**。这些要点适用于所有类型的口播内容，无论是金融、装修、养生还是情感口播。

## 二、案例解析：情感口播账号“种小草”

我们以学员账号“种小草”为例。这是一位40岁带三个娃的宝妈，从事情感口播。她的一条视频播放量超过8000万，并借此涨粉30万。

她成功的关键在于**精准吸引目标群体**。她的所有内容都围绕特定群体的兴趣点展开，因此涨粉快且粉丝粘性高。

### 1. 构图与画幅

抖音是**9:16**的竖屏平台。但在拍摄时，你可以选择横屏或竖屏构图。案例中的账号使用的是**4:3**的横屏构图。

**重要原则**：保持画幅统一。如果你选择横屏，所有视频都应使用横屏；如果选择竖屏，则全部使用竖屏。风格统一是专业度的体现。

在构图上，人物主体应占据画面中央约**70%** 的面积。避免怼脸拍摄，应在画面的左、右、上方适当留白，使画面看起来清晰、舒适。

![](img/6780feb8c57f966bcdf6b9f446d615e0_3.png)

### 2. 人物与表现

口播视频的核心是**内容传达**，而非主播的颜值。你只需要看起来清晰、体面即可。

如果你对自己的外貌不自信，可以开启美颜滤镜。适度使用美颜工具，让自己呈现更佳状态是完全可行的。

### 3. 内容与受众定位

精准的内容定位是吸引目标受众的关键。我们来看她那条涨粉30万的视频主题：“跟婆婆在一起住是什么样的感受”。

这个主题天然地吸引了**已婚女性群体**，尤其是对婆媳关系有切身感受的女性。年轻未婚女性或男性对此的共鸣感较弱。因此，她的内容牢牢锁定了“已婚女性”这个精准的垂直群体。

## 三、拍摄实操技巧

![](img/6780feb8c57f966bcdf6b9f446d615e0_5.png)

以下是拍摄口播视频的几个实用技巧：

*   **关于提词**：如果记不住文案，无需硬背。可以在剪辑环节使用提词器软件，实现边看边录，大大降低拍摄难度。
*   **关于收声**：清晰的声音至关重要。建议购买一个**手机收声麦克风**（价格通常在十几元），这能有效提升音频质量，让观众听得更清楚。
*   **关于场景**：拍摄地点不限，室内或户外均可。关键在于**背景要整洁美观**，避免杂乱无章的画面干扰观众注意力。

本节课中我们一起学习了口播视频的拍摄方法。我们通过“种小草”的案例，重点分析了如何通过统一的画幅、舒适的构图、清晰的内容表达以及精准的受众定位来制作成功的口播视频。记住，口播的核心是价值传递，运用好工具和技巧，你也能拍出吸引人的优质内容。
