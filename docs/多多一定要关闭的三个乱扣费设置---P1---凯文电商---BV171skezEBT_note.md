# 拼多多店铺运营：P1：必须关闭的三个乱扣费设置 🔧

在本节课中，我们将学习如何识别并关闭拼多多店铺后台三个可能导致意外扣费的设置。这些设置若未妥善管理，可能会侵蚀您的利润。我们将分步讲解关闭方法，并介绍如何设置子账号权限以防止误操作。

上一节我们概述了课程目标，本节中我们来看看具体需要关闭哪三个设置。

以下是三个需要重点检查并关闭的功能：

1.  **退货包运费**
    想要关闭此功能，请按以下步骤操作：
    *   进入店铺后台的【售后管理】。
    *   找到【消费者体验】板块。
    *   点击右侧的【更多】选项。
    *   在最后选择【停用】即可。

2.  **智能营销**
    此功能可能自动发起营销活动并产生费用。关闭步骤如下：
    *   进入【营销工具】下的【智能营销】。
    *   点击【设置记录】。
    *   选择对应的产品。
    *   点击右侧的【详情】。
    *   进入详情页面后，点击【暂停】即可关闭。

3.  **自动跟价**
    此功能可能导致商品价格被自动调整。关闭方法如下：
    *   打开【营销活动】页面。
    *   将鼠标悬停在活动说明区域，点击【说明】。
    *   在弹出的说明页面中，划到最底部。
    *   即可看到并点击【关闭】选项。

我们已经了解了如何关闭三个主要的扣费设置。接下来，为了防止未来在后台操作时误触开通其他付费工具，我们可以通过设置子账号权限来创建一个更安全的操作环境。

以下是设置限制权限的子账号步骤：

*   **创建子账号**：打开后台的【子账号管理】，点击【新增子账号】。在权限选择时，关键的三项按如下设置：
    *   `商品管理`：取消 **商品编辑** 权限。
    *   `店铺营销`：取消 **营销工具** 权限。
    *   `发货管理`：取消 **运费模板** 权限。
*   **日常操作**：点击保存后，日常运营使用此子账号登录后台即可。
*   **修改设置**：当需要修改商品、营销或运费模板等重要设置时，再使用主账号登录操作。

本节课中我们一起学习了如何关闭拼多多店铺后台的“退货包运费”、“智能营销”和“自动跟价”这三个可能造成乱扣费的设置，并掌握了通过设置子账号权限来预防误操作的方法。定期检查这些设置，有助于您更好地控制店铺成本，保护经营利润。
