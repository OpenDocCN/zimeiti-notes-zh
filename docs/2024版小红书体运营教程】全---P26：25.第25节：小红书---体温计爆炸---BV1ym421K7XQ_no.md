# 小红书运营教程：P26：小红书暴力玩法揭秘！吸粉推广篇！

在本节课中，我们将要学习小红书推广的两种核心方式：自建账号矩阵运营与官方付费广告投放。我们将理清两者的关系，并详细介绍各自的运作模式与适用场景。

## 账号类型解析

上一节我们介绍了小红书的基本玩法，本节中我们来看看平台上的账号类型。了解不同类型的账号及其用途，是制定推广策略的基础。

小红书账号主要可分为以下三类：

1.  **企业号**：主要作用是树立官方品牌形象。拥有官方认证标志，可以展示品牌名称、地址等信息，并享有官方特权，便于开展业务。
2.  **个人号**：这是目前大多数用户运营的账号类型。其特点是**灵活性高**，可以是品牌员工或消费者身份，便于塑造个人品牌特色，也适合品牌相关人物出镜。
3.  **引流号**：这类账号在之前的课程中被称为“炮灰号”。其作用通常是辅助引流，可以批量管理，重要性较低，在紧急情况下可作为备用小号使用，一般无需进行精细的个性规划。

通常，品牌推广会通过企业号、个人号或引流号发布笔记来实现引流。

## 企业号矩阵运营建议

了解了账号分类后，我们来看看如何系统性地运营它们。打造一个有效的账号矩阵，能最大化推广效果。

![](img/ceb437af773299b8879e75d781645ea2_1.png)

以下是打造企业号矩阵的一些具体建议：

![](img/ceb437af773299b8879e75d781645ea2_2.png)

![](img/ceb437af773299b8879e75d781645ea2_3.png)

*   **积极参与品牌活动**：例如，婚纱、服装、美妆等品类的企业号举办试用或团购活动时，可以动用多个小号去参与，增加活动热度与真实性。
*   **个人号强化品牌属性**：在个人号的笔记内容中，应多次暗示品牌的定位。例如，品牌目标用户是城市年轻人还是职场新人，需要通过内容持续传递。
*   **控制产品发布频率**：个人号发布品牌产品内容的频率不宜过高。一个较好的比例是 **3:7**，即30%的内容用于营销推广，70%的内容用于种草分享或知识科普。
*   **利用引流号进行评论引流**：当企业号发布内容后，可以使用引流号在评论区进行互动和引导，实现更隐蔽和有效的引流。

## 官方付费广告（CPC）介绍

![](img/ceb437af773299b8879e75d781645ea2_9.png)

除了自建账号矩阵，另一种高效的推广方式是借助平台力量。接下来，我们看看小红书官方的付费广告系统。

![](img/ceb437af773299b8879e75d781645ea2_11.png)

小红书官方广告主要指 **CPC（Cost Per Click）竞价广告**。这是一种按点击付费的广告模式。

**申请流程**：
1.  首先，需要拥有一个认证的企业号。
2.  然后，访问小红书官网，在“业务合作” -> “推广合作”板块，按照官方指引提交资质并缴纳审核费用，完成认证后即可开通广告投放权限。

**广告类型**：
官方信息流广告主要分为两大类：
1.  **信息流广告**：用户浏览小红书首页信息时，不定时出现在信息流中的广告。
2.  **搜索广告**：当用户使用关键词搜索时（例如“帽子”），系统根据算法匹配并展示的相关广告。这种广告模式与百度搜索竞价类似，通过**对关键词出价**来竞争不同的广告展示位。

**广告位排名**：
当前信息流广告位主要分为以下几类：第3位、第13位、第23位等。出价不同，获得的广告位也不同。

![](img/ceb437af773299b8879e75d781645ea2_13.png)

**注意事项**：
官方广告投放涉及较高费用。建议在资金允许的情况下考虑与官方合作投放竞价广告，因为它能直接获得由平台算法推荐的精准流量。关于投放的常见问题，可在官方帮助页面查询。
