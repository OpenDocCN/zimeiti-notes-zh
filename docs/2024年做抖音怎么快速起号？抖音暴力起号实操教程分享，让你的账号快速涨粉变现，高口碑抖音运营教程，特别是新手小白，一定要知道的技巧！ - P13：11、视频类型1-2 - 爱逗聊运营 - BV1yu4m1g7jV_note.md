# 抖音运营实战：P13：11、视频类型1-2 📚

在本节课中，我们将要学习抖音账号运营中一个至关重要的环节：**人设的建立**。我们将深入探讨强人设与弱人设的区别，并详细介绍一种核心的视频内容形式——**口播**。掌握这些概念，将帮助你更清晰地规划账号内容，吸引精准粉丝。

上一节我们解决了文案问题，本节中我们来看看内容创作的下一个关键步骤。从涨粉到变现并非一条直线，中间还有一个核心环节：**建立人设**。建立人设的目的不是为了炫耀，而是为了在众多同类账号中脱颖而出。例如，有十个人都做装修内容，如果其中一位“小王同学”树立了一个幽默搞笑的人设，观众就更容易记住他，从而忽略其他九位。当观众喜欢你这个人时，他们自然就成为你的粉丝了。

这就是人设的作用。我们来学习这个知识点，请边听边记。

## 人设的两种类型：强与弱

人设主要分为两种：**强人设**与**弱人设**。区分清楚这两种人设，你就能快速判断任何视频的类型。

![](img/804b08c3317edc7fa70844a9a9d827a8_1.png)

首先，**强人设**是什么意思呢？

![](img/804b08c3317edc7fa70844a9a9d827a8_3.png)

强人设是指以**创作者个人**为切入点的内容。它必须与你自己密切相关。例如：
*   分享你的行业经验（如装修、办公、房产、汽车、餐饮、婚介、法律咨询、电商等）。
*   分享你的兴趣爱好（如美食、旅行、手工、宠物、星座占卜、美妆、护肤、服装穿搭、游戏、摄影等）。

![](img/804b08c3317edc7fa70844a9a9d827a8_1.png)
![](img/804b08c3317edc7fa70844a9a9d827a8_3.png)

简而言之，强人设就是分享你个人的生活、工作或爱好，并且**大部分需要真人出镜**，因为内容与你本人直接相关。

那么，什么是**弱人设**呢？

弱人设是指内容与创作者本人**没有直接关系**，核心目的是为了把账号做好。例如：
*   **情感语录类账号**：发布配乐的情感文字图片或视频，无真人出镜或配音。
*   **影视解说类账号**：虽然有人声解说电影，但观众关注的是电影内容和解说本身，而非解说者这个人。

弱人设的视频，创作者本人通常不是内容的焦点。

## 核心内容形式：口播详解

理解了人设，我们来看具体的内容创作。只要内容与你个人相关，都可以制作成视频。我们首先详细讲解一种非常高效且易操作的形式：**口播**。

口播，简单说就是**用嘴巴讲**。它涵盖多种领域，目的各有不同：

以下是主要的口播类型与目的：
*   **讲故事类口播**：分享客户故事、从业经历、人生感悟等。故事不一定完全真实，可以从文案平台寻找有趣素材。目的是让观众觉得**听起来有意思**。
*   **教知识类口播**：分享育儿、金融、养生、装修、房产、汽车等领域的专业知识。目的是让观众在几十秒内获得**收获感**，觉得学到了东西。

口播非常适合许多领域，如法律、保险、金融、装修知识科普等。它的优势在于**拍摄简单**：只需准备好文案，用手机支架固定好就能拍摄。

针对常见顾虑，这里做出解答：
*   **口才不好能做口播吗？**  
    **可以**。口播是提前准备好稿子再录制，并非临场发挥。大部分口播者的“好口才”都源于充分准备。
*   **普通话不好能做口播吗？**  
    **可以**。如果你的目标客户是同城或有地域属性，使用方言可能效果更好，显得亲切、有趣。同时，你可以通过添加字幕来解决听众的理解问题。

本节课中我们一起学习了抖音账号运营中**人设建立**的重要性，明确了**强人设**与**弱人设**的区别。接着，我们深入探讨了**口播**这种核心视频形式，了解了它的类型、优势以及如何克服拍摄中的常见障碍。掌握这些基础，是规划账号内容、实现有效涨粉的关键第一步。
