# 新媒体运营系统课：P7：2.平台现状 🧭

在本节课中，我们将要学习自媒体行业的平台现状，分析其发展环境、政策支持、市场规模以及从业者与用户的比例，帮助你判断当前是否适合进入这个领域。

上一节我们了解了自媒体的基本概念，本节中我们来看看这个行业所处的具体环境。

## 政策支持 📜

现阶段，整个社会氛围比较浮躁，很多人都想赚钱。过去有人采取不合法的手段（例如前往缅甸等地）谋取利益，这类行为是国家明令禁止的，对我们没有任何好处。

与之相反，自媒体行业是国家明确支持的领域。国家已将支持文件下达到各个地方政府。

以下是部分地方政府发布的政策性文件示例：
*   荆州市发布了“六项措施助力自媒体及新媒体高质量发展”的文件。
*   广西壮族自治区、四川省人民政府、浙江省人民政府、杭州市人民政府、江西省赣州市人民政府等都发布了相应的政策性文件。

这些文件指出，要坚持正确导向，内容为王。这意味着，未来作为自媒体创作者，我们创作的文章和视频，是结算收益最重要的方式和途径。

国家已经下达了明确的政府文件予以支持，因此，自媒体行业是可以进入并从事的。

## 平台激励 💰

![](img/4d0932f953d951b6665d5f6669c5c118_4.png)

除了政策支持，自媒体平台也响应国家号召，出资鼓励用户创作视频。

平台具体出资多少呢？我们可以通过一个视频来了解。

视频关键信息：西瓜视频决定未来一年额外拿出**20亿元**补贴中视频创作者。这20亿元是真金白银的纯补贴，不包含任何商单、直播和电商的收入，并且上不封顶。这意味着，创作者通过制作视频，可以直接从这20亿元中获得收益。

## 市场规模与供需关系 📊

现阶段自媒体行业还有一个显著优势：用户数量非常庞大。

根据2022年《中国互联网络发展状况统计报告》，我国网民规模约为**9.4亿人**，接近10亿。这些网民都具备观看视频的条件，每天都会刷视频。

![](img/4d0932f953d951b6665d5f6669c5c118_8.png)

有如此庞大的观看群体，那么制作视频的人有多少呢？

以下是相关数据：
*   截止去年，我国全职从事自媒体的人数为 **370万人**。
*   兼职从事自媒体的人数为 **600万人**。
*   两者相加，从事视频创作的人数约为 **970万人**，可近似看作1000万人。

我们可以用一个简单的比例来理解这个市场：
*   **观看视频人数 (A)**: 9.4亿 ≈ 10亿
*   **制作视频人数 (B)**: 0.1亿
*   **比例关系**: A / B ≈ 100 / 1

这相当于每100个看视频的人中，只有1个人在制作视频。这就产生了一个问题：一个人制作的视频，能否满足一百个人的观看需求？答案显然是不够的。视频内容的生产速度远远跟不上消费速度。

这正是国家支持自媒体行业发展的重要原因。从我们观看设备的变迁——从黑白电视到彩色电视，再到如今可以连接WiFi的智能电视——可以看出，互联网视频是未来的发展趋势。未来，我们制作的视频将出现在更多终端上，用户群体还会持续扩张。

![](img/4d0932f953d951b6665d5f6669c5c118_8.png)

本节课中我们一起学习了自媒体行业的平台现状。我们了解到，该行业拥有国家政策的明确支持，平台方也提供了巨额的资金补贴。更重要的是，市场呈现出巨大的供需缺口：近10亿的视频用户对应仅约1000万的内容创作者，这意味着存在广阔的发展空间和机会。因此，当前是进入自媒体领域的一个有利时机。
