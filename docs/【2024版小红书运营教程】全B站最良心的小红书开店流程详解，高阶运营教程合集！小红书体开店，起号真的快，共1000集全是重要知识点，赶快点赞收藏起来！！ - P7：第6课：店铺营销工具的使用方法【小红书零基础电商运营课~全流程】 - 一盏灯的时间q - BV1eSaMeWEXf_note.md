# 小红书电商运营课：P7：店铺营销工具使用指南 🛠️

在本节课中，我们将学习小红书店铺后台营销工具的使用方法。掌握这些工具，可以帮助你设置商品促销活动，有效提升店铺的吸引力与转化率。

上一节我们介绍了商品上架流程，本节中我们来看看如何利用营销工具为商品设置优惠活动。

## 营销活动报名

营销工具位于店铺后台的“营销”板块。第一个功能是“营销活动”中的“活动报名”。

以下是活动报名的操作步骤：
*   点击“可报名的活动”，系统会展示当前店铺有资格参与的平台活动。
*   例如，在特定时间段内，可能会显示“三八节”活动。
*   点击“报名活动”可以查看具体参与条件，例如店铺评分需高于4分，或对商品价格有特定要求。
*   活动通常设有优惠门槛，例如“满300减40”。你需要评估自身利润能否承担此折扣，再决定是否报名。
*   对于“双十一”、“双十二”等大型活动，建议报名参与。对于日常小型活动，可将精力更多投入在笔记内容创作上。

## 商品促销设置

“营销管理”下的“商品促销”功能，允许你对商品进行折扣设置。我们主要使用“多品促销”，而非“单品促销”。

以下是核心的促销工具介绍：
*   **满额立减**：当订单金额达到指定数额时，可减免一定金额。
*   **满件打折**：当购买商品达到指定件数时，可享受折扣优惠。
*   其他三种工具（如限时折扣）使用频率较低，本节暂不展开。

### 满额立减设置详解

接下来，我们详细讲解“满额立减”的设置方法。

1.  **创建活动**：点击“满额立减”，创建一个新活动。
2.  **设置基本信息**：
    *   **活动名称**：可自定义，例如“春季优惠”。
    *   **活动时间**：建议设置较长的有效期，例如3个月以上，以减少频繁操作。
3.  **选择商品范围**：通常选择“全部商品”进行全店促销，操作更便捷。
4.  **设置优惠规则**：
    *   **满减门槛**：通常设置为店铺的平均客单价。例如，店铺商品多在50元以上，可设置为“满50元”。
    *   **减免金额**：设置优惠力度，例如“减5元”。
5.  **完成创建**：点击“完成”即可。如需修改或停止活动，可随时“暂停”或“删除”。

### 满件打折设置详解

“满件打折”的设置逻辑与“满额立减”类似。

![](img/a77b8512dbe57c26cc36c38bb302f50b_1.png)

1.  **创建活动**：点击“满件打折”创建新活动。
2.  **设置时间与规则**：
    *   设置活动时间，同样建议3个月以上。
    *   在优惠规则中，设置为“满1件打9折”或“满1件打8折”。
3.  **完成创建**：点击“完成”，全店所有商品将自动应用此折扣。

**核心建议**：对于新店，为了快速积累销量和权重，建议设置“满件打折”活动，例如全店商品打8折或9折。这能有效提升出单速度。公式可以表示为：
`折后售价 = 商品原价 × 折扣率`
当店铺权重提升、销量稳定后，可以取消此活动，恢复利润更高的原价销售模式。

![](img/a77b8512dbe57c26cc36c38bb302f50b_3.png)

**重要提示**：“满额立减”与“满件打折”活动通常不能同时生效，选择其一即可。对于新店起步，更推荐使用“满件打折”。
