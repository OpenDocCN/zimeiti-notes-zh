# 视频号运营教程：P3：视频号基础功能讲解 🎬

在本节课中，我们将学习视频号的基础功能设置与界面操作。这是搭建账号的第一步，掌握这些知识能帮助你更高效地管理自己的视频号。

![](img/41ac0c93029c6a2f8ad994b3c5c98ece_3.png)

## 视频号入口与开通

视频号主要有两个进入端口。第一个端口位于微信“发现”页的“朋友圈”下方。第二个端口在微信“我”的页面中，点击“设置”即可找到。

![](img/41ac0c93029c6a2f8ad994b3c5c98ece_5.png)

![](img/41ac0c93029c6a2f8ad994b3c5c98ece_7.png)

如果你的“朋友圈”下方没有视频号入口，可以按照以下步骤开启：
1.  点击微信右下角的“我”。
2.  进入“设置”。
3.  选择“通用”。
4.  点击“发现页管理”。
5.  找到“视频号”选项并开启右侧开关。

![](img/41ac0c93029c6a2f8ad994b3c5c98ece_9.png)

开通视频号非常简单。进入视频号页面后，如果尚未开通，系统会提示你“发表视频”。只需按照指引输入账号名称，即可成功创建视频号。

![](img/41ac0c93029c6a2f8ad994b3c5c98ece_11.png)

![](img/41ac0c93029c6a2f8ad994b3c5c98ece_12.png)

## 视频号主界面功能

进入视频号后，主界面分为几个主要板块。

**推荐**页面会根据你的观看喜好，由系统算法推送相关视频。

**朋友点赞**页面会展示你的微信好友点赞过的视频。这意味着，如果你的朋友点赞了某个作品，你可能会在这个页面看到它。

**关注**页面集中展示了你所关注的所有视频号发布的最新内容。

界面顶部的**搜索框**（放大镜图标）功能强大。你可以用它搜索具体的视频号名称，也可以输入商品关键词（例如“保鲜膜”）来寻找相关的带货账号，这对后期寻找对标账号很有帮助。

## 个人主页与隐私设置

点击右下角头像，可以进入个人中心。这里可以查看你点赞、收藏过的内容以及关注列表。“消息”是私信入口，“隐私”设置则非常重要。

在“隐私”设置中，你可以管理点赞的可见性。例如，你可以设置不让特定好友看到你点赞的作品，或者选择不看某些好友点赞过的内容。这对于保护个人隐私或避免让熟人看到你的商业行为很有用。

![](img/41ac0c93029c6a2f8ad994b3c5c98ece_14.png)

点击进入你自己的视频号主页，再点击右上角的三个点，进入“资料设置”。在这里，你可以修改头像、名称（一年可修改5次）、性别、地区和简介。

![](img/41ac0c93029c6a2f8ad994b3c5c98ece_15.png)

简介的撰写可以参考一个通用话术：`视频同款购买方式：点下面的商品橱窗，找到你想要的商品下单即可。` 你也可以模仿优秀对标账号的写法。简介可以无限次修改。

在“隐私与权限”中，有一个关键选项是“在个人名片展示视频号”。开启后，你的微信好友在其通讯录中点击你的名片时，能看到你的视频号。如果不想展示，可以在此关闭。

此外，你还可以设置作品对特定好友“不可见”。在发布设置中，选择“公开，但不推荐给朋友”，并添加不想让其看到的好友名单即可。

## 商品橱窗与数据解读

在个人主页设置中，找到“管理橱窗”可以进入你的带货后台。我们也可以点开其他博主的橱窗进行观察。

查看他人橱窗时，会看到两个关键数据：
*   **“已售 X.X万”**：这个数据位于橱窗顶部，表示**该博主所有带货商品的历史总销量**。
*   **“已售 X.X万”**：这个数据位于单个商品下方，表示**该商品被所有带货博主销售的历史总销量**。

![](img/41ac0c93029c6a2f8ad994b3c5c98ece_17.png)

这两个数据含义不同，请注意区分。

## 作品管理与评论区互动

进入自己发布的任意作品页面，点击右上角的三个点，可以对作品进行多项管理操作：
*   **置顶**：将重要作品固定在主页顶部。
*   **上热门**：付费为视频增加曝光（加热）。
*   **关闭评论**：禁止他人评论此作品。
*   **仅自己可见/删除**：隐藏或删除作品。
*   **添加提示**：为视频添加文字说明（通常较少使用）。

![](img/41ac0c93029c6a2f8ad994b3c5c98ece_19.png)

![](img/41ac0c93029c6a2f8ad994b3c5c98ece_20.png)

![](img/41ac0c93029c6a2f8ad994b3c5c98ece_22.png)

评论区是重要的互动和引导区域。对于不当评论，可以**长按该评论并选择删除**。

![](img/41ac0c93029c6a2f8ad994b3c5c98ece_24.png)

视频号评论目前没有官方“置顶”功能，但评论会按点赞量排序。因此，你可以通过**用小号或请朋友为你的引导性评论（如“点击下方橱窗购买”）点赞**，使其获得高赞而自然排在评论区前列，方便观众看到。

![](img/41ac0c93029c6a2f8ad994b3c5c98ece_26.png)
