# 微信视频号运营教程：P21：19.实操步骤

在本节课中，我们将学习如何从零开始，一步步开通并设置微信视频号的商品橱窗功能。这是实现视频号带货变现的基础操作。

上一节我们介绍了视频号的基础概念，本节中我们来看看具体的开通与上架实操。

## 第一步：打开微信
首先，在您的手机上启动微信应用。

## 第二步：进入“发现”页
在微信主界面底部，找到并点击“发现”选项卡。

![](img/2c3606a4746ef550c6ff0394f5c7bed1_1.png)

![](img/2c3606a4746ef550c6ff0394f5c7bed1_2.png)

## 第三步：找到视频号入口
在“发现”页面中，将视线向上移动。您会看到“视频号”的入口，其图标是一个黄色的蝴蝶形状。点击该入口进入视频号。

![](img/2c3606a4746ef550c6ff0394f5c7bed1_1.png)
![](img/2c3606a4746ef550c6ff0394f5c7bed1_2.png)

![](img/2c3606a4746ef550c6ff0394f5c7bed1_4.png)

![](img/2c3606a4746ef550c6ff0394f5c7bed1_6.png)

## 第四步：进入个人中心
进入视频号后，在页面右上角找到一个人形头像图标，点击进入个人中心。

![](img/2c3606a4746ef550c6ff0394f5c7bed1_4.png)
![](img/2c3606a4746ef550c6ff0394f5c7bed1_6.png)

## 第五步：进入创作者中心
在个人中心页面，向下滑动屏幕，直到最底部。找到“创作者中心”并点击进入。如果未显示此选项，请先发布一条视频作品。

![](img/2c3606a4746ef550c6ff0394f5c7bed1_8.png)

## 第六步：找到带货中心
进入创作者中心后，在页面中找到“带货中心”的入口，其图标类似一个小卖部。

![](img/2c3606a4746ef550c6ff0394f5c7bed1_9.png)

![](img/2c3606a4746ef550c6ff0394f5c7bed1_8.png)
![](img/2c3606a4746ef550c6ff0394f5c7bed1_9.png)

## 第七步：开通橱窗权限
点击“带货中心”后，系统会根据您的账号状态进行跳转。

以下是两种情况的处理方式：

### 情况一：未开通权限
如果您的账号尚未开通商品橱窗功能，点击后会跳转到开通界面。

![](img/2c3606a4746ef550c6ff0394f5c7bed1_11.png)

1.  **选择带货身份**：页面会要求您选择带货身份。通常会出现“我是商家”、“我是达人”等选项。
    *   **核心操作**：请选择 **`我是达人`**。
    *   **说明**：即使您是厂家，也建议先选择“达人”身份开通，以便使用官方精选联盟的商品。

2.  **缴纳保证金**：选择身份后，会跳转到保证金缴纳界面。
    *   **核心操作**：点击 **`去交纳`** 按钮，按照提示通过微信支付100元保证金即可完成开通。

![](img/2c3606a4746ef550c6ff0394f5c7bed1_11.png)
![](img/2c3606a4746ef550c6ff0394f5c7bed1_13.png)

### 情况二：已开通权限
如果您的账号已经开通了商品橱窗功能，点击“带货中心”后会直接进入商品上架与管理界面。

![](img/2c3606a4746ef550c6ff0394f5c7bed1_13.png)

## 操作流程总结
为了方便理解，整个操作流程可以概括为以下路径：

```
打开微信 -> 点击“发现” -> 进入“视频号” -> 点击右上角个人头像 -> 下滑进入“创作者中心” -> 点击“带货中心” -> (根据状态) 选择身份并缴纳保证金 或 直接上架商品
```

![](img/2c3606a4746ef550c6ff0394f5c7bed1_15.png)

![](img/2c3606a4746ef550c6ff0394f5c7bed1_15.png)

本节课中我们一起学习了微信视频号商品橱窗功能的具体开通步骤。关键点在于通过“创作者中心”找到“带货中心”，并根据引导完成达人身份选择和保证金缴纳。完成这些步骤后，您就具备了在视频号中带货的基础条件，接下来就可以开始选品和上架商品了。
