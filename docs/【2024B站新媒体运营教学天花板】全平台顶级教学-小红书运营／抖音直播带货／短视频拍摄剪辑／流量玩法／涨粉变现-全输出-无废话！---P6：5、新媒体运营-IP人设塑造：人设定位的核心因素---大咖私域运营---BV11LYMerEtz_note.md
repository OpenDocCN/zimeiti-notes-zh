**1. 角色**
角色，简单来说，就是你在特定赛道中扮演的专业身份。你选择什么赛道、准备输出什么方向的内容，就需要让别人知道你提供的内容具备专业性。
*   **公式：** `人设角色 = 赛道领域 + 专家身份`
*   **例如：**
    *   **装修博主：** 你需要展示自己确实有房子在装修或已装修完成，通过记录过程、分享避坑经验来建立专业形象。
    *   **母婴赛道：** 你可以是分享亲身经验的**宝妈**，也可以是提供专业知识的**育儿护理专家**。
    *   **情感赛道：** 你可以是提供建议的**心理学顾问**，也可以是分享自身蜕变经历的**成长型人设**（例如，分享如何走出职场困境、创业失败后重新振作的故事）。

**2. 内容**
内容是人设的载体。你提供的内容必须与你选择的赛道高度一致，这关系到账号的垂直度和平台标签。
*   **核心要求：** 内容需吸引用户、提供价值，并在平台上有一定特色。
*   **重要性：** 保持内容垂直至关重要。如果今天发教育内容，明天发宠物视频，平台无法为你打上精准标签，会扰乱推荐算法，导致流量分发不稳定，影响账号成长。
*   **原则：** `坚持发布与选定赛道强相关的高质量内容`。

**3. 形式**
形式指的是内容呈现的具体方式，它直接影响用户的观感和记忆。
*   **拍摄场景：** 是固定在室内，还是在室外多种场景切换？例如，固定在书房、在上班路上、在车内等。可以参考对标账号的场景选择。
*   **声音呈现：** 是自己出镜讲解，还是使用配音？需要评估自己能否流畅地进行口头表达。
*   **内容类型：** 是**剧情演绎型**、**干货分享型**，还是**生活记录型**？每种类型各有优劣：
    *   **干货型** 需要扎实的文案脚本。
    *   **生活记录型** 需要丰富的拍摄素材。
    *   **剧情型** 创作难度相对较高。

**4. 记忆点**
记忆点是让用户快速记住并识别你的关键。它需要通过固定的设计反复强化。
*   **常见的记忆点设计包括：**
    *   **视频前几秒的固定动作/特效/开场白**：这些元素需要**前置**，在视频开头就给予用户强烈的视觉或听觉冲击，加深印象。
    *   **统一的BGM（背景音乐）**：贯穿视频全程的特定音乐，能形成独特的听觉标识。
    *   **统一出现的物品或服装风格**：例如，视频中总是出现的某个道具、背景摆设，或固定风格的着装，这些能**贯穿内容全局**，强化整体形象。
