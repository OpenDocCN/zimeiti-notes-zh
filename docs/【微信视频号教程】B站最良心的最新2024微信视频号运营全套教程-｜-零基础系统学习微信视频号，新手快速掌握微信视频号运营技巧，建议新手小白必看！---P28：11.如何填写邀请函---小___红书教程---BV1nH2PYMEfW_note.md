# 微信视频号运营教程：P28：11.如何填写邀请函 📝

在本节课中，我们将学习如何正确填写视频号创作者计划的邀请函。我们将一步步地找到并填写所需的账号信息，确保提交过程准确无误。

![](img/1f660ff9dc89eee59a47f4e49f2a209b_1.png)

上一节我们介绍了如何注册视频号，本节中我们来看看如何填写后续的邀请函信息。

![](img/1f660ff9dc89eee59a47f4e49f2a209b_3.png)

## 第一步：打开微信
首先，您需要打开手机上的微信应用程序。

![](img/1f660ff9dc89eee59a47f4e49f2a209b_4.png)

## 第二步：进入“发现”页面
打开微信后，您需要点击底部导航栏的“发现”选项。

## 第三步：找到“视频号”入口
在“发现”页面中，向上滑动屏幕。您会看到一个名为“视频号”的入口（图标为黄色蝴蝶形状）。点击进入。

![](img/1f660ff9dc89eee59a47f4e49f2a209b_6.png)

![](img/1f660ff9dc89eee59a47f4e49f2a209b_7.png)

![](img/1f660ff9dc89eee59a47f4e49f2a209b_3.png)
![](img/1f660ff9dc89eee59a47f4e49f2a209b_4.png)

## 第四步：进入个人主页
进入视频号页面后，点击右上角的个人头像图标，进入个人主页。

![](img/1f660ff9dc89eee59a47f4e49f2a209b_6.png)

## 第五步：查看账号信息
在个人主页中向下滑动，找到“我的视频号”部分。这里会显示您的**账号昵称**和**粉丝数量**。
例如，显示为“天道酬勤，39917个粉丝”。

![](img/1f660ff9dc89eee59a47f4e49f2a209b_7.png)

![](img/1f660ff9dc89eee59a47f4e49f2a209b_9.png)

![](img/1f660ff9dc89eee59a47f4e49f2a209b_10.png)

![](img/1f660ff9dc89eee59a47f4e49f2a209b_11.png)

以下是填写邀请函的具体步骤：

1.  **账号昵称**：填写您视频号主页显示的昵称，例如“天道酬勤”。
2.  **粉丝数量**：填写您视频号主页显示的粉丝数，例如“39917”。
3.  **活动邀请人**：根据您的专属渠道，填写指定的邀请人姓名，例如“张良老师”。
4.  **实名信息**：填写您本人的真实姓名，例如“周建国”。
5.  **结算手机号**：填写您用于接收验证码和结算的手机号码，例如“13812345678”。注意号码间不要有空格。

## 第六步：提交信息
所有信息填写完毕后，点击页面上的蓝色“提交”按钮。系统会弹出确认提示“确定要提交吗？”，请点击“确认”完成提交。

![](img/1f660ff9dc89eee59a47f4e49f2a209b_13.png)

![](img/1f660ff9dc89eee59a47f4e49f2a209b_14.png)

![](img/1f660ff9dc89eee59a47f4e49f2a209b_9.png)
![](img/1f660ff9dc89eee59a47f4e49f2a209b_10.png)
![](img/1f660ff9dc89eee59a47f4e49f2a209b_11.png)

提交成功后，页面会提示“已成功加入我们的创作者分成计划”。

![](img/1f660ff9dc89eee59a47f4e49f2a209b_16.png)

![](img/1f660ff9dc89eee59a47f4e49f2a209b_18.png)
