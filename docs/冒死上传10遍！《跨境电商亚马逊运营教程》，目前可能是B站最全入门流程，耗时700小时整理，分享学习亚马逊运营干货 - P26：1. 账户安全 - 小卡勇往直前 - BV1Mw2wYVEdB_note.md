# 亚马逊运营教程：P26：账户安全 🔐

在本节课中，我们将要学习亚马逊运营中至关重要的一环——账户安全。了解并规避账户风险，是确保店铺长期稳定运营的基础。

## 概述

亚马逊账户安全是运营的首要前提。一旦账户销售权限被移除（即店铺被关闭），将导致严重后果，尤其是对于使用FBA（亚马逊物流）的卖家，仓库中的库存将面临巨大损失。因此，理解导致账户被封禁的常见原因并提前预防至关重要。

## 账户被移除销售权限的主要原因

上一节我们概述了账户安全的重要性，本节中我们来看看导致账户销售权限被移除的几种主要情况。

以下是导致账户被封禁的四种核心原因：

1.  **绩效指标不达标**
    *   这指的是卖家账户的绩效表现未能达到亚马逊设定的标准。具体可能包括：
        *   产品退货率过高。
        *   买家差评率过高。
        *   产品质量问题引发多个买家投诉。当投诉累积到一定程度，亚马逊会判定产品存在问题，从而移除该店铺的销售权限。

2.  **出售侵权产品**
    *   侵权是跨境电商区别于国内电商的一大特点。国外对知识产权保护极为严格。
    *   例如，在淘宝上售卖印有Hello Kitty、米老鼠或迪士尼《冰雪奇缘》图案的T恤可能很常见，但在亚马逊、速卖通等跨境平台上，这属于严重的侵权行为。
    *   侵权不限于完全相同的图案，外观近似、长得差不多的产品也可能构成侵权。
    *   一旦确认出售侵权产品，店铺销售权限基本会被移除。在某些平台，如Wish，不仅会关店，账户资金也可能无法取出。

    ![](img/4d5e979b0d33838b247610c4903ae901_1.png)
    ![](img/4d5e979b0d33838b247610c4903ae901_3.png)

3.  **账户关联**
    *   亚马逊要求一个公司原则上只经营一家店铺。如果一家公司运营多个账号并被亚马逊检测到关联，可能导致所有关联账户被关闭。
    *   以下是几种常见的导致关联的情况：
        *   **IP关联**：在同一网络环境下登录两个不同的亚马逊卖家账户。
        *   **设备关联**：在同一台电脑或主机上登录过两个不同的亚马逊账户。
        *   **账户信息关联**：注册两个不同的美国站店铺时，使用了相同的邮箱。
        *   **信用卡关联**：使用同一张信用卡注册多个美国站店铺。
        *   **收款账户关联**：两个不同的美国站店铺绑定了同一个第三方收款账户（如P卡、WF卡）。
        *   **UPC关联**：在两个不同的店铺上传产品时，使用了相同的UPC码。UPC码通常需要在淘宝等渠道购买，每个约五六毛钱。

4.  **缺乏相关认证**
    *   某些产品类目，特别是涉及安全健康的产品，需要提供相应的认证才能销售。
    *   例如，玩具类目因为主要面向儿童，亚马逊会要求提供关于产品健康、成分安全等方面的认证。如果无法提供，则无法在该类目下销售。

## 总结

![](img/4d5e979b0d33838b247610c4903ae901_5.png)

![](img/4d5e979b0d33838b247610c4903ae901_6.png)

本节课中我们一起学习了维护亚马逊账户安全的核心要点。关键是要避免**绩效不达标**、**销售侵权产品**、**触发账户关联**以及**销售需要认证但未获认证的产品**。尤其是选品时，务必远离侵权产品；操作账户时，确保网络、设备、信息的独立性。牢记这些原则，是保障店铺安全运营的第一步。
