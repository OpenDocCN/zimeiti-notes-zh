## 导入与基础剪辑 ✨

打开剪映后，第一步是导入素材。

以下是导入与初步剪辑的步骤：

1.  **导入素材**：点击左上角的“导入”按钮，选择电脑中的视频文件。
    ![](img/2b4474b6271a30d74aca40f4a90d5ec5_1.png)

2.  **预览素材**：素材导入后，可在右上角的预览区查看内容。
    ![](img/2b4474b6271a30d74aca40f4a90d5ec5_3.png)

3.  **添加到轨道**：将素材拖拽至下方的时间轴轨道，或点击素材缩略图上的“+”号。
    ![](img/2b4474b6271a30d74aca40f4a90d5ec5_5.png)

4.  **分割视频**：将时间线指针移动到需要切割的位置，点击工具栏的“分割”按钮。一段视频可被分割成多段。
    ```python
    # 操作逻辑示意：在时间点t进行分割
    if current_time == t:
        split(video_clip)
    ```

5.  **调整顺序**：分割后的视频片段可以拖拽，重新排列组合。

6.  **删除片段**：选中不需要的片段（如开头部分），按`Delete`键删除，或拖动片段边缘进行裁剪。

## 添加音频与字幕 🎵

光有画面不够，我们还需要添加背景音乐和字幕。

### 添加背景音乐

在左侧功能栏点击“音频”。你可以使用软件自带的音乐库，选择平台热门的音乐，这有助于增加视频的吸引力。选中音乐后，拖拽到音频轨道即可。

### 添加字幕

![](img/2b4474b6271a30d74aca40f4a90d5ec5_7.png)

为视频添加字幕非常重要。观众可能在嘈杂环境或不方便外放的环境中观看，字幕能确保信息传递。

以下是添加字幕的方法：

![](img/2b4474b6271a30d74aca40f4a90d5ec5_9.png)

![](img/2b4474b6271a30d74aca40f4a90d5ec5_11.png)

1.  **手动添加**：点击“文本” -> “新建文本”，输入文字内容。在轨道上拖动字幕条可以调整其出现的时间段。

2.  **智能识别**：
    *   **识别字幕**：点击“文本” -> “智能字幕” -> “开始识别”。软件会自动识别视频中的语音并生成字幕，识别后需人工核对修改。
        ![](img/2b4474b6271a30d74aca40f4a90d5ec5_7.png)
    *   **识别歌词**：如果添加的是歌曲，使用“识别歌词”功能通常更准确。
        ![](img/2b4474b6271a30d74aca40f4a90d5ec5_9.png)

![](img/2b4474b6271a30d74aca40f4a90d5ec5_13.png)

3.  **美化字幕**：选中字幕片段，在右上角的参数调节区可以修改字体、颜色、大小、动画等样式。
    ![](img/2b4474b6271a30d74aca40f4a90d5ec5_15.png)

![](img/2b4474b6271a30d74aca40f4a90d5ec5_15.png)

## 使用转场与滤镜 🎨

为了使视频片段之间的衔接更自然，画面更美观，我们需要使用转场和滤镜。

### 添加转场

在视频片段之间，直接切换可能显得生硬。点击左侧“转场”，选择喜欢的转场效果（如叠化、闪黑），将其拖拽到两个视频片段之间即可。拖动转场效果条可以调整转场时长。
![](img/2b4474b6271a30d74aca40f4a90d5ec5_11.png)

### 添加滤镜

滤镜可以统一或改变视频的色调和风格。点击左侧“滤镜”，根据视频内容选择：
*   **风景视频**：选择风景类滤镜。
*   **美食视频**：选择暖色调滤镜。
*   **人物视频**：选择人像类滤镜。
将选中的滤镜拖到视频轨道上方即可生效，也可以通过调整强度控制效果。

![](img/2b4474b6271a30d74aca40f4a90d5ec5_17.png)

## 利用模板提升效率 📐

![](img/2b4474b6271a30d74aca40f4a90d5ec5_19.png)

对于片头、片尾或特定风格的视频（如Vlog、美食），使用模板能极大提升制作效率。

![](img/2b4474b6271a30d74aca40f4a90d5ec5_21.png)

点击左侧“模板”，可以看到大量预设好的视频模板。选择一个模板导入后，通常只需替换其中的图片或视频素材即可快速成片。
![](img/2b4474b6271a30d74aca40f4a90d5ec5_17.png)

例如，一个“点赞关注”的片尾模板，导入后点击“替换素材”，上传自己的图片，模板中的元素就会自动更新。
![](img/2b4474b6271a30d74aca40f4a90d5ec5_19.png)
![](img/2b4474b6271a30d74aca40f4a90d5ec5_21.png)

## 导出最终视频 📤

![](img/2b4474b6271a30d74aca40f4a90d5ec5_23.png)

所有剪辑工作完成后，最后一步是导出视频。

1.  点击右上角的“导出”按钮。
    ![](img/2b4474b6271a30d74aca40f4a90d5ec5_23.png)

2.  设置导出参数：
    *   **保存位置**：选择视频在电脑上的保存文件夹。
    *   **分辨率**：选择`1080P`即可。目前主流平台支持的最佳画质即为1080P，4K并非必需。
    *   **格式**：选择`MP4`。
    *   其他参数（帧率、码率）可保持默认。

![](img/2b4474b6271a30d74aca40f4a90d5ec5_25.png)

3.  点击“导出”按钮，等待渲染完成。
    ![](img/2b4474b6271a30d74aca40f4a90d5ec5_25.png)
