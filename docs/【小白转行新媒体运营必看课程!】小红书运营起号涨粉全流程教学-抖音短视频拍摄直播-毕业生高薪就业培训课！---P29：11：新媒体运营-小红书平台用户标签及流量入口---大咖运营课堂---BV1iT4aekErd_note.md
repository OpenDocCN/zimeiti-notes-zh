# 新媒体运营：P29：11：小红书平台用户标签及流量入口 📊

在本节课中，我们将要学习小红书平台如何为用户打上标签，以及平台有哪些主要的流量入口。理解这些机制，有助于我们分析笔记如何被系统推荐给精准用户，从而优化运营策略。

上一节我们介绍了系统如何判定内容标签，本节中我们来看看平台如何判定用户标签。这个过程可以帮助我们反推笔记被推荐给用户的逻辑，从而思考如何让笔记被更多精准用户看到。

## 用户标签判定机制

系统为用户打标签的主要依据是用户的**浏览行为**。用户注册时选择的兴趣范围参考价值不大，它仅作为初期内容方向的引导，无法进入系统深层的用户标签池。

系统根据用户的浏览行为来判定其兴趣分类。以下是判定用户标签的几个关键行为指标：

以下是判定用户标签的几个关键行为指标：

1.  **点击率**
    *   **公式**：`点击率 = (点击某类内容的次数) / (系统推送的总次数)`
    *   系统在发现页向用户推送多种内容（如美妆、家居、母婴）。用户点击哪类内容更多，系统就能计算出其偏好。例如，用户点击10条内容，其中美妆3次、家居2次、母婴0次，系统会判定该用户对母婴内容不感兴趣，并持续推送更多美妆内容进行试探。

2.  **停留时长**
    *   用户点开笔记后的观看时长。如果用户频繁划走某类笔记，说明内容不吸引他。反之，如果用户对某篇平常较少点击的笔记（如家居）停留很久，仔细阅读图片和评论，系统可能判定其有潜在需求（如装修、购买特定商品），并相应调整推荐。

3.  **页面浏览量**
    *   用户浏览页面的深度。例如，用户不仅看了某篇笔记，还点进博主主页，并连续看了第二、第三篇笔记。这个数值（PV）对账号健康度有积极影响。主页内容越精致，越能提升用户的浏览深度。

4.  **互动行为**
    *   包括点赞、收藏、评论、关注。这些数据直接提升笔记的互动率，有助于笔记进入更大的推荐流量池，同时也能带动账号内其他笔记的曝光和粉丝增长。

通过分析这些行为，系统会为用户建立一个标签云图，其中关联性强的标签（如“美妆”）代表用户的核心兴趣，系统会据此主要推送相关领域的内容。

## 平台流量分发入口

了解用户标签后，我们来看小红书平台主要的流量分发入口。平台通过不同入口满足用户多样化需求。

小红书目前主要有四个流量分发入口：

以下是四个主要的流量分发入口：

1.  **发现页**
    *   这是最主要的流量来源。内容包含：
        *   已关注达人发布的内容。
        *   根据用户标签（点击率、兴趣匹配）推荐的、系统认为用户可能喜欢的内容。
        *   部分发散性内容，以增加平台内容的多样性，探索用户潜在兴趣。

2.  **关注页**
    *   这里集中展示用户所有已关注博主发布的最新笔记。对于粉丝而言，这是查看核心关注者动态的直接入口。

3.  **附近页**
    *   根据手机定位，推送同城或附近区域的内容。此入口以具有强地域特征的内容为主，例如美食、本地旅游、线下店铺推广等。对于运营线下实体店（如美容院、餐厅）的账号，需要重点关注此入口的流量。

4.  **购物页**
    *   这是小红书向电商方向发展的重要入口，直接满足用户的购物需求。用户可在此搜索商品、进入商城或观看带货直播。但需注意，目前许多用户仍习惯通过搜索笔记来“种草”，而非直接进入购物页面。
