## 如何享受软件服务费返还政策

![](img/59a62b7b0b864f023191a0fa178636ce_6.png)

上一节我们介绍了课程的整体内容，本节中我们来看看如何获得软件服务费的返还资格。

![](img/59a62b7b0b864f023191a0fa178636ce_8.png)

![](img/59a62b7b0b864f023191a0fa178636ce_10.png)

只有与平台签订了《软件服务费返还协议》的商家，才能享受软件服务费返还政策。

![](img/59a62b7b0b864f023191a0fa178636ce_12.png)

以下是查看协议状态的方法：
*   商家需使用主账号登录抖音来客。
*   进入“店铺管理” -> “资质中心” -> “合同协议”中查看协议。
*   协议的生效时间，决定了从何时开始的订单会参与软件服务费的返还计算。

![](img/59a62b7b0b864f023191a0fa178636ce_14.png)

## 怎样进行服务费返还对账

![](img/59a62b7b0b864f023191a0fa178636ce_18.png)

![](img/59a62b7b0b864f023191a0fa178636ce_20.png)

了解了返还资格后，接下来我们学习如何进行详细的对账操作。这个过程分为几个关键步骤。

### 第一步：查看平台推送的账单

![](img/59a62b7b0b864f023191a0fa178636ce_22.png)

平台会在每月中上旬，向有返还资格的商家推送上一个月的服务费返还账单。

![](img/59a62b7b0b864f023191a0fa178636ce_24.png)

以下是查看账单的途径：
*   **站内信通知**：商家会在抖音来客的站内信中收到账单通知。
*   **电脑端查看**：使用商家主账号登录抖音来客电脑端，点击“财务管理” -> “服务费返还”进行查看。

![](img/59a62b7b0b864f023191a0fa178636ce_26.png)

**注意**：如果未收到账单或账单金额为零，可能由以下原因导致：
1.  上月前未成功签订《软件服务费返还协议》。
2.  上月前未成功激活抖音来客。
3.  上月没有符合返还条件的订单。
4.  未使用商家主账号，或在PC端未切换到第一个门店查看。

### 第二步：下载服务费返还账单

![](img/59a62b7b0b864f023191a0fa178636ce_28.png)

确认账单后，需要进行下载和核对。再次强调，必须使用商家主账号操作。

![](img/59a62b7b0b864f023191a0fa178636ce_30.png)

以下是下载账单的步骤：
*   登录抖音来客电脑端，进入“财务管理” -> “服务费返还”。
*   点击对应月份的“去确认分账详情”，进入返还账户汇总页。
*   在汇总页点击“导出数据”，即可下载包含每笔订单详细收取与返还信息的文件。

### 第三步：核对返还订单数量

![](img/59a62b7b0b864f023191a0fa178636ce_32.png)

下载明细后，首先要核对订单数量是否正确。

服务费返还是根据协议生效后，在周期内“提现发起”的订单来计算的。例如，协议于7月7日生效，则7月返还账单包含的是7月7日至7月31日期间“提现发起”的订单。

![](img/59a62b7b0b864f023191a0fa178636ce_34.png)

**核对方法**：将下载的返还订单明细中的订单量，与同期在“财务管理” -> “提现记录”中导出的分账明细订单数量进行比对。若数量一致，则订单数量正确。

![](img/59a62b7b0b864f023191a0fa178636ce_36.png)

### 第四步：核对订单返还金额

![](img/59a62b7b0b864f023191a0fa178636ce_38.png)

订单数量核对无误后，接下来需要核对每笔订单的返还金额是否正确。

**返还金额的计算公式**为：
`订单返还金额 = (软件服务费 - 支付服务费) × 返还比例`

![](img/59a62b7b0b864f023191a0fa178636ce_40.png)

**公式解析**：
*   **返还基数**：软件服务费需先扣除0.6%的支付服务费。
*   **返还比例**：不同订单可能适用不同比例，常见的有50%和100%。
    *   **50%比例**：适用于符合合规经营要求的商家订单。
    *   **100%比例**：适用于符合“疫情保护”或“小微商家保护”政策的订单。
*   **计算过程**：每笔订单按公式计算后，会四舍五入保留两位小数，所有订单汇总即得总返还金额。

![](img/59a62b7b0b864f023191a0fa178636ce_42.png)

**关于100%返还的特殊情况**：
1.  **疫情保护**：若某城市因静态管控导致大面积门店无法经营，则该城市特定月份的佣金予以100%返还。
2.  **小微商家保护**：针对2022年6月1日前已入驻，且在2022年6月或7月单月有效核销GMV不高于3000元的商家，平台予以100%返还（该政策仅在6、7月实施）。

**附：软件服务费计算公式**
`软件服务费 = (用户实付 + 抖音支付优惠 + 平台补贴金额) × 软件服务费率`
其中，`(用户实付 + 抖音支付优惠 + 平台补贴金额)` 即为订单实收金额。

![](img/59a62b7b0b864f023191a0fa178636ce_44.png)

![](img/59a62b7b0b864f023191a0fa178636ce_46.png)

### 第五步：确认账单

完成所有核对后，即可确认账单。

商家在收到账单后有10天时间进行对账。在“服务费返还”页面点击“去确认”，进入汇总页完成确认。若核对中发现任何问题，可联系抖音来客右上角的“小助手”发起咨询。

## 什么时候可以收到服务费返还

完成对账确认后，最后我们关心的是款项何时到账。

![](img/59a62b7b0b864f023191a0fa178636ce_48.png)

按照协议约定，商家确认账单后，平台会在**60天内**将软件服务费返还至被收取费用的原账户。

![](img/59a62b7b0b864f023191a0fa178636ce_50.png)

**注意**：如果提现失败，记录会有提示，你可以多次手动重试。若多次尝试仍不成功，需检查账户状态或联系平台处理。
