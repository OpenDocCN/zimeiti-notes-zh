# 小红书运营课程：P24：小红书笔记必备，六大爆文模板全解析

在本节课中，我们将学习如何运用特定的笔记模板，在不同场景下撰写出更容易被小红书平台推荐至首页的优质笔记。课程将重点解析两种核心的爆文模板及其应用技巧。

## 专业教程类模板 📝

上一节我们探讨了笔记的基础结构，本节中我们来看看第一种高效模板：专业教程类。这类模板适用于所有具有明确步骤或流程的内容领域。

专业教程类模板主要适用于美妆、瘦身、美食、软件操作、美容、摄影等领域。这些内容通常遵循“第一步、第二步”的步骤逻辑。

以下是撰写专业教程类笔记的具体建议：

*   **首图设计**：建议采用**对比效果图**或**步骤分解图**。
    *   **对比效果**：例如展示使用某款美妆产品前后的状态对比，或自家产品与别家产品的效果差异。
    *   **步骤分解**：例如摄影教程，首图可展示第一步购买配件、第二步组装、第三步选景、第四步拍照等关键步骤。
*   **标题拟定**：标题需突出内容的特别之处。关于标题的具体技巧，大家可以回顾我们之前关于标题起法的课程内容。
*   **正文撰写**：正文内容应条理清晰，并包含必要元素。
    *   内容需包含相关的**工具**、**材料**或具体**步骤**。
    *   书写时务必逻辑分明，避免信息混杂。建议采用大点套小点（如1.1， 1.2）的结构，并为每个关键步骤配图，使笔记看起来清爽易懂。
    *   对于需要提前准备工具或材料的教程，可以在正文开头单独设立“**准备工作**”部分进行说明。

## 视频笔记类模板 🎬

了解了图文教程的写法后，我们再来看看当下非常流行的视频笔记模板。掌握视频笔记的要点，能显著提升内容的吸引力。

视频笔记模板需重点关注以下几个核心要素：

*   **视频时长**：初期建议控制在**1分钟**以内。获得平台1分钟权限后，可根据内容需要，逐步尝试制作**5分钟**或更长的视频。
*   **视频比例**：必须采用**3:4**的竖屏比例。其公式为 **`高度 : 宽度 = 3 : 4`**。请勿使用传统的4:3横屏比例。

### 提升视频吸引力的关键技巧

当前许多视频面临的问题是开头几秒无法抓住观众眼球，导致用户迅速划走。那么，如何制作出开头就极具吸引力的视频呢？

这里以“毒舌电影”的视频为例进行分析。在介绍电影时，最吸引人的往往是剧情的高潮或最精彩片段，而非平淡的背景铺垫。

毒舌电影的开头方式值得借鉴，也是众多短视频博主正在使用的方法：
> “他已经饿了两天，身无分文，想吃霸王餐……”

*   视频前**2秒**，直接呈现故事中最有趣、最冲突的片段。
*   仅用**3到5句话**，快速展示电影最精彩的部分。
*   **5秒后**，才简要介绍电影年代和核心主题。
*   大约到**13秒**，才开始介绍主角背景。

我们可以借鉴这种“**重点前置**”的叙事结构。例如，在拍摄产品对比类视频时，不应按部就班地从第一步开始讲解。

更吸引人的开头应该是：
> “这是使用我家产品的效果，这是使用别家产品的效果。为何差距如此之大？本期视频为你揭晓真正答案。”

这种开头方式能瞬间抓住观众注意力，对提升视频完播率和互动率有极大帮助。

---

本节课中我们一起学习了小红书的两大爆文笔记模板：**专业教程类模板**与**视频笔记类模板**。关键要点在于：教程类笔记需**步骤清晰、图文结合**；视频类笔记需**控制时长与比例**，并运用**重点前置**的技巧在开头瞬间抓住用户眼球。掌握这些模板，能帮助你更高效地创作出优质内容。
