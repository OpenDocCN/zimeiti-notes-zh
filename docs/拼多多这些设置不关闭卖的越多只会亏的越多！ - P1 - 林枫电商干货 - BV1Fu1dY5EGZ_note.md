# 拼多多运营避坑指南：1：关闭后台扣费设置，防止“卖得越多亏得越多” 🛒💸

在本节课中，我们将学习如何识别并关闭拼多多商家后台中可能导致“订单越多，亏损越多”的几个关键扣费设置。这些设置若未妥善管理，会持续产生额外费用，侵蚀你的利润。

许多商家发现，随着订单量增长，亏损反而加剧。这通常是因为商品定价已是全网最低，同时订单依赖高成本的广告推广获取。然而，除了这些显性成本，后台一些默认开启的设置也会产生隐性扣费，进一步加剧亏损。

上一节我们介绍了问题的普遍性，本节中我们来看看具体需要关闭哪些设置以及如何操作。

以下是三个必须重点关注和关闭的扣费项目：

1.  **消费者体验计划**：该计划会从每笔完成的订单中扣除一笔费用，金额从几毛到几块钱不等。订单量越大，总扣费越高。
2.  **自动跟价**：此功能会自动根据竞争对手的价格调整你的商品售价，可能导致你被迫进入无利可图的价格战。
3.  **仅退款处理**：虽然无法直接关闭“仅退款”功能，但通过正确的后台设置，可以显著减少因“仅退款”造成的损失。

接下来，我们将逐一讲解每个项目的具体关闭或优化方法。

## 关闭“消费者体验计划” 🚫

首先，我们来处理“消费者体验计划”。这个计划会按订单扣费，积少成多。

操作路径为：登录商家后台 -> 进入【售后管理】 -> 找到并点击【消费者体验计划】。

在计划详情页面，你可以看到扣费说明。要关闭它，请点击页面右上角的【更多】按钮，然后选择【停用计划】即可。

![](img/507d0f456e932b4d4e3e3545063f0048_1.png)

## 关闭“自动跟价”功能 ⚙️

![](img/507d0f456e932b4d4e3e3545063f0048_2.png)

接下来，我们关闭可能导致利润受损的“自动跟价”功能。

操作路径为：进入商家后台的【营销活动】页面。找到“自动跟价”相关的设置或说明（通常旁边有一个**问号图标**）。

![](img/507d0f456e932b4d4e3e3545063f0048_3.png)

点击该问号图标，查看“自动跟价说明”。在说明页面中，你会找到【关闭】按钮，点击即可禁用此功能。

![](img/507d0f456e932b4d4e3e3545063f0048_4.png)

## 优化“仅退款”设置 🛡️

![](img/507d0f456e932b4d4e3e3545063f0048_5.png)

最后，我们处理“仅退款”问题。平台没有直接关闭“仅退款”的选项，但可以通过设置来有效防御。

![](img/507d0f456e932b4d4e3e3545063f0048_6.png)

核心方法是完善店铺和商品信息，提升客服响应速度，以减少买家发起仅退款的几率。具体设置方法（如商品描述、售后服务条款等）可参考相关的专题教程。

![](img/507d0f456e932b4d4e3e3545063f0048_7.png)

此外，对于已经产生的、不合理的仅退款订单，平台现已支持商家**不限次数申诉**，并且提高了申诉成功率。商家应积极利用此功能维护自身权益。

![](img/507d0f456e932b4d4e3e3545063f0048_8.png)

---

本节课中我们一起学习了如何关闭拼多多后台的“消费者体验计划”和“自动跟价”功能，并了解了优化设置以减少“仅退款”损失的策略。及时检查并调整这些设置，是防止订单增长却利润下滑的关键一步。尚未关闭这些功能的商家，建议尽快按教程操作。
