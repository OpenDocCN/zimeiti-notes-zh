## 剪辑软件与素材导入 🎬

上一节我们介绍了视频剪辑前的准备工作，本节中我们来看看具体的操作。我们将使用**剪映**App进行剪辑，请确保下载的是黑色图标版本。

![](img/5ad6d51a740b190da8887089830ac3db_6.png)

以下是操作步骤：
1.  打开剪映App，点击“开始创作”。
2.  从手机相册中选择需要剪辑的视频素材，点击“添加”导入。

导入后，视频会出现在剪辑轨道上。如果你的视频由多个片段组成，可以先将第一个片段剪辑好，再点击轨道尾部的“+”号继续添加并剪辑下一个片段，这样可以保持工作流程清晰有序。

## 视频基础剪辑：构图与裁剪 ✂️

视频导入后，第一步是调整画面构图和比例。

以下是调整步骤：
1.  点击轨道上的视频片段，下方工具栏会向右滑动。
2.  找到并点击“编辑”按钮（图标通常是一个方框）。
3.  进入编辑界面后，首先点击“旋转”功能，将画面调整到正确的方向。
4.  如果需要调整画面比例，可以点击“裁剪”功能，选择如 **1:1**、**4:3**、**2:1** 等预设比例，或手动调整裁剪框。

![](img/5ad6d51a740b190da8887089830ac3db_14.png)

调整完毕后，返回主剪辑界面。

## 视频基础剪辑：分割与删除 🗑️

调整好构图后，我们需要修剪掉视频中无用的部分，例如开头沉默、结尾多余的动作等。

![](img/5ad6d51a740b190da8887089830ac3db_20.png)

以下是修剪步骤：
1.  播放视频，找到需要剪掉的起始位置，将时间轴（白色竖线）移动到此位置。
2.  点击选中视频片段，在下方工具栏中找到并点击“分割”按钮。视频会在此处被切成两段。
3.  点击选中需要删除的片段（如前一段或后一段），点击“删除”按钮即可移除。

![](img/5ad6d51a740b190da8887089830ac3db_21.png)

![](img/5ad6d51a740b190da8887089830ac3db_22.png)

![](img/5ad6d51a740b190da8887089830ac3db_23.png)

核心操作可以总结为：**选中 -> 分割 -> 选中 -> 删除**。重复此流程，可以流畅地删除任何不需要的片段。

## 添加与编辑背景音乐 🎵

![](img/5ad6d51a740b190da8887089830ac3db_31.png)

视频内容修剪完成后，下一步是添加背景音乐。

以下是添加音乐的步骤：
1.  点击底部工具栏的“音频”按钮。
2.  如果你已有准备好的配音文件（如从淘金阁生成），可以点击“提取音乐”，从视频中导入音频。
3.  如果没有，则点击“添加音乐”，在音乐库中根据分类（如美食、美妆）或搜索查找合适的歌曲。点击歌曲头像可以预览，选定后点击“使用”即可添加。

如果添加的音乐过长，其剪辑方法与视频相同：将时间轴移到需要截断的位置，**选中音乐轨道 -> 点击“分割” -> 选中多余部分 -> 点击“删除”**。你还可以选中音乐轨道，点击“音量”来调整音乐声音的大小。

## 自动添加与修改字幕 📝

![](img/5ad6d51a740b190da8887089830ac3db_37.png)

添加音乐后，一个完整的视频还需要字幕。手动添加字幕效率很低，我们可以使用自动识别功能。

![](img/5ad6d51a740b190da8887089830ac3db_39.png)

以下是添加字幕的步骤：
1.  点击底部工具栏的“文字”按钮。
2.  选择“识别字幕”，在弹出的菜单中选择“仅识别视频原声”（如果视频已添加背景音乐，避免识别出歌词）。
3.  点击“开始识别”，系统会自动为视频中的人声部分生成字幕。
4.  识别完成后，可以拖动字幕调整位置。双击字幕可以修改错别字，还可以在右侧的“样式”选项中更改字体、颜色等（例如，穿黄衣服可以搭配黄色字体）。

## 视频导出设置与分享 📤

![](img/5ad6d51a740b190da8887089830ac3db_43.png)

视频、音乐、字幕都处理完毕后，就可以准备导出了。导出前需要进行关键设置，以保证视频清晰度。

![](img/5ad6d51a740b190da8887089830ac3db_45.png)

![](img/5ad6d51a740b190da8887089830ac3db_46.png)

以下是导出设置：
1.  点击右上角的“导出”按钮。
2.  在导出设置中，将“帧率”调整为 **30**，这是适合抖音平台的帧率。
3.  将“分辨率”拉到最高（如1080P），分辨率越高视频越清晰。
4.  设置完成后，点击“导出”。导出过程中请勿锁屏或切换程序，耐心等待完成。
5.  导出后，可以直接通过剪映分享到抖音平台。

![](img/5ad6d51a740b190da8887089830ac3db_50.png)

![](img/5ad6d51a740b190da8887089830ac3db_51.png)

![](img/5ad6d51a740b190da8887089830ac3db_52.png)

以下是爱提词的使用方法：
1.  下载并打开“爱提词”App。
2.  点击右下角的“+”号，创建新台词稿。你可以直接输入或粘贴准备好的文案，并设置一个标题。
3.  保存后，点击“开始提词”，选择“录制模式”。
4.  在提词界面，你可以一边看着滚动显示的台词，一边进行录制，无需背诵。
5.  点击界面中间的设置按钮，可以调整字体大小，方便阅读。
6.  点击左下角的“美颜”功能，可以开启瘦脸、大眼、磨皮等效果，让拍摄效果更自然美观。
