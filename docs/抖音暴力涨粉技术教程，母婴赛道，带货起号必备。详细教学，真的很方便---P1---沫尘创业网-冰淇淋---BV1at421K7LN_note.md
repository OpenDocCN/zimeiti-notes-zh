# 抖音母婴赛道暴力涨粉教程：P1：三天千粉起号实操指南 🚀

在本节课中，我们将学习一种针对母婴赛道的抖音快速涨粉方法。该方法通过制作特定类型的视频内容，旨在三天内吸引超过1000名粉丝，为后续带货打下基础。教程将详细拆解从素材准备到视频制作、发布的完整流程。

## 账号选择与风险提示 ⚠️

上一节我们介绍了课程目标，本节中我们来看看操作前的准备工作。首先，账号的选择至关重要。

建议使用注册时间超过一个月、且近期未发布过作品的老号进行操作。这类账号通常具有较高的平台权重。例如，家人长期使用但只刷视频不发作品的账号是理想选择。

## 素材准备与下载 🎬

![](img/1c0e5706fd85774e9050a12307780688_5.png)

准备好账号后，接下来我们需要准备视频素材。以下是具体的操作步骤。

![](img/1c0e5706fd85774e9050a12307780688_7.png)

![](img/1c0e5706fd85774e9050a12307780688_9.png)

1.  **下载解压类视频**：打开快手APP，搜索“解压视频”。选择两个类似示例的视频，通过分享功能复制链接，并使用微信小程序等工具去除水印后下载。
2.  **下载奥特曼剧情视频**：在快手搜索“奥特曼”，寻找带有剧情的片段（例如“贝利亚对战”这类）。同样复制链接并去水印下载。

至此，我们共需准备三个视频素材：两个解压视频，一个奥特曼剧情视频。

## 视频剪辑与合成 ✂️

![](img/1c0e5706fd85774e9050a12307780688_17.png)

素材准备完毕后，我们进入核心的视频制作环节。我们将使用剪映APP将多个视频合成为一个。

![](img/1c0e5706fd85774e9050a12307780688_19.png)

1.  **导入与调整主画面**：打开剪映，点击“开始创作”，首先导入下载好的奥特曼剧情视频。导入后，选中视频轨道，点击“编辑”->“调整大小”，将画面适当调小，使其能完整显示在屏幕左下方。
2.  **添加画中画（解压视频）**：点击“新建画中画”，导入第一个解压视频。同样使用“编辑”->“调整大小”功能将其调小，并移动位置，使其与主画面并排对齐。重复此步骤，添加第二个解压视频，并调整至合适大小与位置。
3.  **调整奥特曼主视频**：再次选中底层的奥特曼视频轨道，使用“调整大小”功能将其画面放大至铺满整个屏幕背景。
4.  **添加统一特效**：点击“特效”->“画面特效”，搜索并添加“雪花”特效。添加后，务必点击“作用对象”，选择“全局”，使特效应用于整个合成视频。
5.  **处理视频时长**：确保所有视频轨道的时长一致。如果长度不同，可以拖动轨道末尾进行裁剪，保留需要的部分。
6.  **添加真人出镜层（可选）**：点击“新建画中画”，导入一段自己拍摄的真人视频。将画面铺满后，点击“混合模式”，选择“绿色”。通过调整透明度滑块，使真人图层半透明地叠加在背景上，形成若隐若现的效果。
