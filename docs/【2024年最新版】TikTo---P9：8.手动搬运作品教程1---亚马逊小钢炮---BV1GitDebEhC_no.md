## 素材来源与选择

![](img/9690a7c632c4878a880b37e2d53af62f_6.png)

进行搬运前，首先需要寻找合适的视频素材。素材可以来源于抖音、快手、火山、微视等国内主流短视频平台。

![](img/9690a7c632c4878a880b37e2d53af62f_8.png)

以下是选择素材时需要注意的关键点：
*   **避免复杂对白**：尽量选择没有人物对白或对白简单的视频。因为目标观众是外国人，复杂的语言内容他们可能无法理解。
*   **选择通用内容**：优先选择宠物、风景、美食、手工、运动等视觉冲击力强、无需语言也能看懂的内容。例如，可爱的宠物视频在全球范围内都受欢迎。
*   **注意版权与肖像权**：尽量避免使用有明显商业标识或涉及他人肖像权的视频，以减少潜在风险。

## 去水印操作步骤

![](img/9690a7c632c4878a880b37e2d53af62f_14.png)

![](img/9690a7c632c4878a880b37e2d53af62f_15.png)

如果视频无法直接保存到相册，通常是因为作者设置了权限。这时，我们需要通过复制链接的方式，借助工具去除平台水印。

1.  **复制视频链接**：在抖音APP中，打开目标视频，点击右下角的“分享”按钮。在弹出的菜单中，向左滑动找到“复制链接”选项并点击。
    ```
    操作路径：视频页面 -> 点击分享按钮 -> 向左滑动 -> 点击“复制链接”
    ```

![](img/9690a7c632c4878a880b37e2d53af62f_17.png)

2.  **使用去水印工具**：打开微信，在顶部搜索框中搜索小程序“青抖”。进入小程序后，找到“去水印”功能。

![](img/9690a7c632c4878a880b37e2d53af62f_19.png)

3.  **解析并保存视频**：将刚才复制的链接粘贴到指定输入框，小程序会自动解析视频。解析完成后，你会看到一个去除了左上角和右下角水印的高清视频，点击“保存到相册”即可。

![](img/9690a7c632c4878a880b37e2d53af62f_21.png)

![](img/9690a7c632c4878a880b37e2d53af62f_23.png)

## 二次剪辑简介

![](img/9690a7c632c4878a880b37e2d53af62f_25.png)

将去好水印的视频保存到手机相册后，工作并未结束。直接发布搬运的视频容易被平台识别为重复内容。因此，需要进行二次剪辑，以增加视频的原创性。

常用的剪辑工具是“剪映”，功能全面且免费，完全能满足基础剪辑需求。在接下来的课程中，我们将详细讲解如何进行有效的二次剪辑。

![](img/9690a7c632c4878a880b37e2d53af62f_27.png)

![](img/9690a7c632c4878a880b37e2d53af62f_28.png)

本节课中我们一起学习了手动搬运视频的核心步骤：从国内平台选取合适素材，利用“复制链接”和“青抖”小程序去除水印，并了解了后续进行二次剪辑的必要性。这是构建TikTok内容库的实用入门方法。
