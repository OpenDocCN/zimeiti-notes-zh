# 抖音运营实战：P13：7. 视频剪辑实操（下） ✂️

## 概述
在本节课中，我们将继续学习视频剪辑的核心步骤，重点掌握如何为多片段视频添加转场、统一视频比例以及添加字幕。这些操作是提升视频观感、保证完播率的关键。

上一节我们介绍了视频素材的导入与初步剪辑，本节中我们来看看如何将这些片段流畅地衔接起来，并完善视频的最终呈现。

---

## 添加转场：提升视频流畅度

![](img/c44d81733968b751a1a60cc4059d6565_1.png)

我们剪辑的视频通常由多个片段组成，每个片段时长可能在十秒左右。这些片段之间需要有自然的衔接，因此必须添加转场效果。

![](img/c44d81733968b751a1a60cc4059d6565_3.png)

转场的作用是让用户明确感知到视频内容进入了下一个不同的片段。如果不添加转场，观众在切换到新片段时可能会感到突兀，甚至退出视频查看自己是否点错了内容。一旦用户退出，视频的**完播率**就会直线下降，而完播率直接影响视频的推荐量和最终收益。

![](img/c44d81733968b751a1a60cc4059d6565_5.png)

以下是添加转场的具体步骤：

![](img/c44d81733968b751a1a60cc4059d6565_7.png)

1.  在视频编辑轨道上，找到右侧的 **“+”** 号按钮并点击。
2.  在弹出的菜单中，选择 **“素材库”**。
3.  在素材库的 **“热门”** 分类下，可以找到大量通用的转场素材，例如搞笑的动画或过渡效果。目前大多数视频使用的转场都来源于此。
4.  选择转场时，注意时长。避免选择过长的转场（如16秒），观众通常没有耐心看完。最合适的转场时长应控制在 **3到5秒** 之间，我们以选择一个3秒的转场为例。

![](img/c44d81733968b751a1a60cc4059d6565_9.png)

添加好转场素材后，还需要进行关键一步：统一视频比例。

![](img/c44d81733968b751a1a60cc4059d6565_11.png)

---

## 统一画布比例：保持视觉一致

![](img/c44d81733968b751a1a60cc4059d6565_13.png)

所有视频元素，包括转场，都必须保持统一的画布比例，即 **16:9** 的横屏比例。

![](img/c44d81733968b751a1a60cc4059d6565_15.png)

以下是统一转场素材比例的步骤：

1.  选中已添加的转场素材片段。
2.  在编辑菜单中找到 **“画布模糊”** 或类似功能（不同剪辑App名称可能略有不同）。
3.  选择第一个清晰的画面背景选项。这个操作会将转场素材的背景适配为标准的16:9横屏画布。
4.  最后，点击右下角的 **对勾（✓）** 确认设置。

![](img/c44d81733968b751a1a60cc4059d6565_17.png)

至此，视频的多片段衔接与比例统一就完成了。记住这个流程：裁剪视频比例 -> 添加转场 -> 统一转场画布比例。

## 添加字幕：提升信息传达效率

![](img/c44d81733968b751a1a60cc4059d6565_19.png)

视频剪辑的下一步是为视频添加字幕（文案），这能帮助观众更好地理解内容，尤其在无声播放或环境嘈杂时。

![](img/c44d81733968b751a1a60cc4059d6565_21.png)

以下是添加字幕的步骤：

1.  点击手机剪辑软件左下角的 **“文字”** 或 **“T”** 图标选项。
2.  选择 **“新建文本”**，然后点击右侧的 **“识别字幕”** 功能。
3.  点击 **“开始匹配”**。软件会自动识别视频中的语音，并生成对应的字幕文本。
4.  字幕生成后，可以进行微调。拖动字幕文本框右下角，可以放大字幕。
5.  将放大后的字幕文本框拖动到视频画面的最下方，这个位置通常观感最舒适。调整好第一句字幕后，后续字幕通常会同步应用此样式，非常方便。

---

## 核心要点与总结

![](img/c44d81733968b751a1a60cc4059d6565_23.png)

本节课我们一起学习了视频剪辑后半部分的核心操作：

1.  **添加转场**：在多视频片段间插入3-5秒的转场素材，提升流畅度，保障完播率。公式表示为：`视频完播率 ∝ 观看体验流畅度`。
2.  **统一比例**：确保所有视频片段和转场都设置为 **16:9** 的横屏比例，保持视觉一致。
3.  **添加字幕**：利用 **“识别字幕”** 功能快速生成字幕，并调整其大小和位置至画面底部。

![](img/c44d81733968b751a1a60cc4059d6565_25.png)

整个视频制作流程需要严格遵循步骤：寻找合适时长的素材 -> 按16:9比例裁剪 -> 添加转场并统一比例 -> 最后识别并添加字幕。避免使用过长或不符合规范的素材，这会影响平台的审核通过率。

![](img/c44d81733968b751a1a60cc4059d6565_27.png)

只要按部就班地操作，视频剪辑并不难。掌握这些技巧，你的视频将更具专业性和吸引力。
