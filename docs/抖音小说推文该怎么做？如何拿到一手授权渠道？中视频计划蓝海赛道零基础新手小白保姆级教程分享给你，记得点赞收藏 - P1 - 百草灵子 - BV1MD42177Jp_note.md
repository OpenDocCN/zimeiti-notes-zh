# 抖音小说推文项目：P1：零基础新手保姆级教程 📚

在本节课中，我们将学习一个无需实拍、无需原创文案的抖音变现项目——小说推文。我们将了解其运作原理、核心操作步骤以及如何获取关键授权，帮助零基础新手快速入门。

## 项目原理与优势 💡

上一节我们介绍了课程主题，本节中我们来看看这个项目的核心原理和优势。

这个玩法是近期非常火爆的小说推文项目。它通过发布视频为小说平台推荐读者，从而获取佣金。

这种视频无需真人实拍出镜，也无需自行撰写文案。文案直接复制小说原文段落即可。同时，它也不需要投放“抖加”进行推广，操作难度非常小。

可以说，这是最适合新手起步的项目之一。

## 视频案例与变现模式分析 🎬

了解了基本原理后，我们通过一个具体案例来看看它是如何运作和变现的。

请看一个抖音视频案例：视频文案是“我看着身边的男朋友陷入沉思。我运营了我们大学的表白墙账号……”，画面则播放着解压类视频。这条视频获得了2000多个点赞和几十万播放量。

不要小看这几十万播放量，仅这一条视频就为创作者带来了两三千元的收益。

那么，这种视频是如何变现的呢？难道仅靠“中视频计划”的播放量收益就能达到这个数额吗？答案是否定的。

除了参与“中视频计划”获取播放收益外，**最关键的一点在于视频带有CPA（按行动付费）推广的性质**。虽然视频播放的是解压画面，但文案一直在朗读某部小说的内容。只要有人通过这个视频去搜索小说的完整版，创作者每为小说平台成功推荐一位读者，就能获得约十元左右的佣金。这部分收益不亚于“中视频计划”。

如果你查看这类视频的评论区，通常会发现作者置顶了一条引导用户寻找小说完整版的评论。玄机就在这条置顶评论里。

作者让你去搜索的“关键词”，可能并非小说原名，而是平台与作者约定好的特定关键词。用户通过这个关键词在平台上找到小说并成为用户后，就算作者成功为平台推荐了一名用户。这个关键词就是作者与平台之间的约定。

## 项目操作全流程 🛠️

我们已经明白了项目的盈利逻辑，接下来进入核心环节，看看具体如何操作。

![](img/683f815b8091b6e12c47b80e05b8ccd6_1.png)

要操作这个项目，主要分为两大步骤。

### 第一步：成为推荐官，获取授权

第一步是成为小说平台的“推荐官”，拿到授权。这意味着我们需要获得每部小说对应的推广关键词。

关于如何具体获取授权，由于平台规则限制，无法在正文中直接说明。我将在教程结尾部分告知大家操作方法。

### 第二步：制作发布视频

获取授权后，至关重要的一步就是视频制作。实际上，制作过程非常简单。

以下是制作视频的具体步骤：

1.  **准备文案**：无需自己写文案。直接从小说中复制那些比较经典、能引发好奇心的段落，大约1000字左右。
2.  **生成配音**：将复制好的文案粘贴到AI配音工具中，选择一个听起来舒服的主播声音，然后导出配音音频文件。
3.  **准备画面**：寻找一些解压类视频素材（如一开始展示的案例）。**注意**：不要只使用一个解压视频，而是选取多个进行重新组合剪辑。
4.  **合成视频**：使用剪映等剪辑软件，导入配音音频和解压视频素材。将解压视频的原声调小或关闭，确保配音清晰。最后导出成品视频。

这样的视频生产速度非常快，基本上十分钟以内就能完成一个。

## 收益测算与项目前景 📈

掌握了制作方法，我们最后来算一笔账，看看这个项目的收益潜力如何。

根据了解，小说推文项目在抖音上大约每1万次播放，能产生2到5个订单。每单佣金大约在十元左右。

假设你操作十个抖音号，每个号每天发布三条视频。即使每条视频只有抖音给予的500基础播放量，日总播放量也能达到1.5万，预计能产生5单以上收益。

事实上，我们圈子内有很多小伙伴，一个人就能操作二三十个账号，这毫不夸张。据观察，基本上每发布100条作品，其中肯定会产生一条爆款视频。一旦出现爆款，播放量会非常高，仅这一条视频就有可能带来数千元的收益。
