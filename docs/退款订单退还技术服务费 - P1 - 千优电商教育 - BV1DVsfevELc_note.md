# 拼多多商家教程：P1：退款订单技术服务费退还政策解读 🛒

在本节课中，我们将要学习拼多多平台于近期发布的一项关于技术服务费的重要政策调整。这项调整直接关系到商家的经营成本，理解其具体内容对店铺运营至关重要。

上一节我们介绍了课程主题，本节中我们来看看这项政策的具体内容与背景。

拼多多平台发布了一项新政策。该政策规定，自9月24日起，所有发生退款的订单，其对应的技术服务费将退还给商家。拼多多平台原先的规则是，无论用户最终是完成订单还是申请退款，商家都需要支付订单金额千分之六的技术服务费，且该费用不予退还。新政策执行后，这部分费用在订单退款时将予以返还。

目前尚不清楚拼多多平台推出此项政策的具体战略意图。但从9月份开始，平台面向商家的一系列政策看上去呈现出利好的趋势。
