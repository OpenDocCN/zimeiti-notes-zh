在本节课中，我们将学习如何调整视频素材的颜色，这是提升视频观感的关键一步。我们将重点介绍两种方法：使用预设的“滤镜”和手动进行“调节”。

上一节我们介绍了基础的剪辑操作，本节中我们来看看如何美化视频画面。在许多视频剪辑软件中，滤镜是一个非常实用的功能。

![](img/d20709075295c85635e5cef3271db005_0.png)

滤镜功能提供多种预设效果，能有效掩盖拍摄画面中的缺陷，对于不熟悉调色的用户来说是一个好用的工具。它类似于常用的预设模板，无需手动调整亮度、饱和度等参数。

![](img/d20709075295c85635e5cef3271db005_8.png)

![](img/d20709075295c85635e5cef3271db005_10.png)

![](img/d20709075295c85635e5cef3271db005_12.png)

以下是使用滤镜的基本步骤：

![](img/d20709075295c85635e5cef3271db005_14.png)

![](img/d20709075295c85635e5cef3271db005_16.png)

![](img/d20709075295c85635e5cef3271db005_18.png)

1.  将素材拖入时间轴。
2.  在上方工具栏中找到并单击“滤镜”工具。
3.  在打开的滤镜面板中，根据素材类型（如风景、美食、人像）选择分类。
4.  单击不同滤镜可实时预览效果。
5.  将选中的滤镜拖拽到时间轴上，会生成一个独立的滤镜轨道。
6.  调整滤镜轨道的时长，使其与视频素材对齐。
7.  选中滤镜轨道，可在右侧参数面板调整其应用强度。

接下来，我们重点讲解手动调节的核心参数：

![](img/d20709075295c85635e5cef3271db005_73.png)

![](img/d20709075295c85635e5cef3271db005_75.png)

![](img/d20709075295c85635e5cef3271db005_77.png)

![](img/d20709075295c85635e5cef3271db005_79.png)

*   **色温**：调整画面色彩的冷暖倾向。
    *   **公式**：`色温值 ↑ → 画面偏暖（黄/红）；色温值 ↓ → 画面偏冷（蓝）`
*   **色调**：调整画面色彩的颜色倾向。
    *   **公式**：`色调值 ↑ → 画面偏绿；色调值 ↓ → 画面偏紫红`
*   **饱和度**：调整画面色彩的鲜艳程度。
    *   **公式**：`饱和度值 ↑ → 色彩更鲜艳；饱和度值 ↓ → 色彩更灰暗`
*   **亮度**：调整画面的整体明暗程度。
    *   **公式**：`亮度值 ↑ → 画面更亮；亮度值 ↓ → 画面更暗`
*   **对比度**：调整画面中明暗部分的差异。
    *   **公式**：`对比度值 ↑ → 黑白层次更分明，色彩更浓郁；对比度值 ↓ → 画面更平淡`
*   **高光**：调整画面中最亮部分的明暗。
*   **阴影**：调整画面中最暗部分的明暗。
*   **光感**：综合调整画面的光照感。
*   **效果类参数**：
    *   **锐化**：增加画面边缘清晰度。
    *   **颗粒**：为画面添加胶片颗粒感。
    *   **褪色**：降低画面整体色彩浓度。
    *   **暗角**：使画面四角变暗，突出中心。

调色是一门专业学问，本课仅做基础功能介绍。对于初学者，**推荐优先使用滤镜预设**，这能极大提升工作效率。当使用手动调节时，务必明确调节目的，避免盲目调整。

![](img/d20709075295c85635e5cef3271db005_101.png)

![](img/d20709075295c85635e5cef3271db005_103.png)

![](img/d20709075295c85635e5cef3271db005_105.png)

本节课中我们一起学习了视频调色的两种方法：便捷的**滤镜预设**和精细的**手动调节**。掌握这些工具，能帮助你快速提升视频的视觉质感，使其更符合小红书的平台调性。记住，对于起步阶段，善用现成滤镜是高效又出彩的选择。
