# 微信视频号运营教程：P35：19.实操步骤

在本节课中，我们将学习如何开通视频号的商品橱窗功能，这是进行视频号带货的基础操作。整个过程简单直接，我们将分步进行讲解。

上一节我们介绍了视频号运营的基本概念，本节中我们来看看如何具体开通带货权限。

## 第一步：打开微信
首先，我们需要启动微信应用。这是所有后续操作的基础。

![](img/c2160bfbca5c38191d5698a766612d54_1.png)

## 第二步：进入“发现”页
打开微信后，点击底部导航栏的 **“发现”** 选项。

![](img/c2160bfbca5c38191d5698a766612d54_2.png)

## 第三步：找到视频号入口
在“发现”页面，将视线向上移动。您会看到 **“视频号”** 的文字入口，其图标是一个黄色的蝴蝶形状。点击此处进入视频号。

![](img/c2160bfbca5c38191d5698a766612d54_4.png)

![](img/c2160bfbca5c38191d5698a766612d54_6.png)

![](img/c2160bfbca5c38191d5698a766612d54_1.png)
![](img/c2160bfbca5c38191d5698a766612d54_2.png)

## 第四步：进入个人中心
进入视频号后，在页面右上角找到一个人形头像图标，点击它进入个人中心。

![](img/c2160bfbca5c38191d5698a766612d54_4.png)
![](img/c2160bfbca5c38191d5698a766612d54_6.png)

![](img/c2160bfbca5c38191d5698a766612d54_8.png)

![](img/c2160bfbca5c38191d5698a766612d54_9.png)

## 第五步：进入创作者中心
在个人中心页面，向下滑动，直到页面最底部。找到并点击 **“创作者中心”**。如果未看到此选项，可能是因为账号为新注册且未发布过作品，发布一条作品后即可出现。

## 第六步：找到带货中心
进入创作者中心后，在页面中找到 **“带货中心”** 的入口，其图标类似一个小卖部。

![](img/c2160bfbca5c38191d5698a766612d54_8.png)
![](img/c2160bfbca5c38191d5698a766612d54_9.png)

## 第七步：开通橱窗权限
点击“带货中心”后，系统会根据您的账号状态进行跳转。以下是两种情况的处理方式：

![](img/c2160bfbca5c38191d5698a766612d54_11.png)

如果尚未开通橱窗权限，点击后会跳转到开通界面。

以下是开通权限的具体步骤：
1.  **选择带货身份**：页面会要求您选择身份。请务必选择 **“我是达人”** 这个选项。
    ![](img/c2160bfbca5c38191d5698a766612d54_11.png)
2.  **缴纳保证金**：选择“我是达人”后，会跳转到保证金缴纳界面。您需要点击 **“去交纳”**，并通过微信支付100元保证金即可完成开通。
    ![](img/c2160bfbca5c38191d5698a766612d54_13.png)

如果已经开通了橱窗权限，点击“带货中心”则会直接进入商品上架管理界面。

## 操作流程总结
为了方便记忆，我们将整个操作流程总结如下：

![](img/c2160bfbca5c38191d5698a766612d54_13.png)

1.  **打开微信** -> `启动微信应用`
2.  **点击“发现”** -> `进入发现页面`
3.  **进入“视频号”** -> `点击黄色蝴蝶图标`
4.  **点击右上角头像** -> `进入个人中心`
5.  **进入“创作者中心”** -> `在个人中心底部`
6.  **点击“带货中心”** -> `图标类似小卖部`
7.  **开通或管理橱窗**：
    *   **未开通**：选择`我是达人` -> 缴纳`100元保证金`
    *   **已开通**：直接进入商品管理界面

![](img/c2160bfbca5c38191d5698a766612d54_15.png)

![](img/c2160bfbca5c38191d5698a766612d54_15.png)

本节课中我们一起学习了开通视频号商品橱窗的完整步骤。核心操作在于进入创作者中心的带货中心，并根据引导完成达人身份选择和保证金缴纳。开通后，您就可以在视频号中上架商品进行带货了。整个过程无需粉丝门槛，操作简便。
