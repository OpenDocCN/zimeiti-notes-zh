# 小红书电商运营课：P21：第8课：如何在抖店选品 🛍️

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_1.png)

在本节课中，我们将学习如何利用抖音的“抖店”平台进行选品。这是一种低成本、高效率的方法，可以帮助我们发现那些在小红书平台上也具备爆款潜力的商品。

## 为什么要在抖店选品？ 🤔

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_3.png)

上一节我们介绍了选品的重要性，本节中我们来看看为什么选择抖店作为选品来源。核心逻辑在于：互联网爆款具有共性。抖音和小红书同属新媒体平台，且抖音拥有成熟的电商体系（即抖店）。因此，在抖音电商中销售表现良好的商品，有很大概率在小红书平台也能获得成功。这就是我们选择在抖店选品的原因。

## 选品前的准备工作 🛠️

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_11.png)

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_13.png)

在开始具体操作前，我们需要做一些简单的准备。抖店选品有多种方法，包括使用付费数据软件。但为了坚持轻资产、低成本创业的原则，我们将使用抖音橱窗的“选品广场”功能，这是一个免费且有效的工具。

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_15.png)

以下是使用选品广场的前提条件：
*   **拥有开通橱窗的抖音账号**：通常个人抖音号可能已经开通。若未开通，需要缴纳**500元**保证金进行开通（此保证金后续可退还）。
*   **或借用他人账号**：如果身边有朋友已开通橱窗，可以暂时借用其账号进行操作。

## 抖店选品实操步骤 📱

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_25.png)

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_27.png)

现在，我们开始一步步学习如何在抖音橱窗的选品广场中找到潜力商品。

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_29.png)

**第一步：进入选品广场**
1.  打开抖音APP，进入个人主页。
2.  点击“橱窗”入口。
3.  在橱窗首页，找到并点击“选品广场”。

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_31.png)

**第二步：利用榜单筛选商品**
进入选品广场后，平台会展示各类商品。为了高效找到潜力品，我们使用榜单功能。
1.  点击顶部的“搜索”框。
2.  清空搜索词，页面会自动下拉。
3.  此时你会看到几个关键榜单：“爆款热卖榜”、“搜索热度榜”和“搜索趋势榜”。这些榜单基于销售数据和搜索趋势排序，是我们选品的重要参考。

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_33.png)

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_35.png)

## 核心选品原则 📝

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_37.png)

在浏览商品时，我们需要依据几个核心原则进行判断，而不仅仅是看销量。以下是筛选潜力品的四大原则：

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_39.png)

1.  **趋势向上**：商品的整体销售曲线应呈现上升趋势，而不是下降或平稳。
2.  **款式普适**：商品款式不应过于挑人（例如，过于独特或挑身材的服装），否则容易导致高退货率。
3.  **应季性强**：选择当前季节需要的商品，转化率会更高。
4.  **销量适中**：对于服装等类目，建议选择月销量在 **1000件以上，5000件以下** 的商品。这个区间表明商品有一定热度，但竞争可能尚未白热化，属于“趋势品”。此标准可灵活调整，对于全新商品，销量更低也可考虑。

## 实战案例演示 👗

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_46.png)

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_48.png)

让我们以“童装”为例，实际演练一遍选品流程。

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_50.png)

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_52.png)

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_54.png)

1.  **搜索与初选**：在选品广场搜索“亲子童装”。从结果中，我们选择“女童格子套装”这个细分品类进行查看。
2.  **应用原则筛选**：在列出的众多商品中，我们需要根据上述原则进行判断。
    *   **看趋势**：点击感兴趣的商品，查看其销售趋势图。忽略短期波动，观察**整体曲线是否向上**。
    *   **看款式**：商品图片展示的款式是否大方、流行，适合多数人。
    *   **看季节**：确认这是春季适穿的服装。
    *   **看销量**：检查月销量是否符合我们的区间（例如，1000-5000件）。如果符合条件的很少，可以适当放宽标准至500件甚至100件，尤其是对新款。
3.  **确定候选商品**：找到一款符合多项原则的商品（例如，趋势向上、款式普适、应季、月销1000多件）。点击进入商品详情页。

## 货源查找与利润测算 💰

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_60.png)

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_62.png)

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_64.png)

在抖店选定潜在爆款后，下一步是寻找低价货源并计算利润空间。我们通常使用拼多多（简称“多多”）作为货源平台。

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_66.png)

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_68.png)

以下是具体操作：
1.  **截图**：在抖店商品详情页，对商品首屏进行截图。
2.  **以图搜货**：打开拼多多APP，使用图片搜索功能，上传刚才的截图。
3.  **筛选货源**：在搜索结果中，寻找同款或类似款商品。关注两个关键点：
    *   **评价数**：拼多多的销量数据可能不准确，但**评价数**相对真实。一个拥有几百个评价的商品，其销量通常是真实的。
    *   **价格**：对比抖店的售价和拼多多的进货价，计算差价。
4.  **利润计算**：找到合适的货源后，计算单件利润。
    *   **公式**：`单件利润 ≈ 抖店售价 - 拼多多进货价`
    *   **案例**：若抖店售价为**99元**，拼多多进货价为**49元**，则单件利润约为**50元**。这表明服装类目的利润空间可能很大。

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_70.png)

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_72.png)

**定价提示**：在小红书发布时，定价可以参考抖店的售价，甚至可以根据平台调性适当调整，无需过于低价。

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_74.png)

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_76.png)

## 课程总结 🎯

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_80.png)

![](img/06a10201471fc6c6a9cc3e5d619a2d0c_82.png)

本节课中我们一起学习了如何利用抖音“抖店”平台为小红书电商选品。我们首先理解了跨平台选品的逻辑，然后完成了从进入选品广场、利用榜单、应用四大选品原则筛选商品，到最终在拼多多查找货源并计算利润的全流程。记住核心在于寻找**有上升趋势、款式普适、应季且销量适中**的“趋势品”，并通过比价确保有合理的利润空间。这套方法能帮助你在没有昂贵数据工具的情况下，高效地找到潜在爆款。
