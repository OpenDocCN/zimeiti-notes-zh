上一节我们介绍了在抖音寻找同行的方法，本节中我们来看看在小红书平台如何操作。小红书的核心内容形式是图文，因此我们的主要目标是寻找优质的图文达人，并将其内容转化为视频，以获得更好的效果。

## 寻找对标账号的方法

![](img/32848607951ae0b4bd73791653928743_5.png)

![](img/32848607951ae0b4bd73791653928743_7.png)

![](img/32848607951ae0b4bd73791653928743_11.png)

1.  **利用数据软件筛选**：如果条件允许，可以使用支持小红书筛选功能的数据分析工具来寻找优质账号。

    ![](img/32848607951ae0b4bd73791653928743_3.png)

## 养号与内容推荐机制

![](img/32848607951ae0b4bd73791653928743_23.png)

![](img/32848607951ae0b4bd73791653928743_25.png)

“养号”的目的是让小红书算法识别您的账号兴趣，从而为您推荐更多垂直领域的内容。以下是养号的核心操作：

![](img/32848607951ae0b4bd73791653928743_27.png)

![](img/32848607951ae0b4bd73791653928743_29.png)

*   **主动搜索与关注**：前期，您需要主动搜索您的产品领域（例如“童装”），并大量关注该领域的图文达人。
    ![](img/32848607951ae0b4bd73791653928743_11.png)

![](img/32848607951ae0b4bd73791653928743_31.png)

![](img/32848607951ae0b4bd73791653928743_33.png)

*   **互动行为影响推荐**：平台的推荐机制会根据您的行为进行调整。如果您关注图文类内容多，系统就会为您推荐更多图文；关注带货内容多，则会推荐更多带货笔记。
    ![](img/32848607951ae0b4bd73791653928743_13.png)

![](img/32848607951ae0b4bd73791653928743_34.png)

![](img/32848607951ae0b4bd73791653928743_36.png)

*   **识别优质账号**：在关注过程中，要学会判断账号质量。关注那些内容专业、以图文为主、笔记爆款概率（点赞量高）大的账号。
    ![](img/32848607951ae0b4bd73791653928743_15.png)
    ![](img/32848607951ae0b4bd73791653928743_17.png)
    ![](img/32848607951ae0b4bd73791653928743_21.png)
    ![](img/32848607951ae0b4bd73791653928743_23.png)

![](img/32848607951ae0b4bd73791653928743_38.png)

![](img/32848607951ae0b4bd73791653928743_40.png)

*   **利用系统推荐**：在关注某个优质达人后，平台会推荐“可能感兴趣的人”，这些账号属性通常非常相似，可以批量点击关注，以快速强化账号标签。
    ![](img/32848607951ae0b4bd73791653928743_29.png)
    ![](img/32848607951ae0b4bd73791653928743_31.png)

![](img/32848607951ae0b4bd73791653928743_42.png)

![](img/32848607951ae0b4bd73791653928743_44.png)

完成上述操作后，您的“推荐”页面将逐渐被垂直领域的内容填满，这标志着养号初步成功。

## 内容判断与执行策略

![](img/32848607951ae0b4bd73791653928743_52.png)

![](img/32848607951ae0b4bd73791653928743_54.png)

寻找对标账号不仅是为了关注，更是为了学习和快速反应。以下是关键的执行策略：

![](img/32848607951ae0b4bd73791653928743_56.png)

![](img/32848607951ae0b4bd73791653928743_58.png)

*   **聚焦图文达人**：主要寻找以发布**图文笔记**为主的达人。对于视频博主，如果其爆款频率不高，可以只参考其单个视频内容，而不必关注整个账号。
    ![](img/32848607951ae0b4bd73791653928743_44.png)

![](img/32848607951ae0b4bd73791653928743_60.png)

![](img/32848607951ae0b4bd73791653928743_62.png)

*   **培养爆款预感**：通过大量观察，您会逐渐培养出对内容能否成为爆款的判断力。例如，一条上午发布就已获得50个赞的笔记，有很大可能在全天获得上百个赞。

![](img/32848607951ae0b4bd73791653928743_64.png)

![](img/32848607951ae0b4bd73791653928743_66.png)

*   **快速执行“搬运”**：当您判断一条笔记有爆款趋势但尚未完全爆发时，应果断行动。将其内容（图文）**抄录**过来，并转化为视频形式，在您自己的账号上发布，以抓住流量窗口。
    ![](img/32848607951ae0b4bd73791653928743_72.png)

![](img/32848607951ae0b4bd73791653928743_68.png)

![](img/32848607951ae0b4bd73791653928743_70.png)

**核心公式可以概括为：**
**寻找优质图文对标 -> 养号强化垂直标签 -> 判断爆款趋势 -> 快速转化发布**
