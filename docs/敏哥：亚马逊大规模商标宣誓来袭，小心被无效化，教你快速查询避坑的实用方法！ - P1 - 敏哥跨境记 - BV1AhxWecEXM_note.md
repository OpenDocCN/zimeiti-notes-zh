# 亚马逊商标运营：第1章：商标宣誓查询与风控指南

在本节课中，我们将学习亚马逊商标运营中一个至关重要的环节——商标宣誓。我们将了解其重要性、关键时间点，并掌握如何快速查询自己商标的宣誓截止日期，以规避商标被无效化的风险。

## 商标宣誓的重要性

上一节我们提到了商标运营的整体性，本节中我们来看看一个常被忽略的细节：商标宣誓。

商标注册完成并非一劳永逸。根据规定，商标注册后大约每5年需要进行一次宣誓。如果未能按时完成宣誓，商标将被宣告无效。这可能导致你失去商标权，甚至可能被他人抢注，反过来投诉你侵权。

因此，核实并记录名下所有商标的宣誓截止时间至关重要。建议将信息复制到备忘录中，并设置好提醒。一旦进入可宣誓的时间窗口，应立即安排处理。

## 如何查询宣誓截止时间

了解了宣誓的重要性后，接下来我们学习具体的查询方法。以下是查询商标宣誓截止日期的详细步骤。

1.  访问美国专利商标局（USPTO）官网。
2.  找到 **“Trademark Search System”** 入口并点击进入。
3.  在搜索框中输入你的商标名称或注册号进行查询。
4.  在查询结果中，找到并点击目标商标的详细信息记录。
5.  在商标详情页面中，定位并点击 **“Maintenance”** 按钮。
6.  点击后，页面将显示三个关键的时间点。

## 理解三个关键时间点

完成查询后，页面会显示三个日期，理解它们的含义是正确规划宣誓操作的核心。

*   **第一个时间**：这是可以开始提交宣誓文件的最早日期。公式表示为：**可提交起始日**。
*   **第二个时间**：这是免费提交宣誓的截止日期。在此日期前完成，无需缴纳额外官费。公式表示为：**免费截止日**。
*   **第三个时间**：这是付费提交宣誓的最终截止日期。在此日期前仍可提交，但需支付额外罚款。超过此日期，商标将直接失效。公式表示为：**最终截止日**。

这三个时间点是你商标维护的关键节点。**最佳操作是在第一个时间和第二个时间之间完成宣誓**，这样可以完全避免任何潜在的费用和风险。
