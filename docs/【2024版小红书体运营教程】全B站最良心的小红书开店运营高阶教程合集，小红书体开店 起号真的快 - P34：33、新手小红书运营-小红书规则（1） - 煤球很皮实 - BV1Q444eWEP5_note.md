## 详解推荐规则

推荐规则是指小红书系统如何判定账号，并决定为其分配多少流量和曝光。系统会录入你的账号信息、分析发布的笔记内容，从而为你的账号打上标签（例如：美妆达人、母婴博主、数码爱好者）。基于这个标签，系统会将你的内容推荐给可能感兴趣的用户。

**核心公式**：`账号信息 + 笔记内容 => 系统标签 => 推荐流量`

![](img/cd2ba792d4272e097b81f541c732bb25_19.png)

你要想做好数据，就必须理解并利用好这套推荐机制中的权重分配。推荐规则主要包含以下三个子项：

### 1. 用户行为规范

用户行为规范明确了在账号日常运营中，哪些行为是禁止的，哪些是允许的。以下是必须避免的**敏感行为**（用 `禁止` 表示）：

*   `禁止` 在**任何内容**中包含营销词汇，例如：
    *   **微信号**
    *   **二维码**
    *   **“代购”字样**
    *   **第三方平台水印**
    *   **宣传“某某店铺”**

![](img/cd2ba792d4272e097b81f541c732bb25_21.png)

**含义解析**：这意味着你不能直接在笔记或商品描述中，引导用户去你的其他店铺、微信或QQ群购买产品。例如，你不能在笔记里写“想买同款请加VX：123”。

![](img/cd2ba792d4272e097b81f541c732bb25_23.png)

![](img/cd2ba792d4272e097b81f541c732bb25_25.png)

**应对技巧**：虽然直接宣传被禁止，但可以通过一些间接方式引导。例如，你可以用主账号在评论区以“官方”身份置顶一条评论，并@你的小号（该小号关联了商品），从而实现流量的内部流转。这需要一定的技巧和账号矩阵配合。

![](img/cd2ba792d4272e097b81f541c732bb25_27.png)

以下是**允许的行为**（用 `允许` 表示）：

*   `允许` 在个人资料中留下**邮箱**。
*   `允许` 提及**微博**账号。
*   `允许` 使用**谐音、话术**间接引导（例如，将“微信”写作“V星”）。
    *   **注意**：即使在私聊中引导，每天也最好不超过6次，频繁操作仍有风险。

![](img/cd2ba792d4272e097b81f541c732bb25_29.png)

### 2. 内容与商品规范

小红书对于商品和内容推广相对宽容，但仍有严格的红线。以下是**绝对不能涉及的内容**：

*   `禁止` 提供未经认证的**医疗建议**（例如，“这个病去XX医院看最好”）。
*   `禁止` 提供**投资建议**（例如，“这只股票必涨”、“那个楼盘买了肯定赚”）。
    *   **解释**：这类内容涉及法律风险和诈骗，平台监管极其严格。
*   `禁止` 在同一台电脑上**同时登录超过3个账号**。
    *   **解释**：平台会检测设备信息。建议使用“雷电模拟器”等工具进行IP和数据隔离来管理多账号，且同时登录数不要超过3个，否则容易被判定为营销号群控。

![](img/cd2ba792d4272e097b81f541c732bb25_31.png)

### 3. 图文/视频笔记发布规范

在发布笔记时，需要遵守以下规范：

*   `禁止` **诱导用户互动**，例如：“关注抽奖”、“点赞领取资料”。
    *   **解释**：平台明令禁止这种直接的利诱式互动。但可以通过更软性的方式，如在店铺活动中设置标题，间接吸引用户，不过其搜索权重较低。
*   `必须` 保证内容的**高原创度**。
    *   **核心逻辑**：原创度越高，前期获得的基础流量扶持就越大。
    *   **反面案例**：直接搬运其他平台的图片和文案，账号权重会很低，甚至作废。
    *   **建议**：对于新人，即使原创度只有50%，也比完全搬运要好。小红书对新人的友好度依然较高。
