# 小红书运营指南：P4：小红书的检测规则与运营注意事项 🔍

在本节课中，我们将详细拆解小红书的平台检测规则。了解这些规则是避免账号违规、确保内容安全发布的基础。我们将从平台检测的多个维度入手，并介绍实用的工具来辅助你的运营工作。

## 平台检测的核心维度

小红书的检测系统主要针对以下几个关键部分进行审查，以确保平台内容的合规性。

以下是平台重点检测的四个方向：

1.  **个人资料、笔记内容、私信及评论**：这些位置不得包含微信号、微信二维码、代购信息、水印，以及推广淘宝店铺等营销词汇。
2.  **手机设备**：平台会检测登录设备。建议同一台设备最多只登录三个账号，超出可能增加风险。
3.  **笔记内容**：内容中不得出现诱导用户的行为，例如“关注领奖”、“点赞抽奖”等。这类行为是平台明令禁止的。
4.  **内容原创度**：小红书对原创内容的保护程度很高。因此，切勿将同一篇文章在多个账号上发布。坚持一篇文章只用一个账号发布的原则。

## 敏感词检测与规避方法

除了上述维度，平台另一项重要的检测规则是针对**敏感词**的过滤。涉及手机卡、金融、电子烟、代购、医疗用品等领域的词汇需要格外谨慎。

那么，在撰写笔记时，如果不慎包含了未知的敏感词该怎么办？这时就需要借助外部工具进行检测。

以下是两个实用的敏感词检测网站，可以帮助你提前规避风险：

*   **网站一**：这是一个便捷的在线检测工具。你只需将写好的文案粘贴到输入框中，点击“立即检测”，系统便会高亮标记出被识别为小红书敏感词的词汇。例如，输入包含“代购”的文案，该词会被直接标出。
