## 素材寻找与文案提取
上一节我们介绍了二创的基本思路，本节中我们来看看具体的操作步骤。首先，我们需要找到合适的原始视频素材并提取其文案。

![](img/81e224926e03a7fb37ea8a12e50400c9_13.png)

1.  **寻找素材**：以书籍《走进大学城》为例，在抖音搜索栏选择“商品”并点击“销量”，可以找到相关的带货视频。例如，视频“不读书的人生”就是一个可用的素材。
2.  **分析视频**：观察选中的视频，例如一个时长1分58秒、讲述大学意义的视频。注意视频前后部分的声音可能不同，这为我们分段处理提供了条件。
3.  **下载与去水印**：复制视频链接，使用去水印工具下载无水印版本。

## 视频剪辑与文案处理
获取原始视频后，下一步是在剪辑软件中进行初步处理，并提取出我们需要的文案。

1.  **导入与分割**：将去水印后的视频导入剪映。在声音发生变化的位置（例如53秒处）将视频分割，保留我们需要的片段（通常是前半部分的文案），然后导出这个片段。
2.  **提取文案**：使用“配音神器”等工具，通过“视频提取文案”功能，将上一步导出的视频片段中的文字内容提取出来。

## 素材替换与画面重组
这是二创的核心环节，通过替换原始视频中的部分画面，形成新的视觉内容。

以下是替换画面的具体操作步骤：
*   **定位替换点**：在剪映中，将需要替换的原始视频片段（例如一段3秒的“毕业典礼”画面）删除。
*   **寻找新素材**：根据原片段内容（如“毕业典礼”），在抖音搜索相关视频，下载并去水印。
*   **插入新素材**：将找到的新视频素材导入剪映，放置在刚才删除片段的位置，并裁剪至与删除片段相同的时长。
*   **添加转场与静音**：在片段衔接处添加“叠化”等转场效果。务必关闭新增片段的原声，将音量调整为0。
*   **重复操作**：用同样的方法，替换视频中其他可以更换的画面，例如“打高尔夫”、“宿舍环境”等片段。目标是替换1到2个核心画面素材。

## 音频替换与最终合成
画面重组完成后，我们需要用新的配音替换原声，并完成视频的最后调整。

1.  **生成新配音**：在“配音神器”中，使用提取出的文案，选择一个合适的声音生成新的配音音频，并导出。
2.  **替换音频**：回到剪映，关闭原始视频的所有原声。导入新生成的配音音频，将其与视频画面对齐。
3.  **音画对齐**：检查音频与视频的长度是否匹配。如果音频长，可以适当加快视频速度或添加一些画面；如果音频短，则可以减慢视频速度或裁剪多余画面。使用`常规变速`功能进行调整。
4.  **处理原声片段**：如果视频后半部分需要使用原始视频的人声（例如人物讲述），可以保留该段音频，但可以替换对应的画面，并尽量缩短人物正面特写的时长，以避免版权或肖像权风险。
5.  **添加背景音乐与字幕**：为视频添加一段合适的背景音乐。然后，使用剪映的`识别字幕`功能，为新的配音生成字幕。通过调整字幕的样式和位置，覆盖掉原始视频可能残留的字幕。
6.  **检查与导出**：最后，确保视频中没有明显的其他平台水印（可通过放大画面遮盖或去水印处理）。导出视频时，选择分辨率为`1080P`，帧率为`60帧`，码率选择推荐值即可。

## 总结
本节课我们一起学习了书单视频二创的完整流程。关键步骤可以总结为：**寻找素材 -> 提取文案 -> 替换画面 -> 生成新配音 -> 音画对齐 -> 添加背景音乐与字幕 -> 导出**。初学者在前期操作可能会比较慢，但通过多次练习，熟能生巧，制作效率会大幅提升。请务必动手实操，从处理几个视频开始，逐步掌握所有技巧。
