# 拼多多运营教程：P1：快速拉升店铺DSR评分与流量

在本节课中，我们将学习一个核心的运营技巧：当你的拼多多店铺流量出现下滑时，如何通过快速提升店铺DSR评分来有效拉升流量。这个方法操作简单，适合新手，通常能在四五天内看到效果。

## 核心问题：DSR评分影响流量

上一节我们介绍了流量下滑的普遍现象，本节中我们来看看其核心原因之一。影响店铺流量的一个核心因素是**店铺的DSR评分**。一旦你发现DSR评分降低，就必须想办法提升它。

## 操作步骤：通过营销活动提升评分

以下是利用拼多多后台营销工具“助力夺宝”来提升DSR评分的具体步骤。这个方法有效且可操作。

1.  **进入营销活动页面**
    首先，打开拼多多商家后台，找到“店铺营销”下的“营销玩法”。点击“助力夺宝”旁的“设置”按钮，然后选择“创建活动”。

2.  **填写活动基础信息**
    在创建页面，按以下要求填写：
    *   **活动名称**：可随意填写，用户不可见。
    *   **活动时间**：最少设置3天，以满足发货时效要求。
    *   **活动商品**：添加需要提升评分的商品。

3.  **定位低评分商品**
    如何找到店铺里评分低的商品呢？方法很简单。
    回到后台，进入“数据中心”下的“服务数据”。点击“评价数据”并向下滑动，即可查看店铺产品近30天的数据。找到评分**低于4.5分**的商品链接。

4.  **添加商品并执行活动**
    回到“助力夺宝”活动创建页面，将上一步找到的低评分商品添加进去。然后，你需要准备几个“小号”（备用账号），根据需要的单量进行“拉新助力”操作。

## 效果预估与辅助工具

执行上述操作后，大约5天左右，你的店铺评分就能够有效拉升。

如果你不知道具体需要做多少单，可以使用以下**公式**进行估算，或借助我们提供的计算表格：
**目标提升单量 ≈ (目标DSR分数 - 当前DSR分数) * 近期有效评价总数**
