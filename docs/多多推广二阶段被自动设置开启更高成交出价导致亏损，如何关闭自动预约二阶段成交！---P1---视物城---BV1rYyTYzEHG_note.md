# 拼多多推广教程：第1章：如何关闭自动预约二阶段成交出价 🔧

在本节课中，我们将要学习一个关键的拼多多推广设置：如何关闭系统自动设置的“预约二阶段成交出价”。这个功能可能导致您的推广计划在进入第二阶段时，成交出价被自动大幅提高，从而造成不必要的广告花费和亏损。我们将通过清晰的步骤，指导您如何找到并修改这一设置。

## 问题概述

近期，许多商家发现其拼多多推广计划存在一个问题：第一阶段推广结束后，系统会自动开启“预约二阶段”功能，并设置一个远高于第一阶段出价的成交出价。例如，您可能设置了一阶段成交出价为4元，但系统在二阶段自动将其调整为9元或更高。此过程无需人工操作，且关闭入口隐蔽，即使删除旧计划并新建，该选项仍会被自动勾选。

## 操作步骤详解

以下是关闭自动预约二阶段成交出价的具体操作流程。请跟随步骤逐一进行设置。

### 第一步：检查并修改现有计划

首先，我们需要对正在推广中的现有计划进行检查和修改。

1.  进入拼多多商家后台的“推广计划”页面。
2.  找到您需要检查的计划，点击“成交出价”下方的“编辑”按钮。
3.  在编辑页面中，您会看到“预约二阶段成交出价”选项及其金额（例如，显示为10.14元）。这意味着如果计划进入二阶段，将自动使用此出价。
4.  将该出价修改为您预期的合理金额（例如，修改为5元），然后点击“确定”保存。

### 第二步：创建新计划时的关键设置

上一节我们介绍了如何修改现有计划，本节中我们来看看在创建全新推广计划时，如何避免再次被自动设置高额出价。

1.  点击“新建推广计划”，进入计划创建页面。
2.  在设置“成交出价”时，您会发现系统已自动勾选“预约进入二阶段”并预设了一个出价。
3.  无论您如何切换或尝试，这个预约选项都可能默认存在。此时，需要手动进行干预。
4.  请找到并点击“预约二阶段成交出价”的编辑区域。
5.  将其中的金额修改为您的目标出价（例如，5元）。
6.  **至关重要的一步**：在最终创建计划前，请务必先关闭页面下方的“全局优选起量”等自动优化选项。
7.  完成所有设置后，再点击“确定建立”并开启推广。

### 第三步：计划创建后的立即检查

计划创建成功后，并非万事大吉。请立即进行以下检查，以防万一。

1.  进入刚刚创建好的推广计划详情页。
2.  再次点击“成交出价”进行查看。
3.  如果发现“预约二阶段成交出价”仍不是您设置的金额，请**快速将其修改正确**。
4.  请注意：一旦计划进入二阶段并被设置了高额出价，如果在12小时内未作修改，该出价可能会被系统“锁死”，届时将无法下调，导致持续亏损。

## 核心要点与总结

本节课中我们一起学习了如何管理拼多多推广中的“预约二阶段成交出价”功能。我们来总结一下几个核心要点：

*   **核心问题**：系统自动开启二阶段预约，并**默认为一个较高的成交出价**，公式可表示为：`二阶段实际出价 = 系统预设值（远高于一阶段出价）`。
*   **解决方案**：无论是修改旧计划还是创建新计划，都必须**手动编辑“预约二阶段成交出价”** 选项，将其设置为可控的数值。
*   **关键动作**：在创建任何新计划时，记住一个固定操作顺序：`修改预约出价 -> 关闭自动优化选项 -> 确认创建`。
*   **风险提示**：务必在计划开启后立即复查设置。警惕出价被“锁死”的风险，其代码逻辑可理解为：`if (进入二阶段 && 出价未在12小时内手动修改) { 出价状态 = 锁定; }`。

请务必告知您的运营人员，在建立所有推广计划时，都要主动检查并修改预约二阶段出价，避免因平台自动调整而在不知不觉中承受亏损。通过精细化的手动设置，才能牢牢掌控推广成本与效果。
