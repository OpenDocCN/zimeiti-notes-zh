# 抖音运营与剪辑：P34：视频剪辑实操（下） 🎬

在本节课中，我们将继续学习视频剪辑的核心操作，重点掌握如何为多片段视频添加转场、统一画面比例以及添加字幕。这些步骤对于提升视频的流畅度、观感和平台推荐权重至关重要。

上一节我们介绍了视频素材的导入与基础裁剪，本节中我们来看看如何将这些片段流畅地组合成一个完整的作品。

## 添加视频转场

![](img/1f63d8c7dde544646b2e0d230491a219_3.png)

每个视频通常由多个短片段组成，片段之间需要有自然的衔接。添加转场就是为了明确告知用户，当前内容已切换至另一个不同的片段。

![](img/1f63d8c7dde544646b2e0d230491a219_5.png)

![](img/1f63d8c7dde544646b2e0d230491a219_7.png)

如果不添加转场，当观众看到画面突然变化时，可能会产生困惑，甚至退出视频确认自己观看的内容。这个退出动作会直接导致视频的**完播率**下降。

![](img/1f63d8c7dde544646b2e0d230491a219_9.png)

**核心公式**：
`视频收益 ∝ 完播率`

![](img/1f63d8c7dde544646b2e0d230491a219_11.png)

![](img/1f63d8c7dde544646b2e0d230491a219_13.png)

因此，添加转场是必要的。以下是添加转场的具体步骤：

1.  在视频轨道上，找到并点击右侧的“加号”按钮。
2.  在弹出的菜单中，选择“素材库”。
3.  在素材库的“热门”分类中，可以找到大量转场素材，例如搞笑的片段或动态效果。
4.  选择转场时，注意时长。过长的转场（如16秒）会让观众失去耐心。应尽量选择**3~5秒**的转场素材，3秒最为合适。
5.  选中合适的转场素材，点击添加即可。

![](img/1f63d8c7dde544646b2e0d230491a219_15.png)

![](img/1f63d8c7dde544646b2e0d230491a219_17.png)

## 统一转场画面比例

添加转场后，必须确保其画面比例与主视频保持一致，即**16:9**的横屏比例。

![](img/1f63d8c7dde544646b2e0d230491a219_19.png)

操作步骤如下：
1.  选中已添加的转场片段。
2.  在编辑菜单中找到“画布模糊”或“背景”选项。
3.  选择第一个清晰的画面背景模式，系统会自动将转场素材适配为16:9的横屏。
4.  最后，点击右下角的“对勾”确认设置。

至此，我们完成了视频比例的统一和转场的添加。记住，寻找素材时也应严格筛选，避免使用过长的视频，以免在平台审核时遇到问题。

![](img/1f63d8c7dde544646b2e0d230491a219_21.png)

## 为视频添加字幕

接下来，我们需要为视频添加字幕（文案），以提升视频的可看性和信息传达效率。

![](img/1f63d8c7dde544646b2e0d230491a219_23.png)

操作步骤如下：
1.  点击手机剪辑界面左下角的“文字”选项。
2.  选择“新建文本”，然后点击右侧的“识别字幕”功能。
3.  点击“开始匹配”，系统会自动识别视频或音频中的人物对话，并生成对应的字幕文本。
4.  字幕生成后，可以进行微调。拖动字幕文本框右下角，可以放大字幕。
5.  将放大后的字幕文本框，拖动到视频画面最下方的位置，这样观看最舒适。
6.  调整好第一句字幕后，后续字幕通常会同步应用此样式，非常方便。

![](img/1f63d8c7dde544646b2e0d230491a219_25.png)
