# 微信视频号运营教程：P9：7.账号定位

## 概述
在本节课中，我们将学习微信视频号账号定位的核心环节，特别是如何解决初期缺乏原创视频素材的问题。我们将介绍获取高质量剪辑素材的方法，并明确账号启动前的必要准备步骤。

## 账号启动前的必要操作
上一节我们讨论了账号的基础设置，本节我们来看看启动前的一个关键操作。

有同学询问是否需要隐藏账号之前的作品。这个操作是必要的。

**操作公式：** `新账号启动 = 隐藏历史作品 + 统一新内容方向`

![](img/4a067c8b16db2d92590f98b8a7b994ab_1.png)

## 初期执行策略
明确了基本操作后，我们需要一个清晰的执行计划。以下是初期的核心策略，请按顺序执行。

1.  **首要目标：获取100个粉丝。** 这是开通视频号基础功能的关键门槛。
2.  **发布频率：保持每日更新一条视频。** 稳定的更新是吸引和留住粉丝的基础。
3.  **执行顺序：先集中精力达成100粉丝的目标，再专注于解决素材问题。** 应逐个解决问题，避免同时处理多个难题导致精力分散。

![](img/4a067c8b16db2d92590f98b8a7b994ab_3.png)

许多同学面临不会拍摄、不会剪辑、没有专业素材的困境。
