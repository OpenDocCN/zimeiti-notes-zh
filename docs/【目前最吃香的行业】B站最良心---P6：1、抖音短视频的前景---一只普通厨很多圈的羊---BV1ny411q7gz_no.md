## 流量扶持与用户分类

![](img/4bfa543012e77f1d0294e5dfb10d9a64_3.png)

这体现了平台巨大的流量扶持。同时，需要明确抖音上存在两类人。以下是这两类人的定义：

*   **玩家**：这类人单纯地玩抖音，没想过从抖音上赚到钱。
*   **创作者**：这类人希望自己制作抖音内容来获取收益。你们都可以称作创作者。

## 为何选择抖音平台

为什么在众多短视频平台中，我们选择抖音短视频呢？就像开实体店铺需要选择人流量大的地段一样，做短视频也必须选择一个流量大的平台。抖音的流量数据如下：

*   **日活跃用户**：`8亿` 以上。
*   **月活跃用户**：`40亿` 人次以上。

这是一个巨大的流量池。流量大，竞争压力是否也很大？答案是不会，因为创作者只有约5000万。这里的“创作者”定义广泛，并非特指网红达人。例如，邻居家的老太太用手机拍下自己做的饭传到抖音上，她也算创作者。因此，你90%的竞争压力，都来自于身边的普通人。

## 入门门槛与创作风口

上一节我们分析了平台的流量优势，本节中我们来看看入门的实际门槛和当前机遇。

做抖音短视频不需要购买电脑或摄像机。只需要一部智能手机和一个手机号，就可以注册抖音账号。抖音倡导全民能够通过短视频的形式分享和记录自己的生活。

那么，当前有什么样的风口值得我们去做呢？首先，抖音计划“抖音将帮助1000名优质的艺术创作者涨粉超过100万”。有的同学可能会说不会创作。这不用担心，正因为不会，才需要学习。市场庞大，竞争相对充足，行业还很容易产出内容。

## 内容创作与涨粉逻辑

现在是一个大数据时代。平台算法会根据你的喜好推荐内容，例如：
```python
# 简化的推荐逻辑示意
if 用户喜欢看帅哥:
    推荐内容 = 有腹肌的帅哥
elif 用户喜欢看美女:
    推荐内容 = 长腿美女
```
这导致我们每个人看到的内容可能比较片面。但对于创作者而言，这恰恰是机会。

无论你是宝妈、实体店主还是普通上班族，都可以创作内容：
*   **宝妈**：可以拍宝宝或自己做的饭。
*   **实体店主**：可以拍产品进行引流。
*   **普通上班族**：可以分享日常生活。

在月活跃40亿用户的群体中，只要你创作的内容不触犯法律和道德红线，不是低俗内容，总有一部分人会喜欢上你。如果你目前只有几十个粉丝，很可能是因为喜欢你的人还没有刷到过你的作品。

因此，对于新手小白来说，心情迫切可以理解，但不要把涨粉作为第一个目标。我们的第一个目标应该是：**先把流量涨起来，把播放量涨起来**。粉丝自然会随之而来。
