## 一坑：坑产排名 📈

![](img/eb0f4d441f4e792468e8bc6cac838388_8.png)

![](img/eb0f4d441f4e792468e8bc6cac838388_9.png)

“一坑”指的是坑产排名。产品上架后，不要直接去修改成百万销量或做大量评价。这种做法容易被平台稽查，导致链接失效。正确的做法是先通过操作坑产来获取排名权重，之后再补充基础销量和评价。

上一节我们介绍了“一坑”的概念，本节中我们来看看如何具体操作坑产排名。

### 如何计算并操作坑产

![](img/eb0f4d441f4e792468e8bc6cac838388_15.png)

![](img/eb0f4d441f4e792468e8bc6cac838388_16.png)

![](img/eb0f4d441f4e792468e8bc6cac838388_17.png)

首先，我们需要计算出同行优秀商家的坑产数据，并以此为目标制定自己的操作计划。

![](img/eb0f4d441f4e792468e8bc6cac838388_19.png)

![](img/eb0f4d441f4e792468e8bc6cac838388_20.png)

以下是具体操作步骤：

1.  **进入店铺后台**：打开拼多多商家后台，找到“数据中心”下的“流量数据”。
2.  **记录关键数据**：在流量数据页面，我们需要记录两个关键数据：**访客数**和**成交UV价值**。注意，不是记录自己店铺的数据，而是记录“同行同层级优秀值”的数据。
    *   在“访客数”和“成交UV价值”图表中，通常有三条线，分别代表：**蓝色（自己店铺）**、**绿色（同行同层平均值）**、**黄色（同行同层优秀值）**。
    *   将鼠标悬停在黄色曲线上，即可看到具体的优秀值数据。
3.  **计算同行优秀GMV**：GMV（商品交易总额）即坑产。计算公式为：
    `同行优秀GMV = 同行优秀访客数 × 同行优秀UV价值`
4.  **制定7天操作单量**：根据计算出的同行优秀GMV，结合自己产品的客单价，计算出7天内需要操作的总单量。计算公式为：
    `7天总操作单量 = 同行优秀GMV ÷ 产品客单价`
5.  **执行螺旋增长计划**：得到总单量后，不能平均分配或随意操作。需要制定一个**螺旋增长计划**，即单量逐日递增。
    *   将总单量分配到7天中。
    *   确保每天的订单量都比前一天有所增长（例如：第1天10单，第2天15单，第3天20单……）。
    *   按照这个增长计划去执行，可以有效提升坑产排名。

![](img/eb0f4d441f4e792468e8bc6cac838388_22.png)

完成坑产操作并获取一定权重后，就可以安全地为链接补充基础销量和评价了。

## 二率：点击率与转化率 🎯

### 如何通过标准推广进行测图测款

![](img/eb0f4d441f4e792468e8bc6cac838388_31.png)

![](img/eb0f4d441f4e792468e8bc6cac838388_32.png)

以下是使用标准推广计划测试主图点击率和产品转化率的具体步骤：

![](img/eb0f4d441f4e792468e8bc6cac838388_34.png)

![](img/eb0f4d441f4e792468e8bc6cac838388_35.png)

1.  **创建标准推广计划**：在商家后台的“推广中心”，选择“标准推广”，新建一个计划。
2.  **添加推广商品**：选择你需要测试的新品添加到计划中。
3.  **设置计划参数**：
    *   **日限额**：建议先设置为100元。
    *   **成交出价**：
        *   如果产品客单价在50元以内，按照热销SKU的价格出价（例如，热销款卖13元，出价就设13元）。
        *   如果产品客单价在50元以上，按照单笔利润出价（例如，一单利润30元，出价就设30元）。
    *   **其他设置**：“询单出价”、“收藏出价”、“关注出价”和“急速起量”功能都选择关闭。
4.  **设置测试创意**：
    *   在创建好的计划中，点击“更多” -> “创意” -> “编辑创意”。
    *   从图库中选择3-4张需要测试的主图，添加到创意中。
5.  **分析测试数据**：让计划跑3-7天，然后观察数据。重点关注：
    *   **点击率**：主图点击率最好能达到10%以上。
    *   **转化率**：产品转化率最好能达到5%以上。
    *   **注意**：不同类目的合格标准不同，但最低不能低于行业平均值。选择数据最好的图片作为主图，并优化产品详情以提升转化。

![](img/eb0f4d441f4e792468e8bc6cac838388_36.png)

![](img/eb0f4d441f4e792468e8bc6cac838388_37.png)

![](img/eb0f4d441f4e792468e8bc6cac838388_38.png)

![](img/eb0f4d441f4e792468e8bc6cac838388_39.png)

## 三放大：流量与单量放大 📢

![](img/eb0f4d441f4e792468e8bc6cac838388_46.png)

当产品通过了“二率”测试，具备了基本的市场竞争力后，就可以进行“三放大”，即**流量放大**和**单量放大**。主要有两种方式：报名平台活动和开启全站推广。

![](img/eb0f4d441f4e792468e8bc6cac838388_48.png)

![](img/eb0f4d441f4e792468e8bc6cac838388_49.png)

### 放大方式一：报名活动

![](img/eb0f4d441f4e792468e8bc6cac838388_50.png)

对于有实力的商家，报名大型平台活动是快速放大流量和销量的有效途径。

以下是主要的活动类型：

![](img/eb0f4d441f4e792468e8bc6cac838388_56.png)

![](img/eb0f4d441f4e792468e8bc6cac838388_57.png)

*   **584活动**：即“5.8折大促”，是拼多多的大型促销活动，流量巨大。
*   **百万爆款/热销活动**：针对有爆款潜质的商品，能获得大量资源扶持。
*   **首页资源位活动**：在坑产和销售指标表现优异后，有机会被邀请参加，曝光量极高。
你可以查看后台营销活动中心，根据自己的产品条件报名相应的活动。

### 放大方式二：开启全站推广

![](img/eb0f4d441f4e792468e8bc6cac838388_63.png)

除了活动，付费推广是更可控的放大手段。在放大阶段，我们使用**全站推广**，因为它覆盖的流量渠道更广。

全站推广分为两个阶段，操作方式不同：

![](img/eb0f4d441f4e792468e8bc6cac838388_65.png)

![](img/eb0f4d441f4e792468e8bc6cac838388_66.png)

![](img/eb0f4d441f4e792468e8bc6cac838388_67.png)

![](img/eb0f4d441f4e792468e8bc6cac838388_68.png)

**第一阶段：快速通过期**

1.  **新建全站计划**：在“推广中心”选择“全站推广”，新建计划并添加商品。
2.  **设置计划参数**：
    *   **日限额**：建议初期不设置上限，或设置一个较高的金额。
    *   **目标投产比（ROI）**：按照**保本ROI**来设置。计算公式为：
        `保本ROI = 产品客单价 ÷ 单笔利润`
        *   例如：客单价100元，利润40元，保本ROI = 100 / 40 = 2.5。此时出价设为2.5即可。
    *   **其他设置**：同样关闭“询单/收藏/关注出价”和“急速起量”。
3.  **跑数据**：以此设置跑3天，目的是快速积累成交数据，顺利度过第一阶段。

![](img/eb0f4d441f4e792468e8bc6cac838388_70.png)

![](img/eb0f4d441f4e792468e8bc6cac838388_71.png)

![](img/eb0f4d441f4e792468e8bc6cac838388_72.png)

**第二阶段：托价放量期**

![](img/eb0f4d441f4e792468e8bc6cac838388_73.png)

![](img/eb0f4d441f4e792468e8bc6cac838388_74.png)

![](img/eb0f4d441f4e792468e8bc6cac838388_75.png)

![](img/eb0f4d441f4e792468e8bc6cac838388_76.png)

进入第二阶段后，开始进行精细化的托价和放量操作。

![](img/eb0f4d441f4e792468e8bc6cac838388_77.png)

以下是操作流程：

![](img/eb0f4d441f4e792468e8bc6cac838388_79.png)

1.  **设置日限额**：根据第一阶段的花费情况，设置一个初始日限额（如100-300元）。
2.  **观察消耗速度**：如果日限额能很快消耗完，且转化数据稳定，则第二天可以增加日限额（如从100元加到200元）。
3.  **逐步托价（提升ROI）**：在增加日限额的同时，可以小幅提升目标投产比（ROI），即“托价”。
    *   例如：第一天ROI设为2.0，日限额100元很快花完；第二天日限额加到200元，ROI可以提升到2.2；第三天日限额加到300元，ROI提升到2.4。
    *   每次提价幅度要小（0.1-0.3），不可操之过急。
4.  **监控与调整**：在托价过程中，密切观察计划的曝光量、点击率和转化率。
    *   如果数据稳定甚至增长，可以继续托价和放量。
    *   如果数据出现明显下滑，则停止托价，维持当前ROI稳定跑一段时间。
    *   托价有峰值，当无法继续提升时，保持稳定投放即可。

**组合策略**：在实际操作中，可以将全站推广与平台活动结合，实现流量和销量的最大化。

本节课中我们一起学习了拼多多运营的核心口诀“一坑、二率、三放大”的完整操作流程。

![](img/eb0f4d441f4e792468e8bc6cac838388_88.png)

*   **“一坑”** 是基础，通过计算同行优秀坑产并执行螺旋增长计划，为产品获取初始排名权重。
*   **“二率”** 是关键，通过标准推广测图测款，优化主图点击率和产品转化率，确保产品的市场竞争力。
*   **“三放大”** 是目标，通过报名活动或开启全站推广并进行托价放量操作，最终实现流量和单量的规模化增长。
