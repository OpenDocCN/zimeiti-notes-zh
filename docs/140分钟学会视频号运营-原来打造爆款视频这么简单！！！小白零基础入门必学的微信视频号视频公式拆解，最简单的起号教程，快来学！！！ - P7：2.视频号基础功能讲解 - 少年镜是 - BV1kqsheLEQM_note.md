# 视频号运营：P7：视频号基础功能讲解 🎬

在本节课中，我们将系统学习微信视频号的基础功能设置与界面操作。这是开启视频号带货之旅的第一步，掌握这些基础操作能帮助你更高效地管理账号和内容。

## 视频号入口与开通

![](img/49aa2cac4d7ff288912311348ea70aee_5.png)

![](img/49aa2cac4d7ff288912311348ea70aee_7.png)

视频号主要有两个进入端口。第一个端口位于微信“发现”页的“朋友圈”下方。第二个端口是直接点击微信底部菜单栏的“我”，然后选择“视频号”。

![](img/49aa2cac4d7ff288912311348ea70aee_9.png)

![](img/49aa2cac4d7ff288912311348ea70aee_11.png)

如果你的“朋友圈”下方没有视频号入口，可以按以下步骤开启：
1.  进入微信“我” -> “设置”。
2.  选择“通用”。
3.  点击“发现页管理”。
4.  找到“视频号”选项并开启其开关。

![](img/49aa2cac4d7ff288912311348ea70aee_13.png)

开通视频号非常简单。如果你还没有视频号，点击“发表视频”按钮，按照提示输入账号名称等信息即可完成开通。

## 视频号主界面功能

进入视频号后，你会看到几个主要标签页，它们构成了内容分发的核心路径。

以下是各个标签页的具体功能：
*   **推荐**：系统根据你的观看喜好，智能推送你可能感兴趣的视频。
*   **朋友点赞**：展示你的微信好友点赞过的视频。这是视频号社交推荐的重要体现。
*   **关注**：集中展示你已关注的视频号创作者发布的最新内容。
*   **搜索**（放大镜图标）：你可以通过这里搜索具体的账号名称或商品关键词（例如“保鲜膜”），来寻找对标账号或相关视频内容。

## 个人主页与账号设置

点击底部导航栏的“我”，即可进入你的视频号个人主页。点击右上角的三个点（...），进入“资料设置”，可以对账号信息进行详细配置。

![](img/49aa2cac4d7ff288912311348ea70aee_15.png)

![](img/49aa2cac4d7ff288912311348ea70aee_16.png)

以下是账号设置的核心选项：
*   **头像、名字、简介**：可以随时修改头像和简介。名字一年内可修改**5**次。简介可以参考话术：“视频同款购买方式：点下面的商品橱窗，找到你想要的商品下单即可。”
*   **视频号ID**：在资料页长按可复制。在与商家设置**定向佣金**时，需要提供此ID。
*   **隐私与权限**：
    *   **在个人名片展示视频号**：控制是否在微信个人资料页显示你的视频号。如果不想让熟人看到，可以关闭。
    *   **作品权限**：可以设置作品“公开，但不推荐给朋友”，或指定“不让谁看”某些联系人。
    *   **点赞隐私**：在“隐私”设置中，可以管理“不让他（她）看我点赞过的视频”，保护自己的互动痕迹。

## 作品管理与互动功能

在个人主页点击任意一个自己发布的视频，进入详情页后，点击右上角的三个点（...），可以对单条作品进行管理。

以下是作品管理的常用操作：
*   **置顶**：将重要作品固定在个人主页顶部。
*   **上热门（加热）**：付费推广作品，让内容被更多人看到。
*   **权限管理**：可以“关闭评论”或设置为“仅自己可见”。
*   **删除**：移除不需要的作品。
*   **评论管理**：对于不友善的评论，可以**长按**后选择“删除”。视频号评论目前没有官方置顶功能，但可以通过**点赞**将重要的引导性评论（如购买指引）顶到前排。

## 商品橱窗与数据查看

![](img/49aa2cac4d7ff288912311348ea70aee_18.png)

在个人主页或账号设置中，可以找到“管理橱窗”的入口，点击即可进入带货后台。

![](img/49aa2cac4d7ff288912311348ea70aee_20.png)

查看他人橱窗时，会显示两个关键数据：
*   **账号“已售”数**：表示该博主所有带货商品的历史总销量。例如：`已售 2.4万`。
*   **单个商品“已售”数**：表示该商品在所有带货博主那里的历史总销量。例如：`已售 5.9万`。

![](img/49aa2cac4d7ff288912311348ea70aee_22.png)

![](img/49aa2cac4d7ff288912311348ea70aee_23.png)

![](img/49aa2cac4d7ff288912311348ea70aee_25.png)

这两个数据分别反映了博主的带货能力和商品的市场热度。
