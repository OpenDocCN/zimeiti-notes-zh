## 概述

![](img/d737b54b4425391641b786a7e57e2ed7_5.png)

商品数据是平台考核商品排名、决定流量分配的关键依据。优化商品数据可以有效提升商品权重，从而获得更多自然曝光和访客。本节将详细解析平台考核哪些数据、这些数据如何影响权重，以及如何进行优化。

## 商品数据对权重的影响

![](img/d737b54b4425391641b786a7e57e2ed7_22.png)

![](img/d737b54b4425391641b786a7e57e2ed7_23.png)

商品排名会严格考核商品数据。虽然大多数人都知道这一点，但对于具体考核机制可能并不清楚。接下来我们将详细讲解。

![](img/d737b54b4425391641b786a7e57e2ed7_25.png)

![](img/d737b54b4425391641b786a7e57e2ed7_26.png)

平台考核的数据维度覆盖从曝光到成交乃至售后的全过程。其中，**点击率、销量和转化率**是重点考核部分。

![](img/d737b54b4425391641b786a7e57e2ed7_28.png)

![](img/d737b54b4425391641b786a7e57e2ed7_30.png)

### 1. 点击率考核 🎯

点击率是成交前的关键数据。更高的点击率意味着产品对用户吸引力更强，平台会因此给予更多推荐加权。

![](img/d737b54b4425391641b786a7e57e2ed7_32.png)

![](img/d737b54b4425391641b786a7e57e2ed7_33.png)

我们可以通过一个例子来理解：假设有商品A和商品B，在相同曝光下，商品A点击率为1%，商品B点击率为10%。即使不考虑后续成交，平台也会判定商品B更受用户喜爱，从而在初期给予商品B更多曝光和访客。如果后续成交数据正常，商品B获得的长期流量支持将远高于商品A。

![](img/d737b54b4425391641b786a7e57e2ed7_34.png)

![](img/d737b54b4425391641b786a7e57e2ed7_35.png)

![](img/d737b54b4425391641b786a7e57e2ed7_37.png)

**核心公式**：
`点击率 = (点击次数 / 曝光次数) * 100%`

![](img/d737b54b4425391641b786a7e57e2ed7_38.png)

在曝光量相同的情况下，点击率直接决定了访客量的多少。访客越多，成交概率自然越高，商品权重也随之提升。

![](img/d737b54b4425391641b786a7e57e2ed7_40.png)

### 2. 销量考核 📊

![](img/d737b54b4425391641b786a7e57e2ed7_41.png)

![](img/d737b54b4425391641b786a7e57e2ed7_42.png)

![](img/d737b54b4425391641b786a7e57e2ed7_43.png)

![](img/d737b54b4425391641b786a7e57e2ed7_44.png)

![](img/d737b54b4425391641b786a7e57e2ed7_45.png)

销量考核并非简单的看订单数量（单量），它是一个多维度的综合评估，主要包括：**单量、成交额和UV价值**。

![](img/d737b54b4425391641b786a7e57e2ed7_46.png)

![](img/d737b54b4425391641b786a7e57e2ed7_47.png)

*   **单量**：代表市场对产品的认可度和市场份额。单量越高，权重通常越高。
*   **成交额与客单价**：平台同时考核成交额。如果只考核单量，低客单价商品将永远占优，这不合理。因此，高客单价和高成交额的商品在排名计算中同样重要。
    **核心公式**：
    `客单价 = 成交额 / 销量`
    在对应价格区间的产品中，你的销量和成交额越高，权重就越好。
*   **UV价值**：这是非常重要的考核指标。它衡量的是每个访客带来的平均销售额。
    **核心公式**：
    `UV价值 = 成交额 / 访客数`
    举例来说，A商品100个访客成交100元，UV价值为1；B商品50个访客成交100元，UV价值为2。平台会判定B商品对流量的利用效率更高，从而在排名时给予其更靠前的位置和更多曝光。

![](img/d737b54b4425391641b786a7e57e2ed7_49.png)

### 3. 转化率考核 🔄

![](img/d737b54b4425391641b786a7e57e2ed7_51.png)

![](img/d737b54b4425391641b786a7e57e2ed7_52.png)

转化率与上述各项数据紧密关联。在相同访客进店的情况下，转化率越高，意味着成交单量越多，成交额越高，UV价值也越高，从而商品权重也越高。

![](img/d737b54b4425391641b786a7e57e2ed7_54.png)

**核心公式**：
`转化率 = (成交订单数 / 访客数) * 100%`

![](img/d737b54b4425391641b786a7e57e2ed7_56.png)

![](img/d737b54b4425391641b786a7e57e2ed7_57.png)

![](img/d737b54b4425391641b786a7e57e2ed7_58.png)

转化率是衡量商品变现能力的最直接指标，优化转化率是提升权重的重中之重。

![](img/d737b54b4425391641b786a7e57e2ed7_60.png)

## 如何优化商品数据

![](img/d737b54b4425391641b786a7e57e2ed7_62.png)

上一节我们介绍了平台考核的核心数据指标，本节我们来看看具体的优化方法。

![](img/d737b54b4425391641b786a7e57e2ed7_63.png)

![](img/d737b54b4425391641b786a7e57e2ed7_64.png)

![](img/d737b54b4425391641b786a7e57e2ed7_65.png)

其实方法并不复杂：**针对单品进行数据分析，密切关注流量变化情况，并重点跟进商品的转化率、UV价值和销量等数据。**

![](img/d737b54b4425391641b786a7e57e2ed7_67.png)

![](img/d737b54b4425391641b786a7e57e2ed7_68.png)

以下是优化时需要关注的要点：

![](img/d737b54b4425391641b786a7e57e2ed7_69.png)

*   **建立数据监控习惯**：各项数据指标是相辅相成变化的。例如，当访客上升时，要检查转化率、UV价值、单量是否达到了预期标准。
*   **进行对比分析**：如果发现某天访客开始下降，应立即回顾前一段时间的数据，分析是否是转化率、UV价值或单量出现了下滑，并找出原因。
*   **持续维护与优化**：根据数据分析结果，调整商品主图、标题、详情页、价格或营销活动，以持续优化各项数据指标。
