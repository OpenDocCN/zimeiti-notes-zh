# 小红书运营：P29：小红书笔记引流：平台流量分发机制 🚀

![](img/99202505b8b26f5fc95308aa1003884d_1.png)

在本节课中，我们将要学习小红书平台的核心流量分发机制。理解这套算法，是提升笔记曝光和排名的关键。我们将拆解其工作原理，让你明白平台如何决定将你的内容展示给哪些用户。

## 概述

小红书的内容展示页面与抖音等平台不同。它一次展示20篇笔记，用户需要下滑才能刷新到下一页的另外20篇。在这20个展示位中，通常包含约2个付费推广位（类似淘宝直通车）。我们的核心目标，是让自己的笔记在这20个展示位中，尤其是前几名，获得更高的排名。

## 核心评分系统：CES模型

平台通过一套综合评分系统来决定笔记的排名。这套系统可以简称为 **CES模型**，它代表了用户与笔记互动的主要行为：

*   **C** - 评论
*   **E** - 关注
*   **S** - 收藏、转发、点赞

每种互动行为都被赋予了不同的权重分值，共同构成一篇笔记的“基础评分”。

以下是CES模型中各项互动行为的具体分值：

*   **点赞**：1分
*   **收藏**：1分
*   **转发**：4分
*   **评论**：10分
*   **关注**：8分

**计算公式**：`单用户最高贡献分 = 1（点赞） + 1（收藏） + 4（转发） + 10（评论） + 8（关注） = 24分`

这意味着，如果一个用户对你的笔记完成了全部五项互动行为，他将为这篇笔记贡献24分。系统会综合计算笔记获得的所有互动分值，来为其进行基础排名。

## 流量分发的完整流程

上一节我们介绍了核心的CES评分模型，本节中我们来看看平台是如何利用这个模型进行流量分发的完整流程。

1.  **内容发布与标签匹配**：当你发布一篇笔记（包括图片和文案）后，系统会为其打上多个内容标签。这些标签会与你账号前期设定的领域标签（如美妆、美食、运动）相结合。
2.  **初始曝光（冷启动）**：笔记首先会获得一个基础的初始曝光量，通常在**150到200次展示**之间（对于原创内容）。这相当于一次“测试”。
3.  **数据考核与下一波推流**：系统会将你的笔记放入一个包含20篇同类型笔记的“竞赛池”中进行展示。平台会根据初始曝光产生的数据，决定是否给予下一波更大的流量。这个过程是分批次进行的，而不是一次性给足流量。

以下是决定笔记能否进入下一轮推广的关键数据指标：

*   **点击率**：用户在看到笔记封面和标题后，点击进入笔记页面的比率。这是所有后续互动的前提，**通常需要达到13%-15%以上**才有机会获得更多流量。
*   **互动率（CES）**：用户点击进入后，产生的点赞、收藏、评论、转发、关注等行为。这些数据直接贡献CES评分。
*   **完播率**：对于视频笔记，用户完整观看视频的比例。这反映了内容吸引力。
*   **互动值**：创作者与评论用户之间的回复互动频率。高互动值表明内容能引发讨论，质量较高。

**流程简述**：笔记获得初始曝光 → 考核点击率与互动数据（CES）→ 数据达标则进入第二波更大流量推广 → 继续考核 → 数据优秀则可能获得第三波翻倍流量，形成阶梯式增长。如果数据不达标，推广就会逐渐停止，笔记生命周期可能只有3天左右。

## 算法核心与公平性

无论数据指标如何变化，小红书流量分发的核心算法始终是**考察笔记带来的用户交互行为**。粉丝量不多的新账号，只要笔记内容优质，能引发高点击和高互动（CES评分高），同样有机会被系统推荐给更多用户，实现排名的提升。

平台算法相对公平，它为所有笔记提供了基础的展示机会（零层级曝光）。排名能提升到多高，完全取决于你的内容能否在“竞赛池”中通过数据击败同期其他笔记。

## 总结

本节课中我们一起学习了小红书的流量分发机制。我们了解到：

![](img/99202505b8b26f5fc95308aa1003884d_3.png)

1.  平台通过 **CES模型**（评论、关注、收藏/转发/点赞）对笔记进行综合评分和排名。
2.  流量分发是**分批次进行**的，系统根据每一波曝光产生的**点击率、互动率、完播率等数据**决定是否给予下一波流量。
3.  算法的核心是**用户交互行为**，而非单纯的粉丝量。优质内容即使在新账号上也能获得推荐。
4.  理解这套机制后，我们就应努力优化笔记的**封面标题（提升点击率）** 和**内容质量（提升CES互动分）**，从而在平台的流量竞赛中赢得更高排名，获得更多曝光。
