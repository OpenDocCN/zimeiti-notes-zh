# 抖音运营实战：P14：8. 结算收益

![](img/c54abd76ff883e0cbea9190264f453a7_0.png)

## 概述
在本节课中，我们将学习抖音账号如何通过影视推广项目结算收益。课程将重点介绍两种主要的收益来源：平台播放量奖金和参与“中视频计划”获得的多平台分成。掌握这些方法，可以帮助你在运营账号时实现收益最大化。

---

上一节我们介绍了账号运营的基础，本节中我们来看看如何将流量转化为实际收益。

收益结算的核心在于利用现有资源拓展收入渠道。单纯进行视频剪辑可以获得一份收益。如果在剪辑视频的基础上添加解说内容，并通过特定渠道发布，则有机会获得双份收益。

以下是实现双份收益的具体路径：

1.  **第一份收益：平台播放量奖金**
    将剪辑或解说后的影视推广视频发布在抖音平台。视频产生有效播放后，平台会根据规则结算播放量奖金。

2.  **第二份收益：中视频计划分成**
    参与“中视频计划”是获取额外收益的关键。“中视频计划”是字节跳动旗下抖音、今日头条、西瓜视频三个平台联合推出的创作者激励计划。

    其运作机制是：创作者制作一个符合要求的视频（如添加解说的影视内容），通过计划入口发布。该视频会在三个平台同步分发，所产生的播放量收益将由三个平台共同结算给创作者。

因此，一个添加了解说的影视推广视频，可以同时获得抖音的播放奖金，以及“中视频计划”带来的头条和西瓜视频的分成，从而实现收益翻倍。

## 总结
本节课我们一起学习了抖音影视推广项目的收益结算方式。核心在于理解并利用“中视频计划”，通过为一个视频添加解说，实现在抖音、今日头条、西瓜视频三个平台同时获取播放收益，达到双份收益的效果。这要求创作者在内容制作上付出更多努力，但回报也相应更高。
