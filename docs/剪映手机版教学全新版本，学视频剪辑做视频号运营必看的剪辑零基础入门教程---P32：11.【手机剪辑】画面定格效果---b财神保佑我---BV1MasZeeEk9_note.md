# 剪映手机版教学：P32：11. 画面定格效果制作教程

在本节课中，我们将学习如何使用剪映手机版制作一个富有动感的画面定格效果。这种效果常用于强调视频中的某个精彩瞬间，是视频剪辑中非常实用且炫酷的技巧。

## 概述
画面定格效果的核心是**冻结某一帧画面**，并对其施加动画和特效，使其从原视频中“跳脱”出来，形成视觉焦点。整个过程涉及定格功能、画中画、关键帧动画以及特效的叠加使用。

---

## 步骤详解

上一节我们了解了定格效果的基本概念，本节中，我们将一步步拆解其制作方法。

### 第一步：导入与定格
首先，在剪映中导入你的视频素材。将时间线移动到你想要实现定格效果的那一帧画面。

**操作**：点击下方工具栏中的 **“定格”** 按钮。软件会自动生成一个时长为3秒的静态图片片段。

### 第二步：素材整理
生成定格画面后，时间线上会多出一段静态画面。为了流程清晰，我们需要整理时间线。

以下是需要进行的操作：
*   删除定格点之后**多余的原始视频片段**，只保留定格前的部分和定格画面本身。
*   选中生成的**定格画面片段**，点击“复制”，得到一份相同的片段。

### 第三步：应用画中画
复制完成后，我们需要让两个相同的画面产生层次感。

**操作**：选中复制出来的定格片段，在工具栏中找到 **“切画中画”** 功能。将其切入画中画轨道，并拖动其开头，与主轨道上定格画面的开头**严格对齐**。

### 第四步：添加基础特效
现在，我们为作为底层的定格画面（主轨道）添加一个动态特效。

**操作**：点击下方工具栏的 **“特效”** -> **“画面特效”**。在 **“基础”** 分类下，找到 **“模菇”** 特效并添加。该特效会使画面产生轻微缩放和抖动，增加动感。

### 第五步：制作关键帧动画
这是让定格画面“活”起来的关键。我们将对画中画图层制作一个缩小并倾斜的动画。

以下是关键帧设置步骤：
1.  选中**画中画图层**，在其开头位置添加一个**关键帧**（点击菱形“+”号）。
2.  将时间线移动到该画中画图层的**结尾处**。
3.  在此状态下，双指在预览区**缩小**画面，并稍微**旋转**，制造一个倾斜退出的效果。软件会自动在结尾处生成第二个关键帧。

### 第六步：叠加氛围特效
为了增强视觉效果，我们还需要添加两种氛围特效。

**操作流程**：
1.  为主轨道（第一个）的定格画面添加 **“变清晰II”** 特效（位于“基础”分类），模拟镜头对焦的感觉。
2.  为画中画图层添加一个边框特效，例如 **“粉黄渐变”**（位于“边框”分类）。添加后，务必在作用对象中选择 **“画中画”**，使其只应用于上层画面。

### 第七步：添加转场与音乐
最后，我们来完善细节，让过渡更自然，并配上音乐。

**操作**：
*   **转场**：在主视频与定格画面之间的**小白块**（转场按钮）上点击，添加一个 **“闪白”** 转场，使定格瞬间更醒目。
*   **音乐**：点击 **“音频”** -> **“抖音收藏”**，添加你准备好的背景音乐，并根据视频长度裁剪音乐。

## 效果预览与总结
点击播放按钮，即可查看完整的画面定格效果。本节课中，我们一起学习了制作画面定格效果的完整流程：从**导入定格**，到**复制切画**，再到**关键帧动画**与**多层特效**的叠加应用。掌握这个技巧，你就能轻松为视频中的高光时刻增添电影般的视觉冲击力。
