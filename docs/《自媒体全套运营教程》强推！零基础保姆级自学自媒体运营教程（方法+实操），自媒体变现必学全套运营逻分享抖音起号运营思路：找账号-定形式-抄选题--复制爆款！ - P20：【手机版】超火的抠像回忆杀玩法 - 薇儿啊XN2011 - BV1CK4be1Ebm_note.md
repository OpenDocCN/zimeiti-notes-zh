# 《自媒体全套运营教程》：P20：手机版超火抠像回忆杀玩法 🎬

![](img/f1a7283540a0eded269349320c24e88d_1.png)

在本节课中，我们将学习如何制作流行的“抠像回忆杀”视频。我们将重点掌握滤镜与滤镜图层的区别、智能抠像功能，并复习卡点剪辑技巧。通过本教程，你将能够独立完成一个具有电影感回忆效果的作品。

---

## 概述

本节课的核心是制作一个“抠像回忆杀”视频。我们将从导入素材开始，学习如何利用智能抠像分离人物与背景，并运用滤镜和特效来营造复古氛围。同时，我们会复习卡点技巧，确保视频节奏与音乐完美匹配。

---

## 素材准备与音乐导入

首先，我们需要导入视频素材并添加背景音乐。

1.  点击“开始创作”，导入准备好的环境视频素材。
2.  关闭视频原声。
3.  点击“添加音频” -> “提取音乐”，导入准备好的背景音乐。

**注意**：本案例的视频画面已根据音乐节拍预先剪辑好。如果你自行剪辑，需要根据音乐重音进行卡点，使画面切换与节奏同步，效果会更好。

---

## 添加画中画与智能抠像

上一节我们导入了背景视频和音乐，本节中我们来看看如何添加人物素材并进行智能抠像。

1.  点击后方的“添加”按钮，导入人物回头的视频。
2.  选中新导入的人物视频，点击“切画中画”，将其放入画中画轨道。
3.  此时，我们得到了主轨的环境视频和画中画轨道的人物视频。

接下来，我们需要根据音乐节奏对人物视频进行处理。

4.  为了精确定位，可以返回主菜单查看音乐的波形图。双手滑动放大时间轴，找到音乐开始的强节奏点。
5.  在音乐起始的强节奏点处，选中画中画视频，点击“分割”，将其一分为二。
6.  选中分割后的后半段人物视频，点击“抠像” -> “智能抠像”。等待抠像完成并勾选确认。

**效果说明**：前半段人物视频保持原样，与背景融合；后半段经过智能抠像，人物被单独提取出来，背景得以显现，形成“回忆浮现”的视觉效果。

## 滤镜的应用：单独滤镜 vs. 滤镜图层

在完成抠像后，我们需要为背景添加复古的黑白滤镜。添加滤镜有两种方式，它们的效果有重要区别。

**第一种方式：为单个视频添加滤镜**
*   选中需要添加滤镜的视频片段（例如主轨道背景视频）。
*   点击下方菜单栏的“滤镜”。
*   在滤镜列表中选择“黑白”分类下的“褪色”滤镜。
*   将滤镜强度调整为100%，并勾选确认。

**代码/操作描述**：
```
选中视频片段 -> 点击【滤镜】 -> 选择【黑白-褪色】 -> 强度:100% -> 确认
```

**第二种方式：通过主菜单添加滤镜（生成滤镜图层）**
*   点击左上角返回按钮，直至回到主菜单界面。
*   点击主菜单下方的“滤镜”按钮。
*   同样选择“黑白-褪色”滤镜，强度100%，并勾选确认。

**核心概念区别**：
*   **单独滤镜**：只作用于你选中的那一个视频片段。
*   **滤镜图层**：会在时间轴上生成一个独立的蓝色滤镜轨道。这个图层会覆盖其长度范围内的所有视频轨道（包括主轨道和画中画轨道），实现全局调色效果。

**结论**：若只需改变某个片段的颜色，使用第一种方式；若需要为整个片段（包含所有画中画）统一添加滤镜，则使用第二种方式。在本案例中，我们为主轨道背景视频使用第一种方式添加“褪色”滤镜。

## 添加画面特效

为了增强复古回忆的氛围，我们可以为视频叠加多个画面特效。

![](img/f1a7283540a0eded269349320c24e88d_3.png)

以下是添加特效的步骤：
1.  返回主菜单，点击“特效”。
2.  选择“画面特效”。
3.  在特效列表中，首先添加“复古”分类下的“黑色噪点”特效。
4.  再次点击“画面特效”，添加“投影”分类下的“车窗影”特效。
5.  在时间轴上，将这两个特效图层的长度拖动至与整个视频时长一致，确保特效全程生效。

**关键点**：特效图层与滤镜图层类似，只对其覆盖的时间段内的画面生效。必须确保特效条的长度覆盖所有需要特效的画面。

---

## 微调与最终导出

最后，我们需要检查视频并做最终调整。

1.  播放预览视频，发现画中画人物视频可能比背景视频短。
2.  选中人物视频，点击“变速” -> “常规变速”，将速度调整为0.8倍（慢放），使其时长延长。
3.  将延长后多余的部分分割删除，使其与主视频对齐。
4.  再次完整播放预览，确认卡点、抠像、滤镜和特效均达到预期效果。
5.  点击右上角“导出”按钮，生成最终视频。

## 总结

本节课中，我们一起学习了“抠像回忆杀”视频的制作全流程：
1.  **复习卡点技巧**：依据音乐波形进行画面分割与切换。
2.  **运用智能抠像**：分离人物与背景，创造视觉焦点变化。
3.  **区分滤镜应用**：掌握了为单个片段添加滤镜与通过主菜单添加全局滤镜图层的核心区别。
4.  **叠加画面特效**：使用“黑色噪点”和“车窗影”等特效增强复古氛围。
5.  **进行最终微调**：通过变速功能调整素材时长，并完成导出。
