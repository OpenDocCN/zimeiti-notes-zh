## 流量模式概述 📊

![](img/ff59b7cb1ada2293b196ecb3e087120d_5.png)

![](img/ff59b7cb1ada2293b196ecb3e087120d_6.png)

上一节我们介绍了店铺的基础设置，本节中我们来看看店铺的血液——流量。小红书的流量模式主要分为两组：**笔记引流**和**搜索推荐引流**。理解这两种模式是做好店铺运营的关键。



## 一、笔记引流：流量占比的“大头” 📝

笔记引流是整个小红书流量体系中占比最大的部分。它指的是通过发布与商品相关的笔记内容，吸引用户点击进入店铺的引流方式。

*   **流量占比公式**：`笔记引流占比 ≈ 70%`
*   **特点**：流量大，但用户目标不明确，转化率相对较低。
*   **核心逻辑**：通过优质内容吸引潜在客户的偶然关注，为店铺带来人气和曝光。

以下是关于笔记引流需要了解的几个关键点：

1.  **发布频率**：对于新店，建议每天发布1-2篇单品笔记。中后期可以调整为每天1篇或每2-3天1篇，但发布节奏需要固定下来。
2.  **发布上限**：小红书平台对单品笔记的每日发布数量有限制，超过两个可能会导致流量权重下降。
3.  **笔记效果周期**：笔记的引流效果并非永久。
    *   **小型笔记**（效果一般）：引流效果持续约3天。
    *   **中型笔记**（效果较好）：引流效果持续约7天。
    *   **大型/爆款笔记**（效果极佳）：引流效果最多持续15天。
4.  **不稳定性**：笔记的流量波动较大，无法精确预测能带来多少进入店铺的有效访客。因此，需要持续更新笔记来维持流量。



## 二、搜索推荐引流：转化率的“核心” 🔍

搜索推荐引流是用户通过搜索关键词找到并进入店铺的流量。虽然它的整体流量占比小于笔记引流，但其转化价值极高。

*   **流量占比公式**：`搜索推荐引流占比 ≈ 30%`
*   **特点**：流量精准，用户购买意图明确，转化率非常高。
*   **核心逻辑**：满足用户的主动搜索需求，是带来稳定订单的核心渠道。

搜索推荐引流主要包含以下三种形式：

![](img/ff59b7cb1ada2293b196ecb3e087120d_8.png)

![](img/ff59b7cb1ada2293b196ecb3e087120d_10.png)

1.  **关键词引流**：用户在搜索框输入关键词（如“夏季连衣裙”、“平价水杯”），如果你的商品标题、标签中包含了这些关键词并且排名靠前，就有机会被展示。关键词可以组合成长尾词，每个组合都有独立的排名。
2.  **排名引流**：指商品在某个关键词搜索结果中的自然排名位置。排名越靠前，获得的曝光和点击就越多。
3.  **付费引流**：即通过小红书平台的付费推广工具（如薯条、信息流广告）来购买流量。**特别注意**：对于新店，不建议前期就大量投入付费引流。应在店铺有稳定基础销量和口碑后，用于突破增长瓶颈。过早依赖付费流量，一旦停止投入，店铺数据可能大幅下滑。



## 核心对比与运营重点 ⚖️

上一节我们分别介绍了两种引流模式，现在我们来总结一下它们的核心差异与运营侧重点。

*   **笔记引流（70%）**：目标是“广撒网”，扩大品牌和商品的曝光度，积累店铺人气。操作重心在于**持续产出优质的笔记内容**。
*   **搜索推荐引流（30%）**：目标是“精准捕捞”，直接获取有购买意向的客户。操作重心在于**优化商品标题、关键词，提升搜索排名**。

**重要提示**：切勿因为搜索引流只占30%而忽视它。实际上，这30%的精准流量所带来的订单转化，往往比70%的泛流量更有价值。两者相辅相成，缺一不可。前期可以侧重笔记引流打开局面，中后期必须优化搜索引流以提升稳定收益。
