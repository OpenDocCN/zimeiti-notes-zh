# 小红书运营：P17：16、小红书笔记流量运营-抖音选视频-什么样的视频是好视频 🎬

在本节课中，我们将学习如何从抖音平台筛选出适合搬运到小红书的“好视频”。我们将深入探讨好视频的判断逻辑、具体筛选指标，并通过实操演示，让你掌握快速找到潜力爆款视频的方法。

## 好视频的核心逻辑 🔍

上一节我们介绍了如何寻找对标关键词，本节中我们来看看如何利用这些关键词找到高质量的视频源。其核心逻辑在于利用 **平台发布信息差**。

许多内容创作者（例如“小杨哥”这类达人）通常只专注于一个主流平台（如抖音）发布内容。由于对小红书平台的认知和投入度不高，他们很少会将同一内容同步到小红书。这就产生了信息差：在抖音上已经验证过的爆款视频，有很大概率在小红书上也能获得成功。相比之下，视频号等平台与抖音的用户重叠度高，这种信息差效应较弱。

## 好视频的筛选指标 📊

![](img/d33e7e6f26081c7cd73582292a7465e1_1.png)

理解了逻辑后，我们需要具体的指标来指导筛选。以下是针对抖音平台筛选视频时需要关注的几个关键点：

![](img/d33e7e6f26081c7cd73582292a7465e1_2.png)

![](img/d33e7e6f26081c7cd73582292a7465e1_4.png)

![](img/d33e7e6f26081c7cd73582292a7465e1_6.png)

以下是筛选时必须卡住的四个核心指标：

![](img/d33e7e6f26081c7cd73582292a7465e1_8.png)

![](img/d33e7e6f26081c7cd73582292a7465e1_10.png)

1.  **发布时间**：视频必须在 **24小时** 内发布。公式表示为：`发布时间 ≤ 24小时`。
2.  **点赞表现**：该视频的点赞量，需要显著高于该账号日常视频的平均点赞量。
3.  **内容风险**：尽量避免使用有真人模特露脸、或风格过于专业（如知名网红）的视频，以降低被平台识别为搬运或侵权的风险。对于童装、母婴等类目，因模特复用率高，相对安全。
4.  **视频形式**：直接由图片生成的 **幻灯片式视频** 不要选用，这类内容缺乏竞争力。

![](img/d33e7e6f26081c7cd73582292a7465e1_62.png)

## 总结 📝

![](img/d33e7e6f26081c7cd73582292a7465e1_85.png)

![](img/d33e7e6f26081c7cd73582292a7465e1_87.png)

本节课中我们一起学习了从抖音筛选“好视频”的方法。关键在于利用 **平台发布信息差**，并严格卡住 **24小时内发布**、**点赞量显著高于账号日常水平**、**规避高风险内容**、**拒绝幻灯片视频** 这四项指标。记住不同类目的点赞量参考线（女装300+，童装150+），就能高效地找到适合搬运到小红书的潜力爆款视频，为你的无货源电商运营提供优质内容素材。
