# 微信视频号教程：P41：如何扒取直播间话术

在本节课中，我们将学习如何利用工具高效扒取其他成功直播间的话术，并将其转化为自己可用的脚本，从而快速提升直播带货能力。

上一节我们探讨了如何寻找和复制爆款商品与视频。本节中，我们来看看如何获取直播的核心——话术。许多商品的销售具有时效性，例如年货，掌握对应的话术是成功开播的关键。

## 🛠️ 所需工具与准备

![](img/69c45c7ff12f0ee7735c3535053f4bd9_3.png)

扒取话术需要使用一个名为“飞书妙计”的工具。以下是具体操作步骤。

首先，在手机或电脑上下载并安装“飞书”应用。登录后，进入应用主界面。

## 📝 核心操作步骤

以下是使用飞书妙计录制并导出直播间话术的完整流程。

1.  **找到工具**
    在飞书应用中，点击底部菜单栏的“工作台”。在工作台页面中，找到并点击“飞书妙计”功能。

2.  **开始录制**
    进入飞书妙计后，点击界面右下角的**语音按钮**，工具即开始录音。此时，切换到你想学习的直播间，让飞书妙计在后台录制直播间的所有声音。

3.  **结束与导出**
    录制一段时间（建议15分钟左右以获取完整话术结构）后，返回飞书妙计点击完成。系统会自动将录音转换为文字。处理完成后，点击生成的记录，选择“分享”功能。分享权限选择“可编辑”或“可阅读”均可。

4.  **提取与整理话术**
    复制生成的分享链接，将其发送到微信并打开。打开后即可看到直播话术的完整文字稿。你可以复制这些文字，进行编辑和提炼，去除个人语气词，保留产品介绍、互动和逼单的核心逻辑，最终形成属于自己的直播脚本。

## 💡 应用与总结

![](img/69c45c7ff12f0ee7735c3535053f4bd9_5.png)

本节课中我们一起学习了如何借助“飞书妙计”工具扒取直播间话术。你可以将整理好的话术写在纸上或存入备忘录，在直播时参考使用。这个方法能帮助你快速学习优秀主播的成交技巧，缩短自己摸索的时间。掌握话术后，结合之前学习的选品与视频制作，你就可以更自信地开启自己的直播带货之旅了。
