## 什么是自媒体？

了解了课程结构后，我们现在来探讨最核心的问题：究竟什么是自媒体？

用最直白的话解释，自媒体就是：**你自己在互联网平台上创建并运营的个人媒体账户**。

其运作流程可以用一个简单的公式表示：

**注册账号 + 发布内容（文章/视频） + 获得观看 = 获取收益**

![](img/bc1454290861f4b24b5707035b01c8c7_1.png)

![](img/bc1454290861f4b24b5707035b01c8c7_2.png)

具体步骤如下：
1.  打开手机或电脑，搜索一个自媒体平台（如今日头条、百家号、B站等）。
2.  在该平台注册一个属于你自己的账号。
3.  在你的账号上发布自己创作的文章或视频。
4.  当内容有人观看时，你便可以依据平台的规则，通过播放量或阅读量来结算收益。

![](img/bc1454290861f4b24b5707035b01c8c7_3.png)

这个过程所代表的个人化、去中心化的内容创作与传播模式，就是自媒体。
