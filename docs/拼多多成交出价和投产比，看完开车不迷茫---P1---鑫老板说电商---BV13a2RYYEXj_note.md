# 拼多多直通车教程：P1：成交出价与投产比选择指南 🚗

在本节课中，我们将要学习拼多多直通车推广中“成交出价”与“投产比出价”的核心区别与选择策略。这对于新手卖家而言是一个常见困惑点，理解两者的本质将帮助你做出更明智的推广决策。

## 核心概念解析

上一节我们介绍了课程目标，本节中我们来看看两种出价模式的基本定义。

*   **成交出价**：你为每一笔通过直通车带来的**成交订单**所支付的固定金额。其本质是一种**按效果付费（CPS）**的合作模式。
    *   **公式**：`广告花费 = 成交订单数 × 成交出价`
*   **投产比出价**：你设定一个目标投入产出比（ROI），系统自动调整出价，力求使`总交易额 / 广告花费`接近你设定的比值。其本质是一种**利润分成**的合作模式。
    *   **公式**：`目标投产比(ROI) = 总交易额 / 广告花费`

## 选择逻辑：平台与卖家的双视角

理解了基本概念后，我们需要从平台和卖家自身两个角度来思考如何选择。

### 平台的合作偏好

你可以将平台视为一个流量供应商。平台更倾向于与哪种出价模式的卖家合作呢？

以下是基于平台利益的简单分析：

1.  **偏好成交出价**：在商品转化能力相近的情况下，平台更愿意推广采用“成交出价”的商品。因为平台每带来一单成交，就能获得一笔**固定且明确的收入**，风险更低，结算简单。
2.  **慎对投产比出价**：采用“投产比出价”意味着平台需要与你共享利润。若你的商品转化率不佳，平台投入流量后可能无法获得满意分成，因此合作意愿会降低。除非你的产品能带来巨大的规模效应。

### 卖家的选择策略

从卖家，尤其是新手卖家的角度出发，选择的核心在于**风险控制**与**产品特性**。

以下是针对不同卖家的选择建议：

*   **对新手/小卖家**：在初期对市场转化率未知的情况下，**成交出价是更稳妥的选择**。它让你能够以相对可控的单次成本测试市场，避免因转化不佳而在“投产比”模式下产生不可控的巨额花费。
*   **对低价标品卖家**：如果你的商品客单价极低（例如10元以内），例如日常快消品，由于单笔订单利润微薄，无法承受较高的固定成交出价（平台均值可能在4元以上）。此时，**应选择投产比出价**。这类产品靠规模取胜，平台也愿意通过分成模式合作，以覆盖海量用户。
*   **对中高客单价卖家**：大部分中高客单价卖家可以**前期从成交出价开始**，用于测试和积累数据。待销售稳定、转化率可控后，如果追求更稳定的广告花费占比，可以**考虑转为投产比出价**。

## 总结与行动指南

本节课中我们一起学习了拼多多直通车“成交出价”与“投产比出价”的核心逻辑。

*   **本质**：成交出价是**“按单固定付费”**，投产比是**“按利润分成”**。
*   **选择心法**：没有绝对的好坏，只有适合与否。
    *   **求稳、测款、新手入门**，优先选择**成交出价**。
    *   **客单价极低、追求规模、转化稳定**的标品，可以考虑**投产比出价**。
*   **最终建议**：对于绝大多数新手卖家，从**成交出价**起步是风险最低、最易上手的选择。在积累足够数据并对自身产品转化能力有清晰认知后，再评估是否调整策略。
