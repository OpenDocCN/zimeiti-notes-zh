# 微信视频号教程：P34：2. 如何去视频里的字幕 🎬

在本节课中，我们将学习如何去除视频中的字幕。我们将介绍两种实用的方法：使用视频编辑软件的蒙版功能，以及借助微信小程序进行一键处理。掌握这些技巧，可以帮助你更灵活地处理素材，制作出更专业的视频内容。

---

## 方法一：使用蒙版功能逐段去除字幕

上一节我们介绍了视频剪辑的基础概念，本节中我们来看看如何利用蒙版功能处理字幕位置变化的视频。

![](img/ec323fa47e594da787ae6b4895acfb9c_1.png)

如果视频中的字幕出现在不同位置，例如前半部分在中间，后半部分在底部，我们需要先将视频分割成段落，然后分别处理。

以下是具体操作步骤：

![](img/ec323fa47e594da787ae6b4895acfb9c_3.png)

1.  **分割视频**：在视频编辑软件中，找到字幕位置发生变化的时间点，将视频分割成独立的片段。
    ![](img/ec323fa47e594da787ae6b4895acfb9c_1.png)

2.  **应用矩形蒙版**：选中需要处理的片段，添加一个矩形蒙版。
    ![](img/ec323fa47e594da787ae6b4895acfb9c_3.png)

3.  **调整蒙版位置**：将矩形蒙版调整到完全覆盖字幕的区域。

4.  **选择反转效果**：点击蒙版设置中的“反转”选项。

5.  **设置背景模糊**：返回主编辑界面，找到“背景”选项，选择“画布模糊”效果。这样，原字幕区域就会被模糊的背景所替代，从而实现去字幕的效果。

对于字幕始终在同一位置的视频，则无需分割，直接对整个视频应用上述蒙版操作即可。

## 方法二：使用微信小程序一键去除字幕

除了手动编辑，我们还可以利用更便捷的工具。接下来，我们看看如何使用微信小程序快速去除字幕。

在微信中搜索“去字幕”，可以找到许多相关小程序，其原理与去水印工具类似。

以下是使用小程序的操作流程：

1.  **选择小程序**：在搜索结果中，选择一个功能可靠的小程序，例如“3D去字幕”。
    ![](img/ec323fa47e594da787ae6b4895acfb9c_5.png)

2.  **导入视频**：进入小程序后，点击“视频去字幕”功能，将需要处理的视频导入。

3.  **调整与提交**：系统会自动识别字幕区域，你可以手动调整选框的位置和大小，确保覆盖所有字幕。确认后，提交进行去字幕处理。
    ![](img/ec323fa47e594da787ae6b4895acfb9c_7.png)

![](img/ec323fa47e594da787ae6b4895acfb9c_7.png)

4.  **获取结果**：处理完成后，在小程序内或下载链接中即可获取去除了字幕的视频。

**核心提示**：小程序处理短视频效果较好。对于长视频或字幕位置复杂的情况，其效果与手动蒙版处理相似，可能会留有轻微痕迹。对于字幕位置变化的视频，可以先将视频分割，再分段使用小程序处理。

---

本节课中我们一起学习了两种去除视频字幕的方法：**使用编辑软件的蒙版功能进行精细处理**，以及**利用微信小程序实现快速一键去除**。你可以根据视频的复杂程度和自身需求，选择最适合的方法来优化你的视频素材。
