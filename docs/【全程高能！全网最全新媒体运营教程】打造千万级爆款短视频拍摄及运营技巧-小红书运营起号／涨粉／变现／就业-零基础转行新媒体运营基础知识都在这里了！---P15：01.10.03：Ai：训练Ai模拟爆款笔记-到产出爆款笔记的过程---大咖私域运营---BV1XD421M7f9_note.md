# 新媒体运营：P15：01.10.03：🤖 训练AI模拟爆款笔记到产出爆款笔记的过程

在本节课中，我们将学习如何训练AI（如ChatGPT）分析爆款笔记，并最终生成具备爆款潜质的笔记文案。整个过程分为两个核心步骤：**分析学习**与**模仿创作**。

上一节我们介绍了通过“喂资料”的方式，让AI理解特定平台的文案风格。虽然上一节的旅游攻略案例效果不理想，但方法是正确的。本节中，我们来看看如何通过分析具体的爆款笔记，让AI学习其成功要素，并指导其创作。

## 分析爆款笔记：让AI学习成功要素

![](img/d212d3b3b120451a8bc555d64d9e8056_3.png)

![](img/d212d3b3b120451a8bc555d64d9e8056_5.png)

![](img/d212d3b3b120451a8bc555d64d9e8056_7.png)

要让AI学会创作爆款笔记，首先需要让它理解“什么是爆款”。我们可以通过提供真实的爆款笔记文案，并要求AI从以下几个维度进行分析：

以下是要求AI分析爆款笔记的几个核心指令点：

![](img/d212d3b3b120451a8bc555d64d9e8056_9.png)

![](img/d212d3b3b120451a8bc555d64d9e8056_11.png)

1.  **爆款因素分析**：分析文案为何能成为爆款，例如是否引发了情感共鸣、提供了实用价值等。
2.  **整体结构分析**：拆解文案的“起承转合”。
    *   **开篇**：用什么方式（如提问、感叹、故事）吸引用户。
    *   **中间**：用什么内容（如痛点分析、经验分享、干货方法）留住用户，提升停留时长。
    *   **结尾**：用什么“钩子”（如鼓励、引导私信、关注）促使用户互动。
3.  **关键词分析**：分析笔记中是否刻意植入了特定关键词，以及关键词的出现频率和分布方式。

## 实战演练：以“考研”领域为例

![](img/d212d3b3b120451a8bc555d64d9e8056_17.png)

![](img/d212d3b3b120451a8bc555d64d9e8056_19.png)

我们以小红书平台的“考研”领域爆款笔记为例，进行演示。

![](img/d212d3b3b120451a8bc555d64d9e8056_21.png)

![](img/d212d3b3b120451a8bc555d64d9e8056_23.png)

**第一步：收集并输入爆款笔记文案**
首先，我们需要找到几篇点赞量高的考研相关笔记，将其全文复制。

![](img/d212d3b3b120451a8bc555d64d9e8056_25.png)

![](img/d212d3b3b120451a8bc555d64d9e8056_27.png)

**第二步：向AI下达分析指令**
接下来，在ChatGPT中新建对话，并输入类似以下指令：
```
请你分析以下提供的小红书平台上的考研爆款笔记文案。
分析要求：
1. 分析每篇文案的爆款因素（如：情感共鸣、实用干货等）。
2. 分析每篇文案的整体结构：开篇如何吸引用户？中间如何戳中用户痛点？结尾如何引导互动或关注？
3. 分析每篇笔记是否设计了关键词。如果有，指出是哪个关键词，并说明其出现频率。
```
然后将复制好的笔记文案粘贴给AI。

![](img/d212d3b3b120451a8bc555d64d9e8056_29.png)

![](img/d212d3b3b120451a8bc555d64d9e8056_31.png)

**第三步：解读AI的分析结果**
AI会返回类似以下的分析：
*   **爆款因素**：情感共鸣（如备考焦虑）、逆袭故事、提供详细备考计划、时间节点明确、正能量鼓励、使用表情符号增加亲和力等。
*   **结构分析**：开篇用焦虑情绪引发共鸣 → 中间分享个人逆袭经验和具体方法 → 结尾鼓励并引导私信。
*   **关键词分析**：多数笔记没有明显刻意设计的关键词，词汇多为“考研”、“备考”等直接常用词。

通过分析多篇笔记，AI会总结出该领域爆款内容的常见模式。

## 整合要素并生成大纲

在AI分析完多篇笔记后，我们需要让它将分析出的爆款因素进行整合与去重，形成一个清晰的要素列表。

可以给AI如下指令：
```
请将你前面分析过的所有考研爆款笔记的爆款因素总结一下，整合成一个清晰的列表。
```
AI可能会总结出：情感共鸣、逆袭故事、明确备考计划、时间管理建议、资源技巧分享、正能量鼓励、使用表情符号等。

得到要素列表后，即可指令AI根据这些要素创作新笔记的大纲。
```
我需要写一篇关于考研的小红书笔记。请结合你刚才总结的爆款因素，为我规划一个具备爆款潜质的笔记大纲。大纲需要细分到每个部分的具体要点。
```
AI生成的大纲可能包括：吸引人的开篇（故事引入）、主题介绍、专业选择分析、各科目备考规划、时间管理与心态调整、结尾互动引导等。

![](img/d212d3b3b120451a8bc555d64d9e8056_33.png)

## 根据大纲生成完整文案

大纲确认无误后，即可让AI根据大纲和爆款要素生成完整文案。

指令可以如下：
```
请根据你上面提供的大纲，撰写一篇完整的小红书笔记文案。
要求：
1. 文案结构严格遵循爆款笔记的结构。
2. 开篇需要包含一个简单的“逆袭”小故事来引发读者情绪。
3. 文中需要适当使用emoji表情。
4. 尝试将“考研全流程”、“考研政治”等关键词自然地融入文案中。
```
AI将生成一篇包含个人经历、详细建议、鼓励话语和互动引导的完整笔记。这篇文案已经具备了爆款笔记的基本框架和要素。

## 优化与进阶

对于生成的初稿，我们可以进一步优化：
*   **调整语气和细节**：让文案更贴近真实个人分享。
*   **丰富与简洁**：在信息量充足的前提下，让表达更精炼、易读。
*   **利用高级工具**：使用ChatGPT-4.0或创建自定义的GPTs，通过上传更多风格范例和表情库，打造更懂你需求的专属AI助手，避免在长对话中遗忘前文指令的问题。

本节课中我们一起学习了如何通过“分析-总结-模仿”的流程，训练AI从理解爆款笔记到生成爆款笔记。关键在于为AI提供清晰的分析维度和创作指令，引导其学习成功案例中的结构、要素和风格，从而为我们产出高质量的内容初稿。这是一个可复制的流程，可以应用于不同内容领域。
