## 图片使用要求

图片是文章视觉的重要组成部分，合理使用图片能极大提升文章的可读性。

![](img/999955deda9d02c9f2645961d0135acc_7.png)

![](img/999955deda9d02c9f2645961d0135acc_8.png)

### 1. 为图片预留水印空间

![](img/999955deda9d02c9f2645961d0135acc_9.png)

![](img/999955deda9d02c9f2645961d0135acc_11.png)

在添加水印时，需注意避免水印覆盖在二维码等关键信息上。例如，在黑色二维码上添加水印会严重影响识别效果。

![](img/999955deda9d02c9f2645961d0135acc_13.png)

**核心操作**：在设计或选择图片时，应提前规划好水印位置。
```plaintext
[图片区域] - [预留空白区] - [二维码/关键信息区]
```

![](img/999955deda9d02c9f2645961d0135acc_15.png)

### 2. 确保图片风格统一

![](img/999955deda9d02c9f2645961d0135acc_17.png)

针对同一篇文章，所使用的所有图片应保持风格一致。风格杂乱的图片会破坏文章的整体美感和专业性。

以下是两种风格的对比：
*   **统一风格**：所有图片采用计时专业拍摄风格，视觉连贯。
*   **杂乱风格**：图片质量参差不齐，风格混搭（如卡通图、实拍图、低清图交替出现），导致文章视觉效果不佳。

![](img/999955deda9d02c9f2645961d0135acc_19.png)

![](img/999955deda9d02c9f2645961d0135acc_20.png)

### 3. 优先选择横长型图片

![](img/999955deda9d02c9f2645961d0135acc_22.png)

在移动端阅读时，横长型（左右顶格）图片能带来更佳的浏览体验。

![](img/999955deda9d02c9f2645961d0135acc_23.png)

*   **竖长图片问题**：在手机上会被放大至顶满左右两侧，占据一整屏，影响阅读流。
*   **小尺寸图片问题**：在手机两侧留下空白，与顶格的文字形成不协调的视觉缺口。
*   **横长图片优势**：无论原始尺寸如何，在手机上都会被自动压缩至与屏幕同宽，视觉效果统一、美观。

![](img/999955deda9d02c9f2645961d0135acc_25.png)

**视觉公式**：
```
横长图片宽度 > 竖长图片宽度 （在移动端显示时）
横长图片显示一致性 > 混合尺寸图片显示一致性
```

![](img/999955deda9d02c9f2645961d0135acc_26.png)

![](img/999955deda9d02c9f2645961d0135acc_27.png)

### 4. 适当使用动态素材

![](img/999955deda9d02c9f2645961d0135acc_29.png)

在满足上述要求的基础上，可以适当选用**GIF动图**等动态素材。动态内容比静态图片更具吸引力，能增加文章的趣味性和可读性。但需注意，不宜使用过长的视频，以免增加读者加载和观看的负担。

## 配色方案

![](img/999955deda9d02c9f2645961d0135acc_34.png)

配色是塑造文章视觉风格和品牌调性的关键。

![](img/999955deda9d02c9f2645961d0135acc_36.png)

### 1. 控制颜色数量

![](img/999955deda9d02c9f2645961d0135acc_38.png)

![](img/999955deda9d02c9f2645961d0135acc_40.png)

全文尽量使用不超过三种颜色。颜色过多会导致页面杂乱，分散读者注意力。

### 2. 统一内容颜色

*   **正文**：全文应使用一种统一的颜色。
*   **小标题**：所有小标题应使用另一种统一的颜色，并与正文颜色形成区分。例如，正文用黑色，小标题用紫色。

![](img/999955deda9d02c9f2645961d0135acc_42.png)

**代码示例（概念性）**：
```css
body-text { color: #333333; } /* 正文颜色 */
subtitle { color: #7B1FA2; } /* 小标题颜色 */
```

### 3. 配色与风格匹配

![](img/999955deda9d02c9f2645961d0135acc_44.png)

所选择的配色风格应与文章的整体风格或公司的品牌色（如Logo颜色）相匹配。
*   **文艺风格文章**：可选用淡紫、浅灰等柔和色调。
*   **公司品牌色为蓝色**：在标题、重点标注或装饰素材上可多用蓝色，以强化品牌形象。

![](img/999955deda9d02c9f2645961d0135acc_46.png)

## 其他排版技巧

![](img/999955deda9d02c9f2645961d0135acc_51.png)

### 1. 善用小标题分解长文

![](img/999955deda9d02c9f2645961d0135acc_53.png)

当文章较长（例如超过十段）时，应使用小标题（如1、2、3、4）将内容结构化。这样做有两个好处：一是使版面更美观，二是降低读者的阅读疲劳感，帮助其快速抓住重点。

![](img/999955deda9d02c9f2645961d0135acc_55.png)

![](img/999955deda9d02c9f2645961d0135acc_56.png)

### 2. 避免过度使用排版模板

许多排版编辑器提供丰富的装饰模板（如特殊的分隔线、边框、背景）。在一篇文章中过度使用不同风格的模板，会使页面显得花哨和凌乱，影响核心内容的传达。应保持克制，确保排版为内容服务。

![](img/999955deda9d02c9f2645961d0135acc_58.png)

### 3. 首尾结构呼应

如果文章开头使用了引语、摘要等特殊版式，结尾处也可以采用相似的设计元素进行呼应。这种首尾呼应的结构能使文章在形式上更完整、更美观。

![](img/999955deda9d02c9f2645961d0135acc_60.png)

### 4. 优化尾部二维码引导

![](img/999955deda9d02c9f2645961d0135acc_61.png)

![](img/999955deda9d02c9f2645961d0135acc_62.png)

在文章末尾放置公众号二维码时，不应只有“长按关注”四个字。应选择包含**简要文字介绍**的二维码模板，例如说明公众号能提供什么价值、属于哪个领域。这能有效提升转化率，给用户一个关注的明确理由。

**引导公式**：
```
有效引导 = 二维码 + 价值描述（例如：“关注我们，获取更多运营干货”）
无效引导 = 二维码 + “长按关注”
```

![](img/999955deda9d02c9f2645961d0135acc_64.png)

![](img/999955deda9d02c9f2645961d0135acc_65.png)
