## 核心理念与步骤 📝

上一节我们介绍了课程背景，本节中我们来看看起号期的核心投放理念与具体操作步骤。

千川在建模期面临的主要挑战是缺乏历史数据支撑，导致计划可能跑不动。解决此问题的核心思路是：**人群精准，由浅入深，逐步放量**。

以下是具体的操作步骤：
1.  **人群精准是基础**：确保投放人群足够精准，这是计划能跑动的前提。
2.  **由浅带深可放量**：初期通过相对宽泛的人群进行探索，再逐步向更精准的人群收缩。
3.  **多计划并行测试**：同时投放多条计划（例如10条），观察哪条计划先有数据反馈，以找到突破口。
4.  **前期避免使用放量投放**：建模期使用放量投放容易导致成本失控，应使用控成本投放。

## 人群分析与打包 🎯

理解了核心步骤后，我们需要学习如何找到并打包精准人群。这是搭建有效计划的第一步。

### 如何分析自身账号人群

![](img/c6ac061f828746bc70f2866ecf466b81_12.png)

![](img/c6ac061f828746bc70f2866ecf466b81_14.png)

如果你运营的店铺已有成交数据，可以通过抖音电商罗盘进行人群分析。

操作路径：**抖音电商罗盘 -> 人群 -> 人群洞察**。
1.  将时间范围调整为**30天**。
2.  在“人群特征”下，选择**支付**用户（而非进入用户）。
3.  查看支付用户中占比最高的几类人群，这些就是你的核心成交人群。例如，某账号核心人群可能是“小镇中老年”、“新锐白领”、“资深中产”、“都市银发”。
4.  同时，在“类目偏好”中，可以查看支付用户高频购买的类目，例如“服饰鞋包”、“美妆”等。

**核心公式**：
```
核心投放人群 = 支付用户中占比最高的几类人群 + 高频购买类目兴趣人群
```

### 如何分析竞争对手人群（新账号适用）

对于新账号或无数据的账号，可以通过分析竞争对手来获取人群参考。

操作路径：**精选联盟 -> 达人合作 -> 搜索竞争对手账号**。
1.  进入目标达人主页，查看其**粉丝分析**或**直播间观众分析**。
2.  重点关注“人群画像”中的**八大人群**分布（如小镇中老年、资深中产等）以及**年龄、性别**分布。
3.  同时，在“商品分析”或类似板块，查看其核心成交**类目**。
4.  将竞争对手的高频人群和类目，作为你初期投放的参考方向。

### 人群打包实战操作

分析出人群后，需要在千川的“达摩盘”或类似人群工具中进行打包。

![](img/c6ac061f828746bc70f2866ecf466b81_16.png)

![](img/c6ac061f828746bc70f2866ecf466b81_18.png)

以下是打包的核心逻辑与示例：
1.  **打包核心八大人群**：例如，针对“都市银发”人群单独打包。在打包时，可以进一步交集“消费能力预测”（选择中、高消费能力），并卡上“女性”、“41-50岁及50岁以上”的年龄性别，形成一个相对精准的包。
    *   **打包命名示例**：`都市银发_41+_女_中高消费`
2.  **打包类目兴趣人群**：在“行业偏好”或“行为兴趣”中，找到与产品相关的核心类目（如“服饰鞋包-女装”），并选择“90天内高频次”购买或关注此类目的人群进行打包。这类包通常较大，但能保证足够的探索量。
    *   **打包命名示例**：`女装_90天高频_潜在人群`
3.  **打包核心交集人群**：将“核心八大人群包”与“类目兴趣人群包”进行交集，得到更精准的人群。例如：`（都市银发_女_41+）交集（女装_90天高频）`。
4.  **确保人群包覆盖量级**：单个计划定向的人群包，覆盖人数建议不低于**5000万**。如果包太小，系统难以探索，计划可能无法跑动。

![](img/c6ac061f828746bc70f2866ecf466b81_20.png)

![](img/c6ac061f828746bc70f2866ecf466b81_22.png)

**关键点**：前期可以建立多个不同侧重点的人群包（例如3-5个），用于多计划测试。

### 新建广告计划

![](img/c6ac061f828746bc70f2866ecf466b81_32.png)

1.  **选择推广目标**：进入千川后台，选择 **“专业推广”**。
2.  **设置投放模式**：
    *   **优化目标**：前期起号选择 **“成交”** 或 **“直接成交”**。避免使用深度转化目标（如ROI）或“放量投放”。
    *   **投放方式**：选择 **“控成本投放”**。
3.  **设置预算与出价**：
    *   **日预算**：计算公式为 `（产品出价 * 30）`。例如产品出价100元，则日预算至少设置为3000元，以满足系统跑量的基础单量要求。
    *   **出价**：参考历史成交成本或保本ROI反推出的价格。新账户可从此价格开始测试。
4.  **设置定向人群**：
    *   选择 **“自定义定向”**。
    *   在“更多人群”中，上传或选择你在达摩盘中打包好的 **核心人群包**。
    *   同时，在基础定向中，仍需卡上**性别**和**年龄**（如：女，41-50岁，50岁以上），进行双重锁定，确保精准。
    *   **行为兴趣**和**达人定向**在前期可以暂不设置，主要依靠达摩盘人群包。
    *   **关闭智能放量**功能。
5.  **设置创意与分类**：
    *   选择准备好的视频或直播间画面。
    *   在“创意分类”和“创意标签”中，准确填写产品类目（如：服饰鞋包-女装-连衣裙）及核心关键词（如：中老年女装、马天奴）。

### 使用工具批量建计划（以有米有数为例）

为了提高效率，可以使用第三方工具（如有米有数）进行批量计划搭建。

![](img/c6ac061f828746bc70f2866ecf466b81_40.png)

1.  **创建定向包**：在工具中，将达摩盘打包好的人群包，创建为独立的“定向包”。
2.  **创建文案组**：准备多组广告文案，创建为“文案组”。
3.  **创建创意组**：上传多个视频素材，创建为“创意组”。
4.  **智能组合**：利用工具的“智能组合”功能，将不同的“定向包”、“文案组”、“创意组”进行排列组合，一键生成多条测试计划（例如：3个人群包 * 2个文案组 * 2个视频 = 12条计划）。

### 出价调整与计划优化

![](img/c6ac061f828746bc70f2866ecf466b81_46.png)

![](img/c6ac061f828746bc70f2866ecf466b81_48.png)

计划搭建完成后，进入关键的观察与调整阶段。

![](img/c6ac061f828746bc70f2866ecf466b81_50.png)

1.  **观察期**：计划开启后，观察1-2小时，看是否有曝光、点击和转化。
2.  **调价策略**：如果计划消耗很慢或不动，采取 **“小步快跑”** 的提价策略。
    *   在计划列表，选中不动计划，进行 **“批量修改出价”**。
    *   每次提价幅度为 **5%-10%**。例如，原出价100元，可上调至105-110元。
    *   每隔一段时间（如1小时）观察一次，若仍无动静，则继续按比例提价，直至计划开始跑量。
3.  **复制与重启**：如果提价效果不佳，可以选中原有计划，使用 **“批量复制”** 功能。在复制时，直接设置一个更高的出价（例如上调10%-20%），生成新计划进行测试。旧计划可以暂停。
4.  **核心逻辑**：通过不断提价和复制新建，刺激系统探索，直到找到能跑动的人群和出价点。
