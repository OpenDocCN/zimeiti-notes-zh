# 抖音运营新手起号：P39：如何衡量选题 📊

在本节课中，我们将学习如何判断一个选题是否优秀。我们将介绍五个核心的衡量标准，并通过具体例子帮助你理解如何应用这些标准来筛选和优化你的视频选题。

上一节我们介绍了如何寻找和生成选题，本节中我们来看看如何从众多想法中，挑选出真正有潜力的好选题。

---

## 选题衡量标准

以下是衡量选题好坏的五个核心标准。一个好的选题通常需要满足其中至少两到三点。

![](img/47f472aadef1eb816a6ea5e77922c9b4_1.png)

### 1. 低成本
用户倾向于选择那些能以最小付出（如时间、精力、金钱）获得最大效果的方案。例如，“玩手机躺着也能减肥”之所以吸引人，就是因为它承诺了“低成本”的达成方式。

**核心公式**：`用户获得感 / 用户付出成本 = 选题吸引力`。比值越高，选题吸引力越强。

### 2. 精准
选题需要精准地指向特定用户群体的需求或兴趣。例如，“新手必备的化妆全流程”就精准地服务于化妆初学者。

### 3. 熟悉
利用用户熟悉的场景、经历或情感，能快速引发共鸣。例如，“你有没有在电影院里碰见这样一类人？”这类选题利用了大众的共同记忆。

### 4. 高频
选题涉及的需求或场景应在用户生活中频繁出现。例如，“厨房收纳七件神器”针对的是日常家务痛点，属于高频需求。

### 5. 痛点
选题需要直击用户尚未被满足的强烈需求或烦恼。例如，对于想买某款口红但觉得太贵的用户，“领取大额优惠券”的选题就直接击中了“价格过高”的痛点。

---

## 标准应用与选题筛选

我们经常综合使用“低成本”和“痛点”这两个标准。例如，一个选题承诺“在家不用化妆就有男孩子来求婚”，它同时满足了“低成本”（无需付出）和“痛点”（渴望脱单）的要求。

当你产生一系列选题后，可以逐一用这五个标准去衡量。符合标准越多的选题，成为爆款的可能性就越大。

如果觉得所有选题都不够好，可以回到文案本身进行调整和优化。
