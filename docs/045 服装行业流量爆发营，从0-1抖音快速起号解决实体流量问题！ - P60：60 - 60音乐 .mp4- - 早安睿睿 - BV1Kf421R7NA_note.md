## 添加背景音乐 🎶

![](img/279e72440e24905134658e0ddfc9af8f_5.png)

首先，将你想要使用的视频素材添加到剪辑项目的时间线中。

![](img/279e72440e24905134658e0ddfc9af8f_7.png)

完成素材导入后，点击界面上的 **“音频”** 按钮。

此时会出现二级工具栏，请点击其中的 **“音乐”** 选项。

软件的音乐库中提供了大量自带的音乐。使用前需要先点击下载，下载完成后即可使用。

![](img/279e72440e24905134658e0ddfc9af8f_9.png)

以下是音乐库中的几个主要分类：

*   **收藏**：这里存放的是你在软件音乐库中手动收藏的音乐。
*   **抖音收藏**：登录账号后，会同步你抖音账号中收藏的音乐。
*   **导入音乐**：可以从其他平台（如网易云音乐、抖音）复制音乐链接，粘贴到这里下载使用。**注意：使用外部音乐需留意版权问题，推荐使用无版权音乐。**
*   **提取音乐**：此功能可以直接从已有视频中提取音频，作为新视频的背景音乐。

![](img/279e72440e24905134658e0ddfc9af8f_11.png)

建议平时多收藏一些熟悉的音乐，以便在后期剪辑时快速调用，提高效率。

## 添加音效与录音 🎤

![](img/279e72440e24905134658e0ddfc9af8f_13.png)

除了背景音乐，音效也能极大增强视频的表现力。

![](img/279e72440e24905134658e0ddfc9af8f_15.png)

点击 **“音效”**，软件内置了丰富的音效资源，基本可以满足日常需求。添加方法与添加音乐类似：下载后即可使用。

最后是录音功能。点击 **“录音”**，按住红色按钮开始录制，松开手即结束。确认后，录音文件会出现在视频轨道的下方。

## 音频剪辑基础 ✂️

学习完如何添加音频后，现在学习如何对音频进行精细剪辑。

选中时间线上的音频片段，点击 **“音频”** 编辑选项，可以进行以下操作：

*   **音量**：默认音量为100，最高可支持10倍调整。`音量 = 基础值 × 调整系数`
*   **淡化**：为音频添加淡入或淡出效果，让开始和结束更自然平滑。
*   **分割**：将一段音频分割成两段。`分割点 = 时间线光标位置`
*   **踩点**：在音频上添加标记点，用于卡点视频制作，下文会详细讲解。
*   **删除**：直接删除选中的音频片段。
*   **变速**：对音频进行加速或减速处理。
*   **复制**：复制当前选中的音频片段。

## 制作卡点视频 🎬

下面，我们利用刚学的知识，带领大家制作一个简单的卡点视频。

首先点击 **“添加音频”**。对于新手，建议选择节奏舒缓、节拍明显的音乐，这样更容易找准卡点位置。

我们使用一段预先选好的音乐。导入音乐后，可以手动在音频波形图的波峰位置添加标记点。如果点错了，可以将光标移动到点上，然后点击删除。

对于新手，更推荐使用 **“自动踩点”** 功能。软件通常提供两种自动踩点模式（如“节拍一”、“节拍二”），可以都试听一遍，选择节奏点最符合你素材数量的那一个。

选择“节拍一”作为示例。踩点结束后，现在来调整视频素材的长度。

卡点的核心在于，让每个视频镜头的切换刚好发生在音乐节拍点上。拖动视频片段的尾部，它会自动吸附到附近的标记点上，操作非常方便。

按照此方法，调整后面所有素材的长度，使其与节拍点对齐。

当所有素材都剪辑完毕后，音乐可能还有剩余。我们可以将后面多余的音频分割并删除。最后，在这段音频的结尾添加一个“淡出”效果，让视频结束得更舒缓。
