# 微信视频号运营从0~1全攻略：P16：视频号保证金、带货权限与直播权限解读 🧾

在本节课中，我们将详细解读视频号的带货身份、带货权限以及直播权限的具体规则与要求。这些是开启视频号电商功能的基础，理解它们能帮助你更好地规划运营路径。

## 带货身份选择

上一节我们介绍了创作者中心的入口，本节我们来看看具体的功能设置。在创作者中心，点击右上角的个人头像，即可查看并设置你的带货身份。

以下是两种主要的带货身份：
*   **达人**：通常指个人创作者，通过分享内容来推广商品。
*   **商家**：指拥有营业执照、开设了视频号小店的经营主体。

这个身份初期可以选择，后期如果发现选择错误，也可以进行修改。

## 带货权限详解

了解身份后，我们来看看具体的带货权限。在相关页面中，你会看到三个带货权限板块：视频带货、橱窗带货和直播带货。它们分别代表不同的商品销售方式。

### 1. 视频带货
视频带货指的是在发布的短视频中挂载购物车，类似于抖音的“小黄车”功能。用户观看视频时，可以看到一个橙色的购物车图标，点击即可购买视频中推荐的商品。

**核心要求**：
*   视频内容必须为**原创**。
*   账号需要拥有**1000个有效粉丝**。

对于从事内容搬运的账号，通常很难开通此权限。即便开通，一旦发生违规，此权限也可能被永久关闭。届时，虽然不能在视频中挂车，但仍可通过橱窗销售商品。

![](img/9a743a2154844204c65567b4c0287974_1.png)

![](img/9a743a2154844204c65567b4c0287974_3.png)

### 2. 橱窗带货
橱窗带货是一个相对稳定的商品展示与销售入口。无论是否开通视频带货或直播带货权限，只要开通了商品橱窗，用户就可以通过你的主页进入橱窗浏览和购买商品。这是最基础、最常用的带货方式。

![](img/9a743a2154844204c65567b4c0287974_4.png)

![](img/9a743a2154844204c65567b4c0287974_5.png)

### 3. 直播带货
直播带货指在直播过程中销售商品。这是目前转化效率较高的形式，但其开通条件也更为严格。

![](img/9a743a2154844204c65567b4c0287974_7.png)

![](img/9a743a2154844204c65567b4c0287974_9.png)

**核心条件**：
*   **身份实名限制**：目前，一个自然人仅能实名认证**2个**视频号（过往可认证5个）。这与你实名多少个微信无关，最终以视频号实名认证为准。
*   **粉丝数要求**：若以“达人”身份开播，需要在直播间挂载商品，必须满足 **1000个有效粉丝** 的要求。未达此标准，直播间无法添加商品。
    *   代码示例：检查直播权限的函数逻辑可能类似于 `if (fansCount >= 1000) { enableAddGoods(); }`。
*   **替代方案**：如果粉丝数不足1000，可以通过开通**视频号小店**（需营业执照）来获得直播带货权限。这是绕过粉丝门槛、直接以商家身份开播的有效方法。

![](img/9a743a2154844204c65567b4c0287974_11.png)

## 权限实操对比

![](img/9a743a2154844204c65567b4c0287974_13.png)

为了更直观地理解，以下是不同情况下的权限状态对比：

| 权限状态 | 视频带货 (挂车) | 橱窗带货 | 直播带货 (挂商品) |
| :--- | :--- | :--- | :--- |
| **未达1000粉 (达人)** | ❌ 不可用 | ✅ 可用 | ❌ 不可用 |
| **已达1000粉 (达人)** | ✅ 可用 (需原创) | ✅ 可用 | ✅ 可用 |
| **已开通视频号小店** | ✅ 可用 (需原创) | ✅ 可用 | ✅ 可用 (无视粉丝数) |

![](img/9a743a2154844204c65567b4c0287974_15.png)

![](img/9a743a2154844204c65567b4c0287974_17.png)

如图所示，开通视频号小店是获取完整带货权限、尤其是直播权限的关键路径。关于如何办理营业执照及开通视频号小店的具体流程，我们将在后续课程中详细讲解。

## 总结

本节课我们一起学习了视频号电商功能的核心权限规则。
*   带货身份分为**达人**和**商家**，可后期修改。
*   **视频带货**要求高（原创+1000粉），且不稳定。
*   **橱窗带货**是基础且稳定的销售渠道。
*   **直播带货**权限受实名认证数量（最多2个）和粉丝数（1000粉）限制，或可通过开通**视频号小店**获取。

![](img/9a743a2154844204c65567b4c0287974_19.png)

理解这些规则，能帮助你在运营初期明确目标，是选择先积累粉丝，还是直接通过开店进入直播赛道。
