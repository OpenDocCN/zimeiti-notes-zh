## 推荐机制的核心：社交链驱动

![](img/a58c7f1ac2a9326db4bb4a5fd03d65dd_6.png)

上一节我们介绍了视频号的平台属性，本节中我们来看看其独特的推荐机制。视频号的推荐机制与抖音不同，其核心是基于**社交链**进行“强启动”。

**社交链**是指：当用户A点赞了某个视频，该视频会出现在用户A所有好友的推荐流中。这与抖音不同，在抖音中，用户可以选择关闭“朋友点赞”可见，点赞行为更偏向个人隐私。而在视频号，点赞是一个公开的社交行为，能直接触发对好友圈的推荐。

我们可以用一个公式来理解其初始传播逻辑：
```
用户A（观看并点赞） -> 用户A的所有好友（收到推荐） -> 好友中产生新的点赞 -> 新点赞用户的好友（收到推荐）...
```
这个过程会像涟漪一样，从初始点（用户A）开始，通过社交关系一层层向外扩散。

## 流量传播的两级模型

为了更清晰地理解流量如何蔓延，我们可以将流量分为一级流量和二级流量。

*   **一级流量**：指你的现有粉丝或初始观看用户。当你发布内容后，系统会首先将内容推送给这部分用户。
*   **二级流量**：指通过一级流量用户的社交行为（点赞、转发）所触达的新用户，即“朋友的朋友”。

以下是流量扩散的完整路径：
1.  内容发布后，推送给**一级流量**（你的粉丝）。
2.  一级流量中部分用户产生**点赞/转发**。
3.  他们的**二级流量**（其好友）会看到该视频。
4.  二级流量中若再产生互动，则会继续辐射至三级流量（好友的好友），如此循环。

当内容通过社交链传播，其互动数据（点赞、转发、完播率等）达到系统设定的某个**阈值**后，平台会判定该内容为优质内容，并将其放入更大的**公域流量池**，推荐给无数陌生人。至此，内容便完成了从“私域社交启动”到“公域爆发”的关键一跃。

## 驱动互动的关键：情感共鸣

理解了流量如何扩散后，下一个核心问题是：如何促使观众点赞，从而启动社交链？平台判断内容能否进入公域池，主要依据用户的反馈数据，其权重顺序大致为：
1.  **点赞/转发率**
2.  **完播率**
3.  **评论/关注率**

其中，**点赞是启动社交链的阀门**。根据运营经验，最能促进高点赞和高互动的因素是 **“动情”**。

“情”包含多种能引发观众强烈共鸣的情感，例如：
*   **感动之情**：如公益善举、亲情母爱。
*   **骄傲之情**：如家国成就、文化自信。
*   **共鸣之情**：如女性成长、职场心声，让特定群体感到“说出了我的心里话”。

情感内容最容易突破用户心理防线，促使他们做出点赞、评论等互动行为。这也解释了为何视频号平台的头部爆款账号中，情感类内容占比极高。

## 平台的流量分级倾斜

视频号平台会对不同性质的账号给予不同程度的流量扶持。我们可以将流量倾斜分为三个等级：

以下是各级别流量倾斜的对象及特点：
*   **一级倾斜（最大扶持）**：**头部大号**（如顶级情感号）。平台会直接向其注入大量公域流量，助其持续保持在榜单前列。
*   **二级倾斜（中度扶持）**：**MCN机构旗下账号**。平台会提供一定的初始流量包或倾斜政策，后续根据内容反馈数据决定是否追加流量。
*   **三级倾斜（竞争性扶持）**：**素人白号**。对于不属于前两类的普通账号，平台几乎无初始扶持。流量获取完全依赖于内容本身的**高互动、高反馈数据**。只有数据达标，才有机会进入更大的流量池。

![](img/a58c7f1ac2a9326db4bb4a5fd03d65dd_11.png)

对于大多数新手而言，我们处于第三级，这意味着我们必须依靠优质内容和运营策略来撬动流量。

![](img/a58c7f1ac2a9326db4bb4a5fd03d65dd_12.png)

## 冷启动策略：从公域获取初始流量

一个核心问题随之产生：对于一个没有粉丝基础的新号，如何完成最初的“冷启动”？

解决方案是：**从外部公域平台获取初始流量**。不要只局限于在视频号内部发内容。可以同步运营抖音、小红书等公域平台，将这些平台吸引来的用户引导至你的视频号（例如，在个性签名中注明）。这批外部引入的用户，就成为了你在视频号内的**初始一级流量**。利用这批种子用户，努力提升内容的互动数据，从而启动视频号内部的社交推荐机制，并最终冲击公域流量池。

## 总结

![](img/a58c7f1ac2a9326db4bb4a5fd03d65dd_14.png)

![](img/a58c7f1ac2a9326db4bb4a5fd03d65dd_15.png)

本节课我们一起学习了视频号推荐机制的核心要点：
1.  **机制核心**：基于**社交链**启动，点赞行为是触发推荐的关键。
2.  **流量路径**：遵循 **“一级流量 -> 社交互动 -> 辐射二级流量 -> 数据达标 -> 进入公域池”** 的模型。
3.  **内容关键**：**情感共鸣**是驱动高点赞、高互动的有效手段。
4.  **平台规则**：流量分配有倾斜，素人账号需靠**数据**取胜。
5.  **起号策略**：善用外部公域平台为视频号进行**冷启动**。
