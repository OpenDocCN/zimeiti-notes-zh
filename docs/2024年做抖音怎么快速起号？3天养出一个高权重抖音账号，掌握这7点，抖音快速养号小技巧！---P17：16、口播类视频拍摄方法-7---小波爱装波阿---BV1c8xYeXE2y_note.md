# 抖音运营教程：P17：口播类视频拍摄方法

## 概述
在本节课中，我们将学习口播类视频的具体拍摄方法。掌握正确的拍摄技巧，可以有效提升视频的观感与专业性，从而吸引更多目标观众。

## 拍摄的核心要点
拍摄环节需要注意以下几点：构图比例、主题突出、事物清晰、光线色彩和内容连贯。这些要点是制作优质视频的基础。

为了便于理解，我们将以两种最常见的视频类型为例进行讲解：口播类视频和日常生活类视频。这两种类型几乎涵盖了所有拍摄场景。

上一节我们介绍了拍摄的通用原则，本节中我们来看看第一种类型——口播视频的具体拍摄方法。

---

## 口播视频拍摄详解
首先，我们分析一个成功案例。学员“种小草”是一位情感口播博主，她的一条关于婆媳关系的视频播放量超过8000万，并借此涨粉30万。

她的成功关键在于内容精准吸引了目标群体。例如，她的视频主题明确针对已婚女性群体，这类观众更容易与内容产生共鸣，因此转化和涨粉效率极高。

![](img/594ba350a755f1834034fcfb6d66815d_3.png)

### 构图与画幅
抖音是九比十六的竖屏平台。在拍摄时，你需要统一视频的画幅比例。

以下是关于画幅选择的要点：
*   **统一性**：选择横屏（如四比三）或竖屏后，应保持所有视频风格一致，避免混用。
*   **人物构图**：人物主体应占据画面约70%的面积，位于画面中央。画面四周需适当留白，使观感舒适清晰。
*   **避免错误**：不应将镜头过于贴近脸部拍摄。

### 人物与表现
口播视频的核心是内容传递，而非主播的容貌。

以下是关于人物表现的要点：
*   **内容为王**：观众关注的是你传达的知识、观点或情感。
*   **形象得体**：主播只需看起来清晰、体面即可。如果对自己的外貌不自信，可以开启适度的美颜滤镜。
*   **工具辅助**：为保证声音清晰，建议使用手机收声麦克风，这是一个成本低廉但效果显著的设备。

### 内容与录制
对于口播创作者，记忆台词可能是一个挑战。

![](img/594ba350a755f1834034fcfb6d66815d_5.png)

以下是关于内容录制的要点：
*   **提词工具**：无需背诵全文。可以借助提词器应用，实现边看边录，大幅降低录制难度。
*   **环境选择**：拍摄地点不限（室内或户外均可），但需确保背景整洁、美观，不分散观众注意力。

## 总结
本节课我们一起学习了口播类视频的拍摄方法。我们明确了构图统一、人物得体、声音清晰、善用工具（如美颜、麦克风、提词器）等核心技巧，并通过案例了解到精准定位目标观众群体是内容成功的关键。掌握这些方法，你也能制作出专业、吸引人的口播视频。
