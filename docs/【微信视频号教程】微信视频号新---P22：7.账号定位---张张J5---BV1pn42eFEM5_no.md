# 微信视频号教程：P22：7.账号定位

在本节课中，我们将要学习如何解决视频号运营初期的两个核心问题：快速获取100个初始粉丝以及获取高质量的原创视频素材。我们将通过具体的方法和资源支持，帮助你顺利启动账号。

上一节我们介绍了账号定位的基础概念，本节中我们来看看如何解决启动期的实际困难。

## 解决初始粉丝问题

![](img/be8f7c4b73b4e16b3d84bfe5d0aa2829_3.png)

首先，我们需要解决获取100个初始粉丝的问题。这是账号启动的第一步。

以下是获取初始粉丝的具体步骤：
1.  **隐藏过往作品**：在开始新定位运营前，建议将账号之前的作品隐藏。
2.  **保持发布频率**：素材准备无需着急，保持每天发布一条视频的节奏。

所以，我们的策略是优先解决100个粉丝的目标，再逐步解决素材问题，一步步来。

## 解决原创视频素材问题

接下来解决原创视频素材的问题。很多同学面临的困境是不会拍摄、不会剪辑，缺乏专业素材。

为此，我们为每位学员提供一项重磅福利：**领取30个原创视频素材**。

这些素材来自我们投入近200万元搭建的独家素材库，由专业团队精心剪辑制作，涵盖搞笑娱乐、生活妙招等多种类型，质量很高。

![](img/be8f7c4b73b4e16b3d84bfe5d0aa2829_9.png)

以下是领取素材的三个前提条件：
1.  **课堂互动**：在直播间公屏上打出“想”字。
2.  **积极互动**：在课程中保持积极的互动状态。
3.  **完成听课**：我们的后台会统计听课时长与互动数据。

满足以上条件后，即可按以下步骤领取：
1.  找到向你发送听课链接的助理老师。
2.  向助理老师说明是“张老师”让来领取素材。

这种方式既是对你学习态度的检验，也能确保资源给到真正需要的学员。
