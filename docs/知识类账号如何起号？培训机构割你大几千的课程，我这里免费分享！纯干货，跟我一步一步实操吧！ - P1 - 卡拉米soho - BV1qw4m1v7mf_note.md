# 知识类账号起号教程：P1：从零到一制作知识类视频

在本节课中，我们将学习如何从零开始制作一个知识类短视频。整个过程分为四个核心步骤：寻找对标视频、利用AI仿写文案、将文案转为语音、以及使用免费素材进行剪辑。我们将使用完全免费的工具，一步步完成实操。

## 概述：知识类视频制作的四个步骤

知识类视频的制作流程可以总结为以下四个部分：
1.  寻找对标视频。
2.  使用AI工具仿写文案。
3.  将文案转换为外语语音。
4.  使用免费素材和工具完成视频剪辑。

接下来，我们将逐一进行详细操作。

## 第二步：提取视频文案

![](img/fc79dec7704b5d7ec1623cd196f31330_23.png)

## 第三步：使用AI仿写与生成语音

![](img/fc79dec7704b5d7ec1623cd196f31330_31.png)

我们已经获得了原始文案，接下来需要利用AI工具进行仿写，并生成新的语音。

复制字幕文本，打开ChatGPT。

![](img/fc79dec7704b5d7ec1623cd196f31330_27.png)

![](img/fc79dec7704b5d7ec1623cd196f31330_33.png)

首先，让ChatGPT翻译原文以便理解。然后，输入指令让ChatGPT根据原文风格和主题进行仿写。
**指令示例**：`请根据下面这篇文章的风格和主题，帮我仿写一段类似的新文案。`

ChatGPT生成多个版本后，选择其中一个（例如第五个）并复制。

打开一个免费的文本转语音（TTS）工具网站，例如TTSMaker。将仿写好的文案粘贴进去。

在工具中选择一个合适的外语发音人声，并试听效果。
**代码示例（伪代码）**：`选择语音 -> 输入文本 -> 合成语音`

![](img/fc79dec7704b5d7ec1623cd196f31330_35.png)

完成设置后，生成语音文件并下载。

![](img/fc79dec7704b5d7ec1623cd196f31330_37.png)

![](img/fc79dec7704b5d7ec1623cd196f31330_29.png)

## 第四步：剪辑与合成最终视频

现在，我们拥有了全新的文案和语音，本节将学习如何搭配免费素材完成最终剪辑。

再次打开剪映，将下载好的语音文件拖入音频轨道。

![](img/fc79dec7704b5d7ec1623cd196f31330_31.png)

试听音频，确认无误。

![](img/fc79dec7704b5d7ec1623cd196f31330_33.png)

分析文案主题（例如“正能量”、“哲学思考”），根据主题寻找匹配的免费视频素材。可以使用Pexels、Pixabay等无版权素材网站。

在素材网站搜索关键词（如“跑步”、“健身”），下载合适的视频片段。

![](img/fc79dec7704b5d7ec1623cd196f31330_35.png)

![](img/fc79dec7704b5d7ec1623cd196f31330_37.png)

以下是剪辑的核心操作列表：

![](img/fc79dec7704b5d7ec1623cd196f31330_39.png)

*   **设置比例**：将视频比例设置为9:16（竖屏）。
*   **调整素材**：将横屏素材拉伸适配竖屏比例，并将所有视频片段与音频长度对齐。
*   **统一调色**：选中所有视频片段，在“调节”选项中，使用HSL工具进行颜色统一和优化。
*   **关闭原声**：关闭所有素材视频自带的音频轨道。
*   **添加滤镜**：为视频添加合适的滤镜（如“青橙”、“电影感”），并调整滤镜强度。
*   **添加转场**：在视频片段之间添加转场效果（如“闪黑”、“叠化”）。
*   **添加字幕**：使用“识别字幕”功能为语音生成字幕，调整字体、大小和位置，并可以添加入场动画。
*   **最终导出**：完成所有剪辑后，导出视频。

## 要点总结与注意事项

本节课中我们一起学习了制作知识类视频的完整流程。最后有几点需要特别注意：

1.  **素材版权**：务必使用明确标注为“免费可商用”或无版权的视频、图片素材。
2.  **内容契合度**：挑选的视觉素材必须与文案主题高度契合。例如，正能量文案应搭配积极、明亮的画面。
3.  **用心打磨**：演示过程追求效率，实际操作中应在素材筛选、剪辑细节（如调色、字幕动画、转场）上花费更多心思，提升视频质感。

只要遵循以上步骤和要点，你就能制作出属于自己的知识类视频。
