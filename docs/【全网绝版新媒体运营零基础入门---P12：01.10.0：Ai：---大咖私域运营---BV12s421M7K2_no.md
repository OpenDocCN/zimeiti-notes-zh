## 渠道一：微软 Edge 浏览器 Copilot 🌐

上一节我们介绍了AI对运营的重要性，本节中我们来看看第一个推荐渠道：微软Edge浏览器的Copilot功能。

这是微软基于ChatGPT打造、集成在Windows系统中的功能。它基于4.0模型开发，在智能程度上与官方4.0版本有一定匹配度，是普通用户可用的方式之一。

**使用前提：**
1.  需要登录微软账号（可使用Outlook邮箱注册）。
2.  需要“梯子”（VPN）才能正常访问该服务。

**功能界面与模式：**
Copilot提供“更有创造力”、“平衡”和“精确”三种对话模式。
*   **更有创造力**：AI的回答更具开放性和创意。
*   **平衡**：介于“精确”和“更有创造力”之间，适合一般用途。
*   **精确**：当AI回答过于局限或未达要求时，可尝试此模式以获得更严谨的回复。

**基本操作示例：**
我们可以使用“更有创造力”模式，让它生成一个短视频脚本。
**代码/指令示例：**
```
请帮我写一个关于装修的短视频脚本，时长控制在3分钟左右。
```

**额外优势：**
浏览器版本的Copilot可以分析你当前打开的网页内容，这在右侧工具栏中实现。

**主要缺点：**
尽管Copilot基于4.0模型，但存在一些限制：
1.  **精准度稍逊**：经过微软调教后，在某些方面与官方ChatGPT 4.0存在差距。
2.  **连续对话能力受限**：无法在当前对话中持续深入提问多次，可能需要开启新对话窗口。
3.  **记忆功能有所缺失**：在读取历史对话和基于上下文优化回答方面不如官方版本强大。

**获取方式：**
如果你的Edge浏览器没有Copilot图标，可以尝试下载最新版的Edge Dev（测试版）。在浏览器设置中点击“关于Microsoft Edge”，即可检查并更新至最新版本。

## 渠道二：第三方集成平台与共享账号 🛠️

了解了浏览器内置方案后，我们来看看一些由开发者搭建的第三方平台，它们集成了ChatGPT的API，方便国内用户使用。

以下是几个可用的平台推荐：

**1. 灵感（网站A）**
该平台集成了多种AI模型，包括GPT-3.5、GPT-4.0（含联网版）以及国内主流模型（如讯飞星火、文心一言等）。

**主要特点：**
*   **模型丰富**：集成市面上多数主流AI模型。
*   **GPT-4有限免费**：新注册用户可在一定次数或时间内免费使用GPT-4。
*   **无需梯子**：在国内可直接访问。
*   **缺点**：生成速度可能较慢。

**2. Wee（网站B）**
该平台提供GPT-3.5和GPT-4.0模型，响应速度较快。

**主要特点：**
*   **响应速度快**：达到正常可用水准。
*   **附带功能**：具备语音回答等功能。
*   **缺点**：
    *   需要梯子。
    *   免费额度（初始1万token）通常仅适用于GPT-3.5模型，尝试GPT-4会提示额度不足。
    *   连续对话能力可能一般。

**3. ChatGPT共享账号平台**
这是一个网络共享平台，汇集了用户分享的ChatGPT账号。

![](img/7248c246cc1ab5be19449978ef67174d_5.png)

**平台使用逻辑：**
*   **绿色**：账号空闲。
*   **橙色**：账号近期被使用过。
*   **红色**：账号可能正在被使用或使用频繁。
*   **金色（Plus）**：代表付费的GPT-4账号，可用但稀缺。

![](img/7248c246cc1ab5be19449978ef67174d_7.png)

**注意事项：**
*   共享的GPT-4账号有使用限制（例如每3小时约30条消息），可能遇到“使用上限”提示。
*   GPT-3.5账号通常较多且免费可用。
*   **此方法也需要梯子。**

## 渠道三：国内AI大模型替代方案 🇨🇳

如果无法使用需要梯子的渠道，我们可以转向国内公司出品的一些AI大模型。它们在国内可直接访问，但并非ChatGPT。

以下是几个推荐：

**1. 百度文心一言**
百度的AI对话模型，有免费版和付费的“4.0”版本。对于新媒体运营的辅助工作（如获取灵感、生成基础文案）基本够用。专业深度可能因需求而异。

**2. 字节跳动豆包AI**
字节跳动出品的AI助手。同样可以完成脚本生成等任务，会提供景别、画面内容、台词等结构化信息。

**3. 其他国内模型**
如阿里的通义千问、昆仑万维的天工大模型等，都可以作为备选工具。

**使用建议：**
你可以从这些国内模型中挑选一个用起来最顺手、最能提升你工作效率的进行熟练使用。结合我们之前课程教过的“如何训练你的AI模型”等方法，让其更好地为你服务。

## 官方ChatGPT 4.0效果对比 ⚖️

最后，我们对比一下官方ChatGPT 4.0的生成效果。使用相同的指令：

**代码/指令示例：**
```
请帮我写一份关于装修的短视频脚本，文案要求镜头丰富，语言吸引人。
```

官方GPT-4生成的脚本在场景构建、叙事节奏和镜头语言的丰富性上，通常展现出更强的创造力和逻辑性。这背后有多方面原因。对于我们运营人员而言，原则是：**什么工具能提升效率，就用什么工具。**
