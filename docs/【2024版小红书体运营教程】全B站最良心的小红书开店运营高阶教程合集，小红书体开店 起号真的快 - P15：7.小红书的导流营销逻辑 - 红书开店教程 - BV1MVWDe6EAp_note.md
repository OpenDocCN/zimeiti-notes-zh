## 平台核心角色与内容

![](img/9cd4b63dc3c648b52210962949f39836_28.png)

![](img/9cd4b63dc3c648b52210962949f39836_30.png)

![](img/9cd4b63dc3c648b52210962949f39836_32.png)

![](img/9cd4b63dc3c648b52210962949f39836_34.png)

小红书平台上的用户被称为“小红薯”，这既包括个人用户，也包括企业用户。用户通过创作和分享内容来互动，这些内容在平台上被称为“笔记”。

![](img/9cd4b63dc3c648b52210962949f39836_36.png)

![](img/9cd4b63dc3c648b52210962949f39836_38.png)

![](img/9cd4b63dc3c648b52210962949f39836_40.png)

![](img/9cd4b63dc3c648b52210962949f39836_42.png)

“笔记”这个名称源于平台的初衷：希望用户能持续记录和分享关于商品或购物的真实体验。每当用户购买到好物或体验到优质商品时，就可以通过发布笔记来进行记录和分享。

![](img/9cd4b63dc3c648b52210962949f39836_46.png)

## 个人小红书的导流路径

个人小红书用户可以通过分享笔记来启动导流过程。以下是典型的操作流程：

![](img/9cd4b63dc3c648b52210962949f39836_58.png)

![](img/9cd4b63dc3c648b52210962949f39836_60.png)

![](img/9cd4b63dc3c648b52210962949f39836_62.png)

![](img/9cd4b63dc3c648b52210962949f39836_64.png)

1.  **内容创作与分享**：用户针对某个商家的具体产品、单品或系列商品进行分享。例如，购买了一件显白、显瘦、面料舒适的上衣后，发布使用体验笔记。
2.  **吸引潜在消费者**：其他用户看到这篇高质量的分享笔记后，可能对该商品产生兴趣。
3.  **流量转化**：感兴趣的读者会询问购买渠道，进而被引导至商家的商品页面、店铺或商城。
4.  **完成销售**：最终在商家处产生购买行为，完成从内容到销售的闭环。

![](img/9cd4b63dc3c648b52210962949f39836_66.png)

![](img/9cd4b63dc3c648b52210962949f39836_68.png)

![](img/9cd4b63dc3c648b52210962949f39836_70.png)

![](img/9cd4b63dc3c648b52210962949f39836_72.png)

此外，个人创作者还有另一种变现路径：当你的笔记内容质量很高，且领域垂直（如美妆），可能会被相关领域的商家主动关注。商家会寻求合作，例如进行产品推广，从而实现个人创作的变现。

企业类小红书账号是另一种常见且重要的运营身份。其导流逻辑如下：

![](img/9cd4b63dc3c648b52210962949f39836_88.png)

![](img/9cd4b63dc3c648b52210962949f39836_90.png)

![](img/9cd4b63dc3c648b52210962949f39836_92.png)

![](img/9cd4b63dc3c648b52210962949f39836_94.png)

1.  **官方内容输出**：企业围绕自家商品创作笔记，进行推荐和介绍（即“安利”）。
2.  **价值点释放**：通过这些内容，向目标用户传递产品的利好信息、优点与核心价值。
3.  **激发兴趣**：目标消费者看到内容后，对产品产生兴趣或购买欲望。
4.  **引导转化**：企业将产生的兴趣流量，引导至自己的官方店铺或商城。
5.  **实现销售**：用户在店铺内完成购买，为企业带来营收。
