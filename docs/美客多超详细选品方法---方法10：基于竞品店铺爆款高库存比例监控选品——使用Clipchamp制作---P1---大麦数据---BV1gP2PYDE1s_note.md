# 美客多选品教程：方法10：基于竞品店铺爆款高库存比例监控选品

## 概述
在本节课中，我们将学习一种基于数据监控的选品方法。核心思路是：通过监控竞争对手店铺中爆款商品的库存与销量比例，发现市场机会，从而指导我们进行选品和补货决策。

![](img/bf093dfd24fdfd091910dc3dd6990268_0.png)

---

## 核心概念与原理
上一节我们介绍了选品方法的多样性，本节中我们来看看如何利用竞品数据找到市场缺口。

这种方法的原理基于一个简单的市场逻辑：当一个热销商品出现库存不足时（即库存量小于月销量），意味着市场需求未被完全满足，存在市场机会。我们可以通过监控这一指标，快速发现并填补市场空白。

其核心判断公式为：
**库存月销量比 = 库存数量 / 月销量**
当 **库存月销量比 < 1** 时，表明该商品库存紧张，是潜在的切入机会。

---

## 操作步骤详解
理解了核心原理后，接下来我们看看具体的操作步骤。以下是利用“大麦数据”工具执行此方法的完整流程。

1.  **定位竞品店铺**
    首先，你需要找到与自身供应链能力、产品定位相似的竞争对手店铺作为观察对象。

2.  **分析店铺爆款数据**
    进入“大麦数据”平台的“竞品店铺分析”或“店铺列表分析”功能模块。在此模块中，可以查看目标店铺内所有商品的详细数据。

3.  **监控关键指标**
    在商品数据列表中，重点关注“库存数量”与“月销量”这两个字段，并计算或直接查看其比例。

![](img/bf093dfd24fdfd091910dc3dd6990268_2.png)

4.  **筛选机会商品**
    寻找那些 **库存月销量比 < 1** 的商品。这些商品就是正在热销但可能面临断货风险的“爆款”。

![](img/bf093dfd24fdfd091910dc3dd6990268_3.png)

---

## 机会分析与执行策略
当我们筛选出目标商品后，并不意味着可以盲目跟卖。以下是需要进行的后续分析和行动步骤。

![](img/bf093dfd24fdfd091910dc3dd6990268_4.png)

*   **商品分析**：深入研究该爆款商品的标题、主图、详情页和用户评价，理解其吸引消费者的核心卖点。
*   **市场调研**：参考该商品所在品类的价格分布区间，结合自身品牌和产品成本，制定一个有竞争力的定价策略，与竞品保持适当价差。
*   **行动决策**：如果该商品与你的供应链匹配，可以考虑快速开发相似商品并上架推广，以承接因竞品库存不足而溢出的市场需求。

![](img/bf093dfd24fdfd091910dc3dd6990268_5.png)

## 总结
本节课中，我们一起学习了“基于竞品店铺爆款高库存比例监控”的选品方法。我们掌握了其核心公式 **库存月销量比 < 1** 作为机会信号，并了解了从数据监控、机会筛选到最终执行上架的全流程。这种方法能帮助你敏锐地捕捉市场动态，快速响应，是一种高效的数据驱动型选品策略。
