# 小红书运营教程：P29：28、小红书笔记引流：平台流量结构 📊

![](img/09a5fb39a513d193009898f13c2cdf0e_1.png)

在本节课中，我们将要学习小红书平台的流量结构。了解流量从哪里来，以及不同流量渠道的特点，是制定有效运营策略的基础。我们将重点分析发现页、关注页和搜索页这三个核心流量来源，并探讨如何根据它们的特点来优化我们的笔记发布策略。

上一节我们介绍了笔记制作的基础，本节中我们来看看笔记发布后，平台是如何分配流量的。

## 什么是流量结构？

流量结构指的是小红书平台整体流量的分布和来源。理解流量结构，能让我们明白不同类型的笔记投放到不同的渠道，会获得怎样的效果。这对于规划内容方向和推广策略至关重要。

## 小红书的核心流量渠道

小红书整体的流量结构主要分为三个部分：**发现页**、**关注页**和**搜索页**。打开小红书App，主页面通常显示“发现”和“关注”，旁边的放大镜图标则代表“搜索”。

关于“本地”页面：它未在上述三个主页面中重点标注，因为它更适用于特定场景。“本地”页面显示地域性内容，适合用于寻找素材或分析本地用户喜好。例如，如果你是户外或美食博主，介绍地方特色，可以使用它。

**但请注意**：除非你是特定类型的博主（如本地美食探店），否则在推广商品或进行引流时，应尽量避免过度依赖“本地”引流。因为基于地域的推送会打乱账号的整体标签。平台会优先向附近用户推送发布时间短、互动好的内容。如果你的目标是面向全国市场，专注于“本地”反而会限制发展。

因此，我们运营的核心应聚焦于以下三个主要流量渠道。

### 1. 发现页：流量的主阵地 🔥

![](img/09a5fb39a513d193009898f13c2cdf0e_3.png)

发现页是整个小红书**最重要**的流量来源，约占平台总流量的 **70%**。它也是平台进行流量层级测试的主要场所。一篇笔记能获得多大规模的曝光，很大程度上取决于它在发现页获得的用户反馈（如点赞、评论、收藏）。

**核心机制**：当你发布一篇笔记后，平台会根据你的账号权重，给予一个初始的基础曝光量（例如1000或2000）。这**初始曝光量的100%** 几乎都会投放在发现页。后续能否获得更大流量，则取决于笔记在发现页的互动数据。



### 2. 搜索页：增长最快的精准流量 🔍

搜索页（即通过放大镜搜索）的流量占比仅次于发现页，约为 **20%**。这是小红书平台**增长最快**的流量板块。

**为什么重要**：小红书的月活跃用户已非常庞大，新增用户速度会放缓。因此，平台内用户**主动搜索**的行为变得尤为关键。搜索流量代表了用户的明确意图，转化率极高。据统计，小红书平台内直接成交的订单，有约 **70%** 来源于搜索页。

**运营提示**：搜索页的流量虽然优质，但新账号初期很难直接获取。因为它依赖于商品关键词、笔记权重、历史成交和评价等综合数据。这部分流量是我们账号和店铺成长到一定阶段后，需要重点去“抢占”的市场。



### 3. 关注页：最精准的私域流量 ❤️

关注页的流量占比最小，大约只有 **10%** 甚至更低（约7%）。虽然量小，但这部分用户是**最精准**的。他们已经关注了你，对你的内容感兴趣，因此互动和转化相对容易。

**核心特点**：关注页的流量**不影响流量的层级测试数据**。也就是说，即使你有1万粉丝，笔记发布后全部通过关注页阅读，达到了1万阅读量，这篇笔记也未必能进入更大的公域流量池（如发现页）。因为平台判断笔记是否“爆”，主要依据其在**发现页**的初始曝光和互动率。

**运营提示**：关注页的价值在于维护核心用户、增强粘性，并可能引导至其他私域平台（如微信群）。但运营前期，不应将主要精力放在追求关注页流量上。正确的顺序是：先做好发现页内容，再优化搜索排名，最后经营好关注页的粉丝。

## 总结与下节预告

本节课我们一起学习了小红书平台的流量结构。我们了解到：
*   **发现页**是流量核心（70%），决定笔记的爆发潜力。
*   **搜索页**是精准流量和成交高地（20%），需后期重点优化。
*   **关注页**是精准私域流量（约10%），用于维护核心用户。

理解这个结构，能帮助我们有条不紊地规划运营：前期聚焦发现页，积累权重和数据；中期优化搜索关键词，承接精准流量；长期经营关注页，培养忠实用户。
