# 抖音短视频AI创作教程：P8：5.视频剪辑找素材 🎬

在本节课中，我们将要学习视频剪辑类内容创作的第一步：如何高效、合法地寻找素材。这是制作优质视频的基础，也是避免违规的关键。

上一节我们介绍了视频创作的宏观思路，本节中我们来看看具体如何获取剪辑所需的原始材料。

## 素材来源平台

![](img/c6ae16531321017fff47c6e9fc20e581_3.png)

视频剪辑内容涵盖广泛，例如电影剪辑、电视剧剪辑或综艺节目剪辑。若要进行此类创作，首要任务是寻找素材。寻找素材可以从以下几个主要平台入手。

以下是三个主要的素材来源类别：

1.  **自媒体平台**：众多视频自媒体平台是素材的宝库。例如，下图所示的30多个平台均可作为素材来源。这些平台内容多样，是寻找素材的起点。
    
    ![](img/c6ae16531321017fff47c6e9fc20e581_1.png)
    
    ![](img/c6ae16531321017fff47c6e9fc20e581_3.png)

2.  **专业影视/素材网站**：专门提供影视内容或各类素材的网站是另一个重要来源。
    *   **影视类网站**：例如豆瓣电影等，这类网站通常提供大量影视素材。本人整理了近50个无版权要求的影视网站，可供使用。
    *   **综合类素材网站**：若不想局限于影视，还可寻找动漫、综艺、搞笑、旅游、美食等类型的素材。本人也整理了超过60个此类网站，涵盖图片、音效、各国视频等丰富资源。

3.  **影视推广平台**：这类平台（常与抖音、快手等短视频平台合作）的特殊之处在于，使用其提供的影视素材创作视频，可以依靠播放量直接获得收益，而不仅仅是用于涨粉。这一点后续会重点讲解。

## 关键注意事项：正确取材方式

在寻找素材的过程中，有一个至关重要的原则必须遵守：**切勿直接下载并搬运整个视频**。

平台对搬运行为的审核非常严格，95%的此类操作都无法通过。正确的取材方式是：

**从大量视频中，提取出符合自己视频主题的片段，每个片段的时长建议控制在10秒左右。**

例如，计划制作一个“沈腾搞笑名场面”的合集，那么所有选取的素材片段都应包含沈腾本人，且每个搞笑片段时长约为10秒。这种方式既能保证内容原创度，也能有效规避平台违规风险。

![](img/c6ae16531321017fff47c6e9fc20e581_13.png)

## 从素材到成片：工具与主题

找到素材后，下一步是进行剪辑。最常用的剪辑软件是**剪映**。在开始剪辑前，必须明确视频的**主题**。

例如，确定主题为“沈腾搞笑名场面”。那么，整个视频的所有素材片段都应围绕沈腾的搞笑瞬间展开，确保内容统一。
