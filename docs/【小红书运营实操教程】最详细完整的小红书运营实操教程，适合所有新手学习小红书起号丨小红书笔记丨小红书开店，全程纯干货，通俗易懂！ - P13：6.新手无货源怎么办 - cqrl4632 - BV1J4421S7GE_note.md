## 货源现状分析
上一节我们介绍了店铺的基础设置，本节中我们来看看新手最关心的货源问题。很多新手学员表示自己没有现成的货源和产品。

对于有货源的学员，如果供应链有价格优势，后续会教授如何定价和打造爆款。但观察发现，更多学员属于“无货源”的情况。

## 核心解决方案：代发模式
没有货源并不意味着无法通过小红书开店赚钱。解决方案是从“代发”模式做起。

**代发** 指的是：我们作为卖家，不自己囤货，而是从线上厂家直接拿货。当有顾客下单后，由厂家直接发货给顾客。我们作为中间商，赚取销售价与进货价之间的差价。

用简单的公式表示这个商业模式：
`利润 = 商品销售价 - 商品进货价 - 运费等成本`

## 小红书的高利润优势
在小红书平台，我们能够保证货品达到 **40% 到 60%** 以上的利润。这是因为小红书平台80%的用户是中高端女性，消费能力强，客单价高，因此更容易打造高利润爆款商品。

## 利润实例测算
为了验证利润，我们以一个学生售卖的“儿童拼图”为例进行分析。

1.  **商品售价**：58元。
2.  **寻找货源**：在1688平台通过图片搜索同款商品，找到进货价。我们以较高的进货价27元计算。
3.  **利润计算**：
    *   毛利 = 58 - 27 = 31元
    *   假设运费成本为8元
    *   纯利润 = 31 - 8 = 23元
4.  **利润率计算**：
    `利润率 = 纯利润 / 售价 = 23 / 58 ≈ 39.7%`

该单品利润率接近40%，验证了前述的利润空间。

## 销量与收益展示
利润高，关键还要能卖出去。查看该学生的店铺后台数据，该款儿童拼图累计销量为4529单。

以下是总收益计算：
`总收益 = 单件利润 * 销售数量 = 23元 * 4529单 ≈ 104,167元`

![](img/964fe774f4abd9b8af72146254671676_10.png)

仅这一款单品，就创造了超过10万元的利润。这证明了只要选品正确，即使是无货源的新手，也能在小红书获得可观的收益。如果店铺上架20-30个这样的单品，总收益将更加可观。
