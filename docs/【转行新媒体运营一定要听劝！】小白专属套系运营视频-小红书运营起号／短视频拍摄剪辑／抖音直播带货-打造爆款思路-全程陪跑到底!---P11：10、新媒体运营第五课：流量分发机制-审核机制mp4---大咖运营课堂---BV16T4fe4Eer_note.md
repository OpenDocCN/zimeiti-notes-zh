## 内容审核机制 🔍

上一节我们介绍了内容定位，本节中我们来看看内容发布后的第一道关卡：审核机制。所有平台的内容审核主要分为两大块：**机器审核**和**人工审核**。绝大部分内容的前期审核工作由机器完成。

### 机器审核

![](img/b3712c68b35e69be2712ad52551229b6_9.png)

![](img/b3712c68b35e69be2712ad52551229b6_11.png)

内容发布后，首先由机器审核其合规性。合规性主要包含以下两个方面：

![](img/b3712c68b35e69be2712ad52551229b6_13.png)

以下是机器审核的两个核心检查点：

1.  **敏感词检测**：系统会检测内容（包括图文、口播文案、页面元素）中是否包含平台禁止的敏感词汇。敏感词范围广泛，例如违反广告法的词汇、法律禁止的词汇、政治敏感内容以及平台明确禁止的“黑五类”等赛道内容。一旦触发，内容可能被**直接限流**或**审核不通过**。
    *   **公式表示**：`内容合规性 = 无敏感词 + 无违规信息`
2.  **营销广告检测**：平台会识别硬性广告内容。对于基于兴趣分发（如短视频平台）或种草分享（如图文平台）的生态，直接推广产品、服务、价格等营销信息容易被判定为广告，导致初期流量受损。因此，初期内容应以分享和价值提供为主，而非直接广告。
    *   **代码逻辑示例**：
        ```python
        if content.contains(‘产品价格’, ‘立即购买’, ‘联系我们’):
            risk_level = ‘高’ # 可能被判定为营销广告
        ```

![](img/b3712c68b35e69be2712ad52551229b6_15.png)

![](img/b3712c68b35e69be2712ad52551229b6_17.png)

![](img/b3712c68b35e69be2712ad52551229b6_19.png)

![](img/b3712c68b35e69be2712ad52551229b6_21.png)

通过以上两点后，内容进入**去重检测**阶段。

![](img/b3712c68b35e69be2712ad52551229b6_23.png)

### 去重检测

每个平台都有自己的内容数据库。新内容发布后，系统会提取其特征，并与数据库中的历史内容进行比对。

![](img/b3712c68b35e69be2712ad52551229b6_25.png)

以下是去重检测的运作方式与结果：

*   **运作原理**：系统通过算法提取内容特征（如文本指纹、画面特征），并与数据库比对。
*   **判定结果**：若发现**高度重合或完全抄袭**的内容，系统会判定为抄袭。结果通常是**打回**或**限流**。
*   **应对策略**：创作时应组合他人内容的优秀价值点，但必须进行**调整、优化和重新表述**。可以使用AI工具辅助优化，但需注意合理使用。

![](img/b3712c68b35e69be2712ad52551229b6_27.png)

### 人工审核的介入

![](img/b3712c68b35e69be2712ad52551229b6_29.png)

![](img/b3712c68b35e69be2712ad52551229b6_31.png)

那么，人工审核在何时介入呢？主要有两种情况：

![](img/b3712c68b35e69be2712ad52551229b6_33.png)

![](img/b3712c68b35e69be2712ad52551229b6_35.png)

以下是人工审核介入的两种主要场景：

![](img/b3712c68b35e69be2712ad52551229b6_37.png)

![](img/b3712c68b35e69be2712ad52551229b6_39.png)

![](img/b3712c68b35e69be2712ad52551229b6_41.png)

1.  **机器审核存疑**：当机器审核发现内容可能存在敏感词或违规风险，但无法完全确定时，会**拉长审核时长**，并将内容提交给人工审核员进行最终判断。通常，被提交到人工审核的内容，通过率较低。
2.  **日常人工排查**：平台会有审核团队日常巡查站内内容。有时内容发布初期正常，但后期在排查中被发现违规，从而被打回或提示违规。这可能是由于他人举报，也可能是人工巡查的结果。

![](img/b3712c68b35e69be2712ad52551229b6_43.png)

人工审核的存在是因为**任何平台的规则和系统审核都不是完美的**，存在漏洞。一些内容可能通过特殊方式绕过机器审核，但会在人工排查或用户举报环节暴露风险。

## 流量分发机制 🚀

![](img/b3712c68b35e69be2712ad52551229b6_45.png)

内容审核通过后，便进入流量分发阶段。这个过程决定了你的内容会被推送给谁。

![](img/b3712c68b35e69be2712ad52551229b6_47.png)

![](img/b3712c68b35e69be2712ad52551229b6_49.png)

![](img/b3712c68b35e69be2712ad52551229b6_51.png)

### 第一步：特征提取与初步分类

系统首先需要理解你的内容是什么，以及应该分发给谁。这通过**特征提取**和**初步分类**完成。

![](img/b3712c68b35e69be2712ad52551229b6_53.png)

系统主要通过以下三个维度来分析内容特征：

![](img/b3712c68b35e69be2712ad52551229b6_55.png)

![](img/b3712c68b35e69be2712ad52551229b6_57.png)

1.  **关键词分析**：系统通过语义分析、中文分词等技术，从标题、正文、语音识别字幕、图片文案中提取核心主题关键词。
2.  **Tag标签**：发布时手动添加的话题或标签，为系统提供**参考价值**，辅助判断内容分类，但并非决定性因素。
3.  **账号历史数据**：系统会参考账号以往发布内容的数据表现。如果历史内容集中在某一领域（如护肤），且获得了相关兴趣用户的互动，那么新内容在该领域的分类权重会得到**加成**。这也解释了为何需要保持**账号垂直度**。

![](img/b3712c68b35e69be2712ad52551229b6_59.png)

![](img/b3712c68b35e69be2712ad52551229b6_61.png)

综合以上维度，系统会对内容进行初步分类并打分。例如，一篇护肤笔记可能获得一个初始分类分数，但这个分数在初期通常不高。

### 第二步：数据测试与内容质量评估

![](img/b3712c68b35e69be2712ad52551229b6_63.png)

完成初步分类后，系统会将内容分发给一小部分用户进行**数据测试**，并根据他们的行为反馈来评估内容质量，决定是否给予更多流量。

![](img/b3712c68b35e69be2712ad52551229b6_65.png)

![](img/b3712c68b35e69be2712ad52551229b6_67.png)

用户的行为反馈主要通过以下几个核心数据维度来衡量：

![](img/b3712c68b35e69be2712ad52551229b6_69.png)

*   **点击率 (CTR)**：`点击率 = 点击次数 / 曝光次数`。高点击率意味着封面和标题吸引力强。
*   **停留时长/平均播放时长**：用户观看内容的平均时长。对于图文，还包括翻页、查看评论等深度交互行为。
*   **页面浏览深度 (PV)**：用户从点击内容到浏览主页、查看其他笔记所经过的页面数量。PV值越高，说明用户对账号兴趣越大。
*   **互动行为**：包括点赞、收藏、评论、分享、关注等。这些行为都能为内容带来数据增益。

![](img/b3712c68b35e69be2712ad52551229b6_71.png)

![](img/b3712c68b35e69be2712ad52551229b6_73.png)

**内容优质与否，正是由这四大层面的用户行为数据共同决定的。** 在内容创作前期，就需要思考如何优化这些数据点。

![](img/b3712c68b35e69be2712ad52551229b6_75.png)

![](img/b3712c68b35e69be2712ad52551229b6_77.png)

### 平台主要流量入口

![](img/b3712c68b35e69be2712ad52551229b6_81.png)

![](img/b3712c68b35e69be2712ad52551229b6_83.png)

![](img/b3712c68b35e69be2712ad52551229b6_85.png)

了解分发机制后，我们还需要知道用户从哪里看到内容。以某图文平台为例，主要流量入口如下：

![](img/b3712c68b35e69be2712ad52551229b6_87.png)

![](img/b3712c68b35e69be2712ad52551229b6_89.png)

平台的核心流量通常来自以下几个场景：

![](img/b3712c68b35e69be2712ad52551229b6_91.png)

1.  **发现页/首页**：这是最主要的流量入口，基于用户的兴趣标签进行个性化推荐，同时也会混合推送站内的热门优质内容，动态调整用户标签。
2.  **关注页**：用户在这里查看已关注博主的新内容，目的性非常强，流量精准但总量相对较小。
3.  **同城**：满足用户基于地理位置的需求，如寻找本地美食、住宿、娱乐等，需求明确。
4.  **搜索入口**：用户主动搜索关键词，意图明确，是获取精准流量的重要渠道。
5.  **购物入口**：用户带有明确的购物需求，是电商转化的重要场景。

![](img/b3712c68b35e69be2712ad52551229b6_93.png)

## 总结 📝

![](img/b3712c68b35e69be2712ad52551229b6_95.png)

本节课中，我们一起学习了新媒体平台流量分发与审核的全流程：

1.  **审核机制**：内容发布后需通过**机器审核**（敏感词、营销广告、去重检测）和可能的**人工审核**（处理存疑内容或日常排查）。
2.  **分发流程**：审核通过后，系统通过**特征提取**（关键词、标签、历史数据）对内容进行**初步分类**。然后进行**数据测试**，根据用户的**点击率、停留时长、页面浏览深度和互动行为**来评估内容质量，并决定后续流量分配。
3.  **流量入口**：内容最终通过**发现页、关注页、同城、搜索、购物**等入口触达用户。

![](img/b3712c68b35e69be2712ad52551229b6_97.png)

![](img/b3712c68b35e69be2712ad52551229b6_99.png)

理解这套“后台逻辑”，能帮助我们有针对性地优化内容策略，从选题、创作到发布后的互动，每一步都朝着提升关键数据指标努力，从而更有效地获取流量。
