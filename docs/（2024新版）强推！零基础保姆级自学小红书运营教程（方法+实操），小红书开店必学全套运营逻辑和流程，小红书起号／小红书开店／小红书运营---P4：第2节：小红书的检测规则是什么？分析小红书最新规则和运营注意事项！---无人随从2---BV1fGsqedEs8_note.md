# 小红书运营教程：P4：第2节：小红书平台检测规则详解 🔍

在本节课中，我们将要学习小红书平台的核心检测规则。了解这些规则是避免账号违规、确保内容安全发布的基础。我们将从平台检测的多个维度进行分析，并介绍实用的自查工具。

## 平台检测的核心维度

小红书的检测系统主要针对以下几个方面的内容进行审核，以确保平台环境的合规性。

以下是平台重点检测的四个区域：

1.  **个人资料、笔记内容、私信及评论**：这些位置不得出现微信号、微信二维码、代购、水印、推广淘宝店铺等明显的营销词汇。
2.  **手机设备**：平台会记录设备信息。建议同一台设备最多只登录三个账号，以避免关联风险。
3.  **笔记内容**：内容中不得诱导用户，例如使用“关注领奖”、“点赞抽奖”等话术。这类行为被平台明令禁止。
4.  **内容原创度**：小红书对原创内容的保护程度很高。切勿将同一篇文章通过多个账号发布，应遵循“一篇文章一个账号”的原则。

上一节我们介绍了平台检测的基本范围，本节中我们来看看一个更为具体和关键的规则：敏感词检测。

## 敏感词检测与规避方法

除了上述通用规则，小红书还拥有一套敏感的词汇检测机制。某些特定领域的词汇容易被系统判定为违规。

以下是一些需要尽量避免触碰的领域关键词示例：

*   手机卡
*   金融
*   电子烟
*   代购
*   医疗用品

如果在撰写内容时不慎加入了未知的敏感词，可能会导致笔记无法通过审核。此时，我们可以借助外部工具进行预先检测。

本节课中我们一起学习了小红书平台的检测规则。核心要点包括：了解平台对资料、设备、内容和原创度的审核重点，并掌握使用敏感词检测工具进行自查的方法。遵守这些规则是账号健康运营、内容安全传播的第一步。
