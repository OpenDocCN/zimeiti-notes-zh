# 视频号运营教程：P33：如何寻找书单类对标账号 📚

在本节课中，我们将学习在短视频平台（如抖音）上，如何高效地寻找与自身定位相符的对标账号。我们将以“书单类”账号为例，介绍两种核心方法，帮助你快速锁定模仿和学习的对象。

上一节我们介绍了寻找对标账号的重要性，本节中我们来看看具体如何操作。

## 方法一：利用“关注”推荐功能

![](img/36deb34b335739a7e897c43aa6bc2c32_1.png)

此方法的核心是利用平台的推荐算法。当你关注一个账号时，平台会立即推荐一批与你所关注账号标签相似的其他账号。

操作步骤如下：
1.  在平台搜索框输入你的目标关键词（例如“书单”）。
2.  浏览搜索结果，找到一个视频类型符合你预期的账号。
3.  点击进入该账号主页，确认其内容是你想模仿的类型。
4.  点击“关注”按钮。
5.  关注后，平台会立即在页面底部弹出“你可能感兴趣”的账号列表。这些账号通常与当前账号类型高度相似。
6.  逐一查看这些推荐账号的作品，若符合你的要求，则继续点击关注。
7.  重复此过程，可以在短时间内关注数十个同类账号。此后，平台的信息流会为你推荐更多相似账号。

![](img/36deb34b335739a7e897c43aa6bc2c32_3.png)

这是一种高效的寻找方式。

![](img/36deb34b335739a7e897c43aa6bc2c32_5.png)

## 方法二：通过商品“达人秀”寻找

![](img/36deb34b335739a7e897c43aa6bc2c32_7.png)

此方法适用于已有明确产品（如具体书籍）的情况。通过搜索商品并查看其推广视频，可以找到大量正在销售该商品的账号。

操作步骤如下：
1.  在平台搜索框输入具体的商品名称（例如“墨菲定律漫画版”）。
2.  在搜索结果页，点击顶部导航栏的“商品”分类。
3.  在商品列表中，选择按“销量”排序，筛选出销量最高的商品。
4.  点击进入目标商品详情页。
5.  在商品页中，找到“达人推荐”或“买家秀”板块。这里汇集了所有推广过该商品的短视频。
6.  浏览这些推荐视频，你会发现大量账号都在用相似的内容形式（如影视混剪、名人语录剪辑）推广同类商品。
7.  点击进入这些账号的主页，观察其视频类型和销售数据。作品数量不多但销量高的账号，尤其值得研究和模仿。

![](img/36deb34b335739a7e897c43aa6bc2c32_9.png)

这是另一种寻找对标账号的方式。一个商品链接下的“达人推荐”可能包含数百个账号，为你提供了丰富的学习样本。

---

本节课中我们一起学习了两种寻找书单类对标账号的实用方法：一是通过关注账号触发平台推荐，二是通过搜索具体商品查看达人推广列表。掌握这两种方法，你就能快速建立起自己的学习库，为后续的内容创作和账号运营打下坚实基础。
