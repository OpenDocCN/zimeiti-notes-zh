# 抖音私信自动回复卡片教程：P1：实现私信卡片跳转第三方微信

在本节课中，我们将学习如何在抖音私信中设置自动回复消息卡片，并实现点击卡片直接跳转到微信添加好友的功能。此功能常用于将抖音用户引流至微信私域。

上一节我们介绍了课程目标，本节中我们来看看具体的实现步骤。

## 核心原理概述

该功能的核心是**利用抖音企业服务中心的“自动回复”功能**，在欢迎语中附加一个“消息卡片”。此卡片内嵌了一个指向微信的跳转链接（外链）。当用户点击该卡片时，便会触发跳转动作。

其流程可简化为以下公式：
**用户私信 → 触发自动回复 → 收到欢迎语与消息卡片 → 点击卡片 → 跳转至微信**

## 详细配置步骤

![](img/014c28b60929eacc4cf4dd882e8038f7_1.png)

以下是配置抖音私信自动回复卡片的完整操作流程。

![](img/014c28b60929eacc4cf4dd882e8038f7_2.png)

1.  **账号准备**
    你的抖音账号需要完成**企业认证**，成为蓝V或半蓝V账号，才能使用企业服务中心的相关功能。

![](img/014c28b60929eacc4cf4dd882e8038f7_3.png)

2.  **进入设置页面**
    通过抖音APP登录**企业服务中心**。在页面中向下滑动，找到并点击“**自动化设置**”选项。

3.  **开启自动回复**
    在自动化设置中，选择“**自动回复**”功能。将自动回复的**开关按钮设置为开启状态**。

    ![](img/014c28b60929eacc4cf4dd882e8038f7_1.png)

4.  **配置欢迎语与卡片**
    在自动回复设置页面，找到“**欢迎语**”配置区域。
    *   **欢迎语文案**：根据你的行业或需求，设置一句引导用户点击的欢迎语文本。
    *   **附加内容**：选择“**消息卡片**”作为附加内容。
    *   **卡片链接**：将能够跳转到微信小程序或页面的**外链**植入到消息卡片中。此外链通常需要通过“聚量引擎”和“橙子建站”等工具创建跳转落地页来生成。

    ![](img/014c28b60929eacc4cf4dd882e8038f7_2.png)

5.  **保存生效**
    完成以上设置后，点击页面下方的“**保存**”按钮。设置即刻生效。

    ![](img/014c28b60929eacc4cf4dd882e8038f7_3.png)

## 功能效果验证

配置成功后，当用户首次进入你的抖音私信界面时，系统将**自动发送**你预设的引导文案和消息卡片。用户点击该卡片，即可直接跳转至微信完成添加好友等后续操作。

本节课中我们一起学习了如何从零开始配置抖音私信自动回复跳转卡片。关键点在于使用企业号权限，在自动回复中配置包含跳转外链的消息卡片。掌握此方法，你可以有效地将抖音公域流量引导至微信私域。
