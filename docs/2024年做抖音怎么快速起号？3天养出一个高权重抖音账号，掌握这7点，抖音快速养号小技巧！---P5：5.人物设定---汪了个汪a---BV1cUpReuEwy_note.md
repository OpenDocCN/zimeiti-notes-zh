## 人设策划的核心概念

![](img/00b37914d60983d89bc6ac06471d5d43_8.png)

在抖音中策划人设，类似于在游戏中创建角色，需要预先设定角色的各项属性。但与虚拟游戏角色不同，抖音中的人物设定是**人生角色**，因此必须更具**真实性**。

![](img/00b37914d60983d89bc6ac06471d5d43_10.png)

无论是短视频还是直播，一个成功的IP（人设）能够解决粉丝的信任问题，促进内容的裂变传播，并具备强大的商业变现能力。因此，我们运营账号的核心目的，不仅在于获取流量，更在于**打造IP**。打造IP的本质，是在特定场景或情境中，完成**账号人格化**的过程。

![](img/00b37914d60983d89bc6ac06471d5d43_12.png)

这个过程可以概括为以下公式：
**成功IP = 账号人格化 + 情感注入 + 故事塑造 + 符号化标签**

![](img/00b37914d60983d89bc6ac06471d5d43_14.png)

## 人设策划的六个维度

![](img/00b37914d60983d89bc6ac06471d5d43_16.png)

以下是构建账号人设需要着手策划的六个具体方面：

![](img/00b37914d60983d89bc6ac06471d5d43_18.png)

1.  **名字/昵称**
    一个IP首先需要一个称呼，这是其传播的名片。它如同一个品牌的名称，需要明确且易于记忆。

2.  **职业/身份**
    人格化必然具备身份。这是用户认识并联想该IP的入口，例如老师、律师、创业者等。

3.  **社会称谓**
    一个恰当的称谓能快速拉近与用户的距离，产生情感共鸣。例如“XX哥”、“XX姐”、“XX老师”。

4.  **形象/第一印象**
    从用户视角塑造IP的外在形象，使其更容易占据用户心智。例如大叔形象、甜美可爱、专业干练等。

5.  **性格与特点**
    一个具有鲜明性格特点的IP，记忆点更强，也显得更真实。过于完美的形象反而容易脱离大众，例如“暴躁但暖心”、“吃货属性”等。

6.  **外在标识与认知**
    借助社会普遍认知（刻板印象）来快速定位IP。例如，特定的穿着打扮可以暗示其财富地位、专业领域或生活态度。

## 超级符号的提炼与应用

完成上述基础人设策划后，需要将其落实到账号搭建（头像、昵称、简介等“五件套”）和内容创作中。更重要的是，要通过持续的**内容策划**，为人设提炼出独特的**记忆点**，即“超级符号”。

超级符号可以从以下几个维度进行提炼：

*   **画风（视觉）**：独特的场景、道具、服饰风格。
*   **听觉**：标志性的BGM、特效音、配音声线或口头禅。
*   **文案**：简洁有力、反映核心优势的提炼词或口号。
*   **情绪与表情**：标志性的情绪传递方式，如大笑、惊讶、无奈等表情。
*   **语言特色**：方言、特殊口音、语速语调（如慢条斯理、抑扬顿挫）。
*   **动作习惯**：标志性的肢体动作，如撩头发、扶眼镜、特定手势等。

提炼好超级符号后，在后续的短视频创作中，应有意识地在视频中反复呈现这些元素，不断强化用户记忆，从而让人设深入人心。

本节课中我们一起学习了抖音账号**人物设定**的完整方法论。我们从理解人设的重要性出发，详细拆解了构建人设的六个具体维度，并最终落脚于如何提炼和应用“超级符号”来强化人设记忆点。记住，一个成功的人设是真实人格与精心策划的结合体，是你在抖音海洋中脱颖而出的关键身份标识。
