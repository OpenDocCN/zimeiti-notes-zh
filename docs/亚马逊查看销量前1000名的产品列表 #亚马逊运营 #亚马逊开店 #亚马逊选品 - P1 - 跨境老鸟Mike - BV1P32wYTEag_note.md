# 亚马逊运营：P1：如何查看销量前1000名的产品列表 🔍

在本节课中，我们将学习一个实用的亚马逊运营技巧：如何查看特定小类目下销量排名前1000名的产品列表。这对于选品和市场分析至关重要。

上一节我们介绍了查看销量排行榜的重要性，本节中我们来看看具体的操作步骤。整个过程只需三步即可完成。

以下是查看销量排名的具体操作流程：

1.  **定位并进入小类目排名页面**
    打开一个目标类目的产品详情页，找到产品信息中的“小类目排名”（通常位于“产品信息”或“排名”部分）。点击该排名链接，进入该类目的排行榜页面。

2.  **复制小类目节点编号**
    进入排行榜页面后，观察浏览器地址栏的URL。在URL中找到代表该类目的唯一节点编号（通常是一串字母数字组合），并将其复制下来。

![](img/c091a5f11064746e430a4db30ef8a61a_1.png)

3.  **生成按销量排序的列表**
    准备一个特定的查询网址模板，其结构通常为：`https://www.amazon.com/best-sellers/zgbs/【此处为节点编号】`。将第二步复制的节点编号替换到网址模板的对应位置。在生成的页面中，将排序方式从默认的“最畅销”调整为“**按卖家数量排序**”或类似的选项（具体名称可能为“Best Sellers”或“Top Sellers”），即可看到该类目下按销量综合排名的完整产品列表。

![](img/c091a5f11064746e430a4db30ef8a61a_2.png)

通过以上三个步骤，你就可以突破亚马逊前台通常只显示前100名畅销品的限制，浏览到更全面的类目销量数据，为选品决策提供有力支持。

本节课中我们一起学习了如何通过获取小类目节点编号并修改查询参数，来查看亚马逊上某个类目销量排名前1000的产品列表。这个方法能帮助你进行更深入的市场分析和竞品研究。
