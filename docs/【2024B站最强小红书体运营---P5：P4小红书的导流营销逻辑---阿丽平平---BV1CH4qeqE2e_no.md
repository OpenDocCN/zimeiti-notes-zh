小红书平台上的用户被称为“小红薯”，这包括个人用户和企业用户。个人小红薯可以在平台上分享内容，这些内容被称为“笔记”。

![](img/b35a430c3a0303072c207361d609f4a0_34.png)

![](img/b35a430c3a0303072c207361d609f4a0_36.png)

![](img/b35a430c3a0303072c207361d609f4a0_38.png)

![](img/b35a430c3a0303072c207361d609f4a0_40.png)

“笔记”这一名称的由来，源于小红书希望用户能持续创作并分享关于商品或购物的体验。用户购买商品或发现好物后，进行记录和分享，这个过程就如同记笔记。

![](img/b35a430c3a0303072c207361d609f4a0_42.png)

![](img/b35a430c3a0303072c207361d609f4a0_44.png)

![](img/b35a430c3a0303072c207361d609f4a0_46.png)

![](img/b35a430c3a0303072c207361d609f4a0_48.png)

![](img/b35a430c3a0303072c207361d609f4a0_50.png)

个人小红薯可以针对特定商家的产品、某个单品或一系列商品进行分享。例如，购买了一件显白显瘦、面料舒适的上衣后，用户可以通过小红书分享使用体验。

![](img/b35a430c3a0303072c207361d609f4a0_52.png)

![](img/b35a430c3a0303072c207361d609f4a0_54.png)

![](img/b35a430c3a0303072c207361d609f4a0_56.png)

![](img/b35a430c3a0303072c207361d609f4a0_58.png)

看到笔记的其他用户如果产生兴趣，可能会关注该商品，并询问购买渠道。这样就为商家带来了潜在的流量。

![](img/b35a430c3a0303072c207361d609f4a0_60.png)

![](img/b35a430c3a0303072c207361d609f4a0_62.png)

![](img/b35a430c3a0303072c207361d609f4a0_64.png)

![](img/b35a430c3a0303072c207361d609f4a0_66.png)

![](img/b35a430c3a0303072c207361d609f4a0_68.png)

这种流量引导的路径可以概括为以下公式：
**个人笔记 -> 吸引兴趣用户 -> 引导至商品/店铺 -> 产生销售**

![](img/b35a430c3a0303072c207361d609f4a0_70.png)

![](img/b35a430c3a0303072c207361d609f4a0_72.png)

![](img/b35a430c3a0303072c207361d609f4a0_74.png)

![](img/b35a430c3a0303072c207361d609f4a0_76.png)

同样，如果个人小红薯创作的内容非常优秀，也可能被相关商家主动关注，进而达成商业合作，实现内容变现。例如，一个专注美妆内容的小红薯，其高质量笔记可能吸引美妆品牌寻求合作。

除了个人账号，小红书还存在第二种常见账号类型：企业小红书账号。

![](img/b35a430c3a0303072c207361d609f4a0_88.png)

![](img/b35a430c3a0303072c207361d609f4a0_90.png)

![](img/b35a430c3a0303072c207361d609f4a0_92.png)

![](img/b35a430c3a0303072c207361d609f4a0_94.png)

企业账号通常用于推介自家的商品，即向目标消费者“安利”产品。企业通过平台向用户释放产品的利好信息与价值点。

![](img/b35a430c3a0303072c207361d609f4a0_96.png)

![](img/b35a430c3a0303072c207361d609f4a0_98.png)

![](img/b35a430c3a0303072c207361d609f4a0_100.png)

![](img/b35a430c3a0303072c207361d609f4a0_102.png)

![](img/b35a430c3a0303072c207361d609f4a0_104.png)

目标消费者看到这些内容后，可能对产品产生兴趣或购买欲望，从而被引导至企业的店铺或商城，最终促成销售。这一路径可概括为：
**企业内容 -> 触达目标用户 -> 激发兴趣 -> 引导至店铺/商城 -> 产生销售**

![](img/b35a430c3a0303072c207361d609f4a0_106.png)

![](img/b35a430c3a0303072c207361d609f4a0_108.png)

![](img/b35a430c3a0303072c207361d609f4a0_110.png)

![](img/b35a430c3a0303072c207361d609f4a0_112.png)

此外，也存在部分个人小红薯，其创作笔记不为带货或合作，仅将平台作为个人记录与分享的空间，或将其视为自媒体运营。但从专业运营角度出发，应发挥小红书最根本的价值。

![](img/b35a430c3a0303072c207361d609f4a0_114.png)

![](img/b35a430c3a0303072c207361d609f4a0_116.png)

![](img/b35a430c3a0303072c207361d609f4a0_118.png)

![](img/b35a430c3a0303072c207361d609f4a0_120.png)

小红书最根本的价值在于，无论是个人还是企业账号，都可通过内容创造流量，并将流量进行合理转化。核心逻辑是：
**通过内容创造流量 -> 将流量转化为商业价值 -> 实现变现营收**

![](img/b35a430c3a0303072c207361d609f4a0_122.png)

![](img/b35a430c3a0303072c207361d609f4a0_124.png)

![](img/b35a430c3a0303072c207361d609f4a0_126.png)

![](img/b35a430c3a0303072c207361d609f4a0_128.png)

![](img/b35a430c3a0303072c207361d609f4a0_130.png)

这个逻辑就是小红书平台最根本的导流与营销逻辑。

本节课中我们一起学习了小红书的导流营销逻辑，理解了“笔记”如何作为流量起点，以及个人与企业账号如何通过内容将流量引导至商业环节，最终实现变现。这是进行小红书运营的底层思维框架。
