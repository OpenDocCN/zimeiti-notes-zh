# 亚马逊机器人探秘：1：走进AI驱动的运营中心

在本节课中，我们将跟随记者的镜头，探访亚马逊的AI驱动运营中心，了解其中三种核心机器人的工作原理与特点。

上一节我们介绍了课程概述，本节中我们来看看亚马逊运营中心里最常见的机器人。

## 🤖 赫尔克里：自主搬运工

赫尔克里是亚马逊目前最常见的机器人。它们会独立做出关于自己怎样移动的决定，将货架上的库存运送到配货的员工处。

---

上一节我们认识了负责搬运货架的赫尔克里，接下来我们看看负责包裹分拣的机器人。

## 🦾 罗宾：包裹分拣专家

罗宾是亚马逊机器人团队的第一台机器人系统，它在包裹被送到出库台装车之前进行分拣。

以下是罗宾的工作流程：
*   罗宾会从传送带上抓取包裹。
*   它将包裹放到机器的驱动装置“罗何克林”上。
*   但当被运送到运营中心的下一部分时，包裹可能会掉落。

---

在了解了负责分拣的罗宾之后，最后我们来看看能与员工协同工作的全新机器人。

## 🟢 普罗透斯：自主移动的可爱伙伴

普罗透斯是有着大眼睛的绿色呆萌机器人。它们是亚马逊首个完全自主的移动机器人，能将装有包裹的小车运送到出库台。

由于它们要跟员工一起共事，所以设计得非常注重人机交互安全。其核心行为逻辑可以用以下伪代码描述：

```python
while 执行运送任务:
    if 检测到前方有员工挡路:
        停止移动()
        发出“哔哔”声() # 提示“嘿，你挡住我了”
    否则:
        继续沿路径移动()
```

---

本节课中我们一起学习了亚马逊运营中心里的三种核心机器人：自主搬运货架的**赫尔克里**、负责包裹分拣的**罗宾**，以及能安全与人协作、运送包裹小车的**普罗透斯**。这些AI驱动的机器人协同工作，共同构成了亚马逊高效运营体系的重要一环。
