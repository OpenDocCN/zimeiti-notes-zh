# 直播带货小店随心推3天起号实操方法：P1：七巷论千川实战分享 🚀

在本节课中，我们将要学习如何利用抖音的“随心推”功能，在三天内快速为你的直播带货小店起号。我们将通过三个清晰、可操作的步骤，帮助你建立精准的流量模型，提升投放效果。

上一节我们介绍了课程目标，本节中我们来看看起号实操的第一个核心步骤。

## 第一步：前期投放设置与模型建立

![](img/ee962fbcc41c8aec729174d1536ab405_1.png)

前期使用“随心推”起号时，正确的投放设置至关重要。以下是具体的操作要点：

*   **加热方式**：选择“短视频加热直播间”。因为短视频能预先筛选流量，让进入直播间的用户更精准，从而在起号阶段更容易实现转化。
*   **投放目标**：选择“成交”。我们的核心目标是直接获取有购买意向的用户。
*   **投放计划类型**：前期不要选择“系统推荐”，而应选择“自定义”。在自定义选项中，选择“达人相似粉丝”。

![](img/ee962fbcc41c8aec729174d1536ab405_3.png)

![](img/ee962fbcc41c8aec729174d1536ab405_1.png)

接下来，我们需要为这个新视频寻找精准的流量来源。以下是选择对标达人的方法：

*   在“达人相似粉丝”选项中，尽可能选择满30个与你产品、风格高度对标的达人账号。

![](img/ee962fbcc41c8aec729174d1536ab405_3.png)

![](img/ee962fbcc41c8aec729174d1536ab405_5.png)

**原因解析**：对于一个全新的视频，它没有任何历史成交数据积累。如果一开始就投放“系统推荐”，系统因缺乏数据参考，很容易将流量跑泛，导致成交转化率（ROI）难以提升。我们投放“达人相似”的目的，就是让这个新视频能够触达同行账号的精准粉丝群体，为视频“喂养”初始的成交标签。

![](img/ee962fbcc41c8aec729174d1536ab405_5.png)

当这个视频通过“达人相似”投放累积了大约30到50个成交订单后，其流量模型就会变得相对精准。这是完成起号的第一步。

## 第二步：单计划测试与标签累积

在第一步的基础上，我们需要高效地完成标签累积。操作原则如下：

*   **一个计划对应一个视频**：每个新视频都独立创建一个投放计划。
*   **小额叠投**：按照第一步的设置（投“达人相似”），以100元为单位进行小额多次的叠加投放。
*   **成功标准**：当一个视频通过这种方式累积了30到50个成交订单时，即可认为该视频的精准流量模型已初步建立成功。

## 第三步：模型优化与投产稳定

当视频模型建立成功后，我们的投放策略需要进行调整，以扩大流量并稳定效果。以下是操作流程：

1.  **切换流量池**：将投放计划从“达人相似”改为“系统推荐”，投放目标仍为“成交”。继续以100元为单位进行小额叠投，目的是提升成交转化率（ROI）。
2.  **监控与调整**：投放一段时间后，如果发现ROI开始持续下滑，这表明该视频的流量可能又开始变得泛化。
3.  **稳定投产**：此时，需要将投放目标从“成交”更改为“支付ROI”。通过设定一个理想的ROI值，让系统去稳定投放效果，锁定利润。

**总结一下**：如果是新号使用“随心推”起号，只需遵循以上三个步骤。第一步，通过“达人相似”投放为视频建立30-50单的初始成交标签。第二步，通过单计划小额叠投完成标签累积。第三步，模型成功后转投“系统推荐-成交”以提升ROI，当效果下滑时切换至“支付ROI”来稳定最终投产。

本节课中我们一起学习了“随心推”三天起号的完整实操流程，从建立精准初始模型到优化放大稳定效果，每一步都至关重要。掌握这个方法，能帮助你在起号阶段更高效地利用广告预算，获得精准流量。
