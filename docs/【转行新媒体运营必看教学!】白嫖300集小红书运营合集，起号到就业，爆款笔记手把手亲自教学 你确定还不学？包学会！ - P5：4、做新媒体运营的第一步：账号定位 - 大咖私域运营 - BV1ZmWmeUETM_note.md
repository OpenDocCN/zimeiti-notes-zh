无论你是想做短视频、图文笔记还是直播，第一步都必须明确自己要做什么。这个决策过程就是**账号定位**，它决定了你的内容方向，例如是成为家居博主、母婴博主，还是专注于摄影、校园生活、VLOG或宠物等领域。定位不仅包括赛道选择，还涉及人设打造、商业内容规划等整体布局。

2.  **内容展现**：在内容中形成统一标识，例如固定的开场白、口头禅、标准动作或背景音乐。这有助于用户在众多内容中快速识别并记住你。

内容定位需要解决三个问题：

![](img/34f586433c4b220f3711dfbee7c4517b_243.png)

![](img/34f586433c4b220f3711dfbee7c4517b_245.png)

![](img/34f586433c4b220f3711dfbee7c4517b_247.png)

![](img/34f586433c4b220f3711dfbee7c4517b_249.png)

![](img/34f586433c4b220f3711dfbee7c4517b_251.png)

![](img/34f586433c4b220f3711dfbee7c4517b_253.png)

![](img/34f586433c4b220f3711dfbee7c4517b_255.png)

1.  **内容主题**：你的内容想表达什么？是知识分享、经验总结，还是情感吐槽？主题必须清晰。
2.  **内容吸引力**：如何让你的内容在同类中脱颖而出，吸引用户点击和互动？例如，家居博主可以分享“平价好物改造技巧”，穿搭博主可以聚焦“小个子通勤穿搭”等细分领域。
3.  **平台推流**：如何让内容获得更多平台推荐？这取决于：
    *   **用户行为**：初始流量池用户的互动数据（点赞、评论、完播率）。
    *   **内容赛道**：选择当前平台内，你所在赛道里用户基数更大、更感兴趣的子话题进行创作。例如，在穿搭领域，“穿搭技巧”可能比“店铺测评”的潜在受众更广。

![](img/34f586433c4b220f3711dfbee7c4517b_257.png)

![](img/34f586433c4b220f3711dfbee7c4517b_259.png)

![](img/34f586433c4b220f3711dfbee7c4517b_261.png)

![](img/34f586433c4b220f3711dfbee7c4517b_263.png)

![](img/34f586433c4b220f3711dfbee7c4517b_265.png)

![](img/34f586433c4b220f3711dfbee7c4517b_267.png)

**代码示例（内容定位逻辑）**：
```python
def 内容定位(主题, 创意角度, 平台趋势):
    if 主题清晰 and 创意角度独特 and 符合平台趋势:
        return “优质内容，易获推荐”
    else:
        return “需优化定位”
```

*   **内容定位**：解决用户痛点，如“宝宝不爱吃奶粉怎么办？”、“孩子睡眠不规律如何调整？”。

*   **内容定位**：分享“产后修复标准动作”、“妊娠纹科学处理”、“应对产后焦虑”等知识。

本节课中我们一起学习了账号定位的三个核心维度：**人设定位**、**商业定位**和**内容定位**。
*   **人设定位**解决“我是谁”的问题，旨在建立信任感和记忆点。
*   **商业定位**解决“为何而做”的问题，需要前瞻性地规划变现路径。
*   **内容定位**解决“做什么”的问题，确保内容主题清晰、有吸引力且符合平台流量逻辑。
