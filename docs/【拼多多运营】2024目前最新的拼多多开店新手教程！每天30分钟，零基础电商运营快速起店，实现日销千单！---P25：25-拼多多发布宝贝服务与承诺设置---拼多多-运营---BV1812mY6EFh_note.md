# 拼多多运营：P25：拼多多发布宝贝服务与承诺设置 🛒

在本节课中，我们将学习拼多多发布商品流程的最后一步：如何正确设置商品的服务与承诺。这是确保商品顺利上架并获取平台流量支持的关键环节。

上一节我们介绍了商品规格与描述的填写，本节中我们来看看发布前的最后设置项。

## 了解商品类型

在开始设置服务与承诺前，首先需要了解拼多多的商品类型。这决定了后续需要填写的不同信息。

打开拼多多商家后台，点击“发布商品”后，选择任意类目进入编辑页面。将页面拉至最下方，在“服务与承诺”部分之前，可以看到商品类型选项。

拼多多的商品类型主要分为三种：
1.  **普通商品**
2.  **海外进口商品**
3.  **海外CC个人行邮**（包含“直供”和“直邮”两种模式）

发布商品时，页面左侧的“发布助手”会提供填写指引。如果选择海外商品，必须根据提示选择对应的商品来源地，例如：`货源地: 澳大利亚`。

## 设置服务与承诺

以下是设置“服务与承诺”部分时需要关注的几个核心项目。

### 1. 商品新旧状态
此选项用于声明商品是否为二手商品。
*   **建议**：默认选择“非二手”。因为二手商品在平台上的销售通常不如新品顺畅，普通商品也无此选项。

![](img/9aa659fed123c59ecd64ab93848e1739_4.png)

### 2. 发货时间承诺
此设置影响商品是否带有“急速发货”服务标签，对商品权重和转化率有直接影响。
*   **24小时发货**：承诺当日16点前完成的订单，在当日24点前发出。选择此项，商品会显示“急速发货”标签。
*   **48小时发货**：承诺在48小时内发货。选择此项，商品**不会**显示“急速发货”标签。
*   **当日发货**：与24小时发货类似，也能获得“急速发货”标签。

![](img/9aa659fed123c59ecd64ab93848e1739_5.png)

**核心公式**：`选择24小时/当日发货 = 获得“急速发货”标签 = 提升商品权重与转化率`

### 3. 是否预售
此选项声明商品是否为预售商品。
*   **非预售**：绝大多数普通商品的选择。
*   **预售**：适用于季节性商品（如应季水果）或需要备货周期的商品。选择预售后，必须填写准确的“承诺发货时间”。
    *   **规则**：必须在承诺发货日期的当天结束前（即次日0点前）完成发货，否则将被判违规。
    *   **示例**：设置预售时间为7月6日，则必须在7月7日0点前发出货物。

### 4. 运费模板
需要为商品选择一个已设置好的运费模板。
*   **操作**：直接在下拉菜单中选择已创建的模板即可。运费模板的具体创建方法在前面的课程中已详细讲解。

### 5. 拼单设置
此设置决定成团所需人数。
*   **建议**：选择“两人团”。这有助于降低成团门槛，促进订单转化。

### 6. 服务承诺
平台强制要求开通的服务，务必勾选。
*   **7天无理由退货**：必须开通。
*   **假一赔十**：必须开通。

完成以上所有设置后，点击“提交并上架”，商品即可成功发布。
