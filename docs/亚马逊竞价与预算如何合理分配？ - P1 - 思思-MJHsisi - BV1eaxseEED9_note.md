# 亚马逊广告投放：P1：竞价与预算的合理分配

在本节课中，我们将学习亚马逊广告的核心概念——竞价与预算的合理分配方法。理解并掌握这两者的关系，是控制广告成本、提升广告效果的关键。

上一节我们介绍了课程主题，本节中我们来看看竞价与预算的基本定义及其重要性。

![](img/bb3624fefae725021a49b80b422a3d40_1.png)

## 竞价与预算的基本概念

**竞价** 是指你愿意为一次广告点击支付的最高金额。其公式可以表示为：
`最高每次点击费用（Max CPC） = 你的出价`
这个出价会影响你的广告展示频率和排名。

![](img/bb3624fefae725021a49b80b422a3d40_2.png)

**预算** 是指你为广告活动设定的每日总花费上限。其公式可以表示为：
`每日预算 = 单日最高广告花费`
预算决定了你的广告活动可以持续多久以及消耗的速度。

理解了基本概念后，接下来我们探讨如何为两者设置合理的数值。

![](img/bb3624fefae725021a49b80b422a3d40_4.png)

## 竞价设置策略

设置竞价时，需要考虑市场竞争程度和你的广告目标。以下是几种常见的竞价策略：

*   **手动竞价**：由你直接为关键词设定固定出价。这提供了完全的控制权。
*   **动态竞价-仅降低**：系统会在广告不太可能转化时自动降低你的出价，以节省成本。
*   **动态竞价-提高和降低**：系统会根据转化可能性动态调整出价，可能提高也可能降低。

选择哪种策略，取决于你对广告数据的掌握程度和风险承受能力。

![](img/bb3624fefae725021a49b80b422a3d40_6.png)

## 预算分配原则

预算分配需要平衡广告覆盖面和投资回报率。以下是分配预算时需遵循的核心原则：

![](img/bb3624fefae725021a49b80b422a3d40_7.png)

*   **根据业务目标分配**：新品推广期可分配较高预算，成熟产品则可适度收紧。
*   **遵循二八法则**：将大部分预算（如80%）分配给表现最好的广告活动或产品。
*   **保持连续性**：预算设置应保证广告全天稳定展示，避免因预算过早耗尽而错过流量。

## 实操步骤：优化竞价与预算

以下是一个从初始设置到持续优化的完整流程：

1.  **初始测试阶段**：为新广告活动设置一个中等水平的竞价和保守的每日预算，用于收集数据。
2.  **数据分析阶段**：运行1-2周后，分析ACOS（广告销售成本比）、点击率和转化率等核心指标。
3.  **调整优化阶段**：对高转化、低ACOS的关键词提高竞价和预算；对表现差的关键词降低竞价或暂停。
4.  **周期性复审**：每周或每两周重复一次分析优化过程，使投放策略紧跟市场变化。

本节课中我们一起学习了亚马逊广告竞价与预算的分配方法。我们明确了竞价和预算的定义，探讨了不同的竞价策略与预算分配原则，并梳理了从测试到优化的完整操作流程。合理分配竞价与预算是一个动态、持续的过程，需要基于数据不断调整，才能实现广告效果的最大化。
