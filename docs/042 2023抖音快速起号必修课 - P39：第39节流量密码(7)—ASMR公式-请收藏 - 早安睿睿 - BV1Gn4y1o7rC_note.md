## 什么是ASMR？🧠

上一节我们介绍了其他流量密码，本节中我们来看看ASMR。ASMR是四个英文单词首字母的组合，其英文翻译为“自主性感官经络反应”，俗称“颅内高潮”。

所谓“颅内高潮”，是指通过视觉、听觉、触觉、味觉和嗅觉的共振，产生一种令人平静、愉悦的感觉。这种兴奋感起源于头部，并可能扩展到脊椎。有多种方式可以触发这种感觉。

![](img/d8b2a12834210025cd7761afa15bd09e_1.png)

以下是触发ASMR感觉的一些常见方式：
*   聆听特定语言（如俄语）。
*   关注嘴唇、牙齿的活动。
*   倾听咀嚼声。
*   感受挠痒等触觉刺激。

观看ASMR视频的人中，有98%是为了放松自我、帮助催眠或助眠，还有一些人是为了缓解压力。抖音上存在大量非常火爆的ASMR视频和直播间，因为许多用户希望通过抖音放松日常工作带来的压力。

了解了ASMR的基本概念后，我们通过具体案例来看看它的威力。一个ASMR短视频的流量密码究竟是什么？

我们观察一个播西瓜的短视频。该视频获得了211.7万点赞、12.2万评论、13.5万收藏和27.5万转发，是一则爆款视频，播放量起码上亿。发布该视频的账号“蔡澜”通过制作ASMR内容，在几个月内涨粉超过200万。其内容主要是剥西瓜、剥鸡蛋等。

首先，我们分析“剥西瓜”这个案例。

![](img/d8b2a12834210025cd7761afa15bd09e_1.png)

这个短视频时间很短，但能让观众在听觉、视觉乃至对场景的味觉联想上产生共鸣。这就是美食类ASMR短视频带来的效果。它的制作有以下几个要点：
1.  **专业设备**：可能需要专业的收音器材（如麦克风、声卡）及软件进行采集编辑。
2.  **精美拍摄**：画面拍摄得非常精美、清晰、高清。
3.  **核心声音**：视频中充满动作产生的声音，如摩擦声。
4.  **无语言呈现**：整个视频不说话，完全依靠声音、动作和画面来展现。
5.  **特定道具**：需要使用如西瓜这样的道具来产生特定声音和共振。

## ASMR爆款公式 🧱

![](img/d8b2a12834210025cd7761afa15bd09e_13.png)

通过以上案例，我们可以总结出ASMR爆款短视频的创作公式。该公式主要包含四个维度：

**ASMR爆款公式 = 道具 + 视觉 + 声音 + 场景**

以下是每个维度的具体解释：
*   **道具**：指视频中使用的核心物品，如西瓜、鸡蛋、毛巾、肥皂等。
*   **视觉**：要求画面拍得极致美好，追求视觉上的享受。
*   **声音**：是ASMR的灵魂，主要包括咀嚼声、动作声（如刮擦、破碎声）和口音（如唇齿音）。
*   **场景**：指拍摄所处的环境，如在房间内、车内、锅中等，不同的场景能营造不同的氛围。

这个公式构成了制作爆款ASMR内容的基本框架。

## 公式应用：刮香皂案例 🧼

我们利用上述公式，再分析一个“刮香皂”的案例。

这个短视频仅十几秒，内容看似简单：一个人用一把刀刮一块（可能是）香皂。它没有明确的叙事内容，但却能让观众产生催眠、减压、放松、治愈的效果，促使很多人停下来看完。该视频获得了135万点赞、6.7万评论和1.3万转发。

![](img/d8b2a12834210025cd7761afa15bd09e_15.png)

从公式角度拆解这个视频：
*   **道具**：一把刀和一块香皂。
*   **视觉**：画面简洁，色彩可能经过设计（如香皂的颜色）。
*   **声音**：刀刮香皂时产生的独特摩擦声。
*   **场景**：非常简单，就是在桌面上操作，没有复杂布景。

这个案例证明，将ASMR方式应用得当，可以带来巨大流量。这种方式甚至可以迁移到直播间，对促进转化非常有效。在抖音的某些变现模式（如“体验销售模式”）中，就有直播间模仿这种刮香皂的ASMR形式，并取得了火爆的效果。
