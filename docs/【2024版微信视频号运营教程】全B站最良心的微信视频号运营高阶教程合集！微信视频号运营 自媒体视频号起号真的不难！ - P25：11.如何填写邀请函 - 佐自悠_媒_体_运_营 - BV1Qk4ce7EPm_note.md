# 微信视频号运营教程：P25：11.如何填写邀请函 📝

在本节课中，我们将学习如何正确填写视频号创作者计划的邀请函。这是一个关键步骤，关系到您能否成功加入计划并开始运营。我们将一步步拆解，确保您能清晰、准确地完成所有信息的填写。

![](img/5f41c77149a4e4f12497d7034130f354_1.png)

---

## 第一步：找到您的视频号账号信息 🔍

![](img/5f41c77149a4e4f12497d7034130f354_3.png)

![](img/5f41c77149a4e4f12497d7034130f354_4.png)

上一节我们介绍了视频号的基础概念，本节中我们来看看如何找到自己的账号昵称和粉丝数量。这是填写邀请函的第一步。

以下是找到账号信息的详细步骤：

1.  **打开微信应用**：在您的手机上找到并打开微信软件。

![](img/5f41c77149a4e4f12497d7034130f354_6.png)

![](img/5f41c77149a4e4f12497d7034130f354_7.png)

2.  **进入“发现”页面**：在微信底部导航栏中，点击“发现”选项。

3.  **找到“视频号”入口**：在“发现”页面中，向上滑动屏幕，找到并点击“视频号”入口（图标为黄色蝴蝶形状）。

4.  **进入个人主页**：点击视频号页面右上角的个人头像图标，进入个人主页。

5.  **查看账号信息**：在个人主页中向下滑动，找到“我的视频号”区域。这里会显示您的**账号昵称**和**粉丝数量**。例如，昵称为“天道酬勤”，粉丝数为“39917”。

## 第二步：填写邀请函表格 📄

![](img/5f41c77149a4e4f12497d7034130f354_9.png)

成功找到账号信息后，接下来我们需要将这些信息填入邀请函。请确保每一项都准确无误。

![](img/5f41c77149a4e4f12497d7034130f354_10.png)

![](img/5f41c77149a4e4f12497d7034130f354_11.png)

以下是填写邀请函的具体项目：

1.  **视频号账号昵称**：填写您在上一步中查看到的账号昵称，例如“天道酬勤”。
2.  **粉丝数量**：填写您当前的粉丝数，例如“39917”。
3.  **活动邀请人**：由于是通过特定渠道加入，此处需填写邀请人姓名，例如“张良”或“张良老师”。
4.  **实名信息**：此处必须填写您本人的真实姓名，例如“周建国”。
5.  **结算手机号**：填写您用于接收验证码和后续结算的手机号码，例如 `13812345678`。请注意，号码中不应有空格。

## 第三步：提交与确认 ✅

![](img/5f41c77149a4e4f12497d7034130f354_13.png)

![](img/5f41c77149a4e4f12497d7034130f354_14.png)

所有信息填写完毕后，最后一步是提交并确认。

1.  仔细检查所有已填写的信息。
2.  点击页面上的蓝色“提交”按钮。
3.  系统会弹出“确定要提交吗？”的确认框，点击“确认”完成提交。
4.  提交成功后，页面会显示“已成功加入我们的创作者分成计划”的提示。
