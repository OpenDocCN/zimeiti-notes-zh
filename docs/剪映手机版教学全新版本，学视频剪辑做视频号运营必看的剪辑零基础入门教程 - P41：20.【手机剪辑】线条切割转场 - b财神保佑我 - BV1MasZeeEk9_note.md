# 视频剪辑入门：P41：线条切割转场教程

在本节课中，我们将学习如何使用剪映手机版制作一个动态的线条切割转场效果。这种转场视觉冲击力强，适合用于视频号运营或各类短视频剪辑，能让你的视频衔接更加酷炫流畅。

## 🎬 准备素材与初始设置

首先，在剪辑项目中导入你需要进行转场衔接的两段视频素材。

接下来，我们需要制作一个用于产生切割线条的动画。以下是具体步骤：

1.  点击“画中画”功能，新增一个画中画图层。
2.  在画中画图层中，导入一段纯白色的背景素材。
3.  选中该白底素材，在底部工具栏中找到并点击“蒙版”选项。
4.  在蒙版类型中，选择“镜面蒙版”。
5.  在预览窗口中，将镜面蒙版的两条线向内拖动，将其压缩成一条细线。
6.  将这条细线旋转至90度，使其呈垂直状态。
7.  将时间线移动到视频的第1秒钟位置，在此处添加一个**关键帧**。

## 📐 制作线条展开动画

上一节我们设置了线条的初始状态，本节中我们来看看如何让它动态展开。

1.  将时间线移回视频的0秒起始位置。
2.  在预览窗口中，将那条垂直的细线蒙版向上拖动，直至完全移出画面顶部。
3.  操作完成后，系统会自动在0秒处生成一个关键帧。这样我们就得到了一个线条从画面上方移入并定格的动画。
4.  将这段制作好的动画视频导出，备用。

## ✂️ 应用切割转场效果

现在，我们开始将切割效果应用到实际的视频转场中。

1.  新建一个剪辑项目，导入你希望作为转场后出现的第二段视频素材。
2.  通过“画中画”功能，将刚才导出的线条动画视频添加到上层。
3.  将时间线移动到线条动画完全展开成垂直细线的那个时间点（即我们之前在第1秒打的关键帧位置）。
4.  在此位置，将画中画视频**分割**成前后两段。

以下是针对分割后两部分视频的具体操作：

*   **对于前半部分视频（线条出现的过程）：**
    1.  选中这部分视频，在工具栏中找到并点击“蒙版”。
    2.  选择“线性蒙版”。
    3.  将线性蒙版旋转90度，使其变为垂直分割线。
    4.  此时，蒙版左侧显示的是下层主视频，右侧显示的是画中画的白底。我们需要复制一份来处理另一侧。

*   **对于后半部分视频及复制层：**
    1.  将前半部分视频复制一份，并将其起始点与分割点对齐。
    2.  选中复制的这一层，进入“蒙版”设置。
    3.  点击“反转”按钮。现在，这一层蒙版的左侧变为白底，右侧显示下层视频。
    4.  最后，为这两层画中画视频分别添加出场动画：给左边视频添加“向左滑出”动画，给右边视频添加“向右滑出”动画。

## 🎉 最终调整与完成

效果基本完成后，我们进行最后的清理工作。

检查时间轴，将画中画图层多余的部分（如动画结束后的白底）进行分割并删除，确保只保留必要的转场动画段落。这样，一个完整的线条切割转场效果就制作完成了。

---

本节课中我们一起学习了如何在剪映中制作线条切割转场。关键步骤在于利用**镜面蒙版**制作线条动画，再通过**线性蒙版**和**反转**功能配合**出场动画**，实现视频被线条切割并滑向两侧的视觉效果。多加练习，你就能熟练地将它运用到自己的视频创作中。
