# 视频号分成计划祝福类赛道：P20：18.活动介绍 🎬

在本节课中，我们将详细拆解视频号商家联盟活动的具体内容，包括活动背景、核心福利、参与流程以及官方提供的强大支持。通过本次学习，你将全面了解如何参与此次活动并获取相关资源。

## 活动背景与理念

![](img/99568040fb684980513b9dbfc01813a1_1.png)

![](img/99568040fb684980513b9dbfc01813a1_2.png)

为促进2024年中国电商行业的新发展，并为视频化带货打下坚实的技术基础，特推出本次商家联盟活动。本次活动由短视频教研部负责人张良老师全程对接，并以“公平公正同行”为核心理念展开。

张老师作为本次活动的全国总负责人，拥有独家对接权限。这意味着，只有通过张老师这里才能对接到的资源，在其他地方无法获得。这体现了张老师的实力与背景。

![](img/99568040fb684980513b9dbfc01813a1_4.png)

## 活动核心福利详解

![](img/99568040fb684980513b9dbfc01813a1_6.png)

上一节我们介绍了活动的背景，本节中我们来看看活动提供的具体福利。许多学员担心没有素材或流量，本次活动对此提供了强力支持。

以下是官方为参与者准备的核心福利：

![](img/99568040fb684980513b9dbfc01813a1_8.png)

1.  **官方预制视频素材**：针对“没有素材”的问题，官方已为每个参与账号准备了30条高质量视频。参与者每天发布一条即可。

    ![](img/99568040fb684980513b9dbfc01813a1_2.png)

## 激励政策与结算方式

除了素材和流量，本次活动还设置了极具吸引力的激励政策，以增强参与者的信心。

![](img/99568040fb684980513b9dbfc01813a1_15.png)

*   **保底收益**：参与者每发布一条符合要求的视频，即可获得300元保底收益。
*   **收益计算**：如果发布全部30条视频，总保底收益为 **`30 * 300 = 9000`** 元。
*   **结算周期**：收益结算为“日结”模式，今天发布视频，明天即可到账。该活动由微信视频号发起，并指定张良院长作为唯一对接人。

## 活动期限与参与条件

了解福利后，我们还需要关注活动的时间窗口和参与要求。本次活动设有明确的截止日期。

活动于1月13日正式开始，并将于8月10日结束。活动发起方强调，9000元的奖励机会人人平等，但需要参与者主动把握。需要注意的是，参与活动并获取收益有一个基本前提：**必须将视频发布出去，并成功挂上商品链接**。这意味着参与者仍需学习基本的实操步骤。
