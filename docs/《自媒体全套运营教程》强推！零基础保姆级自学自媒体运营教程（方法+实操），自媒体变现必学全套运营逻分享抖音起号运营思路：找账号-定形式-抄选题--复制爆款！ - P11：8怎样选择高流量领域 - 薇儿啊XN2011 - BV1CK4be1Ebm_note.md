# 《自媒体全套运营教程》：P11：8 怎样选择高流量领域 🎯

在本节课中，我们将要学习如何选择一个高流量的自媒体领域。这是获得平台推荐、吸引观众并最终实现收益的关键一步。没有流量，内容就无人观看，也就无法产生收入。因此，选对平台正在重点推广的领域至关重要。

上一节我们介绍了账号定位与形式选择，本节中我们来看看如何精准选择高流量领域，并规避平台禁忌。

## 核心原则：流量与合规

首先，你需要明白收益上不去的核心原因通常是缺乏流量。没有流量意味着没有观众观看你的视频，自然也就没有收入。因此，选择平台正在大力推流的领域，是获得更好收益的基础。

其次，你必须严格规避平台的禁忌内容。平台通常禁止出现抽烟、喝酒、纹身或穿着暴露等元素。如果你的视频流量不高，可以从以下两方面自查：

![](img/8d0770503805c89bda75aa726c9ddcd6_1.png)

1.  检查你的内容领域是否为平台当前推广的领域。
2.  检查你的素材中是否包含上述违规内容。

![](img/8d0770503805c89bda75aa726c9ddcd6_3.png)

一旦视频因违规内容被限流，其传播效果将大打折扣。

![](img/8d0770503805c89bda75aa726c9ddcd6_5.png)

## 领域细分与专注深耕

![](img/8d0770503805c89bda75aa726c9ddcd6_17.png)

选定大领域后，还需要进一步专注。很多人选择了一个领域，却做得过于宽泛，这反而不利于账号成长。

例如，在美食领域，有人同时做探店、试吃、文化历史和配料盘点。这种做法并不好。你应该在选定的大领域内，明确其细分方向。

![](img/8d0770503805c89bda75aa726c9ddcd6_19.png)

以下是选择细分领域的建议：

*   **深入研究**：搞清楚你所选大领域包含哪些细分方向。
*   **精准选择**：在细分领域中，最多选择1到2个方向深耕。
*   **保持专注**：专注于少数方向，有助于平台识别你的内容标签和目标人群。

如果在一个大领域内做四五个细分方向，平台算法将无法准确判断你的内容定位，难以给你推荐精准的流量和匹配相应的粉丝，最终影响收益效果。让平台清楚知道你在做什么、你的目标受众是谁，它才能为你推送更精确的流量。
