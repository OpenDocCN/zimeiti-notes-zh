# 新媒体运营教程：P19：使用剪映新功能去除视频中不要的人物 🎬

在本节课中，我们将学习如何使用剪映软件的一项全新功能——“局部重绘”，来轻松去除照片或视频画面中不需要的人物或物体。这项功能操作简单，效果出色，并且目前完全免费。

## 功能概述与准备

剪映近期推出了“局部重绘”功能，它能在短时间内智能地抹除画面中的指定元素，并自动填充背景，使画面看起来完整自然。

![](img/59a1fc1e516d143efb9ce499b17cf4d9_3.png)

以下是开始操作前的准备工作。

1.  打开手机上的剪映应用。
2.  点击主界面中的 **`开始创作`** 按钮。
3.  从相册中选择需要处理的照片或视频片段。
4.  勾选右上角的选择圆点，并点击右下角的 **`高清`** 选项，最后点击 **`添加`** 按钮将素材导入项目。

## 核心操作：去除不想要的人物

上一节我们完成了素材导入，本节中我们来看看如何使用“局部重绘”功能来消除画面中的人物。

导入素材后，在时间轴上选中该素材。屏幕下方会弹出工具栏，请向左滑动工具栏，找到并点击 **`局部重绘`** 功能。

进入功能界面后，你会看到“消除”选项。点击它，然后直接用手指在需要去除的人物上进行涂抹。涂抹完成后，点击下方的 **`开始消除`** 按钮。

系统通常会在十秒左右完成处理。处理完成后，你可以看到人物已被移除，且背景填充得非常自然，几乎看不出修改痕迹。你可以通过对比原图和处理后的图片来观察效果。

![](img/59a1fc1e516d143efb9ce499b17cf4d9_3.png)

## 功能拓展：替换画面元素

除了消除，“局部重绘”功能还有一个强大的玩法：替换。我们可以将画面中的某个物体替换成其他东西。

例如，我们想将天空中的一朵云替换成一架飞机。操作步骤如下。

1.  同样选中素材，进入 **`局部重绘`** 功能。
2.  用手指圈出或涂抹想要替换的云朵区域。
3.  在下方文本框中输入想要替换成的物体名称，例如 **`飞机`**。
4.  点击 **`开始生成`** 按钮。

稍等片刻，你就会发现云朵已经变成了飞机。系统还会提供几种不同的飞机样式供你选择。

![](img/59a1fc1e516d143efb9ce499b17cf4d9_5.png)

## 进阶应用：为画面添加新元素

我们还可以利用这个功能，在原本空白的区域添加全新的元素。

假设我们想在画面中的人物旁边添加一只小狗。操作方法与替换类似。

1.  进入 **`局部重绘`** 功能。
2.  在人物旁边的空白处画一个圈，指定生成区域。
3.  在文本框中输入 **`小狗`** 或 **`狗狗`**。
4.  点击 **`开始生成`**。

生成完成后，点击 **`应用`** 即可将效果保存到素材上。对比添加小狗前后的画面，你会发现这个功能非常智能和实用。

![](img/59a1fc1e516d143efb9ce499b17cf4d9_7.png)

## 课程总结

本节课中，我们一起学习了剪映“局部重绘”功能的三种核心用法：
*   **消除物体**：涂抹并移除画面中不想要的元素。
*   **替换物体**：将画面中的指定物体替换为其他内容。
*   **添加物体**：在画面的空白区域生成全新的元素。

这项功能极大地简化了视频编辑中画面修改的难度，即使是初学者也能快速上手，制作出干净、专业的画面效果。
