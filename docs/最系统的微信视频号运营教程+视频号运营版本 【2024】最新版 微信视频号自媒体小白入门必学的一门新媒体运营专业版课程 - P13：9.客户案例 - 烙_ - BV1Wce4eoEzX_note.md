# 微信视频号运营教程：P13：9.客户案例 📖

上一节我们介绍了视频号运营的基础流程，本节中我们来看看如何将这些知识应用于实际教学，特别是面对特殊学员群体时。

![](img/95ca29d85ba3061d9a1b31eca71b2510_5.png)

当时，我按照标准步骤流程指导周建国（周叔）进行操作。我在去年6月20日中午12点49分给他发送了操作指引。大约10分钟后，到了12点58分，我发现他没有回复。



![](img/95ca29d85ba3061d9a1b31eca71b2510_1.png)

![](img/95ca29d85ba3061d9a1b31eca71b2510_2.png)

![](img/95ca29d85ba3061d9a1b31eca71b2510_3.png)

![](img/95ca29d85ba3061d9a1b31eca71b2510_4.png)

![](img/95ca29d85ba3061d9a1b31eca71b2510_5.png)

![](img/95ca29d85ba3061d9a1b31eca71b2510_7.png)

我猜测他可能遇到了问题，于是主动询问。周叔回复说，他记性不太好，总是操作不好。这种情况很常见，尤其是一些年长的学员，他们有时会不好意思提问，或担心自己学得慢、显得笨拙。

以下是教学中的一个重要原则：
*   **不要与他人比较**：你只需要和自己比，只要今天比昨天有进步就可以。

他表示还是不会操作。于是我提出通过视频电话直接指导。他最初拒绝了，接通后，我与他进行了约30分钟的视频通话。我记得很清楚，这个过程我教了他四遍。



![](img/95ca29d85ba3061d9a1b31eca71b2510_9.png)

![](img/95ca29d85ba3061d9a1b31eca71b2510_7.png)

![](img/95ca29d85ba3061d9a1b31eca71b2510_10.png)

教学四遍并非因为他不肯学。问题在于，我让他按我箭头所指找到视频号图标（一个黄色的、像狐狸尾巴的图标），但他始终找不到。我让他截图发过来，才发现我们的手机界面不一样。



![](img/95ca29d85ba3061d9a1b31eca71b2510_9.png)

![](img/95ca29d85ba3061d9a1b31eca71b2510_10.png)

![](img/95ca29d85ba3061d9a1b31eca71b2510_12.png)

原因在于他的手机开启了**老年模式**，导致图标位置和大小都发生了变化。为了解决这个问题，我想了一个办法。

以下是具体的解决方案：
*   我用一台手机给周叔打视频电话。
*   用另一台手机，将显示模式也调整为老年模式，使界面与他的一致。
*   在视频中，我点哪里，他就跟着点哪里。

![](img/95ca29d85ba3061d9a1b31eca71b2510_14.png)

这个办法虽然看起来“笨”，但效果很好。它说明了一个道理：没有教不会的学生，只有不会教的老师。注册完成后，我给他发送了一张专属海报。



![](img/95ca29d85ba3061d9a1b31eca71b2510_12.png)

![](img/95ca29d85ba3061d9a1b31eca71b2510_16.png)

正常情况下，参与“创作者分成计划”需要100个粉丝和原创视频。周叔刚注册，显然不符合条件。但通过我的独家内测通道，即使是零粉丝、零作品、零点赞的账号，也可以直接加入。



![](img/95ca29d85ba3061d9a1b31eca71b2510_17.png)

![](img/95ca29d85ba3061d9a1b31eca71b2510_14.png)

这是我的第二个福利。以下是内测通道的参与方式：
*   海报下方有一个专属二维码。
*   扫描二维码即可添加官方策划人员。



![](img/95ca29d85ba3061d9a1b2510_16.png)

![](img/95ca29d85ba3061d9a1b31eca71b2510_19.png)

![](img/95ca29d85ba3061d9a1b2510_17.png)

需要说明的是，这个二维码是**一人一码**，现在扫描无效。扫描后还需将信息录入后台系统，收益才能正确发放到账户。



![](img/95ca29d85ba3061d9a1b31eca71b2510_19.png)

接下来讲解大家最关心的收益部分。之前提过播放量收益公式：**1000播放量 ≈ 10元**。我之所以愿意指导63岁的周叔操作，是因为这个玩法有保底收益。



![](img/95ca29d85ba3061d9a1b31eca71b2510_21.png)

![](img/95ca29d85ba3061d9a1b31eca71b2510_21.png)

保底规则如下：
*   **发布一条视频即可获得80元保底收益**，无论数据好坏，即使是零播放。
*   经我申请，一个账号**每天最多可发布3条**符合要求的视频。
*   因此，单账号每日保底收益上限为：**3条 × 80元/条 = 240元**。

一个账号最多发3条。虽然理论上可以操作多个账号，但目前平台规则是**一个身份证只能绑定两个账号**，且需要遵循“一人一卡一手机”的原则以防封号。经过测试，一个人操作3-4个账号是比较合理的上限，太多会忙不过来。
