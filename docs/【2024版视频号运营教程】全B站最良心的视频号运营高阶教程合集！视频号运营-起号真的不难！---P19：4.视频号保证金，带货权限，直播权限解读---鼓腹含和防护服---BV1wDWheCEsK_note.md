# 视频号运营教程：P19：视频号保证金、带货权限与直播权限解读 🎬

在本节课中，我们将详细解读视频号的保证金、带货权限和直播权限。这些是开启视频号商业功能的基础，理解它们能帮助你明确运营方向，避免违规操作。

## 创作者中心与身份选择

上一节我们介绍了视频号的基础设置，本节中我们来看看如何查看和设置你的带货身份。

进入视频号，点开右上角的个人头像，即可进入“创作者中心”。在这里，你可以看到自己的带货身份。身份主要分为两种：**达人**和**商家**。这个身份后期可以修改，如果选错也无需担心。

“经营资质”部分通常无需个人用户特别关注。

## 带货权限详解

了解身份后，我们来看看具体的带货权限。带货权限主要分为三个板块：视频带货、橱窗带货和直播带货。

以下是三种带货方式的详细说明：

![](img/76be6bf4a4a004b517767ed316d36e9b_1.png)

*   **视频带货**
    *   **含义**：指在发布的短视频中挂载商品链接，类似抖音的“小黄车”。用户观看视频时，屏幕会出现一个橙色的购物车图标，点击即可购买。
    *   **开通条件**：需要满足 **1000个有效粉丝**，且视频内容必须为**原创**。搬运或混剪的账号很难开通此权限。
    *   **注意事项**：此权限非常严格，一旦发生违规，权限会被立即封禁。封禁后，视频无法挂车，但橱窗功能仍可使用。

![](img/76be6bf4a4a004b517767ed316d36e9b_3.png)

![](img/76be6bf4a4a004b517767ed316d36e9b_4.png)

![](img/76be6bf4a4a004b517767ed316d36e9b_5.png)

*   **橱窗带货**
    *   **含义**：在视频号主页展示商品橱窗，用户点击进入后可浏览和购买商品。这是最基础、最稳定的带货方式。
    *   **优势**：对粉丝数和原创性要求相对较低（具体需参考官方最新规则），是大多数新手和搬运账号的主要带货途径。

![](img/76be6bf4a4a004b517767ed316d36e9b_7.png)

*   **直播带货**
    *   **含义**：在直播过程中展示和销售商品。
    *   **开通条件**：这是一个重点。目前开通直播带货权限需满足以下**任一**条件：
        1.  视频号拥有 **1000个有效粉丝**。
        2.  账号成功开通了**视频号小店**（需营业执照）。
    *   **重要规则**：一个身份证目前最多只能实名认证**2个**视频号用于直播。即使你有5个实名微信，也只能选择其中2个进行视频号直播认证。

![](img/76be6bf4a4a004b517767ed316d36e9b_11.png)

## 权限实操演示与要点

![](img/76be6bf4a4a004b517767ed316d36e9b_13.png)

为了让大家更清楚，这里结合实际情况进行说明。

![](img/76be6bf4a4a004b517767ed316d36e9b_15.png)

![](img/76be6bf4a4a004b517767ed316d36e9b_17.png)

*   **视频带货示例**：在原创视频发布后，约3-5秒屏幕会弹出橙色购物车。用户也可在评论区找到商品链接。
*   **直播权限验证**：
    *   粉丝数达标（如3000粉）的账号，发起直播时可正常添加商品。
    *   粉丝数未达标（如不足1000粉）的账号，发起直播时点击“商品”会提示“未达到直播条件”，无法在直播间挂载商品。但此时**橱窗功能依然可用**。

![](img/76be6bf4a4a004b517767ed316d36e9b_19.png)

对于无法满足1000粉丝要求的账号，解决方案是**开通视频号小店**。通过办理一个营业执照（后续课程会详细讲解办理流程），即可绕过粉丝门槛，直接获得直播带货权限。

## 总结

本节课我们一起学习了视频号的核心商业权限。
1.  **带货身份**分为达人和商家，可后期修改。
2.  **视频带货**权限要求高（1000原创粉），易被封禁。
3.  **橱窗带货**是最通用和稳定的销售渠道。
4.  **直播带货**的关键条件是：**1000有效粉丝** 或 **开通视频号小店**。同时，每人仅能实名认证2个直播账号。

![](img/76be6bf4a4a004b517767ed316d36e9b_25.png)

请根据自身情况选择适合的带货方式。对于新手，建议先从运营橱窗开始积累，同时了解开店流程，为后续开展直播做好准备。
