# 抖音运营实战：P20：16、口播类视频拍摄方法-7

## 概述
在本节课中，我们将学习口播类视频的具体拍摄方法。我们将通过一个成功案例，详细拆解其构图、内容定位、拍摄技巧等核心要素，帮助你掌握制作高质量口播视频的关键步骤。

上一节我们介绍了视频拍摄的通用注意事项，本节中我们来看看**口播类视频**的具体拍摄方法。

口播类视频的拍摄方法具有通用性。无论是金融、装修、养生还是情感口播，其拍摄核心逻辑都是一致的。同样，记录日常生活、拍摄段子或美食视频，也遵循相似的拍摄原则。因此，掌握口播和日常记录这两种类型的拍摄方法，几乎就能覆盖所有内容形式。

首先，我们分析第一个类型：口播视频。这里引用一位学员“种小草”的成功案例。她是一位40岁、带着三个孩子的宝妈，专注于情感口播领域。她的一条视频播放量超过8000万，并借此涨粉30万。

为什么她的视频能如此成功？核心在于**精准吸引目标群体**。她的内容专门针对能产生共鸣的特定人群。例如，如果我的目标群体是20多岁的学生，那么我所有内容都围绕学生感兴趣的话题展开，这样涨粉就会既快又精准。

以下是口播视频拍摄需要关注的几个核心要点：

**1. 构图与画幅比例**
抖音平台采用**9:16**的竖屏格式。关键在于视频内容内部是选择横屏还是竖屏构图。案例中的视频采用了**4:3**的横屏构图。
*   **一致性**：确定一种画幅（横屏或竖屏）后，应保持所有视频统一，避免混用，以维持账号风格的稳定性。
*   **人物位置**：人物主体应占据画面中央约70%的面积。避免脸部怼满屏幕，应在左、右、上方适当留白，使画面看起来清晰、舒适。

**2. 人物表现与设备**
*   **内容大于颜值**：口播视频的核心是传递内容和知识。人物只需看起来清晰、体面即可，不要求特别高的颜值。可以使用美颜滤镜进行适当修饰，但切忌效果过于夸张。
*   **保证声音清晰**：口播视频中，清晰的声音至关重要。建议使用一个**手机收声麦克风**（价格低廉，约十几元），能显著提升音频质量。
*   **提词工具**：如果担心记不住文案，可以在剪辑环节使用提词器软件，实现边看边录，降低拍摄难度。

![](img/eeb98d5031750972920b40b42a330121_3.png)

**3. 背景与环境**
拍摄地点没有限制，可以在户外或家中。关键是**背景要整洁、美观**，避免杂乱无章的画面影响观感。

接下来，我们通过具体视频内容，分析如何精准定位受众。请看案例视频的文案节选：

> “跟婆婆在一起住是什么样的感受？我跟我婆婆在一起住了16年了……我最大的感受就是这十多年以来，我娘家人很少在我家住上几天……这对于我内心来说其实是一个很大的缺失。”

现在，请思考：这个视频想要吸引哪个群体？

![](img/eeb98d5031750972920b40b42a330121_5.png)

答案是：**已婚女性群体**。婆媳关系这个话题，对于未婚年轻女性或男性而言，共鸣感远不如已经步入婚姻、亲身经历此关系的女性群体深刻。视频内容正是精准击中了这部分受众的情感痛点，从而引发了强烈的共鸣和传播。

## 总结
本节课我们一起学习了口播类视频的拍摄方法。我们了解到，成功的口播视频需要做到：**构图稳定统一**、**人物表现得体**、**声音清晰可靠**，并且最重要的是，**内容要精准针对目标受众**，引发特定群体的共鸣。通过分析“种小草”的案例，我们看到了将这些要点付诸实践所带来的显著效果。掌握这些方法，你也能制作出吸引人的口播内容。
