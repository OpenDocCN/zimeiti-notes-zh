## 一、商品准备 🛍️

![](img/acc81ca21d03ef1737c71aaff716e8a9_7.png)

商品是店铺的核心。前期规划中，商品准备可分为两个关键部分：确定产品基础数量与掌握商品发布技巧。

### 1. 产品基础数量要求

整个店铺的商品数量应控制在 **7至15个** 之间。这是为了维持一个健康的店铺动销率，便于集中运营和推广。

以下是不同店铺模式下的产品搭配方案：

*   **多店模式**：每个店铺准备 **7件** 产品即可。
*   **单店模式**：建议准备 **15件** 产品，但需分阶段上架和推广。

产品搭配遵循“主产品 + 副产品 + 补量产品”的结构：

*   **主产品**：店铺的核心爆款，销量最高。
*   **副产品**：辅助销售，能产生少量订单（如1-3单）。
*   **补量产品**：无需产生实际销量，主要用于丰富店铺品类，提升店铺权重。

**具体搭配公式如下：**

*   **初始搭配（适用于前7件产品）**：
    `1件主产品 + 2件副产品 + 4件补量产品`
    或
    `1件主产品 + 1件副产品 + 5件补量产品`

*   **推广第二件主产品时**：
    下架1件补量产品，上新第2件主产品。搭配变为：
    `2件主产品 + 1件副产品 + 4件补量产品`

*   **推广第三件主产品时**：
    搭配变为：
    `3件主产品 + 0件副产品 + 4件补量产品`

当成功打造出3款有销量的主产品后，店铺权重已建立，剩余的补量产品便可逐步转化为可销售的商品。**切记，店铺总商品数不要超过15件**，否则会影响动销率，增加运营难度。

### 2. 商品发布技巧

商品上架的节奏和时机同样关键，目的是为了让小红书系统更好地“认识”并收录你的店铺。

以下是商品发布的核心步骤与注意事项：

1.  **首日上传**：先上传 **1到2件副产品**。此时**不要发布任何推广笔记**，目的仅是让系统收录你的店铺和商品，获得基础排名。
2.  **次日上传主产品**：店铺正常运行一天后，在**流量高峰期**上传你的**主产品**。主产品上架30分钟至1小时后，再开始进行宣传推广。
3.  **控制上架节奏**：每天上架的新商品不要超过3件。如果时间充裕，每天上架1-2件效果更佳。
4.  **统一流量高峰**：所有主产品的上架和推广，应固定在**同一个流量高峰期**（例如，统一在晚上7-9点，或统一在中午12-2点）。这有助于集中权重，提升搜索排名。副产品则可以在任意时间段上架。
5.  **重要原则**：**绝对避免**第一天就直接上传主产品并开始猛推。在系统未收录店铺前操作，权重积累效率极低，几乎是无效劳动。

## 二、基础数据准备 📊

完成商品准备后，便需要为店铺启动进行基础数据规划。这部分内容我们将在下一节课详细展开。

简单来说，基础数据准备包括：确定目标关键词、规划初始销量与评价、设置店铺基础信息（如头像、简介）等。这些工作将为你的首次推广指明方向。

本节课中我们一起学习了小红书店铺启动前期的**商品准备**。我们明确了店铺产品的数量框架（7-15件），掌握了“主、副、补”的产品搭配逻辑，并学会了通过分阶段、择时上架来获取系统初始权重的技巧。这些是店铺从0到1冷启动的关键。
