上一节我们介绍了视频素材的导入与基础比例调整，本节中我们来看看如何通过添加转场和字幕，让视频内容衔接更自然，信息传达更清晰。

![](img/416c20bb5181786f253315e302e56969_3.png)

## 添加视频转场

![](img/416c20bb5181786f253315e302e56969_5.png)

![](img/416c20bb5181786f253315e302e56969_7.png)

每个视频通常由多个片段组成，片段之间需要有自然的衔接。添加转场效果可以明确提示用户内容已切换，避免因画面突然变化导致用户困惑甚至退出，这对于维持视频的**完播率**至关重要，而完播率直接影响视频的推荐与收益。

![](img/416c20bb5181786f253315e302e56969_9.png)

![](img/416c20bb5181786f253315e302e56969_11.png)

**完播率公式**：`完播率 = 完整观看视频的用户数 / 视频的总播放次数`

![](img/416c20bb5181786f253315e302e56969_13.png)

以下是添加转场的具体步骤：

![](img/416c20bb5181786f253315e302e56969_15.png)

![](img/416c20bb5181786f253315e302e56969_17.png)

1.  在视频编辑轨道上，找到并点击右侧的“**+**”号按钮。
2.  在弹出的菜单中，选择“**素材库**”选项。
3.  在素材库的“**热门**”分类中，可以找到多种转场素材，例如搞笑片段或动态效果。
4.  选择转场素材时，应注意时长。过长的转场会影响观看体验，建议选择**3至5秒**时长的素材。
5.  点击选中合适的转场素材，将其添加到视频轨道中。

添加转场后，必须确保其画面比例与主视频保持一致（例如16:9）。选中转场片段，在编辑选项中通常可以找到“**画布模糊**”或“背景”设置，选择清晰匹配的比例选项，然后点击确认。

![](img/416c20bb5181786f253315e302e56969_19.png)

## 添加视频字幕

![](img/416c20bb5181786f253315e302e56969_21.png)

字幕能帮助用户更好地理解视频内容，尤其在环境嘈杂或不方便外放时。添加字幕可以进一步提升视频的观看完成度和信息接收效率。

![](img/416c20bb5181786f253315e302e56969_23.png)

以下是添加字幕的具体步骤：

1.  在编辑界面左下角找到并点击“**文字**”选项。
2.  在文字功能中，选择“**识别字幕**”功能。
3.  点击“**开始匹配**”，软件会自动识别视频中的语音并生成字幕文本。
4.  字幕生成后，可以对其进行调整。拖动字幕框右下角可**放大**字幕。
5.  将调整好大小和样式的字幕文本框，拖动到视频画面底部合适的位置。通常只需调整第一段字幕的样式，后续字幕会自动应用相同样式。

![](img/416c20bb5181786f253315e302e56969_25.png)

![](img/416c20bb5181786f253315e302e56969_27.png)

本节课中我们一起学习了视频剪辑中两个关键的后处理步骤：**添加转场**和**添加字幕**。通过添加转场，我们可以平滑连接不同视频片段，提升观看流畅度以维护完播率；通过添加字幕，我们可以确保视频信息有效传达，覆盖更广泛的观看场景。记住，保持所有元素的**比例一致**（16:9）和**操作规范**，是制作出平台友好型视频的基础。
