新品标签是平台对原创且款式新颖的产品，在满足内部考核后自动授予的标识。打标后，产品能获得更好的推荐曝光和更多展示。

![](img/8d491e452afcf788aabc4f4635440eee_7.png)

新品标的优势包括：
*   在所有端口获得更多曝光。
*   为店铺带来访客的大幅提升。
*   提报活动时，通过率大大增加。
*   报名日常活动后，能在新品特定展示区获得曝光。
*   增加店铺活跃度，提升用户粘性和复购率。

## 如何获取新品加权？🎯

![](img/8d491e452afcf788aabc4f4635440eee_34.png)

*   商品款式新颖，属于新品。
*   系统会识别前一天创建的商品。
*   商品信息需完整详细，款式图片需美观。

满足以上要求，获得新品标的几率就很大。即使类目没有新品标，也要完成新品上传以获取推荐，因为新品推荐是所有类目都有的机制。

## 如何保证图片原创性？🖼️

保证图片原创性的最直接方法是自己拍摄。如果缺乏摄影能力，可以尝试制作“伪原创”图片。

电脑识别图片时，识别的是图片的像素信息，包括像素数量、分布和颜色等。通过修改这些像素信息，可以让平台判定图片为原创。

以下是使用Photoshop制作伪原创图片的步骤：



![](img/8d491e452afcf788aabc4f4635440eee_56.png)

![](img/8d491e452afcf788aabc4f4635440eee_56.png)

1.  将图片导入PS。
2.  在图片上层添加一张颜色构成复杂的图片进行覆盖。
3.  在不影响视觉观感的前提下，将上层图片的**不透明度调低**，使底层产品图清晰可见。
4.  保存并导出图片。



![](img/8d491e452afcf788aabc4f4635440eee_62.png)

![](img/8d491e452afcf788aabc4f4635440eee_62.png)

此操作虽不能完全规避同款识别，但能有效改变图片的像素信息，提升系统对其原创性的判定，从而增加新品推荐权重。如果对操作过程不清楚，可以联系我获取进一步指导。
