# AI新媒体运营教程：P14：01.10.03：AI训练与爆款笔记产出 🚀

## 概述
在本节课中，我们将学习如何训练AI（如ChatGPT）分析爆款笔记，并基于分析结果生成具备爆款潜质的新笔记。我们将从提供样本、分析爆款因素，到最终生成完整文案，走完一个完整的AI辅助创作流程。

上一节我们介绍了通过提供资料让AI学习特定平台风格的方法。虽然以旅游攻略为例的结果可能不理想，但方法是正确的。本节中，我们来看看如何通过分析具体的爆款内容，让AI学习其成功要素并应用于创作。

## 爆款笔记分析要点
我们需要让AI分析爆款内容，并从中提取可学习的规律。以下是几个核心分析点：

![](img/9ec662b0b2120795ad7c8c1368df11f1_7.png)

![](img/9ec662b0b2120795ad7c8c1368df11f1_9.png)

![](img/9ec662b0b2120795ad7c8c1368df11f1_11.png)

![](img/9ec662b0b2120795ad7c8c1368df11f1_13.png)

1.  **爆款原因分析**：分析内容为何能成为爆款，找出关键吸引点。
2.  **内容结构分析**：拆解文案的框架。
    *   **开篇**：用什么方式吸引用户继续阅读。
    *   **中间**：提供哪些内容来戳中用户痛点，提升用户停留时长。
    *   **结尾**：设置什么“钩子”来引导用户互动（如私信、关注）。
3.  **关键词策略分析**：分析笔记中是否植入了特定关键词。
    *   如果有，关键词是什么？
    *   关键词的出现频率和分布结构是怎样的？

### 第二步：向AI下达分析指令
接下来，我们需要给AI（这里使用ChatGPT 3.5）清晰的指令，让它按照我们的要求进行分析。

![](img/9ec662b0b2120795ad7c8c1368df11f1_29.png)

![](img/9ec662b0b2120795ad7c8c1368df11f1_33.png)

**提示词示例**：
```
我给你提供一篇小红书平台上的考研爆款笔记文案，请你分析：
1. 爆款因素：为什么这篇笔记能火？
2. 整体结构：
   - 开篇用什么吸引用户？
   - 中间用什么内容戳中用户痛点？
   - 结尾用什么方式引导用户互动或关注？
3. 关键词分析：分析整篇文案是否设计了关键词。如果有，请指出关键词是什么、出现频率如何。
```

### 第三步：解析AI的分析结果
AI会返回类似以下的分析报告：
*   **爆款因素**：情感共鸣（如开篇营造焦虑感）、逆袭故事、提供实用备考建议、使用表情符号增加亲和力、结尾给予正能量鼓励。
*   **结构分析**：开篇用具体困境引发共鸣，中间分享详细备考方法和资源，结尾鼓励并引导互动。
*   **关键词分析**：本篇笔记未发现明显基于中文分词法设计的关键词，多为常用词汇。

为了获得更全面的规律，我们可以重复第一到第三步，多分析几篇不同特点的爆款笔记，让AI学习更多模式。

### 第四步：整合爆款因素并生成大纲
在AI分析了多篇样本后，我们需要让它总结出共通的爆款要素。

**提示词示例**：
```
请将你前面分析过的几篇考研爆款笔记的爆款因素总结一下，去重后整合成一个完整的列表。
```

![](img/9ec662b0b2120795ad7c8c1368df11f1_56.png)

AI可能会总结出如下要点：**情感共鸣、逆袭故事、明确的备考计划、针对性建议、时间节点明确、具体资源/技巧分享、正能量鼓励、使用表情符号**等。

![](img/9ec662b0b2120795ad7c8c1368df11f1_58.png)

得到爆款因素列表后，即可指令AI根据这些因素创作新笔记的大纲。

**提示词示例**：
```
现在，请结合你总结的爆款笔记因素，为我规划一篇关于考研的小红书笔记大纲。大纲需要细分到具体的小节点。
```

![](img/9ec662b0b2120795ad7c8c1368df11f1_60.png)

![](img/9ec662b0b2120795ad7c8c1368df11f1_61.png)

### 第五步：根据大纲生成完整文案
大纲确认无误后，即可让AI生成完整文案。

**提示词示例**：
```
请根据你提供的大纲，撰写完整的笔记文案。要求：
1. 文案结构严格遵循爆款笔记因素。
2. 包含一个简单的“逆袭”小故事来引入情绪。
3. 包含适量的emoji表情。
4. 尝试在文案中融入“考研”等相关关键词。
```

## 结果评估与优化
AI生成的初版文案可能已经具备基本框架和爆款元素，例如：
*   开篇用“我理解那种感觉…”引发共鸣。
*   中间提供分科目、分时间节点的具体备考建议。
*   结尾鼓励并引导互动。

但对于更高要求的运营，可能还需进一步优化，使其更精炼、丰富或更具个人风格。可以：
*   调整指令的措辞（如“适量表情”与“丰富表情”效果不同）。
*   使用更强大的模型（如GPT-4）。
*   利用自定义GPTs（GPTs）功能，上传专属的风格指南、表情库和范文，打造个性化的AI助手，获得更稳定、更符合需求的输出。

## 总结
本节课我们一起学习了利用AI辅助创作爆款笔记的完整流程：
1.  **提供样本**：寻找并输入目标领域的爆款笔记。
2.  **下达指令**：要求AI分析爆款因素、内容结构和关键词。
3.  **整合规律**：让AI总结多篇笔记的共通爆款要素。
4.  **生成大纲**：基于爆款要素列出内容大纲。
5.  **产出文案**：根据大纲生成融合了爆款元素的完整笔记。
