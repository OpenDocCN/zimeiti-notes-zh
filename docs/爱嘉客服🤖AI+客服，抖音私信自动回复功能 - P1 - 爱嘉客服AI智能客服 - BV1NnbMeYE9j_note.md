# 爱嘉客服🤖AI+客服：P1：抖音私信自动回复功能配置教程

在本节课中，我们将学习如何配置爱嘉客服AI智能客服的抖音私信自动回复功能。你将了解如何设置账号、配置AI参数，并启动自动回复服务。

## 概述

本教程将引导你完成爱嘉客服AI智能客服在抖音私信平台上的自动回复功能配置。我们将从登录平台开始，逐步讲解关键设置项，并演示如何启动和验证AI客服的工作状态。

## 平台登录与选择

首先，登录爱嘉客服AI智能客服的管理后台。登录后，你将看到当前支持的一系列平台列表。

以下是可选的平台示例：
*   抖音私信
*   微信客服
*   其他社交媒体平台

![](img/20a2a83112208d0532dfa38b41ea5f63_1.png)

在本次教程中，我们需要选择 **“抖音私信”** 平台以进行后续配置。

## 基础账号配置

![](img/20a2a83112208d0532dfa38b41ea5f63_2.png)

选择抖音私信平台后，进入配置页面。这里有几个关键设置项。

第一个关键设置是填写你的抖音账号名称。这个设置项决定了AI客服的回复触发条件。

以下是关于此设置的两种模式说明：
*   **填写抖音账号名称**：AI客服将仅在抖音群聊中，当用户**@你**时，才会触发自动回复。
*   **不填写抖音账号名称**：AI客服将对所有接收到的私信进行**全量自动回复**。

![](img/20a2a83112208d0532dfa38b41ea5f63_4.png)

因此，根据你的业务需求，决定是否填写此信息至关重要。

## AI参数配置

![](img/20a2a83112208d0532dfa38b41ea5f63_6.png)

上一节我们介绍了账号的基础配置，本节中我们来看看如何配置AI的核心参数。

在配置页面中，你需要设置所有平台通用的AI服务地址和API密钥（通常标记为`T`）。完成填写后，点击“保存”按钮。

![](img/20a2a83112208d0532dfa38b41ea5f63_8.png)

如果你的某些平台需要使用不同的AI模型或参数，可以在专门的区域进行独立配置。这为多平台管理提供了灵活性。

## 启动与验证

所有配置完成后，即可直接启动AI客服服务。启动后，系统会自动打开抖音私信的网页管理界面。

![](img/20a2a83112208d0532dfa38b41ea5f63_10.png)

启动后，AI客服开始工作。当识别到新的未读消息时，系统会进行检测。

例如，当检测到有两条未回复信息时，AI客服会自动生成并发送两条回复消息。这个过程在群聊或私信场景中均会生效。

![](img/20a2a83112208d0532dfa38b41ea5f63_11.png)

## 配置要点回顾

记得，若要实现“仅在群聊被@时才回复”的功能，务必填写你的抖音账号名称。

![](img/20a2a83112208d0532dfa38b41ea5f63_13.png)

否则，系统将进入全量回复模式，对所有消息进行自动响应。

## 总结

本节课中我们一起学习了爱嘉客服AI智能客服的抖音私信自动回复功能配置。我们涵盖了从平台选择、账号设置、AI参数配置到服务启动与验证的全过程。关键点在于理解**填写抖音账号名称**与**回复触发条件**（@回复或全量回复）之间的关系。正确配置后，你的专属AI客服即可高效处理抖音私信。
