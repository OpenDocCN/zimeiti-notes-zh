# 小红书电商运营课：P48：优惠券与活动设置详解 🛒

在本节课中，我们将学习如何在小红书店铺后台设置优惠券、折扣以及如何选择合适的平台活动。这些营销工具是提升转化率、吸引新客户的关键。

上一节我们介绍了店铺的基础设置，本节中我们来看看如何利用营销工具来促进销售。

## 优惠券设置

优惠券的设置入口位于店铺后台的“营销-优惠券中心”。优惠券主要分为几种类型，其中“全网自动推广券”和“新人券”是最常用、最基础的两种。

以下是设置优惠券的具体步骤：

1.  **创建优惠券**：点击“创建优惠券”，选择“全网自动推广券”或“新人券”。
2.  **填写优惠名称**：名称仅用于后台区分，可自定义，例如“满5减4”。
3.  **设置适用范围**：选择“店铺”则全店商品可用；选择“商品”则需为单个商品单独设置，适合产品较少或需要差异化定价的情况。
4.  **设置领取与使用时间**：
    *   **可领取时间**：建议设置较长，例如一年，避免因优惠券过期未察觉而影响转化。
    *   **可使用时间**：建议选择“相对时间”（如领取后1天或2天内有效），而非“绝对时间”。这能避免因未来价格变动导致利润计算失误或亏本。
5.  **设置优惠规则**：例如，输入 `满5减4`。平台支持设置阶梯式优惠，例如 `满30减5`、`满60减12`，这适用于报名某些平台活动。
6.  **提交创建**：确认信息无误后，点击“确定提交”即可。

## 折扣促销设置

![](img/92392755fe40c0b2636f977384b8b030_6.png)

除了优惠券，还可以通过“商品促销”来设置店铺折扣，操作更为简单。

以下是设置折扣促销的步骤：

1.  **进入促销设置**：在后台找到“营销-商品促销-多品促销”，选择“满件打折”。
2.  **设置活动信息**：
    *   **活动名称**：自定义用于区分的名称。
    *   **活动时间**：同样建议设置较长周期，例如一年。
    *   **促销规则**：选择“全部商品”，设置规则为 `满1件打7.5折`（折扣率可根据利润自行调整，通常保证30%-40%的利润率）。
3.  **完成创建**：点击完成即可。已创建的活动可以随时编辑调整。

## 如何参加平台活动

平台活动（如双十一）能带来巨大流量，但报名有门槛和策略。

以下是报名活动的核心要点与策略：

**活动报名基础要求**：
*   必须缴纳店铺保证金并开通运费险。
*   店铺的售后率、发货率等指标需符合平台标准。

**活动选择与策略**：
平台活动通常分为不同类型，需要根据店铺情况选择：

1.  **适合新品或无销量商品的活動**：此类活动允许你在报名前调整商品折扣。但必须精确计算利润，避免与店铺现有优惠（如折扣、优惠券）叠加后亏本。
    *   **利润计算示例**：假设商品售价100元，店铺设有8折促销和满5减4优惠券，同时活动要求立减15%。
        *   折后价：`100 * 0.8 = 80元`
        *   活动价：`80 * (1 - 0.15) = 68元`
        *   券后价：`68 - 4 = 64元`
        *   若成本为47.5元，则利润极低，需将店铺折扣调整为9折等以保留利润空间。

2.  **常见跨店满减活动（如满300减50）**：此类活动与店铺折扣**互斥**，即生效时会覆盖店铺原有的折扣。因此，定价策略需要调整。
    *   **定价策略**：若想保证活动后到手价为100元，则需反向计算报名价：`100 / 0.8 = 125元`（假设活动折扣为8折）。此时，可以配合使用**阶梯式优惠券**来微调最终利润。
    *   **重要原则**：同一商品切勿同时报名两个可能叠加的活动，否则极易导致亏损。

**活动流量扶持**：
报名活动后，在发布笔记时务必带上活动话题（如#双十一#），有机会获得平台赠送的“曝光券”，增加笔记的曝光量。请注意，曝光券提升的是“曝光次数”，而非直接等于“阅读数”。

此外，可以积极尝试平台新推出的活动玩法（如商品抽签），以获得流量扶持。
