# 新媒体运营速成：P15：01.10.03：AI训练与爆款笔记产出 🚀

在本节课中，我们将学习如何训练AI（以ChatGPT为例）分析爆款笔记，并最终生成具备爆款潜质的新笔记。整个过程分为两步：首先让AI学习并总结爆款内容的成功要素，然后指导它基于这些要素创作新内容。

上一节我们介绍了通过“喂资料”的方式，让AI理解特定平台的文案风格。虽然上一节的旅游攻略案例效果不理想，但方法是正确的。本节中，我们来看看如何通过分析具体的爆款笔记，让AI学习更精细的创作技巧。

## 第一步：指导AI分析爆款笔记

![](img/a2d4e54a75c8627647b89e85dc60f174_5.png)

![](img/a2d4e54a75c8627647b89e85dc60f174_7.png)

![](img/a2d4e54a75c8627647b89e85dc60f174_9.png)

![](img/a2d4e54a75c8627647b89e85dc60f174_11.png)

我们的目标是让AI深入分析我们提供的爆款笔记，并总结出使其成为爆款的关键点。以下是需要AI重点分析的几个维度：

1.  **爆款因素**：笔记中哪些元素（如情感共鸣、实用建议、逆袭故事）可能促使其走红。
2.  **整体结构**：
    *   **开篇**：用什么方式（如提问、制造焦虑、引用名人）吸引用户继续阅读。
    *   **中间**：提供了哪些内容来戳中用户痛点，提升用户停留时长。
    *   **结尾**：如何设置“钩子”（如鼓励互动、引导私信、提供福利）促使用户行动。
3.  **关键词策略**：笔记是否刻意植入了特定关键词，其出现的频率和分布方式如何。

![](img/a2d4e54a75c8627647b89e85dc60f174_13.png)

![](img/a2d4e54a75c8627647b89e85dc60f174_15.png)

接下来，我们需要向AI发出明确的指令。一个有效的指令范例如下：

![](img/a2d4e54a75c8627647b89e85dc60f174_17.png)

```
你是一个小红书爆款笔记分析师。我将为你提供一篇小红书平台上的爆款笔记文案，请你分析：
1.  它的爆款因素有哪些？
2.  它的整体结构：开篇如何吸引用户？中间如何戳中痛点？结尾如何引导互动？
3.  它是否设计了关键词？如果有，是哪个词？出现频率如何？
```

## 第二步：实战演练——以“考研”领域为例

我们以“考研”领域的爆款笔记为例进行演示。

![](img/a2d4e54a75c8627647b89e85dc60f174_25.png)

![](img/a2d4e54a75c8627647b89e85dc60f174_27.png)

![](img/a2d4e54a75c8627647b89e85dc60f174_29.png)

![](img/a2d4e54a75c8627647b89e85dc60f174_30.png)

首先，我们寻找一篇数据不错的爆款笔记（例如点赞量3000+），并将其全文复制。

![](img/a2d4e54a75c8627647b89e85dc60f174_32.png)

![](img/a2d4e54a75c8627647b89e85dc60f174_34.png)

![](img/a2d4e54a75c8627647b89e85dc60f174_36.png)

**操作记录：**
1.  将笔记文案粘贴给AI（例如ChatGPT 3.5），并附上上述分析指令。
2.  AI返回分析结果。例如，它可能指出该笔记的爆款因素包括：“情感共鸣（开篇制造焦虑）”、“提供明确的备考计划”、“使用表情符号增加亲和力”、“结尾给予正能量鼓励”。同时，AI判断该笔记**没有明显设计的关键词**。
3.  为了获得更全面的总结，我们应提供多篇（建议3-5篇）同领域的爆款笔记给AI分析。重复步骤1和2。

## 第三步：整合爆款要素并生成新笔记大纲

在AI分析了数篇爆款笔记后，我们要求它将这些分析结果进行整合、去重，总结出该领域爆款笔记的通用要素列表。

![](img/a2d4e54a75c8627647b89e85dc60f174_51.png)

![](img/a2d4e54a75c8627647b89e85dc60f174_53.png)

**操作指令示例：**
```
请将你前面分析过的几篇考研爆款笔记的爆款因素总结一下，整合成一个完整的列表。
```

AI可能会总结出如下要点：情感共鸣、逆袭故事、明确的备考计划、时间节点清晰、提供具体资源/技巧、正能量鼓励、使用表情符号等。

![](img/a2d4e54a75c8627647b89e85dc60f174_55.png)

![](img/a2d4e54a75c8627647b89e85dc60f174_57.png)

获得通用要素列表后，我们就可以指导AI进行创作了。首先，让它根据这些要素生成一个新笔记的大纲。

**操作指令示例：**
```
我现在要写一篇关于考研的小红书笔记。请结合你刚才总结的爆款笔记因素，为我规划一个具备爆款潜质的笔记大纲。大纲需要细分到具体的小节点。
```

AI生成的大纲可能包含：吸引人的开篇、专业选择分析、各科目备考规划、时间管理技巧、心态调整方法、互动引导等部分。

![](img/a2d4e54a75c8627647b89e85dc60f174_59.png)

![](img/a2d4e54a75c8627647b89e85dc60f174_61.png)

![](img/a2d4e54a75c8627647b89e85dc60f174_62.png)

## 第四步：根据大纲生成完整文案

![](img/a2d4e54a75c8627647b89e85dc60f174_63.png)

如果对大纲满意，我们就可以命令AI根据这个大纲和之前总结的爆款要素，撰写完整的笔记文案。

**操作指令示例：**
```
请根据你上面提供的大纲，撰写一篇完整的小红书笔记文案。要求如下：
1.  结构严格遵循爆款笔记的要素。
2.  开篇融入一个简单的“逆袭”小故事，引发读者共鸣。
3.  文案中要包含适量的emoji表情。
4.  尝试以“考研全流程”、“考研政治”等作为关键词，自然地融入文案中。
```

随后，AI会生成一篇完整的笔记。初稿可能在某些部分（如语言流畅度、细节丰富度）有所不足，但这已经为运营人员提供了一个高质量的基础稿。我们可以在此基础上进行人工优化和调整，使其更精炼、更有吸引力。

**进阶提示**：使用更强大的模型（如GPT-4）或创建自定义的GPTs，通过上传更多风格范例和具体要求，可以打造出更懂你、产出质量更稳定的专属AI助手，从而进一步提升效率。

![](img/a2d4e54a75c8627647b89e85dc60f174_65.png)

## 总结

![](img/a2d4e54a75c8627647b89e85dc60f174_67.png)

![](img/a2d4e54a75c8627647b89e85dc60f174_69.png)

本节课中，我们一起学习了利用AI辅助创作爆款笔记的全流程。关键步骤是：**提供爆款案例 -> 指令AI分析总结 -> 基于总结要素生成大纲 -> 依据大纲产出文案**。这个方法的核心在于通过“投喂”和“分析”，让AI从优秀案例中学习底层逻辑，从而模仿并生成符合平台调性的新内容。掌握这个流程，能显著提升你在小红书等平台的内容创作效率和质量。
