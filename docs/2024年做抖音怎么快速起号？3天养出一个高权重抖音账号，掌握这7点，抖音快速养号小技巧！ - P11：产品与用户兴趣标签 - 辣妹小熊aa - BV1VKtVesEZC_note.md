# 抖音运营教程：P11：产品与用户兴趣标签 🧩

在本节课中，我们将学习一种名为“九宫格选题法”的实用技巧。该方法能帮助你系统性地为抖音账号（或其他内容平台）挖掘源源不断的创作灵感。我们将以健身软件Keep为例，详细拆解如何结合产品属性和用户兴趣来生成优质选题。

上一节我们介绍了选题的重要性，本节中我们来看看如何具体操作。

## 产品标签分析

首先，我们需要分析产品本身的特点。我们将Keep这个产品放在中心，围绕它联想并填充其核心属性。

以下是Keep产品可能具备的标签：

1.  **线上健身软件**：这是产品的核心定位。
2.  **专业健身教练**：平台提供专业指导资源。
3.  **饮食搭配建议**：提供与健身配套的营养方案。
4.  **减肥/塑形攻略**：满足用户的瘦身与体型管理需求。
5.  **健身干货知识**：传授科学的运动方法与理论。
6.  **打卡运动制度**：通过打卡机制培养用户运动习惯。
7.  **主要用户分布**：产品初期可能聚焦于一二线城市用户。

完成产品标签填充后，我们暂时将其放在一边。

## 用户兴趣标签分析

接下来，我们分析目标用户的兴趣。将“健身爱好者/想要减肥的人”放在中心，开始联想他们可能还关心哪些领域。

以下是围绕目标用户可能延伸的兴趣标签：

![](img/928960378bf84c131c2e99cbb32d4627_1.png)

1.  **健身房**：用户可能同时有线下健身的需求或经历。
2.  **健身装备**：如运动服饰、跑鞋、器械等。
3.  **轻脂美食**：对健康、低卡路里的餐食感兴趣。
4.  **旅行**：热爱生活的人往往也热爱探索。
5.  **星座/明星**：普遍的娱乐与社交话题。
6.  **心灵鸡汤**：关注个人成长与激励内容。
7.  **时尚穿搭**：与塑形后的外在展示相关。
8.  **生活热爱者**：将用户画像从“健身者”扩大到更广泛的“积极生活者”。

现在，我们针对产品和用户填满了九宫格。

![](img/928960378bf84c131c2e99cbb32d4627_1.png)

## 生成选题：连连看组合法

填满九宫格后，我们进入核心环节——生成具体选题。方法是进行“连连看”，将产品标签与用户兴趣标签进行随机或创意组合，碰撞出选题火花。

以下是几个组合示例：

*   **组合1：饮食搭配 + 旅行**
    *   可能的选题方向：`外出旅行时，如何享受美食又能控制体重？`

*   **组合2：运动方法 + 明星**
    *   可能的选题方向：`赵丽颖都在练的5个居家健身动作，你一定要学！`

*   **组合3：科学（健身） + 健身房**
    *   可能的选题方向：`去健身房别再瞎练了！这3个科学原则让你效率翻倍。`

通过这种组合，理论上可以衍生出无数选题。但组合出的想法可能比较初步，这时需要借助搜索工具进行完善。

例如，针对“科学健身”这个方向，可以去百度搜索“健身 科学”，查看相关的搜索联想和长尾词，从中获取更具体、更符合用户搜索习惯的选题灵感。

本节课中我们一起学习了“九宫格选题法”。我们首先分别梳理了产品的核心标签和用户的兴趣标签，然后通过将这两类标签进行创意组合，来批量生成视频选题方向。这个方法能有效解决“不知道拍什么”的难题，并帮助你建立起自己的选题素材库，让内容创作变得系统而高效。
