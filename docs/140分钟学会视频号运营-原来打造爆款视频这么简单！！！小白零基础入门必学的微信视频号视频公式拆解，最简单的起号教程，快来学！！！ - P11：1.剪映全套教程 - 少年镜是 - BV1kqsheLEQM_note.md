在本节课中，我们将学习如何从零开始自学剪映软件，掌握视频剪辑的基础操作。课程将提供清晰的自学路径和资源指引，帮助你快速上手。

## 课程概述

![](img/a60a93bbab01b5f22c23a623850e052b_1.png)

短视频剪辑是视频号运营的基础技能。目前，剪映是抖音官方推出的、功能强大且使用广泛的免费剪辑软件。即使你没有任何剪辑基础，通过系统学习也能快速掌握。

## 自学路径与方法

![](img/a60a93bbab01b5f22c23a623850e052b_5.png)

上一节我们介绍了剪辑的重要性，本节中我们来看看如何高效地自学剪映。

### 1. 获取软件与官方课程

![](img/a60a93bbab01b5f22c23a623850e052b_7.png)

首先，从未使用过剪映的用户需要在手机应用商店下载“剪映”App。其官方账号提供了最权威、最系统的免费学习资源。

![](img/a60a93bbab01b5f22c23a623850e052b_9.png)

以下是找到并学习官方课程的步骤：

![](img/a60a93bbab01b5f22c23a623850e052b_11.png)

1.  打开剪映App，点击首页底部的“创作者课堂”。
2.  在搜索框中输入“剪映小助手”并搜索。
3.  在搜索结果中点击“用户”标签，找到并关注名为“剪映”的官方账号。
4.  进入该账号主页，点击“课程”栏目。
5.  重点学习《剪映功能大全》和《新手常见问题》这两个免费课程系列。

![](img/a60a93bbab01b5f22c23a623850e052b_13.png)

《剪映功能大全》包含87节课，《新手常见问题》包含10节课。每节课时长通常在几十秒到两分钟左右，总学习时间约两小时。学完即可从零基础达到熟练操作的水平。

![](img/a60a93bbab01b5f22c23a623850e052b_15.png)

### 2. 利用其他平台优质资源

![](img/a60a93bbab01b5f22c23a623850e052b_17.png)

除了剪映App内的官方课程，抖音平台也有许多优质的教学账号，其视频讲解通常更简短精炼。

以下是推荐的搜索与学习方法：

![](img/a60a93bbab01b5f22c23a623850e052b_19.png)

1.  打开抖音App，在搜索框直接搜索抖音号 **`妮妮6868618`**（用户“妮妮短视频”）。
2.  关注该账号，其主页有系统的剪映教程合集，每集时长约几分钟，讲解细致。
3.  此外，你还可以在抖音搜索关键词如“**零基础学剪辑**”，会找到大量教学账号。观看几个不同博主的视频，就能全面掌握基础功能。

## 核心操作示意

在学习过程中，你会频繁使用一个核心操作逻辑：
**`点击视频片段 -> 下方出现二级编辑工具栏`**
这个工具栏包含了分割、变速、滤镜、动画等所有关键功能，是进行一切剪辑操作的基础。
