# 抖音同城生活-健康垂类0到1运营：P39：39-6.直播间视觉设计拆解 👁️

![](img/e53c76282f6f4562bebb17892a109c3c_0.png)

在本节课中，我们将要学习本地生活直播间视觉设计的核心要素。视觉设计是吸引用户、传递信息、提升转化率的关键环节，尤其对于医疗健康这类垂类账号至关重要。

![](img/e53c76282f6f4562bebb17892a109c3c_2.png)

上一节我们介绍了直播的整体流程，本节中我们来看看如何通过专业的视觉设计，让直播间在第一时间抓住用户眼球。

![](img/e53c76282f6f4562bebb17892a109c3c_4.png)

对于本地生活直播间而言，视觉设计非常重要。医疗机构通常配备有美工设计师，在拥有美工的情况下，只需让他们参考以下设计环节即可。

以下是直播间必须包含的几个视觉设计环节，可以让美工直接参考制作。

*   **预热海报**：此海报可用于发布在朋友圈、社群或粉丝群。设计上基本可以参照优秀案例进行制作。
*   **直播聚合页**：聚合页的设计样式基本固定，可参考示例进行布局。
*   **直播间天面（封面/背景）**：这部分至关重要。用户刷抖音时，首先看到的是直播间的曝光画面，而非直接进入直播间。一个出色的天面设计能瞬间吸引用户点击进入。理想的天面应做到让用户进入直播间后，无需主播开口就能明白直播主题。天面质量直接影响了直播间的**曝光进入率**、用户**停留时长**以及**团购商品的转化数据**。
*   **手持提示卡**：手持卡的重要性较高，它主要影响主播介绍团购产品的效率，以及最终商品的**成交率**。

所有关于直播间的视觉设计，建议让视觉设计师或美工直接参照优秀案例进行平行替换，在符合自身品牌调性的前提下，确保以上所有环节都完备。这是新直播团队与原有美工设计能力的一次结合。

![](img/e53c76282f6f4562bebb17892a109c3c_6.png)

本节课中我们一起学习了直播间视觉设计的四大核心组件：预热海报、聚合页、天面和手持卡。理解并应用这些设计，能有效提升直播间的吸引力与转化效率，是健康垂类账号从0到1运营中不可或缺的一环。
