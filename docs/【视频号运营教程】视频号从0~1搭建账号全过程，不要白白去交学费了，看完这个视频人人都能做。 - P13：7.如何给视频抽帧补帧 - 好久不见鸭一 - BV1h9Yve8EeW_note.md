# 视频号运营教程：P13：7.如何给视频抽帧补帧 🎬

在本节课中，我们将学习视频剪辑中的一个实用技巧——抽帧与补帧。这个操作可以帮助我们微调视频时长，同时保持视频内容的流畅性，是视频二次创作中的常用手法。

上一节我们介绍了视频剪辑的基础操作“掐头去尾”，本节中我们来看看如何通过抽帧和补帧来进一步处理视频素材。

## 理解帧与时长

首先，我们需要理解视频的基本构成。视频是由一系列连续的静态图片（即“帧”）快速播放形成的。在常见的视频格式中，**每秒通常包含30帧（30 FPS）**。这意味着一个18秒的视频，总共包含 `18秒 * 30帧/秒 = 540帧`。

## 什么是抽帧与补帧？

*   **抽帧**：指从视频中删除某一帧或几帧画面。
*   **补帧**：指在视频中复制并插入已有的某一帧画面。

这两个操作通常配合使用，可以在几乎不影响观众观感的前提下，微调视频的总体时长。

以下是进行抽帧和补帧的具体步骤。

## 操作步骤详解

1.  **放大时间轴**：将视频导入剪辑软件后，首先需要将时间轴刻度放到最大，以便精确到每一帧进行操作。在最大刻度下，你可以清晰地看到从 `00:00:00` 到 `00:00:01` 之间共有30个帧（F）。

2.  **执行抽帧**：
    *   在时间轴上找到你想要删除的某一帧，例如第20帧（20F）。
    *   在此帧的位置进行**分割**。
    *   紧接着，在第21帧（21F）的位置再次进行**分割**。这样，你就单独选中了这一帧的画面。
    *   将这一小段（即这一帧）**删除**。这个删除单帧画面的过程就是“抽帧”。

3.  **执行补帧**：
    *   在视频的其他部分（通常是非关键动作画面），选中一段非常短的片段（例如某一帧）。
    *   对这个选中的片段执行**复制**，然后**粘贴**到时间轴的其他位置。这个复制并插入帧的过程就是“补帧”。

**核心操作原则**：对于一个视频，通常抽3帧、补3帧即可，这样对视频整体的流畅度影响最小。

## 操作要点与总结

![](img/9fd55775004761472d810c6efa1e13bf_2.png)

本节课中我们一起学习了抽帧与补帧的技巧。关键点在于，这些操作应选择在画面动作幅度小、相对静止的部分进行，例如风景空镜或人物特写停顿处。通过精准地删除或复制单帧，我们可以在观众难以察觉的情况下调整视频节奏和总时长，使其更符合平台发布要求或背景音乐节奏。

记住，熟练运用“掐头去尾”、“抽帧补帧”这些基础技巧，是高效完成视频二次创作的重要步骤。
