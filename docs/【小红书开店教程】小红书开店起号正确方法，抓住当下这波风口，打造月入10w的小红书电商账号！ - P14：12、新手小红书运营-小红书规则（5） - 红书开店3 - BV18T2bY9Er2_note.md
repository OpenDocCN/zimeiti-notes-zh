## 笔记被推荐的核心逻辑

笔记发布后，平台会给予一个基础的初始展示量。能否获得更多流量，取决于笔记在初始流量池中的表现。其核心逻辑遵循一个两级递进的考核机制。

### 第一级：打开率

打开率是笔记获得更多推荐的第一道门槛。当用户打开小红书App，首页“发现”页会展示4到6篇笔记。系统会将你的笔记与其他同类型笔记一起，放入一个约20篇笔记的初始测试流量池中。

**核心公式**：`打开率 = 点击量 / 展示量`

在这个初始池中，你的笔记需要达到**平均11%的打开率**，才能进入下一级流量池。例如：
*   如果系统给了你1000次展示，你需要获得至少110次点击（小眼睛）。
*   如果给了你2000次展示，则需要至少220次点击。

若打开率不达标，笔记通常只会获得这一波基础推荐，后续流量增长会非常缓慢。打开率低通常意味着封面或标题吸引力不足。

### 第二级：互动率

当笔记通过第一级考核，进入更大的流量池（例如2000-5000次展示）后，平台会考核第二项指标：互动率。

![](img/8a176fe1a3f66b07cee7ba9299f060d6_9.png)

**核心公式**：`互动率 ≈ 互动数（评论为主）/ 点击量`

![](img/8a176fe1a3f66b07cee7ba9299f060d6_11.png)

![](img/8a176fe1a3f66b07cee7ba9299f060d6_13.png)

在这个阶段，笔记的**平均互动率需要超过3%**。例如，如果有110人点击了笔记，那么至少需要产生4条评论。互动率越高，笔记被继续推荐到万级甚至十万级展示池的机会就越大。

![](img/8a176fe1a3f66b07cee7ba9299f060d6_15.png)

**总结升级路径**：`基础展示 → 打开率达标 → 更大展示 → 互动率达标 → 持续推荐`
对于新账号，通常能触及2-3级流量池。高权重老账号的起点更高，有机会冲击更高级别的流量。

![](img/8a176fe1a3f66b07cee7ba9299f060d6_17.png)

### 综合评分指标

![](img/8a176fe1a3f66b07cee7ba9299f060d6_19.png)

除了打开率和互动率这两个基础门槛，平台还有一个综合评分体系，用于精细调整推荐权重。主要包括：
*   **点赞**
*   **收藏**
*   **关注**（用户通过该笔记关注你）
*   **分享**

这些行为都会为笔记增加“评分”。但请注意，**必须先满足11%打开率和3%互动率这两个基础属性**，后续的点赞、收藏等加分项才能充分发挥作用。

## 必须避免的运营“坑”

了解推荐逻辑后，我们还需要规避平台禁忌，以防账号被降权。以下是新手常见的错误操作列表：

*   **新号不要立刻发布笔记**：新注册的账号需要“养号”，即先模拟正常用户浏览、互动几天，再发布内容。当天注册当天发笔记，会浪费平台的初始权重，导致后续流量低迷。
*   **违规会严重降权**：任何违规行为（如引流、抄袭、发布违禁内容）都会导致账号被降权，笔记推荐和搜索排名大幅下降，恢复周期长且困难。
*   **标签切忌反复修改**：账号、店铺或笔记的人群标签一旦确定，不要频繁修改。尤其是在账号已有稳定流量后修改标签，极易导致限流。
*   **不要一机多号**：尽量避免一台手机频繁切换登录多个小红书账号，这容易被系统判定为营销号。理想状态是“一机一卡一号”。
*   **谨慎处理引流信息**：不要在笔记正文、评论或私信中直接留下微信号、手机号等联系方式，或使用“找我购买”、“私聊”等明显引导站外交易的词汇。
