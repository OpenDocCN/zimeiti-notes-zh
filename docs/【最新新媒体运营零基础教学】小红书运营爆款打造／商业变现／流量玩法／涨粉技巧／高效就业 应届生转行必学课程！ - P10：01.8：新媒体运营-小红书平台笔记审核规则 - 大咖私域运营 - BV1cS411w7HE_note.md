# 新媒体运营：P10：01.8：小红书平台笔记审核规则 📝

在本节课中，我们将要学习小红书平台笔记的流量推荐与分发机制。了解笔记从发布到推荐给用户的完整流程，特别是其中的审核规则，是做好小红书运营的基础。

## 概述

小红书与许多新媒体平台一样，采用“千人千面”的内容展现形式，即根据用户的特定标签推送特定内容。本节将详细解析一篇笔记发布后，直至被推荐给用户所经历的各个阶段。

## 笔记审核的三个阶段

笔记发布后，小红书系统会对其进行审核。通常，没有问题的笔记在机器审核阶段即可通过。整个审核流程主要包含以下三个阶段。

![](img/922c006a947add2e5f30ff62834d46bf_1.png)

### 第一阶段：机器审核

笔记首先会被上传至审核系统。该系统会根据预设的基本规则进行初步审核。

![](img/922c006a947add2e5f30ff62834d46bf_3.png)

![](img/922c006a947add2e5f30ff62834d46bf_4.png)

![](img/922c006a947add2e5f30ff62834d46bf_5.png)

以下是机器审核的主要规则：
*   **敏感词与敏感图片**：系统会检测文本和图片中是否包含违规内容。
*   **内容质量评估**：系统会评估笔记的基础质量，例如：
    *   笔记的字数。
    *   广告词出现的频率。
    *   图片清晰度，以及图片中的文字、物体是否能被系统识别。

![](img/922c006a947add2e5f30ff62834d46bf_10.png)

如果笔记在此阶段发现明确违规，可能会被直接封禁。如果内容存在疑似违规或机器难以判断的情况，则会被标记并转入下一阶段。

### 第二阶段：去重检测

通过初步机器审核的笔记，会进入去重检测阶段。系统会将你的笔记与平台数据库中的其他内容进行比对，检查相似度。

![](img/922c006a947add2e5f30ff62834d46bf_16.png)

![](img/922c006a947add2e5f30ff62834d46bf_17.png)

去重检测主要包括以下方面：
*   **图片查重**：检查图片是否为直接抄袭或复制他人。
*   **文案查重**：检测文案内容的重复比例，以及段落是否完全重复。

![](img/922c006a947add2e5f30ff62834d46bf_18.png)

![](img/922c006a947add2e5f30ff62834d46bf_19.png)

![](img/922c006a947add2e5f30ff62834d46bf_20.png)

![](img/922c006a947add2e5f30ff62834d46bf_21.png)

如果笔记原创度极低（例如原封不动抄袭），重复率检测接近100%，则会被直接封禁。如果对图片进行裁剪、加滤镜、调整内容，或对文案段落进行修改、扩充，则有可能通过此阶段检测。

### 第三阶段：人工审核

![](img/922c006a947add2e5f30ff62834d46bf_27.png)

人工审核主要针对机器审核标记的“疑似违规”内容进行二次复核。

![](img/922c006a947add2e5f30ff62834d46bf_29.png)

![](img/922c006a947add2e5f30ff62834d46bf_30.png)

需要注意的是：
*   由于内容量巨大且人工成本高，并非所有笔记都会进入此阶段。
*   在某些特殊领域（如政治、色情擦边、法律严禁内容），人工审核的力度会非常大。
*   目前系统算法已较为完善，触发明确红线的笔记通常由机器直接处理，较少进入人工复核。用户若对结果有异议，可以发起申诉。

对于运营者，特别是新号，应尽量避免触发审核规则，因为即使是少数违规，也可能对账号产生隐形的负面影响。

## 账号权重与流量分发

![](img/922c006a947add2e5f30ff62834d46bf_38.png)

上一节我们介绍了笔记审核的具体流程，本节中我们来看看影响笔记最终流量分发的另一个核心概念：账号权重。

![](img/922c006a947add2e5f30ff62834d46bf_39.png)

![](img/922c006a947add2e5f30ff62834d46bf_40.png)

![](img/922c006a947add2e5f30ff62834d46bf_41.png)

![](img/922c006a947add2e5f30ff62834d46bf_42.png)

在机器审核阶段，系统除了判断违规，还会为笔记打一个“基础质量分”。而笔记最终能获得多少初始曝光，还与发布账号的“权重”密切相关。我们可以将“权重”理解为一个系统对账号的综合评分。

![](img/922c006a947add2e5f30ff62834d46bf_43.png)

![](img/922c006a947add2e5f30ff62834d46bf_44.png)

![](img/922c006a947add2e5f30ff62834d46bf_45.png)

![](img/922c006a947add2e5f30ff62834d46bf_46.png)

![](img/922c006a947add2e5f30ff62834d46bf_47.png)

**权重分值（假设）**：`新号权重 ≈ 0`；`优质账号权重 → 10`

以下是不同权重阶段的特点：
*   **新号阶段（低权重）**：新注册账号权重接近0。此阶段笔记初始曝光量通常很低（如30-40次浏览），且因为账号标签未形成，内容无法精准推送给目标用户。此时重点在于**避免违规**、**保持稳定更新**、**提升内容垂直度**，以积累系统信任，逐步提升权重。
*   **成熟账号阶段（高权重）**：当账号持续发布优质垂直内容（例如专注彩妆领域），系统会为其打上领域标签，并赋予更高的权重分。高权重账号发布的笔记，通常会获得更高的初始曝光基数。

权重是系统内部的评估机制，用户无法查看具体数值，但它切实影响着每篇笔记的冷启动流量。新号运营初期，不应过分追求单篇笔记的爆火，而应专注于通过合规、优质、垂直的运营来稳步提升账号权重。

## 总结

本节课中我们一起学习了小红书笔记的审核与分发机制。我们了解到，笔记发布后需经历**机器审核**、**去重检测**和**人工审核**三道关卡，任何违规行为都可能导致笔记无法正常曝光。同时，发布账号的**权重**直接影响笔记的初始流量，新号需要通过持续、合规、垂直的运营来积累权重，为后续的内容爆发打下基础。理解这些规则，是有效进行小红书运营的第一步。
