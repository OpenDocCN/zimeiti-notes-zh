# 拼多多运营实操教程：P15：直通车ROI及利润计算 💰

## 拼多多付费推广模式

![](img/872a6a105634dd95bd4e9bf2e377617d_3.png)

![](img/872a6a105634dd95bd4e9bf2e377617d_5.png)

拼多多平台主要有两种付费推广模式：**标准推广**和**全站推广**。了解它们的适用场景是制定有效推广策略的第一步。

![](img/872a6a105634dd95bd4e9bf2e377617d_6.png)

### 标准推广：新品启动利器 🚀

![](img/872a6a105634dd95bd4e9bf2e377617d_8.png)

上一节我们介绍了两种推广模式，本节中我们来看看标准推广的具体应用。标准推广适用于新品或新链接的初期推广阶段。其优势在于前期流量相对精准，有助于提升产品的转化率。

![](img/872a6a105634dd95bd4e9bf2e377617d_9.png)

![](img/872a6a105634dd95bd4e9bf2e377617d_10.png)

以下是创建标准推广计划的具体步骤：

1.  **进入后台**：登录拼多多商家后台，点击“标准推广”。
2.  **添加产品**：选择你需要推广的商品添加到计划中。
3.  **设置日限额**：根据你的预算设置每日最高花费。预算较少可设为100元，预算充足可设为200-300元。
4.  **设置出价**：需要设置四种出价：
    *   **成交出价**：这是核心出价，**不得超过你的单笔利润**。例如，商品售价50元，利润20元，则成交出价最高只能设为20元。
    *   **询单/关注/收藏出价**：系统会给出一个推荐出价区间。**前期不要直接采用推荐价**，建议按照以下公式设置：
        `出价 = 系统推荐最低出价 × 70%`
    按照以上设置创建计划后，先运行7天，目标是快速积累**20单以上**的成交。这是因为新计划有一个“学习期”，达到一定成交单数后，系统才能为你匹配更精准的人群标签。

![](img/872a6a105634dd95bd4e9bf2e377617d_13.png)

### 全站推广：追求稳定投产 📈

![](img/872a6a105634dd95bd4e9bf2e377617d_15.png)

当你的标准推广计划成功积累超过20单成交后，就可以考虑转向全站推广。全站推广与标准推广的核心区别在于出价方式：它不再让你设置具体的金额出价，而是要求你设定一个 **“目标投产比（ROI）”**。

## 核心计算公式详解

![](img/872a6a105634dd95bd4e9bf2e377617d_17.png)

![](img/872a6a105634dd95bd4e9bf2e377617d_18.png)

要玩转全站推广，你必须掌握以下几个核心计算公式。

![](img/872a6a105634dd95bd4e9bf2e377617d_19.png)

![](img/872a6a105634dd95bd4e9bf2e377617d_20.png)

![](img/872a6a105634dd95bd4e9bf2e377617d_21.png)

### 1. 实际投产比（ROI）

![](img/872a6a105634dd95bd4e9bf2e377617d_22.png)

实际投产比衡量的是广告花费带来的销售额效果。
**公式**：
`实际ROI = 交易额 / 总花费`
**示例**：直通车花费1000元，带来5000元交易额。
`实际ROI = 5000 / 1000 = 5`

![](img/872a6a105634dd95bd4e9bf2e377617d_24.png)

### 2. 保本ROI（盈亏平衡线）

![](img/872a6a105634dd95bd4e9bf2e377617d_25.png)

这是推广不亏不赚的临界点，也是全站推广初期设置目标投产比的重要参考。
**公式**：
`保本ROI = 客单价 / 利润`
**示例**：商品售价100元，利润50元。
`保本ROI = 100 / 50 = 2`
这意味着，当你的计划ROI达到2时，理论上刚好保本。但实际运营中还需考虑其他成本。

### 3. 盈利计算

![](img/872a6a105634dd95bd4e9bf2e377617d_27.png)

要精确计算推广带来的真实利润，需要使用以下公式。
**公式**：
`盈利 = (交易额 - 退款金额) × 利润率 - 总花费`
**示例**：交易额5000元，退款500元，利润率50%，直通车花费1000元。
`盈利 = (5000 - 500) × 50% - 1000 = 1250元`

![](img/872a6a105634dd95bd4e9bf2e377617d_28.png)

### 4. 净投产（计入退款后保本ROI）

考虑到退款情况，我们需要计算一个更安全的保本ROI，即净投产。
**公式**：
`净投产 = 保本ROI / [(1 - 退货率) × 1.1]`
**示例**：保本ROI为2，假设退货率为10%（即0.1）。
`净投产 = 2 / [(1 - 0.1) × 1.1] = 2 / (0.9 × 1.1) ≈ 2.44`
这意味着，当你的计划ROI达到**2.44**时，即使存在退款，也能确保不亏损。

### 5. 最佳投产（理想盈利ROI）

![](img/872a6a105634dd95bd4e9bf2e377617d_30.png)

我们推广的最终目的是盈利，因此需要设定一个理想的ROI目标。
**公式**：
`最佳投产 = 净投产 × 1.3`
**示例**：净投产为2.44。
`最佳投产 = 2.44 × 1.3 ≈ 3.17`
因此，**ROI在2.44到3.17之间**是一个健康且盈利的状态。低于2.44可能亏损，高于3.17则盈利更佳。
