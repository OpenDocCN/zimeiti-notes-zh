# 新媒体运营教程：P31：29、运营工具篇-运营必备营销日历 📅

在本节课中，我们将要学习一个对运营工作至关重要的工具——营销日历。我们将了解它的定义、核心功能以及如何利用它来规划内容、蹭取热点，从而有效提升账号流量。

## 什么是营销日历？🤔

上一节我们介绍了运营需要关注热点，本节中我们来看看实现这一目标的具体工具。营销日历是一个根据日期，系统化展示每日热点事件、节日、纪念日等内容的工具。作为运营，了解并利用这些时间节点，是快速为短视频或内容获取流量的有效方式。

![](img/258a138c147d04381d607649a8f914cf_1.png)

## 营销日历的核心功能 🔍

营销日历的界面会清晰列出每一天的相关热点。这些热点可能涵盖历史人物纪念日、公司成立日、重要文化事件或国际节日等。

![](img/258a138c147d04381d607649a8f914cf_1.png)

以下是营销日历通常包含的热点类型：
*   **历史纪念日**：例如“中国物理学之父逝世周年”。
*   **企业事件**：例如“宝马公司成立”。
*   **文化首演/发布**：例如“《茶花女》在威尼斯首演”。
*   **国际节日与社会事件**：例如“国际妇女节”、“马航MH370事件纪念”。

运营者可以根据这些日期，提前规划与之相关的内容创作。

![](img/258a138c147d04381d607649a8f914cf_3.png)
