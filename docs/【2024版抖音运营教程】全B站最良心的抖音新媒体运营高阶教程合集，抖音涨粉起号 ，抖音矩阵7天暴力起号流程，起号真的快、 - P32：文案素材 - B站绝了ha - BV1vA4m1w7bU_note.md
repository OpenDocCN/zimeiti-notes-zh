# 抖音运营教程：P32：文案素材与视频制作 🧩

在本节课中，我们将学习如何为视频创意寻找文案素材，并了解一种将文字内容转化为视频的实用方法。我们将介绍一个强大的工具，并讲解如何合法、高效地利用这些素材进行二次创作。

![](img/2f243157882216b876d8cb5b84880c31_1.png)

上一节我们介绍了视频创意的确定方法，本节中我们来看看如何为已确定的创意寻找合适的文案内容。

![](img/2f243157882216b876d8cb5b84880c31_3.png)

如果视频创意已经确定，下一步就是搜索相应的文案。直接搬运他人视频素材并进行处理是一种方式。但如果希望根据特定创意搜索文案，并以另一种形式自主制作视频，则需要掌握寻找文案素材的方法。

![](img/2f243157882216b876d8cb5b84880c31_5.png)

![](img/2f243157882216b876d8cb5b84880c31_7.png)

文案素材方面，推荐一个实用的工具：**淘金阁**。淘金阁是一个网站。

![](img/2f243157882216b876d8cb5b84880c31_9.png)

以下是淘金阁网站的核心功能界面示意：
```
[自媒体全网搜索引擎界面]
```

![](img/2f243157882216b876d8cb5b84880c31_11.png)

该工具是一个“自媒体全网搜索引擎”。点击进入后，可以看到其中汇集了大量文章。

![](img/2f243157882216b876d8cb5b84880c31_13.png)

它搜集了众多内容平台的素材，来源渠道包括：
*   今日头条
*   百家号
*   企鹅号
*   趣头条
*   一点资讯
*   QQ看点
*   凤凰网
*   网易资讯
*   搜狐网
*   东方网
*   简书
*   快资讯

![](img/2f243157882216b876d8cb5b84880c31_15.png)

![](img/2f243157882216b876d8cb5b84880c31_17.png)

因此，淘金阁拥有一个庞大的文案素材库。

![](img/2f243157882216b876d8cb5b84880c31_19.png)

在该搜索引擎中，可以输入目标关键词进行查找。例如，搜索“星座”。

![](img/2f243157882216b876d8cb5b84880c31_21.png)

输入关键词并回车后，点击“开始抓取”按钮。

![](img/2f243157882216b876d8cb5b84880c31_23.png)

![](img/2f243157882216b876d8cb5b84880c31_25.png)

下方会展示抓取到的所有相关文章。我们的目的是寻找文案，所以这里提供的是文章素材。

![](img/2f243157882216b876d8cb5b84880c31_27.png)

![](img/2f243157882216b876d8cb5b84880c31_29.png)

将别人的文章素材转化为自己的视频，这与直接搬运他人视频有本质区别。这不属于搬运范畴。

![](img/2f243157882216b876d8cb5b84880c31_31.png)

我们可以随意点开一篇文章进行查看，例如“分手后也能挽回最放不下前任的三大星座”。

![](img/2f243157882216b876d8cb5b84880c31_33.png)

点击后即可直接查看文章全文，也就是我们所需的文案内容。

![](img/2f243157882216b876d8cb5b84880c31_35.png)

![](img/2f243157882216b876d8cb5b84880c31_37.png)

文章内会包含具体的星座分析，例如巨蟹座、双鱼座等描述。这些文案可以直接复制粘贴，用于自己的视频创作。

![](img/2f243157882216b876d8cb5b84880c31_39.png)

找到文案后，接下来就是视频制作环节。

现在介绍一种制作视频的形式。关于视频创意，之前提到的5118热词搜索以及本节课的淘金阁文案搜索，都是有效的工具，需要大家在课后多加练习使用。

接下来进入视频制作步骤。视频制作可以采用真人出镜拍摄，也可以采用音乐影视混剪等方式。音乐影视混剪可以使用专门的网站或录屏工具。之前分享过的台词混剪网站、以及如 **mvcat.com** 这类提供电影下载的网站，都可以作为影视混剪的素材来源。

![](img/2f243157882216b876d8cb5b84880c31_41.png)

本节课中我们一起学习了如何利用“淘金阁”等工具搜索文案素材，并了解了将文章合法转化为视频内容的基本思路。关键在于理解使用文字素材进行再创作与直接搬运视频的区别，并掌握多种视频制作形式的可能性。
