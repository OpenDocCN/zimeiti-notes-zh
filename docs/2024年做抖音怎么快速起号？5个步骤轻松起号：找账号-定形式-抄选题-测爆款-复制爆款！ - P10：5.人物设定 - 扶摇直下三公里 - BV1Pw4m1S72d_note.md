# 抖音快速起号教程：P10：人物设定 🎭

在本节课中，我们将学习抖音账号运营中至关重要的一环——人物设定。一个清晰、真实且富有吸引力的人设，是解决粉丝信任、实现裂变传播和高效变现的基础。

![](img/9997581d0caccd246ecccd3ccf40015b_2.png)

上一节我们介绍了如何测试爆款内容，本节中我们来看看如何为你的账号塑造一个成功的“人物角色”。

![](img/9997581d0caccd246ecccd3ccf40015b_4.png)

## 什么是账号人设？

![](img/9997581d0caccd246ecccd3ccf40015b_6.png)

在抖音中策划人设，就像在游戏中预先选择并设定一个角色。这包括角色的称呼、职业、技能点等属性。但与游戏角色不同的是，抖音中的人物角色是“人生角色”，因此需要具备更强的真实感。

![](img/9997581d0caccd246ecccd3ccf40015b_8.png)

无论是短视频还是直播，一个成功的IP能够解决粉丝信任问题，起到裂变传播的作用，并具备很强的变现能力。因此，我们运营账号、创作内容和策划选题的核心目的，就是在获取流量的同时打造IP。打造IP的核心，就是在特定场景或情境中，完成账号的“人格化”过程。

![](img/9997581d0caccd246ecccd3ccf40015b_10.png)

## 如何策划账号人设？

![](img/9997581d0caccd246ecccd3ccf40015b_12.png)

账号人格化，即我们通常所说的“账号人设”。对于人设的策划，可以从以下几个具体方面着手。

![](img/9997581d0caccd246ecccd3ccf40015b_14.png)

以下是策划人设的六个核心维度：

![](img/9997581d0caccd246ecccd3ccf40015b_16.png)

1.  **名字/姓氏/昵称**
    *   一个IP首先需要一个叫法，这是IP传播的名片。它就像一个品牌的名称或商标，至关重要。

![](img/9997581d0caccd246ecccd3ccf40015b_18.png)

2.  **职业/职务/职称**
    *   人格化必然要具备身份。这是用户认识这个人或IP的联想入口，例如：老师、教授、博士、律师。

3.  **称呼/社会关系**
    *   一个IP有了亲切的称呼，能快速拉近与用户的距离，起到“移情”作用。例如：XX哥、XX大爷、XX姥姥。

4.  **形象/第一印象**
    *   以用户的视角看待IP，使IP形象化，更容易占据用户心智。例如：大叔形象、萌萌哒、铁憨憨、甜美可爱、光头。

5.  **性格/爱好/特点**
    *   一个具有鲜明特点的IP，除了记忆点强，真实性也更高。过于完美的形象容易脱离大众。例如：暴躁、娘炮、爱吃、爱喝酒。

6.  **穿着打扮/财富地位**
    *   这属于社会刻板印象和大众普遍认知。借助用户的固有思维，可以快速为IP在用户心智中定位。例如：有钱代表会赚钱，开豪车代表成功人士，打扮土气可能代表老师。

## 如何提炼人设的“超级符号”？

将以上人设策划好后，就可以结合你自身的产品或服务优势，落实到账号搭建中（即抖音账号的“五件套”：头像、昵称、背景图、简介、视频封面）。

除了对账号IP进行人格化策划，还需要通过选题和内容策划，为你的人设提炼出易于记忆的“超级符号”。

以下是提炼超级符号的几个方向：

*   **画风（视觉）**：如特定的**场景**、**道具**、**服饰**。
*   **听觉**：如标志性的**BGM**、**特效音**、**配音声线特点**。
*   **文案**：能够反映优势且容易记忆的**提炼词**，例如：“XX方法，简单粗暴”。
*   **表情/情绪**：如标志性的**笑**、**惊恐**、**无奈**等表情包。
*   **原特色**：如**方言口音**、**错发音**、**网络语**、**慢条斯理**或**抑扬顿挫**的说话方式。
*   **动作**：习惯性的**肢体动作**，例如：**撩头发**、**扶眼镜**。

超级符号提炼好之后，在进行短视频创作时，就可以不断地在视频中重复呈现这些元素，强化观众的记忆。

本节课中，我们一起学习了抖音账号“人物设定”的核心方法与步骤。我们了解到，人设策划需要从名字、职业、形象、性格等多个维度构建一个真实、立体的“人生角色”，并在此基础上提炼出视觉、听觉、行为上的“超级符号”，通过持续的内容输出强化记忆点。一个成功的人设，是账号获得信任、实现传播与变现的坚实基础。
