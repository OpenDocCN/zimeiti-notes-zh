在本节课中，我们将通过实际操作，学习如何使用剪映App完成一条视频从导入到导出的完整剪辑流程。课程将涵盖视频裁剪、音频添加、字幕生成等核心步骤，并额外介绍一款实用的提词工具，帮助你高效制作口播视频。

上一节我们介绍了视频剪辑前的准备工作，本节中我们来看看如何使用剪映App进行具体的剪辑操作。

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_9.png)

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_11.png)

剪辑环节采用直接实操的方式进行。请准备好你的笔记，跟随步骤一起操作，以确保掌握每个细节。

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_13.png)

首先，请确保你下载的是**剪映（黑色图标版本）**，它由抖音官方推出，包含手机版和电脑版。本节课我们先讲解手机版的操作。

**第一步：导入与初步调整**

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_15.png)

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_17.png)

打开剪映App，点击“开始创作”，然后选择你需要剪辑的视频素材并“添加”到项目中。如果你的视频由多个片段（如生活Vlog的开头、中间、结尾）组成，建议先导入并剪辑好第一条，再点击“+”号导入第二条，以此类推，避免混乱。

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_19.png)

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_21.png)

以下是视频初步调整的步骤：
1.  **调整构图与比例**：点击时间轴上的视频片段，向右滑动底部工具栏，找到并点击“编辑”（方块图标）。
2.  **旋转画面**：在编辑菜单中，点击左侧的“旋转”功能，将画面调整到正确的方向。
3.  **裁剪比例**：点击第三个“裁剪”功能，你可以选择 `1:1`、`4:3`、`16:9` 等不同比例，以适应抖音等平台的发布要求。调整完毕后，点击对勾确认。

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_23.png)

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_25.png)

**第二步：修剪视频内容**

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_27.png)

初步调整后，需要修剪掉视频中无用的部分，如开场沉默、结尾冗余动作等。

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_29.png)

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_31.png)

播放视频，找到需要剪掉的部分。时间轴上的**白色竖线就是“剪刀”的定位点**。将竖线移动到需要分割的位置，然后点击视频片段，选择底部工具栏的“分割”功能。视频会被分成两段，选中不需要的那一段，点击“删除”即可。

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_33.png)

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_35.png)

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_37.png)

核心操作可以总结为：**选中 -> 分割 -> 选中 -> 删除**。重复此步骤，修剪掉所有多余部分。

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_39.png)

**第三步：添加与编辑背景音乐**

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_41.png)

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_43.png)

视频修剪完毕后，接下来添加背景音乐。点击底部工具栏的“音频”。

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_45.png)

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_47.png)

以下是添加音乐的两种方法：
*   **提取音乐**：如果你已使用“淘金阁”等工具生成了配音文件，可以点击“提取音乐”，直接从手机视频中导入音频。
*   **添加音乐**：点击“添加音乐”，App内提供了丰富的音乐库，你可以根据分类（如美食、美妆、萌宠）选择，或直接搜索歌曲。点击歌曲旁边的播放图标可以预览，选定后点击“使用”即可导入。

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_49.png)

如果音乐长度超过视频，同样使用 **选中 -> 分割 -> 选中 -> 删除** 的方法进行裁剪。你还可以选中音乐轨道，点击“音量”来调整背景音乐的大小，使其不掩盖人声。

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_51.png)

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_53.png)

**第四步：自动添加与修改字幕**

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_55.png)

添加完音频后，下一步是添加字幕。点击底部工具栏的“文字”，然后选择“识别字幕”。

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_57.png)

为了更精准，建议选择“仅视频”开始识别。系统会自动分析视频中的人声并生成字幕，这比手动打字高效得多。

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_59.png)

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_61.png)

识别完成后，检查字幕是否有错别字。**双击**字幕条即可进入编辑模式，进行修改。你还可以在这里调整字幕的字体、样式、颜色（例如，根据衣服颜色搭配字幕颜色），并将其拖动到画面中合适的位置。

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_63.png)

**第五步：导出高清视频**

在导出最终视频前，有一个关键设置能保证视频清晰度。点击右上角的“导出”按钮，在导出设置中：
*   **帧率 (fps)**：设置为 `30`。这是抖音平台最合适的帧率。
*   **分辨率**：尽可能拉满（如1080p或更高）。分辨率越高，视频越清晰。

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_65.png)

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_67.png)

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_69.png)

设置完成后，点击“导出”。**导出过程中请勿锁屏或切换程序**，耐心等待导出完成。导出后，可以直接通过剪映分享到抖音。

掌握了基础剪辑流程后，对于制作口播视频的同学，这里额外推荐一个能提升效率的工具。

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_75.png)

**口播提词工具：爱提词**

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_77.png)

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_79.png)

这款软件可以解决你记不住台词、需要反复背诵的问题。

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_81.png)

![](img/e15ff12037f2c4d46a8d9f16bb2fdd64_83.png)

1.  **添加台词**：打开“爱提词”App，点击右下角“+”号，创建新文稿。你可以直接输入或粘贴你的口播文案，并为其设置一个标题（如“情感语录”），然后保存。
2.  **使用提词器**：保存后，点击“开始提词”，选择“录制模式”。此时，你的台词会以悬浮窗形式显示在手机屏幕上，你可以一边看着台词，一边进行录制，无需背诵。
3.  **个性化设置**：在提词界面，点击中间区域的设置按钮，可以调整**字体大小**，方便近视的同学观看。点击左下角的“美颜”功能，可以开启瘦脸、大眼、磨皮等效果，让拍摄画面更美观自然，且效果比一些美颜相机更真实。
4.  **灵感来源**：App内还设有“爆款文案”板块，涵盖了职场、情感、生活等多个领域的文案素材，可供你参考和获取灵感。

本节课中我们一起学习了使用剪映App进行视频剪辑的完整流程，包括**视频导入与比例调整、内容修剪、音频添加、字幕识别与编辑，以及高清导出设置**。同时，我们还介绍了“爱提词”这款工具，它能有效帮助口播创作者解决记词难题并提升拍摄效果。记住核心操作 **选中 -> 分割 -> 删除**，并善用自动识别功能，你就能快速上手，制作出完整的短视频作品。
