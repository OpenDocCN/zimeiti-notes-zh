# 《抖音全套运营教程》：P9：产品与用户兴趣标签 🧩

在本节课中，我们将学习如何利用“九宫格选题法”，通过分析产品属性和用户兴趣标签，为抖音账号创作源源不断的视频选题。我们将以健身软件Keep为例，详细拆解这一方法。

上一节我们介绍了选题的重要性，本节中我们来看看如何系统地挖掘选题方向。核心思路是围绕两个中心点展开联想：**产品相关标签**和**用户兴趣标签**。

## 产品相关标签分析

首先，我们聚焦于产品本身。将你的核心产品（例如Keep）放在中心位置，围绕它联想其具备的所有特点和属性。

以下是围绕Keep这款健身软件可能联想到的产品标签：

1.  **线上健身软件**：产品的核心形态。
2.  **专业健身教练**：产品提供的核心服务或资源。
3.  **饮食搭配建议**：产品提供的附加价值。
4.  **减肥/塑形攻略**：产品解决的主要用户需求。
5.  **健身干货知识**：产品提供的教育内容。
6.  **科学运动方法**：产品倡导的理念。
7.  **打卡运动制度**：产品的运营或激励模式。
8.  **一二线城市用户为主**：产品的主要用户画像。

将这些关键词填入九宫格中产品周围的八个空格。这一步的目的是穷尽产品自身的所有卖点和特征。

## 用户兴趣标签分析

接下来，我们转向用户。将目标用户群体（例如“健身爱好者”）放在另一个九宫格的中心，开始联想这类人群可能感兴趣的其他事物。

以下是围绕“健身爱好者”可能联想到的兴趣标签：

![](img/26c06b2d39d45d1cc4b75666af1bd5a4_1.png)

1.  **健身房**：线下健身的场所。
2.  **健身装备**：如运动鞋、紧身衣等。
3.  **轻脂美食/减肥餐**：与健康饮食相关的内容。
4.  **旅行**：热爱生活的一种表现。
5.  **星座/明星**：普遍的娱乐兴趣点。
6.  **鸡汤文**：激励、情感类内容。
7.  **时尚穿搭**：与外形、品味相关。
8.  **生活技巧**：提升生活品质的内容。

这一步的关键在于**将用户标签从核心兴趣向外围扩展**。例如，从“健身”扩展到“热爱生活”，从而覆盖更广泛的内容领域。

## 生成选题：连连看组合法

现在，我们手头有两个填满的九宫格：一个产品标签格，一个用户兴趣标签格。如何生成具体的视频选题呢？答案是进行**组合联想**。

**核心操作**：从产品标签和用户兴趣标签中各选一个关键词，将它们“连接”起来，构思一个完整的视频标题或主题。

以下是几个组合示例：

*   **组合1**：`饮食搭配`（产品标签） + `旅行`（用户兴趣标签）
    *   **生成选题**：《外出旅行时，如何享受美食又能控制体重？》
*   **组合2**：`运动方法`（产品标签） + `明星`（用户兴趣标签）
    *   **生成选题**：《赵丽颖都在练的5个高效燃脂动作，赶紧学起来！》
*   **组合3**：`科学`（产品标签，来自“科学运动方法”） + `健身房`（用户兴趣标签）
    *   **生成选题**：《在健身房用错器械等于白练！这3个科学原则你必须知道。》

通过这种“连连看”的方式，你可以轻松组合出数十个甚至上百个选题方向。例如，`打卡制度` + `鸡汤文` 可以结合为《自律X天，我如何靠打卡遇见更好的自己？》。

## 优化与素材库建立

初学者在组合时可能会感到思路有限，这是正常现象。此时，可以借助搜索引擎工具进行拓展。

**操作建议**：将组合出的关键词（如“健身 科学 方法”）输入百度等搜索框，查看搜索联想和相关文章标题，它们能为你提供更丰富、更符合网络语感的选题句式。

将所有通过此方法产生的选题记录下来，就初步建立了你的专属**视频选题素材库**。这能有效解决“不知道拍什么”的创作瓶颈。

本节课中我们一起学习了“九宫格选题法”。我们首先分别梳理了**产品自身标签**和**用户兴趣标签**，然后通过**组合联想**的方式将二者连接，批量生成具体视频选题。这个方法的核心在于打破思维局限，通过结构化联想，让创意源源不断。记住，一个好的选题是连接产品价值与用户兴趣的桥梁。
