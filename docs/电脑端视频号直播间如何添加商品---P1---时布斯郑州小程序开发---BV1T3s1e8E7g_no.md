# 视频号直播运营：第1章：电脑端直播间商品添加与管理

在本节课中，我们将学习如何在电脑端为视频号直播间添加商品，即如何挂上“小黄车”，并对已添加的商品进行管理操作。

上一节我们介绍了课程主题，本节中我们来看看具体的操作步骤。

![](img/b6f4a2ec30394001e967bf677c53dcae_0.png)

## 登录与进入商品管理

首先，你需要登录视频号助手后台。在左侧导航栏中找到并点击“直播”选项，然后选择“直播商品管理”进入商品管理页面。

## 添加商品到直播间

想为直播间添加商品，请点击页面右上角的“添加商品”按钮。系统会展示你橱窗内的商品列表。

以下是添加商品的具体步骤：
1.  在商品列表中，勾选你希望添加到直播间的商品。
2.  点击列表下方的“添加”按钮。

操作完成后，所选商品就会出现在直播间的商品列表中。

## 管理直播间商品

商品添加成功后，你可以对它们进行多种管理操作，例如预热、改价、调整库存或删除。

![](img/b6f4a2ec30394001e967bf677c53dcae_2.png)

以下是关键的管理须知：
*   **改价功能**：只有来自**你自己微信小店**的商品才可以修改价格。
*   **橱窗商品限制**：直接从橱窗添加的商品无法在直播间内改价。若需改价，必须先将商品上架到你自己的微信小店中。

本节课中我们一起学习了通过电脑端为视频号直播间添加和管理商品的完整流程。核心操作包括登录后台、从商品橱窗中勾选添加，以及了解只有小店商品支持改价这一重要规则。
