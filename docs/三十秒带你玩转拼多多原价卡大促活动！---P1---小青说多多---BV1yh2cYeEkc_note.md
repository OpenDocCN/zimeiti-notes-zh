# 拼多多运营教程：P1：原价卡大促活动参与指南 🛒

在本节课中，我们将学习如何参与拼多多的“原价卡”大促活动。这是一种通过特定方式提报商品，从而在活动中以接近原价销售商品的运营技巧。教程将分步讲解，确保新手也能轻松理解和操作。

## 第一步：搜索并提报核心活动

首先，在商家后台的搜索框中，输入活动ID **21603** 进行搜索。这是一个要求商品打 **9.9折** 即可报名的评价活动。

如果你的后台未显示此活动，则搜索活动ID **21604**。该活动允许 **9.8折** 的商品报名。

若以上两个活动均未找到，请直接搜索活动ID **21605**。此活动接受 **9.7折** 的商品报名。

**核心操作**：以上三个活动，你只需根据自身情况，选择其中一个进行提报即可。

## 第二步：提报辅助活动

完成第一步后，我们来进行第二步以增加活动覆盖面。在搜索框中输入活动ID **21536**，这是搜索专区活动，直接提报你的商品。

此外，活动ID **20981** 是一个长期有效的活动，同样建议你直接提报。

## 第三步：关键设置实现“原价卡”

最后一步是实现“原价卡”效果的关键操作。请返回到你在第一步中提报的那个活动（即21603、21604或21605中的一个）。

在该活动的商品编辑页面，找到并修改你商品中**最低价的那个SKU**的价格。将其价格修改为与其他两个SKU的活动价格相同，然后提交提报。

通过以上设置，即可实现“原价卡”大促的效果。
