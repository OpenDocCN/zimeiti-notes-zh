# 抖音运营指南：P20：基础篇丨新手如何做好内容定位 🎯

在本节课中，我们将学习如何为抖音账号进行精准的内容定位。选题是内容创作的基础，一个好的选题方向能让你事半功倍，避免陷入“不知道发什么”的困境。

![](img/1e5cdf0145fc9fb95f1e90fea4d6e47b_1.png)

---

你现在为抖音内容流的泪，都是当初确定选题时，脑子里进的水。你是否为想选题想到头秃，不知道明天该发布什么内容？今天教你一套公式，一分钟帮你解决这个苦恼。

会找选题的人，能有更多时间专注于视频制作。每天发布三条，一个月就是九十条。依靠数量积累都能形成优势，根本不需要额外投入。而且，擅长选题的人能比你更精准地找到用户痛点，也更容易吸引精准粉丝驻足观看，从而快速为账号打上标签。

![](img/1e5cdf0145fc9fb95f1e90fea4d6e47b_3.png)

那么，有什么办法可以解决选题难题呢？

当然有。这是一套百万粉丝博主常用的选题技巧。为了方便理解，我将其总结为一个公式：**内容定位 + 用户痛点 + 借势热点**。

![](img/1e5cdf0145fc9fb95f1e90fea4d6e47b_5.png)

用好这个公式，你的内容方向一定不会跑偏。例如，你想和我一样做一个知识博主，就需要去挖掘行业内的关键词，例如：播放量、粉丝量、定位、选题等。这些都是客观关键词，不包含主观描述。

然后，在这个基础上寻找用户的痛点，例如：便捷、高效、技巧、教学、内容输出等。最后，去结合近期的热点话题。将这三个部分进行任意组合即可。

为了加深理解，我们把这个公式套用在其他行业再试试。例如美妆行业，内容定位可以是：美白、除皱、口红。用户痛点可以是：平价好用、明星同款、避坑扫雷。最后借势热点进行排列组合，就能产生诸如“迪丽热巴同款美白面膜推荐”、“七夕千万别送女朋友口红”等具体选题。

以上我们已经举了两个例子。至于举一反三，就交给你了。你可以根据自己所在行业的特性，将你的思考过程在评论区分享出来。

**本节课总结**

本节课我们一起学习了如何运用 **“内容定位 + 用户痛点 + 借势热点”** 这个核心公式来高效确定抖音内容选题。掌握这个方法，你可以系统性地挖掘用户需求，结合热点趋势，从而产出更精准、更易获得流量的内容，为账号快速成长打下坚实基础。
