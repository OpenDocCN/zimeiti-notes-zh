# 拼多多运营：P29：物流服务指标详解

在本节课中，我们将要学习拼多多店铺运营的核心指标之一——物流服务指标。理解并管理好这个指标，对于维持店铺健康、获取平台流量至关重要。

## 核心指标概述

上一节我们介绍了拼多多店铺运营有多个核心考核指标。本节中，我们来看看其中非常关键的一项：物流服务指标。平台会通过这项指标来评估店铺的物流履约能力。

## 指标定义与计算

物流服务指标主要考核两个数据：**物流服务异常订单数**和**物流服务异常率**。如果这些数据异常增高，可能导致店铺获得的推荐流量减少。

其考核标准基于近30天内，因以下原因产生的订单：
*   虚假发货
*   物流时效问题
*   发生有效的物流投诉
*   因物流服务DSR评分为1分而产生的售后退款

**需要注意**：除虚假发货外，收货地为新疆地区的订单，不计入服务异常订单。因此，发往新疆、西藏等偏远地区的包裹若出现物流延迟等问题，通常不会被记为异常。

### 异常率计算公式

物流服务异常率的计算方式如下：

**物流服务异常率 = (近30天异常订单总数 / (近30天异常订单总数 + 近30天成团订单数)) × 100%**

这个公式的核心是：异常订单数占（异常订单数+总成团订单数）的比例。

## 如何查看与管理

了解指标后，我们需要知道在哪里查看和处理相关问题。

以下是查看物流异常订单的步骤：
1.  登录拼多多商家后台。
2.  找到并进入“包裹中心”。
3.  在包裹中心，可以筛选查看各类异常订单，例如“揽收超时”、“派签超时”等。
4.  对于已出现异常状态的订单，务必及时点击“处理”并标记为“已处理”，以降低损失。

## 考核不达标的处罚

![](img/92db5e17fd18b9bfd2f7a4b6b267228a_1.png)

物流服务指标若未达到平台要求，可能会引发一系列处罚，严重程度取决于违规情况。

处罚主要分为以下两种情形：
*   **情形一**：若店铺的两项物流指标均持续高于同行均值的5倍，店铺将面临**二级限制**，处罚包括全店商品降权。
*   **情形二**：若两项指标持续高于类目均值的5倍，平台会采取更严厉的措施，如下架资源位、禁止上新资源位、全店商品降权等。

![](img/92db5e17fd18b9bfd2f7a4b6b267228a_3.png)

**二级限制的后果非常严重**，它意味着：
*   资源位被移除且禁止上新。
*   无法开广告推广。
*   不能上新或上架商品。
*   全店商品被降权。

这几乎会导致店铺运营陷入停滞状态，因此必须高度重视。

## 总结

本节课中我们一起学习了拼多多核心的物流服务指标。我们了解了该指标的定义、计算方法、在后台的查看路径，以及不达标可能带来的严重处罚。请务必记住，在拼多多平台，遵守规则是运营的基础，物流服务的质量直接影响店铺的流量和生存。在日常操作中，需要密切关注物流动态，及时处理异常，确保指标健康。
