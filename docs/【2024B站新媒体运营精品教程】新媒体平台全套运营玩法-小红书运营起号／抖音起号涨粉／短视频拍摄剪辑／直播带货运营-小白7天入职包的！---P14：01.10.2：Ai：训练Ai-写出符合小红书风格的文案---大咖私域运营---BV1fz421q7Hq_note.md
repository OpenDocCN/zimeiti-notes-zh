# 新媒体运营：P14：01.10.2：🤖 训练AI写出符合小红书风格的文案

在本节课中，我们将学习如何训练AI（如ChatGPT）来撰写符合小红书平台风格的文案。上一节我们介绍了如何让AI生成爆款标题，本节我们将深入探讨如何让AI理解并创作出符合平台调性的完整内容。

## 概述：从标题到内容的AI训练

上一节我们学习了如何让AI写出符合心意的爆款标题。这个方法不仅适用于小红书，也适用于抖音、哔哩哔哩等平台。每个平台都有其独特的关键词和文化，以及大众认可的标题形式。通过提炼这些重点关键词并建立自己的标题库，每次创作时让AI结合主题和关键词库进行生成，就能得到符合平台风格的标题。

关于AI写标题的内容就介绍到这里。接下来，我们需要重点研究如何用AI写出符合爆款结构或平台风格的内容体系。

## 核心：让AI理解平台文案风格

大部分内容ChatGPT都能生成，但如何写出符合特定平台风格的内容，需要讲究几个关键点。

第一个关键点是**文案风格**。为了让AI了解我们需要的风格，必须提供案例。你需要告诉AI：“我提供这些内容作为参考，希望你也写出类似风格的内容。” 这就需要给AI“喂”材料，即提供一定量的他人爆款文章。

![](img/f2c2ea3108aa884d4e3242eb615a9310_1.png)

以下是训练AI理解平台风格的核心步骤：

1.  **提供分析样本**：让AI分析爆款文章中的专用词汇、语气和平台用语（例如小红书的“宝子”等）。
2.  **指令明确**：明确告知AI需要学习和分析的具体方面。
3.  **持续反馈**：要求AI在每次学习后给出反馈，确认其已理解。

![](img/f2c2ea3108aa884d4e3242eb615a9310_3.png)

## 实战演练：训练AI撰写旅游攻略

我们来进行一个简单的实操，目标是训练AI撰写一篇关于“张家界5日游”的小红书风格攻略。

首先，我们需要给AI清晰的指令。以下是一个示例指令：

![](img/f2c2ea3108aa884d4e3242eb615a9310_5.png)

![](img/f2c2ea3108aa884d4e3242eb615a9310_6.png)

```
接下来我会给你提供一些小红书上的爆款笔记文案。你需要根据这些文案，分析小红书的文案撰写风格、特定的平台用语、词汇、语气等。每次我提供文案后，请告诉我你已经理解并学习到了，同时给出你对这篇文案的风格、语气、类型等相关分析。
```

![](img/f2c2ea3108aa884d4e3242eb615a9310_8.png)

这个指令明确了AI的任务，并要求其给出正向反馈，让我们知道它是否已学习。

![](img/f2c2ea3108aa884d4e3242eb615a9310_10.png)

### 第一步：提供学习材料

我们以“张家界旅游攻略”为例，开始向AI提供文案材料。请注意，提供的材料应具有鲜明的平台风格特色，而非通用的行程列表（如“第一天、第二天”）。

![](img/f2c2ea3108aa884d4e3242eb615a9310_12.png)

*   **材料一**：一篇以个人游玩经历为主线，强调“血泪教训”和“不踩坑建议”的攻略。这种文案能增加真实感和亲近感。
*   **材料二**：一篇使用“姐妹们”等平台常用称呼，语气活泼、带有强烈个人分享色彩的种草笔记。
*   **材料三**：一篇情绪鲜明、带有“避坑”或“吐槽”性质的文案，能体现用户的不满或惊喜等真实情感。

**关键点**：在“喂”资料时，目标是让AI理解小红书平台分享类内容特有的表述方式（如第一人称叙述）、语气以及“宝子”、“绝绝子”等平台用语。应选择风格各异的文案，而非千篇一律的行程清单。

### 第二步：要求AI进行创作

在提供多篇范文并得到AI的学习反馈后，即可发出创作指令。例如：

```
请结合你的学习，帮我写一篇具有爆款潜质的“张家界5日游攻略”小红书笔记。内容需涵盖丰富的小红书风格元素和表情。同时，请为这篇笔记生成十个符合小红书旅游赛道风格的爆款标题。
```

### 第三步：评估与优化AI输出

AI生成的初稿可能不尽如人意。例如，它可能只罗列了“天门山景区”、“国家森林公园”等景点名称，描述得像诗歌一样优美，但缺乏实用信息。

一篇优秀的旅游攻略需要：
*   **具体路线**：清晰的每日行程安排。
*   **亮点与避坑点**：每个景点的特色、必体验项目以及需要避免的“坑”。
*   **个人化理由**：为什么推荐某个景点或路线，而不仅仅是描述。

如果AI输出不理想，可能的原因包括：
1.  提供的学习材料质量不足或风格单一。
2.  指令不够具体。可以尝试分步指令，例如先让AI列出张家界所有主要景点，再针对每个景点规划路线、分析特色和避坑点。
3.  模型能力限制。付费的GPT-4模型通常比免费的GPT-3.5更聪明，理解力和创作力更强。

### 关于标题生成的补充

在内容创作后直接让AI生成标题，效果可能不如专门训练。更好的做法是：
**开启一个新的对话**，运用上一节课专门训练标题的指令和关键词库，来为这篇特定的“张家界5日游攻略”生成标题。这样生成的标题通常更精准、更具爆款潜力。

## 总结与核心要点

本节课我们一起学习了如何训练AI撰写符合平台风格的文案。

**核心要点总结**：
1.  **明确指令是关键**：清晰告诉AI你的需求和学习目标。
2.  **“喂”对材料是基础**：提供多样化的、具有鲜明平台风格的爆款文案作为学习样本。
3.  **分步优化是方法**：创作可以分步进行（如先列框架，再丰富细节），标题生成最好单独训练。
4.  **持续迭代是过程**：AI训练不是一蹴而就的，需要根据输出结果不断调整指令和输入材料。

**核心公式**：
`优质AI文案 = 清晰指令 + 高质量风格样本 + 针对性反馈与优化`

通过本课的方法，你可以将AI培养成理解你所在领域（如美妆、考研、科技等）和平台调性的“专属助手”，从而高效产出符合要求的优质内容。
