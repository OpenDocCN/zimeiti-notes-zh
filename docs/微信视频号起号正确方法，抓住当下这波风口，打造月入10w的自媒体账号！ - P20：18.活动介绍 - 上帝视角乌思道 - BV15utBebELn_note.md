# 微信视频号运营教程：P20：18.活动介绍

## 概述
在本节课中，我们将详细介绍一项由微信视频号官方发起的“商家联盟”活动。该活动旨在为视频化带货电商提供支持，包含官方提供的视频素材、流量扶持以及现金激励。我们将逐一解析活动的核心内容、参与方式以及具体操作步骤。

## 活动背景与负责人
为促进2024年中国行业的新发展，并为视频化带货电商奠定技术基础，微信视频号特推出“商家联盟”活动。本次活动由短视频教研部负责人张良老师全程对接。

本次活动以公平公正为核心理念。张老师作为本次活动的全国总负责人，是官方指定的唯一对接人。

![](img/a3d041e88b9d3d696990107235fbcb15_4.png)

## 活动核心扶持内容
上一节我们介绍了活动的背景，本节中我们来看看活动提供的具体扶持内容。官方为参与者准备了多项实质性支持。

![](img/a3d041e88b9d3d696990107235fbcb15_6.png)

以下是本次活动提供的核心资源：

1.  **官方视频素材**：针对参与者缺乏优质拍摄素材的问题，官方已预先准备好大量高质量产品视频。每个参与账号可获得30条视频素材，满足每日发布一条的需求。
    ![](img/a3d041e88b9d3d696990107235fbcb15_1.png)
    ![](img/a3d041e88b9d3d696990107235fbcb15_2.png)

![](img/a3d041e88b9d3d696990107235fbcb15_8.png)

2.  **流量扶持**：每个参与账号将获得90万的流量扶持。这能有效解决视频发布后曝光量不足、商品卖不出去的担忧。

![](img/a3d041e88b9d3d696990107235fbcb15_10.png)

## 素材库详解与使用
了解了核心扶持内容后，我们具体看看官方素材库的构成和使用方法。素材库品类丰富，操作便捷。

![](img/a3d041e88b9d3d696990107235fbcb15_12.png)

官方素材库覆盖多种商品类别，例如中秋节礼品、餐巾纸、牛奶、火锅底料、洗漱手套、不锈钢杯等。

## 现金激励与结算规则
除了素材和流量，本次活动还提供了直接的现金激励，以增强参与者的信心。

![](img/a3d041e88b9d3d696990107235fbcb15_15.png)

活动提供保底收益：参与者每发布一条符合要求的带货视频，即可获得300元奖励。结算周期为“今天发布，明天到账”，采用日结模式。
![](img/a3d041e88b9d3d696990107235fbcb15_15.png)

根据公式计算，若完成30条视频的发布任务，总收益为：
`300元/条 * 30条 = 9000元`

## 活动期限与参与条件
需要注意的是，本次活动设有明确的期限，并非长期开放。

本次活动于1月13日正式开始，并计划于8月10日结束。活动由微信视频号官方发起，张良老师为指定负责人。
