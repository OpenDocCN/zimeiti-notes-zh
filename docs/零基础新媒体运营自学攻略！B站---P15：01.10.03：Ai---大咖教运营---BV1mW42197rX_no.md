# AI新媒体运营实战：P15：01.10.03：AI训练与爆款笔记生成 🚀

## 概述
在本节课中，我们将学习如何训练AI（以ChatGPT为例）分析爆款笔记，并基于分析结果生成具备爆款潜质的新笔记。我们将从提供爆款文案、引导AI分析，到最终生成完整笔记大纲和文案，一步步拆解这个过程。

上一节我们介绍了通过“喂资料”的方式让AI理解特定平台的风格和用语。虽然上一节以旅游攻略为例的成果可能不理想，但方法是正确的。关键在于为AI提供足够且优质的样本，让它从中学习。

本节中，我们来看看如何通过分析具体的爆款内容，让AI学习其成功要素，并最终产出新的笔记。

## 第一步：引导AI分析爆款笔记

![](img/405e4424465eb6f05caf664a3f06a8e2_5.png)

首先，我们需要找到目标领域的爆款笔记作为分析样本。本节以“考研”领域为例。

以下是引导AI进行分析的核心指令要点，我们需要让AI关注以下几个维度：

![](img/405e4424465eb6f05caf664a3f06a8e2_7.png)

![](img/405e4424465eb6f05caf664a3f06a8e2_9.png)

1.  **爆款因素**：分析内容为何能成为爆款。
2.  **整体结构**：
    *   **开篇**：用什么方式吸引用户。
    *   **中间**：用什么内容戳中用户痛点，提升停留时长。
    *   **结尾**：用什么“钩子”促使用户互动（如私信、关注）。
3.  **关键词分析**：笔记中是否植入了特定关键词，其出现频率和铺设结构如何。

我们可以将指令整合如下：
```
请你分析提供的小红书考研爆款笔记文案。
1. 分析其爆款因素。
2. 分析整体结构：开篇如何吸引用户；中间如何戳中痛点；结尾如何引导互动。
3. 分析笔记是否结合中文分词法设计了关键词。如有，指出关键词及其出现频率；如无，请说明。
```

![](img/405e4424465eb6f05caf664a3f06a8e2_11.png)

将找到的爆款文案内容粘贴给AI，并发送上述指令。

## 第二步：整合分析结果

![](img/405e4424465eb6f05caf664a3f06a8e2_19.png)

![](img/405e4424465eb6f05caf664a3f06a8e2_20.png)

![](img/405e4424465eb6f05caf664a3f06a8e2_22.png)

在向AI提供了多篇爆款笔记并获取分析报告后，我们需要让AI对分析结果进行整合与去重。

请向AI发送如下指令：
```
请将你之前分析过的所有考研爆款笔记的爆款因素总结一下，整合成一个清晰的列表。
```

![](img/405e4424465eb6f05caf664a3f06a8e2_24.png)

AI可能会遗漏部分信息，我们可以提醒它：
```
我前面一共让你分析了四篇笔记，请重新回顾并总结。
```

通过这一步，我们可以得到一个关于“考研爆款笔记”的通用成功要素清单，例如：情感共鸣、逆袭故事、明确备考计划、针对性建议、时间节点、资源技巧分享、正能量鼓励、使用表情符号等。

## 第三步：基于爆款要素生成笔记大纲

获得爆款要素清单后，我们可以指令AI基于这些要素，为新笔记规划一个具备爆款潜质的大纲。

指令如下：
```
我需要写一篇关于考研的小红书笔记。请结合你前面总结的爆款笔记因素，帮我梳理一个具备爆款因素的笔记大纲。每个大纲节点需要进一步细分出更小的要点。在我确认大纲没问题后，再生成具体文案。
```

AI生成的大纲可能包含：吸引人的开篇、主题介绍、专业选择分析、各科目备考规划、时间管理、心态调整、互动引导等部分。

---

## 第四步：根据大纲生成完整文案

![](img/405e4424465eb6f05caf664a3f06a8e2_30.png)

对大纲满意后，我们就可以让AI填充内容，生成完整的笔记文案。此时需要给出更具体的写作要求。

指令示例如下：
```
请按照你提供的大纲，写出具体的笔记文案。
要求：
1. 文案结构需严格遵循爆款笔记的因素和结构。
2. 需要包含一个简单的“逆袭小故事”来引入读者情绪。
3. 文案中需包含一定量的emoji表情。
4. 文案中需结合中文分词法的方式融入关键词，关键词是：“考研”、“考研政治”。
```

AI会根据要求生成一篇包含情感共鸣、个人经历、详细建议和鼓励话语的完整文案。生成的文案在专业度和结构上通常已具备基础，但可能仍需根据个人风格和具体需求进行优化调整。

## 总结与进阶提示

本节课我们一起学习了利用AI辅助创作爆款笔记的完整流程：**提供样本 -> 分析要素 -> 总结规律 -> 生成大纲 -> 产出文案**。

![](img/405e4424465eb6f05caf664a3f06a8e2_32.png)

*   **流程核心**：本质是让AI从优秀案例中学习模式，并应用该模式进行创作。
*   **指令精度**：与AI对话时，用词需准确。例如，“包含丰富的emoji”和“包含一定量的emoji”会导致不同结果。
*   **模型差异**：使用更强大的模型（如GPT-4）或定制化AI助手（如GPTs），可以获得更优质、更符合个性化需求的输出。你可以为定制助手提供专属的排版风格、常用表情包和文案范例，使其创作更贴合你的需求。

![](img/405e4424465eb6f05caf664a3f06a8e2_34.png)

通过本课的方法，你可以将AI训练成得力的内容创作助手，高效产出符合平台调性、具备爆款潜质的笔记草稿，为你的新媒体运营工作大幅提效。
