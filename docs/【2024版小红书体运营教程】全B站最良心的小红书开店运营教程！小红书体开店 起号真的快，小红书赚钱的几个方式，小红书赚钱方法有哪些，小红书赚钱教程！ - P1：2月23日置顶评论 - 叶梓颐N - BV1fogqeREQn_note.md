## 开店流程 📝
上一节我们介绍了课程的整体框架，本节中我们来看看如何开设你的小红书店铺。开店是启动小红书电商的第一步，流程非常简单。

1.  打开小红书应用，点击左上角的菜单图标。
2.  选择“创作中心”。
3.  点击“更多服务”选项。
4.  选择“店铺”，点击“立即入住”。
5.  按照官方要求提交资料完成入住。

**店铺类型说明**：
小红书店铺主要分为个人店和企业店。
*   **个人店**：包含“个人店”和“个体工商户店”。
    *   **个人店**：仅需身份证和1000元保证金。
    *   **个体工商户店**：需要身份证、营业执照和1000元保证金。
*   **企业店**：包含普通企业店、专卖店和旗舰店。均需营业执照，并需缴纳600元年费。

**重要提示**：对于初学者，建议直接选择**个人店**。经过实测，平台目前并未对企业店有流量倾斜。个人店与企业店在流量获取上效果相同。

![](img/e6f4ab59d0e1ae1722ab12520f1d9e68_5.png)

**开店成本**：
以下是开店涉及的主要成本：
*   **保证金**：1000元（不做可退）。
*   **平台扣点**：约为营业额的 **5%**。
*   **支付渠道费**：每单约 **0.7%**。
*   **流动资金**：初期可采用无货源模式，准备2000-3000元即可启动。

**核心观点**：小红书不适合“店群”模式。因为其电商自然流量占比低，铺再多店铺效果也有限。正确的做法是建立“账号矩阵”，通过多发笔记来获取流量，这将在后续章节详细说明。

## 选品策略 🎯
成功开店后，下一步就是决定卖什么。选品是电商成功的关键，但方法并不复杂。核心逻辑是：利用其他成熟平台的爆款数据来辅助决策，而不是凭空猜测。

![](img/e6f4ab59d0e1ae1722ab12520f1d9e68_7.png)

以下是三个推荐的选品渠道：

1.  **电商平台（如淘宝、京东）**：寻找符合小红书平台调性（如高颜值、新奇特、生活方式类）的爆款产品。已有案例显示，将其他平台的爆款直接搬运到小红书，第一篇笔记就可能出单。
2.  **抖音精选联盟**：查看抖音精选联盟的排行榜，了解当前热卖商品。筛选出符合小红书用户画像的产品，直接进行测试。
3.  **拼多多销量排行榜**：在拼多多的类目销量排行榜中，选择适合小红书的品类，将其中的爆品搬运到小红书进行测试。

**选品公式**：
`成功选品 = 其他平台爆款 × 与小红书用户匹配度`

我们只需判断排行榜上的爆品是否有可能匹配小红书用户（主要是年轻女性、追求品质和生活方式的人群）。如果匹配，就搬运过来测试。这相当于借助了大卖家的市场验证能力，是最高效、最稳妥的选品方法。

## 流量获取与内容创作 💡
产品上架后，最重要的环节就是获取流量。小红书本质是“种草”社区，店铺超过90%的流量都来自于发布的笔记。因此，**多发优质笔记远比多开店铺更重要**。

流量通路是：`用户看到笔记 → 对产品产生兴趣 → 进入店铺购买`。

**如何创作有效的种草笔记？**
直接发布硬广效果很差。必须从**素人（真实用户）视角**切入，创作真实的体验分享内容。

以下是制作种草笔记的简单方法：

1.  **素材来源**：去淘宝、京东、抖音等平台的商品“买家秀”或评论中，寻找好看、真实的图片或视频素材。
2.  **文案撰写**：基于买家的真实评价进行修改和润色。这些评价本身就是用户的使用体验，天然适合作为种草文案。

![](img/e6f4ab59d0e1ae1722ab12520f1d9e68_9.png)

**核心代码（比喻）**：
```plaintext
# 错误示范：硬广
post_content = “快来买！这款产品超棒！#广告”

# 正确示范：素人种草
post_content = “无意中淘到的宝藏！用了半个月，皮肤真的变好了…#好物分享 #真实体验”
```

当单个账号的运营流程（SOP）跑通后，可以通过批量注册小红书账号、批量发布笔记的“账号矩阵”方式，为店铺引流，放大收益。
