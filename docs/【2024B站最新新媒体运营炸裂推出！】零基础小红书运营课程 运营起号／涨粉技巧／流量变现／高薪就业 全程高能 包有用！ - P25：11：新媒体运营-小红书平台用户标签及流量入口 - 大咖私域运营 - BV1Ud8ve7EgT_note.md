## 用户标签的来源与判定

小红书系统为用户打标签，主要依据是用户的**浏览行为**。用户刚注册账号时所选择的兴趣范围，参考价值不大。它只是平台在初期对用户内容方向的一个引导，只能进行初步筛选，无法进入系统最深层的用户标签池。

### 1. 点击率
**点击率**是指系统在发现页给用户推送多个内容后，用户点击各类内容的概率。例如，系统推送了明星、美妆、家居、母婴等内容。用户点击美妆内容三次，家居内容两次，母婴内容一次或未点击。系统便能判定该用户对母婴内容不感兴趣，可能不是宝妈；而对美妆内容感兴趣，后续会持续推送更多美妆领域的内容进行试探。通过点击率的逻辑，系统能为用户打上**强关联**和**弱关联**的标签。

### 2. 停留时长
**停留时长**是另一个关键行为指标。例如，用户虽然常点击美妆内容，但点进去很快就划走，说明这些具体笔记可能不是他喜欢的。系统会继续推送美妆领域内其他细分内容（如上粉、护肤等）进行测试。反之，如果用户偶然点击了一个平时很少点的家居类笔记，并停留了很长时间，仔细阅读图片和评论，系统就可能判断该用户近期有装修或购买家居用品（如盘子）的需求，从而调整标签，推送相关商品或笔记。

### 3. 页面浏览量（PV）
**PV** 可以理解为用户浏览页面的深度。例如，用户不仅看了一篇装修笔记，还点进了博主主页，并连续看了第二、第三、第四篇笔记。这个浏览页面数就是PV。如果账号主页内容精致，就能提升用户的浏览深度。这个数值对账号健康度评分有非常大的帮助。浏览深度足够时，还可能引发**关注**行为，这对账号质量是永久的提升。

### 4. 互动行为
互动行为包括**点赞、收藏、评论、关注**。这些数据是我们在创作每篇笔记时都需要努力提升的。互动数据高，笔记获得进入更大推荐流量池的几率就更高，同时也能带动账号内其他笔记的流量，并提升粉丝数。

以下是各流量入口的详细介绍：

![](img/4d36f30f8c203a702f6fff1af3ffbe1a_148.png)

![](img/4d36f30f8c203a702f6fff1af3ffbe1a_150.png)

![](img/4d36f30f8c203a702f6fff1af3ffbe1a_156.png)

![](img/4d36f30f8c203a702f6fff1af3ffbe1a_158.png)

*   **发现页**：这是最主要的流量来源。发现页呈现的内容包括：
    *   用户已关注的达人发布的内容。
    *   系统根据用户标签（点击率、停留时长等）判断其可能喜欢的内容。
    *   系统为了平台内容多样化，偶尔推荐的一些用户可能不感兴趣或未接触过的内容。
*   **关注页**：这里只展示用户**已关注博主**发布的内容。对于粉丝而言，这是查看强关联博主更新的直接入口。
*   **附近页**：此入口根据手机定位，推送**附近及同城**的相关内容。内容通常以美食、本地旅游、线下店铺等强地域特征的需求为主。适合线下实体店（如美容院、餐厅）进行本地推广。
*   **购物页**：这是小红书向电商方向发展的重要入口，直接匹配用户的购物需求。目前，更多用户习惯通过搜索笔记来“种草”，直接进入商城购物的人群仍在培养中。

作为内容创作者（博主），我们需要平衡自己发布内容的价值：既要能为**用户**提供有价值的信息，也要能为**平台**留住用户、增加活跃度。平台则负责为**用户**精准匹配内容，同时为**博主**提供展示和变现的渠道。这三者形成了一个相互依存、共同成长的串联关系。

本节课中我们一起学习了小红书平台判定用户标签的核心依据——浏览行为（包括点击率、停留时长、PV和互动），以及笔记进行分发的四大流量入口（发现页、关注页、附近页、购物页）。理解这些机制，是做好小红书运营、实现精准引流和内容变现的基础。
