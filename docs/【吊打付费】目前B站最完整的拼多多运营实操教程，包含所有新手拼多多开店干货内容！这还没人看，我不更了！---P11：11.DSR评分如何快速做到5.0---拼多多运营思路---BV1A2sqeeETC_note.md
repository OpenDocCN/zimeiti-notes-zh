## 概述

![](img/1f50058788879bf0f89ac502d3a95d61_4.png)

![](img/1f50058788879bf0f89ac502d3a95d61_5.png)

DSR评分由商品描述、物流服务和卖家服务三部分组成，是影响店铺排名和消费者信任的关键指标。一个高的DSR评分能显著提升店铺的免费流量。本节课程将分享三种经过验证的、可自行操作的提分方法。

![](img/1f50058788879bf0f89ac502d3a95d61_6.png)

## 方法一：报名“百万爆款”营销活动

上一节我们了解了DSR评分的重要性，本节中我们首先来看看如何通过官方活动提升评分。第一种方法是报名平台营销活动，利用活动流量快速积累有效订单和好评。

![](img/1f50058788879bf0f89ac502d3a95d61_8.png)

以下是具体操作步骤：

1.  **进入商家后台**：登录拼多多商家后台。
2.  **找到活动入口**：在后台找到“店铺营销”，点击“营销活动”，寻找名为“**百万爆款**”的活动。
3.  **检查报名资格**：若无法报名，通常有两种原因：
    *   未缴纳店铺保证金。
    *   店铺领航员评分低于30%。若领航员评分过低，可先通过“批发”渠道进行交易以提升评分。
4.  **设置并提交活动**：该活动折扣力度较大，通常3-5周能快速起量。若仅为提升DSR评分，可选择满足活动最低要求的库存量进行提交，等待平台审核。

> **核心概念**：**领航员评分**是拼多多对店铺综合服务能力的考核指标，需保持在一定水平以上才能参与部分活动。

## 方法二：利用“多多进宝”推广渠道

![](img/1f50058788879bf0f89ac502d3a95d61_12.png)

通过营销活动可以快速起量，但若想更自主地控制提升过程，我们可以利用“多多进宝”这个官方推广工具。接下来，我们看看如何通过它来积累有效订单。

![](img/1f50058788879bf0f89ac502d3a95d61_14.png)

以下是具体操作步骤：

![](img/1f50058788879bf0f89ac502d3a95d61_15.png)

![](img/1f50058788879bf0f89ac502d3a95d61_16.png)

1.  **开通并设置推广**：在商家后台找到“多多进宝”，点击“推广设置”。选择“单品推广”，创建推广计划，添加需要提升DSR评分的商品链接。
2.  **设置优惠与佣金**：进入设置页面后，将优惠券面额和佣金比例均设置为**最低**要求，然后点击“立即推广”。
3.  **获取推广链接**：计划创建成功后，状态显示为“推广中”。在此页面找到“复制链接”，复制该短链接。
4.  **转换并分享链接**：
    *   访问“多多进宝”官网，注册成为“团长”账号。
    *   登录后，在“推广管理”中找到“转链工具”，将复制的短链接转化为**推广链接**。
    *   将此推广链接分享给他人（如亲友、社群），引导他们通过此链接下单购买。

通过“多多进宝”官方渠道产生的订单，大部分会被记为有效订单，从而安全、有效地帮助提升DSR评分。

![](img/1f50058788879bf0f89ac502d3a95d61_18.png)

![](img/1f50058788879bf0f89ac502d3a95d61_19.png)

> **核心概念**：**多多进宝**是拼多多的官方营销推广平台，卖家设置佣金，推广者（团长）帮助推广商品，成交后按佣金结算。

## 方法三：设置“评价有礼”激励工具

对于已有一定订单基础的店铺，“评价有礼”是一个激励买家主动给出好评的有效工具。本节我们来看看如何利用这个营销工具来优化评价管理。

![](img/1f50058788879bf0f89ac502d3a95d61_21.png)

![](img/1f50058788879bf0f89ac502d3a95d61_22.png)

![](img/1f50058788879bf0f89ac502d3a95d61_23.png)

以下是具体操作步骤：

![](img/1f50058788879bf0f89ac502d3a95d61_24.png)

1.  **找到营销工具**：在商家后台，进入“店铺营销”下的“营销工具”。
2.  **创建“评价有礼”活动**：在工具列表中找到“评价有礼”，点击创建。
3.  **设置返现金额与预算**：
    *   根据商品客单价和自身利润空间，设置评价返现的金额（例如0.8元、1元、2元等）。
    *   设置活动的总预算。
4.  **配合引流（可选）**：对于店铺订单量较少的卖家，可以配合开通“直通车”广告，引入更多真实流量和订单，再通过“评价有礼”活动激励好评，双管齐下提升DSR评分。

当DSR评分提升至高位后，店铺将能吸引更多的平台免费推荐流量。

![](img/1f50058788879bf0f89ac502d3a95d61_26.png)

> **核心概念**：**评价有礼**是拼多多提供的官方工具，卖家可设置现金奖励，激励买家在收货后发表带图好评。

2.  利用**“多多进宝”**推广渠道，通过官方合作方式获取安全有效的订单与评价。
3.  设置**“评价有礼”**营销工具，直接激励已购买家留下好评。
