# 微信公众号运营：P61：4.04-有赞微页面建设 🛠️

在本节课中，我们将学习如何利用有赞平台搭建店铺的微页面。我们将理解商品、页面与店铺之间的关系，并掌握创建、编辑和链接微页面的具体操作步骤。

---

## 概述

上一节我们介绍了有赞店铺搭建的第一步——上传商品。本节中，我们来看看如何将这些商品组织起来，构建一个完整的店铺页面。核心在于理解 **商品组成页面，页面组成店铺** 的逻辑。

---

## 微页面的概念与创建

![](img/8b974a6eb6694bafdc5a4818ea217342_1.png)

首先，我们需要明确微页面的作用。在有赞后台的“店铺”模块中，找到“内容创作-微页面”功能。微页面就像一个网页，你可以选择模板，并在其中填充内容（主要是商品）。

![](img/8b974a6eb6694bafdc5a4818ea217342_3.png)

以下是创建微页面的步骤：
1.  点击“新建微页面”。
2.  从系统提供的多种模板中选择一个适合你业务类型的模板（例如，教育类可以选择简洁、知识展示型的模板）。
3.  点击“使用模板”后，进入编辑页面。
4.  为页面设置名称和描述，例如名称设为“新媒体书籍”，描述设为“必读经典新媒体书籍”。
5.  为页面创建分类，便于管理。例如，可以创建一个名为“书籍”的分类，描述为“互联网好书推荐”。

![](img/8b974a6eb6694bafdc5a4818ea217342_5.png)

![](img/8b974a6eb6694bafdc5a4818ea217342_7.png)

**核心关系公式**：
`单个商品 + 单个商品 + ... = 微页面`
`微页面A + 微页面B + ... = 店铺`

---

## 编辑与定制微页面

选择了模板后，页面上的内容（如图片、文字）都是模板自带的，我们需要将其替换为自己的素材。

![](img/8b974a6eb6694bafdc5a4818ea217342_9.png)

![](img/8b974a6eb6694bafdc5a4818ea217342_11.png)

以下是替换页面内容的方法：
1.  点击页面上的图片或文字模块进行编辑。
2.  对于图片，需要上传符合尺寸要求（例如750x350像素）的自有图片。你可以使用设计工具制作，或在网络上寻找合适的素材图片。
3.  上传图片后，可以为其设置标题。
4.  最关键的一步是设置**链接跳转**。点击图片后，可以设置点击该图片后跳转到的目标。

![](img/8b974a6eb6694bafdc5a4818ea217342_13.png)

**链接跳转代码示例（逻辑描述）**：
```
当用户点击[图片A]时，执行跳转至[商品详情页X]
```

![](img/8b974a6eb6694bafdc5a4818ea217342_15.png)

## 理解页面与商品的链接逻辑

页面不仅仅是商品的静态展示，更重要的是提供交互入口。一个设计良好的店铺首页（本身也是一个微页面），其上的各个模块应该能引导用户前往更具体的商品页面或其他活动页面。

![](img/8b974a6eb6694bafdc5a4818ea217342_17.png)

我们可以这样理解结构：
1.  首先，由不同的商品组合成一个展示页面（例如“热门书籍推荐”页面）。
2.  接着，不同的页面（如“首页”、“书籍列表页”、“活动页”）共同组成了整个店铺的导航体系。
3.  在首页这个“总页面”上，放置的图片或按钮，其链接可以指向具体的商品，也可以指向其他功能页面（如领券中心、商品分类页）。

![](img/8b974a6eb6694bafdc5a4818ea217342_19.png)

**逻辑流程**：
`店铺首页（微页面）` -> `点击模块A` -> `跳转到商品详情页`
`店铺首页（微页面）` -> `点击模块B` -> `跳转到活动页面（另一个微页面）`

## 页面模块的调整

![](img/8b974a6eb6694bafdc5a4818ea217342_25.png)

模板中可能存在你不需要的模块。有赞的编辑器允许你自由增删模块。

![](img/8b974a6eb6694bafdc5a4818ea217342_27.png)

以下是调整页面布局的方法：
1.  将鼠标悬停在不需要的模块上，会出现删除图标，点击即可删除该模块。
2.  你也可以通过拖动来调整模块的上下顺序。
3.  如果删除了所有模块后想重新添加，可以使用编辑器左侧的组件库，重新添加图片、商品列表、文本等模块。

## 总结

本节课中我们一起学习了有赞微页面的建设。我们首先明确了**商品构成页面，页面构成店铺**的核心逻辑。然后，我们逐步实践了如何创建微页面、选择模板、替换内容素材，并重点设置了页面的链接跳转，将页面与商品或其他页面串联起来。最后，我们还学会了如何自由调整页面的布局和模块。掌握这些，你就能够搭建出一个结构清晰、引导流畅的店铺前端页面了。
