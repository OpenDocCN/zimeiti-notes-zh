# 微信视频号教程：P27：视频特效添加指南 🎬

在本节课中，我们将学习如何在微信视频号编辑中添加视频特效。特效能为视频增色，但使用不当也可能影响观感。我们将重点介绍如何正确添加“画面特效”和“人物特效”，并确保它们作用于正确的视频轨道上。

上一节我们介绍了画中画功能，本节中我们来看看如何为视频添加特效。

## 添加画面特效 ✨

在编辑界面，点击“画中画”后面的“特效”按钮。特效分为“画面特效”和“人物特效”两类。

以下是添加画面特效到主视频的步骤：
1.  点击“画面特效”。
2.  在特效列表中选择一个合适的特效，例如“星星灯”或“星星坠落”。
3.  点击选中的特效，将其添加到时间轴。
4.  可以调整特效的“速度”和“透明度”等参数。对于主视频，通常将透明度适当调低，避免过于刺眼。
5.  点击“对勾”确认添加。

这样，特效就被成功添加到了主视频画面上。像星星这类特效，添加在主视频上通常不会显得难看。

## 特效的作用对象 🎯

并非所有特效都适合添加在主视频上。如果添加不当，可能会导致主视频内容模糊或变形。

以下是调整特效作用对象的步骤：
1.  添加一个特效，例如“模糊”特效。
2.  如果发现整个主视频变模糊了，说明特效作用对象错误。
3.  在特效参数中，找到“作用对象”选项。
4.  将作用对象从“主视频”更改为“画中画”。
5.  点击“对勾”确认。此时，模糊效果将只作用于画中画图层，主视频会恢复清晰。

这个原则同样适用于“人物特效”。例如，添加“闪白”人物特效时，也应将其作用对象设置为“画中画”，以免主视频人物出现异常闪烁。

**核心操作公式**：`特效作用对象 = 画中画`。这样可以确保主视频内容不受干扰。

## 给初学者的建议 👍

对于初学者，特效的使用应遵循简洁原则。

以下是给初学者的具体建议：
*   **优先使用基础特效**：可以只使用“画面特效”和“人物特效”中的少数几种，并将它们的作用对象都设置为“画中画”。
*   **谨慎使用氛围特效**：在“画面特效”的“氛围”分类中，有“爱心”、“彩带”、“烟花”等特效。这些特效适合用于视频结尾营造温馨或庆祝气氛。
*   **宁缺毋滥**：如果不确定某个特效是否合适，或者添加后感觉视频变得杂乱，建议不要添加。删除不合适的特效，保持视频干净整洁比堆砌特效更重要。

本节课中我们一起学习了微信视频号中特效的添加方法。关键点在于通过“作用对象”设置，将特效精准应用到“画中画”图层，从而保护主视频的清晰与完整。对于初学者，从少量、简单的特效开始尝试是更稳妥的做法。
