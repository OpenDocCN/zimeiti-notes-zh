其核心操作可以概括为一个简单的**公式**：
`输入视频链接` -> `点击提取` -> `获得纯文本文案`

### 第一步：获取视频链接
首先，在你想要提取文案的短视频APP中，找到“分享”按钮，复制视频链接。

### 第二步：粘贴并提取
打开该在线工具的网页，将复制的链接粘贴到输入框中。然后，点击“提取文案”按钮。

### 第三步：处理与使用
稍等片刻（通常不超过2分钟），工具就会将识别出的文案显示出来。你可以直接复制纯文本，也可以使用“一键格式化”功能，让文案按照字幕的段落格式排列，更便于阅读和修改。此外，该工具也支持上传本地视频文件来提取文案。
