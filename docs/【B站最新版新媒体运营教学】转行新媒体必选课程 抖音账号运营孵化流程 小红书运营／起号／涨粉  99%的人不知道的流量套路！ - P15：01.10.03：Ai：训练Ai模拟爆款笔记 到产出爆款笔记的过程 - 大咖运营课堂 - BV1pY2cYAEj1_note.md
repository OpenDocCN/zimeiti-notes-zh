## 分析爆款内容的核心要点

![](img/9d9ae0e41ccad3b55e0a51d30194e4b0_11.png)

![](img/9d9ae0e41ccad3b55e0a51d30194e4b0_13.png)

![](img/9d9ae0e41ccad3b55e0a51d30194e4b0_15.png)

![](img/9d9ae0e41ccad3b55e0a51d30194e4b0_17.png)

![](img/9d9ae0e41ccad3b55e0a51d30194e4b0_19.png)

我们的目标是让AI深入分析爆款笔记，并提取出使其成为爆款的关键因素。以下是需要AI重点分析的几个方面：

*   **爆款因素分析**：分析内容中哪些点可能导致了其高流量和高互动。
*   **内容结构拆解**：拆解文案的框架。
    *   **开篇**：用什么方式吸引用户继续阅读。
    *   **中间**：提供了哪些内容来戳中用户痛点，提升用户停留时长。
    *   **结尾**：设置了什么“钩子”来引导用户互动（如评论、私信、关注）。
*   **关键词策略分析**：分析笔记中是否植入了特定关键词，以及关键词的出现频率和铺设方式。

### 第一步：提供爆款笔记并下达分析指令

![](img/9d9ae0e41ccad3b55e0a51d30194e4b0_38.png)

![](img/9d9ae0e41ccad3b55e0a51d30194e4b0_40.png)

![](img/9d9ae0e41ccad3b55e0a51d30194e4b0_42.png)

首先，我们需要找到目标领域的爆款笔记原文。然后，向AI（如ChatGPT 3.5）下达清晰的分析指令。

![](img/9d9ae0e41ccad3b55e0a51d30194e4b0_44.png)

![](img/9d9ae0e41ccad3b55e0a51d30194e4b0_46.png)

![](img/9d9ae0e41ccad3b55e0a51d30194e4b0_48.png)

![](img/9d9ae0e41ccad3b55e0a51d30194e4b0_50.png)

以下是给AI的指令示例：
```
我给你提供一些关于小红书平台上的考研相关爆款笔记文案，你需要分析这些文案的爆款因素。
1. 分析文案的整体结构：开篇用什么吸引用户，中间用什么内容戳中用户痛点，结尾用什么方式让用户形成互动或私信关注等行为。
2. 分析每一篇笔记的关键词：根据整篇文案分析，该笔记是否有结合中文分词法设计关键词。没有则告知“没有关键词”，有则告诉我是哪个关键词，以及该关键词的出现频率。
```
随后，将找到的爆款笔记文案粘贴给AI，等待其分析结果。

### 第二步：整合与总结爆款要素

![](img/9d9ae0e41ccad3b55e0a51d30194e4b0_59.png)

![](img/9d9ae0e41ccad3b55e0a51d30194e4b0_61.png)

![](img/9d9ae0e41ccad3b55e0a51d30194e4b0_62.png)

![](img/9d9ae0e41ccad3b55e0a51d30194e4b0_64.png)

![](img/9d9ae0e41ccad3b55e0a51d30194e4b0_66.png)

![](img/9d9ae0e41ccad3b55e0a51d30194e4b0_67.png)

在AI分析了多篇笔记（例如4篇）后，我们需要让它对分析结果进行整合去重，提炼出该领域爆款笔记的共性要素。

![](img/9d9ae0e41ccad3b55e0a51d30194e4b0_69.png)

可以给AI如下指令：
```
请将你前面分析过的关于考研的几篇笔记的爆款因素总结一下，整合成一个清晰的列表。
```
AI可能会总结出诸如“情感共鸣”、“提供明确备考计划”、“时间节点清晰”、“正能量鼓励”等共性要素。此时需要检查其总结是否全面，必要时可提醒AI回顾之前分析过的具体篇目。

### 第三步：基于爆款要素生成新笔记大纲

![](img/9d9ae0e41ccad3b55e0a51d30194e4b0_71.png)

在AI掌握了爆款要素后，我们可以指令其基于这些要素，为新笔记规划一个具备爆款潜力的框架。

![](img/9d9ae0e41ccad3b55e0a51d30194e4b0_73.png)

指令如下：
```
我要写一篇关于考研的小红书笔记。请结合你前面分析的爆款笔记因素，帮我梳理一个具备爆款因素的笔记大纲。每个大纲节点要细分成不同的小节点。在我确定大纲没问题后，再生成完整文案。
```
AI生成的大纲可能包含“开篇引入”、“专业选择分析”、“各科目备考规划”、“时间管理与心态调整”、“结尾互动引导”等部分。这个大纲为我们后续的文案创作提供了清晰的结构。

### 第四步：生成完整笔记文案

![](img/9d9ae0e41ccad3b55e0a51d30194e4b0_75.png)

![](img/9d9ae0e41ccad3b55e0a51d30194e4b0_76.png)

最后，我们要求AI根据确定的大纲和爆款要素，生成完整的笔记文案。此时可以加入更细致的风格要求。

指令示例：
```
请按照你提供的大纲，写出具体的文案。文案结构要严格按照爆款笔记因素来写，并且需要：
1. 提供一个简单的逆袭小故事，引入读者的情绪。
2. 包含一定量的emoji表情。
3. 结合中文分词法的方式融入关键词，如“考研”、“政治备考”等。
```
AI会根据指令生成一篇融合了情感共鸣、实用建议、清晰结构和互动引导的笔记初稿。这篇文案对于一般运营需求可能已足够，但我们可以在此基础上进行人工优化和调整，使其更精炼、丰富。

## 进阶技巧与工具

在基础流程之上，我们还可以利用更高级的工具提升效率和质量。

![](img/9d9ae0e41ccad3b55e0a51d30194e4b0_78.png)

*   **使用ChatGPT 4.0**：相比3.5版本，4.0在理解复杂指令、内容连贯性和创意方面通常表现更佳。
*   **创建自定义GPTs**：利用ChatGPT的GPTs功能，可以创建一个专属的“小红书文案助手”。你可以上传平台风格示例、专用表情包列表等资料供其学习，从而打造出更符合你个人或品牌需求的AI助手，避免每次都需要重复提供背景指令。

![](img/9d9ae0e41ccad3b55e0a51d30194e4b0_80.png)

![](img/9d9ae0e41ccad3b55e0a51d30194e4b0_82.png)

本节课中我们一起学习了利用AI从分析爆款笔记到生成新笔记的完整流程。关键在于为AI提供清晰的分析指令和爆款样本，引导其总结规律，并最终基于这些规律进行结构化创作。掌握这个方法，能有效提升新媒体内容创作的效率和质量。
