# 短视频IP打造：第5节：市场研究：洞察平台、行业与竞品，建立全局视野

![](img/b9ebfcf9a497631d9b73a6c564d1739b_1.png)

在本节课中，我们将学习如何进行市场研究。掌握洞察平台规则、行业趋势和竞争对手的方法，能帮助你建立全局视野，为内容创作找到明确方向，避免盲目努力。

## 为什么要进行市场研究？

上一节我们探讨了账号定位，本节中我们来看看如何通过外部研究来验证和优化你的方向。不了解市场，可能会错过行业发展和转型的关键时机。不了解平台规则，容易导致违规，使努力前功尽弃。不了解竞争对手，则很容易在激烈的竞争中丧失优势。

## 如何掌握市场趋势？

掌握市场趋势，能让你对整个行业环境有全局认识，从而找准方向、顺势而为。市场趋势信息通常来自公开的行业报告。

![](img/b9ebfcf9a497631d9b73a6c564d1739b_3.png)

以下是获取行业报告的常用方法：

*   **搜索关键词**：在搜索引擎中使用“短视频报告”、“抖音报告”、“快手报告”等定向关键词进行搜索。
*   **推荐数据网站**：强烈推荐以下几个行业常用的数据资讯网站，可以常去浏览学习：
    *   中文互联网数据资讯网
    *   艾瑞网
    *   CBNdata（第一财经数据中心）

## 如何利用行业报告？

![](img/b9ebfcf9a497631d9b73a6c564d1739b_5.png)

内容产业服务平台“新榜”和“卡思数据”是了解视频内容行业风向的重要渠道。它们提供各行业的互联网趋势报告、人群消费报告和行业白皮书。

一份典型的行业报告通常包含以下几个部分：

1.  **数据来源与统计方法**：帮助你判断报告的权威性和参考价值。
2.  **结论概述与关键数据**：通常位于报告的开篇或结尾，是整个报告最核心的洞察发现，应重点阅读。
3.  **具体的分类分析**：对数据进行详细解读，每页数据分析的标题通常就是结论性总结。

例如，一份报告中的关键数据显示：**短视频平均观看时长约为8.9秒**。这个数据告诉我们，在创作内容时，必须思考如何在**前8.9秒**内抓住用户注意力，无论是通过特定的音乐、表演，还是控制视频时长。

## 从报告中洞察趋势与机会

阅读报告时，要关注趋势背后的洞察：它是为什么增长或衰退？增长时要抓住机会，衰退时则需尽量避免。

以下是基于“卡思数据”2019年短视频KOL报告的一些趋势分析：

*   **内容类型变化**：娱乐化内容占比下降，泛生活、垂直领域（如时尚美妆、汽车）内容增长迅速。这表明用户需求正从单纯娱乐向**获取知识、实现自我价值**转移。
*   **平台差异**：
    *   **抖音**：时尚、情感、美妆类内容竞争已非常激烈。
    *   **快手**：“小哥哥”类内容超越搞笑成为最强类别，美食、萌宠类账号增长迅速，粉丝粘性高。
    *   **B站**：游戏类内容超越鬼畜成为榜首，小众特色类别越发强势。
*   **带货情况参考**：
    *   在抖音，粉丝量10-30万的尾部账号开设商品橱窗最多，平均**周更视频3.9条**。
    *   在快手，粉丝过百万的红人超九成开通直播，平均**周播5场，单场2小时**。
    *   从品类看，**时尚穿搭、美食类KOL**的带货能力更优秀。

这些信息可以帮助你制定内容策略，例如确定更新频率和选择有潜力的内容赛道。

## 如何掌握平台规则与动态？

了解平台信息，可以直接搜索“抖音数据报告”、“快手内容生态报告”等，或在上述行业网站查找。

![](img/b9ebfcf9a497631d9b73a6c564d1739b_7.png)

以抖音报告为例，报告中提到的年度情绪关键词（如“我太难了”、“加油”）揭示了大众的共鸣点：**困难与失败是共同主题，但振作与正能量是持续的需求**。在创作中加入此类方向的内容，更容易获得平台推荐和用户认可。

除了报告，还可以通过以下官方渠道学习：

*   关注 **抖音小助手**，了解热门视频内容。
*   关注 **电商小助手**，学习开通购物车。
*   关注 **巨量课堂**，掌握运营机制与发视频技巧。
*   在抖音的“抖加推广-运营学院”或快手的“作品推广-推广小课堂”中学习审核规则。

## 如何深入研究竞争对手？

锁定一些优秀账号进行学习研究至关重要。一方面分析同类竞品的数据，另一方面解析其运营规律和内容。

以下是分析竞争对手的方法：

首先，可以从宏观维度建立分析表格，包含以下字段：
`品类标签、二级标签、账号名称、粉丝数、视频形式（横/竖屏）、地区、人设特点、内容风格、差异化亮点`。

其次，对单条视频内容进行微观分析。从账号的第一条视频开始，按`发布时间、标题、故事梗概、热评、点赞数、评论量、转发量`等维度记录。这种文字化描述能直观地发现可借鉴之处。

例如，分析一个增长迅速的剧情账号“将时期”，会发现其内容**信息密度高、剧情紧凑、反转设置巧妙**，情绪张力大，因此用户代入感强，评论区讨论热烈。
