上一节我们介绍了视频剪辑前的准备工作，本节中我们来看看如何使用剪映App进行具体的剪辑操作。

首先，请确保你下载的是剪映App（黑色图标版本）。它提供手机版和电脑版，我们先从手机版开始讲解。

![](img/01c3cfd07549c9c17b1a87131ea56019_9.png)

![](img/01c3cfd07549c9c17b1a87131ea56019_11.png)

**第一步：导入素材与调整构图**

![](img/01c3cfd07549c9c17b1a87131ea56019_13.png)

打开剪映App，点击“开始创作”，然后选择你需要剪辑的视频片段并“添加”到项目中。如果你的视频由多个片段组成，建议先导入并剪辑完一个片段，再点击“+”号导入下一个，这样操作更清晰。

以下是调整视频构图和比例的具体步骤：
1.  选中时间轴上的视频片段。
2.  在底部工具栏中向右滑动，找到并点击“编辑”。
3.  点击左侧的“旋转”按钮，将视频调整到正确的方向。
4.  如需裁剪画面比例，点击“裁剪”，可以选择 **1:1**、**4:3**、**16:9** 等常用比例。

![](img/01c3cfd07549c9c17b1a87131ea56019_15.png)

![](img/01c3cfd07549c9c17b1a87131ea56019_17.png)

调整完毕后，点击播放预览视频。

![](img/01c3cfd07549c9c17b1a87131ea56019_19.png)

**第二步：裁剪无用片段**

![](img/01c3cfd07549c9c17b1a87131ea56019_21.png)

在预览时，需要剪掉视频开头、结尾或中间无用的部分（如长时间沉默或无关动作）。

![](img/01c3cfd07549c9c17b1a87131ea56019_23.png)

![](img/01c3cfd07549c9c17b1a87131ea56019_25.png)

裁剪片段的核心操作是：
1.  将时间线（白色竖线）移动到需要分割的位置。
2.  选中视频片段，点击底部工具栏的“分割”按钮。此时视频被切成两段。
3.  选中需要删除的片段，点击“删除”按钮。

![](img/01c3cfd07549c9c17b1a87131ea56019_27.png)

![](img/01c3cfd07549c9c17b1a87131ea56019_29.png)

重复 **选中 -> 分割 -> 选中 -> 删除** 这个流程，即可流畅地去除所有不需要的部分。

![](img/01c3cfd07549c9c17b1a87131ea56019_31.png)

**第三步：添加与编辑背景音乐**

![](img/01c3cfd07549c9c17b1a87131ea56019_33.png)

![](img/01c3cfd07549c9c17b1a87131ea56019_35.png)

视频裁剪好后，需要添加背景音乐。点击底部工具栏的“音频”。

![](img/01c3cfd07549c9c17b1a87131ea56019_37.png)

添加音乐有两种主要方式：
*   **提取音乐**：如果你已经用其他工具（如淘金阁）生成了配音文件，可以点击“提取音乐”，直接从手机视频中提取音频。
*   **添加音乐**：点击“音乐”，剪映内置了丰富的音乐库。你可以按分类（美食、美妆、萌宠等）查找，或直接搜索歌曲。点击歌曲旁边的播放图标可以预览，选定后点击“使用”即可添加。

![](img/01c3cfd07549c9c17b1a87131ea56019_39.png)

![](img/01c3cfd07549c9c17b1a87131ea56019_41.png)

编辑添加的音乐：
*   **裁剪长度**：如果音乐过长，操作与裁剪视频相同：移动时间线到音乐轨道，**选中 -> 分割 -> 选中 -> 删除**。
*   **调整音量**：选中音乐轨道，点击“音量”滑块，可以调大或调小背景音乐的音量，避免盖过人声。

![](img/01c3cfd07549c9c17b1a87131ea56019_43.png)

**第四步：自动识别与添加字幕**

![](img/01c3cfd07549c9c17b1a87131ea56019_45.png)

![](img/01c3cfd07549c9c17b1a87131ea56019_47.png)

为视频添加字幕可以提升观看体验。点击底部工具栏的“文字”。

![](img/01c3cfd07549c9c17b1a87131ea56019_49.png)

高效添加字幕的方法是使用自动识别功能：
1.  点击“识别字幕”。
2.  选择“仅视频”（避免识别到背景音乐里的歌词）。
3.  点击“开始识别”。系统会自动识别视频中的语音并生成字幕。

![](img/01c3cfd07549c9c17b1a87131ea56019_51.png)

识别完成后，你可以：
*   **检查错别字**：双击字幕条，修改识别错误的文字。
*   **调整样式**：选中字幕，可以更改字体、颜色（如配合衣服颜色）、大小等，让字幕更美观。

![](img/01c3cfd07549c9c17b1a87131ea56019_53.png)

**第五步：导出高清视频**

![](img/01c3cfd07549c9c17b1a87131ea56019_55.png)

在导出最终视频前，需要进行关键设置以保证清晰度。

![](img/01c3cfd07549c9c17b1a87131ea56019_57.png)

点击右上角的“导出”按钮，在导出设置中：
*   **帧率 (FPS)**：设置为 **30**。这是抖音平台最合适的帧率。
*   **分辨率**：尽可能选择最高选项（如1080p或4K）。分辨率越高，视频越清晰。

![](img/01c3cfd07549c9c17b1a87131ea56019_59.png)

![](img/01c3cfd07549c9c17b1a87131ea56019_61.png)

请记住导出设置的公式：**帧率 = 30，分辨率 = 拉满**。

设置完成后，点击“导出”。**请注意：导出过程中请勿锁屏或切换手机应用**，耐心等待导出完成。导出后，可以直接通过剪映分享到抖音。

![](img/01c3cfd07549c9c17b1a87131ea56019_63.png)

在掌握了基础的剪辑流程后，我们为口播类视频创作者推荐一个实用工具。

**辅助工具推荐：爱提词**

![](img/01c3cfd07549c9c17b1a87131ea56019_69.png)

![](img/01c3cfd07549c9c17b1a87131ea56019_71.png)

对于需要面对镜头讲话的创作者，记台词是一大难题。“爱提词”这款App可以完美解决这个问题。

![](img/01c3cfd07549c9c17b1a87131ea56019_73.png)

它的使用方法如下：
1.  打开App，点击右下角的“+”号。
2.  “添加”你的台词文本，可以手动输入或粘贴复制好的文案。
3.  保存后，点击“开始提词”，选择“录制模式”。
4.  此时手机屏幕会像提词器一样滚动显示台词，你可以看着屏幕流畅录制，无需背诵。

![](img/01c3cfd07549c9c17b1a87131ea56019_75.png)

此外，该工具还提供一些实用功能：
*   **调整字体**：在设置中可放大字体，方便近视用户看清。
*   **内置美颜**：点击左下角的“美颜”功能，可以调节瘦脸、大眼、磨皮等参数，让拍摄效果更自然美观，不同于过度美颜的相机App。
*   **爆款文案参考**：App内还提供了职场、情感、生活等多个领域的爆款文案库，可供创作参考。

![](img/01c3cfd07549c9c17b1a87131ea56019_79.png)

**本节课总结**

![](img/01c3cfd07549c9c17b1a87131ea56019_85.png)

本节课我们一起学习了使用剪映App进行视频剪辑的全流程：
1.  **导入与构图**：导入素材，调整旋转与画面比例。
2.  **裁剪片段**：使用 **分割** 与 **删除** 功能去除无用部分。
3.  **添加音频**：为视频添加并裁剪背景音乐，调整音量。
4.  **添加字幕**：利用“识别字幕”功能快速生成并修改字幕。
5.  **导出设置**：按照 **帧率30，分辨率拉满** 的设置导出高清视频。
6.  **辅助工具**：介绍了“爱提词”App，帮助口播创作者解决记台词和拍摄美颜的问题。

![](img/01c3cfd07549c9c17b1a87131ea56019_87.png)

请跟随步骤实际操作一遍，就能掌握基础的视频剪辑技能。
