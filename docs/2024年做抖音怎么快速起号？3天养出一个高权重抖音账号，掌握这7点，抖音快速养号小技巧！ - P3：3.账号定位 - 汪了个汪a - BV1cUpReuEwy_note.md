# 抖音运营教程：P3：账号定位 🎯

![](img/cac4e780b03de6798cb59dd42a061f45_0.png)

在本节课中，我们将要学习抖音账号运营中至关重要的一步：**账号定位**。清晰的定位是账号快速起号、获得高权重的基础。我们将通过反思三个核心问题来构建你的账号定位框架。

![](img/cac4e780b03de6798cb59dd42a061f45_2.png)

## 你是谁？—— 确定账号类型与主体

上一节我们介绍了赛道选择，本节中我们来看看如何进一步细化。首先，你需要确定账号的类型。这不仅仅是选择一个大类赛道（一级垂类），更要思考其下的细分领域（二级垂类）。

**公式：账号垂直度 = 一级垂类 + 二级垂类**

例如，你选择“运动健身”作为一级垂类，而你最擅长“瑜伽教学”，那么“瑜伽教学”就是你的二级垂类。通过定位细分类目，可以确保内容的垂直与专业。

其次，需要确定账号的呈现主体。在抖音商业账号中，主体主要分为两类：

以下是两种账号主体的区别：

*   **商家账号**：主体是公司或品牌。内容需偏向商业化，但应通过亲切的内容拉近与粉丝的距离，避免过于严肃，以符合抖音的娱乐属性。
*   **达人账号**：主体是个人。首要解决的是信任问题，需要通过IP打造，树立积极、正面的人设。

## 你是做什么的？—— 明确账号内容与价值

明确了“你是谁”之后，接下来要清晰地告诉粉丝“你是做什么的”。最直接有效的方式是进行**抖音官方认证**。

*   **个人达人**：可申请**黄V认证**，展示职业身份。
*   **企业商家**：可申请**蓝V认证**，获得企业背书。

认证后，粉丝能清晰看到你的认证标签，快速了解你的专业领域。即使在认证前，也可以通过以下账号设置来传达信息：

以下是传达账号业务的设置要点：

*   **头像**：与业务或人设相关。
*   **背景图**：展示业务范围或核心价值。
*   **简介**：一句话清晰说明你能提供什么。
*   **视频封面**：风格统一，突出主题。

## 为什么要关注你？—— 提供核心价值

最后，也是最关键的一步，你需要思考能为粉丝提供什么样的独特价值。在抖音，用户关注一个账号通常基于以下几种价值需求：

以下是抖音账号提供的五种核心价值类型：

1.  **知识价值**：为用户提供信息增量，减少知识盲区。
    *   **认知型**：如人生思考、行业经验、个人成长。
    *   **干货型**：如具体技能、方法步骤、经验总结。
    *   **常识型**：如生活、健康、科学常识的解读。
    *   *提供给粉丝的价值是 **认同感** 与 **获得感**。*

2.  **情绪价值**：通过内容引发观众的情感共鸣或波动。
    *   **类型**：开心愉悦、关系共鸣（如情侣、家庭）、情绪反转、怀旧、好奇、震惊等。
    *   *提供给粉丝的价值是 **共鸣**，直接感染观众情绪。*

3.  **感觉价值**：通过视听体验让用户获得美的享受或冲击。
    *   **来源**：画面美感、构图、视觉冲击、个人魅力、声音节奏等。
    *   **常见领域**：颜值、萌宠、美景、才艺、文化等。
    *   *提供给粉丝的价值是 **共情**，引发用户在意识层面的连接。*

4.  **体验价值**：带领用户见识或感受未曾经历的事物与生活方式。
    *   **常见领域**：美食探店与制作、旅行探险、运动挑战、角色扮演等。
    *   *提供给粉丝的价值是 **满足好奇心** 与 **获得体验感**。*

5.  **期待价值**：通过内容设计，让用户对下一次更新产生强烈期待。
    *   **方法**：在视频中设置悬念，不一次性展示所有精彩内容。
    *   *其目的是促使用户点击 **关注**，以便持续观看。*

本节课中我们一起学习了抖音账号定位的完整框架。你需要依次回答“你是谁”（确定类型与主体）、“你是做什么的”（明确内容与认证）以及“为什么要关注你”（提供核心价值）这三个问题。一个清晰、垂直且能持续提供价值的账号定位，是你在抖音快速起号、获得平台推荐与粉丝青睐的坚实基础。下一节，我们将探讨如何基于此定位来规划具体的内容。
