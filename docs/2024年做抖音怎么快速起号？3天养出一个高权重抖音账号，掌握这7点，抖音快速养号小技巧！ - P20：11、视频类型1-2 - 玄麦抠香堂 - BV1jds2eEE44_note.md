# 抖音运营：P20：11、视频类型1-2 - 人设与口播 📚

在本节课中，我们将要学习抖音账号运营中至关重要的两个概念：**人设**的建立与区分，以及**口播**视频的制作要点。理解并应用这些知识，能帮助你的内容在众多账号中脱颖而出。

上一节我们介绍了文案的重要性，本节中我们来看看如何通过塑造人设和选择内容形式来吸引并留住粉丝。

从直接涨粉到实现变现，中间有一个关键环节，就是建立你的**人设**。建立人设的目的不是为了炫耀，而是为了让你在众多同类创作者中被记住。例如，有十个人都做装修内容，如果其中一位创作者塑造了幽默搞笑的人设，观众就更容易记住他，从而成为他的粉丝。

接下来，我们来学习这个知识点。人设主要分为两种类型：**强人设**与**弱人设**。区分清楚这两种人设，有助于你分析和创作视频。

![](img/7878a08fcd9904b120793a746b3b6cfd_1.png)

以下是两种人设的具体定义：

![](img/7878a08fcd9904b120793a746b3b6cfd_3.png)

*   **强人设**：以创作者**个人**为切入点的内容。这与你自身密切相关，例如分享你的行业经验、专业技能或个人生活。常见的领域包括装修、法律咨询、餐饮、电商等。或者，以你的**兴趣爱好**为切入点，例如美食、旅行、手工、美妆、游戏、摄影等。这类内容通常需要分享你的工作、生活或爱好，并且大部分情况下需要**真人出镜**，因为核心是与“你”这个人建立联系。
*   **弱人设**：内容与创作者**个人**关联度较低，核心目的是运营好账号本身。例如，制作纯情感语录配音乐的视频、影视解说类视频等。这些内容可能不露脸，或即使有配音，其目的也是为了服务内容（如解说电影），而非塑造配音者本身的个人形象。

区分清楚强弱人设是选择内容方向的基础。现在，我们来看看具体的内容制作环节。

只要内容与你个人相关，都可以作为视频素材。我们首先详细讲解第一种广泛适用的形式：**讲故事**。

讲故事可以分享客户故事、从业经历或人生感悟。故事本身不要求完全真实，可以从文案平台寻找有趣的故事进行分享。而讲故事最直接的方式，就是通过**口播**来实现。

口播，即用嘴巴讲述内容。它拥有丰富的分类，能满足不同需求：

*   **讲故事类口播**（如情感、养生口播）：目的是让观众觉得内容有趣、有共鸣。
*   **传授知识类口播**（如育儿、金融、装修、房产知识口播）：目的是在几十秒内让观众获得“收获感”，觉得学到了东西。

无论你身处哪个领域，只要涉及知识分享或故事讲述，都可以采用口播形式。它的优势在于拍摄简单（一部手机即可），且内容依赖于提前准备好的文案。

对于常见的顾虑，这里有两个明确的解答：

1.  **口才不好能否做口播？**
    *   可以。口播是依据**提前准备好的稿子**进行录制，并非临场发挥。
2.  **普通话不标准能否做口播？**
    *   可以。如果你的目标受众是同城或有地域属性，使用**方言**可能效果更好，能增加亲切感和趣味性。同时，可以通过添加字幕来确保信息传递。

口播是一种易于上手且效果显著的内容形式，在后续课程中我们会详细讲解其拍摄技巧。

本节课中我们一起学习了如何通过建立**强人设**或**弱人设**来让账号更具辨识度，并深入探讨了**口播**这一核心视频形式的特点、分类及常见问题的解决方案。掌握这些，你就为创作吸引人的内容打下了坚实基础。
