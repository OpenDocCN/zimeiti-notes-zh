# 抖音运营新手起号：P23：产品与用户兴趣标签 🧩

在本节课中，我们将学习一种实用的选题方法——九宫格选题法。我们将以健身软件Keep为例，详细拆解如何通过分析产品标签和用户兴趣标签，来生成源源不断的视频创作灵感。

上一节我们探讨了选题的重要性，本节中我们来看看如何具体操作，将抽象的想法转化为具体的视频主题。

## 产品标签分析

首先，我们需要分析产品本身的特点。将目标产品（例如Keep）置于九宫格的中心，然后围绕它，联想并填充与其直接相关的8个核心属性。

以下是围绕Keep健身软件可以联想到的产品标签：

1.  **线上健身软件**：产品的核心形态。
2.  **专业健身教练**：平台提供的核心资源。
3.  **饮食搭配**：产品提供的增值服务。
4.  **减肥攻略**：满足用户的核心痛点之一。
5.  **健身干货知识**：内容价值所在。
6.  **科学运动方法**：产品的专业性和可信度。
7.  **打卡运动制度**：产品的互动与激励功能。
8.  **一二线城市用户为主**：产品的主要用户画像。

填充完产品标签后，我们暂时将其放在一边。接下来，我们把视角切换到用户身上。

## 用户兴趣标签分析

现在，我们来分析使用该产品的用户群体。将核心用户（例如“健身爱好者”）放在另一个九宫格的中心，然后发散思维，思考这类人群还对哪些事物感兴趣。

以下是围绕“健身爱好者”可以联想到的兴趣标签：

![](img/72e623cf3dd0e774c4074b3254397f1a_1.png)

1.  **健身房**：线下健身的常见场所。
2.  **健身装备**：如运动鞋、紧身衣等。
3.  **轻脂美食/减肥餐**：与健身目标相关的饮食兴趣。
4.  **旅行**：作为热爱生活的一种表现。
5.  **星座**：普遍的休闲兴趣点。
6.  **明星**：娱乐关注点。
7.  **鸡汤文**：情感或励志需求。
8.  **时尚穿搭**：与外形管理相关的兴趣。

通过这种方式，我们将用户标签从“健身爱好者”扩展到了“生活热爱者”，从而获得了更广泛的选题切入点。

## 生成选题：连连看组合法

当我们填满了两个九宫格后，就拥有了一个丰富的标签库。接下来，就是创造选题的关键步骤：从“产品标签”和“用户兴趣标签”中分别选取一个进行组合，像玩“连连看”一样，将它们串联成一个完整的视频主题。

以下是几个组合示例：

*   **组合1：饮食搭配 + 旅行**
    *   **生成选题**：《外出旅行时，如何享受美食又能控制体重？》

*   **组合2：运动方法 + 明星**
    *   **生成选题**：《赵丽颖都在练的5个高效瘦身动作，一学就会！》

*   **组合3：科学 + 健身房**
    *   **生成选题**：《别再瞎练了！用这3个科学方法，让健身房效果翻倍。》

如果组合后暂时想不出好的文案，可以借助搜索工具。例如，搜索“科学 健身”等关键词，查看相关的长尾词或热门话题，从中获取灵感。

## 如何判断选题质量

通过“连连看”可以产生大量选题，但并非所有组合都是好选题。一个好的选题通常需要具备以下几点：
*   **符合逻辑**：两个标签的组合要自然，能自圆其说。
*   **吸引目标用户**：内容要切中用户的核心需求或兴趣点。
*   **具有操作性**：能够基于这个选题拍摄出具体的视频内容。

例如，“健身房”与“科学”的组合是合理的，但若强行组合“健身房”与“撩妹”，则可能偏离核心用户的实际需求，不是一个好选题。

本节课中我们一起学习了九宫格选题法的具体应用。我们首先分别梳理了**产品标签**和**用户兴趣标签**，然后通过**连连看组合法**将不同标签进行碰撞，从而生成具体的视频选题。这个方法能有效解决“不知道拍什么”的难题，帮助你建立起持续的选题素材库。记住，关键在于充分发散思维填充标签，并确保最终的组合逻辑自洽、能吸引目标观众。
