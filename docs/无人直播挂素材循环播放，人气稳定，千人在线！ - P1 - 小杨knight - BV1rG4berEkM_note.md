# 无人直播教程：P1：核心原理与准备工作

在本节课中，我们将学习无人直播的核心原理，并完成开播前的所有准备工作。无人直播的核心在于通过技术手段，实现直播内容的自动化循环播放，从而达到稳定人气和在线人数的效果。

## 核心原理

上一节我们介绍了课程目标，本节中我们来看看无人直播是如何运作的。其基本原理是模拟一个真实的直播推流过程。我们并非使用摄像头实时采集画面，而是将一个预先制作好的视频文件（即“素材”）作为信号源，通过推流软件持续不断地发送给直播平台。

这个过程可以用一个简单的公式来描述：
**直播推流 = 视频素材 + 循环播放逻辑 + 推流软件**

其中，**视频素材**是内容基础，**循环播放逻辑**确保直播不会中断，**推流软件**则是连接本地电脑与直播平台的桥梁。只要推流不停止，平台就会认为这是一个正在进行的直播，从而实现“无人值守”。

## 准备工作

理解了核心原理后，我们需要为实战操作做好准备。以下是成功开启无人直播所需的三个关键要素。

1.  **视频素材**
    你需要准备一个或多个用于循环播放的视频文件。建议内容精炼、画质清晰，时长不宜过短，以避免频繁重复导致观众察觉。

2.  **推流软件**
    推荐使用 OBS Studio。这是一款免费开源的直播推流软件，功能强大且稳定，是大多数主播的选择。

3.  **直播平台推流地址与密钥**
    这是连接你电脑和直播平台的“钥匙”。你需要在抖音、快手、B站等目标直播平台的后台，找到“直播设置”或“开播工具”栏目，获取以 `rtmp://` 或 `rtmps://` 开头的**服务器地址**和一串由字母数字组成的**串流密钥**。

## 软件配置步骤

现在，我们已经准备好了所有材料，接下来进入具体的软件配置环节。请按照以下步骤操作。

以下是OBS Studio的基本设置流程：

*   **添加视频源**
    在OBS主界面下方的“来源”面板中，点击“+”号，选择“媒体源”或“VLC视频源”。在弹出的窗口中，取消勾选“本地文件”循环（若使用媒体源），然后选择你准备好的视频文件。对于无人直播，更推荐使用“VLC视频源”，因为它内置了更稳定的循环播放功能。

*   **设置推流参数**
    点击OBS菜单栏的“文件” -> “设置”。在设置窗口中，选择“推流”选项卡。将你在直播平台获取的“服务器地址”和“串流密钥”分别填入对应栏目。

*   **配置输出参数**
    仍在设置窗口中，切换到“输出”选项卡。将输出模式改为“高级”。在“串流”编码器部分，建议选择“软件(x264)”。视频比特率根据你的网络上传速度设置，通常2000-6000 Kbps即可平衡清晰度与流畅度。关键帧间隔设为2秒。

*   **开始推流**
    所有设置完成后，点击OBS主界面右下角的“开始推流”按钮。如果一切正常，OBS的状态栏将显示绿色的“推流中”和稳定的帧率与码率。此时，你的直播画面已经开始在平台上循环播放了。

## 总结

本节课中我们一起学习了无人直播的基础知识。我们首先了解了其**通过循环播放视频素材并持续推流来模拟真实直播**的核心原理。然后，我们完成了**视频素材、推流软件OBS、平台推流密钥**这三项关键准备工作。最后，我们一步步演练了在OBS中**配置视频源、设置推流地址与输出参数**的完整流程。掌握这些步骤，你就已经具备了开启一场基础无人直播的能力。
