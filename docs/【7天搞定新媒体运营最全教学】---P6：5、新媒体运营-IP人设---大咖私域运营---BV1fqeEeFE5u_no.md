### 1. 角色

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_60.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_62.png)

角色，简单来说，就是你在特定赛道中扮演的专业身份。你选择什么赛道、准备输出什么方向的内容，就需要让别人知道你提供的内容具备专业性。

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_80.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_82.png)

以下是几种常见的角色定位示例：

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_84.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_86.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_88.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_92.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_94.png)

*   **特定领域的实践者/记录者**：例如，作为一名装修博主，你需要展示自己真实的装修过程、遇到的坑以及最终成果，通过前后对比来建立可信度。
*   **特定身份的经验分享者**：例如，在母婴赛道，最常见的角色是“宝妈”，分享亲身育儿经验。
*   **专业领域的顾问/专家**：同样在母婴赛道，你也可以以“育儿护理专家”的身份出现，这不一定要求你是宝妈，但需要展现专业背景或知识。
*   **成长故事的讲述者**：在情感或成长类赛道，你可以塑造“成长型人设”。例如，一位30岁女性分享自己如何应对家庭、创业、职场中的挑战，并最终走出困境的故事。

### 2. 内容

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_120.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_122.png)

内容是人设的载体。关于内容定位，我们之前强调过要吸引用户、提供价值并具备平台特色。在人设塑造中，内容还需要保持高度垂直。

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_132.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_134.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_136.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_138.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_140.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_142.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_144.png)

你必须持续输出与你所选赛道紧密相关的内容。这关系到账号的垂直度和标签化，直接影响账号的成长和平台的流量分发。如果你的内容方向混乱（例如第一篇讲教育，第二篇讲宠物），平台就无法为你打上精准标签，推荐给感兴趣的用户，从而影响视频的数据表现和推荐效果。

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_146.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_148.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_150.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_152.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_154.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_156.png)

**核心要点**：`内容垂直度 = 持续输出同一赛道的高价值内容`

形式是指你呈现内容的具体方式。它决定了用户感知你的“界面”是什么样的。

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_182.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_192.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_194.png)

以下是形式需要考虑的几个方面：

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_196.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_198.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_200.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_202.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_204.png)

*   **产出场景**：你的内容是在固定的室内场景拍摄，还是在多变的室外场景（如上班路上、面试途中）记录？确定拍摄场景有助于形成统一的视觉风格。
*   **声音呈现**：你是亲自出镜讲解，还是使用配音？这取决于你个人表达的流畅度和舒适度。
*   **内容类型**：你的内容是以“剧情演绎”、“干货分享”还是“生活记录”的形式呈现？每种类型各有优劣：
    *   **剧情型**：创作难度较高，但容易吸引眼球。
    *   **干货型**：需要精心准备文案和脚本，强调知识密度。
    *   **生活记录型**：需要积累大量日常拍摄素材，强调真实感。

### 4. 记忆点

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_218.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_220.png)

记忆点是让用户记住你的关键。你需要设计一些固定元素，反复刻入用户的脑海。

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_232.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_234.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_240.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_242.png)

以下是打造记忆点的几种有效方式：

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_244.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_246.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_248.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_250.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_252.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_254.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_256.png)

*   **视频前几秒的固定动作、特效或开场白**：这些元素在视频开头重复出现，能形成强烈的视觉或听觉冲击，加深用户印象。例如，每次视频都以一个标志性动作或口号开始。
*   **统一的背景音乐（BGM）**：一首独特的BGM贯穿你的多个视频，当音乐响起时，用户即使没看到画面也能联想到你。
*   **统一出现的物品或服装风格**：一个始终出现在背景中的特定物品，或一种固定的着装风格（如职场风、田园风），都能贯穿内容全局，成为你的视觉符号。

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_258.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_260.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_262.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_264.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_266.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_268.png)

**记忆点设计原则**：
*   **前置强化**：将固定动作、特效、开场白等放在视频最前面，利用用户刷主页时的连续观看加深记忆。
*   **全局贯穿**：让BGM、标志性物品或服装风格贯穿整个视频乃至所有内容，形成整体氛围。

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_270.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_272.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_278.png)

![](img/3d59cfb1b4fe9df6cc936a05c1d3d5a1_280.png)

本节课中我们一起学习了IP人设塑造的四个核心因素：**角色**、**内容**、**形式**和**记忆点**。理解并运用好这四点，就能为你的账号建立一个清晰、可信且令人印象深刻的人设基础。下一节课，我们将继续探讨人设定位的配套设置，让你的IP形象更加丰满。
