# 千川运营训练营：11：直播间的投流素材 📺

在本节课中，我们将学习如何为抖音千川广告寻找、分析并制作高投资回报率的直播引流素材。掌握这些方法，可以帮助你更高效地制作出能够跑量的优质视频。

![](img/4183e31250d0d7468dcee95e1a6e6447_1.png)

上一节我们介绍了千川投放的基础框架，本节中我们来看看如何具体准备投放的核心——视频素材。

## 高ROI素材的查找方法

首先，我们需要知道如何找到行业内效果好的投放素材。以下是具体操作步骤。

![](img/4183e31250d0d7468dcee95e1a6e6447_3.png)

1.  使用专业工具“有米云”进行查找。
2.  进入软件后，点击“创意洞察”功能。
3.  接着选择“直播间的引流创意”选项。
4.  在搜索框中输入你的产品类目关键词（例如“鲨鱼裤”）或目标达人名称进行搜索。
5.  搜索结果可按“总曝光”等数据进行降序排列，排在前列的就是当前全网曝光量高、效果好的素材。

通过以上步骤，你可以快速定位到值得参考的优质素材案例。

## 优质素材的分析方法

找到优质素材后，下一步是进行拆解分析。核心方法是**分解视频的分镜与内容**。

1.  **数分镜**：计算整个视频由多少个镜头（分镜）组成。
2.  **析内容**：分析每个分镜具体展示了什么内容（如产品特写、使用场景、人物动作等）。
3.  **学结构**：观察视频的整体节奏和镜头之间的衔接方式。

例如，一个高曝光的“鲨鱼裤”视频可能包含15个分镜，分别展示了产品细节、上身效果、搭配展示等。而另一个视频可能只用一个长镜头展示产品卖点。分析的目的在于理解其成功逻辑。

## 高ROI素材的拍摄与制作

分析完成后，即可着手制作自己的素材。最直接有效的方法是**对标与复制**。

如果你不擅长拍摄，可以寻找专业的拍摄团队，并明确要求他们对照你找到的优质素材进行仿拍。核心是复制其分镜设计、内容呈现和表现方式。

## 素材制作的三大核心规则

为了保证素材能够顺利通过审核并有效跑量，制作时必须遵守以下三条核心规则。

1.  **时长限制**：视频素材时长必须控制在**20秒以内**。
2.  **配音要求**：视频中**不要有明确讲解产品卖点的配音**，这类内容容易导致审核不通过或跑量困难。
3.  **文案规范**：视频字幕或描述等文案中，**不要包含具体的优惠信息**（如“拍两条送两双袜子”），这也容易影响投放效果。

只要满足以上三点，素材就能正常投放，并具备跑量的潜力。
