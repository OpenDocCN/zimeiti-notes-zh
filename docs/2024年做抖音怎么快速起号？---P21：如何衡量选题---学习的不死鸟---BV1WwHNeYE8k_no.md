# 抖音运营教程：P21：如何衡量选题 📊

在本节课中，我们将学习如何判断一个抖音选题是否优秀。我们将介绍五个核心标准，帮助你从众多想法中筛选出最具潜力的视频主题。

上一节我们介绍了如何寻找和拓展选题，本节中我们来看看如何评估这些选题的质量。

![](img/eaa9c507270b731dc4e38bcf5c8d8495_1.png)

以下是衡量选题好坏的五个核心标准：

1.  **低成本**：用户希望以最小的付出（如时间、精力、金钱）获得最大效果。例如，“玩手机躺着也能减肥”就比“50种减肥方法”更具吸引力。
    *   **公式**：用户感知价值 > 用户付出成本
2.  **精准**：选题需要明确指向特定人群的特定需求。例如，“新手必备的化妆全流程”就精准定位了化妆初学者。
3.  **熟悉**：利用用户熟悉的场景或共同记忆，能快速引发共鸣。例如，“你有没有在电影院里遇到过这种人？”
4.  **高频**：选题涉及的需求或场景在目标用户生活中出现频率高。例如，“厨房收纳”、“日常穿搭”都属于高频话题。
5.  **痛点**：直击用户最迫切、最强烈的需求或烦恼。例如，对于想买某款口红但觉得太贵的人，“领取大额优惠券”就戳中了“价格高”的痛点。

在实际操作中，**低成本**和**痛点**是两个最常用、最有效的衡量维度。一个选题如果能同时满足多个标准，其成为爆款的概率就会大大增加。

---

我们基本流程是：先寻找选题灵感，然后通过九宫格等方法拓展出大量选题，最后用这五个标准进行衡量和筛选，选出最值得拍摄的题目。

当然，并非每个选题都能完美符合所有标准。如果觉得所有选题都不够好，我们可以返回去，根据文案脚本或用户反馈进行优化和调整。
