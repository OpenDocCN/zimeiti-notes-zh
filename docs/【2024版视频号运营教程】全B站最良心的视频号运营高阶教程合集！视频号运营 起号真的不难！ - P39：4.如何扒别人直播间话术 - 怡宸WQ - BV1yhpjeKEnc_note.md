# 视频号运营教程：P39：如何扒取直播间话术

在本节课中，我们将学习一个实用的技巧：如何高效地扒取其他成功直播间的话术，并将其转化为自己可用的脚本。这对于快速学习和模仿优秀主播的销售与互动策略至关重要。

## 概述

在视频号运营中，除了选品和制作视频，直播话术是影响转化率的关键因素。当你发现某个直播间销售数据很好，并且其产品易于复制时，学习他们的话术就成为了快速起号的有效方法。然而，直接记忆或记录直播内容效率低下。本节将介绍一个利用工具高效扒取并整理直播间话术的流程。

## 准备工作：所需工具

进行话术扒取，你需要准备一个核心工具：**飞书**。具体来说，我们需要使用飞书应用内的“飞书妙计”功能。这是一个能够实时录音并转写成文字的工具。

## 操作步骤详解

以下是使用飞书妙计扒取直播间话术的完整步骤。

![](img/3d575de2919a8db2cd5c3effbf53f7f2_1.png)

### 第一步：打开飞书妙计

![](img/3d575de2919a8db2cd5c3effbf53f7f2_3.png)

首先，在飞书应用中，点击底部菜单栏的 **“工作台”**。在工作台页面中，找到并点击 **“飞书妙计”** 功能。

### 第二步：开始录制

进入飞书妙计后，你会看到一个明显的录音按钮（通常位于界面右下角）。点击这个按钮，即可开始录音。

此时，你可以切换到视频号或其他直播平台，打开你想要学习和模仿的**目标直播间**。飞书妙计会在后台持续录制直播间发出的所有声音，包括主播的介绍、互动、逼单话术等。

### 第三步：结束并生成文稿

录制一段时间（建议**15分钟左右**，足以覆盖一个完整的销售循环）后，返回飞书妙计，点击完成或停止录制。

系统会自动将录音内容转换成文字稿。这个过程可能需要几分钟时间处理。

### 第四步：导出与编辑文字稿

文稿生成后，在飞书妙计内找到分享选项。选择生成一个 **“可编辑”** 的链接。

复制此链接，并将其发送到微信的文件传输助手或任意聊天窗口。在微信中打开该链接，你就能看到整理好的直播文字内容。

接下来，你可以复制全部文本，并将其粘贴到文档编辑工具（如Word、备忘录或飞书文档）中。

### 第五步：提炼与形成自己的话术

这是最关键的一步。你不需要照搬所有内容，而是需要分析和提炼。

以下是话术提炼的核心要点：
*   **分析结构**：观察主播是如何开场、介绍产品、解决疑虑、促进下单（逼单）以及结束的。
*   **提取金句**：记录下那些富有感染力、能打动人的句子和关键词。
*   **去除个性化内容**：删除主播与特定观众互动的闲聊、重复或口误的部分。
*   **重组与改编**：将提取出的核心话术，用自己的语言和风格重新组织，形成适合你自己的直播脚本。

整理好后，你可以将最终版话术写在提示卡上，或存入手机备忘录。开播时如果忘记，可以随时查看，确保直播过程流畅自然。

## 总结

本节课我们一起学习了如何利用“飞书妙计”工具扒取其他直播间的优秀话术。这个方法的核心步骤是：**录制 -> 转写 -> 编辑 -> 提炼**。掌握这个技巧，能帮助你快速拆解和学习同行成功的直播经验，有效构建你自己的直播话术体系，从而提升直播间的互动和销售能力。记住，学习的目的是为了创新，最终形成你个人独特的直播风格。
