# 跨境电商副业指南：P6：4.准备工作-邮箱注册（电脑端） 📧

在本节课程中，我们将学习如何为跨境电商项目注册一个至关重要的工具——海外邮箱。邮箱是注册TikTok账号、苹果ID以及其他海外平台的基础，因此掌握其注册方法非常重要。

上一节我们介绍了准备工作的重要性，本节中我们来看看如何具体操作，完成Outlook邮箱的电脑端注册。

---

## 为什么选择Outlook邮箱？

在开始注册前，需要了解为何推荐使用Outlook邮箱。以下是几个核心原因：

![](img/40ae4104efafe6b2a2a02e10b6f1e8eb_2.png)

1.  **注册简便**：流程简单，无需绑定手机号即可完成。
2.  **数量充足**：在同一个IP地址下，每天可免费注册**3个**邮箱账号。
3.  **灵活扩展**：若配合科学上网工具更换IP地址，则可实现邮箱账号的“无限”注册，完全满足项目需求。

## 电脑端注册Outlook邮箱全流程

以下是详细的电脑端注册步骤，请按照指引操作。

**第一步：访问官网**
打开任意浏览器（如微软Edge、谷歌Chrome、360浏览器等），在搜索框中输入“outlook”进行搜索。在搜索结果中，点击进入Outlook的官方网站。



**第二步：创建账户**
进入官网后，在页面中找到“创建免费账户”的按钮并点击。页面加载可能需要一些时间，请耐心等待。



**第三步：设置邮箱地址**
在创建页面，你需要输入一个未被占用的电子邮件用户名。例如，可以尝试组合字母和数字，如 `Carl6688`。若提示“用户名已被占用”，则需更换名称，直到系统提示可用为止。确定后，点击“下一步”。

**第四步：设置密码与个人信息**
接下来，需要设置账户密码，请自行设计并牢记。然后，填写英文姓名（如 `Carl`）和出生日期。**请注意，出生日期必须确保年龄大于18周岁**，例如选择1996年。地区选择“中国”即可。填写完毕后，点击“下一步”。

**第五步：完成人机验证**
系统可能会要求进行图形验证，以证明操作者不是机器人。例如，可能会要求你“选出图片中的所有骆驼”。按照提示完成验证即可，这是简单的傻瓜式操作。

**第六步：完成注册**
验证通过后，系统可能会询问“是否保持登录状态”，可以选择“否”。随后，页面将自动跳转进入新注册的邮箱界面。当看到“欢迎加入Outlook”等提示信息时，即表示邮箱注册成功。



**第七步：查看账户信息**
要确认注册的完整邮箱地址，可以点击页面右上角的账户头像或名称缩写（如 `CA`）。在弹出的账户信息中，即可看到完整的邮箱地址，格式为 `[你设置的用户名]@outlook.com`。
