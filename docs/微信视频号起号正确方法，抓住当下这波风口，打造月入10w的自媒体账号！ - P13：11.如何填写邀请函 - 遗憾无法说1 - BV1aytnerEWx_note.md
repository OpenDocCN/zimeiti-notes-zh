# 微信视频号运营教程：P13：11.如何填写邀请函

在本节课中，我们将学习如何正确填写视频号创作者计划的邀请函。这是加入计划、获取官方资源支持的关键一步。整个过程简单明了，我们将一步步进行拆解。

## 概述
填写邀请函是加入视频号创作者计划的必要流程。本节将详细讲解从查找个人账号信息到最终提交申请的每一个步骤，确保信息准确无误。

![](img/bcb7b903e108ee0283a1f72753d9dadc_1.png)

---

上一节我们介绍了如何注册视频号，本节中我们来看看如何填写官方的邀请函以加入创作者计划。

### 第一步：打开微信应用
操作的第一步是启动手机上的微信应用。

![](img/bcb7b903e108ee0283a1f72753d9dadc_3.png)

以下是具体操作步骤：
1.  在手机主屏幕或应用列表中找到微信图标。
2.  点击图标，打开微信应用。

![](img/bcb7b903e108ee0283a1f72753d9dadc_4.png)

### 第二步：进入“发现”页面
打开微信后，我们需要进入包含“视频号”入口的页面。

以下是具体操作步骤：
1.  在微信底部导航栏中，找到并点击 **“发现”**。
2.  成功进入“发现”页面。

![](img/bcb7b903e108ee0283a1f72753d9dadc_6.png)

![](img/bcb7b903e108ee0283a1f72753d9dadc_1.png)

### 第三步：找到并进入“视频号”
在“发现”页面中，我们可以找到视频号的入口。

![](img/bcb7b903e108ee0283a1f72753d9dadc_8.png)

以下是具体操作步骤：
1.  在“发现”页面中向上滑动。
2.  找到标有 **“视频号”** 字样的入口（其图标通常类似一只黄色的蝴蝶）。
3.  点击该入口，进入视频号界面。

![](img/bcb7b903e108ee0283a1f72753d9dadc_9.png)

![](img/bcb7b903e108ee0283a1f72753d9dadc_3.png)
![](img/bcb7b903e108ee0283a1f72753d9dadc_4.png)

### 第四步：进入个人主页
进入视频号界面后，下一步是找到并进入自己的个人主页，以查看账号信息。

以下是具体操作步骤：
1.  点击视频号界面右上角的 **个人头像图标**。
2.  进入个人主页。

![](img/bcb7b903e108ee0283a1f72753d9dadc_6.png)

### 第五步：查看账号昵称与粉丝数
在个人主页中，我们可以找到填写邀请函所需的核心信息。

以下是具体操作步骤：
1.  在个人主页中向下滑动。
2.  找到 **“我的视频号”** 栏目。
3.  该栏目会明确显示你的**视频号昵称**和**粉丝数量**。
    *   例如：`天道酬勤，39917个粉丝`
    *   其中 **“天道酬勤”** 即为账号昵称。
    *   **“39917”** 即为粉丝数量。

![](img/bcb7b903e108ee0283a1f72753d9dadc_8.png)
![](img/bcb7b903e108ee0283a1f72753d9dadc_9.png)

---

在成功查找到自己的账号信息后，我们就可以开始填写邀请函的各个字段了。

![](img/bcb7b903e108ee0283a1f72753d9dadc_11.png)

![](img/bcb7b903e108ee0283a1f72753d9dadc_12.png)

### 邀请函填写详解
邀请函通常包含以下几个必填项，请根据以下指引准确填写：

![](img/bcb7b903e108ee0283a1f72753d9dadc_13.png)

1.  **视频号账号昵称**
    *   **填写内容**：直接填写你在第五步中查看到的账号昵称，例如：`天道酬勤`。

2.  **粉丝数量**
    *   **填写内容**：填写你在第五步中查看到的粉丝数字，例如：`39917`。

3.  **活动邀请人**
    *   **填写内容**：根据你的邀请渠道，填写指定的邀请人姓名，例如：`张良` 或 `张良老师`。

4.  **实名信息**
    *   **填写内容**：必须填写你本人的真实姓名，与身份证信息一致，例如：`周建国`。

5.  **结算手机号**
    *   **填写内容**：填写你本人用于接收验证码和后续结算的手机号码，例如：`13812345678`。
    *   **注意**：号码需连续填写，中间不要有空格。

### 最终提交与确认
所有信息填写完毕后，需要进行最终提交。

以下是具体操作步骤：
1.  检查所有填写的信息是否准确无误。
2.  点击页面上的 **“提交”** 或 **蓝色提交按钮**。
3.  系统会弹出确认框（如“确定要提交吗？”），点击 **“确认”**。
4.  提交成功后，页面通常会显示“已成功加入创作者计划”等提示。

### 附加内容：关于创作素材
成功加入创作者计划后，你将有机会获得官方提供的正版素材库支持，用于内容创作。

![](img/bcb7b903e108ee0283a1f72753d9dadc_20.png)

![](img/bcb7b903e108ee0283a1f72753d9dadc_22.png)

例如，素材库中可能包含已获得版权授权的影视动漫内容，如：
*   `还珠格格`
*   `我不是药神`
*   `大话西游`

这些素材可以有效解决创作者在内容制作时的版权顾虑，提升创作效率与安全性。

## 总结
本节课中我们一起学习了填写视频号创作者计划邀请函的完整流程。关键点在于：首先，通过微信发现页进入视频号并找到个人主页，获取准确的**账号昵称**和**粉丝数**；然后，按照要求填写邀请人、实名信息及手机号；最后，仔细检查并提交申请。确保信息真实准确是成功加入计划的基础。
