# TikTok自动化教程：P1：使用TTY工具自动发布视频

在本节课中，我们将学习如何使用名为“TTY-天天”的工具，实现TikTok视频的自动化发布。我们将从参数设置开始，逐步讲解如何准备视频文件、配置发布选项，并最终完成自动化发布流程。

## 🛠️ 参数设置与基础配置

上一节我们介绍了课程目标，本节中我们来看看如何配置工具的核心参数。首先，点击打开工具的“参数设置”界面。

以下是发布视频前必须填写的四项核心参数：
*   **发布视频前的养号时间**：设置在发布视频前，账号需要进行的准备时间。
*   **发布视频后的养号时间**：设置在发布视频后，账号需要进行的维护时间。
*   **视频音效**：此选项有三种模式可供选择。
    *   **无音效**：发布的视频将不使用任何额外音效。
    *   **随机音效**：工具会从音效库中随机选择一个音效添加到视频。
    *   **收藏音效**：工具会使用您预先收藏的特定音效。
*   **挑战与定位**：根据您的需要，决定是否在发布视频时添加挑战话题或地理位置信息。

在参数区域的右侧，您可以设置**单轮发布视频的数量**，即一次自动化流程中计划发布的视频个数。

## 📁 准备视频与文案素材

了解了基础配置后，接下来我们需要准备要发布的视频和文案素材。这是自动化流程成功运行的关键。

首先，找到“视频文件路径”设置项。首次使用时，需要点击“初始化目录”按钮。此操作会自动在您指定的路径下创建一个文件夹和一个示例文件。

初始化完成后，请打开该文件夹。您需要将计划发布的视频文件放入此路径内。

接着，打开同一目录下的“标签文本”文件。在此文件中，您可以编写视频的文案和标签。请注意：
*   每个标签单独占一行。
*   输入标签时**不需要**添加井号（`#`）。
*   在标签之后或单独的区域，可以填写视频的正文内容文案。

所有素材准备就绪后，关闭文件夹窗口。

## ▶️ 执行自动化发布流程

素材准备完成后，我们就可以启动自动化发布了。首先，在主界面点击“运行表格”旁的“刷新”按钮，以获取当前可用的模拟器设备列表。

![](img/fbc52c67ced0da737b6b33af92156962_1.png)

从列表中勾选您想要运行自动化任务的模拟器设备。

在“功能”选项中，确保勾选了“发布视频”。然后，点击“开始”按钮，并打开“操作日志”窗口以观察运行状态。

此时，工具将自动执行以下流程：
1.  自动打开TikTok应用。
2.  从本地文件夹上传视频到模拟器的相册中。
3.  开始发布视频流程：
    *   从模拟器相册选择视频。
    *   选择音乐（根据之前设置的“随机音效”等模式）。
    *   点击添加“视频挑战”。
    *   输入预先准备好的文案和标签。
    *   添加地理位置信息。
4.  点击发布按钮，完成视频上传。
5.  发布完成后，工具会自动删除本地和模拟器相册中的视频源文件。
6.  流程结束，任务完成。
