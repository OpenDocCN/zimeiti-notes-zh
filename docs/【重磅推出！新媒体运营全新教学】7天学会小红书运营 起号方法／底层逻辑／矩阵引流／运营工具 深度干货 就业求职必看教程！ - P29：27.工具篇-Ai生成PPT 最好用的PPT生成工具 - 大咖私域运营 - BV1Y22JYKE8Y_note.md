# 小红书运营：P29：27.工具篇-AI生成PPT 🚀

在本节课中，我们将学习一个非常实用的AI工具，它能根据主题或现有文档，自动生成结构清晰、内容详实的PPT。这个工具能极大提升内容创作的效率。

## 工具简介与模式

上一节我们介绍了多种运营工具，本节中我们来看看如何利用AI辅助内容制作。这个AI生成PPT工具主要提供两种工作模式。

以下是工具的两种核心模式：

1.  **主题生成模式**：用户输入一个主题，工具会结合类似ChatGPT的生成式AI能力，自动创作出完整的PPT全文。
2.  **文档生成模式**：用户上传一篇已有的文档（如TXT、PDF），AI助手会先智能总结文档大纲，然后基于此大纲生成详细的PPT内容。

## 内容编辑与模板应用

生成初步内容后，我们可以对PPT进行进一步的优化和美化。这是让PPT更加符合个人需求的关键步骤。

![](img/3b45c5f37d18e16bc6a2fa4db03e11be_13.png)

工具右侧提供了“更换模板”的选项。虽然当前内置的模板数量有限，但切换速度非常快。我们可以选择一个合适的模板来快速改变整体风格。

![](img/3b45c5f37d18e16bc6a2fa4db03e11be_13.png)

这个工具最突出的优势在于，它能基于我们提供的**零散内容或大纲**，自动归纳、拓展并生成详细的PPT正文。这与许多其他工具不同，那些工具往往要求用户提供完整的讲稿，它们只负责格式排版。

在编辑界面，我们还可以对每一页的具体内容进行调整。

以下是可进行的编辑操作：
*   修改文字内容与格式。
*   替换或调整图形、形状。
*   调整页面布局。

![](img/3b45c5f37d18e16bc6a2fa4db03e11be_15.png)

如果对在线编辑效果满意，我们可以直接下载PPT文件到本地，使用PowerPoint或Keynote进行更深度的自定义调整。
