# 抖音短视频AI创作教程：P2：抄作业教程 📝

![](img/385b7248ceb42ac9e50c99d014357e10_0.png)

在本节课中，我们将学习一种通过“抄作业”方式，利用AI工具辅助创作抖音温情故事视频的完整方法。该方法操作简单，适合新手，核心在于借鉴爆款内容并加以处理，以实现高效发布和获取收益。

---

上一节我们介绍了利用AI进行视频创作的潜力，本节中我们来看看如何具体执行“抄作业”这个方法。

## 第一步：寻找爆款选题 🔍

首先，需要在抖音确定一个容易引发共鸣、容易获得高播放量的视频赛道。这里推荐“温情故事”类视频。

以下是具体操作步骤：
1.  打开抖音，在搜索框内输入“温情故事”。
2.  点击顶部的“视频”板块。
3.  将筛选条件设置为“最多点赞”。
4.  此时，页面会展示大量点赞量高达百万的爆款温情视频。

## 第二步：提取并处理文案 ✂️

找到目标视频后，下一步是获取其文案内容，但切记不能直接照搬。

以下是具体操作步骤：
1.  随意点击一个爆款视频进入播放页。
2.  点击视频右下角的“分享”按钮。
3.  在弹出的菜单中，找到并点击“复制链接”。
4.  打开一款文案提取工具（如某些微信小程序或在线网站）。
5.  将复制的链接粘贴到工具的“链接转文案”功能中，即可提取出视频的完整文案。
6.  **关键步骤**：不要直接使用提取的原文案。请通读全文，理解故事脉络后，用自己的语言和表达方式将故事重新讲述一遍。这样可以有效避免违规风险。

## 第三步：AI一键生成视频 🎬

拥有自己的文案后，即可利用剪辑软件的AI功能快速生成视频。

以下是具体操作步骤：
1.  打开“剪映”APP。
2.  将第二步中自己改写好的文案导入或粘贴到剪映中。
3.  使用剪映的“一键成片”功能。
4.  软件会自动根据文案内容匹配相关视频画面，并生成同步的语音配音。整个过程无需手动剪辑。

## 第四步：发布与变现 💰

视频制作完成后，需要通过正确的渠道发布以参与中视频计划获取收益。

以下是具体操作步骤：
1.  在抖音顶部搜索框搜索“中视频计划”。
2.  点击进入“中视频伙伴计划”的官方入口。
3.  通过该入口发布你制作好的视频。
4.  发布时，务必勾选“同步到西瓜视频和今日头条”。
5.  此后，你的视频在抖音、西瓜视频、今日头条三个平台产生的播放量，都会为你带来收益。

---

本节课中我们一起学习了“抄作业”制作抖音视频的四步法。这个方法的核心公式可以总结为：

**爆款收益 = 爆款选题 + 爆款文案（改编）+ AI工具辅助**

其简单在于流程固定，不简单在于需要你耐心执行，并能对爆款文案进行有效的理解和转化。只要能够坚持操作，就有机会在平台上获得收益。
