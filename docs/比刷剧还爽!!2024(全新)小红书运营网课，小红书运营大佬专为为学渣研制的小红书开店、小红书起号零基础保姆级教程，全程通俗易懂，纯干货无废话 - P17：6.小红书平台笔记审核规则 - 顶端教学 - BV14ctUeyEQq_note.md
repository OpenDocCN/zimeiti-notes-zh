## 笔记审核的三个阶段

笔记上传后，首先会进入审核系统。这个过程主要分为三个阶段：机器审核、去重检测和人工审核（如需要）。

### 1. 机器审核
系统会根据预设的基本规则对笔记进行初步筛查。这些规则主要包括：
*   **敏感词检测**：识别文案中的违规或敏感词汇。
*   **敏感图片识别**：通过图像识别技术检测图片是否包含违规内容。

如果笔记内容明确违规，可能会被直接处理。如果内容处于疑似违规的灰色地带，机器无法准确判断，则会标记该笔记并转入人工审核阶段。

### 2. 去重检测
通过机器审核的笔记，会进入去重检测阶段。系统会将你的笔记与平台数据库中的现有内容进行比对，计算相似度。比对维度包括：
*   **图片相似度**：是否为直接复制、盗用他人的图片。
*   **文案重复率**：文案内容、段落结构在数据库中的重复比例。

以下是判断逻辑的简化表示：
```python
if 笔记相似度 == 100%:
    处理方式 = “直接封禁”
elif 笔记相似度 > 阈值:
    处理方式 = “限流或转入人工审核”
else:
    处理方式 = “通过检测”
```
因此，直接抄袭的笔记（相似度100%）会被封禁。如果对图片进行裁剪、加滤镜，或对文案进行改写、调整结构，则有可能降低相似度，通过此关。

### 3. 人工审核
人工审核主要针对机器标记的“疑似违规”内容进行二次复核。由于内容量巨大且人工成本高，并非所有笔记都会经历此阶段。人工审核资源会重点向特定高风险领域倾斜，例如：
*   政治相关
*   色情擦边
*   其他法律严禁内容

![](img/bc8e814e498cf8f805dc8af2f1b0130e_8.png)

目前，平台的机器审核算法已相当成熟，误判率较低。一旦笔记触发明确的违规红线，通常会被直接处理，很少再进入人工复核。如果创作者认为自己的笔记被误判，可以通过官方渠道申诉，但新账号应尽量避免违规，以免对账号权重产生隐形负面影响。

在了解了审核阶段后，我们来看看系统如何评估一篇笔记的“基础质量”。这决定了笔记的初始权重。

## 笔记质量评估与初始权重

除了审核违规内容，机器在审核阶段还会对笔记的“基础质量”进行打分。评估维度包括：
*   **笔记字数**：内容是否充实。
*   **广告词频率**：是否含有过多营销词汇。
*   **图片质量**：是否清晰，系统能否识别图中物体和文字。
*   **内容识别度**：文案中的关键信息是否易于提取。

如果笔记图片高清、内容充实且易于识别，系统会给予一个相对较高的**初始权重分**。这个分数是笔记进入下一阶段——用户匹配和流量分发——的起点。

---

笔记的初始权重会受到发布账号本身状态的影响。接下来，我们探讨账号权重这个核心概念。

## 账号权重的影响

![](img/bc8e814e498cf8f805dc8af2f1b0130e_10.png)

账号权重是系统对账号整体质量的一个隐形评分，它直接影响笔记发布后获得的初始流量池大小。权重并非固定不变，而是随着账号行为动态变化的。

以下是账号在不同阶段的权重特点：

**新号阶段（低权重期）**
*   **权重状态**：可理解为“零权重”或极低权重。
*   **流量特征**：笔记初始推荐量通常很低（如几十个浏览量）。
*   **核心任务**：此阶段的核心是“养号”。需要特别注意：
    *   **避免违规**：坚决不发布违禁词、违规内容。
    *   **保持垂直**：持续发布同一领域的优质内容。
    *   **稳定更新**：维持规律的发布频率。
*   **原因分析**：新号缺乏标签，系统无法将你的笔记精准推送给可能感兴趣的用户，导致互动率低，笔记难以突破初始流量池。

**成熟账号阶段（高权重期）**
*   **权重状态**：在特定领域内积累了一定权重分（例如，假设系统评分范围是0-10，此时可能达到5-8分）。
*   **流量特征**：笔记发布的初始流量池会显著扩大。
*   **获得方式**：通过长期、稳定、垂直地发布优质内容，并获得良好的用户互动（点赞、收藏、评论、长停留时间），系统会逐步提升账号在对应领域的权重。

**重要提示**：账号权重是系统内部的、非公开的综合评价指标。我们将其理解为“权重”，是为了更直观地理解账号状态对流量的影响。对于新账号，首要目标是避免违规并通过优质行为积累权重，而不是期待初期就获得大量流量。
