## 两种生成方式 🔧

![](img/483027275dacab1e82d6bb65442d2804_3.png)

该工具主要提供两种生成PPT的方式：
1.  **AI智能生成**：直接输入主题，由AI自动构思大纲和内容。

我通常使用“导入本地大纲”的方式，这样对内容的把控度更高。

### 方式一：导入本地大纲生成

以下是使用“导入本地大纲”功能的具体步骤。

首先，我们需要准备一份详细的大纲文档。可以使用ChatGPT等AI助手来辅助撰写。

**示例：使用ChatGPT生成大纲**
我们可以向ChatGPT发出如下指令：
```plaintext
请以“装修行业在小红书平台的运营推广计划”为主题，撰写一份详细的PPT大纲，并完善每一个小节点的具体内容信息，保持清晰的结构和格式排版。
```
AI生成的大纲可能在某些细节上不够详尽，例如“教育内容制作”部分可能缺乏拓展。我们可以通过完善指令要求AI补充，但本节课中我们将基于现有大纲进行演示。

准备好大纲文档后，即可在工具中操作。

**操作步骤列表：**
1.  在工具界面选择“导入本地大纲”。
2.  选择“导入本地文件”，上传准备好的大纲文档。
3.  工具会快速梳理文档的大纲结构。
4.  在丰富的模板库中，挑选一个适合主题（如装修行业）的模板风格。
5.  点击“生成PPT”，等待生成完成。

生成完成后，如果发现某些内容没有正确加载（例如，“内容趋势”下方的具体说明缺失），可以在工具的“大纲编辑”环节进行补充。

**编辑与调整：**
生成后的PPT可以在线编辑，右侧工具栏提供完整的设置选项。
*   **文字/形状/背景设置**：可调整页面各元素。
*   **页面排版更换**：单击页面，可选择其他预设排版一键更换。
*   **整体模板替换**：通过“模板替换”功能，可以快速为整个PPT更换另一套模板样式，响应速度很快。

### 方式二：AI智能生成

![](img/483027275dacab1e82d6bb65442d2804_8.png)

![](img/483027275dacab1e82d6bb65442d2804_10.png)

接下来，我们尝试使用“AI智能生成”功能。

**操作步骤列表：**
1.  在工具界面选择“AI智能生成”。
2.  在输入框中填写明确的主题，例如：“装修行业在小红书平台的运营推广计划”。
3.  点击生成按钮，AI会开始自动构思大纲。
4.  大纲生成后，从模板库中挑选合适的样式。
5.  点击“生成PPT”，AI将根据主题自动生成完整的PPT文案和页面。

这种方式下，AI会尝试生成更详细的页面内容。生成完毕后，同样可以进入编辑界面，对所有字体、样式、模板进行调整，或编辑大纲的具体内容。
