# 小红书运营实操课：P5：第4课：小红书商品电脑上架实操 💻

在本节课中，我们将学习如何通过电脑端为小红书店铺上架商品。电脑上架的优势在于可以批量操作，效率更高。我们将详细介绍所需的工具、设置步骤以及上架后的检查流程。

上一节我们介绍了小红书店铺的基础设置，本节中我们来看看如何具体上架一个商品。

## 上架方式与工具介绍

![](img/784629a68c552613bff466d1675dfe23_1.png)

小红书带货的商品上架主要分为两种方式：电脑上架和手机上架。电脑上架操作更便捷，支持批量处理。

![](img/784629a68c552613bff466d1675dfe23_3.png)

![](img/784629a68c552613bff466d1675dfe23_5.png)

以下是常用的商品上架辅助工具：

*   **蚂蚁搬家**：每月19元。功能最常用，支持搬运小红书、抖音链接以及1688链接。
*   **沐浴助手**：每月15元。仅支持搬运1688链接。
*   **欢乐逛**：每月15元。
*   **妙手**：每月15元。

其中，**蚂蚁搬家**是使用最广泛的工具。

## 工具安装与授权

如何找到并安装“蚂蚁搬家”工具？操作步骤如下：

1.  打开小红书商家后台。
2.  点击导航栏中的 **“服务”**。
3.  点击 **“前往服务市场”**。
4.  在服务市场的搜索栏中，输入 **“蚂蚁搬家”** 并进行搜索。
5.  找到该工具后，点击进入详情页，选择 **“订购”** 或 **“获取”**。
6.  根据提示完成授权登录。

授权成功后，下次使用时，可以在“服务” -> **“我的服务”** 中直接找到并点击“去使用”。

## 工具核心设置

使用工具前，需要进行几项关键设置，以确保商品信息正确无误。

以下是“蚂蚁搬家”工具的常规设置项：

*   **常规设置**：打开“过滤商品”选项。
*   **品牌设置**：选择 **“我不使用品牌”**，因为多数搬运商品为白牌（无品牌）。
*   **运费模板**：选择已设置好的包邮模板（需提前创建）。
*   **物流模板**：选择支持“快递”的模板。若未创建，需退出工具重新登录后才能添加。
*   **库存设置**：一般设置为 **`10000`**。
*   **价格设置**：
    *   **售卖价公式**：`原价 * 200% + 5`
    *   **市场价公式**：`原价 * 200% + 5`
    *   对于其他不支持百分比设置的软件，直接填写倍数 **`2`** 并加5即可。
    *   **注意**：不要填写“区间价”，否则可能出错。
*   **图片设置**：按需设置（后续课程会详细讲解）。
*   **SKU与条形码**：可按需查看或添加。

**设置高价的原因**：为后续设置店铺满减券、折扣券或参加平台活动预留利润空间，确保利润率不低于30%。

## 商品上架实操步骤

设置完成后，即可开始搬运并上架商品。我们以搬运1688商品为例。

![](img/784629a68c552613bff466d1675dfe23_7.png)

以下是具体的操作流程：

![](img/784629a68c552613bff466d1675dfe23_9.png)

1.  **复制商品链接**：在1688网站找到目标商品，复制其商品详情页的链接。
2.  **在工具中粘贴链接**：将复制的链接粘贴到“蚂蚁搬家”工具的搜索或抓取框中，点击“搜索”或“抓取”。
3.  **确认并复制**：工具解析出商品信息后，点击“复制到店铺”。
4.  **查看上架状态**：在工具的 **“任务列表”** 中，可以查看商品的上架处理状态。**绿色标识**代表“处理成功”。

## 上架后检查与提交

商品显示上架成功后，必须进入小红书后台进行最终检查，确保符合平台规范。

以下是必须检查的关键项目：

1.  进入小红书后台的 **“商品管理”** -> **“商品列表”**，找到刚上架的商品。
2.  点击商品右侧的 **“编辑”** 按钮。
3.  仔细检查：
    *   **商品图片**：删除所有带有第三方平台水印、Logo或其他平台文字介绍的图片。
    *   **商品描述**：检查并删除正文中提及其他平台的描述。
    *   **物流方式**：实物商品选择 **“物流配送”**；虚拟课程/服务选择 **“无物流发货”**。
    *   **商品类型**：选择 **“国内贸易”**。
    *   **预售设置**：选择 **“非预售”**。
    *   **发货时间**：一般设置为 **“48小时”** 内发货。
    *   **运费模板**：确认选择正确的包邮模板。
4.  确认所有信息无误后，点击 **“提交商品”** 完成上架。
