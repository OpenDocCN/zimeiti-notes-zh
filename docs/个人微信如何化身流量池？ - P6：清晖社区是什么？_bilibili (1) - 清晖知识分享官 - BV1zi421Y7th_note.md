# 清晖社区介绍：P6：清晖社区是什么？ 🏘️

在本节课中，我们将要学习清晖社区的定义、核心功能与价值，帮助你全面了解这个综合性成长平台。

## 概述

清辉社区是一个集学习交流、交友于一体的综合性成长平台。

## 核心功能与价值

上一节我们概述了清晖社区的平台性质，本节中我们来看看它的具体关注点与服务。

清辉社区主要关注清辉学院考前考后的学习发展以及持有证书后的证书管理。

以下是加入清晖社区可以享受的核心服务：
*   一站式考后学习支持。
*   专业的证书管理服务。

## 总结

本节课中我们一起学习了清晖社区。它是一个集学习、交流与交友于一体的成长平台，专注于为学员提供从考试到考后学习乃至证书管理的全程支持。山水清辉，你我同行。
