在本节课中，我们将学习第二款强大的AI生成PPT工具。这款工具在生成速度、稳定性和模板丰富度方面具有优势，能与上一节课分享的工具形成互补，共同提升PPT制作效率。

## 工具使用方式概述

该工具主要提供两种生成方式：**AI智能生成**和**导入本地大纲**。通常，导入精心准备的本地大纲能获得更符合预期的效果。

以下是两种方式的具体操作步骤。

### 方式一：导入本地大纲生成

首先，我们需要准备一份详细的PPT内容大纲。可以使用ChatGPT等AI助手辅助生成。

![](img/c52fe9604bd40a0f375086db78a06430_1.png)

![](img/c52fe9604bd40a0f375086db78a06430_3.png)

例如，向ChatGPT输入指令：
```
请为“装修行业在小红书平台的运营推广计划”这个主题，生成一份详细的PPT大纲，并完善每一个小节点的具体内容信息，保持好结构和格式排版。
```

![](img/c52fe9604bd40a0f375086db78a06430_5.png)

![](img/c52fe9604bd40a0f375086db78a06430_6.png)

获得大纲文本后，即可进入工具操作。

1.  在工具界面选择“导入本地大纲”。
2.  上传包含大纲内容的文档文件。
3.  工具会快速解析文档，提取出大纲结构。
4.  从丰富的模板库中，选择一个适合行业风格（如装修行业）的模板。
5.  点击“生成PPT”。

生成后，若发现部分内容未完整加载，可进入“大纲编辑”界面进行补充。在对应章节下，通过回车键添加详细内容。

对于不满意的单页排版，可以单击该页面，从右侧提供的多种布局中选择一键更换。

### 方式二：AI智能生成

如果希望工具从头开始自动生成内容，可以使用此方式。

1.  在工具界面选择“AI智能生成”。
2.  在输入框中直接填写PPT主题，例如：“装修行业在小红书平台的运营推广计划”。
3.  点击生成按钮，工具将自动构思大纲并填充详细内容。
4.  同样，在生成后挑选合适的模板。
5.  点击“生成PPT”并进入编辑界面。

在此方式下，工具会根据主题自动生成完整的文案内容。之后，用户同样可以在编辑界面调整字体、样式、更换模板，或通过“大纲编辑”功能修改具体内容。

## 工具特点与使用技巧

这款工具反应迅速，模板多样，基本能满足PPT编辑的常规需求。为了获得最佳效果，请注意以下两点：

*   **内容准备是关键**：无论是AI生成还是导入大纲，提供**结构清晰、细节丰富**的原始内容，能显著提升最终PPT的质量。
*   **灵活运用编辑功能**：生成初稿后，善用“模板替换”、“页面布局更换”等功能进行微调，可以使PPT更符合个人审美与专业要求。
