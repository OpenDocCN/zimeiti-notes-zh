## 概述
在本节课中，我们将学习如何训练AI（如ChatGPT）来撰写符合小红书平台风格的文案。上一节我们介绍了如何让AI生成爆款标题，本节我们将深入探讨如何让AI理解并创作出符合平台调性的完整内容。

## 从标题到内容：训练AI的核心思路
上一节我们介绍了如何让AI写出符合平台风格的标题。这个方法不仅适用于小红书，也适用于抖音、哔哩哔哩等其他平台。每个平台都有其独特的关键词和文化，以及大众认可的标题形式。关键在于提取这些重点关键词，整理成自己的“标题库”。每次需要写标题时，将库中的关键词和你的主题结合，AI就能生成符合平台风格的标题。

关于AI写标题的内容就介绍到这里。接下来，我们需要重点研究如何让AI写出符合爆款结构或平台风格的内容体系。

## 训练AI理解文案风格
大部分内容ChatGPT都能生成，但如何让它写出符合特定平台风格的内容呢？这需要讲究几个关键点。

第一个关键点是**文案风格**。你必须让AI了解你想要的风格。为此，你需要提供案例，告诉AI：“我提供这些内容作为参考，我希望你写出类似风格的内容。” 这意味着你需要给AI“喂”一定量的、其他人的爆款文章。

这些文章能帮助AI分析出平台特有的风格，例如：
*   女性化的专用用语。
*   少女般活泼气息的词语。
*   平台专用语，如小红书上常见的“宝子”等称呼。

![](img/458e56ced7b37aac3cab13cdff3f65d8_1.png)

我们的目标是让ChatGPT通过分析我们提供的材料，自行总结出这些风格特点，并记忆在其“资料库”中，以便后续根据这些特点进行创作。

以下是训练AI理解文案风格的具体步骤：

首先，我们需要明确告知AI接下来的任务。你可以使用类似以下的指令：

```
接下来，我会为你提供一些小红书上的爆款笔记文案。你需要根据这些文案，分析小红书的文案撰写风格、特定的平台用语、爆款词汇、语气等。
每接收一篇文案后，请告诉我你已经理解并学习到了，同时给出你对这篇文案的风格、语气、类型等相关分析。
```

这个指令明确了AI需要做的事情：学习我们提供的资料库，并给出相应的分析和反馈。反馈很重要，它能让我们确认AI是否真的“学会”了。

## 实战演练：以旅游攻略为例
我们以“张家界旅游攻略”为例，进行一个简单的实操。

![](img/458e56ced7b37aac3cab13cdff3f65d8_10.png)

**第一步：提供第一篇文案**
我们找到一篇关于张家界的笔记，将标题和正文内容提供给AI。最好将标题和正文分开标注。

```
标题：省事儿又省钱！张家界5日游超全攻略
文案：[此处粘贴完整的文案内容]
```

AI收到后，可能会给出如下分析：
*   “以个人游玩经历为主线，增加了真实感和亲近感。”
*   “以‘血泪教训’和‘不踩坑建议’为标题亮点，强调了作者的经验教训，传达了一种亲切的分享心态。”
*   “语气类型为担忧和焦虑，同时表达祝愿。”

这个分析过程，就是AI在学习这篇文案的风格。

**第二步：持续提供多样化的文案**
训练AI不是一次性的工作。在同一个对话中持续提供资料，AI会保留对之前内容的“记忆”。

我们需要寻找多种风格、有特色的文案喂给AI，而不仅仅是通用的行程列表（如“第一天、第二天…”）。重点在于让AI学习：
*   作者是用第几人称（我、你、他）来表述的？
*   平台特有的用语和语气是什么？
*   内容是如何组织并突出亮点的？

例如，我们可以寻找“吐槽避坑”类、“强烈种草”类、“情感分享”类等不同风格的笔记提供给AI。

**注意**：你提供的资料质量直接决定AI的学习效果。应选择能鲜明体现小红书行文风格的文案。

## 让AI进行创作
在提供了一定量的学习材料后，就可以让AI尝试创作了。

我们可以给出创作指令：
```
请结合你的学习，帮我写一篇具有爆款潜质的“张家界5日游攻略”小红书笔记。内容需涵盖丰富的小红书风格的emoji表情，同时配上十个符合小红书旅游赛道风格的爆款标题。
```

**初次创作结果分析**
AI生成的初稿可能不尽如人意。例如，它可能只罗列了景点名称（如“天门山景区”、“大峡谷”），但缺乏实用细节：
*   景点有哪些具体亮点和必看之处？
*   路线应该如何规划更合理？
*   有哪些需要避开的“坑”？

这反映出两个问题：
1.  **训练材料不足或不够典型**：我们提供的旅游攻略类文案如果大多只是行程列表，AI就学不到更深层的“避坑”、“种草”等技巧。
2.  **指令可以更细化**：我们可以分步骤指导AI。例如，先让它列出张家界所有主要景点，再针对每个景点规划路线、分析特色和注意事项，最后整合成一篇攻略。

**优化创作：结合专项指令**
对于标题，我们可以直接应用上一节课的“爆款标题指令”和“爆款关键词库”，让AI在专门的对话中生成，这样效果会更精准。

例如，创建一个新对话，输入：
```
请扮演一位小红书旅游博主，根据以下爆款关键词：[明星同款、C位、出片、氛围感、超低价]，为“张家界5日游攻略”生成10个爆款标题。
```

这样生成的标题会更符合小红书的流行趋势。

## 核心总结与工具选择
本节课的核心是告诉大家如何训练AI成为你所在领域的“专家”。

1.  **理解要求**：AI知识丰富，但不一定理解你的具体领域要求。我们需要通过提供案例（喂资料）和清晰指令，让它理解我们的真实需求。
2.  **持续训练**：训练是一个持续的过程，提供的资料越丰富、越典型，AI的输出就越精准。
3.  **工具差异**：高级模型（如GPT-4）在理解复杂指令、生成创意内容方面通常比基础模型（如GPT-3.5）聪明得多，能显著提升创作质量。
