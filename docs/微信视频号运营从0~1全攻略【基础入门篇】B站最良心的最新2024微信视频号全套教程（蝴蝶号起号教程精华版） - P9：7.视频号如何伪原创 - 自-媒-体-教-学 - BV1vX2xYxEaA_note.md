## 平台原创扶持政策
上一节我们介绍了发布作品的基础步骤。平台本身也支持原创，并设有创作者激励计划，对原创内容提供流量倾斜。

以下是创作者激励计划的具体内容：
*   **流量扶持**：平台计划让1000万原创作者获得流量。
*   **收入激励**：让百万优质创作者获得收入。
*   **零粉阶段**：平台会赠送2500到5万点的流量券，相当于直接推送观众。
*   **千粉阶段**：可加入重点作者扶持计划，优质视频可申报专项奖金。
*   **万粉及百万粉阶段**：可申报的专项奖金更多。

因此，原创内容能获得平台一级一级的流量池推进。当然，这需要你具备相应的技术能力，例如会拍摄、会编写台本。

## 寻找灵感与选品定位
在开始搬运流程前，你可以从其他平台寻找灵感。例如，可以使用西瓜数据、飞瓜数据、微信指数等工具。

在视频号内部，你也可以查看排行榜。例如，如果你想做影视剧类账号，可以查看影视剧排行榜，研究做得好的账号，借鉴他们的优点。音乐类账号同理。这能帮助你了解什么内容容易火。

伪原创有一套完整的流程。首先，你需要进行选品，确定你要卖什么商品。确定好商品后，再去找相关的视频内容。

以下是选品的具体操作步骤：
1.  在视频号主页，点击右上角的小人头图标。
2.  进入个人页面后，找到底部的“创作者中心”。
3.  点击进入创作者中心，可以在这里进行选品规划。

![](img/cd2157694d915b84c5439d3da50ab0c0_8.png)
![](img/cd2157694d915b84c5439d3da50ab0c0_9.png)

你也可以去其他电商平台选品，例如阿里巴巴1688或拼多多，确定要带什么货。

![](img/cd2157694d915b84c5439d3da50ab0c0_8.png)

**公式：账号定位 ≈ 带货品类**
例如：
*   做音乐类账号，可以卖车载U盘或接音乐广告。
*   做影视剧类账号，可以接广告或通过内容吸引粉丝后卖货。
*   做宠物类账号，可以卖宠物相关产品。

![](img/cd2157694d915b84c5439d3da50ab0c0_9.png)

假设我要卖宠物类产品，我就定位一个宠物类型的账号，然后去搬运宠物相关的内容。

## 伪原创核心操作流程
确定了账号定位和选品后，接下来是具体的伪原创操作。这个过程不是直接下载发布，而是需要进行处理。

以下是伪原创的核心步骤：
1.  **寻找对标账号**：在其他平台（如抖音）发现一个不错的账号。
2.  **检查重复性**：必须查看该账号的内容是否已在视频号被搬运过。如果已有人搬运，你再发就会造成内容重复。
3.  **确认目标**：如果确认该账号内容在视频号未被搬运，则可将其确定为搬运对象。
4.  **去水印与二次剪辑**：找到具体作品后，不能直接下载发布。第一步是去除原视频的水印。因为平台支持原创，直接搬运带他人水印的作品可能导致账号违规。

![](img/cd2157694d915b84c5439d3da50ab0c0_11.png)

记住，伪原创的关键在于“伪”和“创”，需要对原内容进行加工，使其成为“新的”内容。
