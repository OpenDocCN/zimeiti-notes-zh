# 新媒体运营：P30：小红书笔记引流：平台流量分发机制 📊

![](img/1be552a45ef5426139557cfd6521c24e_1.png)

在本节课中，我们将要学习小红书平台的核心流量分发机制。了解这套算法，是提升笔记曝光、获得更多推荐流量的关键。我们将拆解其评分系统、分发流程以及影响排名的核心因素。

上一节我们介绍了小红书运营的基础框架，本节中我们来看看平台是如何决定将你的笔记展示给哪些用户的。

## 概述：流量分发的核心逻辑

小红书的流量分发并非随机，而是基于一套名为 **S1CCES** 的综合评分系统。这套系统通过量化用户的交互行为，对笔记进行评分和排名，从而决定其展示的优先级和流量规模。

## 核心评分系统：S1CCES

平台通过计算用户对笔记的交互行为来评分，各项行为的权重不同：

*   **点赞**：**1分**
*   **收藏**：**1分**
*   **转发**：**4分**
*   **评论**：**10分**
*   **关注**：**8分**

如果一个用户完成以上所有互动，单用户可为该笔记贡献 **18分**。系统根据笔记获得的总分，在同批次展示的笔记中进行排名。

## 流量分发流程详解

### 1. 初始曝光与标签匹配

笔记发布后，系统会将其内容（图片、文案）拆解为多个标签。随后，笔记会被推送给近期对相关标签内容感兴趣的用户。初始阶段，平台会给予笔记一个基础曝光量（例如1000次）。

### 2. 分波次推流机制

流量分发是分阶段进行的，并非一次性完成。例如，基础曝光量可能分4-5波进行推送（如400、300、200、100次）。能否进入下一波更大规模的推流，取决于前一波流量的数据表现。

以下是影响进入下一波推流的关键数据指标：

*   **点击率**：用户在看到封面和标题后，点击进入笔记页面的比率。这是所有后续互动的前提。
*   **互动率**：用户点赞、收藏、转发、评论、关注等行为的整体比率。
*   **完播率**：（针对视频笔记）用户完整观看视频的比例。
*   **互动值**：创作者与用户在评论区的互动频率与质量。

### 3. 排名与展示逻辑

小红书信息流每页展示20篇笔记，用户下滑时进行翻页。笔记的排名（第1-20名）由 **S1CCES综合评分** 决定。

*   排名越靠前，获得的曝光机会越多。
*   如果在当前批次（如20篇）中排名前列（例如前5名），笔记将有资格进入更大量级的流量池进行下一轮竞争。

## 核心要点与总结

本节课中我们一起学习了小红书流量分发的核心机制，可以总结为以下几点：

1.  **算法核心**：平台的核心算法始终围绕 **笔记带来的粉丝交互行为** 进行评估。内容质量最终通过用户的主动互动来体现。
2.  **流程关键**：**点击率是“入场券”**。没有点击，后续的评分无从谈起。优质封面和标题至关重要。
3.  **公平性与机会**：平台有基础流量保障，即使是新账号或低粉丝达人的优质内容，只要互动数据好（点击率、互动率高），也能通过阶梯式算法获得更多推荐。
4.  **持续优化方向**：要提升笔记排名，必须持续优化内容，以提升 **点击率、互动率、完播率**，从而获得更高的 **S1CCES综合评分**。
