# 视频号运营：P17：7.如何给视频抽帧补帧 🎬

在本节课中，我们将学习视频剪辑中的一个重要技巧——抽帧与补帧。这个操作可以帮助我们微调视频时长，同时避免触发平台的重复检测机制，是制作原创感内容的实用方法。

上一节我们介绍了视频的“掐头去尾”，本节中我们来看看如何通过抽帧和补帧进一步处理视频素材。

## 核心概念与准备工作

首先，我们需要理解帧的概念。视频是由一系列连续的静态图片（即“帧”）快速播放形成的。在大多数情况下，视频的播放速度为**每秒30帧（30 FPS）**。这意味着，一秒钟的视频由30张图片组成。

**公式：视频时长（秒） × 帧率（FPS） = 总帧数**

在进行操作前，请将视频轨道的时间线**拉伸到最长的刻度**，以便我们能精确地对每一帧进行操作。

## 如何进行抽帧

抽帧，指的是从视频中删除某一帧或几帧的画面。

以下是抽帧的具体步骤：
1.  在时间线上，找到你想要抽帧的位置。例如，在视频的第1秒，帧数会显示为 `00:00:00` 到 `00:00:01`，中间包含30个帧（F）。
2.  选中其中一帧，比如第20帧，在此处进行**分割**。
3.  紧接着，在第21帧的位置再次进行**分割**。这样，你就选中了单独的一帧。
4.  最后，**删除**这被单独分割出来的一帧片段。

对于一个十几秒的视频，通常**抽3帧**即可，这样对视频整体的流畅度影响微乎其微。

## 如何进行补帧

补帧，是指在视频中复制并插入已有的某一帧，以填补因抽帧或其他原因减少的帧数，保持视频总时长不变。

以下是补帧的具体步骤：
1.  在完成抽帧后，视频轨道上会出现一个微小的空隙。
2.  选中与空隙相邻的前一帧或后一帧的片段。
3.  对该片段执行**复制**操作，并将复制出的片段粘贴到空隙处。

你可以用同样的方法，在视频的其他位置也进行补帧操作。通过“抽3帧补3帧”的方式，视频的总时长和主体内容保持不变，但帧序列已经发生了细微变化。

## 操作要点与总结

![](img/b84c6a44cd65a5bc6efc339f8b58838d_2.png)

本节课中我们一起学习了抽帧与补帧的技巧。关键点在于：
*   **目的**：微调视频，增加原创性，规避重复审核。
*   **核心操作**：在最大化的时间线上，对以“帧”为单位的片段进行**分割、删除、复制**。
*   **建议频率**：一个短视频进行**抽3补3**即可，过度操作可能影响观感。
*   **最终效果**：视频整体流畅度不受影响，但帧序列已与原视频不同。

请记住，在完成视频的“掐头去尾”后，将抽帧补帧作为一项标准流程来操作，能使你的作品更具独特性。
