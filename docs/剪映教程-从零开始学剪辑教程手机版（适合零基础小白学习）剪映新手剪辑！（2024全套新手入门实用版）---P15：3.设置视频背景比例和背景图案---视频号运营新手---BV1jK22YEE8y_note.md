# 剪映教程：P15：3. 设置视频背景比例和背景图案 🎬

在本节课中，我们将学习如何设置视频的背景比例和背景图案。这是视频剪辑的基础操作，能让你的视频适应不同平台（如抖音、快手）的播放要求，并通过背景美化提升视频的整体观感。

![](img/73837e27e71e9253008085bc57e85f88_1.png)

## 了解视频比例 📐

在开始操作前，我们先了解一下传统视频平台与新兴短视频平台在视频比例上的区别。

在抖音和快手等平台出现之前，我们习惯于观看横向的宽屏视频。而在这些短视频平台流行之后，竖向（或称竖屏）视频开始成为主流。

![](img/73837e27e71e9253008085bc57e85f88_3.png)

## 设置视频比例

上一节我们了解了比例的背景知识，本节中我们来看看如何在剪映中具体设置视频比例。

剪映的视频比例由你导入的第一个素材决定。例如，如果你首先导入了一个16:9的横向视频，那么整个项目的初始比例就会被设定为16:9。

以下是设置比例的步骤：
1.  点击底部工具栏中的 **`比例`** 按钮。
2.  剪映提供了多种常用比例供你选择，主要包括：**`9:16`**（竖屏）、**`3:4`**、**`1:1`**（正方形）、**`4:3`** 和 **`16:9`**（横屏）。

## 设置视频背景

设置好视频比例后，我们常常需要为视频添加背景。特别是当原始素材尺寸与项目比例不匹配时，背景就显得尤为重要。

在设置背景前，通常需要先将主视频素材缩小，为背景留出空间。操作方法是：选中视频片段，在预览窗口中用**双指捏合**的手势将其缩小。

视频缩小后，点击返回，然后点击 **`背景`** 按钮。剪映提供了三种背景类型：

*   **画布颜色**：允许你选择一种纯色作为背景。软件提供了丰富的颜色选项。
*   **画布样式**：在这里，你可以选择手机相册中的图片，或者使用剪映内置提供的多种精美图案作为背景。
*   **画布模糊**：此功能可以将你的视频画面进行模糊处理，然后作为背景。软件通常提供 **`4`** 种不同程度的模糊选项供你选择。

你可以点击 **`应用到全部`** 按钮，将选中的背景效果一键应用到时间轴上的所有视频片段。

## 案例演示：制作一个带边框的可爱视频

现在我们已经学会了设置比例和背景，接下来通过一个简单的案例，将这些知识结合起来，制作一个带有白色边框的可爱风格视频。

由于审美因人而异，以下步骤旨在提供一个清晰的制作思路。

1.  **调整视频边框**：在预览窗口中，用**双指分开**的手势，将视频画面缩小，从而在四周留出等宽的空白边框区域。
2.  **设置边框颜色**：进入 **`背景`** -> **`画布颜色`**，选择纯白色，这样留出的边框就会变成白色。
3.  **精细调整**：如果发现四边白框宽度不一致，可以通过**裁剪**原始视频素材，或者微调其位置，来确保边框宽度均匀美观。
4.  **添加文本装饰**：点击 **`文本`** -> **`新建文本`**，输入文字（例如“我的第一个视频”）。然后可以为其设置一个有趣的样式，比如**气泡**样式，并添加一个**收拢**动画，让文字出现得更生动。
5.  **调整时长与预览**：最后，统一调整视频和文本片段的长度，然后点击播放按钮进行预览，检查最终效果。
