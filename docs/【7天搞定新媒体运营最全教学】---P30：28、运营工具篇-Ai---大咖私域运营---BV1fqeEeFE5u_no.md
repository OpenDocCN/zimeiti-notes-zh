## 工具使用方式

该工具主要提供两种生成方式：**AI智能生成**和**导入本地大纲**。通常，导入本地大纲的方式能获得更可控、更符合预期的结果。

以下是两种方式的具体操作步骤：

1.  **导入本地大纲**
    *   首先，使用ChatGPT等AI工具撰写一个详细的主题大纲。例如，可以输入指令：“帮我完善‘装修行业在小红书平台的运营推广计划’的每一个小节点的具体内容信息，保持好结构和格式排版。”
    *   将生成的大纲内容复制，进入PPT生成工具，选择“导入本地大纲”功能。
    *   将内容粘贴或上传，工具会快速解析大纲结构。
    *   随后，从丰富的模板库中挑选一个适合行业风格（如装修行业）的模板。
    *   点击“生成PPT”，工具便会基于大纲和模板自动生成幻灯片。

2.  **AI智能生成**
    *   在工具中选择“AI智能生成”功能。
    *   直接在输入框中输入PPT主题，例如：“装修行业在小红书平台的运营推广计划”。
    *   工具会自动构思并生成大纲，然后同样进入模板选择界面。
    *   选择模板后，工具会生成完整的PPT文案和页面。

## 生成后的编辑与优化

无论采用哪种方式生成，初步得到的PPT都可能需要进一步调整和优化。

以下是编辑与优化的关键操作：

*   **内容填充与修正**：如果发现某些页面的内容缺失（例如，大纲中“内容趋势”下的具体分析未显示），可以进入“大纲编辑”界面，在对应的章节下补充详细内容，然后重新生成或更新页面。
*   **页面布局调整**：对单张幻灯片的排版不满意，可以单击该页面，工具会提供多种备选布局样式供一键更换。
    ```plaintext
    操作路径：单击幻灯片 -> 选择“更换布局” -> 挑选新样式
    ```
*   **整体模板更换**：即使PPT已生成，也可以随时在“模板替换”功能中为整个演示文稿更换另一套模板，响应速度很快。
*   **元素自定义**：右侧编辑栏提供文字、形状、背景等设置，支持在线更换图片，可以满足基础的个性化编辑需求。
