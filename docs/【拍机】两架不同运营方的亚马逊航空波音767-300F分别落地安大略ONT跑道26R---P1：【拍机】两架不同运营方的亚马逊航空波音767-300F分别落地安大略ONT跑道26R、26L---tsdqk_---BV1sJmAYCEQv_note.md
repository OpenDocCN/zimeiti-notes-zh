# 航空摄影入门：第1章：如何拍摄飞机降落

在本节课中，我们将学习航空摄影的基础知识，特别是如何拍摄飞机降落的场景。我们将以两架不同运营商的亚马逊航空波音767-300F货机降落在安大略国际机场（ONT）为例，讲解拍摄前的准备、拍摄时的技巧以及后期整理的核心步骤。

上一节我们介绍了课程目标，本节中我们来看看拍摄飞机降落需要做哪些具体准备。

## 第一步：拍摄前的准备工作

成功的拍摄始于充分的准备。以下是拍摄飞机降落前需要完成的几个关键步骤。

1.  **了解机场与跑道信息**：首先，需要确定目标机场的布局和常用跑道。例如，安大略国际机场（ONT）的26R和26L跑道是常见的降落跑道。了解跑道方向有助于判断飞机的进场路线和光线角度。
2.  **查询航班时刻表**：使用航班追踪软件或网站，查询目标机型（如波音767-300F）的预计到达时间。这能帮助你提前到达最佳拍摄位置，不错过关键瞬间。
3.  **选择合适器材**：航空摄影通常需要长焦镜头，例如**焦距为70-200mm或100-400mm的镜头**，以便捕捉远处降落的飞机。同时，稳固的三脚架和充足的存储卡、电池也必不可少。
4.  **勘察拍摄地点**：提前抵达机场外围，寻找安全且合法的公共区域作为拍摄点。确保该位置视野开阔，无遮挡，并且背景简洁。

准备工作完成后，我们就可以进入实战拍摄环节了。

## 第二步：拍摄时的核心技巧

![](img/a87657e1952943082c0513418a516aae_0.png)

当飞机进入视野时，掌握正确的拍摄技巧至关重要。以下是拍摄过程中的几个要点。

1.  **设置相机参数**：为了定格飞机高速运动的瞬间，需要使用较快的快门速度。建议设置：**快门速度 ≥ 1/1000秒**，光圈在f/5.6至f/8之间以保证景深，ISO根据光线条件自动或手动调整。
2.  **使用连续自动对焦**：将相机的对焦模式设置为连续自动对焦（如AI-SERVO或AF-C），并启用高速连拍功能，以确保在一系列照片中能捕捉到最清晰的一帧。
3.  **构图与时机**：在飞机对准跑道下滑的阶段进行跟踪拍摄。构图时，可以在画面中为飞机预留前进方向的空间，使画面更具动感。同时，注意捕捉起落架放下的状态和轮胎触地瞬间的烟尘。
4.  **记录运营方信息**：如果拍摄目标是展示不同运营商的飞机（如视频中的两架亚马逊航空767，可能分别由阿特拉斯航空或ATSG等公司运营），需要仔细观察并记录飞机的喷涂、注册号等信息。

拍摄到满意的照片后，工作并未结束，我们还需要对素材进行整理和归档。

![](img/a87657e1952943082c0513418a516aae_1.png)

## 第三步：后期整理与归档

整理工作能让你快速找到所需素材，并为分享或分析做好准备。以下是整理步骤。

1.  **导入与筛选**：将照片导入电脑，快速浏览并筛选出对焦清晰、构图理想、瞬间精彩的图片。
2.  **基础后期处理**：使用软件进行简单的调整，如修正曝光、调整白平衡、进行适度裁剪以优化构图。处理原则是还原现场观感，避免过度修饰。
3.  **规范命名与添加元数据**：为文件命名，例如采用“日期_机场_机型_注册号”的格式（**20231030_ONT_B763_N1234A.jpg**）。同时在照片元数据中填入关键词，如“波音767”、“亚马逊航空”、“安大略机场”等，便于日后搜索。
4.  **分类存储**：建立清晰的文件夹结构进行存储。可以按日期、机场、航空公司或机型等维度进行分类，确保资料井井有条。

本节课中我们一起学习了航空摄影中拍摄飞机降落的完整流程。我们从拍摄前的信息查询与器材准备开始，接着探讨了实战中的相机设置与构图技巧，最后完成了照片的后期整理与归档。掌握这些基础步骤，你就能系统地记录下每一次精彩的飞机降落瞬间。
