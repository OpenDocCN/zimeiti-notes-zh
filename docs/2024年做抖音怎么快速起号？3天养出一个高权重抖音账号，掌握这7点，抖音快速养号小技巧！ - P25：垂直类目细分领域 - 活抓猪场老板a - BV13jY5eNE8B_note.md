# 抖音运营教程：P25：垂直类目细分领域

在本节课中，我们将深入探讨抖音账号运营中“垂直度”的核心概念。我们将学习如何将宽泛的领域逐级细分，从而打造出更具吸引力和高权重的账号定位。

上一节我们介绍了内容标签的重要性，本节中我们来看看如何将“垂直”这个概念落到实处，进行更精细的领域划分。

我们常说做账号要垂直，但垂直本身也有不同的层级。仅仅选择一个大的方向（如美食、游戏）只是第一步，这属于一级垂直领域。为了实现更精准的流量获取和粉丝积累，我们需要进一步细分。

以下是垂直类目的分级体系，我们以“游戏”领域为例进行说明：

*   **一级垂直领域**：这是最宽泛的分类。例如，你决定做“游戏”内容。
    *   `领域 = “游戏”`
*   **二级垂直领域**：这是一级领域下的具体分支。在“游戏”下，可以细分为手游、电竞、网游等。
    *   `领域 = “游戏” -> “手游”`
*   **三级垂直领域**：这是二级领域的进一步聚焦。例如，在“手游”中，你可以专门做“王者荣耀”或“和平精英”的内容。
    *   `领域 = “游戏” -> “手游” -> “王者荣耀”`
*   **四级垂直领域（可继续细分）**：这是更极致的细分。例如，在“王者荣耀”中，你可以只制作关于“安琪拉”这个英雄的教学、对战视频。或者在“和平精英”中，只发布“雨林地图”的攻略或只创作“游戏剧情”类短片。
    *   `领域 = “游戏” -> “手游” -> “王者荣耀” -> “英雄：安琪拉”`
    *   `领域 = “游戏” -> “手游” -> “和平精英” -> “地图：雨林”`

大领域和小领域都可以作为你的选择，这取决于你自己的兴趣和运营策略。理解这个细分逻辑，能帮助你找到竞争更小、粉丝粘性更高的切入点。
