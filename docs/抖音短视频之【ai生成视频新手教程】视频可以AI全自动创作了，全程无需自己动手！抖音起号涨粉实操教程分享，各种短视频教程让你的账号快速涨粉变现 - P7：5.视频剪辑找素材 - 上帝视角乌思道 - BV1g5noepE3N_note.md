# 抖音短视频AI生成教程：P7：5.视频剪辑找素材 🎬

在本节课中，我们将学习如何为视频剪辑寻找合适的素材。这是创作视频的第一步，也是至关重要的一环。我们将介绍多种寻找素材的渠道，并讲解如何正确、高效地获取和使用这些素材，避免常见的违规问题。

上一节我们介绍了视频剪辑的多种类型，本节中我们来看看如何为这些剪辑寻找素材。

## 寻找素材的平台

寻找素材，我们主要可以从以下三类平台入手。

以下是三类主要的素材来源平台：

1.  **所有视频平台**：众多自媒体平台是寻找素材的重要来源。例如，下图所示的30多个自媒体平台都提供了丰富的视频素材。
    ![](img/07226dd691206b2082e167248e09d777_1.png)

2.  **影视网站**：专门的影视网站是获取影视类素材的宝库。例如，豆瓣电影这类网站。我自己整理了近50个无版权要求的影视网站，其中的素材均可自由使用。此外，针对非影视类需求（如动漫、综艺、搞笑、旅游、美食等），我也整理了60多个涵盖各类主题的素材网站。

3.  **影视推广平台**：这类平台（如抖音、快手上的某些合作内容）的素材有其特殊性。在抖音或快手直接发布视频通常只能用于涨粉，难以直接变现。但使用这些影视推广平台提供的官方素材进行创作，则可以依靠播放量获得平台奖金。这一点我们稍后会重点讲解。

## 如何正确搜索素材

![](img/07226dd691206b2082e167248e09d777_5.png)

掌握了素材来源后，我们来看看具体的搜索方法。以在“西瓜视频”网站搜索为例。

![](img/07226dd691206b2082e167248e09d777_7.png)

以下是搜索素材的具体步骤：

1.  打开百度，搜索并进入目标自媒体平台（如西瓜视频）。
    ![](img/07226dd691206b2082e167248e09d777_5.png)

![](img/07226dd691206b2082e167248e09d777_9.png)

2.  在平台内使用**关键词**进行搜索。例如，想找综艺或网红直播片段，可直接搜索“大杨哥”、“小杨哥”等名字。
    ![](img/07226dd691206b2082e167248e09d777_9.png)
    ![](img/07226dd691206b2082e167248e09d777_11.png)

3.  同理，搜索明星素材，如“沈腾”，也能找到大量相关内容。
    ![](img/07226dd691206b2082e167248e09d777_13.png)

## 关键注意事项：避免“搬运”

找到素材后，使用方法至关重要。这里必须强调一个核心原则：**不要直接下载并发布整个视频**。

这种做法就是所谓的“搬运”。目前，超过95%的搬运内容都无法通过平台审核。正确的做法是：

![](img/07226dd691206b2082e167248e09d777_13.png)

**从大量视频中，提取符合自己视频主题的片段，每个片段的时长控制在10秒左右。**

例如，我们要制作一个主题为“沈腾搞笑名场面”的视频。
![](img/07226dd691206b2082e167248e09d777_15.png)

那么，我们就需要从搜索到的关于沈腾的各种视频里，找出那些搞笑的、时长约10秒的片段。使用这些片段进行二次创作，才能有效避免违规，顺利通过审核。
