# 电商维权教程：P1：应对“仅退款”争议的申诉策略 🛡️

在本节课中，我们将学习一种应对电商平台“仅退款”争议的高效申诉方法。该方法通过特定的沟通策略，旨在帮助卖家有效追回货款。

上个月我曾分享一个视频，讲解如何用一句话追回“仅退款”的货款，但该内容被平台投诉下架。本次我将重新讲解这一方法。方法其实很简单，如果你遇到了“仅退款”的情况，无需担心。

你只需要使用特定的几句话，就能以较高的成功率追回货款。遇到“仅退款”时，请直接前往订单后台选择申诉。申诉过程中，其他材料都不重要，甚至图片可以随意上传，关键在于你撰写的申诉文案。

撰写文案没有任何技巧，核心就是严厉地斥责对方。你斥责得越严厉，用语越激烈，情绪越激动，平台为你退款的成功率反而越高。在之前附有图文教程的视频案例中，申诉了7个订单，仅1个失败。

那个失败的订单没有斥责对方，而是心平气和地讲道理，结果没有成功。其余6个订单均采用了严厉斥责的方式，基本都申诉成功了。因此，如果你遭遇“仅退款”争议，不要害怕，直接发起申诉。

申诉时，核心就是严厉斥责对方。只要你这样做了，货款追回的概率就非常高。
