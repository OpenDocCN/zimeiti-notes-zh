# 微信小店专题：P1：全面开放注册与核心功能详解

在本节课中，我们将学习微信小店（由视频号小店升级而来）全面开放注册后的核心变化与操作要点。课程将涵盖正确的注册入口、界面功能更新、优选联盟新规以及教育类目入驻的注意事项。

## 正确的注册入口

上一节我们介绍了课程主题，本节中我们来看看如何正确注册微信小店。

![](img/6b96ced2af5464bbb72041389becec2d_1.png)

微信小店已于8月25日全面开放注册。许多用户可能会在百度搜索“微信小店”，但搜索结果中的网址均不正确。

以下是正确的注册网址，您可以直接保存：
```
https://channels.weixin.qq.com/shop
```

访问该网址后，界面与之前的视频号小店完全一致。您只需使用微信扫码，即可快速注册。在资料齐全的情况下，审核流程迅速，通常**10至15分钟**即可完成注册。



## 核心功能界面解析

![](img/6b96ced2af5464bbb72041389becec2d_3.png)

![](img/6b96ced2af5464bbb72041389becec2d_4.png)

了解了注册方式后，我们进入后台，看看界面有哪些核心变化。

![](img/6b96ced2af5464bbb72041389becec2d_5.png)

![](img/6b96ced2af5464bbb72041389becec2d_6.png)

整体界面布局与视频号小店几乎完全相同。但其中一项关键更新是客服系统。



![](img/6b96ced2af5464bbb72041389becec2d_7.png)

![](img/6b96ced2af5464bbb72041389becec2d_1.png)

以下是客服功能的详细说明：
*   **独立客服系统**：新版微信小店采用了独立的客服系统，不再强制绑定企业微信。这与旧版的“微信小商店”客服模式一致。
*   **升级建议**：如果您已在运营视频号小店，不建议立即升级现有店铺。更好的方式是**注册一个新店铺**进行测试和运营，因为新旧系统的客服体系已分离。



## 优选联盟新规详解

![](img/6b96ced2af5464bbb72041389becec2d_9.png)

接下来，我们探讨对于分销至关重要的“优选联盟”功能发生的变化。

![](img/6b96ced2af5464bbb72041389becec2d_11.png)

优选联盟的入口和运费险设置与之前类似。新店只需购买联盟运费险即可。最大的区别在于准入规则。

以下是优选联盟的主要更新点：
*   **主体资格放宽**：**个体工商户**现在也可以申请加入优选联盟。
*   **观察期规则变更**：旧规则要求完成**50单**，新规则变更为**当日销售总额达到5000元**。对于知识付费等客单价较高的品类（例如299元以上），需要计算达成此目标所需的订单量。
*   **快速通过观察期**：您也可以选择完成**30个有效结算订单**来通过观察期。有效订单的结算周期约为2天。如果用户能及时确认收货，整个过程可压缩至半个月左右。



## 联盟清退与品控指标

![](img/6b96ced2af5464bbb72041389becec2d_20.png)

成功加入联盟后，维持资格同样重要。本节我们来看看可能导致清退的关键指标。

![](img/6b96ced2af5464bbb72041389becec2d_21.png)

清退规则中，**品退率**是需要重点关注的数据。



![](img/6b96ced2af5464bbb72041389becec2d_18.png)

以下是关于清退风险的核心要点：
*   **关键指标**：在后台可以查看“30天品退单数”和“30天品退率”。
*   **风险阈值**：务必确保**品退单数维持在5单以内**。即使已加入联盟，一旦品退单数超标，也可能被清退。因此，交付与服务团队需与用户做好有效沟通，减少退款纠纷。

## 保证金与教育类目资质

![](img/6b96ced2af5464bbb72041389becec2d_25.png)

最后，我们针对教育知识付费赛道的老师，讲解保证金和特殊资质要求。

![](img/6b96ced2af5464bbb72041389becec2d_27.png)

保证金金额根据主体类型有所不同，通常个体工商户的保证金会少一些。对于教育类目，有额外的资质要求。



![](img/6b96ced2af5464bbb72041389becec2d_28.png)

![](img/6b96ced2af5464bbb72041389becec2d_29.png)

![](img/6b96ced2af5464bbb72041389becec2d_27.png)

![](img/6b96ced2af5464bbb72041389becec2d_28.png)

![](img/6b96ced2af5464bbb72041389becec2d_29.png)

![](img/6b96ced2af5464bbb72041389becec2d_31.png)

以下是教育类目入驻的关键步骤：
1.  **选择类目**：在发布商品时，选择“教育培训”类目。
2.  **准备资质**：主要需要**ICP备案**（互联网信息服务备案）。从经济效益和办理效率考虑，ICP备案比办理许可证更高效。
3.  **办理周期**：ICP备案的办理周期至少需要**15天**，请提前准备。
4.  **关联证明**：如果同一个人拥有多家公司，只需办理一个主体的ICP备案，其他公司可使用“授权模式”并提供关联关系证明模板。



![](img/6b96ced2af5464bbb72041389becec2d_33.png)

![](img/6b96ced2af5464bbb72041389becec2d_31.png)

![](img/6b96ced2af5464bbb72041389becec2d_35.png)
