# 新媒体运营从入门到精通：P29：11：小红书平台用户标签及流量入口 📊

在本节课中，我们将要学习小红书平台如何为用户打上标签，以及笔记是如何通过不同的流量入口被分发给用户的。理解这些底层逻辑，有助于我们更好地规划内容，让笔记精准触达目标受众。

上一节我们介绍了小红书系统如何判定内容标签，本节中我们来看看平台如何判定用户标签。理解这个过程，可以帮助我们分析和了解自己的标签，并反推笔记是如何被系统推荐给用户的，从而找到让笔记被精准用户看到的方法。

## 用户标签判定逻辑

小红书系统为用户打标签，主要依据用户的**浏览行为**。用户刚注册时选择的兴趣范围，参考价值不大，它仅作为平台对用户的内容方向引导和初步筛选，无法进入系统最深层的用户标签池。

![](img/193db5df8616486fd22a3a0c6737fc8c_3.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_5.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_7.png)

系统判定用户的兴趣分类，完全基于其浏览行为。以下是系统分析用户行为的主要维度：

![](img/193db5df8616486fd22a3a0c6737fc8c_9.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_11.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_13.png)

### 1. 点击率
**点击率**是指系统在发现页向用户推送多个内容后，用户点击各类内容的概率。例如，系统可能同时推送明星、美妆、家居、母婴等内容。用户点击哪一类内容更多，系统就能通过算法判断其兴趣偏好。

![](img/193db5df8616486fd22a3a0c6737fc8c_15.png)

**公式示例**：`点击率 = (点击某类内容的次数) / (该类内容被展示的总次数)`

![](img/193db5df8616486fd22a3a0c6737fc8c_17.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_19.png)

如果用户点击了10条内容，其中美妆类3次，家居类2次，母婴类0次，系统会判定该用户对母婴内容不感兴趣，可能不是宝妈，而对美妆内容感兴趣，从而持续推送更多美妆内容进行试探。

### 2. 停留时长
**停留时长**指用户点开一篇笔记后观看的时长。即使一个用户经常点击美妆内容，但如果每次都快速划走，系统会认为这些具体笔记不符合其喜好，从而尝试推送美妆领域内其他细分内容（如上妆、护肤等）。

![](img/193db5df8616486fd22a3a0c6737fc8c_21.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_23.png)

反之，如果用户偶然点击了平时很少看的内容（如家居），却停留了很长时间，仔细浏览图片和评论，系统可能会判断该用户有潜在需求（如装修或购买特定家居产品），从而调整标签，推送相关笔记或商品。

![](img/193db5df8616486fd22a3a0c6737fc8c_25.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_27.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_29.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_31.png)

### 3. 页面浏览量（PV）
**PV** 指用户浏览页面的深度。例如，用户点开一篇装修笔记后，不仅看完了该笔记，还进入了博主主页，并连续观看了该博主的第二、第三篇笔记。这个浏览的页面数就是PV。

![](img/193db5df8616486fd22a3a0c6737fc8c_33.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_35.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_37.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_39.png)

如果账号主页内容精致、垂直，就能有效提升用户的PV。这个数值对账号健康度和系统评分有非常大的帮助，尤其是在粉丝基数较小时，能有效提升账号质量。

![](img/193db5df8616486fd22a3a0c6737fc8c_41.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_43.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_45.png)

### 4. 互动行为
互动行为包括**点赞、收藏、评论、关注**。这些数据是衡量笔记质量的关键指标。

![](img/193db5df8616486fd22a3a0c6737fc8c_47.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_49.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_51.png)

*   **点赞/收藏**：数值高能帮助笔记进入更大的推荐流量池。
*   **评论**：增加内容互动度和热度。
*   **关注**：是用户对账号价值的最高认可，能永久提升账号权重和粉丝基数。

![](img/193db5df8616486fd22a3a0c6737fc8c_53.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_55.png)

提升这些互动数据，是运营小红书账号的核心目标之一，它们直接影响笔记的推荐范围和账号的商业变现能力。

![](img/193db5df8616486fd22a3a0c6737fc8c_57.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_59.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_61.png)

## 平台、用户与博主的关系

![](img/193db5df8616486fd22a3a0c6737fc8c_63.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_65.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_67.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_69.png)

系统通过上述行为为用户打上标签（例如美妆、母婴、家居），并形成一个标签云图。标签的大小代表用户对该领域内容的偏好强度。系统会持续给用户推送其强标签相关的内容，以提升用户的获得感与停留时长，这是平台的核心目标。

![](img/193db5df8616486fd22a3a0c6737fc8c_71.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_73.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_75.png)

这构成了**平台、用户、博主**三方之间的价值循环：
*   **博主** 创作内容，为用户提供价值，同时也为平台贡献内容。
*   **平台** 通过算法连接用户与内容，为用户提供价值（推荐喜欢的内容），也为博主提供曝光机会。
*   **用户** 消费内容，其行为数据反馈给平台，优化整个推荐系统。

## 小红书流量分发入口

![](img/193db5df8616486fd22a3a0c6737fc8c_83.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_85.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_87.png)

了解用户标签后，我们来看笔记是通过哪些入口分发给用户的。小红书主要有四个流量分发入口：

![](img/193db5df8616486fd22a3a0c6737fc8c_91.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_93.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_95.png)

以下是各流量入口的详细说明：

![](img/193db5df8616486fd22a3a0c6737fc8c_97.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_99.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_101.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_102.png)

1.  **发现页**：这是最主要的流量来源，包含三部分内容：
    *   **关注达人**：用户已关注的博主发布的新内容。
    *   **兴趣内容匹配**：系统根据用户标签（点击率、停留时长等）推荐的、其可能喜欢的内容。
    *   **发散性内容匹配**：平台为了内容多样化，也会试探性推荐一些用户可能不常看但潜在感兴趣的内容。

![](img/193db5df8616486fd22a3a0c6737fc8c_104.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_106.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_108.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_110.png)

2.  **关注页**：这里集中展示用户所有关注博主的更新内容。当用户关注数成百上千时，可能会在此页面浏览。

![](img/193db5df8616486fd22a3a0c6737fc8c_112.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_114.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_116.png)

3.  **附近页**：基于手机定位，推送同城或附近的内容。此入口强地域特征，适合**美食、本地旅游、线下店推广**（如美容院、餐厅）等内容。做线下实体店推广时，应特别注意这个入口的运营和广告投放。

![](img/193db5df8616486fd22a3a0c6737fc8c_118.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_120.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_122.png)

4.  **购物页**：这是小红书向电商化发展的重要入口，直接对接商品和直播带货。目前部分用户尚未形成在此直接购物的强习惯，更多仍是通过搜索笔记来“种草”。此入口直接匹配用户的购物需求。

![](img/193db5df8616486fd22a3a0c6737fc8c_126.png)

![](img/193db5df8616486fd22a3a0c6737fc8c_128.png)
