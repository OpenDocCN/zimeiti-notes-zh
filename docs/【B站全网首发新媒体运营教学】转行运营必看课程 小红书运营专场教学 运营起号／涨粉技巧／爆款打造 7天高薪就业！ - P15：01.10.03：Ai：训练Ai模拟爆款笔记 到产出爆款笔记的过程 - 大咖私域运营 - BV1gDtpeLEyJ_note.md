# 小红书运营：P15：01.10.03：AI训练与爆款笔记产出 🚀

## 概述
在本节课中，我们将学习如何训练AI（以ChatGPT为例）分析爆款笔记，并基于分析结果生成具备爆款潜质的新笔记。我们将从提供资料、分析爆款因素，到最终生成文案，完整走一遍流程。

上一节我们介绍了通过提供资料让AI理解平台风格的方法。虽然示例结果可能不完美，但方法是正确的。本节中，我们来看看如何通过分析具体的爆款内容，让AI学习并产出高质量的笔记。

## 核心分析要点
我们需要引导AI分析爆款内容时，关注以下几个核心点：

1.  **爆款因素**：分析内容为何能成为爆款，例如是否引发了情感共鸣、提供了实用价值等。
2.  **内容结构**：分析文案的起承转合。
    *   **开篇**：用什么方式吸引用户继续阅读。
    *   **中间**：提供了哪些内容来戳中用户痛点，提升停留时长。
    *   **结尾**：如何设置“钩子”，引导用户互动（如私信、关注）。
3.  **关键词策略**：分析笔记中是否植入了特定关键词，以及关键词的出现频率和分布方式。

## 实战步骤：从分析到产出

![](img/d36d5dab1a5d8813fe8d4d10a845dc93_9.png)

![](img/d36d5dab1a5d8813fe8d4d10a845dc93_11.png)

### 第一步：提供爆款文案并下达分析指令
首先，我们需要找到目标领域的爆款笔记文案。本节课以“考研”领域为例。

![](img/d36d5dab1a5d8813fe8d4d10a845dc93_13.png)

![](img/d36d5dab1a5d8813fe8d4d10a845dc93_15.png)

找到文案后，向AI下达清晰的分析指令。指令模板如下：

![](img/d36d5dab1a5d8813fe8d4d10a845dc93_17.png)

```
我给你提供一些关于小红书平台上的考研相关爆款笔记文案，你需要分析这些文案的爆款因素。
分析要求：
1.  分析文案的整体结构：开篇用什么吸引用户，中间用什么内容戳中用户痛点，结尾用什么方式让用户形成互动或私信关注等行为。
2.  分析我给你的每一篇笔记的关键词：根据整篇文案分析，该笔记是否有结合中文分词法设计关键词。没有的话，请告诉我“没有关键词”；有的话，告诉我是哪个关键词，以及该关键词的出现频率。
```

![](img/d36d5dab1a5d8813fe8d4d10a845dc93_19.png)

将找到的爆款文案粘贴给AI，等待其分析结果。AI可能会输出类似“情感共鸣”、“逆袭故事”、“提供明确备考计划”等爆款因素。

![](img/d36d5dab1a5d8813fe8d4d10a845dc93_21.png)

### 第二步：整合分析结果，总结爆款因素
在AI分析了多篇笔记后，我们需要让它对分析结果进行整合与去重，总结出该领域爆款笔记的共性。

可以给AI如下指令：
```
请将你前面分析过的关于考研的几篇笔记的爆款因素总结一下，整合成一个清晰的列表。
```

![](img/d36d5dab1a5d8813fe8d4d10a845dc93_23.png)

![](img/d36d5dab1a5d8813fe8d4d10a845dc93_25.png)

AI会生成一个汇总列表，例如：
*   情感共鸣（如焦虑、担忧）
*   提供逆袭故事或个人经历
*   给出明确的备考计划与时间节点
*   分享实用的技巧与资源推荐
*   正能量鼓励
*   使用表情符号增加亲和力

### 第三步：基于爆款因素，生成笔记大纲
有了爆款因素列表后，我们可以指令AI根据这些因素，规划一篇新笔记的框架。

指令示例：
```
我要写一篇关于考研的小红书笔记。请结合你前面分析的爆款笔记因素，帮我梳理一个具备爆款潜质的笔记大纲。每个大纲节点需要进一步细分出小节点。在我确认大纲没问题后，再生成完整文案。
```

AI可能会给出如下结构的大纲：
1.  **开篇：吸引注意**
    *   用引人入胜的话语或问题开场。
    *   简要介绍笔记主题。
2.  **核心内容：干货分享**
    *   专业选择分析。
    *   分科目（政治、英语、数学、专业课）备考攻略。
    *   时间规划与管理建议。
3.  **结尾：互动引导**
    *   鼓励读者，传递正能量。
    *   引导提问、私信或关注。

### 第四步：根据大纲，生成完整文案
确认大纲符合要求后，即可让AI根据大纲和爆款因素撰写完整文案。此时可以提出更细致的要求。

指令示例：
```
请按照上述大纲，写出具体的文案。要求如下：
1.  文案结构需严格遵循爆款笔记的因素和结构。
2.  需要包含一个简单的“逆袭小故事”来引入读者情绪。
3.  文案中需包含适量的emoji表情。
4.  文案中需尝试结合中文分词法的方式融入“考研”等相关关键词。
```

AI会根据指令生成一篇包含故事引入、干货分点、表情符号和互动引导的完整笔记文案。生成的文案可能已经具备不错的基础，但对于更高要求，可以在此基础上进行人工优化和调整。

---

## 进阶提示与工具
*   **指令的精确性**：与AI对话时，用词需精确。例如，“包含适量的emoji表情”比“包含丰富的emoji表情”更能控制输出效果。
*   **模型差异**：使用更强大的模型（如GPT-4）通常会获得更优质、更智能的输出结果。
*   **定制化助手**：可以利用ChatGPT的“GPTs”功能，创建专属的小红书文案助手。通过上传平台风格指南、专属表情包库、优秀文案范例等文件，让AI更深入地学习你的特定风格，从而生成更符合需求的文案，避免遗忘上下文的问题。

---

## 总结
本节课我们一起学习了利用AI辅助小红书运营的核心流程：
1.  **提供资料**：让AI分析现有爆款笔记。
2.  **总结规律**：提炼出可复用的爆款因素和内容结构。
3.  **生成大纲**：基于规律规划新笔记框架。
4.  **产出文案**：根据框架和细节要求生成完整内容。
