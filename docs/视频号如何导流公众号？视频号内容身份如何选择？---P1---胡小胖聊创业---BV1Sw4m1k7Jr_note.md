# 视频号运营指南：P1：视频号导流公众号与内容身份选择

在本节课中，我们将要学习如何将视频号的流量引导至公众号，并深入探讨视频号“内容身份”这一核心设置的选择策略。理解这两点对于规划你的账号矩阵和粉丝增长路径至关重要。

## 视频号导流至公众号的方法

上一节我们介绍了课程目标，本节中我们来看看如何具体操作，将视频号的关注者引导至你的公众号。整个过程非常简单，仅需两个步骤即可完成。

以下是具体的操作步骤：

![](img/7a5d27ca61c07876159530326f5600b4_1.png)

1.  **绑定视频号**：登录你的微信公众号后台，在设置中找到“视频号绑定”功能，完成与你的视频号的绑定操作。
2.  **设置内容身份**：在绑定设置中，将“视频号内容身份”选项设置为“公众号身份”。

![](img/7a5d27ca61c07876159530326f5600b4_2.png)

完成以上设置后，你的视频号个人主页将会显示公众号信息。当粉丝在视频号中点击“关注”按钮时，系统将直接引导他们关注绑定的公众号。通过这种方式，我们便成功实现了将视频号流量导入公众号的目的。

## 理解“内容身份”及其影响

![](img/7a5d27ca61c07876159530326f5600b4_4.png)

我们已经学会了如何导流，但直接选择“公众号身份”会带来一个明显的弊端：此后，粉丝接收你视频号新内容的途径将仅限于“订阅号消息”列表，而无法通过“视频号”入口直接接收。这是因为粉丝关注的主体是公众号，而非视频号本身。

那么，为什么会这样呢？这就需要我们来深入了解“视频号内容身份”这个核心概念。

在公众号后台的视频号绑定设置中，“视频号内容身份”是一个关键选项，它包含两个选择：
*   **公众号身份**
*   **视频号身份**

这个选项决定了你的视频号对外展示的身份以及粉丝关注行为的最终去向。其逻辑可以用以下代码块表示：

```plaintext
if (内容身份 == “公众号身份”) {
    展示身份 = 公众号;
    引导关注 = 公众号;
} else if (内容身份 == “视频号身份”) {
    展示身份 = 视频号;
    引导关注 = 视频号;
}
```

**简单来说**：选择“公众号身份”，视频号主页和直播间将展示公众号信息，并引导用户关注公众号；选择“视频号身份”，则展示视频号信息，并引导用户关注视频号本身。

## 如何选择“内容身份”？

既然“内容身份”选项让你可以选择粉丝关注的主体是视频号还是公众号，那么我们该如何做出明智的选择呢？

这里提供一个基于实践经验的参考方案，主要依据你的粉丝量级和业务需求来决定。

以下是具体的策略建议：

*   **粉丝量1万以内**：建议选择 **“视频号身份”**。
    *   **原因**：视频号平台规定，粉丝达到1万后可以申请“兴趣认证”或“职业认证”（原“黄V认证”），从而获得官方标识、流量扶持等诸多权益。前期应优先积累视频号本身的粉丝，以满足认证条件，不浪费平台给予的成长机会。

![](img/7a5d27ca61c07876159530326f5600b4_6.png)

*   **粉丝量超过1万后**：可以根据你的实际需求进行选择。
    *   **如果你有后端业务**（如课程、咨询、商品销售等），需要沉淀私域流量或获取销售线索，建议选择 **“公众号身份”**。因为公众号在内容沉淀、用户互动、菜单设置和营销转化方面功能更为成熟和方便。
    *   **如果你没有强烈的后端业务需求**，且对公众号的依赖较弱，则可以选择继续保持 **“视频号身份”**，专注于视频号平台的内容创作与粉丝运营。
