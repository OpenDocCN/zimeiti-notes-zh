# 新媒体运营：P15：01.10.03：🤖 训练AI模拟爆款笔记到产出爆款笔记的过程

在本节课中，我们将学习如何训练AI（如ChatGPT）分析爆款笔记，并最终生成具备爆款潜质的新笔记。我们将从提供资料、分析爆款因素，到整合要点并生成文案，完成一个完整的流程。

上一节我们介绍了通过“喂资料”的方式，让AI理解特定平台的文案风格和用语。虽然上一节以旅游攻略为例的成果可能不理想，但方法是正确的。关键在于让AI透彻分析你提供的文章，从而学习到该平台的文案写作方式。

本节中，我们来看看如何让AI分析具体的爆款内容，并学习其成功要素，最终用于产出我们自己的笔记。

以下是分析爆款内容时需要关注的几个核心点：

1.  **爆款因素分析**：分析内容为何能成为爆款，例如情感共鸣、实用价值、故事性等。
2.  **整体结构分析**：分析文案的结构布局。
    *   **开篇**：用什么方式吸引用户继续阅读。
    *   **中间**：提供哪些内容来戳中用户痛点，提升用户停留时长。
    *   **结尾**：设置什么“钩子”来引导用户互动（如私信、关注）。
3.  **关键词分析**：分析笔记中是否植入了特定关键词。
    *   如果有，关键词是什么？
    *   关键词的出现频率和分布结构是怎样的？

## 第一步：提供爆款文案并下达分析指令

首先，我们需要找到目标领域的爆款笔记文案。本节课以“考研”领域为例。

![](img/9909a904ec187bef55e99b4dde32dc01_13.png)

![](img/9909a904ec187bef55e99b4dde32dc01_15.png)

找到文案后，我们需要给AI下达清晰的分析指令。指令应包含我们上述提到的所有分析维度。

![](img/9909a904ec187bef55e99b4dde32dc01_17.png)

![](img/9909a904ec187bef55e99b4dde32dc01_19.png)

**示例指令如下：**

```
我给你提供一篇小红书平台上关于考研的爆款笔记文案，你需要分析：
1. 爆款因素：这篇文案为什么能火？列出具体原因。
2. 整体结构：
   - 开篇用什么方法吸引用户？
   - 中间用什么内容戳中用户痛点？
   - 结尾用什么方式引导用户互动（如私信、关注）？
3. 关键词分析：分析整篇文案是否结合中文分词法设计了关键词。如果没有，请说明。如果有，请指出是哪个关键词，并说明其出现频率。
```

![](img/9909a904ec187bef55e99b4dde32dc01_23.png)

将找到的爆款文案和这条指令一并提交给AI。

## 第三步：整合爆款因素并生成大纲

![](img/9909a904ec187bef55e99b4dde32dc01_50.png)

![](img/9909a904ec187bef55e99b4dde32dc01_52.png)

![](img/9909a904ec187bef55e99b4dde32dc01_53.png)

在AI分析了足够多的样本后，我们需要让它进行总结和整合。

**可以下达如下指令：**

```
请将你前面分析过的所有关于考研的爆款笔记因素总结一下，去重后整合成一个完整的爆款因素列表。
```

![](img/9909a904ec187bef55e99b4dde32dc01_55.png)

![](img/9909a904ec187bef55e99b4dde32dc01_57.png)

AI会生成一个整合后的列表，例如：
*   情感共鸣（开篇制造焦虑或担忧）
*   逆袭故事分享
*   提供明确、详细的备考计划
*   给出针对性建议和实用技巧
*   强调时间管理和重要节点
*   结尾正能量鼓励并引导互动
*   使用表情符号增加亲和力

接下来，基于这些爆款因素，让AI为我们规划一篇新笔记的大纲。

![](img/9909a904ec187bef55e99b4dde32dc01_59.png)

**指令如下：**

```
现在，请结合你总结出的爆款笔记因素，为我规划一篇关于考研的小红书笔记大纲。大纲需要结构清晰，每个主要部分下要列出细分的要点。
```

AI会生成一个包含引言、专业选择、各科备考计划、时间管理、心态调整、结尾互动等部分的大纲。检查这个大纲是否涵盖了爆款因素，并进行调整。

![](img/9909a904ec187bef55e99b4dde32dc01_61.png)

![](img/9909a904ec187bef55e99b4dde32dc01_62.png)

## 第四步：根据大纲生成完整文案

大纲确认无误后，就可以让AI根据大纲和爆款因素生成完整的笔记文案了。

**最终的生成指令需要更具体：**

```
请按照你提供的大纲，撰写一篇完整的小红书考研备考笔记文案。
要求：
1. 文案结构严格遵循爆款笔记的结构。
2. 开篇需要融入一个简单的“逆袭”小故事来引发读者共鸣。
3. 文中需要包含适量的emoji表情。
4. 需要自然地融入核心关键词，例如“考研”、“备考计划”、“政治备考”等。
```

AI会根据所有指令生成一篇初稿。这篇初稿已经具备了爆款笔记的基本框架和元素。对于一般运营需求，它可能已经足够。当然，你还可以在此基础上进行人工优化和调整，使其更精炼、更生动。

![](img/9909a904ec187bef55e99b4dde32dc01_64.png)

## 进阶提示：使用定制化AI助手

在演示中，我们使用了ChatGPT 3.5。如果你使用更强大的模型（如GPT-4）或利用“GPTs”功能创建自定义助手，效果会更好。
你可以创建一个专属的“小红书文案助手”，在它的知识库中上传：
*   平台专用的表情符号列表
*   多种风格的小红书文案范例
*   你的账号特定要求
这样，AI助手就能更稳定、更精准地输出符合你需求的文案，避免在长对话中遗忘之前的指令。

本节课中，我们一起学习了从训练AI分析爆款笔记到生成新笔记的完整流程。关键步骤包括：提供样本并下达精准分析指令、整合爆款因素、规划大纲、最终生成文案。掌握这个方法，你可以将其复制到任何内容领域，高效地产出具备爆款潜质的笔记内容。
