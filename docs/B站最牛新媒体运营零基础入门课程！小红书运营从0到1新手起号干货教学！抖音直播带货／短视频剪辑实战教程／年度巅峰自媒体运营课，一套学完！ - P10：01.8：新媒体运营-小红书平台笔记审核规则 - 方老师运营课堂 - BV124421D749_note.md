上一节我们介绍了课程概述，本节中我们来看看小红书笔记发布后的第一个关键环节：**审核阶段**。

笔记发布后，会依次进入三个审核阶段：**机器审核**、**去重检测**和**人工审核**。大部分正常笔记在机器审核阶段即可通过。

### 机器审核
笔记上传后，首先进入审核系统。系统会根据预设规则进行初步判断。

以下是机器审核的主要规则：
*   **敏感内容检测**：识别笔记中的敏感词、敏感图片等违规内容。
*   **基础质量评估**：评估笔记的字数、广告词出现频率、图片清晰度及文字可识别性。

如果笔记存在明确违规，会被直接封禁。如果内容疑似违规或机器难以判断，则会被标记并转入人工审核阶段。若笔记无问题，则进入下一阶段。

上一节我们了解了机器审核，本节中我们来看看笔记如何避免被判定为抄袭，即**去重检测**阶段。

![](img/329b6ef866e773067133fac93478c25d_10.png)

通过机器审核的笔记，会进入去重检测阶段。系统会将你的笔记与平台数据库中的现有内容进行比对，计算相似度。

以下是去重检测的核心比对维度：
*   **图片相似度**：检测图片是否为直接复制或轻微调整（如裁剪、加滤镜）。
*   **文案重复率**：检测文案内容、段落在数据库中的重复比例。

如果笔记被检测为高度重复或抄袭，例如 `相似度 == 100%`，则会被封禁。如果对素材进行了足够多的原创性修改，使系统无法判定为抄袭，笔记即可通过此关。

---

在笔记通过审核后，平台如何决定将它推送给谁呢？这涉及到**账号权重**和**内容匹配**。

系统会为每个账号评估一个隐形的**权重分**，这会影响笔记的初始推荐流量。权重分并非公开数据，但我们可以理解其运作逻辑。

以下是账号权重在不同阶段的特点：
*   **新号阶段（低权重）**：新注册账号权重接近零。笔记初始推荐流量很少（如30-40次展示），且因账号标签未形成，推送不精准。此阶段核心是“养号”，需保持内容垂直、持续更新、严格避免违规。
*   **成熟账号（高权重）**：持续发布优质垂直内容后，账号在特定领域（如彩妆）的权重会提升。高权重账号发布的笔记，能获得更高的初始推荐流量基数和更精准的受众推送。

假设权重范围是 `0 到 10`，新号权重为 `0`，而优质账号权重可能达到 `3` 或 `4`。权重越高，笔记的冷启动效果越好。

最后，我们来了解一下作为补充的**人工审核**阶段。

人工审核主要针对机器标记的**违规**或**疑似违规**内容进行二次复核。由于内容量巨大，并非所有笔记都会经历人工审核。

以下是人工审核的要点：
*   **审核范围**：主要集中在政治、色情擦边、法律禁止等高风险领域。
*   **当前趋势**：随着算法进步，机器审核准确率提升。触发明确红线的笔记，很可能不经人工复核直接处理。
*   **申诉渠道**：如果认为笔记被误判，可以通过官方渠道申诉。但对于新号，建议以预防为主，因为违规记录可能产生隐形的限流等负面影响。

本节课中我们一起学习了小红书笔记的审核与推荐机制。我们了解到，笔记发布后需通过**机器审核**、**去重检测**，并可能进入**人工审核**。同时，**账号权重**这一隐形分数深刻影响着笔记的初始流量。作为运营者，理解这些规则有助于我们创作合规、优质的内容，平稳度过新号养号期，并逐步提升账号权重，从而获得更好的曝光效果。
