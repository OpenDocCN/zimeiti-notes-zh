## 第一步：获取免费授权与素材 🗂️

授权是推广的第一步，核心是免费获取官方正版素材。以下是两种主要方法。

![](img/276de56aa07473530a6d736ea72444a2_17.png)

### 方法一：通过“巨量星图”获取

![](img/276de56aa07473530a6d736ea72444a2_19.png)

此方法通过官方渠道申请，完全免费。

![](img/276de56aa07473530a6d736ea72444a2_21.png)

1.  打开抖音，搜索“**巨量星图**”并进入官方小程序。
2.  在页面中选择“**我是达人**”身份，进入星图页面。
3.  在搜索框中搜索“**短剧**”，会出现大量短剧项目。
4.  选择**分佣比例较高**（例如70%、80%）的剧集进行合作。避免选择分佣比例过低（如1%）的项目。
5.  进入具体剧集页面后，下滑找到“素材”区域。
6.  复制素材链接，在浏览器中打开即可下载高清视频素材到手机相册。

![](img/276de56aa07473530a6d736ea72444a2_23.png)

![](img/276de56aa07473530a6d736ea72444a2_25.png)

### 方法二：通过“小程序推广计划”获取

![](img/276de56aa07473530a6d736ea72444a2_27.png)

此方法同样免费，并可参考高收益案例。

1.  在抖音搜索“**小程序推广计划**”并进入。
2.  在“热门小程序”中，寻找并进入收益较高的短剧剧场（如“千亿剧场”）。
3.  选择一部当前最高收益较高的剧集进入。
4.  点击“**视频榜单**”，可以查看其他推广者的视频收益与剪辑方式，作为参考。
5.  要获取素材，返回剧集主页，点击“**体验**”按钮。
6.  在体验页面中，使用手机的**录屏功能**，录制你需要的剧集片段作为素材。

## 第二步：视频剪辑与去重处理 ✂️

![](img/276de56aa07473530a6d736ea72444a2_29.png)

![](img/276de56aa07473530a6d736ea72444a2_31.png)

获取素材后，需进行二次剪辑以避免内容重复。我们将素材导入“剪映”App进行操作。

![](img/276de56aa07473530a6d736ea72444a2_33.png)

以下是核心的去重与增强操作：

![](img/276de56aa07473530a6d736ea72444a2_35.png)

*   **去除字幕**：选中视频片段，**双手放大画面**，直至原有字幕移出画面之外。
*   **添加滤镜**：为视频添加合适的滤镜，改变原片色调。
*   **添加贴纸/画中画**：加入自己的贴纸、文字或画中画图层，增加原创元素。
*   **调整速度与顺序**：适当调整片段播放速度或重新排列顺序。

![](img/276de56aa07473530a6d736ea72444a2_37.png)

![](img/276de56aa07473530a6d736ea72444a2_39.png)

对于时间有限的创作者，可以使用自动化剪辑工具提高效率。

![](img/276de56aa07473530a6d736ea72444a2_41.png)

1.  打开特定的“**短剧二剪**”工具。
2.  上传最多15个视频素材。
3.  系统会自动进行混剪、去重等处理，生成一个全新的视频版本。

![](img/276de56aa07473530a6d736ea72444a2_43.png)

（注：如需此类工具，可在教程指定处留言获取。）

![](img/276de56aa07473530a6d736ea72444a2_45.png)

剪辑完成后，点击“导出”，将成品视频保存至手机。

![](img/276de56aa07473530a6d736ea72444a2_47.png)

![](img/276de56aa07473530a6d736ea72444a2_49.png)

## 第三步：发布视频与挂载链接 📤

![](img/276de56aa07473530a6d736ea72444a2_51.png)

最后一步是发布视频并正确挂上推广链接，这样才能产生收益。

![](img/276de56aa07473530a6d736ea72444a2_53.png)

1.  回到抖音的“**小程序推广计划**”，找到你准备推广的那部短剧。
2.  **从该剧集的专属入口点击“上传视频”**。
3.  选择你刚刚剪辑好的视频，进入发布页面。
4.  此时，推广链接已自动挂载到视频上。
5.  编辑标题、封面后，直接点击发布即可。

![](img/276de56aa07473530a6d736ea72444a2_55.png)

当用户通过你的视频下载小程序或开通会员，你就能获得相应的推广收益。

![](img/276de56aa07473530a6d736ea72444a2_57.png)
