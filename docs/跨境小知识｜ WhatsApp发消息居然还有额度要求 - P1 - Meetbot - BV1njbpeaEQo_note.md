# WhatsApp营销教程：P1：了解消息额度与升级机制 📊

在本节课中，我们将要学习WhatsApp企业账号的消息额度系统。这是一个关键概念，决定了您能在24小时内主动向多少客户发送消息。理解并管理好您的消息额度，是进行有效WhatsApp营销的基础。

## 什么是消息额度？📈

消息额度，特指企业账号在连续24小时内，可以主动向客户发送消息的数量上限。这与个人账号的使用方式有显著区别。

## 消息额度等级体系 🏆

WhatsApp为企业账号设定了明确的消息额度等级。以下是五个不同的额度级别：

*   **第一级：** 250条/24小时
*   **第二级：** 1，000条/24小时
*   **第三级：** 10，000条/24小时
*   **第四级：** 100，000条/24小时
*   **第五级：** 无限额

## 初始额度与验证状态 🔑

您的初始消息额度取决于账号的验证状态。这直接关联到您的商业信誉和WhatsApp的信任级别。

*   **完成商业平台（BM）企业验证：** 初始额度为 **1，000条/天**。
*   **BM申请并获得“已验证”徽章：** 初始额度可高达 **100，000条/天**。

## 额度升级机制 ⬆️

在上一节我们了解了初始额度，本节中我们来看看如何提升您的消息额度。升级并非自动发生，需要满足特定条件。

当同时满足以下两个条件时，您的账号额度将在24小时后自动升级到下一个更高等级：

1.  **发送量条件：** 过去7天内发送的消息总数超过当前额度等级的一半。
    *   *例如，当前额度为1，000条/天，则过去7天总发送量需超过 `1，000 / 2 * 7 = 3，500` 条。*
2.  **账号质量条件：** 账号质量评级为“中”或“高”。这通常基于用户反馈（如举报、拉黑）等因素评定。

## 额度降级机制 ⬇️

保持良好的账号质量至关重要。如果您的账号表现不佳，额度可能会被降低。

*   **降级条件：** 如果账号连续7天被用户标记（如举报为垃圾信息），消息额度将会下降一个等级。
