# 新媒体运营：P29：11：小红书平台用户标签及流量入口 📊

在本节课中，我们将学习小红书平台如何为用户打上标签，以及笔记是如何通过不同的流量入口被推荐给用户的。理解这些底层逻辑，有助于我们更好地定位内容，并让笔记被更多精准用户看到。

上一节我们介绍了小红书如何判定内容标签，本节中我们来看看平台如何判定用户标签。

## 用户标签的判定依据

![](img/f141efce6a63b34833f0578128f1b6ed_8.png)

![](img/f141efce6a63b34833f0578128f1b6ed_9.png)

![](img/f141efce6a63b34833f0578128f1b6ed_10.png)

![](img/f141efce6a63b34833f0578128f1b6ed_11.png)

![](img/f141efce6a63b34833f0578128f1b6ed_12.png)

系统为用户打标签的主要依据是用户的**浏览行为**。用户刚注册账号时所选择的兴趣范围，参考价值不大。这仅是平台在初期对用户内容方向的一个引导，只能进行初步筛选，无法进入小红书系统最深层的用户标签池。

![](img/f141efce6a63b34833f0578128f1b6ed_14.png)

![](img/f141efce6a63b34833f0578128f1b6ed_16.png)

![](img/f141efce6a63b34833f0578128f1b6ed_18.png)

![](img/f141efce6a63b34833f0578128f1b6ed_19.png)

系统判定用户的兴趣分类，是根据用户的浏览行为来决定的。以下是几个关键的行为指标。

以下是判定用户标签的几个核心行为指标：

![](img/f141efce6a63b34833f0578128f1b6ed_21.png)

![](img/f141efce6a63b34833f0578128f1b6ed_22.png)

![](img/f141efce6a63b34833f0578128f1b6ed_23.png)

![](img/f141efce6a63b34833f0578128f1b6ed_24.png)

![](img/f141efce6a63b34833f0578128f1b6ed_25.png)

1.  **点击率**
    *   **点击率**是指系统在发现页给用户推送多个内容（如明星、美妆、家居、母婴、装修等）后，用户点击哪一类内容更多。
    *   例如，用户点击10个内容，其中美妆类点了3次，家居类点了2次，母婴类点了1次或根本没点。系统会判定该用户对母婴内容不感兴趣，可能不是宝妈。
    *   如果用户对美妆类内容感兴趣，系统将持续推送更多美妆领域的内容进行试探。通过点击率的逻辑，系统能为用户打上强关联或弱关联的标签。

2.  **停留时长**
    *   停留时长是分析用户喜好的另一个重要行为。
    *   例如，用户喜欢美妆内容，但很多美妆笔记点开后就划走了，说明这些笔记不是他喜欢的。系统会尝试推送美妆领域内其他细分内容（如上粉、护肤等）来观察用户行为。
    *   如果用户偶然点开一个家居类笔记（平时很少点），但停留了很久，仔细浏览图片和评论，系统可能判定该用户有装修需求或想购买相关商品（如盘子），从而考虑推送更多相关笔记或商品。

3.  **页面浏览量**
    *   **页面浏览量**可以理解为用户浏览页面的深度，这个数值有一定价值。
    *   例如，用户刷到一篇装修笔记，点进去后不仅看了这篇笔记，还进入了博主主页，并浏览了第二、第三、第四篇笔记。这说明用户浏览深度足够。
    *   如果你的账号主页内容做得非常精致，就能提升用户在你页面的浏览数。这个数值对账号健康度评分有很大帮助。浏览深度足够时，还可能促使用户产生**关注**行为。

![](img/f141efce6a63b34833f0578128f1b6ed_27.png)

![](img/f141efce6a63b34833f0578128f1b6ed_28.png)

4.  **互动行为**
    *   互动行为包括点赞、收藏、评论、关注。
    *   这些数据是我们在创作每篇笔记时都需要努力提升的。点赞、收藏数值高，笔记获得进入更大推荐流量池的几率更高，也可能带动账号内其他笔记的曝光，提升粉丝数。
    *   互动行为能够直接提升笔记的下一层推荐流量值。

![](img/f141efce6a63b34833f0578128f1b6ed_30.png)

![](img/f141efce6a63b34833f0578128f1b6ed_31.png)

## 流量分发入口

![](img/f141efce6a63b34833f0578128f1b6ed_32.png)

![](img/f141efce6a63b34833f0578128f1b6ed_33.png)

了解用户标签后，我们来看看笔记是通过哪些入口分发给用户的。小红书平台主要有四个流量分发入口：发现页、关注页、附近页和购物。

![](img/f141efce6a63b34833f0578128f1b6ed_35.png)

![](img/f141efce6a63b34833f0578128f1b6ed_36.png)

![](img/f141efce6a63b34833f0578128f1b6ed_37.png)

以下是小红书平台的四个主要流量入口及其特点：

1.  **发现页**
    *   发现页呈现的内容主要包含：
        *   用户已关注的达人发布的内容。
        *   根据用户标签（点击率、浏览行为）推荐的、系统认为用户可能喜欢的内容。
        *   一些用户可能不感兴趣或从未刷过的内容，以增加平台的多样性和探索性。
    *   发现页的价值在于满足用户多样化的需求（如看攻略、教程、趣事分享）。作为博主，你的笔记和账号后期能获得的主要流量也来源于发现页。

![](img/f141efce6a63b34833f0578128f1b6ed_39.png)

![](img/f141efce6a63b34833f0578128f1b6ed_40.png)

![](img/f141efce6a63b34833f0578128f1b6ed_42.png)

2.  **关注页**
    *   关注页的内容非常简单，就是你关注的所有博主发布的笔记。
    *   用户通常从这里直接查看强关注对象的内容。虽然有些人关注列表很长，但关注页仍是找到特定博主内容的途径之一。

![](img/f141efce6a63b34833f0578128f1b6ed_43.png)

![](img/f141efce6a63b34833f0578128f1b6ed_45.png)

![](img/f141efce6a63b34833f0578128f1b6ed_47.png)

![](img/f141efce6a63b34833f0578128f1b6ed_49.png)

![](img/f141efce6a63b34833f0578128f1b6ed_51.png)

3.  **附近页**
    *   附近页类似于早期微信“附近的人”功能，根据手机定位推送附近同城的内容。
    *   这个板块主要以美食、当地旅游等强地域性需求的内容为主。
    *   如果你运营线下实体店（如美容院、餐厅、景点），希望通过小红书推广，就需要特别注意附近人群的推送。在做信息流广告时，可以侧重附近人群的兴趣投放。

![](img/f141efce6a63b34833f0578128f1b6ed_52.png)

![](img/f141efce6a63b34833f0578128f1b6ed_54.png)

4.  **购物页**
    *   购物入口是小红书变现及向电商方向发展的重要板块，主要匹配用户的购物需求。
    *   目前，部分用户还未完全养成在小红书购物的强习惯，更多人可能仍倾向于传统电商平台或抖音。许多用户会通过搜索笔记来了解商品，但笔记中广告出现频率过高也是当前用户反馈较多的问题。

![](img/f141efce6a63b34833f0578128f1b6ed_56.png)

![](img/f141efce6a63b34833f0578128f1b6ed_58.png)

## 总结

![](img/f141efce6a63b34833f0578128f1b6ed_61.png)

本节课中，我们一起学习了小红书平台用户标签的判定机制和四大流量分发入口。

![](img/f141efce6a63b34833f0578128f1b6ed_63.png)

我们了解到，系统主要通过用户的**点击率**、**停留时长**、**页面浏览量**和**互动行为**来为用户打上标签，从而推荐其可能感兴趣的内容。同时，笔记主要通过**发现页**、**关注页**、**附近页**和**购物页**这四个入口触达用户。
