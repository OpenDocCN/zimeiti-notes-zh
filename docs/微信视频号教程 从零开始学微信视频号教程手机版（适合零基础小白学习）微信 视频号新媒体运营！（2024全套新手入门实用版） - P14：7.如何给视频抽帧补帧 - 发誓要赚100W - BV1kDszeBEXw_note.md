# 微信视频号教程：P14：7.如何给视频抽帧补帧 🎬

在本节课中，我们将学习视频剪辑中的一个实用技巧：抽帧与补帧。这个操作可以帮助我们微调视频时长，同时不影响视频的整体流畅度。

上一节我们介绍了视频的“掐头去尾”，本节中我们来看看如何通过抽帧和补帧来进一步优化视频。

## 理解帧与时长

首先，我们需要理解一个基本概念：视频是由一帧一帧的连续画面组成的。在大多数手机视频中，**每秒包含30帧**（即 `30 FPS`）。这意味着一个18秒的视频，总共由 `18秒 × 30帧/秒 = 540帧` 画面构成。

## 什么是抽帧与补帧？

*   **抽帧**：指从视频中删除某一帧或几帧画面。
*   **补帧**：指将视频中的某一帧画面复制并插入到其他位置。

这两个操作通常配合使用，可以在几乎不影响观感的前提下，对视频时长进行微调。

以下是进行抽帧与补帧的具体步骤：

1.  **放大时间轴**：将视频轨道的时间轴拉到最长的刻度，以便我们能精确地看到每一帧。

2.  **定位帧**：观察时间码，例如从 `00:00:00` 到 `00:00:01` 这一秒内，就包含了30个帧（F）。

3.  **执行抽帧**：
    *   在你想删除的某一帧位置（例如第20帧处）进行分割。
    *   紧接着在下一帧（第21帧处）再次分割。
    *   选中这单独的一帧片段，将其删除。这个操作就完成了“抽帧”。

4.  **执行补帧**：
    *   在视频的其他位置（例如结尾附近），将时间轴拉长并进行分割，创造出一个空隙。
    *   选中之前步骤中分割出来的任意一帧片段。
    *   点击“复制”，然后将复制出来的帧粘贴到刚才创建的空隙中。这个操作就是“补帧”。

## 操作要点与建议

*   对于一个完整的视频，通常**抽3帧、补3帧**即可，这样对视频的整体流畅度影响最小。
*   抽帧和补帧的位置可以分散在视频的不同段落。
*   完成操作后，视频的总时长和主体内容保持不变，但内部帧序列经过了微调。
