# 自媒体运营系统课：P5：2.自媒体平台现状 📊

在本节课中，我们将要学习自媒体行业的现状，分析其可行性与发展前景，并了解国家政策与平台支持，帮助你判断是否值得进入这个领域。

上一节我们介绍了什么是自媒体，本节中我们来看看这个行业是否值得去做，以及它为我们提供了哪些机会。

## 行业可行性分析

很多同学在考虑兼职、转行或寻找副业时，都会对这个行业能否真正带来收益产生疑问。近年来，由于各种原因，许多上班族希望寻找额外收入来源。

以下是需要注意的关键点：
*   必须避免抱有“一夜暴富”的不切实际心态。
*   绝不能采取非法手段，例如前往某些地区从事违法活动，这不仅没有好处，还会危害人身安全。

现阶段，自媒体行业是国家明确支持的合法行业。国家已将支持文件下达到各级地方政府。

既然有了国家文件支持，我们可以放心去做。但必须强调一点：国内自媒体平台审核严格，**黄、赌、毒及擦边内容绝对不能做**。例如，某些海外平台上的内容不符合国内规定，不能搬运到国内平台。

有句话说得好：跟着国家的脚步走，一定能赚到钱。现在政策支持我们成为视频创作者，那么平台具体给了我们哪些支持呢？

## 平台资金扶持

![](img/c26822152f78a33052b9761a49a46213_7.png)

首先，我们通过一个视频了解一下平台的扶持计划。西瓜视频曾宣布，未来一年将额外拿出20亿元补贴中视频创作人。

![](img/c26822152f78a33052f78a33052b9761a49a46213_7.png)

从这个视频中，我们可以获取两个核心信息：
1.  扶持金额是**20亿元**。
2.  这20亿元是**纯补贴**，不包含任何商单、直播和电商的收入，是纯播放量收益。

这说明了什么？它说明作为自媒体创作者，**赚钱的路径不只有带货一条**。通过制作视频或文章，依靠播放量同样可以实现变现。这20亿元，就是平台为我们准备好的、可以通过创作直接获取的收益。

## 市场现状与前景

除了资金支持，自媒体行业本身拥有巨大的流量池。根据相关报告，现阶段中国网民规模约为9.4亿人。从中小学生到成年人，各个年龄段的人都在刷视频。

那么，做视频的人有多少呢？截至去年，数据显示：
*   全职从事自媒体：**370万**
*   兼职从事自媒体：**600万**
*   总计约 **970万**，可近似看作1000万人。

这就形成了一个比例关系：**约10亿看视频的人 vs 约1000万做视频的人**。相当于每100个看视频的人，只有1个人在生产内容。

请问，1个人生产的视频，足够100个人看吗？显然不够。一个人制作一个视频可能需要半小时，而观众刷完一个视频只需几秒或几分钟。这种**供需不平衡**，正是国家支持自媒体行业发展的重要原因。

很多同学担心行业是否已经饱和。我们来看行业趋势分析：在大数据时代背景下，**自媒体行业远远没有饱和**。

“远远没有”这个词非常重要。它意味着在可预见的未来（例如未来十年），你都可以将自媒体作为兼职或全职来发展，并从中获得收益。
