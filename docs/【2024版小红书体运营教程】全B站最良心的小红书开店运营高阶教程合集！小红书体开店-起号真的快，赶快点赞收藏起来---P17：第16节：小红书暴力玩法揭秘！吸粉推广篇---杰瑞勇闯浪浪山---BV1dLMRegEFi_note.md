## 账号类型解析

上一节我们介绍了小红书的基础运营，本节中我们来看看平台上的账号类型。了解不同类型的账号及其用途，是制定有效推广策略的第一步。

小红书账号主要分为以下三类：

1.  **企业号**：主要作用是树立官方品牌形象。例如，可以拥有官方认证标志、展示品牌名称与地址等特权，便于开展业务。
2.  **个人号**：这是目前大多数用户运营的账号类型。其特点是**灵活性高**，可以是品牌员工或消费者身份，更容易塑造个人品牌特色，也适合品牌相关人物出镜。
3.  **引流号**：这类账号在之前的课程中也有提及，俗称“炮灰号”。其作用通常是辅助引流，可以多个存在，无需精细运营，在需要时可作为备用小号使用。这类账号一般没有太多个性化规划。

通常，品牌推广会通过企业号、个人号或引流号发布笔记来进行引流。

## 打造企业号矩阵的建议

了解了账号类型后，我们来看看如何系统性地运用它们。本节将提供打造企业号运营矩阵的具体建议，以协同增效。

以下是构建企业号矩阵的四点核心建议：

![](img/e8b2ab2cba453003c09ba329c109bfdd_1.png)

![](img/e8b2ab2cba453003c09ba329c109bfdd_2.png)

*   **积极参与品牌活动**：对于婚纱、服装、美妆等品类的企业号，可以组织试用、团购等活动，并利用小号积极参与，增加活动热度与真实性。
*   **个人号强化品牌属性**：在个人号的笔记内容中，应多次暗示品牌的定位属性。例如，品牌目标用户是都市青年还是职场新人，需要通过内容持续传递。
*   **控制产品发布频率**：个人号发布品牌产品内容的频率不宜过高。一个较好的比例是 **营销内容：种草/科普内容 = 3：7**。即30%的内容用于直接营销，70%的内容用于价值输出。
*   **利用引流号进行评论引流**：当企业号需要直接引流时，最佳方式是利用引流号在其笔记评论区进行互动和引导，这样更为自然且安全。

## 官方广告投放详解

![](img/e8b2ab2cba453003c09ba329c109bfdd_7.png)

![](img/e8b2ab2cba453003c09ba329c109bfdd_9.png)

除了自建账号矩阵，另一种高效的推广方式是借助平台力量。本节中我们来看看小红书官方的广告投放系统。

小红书官方广告主要指 **小红书CPC广告**，即按点击付费的竞价广告。要使用此功能，必须先认证企业号。

![](img/e8b2ab2cba453003c09ba329c109bfdd_11.png)

**企业号认证流程**：在小红书官网的“业务合作”板块，按照指引提交相关资质并支付审核费用即可完成认证。

认证企业号后，便可投放信息流广告。官方信息流广告主要分为两大类：

1.  **信息流广告**：用户浏览小红书首页信息时，不定时出现在信息流中的广告。
2.  **搜索广告**：当用户使用关键词搜索时出现的广告。其原理类似于百度竞价，通过对关键词出价来竞争不同的广告位排名。当前信息流广告位主要分为：**第3位、第13位、第23位**。

**搜索广告的匹配逻辑示例**：
```python
# 简化的广告匹配逻辑（示意）
if 用户搜索词 in [“考研”, “考研资料”]:
    可能推荐广告 = “能量补充商品”或“考研课程”
elif 用户搜索词 == “帽子”:
    可能推荐广告 = “大衣”或“围巾” # 搭配推荐
```
官方会根据算法，在用户搜索核心词时，智能推荐相关联的商品或服务。

![](img/e8b2ab2cba453003c09ba329c109bfdd_13.png)

关于广告投放的常见问题，可在小红书广告平台的官方问答区查阅。官方广告投放费用较高，但在资金允许的情况下，建议考虑合作，因为它能直接获得平台推荐的精准流量。
