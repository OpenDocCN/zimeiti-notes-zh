# 小红书运营实操课：P3：03_第2课：如何开通小红书店铺 🛍️

在本节课中，我们将学习如何开通小红书店铺。我们将了解店铺类型、认证账号的区别、保证金规则，并手把手演示完整的开店流程。

## 核心概念区分

在开始开店流程前，需要明确两个核心概念：**小红书账号**与**小红书店铺**。

*   **小红书账号**：指用户用于浏览、发布内容的个人或企业身份。主要分为两种：
    1.  **个人认证号**：即普通用户账号。
    2.  **企业认证号**：带有蓝色“企业号”标识的账号，需缴纳600元认证费。

*   **小红书店铺**：指在账号内开通的、用于销售商品的线上店铺。它与账号是绑定关系。

上一节我们介绍了账号与店铺的基本概念，本节中我们来看看账号类型与店铺类型之间的关联与限制。

## 账号与店铺的关联规则

![](img/2d8d79baf7a9f7f8d888ec37b355e849_1.png)

以下是账号类型与可开通店铺类型的具体规则：

![](img/2d8d79baf7a9f7f8d888ec37b355e849_3.png)

*   **个人认证号**：一个身份（个人或个体户）**只能开通一家**个人店或个体店。
*   **企业认证号**：可以开通企业店，并且**能绑定开通两家个体店**。

对于大多数新手，建议从**个人认证号**开通**个人店**开始，操作简单，成本较低。

## 小红书店铺的三种类型

小红书店铺主要分为三种类型，它们的开通条件和权限各不相同。

以下是三种店铺类型的详细说明：

1.  **个人店**
    *   **开通条件**：仅需身份证。注册当天通常即可通过审核。
    *   **命名规则**：格式为 `{自定义名称}的店`，例如“玲玲工厂童装的店”。名称需专业，以建立可信人设。
    *   **可售类目**：覆盖大部分基础类目，如服饰鞋包、日用百货等。具体需在开通时查看。
    *   **注意事项**：开店前需在平台搜索店铺名称，检查是否重名。若名称已被占用，需等待7天才能再次申请。

![](img/2d8d79baf7a9f7f8d888ec37b355e849_5.png)

![](img/2d8d79baf7a9f7f8d888ec37b355e849_7.png)

2.  **个体店**
    *   **开通条件**：需要**营业执照**。执照可通过淘宝代办理（约100元）、工商局或自行网上申请获得。建议选择“互联网销售”等宽泛经营类目。
    *   **可售类目**：比个人店更丰富。
    *   **数量规则**：**一个营业执照可开通两家个体店**。

3.  **企业店**
    *   **类型细分**：分为普通企业店、旗舰店、专卖店。开通需相应品牌授权或资质，流程较复杂。
    *   **风险提示**：购买资质开店存在风险，一旦涉及商品打假维权会比较麻烦。
    *   **命名规则**：
        *   普通企业店：`{自定义名称}的店`
        *   专卖店：`{品牌或公司名称}专卖店`
    *   **通用规则**：店铺名不能超过45个字，且不能出现未经授权的Logo、大牌名称或商标信息。

![](img/2d8d79baf7a9f7f8d888ec37b355e849_9.png)

了解了店铺类型后，接下来我们需要关注开店的一项重要成本：保证金。

![](img/2d8d79baf7a9f7f8d888ec37b355e849_11.png)

## 店铺保证金说明

![](img/2d8d79baf7a9f7f8d888ec37b355e849_13.png)

保证金是平台为确保商家履约而收取的押金，不同店铺类型规则不同。

![](img/2d8d79baf7a9f7f8d888ec37b355e849_15.png)

![](img/2d8d79baf7a9f7f8d888ec37b355e849_17.png)

*   **个人店**：保证金通常为1000元，但**可以暂不缴纳**。不过，若不缴纳，将无法开通运费险和微信支付等功能。
*   **个体店**：**必须缴纳**1000元保证金。
*   **企业店**：**必须缴纳**保证金，否则无法发布商品。
*   **注意**：保证金金额可能因类目不同而有浮动，但大部分品类为1000元。

## 手把手开店流程演示 📱

现在，我们进入实操环节，演示如何用手机开通个人店铺。

![](img/2d8d79baf7a9f7f8d888ec37b355e849_19.png)

1.  **进入开店入口**
    登录小红书账号，点击首页右下角“我”，然后点击右上角“三条杠”菜单，选择“创作中心”。

2.  **选择开通店铺**
    在创作中心页面，找到并点击“开通店铺”或类似选项，然后点击“立即开店”。

3.  **选择店铺类型**
    在店铺类型页面，选择“个人店”。可以点击“可售类目”查看你的商品是否在允许范围内。

4.  **填写经营信息**
    *   勾选“定制商品”选项（通常适用）。
    *   选择经营类目（如服饰鞋帽、家居百货等）。类目后期可以添加或修改，初期可先选择主要经营的。
    *   确认保证金金额（可暂不支付）。

5.  **填写身份信息并提交**
    *   根据提示，完整填写个人身份信息。
    *   “邀请码”可不填。
    *   阅读并同意协议，点击提交。

6.  **等待审核**
    提交后，店铺进入审核流程。**通常当天就能审核通过**。审核通过后，即可开始上架商品。
