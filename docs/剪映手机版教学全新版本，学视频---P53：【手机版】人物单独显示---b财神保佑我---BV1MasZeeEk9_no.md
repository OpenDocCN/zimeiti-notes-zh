# 剪映手机版教学：P53：人物单独显示效果

在本节课中，我们将学习如何制作“人物单独显示”效果。这种效果常用于视频号运营和人物介绍，能让画面主体（人物）从背景中突出，营造出强烈的视觉焦点。我们将通过两个案例，从基础到进阶，详细讲解图层关系、抠像、滤镜、特效和关键帧等核心功能的运用。

## 概述与案例预览

上一节我们学习了画中画的基础操作，本节中我们来看看如何利用画中画和抠像功能，制作人物定格和色彩渐变两种“人物单独显示”效果。掌握这些技巧，能让你的视频更具专业感和创意。

---

### 第一部分：人物定格介绍效果

首先，我们学习第一个案例：让人物在某一瞬间定格，背景变为黑白，而人物保持彩色并添加动态效果。

#### 1. 导入素材与音乐

点击“开始创作”，导入准备好的视频素材。
将视频添加到轨道后，关闭视频原声。
点击“添加音频” -> “提取音乐”，从其他视频中导入合适的背景音乐。

#### 2. 制作定格画面

在时间轴上选中视频，滑动到人物动作合适的瞬间（例如手放在额头时）。
点击底部菜单的“定格”功能。这会在该时间点生成一张静态图片。
删除定格点之后多余的视频片段。
调整定格图片的时长，使其与音乐的高潮部分对齐。

#### 3. 创建画中画并抠像

选中定格图片，点击“复制”。
选中复制出来的片段，点击“切画中画”，并将其拖到与主轨道视频开头对齐。
现在，我们有了上下两层相同的画面。
选中画中画图层，点击“抠像” -> “智能抠像”。系统会自动识别人物。
抠像成功后，画中画图层就只显示被抠出的人物，下层则是完整的原始背景。

**核心概念解释**：
此时，视频的**图层关系**如下：
*   **底层**：主轨道视频（原始背景）。
*   **上层**：画中画图层（已抠出的人物）。
上层人物会遮挡下层的背景。

#### 4. 处理背景与添加特效

选中底层的主轨道视频，点击“滤镜”。
在“黑白”滤镜组中，选择一个效果（如“墨片”），并将强度拉至100。这样背景就变成了黑白。
返回主菜单，点击“特效” -> “画面特效”。
在“综艺”分类下，找到“冲刺”特效并添加。
将特效轨道的长度拉至与视频一致。
默认特效只作用于“主视频”。点击选中特效轨道，在“作用对象”中选择“全局”，让特效同时影响主视频和画中画。

#### 5. 添加动态文字

返回主菜单，点击“文本” -> “新建文本”，输入标题（如“狂飙 高启强”）。
为文字选择“花字”样式和创意“字体”（如“超级战甲”）。
调整文字大小和位置，使其在画面中居中。
选中文字轨道，点击“动画”，在“循环”类目下选择一个动画（如“摇荡”），并可调节动画速度和幅度。
最后，将文字轨道的时长与定格画面部分对齐。

#### 本节小结

在本部分，我们一起制作了人物定格效果。关键步骤是：**复制图层 -> 切为画中画 -> 智能抠像 -> 为背景添加滤镜**。我们理解了**图层上下关系**和特效的**作用对象**设置，这是实现复杂效果的基础。

### 第二部分：人物色彩渐变效果

接下来，我们通过第二个案例巩固所学，并学习新的“关键帧”功能，实现人物保持彩色而背景逐渐变为黑白的效果。

#### 1. 复制视频创建图层

导入一段人物行走的视频。
选中视频，点击“复制”，得到两段相同的视频。
选中前一段视频，点击“切画中画”，并将其拖到与主视频开头对齐。
这样，我们同样得到了上下两层相同的视频画面。

#### 2. 抠出人物

选中上层的画中画视频，点击“抠像” -> “智能抠像”，等待抠像完成。
此时，画中画图层只显示被抠出的人物。

#### 3. 使用关键帧制作背景渐变

选中下层的主视频轨道，点击“滤镜”，添加一个黑白滤镜（如“墨片”）。
将时间线移动到视频的**结尾处**，点击轨道上的**添加关键帧**按钮（菱形图标）。这个关键帧记录了“滤镜强度为100”的状态。
将时间线移动到视频的**开头处**，再次进入“滤镜”设置，将滤镜强度滑块拖到**0**。系统会自动在此处生成第二个关键帧，记录“滤镜强度为0”的状态。

**核心概念解释**：
**关键帧**是记录画面属性变化的关键点。系统会自动在两个关键帧之间创建平滑的过渡动画。
公式表示：`起始关键帧（属性A值） -> 结束关键帧（属性B值） = 自动生成属性从A到B的动画`

现在播放，可以看到背景从彩色逐渐变成了黑白，而画中画的人物始终保持彩色。

#### 4. 添加音乐与字幕

点击“添加音频” -> “提取音乐”，为视频配上抒情的背景音乐。
根据音乐节奏和台词，点击“文本” -> “新建文本”添加字幕。
为字幕选择合适的字体（如复古字体）和样式（如添加描边）。
通过复制和修改文本轨道，让字幕的出现时间与语音同步。

#### 本节小结

在本部分，我们制作了色彩渐变效果。核心是使用**关键帧**来控制滤镜强度的变化，从而让背景产生动态的色彩过渡。这进一步深化了我们对图层叠加和动态效果控制的理解。

---

### 课程总结

本节课中，我们一起学习了两种“人物单独显示”效果的制作方法：
1.  **定格突出效果**：通过定格、抠像、滤镜和特效的组合，营造瞬间的视觉冲击。
2.  **动态渐变效果**：通过复制图层、抠像，并配合**关键帧**动画，实现背景色彩随时间平滑变化。

你需要掌握的核心技能包括：
*   **图层管理**：理解主轨道与画中画的上下层遮挡关系。
*   **智能抠像**：快速分离人物与背景。
*   **滤镜与特效应用**：改变画面风格，并设置正确的**作用对象**。
*   **关键帧动画**：让静态的属性（如滤镜强度、位置、大小）产生动态变化。

通过灵活运用这些功能，你可以创造出更多样化的人物展示效果，提升视频的剪辑水平。
