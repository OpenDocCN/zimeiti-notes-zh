# 小红书无货源电商实战：P2：小红书平台检测规则详解 🕵️

在本节课中，我们将要学习小红书平台的核心检测规则。了解这些规则是安全运营账号、避免违规处罚的基础。我们将从平台检测的重点区域、设备管理、内容规范到原创度保护等多个维度进行解析，并介绍实用的敏感词检测工具。

## 平台检测的核心区域

小红书系统会重点监控以下几个区域的内容，以防止违规引流和营销行为。

以下是平台明确禁止在这些区域出现的内容类型：
*   **个人资料、笔记内容、私信及评论**：不得含有微信号、微信二维码、“代购”信息、第三方平台水印（如淘宝）、以及直接推广店铺的营销词汇。
*   **官方许可的例外**：个人资料中**允许留下邮箱地址**。同时，可以使用“微博”等平台名称进行谐音或间接引流，再通过话术引导至微信。具体的操作话术将在后续课程中详细讲解。

## 账号与设备管理规范

上一节我们介绍了内容层面的检测，本节我们来看看账号与设备的安全管理。平台会检测登录设备的关联情况。

关于设备管理，请注意一个关键限制：
*   同一台手机设备**最多建议只登录3个小红书账号**。超出此数量可能被系统判定为营销号集群，增加风险。

## 内容创作的红线

在创作笔记内容时，必须严格遵守平台规定，避免任何诱导互动行为。

以下是严令禁止的诱导用户行为：
*   **关注领奖**：例如，“关注我，抽奖送XX”。
*   **点赞抽奖**：例如，“点赞这条笔记，参与抽奖”。

这些行为都属于**诱导互动**，会触发系统的处罚机制。

## 原创度保护机制

小红书平台对原创内容的保护程度非常高。因此，在推广时需特别注意内容的发布策略。

关于原创度，必须遵守一个核心原则：
*   **一稿一发**：同一篇文章**切勿**在多个小红书账号上重复发布。一篇文章只用一个账号发布一次即可，多发会被系统判定为搬运或 spam，导致限流。

## 敏感词检测与规避方法

除了上述规则，平台还有一个重要的检测机制：**敏感词过滤**。许多领域天然存在高风险词汇。

以下是一些需要尽量避免触碰的高风险领域关键词示例：
*   `手机卡`、`金融`、`电子烟`、`代购`、`医疗用品`等。

在写作时，如果无意中加入了敏感词或不确定某些词汇是否违规，可以使用敏感词检测工具进行排查。这里推荐两个网站：

**1. 即时检测工具**
打开检测网站，将写好的文案粘贴进输入框。

```python
1. 访问检测网站
2. 粘贴文案内容
3. 点击“立即检测”按钮
```
工具会快速扫描并高亮标出文案中的敏感词，例如“代购”。这样你就能及时发现并修改。
