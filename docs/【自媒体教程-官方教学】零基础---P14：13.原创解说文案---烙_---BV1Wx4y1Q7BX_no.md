# 自媒体运营：P14：13.原创解说文案 📝

在本节课中，我们将要学习如何创作原创解说文案。我们将探讨获取文案素材的不同途径，并重点讲解如何用“说人话”的方式写出吸引人的解说内容，这对于提升视频播放量和收益至关重要。

上一节我们介绍了使用外部素材库的方法，本节中我们来看看其他几种获取或创作解说文案的方式。

除了使用一搜猫素材库，很多同学希望有其他选择。当然可以，我自己就拥有一个纯原创的解说文库。

这个文库中的所有文案都是我原创的内容，可以分享给大家使用。这个文库会由我和我的团队实时更新，每篇文案大约两三千字。

你只需要根据我的文案，去寻找对应的视频素材进行合成即可。这是一种比较方便的操作方式。

就除此之外，有很多同学表示，既不想用一搜猫的，也不想用我的文库，而是想自己动手写纯原创文案。

我之前真的遇到过这样的同学，他们非常有底气地表示要自己创作。我支持这种想法，因为纯原创的文案相比借用素材，其视频的播放量收益单价通常会高出两倍。

![](img/5c088d6260972fa0cbdf9aa33b8ead2b_1.png)

这是纯原创文案的显著优势。我自己开设了一门专门的文案课程。

![](img/5c088d6260972fa0cbdf9aa33b8ead2b_1.png)

包括我自己或我的小伙伴撰写文案，其方法和技巧都源于这门课程。很多同学害怕自己写文案，觉得这是一项非常专业的工作，像写作文一样困难。

![](img/5c088d6260972fa0cbdf9aa33b8ead2b_3.png)

其实并非如此。对于想写却又害怕写文案的同学，请记住这句话：**我们的解说文案，说人话就可以了**。

“说人话”的意思，就是使用大白话或口水话。下面我将举例说明。

![](img/5c088d6260972fa0cbdf9aa33b8ead2b_3.png)

例如，当我们向父母或朋友推荐一部电视剧，希望他们也去看时，我们不会像写文章一样有完整的开头、过程和结尾。

我们不会这样说：“妈妈，我今天看了一部电视剧，我觉得很好看。这部剧名叫《XXX》，由某某导演执导，于某年上映。男主角叫A，女主角叫B。第一集讲述了……第二集讲述了……”

你一定会这样说：“妈妈，我今天看了一部电视剧，真的特别好看！那个男主角超级帅，女主角也超级漂亮，他们好般配。剧情是这两个人相爱十年，马上就要结婚了，结果男的婚前出轨。女的知道后去捉奸，找到后扇了男的一巴掌说：‘离婚吧，我不跟你结婚了！’”

这就是我们俗称的“口水话”。解说文案不需要像写作文那样刻板。如果你做得过于复杂，观众为什么还要看你的影视解说呢？他们直接去看原片就好了。

一个典型的例子是《甄嬛传》。这部剧很好看，但我有时间看第一集，没时间看第二集，等到第三集时可能就跟不上剧情了。

当时有很多人做《甄嬛传》的影视解说，并且绝对能赚钱。原剧中有不少文言文对白，不是所有人都能完全看懂。而做解说的人，就是用大白话告诉观众，这一集到底发生了什么事。

在我的团队中，做视频解说的人是最多的。

为什么？就因为影视解说或其他类型的解说视频是最赚钱的领域之一。你们可以看到，做解说的人真的特别多。

有纪录片解说、综艺解说、电视剧解说、动物解说、科普解说、历史解说、动漫解说等等，种类非常多。

大家都愿意做解说，就是因为它并不难，且收益可观。在此也分享一个方向：如果你现在想做解说视频，影视推广是一个选择，做影视解说能起到版权推广的作用。

![](img/5c088d6260972fa0cbdf9aa33b8ead2b_5.png)

如果不想做影视领域，我建议可以考虑动物解说。因为目前在平台上，做动物解说的人相对较少。

![](img/5c088d6260972fa0cbdf9aa33b8ead2b_5.png)

![](img/5c088d6260972fa0cbdf9aa33b8ead2b_7.png)

因此，平台给予的扶持流量是最庞大的。

![](img/5c088d6260972fa0cbdf9aa33b8ead2b_7.png)

本节课中我们一起学习了原创解说文案的创作思路。我们了解了除了使用现成素材库，还可以利用原创文库或自己撰写。核心在于掌握“说人话”的原则，用大白话将故事讲清楚、讲有趣。同时，我们也看到了动物解说等新兴领域的流量潜力。记住，好的解说文案是连接视频与观众、提升收益的关键。
