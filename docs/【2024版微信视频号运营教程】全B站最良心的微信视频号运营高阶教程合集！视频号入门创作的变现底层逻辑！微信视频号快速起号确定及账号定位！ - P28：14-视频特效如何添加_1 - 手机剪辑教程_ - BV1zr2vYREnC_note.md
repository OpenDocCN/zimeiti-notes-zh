# 手机剪辑教程：P28：视频特效如何添加

在本节课中，我们将学习如何在视频剪辑中添加特效。特效能够为视频增添氛围和趣味性，但使用不当也可能影响观感。我们将重点介绍如何正确地为视频主轨道和画中画轨道添加特效，并明确其作用对象。

上一节我们介绍了画中画功能，本节中我们来看看如何为视频添加特效。

## 添加画面特效

在剪辑界面，点击“画中画”功能右侧的“特效”按钮，即可进入特效面板。特效主要分为“画面特效”和“人物特效”两大类。

以下是添加画面特效到主视频的步骤：
1.  点击“画面特效”。
2.  在特效列表中选择一个合适的特效，例如“星星灯”或“星星坠落”。
3.  点击特效将其添加到时间轴。
4.  调整特效的持续时间和参数，例如将**透明度**适当降低，以避免效果过于强烈。
5.  点击对勾确认。

此时，该特效会作用于整个主视频画面。选择像星星这类比较简约的特效，通常不会让主画面显得杂乱。

## 设置特效的作用对象

并非所有特效都适合直接添加到主视频。有时我们需要让特效只作用于画中画图层，以突出主体或创造对比效果。

以下是设置特效作用对象的步骤：
1.  添加一个特效（例如“模糊”特效）到时间轴。
2.  如果发现主视频整体变模糊，说明作用对象不正确。
3.  点击时间轴上的该特效片段，在参数面板中找到“作用对象”选项。
4.  将作用对象从“主视频”更改为“画中画”。
5.  点击对勾确认。

更改后，“模糊”效果将仅应用于画中画视频，而主视频保持清晰。这同样适用于“人物特效”，例如“闪白”效果，将其作用对象设置为画中画，可以避免主视频人物画面出现异常闪烁。

## 给新手的建议

对于剪辑新手，在添加特效时需格外谨慎。复杂的特效若使用不当，容易破坏视频的整体感。

以下是给初学者的核心建议：
*   **慎用主视频特效**：除非确定效果合适，否则尽量避免直接将特效添加到主视频轨道。
*   **明确作用对象**：添加特效后，务必检查并设置其“作用对象”。通常，将特效作用于“画中画”是更安全的选择，可以确保主画面不受干扰。
*   **简化使用**：初期可以只尝试“画面特效”和“人物特效”中的基础选项，并将它们的作用对象都设置为画中画。
*   **谨慎使用氛围特效**：特效分类中的“气氛”类特效（如爱心、彩带、烟花）虽然能营造氛围，但需要较强的节奏感和审美来搭配。若不熟悉，建议暂时不要使用，以免影响视频质量。

本节课中我们一起学习了视频特效的添加方法。关键点在于理解特效的“作用对象”概念，并学会将其精准应用到主视频或画中画轨道上。记住，对于初学者而言，少用、精用特效，并确保主画面清晰不受干扰，是提升视频观感的重要原则。
