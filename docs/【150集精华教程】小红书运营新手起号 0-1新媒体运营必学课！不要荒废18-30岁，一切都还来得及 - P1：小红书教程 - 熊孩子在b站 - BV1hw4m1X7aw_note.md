## 开店 📝

上一节我们介绍了课程的整体框架，本节中我们来看看如何开设小红书店铺。开店是启动小红书电商的第一步，流程非常简单。

以下是开店的具体步骤：
1.  打开小红书应用。
2.  点击左上角的菜单图标。
3.  选择“创作中心”。
4.  点击“更多服务”选项。
5.  选择“店铺”，点击“立即入住”。
6.  按照官方要求提交资料完成入驻。

![](img/239112f7a07fa91d2dc241f0897b6911_5.png)

小红书店铺主要分为个人店和企业店两种形式。

以下是店铺类型说明：
*   **个人店**：分为普通个人店和个体工商户店。
    *   **普通个人店**：仅需身份证和1000元保证金即可开通。
    *   **个体工商户店**：除身份证和保证金外，还需提供营业执照。
*   **企业店**：包括普通企业店、专卖店和旗舰店。均需营业执照，并需缴纳600元年费。一份营业执照可开设三家店铺。

对于新手而言，直接选择个人店即可。实测表明，平台并未对企业店有流量倾斜。初期不建议开设多个店铺（店群），因为小红书的电商自然流量占比低，铺货效果有限。核心应放在内容上。

开店成本主要包括以下几项：
*   **保证金**：1000元（不做可退）。
*   **平台扣点**：为营业额的5%。
*   **支付渠道费**：约为每单的0.7%。
*   **流动资金**：初期可采用无货源模式，准备2000-3000元即可启动。

## 选品 🛍️

成功开店后，下一步就是决定卖什么。选品逻辑的核心是利用市场数据，而非主观猜测。

以下是三个高效的选品渠道：
1.  **某东、某宝**：寻找符合小红书平台调性的爆款产品。已有案例显示，将其他平台的爆款直接搬运至小红书，首篇笔记即可出单。
2.  **某音精选联盟**：查看其排行榜，筛选出符合小红书用户调性的热卖商品，直接搬运测试。
3.  **某多多销量排行榜**：选择适合小红书的类目排行榜，从中挑选爆品进行搬运。

整个选品逻辑可以概括为：**借助成熟平台大卖家的选品能力，来弥补自身不足**。我们只需判断排行榜上的爆品是否可能匹配小红书用户画像，如果匹配，就搬运过来测试。这是最快捷且稳妥的选品方式。

## 获取流量 📈

上一节我们介绍了如何借助市场数据选品，本节中我们来看看如何为商品获取流量。流量是小红书电商成功的关键，其核心来源于笔记内容。

小红书是一个种草平台，店铺超过九成的流量都来自发布的笔记。因此，不应追求店铺数量（店群），而应建立**账号矩阵**，通过尽可能多地发布优质笔记来获取流量。

用户路径是：**看到笔记 -> 对产品产生兴趣 -> 进入店铺完成购买**。发一篇优质笔记的效果远优于开设多个无人问津的店铺。

![](img/239112f7a07fa91d2dc241f0897b6911_9.png)

发笔记的关键在于避免硬广。小红书用户认知度高，硬广难以获得信任和转化。必须以**素人（真实用户）的视角**进行内容创作，站在用户角度做种草分享。

以下是制作种草笔记的简单方法：
1.  前往各大电商平台（如某音、某宝、某东）的目标商品评论区。
2.  寻找高质量的买家秀图片或视频作为素材。
3.  基于买家的真实评价修改和优化文案。

这些买家评价本身就是用户的使用体验，天然适合用作种草内容，且素材取之不尽。

![](img/239112f7a07fa91d2dc241f0897b6911_11.png)

当单个账号的运营模式（SOP）跑通后，即可通过**批量运营多个小红书账号、批量发布笔记**的方式，为店铺进行矩阵式引流，放大经营效果。
