## 直播规划的系统性步骤

上一节我们介绍了直播的重要性，本节中我们来看看如何系统性地规划一场直播。一个完整的直播规划应包含以下步骤。

首先，你需要确认你的直播模式。直播模式主要分为以下几种：

*   **达人直播**：邀请探店效果好的达人为你的机构进行直播。
*   **品牌自播**：由机构自己的团队进行直播。
*   **服务商代播**：委托专业的服务商团队代理直播。

无论选择哪种形式，后续的步骤都必须执行到位。

以下是直播规划的核心步骤列表：

1.  **明确投放预算**：使用**巨量本地推**进行广告投放。预算规划需分步进行：**预算拆解 -> 单场预算投放计划搭建 -> 确认IOI（投入产出比）目标**。例如，设定目标为投入1000元，产出5:1或10:1的ROI，以便后续生意复盘。
2.  **组建自播团队**：确保各个工种到位，分工明确。我们将在下一部分详细讲解团队构成。
3.  **搭建硬件场景**：本地生活类直播非常依赖直播间的场景布置，需提前准备。
4.  **确立主播人设**：筛选并确定达人主播或自有主播的人设风格。
5.  **准备预热内容**：如果是品牌自播，需要提前制作几条预热视频内容；达人直播则通常由达人负责。
6.  **进行直播测流**：本地生活直播的测流相对简单，主要测试不同时间节点投放流量的效果差异。
7.  **编写直播脚本**：医疗健康类直播间的脚本尤为重要，我们将在后续课程中重点讲解脚本的搭建方法。

## 团队构建与人员配置

了解了规划步骤后，我们来看看团队应如何在不同阶段进行构建。团队的人员配置与能力模块匹配度，直接决定了直播间内容的可玩性和最终效果。

商家在不同发展阶段，团队构建和直播流程有所不同：

*   **新手期（3个月内）**：团队设置可以简化，一人可兼任多职。例如，编导可兼任摄影，场控可同时承担中控及其他助理工作。
*   **发展期（3个月以上）**：拥有固定团队和基础粉丝后，需逐步完善团队分工。
*   **成熟期**：团队应高度专业化，各工种职责明确分离。

在配置直播间人员时，应尽可能配齐所有关键能力模块的人员。一个配置完整的团队通常包括以下角色：

*   **直播项目经理（PM）**：负责前期策划、直播中协调、后期复盘。
*   **视觉/中控**：负责直播间画面、贴片、商品上下架、数据监控等。
*   **直播运营**：负责流量投放、互动管理、节奏把控。
*   **主播**：直播内容的核心输出者。
*   **助播**：辅助主播，负责补充讲解、引导互动、处理突发情况。

你也可以按功能将团队分为**画面组、主播组、流量组和视频组**，每组至少需要一人，因此一个最小化的有效团队也至少需要四人协作。

一个成功的直播间绝非单纯靠资金投入，而是依赖团队配合，并始终围绕明确的**IOI目标**展开。通过每日或每场直播后的复盘，可以不断迭代优化，筛选出更高效的团队和更优的执行方案。

## 核心执行要点与总结

决定做直播后，无论是自己操盘还是寻找外部团队，都需要掌握核心要点。

如果你作为操盘手或机构老板，必须清楚每个环节的问题所在，而非做“甩手掌柜”。即使选择外包给专业团队，你也需要懂得其中的配置逻辑，明确每一分钱的去向。在直播投放和人员投入上，方法错误会导致机会成本的巨大损失。

因此，我们主张的打法是：**先用正确的方法做正确的事，拿到初步结果后，再进行复制和放大。**
