## 概述
在本节课中，我们将学习如何通过“二创”的方式制作书单视频。核心思路是：获取一个现有的视频，提取其文案，替换部分画面素材，并重新配音，从而生成一个属于你自己的原创视频。我们将使用剪映和配音神器等工具，一步步完成这个流程。

## 视频素材的获取与处理
上一节我们介绍了二创的基本概念，本节中我们来看看具体的操作步骤。首先，我们需要找到一个合适的原始视频作为基础。

以下是获取和处理原始视频的步骤：
1.  在抖音等平台搜索目标书籍（例如“走进大学城”），并选择“商品”按销量排序，找到相关的带货视频。
2.  进入达人主页，寻找合适的视频素材。例如，一个关于“汕头大学”的视频，时长1分58秒。
3.  复制该视频链接，使用去水印工具去除视频平台的水印。
4.  将去水印后的视频导入剪映。

![](img/b9474656a79e6ae94530199a709c2c3a_13.png)

## 文案的提取与配音
获取视频后，我们需要将其中的文案提取出来，并生成新的配音。

以下是提取文案和生成配音的步骤：
1.  在剪映中，将视频拖入时间轴。如果视频前后声音不一致（例如旁白和人物原声混合），可以在声音变化的时间点（如53秒处）将视频**分割**。
2.  删除不需要的后半部分，只保留含有目标文案的前半部分视频，并将其导出。
3.  打开“配音神器”APP，使用“视频提取文案”功能，导入上一步导出的视频，将文案提取为文字。
4.  在配音神器中选择一个合适的声音（如成熟稳重的女声），将文案转换为新的音频文件并导出。

## 视频画面的替换与二创
现在，我们有了新的配音，接下来需要对原始视频的画面进行“二创”处理，即替换部分素材，使其更具原创性。

以下是替换视频画面的操作步骤：
1.  回到剪映中的原始视频工程。首先，将原视频的音量调为0，或直接删除原音频轨道。
2.  将上一步生成的新配音音频导入剪映，并拖入时间轴。
3.  **核心操作**：根据新配音的内容，逐段替换原始视频的画面。例如：
    *   原视频有一段3秒的“毕业典礼”画面。我们可以在抖音搜索“大学生毕业典礼”视频，下载后替换掉这3秒。
    *   在剪映中，**分割**出要替换的3秒片段，删除它，然后导入新找到的3秒素材放入该位置。
    *   在新旧素材之间添加一个**叠化**转场，使过渡更自然。
    *   务必关闭新增素材的原声。
4.  重复此过程，替换1-2个核心画面（如人物特写、特定场景）。替换的原则是：新素材的格调和含义与原片段大致相符。
5.  如果新配音的时长与视频画面总时长不匹配，可以使用**变速**功能（常规变速）调整视频片段的播放速度，或者通过增删素材来对齐音频与视频。

## 成片调整与导出
所有素材替换完成后，我们需要进行最后的调整，让视频看起来更完整、专业。

以下是最终的调整与导出步骤：
1.  **添加背景音乐**：导入一段合适的、无版权的背景音乐，音量调至低于配音音量。
2.  **生成与遮盖字幕**：
    *   使用剪映的“识别字幕”功能，为新的配音自动生成字幕。
    *   通过调整字幕的**样式**（如背景、字体颜色），确保新字幕能够完全遮盖住原始视频中可能残留的旧字幕。
3.  **检查水印**：确保所有使用的素材都没有明显的其他平台水印。如有水印，可通过**放大**画面或再次去水印来处理。
4.  **导出设置**：导出视频时，选择分辨率为**1080P**，帧率为**60帧**，码率选择推荐值，这样能保证视频的清晰度和流畅度。

## 总结
本节课我们一起学习了书单视频二创的全过程。关键点在于：**获取原始视频 -> 提取文案并重新配音 -> 替换关键画面素材 -> 添加背景音乐和字幕 -> 最终导出**。这个过程的核心是**素材的替换与重组**，只要替换足够的元素，就能有效提升视频的原创度。

请记住，剪辑技能需要练习。从处理一个视频需要一两个小时到十分钟完成，靠的是**熟能生巧**。建议你找到10个左右的视频，按照这个流程反复实操，你的剪辑效率和视频质量一定会快速提升。
