# 微信视频号运营教程：P40：如何扒取直播间话术

在本节课中，我们将学习一个实用的技巧：如何快速获取并借鉴其他成功直播间的话术。这对于想要快速上手直播带货或优化自己直播内容的初学者来说，是一个高效的学习方法。

## 概述与工具准备

上一节我们介绍了视频内容与选品的思路，本节中我们来看看如何学习优秀主播的表达方式。核心在于使用工具录制并转写直播内容。

需要用到的工具是**飞书**。具体操作步骤如下：

![](img/3a6946660d8803dc7793796ed01589d0_1.png)

1.  下载并打开飞书应用。
2.  进入应用后，点击底部导航栏的 **`工作台`**。
3.  在工作台页面中，找到并点击 **`飞书妙记`** 功能。

![](img/3a6946660d8803dc7793796ed01589d0_3.png)

## 录制直播话术

以下是使用飞书妙记录制直播的具体步骤：

1.  **启动录制**：在飞书妙记界面，点击右下角的 **`语音`** 按钮，即可开始录音。
2.  **打开目标直播间**：此时，你可以打开微信视频号或其他平台中，你想要学习和模仿的直播间。
3.  **实时录制**：飞书妙记会实时录制直播间里的所有声音，包括主播的介绍、互动和逼单话术。建议录制**15分钟**左右，这足以让你了解主播介绍产品、营造氛围和促进成交的核心逻辑。

## 导出与整理话术

录制完成后，你需要将音频内容转换为可编辑的文本。

1.  **结束并处理**：录制足够内容后，在飞书妙记中点击 **`完成`**。
2.  **生成文本**：系统会自动将录音转为文字，这个过程可能需要几分钟。
3.  **导出文本**：
    *   转写完成后，点击进入该条记录。
    *   点击 **`分享`** 按钮，选择 **`可编辑`** 或 **`可阅读`** 模式。
    *   复制生成的链接。
4.  **提取话术**：将链接发送到微信并打开，即可看到整理好的直播文字内容。你可以复制全部文本。

## 形成个人话术库

获取原始文本后，关键在于将其转化为你自己的东西。

1.  **提炼核心**：仔细阅读转写出来的文本，提炼出其中的产品介绍框架、互动问答模式和促单技巧。
2.  **编辑优化**：将提炼出的核心话术，结合你自己的产品和语言习惯进行修改和优化。
3.  **备用与练习**：将最终版的话术写在纸上或存入手机备忘录。开播前可以反复练习，直播时如果忘记，也可以作为提词器参考使用。

## 总结

![](img/3a6946660d8803dc7793796ed01589d0_5.png)

本节课中我们一起学习了如何借助“飞书妙记”工具，高效地扒取和学习其他直播间的话术。这个方法的核心步骤是：**录制 -> 转写 -> 提炼 -> 优化**。通过借鉴优秀同行，你可以快速搭建自己的直播话术体系，降低起号难度。记住，学习的目的是为了创新，最终形成具有个人特色的直播风格。
