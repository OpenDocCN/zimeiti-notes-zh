# 自媒体运营教程：P11：8.影视推广平台 🎬

在本节课中，我们将要学习如何通过“影视推广”平台，在抖音、快手等平台发布视频并结算收益。我们将了解影视推广的含义、操作流程以及不同平台的收益计算规则。

---

上一节我们介绍了视频的多平台分发，本节中我们来看看如何在抖音、快手等平台通过特定任务获得收益。

很多同学问到，视频如果发到抖音或快手上面，该如何结算收益。需要明确的是，视频发到除抖音、快手外的30多个自媒体平台，可以依靠**播放量**直接结算收益。但抖音和快手这两个平台有所不同。

就像之前提到的，视频发到抖音和快手，主要作用是**增长粉丝**。有没有同学只想做抖音和快手这两个熟悉的平台，同时又想依靠播放量结算收益呢？答案是肯定的。

所有制作好的视频，可以发布到以下四个平台：
*   抖音
*   快手
*   小红书
*   微信视频号

![](img/4e147bf19048ae84e18ad6f0bf066044_1.png)

这几个平台大家都很熟悉。小红书可能更多年轻用户在使用，而微信视频号则是微信内置的功能。在这几个平台发布后，我们需要通过“影视推广”平台来接取任务，并通过完成任务来结算奖金。

![](img/4e147bf19048ae84e18ad6f0bf066044_2.png)

以下是影视推广的含义与操作流程介绍。

影视推广是指，电视剧、电影或综艺节目需要进行宣传。许多片方会通过影视推广平台发布任务，让创作者制作视频帮助推广，并根据视频的播放量等数据结算奖金。

素材库中的任务非常多，例如爱奇艺、搜狐等平台的任务，列表中都是具体的影视剧名称。

![](img/4e147bf19048ae84e18ad6f0bf066044_1.png)

![](img/4e147bf19048ae84e18ad6f0bf066044_2.png)

操作时，每完成一个电视剧的推广视频，都需要提交任务回填链接。需要填写以下信息：
*   微信号、账号昵称或抖音/快手ID
*   所做推广的具体电视剧名称
*   作品发布后的链接
*   作品发布时间
*   视频播放量的截图

提交完整的回填链接后，平台会据此结算收益。

影视推广的收益通常比单纯靠播放量结算的收益更高。目前短剧推广的任务非常多。许多传媒公司和网红会自制短剧，并发布任务让创作者帮忙宣传，发布到抖音、小红书、视频号等平台。

收益根据播放量要求计算，并且**对账号粉丝数没有要求**。结算时，通常取一部剧最高播放量的三个作品，其播放量总和作为计算依据。

以下是不同平台的收益计算规则。

*   **抖音**：每1万播放量，奖金为50元。
    *   `收益 = (播放量 / 10000) * 50元`
*   **微信视频号**：每1万播放量，奖金为10元。
    *   `收益 = (播放量 / 10000) * 10元`
*   **小红书**：每1000播放量，奖金为30元。
    *   `收益 = (播放量 / 1000) * 30元`

除此之外，像搜狐短剧等任务，可能需要同时在抖音和快手发布。其结算规则依据**点赞数**计算，同样没有粉丝要求。

*   **抖音**：单条作品5000赞奖励100元，1万赞奖励200元。
    *   `收益（5000赞）= 100元`
    *   `收益（10000赞）= 200元`
*   **快手**：单条作品5000赞奖励100元，1万赞奖励180元。
    *   `收益（5000赞）= 100元`
    *   `收益（10000赞）= 180元`

其中，搜狐短剧任务给出的单价相对较高。

本节课中我们一起学习了影视推广的运作模式。关键点在于：通过接取推广任务，在抖音、快手、小红书、视频号发布视频，并根据各平台特定的规则（播放量或点赞数）结算奖金，且对账号粉丝量无门槛要求。这为想在主流平台获取收益的创作者提供了一条有效路径。
