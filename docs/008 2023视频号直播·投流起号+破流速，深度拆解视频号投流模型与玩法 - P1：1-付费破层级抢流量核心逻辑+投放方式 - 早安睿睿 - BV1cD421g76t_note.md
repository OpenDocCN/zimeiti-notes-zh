## 概述：付费破流速的核心思想

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_9.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_11.png)

上一节我们介绍了投流起号的基础框架，本节中我们来看看如何利用付费投流来“破层级、抢流量”。首先需要明确一个核心观点：**破层级的关键在于破流速**，即提升直播间每分钟的进入人数。付费投流的核心作用是加速自然流量的学习过程，并为直播间提供稳定的流量补充，但其最终效果仍取决于“人货场”的承接能力。

付费投流无法直接放大自然流速，如果直播间承接能力不足，即使投入付费流量，自然流也难以增长。层级上不去的根本原因在于“人货场”对自然流的承接能力不足，导致流量被承接能力更强的直播间抢走。

### 第一天投放策略调整

原计划中，在开播后20-30分钟补充2小时互动计划。现调整如下：

*   **保留部分**：开播前预约投放的“商点”和“涨粉”2小时计划保持不变。
*   **调整部分**：在开播后30-40分钟，**同时**进行以下操作：
    1.  补充**长时间的高漏斗计划**（例如互动或成交目标），采用**控成本出价**。
    2.  新建一条**长时间计划**（如12小时），同样采用控成本出价，但出价可适当调高（近期24小时计划可能跑不动）。

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_22.png)

这个新增的长时间计划，核心目的是**在本场其他短时间计划结束后，用于补充流速**，防止自然流突然断崖式下跌，保证直播间能持续开播，为自然流提供充分的学习时间。

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_24.png)

**案例演示**：我们昨晚测试的新号，在两条短计划（商点、涨粉）结束后，在线人数迅速下降。随后我们通过投放长时间的互动和成交计划进行“打底”，稳住了较低的流速，并在此基础上叠加了三条互动计划，成功将代表互动人数的“黄线”拉开，这意味着进入直播间的人群是精准且有停留意愿的。

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_26.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_27.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_29.png)

> **核心公式**：**稳定流速 = 短时间爆量计划 + 长时间打底计划**

### 第二天投放策略调整

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_40.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_42.png)

第二天的策略也需要进行加固，以应对可能出现的流量不稳问题。

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_44.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_45.png)

*   **开播阶段**：可以不进行预约投放，让自然流先进行学习。
*   **开播后20-30分钟**：开始投放计划。
*   **计划调整**：如果发现按最低出价计划跑不动，可以**提高出价进行叠投**来补充流速。
*   **新增环节**：在开播后40分钟左右，可以增加一个**长时间、控成本的“高漏斗计划”**作为全场打底计划，全程不关。这能进一步保障后段的流量安全。

## 付费破流速的适用场景与投放逻辑 🧠

了解策略调整后，我们来看看什么情况下适合使用付费来破流速。

### 何时使用付费破流速？

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_60.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_62.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_63.png)

1.  **流量被抢走时**：当发现自然流量被竞品或同人群直播间吸走时，可以用付费抢回一部分流量。
2.  **人货场已到位，但流速未破时**：这是最理想的情况。直播间本身具备承接能力，只是系统还未学习出来，付费投流可以加速这个过程，实现销量的爆发。
3.  **ROI（投产比）能打平时**：如果付费投放的ROI能够覆盖成本，那么即使自然流没放大，也可以直接以付费为主进行投放。
4.  **高复购产品**：通过持续付费积累粉丝和看播用户，回流人群会像滚雪球一样增大，自然流速也会被叠加放大。

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_65.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_67.png)

### 付费投放的核心逻辑

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_69.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_71.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_73.png)

付费流量是围绕直播间**实时在线人群**进行学习并找相似人群的。如果自然流本身精准，付费流会学得更准；如果自然流泛了，付费流也会跑偏。因此，**控制付费流量的进入比例至关重要**。

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_75.png)

一个重要的经验法则是：**付费流速应尽量控制在整体进入流速的40%以下**。当付费流速占比低于40%时，付费流可以有效地矫正和放大自然流，且投放成本较低。一旦高于这个比例，就有压制甚至打乱自然流人群模型的风险。

> **核心逻辑**：**付费流量占比 < 40%**，有助于自然流学习；**付费流量占比 > 40%**，可能压制自然流。

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_77.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_79.png)

## 投放实战：从测到投的完整流程 🛠️

上一节我们理解了底层逻辑，本节中我们来看看具体的操作流程。付费投放不是一成不变的模板，需要根据直播间特性进行测试和调整。

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_81.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_83.png)

### 第一步：测试最优投放目标

每个直播间的最佳投放目标（互动、涨粉、商点、成交、观众）可能不同，这取决于主播话术和客户行为模式。高客单直播间可能前期商点更准，后期转向成交；有的直播间则互动或涨粉更有效。

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_85.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_86.png)

**测试方法**：
为每个目标单独建立计划，拉满12-24小时进行慢速测试。通过对比“商品曝光率”和“商品点击率”，找到你的“双高计划”（即高漏斗计划）。

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_88.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_90.png)

**一个重要的评估指标**：
在投流后台的自选指标中，添加“**超过1分钟的观看人数**”。计算“花费/超过1分钟观看人数”，这个值可以近似看作获取一个潜在意向客户的成本，比千次曝光成本等指标更具参考意义。

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_92.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_94.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_95.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_96.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_98.png)

### 第二步：测试高漏斗兴趣定向

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_100.png)

确定最优投放目标（例如“互动”）后，用它来测试不同的兴趣定向。

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_102.png)

**测试方法**：
1.  一个计划只圈选1-2个三级类目兴趣。
2.  采用**长时间（12-24小时）、低出价、控成本**的“微付费”方式投放。
3.  目的是用最低成本、最小干扰，测出哪个兴趣定向带来的用户漏斗数据（停留、点击、转化）最好。

**兴趣定向的精度**：
私域人群 > 对标直播间 > 兴趣标签。优先投放私域粉丝；其次，对标直播间（尤其是爆款相似的对标）能快速找到人群；最后，通过兴趣标签进一步收缩和精准化人群。注意，圈选越细，人群越精准，但可能跑量能力越弱。

### 第三步：制定叠投策略

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_104.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_106.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_108.png)

测出高漏斗目标和兴趣后，就可以进行叠投放大。叠投的关键是**避免相同目标计划相互压制**。

**新号期四种基础叠投组合**：
以下是四种以不同目标组合的投放节奏，可根据直播间情况选择或融合使用。

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_110.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_111.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_113.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_115.png)

1.  **微付费持续矫正型**：全部采用长时间计划，通过出价从低到高来调整流速。
    *   低价长时间 -> 中价长时间 -> 高价长时间
2.  **持续增量放大流速型**：针对测出的优质计划，不断加价、缩短时间，快速抢量。
    *   低价长时间 -> 中价中时间 -> 高价短时间
3.  **持续稳定全场流速型**：先保证全场有量，再逐步调节，适合流速需要高度稳定的直播间。
    *   高价长时间 -> 中价中时间 -> 低价短时间
4.  **低价全场辅助矫正型**：全场以低价计划为主，单纯用于辅助自然流矫正，不追求付费流速。
    *   低价长时间 -> 低价中时间 -> 低价短时间

**叠投时间间隔**：
相同目标的计划，投放时间建议间隔**15-20分钟**，给每条计划留出启动和学习的时间，避免相互竞争。

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_119.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_121.png)

### 成熟期账号投放策略

对于已有稳定老粉的成熟账号，策略有所不同：

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_125.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_127.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_129.png)

*   **开播即投**：可以直接预约投放，利用老粉稳在线。
*   **首小时爆破**：开播第一个小时，直接投放一条**2-3小时的高价、高漏斗计划**，利用老粉的互动氛围承接住这波大流量，往往能打爆第一个小时的销售额。
*   **计划衔接**：在每条2-3小时计划结束前**15-20分钟**，补一条新的计划，保证流速不间断。
*   **尝试通投**：如果直播间人群模型非常精准稳定，可以尝试长时间的通投计划（只卡年龄性别），让系统自己去学习，有时ROI也会不错。

### 关键注意事项

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_139.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_141.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_143.png)

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_145.png)

1.  **付费流量不一定精准**：它的精准度依赖于直播间实时在线人群的模型。自然流不准，付费流也很难准。
2.  **监控与关停**：必须实时监控计划数据。对于**高进入率、低漏斗数据**（如只有人进来，没人停留点击）的计划，要果断关停，否则会打乱人群。
3.  **投放工具**：尽量使用**网页版**进行投流，功能更全，且可以手动出价。避免只用直播手机投流，不便于实时调控。
4.  **日播的重要性**：账号坚持日播，有利于培养用户观看习惯，稳定老粉回流，提升投流后的长期回流效率。

### 常见问题解答

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_151.png)

*   **计划跑不动怎么办？**
    *   依次尝试：提高出价 -> 缩短计划时长 -> 提高预算。优先提价。
*   **人群圈太窄跑不动怎么办？**
    *   这是正常现象。可以尝试叠投多条计划，或者适当放宽定向条件。
*   **付费占比过高导致自然流被压制怎么办？**
    *   立即关停或减少付费计划，让自然流重新学习。重点优化人货场承接能力。
*   **如何判断该投哪个目标？**
    *   没有固定答案。必须通过长时间微付费测试，对比不同目标计划下的“超过1分钟观看人数成本”和漏斗数据来决定。

## 总结与实战预告 🚀

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_167.png)

本节课我们一起深入学习了付费投流破层级的核心逻辑和详细操作方法。我们首先根据近期环境调整了起号策略，强调了**用长时间计划补充流速**的重要性。然后，我们明确了付费破流速的关键在于**提升人货场承接能力**，并需将**付费流速占比控制在40%以下**。最后，我们梳理了从**测试最优目标**到**测试兴趣定向**，再到**制定叠投策略**的完整流程，并介绍了新号与成熟号的不同打法。

记住，投流不是机械执行模板，而是**基于数据测试和直播间状态的动态调整过程**。核心思想始终是：**用精准的付费流量，与自然流进行交集学习，互相矫正，最终实现流速和层级的突破。**

![](img/c6fb8c158f5e885e48e94cf1ee8127cf_169.png)
