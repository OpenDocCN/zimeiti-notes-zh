# 新媒体运营系统课：P27：古诗词制作成视频 📜

在本节课中，我们将学习如何将一首古诗词制作成精美的视频。这是一种非常适合不想露脸的自媒体创作者的创作方式，能够稳定地获取收益。整个过程将分为几个清晰的步骤，从获取诗词文本到最终导出成片，我们会手把手带你完成。

![](img/e86f94e6aa0fbdeb9ac3069dc3e4c999_1.png)

## 第一步：获取并提取古诗词文本

![](img/e86f94e6aa0fbdeb9ac3069dc3e4c999_3.png)

首先，我们需要找到一首古诗词。你可以从网上截图，或者直接对书籍进行拍照保存。

以下是提取图片中文字的具体操作：

1.  将含有古诗词的图片发送给任意一位聊天好友。
2.  在聊天窗口中点开这张图片，用手指长按图片。
3.  在弹出的菜单中，选择“提取文字”功能。
4.  系统识别完成后，全选识别出的文字，点击“复制”按钮。

这样，我们就成功将图片中的古诗词转换成了可编辑的文本。

## 第二步：生成诗词朗读音频

上一节我们获取了文本，本节中我们来看看如何为它配上朗读声音。我们将使用剪辑软件来生成音频。

操作流程如下：

1.  打开剪辑软件（如剪映），点击“开始创作”。
2.  在素材库中选择一个**黑色背景**素材，点击“添加”。
3.  在底部工具栏找到“文本”工具，点击“新建文本”。
4.  将之前复制的古诗词文本粘贴到输入框中，点击确认。
5.  再次点击底部工具栏，向左滑动找到“文本朗读”功能。
6.  在声音库中选择一个你喜欢的声音（例如“小姐姐”）。
7.  点击声音选项旁边的三条横杠，可以调节语速。将红色圆点向左拖动至约 **0.8倍** 速，使朗读更清晰自然，然后点击确认。
8.  音频生成后，点击右上角的“导出”按钮，将这段带有朗读的视频保存备用。我们实际只需要其中的音频。

## 第三步：添加背景与同步音频

现在，我们有了诗词的朗读音频。接下来，我们需要为它匹配一个古风背景画面，并让音频与画面同步。

请按以下步骤操作：

1.  再次点击“开始创作”。
2.  在“素材库”的搜索框中输入“**古风背景**”，选择一个合适的背景视频，添加至项目。
3.  关闭视频的原声。
4.  点击底部“音频”工具，选择“提取音乐”。
5.  选中上一步导出的视频文件，并勾选“仅导入视频的声音”，将朗读音频导入进来。
6.  将时间轴移动到音频结束的位置，选中背景视频，点击“分割”，删除后面多余的部分，使视频长度与音频一致。

## 第四步：添加与美化字幕

音频和背景已经对齐，接下来我们需要为朗读内容添加字幕，并进行美化。

以下是具体步骤：

1.  将时间轴移回视频开头，点击底部“文本”工具，选择“识别字幕”，然后点击“开始识别”。
2.  字幕识别出来后，点击任意字幕条，进入编辑界面。
3.  点击“样式”，可以更改字体（例如选择“毛笔行楷”）和颜色（例如粉色）。
4.  调整字幕的大小和位置。
5.  点击“动画”，为字幕添加入场效果。在“入场动画”中找到“**波浪弹入**”效果，并将动画时长设置为约 **2秒**。
6.  点击确认应用所有更改。

## 第五步：添加诗名、作者与装饰元素

字幕美化完成后，我们还需要添加诗名和作者信息，并用一些古风元素装饰视频，让整体更具韵味。

操作如下：

1.  点击底部“新建文本”，输入诗名和作者。
2.  为这段文本选择合适的字体（如“江湖体”）和颜色，并将其缩小后放置在视频上方合适位置。
3.  拖动文本轨道末尾，使其与主视频时长对齐。
4.  点击底部“贴纸”工具，在搜索框中输入“**古风**”。
5.  选择几个喜欢的古风贴纸（如花瓣、印章等）添加到视频中。
6.  调整每个贴纸的位置和大小，并同样拖动其轨道末端，使其持续时间与主视频一致。

## 第六步：添加背景音乐并导出

所有视觉元素都已就位，最后一步是为视频添加一首契合意境的背景音乐，并完成最终导出。

最终调整步骤如下：

1.  点击底部“音频”工具，选择“音乐”。
2.  在音乐库中挑选一首**纯音乐**或古风音乐，点击“使用”添加到时间轴。
3.  将时间轴移到视频结尾，选中音乐轨道，点击“分割”，删除多余的音乐部分。
4.  最后，点击右上角的“导出”按钮，等待渲染完成即可得到最终成片。

本节课中我们一起学习了将古诗词制作成视频的完整流程。我们首先提取了诗词文本并生成了朗读音频，然后为其匹配了古风背景并添加了精美的字幕和装饰元素，最后用背景音乐烘托氛围并导出成片。这种方法无需露脸，操作简单，是自媒体内容创作的一个有效方向。你可以尝试使用不同的诗词、背景和风格，创作出属于自己的独特作品。
