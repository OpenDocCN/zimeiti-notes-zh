# 小红书运营教程：P3-2：小红书平台检测规则详解 🕵️

在本节课中，我们将详细解析小红书平台的检测规则。了解这些规则是避免账号违规、确保内容安全发布的基础。我们将从平台检测的多个维度入手，并学习如何使用工具辅助自查。

## 平台检测的核心维度

小红书平台主要通过以下四个维度对账号和内容进行检测与管理。

### 1. 资料与内容检测

平台会系统性地检测用户的**个人资料**、发布的**笔记内容**、**私信**以及**评论**。

以下是这些区域明确禁止包含的信息：
*   **联系方式**：微信号、微信二维码、手机号。
*   **营销信息**：直接的代购信息、推广淘宝等第三方店铺的词汇。
*   **平台水印**：带有其他平台（如抖音、快手）标识的水印。

**需要注意的是**，平台规则相对宽松的地方是：个人资料中**允许填写邮箱**。此外，可以通过谐音或引导至微博等方式进行间接引流。

### 2. 设备与账号管理

平台会检测账号登录的**手机设备**。为了账号安全，避免被判定为营销号，建议遵循以下原则：
*   同一台移动设备最多只登录**3个**小红书账号。

### 3. 禁止诱导互动

在笔记内容中，**严格禁止**任何诱导用户互动的行为。这包括但不限于：
*   关注领奖
*   点赞抽奖

这类行为会被平台判定为**诱导互动**，违反社区规范。

### 4. 内容原创度保护

小红书平台对**内容原创度**的保护非常严格。在内容推广时，必须注意：
*   同一篇笔记**不要**用多个账号重复发布。
*   坚持**一篇文章对应一个账号**发布的原则。

## 敏感词检测与规避方法

除了上述规则，平台另一项核心检测是针对**敏感词**的。某些领域或词汇极易触发审核，例如：
*   **手机卡**
*   **金融**产品
*   **电子烟**及相关代购
*   **医疗用品**

在创作中，若不小心使用了未知的敏感词，可能导致笔记无法通过审核。因此，发布前进行敏感词自查至关重要。

### 如何使用检测工具

我们可以借助第三方工具进行敏感词检测。以下是两个实用的网站及其使用方法：

![](img/09c9e2bb71a88cbc317375f5c8b141d2_1.png)

**工具一：即时检测网站**

这个工具无需注册，使用便捷。
1.  将写好的笔记全文复制。
2.  粘贴到网站的输入框中。
3.  点击 **`立即检测`** 按钮。
4.  系统会高亮或列出文中所有的平台敏感词，例如“代购”。

![](img/09c9e2bb71a88cbc317375f5c8b141d2_3.png)

**工具二：需注册的检测平台**

这个平台功能类似，但需要先完成注册。
1.  注册并登录后，找到内容检测区域。
2.  将笔记内容粘贴进去。
3.  执行检测操作，平台会返回**过滤结果**，明确指出敏感词。

![](img/09c9e2bb71a88cbc317375f5c8b141d2_5.png)

**操作建议**：在发布任何笔记前，可以先将文案放入第一个工具中进行快速检测，这基本能满足日常的敏感词排查需求。
