# 微信视频号教程：P8：1. 剪映自学指南 📚

在本节课中，我们将学习如何从零开始自学剪映，这是制作微信视频号内容的核心剪辑工具。课程将引导你找到官方及优质的学习资源，帮助你快速掌握基础剪辑技能。

---

## 概述

![](img/8624e0faebdebdfe11f8f49a590b7a63_1.png)

![](img/8624e0faebdebdfe11f8f49a590b7a63_3.png)

剪辑是制作短视频的基本功。如今，许多人已具备一定剪辑基础。如果你从未使用过剪映，本教程将指导你如何高效自学。

![](img/8624e0faebdebdfe11f8f49a590b7a63_5.png)

## 获取与认识剪映

首先，你需要在手机应用商店下载“剪映”App。这是抖音官方的剪辑软件，功能强大且基本免费，足以满足大部分视频制作需求。其他软件如“必剪”与剪映操作类似，但剪映是目前使用最广泛的工具。

![](img/8624e0faebdebdfe11f8f49a590b7a63_7.png)

## 如何自学剪映基础

![](img/8624e0faebdebdfe11f8f49a590b7a63_9.png)

打开剪映App后，你可以通过内置的“创作者课堂”系统学习。以下是具体步骤：

![](img/8624e0faebdebdfe11f8f49a590b7a63_11.png)

1.  在剪映首页底部，找到并点击“创作者课堂”。
2.  在搜索框中输入“剪映小助手”进行搜索。
3.  在搜索结果中，选择“用户”标签页。
4.  第一个结果便是“剪映官方”账号，请关注此账号。
5.  进入官方账号主页，点击“课程”栏目。

![](img/8624e0faebdebdfe11f8f49a590b7a63_13.png)

这里推荐学习两个免费系列课程：
*   **《剪映功能大全》**：共87节课。
*   **《新手常见问题》**：共10节课。

![](img/8624e0faebdebdfe11f8f49a590b7a63_15.png)

![](img/8624e0faebdebdfe11f8f49a590b7a63_17.png)

这些课程每节时长很短，多为几十秒至两分钟。完整学习全部内容大约仅需两小时，即可让你从零基础达到熟练操作的水平。**务必先系统学完基础功能，再进行实际操作。**

## 拓展学习渠道

除了官方课程，抖音平台也是绝佳的学习资源库。你可以在抖音上搜索更多剪辑教学账号。

![](img/8624e0faebdebdfe11f8f49a590b7a63_19.png)

例如，搜索用户 **“妮妮6868618”**。该账号提供的“零基础学剪辑”系列视频较为简短精炼，涵盖全面，适合快速入门。

此外，直接在抖音搜索“剪映全套学习”或“零基础学剪辑”等关键词，也能找到大量优质教学视频。观看几个不同博主的教程，便能基本掌握剪映的核心功能。

本节课我们一起学习了如何自学剪映。核心方法是：首先通过剪映App内的“创作者课堂”学习官方系统课程，其次利用抖音平台搜索“妮妮6868618”等优质博主进行补充学习。剪辑并不难，掌握这些基础功能后，你已具备为微信视频号制作内容的能力。现在，就去开始你的学习与实践吧。
