## 🔍 VC与SC账号的核心区别

了解VC账号，首先要明白它与常见的SC（Seller Central）账号有何不同。这是决定运营策略的基础。

两者的核心区别主要体现在商业模式上：
*   **VC账号**：采用 **`邀请制`**，你是亚马逊的**供应商**，以批发形式将商品卖给亚马逊，由亚马逊负责定价和销售。
*   **SC账号**：采用 **`注册制`**，你是第三方**卖家**，在亚马逊平台上自行定价并销售商品。

这种根本区别导致了在运营权限、结算方式、物流责任等方面的巨大差异。
