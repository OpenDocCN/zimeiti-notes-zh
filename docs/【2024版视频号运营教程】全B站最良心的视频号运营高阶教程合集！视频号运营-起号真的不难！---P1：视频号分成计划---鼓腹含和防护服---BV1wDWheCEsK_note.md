# 视频号运营教程：P1：视频号分成计划详解 🎬

在本节课中，我们将要学习如何快速开通视频号分成计划，了解其核心规则，并掌握两种当前热门的、高收益的赛道操作方法。课程内容从账号注册、开通条件到具体运营策略，旨在为初学者提供一套清晰、可执行的行动指南。

![](img/62b15ef760bccc8a0f3bf0406ed9436d_1.png)

## 概述
视频号分成计划是视频号平台为创作者提供的一种变现方式。只要视频播放量足够高，就能带动视频下方评论区广告的曝光和点击，从而为创作者带来收益。本教程将详细拆解从零开始参与该计划的完整流程。

![](img/62b15ef760bccc8a0f3bf0406ed9436d_3.png)

## 视频号注册与基础 🆔
在开始分成计划之前，你需要拥有一个视频号账号。以下是注册视频号的基本步骤。

视频号注册流程相对简单，主要通过微信进行操作。具体步骤如下：
1.  打开微信，进入“发现”页。
2.  点击“视频号”入口。
3.  根据提示，完成视频号的创建和基础信息设置。

注册完成后，即可开始准备发布内容。

## 开通分成计划的核心条件 🔑
上一节我们介绍了账号注册，本节中我们来看看如何快速开通分成计划并领取收益。账号的启动阶段非常关键，若操作不当，将无法快速开通分成计划。

目前，开通视频号分成计划需要满足以下两个核心条件：
1.  **粉丝数达到100以上**。
2.  **内容符合平台规范**，即需要是优质原创内容，且画风精美。

## 快速过原创的实测方法 ⚡
在过去，新注册的视频号发布十条风景类视频就可能通过原创审核。但现在，仅发布风景视频已很难通过。这里分享一个本人实测有效的快速方法。

以下是具体操作步骤：
1.  **内容形式**：注册新号后，直接进行“怼脸拍”（即人物正面出镜讲话）。
2.  **视频时长**：每条视频需达到9秒以上。
3.  **发布频率**：每天发布5条此类视频。
4.  **开通时间**：持续发布两天（共10条视频）后，通常在第三天即可找到入口申请加入分成计划。

使用此方法，收益可能在前几天较低，但坚持发布，收益会逐步提升。下图展示了使用此方法后，收益从初期几十元到后期单日破千的增长情况。

## 分成计划的申请与关键设置 ⚙️
在满足条件并发布足够视频后，即可申请加入分成计划。以下是申请流程及一个至关重要的设置。

申请开通的流程如下：
1.  在视频号中找到“分成计划”申请入口。
2.  提交申请，等待审核通过。

**关键设置**：通过原创审核后，每次发布视频时，**务必勾选“原创声明”**。如果不开启此选项，即使视频获得千万播放，也无法产生任何收益。

![](img/62b15ef760bccc8a0f3bf0406ed9436d_5.png)

## 收益计算与提升策略 💰
腾讯官方并未给出非常明确的收益计算规则。根据运营经验，收益大致范围与提升方法如下。

**收益计算**：
*   大致范围：每万次播放量预计产生1~3元收益。
*   核心因素：收益具体高低**主要取决于广告的点击率**。即使播放量很高，若无人点击评论区广告，收益也不会理想。

**提升收益的核心策略**：
广告分成收入与视频质量、播放量、发布数量、广告行业选择等多方面因素有关。因此，制作原创视频时，**要想尽一切办法让视频的评论区活跃起来**，互动是驱动广告点击的关键。

## 推荐赛道一：人性语录类 🧠
目前视频号平台内容繁多，选择合适的赛道是成功的一半。以下介绍两种流量大、竞争相对较小的优质赛道。

人性语录领域是当前视频号上非常火爆的领域之一。它通过一句话或一段话，直指人性本质，容易引发30岁以上年龄段用户的共鸣，粉丝质量较高。

![](img/62b15ef760bccc8a0f3bf0406ed9436d_9.png)

以下是操作人性语录类视频的完整流程框架：
1.  **获取文案与素材**：从特定渠道收集能引发共鸣的人性洞察语句及相关图片/视频素材。
2.  **文案配音**：使用配音软件或真人将文案转化为音频。
3.  **视频制作**：将音频与素材结合，剪辑成完整的短视频。

![](img/62b15ef760bccc8a0f3bf0406ed9436d_11.png)

坚持发布至关重要。一个新账号的收益增长需要过程，如下图所示，从初期每天几元、几十元，坚持发布十天后，日收益可显著增长至数百元。

![](img/62b15ef760bccc8a0f3bf0406ed9436d_9.png)

**发布频率建议**：通过原创后，每个账号每天发布3-5个视频为佳，**切勿超过5个**。发布过多可能导致账号权重下降。

![](img/62b15ef760bccc8a0f3bf0406ed9436d_13.png)

## 推荐赛道二：文案鸡汤类 ✨
文案鸡汤类内容在任何内容平台都具有持久的生命力，在视频号同样如此。这类内容通过优美的文字和舒缓的音乐，传递情感与慰藉。

![](img/62b15ef760bccc8a0f3bf0406ed9436d_15.png)

操作文案鸡汤类视频的流程与人性语录类相似，也包含以下几个核心环节：
1.  **获取素材**：收集优质文案、背景音乐和视觉素材。
2.  **配音制作**：为文案配上合适的朗读。
3.  **内容合成**：采用图文或简单视频的形式，将以上元素合成发布。

## 进阶玩法：电视剧解说搬运 📺
除了原创内容，还有一种高效的“搬运+处理”玩法，主要适用于电视剧解说领域。这种玩法的核心在于利用工具对现有视频进行“去重”处理，使其能被平台识别为“原创”。

简单来说，此玩法的操作路径是：**寻找热门解说视频 -> 软件批量去重 -> 发布至视频号变现**。

**操作步骤**：
1.  **寻找素材**：在抖音、快手等平台搜索热门的电视剧解说视频并下载。
2.  **批量去重**：使用专用的去重软件对下载的视频进行多重处理（如缩放、抽帧、加滤镜、修改MD5值等），以通过平台原创审核。
3.  **发布变现**：将处理后的视频发布到视频号，依靠其高传播性获取播放和广告收益。

![](img/62b15ef760bccc8a0f3bf0406ed9436d_19.png)
