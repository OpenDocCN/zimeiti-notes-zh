# 抖音运营教程：P26：垂直类目细分领域 🎯

在本节课中，我们将要学习账号垂直度的分级概念。理解垂直类目的细分，能帮助你更精准地定位内容，吸引目标粉丝。

上一节我们介绍了内容标签的重要性，本节中我们来看看如何将“垂直”这个概念进行更细致的划分。

我们常说做账号要垂直。有些同学可能会简单地认为，选择“美食”或“游戏”这样的大类一直做下去就是垂直。这个理解没错，但这仅仅属于最基础的垂直级别。

垂直领域是有级别划分的，从宽泛到具体，通常可以分为一级、二级、三级甚至更多。以下是各级垂直领域的定义和示例：

**一级垂直领域**：指最宽泛的内容大类。
*   **示例**：`游戏`、`美食`、`美妆`、`旅行`。

**二级垂直领域**：在一级大类下的具体分支。
*   **示例（以“游戏”为例）**：`手游`、`电竞`、`网游`。

**三级垂直领域**：在二级分支下的进一步细分，通常具体到某个产品、主题或风格。
*   **示例（以“手游”下的“王者荣耀”为例）**：`王者荣耀安琪拉教学`、`王者荣耀剧情模式`。

这个分级还可以继续深入。例如，在“王者荣耀安琪拉教学”这个三级领域下，你还可以细分为“安琪拉出装攻略”、“安琪拉连招技巧”等，这可以视为四级垂直领域。

选择哪个级别的垂直领域，完全取决于你自己的规划和目标。大领域受众广但竞争激烈，小领域受众精准且容易建立专业权威。
