上一节我们介绍了如何起号，本节中我们来看看账号运营必须遵守的平台规则。小红书的规则体系决定了你的内容能否被系统推荐、能获得多少初始流量。简单来说，系统会通过分析你的账号信息、发布内容及用户互动行为，为你的账号打上“标签”，并据此决定将你的内容推荐给哪些用户、给予多少曝光。理解这套规则，才能有效提升账号权重，做好数据。

我将小红书的推荐规则总结为以下四个核心要点，它们环环相扣，内容非常丰富。

以下是四个核心要点：
1.  **用户行为规范**

4.  **账号登录限制**

接下来，我们将逐一进行详细解读。

## 一、用户行为规范

用户行为规范主要定义了在平台上哪些行为是被禁止的，哪些是允许的。这直接关系到账号安全，尤其是涉及营销和引流时。

以下是**严格禁止**的行为（用 `[禁止]` 表示）：
*   `[禁止]` **含有直接营销词汇**：包括但不限于“微信号”、“二维码”、“代购”、“水印”、“某某店铺”等。平台严禁在笔记或商品信息中直接进行外部引流或宣传特定店铺。
*   `[禁止]` **在笔记中直接宣传其他账号或商品**：例如，你想将A账号的流量引导至有商品的B账号，不能在A账号的笔记正文中直接宣传B账号。

![](img/3acc6706ec637147e28a5282195945f3_19.png)

以下是**允许采用技巧规避**的行为（用 `[技巧]` 表示）：
*   `[技巧]` **使用谐音或话术间接引导**：例如，将“微信”写作“V心”或“薇信”，在文案中用“看主页瞬间”等话术暗示。
*   `[技巧]` **利用评论区互动**：可以用主账号在评论区置顶留言，并`@`你的小号，通过这种方式进行间接引导。这是目前相对常见的规避方式。
*   `[允许]` **留下邮箱或微博信息**：个人资料中允许存在邮箱和微博公开信息。

> **核心提示**：微信号、二维码、代购、水印、某某店铺是平台最敏感的五大词汇，触犯其中任何一项都可能导致限流或封号。做引流必须寻找其他合规方法。

## 二、内容创作禁忌（自序商品）

这一部分主要针对笔记和商品内容本身的合规性。平台对某些领域的内容有严格限制。

![](img/3acc6706ec637147e28a5282195945f3_21.png)

![](img/3acc6706ec637147e28a5282195945f3_23.png)

![](img/3acc6706ec637147e28a5282195945f3_25.png)

以下是需要特别注意的禁忌：
*   **严禁提供医疗建议**：例如，“我在这家医院治好了，推荐你去”这类内容绝对不允许。涉及个人健康问题需格外谨慎。
*   **严禁提供投资建议**：例如，分析股票涨跌、推荐房产、评价某类资产保值性等均属违规。这涉及金融安全，平台管控极严。
*   **遵守账号登录限制**：在**电脑端**操作时，同一设备同时登录的账号**不建议超过3个**。即使使用模拟器（如雷电模拟器）修改IP，也存在被检测的风险。这是经验之谈，为确保账号安全，登录第4个账号前，请先退出一个。

![](img/3acc6706ec637147e28a5282195945f3_27.png)

> **公式**：`安全内容 = 原创内容 - (医疗建议 + 投资建议 + 过度营销)`

## 三、笔记发布规范（图文阴影）

![](img/3acc6706ec637147e28a5282195945f3_29.png)

这部分规范了发布笔记时的具体操作，直接影响内容的初始推荐。

以下是发布笔记时**不能做**的事情：
*   `[禁止]` **诱导用户互动**：例如，“关注我就抽奖”、“点赞领取资料”等以利益诱导关注、点赞、评论的行为是被明令禁止的。
*   `[禁止]` **搬运抄袭**：直接复制粘贴他人的图片和文案发布，账号权重会变得极低，甚至被视为废号。原创度是获得流量扶持的关键。

![](img/3acc6706ec637147e28a5282195945f3_31.png)

以下是发布笔记时**应该做**的事情：
*   `[推荐]` **追求高原创度**：原创度越高，获得平台基础流量扶持的机会就越大。即使是50%的原创（如自己写文案配网络图），也比完全搬运要好。
*   `[技巧]` **通过店铺活动间接引导**：虽然笔记中不能直接诱导，但可以在店铺内设置“关注店铺享受折扣”等活动，这是一种相对安全的间接运营方式，但搜索权重可能较低。

> **代码逻辑类比**：
> ```python
> if 笔记内容.原创度 >= 80%:
>     获得流量扶持的概率 = “高”
> elif 笔记内容.原创度 >= 50%:
>     获得流量扶持的概率 = “中”
> else: # 完全搬运
>     账号权重 = “极低”
> ```
