在本节课中，我们将要学习小红书平台如何为用户打上标签，以及平台有哪些主要的流量入口。理解这些机制，有助于我们分析自己的内容标签，并掌握笔记如何被系统推荐给精准用户。

上一节我们介绍了系统如何判定内容标签，本节中我们来看看平台如何判定用户标签。这个过程可以帮助我们从背后的逻辑反推，如何让笔记被我们的精准用户看到。

## 用户标签的判定逻辑

系统为用户打标签的主要依据是用户的**浏览行为**。用户刚注册时选择的兴趣范围参考价值不大，它仅作为平台对用户的内容方向引导，无法进入系统深层的用户标签池。

系统根据用户的浏览行为来判定其兴趣分类。以下是几个核心的判定维度：

以下是判定用户标签的几个核心行为维度：

1.  **点击率**
    *   **公式**：`点击率 = (点击某类内容的次数) / (系统推送的该类内容总次数)`
    *   系统在发现页向用户推送多种内容（如美妆、家居、母婴）。用户点击哪一类内容更多，系统就能计算出其偏好。例如，用户点击10条内容，其中美妆3次、家居2次、母婴0次，系统会判定该用户对母婴内容不感兴趣，并持续推送更多美妆内容进行试探。

2.  **停留时长**
    *   用户点开笔记后的观看时长。如果用户频繁点击美妆内容但很快划走，说明这些具体笔记不吸引他，系统会尝试推送美妆领域下的其他细分内容（如上妆、护肤）。反之，如果用户对一条平常很少点击的家居笔记停留很久，仔细阅读图片和评论，系统可能判断他有装修或购买家居用品的潜在需求。

3.  **页面浏览量（PV）**
    *   用户浏览页面的深度。例如，用户不仅看了一篇笔记，还点进博主主页，并连续看了第二、第三篇笔记。提升用户在你主页的浏览深度，能为账号健康度带来非常积极的评分。

4.  **互动行为**
    *   包括点赞、收藏、评论、关注。这些数据直接提升笔记的互动率，是笔记进入更高层级流量池的关键。互动数据高的笔记，获得推荐的机会更大，也能带动账号内其他笔记的流量，并有助于涨粉和后续变现。

通过以上行为，系统会为用户生成一个“标签云图”，其中较大的标签代表用户的强兴趣领域。系统会持续向用户推送其强兴趣领域的内容，以提升用户的获得感与停留时长，从而实现平台、用户与博主三方的价值串联：博主提供有价值的内容给用户，平台则负责将匹配的内容分发给用户。

## 平台的流量分发入口

了解用户标签后，我们来看小红书平台将内容分发给用户的几个主要流量入口。平台主要通过以下四个页面进行流量分发：

以下是小红书四个主要的流量分发入口：

1.  **发现页**
    *   **内容来源**：
        *   用户已关注的达人发布的内容。
        *   系统根据用户标签（点击率、兴趣等）匹配的推荐内容。
        *   系统为增加平台内容多样性而推送的部分试探性内容。
    *   这是笔记和账号获取流量的最主要来源。

2.  **关注页**
    *   **内容来源**：纯粹展示用户所有已关注博主发布的最新笔记。用户通常从这里直接查看强关注对象的更新。

3.  **附近页**
    *   **内容来源**：根据手机定位，推送同城或附近地区的相关内容。主要以美食、本地旅游、线下店铺等具有强地域特征的内容为主。适合线下实体店（如美容院、餐厅）进行本地推广。

4.  **购物页**
    *   **内容来源**：小红书电商和变现的核心入口，直接展示商品及带货内容。部分用户习惯在此搜索商品，但目前更多用户仍倾向于通过搜索笔记来获取购物建议。
