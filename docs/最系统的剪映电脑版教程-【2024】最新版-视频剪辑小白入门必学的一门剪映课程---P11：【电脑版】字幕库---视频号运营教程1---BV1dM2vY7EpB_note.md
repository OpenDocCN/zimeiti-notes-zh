在本节课中，我们将要学习剪映电脑版中“文本”功能的核心模块——字幕库。我们将系统地了解如何为视频添加和美化文字，包括自定义文本、使用文字模板、智能识别字幕与歌词，以及导入外部字幕文件。掌握这些技能，能让你的视频内容更清晰、更具吸引力。

## 文本功能概述 🎬

上一节我们介绍了视频剪辑的基础操作，本节中我们来看看如何为影片添加文字包装。文字是视频包装的重要元素，能有效提升影片的质感和信息传达效果。在剪映中，所有文字相关功能都集中在菜单栏的“文本”选项中。

打开“文本”后，你会看到多个子模块：默认文本、文字模板、智能字幕、识别歌词和本地字幕。接下来，我们将逐一详细讲解。

---

## 默认文本：自定义文字样式 ✍️

默认文本功能允许你完全自定义文字内容与样式。以下是操作步骤：

1.  **添加文本**：点击“默认”选项，然后将“默认文本”拖拽到视频轨道上，或点击“+”号添加到播放头所在位置。
2.  **编辑内容**：在预览窗口双击文本，即可输入你想要的文字，例如“眼睛”。
3.  **调整样式**：选中轨道上的文本，右侧会弹出编辑面板。你可以进行多项调整：
    *   **字体与字号**：在“字体”下拉菜单选择字体，拖动“字号”滑块或直接输入数值调整大小。
    *   **文字样式**：使用 **`B`**（加粗）、**`U`**（下划线）、**`I`**（倾斜）按钮。
    *   **颜色**：点击“颜色”方块，从色板中选择或使用取色器自定义颜色。
    *   **间距**：调整“字间距”和“行间距”（针对多行文字）的数值。
    *   **对齐方式**：设置文字本身的对齐方式（左、中、右）。
    *   **样式与位置**：在“样式”标签下，可以为文字添加描边、背景、阴影、发光等效果。在“位置”选项中，可以调整文字在整个视频画面中的对齐方式。
    *   **动画**：在“动画”标签下，可以为文字设置“入场”、“出场”和“循环”动画效果，并调整动画时长。

**核心概念公式**：
`自定义文本 = 基础内容 + 字体/颜色/间距 + 样式效果（描边/阴影/发光）+ 动画效果`

## 文字模板：快速应用精美效果 🎨

如果你希望快速获得专业级的文字效果，无需手动调整，可以使用“文字模板”功能。文字模板是剪映预设好的、带有复杂动画和样式的文字组合。

以下是使用方法：

1.  **选择模板**：在“文本”菜单下点击“文字模板”，浏览丰富的分类，如“标题”、“片头”、“节日”等。
2.  **应用模板**：将喜欢的模板直接拖拽到视频轨道上。
3.  **修改内容**：选中轨道上的模板，在右侧编辑面板中，找到“文本”选项，修改模板内的文字内容为你需要的文案。
4.  **微调效果**：你仍然可以调整模板的缩放、位置、旋转等基础属性，使其更贴合你的视频画面。

**核心概念**：文字模板是**开箱即用**的解决方案，极大提升了包装效率。

## 智能字幕：自动生成对话字幕 🗣️

为有配音或对话的视频添加字幕是一项常见但繁琐的工作。剪映的“智能字幕”功能可以自动识别音频并生成字幕。

操作步骤如下：

1.  **导入音视频**：将带有配音的视频或单独音频文件导入轨道。
2.  **开始识别**：点击“文本” -> “智能字幕”，然后点击“开始识别”。
3.  **校对与美化**：识别完成后，字幕会自动出现在轨道上。你可以双击字幕条修改错别字，并统一为所有字幕应用“花字”样式或动画效果。

**核心概念代码**（示意流程）：
```plaintext
导入音频 -> 点击“智能识别” -> 自动生成时间轴字幕 -> 批量校对与美化
```

---

## 文稿匹配：精准对齐已有文案 📄

当你有完整的视频解说文案时，“文稿匹配”功能比“智能识别”更精准。它能避免语气词干扰，并减少错别字。

使用方法：

1.  **准备文案与音频**：确保你的配音是根据文案录制的，且节奏一致。
2.  **粘贴文案**：点击“文本” -> “智能字幕” -> “文稿匹配”，将完整的文案粘贴到输入框中。
3.  **开始匹配**：点击“开始匹配”，系统会自动将文案按音频节奏切分成字幕条。

---

## 识别歌词：为音乐视频添加歌词 🎵

此功能专为音乐视频设计，能自动识别歌曲并生成歌词字幕。

操作流程：

1.  **导入音乐**：将歌曲文件导入音频轨道。
2.  **识别歌词**：点击“文本” -> “识别歌词”，然后点击“开始识别”。
3.  **检查修改**：识别完成后，务必检查是否有错别字（如谐音字），并可在右侧面板统一调整歌词的样式和动画。

## 本地字幕：导入外部字幕文件 🤝

在团队协作或使用专业字幕工具时，常会生成 `.srt`、`.lrc` 等格式的字幕文件。剪映支持直接导入这些文件。

![](img/29fccfca97381b02ac7e15f69993cf2a_8.png)

![](img/29fccfca97381b02ac7e15f69993cf2a_9.png)

操作步骤：

1.  **导入文件**：点击“文本” -> “本地字幕”，点击“导入”按钮，选择你的字幕文件（如 `.srt`）。
2.  **拖入轨道**：导入后，将字幕文件从素材库拖拽到视频轨道上，对齐音频起点。
3.  **统一调整**：导入的字幕会成为一个整体，你可以像处理普通字幕一样，为其批量修改字体、颜色、大小和动画。

**核心概念**：本地字幕功能实现了**跨软件、跨人员**的协作流程，提升了工作效率。
