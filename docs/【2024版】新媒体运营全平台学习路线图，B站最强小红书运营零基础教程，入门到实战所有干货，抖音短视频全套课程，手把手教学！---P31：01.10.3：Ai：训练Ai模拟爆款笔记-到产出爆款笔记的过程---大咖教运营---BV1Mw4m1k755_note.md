# 新媒体运营AI实战：P31：01.10.3：🤖 训练AI从分析到产出爆款笔记

在本节课中，我们将学习如何训练AI（以ChatGPT为例）分析爆款笔记，并最终生成具备爆款潜质的新笔记。我们将通过具体的步骤，从提供资料、分析结构到最终产出，手把手带你完成整个过程。

上一节我们介绍了通过“喂资料”的方式，让ChatGPT学习特定平台的文案风格和用语。虽然上一节的旅游攻略案例结果不尽理想，但方法是正确的。本节中，我们来看看如何深化这一过程，让AI通过分析具体的爆款内容，学习其成功要素并用于创作。

## 核心训练思路：让AI学会分析

![](img/df52631e4f6a2f95ed2596653bb42253_5.png)

![](img/df52631e4f6a2f95ed2596653bb42253_7.png)

![](img/df52631e4f6a2f95ed2596653bb42253_9.png)

![](img/df52631e4f6a2f95ed2596653bb42253_11.png)

我们的目标是让ChatGPT成为我们的“爆款分析师”。为了实现这一点，我们需要引导它关注以下几个核心分析维度：

以下是需要AI重点分析的几个方面：
1.  **爆款因素**：分析内容为何能成为爆款，例如是否引发了情感共鸣、提供了实用价值、讲述了逆袭故事等。
2.  **文案结构**：拆解文案的起承转合。
    *   **开篇**：用什么方式（如提问、制造焦虑、引用名人）吸引用户继续阅读。
    *   **中间**：如何通过戳中用户痛点、提供详细解决方案来留住用户，提升停留时长。
    *   **结尾**：如何设置“钩子”（如鼓励互动、引导私信、提供福利）促使用户行动。
3.  **关键词策略**：分析笔记中是否植入了特定关键词，以及关键词的出现频率和分布方式。

### 第一步：提供爆款笔记并下达分析指令

![](img/df52631e4f6a2f95ed2596653bb42253_25.png)

![](img/df52631e4f6a2f95ed2596653bb42253_27.png)

首先，我们需要找到目标爆款笔记，并将其文案复制下来。接着，向ChatGPT下达清晰的分析指令。

![](img/df52631e4f6a2f95ed2596653bb42253_29.png)

![](img/df52631e4f6a2f95ed2596653bb42253_31.png)

![](img/df52631e4f6a2f95ed2596653bb42253_32.png)

![](img/df52631e4f6a2f95ed2596653bb42253_34.png)

**给AI的指令示例**：
```
我给你提供一篇小红书平台上的考研爆款笔记文案，请你分析：
1.  爆款因素：这篇文案为什么能火？列出具体原因。
2.  文案整体结构：
    *   开篇用什么方法吸引用户？
    *   中间用什么内容戳中用户痛点？
    *   结尾用什么方式引导用户互动（如私信、关注）？
3.  关键词分析：根据整篇文案，分析是否有结合中文分词法设计关键词。如果有，指出是哪个关键词，并说明其出现频率和比率；如果没有，请说明。
```
将找到的爆款文案粘贴在指令下方，发送给AI。

### 第二步：迭代分析，总结规律

![](img/df52631e4f6a2f95ed2596653bb42253_43.png)

![](img/df52631e4f6a2f95ed2596653bb42253_45.png)

![](img/df52631e4f6a2f95ed2596653bb42253_46.png)

![](img/df52631e4f6a2f95ed2596653bb42253_48.png)

![](img/df52631e4f6a2f95ed2596653bb42253_50.png)

重复第一步，为AI提供多篇（建议3-5篇）同领域的爆款笔记进行分析。每次分析后，观察AI给出的结论。

![](img/df52631e4f6a2f95ed2596653bb42253_51.png)

![](img/df52631e4f6a2f95ed2596653bb42253_53.png)

**关键操作**：当AI可能“忘记”最初的分析要求或开始跑题时，需要重新编辑提示，强调核心指令，将其“拉回”正轨。

![](img/df52631e4f6a2f95ed2596653bb42253_55.png)

在提供足够样本后，要求AI对分析过的所有爆款因素进行整合去重，总结出该领域爆款笔记的共性。

![](img/df52631e4f6a2f95ed2596653bb42253_57.png)

**整合指令示例**：
```
请将你前面分析过的所有考研爆款笔记的爆款因素总结一下，整合成一个清晰的列表。
```

![](img/df52631e4f6a2f95ed2596653bb42253_59.png)

### 第三步：基于分析结果，生成爆款大纲

![](img/df52631e4f6a2f95ed2596653bb42253_61.png)

在AI学习了爆款规律后，我们可以指令它基于这些规律，创作一篇新笔记的大纲。

**生成大纲指令示例**：
```
现在，请结合你总结的爆款笔记因素，为我规划一篇关于考研的小红书笔记大纲。大纲需要细分到具体的小节点。在我确认大纲没问题后，你再生成完整文案。
```
AI可能会生成如下结构的大纲：
*   1. 开篇：吸引注意
*   2. 主题介绍：考研备考
*   3. 专业选择分析
*   4. 分科备考规划（政治、英语、数学、专业课）
*   5. 时间管理与心态调整
*   6. 结尾：互动引导

![](img/df52631e4f6a2f95ed2596653bb42253_63.png)

### 第四步：根据大纲，生成完整文案

![](img/df52631e4f6a2f95ed2596653bb42253_65.png)

![](img/df52631e4f6a2f95ed2596653bb42253_66.png)

![](img/df52631e4f6a2f95ed2596653bb42253_68.png)

确认大纲符合要求后，下达最终生成指令。这一步的指令需要非常具体，确保产出物符合预期。

**生成文案指令示例**：
```
请按照你提供的大纲，撰写具体的文案。要求如下：
1.  文案结构严格遵循爆款笔记因素。
2.  需要包含一个简单的“逆袭”小故事来引入读者情绪。
3.  文案中要包含emoji表情。
4.  文案中要结合中文分词法的方式融入“考研”这个关键词。
```
**生成的文案示例片段**：
```
当你曾经迷茫，觉得时间不够用，考研上岸遥不可及……我理解那种感觉！😔 但我也亲身经历了整个逆袭的过程。让我来告诉你，只要你下定决心，制定一份合理的**考研**计划，梦想绝非遥不可及！💪

**【明确你的备考时间线】**
从现在（2023年11月）到考前，每一分钟都宝贵！⏰
*   11-12月：打好基础，主攻**考研**政治马原和英语单词。
*   1-3月：强化训练，开始**考研**专业课第一轮复习。
...
（后续为分科详细建议、时间管理方法和鼓励性结尾）
```

## 优化与进阶提示

![](img/df52631e4f6a2f95ed2596653bb42253_70.png)

*   **指令需精确**：与AI对话时，用词需要琢磨。例如，“包含emoji表情”和“包含丰富的emoji表情”会产生不同密度的结果。
*   **模型选择**：使用更强大的模型（如GPT-4）通常会获得更优质、更智能的产出。
*   **创建专属AI助手**：可以利用ChatGPT的“GPTs”功能，上传你整理的平台风格指南、专属表情库和优秀文案范本，打造一个更懂你需求的私人写作助手，避免每次都需要重新提供背景指令。
