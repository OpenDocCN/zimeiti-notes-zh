## 如何在小红书内找对标？📱

接下来，我们具体讲解在小红书站内寻找对标账号的实操步骤。这个方法的核心是利用平台的搜索和推荐机制。

1.  **打开小红书搜索页**，输入你的类目关键词，例如“马面裙”。
2.  **在搜索结果中**，重点寻找粉丝数在1万以下的账号。这类账号通常处于快速上升期，更具参考价值。
3.  **识别混剪搬运类账号**：这类账号通常有几个特点：视频清晰度一般、全程用表情包遮脸或不露脸、持续发布多样化的带货笔记、偶尔会出现点赞上千的爆款、产品价格多为两位数（如99.99元）。
4.  **互动养号**：找到目标账号后，积极点赞、收藏、关注其爆款笔记。平台的大数据算法会因此为你推荐更多同类型账号，形成良性循环。

## 如何在抖音找对标？🎬

除了小红书，抖音也是一个巨大的素材库和灵感来源。在抖音寻找对标的逻辑与小红书类似。

以下是抖音找对标账号的步骤：

![](img/558b2a3c25de16eaacfe52fcfa61dfa8_4.png)

1.  在抖音**搜索类目关键词**，并切换到“视频”栏目。
2.  **筛选视频**，选择发布时间不超过半个月、点赞量较高但发布者粉丝数不超过30万（低粉高赞）的视频。
3.  **循环寻找10-20个**符合标准的账号。
4.  建议使用一个**新的抖音号**专门执行此操作，这样平台会持续为你推荐类似账号，方便长期寻找素材。
