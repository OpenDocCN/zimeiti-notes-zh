# 抖音运营入门：P2：1、抖音短视频的前景 📈

![](img/6c372ee5986ca593942dfcdd6045ef91_1.png)

在本节课中，我们将要学习抖音短视频平台的发展前景，分析其作为内容创作和变现渠道的巨大潜力，并理解为什么它适合新手入门。

![](img/6c372ee5986ca593942dfcdd6045ef91_3.png)

了解抖音短视频的前景，首先需要通过一张图片来认识。

![](img/6c372ee5986ca593942dfcdd6045ef91_1.png)

这张图片是抖音创作者大会的一份数据报告。报告显示，有超过2200万人在抖音的合计收入超过了417亿元。并且，抖音希望将这个数字翻一番，让创作者的总收入达到800亿元。

![](img/6c372ee5986ca593942dfcdd6045ef91_3.png)

这代表了平台巨大的流量扶持力度。同时，这里引入第一个核心概念：抖音用户主要分为两类。

以下是两类用户的定义：
*   **玩家**：指单纯在抖音上娱乐、消费内容，没有计划通过平台赚钱的用户。
*   **创作者**：指主动制作并发布内容，旨在通过抖音获取收益的用户。我们的课程正是面向希望成为创作者的同学。

那么，在众多短视频平台中，我们为什么选择抖音？这类似于开设实体店铺选址的逻辑。

以下是选择抖音的核心原因：
1.  **流量巨大**：抖音日活跃用户超过8亿，月活跃用户更是达到40亿人次。巨大的流量池是变现的基础。
2.  **竞争相对温和**：平台上的创作者数量约为5000万。其中绝大部分是普通用户，你的直接竞争对手并非专业团队，而是身边的普通人。
3.  **入门门槛极低**：只需要一部智能手机和一个手机号即可注册账号开始创作。抖音倡导全民通过短视频记录和分享生活。

接下来，我们看看当前有哪些值得关注的风口。抖音推出了“抖音计划”，旨在扶持特定领域的创作者。

“抖音计划”的核心目标是：**帮助1000名优质的艺术创作者涨粉超过100万**。许多同学可能会担心自己不会创作，但这正是学习的目的。虚心向有经验的人请教是快速成长的捷径。

市场空间广阔，内容需求旺盛。我们处在一个大数据时代，平台会根据用户的喜好推荐内容。这意味着，无论你分享什么——宝宝日常、家常饭菜、店铺产品还是个人生活——只要内容不触犯法律和道德底线，总会有一部分受众喜欢。

对于新手小白而言，涨粉的心情可以理解，但不应将其作为首要目标。

我们的核心策略公式是：**优先提升播放量 → 自然吸引粉丝**。因为喜欢你的人可能还没有刷到过你的作品。因此，第一步是专注于创作能获得流量的内容，粉丝增长便会水到渠成。

本节课中我们一起学习了抖音平台的巨大前景、用户分类、选择抖音的理由以及新手起步的核心策略。记住，在庞大的流量池中，找到自己的细分领域并持续输出内容，是成功的第一步。下一节，我们将开始探讨如何具体搭建和设置你的抖音账号。
