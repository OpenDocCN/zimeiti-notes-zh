# 《抖音全套运营教程》：P4：短视频文案怎么找 📝

在本节课中，我们将学习如何高效地寻找短视频文案，解决内容创作的难题，帮助你在保持作品数量和质量的同时，减轻创作压力。

---

![](img/6c5560fab3c386f53076a13dcc5775b2_1.png)

说做抖音最令人崩溃的就是写文案。群里有一个伙伴刚做短视频一个月，决定不再继续做了。他每天需要写800字的文案稿。

![](img/6c5560fab3c386f53076a13dcc5775b2_1.png)

他的视频没什么人看，他感觉自己快成精神病了。他问我，有没有一种输出方式可以不这么累，在一个人做的时候，也能保持作品数量和质量。我说方法是有，但手段可能不太光彩，我不太推荐。

但是有一个大V就是靠这个办法，积累到了上千万的粉丝。这个办法用一句话就可以说明白：**任何一个图文平台出现过的10万加内容，你都可以用短视频的形式再做一遍。**

![](img/6c5560fab3c386f53076a13dcc5775b2_3.png)

![](img/6c5560fab3c386f53076a13dcc5775b2_3.png)

这是什么意思呢？比如，你在朋友圈看到一篇10万加的文章，大家都在转发。这说明这篇文章已经被至少10万人认同，是能产生情绪共鸣的高质量内容。例如这篇讲拼多多砍价的文章，普通人看后就是满腔愤怒，觉得终于有人发声，给出了口恶气。

你也想蹭这个热点，但自己说三言两语就没内容了。怎么办呢？你可以去公众号、知乎等平台搜索相关话题，然后用口播的形式再拍一遍视频。做其他领域也是一个道理。

比如你觉得健身减肥是一个好项目，赛道卖轻卡零食或减肥代餐有搞头，但你是门外汉不懂。这时依然可以用这个办法。你就去找这个领域的爆款文案，然后用自己的话讲出来。

![](img/6c5560fab3c386f53076a13dcc5775b2_5.png)

看得多了，讲得多了，粉丝也就积累了，你也会逐渐成为这个行业的专家。这是第一个阶段：**通过转换图文爆款为视频内容，来帮助自己快速起号**。因为曾经火过的内容，有很大概率可以再次火起来。

![](img/6c5560fab3c386f53076a13dcc5775b2_5.png)

![](img/6c5560fab3c386f53076a13dcc5775b2_7.png)

但是，第二个阶段就不能一直这么用了。万一让人发现你完全没有自己的观点，会对账号信誉造成损害。

---

![](img/6c5560fab3c386f53076a13dcc5775b2_7.png)

本节课中我们一起学习了寻找短视频文案的两个核心思路。首先，可以通过将图文平台的爆款内容转化为视频形式，来快速启动账号并保证内容质量。其次，在积累一定基础后，需要逐步融入自己的观点和创作，以实现长远发展。
