## 🕒 核心要求一：时间投入

时间是最重要的资源。无论你的身份如何，每天都需要投入固定的时间来学习和操作。

以下是具体的时间要求：
*   **基础要求**：每天至少能抽出**30分钟**的零散时间。这适用于时间较为碎片化的学员（如宝妈、上班族），可以专注于“创作者分成计划”等玩法。
*   **进阶要求**：如果希望全职投入或操作更多玩法（如视频带货、无人直播），则需要每天有**1小时以上**的连续时间。

**收益公式参考**：
*   创作者分成计划：`日收益 ≈ 80元` -> `月收益 ≈ 80元/天 * 30天 = 2400元`
*   视频带货（参与活动）：`日收益 ≈ 300元` -> `月收益 ≈ 300元/天 * 30天 = 9000元`
*   多账号操作收益倍增，例如两个账号做视频带货：`月收益 ≈ 9000元 * 2 = 18000元`

## ⚡ 核心要求二：执行力与态度

光有时间不够，还必须具备强大的执行力和积极的态度。老师出身部队，最看重“快、狠、准”的行事风格。

![](img/1fc9f5e0a714ffb0b04ec98bb174124b_3.png)

以下是两点关键态度要求：
1.  **不拖拉、不偷懒**：必须按时完成学习任务和视频发布计划。执行力体现在每一个互动和操作细节上。
2.  **听话照做，全力以赴**：教学方案是经过验证的。学员需要严格按照指导步骤操作，不要自行其是。只有完全跟随，才能确保学习效果。

## 📝 核心要求三：信息透明

为了让教学方案更具针对性，学员需要向老师坦诚自己的真实情况。

![](img/1fc9f5e0a714ffb0b04ec98bb174124b_5.png)

以下是需要告知的信息类型，老师会据此定制方案：
*   **身份与时间**：例如是宝妈、上班族、学生还是退休人员，每天可支配的时间段。
*   **基础情况**：例如年龄、文化水平（是否识字）、是否有运营经验（新账号/老账号）。
*   **特殊状况**：例如是否有负债、是否急需赚钱等个人压力。

老师曾指导过各类学员，包括56岁的阿姨、63岁不识字的老人，乃至84岁的高龄学员。无论基础如何，只要愿意学，都有对应的方法。

![](img/1fc9f5e0a714ffb0b04ec98bb174124b_7.png)
