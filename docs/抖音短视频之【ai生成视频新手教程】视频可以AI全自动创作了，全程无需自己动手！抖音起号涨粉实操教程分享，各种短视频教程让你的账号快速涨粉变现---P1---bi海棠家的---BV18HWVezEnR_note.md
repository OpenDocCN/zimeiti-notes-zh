## 核心工具介绍

实现全自动创作需要三个核心AI工具的协同工作。

以下是三个必备工具及其作用：

1.  **文案创作工具：ChatGPT**
    *   用于生成高度原创且优质的视频文案。
2.  **配音生成工具：配音神器**
    *   用于将AI生成的文案转换为自然的人声配音。
3.  **视频素材匹配工具：三三搜帧**
    *   用于根据文案和配音，自动匹配高质量、无版权的动态视频画面。

## 分步操作详解

### 第一步：使用ChatGPT生成原创文案

首先，我们需要利用ChatGPT生成视频的核心文案。它的强大之处在于能理解复杂指令并生成富有情感和逻辑的文本。

**操作示例**：
你可以向ChatGPT提出明确的文案需求。例如，输入以下指令：
`请为我创作一个约1000字的中国历史故事，主题是曹操与华佗。`
ChatGPT便会生成相应的故事文案。

如果你对初步结果不满意，可以追加条件进行优化。例如：
`请在刚才的故事中加入谋士荀彧这个人物，并保持总字数不变。`
通过这种交互，你能获得更精细、更独特的原创内容。

### 第二步：使用配音工具生成语音

获得优质文案后，下一步是将其转化为语音。这一步操作简单，能快速得到专业配音。

以下是具体步骤：

1.  打开配音工具（如配音神器），点击“制作配音”。
2.  将ChatGPT生成的文案粘贴到文本框中。
3.  点击“更多主播”，从多种音色风格中选择一个合适的主播。
4.  生成配音并保存为音频文件。

### 第三步：使用三三搜帧自动合成视频

![](img/3f04e340f16c70fad9d1756102c7b046_5.png)

最后也是最关键的一步，是让画面自动匹配文案和配音。三三搜帧工具能智能解析文案含义，并寻找高度相关的动态视频素材。

**核心功能**：
该工具的“AI合成视频”功能，可以自动将配音文件与海量视频画面进行匹配。

![](img/3f04e340f16c70fad9d1756102c7b046_7.png)

**操作流程**：

![](img/3f04e340f16c70fad9d1756102c7b046_9.png)

1.  进入三三搜帧首页，点击“合成视频”。
2.  上传第二步中得到的配音文件。
3.  系统会自动根据每句台词，推荐多个相符的视频画面片段，供你确认或选择。
4.  选择完毕后，点击底部的“生成视频”按钮。
5.  系统将自动合成一个画面与内容高度匹配的原创视频。

**效果预览**：
生成后的视频效果流畅，例如一段关于“康熙皇子夺嫡”的文案，能自动匹配到相应的历史场景画面，观感专业。
