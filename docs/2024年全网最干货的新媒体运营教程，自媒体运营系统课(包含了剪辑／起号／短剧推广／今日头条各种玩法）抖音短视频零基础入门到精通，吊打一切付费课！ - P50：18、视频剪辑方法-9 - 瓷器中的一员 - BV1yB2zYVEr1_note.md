# 新媒体运营系统课：P50：18、视频剪辑方法-9 🎬

在本节课中，我们将通过实际操作，学习如何使用手机版剪映App完成一个短视频的完整剪辑流程。课程将涵盖从素材导入、基础剪辑、添加音乐与字幕，到最终导出的所有关键步骤。此外，我们还会介绍一个对“口播”类视频非常有帮助的提词器软件。

## 剪辑准备与素材导入

上一节我们介绍了剪辑前的准备工作，本节中我们来看看如何开始实际操作。

剪辑环节采用直接实操的方式。请在本环节做好笔记。

此时无需在公屏互动，应抓紧时间记录。我们将使用**剪映（黑色图标版本）**进行剪辑。请确保下载正确。

![](img/21b08c167fb1b50074558a94e3343f13_1.png)

剪映是抖音平台的官方剪辑工具，提供手机版和电脑版。本教程先讲解手机版，以照顾没有电脑的同学。

![](img/21b08c167fb1b50074558a94e3343f13_2.png)

以下是开始创作的步骤：

![](img/21b08c167fb1b50074558a94e3343f13_3.png)

![](img/21b08c167fb1b50074558a94e3343f13_4.png)

1.  打开手机版剪映App。
2.  点击“开始创作”按钮。
3.  从相册中选择需要剪辑的视频素材（例如一段22秒的视频），点击“添加”导入。

![](img/21b08c167fb1b50074558a94e3343f13_1.png)
![](img/21b08c167fb1b50074558a94e3343f13_2.png)
![](img/21b08c167fb1b50074558a94e3343f13_3.png)
![](img/21b08c167fb1b50074558a94e3343f13_4.png)

视频素材成功添加到剪辑轨道。

如果拍摄的是由多个片段（如开头、中间、结尾）组成的生活片段，建议先单独剪辑好第一条。剪辑完成后，点击轨道尾部的“+”号，即可导入并剪辑第二条视频。按顺序逐条导入和剪辑，可以避免混乱。

![](img/21b08c167fb1b50074558a94e3343f13_6.png)

## 基础剪辑：调整构图与裁剪片段

素材导入后，我们首先需要检查并调整视频的基本构图，然后裁剪掉无用的部分。

我们首先剪辑已导入的这条视频。先检查问题所在。

第一步是修正错误的构图比例。解决方法如下：

1.  在剪辑轨道上，点击需要调整的视频片段。
2.  向右滑动底部工具栏，找到并点击“编辑”图标（一个方块形状）。
3.  在编辑菜单中，点击左侧的“旋转”功能，将视频旋转到合适的角度。
4.  若需进一步裁剪画幅比例，可点击第三个“裁剪”功能。这里提供了多种比例选项，如 **1:1**、**4:3**、**2:1** 等，可根据需要自行调整。

![](img/21b08c167fb1b50074558a94e3343f13_6.png)

完成初步调整后，返回主剪辑界面。

![](img/21b08c167fb1b50074558a94e3343f13_8.png)

接下来需要预览视频，听一下内容。点击播放按钮。

![](img/21b08c167fb1b50074558a94e3343f13_9.png)

![](img/21b08c167fb1b50074558a94e3343f13_10.png)

可能会发现视频开头有一段没有说话的空镜头。这些无效部分需要剪掉，否则会浪费观众时间。

![](img/21b08c167fb1b50074558a94e3343f13_11.png)

剪辑时间轴上的**竖线**即代表当前时间指针，也起到“剪刀”的作用。裁剪片段的基本操作是 **“选中 -> 分割 -> 删除”**。

以下是具体操作步骤：

1.  将时间线竖指针移动到需要裁剪片段的起始位置。
2.  点击选中轨道上的视频片段。
3.  点击底部工具栏第一个“分割”按钮，视频即被分割成两段。
4.  点击选中需要删除的前一段视频。
5.  点击“删除”按钮。

![](img/21b08c167fb1b50074558a94e3343f13_8.png)
![](img/21b08c167fb1b50074558a94e3343f13_9.png)

重复此操作。例如，中间或结尾有不需要的部分（如关手机的动作），操作流程一致：**选中片段 -> 分割 -> 选中要删除的部分 -> 删除**。

![](img/21b08c167fb1b50074558a94e3343f13_10.png)
![](img/21b08c167fb1b50074558a94e3343f13_11.png)

请仔细预览完整视频，并进行裁剪。记住核心操作：**选中、分割、删除**。

![](img/21b08c167fb1b50074558a94e3343f13_13.png)

## 添加与编辑背景音乐

![](img/21b08c167fb1b50074558a94e3343f13_15.png)

视频内容剪辑好后，下一步是为视频添加背景音乐。

![](img/21b08c167fb1b50074558a94e3343f13_16.png)

第二个关键步骤是添加“音频”。点击底部工具栏的“音频”按钮。

![](img/21b08c167fb1b50074558a94e3343f13_17.png)

![](img/21b08c167fb1b50074558a94e3343f13_18.png)

如果您之前使用过“淘金阁”等工具进行配音，可以直接点击“提取音乐”，导入已有音频文件。

如果未提前准备音乐，则点击“添加音乐”。剪映内置了丰富的音乐库，按“美食”、“美妆”、“萌宠”等分类。您可以搜索歌曲名，或浏览分类选择。

![](img/21b08c167fb1b50074558a94e3343f13_13.png)
![](img/21b08c167fb1b50074558a94e3343f13_14.png)
![](img/21b08c167fb1b50074558a94e3343f13_15.png)

点击音乐旁边的播放图标可以预览。选定音乐后，点击“使用”即可导入到视频轨道。

![](img/21b08c167fb1b50074558a94e3343f13_16.png)

音乐导入后，可能长度与视频不匹配。剪辑音乐的方法与剪辑视频相同。

如果音乐过长，操作如下：将时间线指针移动到需要裁剪的位置 -> **选中音乐轨道 -> 点击“分割” -> 选中多余部分 -> 点击“删除”**。

![](img/21b08c167fb1b50074558a94e3343f13_17.png)
![](img/21b08c167fb1b50074558a94e3343f13_18.png)

![](img/21b08c167fb1b50074558a94e3343f13_20.png)

调整音乐音量大小：选中音乐轨道，点击左侧第一个“音量”按钮，拖动滑块即可调高或调低音量。

![](img/21b08c167fb1b50074558a94e3343f13_21.png)

## 自动识别与添加字幕

![](img/21b08c167fb1b50074558a94e3343f13_24.png)

添加音乐后，一个完整的视频通常还需要字幕。我们将使用剪映高效的自动识别功能来添加字幕。

![](img/21b08c167fb1b50074558a94e3343f13_25.png)

回到主工具栏，进行第三个关键步骤：添加“文字”。点击底部工具栏的“文字”按钮。

![](img/21b08c167fb1b50074558a94e3343f13_27.png)

无需手动逐个输入字幕。推荐使用高效方法：点击“识别字幕”功能。

在弹出菜单中，选择“仅识别视频原声”。如果视频已添加背景音乐，不建议选择“全部”，否则会连歌词也识别出来。

![](img/21b08c167fb1b50074558a94e3343f13_20.png)
![](img/21b08c167fb1b50074558a94e3343f13_21.png)

![](img/21b08c167fb1b50074558a94e3343f13_29.png)

点击“开始识别”，系统将自动为视频语音生成字幕。

![](img/21b08c167fb1b50074558a94e3343f13_22.png)

识别完成后，字幕会出现在视频轨道上方。可以拖动调整其位置。

![](img/21b08c167fb1b50074558a94e3343f13_23.png)

需要检查并修改错别字：双击字幕条，即可进入编辑模式，进行删除或修改。

![](img/21b08c167fb1b50074558a94e3343f13_24.png)

还可以修改字体样式。例如，点击“样式”，可以更改颜色（如将字体颜色改为黄色以匹配衣服颜色）、字体、大小等。

![](img/21b08c167fb1b50074558a94e3343f13_25.png)
![](img/21b08c167fb1b50074558a94e3343f13_27.png)

![](img/21b08c167fb1b50074558a94e3343f13_31.png)

至此，视频已基本完成：比例已调整，内容已裁剪，音乐已添加，字幕也已自动生成并美化。

## 导出设置与发布

视频制作完成后，在导出前有一些关键设置需要注意，以确保视频清晰度。

点击右上角的“导出”按钮。在导出前，务必调整视频参数，防止平台压缩导致画质下降。

![](img/21b08c167fb1b50074558a94e3343f13_37.png)

对于抖音平台，推荐的导出设置如下：
*   **帧率 (Frame Rate)**：设置为 **30** fps。这个帧率最适合抖音平台。
*   **分辨率 (Resolution)**：尽可能拉满（如1080p或更高）。分辨率越高，视频越清晰。

**核心导出公式：帧率 = 30，分辨率 = 最高可用选项。**

![](img/21b08c167fb1b50074558a94e3343f13_31.png)
![](img/21b08c167fb1b50074558a94e3343f13_32.png)

设置完成后，点击“导出”。导出过程中，请勿锁屏或切换手机应用程序，耐心等待导出完成，以免中断进程。

![](img/21b08c167fb1b50074558a94e3343f13_33.png)

导出完成后，可以直接通过剪映的“分享到抖音”按钮进行发布。

![](img/21b08c167fb1b50074558a94e3343f13_34.png)
![](img/21b08c167fb1b50074558a94e3343f13_35.png)

![](img/21b08c167fb1b50074558a94e3343f13_39.png)

![](img/21b08c167fb1b50074558a94e3343f13_40.png)

请记住以上强调的操作要点。如果仍有疑问，可以在课后咨询。

## 课程总结

本节课中，我们一起学习了使用手机剪映进行短视频剪辑的完整流程：
1.  **导入素材**并学会逐条剪辑。
2.  进行**基础剪辑**，包括调整构图比例和运用 **“选中-分割-删除”** 裁剪片段。
3.  **添加与编辑背景音乐**，并调整音量。
4.  利用**自动识别字幕**功能快速生成并修改字幕。
5.  按照 **帧率30、分辨率最高** 的设置**导出视频**。
6.  认识了辅助口播拍摄的实用工具——**爱提词App**，用于提词和美颜拍摄。

![](img/21b08c167fb1b50074558a94e3343f13_55.png)

通过以上步骤，您已经可以独立完成一个具备基本要素的短视频作品。请务必动手实践，巩固所学技能。
