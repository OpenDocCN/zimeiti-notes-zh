## 1. 开通微信小商店

![](img/bf62b307024536f3680258e36d8b9d92_7.png)

如果你没有自己的产品，可以开通**微信小商店**进行带货。微信小商店是微信官方提供的免费开店工具，背靠微信生态，开通门槛极低。

以下是开通微信小商店的步骤：

![](img/bf62b307024536f3680258e36d8b9d92_9.png)

1.  打开微信，在顶部搜索框搜索“**微信小商店**”。
2.  进入“微信小商店助手”公众号。
3.  点击公众号菜单栏的“**免费开店**”。
4.  选择“**快速开店**”，系统会弹出小程序，完成注册即可。

**优势对比（微信小商店 vs. 抖音橱窗）**：

*   **微信小商店**：0元入驻，0粉丝要求，支持在视频号、公众号、社群、小程序等多场景卖货。
*   **抖音橱窗**：需要500元保证金，发布10条以上视频，并拥有1000粉丝。

可以看出，微信小商店的入门条件更为宽松，更适合新手起步。

## 2. 小商店的两种运营模式

![](img/bf62b307024536f3680258e36d8b9d92_15.png)

开通小商店后，你可以根据自身情况选择两种运营模式。

![](img/bf62b307024536f3680258e36d8b9d92_17.png)

### 模式一：销售自有产品

如果你有货源，可以在小商店中上架自己的商品。

操作路径：进入小商店后台 -> 点击“**新增商品**” -> 上传商品图片、编辑标题与描述等信息。

![](img/bf62b307024536f3680258e36d8b9d92_19.png)

为了让商品图片更美观，你可以使用一些设计工具，例如：
*   创客贴
*   美图秀秀
*   搞定设计

### 模式二：带货赚取佣金（无货源模式）

如果你没有货源，可以通过小商店的“带货中心”帮助其他商家销售商品，从而赚取佣金。

以下是进入选品中心的步骤：

1.  进入视频号个人主页。
2.  点击右上角头像进入“**创作者中心**”。
3.  在创作者中心找到“**带货中心**”。
4.  点击“**去选品**”，即可浏览海量商品。

在选品中心，你可以看到各类商品（如食品、服饰、图书等）及其销售佣金。例如，一本售价59.8元的书，卖出后你可能获得27.44元的佣金。

**佣金率优势**：微信视频号带货的佣金率较高，**可达50%左右**，而其他平台通常为25%-30%。这意味着销售同样的商品，在视频号能获得更高的收益。

## 3. 利用扩展链接带货

![](img/bf62b307024536f3680258e36d8b9d92_25.png)

除了小商店自带选品，你还可以通过“**扩展链接**”功能，销售来自其他电商平台的商品，充当“中间商”赚取差价。

常见的货源平台包括：
*   **阿里巴巴（1688）**：批发采购平台。
*   **淘宝联盟**：淘宝客推广平台。
*   **京东联盟**：京东商品推广平台。
*   **多多团长**：拼多多商品推广平台。

**操作流程**：
1.  在上述平台找到你想推广的商品，复制商品链接。
2.  在微信中搜索“扩展链接”小程序并打开。
3.  将商品链接粘贴到扩展链接小程序中，生成专属推广链接。
4.  发布视频时，在视频下方添加此扩展链接。

![](img/bf62b307024536f3680258e36d8b9d92_27.png)

这样，用户点击链接即可跳转购买，你便能获得相应的推广佣金。

## 4. 收益测算与案例

![](img/bf62b307024536f3680258e36d8b9d92_30.png)

![](img/bf62b307024536f3680258e36d8b9d92_31.png)

了解方法后，我们来算一笔账，看看视频号带货的收益潜力。

**核心公式**：
`预估收入 = 视频播放量（或商店访问量） × 转化率 × 商品佣金`

**假设一个保守 scenario**：
*   单个视频获得 **1万** 次有效曝光（用户访问小店）。
*   转化率取极低的 **1%**（即100人中有1人购买）。
*   商品平均佣金为 **50元**。

![](img/bf62b307024536f3680258e36d8b9d92_33.png)

**计算过程**：
`10,000 × 1% × 50 = 5,000元`
