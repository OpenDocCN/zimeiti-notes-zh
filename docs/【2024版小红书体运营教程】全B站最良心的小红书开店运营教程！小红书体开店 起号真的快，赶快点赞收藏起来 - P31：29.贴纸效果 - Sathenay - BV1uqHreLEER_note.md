# 小红书运营教程：P31：29.贴纸效果 🎨

在本节课中，我们将学习如何在小红书视频编辑中使用“贴纸”功能。贴纸能为你的内容增添趣味性和视觉重点，是提升视频吸引力的重要工具。我们将详细介绍如何添加、调整贴纸，并为其添加动画效果。

## 贴纸工具的位置与分类

![](img/ecdf333eec1770d38b66e8daa1317195_1.png)

上一节我们介绍了视频剪辑的基础操作，本节中我们来看看如何为视频添加装饰性元素。

![](img/ecdf333eec1770d38b66e8daa1317195_3.png)

在顶部工具栏中选择“贴纸”工具，可以进入贴纸素材库。素材库中有多种分类，方便你快速找到所需贴纸。

以下是主要的贴纸分类：
*   **热门**：平台当前流行的贴纸素材。
*   **VIP**：可能需要会员权限使用的专属贴纸。
*   **主题分类**：如“夏日旅行”、“毕业季”等场景化贴纸合集。

![](img/ecdf333eec1770d38b66e8daa1317195_1.png)

这些贴纸可以单击进行下载并使用。

## 如何添加与调整贴纸

了解分类后，接下来我们学习如何将贴纸应用到视频中并进行基础调整。

首先，导入视频素材到时间轴。然后选择“贴纸”工具，找到想要的贴纸，例如一个形状图案。将其直接拖拽到时间轴上，软件会自动为它建立一个独立的贴纸轨道。

添加贴纸后，你可以对其进行多方面的调整：
*   **调整时长**：在时间轴上拖拽贴纸轨道的两端，可以改变其显示时间。
*   **调整位置、大小与角度**：在播放面板中选中贴纸，可以直接用鼠标拖拽来移动位置、缩放大小或旋转角度。
*   **使用参数面板精确调整**：选中贴纸后，右侧参数面板允许你输入精确的数值。
    *   **位置**：通过坐标 `(X, Y)` 精确定位。
    *   **缩放**：调整 `缩放比例` 滑块或输入百分比。
    *   **旋转**：输入 `旋转角度` 数值。
*   **使用对齐工具**：软件提供多种对齐方式，如左对齐、水平居中对齐、右对齐、顶对齐、垂直居中对齐、底对齐等。对于需要统一多个元素位置的模板化制作，使用对齐工具能提高效率。但对于日常灵活编辑，直接拖拽调整通常更为方便。

## 贴纸的进阶操作

掌握了单个贴纸的调整方法后，我们来看看如何对贴纸轨道进行编辑和叠加使用。

选中时间轴上的贴纸轨道，你可以对其进行剪辑操作，代码逻辑类似于视频剪辑：
*   `分割()`：在播放头位置切割贴纸。
*   `删除()`：移除选中的贴纸片段。
*   `镜像()`：水平翻转贴纸图像。

**贴纸叠加**是创造丰富效果的关键。你可以将多个贴纸拖拽到时间轴的不同轨道上，它们会同时显示在视频画面上。例如，将一个箭头动画贴纸叠加在一个静态形状贴纸之上。你可以分别选中每个贴纸轨道，独立调整它们的大小、位置和出现时间。请注意，叠加贴纸时不宜过多过乱，以免影响内容主体。

## 为贴纸添加动画效果

静态贴纸有时略显单调，为其添加动画能让内容更生动。本节我们学习如何为贴纸设置动画。

选中贴纸轨道，在右侧参数面板中可以找到动画选项：
*   **入场动画**：设置贴纸如何出现，例如“淡入”。添加后，贴纸轨道起始处会显示一个箭头标志，代表入场动画。
*   **出场动画**：设置贴纸如何消失，在贴纸轨道的末端添加效果。
*   **循环动画**：为贴纸添加持续循环播放的动画效果。
