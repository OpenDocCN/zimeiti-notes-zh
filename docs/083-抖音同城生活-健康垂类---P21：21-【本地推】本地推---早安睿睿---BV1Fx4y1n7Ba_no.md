# 抖音同城生活-健康垂类0到1运营：P21：21-【本地推】本地推保姆级操作 📍

在本节课中，我们将详细学习巨量本地推的具体操作步骤。从账号入驻到创建推广计划，我们将一步步拆解，帮助你掌握这个强大的本地引流工具。

---

![](img/90f528561fed3462b5c5683954841ae1_0.png)

上一节我们介绍了本地推的基本概念，本节中我们来看看如何具体操作。使用巨量本地推的第一步，是完成抖音来客的入驻与资质认证。

当你完成门店主体的资质认证后，你的抖音主页会出现一个“本地推广”按钮。

点击这个“本地推广”按钮，即可进入相关页面。

进入页面后，你需要找到并点击“营销推广”选项。

以下是创建本地推广告的核心步骤与设置要点：

**1. 创建推广计划**
点击“新建推广”按钮，开始创建你的第一个广告计划。

**2. 选择推广目标**
系统会提供多个推广目标供你选择。对于本地商家，核心目标是吸引顾客到店。
*   **首选目标：** **门店推广**。这是最直接引导用户关注你门店、领取优惠、或导航到店的方式。
*   **备选目标：** 短视频/直播推广。如果你有优质的视频内容或正在进行直播，可以选择此目标来提升内容的热度，间接为门店引流。

**3. 设置推广内容**
你需要选择用哪条视频或哪场直播来进行广告投放。
*   从你的作品库中选择一条**数据表现最好**（如播放量、互动率较高）的视频。
*   确保视频内容与你的门店服务强相关，能清晰展示卖点。

![](img/90f528561fed3462b5c5683954841ae1_2.png)

**4. 设定投放模式与预算**
这是控制广告花费和效果的关键环节。
*   **投放模式：** 新手建议选择 **“控成本投放”** 。你可以设定一个期望的单个转化目标成本（如下单、到店的成本），系统会尽量在这个成本范围内帮你获取客户。
*   **预算设置：** 分为“总预算”和“日预算”。
    *   **公式：总预算 ≥ 日预算 × 计划投放天数**
    *   建议新手从较低的日预算（如100-300元）开始测试，根据效果逐步调整。

**5. 选择投放人群（定向设置）**
这是确保广告展示给潜在客户的核心步骤。本地推的优势在于强大的地理位置定向。
*   **基础定向：**
    *   **地域：** 选择你门店周边**3-5公里**的范围。这是本地服务的黄金辐射圈。
    *   **年龄、性别：** 根据你的目标客户画像进行选择（例如，养生服务可能更偏向中老年群体）。
*   **兴趣定向：** 可以添加“健康”、“养生”、“按摩”、“运动”等与你业务相关的兴趣标签，帮助系统找到更精准的用户。

![](img/90f528561fed3462b5c5683954841ae1_4.png)

**6. 完成支付与上线**
检查所有设置无误后，提交计划并完成支付。广告进入审核状态，审核通过后便会自动开始投放。

---

本节课中我们一起学习了巨量本地推从入驻到创建完整广告计划的每一步操作。关键点在于：**完成来客认证**，以 **“门店推广”** 为核心目标，选择**优质视频内容**，设置合理的**成本与预算**，并利用**地理位置和兴趣标签**进行精准人群定向。掌握这些步骤，你就已经迈出了通过付费流量为本地门店高效引流的第一步。
