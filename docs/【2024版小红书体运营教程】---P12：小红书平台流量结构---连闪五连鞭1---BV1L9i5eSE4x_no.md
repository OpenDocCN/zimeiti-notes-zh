### 平台首页主要流量入口

上一节我们介绍了小红书运营的整体框架，本节中我们来看看流量从哪里开始。打开小红书App，首页是流量的起点，主要有五个核心入口。

以下是首页的五个主要流量入口及其特点：

1.  **发现**：这是默认打开的首页标签，流量入口最大。系统会根据算法为用户推荐可能感兴趣的笔记。
2.  **关注**：展示用户已关注的博主发布的内容，流量取决于用户的关注列表质量和活跃度。
3.  **附近**：基于地理位置推荐内容，对于本地生活、探店等类目比较重要。
4.  **搜索框**：流量第二大的入口。用户主动搜索关键词，意图明确，转化潜力高。
5.  **商城**：直接导向商品购买页面，是电商流量的核心入口之一。

> **注**：课程中展示的首页截图板块可能已更新，当前版本可能有所不同，但以上五个核心入口依然存在。

### 笔记内的流量转化入口

了解了首页的流量来源后，我们深入到单篇笔记内部，看看如何将浏览笔记的流量引导至店铺。在笔记中，主要有两个地方可以插入导流链接。

以下是笔记内的关键导流位置：

*   **笔记正文中**：可以直接插入商品链接，用户点击即可跳转。
*   **笔记图片中**：可以通过添加标签（Tag）的形式插入链接。无论是在图片上打标签，还是在正文中插入商品卡，都会在图片下方展示商品信息。

**代码/形式示例**：
```
【正文推荐】这款包包真的很百搭！点击这里购买 -> [商品链接]
【图片标签】在图片上添加“同款商品”标签，链接至商品详情页。
```
当用户点击这些链接或标签时，便可直接进入你的小红书店铺完成导流。

### 搜索框流量的细分入口

搜索框作为第二大流量入口，其内部结构也值得细分研究。当用户使用搜索功能时，结果会分为三类，对应不同的流量路径。

以下是搜索关键词后的三个主要结果分类：

1.  **全部（默认笔记流）**：展示与关键词相关度高的笔记。优化笔记标题、内容和标签，提升在“全部”中的排名，可以获得可观的被动流量。
2.  **商品**：直接展示与关键词相关的商品列表。优化商品标题和关键词，有助于在这里获得曝光，直接为店铺引流。
3.  **用户**：展示名称或简介中包含关键词的博主。这是寻找潜在合作达人（KOL）的重要途径，可以通过搜索联系他们进行合作。

### 店铺流量来源构成分析

理论需要结合实际数据。我们来看一个店铺的真实流量来源分布，这能直观地告诉我们应该把运营重点放在哪里。

下图展示了一个店铺近30天的流量来源占比：

```
流量来源占比：
- 社区笔记推荐：73%
- 搜索流量：8%
- 其他入口（如商城、关注等）：19%
```

**核心公式**：
**社区笔记流量 + 搜索流量 ≈ 总流量的80%**

这个数据清晰地表明，对于大多数店铺而言，通过发布优质笔记获取社区推荐流量，以及优化关键词获取搜索流量，是运营工作的重中之重。当然，不同类目（如美妆、家居、服饰）的搜索流量占比会有所差异，但“笔记+搜索”的双引擎策略是普遍有效的。
