# 视频号创作分成计划：P19：活动介绍与官方扶持详解 🎬

在本节课中，我们将详细拆解视频号“商家联盟”活动的具体内容、官方提供的重磅扶持政策以及参与活动的核心步骤。你将了解到如何借助官方资源，从零开始轻松参与视频带货。

## 活动背景与核心理念

![](img/8e6e5775c762b3fa4720d07b8a3ce480_1.png)

为促进2024年中国行业的新发展，并为视频化带货电商奠定技术基础，平台特推出“商家联盟”活动。本次活动由短视频教研部负责人张良老师全程对接。

![](img/8e6e5775c762b3fa4720d07b8a3ce480_2.png)

本次活动以“公平公正”为核心理念展开。张老师作为本次活动的全国总负责人，拥有独家对接权限。这意味着，只有通过张老师这里才能对接本次活动，其他地方无法对接。

## 活动奖励与结算方式

![](img/8e6e5775c762b3fa4720d07b8a3ce480_13.png)

了解了丰富的资源扶持后，我们来看最直接的激励部分——活动奖励。本次活动的奖励力度空前，并采用高效的结算方式。

官方为鼓励创作，设置了保底奖励：**每发布一条视频，即可获得300元奖励**。奖励结算为“日结”模式，即今天发布视频，明天奖励到账。

根据奖励规则计算：
```
单条视频奖励 = 300元
总奖励（30条）= 300元/条 * 30条 = 9000元
```
这意味着，完成30条视频的发布任务，总计可获得9000元保底奖励。

![](img/8e6e5775c762b3fa4720d07b8a3ce480_15.png)

## 活动权威性与时效性

本次活动由微信视频号官方发起，并指定张良院长作为全国总负责人进行全权对接。活动具有明确的时效性，并非长期开放。

活动关键日期如下：
*   **活动开幕**：1月11日
*   **活动宣讲**：1月12日
*   **活动正式开始**：1月13日
*   **活动截止**：8月10日

需要特别注意，活动即将结束，机会有限。奖励人人可拿，但前提是必须完成“发布视频”并“挂上商品链接”的操作。这要求参与者仍需学习基本的实操步骤。

![](img/8e6e5775c762b3fa4720d07b8a3ce480_17.png)

## 实操步骤指引

![](img/8e6e5775c762b3fa4720d07b8a3ce480_18.png)

最后，我们进入实操环节。参与活动并获得奖励，需要遵循正确的操作流程。

以下是参与活动的核心操作步骤概览：
1.  通过指定渠道报名参与活动。
2.  从官方素材库下载分配的视频素材。
3.  将视频发布至自己的视频号，并正确挂载商品链接。
4.  等待奖励按日结方式发放。
