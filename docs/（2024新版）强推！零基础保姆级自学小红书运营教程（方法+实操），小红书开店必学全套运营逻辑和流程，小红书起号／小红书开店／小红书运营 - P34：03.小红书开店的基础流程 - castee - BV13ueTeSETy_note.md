## 个人店开通基础

![](img/d3a3ff1c26f7f62099156caef599eacf_7.png)

![](img/d3a3ff1c26f7f62099156caef599eacf_9.png)

开通个人店可以通过手机或电脑完成。手机端可以使用小红书App或专门的商家管理工具“小红书千帆”。以下是开通个人店的核心要求：

![](img/d3a3ff1c26f7f62099156caef599eacf_11.png)

![](img/d3a3ff1c26f7f62099156caef599eacf_13.png)

*   **身份要求**：开店人需提供**身份证**和**银行卡**，且两者必须属于**同一个人**。后续店铺收益将提现至该银行卡。
*   **重要提醒**：由于涉及资金结算，请务必使用自己或非常信任的亲友的身份信息注册，以避免后续可能的经济纠纷。

![](img/d3a3ff1c26f7f62099156caef599eacf_15.png)

![](img/d3a3ff1c26f7f62099156caef599eacf_17.png)

在开店过程中，部分信息可以灵活处理，以快速完成店铺创建：

![](img/d3a3ff1c26f7f62099156caef599eacf_19.png)

*   **经营类目**：前期可**随便选择一个类目**，目的是先将店铺开通。后续可以随时修改经营类目。
*   **店铺名称**：店铺名字可以**先随便取一个**，后续支持修改（通常每7天可修改一次）。
*   **经营地址**：可以**随便填写**，不影响开店。
*   **保证金**：开店过程中，**保证金可以先跳过不交**。待店铺正式运营前再根据平台要求缴纳即可。

![](img/d3a3ff1c26f7f62099156caef599eacf_21.png)

## 个人店开通步骤 📱

![](img/d3a3ff1c26f7f62099156caef599eacf_23.png)

![](img/d3a3ff1c26f7f62099156caef599eacf_25.png)

了解了基础要求后，我们来看具体的操作步骤。以下是详细的图文指引：

![](img/d3a3ff1c26f7f62099156caef599eacf_27.png)

![](img/d3a3ff1c26f7f62099156caef599eacf_29.png)

1.  **第一步：搜索开店入口**
    打开小红书App，在顶部的**搜索栏**中输入“**开店**”两个字进行搜索。

![](img/d3a3ff1c26f7f62099156caef599eacf_31.png)

![](img/d3a3ff1c26f7f62099156caef599eacf_33.png)

2.  **第二步：进入官方开店页面**
    在搜索结果中，找到并点击“**小红书电商官方平台**”提供的“**开店**”入口。

![](img/d3a3ff1c26f7f62099156caef599eacf_35.png)

3.  **第三步：开始创建店铺**
    进入页面后，点击“**立即开店**”按钮。个人店开通是**零费用**的。

![](img/d3a3ff1c26f7f62099156caef599eacf_37.png)

4.  **第四步：选择店铺类型**
    在店铺类型选择页面，选择“**个人**”店。个人店只需身份证；若选择“个体店”，则需要额外提供营业执照。

    > **注意**：店铺类型后期可以相互转换（如个人店转个体店）。企业店涉及税务问题，需谨慎选择。

![](img/d3a3ff1c26f7f62099156caef599eacf_39.png)

5.  **第五步：填写资料并提交**
    *   点击“下一步”，如有服务商邀请码可以跳过。
    *   根据提示上传**身份证正反面照片**。
    *   在“主体类型”中，个人卖家通常选择“**属于零星小额交易，依法无需办理市场主体登记**”。
    *   填写必要的个人信息，部分非必填信息可暂时跳过。
    *   最后确认信息，点击“**立即开店**”。

![](img/d3a3ff1c26f7f62099156caef599eacf_41.png)

![](img/d3a3ff1c26f7f62099156caef599eacf_43.png)

正常情况下，信息填写无误，店铺即可**当天审核通过**。

![](img/d3a3ff1c26f7f62099156caef599eacf_45.png)

![](img/d3a3ff1c26f7f62099156caef599eacf_47.png)

## 关键注意事项与费用 💰

![](img/d3a3ff1c26f7f62099156caef599eacf_49.png)

![](img/d3a3ff1c26f7f62099156caef599eacf_51.png)

上一节我们完成了开店操作，本节中我们来看看开店后需要关注的重要规则和费用。

![](img/d3a3ff1c26f7f62099156caef599eacf_53.png)

*   **提现账户**：店铺营业额将提现至开店时绑定的身份证所属人的银行卡。
*   **店铺命名**：店铺名称为“**你的小红书账号名 + 的店**”。取名前最好先搜索查重，避免重复。
*   **保证金**：个人店保证金通常在**1000元人民币左右**，具体金额根据所选经营类目而定。
*   **平台技术服务费（扣点）**：
    *   平台对成交订单收取一定比例的技术服务费。
    *   **计算公式**：`月度技术服务费 = (当月结算金额 - 1万元 - 退款金额) * 5%`
    *   **规则详解**：一个自然月内，店铺的订单结算总额（扣除退款后）**不超过1万元，则免收技术服务费**。超过1万元的部分，平台按**5%** 的比例收取费用。
    *   **举例**：假设某月店铺结算额为2万元，退款金额为0。则技术服务费为 `(20000 - 10000) * 5% = 500元`。

## 其他店铺类型与特殊类目

![](img/d3a3ff1c26f7f62099156caef599eacf_55.png)

除了个人店，小红书还支持其他类型的店铺，部分商品类目也有特殊资质要求。

![](img/d3a3ff1c26f7f62099156caef599eacf_57.png)

![](img/d3a3ff1c26f7f62099156caef599eacf_59.png)

**个体店/企业店开通**：
若想开通个体店或企业店，需要准备：
*   **个体店**：小红书实名账号 + **营业执照**（建议法人与小红书账号实名人为同一人）。
*   **开通方式**：通过手机下载“**小红书千帆**”（商家版App），或访问其官网进行注册开通。

**特殊经营类目资质要求**：
部分类目对资质有严格规定，以下是常见类目的要求：
*   **食品类**（如酒水、零食、水果）：需提供**营业执照** + **食品经营许可证** + **品牌授权书**。
*   **美妆类**：需提供**化妆品备案凭证**。
*   **家电/3C数码类**：需提供**3C认证证书**。

![](img/d3a3ff1c26f7f62099156caef599eacf_61.png)

> **建议**：如果不熟悉这些行业或无法提供完整资质，新手初期应尽量避免选择这些类目。

![](img/d3a3ff1c26f7f62099156caef599eacf_63.png)

具体的类目资质要求，可以在小红书商家后台的“类目管理”中查询。

![](img/d3a3ff1c26f7f62099156caef599eacf_65.png)
