# 视频号分成计划教程：1：项目介绍与原理

在本节课中，我们将要学习视频号广告分成计划的基本概念、项目原理以及开通条件。这是一个适合新手快速上手的项目，无需文案和复杂制作能力。

## 项目概述与收益展示

视频号分成计划是由视频号官方推出的项目。用户开通分成计划后，发布原创视频即可获得广告分成。我们通过发布明星混剪视频并声明原创，来获取官方的广告收益。

以下是我们的收益情况展示：

*   **老号收益**：一个运营了约一个月的老号，收益相对稳定，基本维持在300元左右，最高可达500多元，最低也有100多元。
*   **新号收益**：一个6月底新启动的账号，收益同样可观，基本稳定在三位数。

## 项目运作原理

上一节我们介绍了项目的收益情况，本节中我们来看看它的具体运作原理。

视频号广告分成计划的本质是：创作者通过发布优质原创内容吸引用户，平台在视频评论区等位置插入广告，并根据广告的互动效果（如点击、评论）与创作者进行分成。

具体到我们的操作，就是制作并发布**明星混剪视频**。当用户观看视频并看到下方的广告时，他们的互动行为就会为我们产生收益。

广告通常出现在视频评论区靠前的位置（例如第五条左右）。虽然广告内容可能不同，但多数视频下都会有此类广告展示。

## 开通分成计划的条件

![](img/780c786dc8e1597048bef3232a6474c4_8.png)

![](img/780c786dc8e1597048bef3232a6474c4_9.png)

了解了原理后，我们需要满足特定条件才能开通分成计划并开始获得收益。

![](img/780c786dc8e1597048bef3232a6474c4_10.png)

![](img/780c786dc8e1597048bef3232a6474c4_11.png)

以下是官方要求的三个开通条件：

![](img/780c786dc8e1597048bef3232a6474c4_12.png)

1.  **有效关注100人以上**：必须通过自然方式涨粉至100人。使用非正规手段刷粉通常无法通过审核。
2.  **符合内容规范**：发布的内容需要遵守平台规则。
3.  **发布优质原创视频**：必须声明原创并发布优质内容。

针对第一个条件“有效关注100人以上”，这里提供一个快速涨粉建议：发布**情感类自拍视频**。这是官方推荐的领域，涨粉速度较快。如果不想出镜自拍，也有其他方法，但核心是必须满足以上三个条件才能成功开通。
