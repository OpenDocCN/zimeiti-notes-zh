# 小红书运营教程：P28：小红书笔记引流：平台流量结构 📊

![](img/9c8302072fd1bfa356ea1e9d317fb7bc_1.png)

在本节课中，我们将要学习小红书平台的整体流量结构。了解流量从哪里来，以及不同流量入口的特点，是制定有效运营策略的基础。掌握这些知识，能帮助我们明确笔记的投放重点，从而更高效地获取流量。

## 什么是流量结构？

上一节我们介绍了笔记引流的基础概念，本节中我们来看看平台流量的具体构成。流量结构指的是小红书平台流量的整体分布和来源。理解流量结构，能让我们知道将笔记投放到何处，以及能获得何种效果。

小红书的整体流量结构主要分为三个部分：**发现页**、**关注页**和**搜索页**。

## 三大流量入口详解

以下是小红书三个核心流量入口的详细介绍。

### 1. 发现页：核心流量池 🔥

发现页是整个小红书最重要的流量来源，约占平台总流量的 **70%**。它也是平台进行流量层级测试的主要阵地。一篇笔记能获得多大规模的曝光，很大程度上取决于它在发现页获得的用户反馈。

**核心机制**：账号权重会影响笔记的基础曝光量。每发布一篇笔记，系统会根据账号权重分配一个基础曝光量（例如100、2000等）。这70%的基础曝光量几乎全部集中在发现页。新账号发布的笔记，其初始流量基本都来自发现页。关注页和搜索页的流量则更倾向于服务那些长期在平台浏览、有明确目标的成熟用户。

![](img/9c8302072fd1bfa356ea1e9d317fb7bc_3.png)

### 2. 搜索页：精准流量与高转化 💎

搜索页的流量占比仅次于发现页，约为 **20%**，并且是小红书流量增长最快的板块。用户通过搜索框（放大镜图标）主动寻找商品或内容，意图非常明确。

**核心特点**：虽然搜索页流量占比为20%，但其产生的消费转化却可能占平台总成交的 **70%** 甚至更高。用户带着明确需求来搜索，因此转化率极高。然而，搜索流量并非新手初期可以轻易获取的，它依赖于商品笔记的权重、关键词优化、历史成交和评价等综合数据。这部分流量是运营后期需要重点攻坚的方向。

### 3. 关注页：精准但量小的私域流量 👥

关注页的流量占比最小，通常不到 **10%**。这部分流量来自已关注你的用户，虽然量小，但用户最为精准，兴趣匹配度高。

**核心机制**：在关注页产生的浏览、互动等数据，**不计入影响笔记爆款与否的初级权重考核**。也就是说，即使你有大量粉丝，笔记发布后仅靠粉丝在关注页阅读，也难以成为爆款。它更多是用于维护核心用户、进行深度互动和引导至其他平台（如微信群）的渠道。运营顺序应是优先做好发现页，再优化搜索页，最后经营关注页。

## 关于“本地页”的特别说明

在APP首页，除了“发现”和“关注”，还有一个“本地”页面。该页面主要展示基于地理位置的内容。

*   **适用场景**：适合户外、美食探店等地域性强的博主，用于内容创作和素材搜集。
*   **运营警告**：对于大多数以销售商品、全国引流为目标的账号，**应尽量避免使用本地页进行推广**。因为地域性推送会打乱账号的全国性标签，将内容局限在局部区域，不利于账号做大。除非你的商业模式就是本地配送服务（如本地熟食店）。

## 总结与下节预告

本节课中我们一起学习了小红书平台的三大流量结构：**发现页（70%，核心流量池）**、**搜索页（20%，高转化区）** 和 **关注页（<10%，精准私域）**。理解这个结构，能帮助我们在不同运营阶段找准发力点：前期主攻发现页获取曝光，中期优化搜索页提升转化，长期经营关注页维护用户。
