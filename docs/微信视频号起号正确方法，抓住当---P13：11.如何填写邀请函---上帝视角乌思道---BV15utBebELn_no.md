# 视频号运营教程：P13：11.如何填写邀请函

在本节课中，我们将学习如何正确填写视频号创作者计划的邀请函。这是加入计划、开启变现的关键一步，每一步都至关重要。

## 概述
填写邀请函是加入视频号创作者计划的必要流程。本节将详细拆解填写步骤，确保您能准确无误地完成信息提交，为后续的账号运营和收益结算打下基础。

![](img/1bf37641abdf43177aa9a5260c050ae0_1.png)

---

## 第一步：打开微信
首先，您需要打开手机上的微信应用程序。



![](img/1bf37641abdf43177aa9a5260c050ae0_3.png)

![](img/1bf37641abdf43177aa9a5260c050ae0_4.png)

![](img/1bf37641abdf43177aa9a5260c050ae0_1.png)

## 第二步：进入“发现”页
打开微信后，点击底部导航栏的“发现”选项。



![](img/1bf37641abdf43177aa9a5260c050ae0_3.png)

## 第三步：找到“视频号”
在“发现”页面中，向上滑动屏幕，找到并点击“视频号”入口。其图标通常是一个黄色的蝴蝶形状。



![](img/1bf37641abdf43177aa9a5260c050ae0_6.png)

![](img/1bf37641abdf43177aa9a5260c050ae0_7.png)

![](img/1bf37641abdf43177aa9a5260c050ae0_4.png)

## 第四步：进入个人主页
进入视频号后，点击页面右上角的个人头像图标，进入个人中心。



![](img/1bf37641abdf43177aa9a5260c050ae0_6.png)

## 第五步：查看账号信息
在个人中心页面，向下滑动，找到“我的视频号”区域。这里会显示您的**账号昵称**和**粉丝数量**。
例如，显示为：`天道酬勤，39917个粉丝`。



![](img/1bf37641abdf43177aa9a5260c050ae0_7.png)

上一节我们找到了账号信息，接下来我们开始正式填写邀请函的各个项目。

以下是邀请函需要填写的具体内容及方法：

1.  **视频号昵称**：填写您在第五步中看到的账号昵称，例如 `天道酬勤`。
2.  **粉丝数量**：填写您在第五步中看到的粉丝数，例如 `39917`。
3.  **活动邀请人**：根据您的专属渠道，填写指定的邀请人姓名，例如 `张良` 或 `张良老师`。
4.  **实名信息**：填写您本人的真实姓名，例如 `周建国`。
5.  **结算手机号**：填写用于接收验证码和后续结算的手机号码，例如 `13812345678`。注意号码间不要有空格。



![](img/1bf37641abdf43177aa9a5260c050ae0_9.png)

![](img/1bf37641abdf43177aa9a5260c050ae0_10.png)

![](img/1bf37641abdf43177aa9a5260c050ae0_11.png)

![](img/1bf37641abdf43177aa9a5260c050ae0_9.png)

![](img/1bf37641abdf43177aa9a5260c050ae0_10.png)

## 第六步：提交并确认
所有信息填写完毕后，仔细检查一遍，然后点击蓝色的“提交”按钮。
系统会弹出“确定要提交吗？”的确认框，点击“确认”即可完成提交。



![](img/1bf37641abdf43177aa9a5260c050ae0_11.png)

![](img/1bf37641abdf43177aa9a5260c050ae0_13.png)

提交成功后，页面会提示“已成功加入创作者分成计划”。

![](img/1bf37641abdf43177aa9a5260c050ae0_14.png)
