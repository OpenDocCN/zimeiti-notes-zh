# 抖音运营入门：P4：抖音短视频的前景 📈

![](img/6b7b2717012f22386b414da58e72116a_1.png)

在本节课中，我们将要学习抖音短视频平台的整体前景，分析其市场潜力、用户构成以及为何它是适合新手创作者起步的平台。

![](img/6b7b2717012f22386b414da58e72116a_3.png)

了解抖音短视频的前景，需要先通过一张图片来认识。

![](img/6b7b2717012f22386b414da58e72116a_1.png)

这张图片是抖音创作者大会的一份数据报告。报告显示，有超过2200万人在抖音的合计收入超过了417亿元。抖音希望将这个数字翻一番，让创作者的总收入达到800亿元。

![](img/6b7b2717012f22386b414da58e72116a_3.png)

这代表了巨大的流量扶持。这里介绍第一个知识点：抖音上有两类人。第一类人叫做“玩家”，他们单纯地玩抖音，没想过从平台上赚钱。另一类人，可以称作“创作者”，他们想通过制作抖音内容来获取收益。区分这两类人很重要，后续还会提到。

那么，为什么我们要选择抖音短视频呢？市面上有快手、火山等短视频平台，为何非得做抖音？这就像开实体店选址，你肯定会找一个流量大、位置好的地方。做短视频也一样，必须选择一个流量大的平台。如果平台自身流量小，就无法分给创作者，也就赚不到钱。

抖音每日的活跃用户超过8亿，月活跃用户更是达到了40亿人次。这是非常庞大的流量基础。

流量大，竞争压力会不会很大？答案是不会。平台上的创作者大约只有5000万。创作者并非特指网红达人。例如，邻居家的老太太用手机拍下自己做的饭传到抖音上，她也算创作者。因此，90%的竞争压力都来自于普通用户。

接着，做抖音短视频需要购买电脑或摄像机吗？不需要。只需要一部智能手机和一个手机号，就可以注册抖音账号。抖音倡导全民通过短视频的形式分享和记录生活，门槛很低。

第三个问题是，有什么样的风口值得我们去做？首先，抖音计划扶持1000名优质的艺术创作者，帮助他们涨粉超过100万。有的同学可能会说，自己不会创作。这不用担心，正是因为不会，才需要学习。虚心请教比自己厉害的人并不丢人。只要对方在你想要学习的领域有本事，他就是你的老师。谦虚好学是进步的关键。

市场庞大，竞争相对温和，行业还很容易产出内容。现在是大数据时代。大数据时代的特点是，平台会根据你的喜好推荐内容。你喜欢看什么，抖音就给你推荐什么。

有的同学是宝妈，不知道拍什么，可以拍宝宝或做的饭菜。有的同学开实体店，想引流，可以拍产品。有的同学是普通上班族，想分享生活，也可以拍。月活跃用户高达40亿，只要你拍的内容不触犯法律和道德红线，不是低俗内容，总有一部分人会喜欢上你。

为什么老师说一定会有人喜欢你，你却只有几十个粉丝呢？因为喜欢你的人可能还没有刷到过你的作品。对于新手来说，心情迫切想涨粉可以理解，但不要把涨粉作为第一个目标。第一个目标应该是先把流量和播放量做起来。播放量涨起来后，粉丝自然就来了。

本节课中我们一起学习了抖音平台的巨大前景、用户分类、低门槛特性以及大数据推荐机制。核心在于理解抖音是一个流量庞大、创作门槛低、且通过算法精准连接内容与观众的平台。对于初学者，首要目标是提升内容曝光和播放量，粉丝增长会随之而来。
