# 视频号运营教程：P46：视频剪辑基础流程 🎬

在本节课中，我们将学习视频号剪辑的一个基础流程。这个流程旨在通过简单的步骤，高效地处理视频素材，添加广告、画中画、特效等元素，最终完成一个适合发布的视频作品。

## 素材预处理 ✂️

![](img/ba2d19bb7fab1eff70be3a696974849b_1.png)

![](img/ba2d19bb7fab1eff70be3a696974849b_3.png)

首先，我们需要对原始视频素材进行一些基础处理。

![](img/ba2d19bb7fab1eff70be3a696974849b_5.png)

1.  **放大画面**：将视频画面稍微放大一点。
2.  **掐头去尾**：剪掉视频开头和结尾不需要的部分。在熟悉剧情或使用创作猫等工具时，可以快速定位剪辑点，避免反复观看浪费时间。
3.  **灵活裁剪**：不必完全按照原片时长，可以根据需要裁切中间的片段。

## 插入广告 📢

上一节我们介绍了素材的预处理，本节中我们来看看如何在视频中插入广告。

![](img/ba2d19bb7fab1eff70be3a696974849b_7.png)

广告通常加在视频中后段，例如八九分钟或更靠后的位置。如果感觉位置不合适，也可以调整到前面。

以下是插入广告的具体步骤：

![](img/ba2d19bb7fab1eff70be3a696974849b_9.png)

1.  在选定的时间点（例如某个表情或对话间隙）将视频分割。
2.  从素材库中选择广告素材插入。对于同一个产品，可以使用不同片段进行组合，增加变化。
3.  确保广告片段之间衔接流畅。可以使用“叠化”转场，并将叠化时长设置为2或3，使过渡更自然。
4.  广告总时长建议控制在1分半到2分钟。如果广告素材中出现价格信息，需要将其剪掉。

## 添加画中画与去重 🖼️

![](img/ba2d19bb7fab1eff70be3a696974849b_11.png)

插入广告后，为了丰富画面和进行视频去重，我们可以添加画中画图层。

![](img/ba2d19bb7fab1eff70be3a696974849b_13.png)

以下是添加画中画的方法：

1.  添加第一张画中画图片（通常使用自己的照片）。
2.  将图片的“混合模式”调整为“叠加”，并将透明度调到最低（2或3）。
3.  将图片放大至铺满全屏。
4.  用同样方法添加第二张画中画图片。
5.  为了进一步去重，可以添加一个“矢量图”素材（在素材库中搜索）。将其混合模式也设为“叠加”，并调整至铺满屏幕。
6.  如果画面长度不够，可以复制画中画图层直至铺满整个时间轴，多余部分则裁切掉。
7.  还可以添加自己的名字水印，并将其放置在边角位置，这也是去重的有效手段。

## 应用特效与滤镜 ✨

添加完画中画，我们可以为视频增加一些视觉效果。

以下是应用特效和滤镜的步骤：

1.  **添加特效**：可以添加1-2个画面特效。如果有VIP权限，可以使用更多特效。将特效的“作用对象”设置为对应的画中画图层。
2.  **添加滤镜**：为整个视频添加一个滤镜，强度建议调到1-3之间，不宜过高。如果视频中插入了广告，务必确保滤镜效果覆盖到视频结尾。

## 添加贴纸与最终导出 🏁

最后，我们进行收尾工作并导出视频。

![](img/ba2d19bb7fab1eff70be3a696974849b_15.png)

以下是最终步骤：

![](img/ba2d19bb7fab1eff70be3a696974849b_17.png)

![](img/ba2d19bb7fab1eff70be3a696974849b_18.png)

1.  **添加贴纸**：在视频的四个角分别添加贴纸，例如小蝴蝶等装饰元素，并将它们的时长拉至与视频结尾对齐。
2.  **分段处理（可选）**：如果想更复杂地处理，可以将视频每20秒分割一段，并为每段更换不同的背景底色。但为了效率，此步骤常被简化。
3.  **导出视频**：将视频以1080P分辨率导出。
4.  **过一遍秒剪**：将导出的视频导入“秒剪”App，不做任何修改，直接通过秒剪发布到视频号。这一步有助于提升视频流量。

![](img/ba2d19bb7fab1eff70be3a696974849b_20.png)

本节课中我们一起学习了视频号剪辑的基础流程，从素材预处理、广告插入、画中画去重，到特效滤镜应用和最终导出。掌握这个流程，你可以高效地完成一个符合平台要求的视频作品。记住，核心在于衔接自然与有效去重。
