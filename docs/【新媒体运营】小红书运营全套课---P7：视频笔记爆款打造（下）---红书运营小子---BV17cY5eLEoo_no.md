上一节我们介绍了视频笔记的基础概念，本节中我们来看看如何解读视频笔记的播放量数据。

![](img/2d4b49a00ca2ff50257d0262e4796f50_5.png)

一个视频笔记的播放长度主要分为四种类型：**有效播放**、**长播**、**完播**和**复播**。这四种类型分别对应着不同的创作考验和用户需求。播放时长直接决定了平台分配给笔记的流量多少。

从用户观看行为来看，视频笔记的跳出率到完播率主要可以分为五个关键时间段：

![](img/2d4b49a00ca2ff50257d0262e4796f50_7.png)

以下是视频播放的五个关键时间段及其应对策略：
*   **0-3秒**：抓住用户注意力，建立观看期待。
*   **3-7秒**：持续进行内容铺垫，维持有效期待。
*   **7-20秒**：内容应一波三折，突破用户原有期待。
*   **20秒至视频结束**：让用户因投入的“沉没成本”而获得满足感。
*   **播放完毕至复播**：内容应令人意犹未尽，促使用户反复观看以满足需求。

![](img/2d4b49a00ca2ff50257d0262e4796f50_9.png)

根据这五个时间段来精心安排视频内容，视频笔记的流量数据会得到显著提升。



![](img/2d4b49a00ca2ff50257d0262e4796f50_11.png)

理解了播放量的构成后，接下来我们学习如何具体提升视频的热度。

以下是提升视频热度的四个实用技巧：
*   **快速入题**：视频开头先公布结果或抛出核心亮点，以引起用户的好奇心。
*   **设置反转**：在视频情节中巧妙添加转折效果，提高内容的可看性和吸引力。
*   **善用音乐**：学会用背景音乐来提升观众对视频的期待值。例如，美妆类产品可以使用轻快活泼的音乐，而茶具类产品则更适合使用舒缓类音乐。
*   **提升画面质感**：视频笔记比图文更注重画面质量。提升画面清晰度、构图和色调，使其看起来更加高级。



![](img/2d4b49a00ca2ff50257d0262e4796f50_13.png)

掌握了提升热度的技巧，我们还需要关注衡量视频质量的四个核心数据指标，即优质短视频的四大要点。

我们可以在小红书的数据分析后台查看视频的**完播率**。通过优化以下四个维度，可以有效提升视频热度，从而提高完播率。

![](img/2d4b49a00ca2ff50257d0262e4796f50_15.png)

以下是优质短视频的四大核心数据指标及其优化方法：
*   **点赞率**：其计算公式为 `点赞率 = 点赞量 / 播放量`。提升点赞率的关键在于引发粉丝共鸣，让粉丝觉得视频“有用”或“质量高”。
*   **评论率**：其计算公式为 `评论率 = 评论量 / 播放量`。提升评论率的重点在于设置具有互动性的标题或话题，大家可以搜索“商品笔记标题策划”查看相关课程。
*   **转发率**：其计算公式为 `转发率 = 转发量 / 播放量`。提升转发率的重点在于视频内容要有足够的价值，让用户产生强烈的分享欲望。可以在视频中多分享干货内容来提高转发量。



![](img/2d4b49a00ca2ff50257d0262e4796f50_17.png)
