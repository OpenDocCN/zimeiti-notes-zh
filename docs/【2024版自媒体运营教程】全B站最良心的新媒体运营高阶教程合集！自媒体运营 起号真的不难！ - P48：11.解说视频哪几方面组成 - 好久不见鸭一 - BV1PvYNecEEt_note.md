## 解说视频的构成

![](img/0e419f35aa518ca7cbe1a061876bea5f_3.png)

在讲解如何制作解说视频之前，我们首先需要明确什么样的视频属于解说视频，以及它由哪几方面组成。

![](img/0e419f35aa518ca7cbe1a061876bea5f_1.png)

我们刚才看到的这个视频，就是一个非常典型的解说类视频。它主要由三个方面组成。

![](img/0e419f35aa518ca7cbe1a061876bea5f_3.png)

解说类视频由以下三方面组成：
1.  **画面/视频**：这是视频的视觉主体。
2.  **文案**：这是视频的解说词内容。
3.  **配音**：这是对文案进行的有声演绎。

## 解说视频的素材来源

上一节我们介绍了解说视频的三个组成部分，本节中我们来看看如何获取其中的核心——文案素材。

对于不会自己写文案或不想写的同学，我们可以去寻找现成的素材来使用。这里有一个关键问题：我们本身做的是视频解说，还能不能去找其他视频的文案来用？

**答案是不能。**

因为这样做与“洗稿”没有区别，存在内容重复和版权风险。那么，我们应该去哪里寻找素材呢？这里需要运用逆向思维。

**逆向思维公式：**
```
视频解说 → 不找视频文案 → 寻找文章或音频
```

我们可以去寻找文章或音频来作为文案素材。这是因为在大多数内容平台，文章和视频是分开审核的，它们之间通常不会出现内容重复度的判定。

这个小技巧是一个独家经验。在早期运营自媒体时，作者不仅做视频，也写文章。当账号增多、原创文章跟不上发布需求时，作者尝试将音频平台（如喜马拉雅、企鹅FM）的内容提取成文字，用作文章素材，成功增加了作品数量且避免了内容重复。

后来，作者将这个方法应用到视频解说领域，同样取得了不错的效果。通过这种方式，可以有效地解决文案来源问题。

![](img/0e419f35aa518ca7cbe1a061876bea5f_5.png)

## 总结

本节课中我们一起学习了：
1.  解说视频的三个核心组成部分：**画面**、**文案**和**配音**。
2.  获取解说文案的**逆向思维方法**：避免直接搬运其他视频文案，转而从文章或音频中寻找素材，以规避内容重复问题。

![](img/0e419f35aa518ca7cbe1a061876bea5f_5.png)

掌握这些基础要素和素材寻找技巧，是制作解说视频的第一步。
