# 微信视频号教程：P40：3. 如何制作直播间贴图 🎨

在本节课中，我们将学习如何为微信视频号直播间制作一个透明背景的贴图。这种贴图常用于展示商品信息、促销活动或品牌标语，能有效吸引观众注意力并传递关键信息。

上一节我们介绍了直播间的其他设置，本节中我们来看看如何制作并添加这种视觉元素。

## 概述：贴图的作用与工具选择

直播间贴图是一种悬浮在直播画面上的文字或图片层。许多直播间使用它来突出“好物推荐”、“限时优惠”等信息。制作这种贴图其实非常简单，无需专业软件或开通会员，使用一款名为“微麦水印相机”（原名“微商水印相机”）的免费应用即可完成。

## 第二步：选择贴图模板

进入应用后，请按照以下步骤操作：

以下是找到并选择合适模板的步骤列表：
1.  在应用首页向下滑动，找到并点击“水印库”选项。
2.  在弹出的窗口中选择“暂不开通”。
3.  点击“更多水印”，进入“水印模板”页面。
4.  在模板库中，找到“横竖文字”分类，并选择第一个或你喜欢的模板。

## 第三步：编辑贴图内容与样式

选择模板后，进入编辑页面。你可以自由修改文字内容、颜色和字体。

![](img/7a46221ac2b1bca6edb1709e46cd9d21_1.png)

以下是编辑贴图的具体操作：
*   **修改文字**：点击文本框，将示例文字替换为你需要的内容，例如“好物推荐”、“老国货品牌”以及具体的价格信息（如“9.9元一盒”）。
*   **更改颜色**：点击“颜色”选项，你可以为文字选择纯色、渐变色或其他你喜欢的颜色。
*   **调整字体**：点击“字体”选项，可以更换不同的字体样式。

![](img/7a46221ac2b1bca6edb1709e46cd9d21_3.png)

## 第四步：关键设置——制作透明背景

![](img/7a46221ac2b1bca6edb1709e46cd9d21_5.png)

这是最重要的一步，确保贴图背景透明，才能完美融入直播画面。

1.  找到“背景色”设置选项。
2.  背景色的选择中，**务必点击带有斜杠（/）标识的“无背景”选项**。这表示背景完全透明。
3.  确保所有背景相关的透明度参数都调整为0。
4.  设置完成后，点击“生成”按钮保存贴图。

![](img/7a46221ac2b1bca6edb1709e46cd9d21_7.png)

**核心概念**：透明背景意味着贴图只有文字部分可见，其后方区域是镂空的，不会遮挡直播画面。其效果可以类比为网页设计中的CSS属性：`background: transparent;`。

![](img/7a46221ac2b1bca6edb1709e46cd9d21_9.png)

## 第五步：在直播间添加贴图

贴图制作保存后，就可以在直播中使用了。

![](img/7a46221ac2b1bca6edb1709e46cd9d21_11.png)

以下是添加贴图到直播间的流程：
1.  在微信视频号中发起直播。
2.  进入直播画面后，找到添加贴图或“画面截图”的功能按钮（通常是一个“+”或图片图标）。
3.  选择“手机相册”，从相册中选取你刚刚制作并保存好的贴图图片。
4.  贴图会出现在直播画面上，你可以用手指拖动来调整其位置，或使用双指缩放来调整其大小。

![](img/7a46221ac2b1bca6edb1709e46cd9d21_12.png)

如果你对贴图的字号或排版不满意，可以返回“微麦水印相机”应用，重新编辑文字大小、间距等参数，再次生成并保存，然后在直播间更新贴图即可。
