# 抖音同城生活-健康垂类0到1运营：05：抖音来客基础设置 💳

![](img/a58df8a3d11d62a43bde29823f81edef_0.png)

![](img/a58df8a3d11d62a43bde29823f81edef_2.png)

在本节课中，我们将学习如何在抖音来客平台完成基础财务与订单管理设置。主要内容包括绑定收款账户、查看核销明细、管理退款、查看提现记录以及申请开票。这些设置是商家在抖音平台开展团购业务、进行资金结算和客户服务的基础。

![](img/a58df8a3d11d62a43bde29823f81edef_4.png)

## 一、绑定收款账户

![](img/a58df8a3d11d62a43bde29823f81edef_6.png)

收款账户是商家接收顾客订单款项的必备设置。只有在成功配置收款账户后，才能进行商品上架等后续操作。

![](img/a58df8a3d11d62a43bde29823f81edef_8.png)

收款账户支持绑定公司账户和门店账户。设置门店收款账户后，核销结算可分配至对应的分门店账户。公司和门店均支持绑定，操作可在抖音来客的PC端或App端完成。

![](img/a58df8a3d11d62a43bde29823f81edef_10.png)

以下是App端的操作步骤：

![](img/a58df8a3d11d62a43bde29823f81edef_12.png)

1.  **打开抖音来客App**：登录并进入首页。
2.  **进入收款账户设置**：点击“门店经营”，然后点击“收款账户”，进入收款账户列表页面。
3.  **填写账户信息**：输入银行卡号，并选择对应的开户银行。
4.  **选择开票协议**：选择是否与平台签订分账开票协议。
    *   如果选择“是”，分门店结算后可以门店主体开具发票。
    *   如果选择“否”，则只能以总公司主体开票。
5.  **提交审核**：阅读并勾选相关协议，点击提交审核。

![](img/a58df8a3d11d62a43bde29823f81edef_14.png)

平台审核时，会向您填写的银行卡号打入一分钱以验证卡号有效性。审核结果将通过短信或系统消息通知。

![](img/a58df8a3d11d62a43bde29823f81edef_16.png)

![](img/a58df8a3d11d62a43bde29823f81edef_18.png)

接下来我们看看PC端的操作步骤：

![](img/a58df8a3d11d62a43bde29823f81edef_20.png)

1.  **打开抖音来客PC端**：登录系统。
2.  **进入收款账户设置**：点击“收款账户”，进入收款账户列表。
3.  **填写并提交**：输入银行卡号、选择开户银行、阅读并勾选协议，最后提交审核。

## 二、查看核销明细

![](img/a58df8a3d11d62a43bde29823f81edef_22.png)

核销明细记录了顾客消费团购券的历史订单，方便商家进行对账和管理。此功能同样支持PC端和App端。

![](img/a58df8a3d11d62a43bde29823f81edef_24.png)

以下是App端的查看方法：

*   点击“门店经营”下的“更多”，选择“核销明细”，即可查看历史核销订单。
*   您可以通过页面上方的核销时间、核销状态、商品等筛选条件快速查找特定订单。

![](img/a58df8a3d11d62a43bde29823f81edef_26.png)

**订单核销操作流程**：
在App首页，可以选择“扫码验券”（使用相机扫描二维码）或“数码验券”（输入12-16位纯数字券码）。
1.  选择适用的门店和需要核销的张数。
2.  核销操作后一小时内可以撤销。
> **温馨提示**：App端暂无下载明细功能，如需下载需到电脑端操作。

![](img/a58df8a3d11d62a43bde29823f81edef_28.png)

PC端查看核销明细的方法如下：
打开抖音来客PC端，登录后点击“订单管理”下的“核销明细”，即可查看和筛选历史核销订单。

![](img/a58df8a3d11d62a43bde29823f81edef_30.png)

## 三、管理退款订单

![](img/a58df8a3d11d62a43bde29823f81edef_32.png)

商家可以在App端处理用户的退款申请。
*   点击“门店经营”下的“更多”，进入“退款管理”。
*   页面会展示所有退款订单，并分为“待处理”、“已处理”和“仲裁中”三种状态，方便您进行分类操作。

![](img/a58df8a3d11d62a43bde29823f81edef_34.png)

## 四、查看提现记录

![](img/a58df8a3d11d62a43bde29823f81edef_36.png)

提现记录功能让商家清晰掌握资金流动情况，支持PC端和App端。

![](img/a58df8a3d11d62a43bde29823f81edef_38.png)

![](img/a58df8a3d11d62a43bde29823f81edef_40.png)

**App端操作步骤**：
1.  首页点击“门店经营”，进入“财务管理”，点击“提现记录”。
2.  搜索框支持按提现时间、账户、状态进行筛选。
3.  支持查看单条提现的详情和分账详情。
4.  如果因收款账户信息有误导致提现失败，支持修改信息后手动点击“重试提现”。

![](img/a58df8a3d11d62a43bde29823f81edef_42.png)

**PC端操作步骤**：
1.  首页点击“财务管理”，点击“提现记录”。
2.  同样支持通过提现时间、账户、状态进行筛选查询。

![](img/a58df8a3d11d62a43bde29823f81edef_44.png)

## 五、申请开票

![](img/a58df8a3d11d62a43bde29823f81edef_46.png)

![](img/a58df8a3d11d62a43bde29823f81edef_48.png)

商家可以为自己在平台的佣金支出申请发票。

![](img/a58df8a3d11d62a43bde29823f81edef_50.png)

**开票主体与类型**：
*   **谁能开票**：仅支持开通了团购功能的主体申请。
*   **开票类型**：支持增值税普通发票、增值税电子普通发票和增值税专用发票。

**开票操作步骤**：
1.  **申请开票资质（首次需要）**：在App首页点击“门店经营”，进入“自助开票”。首次进入需先申请开票资质。
2.  **填写资质信息**：选择发票类型并填写相关信息。
3.  **提交资质**：信息填写完毕后，点击“提交资质”。
4.  **申请开票**：返回“自助开票”页面，勾选需要开票的记录，点击“申请开票”。
5.  **确认提交**：确认开票信息无误后，点击“提交申请”。

![](img/a58df8a3d11d62a43bde29823f81edef_52.png)
