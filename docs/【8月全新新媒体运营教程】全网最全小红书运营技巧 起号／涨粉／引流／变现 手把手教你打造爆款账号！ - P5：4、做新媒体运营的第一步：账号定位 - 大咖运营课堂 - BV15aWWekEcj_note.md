## 什么是账号定位？

无论你是想做短视频、图文笔记还是直播，第一步都必须明确自己要做什么。这就是**账号定位**，即确定账号的垂直发展方向。例如，是成为家居博主、母婴博主，还是专注于摄影、校园生活、VLOG或宠物等领域。对选定赛道进行细致分析，并规划人设、商业内容和整体方向，就是账号定位的核心工作。

## 人设定位：让用户认识并记住你 🎭

![](img/0d0818bd622b4b4923a46f6f9ad081c6_62.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_64.png)

上一节我们介绍了定位的整体概念，本节中我们来看看定位的第一个核心部分：**人设定位**。其根本目的是让用户知道内容提供者是谁，有什么样的背景和经历。

![](img/0d0818bd622b4b4923a46f6f9ad081c6_82.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_84.png)

例如，一个家居博主如果时而分享家居好物，时而发布工作日常或生活碎片，用户就很难判断其专业性，不确定博主是否在家居领域有深耕经验，其内容是否具有参考价值。

人设定位的另一个关键点是**让用户记住你**。打造人设主要依靠以下要素：

![](img/0d0818bd622b4b4923a46f6f9ad081c6_98.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_100.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_104.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_106.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_108.png)

以下是打造人设的基础要素，通常被称为“账号五件套”：
*   **基础形象**：头像、名称、简介、背景图、封面风格。
*   **内容展现**：视频中的标准动作、固定的开场白或口头禅、统一的视频节奏或特效风格。

![](img/0d0818bd622b4b4923a46f6f9ad081c6_110.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_112.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_114.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_116.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_118.png)

许多成功博主会通过高度统一的视频风格，让用户一看到内容就能联想到他。例如，“这是那个讲汽车的博主”或“这是专门带娃滑雪的宝妈”。甚至特定的背景音乐也能成为人设记忆点。

![](img/0d0818bd622b4b4923a46f6f9ad081c6_124.png)

**为什么人设如此重要？** 在选定赛道后，建立一个鲜明的人设能显著提升账号的成长上限。它让用户关注你的理由非常明确，不会因为内容跳跃而失去兴趣。用户是基于明确的预期和价值承诺来关注你的。

![](img/0d0818bd622b4b4923a46f6f9ad081c6_134.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_142.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_146.png)

关于人设，有两点需要注意：
1.  **出镜有利于打造人设**：在短视频平台，通过口播、分享经验（心理、情感、成长等）出镜，能有效建立人设。相比之下，纯图文、不出镜的知识分享类账号（如考研、考公资料号）很难让用户记住背后的运营者。
2.  **人设并非绝对必要**：需要思考你的赛道或账号是否必须强人设。例如，影视解说类账号，用户记住的往往是“账号”本身或独特的解说风格，而非具体的“人”。

## 商业定位：思考未来的变现路径 💰

在明确了“我是谁”之后，我们需要思考“我如何赚钱”，这就是**商业定位**。这是启动前必须想清楚的问题。

![](img/0d0818bd622b4b4923a46f6f9ad081c6_174.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_180.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_182.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_184.png)

你需要考虑：在这个赛道长期投入后，有哪些可行的变现方式？变现前景如何？能否达到预期收益（如超过主业收入或成为稳定副业）？对于企业账号，则需要思考：内容方向能否有效获客、沉淀私域或促进产品营销？

![](img/0d0818bd622b4b4923a46f6f9ad081c6_186.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_188.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_190.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_194.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_196.png)

简而言之，商业定位的核心公式是：
**`投入（精力/时间） -> 账号成长 -> 变现结果`**
你必须确保这个链条最终能跑通。

## 内容定位：规划你的具体产出 📝

有了人设和商业目标，接下来就要规划具体生产什么内容，即**内容定位**。这决定了你的文案、视频脚本应该怎么写。

我们可以从三个层面来理解内容定位：

![](img/0d0818bd622b4b4923a46f6f9ad081c6_236.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_238.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_240.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_242.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_244.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_246.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_248.png)

以下是内容定位需要解决的三个核心问题：
1.  **内容主题**：我到底想表达什么？是分享知识、记录生活，还是表达观点？主题必须清晰，不能模糊不清。
2.  **内容吸引力**：我的内容如何吸引用户？例如，家居博主可以分享房间改造技巧、家居好物测评；穿搭博主可以聚焦于通勤、出游、职场等细分场景的穿搭。关键在于在垂直领域内找到有差异化、能切中目标用户痛点的创意。
3.  **平台推流逻辑**：内容如何获得平台更多的推荐？这取决于两点：
    *   **用户行为反馈**：内容发布后，初始流量池用户的互动数据（点赞、评论、分享、完播率）。
    *   **赛道热度**：你选择的内容细分方向，是否是当前平台内该领域用户最感兴趣的话题。例如，在穿搭领域，如果关注“穿搭技巧”的用户远多于“店铺测评”，那么前期做“穿搭技巧”内容更容易获得基础流量。

![](img/0d0818bd622b4b4923a46f6f9ad081c6_250.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_252.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_254.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_256.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_258.png)

![](img/0d0818bd622b4b4923a46f6f9ad081c6_260.png)

把内容定位这三点研究透彻，你后续的内容生产方向就不会出现大的偏差。

**版本一：专注宝宝养育**
*   **人设定位**：一位宝宝3岁的宝妈，分享3年来生娃、养娃、带娃的真实经历。
*   **商业定位**：围绕宝宝，推荐优质的纸尿裤、奶瓶、童装、食品等用品。
*   **内容定位**：解决宝妈们的急切痛点，如“孩子不爱吃奶粉怎么办？”、“宝宝睡眠不规律如何调整？”、“孩子挑食怎么办？”。

*   **人设定位**：一位产后体重暴涨30斤，并通过努力成功恢复身材甚至有所提升的宝妈。
*   **商业定位**：推荐产后身材恢复相关产品，如塑身衣、外用护肤品、身材管理计划等。
*   **内容定位**：分享产后修复标准动作、妊娠纹科学处理、快速恢复秘诀、产后情绪焦虑应对等。

本节课中我们一起学习了新媒体账号运营的基石——**账号定位**。我们系统地拆解了定位的三个核心组成部分：
1.  **人设定位**（🎭）：解决“我是谁”的问题，旨在建立专业性和用户记忆点。
2.  **商业定位**（💰）：解决“如何盈利”的问题，需要在起步前规划清晰的变现路径。
3.  **内容定位**（📝）：解决“我生产什么”的问题，需要明确主题、打造吸引力并契合平台推流逻辑。
