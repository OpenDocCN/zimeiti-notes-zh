## 第一步：指令设计与内容分析 🔍

![](img/6c93fd979e5432b88de88f1f87684193_18.png)

![](img/6c93fd979e5432b88de88f1f87684193_20.png)

![](img/6c93fd979e5432b88de88f1f87684193_22.png)

首先，我们需要给AI下达清晰的分析指令。指令应涵盖对爆款笔记的多维度拆解。

![](img/6c93fd979e5432b88de88f1f87684193_24.png)

![](img/6c93fd979e5432b88de88f1f87684193_26.png)

![](img/6c93fd979e5432b88de88f1f87684193_28.png)

![](img/6c93fd979e5432b88de88f1f87684193_30.png)

以下是给AI的核心分析指令要点：
1.  **分析爆款因素**：找出内容可能成为爆款的原因。
2.  **分析整体结构**：
    *   **开篇**：用什么方式吸引用户继续阅读。
    *   **中间**：提供了哪些内容来戳中用户痛点，提升停留时长。
    *   **结尾**：如何设置“钩子”引导用户互动（如私信、关注）。
3.  **分析关键词策略**：检查笔记是否结合中文分词法设计了关键词。如果有，指出关键词是什么、出现频率以及铺设结构。

![](img/6c93fd979e5432b88de88f1f87684193_32.png)

![](img/6c93fd979e5432b88de88f1f87684193_34.png)

![](img/6c93fd979e5432b88de88f1f87684193_36.png)

**示例指令（给ChatGPT）**：
```
我给你提供一篇小红书平台上的爆款笔记文案，请你分析：
1. 它的爆款因素有哪些？
2. 它的整体结构：开篇用什么吸引用户？中间用什么内容戳中用户痛点？结尾用什么方式引导用户互动或关注？
3. 分析该笔记是否有结合中文分词法设计关键词。如果有，请告诉我关键词是什么、出现频率和铺设结构；如果没有，请说明。
```

AI对多篇笔记的分析反馈总结如下：
*   **爆款因素**：情感共鸣、逆袭故事、提供明确备考计划/针对性建议、时间节点清晰、推荐具体资源与技巧、正能量鼓励、使用表情符号增加亲和力。
*   **结构**：开篇常以焦虑、担忧等情感或知名人物/话语引入，引发共鸣；中间分享个人经验、提供详细规划和建议；结尾进行鼓励并引导互动。
*   **关键词**：在提供的样本中，AI分析指出多数笔记没有明显设计特定关键词，词汇以直接、常用的为主。

## 第三步：整合分析与生成大纲 📋

在AI分析了多篇样本后，我们需要让它整合所有分析出的爆款因素，去重后形成一个清晰的列表。这有助于我们和AI共同明确“爆款笔记应包含哪些要素”。

![](img/6c93fd979e5432b88de88f1f87684193_81.png)

![](img/6c93fd979e5432b88de88f1f87684193_83.png)

![](img/6c93fd979e5432b88de88f1f87684193_85.png)

![](img/6c93fd979e5432b88de88f1f87684193_86.png)

接着，指令AI根据这些爆款因素，为新的“考研”笔记规划一个详细大纲。

![](img/6c93fd979e5432b88de88f1f87684193_88.png)

![](img/6c93fd979e5432b88de88f1f87684193_90.png)

**示例指令**：
```
请结合你前面分析出的爆款笔记因素，为我规划一篇关于考研的小红书笔记大纲。大纲需要细分到具体的小节点。在我确认大纲没问题后，再生成完整笔记。
```

![](img/6c93fd979e5432b88de88f1f87684193_92.png)

![](img/6c93fd979e5432b88de88f1f87684193_94.png)

AI生成的大纲示例如下：
1.  **引人入胜的开头**：用问题或话语吸引注意，介绍主题。
2.  **专业选择分析**。
3.  **详细时间规划**（分政治、英语、数学、专业课）。
4.  **时间管理与心态调整建议**。
5.  **互动引导**：鼓励提问、提醒关注。

![](img/6c93fd979e5432b88de88f1f87684193_96.png)

## 第四步：根据大纲生成完整文案 ✍️

![](img/6c93fd979e5432b88de88f1f87684193_98.png)

![](img/6c93fd979e5432b88de88f1f87684193_100.png)

![](img/6c93fd979e5432b88de88f1f87684193_101.png)

![](img/6c93fd979e5432b88de88f1f87684193_102.png)

大纲确认后，即可指令AI根据大纲和之前总结的爆款因素，生成完整的笔记文案。此时可以提出更细致的要求。

![](img/6c93fd979e5432b88de88f1f87684193_104.png)

![](img/6c93fd979e5432b88de88f1f87684193_106.png)

**示例指令**：
```
请按照上面确认的大纲，撰写具体的文案。要求：
1. 文案结构严格遵循爆款笔记因素。
2. 包含一个简单的“逆袭”小故事引入，引发读者共鸣。
3. 文案中融入适量的emoji表情。
4. 结合中文分词法，自然地融入“考研”、“备考”等关键词。
```

AI生成的文案会包含情感共鸣的开头、清晰的备考规划、具体的时间节点、资源建议以及鼓励互动的结尾，基本符合一篇合格运营文案的要求。当然，对于更资深的运营者，可以在此基础上提出更精细的优化指令。

## 进阶提示与工具 🚀

![](img/6c93fd979e5432b88de88f1f87684193_120.png)

![](img/6c93fd979e5432b88de88f1f87684193_122.png)

*   **模型差异**：使用更强大的模型（如GPT-4）或专门优化的工具（如OpenAI的GPTs），通常能获得更高质量、更稳定的输出。你可以为GPTs提供专属的风格指南、表情库等文件，打造更个性化的AI助手。
*   **指令优化**：与AI对话时，措辞的细微差别可能导致结果不同。例如，“包含丰富的emoji”和“包含emoji”可能让AI添加的表情数量不同，需要在实践中不断调整和明确你的意图。
*   **迭代优化**：AI生成的第一版文案通常可作为优质初稿。你可以在此基础上，提出具体修改意见（如“让开头更抓人”、“简化第二部分”），进行多轮迭代，使文案更精炼、丰富。

![](img/6c93fd979e5432b88de88f1f87684193_124.png)

![](img/6c93fd979e5432b88de88f1f87684193_126.png)

本节课中，我们一起学习了训练AI产出爆款笔记的完整流程：从设计分析指令、喂给AI样本内容，到整合爆款因素、生成大纲，最后产出完整文案。掌握这个方法，能有效利用AI辅助内容创作，提升运营效率。后续课程将继续分享AI在更多平台和场景下的运营技巧。
