# 小红书无货源电商教程：P30：小红书笔记引流：平台流量分发机制 🧠

![](img/1c07b75eab7692716212a75dc21edebf_1.png)

在本节课中，我们将要学习小红书平台的核心流量分发机制。了解这套算法，是提升笔记曝光和排名的关键。

---

上一节我们介绍了账号定位与内容规划，本节中我们来看看小红书是如何决定将你的笔记展示给哪些用户的。这套机制可以简称为 **CES评分系统**。

![](img/1c07b75eab7692716212a75dc21edebf_1.png)

## 核心算法：CES评分系统

小红书的流量分发并非随机，而是基于一套综合评分算法。这套系统的核心是 **CES**，即**点赞（C）、收藏（S）、评论（C）、转发（E）、关注（S）** 等用户互动行为。每一项互动都有其对应的分数：

*   **点赞**：1分
*   **收藏**：1分
*   **转发**：4分
*   **评论**：10分
*   **关注**：8分

**公式：单用户最高贡献分 = 1（点赞） + 1（收藏） + 4（转发） + 10（评论） + 8（关注） = 24分**

系统会计算你笔记所获得的平均CES分数，并以此作为排名依据。

## 流量分发的具体流程

理解了核心评分，我们来看看流量是如何一步步分发的。

### 1. 初始曝光与排名

小红书的信息流页面一次展示20篇笔记，用户下滑会刷新出新的20篇。你的笔记发布后，系统会为其打上标签，并推送给具有相应兴趣标签的用户，获得约 **1000次** 的基础曝光。

这20篇笔记会有一个内部排名（第1名到第20名）。你的笔记能排在第几名，直接取决于它在此次曝光中获得的CES评分。

### 2. 多轮推流机制

平台的推流不是一次性的，而是分批次进行。例如：

*   第一波推流：400次曝光
*   第二波推流：300次曝光
*   第三波推流：500次曝光

能否获得下一波更大的流量，取决于前一波曝光的数据表现。

以下是决定推流的关键数据指标：

*   **点击率**：用户看到封面和标题后，点击进入笔记的比例。这是所有后续互动的前提。
*   **互动率**：用户点赞、收藏、评论、转发的整体比例。
*   **完播率**：用户完整观看视频或阅读完图文笔记的比例。
*   **互动值**：账号作者与评论用户之间的回复互动情况。

**流程简述**：笔记获得初始曝光 → 用户点击（点击率） → 用户观看（完播率） → 产生互动（互动率、互动值） → 系统计算CES评分 → 根据评分决定是否给予下一轮、更大规模的曝光。

### 3. 阶梯式推荐

只要你的笔记在某一轮曝光中，各项数据（尤其是点击率和CES评分）优于同期其他笔记，系统就会将其推荐给更多用户，形成“数据达标 → 获得更多流量 → 数据再次达标 → 获得更大流量”的阶梯式增长。

即便是粉丝数不多的新账号，只要单篇笔记内容优质，互动数据好，也有机会通过这个机制获得大量曝光。

## 给初学者的核心要点

为了让初学者更好地抓住重点，以下是你在创作笔记时必须关注的核心：

1.  **封面与标题决定点击率**：如果用户不点击，后面的一切评分都无从谈起。封面和标题必须吸引人。
2.  **内容质量决定互动率**：用户点进来后，能否看完、是否愿意点赞评论，完全取决于内容是否对他们有价值、有吸引力。
3.  **积极互动提升互动值**：认真回复用户的每一条评论，这不仅能提升“互动值”指标，也能鼓励更多用户参与讨论。
4.  **平台公平，重在数据**：小红书的分发机制相对公平。一个新账号的优质笔记，通常也能获得 **150-200** 的初始曝光。能否跑出来，全靠内容带来的数据说话。

---

本节课中我们一起学习了小红书的流量分发心脏——CES评分系统与多轮推流机制。记住，**排名 = 点击率 + (点赞+收藏+评论+转发+关注)的综合数据**。你的内容质量直接决定了这些数据，进而决定了平台会给你多少流量。理解这套机制，你就能更有方向地优化自己的笔记。
