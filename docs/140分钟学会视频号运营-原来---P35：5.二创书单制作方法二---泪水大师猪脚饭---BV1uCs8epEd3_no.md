# 视频号运营：P35：5.二创书单制作方法二

在本节课中，我们将学习如何通过“二创”的方式制作书单视频。我们将详细拆解从获取原始素材、提取文案、替换画面到最终合成原创视频的完整流程。这种方法的核心在于对现有视频素材进行重组和优化，从而高效地产出符合平台要求的“新”内容。

## 素材获取与初步处理

![](img/d0477ca26a18817066683236a632c7e2_7.png)

首先，我们需要找到合适的原始视频作为创作基础。

以下是具体操作步骤：
1.  在抖音等平台搜索目标书籍名称，例如“走进大学城”。
2.  在搜索结果中，选择“商品”分类，并按“销量”排序，以找到相关的带货视频。
3.  进入达人主页，寻找可用的视频素材。例如，一个关于“汕头大学”的1分58秒视频，其文案前半部分和后半部分的配音风格不同。

获取视频链接后，需要使用去水印工具去除视频平台的水印，为后续剪辑做好准备。

![](img/d0477ca26a18817066683236a632c7e2_13.png)

## 文案提取与配音制作

得到去水印的视频后，下一步是提取其中的文案并生成新的配音。

以下是具体操作步骤：
1.  将视频导入剪映，在音频发生变化的位置（例如53秒处）进行分割，只保留我们需要的片段（如前半部分），然后导出该片段。
2.  打开“配音神器”等工具，通过“视频提取文案”功能，将上一步导出的视频片段转换为文字。
3.  在配音工具中，为提取出的文案选择合适的AI配音音色，然后导出生成的音频文件。

这样，我们就得到了一个全新的、无版权问题的配音文件。

## 视频画面替换与重组

这是二创的核心环节，即用新素材替换原视频的部分画面，实现“洗稿”。

以下是具体操作步骤：
1.  在剪映中重新导入原始视频。关闭原视频的音量，并将视频轨道拉长，便于精确到每一帧进行操作。
2.  识别并定位需要替换的画面片段。例如，一个持续3秒的“大学生毕业典礼”镜头。
3.  在抖音搜索与片段内容相关的关键词（如“高校 毕业”），下载找到的合适素材，并去水印。
4.  在剪映中，删除原视频中需要替换的3秒片段，将下载的新素材导入并裁剪至同样时长。
5.  在新旧片段之间添加转场效果（如“叠化”），并确保新片段的原声已被关闭。
6.  重复此过程，替换多个片段。例如，将“打高尔夫”的画面替换为搜索到的“美女打高尔夫”视频；将“宿舍单间”画面替换为“精装修室内”视频。替换1-2个核心画面即可显著改变视频观感。

## 音画合成与最终调整

将新配音与新画面进行合成，并完成最后的优化工作。

以下是具体操作步骤：
1.  将之前通过“配音神器”生成的新音频文件导入剪映，对齐视频轨道。
2.  检查音频与视频的长度是否匹配。如果不匹配，可通过 **`常规变速`** 功能调整视频播放速度，或增删视频素材来对齐时长。
3.  对于保留原声的片段（如视频后半部分），可以保留其声音，但替换其画面，并注意避免长时间出现特定人物面部，以减少侵权风险。
4.  为整个视频添加一个统一的背景音乐，提升观看体验。
5.  使用剪映的 **`识别字幕`** 功能，为新的配音生成字幕。通过调整字幕的样式、位置和大小，覆盖掉原始视频可能残留的旧字幕。
6.  最后检查视频，确保没有其他平台的水印（可通过放大画面遮盖），然后以 **`1080P`** 分辨率、**`60帧`** 的推荐设置导出视频。

## 总结与练习建议

本节课中我们一起学习了二创书单视频的完整制作流程：**获取素材 -> 提取文案 -> AI配音 -> 替换画面 -> 音画合成**。

![](img/d0477ca26a18817066683236a632c7e2_15.png)

剪辑技能的核心在于 **`熟能生巧`**。对于零基础学员，初期制作一个视频可能需要一两个小时，但熟练后通常可在10分钟内完成。关键在于动手实践，建议找到10个左右的视频素材，严格按照上述步骤逐一练习。剪映等工具功能直观，通过大量练习，剪辑水平必然会突飞猛进。请记住，成功的捷径正是持续且认真的实操。
