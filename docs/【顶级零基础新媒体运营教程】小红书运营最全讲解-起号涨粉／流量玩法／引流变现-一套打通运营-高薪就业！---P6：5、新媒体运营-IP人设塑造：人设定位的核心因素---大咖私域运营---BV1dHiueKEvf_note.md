### 1. 角色：明确你的专业身份
角色定位的核心在于，你需要根据所选赛道和内容方向，确立一个能让用户信服的专业身份。

![](img/55894649bd6954707e8388af3556365b_70.png)

![](img/55894649bd6954707e8388af3556365b_72.png)

![](img/55894649bd6954707e8388af3556365b_74.png)

![](img/55894649bd6954707e8388af3556365b_76.png)

![](img/55894649bd6954707e8388af3556365b_78.png)

*   **好物分享/测评赛道**：你需要展示你对产品的深入了解和真实体验。
*   **装修博主**：你需要证明自己确实拥有相关经验，例如展示正在装修或已装修好的房子，记录过程、分享避坑经验，并通过前后对比来体现专业性。公式可以概括为：**专业性 = 真实经历 + 过程记录 + 成果展示**。
*   **母婴赛道**：最常见的角色是“宝妈”，但也可以是“育儿护理专家”。后者不一定需要亲身育儿经历，但需要具备相关领域的专业知识或机构背景。
*   **情感赛道**：你可以是“情感大师”、“心理学顾问”，也可以是分享自身成长经历的“成长型人设”。例如，一位女性分享自己如何处理家庭、职场、创业中的问题，这种真实的故事也能塑造出鲜明的人设。

![](img/55894649bd6954707e8388af3556365b_86.png)

![](img/55894649bd6954707e8388af3556365b_88.png)

总而言之，**角色 = 你选择的赛道方向 + 对应的专家身份**。

### 2. 内容：保持垂直与价值
内容是人设的载体。关于内容定位，需要回顾几个核心原则：内容是否吸引用户、是否有价值、是否有平台特色。

![](img/55894649bd6954707e8388af3556365b_98.png)

![](img/55894649bd6954707e8388af3556365b_100.png)

![](img/55894649bd6954707e8388af3556365b_106.png)

![](img/55894649bd6954707e8388af3556365b_108.png)

在人设塑造的语境下，内容的关键在于**垂直与持续**。你必须持续输出与所选赛道强相关的内容。

![](img/55894649bd6954707e8388af3556365b_110.png)

![](img/55894649bd6954707e8388af3556365b_112.png)

![](img/55894649bd6954707e8388af3556365b_114.png)

![](img/55894649bd6954707e8388af3556365b_118.png)

```python
# 伪代码示例：内容垂直度检查
if 当前发布内容.所属赛道 == 账号定位.核心赛道:
    继续发布，强化账号标签
else:
    重新评估，避免内容混杂导致流量分发混乱
```

### 3. 形式：确定内容呈现方式
形式指的是你通过何种具体方式将内容和角色呈现给用户。主要包括以下几个方面：

![](img/55894649bd6954707e8388af3556365b_150.png)

![](img/55894649bd6954707e8388af3556365b_152.png)

![](img/55894649bd6954707e8388af3556365b_154.png)

![](img/55894649bd6954707e8388af3556365b_156.png)

![](img/55894649bd6954707e8388af3556365b_158.png)

*   **产出场景**：你的内容是在固定室内场景拍摄，还是在多变的室外场景？例如，在上班路上、车内、或是特定工作场景中分享。确定拍摄场景有助于形成统一的视觉风格。
*   **声音呈现**：你是亲自出镜讲解，还是只使用配音？这取决于你个人的表达能力和内容风格。现在通过“剪映”等工具实现高质量配音已非常容易。
*   **内容形式**：你的内容主要以哪种形式呈现？
    *   **剧情型**：创作难度较高，需要剧本和表演。
    *   **干货/经验分享型**：需要精心准备文案脚本，输出有价值的信息。
    *   **生活记录型**：需要积累大量的日常拍摄素材，真实感强。

### 4. 记忆点：打造深刻用户印象
记忆点是让用户快速记住并识别你的关键。它需要通过设计一些固定环节，反复强化用户的认知。

![](img/55894649bd6954707e8388af3556365b_184.png)

![](img/55894649bd6954707e8388af3556365b_186.png)

![](img/55894649bd6954707e8388af3556365b_188.png)

![](img/55894649bd6954707e8388af3556365b_190.png)

以下是几种常见的记忆点设计，它们可以归纳为两类：

![](img/55894649bd6954707e8388af3556365b_192.png)

![](img/55894649bd6954707e8388af3556365b_194.png)

![](img/55894649bd6954707e8388af3556365b_196.png)

![](img/55894649bd6954707e8388af3556365b_198.png)

![](img/55894649bd6954707e8388af3556365b_200.png)

![](img/55894649bd6954707e8388af3556365b_202.png)

**A. 前置强化型（在视频开头出现）**
*   **固定动作**：视频前几秒的特定手势或行为。
*   **固定特效**：开场的统一视觉特效。
*   **固定开场白**：每次视频开始时说的第一句话。

![](img/55894649bd6954707e8388af3556365b_204.png)

![](img/55894649bd6954707e8388af3556365b_206.png)

![](img/55894649bd6954707e8388af3556365b_208.png)

![](img/55894649bd6954707e8388af3556365b_210.png)

![](img/55894649bd6954707e8388af3556365b_212.png)

**为什么前置？** 当用户浏览你的多个视频时，每次开头都看到相同的元素，能产生强烈的重复记忆冲击，比将这些元素随机放在视频中间效果更直接。

![](img/55894649bd6954707e8388af3556365b_214.png)

![](img/55894649bd6954707e8388af3556365b_216.png)

![](img/55894649bd6954707e8388af3556365b_218.png)

![](img/55894649bd6954707e8388af3556365b_220.png)

![](img/55894649bd6954707e8388af3556365b_222.png)

![](img/55894649bd6954707e8388af3556365b_224.png)

**B. 全局贯穿型（在整个视频中体现）**
*   **统一BGM**：使用相同或风格一致的背景音乐，音乐响起就能让老粉丝联想到你。
*   **标志性物品**：画面中始终出现的某个特定物品，如特定的台灯、装饰画等。
*   **统一服装风格**：符合你人设的固定着装风格，例如职场风、田园风等。

![](img/55894649bd6954707e8388af3556365b_228.png)

**记忆点公式可以总结为：** **深刻印象 = 前置强化元素 + 全局贯穿元素**。
