## 账号类型解析

上一节我们介绍了小红书的整体运营框架，本节中我们来看看平台上的账号类型。小红书的账号主要可以分为三类。

以下是三类账号的详细说明：

*   **企业号**：主要作用是树立官方品牌形象。企业号拥有官方认证标志，可以展示品牌名称、地址等信息，并享有平台赋予的特权，便于开展商业活动。
*   **个人号**：这是目前大多数用户使用的账号类型。个人号运营灵活，可以由品牌员工或真实消费者运营，更容易塑造个人品牌特色，也适合品牌相关人物出镜。
*   **引流号**：这类账号在之前的课程中被称为“炮灰号”。引流号可以创建多个，主要用于测试内容或特定引流目的，通常无需精细运营，在紧急情况下也可作为备用小号使用。

## 企业号矩阵运营建议

了解了账号分类后，我们聚焦于如何系统化地运营。当前常见的推广方式是企业号、个人号或引流号自行发布笔记进行引流。我们建议打造一个协同运作的账号矩阵。

以下是构建企业号矩阵的四点核心建议：

*   **积极参与品牌活动**：品牌的企业号发起试用、团购等活动时，应使用多个小号积极参与，增加活动热度与真实感。
*   **个人号展示品牌属性**：运营个人号时，应在笔记内容中巧妙暗示品牌定位。例如，品牌目标用户是都市年轻人还是职场新人，需在内容中有所体现。
*   **控制产品发布频率**：个人号发布品牌产品相关内容时，频率不宜过高。建议遵循 **营销内容 : 种草/科普内容 = 3 : 7** 的比例，以维持账号的可信度与吸引力。
*   **利用引流号进行评论引流**：企业号发布内容后，可以使用引流号在评论区进行互动或引导，实现更自然的流量转化。

## 官方广告投放介绍

![](img/3b026ad9b89c3048ed2f9183c0e9204e_5.png)

除了运营自有的账号矩阵，另一种重要的推广方式是借助平台力量。接下来，我们看看小红书的官方广告投放体系。

![](img/3b026ad9b89c3048ed2f9183c0e9204e_7.png)

![](img/3b026ad9b89c3048ed2f9183c0e9204e_9.png)

小红书官方广告即竞价广告，主要分为两大类：

![](img/3b026ad9b89c3048ed2f9183c0e9204e_11.png)

1.  **信息流广告**：当用户浏览小红书首页推荐内容时，系统会在信息流中不定时插入广告。
2.  **搜索广告**：当用户使用关键词进行搜索时，系统会根据算法展示相关的广告内容。例如，用户搜索“考研资料”，系统可能推荐能量补充类商品。

![](img/3b026ad9b89c3048ed2f9183c0e9204e_13.png)

搜索竞价广告的核心逻辑是关键词竞价排名，这与百度的模式类似。广告主对特定关键词出价，出价不同，获得的广告位排名也不同。

目前，信息流广告位主要出现在搜索结果页的第3位、第13位和第23位。

## 官方广告申请流程

如果你想尝试官方广告，需要先完成企业号认证。

申请流程概括如下：
1.  访问小红书官方网站。
2.  在“业务合作”板块中找到“推广合作”。
3.  根据指引提交企业资质并支付审核认证费用。
4.  认证通过后，普通账号即升级为企业号，获得投放信息流与搜索广告的资格。

![](img/3b026ad9b89c3048ed2f9183c0e9204e_15.png)

官方广告投放涉及较高费用，但能直接获取平台推荐流量。在资金允许的情况下，与官方合作投放竞价广告是获取精准流量的有效途径。
