# 抖音同城生活-健康垂类0到1运营：P26：口腔团购组品攻略 🦷

![](img/6da43f78940841ba6f9f5e2b7b3e7a63_0.png)

在本节课中，我们将学习抖音平台口腔类目团购业务的准入规则、资质要求以及可上架的商品品类。这对于计划入驻抖音开展口腔健康服务的商家至关重要。

上一节我们介绍了健康垂类的整体框架，本节中我们来看看口腔这一具体类目的运营攻略。

## 准入规则与资质要求

截至本课程制作时，抖音平台已放宽口腔类目的内测名额限制。以下是具体的准入资质和要求清单。

以下是商家入驻需要准备的核心材料：

*   **医生资质**：需提供至少三名医生的医师执业证。
*   **机构资质**：需提供营业执照、医疗机构执业许可证。
*   **广告资质**：需提供医疗广告审查表及对应的广告成品样片。
*   **法人信息**：需提供法人身份证正反面，以及其手机号用于注册“抖音来客”（手机号可与执照注册号不一致）。
*   **合规记录**：要求一年内无重大医疗事故或相关负面舆情。

## 品牌与门店筛选门槛

除了基础资质，平台对品牌和门店也有特定的筛选标准。

以下是品牌入驻的具体门槛：

*   **门店数量**：品牌旗下POI（兴趣点，即门店）数量需**大于等于3**。这些门店可以不在同一个城市。
*   **商品品类**：上架的商品品类需**大于等于4**。
*   **保证金**：口腔类目需缴纳**20万元**保证金。

## 可上架团购商品品类

目前，平台允许上架的团购商品主要围绕口腔清洁与牙齿美容。

以下是当前准入的具体团品列表：

1.  **洗牙/洁牙**：
    *   超声波洗牙
    *   喷砂洗牙
    *   牙齿抛光
2.  **牙齿美白**：
    *   冷光美白
    *   皓齿美白
    *   植物美白

## 内容与带货规则说明

关于内容创作和销售推广，有两点重要规则需要注意：

*   **达人带货限制**：在每年3月15日（消费者权益日）之前，口腔类目**不允许**达人进行直播带货。商家只能通过官方账号（BGC）进行内容发布或自播。
*   **规则动态性**：3月15日后，平台可能会开放达人带货权限。但鉴于该日期前后的监管敏感性，涉及广告的行为需格外谨慎。

## 入驻后的平台支持

成功入驻并开通服务后，商家将获得平台的支持。

以下是平台提供的主要支持：

![](img/6da43f78940841ba6f9f5e2b7b3e7a63_2.png)

*   **运营支持**：会有专门的平台运营团队提供支持。
*   **IM功能**：审核通过后，商家可优先开通在线即时通讯（IM）功能，便于在线接待咨询、处理客诉。

本节课中我们一起学习了抖音口腔类目的入驻全流程。核心在于备齐**医生资质、机构证照、广告批文**，并满足**3家以上门店**的品牌门槛。当前可经营**洗牙和牙齿美白**两大类目，且需注意**315前的达人带货限制**。成功入驻后，商家将获得平台运营与工具支持。
