# 小红书运营教程：P2：第1节：小红书运营之账号定位上 📚

在本节课中，我们将要学习小红书平台的核心流量入口与推荐机制，并掌握一个能够高效引流和涨粉的账号定位方法。了解这些基础知识，是避免账号运营陷入困境、实现快速起号的第一步。

## 流量入口解析 🚪

![](img/8712dbe42bfd1d27823e712f0f04b988_1.png)

想要通过小红书获取流量，首先需要搞清楚小红书的流量入口有哪些，以及哪些是需要重点关注的。这样可以将精力集中在能快速见效的渠道上。

以下是一篇爆款笔记的用户来源数据分析，它清晰地展示了三大主要流量渠道的占比情况：

![](img/8712dbe42bfd1d27823e712f0f04b988_3.png)

*   **首页推荐**：95%
*   **搜索**：1%
*   **其他来源**：4%

这三个渠道就是我们在小红书运营中会经常用到的。接下来，我们逐一进行讲解。

![](img/8712dbe42bfd1d27823e712f0f04b988_5.png)

![](img/8712dbe42bfd1d27823e712f0f04b988_7.png)

### 1. 首页推荐
首页推荐是小红书最大的流量入口，它由以下几个部分构成：
*   **关注页**：用户关注博主后，博主发布的新笔记会通过这个入口推荐给用户。
*   **地区页**：系统会根据用户的地理位置定位，进行同城内容推荐。
*   **发现页**：这是所有流量入口中占据绝对主导地位的页面。对于一个新账号而言，发现页基本占据了百分之百的流量推荐比重。小红书的推荐机制也主要针对发现页进行。

![](img/8712dbe42bfd1d27823e712f0f04b988_9.png)

小红书的发现页采用“双瀑布流”的展现形式，即信息以左右两列图片流的方式呈现。这意味着，当我们的笔记出现在用户的发现页时，用户是否会点击，与我们的**标题、封面和选题**有直接关联。这部分内容我们将在后续课程中重点讲解。

![](img/8712dbe42bfd1d27823e712f0f04b988_11.png)

从引流角度讲，关注页和地区页带来的流量通常较少，几乎可以忽略不计。因此，运营的核心应聚焦于如何让内容在“发现页”获得推荐。

### 2. 搜索推荐
搜索推荐是指用户主动搜索关键词后，系统根据内容匹配度和笔记活跃度进行综合排序后呈现的结果。它与首页推荐的最大区别在于：首页是**系统主动推荐**，而搜索是**用户主动寻找**。

虽然在新账号初期，搜索带来的流量占比较低，但优化搜索排名对于获取精准用户和长尾流量至关重要。我们后续会专门开设加餐课，教大家如何优化搜索排名。

![](img/8712dbe42bfd1d27823e712f0f04b988_13.png)

### 3. 其他来源
其他来源主要指用户通过分享链接或直接打开笔记的方式访问。对于新号来说，这个渠道难以在短时间内带来大量流量，因此本节课不作重点讨论。

![](img/8712dbe42bfd1d27823e712f0f04b988_15.png)

上一节我们梳理了流量从哪里来，接下来我们需要了解平台是如何分配这些流量的。弄清推荐机制，就能明白一篇笔记是如何获得更大曝光的。

## 两大推荐机制 ⚙️

![](img/8712dbe42bfd1d27823e712f0f04b988_17.png)

小红书的推荐机制主要分为两大块：**个性化推荐**和**关键词搜索推荐**。

![](img/8712dbe42bfd1d27823e712f0f04b988_19.png)

### 1. 个性化推荐（用户消费推荐）
个性化推荐，也称为用户消费推荐。其核心逻辑是：系统通过分析用户的行为（如点击、关注、评论、收藏、停留时长、搜索等），为用户打上“喜好标签”，然后根据这些标签为用户推荐相关内容。

值得注意的是：
*   每个用户身上可能有多个标签。
*   系统会根据你的实时行为动态调整推荐内容。这就是为什么你给一篇笔记点赞后，平台会持续推荐类似内容给你。

除了用户标签，系统还会为每篇笔记打上“笔记标签”。系统会分析我们发布笔记的**标题、封面、文案和内容**，将其推荐给带有相应“喜好标签”的用户。然后，根据这些用户的反馈（互动数据）来评估笔记质量，决定是否将其推荐给更多人。

那么，这个推荐机制是如何运转的呢？我们可以用一个简单的流程来描述：

![](img/8712dbe42bfd1d27823e712f0f04b988_21.png)

1.  **发布与审核**：笔记发布后，首先进入系统审核阶段。
2.  **初始推荐**：审核通过后，系统会给予一个初始流量池（通常为100-200的曝光量）。
3.  **数据评估**：系统根据初始曝光下用户的**点击率、点赞、评论、收藏、停留时长**等数据，综合评估笔记质量。
4.  **流量升级**：如果数据达到标准，笔记就会被推荐到更大的流量池，获得更多曝光。反之，推荐则会停止。

**重要前提**：上述流程成立的条件是账号状态正常且笔记无任何违规。满足条件的账号，作品一般在3~6小时内能达到基础播放量。如果内容质量足够高，可能在20~30分钟内就能突破初始流量池。关于如何诊断账号是否正常，我们也会在加餐课中详细讲解。

![](img/8712dbe42bfd1d27823e712f0f04b988_23.png)

### 2. 关键词搜索推荐
这部分机制与“搜索推荐”流量入口相对应。当用户搜索时，系统会从海量内容中，根据关键词的相关性以及笔记的互动数据、新鲜度等因素进行排序，将最相关、最优质的结果呈现给用户。优化搜索排名的核心就在于围绕关键词进行内容布局和互动维护。
