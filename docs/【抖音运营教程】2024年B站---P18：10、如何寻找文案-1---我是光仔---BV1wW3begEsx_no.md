# 抖音运营教程：P18：10、如何寻找文案-1 📝

在本节课中，我们将要学习如何为抖音视频寻找和准备优质的文案。这是优化作品、吸引粉丝的关键第一步。很多新手卡在“不会写文案”这一步，本节课将提供几个实用的工具和方法，帮助你轻松解决文案来源问题。

---

上一节我们讨论了提升作品质量的重要性，本节中我们来看看如何具体解决文案创作这个难题。如果每天发布的视频内容不吸引人，即使发布频率很高，也难以获得好的流量和粉丝增长。因此，我们的策略是：**宁可降低发布频率，也要确保每条视频的质量**。而优质视频的起点，往往是一个能抓住人的文案。

很多同学表示不会写文案，这没有关系。我们可以借助一些现成的工具和平台来寻找灵感。以下是五个实用的文案素材网站，它们都是网页版，无需下载APP，在手机或电脑浏览器中即可直接使用。

1.  **淘金阁**
2.  （其他四个平台名称，因原文未具体说明，此处保留）

为了方便大家使用，这里提供淘金阁的网址。如果你在搜索引擎中直接搜索“淘金阁”发现广告太多，可以直接使用这个网址进入。

进入淘金阁主页后，你会看到它按类别收录了海量文案素材。以下是部分类别示例：

*   财经
*   军事
*   动漫
*   科技
*   美食美文
*   情感
*   三农
*   体育
*   育儿

![](img/eeb47276c9dcc02434a5fcec52f2425c_3.png)

类别非常丰富，几乎涵盖了所有热门领域。你可以根据自己的账号定位，点击相应类别进行查找。

![](img/eeb47276c9dcc02434a5fcec52f2425c_3.png)

例如，点击“情感”类别后，页面会展示大量相关话题文案，并且通常有多页内容可供浏览。这五个平台提供的素材，对于日常创作来说已经足够。

![](img/eeb47276c9dcc02434a5fcec52f2425c_5.png)

当你面对众多文案不知如何选择时，可以套用一个简单的公式来筛选：**一个特定群体 + 一个核心话题 + 一个疑问句式**。用这个标准去衡量，合适的文案就可以选用。

![](img/eeb47276c9dcc02434a5fcec52f2425c_5.png)

通过以上方法，你基本不会再为“没有文案”而发愁。接下来的问题可能是：在这些分类里找不到我想要的细分内容怎么办？

![](img/eeb47276c9dcc02434a5fcec52f2425c_7.png)

有两种解决方案。有条件的同学可以开通平台会员，使用更高级的搜索功能。不想开通会员的同学，可以注意页面下方的“输入多个关键词”搜索框。

例如，我们可以搜索“宝妈”这个关键词。

![](img/eeb47276c9dcc02434a5fcec52f2425c_9.png)

![](img/eeb47276c9dcc02434a5fcec52f2425c_7.png)

点击“开始抓取”后，系统会列出所有包含“宝妈”相关内容的文案。这个方法在手机和电脑上操作完全一致，非常方便。

![](img/eeb47276c9dcc02434a5fcec52f2425c_11.png)

![](img/eeb47276c9dcc02434a5fcec52f2425c_9.png)

![](img/eeb47276c9dcc02434a5fcec52f2425c_13.png)

解决了文案来源，还有一个常见问题：有的同学担心自己的声音不好听，不适合配音。淘金阁也提供了解决方案。

点击页面右上角的“小工具”，然后选择第四项“配音在线生成”功能。

![](img/eeb47276c9dcc02434a5fcec52f2425c_11.png)

这个功能完全免费，并且提供了极其丰富的音色选择。

![](img/eeb47276c9dcc02434a5fcec52f2425c_15.png)

![](img/eeb47276c9dcc02434a5fcec52f2425c_13.png)

![](img/eeb47276c9dcc02434a5fcec52f2425c_17.png)

以下是部分音色类型：

![](img/eeb47276c9dcc02434a5fcec52f2425c_19.png)

*   **女声类**：亲和女生、温柔女生、甜美女生、自然女声。
*   **男声类**：标准男声、情感解说男声。
*   **场景类**：卖场广播、直播配音。
*   **方言类**：东北话、四川话、粤语、台湾话。
*   **其他**：童声、英文配音等。

![](img/eeb47276c9dcc02434a5fcec52f2425c_15.png)

使用方法很简单：
1.  在左侧“配音文本”框中输入或粘贴你的文案。
2.  在右侧选择你喜欢的声音类型，例如“东北话”。
3.  调整音量、语速和语调等参数。

![](img/eeb47276c9dcc02434a5fcec52f2425c_17.png)
![](img/eeb47276c9dcc02434a5fcec52f2425c_19.png)
![](img/eeb47276c9dcc02434a5fcec52f2425c_21.png)

![](img/eeb47276c9dcc02434a5fcec52f2425c_21.png)

设置完成后，点击“立即生成”。在手机上使用时，可能需要用微信扫码登录一下，但功能依然是免费的。

![](img/eeb47276c9dcc02434a5fcec52f2425c_23.png)
![](img/eeb47276c9dcc02434a5fcec52f2425c_25.png)

![](img/eeb47276c9dcc02434a5fcec52f2425c_23.png)

生成后，在“配音生成列表”中，你可以直接下载生成的音频文件到手机。

![](img/eeb47276c9dcc02434a5fcec52f2425c_25.png)

![](img/eeb47276c9dcc02434a5fcec52f2425c_27.png)
