亚马逊BSR榜单分析：P1：如何查看BSR榜单100名之后的产品

在本节课中，我们将学习三种有效的方法，来查看亚马逊BSR（Best Sellers Rank）榜单中排名100名之后的产品。这对于进行深入的市场分析和选品调研至关重要。

上一节我们介绍了课程目标，本节中我们来看看第一种方法。

### 方法一：使用卖家精灵插件

此方法通过浏览器插件，直接扩展亚马逊官方榜单页面的显示范围。

操作步骤如下：

1.  安装并启用卖家精灵浏览器插件。
2.  进入您想查询的亚马逊产品类目的BSR榜单页面。
3.  点击插件功能区的“查看top400名产品”按钮。
4.  页面将跳转至该类目前400名产品的搜索结果页，产品默认按排名展示。
5.  打开插件的数据抓取功能，点击“加载更多”以获取更多产品数据。
6.  您可以将数据导出，并按“小类排名”字段进行排序，即可清晰查看100名之后的产品。

我们已经学会了通过插件快速抓取数据，接下来看看第二种利用卖家精灵网站功能的方法。

### 方法二：使用卖家精灵“选产品”功能

![](img/680fd849e5e44b04ff511b8da52456b7_1.png)

此方法在卖家精灵网站内部进行操作，利用其强大的筛选和排序系统。

![](img/680fd849e5e44b04ff511b8da52456b7_2.png)

以下是具体步骤：

1.  登录卖家精灵网站，进入“选产品”功能模块。
2.  在筛选条件中，选择您要分析的产品细分类目。
3.  点击“搜索”按钮。
4.  在结果列表中，找到排序选项，选择按“BSR排名升序”进行排列。
5.  排序后，您便可以浏览该细分类目下所有的产品，包括排名100之后的部分。



看完了网站内部工具的使用，最后我们学习一个更直接的技巧，通过修改网址参数来获取完整榜单。

### 方法三：修改URL参数直接访问

此方法通过解析和修改亚马逊页面的URL结构，直接访问包含全部排名的列表页。

操作流程如下：

![](img/680fd849e5e44b04ff511b8da52456b7_4.png)

1.  进入目标产品的亚马逊细分类目页面。
2.  从浏览器地址栏的URL中，复制代表该类目的一串数字ID。其格式通常类似于 `ref=zg_bsnr_XXXXXXXXXX` 中的数字部分。
3.  准备一个固定的链接模板：`https://www.amazon.com/b?node=【你的类目数字ID】`
4.  将模板中的 `【你的类目数字ID】` 替换为您刚复制的数字。
5.  在新页面中打开修改后的完整链接。
6.  在打开的页面中，找到排序筛选器，选择按“**Best Sellers**”排序。
7.  页面将展示该类目下所有按BSR排序的产品，您可以滚动浏览或分页查看全部列表。
