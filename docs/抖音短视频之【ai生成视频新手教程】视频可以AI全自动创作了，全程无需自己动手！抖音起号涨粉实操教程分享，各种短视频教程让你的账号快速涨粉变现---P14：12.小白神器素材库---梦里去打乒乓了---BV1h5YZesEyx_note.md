# AI短视频创作教程：P14：12.小白神器素材库 🧰

上一节我们介绍了AI视频创作的基本概念，本节中我们来看看如何具体获取和处理创作所需的素材。

## 素材库工具介绍

![](img/3e285befc67d6f85d0bba9c15ffd66b8_7.png)

首先，我们需要在微信小程序中找到并打开“一搜猫”工具。这是本节课的核心工具，它将帮助我们解决素材来源问题。

![](img/3e285befc67d6f85d0bba9c15ffd66b8_9.png)

![](img/3e285befc67d6f85d0bba9c15ffd66b8_11.png)

## 如何查找视频素材

![](img/3e285befc67d6f85d0bba9c15ffd66b8_13.png)

以下是查找视频素材的具体步骤：

![](img/3e285befc67d6f85d0bba9c15ffd66b8_15.png)

1.  **利用视频资源功能**：在“一搜猫”首页，可以使用“视频资源”功能。输入关键词（例如“沈腾”），即可搜索到全网与该主题相关的视频内容。
2.  **确定文章主题**：本节课的重点是使用“自媒体资源”功能。这里汇集了各平台的文章。首先，选择你账号所属的领域（如娱乐、影视）。
3.  **选择文章平台**：在领域下方，选择文章来源平台，例如“百家号”。系统会列出该平台的大量文章。
4.  **筛选可用文章**：浏览文章列表，寻找近期热门、你熟悉且适合改编成视频解说的主题。例如，可以选择关于热播剧《县委大院》的文章。
5.  **根据文章找视频**：复制文章的核心主题或剧名（如“县委大院”），然后打开“西瓜视频”APP进行搜索。
6.  **收集视频片段**：在搜索结果中，下载多个相关视频片段。**关键点**：每个片段时长最好控制在10秒左右，并且避免只使用单一作者的素材，以丰富视频内容并降低侵权风险。

通过以上步骤，视频素材部分就准备完成了。

## 如何获取与处理文案

视频素材找到后，下一步是准备解说文案。文案直接来源于我们刚才选中的文章。

以下是处理文案的几种方法：

*   **方法一：自主录音配音**
    *   将文章中的核心文字提取出来，形成解说稿。
    *   打开手机自带的“录音机”应用，亲自朗读文案进行录制。这能保证内容的绝对原创性。
*   **方法二：使用方言配音**
    *   如果担心普通话不标准，使用方言配音是一个极具特色和亲和力的选择。例如，使用四川话、东北话等，能让视频更具辨识度和亲切感。
*   **方法三：利用工具智能配音**
    *   如果不想自己配音，可以使用工具的智能配音功能。将文案复制到“一搜猫”的“智能配音”框中。
    *   在右侧选择喜欢的声音类型（如邻家小姐姐、萝莉音、御姐音等）。
    *   调整语速和音量后，点击“试听配音”。满意后即可生成音频文件。
*   **备选方案：其他配音工具**
    *   除了“一搜猫”，微信小程序中的“配音神器”也是一个专业选择。它是付费服务（约29元/月），但音质和选择可能更丰富。记住“小投资，大回报”的原则，根据自身情况选择。
