## 什么是主题IP
上一节我们介绍了内容定位的重要性，本节中我们来看看如何为内容找到一个核心的“钩子”，即主题IP。

![](img/f941d857afc073a19eb41ff8e4c1d05a_8.png)

首先，我们通过两个视频案例来理解主题IP的作用。

![](img/f941d857afc073a19eb41ff8e4c1d05a_10.png)

![](img/f941d857afc073a19eb41ff8e4c1d05a_0.png)

**案例一：** 一个账号的视频标题为“挑战惹老婆生气的第21天”。视频内容仅是夫妻间的日常嬉戏打闹。如果不看标题，这个视频可能显得普通，看过即忘。但标题“第21天”赋予了它连续性，暗示这是一个系列。观众可能会好奇前20天发生了什么，并期待第22天的内容，从而产生持续关注的欲望。

**案例二：** 账号“小姨恋爱课”的视频内容固定为讲解恋爱技巧。其主题明确，人设清晰（小姨），内容对目标人群（如不擅长恋爱的男性）具有实用性和共鸣感。这使得账号能围绕“恋爱教学”这一主题，持续产出大量相关视频，建立起鲜明的IP。

通过对比可以发现，第一个唱歌账号缺乏明确的主题IP，观众听完歌就可能离开，不会关注其过往或未来的内容。而拥有主题IP的账号，能有效提升观众的留存率和期待感。

![](img/f941d857afc073a19eb41ff8e4c1d05a_12.png)

**核心概念：**
主题IP = 一个能贯穿账号多期视频的、具有连续性和吸引力的核心主题或系列名称。

## 如何策划主题IP
理解了主题IP的重要性后，我们来看看如何为自己的账号策划一个合适的主题IP。

![](img/f941d857afc073a19eb41ff8e4c1d05a_14.png)

以下是几种常见的主题IP策划方法，每种方法都包含一个核心公式。

**公式：主题IP = [前缀] + [垂直领域]**

1.  **名字 + 垂直领域**
    *   **方法介绍：** 以创作者的个人名称或昵称为前缀，结合内容领域。
    *   **举例：**
        *   某某的北漂日记（零二集）
        *   老韩摄影教学（第12期）
        *   猪猪小姐姐的撩汉指南

2.  **人群 + 垂直领域**
    *   **方法介绍：** 以特定人群为前缀，结合该人群关心的内容领域。
    *   **举例：**
        *   别人家男朋友的拍照技巧（第三期）
        *   宝妈育儿小技巧
        *   新手开店避坑指南

3.  **地域 + 垂直领域**
    *   **方法介绍：** 以特定地点为前缀，结合在该地点发生的内容。
    *   **举例：**
        *   西藏自驾之旅（第50天）
        *   环球旅居日本站
        *   广州美食之火锅篇

4.  **场景 + 垂直领域**
    *   **方法介绍：** 以特定场景为前缀，结合在该场景下的内容。
    *   **举例：**
        *   办公室迷惑行为（为什么总有人脚臭？）
        *   儿童房风格装修
        *   婚礼现场的创意小心思

## 总结
本节课中我们一起学习了**主题IP**。我们明白了主题IP是一个能让账号内容形成系列、增强观众粘性的核心策划。关键在于为你的内容找到一个“**前缀**”（如名字、人群、地域、场景），并将其与你的“**垂直领域**”紧密结合。在明确了商业定位后，不要急于拍摄，先花时间策划一个清晰、有吸引力的主题IP，这是打造成功抖音账号的重要一步。
