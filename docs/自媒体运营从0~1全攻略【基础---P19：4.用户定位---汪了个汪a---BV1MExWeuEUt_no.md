# 自媒体运营从0到1全攻略：P19：4.用户定位 👥

在本节课中，我们将要学习自媒体运营中至关重要的一环：用户定位。我们将了解抖音平台的用户特征，并学习如何系统地分析并定位你的目标用户群体，为后续的内容创作和商业变现打下坚实基础。

## 用户定位的核心逻辑

用户定位的核心是思考：我们准备给什么样的人，发什么样的作品。

要理解这一点，我们需要先了解抖音的推荐机制。用户刷到你作品的完整流程如下：

![](img/c6c5da90cbec2ae6cc62c1a5424c9b6d_1.png)

1.  你创作并发布内容。
2.  抖音平台对你的账号和作品进行综合分析。
3.  结合用户的历史兴趣，将你的视频推送给可能感兴趣的用户。
4.  用户观看后，会产生**观看、点赞、关注、评论、转发**等行为反馈。
5.  抖音根据这些正向反馈，将视频推送给更多相似的用户，从而形成热门视频。

因此，想要创作出有潜力的视频，首先必须对你的目标用户进行深入分析。虽然我们运营抖音的终极目的是**变现**，而非单纯追求热度，但视频的播放量是变现的前提。在分析目标用户时，必须将核心用户的**商业价值**考虑进去，并针对他们来拆解和设计视频内容的价值。

这样才能最大限度地保证你发布的视频吸引到的人群是精准且有价值的。

## 抖音平台用户现状

在定位具体目标前，我们先宏观了解抖音用户的整体特征。抖音日活跃用户已突破7亿，人均每日使用时长超过2小时，这意味着在任何时间、任何行业都有海量用户在活跃。

![](img/c6c5da90cbec2ae6cc62c1a5424c9b6d_3.png)

从整体用户画像来看，抖音用户具备以下特征：

![](img/c6c5da90cbec2ae6cc62c1a5424c9b6d_4.png)

![](img/c6c5da90cbec2ae6cc62c1a5424c9b6d_5.png)

*   **性别分布**：女性用户占比约55%。
*   **年龄分布**：18-30岁的用户占比高达71%。
*   **城市分布**：超过60%的用户分布在一、二线城市。
*   **兴趣偏好**：教育、美食、旅游、搞笑和娱乐是总体兴趣偏好的前五名，其中教育和美食占比最高。
*   **消费动机**：年轻群体（18-30岁）的网上购物动机中，有91.8%是出于“自己喜欢或需要买给自己”。
*   **内容偏好**：这部分年轻群体喜爱的人物和形象通常具备以下特质：
    1.  令人心情愉悦或搞笑。
    2.  高颜值（好看、可爱、漂亮）。
    3.  具备爱国情怀和社会正能量。
    4.  展现认真生活和处事的态度。

因此，在创作视频或打造人设时，可以参考以上内容特质。

## 如何定位目标用户

定位目标用户时，数据来源可以是对标账号的分析、行业报告，或是企业自身积累的消费者数据。

以下是构建目标用户画像的基础分析维度：

![](img/c6c5da90cbec2ae6cc62c1a5424c9b6d_7.png)

**基础人口属性**
*   **性别**：男 / 女
*   **年龄**：18岁及以下、18-28岁、28-38岁、38-48岁、48-58岁、58岁及以上
*   **地域**：一线城市、二线城市、三线城市、四线城市等
*   **人群特征**：日常白领、熬夜人群、家庭主妇、学生等

**消费行为特征**
*   **消费水平**：经济型商品购买者 / 豪华型商品购买者
*   **购买时间偏好**：上午、中午、下午、晚上
*   **价格敏感度**：高 / 中 / 低
*   **囤货偏好**：高 / 中 / 低

**兴趣标签**
抖音平台将用户兴趣划分为17个大类，包括：金融、汽车、科技、娱乐、服饰、教育、母婴、法律、体育、游戏、美食、生活、家装、美妆、旅游、房产、商务服务。

## 用户价值分层：5A模型

对于品牌商家而言，抖音采用5A模型对用户进行价值分层，这对于制定营销策略至关重要。

以下是5A人群的具体定义：
*   **A1（了解感知）**：我知道。用户被动接收信息，是品牌知名度的主要来源。
*   **A2（吸引兴趣）**：我喜欢。用户对品牌产生兴趣，形成短期记忆。
*   **A3（主动问询）**：我相信。用户被引发好奇，主动搜索品牌信息，是**最接近转化**的高价值人群。
*   **A4（购买行动）**：我购买。用户完成购买及后续的消费、使用行为。
*   **A5（拥护推荐）**：我推荐。用户成为品牌忠实粉丝，会复购并主动向他人推荐。

此外，还有规模最大的 **O人群（潜在人群）**，即平台内活跃但尚未与品牌产生联系的潜在用户。

![](img/c6c5da90cbec2ae6cc62c1a5424c9b6d_9.png)

## 进阶用户分析：人格维度

如果获取上述传统维度数据有难度，或想更深入地指导内容创作，可以从用户人格层面进行分析。

参考九型人格等理论，针对不同人格特点产出相应内容，例如：
*   **忠诚型人格**：谨慎、多疑、保守。内容需**真实、接地气**，以获取其信任。
*   **理想主义者**：偏感性。内容应侧重提供**情绪价值**。
*   **艺术家型人格**：乐于接受新体验。适合发布**创意类或挑战体验型**内容。
*   **学者型人格**：对知识类内容更感兴趣，且**互动意愿强**。
