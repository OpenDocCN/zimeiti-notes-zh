# 小红书私域引流教程：P1：如何设置自动回复跳转微信卡片 📲

在本节课中，我们将学习如何为小红书账号配置自动回复功能，实现当用户私信时，系统能自动回复一条包含微信二维码的笔记，从而将流量从公域平台引导至私域微信。

上一节我们介绍了课程目标，本节中我们来看看具体的操作步骤。

## 效果展示与原理

当客户在小红书上发送私信后，系统会自动回复一条笔记。客户点击该笔记，即可直接显示您的微信二维码。这种方式可以实现自动引流的目的。

![](img/9e904b021ae648e90ed240ee7fbf7973_1.png)

![](img/9e904b021ae648e90ed240ee7fbf7973_1.png)

## 第一步：登录管理后台

![](img/9e904b021ae648e90ed240ee7fbf7973_3.png)

首先，需要登录红书词语网站的后台管理系统。您需要使用账号和密码进行登录。

**核心操作**：`访问指定网站 -> 输入账号密码登录`

![](img/9e904b021ae648e90ed240ee7fbf7973_3.png)

![](img/9e904b021ae648e90ed240ee7fbf7973_5.png)

> 提示：此账号和密码可向您的代理直接购买获得。

## 第二步：绑定小红书账号

![](img/9e904b021ae648e90ed240ee7fbf7973_7.png)

登录后台后，左侧菜单栏有“账号管理”选项。点击进入后，选择“账号列表”，即可将您的小红书账号绑定至此后台，为后续的自动回复功能做准备。

![](img/9e904b021ae648e90ed240ee7fbf7973_5.png)

以下是绑定账号的具体步骤：

1.  **创建小红书专业号子账号**：登录您的小红书专业号后台，进入“账号管理”->“运营账号”，点击“新建子账号”。
2.  **填写账号信息**：账号名称可随意填写。绑定方式务必选择“邮箱绑定”，可使用QQ邮箱等任何平台邮箱。
3.  **验证邮箱**：点击“发送验证码”，验证码将发送至您填写的邮箱。登录邮箱查收验证码，并复制回小红书后台的验证码输入框，点击“递交”。
4.  **设置子账号密码**：创建成功后，您会收到第二封邮件，提示为子账号设置密码。点击邮件中的链接，完成密码设置。

完成以上步骤后，您就拥有了一个可用于绑定的邮箱账号和密码。

![](img/9e904b021ae648e90ed240ee7fbf7973_7.png)

## 第三步：在后台完成绑定

返回红书词语网站的后台，在绑定账号页面，输入上一步创建的小红书子账号的邮箱和密码。

**核心操作**：`输入邮箱 -> 输入密码 -> 输入从上级购买的卡密 -> 点击确定`

![](img/9e904b021ae648e90ed240ee7fbf7973_9.png)

![](img/9e904b021ae648e90ed240ee7fbf7973_9.png)

绑定成功后，您的账号就会出现在账号列表中。

![](img/9e904b021ae648e90ed240ee7fbf7973_11.png)

![](img/9e904b021ae648e90ed240ee7fbf7973_11.png)

## 第四步：设置自动回复内容

账号绑定完成后，即可开始配置自动回复的具体内容。点击对应账号的“管理”选项。

![](img/9e904b021ae648e90ed240ee7fbf7973_13.png)

![](img/9e904b021ae648e90ed240ee7fbf7973_13.png)

在设置页面，需要填入一个“笔记ID”。这个ID来源于您在小红书上发布的一条特定笔记。

![](img/9e904b021ae648e90ed240ee7fbf7973_15.png)

以下是获取并设置笔记ID的流程：

1.  **创建引流笔记**：在您的小红书账号后台，创建一条新的笔记。这条笔记的封面设计至关重要，直接影响转化率。
2.  **复制笔记链接**：笔记发布后，复制其分享链接。这个链接中就包含了笔记的唯一ID。
3.  **提取并填写ID**：将链接中的笔记ID部分提取出来，填写到后台设置页面的“笔记ID”栏中。

![](img/9e904b021ae648e90ed240ee7fbf7973_15.png)

![](img/9e904b021ae648e90ed240ee7fbf7973_17.png)

![](img/9e904b021ae648e90ed240ee7fbf7973_17.png)

4.  **上传微信二维码**：“跳转链接”可留空，系统会自动生成。您需要将个人微信二维码图片上传到后台。
5.  **保存设置**：完成以上信息填写后，点击“确定”保存设置。

![](img/9e904b021ae648e90ed240ee7fbf7973_19.png)

![](img/9e904b021ae648e90ed240ee7fbf7973_19.png)

至此，整个自动回复跳转微信卡片的流程就设置完成了。当有用户给您的小红书账号发送私信时，系统将自动回复您设置的笔记，用户点击即可看到您的微信二维码，实现从小红书到微信的引流。

![](img/9e904b021ae648e90ed240ee7fbf7973_21.png)

![](img/9e904b021ae648e90ed240ee7fbf7973_21.png)

## 总结

本节课中我们一起学习了小红书自动回复引流功能的完整设置流程。关键步骤包括：登录管理后台、创建并绑定小红书子账号、发布引流笔记并获取ID、最后在后台完成自动回复内容的配置。掌握这个方法，可以帮助您高效地将小红书公域流量引导至微信私域。
