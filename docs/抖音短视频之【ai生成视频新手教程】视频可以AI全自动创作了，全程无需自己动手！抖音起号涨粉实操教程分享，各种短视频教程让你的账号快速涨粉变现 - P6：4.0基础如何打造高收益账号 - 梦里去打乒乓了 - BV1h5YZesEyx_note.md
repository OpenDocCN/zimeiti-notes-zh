# AI短视频创作：P6：4.0 基础如何打造高收益账号

在本节课中，我们将要学习如何解决视频播放量低的问题，并了解如何通过选择正确的平台和内容领域来提升账号表现，从而打造高收益的账号。

## 播放量低迷的原因分析

上一节我们介绍了AI创作视频的潜力，本节中我们来看看为什么有些视频发布后播放量始终很低。

![](img/3fe63c75e4cb84c086c4e91046ef7af0_3.png)

很多同学认为视频可以制作，也相信能赚到钱，但发布的视频播放量总是为零，或者只有几百到一千多。这并非平台针对个人，而是因为自媒体平台众多，规则各异。

![](img/3fe63c75e4cb84c086c4e91046ef7af0_5.png)

## 平台选择的重要性

目前自媒体平台总数超过30个。以下是需要了解的平台列表：

*   今日头条
*   哔哩哔哩 (B站)
*   抖音
*   快手
*   ... (以及其他20多个平台)

这30多个平台未来都可能用到。每个平台的收益结算方式和适合的内容领域各不相同。如果不了解这些差异，就难以获得平台流量扶持。

![](img/3fe63c75e4cb84c086c4e91046ef7af0_7.png)

## 领域与平台的匹配

![](img/3fe63c75e4cb84c086c4e91046ef7af0_9.png)

每一个平台都有其最适合的内容领域。选择错误的领域是许多账号做不起来的主要原因。

例如，如果在今日头条上发布一个适合B站的舞蹈领域内容，就很难获得成功。平台不会将流量浪费在不适合其用户的内容上，而是会扶持那些符合平台调性和用户喜好的领域。

平台流量推送逻辑可以概括为：
**平台算法 → 识别内容领域 → 匹配用户兴趣 → 决定流量分配**

因此，选择与平台匹配的领域至关重要。

## 高流量内容类型选择

在众多平台中，我们应该选择什么样的内容来制作视频呢？首先需要思考用户最喜欢观看什么。

根据大数据分析，用户的点击量、阅读量和完播率最高的三个视频类型如下：

*   **视频剪辑类**：对现有影视、综艺等素材进行创意剪辑。
*   **解说视频类**：对影视、游戏、知识等内容进行配音解说。
*   **混剪视频类**：将多个相关素材混合剪辑，形成新的主题或故事。

**大数据**是指平台通过分析用户行为（如观看时长、互动频率），来精准推荐内容的技术。例如，如果你经常观看美食视频，平台就会向你推送更多美食相关内容。大数据比我们想象的更了解用户的喜好。

## 总结

![](img/3fe63c75e4cb84c086c4e91046ef7af0_11.png)

本节课中我们一起学习了打造高收益账号的基础方法。核心在于理解播放量低并非平台偏见，而是源于对平台规则和内容领域的不熟悉。关键在于：第一，从30多个平台中选择适合自己的；第二，确保创作领域与所选平台高度匹配；第三，优先选择大数据验证过的高流量内容类型，即视频剪辑、解说和混剪。遵循这些原则，才能获得平台的流量扶持，提升账号表现。
