1.  在抖音找到目标视频，点击“分享”按钮复制链接。
2.  打开工具网页，将链接粘贴到指定输入框。
3.  点击“提取文案”按钮。
4.  稍等片刻，工具会自动识别并生成视频的完整文案。
5.  你可以直接复制文案，或使用“格式化”功能，使其按对话段落清晰排列。
