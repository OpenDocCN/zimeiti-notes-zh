# 新媒体运营系统课：P7：3.账号定位 🎯

![](img/cfa0994b9f05b09461459eb1fd213aa3_0.png)

在本节课中，我们将要学习账号定位的核心方法。账号定位是运营的基石，它决定了内容方向、吸引的粉丝群体以及最终的商业价值。一个清晰的定位能让你的账号在众多竞争者中脱颖而出。

## 理解账号定位：三个核心问题

要理解账号定位，需要反思三个核心问题：
1.  你是谁？
2.  你是干什么的？
3.  为什么要关注你？

首先，我们需要确定接下来要做的账号类型。

![](img/cfa0994b9f05b09461459eb1fd213aa3_2.png)

## 确定账号类型：从大类到细分

除了上一节我们介绍的大类赛道（即抖音定义的一级垂类），我们还要思考在一级垂类下的细分赛道是什么。

这好比参加运动会，你报的项目是“田径”，这是你的一级垂类。而你最擅长且具有优势的项目是“百米赛跑”，这就是你的二级垂类。因此，我们可以通过定位账号的细分类目来确定账号的垂直类型。

在抖音中，无论任何行业，只要定位足够清晰，都有机会成功。因为抖音的流量足够大，一旦做起来，增长和变现的速度将非常快。

## 明确账号主体：谁在发声

其次，我们要确定账号主体是谁。账号主体是账号的拥有人，也是最终呈现给粉丝的账号主人公。这需要让粉丝清楚地知道是在看谁的作品。

在抖音商业类账号中，账号主体主要分为两类：
*   **商家账号**：如果账号主体是公司，即为商家账号。这类账号的内容应偏向商业化，并需不断通过内容拉近与粉丝的距离，避免过于严肃。这样既能建立信任感，也能带来好感，更符合抖音的娱乐属性。
*   **达人账号**：如果账号主体是个人，即为达人账号。这类账号首先应解决信任问题，可以通过IP打造的方式，为账号树立一个积极正面的人设。

## 明确账号内容：提供什么价值

然后，我们要明确这个账号是做什么的。最直接的方式是通过抖音认证身份。
*   如果是达人，可以通过认证获得**黄V**。
*   如果是公司，可以通过认证获得**蓝V**。

认证职业后，粉丝可以清晰地看到认证标签，从而明白我们的职业，对这个账号是做什么的也能一清二楚。同时，商家认证蓝V或达人认证黄V，也可以获取平台背书，增加信任感。

在未认证之前，也可以通过账号设置向粉丝明确传达你是做什么的。例如，可以通过**头像、背景图、简介、视频封面**等进行与业务关联的统一设置。

## 思考核心价值：为何关注你

最后，我们需要花时间思考一个核心问题：你将为粉丝提供什么样的价值？在抖音中，粉丝对于博主的价值需求主要有以下几种：

以下是粉丝关注的五种核心价值类型：

1.  **知识价值**
    *   **认知型**：如人生思考、思维方式、读书分享、行业内幕、成功经验、个人提升等。
    *   **干货型**：如技能提升、经验总结、过程步骤、冷知识等。
    *   **常识普及型**：如生活、健康、产品、科学、历史、地区常识的解读和普及。
    *   **提供给粉丝的价值**：认同感，如三观认同、观点认同，减少知识盲区的获得感，进而产生对博主的认同。

2.  **情绪价值**
    *   情绪是影响观众的重要因素。作品中容易让观众接受的情绪类型包括：开心愉悦（搞笑、夸张表情）、人物关系共鸣（情侣、婆媳、家庭、职场关系等）、情绪波动（争议话题、反转反差内容激发）、以及怀旧、震惊、好奇等。
    *   **提供给粉丝的价值**：共鸣。利用表情、文字、画面、声音、肢体动作等准确到位地抒发情绪，直接感染观众。

3.  **感觉价值**
    *   博主通过短视频的画面、文字和声音传递信息。提供的价值包括：视频的画面美感、构图，以及给观众带来的喜欢、治愈、视觉冲击，或对作者人格魅力的仰慕；视频声音带来的节奏感、心跳加速或听觉享受等。常见于颜值、萌宠、美景、才艺、文化类账号。
    *   **提供给粉丝的价值**：共情。因表情、文字、画面、声音、肢体动作的自然表现，让人产生联想，并在情绪、认知等意识层面与作者产生连接，从而间接引发共鸣。

4.  **体验价值**
    *   抖音让用户看到了不一样的生活方式，体验到了未曾体验过的事物。例如流量很大的美食账号（享用与制作体验）、旅行体验、运动体验、角色体验等。
    *   **提供给粉丝的价值**：主要是满足好奇心，带来紧张感、激动、感动、体验获得感等。

5.  **期待价值**
    *   当用户被你的短视频内容吸引，或对作者本人产生好奇时，看完视频可能会意犹未尽，进而点击关注，期待下次更新。许多博主会利用这一点，不将所有内容在一个视频中呈现完，制造“且听下回分解”的感觉。

---

本节课中，我们一起学习了账号定位的完整框架。我们从理解定位的三个核心问题出发，逐步明确了如何确定账号类型、选择账号主体、清晰传达账号内容，并深入分析了能为粉丝提供的五种核心价值。一个成功的账号定位，需要在这些维度上都有清晰、一致的规划。
