# 小红书运营：P6：4.小红书店铺账号测试 🧪

在本节课中，我们将要学习一个至关重要的前期准备工作：如何测试你的小红书店铺账号是否具备基础权重。这是开启小红书店铺运营的第一步，能有效避免后续投入的精力与资源因账号问题而白费。

## 概述

小红书平台与其他平台不同，并非所有新创建的账号都具备初始权重。无论是使用新手机号、新身份证注册的账号，还是使用超过3年、5年的老手机号注册的账号，都存在一定概率（约5%-10%）是“无权重”账号。如果使用无权重账号运营店铺，后续所有努力都可能无效。因此，在正式开店前，对账号进行测试是必要步骤。

## 测试原理与时机

上一节我们强调了测试的重要性，本节中我们来看看具体的测试原理与最佳操作时机。

账号测试的核心是**模拟真实发布内容，并观察其自然流量数据**。通过发布笔记，观察其在最初几分钟及随后几天的浏览量（即“小眼睛”数量），可以判断账号是否被平台赋予了基础流量推送权重。

**关键点**：测试必须在**正式开通店铺之前**进行。不要在完成店铺注册、填写完所有信息后再测试。我们可以在未开店的状态下，直接使用该账号发布笔记来测试。

## 测试操作步骤

以下是账号测试的具体操作步骤，整个过程约需3到5天。

1.  **准备测试账号**：准备2到3个待测试的账号即可。通常这足以覆盖，避免遇到那5%的无权重账号。
2.  **发布测试笔记**：每个账号发布一篇笔记。内容无需复杂，但必须符合以下要求：
    *   **内容形式**：必须是**原创视频**，视频时长需**超过15秒**，建议控制在20秒左右。图片或文字笔记也可，但视频是更稳妥的测试方式。
    *   **内容主题**：可以是你打算经营的同类产品信息，也可以是简单的打招呼、求关注等内容。
3.  **观察初始流量**：笔记发布后约2分钟内，系统会给予一波初始推荐流量。此时“小眼睛”数量通常在20个以上，但这仅代表笔记被推送，**不能**作为账号有权重的最终依据。
4.  **分析核心数据**：需要持续观察3天内的数据。判断账号有权重的标准是：
    *   **每日浏览量**：每天“小眼睛”增长数在50以上。
    *   **累计浏览量**：3天后，该笔记的总“小眼睛”数量达到150以上。
    *   公式表示：`3天总浏览量 >= 150` 且 `日均浏览量 >= 50`。

## 测试注意事项

以下是进行账号测试时必须严格遵守的几点注意事项，任何一点疏忽都可能导致测试失败或结果不准。

*   **内容必须原创**：测试笔记必须是原创内容。直接复制他人的内容，即使账号有权重，该笔记也可能无法获得流量。可以对他人内容进行深度修改，使其成为自己的原创。
*   **禁止同ID操作**：多个测试账号**不能**在同一部手机（同一设备ID）上切换登录操作。小红书平台会识别设备ID，同设备操作会导致账号关联，可能全部被判定为无权重。
    *   **正确做法**：使用家人或朋友的**不同手机**（不同设备）进行测试。如果使用电脑，需要配合特定的浏览器插件来模拟不同设备环境，但操作较复杂。
*   **禁用同一Wi-Fi**：测试期间，所有测试手机**不能连接同一个Wi-Fi网络**。必须使用各自的**手机移动数据流量**进行操作。
    *   **特殊情况**：如果这几部手机曾经连接过同一个Wi-Fi，则需要断开该Wi-Fi，等待**至少一周**后，再使用移动数据流量进行测试。否则，平台可能通过网络环境判定账号关联，导致数据叠加或权重失效。
