## 开店前的准备

上一节我们介绍了小红书开店的整体概念，本节中我们来看看开店前需要准备哪些东西。

![](img/b9daf64e780ab60300e3af59a88d069e_5.png)

首先，开通店铺可以通过手机或电脑完成。手机端可以使用小红书App本身，或者下载名为“小红书千帆”的商家管理应用。

![](img/b9daf64e780ab60300e3af59a88d069e_7.png)

![](img/b9daf64e780ab60300e3af59a88d069e_9.png)

以下是开通个人店所需的核心证件：
*   **身份证**：开店人的有效身份证。
*   **银行卡**：用于后续提现的银行卡。

![](img/b9daf64e780ab60300e3af59a88d069e_11.png)

**核心要求**：身份证与银行卡的持有人**必须是同一个人**。这是因为后续所有经营收入都将提现至该银行卡。如果使用家人或朋友的证件，需确保关系可靠，以避免未来的经济纠纷。

## 开店流程与灵活设置

![](img/b9daf64e780ab60300e3af59a88d069e_19.png)

了解了基本要求后，我们进入具体的开店流程。在操作过程中，部分信息可以灵活填写，优先完成店铺开通。

![](img/b9daf64e780ab60300e3af59a88d069e_21.png)

经营类目在前期可以任意选择一个，目的是先成功开通店铺。店铺开通后，类目可以随时修改。
店铺名称也可以随意填写，后续支持修改（通常有修改周期限制，如7天一次）。
经营地址同样可以随意填写，不影响开店。
**重要提示**：开店过程中的**保证金可以跳过，暂时不缴纳**。店铺成功开通后，再根据平台提示缴纳即可。

## 详细操作步骤

![](img/b9daf64e780ab60300e3af59a88d069e_29.png)

![](img/b9daf64e780ab60300e3af59a88d069e_31.png)

以下是使用手机通过小红书App开通个人店铺的详细步骤：

![](img/b9daf64e780ab60300e3af59a88d069e_33.png)

1.  **搜索开店入口**
    打开小红书App，在首页顶部的搜索栏中输入“开店”并搜索。

![](img/b9daf64e780ab60300e3af59a88d069e_35.png)

2.  **进入官方开店页面**
    在搜索结果中找到并点击“小红书电商”官方账号发布的“开店”相关内容，进入官方开店引导页面。

![](img/b9daf64e780ab60300e3af59a88d069e_37.png)

3.  **开始创建店铺**
    在开店页面点击“立即开店”按钮。个人店开通是免费的。

![](img/b9daf64e780ab60300e3af59a88d069e_39.png)

4.  **选择店铺类型**
    在店铺类型选择页面，选择“个人店”。个人店仅需身份证；若选择“个体店”，则还需要营业执照。

![](img/b9daf64e780ab60300e3af59a88d069e_41.png)

5.  **填写基本信息**
    点击“下一步”，跳过“邀请码”（若无可不填）。然后根据页面指引，上传身份证正反面照片，并填写个人信息。
    在涉及工商资质的选项中，可选择“属于零星小额交易，依法无需办理”。
    填写完毕后，点击“立即开店”提交审核。

![](img/b9daf64e780ab60300e3af59a88d069e_43.png)

![](img/b9daf64e780ab60300e3af59a88d069e_45.png)

**核心流程代码描述**：
```
1. 小红书App -> 搜索“开店” -> 进入官方页面
2. 点击“立即开店” -> 选择“个人店”
3. 填写身份信息 -> 提交审核 -> 店铺开通
```

![](img/b9daf64e780ab60300e3af59a88d069e_47.png)

正常情况下，信息填写无误，店铺审核通过非常快。

## 核心注意事项与规则

店铺开通后，了解相关规则至关重要。以下是几个必须掌握的核心点：

*   **身份一致性**：再次强调，提现金额将进入开户身份证所对应的银行卡，务必保证身份证与银行卡为同一人。
*   **店铺命名**：店铺名称默认与小红书账号名关联，格式通常为“{账号名}的店”。取名前可先查询是否重复。
*   **保证金**：个人店保证金一般为**1000元**左右，具体金额因经营类目而异。
*   **平台技术服务费（扣点）**：平台会对成交订单收取一定比例的技术服务费。
    **计算公式**：`当月技术服务费 = (结算金额 - 1万元) * 5%`
    **规则解释**：一个自然月内，店铺订单结算总额（扣除退款后）不超过1万元时，免收技术服务费。超过1万元的部分，按**5%** 的比例收取费用。
    *举例*：某月结算金额为2万元，则技术服务费为 (20000 - 10000) * 5% = 500元。

## 其他店铺类型与特殊类目

除了个人店，如果你想经营更大规模的业务，可以考虑其他店铺类型。

![](img/b9daf64e780ab60300e3af59a88d069e_61.png)

**个体店/企业店**：需要使用小红书账号持有人的**营业执照**（法人需与账号实名一致）。可通过“小红书千帆”App或官网开通。
**专卖店/旗舰店**：同样遵循上述基本流程，但需要相应的品牌资质文件。

![](img/b9daf64e780ab60300e3af59a88d069e_63.png)

某些特殊商品类目需要额外的资质证明，在开店前务必了解清楚：
*   **食品类**（如酒水、零食、水果）：需提供**营业执照**和**食品经营许可证**，部分情况需要品牌授权。
*   **美妆类**：需提供**化妆品备案凭证**。
*   **家电/3C数码类**：需提供**3C认证证书**。

建议新手在没有相关资质和行业经验时，谨慎选择这些特殊类目。平台对所有类目的资质要求，均可在开店后台的“类目资质”说明中查询。
