# 新媒体运营零基础小白必看课程：P17：02.2：工具篇-AI生成PPT 🚀

在本节课中，我们将学习如何使用一款高效的AI工具来生成PPT。这款工具能根据你提供的主题或现有文档，自动梳理大纲并生成详细的PPT内容，极大地提升内容创作效率。

上一节我们介绍了AI在文案创作中的应用，本节中我们来看看如何利用AI辅助制作演示文稿。

## 工具核心功能

这款AI生成PPT工具主要提供两种模式：

1.  **主题生成模式**：输入一个主题，AI将结合生成式技术（如ChatGPT）自动生成完整的PPT全文。
2.  **文档生成模式**：上传一篇现有的文档（如TXT、PDF），AI助手会自动总结文档大纲，并基于此大纲生成详细的PPT内容。

## 工具的优势与编辑

这款工具最突出的优势在于：**它能基于你提供的大纲或文章，自动分析内容并将其归类到不同的大纲节点下，并生成对应的详细阐述**。

这与许多其他AI PPT工具不同，那些工具往往要求你提供全部完整内容，它只是帮你做格式排版。而本工具能在你只有思路或要点时，帮你填充血肉，完善具体内容，这才是AI在PPT制作中最有价值的应用。

生成后的PPT支持在线编辑。右侧提供了丰富的调整选项。

![](img/4cbe588bef2422d4292a923685790967_19.png)

以下是主要的编辑功能：
*   **更换模板**：虽然当前内置模板数量不算非常多，但切换速度很快，足以满足基础需求。相信未来会持续增加。
*   **调整内容**：你可以直接修改文字、替换形状、调整布局等。
*   **下载本地**：你可以先在线替换好大致模板，然后下载到本地（如PowerPoint或Keynote）中进行更精细的调整。
