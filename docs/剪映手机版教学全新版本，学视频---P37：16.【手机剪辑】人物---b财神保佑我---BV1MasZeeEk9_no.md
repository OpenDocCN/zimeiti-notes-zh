# 视频剪辑入门教程：P37：16. 手机剪辑：人物单独显色

在本节课中，我们将学习如何使用剪映手机版制作“人物单独显色”的视频效果。这种效果能让画面中的人物保留原有色彩，而背景变为黑白或其他单色，从而突出主体。我们将通过分步操作，详细讲解实现这一效果的完整流程。

## 概述

“人物单独显色”是一种流行的视频剪辑技巧，它通过色彩对比来强化视觉焦点。其核心原理是复制视频图层，并为上下两层分别应用不同的色彩处理，最后通过蒙版工具将彩色人物区域精确显示出来。

上一节我们介绍了基础剪辑操作，本节中我们来看看如何利用图层与蒙版实现高级色彩效果。

## 详细步骤

以下是实现人物单独显色效果的具体操作步骤。

1.  **导入并复制视频**
    首先，在剪映中导入一段包含人物的视频素材。选中这段视频，点击工具栏中的 **复制** 功能。

2.  **切为画中画**
    复制完成后，点击 **切画中画** 功能。这样我们就得到了两个完全相同的视频轨道：主轨道和画中画轨道。

3.  **为主轨道添加滤镜**
    选中主轨道视频，点击 **滤镜**。在滤镜库中，选择一个能将色彩简化为单色的滤镜，例如 **牛皮纸滤镜**。此操作将使主轨道视频背景变为单色。

4.  **对齐视频轨道**
    将切下来的画中画视频拖动，使其起始位置与主视频完全对齐，确保两段视频播放同步。

5.  **处理画中画图层**
    选中画中画图层，进行以下操作：
    *   点击 **智能抠像**。软件会自动识别人物主体并将其从背景中分离出来。抠像完成后，画中画图层将只显示彩色人物。
    *   点击 **蒙版**，选择 **圆形蒙版**。此时画面会出现一个圆形蒙版区域。

6.  **调整蒙版**
    将圆形蒙版移动并覆盖到人物区域。通过拖动蒙版边缘的控制点，调整其大小和形状，使其与人物的轮廓大致匹配。这一步的目的是精确控制彩色区域的显示范围。

7.  **预览并导出**
    为视频添加合适的背景音乐。点击播放预览最终效果，确认无误后即可导出视频。

## 核心概念与技巧

*   **图层叠加**：效果依赖于**主轨道（背景层）** 与**画中画轨道（人物层）** 的叠加。上层（画中画）的彩色人物会覆盖下层的单色背景。
*   **蒙版（Mask）**：蒙版是控制图层局部显示或隐藏的工具。其作用可以理解为：`蒙版区域 = 显示区域`。我们使用圆形蒙版来限定彩色人物的显示范围。
*   **智能抠像**：这是实现效果的关键，其算法自动完成了人物与背景的分离，无需手动逐帧抠图。

## 总结

本节课中我们一起学习了“人物单独显色”效果的制作方法。我们首先通过复制和切画中画创建了两个图层，然后分别为背景层去色、为人物层抠像，最后利用蒙版工具进行合成。掌握这个技巧，你就能轻松制作出突出视频主体的创意内容，为你的视频号运营增添亮点。
