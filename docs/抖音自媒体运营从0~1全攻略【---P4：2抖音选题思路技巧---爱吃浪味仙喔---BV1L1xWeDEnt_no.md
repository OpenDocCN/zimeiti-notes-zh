## 核心选题公式

那么，有什么办法解决选题难题呢？当然有。这是一套百万粉丝博主常用的选题技巧。为了方便理解，我将其总结为一个公式：

![](img/e63f590ec811ba3a04149ec24390a7e2_3.png)

**选题公式 = 内容定位 + 用户痛点 + 借势热点**

用好这个公式，你的内容方向就不会跑偏。

![](img/e63f590ec811ba3a04149ec24390a7e2_5.png)

---

![](img/e63f590ec811ba3a04149ec24390a7e2_5.png)

### 公式应用详解

上一节我们介绍了核心公式，本节中我们来看看如何具体应用。以下是公式的三个组成部分及其应用方法：

1.  **内容定位**
    *   确定你的账号所属领域或行业。
    *   **例如**：知识博主领域的关键词包括：`播放量`、`粉丝量`、`定位`、`选题`。

2.  **用户痛点**
    *   挖掘目标用户在该领域最关心、最困扰的问题。
    *   **例如**：用户对`便捷高效`、`实用技巧`、`教学内容`有强烈需求。

3.  **借势热点**
    *   结合当前的社会热点、节日、流行趋势进行创作。
    *   **例如**：结合“春节”热点，可以创作“春节必火选题”或“拜年视频拍摄技巧”等内容。

将以上三者进行任意组合，就能生成大量优质选题。

**应用示例（知识博主）**：
*   **组合**：`选题`（定位） + `高效`（痛点） + `春节`（热点）
*   **生成选题**：《如何用拜年的方式，策划春节必火的选题内容？》或《“张同学”快速涨粉的三个秘诀》。

### 跨行业应用

我们已经以知识博主为例进行了演示，现在将这个公式套用在其他行业试试看。以下是美妆行业的应用示例：

*   **内容定位**：`美白`、`除皱`、`口红`
*   **用户痛点**：`平价好用`、`明星同款`、`避坑扫雷`
*   **借势热点**：结合明星动态或特定节日

**生成选题示例**：
1.  `迪丽热巴同款`（热点） + `美白面膜`（定位） + `推荐`（解决痛点）
    *   **选题**：《迪丽热巴同款美白面膜推荐！》
2.  `七夕`（热点） + `口红`（定位） + `避坑`（痛点）
    *   **选题**：《七夕千万别送女朋友这几支口红！》
