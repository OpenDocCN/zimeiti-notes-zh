# 抖音运营教程：P23：垂直类目细分领域 🎯

在本节课中，我们将要学习账号垂直度的分级概念。理解这一点，能帮助你更精准地定位自己的内容方向，从而吸引更明确的粉丝群体。

上一节我们介绍了内容标签的重要性，本节中我们来看看如何将“垂直”这个概念进行更细致的划分。

我们经常说做账号要做垂直。很多初学者会简单地认为，选择一个大的方向（如美食或游戏）并持续发布相关内容，就是垂直。这个理解没错，但这仅仅是垂直分类的起点。

## 垂直度的级别划分

垂直类目的细分领域通常可以分为多个级别，从宽泛到具体。以下是各级别的定义与示例：

**一级垂直领域**
这是最宽泛的分类，代表一个大的内容方向。
*   **示例**：`游戏`、`美食`、`美妆`、`旅行`。

**二级垂直领域**
在一级分类下，进行更具体的划分。
*   **以“游戏”为例**：`手游`、`电竞`、`网游`。

**三级垂直领域**
在二级分类下，进一步细分到具体的产品、主题或风格。
*   **以“手游”为例**：`王者荣耀`、`和平精英`、`原神`。

**四级垂直领域（或更细）**
在三级分类的基础上，可以继续深入到极其具体的玩法、角色或场景。
*   **以“王者荣耀”为例**：`安琪拉教学`、`打野攻略`、`剧情模式解说`。

所以，垂直类目的细分领域有一级、二级、三级、四级。大领域和小领域都可以作为你的选择，这完全取决于你自己的兴趣和规划。

## 为何要理解细分领域？

因为今天课程后续的某些内容会涉及到这一部分。理解垂直度的分级，能帮助你：
1.  更清晰地规划账号内容。
2.  吸引目标更明确的粉丝，提升粉丝粘性。
3.  在竞争激烈的平台中，找到属于自己的差异化赛道。
