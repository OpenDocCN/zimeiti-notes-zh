在本节课中，我们将深入探讨小红书平台如何为用户打上标签，以及平台内主要的流量入口是如何运作的。理解这些机制，有助于我们反向推导，使自己的笔记更精准地触达目标用户。

上一节我们介绍了系统如何判定内容标签，本节中我们来看看平台如何判定用户标签。这个过程能帮助我们分析和理解自己的标签，以及笔记是如何被系统推荐给用户的。我们可以从背后的逻辑反推，如何让笔记被精准用户看到。

## 用户标签判定机制 👤

系统为用户打标签的主要依据是用户的**浏览行为**。用户注册时选择的兴趣范围参考价值不大，它仅作为平台对内容方向的初步引导和筛选，无法进入系统最深层的用户标签池。

系统判定用户兴趣分类完全基于其浏览行为。以下是系统分析用户行为的主要维度：

以下是判定用户标签的几个关键行为指标：

1.  **点击率**
    *   **公式**：`点击率 = (点击某类内容的次数) / (系统推送的该类内容总次数)`
    *   系统在发现页向用户推送多种内容（如美妆、家居、母婴）。用户点击哪类内容更多，系统就能计算出其点击概率，从而判断其兴趣。例如，用户点击10条内容，其中美妆3次、家居2次、母婴0次，系统会判定其对母婴不感兴趣，并持续推送更多美妆内容进行试探，以此打上强关联或弱关联标签。

2.  **停留时长**
    *   即用户在某篇笔记页面的观看时间。即使同属美妆领域，如果用户点了就划走，说明内容不吸引他。系统会继续推送美妆下的其他细分内容（如上妆、护肤）进行观察。反之，如果用户对平时很少点击的“家居”类内容停留很久，仔细阅读图片和评论，系统可能判断其有装修或购买家居用品的潜在需求。

3.  **页面浏览量**
    *   指用户浏览页面的深度。例如，用户不仅看了某篇装修笔记，还点进博主主页，并连续看了第二、第三、第四篇笔记。这个数值（PV）对账号健康度评分有非常大的帮助。在粉丝基数不大时，提升用户在自己主页的浏览深度，能为账号带来积极的评分。

4.  **互动行为**
    *   包括点赞、收藏、评论、关注。每一个数据都是我们需要努力提升的指标。互动数据高，笔记进入更大推荐流量池的几率更高，也可能带动账号内其他笔记的流量，并有助于粉丝增长和后续变现。关注行为是一个能永久提升账号权重的加分项。

通过以上行为分析，系统会为用户构建一个动态的“兴趣云图”，其中标签的大小代表兴趣的强弱。平台的目标是持续为用户推送其感兴趣的内容，以提升用户的获得感与停留时长。

## 平台流量分发入口 🚪

理解了用户标签，我们再来看看小红书主要的流量分发入口。作为用户可能经常使用，但作为运营者需要分析其区别。

小红书平台主要有四个流量入口：

以下是四个核心流量入口及其特点：

1.  **发现页**
    *   这是笔记获取流量的主要来源。内容包含：
        *   **关注达人**：用户已关注的博主发布的内容。
        *   **兴趣内容匹配**：系统根据用户标签（点击率、停留时长等）推荐的、其可能喜欢的内容。
        *   **发散性内容匹配**：系统也会推荐一些用户可能不感兴趣或未刷到过的内容，以增加平台内容的多样性，满足用户潜在的多方位需求。

2.  **关注页**
    *   这里集中展示用户所有关注博主的笔记。虽然用户直接刷关注页的频率可能不如发现页高，但对于强关注的粉丝而言，这里是稳定触达他们的渠道。

3.  **附近页**
    *   根据手机定位，推送附近及同城的相关内容。此板块以具有强地域特征的内容为主，如美食、本地旅游、探店等。对于运营线下实体店（如美容院、餐厅）的账号，在做信息流广告或内容运营时，需要重点关注附近人群的推送。

4.  **购物页**
    *   这是小红书向电商方向发展的重要入口，直接匹配用户的购物需求。目前，用户习惯仍在培养中，许多人仍倾向于通过搜索笔记来“种草”。该入口涉及直播带货和商城直接购买，是变现的重要渠道。
