# 小红书运营教程：P7：07、小红书平台推荐规则 📖

![](img/e9b0b1c35a6a316af0f28e69d1e02bf9_1.png)

在本节课中，我们将要学习小红书平台的核心推荐规则。理解这些规则是账号运营的基础，能帮助你避免违规，并让内容获得基础的平台流量推荐。

![](img/e9b0b1c35a6a316af0f28e69d1e02bf9_3.png)

上一节我们介绍了课程的整体框架，本节中我们来看看推荐规则的具体内容。

## 推荐规则详解

![](img/e9b0b1c35a6a316af0f28e69d1e02bf9_5.png)

推荐规则可以分解为三个核心部分来理解：用户行为规范、内容发布秩序以及图文影音要求。

### 1. 用户行为规范

用户行为规范定义了在小红书平台上的行为边界，分为禁止行为和允许行为。

以下是用户不能做的行为：
*   **禁止含有营销词汇**：例如“微信号”、“二维码”、“代购”、“水印”、“某某店铺”等直接表述。即使是开店商家，也不能直接使用这些词汇，否则会被判定违规。
*   **禁止使用敏感词**：所有内容不得包含平台定义的敏感词汇。

以下是用户允许做的行为：
*   **允许存在邮箱**：在个人资料或内容中留下邮箱地址是允许的。
*   **允许使用谐音或话术引导**：例如，可以使用“V心”、“攻众号”等谐音词，或通过话术引导用户，但不能直接出现“微信”二字。

需要注意的是，虽然允许使用邮箱，但其引流效果通常不佳。此外，即使用于矩阵运营的小号，也必须遵守上述规范，否则账号容易被封禁，得不偿失。

### 2. 内容发布秩序

内容发布秩序主要涉及内容主题和账号登录管理。

以下是内容方面的注意事项：
*   **禁止提供特定建议**：不得轻易提供医疗或投资方面的具体建议。例如，“你这个病去XX医院治效果好”或“XX楼盘值得投资”这类内容属于违规，会导致封号。

以下是账号登录管理的规则：
*   **限制登录账号数量**：一个移动设备上最多登录3个小红书账号，登录过多会导致账号被限流。
*   **电脑端登录限制**：在电脑端登录时，同样建议不要超过3个账号（包括主账号和子账号）。因为电脑通常使用同一个网络IP，登录过多账号也会引发限流。如需登录更多账号，请先退出已登录的账号。

### 3. 图文影音要求

这部分规则针对笔记内容本身。

以下是内容创作中不能做的：
*   **禁止诱导互动**：不得使用“关注领奖”、“点赞抽奖”、“关注领红包”、“关注评论有奖”等话术诱导用户行为。

以下是关于内容原创性的核心要点：
*   **原创内容权重最高**：平台从未出现过的原创内容，权重最高，发布后会被系统百分之百录入推荐池。
*   **非原创内容权重低**：复制、转发他人已发布过的笔记或文案，权重会偏低，系统可能不会录入。如果借鉴他人内容，必须进行深度修改，包括调整文案、美工和视频剪辑等。

## 规则的核心影响

理解并遵守以上规则的核心目的是：**确保账号能获得基础的平台推荐流量**。

![](img/e9b0b1c35a6a316af0f28e69d1e02bf9_7.png)

一个新账号发布的第一篇合规笔记，通常能获得50到200个“小眼睛”（阅读量），对应的曝光量大约在1000到2000次。如果触犯了上述任何规则，账号可能无法获得这部分基础的曝光数据。

![](img/e9b0b1c35a6a316af0f28e69d1e02bf9_9.png)
