# 小红书运营教程：P3-2：小红书平台检测规则详解 🕵️

在本节课中，我们将要学习小红书平台的核心检测规则。了解这些规则是安全、合规运营账号的基础，能有效避免笔记被限流或账号受处罚。我们将从多个维度分析平台禁止的行为，并提供实用的自查工具与方法。

上一节我们介绍了账号定位，本节中我们来看看平台如何监管内容。小红书的检测系统主要覆盖以下几个关键方面。

## 个人资料与内容检测

小红书平台会系统性地检测用户的**个人资料**、**笔记内容**、**私信**以及**评论**。在这些区域，严格禁止出现以下信息：
*   直接联系方式，如微信号、微信二维码。
*   营销导向词汇，如“代购”、“淘宝店铺”。
*   第三方平台水印。

平台管理相对宽松的方面是：**个人资料中允许填写邮箱地址**。同时，运营者可以使用谐音或引导至微博等平台进行间接引流。

## 设备与账号管理

小红书会检测登录设备的关联性。为了保证账号安全，请注意：
*   同一台移动设备最多建议只登录**3个**小红书账号。
*   避免频繁切换或使用过多账号在同一设备上操作。

## 内容创作规范

在撰写笔记时，必须遵守平台的内容规范，核心是**不得诱导用户互动**。以下是明确禁止的行为：
*   发布“关注领奖”、“点赞抽奖”等利诱性内容。
*   任何引导用户进行指定互动以获取利益的话术。

## 原创度保护

小红书平台对**原创内容**的保护程度非常高。因此，在推广时务必注意：
*   同一篇原创文章，**仅限一个账号**在小红书平台发布。
*   严禁将同一篇文章通过多个账号进行重复发布，这会被系统判定为搬运或 spam 行为。

除了上述基本规则，另一个至关重要的检测环节是**敏感词过滤**。接下来，我们将重点学习如何应对这一问题。

## 敏感词检测与处理

敏感词检测是小红书规则的核心部分。平台禁止讨论的类别包括但不限于：
*   金融投资
*   医疗健康用品
*   电子烟
*   手机卡
*   代购服务

在写作时，若无意中使用了违禁词汇，笔记可能会被限制推荐。为了解决这个问题，我们可以借助外部工具进行自查。

    *   **使用方法**：将写好的笔记正文粘贴到网站的输入框中。
    *   **操作指令**：点击“立即检测”按钮。
    *   **结果**：系统会高亮标记出文本中存在的所有小红书平台敏感词。

本节课中我们一起学习了小红书平台的四大核心检测规则：**资料与内容规范**、**设备管理**、**互动诱导禁令**以及**原创度保护**。同时，我们掌握了利用**敏感词检测网站**进行内容自查的关键技能。理解并遵守这些规则，是账号长期稳定运营的基石。关于更全面的违禁词列表，我们将在后续资料中为大家整理提供。
