# 微信视频号教程：P16：第二节：解剖推荐机制 🔍

在本节课中，我们将要学习微信视频号与抖音截然不同的推荐机制。理解这套基于社交关系的“冷启动”逻辑，是做好视频号运营的基础。如果你是新来的，建议从本系列教程的第一节开始按顺序观看。

上一节我们介绍了视频号的平台属性，本节中我们来看看其核心的推荐机制。视频号的推流方式与抖音不同，切勿将抖音的推荐逻辑直接套用在视频号上。许多培训课程在此存在误导，一旦理解错误，后续运营将步步维艰。

## 核心机制：基于社交链的冷启动

![](img/373da7cab3c81aec5d2c07e16d77c949_3.png)

![](img/373da7cab3c81aec5d2c07e16d77c949_4.png)

视频号推荐机制与抖音最大的不同在于，它基于社交链进行“冷启动”。什么是社交链？我们通过一个例子来说明。

现在你看到了一个视频并点赞，那么你的所有好友都会在他们的“朋友”栏看到你点赞的这个视频内容。这与抖音有本质区别：在抖音，你可以关闭“点赞”可见，保护个人隐私，点赞行为不会分发给朋友。但在视频号，你的点赞行为是公开的，会成为内容分发的起点。

为什么说它基于社交链启动？正是因为你的点赞、转发等互动，会直接将内容推送给你的一度人脉（即你的好友），从而启动在社交关系网内的传播。

![](img/373da7cab3c81aec5d2c07e16d77c949_6.png)

我们可以将这个过程理解为三个板块：

以下是推荐机制启动的三个关键步骤：

1.  **初始用户冷启动**：假设你是我的粉丝（初始用户），你看到我的视频后，觉得内容不错，产生了点赞或转发动作。
2.  **辐射一环社交圈**：一旦你点赞，你的所有好友都会看到我这个视频。如果你的好友也觉得不错并点赞，那么内容将推送到你好友的好友那里，即辐射到二度人脉。
3.  **进入公域流量池**：当内容的互动数据（如点赞量）达到某个**阈值**后，平台会判定其为优质内容，将其从私域社交圈推送到更大的公域流量池，让更多陌生人看到。

![](img/373da7cab3c81aec5d2c07e16d77c949_8.png)

![](img/373da7cab3c81aec5d2c07e16d77c949_9.png)

**阈值**是一个关键概念，它指触发平台公域推荐所需达到的数据标准。不同领域阈值不同，例如，一个知识博主的阈值可能是 **200次点赞** 辐射到 **2000次观看**。

因此，你在刷视频号时，既会看到好友点赞推荐的内容（社交推荐），也会看到完全不认识的人发布的内容（公域推荐）。视频号的推荐机制正是这两者的结合。

## 流量传播模型：一级与二级流量

为了更清晰地理解，我们可以将流量分为一级流量和二级流量。

*   **一级流量**：指你的粉丝。例如，你的账号有1万粉丝，发布内容后首先会推送给这1万人。
*   **二级流量**：指通过粉丝互动带来的新流量。当粉丝点赞后，他的好友（二级流量）就会看到内容。如果二级流量继续互动，则会辐射到三级、四级流量，实现裂变传播。

**传播模型可简化为：一级流量（粉丝）互动 → 辐射至二级流量（粉丝的好友） → 持续裂变。**

## 如何利用机制打造爆款？💥

理解了推荐机制，我们从运营层面看，要想做出爆款，关键在于驱动用户产生**点赞和转发**动作。平台正是依据用户反馈数据来决定是否将内容推入公域池。

以下是平台评估内容质量的几个核心用户反馈维度：

1.  **点赞/转发率**：权重最高。点赞直接触发社交推荐，是冷启动的关键。
2.  **完播率**：用户观看视频的完成程度，反映内容吸引力。
3.  **互动率**：主要指评论和关注率，反映内容引发的共鸣深度。

![](img/373da7cab3c81aec5d2c07e16d77c949_11.png)

那么，如何获得高点赞和高转发呢？根据我们的经验，一个非常有效的方法是：**以“情”动人**。

“情”能有效突破用户心理防线，促进高互动，从而催生爆款。这种“情”包括但不限于：

*   **感动之情**：如公益善举、亲情、爱情故事。
*   **骄傲之情**：如家国情怀、民族自豪感相关的内容。
*   **共鸣之情**：如引发特定群体（如宝妈、职场人）深度共鸣的内容。

这也解释了为什么视频号头部爆款中，情感类账号占比非常高。情感内容是触发社交互动最直接的催化剂。

## 平台的流量分级倾斜 📊

平台流量并非平均分配，存在明显的分级倾斜。我们将流量倾斜分为三级：

以下是平台流量倾斜的三个主要级别：

![](img/373da7cab3c81aec5d2c07e16d77c949_13.png)

![](img/373da7cab3c81aec5d2c07e16d77c949_14.png)

1.  **一级倾斜（最大）**：面向**头部账号**（如情感领域大号）。平台会给予其巨大的公域流量扶持，使其长期占据榜单前列。
2.  **二级倾斜（中等）**：面向**MCN机构旗下账号**。平台会提供一定的初始流量包或倾斜，然后根据内容反馈决定是否追加流量。
3.  **三级倾斜（依赖内容）**：面向**素人白号**。这类账号没有先天流量优势，必须依靠内容本身获得高互动、高反馈（高点赞/转发），才有机会闯入公域流量池成为爆款。

## 关键问题与运营策略 🎯

基于以上机制，一个核心问题浮现：对于一个没有初始粉丝（冷启动用户）的新号，该如何启动？

正确的运营策略应该是：**从公域获取初始流量，反哺视频号冷启动。**

具体操作是：不要只局限于视频号一个平台。你可以先在抖音、小红书等公域属性强的平台运营，积累第一批粉丝和影响力。然后，将这部分用户引导至你的视频号，作为你的“初始用户”。拥有了这批种子用户后，你便可以专注于在视频号内创作能引发他们高互动的内容，从而启动社交推荐链，并有机会冲击公域流量池。
