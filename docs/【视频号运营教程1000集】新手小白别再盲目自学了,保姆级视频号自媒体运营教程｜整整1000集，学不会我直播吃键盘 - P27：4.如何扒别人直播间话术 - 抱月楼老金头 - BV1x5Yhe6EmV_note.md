# 视频号运营教程：P27：如何扒取直播间话术 📝

在本节课中，我们将学习一个实用的技巧：如何高效地扒取其他成功直播间的话术。掌握这个方法，可以帮助你快速模仿和学习优秀主播的销售与互动策略，从而优化自己的直播内容。

---

## 核心思路

上一节我们介绍了视频内容与选品的复制方法，本节中我们来看看如何学习直播的核心——话术。

![](img/390d3f0058ae184fa2c58cd543229014_1.png)

通用的话术学习方法是：观察一个直播间，如果对方销售情况很好，并且他所售卖的商品或发布的视频内容你可以低成本复制，那么接下来关键就是研究他的直播话术。某些商品可能只在特定时段（例如过年时的年货）畅销，如果你不知道对应的话术如何组织，这个方法就能派上用场。

![](img/390d3f0058ae184fa2c58cd543229014_3.png)

## 所需工具

你需要使用一个名为“飞书”的工具，具体是其内置的“飞书妙计”功能。

## 操作步骤详解

以下是使用飞书妙计录制并导出直播间话术的完整流程。

1.  **打开工具**
    在飞书应用中，点击底部导航栏的 **`工作台`**。在工作台页面里，找到并点击 **`飞书妙计`** 功能。

2.  **开始录制**
    进入飞书妙计后，点击界面右下角的 **`语音`** 按钮，即可开始录音。此时，你可以打开任意一个你想模仿的直播间。

    > 该功能会实时录制直播间里的所有声音，包括主播的介绍、互动以及背景音。

3.  **录制时长建议**
    录制直播间声音大约 **15分钟**，这段时间通常足以让你了解主播是如何介绍产品、进行互动以及完成促单（逼单）的。

4.  **结束并导出**
    录制完成后，点击 **`完成`** 结束录音。系统会自动生成一个文字记录。点击进入这条记录，找到 **`分享`** 按钮。

5.  **获取文本**
    在分享选项中，选择 **`可编辑`** 或 **`可阅读`** 模式，然后复制生成的链接。将此链接发送到微信，并在微信中打开。

6.  **整理与运用**
    打开链接后，你将看到系统自动转写的直播间话术文本。你可以复制全部内容，然后进行编辑和提炼，最终形成适合你自己风格和产品的直播话术。

    > 你可以将整理好的话术写在纸上，或存入手机备忘录。开播时如果忘记，可以对照着念，以此保证直播流程的顺畅。

## 总结

![](img/390d3f0058ae184fa2c58cd543229014_5.png)

本节课我们一起学习了如何利用“飞书妙计”工具扒取并分析他人直播间的话术。关键步骤是：**`打开飞书工作台 -> 使用飞书妙计录音 -> 录制目标直播间约15分钟 -> 导出并分享文本 -> 在微信中打开链接获取文字稿 -> 编辑提炼成自己的话术`**。通过这个方法，你可以快速吸收优秀同行的经验，有效提升自己的直播表达能力。
