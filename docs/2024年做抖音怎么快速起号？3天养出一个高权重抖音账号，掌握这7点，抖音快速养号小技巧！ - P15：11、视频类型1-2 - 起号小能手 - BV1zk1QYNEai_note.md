# 抖音运营指南：P15：11、视频类型1-2 - 人设与口播

在本节课中，我们将学习抖音账号运营中至关重要的两个概念：**人设**的建立与区分，以及**口播**这种常见视频形式的具体应用方法。理解并掌握这些知识，是制作吸引人内容、实现账号增长的基础。

上一节我们讨论了文案的重要性，本节中我们来看看内容创作中更深层的一环：人设。从涨粉到变现并非一条直线，中间有一个关键环节是建立你的“人设”。人设不是为了炫耀，而是为了在众多同类创作者中让你被记住。例如，有十个人都做装修内容，如果其中一位建立了幽默搞笑的人设，观众就更容易记住他，进而喜欢他、关注他，成为他的粉丝。

这就是人设的作用。接下来我们讲解这个知识点，请边听边记。人设主要分为两种：**强人设**和**弱人设**。区分清楚这两种人设后，你看到任何视频都能将其归类。

首先，**强人设**是什么意思？强人设是以创作者**个人**为切入点的内容。它必须与你自己密切相关。

以下是强人设常见的切入点：

*   **个人经验与职业**：分享你的行业经验，例如装修、办公、房产、汽车、餐饮、婚介、法律咨询、电商等领域。
*   **个人兴趣与生活**：分享你的兴趣爱好，例如美食、旅行、手工、宠物、星座占卜、美妆、护肤、服装穿搭、游戏、摄影等。

强人设的核心是分享与你个人相关的工作、生活和爱好，并且大部分需要真人出镜。

那么，**弱人设**就是与创作者本人关系不大的内容，核心目的是运营好账号本身。

以下是弱人设的例子：

*   **情感语录账号**：发布情感文案配音乐，无真人出镜或配音。
*   **影视解说账号**：虽然有人声解说电影，但重点在电影内容，与解说者个人特质关联弱。

弱人设的内容更侧重于主题本身，而非突出创作者个人。

理解了人设的强弱之分，我们才能更好地规划内容。接下来，我们看看具体的内容形式。只要内容与你个人相关，都可以制作成视频。我们首先讲解第一种广泛应用的形式：**讲故事**。

讲故事可以分享客户故事、从业经历或人生感悟。故事本身不要求完全真实，可以从文案平台寻找有趣的故事进行分享。而讲故事最常用的方式就是**口播**。

口播即用嘴巴讲述，它有很多分类。以下是常见的口播类型：

*   **故事型口播**：如讲述各种经历、故事，目的是让观众觉得有趣、有共鸣。
*   **知识型口播**：如育儿、金融、财经、养生、装修、房产、汽车等领域的知识分享，目的是让观众在几十秒内获得“收获感”，觉得学到了东西。

口播形式适合众多领域，如法律、保险、金融、装修知识科普等。它的优点是拍摄简单，只需用手机录制，关键在于准备好文案。

对于口播，初学者常有两个疑问：

1.  **口才不好能做口播吗？**
    *   可以。口播是**提前准备好稿子**再录制，并非临场发挥。很多口播者的流畅表达都源于充分准备。
2.  **普通话不好能做口播吗？**
    *   可以。如果你的目标客户在同城，使用**方言**可能效果更好，能增加亲切感或趣味性。同时，可以配上字幕确保观众理解。

口播是一种非常实用且易于上手的内容形式，在后续课程中我们会详细讲解其拍摄技巧。

本节课中我们一起学习了**强人设**与**弱人设**的核心区别，并深入探讨了**口播**这种内容形式的特点、分类及常见问题的解决方案。明确人设方向并掌握口播技巧，是打造高权重抖音账号的重要步骤。
