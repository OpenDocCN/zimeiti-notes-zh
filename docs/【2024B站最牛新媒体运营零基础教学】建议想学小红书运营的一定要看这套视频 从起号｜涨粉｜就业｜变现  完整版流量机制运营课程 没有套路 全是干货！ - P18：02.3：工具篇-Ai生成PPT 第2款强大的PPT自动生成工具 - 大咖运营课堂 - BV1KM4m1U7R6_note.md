# 新媒体运营：P18：工具篇-AI生成PPT（第二款）🚀

在本节课中，我们将学习第二款强大的AI生成PPT工具。这款工具在生成速度、稳定性和模板丰富度方面具有优势，能与上一节介绍的工具形成互补，共同提升我们的PPT制作效率。

上一节我们介绍了一款AI生成PPT工具，本节中我们来看看另一款同样高效但侧重点不同的工具。

## 工具核心优势 ⚡

这款工具主要有两个优势。第一是生成速度快且稳定。第二是它拥有数量庞大、样式丰富的模板库，方便我们进行细节调整和风格选择。

要获得理想的生成效果，关键在于我们需要准备一份**内容详细、结构清晰的大纲**。工具会基于我们提供的大纲内容来分析并匹配合适的模板。

## 两种生成方式 🛠️

该工具提供两种核心的PPT生成方式。

以下是两种方式的具体说明：
1.  **导入本地大纲**：将预先准备好的文档大纲导入，由工具自动生成PPT。这是更常用、控制度更高的方式。
2.  **AI智能生成**：直接输入主题，由AI自动构思大纲并生成完整的PPT内容和页面。

## 实战演示：导入本地大纲 📄

我们首先演示“导入本地大纲”的方式。为了获得大纲，我们可以先借助ChatGPT来生成。

我们向ChatGPT输入指令：
```text
请为“装修行业在小红书平台的运营推广计划”生成一份详细的大纲，并完善每一个小节点的具体内容信息，保持好结构和格式排版。
```

获得大纲文本后，我们进入PPT生成工具的操作界面。

以下是操作步骤：
1.  选择“导入本地大纲”功能。
2.  上传包含大纲内容的本地文件。
3.  工具会快速解析文档结构，提取出大纲。
4.  在丰富的模板库中，选择一个符合“装修行业”风格的模板。
5.  点击“生成PPT”，稍等片刻即可完成。

生成完成后，我们进入编辑界面。如果发现某些页面的内容有缺失，是因为原始大纲中对应节点的内容不够详细。

此时，我们可以使用“大纲编辑”功能进行补充。例如，在“内容趋势”节点下，我们可以回车新增具体描述（如“当前流行的内容类型”），然后让工具重新生成该页面。

在编辑界面中，我们可以进行全面的调整：
*   **文字/形状/背景设置**：调整页面上各元素的样式。
*   **页面布局更换**：如果对某一页的排版不满意，可以单击该页面，从推荐的多种布局中选择一键更换。
*   **整体模板替换**：生成后仍可点击“模板替换”，为整个PPT快速更换另一套设计风格，且响应速度很快。

## 实战演示：AI智能生成 🧠

接下来，我们演示“AI智能生成”方式。

以下是操作步骤：
1.  选择“AI智能生成”功能。
2.  在输入框中填入明确的主题，例如：“装修行业在小红书平台的运营推广计划”。
3.  点击生成，AI会先自动构思一份详细的大纲。
4.  大纲确认后，同样进入模板选择步骤，挑选一个合适的样式。
5.  点击“生成PPT”，AI将根据主题和大纲，自动合成每一页的详细文案内容。

这种方式生成的PPT，其页面内容完整度通常更高，因为AI自行填充了细节文案。生成后，我们同样可以进入编辑界面，对字体、样式、模板等进行自定义调整，或通过“大纲编辑”功能对内容进行修正。
