在本节课中，我们将要学习剪映App中一个非常实用且高效的功能——“剪同款”。这个功能可以帮助你快速套用热门模板，制作出风格统一的视频。我们将分步讲解如何使用剪同款、如何替换模板中的素材以及如何更换模板的音乐，让初学者也能轻松上手。

## 认识“剪同款”功能

上一节我们介绍了剪映的基础界面，本节中我们来看看“剪同款”功能。在剪映App的底部菜单栏中，你可以找到“剪同款”选项。这个功能与抖音的“剪同款”类似，允许你使用他人创作的视频模板。

![](img/ebc4993303736fc20c0b39683b00339e_1.png)
![](img/ebc4993303736fc20c0b39683b00339e_2.png)

以下是“剪同款”界面的主要区域：
*   **推荐/关注**：这里会展示系统推荐的模板或你关注的创作者发布的模板。
*   **模板分类**：例如“卡点”音乐类模板，非常受欢迎。
*   **模板信息**：每个模板会显示使用量，部分模板需要“解锁草稿”才能获得更灵活的编辑权限。

## 如何使用“剪同款”制作视频

了解了功能入口后，我们来看看具体如何操作。使用“剪同款”制作视频的过程非常简单直接。

操作步骤如下：
1.  在“剪同款”页面，浏览并选择一个你喜欢的模板，点击进入。
2.  点击模板下方的 **“剪同款”** 按钮。
3.  系统会提示你选择照片或视频素材。根据模板要求（例如显示需要10个素材），依次点击选择手机相册中的内容。
4.  选择完成后，点击 **“下一步”**。你也可以选择 **“直接拍摄”** 来现场录制素材。
5.  进入预览和编辑页面，确认无误后点击 **“导出”** 即可生成视频。

## 如何替换模板中的素材

有时我们对自动生成的视频效果不满意，希望替换或调整某个片段。本节中我们学习如何替换模板中的原始素材。

操作步骤如下：
1.  在生成视频的编辑页面，找到并点击 **“编辑”** 按钮。
2.  在时间轴上，找到你想要替换的素材片段，点击它。
3.  下方会出现操作菜单，点击 **“替换”** 按钮。
4.  从手机相册中重新选择新的照片或视频。
5.  如果对素材的时长或片段不满意，可以点击 **“裁剪”** 按钮。通过拖动时间轴选取框，可以精确选择要保留的视频部分。
6.  重复以上步骤，替换或裁剪其他素材片段。
7.  所有调整完成后，点击 **“导出”** 保存最终视频。

## 如何更换模板的音乐

模板自带的音乐可能不符合你的需求。下面我们学习如何为“剪同款”生成的视频更换背景音乐。

操作步骤如下：
1.  首先，使用“剪同款”功能生成并导出一个视频。
2.  返回剪映首页，点击 **“开始创作”**，导入刚才生成的视频。
3.  在剪辑界面，选中视频轨道，点击下方工具栏中的 **“音量”** 按钮，将 **“原声”** 音量拖到0，以关闭模板原音。
4.  点击下方工具栏的 **“音频”** -> **“音乐”**，从音乐库中选择一首新歌曲，点击 **“使用”**。
5.  添加新音乐后，如果音乐长度与视频不匹配，可以选中音频轨道，将多余的部分分割删除。
6.  调整完成后，导出视频即可。这样就完成了模板音乐的替换。

本节课中我们一起学习了剪映“剪同款”功能的核心用法。我们掌握了如何套用模板快速生成视频、如何替换模板内的素材以及如何更换模板的背景音乐。这个功能对于快速响应抖音热点、制作风格化视频非常有帮助。你自己制作的视频，同样可以上传成为模板供他人使用。
