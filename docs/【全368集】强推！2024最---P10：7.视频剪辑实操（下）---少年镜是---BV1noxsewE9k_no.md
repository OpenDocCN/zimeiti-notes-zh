# 自媒体运营教程：P10：7. 视频剪辑实操（下）

上一节我们介绍了视频素材的导入与基础剪辑，本节中我们来看看如何通过添加转场和字幕，让视频内容衔接更自然，信息传达更清晰。

## 添加视频转场

![](img/9e084eea29e28b0b9c3752c90adb4409_5.png)

![](img/9e084eea29e28b0b9c3752c90adb4409_6.png)

![](img/9e084eea29e28b0b9c3752c90adb4409_7.png)

一个完整的视频通常由多个片段组成。为了让观众明确感知到不同片段之间的内容转换，避免观看时产生困惑，我们需要在片段之间添加转场。

![](img/9e084eea29e28b0b9c3752c90adb4409_8.png)

![](img/9e084eea29e28b0b9c3752c90adb4409_9.png)

**为什么必须添加转场？**
*   **明确内容切换**：转场能提示用户，当前片段已结束，即将开始新的内容。
*   **提升观看体验**：若无转场，观众在切换到不同内容时可能会感到突兀，甚至退出视频查看标题，这将直接导致**视频完播率下降**。
*   **影响收益**：完播率是平台评估视频质量的重要指标，会直接影响视频的推荐量和最终收益。

**如何添加转场？**
以下是添加转场的具体步骤：
1.  在视频轨道上，找到并点击右侧的 **`+`** 号按钮。
2.  在弹出的菜单中，选择 **`素材库`**。
3.  在素材库的 **`热门`** 分类中，可以找到大量预设的转场素材（例如搞笑片段、动画效果等）。
4.  选择转场时，请注意时长。**建议选择3到5秒的转场**，时长过久（如16秒）会影响视频节奏，降低观众耐心。
5.  选中合适的转场素材，点击添加。

![](img/9e084eea29e28b0b9c3752c90adb4409_11.png)

![](img/9e084eea29e28b0b9c3752c90adb4409_12.png)

**调整转场视频比例**
添加转场后，必须确保其画面比例与主视频保持一致（通常为**16:9**的横屏比例）。
1.  选中已添加的转场片段。
2.  在编辑菜单中找到 **`画布模糊`** 或 **`背景`** 选项。
3.  选择 **`画布模糊`** 下的清晰样式，系统会自动将背景填充为16:9的横屏。
4.  最后，点击右下角的 **`√`** 确认。

## 为视频添加字幕

![](img/9e084eea29e28b0b9c3752c90adb4409_14.png)

![](img/9e084eea29e28b0b9c3752c90adb4409_15.png)

字幕能帮助观众更好地理解视频内容，尤其在嘈杂环境或无声观看时。添加字幕可以显著提升视频的专业度和信息传达效率。

![](img/9e084eea29e28b0b9c3752c90adb4409_17.png)

**如何快速添加字幕？**
以下是使用剪映自动识别字幕的方法：
1.  点击手机剪辑界面左下角的 **`文字`** 选项。
2.  选择 **`新建文本`**，然后点击右侧的 **`识别字幕`** 功能。
3.  点击 **`开始匹配`**，软件会自动识别视频中的语音，并生成对应的字幕文本。
4.  字幕生成后，可以进行样式调整：
    *   **放大字幕**：拖动字幕文本框右下角，向外拉伸即可放大。
    *   **调整位置**：将放大后的字幕文本框，拖动到视频画面底部的位置，通常观感最舒适。
    *   **注意**：调整好第一段字幕后，后续识别的字幕通常会保持相同的样式设置，无需重复调整。

![](img/9e084eea29e28b0b9c3752c90adb4409_18.png)

![](img/9e084eea29e28b0b9c3752c90adb4409_19.png)

![](img/9e084eea29e28b0b9c3752c90adb4409_20.png)

本节课中我们一起学习了视频剪辑后半部分的核心操作：为多片段视频添加转场以提升流畅度与完播率，以及为视频添加字幕以增强信息传达效果。记住，保持**16:9**的横屏比例、使用**3-5秒**的转场、并利用**识别字幕**功能，是高效制作合格自媒体视频的关键步骤。严格按照这些步骤操作，你的视频将更具专业性，更容易通过平台审核并获得更好的传播效果。
