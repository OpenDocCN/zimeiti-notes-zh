# 千川付费撬动自然流：1：核心逻辑与数据复盘 📊

在本节课中，我们将学习如何通过千川付费投放有效撬动直播间自然流量。我们将通过一个真实的实操案例复盘，解析其背后的核心逻辑与关键步骤，帮助你理解付费流量如何为自然流量服务。

## 数据增长轨迹 📈

上一节我们介绍了课程目标，本节中我们来看看实现增长的具体数据表现。以下是案例账号从起号到自然流爆发的关键数据节点：

*   **6月9日**：直播间成交额（GMV）为8103元。此时自然流推流速度约为4-8人/秒，表明这并非一个本身具备大流量的账号。
*   **6月10日**：成交额约1万元。
*   **6月29日**：成交额开始显著增长，达到2.5万元，千次观看成交金额（GPM）为3100元。
*   **7月3日**：成交额2.7万元，GPM为2900元。
*   **7月6日**：成交额突破至5万元，GPM提升至3800元。此时自然流推流速度已增长至约17-19人/秒。
*   **7月9日**：成交额做到7万元。自然流推流速度从开播时的约10人/秒持续上涨，下播时达到77人/秒。
*   **7月13日至16日**：成交额在11万至18万之间波动。在成交18万的场次中，自然流推流速度峰值达到197-228人/秒。

可以看到，随着付费投放策略的推进和直播间数据的优化，自然流量的推流速度发生了显著提升。

## 三大核心前提 🔑

在了解具体操作步骤前，我们必须先明确几个至关重要的前提。如果这些基础不牢固，后续所有操作都可能事倍功半。

以下是实现付费撬动自然流的三个核心重点：

1.  **品类的选择**：并非所有商品都适合通过付费撬动自然流。有些品类天然难以打出自然流量。在选择赛道前，需评估品类的流量潜力。
2.  **必须采用直投直播间画面**：计划类型至关重要。只有投放“直投直播间”计划，计划产生的各项成交数据（如GPM、转化率）才会累积在**直播间本身**，从而影响直播间的推荐标签。如果一直投放短视频，数据则累积在短视频上，直播间标签难以建立。
    *   `计划载体 = 短视频 -> 数据累积于短视频`
    *   `计划载体 = 直播间直投 -> 数据累积于直播间`
3.  **人货场的转化能力**：主播、产品、场景布局必须具备强大的转化能力。平台推流的根本逻辑是“先有成交，后有推流”。所有技巧都需建立在扎实的“人货场”基础上，才能产生有效的成交数据。

## 实操起号步骤 🚀

明确了核心前提后，我们进入具体的操作环节。本节将分步讲解如何从新号开始，通过付费投放拉动自然流量。

### 第一步：拉高千次观看成交金额（GPM）

起号初期，首要目标不是追求付费投入产出比（ROI），而是全力拉高直播间的**千次观看成交金额（GPM）**。

**逻辑解析**：GPM的高低综合反映了**流量精准度**和**直播间承接能力**。流量精准但承接差，GPM不高；承接好但流量不精准，GPM也不会高。因此，高GPM是直播间获得系统大量推流的前提，它代表了高转化率、高成交密度以及优于同行的赛马价值。

**计划打法**：此阶段主要使用**人群包计划**进行投放。虽然人群包流量成本较高、ROI可能偏低，但其核心作用是帮助直播间快速打上精准标签，从而拉升GPM。例如，案例中在7月16日的场次，一条通投搭配人群包（成长人群包-5）的计划消耗了3700元，核心目的就是打标签、拉GPM。

### 第二步：利用高GPM撬动自然流

当直播间通过付费建立起较高的GPM后，就具备了撬动自然流量的基础。

**核心关系**：GPM本身不直接决定推流速度，但它代表了流量质量和承接能力。高GPM使得做出高**成交密度**变得更加容易，而成交密度是影响推流速度的关键因素。

**操作要点**：在GPM拉高后，可以尝试用付费计划去获取一波更高价值的流量。只要直播间能承接住（成交数据好），赛马价值足够，下一场直播的自然流量就极有可能被撬动。案例中数据曲线的几次跃升，都符合这一规律。

### 第三步：避免过度收割导致流量衰退

付费投放是一把双刃剑，使用不当反而会抑制自然流。一个常见的误区是过度依赖“托管ROI”这类强收割计划。

**风险原理**：每一场直播都会积累大量意向客户（即A3人群）。托管ROI计划能高效收割这些存量人群，短期内显著提升ROI，让人误以为找到了“版本答案”。但这本质上是“吃老本”。当存量人群被消耗殆尽，而付费计划又没有为直播间注入新的精准人群（打标签）时，直播间的GPM和付费ROI就会开始下滑，自然流也随之减少。

**案例警示**：例如，某场直播广告流占比7%，整体推流68人/秒，卖货5.8万元。如果此时为了快速提升ROI而全力投放托管计划，短期内可能见效，但几场后往往面临流量衰退。

## 总结与回顾 🎯

本节课中，我们一起学习了通过千川付费投放撬动自然流量的完整逻辑与实操方法。

我们首先回顾了一个成功的增长案例，看到了自然流推流速度随付费策略推进而显著提升的数据轨迹。接着，我们明确了三大核心前提：**选对品类**、**坚持直投**、**夯实人货场**。然后，我们拆解了起号的三步核心操作：**初期不计ROI，专注用人群包计划拉高GPM**；**利用高GPM代表的承接能力去撬动自然流量**；**警惕过度使用托管ROI等收割计划，避免“吃老本”导致的流量衰退**。

记住，付费流量的核心作用在起号阶段是 **“打标签”和“拉数据”** ，为自然流量铺路，而非单纯追求即时ROI。理解并平衡好“拉新”（打标签）与“收割”的关系，是持续获得自然流的关键。
