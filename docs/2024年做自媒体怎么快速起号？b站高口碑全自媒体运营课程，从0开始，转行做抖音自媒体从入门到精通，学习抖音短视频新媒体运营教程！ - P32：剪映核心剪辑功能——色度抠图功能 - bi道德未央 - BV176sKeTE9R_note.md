# 自媒体运营课程：P32：剪映核心剪辑功能——色度抠图功能 🎬

在本节课中，我们将要学习剪映App中的一个核心剪辑功能——**色度抠图**。这个功能非常强大，能帮助我们将绿幕背景的视频素材完美地抠出，并合成到其他视频中，从而制作出各种特效视频。对于初学者而言，掌握这个功能是提升视频创意和制作效率的关键。

## 色度抠图功能简介

上一节我们介绍了剪映的基础剪辑工具，本节中我们来看看**色度抠图**。色度抠图是手机剪辑中一个非常强大的功能，它允许我们移除视频中的特定颜色（通常是绿色或蓝色背景），只保留前景主体。这个功能类似于专业软件如Adobe Premiere或After Effects中的绿幕抠像技术，但现在在手机端的剪映App中也能轻松实现。

通过使用色度抠图，我们可以方便地将特效视频素材（如绿幕素材）合成到自己的视频项目中，极大地丰富了视频的视觉效果。

## 色度抠图实操演示

![](img/2f38a297ba0f30780529bd198e32b273_1.png)

了解了色度抠图的基本概念后，接下来我们通过一个具体的例子来学习如何使用它。我们将制作一个“飞机飞过天空”的特效视频。

以下是操作步骤：

1.  **导入主视频素材**：打开剪映，点击“开始创作”，首先导入一段天空背景的视频作为主场景。

2.  **导入绿幕素材**：点击底部工具栏的“画中画”功能，选择“新增画中画”，导入一段带有绿色背景的飞机特效视频素材。

![](img/2f38a297ba0f30780529bd198e32b273_3.png)

![](img/2f38a297ba0f30780529bd198e32b273_4.png)

3.  **查看原始效果**：在未进行抠图前，画中画视频会覆盖在主视频上，并显示其原始的绿色背景。

![](img/2f38a297ba0f30780529bd198e32b273_6.png)

4.  **应用色度抠图**：选中画中画轨道上的飞机素材，在底部工具栏中找到“色度抠图”选项（通常位于“蒙版”功能旁边）。

![](img/2f38a297ba0f30780529bd198e32b273_8.png)

5.  **取色与调整**：
    *   点击“取色器”（一个圆形图标），将其移动到画面中的绿色背景区域，选取要扣除的颜色。
    *   随后，调整“强度”滑块，逐步增加数值，直到绿色背景完全消失，只留下飞机主体。
    *   如果抠图后主体边缘有残留的杂色或显得生硬，可以适当调整“阴影”滑块，使合成效果更自然。

![](img/2f38a297ba0f30780529bd198e32b273_9.png)

6.  **调整与优化**：
    *   **裁剪**：如果素材带有水印或不必要的部分，可以选中素材后，点击“编辑”->“裁剪”，将多余部分裁掉。
    *   **位置与动画**：调整飞机在画面中的位置，使其飞行轨迹更符合逻辑。还可以为飞机的入场添加“动画”效果（如“渐显”），让合成更流畅。
    *   **时长修剪**：将主视频与画中画素材的时长对齐，剪掉多余的部分，确保视频节奏一致。

完成以上步骤后，预览视频，你就会看到飞机在天空飞过的合成特效了。

![](img/2f38a297ba0f30780529bd198e32b273_11.png)

## 课程总结
