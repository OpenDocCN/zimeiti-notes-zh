# 自媒体运营课程：P16：13.原创解说文案

在本节课中，我们将要学习如何创作原创解说文案。我们将探讨获取文案素材的不同途径，并重点讲解如何用简单直白的方式写出吸引人的解说文案。

## 获取文案素材的途径

上一节我们介绍了影视解说的基本概念，本节中我们来看看如何获取解说所需的文案素材。主要有三种方式可供选择。

### 使用外部素材库

第一种方式是使用如“一搜猫”等外部素材库。但很多同学可能希望使用其他资源。

### 使用讲师提供的原创文库

除了外部素材库，讲师本人拥有一个持续更新的原创解说文库。该文库中的每个文案均为纯原创内容，字数约在两三千字左右。

以下是使用该文库的流程：
1.  获取讲师提供的原创文案。
2.  根据文案内容，去寻找对应的视频素材。
3.  将文案与视频素材进行合成，制作成完整的解说视频。

![](img/6f596f4524b611220b112348d5352900_1.png)

这种方式为创作者提供了现成的优质文案，简化了内容创作过程。

### 自主创作纯原创文案

当然，也有很多同学希望完全自主创作纯原创文案。这种想法值得鼓励，因为纯原创文案拥有显著优势。

![](img/6f596f4524b611220b112348d5352900_3.png)

**核心优势公式：**
`纯原创文案收益 ≈ 使用素材库文案收益 × 2`

纯原创内容的播放量收益单价，通常能达到使用现有素材文案的两倍左右。这是因为平台更青睐和鼓励原创内容。

## 如何创作“说人话”的解说文案

对于想尝试原创但又感到畏惧的同学，请记住一个核心原则：**解说文案就是说人话**。

“说人话”意味着使用大白话、口水话，就像平时给朋友安利一部电影或电视剧一样。它的目的是把事情讲清楚、讲有趣，而不是写成结构严谨的作文。

### 错误 vs 正确的表达方式

为了更直观地理解，我们来对比一下两种不同的表达方式。

**错误的表达方式（像写作文）：**
> “妈妈，我今天看了一部电视剧。该剧名为《XX》，由某导演于某年执导上映。男主角叫A，女主角叫B。第一集讲述了……第二集讲述了……”

**正确的表达方式（说人话）：**
> “妈，我今天看了一部剧，特别好看！男主帅女主美，他俩特别般配。剧情是，他们相爱十年都快结婚了，结果男的居然婚前出轨！女的知道后去捉奸，当场扇了他一巴掌说‘离婚，这婚我不结了！’”

后一种方式生动、有情绪、易于理解，这才是观众想听的“解说”。

### 解说文案的核心价值

解说文案的核心价值在于“翻译”和“提炼”。以《甄嬛传》为例，很多观众可能没时间追完全剧，或者看不懂剧中的文言文对白。

**解说文案的作用代码描述：**
```plaintext
function 解说文案(原始剧集) {
    let 核心情节 = 提取关键事件(原始剧集);
    let 白话版本 = 转换为口语化描述(核心情节);
    return 白话版本; // 输出观众能轻松理解的故事概要
}
```
影视解说者正是通过大白话，将复杂的故事提炼成易懂的片段，满足了观众“快速了解剧情”的需求，因此这个领域始终拥有巨大的市场。

## 高潜力解说领域推荐

正因为解说视频需求大、创作相对直接，它成为了自媒体中非常赚钱的领域。在讲师自己的团队中，从事各类解说创作的伙伴也最多。

以下是常见的解说视频类型：
*   影视剧解说
*   纪录片解说
*   综艺节目解说
*   动物世界解说
*   科普知识解说
*   历史故事解说
*   动漫解说

### 重点领域建议

对于新手，如果想从事影视推广，影视解说是一个有版权保障的明确方向。如果不想做影视类，这里有一个特别推荐：

![](img/6f596f4524b611220b112348d5352900_5.png)

**动物解说**
目前平台上专注动物解说的创作者相对较少，而该领域的内容通常能获得平台较大的流量扶持，竞争蓝海，增长潜力巨大。
