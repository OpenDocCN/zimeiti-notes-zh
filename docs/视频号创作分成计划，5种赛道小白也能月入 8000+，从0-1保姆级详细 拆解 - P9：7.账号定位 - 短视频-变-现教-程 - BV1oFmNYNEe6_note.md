## 解决初始粉丝问题

![](img/c0e82cdf0cd6ba9db5434802091b9c2d_3.png)

首先，需要明确一点：在开始正式运营前，建议将账号之前的作品隐藏。这是为了确保账号内容垂直，给新观众一个清晰的定位印象。

获取初始100个粉丝是第一步。操作很简单：保持稳定的内容更新频率，**一天发布一条视频**。不要急于寻找复杂素材，先专注于完成“获取100粉丝”这个小目标，再解决后续的素材问题。一步步来，思路会更清晰。
