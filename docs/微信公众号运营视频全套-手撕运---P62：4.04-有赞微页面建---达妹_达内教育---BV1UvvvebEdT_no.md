# 微信公众号运营：P62：4.04-有赞微页面建设~2 🛠️

在本节课中，我们将学习如何使用有赞平台建设微页面。我们将了解如何添加和配置各种组件，如商品、图片广告、优惠券等，并学习如何通过组合多个页面来构建一个结构清晰、功能丰富的店铺主页。

## 微页面组件添加与配置

![](img/2edd4eee4c7ff04f118f4a4a756a2ef5_3.png)

![](img/2edd4eee4c7ff04f118f4a4a756a2ef5_5.png)

上一节我们介绍了微页面的基本概念，本节中我们来看看如何向页面中添加和配置具体的组件。

![](img/2edd4eee4c7ff04f118f4a4a756a2ef5_7.png)

微页面的建设基于组件。你可以通过添加不同的组件来自由设计页面布局和内容。

![](img/2edd4eee4c7ff04f118f4a4a756a2ef5_9.png)

以下是可添加的核心组件类型及其配置方法：

*   **商品组件**：用于展示店铺商品。添加后，可以选择不同的展示模板，例如一行三个小商品、一行一个大两个小，或横向滑动展示。
*   **图片广告组件**：用于展示促销或宣传图片。可以放置在页面的任意位置，并设置点击后跳转的链接。
*   **营销组件**：用于开展促销活动。
    *   **优惠券**：添加后可让用户直接领取。
    *   **秒杀**：添加秒杀活动专区，并可设置一行展示的商品数量（如一行三个）。
*   **其他功能组件**：
    *   **关注公众号**：添加引导用户关注公众号的模块，使用前需在后台完成相关设置。
    *   **标题**：添加纯文本标题，用于划分板块或突出主题。
    *   **辅助线与空白**：用于调整页面元素的间距和视觉区分。
        *   **辅助线**：在元素间添加一条分隔线。
        *   **辅助空白**：添加一个空白区域来增加间距。

## 组件内容绑定与跳转设置

添加组件后，需要为其绑定具体的内容或设置跳转目标。

![](img/2edd4eee4c7ff04f118f4a4a756a2ef5_11.png)

![](img/2edd4eee4c7ff04f118f4a4a756a2ef5_13.png)

对于商品和秒杀组件，需要从你的商品库中选择具体的商品进行展示。如果组件内没有商品，它将无法显示内容。

![](img/2edd4eee4c7ff04f118f4a4a756a2ef5_15.png)

![](img/2edd4eee4c7ff04f118f4a4a756a2ef5_17.png)

对于图片广告等可点击的组件，可以设置其跳转目标。跳转目标可以是：
*   具体的商品详情页。
*   已创建的其他微页面。
*   商品分类页（如“最新”或“最热”分类）。

## 页面保存与多层结构

![](img/2edd4eee4c7ff04f118f4a4a756a2ef5_19.png)

配置好所有组件后，点击“保存并继续”，该微页面就会发布并出现在你的店铺页面列表中。

微页面的强大之处在于可以创建多层结构。你可以：
1.  创建多个专注于不同商品或主题的**子页面**（例如“书籍页面”、“视频页面”）。
2.  创建一个**主页**，在主页上使用图片广告等组件，将这些子页面链接起来。
3.  用户点击主页上的图片，即可进入相应的子页面浏览更多商品，从而实现从主页 -> 子页面 -> 商品详情页的流畅导航体验。

这种结构类似于大型电商网站的首页，通过入口图引导用户进入不同的活动专区或商品分类页。

![](img/2edd4eee4c7ff04f118f4a4a756a2ef5_21.png)

![](img/2edd4eee4c7ff04f118f4a4a756a2ef5_23.png)
