# 亚马逊跨境电商入门教程：P11：12、如何注册店铺：资料准备 📝

在本节课中，我们将要学习注册亚马逊店铺前必须准备的各项资料。了解并备齐这些资料，是开启亚马逊运营之旅的第一步。

上一节我们介绍了如何选择目标站点，本节中我们来看看注册店铺需要准备哪些核心材料。

## 资料清单概述

注册亚马逊店铺需要准备六项核心资料。以下是每一项资料的详细说明与注意事项。

### 1. 营业执照彩色扫描件

第一份需要准备的资料是**营业执照的彩色扫描件**（即清晰的照片）。

*   **执照类型**：可以是**企业营业执照**，也可以是**个体工商户营业执照**。
*   **办理途径**：可在当地工商局免费办理。
*   **解决方案**：若没有营业执照，可借用家人或朋友的执照，或寻找代办机构/课程助理老师协助办理，费用通常在几百至一千元人民币左右。

关于营业执照，有以下具体要求：

*   **注册资本与年限**：无硬性要求。无论是1元还是1000万注册资本，无论是新办还是已注册多年的执照，均可使用。
*   **经营范围**：执照经营范围需**带有销售性质**即可（例如：百货零售、服装销售）。
*   **重要提醒**：**切勿**在经营范围中添加“**进出口贸易**”字样。添加此项后，按规定需额外办理进出口手续，属于画蛇添足。
*   **经营范围与销售产品**：执照的经营范围**不需要**与你在亚马逊店铺中计划销售的产品保持一致。例如，使用超市的营业执照，完全可以在亚马逊上销售服装。

### 2. 法人身份证正反面照片

第二份需要准备的资料是**法人身份证的正反面清晰照片**。

这里有一个关键点：**营业执照上的法人，必须与这份身份证的持有人为同一人**。而后续的其他资料，则可以与这两份资料的所有人不同。

### 3. 付款信用卡

第三份需要准备的资料是用于支付店铺月租的**付款信用卡**。

*   **唯一作用**：此信用卡**唯一的作用**就是支付亚马逊的月度店铺租金（目前为39.99美元）。
*   **卡片类型**：需要使用**全币种信用卡**，首选带有 **`Visa`** 或 **`Mastercard`** 标识的卡片。
*   **办理银行**：国内各大银行（如工商银行、建设银行、招商银行、中信银行等）均可办理。
*   **替代方案**：如果没有信用卡，可以借用他人的，或花费约100元购买一张虚拟信用卡，每月按时存入足额月租即可。
*   **扣款逻辑**：此卡仅在店铺无销售额或余额不足时用于扣缴月租。当店铺产生营业额后，亚马逊会优先从店铺余额中扣除月租。

### 4. 海外收款账户

第四份需要准备的资料是**海外收款账户**。

![](img/f03b8f1f1987ec11ebeb9d1265e88282_3.png)

*   **核心功能**：它相当于**国际版的支付宝**。当你的亚马逊店铺产生销售额后，资金会先到达此账户，然后你可以从此账户提现至你个人的国内银行卡。
*   **如何获取**：可以通过第三方支付服务商（如Payoneer、PingPong、万里汇等）的官网免费注册，流程简单，仅需绑定个人身份证和国内收款银行卡即可。

### 5. 电话号码

第五份需要准备的资料是一个**有效的国内手机号码**，用于接收验证码和官方通知。

### 6. 电子邮箱

第六份需要准备的资料是一个**电子邮箱地址**。

*   **邮箱类型**：`163邮箱`、`126邮箱`、`QQ邮箱`等均可使用。
*   **个人推荐**：建议使用 **`163邮箱`**，因其在国内外邮件收发稳定性上表现较好。
