# 剪映手机版教学：P45：文字层级功能详解 🎬

在本节课中，我们将深入学习剪映手机版中一个非常实用的功能——**文字层级**。通过掌握图层顺序的调整，你可以轻松实现文字在视频人物前或人物后显示的效果，从而制作出更具层次感和专业感的视频。

---

上一节我们介绍了通过画中画和混合模式制作文字效果。本节中，我们来看看如何利用文本自带的“层级”功能，更直接地控制文字与视频画面的前后关系。

以下是实现文字位于抠像人物后方的具体操作步骤：

1.  **准备素材**：选中主轨道视频，点击“复制”。将复制出的这段视频切入画中画轨道，并与主视频对齐。接着，对这段画中画视频进行“智能抠像”。

2.  **创建文本**：返回主菜单，点击“文本”，选择“新建文本”。输入你想要的文字内容，例如“从广州路过”。为文字选择一种字体（如创意字体中的“苏打体”），并调整到合适大小。

3.  **调整文本层级**：选中刚刚创建的文本，在下方工具栏中找到并点击“层级”。在弹出的窗口中，点击顶部选项，将“仅文字贴纸”切换为“全部轨道”。

4.  **改变图层顺序**：此时，列表中会显示包括“画中画”在内的所有轨道。找到代表文字的图层，按住并拖动，将其放置在“画中画”图层的**下方**。这样，文字就移动到了抠像人物的后面。

5.  **为文字添加动画**：选中文本，在开头位置添加一个**关键帧**，并将文字拖到视频画面右侧之外。接着，将时间轴移动到视频结尾处，将文字拖回画面左侧之外。这样就为文字创建了一个从右向左滑入再滑出的动画效果。

通过以上步骤，我们同样可以实现文字在人物背后移动的效果。这与上节课先导出文字视频再通过“绿色”混合模式叠加的方法不同，本节课是直接利用文本的“层级”功能来调整顺序。

---

为了帮助大家更好地理解，我们来梳理一下本节涉及的**核心概念**：

*   **图层关系**：视频剪辑可以理解为多个图层的叠加。通常，**上方的图层会遮盖下方的图层**。在本例中，涉及的图层从下到上依次是：
    1.  **主轨道视频**（背景）。
    2.  **文本图层**（本节课我们将其调整到了这个位置）。
    3.  **画中画图层**（抠像人物）。

*   **层级功能**：“层级”窗口是调整这些图层前后顺序的控制面板。切换到“全部轨道”模式，可以管理所有类型的轨道（视频、画中画、文本、贴纸等）的显示顺序。

---

本节课中我们一起学习了剪映手机版的“文字层级”功能。关键点在于理解**全部轨道**下的图层顺序调整，这能让你更灵活地控制文字、贴纸与视频画面之间的前后遮挡关系，是制作复杂画面效果的基础。请务必理解主轨道、画中画轨道和文本轨道之间的层次关系，并通过课后练习来巩固这一重要技能。
