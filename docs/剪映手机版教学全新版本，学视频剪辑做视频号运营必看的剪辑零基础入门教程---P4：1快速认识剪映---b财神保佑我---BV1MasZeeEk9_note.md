## 1. 剪映是什么？ 🤔

剪映是由抖音官方推出的一款手机视频编辑工具，可用于手机短视频的剪辑、制作和发布。它拥有全面且简洁的剪辑操作功能，支持**变速**、**滤镜**、**转场**等多种剪辑效果，并带有抖音官方的丰富音乐素材资源。

剪映的开发和运营公司是深圳脸萌科技有限公司。该公司已被字节跳动（抖音母公司）收购。为了推动抖音及头条系在短视频赛道的发展，官方于2019年3月推出了剪映APP。

了解这一点很重要，因为它意味着通过剪映制作并发布的视频，在抖音平台上能获得官方的流量扶持和推荐，这对想在抖音发展的用户有极大帮助。

如果您的手机尚未安装剪映，建议在课程结束后立即下载。我们后续的教程都将围绕此APP展开。

![](img/d7ae0839f455708933d15d04850504ac_7.png)

**小结：剪映是什么？**
剪映是由抖音官方推出的一款**完全免费**且超级好用的手机剪辑APP。与许多需要付费开通会员的剪辑软件（如快剪辑、爱剪辑、Videoleap、巧影等）不同，剪映完全免费，界面简洁，功能强大，对新手非常友好，并可一键分享至抖音平台。

## 2. 剪映做短视频的优势有哪些？ ✨

上一节我们介绍了剪映的定义，本节中我们来看看它为何备受推崇。使用剪映制作短视频主要有五大优势。

以下是剪映的五大优势：

1.  **简单好用**：操作界面和功能设计简洁直观，零基础用户也能快速上手，没有复杂的剪辑逻辑。
2.  **完全免费**：无需开通会员或为素材、特效付费，是一款真正的免费神器。
3.  **功能强大**：尽管版本持续更新，但现有功能已非常强大，能满足抖音短视频制作的绝大部分需求，例如：
    *   **文本**、**音频**、**贴纸**、**特效**编辑。
    *   近期上线的**色度抠图**功能，可通过画中画功能处理绿幕素材，实现合成特效视频。
4.  **海量素材**：内置大量抖音热门的视频素材和**BGM（背景音乐）**，方便用户快速制作同款热门视频，无需自行寻找素材。
5.  **官方推荐**：剪映与抖音同属字节跳动公司。通过剪映剪辑并发布的视频，在抖音平台会获得默认的流量推荐和扶持。这也是很多抖音视频下方标注“剪映分享”的原因。

**小结：使用剪映制作短视频的五大优势**
1.  简单好用。
2.  完全免费。
3.  功能强大。
4.  海量素材。
5.  官方推荐。

## 3. 哪些人适合用剪映做短视频？ 🎯

![](img/d7ae0839f455708933d15d04850504ac_14.png)

![](img/d7ae0839f455708933d15d04850504ac_16.png)

刚才我们了解了剪映的优势，那么它最适合哪些人群使用呢？虽然剪映对新手非常友好，但它并非适用于所有人。我们归纳了三类特别适合使用剪映的人群。

![](img/d7ae0839f455708933d15d04850504ac_18.png)

以下是最适合使用剪映的三类人群：

1.  **喜欢拍摄的人群**：在移动互联网时代，人们倾向于用手机完成各种操作。相比专业的电脑剪辑软件（如PR、AE），剪映能让用户随时随地快速完成视频剪辑，提升效率。
2.  **愿意分享的人群**：抖音等短视频平台鼓励用户分享生活点滴、故事与感悟。如果你有内容渴望记录并分享，剪映是便捷的制作工具。
3.  **短视频新手/小白**：对于想在短视频领域深耕或尝试自媒体创作的新手，剪映是绝佳的入门工具。它技术要求低，操作简单，能让新手快速上手并产出作品。

**小结：哪些人适合用剪映做短视频？**
1.  喜欢拍摄的人群。
2.  愿意分享的人群。
3.  短视频新手。
