# 抖音运营实战：P24：视频剪辑与提词工具使用教程 🎬

![](img/49509797a202b7ac4e265702de22128c_1.png)

在本节课中，我们将学习如何使用剪映App进行视频剪辑，以及如何利用爱提词App辅助口播视频的拍摄。课程将包含从导入素材、剪辑、添加音乐字幕到最终导出的完整流程，并介绍一个能提升口播效率的实用工具。

上一节我们介绍了视频策划与素材准备，本节中我们来看看如何将这些素材剪辑成完整的视频。我们将使用抖音官方出品的“剪映”App进行实操教学。

![](img/49509797a202b7ac4e265702de22128c_9.png)

首先，请确保在手机应用商店下载**黑色图标**的“剪映”App。它同时提供手机版和电脑版，本节课我们以手机版为例进行讲解。

![](img/49509797a202b7ac4e265702de22128c_11.png)

![](img/49509797a202b7ac4e265702de22128c_13.png)

打开剪映App，点击“开始创作”按钮。

![](img/49509797a202b7ac4e265702de22128c_1.png)

在相册中选择你需要剪辑的视频片段，点击“添加”将其导入工程。如果你的视频由多个片段（如开头、中间、结尾）组成，建议**一次只导入一个片段**进行剪辑，完成后再点击轨道尾部的“+”号添加下一个片段。这样可以避免剪辑顺序混乱。

![](img/49509797a202b7ac4e265702de22128c_15.png)

![](img/49509797a202b7ac4e265702de22128c_17.png)

我们将以一条22秒的视频为例进行剪辑。添加后，我们首先检查并调整视频的构图与比例。

![](img/49509797a202b7ac4e265702de22128c_19.png)

**第一步：调整视频构图与比例**
1.  点击时间轨道上的视频片段，使其被选中。
2.  在底部工具栏中向右滑动，找到并点击“编辑”图标（一个方框形状）。
3.  进入编辑界面后，点击左侧的“旋转”按钮，将视频调整到正确的方向。
4.  如果需要裁剪画面比例，点击第三个“裁剪”按钮。这里提供了多种比例选项，如**1:1**、**4:3**、**16:9**等，你可以根据抖音的展示需求或个人喜好进行选择。

![](img/49509797a202b7ac4e265702de22128c_21.png)

![](img/49509797a202b7ac4e265702de22128c_23.png)

调整完毕后，点击右下角“√”确认，并返回主编辑界面。

![](img/49509797a202b7ac4e265702de22128c_25.png)

![](img/49509797a202b7ac4e265702de22128c_9.png)

![](img/49509797a202b7ac4e265702de22128c_27.png)

![](img/49509797a202b7ac4e265702de22128c_29.png)

**第二步：剪辑视频内容**
接下来，我们需要剪掉视频中无效或多余的部分。播放视频，找到需要裁剪的起始位置。

![](img/49509797a202b7ac4e265702de22128c_31.png)

![](img/49509797a202b7ac4e265702de22128c_33.png)

以下是剪辑操作的核心步骤：
1.  将时间线（白色竖线）移动到需要分割的位置。
2.  选中视频片段，点击底部工具栏中的“分割”按钮。视频将在时间线位置被切成两段。
3.  选中需要删除的片段，点击“删除”按钮。

![](img/49509797a202b7ac4e265702de22128c_35.png)

这个操作可以概括为一个简单流程：`选中片段 -> 点击“分割” -> 选中多余部分 -> 点击“删除”`。重复此操作，即可去除视频开头、中间或结尾所有不需要的部分。

![](img/49509797a202b7ac4e265702de22128c_37.png)

![](img/49509797a202b7ac4e265702de22128c_39.png)

![](img/49509797a202b7ac4e265702de22128c_15.png)

![](img/49509797a202b7ac4e265702de22128c_41.png)

![](img/49509797a202b7ac4e265702de22128c_43.png)

**第三步：添加背景音乐**
视频内容剪辑好后，我们需要为其添加背景音乐。点击底部工具栏中的“音频”按钮。

![](img/49509797a202b7ac4e265702de22128c_45.png)

添加音乐有两种常用方式：
1.  **提取音乐**：如果你已经用其他工具（如淘金阁）生成了配音或音乐文件，可以点击“提取音乐”，直接从手机视频中提取音频。
2.  **添加音乐**：点击“音乐”，进入剪映自带的音乐库。你可以根据分类（如美食、美妆、vlog）选择，或直接搜索歌曲名称。点击歌曲旁边的“下载”图标即可试听，满意后点击“使用”添加到轨道。

![](img/49509797a202b7ac4e265702de22128c_47.png)

添加音乐后，如果音乐长度超过视频，可以使用同样的`分割->删除`方法进行裁剪。你还可以选中音乐轨道，点击“音量”按钮，调整背景音乐的音量大小，使其不掩盖人声。

![](img/49509797a202b7ac4e265702de22128c_49.png)

![](img/49509797a202b7ac4e265702de22128c_51.png)

![](img/49509797a202b7ac4e265702de22128c_27.png)

**第四步：自动识别并添加字幕**
字幕能极大提升视频的观看体验。点击底部工具栏的“文本”按钮，然后选择“识别字幕”。

![](img/49509797a202b7ac4e265702de22128c_53.png)

![](img/49509797a202b7ac4e265702de22128c_55.png)

在弹出窗口中，建议选择“仅视频”，以避免识别到背景音乐里的歌词。点击“开始识别”，系统会自动为视频中的人声生成字幕，无需手动输入。

![](img/49509797a202b7ac4e265702de22128c_57.png)

识别完成后，检查字幕是否有错别字。双击字幕条即可进行修改。你还可以选中字幕，在“样式”和“花字”选项中，更改字体、颜色、大小等，让字幕更美观。

![](img/49509797a202b7ac4e265702de22128c_59.png)

![](img/49509797a202b7ac4e265702de22128c_61.png)

![](img/49509797a202b7ac4e265702de22128c_37.png)

**第五步：调整参数并导出视频**
在导出前，有一个关键步骤能保证视频清晰度。点击右上角的“导出”按钮。

![](img/49509797a202b7ac4e265702de22128c_63.png)

在导出设置中，请调整以下两个参数：
*   **帧率 (fps)**：设置为 **30**。这是抖音平台最适配的帧率。
*   **分辨率**：尽可能选择最高选项（如1080p或4K）。分辨率越高，视频越清晰。

![](img/49509797a202b7ac4e265702de22128c_65.png)

![](img/49509797a202b7ac4e265702de22128c_67.png)

设置完成后，点击“导出”。**导出过程中请勿锁屏或切换App**，耐心等待导出完成。导出后，你可以直接通过剪映的“分享到抖音”按钮发布作品。

![](img/49509797a202b7ac4e265702de22128c_51.png)

![](img/49509797a202b7ac4e265702de22128c_69.png)

![](img/49509797a202b7ac4e265702de22128c_71.png)

![](img/49509797a202b7ac4e265702de22128c_73.png)

![](img/49509797a202b7ac4e265702de22128c_75.png)

**口播提词神器：爱提词App**
拍摄口播视频时，记不住或背不下台词是常见难题。“爱提词”这款App可以完美解决这个问题。

![](img/49509797a202b7ac4e265702de22128c_77.png)

![](img/49509797a202b7ac4e265702de22128c_79.png)

1.  **添加台词**：打开爱提词App，点击右下角“+”号，创建新台词稿。你可以直接输入或粘贴复制好的文案，并为其设置一个标题（如“情感语录”），然后保存。
2.  **使用提词器**：保存后，点击“开始提词”，选择“录制模式”。此时，你准备好的台词会以悬浮窗形式滚动显示在手机屏幕上，你可以一边看着台词，一边进行录制，完全无需背诵。
3.  **个性化设置**：点击屏幕中央的“设置”按钮，可以调整台词滚动的速度、字体大小和颜色，方便阅读。
4.  **内置美颜与文案库**：该App还提供了基础的美颜功能（如瘦脸、大眼、磨皮），拍摄效果比原相机更自然。此外，它的“爆款文案”库中包含了职场、情感、生活等多个领域的现成文案，可供参考和灵感启发。
