# 新媒体运营精品教程：P16：02.1：工具篇-全平台短视频文案一键提取 📝

在本节课中，我们将学习一个非常实用的运营工具，它能帮助我们从抖音、小红书、哔哩哔哩等多个平台的短视频中，一键提取出完整的文案内容。掌握这个工具，可以极大地提升我们收集素材和创作内容的效率。

## 工具介绍与优势

上一节我们介绍了文案提取的需求，本节中我们来看看这个强大的在线工具。传统提取文案的方法通常比较繁琐，例如使用“轻抖”小程序，或者先将视频下载下来，再导入“剪映”等软件进行识别。这些方法往往只适用于单一平台，操作步骤较多。

而今天分享的这个工具，其核心优势在于**全平台在线一键提取**。你只需要获得目标短视频的链接，将其粘贴到工具中，即可快速获取文案，无需下载视频或切换多个软件。

以下是该工具的主要特点：
*   **支持多平台**：适用于抖音、快手、小红书、哔哩哔哩、视频号等主流平台。
*   **在线操作**：无需安装软件，在网页中即可完成。
*   **操作简便**：核心操作只有两步：`复制链接` -> `点击提取`。
*   **附加功能**：提供一键复制、文案格式化（整理为字幕格式）以及本地视频上传识别功能。

### 示例一：提取抖音视频文案

首先，我们打开抖音，找到一个想要提取文案的视频，例如一段“情感语录”。

![](img/bb26169ed94fd55fa4d3d968c7b0ab50_5.png)

![](img/bb26169ed94fd55fa4d3d968c7b0ab50_7.png)

![](img/bb26169ed94fd55fa4d3d968c7b0ab50_1.png)

**操作步骤如下：**
1.  在抖音APP中，点击视频分享按钮，选择“复制链接”。
2.  打开文案提取工具网页，将复制好的链接粘贴到输入框中。
3.  点击“提取文案”按钮。

![](img/bb26169ed94fd55fa4d3d968c7b0ab50_9.png)

![](img/bb26169ed94fd55fa4d3d968c7b0ab50_3.png)

稍等片刻，工具就会自动识别并显示出视频的完整文案。

![](img/bb26169ed94fd55fa4d3d968c7b0ab50_11.png)

![](img/bb26169ed94fd55fa4d3d968c7b0ab50_5.png)

![](img/bb26169ed94fd55fa4d3d968c7b0ab50_13.png)

提取出的文案内容准确，例如：“听完你就释怀了，村上春树说，我动了离开你的念头，不是因为你不好，也不是因为不爱了……”。我们可以直接点击“复制”使用，或使用“格式化”功能让其更易读。

### 示例二：提取哔哩哔哩视频文案

接下来，我们测试一下对哔哩哔哩平台视频的支持情况。同样，先找到一个B站视频并复制其链接。

![](img/bb26169ed94fd55fa4d3d968c7b0ab50_7.png)

**操作步骤完全相同：**
1.  复制B站视频链接。
2.  将链接粘贴到工具输入框。
3.  点击“提取文案”。

![](img/bb26169ed94fd55fa4d3d968c7b0ab50_9.png)

工具成功提取出了B站视频的文案，例如：“新手做直播电商，怎么才能打造出直播爆品？在直播销售的不同阶段……”。这证明了该工具跨平台提取的准确性和高效性。
