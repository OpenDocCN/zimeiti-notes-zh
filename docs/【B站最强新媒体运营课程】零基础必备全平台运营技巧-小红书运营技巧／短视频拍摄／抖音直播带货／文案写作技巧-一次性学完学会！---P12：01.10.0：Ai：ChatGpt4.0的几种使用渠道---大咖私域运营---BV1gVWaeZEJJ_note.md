# 新媒体运营与AI应用：P12：01.10.0：ChatGPT 4.0的几种使用渠道 🚀

## 概述
在本节课中，我们将学习如何获取和使用目前最强大的AI对话模型——ChatGPT 4.0。对于新媒体运营人员而言，掌握AI工具可以极大地提升工作效率，例如撰写脚本、制作PPT和生成报告。我们将介绍几种不同的使用渠道，包括免费和付费选项，并分析各自的优缺点。

## 微软Edge浏览器Copilot
上一节我们提到了AI对运营工作的重要性，本节中我们首先来看看一个集成在Windows系统中的便捷工具。

![](img/6373b3fa8a8c35d160a369c3bccce2b5_1.png)

![](img/6373b3fa8a8c35d160a369c3bccce2b5_3.png)

微软基于ChatGPT 4.0模型，在Edge浏览器中内置了名为“Copilot”的AI助手。它提供了三种对话模式：
*   **更有创造力**：AI的回答更具开放性和想象力。
*   **平衡**：介于创造力和精确性之间，适合一般用途。
*   **更精确**：回答更为严谨和准确，适合专业内容。

**使用前提**：
1.  需要登录微软账号（可使用Outlook邮箱注册）。
2.  需要能够访问国际网络的工具（俗称“梯子”）。

**使用方法**：
在登录账号并设置好网络后，打开Edge浏览器，即可在侧边栏或系统任务栏找到Copilot图标并开始对话。例如，你可以输入：
```text
请帮我写一个关于装修的短视频脚本，时长控制在3分钟左右。
```
Copilot会基于你的要求生成相应的内容。它的一个优势是能够分析你当前打开的网页内容。

**优点与缺点**：
*   **优点**：免费、集成度高、使用方便，可进行一定程度的连续对话。
*   **缺点**：
    1.  **精准度稍逊**：相比官方ChatGPT 4.0，其回答经过微软的定制和限制，在某些方面可能不够精准。
    2.  **连续对话能力受限**：无法进行非常深度的连续提问，有时需要开启新对话。
    3.  **记忆功能有缺失**：对历史对话内容的记忆和分析能力不如官方版本完整。

**获取方式**：
确保你的Edge浏览器更新到最新版本。如果未找到Copilot，可以尝试下载Edge的Dev（开发者）测试版，该版本通常会包含最新功能。

## 第三方集成平台与共享账号
了解了系统集成的工具后，我们来看看由开发者搭建的第三方平台，它们提供了另一种使用ChatGPT的途径。

以下是几个可用的第三方渠道：

**1. 灵感（网站A）**
*   **特点**：集成了多种AI模型，包括GPT-3.5、GPT-4.0（含联网版）以及国内主流模型（如讯飞星火、文心一言等）。
*   **使用**：新注册用户有一定免费使用额度（例如7天内无限次使用，但GPT-4有次数限制）。
*   **优点**：模型丰富，GPT-4在一定额度内免费，无需特殊网络工具。
*   **缺点**：响应速度可能较慢。

**2. Wee（网站B）**
*   **特点**：提供GPT-3.5和GPT-4.0模型，响应速度较快。
*   **使用**：新用户赠送一定量的免费使用额度（Token），但通常仅适用于GPT-3.5。想试用GPT-4可能需要付费。
*   **优点**：速度快，体验接近官方。
*   **缺点**：免费额度有限，且需要特殊网络工具才能访问。

**3. ChatGPT共享账号平台**
*   **特点**：一个汇集了用户分享的ChatGPT账号的网站。账号状态用颜色标识：
    *   **绿色**：空闲。
    *   **橙色**：近期被使用过。
    *   **红色**：正在被使用或使用频繁。
    *   **金色边框**：通常是付费的Plus账号，可使用GPT-4。
*   **使用**：找到绿色或可用的账号，点击后设置密码即可登录使用。
*   **优点**：可以免费使用GPT-3.5，偶尔可能找到可用的GPT-4 Plus账号。
*   **缺点**：账号稳定性差，容易被封或达到使用上限；同样需要特殊网络工具。

> **注意**：使用第三方平台和共享账号存在隐私和安全风险，请谨慎评估，切勿输入敏感信息。

## 国内AI大模型替代方案
如果无法使用需要特殊网络工具的渠道，国内公司也推出了优秀的AI大模型，它们是可靠的替代选择。

*   **百度文心一言**：百度推出的AI对话模型，提供基础版和付费的“文心一言4.0”版本。在文案生成、创意辅助方面能满足基本运营需求。
*   **字节跳动豆包AI**：字节跳动推出的AI助手，同样可以用于生成脚本、提供创意灵感等。
*   **阿里通义千问**：阿里巴巴推出的AI模型。
*   **昆仑万维天工AI**：国内另一款表现不错的AI大模型。

**使用体验**：
这些国内模型在通用任务上表现良好，能够帮助运营人员完成诸如撰写短视频脚本、生成文案等工作。例如，向它们提问“请帮我写一个关于装修的短视频脚本文案，要求镜头丰富，语言吸引人”，都能得到结构完整、包含景别、画面、台词等要素的回复。

**与GPT-4的对比**：
虽然国内模型进步迅速，但在逻辑复杂性、创意深度和多轮对话的连贯性上，与顶尖的GPT-4模型相比仍存在差距。这受限于多种因素，并非单纯的技术问题。对于运营工作，选择的原则是：**哪个工具能更高效地提升你的工作效果，就使用哪个。**

## 官方ChatGPT 4.0体验
最后，我们直观感受一下官方ChatGPT 4.0的能力。使用同样的提示词“请帮我写一份关于装修的短视频脚本，文案要求镜头丰富，语言吸引人”，GPT-4生成的脚本在场景构建、叙事节奏和语言感染力上通常更为出色和细腻，例如它会构思出“变形计：从旧居到梦想家园”这样的主题，并详细描述镜头运动和情绪铺垫。

这展示了顶级AI模型在创意辅助方面的强大潜力。对于有条件的新媒体运营者，深度体验和使用GPT-4将是提升内容产出的利器。
