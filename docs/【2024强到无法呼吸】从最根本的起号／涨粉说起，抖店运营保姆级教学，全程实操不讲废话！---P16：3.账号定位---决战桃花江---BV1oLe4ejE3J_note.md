# 抖音运营教程：P16：3.账号定位 🎯

![](img/ec7258c48b6faa524e41a892fd839be1_0.png)

在本节课中，我们将要学习抖音账号定位的核心方法。账号定位是运营的基石，它决定了内容的方向、吸引的粉丝类型以及最终的商业价值。一个清晰的定位能让你的运营事半功倍。

![](img/ec7258c48b6faa524e41a892fd839be1_2.png)

## 理解账号定位的三个核心问题

要理解账号定位，必须反思三个根本问题：**你是谁**，**你是干什么的**，以及**为什么要关注你**。这三个问题构成了账号的灵魂。

## 确定账号类型与细分赛道

上一节我们介绍了定位的核心问题，本节中我们来看看如何具体确定账号类型。

首先，我们需要确定要做的账号类型。除了之前章节提到的大类赛道（即抖音定义的一级垂类），我们还需思考在一级垂类下更细分的赛道是什么。

**公式：账号垂直度 = 一级垂类 + 二级垂类**

这好比参加运动会，你报的项目“田径”是一级垂类，而你最擅长且有优势的“百米赛跑”就是二级垂类。通过定位账号的细分类目，可以确定账号的垂直类型。

在抖音中，无论任何行业，只要定位足够清晰，都有机会成功。因为抖音流量巨大，一旦做起来，增长和裂变的速度会非常快。

## 明确账号主体

确定了赛道方向后，接下来需要明确账号的“主人公”是谁。

账号主体是指账号的拥有人，也是最终呈现给粉丝的账号主人公。必须让粉丝清楚知道是在看谁的作品。在抖音商业类账号中，账号主体主要分为两类：

以下是两种主要的账号主体类型：

*   **商家账号**：如果账号主体是公司，即为商家账号。其内容需偏向商业化，并需不断通过内容拉近与粉丝的距离，避免过于严肃。这样既能建立信任感，也能带来好感，更符合抖音的娱乐属性。
*   **达人账号**：如果账号主体是个人，即为达人账号。首要任务是解决信任问题。可以通过IP打造的方式，为账号树立一个积极正面的人设。

## 明确账号内容与职业

明确了“谁在运营”之后，下一步要明确“这个账号是做什么的”。

最直接的方式是通过抖音认证身份，即认证职业。达人可以通过认证获得**黄V**，公司可以通过认证获得**蓝V**。认证后，粉丝可以清晰看到认证标签，从而明白账号主体的职业和账号的大致内容方向。

同时，商家认证蓝V或达人认证黄V，也能获取平台背书，增加信任感。

在未认证之前，也可以通过账号设置向粉丝明确传达业务信息。以下是可进行统一设置的要素：

*   头像
*   背景图
*   个人简介
*   视频封面

通过对自身商业模式和账号类型的确定，将这些元素进行与业务关联的统一设置。

## 思考为粉丝提供的价值

最后，我们需要花时间深入思考一个关键问题：**你将为粉丝提供什么样的价值？**

在抖音中，粉丝对博主的价值需求主要分为以下几类。以下是五种核心价值类型：

1.  **知识价值**
    *   **认知型**：如人生思考、思维方式、读书分享、行业内幕、成功经验、个人提升。
    *   **干货型**：如技能提升、经验总结、过程步骤、冷知识。
    *   **常识普及型**：如生活、健康、产品、科学、历史、地区常识的解读。
    *   **提供价值**：认同感（如三观、观点认同）、减少知识盲区的获得感，进而产生对作者的认同。

2.  **情绪价值**
    *   情绪是影响观众的重要因素。容易引发共鸣的情绪类型包括：开心愉悦（搞笑、夸张）、关系共鸣（情侣、婆媳、家庭等）、情绪波动（争议话题）、反转反差、怀旧、emo、震惊、好奇等。
    *   **提供价值**：共鸣。利用表情、文字、画面、声音、肢体动作准确抒发情绪，直接感染观众。

3.  **感觉价值**
    *   博主通过短视频的画面、文字和声音传递信息。提供的价值包括：视频的画面美感、构图，以及给观众带来的喜欢、治愈、视觉冲击感，或个人魅力带来的仰慕感；声音带来的节奏感、心跳加速或听觉享受。
    *   **典型账号**：颜值类、萌宠类、美景类、才艺类、明星、文化类等。
    *   **提供价值**：共情。因表情、文字、画面、声音、肢体动作的自然表现，让人产生联想，并在情绪、认知等意识层面与作者产生连接，从而间接引发共鸣。

4.  **体验价值**
    *   抖音让用户看到了不一样的生活方式，体验到了未曾经历的事物。这就是抖音提供的体验价值。
    *   **典型账号**：流量最多的美食账号（享用、制作体验）、各类旅行、运动、角色体验账号。
    *   **提供价值**：主要是满足好奇心，带来紧张感、激动、感动、获得感。

5.  **期待价值**
    *   当用户被你的短视频内容、观看体验或对作者本人产生好奇时，看完视频会意犹未尽，从而点击关注，期待下次更新。
    *   许多博主会利用这一点，不会在一个视频中呈现所有内容，以制造悬念，留住粉丝。

## 总结

本节课中，我们一起学习了抖音账号定位的系统方法。我们从反思“你是谁、做什么、为什么关注你”三个核心问题开始，逐步确定了账号的细分赛道、运营主体、内容方向，并深入分析了能为粉丝提供的五种核心价值：**知识、情绪、感觉、体验、期待**。一个清晰、独特的账号定位，是你在抖音海洋中吸引目标粉丝、实现持续增长的导航仪。
