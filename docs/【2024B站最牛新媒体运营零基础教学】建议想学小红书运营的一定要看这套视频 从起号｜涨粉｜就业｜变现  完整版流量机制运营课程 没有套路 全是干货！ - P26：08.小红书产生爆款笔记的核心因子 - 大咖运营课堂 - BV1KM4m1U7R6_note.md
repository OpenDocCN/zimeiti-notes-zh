## 1. 笔记收录 📥

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_44.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_46.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_48.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_50.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_52.png)

笔记收录是笔记获得流量的基础。它指的是笔记发布后，能否通过小红书平台的搜索功能被完整地找到。

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_54.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_56.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_58.png)

平台采用**千人千面**的分发算法。笔记发布后，系统会根据其原创度、内容质量等进行初步评估。表现良好的笔记会被推送到公域流量池，并可能被平台收录。收录意味着笔记可以被搜索到，并有机会参与关键词排名。反之，如果笔记在公域流量池中表现不佳（如点击率低、互动差），则可能不会被收录，后续流量也会受限。

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_60.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_62.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_64.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_66.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_68.png)

以下是影响笔记收录的几个关键点：

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_70.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_72.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_74.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_76.png)

*   **原创度**：平台会检测笔记内容与其他内容的**重合度**。笔记需要有自己的独特观点和价值。
*   **规避违规词汇**：小红书平台管理严格。笔记中若包含**玄学、伪科学、封建迷信**等违规词汇，可能导致笔记被限流或仅自己可见。
*   **初始数据反馈**：笔记发布后获得的初始互动数据至关重要，主要包括：
    *   **点击率 (CTR)**：`点击率 = 点击次数 / 曝光次数`。低点击率说明封面或标题吸引力不足。
    *   **停留时长**：用户点进笔记后的观看时间。停留时间过短（如仅3-5秒）通常意味着内容吸引力不够；**30秒以上**的停留时长是比较合格的标准。
    *   **互动率**：包括点赞、收藏、评论、关注、分享等行为。这些数据直接影响平台对笔记质量的判断。

## 2. 笔记权重分 ⚖️

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_86.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_88.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_90.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_92.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_94.png)

在了解了笔记收录的基础后，我们来看看更深层次的机制——笔记权重分。它决定了单篇笔记能获得多少初始流量。

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_96.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_98.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_100.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_102.png)

笔记权重分是一个综合评分，它受两方面影响：
1.  **账号权重**：运营时间越长、历史数据越好的账号，其新笔记的**初始权重分**通常越高。这就是为什么大V发笔记更容易获得高曝光。
2.  **单篇笔记的数据反馈**：如上文所述，笔记发布后的点击率、停留时长、互动率等数据会动态影响其权重分。

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_104.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_106.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_108.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_110.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_112.png)

**公式表示**：`笔记权重分 = f(账号基础权重， 笔记实时数据反馈)`
权重分高的笔记，会获得平台更多的流量推荐。

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_116.png)

## 3. 封面设计 🎨

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_122.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_124.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_126.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_128.png)

笔记的权重分决定了它能被多少人“看到”，而封面设计则决定了有多少人愿意“点进来”看。封面是决定笔记**点击率**的核心。

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_130.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_132.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_134.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_136.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_138.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_140.png)

一个优秀的封面需要做到：
*   **视觉冲击**：使用惊艳、令人向往的场景图。
*   **价值前置**：在封面上直接展示用户最关心的内容或结果，让用户一眼就能看到价值点。
*   **激发好奇**：设计能引发用户点击欲望的封面，让用户产生“我必须点进去看看里面到底讲了什么”的想法。

## 4. 需求匹配度 🔗

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_150.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_152.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_154.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_156.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_158.png)

用户点击进来后，能否留住他们并产生互动，就取决于内容是否与他们的需求高度匹配。

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_160.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_162.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_164.png)

![](img/f7c347f2061451e66b3ed1ff5b5da9ac_166.png)

需求匹配度主要体现在两个方面：
1.  **选题精准**：基于关键词规划，优先创作用户**当下最关心、最痛点的内容**。例如，在装修领域，“小户型装修避坑要点”比“装修风格大全”更能匹配精准用户的即时需求。
2.  **内容直给**：避免冗长铺垫。直接在文案、图片或视频开头，呈现用户寻找的**核心攻略、解决方案或关键信息点**。例如，直接列出“阳台改造3个技巧”、“电路布局5个注意事项”。这样能快速满足用户需求，提升停留时长和互动意愿。
