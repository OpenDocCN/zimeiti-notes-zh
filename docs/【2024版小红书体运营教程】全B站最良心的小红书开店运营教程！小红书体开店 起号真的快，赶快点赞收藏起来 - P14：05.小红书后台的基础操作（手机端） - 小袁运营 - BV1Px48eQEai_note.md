# 小红书运营教程：P14：05.小红书后台的基础操作（手机端） 📱

![](img/f971f7dd79107a2ca59d6b28b361de50_0.png)

在本节课中，我们将学习如何通过手机端操作小红书店铺后台（千帆系统），包括基础设置、地址管理、运费模板配置以及优惠券创建等核心功能。

---

上一节我们介绍了开店的基本流程，本节中我们来看看如何通过手机App管理店铺后台。我们将以小红书千帆系统为例，进行详细演示。

首先，你需要在手机应用商店搜索并下载“小红书千帆”App。这是官方店铺管理后台。

打开App后，你会看到初始页面。对于一个新店，页面内容通常较少。

**消息中心**
点击底部的“消息”图标。这里是客服接待和系统通知的中心。
以下是消息中心的主要功能：
*   **客服消息**：买家咨询会显示在这里，你可以直接回复。
*   **系统通知**：包括订单通知、违规提醒等重要信息都会在此处显示。
*   **客服配置**：可以在此进行客服相关的设置。

**首页与工具**
返回“首页”，你会看到“去初始化”和“全部工具”等选项。“全部工具”是后台功能的集合。
以下是“全部工具”内的核心功能列表：
*   **发布作品/商品**：创建新的商品或笔记。
*   **管理商品/订单**：查看和管理所有上架商品及用户订单。
*   **数据中心**：查看店铺运营数据。
*   **店铺装修**：设置店铺首页样式。
*   **营销工具**：创建优惠券等活动。

**地址与物流设置**
这是开店后的首要操作。点击“成长”或相关入口，找到“地址管理”。
你需要设置发货地址和退货地址。对于一件代发的模式，地址可以填写你常用的联系地址。
关键公式为：**发货地址 = 联系人 + 地区 + 详细地址 + 手机号**
**请注意**：退货地址务必填写真实有效的地址，不要选择“联系客服”。如果填写“联系客服”，平台可能不会通过审核，影响后续退货流程。

设置完地址后，需要配置“运费模板”。
以下是设置运费模板的步骤：
1.  进入“物流管理”或“运费模板”设置。
2.  点击“新建运费模板”。
3.  为模板命名（例如：“全国包邮（除偏远）”）。
4.  设置配送区域和运费。例如，可以选择为新疆、西藏等地区设置“不配送”或“买家承担运费”。
5.  保存模板，在发布商品时即可选用。

**营销功能：优惠券**
接下来，我们看看如何设置优惠券来吸引顾客。在“营销”或“优惠券”板块，点击“创建优惠券”。
你可以创建多种类型的优惠券：
*   **店铺通用券**：全店商品可用。
*   **指定商品券**：仅限特定商品使用。
*   **店铺新人券**：针对首次进店用户。

**支付与保证金**
为了店铺正常收款和长期运营，需要完成两项重要设置：
1.  **开通支付**：在设置中绑定微信支付和支付宝。买家下单后，款项才能正常结算到你账户。
2.  **缴纳保证金**：根据你经营的商品类目，系统会要求缴纳相应额度的保证金。这是平台对商家的一种履约保障。建议打算长期经营的商家及时缴纳。

本节课中我们一起学习了小红书手机端后台的基础操作。我们了解了如何查看消息、使用全部工具、设置发货与退货地址、配置运费模板、创建优惠券活动，以及开通支付和缴纳保证金。这些是店铺日常运营的基石，请多加练习和摸索，实际操作起来并不复杂。
