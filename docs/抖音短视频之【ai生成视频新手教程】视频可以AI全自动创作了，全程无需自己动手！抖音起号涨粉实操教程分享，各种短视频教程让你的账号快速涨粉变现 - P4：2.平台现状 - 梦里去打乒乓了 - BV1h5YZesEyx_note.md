# AI生成视频新手教程：P4：2.平台现状 📊

在本节课中，我们将要学习自媒体行业的平台现状，分析其政策支持、平台激励与市场潜力，帮助你判断进入这个行业的可行性。

上一节我们介绍了什么是自媒体，本节中我们来看看这个行业的平台现状如何，以及它是否值得去做。

## 政策支持 ✅

现阶段，整个社会氛围比较浮躁，许多人都想赚钱。但需要注意的是，过去有人采取非法手段牟利，例如前往缅甸等地的危险行为，这些已被国家明令禁止，对我们没有任何好处。

与之相反，自媒体行业目前属于国家支持的范畴。国家已将支持文件下达到各个地方政府。

*   **荆州市**：于今年6月2日由人社局发布了“六项措施”，文件中指出要“加大政策宣传力度，有力支持自媒体以及新媒体行业的蓬勃发展”。

## 平台激励 💰

除了有政策性文件支持我们制作视频，自媒体平台也响应国家号召，出资鼓励创作。

那么平台具体出多少钱呢？通过一个视频可以了解到关键信息。



![](img/dc1141ff2f8f86b9d8bd4906ffcb57b7_7.png)

![](img/dc1141ff2f8f86b9d8bd4906ffcb57b7_9.png)

视频中提到，西瓜视频决定在未来一年额外拿出**20亿元**，专门补贴中视频创作人。这20亿元是**真金白银的纯补贴**，不包含任何商单、直播和电商的收入。平台还会探索“保底+分成”的模式来助力创作人职业化。并且，这20亿元只是保底，实际补贴金额上不封顶。



![](img/dc1141ff2f8f86b9d4906ffcb57b7_9.png)

简单来说，我们未来制作视频，就能有机会从这20亿元乃至更多的补贴中获得收益。

## 市场潜力 📈

现阶段自媒体行业还有一个巨大优势：用户基数非常庞大。

根据2022年9月9日中国研究生网站发布的自媒体行业市场调查与发展前景分析报告指出：

*   **我国网民规模**：达到**9.4亿人**，接近10亿。这些就是看视频的潜在用户。
*   **自媒体从业者规模**：截至去年，我国全职从事自媒体的人数为370万，兼职为600万，合计约**970万人**。我们可以近似理解为有**1000万**人在制作视频。

由此可以计算出一个简单的市场供需关系：

`观看视频用户数 / 制作视频用户数 ≈ 9.4亿 / 0.1亿 ≈ 94 : 1`

这相当于大约每**100个**看视频的人，才对应**1个**制作视频的人。一个人制作视频的速度，远远无法满足上百人观看视频的需求。这种巨大的内容缺口，正是国家支持自媒体行业发展的重要原因。

从发展趋势也能看出，我们从小时候看的黑白电视，到后来的彩电，再到如今可以连接WIFI的智能电视，未来我们制作的视频将拥有更广阔的展示平台。你可以想象，其用户数量是多么庞大。

本节课中我们一起学习了自媒体行业的平台现状。我们了解到，该行业拥有**国家政策的大力支持**，平台方也提供了**真金白银的巨额补贴**激励创作，并且市场存在**巨大的用户基数和内容需求缺口**。这些现状共同表明，自媒体是一个值得进入且充满潜力的领域。
