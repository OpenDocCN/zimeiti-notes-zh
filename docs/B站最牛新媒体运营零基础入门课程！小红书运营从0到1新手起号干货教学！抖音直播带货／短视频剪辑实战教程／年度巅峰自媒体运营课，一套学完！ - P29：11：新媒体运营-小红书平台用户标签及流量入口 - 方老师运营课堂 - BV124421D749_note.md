在本节课中，我们将学习小红书平台如何为用户打上标签，以及平台的主要流量入口有哪些。理解这些机制，有助于我们分析笔记如何被推荐给用户，并反推出让内容触达精准用户的方法。

上一节我们介绍了系统如何判定内容标签，本节中我们来看看平台如何判定用户标签。

## 用户标签判定机制 🔍

平台主要通过分析用户的**浏览行为**来判定其标签。用户注册时选择的兴趣范围参考价值不大，仅作为初期内容方向的引导，无法进入系统深层的用户标签池。

以下是判定用户标签的核心行为指标：

1.  **点击率**
    *   **定义**：系统在发现页向用户推送多种内容（如美妆、家居、母婴），用户点击某一类内容的频率。
    *   **逻辑**：系统通过`用户点击某类内容的次数 / 系统推送的该类内容总数`来计算点击率。点击率高的类别会被判定为用户强兴趣领域。

2.  **停留时长**
    *   **定义**：用户点开一篇笔记后观看的持续时间。
    *   **逻辑**：即使同属美妆领域，若用户快速划走某些笔记，说明内容不吸引他。系统会继续推送该领域下其他细分内容（如上妆、护肤）来试探。反之，若用户对偶然推送的家居内容停留很久，系统可能判定其有装修或购买家居用品的潜在需求。

3.  **页面浏览量（PV）**
    *   **定义**：用户浏览页面的深度，例如不仅看了单篇笔记，还进入了博主主页并查看了多篇内容。
    *   **逻辑**：PV值高代表用户对你账号的内容兴趣浓厚。提升主页内容质量，可以增加用户的PV，这对账号健康度和评分有积极帮助。

4.  **互动行为**
    *   **定义**：包括点赞、收藏、评论、关注等用户主动操作。
    *   **逻辑**：这些数据直接提升笔记的互动率。高互动笔记更容易进入下一层级的推荐流量池，同时也能带动账号内其他笔记的曝光，并有助于粉丝增长。

通过上述行为分析，系统会为用户生成一个标签云图。例如，一个用户的“美妆”标签最大，说明他是该领域的重度兴趣用户，系统便会持续为其推荐美妆内容。

## 平台流量分发入口 🚪

了解用户标签后，我们来看看小红书主要的流量分发入口。平台主要通过以下四个页面分发内容：

以下是小红书四大核心流量入口：

*   **发现页**
    *   **内容构成**：
        1.  已关注达人发布的新内容。
        2.  根据用户标签（点击率、兴趣等）匹配的推荐内容。
        3.  部分平台为了内容多样化而推送的试探性内容。
    *   **核心价值**：这是笔记和账号获取流量的最主要来源。

*   **关注页**
    *   **内容构成**：纯粹展示用户所有已关注博主的笔记。
    *   **核心价值**：用于查看固定关注对象的内容，流量相对稳定但有限。

*   **附近页**
    *   **内容构成**：根据手机定位，推送同城或附近地区的相关内容。
    *   **核心价值**：强地域性需求入口，非常适合线下实体店（如美容院、餐厅、景点）的本地化推广。

![](img/408c3547aae3f8e2de7d13ffbb47af80_3.png)

*   **购物页**
    *   **内容构成**：小红书电商和直播带货的主要入口，匹配用户的购物需求。
    *   **核心价值**：平台商业变现的核心板块，但用户直接进入商城搜索的习惯仍在培养中，许多购物需求仍始于笔记搜索。
