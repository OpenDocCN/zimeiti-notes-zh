# 抖音运营教程：P34：基础篇丨爆款文案套路 📝

![](img/2acdea719dffd7e0e395fe4f1f7c0151_0.png)

在本节课中，我们将学习一个适用于各行各业的爆款文案创作套路。这个套路的核心在于快速吸引目标用户，并引导他们完成观看、互动等一系列行为，从而提升视频的完播率和互动率。

在内容为王的时代，运营者必须具备脚本创作能力。而创作好文案的关键，是让观众认为内容与自己有关，即引起共鸣。如果内容与观众无关，他们就不会停留观看。

那么，如何创作出能引起共鸣的文案呢？接下来，我们将介绍一个万能公式。

## 爆款文案万能公式

![](img/2acdea719dffd7e0e395fe4f1f7c0151_2.png)

这个万能公式的结构是：**现象 + 危害 + 原因和解决办法**。它的每一步都承担着特定的作用，共同构成一个吸引人的完整叙事。

以下是该公式每个部分的具体解释：

*   **现象**：描述一个普遍存在的问题或场景。作用是筛选目标用户，让他们感觉“这说的就是我”，从而停留观看。
*   **危害**：阐述该现象带来的负面影响或糟糕后果。作用是放大用户的焦虑或痛点情绪，防止他们在5秒内划走视频。
*   **原因**：简要分析问题产生的原因。作用是引导观众继续观看，寻求答案，从而提高视频的完播率。
*   **解决办法**：给出具体的解决方案或建议。作用是提供价值，鼓励观众点赞、收藏，提升视频的互动率。

## 公式应用案例

为了帮助你更好地理解，我们来看两个不同行业的应用案例。

### 案例一：服装行业

*   **现象**：梨形身材的女孩太难买裤子了。
*   **危害**：挑错了样式显得腿又粗又短。
*   **原因**：但梨形身材是天生的，很难改变。
*   **解决办法**：而这三种裤子不仅显瘦，还显腿长，穿上第二天就被小哥哥搭讪了，姐妹们赶快收藏起来吧。

这段文案开头就锁定了“梨形身材”的目标客户，并通过描述危害引起共鸣，最后给出解决方案，鼓励互动，完美契合了公式。

### 案例二：运营行业

*   **现象**：每天用心制作短视频，但发完只有100播放。
*   **危害**：继续下去只是浪费时间，徒增焦虑。
*   **原因**：这跟你没有系统的运营底层逻辑有关。
*   **解决办法**：现在去看完我的前三条视频，不花一分钱也能学会运营知识，还不快快行动起来。

![](img/2acdea719dffd7e0e395fe4f1f7c0151_4.png)

这个案例同样用现象吸引目标用户（播放量低的创作者），用危害加深焦虑，最后引导用户观看指定视频作为解决方案。

用好这个套路，无论什么行业，都可以快速套用生成吸引人的文案。这种不足30秒的短文脚本，制作起来轻松，尤其适合进行矩阵化运营，能极大提升内容产出效率。

![](img/2acdea719dffd7e0e395fe4f1f7c0151_6.png)

本节课我们一起学习了爆款文案的万能公式：**现象 + 危害 + 原因和解决办法**。我们通过两个案例了解了如何在不同行业中应用此公式。你已经掌握了“举一反二”的方法，剩下的“反三”就交给你了。你可以根据自己行业的特性，运用这个公式创作你的文案，并将思考过程付诸实践。
