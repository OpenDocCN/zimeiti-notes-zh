# 剪映手机版教学：P6：剪辑核心功能——滤镜调色 🎨

在本节课中，我们将要学习剪映的核心剪辑功能之一：滤镜与调色。许多初学者拍摄的视频色彩不够鲜亮，画面显得单调。通过学习滤镜和调色的应用，我们可以显著提升视频的视觉质感，使其色彩丰富、画面细腻。

## 为什么需要调色？ 🎬

如何快速提升视频的质感？答案就是调色。在传统的专业剪辑软件（如PR、达芬奇）中，调色功能非常强大但操作复杂。对于初学者而言，剪映提供了简单易用的滤镜和调色工具，能够帮助我们轻松实现类似的效果。

上一节我们介绍了视频剪辑的基础操作，本节中我们来看看如何为视频“上色”。

## 滤镜与调色的区别

在剪映中，调色主要通过两个部分实现：**滤镜**和**调节**。

*   **滤镜**：可以理解为已经预设好的色彩模板。我们只需选择并套用，就能快速改变视频的整体风格。
*   **调节**：允许我们对视频的色彩参数进行个性化、精细化的手动调整，例如单独让天空更蓝或草地更绿。

接下来，我们将通过实战演示来具体学习如何使用它们。

## 实战：为视频应用滤镜与调色

我们将导入一段素材进行演示。原素材中的树木不够绿，天空不够蓝，我们的目标是让色彩更加鲜亮。

以下是操作步骤：

1.  **导入素材**：点击“开始创作”，导入需要调色的视频片段。
2.  **应用滤镜**：
    *   点击底部工具栏的 **“滤镜”** 按钮。
    *   在滤镜库中选择一个喜欢的风格，例如 **“鲜亮”**。
    *   应用后，可以立即看到视频色彩发生显著变化。通过对比原视频，可以直观感受滤镜的效果。
3.  **进行个性化调节**：
    *   如果对滤镜效果仍有不满意的地方，可以点击 **“新增调节”**。
    *   这里提供了多个参数供我们精细调整：
        *   **亮度**：控制画面明暗。
        *   **对比度**：增强或减弱明暗对比。
        *   **饱和度**：**增加此值可以让色彩更鲜艳**。
        *   **锐化**：让画面细节更清晰。
        *   **高光/阴影**：分别调整亮部和暗部的细节。
        *   **色温**：调整画面冷暖色调。
        *   **色调**：调整画面整体色彩倾向。
    *   根据需求调整参数。例如，可以**稍微降低亮度，提升对比度和饱和度**，并微调色温让天空更蓝。
4.  **对比效果**：调整完成后，通过播放对比原视频与调色后的视频，观察天空是否更蓝、树木是否更绿。

需要注意的是，调整人物、美食等不同主体时，侧重的颜色和参数会有所不同，但基本方法是一致的：先尝试滤镜快速定调，再通过调节功能进行微调。
