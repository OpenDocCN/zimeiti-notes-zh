## 笔记被推荐的核心逻辑

笔记发布后，平台会给予一个基础的展示量。这个展示量会根据账号权重有所不同。笔记能否获得更多流量，取决于它能否通过平台的“数据考核”，实现层级的跨越。

### 第一关键：打开率

打开率是笔记获得更多推荐的第一道门槛。当用户打开小红书App，首页“发现”页会展示4到6篇笔记。平台会一次性推送约20篇笔记，用户浏览完这20篇后，页面会刷新。

**核心公式**：`打开率 = 笔记点击量 / 笔记展示量`

在这批同类型笔记的竞争中，你的笔记**平均打开率必须超过11%**，才能进入下一流量池。

*   **举例**：如果平台给你的笔记基础展示量是1000次，那么你需要获得至少110次点击（小眼睛），平台才会认为你的笔记有吸引力，并给予你2000到5000次的下一级展示量。
*   **结果**：如果打开率达不到11%，笔记通常只会获得一波基础推荐，之后流量便会停滞。

### 第二关键：互动率

当笔记通过打开率考核，进入更大流量池后，**互动率**就成为能否继续升级的关键。

![](img/c94942707fa1670942d123f272bfab55_9.png)

**核心公式**：`互动率 = （点赞+收藏+评论+分享等互动数） / 笔记点击量`

![](img/c94942707fa1670942d123f272bfab55_11.png)

![](img/c94942707fa1670942d123f272bfab55_13.png)

笔记的**平均互动率需要达到3%以上**，才有机会冲击更高的流量层级。

![](img/c94942707fa1670942d123f272bfab55_15.png)

*   **举例**：假设你的笔记获得了110次点击，那么至少需要有约4次互动（评论、点赞等），互动率才算达标。
*   **升级路径**：笔记通常遵循“**先点击，后互动**”的升级路径。新号一般有机会冲击到第三级流量池。老号因基础权重高，起步层级更高，有机会冲击第四、第五级。更高级别的流量（第六级及以上）通常需要结合内容质量与商业投放。

![](img/c94942707fa1670942d123f272bfab55_17.png)

### 整体评分体系

![](img/c94942707fa1670942d123f272bfab55_19.png)

除了打开率和互动率这两个基础属性，平台还有一个综合评分体系，用于评估笔记的全面表现。

以下是影响笔记权重的核心互动行为及其评分（示意）：
```python
# 笔记互动权重评分（示意）
engagement_score = (
    like_count * 1 +          # 点赞权重
    comment_count * 2 +       # 评论权重较高
    collect_count * 3 +       # 收藏权重高
    share_count * 4 +         # 分享权重最高
    follow_from_note * 5      # 通过笔记关注账号权重极高
)
```
**重要提示**：这个综合评分建立在打开率和互动率达标的基础上。如果基础数据不达标，后续的点赞、收藏等权重加成效果也会非常有限。

## 必须避免的运营“坑”

了解推荐逻辑后，我们还需要避开平台规则中的常见陷阱，以防账号被降权或限流。

以下是新手运营小红书时必须注意的禁忌事项列表：

![](img/c94942707fa1670942d123f272bfab55_21.png)

![](img/c94942707fa1670942d123f272bfab55_23.png)

*   **新号切忌立刻发笔记**：新注册的账号需要先“养号”，通过浏览、互动等行为模拟真实用户，为账号打上标签。当天注册当天发笔记，会浪费平台的初始权重推荐，导致后续流量低迷。
*   **违规必然导致降权**：任何违规行为（如引流、抄袭、发布违禁内容）都会导致账号被降权，笔记推荐和搜索排名会大幅下降，恢复难度极大。
*   **标签切忌反复修改**：账号、店铺或笔记的人群标签一旦确定，不要频繁修改。尤其是在账号已有一定流量后修改核心标签，极易引发限流。
*   **禁止一机多号登录**：强烈建议遵循“一机一卡一号”的原则。一台设备登录多个账号，或一个账号在多个设备频繁切换，会被系统判定为异常营销行为。
*   **谨慎处理引流信息**：在笔记或私信中直接发布微信号、二维码、诱导性外链等内容是高风险行为。需要采用平台允许的合规方式进行引流。
*   **谨慎处理用户互动**：对于用户的留言和私信，应积极、友善地回复。避免出现争吵、删除负面评论等不当处理方式。
