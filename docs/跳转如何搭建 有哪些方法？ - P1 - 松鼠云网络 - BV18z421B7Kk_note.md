# 抖音引流至微信：P1：跳转搭建方法与形式详解 🚀

在本节课中，我们将学习如何从抖音平台引流至微信，并详细介绍目前主流的七种跳转搭建方法。无论你是商家还是运营者，掌握这些方法对于构建营销闭环至关重要。

首先需要明确一个前提：无论采用哪种跳转形式，都需要**认证抖音企业号（蓝V）**，或者在**半蓝V（企业号试用期）**的状态下才可进行操作。

---

## 七种抖音至微信跳转方法 📋

![](img/3b717f02b883746ec85eedc825457311_1.png)

上一节我们明确了操作前提，本节中我们来看看具体的七种跳转方法。以下是每种方法的详细说明。

![](img/3b717f02b883746ec85eedc825457311_2.png)

### 1. 私信跳转
用户进入账号私信界面后，通过**智能回复**、**自动回复卡片**或**链接**的形式完成向微信的跳转。

![](img/3b717f02b883746ec85eedc825457311_3.png)

![](img/3b717f02b883746ec85eedc825457311_4.png)

### 2. 预约跳转
在视频左下方，通过配置企业号“私域经营工具”中的**在线咨询组件**。用户点击后会进入私信界面，随后通过手动或自动发送卡片/链接的形式完成跳转。

![](img/3b717f02b883746ec85eedc825457311_5.png)

### 3. 直播跳转
通过抖音直播间进行跳转，主要分为两种形式：
*   **小风车跳转**
*   **星图跳转**

![](img/3b717f02b883746ec85eedc825457311_6.png)

### 4. 群聊跳转
通过抖音群聊进行跳转。可以在群聊的**欢迎语**中设置卡片、链接或超链接，引导用户完成跳转。

![](img/3b717f02b883746ec85eedc825457311_7.png)

![](img/3b717f02b883746ec85eedc825457311_8.png)

### 5. 星图跳转
通过**巨量星图**平台发布任务，抖音号以接单的形式挂载小风车，从而实现跳转。

![](img/3b717f02b883746ec85eedc825457311_9.png)

### 6. 短视频直跳（高级预约跳转）
这种跳转与第二种“预约跳转”不同。用户点击视频左下角的“预约”按钮后，会**直接跳转至微信**等第三方平台，**不经过私信通道**。此外，该功能还可以授权给员工号进行挂载使用。
其核心流程可简化为：`用户点击视频预约按钮 -> 直接打开微信/外部链接`

![](img/3b717f02b883746ec85eedc825457311_10.png)

### 7. 投放信息流跳转
本质上是通过**抖音广告**的形式完成跳转。在信息流广告中设置跳转组件，引导用户至微信。

## 如何选择适合的跳转形式？ 🤔

![](img/3b717f02b883746ec85eedc825457311_12.png)

以上便是七种不同形式的抖音至微信跳转方法，在实际应用中，这些方法还可以进行组合使用。

![](img/3b717f02b883746ec85eedc825457311_13.png)

![](img/3b717f02b883746ec85eedc825457311_14.png)

那么，哪种跳转形式最适合您的业务引流呢？这需要根据您的**营销目标**、**预算**和**用户互动场景**（如日常内容、直播、广告投放）来综合决定。例如，追求直接转化可考虑“短视频直跳”或“信息流跳转”；注重用户沉淀和长期运营，则“私信跳转”和“群聊跳转”可能更合适。

![](img/3b717f02b883746ec85eedc825457311_15.png)
