# 视频号运营教程：P28：如何开启一场直播 🎬

在本节课中，我们将学习如何从零开始，完整地准备并开启一场视频号直播。整个过程包括开播前的准备、直播中的设置与操作，以及直播结束后的处理。

---

## 开播前的准备

上一节我们介绍了视频内容与选品，本节中我们来看看如何为直播做具体准备。

在确定好商品并发布预热视频后，需要等待视频获得约1000左右的播放量。此时，应提前准备好直播话术。

![](img/7b1a234b1910a18aea28c3bac83e6d91_1.png)

如果担心直播开始时没有观众进入，可以提前将用于补流量的视频准备好，并保存到草稿箱中，以便随时发布。

![](img/7b1a234b1910a18aea28c3bac83e6d91_2.png)

当你没有时间或不方便立即直播，但预热视频流量很好时，务必先创建一个直播预告。例如，预告你将在某个具体时间开播。这样，想观看直播的用户可以提前预约。当你开播时，预约用户会第一时间收到开播提醒。

![](img/7b1a234b1910a18aea28c3bac83e6d91_3.png)

## 发起直播与基础设置

以下是发起直播并进行基础设置的步骤。

![](img/7b1a234b1910a18aea28c3bac83e6d91_6.png)

1.  点击“发起直播”按钮，然后选择“直播”功能。

    ![](img/7b1a234b1910a18aea28c3bac83e6d91_1.png)
    ![](img/7b1a234b1910a18aea28c3bac83e6d91_2.png)
    ![](img/7b1a234b1910a18aea28c3bac83e6d91_3.png)
    ![](img/7b1a234b1910a18aea28c3bac83e6d91_4.png)

2.  设置直播主题。主题一般可写为“视频同款热卖中”。你也可以参考对标账号的写法进行设置。

3.  提前添加商品。在“商品”区域，添加你计划售卖的商品，支持添加单品。

    ![](img/7b1a234b1910a18aea28c3bac83e6d91_6.png)

4.  设置直播画面。在“画面”设置中，可以使用美颜、滤镜、人物特效等功能。根据个人需求进行调整。

![](img/7b1a234b1910a18aea28c3bac83e6d91_8.png)

5.  添加直播贴图。可以参考上节课的内容，使用“微微商水印相机”等工具制作简单的介绍贴图。对标账号如何做，你就可以如何做。

    ![](img/7b1a234b1910a18aea28c3bac83e6d91_8.png)

![](img/7b1a234b1910a18aea28c3bac83e6d91_10.png)

6.  对于纯卖货的直播，无需调整“更多”设置中的复杂选项。

    ![](img/7b1a234b1910a18aea28c3bac83e6d91_10.png)
    ![](img/7b1a234b1910a18aea28c3bac83e6d91_12.png)

![](img/7b1a234b1910a18aea28c3bac83e6d91_12.png)

7.  设置直播可见范围。为避免无关用户进入影响直播，可将直播模式设置为“部分可见”。

    ![](img/7b1a234b1910a18aea28c3bac83e6d91_14.png)
    ![](img/7b1a234b1910a18aea28c3bac83e6d91_16.png)

8.  标记地理位置。卖货直播需要标记位置，可以随便标记一个地点，然后点击“开始直播”即可。

    ![](img/7b1a234b1910a18aea28c3bac83e6d91_18.png)

![](img/7b1a234b1910a18aea28c3bac83e6d91_14.png)

## 直播中的关键操作

开始直播后，有以下几项关键操作需要注意。

![](img/7b1a234b1910a18aea28c3bac83e6d91_18.png)

1.  讲解商品时，务必点击打开购物车。这样观众才能看到商品弹窗，有助于提升转化率。

    ![](img/7b1a234b1910a18aea28c3bac83e6d91_20.png)

![](img/7b1a234b1910a18aea28c3bac83e6d91_20.png)

2.  直播开始后，可以继续添加或调整贴图。

3.  可以在直播时添加背景音乐，这不会影响直播的正常进行。

4.  管理评论。如果某条观众评论对你特别有利，可以长按该评论，然后选择“点上墙”。这样新进入直播间的用户就能看到这条评论。

    ![](img/7b1a234b1910a18aea28c3bac83e6d91_22.png)

5.  注意直播话术，尽量避免使用违规词汇。

---

![](img/7b1a234b1910a18aea28c3bac83e6d91_22.png)

## 结束直播与后续操作

直播结束后，需要进行收尾工作。

1.  点击直播间右上角的“叉号”即可结束直播。

2.  在直播过程中，你可以将直播间分享给其他人。

3.  结束直播后，可以选择“生成直播回放”来保存本次直播。

    ![](img/7b1a234b1910a18aea28c3bac83e6d91_24.png)

![](img/7b1a234b1910a18aea28c3bac83e6d91_24.png)

本节课中我们一起学习了开启视频号直播的全流程：从开播前的预告与话术准备，到直播中的商品讲解、互动功能使用，再到最后的结束与回放生成。掌握这些步骤，你就能顺利地开启并完成一场直播。
