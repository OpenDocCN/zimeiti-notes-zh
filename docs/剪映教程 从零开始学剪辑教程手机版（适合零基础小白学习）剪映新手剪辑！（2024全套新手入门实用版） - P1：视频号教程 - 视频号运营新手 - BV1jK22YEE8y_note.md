## 开通分成计划的核心条件

注册完视频号后，如何快速开通分成计划并领取收益？账号的启动阶段非常关键。如果启动不到位，将无法快速开通分成计划。

目前，开通视频号分成计划需要满足以下两个条件：
1.  **粉丝数达到100以上**。
2.  **内容符合平台规范**，即优质原创、画风精美。

可能在前段时间，新注册的视频号拍摄十条风景视频就能通过原创审核。但现在，仅拍摄风景类视频很难通过。

这里我给大家一个方法：申请注册完新的视频号后，直接进行“怼脸拍”。每条视频时长需达到9秒以上，每天发布5条。**实测第三天就能开通分成计划**。

开通后的前几天收益可能较低，但坚持发布，收益会逐步提升。

## 分成计划开通流程

![](img/ef07ae2c24f80e867331fc0f4d0d75a5_6.png)

上一节我们了解了开通条件，本节中我们来看看具体的操作流程。

开通分成计划的第一步，是找到视频号的分成计划入口。

**具体操作步骤如下：**
1.  每天发布5条“怼脸拍”视频，持续两天，共计发布10条。
2.  第三天，在视频号内找到“分成计划”入口，直接申请加入。
3.  申请通过后，切记：**发布原创视频时，一定要勾选“原创声明”**。

如果你不点开原创声明，即使视频获得几千万的播放量，也不会产生任何收益。

## 收益计算与提升策略

目前，腾讯官方没有给出非常明确的收益计算规则。根据我的运营经验，大致范围是：
```
收益估算 ≈ 每万次播放产生1到3元人民币
```
具体收益仍取决于广告点击情况。有时即使播放数据很好，但如果没有人点击视频评论区或贴片的广告，视频收益也不会高。

广告分成收入与作者视频质量、视频播放量、投稿数量、广告行业选择等多方面因素有关。

因此，我们在制作原创视频时，要想尽一切办法**让视频的评论区活跃起来**。

## 推荐赛道：人性语录类

在众多内容形式中，我推荐两个目前操作者相对较少但流量巨大的赛道。第一个是**人性语录类**。

对于现在想入局视频号的新手，我不推荐一开始就做“祝福类”内容，因为做的人太多，竞争过于激烈。

人性领域在视频号中，是除了祝福类之外最火的领域之一。它与文案号不同，人性语录类视频通常通过一句话或一段话，直接揭示人性的本质。这个领域的粉丝质量通常较高，年龄段多在30岁以上。

我运营的账号收益，主要也来自人性领域，收益相当可观。

关于人性语录类视频的完整制作流程，包括：
*   获取文案
*   获取素材
*   文案配音
*   视频制作

我已将每一步都整理成详细的文档。因为逐步讲解过程较为枯燥，你们只需按照我整理的步骤一比一操作，就能制作出像同行那样流量很高的视频。

**重要提醒：** 开通原创后，不要因为发了几天视频收益不高就停止更新。一定要坚持下来。以我的账号为例，收益从最初的几元、几十元，通过坚持才逐步提升。

开通原创后，每个账号每天最好发布3到5个视频，切记不要超过5个。网上有人建议一天发8、9个甚至10个，但根据当前平台规则，单日发布数量过多可能导致账号权重下降。每天3到5个，坚持至少7天，才能见到明显效果。
