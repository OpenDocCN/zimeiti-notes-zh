# 小红书运营教程：P25：小红书暴力玩法揭秘！吸粉推广篇 🔍

在本节课中，我们将要学习小红书推广的两种核心路径：自建账号矩阵推广与官方付费广告投放。我们将理清两者的关系，并详细介绍各自的运作方式与适用场景。

## 账号类型解析

上一节我们介绍了小红书的整体运营逻辑，本节中我们来看看平台上的账号类型。了解账号分类是制定推广策略的基础。

小红书账号主要分为以下三类：

1.  **企业号**：主要作用是树立官方品牌形象。它可以拥有官方认证标志，并享有展示品牌名称、地址等特权，便于开展业务。
2.  **个人号**：这是目前大多数用户运营的账号类型。其特点是**灵活性高**，可以是品牌员工或消费者身份，更容易塑造个人品牌特色，也适合品牌相关人物出镜。
3.  **引流号**：俗称“炮灰号”。这类账号可以批量创建，主要用于特定引流目的，用后即可舍弃，或在紧急情况下作为备用小号。通常**没有精细的个性规划**。

## 企业号矩阵运营建议

了解了账号分类后，我们聚焦于如何有效利用它们。目前常见的推广方式是企业号、个人号或引流号自行发布笔记进行引流。以下是打造企业号矩阵的一些具体建议：

1.  **积极参与品牌活动**：例如，婚纱、服装、美妆等品类的企业号发起试用或团购活动时，可以动用多个小号去参与，增加活动热度。
2.  **个人号展示品牌属性**：在个人号的笔记内容中，应多次暗示品牌的定位。例如，品牌目标用户是**都市年轻人**还是**职场新人**，需要通过内容持续强化。
3.  **控制营销内容比例**：个人号发布品牌产品时，需注意内容比例。建议遵循 **“30%营销内容，70%种草或科普内容”** 的原则。这个比例能较好地平衡推广与用户体验。
4.  **利用引流号进行评论引流**：当企业号需要直接引流时，最佳做法是使用引流号在其笔记评论区进行互动和引导，而非直接用企业号操作。

## 官方付费广告（CPC）介绍

![](img/65ba5b92af20af77a3d1f17fc2679b7f_3.png)

![](img/65ba5b92af20af77a3d1f17fc2679b7f_5.png)

除了自建账号矩阵，另一种高效的推广方式是借助平台力量。接下来，我们看看小红书官方的付费广告系统。

![](img/65ba5b92af20af77a3d1f17fc2679b7f_7.png)

![](img/65ba5b92af20af77a3d1f17fc2679b7f_9.png)

小红书官方广告即 **CPC（Cost Per Click）竞价广告**。要使用此功能，必须先认证企业号。

![](img/65ba5b92af20af77a3d1f17fc2679b7f_11.png)

**企业号认证流程**：
1.  访问小红书官网。
2.  找到“业务合作”或“推广合作”入口。
3.  按照指引提交相关企业资质并支付审核费用。
4.  审核通过后，普通账号即升级为官方认证的企业号。

拥有企业号后，即可投放信息流广告。官方广告主要分为两大类：

1.  **信息流广告**：用户浏览小红书首页信息时，系统会不定期间隔插入的广告。
2.  **搜索广告**：当用户主动搜索关键词时出现的广告。其原理类似于**百度搜索竞价排名**，通过对关键词出价来竞争不同的广告位。例如，用户搜索“考研资料”，系统可能根据算法推荐相关的能量补充商品。

当前信息流广告位常见于第3位、第13位和第23位等位置。关于广告投放的更多常见问题，可在小红书广告平台的官方问答区查看。

![](img/65ba5b92af20af77a3d1f17fc2679b7f_13.png)

**请注意**：官方广告投放费用较高。在资金允许的情况下，建议考虑与官方合作投放竞价广告，因为它能直接获得由平台算法推荐的精准流量。
