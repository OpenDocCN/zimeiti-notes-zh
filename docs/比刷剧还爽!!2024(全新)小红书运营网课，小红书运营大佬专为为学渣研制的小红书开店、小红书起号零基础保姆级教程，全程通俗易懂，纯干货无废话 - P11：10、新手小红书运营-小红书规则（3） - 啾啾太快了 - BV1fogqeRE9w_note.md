上一节我们介绍了小红书的基本规则框架，本节中我们来看看平台具体的“推流规则”。所谓推流规则，是指小红书根据店铺或账号的权重，决定如何分配和展示流量的机制。

![](img/75932d5157b10a91a082096ce94f4aa6_1.png)

推流主要发生在“发现页”。平台的推流并非随机，而是基于一套精密的标签系统进行筛选。这套系统主要依赖三个层面的标签：账号标签、笔记标签、以及话题/关键词标签。

![](img/75932d5157b10a91a082096ce94f4aa6_3.png)

## 三重标签系统解析

以下是平台用于判定和推流的三重标签：

1.  **账号标签**
    账号标签在你起号初期就已开始形成。它基于：
    *   账号资料设置（如昵称、简介、头像）。
    *   你选择的兴趣爱好选项。
    *   历史浏览、互动行为（即“养号”阶段观看同领域内容的目的）。

2.  **笔记标签**
    笔记标签指单篇笔记内容所对应的人群属性。平台通过分析笔记内容（文字、图片、视频）来判定其类型。
    *   **产品笔记**：直接带有商品链接的笔记，主要匹配搜索流量。即用户通过关键词搜索后才能看到。
    *   **引流笔记**：不带商品，纯内容分享，旨在引导用户兴趣后主动搜索产品。两种笔记的初始推流逻辑略有不同。
    笔记标签的核心是 **人群匹配**。例如，你发布一篇关于“抗初老精华”的笔记，平台会判断其适合28-35岁的用户。如果点击观看的用户中，这个年龄段的比例最高（例如超过40%），平台后续就会将笔记更多地推送给这个年龄段的兴趣用户。

3.  **话题与关键词标签**
    这是最直接的匹配层。发布笔记时设置的标题、话题以及文中重复出现的关键词至关重要。
    *   **公式**：`笔记曝光 ≈ f(账号标签， 笔记标签， 关键词匹配度)`
    例如，你的标题中含有“儿童早教”，那么近期搜索或浏览过“儿童早教”相关内容的用户，就更容易在发现页看到你的笔记。

**平台推流的完整逻辑链是**：首先根据你的账号标签进行初筛，再结合笔记内容标签进行二次匹配，最后通过关键词进行精准触达。这套三重匹配机制旨在推送最精准的流量。

## 推流的层级与放大效应

理解了标签系统，我们再来看看流量是如何从少到多扩散的。

平台会先将你的笔记推送给**同领域的一小部分用户**（例如2000-3000人）。这部分用户是根据上述三重标签筛选出的“可能感兴趣人群”。

这部分初始流量的数据反馈（点击率、点赞、收藏、评论、转发）至关重要。数据越好，平台算法就越认为你的笔记优质且受目标人群欢迎。

**代码逻辑模拟**：
```python
if 初始推送数据反馈(点赞率, 收藏率等) > 行业平均阈值:
    扩大推送范围至下一流量池（例如1万-2万人）
else:
    停止推荐，笔记流量停滞
```
因此，流量的增长是有层级的。只有在上一个层级获得良好的互动数据，才能进入下一个更大的流量池，获得更多曝光。

---

## 标签的重要性与修正成本

精准的标签是高效获取流量的前提。如果标签混乱，会导致推送人群不精准。

*   **错误示例**：你是美妆博主，但账号因历史行为被打上了“汽车”标签。平台将你的美妆笔记推给汽车爱好者，点击率必然很低。
*   **后果**：平台认为笔记不受欢迎，停止推荐，即使内容优质也无法获得曝光。

**更严重的是，修正错误标签的成本很高**：
*   **小标签修正**（微调人群属性）：约需7天。
*   **大标签修正**（完全改变领域，如从“服装”改为“美妆”）：约需15天。
    修正方法主要是通过主动、集中地搜索、浏览、互动目标领域的内容，来“告诉”平台你的新兴趣。

这正说明了**前期养号和精准定位的极端重要性**。前期花3天做好标签，远比后期花15天修正标签要高效得多。

## 核心行动指南

根据以上规则，我们可以总结出运营前期的核心动作：

1.  **明确账号定位**：在起号前就想清楚，你要服务哪一类具体人群。
2.  **执行养号流程**：主动、连续地浏览、互动目标领域的内容，打好“账号标签”基础。
3.  **内容紧扣标签**：发布笔记时，确保内容、标题、话题与你定位的领域高度一致，强化“笔记标签”和“关键词标签”。
4.  **优化初始数据**：笔记发布后，可通过一些方式（如分享到社群）引导初始互动，提升第一波流量池的数据反馈，撬动更大推荐。
