在本节课中，我们将要学习如何为新媒体账号进行人设定位。许多初学者在运营账号时，常常会为人设定位感到困惑和头疼，不知道如何塑造一个成功且可信的形象。本节课将为你拆解人设定位的核心因素，帮助你理清思路，为账号的长期发展打下坚实基础。

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_16.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_20.png)

人设的打造并非一蹴而就，它是在账号的运营过程中，随着内容方向的明确和账号的成长而逐步凸显的。我们前期所做的人设定位工作，主要是为了固化后期账号内容中需要融入的特征点。如果你在运营个人或企业账号时，特别纠结于人设问题，那么回顾本节课的内容将帮助你理解人设的本质及其打造方法。

## 人设定位的四大核心因素

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_32.png)

### 1. 角色：明确你的专业身份

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_46.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_48.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_50.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_52.png)

角色是指你在特定内容赛道中所扮演的专业身份。你需要让用户相信，你提供的内容具有专业性。

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_54.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_56.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_58.png)

以下是不同赛道的角色定位示例：

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_60.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_62.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_64.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_66.png)

*   **装修博主**：你需要展示自己确实拥有正在装修或已装修好的房子，通过记录过程、分享避坑经验、展示前后对比来建立专业形象。
*   **母婴赛道**：最常见的角色是“宝妈”。但根据内容方向，你也可以定位为“护理专家”，这不一定要求你自身是宝妈，可以是相关领域的专业人员。
*   **情感赛道**：你可以是“心理学顾问”或“情感大师”，分享情感、成长、励志类内容。另一种是“成长型人设”，例如分享一位30岁女性如何应对家庭、职场、创业挑战的个人成长故事。

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_68.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_70.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_74.png)

**核心公式**：`人设角色 = 内容赛道方向 + 专家身份`

### 2. 内容：保持垂直与价值

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_84.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_86.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_88.png)

内容定位需要确保内容吸引用户、提供价值并具备平台特色。在人设定位中，内容必须与你选择的赛道高度相关。

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_90.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_92.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_94.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_96.png)

这一点至关重要，因为它关系到账号的垂直度和平台标签的建立。垂直领域和清晰的账号标签直接影响账号的成长和后续的流量分发。

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_98.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_100.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_102.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_104.png)

**逻辑示例**：
如果你的第一篇内容关于儿童教育，第二篇却分享宠物日常，那么之前因教育内容关注你的用户，很可能对宠物内容不感兴趣。这会导致用户划走视频，数据表现不佳，从而向平台反馈“内容不受欢迎”，最终影响推荐流量。

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_106.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_108.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_110.png)

**核心原则**：你选择什么赛道，就持续提供该赛道相关的内容。

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_112.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_114.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_116.png)

### 3. 形式：确定内容的呈现方式

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_120.png)

形式指的是内容产出的具体方式，主要包括以下几个方面：

以下是关于内容形式的几个关键考虑点：

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_136.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_138.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_140.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_142.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_144.png)

*   **拍摄场景**：你的内容是在固定室内场景、变化的室外场景，还是在车内等特定场景拍摄？可以参考对标账号的常用场景。
*   **声音呈现**：你是自己出镜讲解，还是不出镜仅使用配音？现在通过剪辑软件（如剪映）用配音呈现对话场景也很常见。你需要评估自己的表达是否流畅。
*   **内容类型**：你的内容是以**剧情**、**干货分享**还是**生活记录**的形式呈现？
    *   **干货型**：需要精心准备文案脚本。
    *   **生活记录型**：需要积累大量的日常拍摄素材。
    *   **剧情型**：创作难度相对较高。

### 4. 记忆点：让用户记住你的关键

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_152.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_154.png)

记忆点是让用户能够记住并识别你的独特设计。它需要通过固定的环节反复强化，刻入用户心智。

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_156.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_158.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_160.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_162.png)

以下是几种常见的记忆点设计：

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_164.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_166.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_168.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_170.png)

*   **视频前几秒的固定动作或特效**
*   **统一的开场白**
*   **贯穿视频的固定背景音乐（BGM）**
*   **统一出现的标志性物品**
*   **风格统一的服装**

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_172.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_174.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_176.png)

![](img/967e61bb7e1d3afe6c5a581f8d47a7f7_180.png)

**设计逻辑分析**：
*   **前置设计**：固定动作、特效、开场白等需要放在**视频开头**。当用户浏览你的主页时，每个视频开头都采用相似的格式，能极大地强化记忆。
*   **全局贯穿**：统一的BGM和标志性物品（如背景摆设、服装风格）则**贯穿整个视频**。例如，一听到特定的BGM，用户就能联想到你；统一的视觉风格也能加深印象。

1.  **角色**是你在赛道中的专业身份。
2.  **内容**需要保持垂直，提供持续价值。
3.  **形式**决定了内容如何被呈现和制作。
4.  **记忆点**是通过精心设计的固定环节，让用户牢牢记住你的关键。
