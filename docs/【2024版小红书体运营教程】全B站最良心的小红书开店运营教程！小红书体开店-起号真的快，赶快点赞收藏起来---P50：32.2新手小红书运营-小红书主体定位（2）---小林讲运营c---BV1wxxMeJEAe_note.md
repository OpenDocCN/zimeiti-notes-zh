在本节课中，我们将要学习小红书店铺运营中至关重要的“人群定位”。人群定位是店铺精准引流和转化的基础，理解并掌握它，能让你的店铺运营事半功倍。

上一节我们介绍了店铺定位的基础概念，本节中我们来看看如何为店铺进行精准的人群定位。

### 人群定位的核心逻辑

小红书店铺的人群定位，其核心逻辑与账号定位相似，但更侧重于“人”的维度。不要小看小红书的搜索体系，我们了解这些内容，是为店铺定位所做的技术准备。只有了解这些，我们才知道后续店铺应如何操作。

### 人群定位的五个维度

以下是进行人群定位时需要分析的五个关键维度，它们共同构成了店铺的目标用户画像。

1.  **个人属性**
    用户的个人属性包括性别、年龄范围和喜欢的内容。这与我们最初选择店铺类型时的考量是相似的。做店铺与做账号不同，账号的属性需要与店铺属性相匹配，才能有效推广。若只做了账号属性而忽略了店铺属性，引流的人群将不精准。因此，我们需要根据产品来选择目标用户的个人属性，这会将目标范围进一步缩小，因为小红书的产品搜索推广比大众推广更精准。选择方式与之前相同，但这次是以店铺为核心来确定人群。

2.  **消费能力**
    用户的消费能力是根据其年度总购买力，分12个月计算平均值来评估的。公式可以理解为：
    `月均消费能力 ≈ 年度总消费 / 12`
    小红书系统会通过用户的购买行为（如产品类型、价格、搜索方式）来确认其消费等级。因此，我们在为产品定价时，必须根据目标用户的消费能力来确定，才能吸引到这部分人群。

3.  **购买类型**
    系统分析用户购买行为后，通常只会给用户推广最多三个类型的产品。我们可以将其分为日用品、生活用品、户外用品、家具玩具等大类目。在自然搜索时，基本只显示三个类型的产品，且根据用户最近的购买行为来区分。这是由于小红书的“千人千面”算法非常明显，用户经常浏览什么内容，系统就会展示什么内容，通常两到三天即可定型。前三个类型相对固定，后续搜索才会逐渐调整。

4.  **购买地域**
    小红书系统会根据用户的收货地址来分析不同地域的购买偏好。例如，分析北上广深等城市用户购买服装、食品或其他产品的占比。如果你的店铺产品属性恰好符合北京用户的偏好，那么系统在分配流量时，你的店铺出现在北京用户浏览页面的几率就会增大。系统会将对应目标的产品，分享给对应目标地域的人群。

5.  **购买习惯**
    用户的购买习惯也有划分，例如“货比三家”、长时间浏览后再购买等。这是根据个人用户的搜索和兴趣来判断的。兴趣搜索越多，相应产品的推广就越广。但需要注意的是，如果用户在你的店铺浏览后却去别人的店铺成交，那么系统下一步推广别人店铺属性的权重就会高于你的店铺。虽然过去平台间存在一些利用此规则的竞争手段，现在已不常用，但这种潜意识的流量影响依然存在，需要我们注意。

### 综合应用与前期策略

![](img/997bb52583ac7fcfdf9e93cf9a895a6c_8.png)

![](img/997bb52583ac7fcfdf9e93cf9a895a6c_10.png)

综合来说，我们在操作店铺时，流程通常是：先进行**店铺定位**、**选品类产品**、初步了解**价格**定型，然后保持**产品时效性**，接着就是进行**人群定位**。

![](img/997bb52583ac7fcfdf9e93cf9a895a6c_12.png)

对于人群定位中的个人属性、消费能力、购买类型、购买地域、购买习惯这五点，它们是系统匹配流量的关键。如果你想将店铺做大，就需要深入了解这三点（个人属性、消费能力、购买类型）。如果只是前期熟悉节奏，可以暂不深入操作。

因为一旦精准设定了这些维度，虽然推广人群属性会更精准，但初始的展示量可能会减少。例如，原本可能有2000-3000的展示量，精准设定后可能只有1000左右，但人群更精准。因此，在前期推广时，若想先测试或拉取较广范围的人群，可以暂缓深入操作这三点，先完成前面的基础定位即可。

这就是店铺定位与人群定位的综合应用方式。后续的价格定位，实际上是这三个内容（购买类型、地域、习惯）的延伸。价格定位完成后，才能更精准地匹配这三个维度。

如果你前期不想操作这些，也可以自行定价，这并不影响。但需要注意，前期做完这些精准设定后，店铺和账号的初始流量属性可能会减少近一半，不过后续的数据提升效果会更好，这只是前期需要承受的影响。

本节课中我们一起学习了小红书店铺人群定位的五个核心维度及其应用策略。理解并善用这些维度，能帮助你在运营初期建立清晰的用户画像，为后续的精准营销和流量转化打下坚实基础。下一节，我们将深入探讨“价格定位”如何与人群定位相结合，优化你的整体运营策略。
