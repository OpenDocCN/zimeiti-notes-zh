# 新媒体运营入门：P2：1、流量分发算法与创作者标签

![](img/389d7fddee0307fc64172a7ff12cc81a_1.png)

在本节课中，我们将要学习新媒体运营的核心基础——平台的流量分发算法。理解这套算法，是做好短视频、图文笔记等内容运营的第一步。我们将重点探讨“创作者标签”这一关键概念，了解它如何决定你的内容被推送给谁，以及如何影响你的流量数据。

![](img/389d7fddee0307fc64172a7ff12cc81a_3.png)

## 什么是流量分发算法？

![](img/389d7fddee0307fc64172a7ff12cc81a_5.png)

![](img/389d7fddee0307fc64172a7ff12cc81a_7.png)

无论我们发布短视频还是图文笔记，内容发布后都会获得播放量（小眼睛）。从最初的三五百播放，增长到几千甚至几万，直至成为爆款，这背后的推动力就是平台的流量分发算法。它决定了流量的构成与来源。

![](img/389d7fddee0307fc64172a7ff12cc81a_9.png)

要理解这套算法，我们首先需要了解两个核心概念：**创作者标签**和**用户标签**。本节我们先聚焦于创作者标签。

## 什么是创作者标签？

当我们发布第一个内容后，平台会将我们的账号识别为“创作者”。但此时存在一个问题：虽然我们发布了内容，平台却无法立刻准确知道我们的内容是关于什么的。

你可能会想，我们在标题、文案、标签（Tag）、封面等处都明确说明了内容主题（例如“女性穿搭”），平台为什么识别不了？原因在于，当前所有平台的内容判断机制都依赖于**机器审核**，而非人工判断。平台通过强大的算法和大数据分析系统，自动判断内容的合规性与主题。

![](img/389d7fddee0307fc64172a7ff12cc81a_11.png)

这意味着，在刚成为创作者时，你的账号**没有精准的标签**。那么，最初的播放量从何而来？我们可以理解为，这些初始流量是**泛流量**。

![](img/389d7fddee0307fc64172a7ff12cc81a_13.png)

**泛流量**是指分发给不精准用户的流量。例如，你做女性穿搭内容，系统可能将其分发给中年男性或青少年。因为系统尚无法判断你的内容主题，它不会单纯依据你的标题或标签，而是通过**观察分发人群对内容的反应**来判断。

因此，平台需要为创作者打上一个标签，用以定义创作者所在的赛道和产出内容的类型。这就是**创作者标签**。前期流量上不去、互动数据差，往往就是因为创作者标签不精准。

![](img/389d7fddee0307fc64172a7ff12cc81a_15.png)

## 创作者标签的作用

![](img/389d7fddee0307fc64172a7ff12cc81a_17.png)

![](img/389d7fddee0307fc64172a7ff12cc81a_19.png)

创作者标签的作用很简单：**系统用于判断你的内容是关于什么的**。

![](img/389d7fddee0307fc64172a7ff12cc81a_21.png)

这个判断直接决定了一个关键结果：**你的内容最终会被分发给哪些人群**。

![](img/389d7fddee0307fc64172a7ff12cc81a_23.png)

![](img/389d7fddee0307fc64172a7ff12cc81a_25.png)

![](img/389d7fddee0307fc64172a7ff12cc81a_27.png)

例如，一个账号长期创作“女装穿搭”内容。系统成功识别后，后续就会将内容推荐给对女装穿搭感兴趣或曾有相关浏览行为的人群，而非泛流量。

在这种精准推荐下，会产生两种结果：

*   **分发精准**：你的流量数据会**越来越高**。
*   **分发不精准**：你的流量数据会**越来越差**。

这很好理解：如果用户对你的内容感兴趣，他们会有更长的停留时间，并在内容优质的前提下，提供点赞、收藏、评论等数据支持。反之，如果分发给错误的人群（泛流量），用户会迅速划走，导致**跳出率**极高。

在短视频平台，即使瞬间划走也算一个播放量，但这种播放量的数据贡献极差。一旦此类数据过多，平台可能判定创作者内容质量差，从而**限制流量分发**，也就是常说的“限流”。许多账号后期流量起不来或越来越差，这正是核心原因之一。例如，企业账号初期就发布硬性产品营销内容，在不精准的流量池中，极易导致数据差，长期积累会使账号难以成长。

## 账号初期：没有标签的阶段

![](img/389d7fddee0307fc64172a7ff12cc81a_29.png)

我们知道，账号初始阶段没有准确标签，系统会分发泛流量。泛流量具有以下特性：
*   性别不精准
*   年龄不精准
*   兴趣爱好不精准

![](img/389d7fddee0307fc64172a7ff12cc81a_31.png)

对于垂直赛道（如穿搭、美妆）的内容，面对这些不精准流量，账号的完播率、跳出率等数据通常不乐观。

那么，这个阶段如何把流量拉起来？答案是：创作**娱乐性内容**，即泛流量人群普遍喜欢的内容。

以下是泛流量人群普遍喜欢的内容类型：
*   搞笑段子
*   新闻热点
*   吃瓜、猎奇类内容

这类内容能覆盖绝大部分人群的兴趣。例如，1万个泛流量中，可能只有10个人对特定穿搭内容感兴趣，基数太小，难以支撑起优质数据。但如果是搞笑或热点内容，感兴趣的可能有八九千人。这就是为什么初期做综艺剪辑、明星八卦等内容流量更容易起来——因为它们契合了大多数人刷短视频为了**娱乐**和**打发时间**的核心需求。

![](img/389d7fddee0307fc64172a7ff12cc81a_33.png)

当然，选择这类内容时，需要考虑它是否是你的目标赛道，以及后续的变现、发展和涨粉路径。

除了提供初始流量，平台在账号无标签阶段分发泛流量，还有另一个重要目的：**对我们的内容进行测试**，以判断内容究竟是关于什么的。

![](img/389d7fddee0307fc64172a7ff12cc81a_35.png)
