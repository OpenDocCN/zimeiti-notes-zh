# 视频号创作分成计划：P7：5.实操教学-账号设置 🛠️

在本节课中，我们将学习如何从零开始设置一个专业的视频号账号。账号设置是吸引观众的第一步，相当于店铺的“门面装修”，良好的第一印象有助于获得更多关注。

上一节我们介绍了账号定位，本节中我们来看看如何具体设置账号信息。

## 开通与基础设置

![](img/9af4227ee8578794f41d7aedec10e457_1.png)

第一步是开通视频号。确保你的微信已升级到最新版本，并在设置中打开视频号功能。

![](img/9af4227ee8578794f41d7aedec10e457_2.png)

开通后，在微信发现页会出现一个蝴蝶形状的图标，这就是视频号的入口。

## 账号名称设置

起名字时，可以借助一些工具找到热度较高的词语，将其融入账号名，以获取搜索流量。

以下是设置账号名称的具体方法：

1.  **使用“微信指数”小程序**：在微信右上角的“搜一搜”框内，搜索“微信指数”小程序。该工具每天有约5亿用户使用，类似于百度指数。
2.  **查询热词**：在小程序中搜索你感兴趣领域的关键词（如“影视”、“情感”、“家居”），查看其搜索量。选择搜索量较高的词语融入账号名。
3.  **利用其他数据平台**：例如“西瓜数据”等平台，可以查看各内容领域的流行趋势和受欢迎话题。

**核心逻辑**：互联网经济中存在信息差。通过工具发现高热度词汇并用于命名，可以为账号带来免费的搜索流量。

## 头像设置

![](img/9af4227ee8578794f41d7aedec10e457_4.png)

![](img/9af4227ee8578794f41d7aedec10e457_5.png)

头像代表账号形象，需注意避免侵权和过于随意。

以下是头像设置的两个注意事项：

1.  **避免使用明星/偶像头像**：此类头像存在侵权风险。
2.  **避免使用风景照**：过于随意，缺乏辨识度。

头像风格应与账号定位一致：
*   **可爱路线**：使用可爱、卖萌风格的头像。
*   **专业路线**：使用个人形象照或职业照。
*   **品牌/产品路线**：直接使用品牌Logo或产品名称。

## 个人简介设置

![](img/9af4227ee8578794f41d7aedec10e457_7.png)

简介是展示账号价值、引导关注的关键区域，最长可写十行，应充分利用。

以下是丰富个人简介的几个方向：

1.  **引导关注**：例如首行写“点击右上方黄色字加关注”，并搭配表情符号。
2.  **说明直播时间**：告知用户固定的直播日程。
3.  **阐述账号价值**：清晰说明账号能提供什么内容或帮助（如心灵慰藉、生活技巧、影视推荐）。
4.  **账号互推**：如果你运营多个账号或有公众号，可以在此引导，实现粉丝互通。

**核心要点**：简介需要回答“用户为什么要关注你”这个问题，信息丰富的简介能提升关注转化率。

![](img/9af4227ee8578794f41d7aedec10e457_9.png)

## 账号认证

账号认证分为个人认证和企业认证，能提升账号的权威性和流量权重。

以下是关于认证的要点：

1.  **个人认证**：完成实名后，根据提示选择兴趣、知识教育等类别，上传相关资质即可。
2.  **企业认证**：需要上传营业执照和企业法定代表人身份证。企业号更像官方账号，通常享有更多流量扶持和优先权益。
3.  **认证注意事项**：
    *   认证名称最好与关联的公众号保持一致。
    *   每年有两次认证机会，失败后可能需要等待次年或注册新号，请谨慎操作。

**补充说明**：即使没有自己的企业，也可以通过其他方法完成企业认证，具体方法可在后续课程中探讨。
