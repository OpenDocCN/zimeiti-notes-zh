# 抖音运营教程：P8：3、如何完善个人资料 🧑‍💼

在本节课中，我们将学习如何完善抖音的个人资料。一个专业且吸引人的个人主页是吸引粉丝、建立信任的第一步。我们将详细讲解头像、昵称和个人简介的设置方法与核心原则。

---

## 头像选择

上一节我们介绍了个人资料的重要性，本节中我们来看看如何选择头像。

头像应清晰、专业并与账号定位相关。以下是几种常见的头像选择方案：

*   有实体店的同学，可以使用实体店铺的照片。
*   没有实体店的同学，可以使用个人自拍照或生活照。
*   宝妈类型的同学，可以使用和宝宝的合照。
*   做宠物类型的同学，可以使用宠物的照片。

---

## 昵称设置

设置好头像后，一个关键的步骤是起一个好记且专业的昵称。**切勿使用自己的真实姓名**（如：小月、兰亭）。

因为普通人的名字会被平台判定为“玩家”而非“创作者”，从而影响流量推荐。同时，暴露个人信息也存在风险。明星可以使用本名，因为他们本身具有知名度和号召力。

那么，我们应该如何起名呢？核心原则是：**名字 + 定位/职业/地区**。

以下是具体的命名方法，请选择其中一种：

1.  **名字 + 定位**：将昵称与你的内容领域结合。
    *   公式：`{昵称} + {领域}`
    *   例如：小月说游戏、小月小厨房、小月聊房、小月说车、小月穿搭。

2.  **名字 + 职业**：将昵称与你的职业身份结合。
    *   公式：`{职业} + {昵称}` 或 `{昵称} + {职业}`
    *   例如：宝妈小月、律师小月、摄影师小月、造型师小月。

3.  **名字 + 地区**：适合具有地域特色的账号（如三农、探店）。
    *   公式：`{地区} + {昵称}` 或 `{昵称}在{地区}`
    *   例如：赣州月姐、小月在北漂、小月在海外。

**起名注意事项**：
*   控制在五个字以内，便于记忆。
*   避免使用繁体字和特殊符号（如：❤️✨），不利于用户搜索和识别。

## 个人简介撰写

![](img/9c89dadd073fe7882165484d482c6f92_1.png)

昵称设置完成后，我们需要用一段简短的文字来介绍自己，这就是个人简介。它位于个人主页的醒目位置。

一份优秀的个人简介应围绕以下三个核心要素展开：

1.  **我是谁**：介绍你的身份或特点。
2.  **我是做什么的**：明确你的专业领域或内容方向。
3.  **关注我能得到什么**：向用户说明你能提供的价值。

![](img/9c89dadd073fe7882165484d482c6f92_3.png)

**示例**（母婴账号）：
> 中年大叔依然保持母婴初衷（我是谁）。从事母婴行业13年，做母婴商贸8年有余（我是做什么的）。关注我，分享育儿经验（关注我能得到什么）。

你可以根据自己的性格（幽默、可爱等）来调整简介的语气，使其更生动活泼。

## 重要注意事项：联系方式

在完善资料时，有一个常见的误区需要特别注意：**不要在个人简介或主页任意位置留下微信、电话等外部联系方式。**

这是平台明确的违规引流行为。抖音与微信（腾讯）属于不同公司，在抖音平台引导用户添加微信，会被视为将平台流量引向他处。

**允许挂载联系方式的情况只有两种**：
1.  已认证**企业蓝V账号**，可以展示官方联系电话。
2.  粉丝达到1万并成功入驻**抖音星图平台**（关于星图，我们将在后续变现章节详细讲解）。

如果已经违规挂载，请立即删除。

本节课中我们一起学习了完善抖音个人资料的完整流程。我们明确了头像的选择应与定位相符，昵称应采用“名字+定位/职业/地区”的公式，个人简介需清晰包含“我是谁、做什么、有何价值”三要素，并切记不要违规放置外部联系方式。一个好的个人资料是账号成功的基石，请务必认真设置。
