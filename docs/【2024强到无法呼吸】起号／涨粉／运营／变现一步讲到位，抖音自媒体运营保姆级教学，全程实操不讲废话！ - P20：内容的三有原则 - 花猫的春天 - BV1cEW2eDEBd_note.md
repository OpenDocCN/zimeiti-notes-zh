# 抖音自媒体运营：P20：内容的三有原则

在本节课中，我们将学习如何确定内容定位，并深入理解一个核心原则——“三有原则”。这个原则是判断内容能否吸引用户的关键。我们将通过分析具体案例，帮助你理解如何创作出用户真正喜爱的作品。

在上一节，我们探讨了内容定位的重要性。本节中，我们来看看如何具体判断什么样的内容能吸引用户。用户喜欢一个作品，通常是因为这个作品满足了他的某些需求。

![](img/c2038f71bc5eb72bc7d01684fc005786_11.png)

![](img/c2038f71bc5eb72bc7d01684fc005786_12.png)

以下是三个核心原则，它们被称为“三有原则”：

![](img/c2038f71bc5eb72bc7d01684fc005786_14.png)

1.  **有用处**：内容能为用户提供实际价值或帮助。例如，教程、攻略、干货分享等。
    *   **示例**：一个标题为“一秒钟让素人变女神的打光方式”的视频，提供了实用的摄影技巧，对想提升拍照水平的用户非常有用。
    *   **另一个示例**：一个提供《卡农》简易版钢琴简谱的视频，对于钢琴初学者来说是有用的学习资料。

2.  **有兴趣**：内容本身具有趣味性，能引发用户的好奇心或喜爱。例如，萌宠、颜值、炫技、情景剧等。
    *   **示例**：一个关于可爱小猫的宠物视频，能吸引喜欢宠物的用户观看和点赞。

![](img/c2038f71bc5eb72bc7d01684fc005786_16.png)

3.  **有共鸣**：内容能触动用户的情感，引发内心的认同感。例如，励志故事、情感鸡汤、生活感悟等。
    *   **示例**：一个探讨“男孩子长大后话变少”的情感视频，描述了默默承受压力的心境，容易让有类似经历的观众产生共鸣，从而关注发布者。

简单来说，**三有原则 = 有用处 + 有兴趣 + 有共鸣**。抖音上大多数受欢迎的视频都符合这三点中的至少一点。

理解了“三有原则”后，你可能会想：内容类型这么多，我到底该选择哪一类呢？先别急着做决定。我们还需要解决一个更常见的问题：为什么有些内容看起来不错，却无法有效涨粉？

![](img/c2038f71bc5eb72bc7d01684fc005786_20.png)

![](img/c2038f71bc5eb72bc7d01684fc005786_22.png)

让我们分析一个具体案例。有一个名为“贺鸣酒”的账号，其视频制作精良，单个视频获得了5.8万点赞，但粉丝数只有5173。计算一下**粉丝转化率**：

`粉丝数 / 总获赞数 ≈ 5173 / 58000 ≈ 0.09`

这意味着大约每获得100个点赞，才能换来9个粉丝，转化率偏低。账号主页显示其内容均为唱歌，领域是垂直的。那么问题出在哪里？

主要有两个原因：**主题IP不清晰**和**账号搭建不完善**。本节我们先探讨第一个问题：主题IP。

**垂直**是指账号持续专注于同一领域（如只做美食、只做唱歌）。这对于普通人做账号至关重要，因为用户通常因内容价值而关注，而非博主本人。如果内容杂乱，用户就找不到持续关注的理由。

然而，仅仅“垂直”还不够。在垂直领域内，你需要一个更鲜明、更具吸引力的**主题IP**。它让你的账号在同类内容中脱颖而出，给用户一个“必须关注你而不是别人”的理由。下一节，我们将详细讲解如何打造强有力的主题IP。
