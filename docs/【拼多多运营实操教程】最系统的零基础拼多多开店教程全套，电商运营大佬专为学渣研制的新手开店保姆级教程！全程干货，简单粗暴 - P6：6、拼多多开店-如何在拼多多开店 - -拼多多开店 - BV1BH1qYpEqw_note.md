# 拼多多运营实操教程：P6：如何在拼多多开店 🛒

在本节课中，我们将学习如何在拼多多平台完成店铺的入驻流程。无论您是想开个人店铺还是企业店铺，本教程都将为您提供清晰、简单的操作指引。

## 开店前的准备

在开始操作前，您需要明确自己的店铺类型和主营类目，并准备好相应的资质材料。拼多多主要分为**个人店铺**和**企业店铺**，两者的入驻流程基本一致，主要区别在于所需提交的资质材料不同。本节将以个人店铺为例进行说明。

![](img/150b9ddd32a7c064bb397172fc6f4acc_1.png)

## 入驻操作流程

拼多多支持通过电脑端和手机端两种方式完成店铺入驻。以下是两种方式的具体步骤。

### 电脑端入驻流程

上一节我们介绍了开店前的准备，本节中我们来看看如何在电脑上完成入驻。

以下是电脑端入驻的具体步骤：
1.  **访问官网**：在浏览器中搜索并进入拼多多官方网站。
2.  **点击入驻**：在官网首页顶部的菜单栏中，找到并点击“商家入驻”。
3.  **开始注册**：
    *   在打开的页面中，输入一个**从未在拼多多开过店**的手机号码。
    *   设置您的店铺登录密码。
    *   输入手机收到的验证码。
    *   点击“**0元入驻**”。
4.  **填写店铺信息**：
    *   设置店铺名称。
    *   选择主营类目。如果您尚未确定具体售卖商品，**一律选择“普通商品”**。选择“医药健康”类目后无法更改，而“普通商品”类目后期可补充资料进行更换。
    *   根据页面提示，上传开店人的身份证正反面照片等个人信息。
5.  **完成实名认证**：信息提交后，系统会弹出一个二维码。请使用微信扫描该二维码，按照指引完成人脸识别实名认证。

![](img/150b9ddd32a7c064bb397172fc6f4acc_3.png)

完成以上步骤后，提交审核即可。个人店铺的审核通常会在**两个工作日**内完成。

### 手机端入驻流程

电脑端的流程介绍完毕，现在我们来看看更为便捷的手机端如何操作。

以下是手机端入驻的具体步骤：
1.  **下载APP**：在手机应用商城搜索并下载“**拼多多商家版**”APP。
2.  **打开应用**：打开APP后，在首页点击“**0元开店**”。
3.  **注册账号**：
    *   输入一个未注册过的手机号码。
    *   填写收到的短信验证码。
    *   点击“立即开店”。
4.  **选择店铺类型**：根据自身情况（个人、个体工商户、企业等）选择店铺类型，不同类型需提交的资料不同。
5.  **提交资料与认证**：
    *   按照提示上传身份证照片。
    *   设置店铺名称。
    *   完成人脸识别验证。

资料全部提交后，等待平台审核即可，审核时效同样为**两个工作日**以内。

## 总结与注意事项

本节课中，我们一起学习了在拼多多开店的完整流程。我们来总结一下核心要点和注意事项：
*   **流程选择**：您可以通过**拼多多官网（电脑端）** 或 **拼多多商家版APP（手机端）** 完成入驻。
*   **核心操作**：关键步骤是使用新手机号注册、选择类目（建议新手选“**普通商品**”）、提交身份信息并完成**人脸识别实名认证**。
*   **审核时间**：个人店铺审核速度快，一般在**2个工作日内**完成。
*   **资料准备**：开店前请根据店铺类型准备好相应的资质材料，如身份证、营业执照等。
