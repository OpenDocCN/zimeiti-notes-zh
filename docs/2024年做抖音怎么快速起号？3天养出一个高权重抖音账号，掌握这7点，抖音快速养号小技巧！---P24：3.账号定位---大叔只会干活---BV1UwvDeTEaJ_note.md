# 抖音运营教程：P24：3.账号定位

![](img/83e9aa28e8e5c49a72d57d227b14967e_0.png)

在本节课中，我们将要学习抖音账号运营中至关重要的一步：**账号定位**。清晰的定位是账号获得精准流量、快速成长的基础。我们将通过反思三个核心问题来构建你的账号定位框架。

![](img/83e9aa28e8e5c49a72d57d227b14967e_2.png)

## 概述：定位的三个核心问题

对账号定位的理解需要反思三个问题：**你是谁**，**你是干什么的**，以及**为什么要关注你**。围绕这三个问题展开思考，是建立高权重账号的第一步。

## 确定账号类型与细分赛道

上一节我们介绍了定位的核心问题，本节中我们来看看如何具体确定账号的类型。

首先，要确定接下来要做的账号类型。除了之前章节提到的大类赛道（即抖音定义的一级垂类），还需要思考在一级垂类下细分的二级赛道是什么。

**公式：账号垂直度 = 一级垂类 + 二级垂类**

例如，参加运动会时，你报的项目是“田径”，这是一级垂类。而你最擅长且具有优势的项目是“百米赛跑”，这就是你的二级垂类。通过定位账号的细分类目，可以明确账号的垂直类型。

在抖音中，无论任何行业，只要定位足够清晰，都有机会成功。因为抖音流量巨大，一旦做起来，增长和裂变速度会非常快。

## 明确账号主体

确定了赛道方向后，接下来需要明确账号的呈现主体是谁。

账号主体是指账号的拥有人，以及最终呈现给粉丝的“主人公”。需要让粉丝清楚知道是在看谁的作品。在抖音商业类账号中，账号主体主要分为两类：

以下是两种主要的账号主体类型：

*   **商家账号**：如果账号主体是公司，即为商家账号。其内容应偏向商业化，并需不断通过内容拉近与粉丝的距离，避免过于严肃。这样既能建立信任感，也能带来好感，更符合抖音的娱乐属性。
*   **达人账号**：如果账号主体是个人，即为达人账号。首先需要解决的是信任问题。可以通过IP打造的方式，为账号树立一个积极、正面的人设。

## 明确账号内容与认证

明确了“你是谁”之后，下一步要解决“你是干什么的”这个问题。

最直接的方式是通过抖音进行身份认证，即展示你的职业。达人可以通过认证获得**黄V**，公司可以通过认证获得**蓝V**。认证后，粉丝可以清晰看到认证标签，从而明白你的职业和账号大致内容。

同时，商家认证蓝V或达人认证黄V也能获取平台背书，增加信任感。

在未认证之前，也可以通过账号设置向粉丝明确传达你是做什么的。以下是可进行统一设置的要素：

*   头像
*   背景图
*   个人简介
*   视频封面

通过对自身商业模式和账号类型的确定，将这些要素进行与业务关联的统一设置。

## 提供核心价值：粉丝为何关注你

最后，需要花时间思考最关键的问题：**你将为粉丝提供什么样的价值？** 这是决定用户是否关注你的核心。

在抖音中，粉丝对博主的价值需求主要分为以下几类：

以下是五种主要的粉丝价值需求：

1.  **知识价值**
    *   **认知型**：如人生思考、思维方式、读书分享、行业内幕、成功经验、个人提升等。
    *   **干货型**：如技能提升、经验总结、操作步骤、冷知识等。
    *   **常识普及型**：如生活、健康、产品、科学、历史、地区等常识的解读与普及。
    *   **提供给粉丝的价值**：**认同感**。如三观认同、观点认同，减少知识盲区带来的获得感，进而产生对作者的认同。

2.  **情绪价值**
    *   情绪是影响观众的重要因素。作品中容易引发共鸣的情绪类型包括：开心愉悦（搞笑、夸张）、关系共鸣（情侣、婆媳、家庭、职场等）、情绪波动（争议话题、反转反差内容）、以及怀旧、伤感、震惊、好奇等。
    *   **提供给粉丝的价值**：**共鸣**。利用表情、文字、画面、声音、肢体动作等准确抒发情绪，直接感染观众。

3.  **感觉价值**
    *   博主通过短视频的画面、文字和声音传递信息。提供的价值包括：视频的画面美感、构图，以及给观众带来的喜欢、治愈、视觉冲击感，或个人魅力带来的仰慕感。也包括声音带来的节奏感、心跳加速或听觉享受。
    *   **涉及的账号类型**：颜值类、萌宠类、美景类、才艺类、明星类、文化类等。
    *   **提供给粉丝的价值**：**共情**。因表情、文字、画面、声音、肢体动作等的自然表现，让人产生联想，并在情绪、认知等意识层面与作者产生连接，从而间接引发共鸣。

4.  **体验价值**
    *   抖音让用户看到了不一样的生活方式，体验到了未曾经历的事物。例如流量很大的美食账号（享用与制作体验）、旅行体验、运动体验、角色体验等。
    *   **提供给粉丝的价值**：主要是**好奇心的满足**，以及紧张感、激动、感动、获得感等。

5.  **期待价值**
    *   当用户被你的短视频内容、观看体验或对作者本人产生好奇时，看完视频可能会意犹未尽，从而点击关注，期待下次更新。
    *   许多博主会利用这一点，不将所有内容在一个视频中呈现完，以制造悬念，留住粉丝的期待。

## 总结

本节课中，我们一起学习了抖音账号定位的完整框架。我们从反思“你是谁、你是干什么的、为什么关注你”这三个核心问题出发，逐步明确了如何确定账号的细分赛道、选择账号主体、通过认证和设置明确内容方向，并最终落脚于为粉丝提供知识、情绪、感觉、体验、期待这五大核心价值。一个清晰、垂直且能持续提供价值的账号定位，是你在抖音快速起号、养出高权重账号的坚实基石。
