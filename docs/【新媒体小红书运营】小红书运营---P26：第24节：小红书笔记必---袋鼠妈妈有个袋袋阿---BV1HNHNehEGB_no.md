# 小红书运营：P26：小红书笔记必备六大爆文模板全解析

在本节课中，我们将学习小红书笔记的六大爆文模板。这些模板旨在帮助你在特定场景下，撰写出更容易被推荐至首页的笔记。我们将详细解析每种模板的结构与核心要点，助你轻松起号。

## 专业教程类模板

上一节我们介绍了爆文模板的总体分类，本节中我们来看看第一种类型：专业教程类模板。这类模板适用于有明确步骤或流程的内容领域。

专业教程类模板适用于**美妆、瘦身、美食、软件操作、美容摄影**等内容。这些内容通常具有清晰的步骤，例如第一步做什么、第二步做什么。

以下是撰写专业教程类笔记的具体建议：

*   **首图类型**：建议使用**对比效果图**或**步骤分解图**。
    *   **对比效果**：例如展示使用某款美妆产品前后的效果对比，或自家产品与别家产品的效果对比。
    *   **步骤分解**：例如摄影教程，首图可以展示第一步购买配件、第二步组装、第三步选择场景、第四步拍照等关键步骤。
*   **标题要点**：标题应突出内容的特别之处。可以参考我们之前课程中关于标题起法的规律。
*   **正文结构**：正文内容应包含所需的**工具、材料**和具体**步骤**。
    *   书写时务必条理清晰，避免内容混杂。建议采用大点套小点的结构，例如：
        ```
        一、 核心步骤
            1. 第一步详解（可配图）
            2. 第二步详解（可配图）
        二、 注意事项
        ```
    *   若教程需要提前准备，可在正文开头单独列出“**准备工作**”，写明需要的工具和材料。

## 视频笔记类模板

了解了图文类的教程模板后，我们再来看看视频笔记的创作要点。视频笔记的核心在于快速吸引用户并传递信息。

视频笔记有两大基本要素：

1.  **视频时长**：初期建议**1分钟**以内。获得1分钟权限后，可根据内容需要制作**5分钟或10分钟**的视频。
2.  **视频比例**：必须采用**3:4**的竖屏比例。其公式为 **`高度 : 宽度 = 3 : 4`**。这是一个竖长的长方形，与传统的4:3横屏比例不同。

当前许多视频面临的问题是，开头几秒无法抓住观众眼球，导致用户迅速划走。那么，如何制作有吸引力的视频开头呢？

关键在于 **“重点前置”** ，即将最精彩、最核心的内容放在视频开头。我们可以参考“毒舌电影”的讲解方式：

> 例子：他在讲解一部电影时，开头直接呈现剧情最高潮或最有趣的片段，例如“他已经饿了两天，身无分文，想吃霸王餐”。仅用几秒钟就抛出核心冲突，吸引用户看下去，然后再介绍电影背景等信息。

将这种思维应用到产品推广视频中，应该这样做：

> 错误示范：按部就班地说“第一步……第二步……”。
>
> 正确示范：开头直接展示对比——“这是用了我们家产品的效果，这是用了别人家产品的效果。为什么差距这么大？今天这一集告诉你答案。”

这种“重点前置”的方法能极大提升视频开头的吸引力，对作品结构和叙事逻辑也很有帮助。

---

本节课中我们一起学习了小红书的两大核心爆文模板：**专业教程类模板**与**视频笔记类模板**。关键要点在于：教程类笔记需**步骤清晰、对比鲜明**；视频类笔记需**把控时长与比例，并运用“重点前置”技巧**快速抓住用户注意力。掌握这些模板，能有效提升你笔记的推荐概率。
