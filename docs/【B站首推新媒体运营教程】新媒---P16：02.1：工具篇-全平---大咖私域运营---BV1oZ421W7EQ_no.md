# 新媒体运营教程：P16：02.1：工具篇-全平台短视频文案一键提取 📝

在本节课中，我们将学习一个非常实用的运营工具，它能帮助我们从多个平台的短视频中，一键提取出完整的文案内容。掌握这个工具，可以极大地提升我们收集素材和创作内容的效率。

上一节我们介绍了文案的重要性，本节中我们来看看如何高效地获取文案素材。以往，当我们看到优秀的短视频时，想要借鉴或分析其文案，通常需要手动记录，或者使用一些平台特定的工具（如“轻抖”小程序）或视频剪辑软件（如剪映）来提取字幕。这些方法往往效率较低，且难以跨平台操作。

为了解决跨平台提取文案的难题，今天分享一个强大的在线工具。它支持抖音、快手、小红书、哔哩哔哩、视频号等多个主流平台。你只需将短视频的链接粘贴到工具中，即可在线一键提取出完整的文案。

以下是该工具的核心操作流程：

![](img/ff4866c852c04eb49366764759fc7470_1.png)

1.  **复制链接**：在目标平台（如抖音）找到你想提取文案的短视频，复制其分享链接。
2.  **粘贴提取**：打开该在线工具网页，将复制的链接粘贴到指定输入框。
3.  **获取文案**：点击“提取文案”按钮，稍等片刻，工具就会自动生成完整的文字内容。
4.  **复制与格式化**：提取出的文案可以一键复制。工具还提供“格式化”功能，能将文案按对话或字幕段落清晰排版，方便后续使用。

![](img/ff4866c852c04eb49366764759fc7470_3.png)

为了让大家更直观地理解，我们进行实际操作演示。

![](img/ff4866c852c04eb49366764759fc7470_4.png)

![](img/ff4866c852c04eb49366764759fc7470_6.png)

首先，我们尝试提取一个抖音短视频的文案。我们搜索“情感语录”并找到一个合适的视频，复制其链接。

![](img/ff4866c852c04eb49364759fc7470_3.png)

![](img/ff4866c852c04eb49364759fc7470_4.png)

将链接粘贴到工具中并点击提取，很快便得到了文案结果：“听完你就释怀了，村上春树说，我动了离开你的念头，不是因为你不好，也不是因为不爱了……”。经对比，提取的文案准确无误。

![](img/ff4866c852c04eb49364759fc7470_6.png)

接下来，我们测试其对哔哩哔哩平台的支持。同样，复制一个B站视频的链接（通常以 `BV` 号开头）。

将B站视频链接粘贴到工具中并提取，成功得到了文案：“新手做直播电商，怎么才能打造出直播爆品，在直播销售的不同阶段……”。这证明了该工具出色的跨平台兼容性和准确性。

除了在线提取，该工具也支持上传本地视频文件来提取文案，功能非常全面。

这个工具在运营工作中用途广泛。提取出的文案，我们可以直接用于分析学习，也可以经过修改和调整，作为我们自己创作口播视频、文章或脚本的素材基础。它集成了多个平台专用工具的功能，能节省大量寻找、下载和转换的时间。

本节课中我们一起学习了一个强大的全平台短视频文案一键提取工具。它操作简单，只需一个链接即可跨平台工作，能有效提升我们收集灵感、分析内容和进行二次创作的效率。希望这个工具能为你的运营工作带来实质性的帮助。
