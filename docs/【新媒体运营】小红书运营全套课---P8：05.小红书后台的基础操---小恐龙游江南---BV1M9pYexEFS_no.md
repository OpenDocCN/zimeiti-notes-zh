# 小红书运营全套课程：P8：05.小红书后台的基础操作（手机端）📱

![](img/1fc14038a001177a4dfcc39923a5b677_0.png)

## 概述
在本节课中，我们将学习如何使用手机端的小红书商家后台——千帆系统。我们将从下载应用开始，逐步了解后台的核心功能模块，包括消息管理、店铺信息设置、商品发布、物流与地址配置以及营销工具的使用。通过本教程，你将掌握在手机上管理小红书店铺的基础操作流程。

## 课程内容

### 1. 下载与登录千帆系统
首先，我们需要在手机应用商店中搜索并下载“小红书千帆”应用。这是小红书官方的商家后台管理系统。

下载完成后，打开应用并登录你的店铺账号。登录后，你将看到后台的主页面。

### 2. 消息中心管理
在后台首页，你可以找到“消息”入口。点击进入消息中心。

以下是消息中心的主要功能：
*   **客服接待**：当买家咨询店铺商品或服务时，你可以在这里进行回复。
*   **系统通知**：平台会在这里发送各类通知，例如订单提醒、违规警告或活动公告。
*   **客服配置**：你可以在此处查看和管理店铺的客服设置。

消息中心是处理客户沟通和接收平台重要信息的关键区域。

### 3. 店铺初始化与工具总览
返回首页，你会看到“初始化”区域和“全部工具”入口。初始化是开设新店后的首要步骤。

点击“全部工具”，这里集成了店铺运营的所有核心功能。以下是主要工具列表：
*   **发布作品**：用于发布新的商品笔记。
*   **发布商品/管理商品**：上架新商品或管理已上架的商品。
*   **订单管理**：处理全部订单和售后订单。
*   **数据中心**：查看店铺的运营数据。
*   **商品笔记**：管理已关联商品的笔记。
*   **类目与品牌**：设置商品发布的类目和品牌信息。
*   **店铺装修**：装修你的店铺主页。
*   **群管理**：管理粉丝群聊。

### 4. 设置发货地址与物流模板
完成店铺初始化是运营的第一步，其中最关键的是设置发货地址。

**操作路径**：在“成长”任务或“全部工具”的“物流管理”中，可以找到“地址管理”。

**地址填写步骤**：
1.  点击“添加新地址”。
2.  填写联系人（可填自己）、所在地区及详细地址。
3.  填写有效的手机号码。
4.  **重要**：在“买家留言”或备注栏，**务必填写“退货请联系客服”**。如果直接填写详细地址，平台可能不会审核通过，这会影响到后续的退货流程。

设置好地址后，接下来需要配置物流模板。

**操作路径**：在“物流管理”中设置“运费模板”。

**模板配置说明**：
*   平台提供默认模板（如“全国包邮”）。
*   你可以创建新模板，例如：
    *   对新疆、西藏等偏远地区设置“不配送”。
    *   设置“买家付邮费”模板，指定非偏远地区邮费或满额包邮条件。
    *   代码示例（逻辑描述）：
        ```python
        if 地区 in [“新疆”, “西藏”]:
            配送方案 = “不支持配送”
        elif 订单金额 >= 88:
            配送方案 = “包邮”
        else:
            配送方案 = “邮费到付”
        ```

合理设置物流模板可以有效控制运营成本。

### 5. 营销工具：优惠券设置
营销工具能帮助提升店铺销量。我们以设置优惠券为例。

**操作路径**：在“全部工具”中找到“营销”或直接搜索“优惠券”，点击“创建优惠券”。

**优惠券类型**：
*   **店铺通用券**：全店商品均可使用的优惠券。
*   **指定商品券**：仅限特定商品使用的优惠券。
*   **店铺新人券**：针对首次访问店铺的新用户发放的优惠券。

创建后，你可以在“优惠券管理”中查看正在生效、已失效的优惠券，并参与平台推广活动。

![](img/1fc14038a001177a4dfcc39923a5b677_2.png)

### 6. 店铺认证与保证金缴纳
为了店铺能长期正常运营，需要完成认证并缴纳保证金。

**认证与支付设置**：
1.  在“店铺设置”中完成店铺类型（如个人店）认证。
2.  **必须**开通“微信支付”和“支付宝支付”功能。按照页面指引，绑定你的身份证和银行卡信息。
3.  根据平台提示，缴纳对应商品类目的保证金。保证金金额因经营类目不同而异，是店铺正常交易和获得消费者信任的保障。

## 总结
本节课我们一起学习了小红书千帆系统手机端的基础操作。我们从下载登录开始，依次了解了消息管理、店铺工具总览、发货地址与物流模板的设置、优惠券营销工具的创建，以及最后的店铺认证与保证金缴纳。整个流程逻辑清晰，操作并不复杂，关键在于细心完成每一项设置，特别是地址备注和支付开通环节。建议你多进入后台实际操作和摸索，熟悉各个功能的位置与用法，为后续的商品上架和店铺运营打下坚实基础。
