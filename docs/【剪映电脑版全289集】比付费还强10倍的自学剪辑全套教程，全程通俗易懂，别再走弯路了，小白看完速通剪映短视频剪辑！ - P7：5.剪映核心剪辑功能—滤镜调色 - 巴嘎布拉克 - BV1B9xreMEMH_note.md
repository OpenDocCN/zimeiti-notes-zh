# 剪映核心剪辑功能：P7：滤镜调色 🎨

在本节课中，我们将要学习剪映的核心剪辑功能之一：滤镜与调色。许多初学者在拍摄视频后，常感到视频色彩不够鲜亮，画面显得单调。实际上，许多视觉效果出色的视频并非拍摄难度高，而是在后期剪辑中巧妙地运用了滤镜和调色。本节课将指导你如何利用剪映的滤镜和调色功能，让你的视频色彩变得鲜亮、富有层次。

## 为什么需要调色？ 🖼️

![](img/e4ad9bb10df1794b46185f2c72f00df1_1.png)

如何快速提升视频的质感？答案是调色。在传统的视频剪辑软件中，例如 Premiere Pro，用户可以通过复杂的调色面板或专业插件进行调色。还有像达芬奇这样的专业调色软件。但对于剪辑新手而言，这些工具学习成本较高。

剪映为初学者提供了便捷的滤镜和调色功能。滤镜是预设好的色彩模板，直接套用即可改变视频风格。调色则允许你对视频进行个性化调整，例如让天空更蓝或让草地更绿。

上一节我们介绍了调色的基本概念，本节中我们来看看如何在剪映中进行实际操作。

## 实战：使用滤镜与调色 ✨

![](img/e4ad9bb10df1794b46185f2c72f00df1_3.png)

以下是使用滤镜和调色功能的具体步骤。

![](img/e4ad9bb10df1794b46185f2c72f00df1_4.png)

首先，开始创作并导入一段素材。我们以此段原素材为例，可以看到画面中的树木不够绿，天空不够蓝。我们的调色目标是让天空更蓝，植物更绿，使画面色彩更鲜亮。

### 1. 应用滤镜

剪映内置了多种滤镜模板，可以快速改变视频风格。

*   在编辑界面右下角，点击 **“滤镜”** 按钮。
*   在滤镜库中选择一个模板，例如 **“鲜亮”**。
*   应用后，可以立即看到视频色彩发生显著变化。可以尝试切换不同滤镜，预览效果。

我们以“鲜亮”滤镜为例，对比原视频和应用后的视频，变化非常明显。

### 2. 进行个性化调色

如果对滤镜效果不完全满意，或需要进行更精细的调整，可以使用“调节”功能。

以下是“调节”功能中的核心参数及其作用：

*   **亮度**：调整画面整体明暗。公式可理解为 `最终亮度 = 原始亮度 + 调整值`。
*   **对比度**：增强亮部与暗部的差异，使画面更立体。
*   **饱和度**：控制色彩鲜艳程度。提高饱和度能让颜色更鲜明。
*   **锐化**：增强画面边缘细节，使影像更清晰。
*   **高光**：调整画面中最亮区域的明暗。
*   **阴影**：调整画面中最暗区域的明暗。
*   **色温**：调整画面冷暖色调。向蓝色调整偏冷，向黄色调整偏暖。
*   **色调**：调整画面整体色彩倾向。

以当前视频为例，在应用“鲜亮”滤镜后，我们可以进一步微调：
1.  略微降低**亮度**。
2.  适当提升**对比度**和**饱和度**，让色彩更浓郁。
3.  微调**锐化**和**高光**，使画面更细腻柔和。
4.  调整**色温**和**色调**，让天空的蓝色和植物的绿色更符合预期。

调整完成后，可以对比原视频与最终效果。可以看到，调整后的视频天空更蓝，树木更绿，整体质感得到了飞跃性提升。

![](img/e4ad9bb10df1794b46185f2c72f00df1_6.png)

需要注意的是，调整对象不同（如人物、美食、风景），需要调整的色彩参数侧重点也不同，但方法是一致的：先尝试滤镜，再通过调节功能进行个性化微调。

## 课程总结 📝

本节课中我们一起学习了剪映的核心剪辑功能——滤镜与调色。剪映的滤镜库和调色面板提供了强大的色彩调整功能。在制作视频时，我们可以根据需求选择合适的滤镜模板，或通过细致的参数调节来提升视频的色彩表现力与丰富度。有效运用这些功能，能显著提升视频的整体质量和视觉质感。
