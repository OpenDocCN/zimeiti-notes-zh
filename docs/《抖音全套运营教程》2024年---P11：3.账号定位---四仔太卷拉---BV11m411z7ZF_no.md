# 《抖音全套运营教程》：P11：3.账号定位 🎯

![](img/de940a07f8fa5856a889c8afbff3e641_0.png)

在本节课中，我们将要学习抖音账号定位的核心方法。一个清晰的定位是账号成功的基石，它能帮助你明确方向、吸引精准粉丝并实现快速成长。

![](img/de940a07f8fa5856a889c8afbff3e641_2.png)

上一节我们介绍了赛道选择，本节中我们来看看如何具体定位你的账号。理解账号定位，需要反思三个核心问题：你是谁，你是干什么的，以及用户为什么要关注你。

## 确定账号类型与细分赛道

首先，我们需要确定要做的账号类型。除了之前章节提到的大类赛道（即抖音的一级垂类），我们还需思考在一级垂类下的细分赛道是什么。

这好比参加运动会，你报的项目是“田径”，这是一级垂类。而你最擅长且具有优势的“百米赛跑”，就是你的二级垂类。通过定位账号的细分类目，可以确定账号的垂直类型。

在抖音中，无论任何行业，只要定位足够清晰，都有成功的机会。因为抖音流量巨大，一旦做起来，增长和裂变的速度会非常快。

## 明确账号主体

其次，我们要确定账号主体是谁。账号主体是账号的拥有人，也是最终呈现给粉丝的主人公，即让粉丝明确知道在看谁的作品。

在抖音商业类账号中，账号主体主要分为两类：
*   **商家账号**：主体是公司。内容应偏商业化，并需不断通过内容拉近与粉丝的距离，避免过于严肃。这既能建立信任感，也能带来好感，更符合抖音的娱乐属性。
*   **达人账号**：主体是个人。首要解决的是信任问题，可以通过IP打造的方式，为账号树立一个积极、正面的人设。

## 明确账号内容与价值

然后，我们需要明确这个账号是做什么的。最直接的方式是通过抖音认证身份（即职业）。达人可认证黄V，公司可认证蓝V。认证后，粉丝能清晰看到认证标签，从而明白你的职业和账号内容方向。同时，认证也能获取平台背书，增加信任感。

在未认证前，也可以通过账号设置向粉丝清晰传达业务信息。以下是可进行统一设置的要素：
*   头像
*   背景图
*   简介
*   视频封面

最后，必须花时间思考：你将为粉丝提供什么样的独特价值？在抖音中，粉丝对博主的价值需求主要分为以下几类：

**1. 知识价值**
抖音中的知识一般分为几种类型：
*   **认知型**：如人生思考、思维方式、读书分享、行业内幕、成功经验、个人提升等。
*   **干货型**：如技能提升、经验总结、操作步骤、冷知识等。
*   **常识普及型**：如生活、健康、产品、科学、历史、地区等常识的解读和普及。

这类账号提供的价值是 **认同感**（如三观认同、观点认同）和 **获得感**（减少知识盲区），进而让粉丝产生自我认同。

**2. 情绪价值**
情绪是影响观众的重要因素。作品中容易引发共鸣的情绪类型包括：
*   **开心愉悦**：如搞笑、夸张表情、感染类内容。
*   **关系共鸣**：如情侣、婆媳、夫妻、家庭、上下级、师生等人物关系。
*   **情绪波动**：由争议性话题、内容反转/反差、反常识开场等激发的情绪。
*   此外，怀旧、emo、震惊、好奇、颠覆三观等也是用户容易产生的情绪。

这类账号提供的价值是 **共鸣**，即利用表情、文字、画面、声音、肢体动作等准确到位地抒发情绪，直接感染观众。

**3. 感觉价值**
博主通过短视频的画面、文字和声音传递信息。其提供的价值还包括：
*   **视觉享受**：画面美感、构图、以及带来的喜欢、治愈、视觉冲击感，或对作者人格魅力的仰慕。
*   **听觉享受**：视频声音带来的节奏感、心跳加速或听觉愉悦。
*   常见于颜值、萌宠、美景、才艺、明星、文化等类目账号。

这类账号提供的价值是 **共情**，即因表情、文字、画面、声音、肢体动作的自然表现，让人产生联想，并在情绪、认知等意识层面与作者产生连接，从而间接引发共鸣。

**4. 体验价值**
抖音让用户看到了不一样的生活方式，体验到了未曾经历的事物。抖音为用户提供的正是体验价值。例如：
*   **美食账号**：美食享用与制作体验。
*   **旅行、运动、角色扮演等账号**：提供各类沉浸式体验。

这类账号提供给粉丝的价值主要是 **好奇心的满足**，以及紧张感、激动、感动、获得感等体验。

**5. 期待价值**
当用户被你的视频内容、观看体验或对作者本人产生好奇时，可能会在看完后意犹未尽，从而点击关注，期待下次更新。许多博主会利用这一点，不将所有内容在一个视频中呈现完，以制造持续吸引力。

本节课中我们一起学习了抖音账号定位的完整框架。你需要依次确定账号的细分赛道、明确账号主体（商家或个人）、并通过认证或设置清晰传达账号内容。最关键的是，必须深入思考你将为粉丝提供何种核心价值——是知识、情绪、感觉、体验还是持续的期待。一个精准、有价值的定位，是你在抖音海洋中脱颖而出的第一步。
