# 电商入门教程：P1：公司业务与学员案例介绍 🏢

在本节课中，我们将了解一家电商公司的核心业务模式，并通过一位新学员的真实案例，展示从零开始学习电商的初步成果。

## 公司业务概述

我们公司的主营业务分为三个部分。

以下是具体的业务方向：

*   **自主运营店铺**：我们自己在电商平台上开设并运营店铺。
*   **线上培训学员**：我们主要通过线上方式指导学员学习电商运营。
*   **客户店铺托管**：我们为一些客户提供店铺代运营服务。

上一节我们介绍了公司的整体业务框架，本节中我们来看看线下学习的实际案例。

## 学员案例分享

我们有一位来自浙江的线下学员。他是一位05后，刚从学校毕业，此前完全没有电商经验，是一名纯新手。

他的学习经历如下：起初，由他的父亲陪同前来我们公司进行实地考察。考察结束后，他决定留下进行线下学习。

以下是关于他学习进度的具体信息：

*   **当前学习阶段**：今天是学习的第三天。
*   **实操启动时间**：从昨天正式开始操作店铺。
*   **初步成果**：在启动实操的昨天，他的店铺已经出了四笔订单。

## 学习渠道说明

如果你也对电商感兴趣，但不知如何入手，可以考虑跟随我们学习。

我们提供两种学习方式：

*   **主要方式**：线上学习。
*   **可选方式**：线下学习。如果你希望线下学习，也欢迎前来，可以与这位学员作伴。

本节课中我们一起学习了电商公司的典型业务构成，并通过一位新手学员的案例，看到了电商学习可以快速取得初步成果。无论选择线上还是线下，系统性的学习是开启电商之路的有效途径。
