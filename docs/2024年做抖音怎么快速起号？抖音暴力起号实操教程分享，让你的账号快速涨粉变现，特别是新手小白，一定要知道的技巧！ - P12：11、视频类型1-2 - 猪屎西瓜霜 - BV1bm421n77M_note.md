# 抖音运营实战：P12：11、视频类型1-2 - 人设与口播详解 🎬

在本节课中，我们将深入探讨抖音账号运营中至关重要的“人设”概念，并详细讲解一种核心的视频制作形式——口播。理解并应用这些知识，是连接内容创作与粉丝增长、实现变现的关键环节。

上一节我们解决了文案问题，本节中我们来看看内容创作的下一个核心环节。许多教程会直接谈论涨粉与变现，但在这两者之间，还有一个不可或缺的桥梁，那就是建立你的“人设”。建立人设并非为了炫耀，而是为了在众多同类创作者中脱颖而出。例如，有十个人都做装修内容，如果其中一位“小王同学”树立了幽默搞笑的人设，观众就更容易记住他，从而忽略其他九位。当观众因为喜欢你这个人而关注你时，他们就成为了你的粉丝。这就是人设的价值。

现在，我们来系统学习这个知识点。人设主要分为两种类型：**强人设**与**弱人设**。区分清楚这两种人设，有助于你精准定位自己的账号方向。

以下是两种人设的具体定义与示例：

![](img/84d3241a80928cd16b2115302bc95da9_1.png)

*   **强人设**
    *   **定义**：以创作者**个人**为绝对核心的切入点。内容与创作者自身的生活、工作、经验或兴趣紧密相关。
    *   **公式**：**视频内容 ∝ 创作者个人 (经验/生活/兴趣)**
    *   **常见领域示例**：
        1.  行业经验分享：装修、办公设计、房产、汽车、餐饮、婚介、法律咨询、电商等。
        2.  兴趣爱好分享：美食、旅行、手工艺、宠物、星座占卜、美妆、护肤、服装穿搭、游戏、摄影等。
    *   **特点**：此类账号大多需要**真人出镜**，因为内容与“人”本身强绑定。

![](img/84d3241a80928cd16b2115302bc95da9_3.png)

*   **弱人设**
    *   **定义**：内容核心**并非**创作者本人，而是为了做好账号而选择的特定内容形式。创作者个人色彩较淡。
    *   **公式**：**视频内容 ⊥ 创作者个人 (工具属性 > 个人属性)**
    *   **常见领域示例**：
        1.  情感语录：搭配音乐的文字图片或视频。
        2.  影视解说：以电影画面为主，配音为辅进行讲解。
    *   **特点**：可能无需真人出镜，或出镜/出声仅为服务内容本身，观众关注的是内容而非创作者个人。

清晰区分强、弱人设是选择后续内容形式的基础。接下来，我们将聚焦于一种非常适合强人设账号，且实操性极强的视频类型。

对于强人设账号，任何与你个人相关的内容都可以成为视频素材。我们首先讲解第一种广泛应用的形式：**讲故事**。讲故事可以分享客户案例、行业经历或人生感悟。故事本身不要求100%真实，可以从各类文案平台寻找灵感素材。而讲故事最直接的方式，就是通过“口播”来实现。

口播，即对着镜头进行讲述。它拥有丰富的分类，能适应不同领域的需求。

以下是口播的主要类型与价值：

*   **故事型口播**：例如讲述一个有趣的客户故事或人生经历。其核心价值在于**趣味性**，让观众觉得“听起来很有意思”。
*   **知识型口播**：例如分享育儿心得、金融知识、装修干货、房产政策等。其核心价值在于提供**收获感**，让观众在几十秒内觉得“学到了东西”。

无论你身处哪个领域，只要涉及知识、经验或观点的输出，口播都是一种高效的选择。它的拍摄门槛较低，通常只需一部手机和一个稳定的支架。

针对同学们常见的顾虑，这里有两个关键解答：

1.  **口才不好能做口播吗？**
    *   **答案**：可以。口播是**提前准备文案稿**，然后对照录制，并非临场发挥。大部分优秀的口播者都依赖于充分的准备。
2.  **普通话不标准能做口播吗？**
    *   **答案**：可以，有时甚至更具优势。如果你的目标客户是同城或有地域属性，使用方言可能更显亲切，效果更好。同时，可以通过添加字幕来确保所有观众都能理解内容。

本节课中我们一起学习了抖音账号运营中“人设”的概念及其分类（强人设与弱人设），并深入探讨了适用于强人设账号的“口播”视频形式。理解人设是吸引精准粉丝的前提，而掌握口播技巧则是高效输出内容的有力工具。在接下来的课程中，我们将继续讲解其他视频类型及具体的拍摄方法。
