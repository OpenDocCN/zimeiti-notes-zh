# 跨境电商亚马逊运营教程：P3：02、跨境电商的现状 🛒

![](img/d0b46f6f442841f84d4b6cb892b23722_0.png)

在本节课中，我们将要学习跨境电商的现状，分析其用户规模、消费习惯以及中国卖家所具备的核心优势。通过对比中美市场的产品价格与成本，我们将理解跨境电商盈利的关键所在。

---

![](img/d0b46f6f442841f84d4b6cb892b23722_2.png)

为了打消大家的顾虑，我们一起来对跨境电商的现状及其未来走势进行一个简单的了解。我们先来了解跨境电商的现状。现阶段跨境电商的用户规模已经超过了20亿。

有超过20亿的用户群体在使用跨境电商服务。这些用户的消费习惯是大家比较关注的一个问题，即外国人的消费习惯。如何去了解呢？其实特别简单。

如何去了解呢？

![](img/d0b46f6f442841f84d4b6cb892b23722_2.png)

我们可以直接打开亚马逊美国站的官网。进入亚马逊美国站官网后，可以在这里随机搜索一些产品，做一些产品对比，看看这些产品在美国卖多少钱、卖得怎么样，同时对比它们在中国卖多少钱。

以下是进行产品对比的方法：

*   以“拖鞋”为例进行搜索。
*   观察搜索结果中拖鞋在美国的售价，例如：34.88美元、39.24美元、35.22美元、19.21美元等。
*   可以看到，拖鞋在亚马逊上的价格大约在十几到三十几美元。

![](img/d0b46f6f442841f84d4b6cb892b23722_4.png)

这个价格感觉贵不贵？可能感觉并不特别贵。但是，现在不要忽略一个问题：所有产品的标价货币是美元。

例如，一双儿童卡通拖鞋在亚马逊的价格是29.98美元。29.98美元等于多少人民币呢？我们在百度上搜索一个工具，叫做“汇率换算计算器”。

![](img/d0b46f6f442841f84d4b6cb892b23722_4.png)

百度搜索汇率换算计算器后，会有一个免费的实时转换工具。这双卡通拖鞋价格是29.98美元。29.98美元大约等于193.43元人民币，将近200元。

大家可以想一下，这样的拖鞋在超市购买需要花多少钱？可能顶多30元，甚至在拼多多9.9元就可以包邮买到。通过对比，我们发现这样的产品在国外可以卖到将近200元。这说明国外消费者的消费水平相对较高。

那么，他们消费水平高的核心原因在哪里？在我们中国，普通人一个月的收入大概是三五千元。但是，一个普通的美国人，一个月的工资也是三五千，不过他们发放的是美元。

![](img/d0b46f6f442841f84d4b6cb892b23722_6.png)

假如一个美国人一个月收入3000美元。3000美元等于多少人民币呢？看一下，3000美元等于人民币19356元。

![](img/d0b46f6f442841f84d4b6cb892b23722_6.png)

![](img/d0b46f6f442841f84d4b6cb892b23722_8.png)

所以，在一个美国人眼里，花20美元买一双拖鞋，感觉就像我们花20元人民币一样。但是这20美元等于人民币将近130元。因为人民币与美元之间存在一个东西，叫做**汇率差**。

这个点需要记住。我们做亚马逊、做跨境电商的主要收益来源就在于赚取这个**汇率差**。

了解了利润来源，可能会有同学问：邮费是不是特别贵？比如那双卡通拖鞋，在国外卖190元，我的进货价按40元算，还有150元利润，那邮费是不是要200元？

接下来，我跟大家讲一下邮费问题。邮费其实没有各位想象那么贵。我打开一个物流报价表给大家简单介绍一下。

![](img/d0b46f6f442841f84d4b6cb892b23722_8.png)

首先聊一下邮费问题。我打开的是云图物流，它在国际物流行业的地位类似于国内的顺丰，价格稍贵一点，但服务相对较好。我们以云图物流为例，计算一双卡通拖鞋从中国邮寄到美国的运费。

在计算运费前，要知道这双拖鞋不会特别重，大概两三百克左右。假如一双拖鞋重量是300克，即0.3千克。

以下是运费计算过程：

*   运费计算公式：`重量（公斤） * 单价（元/公斤）`。
*   假设单价为70元/公斤，那么运费为 `0.3 * 70 = 21` 元。
*   国际物流公司还会收取一个挂号费来处理海关清关等事宜，正常一双拖鞋的挂号费是16元。
*   因此，总费用为 `21 + 16 = 37` 元。

一双拖鞋从中国邮到美国，总费用大约37元。相比于我们的利润，这个费用是能够接受的。

同时要告诉大家，如果走海运，通过货代公司，费用最低可以做到13-14元人民币每公斤。这意味着邮寄三四双拖鞋到美国，总运费也就四五十元，平均下来一双拖鞋的邮费可能才五六元。国际物流费用真没想象那么贵。海运是最便宜的，因为船大装的东西多。当然，走空运价格会贵一点，后期应尽量少选择空运以节省成本。

了解完市场与成本后，我们常说“知己知彼，方能百战不殆”。那么，中国人做跨境电商有什么优势吗？

首先，中国是一个农业比较发达的国家，这要感谢袁隆平老先生。农业发达也预示着轻工业很发达。什么是轻工业呢？像家里常用的锅碗瓢盆、桌椅板凳，都属于轻工业用品。

![](img/d0b46f6f442841f84d4b6cb892b23722_10.png)

中国具备全球最完善的工业体系，我们是全世界第一出口大国和制造大国，全球有30%以上的轻工业产品来自中国制造。“Made in China”这个词大家都听说过。中国在世界上被冠以“世界工厂”的美称。因为中国制造的商品在国际上可以被称之为物美价廉的代表。

为什么这样讲？

![](img/d0b46f6f442841f84d4b6cb892b23722_12.png)

![](img/d0b46f6f442841f84d4b6cb892b23722_14.png)

在亚马逊美国站搜索保温杯，观察其价格。

一个常见的保温杯在亚马逊的价格是25.99美元。25.99美元等于多少人民币呢？算一下，25.99美元约等于167.68元人民币。

![](img/d0b46f6f442841f84d4b6cb892b23722_18.png)

第二款产品：烟灰缸。

![](img/d0b46f6f442841f84d4b6cb892b23722_18.png)

搜索烟灰缸，观察其价格。一种玻璃制的烟灰缸，在国内2元店可能卖2元一个。在亚马逊，两个装的售价是12.99美元。12.99美元等于多少人民币呢？约等于83元人民币，平均一个40多元。

![](img/d0b6f6f442841f84d4b6cb892b23722_20.png)

对比完之后就很清晰了。这类产品在国内价格相对较低，而在国外价格相对较高。原因在哪里？

![](img/d0b46f6f442841f84d4b6cb892b23722_20.png)

第一个原因：中国的人工成本比较低。

为什么说中国的人工成本低？给大家算一笔账：

*   假设一个普通中国人一个月挣3000元人民币。
*   一个月工作26天，日工资为 `3000 / 26 ≈ 115` 元。
*   一天工作8小时，时薪为 `115 / 8 ≈ 14` 元。
*   计算公式：`时薪 = (月薪 / 每月工作天数) / 每日工作小时数`。

一个普通中国人一个小时的工资大约是14元人民币。

同样道理，一个普通美国人一个月也挣3000，但发的是美元。

*   假设一个美国人一个月挣3000美元。
*   同样工作26天，每天8小时。
*   时薪为 `(3000 / 26) / 8 ≈ 14.42` 美元。
*   14.42美元换算成人民币约为90元。

一个美国人每小时的工资换算成人民币约为90元。看到这里就应该明白，为什么90年代那么多人喜欢去美国打工，因为感觉美国工资高；为什么富士康这些工厂喜欢建在中国，因为中国的人工成本低。当然，这只是现状。未来当我们成为发达国家后，这也会有所改观，但在通往发达国家的路上，需要我们每一个中国人一起努力。

第二个原因：原材料价格也低。

中国领土面积大，拥有960万平方公里的领土面积，因此原材料价格也低。人工成本低，原材料价格低，那么我们生产出来的商品在整个国际上就可以被称之为物美价廉的代表。这就是咱们中国人的优势。

第三个原因：做电商这个事，我们比较有经验。

像京东、淘宝、拼多多、抖音直播带货等，各种各样的电商形式，不管是现在、过去还是未来，中国一定会是跨境电商发展的主要推动力量。这就是我们中国人的优势。
