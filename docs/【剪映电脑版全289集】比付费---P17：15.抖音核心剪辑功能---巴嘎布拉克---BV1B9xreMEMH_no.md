# 剪映电脑版全教程：P17：抖音核心剪辑功能—特效 🎬

![](img/672fbe08e50d2684dfe737033d849914_1.png)

在本节课中，我们将要学习剪映软件中一个非常有趣且强大的功能——特效。特效功能借鉴了抖音等短视频平台的拍摄道具，通过在视频中添加各种动态效果，可以让你的视频变得更加生动、有趣，并大幅提升视频的观感和氛围。

上一节我们介绍了其他剪辑功能，本节中我们来看看如何为视频添加特效。

## 特效功能概述

![](img/672fbe08e50d2684dfe737033d849914_3.png)

![](img/672fbe08e50d2684dfe737033d849914_5.png)

剪映的特效功能内置了丰富的特效素材库。在视频中植入合适的特效元素，可以极大地提升视频的整体画面感觉和氛围。尤其是在制作变装类、反转类或美妆类视频时，特效的使用频率非常高。因此，掌握特效的运用，对于提升视频的最终效果至关重要。

## 实战演练：添加特效

![](img/672fbe08e50d2684dfe737033d849914_11.png)

以下是添加特效的具体操作步骤。我们将通过一个实例来演示如何为视频添加多个特效。

![](img/672fbe08e50d2684dfe737033d849914_13.png)

首先，我们观察一个成品视频案例。这个视频运用了多种特效，例如开场时的“模糊变清晰”、舞蹈时的背景“妖气”效果，以及结尾的“横纹制造”抖动效果。

![](img/672fbe08e50d2684dfe737033d849914_15.png)

1.  **导入素材并定位**：在剪映中新建项目，导入你的视频素材。将时间轴移动到需要添加特效的起始位置。
2.  **添加第一个特效（模糊变清晰）**：
    *   点击底部工具栏的 **`特效`** 按钮。
    *   在特效面板中，选择 **`基础`** 分类。
    *   找到 **`变清晰`** 特效，点击即可添加到时间轴。
    *   在时间轴上拖动特效轨道的边缘，可以调整特效的持续时间。
3.  **添加第二个特效（妖气背景）**：
    *   再次点击 **`特效`** 按钮。
    *   选择 **`梦幻`** 分类。
    *   找到 **`妖气`** 特效，点击添加。
    *   同样，在时间轴上调整其位置和时长，使其与舞蹈片段对齐。
4.  **添加第三个特效（横纹制造抖动）**：
    *   继续点击 **`特效`** 按钮。
    *   选择 **`动感`** 分类。
    *   找到 **`横纹制造`** 特效，点击添加。
    *   将其拖动到甩头发的片段位置，并调整时长。
5.  **特效叠加与调整**：剪映支持多个特效叠加使用。你可以通过反复点击“新增特效”来尝试不同效果的组合。在时间轴上，你可以自由拖动、裁剪每个特效片段，以精确控制其出现和结束的时机。

![](img/672fbe08e50d2684dfe737033d849914_17.png)

## 常用特效分类简介

![](img/672fbe08e50d2684dfe737033d849914_19.png)

剪映的特效库种类繁多，以下是一些常用且效果突出的分类，建议在创作中多加尝试：

*   **基础类**：包含 **`开幕`**、**`闭幕`**、**`变清晰`**、**`变模糊`**、**`放大`**、**`抖动`** 等，常用于视频转场或强调某个瞬间。
*   **梦幻类**：如 **`妖气`**、**`星火`**、**`金粉`** 等，能为视频增添唯美或神秘的氛围。
*   **动感类**：如 **`横纹制造`**、**`闪烁`**、**`冲击波`** 等，适合用于需要强节奏感和视觉冲击力的片段。
*   **复古类**：如 **`录像机`**、**`噪点`**、**`相机方格`** 等，可以轻松营造怀旧或电影感的风格。

## 总结与建议

![](img/672fbe08e50d2684dfe737033d849914_21.png)

本节课中我们一起学习了剪映核心剪辑功能之一的“特效”。我们了解了特效的作用，并通过一步步实操，掌握了如何为视频添加、调整和叠加多种特效。

特效是提升视频表现力的利器。剪映提供了海量的特效素材，建议你在后期的视频创作中，根据内容主题大胆尝试和组合不同的特效。持续的练习与尝试，将帮助你更快地掌握特效使用的精髓，创作出更具吸引力和专业感的短视频作品。
