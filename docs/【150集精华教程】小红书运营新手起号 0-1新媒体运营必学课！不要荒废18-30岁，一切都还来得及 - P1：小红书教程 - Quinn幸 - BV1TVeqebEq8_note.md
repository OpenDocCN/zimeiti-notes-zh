## 开店 📝

![](img/2db3f5db7258c723b4296f298cf4fd46_3.png)

上一节我们介绍了课程概述，本节中我们来看看如何开设小红书店铺。开店是整个流程的第一步，操作非常简单。

首先，打开小红书应用，点击左上角的菜单图标。

![](img/2db3f5db7258c723b4296f298cf4fd46_5.png)

然后，选择“创作中心”，点击“更多服务”选项，找到并选择“店铺”，最后点击“立即入住”即可。按照官方要求提交资料完成入住。

![](img/2db3f5db7258c723b4296f298cf4fd46_5.png)

以下是小红书店铺的两种主要形式：

*   **个人店**：分为仅需身份证和1000元保证金的个人店，以及需要额外营业执照的个体工商户店。
*   **企业店**：包括普通企业店、专卖店和旗舰店，均需要营业执照并缴纳600元年费。

对于新手，建议直接开通个人店。实测表明，平台并未对企业店有流量倾斜。个人店只能开一家，而一份营业执照可开三家店。但初期专注于做好一个店铺更为重要。

小红书电商不适合以“店群”模式操作，因为其电商自然流量占比很低。铺再多的店铺和商品效果有限。正确的做法是建立“账号矩阵”，这将在后续流量部分详细说明。

开店的成本主要包括以下几项：

![](img/2db3f5db7258c723b4296f298cf4fd46_7.png)

1.  **保证金**：1000元（不做可退）。
2.  **平台扣点**：营业额的5%。
3.  **支付渠道费**：每单约0.7%。
4.  **流动资金**：初期可采用无货源模式，准备2000-3000元即可启动。

## 选品 🛍️

完成开店后，下一步就是决定卖什么。选品逻辑的核心是利用市场数据，而非凭空想象。

我们应该在其他平台的爆款商品中，寻找符合小红书平台调性的产品进行测试。以下是三个推荐的选品渠道：

1.  **某东、某宝**：寻找具有小红书调性的爆款产品。有电商朋友将其淘宝爆款搬到小红书，第一篇笔记就出单，第二篇笔记单量破百。
2.  **某音精选联盟**：查看精选联盟排行榜，了解现阶段热卖商品。将符合小红书用户调性的产品直接搬运测试。
3.  **某多多销量排行榜**：选择适合小红书的类目排行榜，从中挑选爆品进行搬运。

整个选品逻辑是借助大卖家的选品能力来弥补自身不足。只需判断排行榜上的爆品是否可能匹配小红书用户，如果匹配，就搬运过来测试。这是最快捷且稳妥的选品方式，成功率远高于盲目测试。

## 获取流量 📈

产品上架后，就进入了最重要的环节——获取流量。小红书是种草平台，店铺九成以上的流量都来自于发布的笔记。因此，不应铺“店群”，而应做“账号矩阵”，通过多发笔记来获取流量。

用户通过你的笔记对产品产生兴趣，然后进入店铺完成购买。多发笔记的效果远优于堆砌多个店铺。

发笔记的方式是小红书电商成功的关键。直接发布硬广效果很差，因为小红书用户认知较高，难以获得信任和转化。

![](img/2db3f5db7258c723b4296f298cf4fd46_9.png)

发笔记时，必须从素人（即普通用户）的角度切入，做种草内容，而非官方视角。

![](img/2db3f5db7258c723b4296f298cf4fd46_11.png)

制作这种种草笔记的方法很简单：去各大电商平台（如某音、某宝、某东）的商品评论区和买家秀中，寻找好看的素材直接使用。

![](img/2db3f5db7258c723b4296f298cf4fd46_13.png)

![](img/2db3f5db7258c723b4296f298cf4fd46_11.png)

文案则基于买家评价进行修改。这些评价是用户的真实体验，天然适合用作种草笔记，且素材取之不尽。

![](img/2db3f5db7258c723b4296f298cf4fd46_13.png)

当单个账号的运营流程（SOP）跑通后，就可以开始通过“账号矩阵”来放大流量。即批量运营小红书账号，批量发布笔记为店铺引流。
