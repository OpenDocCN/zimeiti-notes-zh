# 抖音运营新手起号：P13：变现篇丨抖音沙雕动画制作玩法 🎬

在本节课中，我们将学习一种被称为“沙雕动画”的视频制作方法。这是一种操作简单、无需复杂技术或原创剧本，却能有效吸引流量的内容形式。我们将分步讲解如何利用一部手机完成制作，并介绍一种高效的“懒人”配音方法，帮助你快速起步并实现变现。

## 制作工具与基础设置 🛠️

上一节我们概述了沙雕动画的特点，本节中我们来看看具体的制作工具和初始设置。

![](img/8c86dd248160b5338f1b64844773ecd7_1.png)

这种动画的制作过程并不复杂。你只需要一部手机和一个特定软件即可开始。首先，打开制作软件，进行基础的项目设置。

以下是初始设置的关键步骤：
*   **选择比例**：根据发布平台选择视频比例。发布抖音选择 **9:16**，发布西瓜视频则选择 **16:9**。
*   **设置分辨率**：将分辨率设置为 **1080P**，以保证视频清晰度。
*   **调整帧率**：保持软件默认的帧率设置即可，无需改动。

完成这些基础设置后，我们就可以进入场景和角色的选择阶段了。

## 场景与角色选择 🏞️👤

设置好项目参数后，接下来要为动画搭建舞台和选择演员。

![](img/8c86dd248160b5338f1b64844773ecd7_5.png)

软件的场景库提供了丰富的背景选项。例如，你可以选择一个“古代”风格的分类，然后向右滑动浏览该分类下的所有具体场景。角色的选择方式与此类似，软件的角色库中有多种分类，包含沙雕、武侠、斗罗大陆、奥特曼等多种风格的角色模型，非常丰富。

![](img/8c86dd248160b5338f1b64844773ecd7_5.png)

## 动画制作与合成 🎞️

选好场景和角色后，我们就可以开始让静态的画面动起来了。

其操作逻辑与手机版剪映类似，非常简单直观。点击场景中的角色，选择“添加动作”，然后点击播放预览，人物就会按照设定的动作运动。如果你对当前动作不满意，可以随时更换。你还可以在场景中添加更多角色，以丰富故事情节。

最后，导入准备好的音频文件，软件即可合成出一段完整的动画视频。

## “懒人”配音与流量获取策略 🎤

制作出动画画面后，配音是赋予视频灵魂的关键。下面介绍一种无需自己编写段子的高效方法。

这个方法的核心是“拍同款”。你可以寻找平台上粉丝量较多的搞笑博主，直接使用他们视频的“原声”来为你的沙雕动画配音。

采用这种方法主要有以下三个好处：
*   **保证原创度**：画面是你自己制作的动画，与别人的音频结合后，整体作品的原创度依然很高，不易被平台限制流量，从而保障收益。
*   **蹭取流量**：原声视频的粉丝可能会通过“原声”入口，浏览所有使用该声音的作品，这为你的视频带来了额外的曝光机会。
*   **提升效率**：省去了构思剧本和录制配音的时间，可以更专注于动画制作，从而有能力运营多个账号，增加总收益。

有时，你缺少的并非资源，而是发现有效资源的方法。
