# 剪映核心剪辑教程：P9：7.抖音常用素材片段 📁

在本节课中，我们将学习剪映的核心剪辑功能之一：如何使用剪映自带的“素材库”功能。这个功能内置了大量常用的视频片段，能帮助初学者快速找到并应用素材，从而节省寻找素材的时间，提高视频制作效率。

上一节我们介绍了其他剪辑功能，本节中我们来看看如何利用剪映内置的素材库来丰富你的视频内容。

## 素材库的位置与作用

许多新手在制作视频时，常常苦恼于素材的来源和使用方法。实际上，剪映App自带了大量实用的视频素材，例如黑屏、白屏、雪花故障效果等。这些素材在制作沙雕搞笑类段子或日常VLOG时非常有用，可以直接套用，极大地便利了新手，减少了不必要的时间浪费。

那么，剪映的素材库在哪里呢？我们一起来看一下。

![](img/9d2a6cb49cfe56d6c9220c8293fabc88_3.png)

## 素材库的内容与使用

这些内置素材正是我们前面提到的，例如雪花效果。

![](img/9d2a6cb49cfe56d6c9220c8293fabc88_11.png)

在制作日常或沙雕视频时经常会用到。素材库还提供了许多绿幕素材，例如小鱼、飞机等，方便进行抠像合成。

我们可以预览这些素材，它们都是剪映自带的一些特效或视频片段。

## 核心操作总结

![](img/9d2a6cb49cfe56d6c9220c8293fabc88_21.png)

使用素材库的核心操作非常简单：
1.  点击底部工具栏的 `开始创作`。
2.  在导入界面选择右侧的 `素材库` 选项卡。
3.  浏览或搜索需要的素材类别。
4.  点击素材预览，并点击 `+` 号添加到时间线。

这种方式非常方便，各位新手无需再费力寻找其他素材，可以直接在这里调用。
