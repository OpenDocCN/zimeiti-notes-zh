# 抖音运营入门：P10：1、抖音短视频的前景 📈

![](img/fcab1b78b257594b483ba432ed7a6629_1.png)

在本节课中，我们将要学习抖音短视频平台的发展前景，分析其市场潜力与用户基础，并理解为什么它适合新手创作者起步。

![](img/fcab1b78b257594b483ba432ed7a6629_3.png)

了解抖音短视频的前景，需要先通过一张图片来认识。

![](img/fcab1b78b257594b483ba432ed7a6629_1.png)

这张图片是抖音创作者大会的一份数据报告。报告显示，有超过2200万人在抖音的合计收入超过了417亿元。抖音希望将这个数字翻一番，让创作者的总收入达到800亿元。

![](img/fcab1b78b257594b483ba432ed7a6629_3.png)

这代表了巨大的流量扶持。同时，这里介绍第一个知识点：抖音上有两类人。第一类人叫做“玩家”，他们单纯地玩抖音，没想过从中赚钱。另一类人，可以称作“创作者”，他们想通过制作抖音内容来获取收益。区分这两类人很重要，后续还会提到。

那么，为什么我们要选择抖音短视频呢？市面上有快手、火山等短视频平台，为什么非得做抖音？这就像开实体店选址，你肯定会找一个流量大、适合开店的地方。做短视频也一样，必须选择一个流量大的平台。如果平台自身流量小，就无法分给创作者，我们也赚不到钱。

抖音的日活跃用户达到了8亿以上，月活跃用户更是达到了40亿人次以上。这是非常庞大的流量基础。

流量大，竞争压力会不会很大？答案是不会。抖音上的创作者大约只有5000万。这里的“创作者”并非专指网红达人。例如，邻居家的老太太用手机拍下自己做的饭传到抖音上，她也算创作者。因此，你90%的竞争压力，实际上来自于身边的普通用户。

接着，做抖音短视频需要购买电脑或摄像机吗？不需要。只需要一部智能手机和一个手机号，就可以注册抖音账号。抖音倡导全民通过短视频的形式分享和记录生活，门槛很低。

那么，当前有什么样的风口值得我们去做呢？首先是“抖音计划”。该计划表示，抖音将帮助1000名优质的艺术创作者涨粉超过100万。有的同学可能会说，自己不会创作。这不用担心，如果你什么都会，就不需要来听课了。正是因为不会，才需要学习。

市场空间充足，行业也容易产出内容。现在是大数据时代。大数据时代的特点是，平台会根据你的喜好推荐内容。你喜欢看什么，抖音就给你推荐什么。因此，每个人看到的内容都可能比较片面。

以下是不同人群可以创作的内容方向举例：
*   如果你是宝妈，可以拍摄宝宝或自己做的饭菜。
*   如果你是实体店主，可以拍摄产品进行引流。
*   如果你是普通上班族，可以分享日常生活。

为什么说这些内容都可行？因为面对月活跃40亿的用户群体，只要你的内容不触犯法律和道德红线，不是低俗内容，总有一部分人会喜欢上你。一定会有人喜欢你的内容。

那么，为什么你觉得只有几十个粉丝呢？是因为喜欢你内容的那部分人，还没有刷到过你的作品。他们无法看到，自然无法成为你的粉丝。

所以，对于新手小白来说，大家都想涨粉，心情迫切可以理解。但说句实在话，不要把涨粉作为第一个目标。我们的第一个目标应该是：**先把流量（播放量）做起来**。播放量涨起来之后，粉丝自然就来了。

本节课中我们一起学习了抖音平台的巨大前景、用户分类、低门槛特性以及适合新手的创作方向。核心在于理解平台流量基础庞大，竞争并非想象中激烈，并且起步的关键是优先提升内容播放量，而非单纯追求粉丝数。
