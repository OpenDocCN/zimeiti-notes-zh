# 小红书开店运营教程：P7：06. 小红书后台的基础操作（电脑端） 💻

在本节课中，我们将学习如何通过电脑端操作小红书商家后台，重点掌握保证金、支付方式、运费保和地址设置等核心功能。这些是店铺正常运营和提升转化率的基础。

![](img/76b0ae16f38aadb2673fb7adedf24320_1.png)

上一节我们介绍了手机端的操作，本节中我们来看看电脑端后台的具体设置。

![](img/76b0ae16f38aadb2673fb7adedf24320_3.png)

## 一、 保证金管理 💰

保证金是开店的基础环节。前期店铺可以暂不缴纳保证金，但若想长期运营或开通更多支付方式，则建议缴纳。

保证金的金额因店铺类型和经营类目而异。以下是不同店铺类型的大致保证金范围：
*   **个人店**：约1000元人民币。
*   **个体店**：约1000元人民币（部分特殊类目如二手奢侈品、珠宝玉石等个人店无法开通，需升级为个体店或企业店）。
*   **企业店**：2000元至20000元人民币不等。

![](img/76b0ae16f38aadb2673fb7adedf24320_5.png)

**操作路径**：登录小红书千帆后台 -> 点击左侧「资金」 -> 选择「店铺保证金管理」。

![](img/76b0ae16f38aadb2673fb7adedf24320_7.png)

在该页面，你可以进行充值、查询待补缴金额或申请退款。保证金在店铺关闭且无违规纠纷的情况下可以全额退还。平台扣除保证金主要用于处理店铺未履行的客户售后纠纷。

![](img/76b0ae16f38aadb2673fb7adedf24320_9.png)

## 二、 支付方式开通 💳

![](img/76b0ae16f38aadb2673fb7adedf24320_11.png)

开通多元支付方式能方便更多消费者下单，对提升订单转化率有帮助。主要支持**支付宝**和**微信支付**两种方式。

**重要提示**：开通微信支付功能，**必须先缴纳店铺保证金**。否则，买家无法使用微信付款。

![](img/76b0ae16f38aadb2673fb7adedf24320_13.png)

**操作路径**：登录小红书千帆后台 -> 点击左侧「资金」 -> 选择「支付设置」。

以下是两种支付方式的开通要点：
*   **支付宝**：提交资料后，通常可以当天完成开通。
*   **微信支付**：缴纳保证金后，需等待至**第二天**才能进行设置绑定。提现金额将进入店铺注册时使用的法人账户。

![](img/76b0ae16f38aadb2673fb7adedf24320_15.png)

![](img/76b0ae16f38aadb2673fb7adedf24320_17.png)

## 三、 运费保设置 🛡️

运费保是一项推荐开通的服务，它能有效降低消费者的下单顾虑，从而提升订单转化率。

运费保的作用是：当消费者发起“七天无理由退货”时，系统会自动补贴退货运费。这能减少因运费问题产生的纠纷，优化购物体验。

**操作路径**：登录小红书千帆后台 -> 点击左侧「售后」 -> 选择「运费保」。

![](img/76b0ae16f38aadb2673fb7adedf24320_19.png)

在运费保页面，点击“开通全店运费保服务”即可启用该功能。

![](img/76b0ae16f38aadb2673fb7adedf24320_21.png)

## 四、 地址与物流设置 📦

![](img/76b0ae16f38aadb2673fb7adedf24320_23.png)

正确的地址设置对于订单处理和售后至关重要，尤其对于一件代发（无货源）模式的商家。

![](img/76b0ae16f38aadb2673fb7adedf24320_25.png)

**操作路径**：登录小红书千帆后台 -> 点击左侧「订单」 -> 选择「物流工具」。

![](img/76b0ae16f38aadb2673fb7adedf24320_27.png)

在物流工具中，你需要完成以下设置：

![](img/76b0ae16f38aadb2673fb7adedf24320_29.png)

1.  **新建退货地址**：点击「新建地址」。
    *   **建议填写方式**：地址可填写供应商所在地或一个偏远地区（如新疆），并在地址末尾备注“**退货前请联系客服**”。这是为了避免消费者直接按此地址退货，导致你无法及时处理。
    *   **有货源商家**：直接填写自己的真实仓库地址即可。

2.  **设置运费模板**：根据你的商品发货地、是否包邮等规则，设置相应的运费模板，并在发布商品时应用。

![](img/76b0ae16f38aadb2673fb7adedf24320_31.png)

本节课中我们一起学习了小红书电脑端后台的四个核心设置：保证金的缴纳与管理、支付方式的开通、运费保服务的优势与开通，以及地址和运费模板的设置。熟练掌握这些操作，能为你的店铺打下坚实的运营基础，并有效提升客户体验与订单转化率。
