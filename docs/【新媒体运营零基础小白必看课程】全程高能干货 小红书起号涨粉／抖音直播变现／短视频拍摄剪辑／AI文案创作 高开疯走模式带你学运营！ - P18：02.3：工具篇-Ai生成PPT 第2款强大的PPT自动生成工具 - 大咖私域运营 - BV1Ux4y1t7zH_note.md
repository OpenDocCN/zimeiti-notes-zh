# 新媒体运营零基础小白必看课程：P18：02.3：工具篇-AI生成PPT 🛠️

## 概述
在本节课中，我们将学习第二款强大的AI生成PPT工具。这款工具在生成速度、稳定性和模板丰富度方面具有优势，能与上一节课分享的工具形成互补，共同提升我们制作演示文稿的效率。

上一节我们介绍了第一款AI生成PPT工具，本节中我们来看看另一款同样高效的工具。

## 工具核心优势与使用前提
这款工具的主要优势在于生成速度快、运行稳定，并且提供了数量庞大的模板库，方便我们进行样式选择和细节调整。

要充分发挥其效能，有一个重要前提：我们需要准备一份内容详细、结构清晰的大纲或文档。工具会基于我们提供的内容进行分析，从而生成质量更高的PPT模板。

## 两种生成方式
该工具主要提供两种生成PPT的方式：
1.  **AI智能生成**：直接输入主题，由AI自动构思大纲并生成内容。
2.  **导入本地大纲**：导入我们预先准备好的详细文档或大纲，由工具基于此生成PPT。

我个人更常使用“导入本地大纲”的方式，因为可控性更强。

## 实践操作：导入本地大纲生成
以下是使用“导入本地大纲”功能的具体步骤。

![](img/23fb2dcb7890de21d6c1b02d1cc4926e_3.png)

![](img/23fb2dcb7890de21d6c1b02d1cc4926e_5.png)

首先，我们需要准备一份详细的大纲。我们可以借助ChatGPT等AI助手来辅助完成。例如，我们可以向ChatGPT发出如下指令：
```plaintext
请以“装修行业在小红书平台的运营推广计划”为主题，撰写一份详细的PPT大纲，并完善每一个小节点的具体内容信息，保持好结构和格式排版。
```
获得生成的大纲文档后，即可进入工具操作。

![](img/23fb2dcb7890de21d6c1b02d1cc4926e_6.png)

第一步，在工具界面选择“导入本地大纲”功能，并上传我们准备好的文档文件。

第二步，工具会快速解析文档结构，提取出大纲。接着，我们需要从丰富的模板库中，挑选一个与内容主题（如装修行业）风格相符的模板。

第三步，点击“生成PPT”，工具便会基于我们提供的内容和选择的模板，快速合成一份初步的演示文稿。

生成完成后，我们进入编辑界面。如果发现某些页面的内容有缺失或排版不理想，可以进行手动调整。

*   **编辑内容**：在右侧的“大纲编辑”区域，可以补充或修改具体内容。
*   **调整排版**：点击页面中的元素，可以使用“文字设置”、“形状设置”等功能进行调整。
*   **更换版式**：如果对某一页的布局不满意，可以单击该页面，从工具提供的多种备选版式中一键更换。
*   **整体换肤**：如果对整套模板的风格不满意，还可以使用“模板替换”功能，为全部页面快速更换另一套设计模板，且内容会自动适配。

## 实践操作：AI智能生成
接下来，我们尝试“AI智能生成”方式。

第一步，在工具界面选择“AI智能生成”功能。

第二步，在输入框中直接键入PPT主题，例如“装修行业在小红书平台的运营推广计划”，然后点击生成。

第三步，工具会先自动构思并生成一份详细的大纲。我们确认后，同样需要从模板库中挑选一个合适的风格。

第四步，点击“生成PPT”，工具将根据主题自动生成完整的文案内容，并套用模板形成PPT。之后，我们同样可以进入编辑界面，对内容、排版和模板进行精细化的调整。
