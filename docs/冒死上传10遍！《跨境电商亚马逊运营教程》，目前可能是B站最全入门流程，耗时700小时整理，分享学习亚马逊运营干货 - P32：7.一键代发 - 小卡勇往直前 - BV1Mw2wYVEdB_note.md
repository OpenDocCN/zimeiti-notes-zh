## 概述
在本节课中，我们将要学习亚马逊运营中的“一键代发”模式。这是一种特别适合个人卖家或新手的运营方式，可以降低启动成本和库存风险。我们将介绍一个实用的工具网站，并详细讲解如何利用它进行选品和操作。

上一节我们介绍了平台站内选品和二次筛选法。本节中我们来看看一种更轻量级的运营模式——一键代发。

![](img/3f7836a4f9a5bbb52a5d1ee2129ff78e_12.png)

很多同学运营的是个人店铺。个人店铺采用自发货模式是完全可行的。这里给大家介绍一个实用的网站。

这个网站叫做“越狱大卖网”。FBA是亚马逊物流服务的简称，类似于京东自营的仓库体系，是亚马逊官方运营的海外仓统称。

以下是该网站的一个功能演示：
*   该网站提供了一个工具箱，内含“销量预测”功能。
*   我们可以输入产品的销售排名（例如，服饰鞋子类目下的第174名），系统会预估出其月销量（例如，7400多双）。
*   产品排名依据实时销量每日动态变化。

![](img/3f7836a4f9a5bbb52a5d1ee2129ff78e_14.png)

对于新卖家而言，如果尚未接触过这个行业，可以尝试从自发货模式开始。自发货相比FBA优势较少，但作为起步是可行的选择。

这里推荐的“越狱大卖网”对自发货卖家很有帮助。我们进入网站的亚马逊专区进行查看。

以下是该平台产品的特点：
*   产品页面信息完整，包含图片、商品描述、五点特性、商品属性及标题。
*   其仓库位于美国洛杉矶，属于海外仓。
*   平台上标注的价格（如110元）已包含物流费用，卖家支付此费用后，平台会直接将货品送达买家手中。
*   平台承诺一般在两天内发货。

![](img/3f7836a4f9a5bbb52a5d1ee2129ff78e_16.png)

如果你不想直接采购该平台的产品，也可以利用其信息。有些同学英语能力可能不足，而该平台已准备好完整的商品信息。

![](img/3f7836a4f9a5bbb52a5d1ee2129ff78e_18.png)

该平台的产品通常由经验丰富的亚马逊运营定期更新，例如会提前准备季节性产品（如圣诞节商品）。虽然价格可能较高，但其从美国海外仓直接发货，物流速度很快。

不一定非要局限于在1688上寻找货源。这个平台为人民币结算（例如435元），对做自发货的同学来说操作非常方便。

需要明确的是，并非强制使用该平台。我们的建议是：做自发货的同学可以参考该平台的产品信息。

以下是具体的操作思路：
1.  将该平台上的产品图片和描述下载下来。
2.  将这些信息原样上架到自己的亚马逊店铺。
3.  出单后，去该平台下单并支付货款。
4.  平台会直接帮你将货物发给亚马逊的买家。

这种行为就是“一件代发”。有些同学一开始不想做FBA，那么一件代发是较为合适的选择。该平台上的产品通常是普通商品，基本不存在侵权风险。

我们可以简单计算一下售价。例如，一个成本400元的产品，想要30%的利润，则初步定价为：
`400 * 1.3 = 520元`
亚马逊平台佣金约为15%，因此最终售价需为：
`520 / 0.85 ≈ 611.76元`
换算成美元，大约定价为88美金。关于定价的详细方法，后续课程会进一步讲解。

![](img/3f7836a4f9a5bbb52a5d1ee2129ff78e_20.png)

一件代发是自发货（FBM）的一种形式，并非必须自己发货。FBM卖家可以利用这种模式，由供应商直接发货给买家，从美国仓发货通常只需三四天即可送达。

## 总结
本节课中我们一起学习了亚马逊的“一键代发”（一件代发）模式。我们了解了一个实用的工具网站“越狱大卖网”，并掌握了如何利用该平台的信息进行选品、上架以及由供应商直接发货的操作流程。这种模式降低了新手卖家的启动门槛和资金压力，是开始亚马逊自发货（FBM）业务的一个有效途径。
