# AI新媒体运营：P15：01.10.03：🤖 训练AI模拟爆款笔记到产出爆款笔记的过程

在本节课中，我们将学习如何通过分析平台上的爆款笔记，训练AI理解其成功要素，并最终指导AI生成具备爆款潜力的新笔记。这是一个从“喂资料”到“出成品”的系统化过程。

上一节我们介绍了通过“喂资料”的方式，让ChatGPT学习特定平台的用语和内容风格。虽然上一节以旅游攻略为例的成果可能不尽理想，但方法是正确的。核心在于，如果你能向AI提供你所在赛道的优质内容，让它透彻分析并总结规律，AI就能学习到该平台的文案写作方式和风格。

## 分析爆款笔记的核心要点

![](img/61b01d98fd9e1464585f036abc06f562_9.png)

![](img/61b01d98fd9e1464585f036abc06f562_11.png)

![](img/61b01d98fd9e1464585f036abc06f562_12.png)

![](img/61b01d98fd9e1464585f036abc06f562_16.png)

要让AI学会创作爆款，首先需要教会它识别爆款。这不仅仅是看内容，更是分析其内在结构、情感策略和关键词布局。以下是需要引导AI重点分析的几个维度：

1.  **爆款因素分析**：分析内容为何能成为爆款。是触动了用户情感，提供了稀缺信息，还是讲述了逆袭故事？
2.  **整体结构拆解**：将笔记解构为“开头-中间-结尾”。
    *   **开篇**：用什么方式（如提问、制造焦虑、引用名人）吸引用户继续阅读。
    *   **中间**：提供了哪些具体内容来戳中用户痛点，满足需求，从而提升用户停留时长。
    *   **结尾**：设置了什么“钩子”（如鼓励互动、提供联系方式、引导私信）来促成用户行动。
3.  **关键词策略分析**：检查笔记中是否植入了特定关键词，并分析其出现频率和分布方式。这有助于理解平台的搜索和推荐机制。

### 第一步：收集与提供爆款笔记

首先，我们需要找到该领域内数据表现好（如高点赞、高收藏）的笔记，并将文案内容提供给AI。我们选取了几篇考研类爆款笔记，内容涉及备考经验、时间规划、心态调整等。

### 第二步：指令AI进行分析

![](img/61b01d98fd9e1464585f036abc06f562_32.png)

![](img/61b01d98fd9e1464585f036abc06f562_34.png)

![](img/61b01d98fd9e1464585f036abc06f562_36.png)

我们向ChatGPT（此处以3.5版本为例）发出明确的指令，要求它根据上述要点进行分析。指令模板如下：

![](img/61b01d98fd9e1464585f036abc06f562_38.png)

![](img/61b01d98fd9e1464585f036abc06f562_40.png)

```
我给你提供一篇小红书平台上关于考研的爆款笔记文案，请你分析：
1.  它的爆款因素有哪些？（例如：情感共鸣、实用建议、逆袭故事等）
2.  它的整体结构是什么？
    - 开篇用什么吸引了用户？
    - 中间用什么内容戳中了用户痛点？
    - 结尾用什么方式促使用户互动或关注？
3.  分析该笔记是否有结合中文分词法设计关键词。如果有，指出是哪个关键词，以及它的出现频率。
```

随后，我们将多篇考研爆款笔记的文案依次输入。AI给出了分析结果，例如：
*   **爆款因素**：情感共鸣（开头制造焦虑）、逆袭故事、提供详细备考计划、时间节点明确、正能量鼓励、使用表情符号增加亲和力。
*   **结构分析**：开头用自身困境引发共鸣；中间分科目给出具体方法和资源；结尾鼓励并引导互动。
*   **关键词分析**：多数笔记没有刻意设计强关键词，词汇以常用词为主。

### 第三步：整合分析结果并生成大纲

在AI分析了多篇笔记后，我们需要让它将分析出的爆款因素进行去重和整合，形成一个通用的“爆款要素清单”。

![](img/61b01d98fd9e1464585f036abc06f562_53.png)

![](img/61b01d98fd9e1464585f036abc06f562_55.png)

接着，我们指令AI基于这个清单，为我们规划一篇新的考研笔记大纲。指令如下：

```
请结合你前面分析的爆款笔记因素，为我规划一篇关于考研的小红书笔记大纲。大纲需要细分到具体的小节点。在我确认大纲后，再生成完整文案。
```

![](img/61b01d98fd9e1464585f036abc06f562_57.png)

AI生成的大纲示例如下：
*   1. 引人入胜的开头（引入逆袭小故事，引发共鸣）
*   2. 主题介绍（明确笔记价值）
*   3. 专业选择建议
*   4. 详细时间规划（分月/阶段）
*   5. 分科目备考攻略（政治、英语、数学、专业课）
*   6. 时间管理与心态调整方法
*   7. 结尾互动引导（鼓励提问、关注）

### 第四步：根据大纲生成完整文案

大纲确认后，我们给出最终生成指令，要求AI严格按照爆款结构和要素进行创作：

![](img/61b01d98fd9e1464585f036abc06f562_59.png)

![](img/61b01d98fd9e1464585f036abc06f562_60.png)

![](img/61b01d98fd9e1464585f036abc06f562_62.png)

```
请按照上述大纲，撰写具体的文案。要求：
1.  文案结构严格遵循爆款笔记因素。
2.  包含一个简单的逆袭小故事引入。
3.  使用适量的emoji表情。
4.  尝试在文案中自然融入“考研”、“备考”等关键词。
```

AI生成的文案示例开头：
> “当你曾经迷茫，觉得时间不够用，考研上岸遥不可及……我理解那种感觉！我也亲身经历了整个逆袭过程。让我告诉你，只要你下定决心，制定合理计划，梦想绝非遥不可及！💪”
>
> **【明确你的备考时间线】**
> 2023.11-2024.2：基础阶段（英语单词、专业课一轮）...
> （后续为分科目具体建议和鼓励性结尾）

这篇生成的文案在结构、情感点和实用性上已经具备了爆款笔记的雏形，对于运营新手来说是一个合格的起点。当然，还可以进一步优化，使其更精炼、更具个人特色。

## 进阶提示与总结

本节课中我们一起学习了从分析爆款到生成笔记的完整AI训练流程。有几个关键点值得总结：

1.  **指令的明确性至关重要**：AI的理解依赖于清晰的指令。你需要明确告诉它分析什么、如何分析、最终产出什么。
2.  **提供足够多的样本**：分析的爆款笔记越多，AI总结的规律可能越具普适性。
3.  **迭代与优化**：生成的第一版文案通常只是起点。你可以要求AI基于你的反馈进行多次修改，调整语气、增加细节或改变结构。
4.  **利用高级工具**：ChatGPT 4.0或自定义GPTs（GPT商店中的定制助手）在理解复杂指令、保持长上下文记忆方面表现更佳。你可以创建一个专属的“小红书文案助手”，预先植入平台规则、偏好表情和你的内容风格，从而获得更高质量的产出。
