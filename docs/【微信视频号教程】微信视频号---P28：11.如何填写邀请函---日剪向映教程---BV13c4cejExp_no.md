# 微信视频号教程：P28：11.如何填写邀请函

在本节课中，我们将学习如何正确填写视频号创作者计划的邀请函。这是加入计划、获取官方素材和收益权限的关键一步。我们将一步步拆解填写流程，确保信息准确无误。

## 概述

![](img/bd893946fd15e9247889d55a9386efc1_1.png)

填写邀请函是加入视频号创作者计划前的必要操作。本节将详细讲解如何找到并填写账号昵称、粉丝量、邀请人、实名信息及结算手机号这五项核心内容，并最终完成提交。

---

## 查找视频号账号信息

![](img/bd893946fd15e9247889d55a9386efc1_3.png)

![](img/bd893946fd15e9247889d55a9386efc1_4.png)

上一节我们介绍了如何注册视频号，本节中我们来看看如何找到自己的账号昵称和粉丝数量。这是填写邀请函的第一步。

以下是查找账号信息的详细步骤：

1.  **打开微信应用**：在手机上找到并点击微信图标，启动微信应用。

2.  **进入“发现”页面**：在微信底部导航栏中，点击“发现”选项。

![](img/bd893946fd15e9247889d55a9386efc1_6.png)

3.  **找到“视频号”入口**：在“发现”页面中，向上滑动屏幕，找到并点击“视频号”入口。其图标通常为黄色，形似一只蝴蝶。

    ![](img/bd893946fd15e9247889d55a9386efc1_3.png)
    ![](img/bd893946fd15e9247889d55a9386efc1_4.png)

![](img/bd893946fd15e9247889d55a9386efc1_7.png)

4.  **进入个人主页**：进入视频号页面后，点击右上角的个人头像图标。

    ![](img/bd893946fd15e9247889d55a9386efc1_6.png)

5.  **查看账号信息**：点击头像后进入新页面，向下滑动，找到“我的视频号”栏目。这里会明确显示你的**账号昵称**和**粉丝数量**。例如：`天道酬勤，39917个粉丝`。

    ![](img/bd893946fd15e9247889d55a9386efc1_7.png)

至此，你已成功获取到需要填写的账号昵称和粉丝量数据。

---

## 填写邀请函五项内容

获取账号信息后，接下来我们开始正式填写邀请函。邀请函通常包含五个必填项。

以下是需要填写的五项内容及填写规范：

1.  **视频号账号昵称与粉丝量**：将上一步查找到的昵称和粉丝数填写到对应位置。例如，昵称填“天道酬勤”，粉丝量填“39917”。

![](img/bd893946fd15e9247889d55a9386efc1_9.png)

![](img/bd893946fd15e9247889d55a9386efc1_10.png)

2.  **活动邀请人**：由于是通过特定渠道加入，此处需填写指定邀请人姓名，例如“张良”或“张良老师”。

![](img/bd893946fd15e9247889d55a9386efc1_11.png)

3.  **实名信息**：此处必须填写你本人的真实姓名，与身份证信息一致。例如，姓名是“周建国”，则填写“周建国”。

4.  **结算手机号**：填写用于接收验证码和后续收益结算的手机号码。格式为11位连续数字，例如 `13812345678`，中间不能有空格。

5.  **提交确认**：所有信息填写无误后，点击页面底部的蓝色“提交”按钮。系统会弹出“确定要提交吗？”的确认框，点击“确认”即可完成提交。

    ![](img/bd893946fd15e9247889d55a9386efc1_9.png)
    ![](img/bd893946fd15e9247889d55a9386efc1_10.png)
    ![](img/bd893946fd15e9386efc1_11.png)

提交成功后，页面会提示“已成功加入创作者分成计划”。

![](img/bd893946fd15e9247889d55a9386efc1_13.png)

## 提交后的素材获取与发布

成功提交邀请函后，你将正式加入计划。作为新创作者，你将获得由官方或导师提供的起步素材，用于发布你的第一个视频。

导师会将剪辑好的视频素材直接发送给你。你只需登录自己的视频号，将收到的素材按顺序发布即可。例如，导师可能会发送三个已编号的影视素材文件，你按照 `1, 2, 3` 的顺序发布。

![](img/bd893946fd15e9247889d55a9386efc1_16.png)

这些素材来源于拥有正规版权的素材库，确保了内容的合规性，避免了侵权风险。素材库中包含了大量经典影视剧、动漫等资源，例如《还珠格格》、《我不是药神》、《大话西游》等。
