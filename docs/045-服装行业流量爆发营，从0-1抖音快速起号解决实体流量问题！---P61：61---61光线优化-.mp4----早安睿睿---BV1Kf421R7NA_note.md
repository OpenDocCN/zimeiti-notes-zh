# 服装行业流量爆发营：P61：光线优化 💡

在本节课中，我们将学习如何优化拍摄视频时的光线。好的光线能显著提升视频的视觉质量和专业感，对于吸引观众至关重要。我们将从三个核心光源入手，讲解如何布置简单而有效的光线环境。

## 光线优化的三个核心要素

光线优化主要围绕三个清晰的光源展开：主光、辅光和背景光。

![](img/3794b86b39b5124e0215eb92e7ec8a97_1.png)

### 1. 主光：照亮你的主体

主光是可以将你的人脸单独打亮的光源。它的作用是确保主体清晰、明亮。
*   **光源类型**：可以是自然光、环形灯或台灯。
*   **核心作用**：主光能大幅提升视觉观感。例如，在灯光通常不够明亮或比较杂乱的店铺环境中，在你和顾客之间增加一个主光源，能呈现出完全不同的效果。

在布置好主光之后，接下来我们来看看辅光。

### 2. 辅光：辅助与补充

顾名思义，辅光是辅助光源。
*   **核心作用**：它用于打亮主体另一侧的阴影面，或打亮发丝，从而提升画面的整体质感，使光线过渡更自然。

了解了如何照亮主体和补充阴影后，我们还需要考虑如何让人物从背景中凸显出来。

### 3. 背景光：分离主体与背景

背景光是能使人物和背景分离开来的光源。
*   **核心作用**：它通过照亮背景或制造光效，在人物背后形成层次，增强画面的立体感和空间感。

## 光线布置的核心原则

在学会了控制三种光线后，需要记住核心原则：光线布置并非越亮越好。
*   **追求简单统一**：为了保证画面的清晰度和颜色统一，光线应尽可能简单。
*   **创造层次感**：简单而有层次的光线才是最优选择。这意味着通过主光、辅光、背景光的合理搭配，形成明暗过渡，突出主体。
