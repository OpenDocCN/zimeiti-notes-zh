# 自媒体运营入门：P1：小说推文项目详解 🚀

在本节课中，我们将学习一种无需实拍、无需原创文案即可参与的抖音自媒体玩法——小说推文项目。我们将从项目原理、操作步骤到收益计算，完整地解析这个适合新手的入门项目。

## 项目原理与案例展示

上一段我们提到了一个无需复杂操作的自媒体项目，本节中我们来看看它的具体原理。

一个抖音视频展示了“我运营大学表白墙账号”的故事。该视频获得了2000多个点赞和几十万次播放。

不要小看这几十万次播放。仅这一条视频就为创作者带来了两三千元的收益。最关键的是，这类视频无需真人实拍出镜，也无需自行撰写文案。文案内容是复制粘贴的，并且无需投放“抖加”进行推广，操作难度非常小。

这可以说是最适合新手入门操作的一个项目。有人可能会问，这种视频是如何变现的？难道仅通过“中视频计划”，一条视频就能变现两三千元吗？答案是否定的。

因为这个视频除了参加“中视频计划”获取播放量收益之外，最关键的一点在于它具有CPA（按效果付费）属性。虽然视频画面播放的是解压内容，但文案一直在朗读某部小说的片段。只要有人通过这个视频去搜索这部小说的完整版，那么创作者每为小说平台成功推荐一位读者，就能获得大约十元左右的佣金。这项收益不亚于“中视频计划”。

如果查看这类视频的评论区，会发现作者通常会置顶一条引导用户去寻找小说完整版的评论。玄机就在这条置顶评论里。他让你去搜索的关键词，可能并非小说原本的名字，而是平台与作者约定好的特定关键词。只要用户通过这个关键词在平台上找到小说并成为用户，就算作者成功为该平台推荐了一名用户。这个关键词就是作者与平台之间的约定。

这个玩法在平台上已经有一些人在操作，并且做得非常不错。这就是“小说推文”项目。其原理是通过在抖音发布作品，为小说推荐用户，从而获取佣金。就是这样一个简单的模式。我们圈内就有人在做这个项目，已经达到了月入数万元的水平。

## 项目操作步骤详解

了解了项目的基本原理后，本节中我们将重点拆解其具体的操作流程。

要操作这个项目，主要分为两大步骤。

![](img/5da44602d6d7a8307600475f2c2fc959_1.png)

**第一步：成为推荐官并获取授权**

你需要成为“推荐官”，拿到小说平台的授权。也就是获取每部小说对应的推荐关键词。至于具体如何授权，无法在正文中直接说明。在视频结尾会告知大家操作方法。

**第二步：制作视频内容**

接下来是很重要的一步：视频制作。其实视频制作非常简单，因为它根本不需要你写文案。

以下是制作视频的具体步骤列表：

1.  **复制小说文案**：直接使用小说中比较经典、能够引发好奇心的段落。复制大约1000字左右的内容。
2.  **生成配音**：将复制好的文案粘贴到AI配音工具中。选择一个听起来比较舒服的主播声音，然后直接导出配音音频文件。
3.  **准备视频素材**：寻找一些解压类的视频素材，就像课程开头展示的视频一样。此处需要注意，解压视频千万不要只使用一个，而是选取多个进行重新组合。
4.  **合成视频**：将解压视频的原声调小，使用剪映等软件导入配音音频和视频素材，进行合成后导出。

这样一个视频就完成了。这种视频的生产速度非常快，基本上一个视频在10分钟以内就能搞定。

## 收益分析与总结

最后，我们来算一笔账，并对本节课的内容进行总结。

经过了解，小说推文项目基本上在抖音上每获得1万次播放，就能够产生2到5个订单。每单的收益大约在十元左右。

我们可以设想一下：假如你在抖音上运营十个账号，每个账号每天发布三条视频。每条视频即使只获得抖音给予的500个基础播放，总共也有15000个播放量，这至少能产生5单以上的收益。

事实上，我们圈内有很多小伙伴，一个人就操作二三十个账号，这毫不夸张。据观察，基本上每发布100条作品，其中肯定会有一条成为爆款视频。一旦出现爆款视频，播放量就会非常高，仅这一条视频就有可能带来好几千元的收益。
