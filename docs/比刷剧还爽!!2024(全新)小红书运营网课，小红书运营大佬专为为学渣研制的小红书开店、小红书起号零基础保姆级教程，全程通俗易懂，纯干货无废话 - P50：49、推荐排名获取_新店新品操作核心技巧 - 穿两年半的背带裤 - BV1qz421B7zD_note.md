## 平台排名逻辑与新品扶持

上一节我们介绍了店铺运营的整体框架，本节中我们来看看平台对新品排名的具体考核逻辑。

拼多多平台对所有商品（包括新品）都会给予基础曝光。所谓的“新品扶持”，并非直接给予更多流量，而是在新品上架初期（通常为30天），平台会默认新店的某些数据（如DSR、店铺层级、服务指标等）考核达标，从而让新品有机会进入推荐池参与排名竞争。

商品排名主要考核以下四个维度：
*   **店铺数据**：如DSR评分、店铺层级、服务指标等。
*   **商品质量**：如标题、主图、详情页、价格、属性等。
*   **商品数据**：如上新时间、销量、评价、转化率、UV价值等。
*   **处罚记录**：店铺是否有违规处罚。

对于新店新品，平台会优先考核**商品质量**，给予一个基础权重，然后根据这个权重在“新品推荐位”进行展示。你的商品质量得分越高，在新品池中的排名就越靠前，获得的初始曝光机会也就越多。

## 从曝光到点击的关键路径

理解了平台如何分配初始曝光后，我们需要解决下一个问题：如何将这些曝光转化为真实的访客（点击）。

![](img/a6f5b0a52b3e922ed5b8597cad098c5b_1.png)

平台给予的是**曝光**（Exposure），即商品被系统收录并可能展示给用户。只有当商品出现在用户当前可视的屏幕范围内时，才称为**展现**（Impression）。用户看到商品后产生兴趣并点击，才形成**访客**（Click/Visit）。

![](img/a6f5b0a52b3e922ed5b8597cad098c5b_3.png)

因此，提升访客的关键路径是：
**提升商品排名 → 增加曝光成为展现的机会 → 优化点击吸引力 → 获得访客**

以下是影响“展现”形成“点击”的核心因素分析：

*   **主图**：**这是最核心的因素**。用户在信息流中瞬间决定是否点击，主图起到了90%以上的作用。
*   **标题**：影响搜索匹配度和人群精准度，对于通过搜索渠道的点击非常重要。
*   **价格**：在系统推荐阶段，价格的影响相对次要，因为系统会推荐给可能接受该价格区间的人群。但在影响点击决策时，有竞争力的价格或清晰的优惠信息能提升点击率。
*   **营销文案/卖点**：在主图上或价格旁添加戳中用户痛点的文案，能有效提升点击欲望。
*   **销量/评价**：有一定影响，能增加信任感，但对于全新新品来说，这部分数据薄弱，不是前期核心。
*   **品牌/黑标**：能显著增加信任度和点击率，但需要一定条件才能获取。
*   **SKU/详情页**：**在点击阶段几乎无影响**。它们主要影响用户进入商品页后的转化决策。

**结论**：对于新店新品，想要把平台给予的曝光转化为点击，必须在**主图优化**和**标题撰写**上下足功夫，其中**主图是重中之重**。

## 核心实操：打造高权重原创主图

上一节我们明确了主图的核心地位，本节将分享一个能有效提升新品权重、避免同质化的原创主图制作技巧——**颜色叠加法**。

很多商家使用供应商或同行的图片，导致商品被系统识别为“同款”或“相似款”，从而丧失新品权重，陷入比价困境。即使自己拍摄，同质化严重的标品也很难避免这个问题。

**颜色叠加法**的原理是：在不影响人工识别商品主体的前提下，通过添加一个半透明的、复杂的颜色图层，改变图片的底层像素色彩分布，从而使系统识别为一张全新的原创图片。

**操作步骤如下：**

1.  准备商品图：在PS中打开你的商品主图。
2.  叠加颜色层：在网上任意寻找一张色彩复杂丰富的图片（如油画、渐变色彩图等），导入PS，覆盖在整个商品图上。
3.  调整图层：将叠加的彩色图片图层进行**自由变换**，拉伸至完全覆盖背景。
4.  降低不透明度：将该彩色图层的**不透明度调至极低**（例如2%-5%），确保商品主体依然清晰可见。
5.  导出图片：保存这张处理后的图片作为你的主图。

**代码/公式描述此方法的核心：**
`最终主图 = 原始商品图 + (复杂色彩图层 * 极低不透明度)`

**此方法的优势：**
*   **保证原创性**：系统识别为全新图片，获得更高新品权重。
*   **避免同款**：极大降低被识别为同款/相似款的风险。
*   **操作简单**：无需高超的PS技术，可批量操作。
*   **独一无二**：即使他人使用相同方法，因选择的色彩图和变形程度不同，结果也完全不同，你的图片仍具有独特性。

![](img/a6f5b0a52b3e922ed5b8597cad098c5b_5.png)

制作好高权重的原创主图后，再结合精准的卖点文案，你的新品点击率将会得到有力保障。

## 付费推广的正确使用时机

在课程的最后，我们需要明确一个重要的运营原则：新店新品前期**不建议**开付费推广。

付费推广（如搜索/场景推广）就像一个“凳子”，它垫高了你的整体曝光基数，让你不努力也能获得展示。但这会导致一个严重问题：**平台原本应该给你的自然推荐曝光，会被付费曝光覆盖**。

具体逻辑是：
1.  新品期，平台会给你自然推荐曝光。
2.  如果你同时开付费，用户点击大概率会先发生在付费展示位上。
3.  导致自然推荐曝光得不到应有的点击、转化等数据反馈。
4.  系统会判定你的商品在自然流量中表现不佳，从而减少甚至停止自然流量的推荐。
5.  结果就是：**一停付费，流量立刻消失**，店铺陷入“不付费就没流量”的恶性循环。

**正确的付费推广时机是**：当你的商品通过自然流量运营，各项数据（转化率、UV价值、成交额）已经稳定且优秀，但流量遇到了瓶颈，无法再突破时。此时，利用付费推广去**放大**和**拓展**流量池，才是有效的。
