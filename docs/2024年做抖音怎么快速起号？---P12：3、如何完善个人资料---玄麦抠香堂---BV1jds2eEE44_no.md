# 抖音运营入门：P12：3、如何完善个人资料 🧑‍💼

在本节课中，我们将学习如何完善抖音账号的个人资料。一个专业、清晰且符合平台规则的个人资料，是打造高权重账号、吸引精准粉丝的第一步。我们将重点讲解头像、昵称和个人简介的设置方法与核心原则。

上一节我们介绍了养号的基本概念，本节中我们来看看如何通过完善个人资料来提升账号的专业度。

## 头像设置

以下是头像设置的具体建议：

*   **有实体店**：可以使用实体店铺的照片作为头像。
*   **无实体店**：可以使用个人自拍照或生活照。
*   **宝妈群体**：可以使用与宝宝的合照。
*   **宠物领域**：可以使用宠物的照片。

## 昵称设置

昵称是用户对你的第一印象，设置不当可能导致账号被判定为普通玩家而非创作者，从而影响流量推荐。**禁止使用自己的真实姓名**（如：小月、兰亭），除非你是知名公众人物。

以下是正确的昵称设置方法，需满足 **`昵称长度 ≤ 5个汉字`** 的原则，并避免使用繁体字和特殊符号（如：❤️）。

**核心命名公式为：**
1.  **昵称 + 领域定位**
    *   示例：`小月说游戏`、`小月小厨房`、`小月聊房`、`小月说车`、`小月穿搭`
2.  **昵称 + 职业身份**
    *   示例：`宝妈小月`、`律师小月`、`摄影师小月`、`造型师小月`
3.  **昵称 + 所在地区**
    *   示例：`赣州月姐`、`小月在北漂`、`小月在海外`

## 个人简介撰写

![](img/8ca0bda46bf822db5075582b3e391128_1.png)

个人简介位于抖音主页，是展示账号价值的关键。撰写时应围绕以下三个核心要素展开：

1.  **我是谁**：介绍自己的基本身份或特点。
2.  **我是做什么的**：明确你的专业领域或行业经验。
3.  **关注我能得到什么**：说明你能为粉丝提供的价值。

**示例代码（以母婴账号为例）：**
```
中年大叔依然保持母婴初衷（我是谁）
从事母婴行业13年，做母婴商贸8年有余（我是做什么的）
分享育儿经验（关注我能得到什么）
```

![](img/8ca0bda46bf822db5075582b3e391128_3.png)

撰写风格可根据个人性格调整，或幽默或可爱，无需过于死板。

## 重要规则：联系方式

在个人资料中**切勿随意添加微信、电话等第三方联系方式**。这是跨平台引流行为，违反抖音社区规定，可能导致违规。

**允许挂载联系方式的情况仅限以下两种：**
1.  已认证**企业蓝V账号**。
2.  粉丝数达到**1万**并成功入驻**星图平台**（后续课程会讲解）。

如果已违规添加，请立即删除。
