# 微信视频号运营从0~1全攻略：P20：18.活动介绍

在本节课中，我们将详细介绍一项由官方发起的视频号商家扶持活动。该活动旨在为商家提供从素材到流量的全方位支持，帮助大家快速启动视频带货。

## 活动背景与负责人

![](img/5459325f2d33c33ef59ec883dbe4cf8c_1.png)

为促进2024年中国电商行业的新发展，并为视频化带货打下坚实技术基础，微信视频号官方特推出“商家联盟”活动。本次活动由短视频教研部负责人张良老师全程对接。

![](img/5459325f2d33c33ef59ec883dbe4cf8c_2.png)

本次活动秉持公平、公正、公开的理念展开。张老师作为本次活动的全国总负责人，拥有官方直接授权。这意味着，通过张老师这里可以对接到的资源和支持，在其他地方可能无法获得。

## 活动核心扶持内容

![](img/5459325f2d33c33ef59ec883dbe4cf8c_4.png)

![](img/5459325f2d33c33ef59ec883dbe4cf8c_6.png)

上一节我们介绍了活动的背景，本节中我们来看看活动具体提供了哪些令人心动的支持。以下是本次活动的三大核心扶持：

1.  **官方提供视频素材**：许多同学担心没有高质量的拍摄素材。本次活动官方已为大家准备好了现成的产品视频素材。每个参与活动的账号，官方将直接提供 **30条** 高质量视频。参与者每天发布一条即可。

2.  **赠送巨额流量扶持**：针对“发了视频卖不出去”的担忧，官方为每个参与账号提供 **90万** 的流量扶持。这能极大提升视频的曝光机会。

3.  **素材库丰富且易用**：官方准备的素材库品类齐全，质量上乘，并且支持搜索功能，方便商家快速找到所需素材。例如，中秋节礼品、餐巾纸、牛奶、火锅底料、书籍等各类产品均有对应视频。

## 活动奖励与结算方式

![](img/5459325f2d33c33ef59ec883dbe4cf8c_12.png)

![](img/5459325f2d33c33ef59ec883dbe4cf8c_13.png)

除了素材和流量，本次活动还设置了直接的现金奖励，以增强商家的信心和动力。

官方承诺，参与者每按要求发布一条带货视频，即可获得 **300元** 的奖励。奖励结算方式为 **日结**，即今天发布视频，明天奖励到账。

按此计算，如果完成官方提供的30条视频发布任务，总计可获得：
`300元/条 * 30条 = 9000元`

![](img/5459325f2d33c33ef59ec883dbe4cf8c_15.png)

![](img/5459325f2d33c33ef59ec883dbe4cf8c_15.png)

## 活动期限与参与条件

需要注意的是，任何大型扶持活动都有成本，因此设有明确的期限。本次活动并非长期进行。

活动于今年1月13日正式开始，并定于 **8月10日** 结束。机会具有时效性，官方希望符合条件的同学能把握最后的机会。

![](img/5459325f2d33c33ef59ec883dbe4cf8c_17.png)

参与活动并获得奖励并非毫无条件。参与者必须完成一个核心动作：**将官方提供的视频素材发布出去，并正确挂上商品链接**。这意味着，参与者仍需学习基本的视频号发布与带货操作。

![](img/5459325f2d33c33ef59ec883dbe4cf8c_18.png)
