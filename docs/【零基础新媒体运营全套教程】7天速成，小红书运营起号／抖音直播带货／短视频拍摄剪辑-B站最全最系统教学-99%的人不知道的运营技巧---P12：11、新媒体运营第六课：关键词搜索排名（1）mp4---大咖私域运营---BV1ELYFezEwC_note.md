## 什么是搜索排名？🔎

![](img/88f70cff5a5752a5898be7818ae68188_16.png)

上一节我们介绍了新媒体运营的宏观概念，本节中我们来看看一个具体的流量获取机制：搜索排名。

![](img/88f70cff5a5752a5898be7818ae68188_18.png)

![](img/88f70cff5a5752a5898be7818ae68188_20.png)

![](img/88f70cff5a5752a5898be7818ae68188_22.png)

搜索排名指的是，当用户在平台（如B站、小红书、抖音等）的搜索框中输入特定词语（即关键词）后，系统根据一系列算法规则，将相关内容进行排序并展示的结果列表。

![](img/88f70cff5a5752a5898be7818ae68188_24.png)

![](img/88f70cff5a5752a5898be7818ae68188_26.png)

![](img/88f70cff5a5752a5898be7818ae68188_28.png)

![](img/88f70cff5a5752a5898be7818ae68188_30.png)

为了更直观地理解，我们以B站为例进行演示。当用户在搜索框输入“英语口语”并点击搜索后，平台会返回一系列相关内容。



![](img/88f70cff5a5752a5898be7818ae68188_32.png)

![](img/88f70cff5a5752a5898be7818ae68188_34.png)

![](img/88f70cff5a5752a5898be7818ae68188_1.png)



![](img/88f70cff5a5752a5898be7818ae68188_36.png)

![](img/88f70cff5a5752a5898be7818ae68188_38.png)

![](img/88f70cff5a5752a5898be7818ae68188_40.png)

在分析搜索结果时，我们需要排除几类特殊内容：
*   **平台课程**：属于B站上架的付费课程，其排名规则与普通用户上传的视频不同。
*   **创作推广（广告）**：这是博主购买了关键词广告位的结果。其排名依赖于广告投放，一旦停止付费，排名就会变化。
*   **直播**：属于实时内容，其排名机制也与常规视频不同。



![](img/88f70cff5a5752a5898be7818ae68188_42.png)

![](img/88f70cff5a5752a5898be7818ae68188_44.png)

![](img/88f70cff5a5752a5898be7818ae68188_46.png)

![](img/88f70cff5a5752a5898be7818ae68188_10.png)



![](img/88f70cff5a5752a5898be7818ae68188_48.png)

![](img/88f70cff5a5752a5898be7818ae68188_50.png)

![](img/88f70cff5a5752a5898be7818ae68188_52.png)

我们重点关注 **自然视频的流量排名**。例如，在“英语口语”的搜索结果中，“YOUTUBE口语精选”可能排在第一，“easy english系列”排在第二，后续视频依次排列。



![](img/88f70cff5a5752a5898be7818ae68188_54.png)

![](img/88f70cff5a5752a5898be7818ae68188_56.png)

![](img/88f70cff5a5898be7818ae68188_20.png)



![](img/88f70cff5a5752a5898be7818ae68188_58.png)

这个顺序就是 **搜索排名**。它意味着在特定的搜索场景下，内容会按照与用户搜索意图的相关性等因素进行系统排序。

![](img/88f70cff5a5752a5898be7818ae68188_60.png)

![](img/88f70cff5a5752a5898be7818ae68188_62.png)

## 搜索排名的重要性🚀

![](img/88f70cff5a5752a5898be7818ae68188_64.png)

![](img/88f70cff5a5752a5898be7818ae68188_66.png)

![](img/88f70cff5a5752a5898be7818ae68188_68.png)

从上述演示中，我们可以得出一个核心结论：**在搜索场景下，排名越靠前的内容，获得的点击和流量就越多**。

![](img/88f70cff5a5752a5898be7818ae68188_70.png)

这是因为：
1.  用户主动搜索，代表其有明确、具体的内容需求。
2.  系统将最符合需求的内容优先推荐给用户。
3.  用户通常会优先点击排名第一或靠前的结果。如果该内容满足需求，用户便会停留；若不满足，才会浏览后续内容。

![](img/88f70cff5a5752a5898be7818ae68188_72.png)

因此，提升内容在目标关键词下的搜索排名，是获取精准流量的关键手段。这一原理适用于所有具备搜索功能的平台，包括B站、抖音、小红书、百度等。

![](img/88f70cff5a5752a5898be7818ae68188_74.png)

## 核心概念：关键词🔑

![](img/88f70cff5a5752a5898be7818ae68188_76.png)

![](img/88f70cff5a5752a5898be7818ae68188_78.png)

既然搜索排名如此重要，那么理解“关键词”就是第一步。我们刚才搜索的“英语口语”，本身就是一个关键词。

![](img/88f70cff5a5752a5898be7818ae68188_80.png)

![](img/88f70cff5a5752a5898be7818ae68188_82.png)

![](img/88f70cff5a5752a5898be7818ae68188_84.png)

![](img/88f70cff5a5752a5898be7818ae68188_86.png)

**关键词**代表了用户确切的搜索需求。与基于兴趣的随机浏览不同，搜索行为意味着用户带着明确目的而来。例如，想学“英语口语”才会去搜，而不是漫无目的地刷首页。

![](img/88f70cff5a5752a5898be7818ae68188_88.png)

## 关键词的分类：核心词与长尾词📊

![](img/88f70cff5a5752a5898be7818ae68188_90.png)

![](img/88f70cff5a5752a5898be7818ae68188_92.png)

![](img/88f70cff5a5752a5898be7818ae68188_94.png)

![](img/88f70cff5a5752a5898be7818ae68188_95.png)

![](img/88f70cff5a5752a5898be7818ae68188_96.png)

在运营中，我们通常将行业内的关键词分为两大类：**核心关键词**和**长尾关键词**。

![](img/88f70cff5a5752a5898be7818ae68188_98.png)

![](img/88f70cff5a5752a5898be7818ae68188_100.png)

![](img/88f70cff5a5752a5898be7818ae68188_102.png)

![](img/88f70cff5a5752a5898be7818ae68188_103.png)

每个行业都有属于自己的关键词体系。例如，新媒体运营行业的关键词可能包括“内容运营”、“短视频剪辑”等，而家居装修行业则有“装修避坑”、“小户型设计”等。

![](img/88f70cff5a5752a5898be7818ae68188_105.png)

![](img/88f70cff5a5752a5898be7818ae68188_107.png)

![](img/88f70cff5a5752a5898be7818ae68188_109.png)

![](img/88f70cff5a5752a5898be7818ae68188_111.png)

下面我们以“英语培训”行业为例，详细解析这两种词性。

![](img/88f70cff5a5752a5898be7818ae68188_113.png)

假设我们需要为一家英语提升机构筛选关键词，面临两个选项：“英语”和“英语口语”。哪个更适合作为核心关键词？

![](img/88f70cff5a5752a5898be7818ae68188_115.png)

![](img/88f70cff5a5752a5898be7818ae68188_117.png)

*   **“英语”**：这是一个过于宽泛的学科名词。搜索“英语”的用户意图非常分散，可能是想了解英语专业、查询考试信息，或是学习音标，并不一定代表有系统化提升英语能力的需求。因此，它**不适合**作为精准获客的核心词。
*   **“英语口语”**：这个词范围更聚焦，代表了用户在“英语”学科下的一个具体方向（口语）。用户搜索它，更可能带有学习或提升的目的。因此，它可以作为前端引流的**核心关键词**之一。

![](img/88f70cff5a5752a5898be7818ae68188_119.png)

![](img/88f70cff5a5752a5898be7818ae68188_121.png)

然而，“英语口语”仍然不够精准。我们无法仅凭这个词判断用户具体是想要练习方法、零基础教程，还是直接寻找课程。这时，就需要 **长尾关键词** 来进一步细化需求。

![](img/88f70cff5a5752a5898be7818ae68188_123.png)

![](img/88f70cff5a5752a5898be7818ae68188_125.png)

以下是基于核心词“英语口语”衍生出的部分长尾词示例：
*   英语口语怎么练
*   英语口语练习
*   英语口语零基础
*   英语口语课程

![](img/88f70cff5a5752a5898be7818ae68188_127.png)

![](img/88f70cff5a5752a5898be7818ae68188_129.png)

![](img/88f70cff5a5752a5898be7818ae68188_131.png)

![](img/88f70cff5a5752a5898be7818ae68188_133.png)

这些长尾词能更准确地反映用户的具体疑问和场景。我们的内容创作就可以针对这些长尾词提供解决方案：
*   针对“英语口语怎么练”：标题可以是《三个高效的英语口语练习方法，一般人我不告诉他》。
*   针对“英语口语零基础”：标题可以是《零基础30天快速提升英语口语的黄金法则》。

![](img/88f70cff5a5752a5898be7818ae68188_135.png)

![](img/88f70cff5a5752a5898be7818ae68188_137.png)

长尾词还可以进一步细分。例如，在“英语口语练习”这个长尾词下，又能衍生出更具体的词，如“英语口语练习软件”、“英语口语练习素材（美音）”等。



![](img/88f70cff5a5752a5898be7818ae68188_139.png)

![](img/88f70cff5a5752a5898be7818ae68188_141.png)

![](img/88f70cff5a5752a5898be7818ae68188_143.png)

![](img/88f70cff5a5752a5898be7818ae68188_145.png)

![](img/88f70cff5a5752a5898be7818ae68188_135.png)



![](img/88f70cff5a5752a5898be7818ae68188_147.png)

通过这样的层层细分，我们可以挖掘出大量代表用户不同细分需求的长尾词。一个行业的词库可能包含数百甚至上千个关键词。用户的搜索用词千差万别（如“怎么练”、“练习方法”、“如何提升”），但都指向相似的需求。我们的内容需要尽可能覆盖这些不同的搜索表达。

![](img/88f70cff5a5752a5898be7818ae68188_149.png)

![](img/88f70cff5a5752a5898be7818ae68188_151.png)

同理，我们可以确定其他核心词，如“英语语法”。围绕“英语语法”这个核心词，同样可以拓展出“英语语法零基础”、“英语语法填空技巧”等一系列长尾词。



![](img/88f70cff5a5752a5898be7818ae68188_153.png)

![](img/88f70cff5a5752a5898be7818ae68188_155.png)

![](img/88f70cff5a5752a5898be7818ae68188_157.png)

![](img/88f70cff5a5752a5898be7818ae68188_165.png)



![](img/88f70cff5a5752a5898be7818ae68188_159.png)

![](img/88f70cff5a5752a5898be7818ae68188_161.png)

![](img/88f70cff5a5752a5898be7818ae68188_163.png)

**核心公式**：
`核心关键词` -> 衍生出 -> `长尾关键词1， 长尾关键词2， ...` -> 进一步衍生出 -> `更具体的长尾关键词`

本节课我们一起学习了新媒体运营中的关键词搜索排名。
1.  **搜索排名**是内容在用户搜索结果中的顺序，排名越靠前，获得的精准流量越多。
2.  **关键词**是用户搜索需求的直接体现，是影响排名的核心要素。
3.  关键词分为**核心关键词**（范围较广，代表主要方向）和**长尾关键词**（范围具体，代表细分需求）。
4.  通过核心词与长尾词的划分与拓展，我们可以系统性地了解用户需求，并据此创作出针对性更强的内容，从而提升搜索排名，获取有效流量。

![](img/88f70cff5a5752a5898be7818ae68188_184.png)
