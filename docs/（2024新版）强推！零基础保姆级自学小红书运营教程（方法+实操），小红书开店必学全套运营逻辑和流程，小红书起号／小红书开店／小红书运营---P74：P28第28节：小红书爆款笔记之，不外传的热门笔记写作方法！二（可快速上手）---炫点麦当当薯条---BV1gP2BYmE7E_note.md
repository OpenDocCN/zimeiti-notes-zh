# 小红书运营教程：P28：小红书爆款笔记写作方法（二）🔥

在本节课中，我们将学习如何确立小红书笔记的内容，包括从确定领域到最终成稿的完整流程。我们将重点介绍一个实用的数据分析工具，并讲解标题撰写的核心技巧。

上一节我们介绍了爆款笔记的基础框架，本节中我们来看看如何具体确立笔记内容并进行优化。

## 一、确立笔记内容的五个步骤

撰写笔记前，清晰的内容规划至关重要。以下是确立内容的五个核心步骤。

### 1. 确定内容领域
首先，必须明确你的账号专注的领域。例如，是美妆、护肤、母婴还是其他类别。在之前的课程中已多次强调，领域不明确会导致账号垂直度降低。

账号垂直度降低会引发一系列问题：官方推荐减少、粉丝流失，并形成恶性循环——粉丝越不喜欢你的内容，官方就越不推荐。

### 2. 梳理行业关键词
确定领域后，需要梳理与该领域相关的关键词。在之前的课程中，我们介绍了使用百度工具、爱站网、5118、百度指数等关键词工具进行查询的方法。

本节课不再赘述具体操作。根据你的行业，可以梳理出关键词排行列表。用好这些关键词能极大提升内容效果。

### 3. 搜集同行案例
接下来需要搜集同行的优秀案例。有人可能会问，在小红书内搜索无法看到同行的具体数据，该怎么办？

![](img/25e78989a72e9decdf102754216faf5d_1.png)

解决方法其实很简单。我们推荐使用一个工具：**千瓜数据**。这是一个长期被使用且非常好用的数据分析平台。

![](img/25e78989a72e9decdf102754216faf5d_2.png)

![](img/25e78989a72e9decdf102754216faf5d_3.png)

千瓜数据的优势在于它能提供小红书平台的分析功能，例如：
*   匹配优质达人
*   分析热门数据
*   查看品牌方投放情况

![](img/25e78989a72e9decdf102754216faf5d_5.png)

此外，平台还提供各种榜单。那么，具体如何使用千瓜数据筛选内容呢？下面我们来详细说明。

### 4. 使用工具筛选内容
搜集同行案例后，即可通过这些案例进行内容筛选。筛选完成后，便能整理成自己的内容进行发布。这里我们用千瓜数据来演示。

![](img/25e78989a72e9decdf102754216faf5d_7.png)

在千瓜数据网站左侧栏目中，可以看到许多功能，包括热词排行榜、热词搜索榜、笔记收录查询等。

本节课主要讲解“笔记收录查询”功能。例如，发布笔记后，你可以利用此功能查询笔记是否被官方收录。关于热门内容和热词排行榜，此处不再展示。

另一个主要使用的功能是“达人搜索”。可以按昵称搜索，或按笔记内容搜索。例如，我们搜索“美妆”（此功能需付费），或按昵称搜索，效果类似。

按昵称搜索会显示达人的多种属性，例如彩妆、护肤、母婴、美食饮品、服饰穿搭等分类。大家可以根据自身需求进行筛选。

我们还能看到达人属性，如是否是官方号、品牌号或头部达人。平台提供了大量数据，包括达人对比、排名等。

如果需要使用千瓜数据，可以根据实际需求在淘宝上寻找开通渠道，费用通常比较划算。

拥有此功能后，例如点击某个达人的详情页，就能看到他发布的数据、笔记分类、详情以及笔记的具体类型。例如，笔记的评论词云图显示了高频词汇。

用好这个工具，能帮助我们有效掌握小红书的数据参数。

### 5. 整理并发布内容
说完了千瓜数据的使用，最后我们谈谈内容整理。在之前的课程中讲过，主要利用“总分总”的结构来撰写内容。

![](img/25e78989a72e9decdf102754216faf5d_9.png)

这个结构并不难写。如果觉得困难，建议回顾之前的课程内容。本节课不再重复这一点。

## 二、其他注意事项

内容确立后，标题的优化同样重要。以下是标题撰写的三个注意事项。

1.  **控制字数**：标题字数尽量控制在 **12到18个字** 之内。
2.  **添加符号美化**：标题中可多带一些符号（如🔥、💄、✨）进行美化。因为小红书女性用户居多，美化后的标题能更好地吸引她们的注意力。
3.  **加入具体数据**：在标题中加入具体数据可以增加真实感。例如，像“百万销量”、“3天见效”这样的详细数字，会让人感觉更可信。因此，在拟定标题文字时，可以加入数字。
