# 剪映手机版教学：P47：视频变照片卡点 📸

在本节课中，我们将学习如何制作“视频变照片”卡点效果。你将掌握定格功能的使用，学会根据音乐波形图找准节奏，并了解如何添加文字、滤镜和贴纸来丰富视频效果。

## 课程概述

上一节我们学习了基础的卡点概念。本节中，我们来看看如何将动态视频在特定节拍处“定格”为静态图片，从而制作出富有节奏感的卡点视频。核心操作包括：使用**定格**功能、通过**波形图**手动踩点、以及运用**滤镜**和**贴纸**进行装饰。

## 1. 准备素材与音乐 🎵

首先，我们需要导入视频素材并添加背景音乐。

1.  点击「开始创作」，导入你的视频素材，然后点击「添加」。
2.  点击下方工具栏的「添加音频」，选择「提取音乐」，从你的视频文件中导入背景音乐。

此时，音乐已添加到时间轴。接下来，我们需要为音乐设置节拍点。

## 2. 设置音乐节拍点 🎶

我们将通过自动和手动结合的方式，为音乐打上节拍点。

1.  **剪辑视频开头**：如果视频开头有对话，可以将其保留。从需要卡点的音乐部分开始，将前面的视频片段分割并删除。
2.  **调整音频**：
    *   可以选中视频片段，点击「音量」，适当提高人物对话的音量。
    *   选中音乐片段，点击「淡化」，为其添加「淡入时长」和「淡出时长」，使音乐的开始和结束更自然。
3.  **自动踩点**：选中音乐轨道，点击「节拍」，选择「自动踩点」。系统会生成一些节拍点。
4.  **手动微调节拍点**：自动踩点可能不准确，需要手动调整。
    *   放大时间轴，仔细聆听音乐，找到重音（“哒”或“噔”声）的位置。
    *   **观察波形图**：波形图中凸起较高的地方，通常就是节拍点。
    *   在需要的位置点击「添加点」，删除多余或错误的点，确保每个节拍点之间的距离大致均匀。

完成节拍设置后，点击勾选确认。

## 3. 对齐画面与节拍 🖼️

现在，我们将视频画面与节拍点对齐。

1.  在第一个节拍点处，选中视频，点击「分割」。
2.  拖动分割后的第二段视频，为其寻找一个新的画面。
3.  在下一个节拍点处，再次「分割」，并删除中间不需要的部分，使新画面从该节拍点开始。
4.  重复以上步骤，为前几个节拍点准备好不同的视频画面。

## 4. 使用定格功能 ⏸️

这是本节课的核心操作，将视频瞬间变为静态图片。

1.  在需要“定格”的画面处（例如人物动作的某个瞬间），将视频「分割」。
2.  选中分割出来的这一小段视频，在下方工具栏中找到并点击「定格」。
3.  操作后，时间轴上会生成一张3秒长的静态图片。
4.  将这张图片的末端，拖动对齐到下一个节拍点。时间轴有吸附功能，可以帮你精准对齐。
5.  重复以上步骤，为后续每个需要定格的节拍点制作图片。

完成后预览，效果应是一段动态视频后，紧接着一张静态图片，如此循环。

## 5. 添加滤镜与装饰 🎨

为了让“视频变照片”的对比更明显，我们可以为图片添加滤镜和装饰。

以下是美化图片的步骤：

1.  **添加滤镜**：选中一张定格图片，点击下方「滤镜」，选择一个你喜欢的风格（如“影视级”系列）。可以调整滤镜强度，让颜色变化更明显。为每一张定格图片应用不同的滤镜，以增强变化感。
2.  **添加文字**：在定格图片出现的位置，点击「文本」-「新建文本」，输入你想添加的字幕。
3.  **添加贴纸**：点击「贴纸」，在搜索框中输入关键词（如“电影”），选择喜欢的贴纸添加到画面中。可以拖动调整贴纸的位置和大小。

**关键点**：确保所有文字和贴纸的时长都与它们所属的定格图片完全对齐。可以通过分割和删除来调整其长度。

## 6. 最终检查与导出 ✅

完成所有编辑后，请完整播放一遍视频，检查：
*   卡点是否准确。
*   定格图片、文字、贴纸的时长是否对齐。
*   滤镜效果是否满意。

确认无误后，即可导出你的作品。

## 课程总结

本节课中，我们一起学习了如何制作“视频变照片”卡点效果。核心要点包括：
*   使用 **`定格`** 功能将视频变为静态图片。
*   通过观察**波形图**和手动踩点来找准音乐节奏。
*   运用**滤镜**改变图片色调，利用**文字**和**贴纸**增加画面信息与趣味性。

课后作业是运用本节课所学，独立完成一条属于自己的视频变照片卡点视频。多加练习，你就能熟练掌握这些技巧。
