# 拼多多运营：P14：拼多多个人店铺入驻流程 🛒

在本节课中，我们将要学习拼多多个人店铺的完整入驻流程。整个过程简单直接，只需每天投入少量时间，即使是零基础的电商新手也能快速完成开店。

## 概述

上一节我们介绍了拼多多运营的基础概念，本节中我们来看看如何从零开始，完成个人店铺的注册与开通。

![](img/c10e10abd1203581c9373ff813693b7c_1.png)

## 第一步：访问商家入驻后台

![](img/c10e10abd1203581c9373ff813693b7c_2.png)

首先，需要打开拼多多的商家入驻后台。以下是官方后台链接：
```
https://mms.pinduoduo.com/
```
如果找不到该链接，也可以在百度中搜索“拼多多商家入驻”来找到入口。



![](img/c10e10abd1203581c9373ff813693b7c_1.png)

## 第二步：登录与选择入驻类型

![](img/c10e10abd1203581c9373ff813693b7c_4.png)

打开链接后，会进入登录页面。需要填写个人手机号码，填写完毕后点击“0元入驻”按钮。



![](img/c10e10abd1203581c9373ff813693b7c_2.png)

登录成功后，进入后台页面。这里需要选择入驻类型。

以下是两种主要类型：
*   **普通入驻**：适用于大部分没有营业执照的个人卖家，开设**个人店**。
*   **一般贸易入驻**：适用于有进出口资质的商家。

![](img/c10e10abd1203581c9373ff813693b7c_6.png)

对于新手，我们选择“普通入驻”下的“个人店”，然后点击“下一步”。



![](img/c10e10abd1203581c9373ff813693b7c_4.png)

## 第三步：填写店铺信息与身份认证

进入此步骤后，需要完善店铺的基本信息和进行身份认证。

以下是需要填写和准备的内容列表：
1.  **店铺名称**：为你的店铺起一个名字，例如“女装小店123”。注意名称不能包含“企业”、“旗舰店”等字样。
2.  **登录密码**：设置一个包含**大写字母**和**小写字母**的密码，例如 `Pdd123456`。
3.  **主营类目**：选择你计划销售的商品类目。如果暂时不确定，可以先任意选择一个，因为后期有**一次**修改类目的机会。
4.  **身份证信息**：按照页面提示，上传开店人本人的身份证正反面照片。
5.  **基础信息**：填写个人信息。“是否为出口商家”一项选择“**否**”。
6.  **其他信息**：“第三方店铺链接”和“入驻邀请码”可以暂时不填。

最后，勾选同意协议，点击“提交”。



![](img/c10e10abd1203581c9373ff813693b7c_6.png)

## 第四步：完成注册与登录

信息提交后，注册流程就基本完成了。之后，可以直接使用你设置的手机号和密码，在商家后台登录，开始管理你的店铺。



![](img/c10e10abd1203581c9373ff813693b7c_8.png)

![](img/c10e10abd1203581c9373ff813693b7c_8.png)

![](img/c10e10abd1203581c9373ff813693b7c_10.png)

## 流程总结

本节课中我们一起学习了拼多多个人店铺的入驻流程。整个过程可以总结为以下四个核心步骤：

1.  **打开链接**：访问拼多多商家入驻后台。
2.  **登录选择**：输入手机号，选择“普通入驻”类型。
3.  **填写认证**：设置店铺名、密码、选择类目，并完成身份证上传。
4.  **提交登录**：提交所有信息后，即可登录商家后台。



![](img/c10e10abd1203581c9373ff813693b7c_10.png)
