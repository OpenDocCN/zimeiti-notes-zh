# 微信视频号运营教程：P27：如何扒取直播间话术

在本节课中，我们将学习一个实用的技巧：如何高效地扒取其他成功直播间的话术，并将其转化为自己的直播脚本。这对于初学者快速上手直播带货至关重要。

上一节我们介绍了视频内容与选品的思路，本节中我们来看看如何借鉴他人的成功经验，特别是直播过程中的语言组织。

## 核心工具：飞书妙计

扒取话术需要使用一个工具：**飞书**。具体而言，是飞书应用内的“飞书妙计”功能。这个工具可以实时录制并转写语音内容。

## 操作步骤详解

以下是使用飞书妙计扒取直播间话术的完整流程。

1.  **打开工具**
    在飞书App中，点击底部导航栏的“工作台”。在工作台页面中，找到并点击“飞书妙计”功能。

2.  **开始录制**
    进入飞书妙计后，点击界面右下角的**语音按钮**，即可开始录音。此时，你可以切换到微信视频号，打开你想要学习和模仿的直播间。

3.  **录制过程**
    飞书妙计会实时录制直播间主播的声音（包括背景音乐和互动），并将其转换为文字。建议录制**约15分钟**，这足以覆盖产品介绍、互动和逼单等核心环节。

    ![](img/5a78c57b52ddaadd3ad31d1b051edffd_1.png)

4.  **结束与导出**
    录制完成后，点击“完成”结束。系统需要几分钟时间处理音频并生成完整的文字稿。处理完毕后，点开这条录音记录。

5.  **分享与复制文本**
    在记录详情页，点击“分享”按钮。选择“可编辑”或“可阅读”模式，然后复制生成的链接。将此链接发送到微信聊天窗口并打开。

    ![](img/5a78c57b52ddaadd3ad31d1b051edffd_5.png)

6.  **整理与提炼**
    在微信中打开链接后，你将看到直播话术的完整文字版。你可以复制全部或部分文本，粘贴到文档编辑工具（如Word、备忘录）中。接下来，需要对这些原始话术进行编辑、提炼和重组，形成适合你自己产品和风格的新脚本。

## 话术的应用

提炼出的话术，可以写在纸上，或存入手机备忘录。在你自己开播时，如果忘记台词，可以对照这份脚本进行讲述，从而保证直播流程的顺畅和专业。

![](img/5a78c57b52ddaadd3ad31d1b051edffd_5.png)

本节课中我们一起学习了如何利用“飞书妙计”工具，高效扒取并借鉴他人直播间的优秀话术。掌握这个方法，你可以快速积累直播语言素材，降低起步难度，为打造自己的成功直播间打下坚实基础。
