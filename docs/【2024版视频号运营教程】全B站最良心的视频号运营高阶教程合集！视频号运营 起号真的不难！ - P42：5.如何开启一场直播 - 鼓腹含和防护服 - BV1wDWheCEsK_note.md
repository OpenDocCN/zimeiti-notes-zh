# 视频号运营教程：P42：如何开启一场直播 🎥

在本节课中，我们将学习如何从零开始，在视频号平台成功开启一场直播。我们将详细讲解直播前的准备工作、直播中的关键设置与操作，以及直播结束后的收尾工作。

## 直播前的准备工作

![](img/c3cc077b781ca9ea25bf914e7666ee7b_1.png)

上一节我们介绍了视频内容与选品，本节中我们来看看如何为直播做具体准备。

确定好商品并发布相关视频后，需要等待视频获得约1000的播放量。此时，应提前准备好直播话术。同时，为防止直播开始时观众进入缓慢，需要提前将用于补发引流的视频准备到草稿箱。

![](img/c3cc077b781ca9ea25bf914e7666ee7b_3.png)

![](img/c3cc077b781ca9ea25bf914e7666ee7b_1.png)

如果视频热度上升时，你暂时没有时间或不方便立即直播，务必先创建一个直播预告。预告需明确你计划开播的时间。这样，感兴趣的观众可以提前预约。当你开播时，预约过的观众会第一时间收到提醒。

![](img/c3cc077b781ca9ea25bf914e7666ee7b_3.png)

![](img/c3cc077b781ca9ea25bf914e7666ee7b_5.png)

## 发起直播与基础设置

![](img/c3cc077b781ca9ea25bf914e7666ee7b_7.png)

准备工作就绪后，即可进入开播环节。以下是发起直播的步骤。

首先，点击“发起直播”，然后选择“直播”功能。

![](img/c3cc077b781ca9ea25bf914e7666ee7b_5.png)

![](img/c3cc077b781ca9ea25bf914e7666ee7b_9.png)

在直播设置页面，需要填写直播主题。一个常见的主题格式是“视频同款热卖中”。你也可以参考其他成功账号的写法。接着，在“商品”区域，提前添加好你计划售卖的商品。

![](img/c3cc077b781ca9ea25bf914e7666ee7b_7.png)

添加商品时，一般选择你想主推的单品即可。添加完成后返回。

![](img/c3cc077b781ca9ea25bf914e7666ee7b_11.png)

在“画面”设置中，可以进行美颜、添加滤镜或人物特效等操作。关于“贴图”，可以参考上节课的内容，使用“微微商水印相机”等工具制作简单的介绍贴图。对标账号的做法是很好的参考。

![](img/c3cc077b781ca9ea25bf914e7666ee7b_9.png)

![](img/c3cc077b781ca9ea25bf914e7666ee7b_13.png)

## 直播中的关键操作

![](img/c3cc077b781ca9ea25bf914e7666ee7b_15.png)

设置完成后，就可以开始直播了。但直播过程中有几个关键操作能有效提升效果。

“更多”选项里有很多功能，但对于我们以卖货为主的直播，大部分不需要调整。为了教学演示，我将直播模式改为“仅自己可见”，以避免观众进入影响讲解。

![](img/c3cc077b781ca9ea25bf914e7666ee7b_17.png)

![](img/c3cc077b781ca9ea25bf914e7666ee7b_11.png)

![](img/c3cc077b781ca9ea25bf914e7666ee7b_19.png)

![](img/c3cc077b781ca9ea25bf914e7666ee7b_13.png)

![](img/c3cc077b781ca9ea25bf914e7666ee7b_15.png)

![](img/c3cc077b781ca9ea25bf914e7666ee7b_21.png)

实际卖货直播中，需要标记地理位置。你可以任意标记一个位置，然后点击“开始直播”。

![](img/c3cc077b781ca9ea25bf914e7666ee7b_23.png)

![](img/c3cc077b781ca9ea25bf914e7666ee7b_17.png)

![](img/c3cc077b781ca9ea25bf914e7666ee7b_19.png)

直播开始后，当你讲解到某个商品时，**必须点击打开购物车中对应的商品**。这样，观众的画面中才会弹出该商品卡片，这是提升转化率的关键。

![](img/c3cc077b781ca9ea25bf914e7666ee7b_25.png)

![](img/c3cc077b781ca9ea25bf914e7666ee7b_21.png)

直播时无需过度关注复杂数据，专注于你的话术即可。贴图也可以在开播后添加或调整。此外，可以播放一些背景音乐，这通常不影响直播。

如果某位观众的评论对你特别有利，你可以长按该评论，然后选择“点上墙”。这样，新进入直播间的观众就能看到这条优质评论。

直播过程中，务必注意不要使用平台规定的违规词汇。

## 结束直播与后续

直播结束后，需要正确关闭并处理后续事宜。

直播结束时，点击右上角的关闭按钮即可结束直播。在直播过程中，你也可以将直播间链接分享给他人。

结束直播后，系统会提示你是否生成直播回放，你可以根据需要选择“生成”。

![](img/c3cc077b781ca9ea25bf914e7666ee7b_27.png)

![](img/c3cc077b781ca9ea25bf914e7666ee7b_23.png)

本节课中我们一起学习了开启一场视频号直播的全流程：从开播前的预告与素材准备，到直播中的商品讲解、互动功能使用，再到最后的结束与回放生成。掌握这些步骤，你就能更从容地开启自己的直播之旅。
