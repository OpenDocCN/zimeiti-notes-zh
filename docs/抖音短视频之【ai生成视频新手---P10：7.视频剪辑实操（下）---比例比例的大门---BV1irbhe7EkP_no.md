# AI短视频创作教程：P10：7.视频剪辑实操（下） 🎬

在本节课中，我们将继续学习视频剪辑的实操部分，重点讲解如何为多个视频片段添加转场效果，以及如何为视频添加字幕。这些步骤对于提升视频的流畅度和专业性至关重要。

上一节我们介绍了视频比例和背景的设置，本节中我们来看看如何让多个视频片段之间衔接得更自然。

## 添加转场效果

![](img/378ece7fbaf9cb2c474c5c76ade37eff_1.png)

我们的视频通常由多个约10秒的片段组成，并非一个完整的原始视频。这些片段之间需要有自然的衔接，因此必须添加转场效果。

![](img/378ece7fbaf9cb2c474c5c76ade37eff_3.png)

转场的作用是让用户明确感知到不同片段的内容差异。如果不添加转场，观众在切换到下一个片段时可能会感到突兀，甚至退出视频查看详情。这种行为会直接导致视频的**完播率**下降，而完播率会影响视频的推荐权重和最终收益。

![](img/378ece7fbaf9cb2c474c5c76ade37eff_5.png)

以下是添加转场效果的具体步骤：

![](img/378ece7fbaf9cb2c474c5c76ade37eff_7.png)

1.  在视频编辑轨道上，找到右侧的“加号”按钮并点击。
    ![](img/378ece7fbaf9cb2c474c5c76ade37eff_5.png)

![](img/378ece7fbaf9cb2c474c5c76ade37eff_9.png)

2.  在弹出的菜单中，选择“素材库”选项。
    ![](img/378ece7fbaf9cb2c474c5c76ade37eff_7.png)

![](img/378ece7fbaf9cb2c474c5c76ade37eff_11.png)

3.  在素材库的“热门”分类中，可以找到丰富的转场素材，例如各种动态效果。
    ![](img/378ece7fbaf9cb2c474c5c76ade37eff_11.png)

![](img/378ece7fbaf9cb2c474c5c76ade37eff_13.png)

在选择转场时，请注意一个核心原则：**转场时长不宜过长**。避免选择长达16秒的转场，因为观众通常没有耐心观看不重要的内容。最合适的转场时长应保持在 **3到5秒** 之间。

![](img/378ece7fbaf9cb2c474c5c76ade37eff_15.png)

我们选择一个3秒的转场素材添加进去。添加后，转场视频也必须遵循统一的 **16:9** 横屏比例。在转场素材的编辑界面，找到“画布模糊”选项，选择第一个清晰的画面，背景就会自动适配为16:9的横屏。最后，点击右下角的对勾确认。

![](img/378ece7fbaf9cb2c474c5c76ade37eff_17.png)

## 添加视频字幕

完成转场设置后，下一步是为视频添加文案，即字幕。字幕能有效提升视频的信息传达效率和观看体验。

以下是添加字幕的具体步骤：

![](img/378ece7fbaf9cb2c474c5c76ade37eff_19.png)

1.  在手机编辑界面左下角找到“文字”选项并点击。
    ![](img/378ece7fbaf9cb2c76ade37eff_21.png)

![](img/378ece7fbaf9cb2c474c5c76ade37eff_21.png)

2.  选择“新建文本”，然后在右侧找到“识别字幕”功能并点击。
3.  点击“开始匹配”，系统会自动识别视频或音频中的人物对话，并生成对应的字幕文本。
4.  字幕生成后，可以进行微调。拖动字幕文本框右下角，可以放大字幕。然后将整个文本框拖动到视频最下方的位置，这样观看最舒适。
    ![](img/378ece7fbaf9cb2c474c5c76ade37eff_23.png)

调整好第一段字幕后，后续的字幕样式通常会同步应用，非常方便。

## 核心要点与总结

![](img/378ece7fbaf9cb2c474c5c76ade37eff_23.png)

本节课中我们一起学习了视频剪辑中两个关键的后处理步骤：添加转场和字幕。

*   **转场**：用于衔接不同视频片段，提升流畅度，保障完播率。选择**3-5秒**的转场素材，并确保其比例为 **16:9**。
*   **字幕**：通过“识别字幕”功能自动生成，调整位置和大小以优化观看体验。

![](img/378ece7fbaf9cb2c474c5c76ade37eff_25.png)

整个视频制作过程需要严格按照步骤操作：从寻找合适时长的素材，到设置统一比例，再到添加转场和字幕。平台审核会关注视频的规范性和质量，严谨的制作流程能有效提高审核通过率。记住，视频制作并不难，关键在于细心和遵循正确的步骤。

![](img/378ece7fbaf9cb2c474c5c76ade37eff_27.png)
