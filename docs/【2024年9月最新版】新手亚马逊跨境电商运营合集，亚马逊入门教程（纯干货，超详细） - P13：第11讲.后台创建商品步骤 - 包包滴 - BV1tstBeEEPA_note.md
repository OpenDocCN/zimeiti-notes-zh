# 亚马逊跨境电商运营：P13：后台创建商品步骤 🛒

在本节课中，我们将学习如何在亚马逊卖家后台手动创建商品。我们将详细介绍从填写商品信息、处理品牌与UPC问题，到上传变体以及解决商品禁止显示等关键步骤，确保您能成功上架第一个产品。

上一节我们介绍了库存管理的基础知识，本节中我们来看看如何从零开始创建一个新的商品链接。

## 商品信息填写

首先，我们需要进入卖家后台，将界面语言切换为中文以便操作。随后，在“库存”菜单下选择“添加新商品”。

以下是创建商品时需要填写的核心信息：

1.  **商品编码**：通常选择 **UPC** 或 **EAN**。如果您没有品牌，可以暂时使用系统提供的通用编码选项。
2.  **商品名称**：格式一般为 **品牌名 + 产品核心名称**。例如：`Anker Soundcore 蓝牙耳机`。
3.  **品牌**：如果尚未完成品牌注册，可以选择“无品牌”或填写 `N/A`、`Generic` 等。**制造商**栏可以预先填写您计划注册的商标名称，为日后品牌备案后的修改提供便利。
4.  **商品关键属性**：根据产品实际情况填写，如“是否有保质期”通常选“否”，“包装数量”通常填 `1`。
5.  **报价**：填写您的商品价格（如 `10` 美元）和初始库存数量（如 `10` 个）。
6.  **配送渠道**：初期建议先选择“卖家自行配送”，创建成功后再在库存管理中转换为“亚马逊配送（FBA）”。这只是一个操作习惯，您也可以直接选择亚马逊配送。

## 商品图片上传

图片是影响转化的重要因素。亚马逊通常支持上传最多9张图片。

*   **主图要求**：必须为纯白背景（RGB值：255，255，255），展示产品主体。
*   **辅图作用**：展示细节、功能、尺寸对比或使用场景。
*   **变体图片**：如果商品有变体（如不同颜色），除了为每个子体上传图片外，建议在**变体主题对应的图片栏**中上传一张统一格式的主图。这能确保在前台变体选择区展示的缩略图风格一致，提升页面美观度。

## 创建商品变体

如果您的商品有不同颜色、尺寸等属性，需要创建变体。

![](img/3dc60eabb2a58e2fa9f19b8d22bcc643_13.png)

![](img/3dc60eabb2a58e2fa9f19b8d22bcc643_14.png)

![](img/3dc60eabb2a58e2fa9f19b8d22bcc643_15.png)

以下是创建变体的步骤：

![](img/3dc60eabb2a58e2fa9f19b8d22bcc643_17.png)

1.  在“重要信息”部分，找到“变体”选项并选择“是”。
2.  选择变体主题，如“颜色-尺寸”。
3.  在下方出现的表格中，为每个变体组合（如“黑色-XL”、“灰色-L”）填写独立的**子体SKU**、价格、库存和UPC码。
4.  **SKU命名技巧**：建议采用有规律的命名方式，例如：
    *   父体SKU：`ProductName_Set`
    *   子体SKU：`ProductName_Color_Size`
    这便于日后在库存中快速识别和管理。

## 商品上架与问题修复

![](img/3dc60eabb2a58e2fa9f19b8d22bcc643_19.png)

![](img/3dc60eabb2a58e2fa9f19b8d22bcc643_21.png)

![](img/3dc60eabb2a58e2fa9f19b8d22bcc643_22.png)

点击“保存并完成”后，商品可能不会立即上架成功。

![](img/3dc60eabb2a58e2fa9f19b8d22bcc643_24.png)

1.  **检查“管理库存”**：新创建的商品可能显示为“不可售”状态。
2.  **处理“禁止显示”**：点击“禁止显示”列表或“编辑”按钮，系统会提示具体原因，常见问题包括：
    *   缺少“商品描述”（必填项）。
    *   缺少主图或图片不符合规范。
3.  **使用“修复商品”功能**：根据提示，在“修复商品”页面补全缺失信息（如长描述、图片），并重新提交。信息补全后，商品状态将变为“在售”。
4.  **优化商品信息质量**：进入“商品信息控制面板”，查看亚马逊提供的“改进建议”。补全更多属性（如材质、保养说明等）可以提升商品信息质量，有助于获得更多曝光。

![](img/3dc60eabb2a58e2fa9f19b8d22bcc643_26.png)

本节课中我们一起学习了在亚马逊后台手动创建商品的完整流程。关键点在于准确填写商品信息、正确处理品牌/UPC报错、规范上传图片，以及为多属性商品创建清晰的变体。完成创建后，务必通过“修复商品”功能解决任何“禁止显示”问题，并利用“商品信息控制面板”持续优化Listing，为商品获得更好的搜索排名和曝光打下坚实基础。
