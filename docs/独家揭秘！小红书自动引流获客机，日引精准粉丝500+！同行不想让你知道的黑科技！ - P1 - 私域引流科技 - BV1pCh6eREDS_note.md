# 小红书自动引流教程：P1：软件配置与基础操作 🚀

在本节课中，我们将学习如何使用新版小红书自动采集与曝光软件，包括软件启动、账号登录、浏览器配置以及基础的文章与评论采集功能。

## 概述

本教程将详细介绍软件的安装、初始配置以及核心的采集功能操作。我们将从获取软件开始，逐步完成环境设置，并演示如何通过关键词或作品链接来采集目标用户。

![](img/ae51b3f118ec6201e99ddf240bfb4c18_1.png)

## 软件启动与登录

首先，你需要下载并解压提供的软件文件包。软件主程序名为“红薯采集1.19”。

**步骤一：启动软件并输入密码**
首次打开软件时，需要输入内部密码。内部密码为：`灵动共创牛逼`。输入后即可进入登录界面。

**步骤二：登录或注册账号**
如果你已有账号，直接输入登录即可。如果没有账号，需要在界面中完成注册。

![](img/ae51b3f118ec6201e99ddf240bfb4c18_1.png)

## 浏览器配置

上一节我们完成了账号登录，本节中我们来看看新版软件的一个重要变化：外置浏览器的配置。

新版软件将内置浏览器改为外置浏览器，因此需要手动配置浏览器路径。

以下是配置浏览器的具体步骤：
1.  建议使用提供的“百分百浏览器”安装包进行安装。
2.  在电脑上找到浏览器的快捷方式，右键点击并选择“属性”。
3.  在属性窗口中，复制“目标”栏中的完整路径。
4.  将复制的路径粘贴到软件指定的“浏览器路径”输入框中。

软件包中已包含浏览器的安装文件。配置完成后，你还需要设置“浏览器端口”。每个独立的端口对应一个独立的浏览器会话，这允许你同时运行多个软件实例或多个账号。端口号建议从 `4666` 开始顺序设置，例如 `4666`、`4667`。

设置好路径和端口后，点击“启动浏览器”，软件将自动打开小红书登录页面。在此登录你的小红书账号。

## 内容采集操作

成功登录账号后，我们就可以开始使用软件的核心功能了。软件提供两种采集目标用户的方式：通过特定作品链接采集，或通过关键词搜索文章后再采集。

**方式一：通过作品链接采集**
1.  在小红书APP或网页中找到目标作品。
2.  复制该作品的链接。
3.  将链接粘贴到软件中指定的输入框，即可采集该作品下的所有评论用户。

**方式二：通过关键词采集（演示）**
我们将以引流“创业粉”为例，演示通过关键词搜索并采集的流程。

以下是操作步骤：
1.  在软件的“文章搜索关键词”设置中，输入行业关键词，例如“创业搞钱”。
2.  点击“文章采集”，软件会自动搜索并采集相关文章。采集速度很快，演示时可随时点击“停止采集”。
3.  采集到文章后，需要设置“评论关键词”来筛选精准用户。例如，针对创业粉，可以设置“求带”、“怎么做”等关键词。
4.  设置好评论关键词后，在文章列表中全选目标文章。
5.  点击“采集文章评论”，软件将自动抓取符合关键词条件的评论用户。

![](img/ae51b3f118ec6201e99ddf240bfb4c18_3.png)

**注意**：采集时如果遇到用户信息（昵称、红书号）缺失的情况，说明当前账号可能被风控。此时需要更换账号或通过“滑块验证”解除风控。

## 自动关注功能

![](img/ae51b3f118ec6201e99ddf240bfb4c18_5.png)

用户采集完成后，即可进行自动关注操作。

在采集到的用户列表中，点击“全选”。你可以在软件中设置“关注间隔时间”，为了账号安全，建议将时间设置得长一些（例如60秒以上）。本次演示设置为5-10秒。点击“开始关注”，软件便会自动执行关注操作。关注过程中如遇到“滑块验证”，手动通过即可。

![](img/ae51b3f118ec6201e99ddf240bfb4c18_3.png)

## 实时关注功能应用

除了基础的“采集后关注”模式，软件还提供了更高效的“实时关注”功能。它可以配合采集软件，实现边采集边关注，也支持多账号循环操作。

**实时关注软件配置**
1.  打开文件包中的“实时关注”软件，点击“立即登录”。
2.  同样需要配置“浏览器路径”（与主软件配置方法一致）。
3.  **关键步骤**：必须修改“浏览器端口号”，不能与主采集软件相同。例如主软件用 `4666`，这里可改为 `4667`。
4.  点击“启动浏览器”，登录一个新的小红书账号（**注意：不能与采集软件使用同一账号**）。

**获取并添加账号CK**
实时关注软件需要通过CK（Cookie）登录账号。

以下是提取CK的步骤：
1.  在登录了小红书账号的浏览器页面，按下键盘 `F12` 打开开发者工具。
2.  切换到 `Network` （网络）标签页。
3.  按 `F5` 刷新页面，等待网络请求加载完成。
4.  在请求列表中找到以 `EX` 开头的请求项，点击它。
5.  在右侧详情中切换到 `Headers` （标头）标签页。
6.  向下滚动，找到 `Cookie` 字段，将其冒号后的全部内容复制下来。
7.  在实时关注软件中，点击“添加账号”，粘贴复制的CK内容。

![](img/ae51b3f118ec6201e99ddf240bfb4c18_7.png)

![](img/ae51b3f118ec6201e99ddf240bfb4c18_5.png)

**添加多个操作账号**
如果你需要多个账号同时运行，可以在浏览器中操作：
1.  在浏览器标签页上右键，选择“在右侧新增小号标签页”。
2.  将小红书网址复制到新标签页并打开，登录另一个账号。
3.  重复上述提取CK和添加账号的步骤即可。

![](img/ae51b3f118ec6201e99ddf240bfb4c18_9.png)

![](img/ae51b3f118ec6201e99ddf240bfb4c18_10.png)

![](img/ae51b3f118ec6201e99ddf240bfb4c18_7.png)

**重要提示**：添加完所有账号后，请关闭浏览器中除主标签页外的所有额外小号标签页。

**软件联动与设置**
在实时关注软件中，需要进行关键设置：
*   **操作间隔时间**：建议关注操作间隔设置在60秒以上以保证安全。
*   **单号关注多少切换**：即使只有一个账号也必须设置，例如设为 `100`，表示一个账号关注100个用户后才会切换或停止（如果只有一个账号）。
*   **勾选账号**：在账号列表中，勾选你准备执行操作的账号。

![](img/ae51b3f118ec6201e99ddf240bfb4c18_9.png)
![](img/ae51b3f118ec6201e99ddf240bfb4c18_10.png)

![](img/ae51b3f118ec6201e99ddf240bfb4c18_12.png)

**双软件配合工作流程**
1.  保持实时关注软件在运行状态并完成上述设置。
2.  打开主采集软件，在其界面上找到并勾选“实时关注”选项。
3.  在主采集软件中执行“采集文章评论”操作。
4.  一旦采集到新的用户UID，实时关注软件便会立即开始自动关注，实现“边采边关”的流水线作业。

![](img/ae51b3f118ec6201e99ddf240bfb4c18_12.png)

**新版优化**：当前版本已加入“判断已关注用户”功能，避免对同一用户重复关注。

## 总结

本节课中我们一起学习了新版小红书引流软件的完整操作流程。我们从软件登录和浏览器配置开始，掌握了通过链接和关键词两种方式采集目标用户的方法，并详细讲解了自动关注功能。最后，我们深入探讨了更高效的“实时关注”功能，学习了如何配置多账号以及如何与主采集软件联动工作，从而实现自动化、批量的引流操作。如果在操作中遇到任何问题，可以通过指定渠道联系获取帮助。软件后续将持续更新优化。
