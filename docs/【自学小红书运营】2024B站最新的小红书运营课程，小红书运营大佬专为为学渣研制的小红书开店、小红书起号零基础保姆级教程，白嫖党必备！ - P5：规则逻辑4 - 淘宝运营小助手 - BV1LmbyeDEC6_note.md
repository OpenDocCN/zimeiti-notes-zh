## 推荐规则解析 🔍

![](img/a8f96f429c9ed2e9d572abd840f32bf0_9.png)

![](img/a8f96f429c9ed2e9d572abd840f32bf0_10.png)

![](img/a8f96f429c9ed2e9d572abd840f32bf0_11.png)

上一节我们介绍了课程的整体框架，本节中我们来看看小红书平台的具体推荐规则。推荐规则主要分为用户行为、商品秩序和图文影响三个方面。

以下是推荐规则的具体要点：

1.  **用户行为限制**
    *   **禁止行为**：内容中不得含有明确的营销词汇，例如：`微信`、`二维码`、`代购`、`水印`、`某某店铺`。含有这些词汇的作品不会被系统推荐，多次发布可能涉嫌违规。
    *   **允许行为**：
        *   个人允许留下邮箱，这是一种较慢的引流方式。
        *   微博账号需使用谐音，不能直接展示。
        *   可以使用话术引导用户联系微信，但不能直接展示微信号或二维码。具体做法是将微信号用数字、图标、表情等隔开。建议每天引导次数不超过3次，超过15次则存在风险。

2.  **商品秩序规范**
    *   不得提供医疗建议或投资建议，例如推荐保健品、医疗用品或指导治病。
    *   不得宣称护肤品具有治疗效果或夸大使用效果。
    *   同一设备最多登录3个账号。手机应独立使用，避免连接公共WiFi。

3.  **图文内容影响**
    *   不得诱导用户，例如进行“关注领奖”等虚假宣传。
    *   **内容原创度为最佳**。复制他人内容虽然可能获得曝光，但流量通常只有原创内容的 **1% 到 20%**。例如，原创笔记有2万阅读，复制后最多可能获得4000阅读。二次发布的内容流量会大幅衰减。

## 被推的核心逻辑 🚀

![](img/a8f96f429c9ed2e9d572abd840f32bf0_13.png)

了解了基础推荐规则后，我们进一步探讨笔记如何才能被系统推荐的核心逻辑。这主要分为“推广笔记”和“推广须知”两部分。

以下是“推广笔记”的四种形式：

1.  **品牌或相关方赠送**：品牌方提供的合作笔记，可获得额外流量扶持。
2.  **付费推广**：直接付费购买流量。但对新账号效果有限，可能浪费资金。
3.  **激励试用**：参与平台官方活动，完成任务后可获得流量奖励。
4.  **晒单/抽奖**：发布相关活动笔记，可能获得间接流量扶持。

**作用说明**：这些方式能为新笔记提供初始曝光加成。例如，一篇新笔记的基础曝光量可能在1000-2000，通过参与活动可能提升至2500-3000。前期作用明显，后期效果递减。

接下来是必须遵守的“推广须知”，以避免处罚：

以下是关键的推广行为规范：

1.  **内容垂直**：笔记内容需与账号定位领域一致。跨领域发布（如女装账号发美妆笔记）会导致流量锐减 **80%-90%**，并可能损害账号后续流量权重。
2.  **避开长效产品**：避免推广需要长期使用才能体现效果的产品（如某些保健品），这类宣传在小红书效果不佳且易引发反感。
3.  **发布间隔**：两篇推广笔记发布至少间隔 **3小时** 以上。连续发布会导致第二篇笔记占用第一篇的流量，曝光量大幅降低。
4.  **禁止刷数据**：严禁人工干预数据（刷赞、藏、评）。平台虽非实时严查，但风险极高。
5.  **违规处罚**：
    *   首次违规会收到“管家警告”，笔记流量受限。
    *   单月内违规笔记占比超过 **20%**，账号将受到严厉惩罚，可能基本废置。
