# 小红书运营实操课：P4：第3课：小红书店铺后台的基础设置 📝

![](img/404473ae20b0b488a99e9b84a8de9769_1.png)

在本节课中，我们将学习如何设置小红书店铺的后台。这是店铺运营的基础，正确的设置能确保后续订单处理、资金提现和客户服务的顺利进行。

## 概述

我们将分步讲解店铺信息、物流、资金和售后等核心模块的设置方法。每一部分都至关重要，请确保按照指引完成。

## 1. 店铺信息公示设置

首先，我们需要进入店铺后台，完成基础信息的公示设置。

1.  打开小红书，点击进入“店铺”后台。
2.  找到“信息公示”板块。
3.  填写联系人信息，可以随意填写。
4.  填写你自己的手机号码作为公示联系方式。
5.  填写经营地址，可以是家庭住址或公司地址。

![](img/404473ae20b0b488a99e9b84a8de9769_7.png)

![](img/404473ae20b0b488a99e9b84a8de9769_1.png)

![](img/404473ae20b0b488a99e9b84a8de9769_9.png)

接下来，在“基因内部”部分，需要勾选所有选项。

![](img/404473ae20b0b488a99e9b84a8de9769_3.png)

以下是需要勾选的步骤：
*   勾选全部四个选项。
*   点击“保存”按钮完成设置。

## 2. 物流地址与模板设置

上一节我们完成了店铺信息设置，本节中我们来看看物流相关的配置。正确的物流设置能提升发货效率并影响资金回笼周期。

![](img/404473ae20b0b488a99e9b84a8de9769_15.png)

首先设置发货与退货地址。

1.  点击后台的“物流工具”。
2.  在地址库中“新建地址”。
3.  填写详细的发货地址信息，包括国家、省市区、街道以及具体的门牌号（如几幢几单元几零几）。地址不详细会影响客户确认收货后你的提现周期。
4.  填写你的手机号，座机号可不填。
5.  点击“提交”保存地址。

![](img/404473ae20b0b488a99e9b84a8de9769_7.png)
![](img/404473ae20b0b488a99e9b84a8de9769_9.png)
![](img/404473ae20b0b488a99e9b84a8de9769_11.png)
![](img/404473ae20b0b488a99e9b84a8de9769_13.png)

**地址填写建议：**
*   **发货地址**：可以先填写一个热门发货地区，例如义乌或广州。
*   **退货地址**：初期可以先填写你自己的地址。后续若发生退换货，再将其修改为厂家的地址。

接下来，设置运费模板，这决定了向不同地区客户收取的运费。

1.  在“物流工具”中找到“运费模板”。
2.  点击“新建运费模板”。
3.  模板名称可随意填写（如123456），此名称仅你自己可见。
4.  运费类型选择“买家承担运费”。
5.  计费方式选择“按件计费”。
6.  设置默认运费，例如：首件 `10` 元，每增加一件加 `5` 元。
7.  点击“编辑”并“确定”保存。

![](img/404473ae20b0b488a99e9b84a8de9769_15.png)

![](img/404473ae20b0b488a99e9b84a8de9769_17.png)

**运费设置说明：**
系统默认的“非偏远地区包邮模板”通常不配送新疆、西藏、宁夏、青海、海南、内蒙古、甘肃等地区。如果你想对这些地区单独设置运费，就通过上述新建模板的方式操作。在上架商品时，选择你自定义的这个运费模板即可。

## 3. 资金账户与保证金

![](img/404473ae20b0b488a99e9b84a8de9769_21.png)

物流设置完成后，我们需要绑定资金账户，这是店铺收款和提现的关键。

![](img/404473ae20b0b488a99e9b84a8de9769_23.png)

1.  点击后台的“资金”管理。
2.  进入“货款资金”设置。
3.  绑定银行卡。注意：用谁的身份证开店，提现时就需绑定谁的银行卡。
4.  绑定微信支付账户。需要用开店人的实名微信号扫码认证。
5.  绑定支付宝账户。

![](img/404473ae20b0b488a99e9b84a8de9769_25.png)

![](img/404473ae20b0b488a99e9b84a8de9769_19.png)
![](img/404473ae20b0b488a99e9b84a8de9769_21.png)

**重要提示：**
*   必须缴纳店铺保证金后，才能成功开通微信支付账户。否则顾客无法使用微信购买你的商品。
*   通过微信或支付宝提现，平台会扣除 `0.7%` 的手续费。

## 4. 售后与运费险设置

最后，我们来配置售后相关功能，包括运费险和退货地址策略，这直接影响客户体验和店铺转化率。

首先，了解运费险的开通与关闭。

![](img/404473ae20b0b488a99e9b84a8de9769_31.png)

1.  在后台找到“售后”管理，进入“运费保”设置。
2.  开通后，每卖出一单，平台会自动扣除约 `0.46` 元的运费险费用。
3.  **开通建议**：利润高的产品建议开通，可以提升店铺转化率；利润低的产品（如9.9元百货）可以不开通。
4.  如需关闭，在设置页面点击“关闭”并确认即可。

![](img/404473ae20b0b488a99e9b84a8de9769_33.png)

![](img/404473ae20b0b488a99e9b84a8de9769_27.png)
![](img/404473ae20b0b488a99e9b84a8de9769_29.png)
![](img/404473ae20b0b488a99e9b84a8de9769_31.png)
![](img/404473ae20b0b488a99e9b84a8de9769_33.png)

接下来，设置多地址退换货策略。当你有多个供货厂家时，此功能可自动将退货引导至对应厂家。

1.  在“订单”或“物流工具”中找到“多地址退换货”。
2.  点击“创建策略”。
3.  填写策略名称，建议使用详细的产品词（例如：“冬季加厚棉服”）。
4.  从地址库中选择对应厂家的退货地址。
5.  关联该策略所适用的所有商品SKU。
6.  点击“创建”完成。

设置后，当用户申请“七天无理由退换货”并获系统同意时，系统会自动将你设置的厂家地址推送给买家，方便其直接退货。

## 总结

本节课中，我们一起学习了小红书店铺后台的四大基础设置：
1.  **店铺信息公示**：完善联系方式和地址。
2.  **物流设置**：配置详细的发货/退货地址和自定义运费模板。
3.  **资金账户**：绑定银行卡、微信和支付宝，并注意保证金与手续费。
4.  **售后设置**：根据利润决定是否开通运费险，并设置多地址退货策略以高效处理售后。

![](img/404473ae20b0b488a99e9b84a8de9769_39.png)

完成以上设置，你的店铺就具备了正常运营的基础框架。请务必根据自身情况仔细配置每一项。
