# 自媒体运营教程：P8：7实操教学—剪辑合成 💻

在本节课中，我们将学习如何将翻译好的字幕与视频素材进行合成，并添加动态水印，最终完成一个可用于发布的视频作品。

上一节我们介绍了如何获取并翻译视频字幕，本节中我们来看看如何将字幕与视频结合，并进行最终的剪辑处理。

---

视频素材与字幕文件已准备就绪。首先，我们来检查翻译字幕的效果。

播放视频，观察字幕与画面的匹配程度。可以看到，翻译基本准确。对于中文观众而言，结合画面理解内容没有障碍。

**核心原则**：字幕无需逐字精确翻译，目标是让观众结合画面能理解大意。因此，可以删除部分不影响理解的英文原句或冗余词汇，只保留核心的中文信息。

确认字幕后，需要将其导出。我们只需要中文字幕文件。

![](img/83ba220a24624960073f69bc97f01a36_1.png)

以下是导出字幕的步骤：
1.  在翻译工具中，选择导出功能。
2.  在导出选项中，**仅勾选“中文字幕”**。
3.  将字幕文件保存到本地设备。

![](img/83ba220a24624960073f69bc97f01a36_3.png)

![](img/83ba220a24624960073f69bc97f01a36_5.png)

---

接下来，进入视频剪辑与合成环节。我们将使用“剪映”软件进行操作。

**重要提示**：请确保你使用的是**剪映专业版 3.0 或更高版本**。旧版本可能不支持字幕文件的导入功能。

以下是导入素材与字幕的步骤：
1.  打开剪映专业版，创建新项目。
2.  点击“导入”或直接将之前下载好的**视频素材**拖入素材库。
3.  将视频素材拖到下方的时间轴轨道上。
4.  在界面左侧功能栏找到“文本”选项。
5.  在文本菜单最下方，点击“**本地字幕导入**”。
6.  选择刚刚导出的中文字幕文件，将其导入到时间轴的字幕轨道上。

导入后播放预览，检查字幕是否与视频同步。

---

此时字幕可能大小或样式不合适，需要进行调整。

![](img/83ba220a24624960073f69bc97f01a36_7.png)

以下是调整字幕样式的步骤：
1.  在时间轴上选中字幕轨道。
2.  在右侧的编辑面板中，可以调整**字体、大小、颜色**等属性。
3.  软件也提供了多种“花字”预设样式，可直接选用。
4.  如果发现字幕断句不合理，可以在右侧面板中，根据中文语意**手动调整句子的分段位置**，使阅读更流畅。

调整至视觉效果舒适即可。

---

视频合成后，为防止他人搬运，需要添加专属水印。动态水印比静态水印防护效果更好。

以下是创建动态水印的步骤：
1.  在“文本”功能中，添加一个**默认文本**或选择一个**花字**。
2.  将其拖到时间轴轨道上，并输入你的名称（如“安雅看综艺”）。
3.  将水印文字调整到合适大小，并放置在视频角落（如左上角）。
4.  将水印图层的长度**拉至与视频时长一致**。
5.  在右侧编辑面板，适当降低水印的**不透明度**，使其不太突兀。
6.  制作动态效果：
    *   在时间轴起始处，选中水印图层。
    *   在右侧“位置”参数旁，点击**添加关键帧**图标。
    *   将播放指针移动到时间轴中间位置。
    *   将水印从**左上角拖动到右下角**，软件会自动生成第二个关键帧。
    *   再将播放指针移动到时间轴末尾。
    *   将水印从**右下角拖回左上角**，生成第三个关键帧。

播放预览，水印便会按照设定的路径在屏幕上循环移动。

---

本节课中我们一起学习了视频剪辑合成的完整流程。我们首先检查并导出了中文字幕，随后在剪映软件中将视频与字幕合成，并调整了字幕样式以提升观感。最后，为了保护原创性，我们学习了如何制作并添加一个简单的动态水印。

![](img/83ba220a24624960073f69bc97f01a36_9.png)

对于新手而言，可以从在小平台进行内容搬运和剪辑练习开始，按照上述步骤操作，逐步熟悉自媒体视频制作的全过程。
