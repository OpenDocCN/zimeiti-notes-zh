# 亚马逊广告归因周期结构打法：P1：核心概念与周期分析

在本节课中，我们将学习亚马逊广告归因的核心概念与周期分析。这是构建高效广告策略、打造爆款产品的第一步。我们将解析不同广告形式的底层逻辑，并深入理解亚马逊的广告归因周期模型。

## 概述

亚马逊平台上的头部卖家通常有系统化的广告布局策略。本节课程将分享一种适用于当前亚马逊环境的新广告归因周期打法。内容涵盖四个核心部分：不同广告形式的抓取逻辑、广告归因周期的具体分析、不同阶段的广告结构搭建方法，以及后续的广告数据分析。课程将结合具体实操案例进行讲解。



## 不同广告形式的抓取逻辑

首先，我们需要理解亚马逊几种主要广告形式是如何抓取流量和匹配用户的。这是后续所有策略的基础。

以下是三种核心广告类型的抓取逻辑：

1.  **商品推广**：系统基于用户搜索词与卖家设置的关键词、商品详情信息（如标题、五点描述）进行匹配，从而展示广告。
2.  **品牌推广**：通过品牌旗舰店、品牌视频或定制图片集合的形式，在搜索结果顶部或相关位置展示，旨在提升品牌认知和考虑度。
3.  **展示型推广**：通过**再营销**（针对浏览过你商品或类似商品的用户）和**受众拓展**（根据商品、兴趣或生活方式定位新客户）两种方式，在站内及站外网络进行展示。



## 广告归因周期的具体分析

![](img/17fedb0cf193d448585dabe0e91c140f_1.png)

上一节我们介绍了广告的抓取逻辑，本节中我们来看看亚马逊的广告归因周期。理解这个周期，能帮助我们看清广告如何一步步影响消费者的购买决策。

亚马逊的广告归因周期通常分为四个阶段，对应消费者从认知到购买的完整路径：

![](img/17fedb0cf193d448585dabe0e91c140f_2.png)

1.  **认知阶段**：消费者首次接触你的品牌或产品。此阶段的目标是扩大曝光，广告类型以**展示型推广（受众拓展）**和**品牌推广**为主。
2.  **考虑阶段**：消费者对你的产品产生兴趣，开始进行比较。此阶段可使用**商品推广**覆盖广泛关键词，同时利用**展示型推广（再营销）**重新吸引已浏览过商品的用户。
3.  **转化阶段**：消费者决定购买。此阶段应聚焦于高购买意向的关键词，通过**商品推广**精准投放，并利用**展示型推广**进行最后的转化助推。
4.  **忠诚度阶段**：消费者完成购买后，通过广告鼓励其复购或购买关联商品。此阶段可运用**展示型推广（再营销）**，定位已购买客户群体。



## 总结

本节课中，我们一起学习了亚马逊广告策略的基础框架。我们首先剖析了商品推广、品牌推广和展示型推广各自的流量抓取逻辑。接着，我们深入了解了广告归因的四个周期阶段——认知、考虑、转化与忠诚度，并明确了每个阶段对应的广告目标和主要广告形式。理解这些核心概念是搭建动态广告结构、进行有效数据分析的前提。在接下来的课程中，我们将探讨如何在不同时期搭建具体的广告结构。
