上一节我们介绍了手机端的操作，本节中我们来看看电脑端后台的具体设置。

## 保证金 💰

保证金是开店时需要向平台缴纳的一笔信用担保资金。前期可以暂不缴纳，但若计划长期运营或开通更多支付方式，则建议缴纳。

以下是关于保证金的几个要点：

![](img/44abf2bad2738d40039db1b0c89fd4c2_5.png)

*   **金额**：个人店保证金通常在 **1000元** 左右，具体金额根据所选经营类目而定。企业店保证金在2000元至数万元不等。
*   **查看与缴纳路径**：登录 **小红书千帆系统** -> 点击“资金” -> 选择“店铺保证金管理”。在此页面可以查询应缴金额并进行充值。
*   **可退还性**：与主流电商平台规则一致，当店铺不再经营时，保证金在扣除可能的违规罚款后，余额可以申请退还。
*   **扣款规则**：保证金主要用于保障消费者权益。例如，若商家同意退款但未执行，平台会从中扣款。正常合规经营则不会被扣。
*   **缴纳必要性**：**缴纳保证金是开通微信支付功能的必要条件**。考虑到微信支付的普及率，建议尽早缴纳以提升顾客支付便利性。

## 支付方式开通 💳

开通多样的支付方式能极大提升顾客的下单转化率。小红书店铺支持支付宝和微信支付。

![](img/44abf2bad2738d40039db1b0c89fd4c2_11.png)

支付方式的开通路径如下：

![](img/44abf2bad2738d40039db1b0c89fd4c2_13.png)

1.  进入小红书千帆后台，点击“资金”。
2.  在资金管理页面，找到“支付设置”相关选项。
3.  分别点击“支付宝”和“微信支付”进行开通绑定。

**注意事项**：
*   支付宝支付通常可即时开通。
*   微信支付需在缴纳保证金后的**第二天**才能进行设置。
*   提现金额将进入店铺注册时使用的法人账户（个人店为身份证持有人，个体/企业店为营业执照法人）。

![](img/44abf2bad2738d40039db1b0c89fd4c2_15.png)

## 运费保设置 🛡️

![](img/44abf2bad2738d40039db1b0c89fd4c2_17.png)

运费保是一项平台服务，旨在解决消费者退货时的运费顾虑，从而提升订单转化率并降低交易纠纷。

其核心作用是：当消费者成功退货退款后，系统会自动对运费进行补贴。

以下是开通运费保的步骤：

1.  在后台找到“售后管理”或类似菜单。
2.  查找“运费保”或“退货保障”服务。
3.  阅读服务说明后，点击“开通”即可。

![](img/44abf2bad2738d40039db1b0c89fd4c2_19.png)

![](img/44abf2bad2738d40039db1b0c89fd4c2_21.png)

**建议**：对于大多数店铺，开通运费保是利大于弊的选择，能有效打消顾客下单时的后顾之忧。

![](img/44abf2bad2738d40039db1b0c89fd4c2_23.png)

## 地址设置 📍

![](img/44abf2bad2738d40039db1b0c89fd4c2_25.png)

正确设置发货和退货地址对于订单处理和售后至关重要。电脑端的设置与手机端逻辑一致，但操作界面不同。

![](img/44abf2bad2738d40039db1b0c89fd4c2_27.png)

地址设置的具体路径为：进入后台 -> 找到“订单”或“物流”模块 -> 选择“物流工具” -> 点击“地址库管理” -> “新建地址”。

![](img/44abf2bad2738d40039db1b0c89fd4c2_29.png)

对于不同经营模式的商家，地址设置策略如下：

*   **无货源（一件代发）商家**：退货地址建议填写一个偏远地区（如新疆），并在地址后附加“**退货前请联系客服**”等字样。这是因为实际退货会由供应商处理，需要客服介入引导。
*   **有货源商家**：直接填写真实的仓库或办公地址即可。

![](img/44abf2bad2738d40039db1b0c89fd4c2_31.png)

本节课中我们一起学习了小红书电脑端后台的四个基础操作：保证金的查看与缴纳、支付方式的开通、运费保服务的设置以及地址库的管理。这些是店铺搭建的基石，请务必根据自身情况完成配置，为后续的商品上架与销售打好基础。
