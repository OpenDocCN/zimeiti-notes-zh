# 抖音运营教程：P1：AI自动创作中视频 🚀

![](img/b6080ad4d859ebf0762fdd9a5352f200_1.png)

在本节课中，我们将学习如何利用AI工具实现中视频的全自动创作。这套方法结合了文案生成、配音和视频画面匹配，旨在帮助初学者高效地产出符合平台要求的原创内容。

![](img/b6080ad4d859ebf0762fdd9a5352f200_3.png)

---

上一节我们概述了AI自动创作的可能性，本节中我们来看看具体的实现原理。平台青睐的中视频作品需要满足两个核心条件：**文案足够优秀**，以及**画面高度相关且精彩**。基于此，我们只需三个核心工具即可实现自动化流程。

以下是实现全自动创作所需的三个核心工具：

1.  **文案创作工具：ChatGPT**
    *   这是一个强大的人工智能文本生成模型。通过向其提出具体、精细的问题，它可以生成高度原创的文案。
    *   **核心操作**：向ChatGPT输入指令，例如 `请为我创作一个约1000字的、关于中国历史冷门人物的故事`。

2.  **AI配音工具：例如“配音神器”**
    *   该工具可将文本转换为语音。我们只需将ChatGPT生成的文案粘贴进去，并选择喜欢的主播音色即可生成配音文件。

3.  **视频素材匹配工具：例如“三三搜帧”**
    *   该工具能根据文案或关键词，智能搜索并匹配无版权的、高质量动态视频素材。其“AI合成视频”功能可自动将配音与画面对齐。

---

了解了所需的工具后，接下来我们看看如何将它们串联起来，完成从零到一的视频制作。

以下是具体的操作步骤：

1.  **生成原创文案**
    *   打开ChatGPT，输入你的内容需求。提问越具体，得到的文案质量越高。例如，可以要求它“写一个关于湖南地区的民间传说，字数控制在1200字左右”。
    *   如果对初稿不满意，可以追加条件，如“请在故事中加入一个智慧老人的角色，并保持总字数不变”。

2.  **制作配音**
    *   打开配音工具，将上一步得到的文案粘贴到文本框中。
    *   在“更多主播”中选择合适的音色和风格，生成并保存配音音频文件（通常是MP3格式）。

![](img/b6080ad4d859ebf0762fdd9a5352f200_5.png)

3.  **合成视频**
    *   打开“三三搜帧”工具，进入“AI合成视频”或类似功能页面。
    *   导入上一步保存的配音文件。系统会自动分析台词，并为每一句推荐多个匹配的视频画面片段。
    *   手动检查或调整系统匹配的画面，确保其与文案内容高度相关。最后，点击“生成视频”导出成品。

---

为了让你更直观地理解最终效果，我们可以看一个应用此方法生成的视频案例。

**视频效果示例：**

可以看到，通过上述流程生成的视频，其文案、配音和画面达到了高度的统一，完全符合中视频计划对原创性和质量的要求。
