![](img/1817179d91ceb0f6b3f7dd412b3b44bf_1.png)

在本节课中，我们将要学习小红书平台的核心流量分发机制。了解这套算法，是提升笔记曝光、获得更高排名的关键。我们将拆解其工作原理，并用简单直白的语言和公式进行说明。

## 概述：流量分发的核心逻辑

小红书平台通过一套复杂的算法系统，对用户发布的笔记进行评分和排序，从而决定其展示量和曝光机会。这套系统的核心目标是筛选出高质量、受用户欢迎的内容。

## 流量分发机制详解

上一节我们介绍了流量分发的重要性，本节中我们来看看这套机制具体是如何运作的。其核心可以概括为一个综合评分系统。

### 1. 基础评分：CES互动模型

平台对笔记的初步评估基于用户的互动行为，这套系统可简称为 **CES模型**（点赞、收藏、转发、评论、关注）。每种行为都有对应的分数：

*   **点赞**：1分
*   **收藏**：1分
*   **转发**：4分
*   **评论**：10分
*   **关注**：8分

**公式：单用户最高贡献分 = 1（点赞） + 1（收藏） + 4（转发） + 10（评论） + 8（关注） = 24分**

当一个用户对你的笔记完成所有上述互动时，可为该笔记贡献最高24分。系统会累计所有用户的互动分数，作为笔记的基础排名依据。

### 2. 展示与排名规则

小红书的笔记展示页面与抖音等单列流不同。以下是其展示规则：

*   平台以“页”为单位刷新内容，每页展示约20篇笔记。
*   这20篇笔记中，通常包含少量付费推广笔记（类似淘宝直通车）。
*   笔记会根据综合评分进行排名，决定其出现在第1到第20的哪个位置。
*   用户下滑浏览时，会加载下一页（即第21-40名），以此类推。

我们的核心目标就是让笔记的排名进入前20名，甚至前5名，以获得最优先的曝光。

### 3. 流量分发的完整流程

一篇笔记从发布到获得流量，会经历一个阶梯式的测试和推广过程。

**第一步：打标签与初始曝光**
笔记发布后，系统会根据笔记内容（图片、文案）和账号预设的标签，为其打上多个内容标签。随后，笔记会被推送给对这些标签感兴趣的用户，获得一个**基础曝光量**（例如1000次展示）。

**第二步：多轮流量测试**
初始流量并非一次性给足，而是分批次进行测试。例如，系统可能先分配400次曝光，根据这批曝光的数据表现，决定是否以及如何分配下一波流量（如300次或500次）。

**第三步：核心数据指标考核**
系统在每一轮流量测试中，主要考核以下几个关键指标，以决定是否增加流量推荐：

以下是决定笔记能否进入下一轮推荐的关键数据指标：

1.  **点击率**：用户在看到笔记封面和标题后，点击进入笔记页面的比率。这是所有后续互动的前提。**高点击率依赖于吸引人的封面和标题**。
2.  **互动率（CES）**：用户点击进入后，产生点赞、收藏、转发、评论、关注等行为的比率。这直接反映了内容的质量和吸引力。
3.  **完播率**：对于视频笔记，用户完整观看视频的比例。高完播率表明内容能留住用户。
4.  **互动值**：创作者与评论用户之间的回复、交流频率。积极的互动能进一步提升笔记权重。

**流程总结**：`高点击率` -> `用户进入笔记` -> `产生高互动率(CES)与完播率` -> `创作者积极互动提升互动值` -> `系统计算综合评分提升排名` -> `获得下一轮更大流量推荐`。

如果笔记在第一波测试中点击率和互动数据达标，系统会在后续推荐中加大流量扶持。反之，如果数据不佳，后续流量则会减少甚至停止。通常，一篇笔记的流量推荐周期约为3天。

### 4. 算法核心与公平性

无论具体数据指标如何变化，小红书流量分发的**核心算法**始终是考察笔记带来的**真实用户交互行为**。即使是一个粉丝数不多的新账号，只要单篇笔记内容优质，能引发高点击和高互动，同样有机会被系统推荐给大量用户，形成“阶梯式”曝光增长。

平台的基础曝光量相对公平。通常，一篇原创笔记至少能获得50次以上的基础展示，优质原创内容的基础展示量可能在150-200次之间。

## 总结

本节课中我们一起学习了小红书平台的流量分发机制。关键在于理解其**CES互动评分模型**和**阶梯式流量测试流程**。要想提升笔记排名和曝光，必须从源头做起：优化封面标题以提升**点击率**，创作优质内容以提升**互动率(CES)**和**完播率**，并积极与用户交流以提升**互动值**。掌握这套机制的核心逻辑，就能更有针对性地优化运营策略，让笔记获得更多流量。
