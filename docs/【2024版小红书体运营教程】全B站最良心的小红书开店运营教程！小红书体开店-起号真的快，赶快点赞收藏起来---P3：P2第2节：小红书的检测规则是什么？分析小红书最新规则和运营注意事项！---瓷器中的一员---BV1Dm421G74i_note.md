# 小红书运营教程：P3：小红书平台检测规则详解

在本节课中，我们将详细解析小红书平台的检测规则。了解这些规则是避免账号违规、确保内容安全发布的基础。我们将从多个维度拆解小红书的审查机制，并提供实用的规避方法与工具。

## 检测范围与内容

小红书平台主要检测以下几个方面的内容：个人资料、笔记正文、私信以及评论。

以下是平台明确禁止在这些区域出现的内容：
*   **联系方式**：微信号、微信二维码、手机号码。
*   **营销信息**：代购信息、推广淘宝等第三方店铺的词汇。
*   **不当内容**：水印、其他平台的引流信息。

不过，平台管理也存在一定的灵活性。官方允许在个人资料中留下邮箱地址。同时，用户可以使用谐音词或通过提及微博等方式进行间接引流，再通过特定话术引导至微信。具体的操作技巧将在后续课程中详细讲解。

## 设备与账号管理

除了内容，小红书还会检测登录设备。平台建议，同一台移动设备最多只登录三个账号。这一点需要特别注意，过多的账号关联可能引发风险。

## 笔记内容禁忌

在创作笔记内容时，必须避免任何诱导用户的行为。

以下行为是小红书明令禁止的：
*   以“关注即可领奖”等方式诱导关注。
*   以“点赞参与抽奖”等方式诱导互动。

这些均属于违规的诱导用户行为。

## 内容原创度保护

小红书平台对内容的原创度保护非常严格。在推广时，切忌将同一篇文章通过多个账号在平台内重复发布。正确的做法是一篇文章只用一个账号发布，避免多发。

## 敏感词检测机制与工具

上一节我们介绍了内容创作的基本规则，本节中我们来看看一个至关重要的具体机制：敏感词检测。这是小红书检测规则的核心部分，涉及金融、医疗、烟草等特定领域。

平台对某些关键词非常敏感，例如“手机卡”、“金融”、“电子烟”、“代购”、“医疗用品”等。在创作时应尽量避免触及这些领域。

那么，如果在写作时不慎加入了未知的敏感词该怎么办？这就需要借助外部工具进行预先检测。

![](img/8bb77674e5174ac4676ab876c0b4b653_1.png)

以下是两个实用的敏感词检测网站，可以辅助我们进行内容审查：

![](img/8bb77674e5174ac4676ab876c0b4b653_3.png)

1.  **网站A（免登录）**：这是一个便捷的在线工具。使用时，只需将写好的笔记内容粘贴到检测框中，点击“立即检测”，系统便会高亮显示出所有被识别为小红书敏感词的词汇。例如，如果文中包含“代购”一词，它会被明确标记出来。
    ![](img/8bb77674e5174ac4676ab876c0b4b653_1.png)

2.  **网站B（需注册）**：这个网站功能类似，但可能需要先注册账号。其操作逻辑与网站A一致，输入文本后即可获得过滤后的结果和敏感词提示。
    ![](img/8bb77674e5174ac4676ab876c0b4b653_3.png)
    ![](img/8bb77674e5174ac4676ab876c0b4b653_5.png)

对于日常使用，**网站A** 的功能已经足够。我们只需在发布前，将文章内容放入这些检测工具中扫描一遍，就能有效规避因敏感词导致的违规风险。

![](img/8bb77674e5174ac4676ab876c0b4b653_5.png)
