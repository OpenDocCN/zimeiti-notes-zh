## 概述：平台检测的四大核心领域

小红书的检测系统主要围绕以下四个方面展开，运营者必须严格遵守。

### 1. 内容与资料检测

首先，我们来了解平台对用户资料和发布内容的检测重点。小红书会严格审查个人资料、笔记正文、私信以及评论区。

以下是明确禁止出现的内容类型：
*   **联系方式**：微信号、微信二维码、手机号。
*   **营销词汇**：直接推广淘宝店铺或其他电商平台的用语。
*   **违禁行为**：“代购”信息、非平台水印。
*   **诱导互动**：关注领奖、点赞抽奖等引导用户行为的词汇。

**注意**：平台对个人资料中的邮箱管理相对宽松。一种常见的合规引流方法是使用微博ID或谐音词进行暗示，再通过话术引导至微信。此方法的具体操作将在后续课程中详细讲解。

### 2. 设备与账号关联检测

上一节我们介绍了内容层面的规则，本节我们来看看设备与账号的关联规则。小红书会检测登录设备的关联性。

**核心规则**：同一台移动设备最多建议登录**3个**小红书账号。超过此数量可能被系统判定为营销号或存在风险，影响账号权重。

### 3. 内容原创度保护

接下来，我们探讨平台对原创内容的保护机制。小红书高度重视内容的原创性，严厉打击抄袭和重复发布行为。

**重要原则**：一篇原创笔记，只应在一个账号下发布。切勿将同一篇文章在多个账号上重复发布，此行为极易被系统识别并处罚。

### 4. 敏感词检测与处理

最后，也是最为关键的一环，是平台的敏感词检测系统。这是所有运营者在撰写笔记时都会遇到的挑战。

平台禁止讨论的领域包括但不限于：**手机卡**、**金融**、**电子烟**、**医疗用品**等。涉及这些领域的词汇需格外谨慎。

那么，在写作中如何确保不误用敏感词呢？以下是两个实用的在线检测工具，可以帮助我们提前筛查：

1.  **工具A（无需注册）**：
    *   **使用方法**：将写好的笔记内容粘贴到网站的输入框中。
    *   **执行操作**：点击“立即检测”按钮。
    *   **结果反馈**：系统会高亮标出内容中存在的敏感词，例如“代购”。
    *   **代码示意**：其功能类似于一个文本过滤器。
        ```python
        # 概念性代码，展示检测逻辑
        sensitive_words = ["代购", "微信", "抽奖"]
        user_content = "这是一篇关于代购的笔记..."
        for word in sensitive_words:
            if word in user_content:
                print(f"发现敏感词: {word}")
        ```

2.  **工具B（需注册）**：
    *   该工具功能与工具A类似，同样提供敏感词过滤服务。注册后即可使用，能提供更详细的过滤结果报告。

**操作流程总结**：在笔记发布前，先将文案放入上述任一检测网站进行扫描，替换或删除被标出的敏感词，即可大幅提升内容通过审核的概率。

## 总结

本节课中，我们一起学习了小红书平台的四大核心检测规则：
1.  **内容资料**：严禁出现直接联系方式、硬广及诱导互动信息。
2.  **设备管理**：单台设备登录账号不宜超过3个。
3.  **原创保护**：坚持“一稿一号”，杜绝重复发布。
4.  **敏感词过滤**：善用在线检测工具，在发布前主动清理违禁词汇。

掌握并遵守这些规则，是你在小红书安全、稳定运营的第一步。关于更全面的违禁词列表，我们正在加紧整理，将在后续资料中提供给大家。
