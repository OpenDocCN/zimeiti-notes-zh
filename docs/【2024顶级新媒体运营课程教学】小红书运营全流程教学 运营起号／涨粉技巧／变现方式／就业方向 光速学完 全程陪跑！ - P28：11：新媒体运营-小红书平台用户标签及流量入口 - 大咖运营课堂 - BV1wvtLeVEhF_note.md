# 小红书运营全流程教学：P28：11：新媒体运营-小红书平台用户标签及流量入口 📊

在本节课中，我们将要学习小红书平台如何为用户打上标签，以及平台主要的流量入口有哪些。理解这些机制，能帮助我们更好地分析自己的账号和笔记是如何被系统推荐给目标用户的，从而制定更有效的运营策略。

上一节我们介绍了系统如何判定内容标签，本节中我们来看看平台如何判定用户标签。

## 用户标签判定逻辑

小红书系统主要通过分析用户的**浏览行为**来为其打上标签。用户注册时选择的兴趣范围，仅作为初期内容方向的引导，参考价值不大，无法进入系统深层的用户标签池。

![](img/076f5fc35574a02a7910c06b3f0b6545_1.png)

系统根据以下几种用户行为来判定其兴趣分类：

![](img/076f5fc35574a02a7910c06b3f0b6545_3.png)

![](img/076f5fc35574a02a7910c06b3f0b6545_5.png)

以下是判定用户标签的主要行为指标：

![](img/076f5fc35574a02a7910c06b3f0b6545_7.png)

1.  **点击率**
    *   **定义**：系统在发现页向用户推送多个不同领域的内容（如美妆、家居、母婴），用户点击哪一类内容更多，系统就能计算出其偏好。
    *   **作用**：通过点击概率，系统能为用户打上**强关联**或**弱关联**的标签。例如，用户频繁点击美妆内容，系统会判定其为美妆兴趣用户，并持续推送相关内容进行试探。

2.  **停留时长**
    *   **定义**：用户点开一篇笔记后观看的持续时间。
    *   **作用**：停留时长能更精细地判断用户对具体内容的喜好。例如，用户虽然常看美妆，但对某篇护肤笔记停留时间特别长，系统可能会强化其“护肤”子标签。如果用户对偶然推送的家居内容停留很久，系统可能判断其有装修或购买家居用品的潜在需求。

![](img/076f5fc35574a02a7910c06b3f0b6545_9.png)

![](img/076f5fc35574a02a7910c06b3f0b6545_11.png)

3.  **页面浏览量（PV）**
    *   **定义**：用户浏览页面的深度，例如不仅看了单篇笔记，还进入了博主主页并查看了其他多篇笔记。
    *   **作用**：PV值反映了用户兴趣的浓度。对于博主而言，精心设计的主页能提升用户的PV，这对账号的**健康度评分**有非常大的帮助，即使粉丝基数不大。

![](img/076f5fc35574a02a7910c06b3f0b6545_13.png)

![](img/076f5fc35574a02a7910c06b3f0b6545_15.png)

4.  **互动行为**
    *   **定义**：包括点赞、收藏、评论、关注等用户主动操作。
    *   **作用**：互动数据直接提升笔记的互动率，是笔记进入更高**推荐流量池**的关键。高互动笔记能获得更多曝光，带动账号其他笔记的流量，并有效**提升粉丝数**。

![](img/076f5fc35574a02a7910c06b3f0b6545_17.png)

![](img/076f5fc35574a02a7910c06b3f0b6545_19.png)

## 平台流量分发入口

小红书目前主要有四个流量分发入口，每个入口的内容推荐逻辑各有侧重。

![](img/076f5fc35574a02a7910c06b3f0b6545_29.png)

![](img/076f5fc35574a02a7910c06b3f0b6545_31.png)

以下是四个主要的流量入口及其特点：

![](img/076f5fc35574a02a7910c06b3f0b6545_33.png)

![](img/076f5fc35574a02a7910c06b3f0b6545_35.png)

![](img/076f5fc35574a02a7910c06b3f0b6545_37.png)

*   **发现页**
    *   **内容构成**：
        1.  用户已关注的达人发布的新内容。
        2.  根据用户标签（点击率、停留时长等）匹配的**兴趣内容**。
        3.  系统为了平台内容多样化而推荐的**发散性内容**（用户可能感兴趣的新领域）。
    *   **核心价值**：这是博主获取流量的**主要来源**，目标是满足用户个性化内容消费需求。

*   **关注页**
    *   **内容构成**：纯粹展示用户**已关注的所有达人**发布的笔记。
    *   **核心价值**：用于维护粉丝关系，粉丝可以集中查看所关注博主的更新。

![](img/076f5fc35574a02a7910c06b3f0b6545_39.png)

![](img/076f5fc35574a02a7910c06b3f0b6545_40.png)

![](img/076f5fc35574a02a7910c06b3f0b6545_42.png)

*   **附近页**
    *   **内容构成**：根据手机定位，推送**附近及同城**用户发布的内容。
    *   **核心价值**：主打地域性需求，如本地美食、旅游、线下店推广等。对于运营**线下实体店**的博主，需要重点关注此入口的运营和广告投放。

![](img/076f5fc35574a02a7910c06b3f0b6545_44.png)

![](img/076f5fc35574a02a7910c06b3f0b6545_46.png)

*   **购物页**
    *   **内容构成**：直接与电商、商品推荐相关的笔记和商品信息。
    *   **核心价值**：是小红书实现**变现**和发展电商业务的核心入口，直接匹配用户的购物需求。

![](img/076f5fc35574a02a7910c06b3f0b6545_48.png)

![](img/076f5fc35574a02a7910c06b3f0b6545_50.png)

本节课中我们一起学习了小红书平台判定用户标签的四大行为依据（点击率、停留时长、PV、互动行为），以及笔记获得曝光的四大流量入口（发现页、关注页、附近页、购物页）。理解“用户标签”和“流量入口”这两大底层逻辑，能帮助我们反推运营策略，让笔记更精准地触达目标用户，并为账号的长期发展奠定基础。
