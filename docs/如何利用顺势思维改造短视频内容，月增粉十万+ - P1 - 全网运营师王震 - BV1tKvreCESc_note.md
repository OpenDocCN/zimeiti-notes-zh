# 短视频运营：P1：顺势思维与内容创作

在本节课中，我们将要学习一个在短视频创作中至关重要的概念——顺势思维。我们将探讨它的定义、作用，以及如何巧妙地利用或反向利用它来提升视频的吸引力与互动数据，从而实现快速增长粉丝的目标。

## 什么是顺势思维？🎭

上一节我们提到了顺势思维这个概念，本节中我们来看看它的具体含义。

顺势思维是指人们倾向于沿着他人预设的路径或暗示进行思考的一种惯性思维模式。它源于人类的本能，在漫长的进化过程中形成，例如我们天生害怕黑暗、看到乌云会预判下雨。在内容消费中，观众会不自觉地进入创作者设置的思维轨道。

## 顺势思维在内容中的作用🔍

理解了顺势思维的定义后，我们来看看它在短视频中扮演的角色。

数据本身（如播放量）与视频质量没有绝对关系。观众决定是否继续观看一个视频，更多是基于一种感性的判断，而这种感性判断深受顺势思维的影响。当我们从观众视角看视频时，即使知道顺势思维有认知局限性，也很难跳出。

例如，思考“2秒完播率、5秒完播率、点赞、评论哪个更重要？”这个问题时，如果陷入顺势思维，会在这几个选项中比较。但若能跳出，第一层答案是“播放量”，再跳出一层，核心答案应是“变现能力”。遗憾的是，观众在观看时很难进行这种跳出习惯性思维的思考。

## 顺势思维的两种应用方式🔄

既然大部分观众难以跳出顺势思维，那么作为创作者，我们的任务不是成为陷入思维的“范伟”，而是成为设置思维的“本山老师”。关键在于如何设计内容，让观众顺利进入我们预设的思维轨道。

短视频内容在利用顺势思维上，主要分为两类：

以下是两种主要的应用方式：

1.  **正向利用顺势思维**：创作让观众感觉流畅、无阻碍的沉浸式内容。
    *   **底层逻辑**：顺势推舟。
    *   **例子**：沉浸式撸猫、沉浸式化妆等视频。观众在不知不觉中就看完了视频，其中背景音乐等因素加强了这种顺滑的体验。

2.  **反向利用顺势思维**：故意建立一种预期，然后打破它，制造反差和惊喜。
    *   **底层逻辑**：打破常规，结局出乎意料。
    *   **例子**：反向思维广告、疯狂小杨哥的搞笑视频。观众在某个节点预判了结果，但实际结局却截然不同，从而产生“猜不到结尾”的趣味性。

## 如何在创作中实践顺势思维💡

我们了解了顺势思维的两种模式，现在来看看如何将它们应用到具体的视频创意中。

所谓的“打破常规”，本质上就是对顺势思维的反向利用。在策划内容时，我们可以尝试将日常事件中的“正常顺序”进行反转。

例如，记录“发工资”这件事：
*   **正常顺势思维**：老板拿钱给员工。
*   **反向利用顺势思维**：老板跟员工要工资（现实中较难发生，代入感可能不强）。

我们可以进一步优化，设置为三方关系：老板、财务、普通员工。老板想给员工发工资，但财务不同意，于是老板和财务产生冲突。
*   如果视频采用**员工视角**，结局通常是老板和员工获胜。
*   如果采用**客户视角**，结局则可能是财务获胜。

这类利用身份、关系反转制造冲突的剧情，在平台上非常常见。同样，“00后整顿职场”这类年年热门的话题，也是反向利用顺势思维的典型：按照常规思维，整顿职场的应是资深人士或老板；但通过反向设计，将对立方设置为刚入职的00后，从而制造出强烈的戏剧张力和新鲜感。

**核心创作公式**：
`常规预期 + 关键元素反转 = 新鲜内容`
例如：`[发工资] + [老板→员工要钱] = 剧情冲突`

当然，使用这种方法需要考虑合理性与受众接受度，避免设计过于离谱的主题（如“10后整顿班主任”）。

## 总结📝

![](img/95638b46a102cdaa803f85749411ed32_3.png)

本节课中我们一起学习了顺势思维在短视频创作中的核心作用。我们认识到，大部分观众会本能地进入顺势思维轨道。因此，优秀创作者的核心技能在于主动**设置思维轨道**，通过正向利用制造沉浸感，或通过反向利用制造反差与惊喜，从而牢牢抓住观众的注意力，提升视频的互动与传播数据。用好顺势思维，是驱动视频数据增长的重要引擎。
