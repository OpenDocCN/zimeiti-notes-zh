# 小红书自动化引流教程：P1：设备准备与软件激活 🚀

在本节课中，我们将学习如何为“小红书全自动引流炮机”进行前期设备准备与软件激活。这是运行自动化脚本的基础步骤，请严格按照教程操作。

## 设备与材料准备 💻

上一节我们介绍了课程目标，本节中我们来看看具体需要准备哪些硬件和软件。

以下是激活前必须准备好的物品清单：
*   一台红米Note 9或红米Note 10手机。
*   一台安装Windows 10及以上64位操作系统的台式电脑或性能较好的笔记本电脑。不支持32位系统或苹果电脑。
*   一根可以传输文件的数据线，用于连接手机和电脑。

## 手机开发者设置 ⚙️

设备连接电脑后，我们需要对手机进行必要的设置以允许调试。

1.  打开手机“设置”，登录小米账号。仅登录即可，无需其他操作。
2.  进入“我的设备”。
3.  点击“全部参数与信息”。
4.  连续点击“MIUI版本”5次以上，直到屏幕提示“您已处于开发者模式”。
5.  返回设置，进入“更多设置”。
6.  找到并进入“开发者选项”。
7.  开启“开发者选项”开关。
8.  在开发者选项中，找到并开启“USB调试”及相关的调试选项（通常为3-4个）。
9.  继续下滑，找到“USB配置”，点击进入后选择“传输文件（MTP）”模式。

## 下载所需文件 📥

完成手机设置后，我们需要从提供的网盘链接下载必要的软件和工具。

以下是需要下载的文件及其对应设备：
*   **小红书全自动曝光机（软件）**：下载并安装到手机上。
*   **激活器一**：下载到电脑上。
*   **MT管理器**：下载并安装到手机上。

**注意**：请确保将文件下载到正确的设备（手机或电脑）。

## 创建配置文件 📄

软件下载完成后，我们需要在手机上创建运行脚本所必需的配置文件。

打开手机上的 **MT管理器** 应用，在指定目录下创建7个文本文档。创建方法有两种：直接在MT管理器中点击加号新建，或在电脑上创建后传输到手机。

以下是必须创建的7个文本文档名称（请确保名称与后缀完全一致）：
*   话术.txt
*   私信话术.txt
*   用户UID.txt
*   搜索词.txt
*   （其余3个文件名称请参照视频中演示的完整列表）

## 获取并填写注册码 🔑

配置文件创建完毕后，需要联系客服获取软件的唯一注册码。

1.  从客服处获得注册码。
2.  在MT管理器中，打开名为“注册码.txt”的文本文档。
3.  将获得的注册码填写到该文档中并保存。

## 运行激活器完成激活 ✅

这是激活流程的最后一步，将在电脑上完成。

1.  在电脑上打开之前下载的“激活器一”软件。
2.  用数据线连接手机与电脑。
3.  观察激活器界面，等待其显示“开始查找设备” -> “找到设备” -> “激活成功”。
4.  如果激活失败，请检查：激活器界面显示的数字是否为 **19901, 19902, 19903, 199120**；或检查文件是否被安全软件误删。

## 软件初始设置与验证 🧪

激活成功后，我们打开手机上的软件进行最终设置并验证激活状态。

1.  打开手机上的“小红书全自动曝光机”软件。
2.  点击左上角菜单，进入“系统设置”。
3.  在系统设置中，确保所有选项（如代理模式）与教程视频中演示的完全一致，并已打勾。
4.  检查“激活状态”是否已显示为打勾，以及“激活器端口”和“激活器密码”是否与电脑激活器上显示的一致。
5.  开启所有必要的悬浮窗权限。
6.  点击“保存”按钮。
7.  尝试启动软件，测试是否能正常检测到卡密（注册码）。若提示“卡密已到期”或“超时”，请按下一节方法排查。

## 常见问题排查 🔍

如果最后一步验证失败，请按以下顺序排查问题。

以下是激活失败的常见原因及解决方法：
*   **注册码文件错误**：检查MT管理器中“注册码.txt”文件内容是否与客服提供的一致。
*   **卡密状态异常**：联系客服确认卡密是否在有效期内且未被系统删除。
*   **设备兼容性问题**：如果以上均无误，可能是当前手机设备与系统不兼容。建议尝试为手机刷机或更换另一台设备进行测试。
