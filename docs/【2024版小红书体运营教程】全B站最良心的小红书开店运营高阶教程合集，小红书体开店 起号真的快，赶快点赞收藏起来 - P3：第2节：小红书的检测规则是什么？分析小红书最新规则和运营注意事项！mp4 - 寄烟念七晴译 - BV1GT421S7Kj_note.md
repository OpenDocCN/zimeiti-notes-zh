# 小红书运营教程：P3-2：小红书平台检测规则详解 🕵️

在本节课中，我们将要学习小红书平台的核心检测规则。了解这些规则是安全、有效运营账号的基础，能帮助我们避免违规，确保内容顺利发布和传播。

上一节我们介绍了账号定位，本节中我们来看看平台如何监管内容。小红书的检测系统主要监控以下几个方面，以确保社区环境的合规性。

## 检测范围与禁止内容

小红书平台会系统性地检测用户资料与内容。以下是主要的四个检测方向：

1.  **资料与内容检测**：平台会检测个人资料、笔记内容、私信以及评论。在这些地方，**不得含有**微信号、微信二维码、“代购”字样、水印，以及推广淘宝店铺等明显营销词汇。
2.  **允许的引流方式**：平台管理相对宽松的是，官方个人资料**允许填写邮箱**。同时，可以使用“微博”等平台名称的谐音进行间接引流，再通过话术引导至微信。具体操作技巧将在后续课程详细说明。
3.  **设备登录限制**：小红书会检测手机设备。为确保账号安全，**同一台设备最多建议只登录3个账号**，这一点需要特别注意。
4.  **禁止诱导行为**：在笔记内容中，**不得诱导用户**进行关注领奖、点赞抽奖等互动。这类行为被平台明令禁止，属于违规的诱导操作。

## 内容原创度与发布规则

在内容创作方面，小红书有严格的规定以保护原创。

内容的原创度在小红书平台上受保护程度较高。因此，在推广时，**切勿将同一篇文章在多个账号上重复发布**。正确的做法是，一篇文章只用一个账号发布一次，避免多发。

## 敏感词检测与处理工具

除了上述规则，小红书还设有严格的敏感词检测机制。

敏感词检测是内容发布前的重要环节。涉及“手机卡”、“金融”、“电子烟”、“医疗用品”等领域的词汇，应尽量避免触碰。

若在写作时不慎加入了未知的敏感词，可以使用专门的检测工具进行排查。以下是两个实用的敏感词检测网站使用方法：

**第一个网站（无需注册）**：
1.  打开检测网站。
2.  将写好的文章内容粘贴进输入框。
3.  点击“立即检测”按钮。
4.  系统会高亮显示文章中的敏感词，例如“代购”。

平时在发布文章前，先将内容通过此类网站检测，可以有效识别并修改敏感词，保障笔记顺利通过平台审核。
