#### 第三小节：剪映核心操作功能详解

![](img/a196804cf04a7f7c747e23044c9608fe_24.png)

第三小节是第一部分课程的重中之重，将详细讲解剪映的核心操作功能。我们将围绕“剪辑”功能，展开讲解十五大核心功能。

了解这些功能的技巧，是为了让学员在今后创作时能灵活运用，明白视频制作的逻辑和原理。如果不了解这些，即使在实战案例中，也可能无法理解其设计原理。

![](img/a196804cf04a7f7c747e23044c9608fe_26.png)

![](img/a196804cf04a7f7c747e23044c9608fe_27.png)

以下是剪映的十五大核心功能列表：

![](img/a196804cf04a7f7c747e23044c9608fe_28.png)

*   **分割**：`分割` 功能用于将一段视频或音频切割成多个部分。
*   **音频**：为视频添加背景音乐、音效或录制画外音。
*   **文本**：为视频添加字幕、标题等文字内容。
*   **贴纸**：添加各种动态或静态的图案贴纸。
*   **画中画**：`画中画` 功能允许在主视频上叠加另一个视频图层。
*   **特效**：为视频添加视觉特效，如模糊、闪光、震动等。
*   **滤镜**：为视频画面应用不同的色彩风格。
*   **比例与背景**：调整视频画幅比例（如9:16, 1:1）并设置背景颜色或模糊效果。
*   **转场**：在不同视频片段之间添加过渡效果。
*   **定格**：将视频的某一帧画面变为静态图片持续播放。
*   **倒放**：使视频片段从后向前播放。
*   **蒙版**：`蒙版` 功能用于控制图层的显示区域，实现局部遮挡或融合效果。
*   **变声**：改变视频或音频中声音的音调。
*   **色度抠图**：`色度抠图` 功能用于扣除视频中的纯色背景（如绿幕）。
*   **素材寻找**：介绍如何在抖音或剪映内寻找常用的视频、音频素材。

这十五项功能是当前剪映剪辑中使用最频繁的。掌握它们后，在制作各类视频时都能得心应手地运用所需功能。
