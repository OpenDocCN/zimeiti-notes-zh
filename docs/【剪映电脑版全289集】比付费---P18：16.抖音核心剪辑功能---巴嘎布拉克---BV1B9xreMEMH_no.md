# 剪映电脑版教程：P18：16.抖音核心剪辑功能—贴纸 📌

在本节课中，我们将要学习剪映的核心剪辑功能之一——贴纸。贴纸是提升视频画面质量、增加趣味性和传递信息的重要工具。我们将了解其作用，并学习如何在视频中添加和设置贴纸。

## 贴纸功能概述 🎨

![](img/772d78fb11b5dee413a429177e20ad16_1.png)

上一节我们介绍了其他剪辑功能，本节中我们来看看贴纸。贴纸功能常用于VLOG、美食或旅拍类视频中，用于在视频画面中插入各种装饰性元素。过去，贴纸主要用于照片修图；如今，在视频制作中添加贴纸，同样能显著提升视频的视觉质量和信息传达效果。

![](img/772d78fb11b5dee413a429177e20ad16_1.png)

剪映软件内置了丰富的贴纸素材库。添加贴纸后，我们可以从库中选择大量元素，包括平台热门贴纸，以此来增强视频的炫酷程度。例如，在换装或变身类视频中，贴纸能极大地增强画面表现力。在日常VLOG中，添加贴纸也能提升视频的形象和趣味性。

![](img/772d78fb11b5dee413a429177e20ad16_3.png)

## 贴纸功能实操环节 🛠️

![](img/772d78fb11b5dee413a429177e20ad16_5.png)

![](img/772d78fb11b5dee413a429177e20ad16_6.png)

了解了贴纸的基本概念后，接下来我们进入实际操作环节，学习如何为视频添加和设置贴纸。

![](img/772d78fb11b5dee413a429177e20ad16_3.png)
![](img/772d78fb11b5dee413a4a177e20ad16_5.png)
![](img/772d78fb11b5dee413a4a177e20ad16_6.png)

首先，新建一个项目并导入视频素材。然后，在一级菜单的左下方找到并点击 **“添加贴纸”** 按钮。

以下是添加和设置贴纸的具体步骤：

1.  **选择贴纸**：点击后，会弹出贴纸素材库。库中包含多种类别，如热门表情、文字、装饰元素（太阳、彩虹、蝴蝶、花朵、边框等）。
2.  **添加贴纸**：根据视频题材，选择合适的贴纸，点击即可添加到视频轨道上。
3.  **设置时长**：添加后，可以拖动贴纸轨道两端的边缘来调整其显示时长。可以让它贯穿整个视频，也可以只出现在特定片段。
4.  **预览效果**：调整后播放预览，检查贴纸与视频的配合效果。
5.  **添加动画**：选中贴纸，在右侧功能面板中可以为其设置入场、出场等动画效果，让贴纸的出现方式更生动。

## 课程总结 📝

本节课中我们一起学习了剪映的贴纸功能。使用贴纸可以快速提升视频的画面氛围，让内容更加生动有趣。操作方法是：在剪映中直接点击 **“贴纸”** 按钮，从素材库选择内容插入视频，并可自由设置其显示时长和动画特效。
