# 小红书运营新手起号：P28：小红书账号类型与官方广告投放解析

在本节课中，我们将学习小红书账号的三种主要类型，并对比自建账号运营与官方广告投放两种推广方式的区别与联系，帮助你理清运营思路。

## 账号类型解析

上一节我们介绍了账号定位，本节中我们来看看小红书平台上的账号具体分为哪几类。了解不同类型的账号及其用途，是制定有效运营策略的基础。

小红书账号主要分为以下三类：

1.  **企业号**
    企业号的核心作用是树立官方品牌形象。它拥有官方认证标识，并能行使平台赋予的特权，例如展示品牌名称、地址等信息。这有助于品牌开展相关业务，建立用户信任。

2.  **个人号**
    个人号是目前大多数用户运营的账号类型。它的特点是**灵活性高**。运营者可以是品牌员工、产品消费者或任何个人。个人号能更好地塑造个人品牌特色，品牌相关人物或员工出镜也显得更为自然。

3.  **引流号**
    引流号在之前的课程中也有提及，俗称“炮灰号”。这类账号可以创建多个，主要用于特定目的的引流，其重要性相对较低，可以随意打理或在紧急情况下作为备用小号使用。通常，引流号没有精细的个性与内容规划。

一般而言，品牌的推广渠道主要是通过上述三类账号自行发布笔记进行引流。

## 企业号矩阵运营建议

在了解了账号分类后，我们来看看如何协同使用这些账号，特别是围绕企业号构建运营矩阵。以下是打造企业号矩阵的一些具体建议：

*   **积极参与品牌活动**：品牌的企业号发起试用、团购等活动时，可以动用多个小号（个人号或引流号）去参与，增加活动热度与真实感。
*   **个人号展示品牌属性**：个人号在发布内容时，应多方位暗示其与品牌的关联。例如，如果品牌定位是“都市职场年轻人”，那么个人号的内容也应围绕相关场景和话题展开。
*   **控制营销内容比例**：个人号在发布品牌产品相关内容时，需注意频率。建议遵循 **“三七分”原则**：即30%的内容用于产品营销，70%的内容用于种草分享、知识科普等，以维持账号的可信度与吸引力。
*   **利用小号评论引流**：当企业号发布内容后，可以使用引流号在评论区进行互动或引导，以实现更隐蔽和有效的引流。

## 官方广告投放介绍

![](img/1005b00058d5103be55c2dbe755e982b_5.png)

除了运营自建账号矩阵，另一种重要的推广方式是借助小红书平台官方的广告系统。下面，我们来详细了解这种付费推广方式。

![](img/1005b00058d5103be55c2dbe755e982b_7.png)

小红书官方广告主要指 **“小红书CPC广告”**，即按点击付费的竞价广告。商家通过竞价购买广告位，以获得精准的流量曝光。

![](img/1005b00058d5103be55c2dbe755e982b_9.png)

![](img/1005b00058d5103be55c2dbe755e982b_11.png)

### 广告投放前提与流程

投放官方广告有一个核心前提：**必须拥有一个已认证的企业号**。

![](img/1005b00058d5103be55c2dbe755e982b_13.png)

企业号认证流程如下：
1.  访问小红书官方网站。
2.  找到“业务合作”或相关入口。
3.  进入“推广合作”或“企业号入驻”页面。
4.  按照指引提交企业资质文件并支付审核认证费用。
5.  审核通过后，普通账号即升级为官方认证的企业号。

完成企业号认证后，即可在广告平台中创建和管理广告计划。

### 广告类型与展示位

小红书的信息流广告主要分为两大类：

1.  **信息流广告**：当用户浏览小红书首页推荐流时，系统会不定期间隔插入广告内容。这种广告形式原生，用户体验相对自然。
2.  **搜索广告**：当用户使用关键词进行搜索时，在搜索结果中出现的广告。例如，用户搜索“考研资料”，系统可能匹配推荐相关的学习工具或营养品广告。其核心机制是**关键词竞价排名**，出价越高，广告获得优势排位的可能性越大。

![](img/1005b00058d5103be55c2dbe755e982b_15.png)

目前，信息流广告常见的固定展示位主要是笔记流中的第3位、第13位和第23位等位置。

![](img/1005b00058d5103be55c2dbe755e982b_17.png)

**注意**：官方广告投放涉及较高的费用。在资金允许的情况下，与官方合作投放竞价广告可以直接获取由平台算法推荐的精准流量，效率较高。具体的投放细节、常见问题及最新资费标准，建议直接参考小红书广告平台的官方文档与说明。
