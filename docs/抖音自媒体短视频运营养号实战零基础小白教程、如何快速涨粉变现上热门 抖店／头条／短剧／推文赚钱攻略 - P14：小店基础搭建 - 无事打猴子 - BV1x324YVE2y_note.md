# 抖音自媒体短视频运营：P14：小店基础搭建 🛠️

在本节课中，我们将学习抖音小店开通后，如何进行基础搭建与精细化运营。我们将从支付设置、保障服务、地址管理到运费模板，一步步完成店铺的基础配置。

---

上一节我们介绍了开通小店，本节中我们来看看开通后需要完成哪些基础设置。

首先，登录小店后台，进入“店铺”的“支付方式”设置页面。以下是必须完成的支付功能配置：

![](img/32a49b81541fb01306d1fb67d5f574a9_2.png)

*   **开通所有支付功能**：将页面中列出的四个支付功能全部开启。这是后续正常提现到对公账户或个人账户的前提条件。

支付功能开通后，我们进入“商家保障中心”。

在商家保障中心，我们需要开启“运费险”服务。开通运费险需要支付少量费用（通常几毛钱），但能带来平台流量倾斜。以下是开通运费险的主要好处：

*   **增加商品曝光**：数据显示可提升约19%。
*   **提升转化率**：系统自动理赔退货，能打消顾客下单顾虑。
*   **降低客户投诉**：提供售后保障，改善购物体验。

---

接下来，我们回到店铺首页。

![](img/32a49b81541fb01306d1fb67d5f574a9_8.png)

![](img/32a49b81541fb01306d1fb67d5f574a9_9.png)

在首页向下滑动，找到“急速退款助手”并点击进入。我们需要进行以下设置：

![](img/32a49b81541fb01306d1fb67d5f574a9_10.png)

*   **开启“未发货仅退款”**：系统会自动识别金额小于300元、下单时间在6小时内且未发货的订单，当客户申请退款时实现急速退款。
*   **“退货退款”功能**：此项暂不开启。

---

设置完急速退款，我们再次回到“商家保障中心”。

在商家保障中心页面继续向下滑动，找到“安心购”服务并申请开通。开通“安心购”的要求是完成其列出的各项前置功能开通。你可以检查列表，确保每一项都显示“已开通”状态。全部开通后，即可加入“安心购”。

![](img/32a49b81541fb01306d1fb67d5f574a9_12.png)

“安心购”标志会展示在商品详情页，有助于提高订单转化率和成交率，并能提升商家体验分。

---

开通完保障服务，我们继续处理物流相关设置。

在后台找到“地址库管理”。在这里，我们需要添加并设置默认地址。对于做无货源的学员，这一点尤为重要，因为供应商的仓库地址可能不同。平台近期推出了“闪电退货”规则，用户退货时，系统会自动发送地址库中设置的默认退货地址。

因此，你需要提前与供应商对接，获取准确的退货地址并填写在此。如果某些供应商不提供退货地址，可以采取以下变通方法：

*   **填写一个临时地址**：在详细地址栏中，添加说明文字，例如：“**退货请联系客服**”或“**该地址不作为退货地址，仓库较多，退货前请务必联系客服确认**”。
*   **设为默认**：将此地址同时设置为“默认发货地址”和“默认退货地址”。

这样就能解决部分没有固定退货地址的配置问题。关于如何创建更灵活的多地址退货策略，我们将在下节课详细讲解。

![](img/32a49b81541fb01306d1fb67d5f574a9_14.png)

地址设置完成后，我们来到“运费模板”进行配置。

新店铺通常只有一个不可修改的“全国包邮”默认模板。我们需要新建属于自己的运费模板。点击“新建模板”，按以下步骤操作：

1.  **模板名称**：输入易于识别的名称，如“部分地区包邮”。
2.  **发货地址**：选择或填写你的实际发货地。
3.  **运费设置**：系统提供三种计费方式，建议根据商品类型选择“阶梯计价”：
    *   **阶梯计价**：按件数或重量分段计费。例如：首件0元，每增加一件加0元（即全包邮）。对于服装可按“件”，水果可按“重量”。
    *   **固定运费**：无论数量或重量，统一收取固定运费。
    *   **卖家包邮**：所有订单均由卖家承担运费。
4.  **为偏远地区设置运费**：点击“添加地区”，选择如新疆、西藏、内蒙古、港澳台等偏远地区。为其设置单独的运费规则，例如：首件运费30元，每增加一件加收5-10元（具体需根据物流成本核算）。
5.  **指定条件包邮**（可选）：可设置如“满99元包邮”或“满2件包邮”的规则，通常在全包邮情况下可不设置。
6.  **限制下单区域**（可选）：可添加因疫情等原因无法配送的地区，用户填写该地址时将无法下单。
7.  **保存模板**：完成所有设置后，点击保存。

创建完成后，在运费模板列表即可看到新建的模板及其被多少商品使用。
