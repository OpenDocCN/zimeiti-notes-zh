## 热门内容形式：拍段子指南

了解了成功案例后，我们来看看一种非常受欢迎的内容形式：拍段子。

想做对口型、搞笑剧情、沙雕段子或顾客趣闻这类段子内容的同学，可以在公屏上扣“1”。

![](img/6d3243e5d67f804d02d754748e27ed15_9.png)

![](img/6d3243e5d67f804d02d754748e27ed15_9.png)

关于拍段子，我想给大家看一个实打实的例子。我有一位在工地辛苦上班的学员，他最初自己拍的搞笑视频播放量只有2000或3000。但经过学习后，他给我的反馈是：“老师，我的播放量快要破500万了，评论和涨粉还在增加，我不知道该怎么回复了。”

![](img/6d3243e5d67f804d02d754748e27ed15_11.png)

他很紧张。希望我们将来也能体会到这种播放量暴涨的心情。

![](img/6d3243e5d67f804d02d754748e27ed15_13.png)

![](img/6d3243e5d67f804d02d754748e27ed15_11.png)

同时，拍这种段子并不难。但前提是，你不能是一个特别腼腆、害羞、不好意思的人。拍段子的目的就是让人哈哈一笑，让人觉得有趣、有意思，需要一定的表演成分。
后期如果你不知道拍什么，老师会提供**段子素材**和**拍摄脚本**。这一点完全不用担心。

![](img/6d3243e5d67f804d02d754748e27ed15_13.png)

**核心支持公式：**
```
内容创作 = 可用素材 + 详细脚本 + 个人表演
```
