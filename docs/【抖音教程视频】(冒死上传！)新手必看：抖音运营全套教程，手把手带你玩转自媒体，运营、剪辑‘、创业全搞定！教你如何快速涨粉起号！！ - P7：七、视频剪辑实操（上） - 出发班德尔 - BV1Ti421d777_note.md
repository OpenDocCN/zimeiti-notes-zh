# 抖音运营全套教程：P7：视频剪辑实操（上）🎬

在本节课中，我们将要学习如何使用手机版剪映App进行视频剪辑的基础操作。我们将从确定视频主题开始，逐步完成素材导入、背景添加、文字制作和音频处理，最终调整视频比例以适应不同平台。整个过程将手把手演示，确保初学者能够轻松跟上。

---

![](img/877519df396f708928c1ddc59811fb90_1.png)

上一节我们介绍了视频剪辑的整体流程，本节中我们来看看如何从零开始制作一个视频的开头部分。

首先，制作视频需要一款软件。请在你的手机上下载并安装名为“剪映”的App。其电脑端和手机端的图标外观一致。

![](img/877519df396f708928c1ddc59811fb90_3.png)

![](img/877519df396f708928c1ddc59811fb90_1.png)

![](img/877519df396f708928c1ddc59811fb90_5.png)

安装完成后，打开剪映App。进入主界面后，你会看到一个“+”号按钮，这是“开始创作”的入口，点击它即可新建项目。

![](img/877519df396f708928c1ddc59811fb90_3.png)

![](img/877519df396f708928c1ddc59811fb90_7.png)

在开始剪辑前，需要先确定视频的主题。例如，我们计划制作一个关于“沈腾综艺爆笑名场面”的动漫综艺类视频。主题确定后，你在寻找素材时，必须确保每个视频片段中都包含沈腾本人，否则该素材无法使用，且平台审核时也会考察内容与主题的相关性。

![](img/877519df396f708928c1ddc59811fb90_9.png)

![](img/877519df396f708928c1ddc59811fb90_5.png)

![](img/877519df396f708928c1ddc59811fb90_11.png)

确定主题后，点击“+”号，将你准备好的视频素材导入到剪辑轨道中。

![](img/877519df396f708928c1ddc59811fb90_13.png)

![](img/877519df396f708928c1ddc59811fb90_7.png)

![](img/877519df396f708928c1ddc59811fb90_15.png)

素材添加完成后，我们需要为视频制作一个标题页。现成的主题模板很难找到，需要我们自己制作。

![](img/877519df396f708928c1ddc59811fb90_17.png)

![](img/877519df396f708928c1ddc59811fb90_9.png)

![](img/877519df396f708928c1ddc59811fb90_19.png)

观察视频轨道上方，有一个“+”号按钮。点击它，可以为视频添加新的片段。

![](img/877519df396f708928c1ddc59811fb90_11.png)

![](img/877519df396f708928c1ddc59811fb90_21.png)

点击“+”号后，选择剪映自带的“素材库”。

![](img/877519df396f708928c1ddc59811fb90_23.png)

![](img/877519df396f708928c1ddc59811fb90_13.png)

![](img/877519df396f708928c1ddc59811fb90_25.png)

在素材库的“热门”分类中，有各种动态或静态的背景画面可供选择。

![](img/877519df396f708928c1ddc59811fb90_27.png)

![](img/877519df396f708928c1ddc59811fb90_15.png)

对于零基础的新手，建议选择更简洁的背景，例如纯黑或纯白背景。这样制作出来的标题页面会更干净、工整，不会显得杂乱。这里我们选择一个黑色背景并添加。

![](img/877519df396f708928c1ddc59811fb90_17.png)

![](img/877519df396f708928c1ddc59811fb90_29.png)

背景添加好后，需要添加文字标题。点击轨道下方工具栏中的“文字”选项。

![](img/877519df396f708928c1ddc59811fb90_19.png)

然后选择“新建文本”，会出现一个文本框。在框内输入我们的主题文字，例如“沈腾综艺爆笑名场面”。

![](img/877519df396f708928c1ddc59811fb90_21.png)

![](img/877519df396f708928c1ddc59811fb90_31.png)

白色的默认文字可能不够美观。我们不需要手动调整复杂的样式，可以直接使用“文字模板”。模板区有免费和付费选项，选择一个合适的模板应用即可。

![](img/877519df396f708928c1ddc59811fb90_33.png)

![](img/877519df396f708928c1ddc59811fb90_23.png)

![](img/877519df396f708928c1ddc59811fb90_35.png)

选好模板后，点击右上角的对勾确认。模板会自动下载并应用到文字上。你可以拖动文字框右下角来调整文字大小。

![](img/877519df396f708928c1ddc59811fb90_25.png)

文字样式设置好后，接下来添加语音。点击文字轨道，在下方工具栏中找到“文本朗读”功能。为文字添加语音可以使视频更生动，避免成为“哑巴”视频。

如果你自己会配音，可以使用自己的声音，这样更具原创性。如果不会，则使用“文本朗读”功能。该功能提供多种音色，包括方言、萝莉音、播音腔等。

目前，使用“方言”音色是一种创新选择，因为使用率相对较低，更容易吸引观众注意。你可以试听并选择自己喜欢的声音，例如“广西话”音色。

![](img/877519df396f708928c1ddc59811fb90_27.png)

选择音色后，系统会自动生成音频并下载。试听效果：“沈腾综艺爆笑名场面”。生成后，需要手动调整黑色背景画面的时长，使其与音频长度一致，实现音画同步。

![](img/877519df396f708928c1ddc59811fb90_29.png)

标题页制作完成后，接下来处理主视频素材。请再次确认素材中是否包含沈腾。

![](img/877519df396f708928c1ddc59811fb90_37.png)

现在，开始讲解视频剪辑中的一个关键步骤：调整视频比例。调整比例是因为不同视频平台有各自的格式要求。

绝大多数视频平台（如B站、西瓜视频）要求视频比例为 **16:9** 的横屏。

**例外情况**：抖音平台要求视频比例为 **9:16** 的竖屏。

请务必区分这两个标准。

![](img/877519df396f708928c1ddc59811fb90_31.png)

本节课中我们一起学习了视频剪辑的开篇操作：从安装剪映、确定主题、导入素材，到制作包含背景、文字和语音的标题页，最后了解了调整视频比例以适应不同平台的重要性。这些是制作一个完整视频的基础步骤。
