# 手机剪辑教程：P24：2. 穿越手机效果 📱➡️🎬

在本节课中，我们将学习如何使用剪映App制作一个“手机穿越”的创意视频效果。这个效果能让视频画面仿佛从一部手机中穿越出来，非常适合用于短视频的转场或开场。

## 概述
我们将通过几个核心步骤来完成这个效果：导入素材、使用画中画与色度抠图进行合成，最后添加音乐和特效来增强视觉冲击力。整个过程简单明了，即使是初学者也能轻松跟上。

## 核心步骤详解

上一节我们概述了制作流程，本节中我们来看看每个步骤的具体操作方法。

### 1. 导入与裁剪素材
首先，你需要准备一段主体视频和一个“手机穿越”的绿幕素材。

以下是操作步骤列表：
*   打开剪映App，导入你准备好的主体视频。
*   点击下方工具栏中的 **`画中画`** 功能。
*   选择 **`新增画中画`**，导入手机穿越的绿幕素材。
*   将导入的绿幕素材调整到与主视频相同的时长，并进行裁剪，使其与手机屏幕区域对齐。

### 2. 抠除绿幕
完成素材对齐后，下一步是抠除绿幕素材的绿色背景，让主体视频从手机屏幕中显现出来。

以下是关键操作列表：
*   在时间轴上选中绿幕素材。
*   点击下方工具栏中的 **`抠像`** 选项。
*   选择 **`色度抠图`** 功能。
*   将取色器移动到绿色背景上，确保选中绿色。
*   将 **`强度`** 和 **`阴影`** 参数都调整到 **`100`**，以彻底去除绿色并保留自然阴影。

### 3. 添加音频与特效
画面合成完成后，我们需要添加音频和特效来提升视频的整体氛围和节奏感。

以下是最后的润色步骤列表：
*   点击 **`音频`**，导入一段你喜欢的背景音乐。
*   将时间轴指针移动到音乐的高潮部分。
*   点击 **`特效`**，选择 **`氛围`** 分类下的 **`积粉`** 特效。
*   点击 **`滤镜`**，为视频整体添加一个 **`影视级`** 分类下的 **`激光`** 滤镜。

## 效果预览与总结
完成以上所有步骤后，点击播放按钮即可预览最终的“手机穿越”效果。

本节课中我们一起学习了如何利用剪映的画中画、色度抠图、特效和滤镜功能，将一段普通视频与创意素材结合，制作出吸引眼球的穿越转场效果。关键在于素材的对齐和绿幕抠图的参数调整，掌握这些就能轻松实现类似的创意剪辑。
