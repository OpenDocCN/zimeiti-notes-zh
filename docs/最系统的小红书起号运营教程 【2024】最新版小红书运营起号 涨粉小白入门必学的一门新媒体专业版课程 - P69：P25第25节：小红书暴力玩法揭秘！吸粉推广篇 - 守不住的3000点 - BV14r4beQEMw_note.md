# 小红书运营起号教程：P25：小红书暴力玩法揭秘！吸粉推广篇 💥

在本节课中，我们将要学习小红书账号运营与广告投放的核心策略。我们将理清自建账号推广与官方广告投放的关系，并详细介绍账号类型、矩阵玩法以及官方广告的申请与投放方式。

## 账号类型解析

上一节我们介绍了小红书运营的基础，本节中我们来看看小红书平台上的三种主要账号类型及其特点。

**1. 企业号**
企业号的主要作用是树立官方品牌形象。它拥有官方认证标志，并享有特定特权，例如展示品牌名称、地址等信息，便于开展相关业务。其核心功能是品牌建设与官方发声。

**2. 个人号**
目前大部分用户运营的是个人号。个人号的特点是灵活性高，运营者可以是品牌员工或产品消费者。这使得个人号能承载更多个人品牌特色，品牌相关人物或员工出镜也显得更自然。

**3. 引流号**
引流号在之前的课程中也有提及，俗称“炮灰号”。这类账号可以创建多个，主要用于特定引流目的，用后即可舍弃，或作为日常打理简单、遇紧急情况时使用的小号。引流号通常没有精细的个性规划。

一般的品牌推广或渠道引流，主要通过以上三种账号自行发布笔记来实现。

## 企业号矩阵运营建议

了解了账号分类后，接下来我们探讨如何系统性地运营企业号矩阵。以下是打造企业号矩阵的具体建议：

**1. 积极参与品牌号活动**
若您运营婚纱、服装、美妆等品类的企业号，可以组织小号积极参与官方发起的试用、团购等活动。

**2. 个人号展示品牌属性**
在个人号的笔记内容中，应多次暗示品牌的定位属性。例如，品牌定位是城市年轻人还是职场年轻人，需在内容中清晰体现。

**3. 控制产品发布频率**
个人号发布品牌产品内容的频率不宜过高。建议采用 **三七分** 原则：**30%** 的内容用于营销推广，**70%** 的内容用于种草分享或知识科普。此比例能维持账号健康度。

**4. 利用引流号进行评论引流**
当企业号需要引流时，最佳方式是使用预备的引流小号，在相关笔记的评论区进行引导，从而实现软性引流。

## 官方广告投放详解

![](img/1b4208606c74791377da3758385315a2_7.png)

除了自建账号矩阵，另一种重要的推广方式是借助小红书官方广告。下面我们来看看官方广告的具体形式与申请流程。

![](img/1b4208606c74791377da3758385315a2_9.png)

小红书官方广告主要指 **CBC（Cost Per Click）竞价广告**，即按点击付费的广告模式。

![](img/1b4208606c74791377da3758385315a2_11.png)

### 广告申请流程
官方广告的申请与使用需遵循特定流程。您需要先拥有一个认证的企业号。

![](img/1b4208606c74791377da3758385315a2_13.png)

**企业号注册路径：**
1.  访问小红书官方网站。
2.  找到“业务合作”板块。
3.  进入“推广合作”页面。
4.  根据指引提交企业资质并支付审核认证费用。
5.  审核通过后，普通账号即升级为官方企业号。

拥有企业号后，便可申请投放信息流广告。

### 广告类型与投放位
官方信息流广告主要分为两大类：

**1. 信息流广告**
用户在浏览小红书首页信息流时，系统会不定期插入相关的广告内容。

**2. 搜索广告**
当用户使用关键词进行搜索时，系统会根据算法匹配并展示相关的广告内容。例如，用户搜索“帽子”，可能看到搭配推荐的“大衣”或“外套”广告；搜索“考研资料”，可能看到营养补充品广告。

搜索竞价广告的代表是关键词竞价排名，通过调整对特定关键词的出价，来竞争不同的广告展示位。当前信息流广告位主要分布在以下位置：
*   第3位
*   第13位
*   第23位

![](img/1b4208606c74791377da3758385315a2_15.png)

关于广告投放的常见问题，可在小红书官方平台的帮助中心或相关问答页面找到详细解答。

![](img/1b4208606c74791377da3758385315a2_17.png)

**请注意**：官方广告投放涉及一定的资质要求，且费用通常较高。在资金允许的情况下，建议考虑与官方合作投放竞价广告，因为其流量由平台直接推荐，效率更高。

![](img/1b4208606c74791377da3758385315a2_19.png)
