## 什么是自媒体？🤔

![](img/a7684bb861c1d0851b4ff57dcf3a7f4f_3.png)

上一节我们介绍了课程的整体框架，本节中我们来看看最核心的基础概念：到底什么是自媒体？

用最直白的话来解释：

**自媒体** = **个人** + **媒体平台** + **内容创作** + **流量变现**

![](img/a7684bb861c1d0851b4ff57dcf3a7f4f_5.png)

其操作流程可以简化为以下步骤：

![](img/a7684bb861c1d0851b4ff57dcf3a7f4f_7.png)

1.  打开手机，搜索并进入一个自媒体平台（如：抖音、快手、B站、微信公众号等）。
2.  在该平台注册一个属于你自己的账号。
3.  在你的账号上发布自己创作的内容（文章或视频）。
4.  内容获得观看（阅读），即可依靠流量结算收益。

![](img/a7684bb861c1d0851b4ff57dcf3a7f4f_9.png)

这个过程，就叫做自媒体。它本质上是个人利用互联网平台，创建并传播内容，从而获得影响力或经济回报的行为。
