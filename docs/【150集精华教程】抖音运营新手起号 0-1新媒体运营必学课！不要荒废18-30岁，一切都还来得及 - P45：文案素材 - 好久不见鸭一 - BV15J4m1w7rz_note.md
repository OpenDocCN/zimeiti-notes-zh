# 抖音运营新手起号：P45：文案素材与视频制作 🎬

在本节课中，我们将学习如何为已确定的视频创意寻找文案素材，并了解一种将文章素材转化为视频的创作方法。课程将介绍一个实用的工具网站，并讲解其使用方法。

上一节我们讨论了视频创意的确定，本节中我们来看看如何为创意寻找合适的文案内容。

## 文案素材的获取

![](img/218a6d8a70217018291c621154c91dd1_5.png)

![](img/218a6d8a70217018291c621154c91dd1_7.png)

如果视频创意已经确定，下一步就是搜索相应的文案。直接搬运他人视频素材并进行处理是一种方式。但若希望根据特定创意搜索文案，并以另一种形式自主创作视频，则需要掌握文案获取的方法。

![](img/218a6d8a70217018291c621154c91dd1_9.png)

文案素材方面，推荐一个名为“淘金阁”的工具网站。它是一个自媒体全网搜索引擎。

以下是“淘金阁”网站的主要特点：

![](img/218a6d8a70217018291c621154c91dd1_11.png)

*   它汇集了多个内容平台的图文素材，包括：今日头条、百家号、企鹅号、趣头条、一点资讯、QQ看点、凤凰网、网易资讯、搜狐网、东方网、简书、快资讯等。
*   因此，该网站拥有一个庞大的文案素材库。

![](img/218a6d8a70217018291c621154c91dd1_13.png)

![](img/218a6d8a70217018291c621154c91dd1_15.png)

## 如何使用“淘金阁”

![](img/218a6d8a70217018291c621154c91dd1_17.png)

在网站搜索框中，输入你想要的关键词。例如，搜索“星座”。

![](img/218a6d8a70217018291c621154c91dd1_19.png)

按下回车键后，点击“开始抓取”按钮。

![](img/218a6d8a70217018291c621154c91dd1_21.png)

下方区域会显示抓取到的所有相关文章。请注意，这里获取的是**文章素材**，而非视频素材。

![](img/218a6d8a70217018291c621154c91dd1_23.png)

![](img/218a6d8a70217018291c621154c91dd1_25.png)

将别人的文章素材转化为自己的视频内容，与直接搬运他人视频有本质区别，不属于违规搬运的范畴。

![](img/218a6d8a70217018291c621154c91dd1_27.png)

我们可以随意点开一篇文章进行查看，例如“分手后也能挽回，最放不下前任的三大星座”。

![](img/218a6d8a70217018291c621154c91dd1_29.png)

![](img/218a6d8a70217018291c621154c91dd1_31.png)

点击后即可直接查看文章全文，获得我们想要的文案内容，例如关于巨蟹座、双鱼座等的描述。

![](img/218a6d8a70217018291c621154c91dd1_33.png)

找到文案后，可以直接通过复制粘贴的方式取用，将其转化为自己的视频脚本。

![](img/218a6d8a70217018291c621154c91dd1_35.png)

## 视频制作的形式

![](img/218a6d8a70217018291c621154c91dd1_37.png)

文案找到后，接下来就是视频制作环节。

![](img/218a6d8a70217018291c621154c91dd1_39.png)

今天介绍一种视频制作形式。除了真人出镜拍摄、音乐影视混剪（可通过录屏或特定网站如“mv cat”下载影视素材进行剪辑）等方式外，我们还可以将**文章文案转化为视频**。

这是一种有效的创作方式。关键在于利用“5118”进行热词搜索，以及使用“淘金阁”进行文案搜索。大家需要掌握这些工具的使用方法。

课程结束后，请务必亲自搜索尝试，寻找自己能用的素材。

![](img/218a6d8a70217018291c621154c91dd1_41.png)

本节课中我们一起学习了如何利用“淘金阁”网站搜索文案素材，并了解了将文章内容转化为视频脚本进行再创作的方法。这为自主视频制作提供了一条清晰的路径。
