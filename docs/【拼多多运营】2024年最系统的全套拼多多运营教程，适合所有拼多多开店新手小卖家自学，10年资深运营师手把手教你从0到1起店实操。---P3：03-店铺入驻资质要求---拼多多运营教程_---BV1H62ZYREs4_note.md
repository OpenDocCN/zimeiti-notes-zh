# 拼多多运营教程：P3：店铺入驻资质要求 📋

在本节课中，我们将学习在拼多多平台开店前需要准备哪些资质和材料。了解清楚这些要求，是成功开设店铺的第一步。

## 概述

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_1.png)

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_2.png)

本节课是拼多多新手实操运营白皮书系列课程的第三章。我们将详细讲解开设个人店铺和企业店铺所需的不同资质证明，包括身份证、手机号、营业执照以及其他特定文件的要求。

## 个人店铺资质要求

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_4.png)

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_5.png)

上一节我们介绍了拼多多店铺的基本类型。本节中，我们来看看开设个人店铺需要准备哪些材料。

个人店铺根据经营主体不同，主要分为两类：以**个人**为主体的店铺和以**个体工商户**为主体的店铺。

### 个人主体店铺

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_7.png)

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_8.png)

以下是开设个人主体店铺需要准备的核心材料：

*   **身份证**：需要提供**大陆居民身份证**的原件正反面清晰照片。证件有效期需大于3个月，且照片不可倒置。
*   **手机号码**：需要一个**未注册过拼多多店铺**的手机号码。官方允许退店后复用该号码，但实践中可能存在隐形限制，建议谨慎使用。

关于**经营范围**，根据《电子商务法》第十条规定，符合以下情形无需办理营业执照：
1.  个人销售自产农副产品、家庭手工业产品。
2.  个人利用自身技能从事依法无需取得许可的便民劳务活动。
3.  个人进行零星小额交易活动。
4.  依照法律、行政法规不需要进行登记的其他情形。

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_10.png)

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_11.png)

**不属于以上情形的，则需要办理营业执照。** 销售食品、药品等特殊商品还需另行办理相关资质。

### 个体工商户店铺

个体工商户店铺的资质要求与个人主体店铺大体相似，但增加了一项重要文件。

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_13.png)

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_14.png)

以下是开设个体工商户店铺需要准备的材料清单：

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_15.png)

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_16.png)

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_18.png)

*   **身份证**：经营者本人的大陆居民身份证原件正反面清晰照片。
*   **手机号码**：一个未注册过拼多多店铺的手机号码。
*   **个体工商户营业执照**：需提供营业执照原件照片。若为复印件或扫描件，必须加盖公章。执照有效期同样需大于3个月。

准备好上述材料后，即可申请开设对应的个人店铺，流程相对简单。

## 企业店铺资质要求

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_23.png)

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_24.png)

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_25.png)

了解了个人店铺的要求后，本节我们来看看企业店铺的资质要求。企业店铺根据类型不同，所需材料更为复杂。

### 基本身份信息

开设企业店铺，需要提供两个人的身份信息：

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_27.png)

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_28.png)

1.  **企业法定代表人**的身份证：可以是大陆、港澳台或海外地区的身份证。
2.  **店铺管理人**的身份证：**必须为大陆居民身份证**。

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_29.png)

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_30.png)

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_31.png)

其他要求（如清晰度、有效期）与个人店铺相同。

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_32.png)

### 按店铺类型划分的附加资质

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_34.png)

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_35.png)

企业店铺主要分为旗舰店、专卖店、专营店和普通店。不同类型店铺需要提交的附加资质不同。

以下是各类企业店铺所需的附加资质清单：

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_37.png)

*   **旗舰店**
    *   **经营一个或多个自有品牌**：需提供**企业三证**（或三证合一后的营业执照）、**质检报告**、**商标注册证**。
    *   **经营一个授权品牌**：除上述三项外，还需提供**品牌授权书**，且必须为**一级授权**。
    *   **卖场型品牌旗舰店**：通常为平台邀请入驻，对普通卖家参考价值不大。

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_38.png)

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_39.png)

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_40.png)

*   **专卖店**
    *   **经营一个自有品牌**：需提供**企业三证**、**质检报告**、**商标注册证**。
    *   **经营一个授权品牌**：除上述三项外，还需提供**品牌授权书**，授权等级**不能超过二级**。

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_41.png)

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_42.png)

*   **专营店**
    *   **经营多个品牌（无论自有或授权）**：均需提供**企业三证**、**质检报告**、**商标注册证**及相应的**品牌授权书**（如涉及授权品牌）。

*   **普通企业店**
    *   资质要求最简单，通常仅需提供**企业三证**（或三证合一后的营业执照）及**银行开户许可证**即可。

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_44.png)

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_46.png)

> **核心概念提示**：
> *   **企业三证**：通常指《营业执照》、《组织机构代码证》、《税务登记证》。现在多为“三证合一”后的统一社会信用代码营业执照。
> *   **授权等级**：品牌授权链条中的层级，例如：品牌方(一级) -> 总代理(二级) -> 经销商(三级)。级别越高，授权效力通常越强。

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_48.png)

不同商品类目可能有额外的特殊资质要求，这部分内容较多，我们将在后续课程中详细分享。

## 总结

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_50.png)

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_51.png)

本节课我们一起学习了拼多多店铺入驻的资质要求。我们主要掌握了两点：
1.  **个人店铺**（含个体工商户）主要需要**身份证**、**手机号**，并根据经营情况判断是否需要**营业执照**。
2.  **企业店铺**需要提供**法人**和**管理人**的身份证，并根据**旗舰店、专卖店、专营店、普通店**等不同类型，准备相应的**企业三证、质检报告、商标注册证及品牌授权书**等附加资质。

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_52.png)

![](img/ff9978c4e8e2cfcf922ea7e49288ca54_54.png)

确保提前备齐所有符合要求的材料，是顺利通过店铺审核、快速开启运营的关键。如果在实际操作中遇到任何问题，欢迎随时交流探讨。
