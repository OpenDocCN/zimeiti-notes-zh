## 笔记被推荐的核心逻辑

笔记发布后，平台会给予一个基础的展示量（例如新号约1000次展示）。笔记能否获得更多流量，取决于它能否通过平台的“数据考核”，实现层级的跨越。这个考核主要围绕两个核心指标。

### 第一关键：打开率

打开率是指看到你笔记封面的用户中，有多少人点击进入了笔记详情页。这是笔记能否进入下一流量池的第一道门槛。

*   **考核标准**：在同期推荐的约20篇同类型笔记中，你的笔记**打开率需超过11%**。
*   **运作方式**：假设平台给你的笔记**基础展示量（`BaseImpressions`）** 是1000次。你需要获得至少 `1000 * 11% = 110` 次点击（即“小眼睛”数）。达标后，系统才会将你的笔记推荐给更多人，例如将展示量提升到2000-5000次。
*   **未达标后果**：如果打开率低于11%，笔记通常只会获得一波基础推荐，流量很快停滞。

### 第二关键：互动率

互动率是指点击进入笔记的用户中，进行点赞、收藏、评论、转发等互动行为的比例。这是笔记冲击更高流量层级的核心。

![](img/6d2e712117777d146cd6f79a5eb28297_9.png)

*   **考核标准**：笔记的**平均互动率需达到3%以上**。
*   **运作方式**：承接上例，如果有110人点击了笔记（`Clicks = 110`），那么至少需要有 `110 * 3% ≈ 4` 个互动（如评论）。互动率达标后，笔记有资格进入万级甚至十万级的更大流量池。
*   **升级路径**：笔记的推荐升级路径可以简化为：**基础展示 → 打开率达标 → 进入更大流量池 → 互动率达标 → 进入更大流量池**。

![](img/6d2e712117777d146cd6f79a5eb28297_11.png)

![](img/6d2e712117777d146cd6f79a5eb28297_13.png)

![](img/6d2e712117777d146cd6f79a5eb28297_15.png)

**核心公式总结**：
```
晋级条件1: (Clicks / Impressions) > 11%
晋级条件2: (Engagements / Clicks) > 3%
```
其中，`Impressions`为展示量，`Clicks`为点击量，`Engagements`为互动量（评论、点赞等）。

### 账号权重与流量天花板

![](img/6d2e712117777d146cd6f79a5eb28297_19.png)

*   **账号权重影响起点**：账号权重越高，获得的**基础展示量（`BaseImpressions`）** 也越高。
*   **流量层级有上限**：对于新号，通常最多能通过自然流量冲到第三或第四层级。更高的流量层级（如第六级及以上）往往需要结合内容质量和商业投放（如薯条、信息流广告）。

## 必须避免的运营“坑”

了解推荐逻辑后，我们还需要避开平台规则中的常见陷阱，以保护账号健康。以下是运营中需要特别注意的几点：

*   **新号勿立即发笔记**：新注册账号应首先“养号”，模拟正常用户行为（浏览、互动）数日，建立初始标签。立即发笔记会浪费平台给予新号的初始流量权重。
*   **违规必定会降权**：任何违规行为（如导流、抄袭、虚假营销）都会导致笔记推荐减少、搜索排名下降，甚至账号被限流。恢复权重需要很长时间。
*   **标签切忌反复修改**：账号、店铺的人群标签一旦确定，频繁更改会扰乱系统推荐，可能导致流量下滑。
*   **设备登录需专一**：尽量避免一台手机登录多个小红书账号，建议遵循“一机一卡一号”原则，减少关联风险。
*   **谨慎处理引流内容**：在笔记、评论、私信中直接留联系方式或引导至其他平台，是高风险行为，极易被判定违规。
*   **内容原创是底线**：搬运、抄袭他人内容，不仅无法获得推荐，还会受到处罚。
