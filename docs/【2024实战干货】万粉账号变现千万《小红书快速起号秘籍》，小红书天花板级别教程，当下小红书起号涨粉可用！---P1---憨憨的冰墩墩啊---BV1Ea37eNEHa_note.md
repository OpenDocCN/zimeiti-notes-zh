## 项目原理

**公式：假装存钱 = 设定虚拟消费场景 + 记录“支出” + 实际储蓄**

我们通过创作以“假装存钱”为主题的笔记，设定各种生活化、有趣的场景，来引起大众共鸣。这能吸引大量用户参与互动和打卡，从而实现快速吸粉和转化。利用这种模式，账号粉丝量可以快速增长。

## 账号准备

首先，你需要准备一个小红书账号。

*   **账号注册**：必须使用正规手机号注册。新号和老号均可操作，但新号享有平台流量扶持。若使用老号，需将之前不相关的作品隐藏，以保证账号内容垂直度。
*   **养号策略**：无论新号老号，都必须经历养号过程，以提高账号权重和流量精准度。
    *   **增加权重**：可以在小红书商城进行小额消费（如一分钱商品），提升账号信誉。
    *   **刷同类型内容**：每天累计刷30分钟以上与“存钱”、“宠物”、“生活记录”相关的笔记。分多次进行（如早、中、晚各10分钟），并积极关注、点赞、评论。这有助于系统为你的账号打上精准标签。
    *   **积累素材**：在养号过程中，观察并学习同行（对标账号）的内容形式和创意。
