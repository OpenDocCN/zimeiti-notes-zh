# 微信视频号运营教程：P46：手机剪辑基础流程

在本节课中，我们将学习一个适用于初学者的手机剪辑基础流程。这个流程侧重于高效剪辑与基础去重操作，旨在帮助大家快速完成视频制作。

## 概述

本节教程将拆解一个完整的手机剪辑步骤，从素材的初步处理到添加广告、画中画、特效等元素，最后进行导出和二次处理。我们将使用简单直白的语言，确保每位初学者都能跟上操作。

## 剪辑准备与初剪

![](img/de47bd73ebb5f902dfc5dbcdc4c6a746_5.png)

上一节我们介绍了剪辑的整体思路，本节中我们来看看具体的操作步骤。剪辑的第一步是对原始素材进行基础处理。

1.  **放大画面**：将视频画面稍微放大一些。这个操作有助于二次构图和去重。
2.  **掐头去尾**：剪掉视频开头和结尾不必要的部分。在熟悉剧情或使用创作工具生成文案后，可以快速定位剪辑点，避免反复观看浪费时间。
3.  **精简内容**：根据需要对视频中间部分进行裁剪。不必完全保留原片时长，可以删除冗余片段以保持节奏。

---

## 广告素材的添加与衔接

完成初剪后，下一步是在视频中插入广告素材。广告的插入时机和衔接方式至关重要。

![](img/de47bd73ebb5f902dfc5dbcdc4c6a746_7.png)

我通常在视频播放到八九分钟或更靠后的位置添加广告。如果感觉该位置不合适，也可以调整到前面。

以下是插入广告的具体步骤：

1.  **定位分割**：在选定的时间点，将主视频轨道分割开。
2.  **选择广告素材**：从素材库中选择广告片段。可以拼接多个同类产品的不同素材，以增加丰富度。
3.  **检查内容**：确保广告素材中没有直接标明价格（如“多少钱”）。如有价格信息，需将其裁剪掉。
4.  **设置时长**：广告片段的总时长建议控制在**一分半到两分钟**。太短可能效果不佳。
5.  **自然衔接**：为避免广告生硬，需要在广告与前后视频的连接处添加“叠化”转场效果。将转场时长设置为 **2 或 3** 帧，可以使过渡更平滑。

![](img/de47bd73ebb5f902dfc5dbcdc4c6a746_9.png)

## 运用画中画进行去重

添加广告后，我们可以通过画中画功能来进一步处理视频，达到去重和增加原创度的目的。

![](img/de47bd73ebb5f902dfc5dbcdc4c6a746_11.png)

我一般会添加两张个人照片作为画中画图层。

以下是操作步骤：

![](img/de47bd73ebb5f902dfc5dbcdc4c6a746_13.png)

1.  **添加第一张画中画**：导入一张个人照片，将其铺满整个屏幕。
2.  **设置混合模式**：将照片的混合模式调整为“叠加”，并将透明度调到最低（**2 或 3**）。
3.  **添加第二张画中画**：重复上述步骤，添加第二张个人照片。使用自己的照片效果通常更好。
4.  **添加矢量图元素**：在素材库中搜索“矢量图”，选择一个时长较长的横屏或竖屏素材。
5.  **处理矢量图**：将矢量图素材放大，混合模式同样设置为“叠加”。如果素材长度不够，可以通过**复制**片段来铺满整个时间轴；如果过长，则进行裁剪。
6.  **调整音频**：可以适当调慢画中画素材的音量，避免干扰主视频声音。
7.  **添加个人标识**：为了强化去重效果，可以在画面一角添加自己的名字水印。安卓手机可将其置于一角，苹果手机可将其移出画面外。

## 特效、滤镜与贴纸的运用

画中画处理完毕后，可以通过特效和装饰元素进一步提升视频的观感。

我会添加两个特效和一个滤镜。

以下是具体操作：

1.  **添加画面特效**：添加一至两个画面特效。如果有VIP权限，可以选择更多特效。将特效的“作用对象”设置为对应的画中画图层，并让特效覆盖整个片段。
2.  **添加滤镜**：选择一个滤镜，强度调整到 **20-30** 左右即可，避免效果过重。**注意**：如果视频中插入了广告，务必确保滤镜效果覆盖到广告片段之后的部分。
3.  **添加贴纸**：在画面的四个角分别添加一个贴纸，这是常用的装饰方法。选择四个不同的贴纸，将它们放置到合适位置，并拖拽时长使其覆盖整个视频。

## 最终导出与二次处理

所有剪辑操作完成后，便进入导出阶段。我通常会使用“秒剪”App进行最终处理。

以下是导出流程：

![](img/de47bd73ebb5f902dfc5dbcdc4c6a746_15.png)

1.  **分段处理（可选）**：如果想进一步复杂化视频以去重，可以将视频按**每20秒一段**进行分割，并为每一段更换不同的背景底色。
2.  **初次导出**：在剪辑软件中，将视频以 **1080P** 分辨率导出到手机相册。
3.  **使用秒剪处理**：打开“秒剪”App，从相册导入刚刚导出的视频。
4.  **直接发布**：在秒剪中，无需进行任何额外编辑，直接点击“发表视频”，选择发布到视频号即可。近期实践表明，经过秒剪处理的视频，平台流量表现可能更好。

![](img/de47bd73ebb5f902dfc5dbcdc4c6a746_17.png)
