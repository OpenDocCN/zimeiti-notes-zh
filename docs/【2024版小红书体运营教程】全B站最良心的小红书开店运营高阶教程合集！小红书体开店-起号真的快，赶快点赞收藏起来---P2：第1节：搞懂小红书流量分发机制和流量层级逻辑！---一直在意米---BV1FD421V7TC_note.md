## 认识小红书与主要流量板块

![](img/6475e485d2ecbef806a61eae2661647e_3.png)

首先，我们来认识小红书App及其主要的流量来源。小红书的流量主要集中在几个特定板块。

以下是小红书最主要的四个流量入口：

1.  **发现页**
    *   这是最主要的流量来源。平台会根据用户的资料和行为数据进行**特征匹配**，向用户推荐可能感兴趣的笔记或视频。例如，如果账号注册时选择了“宠物”、“美食”等标签，发现页就会优先推荐相关的内容。

2.  **位置页**
    *   该板块的内容基于用户的**地理位置**进行推荐。例如，定位在上海，页面就会展示上海的景点、餐厅、展览等相关笔记，具有很强的地域属性。

![](img/6475e485d2ecbef806a61eae2661647e_5.png)

3.  **关注页**
    *   这里展示用户所关注账号发布的内容。对于新账号，系统也会根据算法推荐一些可能感兴趣的账号。当你的账号运营良好时，你的笔记也可能通过信息流的形式出现在其他用户的关注页中。

4.  **搜索框**
    *   搜索是用户主动寻找信息的关键入口。通过**热门搜索词**，内容创作者可以蹭到热点流量。从搜索而来的用户通常目的明确，用户质量相对较高。

整个小红书的流量主要就分布在这四个板块：发现页、位置页、关注页和搜索框。

![](img/6475e485d2ecbef806a61eae2661647e_7.png)

本节课中我们一起学习了小红书平台的四大核心流量入口：算法推荐的**发现页**、基于地理的**位置页**、粉丝导向的**关注页**以及精准需求的**搜索框**。理解这些流量来源，是后续进行内容策划和引流的基础。
