# 抖音运营教程：P42：产品与用户兴趣标签 🧩

在本节课中，我们将学习一种名为“九宫格选题法”的实用技巧。该方法通过结合产品属性与用户兴趣，帮助创作者系统性地挖掘源源不断的视频选题，解决“不知道拍什么”的难题。

上一节我们介绍了选题的重要性，本节中我们来看看如何具体操作。

## 第一步：构建产品标签九宫格 🏷️

我们以健身软件Keep为例进行讲解。首先，将核心产品“Keep”置于九宫格中心。

以下是围绕产品联想其特点或属性的步骤，请将想到的关键词填入周围的八个格子中。

1.  **线上健身软件**：这是产品的基本形态。
2.  **专业健身教练**：产品提供专业指导服务。
3.  **饮食搭配建议**：产品包含营养学相关内容。
4.  **减肥/健身攻略**：产品提供目标导向的方案。
5.  **运动干货知识**：产品具有教育属性。
6.  **科学运动方法**：强调其内容的权威性。
7.  **打卡运动制度**：产品具有社群或激励体系。
8.  **一二线城市用户为主**：这是产品的用户画像特征。

这些标签源于产品自身的功能、内容或市场定位。完成填充后，产品侧的九宫格就构建完毕了。

## 第二步：构建用户兴趣标签九宫格 👥

接下来，我们分析产品的核心用户。将“健身爱好者/想减肥的人”置于另一个九宫格的中心。

以下是围绕目标用户联想其可能感兴趣的内容，用以扩大选题范围。

1.  **健身房**：线下健身场所，是兴趣的延伸。
2.  **健身装备**：如运动服饰、器材等。
3.  **轻脂美食**：与健康饮食相关的兴趣点。
4.  **旅行**：热爱生活的人可能喜欢的活动。
5.  **星座**：大众化的娱乐兴趣。
6.  **明星**：娱乐八卦，具有高关注度。
7.  **心灵鸡汤**：情感共鸣类内容。
8.  **时尚穿搭**：与外形、生活品味相关。

这一步的目的是将核心用户群扩展为更广泛的“生活热爱者”，从而让选题能触达更多潜在观众。

## 第三步：进行“连连看”组合创意 💡

现在，我们手头有两个填满的九宫格。选题的秘诀在于将两个九宫格中的标签进行随机或创意连接。

以下是具体的操作方法，即从两个九宫格中各选一个标签进行组合，形成一个具体的选题方向。

*   **示例组合1**：`饮食搭配` + `旅行`
    *   **选题方向**：“外出旅行时，如何享受美食又能控制体重？”
*   **示例组合2**：`运动方法` + `明星`
    *   **选题方向**：“赵丽颖都在练的5个高效瘦身动作，你一定要学！”
*   **示例组合3**：`科学` + `健身房`
    *   **选题方向**：“揭秘健身房最科学的训练流程，别再瞎练了！”

通过这种“连连看”的方式，理论上可以衍生出**8 x 8 = 64**种基础组合，每种组合又能启发多个具体的视频选题，从而实现选题的无限拓展。

## 第四步：优化与拓展选题 🔍

然而，初步组合产生的选题想法可能不够完善或吸引人。这时，我们需要借助工具进行优化。

以下是完善选题的具体建议：

1.  **利用搜索工具**：将组合出的关键词（如“科学健身”）输入搜索引擎或抖音搜索框，参考自动联想的短语和热门内容，优化你的标题和角度。
2.  **建立素材库**：将每一个优质的选题灵感记录下来，分类整理，形成你的专属选题素材库。
3.  **判断选题质量**：一个好的选题通常需要具备**用户兴趣**（吸引点击）、**产品关联**（贴合账号定位）和**可操作性**（能够拍摄制作）。需过滤掉不相关或低质的方向。

本节课中我们一起学习了“九宫格选题法”。该方法的核心在于**系统拆解**和**创意连接**：首先拆解产品属性和用户兴趣，形成标签库；然后通过跨标签组合，碰撞出新颖的选题灵感。掌握这一方法，你将能有效解决内容枯竭的问题，为账号带来源源不断的创作思路。
