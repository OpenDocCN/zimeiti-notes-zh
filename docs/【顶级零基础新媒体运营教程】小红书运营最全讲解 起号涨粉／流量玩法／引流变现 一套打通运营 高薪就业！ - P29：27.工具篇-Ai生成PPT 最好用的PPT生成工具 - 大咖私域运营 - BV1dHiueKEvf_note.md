# 顶级零基础新媒体运营教程：P29：27.工具篇-AI生成PPT 🚀

在本节课中，我们将学习如何使用一款高效的AI工具来自动生成PPT。这款工具能根据你提供的主题或现有文档，自动梳理大纲并生成详细的演示文稿内容，极大地提升工作效率。

## 工具简介与模式

上一节我们介绍了多种新媒体运营工具，本节中我们来看看这款专门用于PPT制作的AI工具。该工具主要提供两种生成模式：

1.  **主题生成模式**：你只需输入一个核心主题，工具便会结合类似ChatGPT的生成式AI能力，自动创作出完整的PPT全文。
2.  **文档生成模式**：你可以上传已有的文档（如TXT、PDF文件），AI助手将自动总结文档内容，提炼出大纲，并基于此大纲生成对应的PPT内容。

在众多AI生成PPT的工具中，这款工具因其出色的内容理解和组织能力而备受推荐。主题生成模式产生的PPT通常非常详尽，页数可能多达六七十页，内容全面但偶有重复。接下来，我们将通过实际操作来演示其文档生成功能。

## 编辑与优化生成内容

进入编辑界面后，我们可以看到AI生成的主体内容。

![](img/11dceb7bab2ecb94504723fee5ccf88c_13.png)

该工具的核心优势在于，它并非简单地将你提供的全文进行格式转换，而是能**智能分析内容，并将其归类、填充到不同的大纲节点下**。这对于只有大纲或要点、需要扩展详细内容的场景尤为实用。

虽然当前内置的模板库选择不算非常丰富，但模板切换非常迅速。我们可以通过右侧的“更换模板”功能，为PPT选择一个更合适的视觉风格。

![](img/11dceb7bab2ecb94504723fee5ccf88c_15.png)

此外，在编辑界面右侧，还提供了丰富的调整选项，例如修改文字、替换形状等，允许我们对生成的内容进行精细化调整。

![](img/11dceb7bab2ecb94504723fee5ccf88c_15.png)

![](img/11dceb7bab2ecb94504723fee5ccf88c_17.png)

![](img/11dceb7bab2ecb94504723fee5ccf88c_17.png)

如果对在线编辑器的功能不满意，一个高效的流程是：先在工具内选定一个大致满意的模板，然后将PPT下载到本地，再使用PowerPoint或Keynote等专业软件进行深度调整和美化。
