# 小红书运营：P4：店铺后台基础设置 🛠️

在本节课中，我们将学习小红书店铺开通后，必须完成的几项后台基础设置。这些设置是店铺正常运营和交易的前提，包括缴纳保证金、配置支付方式、设置运费模板与退货地址等。我们将逐一详细讲解，确保你的店铺能够顺利开张。

![](img/fd559f222c43f3def2915687e50933cb_1.png)

## 保证金 💰

店铺开通后，必须缴纳保证金。

![](img/fd559f222c43f3def2915687e50933cb_3.png)

保证金的缴纳路径是：在店铺商家管理首页点击“资金”，然后选择“店铺保证金”，最后点击“充值”。

![](img/fd559f222c43f3def2915687e50933cb_1.png)

![](img/fd559f222c43f3def2915687e50933cb_5.png)

![](img/fd559f222c43f3def2915687e50933cb_3.png)

![](img/fd559f222c43f3def2915687e50933cb_7.png)

大部分类目的保证金金额为 **1000元**。无论是个人店（个体店）还是企业店（工商店），通常都是这个标准。

## 支付方式 💳

完成保证金缴纳后，下一步是设置支付方式。

支付方式在“货款资金”页面进行设置，支持微信支付和支付宝。

![](img/fd559f222c43f3def2915687e50933cb_19.png)

以下是设置时需要注意的关键点：
*   **主体一致性**：你用于收款的支付宝和微信账户，其认证主体必须与开店主体完全一致。
*   **开通流程**：
    *   **支付宝**：通常可以当天完成认证并生效。
    *   **微信支付**：流程稍长。第一天提交认证并等待审核，审核通过后，需要在第二天进行签约操作，签约完成后微信支付才正式生效。

一个常见的错误是忽略了微信支付的签约步骤，导致店铺已产生销售却无法通过微信收款，严重影响顾客付款体验，务必注意。

![](img/fd559f222c43f3def2915687e50933cb_21.png)

## 运费险设置 🛡️

![](img/fd559f222c43f3def2915687e50933cb_23.png)

建议为店铺开通运费险。

![](img/fd559f222c43f3def2915687e50933cb_21.png)

小红书平台非标品（如服饰、家居等款式多样的商品）较多，消费者对运费险比较关注。在实际运营中，爆款笔记的评论区经常会有用户询问是否支持运费险。开通运费险能有效提升这类用户的购买转化率。

![](img/fd559f222c43f3def2915687e50933cb_23.png)

![](img/fd559f222c43f3def2915687e50933cb_25.png)

## 地址与运费模板设置 📦

接下来，我们设置发货/退货地址和运费模板。

在后台点击“交易”，进入“物流工具”进行设置。

![](img/fd559f222c43f3def2915687e50933cb_25.png)

![](img/fd559f222c43f3def2915687e50933cb_27.png)

### 地址设置

建议将“发货地址”和“退货地址”分开设置。
*   **发货地址**：填写一个国内的真实地址即可，例如：`联系人：张三， 电话：13800138000， 地址：XX省XX市XX区XX路XX号`。
*   **退货地址**：**这是无货源模式下的一个关键设置**。由于平台支持“七天无理由退货”，当顾客以此理由申请退货时，系统会自动同意并将**默认退货地址**提供给顾客。如果你填写真实地址，货物将退回给你，但作为无货源卖家，你需要让顾客将货退给上游供应商。

因此，退货地址建议按以下方式填写：
`中国台湾，具体地址一定联系客服发你正确地址`
![](img/fd559f222c43f3def2915687e50933cb_27.png)

这样设置后，当顾客申请退货时，他们会看到此地址并主动联系客服，此时你再将供应商的正确地址发给顾客即可。如果填写“台湾”地址后续可能有问题，也可以使用国内一些偏远地区（如新疆、西藏）的地址代替。目前平台暂无法修改“七天无理由”默认的退货地址逻辑，这是经过与客服确认的可行方案。

![](img/fd559f222c43f3def2915687e50933cb_29.png)

![](img/fd559f222c43f3def2915687e50933cb_29.png)

### 运费模板设置

在“物流工具”中创建和管理运费模板。

![](img/fd559f222c43f3def2915687e50933cb_31.png)

对于新手，通常创建一个“非偏远地区包邮”的模板即可。你可以创建多个模板（如指定地区包邮、全场包邮等），在上架商品时选择对应的模板。

![](img/fd559f222c43f3def2915687e50933cb_31.png)

![](img/fd559f222c43f3def2915687e50933cb_33.png)

## 运费保（运费险）规则详解 📝

![](img/fd559f222c43f3def2915687e50933cb_33.png)

小红书将运费险称为“运费保”，其规则与其他平台略有不同。

![](img/fd559f222c43f3def2915687e50933cb_35.png)

![](img/fd559f222c43f3def2915687e50933cb_37.png)

![](img/fd559f222c43f3def2915687e50933cb_35.png)

主要规则如下：
*   **上门取件**：如果顾客选择快递员上门取件，平台补贴的运费最高为 **20元**。
*   **自行寄回**：如果顾客自行联系快递寄回商品，无论实际支付多少运费，平台统一补贴 **7元**。这部分补贴是为了避免运费纠纷。

![](img/fd559f222c43f3def2915687e50933cb_37.png)

了解这个规则有助于你在处理售后时向顾客解释。

![](img/fd559f222c43f3def2915687e50933cb_39.png)
