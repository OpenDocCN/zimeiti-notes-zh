# 小红书运营系统课：P68：5.渐变粉色效果

在本节课中，我们将学习如何为视频制作一种渐变樱花色的视觉效果。这种效果能让你的视频画面色彩过渡更加柔和、富有氛围感，非常适合用于情感类、美妆类或风景类的内容创作。

## 概述

![](img/6e568d0c7016626446b5cdff216ed3d8_1.png)

我们将通过为视频添加滤镜和关键帧动画，实现色彩从原始状态平滑过渡到樱花粉色，再渐变回原始状态的效果。整个过程主要涉及**关键帧**的运用和**色彩参数**的调节。

---

## 操作步骤详解

上一节我们介绍了视频剪辑的基础操作，本节中我们来看看如何实现这种动态的色彩渐变效果。

首先，你需要准备一段带有文字或其他内容的视频素材，并将其导入到剪辑软件中。

以下是实现渐变樱花色效果的具体步骤：

1.  **定位时间点**：在时间轴上选中视频片段，将播放头移动到你希望色彩**开始变化**的起始位置。
2.  **添加起始关键帧**：在起始位置，为视频的**滤镜**或**调节**属性添加一个关键帧。此时可以暂时不添加任何滤镜，或保持滤镜强度为0。
3.  **选择并应用滤镜**：打开滤镜库，选择一款能营造粉色或樱花色调的滤镜（例如：粉白、樱粉、初恋等）。
4.  **调整色彩参数**：为了效果更佳，可以进一步微调色彩。点击**调节**功能，进行如下设置：
    *   **对比度 (Contrast)**：`+12`
    *   **饱和度 (Saturation)**：`+9`
    *   **色温 (Temperature)**：`+17`
5.  **设置结束关键帧**：将播放头移动到希望色彩**变化结束**的位置，再次添加一个关键帧。此时应确保滤镜效果和调节参数已应用到位。
6.  **完成渐变复原**：将播放头移动到希望色彩**恢复原样**的结束位置，添加最后一个关键帧。在此关键帧处，将**滤镜的强度或效果值调整为0**。这样，软件就会自动创建从粉色渐变回原始画面的动画。
7.  **添加背景音乐**：最后，为视频添加一段合适的背景音乐（例如从抖音收藏的音乐）。将音乐轨道与视频对齐，并分割删除掉多余的音乐部分。

## 效果预览与总结

![](img/6e568d0c7016626446b5cdff216ed3d8_5.png)

按照以上步骤操作后，你就可以预览到视频画面色彩平滑地变为樱花粉色，再缓缓恢复如初的动态效果了。

本节课中我们一起学习了如何利用**关键帧动画**和**色彩调节**来制作渐变樱花色效果。核心在于通过设置**起始状态关键帧**、**效果峰值关键帧**和**结束状态关键帧**，来控制滤镜强度的变化，从而实现色彩的动态渐变。掌握这个技巧后，你可以举一反三，将其应用于其他滤镜或特效，创造出更多丰富的视觉变化。
