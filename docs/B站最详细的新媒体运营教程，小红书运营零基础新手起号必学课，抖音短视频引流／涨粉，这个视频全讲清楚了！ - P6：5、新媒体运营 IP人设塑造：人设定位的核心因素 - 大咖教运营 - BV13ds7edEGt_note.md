### 1. 角色：你的专业身份

![](img/3826f9cb6b2faf937481b33fd9c7a338_82.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_84.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_88.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_90.png)

角色，简单来说，就是你在特定领域中的专业身份。你选择什么赛道、准备输出什么方向的内容，就需要塑造与之对应的专家形象。

![](img/3826f9cb6b2faf937481b33fd9c7a338_92.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_94.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_96.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_98.png)

*   **公式：角色 = 赛道方向 + 专业身份**
*   **示例**：
    *   **装修博主**：你需要展示自己确实有房子在装修或已装修完成，通过记录过程、分享避坑经验、展示前后对比来建立专业性和可信度。
    *   **母婴赛道**：最常见的角色是“宝妈”。但根据内容方向，你也可以塑造为“育儿护理专家”，这不一定要求你自身是宝妈，可以是相关领域的专业人员。
    *   **情感赛道**：可以塑造为“情感大师”、“心理学顾问”或“成长型人设”。成长型人设侧重于分享个人在情感、职场、创业等方面的真实经历和心路历程，例如：“一位30岁女性如何应对家庭与事业挑战”。

![](img/3826f9cb6b2faf937481b33fd9c7a338_108.png)

**核心要点**：你的角色应与你的内容赛道紧密绑定，目标是让用户相信你提供的内容具有专业性。

### 2. 内容：垂直与价值

![](img/3826f9cb6b2faf937481b33fd9c7a338_120.png)

内容是人设的载体。关于内容定位，我们之前强调过几个关键点：是否吸引用户、是否有价值、是否有平台特色。在人设定位的语境下，内容的核心要求是**垂直**。

![](img/3826f9cb6b2faf937481b33fd9c7a338_130.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_132.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_134.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_136.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_138.png)

*   **代码逻辑**：
    ```python
    if 内容主题 == 账号核心赛道:
        有利于打牢账号标签
        有利于平台精准推流
    else:
        可能导致用户流失
        影响平台流量分发效果
    ```
*   **解释**：你必须持续输出与你所选赛道相关的内容。内容不垂直会扰乱账号标签，导致平台将你的内容推送给不感兴趣的用户，从而拉低互动数据（如完播率、点赞），最终影响流量推荐效果。例如，第一篇讲儿童教育，第二篇讲宠物，喜欢教育的用户可能对宠物内容不感兴趣，直接划走，这就向平台传递了负面信号。

![](img/3826f9cb6b2faf937481b33fd9c7a338_140.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_142.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_144.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_146.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_148.png)

**核心要点**：坚持内容垂直，是强化人设、获得稳定流量的基础。

### 3. 形式：呈现内容的方式

![](img/3826f9cb6b2faf937481b33fd9c7a338_160.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_162.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_168.png)

形式指的是你呈现内容的具体方式，主要包括产出场景、声音呈现和内容类型。

![](img/3826f9cb6b2faf937481b33fd9c7a338_170.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_172.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_174.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_176.png)

*   **产出场景**：你的内容在哪里拍摄？是固定的室内场景（如书房、客厅），还是多变的室外场景（如通勤路上、咖啡厅）？可以参考对标账号的常用场景。
*   **声音呈现**：你是亲自出镜讲解，还是使用配音？现在利用“剪映”等工具进行后期配音已非常方便。
*   **内容类型**：你的内容主要采用哪种形式？
    *   **剧情型**：创作难度较高，需要剧本和表演。
    *   **干货/经验分享型**：需要精心准备文案脚本，输出有价值的信息。
    *   **生活记录型**：需要积累大量的日常拍摄素材，真实感强。

![](img/3826f9cb6b2faf937481b33fd9c7a338_178.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_180.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_182.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_184.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_186.png)

**核心要点**：根据你的个人能力和内容方向，选择一种适合且能持续产出的形式，并尽量保持统一。

![](img/3826f9cb6b2faf937481b33fd9c7a338_188.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_190.png)

### 4. 记忆点：让用户记住你的关键

![](img/3826f9cb6b2faf937481b33fd9c7a338_198.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_200.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_202.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_204.png)

记忆点是让人设脱颖而出的“魔法”。它是你刻意设计、并在内容中反复出现的元素，目的是在用户心中留下深刻印象。

![](img/3826f9cb6b2faf937481b33fd9c7a338_206.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_208.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_210.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_212.png)

以下是几种常见的记忆点设计，它们可以根据出现的位置分为两类：

![](img/3826f9cb6b2faf937481b33fd9c7a338_214.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_216.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_218.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_220.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_222.png)

*   **视频开头前置（强化冲击）**：
    *   **固定动作**：每期视频开始时的标志性动作。
    *   **固定特效**：片头使用的统一视觉特效。
    *   **固定开场白**：每期视频的第一句话。
    *   **设计原理**：将这些元素放在视频前几秒，当用户刷到你多个视频时，重复的强刺激能最快速度加深记忆，比放在视频中后部效果更直接。

![](img/3826f9cb6b2faf937481b33fd9c7a338_224.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_226.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_228.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_230.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_232.png)

*   **贯穿视频全局（持续强化）**：
    *   **统一BGM（背景音乐）**：使用同一首或同一风格的音乐，音乐响起用户就能联想到你。
    *   **标志性物品**：始终出现在画面中的某个物品（如特定的台灯、玩偶）。
    *   **统一风格的服装**：符合你人设的特定着装风格（如职场风、田园风）。

![](img/3826f9cb6b2faf937481b33fd9c7a338_234.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_236.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_238.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_240.png)

![](img/3826f9cb6b2faf937481b33fd9c7a338_242.png)

**核心要点**：精心设计你的记忆点，并坚持在每一期内容中应用。前置的元素要“快准狠”，全局元素要“始终如一”，共同作用，让用户牢牢记住你。

3.  **形式**关乎内容的呈现方式，选择适合自己并能持续输出的模式。
4.  **记忆点**是塑造辨识度的关键，需要通过设计并在内容中反复强化。
