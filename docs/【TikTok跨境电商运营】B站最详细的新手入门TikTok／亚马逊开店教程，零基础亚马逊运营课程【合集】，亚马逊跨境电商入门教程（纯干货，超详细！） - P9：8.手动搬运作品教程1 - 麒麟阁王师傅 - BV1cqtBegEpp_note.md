## 寻找与选择素材 🎯

![](img/b2a22d629396820298e1e41b4ce7f0d4_8.png)

首先，我们需要找到合适的视频素材。素材可以来源于抖音、快手、火山、微视等国内短视频平台。

![](img/b2a22d629396820298e1e41b4ce7f0d4_10.png)

以下是选择素材时需要注意的几点：
*   **避免复杂对白**：尽量选择没有人物对白或对白简单的视频，因为目标观众是外国人，语言不通可能影响理解。
*   **内容普适性**：选择具有视觉冲击力、情感共鸣或趣味性的内容，例如宠物、美景、创意手工等，这类内容不受语言限制。
*   **画质清晰**：尽量选择高清视频，以保证二次剪辑后的观看体验。

## 去除视频水印 🚫

如果视频无法直接保存到相册，我们需要通过复制链接的方式去除平台水印。

![](img/b2a22d629396820298e1e41b4ce7f0d4_17.png)

操作步骤如下：
1.  在抖音APP中，打开目标视频，点击右下角的“分享”按钮。
2.  在分享菜单中，向右滑动，找到并点击“复制链接”。
3.  打开微信，在顶部搜索框搜索小程序“青抖”。
4.  进入“青抖”小程序后，找到“去水印”功能并点击。
5.  系统通常会自动识别剪贴板中的链接，点击“确定”进行解析。
6.  解析完成后，视频上的水印（如左上角/右下角的Logo）已被去除，点击“保存到相册”即可获得无水印视频文件。

![](img/b2a22d629396820298e1e41b4ce7f0d4_19.png)

核心操作可以用一个流程表示：
**发现视频 -> 分享 -> 复制链接 -> 微信打开“青抖”小程序 -> 解析去水印 -> 保存视频**

![](img/b2a22d629396820298e1e41b4ce7f0d4_21.png)

## 进行二次剪辑 ✂️

![](img/b2a22d629396820298e1e41b4ce7f0d4_23.png)

![](img/b2a22d629396820298e1e41b4ce7f0d4_25.png)

获得无水印素材后，不能直接发布，必须进行二次剪辑以规避平台审核，这被称为“洗稿”或“深度处理”。

以下是二次剪辑的常见手法列表：
*   **掐头去尾**：剪掉原视频开头和结尾的冗余部分。
*   **变速处理**：对视频整体或局部进行加速或减速。
*   **镜像翻转**：将画面水平翻转，改变视频方向。
*   **调整尺寸**：裁剪或缩放画面，改变视频比例。
*   **添加元素**：加入自己的贴纸、滤镜、转场效果或画中画。
*   **替换音频**：去掉原声，添加TikTok热门的背景音乐或音效。
*   **重新排序**：对视频片段顺序进行调换。
