## 拼图制作：使用美图秀秀

上一节我们介绍了课程的整体目标，本节中我们来看看如何使用美图秀秀进行拼图。拼图功能适用于需要展示多张图片的场景，能让封面信息更丰富。

以下是适合使用拼图功能的笔记类型：
*   **影视剧推荐**：拼接多个经典镜头或台词。
*   **旅游民宿分享**：展示不同角度的环境或特色细节。
*   **穿搭好物分享**：呈现同一套穿搭的不同单品或搭配效果。
*   **美食推荐**：组合食物的全景、特写或制作过程。

**操作核心要点**：
打开美图秀秀，选择“拼图”功能并导入多张照片后，**仅使用“模板”选项**，避免使用“海报”、“自由”或“拼接”模式。模板自带布局和特效，能快速生成美观的封面。

以下是四种特别适合用拼图形式展示的内容：
1.  **同类产品不同款式**：例如，展示同一品牌不同功能的帽子（遮阳帽、防水帽等）。
2.  **使用效果对比图**：例如，展示美白产品在不同使用阶段的效果对比。
3.  **产品图与效果图结合**：左侧放产品实物，右侧放使用后的效果对比。
4.  **影视剧情节拼贴**：拼接剧中关键对话或场景画面。

## 图片加文字：强化信息传达

学会了拼图，我们来看看如何通过添加文字来直接传达笔记核心信息。在图片上添加醒目的标题或要点，能让用户一眼了解笔记内容。

**制作要点如下**：
*   **文字要足够大**：确保在缩略图状态下也能清晰辨认。
*   **形成色彩反差**：如果背景是浅色，文字就用深色；如果背景是深色或蓝色，文字就用白色或亮色（如红色），以突出显示。
*   **保持简洁明了**：避免使用过多字体特效或倾斜文字，保证易读性。
*   **善用“产品图+要点”布局**：将产品置于图片中央，周围用简洁的文字列出核心功效或特点。这种布局要求背景干净，产品突出。

**对于“产品图+要点”式封面，请遵循以下公式**：
```
封面 = 干净纯色背景 + 居中产品图 + 不超过3种字号/颜色的要点文字（建议以黑白为主）
```
这种组合能有效吸引注意力，并快速传递信息。

![](img/0718fda0cc72bb06a8d71d94741b8ff6_3.png)

**针对年轻化账号**（如面向大学生、职场新人），可以采用**文字+气泡+色块**的活泼风格，这可以通过美图秀秀的贴纸和文字功能轻松实现。

## 抠图应用：使用天天P图

在图片上添加文字后，有时我们需要将产品从原图中“抠”出来，以便自由组合。本节我们将学习如何使用天天P图进行抠图。

抠图功能主要适用于以下三类笔记封面：
1.  **穿搭测评**：将上衣、裤子、包包等单品抠图后并列展示。
2.  **干货科普**：例如，将不同类型的美容仪抠图并排，附上名称。
3.  **横向测评**：将同品类多个产品（如所有热门口红）抠图后集中展示在一张图上。

**抠图前的重要注意事项**：
*   **素材选择**：适合抠图的是细节丰富的小物件（如化妆品、首饰），而非汽车、房屋等大型或背景复杂的物体。
*   **背景要求**：抠图原理是基于物体与背景的差异。**背景最好是单一纯色**（如白、绿、蓝），与主体颜色形成鲜明对比，这样软件才能准确识别。
*   **次数限制**：天天P图的免费抠图功能有每日次数限制。用完后，可通过卸载重装APP来重置（但需注意保存好原图）。

**抠图操作步骤**：
1.  打开天天P图，选择“魔法抠图”。
2.  选择喜欢的背景模板。
3.  导入背景干净的图片，用画笔涂抹需要保留的主体部分。
4.  软件会自动抠出主体，完成后保存即可。
