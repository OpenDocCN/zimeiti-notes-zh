## 收音设备选择

除了画面设备，清晰的音频同样至关重要。以下是常见的收音设备类型及其特点：

*   **无线麦克风**：这是推荐的优选方案。它便于主播移动，且收音效果通常较好。市面上如“小蜜蜂”、罗技、索尼等品牌的产品都是不错的选择。
*   **有线麦克风**：适合超长时间直播，无需担心电量问题。但其缺点是线缆可能限制主播活动，并且在多人直播场景下，固定位置的麦克风可能无法均衡地收取所有人的声音。

选择设备需根据自身条件决定。但无论如何，请务必准备一个收音设备（最不济也应使用耳机麦克风），以避免环境杂音对直播效果造成严重影响。

## 直播软件：OBS与直播伴侣

当使用电脑进行直播时，我们需要专业的推流软件。

![](img/de79a3d30c4aded7b4d483e86076c608_4.png)

*   **OBS**：一款功能强大的开源软件，但设置相对复杂。
*   **直播伴侣**：这是平台官方开发的直播软件，基于OBS优化，更易上手，推荐大多数用户使用。

接下来，我们重点讲解直播伴侣的使用方法。

![](img/de79a3d30c4aded7b4d483e86076c608_6.png)

## 直播伴侣核心设置教程

![](img/de79a3d30c4aded7b4d483e86076c608_8.png)

![](img/de79a3d30c4aded7b4d483e86076c608_10.png)

打开直播伴侣后，核心操作是添加直播画面并调整各项参数。

![](img/de79a3d30c4aded7b4d483e86076c608_12.png)

### 添加直播画面

您可以通过以下两种主要方式添加画面源：

1.  **添加摄像头**：如果您连接了专业相机或摄像头，点击“添加素材” -> “添加摄像头”，然后在列表中选择您的设备。
2.  **添加手机画面（投屏）**：
    *   **方式一（有线）**：用USB数据线连接手机与电脑，在“添加素材”中选择“投屏(iOS/Android)”，按提示操作即可。
    *   **方式二（软件辅助）**：使用如 **iVCam** 这类专用投屏软件，可以获得更稳定的效果。在电脑和手机端分别安装该软件后，在直播伴侣的“摄像头”列表中选择“iVCam”即可捕获手机屏幕。

### 调整视频参数

添加画面后，需在右侧调整视频参数以优化直播画质与流畅度。

![](img/de79a3d30c4aded7b4d483e86076c608_14.png)

```plaintext
分辨率：1080P (默认推荐)
帧率(FPS)：30 (默认推荐)
视频格式：RGB
色彩空间：709
色彩范围：局部
```

![](img/de79a3d30c4aded7b4d483e86076c608_16.png)

**参数选择建议**：
*   对于大多数直播品类，**1080P分辨率**和**30帧率**已完全够用。过高的参数会被平台压缩，且会增加观众设备的负载。
*   如果您追求极致的画面质感（如高客单价商品展示），可以考虑将分辨率提升至4K，帧率提升至60。
*   “色彩范围”选择“局部”，通常在展示产品时色彩表现更好。

![](img/de79a3d30c4aded7b4d483e86076c608_18.png)

![](img/de79a3d30c4aded7b4d483e86076c608_20.png)

### 配置直播设置

![](img/de79a3d30c4aded7b4d483e86076c608_22.png)

点击“直播设置”，进入核心推流配置。

![](img/de79a3d30c4aded7b4d483e86076c608_24.png)

![](img/de79a3d30c4aded7b4d483e86076c608_26.png)

```plaintext
清晰度：蓝光 或 自定义
码率：根据电脑性能，可在6000-10000之间调整
帧率：30 (或根据前述建议调整)
视频编码：硬编
高级设置：开启GPU编码
```

![](img/de79a3d30c4aded7b4d483e86076c608_28.png)

![](img/de79a3d30c4aded7b4d483e86076c608_30.png)

**其他功能设置**：
*   **自动录制**：建议每次开播前开启“本地录制”，便于后期复盘。
*   **福袋**：优先选择“粉丝团福袋”，可以促进观众加入粉丝团，为直播间增加互动数据。
*   **互动管理**：可在“在线观众”列表中对发表不良言论的用户进行拉黑或禁言。
*   **直播间设置**：开播前务必填写“直播标题”、“封面图”和“直播介绍”，这有助于吸引精准观众进入直播间。
