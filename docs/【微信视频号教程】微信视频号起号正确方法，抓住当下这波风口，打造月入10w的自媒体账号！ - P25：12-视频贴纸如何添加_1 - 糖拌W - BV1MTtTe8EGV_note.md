# 微信视频号运营教程：P25：12-视频贴纸如何添加

## 概述
在本节课中，我们将学习如何在微信视频号的视频中添加贴纸。贴纸是丰富视频内容、增加趣味性和引导观众注意力的有效工具。我们将详细介绍添加、调整、管理和删除贴纸的具体步骤。

---

上一节我们介绍了如何在视频中添加文字，本节中我们来看看如何为视频添加贴纸。

在视频编辑界面，文字的后面就是“贴纸”功能按钮。点击“贴纸”按钮，会弹出贴纸选择面板。

以下是添加贴纸的基本步骤：

1.  **选择贴纸**：在贴纸库中浏览并选择你喜欢的贴纸。例如，你可以选择一个“猪头”贴纸。
2.  **确认添加**：点击选中的贴纸，然后点击“对勾”按钮，即可将贴纸添加到视频画面上。
3.  **调整贴纸**：贴纸添加后，可能大小或位置不合适。你可以通过双指捏合手势来**缩放**贴纸，也可以通过拖拽来**移动**贴纸的位置。
4.  **调整时长**：贴纸的显示时长默认与添加时的时间点对齐。你需要手动拖拽贴图轨道两端的边缘，使其与视频的尾部对齐，确保贴纸在整个需要的时段内都显示。

有时，添加的贴纸可能因为颜色或位置问题不够明显。例如，一个浅色的“蝴蝶”贴纸放在亮色背景上就可能“看不太清”。这时，你需要调整其位置或更换更醒目的贴纸。

贴纸库提供了丰富的分类，方便你根据视频主题进行选择。

以下是主要的贴纸分类：

*   **表情/种草**：包含各种表情包和好物推荐类贴纸。
*   **季节主题**：如“秋日”、“冬天”等应景的贴纸。
*   **主题分类**：如“旅行”、“天气”等特定场景的贴纸。

你可以根据视频内容，从这些分类中挑选合适的贴纸。例如，讲解星空内容时，可以添加一个“跳动的小星星”贴纸。

为了让操作更清晰，我们以添加“跳动的小星星”贴纸为例，再演示一遍完整流程。

1.  点击“贴纸”按钮。
2.  找到并点击“跳动的小星星”贴纸，然后点击“对勾”确认添加。
3.  将贴纸**缩放**到合适大小，并**移动**到一个不遮挡主体内容但又可见的位置，例如视频角落。
4.  在时间轴上，将该贴纸的轨道长度拖拽至与视频结尾对齐。

添加后，如果觉得贴纸位置不合适，可以随时**拖拽移动**。如果觉得不需要了，可以点击选中贴纸，然后点击“删除”按钮将其移除。

关于贴纸的使用，有几个关键建议需要注意。

*   **数量建议**：一个视频建议添加 **3~5个** 贴纸为宜，过多会显得杂乱。
*   **内容选择**：优先选择免费贴纸，注意避开标注有“VIP”的付费贴纸。
*   **功能运用**：贴纸不仅可以用于装饰，也可以选择带有文字提示的贴纸来辅助说明。

## 总结
本节课中我们一起学习了微信视频号视频贴纸的添加与管理。核心操作包括：从分类库中选择贴纸、添加到画面、进行缩放与移动调整、以及控制贴纸的显示时长。记住，合理使用3-5个贴纸可以增强视频效果，但需确保其不影响主要内容观看，并优先选用免费资源。
