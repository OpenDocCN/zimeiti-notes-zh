# 亚马逊电商运营教程：P25：创建新商品：描述

在本节课中，我们将学习如何为亚马逊商品创建有效的“描述”和“商品特性”（五点描述）。这是优化商品详情页、提升转化率的关键步骤。

上一节我们介绍了商品标题的撰写，本节中我们来看看如何填充描述部分。

## 商品描述与五点描述概述

商品描述是对商品标题的补充和详细说明。而“商品特性”（Bullet Points，常称五点描述）则是以列表形式突出商品的核心卖点。两者都至关重要，能有效传达商品价值并影响消费者的购买决策。

## 如何撰写商品描述

商品描述应详细、全面地介绍商品。我们可以从供应商提供的产品详情中提取信息，将那些未在标题和特性中充分展示的细节补充进来。这样做不仅能完善商品信息，还能增加潜在的搜索关键词。

以下是撰写商品描述的核心步骤：
1.  参考供应商（如1688）的商品详情页。
2.  提取商品的核心功能、材质、工艺等详细信息。
3.  将这些信息组织成通顺、有吸引力的段落。

## 如何撰写五点描述（商品特性）

五点描述是消费者快速了解商品优势的窗口。亚马逊前台在手机端通常只显示前三条，因此必须将最重要的卖点放在最前面。

以下是撰写五点描述的核心步骤：
1.  **确定核心卖点**：分析商品，找出最吸引人的特性（如材质、独特设计、核心功能）。
2.  **排序**：将最重要的三个卖点放在前三条。
3.  **撰写**：每条描述应简洁有力，首字母大写，避免重复标题已包含的信息。
4.  **优化格式**：使用正确的英文大小写规范，确保可读性。

例如，对于一款男士连帽羽绒服，其五点描述可能包括：
*   **大毛领设计**，时尚保暖。
*   **含46%-50%白鸭绒**，轻盈保暖。
*   **防风袖口与口袋**，锁温防寒。
*   **高密度面料**，防钻绒，不褪色。
*   **多色可选**，满足不同搭配需求。

## 操作注意事项

![](img/69ba88fbbae259fdd9dbc86e5c99da84_3.png)

在后台填写时，需注意格式规范。从其他页面复制文本后，应手动调整大小写，确保每条描述的首字母大写，而非全部大写或全部小写。直接粘贴使用可能影响页面的专业度。

![](img/69ba88fbbae259fdd9dbc86e5c99da84_5.png)

本节课中我们一起学习了如何撰写商品描述和五点描述。关键点在于：描述要详细补充，五点要突出核心卖点且重要项前置，同时注意文本格式的规范性。做好这些，能显著提升商品详情页的质量和吸引力。
