# 自媒体运营教程：P8：7实操教学—剪辑合成 🎬

在本节课中，我们将学习如何将翻译好的字幕与视频素材进行剪辑合成，并添加动态水印以防止他人搬运。整个过程将使用剪映专业版软件完成。

## 视频与字幕的合成

上一节我们介绍了如何获取并翻译视频字幕。现在，我们的视频和字幕文件都已准备就绪。本节中，我们来看看如何将它们合成为一个完整的视频。

首先，播放视频以检查翻译效果。可以看到，翻译基本准确。配合画面，仅看中文也能理解视频内容。

**核心原则**：字幕无需完全精确对应英文，只需确保中文表达在画面辅助下能让观众理解即可。可以删除部分冗余或难以理解的英文直译内容。

以下是导出和导入字幕的步骤：
1.  导出翻译好的字幕文件时，**只选择中文字幕**。我们不需要保留英文部分。
2.  在电脑或手机上保存好导出的字幕文件（通常是 `.srt` 或 `.ass` 格式）。

## 使用剪映进行合成

![](img/9c563d79a60fbcc96ad9026c0cf44b8f_1.png)

![](img/9c563d79a60fbcc96ad9026c0cf44b8f_3.png)

接下来，我们需要使用视频剪辑软件将字幕与视频结合。这里推荐使用剪映专业版。

![](img/9c563d79a60fbcc96ad9026c0cf44b8f_5.png)

**重要提示**：请确保你安装的是 **剪映3.0或更高版本**。旧版本可能不支持字幕文件的导入功能。

以下是具体的操作流程：
1.  打开剪映专业版，将之前录屏或下载好的视频素材导入并添加到时间轨道上。
2.  在软件界面找到“文本”选项卡，在最下方选择“本地字幕导入”。
3.  选择你刚刚导出的中文字幕文件，将其导入。字幕会自动添加到视频轨道上方的文本轨道。
4.  点击播放，预览合成效果。

## 字幕的调整与优化

导入字幕后，可能需要调整其样式和位置，以获得更好的观看体验。

以下是可调整的选项：
*   **大小**：拖动控制点调整字幕大小至合适程度。
*   **字体与颜色**：在右侧面板中选择喜欢的字体和颜色。软件也提供了多种预设的花字样式。
*   **断句与位置**：如果发现字幕断句不合理，可以点击字幕片段，在右侧的文本框中根据语意手动调整换行，使语句更通顺。

调整完毕后，再次播放预览，确保字幕显示清晰、美观。

## 添加动态水印

![](img/9c563d79a60fbcc96ad9026c0cf44b8f_7.png)

由于视频素材是搬运的，为防止他人二次搬运我们的成品，需要为视频添加水印。静态水印容易被去除，因此推荐制作**动态水印**。

动态水印是指能在屏幕上移动的水印，增加他人去除的难度。

以下是创建动态水印的步骤：
1.  在“文本”选项卡中，添加一个“默认文本”或选择一个“花字”。
2.  在文本框中输入你的专属标识，例如账号名“安雅看综艺”。
3.  将文字调整到合适大小，并拖动到视频画面的角落（如左上角）。
4.  将文本轨道的长度拉至与视频轨道等长。
5.  适当降低文字的**不透明度**，使其不影响主画面观看。
6.  制作动态效果：
    *   在视频起始处，选中文字，在右侧“位置”参数旁点击**添加关键帧**图标。
    *   将播放指针移动到视频中间位置，然后将文字拖动到画面另一角（如右下角），软件会自动生成新的关键帧。
    *   再将播放指针移动到视频结尾，将文字拖回起始位置（如左上角）。
    *   这样便创建了一个文字从左上角移动到右下角，再移回左上角的动画路径。

预览视频，可以看到水印在屏幕上循环移动。这种动态水印能有效增加他人搬运和处理的成本。

## 总结与建议

本节课中，我们一起学习了视频剪辑合成的完整流程：从导入视频与字幕、调整字幕样式，到创建动态水印以保护内容。

对于新手而言，可以从在小平台进行视频搬运和剪辑练习开始，以此熟悉内容领域和锻炼剪辑技能。请严格按照上述步骤操作，逐步完成你的视频作品。

**核心操作回顾**：
1.  **导出纯中文字幕**。
2.  **使用剪映3.0+导入音视频和字幕**。
3.  **调整字幕样式**（大小、字体、颜色、断句）。
4.  **创建动态水印**（添加文本、设置关键帧动画）。
