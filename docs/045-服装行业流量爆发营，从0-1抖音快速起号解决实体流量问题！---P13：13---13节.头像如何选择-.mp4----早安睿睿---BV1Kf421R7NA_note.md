# 服装行业流量爆发营：13：头像选择与拍摄指南

![](img/762c36a030e339045e6cf53137452879_0.png)

在本节课中，我们将学习抖音账号搭建中一个至关重要的环节——头像的选择与拍摄。一个好的头像能让人一眼记住你，是建立个人品牌形象的第一步。

![](img/762c36a030e339045e6cf53137452879_1.png)

## 头像的重要性

短视频的头像是账号搭建中非常重要的部分。这是别人第一眼记住你的样子。当我们浏览账号时，如果想了解一个人，必须点进他的主页才能看到背景图或封面。但即使不点开主页，也能在圆形的头像标识中看到他的头像。因此，头像是账号搭建中非常关键的部分。

## 常见错误头像分析

我看到很多服装行业从业者的头像选择非常潦草。要么是随便拍下自家昏暗的门头照片放上去，要么是选择一些欧美街拍或明星照片。这些照片一看就不是本人。

## 优质头像的四个标准

一个好的、面向公众的账号，若想通过头像让人一眼记住，需遵循以下四个方法和标准。

### 1. 注意头像尺寸

头像的尺寸需要注意，它是圆形的。因此，你的头像尺寸必须匹配官方规定的尺寸。不要使用过大或过小的古怪尺寸。

### 2. 掌握正确取景

取景指的是头像中呈现的内容。头像应该让人看到你的脸。但很多人使用全身照作为头像。头像在手机上只占很小的比例，使用全身照，别人根本无法看清你的长相，自然无法记住你。

在取景上，只推荐两种方式：
*   **大头照**：只呈现面部。
*   **半身照**：最多到半身。

绝对不要使用全身照。全身照非常不适合用作头像，无论是公开平台还是私人社交（如微信）都不建议。对于公开平台，我建议大家选择自拍，因为自拍的尺寸非常符合头像的取景要求。

### 3. 匹配服装风格

头像的风格，例如拍摄场景、白天或晚上、背景颜色等，都需要考虑。如果你经营的是优雅、轻熟风格的女装，那么头像给人的感觉应该是淡雅的，而不是艳丽的。如果你做的是甜美可爱的韩系服装，那么头像应该给人清新、舒服的感觉，而不是非常火辣、张扬的。

在选择头像时，应尽量靠近自己的服装风格，这样才能更快地与我们的变现目标匹配起来。

此外，提醒大家尽量不要使用过于昏暗的头像。例如在夜晚昏暗的街边，仅凭隐约路灯拍摄的头像，看起来质感会非常差。尽量选择在光线充足环境下拍摄的照片，这样给人的整体感觉会更加舒适。

### 4. 确保真实感

头像需要给人真实感，让人觉得这是一个真实的人。我发现很多服装从业者会使用明星、模特、街拍图或秀场图作为头像，自己觉得非常高大上。

但抖音是一个人与人之间产生交集的平台。你会发现抖音上几百万粉丝的大V，没有一个是公司账号，全部是个人账号。因此，平台非常需要这种人与人之间的亲近感。当用户看到一个账号时，关联到的是一个真实的人，而不是一个T台模特或明星。

所以我建议大家最好使用自己的照片，用自己的脸作为账号头像，这样可以快速拉近你与观众之间的距离。

## 如何拍摄有质感的头像

上一节我们介绍了选择好头像的四个标准，本节中我们来看看如何实际操作，拍出一张有质感的头像。

以下是拍摄高质量头像的几个实用技巧：
*   **光线是关键**：选择白天光线柔和的窗口，或使用补光灯，确保面部光线均匀、明亮。
*   **背景要简洁**：选择干净、单一的背景，避免杂物分散注意力。
*   **表情要自然**：根据你的服装风格，展现微笑、自信或专业的表情。
*   **画质要清晰**：使用手机后置摄像头或高清模式拍摄，确保图片清晰不模糊。
*   **适当后期调整**：可以使用简单的修图软件微调亮度、对比度和色调，但切勿过度修饰失真。
