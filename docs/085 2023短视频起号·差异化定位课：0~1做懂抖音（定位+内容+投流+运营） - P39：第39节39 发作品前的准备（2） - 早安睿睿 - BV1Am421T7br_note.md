# 短视频起号·差异化定位课：39：发作品前的准备（2）

![](img/155647a15ed64e116d589ce0436595f9_0.png)

在本节课中，我们将要学习发布作品前的第三个准备工作：账号门面打造。上一节我们讨论了选题规划，本节中我们来看看如何通过设置账号信息，清晰地告诉观众和平台你是谁、你能做什么。

![](img/155647a15ed64e116d589ce0436595f9_2.png)

账号门面打造，如同入职前的自我介绍，旨在建立清晰的第一印象。它总共包含四个部分：账号名称、头图、简介和头像。

## 账号名称的设定

账号名称是用户对你的第一印象。以下是设定名称的四个核心要点：

1.  **体现赛道与价值**：名称应能让人一眼看出你所属的赛道和能提供的帮助。
2.  **暗示人设形象**：通过名称，用户能大致感知到你的性格或人设特点。
3.  **简洁易记**：名称不宜过长，通常5-6个字为佳。
4.  **考虑简称**：提前思考名称的简称，方便粉丝称呼和传播。

我们来看几个例子：

*   **美食赛道（烘焙）**：
    *   `小溪的烘焙日记`：直接体现赛道和内容（提供烘焙食谱）。
    *   `小西烘焙很用心`：体现了“努力、用心、精致”的人设形象。
*   **儿童玩具种草赛道**：
    *   `飞飞玩益智`：体现内容与益智玩具相关。
    *   `说玩具的暴躁飞`：体现了“脾气直接”的人设形象。
*   **财经赛道（理财）**：
    *   `王大锤学理财`：直接体现内容是讲解理财知识。
    *   `理财大锤很努力`：体现了“勤奋努力”的人设形象。

## 头图的设置方法

![](img/155647a15ed64e116d589ce0436595f9_8.png)

![](img/155647a15ed64e116d589ce0436595f9_10.png)

头图（Banner）是账号主页顶部的图片，作用重大。设置头图主要有三个方向：

![](img/155647a15ed64e116d589ce0436595f9_12.png)

1.  **简洁版**：仅放置个人照片或有意境的图片，突出人设或风格。
2.  **详实版**：在头图上写清楚“你是谁”以及“你能做什么”。
3.  **引导版**：利用头图引导用户进入私域或完成转化。

![](img/155647a15ed64e116d589ce0436595f9_14.png)

以下是具体案例：

![](img/155647a15ed64e116d589ce0436595f9_16.png)

![](img/155647a15ed64e116d589ce0436595f9_18.png)

**简洁版案例**：
*   `小军办公室`：使用团队合影，体现人设。
*   `野生女孩日记本`：使用个人风格化照片，营造氛围。
*   `特特子的家`：使用“求关注”等简单引导语。

![](img/155647a15ed64e116d589ce0436595f9_20.png)

![](img/155647a15ed64e116d589ce0436595f9_22.png)

**详实版案例**：
*   **价值说明**：在头图上明确列出能为用户带来的具体价值，如“提供产品营销定位策略”。
*   **IP背书**：通过展示演讲照片、与名人合影或列出专业履历，强化IP的专业身份和人设可信度。

**引导版案例**：
*   **引导至私域**：直接在头图上放置微信号等联系方式。
*   **展示外站成就**：放置其他平台（如微博）的粉丝数或影响力截图。
*   **引导加入粉丝群**：说明加入粉丝群可获得的福利（如免费资料），目的是快速积累核心用户。

## 简介的撰写逻辑

![](img/155647a15ed64e116d589ce0436595f9_26.png)

简介是详细说明账号价值的核心区域。撰写简介应包含以下六个步骤：

![](img/155647a15ed64e116d589ce0436595f9_28.png)

1.  **说明身份**：告诉用户“你是谁”。
2.  **展示履历**：用经历概述“你为什么是你”（即你的专业性或独特性）。
3.  **阐明内容**：说清楚你在抖音“要做什么”。
4.  **点明价值**：说明你能给用户“带来什么”。
5.  **设置引导**：添加引导用户关注、加群或进入私域的话术。
6.  **备注与提示**：备注要售卖的产品或给出温馨提醒（如“橱窗商品亲测有效”）。

我们通过一个案例来理解。以美食账号`小雨咕咕`为例：
*   **身份**：“我是小雨”。
*   **履历**：“金牛座吃货”（用星座趣味化表达“爱吃、懂吃”的履历）。
*   **内容与价值**：“在抖音分享很多很多好吃的”。
*   （该案例未明显展示引导和备注，但逻辑完整）。

**模拟案例**：假设我们创建一个“减脂餐美食”账号。
1.  **身份**：我是小希（或“正在减肥的小希”）。
2.  **履历**：曾通过健康饮食从150斤减至100斤。
3.  **内容**：每天分享一道创意减脂餐。
4.  **价值**：希望帮助大家健康瘦身。
5.  **引导**：减脂交流欢迎加入粉丝群/添加微信。
6.  **备注**：橱窗内商品均为亲测有效，可放心下单。

## 头像的选择

![](img/155647a15ed64e116d589ce0436595f9_30.png)

头像的设置相对简单，核心原则是**与人设形象保持一致**。
*   如果人设是搞笑、亲切的，可以使用可爱或有趣的图片。
*   如果人设是专业、酷炫的，则应选择稳重或风格化的图片。

![](img/155647a15ed64e116d589ce0436595f9_32.png)

本节课中我们一起学习了账号门面打造的四个部分：名称、头图、简介和头像。通过系统设置这些信息，你的账号就能清晰地传达价值，吸引精准用户。下一节，我们将继续探讨发布作品前的其他准备工作。
