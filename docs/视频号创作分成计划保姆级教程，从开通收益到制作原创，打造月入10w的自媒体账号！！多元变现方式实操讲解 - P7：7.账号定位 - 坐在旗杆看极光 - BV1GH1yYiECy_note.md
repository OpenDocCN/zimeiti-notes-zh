# 视频号创作分成计划保姆级教程：P7：账号定位与素材获取 🎯

在本节课中，我们将学习如何为视频号进行账号定位，并解决新手创作者最关心的原创视频素材问题。我们将从清理旧内容开始，逐步讲解如何获取和利用高质量素材，为后续的内容创作打下坚实基础。

## 账号内容清理与启动准备

上一节我们讨论了账号的基础设置，本节我们来看看启动前的必要准备工作。有同学询问是否需要隐藏账号之前的作品，答案是肯定的。在开始新的运营方向前，清理或隐藏与当前定位不符的历史作品是必要的步骤。

![](img/a36381a59c50da96464c899a14159b25_1.png)

以下是启动前的两个核心步骤：
1.  **清理历史内容**：隐藏或删除与新的账号定位不相关的旧作品，保持内容垂直度。
2.  **达成初始粉丝目标**：首先集中精力获取最初的100个粉丝，这是许多平台功能（如开通收益）的基础门槛。素材准备可以在此之后进行。

![](img/a36381a59c50da96464c899a14159b25_3.png)
