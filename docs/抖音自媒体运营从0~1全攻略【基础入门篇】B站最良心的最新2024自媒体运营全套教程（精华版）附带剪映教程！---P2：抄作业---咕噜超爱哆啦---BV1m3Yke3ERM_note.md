## 教程正文

上一节我们介绍了中视频计划的潜力，本节中我们来看看一种被称为“抄作业”的具体操作方法。该方法的核心是找到已验证成功的爆款内容，并对其进行合规化再创作。

我很清楚，许多人是为了挣钱努力奋斗的年轻人。最近发现一个通过中视频计划搞钱的方法。这个方法就是“抄作业”。只要你能够耐得住寂寞，吃得了做抖音的苦，就能够掌握这个方法。

我在两周前将这个方法告诉了一位圈友。他每天坚持操作，仅用这个简单方法，每天能获得上千元收益。如果你觉得自己能够做这种重复机械的工作，耐得住寂寞，吃得了这份苦，接下来我把这个方法分享给大家。

小白新手完全可以操作，因为步骤非常简单。我在上一期视频推荐大家去做温情故事类的中视频赛道。所以，第一步是在抖音的搜索框里搜索“温情故事”。这类视频非常容易引起情绪共鸣，容易获得高播放量。

**第一步操作**：在抖音搜索框输入“温情故事”，点击进入“视频”板块，然后筛选“最多点赞”。这样你会看到许多百万级别点赞的爆款温情视频。

接下来来到第二步。随便点开一个高赞视频。视频右下角有一个分享按钮，点击后在最下方会出现一排选项，其中有一个是“复制链接”。我们将这个链接复制下来。

然后，我们来到一款文案提取工具。这一步的目的是把视频里的文案提取出来。直接使用工具里的“链接转文案”功能，视频的文案就被提取出来了。

但这里需要注意：文案提取出来后，一定不要直接使用。我们需要把这个文案通篇读一遍，然后根据自己的理解，用自己的语言把这个温情故事再讲述一遍。这样就不会有违规的风险。

**核心处理逻辑**：
```
原爆款文案 -> 提取 -> 理解 -> 用自己的语言复述 = 新合规文案
```

然后来到第三步。我们把处理好的新文案直接导入到剪映软件里，使用剪映的“一键成片”功能。它会自动为文案匹配画面，并根据文案进行配音。完全不需要我们自己剪辑，一个视频就完成了。

最后是视频发布。直接在抖音顶部搜索“中视频计划”，找到“中视频伙伴计划”入口。通过这个入口发布视频，播放量就会计入中视频计划。发布时别忘了同步到西瓜视频和今日头条。在这三个平台上，只要视频有播放量，就有收益。

这个方法看似简单，但简单之中透露着不简单。它的核心在于，我们借鉴了已经被市场验证的爆款文案和爆款选题。而文案和选题是视频能否爆播放最重要的两个因素。

本节课我们一起学习了“抄作业”这一中视频创作方法。其流程可总结为：**搜索爆款 -> 提取文案 -> 理解复述 -> 工具成片 -> 计划发布**。简单的方法关键在于执行，不去执行一切等于零。只要你能坚持操作，就能在抖音上获得收益。
