# 剪映手机版教学：P13：色度抠图功能详解 🎬

在本节课中，我们将学习剪映APP的核心剪辑功能之一——色度抠图。这是一个非常强大的功能，尤其适合用于制作特效类视频。通过本教程，你将学会如何利用色度抠图功能，轻松地将绿幕素材中的特效元素抠出，并合成到你的视频中，从而制作出专业水准的短视频内容。

## 色度抠图功能简介

上一节我们介绍了剪映的基础剪辑功能，本节中我们来看看色度抠图。色度抠图是剪映APP近期新增的一项功能，它极大地简化了在手机端进行视频合成的流程。这个功能的核心原理是**通过选取并移除画面中的特定颜色（如绿色），将该颜色区域变为透明，从而分离出前景物体**。

![](img/b1974cd83ed37bc6291622338c8b3c0d_1.png)

在传统的专业软件（如PR或AE）中，绿幕抠图是合成类视频最常用的技术。现在，剪映将这一功能带到了移动端，让用户能够直接在手机上制作大量特效视频。

## 功能实操演示

了解了色度抠图的基本概念后，接下来我们通过一个具体案例来学习如何操作。我们将制作一个“飞机飞过天空”的特效视频。

![](img/b1974cd83ed37bc6291622338c8b3c0d_3.png)

![](img/b1974cd83ed37bc6291622338c8b3c0d_5.png)

![](img/b1974cd83ed37bc6291622338c8b3c0d_7.png)

以下是操作步骤：

1.  **导入主视频素材**：首先，在剪映中导入一段作为背景的天空视频。
2.  **添加绿幕素材**：点击“画中画”功能，导入第二段带有绿色背景的飞机特效视频素材。此时，绿幕素材会叠加在天空视频之上。
3.  **应用色度抠图**：选中画中画轨道上的绿幕素材，在底部工具栏中找到“色度抠图”选项（通常位于“蒙版”旁边）。
4.  **取色与抠除**：
    *   使用取色器（一个圆形图标）点击画面中的绿色背景。
    *   调整“强度”滑块，逐步将绿色背景抠除干净。如果边缘有残留的绿色杂边，可以适当调整“阴影”滑块，使合成效果更自然。
5.  **调整与剪辑**：
    *   使用“编辑”->“裁剪”功能，裁掉素材中不需要的水印或多余部分。
    *   调整飞机素材的位置、大小，并为其添加“入场动画”（如渐显），使其融入更流畅。
    *   修剪主视频与画中画视频的长度，确保时间线对齐，然后删除多余片段。

![](img/b1974cd83ed37bc6291622338c8b3c0d_9.png)

![](img/b1974cd83ed37bc6291622338c8b3c0d_10.png)

通过以上步骤，我们就能将绿幕中的飞机完美地合成到天空背景中，实现逼真的特效效果。
