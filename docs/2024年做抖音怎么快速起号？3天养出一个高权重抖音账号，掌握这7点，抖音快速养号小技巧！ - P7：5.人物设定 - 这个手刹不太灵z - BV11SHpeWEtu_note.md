# 抖音运营指南：P7：人物设定 🎭

![](img/65a6344714e23183efcd6473128706fb_0.png)

在本节课中，我们将学习如何在抖音上进行人设策划。一个清晰、真实且有吸引力的人物设定，是打造成功账号、获取粉丝信任并实现变现的核心基础。

![](img/65a6344714e23183efcd6473128706fb_2.png)

上一节我们介绍了账号定位的重要性，本节中我们来看看如何具体构建一个账号的“灵魂”——人物设定。

![](img/65a6344714e23183efcd6473128706fb_4.png)

## 人设策划的核心概念

![](img/65a6344714e23183efcd6473128706fb_6.png)

在抖音中做人设，就像在游戏中选角色一样，需要提前对人物角色进行设定。例如称呼、职业、拥有的技能点等。但与游戏角色不同的是，我们在抖音中设定的人物角色是**人生角色**，所以需要更具有真实性。

![](img/65a6344714e23183efcd6473128706fb_8.png)

不管是短视频还是直播，一个成功的IP能够解决粉丝信任的问题。它能够起到裂变传播的作用，当然也具备很强的变现能力。因此，我们做账号，不管是创作内容还是策划选题，目的就是在获取流量的同时打造IP。打造IP的核心，就是在一定的场景或者情境中，进行**账号人格化**的过程。

![](img/65a6344714e23183efcd6473128706fb_10.png)

## 如何策划账号人设

![](img/65a6344714e23183efcd6473128706fb_12.png)

账号人格化，即我们通常所说的账号人设。对于账号人设的策划，我们可以从以下几个方面着手。

![](img/65a6344714e23183efcd6473128706fb_14.png)

以下是构建人设的六个关键维度：

![](img/65a6344714e23183efcd6473128706fb_16.png)

1.  **名字/姓氏/昵称**
    *   一个IP首先得有一个叫法，这是IP传播的“名片”。要明确这个IP叫什么，就像一个品牌的品牌名或商标名一样重要。

![](img/65a6344714e23183efcd6473128706fb_18.png)

2.  **职业/职务/职位/职称**
    *   人格化必然要具备身份。这是用户认识这个人或者这个IP的联想入口。例如：老师、教授、博士、律师等。

3.  **称呼/称谓/社会关系**
    *   一个IP有了特定的称呼，能快速拉近与用户的距离，起到“移情”的作用。例如：XX哥、XX大爷、XX姥姥。

4.  **形象印象/第一印象**
    *   以用户的视角来看待IP，使IP形象化，更容易占据用户心智。例如：大叔形象、萌萌哒、铁憨憨、甜美可爱、光头等。

5.  **性格/爱好/特点**
    *   一个具有鲜明特点的IP，除了具有很强的记忆点之外，还具有很强的真实性。太完美的形象容易脱离大众。例如：暴躁、娘炮、爱吃、爱喝酒等。

6.  **穿着打扮/财富地位**
    *   这属于社会的刻板印象和大众的普遍认知。借助用户的固有思维，可以为IP快速在用户心智中定位。例如：有钱代表会赚钱，开豪车代表成功人士，打扮土气代表老师等。

以上内容策划好之后，就可以结合你自身的产品优势、服务优势等，将其落实到**账号搭建**中，即抖音账号的“五件套”：头像、昵称、背景图、简介、视频封面。

## 提炼人设的“超级符号”

除了对账号IP进行人格化的基础策划之外，还要通过选题和内容策划，为您的人设提炼出**记忆点**，即通常所说的“超级符号”。

超级符号的提炼可以从以下几个方面着手：

以下是提炼超级符号的几个方向：

*   **画风（视觉）**：如特定的场景、道具、服饰等。
*   **听觉**：如标志性的BGM、特效音、配音声线特点等。
*   **文案**：能够反映优势且容易记忆的提炼词。例如：“XX简单粗暴”。
*   **表情/情绪**：如标志性的笑、惊恐、无奈等表情或情绪传递。
*   **原特色**：如方言、口音、错发音、网络语、慢条斯理、抑扬顿挫的说话方式等。
*   **动作**：习惯性的手部或肢体动作。例如：撩头发、扶眼镜等。

**超级符号**提炼好之后，在你进行短视频创作时，就可以不断地在视频中呈现这些元素，从而强化人设，让用户牢牢记住你。

本节课中，我们一起学习了抖音账号人设策划的全过程。从理解人设的真实性与重要性，到具体构建人设的六个维度，再到提炼强化记忆的“超级符号”，每一步都是将账号人格化、赋予其生命力的关键。一个成功的人设，是连接你与粉丝、实现内容价值最大化的桥梁。
