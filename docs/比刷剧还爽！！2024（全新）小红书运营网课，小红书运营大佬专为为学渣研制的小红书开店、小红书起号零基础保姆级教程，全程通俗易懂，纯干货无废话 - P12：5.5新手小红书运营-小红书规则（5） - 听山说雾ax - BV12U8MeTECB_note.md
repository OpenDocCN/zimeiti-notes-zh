## 被推荐的核心逻辑

笔记发布后，平台会根据一系列数据指标决定是否将其推荐给更多用户。这个过程可以理解为“流量升级”。例如，一个新号发布的笔记，初始可能只有约1000次展示（曝光）和50-200次点击（小眼睛）。平台通过何种机制，能让它升级到2000-5000次展示，甚至更高层级呢？了解这个升级过程，有助于我们诊断笔记流量停滞的原因。

以下是决定笔记能否进入下一流量层级的两个核心数据指标：

**1. 打开率 (Click-Through Rate, CTR)**
当用户打开小红书App，首页推荐流会展示4-6篇笔记。平台会一次性测试约20篇同类型笔记。你的笔记若在其中，平台会给予一个初始展示量（例如1000次）。**打开率** 指看到笔记的用户中，有多少人点击进入阅读。

*   **核心公式**：`打开率 = (笔记点击次数 / 笔记展示次数) * 100%`
*   **升级门槛**：你的笔记打开率必须超过同批测试笔记的**平均打开率（约11%）**，才能进入下一流量池。
    *   例如：获得1000次展示，至少需要110次点击（1000 * 11%）。
    *   若达标，展示量可能跃升至2000-5000次；若不达标，流量将停留在初始水平。

**2. 互动率 (Engagement Rate)**
通过第一层打开率考核后，笔记进入更大流量池。此时，**互动率** 成为能否继续升级的关键。

![](img/992e7af8640afbe319d342aefce58eef_9.png)

*   **核心公式**：`互动率 = (点赞、收藏、评论、分享等互动总数 / 笔记点击次数) * 100%`
*   **升级门槛**：互动率需超过**3%**。
    *   例如：笔记有110次点击，互动行为（评论等）至少需达到4次（110 * 3% ≈ 3.3，向上取整）。
    *   互动率达标，笔记有望从几千展示升级至数万甚至十万级别。

**总结升级路径**：**先点击（打开率），后互动（互动率）**。通常，新号通过这两层考核，可升至第三级流量池。账号基础权重越高，起始流量池越大，有机会冲击更高层级。而顶级流量（第六级及以上）通常需要结合内容质量与商业投放。

![](img/992e7af8640afbe319d342aefce58eef_17.png)

除了打开率和互动率，平台还有一个综合评分体系，基于用户的**点赞、关注、收藏、分享**等行为对笔记进行加权。但这一切的前提是，笔记的**基础数据（打开率>11%，互动率>3%）** 必须达标。

![](img/992e7af8640afbe319d342aefce58eef_19.png)

## 必须避免的运营“坑”🚫

了解规则后，更需规避违规操作，以免账号被降权、限流。以下是新手运营中常见的误区：

以下是运营小红书时需要特别注意的禁忌列表：

![](img/992e7af8640afbe319d342aefce58eef_21.png)

*   **新号切勿立即发笔记**：新注册账号需“养号”，模拟正常用户浏览、互动行为数日至一周，建立初始标签。立即发笔记会消耗平台给予新号的初始权重，导致后续流量低迷。
*   **违规必导致降权**：任何违规（如导流、抄袭、违禁内容）都会导致笔记推荐减少、搜索排名下降，甚至账号被限流。一次严重违规可能导致流量下降90%并持续一段时间，恢复难度极大。
*   **标签切忌反复修改**：账号、店铺或笔记的人群标签一旦确定，应保持稳定。在已有一定流量后频繁修改核心标签（如领域、定位），极易引发系统识别混乱，导致限流。
*   **避免一机多号**：原则上，一部手机最好只登录一个小红书账号，并使用独立手机卡。同一设备频繁切换多个账号，易被判定为营销号或工作室操作。
*   **谨慎处理推广与引流**：避免在笔记或评论区直接留下联系方式、二维码或明显导流词汇。私信引流也需谨慎，过度营销易被举报和处罚。
*   **内容原创为王**：搬运、抄袭他人内容风险极高，平台推崇原创优质内容。
