## 工具核心功能概述

今天推荐的工具主要有两大核心功能：
1.  **AI一键成片**：根据文案或热点话题，自动匹配素材和配音，生成完整视频。
2.  **AI文本生成与润色**：辅助进行文案创作、改写与优化。

## AI一键成片功能详解

首先，我们进入工具的首页。界面中可以看到“AI成片”和“AI笔记”两个主要入口。

### 1. 输入文案生成视频

在“AI成片”页面，您可以将预先准备好的视频脚本文案粘贴到输入框中。工具会自动根据文案内容，在网络上搜索匹配的素材，并配上合适的背景音乐和AI配音，从而生成一个完整的视频。

**操作公式**：
```
输入文案 -> AI分析 -> 自动匹配素材与配音 -> 生成视频
```

### 2. 选择文章生成视频

此功能需要您拥有一个已发布原创文章的百家号。登录后，您可以直接选择自己发布过的文章，工具会提取文章内容并自动生成视频。

![](img/ff0acfc7f104039be10d227c9cf29479_3.png)

### 3. 蹭热点自动生成视频 🔥

![](img/ff0acfc7f104039be10d227c9cf29479_5.png)

这是该工具最强大的功能之一。在“AI成片”页面右侧，会实时展示当前的热点话题。

以下是使用此功能的步骤：
1.  **选择热点**：从热点列表中选择一个您想创作的话题。
2.  **AI生成文案**：点击后，AI会在几十秒内自动生成一份关于该热点的视频文案。
3.  **优化文案**：AI生成的文案通常会将吸引人的点放在开头以提升点击率。如果开头不够吸引人，建议手动调整，将核心爆点前置。
4.  **审核文案**：在生成视频前，**必须**仔细检查文案内容，确保其符合法律法规和平台规则，避免传播不实信息。

![](img/ff0acfc7f104039be10d227c9cf29479_7.png)

### 4. 素材匹配与替换

![](img/ff0acfc7f104039be10d227c9cf29479_9.png)

AI会自动为文案匹配素材，但有时匹配可能不精准或素材缺失。

以下是处理素材问题的两种方法：
*   **使用推荐素材**：AI会提供“关键素材推荐”，这些是与热点相关的素材，选用它们能使视频更真实。
*   **手动搜索替换**：如果对自动匹配的素材不满意，可以在素材库中手动搜索关键词，替换成更合适的素材。您也可以上传本地素材。

**重要声明**：对于新闻类视频，如果使用的不是原始视频素材，建议在视频中添加声明，例如：“素材来源于网络，非事件原视频”。

### 5. 调整视频元素

生成视频后，您可以进行多项调整：
*   **模板**：可选择横版或竖版视频模板。
*   **朗读音**：提供多种AI配音音色，并可调整语速。根据视频内容（如紧张的社会新闻）选择合适的语速和音色。
*   **背景音乐**：工具内置了分类音乐库，但选择有限。目前不支持上传自定义音乐。

**技巧**：您可以将生成的视频下载到本地，使用专业剪辑软件（如剪映）更换背景音乐、去除水印或进行精细调整。

### 6. 视频下载与发布

![](img/ff0acfc7f104039be10d227c9cf29479_11.png)

视频制作完成后，按以下步骤操作：
1.  点击右上角的 **“发布视频”** 按钮。
2.  在跳转的页面中，选择 **“生成视频”**（而非直接发布到平台）。
3.  等待几分钟，视频渲染完成后，会出现 **“下载”** 按钮。
4.  将视频下载到本地，经过必要的检查与优化后，即可发布到任何您想要的平台。

![](img/ff0acfc7f104039be10d227c9cf29479_13.png)

**核心提醒**：使用工具时，务必遵守国家法律法规和平台规则，不造谣，不传播非法内容。

## AI笔记（文本生成与润色）功能详解

![](img/ff0acfc7f104039be10d227c9cf29479_20.png)

AI笔记功能可以理解为一个内置的文案创作助手，类似于ChatGPT，能帮助运营者进行多种文本工作。

对于新媒体运营者而言，掌握AI文本工具至关重要。它能帮助我们：
*   **撰写脚本**和**文章文案**。
*   **设计直播话术**。
*   **进行品牌宣传推广**的文案创作。
*   **润色优化**现有文本，使其更流畅、更有吸引力。

![](img/ff0acfc7f104039be10d227c9cf29479_22.png)

以下是该功能的具体应用：

### 1. 文本生成

您可以直接向AI提问或给出主题，让它生成相关文案。例如，输入“地球还有多少年的寿命，怎么保护地球”，AI会生成一篇相关的观点文章。

### 2. 文笔润色

![](img/ff0acfc7f104039be10d227c9cf29479_24.png)

将您已写好的文案粘贴到输入框，AI可以帮您优化语句、调整结构，使文案更具可读性和专业性。

**代码示例（概念性）**：
```python
# 伪代码：展示AI文本处理流程
original_text = “输入您的原始文案”
polished_text = ai_assistant.polish(original_text) # AI润色
new_script = ai_assistant.generate(topic=“给定主题”) # AI生成新文案
```
