## 个人账号规划与定位

做个人账号规划时，可以借鉴引流的思路。其中有一个关键点是个人信息的组织与搭配。

以下是构建个人简介的核心要素：
*   **身高、体重、身材爱好、职业方向**：这些信息是给自己看的，用于评估你是否满足成为某类博主的需求。
*   **信息筛选**：不满足博主形象需求的信息可以屏蔽，符合的信息则保留并放入个人背景简介中。

这类似于交朋友，需要先介绍自己的个人情况，以吸引同类型的人群。例如，如果你是美妆、化妆或美食博主，就需要完美地介绍自己。个人情况介绍得越真实，获得的认可度通常也越高。

同时，这也是给自己做定位。个人账号的内容规划涉及以下几个层面：

![](img/a063407de4bb55794d8eea172600a0a0_7.png)

![](img/a063407de4bb55794d8eea172600a0a0_9.png)

以下是个人账号内容定位的步骤：
1.  **IP定位**：为自己设定一个“人设”。例如，“一线城市90后独居女性，分享每日生活或职场日常”。
2.  **粉丝定位**：确定你的内容要吸引哪类人群。你需要将自己打造成目标人群的“领头羊”。
3.  **内容关键词**：在粉丝定位的大方向下，选择2-3个具体的小话题进行日常分享，例如“美食、独居、职场独立”。
4.  **选题**：为你将要发布的内容确定主题，例如推广书籍、分享经验等。

![](img/a063407de4bb55794d8eea172600a0a0_11.png)

![](img/a063407de4bb55794d8eea172600a0a0_13.png)

确定好这些后，你作为“领头羊”就能更有效地向粉丝介绍吃喝玩乐、推广产品或接广告。

## 网络博主的类型与运营

![](img/a063407de4bb55794d8eea172600a0a0_21.png)

既然决定成为小红书博主，就需要选择自己的内容呈现方式：是做直播博主还是文案博主。

### 直播博主

选择成为直播博主，需要规划以下四个方面的内容。

以下是直播博主需要准备的四个核心要素：
*   **直播内容**：可以选择教育科普、生活分享、时尚穿搭、访谈、电商、心理咨询等任何你擅长的领域。核心是放大自己的专业或兴趣，吸引对应人群。
*   **直播时间**：平台虽无限制，但建议设定固定的直播时间表，将其视为工作时间，有利于培养观众习惯。
*   **内容准备**：前期建议准备至少能支撑2小时直播的3个主题内容或3套直播话术，以避免直播间冷场。
*   **直播场景**：注意背景、前景和整体环境（三位一体）的布置，根据个人条件进行设置。

![](img/a063407de4bb55794d8eea172600a0a0_23.png)

### 文案博主

文案博主相比直播博主，对个人即时表现力的要求较低，因为不熟悉的内容可以查阅资料后再整理成文案笔记发布。

以下是文案博主运营的两个要点：
*   **笔记输出内容**：采用 **`5分之4的实题输出 + 5分之1的猜测内容`** 的策略。保留部分答案，用小号在评论区引导互动，提升账号活跃度。
*   **文案排版**：
    1.  主笔记图片选择3到5张进行排版。
    2.  首图或首段用“猜测内容”吸引用户点击。
    3.  第二部分对核心内容进行解说。
    4.  第三、四部分进行内容延伸。
    5.  第五部分预告下一篇笔记的主题，形成内容循环，吸引用户关注后续更新。

笔记发布时间可参考之前讲解的“商品笔记”发布策略。
