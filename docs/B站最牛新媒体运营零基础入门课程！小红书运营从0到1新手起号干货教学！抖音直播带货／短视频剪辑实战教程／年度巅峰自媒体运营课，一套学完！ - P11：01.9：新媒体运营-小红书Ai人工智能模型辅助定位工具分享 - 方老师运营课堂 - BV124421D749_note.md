# 新媒体运营：P11：01.9：小红书AI人工智能模型辅助定位工具分享 🧠

在本节课中，我们将学习如何利用AI工具辅助进行小红书账号的人设定位。上一节我们介绍了账号定位的四个思考方向，本节我们将看看当自己难以确定方向时，如何借助智能工具来获得灵感和参考。

## 课程概述

定位是账号运营的基础。我们已从“我是谁”、“提供价值”、“差异化”和“持续输出”四个方向探讨了如何让用户产生关注冲动。同时，领域定位应基于自身擅长、用户需求和对标账号分析。然而，部分同学可能仍感困惑。本节将介绍一个基于ChatGPT模型开发的AI工具，它能帮助我们分析和生成定位建议。

## 工具核心功能简介

这个AI工具本质上是一个智能聊天模型，它集成了多项对新媒体运营者实用的功能。

以下是该工具的几个核心功能：

*   **爆款笔记分析**：工具能分析已有爆款笔记的文案，识别其中的“爆款因子”。
*   **爆款复刻（伪原创）**：基于分析出的爆款因子，工具能生成具备类似潜力的新文案。
*   **敏感词检测**：发布前检测文案，避免因敏感词导致限流。
*   **小红书风格文案生成**：根据你提供的选题方向，自动生成符合小红书平台风格的文案。

**注意**：AI生成的内容可作为重要参考，但不能完全依赖。最终决策需结合个人判断。

我们将重点演示如何使用该工具的“人设定位”功能。当你对自身定位感到迷茫时，它可以提供多个方向建议。

![](img/f02f39e3e471f1ef3b07401865807fad_7.png)

你需要向工具提供关于你自己的几个关键信息点。以下是需要准备的信息列表：

1.  **兴趣爱好**：例如，打游戏、看直播、唱歌。
2.  **工作与生活经历**：例如，做过运营、参加过游戏比赛、带领过团队。
3.  **专业知识与技能**：例如，新媒体运营、小红书起号、数学能力。
4.  **可投入时间**：例如，每天3小时。
5.  **目标与期望**：例如，打造个人IP。

我们根据以上信息，向工具提交了一个混合的示例（“打游戏、看直播、唱歌、做过运营、参加过比赛、新媒体运营技能、数学好、每天3小时、目标打造IP”）。工具迅速生成了分析报告。

## 如何有效使用工具

![](img/f02f39e3e471f1ef3b07401865807fad_19.png)

![](img/f02f39e3e471f1ef3b07401865807fad_21.png)

分享这个工具的目的是帮助那些在定位阶段遇到巨大困难的同学。但使用时需注意方法。

![](img/f02f39e3e471f1ef3b07401865807fad_23.png)

![](img/f02f39e3e471f1ef3b07401865807fad_25.png)

以下是有效使用该工具的步骤建议：

![](img/f02f39e3e471f1ef3b07401865807fad_27.png)

1.  **明确输入**：不要像演示中那样输入杂乱信息。应围绕一个你潜在有优势的垂直领域（如“包包收藏与鉴定”）进行详细、聚焦的描述。
2.  **多次尝试**：针对你不同的擅长领域，分别生成多份定位报告。
3.  **对比筛选**：从所有报告中挑选出几个看起来最可行、最匹配的方向。
4.  **深度评估**：结合竞争难度、自身资源、兴趣持久度等因素，做出最终选择。

本节课我们一起学习了如何利用AI智能工具辅助解决小红书账号的人设定位难题。该工具可通过分析爆款、检测敏感词、生成文案等功能，特别是在你缺乏方向时，通过输入你的兴趣、经历、技能等信息，为你生成多个潜在的定位方向报告。

**关键点总结**：
1.  AI工具是**辅助参考**，而非决策唯一依据。
2.  使用时应提供**垂直、聚焦**的个人信息输入。
3.  通过**生成多份报告、对比筛选**来找到最适合自己的方向。
