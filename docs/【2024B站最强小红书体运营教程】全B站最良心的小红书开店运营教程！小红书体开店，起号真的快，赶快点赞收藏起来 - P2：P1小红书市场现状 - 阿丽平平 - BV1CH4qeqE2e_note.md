**1. 小红书平台合作**
这种方式需要你登录小红书的电脑端。在电脑端，你可以找到“创作者服务”板块。该板块内设有官方的合作渠道。如果你是一个内容创作者，希望通过为他人带货或引流来实现流量变现，就可以通过这个官方渠道进行。小红书官方平台整合了多方资源，包括内容创作者、专业的账号运营机构（如MCN机构）以及大量寻求推广的商家。加入这个平台，意味着你有可能直接与商家合作获得推广佣金，也可能因为账号潜力被MCN机构看中，进入其孵化体系，从而有机会成长为头部账号。

![](img/9afef04001b7ee758ed3dccda1f5aa6b_46.png)

![](img/9afef04001b7ee758ed3dccda1f5aa6b_48.png)

![](img/9afef04001b7ee758ed3dccda1f5aa6b_50.png)

操作路径可以总结为：`小红书电脑端 -> 创作者服务 -> 官方合作渠道`。

![](img/9afef04001b7ee758ed3dccda1f5aa6b_54.png)

**2. 自开小店**
无论你是个人、商家还是企业，无论是否拥有营业执照，都可以在小红书上开设自己的小店。你可以在手机小红书App中，通过点击个人页面的“三道杠”菜单，进入“创作者服务中心”，在其中找到“开通店铺”或“成为专业号”的入口并完成开通。开店后，你可以上架自己的商品（个人可销售合规商品，企业可销售自家产品），并通过内容创作吸引用户购买，实现直接的带货变现。

![](img/9afef04001b7ee758ed3dccda1f5aa6b_60.png)

![](img/9afef04001b7ee758ed3dccda1f5aa6b_62.png)

![](img/9afef04001b7ee758ed3dccda1f5aa6b_64.png)

操作路径可以总结为：`手机App -> 个人主页菜单 -> 创作者服务中心 -> 开通店铺`。

![](img/9afef04001b7ee758ed3dccda1f5aa6b_66.png)

![](img/9afef04001b7ee758ed3dccda1f5aa6b_68.png)

![](img/9afef04001b7ee758ed3dccda1f5aa6b_70.png)

**3. 企业号营销**
如果你是企业或品牌方，除了自开小店，还可以选择与小红书商城进行更深度的合作。例如，你可以将商品供应给小红书，成为其自营商品的一部分。你只需负责供货，同时可以利用自己的企业号发布内容，为这些商品引流，促进销售。这种方式将供应链与内容营销相结合，是品牌方高效的营销路径。

本节课中我们一起学习了小红书店铺运营的三种主要方式：通过官方平台合作接单、自行开设小店销售商品、以及作为企业进行深度供应链合作。理解这些路径是进行小红书电商变现的第一步。
