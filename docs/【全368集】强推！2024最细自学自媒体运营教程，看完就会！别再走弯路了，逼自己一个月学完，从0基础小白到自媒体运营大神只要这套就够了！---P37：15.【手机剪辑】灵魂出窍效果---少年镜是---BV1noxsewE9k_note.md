# 手机剪辑教程：P37：15. 灵魂出窍效果制作 🎬

在本节课中，我们将学习如何使用剪映App制作一个流行的“灵魂出窍”视频特效。这个效果常用于音乐卡点视频，能让人物主体产生一种动态分离的视觉冲击。

## 概述
我们将通过调整视频时长、应用组合动画、使用画中画功能以及添加关键帧来控制透明度，最终结合特效来完成整个效果。整个过程步骤清晰，适合初学者跟随操作。

## 详细步骤

上一节我们概述了效果的制作流程，本节中我们来详细分解每一个操作步骤。

以下是核心操作步骤列表：

1.  **调整视频时长**
    *   在剪映中导入视频后，首先根据背景音乐的节拍，裁剪并调整视频的时长，使其与音乐节奏对齐。

2.  **应用组合动画**
    *   选中视频片段，点击下方工具栏中的“动画”。
    *   选择“组合动画”分类，在其中找到并应用“回弹伸缩”效果。
    *   将动画的时长拉满，使其覆盖整个视频片段。

3.  **复制视频并切为画中画**
    *   为当前视频片段制作一个副本。选中视频，点击“复制”。
    *   选中复制出来的视频片段，在工具栏中找到并点击“切画中画”。这样我们就得到了上下重叠的两个相同视频图层。

4.  **设置混合模式与关键帧**
    *   选中下方的画中画图层，点击“混合模式”，将其更改为“绿色”。
    *   在时间轴上找到第二个音乐节拍点（黄点位置），在此为画中画图层添加一个**关键帧**。
    *   将时间线移动到视频结尾处，在此位置将画中画视频**放大**。
    *   接着，将时间线移回第一个关键帧正前方的位置，在此处添加一个新的关键帧，并将画中画图层的**不透明度参数调整为0**。
    *   最后，在视频结尾处（最后一个关键帧），也将**不透明度参数调整为0**。这样，画中画图层就会呈现从透明到显现再回到透明的动画过程。

5.  **添加特效**
    *   点击底部工具栏的“特效”，选择“画面特效”。
    *   在“热门”分类下，找到“荧光线描”效果并添加。
    *   将特效的“强度”或“滤镜”参数**调整至50左右**（根据实际效果微调）。
    *   在特效参数中，将“作用对象”设置为“画中画”图层。
    *   调整特效的时长，使其与视频后两个节拍点的区间对齐。对于更长的音乐，可以在后面的节拍处重复此操作。

## 效果预览与总结
完成以上所有步骤后，“灵魂出窍”效果就制作完成了。你可以播放预览整个视频，检查动画与音乐节拍是否同步。

本节课中我们一起学习了制作“灵魂出窍”视频特效的全过程。关键点在于利用**画中画**复制图层，通过**关键帧**控制其缩放与透明度变化，并最终使用**荧光线描**特效增强视觉表现力。掌握这个流程后，你可以尝试调整关键帧的位置和参数，创造出属于自己的变体效果。
