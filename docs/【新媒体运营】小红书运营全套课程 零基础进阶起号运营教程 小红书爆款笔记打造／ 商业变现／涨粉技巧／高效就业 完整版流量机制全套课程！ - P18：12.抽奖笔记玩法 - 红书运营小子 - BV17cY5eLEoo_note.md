# 新媒体运营：P18：12.抽奖笔记玩法 🎁

在本节课中，我们将要学习如何在小红书平台创建和发布抽奖笔记。这是一种有效的营销工具，可以帮助商家增加店铺曝光、促进成交、增强粉丝粘性并为店铺引流。

## 概述

![](img/205fa5d7527233ad76e7274d85dac2f6_1.png)

抽奖笔记是小红书为专业号用户提供的一种官方营销功能。通过设置奖品和活动规则，可以吸引用户参与互动，从而达到推广目的。接下来，我们将分步讲解如何创建一次合规的抽奖活动。

## 创建抽奖活动

![](img/205fa5d7527233ad76e7274d85dac2f6_3.png)

上一节我们介绍了抽奖笔记的作用，本节中我们来看看如何具体创建一次抽奖活动。整个过程分为在电脑端开通功能、配置活动，以及在手机端发布笔记。

![](img/205fa5d7527233ad76e7274d85dac2f6_5.png)

### 第一步：开通抽奖功能

首先，你需要在电脑端登录小红书专业号平台。开通抽奖功能需要满足以下条件：
*   **账号身份**：必须是**专业号企业身份**或**专业号个人身份**。
*   **粉丝数量**：粉丝量需**大于或等于1000**。

开通后，不同身份的账号拥有不同的创建次数：
*   **专业号企业身份**：每月可创建**3次**抽奖笔记。
*   **专业号个人身份**：每月可创建**1次**抽奖笔记。

### 第二步：配置活动信息

开通功能后，即可创建抽奖活动。以下是具体的配置步骤和注意事项。

![](img/205fa5d7527233ad76e7274d85dac2f6_7.png)

1.  在专业号平台，找到“营销工具”下的“抽奖活动”入口。
2.  点击“创建抽奖活动”。
3.  填写活动的核心信息，包括：
    *   奖品名称
    *   奖品数量
    *   奖品价值
    *   奖品简介
4.  设置活动的参与时间与开奖时间。

**重要限制**：目前平台仅支持可寄送的**实物奖品**，且单次活动的奖品数量**不可超过50份**。

### 第三步：发布抽奖笔记

![](img/205fa5d7527233ad76e7274d85dac2f6_9.png)

电脑后台配置完成后，需要切换到手机App端来发布包含抽奖活动的笔记。

1.  在手机小红书App中，点击底部“+”号，选择笔记封面和编辑内容。
2.  在发布页面，点击“高级选项”。
3.  选择“添加抽奖活动”，并确认关联你刚创建的活动。
4.  点击发布，即完成抽奖笔记的发布。

## 活动中奖与发货管理

![](img/205fa5d7527233ad76e7274d85dac2f6_11.png)

发布抽奖笔记后，如何查看中奖者并完成发货呢？本节我们来了解后续的管理流程。

### 查看与导出中奖名单

开奖后，你可以回到电脑端专业号平台，在“营销工具”->“抽奖活动”页面查看中奖用户。在这里，你可以：
*   查看中奖用户的收件地址与电话。
*   点击“导出表格”，下载包含中奖用户收货信息的表格。

### 填写并上传发货信息

下载中奖名单表格后，你需要填写发货单号等信息。填写完毕后，在同一页面点击“上传发货信息”，将表格上传回平台，以完成发货登记。

![](img/205fa5d7527233ad76e7274d85dac2f6_13.png)

## 注意事项与违规行为

为了保证活动公平和账号安全，你必须避免以下违规行为。平台将违规分为“活动违规”、“笔记违规”和“发奖违规”。

以下是主要的违规类型：
*   **活动违规**：包括诱导刷单、组织恶意活动、滥用用户信息等。
*   **笔记违规**：在抽奖笔记中诱导互动（如强制要求关注、点赞、收藏）、附加不合理的限制条件、或抽奖规则不清晰。
*   **发奖违规**：**不发奖品**或**超过30天未发货**。商家必须在开奖后30天内填写有效的发货单号。
