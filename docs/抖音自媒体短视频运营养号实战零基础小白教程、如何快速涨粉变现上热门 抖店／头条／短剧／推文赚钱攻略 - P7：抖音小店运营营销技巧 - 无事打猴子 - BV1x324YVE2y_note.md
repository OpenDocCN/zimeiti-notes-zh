## 店铺类型介绍

![](img/a431d115ae5403efc822e64bc7bde69b_5.png)

![](img/a431d115ae5403efc822e64bc7bde69b_6.png)

在开店之前，首先需要了解抖音小店的四种主要类型，以便根据自身情况做出合适的选择。

以下是四种店铺类型及其特点：

1.  **普通店**：适合个人或新手商家。常见于销售日用百货、精选好物等。开通此类店铺无需提供商标或品牌授权，仅需办理营业执照即可。若办理的是个体工商户营业执照，则只能选择普通店入驻。
2.  **专营店**：指经营同一品类下多个不同品牌商品的店铺。例如，经营服装类目，可以同时销售耐克、阿迪达斯、安踏等多个品牌的服饰，但需要获得相应的品牌授权。
3.  **专卖店**：指获得某一品牌授权，在特定渠道或区域销售该品牌所有品类商品的店铺。例如，获得“巴拉巴拉”品牌授权后，可以销售该品牌旗下的所有鞋服、配饰等商品。
4.  **旗舰店**：指品牌的官方直营店。一个品牌在抖音平台只能开设一家旗舰店，有权经营该品牌的所有品类商品。

![](img/a431d115ae5403efc822e64bc7bde69b_8.png)

![](img/a431d115ae5403efc822e64bc7bde69b_9.png)

对于大多数新手，尤其是计划采用“无货源”模式的商家，开通**普通店**是性价比最高的选择。

![](img/a431d115ae5403efc822e64bc7bde69b_11.png)

## 开店前准备工作

上一节我们介绍了店铺类型，本节中我们来看看开通小店前需要准备哪些硬件和材料。

开通抖音小店前，你需要准备好以下两样核心硬件：

![](img/a431d115ae5403efc822e64bc7bde69b_14.png)

*   **手机**：用于注册抖音号。该账号后续将用于开通商品橱窗，并绑定小店商品进行推广。建议遵循“一机一卡一账号”的原则。
*   **电脑**：用于登录抖音小店后台，进行店铺管理、商品上架、订单处理等操作，比手机端更便捷高效。

除了硬件，**营业执照**是开店的核心资质。主要分为两类：

*   **企业营业执照**：适用于公司或企业主体。拥有后可以开通所有类型的店铺（普通、专营、专卖、旗舰）。若考虑财务成本，也可为新业务单独办理。
*   **个体工商户营业执照**：适合个人卖家。一个身份证最多可办理5个个体户执照，而一个执照可认证6个抖音账号。这便于进行账号矩阵或店铺矩阵运营，提前储备执照也能规避平台未来可能提高的入驻门槛。

![](img/a431d115ae5403efc822e64bc7bde69b_16.png)

在正式操作前，请对照官方清单，准备好包括营业执照信息、法人身份证、银行账户、联系方式等在内的约17项材料，确保信息准确无误。

![](img/a431d115ae5403efc822e64bc7bde69b_18.png)

## 详细开店操作流程

准备好所有材料后，我们就可以进入正式的开店流程了。这个过程主要在电脑端完成。

![](img/a431d115ae5403efc822e64bc7bde69b_21.png)

![](img/a431d115ae5403efc822e64bc7bde69b_22.png)

![](img/a431d115ae5403efc822e64bc7bde69b_24.png)

![](img/a431d115ae5403efc822e64bc7bde69b_25.png)

以下是按步骤进行的操作指南：

1.  **访问后台**：在电脑浏览器中输入抖音小店官方后台网址。
2.  **开始入驻**：在登录页面点击“入驻抖音小店”，使用一个**未注册过小店**的手机号接收验证码登录。
3.  **选择主体类型**：根据你办理的营业执照类型，选择“个体工商户”或“公司/企业”。
4.  **提交资质信息**：
    *   上传营业执照照片，系统会自动识别并填充企业名称、信用代码等信息，请仔细核对。
    *   上传法定代表人身份证照片，系统会自动识别姓名、身份证号及有效期。
5.  **填写店铺信息**：
    *   **选择店铺类型**：根据自身情况选择（新手建议选“普通店”）。
    *   **设置店铺名称**：如“XX百货”、“XX精选”。
    *   **上传店铺Logo**：可自行上传或使用系统生成的默认Logo。
    *   **选择经营类目**：根据营业执照上的经营范围选择，后期可修改。
    *   **设置店铺管理员**：填写管理员姓名和手机号。
6.  **提交特殊资质（如有）**：如果经营食品、3C数码、图书、化妆品等特殊类目，需要按提示上传相应的许可证（如《食品经营许可证》）、质检报告或品牌授权书。若无，可直接跳过。
7.  **签署协议并等待审核**：确认信息无误后，阅读并同意平台协议，提交审核。审核通常需要1-3个工作日。
8.  **缴纳保证金**：审核通过后，根据所选经营类目缴纳相应的保证金。金额由平台规定，缴纳完成后店铺即正式开通。

## 开通商品橱窗与账号绑定

店铺开通后，为了销售商品，你需要开通抖音商品橱窗功能。这里针对不同账号情况，有两种主要路径。

![](img/a431d115ae5403efc822e64bc7bde69b_31.png)

### 情况一：已有粉丝基础账号（≥1000粉丝）

![](img/a431d115ae5403efc822e64bc7bde69b_33.png)

如果你的抖音账号已满足以下四个条件，可直接申请开通：
1.  已完成实名认证。
2.  已缴纳500元作者保证金。
3.  发布有效视频数≥10条。
4.  粉丝数≥1000。

**注意**：一个已开通橱窗的个人账号，无法直接用其手机号开通小店。系统会提示“暂不支持使用个人身份证入驻”。此时需要通过“官方账号绑定”功能来解决。

### 情况二：零粉丝新账号或解决绑定问题

![](img/a431d115ae5403efc822e64bc7bde69b_35.png)

对于新账号或上述遇到问题的账号，可以通过小店后台进行绑定授权：

1.  **绑定官方账号**：在小店后台，点击店铺名称，进入“官方账号绑定”页面。这里可以绑定一个**主抖音账号**，绑定后**无法解绑**，请谨慎选择。此方法可用于将小店绑定到已有粉丝的账号上。
2.  **绑定授权账号（子账号）**：在抖店后台的“营销中心” - “账号管理”中，可以绑定最多5个抖音账号作为子账号。这些账号可以是零粉丝账号。绑定后，180天内无法解绑或换绑。

![](img/a431d115ae5403efc822e64bc7bde69b_37.png)

**总结**：一个小店最多可绑定6个带货账号（1个官方主账号 + 5个子账号）。账号绑定成功后，需要在抖音APP的“创作者服务中心”缴纳500元**作者保证金**，即可通过该账号的橱窗推广小店商品。

## 核心步骤：如何选品

店铺和橱窗都准备好后，接下来最关键的一步就是选品。选品决定了店铺的成败，无论你是有货源还是无货源，都需要遵循科学的方法。

![](img/a431d115ae5403efc822e64bc7bde69b_39.png)

### 热门品类参考

![](img/a431d115ae5403efc822e64bc7bde69b_41.png)

对于没有货源的商家，可以从以下三大热门且易出单的品类入手筛选：
1.  **生鲜水果/农产品**：如苹果、葡萄、梨、哈密瓜、山楂制品等。属于快消品，复购率高。
2.  **日常百货/生活用品**：如洗洁精、洗衣液、牙膏、牙刷、纸巾、床上四件套等。家庭必备，消耗快。
3.  **家居/厨房用品**：如特定功能的厨房小工具、收纳用品、小型家电等。可通过视频直观展示使用场景，激发购买欲。

### 选品三大维度

在选择具体商品时，务必通过以下三个维度进行考察：
1.  **带货频率**：是否有大量主播频繁带货？可通过数据平台查看。
2.  **平台热度**：在淘宝、拼多多等主流电商平台的数据榜单上，该商品是否有一席之地？避免选择过于冷门的产品。
3.  **精选联盟热度**：在抖音“精选联盟”中搜索该商品，看其热度排名和佣金情况。热度越高，说明达人越愿意推广。

![](img/a431d115ae5403efc822e64bc7bde69b_43.png)

### 利用数据工具分析

![](img/a431d115ae5403efc822e64bc7bde69b_45.png)

我们可以借助第三方数据平台（如飞瓜数据、蝉妈妈）进行深入分析：

![](img/a431d115ae5403efc822e64bc7bde69b_47.png)

*   **查看商品榜单**：在数据平台查看“商品榜”、“实时爆款榜”，筛选自己感兴趣的品类，观察哪些商品销量高、增长快。
*   **分析商品详情**：点击具体商品，查看其“带货主播”数量、近30天销量、关联直播场次等数据。例如，使用公式 `带货达人数量 > 1000` 且 `30日销量 > 10万`，可初步判断为潜力爆款。
*   **考察商品评价**：在商品详情页查看“用户评价”，重点关注好评率与最新差评内容，确保产品本身质量过关，避免售后危机。

### 可视化场景与同行参考

*   **场景化视频参考**：在抖音搜索目标商品，学习优质带货视频如何通过可视化场景展示产品功能、解决用户痛点。例如，展示一款“洗碗巾”吸油快干、不易滋生细菌的特点。
*   **分析同行数据**：在数据平台搜索商品关键词（如“洗碗巾”），按销量排序，研究头部商品的定价、佣金、销量构成（是视频带货多还是直播带货多），以及是哪些核心达人在推广。

![](img/a431d115ae5403efc822e64bc7bde69b_49.png)

### 选品重要指标

综合来看，判断一个品是否值得做，有三个核心指标：
1.  **关联达人数**：越多达人带货，产品起量概率越大。
2.  **关联视频数**：视频内容能持续带来长尾流量。
3.  **关联直播数**：特别是头部达人的直播，是爆单的主要推力。

## 产品上架与优化细节

![](img/a431d115ae5403efc822e64bc7bde69b_54.png)

选好品之后，如何上架并优化商品信息，从而获取免费流量，是接下来的重点。

### 商品标题优化

![](img/a431d115ae5403efc822e64bc7bde69b_56.png)

未来抖音电商会越来越重视搜索流量。优化标题是获取免费搜索流量的关键。标题应控制在30字以内，并包含以下要素：
`标题 = 吸睛词 + 核心词 + 长尾词 + 品牌词（可选）`
例如：“**爆款**森马**连衣裙泡泡袖**2021**秋款新款**收腰**荷叶边裙**”
*   **吸睛词**：爆款
*   **核心词**：连衣裙、秋款新款
*   **长尾词**：泡泡袖、收腰、荷叶边裙
*   **品牌词**：森马（需有授权）

**注意**：做无货源模式时，用软件批量上架需检查标题，**删除**无授权的品牌词和违禁词。

### 商品主图设计

主图的目标是吸引点击。在抖音现阶段，一些具有促销信息、对比强烈的“牛皮癣”主图（如“买一送十”、“有它谁还用洗碗布”）仍有效果，但需确保不违规。主图应清晰、无水印、无他人品牌标识。

### 后台创建商品实操

在抖店后台“商品-商品创建”中，按步骤操作：
1.  选择正确类目。
2.  填写优化后的标题。
3.  上传5张高清主图及白底图（有助于获得推荐流量）。
4.  完善商品详情页。
5.  设置规格（如尺码、颜色）和价格库存。价格可设置“划线价”（原价）和“售价”，形成对比。
6.  设置运费模板（新手可选“标准模板-包邮”）。
7.  选择发货模式（建议“现货发货”，时效选“24小时内”）。
8.  确认所有带*号的信息已填妥，然后发布。

## 进阶策略：截流选品与定价技巧

![](img/a431d115ae5403efc822e64bc7bde69b_62.png)

掌握了基础选品和上架后，我们再来学习两种进阶策略：如何通过截流快速起量，以及如何定价能促进销售。

### 截流选品法

![](img/a431d115ae5403efc822e64bc7bde69b_64.png)

截流选品，即通过更低的价格或更高的佣金，截取同行潜力商品的流量。操作时关注两个维度：
1.  **销量**：在数据平台“实时爆款”中，寻找销量在**1000单以内**（最好低于200单）的同类商品。这类商品正处于上升期，竞争相对较小，更容易通过补单或达人合作实现超越。
2.  **利润**：确保自己有足够的利润空间（建议**30%-50%**），因为需要预留**20%-30%**的佣金给推广达人。

### 产品定价策略

![](img/a431d115ae5403efc822e64bc7bde69b_66.png)

定价前需明确销售模式：是自播自销，还是依赖达人分销？为达人分销需留足佣金空间。以下是四种实用的定价技巧：
1.  **非整数定价法**：利用消费者心理，定价如 `6.99`、`9.9`，比 `7`、`10` 显得更便宜。
2.  **心理定价法**：定价直接击穿用户心理防线，如 `0.39` 元比 `0.9` 元看起来便宜感更强。
3.  **差异定价法**：在同类产品中定一个明显更高的价格（如别人卖 `1.8`，你卖 `6.9`），并辅以更好的主图和详情，塑造“优质优价”的形象，满足部分用户“贵就是好”的心理。
4.  **A+B组合定价法**：通过搭配销售让用户感觉占便宜，如“一件 `10` 元，两件 `15` 元”、“买一送一”、“两瓶到手价 `99`”等。
