# 小红书运营：P43：店铺数据分析及复盘 📊

![](img/a3ec42b417ea2797f41d631a678122d4_1.png)

在本节课中，我们将学习小红书店铺的数据分析及复盘方法。数据分析是店铺运营中至关重要的一环，它能帮助我们理解流量来源、优化商品策略、了解客户群体，并最终提升店铺的整体表现和搜索排名。

上一节我们介绍了店铺的基础设置与运营，本节中我们来看看如何通过数据分析来驱动店铺的持续优化。

## 数据分析的重要性

数据分析是小红书店铺复盘的核心环节。与其他平台相比，小红书的数据分析体系相对清晰明确，其流量权重分布不像其他平台那样复杂。尽管如此，我们仍需重视数据分析，因为它能帮助我们获取那至关重要的30%的精准搜索流量。

笔记本身的分析相对简单，主要围绕笔记的优点、缺点以及如何针对产品或客户群体进行调整。这本质上是对我们内容基本功的检验。小红书70%的流量主体依然是笔记。

下面，我们将详细讲解店铺的流量数据分析体系。

![](img/a3ec42b417ea2797f41d631a678122d4_3.png)

## 数据分析的核心模块

店铺的数据分析是一个系统工程，主要包含以下几个部分：

以下是数据分析的主要构成模块：
1.  **流量分析**
2.  **销售数据分析**
3.  **客户分析**
4.  **商品分析**
5.  **竞争对手分析**
6.  **营销活动分析**（次要）
7.  **店铺运营指标分析**（次要）

这些分析得出的数据，最终会反馈并影响我们笔记推广策略的调整。整体而言，店铺优化以**流量分析为主，数据分析为辅**，因为笔记带来的流量本身不够精准。

![](img/a3ec42b417ea2797f41d631a678122d4_5.png)

### 1. 流量分析

流量分析的核心是追踪访客来源。通过分析“访客来源”，我们可以了解客户是通过笔记、搜索还是付费推广进入店铺的。

以下是流量分析的关键点：
*   **来源渠道**：区分笔记流量、搜索流量、付费推广流量。
*   **流量质量**：观察各渠道流量的转化效果。
*   **用户画像**：分析访客的性别、年龄、消费习惯、行为习惯等数据。

![](img/a3ec42b417ea2797f41d631a678122d4_7.png)

观察整体数据后，我们可以对比不同渠道的流量占比和协同效果（例如，笔记流量占70%，搜索流量占30%），找出异常数据点并针对性地进行修改。

### 2. 销售数据分析

![](img/a3ec42b417ea2797f41d631a678122d4_9.png)

销售数据分析关注店铺的成交情况，其核心目的是**确定我们的顾客群体**。

以下是销售数据分析的要点：
*   **整体销售额与单品销售情况**。
*   **产品客单价对比**。
*   **顾客购买频次**。

通过分析这些数据，我们可以判断顾客的消费能力、常逛的商品类型，以及是否需要优化目标人群或跟进推广其他产品。

![](img/a3ec42b417ea2797f41d631a678122d4_11.png)

### 3. 客户分析与商品分析

在明确了销售数据后，我们才能进行有效的客户分析，构建客户群体画像。简单来说，流量分析和销售分析的最终目的，都是为**客户分析**服务。

访客进入店铺、购买商品并成交后，我们需要分析访客数据，进而分析商品数据。根据访客的反馈，调整我们的主推商品或产品线。

以下是分析逻辑：
*   如果**产品**和**内容**合格，但**客户**不匹配（成交率低），说明引流人群不准，需要修改笔记标题和内容。
*   如果**客户**匹配，但**成交率**低，则说明商品推广方式或产品本身可能有问题。

商品分析主要关注商品的销售量、加购量等市场反馈数据。

### 4. 竞争对手分析

竞争对手分析是提升店铺排名的关键。在小红书商家后台的“数据分析”或通过搜索，可以观察流向同类竞争对手店铺的流量。

以下是竞争对手分析的操作步骤：
1.  **使用新账号**（非自己店铺号）访问竞争对手店铺。
2.  **对比同类型产品**的价格、销量、推广方式。
3.  **观察客服服务**，对比自家服务水平。
4.  **分析对手的日成交金额和转化率**，估算其访客流量和成交转化率。

分析竞争对手的目的不是击垮对方，而是**让自己的店铺数据超越对方**。当你的数据更好时，搜索排名就会更高，你的竞争对手层级也会随之更换。此方法在店铺运营中期尤为实用。

### 5. 营销活动与店铺运营指标分析

*   **营销活动分析**：在小红书平台，笔记本身就是核心的营销活动，平台内的营销活动作用相对有限，可做可不做。但需注意完成店铺等级相关的每日任务。
*   **店铺运营指标分析**：主要包括动态评分（DSR）、退款纠纷率等售后服务质量指标。这些指标反映了店铺的整体运营质量和客户满意度。**需尽量保证评分在4.8左右**，避免过低。

### 6. 搜索排名分析

搜索排名是上述所有数据分析结果的综合体现。小红书系统会根据多项因素给出店铺的整体排名。

以下是影响搜索排名的主要因素：
*   **商品维度**：标题、关键词、销量、评价。
*   **店铺维度**：动态评分（DSR）。
*   **内容维度**：笔记的整体引流效果。

此外，还有独立的**笔记排名**，主要由笔记的展示量和点击率决定。例如，一篇笔记获得3000次展示，若点击率达到20%（即600次点击），则可能成为“小爆文”，获得更多流量；否则，可能只有300-500的基础流量。
