# 小红书运营教程：P24：小红书暴力玩法揭秘！吸粉推广篇

在本节课中，我们将要学习小红书运营中的两种核心推广策略：自建账号矩阵运营与官方广告投放。我们将理清两者的关系，并详细介绍各自的运作方式与适用场景，帮助你根据自身情况选择最合适的推广路径。

## 账号类型解析

上一节我们介绍了推广的整体思路，本节中我们来看看具体执行的基础——账号类型。小红书的账号主要可以分为以下三类。

以下是三种主要账号类型及其特点：

1.  **企业号**
    *   主要作用是树立官方品牌形象。
    *   拥有官方认证标志和特权，例如展示品牌名称、地址等，便于开展业务。

2.  **个人号**
    *   目前大多数用户运营的账号类型，特点是**灵活性高**。
    *   运营者可以是品牌员工或消费者，易于塑造个人品牌特色，适合人物出镜。

3.  **引流号**
    *   在之前的课程中提及过，俗称“炮灰号”。
    *   数量可以是多个，作用相对次要，通常没有精细的个性规划，可用于临时引流或作为备用小号。

通常，品牌推广会通过企业号、个人号或引流号自行发布笔记进行引流。

## 企业号矩阵运营建议

了解了账号分类后，我们来看看如何系统性地运用它们。打造一个有效的企业号矩阵，可以遵循以下建议。

以下是构建企业号矩阵的四点建议：

1.  **积极参与品牌号活动**：若品牌旗下有企业号（如婚纱、服装美妆类），应使用小号积极参与其发起的试用、团购等活动。
2.  **个人号展示品牌属性**：在个人号的笔记内容中，应多次暗示品牌的定位属性，例如目标用户是“城市年轻人”还是“职场新人”。
3.  **控制个人号内容比例**：个人号发布品牌产品内容的频率不宜过高。建议采用 **`营销内容 : 种草/科普内容 = 3 : 7`** 的比例。
4.  **利用引流号进行评论引流**：当企业号需要直接引流时，最佳做法是使用引流号在其笔记评论区进行引导。

## 官方广告投放介绍

![](img/667a463c28b28ca2df37ca68ff27f762_3.png)

除了自建账号矩阵，另一种高效的推广方式是借助平台力量。接下来，我们看看小红书官方的广告系统。

![](img/667a463c28b28ca2df37ca68ff27f762_5.png)

![](img/667a463c28b28ca2df37ca68ff27f762_7.png)

小红书官方广告主要指 **CBC（Cost Per Click）竞价广告**。商家通过竞价购买广告位，以获得流量。

![](img/667a463c28b28ca2df37ca68ff27f762_9.png)

### 投放前提与流程

![](img/667a463c28b28ca2df37ca68ff27f762_11.png)

投放官方广告有一个必要前提：**必须已认证企业号**。

企业号认证流程如下：
1.  访问小红书官网。
2.  找到“业务合作”或“推广合作”入口。
3.  按照指引提交相关资质并支付审核费用。
4.  审核通过后，普通账号即升级为官方认证企业号。

### 广告类型与展示位

拥有企业号后，可投放的广告主要分为两大类：

1.  **信息流广告**：用户浏览小红书首页时，不定时出现在信息流中的广告。
2.  **搜索广告**：当用户使用关键词搜索时出现的广告。其原理是**根据用户搜索意图进行相关推荐**。例如，用户搜索“考研资料”，系统可能推荐补充能量的商品。

搜索竞价广告最具代表性的是百度，小红书借鉴了其模式。通过对关键词出价，竞争不同的广告排名。

![](img/667a463c28b28ca2df37ca68ff27f762_13.png)

目前，信息流广告位主要分为以下三类：
*   第3位
*   第13位
*   第23位

关于广告投放的常见问题，可在小红书广告平台的官方问答区找到解答。官方广告投放费用较高，但在资金允许的情况下，建议考虑合作。因为其流量由平台官方直接推荐，效率通常更高。
