# 视频号创作分成计划：P20：19.实操步骤 - 短视频变现教程 🎬

在本节课中，我们将要学习如何从零开始，在微信视频号中开通并设置“带货中心”，这是参与创作分成计划、实现短视频变现的关键一步。整个过程分为开通权限和上架商品两个主要部分，步骤清晰，操作简单。

上一节我们介绍了视频号变现的基本概念，本节中我们来看看具体的操作流程。以下是详细的实操步骤，请跟随指引一步步进行。

## 第一步：打开微信
首先，你需要打开手机上的微信应用。这是所有操作的起点。

![](img/c8857511d956d85c62a9117af07be50b_1.png)

![](img/c8857511d956d85c62a9117af07be50b_2.png)

## 第二步：进入“发现”页
在微信主界面底部，找到并点击 **“发现”** 选项卡。

## 第三步：找到视频号入口
在“发现”页面中，将视线向上滑动。你会看到一个标有 **“视频号”** 字样的入口，其图标是一个黄色的蝴蝶形状。点击此处进入视频号。

![](img/c8857511d956d85c62a9117af07be50b_4.png)

![](img/c8857511d956d85c62a9117af07be50b_6.png)

![](img/c8857511d956d85c62a9117af07be50b_1.png)
![](img/c8857511d956d85c62a9117af07be50b_2.png)

## 第四步：进入个人中心
进入视频号后，在界面右上角找到一个 **人形头像** 图标。点击这个图标进入个人中心。

![](img/c8857511d956d85c62a9117af07be50b_8.png)

![](img/c8857511d956d85c62a9117af07be50b_4.png)
![](img/c8857511d956d85c62a9117af07be50b_6.png)

![](img/c8857511d956d85c62a9117af07be50b_9.png)

## 第五步：进入创作者中心
在个人中心页面，向下滑动直至最底部。找到并点击 **“创作者中心”**。请注意，如果你是新注册的账号且未发布过作品，此选项可能不会显示。发布一条作品后即可出现。

## 第六步：找到带货中心
进入创作者中心后，在页面中找到 **“带货中心”** 的选项，其图标类似一个小卖部。点击进入。

![](img/c8857511d956d85c62a9117af07be50b_8.png)
![](img/c8857511d956d85c62a9117af07be50b_9.png)

接下来，系统会根据你的账号状态引导你进行后续操作。主要分为两种情况：**开通橱窗权限** 和 **上架商品**。

![](img/c8857511d956d85c62a9117af07be50b_11.png)

### 情况一：开通橱窗权限（针对未开通用户）
如果你从未开通带货功能，点击“带货中心”后会跳转到开通权限的界面。

以下是开通权限的具体步骤：
1.  **选择带货身份**：界面会要求你选择带货身份。选项通常包括“我是商家”、“我是达人”等。
2.  **身份选择**：**请务必选择“我是达人”**。即使你是厂家或拥有自有品牌，也建议先以达人身份开通，因为该身份可以关联平台精选联盟的货源（产品是0成本的）。一个身份证可以认证两个账号，第二个账号可以选择商家身份。
3.  **缴纳保证金**：选择“我是达人”后，会跳转到缴纳保证金的界面。你需要点击 **“去交纳”**，系统将引导你通过微信支付100元保证金。完成支付即开通成功。

![](img/c8857511d956d85c62a9117af07be50b_11.png)
![](img/c8857511d956d85c62a9117af07be50b_13.png)

![](img/c8857511d956d85c62a9117af07be50b_13.png)

### 情况二：上架商品（针对已开通用户）
如果你已经开通了橱窗权限，点击“带货中心”后将直接进入选品上架的界面。你可以从平台的精选联盟中选择商品，将其添加到自己的橱窗中，以便在视频或直播中进行推广。

整个操作的核心逻辑可以总结为：
```
if (用户未开通带货权限) {
    点击“带货中心” -> 引导至【开通流程】-> 选择身份(达人) -> 缴纳保证金 -> 开通成功
} else {
    点击“带货中心” -> 直接进入【选品上架】界面
}
```

本节课中我们一起学习了在微信视频号中开通和设置“带货中心”的完整流程。从打开微信开始，经过发现页、视频号入口、个人中心、创作者中心，最终到达带货中心。我们重点区分了开通权限与上架商品两种情况的处理方式，并明确了以“达人”身份起步的优势。掌握这些步骤，你就完成了短视频变现的基础设施搭建。
