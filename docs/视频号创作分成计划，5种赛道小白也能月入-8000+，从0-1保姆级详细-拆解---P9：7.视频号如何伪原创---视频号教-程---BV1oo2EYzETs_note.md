# 视频号创作分成计划：P9：7. 视频号如何伪原创 🎬

在本节课中，我们将要学习视频号平台对原创内容的扶持政策，并重点拆解如何通过“伪原创”的方式，从零开始创作内容并获取流量。我们将从平台规则讲起，通过实际案例展示伪原创的可行性，并详细介绍从选品到发布的完整操作流程。

---

## 平台原创扶持政策

上一节我们介绍了视频号的基本操作，本节中我们来看看平台对创作者的激励措施。平台本身支持原创，并设有创作者激励计划，对原创内容提供流量倾斜。

以下是创作者激励计划的具体内容：
*   **流量扶持**：平台计划让1000万原创作者获得流量。
*   **收入激励**：让百万优质创作者获得收入。
*   **零粉阶段**：平台会赠送2500到5万点的流量券，相当于直接推送观众。
*   **千粉阶段**：可加入重点作者扶持计划，优质视频可申报专项奖金。
*   **万粉及百万粉阶段**：可申报的专项奖金更多。

因此，平台会逐级将原创内容推进更大的流量池。原创固然好，但这通常需要个人具备拍摄、编剧等技术能力。

## 伪原创的可行性与案例

我看到很多同学表示想做伪原创，并且清楚自己作为新手，目前独立完成原创比较困难。今天我们将重点讲解搬运与二次创作。

让我们看一个案例。这位“大学生中娜”同学在疫情期间封校，课余时间开始研究视频号。在指导下，她通过搬运内容获得了不错的流量。

## 伪原创完整操作流程

伪原创并非直接下载发布，而是有一套完整的流程。平台支持原创，直接搬运带水印的内容可能导致账号违规。因此，我们需要进行“伪”创作。

完整的伪原创流程如下：

![](img/a20e01f2621b55e2ab24a3fa8658740f_8.png)

**第一步：选品与定位**
首先，你需要确定要售卖的商品或账号类型。可以通过以下途径进行选品：
1.  进入视频号，点击右上角个人头像。
2.  进入“创作者中心”。
    ![](img/a20e01f2621b55e2ab24a3fa8658740f_8.png)
    ![](img/a20e01f2621b55e2ab24a3fa8658740f_9.png)
3.  或在其他电商平台（如1688、拼多多）寻找商品灵感。

![](img/a20e01f2621b55e2ab24a3fa8658740f_9.png)

根据你想带的货来确定账号定位。例如：
*   `定位音乐账号` -> 可售卖车载U盘、承接音乐广告。
*   `定位影视账号` -> 可承接广告、售卖相关商品或知识付费内容。
*   `定位宠物账号` -> 可售卖宠物用品。

**第二步：寻找并确定素材源**
确定定位后，去其他平台（如抖音）寻找对应领域的优质账号。关键一步是：**检查该账号的内容是否已被大量搬运到视频号**。如果已有大量重复内容，则应避免再次搬运。

**第三步：下载与去水印**
确定可用的素材源后，不要直接下载发布。首先需要将视频下载下来，并**去除原视频的水印**。这是一个必要的步骤。
![](img/a20e01f2621b55e2ab24a3fa8658740f_11.png)

**第四步：二次剪辑**
去水印后，需要对视频进行二次剪辑，使其看起来像是新作品。这包括但不限于：调整片段顺序、添加新的字幕或贴纸、改变背景音乐、进行调速或添加转场效果等。

**第五步：发布与运营**
完成剪辑后，按照平台规则进行发布，并配合相关的标题、话题标签，同时可以参与平台的热点话题以获得更多曝光。
