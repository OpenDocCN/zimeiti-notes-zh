# 新媒体运营：P33：16：百度AI一键自动生成热点视频工具 🚀

在本节课中，我们将学习如何使用百度AI工具，实现热点视频的自动生成。这个工具能帮助我们快速创作内容，抓住热点，提升运营效率。

![](img/6ecc7b326c946b70eecf55c3cc912de3_1.png)

## 工具界面与核心功能概览

上一节我们介绍了课程主题，本节中我们来看看工具的具体界面和核心功能。

进入工具界面后，首页主要提供两大核心功能：**AI成片**和**AI笔记**。左侧导航栏也可以找到这两个入口。

以下是两大核心功能的简要介绍：
*   **AI成片**：输入文案或选择文章，工具可自动匹配素材和配音，一键生成完整视频。
*   **AI笔记**：一个文本生成与润色工具，类似于ChatGPT，可用于创作文案、脚本等。

## AI成片功能详解

上一节我们了解了工具概览，本节中我们深入探讨“AI成片”功能的具体使用方法。

“AI成片”功能主要分为两种使用方式。

以下是两种生成视频的路径：
1.  **输入外部文案**：将已有的视频脚本或文案粘贴到输入框，工具会自动搜集素材并匹配声音。
2.  **选择文章成片**：需要拥有百家号并发布过原创文章。成功关联后，可直接选择已发布的文章来生成视频。

此外，该功能还有一个强大的“蹭热点”模块。工具右侧会展示实时热点话题。

![](img/6ecc7b326c946b70eecf55c3cc912de3_3.png)

以下是利用热点生成视频的步骤：
1.  在热点列表中选择一个话题。
2.  点击“一键生成文案”，AI会自动撰写该热点事件的视频文案。
3.  生成文案后，可点击“AI一键成片”自动制作视频。

![](img/6ecc7b326c946b70eecf55c3cc912de3_5.png)

![](img/6ecc7b326c946b70eecf55c3cc912de3_7.png)

**注意**：AI生成的文案需人工审核，确保内容合法合规，避免传播不实信息。

## 视频生成后的编辑与优化

![](img/6ecc7b326c946b70eecf55c3cc912de3_9.png)

上一节我们学会了如何生成视频，本节中我们来看看生成后如何进行编辑和优化。

![](img/6ecc7b326c946b70eecf55c3cc912de3_11.png)

视频自动生成后，进入编辑界面。有时AI匹配的素材可能不准确或缺失，需要进行调整。

以下是编辑视频的几个关键环节：
*   **素材替换**：如果某个片段素材不合适，可以点击该片段进行替换。编辑界面提供“推荐素材”搜索和“本地素材”上传功能。
*   **模板选择**：工具提供横版和竖版等多种视频模板，可根据发布平台选择。
*   **朗读音与背景音乐**：
    *   **朗读音**：可以更换配音音色，并调整语速以适应视频节奏。公式表示为：`最终效果 = 配音音色 + 语速调整`。
    *   **背景音乐**：可以在内置音乐库中选择更换，但目前不支持自定义上传。

**技巧**：生成视频并下载到本地后，可以使用专业剪辑软件进行更深度的加工，如替换背景音乐、去除水印等。

## 视频的下载与发布

上一节我们完成了视频的编辑，本节中我们来看看如何下载和发布最终的视频作品。

视频编辑满意后，点击右上角的“发布视频”按钮。

![](img/6ecc7b326c946b70eecf55c3cc912de3_13.png)

以下是下载视频的步骤：
1.  点击“发布视频”后，在弹出的界面中选择“生成视频”。
2.  系统会进入渲染队列，等待几分钟。
3.  视频生成完成后，界面会出现“下载”按钮，点击即可将视频保存到本地。
4.  下载后，可对视频进行最终优化，然后发布到任意平台。

![](img/6ecc7b326c946b70eecf55c3cc912de3_15.png)

**重要提示**：使用任何工具创作和发布内容，都必须遵守国家法律法规和平台规则。

![](img/6ecc7b326c946b70eecf55c3cc912de3_17.png)

## AI笔记（文案助手）功能简介

![](img/6ecc7b326c946b70eecf55c3cc912de3_19.png)

上一节我们掌握了视频生成的全流程，本节中我们简要了解另一个辅助工具“AI笔记”。

“AI笔记”是一个强大的AI文本创作助手，能帮助我们生成和优化文案。

![](img/6ecc7b326c946b70eecf55c3cc912de3_21.png)

以下是AI笔记的主要用途：
*   **文案创作**：输入主题或问题，AI可自动生成文章、脚本、广告语等。
*   **文笔润色**：将已写好的文案粘贴进去，AI可帮助优化语句，使其更流畅、更有吸引力。

这个工具由百度开发，符合中文使用习惯，是新媒体运营人员在文案工作上的得力助手。
