# 微信视频号运营教程：P36：5. 二创书单制作方法二

## 概述
在本节课中，我们将学习如何通过“二创”的方式制作书单类视频。核心思路是：找到目标视频，提取其文案并重新配音，同时替换视频中的部分画面素材，最终生成一个符合平台原创要求的新视频。

## 核心操作流程
上一节我们介绍了二创的基本概念，本节中我们来看看具体的操作步骤。整个过程可以概括为以下几个核心环节：

![](img/c2e3b924d1288dd8b8479349572cbddd_7.png)

![](img/c2e3b924d1288dd8b8479349572cbddd_9.png)

1.  **寻找并下载源视频**：在抖音等平台搜索目标书籍或主题，找到高销量的带货视频作为素材。
2.  **处理源视频**：去除视频水印，并导入剪辑软件进行初步剪辑。
3.  **提取与重制音频**：从处理后的视频中提取文案，并使用配音工具生成新的旁白。
4.  **替换视频素材**：将原视频中的部分画面替换为其他相关素材，完成画面“二创”。
5.  **合成与导出**：将新音频与新画面结合，添加背景音乐和字幕，最终导出成品。

## 详细步骤解析

![](img/c2e3b924d1288dd8b8479349572cbddd_13.png)

### 第一步：获取与准备源视频
首先，我们需要找到一个合适的视频作为改造基础。

*   **搜索目标**：以书籍《走进大学城》为例，在抖音搜索框选择“商品”并筛选“销量”，可以找到相关的带货视频。
*   **选择视频**：从搜索结果中，选择一个结构清晰、文案有吸引力的视频。例如，一个时长约2分钟，前半部分展示大学生活，后半部分引导购书的视频。
*   **下载去水印**：复制该视频链接，使用去水印工具下载无水印版本到本地。

### 第二步：剪辑源视频并提取文案
获取视频后，我们需要将其导入剪辑软件进行处理。

*   **导入与分割**：将下载好的视频导入剪映。如果原视频前后部分声音或风格不一致（例如前半段是展示，后半段是推销），可以在转折点（如53秒处）进行分割。我们通常只需要保留前半部分的展示性内容。
    ```python
    # 伪代码示意：在剪辑时间轴53秒处执行分割操作
    split_video_at(video_clip, time=53)
    ```
*   **导出片段**：将分割后需要的视频片段（如前53秒）导出为一个新视频文件。这个文件将用于提取文案。

### 第三步：提取文案并生成新配音
接下来，我们需要将视频中的话术转化为文字，并制作新的配音。

*   **提取文案**：使用“配音神器”等工具的“视频提取文案”功能，上传第二步导出的视频片段，系统会自动识别并生成文字稿。
*   **生成配音**：在配音神器中选择一个合适的声音（如成熟稳重的女声），将上一步提取的文案输入，生成新的音频文件并导出。

### 第四步：替换视频画面素材
这是实现“二创”的关键步骤，通过替换画面避免直接搬运。

*   **分析原视频**：播放原视频，识别出可以替换的片段。例如：
    *   一段3秒的“毕业典礼”镜头。
    *   一段“打高尔夫”的镜头。
    *   一段展示“宿舍环境”的镜头。
*   **搜索新素材**：在抖音分别搜索“高校毕业典礼”、“美女打高尔夫”、“精装修宿舍”等关键词，找到合适的替代视频，下载并去水印。
*   **执行替换**：回到剪映中的原视频工程。
    1.  删除原视频中需要替换的片段（如那3秒的毕业典礼镜头）。
    2.  将下载好的新素材导入，并裁剪出时长基本相同的片段（如3秒）。
    3.  将新片段放入被删除片段的位置，在连接处添加“叠化”等转场效果。
    4.  **务必关闭**新素材片段的原始音量。
*   **重复操作**：对其余需要替换的画面重复以上“搜索-下载-替换”流程。通常替换1-2个核心画面即可显著改变视频面貌。

### 第五步：合成音频与最终调整
现在，我们将新的配音与修改后的画面结合起来。

*   **添加新配音**：在剪映中，关闭原视频的所有原声。导入第三步生成的新配音音频文件，将其拖入音轨。
*   **音画对齐**：检查新的配音长度是否与视频画面总长度匹配。
    *   **音频过长**：可以适当对视频片段进行“常规变速”（略微调慢），或添加一些相关空镜头来延长画面。
    *   **音频过短**：可以加快部分视频片段速度，或删减冗余画面。
    *   目标是让配音结束时，视频画面也基本结束。
*   **处理视频后半部分**：如果保留原视频后半段（如达人出镜讲解部分），可以保留其原声，但最好也替换掉对应的画面，减少原出镜人的面部特写时长，以降低侵权风险。

### 第六步：添加包装与导出
进行最后的润色，让视频更完整。

*   **添加背景音乐**：在“音频”中选择一首合适的纯音乐作为背景音乐，音量调低，避免遮盖配音。
*   **添加新字幕**：
    1.  使用剪映的“识别字幕”功能，为新的配音自动生成字幕。
    2.  调整字幕的样式、位置和大小，使其**完全覆盖**原视频可能残留的旧字幕。
        ```python
        # 伪代码示意：生成新字幕并覆盖于旧字幕之上
        new_subtitle = generate_subtitle(new_audio)
        new_subtitle.style.position = ‘cover_old’
        ```
*   **最终检查**：播放整个视频，确保无原视频水印、新旧画面过渡自然、音画同步、字幕准确。
*   **导出设置**：选择分辨率1080P，帧率60帧，码率推荐，然后导出视频。

## 总结
本节课我们一起学习了书单类视频的二创制作全流程。其核心公式可总结为：**新视频 = （原视频文案 -> 新配音） + （原视频画面 - 部分旧素材 + 新搜索素材）**。关键在于**文案重组**与**素材替换**。对于初学者，前几个视频制作可能会比较慢，但只要严格按照步骤实操，完成10个左右的视频后，速度和质量都会大幅提升。剪辑是一项熟能生巧的技能，重在坚持练习。
