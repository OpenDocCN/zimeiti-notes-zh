# 抖音运营教程：P9：产品与用户兴趣标签 🧩

在本节课中，我们将学习一种名为“九宫格选题法”的实用技巧。该方法通过分析产品属性和用户兴趣，帮助创作者高效生成源源不断的视频选题，解决“不知道拍什么”的难题。

上一节我们介绍了选题的重要性，本节中我们来看看如何具体操作。我们将以健身软件Keep为例，演示如何运用九宫格法进行选题。

## 第一步：分析产品相关标签

首先，我们需要围绕产品本身，挖掘其核心特点与属性。将目标产品（例如Keep）置于九宫格中心，然后在周围的八个格子中填入你能想到的所有产品相关关键词。

以下是具体操作步骤：

1.  **确定核心产品**：将“Keep健身软件”写在九宫格中心。
2.  **联想产品特点**：围绕该产品，思考并填写其各项功能与属性。
    *   它是一个**线上健身软件**。
    *   它提供**专业健身教练**的指导。
    *   它包含**饮食搭配**建议。
    *   它提供**减肥攻略**。
    *   它传授**健身干货知识**。
    *   它教授**科学的运动方法**。
    *   它具有**打卡运动**的激励制度。
    *   它的主要用户集中在**一二线城市**。

完成这一步后，产品侧的标签就填充完毕了。这些标签代表了你的视频内容可以依托的产品基础。

## 第二步：分析用户兴趣标签

接下来，我们需要分析产品的目标用户，并挖掘他们可能感兴趣的其他领域。将核心用户画像（例如“健身爱好者”）置于另一个九宫格的中心，然后进行发散联想。

![](img/456831987e66196e6aef98e7823059ef_1.png)

以下是具体操作步骤：

1.  **确定核心用户**：将“健身爱好者/想要减肥的人”写在九宫格中心。
2.  **联想用户兴趣**：思考这类用户除了健身，还可能对什么感兴趣。思路要打开，进行广泛联想。
    *   他们可能对**健身房**感兴趣。
    *   他们可能对**健身装备**（如运动鞋、紧身衣）感兴趣。
    *   他们可能对**轻脂美食/减肥餐**感兴趣。
    *   作为热爱生活的人，他们可能对**旅行**感兴趣。
    *   他们可能也对**星座、明星、鸡汤文**等大众兴趣点感兴趣。

通过这一步，我们将用户群从“健身爱好者”扩展到了“热爱生活的人”，从而获得了更广泛的选题灵感来源。

## 第三步：进行“连连看”组合选题

现在，我们手头有了两大组标签：**产品标签**和**用户兴趣标签**。真正的创意来自于将这两组标签进行随机碰撞与组合。

以下是具体操作方法：

*   **核心方法**：从产品标签和用户兴趣标签中各选一个词，将它们结合起来，构思一个完整的视频标题或主题。
*   **组合示例**：
    *   组合 **“饮食搭配”** (产品标签) 与 **“旅行”** (用户兴趣标签)
        *   可能选题：**《外出旅行时，如何享受美食又能控制体重？》**
    *   组合 **“运动方法”** (产品标签) 与 **“明星”** (用户兴趣标签)
        *   可能选题：**《赵丽颖都在练的5个高效瘦身动作，一学就会！》**
    *   组合 **“科学方法”** (产品标签) 与 **“健身房”** (用户兴趣标签)
        *   可能选题：**《避开误区！在健身房必须知道的3个科学健身原则》**

通过这种“连连看”式的组合，你可以轻松衍生出数十个甚至上百个选题方向。如果创意枯竭，可以借助搜索引擎，输入组合关键词（如“健身 科学”），查看相关的长尾词和热门话题，进一步激发灵感。

## 总结与过渡

本节课中，我们一起学习了“九宫格选题法”的三个核心步骤：**分析产品标签**、**分析用户兴趣标签**以及**将两者进行创意组合**。这个方法能系统化地为你构建一个庞大的选题库，实现内容的持续产出。
