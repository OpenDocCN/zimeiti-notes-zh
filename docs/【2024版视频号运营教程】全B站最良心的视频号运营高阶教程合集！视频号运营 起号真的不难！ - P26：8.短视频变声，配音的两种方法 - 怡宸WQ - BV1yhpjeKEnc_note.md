# 视频号运营教程：P26：短视频变声与配音的两种方法 🎬

在本节课中，我们将学习在视频混剪时，如何对视频的声音进行去重处理。我们将介绍两种实用方法：第一种是直接在剪映App内使用变声功能；第二种是通过配音神器提取文案并生成全新配音。这两种方法都能有效改变视频的音频特征，帮助你的内容通过平台审核。

## 方法一：在剪映内直接变声 🎚️

上一节我们概述了课程内容，本节中我们来看看第一种实现声音去重的方法。此方法操作简单，能快速改变原视频的人声音色。

首先，在剪映中导入需要处理的视频素材。选中该视频轨道，在左侧功能面板中找到“声音效果”选项。

![](img/8f60b6e94f13b388f91fcaf805f6659a_1.png)

以下是具体操作步骤：
1.  点击“声音效果”后，会出现多种变声选项。
2.  选择你喜欢的音效，例如“男生”类别下的“大叔”或“小帅”等。
3.  部分音效支持参数微调，你可以根据预览效果进行调节。

通过此方法，原视频的人声会被改变，但背景音乐保持不变。这是一种快捷的初步去重手段。

## 方法二：使用配音神器生成全新配音 🗣️

除了直接变声，我们还可以通过提取文案并重新配音的方式，实现更彻底的音频去重。这种方法会改变全部音频内容。

首先，需要使用“配音神器”这类工具。将视频导入后，利用其“文案提取”功能，把视频中的对白文字提取出来。

为了提高去重效果，建议对提取出的文案进行适当修改。修改原则是保持原意，但调整部分词语或句式。

以下是文案修改示例：
*   **原文**：老人给小孩削皮的时候，我们还不放心。怎么办呢？
*   **修改后**：大人给小孩削皮的时候，我们还不放心。咋办呢？

![](img/8f60b6e94f13b388f91fcaf805f6659a_3.png)

修改完成后，将新文案复制到配音神器的配音界面，选择一个你喜欢的声音角色（如“男生”），然后合成并导出新的配音文件。

![](img/8f60b6e94f13b388f91fcaf805f6659a_5.png)

## 在剪映中替换音频 🔊

现在，我们得到了全新的配音文件。接下来需要在剪映中完成音频替换，以实现最终效果。

![](img/8f60b6e94f13b388f91fcaf805f6659a_7.png)

打开剪映，重新导入原始视频。关闭视频的原声，然后通过“添加音频” -> “提取音乐”功能，导入我们刚刚生成的新配音文件。

此时，视频的原始人声和背景音乐都已被替换。你可以在此基础上，添加新的背景音乐，并调整音量平衡，使配音突出。最后，可以运用之前学过的混剪技巧，对画面进行进一步处理。

本节课中我们一起学习了两种短视频声音去重的方法：在剪映内直接变声，以及通过配音神器提取文案并生成全新配音。掌握这两种方法，能有效提升你混剪视频的原创度，帮助账号更健康地运营。
