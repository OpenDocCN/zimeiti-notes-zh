## 核心分析工具：新榜与微小宝

公众号数据分析主要可以通过两个平台进行：**新榜**和**微小宝**。它们提供了互补的功能，帮助我们全方位了解行业动态。

![](img/80670c6317bf86ff47376f411bffb2ee_8.png)

## 功能一：查看公众号榜单 📈

![](img/80670c6317bf86ff47376f411bffb2ee_10.png)

以下是新榜和微小宝在榜单功能上的具体应用：

*   **新榜**：主要提供公众号的综合排行榜单，支持按不同领域（如科技、教育、生活等）进行筛选，方便我们了解各垂直领域的头部账号。
*   **微小宝**：除了提供类似的排行榜和领域榜单外，还有一个特色功能——**查看排名上升最快的账号榜单**。这个功能对于发现新兴的、增长迅速的账号非常有价值，我们可以学习他们的运营策略。

![](img/80670c6317bf86ff47376f411bffb2ee_12.png)

## 功能二：了解行业平均水平 📊

在微小宝平台中，有一个“数据研究院”模块。它通过采集海量公众号数据，计算出全平台的平均运营水平。

![](img/80670c6317bf86ff47376f411bffb2ee_14.png)

![](img/80670c6317bf86ff47376f411bffb2ee_15.png)

**公式示例：行业平均数据 ≈ 新账号的参考基准**

![](img/80670c6317bf86ff47376f411bffb2ee_17.png)

例如，一份报告可能显示：平均粉丝数约为3万，日均净增粉丝约30个。如果你的新公众号能达到这个数据，就意味着你达到了行业的平均水准。这个功能支持按领域查看，让你对所在细分领域的“及格线”有清晰的认识。

## 功能三：进行竞品对比分析 ⚔️

无论是新榜还是微小宝，都支持对特定的公众号进行深入的数据分析。

![](img/80670c6317bf86ff47376f411bffb2ee_19.png)

**操作逻辑**：`添加目标公众号 -> 查看详细数据报告 -> 与自身账号进行对比`

![](img/80670c6317bf86ff47376f411bffb2ee_21.png)

![](img/80670c6317bf86ff47376f411bffb2ee_23.png)

例如，你可以将自己公司的公众号和竞争对手的公众号添加到监控列表，对比双方的新榜指数、阅读量、发布频率、互动数据等，从而找出差距和优化方向。

## 功能四：分析热门文章 🔥

在微小宝平台上，可以查看不同时间维度的热门文章排行榜。

**代码示例（筛选逻辑）**：
```python
# 查看热门文章
time_range = “最近7天” # 可选：24小时、3天、7天、30天
sort_by = “传播指数” # 或按阅读量、点赞数排序
hot_articles = get_hot_articles(time_range, sort_by)
```

## 功能五：查看违规文章以规范创作 🚫

这个功能对内容创作者至关重要。微小宝平台会公示一些因违规而被处理的文章案例。

以下是常见的违规类型：
*   **涉及低俗、性暗示内容**
*   **过度营销、骚扰用户**
*   **传播不实信息**

定期查看这些违规案例，可以清楚了解平台的内容红线，对自身的创作起到警示和规范作用，避免账号受到处罚。

## 功能六：深入分析特定公众号 🔍

以新榜为例，如果我们对某个账号（如“十点读书”）感兴趣，可以点进去查看其详细的数据面板。

**数据分析维度包括**：
1.  **基础信息**：认证情况、账号简介、所属领域。
2.  **多平台布局**：该机构在微博、头条等其他平台的相关账号。
3.  **指数趋势**：观察其新榜指数（NCII）随时间的变化。稳定在高分（如950以上）代表运营非常稳定且优质。
4.  **内容数据**：发布次数、篇数、10万+文章数、平均阅读量、点赞量等。
5.  **发布规律**：分析其每日发布文章的数量和时间规律。例如，“十点读书”通常每日推送一次，每次8篇文章，非常稳定。
6.  **互动分析**：对于阅读量都很高的大号，**点赞量**（或“在看”数）是比阅读量更关键的互动质量指标，更能反映用户对内容的认可程度。
