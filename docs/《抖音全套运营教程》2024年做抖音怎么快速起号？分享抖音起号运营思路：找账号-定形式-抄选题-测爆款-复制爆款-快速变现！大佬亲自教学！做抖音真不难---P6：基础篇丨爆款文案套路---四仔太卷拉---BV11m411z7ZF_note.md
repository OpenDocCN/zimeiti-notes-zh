# 《抖音全套运营教程》：P6：基础篇丨爆款文案套路 📝

![](img/7d07dc7a19ce475d54173a3ad7443523_0.png)

在本节课中，我们将学习一个适用于各行各业的爆款文案创作套路。掌握这个套路，能帮助你快速创作出吸引用户停留、观看并互动的高质量短视频文案。

---

上一节我们介绍了文案的核心在于“与用户有关”。本节中，我们来看看如何通过一个具体的公式来构建这种关联性。

![](img/7d07dc7a19ce475d54173a3ad7443523_2.png)

这个万能公式是：**现象 + 危害 + 原因 + 解决办法**。

以下是该公式每个部分的具体作用：

*   **现象**：描述一个普遍存在的问题或场景，用于筛选目标用户，让他们感觉“这说的就是我”。
*   **危害**：阐述该问题带来的负面影响或痛苦，用于放大用户的情绪，防止他们在5秒内划走视频。
*   **原因**：分析问题产生的根源，引导用户继续观看以寻求答案，有助于提高视频的完播率。
*   **解决办法**：给出具体的解决方案或建议，满足用户需求，并鼓励他们点赞、收藏以提升互动率。

---

为了帮助你更好地理解，我们来看两个应用此公式的例子。

**例一：服装行业**
> 梨形身材的女孩太难买裤子了（**现象**）。挑错了样式显得腿又粗又短（**危害**）。但梨形身材是天生的，很难改变（**原因**）。而这三种裤子不仅显瘦，还显腿长，穿上第二天就被小哥哥搭讪了，姐妹们赶快收藏起来吧（**解决办法**）。

这段文案开头锁定目标客户，随后层层递进，引导用户看完并产生互动。

**例二：运营行业**
> 每天用心制作短视频，但发完只有100播放（**现象**）。继续下去只是浪费时间，徒增焦虑（**危害**）。这跟你没有系统的运营底层逻辑有关（**原因**）。现在去看完我的前三条视频，不花一分钱也能学会运营知识，还不快快行动起来（**解决办法**）。

![](img/7d07dc7a19ce475d54173a3ad7443523_4.png)

---

通过以上两个例子，我们可以看到，无论什么行业，套用这个公式都能快速生成有效的文案。这类文案通常简短（不足30秒），易于批量生产和矩阵化运营，能极大提升内容创作效率。
