# 新媒体运营：P12：01.10.0：ChatGPT 4.0的几种使用渠道 🚀

在本节课中，我们将学习如何获取和使用目前功能强大的ChatGPT 4.0模型。对于新媒体运营人员而言，掌握AI工具能极大提升工作效率，例如撰写脚本、制作PPT和生成报告。我们将介绍几种体验ChatGPT 4.0的渠道，包括免费和付费选项，并分析各自的优缺点。

## 优先推荐：微软Edge浏览器Copilot 💻

上一节我们介绍了AI对运营工作的重要性，本节中我们来看看最便捷的体验渠道之一：微软Edge浏览器的Copilot功能。它是微软基于ChatGPT 4.0模型打造的AI助手，集成在Windows系统中，智能程度与4.0模型相匹配。

![](img/bd4d1dd05ab0d650e700343bee33d499_1.png)

![](img/bd4d1dd05ab0d650e700343bee33d499_3.png)

使用Copilot有两个前提条件：
1.  登录微软账号（可使用Outlook邮箱注册）。
2.  需要准备“梯子”（VPN）以访问服务。

Copilot提供三种对话模式：
*   **更有创造力**：AI的回答更具开放性和创意。
*   **平衡**：介于创造力和精确性之间，适合一般用途。
*   **更精确**：回答更为严谨和准确，适合专业内容。

例如，我们可以使用“更有创造力”模式来生成一个短视频脚本。输入指令：
```
请帮我写一个关于装修的短视频脚本，时长控制在3分钟左右。
```
AI会快速生成包含场景、画面和台词的结构化脚本。此外，Copilot还能基于你当前浏览的网页内容进行分析，辅助办公和系统设置。

然而，Copilot也存在一些缺点：
1.  **回答精准度**：相较于官方ChatGPT 4.0，经过微软调教的Copilot在某些回答上可能不够精准。
2.  **连续对话能力受限**：在当前对话中持续提问的次数有限，可能需要开启新对话窗口。
3.  **记忆功能有所缺失**：在读取和分析历史对话记录方面不如官方版本强大。

如果你的Edge浏览器没有Copilot图标，可以尝试下载最新的Edge Dev（测试版）浏览器。在设置中检查更新或访问微软官网的Dev频道下载即可。

## 第三方平台与共享账号 🌐

除了官方集成工具，还有许多第三方平台通过调用ChatGPT的API，为国内用户提供了访问渠道。以下是几个可用的平台推荐。

### 平台一：灵感AI

这是一个集成了多种AI模型的平台，包括GPT-3.5、GPT-4.0以及国内主流模型。

该平台的优势如下：
*   **模型丰富**：集成GPT-4.0、GPT-4.0联网版、讯飞星火、文心一言等。
*   **免费额度**：新注册用户可在7天内免费使用GPT-4.0一定次数。
*   **无需梯子**：在国内网络环境下可直接访问。

其缺点是响应速度可能较慢。例如，生成一篇旅游攻略可能需要等待几分钟。

### 平台二：Vee AI

这个平台也提供GPT-3.5和GPT-4.0模型，速度较快。

其特点如下：
*   **响应速度快**：接近官方ChatGPT的响应水准。
*   **附带功能**：支持语音回答和AI绘画。
*   **免费额度**：初始赠送1万token（计算单位），但通常仅适用于GPT-3.5模型。

需要注意的是，使用该平台需要梯子，且免费额度有限。

### 平台三：ChatGPT共享账号平台

这是一个由网友共享ChatGPT账号的网站，可以找到免费的官方账号。

使用方式如下：
1.  访问共享网站，会看到许多账号列表。
2.  账号状态以颜色标识：**绿色**（空闲）、**橙色**（近期使用过）、**红色**（正在使用或使用频繁）。
3.  带有“金光”标识的账号为付费的Plus会员账号，可使用GPT-4.0。

![](img/bd4d1dd05ab0d650e700343bee33d499_7.png)

![](img/bd4d1dd05ab0d650e700343bee33d499_9.png)

你可以尝试点击绿色的免费账号（通常是GPT-3.5）登录使用。付费的GPT-4.0账号由于使用人数多，可能遇到“对话次数达到上限”的提示（官方限制每3小时约30条消息）。这是一个应急使用官方服务的渠道，同样需要梯子。

## 总结与建议 📝

本节课中我们一起学习了体验和使用ChatGPT 4.0的多种渠道。

我们来总结一下关键点：
*   **追求最佳体验**：如果条件允许，使用**官方ChatGPT 4.0**（通过共享账号或自行订阅）能获得最强大的智能辅助。
*   **追求便捷与集成**：Windows用户可使用**Edge浏览器Copilot**，它方便快捷，能满足大部分日常需求。
*   **尝试第三方平台**：**灵感AI**、**Vee AI**等平台提供了折中的选择，部分支持免费使用GPT-4.0，但可能有速度或额度限制。
*   **无梯子需求**：国内出品的**文心一言**、**豆包AI**、**通义千问**等是稳定可靠的替代方案，适合基础运营工作。
