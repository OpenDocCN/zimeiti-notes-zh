# 亚马逊跨境电商入门到精通：P19：7.7、采购需要在亚马逊上面销售的商品

![](img/4d068b8114e42164ade664ab8be24a7a_1.png)

## 概述
在本节课中，我们将学习在亚马逊平台销售商品时必须遵守的核心采购准则。了解并遵循这些政策，是确保店铺合规运营、避免侵权风险、保障销售权益的基础。

![](img/4d068b8114e42164ade664ab8be24a7a_2.png)

---

在考虑于亚马逊销售何种商品时，了解亚马逊的准则至关重要。作为亚马逊卖家，您可以销售来自第三方品牌的商品或您自有品牌的商品。如果您选择成为亚马逊卖家，了解平台对上架商品的期望和政策非常重要。

亚马逊尊重他人的知识产权。作为卖家，您有责任确保您在亚马逊上销售的商品遵守所有适用的法律法规，已获得销售或转售授权，并且不会侵犯版权、专利、商标和宣传权等知识产权。

上一节我们介绍了遵守平台政策的重要性，本节中我们来看看具体的采购准则。以下是您在亚马逊上架销售商品时需要重点考虑的五条重要准则。

### 一、确认您的供应商
作为卖家，您有责任采购并销售正品。如果您要销售品牌商品，直接从品牌商或其直接分销商、批发商处采购商品始终是最佳做法。

如果您通过分销商或批发商购买品牌商品，您需要确认他们与该品牌的关系。以下是验证供应商可靠性的信息来源：
*   他们的营业执照。
*   品牌授权书。
*   办公和仓库地点。
*   商务局评级。
*   可以证明供应商信誉的其他信息来源，例如他们在网上和社交媒体上的形象、供应特定品牌或商品的时长、新闻文章等。

查阅公司网站上的“关于我们”部分也是一个不错的主意，这样可以了解他们的发展历史、当前业绩以及未来计划。

### 二、与品牌商建立关系
尽管您可能能够从值得信赖的供应商处购买商品，但对于某些商品，您可能仍然需要获得品牌商的明确许可才能在亚马逊上销售。因此，与品牌商建立关系非常重要，这样可以确保您符合销售这些品牌商品的所有品牌要求和亚马逊政策。

### 三、保留交易发票
亚马逊可能要求您提供发票等单据，以证明您的商品是正品或您拥有销售授权。采购打算在亚马逊上销售的商品时，一定要保留所有发票。

发票有助于证实您从有效的供应商处购买了相关产品。发票应包含关键信息，如供应商的名称、地址和详细联系信息。在将这些单据提供给亚马逊时，您可以删除价格信息，但单据的其余信息必须清晰可见。该单据应逐项列出您所购买商品的介绍和数量。

对于打算以“新品”状态销售的商品，单据开具日期还应在过去的365天内。

### 四、了解亚马逊的直送政策
亚马逊允许卖家根据直送政策配送商品。您可以在卖家平台的搜索栏中输入 **`直送政策`** 或 **`drop shipping policy`** 来找到该政策。

要正确实施直送，卖家必须首先确认直送方是他们所要销售商品的有效供应商。在确定供应商有效后，一定要签署标有日期的服务级别协议，在您与直送供应商之间正式建立关系。

要遵守亚马逊的直送政策，您必须：
*   是商品的登记卖家。
*   在所有装箱单、发票、外包装以及其他相关信息中注明自己是商品卖家。
*   在配送订单前，删除所有标识第三方直送方的装箱单、发票、外包装或其他信息。
*   负责接受和处理所售商品的客户退货。

需要指出的是，亚马逊严禁零售商间的直送。

### 五、注意无效货源
拍卖或拍卖式环境不能算作正当的货源。严禁在亚马逊上转售打算销毁、处置的商品，或者被制造商、提供商、供货商或零售商规定不得销售的商品。

## 如果您是品牌所有者
如果您拥有自己的商标，您可以作为品牌所有者销售商品。注册商品的品牌所有者可以充分利用亚马逊品牌注册计划所提供的所有权益、保护和计划。亚马逊上也允许销售自有商品、手工或定制商品。

您有责任确保您在亚马逊上销售的商品遵守所有适用的法律法规，已获得销售或转售授权，并且不会侵犯版权、专利、商标和宣传权等知识产权。保留单据很重要，这样可以证实您与自己的商品的关系。

原材料或成品的发票就是一个例子。它们应包含供应商的名称、地址和联系信息，以及您的公司名称、地址和联系信息。发票应逐项列出所购买商品的介绍和数量。在将这些单据提供给亚马逊时，您可以删除价格信息，但单据的其余信息必须清晰可见。

## 总结
本节课中，我们一起学习了在亚马逊进行商品采购时必须遵循的五项核心准则：**确认供应商资质**、**与品牌商建立关系**、**妥善保留交易发票**、**理解并遵守直送政策**，以及**规避无效货源**。无论您是销售第三方品牌商品还是自有品牌商品，遵守这些准则都是保障店铺长期、稳定、合规运营的基石。请务必在采购和上架前仔细核查，确保每一步都符合亚马逊平台的要求。
