**收款账号的作用**类似于支付宝。当顾客在您的亚马逊店铺下单付款后，货款会先存放在这个第三方平台中。之后，您可以将款项提现到绑定的个人银行卡里。

对于个人店铺，我们建议使用**PingPong**，它是最常用且易于操作的收款工具。

## 第二步：开始注册PingPong账户
理解了收款账号的重要性后，我们现在开始注册流程。

以下是注册PingPong账户的步骤：
1.  **访问注册链接**：从老师提供的流程图或资料中找到PingPong的注册网址并访问。
2.  **选择账户类型**：在注册页面，个人店铺请选择“个人账号”，企业店铺可选择“企业账号”。
3.  **填写注册信息**：根据提示填写信息，包括：
    *   地区：中国内地
    *   手机号及验证码
    *   登录密码
    *   （可选）客户经理信息可不填
4.  **创建账户**：点击“立即创建”按钮。

![](img/272d830fee79db3ad45809efdfafec0b_23.png)

> **注意**：北美站和日本站可以使用法人个人的收款账号，但欧洲站通常要求使用企业对公账号。

## 第三步：完善账户资料与实名认证
账户创建成功后，需要进一步完善资料并通过实名认证。

![](img/272d830fee79db3ad45809efdfafec0b_27.png)

![](img/272d830fee79db3ad45809efdfafec0b_28.png)

![](img/272d830fee79db3ad45809efdfafec0b_30.png)

![](img/272d830fee79db3ad45809efdfafec0b_32.png)

以下是完善资料与认证的步骤：
1.  **完善邮箱信息**：在账户后台，将准备好的163邮箱绑定到账户。
2.  **绑定目标平台**：在“平台管理”或类似选项中，选择您要运营的站点（如北美站），并填写店铺名称。
    *   **店铺名称**：建议使用姓名拼音，若提示不可用，可灵活添加数字或字母后缀（例如：`zhangsan_2024`）。**请务必记下这个名称**，后续注册店铺需保持一致。
3.  **完成实名认证**：在账户中心找到“实名认证”入口。
    *   按要求上传**身份证正反面照片**及**手持身份证照片**。
    *   确保照片清晰，手持照需聚焦于身份证信息。
    *   填写身份证上的常住地址。
4.  **等待审核**：提交后，系统会进行审核，通常会在一天内完成。请耐心等待审核通过。

## 第四步：获取收款账户信息
实名认证通过后，您就可以获取注册亚马逊店铺所需的关键信息了。

![](img/272d830fee79db3ad45809efdfafec0b_41.png)

登录PingPong后台，进入“平台管理”或“店铺详情”页面。您会看到两个关键信息：
*   **9位数收款码**：一串9位数字的识别码。
*   **银行账户信息**：系统自动生成的一串虚拟银行账户信息（包括账号、路由号等）。

![](img/272d830fee79db3ad45809efdfafec0b_43.png)

![](img/272d830fee79db3ad45809efdfafec0b_44.png)

**公式/代码示例**：
```
店铺注册所需信息：
收款码 = PingPong后台获取的9位数字
银行账户 = PingPong后台生成的虚拟账户信息
```
