上一节我们介绍了小红书变现的基础，本节中我们来看看具体的商单形式。商单主要分为两种：直发订单和原创订单。

### 直发订单

![](img/7c56bbdee46fab9483f414925b8171de_9.png)

![](img/7c56bbdee46fab9483f414925b8171de_10.png)

![](img/7c56bbdee46fab9483f414925b8171de_11.png)

直发订单是指商家提供完整的文案和图片，创作者只需负责发布。这种订单操作简单，但报价较低。

以下是直发订单的特点：
*   **无需创作**：商家提供所有素材。
*   **流程简单**：创作者仅执行发布操作。
*   **报价低廉**：单条报价通常在 **100元至150元** 人民币。

由于门槛低、竞争激烈，仅依靠直发订单的创作者缺乏市场竞争力。

### 原创订单

原创订单要求创作者自行撰写文案或准备图片。这种订单对创作者能力有要求，因此报价更高。

以下是原创订单的特点：
*   **需要创作**：创作者需撰写文案或制作图片。
*   **价值更高**：体现了创作者的原创能力。
*   **报价可观**：以1000粉丝的账号为例，原创商单的均价约为 **300元** 一条。随着粉丝量增长，报价会进一步提升。

![](img/7c56bbdee46fab9483f414925b8171de_13.png)

平台认证的“原创达人”身份能显著提高接单价格和机会。

## 原创能力决定收入天花板

![](img/7c56bbdee46fab9483f414925b8171de_15.png)

![](img/7c56bbdee46fab9483f414925b8171de_16.png)

我们通过两个案例来直观对比原创能力带来的收入差异。

**案例一：缺乏原创能力的账号**
*   **粉丝数**：6898
*   **接单类型**：直发
*   **单价**：150元
*   **近60天接单量**：4条
*   **估算收入**：`150元/条 * 4条 = 600元`

![](img/7c56bbdee46fab9483f414925b8171de_18.png)

**案例二：具备原创能力的账号**
*   **粉丝数**：5177
*   **接单类型**：原创
*   **单价**：700元
*   **近30天接单量**：37条
*   **估算收入**：`700元/条 * 37条 ≈ 25,900元`

对比可见，原创能力是提升商单收入的核心。

![](img/7c56bbdee46fab9483f414925b8171de_20.png)

## 如何利用AI工具撰写原创文案

![](img/7c56bbdee46fab9483f414925b8171de_22.png)

对于不熟悉的产品，可以利用AI工具辅助生成高质量的种草文案。以下是操作步骤：

1.  **打开AI工具**，选择“种草文案”生成功能。
2.  **输入产品信息**：选择品类（如“美容护肤”），填写产品名称（如“XX美容仪”），并输入商家提供的产品核心卖点。
3.  **选择文案语气**：根据需求选择（如“友好”）。
4.  **生成文案**：点击生成，AI会在几秒内输出一篇包含开头、卖点罗列和结尾的完整文案。
5.  **优化与保存**：如果对初稿不满意，可以点击“换一换”生成不同版本的文案。将满意的文案保存到文档中备用。

**示例代码（模拟AI生成文案的逻辑）**：
```python
# 输入：产品品类、产品名称、核心卖点列表
product_category = "美容护肤"
product_name = "XX美容仪"
selling_points = ["提拉紧致", "深层清洁", "美白亮肤", "导入导出技术"]

# AI模型基于输入生成种草文案
generated_copywriting = ai_model.generate(product_category, product_name, selling_points)
print(generated_copywriting)
```

![](img/7c56bbdee46fab9483f414925b8171de_24.png)

将生成的文案稍作修改后发给商家审核，可以极大提升创作效率和文案质量。

## 接单与运营策略

![](img/7c56bbdee46fab9483f414925b8171de_26.png)

![](img/7c56bbdee46fab9483f414925b8171de_27.png)

掌握了创作方法后，我们来看看接单和账号运营的具体策略。

![](img/7c56bbdee46fab9483f414925b8171de_29.png)

**接单沟通技巧**：
提交AI生成的文案时，应间隔一段时间（如1-1.5小时），营造出亲自创作的感觉。如果商家要求修改，可以再次利用工具快速生成新版本。

![](img/7c56bbdee46fab9483f414925b8171de_30.png)

![](img/7c56bbdee46fab9483f414925b8171de_31.png)

**账号运营目标**：
接商单的主要门槛是粉丝数达到 **1000** 且账号状态正常。初期账号内容不必局限单一垂直领域，可以发布影视解说、情感语录、知识科普等多种内容来吸引流量、快速涨粉。

**资源与合作模式**：
通常，平台（如蒲公英平台）会从商单总额中抽取一部分服务费。如果订单通过合作方（如MCN机构）引入，合作方会从平台服务费中获得分成，而不会影响创作者的实际收入。这是一种共赢的合作模式。

**进阶增收技巧**：
一个高效的增收策略是运营多个账号。用一个账号与商家合作成功后，可以换用其他账号再次联系同一商家，提供不同风格的原创内容进行二次合作，从而将资源价值最大化。

![](img/7c56bbdee46fab9483f414925b8171de_33.png)
