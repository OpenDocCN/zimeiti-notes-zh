# 剪映教程：P17：5.剪辑的两种方法 🎬

在本节课中，我们将要学习视频剪辑的两种核心方法：粗剪与精剪。掌握这两种方法，是完成一个视频作品的基础。但在此之前，充分的准备工作同样至关重要。

## 剪辑前的准备工作 📋

![](img/b65a5a9bb0b0110c5baebb00ad820dd8_1.png)

在开始动手剪辑之前，做好充分的准备能让整个流程事半功倍。以下是三个关键的准备工作步骤。

### 1. 做好选题
尽量选择一些有趣的、能调动观众情绪的内容作为视频主题。

### 2. 准备好脚本
脚本就像写作文一样，它能让你的内容有清晰的结构。一个完整的脚本应包含开头、发展、高潮、转折、结局等环节。简单来说，尽量让你的故事有头有尾，过程完整，并带有一点曲折。

### 3. 设计分镜头
你至少需要在脑子里对视频镜头如何拍摄有一个大概的印象。景别应尽量涵盖近景、中景和远景。拍摄同一动作时，使用组合镜头（多个不同角度的短镜头）往往比单一的长镜头看起来更专业、更有吸引力。

按照以上三步进行准备，再开始拍摄，优点非常明显：拍摄效率高，且不会在剪辑时才发现漏掉了某些必要的镜头。此外，拍摄时请尽量保证画面平稳，如果使用手机稳定器效果会更好。

![](img/b65a5a9bb0b0110c5baebb00ad820dd8_3.png)

做好以上准备工作，你就能为剪辑出一个出色的视频打下坚实的基础。

## 第一种剪辑方法：粗剪 ✂️

![](img/b65a5a9bb0b0110c5baebb00ad820dd8_5.png)

上一节我们介绍了剪辑前的准备工作，本节中我们来看看第一种剪辑方法——粗剪。粗剪的目标是快速搭建视频的骨架和顺序，它主要依赖于四个基础操作。

以下是粗剪所需的四个核心操作：

1.  **拖动**：选中素材后，拖动其开头或结尾的白色边框，可以调整素材的入点和出点，从而删除或恢复头尾部分。
2.  **分割**：将时间线指针移动到想要切割的位置，点击“分割”工具，可以将一段素材切成两段。
3.  **删除**：选中不需要的片段，点击“删除”按钮即可将其移除。
4.  **排序**：长按某个素材片段，当其变成可拖动的小方块后，即可将其移动到时间线上的其他位置，调整片段的先后顺序。

**操作流程示例**：
点击“开始创作”，选择素材并“添加到项目”。若要删除素材中间部分，可以先用“分割”工具在不需要部分的前后各切一刀，将其独立为一个片段，然后选中该片段并“删除”。最后，通过“排序”功能，将所有素材片段按照脚本的顺序排列好。这样，视频的粗剪工作就完成了。

## 第二种剪辑方法：精剪 🔍

完成了视频的粗剪，搭建好整体框架后，本节我们来学习如何对视频进行精细修剪，这就是精剪。精剪关注的是细节，比如卡准节奏、去除冗余帧等。

在剪映中，放大时间线可以更精确地看到每一帧画面，从而进行精细剪辑。剪映支持的最高剪辑精度为**4帧**。这意味着，任何等于或长于4帧的片段都可以被分割；而短于4帧的片段则无法再进行分割。不过，这个精度已能满足绝大多数日常剪辑的需求。

值得注意的是，通过拖动片段的**首尾**进行剪辑，可以实现**逐帧级别**的精细调整，让你能精准控制每个镜头的长度。

通过结合**视频粗剪**（搭建结构）和**视频精剪**（优化细节），你已经能够独立完成一个视频的完整剪辑工作了。

## 总结 📝

![](img/b65a5a9bb0b0110c5baebb00ad820dd8_7.png)

本节课中我们一起学习了视频剪辑的完整流程。首先，我们强调了**选题、脚本、分镜头**这三项剪辑前准备工作的重要性。接着，我们深入探讨了两种核心剪辑方法：使用**拖动、分割、删除、排序**进行快速框架搭建的**粗剪**，以及通过**放大时间线、逐帧修剪**来实现细节优化的**精剪**。掌握这些基础步骤和方法，你就能迈出视频剪辑的第一步。
