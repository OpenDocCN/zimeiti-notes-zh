1.  **角色**
    角色这个东西，说白了你做什么赛道，准备往什么方向输出内容，你就需要让别人知道你提供的内容是有一定专业性的。例如：
    *   **装修博主**：你需要告诉别人你确实有这么一个户型，有这么一个房子，正在或已经完成装修，并记录和分享整个过程。
    *   **母婴赛道**：你可以是“宝妈”，也可以是“育儿护理专家”的身份。
    *   **情感赛道**：你可以是“心理学顾问”或“情感大师”，也可以是分享自身经历的“成长型人设”（例如：分享30岁女性如何处理家庭、职场问题的博主）。
    **核心公式**：`人设角色 = 赛道方向 + 专家身份`

2.  **内容**
    内容这一点很简单。我们讲内容定位时说过，你的内容要参照几个点：是否吸引用户、是否有价值、是否有平台特色。只要满足了这些点，基本上没有大问题。在人设定位这一块，关键是你做某个赛道，就一定要讲跟这个赛道相关的内容，并持续下去。
    这涉及到账号的垂直度和标签问题。我们做新媒体一定要做垂直领域，主要方向就是打上账号标签，因为这影响账号成长和后期流量分发。如果你的内容方向杂乱，平台就无法准确为你匹配受众，从而影响推流效果。
    **核心原则**：`内容垂直 = 持续输出同一赛道的高价值内容`

3.  **形式**
    形式指的是内容的具体呈现方式。主要包括以下几点：
    *   **产出场景**：是在固定的室内场景，还是在多变的室外场景（如上班路上、车内）？
    *   **声音呈现**：是自己出镜讲解，还是不出镜仅使用配音？
    *   **内容类型**：是以“剧情”形式，还是以“干货分享”、“生活记录”的形式来提供内容？每种形式各有优劣势，例如干货型需要丰富的文案脚本，生活记录型则需要大量的拍摄素材。
    确定形式时，可以参考对标账号的做法，并结合自身条件（如是否擅长表达）来决定。

4.  **记忆点**
    记忆点是最重要的一点，它决定了用户如何记住你。记忆点是通过固定设计的环节来实现的，例如：
    *   **视频前几秒的固定动作、特效、开场白**（需前置，以强化冲击力）。
    *   **统一的BGM**（贯穿视频全程，形成听觉标识）。
    *   **统一出现的物品或服装风格**（贯穿内容全局，形成视觉标识）。
    这些记忆点需要反复刻入用户的心里，让他们在刷到的瞬间就能联想到你。设计记忆点时，应将具有强冲击力的元素（如动作、开场白）放在视频开头，将贯穿性元素（如BGM、视觉风格）融入全程。
