# 手机剪辑教程：P38：16. 人物单独显色

## 概述
在本节课中，我们将学习如何制作“人物单独显色”的视频效果。这种效果能让画面中的人物保持彩色，而背景变为黑白或其他单色，从而突出主体。我们将使用手机剪辑软件，通过复制视频、应用滤镜和蒙版等步骤来实现。

---

上一节我们介绍了视频剪辑的基础操作，本节中我们来看看如何实现高级的人物突出效果。

### 第一步：导入并复制视频
首先，你需要将一段包含人物的视频导入到剪辑软件的时间轴（主轨道）中。

接着，选中这段视频，点击“复制”功能。然后，点击“切画中画”选项，将复制出来的视频片段移动到画中画轨道。

**关键操作代码示例（概念性）**：
```
1. 导入视频 -> 主轨道
2. 选中主轨道视频 -> 点击“复制”
3. 点击“切画中画”
```

### 第二步：为主视频添加滤镜
现在，我们选中主轨道上的原始视频。为其添加一个滤镜，目的是改变背景的色调。

在滤镜库中，选择“牛皮纸”滤镜（或其他能产生单色/黑白效果的滤镜）。添加后，整个主视频画面将变为该滤镜效果。

### 第三步：对齐画中画视频
将之前切到画中画轨道的视频片段拖动，使其起始位置与主轨道视频的开头完全对齐。这是为了确保两层画面在时间上同步。

### 第四步：处理画中画图层
接下来，我们需要对画中画图层进行两项关键处理：抠像和蒙版。

首先，选中画中画图层。

以下是需要完成的两个核心步骤：

1.  **智能抠像**：点击“智能抠像”功能。软件会自动识别并抠出画面中的人物主体。完成后，画中画图层将只显示被抠出的人物。
2.  **应用圆形蒙版**：在画中画图层被选中的状态下，找到并点击“蒙版”工具。选择“圆形蒙版”。屏幕上会出现一个圆形蒙版区域，你可以直接拖动它，将其移动到人物脸部或上半身等需要突出显示的位置。最后，调整蒙版的大小和羽化值，使其与人物的轮廓匹配自然。

**核心概念公式**：
```
最终效果 = 主轨道(背景滤镜) + 画中画(抠像人物 + 圆形蒙版局部显示)
```

### 第五步：预览并添加音乐
所有调整完成后，你可以预览视频效果，检查人物显色是否准确、过渡是否自然。

最后，为视频添加合适的背景音乐，完成整个制作。

---

## 总结
本节课中我们一起学习了制作“人物单独显色”效果的全过程。我们通过**复制视频**、为底层添加**滤镜**、对上层进行**智能抠像**并应用**圆形蒙版**，实现了突出彩色人物、压暗背景的视觉对比效果。这个技巧是人物剪辑中的常用方法，希望你多加练习，灵活运用。
