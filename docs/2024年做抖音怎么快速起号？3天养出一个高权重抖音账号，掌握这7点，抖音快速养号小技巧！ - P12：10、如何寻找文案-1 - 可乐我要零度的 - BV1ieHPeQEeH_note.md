# 抖音运营实战：P12：10、如何寻找文案-1

## 概述

在本节课中，我们将要学习如何为抖音视频寻找和准备文案。对于初学者而言，文案是视频内容的核心，好的文案能有效提升视频吸引力，帮助账号吸引并留住粉丝。我们将介绍几个实用的文案来源网站，并讲解如何使用这些工具来生成文案和配音，从而解决“不会写”和“声音不好听”两大难题。

解决了文案来源，我们再来解决另一个常见问题：有的同学担心自己的声音不好听。

点击网站右上角的“小工具”，然后选择第四项“配音在线生成”。

![](img/4536695f8606f81623a56de98b441018_11.png)

![](img/4536695f8606f81623a56de98b441018_15.png)

这个功能完全免费，且提供丰富的音色选择。

![](img/4536695f8606f81623a56de98b441018_17.png)

![](img/4536695f8606f81623a56de98b441018_13.png)

![](img/4536695f8606f81623a56de98b441018_19.png)

以下是配音制作功能的使用展示。你可以看到这里有非常多的音色选项：亲和女生、温柔女生、甜美女生、自然女声、卖场广播等。继续往下翻，还有直播配音、标准男声、标准女声、方言（如湖南话、四川话、粤语、东北话、台湾话）、外语配音，以及精品男配如情感解说、影视解说、娱乐解说等。种类非常多，你可以自行探索。

![](img/4536695f8606f81623a56de98b441018_15.png)

具体操作步骤如下：
1.  在左侧“配音文本”框中输入或粘贴你准备好的文案。
    ![](img/4536695f8606f81623a56de98b441018_17.png)
2.  在右侧选择你喜欢的音色，例如选择“方言”分类下的“大虎（东北话）”。
    ![](img/4536695f8606f81623a56de98b441018_19.png)
3.  调整参数：你可以调节音量大小、语速快慢以及语调高低。
    ![](img/4536695f8606f81623a56de98b441018_21.png)
4.  调整完毕后，点击“立即生成”。
    ![](img/4536695f8606f81623a56de98b441018_23.png)

如果你在手机上使用，可以截图然后用微信扫描页面二维码登录，即可免费使用配音功能，非常方便。

![](img/4536695f8606f81623a56de98b441018_21.png)

![](img/4536695f8606f81623a56de98b441018_25.png)

生成后，在“配音生成列表”中，找到你的作品，点击右下角的下载按钮，即可将配音文件保存到手机。
