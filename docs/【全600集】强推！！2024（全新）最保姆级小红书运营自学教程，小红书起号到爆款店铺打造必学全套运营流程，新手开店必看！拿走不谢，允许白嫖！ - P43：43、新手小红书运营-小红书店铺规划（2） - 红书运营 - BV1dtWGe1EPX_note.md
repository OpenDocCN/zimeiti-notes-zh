在本节课中，我们将学习小红书店铺运营的中期与后期规划。我们将了解中期需要关注的各项排名指标，以及后期发展的两个主要方向。

上一节我们介绍了店铺规划的前期准备，本节中我们来看看店铺运营进入中期后，应该如何进行规划与操作。中期规划的重点在于提升店铺在平台内的各项排名，以获得更多的基础曝光。

![](img/4077583ea8435651e7dcd87e25d8f3a2_11.png)

![](img/4077583ea8435651e7dcd87e25d8f3a2_13.png)

中期规划涉及多个排名系统，其核心目标是争夺平台约20%的基础流量。以下是中期需要关注的核心排名分类：

![](img/4077583ea8435651e7dcd87e25d8f3a2_15.png)

*   **综合排名分类**：这是默认的商品排序方式，平台根据店铺的综合表现对商品进行排名。其考核维度多达16项，主要包括：
    *   商品描述与主图质量
    *   单品综合数据（如转化率、收藏率）
    *   店铺整体交易情况（如销量、信誉、好评率）
    *   新店与新品扶持权重
    *   标题关键词优化等
    公式可以理解为：**综合排名 = f(商品质量，店铺数据，用户反馈，平台权重)**。在这16项中的排名越高，能获取的基础流量就越多。

![](img/4077583ea8435651e7dcd87e25d8f3a2_17.png)

![](img/4077583ea8435651e7dcd87e25d8f3a2_19.png)

*   **价格排名**：根据商品价格进行排序。排名不仅看价格高低，还会考量：
    *   价格的竞争力与真实性
    *   价格的稳定性
    *   价格与销量之间的平衡关系
    *   商品与搜索关键词、类目的匹配度

![](img/4077583ea8435651e7dcd87e25d8f3a2_21.png)

*   **销量排名**：根据商品的销售数量排序。高销量通常意味着高人气，但排名也受类目相关性和商品评价等因素影响。

![](img/4077583ea8435651e7dcd87e25d8f3a2_23.png)

![](img/4077583ea8435651e7dcd87e25d8f3a2_25.png)

*   **关键词排名**：针对用户搜索的特定关键词进行排序。例如，“T恤”、“男士T恤”、“夏季女士T恤”都是不同的关键词赛道。优化特定关键词的排名，可以获取该搜索词下的精准流量。

![](img/4077583ea8435651e7dcd87e25d8f3a2_27.png)

![](img/4077583ea8435651e7dcd87e25d8f3a2_29.png)

![](img/4077583ea8435651e7dcd87e25d8f3a2_31.png)

*   **个性化搜索**：平台根据用户的历史行为（浏览、点赞、购买等）和店铺标签，进行个性化商品推荐。优化此项意味着能吸引更精准的用户群体。

由于中期规划的每个排名系统都包含大量细节，本节课旨在让大家建立整体认知。若对其中某一部分特别感兴趣，建议搜索专项教程进行深入学习。

了解中期规划后，我们来看后期发展。当店铺度过中期，进入稳定或瓶颈阶段时，主要有两个发展方向：

![](img/4077583ea8435651e7dcd87e25d8f3a2_33.png)

1.  **加大力度投入**：如果判断店铺仍有增长空间，可以在此阶段进行战略性投入，突破瓶颈。这通常包括：
    *   更深入的数据分析与店铺诊断
    *   开展付费推广（如信息流广告）
    *   系统性的店铺优化与数据维护
    目标是推动店铺向更高层级发展。

2.  **开设新店**：如果当前店铺已达到品类或投入的上限，可以选择开设新店。将现有店铺成功的运营模式、选品思路和规划经验复制到新店，并进行小范围调整优化。利用已有经验，新店的启动和发展效率通常会更高。

![](img/4077583ea8435651e7dcd87e25d8f3a2_35.png)

本节课中我们一起学习了小红书店铺的中期与后期规划。中期规划的核心是理解并优化**综合排名**、**价格排名**、**销量排名**、**关键词排名**和**个性化搜索**这五大系统，以提升基础流量获取能力。后期规划则提供了 **“深化运营”** 与 **“复制扩张”** 两条发展路径。这些内容构成了店铺从成长到成熟的完整运营思路。
