## 用户标签：平台如何“读懂”你

用户标签是平台为每一位用户建立的兴趣画像。平台通过分析用户的行为数据，来判断用户喜欢什么内容。

以下是平台判断用户兴趣的三个核心数据维度：

1.  **点击率**：用户点击进入视频的概率。
2.  **停留时长/完播率**：用户观看一个视频的时长或是否完整看完。
3.  **互动行为**：包括点赞、收藏、评论和**关注**。

![](img/376e7e13b697f3d5736243662fd88389_5.png)

平台会持续追踪这些数据。例如，如果一个用户被推送了5条美食视频、5条美女视频和5条科技视频，而他点击了全部5条美食视频、2条美女视频和1条科技视频。平台通过算法计算后，可能会得出类似以下的兴趣权重：

```
用户兴趣权重 = {
    “美食”: 5,
    “美女”: 2,
    “科技”: 1
}
```

![](img/376e7e13b697f3d5736243662fd88389_7.png)

基于这个“画像”，平台会认为该用户对美食内容最感兴趣，从而在后续推送中增加美食类内容的比例，以延长用户在平台的停留时间，最终实现商业转化（如直播带货、商品推荐）。反之，如果用户对某类内容（如母婴）点击率为零，平台就会减少或停止推送此类内容。

这就是“信息茧房”或“个性化推荐”的底层逻辑：平台致力于将用户最可能感兴趣的内容推送给他，以提升用户粘性。

## 创作者标签：如何成为领域的“专家”

创作者标签决定了平台将你的内容推荐给哪一类用户。很多人误以为只要多发某一领域的视频，平台就会自动识别。实际上，平台是通过分析你视频吸引来的用户**行为数据**，来反向定义你的账号标签。

假设有两个博主：
*   **博主A**：内容混杂，既发母婴、又发女装、还发生活打卡。
*   **博主B**：内容专注，只发母婴相关。

当博主A发布一条女装视频时，平台会根据其历史数据，主要将视频推送给曾与其母婴内容互动过的用户，同时尝试匹配部分女装兴趣用户。结果很可能是：大部分母婴用户对女装不感兴趣，直接划走，导致该视频的**点击率、完播率、互动数据**变差。平台会因此判定博主A的创作质量不稳定。

![](img/376e7e13b697f3d5736243662fd88389_13.png)

![](img/376e7e13b697f3d5736243662fd88389_15.png)

**平台的核心能力是基于兴趣标签进行精准推送。** 它需要将用户交给每一个领域的“专家”，而不是“杂家”。如果你的账号标签混乱，平台就无法确认你的“专家”身份，从而难以进行稳定和精准的流量推荐。

此外，**关注**这个行为需要单独强调。用户关注一个账号，意味着他认为这个账号能提供**长期、稳定的价值**。一个标签混乱的账号无法给予用户这种确定性预期，从而影响粉丝的积累和忠诚度。

## 核心总结与下节预告

本节课中我们一起学习了：
1.  **用户标签**是平台通过用户的**点击率、完播率、互动行为**数据生成的兴趣画像，用于实现个性化推荐。
2.  **创作者标签**是平台通过分析你内容吸引来的用户**行为数据**，反向定义你的账号领域。保持内容垂直，是为了让平台将你识别为单一领域的“专家”，从而获得更精准的推荐流量。
3.  **“关注”** 是用户对账号长期价值的认可，混乱的标签会削弱这种价值感。

**核心要义**：你必须保证自己的创作者标签**清晰且唯一**，始终专注于一个领域的内容创作。
