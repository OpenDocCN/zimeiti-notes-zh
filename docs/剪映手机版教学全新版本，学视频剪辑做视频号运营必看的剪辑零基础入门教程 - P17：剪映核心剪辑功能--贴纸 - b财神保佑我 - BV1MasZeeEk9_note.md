# 剪映手机版教学：P17：剪映核心剪辑功能--贴纸 🎨

在本节课中，我们将要学习剪映的核心剪辑功能之一——贴纸。贴纸是提升视频趣味性和画面表现力的重要工具，尤其适用于Vlog、美食、旅拍等类型的视频制作。我们将了解其作用，并掌握如何为视频添加和设置贴纸。

## 贴纸功能概述

上一节我们介绍了其他剪辑功能，本节中我们来看看贴纸。贴纸最初常用于照片修图，如今在视频制作中同样扮演着重要角色。通过添加贴纸，可以显著提升视频的画面质量和信息传递效果。

在剪映中，贴纸功能提供了海量的元素库。选择并添加合适的贴纸，能够增强视频的炫酷感和趣味性。例如，在换装、变身或日常Vlog类视频中，贴纸能有效提升画面的生动感和整体形象。

## 贴纸添加与设置实操

![](img/1386571fe26b828c9c411ba4a97c7895_2.png)

以下是贴纸功能的具体操作步骤。

1.  **新建项目与导入素材**
    首先，在剪映中新建一个项目，并导入需要剪辑的视频素材。

2.  **进入贴纸素材库**
    在下方的一级菜单栏中，找到并点击 **“贴纸”** 按钮，即可进入贴纸素材库。

3.  **选择并添加贴纸**
    素材库中提供了丰富的贴纸类别，例如热门表情、日常Vlog、文字、边框、自然元素（如太阳、彩虹、蝴蝶、花朵）等。根据视频题材，选择合适的贴纸，点击即可添加到视频轨道上。

![](img/1386571fe26b828c9c411ba4a97c7895_4.png)

4.  **调整贴纸时长**
    添加后，可以拖动贴纸轨道两端的白边，以调整其显示的持续时间。你可以让贴纸贯穿整个视频，也可以让它只在特定片段出现。

5.  **为贴纸添加动画**
    选中轨道上的贴纸，点击 **“动画”** 按钮，可以为贴纸设置入场、出场及循环动画效果，让它的出现方式更加生动。

![](img/1386571fe26b828c9c411ba4a97c7895_6.png)

## 课程总结

本节课中，我们一起学习了剪映的贴纸功能。贴纸是快速提升视频画面氛围、让内容更加生动有趣的有效工具。操作流程可总结为：**点击贴纸 -> 选择类别 -> 添加至轨道 -> 调整时长 -> 设置动画**。掌握这些步骤，你就能轻松地为视频增添个性化的装饰元素。
