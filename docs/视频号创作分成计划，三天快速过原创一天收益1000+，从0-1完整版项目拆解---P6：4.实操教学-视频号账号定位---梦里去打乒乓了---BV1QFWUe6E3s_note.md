如果你只是将视频号当作朋友圈随意分享生活，那么发任何内容都可以。但如果你希望通过视频号运营来赚取流量、实现盈利，或者为他人提供价值，那么你就需要认真地为账号进行定位。

定位是第一步。它本质上决定了你后期带货的方向和吸引的人群类型，因此必须重视。

另外，如果你在其他平台（如抖音、快手）已有作品或运营过账号，可以直接将内容搬运到视频号。有些在抖音不火的内容，发布到视频号后可能会突然爆火。视频号的播放量增长通常非常直观。所以，已有作品的同学可以尝试这种方式。

![](img/50a7c6212e05555a9bb40ed784d13418_12.png)

以下是关于定位类型的选择，你可以根据自己的兴趣爱好来决定。

![](img/50a7c6212e05555a9bb40ed784d13418_13.png)

我总结了一些比较容易上手的类型：
*   **音乐类**
*   **好物分享类**（生活好物）
*   **生活记录类**（分享精彩生活）
*   **技能展示类**（如唱歌、烹饪等）

请根据你的兴趣爱好或特长来选择。一旦确定，就不要随意更改。

“垂直发布”的意思是：如果你决定做搞笑类内容，就应该持续发布搞笑类视频，而不是今天发搞笑、明天发影视剪辑、后天发音乐表演。内容杂乱无法吸引精准粉丝，这一点需要注意。

账号定位是第一步，确定后不要乱改。之后你需要垂直发布内容。对于新创建的账号，在完成“养号”后，初期可以多发一些视频，例如每天5-6个。最低限度也应保证每天持续更新1-2个。只有持续生产内容，才能有效吸引用户。
