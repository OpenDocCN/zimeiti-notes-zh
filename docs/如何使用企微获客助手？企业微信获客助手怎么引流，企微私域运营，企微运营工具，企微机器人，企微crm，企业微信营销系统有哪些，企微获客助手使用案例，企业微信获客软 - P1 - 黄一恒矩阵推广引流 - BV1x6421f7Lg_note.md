# 企业微信运营工具教程：P1：企微获客助手使用指南

## 软件安装与基础设置

![](img/767515df753cd092db722a1f366c7717_2.png)

上一节我们介绍了课程目标，本节中我们来看看如何正确安装软件并进行必要的基础设置，以确保软件稳定运行。

使用软件前，必须安装 **4.1.2** 版本的企业微信客户端。最新版本无法兼容此助手软件。

以下是安装与基础设置的具体步骤：

1.  **安装指定版本**：首先，下载并安装企业微信 **4.1.2** 版本客户端。
2.  **安装获客助手**：接着，安装“企微获客助手”软件。安装完成后，桌面会出现软件图标。
3.  **启动与多开**：双击图标启动软件。首次启动加载可能较慢，建议重启电脑后再打开以确保功能正常。如需同时运行多个账号，可多次双击图标实现多开。
4.  **登录账号**：点击软件内的登录按钮，并在手机企业微信上确认登录（首次使用需扫码）。
5.  **关闭自动更新**：登录后，点击软件右上角头像，进入“设置” -> “通用”，找到并**取消勾选“自动更新”**选项，防止软件自动升级到不兼容的新版本。
6.  **设置消息免打扰**：在设置中，找到并**勾选“合并免打扰的聊天”**选项。此设置可将所有免打扰群的消息合并到一个窗口，避免海量群消息刷屏。
7.  **定期清理缓存文件**：为防止软件缓存文件占满硬盘，需定期清理。路径通常为：`我的电脑/文档/WeChat Files/WXWork/...`。删除该目录下的缓存文件不影响聊天记录。

![](img/767515df753cd092db722a1f366c7717_4.png)

完成以上设置后，软件环境便已准备就绪。

## 核心功能一：自动化群发

基础设置完成后，我们就可以开始使用软件的核心功能了。首先，我们来了解如何利用软件进行高效的自动化群发。

该助手软件功能虽多，但最常用的是“群发”和“加粉”功能。群发功能支持向好友、外部联系人或群组发送多种类型的内容。

以下是群发功能的使用方法：

*   **选择发送对象**：在群发界面，勾选需要发送消息的“外部联系人”或“群聊”。
*   **编辑发送内容**：点击“群发内容”输入框，可以添加文本、链接、小程序或名片等内容。
*   **执行发送**：编辑好内容后，点击发送按钮即可。软件会自动向所有勾选的对象发送消息。

> **注意**：演示账号数据较少，实际运营中需注意发送频率与内容合规性，避免违规。

## 核心功能二：自动化加粉（爆粉）

掌握了群发功能后，我们来看看另一个核心功能——自动化加粉，这也是实现引流的关键。

“加粉”功能的核心是自动向目标群成员发送好友申请。要实现加粉，首先需要获取大量的潜在客户群。

以下是加粉功能的操作流程与注意事项：

1.  **获取群资源**：通常通过“换群”方式获取。先购买或拥有一些种子群，然后使用专门的换群软件以500个群为单位进行交换。**企微获客助手自带的拉群功能有限制，无法实现大量换群**。
2.  **配置加粉参数**：在软件的“加粉”功能中，勾选目标群，点击“爆粉配置”。可以设置发送间隔等参数，通常勾选“不加密”即可，然后点击确认。
3.  **开始执行**：点击“开始”按钮，软件便会自动向所选群成员发送好友申请。整个过程全自动，无需手动操作。
4.  **账号养护与扩容**：
    *   每个活跃的企业微信账号，每日可发送 **100 到 300 个** 好友申请。
    *   账号需“养号”，即保持常用IP和设备登录，行为正常。
    *   发送量需循序渐进扩容，例如：`第一天:10个 -> 第二天:20个 -> 第五天:50个 -> 第十天:100个 -> 第二十天:200个`。若首日即发送300个，通常无法成功。

通过多开软件，可以同时操作多个账号进行加粉，提升效率。
