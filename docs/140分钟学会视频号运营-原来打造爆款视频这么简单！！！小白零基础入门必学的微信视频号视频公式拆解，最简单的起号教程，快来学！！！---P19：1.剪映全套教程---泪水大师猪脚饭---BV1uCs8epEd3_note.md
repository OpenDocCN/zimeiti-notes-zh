# 视频号运营：P19：1.剪映全套教程

在本节课中，我们将学习如何从零开始自学剪映软件，掌握视频剪辑的基础操作。这是视频号运营中制作视频内容的关键技能。

## 课程概述

剪辑是短视频制作的基础。目前，短视频已普及多年，许多人已具备一定的剪辑基础。如果你从未使用过剪映，本教程将指导你如何高效自学。

![](img/978cb4c056576242bad51bf2a2d5122e_1.png)

## 如何获取与认识剪映

![](img/978cb4c056576242bad51bf2a2d5122e_3.png)

首先，你需要在手机应用商店下载“剪映”App。它是抖音官方的剪辑软件，功能强大且大部分功能免费，足以满足我们的日常剪辑需求，无需开通会员。

![](img/978cb4c056576242bad51bf2a2d5122e_5.png)

接下来，我们将介绍两种高效的自学路径。

## 路径一：通过剪映官方课堂学习

![](img/978cb4c056576242bad51bf2a2d5122e_7.png)

![](img/978cb4c056576242bad51bf2a2d5122e_9.png)

打开剪映App后，你可以通过内置的“创作者课堂”系统学习。

![](img/978cb4c056576242bad51bf2a2d5122e_11.png)

以下是具体操作步骤：

![](img/978cb4c056576242bad51bf2a2d5122e_13.png)

1.  **进入课堂**：在剪映首页最下方，找到并点击“创作者课堂”。
2.  **搜索官方账号**：在课堂页面的搜索框中，输入“剪映小助手”进行搜索。
3.  **关注官方账号**：在搜索结果中，选择“用户”标签页，第一个结果就是“剪映官方账号”，请关注它。
4.  **学习免费课程**：进入官方账号主页，点击“课程”栏目。这里推荐学习《剪映功能大全》和《新手常见问题》这两个免费系列课程。
    *   《剪映功能大全》包含87节课。
    *   《新手常见问题》包含10节课。

![](img/978cb4c056576242bad51bf2a2d5122e_15.png)

这些课程每节时长很短，多为几十秒到两分钟。完整学完所有课程大约需要两小时，即可从零基础达到熟练操作的水平。建议没有剪辑基础的同学务必先系统学习这些基础功能，再进行实操。

## 路径二：通过抖音平台学习

![](img/978cb4c056576242bad51bf2a2d5122e_17.png)

除了剪映App内部，抖音平台也是一个丰富的学习资源库。

![](img/978cb4c056576242bad51bf2a2d5122e_19.png)

你可以在抖音的搜索框中，直接搜索抖音号 **`妮妮6868618`**。这位创作者（妮妮短视频）制作了一系列简短精炼的剪映教程，将核心功能分解成多个短小视频，讲解细致，适合快速入门。

此外，你也可以在抖音搜索“零基础学剪辑”等关键词，会发现大量教授剪辑技巧的账号。观看其中几个高赞视频，就能快速掌握剪映的基础操作。

## 总结与建议

本节课我们一起学习了两种自学剪映的方法：通过剪映App内的官方“创作者课堂”系统学习，以及通过抖音平台搜索特定账号或关键词进行碎片化学习。

![](img/978cb4c056576242bad51bf2a2d5122e_21.png)

剪辑本身并不难，关键在于动手练习。推荐将官方教程与博主“妮妮6868618”的课程结合观看，这些基础功能完全能满足我们制作视频号内容的需求。学完后，请务必多加练习，巩固技能。
