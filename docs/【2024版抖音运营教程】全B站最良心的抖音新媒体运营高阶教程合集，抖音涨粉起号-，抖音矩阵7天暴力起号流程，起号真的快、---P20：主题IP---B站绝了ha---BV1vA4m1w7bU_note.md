## 什么是主题IP？
上一节我们介绍了内容定位的重要性，本节中我们来看看如何为内容注入灵魂，即构建**主题IP**。简单来说，主题IP就是贯穿你一系列视频的核心主题或故事线。它让用户对你的内容产生持续期待，而不仅仅是观看一个孤立的视频。

为了更直观地理解，我们接下来再看两个视频案例。

![](img/dce3f9ac29929c676a81c11093f2f3e7_8.png)

首先我们看左边这个视频。标题是“老婆怀孕以后一直停更的，惹老婆生气系列又回来了”。视频内容是夫妻间的嬉戏打闹。



![](img/dce3f9ac29929c676a81c11093f2f3e7_2.png)

我们再看右边这个视频。标题是“女生说我没有你想象的那么好，该怎么回复”。视频内容是一位“小姨”在黑板前讲解恋爱技巧。



![](img/dce3f9ac29929c676a81c11093f2f3e7_8.png)

好了，这两个视频我们都看过了。



你们有没有发现它们有一个共同的特点？
首先我们来看“挑战惹老婆生气的第21天”这个视频。如果你不看标题文字，只看他和他老婆嬉戏打闹的内容，这似乎只是一个普通的日常视频，镜头和画面也没有特别出彩的地方。

![](img/dce3f9ac29929c676a81c11093f2f3e7_12.png)

但是，他加上了一个主题：“挑战惹老婆生气的第21天”。这个标题引导着用户思考：这是第21天，那是不是还有第20天？之前是怎么惹老婆生气的？用户会因为对这个主题的好奇，而想去查看他之前的视频。这就是主题IP的作用：**它能延伸出无数期内容，让用户产生持续关注的欲望**。

例如，之前很火的“阿拉蕾的装修日记”，就是围绕“装修小院”这个主题，衍生出了多期作品。

右边“小余恋爱课”的视频策划得更绝。他的主题IP是“小姨恋爱课”。这个人设和主题策划得非常好，因为它同时满足了多种用户需求：为不会追女生的直男提供方法、吸引对恋爱感情感兴趣的人、引发有过类似经历用户的共鸣。这个系列视频拍起来也简单，一块黑板就可以拍很多期。

相比之下，如果一个账号没有明确的主题IP（例如，只是一个单纯唱歌的账号），用户听完一首歌点赞后可能就划走了，不会关心他昨天唱了什么、明天要唱什么，账号的粘性就很低。因此，**主题IP的存在对于吸引和留住用户至关重要**。

## 如何策划主题IP？
那么，主题IP我们怎么去设计呢？这里给到大家具体的方法。大家在明确了商业定位（例如，决定做医美行业）后，不要着急开拍。内容定位中还有许多要素需要填充，**主题IP就是我们需要填充的第一个关键要素**。

以下是策划主题IP的几种常用公式和原则：



策划主题IP的核心公式可以归纳为：**【变量】+【垂直领域】**。
这里的【变量】可以是名字、人群、地域或场景，它们与你的【垂直领域】相结合，就能形成一个具体的主题IP。

以下是几种常见的组合方式及例子：

1.  **名字 + 垂直领域**
    *   **公式**：`个人名称/昵称 + 内容领域`
    *   **例子**：
        *   某某的北漂日记（零二集）
        *   老韩摄影教学（第12期）
        *   猪猪小姐姐的撩弟指南之（某某某）

2.  **人群 + 垂直领域**
    *   **公式**：`目标人群 + 内容领域`
    *   **例子**：
        *   别人家男朋友的拍照技巧（第三期）
        *   宝妈育儿小技巧
        *   新手开店避坑指南

3.  **地域 + 垂直领域**
    *   **公式**：`地点 + 内容领域`
    *   **例子**：
        *   西藏自驾之旅（第50天）
        *   环球旅居日本站
        *   广州美食之火锅篇

4.  **场景 + 垂直领域**
    *   **公式**：`特定场景 + 内容领域`
    *   **例子**：
        *   办公室迷惑行为之（为什么总有人脚臭）
        *   儿童房风格装修
        *   婚礼现场的创意小心思

## 总结
本节课中我们一起学习了**主题IP**的概念与策划方法。我们了解到，主题IP是一个能够串联起多期视频的核心主题，它能有效提升用户粘性和内容期待感。通过分析“惹老婆生气系列”和“小姨恋爱课”等案例，我们明白了主题IP的强大作用。最后，我们掌握了策划主题IP的四种实用公式：**名字、人群、地域、场景** 分别与 **垂直领域** 相结合。记住这个核心公式，就能为你自己的账号找到一个清晰、有吸引力的内容主线。
