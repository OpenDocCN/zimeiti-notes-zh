# AI短视频创作：P10：影视推广平台结算指南 🎬

在本节课中，我们将详细探讨如何将AI生成的视频发布到抖音、快手等平台，并通过“影视推广”任务来结算收益。你将了解不同平台的收益规则、任务参与流程以及如何最大化你的创作回报。

很多同学询问，如果想把视频发布到抖音或快手平台，应该如何结算收益。这里需要明确区分不同平台的收益模式。

上一节我们介绍了视频的多平台分发，本节中我们来看看针对特定平台的收益结算方法。

![](img/1fc42590c6e5705db0650d807195dc90_5.png)

根据之前的讲解，视频制作完成后可以发布到30多个自媒体平台。除了抖音和快手，其他平台通常可以依靠视频本身的播放量来直接结算收益。

![](img/1fc42590c6e5705db0650d807195dc90_1.png)

![](img/1fc42590c6e5705db0650d807195dc90_7.png)

然而，对于抖音和快手这两个平台，情况有所不同。正如之前提到的，视频在这两个平台发布，主要作用可能是增长粉丝，而非直接通过播放量获得收益。

![](img/1fc42590c6e5705db0650d807195dc90_3.png)

那么，如果有同学只想或主要运营抖音和快手，同时又希望不仅能涨粉，还能通过播放量获得收益，有没有办法呢？答案是肯定的。

接下来，我们介绍一种可以实现此目标的方法。

视频可以发布在以下四个我们熟悉的平台：**抖音、快手、小红书以及微信视频号**。小红书在年轻用户，特别是95后和女性用户中较为流行。微信视频号则依托于微信这个全民应用，触达非常方便。

![](img/1fc42590c6e5705db0650d807195dc90_5.png)

在这几个平台发布后，我们可以通过参与“影视推广”平台的任务来结算奖金。以下是参与影视推广任务的核心步骤：

1.  **领取任务**：在影视推广平台上，有大量来自爱奇艺、搜狐等平台的任务，每个任务对应特定的电视剧、电影或综艺节目。
2.  **制作与发布视频**：根据任务要求制作推广视频，并发布到上述指定的平台（如抖音、小红书、视频号）。
3.  **提交任务回填**：这是结算收益的关键。你需要到任务页面提交一个“回填链接”，填写以下信息：
    *   你的微信号、账号昵称或平台ID。
    *   所推广的剧集名称。
    *   发布作品的平台链接。
    *   作品发布时间。
    *   作品播放量或点赞量的截图。
4.  **结算收益**：平台审核你提交的信息后，会根据约定的规则为你结算奖金。

影视推广的收益通常比单纯依靠平台播放量分成的收益更高。平台上有丰富的短剧推广任务，这些短剧多由传媒公司或网红制作，他们需要外部创作者帮助宣传。

![](img/1fc42590c6e5705db0650d807195dc90_7.png)

参与这类任务**对你的账号粉丝数没有要求**。收益通常根据你为同一部剧创作的作品中，**播放量最高的三个作品的数据总和**来计算。

不同平台的收益计算规则有所不同，以下是具体的公式：

*   **抖音**：按播放量计算。
    *   `收益 = (总播放量 / 10000) * 50元`
    *   例如，1万播放量可获得50元奖金。
*   **微信视频号**：按播放量计算。
    *   `收益 = (总播放量 / 10000) * 10元`
    *   单价较低，因为用户基数大。
*   **小红书**：按播放量计算，单价最高。
    *   `收益 = (总播放量 / 1000) * 30元`
    *   例如，1000播放量可获得30元奖金。
*   **抖音+快手（部分任务）**：按点赞数计算。
    *   抖音：`5000赞 = 100元`， `10000赞 = 200元`
    *   快手：`10000赞 = 180元`
    *   同样对账号粉丝数无要求。

此外，像搜狐短剧等任务，虽然其平台知名度可能不如前几位，但提供的推广单价往往非常有竞争力。
