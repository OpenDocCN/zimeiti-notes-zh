# 剪映手机版教学：P35：14.镜头跟踪人物效果教程 🎬

在本节课中，我们将学习如何使用剪映手机版制作一个“镜头跟踪人物”的视觉效果。这个效果能让你的镜头像摄像机一样，在合影照片中依次聚焦到不同的人物脸上，最后再展现出全景，非常适合用于视频号等平台的创意内容制作。

## 概述
镜头跟踪效果的核心是通过关键帧控制画面的缩放与位置变化，模拟出镜头移动和聚焦的感觉。我们将使用一张合影照片作为素材，通过一系列关键帧设置来实现。

## 教程步骤

上一节我们概述了效果原理，本节中我们来看看具体的操作步骤。

以下是制作镜头跟踪人物效果的具体流程。

1.  **导入并设置主素材**
    *   准备一张合影照片，将其导入剪映并添加到主轨道。
    *   将这张照片的时长拉长，为后续的关键帧动画预留足够的时间。

2.  **为画面运动添加关键帧**
    *   将时间线移动到视频的**起始位置（0秒）**。
    *   选中主轨道照片，在预览窗口中**放大画面**，使镜头对准第一个要跟踪的人物。
    *   在此位置**打上一个关键帧**，记录下画面当前的缩放和位置状态。

3.  **设置第二个跟踪点**
    *   将时间线移动到**第1秒**的位置。
    *   在预览窗口中，**调整画面**，将镜头移动到并对准第二个人物。
    *   软件会自动在此时间点**打上新的关键帧**。

4.  **设置第三个跟踪点**
    *   将时间线移动到**第2秒**的位置。
    *   再次**调整画面**，将镜头移动到并对准第三个人物。
    *   此操作会**打上第三个关键帧**。

5.  **设置全景结尾**
    *   将时间线移动到视频的**结尾处**。
    *   **调整画面**，缩小镜头以露出整张合影照片。
    *   此操作会**打上最后一个关键帧**。至此，主画面的镜头移动动画已完成。

6.  **添加圆形聚焦遮罩**
    *   点击**画中画**功能，**新增一个画中画轨道**。
    *   从**素材库**中导入一张**纯黑色图片**，并将其添加到画中画轨道。
    *   将这张黑色图片**放大至全屏**。
    *   选中黑色图片，找到**蒙版**工具，选择**圆形**蒙版。
    *   点击**翻转**按钮，使圆形区域变为透明，周围是黑色。
    *   调整圆形蒙版的**羽化值**，使其边缘柔和。

7.  **同步遮罩与镜头运动**
    *   选中画中画黑色图层，将时间线移回主画面**第一个关键帧（0秒）**的位置。
    *   在此处为黑色图层的蒙版形状**打上一个关键帧**，并调整圆形蒙版的大小和位置，使其框住第一个人物。
    *   将时间线移到主画面**第二个关键帧（1秒）**的位置。
    *   同步调整圆形蒙版的大小和位置，使其框住第二个人物，并**打上关键帧**。
    *   将时间线移到主画面**第三个关键帧（2秒）**的位置。
    *   同步调整圆形蒙版，使其框住第三个人物，并**打上关键帧**。

8.  **制作遮罩淡出效果**
    *   将时间线移到视频的**最后（结尾关键帧处）**。
    *   打开蒙版设置，**放大圆形蒙版**，直到它完全超出画面，露出整个背景照片。
    *   将时间线移回**视频开头**。
    *   同样打开蒙版，将圆形蒙版**向下拉出画面**，制作一个从画面外进入的动画效果。

## 总结
本节课中我们一起学习了如何在剪映中制作“镜头跟踪人物”的效果。我们通过为**主画面添加关键帧 (`keyframe`)** 来控制镜头的移动和缩放，模拟跟踪拍摄。同时，利用**画中画**和**圆形蒙版**功能，创建了一个聚焦光圈，并通过同步其关键帧与主画面运动，增强了视觉引导效果。最后，为遮罩添加入场和出场动画，使整体过渡更加流畅自然。完成后，为视频配上合适的音乐即可。
