# 小红书运营教程：P4：小红书的检测规则与运营注意事项 📝

在本节课中，我们将详细解析小红书的平台检测规则。了解这些规则是避免账号违规、确保内容安全发布的基础。我们将从检测范围、设备管理、内容规范到原创度与敏感词检查等多个方面进行系统讲解。

## 平台检测的核心范围

小红书平台主要对以下几个方面的内容进行检测。

以下是具体的检测项目列表：
*   **个人资料**：账号简介、头像等。
*   **笔记内容**：发布的图文或视频笔记正文。
*   **私信交流**：用户之间的私密消息。
*   **评论互动**：笔记下的公开评论。

在这些范围内，平台严格禁止出现以下信息：**微信号、微信二维码、代购、水印、推广淘宝等第三方平台的营销词汇**。

不过，平台管理相对宽松的地方在于：官方**个人资料允许填写邮箱地址**。同时，可以通过提及“微博”等谐音或替代词进行间接引流，再通过话术引导至微信。具体的引流方法与话术技巧，我们将在后续课程中详细展开。

## 账号与设备管理规则

上一节我们介绍了内容检测的范围，本节中我们来看看账号与设备层面的管理规则。

小红书会检测账号登录的**手机设备**。平台建议，**同一台设备最多只登录三个账号**。请务必注意这一点，过多账号在同一设备登录可能被系统判定为异常操作。

## 笔记内容的行为规范

除了基础信息，笔记内容本身的行为引导也有明确限制。

在笔记中，**不得诱导用户互动**。例如，“关注领奖”、“点赞抽奖”等都属于小红书明令禁止的诱导行为。

## 内容的原创度要求

小红书平台对内容的**原创度保护程度较高**。因此，在推广时需要注意：**一篇文章只在一个账号发布**。切勿将同一篇文章在多个小红书账号上重复发布。

## 敏感词的识别与处理

接下来，我们将探讨一个非常关键且实用的部分：敏感词检测。这是运营中经常会遇到的问题。

平台内置了敏感词检测机制。例如，涉及“手机卡”、“金融”、“电子烟”、“代购”、“医疗用品”等领域的词汇，应尽量避免触碰。

![](img/fc44f3a55ee8d517c1573d78ce128fc5_1.png)

那么，在撰写文章时，如果不小心加入了未知的敏感词该怎么办？这时就需要借助外部工具进行检测。

![](img/fc44f3a55ee8d517c1573d78ce128fc5_3.png)

以下是两个常用的敏感词检测网站，可以辅助我们进行内容审查：
1.  第一个网站（无需登录）：将写好的内容粘贴进输入框，点击“立即检测”，系统会高亮标出敏感词，例如“代购”。
    
    ![](img/fc44f3a55ee8d517c1573d78ce128fc5_1.png)

2.  第二个网站（需注册）：使用逻辑类似，注册登录后，将内容粘贴进去进行过滤，即可得到检测结果。
    
    ![](img/fc44f3a55ee8d517c1573d78ce128fc5_3.png)
    
    ![](img/fc44f3a55ee8d517c1573d78ce128fc5_5.png)

通常，使用第一个网站进行检测已经足够。我们只需在发布前，将文章内容放入这些网站检测一遍，就能有效规避大部分敏感词风险。

![](img/fc44f3a55ee8d517c1573d78ce128fc5_5.png)
