# 拼多多开店运营：P26：新品平台排名规则 📊

![](img/8aa8236b033d6fb55f712ddcff1236d3_1.png)

在本节课中，我们将要学习拼多多平台新品排名的核心规则。理解这些规则是获取自然流量、提升商品曝光的基础。我们将从规则的重要性、官方定义、具体考核维度以及排名逻辑四个方面进行详细拆解。

## 规则的重要性

了解平台排名规则，有助于我们明确运营方向，知道平台鼓励什么、考核什么，从而有针对性地优化店铺和商品。

对于商家而言，排名规则能激励良性竞争，促使商家努力提升商品质量和服务水平，从而为店铺创造更大价值。

对于用户而言，合理的排名规则能确保用户在搜索时，快速找到高质量、受欢迎的商品，从而获得更好的购物体验。

![](img/8aa8236b033d6fb55f712ddcff1236d3_3.png)

对于平台而言，优秀的商品和良好的用户体验能扩大市场体量，吸引更多优质商家入驻，形成一个健康、可持续的生态循环。

## 官方排名规则定义

根据拼多多官方规定，平台内商品的排名是**根据一段时间内**的商家商品上线时间、销量、价格、质量、售后、商品评分等综合因素进行动态更新的。

![](img/8aa8236b033d6fb55f712ddcff1236d3_8.png)

![](img/8aa8236b033d6fb55f712ddcff1236d3_9.png)

具体考核内容主要分为四大类：
*   店铺数据
*   商品质量
*   商品数据
*   处罚类

## 排名规则详解

![](img/8aa8236b033d6fb55f712ddcff1236d3_13.png)

商品的排名考核遵循一个核心原则：**优先考核商品本身，再考核店铺整体**。

![](img/8aa8236b033d6fb55f712ddcff1236d3_15.png)

### 1. 商品质量考核

![](img/8aa8236b033d6fb55f712ddcff1236d3_16.png)

![](img/8aa8236b033d6fb55f712ddcff1236d3_18.png)

商品质量是产品上架后，平台首先抓取并考核的部分。它决定了商品的**基础曝光量**。

以下是商品质量包含的核心要素：
*   **标题与关键词**：是否准确、覆盖用户搜索需求。
*   **主图与详情页**：是否清晰、美观、能吸引点击并展示卖点。
*   **价格与属性**：定价是否合理，属性填写是否完整准确。

![](img/8aa8236b033d6fb55f712ddcff1236d3_20.png)

![](img/8aa8236b033d6fb55f712ddcff1236d3_21.png)

![](img/8aa8236b033d6fb55f712ddcff1236d3_22.png)

![](img/8aa8236b033d6fb55f712ddcff1236d3_23.png)

平台会根据这些基础信息判断商品的潜力。用户需求高、质量好的商品，获得的基础曝光会更多。

### 2. 商品数据考核

![](img/8aa8236b033d6fb55f712ddcff1236d3_25.png)

当商品获得基础曝光并产生用户点击后，平台会进入更深层次的**商品数据考核**。这部分数据直接决定了排名的升降。

![](img/8aa8236b033d6fb55f712ddcff1236d3_26.png)

![](img/8aa8236b033d6fb55f712ddcff1236d3_27.png)

![](img/8aa8236b033d6fb55f712ddcff1236d3_28.png)

![](img/8aa8236b033d6fb55f712ddcff1236d3_29.png)

商品数据主要包含以下几项：
*   **销量**：并非单纯指销售件数，而是一个综合指标，包括订单量、销售额、客单价、UV价值（访客平均价值）、同比增长率等。
*   **收藏率**：用户收藏商品的比率。
*   **转化率**：访客转化为下单买家的比率。
*   **上新时间**：新品会有一定的流量扶持期。

![](img/8aa8236b033d6fb55f712ddcff1236d3_30.png)

平台会将这些数据与**同行同类目商品**进行对比。数据表现更好、更受用户欢迎的商品，排名会更靠前。

![](img/8aa8236b033d6fb55f712ddcff1236d3_32.png)

![](img/8aa8236b033d6fb55f712ddcff1236d3_33.png)

### 3. 店铺数据与处罚考核

当商品质量和商品数据相近时，平台会进一步考核**店铺数据**和**处罚记录**来区分排名。

店铺数据考核指标包括：
*   3分钟人工回复率
*   30天投诉率
*   物流服务异常率
*   其他各项服务指标

处罚类主要指店铺的**违规记录**。一旦店铺出现违规，商品排名就会受到负面影响，曝光范围会被压缩。

![](img/8aa8236b033d6fb55f712ddcff1236d3_35.png)

## 排名逻辑可视化解析

我们可以用一个坐标图来形象地理解排名逻辑：

![](img/8aa8236b033d6fb55f712ddcff1236d3_37.png)

1.  **曝光原点**：商品上架时，处于一个初始的曝光位置。
2.  **商品质量决定曝光宽度**：商品质量优化得越好（如标题、主图），曝光能覆盖的**横向范围（宽度）** 就越大。
3.  **商品数据决定曝光高度**：商品数据表现越好（如销量、转化率），曝光能到达的**纵向高度**就越高。
4.  **违规导致原点偏移**：如果店铺出现违规或服务数据不达标，整个曝光原点会向**左下方（负向）移动**。这意味着，即使你的商品质量和数据尚可，有效的曝光范围（图中黄色区域）也会大幅缩水，整体流量减少。

![](img/8aa8236b033d6fb55f712ddcff1236d3_38.png)

**核心结论**：想要获得最大曝光，必须同时做好**商品基础优化**、提升**商品核心数据**，并严格**避免店铺违规**。
