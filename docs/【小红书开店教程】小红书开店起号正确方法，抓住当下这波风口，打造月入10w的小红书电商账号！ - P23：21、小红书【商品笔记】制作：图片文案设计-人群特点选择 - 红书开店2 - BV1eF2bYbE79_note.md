图片与文案的内容非常广泛，但本教程将聚焦于重点，并帮助大家理解其核心。首先，我们从分析目标人群的特点入手。

## 人群特点分析与选择 👥

以下是基于小红书平台用户画像的分析与选择策略：

**1. 用户以年轻高消费力人群为主**
小红书用户以年轻人为主，特别是90后。商品购买力主要集中在一线和新一线城市（如北京、上海、广州、深圳、武汉等）。这些城市的女性用户对自身要求高，具备较强的消费能力。因此，设计笔记时必须考虑该群体的审美偏好和消费习惯。

**2. 注重内容真实性与质量**
用户倾向于信赖专业和真实的评价。因此，文案和图片必须体现商品的实质性内容和真实的用户体验。虚假宣传难以在小红书长久生存，平台用户非常“挑剔”。

**3. 偏好图文并茂的形式**
优秀的笔记需要图片、文字和文案内容三者兼备，缺一不可。具体形式需根据自身产品特点进行设计和划分。

**4. 浓厚的社区氛围利于传播**
小红书具有强大的社区属性，优质内容容易形成“一人传染十人，十人传染百人”的裂变效应。因此，营造氛围、突出潮流、时尚或独特风格至关重要。

**5. 用户习惯在此进行消费决策**
许多用户将小红书作为消费决策的入口，习惯在此搜索产品信息和评价。因此，文案应包含关键信息和正面评价，以帮助用户做出购买决定。

**6. 用户群体多样化，需精准挖掘**
品牌或团队应挖掘不同细分行业和品类的用户特点。即使是一个人操作，也应通过内容包装，突出专业性或社群氛围，吸引目标用户。

**7. 对新消费场景有较高兴趣**
用户对新的消费场景（如口罩妆、新式茶饮、宅家健身等）抱有浓厚兴趣。这为相关品类提供了巨大的市场空间。

**8. 关注性价比**
用户在追求品质的同时，也对价格有一定敏感度。在文案中可以适当强调产品的价格优势或性价比。但针对一二线城市用户，建议定位中高端市场，价格可设定为成本的 **1.5 到 1.7 倍**，以匹配其消费能力。

**9. 对流行趋势敏感**
设计时应把握当前的流行元素。可以参考热门美妆、穿搭博主的风格，将其融入产品图片的设计中。

## 产品特点选择（反向操作） 🔄

除了根据人群选择产品，还可以从产品本身出发，反向寻找适合的人群。这是一种高效的运营思路。

![](img/e45dfff621eaf3fdf63de072ebb9d520_8.png)

![](img/e45dfff621eaf3fdf63de072ebb9d520_9.png)

**核心公式：**
```
目标人群 = 产品定位 + 价格区间 + 类型筛选
```

具体操作是：明确你的产品适合什么样的用户，具有什么特点，然后反向去定位和吸引这部分人群。这与之前“通过人群找产品”的思路形成互补。

![](img/e45dfff621eaf3fdf63de072ebb9d520_11.png)
