# 实体店抖音直播带货保姆级实战起号课：P10：如何开通商品橱窗 🛍️

在本节课中，我们将要学习如何在抖音上开通“商品橱窗”功能。这是进行直播带货或视频带货的关键一步，它允许你分享和销售商品，而无需自己拥有营业执照或囤积货物。

上一节我们介绍了抖音直播的基础设置，本节中我们来看看如何开通商品橱窗功能，从而获得带货资格。

## 商品橱窗与抖音小店的区别

商品橱窗功能与开通抖音小店不同。开通商品橱窗无需你提供营业执照，也无需你自己进货。你可以直接从抖音的“精选联盟”中选择商品进行分享和销售。

这意味着你可以筛选出那些销量高、评分好、性价比高的商品。由商品供应商负责发货和库存管理，你只需专注于内容创作和销售环节。

## 开通商品橱窗的步骤

以下是开通商品橱窗权限的具体操作流程。

1.  **进入创作者服务中心**
    打开抖音APP，点击首页右上角的“三条杠”图标。在弹出的菜单中，选择“创作者服务中心”。

2.  **找到商品橱窗申请入口**
    在创作者服务中心页面，点击“全部分类”。在分类列表中找到“内容变现”板块，并点击其中的“商品橱窗”。

3.  **申请带货权限**
    点击后，页面将显示“商品分享权限申请”。请点击“成为带货达人”。系统会跳转至申请界面，在此点击“带货权限申请”。

4.  **完成实名认证**
    如果你的账号尚未完成实名认证，系统会提示你先进行认证。请按照指引完成实名认证流程。

5.  **缴纳保证金**
    实名认证通过后，需要缴纳500元人民币的保证金。点击充值并完成支付即可。

## 开通必须满足的条件

在申请过程中，除了完成上述操作，还需确保账号满足平台设定的条件。以下是开通商品橱窗必须满足的四个核心条件。

*   **实名认证**：账号已完成实名认证。
*   **保证金**：已成功缴纳500元保证金。
*   **作品数量**：抖音账号已公开发布不少于10个作品。
*   **粉丝数量**：账号粉丝数达到1000或以上。

关于“10个作品”这一条件，请注意：初期不必追求作品的高质量。你可以拍摄一些简单的视频，只要画面不是过于模糊即可。**关键点：务必保证作品为原创，不要搬运他人的内容**。这10个作品可以一次性上传，平台对此没有限制。满足所有条件后，即可成功开通商品橱窗。

## 开通后的操作建议

商品橱窗成功开通后，你可以选择将之前为了凑数而发布的10个作品隐藏或删除。此后，你可以专注于策划和发布高质量、垂直领域的优质内容，以吸引粉丝并促进销售。

本节课中我们一起学习了开通抖音商品橱窗的完整流程、必要条件以及注意事项。总结来说，核心步骤是：**进入创作者服务中心 -> 申请带货权限 -> 完成实名认证与保证金缴纳 -> 确保满足“10个作品”和“1000粉丝”的条件**。开通后，你便可以通过精选联盟选品，开始你的带货之旅了。
