## 小红书推流机制：点击为王，互动为后

上一节我们介绍了课程的整体框架，本节中我们来看看小红书的推流机制。理解这个机制是获得流量的基础。

小红书是一个“点击为王，互动为后”的平台。**点击率**直接决定了你笔记观看量的下限。例如，你发布笔记后获得的初始流量（如200、500、1000等）主要就取决于点击率。而点击率主要由**封面**决定，因此优化封面至关重要。

**互动量**则决定了你笔记观看量的上限。笔记发布后，能否从初始的一两千流量突破到5万、10万，取决于小红书的用户反馈算法，即**CES得分**。

CES得分的计算公式如下：
```
CES = 点赞(1分) + 收藏(2分) + 评论(4分) + 转发(4分) + 关注(8分)
```

接下来，小红书会对你的笔记进行**打标签**。标签分为：
*   **大标签**：约30多个，持续更新。你可以在首页点击小三角查看，例如“露营”、“美妆”。
*   **小标签**：大标签下的细分。例如“露营”下包含“露营装备”、“露营地”等。

内容的分发机制基于以下几点：
1.  **兴趣标签分发**：系统根据用户近期的点击、互动、搜索行为推荐内容。
2.  **重定向机制**：内容会优先推送给与你的笔记产生过互动（点赞、收藏、评论、关注）的用户。
3.  **初次分发曝光量**：由账号最近30天的观看及互动数据决定的**流量层级**决定。流量层级高的账号，初始曝光量也更高。

小红书的**流量池推荐机制**核心是**千次曝光得分**。

千次曝光得分的计算公式为：
```
千次曝光得分 = (CES得分 / 曝光量) * 1000
```

当千次曝光得分高时，笔记会进入更大的流量池。小红书共有八个流量池层级：
*   初始流量池：~200
*   基础流量池：200-500
*   三级流量池：500-5000
*   潜力流量池：5000-2万
*   热门流量池：2万-20万
*   大热门流量池：20万-500万
*   爆款流量池：500万-1000万+
*   全站流量池：1200万-3000万+

得分不足时，笔记通常会卡在200曝光左右，无法扩散。

## 小红书起号原理：观察、识别与入池

理解了推流机制后，我们来看看一个新账号是如何被平台识别和培养的。小红书的起号过程分为三个阶段。

**第一阶段：观察期**
新账号注册后，系统会进入观察期，不确定账号质量，因此会限制流量。这通常表现为新账号发布笔记流量很低或没流量。操作建议是：前期内容需要“脱敏”，避免营销敏感内容，尽量发布利他性的干货内容，帮助账号快速度过观察期。

**第二阶段：识别期**
度过观察期后（快则一周，慢则一个月），系统进入识别期。系统会识别你的内容与哪些标签相关，并推荐给相关人群。内容好坏取决于推送给谁。例如，将露营内容推给喜欢露营的人就是好内容。
此阶段有两种标签：
*   **内容标签**：笔记本身对应的标签。
*   **用户反馈标签**：用户互动行为产生的标签。
系统认定内容与用户匹配度越高，推送力度越大。操作方向是：专注做**搜索流量**，发布用户常搜索的内容，帮助账号快速打上精准标签。

**第三阶段：入池期**
内容发布后前几天流量较低（如两三百），但几天后流量突然增长到三五千并持续增加，这代表笔记已进入流量池。加速入池的一个方法是**投薯条**。投薯条可以强制触发人工审核，快速度过观察期，并提高初始流量。
操作步骤分为两次投放：
1.  第一次投放：选择“点赞收藏”目标，拉高CES得分。
2.  第二次投放：选择“浏览量/播放量”目标。
两次投放间隔24小时即可，主要意义在于加速过审、提高初始流量、帮助账号和内容打上精准标签。

## 小红书起号方法：搜索与互动

掌握了起号原理，本节我们来看看两种具体、可操作的起号方法。

**第一种：搜索起号法**
这是目前行业内最简单有效的起号方法。核心目的是让系统快速识别你的领域，并将内容推送给相关人群，快速获得用户反馈标签。因为搜索流量需求明确，用户反馈也明确，易于打标签。小红书搜索流量占比超过30%。
操作步骤如下：
1.  在你的赛道内搜索核心关键词。
2.  整理搜索结果页的**下拉联想词**。
3.  围绕这些下拉词发布选题内容。

以下是操作时的注意细节：
*   优先选择收藏数大于点赞数的笔记作为参考，这类内容更容易卡关键词排名。
*   如果参考爆款内容，需替换30%以上，避免被判搬运。
*   标题必须包含核心关键词，因为标题权重大于内容、标签和话题。
*   封面可以借鉴已验证的高点击率封面。
*   如果是视频笔记，将文案复制到内容区，方便系统识别。
*   内容避免直接营销广告，否则可能连薯条审核都无法通过。

**第二种：骗评论起号法**
这个方法的核心是利用评论来快速拉高CES得分。因为在CES公式中，一条评论价值4分，相当于4个点赞或2个收藏。
我们可以设计内容引导用户评论。最简单的方法是 **“极限二选一”** ，即让用户在A和B之间做选择。各行各业都可以套用此模板。
另一个方法是 **“简答题”** ，直接向用户提问，如“说出你最喜欢的xx”。核心在于降低用户的评论成本，让用户无需思考即可参与。

## 小红书内容运营：流量与转化的平衡

最后，我们来探讨如何通过内容运营实现流量增长与商业转化的平衡。内容可以分为两种类型。

**第一种：流量型内容**
这类内容流量大，但转化率较低，属于“宽口”内容。例如，做装修的账号发布“2024年最流行的装修风格”，目标受众非常广泛。

**第二种：营销型内容**
这类内容转化率高，但流量相对较小，属于“窄口”内容。例如，同样是装修账号，发布“大平层客厅怎么装修”，则只针对大户型业主。

正确的策略是两者配合使用：
*   **流量型内容**负责搞流量、增加曝光、拉高账号基础权重。
*   **营销型内容**负责做转化、成交订单。

具体应用时，可以理解为“广撒网，深捕捞”。先用流量型内容触达潜在用户（如对西藏感兴趣的人），再用营销型内容转化目标客户（如决定去西藏并寻找定制团的人）。只做流量型内容会导致有曝光无转化；只做营销型内容则无法获取新客户。
