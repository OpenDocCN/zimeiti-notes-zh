# 视频号运营：P25：7.如何给视频抽帧补帧 🎬

在本节课中，我们将学习视频剪辑中的一个重要技巧——抽帧与补帧。这个操作可以帮助我们调整视频时长，同时避免因直接剪辑而导致的画面跳跃或卡顿问题。掌握这个方法，能让你的视频剪辑更加精细和专业。

上一节我们介绍了视频的“掐头去尾”基础剪辑，本节中我们来看看如何通过抽帧和补帧来进一步优化视频的流畅度。

## 理解帧率与操作原理

首先，我们需要理解视频是由一帧一帧的连续画面组成的。在常见的视频格式中，**每秒通常包含30帧（30FPS）**。这意味着每一秒的视频由30张静态图片快速播放而成。

`帧率 (FPS) = 每秒帧数`

“抽帧”是指从视频中删除某一帧画面；“补帧”则是指复制并插入某一帧画面。这两个操作配合使用，可以在不显著影响视频观感的前提下，微调视频的时长和节奏。

## 抽帧操作步骤

以下是进行抽帧的具体步骤：

1.  **放大时间轴**：将视频导入剪辑软件后，首先将时间轴刻度拉到最精细的程度，以便能清晰地看到每一帧。
2.  **定位帧**：找到你想要进行操作的片段。例如，视频开头第00秒到第01秒之间，就包含了30帧。
3.  **分割与删除**：在某一帧（如第20帧）的位置进行分割，紧接着在下一帧（第21帧）的位置再次分割。这样，你就单独选中了这一帧画面。
4.  **删除帧**：将这一小段（即这一帧）删除。这个删除单帧画面的过程，就叫做“抽帧”。

对于一个十几秒的视频，通常**抽3帧左右**即可，过多可能会影响流畅度。

## 补帧操作步骤

完成抽帧后，我们可能需要在其他位置补帧以保持视频总时长的平衡或解决卡顿。以下是补帧的步骤：

1.  **选择参考帧**：在视频中找到一个画面稳定、动作幅度不大的单帧片段，用同样的分割方法将其单独分离出来。
2.  **复制帧**：选中这一帧片段，使用“复制”功能。
3.  **粘贴帧**：将复制好的帧，粘贴到视频中你认为合适的其他位置（例如，在另一个动作平缓处进行分割并粘贴插入）。这个复制并插入帧的过程，就是“补帧”。

补帧的数量通常与抽帧数量保持一致，例如抽3帧则补3帧，以确保视频的整体长度和流畅度不受影响。

## 操作要点与总结

![](img/8f835726be000ee78eaa4bc4cd10510a_1.png)

本节课中我们一起学习了视频剪辑中的抽帧与补帧技术。我们来总结一下核心要点：

![](img/8f835726be000ee78eaa4bc4cd10510a_2.png)

*   **核心目的**：微调视频时长，优化流畅度，避免直接裁剪导致的跳跃感。
*   **关键概念**：视频由连续帧构成，每秒30帧（30FPS）是常见标准。
*   **操作流程**：先通过精细分割删除单帧（抽帧），再复制类似的单帧插入到其他位置（补帧）。
*   **注意事项**：操作需在时间轴放大至最精细模式下进行；抽帧和补帧的数量不宜过多，一般少量（如3次）即可；应选择画面变化小的帧进行操作，对视频整体观感影响最小。

只要遵循上述步骤，你就能轻松完成视频的抽帧与补帧，使你的视频内容更加完美。
