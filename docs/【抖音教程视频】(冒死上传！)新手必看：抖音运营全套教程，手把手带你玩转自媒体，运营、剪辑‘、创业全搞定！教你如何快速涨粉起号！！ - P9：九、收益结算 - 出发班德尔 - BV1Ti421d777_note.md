# 抖音运营教程：P9：收益结算

在本节课中，我们将要学习影视推广项目的收益结算方式，特别是如何通过“中视频计划”获取双份收益，并了解合作的基本要求。

上一节我们介绍了影视推广的基本操作，本节中我们来看看如何最大化我们的收益。

现阶段，我自己的团队也在做影视推广。我们团队每个成员手里至少有十个账号在操作。即使我们批量操作，账号数量依然不够，这些资源我们一个团队无法完全消化。因此，我想寻找一些伙伴，一起来做这个项目。

我可以为你对接资源，但有一个要求：你必须严格按照我的方法制作视频。这一点能做到吗？

我们属于合作共赢的关系。我将资源对接给你，你按照我的方法操作并获取奖金。未来有新的资源，同样会对接给我们团队。这是我们共同期望的结果。

![](img/92cb8e6e5904ec32e0e6665ff741450c_5.png)

**合作模式**：`资源对接 + 标准化操作 = 共同收益`

![](img/92cb8e6e5904ec32e0e6665ff741450c_7.png)

这一点是我所强调的：如果你想在这个平台上取得成功，拥有资源和人脉至关重要。

![](img/92cb8e6e5904ec32e0e6665ff741450c_9.png)

除了单纯的视频剪辑，如果你愿意在视频中添加解说，我们一个视频可以赚取双份收益。

以下是关于“中视频计划”的说明：

*   **平台**：该计划由字节跳动旗下的抖音、今日头条和西瓜视频三个平台联合推出。
*   **性质**：它是针对自媒体视频创作者的一个更高流量分成机制。
*   **规则**：参与计划后，你发布的一个视频，可以在三个平台同时获得播放收益结算。

对于影视推广来说，其优势在于拥有版权。将剪辑好的视频添加解说内容后，通过抖音平台发布：
1.  首先，系统会结算影视推广项目的播放量奖金。
2.  其次，通过“中视频计划”入口，抖音、今日头条、西瓜视频三个平台会共同结算该视频的播放收益。

**双份收益公式**：`影视推广奖金 + 中视频计划分成 = 单视频双份收益`

![](img/92cb8e6e5904ec32e0e6665ff741450c_11.png)

如果你想尝试解说，首先需要强调一点：如果我们想做影视解说，必须确保内容拥有合法版权，这是参与中视频计划的前提。

![](img/92cb8e6e5904ec32e0e6665ff741450c_13.png)

![](img/92cb8e6e5904ec32e0e6665ff741450c_14.png)

本节课中我们一起学习了收益结算的核心方式。关键在于理解并利用“中视频计划”，将拥有版权的影视剪辑内容与解说结合，从而实现一个视频在多个平台获取双份收益。同时，稳定的资源对接和标准化操作是达成合作共赢的基础。
