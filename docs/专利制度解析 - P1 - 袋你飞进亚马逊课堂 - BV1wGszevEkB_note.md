# 亚马逊卖家合规指南：P1：全球专利制度解析

![](img/547f35d151b3a6de7b3612ab499f9e1b_0.png)

在本节课中，我们将学习全球主要市场的专利制度，包括专利的定义、类型、保护期限以及亚马逊卖家应如何确保合规经营，避免侵权风险。

## 概述：什么是专利？

专利是创新者对其发明创造享有的**独占权**。对于亚马逊卖家而言，在向不同国家销售产品前，理解目标市场的专利法规至关重要。

## 专利的三种主要类型

上一节我们介绍了专利的基本概念，本节中我们来看看专利的具体分类。专利主要分为以下三种类型，它们在保护对象和授权条件上有所不同。

以下是三种专利类型的核心区别：

1.  **发明专利**
    *   **保护对象**：针对产品、方法或其改进所提出的**新的技术方案**。
    *   **特点**：创造性要求高，审查严格。

2.  **实用新型专利**
    *   **保护对象**：针对产品的**形状、构造**所提出的适于实用的新技术方案。
    *   **特点**：通常被称为“小专利”，创造性要求相对较低，审查周期较短。

3.  **外观设计专利**
    *   **保护对象**：针对产品的**整体或局部外观**，包括形状、图案、色彩等所作出的富有美感的新设计。
    *   **特点**：不保护技术功能，仅保护视觉设计。

## 🌎 美国专利制度解析

![](img/547f35d151b3a6de7b3612ab499f9e1b_2.png)

了解了专利的基本类型后，我们进入具体国家的制度分析。首先来看美国市场，这是亚马逊卖家的核心战场之一。

*   **发明专利**：保护期限为**20年**，自申请日起计算。
*   **实用新型专利**：美国没有专门的实用新型专利制度。
*   **外观设计专利**：保护期限为**15年**，自授权日起计算。

**卖家行动指南**：
在美国销售产品前，必须在**美国专利商标局（USPTO）** 数据库进行专利检索，公式化表示为：`产品上市前 = 完成USPTO专利检索`，以确保不侵犯现有专利权。

## 🇪🇺 欧盟专利制度解析

接下来，我们将目光转向欧盟市场。欧盟的专利制度与美国有所不同，卖家需要特别注意其统一与并行的特点。

*   **发明专利**：可通过**欧洲专利局（EPO）** 统一申请，在指定成员国生效，保护期限为**20年**。
*   **实用新型专利**：部分欧盟成员国提供保护，但制度因国家而异，需单独了解。
*   **外观设计专利**：在**欧盟知识产权局（EUIPO）** 注册，获得欧盟范围内统一保护，保护期限为**25年**。

**卖家行动指南**：
除了专利检索，还需关注产品上的**CE标志**等合规要求，确保符合欧盟相关法规。

## 🛡️ 亚马逊卖家的专利合规策略

前面我们分别了解了美国和欧盟的专利制度，本节我们将整合这些知识，为亚马逊卖家总结一套可行的合规操作流程。

以下是卖家应遵循的四个关键步骤：

![](img/547f35d151b3a6de7b3612ab499f9e1b_4.png)

1.  **认知差异**：首先必须认识到各国专利制度存在差异，不能一概而论。
2.  **全面检索**：在产品上架前，对目标市场进行全面的专利搜索与侵权风险评估。
3.  **制定策略**：根据检索结果，制定产品开发、采购或销售策略，主动规避潜在侵权产品。
4.  **积极应对**：若收到侵权指控，应积极、专业地应对，分析对方专利的有效性及自身产品是否构成侵权。

![](img/547f35d151b3a6de7b3612ab499f9e1b_5.png)

## 总结

本节课中，我们一起学习了专利的三种基本类型（发明专利、实用新型专利、外观设计专利），并深入解析了美国和欧盟两大市场的专利制度特点与保护期限。最后，我们为亚马逊卖家梳理了从认知差异、全面检索到制定策略和积极应对的完整合规流程。掌握这些知识，能帮助你在全球电商市场中更安全、更稳健地经营业务。
