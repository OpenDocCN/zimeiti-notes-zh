# 剪映教程：P14：抠像与蒙版功能详解 🎬

![](img/5e5a6eae6950421eda0af3ecab927ee5_0.png)

在本节课中，我们将学习剪映中两个非常有趣且强大的工具：**抠像**和**蒙版**。我们将通过制作一系列富有创意和视觉冲击力的视频效果，来掌握这两个工具的核心用法。课程内容将涵盖抠像的三种主要方法以及蒙版的各种应用，确保即使是初学者也能轻松上手。

## 概述：认识抠像与蒙版

上一节我们介绍了关键帧动画，本节中我们来看看如何利用抠像和蒙版进行画面合成。抠像功能允许我们将视频中的特定主体（如人物、物体）从原始背景中分离出来。蒙版功能则能帮助我们以各种形状（如线性、圆形、矩形）来组合或隐藏画面的不同部分，实现无缝的镜头融合。

## 第一部分：抠像功能详解

抠像，顾名思义，就是把画面中的“像”（主体）从背景中“抠”出来。在剪映中，主要有三种抠像方法。

### 1. 色度抠像

色度抠像是利用纯色背景（通常是绿色或蓝色）进行快速抠像的方法。其原理是吸取背景颜色并将其“抠掉”，从而保留前景主体。

**操作步骤如下：**
1.  将带有纯色背景（如绿幕）的素材拖入时间线。
2.  选中素材，在右侧控制面板中找到“抠像”选项。
3.  选择“色度抠像”。
4.  使用“取色器”吸取画面中的背景颜色。
5.  调整“强度”和“阴影”滑块，直到主体被干净地抠出，背景消失。

例如，要将一个绿幕恐龙放入电视画面中：
*   对恐龙素材使用“色度抠像”，吸取绿色并调整参数。
*   将抠好的恐龙素材叠加到电视画面上方，调整大小和位置。
*   利用关键帧，可以为恐龙制作从电视中走出的动画效果。

### 2. 自定义抠像

当背景不是纯色，或者需要更精细地控制抠取区域时，可以使用自定义抠像。它通过手动涂抹来选择要保留或移除的区域。

**操作步骤如下：**
1.  选中素材，在“抠像”选项中选择“自定义抠像”。
2.  使用“智能画笔”工具，在需要保留的主体上涂抹。软件会自动识别并选中相近区域。
3.  如果多选了不需要的区域，可以使用“智能橡皮擦”或“橡皮擦”工具进行擦除修正。
4.  完成后，点击“应用效果”。

此方法适用于背景复杂、主体清晰的镜头，例如从街景中抠出人物。

### 3. 智能抠像

智能抠像是剪映提供的自动化工具，专门用于识别并抠取视频中的人物图像。

**重要说明：**
*   该功能**仅支持对人物图像的智能识别**。对于动物、物体或无人的场景，它可能无法工作或效果不佳。
*   其工作原理是识别人物五官和轮廓。对于此类镜头，它能提供快速便捷的抠像方案。

如果镜头主体不是人物（例如一只熊），则应使用“自定义抠像”来获得更准确的结果。

## 第二部分：蒙版功能详解

蒙版功能通过不同的形状来显示或隐藏图层的部分内容，常用于实现两个或多个画面的创意合成与过渡。

剪映提供了多种蒙版形状：线性、镜面、圆形、矩形、爱心和星形。

**通用操作与参数：**
*   **添加蒙版**：选中轨道上的素材，在右侧面板中选择“蒙版”，然后选择一种形状。
*   **调整位置与大小**：可以直接拖动蒙版边框的控制点，也可以在参数面板中精确调整位置、旋转和大小。
*   **羽化**：调整“羽化”值可以使蒙版边缘变得柔和，让合成效果更加自然。
*   **圆角（矩形蒙版特有）**：调整“圆角”参数可以使矩形的直角变为圆角。

以下是几种常用蒙版的应用示例：

**线性蒙版**：常用于替换天空或制作上下/左右分屏效果。例如，将一段实拍风景的天空，用另一段卡通天空素材替换。
1.  将两段素材上下叠加。
2.  对上层素材（实拍风景）应用“线性蒙版”。
3.  旋转蒙版线至地平线位置，并适当增加羽化值，使交界处过渡自然。

**镜面蒙版**：可以创造出类似“望远镜”或中间保留、两边遮挡的效果，适合用于突出画面中心主体。

**圆形/矩形蒙版**：常用于制作“画中画”或局部聚焦效果。例如，将一个视频画面以圆形或矩形的形式嵌入到另一个背景视频中。

## 第三部分：综合案例实战

现在，我们将综合运用抠像和蒙版，复现课程开始时展示的创意效果。

### 案例一：人物从电视机中爬出
1.  **抠出人物**：将绿幕拍摄的人物素材导入。使用“色度抠像”吸取绿色，初步抠除背景。对于边缘残留的绿色，可以结合“调节”功能，降低绿色饱和度。
2.  **处理穿帮**：由于绿幕可能无法完全覆盖，人物边缘或周围会有穿帮。此时添加一个“矩形蒙版”，框选出电视屏幕区域，只显示该区域内的人物，隐藏屏幕外穿帮的部分。
3.  **合成背景**：将抠好并处理好的人物素材，放置到电视背景素材的上方轨道，调整位置，使人物手臂搭在电视框上，形成爬出的视觉效果。

### 案例二：无实物托举锤子
1.  **分离元素**：分别导入“人物做动作”和“绿幕锤子”两段素材。
2.  **抠出锤子**：对锤子素材使用“自定义抠像”或“色度抠像”，将其干净地抠出。
3.  **制作关键帧动画**：将抠出的锤子素材叠加到人物素材上方。根据人物手部动作，为锤子素材的位置添加关键帧，让锤子随着人物手势“凭空”升起、落下，形成被操控的错觉。
4.  **色彩统一**：通过“调节”功能，微调锤子的亮度、饱和度，使其与人物所在的环境光影更加匹配。

### 案例三：车内飞行特效
1.  **抠出车内景**：导入在绿幕前拍摄的车内人物素材。使用“自定义抠像”精细勾勒出人物、座椅、方向盘等需要保留的部分。
2.  **去除绿幕**：同样使用“色度抠像”和“调节”（降低绿色饱和度）来彻底清除背景的绿色。
3.  **合成动态背景**：将抠好的车内景素材，叠加到一段动态的天空或飞行背景素材上。调整车内景的大小和位置，使其与背景的运动（如晃动）同步，营造出在天空中飞驰的震撼效果。

本节课中，我们一起深入学习了剪映的**抠像**与**蒙版**两大核心合成功能。
*   **抠像**让我们能够将主体从原背景中分离，主要方法有：利用纯色背景的**色度抠像**、适用于复杂背景的**自定义抠像**以及针对人物识别的**智能抠像**。
*   **蒙版**则为我们提供了多种形状（线性、圆形、矩形等）的遮罩工具，通过调整位置、羽化和旋转，可以实现画面间的无缝拼接与创意融合。
