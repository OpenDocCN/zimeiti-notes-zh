上一节我们介绍了视频拍摄的要点，本节中我们来看看如何将拍摄好的素材剪辑成完整的作品。我们将使用抖音官方出品的“剪映”App进行手机端剪辑教学。

![](img/146eb04a487ab1c792e3d87cab0ae63a_7.png)

## 第一步：导入素材与基础调整

首先，打开剪映App，点击“开始创作”按钮。

![](img/146eb04a487ab1c792e3d87cab0ae63a_9.png)

![](img/146eb04a487ab1c792e3d87cab0ae63a_1.png)

![](img/146eb04a487ab1c792e3d87cab0ae63a_11.png)

选择你需要剪辑的视频片段，点击“添加”。如果你的视频由多个片段组成，建议先导入并剪辑好第一条，再通过点击“+”号依次导入后续片段，这样可以避免混乱。

![](img/146eb04a487ab1c792e3d87cab0ae63a_13.png)

![](img/146eb04a487ab1c792e3d87cab0ae63a_5.png)

导入后，第一步是检查并调整画面构图与比例。以下是操作步骤：

1.  点击选中时间轴上的视频片段。
2.  向右滑动底部工具栏，找到并点击“编辑”按钮（图标通常为方块）。
3.  点击“旋转”功能，将画面调整到正确的方向。
4.  如需裁剪画面比例，可点击“裁剪”，选择如 **9:16**（竖屏）、**16:9**（横屏）或 **1:1**（正方形）等合适比例。

![](img/146eb04a487ab1c792e3d87cab0ae63a_15.png)

![](img/146eb04a487ab1c792e3d87cab0ae63a_17.png)

![](img/146eb04a487ab1c792e3d87cab0ae63a_9.png)

## 第二步：裁剪视频内容

![](img/146eb04a487ab1c792e3d87cab0ae63a_19.png)

调整好画面后，需要剪掉视频中无用的部分，如开头沉默、结尾多余动作等。

![](img/146eb04a487ab1c792e3d87cab0ae63a_21.png)

![](img/146eb04a487ab1c792e3d87cab0ae63a_23.png)

操作的核心逻辑是：**选中 -> 分割 -> 删除**。

![](img/146eb04a487ab1c792e3d87cab0ae63a_25.png)

1.  将时间线（白色竖线）移动到需要裁剪的位置。
2.  点击选中视频片段。
3.  点击底部工具栏的“分割”按钮，视频将在时间线处被切成两段。
4.  点击选中需要删除的片段，点击“删除”按钮。

![](img/146eb04a487ab1c792e3d87cab0ae63a_27.png)

![](img/146eb04a487ab1c792e3d87cab0ae63a_15.png)

![](img/146eb04a487ab1c792e3d87cab0ae63a_29.png)

重复以上步骤，仔细检查并剪掉所有不必要的部分，使视频内容紧凑流畅。

![](img/146eb04a487ab1c792e3d87cab0ae63a_31.png)

![](img/146eb04a487ab1c792e3d87cab0ae63a_33.png)

## 第三步：添加与编辑背景音乐

![](img/146eb04a487ab1c792e3d87cab0ae63a_35.png)

视频内容裁剪完毕后，接下来为其添加背景音乐。

![](img/146eb04a487ab1c792e3d87cab0ae63a_37.png)

1.  点击底部工具栏的“音频”按钮。
2.  添加音乐有两种常用方式：
    *   **提取音乐**：如果你已有带配音的视频文件，可点击“提取音乐”，直接导入该视频的音频。
    *   **添加音乐**：点击“音乐”，剪映内置了丰富的音乐库。你可以根据分类（如美食、美妆、vlog）寻找，或直接搜索歌曲名称。点击歌曲旁边的“使用”按钮即可添加。

![](img/146eb04a487ab1c792e3d87cab0ae63a_39.png)

![](img/146eb04a487ab1c792e3d87cab0ae63a_21.png)

![](img/146eb04a487ab1c792e3d87cab0ae63a_41.png)

添加音乐后，可能需要对音频进行剪辑：

![](img/146eb04a487ab1c792e3d87cab0ae63a_43.png)

*   **裁剪长度**：如果音乐过长，同样使用 **选中 -> 分割 -> 删除** 的方法，剪掉多余部分。
*   **调整音量**：选中音频轨道，点击“音量”，拖动滑块即可调整背景音乐大小，避免盖过人声。

![](img/146eb04a487ab1c792e3d87cab0ae63a_45.png)

![](img/146eb04a487ab1c792e3d87cab0ae63a_33.png)

![](img/146eb04a487ab1c792e3d87cab0ae63a_47.png)

## 第四步：自动识别与添加字幕

![](img/146eb04a487ab1c792e3d87cab0ae63a_49.png)

手动添加字幕效率很低，剪映的“识别字幕”功能可以极大提升效率。

![](img/146eb04a487ab1c792e3d87cab0ae63a_51.png)

1.  点击底部工具栏的“文本”按钮。
2.  选择“识别字幕”。
3.  在弹出窗口中，选择“仅视频”（避免识别到背景音乐里的歌词），然后点击“开始识别”。

![](img/146eb04a487ab1c792e3d87cab0ae63a_37.png)

![](img/146eb04a487ab1c792e3d87cab0ae63a_53.png)

系统会自动生成字幕，并匹配到对应的时间点。识别完成后：

![](img/146eb04a487ab1c792e3d87cab0ae63a_55.png)

*   **检查错别字**：双击字幕条，即可修改文本内容。
*   **美化字幕**：选中字幕，可以更改字体、颜色、样式等，使其与视频风格更搭配。

![](img/146eb04a487ab1c792e3d87cab0ae63a_57.png)

![](img/146eb04a487ab1c792e3d87cab0ae63a_45.png)

![](img/146eb04a487ab1c792e3d87cab0ae63a_59.png)

![](img/146eb04a487ab1c792e3d87cab0ae63a_61.png)

## 第五步：导出高清视频

在导出最终视频前，需要进行关键设置以保证清晰度。

1.  点击右上角的“导出”按钮。
2.  在导出设置中，调整参数：
    *   **分辨率**：建议选择最高选项（如1080P），数值越高越清晰。
    *   **帧率**：设置为 **30 fps**，这是抖音平台最合适的帧率。
3.  点击“导出”按钮，等待渲染完成。**导出过程中请勿锁屏或切换App**，以免中断。

![](img/146eb04a487ab1c792e3d87cab0ae63a_63.png)

![](img/146eb04a487ab1c792e3d87cab0ae63a_51.png)

![](img/146eb04a487ab1c792e3d87cab0ae63a_65.png)

导出完成后，可以直接通过剪映分享到抖音发布。

![](img/146eb04a487ab1c792e3d87cab0ae63a_67.png)

“爱提词”是一款提词器软件，可以让你在拍摄时看着台词念，无需背诵。

![](img/146eb04a487ab1c792e3d87cab0ae63a_73.png)

![](img/146eb04a487ab1c792e3d87cab0ae63a_75.png)
