# 新媒体运营AI实战：01.10.2：AI训练：撰写符合小红书风格的文案 ✍️🤖

在本节课中，我们将学习如何训练AI（如ChatGPT）来撰写符合小红书平台风格的文案。上一节我们探讨了如何让AI生成符合平台风格的爆款标题，本节我们将深入核心，学习如何让AI理解并创作出符合平台调性的完整内容。

## 核心思路：让AI理解平台风格

![](img/f1db4ad489e925f49717ce137be9b2f0_1.png)

要让AI写出符合特定平台风格的内容，关键在于“投喂”资料。AI需要通过学习大量该平台的爆款文案，来分析和记忆其独特的语言风格、常用词汇、语气以及内容结构。

![](img/f1db4ad489e925f49717ce137be9b2f0_3.png)

以下是训练AI理解平台风格的核心步骤：

![](img/f1db4ad489e925f49717ce137be9b2f0_5.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_7.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_9.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_10.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_11.png)

1.  **提供指令**：明确告知AI你的目标。例如，要求它分析你提供的文案，并学习其中的风格、语气和平台专用语。
2.  **投喂案例**：向AI提供多篇具有代表性的平台爆款文案作为学习材料。
3.  **获取反馈**：要求AI在每次学习后，给出它对文案风格的分析，以确保它真正理解了。
4.  **应用创作**：在AI学习足够案例后，指令它结合所学，针对特定主题进行创作。

## 实战演练：训练AI撰写旅游攻略

![](img/f1db4ad489e925f49717ce137be9b2f0_19.png)

我们以“张家界旅游攻略”为例，进行一个简单的实操演示。

![](img/f1db4ad489e925f49717ce137be9b2f0_21.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_23.png)

首先，我们需要给AI清晰的指令。我们可以这样开始：

![](img/f1db4ad489e925f49717ce137be9b2f0_25.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_27.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_29.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_31.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_33.png)

```
接下来我会给你提供一些小红书上面的爆款笔记文案。你需要根据这些文案，分析小红书平台的文案撰写风格、特定的平台用语、词汇、语气等。
每当我提供一篇文案后，请告诉我你已经理解，并给出你对这篇文案的风格、语气、类型等相关分析。
```

![](img/f1db4ad489e925f49717ce137be9b2f0_35.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_37.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_39.png)

发出指令后，我们开始向AI“投喂”找到的小红书爆款旅游攻略文案。例如，我们可以提供一篇标题为《省事又省钱！张家界5天4晚不踩坑攻略》的笔记全文。

![](img/f1db4ad489e925f49717ce137be9b2f0_41.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_42.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_44.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_46.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_48.png)

AI在接收并分析后，可能会给出如下反馈：
*   **风格分析**：以个人游玩经历为主线，增加了真实感和亲近感。
*   **语气分析**：标题使用“血泪教训”、“不踩坑”等词，强调了经验分享和避坑建议，语气亲切且带有告诫意味。
*   **平台用语**：文中可能使用了“宝子们”、“真的绝了”等小红书常见口语化词汇。

![](img/f1db4ad489e925f49717ce137be9b2f0_49.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_51.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_53.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_55.png)

接着，我们继续提供第二篇、第三篇不同风格或角度的笔记（例如吐槽踩坑经历、分享小众打卡点的笔记），不断丰富AI对平台文案多样性的理解。

![](img/f1db4ad489e925f49717ce137be9b2f0_57.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_59.png)

**关键点**：在选择投喂的文案时，应注重其**独特性**，而不仅仅是通用信息。例如，避免只提供千篇一律的“第一天去哪，第二天去哪”的路线罗列，而应选择那些具有强烈个人分享色彩、情感表达或独特见解的文案，这样AI才能学到平台的核心“语气”和“网感”。

![](img/f1db4ad489e925f49717ce137be9b2f0_61.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_63.png)

## 指令AI进行创作

![](img/f1db4ad489e925f49717ce137be9b2f0_65.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_67.png)

在向AI投喂了多篇范文后，我们可以指令它进行创作。例如：

![](img/f1db4ad489e925f49717ce137be9b2f0_69.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_71.png)

```
请结合你刚才的学习，帮我撰写一篇具有爆款潜质的“张家界5日游攻略”小红书笔记。要求内容涵盖丰富的小红书风格emoji表情，并同时生成10个符合小红书旅游赛道风格的爆款标题。
```

AI会根据之前的学习生成内容。初次生成的结果可能不尽如人意，内容可能偏向笼统和诗意，而缺乏实用性的细节和真实的“避坑”建议。

![](img/f1db4ad489e925f49717ce137be9b2f0_73.png)

**问题分析与优化**：
这通常是因为我们提供的学习材料（样本）本身深度不够，或者指令不够具体。要获得更优质的输出，我们可以采用更精细的步骤：

![](img/f1db4ad489e925f49717ce137be9b2f0_75.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_77.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_79.png)

1.  **分步指令**：先让AI列出张家界所有主要景点及特点。
2.  **细化要求**：再针对“5日游”这个框架，指令AI为每一天规划路线，并针对每个景点补充“推荐理由”、“必看亮点”和“避坑提示”。
3.  **结合标题库**：在最终生成笔记时，可以结合上一节课的标题关键词库，让AI生成的标题更具爆款潜力。

![](img/f1db4ad489e925f49717ce137be9b2f0_81.png)

## 模型差异与持续优化

![](img/f1db4ad489e925f49717ce137be9b2f0_83.png)

需要注意的是，使用ChatGPT 3.5等免费模型与4.0等更先进的模型，在理解复杂指令和生成内容的质量上会有显著差异。付费模型通常表现更佳。

![](img/f1db4ad489e925f49717ce137be9b2f0_85.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_87.png)

此外，对于不同领域（如美妆、知识分享、科技测评），训练方法相通，但需要寻找并投喂对应领域的优质爆款文案。素材可以从平台站内、百度或其他内容平台搜索获取。

![](img/f1db4ad489e925f49717ce137be9b2f0_89.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_91.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_93.png)

![](img/f1db4ad489e925f49717ce137be9b2f0_95.png)

**核心要义**：训练AI的本质，是将其从一个“通才”变成你所在垂直领域的“专家”。它本身知识渊博，但需要通过我们持续、有针对性的“投喂”和指令，才能精准理解并满足我们的具体需求。

## 总结

本节课我们一起学习了训练AI撰写平台化文案的完整流程。关键在于通过**提供明确指令**和**投喂高质量样本**，教会AI理解特定平台（如小红书）的独特行文风格、用语习惯和内容结构。这是一个需要不断调试和优化的过程，选择多样的样本、给出具体的创作要求，是获得满意结果的重要保障。
