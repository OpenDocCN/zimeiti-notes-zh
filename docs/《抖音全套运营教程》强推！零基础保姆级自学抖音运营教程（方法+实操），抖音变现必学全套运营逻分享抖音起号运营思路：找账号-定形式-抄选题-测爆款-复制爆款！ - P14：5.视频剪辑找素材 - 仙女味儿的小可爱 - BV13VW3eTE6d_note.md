# 抖音全套运营教程：P14：视频剪辑找素材 🎬

在本节课中，我们将要学习视频剪辑类内容创作的第一步：如何高效、合规地寻找视频素材。这是制作原创内容、避免搬运违规的关键环节。

视频剪辑类内容涵盖多种类型，例如电影剪辑、电视剧剪辑或综艺节目剪辑。

---

## 素材来源平台

![](img/7a777d1f4dad0ea99fc35cf589fce690_1.png)

以下是寻找视频素材的三个主要途径。

![](img/7a777d1f4dad0ea99fc35cf589fce690_3.png)

### 1. 视频与自媒体平台

所有视频平台和自媒体平台都是寻找素材的重要来源。下图展示了30多个可用的自媒体平台。

![](img/7a777d1f4dad0ea99fc35cf589fce690_1.png)

这些平台均能提供丰富的视频素材。

### 2. 专业影视与素材网站

第二个途径是专业的影视网站或素材集合网站。例如，豆瓣电影这类网站提供了大量影视信息。

此外，还有专门整理的无版权影视素材网站，数量超过50个。这些网站的素材通常没有版权限制，可以安全使用。

如果不想局限于影视类，还可以使用其他类型的素材网站。这些网站整理了超过60个资源站，涵盖旅游、图片、音效、体育、美食、动漫、综艺、搞笑以及国内外各类视频素材。

### 3. 影视推广平台

第三个途径是影视推广平台。在抖音或快手等平台，普通视频通常只能用于涨粉，难以直接通过播放量变现。

但某些影视推广平台允许创作者使用其提供的影视素材，并能够依靠视频播放量获得奖金。这个平台的具体操作将在后续重点讲解。

## 如何具体查找素材

![](img/7a777d1f4dad0ea99fc35cf589fce690_7.png)

上一节我们介绍了素材的来源，本节中我们来看看如何具体操作。正确查找素材的方式不是直接下载完整视频，而是寻找符合主题的片段。

以下是具体操作步骤：

1.  **打开搜索引擎**：例如，打开手机百度。
2.  **搜索自媒体平台**：以“西瓜视频”为例进行搜索。

    ![](img/7a777d1f4dad0ea99fc35cf589fce690_5.png)

![](img/7a777d1f4dad0ea99fc35cf589fce690_9.png)

3.  **进入平台网页版**：点击搜索结果进入其网页版。

    ![](img/7a777d1f4dad0ea99fc35cf589fce690_7.png)

![](img/7a777d1f4dad0ea99fc35cf589fce690_11.png)

4.  **使用关键词搜索**：确定视频主题和核心人物。例如，想做综艺类内容，可以搜索“大杨哥”、“小杨哥”等网红关键词。

    ![](img/7a777d1f4dad0ea99fc35cf589fce690_9.png)

    搜索后即可找到大量相关视频素材。

    ![](img/7a777d1f4dad0ea99fc35cf589fce690_11.png)

**关键注意事项**：直接下载整个视频属于“搬运”行为，目前平台审核严格，95%的搬运内容无法通过审核。

正确的做法是从众多视频中，提取符合自己视频主题的片段。每个片段的时长应控制在 **10秒钟** 左右。这样剪辑出的视频在平台审核时更容易通过，不易违规。

**举例说明**：如果想制作关于明星沈腾的内容，同样使用关键词“沈腾”进行搜索。

![](img/7a777d1f4dad0ea99fc35cf589f4dad0ea99fc35cf589fce690_13.png)

用此方法可以找到非常丰富的素材。

## 确定主题与准备工具

![](img/7a777d1f4dad0ea99fc35cf589fce690_13.png)

素材找到后，下一步是制作视频。制作视频需要用到剪辑软件，最常用的手机剪辑软件是 **剪映**。

在开始剪辑前，必须首先确定视频的 **主题**。

例如，计划制作一个综艺类视频，主题可以定为“沈腾的搞笑名场面”。那么，所有选取的素材片段都必须包含沈腾这个人，并且围绕“搞笑”这一核心。

![](img/7a777d1f4dad0ea99fc35cf589fce690_15.png)

主题是视频的灵魂，它决定了素材筛选和内容剪辑的方向。

本节课中我们一起学习了为视频剪辑寻找素材的三大途径、具体搜索方法以及确定视频主题的重要性。核心要点是：**通过关键词在各大平台搜索素材，并截取约10秒的片段进行二次创作，而非直接搬运，同时明确视频主题以指导素材筛选。** 掌握这些，你就能迈出原创视频制作的第一步。
