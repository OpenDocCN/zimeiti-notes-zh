# 小红书运营教程：P19：如何发笔记挂车、挂团购券、二次编辑与定时发布 🛒📅

在本节课中，我们将学习商家、买手和实体店如何在小红书笔记中挂载商品链接或团购券。课程将涵盖发布流程、不同身份的挂载方式、定时发布功能以及笔记爆火后的二次编辑方法。

---

## 发布前的准备

无论你是商家、买手还是实体店，发布笔记的第一步是相同的。打开小红书App，点击底部导航栏的“+”号，上传你的作品（图片或视频）。

上传完成后，进入编辑页面填写标题和文案。关于如何撰写优质标题和文案，可以参考本教程的其他章节。本节我们重点关注如何挂载商品或团购链接。

---

## 如何挂载商品或团购链接

在笔记编辑页面，标题和文案区域的下方，你会看到一个名为“添加商品”或“门店推广”的选项。点击这个选项，你将进入链接挂载界面。

根据你的身份和目的，选择以下三种方式之一：

*   **店内商品**：适用于**开通了小红书店铺的商家**。选择此项，可以直接挂载你自己店铺中已上架的商品。
    *   **操作**：点击“店内商品” -> 从你的店铺商品列表中选择目标商品 -> 确认添加。
*   **商品合作**：适用于**粉丝数超过1000的买手**或**达人**。选择此项，可以进入选品中心，挂载与其他商家合作推广的商品。
    *   **操作**：点击“商品合作” -> 前往选品中心 -> 浏览并选择你想推广的商品 -> 申请合作并添加。
*   **探店合作**：适用于**本地实体店商家**或**探店达人**。选择此项，可以挂载本地商户的团购优惠券。
    *   **操作**：点击“探店合作” -> 进入探店广场 -> 搜索或浏览目标团购券 -> 点击“加入合作”并添加。

**核心概念区分**：
*   **商家**：拥有自家店铺，使用“店内商品”。
*   **买手/达人**：无自有店铺，帮他人卖货，使用“商品合作”。
*   **实体店/探店达人**：推广本地团购，使用“探店合作”。

链接成功添加后，会在笔记左下角显示。商品链接显示为“购物车”，团购券链接显示为“优惠团购”。

---

## 如何使用定时发布功能

如果你不希望笔记立即发布，可以使用定时发布功能。

在笔记编辑页面最下方，找到并点击“高级选项”。在弹出的菜单中，开启“定时发布”开关。

系统会允许你设置一个未来的日期和时间。例如，你可以设置为 `2024-05-20 23:59`。设置完成后点击“完成”。

笔记将保存为草稿，并在你设定的时间自动发布。

---

## 如何进行二次编辑

当一篇已发布的笔记数据表现很好时，你可以通过二次编辑来优化它，例如补充挂载商品链接。

1.  进入你的小红书个人主页，找到需要编辑的笔记。
2.  点击笔记右上角的“...”菜单，选择“编辑”。
3.  进入编辑页面后，你可以像发布新笔记时一样，在“添加商品”或“门店推广”处挂载新的商品或团购链接。
4.  完成修改后，点击“发布”即可更新笔记。

**注意**：二次编辑并发布后，笔记的发布时间不会改变，但内容会更新为最新版本。
