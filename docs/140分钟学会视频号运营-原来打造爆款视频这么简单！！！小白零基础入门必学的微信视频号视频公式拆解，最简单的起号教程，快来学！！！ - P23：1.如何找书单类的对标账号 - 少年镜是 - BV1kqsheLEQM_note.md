# 短视频运营：P23：如何寻找书单类对标账号 🔍

在本节课中，我们将学习如何在抖音平台上，为你的短视频账号（特别是书单类）寻找合适的对标账号。掌握这个方法，能帮助你快速了解目标领域，并找到可参考的模仿对象。

上一节我们介绍了寻找对标账号的重要性，本节中我们来看看具体如何操作。

## 通过“关注”功能寻找对标账号

第一种方法是利用抖音的推荐算法。当你找到一个符合你预期的账号时，可以通过关注它来触发系统推荐。

![](img/92e288d221acecf3038285dec7c120fd_1.png)

以下是具体操作步骤：

1.  **搜索关键词**：在抖音搜索框输入你想做的领域关键词，例如“书单”。
2.  **筛选目标账号**：从搜索结果中，找到一个视频类型与你预期相符（例如非真人出镜的书单视频）的账号。
3.  **进入主页并关注**：点击该账号头像进入其主页，浏览视频确认类型符合后，点击“关注”按钮。
4.  **利用系统推荐**：点击关注后，抖音会立刻在页面底部弹出“你可能感兴趣”的账号列表。这些账号通常与刚关注的账号标签相似，是潜在的对标账号。
5.  **逐个筛选**：点开这些推荐账号，检查其作品是否是你想模仿的类型。如果是，就关注；如果不是，则返回继续查看下一个。

![](img/92e288d221acecf3038285dec7c120fd_1.png)

![](img/92e288d221acecf3038285dec7c120fd_3.png)

![](img/92e288d221acecf3038285dec7c120fd_4.png)

通过这种方式，你可以在两三分钟内关注几十个同类账号。之后，抖音的信息流会持续为你推荐更多相似账号。

## 通过“商品”搜索寻找对标账号

![](img/92e288d221acecf3038285dec7c120fd_6.png)

第二种方法是通过具体的商品（例如某本书）来反向寻找推广它的账号。这种方法尤其适合带货类内容。

![](img/92e288d221acecf3038285dec7c120fd_8.png)

以下是具体操作步骤：

1.  **搜索具体商品**：在抖音搜索框直接输入你想推广的具体商品名，例如“墨菲定律漫画版”。
2.  **切换至商品标签**：搜索结果默认是“综合”页，你需要点击顶部的“商品”标签。
3.  **按销量排序**：进入商品页面后，点击“销量”进行排序，找到销量最高的商品链接。
4.  **进入商品详情页**：点击该高销量商品，进入其商品详情页。
5.  **查看“达人推荐”**：在商品详情页中，找到“达人推荐”或“买家秀”板块。这里汇集了大量推广过该商品的短视频账号。
6.  **筛选并关注**：点开这些达人账号，观察其视频类型（通常是影视混剪、语录剪辑等）和带货数据。选择作品不多但销量高的账号作为重点模仿对象。

一个热销商品下可能关联数百个推广视频，这意味着你能一次性发现大量同领域账号，视频类型和商业模式都非常相似。

本节课中我们一起学习了两种寻找对标账号的核心方法：**通过关注触发系统推荐**和**通过商品页查看达人推荐**。掌握这两种方法，你就能高效、精准地找到大量可参考的同行账号，为后续的内容模仿与创新打下坚实基础。
