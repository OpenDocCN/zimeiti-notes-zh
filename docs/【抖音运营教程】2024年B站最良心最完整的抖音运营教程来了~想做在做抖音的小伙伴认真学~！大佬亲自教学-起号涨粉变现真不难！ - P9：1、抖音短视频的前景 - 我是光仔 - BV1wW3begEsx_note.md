# 抖音运营教程：P9：1、抖音短视频的前景 📈

![](img/3d497926d267e3168e33a2f0f9d116c5_1.png)

在本节课中，我们将要学习抖音短视频平台的发展前景，分析其市场潜力与机遇，并明确作为创作者入局的可行性。

![](img/3d497926d267e3168e33a2f0f9d116c5_3.png)

了解抖音短视频的前景，需要先通过一张图片来认识。

![](img/3d497926d267e3168e33a2f0f9d116c5_1.png)

这张图片是抖音创作者大会的一份数据报告。报告显示，有超过2200万人在抖音的合计收入超过了417亿元。抖音希望将这个数字翻一番，让创作者的总收入达到800亿元。

![](img/3d497926d267e3168e33a2f0f9d116c5_3.png)

这体现了平台巨大的流量扶持力度。同时，需要明确抖音上存在两类用户。以下是这两类用户的定义：
*   **玩家**：指单纯使用抖音娱乐，未考虑从中赚钱的用户。
*   **创作者**：指计划通过制作抖音内容来获取收益的用户。本课程的学员都属于这一类。

上一节我们区分了用户类型，本节中我们来看看为什么选择抖音平台。在众多短视频平台中，为何要专注于抖音？可以类比开实体店选址的逻辑：必须选择流量大、位置好的地方。做短视频同样需要选择一个流量庞大的平台。平台自身缺乏流量，则无法分配给创作者，创作者也就难以获利。

抖音的日活跃用户超过8亿，月活跃用户更是达到40亿人次。这是一个极其庞大的流量池。流量巨大，竞争压力是否也会很大？答案是否定的。平台上的创作者数量约为5000万。这里的“创作者”定义广泛，并非特指网红达人。例如，一位邻居老太太用手机拍摄自己做饭的视频并上传到抖音，她也算创作者。因此，90%的竞争压力实际上来自于身边的普通人。

接着，我们探讨做抖音短视频的硬件门槛。制作抖音短视频不需要购买电脑或专业摄像机。只需要一部智能手机和一个手机号，就可以注册抖音账号。抖音倡导全民通过短视频形式分享和记录生活，入门非常简单。

那么，当前有哪些具体的风口值得我们去把握呢？首先，抖音平台有明确的扶持计划，例如“抖音将帮助1000名优质的艺术创作者涨粉超过100万”。有的学员可能会担心自己不会创作。这无需忧虑，正因为存在知识盲区，才需要系统学习。学习的态度至关重要：虚心请教专业人士并不可耻，这能让人少走弯路。学习应以谦逊好学的态度进行。

市场空间巨大，竞争环境相对宽松，且内容产出门槛较低。我们正处于大数据时代，平台算法会根据用户喜好推荐内容。你喜欢看什么，抖音就会给你推荐什么。这意味着，只要内容不触犯法律和道德底线，不是低俗内容，总会有一部分受众喜欢你的作品。为什么很多初学者只有几十个粉丝？因为喜欢你作品的那部分人还没有刷到过你的视频。

因此，对于新手而言，首要目标不应急于涨粉。我们的第一个目标应该是提升视频的流量和播放量。播放量增长起来后，粉丝自然会随之而来。

本节课中我们一起学习了抖音平台的巨大前景、用户分类、选择抖音的原因、低门槛的创作条件以及新手起步的核心策略。关键在于理解平台流量逻辑，并采取先提升播放量、再自然涨粉的务实路径。
