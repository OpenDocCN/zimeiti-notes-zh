# 小红书运营实操课：P7：店铺营销工具的使用方法 🛠️

在本节课中，我们将学习如何利用小红书店铺后台的营销工具来设置商品促销活动，以提升店铺的吸引力和转化率。掌握这些工具的使用方法，是进行有效店铺运营的关键一步。

## 概述

上一节我们介绍了商品上架的基础操作。本节中，我们来看看如何为已上架的商品设置促销活动，例如折扣和优惠券，以吸引顾客并促进销售。我们将重点讲解营销中心的核心功能。

## 营销活动报名

首先，进入小红书店铺后台，点击“营销”板块。在“营销活动”中，有一个“活动报名”功能。

以下是操作步骤：
1.  点击“可报名的活动”，系统会展示当前店铺有资格参与的官方活动（例如：三八节活动）。
2.  点击具体活动进入详情页，查看报名条件（例如：店铺评分需高于4分，商品价格要求等）。
3.  评估活动优惠力度（例如：满300减40）与自身成本，决定是否报名。
    *   **建议**：大型购物节（如双11、双12）建议报名参与。日常小型活动可根据精力分配决定，建议将主要精力放在笔记内容创作上。

## 商品促销工具

在“营销管理” -> “商品促销”中，提供了多种促销工具。我们主要使用“单品促销”和“多品促销”。

*   **单品促销**：针对单个商品设置折扣。
*   **多品促销**：针对多个或全部商品设置促销规则，更为常用。

在多品促销中，最核心的两个工具是“满额立减”和“满件打折”，中间的几个工具基本可以忽略。

### 1. 满额立减设置

满额立减是指顾客订单金额达到指定数额后，可立即减免一定金额。

以下是创建步骤：
1.  点击“满额立减”，创建新活动。
2.  填写活动名称（例如：123）。
3.  设置活动时间，建议持续时间在**三个月以上**，以保持店铺促销的稳定性。
4.  选择“全部商品”参与活动（通常建议全店设置，操作更简便）。
5.  设置优惠条件。例如，店铺平均客单价为50元，可设置为“满50元减5元”。
6.  点击完成。活动创建后，可随时暂停或删除。

### 2. 满件打折设置

满件打折是指顾客购买商品达到指定件数后，可享受折扣优惠。

以下是创建步骤：
1.  点击“满件打折”，创建新活动。
2.  填写活动名称，设置活动时间（同样建议三个月以上）。
3.  设置优惠条件。例如，设置“买1件打9折”。
    *   **核心建议**：对于新店，为了快速积累销量和店铺权重，非常建议设置此折扣。虽然通过“两倍上传”法利润空间较高，但适当的折扣能有效促进前期出单。
    *   **折扣力度**：可以根据自身情况设置9折或8折。若想快速起量，可设置8折；若对利润要求较高，设置9折或不打折亦可。
4.  点击完成。

![](img/4e8d3e5b6114fbd3b758ab9fdb282c6d_1.png)

**重要提示**：“满额立减”和“满件打折”活动通常不能同时生效，只需选择其中一个设置即可。对于新店，建议优先使用“满件打折”工具。

![](img/4e8d3e5b6114fbd3b758ab9fdb282c6d_3.png)

设置成功后，全店商品将按照规则享受折扣。待店铺权重提升、销量稳定后，可以取消此活动，恢复原价销售。

## 其他工具说明

店铺后台“营销”板块下的其他工具，如“善单有礼”等，对于初期运营并非必需，可以暂时忽略。集中精力用好上述两个核心促销工具即可。

![](img/4e8d3e5b6114fbd3b758ab9fdb282c6d_7.png)

## 总结

本节课我们一起学习了小红书店铺营销工具的核心使用方法。我们主要掌握了：
1.  如何查看和报名平台官方活动。
2.  如何创建“满额立减”促销，其公式为：**订单金额 ≥ X元 → 立减 Y元**。
3.  如何创建“满件打折”促销，其公式为：**购买件数 ≥ N件 → 享受 Z折**。
4.  明确了新店初期通过设置“满件打折”（如8折或9折）来促进出单的策略，以及后期调整的思路。

![](img/4e8d3e5b6114fbd3b758ab9fdb282c6d_9.png)

合理运用这些营销工具，能够有效提升店铺的竞争力，帮助店铺度过冷启动期。
