# 抖音运营教程：P5：3.账号定位 🎯

![](img/4a06a1d45057bd713aaeb2bd646fd6d7_0.png)

在本节课中，我们将要学习抖音账号运营中至关重要的一步：**账号定位**。清晰的定位是账号快速起号、获得高权重的基础。我们将通过反思三个核心问题来构建你的账号定位框架。

![](img/4a06a1d45057bd713aaeb2bd646fd6d7_2.png)

---

上一节我们介绍了抖音的赛道选择，本节中我们来看看如何为你的账号进行精准定位。对账号定位的理解，需要反思三个问题：**你是谁**、**你是干什么的**、**为什么要关注你**。

### 确定账号类型与细分赛道

首先，我们要确定接下来要运营的账号类型。除了之前章节提到的大类赛道（即抖音定义的一级垂类），我们还需思考在一级垂类下更细分的赛道是什么。

这好比参加运动会，你报的项目是“田径”，这是你的一级垂类。而你最擅长且具有优势的项目是“百米赛跑”，这就是你的二级垂类。因此，通过定位账号的细分类目，可以确定账号的垂直类型。

在抖音中，无论任何行业，只要你的定位足够清晰，都有成功的机会。因为抖音的流量池足够庞大，一旦账号做起来，其增长和裂变速度将非常快。

### 确定账号主体

其次，我们要确定账号主体是谁。账号主体即该账号的拥有人，也是最终呈现给粉丝的账号主人公。这能让粉丝清晰地知道是在观看谁的作品。

在抖音商业类账号中，账号主体主要分为两类：
*   **商家账号**：如果账号主体是公司，即为商家账号。其内容应偏向商业化，并需不断通过内容拉近与粉丝的距离，避免过于严肃。这样既能建立信任感，也能带来好感，更符合抖音的娱乐属性。
*   **达人账号**：如果账号主体是个人，即为达人账号。这类账号首先需要解决信任问题。可以通过**IP打造**的方式，为账号树立一个积极、正面的人设。

### 明确账号内容与价值

然后，我们需要明确这个账号是做什么的。最直接的方式是通过抖音的官方认证来表明身份。

*   如果是个人达人，可以申请**黄V认证**。
*   如果是公司，可以申请**蓝V认证**。

认证后，粉丝可以清晰地看到认证标签，从而明白我们的职业和账号的大致方向。同时，认证也能获取平台背书，增加信任感。

在未进行官方认证前，也可以通过账号的基础设置向粉丝传达你是做什么的。以下是可进行统一设置的要素：
*   头像
*   背景图
*   个人简介
*   视频封面

通过对自身商业模式和账号类型的确定，将这些要素进行与业务关联的统一设计。

### 思考为粉丝提供的价值

最后，我们需要花时间思考一个核心问题：**你将为粉丝提供什么样的价值？** 在抖音中，粉丝对博主的价值需求主要可分为以下几类：

以下是粉丝价值需求的几种主要类型：

1.  **知识价值**
    *   **认知型**：如人生思考、思维方式、读书分享、行业内幕、成功经验、个人提升等。
    *   **干货型**：如技能提升、经验总结、操作步骤、冷知识等。
    *   **常识普及型**：如生活、健康、产品、科学、历史、地区等常识的解读与普及。
    *   **提供给粉丝的价值**：**认同感**。如三观认同、观点认同，减少知识盲区带来的获得感，进而产生对作者的认同。

2.  **情绪价值**
    *   情绪是影响观众的重要因素。容易引发共鸣的情绪类型包括：开心愉悦（搞笑、夸张表情）、关系共鸣（情侣、婆媳、家庭、职场等）、情绪波动（争议话题、反转反差内容）、以及怀旧、伤感、震惊、好奇等。
    *   **提供给粉丝的价值**：**共鸣**。利用表情、文字、画面、声音、肢体动作等准确到位地抒发情绪，直接感染观众。

3.  **感觉价值**
    *   博主通过短视频的画面、文字和声音传递信息。提供的价值包括：视频的画面美感、构图，以及给观众带来的喜欢、治愈、视觉冲击感，或个人魅力带来的仰慕感；也包括视频声音带来的节奏感、心跳加速或听觉享受。
    *   常见类型如：颜值类、萌宠类、美景类、才艺类、明星类、文化类等。
    *   **提供给粉丝的价值**：**共情**。因表情、文字、画面、声音、肢体动作等的自然表现，让人产生联想，并在情绪、认知等意识层面与作者产生连接，从而间接引发共鸣。

4.  **体验价值**
    *   抖音让用户看到了不一样的生活方式，体验到了未曾体验过的事物。例如流量很大的美食账号（享用与制作体验）、旅行体验、运动体验、角色体验等。
    *   **提供给粉丝的价值**：主要是**满足好奇心**，带来紧张感、激动、感动或获得感。

5.  **期待价值**
    *   当用户被你的视频内容、观看体验或对作者本人产生好奇时，可能会意犹未尽，进而点击关注，期待下次更新。许多博主会利用这一点，不将所有内容在一个视频中呈现完，以制造悬念。
