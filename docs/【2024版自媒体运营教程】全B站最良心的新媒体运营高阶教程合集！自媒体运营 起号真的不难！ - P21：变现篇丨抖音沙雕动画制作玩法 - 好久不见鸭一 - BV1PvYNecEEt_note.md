# 自媒体运营教程：P21：变现篇丨抖音沙雕动画制作玩法 🎬

在本节课中，我们将学习一种制作抖音沙雕动画的简易方法。这种方法无需复杂技术或原创脚本，仅需一部手机即可完成，旨在帮助初学者快速上手并实现流量获取与变现。

## 制作工具与基础设置 📱

上一节我们介绍了沙雕动画玩法的整体思路，本节中我们来看看具体的制作工具和初始设置。

![](img/19fa5189bb5ff85418d629b8ea638c75_1.png)

制作沙雕动画，你只需要一部手机和一个特定软件。启动软件后，首先需要进行基础设置。

以下是关键设置步骤：
*   **选择比例**：根据发布平台选择。抖音平台建议使用 **9:16** 的竖屏比例；西瓜视频等平台则建议使用 **16:9** 的横屏比例。
*   **设置分辨率**：为保证视频清晰度，分辨率应选择 **1080P**。
*   **设置帧率**：保持软件默认的帧率设置即可，无需改动。

完成这些设置后，你的制作环境就准备就绪了。

## 选择场景与角色 🏞️👤

设置好基础参数后，接下来要为动画搭建舞台和选择演员。

![](img/19fa5189bb5ff85418d629b8ea638c75_5.png)

软件的场景库提供了丰富的背景选项。例如，你可以选择一个“古代”场景分类。

![](img/19fa5189bb5ff85418d629b8ea638c75_3.png)

向右滑动，你可以看到该分类下的所有具体场景。角色的选择方式与此类似，角色库中有多种分类，例如沙雕风格、武侠风格、斗罗大陆或奥特曼等主题角色，内容非常丰富。

![](img/19fa5189bb5ff85418d629b8ea638c75_5.png)

## 动画制作与合成 🎞️

当场景和角色都选定后，就可以开始制作动画了。其操作逻辑与手机版剪映类似，非常简单直观。

点击已添加的角色，选择“添加动作”。点击播放按钮，角色就会执行所选动作。如果你对当前动作不满意，可以随时更换。你还可以添加更多角色来丰富剧情。最后，导入你的音频文件，软件即可合成一段完整的动画视频。

## 高效配音与流量玩法 🎤

动画制作本身并不复杂，而配音内容才是沙雕动画吸引流量的灵魂。这里教你一个“懒人”方法，无需自己编写搞笑段子也能获得流量和收益。

这个方法的核心是“拍同款”。一些聪明的博主已经开始使用此方法。具体操作是，寻找搞笑博主的原创视频，直接使用他们的原声来制作你的沙雕动画。

这样做主要有三个好处：
*   **保证原创度**：视频形式是你的原创动画，不易被平台判定为重复或抄袭，有利于播放量和收益。
*   **蹭取流量**：你可以借助原声博主的流量。许多用户喜欢点击原声，观看其他人的模仿作品，这为你带来了额外的曝光机会。
*   **提升效率**：省去了构思段子和亲自配音的环节，节省了大量时间和精力。

节省下来的时间，你可以用于运营更多账号，从而实现多账号的流量与收益增长。有时，你缺乏的并非资源，而是发现有效资源的方法。
