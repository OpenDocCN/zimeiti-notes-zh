# 抖音运营教程：P11：策划内容定位

在本节课中，我们将要学习如何为你的抖音账号策划一个清晰且具有竞争力的内容定位。仅仅选择一个热门主题是不够的，我们还需要深入规划构成视频的各个核心要素。

![](img/2a08a5453c95b8a7511cc2fb31cc9f60_1.png)

上一节我们讨论了如何构思主题IP，本节中我们来看看如何将这个主题具体化，使其脱颖而出。

很多同学会担心：我选择的主题（例如“摄影教学”、“美食探店”）已经有很多人在做了，内容同质化严重，我的账号还能火起来吗？这个问题的关键，在于主题之下的**核心内容**是否足够具体和独特。

选定“摄影教学”这个主题IP后，更具体的问题随之而来：教什么内容？拍人像还是风景？用手机还是相机？这些都需要明确。因此，除了主题IP，我们还需要规划以下更细致的维度：

以下是构成内容定位的几个核心要素：
*   **身份**：你以什么身份出镜？（例如：专业摄影师、摄影爱好者、搞笑博主）
*   **场景**：在什么环境下拍摄？（例如：工作室、街头、自然风光）
*   **道具与动作**：使用什么道具？设计什么标志性动作？
*   **态度**：表达怎样的情绪或价值观？（例如：专业严谨、幽默搞笑、温暖治愈）
*   **机位与结构**：采用什么拍摄视角？视频的叙事结构是怎样的？

这些要素越丰富、越具体，你的内容定位就越明确，后续的内容产出也会越容易。一个成功的案例是账号“暴躁的小峰”，其核心就在于鲜明地选取了“暴躁”作为人物态度，从而形成了独特的记忆点。

为了让大家更直观地理解，我们来看一个具体案例：“仙女酵母”。

（此处为视频内容描述：博主“仙女酵母”在具有欧洲古典风格的场景中，扮演一位公主，演绎了一段在咖啡馆遗忘物品的日常小剧情。）

![](img/2a08a5453c95b8a7511cc2fb31cc9f60_3.png)

![](img/2a08a5453c95b8a7511cc2fb31cc9f60_5.png)

我们来分析一下这个视频。单看段子剧情，它并不新颖。但这个视频获得了数百万点赞，账号粉丝超过500万。其成功的关键在于精心的**策划**。

![](img/2a08a5453c95b8a7511cc2fb31cc9f60_7.png)

![](img/2a08a5453c95b8a7511cc2fb31cc9f60_9.png)

“仙女酵母”的独特之处在于：
*   **身份**：设定为“公主”。
*   **场景**：搭建了具有欧洲古典风格的场景。
*   **道具与服装**：使用了相应的复古服装与道具。

这种在**身份、场景、道具**上的独特打造，使其内容即使剧情普通，也能在众多账号中脱颖而出。这本质上属于**人物IP的打造**。

除了人物IP，另一种常见且有效的内容定位方法是打造独特的**场景IP**。

（此处为视频内容描述：博主“你的子笺子凛”始终在一个布置温馨、有巨大窗户的固定房间内，以固定视角拍摄弹唱视频。）

这个账号的魔力在于其极致的**场景统一性**。无论弹唱什么歌曲，其**场景、机位、人物状态**都保持高度一致，形成了一个强烈、稳定的视觉符号，让用户产生深深的陪伴感和归属感。

![](img/2a08a5453c95b8a7511cc2fb31cc9f60_11.png)

![](img/2a08a5453c95b8a7511cc2fb31cc9f60_13.png)

本节课中我们一起学习了如何深化内容定位。核心在于，不要停留在宽泛的主题层面，而要深入规划**身份、场景、道具、态度、机位**等具体要素。通过打造独特的**人物IP**或**场景IP**，即使内容同质化，你的账号也能凭借鲜明的风格脱颖而出。记住，定位越具体，创作越轻松，账号也越容易被人记住。
