# 微信视频号运营从0~1全攻略：P30：如何寻找书单类对标账号 📚

在本节课中，我们将学习在短视频平台（如抖音）上寻找书单类对标账号的两种核心方法。掌握这些方法，能帮助你快速定位并学习同领域优秀账号的运营模式。

上一节我们介绍了寻找对标账号的重要性，本节中我们来看看具体如何操作。

## 方法一：利用“关注”推荐功能

![](img/597f37753fd1a812f57560d4c051f205_1.png)

此方法的核心是利用平台的推荐算法。当你关注一个账号后，平台会基于该账号的标签，为你推荐相似账号。

具体操作步骤如下：

![](img/597f37753fd1a812f57560d4c051f205_3.png)

![](img/597f37753fd1a812f57560d4c051f205_4.png)

1.  **搜索与筛选**：在平台搜索框输入你计划做的领域关键词，例如“书单”。
2.  **初步评估**：在搜索结果中，点开一个非真人出镜、视频类型符合你预期的账号主页。
3.  **触发推荐**：在该账号主页点击“关注”按钮。
4.  **批量筛选**：点击关注后，页面下方会弹出“你可能感兴趣”的账号列表。这些账号通常与当前账号类型高度相似。
5.  **逐个确认与关注**：点开这些推荐账号，查看其视频内容。如果内容是你想模仿的类型，即可点击关注。如果不是，则返回继续浏览下一个推荐账号。

通过此方法，你可以在两三分钟内关注数十个同类账号。此后，平台的信息流会持续为你推荐更多相似账号。

![](img/597f37753fd1a812f57560d4c051f205_8.png)

## 方法二：通过商品“达人秀”寻找

此方法通过具体商品链接，寻找推广过该商品的大量账号，效率极高。

以下是具体操作流程：

1.  **搜索目标商品**：在平台搜索框直接输入你想推广的书籍名称，例如“墨菲定律漫画版”。
2.  **切换至商品筛选**：在搜索结果页，将标签从“综合”切换为“商品”。
3.  **按销量排序**：点击“销量”进行排序，找到销量最高的商品链接。
4.  **进入商品详情页**：点击进入该高销量商品的详情页面。
5.  **查找“达人秀/买家秀”**：在商品详情页中，找到“达人秀”或“买家秀”板块。这里汇集了大量推广过此商品的短视频账号。
6.  **分析与关注**：浏览这些达人发布的视频，他们的账号类型（如影视混剪、语录剪辑）和内容风格通常高度一致。你可以从中筛选出作品数据好、适合模仿的账号进行关注。

![](img/597f37753fd1a812f57560d4c051f205_10.png)

例如，一个商品可能有592个达人视频，这意味着有数百个同类型账号可供你参考和学习。其他品类（如玩具、渔具）的寻找方法与此类似。

本节课中我们一起学习了寻找对标账号的两种实用方法：一是利用关注后的系统推荐功能进行批量筛选；二是通过具体商品的“达人秀”板块高效发现大量同领域账号。掌握这两种方法，你就能快速建立起自己的学习库，为后续的内容创作打下坚实基础。
