# 视频号运营：P39：如何扒取直播间话术 🎤

在本节课中，我们将学习一个实用的技巧：如何借鉴并拆解其他成功直播间的话术。掌握这个方法，可以帮助你快速学习优秀主播的销售逻辑和表达方式，从而优化自己的直播内容。

## 概述

上一节我们介绍了视频内容的创作方法，本节中我们来看看如何从他人的直播中汲取经验。许多商品的销售具有时效性，例如年货，而优秀的话术是促成交易的关键。通过学习如何记录和分析他人直播间的话术，你可以将其提炼并转化为适合自己的脚本。

## 所需工具

你需要使用一个名为“飞书”的工具。具体操作步骤如下：

![](img/3c91871440110a4fd4ea685ee8146758_1.png)

![](img/3c91871440110a4fd4ea685ee8146758_3.png)

以下是使用飞书进行话术录制的核心步骤：

1.  下载并打开“飞书”应用。
2.  点击底部菜单栏的“工作台”。
3.  在工作台中，找到并点击“飞书妙计”功能。

## 操作步骤详解

找到工具后，就可以开始录制了。整个过程非常简单。

以下是具体的操作流程：

1.  在飞书妙计界面，点击右下角的语音按钮，开始录音。
2.  打开你想要学习和模仿的直播间。
3.  让飞书妙计录制直播间的声音约15分钟。这段时间通常足以覆盖产品介绍、互动和促单（逼单）等核心环节。
4.  录制完成后，点击“完成”结束。

## 话术导出与整理

录制完成后，你需要将音频内容转换为可编辑的文本。

以下是导出和整理文本的方法：

1.  在飞书妙计中找到录制好的文件并点开。
2.  点击“分享”按钮，选择“可编辑”或“可阅读”模式。
3.  复制生成的链接。
4.  将链接发送到微信，并在微信中打开。此时，直播的语音内容已被转换为文字。
5.  复制全部文本，粘贴到文档编辑器中（如Word、备忘录或记事本）。
6.  仔细阅读文本，提炼出其中的产品介绍逻辑、互动问答和促单话术。
7.  根据你自身的产品和风格，修改并形成属于你自己的直播话术脚本。开播时可以将脚本放在旁边作为提示。

**核心操作代码示意**：
整个流程可以简化为：`打开飞书妙计 -> 开始录音 -> 播放目标直播间 -> 结束录音 -> 导出文本 -> 编辑整理`。
