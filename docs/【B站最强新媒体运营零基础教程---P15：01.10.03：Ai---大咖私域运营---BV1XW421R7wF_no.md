# 新媒体运营AI实战：P15：01.10.03：🤖 训练AI从分析到产出爆款笔记

## 概述
在本节课中，我们将学习如何训练AI（以ChatGPT为例）分析爆款笔记，并最终生成具备爆款潜质的新笔记。我们将通过一个考研领域的案例，完整演示从“喂资料”到“出文案”的全过程。

上一节我们介绍了通过提供资料让AI学习平台风格的方法。虽然上一节的旅游攻略案例效果不理想，但方法是正确的。本节中，我们来看看如何通过分析具体的爆款内容，让AI学习其成功要素，并用于创作。

## 第一步：明确分析指令
要让AI有效分析，我们需要给出清晰、具体的指令。以下是分析爆款笔记时需要关注的几个核心点。

我们需要让AI分析以下内容：
1.  **爆款因素**：分析内容为何能成为爆款。
2.  **文案结构**：
    *   **开篇**：用什么方式吸引用户继续阅读。
    *   **中间**：提供了哪些内容来戳中用户痛点，提升停留时长。
    *   **结尾**：设置了什么“钩子”来引导用户互动（如私信、关注）。
3.  **关键词策略**：分析笔记是否植入了特定关键词。如果有，其**出现频率**和**铺设结构**是怎样的。

![](img/e3058d6a9d2b06634e02f40464d131ba_5.png)

![](img/e3058d6a9d2b06634e02f40464d131ba_7.png)

![](img/e3058d6a9d2b06634e02f40464d131ba_9.png)

![](img/e3058d6a9d2b06634e02f40464d131ba_11.png)

我们可以将以上要求整合成给AI的指令。一个示例指令如下：
```
你是一名小红书爆款笔记分析师。我将提供一篇小红书平台的爆款笔记文案，请你分析：
1.  爆款因素：它为什么能火？
2.  整体结构：开篇如何吸引用户？中间如何戳中痛点？结尾如何引导互动？
3.  关键词分析：笔记是否结合中文分词法设计了关键词？如果有，是哪个词？出现频率（比率）如何？
```

## 第二步：寻找并投喂爆款样本
接下来，我们需要寻找目标领域（本例为“考研”）的真实爆款笔记，并将文案内容提供给AI进行分析。

![](img/e3058d6a9d2b06634e02f40464d131ba_17.png)

![](img/e3058d6a9d2b06634e02f40464d131ba_19.png)

我们以一篇考研领域点赞数3000+的笔记为例。将文案复制后，连同上述分析指令，一并提交给AI（如ChatGPT 3.5）。

![](img/e3058d6a9d2b06634e02f40464d131ba_21.png)

![](img/e3058d6a9d2b06634e02f40464d131ba_22.png)

AI给出了分析结果：
*   **爆款因素**：情感共鸣（开篇营造焦虑感）、逆袭故事、提供实用备考建议、使用表情符号增加亲和力、结尾传递正能量。
*   **结构分析**：符合“情感引入-经验分享-鼓励互动”的常见结构。
*   **关键词分析**：本篇笔记没有明显刻意设计的关键词。

![](img/e3058d6a9d2b06634e02f40464d131ba_24.png)

为了获得更全面的认知，我们应继续提供多篇同领域爆款笔记给AI分析。这个过程就是“喂资料”，让AI积累对该领域爆款模式的认知。

## 第三步：整合爆款要素
在AI分析了多篇样本后，我们需要让它对分析结果进行整合与去重，提炼出通用的爆款要素列表。

我们可以指令AI：“请将你前面分析过的几篇考研笔记的爆款因素总结一下，整合成一个去重后的列表。”

AI可能会总结出如下要点：
*   情感共鸣与逆袭故事引入
*   提供明确、详细的备考计划
*   给出针对性的方法与建议
*   强调时间管理与重要节点
*   推荐具体资源与技巧
*   使用表情符号增强感染力
*   结尾进行正能量鼓励并引导互动

**注意**：AI有时会“遗忘”或遗漏之前分析过的某些点（如“逆袭故事”），这时需要我们在指令中提醒它回顾之前的对话内容。

![](img/e3058d6a9d2b06634e02f40464d131ba_26.png)

![](img/e3058d6a9d2b06634e02f40464d131ba_28.png)

## 第四步：生成大纲与文案
有了爆款要素的认知后，我们就可以指令AI进行创作了。建议分两步走：先出大纲，再写正文。

![](img/e3058d6a9d2b06634e02f40464d131ba_29.png)

![](img/e3058d6a9d2b06634e02f40464d131ba_30.png)

首先，指令AI生成大纲：
```
请结合你前面分析的爆款笔记因素，为我规划一篇关于考研的小红书笔记大纲。大纲需要具备爆款潜力，每个主要部分下请列出细分的要点。
```
AI生成的大纲可能包含：吸引人的开篇、专业选择分析、各科备考规划、时间管理、心态调整、互动引导等部分。

如果大纲符合要求，我们就可以进行下一步，指令AI根据大纲和爆款要素撰写完整文案：
```
请按照你提供的大纲，撰写具体的笔记文案。要求：
1.  结构严格遵循爆款笔记的要素。
2.  包含一个简单的“逆袭小故事”来引入读者情绪。
3.  文案中要使用适量的emoji表情。
4.  需要结合中文分词法，自然地融入“考研”等相关关键词。
```

## 第五步：优化与进阶
AI生成的初版文案可能已经具备可用性，但通常还有优化空间。我们可以：
*   **人工润色**：在AI文案基础上，调整语气，丰富细节，使其更精炼、更打动人。
*   **使用更高级工具**：ChatGPT 4.0或自定义的GPTs（AI智能体）通常能产生更高质量、更稳定的内容。你可以创建一个专属的“小红书文案助手”GPTs，预先上传平台风格指南、专属表情包库等资料，使其输出更符合你的个性化需求。

## 总结
本节课我们一起学习了利用AI辅助生产爆款笔记的完整流程：
1.  **下达精准指令**，让AI从结构、要素、关键词等多维度分析爆款。
2.  **投喂多个样本**，让AI学习领域内的成功模式。
3.  **整合爆款要素**，提炼可复用的方法论。
4.  **分步生成内容**，先大纲后文案，可控性更强。
5.  **持续优化迭代**，结合人工润色或进阶工具提升质量。
