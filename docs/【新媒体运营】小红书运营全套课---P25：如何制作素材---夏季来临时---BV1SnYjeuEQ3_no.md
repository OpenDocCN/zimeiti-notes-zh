# 新媒体运营：P25：如何制作素材 - 夏季来临时

## 概述
在本节课中，我们将学习如何为小红书笔记制作无水印素材。课程将详细介绍一款名为“龙猫工具大师”的工具，并演示如何用它从不同平台下载和保存视频素材。

## 工具准备与登录
首先，我们需要使用“龙猫工具大师”这款工具。请提前在手机应用商店下载并安装此应用。

打开应用后，需要使用微信账号进行登录。该工具完全免费，但有时加载速度可能较慢，请耐心等待。

登录成功后，进入应用主界面。

## 获取使用次数
上一节我们介绍了工具的登录，本节中我们来看看如何确保有足够的使用次数。

工具的部分功能可能需要消耗次数。如果次数不足，可以通过完成任务来获取。

以下是获取额外次数的步骤：
*   在应用内找到“我的”页面。
*   进入“任务中心”。
*   点击“签到”或完成其他指定任务。
*   完成任务后，即可领取额外的使用次数。

## 下载小红书素材
掌握了次数获取方法后，我们现在开始学习如何下载小红书上的素材。

操作过程非常简单，无需在工具内进行复杂设置。

以下是下载小红书视频素材的步骤：
1.  在小红书APP中找到你想要的视频。
2.  点击“复制链接”。
3.  直接打开“龙猫工具大师”应用。
4.  应用会提示“允许粘贴”，点击“允许”。
5.  工具会询问“提取已复制的链接吗？”，点击“提取”。
6.  稍等片刻，素材便会解析出来。
7.  点击“保存”按钮，即可将无水印视频保存到手机相册。

保存完成后，你可以在相册中找到下载好的素材，这些素材可以直接用于你的笔记创作。

## 下载抖音素材
除了小红书，我们同样可以下载抖音平台的视频素材，操作方法基本一致。

以下是下载抖音视频素材的步骤：
1.  在抖音APP中找到目标视频（例如，搜索“手机壳”）。
2.  点击“复制链接”。
3.  打开“龙猫工具大师”应用。
4.  允许粘贴后，点击“提取”。
5.  等待视频素材解析完成。
6.  点击“保存到相册”，素材便保存完毕。

这样，你就获得了可以自由使用的抖音视频素材。

## 总结
本节课中，我们一起学习了如何为小红书运营制作无水印素材。核心内容包括：准备并登录“龙猫工具大师”工具、通过任务中心获取使用次数、逐步演示从小红书和抖音平台下载视频素材的方法，并简要介绍了其他替代工具。掌握这些技能，能帮助你更高效地收集和制作笔记所需素材。
