# 抖音运营指南：P26：5.人物设定

![](img/e9aba9e316487b5fc46095e32f805644_0.png)

## 概述
在本节课程中，我们将学习抖音账号运营中一个至关重要的环节：**人物设定**（简称“人设”）。我们将探讨如何像设计游戏角色一样，为你的抖音账号策划一个真实、立体且富有吸引力的人格化形象，并了解如何通过“超级符号”强化观众的记忆点。

![](img/e9aba9e316487b5fc46095e32f805644_2.png)

## 人设策划的核心概念
上一节我们讨论了账号定位，本节中我们来看看如何将定位具体化为一个鲜活的人物。在抖音中策划人设，类似于在游戏中创建角色，需要预先设定其称呼、职业、技能等属性。

![](img/e9aba9e316487b5fc46095e32f805644_4.png)

但与游戏角色不同，抖音中的人物设定是**人生角色**，因此必须更具真实感。无论是短视频还是直播，一个成功的IP（知识产权/形象）能有效解决粉丝的信任问题，促进内容裂变传播，并具备强大的商业变现潜力。

![](img/e9aba9e316487b5fc46095e32f805644_6.png)

因此，我们运营账号，无论是创作内容还是策划选题，核心目标都是在获取流量的同时**打造IP**。打造IP的本质，是在特定场景或情境中，完成账号的**人格化**过程。

![](img/e9aba9e316487b5fc46095e32f805644_8.png)

**账号人格化公式**：
```
成功IP = 人格化账号 + 情感注入 + 故事塑造 + 符号化标签
```

账号人格化，即我们通常所说的**账号人设**。

![](img/e9aba9e316487b5fc46095e32f805644_10.png)

## 人设策划的六个维度
对于账号人设的策划，我们可以从以下六个具体方面着手。以下是详细的策划要点：

![](img/e9aba9e316487b5fc46095e32f805644_12.png)

**1. 名字/昵称**
一个IP首先需要一个称呼，这是其传播的名片。必须明确IP叫什么，这如同一个品牌的名称或商标一样关键。

![](img/e9aba9e316487b5fc46095e32f805644_14.png)

**2. 职业/身份**
人格化必然具备身份。职业或职务是用户认识这个IP的联想入口，例如：老师、教授、律师、创业者等。

![](img/e9aba9e316487b5fc46095e32f805644_16.png)

**3. 称呼/社会关系**
一个恰当的称呼能快速拉近与用户的距离，产生情感共鸣。例如：XX哥、XX姐、XX叔、XX姥等。

![](img/e9aba9e316487b5fc46095e32f805644_18.png)

**4. 形象/第一印象**
从用户视角塑造IP形象，使其更容易占据用户心智。例如：大叔形象、甜美可爱、硬汉风格、时尚达人等。

**5. 性格/爱好**
一个具有鲜明特点的IP，除了记忆点强，也更具真实性。过于完美的形象反而容易脱离大众。例如：暴躁但热心、吃货属性、幽默风趣等。

**6. 穿着打扮/社会地位**
借助社会普遍认知和刻板印象，可以快速在用户心中为IP定位。例如：专业着装代表权威，休闲打扮显得亲民，特定的道具或环境暗示其财富或兴趣。

## 从人设到账号搭建
以上六个维度的内容策划完成后，即可结合你自身的产品或服务优势，将其落实到抖音账号的**“五件套”**中：
*   **头像**
*   **昵称**
*   **背景图**
*   **简介**
*   **视频封面风格**

## 提炼人设的“超级符号”
除了对账号进行人格化策划，还需通过持续的选题与内容策划，为你的人设提炼出易于记忆的**“超级符号”**。

超级符号可以从以下几个维度进行提炼：

**1. 画风（视觉）**
固定场景、标志性道具、特色服饰等。

**2. 听觉**
专属BGM（背景音乐）、特效音、独特的配音或声线。

**3. 文案**
能够反映账号核心优势且朗朗上口的提炼词或口号。例如：“方法简单粗暴”、“专注搞定XX问题”。

**4. 表情与情绪**
标志性的表情传递，如大笑、惊讶、无奈等，形成个人表情包。

**5. 语言特色**
独特的方言、口音、口头禅、语速（如慢条斯理或抑扬顿挫）。

**6. 动作习惯**
习惯性的肢体动作，例如推眼镜、特定手势、开场动作等。

超级符号提炼好后，在进行短视频创作时，应有意识地在视频中反复呈现这些元素，不断强化观众的记忆。

## 总结
本节课我们一起学习了抖音账号**人物设定**的完整方法。我们首先明确了人设即**账号人格化**的核心，然后从六个维度详细拆解了如何策划一个立体的人设，并指导大家将其应用于账号基础搭建。最后，我们探讨了如何通过多维度提炼**超级符号**，来强化人设的独特性和记忆点。一个清晰、真实且符号化的人设，是抖音账号获取信任、实现传播与变现的坚实基础。
