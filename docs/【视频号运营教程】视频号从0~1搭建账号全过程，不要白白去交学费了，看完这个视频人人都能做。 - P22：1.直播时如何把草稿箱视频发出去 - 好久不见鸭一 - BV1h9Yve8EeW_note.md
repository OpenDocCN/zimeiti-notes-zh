在本节课中，我们将学习一个非常实用的直播技巧：如何在直播过程中，将提前准备好的草稿箱视频发布出去，以吸引更多观众进入直播间。同时，我们也会探讨一些提升直播间初始人气的有效方法。

## 直播中发布草稿箱视频的步骤 📤

![](img/569e3415a6d48d513257c150a01a715e_4.png)

上一节我们介绍了直播前的准备工作，本节中我们来看看如何在直播中操作。当直播间观众较少时，可以通过发布视频来吸引流量。以下是具体操作步骤：

![](img/569e3415a6d48d513257c150a01a715e_6.png)

1.  点击直播间右上角的三个点菜单。
2.  选择“最小化”直播间，将直播画面暂时收起。
3.  此时，返回视频号主页。
4.  找到并点击你提前准备好的草稿箱视频。
5.  点击“发表”按钮。

**核心操作**：必须通过**最小化直播间 -> 返回主页 -> 从草稿箱发布**这个路径。如果直接在直播界面点击“发表视频”的入口，视频将无法成功发布。

![](img/569e3415a6d48d513257c150a01a715e_8.png)

> 注意：发布时系统可能会提示“正在直播中，请稍后”，但视频依然可以正常发布出去。

## 提升直播间初始人气的策略 🚀

仅仅发布视频还不够，我们还需要主动为直播间引入第一批观众。以下是几种有效的方法：

一句话介绍，以下是提升直播间人气的具体操作列表：

![](img/569e3415a6d48d513257c150a01a715e_14.png)

*   **分享直播间**：利用直播间的“分享”功能，将直播间链接转发到朋友圈或直接发送给好友。
*   **利用社群**：提前准备一些微信群（如业主群、兴趣群、粉丝群），在开播时将直播间分享到这些群里。
*   **使用小号**：如果不想用主号在个人社交圈分享，可以注册小号，将直播间分享给小号加入的各种群聊中。
*   **利用平台规则**：视频号官方有流量扶持机制。简单来说，如果你能从微信私域（如微信群、朋友圈）引导一定数量的用户进入直播间，平台通常会从公域给你推荐更多流量。其逻辑可以概括为：
    **私域引导流量 ≈ 公域推荐流量**

## 自查与优化直播表现 🧐

如果尝试了以上方法，直播间仍然留不住人，观众进入后很快离开，那么问题可能出在直播内容本身。

一句话介绍，以下是需要自我检查的几个方面：

*   **直播话术**：你的讲解是否清晰、有吸引力？需要不断练习和改进。
*   **表现力**：你是否精神饱满、充满热情？死气沉沉的状态会让观众立刻离开。
*   **个人能力**：直播是一项需要综合能力（表达、互动、控场）的工作，需要持续学习和锻炼。

遇到问题，首先从自身找原因，优化直播内容和状态，而不是单纯归咎于产品或外部因素。
