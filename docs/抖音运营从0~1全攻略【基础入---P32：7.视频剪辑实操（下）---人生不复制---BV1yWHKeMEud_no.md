# 抖音运营从0到1全攻略：P32：7.视频剪辑实操（下）

在本节课中，我们将继续学习视频剪辑的核心操作，重点掌握如何为视频片段添加转场效果以及如何添加和调整字幕。这些步骤对于提升视频的流畅度与专业度至关重要。

上一节我们介绍了视频比例调整与背景设置，本节中我们来看看如何让多个视频片段之间衔接得更自然。

![](img/a6fed216f2356083da5abd6593158c3c_1.png)

## 添加视频转场

![](img/a6fed216f2356083da5abd6593158c3c_3.png)

我们的视频通常由多个约10秒的片段组成。这些片段之间需要有明确的衔接，因此必须添加转场效果。转场的作用是提示用户，当前内容已切换到另一个不同的片段。

![](img/a6fed216f2356083da5abd6593158c3c_5.png)

如果不添加转场，观众在观看时可能会感到突兀，产生“视频内容怎么突然变了”的疑惑。这种困惑可能导致观众退出视频查看详情，从而**严重降低视频的完播率**。而完播率直接影响视频的推荐权重和最终收益。

![](img/a6fed216f2356083da5abd6593158c3c_7.png)

以下是添加转场的具体步骤：

![](img/a6fed216f2356083da5abd6593158c3c_9.png)

1.  在视频编辑轨道上，找到并点击片段右侧的 **`+`** 号按钮。
2.  在弹出的菜单中，选择 **`素材库`**。
3.  在素材库的 **`热门`** 分类中，可以找到丰富的转场素材，例如各种动态切换效果。
4.  选择转场时，注意时长不宜过长。尽量选择 **`3至5秒`** 的转场素材，以保证节奏紧凑，避免观众失去耐心。最常用的是3秒左右的转场。
5.  选中合适的转场素材，点击添加。

![](img/a6fed216f2356083da5abd6593158c3c_11.png)

![](img/a6fed216f2356083da5abd6593158c3c_13.png)

转场素材添加后，同样需要调整其画面比例，使其与主视频保持一致（例如16:9）。

以下是调整转场素材比例的步骤：

![](img/a6fed216f2356083da5abd6593158c3c_15.png)

1.  选中已添加的转场片段。
2.  在编辑菜单中找到 **`画布模糊`** 或背景设置选项。
3.  选择清晰的画面背景，确保其符合 **`16:9`** 的横屏比例。
4.  最后，点击右下角的对勾 **`√`** 确认设置。

![](img/a6fed216f2356083da5abd6593158c3c_17.png)

## 添加与调整视频字幕

视频转场设置完毕后，下一步是为视频添加字幕（文案）。清晰的字幕能提升观看体验，尤其适用于在无声环境下观看视频的用户。

![](img/a6fed216f2356083da5abd6593158c3c_19.png)

以下是添加和调整字幕的步骤：

![](img/a6fed216f2356083da5abd6593158c3c_21.png)

1.  点击手机编辑界面左下角的 **`文字`** 选项。
2.  选择 **`新建文本`** 右侧的 **`识别字幕`** 功能。
3.  点击 **`开始匹配`**，软件将自动识别视频或音频中的人物对话，并生成对应的字幕文本。
4.  字幕生成后，需要进行调整以优化视觉效果。拖动字幕文本框右下角，可以放大字幕。
5.  将放大后的字幕文本框拖动到视频画面的最下方，这个位置通常最便于阅读且不遮挡主要内容。
6.  调整好第一段字幕后，后续字幕通常会应用相同的样式，无需重复调整。

## 核心要点与总结

本节课我们一起学习了视频剪辑后半部分的关键操作。

![](img/a6fed216f2356083da5abd6593158c3c_23.png)

首先，我们了解了添加**视频转场**的必要性：它能明确区分内容片段，提升观看流畅度，是保障**视频完播率**的重要环节。操作上，我们从素材库选择3-5秒的转场，并确保其画面比例与主视频一致。

![](img/a6fed216f2356083da5abd6593158c3c_25.png)

接着，我们学习了如何添加**字幕**：通过“识别字幕”功能自动生成，然后调整其大小和位置至视频底部，以增强视频的可读性和专业性。

![](img/a6fed216f2356083da5abd6593158c3c_27.png)

最后，请记住制作视频的核心原则：严格遵循平台要求的**比例**（如16:9），精心挑选和处理**素材**，避免使用过长的片段或转场，以确保内容顺利通过平台审核。整个流程并不复杂，关键在于细心和规范操作。
