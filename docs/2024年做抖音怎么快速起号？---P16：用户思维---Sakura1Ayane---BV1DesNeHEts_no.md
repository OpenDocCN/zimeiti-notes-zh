## 从销售思维到用户思维

![](img/e62f38eb493db30d6b7223a4067a182f_24.png)

![](img/e62f38eb493db30d6b7223a4067a182f_26.png)

如何才能在抖音上达到引流和销售的效果？如果仅以销售的思维去做，你会怎么做？

![](img/e62f38eb493db30d6b7223a4067a182f_30.png)

![](img/e62f38eb493db30d6b7223a4067a182f_32.png)

以销售的思维去想，取决于客户的需求。这没错。那么，我们如何才能知道客户的需求呢？

![](img/e62f38eb493db30d6b7223a4067a182f_38.png)

![](img/e62f38eb493db30d6b7223a4067a182f_40.png)

![](img/e62f38eb493db30d6b7223a4067a182f_42.png)

其实大家可以想一下，我们以销售的思维去做，会想到大家喜欢什么、用什么。我们这些技巧和话术，如果单纯用销售的思维去想，可能会这样说：

![](img/e62f38eb493db30d6b7223a4067a182f_44.png)

![](img/e62f38eb493db30d6b7223a4067a182f_46.png)

![](img/e62f38eb493db30d6b7223a4067a182f_48.png)

*   我是医美机构，我能让你变美。
*   你来到我的机构之后，我让你有一个蜕变。
*   我们这里的医生特别专业、特别权威。
*   或者说，我这本书特别好，只需要59块钱，今天是七夕节，我还降了十块钱，你买吧，特别优惠。

![](img/e62f38eb493db30d6b7223a4067a182f_50.png)

![](img/e62f38eb493db30d6b7223a4067a182f_52.png)

![](img/e62f38eb493db30d6b7223a4067a182f_54.png)

但是，这样的内容在淘宝上或许可以，在抖音上却行不通。为什么？

![](img/e62f38eb493db30d6b7223a4067a182f_56.png)

![](img/e62f38eb493db30d6b7223a4067a182f_58.png)

![](img/e62f38eb493db30d6b7223a4067a182f_60.png)

因为淘宝是**电商平台**，而抖音是一个**娱乐平台**。用户打开淘宝，通常是因为有明确的购物需求，或者做好了花钱的准备。所以，直接的价格促销（如“59元，七夕节降10元”）可能有效。

![](img/e62f38eb493db30d6b7223a4067a182f_62.png)

![](img/e62f38eb493db30d6b7223a4067a182f_64.png)

![](img/e62f38eb493db30d6b7223a4067a182f_66.png)

然而，抖音是一个娱乐软件。用户打开抖音是为了找乐子、图开心。此时如果直接进行推销，用户可能会感到厌烦。即使再便宜，用户也可能觉得“关我什么事？我是来娱乐的”。

![](img/e62f38eb493db30d6b7223a4067a182f_68.png)

![](img/e62f38eb493db30d6b7223a4067a182f_70.png)

![](img/e62f38eb493db30d6b7223a4067a182f_72.png)

所以，我们必须转换思维，站在用户的角度上思考。

![](img/e62f38eb493db30d6b7223a4067a182f_76.png)

## 核心：用户思维

![](img/e62f38eb493db30d6b7223a4067a182f_82.png)

![](img/e62f38eb493db30d6b7223a4067a182f_84.png)

![](img/e62f38eb493db30d6b7223a4067a182f_86.png)

互联网思维有很多种，市面上流传着各种概念。我们可以先不管其他，但永远要记住第一个也是最重要的思维：**用户思维**。永远都要想用户喜欢什么。

![](img/e62f38eb493db30d6b7223a4067a182f_88.png)

![](img/e62f38eb493db30d6b7223a4067a182f_90.png)

![](img/e62f38eb493db30d6b7223a4067a182f_92.png)

我们需要从“我想卖什么”转变为“用户想看什么、需要什么”。这要求我们具体到视频的每一秒，都要考虑用户看完后的反应。

![](img/e62f38eb493db30d6b7223a4067a182f_94.png)

![](img/e62f38eb493db30d6b7223a4067a182f_96.png)

![](img/e62f38eb493db30d6b7223a4067a182f_98.png)

以下是运用用户思维的两个关键实践：

### 1. 内容排序：利益前置

![](img/e62f38eb493db30d6b7223a4067a182f_106.png)

![](img/e62f38eb493db30d6b7223a4067a182f_108.png)

![](img/e62f38eb493db30d6b7223a4067a182f_110.png)

![](img/e62f38eb493db30d6b7223a4067a182f_112.png)

我们以教授“打光技巧”为例。假设一个视频包含五个知识点：测光、底光、蝴蝶光、轮廓光等。

![](img/e62f38eb493db30d6b7223a4067a182f_114.png)

![](img/e62f38eb493db30d6b7223a4067a182f_116.png)

![](img/e62f38eb493db30d6b7223a4067a182f_118.png)

如果只是随机地讲解每个光怎么打、有什么效果（例如：测光让视频更清晰，底光让画面更恐怖，蝴蝶光让人更美丽），用户可能觉得“哦，还行吧”，但触动不大。

![](img/e62f38eb493db30d6b7223a4067a182f_120.png)

![](img/e62f38eb493db30d6b7223a4067a182f_122.png)

![](img/e62f38eb493db30d6b7223a4067a182f_124.png)

关键在于，我们要分析哪个知识点能给用户带来最直接、最强烈的**利益获得感**。例如，“蝴蝶光能让女生更漂亮”就比“底光能让画面更恐怖”更具吸引力，因为“变漂亮”是很多用户的直接利益诉求。

![](img/e62f38eb493db30d6b7223a4067a182f_126.png)

![](img/e62f38eb493db30d6b7223a4067a182f_128.png)

![](img/e62f38eb493db30d6b7223a4067a182f_130.png)

因此，在安排内容时，我们应该把更吸引人、能让用户有利益获得感的点放在视频最前面，以抓住用户的注意力。

![](img/e62f38eb493db30d6b7223a4067a182f_132.png)

![](img/e62f38eb493db30d6b7223a4067a182f_134.png)

![](img/e62f38eb493db30d6b7223a4067a182f_136.png)

**公式：内容吸引力 = 用户感知利益强度 × 呈现顺序优先级**

![](img/e62f38eb493db30d6b7223a4067a182f_138.png)

![](img/e62f38eb493db30d6b7223a4067a182f_140.png)

### 2. 标题设计：关联用户利益

![](img/e62f38eb493db30d6b7223a4067a182f_144.png)

![](img/e62f38eb493db30d6b7223a4067a182f_146.png)

![](img/e62f38eb493db30d6b7223a4067a182f_148.png)

![](img/e62f38eb493db30d6b7223a4067a182f_150.png)

标题是吸引用户点击的第一道门。同样以“打光教学”为例，对比以下两种标题：

![](img/e62f38eb493db30d6b7223a4067a182f_152.png)

![](img/e62f38eb493db30d6b7223a4067a182f_154.png)

![](img/e62f38eb493db30d6b7223a4067a182f_156.png)

*   **普通标题**：“人人必会的三个打光神技” 或 “教你三种打光方式”。
    *   用户心理：“打光关我啥事？” 缺乏利益关联。
*   **用户思维标题**：“能把姑娘变漂亮的打光方式”。
    *   用户心理：“变漂亮？这个对我/我拍的人有用。” 直接点明用户可获得的利益。

![](img/e62f38eb493db30d6b7223a4067a182f_158.png)

![](img/e62f38eb493db30d6b7223a4067a182f_160.png)

![](img/e62f38eb493db30d6b7223a4067a182f_162.png)

标题的作用不是概括内容，而是**向用户承诺观看视频能获得的价值**。一个具有“利益获得感”的标题能显著提高视频的点击率。

![](img/e62f38eb493db30d6b7223a4067a182f_164.png)

![](img/e62f38eb493db30d6b7223a4067a182f_166.png)

![](img/e62f38eb493db30d6b7223a4067a182f_168.png)

**代码示例（标题思维转换）：**
```plaintext
# 传统思维标题
title = “知识点” + “教学”
# 用户思维标题
title = “你能获得的好处” + “（通过知识点）”
```

![](img/e62f38eb493db30d6b7223a4067a182f_170.png)

![](img/e62f38eb493db30d6b7223a4067a182f_172.png)

![](img/e62f38eb493db30d6b7223a4067a182f_174.png)

总之，无论是视频内容的结构，还是标题的撰写，每一秒、每一个字都需要考虑用户。思考他们到底想看什么、需要什么，并将你能提供的价值与之精准匹配。

![](img/e62f38eb493db30d6b7223a4067a182f_176.png)

![](img/e62f38eb493db30d6b7223a4067a182f_178.png)

![](img/e62f38eb493db30d6b7223a4067a182f_180.png)

本节课中我们一起学习了抖音运营中的“用户思维”。核心在于摆脱生硬的销售逻辑，始终从娱乐平台用户的视角出发，通过“利益前置”的内容排序和“关联利益”的标题设计，将你的价值转化为用户感兴趣、能获得的内容。记住，在抖音，你不是在推销，而是在提供一种令人愉悦的解决方案。
