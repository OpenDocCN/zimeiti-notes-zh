# 小红书运营教程：P8：5. 小红书的3种店铺类型 🏪

在本节课中，我们将要学习小红书平台上的三种主要店铺类型：个人店、个体店和企业店。了解它们之间的区别、保证金要求、可售类目以及开店门槛，是选择适合自己店铺模式的第一步。

![](img/b8deea262bca3f7e925dfbff061a2222_1.png)

上一节我们介绍了小红书开店的基础优势，本节中我们来看看具体的店铺分类。

![](img/b8deea262bca3f7e925dfbff061a2222_2.png)

小红书店铺总体上分为三种类型：**个人店**、**个体店**和**企业店**。企业店中还可细分为旗舰店与专卖店，但本节我们主要讨论这三种基础类型。

---

## 1. 个人店 👤

个人店是门槛最低的店铺类型。

![](img/b8deea262bca3f7e925dfbff061a2222_4.png)

以下是个人店的核心特点：

*   **保证金**：**1000元**。平台允许先销售产品，再补交保证金。
*   **销售限制**：虽然可以先卖后交，但不建议这样做。因为未缴纳保证金时，店铺无法参与平台活动，用户也无法使用微信支付。保证金在关店时可退还。
*   **可售类目**：只能上架平台规定的**部分类目**。具体可参考《小红书经营大类一览表》，通常非电器等特殊类目的商品都可以销售。
*   **开店门槛**：仅需**实名认证**。支持零粉丝开店，无需营业执照。店铺名称和头像可以随意更改。

## 2. 个体店 🏬

个体店适合拥有个体工商户营业执照的卖家。

以下是个体店的核心特点：

![](img/b8deea262bca3f7e925dfbff061a2222_9.png)

*   **保证金**：**1000元**。与个人店不同，**必须先缴纳保证金才能上架销售产品**。
*   **可售类目**：可上架类目比个人店**更多**。可以销售代理或授权的品牌产品，例如获得授权的电器类商品。
*   **开店门槛**：
    1.  需要**个体工商户营业执照**。
    2.  需要认证**个体企业号**，认证费为**600元/年**。
    3.  支持零粉丝开店。
    4.  **店铺名称和认证类目不可随意更改**。如需修改店铺名称，需要重新缴纳600元认证费。一个营业执照最多可认证**2个**企业号（根据新规）。

![](img/b8deea262bca3f7e925dfbff061a2222_9.png)

---

## 3. 企业店 🏢

企业店适合拥有企业营业执照的商家，功能最全面。

以下是企业店的核心特点：

*   **保证金**：基础保证金为**2000元**以上。若升级为**专卖店**或**旗舰店**，保证金通常为**20000元**。同样需要先交保证金后经营。
*   **可售类目**：可销售类目**最多**，几乎涵盖所有产品（除仅限平台邀约的类目）。可以开设品牌旗舰店或品牌专卖店。
*   **开店门槛**：
    1.  需要**企业营业执照**。
    2.  需要认证**企业号**，认证费为**600元/年**。
    3.  支持零粉丝开店。
    4.  **店铺名称和认证类目不可随意更改**。
        *   **重点提示**：经营类目可以**申请新增**，但需要提交资质并通过平台审核。例如，原店铺卖杯子（家居类目），想新增卖项链（饰品类目），只需申请新增类目并通过审核即可，**无需额外缴费**。但修改店铺名称仍需重新缴纳认证费。

## 核心对比与运营逻辑 🔄

了解完三种店铺类型后，我们需要理解小红书的独特电商逻辑。

小红书店铺与淘宝、天猫等传统电商平台有本质区别，核心在于**流量来源**。小红书更注重**内容驱动**。

![](img/b8deea262bca3f7e925dfbff061a2222_13.png)

![](img/b8deea262bca3f7e925dfbff061a2222_14.png)

典型的消费链路是：
`用户搜索或推荐刷到笔记 -> 通过笔记内容被“种草” -> 产生兴趣并做出消费决策 -> 进入店铺购买产品`

![](img/b8deea262bca3f7e925dfbff061a2222_15.png)

因此，平台对于“无货源店群”模式的打击，对于真心致力于通过优质内容做好电商的卖家而言是利好。因为在总流量不变的情况下，低质竞争对手的减少意味着更良性的竞争环境。

本节课中我们一起学习了小红书的三种店铺类型：**个人店**、**个体店**和**企业店**。关键要掌握它们的**保证金**规则、**可售类目**范围以及不同的**开店门槛**（如营业执照和认证要求）。同时，理解了小红书**“内容种草，店铺成交”**的核心运营逻辑，这将帮助你在选择店铺类型和制定运营策略时做出更明智的决策。
