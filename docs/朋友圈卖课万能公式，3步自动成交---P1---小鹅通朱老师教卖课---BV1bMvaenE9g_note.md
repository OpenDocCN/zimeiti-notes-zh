# 朋友圈卖课万能公式：P1：三步打造自动成交的朋友圈

在本节课中，我们将学习一套专为知识博主设计的、简单实用的朋友圈内容框架。掌握这个框架，可以有效提升你的专业形象，吸引潜在学员，并最终促进课程销售。

## 概述

打造优质的朋友圈是知识博主的必备技能。一个精心设计的朋友圈能够建立信任、展示价值，从而让潜在客户主动为你的课程付费。本节将拆解一个由三部分构成的万能公式，帮助你系统化地规划朋友圈内容。

## 朋友圈内容三大核心板块

上一节我们概述了朋友圈的重要性，本节中我们来看看构成高转化朋友圈的三个具体内容方向。以下是三个核心板块：

1.  **产品相关**
    *   发布与你的行业和课程产品相关的专业解决方案。
    *   目标是让用户感受到你在这个领域的专业性和深度。

2.  **人设相关**
    *   人们倾向于与阳光、正面、真实、接地气的人交朋友，朋友圈内容也应遵循此理。
    *   可以分享工作心得、工作趣事、团队生活（如部门聚餐）等内容。

3.  **用户相关**
    *   精心收集并展示用户证言。
    *   内容可包括：学员学习后的积极评价、通过学习取得的具体成绩（如销量提升、行业排名、获得奖状荣誉）、以及得到哪些权威机构的认可。
    *   还可以展示你曾服务过哪些知名客户或案例。
