## 账号准备与防关联方案 📱

![](img/86278012a3742aa7329aa311fb678c97_11.png)

既然要进行矩阵运营，就需要准备多个账号进行操作。我们建议每人至少准备5个矩阵账号，并解决账号防关联的技术问题。

### 账号注册与养号

![](img/86278012a3742aa7329aa311fb678c97_13.png)

以下是账号准备的具体步骤：
1.  **注册**：推荐使用手机卡注册。可以去某宝购买或去营业厅办理“0月租”的手机卡来注册账号。
2.  **养号与换绑**：账号注册后，先养号7~10天。之后将账号换绑到常用的流量卡上。这样，初始注册用的手机卡可以循环用于注册新账号。

### 设备与网络分配方案

为了管理多个账号并避免关联，需要合理的设备分配方案。以下是三种推荐方案：

*   **新手方案（成本最低）**：
    *   准备一台安卓手机（推荐小米，因其支持应用分身功能）。
    *   在本机登录2-3个账号，在应用分身上再登录2-3个账号。
    *   网络分配：本机账号使用A网络（如手机流量），分身账号使用B网络（如不同Wi-Fi）。**关键：避免所有账号同时使用同一个Wi-Fi**。

*   **一机一号方案（最稳定）**：
    *   适合长期进行矩阵运营。方案为每个账号配备独立的手机和手机卡。手机卡和手机的购买方式同上。

*   **云手机方案（适合有技术基础的团队）**：
    *   推荐购买“云手指”或其他品牌的云手机。

### 账号防关联详解

我们需要防止两种主要关联：设备关联和IP关联。
*   **设备关联**：通过同一台电脑浏览器或手机登录可能触发。
*   **IP关联**：通过同一IP地址（无论是电脑还是手机）上网可能触发。

那么，哪些账号需要做防关联处理？
*   **需要做防关联的账号类型**：经常进行平台不允许的操作、经常发布擦边内容、或所在行业封禁率较高的账号。
*   **不需要做防关联的账号类型**：发布正规行业内容、行为合规的大号；个人仅操作2-3个账号的情况。

为了解决关联问题，我们团队使用一款矩阵营销账号管理工具。该工具能为每个账号模拟独立的电脑环境和独立的IP地址。代码如下所示，展示了其核心功能逻辑：
```python
# 模拟工具为每个账号创建独立环境
for account in account_list:
    create_isolated_browser_environment(account)  # 创建独立浏览器环境
    assign_unique_ip_address(account)             # 分配独立IP地址
```
这样可以有效避免因IP或设备环境关联导致的账号封禁或限流。该软件的具体使用方法将在后面演示。

## 新号养成计划 🌱

账号准备妥当后，下一步是养号。新号必须完成两件事：**快速定位**和**账号包装**。

前10天，每天需要执行以下操作：
1.  浏览10~30篇自己行业相关的笔记。
2.  点赞5~20篇相关笔记。
3.  评论行业相关笔记2~10次。
4.  发布1~2篇笔记。

![](img/86278012a3742aa7329aa311fb678c97_19.png)

3.  **网络设置**：可选用本机网络或代理IP。点击下一步。

### 配置自动化运营计划

添加账号后，点击“AI运营”，进入小红书自动化设置界面。
1.  点击“添加计划”，选择要运营的账号。
2.  勾选需要的自动化功能：**自动活跃**、**自动AI点赞**、**自动AI评论**。如需发文，也勾选“自动发文”。
3.  在“账号关键词”中输入行业主关键词，例如“营销”、“引流”、“获客”。
4.  评论内容可以自定义，也可以使用“AI随机生成”。点击“使用AI生成评论”和“系统生成AI提示词”即可。
