### 1. 角色 🎭
角色，简单来说，就是你选择的赛道和内容输出方向。你需要让别人知道你提供的内容具有一定的专业性。
*   **好物分享/测评博主**：你需要展示你对产品的深入了解和客观评价。
*   **装修博主**：你需要告诉别人你确实有正在装修或已装修好的房子，并记录整个过程，分享遇到的坑和最终成果，展示前后对比。
*   **母婴赛道**：最常见的角色是“宝妈”。但根据内容方向，你也可以塑造为“护理专家”的角色，这不一定要求你自身是宝妈，可以是某机构或专业领域的人员。
*   **情感赛道**：你可以是“心理学顾问”、“情感大师”，分享关于情感、成长、励志的话题。此外，你还可以打造“成长型人设”，例如分享一位30岁女性如何处理家庭、创业、职场中遇到的问题，以及如何从失败中走出来。

### 2. 内容 📝
内容定位我们之前已经讨论过。在人设定位中，内容的关键在于持续性与垂直度。
你的内容需要吸引用户、提供价值，并在平台上有一定特色。在人设定位层面，你做某个赛道，就必须持续输出与该赛道相关的内容。
这涉及到账号的垂直度和标签化问题。我们一直强调要做垂直领域，目的是打牢账号标签，这直接影响账号的成长和后期流量分发。
**公式：账号垂直度 = 内容方向一致性**
如果你的第一篇内容是关于儿童教育，第二篇却是宠物VLOG，那么平台推荐给你的用户可能对第二类内容不感兴趣，导致数据下滑。平台会因此判定内容不受欢迎，从而影响后续推流效果。
请记住：你做什么赛道，就提供什么内容。

### 3. 形式 🎬
形式指的是内容呈现的具体方式。你需要考虑以下几个层面：
*   **内容产出场景**：你的内容是在固定室内场景拍摄，还是在不同的室外场景切换？例如，总是在上班路上、车内或特定工作场景中分享心得。场景的选择也可以参考对标账号。
*   **声音与出镜**：你是亲自出镜讲解，还是不出镜仅使用配音？现在很多账号采用朋友对话出镜但后期配音的形式，这通过“剪映”等工具很容易实现。你需要评估自己的表达能力和呈现方式。
*   **内容形式**：你的内容是以剧情、干货分享还是生活记录的形式呈现？
    *   **剧情型**：创作难度相对较高。
    *   **干货型**：需要准备丰富的文案脚本。
    *   **生活记录型**：需要大量的日常拍摄素材。
    每种形式都有其优劣势，需要根据自身情况选择。

### 4. 记忆点 🔥
记忆点是让人设深入人心、让用户记住你的关键。它需要通过固定的设计环节反复强化。
**代码示例（记忆点清单）：**
```
记忆点 = [
    “视频前几秒的固定动作/特效”，
    “标志性开场白”，
    “统一的背景音乐(BGM)”，
    “贯穿始终的特定物品/服装风格”
]
```
*   **为何前置？**：固定动作、特效、开场白这些元素放在视频开头，能在一开始就给用户强烈的冲击和记忆。当用户浏览你的主页多个视频时，每次开头都看到相似的画面，记忆会不断加深，效果远好于将这些点随机放在视频中段。
*   **BGM与全局元素**：统一的BGM能串联整个视频，当音乐响起，用户即使没看到画面也能联想到你。特定的物品（如桌面的摆件）或统一的服装风格（如职场穿搭、田园风），则能贯穿内容全局，强化视觉记忆。
