# AI短视频创作教程：P16：13.原创解说文案 📝

在本节课中，我们将要学习如何获取和创作原创解说文案。这是制作高质量AI短视频的关键一步，好的文案能直接提升视频的播放量和收益。

上一节我们介绍了使用“一搜猫”素材库来获取文案。本节中我们来看看，如果不想使用现成素材库，还有哪些其他途径可以获得解说文案。

## 途径一：使用我的原创解说文库

除了素材库，我拥有一个纯原创的解说文库。这个文库里的所有文案内容均为原创，我可以分享给大家使用。我和我的团队会持续更新这个文库，其中每篇文案大约有两三千字。

![](img/504702a920f4be1583922cfdc9de7ebe_1.png)

你只需要根据这些文案，去匹配和寻找相应的视频素材进行合成即可。这是一种非常便捷的方式。

## 途径二：自行创作纯原创文案

![](img/504702a920f4be1583922cfdc9de7ebe_3.png)

有很多同学表示，既不想用素材库，也不想用我的文库，而是希望自己动手写纯原创文案。我非常支持这个想法，因为纯原创文案相比借用素材的文案，其视频的播放收益单价通常会高出**两倍**。

如果你对自行创作文案感到畏惧，请记住一个核心原则：**解说文案就是说人话**。所谓“说人话”，就是用大白话、口水话把事情讲清楚。

以下是“说人话”文案的创作思路：

想象你向朋友推荐一部电视剧，你不会像写作文一样有完整的起承转合。你不会说：“妈妈，我在X月X日看了一部由XX导演执导、于X年上映的电视剧。男主角叫A，女主角叫B。第一集讲述了……第二集……”

你更可能会这样说：“妈妈，我今天看了一部剧，特别好看！男主帅女主美，他俩特别般配。故事讲的是他们相爱十年快结婚了，结果男的婚前出轨，被女的抓个正着，女的一巴掌扇过去说‘离婚吧’。剧情特别带劲！”

这就是“口水话”式的解说。观众来看解说，是为了快速了解故事梗概，而不是听你复述剧本或分析文言文（例如《甄嬛传》中的台词）。解说的价值就在于将复杂或冗长的内容，用通俗易懂的语言提炼出来。

## 解说领域的优势与选择

在我的团队中，从事视频解说的人是最多的。原因在于，解说类视频是**最赚钱的领域之一**。

解说涵盖的范围非常广泛，包括：
*   影视剧解说
*   纪录片解说
*   综艺节目解说
*   动物世界解说
*   科普知识解说
*   历史故事解说
*   动漫解说

如果你想开始制作解说视频，影视解说是一个有明确版权合作模式（影视推广）的成熟方向。如果你不想做影视类，那么我强烈推荐**动物解说**。

目前，在平台上专注做动物解说的人相对较少，而平台对此类内容的扶持流量却非常庞大，这是一个很好的机会。
