## 开通微信小商店 🛒

![](img/f0bea777c996dd08954e0517c84b6115_7.png)

无论你是否有自己的产品，都可以通过微信小商店进行带货。以下是开通微信小商店的步骤。

![](img/f0bea777c996dd08954e0517c84b6115_9.png)

1.  打开微信，在顶部搜索框搜索“微信小商店”。
2.  进入“微信小商店”公众号。
3.  点击公众号菜单栏中的“免费开店”。
4.  根据提示完成“快速开店”流程。

微信小商店的优势在于**零成本、零门槛**。与抖音橱窗需要500元押金、10条视频和1000粉丝的要求相比，微信小商店无需任何费用和粉丝基础即可开通，真正做到“人人可开店”。

如果你有自己的产品，可以在微信小商店中直接上架销售。

![](img/f0bea777c996dd08954e0517c84b6115_17.png)

在微信小商店管理后台，点击“新增商品”，上传商品图片、编辑标题和描述等信息即可完成上架。为了让商品图片更美观，可以使用“创客贴”、“美图秀秀”或“搞定设计”等在线设计工具进行美化。

## 无货源：带货赚取佣金 🤝

![](img/f0bea777c996dd08954e0517c84b6115_19.png)

如果你没有自己的货源，可以通过为其他商家的商品带货来赚取佣金。佣金比例可观，是视频号当前的重要红利。

以下是进入选品中心的路径和选品方法。

1.  进入视频号个人主页，点击右上角人头图标进入“创作者中心”。
2.  在创作者中心找到并进入“带货中心”。
3.  在“选品广场”中，可以根据食品、服饰、美妆等类目筛选商品。
4.  选择佣金率高、销量好的商品进行推广。

微信小商店带货中心的佣金率**高达50%**，相比之下，抖音的佣金率通常在25%-30%左右。这意味着销售同样100元的商品，在视频号赚取的佣金会更高。

## 扩展外部商品链接 🔗

![](img/f0bea777c996dd08954e0517c84b6115_25.png)

除了小商店内部的选品中心，你还可以通过“扩展链接”功能，带货来自其他电商平台的商品，例如阿里巴巴（1688）、淘宝联盟、京东联盟、拼多多团长等。

以下是使用扩展链接带货的简要步骤。

![](img/f0bea777c996dd08954e0517c84b6115_27.png)

1.  在其他电商平台（如1688）找到你想推广的商品，复制商品链接。
2.  在微信中搜索并使用“扩展链接”小程序。
3.  将复制的商品链接粘贴到扩展链接小程序中，生成专属推广链接。
4.  发布视频时，在视频下方挂载此链接。

![](img/f0bea777c996dd08954e0517c84b6115_29.png)

![](img/f0bea777c996dd08954e0517c84b6115_30.png)

这样，观众点击链接即可跳转到原平台购买，你同样可以赚取推广佣金。

![](img/f0bea777c996dd08954e0517c84b6115_31.png)
