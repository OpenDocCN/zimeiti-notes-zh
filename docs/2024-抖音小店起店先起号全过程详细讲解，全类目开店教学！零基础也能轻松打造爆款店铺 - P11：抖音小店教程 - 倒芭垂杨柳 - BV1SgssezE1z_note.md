## 一、选品策略：找到你的流量密码

上一节我们了解了课程的整体框架，本节中我们来看看如何迈出成功的第一步——选品。选品是店铺的基石，决定了你能否获得平台的自然流量。我们将学习两种核心的选品方法。

![](img/b7715491f60f306c4c2701dd6f52ed18_37.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_39.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_41.png)

### 1. 搜索流量选品法
此方法适用于通过关键词搜索进入店铺的流量。目标是找到搜索热度高、竞争相对较小的商品。

![](img/b7715491f60f306c4c2701dd6f52ed18_43.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_45.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_47.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_49.png)

以下是具体操作步骤：

![](img/b7715491f60f306c4c2701dd6f52ed18_51.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_53.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_55.png)

1.  登录抖音小店后台，进入“电商罗盘”。
2.  点击“引流”，选择“行业搜索词”。
3.  在左侧选择你的商品类目（例如：玩具、家纺）。
4.  在右侧的榜单中，选择按“成交指数”或“热度指数”排序。
5.  找到感兴趣的关键词（例如：“床垫四件套”），点击右侧的“搜索词诊断”。
6.  在诊断页面下方，平台会展示与该关键词相关的、表现优异的商品。这些就是你可以直接参考或上架的同款潜力商品。

![](img/b7715491f60f306c4c2701dd6f52ed18_57.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_59.png)

**核心公式**：**潜力商品 = 高成交指数关键词 + 搜索词诊断中的优质关联商品**

### 2. 商城推荐（截流）选品法
此方法适用于从抖音商城首页“猜你喜欢”等推荐位进入的流量。核心思路是“截流”，即复制已成功商家的爆款。

![](img/b7715491f60f306c4c2701dd6f52ed18_61.png)

以下是具体操作步骤：

![](img/b7715491f60f306c4c2701dd6f52ed18_63.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_65.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_67.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_69.png)

1.  在电商罗盘中，进入“商城概览”，点击“榜单”。
2.  选择“商城推荐”榜单，并筛选你的目标类目（例如：女装）。
3.  在榜单中寻找符合以下特征的店铺：
    *   **成交来源**：主要为“商品卡”（而非直播间）。
    *   **店铺特征**：商品定价常带有小数点后两位（如39.90元），这是无货源店铺的常见特征。
    *   **商品数量**：店铺商品数在100-200个左右。
4.  进入这类店铺，将其销量高（100单以上）的商品主图保存下来。
5.  使用拼多多等货源平台的“以图搜款”功能，找到同款货源，计算利润空间（售价 - 进货价 - 平台扣点）。利润可观即可上架。

**核心逻辑**：**成功店铺的爆款 = 已被市场验证的流量产品 = 你的上架首选**

## 二、基础设置与优化

选好产品后，我们需要对其进行基础优化，以提升商品的竞争力。本节将介绍标题优化和定价策略。

![](img/b7715491f60f306c4c2701dd6f52ed18_71.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_73.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_75.png)

### 1. 标题优化
标题是获取搜索流量的关键。优化标题有两个核心方法：

*   **简易法**：在抖店后台“搜索”下的“商品诊断”中，平台会直接推荐“添加热词”，可将这些词加入标题。
*   **精细法**：从“行业搜索词”中下载与你商品相关的关键词榜单，按热度排序。遵循“**首尾权重高，小词放中间**”的原则，将高热词放在标题开头和结尾。

### 2. 定价策略
建议采用“高开低走”的定价策略：
*   **初始定价**：参考同行售价，以“进货价 × 2 + 5”左右的公式初步定价。
*   **后期调整**：通过设置“限时限量购”、“优惠券”等活动来调整最终售价。这样既能预留利润空间，也能通过活动吸引用户。

## 三、起店核心流程：激活与放大流量

![](img/b7715491f60f306c4c2701dd6f52ed18_83.png)

店铺和商品上架后，如何从零启动并获得平台推荐？这是最关键的一步。本节我们将拆解完整的起店流程。

![](img/b7715491f60f306c4c2701dd6f52ed18_85.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_87.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_89.png)

### 阶段一：激活流量与出体验分
一家新店，在没有体验分时，主要只能获得搜索流量。要获得商城推荐流量，必须满足两个条件：**体验分 > 4.3** 和 **商品层级达到 G2 以上**。

![](img/b7715491f60f306c4c2701dd6f52ed18_91.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_93.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_95.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_97.png)

因此，初期核心任务是：**通过一轮“动销”，同时解决“出体验分”和“提升商品层级”两个问题。**

![](img/b7715491f60f306c4c2701dd6f52ed18_99.png)

**操作方案**：
1.  **全店动销**：为新店上架的约100个商品，每个安排1-2单销量。
2.  **执行方式**：通过合作达人直播间，以略低于售价的“AA单”形式真实卖出。这是最稳妥、最不易被清洗销量的方式。
3.  **成本计算**：动销成本（约15元/单） + 拍单垫资（商品成本价） - 店铺实收货款（买家支付价）。总成本可控制在1000元左右。
4.  **效果**：完成动销后，店铺体验分出现，大部分商品层级从G1提升至G2，商城推荐流量开始激活。

### 阶段二：数据化运营，放大流量
动销后，店铺会获得初始流量。此时需要通过数据运营，将流量放大。运营分为“广度”和“深度”两个维度。

![](img/b7715491f60f306c4c2701dd6f52ed18_101.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_103.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_105.png)

**1. 店铺广度运营（针对新店初期）**
目标是提升店铺整体权重，撬动更高层级的流量池。主要考核**全店转化率**和**成交金额**。

![](img/b7715491f60f306c4c2701dd6f52ed18_107.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_109.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_111.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_113.png)

*   **补单计算公式**：
    *   **目标订单数** = 近7天店铺总点击人数 × 同行优秀转化率百分比
    *   **需补单数** = 目标订单数 - 近7天真实订单数
    *   将需补的单量，按各流量渠道（搜索/商城推荐）的占比，分配到主要出单的商品上。

![](img/b7715491f60f306c4c2701dd6f52ed18_115.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_117.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_119.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_121.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_123.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_125.png)

*   **提升成交金额（坑产）**：
    *   **目的**：冲击类目榜单排名，获取排名权重。
    *   **方法**：分析榜单中同行的日成交额，通过“退款单”在3-5天内递增做到同等或更高坑产。
        *   **退款单操作**：为商品设置高额“私域渠道优惠券”，让熟人下单大额订单（不发货），次日退款。此举能快速拉高成交额。

**2. 单品深度运营（针对已起流量的爆款）**
当某个商品流量开始增长时，需对其进行精细化维护，考核**单品转化率**、**成交金额**和**收藏加购率**。

![](img/b7715491f60f306c4c2701dd6f52ed18_127.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_129.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_131.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_133.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_135.png)

![](img/b7715491f60f306c4c2701dd6f52ed18_137.png)

*   **单品补单公式**：
    *   **目标订单数** = 该商品当日曝光人数 × 目标转化率（如10%）
    *   **需补单数** = 目标订单数 - 当日真实订单数
*   **关键词深度运营**：
    *   在商品“引流搜索词”中，点击关键词右侧的“搜索词诊断”。
    *   平台会直接给出该关键词下，需提升的“点击人数”和“成交人数”建议值，据此进行补单。

**核心循环**：**分析实时数据 -> 计算需补量 -> 执行补单 -> 观察流量变化 -> 继续分析数据...**

## 四、总结与进阶思路

![](img/b7715491f60f306c4c2701dd6f52ed18_149.png)

本节课中我们一起学习了抖音小店从选品到起店的全流程。我们首先掌握了两种核心选品方法，然后学习了商品基础优化技巧。接着，我们深入剖析了起店的核心流程：通过一轮动销激活店铺，再通过数据化运营（先广度后深度）将流量持续放大。

**流程回顾**：
1.  **选品**：利用搜索词和榜单，找到有流量的产品。
2.  **上架优化**：优化标题，采用“高开低走”定价。
3.  **激活启动**：全店动销，快速出体验分并提升商品层级。
4.  **数据运营**：
    *   **初期（广度）**：提升全店转化率和成交金额，撬动流量层级。
    *   **中期（深度）**：针对爆款单品，维护其转化率、坑产和收藏加购数据。
5.  **持续迭代**：下架无流量商品，通过“商城推荐分析”和“搜索词诊断”不断上新潜力款。

**进阶思路**：
*   **类目选择**：优先选择有门槛的类目（如保证金高、需特殊认证），竞争相对较小。
*   **店铺裂变**：当一套流程跑通后，可复制此模式，同时运营多家店铺，形成店群。
*   **长期维护**：重视好评回复、体验分维护，并随着季节和市场变化调整选品。

![](img/b7715491f60f306c4c2701dd6f52ed18_151.png)

记住，成功的运营是**标准化流程**与**实时数据反馈**的结合。掌握这套方法，并保持高效的执行力，你就能在抖音小店中稳步前进，打造出属于自己的爆款店铺。
