# 微信视频号运营教程：P35：4. 二创书单制作方法一

## 概述
在本节课中，我们将学习如何为书单类视频进行二次创作。这是一种通过替换和重组现有视频素材，来制作原创度更高内容的方法。我们将从寻找素材开始，逐步讲解替换素材、处理音频和字幕的完整流程。

---

## 剪辑是必备技能
上一节我们介绍了视频制作的基础，本节中我们来看看剪辑技能的重要性。剪辑是制作任何类型视频的必备技能。很多新手刚开始不熟悉，觉得混剪和剪辑很难。根源在于剪辑零基础。

你可以先拿十几个视频练习，熟能生巧。剪辑并不难。花一两个小时剪辑几个视频，熟悉每个功能的作用，再看课程就会觉得非常简单。

## 书单视频的优势
接下来，我们探讨一种具体的视频类型。我建议做纯视频带货的学员尝试制作书单视频。书单视频一直是常青树类型。只要学员认真操作，基本上都能出单。

除了按照之前课程（混剪一、二、三）的方法制作，我们还可以进行二次创作。你会发现很多书单视频都是二次创作的，素材全部来源于网络，并非自己实拍。

二次创作，说白了就是将各种视频素材打乱顺序重新组合。这种方式比用单条视频混剪出来的原创度更高。

## 书单账号的底层逻辑
在开始实操前，理解逻辑很重要。做书单账号的逻辑，是通过一个视频（如影视剧片段或一段小故事）讲出一个道理，最后推荐某一本书，引起读者的购买欲，从而赚钱。

![](img/2d8ace207e072144cc32f3fcbc38cd69_15.png)

你多刷这类视频，就能明白其中的逻辑。大家先按照上面发的书单去搜索，然后通过“达人秀”功能寻找对标账号，自己刷一刷，找找感觉，就能明白这种账号的底层逻辑。

这种形式比直接做好物推荐（如直接说保鲜膜、垃圾袋有多好）的硬广要好得多。

![](img/2d8ace207e072144cc32f3fcbc38cd69_17.png)

## 二次创作实操步骤
本节我们将进入核心的实操环节，看看二次创作时具体如何操作。

### 第一步：寻找对标视频与素材
1.  打开抖音，直接搜索目标书籍，例如“女孩要懂得保护自己”。
2.  点击“商品”，再点击“销量”。找到销量几万以上的商品，其下方通常有“达人秀”。
3.  点开“达人秀”，这里汇集了大量卖家为这本书制作的视频。选择点赞量较高的视频作为参考和分析对象。

![](img/2d8ace207e072144cc32f3fcbc38cd69_19.png)

### 第二步：分析视频结构与提取素材
观看选定的对标视频。例如，一个书单视频可能包含以下结构：
*   开头：提出问题或观点（如“性教育的知识，你不教给孩子...”）。
*   中间：用多个网络素材片段（如女老师讲课、学生在操场、女生结伴出行、学生挑灯夜读）来支撑观点。
*   结尾：展示书籍封面，完成推荐。

![](img/2d8ace207e072144cc32f3fcbc38cd69_21.png)

![](img/2d8ace207e072144cc32f3fcbc38cd69_23.png)

我们的二次创作，就是将其中部分或全部网络素材片段进行替换或重新排序。

### 第三步：处理原视频与文案
1.  **保存视频**：复制视频链接，使用去水印工具（如“配音神器”）保存视频到相册。
    ```
    [操作：复制链接 -> 打开去水印工具 -> 粘贴 -> 提取视频 -> 保存]
    ```
2.  **提取文案**：使用工具提取视频的文案（字幕文本）。
3.  **生成新配音**：将提取的文案导入配音软件（如剪映的“文本朗读”或第三方工具），生成新的音频文件。注意检查错别字，并可以调整语速、添加停顿。
    ```
    [示例：选择“女生情感”音色 -> 输入文案 -> 调整语速/停顿 -> 合成配音 -> 导出音频]
    ```

### 第四步：在剪辑软件中重组素材
1.  **导入与遮盖**：将去水印的原视频导入剪映。首先，用矩形蒙版和模糊背景功能遮盖掉原视频的字幕。
    ```
    [操作：选中视频 -> 点击“蒙版” -> 选择“矩形” -> 调整覆盖字幕 -> 点击“反转” -> 点击“背景” -> 选择“画布模糊”]
    ```
2.  **关闭原声**：关闭视频轨道的原声。
3.  **替换素材片段**：
    *   **方法A（替换）**：将光标移动到要替换的片段处，点击“分割”。选中该片段，点击“替换”，从素材库或本地导入新素材。
    *   **方法B（删除并新增）**：将原片段“分割”后“删除”。然后将光标移动到该位置，点击“添加”导入新素材。新素材时长需与原片段大致相同，并关闭其原声。
4.  **寻找新素材**：可以在剪映“素材库”搜索关键词（如“老师讲课”），或直接去抖音搜索相关场景（如“女讲师培训”），下载合适的片段（尽量去水印）后使用。
5.  **重复操作**：按照视频逻辑，逐一替换其他需要更换的素材片段（如学生场景、学习场景等）。

![](img/2d8ace207e072144cc32f3fcbc38cd69_25.png)

![](img/2d8ace207e072144cc32f3fcbc38cd69_27.png)

### 第五步：添加音频与字幕
1.  **导入新配音**：将第三步生成的新配音音频导入剪辑软件，拖到音频轨道。
2.  **对齐音频**：根据画面内容，对音频进行分割、移动，使其与视频画面匹配。
3.  **添加背景音乐**：添加合适的、不喧宾夺主的背景音乐，调整音量低于配音。
4.  **生成新字幕**：使用“文本识别字幕”功能，识别新配音生成字幕。调整字幕样式和位置，覆盖在之前被模糊处理的旧字幕区域。
    ```
    [操作：点击“文本” -> “识别字幕” -> 开始识别 -> 调整样式/位置]
    ```

![](img/2d8ace207e072144cc32f3fcbc38cd69_29.png)

![](img/2d8ace207e072144cc32f3fcbc38cd69_31.png)

### 第六步：导出视频
检查整个视频的流畅度，确保音画同步、字幕清晰。最后将帧率调到60，导出视频。
```
[操作：点击“导出” -> 分辨率1080p -> 帧率60 -> 导出]
```

![](img/2d8ace207e072144cc32f3fcbc38cd69_33.png)

![](img/2d8ace207e072144cc32f3fcbc38cd69_35.png)

## 总结
本节课我们一起学习了书单视频的二次创作方法。核心思路是：**分析对标视频结构 -> 提取文案并生成新配音 -> 替换原视频中的网络素材片段 -> 组合成新视频并添加新字幕**。这种方法通过替换核心素材，能有效提升视频的原创度。关键在于多练习，熟能生巧。请务必亲自实操整个流程。
