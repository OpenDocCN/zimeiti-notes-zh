## 理解AI与运营的结合点

运营工作的核心是内容创作。无论是视频还是图文笔记，其基础都是内容。AI工具，如ChatGPT，能够辅助我们完成从选题、文案撰写到标题设计等一系列工作，降低了对特定领域专业知识深度的依赖。

然而，许多人在使用AI工具时，面临一个核心问题：沟通效率。要让AI理解我们的意图并产出令人满意的结果，关键在于提供清晰、具体的指令。

以下是知识/学习赛道的部分爆款关键词示例：
*   逆袭
*   学霸秘籍
*   干货
*   自律
*   高效
*   宝藏
*   上岸
*   复盘
*   书单
*   时间管理

![](img/859a6ea8e60e1e1fd06d8399c80918ca_65.png)

**第二步：提出具体格式要求**
明确的格式要求能让AI的输出更规整、更可用。

![](img/859a6ea8e60e1e1fd06d8399c80918ca_69.png)

![](img/859a6ea8e60e1e1fd06d8399c80918ca_71.png)

![](img/859a6ea8e60e1e1fd06d8399c80918ca_73.png)

**公式：优化指令结构**
```
角色设定 + 核心任务 + 关键词参考 + 具体格式要求
```

![](img/859a6ea8e60e1e1fd06d8399c80918ca_75.png)

![](img/859a6ea8e60e1e1fd06d8399c80918ca_77.png)

**代码示例：优化后的指令**
```
我要写一篇关于“应届生考研”的小红书笔记，请根据以下爆款关键词，帮我生成十个爆款标题。要求：标题在20个字以内，并包含合适的emoji表情。

![](img/859a6ea8e60e1e1fd06d8399c80918ca_79.png)

![](img/859a6ea8e60e1e1fd06d8399c80918ca_81.png)

爆款关键词：逆袭、学霸秘籍、干货、自律、高效、宝藏、上岸、复盘、书单、时间管理、规划、重点、冲刺、真题、心态、方法、总结、经验贴、避坑、指南。
```
