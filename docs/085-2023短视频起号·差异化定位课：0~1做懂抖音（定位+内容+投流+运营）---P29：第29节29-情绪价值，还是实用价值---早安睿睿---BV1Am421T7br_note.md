# 短视频起号·差异化定位课：29：情绪价值与实用价值 📊

![](img/e2ae5b4229212d6fd922d11b7fb628db_0.png)

![](img/e2ae5b4229212d6fd922d11b7fb628db_2.png)

在本节课中，我们将学习短视频内容创作中一个核心的底层逻辑：如何判断并选择为你的用户提供**情绪价值**还是**实用价值**。理解这一点，是做好内容选题、实现账号变现的关键第一步。

![](img/e2ae5b4229212d6fd922d11b7fb628db_4.png)

---

![](img/e2ae5b4229212d6fd922d11b7fb628db_6.png)

在深入探讨如何具体做选题之前，我们首先需要建立一个清晰的逻辑框架。这个框架的核心在于明确：你制作的视频内容，究竟要为用户提供哪种价值。

![](img/e2ae5b4229212d6fd922d11b7fb628db_8.png)

## 建立核心逻辑：利他性

首先，我们来梳理一个核心概念：**利他性**。

利他性的含义是，让你的内容能够使他人获得愉悦感，或者让他人获得实际的收获。这是因为在抖音这样的平台上，每个创作者对于观众而言，最初都是陌生人。

![](img/e2ae5b4229212d6fd922d11b7fb628db_10.png)

![](img/e2ae5b4229212d6fd922d11b7fb628db_12.png)

这与朋友圈不同。在朋友圈发布内容，亲朋好友可能会因为认识你而点赞互动。但在抖音，没有人认识你，也没有人了解你。因此，我们唯一需要满足的一点，就是**取悦观众**。只有让刷到你视频的用户觉得这条内容“有意思”或者“有用”，他们才可能点赞、评论或关注你。

![](img/e2ae5b4229212d6fd922d11b7fb628db_14.png)

**所以，在抖音创作要做的第一件事，就是坚持利他性。** 你发布的所有内容，都必须能为他人带来帮助。

![](img/e2ae5b4229212d6fd922d11b7fb628db_16.png)

## 实现利他性的两个层面

那么，如何实现利他性呢？主要有两个层面：

![](img/e2ae5b4229212d6fd922d11b7fb628db_18.png)

1.  **做到有趣**：为他人提供**情绪价值**，在精神层面让对方获得满足。
    *   例如，你的视频能让观众看完后感到治愈、开心、满足或产生强烈共鸣。

2.  **做到有用**：为他人提供**实用价值**，在物质或技能层面让对方觉得切实有用。
    *   例如，分享做菜技巧、美妆教程、装修指南、买车买房攻略等。你提供的观点或方法，能让对方获得实际的技能提升或知识增长。

![](img/e2ae5b4229212d6fd922d11b7fb628db_20.png)

上一节我们介绍了利他性的两个方向，接下来我们分别深入看看它们的具体表现和特点。

### 深入理解：情绪价值（有趣）

“有趣”的核心是在精神层面满足用户的情绪需求。这种情绪不限于欢笑，也包括共鸣、愉悦、感动、治愈等。

![](img/e2ae5b4229212d6fd922d11b7fb628db_22.png)

以下是主要提供情绪价值的赛道类型：
*   **剧情号**：传递正能量或引发深度共鸣。
*   **搞笑段子**：制造欢乐，让人开心。
*   **音乐号**：带来身心愉悦或治愈感。
*   **影视解说号**：满足娱乐需求或引发共情。
*   **舞蹈/颜值达人**：带来视觉享受和情绪满足。

![](img/e2ae5b4229212d6fd922d11b7fb628db_24.png)

创作这类内容，就实现了“有趣”层面的利他。**然而，提供纯粹的情绪价值往往难以直接变现。**

![](img/e2ae5b4229212d6fd922d11b7fb628db_26.png)

因为用户关注你是由于情绪得到了满足，但“情绪”本身难以作为商品直接售卖。无论是让人开心、共鸣还是治愈，都缺乏一个具体的产品载体。这正是许多搞笑剧情号、音乐达人号等账号变现困难的主要原因。

### 深入理解：实用价值（有用）

![](img/e2ae5b4229212d6fd922d11b7fb628db_28.png)

“有用”的核心是提供**实用价值**，即你的视频内容能让观众觉得对实际生活有所帮助。

![](img/e2ae5b4229212d6fd922d11b7fb628db_30.png)

这类内容通常集中在以下**垂直赛道**：
*   美食教程
*   美妆教学
*   种草测评
*   亲子教育
*   汽车知识
*   买房攻略
*   财经科普

![](img/e2ae5b4229212d6fd922d11b7fb628db_32.png)

观看这些博主的视频后，用户能在对应领域获得实际的帮助和提升。

![](img/e2ae5b4229212d6fd922d11b7fb628db_34.png)

**相比之下，提供实用价值更容易实现变现。**

![](img/e2ae5b4229212d6fd922d11b7fb628db_36.png)

例如：
*   用户关注美食博主，是为了提升厨艺、学习新菜。博主在内容中植入调味品，用户会认为使用该产品能提升做饭水平，从而容易产生购买。
*   用户关注汽车博主，是为了在买车时避坑、少走弯路。博主推荐相关产品或服务，转化路径就非常自然。

![](img/e2ae5b4229212d6fd922d11b7fb628db_38.png)

因此，**实用价值**因为与用户的具体需求、实际问题解决方案紧密绑定，离商业变现更近一步。

![](img/e2ae5b4229212d6fd922d11b7fb628db_40.png)

## 从变现角度出发的选择

从实现账号商业价值的角度出发，我们应该选择创作哪种价值的选题呢？

答案很明确：**我们应该优先选择为用户提供实用价值的选题。**

![](img/e2ae5b4229212d6fd922d11b7fb628db_42.png)

正如前文所分析的，情绪价值缺乏直接的变现载体，难以商业化。而我们大多数新手运营抖音账号，目的不是为了打发时间，而是为了获得最直接的变现回报。

![](img/e2ae5b4229212d6fd922d11b7fb628db_44.png)

因此，我们要做的事情就是**提供实用价值**。

## 课程总结

本节课我们一起学习了短视频内容创作的价值选择逻辑：

1.  我们首先建立了**利他性**这一核心原则，即在抖音平台必须通过提供价值来吸引陌生人。
2.  实现利他有两大路径：提供**情绪价值**（有趣）和提供**实用价值**（有用）。
3.  我们深入分析了两种价值的特点：情绪价值能满足精神需求但**难变现**；实用价值能解决实际问题且**易变现**。
4.  最终，从变现目标出发，我们得出结论：**应优先创作提供实用价值的选题**。

![](img/e2ae5b4229212d6fd922d11b7fb628db_46.png)

理清了“实用价值”与“情绪价值”的区别与选择后，下一节课我们将具体讲解“如何基于实用价值进行选题策划”，思路就会更加清晰。因为我们已经明确，选题首先要利他，其次要提供切实有用的实用价值。
