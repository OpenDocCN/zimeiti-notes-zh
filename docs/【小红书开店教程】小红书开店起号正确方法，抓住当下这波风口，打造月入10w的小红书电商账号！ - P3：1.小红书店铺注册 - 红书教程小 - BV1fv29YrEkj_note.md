# 小红书电商教程：P3：小红书店铺注册 📝

![](img/121fba6d52105d42f400b373a608b449_1.png)

在本节课中，我们将学习如何从零开始，完成小红书个人店铺的注册流程。这是开启小红书电商之旅的第一步，我们将详细拆解每一步操作，确保你能顺利完成。

---

上一节我们介绍了开店前的准备工作，本节中我们来看看如何具体操作，完成店铺的注册。

### 注册前的准备
注册店铺前，你需要准备好一部智能手机。使用个人手机或工作手机均可。首先，在这部手机上下载并安装小红书App。

### 账号登录与入口寻找
1.  **打开小红书App**，使用手机号登录你的小红书账号。
2.  登录后，点击App右下角的 **“我”** ，进入个人中心。
3.  在个人中心页面的右上角（或左上角），点击 **“三条杠”** 图标菜单。
4.  在弹出的菜单中，选择 **“创作中心”**。

以下是进入创作中心后的操作步骤：
*   在创作中心页面，找到并点击 **“更多服务”**。
*   在更多服务页面中，大约在第三排的位置，找到 **“开通店铺”** 的选项框并点击。

### 店铺开通与认证
点击“开通店铺”后，系统会跳转到一个界面。这个界面可能包含一些广告推广信息，你只需找到并点击 **“立即认证”** 按钮即可开始注册流程。

点击后，请严格按照系统提示的步骤进行操作。

### 提交材料与选择店铺类型
在认证过程中，你需要提交提前准备好的材料，主要是身份证的正反面照片。

接下来是选择店铺类型，这是一个需要注意的环节。小红书店铺主要分为以下几种类型：
*   **个人店**
*   **企业店**
*   **旗舰店** 等

对于刚接触小红书电商的新手，我们强烈建议选择开通 **个人店**。个人店门槛较低，非常适合用于前期熟悉平台的整体操作流程和规则。先通过个人店实践，积累经验，之后再考虑是否升级店铺类型，这是更稳妥的策略。

### 填写店铺信息
选择“个人店”后，需要填写店铺的基本信息。这些信息包括店铺名称、经营类目等。

**请注意**：这些信息在店铺开通后是可以修改的，但建议尽量一次性填写准确，避免后续频繁修改。

### 完成注册与等待审核
填写完所有信息并提交后，就完成了店铺的注册申请。小红书平台会对申请进行审核。

**审核时间说明**：官方反馈时间约为3天，但根据经验，通常**5分钟到1天左右**就能出结果。与其他成熟电商平台相比，小红书的系统响应可能稍慢，这是正常现象，请耐心等待。

### 重要补充：下载客服工作台
店铺注册成功后，还有一件非常重要的事情：在手机上下载 **“小红书客服工作台”** App。

这个App是独立于小红书主App的，专门用于处理店铺的售前售后咨询。当有顾客在你的店铺下单或咨询时，你需要通过这个“客服工作台”与对方进行沟通，在主App内无法直接完成聊天功能。因此，务必提前下载好。
