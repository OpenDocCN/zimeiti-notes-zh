# 小红书运营全套课程：P7：05.小红书后台的基础操作（手机端）📱

![](img/74f61991e249452c33faf84ca18c9169_0.png)

在本节课中，我们将学习如何通过手机端操作小红书商家后台（千帆系统），完成店铺的基础设置，包括地址管理、运费模板配置、优惠券创建以及支付方式开通等核心操作。

---

上一节我们了解了小红书运营的整体框架，本节中我们来看看如何通过手机APP来设置和管理你的店铺后台。

首先，你需要下载并登录“小红书千帆”商家后台APP。在手机应用商店搜索“小红书千帆”即可找到并下载。

打开APP后，你会看到首页。对于一个新店铺，首页可能显示“去初始化”的提示。页面底部有“首页”、“消息”、“全部工具”等主要导航栏。

**消息中心**是处理客服沟通和平台通知的地方。当有用户咨询商品或订单问题时，你需要在这里进行回复。同时，店铺的违规通知等重要信息也会在此显示。

接下来，我们进入核心的“全部工具”模块。这里是进行店铺日常运营管理的主要入口。

以下是“全部工具”中包含的主要功能列表：
*   **发布作品/笔记管理**：用于发布新的商品笔记或管理已发布的笔记。
*   **商品管理**：可以发布新商品，或对店铺内所有商品进行上架、下架、编辑等操作。
*   **订单管理**：处理全部订单，包括待发货、已发货、售后订单等。
*   **数据中心**：查看店铺的流量、销售等数据报表。
*   **营销工具**：创建优惠券、报名平台活动等。
*   **店铺装修**：设置店铺首页的展示样式。
*   **地址与物流**：管理发货地址和设置运费模板。

---

上一部分我们浏览了后台的主要功能，现在我们来详细讲解几个必须完成的基础设置。

**第一步：设置发货地址与退货地址**
这是开店后的首要操作。路径为：**成长中心 -> 地址管理**。
你需要填写准确的发货地址和退货地址。即使你做一件代发，也需要填写一个有效的地址。填写时需注意：
*   联系人、电话需真实有效。
*   “退货地址”一项，**务必直接填写具体地址**，不要选择“线上联系客服”。如果选择后者，平台审核可能无法通过，会影响后续的退货流程。

**第二步：配置运费模板**
在地址管理页面，你可以继续设置“运费模板”。这对于控制物流成本至关重要。
你可以创建多个模板，例如：
*   **全国包邮模板**：指定所有地区免运费。
*   **部分地区付费模板**：例如，设置新疆、西藏等偏远地区不配送或由买家承担运费。
*   系统会有默认模板，但建议根据你的商品利润和发货策略进行自定义设置。

**第三步：创建营销优惠券**
优惠券是促进销售的重要工具。路径为：**营销 -> 优惠券/活动报名**。
在这里，你可以创建多种类型的优惠券：
*   `店铺通用券`：全店商品可用。
*   `指定商品券`：仅限特定商品使用。
*   `店铺新人券`：针对首次进店用户发放。
创建时需设置优惠券金额、使用门槛、有效期和发放数量。

**第四步：开通支付方式与缴纳保证金**
为了确保店铺能正常收款并长期运营，你需要完成两项重要任务：
1.  **开通支付方式**：在设置中，按照指引绑定你的微信支付和支付宝。这样买家才能通过这两种主流方式付款。
2.  **缴纳保证金**：根据你经营的商品类目不同，需要缴纳相应金额的保证金。这是平台对商家履约能力的保障，建议打算长期经营的商家及时完成缴纳。
