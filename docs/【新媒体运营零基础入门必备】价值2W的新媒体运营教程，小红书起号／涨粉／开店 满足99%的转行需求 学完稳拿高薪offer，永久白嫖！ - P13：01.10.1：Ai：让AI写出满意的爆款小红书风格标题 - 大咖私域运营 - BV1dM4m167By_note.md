# 新媒体运营：P13：01.10.1：AI：让AI写出满意的爆款小红书风格标题 🚀

在本节课中，我们将要学习如何利用AI工具，特别是ChatGPT，来生成符合小红书平台风格的爆款标题。我们将从理解AI与运营的结合开始，逐步拆解如何通过精准的指令，让AI输出我们真正需要的高质量标题。

## 概述：AI与新媒体运营的结合

![](img/37075957b641a8d8ef3383a7dcb9f7dd_3.png)

随着ChatGPT等AI工具的普及，AI在各行各业的应用已非常广泛。对于运营人员而言，掌握AI工具变得至关重要。运营的核心是内容，无论是视频还是图文笔记，其本质都是内容的产出。AI工具可以帮助我们生成文案、设计脚本、植入关键词，甚至创作吸引人的标题。

然而，许多人在使用AI工具时，虽然知道其功能，却无法充分发挥其潜力。这通常是因为与AI的“沟通”存在问题。要让AI理解我们的意图并输出满意的结果，关键在于掌握正确的指令逻辑。本节课程将聚焦于小红书标题的创作，展示如何通过优化指令，让AI生成具有平台特色、能有效提升点击率的爆款标题。

## 如何撰写有效的AI指令

![](img/37075957b641a8d8ef3383a7dcb9f7dd_39.png)

![](img/37075957b641a8d8ef3383a7dcb9f7dd_41.png)

![](img/37075957b641a8d8ef3383a7dcb9f7dd_43.png)

![](img/37075957b641a8d8ef3383a7dcb9f7dd_45.png)

![](img/37075957b641a8d8ef3383a7dcb9f7dd_47.png)

以下是撰写有效AI指令的关键步骤，其核心逻辑是为AI提供充足的上下文和明确的要求。

![](img/37075957b641a8d8ef3383a7dcb9f7dd_49.png)

![](img/37075957b641a8d8ef3383a7dcb9f7dd_51.png)

![](img/37075957b641a8d8ef3383a7dcb9f7dd_53.png)

![](img/37075957b641a8d8ef3383a7dcb9f7dd_55.png)

1.  **明确任务与背景**：清晰告诉AI你要做什么（写小红书笔记标题）、关于什么主题（例如：应届生考研）。
2.  **提供风格参考（关键词）**：给予AI一批能代表小红书爆款风格和垂直领域痛点的关键词，让它学习模仿。
3.  **设定具体格式要求**：规定标题的字数限制、是否需要添加表情符号（Emoji）等。
4.  **组合成完整指令**：将以上所有要素组合成一段清晰的提示语。

![](img/37075957b641a8d8ef3383a7dcb9f7dd_57.png)

![](img/37075957b641a8d8ef3383a7dcb9f7dd_59.png)

![](img/37075957b641a8d8ef3383a7dcb9f7dd_61.png)

![](img/37075957b641a8d8ef3383a7dcb9f7dd_63.png)

![](img/37075957b641a8d8ef3383a7dcb9f7dd_65.png)

一个优化后的指令模板如下：
```
我要写一篇关于[你的主题，例如：应届生考研]的小红书笔记。请根据以下爆款关键词，帮我生成10个爆款标题。要求：每个标题20个字以内，并加上小红书风格的Emoji表情。

![](img/37075957b641a8d8ef3383a7dcb9f7dd_67.png)

![](img/37075957b641a8d8ef3383a7dcb9f7dd_69.png)

爆款关键词：[这里列出你准备好的关键词，例如：逆袭、上岸、干货、避坑、自律、规划、效率、书单、真题、心态]
```

![](img/37075957b641a8d8ef3383a7dcb9f7dd_71.png)

![](img/37075957b641a8d8ef3383a7dcb9f7dd_73.png)

![](img/37075957b641a8d8ef3383a7dcb9f7dd_75.png)

## 总结与下节预告

![](img/37075957b641a8d8ef3383a7dcb9f7dd_105.png)

本节课中我们一起学习了如何通过优化指令，让AI工具生成符合小红书风格的爆款标题。关键点在于：不能只给AI一个简单的任务，而要为其提供充足的“养料”——即代表平台风格的爆款关键词和具体的格式要求。这样，AI才能更好地理解我们的需求，输出高质量、高点击率的标题。

![](img/37075957b641a8d8ef3383a7dcb9f7dd_107.png)

![](img/37075957b641a8d8ef3383a7dcb9f7dd_109.png)

![](img/37075957b641a8d8ef3383a7dcb9f7dd_111.png)

![](img/37075957b641a8d8ef3383a7dcb9f7dd_113.png)

然而，仅靠我们提供的关键词，有时可能会限制AI的思维发散。为了获得更丰富、更具创意的标题，我们需要学会给AI“喂”更全面的资料。这就是我们下一节课要探讨的内容：如何通过投喂更多样化的资料，进一步激发AI的创作潜力，使其产出更令人惊喜的成果。
