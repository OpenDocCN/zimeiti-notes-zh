## 概述
在本节课中，我们将学习如何在微信视频号中开通商品橱窗功能，并完成选品与上架的具体操作步骤。这是通过视频号进行带货变现的基础环节。

## 第一步：打开微信应用 📱
首先，在您的手机上找到并打开微信应用。

![](img/86af5b5e636d7deec2d8489db54591f0_7.png)

![](img/86af5b5e636d7deec2d8489db54591f0_8.png)

![](img/86af5b5e636d7deec2d8489db54591f0_1.png)
![](img/86af5b5e636d7deec2d8489db54591f0_2.png)

![](img/86af5b5e636d7deec2d8489db54591f0_10.png)

## 第二步：进入“发现”页面
打开微信后，点击底部导航栏的“发现”选项。

![](img/86af5b5e636d7deec2d8489db54591f0_11.png)

## 第三步：找到视频号入口
在“发现”页面中，找到并点击“视频号”入口（通常是一个黄色的图标）。

![](img/86af5b5e636d7deec2d8489db54591f0_4.png)
![](img/86af5b5e636d7deec2d8489db54591f0_5.png)

## 第四步：进入个人中心
进入视频号后，点击页面右上角的个人头像图标。

![](img/86af5b5e636d7deec2d8489db54591f0_13.png)

![](img/86af5b5e636d7deec2d8489db54591f0_14.png)

![](img/86af5b5e636d7deec2d8489db54591f0_7.png)
![](img/86af5b5e636d7deec2d8489db54591f0_8.png)

## 第五步：进入创作中心
在个人页面中，将屏幕滑动到最下方，找到并点击“创作中心”。如果未显示此选项，请先发布一条视频。

![](img/86af5b5e636d7deec2d8489db54591f0_16.png)

## 第六步：找到带货中心
进入创作中心后，找到“带货中心”并点击进入。

![](img/86af5b5e636d7deec2d8489db54591f0_10.png)
![](img/86af5b5e636d7deec2d8489db54591f0_11.png)

**核心操作**：如果您尚未开通橱窗权限，点击“带货中心”后会跳转到开通页面。通常需要完成实名认证并缴纳保证金（例如，选择“达人”身份并交费）。

![](img/86af5b5e636d7deec2d8489db54591f0_18.png)

## 第七步：进入选品中心
成功开通权限后，再次进入“带货中心”。您将看到类似“天道酬勤399”这样的橱窗界面。点击“选品中心”。

![](img/86af5b5e636d7deec2d8489db54591f0_20.png)

![](img/86af5b5e636d7deec2d8489db54591f0_22.png)

![](img/86af5b5e636d7deec2d8489db54591f0_13.png)
![](img/86af5b5e636d7deec2d8489db54591f0_14.png)

![](img/86af5b5e636d7deec2d8489db54591f0_24.png)

## 第八步：选择并上架商品
在选品中心，您可以看到大量可供带货的商品。以下是操作步骤：

1.  **浏览商品**：例如，找到一个“菜板”商品。
2.  **添加橱窗**：点击商品下方的“加橱窗”按钮。
3.  **确认上架**：点击后，该商品即成功添加到您的视频号橱窗中。

![](img/86af5b5e636d7deec2d8489db54591f0_16.png)

**核心代码/流程示意**：
```
操作流程 = 选品中心 -> 选择商品 -> 点击“加橱窗” -> 商品上架成功
```

![](img/86af5b5e636d7deec2d8489db54591f0_26.png)

## 案例分享：从零到爆单的启示 💡
上一节我们介绍了基础的操作步骤，本节中我们来看看一个真实案例，理解正确方法的重要性。

![](img/86af5b5e636d7deec2d8489db54591f0_28.png)

很多新手独自操作时会感到进展缓慢，这通常不是因为不够努力，而是因为缺乏经验，容易走弯路。有人指导可以显著提高效率。

以下是一个学员案例：
*   **初始状态**：一位退休在家的学员，视频号只有2个粉丝（还是好友），账号内容杂乱，定位不清晰、不垂直、不活跃。
*   **关键调整**：在指导下，明确了账号定位，并使用了称为“黑科技爆款参谋”的工具来寻找最新爆款商品。
*   **成果展示**：经过约5天的学习与实践，该学员在一天内爆单，数据如下：
    *   **成交单数**：3247单
    *   **当日收入**：97122.6元

![](img/86af5b5e636d7deec2d8489db54591f0_30.png)

![](img/86af5b5e636d7deec2d8489db54591f0_30.png)

这个案例说明了几个核心要点：
1.  **账号基础至关重要**：混乱的账号（定位不清、内容不垂直）难以获得流量和转化。
2.  **工具与选品是放大器**：借助数据分析工具（爆款参谋）可以高效找到潜力商品。
3.  **方法决定效率**：系统性的学习和正确的操作流程，能让普通人也能快速取得显著成果。

**核心公式**：
```
成功变现 ≈ 清晰的账号定位 + 垂直的优质内容 + 高效的爆款选品 + 正确的操作流程
```
