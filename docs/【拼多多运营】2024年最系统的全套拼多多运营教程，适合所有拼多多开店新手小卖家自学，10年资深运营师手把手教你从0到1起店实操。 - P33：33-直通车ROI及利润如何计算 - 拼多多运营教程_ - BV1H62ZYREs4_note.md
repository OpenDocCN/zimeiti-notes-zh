# 拼多多运营教程：P33：直通车ROI及利润计算 💰

在本节课中，我们将学习拼多多直通车推广的核心概念，重点掌握ROI（投资回报率）和利润的计算方法。理解这些公式是判断推广盈亏、优化投放策略的基础。

## 付费推广的两种模式

![](img/98dda83f84b27a4cabee7784ab6fa100_1.png)

![](img/98dda83f84b27a4cabee7784ab6fa100_2.png)

拼多多平台主要提供两种付费推广模式：**标准推广**和**全站推广**。它们适用于不同的运营阶段。

![](img/98dda83f84b27a4cabee7784ab6fa100_3.png)

![](img/98dda83f84b27a4cabee7784ab6fa100_4.png)

上一节我们介绍了推广的基本概念，本节中我们来看看如何具体操作标准推广。

![](img/98dda83f84b27a4cabee7784ab6fa100_6.png)

### 标准推广：新品启动利器

![](img/98dda83f84b27a4cabee7784ab6fa100_7.png)

标准推广适用于新品新链接的启动期。其优势在于前期流量相对精准，有助于提升商品的转化率。

以下是创建标准推广计划的具体步骤：

1.  **进入后台**：登录拼多多商家后台，点击“标准推广”。
2.  **添加产品**：选择你需要推广的商品，将其添加到推广计划中。
3.  **设置日限额**：根据你的预算设置每日最高花费。预算少可设为100元，预算充足可设为200-300元。
4.  **设置出价**：需要设置四种出价：
    *   **成交出价**：**不得超过单笔利润**。例如，商品售价50元，利润20元，则成交出价最高为20元。
    *   **询单/关注/收藏出价**：系统会给出推荐出价区间。**前期建议按系统推荐最低出价的70%进行设置**。

![](img/98dda83f84b27a4cabee7784ab6fa100_9.png)

![](img/98dda83f84b27a4cabee7784ab6fa100_10.png)

按照以上方式设置完成后，先让计划运行7天，目标是快速积累**20单以上**的成交。这是因为新计划处于“学习期”，积累足够订单后，系统才能探索并打上精准的人群标签。

![](img/98dda83f84b27a4cabee7784ab6fa100_11.png)

## 全站推广与ROI核心计算

当标准推广积累超过20单成交后，可以考虑转向全站推广。全站推广与标准推广的关键区别在于出价方式：它要求设置 **“目标投产比（ROI）”**，而非具体的金额。

![](img/98dda83f84b27a4cabee7784ab6fa100_13.png)

![](img/98dda83f84b27a4cabee7784ab6fa100_14.png)

要玩转全站推广，必须掌握以下几个核心计算公式。

![](img/98dda83f84b27a4cabee7784ab6fa100_16.png)

### 1. 实际投产比（ROI）

实际投产比衡量的是广告花费带来的交易额效果。

**公式**：`实际ROI = 交易额 / 总花费`

**示例**：直通车花费1000元，带来5000元交易额。
`实际ROI = 5000 / 1000 = 5`

![](img/98dda83f84b27a4cabee7784ab6fa100_18.png)

![](img/98dda83f84b27a4cabee7784ab6fa100_19.png)

### 2. 保本投产比（盈亏平衡线）

![](img/98dda83f84b27a4cabee7784ab6fa100_20.png)

![](img/98dda83f84b27a4cabee7784ab6fa100_21.png)

这是全站推广中需要设置的核心数值，达到此ROI则不亏本。

![](img/98dda83f84b27a4cabee7784ab6fa100_22.png)

![](img/98dda83f84b27a4cabee7784ab6fa100_23.png)

**公式**：`保本ROI = 客单价 / 利润`

![](img/98dda83f84b27a4cabee7784ab6fa100_24.png)

![](img/98dda83f84b27a4cabee7784ab6fa100_25.png)

**示例**：商品售价100元，利润50元。
`保本ROI = 100 / 50 = 2`

但需要注意，达到保本ROI并不代表绝对安全，因为运营中还存在退款、平台扣点等额外成本。

### 3. 盈利计算

![](img/98dda83f84b27a4cabee7784ab6fa100_27.png)

要精确计算推广带来的真实利润，需使用以下公式。

![](img/98dda83f84b27a4cabee7784ab6fa100_28.png)

![](img/98dda83f84b27a4cabee7784ab6fa100_29.png)

**公式**：`盈利 = (交易额 - 退款金额) × 利润率 - 总花费`

![](img/98dda83f84b27a4cabee7784ab6fa100_30.png)

**示例**：交易额5000元，退款500元，利润率50%，直通车花费1000元。
`盈利 = (5000 - 500) × 50% - 1000 = 1250元`

### 4. 净投产比（计入退款）

为保证在存在退款的情况下仍不亏损，需要计算净投产比。

**公式**：`净投产比 = 保本ROI / [(1 - 退货率) × 1.1]`

![](img/98dda83f84b27a4cabee7784ab6fa100_32.png)

![](img/98dda83f84b27a4cabee7784ab6fa100_33.png)

**示例**：保本ROI为2，退货率为10%（即0.1）。
`净投产比 = 2 / [(1 - 0.1) × 1.1] = 2 / (0.9 × 1.1) ≈ 2.44`

这意味着，当你的直通车ROI达到**2.44**时，即使有退款也不会亏本。

### 5. 最佳投产比（盈利区间）

我们开车的最终目的是盈利，因此需要找到一个健康的盈利区间。

**公式**：`最佳投产比 = 净投产比 × 1.3`

![](img/98dda83f84b27a4cabee7784ab6fa100_35.png)

**示例**：净投产比为2.44。
`最佳投产比 = 2.44 × 1.3 ≈ 3.17`

**结论**：当直通车ROI维持在 **2.44 到 3.17** 之间时，推广处于良性盈利状态。若ROI低于2.44，则可能亏损；高于此区间则盈利更佳。

2.  我们重点学习了用于**全站推广**的五个关键公式：
    *   **实际ROI** (`交易额 / 总花费`)：衡量广告效果。
    *   **保本ROI** (`客单价 / 利润`)：盈亏平衡点。
    *   **盈利计算** (`(交易额-退款)×利润率 - 花费`)：计算真实利润。
    *   **净投产比**：计入退款成本后的安全线。
    *   **最佳投产比**：理想的盈利区间。
