## 渠道一：微软Edge浏览器与Copilot 🌐

首先，我们来看看如何通过微软的Edge浏览器体验AI功能。这是优先推荐给大多数用户的渠道。

微软基于ChatGPT打造了一个名为“Copilot”的功能，集成在Edge浏览器中。它基于GPT-4模型打造，智能程度与4.0版本有一定匹配度，是普通用户可便捷使用的方式。

使用Copilot有两个前提条件：
1.  登录微软账号（可使用Outlook邮箱注册）。
2.  需要“梯子”（VPN）才能访问该服务。

Copilot通常集成在浏览器右侧工具栏。在最新版的Windows 11中，它也可能出现在任务栏。打开后，界面会提供三种对话模式：
*   **更有创造力**：AI的回答更具开放性和创造性。
*   **平衡**：介于创造力和精确性之间，适合一般用途。
*   **更精确**：回答更为严谨和准确，适合专业内容。

**示例：生成短视频脚本**
我们可以使用“更有创造力”模式，输入指令：
```
请帮我写一个关于装修的短视频脚本，时长控制在3分钟左右。
```
AI会开始生成包含场景、画面、台词和时长的详细脚本。

Copilot的优势在于它能分析你当前打开的网页内容。其缺点在于：
1.  回答精准度略低于官方ChatGPT 4.0。
2.  连续对话功能受限，无法在当前窗口无限次提问。
3.  记忆功能有所缺失，难以基于历史对话优化后续回答。

如果你的Edge浏览器没有Copilot图标，可以尝试下载最新版的Edge Dev（测试版）。你可以在微软官网的Edge下载页面，找到“Dev频道”进行下载。或者，在浏览器设置中检查并更新至最新版本。

## 渠道二：第三方集成平台与共享账号 🔌

上一节我们介绍了系统集成的工具，本节中我们来看看由开发者搭建的第三方平台。这些平台通过调用ChatGPT的API，让国内用户也能方便地使用。

以下是几个可用的第三方平台：

**1. 灵感AI**
这个网站集成了多种AI模型，包括GPT-3.5、GPT-4（含联网版）以及国内主流模型（如讯飞星火、文心一言等）。
*   **优势**：模型丰富；新注册用户有一定次数的GPT-4免费使用额度。
*   **缺点**：生成速度可能较慢。
*   **费用**：提供付费套餐，如19.9元包月。

**2. Wee Chat**
这个网站提供GPT-3.5和GPT-4模型。
*   **优势**：响应速度较快；具备语音回答功能。
*   **缺点**：免费额度（1万token）通常仅限GPT-3.5使用；需要“梯子”。
*   **费用**：提供套餐，例如约5美元兑换100万文字token。

**3. ChatGPT共享账号平台**
这是一个共享官方ChatGPT账号的网站。账号状态以颜色标示：
*   **绿色**：空闲。
*   **橙色**：近期被使用过。
*   **红色**：正在被使用或使用频繁。
*   **金色边框**：代表是付费的ChatGPT Plus（GPT-4）账号。

你可以尝试使用这些共享账号登录官方ChatGPT网站。GPT-3.5的免费账号较多，而GPT-4的Plus账号相对稀缺且可能有使用频率限制（例如每3小时约30条消息）。这是一个应急使用GPT-4的途径。

以下是几个主要的国内AI模型：

**1. 百度文心一言**
百度的AI对话产品。提供基础版和付费的“文心一言4.0”版本。在撰写文案、获取灵感等基础运营任务上足够使用。你可以尝试其付费版以体验更强大的功能。

**2. 字节跳动豆包AI**
字节跳动推出的AI助手。同样可以完成脚本生成、文案撰写等任务，回答结构清晰。

**3. 其他模型**
此外，还有**阿里巴巴的通义千问**、**昆仑万维的天工AI**等，都是可供选择的国内大模型。

这些国内模型在通用任务上表现不错，但与最顶尖的GPT-4在复杂逻辑、创造性及深度理解上仍存在差距。选择时，可以挑选一个用起来最顺手、最能提升你工作效率的工具进行深入学习。

## 总结与对比

本节课中，我们一起学习了体验和使用ChatGPT 4.0的几种主要渠道：

1.  **微软Copilot**：集成度高，使用方便，但功能有阉割，需要“梯子”。
2.  **第三方平台**：提供更多模型选择和免费额度，但速度和稳定性不一，多数需“梯子”。
3.  **共享账号**：可直接体验官方服务，但账号不稳定，有使用限制，需“梯子”。
4.  **国内大模型**：访问便捷，完全免费或成本较低，适合基础应用，但能力上限相对固定。
