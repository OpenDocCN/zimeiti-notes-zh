# 微信视频号运营教程：P22：3.如何给短视频配音 🎤

在本节课中，我们将学习如何为短视频进行配音。有时，原始视频的配音效果不佳，我们可以将其替换为新的配音。本节将介绍配音工具的选择、使用方法以及如何将新配音应用到视频中。

## 配音工具介绍

上一节我们介绍了视频剪辑的基础操作，本节中我们来看看如何为视频更换配音。配音工具有很多种，其中“配音神器”是一个功能强大的选择。

![](img/41bffae68e0606eb445577c150fe7a74_1.png)

除了配音功能，“配音神器”还集成了去水印、文案提取、修改视频MD5值等多种工具。修改MD5值可以增加视频的独特性，但仅靠修改MD5值发布到视频号是不够的，后续仍需进行视频混剪操作。关于如何混剪，我们将在下节课详细讲解。

## 如何搜索相关工具

![](img/41bffae68e0606eb445577c150fe7a74_3.png)

无论是去水印、修改MD5还是配音，我们都可以在微信中搜索到对应的小程序工具。

以下是搜索方法：
*   在微信搜索框输入关键词，如“修改MD5小程序”或“去水印”。
*   选择“小程序”分类，即可找到大量相关工具。

![](img/41bffae68e0606eb445577c150fe7a74_5.png)

我推荐使用“配音神器”和“青抖”，其中“配音神器”是首选。如果某个工具暂时无法使用，直接搜索并更换另一个即可。“配音神器”的优势在于，除了配音功能需要消耗金币（或开通会员），其他如文案提取、去水印等功能都是免费的。

![](img/41bffae68e0606eb445577c150fe7a74_7.png)

## “配音神器”使用与金币获取

“配音神器”提供了每日签到领取金币的福利。生成一段视频配音通常需要消耗3个金币。

以下是金币获取与会员说明：
*   **每日签到**：坚持签到，获得的金币数量会逐渐增加。
*   **开通会员**：如果金币不够用，可以考虑开通会员。会员分为月卡（29元/月）或终身卡（99元）。

## 制作配音的步骤

现在，我们来看看如何使用“配音神器”制作一段配音。

以下是具体操作步骤：
1.  打开“配音神器”小程序，点击配音功能。
2.  清除默认文字，将你准备好的文案脚本输入到文本框中。
3.  在声音库中选择你喜欢的声音类型（如女声、男声等），可以点击试听。
4.  选择好声音后，点击“合成配音”。
5.  合成完成后，选择导出方式。通常可以选择“保存配音视频到相册”或“保存字幕视频到相册”，两者导出的视频样式不同。

**注意**：在合成和保存过程中，请勿退出小程序界面。

![](img/41bffae68e0606eb445577c150fe7a74_9.png)

## 在剪映中替换视频原声

获得配音文件后，我们需要在剪辑软件（如剪映）中替换掉视频的原始声音。

![](img/41bffae68e0606eb445577c150fe7a74_11.png)

以下是替换音频的步骤：
1.  将目标视频导入剪映。
2.  关闭视频的“原声”。
3.  点击“添加音频” -> “提取音乐”，然后选择你刚刚保存的配音视频文件。
4.  导入后，视频的原声即被新的配音替换。

**代码示例**（示意流程）：
```plaintext
1. 导入视频素材 -> 关闭原声
2. 添加音频 -> 选择“提取音乐”
3. 从相册导入配音文件 -> 完成替换
```
