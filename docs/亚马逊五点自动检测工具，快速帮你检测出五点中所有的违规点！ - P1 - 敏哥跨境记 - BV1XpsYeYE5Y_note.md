# 亚马逊五点描述优化教程：P1：使用卖家精灵工具进行五点描述违规检测与优化

在本节课中，我们将学习如何利用卖家精灵工具，快速检测并优化亚马逊商品五点描述中的违规内容，以符合平台最新的政策要求。

## 概述：亚马逊五点描述新规

亚马逊五点描述的新规已正式实施。此次更新主要限制使用特殊符号、表情以及特定短语。一旦亚马逊检测到五点描述存在违规，平台将直接对商品详情页进行修改。因此，卖家必须确保自己的五点描述整改到位。

## 卖家精灵五点检测工具介绍

上一节我们了解了新规的重要性，本节中我们来看看如何借助工具高效完成合规检查。卖家精灵平台已上线五点描述违规检测工具。

该工具能自动扫描五点描述，识别所有潜在的违规点，并提供具体的优化建议。卖家可以根据这些建议进行针对性修改，确保内容完全合规。

以下是使用该工具的具体操作步骤：

1.  登录卖家精灵工具。
2.  找到并点击五点描述编辑区域上方的“违规检测”功能按钮。
3.  系统将自动运行检测程序。
4.  查看检测报告及系统给出的优化建议。
5.  根据建议内容逐一修改五点描述。

建议各位卖家尽快使用此工具检查自己的商品列表。
