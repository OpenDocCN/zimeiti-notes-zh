# 微信公众号运营：P45：3.04-微信上墙增粉02~2 🎯

在本节课中，我们将学习如何正确设置微信上墙活动中的抽奖顺序，并了解其他互动功能（如摇一摇）的配置方法。我们将通过具体的后台操作步骤，确保活动流程顺畅，避免奖品顺序错乱的问题。

---

上一节我们介绍了微信上墙的基本概念，本节中我们来看看如何设置抽奖环节，特别是解决奖品顺序问题。

![](img/a5c60c4cfe961a803763549c5c5b99bf_1.png)

正确的流程是：先展示并抽取测试奖品，然后按照既定顺序（如三等奖、二等奖、一等奖）依次抽取。如果顺序设置不当，例如抽完测试奖后直接跳到一等奖，就会打乱流程。

![](img/a5c60c4cfe961a803763549c5c5b99bf_3.png)

以下是解决此问题的核心方法：

1.  **预先排序奖项**：在后台设置奖项时，必须按照你希望抽取的顺序排列。例如，可以设置为“测试奖 -> 三等奖 -> 二等奖 -> 一等奖”或“测试奖 -> 一等奖 -> 二等奖 -> 三等奖”。
2.  **使用“上一项/下一项”切换**：在手机控制端，通过“上一项”或“下一项”按钮，在已排好序的奖项列表间切换，从而控制前台大屏幕显示的当前奖项。

所以，如果你发现默认的抽奖顺序是“测试奖 -> 一等奖 -> 二等奖”，这是不正确的。现在我们需要进入后台进行调整。



![](img/a5c60c4cfe961a803763549c5c5b99bf_1.png)

首先，我需要编辑奖项数据。当前还没有中奖数据，我将进行编辑。



![](img/a5c60c4cfe961a803763549c5c5b99bf_5.png)

![](img/a5c60c4cfe961a803763549c5c5b99bf_3.png)

我将第一个奖项设置为“三等奖”，并设置抽取三人。请记住这个顺序。接着，将下一个奖项设置为“二等奖”，奖品数量和价值保持不变，然后点击保存。

![](img/a5c60c4cfe961a803763549c5c5b99bf_7.png)

此时，无论奖项列表如何显示，它们的顺序已经是排好的（测试奖、三等奖、二等奖、一等奖）。因此，你可以先抽取测试奖，然后在手机上点击“下一项”，将大屏幕切换到三等奖进行抽取，再点“下一项”切换到二等奖，以此类推。

如果你还不清楚，接下来看我操作一遍就会明白。现在奖项已设置完成，我需要到前台进行演示。



![](img/a5c60c4cfe961a803763549c5c5b99bf_9.png)

![](img/a5c60c4fe961a803763549c5c5b99bf_5.png)

首先，将大屏幕界面调至基础内容。用户需要扫码参与，但作为运营人员，我需要在后台进行控制。



![](img/a5c60c4cfe961a803763549c5c5b99bf_7.png)

我需要用自己的手机扫描屏幕上的控制台二维码。扫描后，手机上将出现控制界面。



![](img/a5c60c4cfe961a803763549c5c5b99bf_9.png)

我可以在手机上控制大屏幕。例如，点击一下，二维码隐藏；再点击，二维码显示。点击“弹幕”，活动现场的弹幕消息就会显示在大屏幕上。

活动开始时，我可以引导观众：“大家好，欢迎参与我们的微信互动。有任何问题或想说的话，请扫码并回复‘666’参与互动。” 观众扫码回复后即可发送弹幕。之后，我可以隐藏二维码，专注于与弹幕互动。

如果需要抽奖，直接点击“抽奖”。现在大屏幕上已显示奖项。我要先关闭弹幕。然后在手机上操作：当前显示的是“测试奖品”，我可以点击“开始抽奖”进行测试。由于只有我一人参与，自然是我中奖。

关键点来了：现在需要切换到正式奖项。手机控制端有“上一项”和“下一项”按钮。我点击“下一项”，屏幕切换到“三等奖”；再点“下一项”，切换到“二等奖”；继续点“下一项”，切换到“一等奖”。这样，顺序就完全可控了。

理解我刚才的操作了吗？如果我没有预先将三等奖、二等奖排序，那么点击“下一项”时，顺序可能会变成“测试奖 -> 二等奖 -> 一等奖 -> 三等奖”，这就乱了。所以必须让它们的顺序是连续的（如123或321）。如果是倒序321，则使用“上一项”按钮来切换。

因此，你需要记住的流程是：设置时，先设一个测试奖作为默认展示项，然后按321或123的顺序排好其他奖项。抽奖时，先抽测试奖，然后用“上一项”或“下一项”切换到下一个奖项进行抽取。

抽奖结束后，你可以点击“消息”回到主界面，再点击“弹幕”重新开启弹幕互动。当然，除了消息和抽奖，平台还提供了其他丰富的互动功能。

---

在理解了抽奖设置后，我们来看看其他活跃现场气氛的功能。在“互动现场”中，“竞技PK”类功能就非常有效。

例如，我可以演示如何设置一个“摇一摇”活动。点击新建“摇一摇”，输入标题，比如“新媒体最佳臂力排行榜”。活动持续时间可设置为30秒，让现场几百人同时摇动手机。

你可以为活动添加趣味描述，比如“摇得最快，证明是单身贵族！” 在双十一等场合特别合适。我们这里保持正经，就写“最佳臂力排行榜，限时30秒”。

“最大摇动次数”通常无需修改。“显示最终排名”数量可以设为5或10名，然后根据排名发放奖品。我这里先默认显示10名。

活动时，现场参与者的微信头像和昵称会实时显示在大屏幕上，并伴有动态效果。我们可以选择默认效果。

“参与条件”可以设置，例如“本轮前10名不能参与后续轮次”，以避免同一批人反复获奖，当然也可以选择“不限”。

你还可以“按名次设置奖品”，为第1到第10名分别配置奖品。这里我们先不设置，直接点击保存。



![](img/a5c60c4cfe961a803763549c5c5b99bf_11.png)



![](img/a5c60c4cfe961a803763549c5c5b99bf_12.png)

保存后，活动处于“展示中但未开始”的状态。在前台大屏幕上，用户可以向下滑动看到这个“摇一摇”活动。



![](img/a5c60c4cfe961a803763549c5c5b99bf_14.png)

我可以先关闭弹幕显示。然后，点击“开始摇一摇”。所有通过公众号关键词链接进入该活动的用户，在手机页面上都可以参与摇一摇。我作为主持人喊“3，2，1，开始！”并点击开始，倒计时就会启动，大家开始摇手机。

摇动结束后，系统会显示排名。目前因为没有其他用户加入，所以无法演示摇动效果。这就是“摇一摇”活动的基本设置。
