# 剪映电脑版教程：P19：17.抖音核心剪辑功能—文本 📝

在本节课中，我们将要学习剪映电脑版中一个至关重要的功能——文本。一条完整的视频不仅包含画面和音频，文本（或称字幕）也是传递主题、思想和相关信息的关键元素。通过添加文本，可以让观众更准确、更快速地理解视频内容。

上一节我们介绍了音频功能，本节中我们来看看如何为视频添加和设置文本。

![](img/df97241233c080a09f4f412113682034_1.png)

## 概述：文本功能简介

剪映的文本功能允许我们自由地为视频添加字幕。添加的字幕可以单独设置样式，包括字体、字号、颜色、间距以及动画效果。

此外，剪映还提供了两个高效的功能：**字幕自动识别**和**歌词识别**。在已添加音频的情况下，可以使用这两个功能一键为视频快速生成字幕，极大地降低了操作难度和时间成本。

文本功能区域还包含**贴纸**的添加选项。关于文本功能的基本介绍就到这里，接下来我们进入实际操作环节。

![](img/df97241233c080a09f4f412113682034_3.png)

## 功能实操演练

文本功能位于一级菜单栏，点击“文字”按钮即可进入。其下包含四个主要选项：**新建文本**、**识别字幕**、**识别歌词**和**贴纸**。

以下是各功能的详细操作步骤。

### 1. 新建文本（手动添加）

手动添加文本是最基础的方式。点击“新建文本”，在输入框中键入所需文字，例如“剪映视频剪辑教程”。

输入完成后，时间线上会新增一个文本轨道。点击底部“样式”选项，可以对文字进行个性化设置。

以下是可调整的样式选项：
*   **字体**：可以更改文字的字体，例如选择流行的“新青年体”。
*   **样式**：包括描边、标签、阴影等。可以设置描边的颜色（红、灰、黄、蓝）和粗细。
*   **花字**：可以直接应用抖音官方提供的各种特色艺术字体。
*   **气泡**：可以选择综艺节目或VLOG中常见的对话气泡样式。
*   **动画**：可以为文字添加入场、出场及循环动画。例如，添加一个“螺旋上升”的入场动画，并设置动画的持续时间。

![](img/df97241233c080a09f4f412113682034_5.png)

设置完成后保存，即可在预览区看到添加了动画和样式的字幕效果。

### 2. 识别字幕（语音转文字）

当视频本身包含人物对话或旁白时，可以使用“识别字幕”功能自动生成字幕。

![](img/df97241233c080a09f4f412113682034_7.png)

![](img/df97241233c080a09f4f412113682034_8.png)

首先，确保视频轨道上已添加包含人声的音频。然后，点击“文字”菜单下的“识别字幕”按钮，再点击“开始识别”。

系统会自动分析音频，并将识别出的文字以字幕形式添加到视频中。识别完成后，可以像手动添加的字幕一样，对自动生成的字幕进行样式和位置的调整。

### 3. 识别歌词（音乐转字幕）

![](img/df97241233c080a09f4f412113682034_10.png)

![](img/df97241233c080a09f4f412113682034_12.png)

如果视频中添加了带有歌词的歌曲，可以使用“识别歌词”功能快速生成歌词字幕。

![](img/df97241233c080a09f4f412113682034_14.png)

首先，将带歌词的歌曲添加到音频轨道。然后，点击“文字”菜单下的“识别歌词”按钮，并点击“开始识别”。

![](img/df97241233c080a09f4f412113682034_16.png)

识别完成后，歌词会以字幕形式出现在视频画面上。同样，我们可以对这些歌词字幕进行美化，例如调整位置、更换字体、添加描边或动画效果，以提升视频的整体意境。

### 4. 贴纸功能

除了文字，在“文字”功能区还可以找到“贴纸”选项。这里提供了丰富的动态或静态贴纸素材，可以用于装饰视频画面，增加趣味性。

![](img/df97241233c080a09f4f412113682034_18.png)

## 总结

![](img/df97241233c080a09f4f412113682034_19.png)

![](img/df97241233c080a09f4f412113682034_21.png)

本节课我们一起学习了剪映的文本功能。通过文本功能，可以快速为视频添加字幕，并能对字幕的颜色、字体、字号、动画、间距等进行一系列个性化调整。

**核心操作可归纳为以下公式：**
**添加字幕 = 新建文本（手动） 或 识别字幕/歌词（自动）**
**美化字幕 = 调整样式（字体/颜色/动画）**

![](img/df97241233c080a09f4f412113682034_23.png)

合理使用文本功能，能显著提升视频信息传达的效率和内容质量。至此，关于剪映核心剪辑功能中文本部分的讲解就全部结束了。
