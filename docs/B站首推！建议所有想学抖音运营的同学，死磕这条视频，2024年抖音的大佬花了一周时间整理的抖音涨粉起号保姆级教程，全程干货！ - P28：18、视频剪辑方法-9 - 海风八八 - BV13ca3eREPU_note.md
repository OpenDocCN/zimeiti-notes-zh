## 剪辑前的准备 📱

![](img/fcd202286bfa04f4059aeb49dbeb5e77_13.png)

上一节我们介绍了视频策划与拍摄，本节中我们来看看如何将拍摄的素材剪辑成片。首先需要准备好剪辑工具。

![](img/fcd202286bfa04f4059aeb49dbeb5e77_15.png)

![](img/fcd202286bfa04f4059aeb49dbeb5e77_17.png)

以下是剪辑前的准备工作：
*   **软件选择**：使用**剪映**App进行剪辑。请认准黑色图标版本，它提供手机版和电脑版。
*   **素材导入**：打开剪映，点击“开始创作”，从相册中选择需要剪辑的视频片段，点击“添加”导入。

## 基础剪辑操作 ✨

![](img/fcd202286bfa04f4059aeb49dbeb5e77_27.png)

![](img/fcd202286bfa04f4059aeb49dbeb5e77_29.png)

导入素材后，我们开始进行基础的视频处理。核心操作围绕“选中”、“分割”和“删除”展开。

![](img/fcd202286bfa04f4059aeb49dbeb5e77_31.png)

![](img/fcd202286bfa04f4059aeb49dbeb5e77_33.png)

![](img/fcd202286bfa04f4059aeb49dbeb5e77_35.png)

以下是基础剪辑的步骤：
1.  **调整画面构图**：点击轨道上的视频片段，向右滑动工具栏，找到并点击“编辑”。点击“旋转”调整画面方向；点击“裁剪”可以调整画面比例（如9:16、1:1、4:3等）。
2.  **剪除无用片段**：播放视频，找到需要删除的片段（如开头沉默、结尾多余动作）。将时间线（白色竖线）移动到分割点，点击视频片段，选择“分割”。然后点击需要删除的片段，选择“删除”。
    *   **核心操作公式**：`选中视频 -> 分割 -> 选中多余片段 -> 删除`
3.  **添加背景音乐**：点击底部工具栏的“音频”。如果已有配音文件，可点击“提取音乐”导入。若无，则点击“添加音乐”，从推荐或分类中选择合适的音乐，点击“使用”。
4.  **剪辑音频**：如果音乐过长，操作与剪辑视频相同：选中音频轨道，在需要截断处“分割”，然后删除多余部分。可以选中音频，调整“音量”大小。

## 添加与美化字幕 🆎

![](img/fcd202286bfa04f4059aeb49dbeb5e77_45.png)

![](img/fcd202286bfa04f4059aeb49dbeb5e77_47.png)

视频和音频剪辑完成后，需要添加字幕。剪映提供了高效的自动识别功能。

![](img/fcd202286bfa04f4059aeb49dbeb5e77_49.png)

![](img/fcd202286bfa04f4059aeb49dbeb5e77_51.png)

以下是添加字幕的步骤：
1.  点击底部工具栏的“文字”。
2.  选择“识别字幕”，点击“仅视频”（避免识别到背景音乐歌词），然后开始识别。
3.  识别完成后，字幕会自动添加到时间轴。检查是否有错别字，双击字幕框即可修改。
4.  可以进一步美化字幕：选中字幕，修改字体、样式、颜色等，使其与视频风格更搭配。

## 导出高清视频 ⚙️

![](img/fcd202286bfa04f4059aeb49dbeb5e77_59.png)

![](img/fcd202286bfa04f4059aeb49dbeb5e77_61.png)

所有剪辑步骤完成后，在导出前需要进行关键设置，以保证视频清晰度。

![](img/fcd202286bfa04f4059aeb49dbeb5e77_63.png)

以下是导出高清视频的设置：
*   点击右上角“导出”按钮。
*   在导出设置中，调整参数：
    *   **帧率 (fps)**：设置为 **30**。这是抖音平台的推荐帧率。
    *   **分辨率**：尽可能拉满（如1080p）。分辨率越高，视频越清晰。
*   **导出公式**：`帧率 = 30`，`分辨率 = 最高`
*   设置完成后，点击“导出”。导出过程中请勿锁屏或切换App，耐心等待完成即可。导出后可直接分享到抖音。

## 口播提词工具推荐 🎤

![](img/fcd202286bfa04f4059aeb49dbeb5e77_73.png)

![](img/fcd202286bfa04f4059aeb49dbeb5e77_75.png)

对于需要录制口播视频的同学，背诵台词是一大难题。接下来介绍一个能提升效率的辅助工具。

![](img/fcd202286bfa04f4059aeb49dbeb5e77_77.png)

![](img/fcd202286bfa04f4059aeb49dbeb5e77_79.png)

以下是“爱提词”App的使用方法：
1.  **下载软件**：在应用商店搜索下载“爱提词”App。
2.  **导入台词**：打开App，点击右下角“+”号，创建新提词板。输入标题（如“情感语录”），将准备好的台词粘贴或输入到正文区，点击保存。
3.  **提词拍摄**：保存后，点击“开始提词”，选择“录制模式”。此时手机屏幕会以悬浮窗形式滚动显示台词，你可以一边看词一边录制，无需背诵。
4.  **美化与设置**：
    *   点击屏幕中央的“设置”图标，可以调整台词字体大小、滚动速度等。
    *   点击左下角的“美颜”图标，可以开启瘦脸、大眼、磨皮、美白等美颜效果，让拍摄效果更自然美观。
5.  **灵感参考**：该App内还提供“爆款文案”板块，涵盖职场、情感、生活等多个领域，可供创作时参考。
