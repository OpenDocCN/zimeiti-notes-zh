# 新媒体运营零基础教程：P17：02.2：工具篇-AI生成PPT 🚀

在本节课中，我们将学习如何使用一款高效的AI工具来自动生成PPT。这款工具能根据你提供的主题或现有文档，自动梳理大纲并生成详细的演示文稿内容，非常适合快速制作专业演示材料。

## 工具简介与核心功能

上一节我们介绍了多种运营工具，本节中我们来看看这款AI生成PPT工具。它主要提供两种生成模式：

1.  **主题生成模式**：输入一个主题，工具会结合生成式AI（如ChatGPT）自动创作全文并生成PPT。
    *代码示例：`输入主题 -> AI生成全文 -> 自动排版为PPT`*
2.  **文档生成模式**：上传现有的文档（如TXT、PDF），AI助手会自动总结文档大纲，并基于此大纲生成PPT内容。
    *公式表示：`上传文档 -> AI提取大纲 -> 基于大纲生成PPT内容`*

我试用过许多AI生成PPT的工具，这款在内容理解和生成方面表现突出。

## 编辑与优化生成结果

PPT内容生成后，我们可以对其进行调整和美化。

工具右侧提供了编辑面板，允许用户进行多项调整。以下是主要的编辑功能：

*   **更换模板**：工具提供多种模板可供选择，切换迅速。虽然当前模板库可能不算庞大，但足以满足基础需求，并且未来很可能会增加。
*   **调整内容**：你可以直接修改PPT页面上的文字、形状等元素。
*   **本地深度编辑**：你可以将生成好的PPT下载到本地，使用PowerPoint或Keynote等软件进行更精细的调整。
