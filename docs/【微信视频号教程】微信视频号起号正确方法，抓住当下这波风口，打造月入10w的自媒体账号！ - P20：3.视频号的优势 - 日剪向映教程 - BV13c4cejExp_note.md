在本节课中，我们将要学习微信视频号的三大核心优势。理解这些优势，能帮助你明白为何视频号是当前值得投入的创业风口。

![](img/c4a2fed5fe86a780c592d4353b445f18_5.png)

第三个点也是很重要的点，就是稳定性和持久性。就一句话，各位，你有可能卸载抖音、快手、小红书，但是你绝对不会卸载咱们的微信。为什么这样讲呢？因为微信现在已经成为我们的习惯了。以前联系亲朋好友是打手机电话，现在基本上都是微信消息、微信语音、微信视频。而且还有一个点，这个是在我们去年到去年的时候，我们过春节的时候。大家可以看一下，总共有多少红包？各位同学除夕夜抢了多少红包呢？抢了有50.8个亿。过年期间抢红包已经变成我们一个习俗了。所以说也是一样的，我们做任何事情你不可能说一天两天，一两个月就不搞了，重新再来肯定是不划算的。所以说视频号是非常非常稳定的。

## 优势三：平台全力扶持与巨大增长

最重要的点，一个东西他到底能不能搞得起来，还是得看背后的大老板，他愿不愿意去投钱。腾讯集团已经强烈表示了2024年会全力发展咱们的视频号。怎么去开展呢？这个不是张老师我讲的，是我们这个马化腾马总讲的。我告诉你视频号在今年的机会是非常之大的。2024年全力发展视频号的直播电商，视频号的春天已经来了，马化腾最新发言。腾讯公司的董事会主席、首席执行官马化腾先生，我们来看一下。整个视频号在2023年增长了多少？**增长300%**。这个不是说三倍，而是300%。看清楚，同比增长47%，GMV增长50.5%，直播带货规模增长7倍，超过1000亿。这种种数据说明视频号的风口已经来了。

## 课程预告与准备

既然你们今天来了，张老师我就会给大家去讲。说实话真的非常之简单。所以说你今天不去跳广场舞，不去打麻将，你怎么抓住机会这么大呢？包括很多同学刚才很着急说老师赶快教我怎么做变现玩法，不要着急玩法他已经来了。今天会给大家带来两个非常暴力的玩法，一个是我们的保底变现玩法，一个叫我们的翻身暴富玩法。保底变现玩法，你学会它，日入三位数是没有任何问题的，也就是说一天挣个几百块钱。翻身暴富玩法，你学会它日入5位数也是没问题，一天挣个万把块钱。可能很多同学有点说老师这么挣钱吗？听清楚了，就是这么的挣钱，就是这么的暴力。但是在讲玩法之前，我要确认一件事情。因为刚刚在正式开讲之前，官方跟张老师说，说要照顾很多的新手同学。所以说张老师我现在要确认一下，这些新手同学，张老师我现在说话的这个语速、大小，大家觉得合不合适。如果觉得合适，觉得可以同学给老师打上可以两个字。如果说老师你这个说话语速有点快了，那张老师我就讲慢一点，老师你这个声音小一点，我就调大一点。因为正式上课可能要测试咱们的设备。
