# 从0开始做公众号：P1：使用布局功能提升排版自由度

在本节课中，我们将要学习微信公众号排版中的“布局”功能。通过使用布局，你可以将推文页面划分为多个独立区域，从而像使用手机分屏一样，在同一个屏幕上同时操作不同的内容模块，极大地提升排版的灵活性和创意空间。

## 什么是布局功能？

上一节我们介绍了排版的基础概念，本节中我们来看看“布局”功能。布局功能的核心是将整个推文编辑区域分割成多个部分。你可以将其理解为将手机屏幕进行分屏，每一块分屏区域都可以独立地插入图片、文字或样式模板。

## 如何在编辑器中找到并使用布局？

理解了布局的概念后，我们进入实践环节，看看如何在编辑器中操作。

我们打开135编辑器，进入编辑界面。首先，点击左侧工具栏的“样式”选项。接下来，点击“布局”，并选择“基础布局”。

在编辑器中，系统提供了8个常用的基础布局效果。以下是这些布局的简要说明，你可以根据自己的排版需求进行选择：
*   两栏布局：将区域平均分为左右两部分。
*   三栏布局：将区域平均分为左、中、右三部分。
*   左图右文布局：左侧区域侧重放图，右侧区域侧重放文。
*   上文下图布局：上方区域放文字，下方区域放图片。
*   以及其他几种常见的图文组合布局。

我现在选择“两栏布局”为大家进行演示。

## 布局功能的应用与优势

选择了“两栏布局”后，编辑区域就会被划分为两个独立的小屏幕。你可以在一边放置图片，另一边放置样式模板或文字。

如果不使用布局功能，整个编辑区域就是一个整体，只能进行单一的模板操作，自由度较低。布局功能最大的好处就是提供了极高的自由度，你可以自由组合内容。

我们再来和单个无布局的排版方式对比一下。单个布局下，所有元素必须线性排列；而使用分栏布局后，图文可以并排展示，信息呈现更紧凑、美观。

当然，布局的玩法不仅限于“一边图片，一边文字”这种基础形式。接下来，我们尝试一点更高级的组合应用。

## 实现高级滑动效果

![](img/bcf785c7e42d493fb2e1a4c021f6b8b2_1.png)

除了静态分栏，我们还可以利用布局组合实现动态效果，例如内容滑动。

首先，选择一个“两栏布局”。然后，在其中一个分栏内再次点击“布局”，选择一个“上下滑动”的布局样式。

接着，找到一个多文字的样式模板，将其放入这个新创建的滑动布局中。这个操作相当于在一个固定的容器里，嵌套了一个可以滚动查看内容的区域。

**代码/结构示意：**
```
[两栏布局]
├── [左侧分栏：普通内容]
└── [右侧分栏：上下滑动布局]
    └── [可滚动的内容块，如长图或长文]
```

拥有了这个滑动效果后，你就可以在里面放置一张长图。当你想在推文中展示历史文章合集时，就可以将往期推文生成一张长图，然后通过这种方式嵌入进来，读者上下滑动即可浏览。

需要注意的是，如果直接将长图放在普通的右侧分栏里，是不会产生滑动效果的。因此，务必记得在目标分栏内先插入一个具有滑动效果的布局容器。

但是，这种方法是一种利用现有布局工具实现滑动效果的简便方法。更多复杂的动画效果和更正规的H5制作方法，需要进入135编辑器的“SVG编辑器”模块中实现。

## 课程总结

本节课中我们一起学习了微信公众号排版中的布局功能。我们了解了布局的核心是将编辑区域分块，从而提升排版自由度；学习了在135编辑器中如何找到并使用基础布局；并通过实例演示了如何组合布局来实现图文并排以及高级的滑动展示效果。掌握布局功能，能让你的公众号推文结构更灵活，视觉表现更丰富。
