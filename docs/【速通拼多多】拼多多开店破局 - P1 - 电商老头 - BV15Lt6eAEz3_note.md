# 拼多多开店教程：P1：必须关闭的店铺设置与费用规避 🛒

在本节课中，我们将学习拼多多开店初期必须关闭的几个关键设置。这些设置如果保持开启，可能会导致平台自动扣费，严重侵蚀你的利润。了解并关闭它们，是店铺健康运营的第一步。

---

## 🎯 核心问题：为何“卖得越多，亏得越快”？

有一种说法是“卖的越多，死的越快”。这并非危言耸听。其根本原因在于，拼多多平台存在许多自动扣费的设置，项目繁多。扣费严重的订单，一单甚至可能被扣除几十到几百元。如果不提前关闭这些设置，等到结算时就会发现利润所剩无几，甚至亏损。

那么，具体是哪些设置需要警惕呢？

---

## ⚙️ 必须关闭的三大设置

以下是三个必须重点关注并关闭的设置项，它们分别是：消费者体验计划、仅退款和自动跟价。

1.  **消费者体验计划**
    *   **作用与风险**：该计划旨在提升消费者购物体验，但会从每笔订单中扣除一定费用。
    *   **关闭方法**：进入【售后管理】页面，找到消费者体验计划。你可以看到每笔订单会被扣除几毛到几块钱不等的费用。点击页面右上角的“更多”选项，然后选择“停用计划”即可关闭。

2.  **自动跟价**
    *   **作用与风险**：此功能会自动让你的商品价格跟随竞争对手调整，可能导致你陷入无利可图的价格战。
    *   **关闭方法**：前往【营销活动】页面，找到相关设置旁的“问号”图标，查看自动跟价说明。将页面滑动至最底部，即可找到关闭该功能的选项。

3.  **仅退款**
    *   **特殊说明**：“仅退款”功能没有直接的关闭按钮。但是，通过合理的店铺设置和运营策略，可以大幅减少此类情况的发生，预计能减少80%的“仅退款”订单。具体的设置方法需要参考更详细的专项教程。
