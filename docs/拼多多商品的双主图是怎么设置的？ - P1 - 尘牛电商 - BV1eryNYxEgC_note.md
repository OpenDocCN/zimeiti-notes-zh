# 拼多多商品主图设置教程：P1：双主图设置详解 🛍️

在本节课中，我们将学习拼多多商家必须掌握的商品主图设置，特别是“双主图”的概念与上传方法。正确设置主图是获取平台推荐流量的关键步骤。

## 概述：什么是双主图？

在拼多多购物时，您可能会注意到同一个商品页面中，有时会出现方形图和长图两种不同比例的图片。这两种图片共同构成了商品的“双主图”。它们分别在不同的展示位出现，是商家获取流量的重要工具。

上一节我们介绍了双主图的基本概念，本节中我们来看看它们具体是如何设置的。

## 设置入口与步骤

设置双主图的操作在商家后台完成。以下是具体操作路径：

1.  登录拼多多商家后台。
2.  找到并进入 **`商品管理`** 模块。
3.  点击 **`商品素材`** 选项。
4.  进入 **`图文素材`** 页面，即可开始上传。

在图文素材页面，您会看到三个主要的图片上传区域，对应三种不同类型的商品主图。

## 三种主图类型详解

以下是拼多多要求商家上传的三种核心商品图片，它们各自有不同的作用和展示场景。

*   **白底图**
    *   **作用**：每个商品都必须上传。上传白底图有助于商品在平台首页获得更多的推荐流量。
    *   **公式**：`白底图` = `首页推荐流量`的必备条件。

![](img/ad6be11a4f5d5f039e948de4869e966c_1.png)

*   **长图**
    *   **作用**：能够展现更多的商品细节和内容。对于服饰、百货、箱包等类目的商品，使用长图能更有效地吸引买家，从而**提高主图的点击率**。
    *   **适用类目**：服饰、百货、箱包等。

*   **场景图（活动图）**
    *   **作用**：这是报名参加平台营销活动的必备条件之一。上传后，商品将在首页、各类频道活动页面等场景获得展示机会。
    *   **代码逻辑**：`if (想要报名平台活动) { then 必须上传场景图 }`

![](img/ad6be11a4f5d5f039e948de4869e966c_2.png)

## 核心建议与总结

为了保证商品能最大范围地曝光并获取流量，建议商家将以上三种主图全部上传。这样，商品就有机会在不同的展示位置（如首页推荐、活动频道、搜索列表等）出现，从而**分别获取到不同展现位的流量**。

![](img/ad6be11a4f5d5f039e948de4869e966c_3.png)
