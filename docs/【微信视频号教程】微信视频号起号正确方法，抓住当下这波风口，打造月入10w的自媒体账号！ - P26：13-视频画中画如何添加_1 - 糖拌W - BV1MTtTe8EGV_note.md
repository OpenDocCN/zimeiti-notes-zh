# 微信视频号教程：P26：视频画中画添加方法 🎬

在本节课中，我们将学习如何在微信视频号剪辑中添加“画中画”效果。画中画功能允许你在主视频之上叠加额外的图片或视频，从而丰富内容层次。我们将详细介绍三种画中画素材的添加与处理方法。

## 概述

画中画是视频剪辑中常用的效果，它能让你的视频内容更丰富。本节教程将分步讲解如何添加自拍图、剧情视频和矢量图/视频这三种画中画素材，并调整它们与主视频的融合效果。

## 添加画中画的基本步骤

首先，在剪辑界面找到“画中画”功能入口并点击。接着，选择“新增画中画”来添加素材。画中画素材主要分为三种类型：自拍图片、剧情视频以及矢量图或矢量视频。

以下是添加画中画的通用操作流程：

1.  **选择素材**：从手机相册或素材库中选择要添加的图片或视频。
2.  **调整层级**：添加后，画中画素材会默认覆盖在主视频上方。
3.  **关闭声音**（针对视频素材）：为避免与主视频声音冲突，需要关闭画中画视频的原始声音。
4.  **应用混合模式**：使用“混合模式”功能（如绿色、叠加等）使画中画素材与主视频融合。
5.  **调整不透明度**：将不透明度数值调低（通常在5以内），使叠加效果更自然。
6.  **调整时长与位置**：确保画中画素材的时长不超过主视频，并可通过拖拽调整其出现的时间点。

## 第一种：添加自拍图片或视频

上一节我们介绍了基本流程，本节中我们来看看如何添加自拍类素材。

**操作步骤如下：**

1.  点击“新增画中画”，从相册中选择一段自拍视频或图片。
2.  素材添加后，如果它是视频，**第一步必须关闭其原声**。点击选中画中画轨道，在出现的工具栏中找到“音量”图标，将音量滑块拖至最左侧。
    *   **关键区分**：操作前需确认选中的是画中画轨道（其四周有白色框线），而非主视频轨道。
3.  接着，点击工具栏中的“混合模式”。通常选择“绿色”或“叠加”效果。
4.  选择混合模式后，将下方的“不透明度”数值调整到5以内（例如1、2、3）。
5.  点击确认（对勾图标）。此时，画中画素材会以半透明状态叠加在主视频上。
6.  最后，将画中画素材的尾部拖拽至与主视频对齐，确保全程覆盖。

## 第二种：添加剧情类视频

处理完自拍素材，接下来我们学习添加已有的剧情视频片段作为画中画。

以下是添加剧情视频的要点：

*   尽量选择时长短于主视频的剧情片段。
*   添加后，**首要操作同样是关闭该视频的原声**。
*   随后应用“混合模式”（如绿色）并调低不透明度，步骤与添加自拍视频一致。
*   如果添加的剧情视频时长过短，可以选中该片段，在工具栏中向后滑动找到“复制”功能，进行复制拼接，以延长其持续时间。
*   如果画中画视频过长，可以拖拽其两端进行裁剪，或使用“分割”工具删除多余部分，确保其不超出主视频时长。

## 第三种：添加矢量图或矢量视频

最后，我们来看看如何利用素材库中的矢量素材。

**具体操作如下：**

1.  点击“新增画中画”，选择“素材库”。
2.  在素材库中，你可以选择纯色背景（视为图片）或各种矢量动画视频。
3.  选择所需的矢量视频，点击添加。
4.  **对于视频素材，重复关键步骤：关闭原声 -> 选择“混合模式”（如绿色）-> 调整不透明度至2左右**。
5.  同样，需要控制矢量视频的时长，可通过“复制”拼接或“分割”删除来匹配主视频长度。
6.  所有画中画素材都可以在屏幕上通过双指捏合进行放大或缩小，或通过长按拖拽来移动位置。

## 总结

本节课中我们一起学习了微信视频号中添加画中画的方法。核心在于掌握**关闭画中画视频原声**、使用**混合模式**以及调整**不透明度**这三个关键步骤，无论添加哪种类型的素材都适用。通过灵活运用画中画功能，你可以为视频添加个性化的装饰、补充情节或营造氛围，而观众在最终成片中可能并不会明显察觉这些叠加的元素，从而达到润物细无声的增强效果。
