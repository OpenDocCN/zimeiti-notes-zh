# 抖音运营：P13：6.视频剪辑实操（上） ✂️

在本节课中，我们将学习如何使用剪映App进行视频剪辑的基础操作，包括如何导入素材、制作主题背景、添加文字与配音，以及调整视频比例。这些是制作一个合格短视频的关键第一步。

![](img/a169543a3ffbed73fad6ec89a52dbc92_1.png)

现在，请确保你能看到我的手机屏幕。首先，我们需要在自己的手机上找到并打开“剪映”软件。手机端和电脑端的剪映图标完全相同。

![](img/a169543a3ffbed73fad6ec89a52dbc92_3.png)

找到软件后，直接点击打开。

![](img/a169543a3ffbed73fad6ec89a52dbc92_5.png)

![](img/a169543a3ffbed73fad6ec89a52dbc92_1.png)

![](img/a169543a3ffbed73fad6ec89a52dbc92_7.png)

打开后，你会看到一个带有加号的“开始创作”界面。

![](img/a169543a3ffbed73fad6ec89a52dbc92_9.png)

![](img/a169543a3ffbed73fad6ec89a52dbc92_3.png)

我们直接点击这个加号，然后选择并添加一个自己准备好的视频素材。

![](img/a169543a3ffbed73fad6ec89a52dbc92_11.png)

![](img/a169543a3ffbed73fad6ec89a52dbc92_5.png)

![](img/a169543a3ffbed73fad6ec89a52dbc92_13.png)

素材添加完成后，界面如下所示。

![](img/a169543a3ffbed73fad6ec89a52dbc92_15.png)

![](img/a169543a3ffbed73fad6ec89a52dbc92_7.png)

![](img/a169543a3ffbed73fad6ec89a52dbc92_17.png)

首先需要明确一点：我们做视频时，素材可以找到，但视频的主题背景通常需要自己制作。那么如何制作呢？在视频轨道的右侧，有一个加号按钮。

![](img/a169543a3ffbed73fad6ec89a52dbc92_11.png)

![](img/a169543a3ffbed73fad6ec89a52dbc92_19.png)

点击这个加号，然后找到剪映自带的“素材库”。

![](img/a169543a3ffbed73fad6ec89a52dbc92_21.png)

![](img/a169543a3ffbed73fad6ec89a52dbc92_13.png)

“素材库”的位置已圈出。点开素材库后，在“热门”分类下可以看到许多背景主题。

![](img/a169543a3ffbed73fad6ec89a52dbc92_23.png)

![](img/a169543a3ffbed73fad6ec89a52dbc92_17.png)

背景种类非常多。例如图中圈出的这些类型都可以使用。但更建议选择简单明了的背景，让用户一眼就能明白视频的主题。我们选择一个黑色的背景添加进去。

![](img/a169543a3ffbed73fad6ec89a52dbc92_19.png)

背景添加完成后，界面如下。

![](img/a169543a3ffbed73fad6ec89a52dbc92_21.png)

背景有了，接下来需要添加主题文字。以下是添加主题文字的步骤：

1.  点击左下角的“文字”选项。
2.  点击最左侧的“新建文本”。
3.  此时会出现一个文本框。

![](img/a169543a3ffbed73fad6ec89a52dbc92_25.png)

![](img/a169543a3ffbed73fad6ec89a52dbc92_23.png)

现在，输入我们视频的主题。主题必须明确。例如，我们今天要制作“沈腾的搞笑名场面”，那么“沈腾”就是核心主题人物。你找的所有素材中都必须有他出现。如果素材中没有他，则无法使用，需要重新寻找。

![](img/a169543a3ffbed73fad6ec89a52dbc92_27.png)

首先，输入主题人物和内容，例如：“沈腾综艺爆笑名场面”。如果你的素材是影视相关，就输入“影视”；如果是综艺相关，就输入“综艺”。

主题确定后，我们看到文字是白色的，不太醒目。人们通常更喜欢色彩鲜艳、抓人眼球的东西。接下来，我们不去手动调整“样式”和“花字”，而是直接使用更便捷的“文字模板”。

![](img/a169543a3ffbed73fad6ec89a52dbc92_29.png)

![](img/a169543a3ffbed73fad6ec89a52dbc92_25.png)

文字模板有收费和免费的，可根据实际情况选择。我们随便选择一个模板，然后点击右上角的对勾确认。

![](img/a169543a3ffbed73fad6ec89a52dbc92_31.png)

![](img/a169543a3ffbed73fad6ec89a52dbc92_27.png)

主题背景制作完成后，我们可以手动拖动文本框右下角，将文字放大一些，使其更清晰。

![](img/a169543a3ffbed73fad6ec89a52dbc92_33.png)

![](img/a169543a3ffbed73fad6ec89a52dbc92_29.png)

现在，主题有了，但视频不能没有声音。用户不喜欢看“哑巴”视频。那么，如何添加声音呢？如果你会配音，可以打开手机录音机自己录制。如果不会，也没有关系，我们可以使用剪映的“文本朗读”功能。

![](img/a169543a3ffbed73fad6ec89a52dbc92_31.png)

点击“文本朗读”后，会出现许多声音选项，包括方言、萝莉音、播音腔等。

![](img/a169543a3ffbed73fad6ec89a52dbc92_33.png)

![](img/a169543a3ffbed73fad6ec89a52dbc92_35.png)

目前，方言音在各大平台的使用率相对较低，但很有吸引力。例如，广西方言就很有趣。我们可以试听几种方言效果，然后选择一个喜欢的，比如广西方言。选择完成后，点击右下角的对勾，系统会自动下载该音频。

![](img/a169543a3ffbed73fad6ec89a52dbc92_35.png)

音频下载好后，我们可以试听一下：“沈腾综艺爆笑名场面”。现在声音有了。接着，手动选中黑色背景轨道，将其长度缩短，使其与音频时长同步。

![](img/a169543a3ffbed73fad6ec89a52dbc92_37.png)

主题背景制作完成后，接下来是处理视频本身。首先确认素材中是否有沈腾这个人。如果有，素材可用；如果没有，则不能用。

在讲解过程中，我会边操作边讲解细节。首先，第一点：视频必须是 **16:9** 的横屏比例。这是大多数平台的硬性标准（抖音除外，抖音要求 **9:16** 的竖屏）。如果比例不对，平台审核可能无法通过。

那么，如何调整比例呢？例如，当前这个视频明显不是16:9的横屏。最简单的方法是找到手机下方的“比例”选项。

点击“比例”，选择“16:9”的横屏选项（图中已圈出）。然后观察视频素材是否随之调整。如果视频画面移动并适配了，说明这种方法有效。

如果像这个视频一样，选择比例后画面一动不动，又该如何手动调整呢？这样可以提高素材的可用率。以下是手动调整的步骤：

1.  点击左下角的“剪辑”选项。
2.  选择里面的“编辑”。
3.  再选择“裁剪”。

![](img/a169543a3ffbed73fad6ec89a52dbc92_39.png)

在裁剪过程中，请注意：原视频自带的字幕不要保留，直接裁掉，我们之后可以自己添加。同时，视频边框要少留一些，留多了会出现黑边，影响观感。

![](img/a169543a3ffbed73fad6ec89a52dbc92_39.png)

![](img/a169543a3ffbed73fad6ec89a52dbc92_41.png)

裁剪完成后，点击右下角的对勾。

![](img/a169543a3ffbed73fad6ec89a52dbc92_43.png)

![](img/a169543a3ffbed73fad6ec89a52dbc92_41.png)

现在可以看到，视频画面放大了，但它仍然不是一个标准的16:9横屏（如果是，就不会有黑色背景了）。接下来，找到手机右下角的“背景”选项。

![](img/a169543a3ffbed73fad6ec89a52dbc92_43.png)

点击“背景”，然后选择“画布模糊”，并调整到第一个清晰度。

![](img/a169543a3ffbed73fad6ec89a52dbc92_45.png)

现在再看，视频是否已经变成了一个完整的16:9横屏了呢？

![](img/a169543a3ffbed73fad6ec89a52dbc92_45.png)

本节课中，我们一起学习了视频剪辑的初步操作：从导入素材、制作主题背景、添加文字和配音，到关键的视频比例调整。掌握这些基础步骤，是制作出符合平台要求的短视频的前提。下节课我们将继续深入讲解更多的剪辑技巧。
