## 千川投放的核心考核指标

![](img/d4d677094974605b6ef4aba258eefada_4.png)

上一节我们介绍了千川计划的基础搭建。本节中，我们来看看驱动千川投放决策的三个核心数据指标。理解这些指标是计算盈亏平衡点的基础。

千川投放主要考核以下三个指标：
*   **退货率**：用户收到商品后申请退货的比例。
*   **利润率**：扣除所有成本（除广告费外）后，单笔订单的利润占售价的比例。
*   **客单价**：平均每个订单的成交金额。

## 如何计算保本ROI

了解了核心指标后，我们需要一个公式来将它们联系起来，以判断广告投放是否保本。这个公式就是保本ROI计算公式。

![](img/d4d677094974605b6ef4aba258eefada_6.png)

千川的保本ROI计算公式如下：

**保本ROI = 1 / [(1 - 退货率) × 利润率]**

![](img/d4d677094974605b6ef4aba258eefada_8.png)

这个公式的推导逻辑是：
1.  `(1 - 退货率)` 代表你实际成功销售出去的金额比例，即**真实成交额**。
2.  `真实成交额 × 利润率` 等于**实际利润**。
3.  保本ROI意味着广告花费等于实际利润，因此用 `1 / 实际利润率` 即可得出。

![](img/d4d677094974605b6ef4aba258eefada_10.png)

为了让计算更直观，我们可以用具体金额来演算。以下是两个不同类目的计算示例：

**示例一：高客单数码产品**
*   客单价：1500元
*   退货率：20%
*   利润率：50%
计算过程：
1.  真实成交额：1500 × (1 - 0.2) = 1200元
2.  实际利润：1200 × 0.5 = 600元
3.  实际利润率：600 / 1500 = 0.4
4.  保本ROI：1 / 0.4 = 2.5
**结论：** 该数码产品的千川广告ROI达到2.5即可保本。

**示例二：高客单女装**
*   客单价：1000元
*   退货率：70%
*   利润率：50%
计算过程：
1.  真实成交额：1000 × (1 - 0.7) = 300元
2.  实际利润：300 × 0.5 = 150元
3.  实际利润率：150 / 1000 = 0.15
4.  保本ROI：1 / 0.15 ≈ 6.67
**结论：** 该女装产品的千川广告ROI需要达到约6.67才能保本。

![](img/d4d677094974605b6ef4aba258eefada_12.png)

**核心要点：** 只要你的广告ROI达到或超过这个保本值，就意味着千川投放本身不亏损。在此前提下，你就可以放心地放大广告预算，获取更多流量和订单。

## 出价与调价操作指南

![](img/d4d677094974605b6ef4aba258eefada_14.png)

计算出保本ROI后，下一步就是将其应用到广告计划的出价中。以下是具体的操作步骤和策略。

![](img/d4d677094974605b6ef4aba258eefada_16.png)

![](img/d4d677094974605b6ef4aba258eefada_18.png)

首先，你需要根据保本ROI和客单价，反推出初始出价。
**出价公式：出价 = 客单价 / 保本ROI**

例如：
*   客单价100元，保本ROI为2，则初始出价应为 `100 / 2 = 50` 元（以“成交”为目标出价）。
*   客单价1500元，保本ROI为2.5（如前文数码产品），则初始出价应为 `1500 / 2.5 = 600` 元。

对于新账户，较高的出价（如600元）可能无法立刻跑动（即获得曝光）。这时，你需要采用“阶梯提价”策略。

以下是提价的具体操作流程：
1.  以计算出的出价（如600元）创建计划。
2.  若计划消耗缓慢或没有曝光，不要直接修改原计划的价格（频繁改价会影响系统稳定性）。
3.  **正确的做法是：** 复制原计划，在新计划中将出价提高约5%（例如从600元提高到630元）。
4.  观察15-30分钟，若新计划仍无起色，再次复制计划，继续提价5%（如提高到660元）。
5.  重复此过程，直到计划开始稳定消耗。

同理，如果某个计划消耗过快、成本过高，你需要“阶梯降价”。

以下是降价的具体操作流程：
1.  复制当前跑得飞快的计划。
2.  在新计划中将出价降低约10%。
3.  观察效果，如果依然消耗过快，则继续复制并降价10%。
4.  通过多次复制和降价，逐步将广告成本调整至理想区间。

![](img/d4d677094974605b6ef4aba258eefada_20.png)

**关键提醒：** 所有调价操作都应通过“复制计划-设置新出价”来完成，而非直接修改已有计划。这是维持计划稳定性和获得系统推荐流量的重要技巧。

本节课中，我们一起学习了千川投放的底层逻辑。首先，我们明确了退货率、利润率和客单价三个核心指标。接着，我们掌握了计算保本ROI的公式与方法，这是判断投放盈亏的尺子。最后，我们详细讲解了如何根据保本ROI进行科学出价，以及通过“复制计划”进行阶梯式调价的具体操作。理解并运用这些知识，是实现千川广告高效起号和稳定增长的基础。下一节课，我们将深入讲解具体创建广告计划的策略与技巧。
