# 剪映教程：P4：剪映APP界面结构拆解 📱

在本节课中，我们将要学习剪映APP的整体界面结构。了解每个功能模块的位置和用途，是熟练使用剪映进行视频创作的第一步。

![](img/2e3d261a7ce873a27e0588397d78008e_1.png)

## 概述

剪映APP的界面设计非常简洁直观，主要包含四个核心模块。熟悉这四个模块是我们开启剪辑之路的正确方式。

## 界面四大模块介绍

以下是剪映APP底部的四个主要功能模块，我们将逐一进行介绍。

1.  **剪辑**
    *   启动剪映APP后，默认进入的首页就是“剪辑”页面。
    *   要开始剪辑视频，需要点击顶部的 **`开始创作`** 按钮，创建新项目并导入素材。
    *   在首页界面中，还可以查看和管理之前剪辑过的项目草稿。

2.  **剪同款**
    *   剪映APP内置了大量短视频模板。
    *   对于新手，如果看到喜欢的爆款视频但不会制作，可以使用“剪同款”功能。
    *   用户只需选择喜欢的模板，然后按照要求导入自己的视频或图片素材，即可一键生成风格相似的同款视频。
    *   模板按类别划分，如卡点视频、毕业季、旅行Vlog等，方便用户查找。

![](img/2e3d261a7ce873a27e0588397d78008e_3.png)

3.  **消息**
    *   剪映APP与抖音账号互通。
    *   授权登录抖音账号后，可以在此模块接收抖音官方的消息、作品被点赞或评论的通知。

4.  **我的**
    *   这是个人信息页面。
    *   成功授权抖音账号后，可以在此查看和编辑个人资料，如头像、昵称、抖音号、粉丝量等。
    *   也可以直接跳转到关联的抖音账号主页。

![](img/2e3d261a7ce873a27e0588397d78008e_5.png)

上一节我们介绍了四大模块的理论知识，本节中我们来看看软件的实际操作界面。

### 1. 剪同款功能演示

启动剪映APP，首先进入的是“剪同款”页面。这里展示了官方推荐的大量模板。

![](img/2e3d261a7ce873a27e0588397d78008e_11.png)

*   我们可以浏览“热门”、“卡点”、“Vlog”等不同类别的模板。
*   选择任意模板（例如一个旅行Vlog模板）可以预览效果。
*   点击 **`剪同款`** 按钮，根据提示导入所需数量的素材片段。
*   导入后，可以对每个素材片段进行编辑，例如替换、裁剪。
*   也可以修改模板中的预设文字。
*   预览效果满意后，点击 **`导出`**，设置分辨率（通常推荐1080P）即可生成视频。

![](img/2e3d261a7ce873a27e0588397d78008e_13.png)

### 2. 剪辑功能界面详解

回到首页，点击“剪辑”模块，我们来了解核心的剪辑界面。

点击顶部的 **`开始创作`**，导入素材并 **`添加到项目`**，就进入了剪辑主界面。

![](img/2e3d261a7ce873a27e0588397d78008e_15.png)

主界面主要分为三个区域：

![](img/2e3d261a7ce873a27e0588397d78008e_17.png)

*   **视频预览窗口**：顶部区域，用于实时预览视频编辑效果。
*   **视频轨道（时间轴）**：中间区域，所有视频、音频、文字等素材按时间顺序排列在此。中间有一条垂直的 **`时间线`**，它是当前操作的基准点，用于定位、分割和添加效果。
*   **底部工具栏**：底部区域，包含各种编辑工具。
    *   **一级工具栏**：包括 **`剪辑`**、**`音频`**、**`文字`**、**`画中画`**、**`特效`**、**`滤镜`**、**`比例`**、**`调节`** 等主要功能按钮。
    *   选中轨道上的素材后，下方会弹出对应的二级编辑菜单，如 **`分割`**、**`变速`**、**`动画`**、**`蒙版`** 等。

## 总结

![](img/2e3d261a7ce873a27e0588397d78008e_27.png)

本节课中，我们一起学习了剪映APP的界面结构。它主要由 **`剪辑`**、**`剪同款`**、**`消息`** 和 **`我的`** 四个模块组成。对于内容创作者而言，最核心的是 **`剪辑`** 和 **`剪同款`** 功能。熟悉这两个功能界面，是我们快速上手剪映、高效创作视频的坚实基础。

![](img/2e3d261a7ce873a27e0588397d78008e_29.png)

剪映APP是一款简洁易用的剪辑工具，非常适合新手小白和短视频创作者使用。
