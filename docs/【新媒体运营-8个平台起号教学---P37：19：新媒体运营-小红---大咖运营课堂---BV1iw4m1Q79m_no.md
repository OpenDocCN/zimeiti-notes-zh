## 定位与对标：为选题奠定基础 🎯

在开始具体选题前，账号需要明确两个基础：**定位**与**对标**。这是确保后续内容方向不跑偏的前提。

### 账号定位：选择你的细分赛道

定位即确定账号要深耕的细分领域。例如，美妆、母婴、家居好物分享等。明确的定位能保证内容的垂直度，帮助平台为你的账号打上“领域专家”的标签。

选择细分赛道需遵循几个原则：

以下是定位时需要遵循的几个原则：

1.  **避免过度竞争**：初期不要选择过于热门、宽泛的头部领域（如整个“美妆”）。应寻找其中的细分赛道（如“男士美妆”、“美容仪器”、“口腔护理”），以缩短新手期，快速出成绩。
2.  **避免完全冷门**：确保所选赛道在小红书平台上有一定的用户群体和内容数据。如果完全无人涉及，可能意味着该领域不适合在此平台发展。
3.  **确保可持续创作**：选择你了解、能持续产出内容的领域。不要求你是专家，但需要对该领域有基本认知，并能规划长期内容。

### 寻找对标账号：站在巨人的肩膀上

确定定位后，需要寻找至少5个同领域的优质账号作为对标。对标账号的作用是为你提供**创作方向、风格和内容**的参考。

分析对标账号时，需关注以下几点：

以下是分析对标账号时需要关注的核心要素：

*   **初期数据**：重点研究对标账号运营初期的笔记数据和增长情况。近期（如近3-6个月）起号的账号，其初期策略更具参考价值。
*   **更新频率**：观察其内容更新节奏。
*   **使用标签**：记录其常用的热门标签，作为你后期发布的参考。

通过对标分析，你可以了解该领域用户和平台喜欢什么样的内容。

## 账号初期内容选题四大法 🔍

当定位与对标清晰后，便可进入具体的内容选题阶段。对于新账号，选题的核心逻辑是：**让平台快速注意到你，并喜欢你的内容**。以下是四个高效的选题方向。

### 1. 搜索发现：紧跟平台热点

“搜索发现”位于小红书首页的搜索框下方，类似于抖音的热点榜。它展示了平台当前的热门搜索词。

**操作公式**：`进入首页 -> 点击搜索图标 -> 查看“搜索发现”区域`

如果“搜索发现”中出现与你领域相关的热点，应立刻抓住。可以围绕该热点连续发布3天笔记。即使笔记未成爆款，蹭热点也能增加被平台收录的机会，让系统“注意到”你的账号，从而提升账号的潜在权重。

### 2. 搜索框：洞察用户真实需求

在搜索框输入你的领域关键词（如“穿搭”），在点击搜索前，下方会出现一系列“搜索提示词”。

![](img/d00cc7ade736b92f2a66cc2cb9eae586_5.png)

**操作公式**：`在搜索框输入核心关键词 -> 观察并记录下方出现的提示词`

![](img/d00cc7ade736b92f2a66cc2cb9eae586_7.png)

![](img/d00cc7ade736b92f2a66cc2cb9eae586_9.png)

这些提示词是平台根据用户搜索关联行为大数据生成的，代表了**高频、高权重的用户需求**。新账号应优先围绕这些提示词创作内容。

![](img/d00cc7ade736b92f2a66cc2cb9eae586_11.png)

**执行策略**：
1.  筛选出与业务强相关的提示词。
2.  每天选择2-3个词，作为当天笔记的创作主题。
3.  以3-4天为一个周期，循环覆盖这些高需求关键词。

![](img/d00cc7ade736b92f2a66cc2cb9eae586_13.png)

![](img/d00cc7ade736b92f2a66cc2cb9eae586_15.png)

这样做是为了让内容最大限度地匹配平台认为的“用户喜欢的内容”，从而获得更多推荐。

### 3. 创作灵感：获取流量扶持与加速定位

![](img/d00cc7ade736b92f2a66cc2cb9eae586_17.png)

![](img/d00cc7ade736b92f2a66cc2cb9eae586_19.png)

![](img/d00cc7ade736b92f2a66cc2cb9eae586_21.png)

“创作灵感”位于个人主页，是平台鼓励创作者参与的话题活动入口。

![](img/d00cc7ade736b92f2a66cc2cb9eae586_23.png)

**操作公式**：`进入个人主页 -> 找到并点击“创作灵感” -> 选择相关话题参与发布`

它的两大作用是：
*   **加速账号定位**：持续参与垂直领域的话题发布，能帮助平台更快地为你的账号打上精准标签。
*   **获得流量扶持**：新关注“创作灵感”或参与订阅话题，通常会获得平台赠送的初始流量券（如200曝光），能为笔记带来数据增益。

### 4. 推荐栏与全网热点：拓宽选题视野

![](img/d00cc7ade736b92f2a66cc2cb9eae586_25.png)

![](img/d00cc7ade736b92f2a66cc2cb9eae586_27.png)

*   **小红书推荐栏**：在搜索结果页，关键词下方有一排细分标签（如搜索“秋天穿搭”后，出现的“梨形身材”、“微胖大学生”等）。这些是平台推荐的关联细分方向。你的内容应尽量覆盖这些细分需求，在标题和文案中融入这些关键词。
*   **全网热点**：关注其他平台（如抖音、微博、B站）的热点，寻找可跨界结合到自身领域的内容。可以使用“今日热榜”等工具一站式查看各平台实时热榜。
