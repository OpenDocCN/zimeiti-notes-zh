# 视频号运营教程：P22：7.账号定位

在本节课中，我们将要学习视频号运营中关于账号定位的两个核心问题：如何处理旧作品以及如何获取原创视频素材。我们将分步拆解，确保初学者也能轻松掌握。

## 处理旧作品与发布频率

![](img/c0c7719c0bce235a20b81dc04b7ba3b0_1.png)

上一节我们介绍了账号定位的重要性，本节中我们来看看如何为账号的初期运营做准备。首先需要明确的是，在确定新的账号定位后，需要将之前不符合定位的作品隐藏。

![](img/c0c7719c0bce235a20b81dc04b7ba3b0_3.png)

发布新作品时，请遵循以下原则：
*   素材无需着急，保持一天发布一条的频率即可。

我们的首要目标是先积累100个粉丝，然后再系统性地解决素材问题。请按顺序逐个解决这些问题。

## 如何领取素材

如需领取这30个视频素材，需要满足以下条件并联系您的助理老师：
1.  **课堂互动**：在直播间公屏上发送指定互动内容（如“想”字）。
2.  **积极互动**：全程保持积极的课堂互动。
3.  **完成听课**：后台会统计听课时长与互动数据。

领取时，请务必向发放听课链接的助理老师说明是“张老师”同意领取的。这是获得素材的关键步骤。
