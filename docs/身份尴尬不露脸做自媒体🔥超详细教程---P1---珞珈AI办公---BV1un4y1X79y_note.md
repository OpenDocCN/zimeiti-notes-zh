# 自媒体入门：P1：不露脸不露声的自媒体创作指南🔥

在本节课中，我们将要学习如何在不露脸、不露声的情况下，轻松开启自媒体创作。我们将通过软件工具和拍摄技巧，实现身份隐私保护，同时制作出吸引人的内容。

## 概述

新手想做自媒体，但不想露脸，也不想露声音，这是完全可以实现的。本节将分享不露脸也能做自媒体的详细教程，帮助你轻松制作出专业效果的内容。

## 核心步骤详解

上一节我们概述了不露脸创作的可能性，本节中我们来看看具体的实现步骤。以下是利用软件工具进行形象、声音和背景替换的三个核心操作。

**第一步：更换头像**
1.  打开指定软件，点击“特效”。
2.  选择“人物头效”或“形象”功能。
3.  从提供的众多形象中选择一个，替换原始脸谱。

**第二步：更换声音**
1.  在编辑界面选中视频片段。
2.  点击“声音”或“音效”选项。
3.  从提供的多种音色中选择一个进行替换。

**第三步：更换背景**
1.  选中视频片段，点击“抠图”功能，选择“智能抠像”将人物分离出来。
2.  返回编辑界面，点击“背景”选项。
3.  选择软件内置的背景或从手机相册导入图片作为新背景。

完成以上三步修改后，视频中的人物形象将得到彻底改变。

## 补充拍摄技巧

除了依赖后期软件处理，我们还可以在前期拍摄时下功夫。以下是两种无需露脸的实用拍摄方法。

**方法一：调整拍摄角度**
*   拍摄时，可以避免露出头部，只拍摄身体中间的部分（如手部、工作台面等）。

![](img/ecfca0be64af13ee41a8e9699adfb63a_2.png)

**方法二：拍摄空场景**
*   直接拍摄场景画面，人物完全不出现在镜头中。
