# 微信视频号教程：P10：如何给短视频配音 🎙️

在本节课中，我们将学习如何为短视频进行配音。当您觉得原视频的配音效果不佳时，可以将其替换为自己的配音。我们将介绍配音工具的选择、具体操作步骤以及如何将新配音应用到视频中。

## 配音工具介绍

上一节我们介绍了视频剪辑的基础概念，本节中我们来看看配音所需的工具。配音工具种类繁多，您可以根据需求选择。

以下是几种常用的配音工具及其功能：

![](img/295ac26536fa880b9ce8a983bf4ce02a_1.png)

*   **配音神器**：这是一个功能全面的工具，不仅可以配音，还提供文案提取、视频去水印、修改视频MD5值等多种功能。
*   **微信小程序**：在微信中搜索“配音”、“去水印”或“修改MD5”等关键词，可以找到大量相关的小程序工具，例如“青抖”等。

## 工具获取与使用思路

无论您需要去水印、配音还是修改视频的MD5值，都可以通过以下方式找到工具：

![](img/295ac26536fa880b9ce8a983bf4ce02a_3.png)

1.  **打开微信**，点击顶部的搜索框。
2.  **输入关键词**，如“去水印小程序”或“修改MD5小程序”。
3.  在搜索结果中选择“小程序”分类，即可找到众多相关工具。

![](img/295ac26536fa880b9ce8a983bf4ce02a_5.png)

**提示**：不同工具的可用性可能变化，如果某个工具暂时无法使用，可以尝试搜索并使用其他同类工具。

## 以“配音神器”为例进行配音

![](img/295ac26536fa880b9ce8a983bf4ce02a_7.png)

“配音神器”是一个推荐的工具，其大部分功能（如文案提取、去水印）是免费的，仅配音功能需要消耗金币或开通会员。

以下是使用“配音神器”制作配音的详细步骤：

1.  **打开工具**：在微信中搜索并打开“配音神器”小程序。
2.  **准备文案**：点击配音功能，清除默认文字，将您准备好的视频文案脚本输入到文本框中。
3.  **选择音色**：在声音库中选择喜欢的配音音色（如男声、女声、童声等），可以点击试听效果。
4.  **生成配音**：选择好音色后，点击“合成配音”按钮。
5.  **保存配音**：合成完成后，选择“保存配音视频到相册”或“保存字幕视频到相册”。两者区别在于后者会生成带字幕的视频，前者则只有音频。
6.  **等待完成**：保存过程中请勿退出小程序，等待保存完成提示。

**获取免费金币**：您可以每天在小程序的“每日福利”中签到，以领取免费金币，用于生成配音，无需立即开通会员。

## 在剪映中替换视频原声

获得配音文件后，下一步就是将其替换到原视频中。我们以“剪映”App为例进行操作。

1.  **导入视频**：在剪映中导入需要替换声音的视频素材。
2.  **关闭原声**：选中视频轨道，点击“关闭原声”按钮。
3.  **添加新音频**：
    *   点击底部工具栏的“音频”。
    *   选择“提取音乐”，从手机相册中选择刚刚保存好的配音视频文件。
4.  **完成替换**：导入后，新的配音音频会出现在时间轴上，这样就完成了视频原声的替换。

**公式/代码示意**：
替换流程可以简化为：`原视频 - 原声 + 新配音音频 = 新视频`

![](img/295ac26536fa880b9ce8a983bf4ce02a_9.png)

**注意**：演示中使用的视频和文案可能不匹配，这仅为操作演示，您在实际操作时需确保内容一致。

## 关于视频去重与MD5值

![](img/295ac26536fa880b9ce8a983bf4ce02a_11.png)

修改视频的MD5值是一种基础的视频去重手段，可以提高内容的原创度。虽然仅修改MD5值不足以完全避免平台检测，但结合后续课程将讲到的**混剪**技巧，能更有效地提升视频通过审核的几率。

您可以在“配音神器”的“更多工具”中找到“视频去重”功能来修改MD5值。

本节课中我们一起学习了如何为短视频配音。核心内容包括：认识了“配音神器”等配音工具，掌握了从文案输入、音色选择到生成配音的全过程，并学会了在剪映中替换视频原声的方法。同时，我们也了解了修改MD5值作为辅助去重手段的思路。记住，制作吸引人的视频，清晰合适的配音是关键一步。
