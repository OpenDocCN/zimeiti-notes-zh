# 微信视频号教程：P14：视频号介绍与起号准备 🎬

在本节课中，我们将要学习微信视频号的基础介绍、变现模式以及开始操作前需要做的准备工作。内容简单直白，适合零基础的初学者。

## 概述

视频号是微信平台内置的功能。打开微信，在“发现”页和“我”的页面都能找到视频号入口。我从去年开始接触并运营视频号。在此之前，我运营过抖音和快手。

我们团队创造过的最高记录是一场直播达成31万营业额。不到十小时的直播，营业额达到十几万的场次非常多。营业额在10万以内、七八万或几万的直播也非常多。

## 视频号后台与入口

上一节我们介绍了视频号的概况，本节中我们来看看如何进入和操作视频号后台。

如果你是卖家，可以从微信“发现”页点击进入“视频号”。页面上方有一个小人头图标，点击即可进入个人中心。

进入后，你会看到“创作者服务中心”、“视频号消息”、“视频号私信”等选项。这里就是整个视频号的后台。后台功能包括“加热工具”、“账号诊断”、“直播收入”、“带货中心”等。

开始带货时，都需要从这里进入。在后台可以查看账号数据，例如每日出单情况。选品广场和佣金数据也在这里查看。

以我的账号为例，已打款金额为3万5千元，另有一笔1万多的款项，加起来这个账号总共收入超过4万元。

## 视频号如何赚钱

了解了后台操作，接下来我们看看视频号的核心变现模式。

视频号通过**视频自然流量**加**直播转化**来实现变现。拍摄视频只需要一部手机即可操作。直播则需要准备一些简单的工具。

以下是直播需要准备的基础工具列表：
*   **散热器**：这是每人必须准备的设备。
*   **桌面支架**：可以在拼多多等平台购买，价格从几元到十几元不等。
*   **补光灯**：如果家里灯光不够亮，可以购买一个几十元的直播补光灯。如果光线充足则不需要。
*   **背景布**：如果家里背景不够美观，可以购买背景布进行装饰。在拼多多、淘宝等平台都可以买到。

## 视频号的优势与规则

准备好工具后，我们还需要了解这个平台的特点。

视频号带货支持**零粉丝**起步，这与许多其他平台不同。每个人可以注册**五个**视频号账号，每个账号只需要缴纳**100元**的保证金。

视频号的规则相对简单，因此普通人和新手在上面很容易实现变现。

## 直播界面特点

现在，让我们具体了解一下视频号的直播界面。

从“发现”页点击“直播”进入。在直播广场，观众只能看到“XXX人看过”，但**看不到直播间实时在线人数**。

直播间实时在线人数**只有主播本人能看到**，观众是看不见的。因此，直播时不必过于紧张在线人数的多少。
