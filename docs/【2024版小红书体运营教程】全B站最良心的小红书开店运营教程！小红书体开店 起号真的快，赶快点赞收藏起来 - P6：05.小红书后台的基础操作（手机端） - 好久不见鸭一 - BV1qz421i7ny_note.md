# 小红书开店运营教程：P6：05. 小红书后台手机端基础操作 📱

![](img/f3882a841b3580e053c29053897331e6_0.png)

在本节课中，我们将学习如何通过手机端操作小红书店铺后台（千帆系统），完成店铺的基础设置，包括地址管理、物流模板配置、优惠券创建以及支付方式开通等核心功能。

![](img/f3882a841b3580e053c29053897331e6_2.png)

---

上一节我们介绍了开店前的准备工作，本节中我们来看看如何通过手机App来管理你的店铺后台。

首先，你需要下载并安装“小红书千帆”App。你可以在手机的应用商店中直接搜索“小红书千帆”进行下载安装。这是管理小红书店铺的官方后台工具。

打开App后，你会看到初始页面。对于一个新店，页面内容通常比较简单。

**消息中心**
点击底部的“消息”图标，可以进入客服与通知中心。这里是处理买家咨询、查看平台通知（如违规提醒）的地方。所有与店铺相关的沟通和系统消息都会在这里显示。

**首页与工具**
回到“首页”，你会看到“去初始化”的提示，引导你完成基础信息填写。点击“全部工具”，可以进入店铺管理的核心功能区。

以下是“全部工具”中包含的主要功能列表：
*   **发布作品/商品**：用于发布新的商品或营销笔记。
*   **商品管理**：管理已上架的全部商品。
*   **订单管理**：处理全部订单及售后订单。
*   **数据中心**：查看店铺的经营数据。
*   **店铺装修**：设置店铺首页的展示样式。
*   **群管理**：管理粉丝群聊。

---

上一部分我们浏览了后台的主要模块，接下来我们详细讲解几个必须完成的基础设置。

**1. 填写发货地址**
这是开店后的首要操作。你可以在“首页”的“成长”任务中找到“地址”填写项，也可以在“全部工具”的相关设置中找到。

地址填写步骤如下：
1.  填写联系人姓名（可填自己）。
2.  选择所在地区（中国大陆）。
3.  填写详细地址。对于做一件代发的商家，此地址可灵活填写。
4.  填写你的手机号码。
5.  **关键点**：在“买家留言”或类似字段中，**务必填写有效的客服联系方式**（如“退货前请先联系客服”）。如果此处填写“联系客服”，平台可能不会通过审核，影响后续退货流程。

**2. 设置物流模板**
在地址管理页面，通常可以找到“物流”或“运费模板”的设置入口。

你可以创建不同的运费模板，例如：
*   `偏远地区不配送`：可设置为新疆、西藏等地不发货。
*   `买家承担邮费`：为指定地区设置额外的运费。
*   `全国包邮`：系统通常有默认模板（如“全国包邮”），但需注意是否包含偏远地区。

对于新手，可以直接使用或修改系统默认的模板，但务必根据你的发货政策检查偏远地区的设置。

**3. 创建优惠券**
优惠券是重要的营销工具。你可以在“全部工具”中找到“营销”或“优惠券”入口。

进入后，你可以创建多种类型的优惠券：
*   `店铺通用券`：全店商品可用。
*   `指定商品券`：仅限特定商品使用。
*   `店铺新人券`：针对新客的专属优惠。

创建后，你可以在后台管理“生效中”、“已失效”等不同状态的优惠券。

**4. 完成认证与缴纳保证金**
在“成长”任务或店铺设置中，需要完成店铺类型认证（如个人店）并缴纳保证金。

**重要提示**：
*   若计划长期经营，建议缴纳保证金。
*   必须按照流程开通**微信支付**和**支付宝支付**功能。这通常需要绑定你的身份证和银行卡信息。
*   保证金的金额根据你经营的商品**类目**不同而有所差异，请按系统提示金额缴纳。
