# 抖音运营实战：P13：10、如何寻找文案-1 📝

![](img/a3fb589c364ba4b5d6d1465dfa904bf0_0.png)

在本节课中，我们将学习如何为抖音视频寻找和准备文案。对于新手而言，文案是视频内容的核心，好的文案能有效提升视频吸引力，帮助账号快速涨粉。本节将重点介绍几个实用的文案素材网站，并演示如何利用它们生成配音，一站式解决文案与声音的难题。

---

上一节我们讨论了优化作品的重要性，本节中我们来看看如何具体解决文案创作这个首要问题。很多同学在起步阶段不会写文案，这没有关系，我们可以借助一些现成的工具和资源。

![](img/a3fb589c364ba4b5d6d1465dfa904bf0_2.png)

以下是五个实用的文案素材网站，它们都是网页版，无需下载APP，在手机或电脑浏览器中搜索即可使用。

![](img/a3fb589c364ba4b5d6d1465dfa904bf0_4.png)

1.  淘金阁
2.  （其他四个网站名称，因原文未具体说明，此处保留原文结构）

![](img/a3fb589c364ba4b5d6d1465dfa904bf0_6.png)

我们以“淘金阁”为例进行详细讲解。如果你在搜索引擎中直接搜索“淘金阁”广告太多，可以直接使用其网址访问。

![](img/a3fb589c364ba4b5d6d1465dfa904bf0_8.png)

进入淘金阁网站后，你会看到主页有非常丰富的分类，例如：财经、军事、动漫、科技、美食、美文、星座、体育、旅游、育儿、汽车、社会、娱乐、时尚、房产、国际、宠物等。

![](img/a3fb589c364ba4b5d6d1465dfa904bf0_10.png)

假设我们需要情感类文案，就点击“情感”分类。页面会展示大量相关话题，并且通常有多页内容可供浏览。之前让大家保存的“一个群体加一个核心话题加疑问”的公式，此时就可以用来筛选合适的文案。

![](img/a3fb589c364ba4b5d6d1465dfa904bf0_12.png)

**公式：目标群体 + 核心话题 + 疑问句式**

![](img/a3fb589c364ba4b5d6d1465dfa904bf0_14.png)

如果你在现有分类中找不到想要的内容，可以使用关键词搜索功能。例如，在搜索框输入“宝妈”，然后点击“开始抓取”，系统就会为你抓取全网包含“宝妈”关键词的文案素材。

解决了文案来源问题，接下来处理配音。很多同学担心自己的声音不够好听，淘金阁也提供了免费的配音生成工具。

![](img/a3fb589c364ba4b5d6d1465dfa904bf0_16.png)

在网站右上角找到“小工具”入口，点击进入后选择“配音在线生成”功能。这个工具提供了极其丰富的音色选择，包括：
*   亲和女生、温柔女生、甜美女生
*   自然女生、卖场广播
*   直播配音（亲切女生、活力女生）
*   通用场景（标准男生、标准女生）
*   特色音色（湖南口音、童声）
*   英文配音
*   方言配音（四川话、粤语、东北话、台湾话）
*   外语配音
*   精品男配（情感解说、影视解说、娱乐解说）

使用方法很简单：
1.  在左侧“配音文本”框中输入或粘贴你的文案。
2.  在右侧选择你喜欢的声音类型，例如“方言”下的“东北话”。
3.  调整音量、语速和语调参数。
4.  点击“立即生成”。

![](img/a3fb589c364ba4b5d6d1465dfa904bf0_18.png)

生成后，你可以在“配音生成列表”中下载音频文件到手机，以便在后续剪辑环节使用。手机用户可以通过微信扫码登录来免费使用这些功能。

![](img/a3fb589c364ba4b5d6d1465dfa904bf0_20.png)
