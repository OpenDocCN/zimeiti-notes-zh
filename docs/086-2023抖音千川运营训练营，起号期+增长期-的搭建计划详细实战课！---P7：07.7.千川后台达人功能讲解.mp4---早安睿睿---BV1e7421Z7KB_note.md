# 千川运营训练营：07.7：千川后台达人相似功能详解 🎯

![](img/c5358e7b61004cb76a0a73d296d4feb6_0.png)

![](img/c5358e7b61004cb76a0a73d296d4feb6_2.png)

在本节课中，我们将要学习千川后台的“达人相似”功能。这是一个通过定向与特定达人有过互动行为的用户，来精准触达潜在客户的重要工具。我们将详细讲解其设置方法、核心逻辑以及如何有效圈定目标人群。

上一节我们介绍了千川计划的基础搭建，本节中我们来看看如何利用“达人相似”功能进行更精细的人群定向。

## 功能入口与基础设置

首先，在千川后台创建专业计划时，可以在“更多人群”下的“自定义”选项中，找到“达人相似”功能。

以下是设置“达人相似”功能时需要关注的几个核心点：

![](img/c5358e7b61004cb76a0a73d296d4feb6_4.png)

*   **互动行为选择**：系统要求选择定向用户与达人发生何种互动。商品互动是精准行为，建议勾选。直播互动方面，建议将“打赏”、“评论”、“有效观看”全部勾选。“观看”行为过于宽泛，不建议选择。视频互动中的“点赞”、“分享”、“评论”可根据情况选择。
*   **关注行为处理**：“关注”达人的用户是其粉丝，与你的目标客户可能不完全重合，因此不建议勾选。
*   **互动时间范围**：建议选择“近30天”内的互动用户。“近15天”的时间范围过窄，可能导致覆盖人群不足。

![](img/c5358e7b61004cb76a0a73d296d4feb6_6.png)

## 人群覆盖量的核心要求

![](img/c5358e7b61004cb76a0a73d296d4feb6_8.png)

使用“达人相似”功能有一个关键前提：**圈定的人群总量需达到1000万以上**。如果覆盖人数不足1000万，计划很可能无法正常消耗和跑量。

![](img/c5358e7b61004cb76a0a73d296d4feb6_10.png)

因此，在添加达人或达人组合后，务必观察系统估算的覆盖人群数量，确保其超过1000万的门槛。

![](img/c5358e7b61004cb76a0a73d296d4feb6_12.png)

## 如何有效圈定达人

如果只添加自己直接的竞争对手，覆盖人群可能达不到1000万的要求。这时，我们需要拓展思路，去圈定“上下游达人”。

上下游达人是指，其粉丝群体与你的目标客户高度重合，但并非你的直接竞争对手。例如，如果你是销售女装（目标客群为30-40岁女性），那么美妆、护肤等同样以此年龄段女性为主要客群的达人，就是你的上下游达人。他们的粉丝也是你的潜在客户。

![](img/c5358e7b61004cb76a0a73d296d4feb6_14.png)

![](img/c5358e7b61004cb76a0a73d296d4feb6_16.png)

以下是寻找和添加上下游达人的具体方法：

1.  **利用数据平台**：可以借助蝉妈妈、飞瓜数据等第三方平台，搜索与你产品相关的品类（如“口红”），查看正在为该品类商品带货的达人列表。
2.  **分析带货达人**：在数据平台中，进入具体商品页的“达人分析”，可以看到为其带货的达人及其粉丝量级。
3.  **筛选与添加**：从中筛选出粉丝量较大、且你认为其粉丝画像与你目标客户匹配的达人。将这些达人的账号名称，准确填写到千川的“达人相似”搜索框中添加即可。

通过添加一系列直接竞争对手和上下游达人，可以有效将覆盖人群扩大至1000万以上，满足计划启动的条件。

## 功能总结与组合使用

“达人相似”功能的设置逻辑就是以上内容。在实际使用时，可以与其他定向条件（如年龄、性别等）组合使用，以实现更精准的投放。
