## 案例视频分析：强人设还是弱人设？

![](img/8cfd8511c79f5db87ccbce187d05a1ae_9.png)

![](img/8cfd8511c79f5db87ccbce187d05a1ae_11.png)

请看以下视频脚本与画面描述，并判断这是“强人设”还是“弱人设”视频。

![](img/8cfd8511c79f5db87ccbce187d05a1ae_13.png)

**视频脚本内容：**
> 穷人改变命运最大的难点就是他的父母，还有他的原生阶层。如果他生活在父母身边，父母对他施加很大的影响力，基本上他这辈子就没机会改变命运了。因为你想一个问题，他爹妈如果观点是对的，那他爹妈怎么会把他生成一个穷人？他爹妈在那么多年的社会实践中，最终变成了一个穷人，一定是因为他的观点是有很多问题的。所以他深受他父母影响，他就绝对完蛋。
>
> 所以你会发现一个现象，那就是穷人翻身必定不是在老家，他一定是去别的城市，远离父母，自己一个人独自打拼。父母管不到他了，那这个时候他就有机会改变自己思维了。
>
> 第二点就是他的朋友。还是那个话，如果他是个穷人，他朋友也是穷人。朋友们如果想问题的方法是对的，解决问题的方法如果是正确的，他朋友怎么会穷呢？能跟他混到一个圈子里面去的人，天然的处理问题的方法也是错的。
>
> 所以当你掉到坑里面去的时候，一不要指望父母，二不要指望朋友。他们要是能帮助你出坑，他们自己就不在坑里待着了，也就不会认识你了。
>
> 第三点就是在于穷人有的真的就是缺本金，真的缺机会。有一个机会来的时候呢，身上缺的那个3万5万关键的钱，他就没办法发力。他使不上力，他也没胆子去借，他也不确定这个事情能不能抓得住，万一抓不住，他怕背债，怕去借杠杆。其实很多有钱人都是敢赌的，是敢借债的，是敢去把自己所有的东西`all in`去付出的。富贵险中求，一点险都不冒，是不可能改变阶级的。
>
> 最后我觉得就是他们可能内心还是虚弱的。因为长久以来在这个社会上，处于一个比较边缘的状态，没有太多人相信他，没有人太多人关注他，所以他习惯性的会不自信，习惯性的逃避责任，逃避压力。会导致身边的人也不敢相信他，依靠他。他这样的话就团结不了队伍，团结不了队伍的人，他怎么可能做大呢？
>
> 我觉得主要是这四点。所以想要改变命运呢，第一离开自己的家乡，远离自己的父母；第二认识一帮新朋友，认识一些跟自己原有阶层不一样的朋友；第三点要有胆子，要有一定的风险意识，在能力范围内敢于去`all in`，敢于去承担风险；第四那就是自强自信。这个东西你不自信，没有人能帮你的。

## 案例分析结论与技巧解析

![](img/8cfd8511c79f5db87ccbce187d05a1ae_23.png)

![](img/8cfd8511c79f5db87ccbce187d05a1ae_25.png)

经过分析，该视频是一个典型的 **弱人设** 视频。

![](img/8cfd8511c79f5db87ccbce187d05a1ae_27.png)

### 为什么是“弱人设”？

![](img/8cfd8511c79f5db87ccbce187d05a1ae_29.png)

![](img/8cfd8511c79f5db87ccbce187d05a1ae_31.png)

判断依据如下：
1.  **内容与发布者无关**：视频中讨论的是“穷人”和“富人”的普遍现象，但并未涉及视频发布者“猪场老板”自身的经历、观点或故事。
2.  **声画分离**：视频的**画面**（养猪场场景）与**声音**（讲述阶层跨越的观点）是完全不同的两件事，没有直接联系。

![](img/8cfd8511c79f5db87ccbce187d05a1ae_33.png)

### 为什么“弱人设”视频能火？

![](img/8cfd8511c79f5db87ccbce187d05a1ae_35.png)

![](img/8cfd8511c79f5db87ccbce187d05a1ae_37.png)

这个视频之所以能获得大量播放，关键在于它运用了 **“双向吸引”** 策略。

![](img/8cfd8511c79f5db87ccbce187d05a1ae_39.png)

以下是该策略吸引流量的两个核心维度：

![](img/8cfd8511c79f5db87ccbce187d05a1ae_41.png)

![](img/8cfd8511c79f5db87ccbce187d05a1ae_43.png)

*   **吸引维度一：画面吸引力**。一部分用户被“活抓猪场老板”这种独特、生活化或能满足“治愈强迫症”等心理需求的画面所吸引，从而停留观看。
*   **吸引维度二：内容共鸣力**。另一部分用户对视频中关于“原生家庭”、“阶层突破”、“创业心态”的观点产生强烈共鸣，认为“说得太好了”，从而完成播放、点赞甚至转发。

![](img/8cfd8511c79f5db87ccbce187d05a1ae_45.png)

**公式表示即：视频热度 ≈ 画面吸引力 + 内容共鸣力**

![](img/8cfd8511c79f5db87ccbce187d05a1ae_47.png)

### 如何将“双向吸引”技巧用于你的视频？

![](img/8cfd8511c79f5db87ccbce187d05a1ae_49.png)

![](img/8cfd8511c79f5db87ccbce187d05a1ae_51.png)

无论你是什么行业的创作者，都可以借鉴这个方法，为你的常规内容注入新的吸引力。

![](img/8cfd8511c79f5db87ccbce187d05a1ae_53.png)

以下是针对不同创作者的应用举例：

*   **宝妈博主**：如果你拍摄宝宝日常或美食制作没人看，可以尝试为画面**配上**一个搞笑的家庭趣事或情感故事音频。
*   **实体店/工厂主**：如果你拍摄产品展示或工厂流程没人看，可以尝试为画面**配上**你的从业经历、创业故事或行业洞察的音频。

![](img/8cfd8511c79f5db87ccbce187d05a1ae_55.png)

**代码式描述创作思路：**
```plaintext
if 你的常规内容（画面）吸引力不足：
    为画面匹配一个具有强共鸣或强吸引力的音频（观点、故事、知识）。
else:
    持续优化你的核心人设内容。
```

![](img/8cfd8511c79f5db87ccbce187d05a1ae_57.png)

这种方法的核心是，让你的视频既能通过**画面**吸引一类用户，又能通过**音频内容**吸引另一类用户，从而扩大视频的受众面，提高完播率和互动数据。

本节课中我们一起学习了如何通过案例分析区分“强/弱人设”，并掌握了“双向吸引”这一实用的视频创作技巧。许多视频缺乏流量的原因，往往只是缺少这样的小技巧。抖音运营是一个需要不断学习和测试的行业，掌握底层逻辑并灵活运用，是起号的关键。
