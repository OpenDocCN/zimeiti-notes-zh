# 抖音运营：042：爆款短视频选题的万能公式 📝

在本节课中，我们将学习一个用于生成爆款短视频选题的万能公式。这个公式通过拆解用户、场景、关系、动机和产品等核心要素，帮助你系统化地批量生产内容创意，解决“不知道拍什么”的难题。

![](img/48f0e55b49c84e4d81148af84820fad1_1.png)

上一节我们介绍了内容策划的基础，本节中我们来看看如何具体应用一个结构化公式来挖掘选题。

这个万能公式是我通过总结大量爆款文案和抖音运营技巧得出的。其核心是将内容选题拆解为五个关键部分进行排列组合。具体内容分为以下五大块：
1.  **用户标签**
2.  **用户场景**
3.  **用户关系**
4.  **购买理由**
5.  **产品差异化**

---

### 公式详解与应用

当你为自己的赛道、产品或内容方向批量生产内容感到无从下手时，可以遵循以下步骤：

首先，明确你的定位，并罗列出核心的**用户标签**。一个人的标签可能多达数百个，你只需聚焦最核心的即可。

以下是常见的用户标签维度：
*   年龄
*   性别
*   性格
*   教育程度
*   居住区域
*   婚姻状态

其次，确定**用户场景**，即内容发生的主要环境。

常见的用户场景包括：
*   家里
*   超市
*   学校
*   餐饮场所
*   公司
*   车上

接着，定义内容中涉及的**用户关系**，这决定了内容的互动和情感基调。

常见的用户关系有：
*   夫妻关系
*   婆媳关系
*   亲子关系
*   上下级关系
*   生意伙伴关系

然后，思考驱动用户的**购买理由**（或行为动机），即他们为何会被你的内容吸引或产生购买行为。

核心的购买理由包括：
*   好吃
*   好奇
*   新鲜
*   健康
*   方便
*   便宜

最后，提炼你的**产品差异化**优势，即你的产品或服务与竞品的不同之处。

产品差异化可以从多个维度挖掘：
*   **质量**：更耐用、更安全。
*   **来源**：特定原产地（如：云南西双版纳古树茶）。
*   **品牌**：品牌历史、口碑。
*   **文化/概念**：独特的设计理念或文化内涵。
*   **服务**：更优质的售后或体验。

将以上五个要素进行排列组合，就能源源不断地产生新的选题，彻底解决“内容荒”的问题。

### 万能公式应用案例

为了帮助你更好地理解，以下是几个应用该公式的具体案例：

**案例一：职场新人穿搭**
*   **用户标签**：22岁，刚毕业的年轻人
*   **用户场景**：谈客户时
*   **用户关系**：商务关系
*   **产品差异化**：纯羊毛面料、欧式风格的西服
*   **购买理由**：显得成熟、有能力
*   **生成选题**：《22岁新人谈客户，穿什么西服能显得成熟有能力？试试纯羊毛欧式款！》

**案例二：婆媳关系维护**
*   **用户标签**：30岁左右的宝妈
*   **用户场景**：家里
*   **用户关系**：婆媳关系
*   **产品差异化**：100%纯棉、高端中国风旗袍
*   **购买理由**：讨好婆婆开心
*   **生成选题**：《30岁宝妈必看！送婆婆这件纯棉中国风旗袍，婆媳关系瞬间升温！》

**案例三：职场人情往来**
*   **用户标签**：打工人
*   **用户场景**：公司
*   **用户关系**：上下级关系
*   **产品差异化**：百年古树普洱茶
*   **购买理由**：上档次、不失礼、有效“拍马屁”
*   **生成选题**：《打工人如何给老板送礼？选百年古树茶，有档次又不尴尬！》

**案例四：亲子早餐难题**
*   **用户标签**：三岁孩子他妈
*   **用户场景**：家里
*   **用户关系**：亲子关系
*   **产品差异化**：省时省事的空气炸锅
*   **购买理由**：让孩子爱吃、不挑食、营养方便
*   **生成选题**：《三岁宝妈早餐秘籍！用空气炸锅做，孩子抢着吃，营养又省事！》

**案例五：厨房技巧分享**
*   **用户标签**：喜欢做菜的人
*   **用户场景**：家里厨房
*   **用户关系**：自我提升
*   **产品差异化**：轻而薄的锅
*   **购买理由**：解决炒菜粘锅的痛点
*   **生成选题**：《爱做菜的人必学！轻薄锅的开锅秘诀，从此炒菜不粘！》
