# 抖音运营：P32：基础篇丨爆款文案套路 📝

![](img/f2e8f1c5d9c7db50488ee9f9ef34ba6e_0.png)

在本节课中，我们将要学习一个适用于各行各业的爆款文案创作套路。掌握这个核心公式，可以帮助你快速创作出能引发共鸣、提升视频数据的高质量文案。

在内容为王的时代，运营者必须具备脚本创作能力。要拥有这种能力，首先需要弄清楚什么是好文案。好文案的核心只有一点：**要让对方认为与自己有关**，也就是引起共鸣。如果用户不关心你所说的内容，视频就不可能有人看。

明白了核心目标后，具体应该怎么做呢？接下来，我们来看一个万能公式。

![](img/f2e8f1c5d9c7db50488ee9f9ef34ba6e_2.png)

## 爆款文案万能公式

以下是构建爆款文案的四个步骤，其核心逻辑是：**现象 + 危害 + 原因 + 解决办法**。

*   **现象**：描述一个普遍存在的问题或场景，用于筛选目标用户，让他们感觉“与我有关”，从而愿意停留观看。
*   **危害**：阐述该问题会带来的负面影响或不良后果，负责放大用户的焦虑或痛点情绪，防止他们在5秒内划走视频。
*   **原因**：分析问题产生的根本原因，引导用户继续观看，寻找答案，从而提高视频的完播率。
*   **解决办法**：给出具体、可行的解决方案，并鼓励用户点赞、收藏，以此提升视频的互动率。

上一节我们介绍了公式的结构，本节中我们来看看如何将这个公式应用到具体案例中。

## 公式应用案例

以下是两个不同行业的应用示例，帮助你更好地理解这个套路。

**案例一：服装行业**
*   **现象**：梨形身材的女孩太难买裤子了。
*   **危害**：挑错了样式显得腿又粗又短。
*   **原因**：梨形身材是天生的，很难改变。
*   **解决办法**：而这三种裤子不仅显瘦，还显腿长，穿上第二天就被小哥哥搭讪了，姐妹们赶快收藏起来吧！

这段文案完美契合了公式。开头一句话就锁定了“梨形身材”的目标客户，然后通过阐述危害和原因，吸引她们看完视频寻找解决方案。这样吸引来的粉丝非常精准，后续变现也更容易。

![](img/f2e8f1c5d9c7db50488ee9f9ef34ba6e_4.png)

**案例二：运营行业**
*   **现象**：每天用心制作短视频，但发完只有100播放。
*   **危害**：继续下去只是浪费时间，徒增焦虑。
*   **原因**：这跟你没有系统的运营底层逻辑有关。
*   **解决办法**：现在去看完我的前三条视频，不花一分钱也能学会运营知识，还不快快行动起来！

用好这个套路，不管什么行业都可以往里套。这类文案通常不足30秒，拍摄起来也很轻松，尤其适合进行账号矩阵化运营，效率会非常高。
