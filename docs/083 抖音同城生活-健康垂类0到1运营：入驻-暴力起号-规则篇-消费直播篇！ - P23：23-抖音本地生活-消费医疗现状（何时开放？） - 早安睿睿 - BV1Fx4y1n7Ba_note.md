在本节课中，我们将要学习抖音本地生活板块中，与消费医疗相关的现状与规则。我们会了解当前哪些类目开放、哪些在内测，以及“报白”等核心概念的含义，帮助你清晰认识这个赛道的准入门槛和合规要求。

上一节我们介绍了抖音本地生活医疗垂类的整体框架，本节中我们来看看消费医疗类目的具体现状。

目前，抖音本地生活消费医疗的现状如下：自2022年12月起，平台已对口腔、医美、月子中心等消费医疗类目启动内测。内测名单采用定向邀请制，仅限行业头部机构参与。例如，全国口腔类目仅上线了三家机构，医美类目则是在三个省份各选一家头部机构进行测试。

目前，眼镜、体检、推拿、美容等类目已正常开放。这些类目只需完成“报白”流程即可入驻。这里需要解释一下“报白”这个概念。

**报白**，即报备白名单。简单来说，某些行业尚未完全向所有商家开放，但如果抖音本地生活的运营人员认为服务商提交的申请名单资质可靠、符合要求，便会将其纳入白名单，允许其入驻。目前，医美、口腔、眼科等类目仍处于内测阶段，因此“定向邀约”和“报白”存在一定区别。

市场上存在一些声称能“强行上架”或“报白”的服务，它们通常是利用了平台的某些漏洞或未被公开的规则进行操作。这类方式并非官方认可的“报白”，存在较大风险。

预计到2023年3月，平台将对消费医疗类目进行全面开放的评估。根据现有信息，口腔类目有望在3月15日后正式开放。当然，即使是正式开放，平台也会设置相应的资质和运营门槛。
