# 抖音运营教程：P24：16、口播类视频拍摄方法-7 🎬

在本节课中，我们将要学习口播类视频的具体拍摄方法。我们将通过一个成功案例，详细拆解拍摄时需要注意的构图、内容定位等核心要素，帮助你掌握制作高质量口播视频的技巧。

上一节我们介绍了视频拍摄的通用注意事项，本节中我们来看看**口播类视频**的具体拍摄方法。口播类视频涵盖金融、装修、养生、情感等多个领域，但其拍摄的核心逻辑是相通的。

首先，我们分析一个学员的成功案例。账号名为“种小草”，是一位40岁带三个孩子的宝妈，专注于情感口播。她的一条视频播放量超过8000万，并借此涨粉30万。她的成功关键在于**内容精准吸引了目标群体**。例如，如果你的内容针对20多岁的学生党，那么所有视频都应围绕学生感兴趣的话题展开，这样涨粉才会又快又精准。

以下是拍摄口播视频时需要关注的几个核心要点：

**1. 构图与比例**
抖音是**9:16**的竖屏平台。关键在于视频内容本身采用横屏还是竖屏。案例中的视频采用了**4:3**的横屏构图。需要强调的是，风格必须统一：如果选择横屏，则所有视频都应保持横屏；如果选择竖屏，则全部使用竖屏。人物主体应占据画面中央约**70%** 的面积，避免脸部怼满屏幕。画面左、右、上方需要适当留白，使整体观感清晰舒适。

**2. 人物与表现**
如果你认为自己外貌不够出众，是否还能做口播？答案是肯定的。抖音的成功并非完全依赖颜值。口播的核心在于**传递内容和知识**。只要出镜形象清晰、体面即可。可以利用美颜滤镜进行适当修饰，但注意不要过度夸张。

![](img/16f51923987119b8d10d7ac8cb320c6c_3.png)

**3. 内容与定位**
内容是吸引目标群体的核心。我们通过案例视频的具体内容来分析其定位。

> 视频文案示例：
> “跟婆婆在一起住是什么样的感受？我跟我婆婆在一起住了16年了...我最大的感受就是这十多年以来，我娘家人很少在我家住上几天...这对于我内心来说其实是一个很大的缺失。”

这段内容讨论的是婆媳关系。那么，它最能引起哪个群体的共鸣？显然，对婆媳关系感触最深的不是男性，也不是未婚的年轻女性，而是**已婚的女性群体**。这个视频正是精准地吸引了这部分受众，从而引发了广泛的共鸣和传播。

**4. 拍摄场景与准备**
口播视频不限于户外拍摄，在家中也完全可以进行。关键是**背景要整洁美观**，避免杂乱。如果担心记不住台词，可以在剪辑环节利用提词器辅助，实现边看边录，降低拍摄难度。

**5. 收音清晰度**
清晰的声音是口播视频的基础。为了保证音质，建议购买一个**手机收声麦克风**。这类设备价格便宜（约十几元），在淘宝、京东等平台均可购得，能有效提升视频的听觉体验。

本节课中我们一起学习了口播类视频的拍摄方法。我们了解到，成功的口播视频需要**统一的构图比例**、**得体的人物表现**、**精准的内容定位**以吸引目标群体、**整洁的拍摄环境**以及**清晰的收音效果**。掌握这些要点，你也能制作出吸引人的优质口播内容。
