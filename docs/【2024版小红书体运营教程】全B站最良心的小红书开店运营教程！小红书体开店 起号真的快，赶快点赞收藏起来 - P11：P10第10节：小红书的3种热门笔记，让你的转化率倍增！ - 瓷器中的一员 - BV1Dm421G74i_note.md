# 小红书运营教程：P11：三种热门笔记类型详解

在本节课中，我们将详细讲解小红书的三种热门笔记类型。掌握这些笔记的写作方法，可以有效提升内容的转化率。建议在学习过程中做好笔记，以便更好地理解和应用。

## 教育类笔记

上一节我们介绍了笔记类型的重要性，本节中我们来看看第一种类型：教育类笔记。这类笔记的核心是传授知识或技能，适合美妆、瘦身、美食、P图软件、美容摄影等行业。在撰写时，可以将产品自然地植入教学内容中。

教育类笔记的封面首图通常有三种构成方式：

*   **对比图**：展示使用产品前后的效果对比。例如，口红涂在手上与涂在嘴上的颜色对比。
*   **效果图**：直接展示使用产品后的最终效果。例如，眼影上妆后的整体妆容效果。
*   **步骤图**：分解展示操作过程。例如，“五步告别圆肩驼背”教程中，展示每一步的动作或所需辅助工具。

以下是教育类笔记正文内容的通用写作模板，通常包含五个部分：

1.  **使用工具或材料**：列出所需的产品及辅助工具。
2.  **操作步骤**：详细说明具体的操作过程。
3.  **注意事项**：提示操作中需要避免的问题或关键点。
4.  **产品安利**：在讲解过程中，自然地推荐你觉得好用的产品。**注意：介绍产品的文字篇幅建议不超过全文的30%**，以避免被判定为硬性广告。
5.  **常见问题解答**：针对操作中可能出现的失败情况（例如“为什么贴不上”）进行解答，这比展示成功案例更能吸引读者。

## 合集类笔记

了解了教育类笔记后，我们来看第二种适用范围极广的类型：合集类笔记。这类笔记通过集合推荐多种产品，几乎适合所有行业。其核心是针对特定场景、人群或需求，提供一系列解决方案。

为了避免写作角度单一，以下是撰写合集类笔记时可以切入的七个不同角度：

1.  **针对人群**：例如，《学生党开学彩妆攻略》、《上班族代步小电动车推荐》。
2.  **针对成本**：例如，《100元内平价护肤品清单》、《五块钱一个的网红早餐包》。
3.  **针对效果**：例如，《让男朋友合不拢嘴的夏季穿搭》、《瘦身必备零食推荐》。
4.  **针对地区**：例如，《香港平价美食推荐》、《南京本地网红店打卡》。
5.  **针对场景**：例如，《旅游必备清单》、《居家清洁好物指南》。
6.  **针对产品属性**：例如，《微电流美容仪测评》、《某某色系口红合集》。
7.  **针对品牌**：例如，《体察内部员工隐藏菜单》、《星巴克夏季新品测评》。

合集类笔记的首图主要有两种形式：

*   **产品拼图**：使用美图秀秀等软件将多个产品图片拼接在一起。
*   **多产品摆拍**：将相关产品有序地摆放在一起进行拍摄，例如摊开旅行箱内所有物品。

撰写合集类笔记正文时，需注意均匀分布对各产品的介绍篇幅，并客观说明其优点与缺点，这样可以有效降低“硬广”嫌疑。同时，在正文中适当提及产品名称，有助于平台搜索引擎抓取关键词。

## 视频类笔记

最后，我们来看目前平台流量扶持力度最大的第三种类型：视频类笔记。如果条件允许，非常推荐大家尝试制作视频内容。

制作视频笔记需要准备以下工具：

*   **拍摄工具**：手机、专业相机、三脚架、补光灯。预算充足可考虑简易摄影棚。
*   **剪辑工具**：手机APP（如剪映），或电脑专业软件（如PR）。

关于视频笔记，有几个关键参数需要注意：

*   **视频时长**：小红书默认视频时长为1分钟。持续发布优质视频，达到一定标准后，可申请开通5分钟甚至更长视频权限。
*   **视频尺寸**：推荐两种比例：
    *   **3:4（竖屏）**：视觉冲击力强，适合大部分内容。
    *   **4:3（横屏）**：画面更精致，适合展示化妆品、护肤品等细节。

在内容策划上，可以结合所在行业，制作教程、测评、好物分享或生活VLOG等类型的视频。**一个有效的学习方法是：多研究、总结行业内成功视频博主的经验**，分析他们的视频结构（如开头、中间、结尾如何安排）、构图、选品和脚本，并进行模仿和创新。

---

本节课中我们一起学习了小红书的三种核心笔记类型：**教育类笔记**、**合集类笔记**和**视频类笔记**。每种类型都有其适用的场景和写作要点，掌握并灵活运用它们，是提升笔记吸引力和转化率的关键。请结合自身行业特点，选择最适合的类型进行创作实践。
