## 注册前的准备与注意事项

在开始注册之前，需要了解几个核心概念和准备工作。首先，亚马逊的注册流程在不同站点（如美国站、欧洲站）基本一致。其次，注册一个主要站点（如美国站）后，其他站点账户会同步生成，但处于未激活状态，后续可在后台手动激活。

![](img/b337f7fc3393a3abbcaa4e4897bf705b_7.png)

以下是注册前必须准备好的材料清单：
*   **公司营业执照**：需为最新且有效的营业执照。
*   **法人身份证**：需在有效期内。
*   **双币信用卡**：支持美元消费，如Visa或MasterCard。
*   **收款账户**：用于接收亚马逊销售款项，例如连连支付、PingPong等第三方收款工具。
*   **电子邮箱和手机号**：用于接收验证码和官方通知，建议使用未注册过亚马逊的邮箱。

![](img/b337f7fc3393a3abbcaa4e4897bf705b_9.png)

![](img/b337f7fc3393a3abbcaa4e4897bf705b_11.png)

**重要提示**：在模拟注册或观看教程时，**切勿使用真实的公司和个人信息进行填写**，以免与未来或现有的亚马逊账户产生关联，导致封号风险。

## 模拟注册流程逐步解析

上一节我们介绍了注册前的准备工作，本节中我们将一步步解析模拟注册的完整流程。

### 第一步：创建登录账户

首先，访问亚马逊卖家注册页面。以下是创建初始账户的步骤：
1.  **填写姓名**：输入法人姓名，例如 `田丽琴`。
2.  **设置邮箱和密码**：使用准备好的邮箱地址，并设置一个强密码。
3.  **验证邮箱**：登录邮箱查收亚马逊发送的验证码（例如 `231416`），并在网页上输入以完成验证。
4.  **验证手机号**：添加中国大陆手机号，接收并输入短信验证码，完成手机绑定。

![](img/b337f7fc3393a3abbcaa4e4897bf705b_15.png)

完成以上步骤后，即成功创建亚马逊卖家账户，进入正式的店铺信息填写环节。

![](img/b337f7fc3393a3abbcaa4e4897bf705b_17.png)

### 第二步：填写公司信息

接下来需要填写详细的营业执照信息。请确保所有信息与营业执照完全一致。

1.  **选择公司所在地**：选择 `中国`。
2.  **选择公司类型**：通常选择 `私有企业`。
3.  **填写公司名称**：
    *   中文全称：例如 `杭州博辰网络科技有限公司`。
    *   英文/拼音：需全部小写且无空格，例如 `hangzhoubochenwangluokejiyouxiangongsi`。
4.  **填写公司注册号**：输入营业执照上的“统一社会信用代码”，例如 `9111010117822640812P`。
5.  **填写公司地址**：完整填写营业执照上的注册地址，包括省、市、区、街道和具体门牌号。邮编可通过网络查询确认（例如 `310016`）。

**关键点**：此步骤信息提交后**无法返回修改**，务必仔细核对。

### 第三步：填写法人个人信息

此部分需要填写公司法人的详细信息，需与身份证信息严格对应。

1.  **姓名与国籍**：填写法人身份证上的姓名（`田丽琴`）和国籍（`中国`）。
2.  **出生日期与身份证号**：按身份证信息准确填写。
3.  **居住地址**：这是一个重要字段。建议填写**法人身份证地址**或**信用卡账单地址**。若店铺触发审核（如二审），提供与此地址一致的账单有助于通过审核。
4.  **确认受益人**：根据公司股权结构如实勾选。若为个人独资，可同时勾选“企业受益人”和“法人代表”；若有其他持股超过25%的股东，需额外添加其信息。

### 第四步：设置收款与付款方式

店铺需要绑定收款账户以接收货款，并绑定信用卡以支付月费和其他销售费用。

1.  **设置收款账户**：
    *   选择已准备好的收款服务商（如 `连连支付`）。
    *   填写账户持有人姓名（法人姓名）和账户识别码（可在收款平台后台找到）。
2.  **绑定信用卡**：
    *   填写信用卡卡号、有效期（格式：`09/2024`）、持卡人姓名。
    *   **账单地址**：务必更改为该信用卡的实际账单地址，并确保与之前填写的“居住地址”之一相符，以应对可能的地址验证。

### 第五步：店铺信息与身份验证

![](img/b337f7fc3393a3abbcaa4e4897bf705b_19.png)

最后一步是完善店铺信息并提交资质文件进行审核。

![](img/b337f7fc3393a3abbcaa4e4897bf705b_21.png)

1.  **设置店铺名称**：可先设定一个名称，店铺注册成功后可以修改。
2.  **产品信息**：关于UPC编码和品牌问题，若无品牌备案，通常选择 `否`。
3.  **提交验证文件**：
    *   需上传**营业执照**、**法人身份证正反面**的彩色扫描件或高清照片。
    *   **要求**：文件必须清晰、完整，四个边角均需露出。
4.  **最终核对与提交**：在点击最终提交按钮前，请再次全面核对所有已填写的信息。

### 第六步：视频认证与完成注册

提交资料后，系统会安排视频通话进行真人验证。

1.  **预约视频时间**：在系统提供的日历中选择一个方便的时间（通常为工作日）。
2.  **准备视频认证**：
    *   **设备**：确保电脑或手机的摄像头、麦克风功能正常。
    *   **材料**：法人需本人到场，并准备好**身份证原件**和**营业执照原件**。
    *   **网络**：如使用手机认证，**务必使用手机4G/5G流量，切勿连接Wi-Fi**，以防IP关联风险。
3.  **进行认证**：按时通过预留的链接进入视频房间。认证官会询问几个简单问题，如确认是否为本人、身份证号、计划销售什么产品等，过程约5-15分钟。
4.  **等待审核结果**：视频认证结束后，保存好登录信息，等待亚马逊的审核结果通知即可。

## 总结
本节课我们一起学习了亚马逊店铺注册的全流程。我们从注册前的材料准备讲起，逐步完成了创建账户、填写公司与法人信息、绑定收款与信用卡、提交资质审核，直至最后的视频认证。请记住，信息真实准确、文件清晰完整、注意防关联是注册成功的三大关键。熟练掌握此流程，你就能独立完成亚马逊卖家账户的注册，迈出跨境电商的第一步。
