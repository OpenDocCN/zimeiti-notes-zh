# 美客多选品教程：方法9：基于竞品店铺上新监控选品

![](img/59970cd97e477a649bb310821c39d85b_0.png)

## 概述
在本节课中，我们将学习一种高效的选品方法：通过监控优秀竞品店铺的每日上新情况，从中分析并筛选出有潜力的产品进行跟进。这种方法尤其适合供应链能力较强的卖家，能够帮助我们在产品早期抢占市场先机。

## 核心原理与优势
卖家可以通过监控优秀标杆店铺的每日上新情况，分析并选出其中最有潜力的产品进行跟进。这种选品方法更适合供应链能力较强的卖家。参考有竞争力店铺的原因是，他们的产品选择都经过了多重筛选，推广成功的概率较大。

趁他们的产品还在孵化状态就去跟进，比等他们推广成功后再跟进，更能够抢占市场先机。

## 操作步骤：如何监控竞品上新
那么，如何准确监控竞品店铺的上新情况呢？以下是具体的操作流程。

进入“大麦数据”工具，找到“商品竞品店铺”功能模块。将店铺级别设置为 **`green`**。选择你的目标竞品店铺，并筛选条件为：月销量大于 **`100`**，上架日期在最近 **`30`** 至 **`90`** 天。按照以上步骤操作，就可以顺利筛查出目标上新商品。

## 总结
本节课我们一起学习了基于竞品店铺上新监控的选品方法。我们了解了该方法的原理与优势，并掌握了利用“大麦数据”工具设置筛选条件、监控竞品店铺上新商品的具体操作步骤。通过早期跟进竞品的潜力新品，卖家可以更有效地抢占市场先机。
