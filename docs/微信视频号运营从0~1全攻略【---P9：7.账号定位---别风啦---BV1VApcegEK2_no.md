# 微信视频号运营从0到1全攻略：P9：7. 账号定位与素材获取 🎯

在本节课中，我们将要学习微信视频号运营中两个非常关键的基础步骤：如何正确处理已有账号内容，以及如何获取高质量的原创视频素材。这对于从零开始运营视频号至关重要。

上一节我们介绍了账号定位的重要性，本节中我们来看看如何为账号的启动做好内容准备。

## 账号内容初始化

![](img/0aea618322c7a871cf7971d8da784960_1.png)

有同学询问是否需要隐藏账号之前的作品。答案是肯定的，需要隐藏。这是为了确保账号以全新的、定位清晰的形象面向观众和平台推荐算法。

操作很简单：进入账号后台，将历史作品设置为“仅自己可见”或直接删除。这样做的目的是避免旧内容干扰新账号的垂直定位。

![](img/0aea618322c7a871cf7971d8da784960_3.png)

## 分步执行运营策略

启动初期，目标需要分步完成。首先集中精力解决获取**100个初始粉丝**的问题，然后再系统性地解决**视频素材**的问题。请按顺序执行，不要急于求成。

以下是分步策略的核心要点：
*   **第一步：获取100粉丝**。这是账号获得基础权重和初步流量的关键门槛。
*   **第二步：解决素材问题**。在拥有基础粉丝后，持续发布优质内容来巩固和扩大流量。

## 原创视频素材解决方案

很多同学面临“不会拍摄、不会剪辑、没有专业素材”的困境。针对这一普遍情况，我们提供了一个实质性的解决方案：为每位认真学习的同学提供**30个已剪辑好的高质量视频素材**。

这些素材来自一个投入巨大成本建立的独家素材库。该素材库具有以下特点：
*   **专业制作**：素材由专业剪辑师和选材师团队制作，单个剪辑师月成本高昂，确保了素材质量。
*   **品类丰富**：涵盖搞笑娱乐、生活妙招、情感等多种领域，满足不同账号定位需求。
*   **避免重复**：素材采用领取后链接失效的机制，保证分发给不同用户的素材不重复。

为了让您直观了解素材质量，以下是一个搞笑娱乐类素材的示例描述：
> 一段节奏轻快、配有动感音乐的视频片段，内容幽默诙谐，可能包含反转情节或可爱元素（如小朋友或动物），能有效吸引观众停留并互动。

![](img/0aea618322c7a871cf7971d8da784960_9.png)

领取素材需要满足几个简单的条件，这既是对课程的尊重，也是筛选真正有意学习者的方式。

以下是领取素材的具体步骤和要求：

2.  **积极互动**：在课程直播期间保持积极的评论和互动。

**请注意**：礼貌地联系助理老师并提及讲师名号，是顺利获得福利的有效方式。
