## 剪辑前的准备 🛠️

上一节我们介绍了视频拍摄的要点，本节中我们来看看如何将拍摄好的素材剪辑成完整的视频。请准备好你的手机，并确保已下载正确的剪映App。

![](img/0e2213636a77fcb06116a9aef29a0a7c_9.png)

![](img/0e2213636a77fcb06116a9aef29a0a7c_11.png)

*   **软件选择**：使用剪映App的黑色图标版本，不要下载错误。
*   **版本说明**：剪映有手机版和电脑版，本节课以手机版操作为主。

![](img/0e2213636a77fcb06116a9aef29a0a7c_13.png)

## 开始剪辑：导入与基础调整 🎬

打开剪映App，点击“开始创作”按钮，进入剪辑界面。

![](img/0e2213636a77fcb06116a9aef29a0a7c_15.png)

![](img/0e2213636a77fcb06116a9aef29a0a7c_17.png)

以下是导入视频并进行基础调整的步骤：

![](img/0e2213636a77fcb06116a9aef29a0a7c_19.png)

1.  **导入视频**：从相册中选择需要剪辑的视频片段，点击“添加”将其导入到剪辑轨道中。如果视频由多个片段组成，建议先导入并剪辑完一个片段后，再点击“+”号导入下一个片段，以此类推，避免混乱。
2.  **调整构图与比例**：
    *   点击轨道上的视频片段。
    *   在底部工具栏中向右滑动，找到并点击“编辑”按钮（图标通常是一个方块）。
    *   点击“旋转”功能，将视频画面调整到正确的方向。
    *   如需裁剪画面比例，可点击“裁剪”功能，选择如1:1、4:3、16:9等预设比例或自由裁剪。

![](img/0e2213636a77fcb06116a9aef29a0a7c_21.png)

![](img/0e2213636a77fcb06116a9aef29a0a7c_23.png)

## 剪辑核心：分割与删除 ✂️

![](img/0e2213636a77fcb06116a9aef29a0a7c_25.png)

基础调整完成后，需要对视频内容进行精剪，去除多余部分。

![](img/0e2213636a77fcb06116a9aef29a0a7c_27.png)

![](img/0e2213636a77fcb06116a9aef29a0a7c_29.png)

剪辑的核心操作是“分割”与“删除”。时间轴上的白色竖线代表当前播放指示器，也起到“剪刀”的作用。

![](img/0e2213636a77fcb06116a9aef29a0a7c_31.png)

![](img/0e2213636a77fcb06116a9aef29a0a7c_33.png)

*   **操作流程**：将白色竖线移动到需要裁剪的位置 -> 点击选中视频片段 -> 点击底部工具栏的“分割”按钮 -> 视频被切成两段 -> 选中需要删除的片段 -> 点击“删除”按钮。
*   **应用场景**：此方法适用于删除视频开头、中间或结尾处任何不需要的片段（如无说话的静默部分、无关动作等）。

![](img/0e2213636a77fcb06116a9aef29a0a7c_35.png)

## 添加背景音乐 🎵

![](img/0e2213636a77fcb06116a9aef29a0a7c_37.png)

![](img/0e2213636a77fcb06116a9aef29a0a7c_39.png)

视频内容修剪完毕后，需要添加合适的背景音乐来提升观感。

![](img/0e2213636a77fcb06116a9aef29a0a7c_41.png)

![](img/0e2213636a77fcb06116a9aef29a0a7c_43.png)

点击底部工具栏的“音频”按钮，进入音乐添加界面。

![](img/0e2213636a77fcb06116a9aef29a0a7c_45.png)

*   **提取音乐**：如果你已经使用“淘金阁”等工具生成了配音文件，可以点击“提取音乐”，直接从视频文件中导入音频。
*   **添加音乐**：如果没有预配音，则点击“添加音乐”。剪映内置了丰富的音乐库，你可以根据视频内容（如美食、美妆、萌宠等分类）搜索或选择音乐。点击音乐可以预览，选定后点击“使用”即可添加到轨道。
*   **剪辑音乐**：如果添加的音乐过长，处理方法与剪辑视频相同：选中音频轨道 -> 移动竖线到需要裁剪的位置 -> 点击“分割” -> 选中多余部分 -> 点击“删除”。
*   **调整音量**：选中音频轨道，点击“音量”按钮，可以拖动滑块调整背景音乐的音量大小，使其不会掩盖人声。

![](img/0e2213636a77fcb06116a9aef29a0a7c_47.png)

## 添加与修改字幕 📝

![](img/0e2213636a77fcb06116a9aef29a0a7c_49.png)

![](img/0e2213636a77fcb06116a9aef29a0a7c_51.png)

添加字幕能让观众更清晰地理解视频内容。剪映提供了高效的自动识别功能。

点击底部工具栏的“文字”按钮，然后选择“识别字幕”。

![](img/0e2213636a77fcb06116a9aef29a0a7c_53.png)

![](img/0e2213636a77fcb06116a9aef29a0a7c_55.png)

*   **识别设置**：在弹出的选项中，建议选择“仅视频”，这样可以避免将背景音乐里的歌词也识别为字幕。
*   **修改字幕**：识别完成后，字幕会自动添加到视频中。你需要：
    *   **检查错别字**：双击字幕条，可以对识别错误的文字进行修改。
    *   **调整样式**：选中字幕，可以修改字体、颜色、大小、位置等。例如，可以根据画面主色调选择匹配的字幕颜色。

![](img/0e2213636a77fcb06116a9aef29a0a7c_57.png)

## 导出视频前的关键设置 ⚙️

![](img/0e2213636a77fcb06116a9aef29a0a7c_59.png)

视频剪辑全部完成后，在导出前需要进行关键设置，以保证视频清晰度。

![](img/0e2213636a77fcb06116a9aef29a0a7c_61.png)

点击右上角的“导出”按钮，在导出设置界面中：

![](img/0e2213636a77fcb06116a9aef29a0a7c_63.png)

*   **帧率 (FPS)**：设置为 **30**。这个帧率对于抖音平台最为合适，能平衡流畅度与文件大小。
*   **分辨率**：尽可能选择最高的选项（如1080p）。分辨率越高，视频越清晰。
*   **导出注意事项**：点击“导出”后，请耐心等待，不要锁屏或切换到其他应用程序，以免中断导出过程。

![](img/0e2213636a77fcb06116a9aef29a0a7c_65.png)

![](img/0e2213636a77fcb06116a9aef29a0a7c_67.png)

导出完成后，可以直接通过剪映分享到抖音平台。

## 辅助工具推荐：爱提词 🗣️📱

对于需要录制口播视频的同学，记台词是一大难题。接下来我们介绍一个能有效解决这个问题的辅助工具——“爱提词”App。

![](img/0e2213636a77fcb06116a9aef29a0a7c_73.png)

![](img/0e2213636a77fcb06116a9aef29a0a7c_75.png)

“爱提词”是一款提词器软件，能让你在录制时看着台词念，无需背诵。

![](img/0e2213636a77fcb06116a9aef29a0a7c_77.png)

![](img/0e2213636a77fcb06116a9aef29a0a7c_79.png)

以下是其主要功能和使用方法：

![](img/0e2213636a77fcb06116a9aef29a0a7c_81.png)

![](img/0e2213636a77fcb06116a9aef29a0a7c_83.png)

1.  **添加台词**：打开App，点击右下角“+”号，创建新台词稿。你可以直接输入或粘贴复制好的文案，并为其设置一个标题（如“情感语录”），然后保存。
2.  **开始提词**：保存后，点击“开始提词”，选择“录制模式”。此时，台词会以悬浮窗形式显示在手机屏幕上，你可以一边看着台词，一边进行录制。
3.  **个性化设置**：
    *   **调整字体**：点击屏幕中央的设置按钮，可以调整台词文字的字体、大小和滚动速度，方便近视的同学观看。
    *   **内置美颜**：点击左下角的“美颜”按钮，可以开启瘦脸、大眼、磨皮、美白等美颜功能。它的优势在于美颜效果相对自然，不会像一些美颜相机那样滤镜过重。
4.  **爆款文案参考**：App内还提供了“爆款文案”板块，分类整理了职场、情感、生活、教育等多个领域的文案素材，可供创作时参考和借鉴。
