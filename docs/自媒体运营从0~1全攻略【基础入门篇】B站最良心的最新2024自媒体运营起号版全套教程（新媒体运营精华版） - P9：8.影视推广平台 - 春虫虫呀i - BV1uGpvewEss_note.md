# 自媒体运营从0到1全攻略：P9：影视推广平台详解 🎬

在本节课中，我们将要学习如何通过“影视推广”平台，在抖音、快手等熟悉的平台上发布视频并结算收益。我们将详细讲解其运作模式、任务流程以及各平台的收益计算方式。

---

上一节我们介绍了视频在多平台发布的基础逻辑，本节中我们来看看一个特殊的收益结算方式——影视推广。

很多同学问到，如果想把视频发布到抖音或快手平台，该如何结算收益。需要明确的是，将视频发布到除抖音、快手外的其他30多个自媒体平台，通常可以依靠视频本身的播放量直接结算收益。然而，抖音和快手这两个平台有所不同。

就像之前提到的，视频在这两个平台发布，最主要的作用是增长粉丝。但有些同学可能不仅想运营抖音和快手来涨粉，同时也希望通过播放量来获得收益。

所有制作完成的视频，可以发布到以下四个大家熟悉的平台：
*   抖音
*   快手
*   小红书
*   微信视频号

小红书平台可能更多被女性或年轻用户使用。微信视频号则是微信内置的功能，人人可用。在这几个平台发布视频后，我们需要通过“影视推广”平台来接取任务，并通过完成任务来结算奖金。

那么，影视推广具体是什么意思？影视推广指的是，每一部电视剧、电影或综艺节目都需要进行宣传。现在许多广告商或制片方，会利用影视推广平台，让创作者制作视频帮助推广内容。我们可以通过完成推广任务，依靠视频的播放量来结算奖金。

每个视频制作完成后，需要提交一个任务回填链接来报名。以下是需要填写的信息：
*   个人的微信号、账号昵称，或抖音/快手的ID
*   所推广的具体电视剧名称
*   发布作品的平台链接
*   作品的发布时间
*   视频播放量的截图

回填链接提交并审核通过后，平台就会结算收益。影视推广的收益通常比单纯依靠平台播放量分成的收益要高。

现在，让我们具体看看影视推广的收益情况。目前平台上有非常多的短剧推广任务。许多传媒公司或网红会自制短剧，他们同样需要宣传，因此会发布任务让我们制作视频并发布到指定平台。

短剧任务通常可发布至以下平台：
*   抖音
*   小红书
*   微信视频号

收益根据播放量要求计算，并且**对账号粉丝数没有要求**。结算时，通常会取一部剧推广视频中，**播放量最高的三个作品，其播放量总和**作为计算依据。

各平台收益计算标准如下（以每万次播放的奖金为例）：
*   **抖音**：1万播放量奖金为 **50元**。（对比：头条号1万播放量收益约为34元）
*   **微信视频号**：1万播放量奖金为 **10元**。单价较低是因为微信用户基数庞大，触达容易。
*   **小红书**：单价最高，**1000播放量**奖金即为 **30元**。

此外，播放量超过基础要求后，每增加一定播放量通常还会有额外奖金。

除了上述按播放量结算的任务，还有一类任务需要同时发布到抖音和快手平台，并按**点赞数**结算。

这类任务同样没有粉丝要求。结算标准如下：
*   **抖音**：单条作品达到5000赞奖励100元；达到1万赞奖励200元。
*   **快手**：单条作品达到5000赞奖励100元；达到1万赞奖励180元。

在众多任务中，**搜狐短剧**推广任务给出的单价通常是最高的。

本节课中我们一起学习了影视推广平台的运作模式。我们了解到，通过在抖音、快手、小红书、视频号发布指定的推广视频，并完成平台任务回填，即可结算收益。关键点在于：**收益与账号粉丝数无关**，主要依据**播放量**或**点赞数**计算，且影视推广的收益单价往往高于普通平台流量分成。这为想在熟悉平台同时实现涨粉和变现的创作者，提供了一条可行的路径。
