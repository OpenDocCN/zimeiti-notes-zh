## 一、拼图制作：美图秀秀
上一节我们介绍了核心工具，本节中我们来看看如何利用美图秀秀进行拼图。拼图适用于需要展示多张关联图片的场景。

以下是适合使用拼图功能的笔记类型：
*   电影/电视剧推荐
*   旅游/民宿分享
*   穿搭/好物分享
*   美食推荐

**操作要点：**
打开美图秀秀，选择“拼图”功能，导入多张图片。**关键点在于模板的选择**：请避免使用“海报”、“自由”或“拼接”模式，**仅使用“模板”**。依靠模板的自带布局和特效，就能组合出适合小红书封面的拼图。

以下是四类适合做拼图的图片场景：
1.  **同类产品不同款式**：例如，展示同一品牌不同款式的帽子。
2.  **效果对比图**：例如，使用美白产品在不同阶段的效果对比。
3.  **产品图+效果图**：展示产品本身与使用后的效果对比。
4.  **影视剧截图**：拼接经典剧情画面或对话截图。

## 二、图片加文字：美图秀秀
除了拼图，为图片添加醒目文字也是制作首图的常用方法。文字能直接传达笔记核心信息。

**制作要点如下：**
1.  **文字要足够大**：确保用户一眼就能看清标题。
2.  **形成反差**：文字颜色需与图片背景形成鲜明对比。例如，浅色背景用深色字，深色背景用浅色字。
3.  **简洁明了**：避免使用过多字体和倾斜特效，保证易读性。
4.  **背景干净**：纯色或简洁的背景更利于突出文字和产品。
5.  **控制字体种类**：单张图片使用的字体**最好不超过3种**，并以黑、白等基础色为主。

对于需要罗列产品特点的图片，可以采用“中心产品图+周围文字说明”的布局，让用户通过封面就能快速了解产品核心卖点。

## 三、抠图应用：天天P图
接下来，我们学习如何利用天天P图进行抠图。抠图功能能将产品主体从原图中分离出来，便于进行创意组合。

以下是三类适合使用抠图的笔记场景：
1.  **穿搭测评**：将整套穿搭单品抠出，并列展示。
2.  **干货科普**：例如，将不同类别的美容仪产品抠出并标注名称。
3.  **横向测评**：将同类型多个产品抠图后集中展示在一张图上。

**抠图注意事项：**
在开始操作前，需要了解以下几点：
*   **素材选择**：适合抠图的是细节丰富的小物件（如化妆品、首饰），而非汽车、房屋等大场景。
*   **背景要求**：抠图原理是识别主体与背景的差异。因此，**背景越干净、单一**（纯色为佳），主体越突出，抠图效果越好。
*   **次数限制**：该App的免费抠图功能有每日次数限制。用完后，可通过卸载重装App来重置。

**抠图步骤：**
1.  打开天天P图，选择“魔法抠图”功能。
2.  选择喜欢的背景模板。
3.  导入需要抠图的照片。
4.  用手指涂抹需要保留的主体部分，软件会自动识别边缘并完成抠图。
5.  将抠出的素材保存或直接使用。
