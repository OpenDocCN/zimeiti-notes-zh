# 抖音运营指南：P17：11、视频类型1-2 - 人设与口播

在本节课中，我们将要学习抖音账号运营中两个核心概念：**人设**的建立与区分，以及**口播**这种视频形式的具体应用。理解并掌握这些内容，是连接内容创作与粉丝增长、实现变现的关键环节。

上一节我们探讨了文案的重要性，本节中我们来看看如何通过塑造人设和选择视频形式来提升账号吸引力。

## 人设的建立与区分

从单纯追求涨粉到最终实现变现，中间有一个必不可少的环节，即建立你的**人设**。建立人设的目的并非为了炫耀，而是为了在众多同类创作者中脱颖而出。例如，在十个做装修的账号中，如果有一位创作者塑造了幽默搞笑的人设，他就更容易被观众记住。当观众喜欢你这个人时，他们自然就会成为你的粉丝。

![](img/be60e6da41bb8a9a57c626a6dfd48bbd_1.png)

人设主要分为两种类型：**强人设**与**弱人设**。区分清楚这两种人设，有助于你准确判断视频类型并规划自己的创作方向。

![](img/be60e6da41bb8a9a57c626a6dfd48bbd_3.png)

以下是两种人设的具体定义：

*   **强人设**：以创作者**个人**为切入点的内容。这包括分享与你自身相关的行业经验、工作生活或个人兴趣爱好。内容核心与你本人紧密相连。
    *   **公式**：**强人设 = 个人经验/生活/爱好 + 真人出镜（常见）**
    *   **示例领域**：装修、办公、房产、汽车、餐饮、婚介、法律咨询、电商、美食、旅行、手工、宠物、星座、美妆、护肤、服装穿搭、游戏、摄影等。

*   **弱人设**：内容核心与创作者**个人**关联度较低，主要目的是运营好账号本身。视频焦点在于内容主题，而非塑造创作者的个人形象。
    *   **公式**：**弱人设 = 主题内容（如文案、影视） ± 创作者声音/形象**
    *   **示例类型**：情感语录配乐视频、影视解说等。在这些视频中，创作者的声音或形象主要是为内容服务，而非展示个人特质。

## 视频内容形式：口播详解

明确了人设方向后，任何与你个人相关的内容都可以转化为视频。接下来，我们详细讲解第一种常见且高效的内容形式：**口播**。

口播，即用口头讲述的方式制作视频，是分享故事和知识的强大工具。它主要分为两大类：

1.  **讲故事型口播**：分享客户故事、从业经历或人生感悟。故事本身不要求完全真实，可以从文案平台寻找有趣素材。这类口播的目的是让观众觉得内容有意思，吸引他们听下去。
2.  **教知识型口播**：分享特定领域的专业知识或实用信息。例如育儿、金融、养生、装修、房产、汽车等领域的知识科普。这类口播的目的是让观众在几十秒内获得“收获感”，觉得学到了东西。

以下是关于口播的常见问题与解答：

*   **哪些领域适合做口播？**
    *   几乎所有知识型或经验分享型领域都适合，如法律、保险、金融、装修、汽车、房产知识等。
*   **口播难拍吗？**
    *   口播拍摄相对简单。只需准备好手机和文案，将手机固定后录制即可。
*   **口才不好能做口播吗？**
    *   可以。口播依赖的是**提前准备好的文案稿**，并非临场发挥。创作者都是依据稿子进行录制。
*   **普通话不标准能做口播吗？**
    *   可以。如果你的目标客户是同城或有地域属性，使用**方言**可能效果更佳，能增加亲切感和趣味性。同时，可以通过添加字幕来确保信息传达无障碍。

本节课中我们一起学习了抖音账号运营中“人设”的概念及其“强”、“弱”分类，并深入探讨了“口播”这一视频形式的具体应用、分类和实操要点。建立清晰的人设是吸引目标粉丝的基础，而熟练运用口播则是高效输出内容、传递价值的关键技能。掌握这两点，能为你的抖音账号快速成长奠定坚实基础。
