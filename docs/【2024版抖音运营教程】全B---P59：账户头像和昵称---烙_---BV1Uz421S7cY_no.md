# 抖音运营教程：P59：账户头像和昵称 🎯

![](img/f3995688779b8d583f301d848cecdc31_1.png)

在本节课中，我们将要学习抖音账号搭建中最基础也最重要的一步：如何设置头像和昵称。一个清晰、专业的头像和昵称是吸引用户关注、传递账号价值的第一步。

## 头像的选择标准

![](img/f3995688779b8d583f301d848cecdc31_3.png)

上一节我们介绍了账号搭建的重要性，本节中我们来看看头像的具体选择标准。头像的选择应基于你的内容定位。

*   **有人物出镜的内容**：如果你的视频内容中有人物出镜，那么最好直接使用该人物的照片作为头像。
*   **无人物出镜的内容**：如果你的内容不涉及人物出镜，则可以考虑使用与行业相关的关键词、图标、文字或风景图片作为头像。

以下是几个头像选择的例子：

![](img/f3995688779b8d583f301d848cecdc31_1.png)

*   **“秋叶PPT”**：该账号专注于办公软件教学，其头像直接使用了Excel软件的图标，能清晰传递账号价值。
*   **“好物种草”**：该账号使用纯文字作为头像，直接明了。
*   **“恋爱指南”**：该账号内容有人物出镜，因此直接使用人物照片作为头像。

一个好的头像能有效传递价值，让用户产生“关注他可能对我有用”的想法。

![](img/f3995688779b8d583f301d848cecdc31_5.png)

### 头像设置的禁忌

在设置头像时，需要注意以下几点禁忌：

*   **尺寸要适中**：头像不要过大或过小，确保在各类显示场景下都清晰可辨。
*   **人物头像注意事项**：使用人物头像时，脸部不宜过大（显得突兀）或过小（看不清）。尽量避免使用多人合照，最多使用两人的合照。

![](img/f3995688779b8d583f301d848cecdc31_5.png)

## 昵称的设置技巧

讲完了头像，接下来我们看看昵称的设置。一个好的昵称应该能让用户一眼就知道你是做什么的。

以下是几个昵称的例子：

*   **瑞普PPT**：直接包含行业关键词“PPT”。
*   **美妆师咕咚**：包含行业关键词“美妆”和人设“咕咚”。
*   **跟蒋晖学电商**：包含行业关键词“电商”、人设“蒋晖”和价值体现“学”。

这类昵称吸粉能力较强，因为目标用户能快速识别其价值。我们可以通过“粉赞比”（粉丝数：总获赞数）来粗略评估一个账号的吸粉效率。在竞争激烈的领域（如美妆、美食），较好的粉赞比约为 **7:1 或 8:1**。在细分领域（如PPT教学、电商教程），较好的粉赞比可能达到 **2:1 到 4:1**。

### 关于“赞少粉多”现象的解释

有时我们会看到一些账号点赞数不多但粉丝量很大，这通常有以下几种原因：

1.  **删除爆款视频**：创作者删除了过去点赞量很高的视频，但通过那些视频积累的粉丝被保留了下来。
2.  **以直播吸粉为主**：一些账号（尤其是带货主播）主要依靠直播而非视频内容来吸引粉丝，因此视频点赞数相对较少。
3.  **跨平台迁移**：创作者从其他平台（如与抖音打通的今日头条）迁移过来，粉丝数据同步，但抖音本身的视频作品和点赞数不多。

### 取昵称的三个核心建议

以下是设置昵称的三个关键建议：

1.  **包含行业关键词**：例如“PPT”、“美妆”、“电商”等，让用户知道你的领域。
    *   `昵称 = 行业关键词 + ...`
2.  **加入人设**：加入你的名字或称号，使账号更具人格化和辨识度。
    *   `昵称 = ... + 人设（如：咕咚、蒋晖）+ ...`
3.  **体现价值**：使用“教学”、“学”、“指南”等字眼，明确告诉用户你能提供什么价值。
    *   `昵称 = ... + 价值体现（如：教学、学）`
