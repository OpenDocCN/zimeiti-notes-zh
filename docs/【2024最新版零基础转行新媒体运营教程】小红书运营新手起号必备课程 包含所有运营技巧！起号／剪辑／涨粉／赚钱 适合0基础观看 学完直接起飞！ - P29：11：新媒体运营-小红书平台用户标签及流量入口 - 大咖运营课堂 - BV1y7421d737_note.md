在本节课中，我们将深入探讨小红书平台如何为用户打上标签，以及平台主要的流量入口有哪些。理解这些底层逻辑，将帮助你更好地分析自己的账号，并制定有效的运营策略，让你的笔记被更多精准用户看到。

上一节我们介绍了小红书如何判定内容标签，本节中我们来看看平台如何判定用户标签。这个过程能帮助我们理解笔记是如何被系统推荐给用户的，并可以反推如何让笔记被目标用户看到。

## 用户标签判定逻辑

系统为用户打标签的主要依据是用户的**浏览行为**。用户注册时选择的兴趣范围，其参考价值不大，仅作为初期内容方向的引导，无法进入系统深层的用户标签池。

系统通过分析用户的一系列互动行为来判定其兴趣分类。以下是主要的判定维度：

### 1. 点击率
**点击率**是指系统在发现页向用户推送多个内容（如明星、美妆、家居、母婴等）后，用户点击各类内容的概率。例如，用户点击了10条内容，其中美妆类3次，家居类2次，母婴类0次。系统会判定该用户对母婴内容不感兴趣，可能不是宝妈，而对美妆内容更感兴趣。系统会持续推送更多美妆内容来试探用户，从而为用户打上**强关联**或**弱关联**的标签。

### 2. 停留时长
**停留时长**是另一个关键行为。例如，用户喜欢美妆内容，但点开某些美妆笔记后很快划走，说明这些笔记不吸引他。系统会尝试推送美妆领域下的其他细分内容（如上粉、护肤）或其他领域内容来观察。如果用户点开一篇平常很少点击的家居笔记，并停留了很长时间，仔细阅读图片和评论，系统可能判定他有装修或购买家居用品的需求，从而调整推荐策略。

### 3. 页面浏览量
**页面浏览量**可以理解为用户浏览的深度。例如，用户点开一篇装修笔记后，不仅看了这篇笔记，还进入了博主主页，并浏览了第二、第三、第四篇笔记。这个浏览页面的数量就是PV值。提升用户在你主页的浏览深度，能有效增加账号的健康度评分。

### 4. 互动行为
互动行为包括**点赞、收藏、评论、关注**。这些数据是衡量笔记质量的核心指标。提升这些数据不仅能增加笔记进入更大推荐流量池的几率，还能带动账号内其他笔记的流量，并有助于未来接单和变现。

## 平台流量分发入口

了解用户标签后，我们来看看小红书主要的流量分发入口。平台主要有四个流量入口，各自承担不同的功能。

以下是各流量入口的详细介绍：

*   **发现页**：这是最主要的流量来源。其内容包含：
    *   用户已关注的达人发布的内容。
    *   根据用户标签（点击率、停留时长等）匹配的、系统认为用户可能喜欢的兴趣内容。
    *   一些用户未曾接触过的发散性内容，以增加平台内容的多样性，满足用户潜在的多方位需求。

*   **关注页**：这里集中展示用户所有已关注博主发布的最新笔记。当用户关注列表过长时，可以在这里查找特定博主的内容。

*   **附近页**：此入口基于手机定位，向用户推送同城或附近的内容。内容多以**美食、本地旅游、线下店铺**等具有强地域特征的需求为主。对于运营线下实体店（如美容院、餐厅）的账号，需要重点关注这个入口，并在投放信息流广告时考虑附近人群的推送。

*   **购物页**：这是小红书向电商化发展的重要入口，直接满足用户的购物需求。目前，部分用户尚未养成在小红书直接购物的强习惯，更多人仍倾向于先搜索相关笔记（种草）再做决策。
