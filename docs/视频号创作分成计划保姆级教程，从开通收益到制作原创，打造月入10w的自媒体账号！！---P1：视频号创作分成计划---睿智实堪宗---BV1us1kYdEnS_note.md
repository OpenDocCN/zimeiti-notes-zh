## 项目原理

视频号创作分成计划的原理非常简单。其核心逻辑是：**创作者满足平台条件 → 平台在视频中匹配广告 → 用户观看或点击广告 → 创作者获得收益**。

这个过程类似于其他平台的中视频计划。创作者需要先开通“创作分成计划”功能，之后平台会在你发布的视频下方或播放过程中自动插入广告。这些广告产生的曝光和点击，就会为你带来收益。

## 项目门槛

开通创作分成计划需要满足三个核心条件，这也是项目的关键门槛。

以下是具体的开通要求：
1.  **粉丝数量**：需要达到100个有效粉丝。
2.  **内容规范**：发布的内容必须符合平台规范，不能涉及违规、血腥、色情等不良信息，且视频画风需要精美。
3.  **原创作者认定**：平台需要认定你是一个优质的原创作者。

![](img/abe49eadc434253189a67f88d7ac536f_7.png)

对于许多新手来说，如何满足这些条件，特别是“成为优质原创作者”这一条，存在信息差。我们将在后续课程中详细讲解如何高效、正确地达成这些条件。

## 项目优势

![](img/abe49eadc434253189a67f88d7ac536f_9.png)

接下来，我们来看看这个项目的几大优势。

以下是该计划的主要优势点：
*   **官方扶持与流量大**：视频号创作分成计划是今年（课程录制时）才推出的官方项目。对于通过计划的创作者，平台会给予流量扶持，视频更容易获得大量曝光。
*   **正处于风口期**：目前该项目类似于前两年的中视频计划初期，属于红利期。入局的创作者较少，竞争相对较小，视频收益和流量都处于较高水平。这被认为是下半年的风口项目。
*   **收益潜力可观**：根据操作案例，单条视频的收益可以很高（例如展示的案例中，有单条视频收益超过1000元），且多个账号操作可以带来稳定收益。

# 视频号创作分成计划保姆级教程：P2：如何开通创作分成计划

上一节我们介绍了项目的原理和优势，本节中我们来看看开通创作分成计划的具体方法和关键注意事项。

![](img/abe49eadc434253189a67f88d7ac536f_11.png)

开通计划有一个至关重要的前提：**在未满足开通条件前，绝对不要点击“视频变现任务”**。如果先点击了“视频变现任务”，后续将很难甚至无法开通“创作分成计划”。这是许多教程会误导新手的关键点。

正确的路径是：满足条件后，在创作者服务中心会直接出现“创作分成计划”的入口。而“视频变现任务”是另一个功能，切勿混淆。

## 前期作品发布策略

![](img/abe49eadc434253189a67f88d7ac536f_17.png)

为了满足“优质原创作者”的认定条件，在涨粉到100之前，我们需要发布一系列特定作品来“塑造”账号形象。

![](img/abe49eadc434253189a67f88d7ac536f_19.png)

以下是前期发布作品的核心要求：
*   **拍摄方式**：必须使用自己的手机亲自拍摄。
*   **内容主题**：建议拍摄户外风景、花花草草。画面要清晰。
*   **出镜与文案**：不露脸，不添加任何文案或字幕。
*   **背景音乐**：可以添加纯音乐，也可以不加。
*   **发布频率**：每天发布3-5个视频。
*   **视频时长**：每个视频控制在13-15秒之间。

这种视频的目的是向平台证明你是一个**有原创拍摄能力**的创作者，而不是搬运号。

## 内容垂直度的重要性

在发布这类视频时，**内容垂直**至关重要。平台会根据你发布的内容类型来判定你的创作领域。

![](img/abe49eadc434253189a67f88d7ac536f_28.png)

以下是保持垂直度的关键点：
*   **视频比例固定**：如果发横屏视频，就一直发横屏；如果发竖屏，就一直发竖屏。
*   **内容主题聚焦**：如果拍花草，就一直拍花草。不要今天拍花草，明天拍山川，后天拍街道。内容杂乱会让平台无法对你的账号进行定位，从而影响“优质原创作者”的认定。

发布这类视频的主要目的是通过原创认证，但它们通常涨粉较慢。下一节课，我们将专门讲解如何快速涨粉到100。

# 视频号创作分成计划保姆级教程：P3：快速涨粉的四大技巧

上一节我们讲解了如何通过发布原创视频让平台认可我们，本节我们来看看如何快速吸引粉丝，达到100粉的开通门槛。

视频号的主要用户群体是中老年人，因此我们的涨粉内容需要符合他们的喜好。以下是四种经过验证的有效涨粉视频类型。

## 四大涨粉视频类型

以下是具体可以制作的视频类型及其要点：

1.  **财神开门特效视频**：使用“财神开门”等喜庆模板，替换背景图为实拍街道或家庭场景，寓意美好，深受中老年用户喜爱。
2.  **喜鹊报喜视频**：使用“喜鹊报喜”模板，将背景图替换为从小红书等平台下载的梅花等吉祥图片，搭配祝福文案。
3.  **老年人爱情故事**：搜集老年人爱情主题的图片（可从小红书获取），用剪映拼接成视频，添加感人的文案和背景音乐（如《牵手》），讲述相守一生的故事。
4.  **祝福文案视频**：寻找热门祝福类视频，通过录屏提取其配音。然后搭配从小红书下载的优美风景视频，重新制作字幕，合成新的祝福视频。

## 制作流程示例（以祝福文案视频为例）

以下是制作一个祝福文案视频的通用步骤：

![](img/abe49eadc434253189a67f88d7ac536f_32.png)

1.  **找配音**：在视频号搜索“祝福”等关键词，找到点赞高的视频，用手机录屏保存。
2.  **找背景素材**：在小红书搜索“风景视频”，使用多平台下载工具保存高清风景素材。
3.  **视频合成**：将风景视频导入剪映，将录屏视频的音频分离出来并拖入轨道，关闭风景视频原声。
4.  **添加字幕**：根据配音，使用“智能字幕”功能生成字幕，并调整字体、颜色和大小。
5.  **导出发布**：导出视频，在视频号发布时，参考同行撰写类似的正能量祝福文案。

![](img/abe49eadc434253189a67f88d7ac536f_34.png)

这些视频内容精准击中目标用户心理，互动数据和涨粉速度都非常快。坚持发布，通常两三天即可达到100粉丝。

## 作品制作方法（以搞笑视频为例）

以下是以搞笑视频为例，演示一个完整的混剪二次创作流程：

1.  **搜集素材**：在抖音等平台搜索“搞笑视频”，找到多个目标素材。
2.  **下载素材**：使用去水印下载工具，将视频素材保存到本地。
3.  **导入剪辑**：将下载的多个搞笑视频导入剪映。
4.  **选取片段**：从每个原视频中，截取最精彩、最好笑的片段（每个片段几秒到十几秒）。
5.  **拼接重组**：将这些来自不同原视频的片段打乱顺序，重新拼接成一个新的完整视频。
6.  **添加转场与特效**：在片段之间添加搞笑的转场动画或贴纸，增强趣味性。
7.  **进行驱虫处理（关键步骤）**：这是避免被平台判定为搬运的核心操作。可以通过以下方式实现：
    *   **添加画中画**：新建复合片段后，叠加一个半透明的、动态的装饰素材（如星光、扫光动画），将混合模式设置为“绿色”或“叠加”，并降低不透明度。
    *   **调整参数**：为视频添加滤镜、调节色调、对比度等。
    *   **添加元素**：在视频角落添加不影响主画面的贴纸或文字。
8.  **添加背景音乐**：配上合适的背景音乐。
9.  **导出发布**：导出视频，发布到视频号。

其他赛道的制作流程与此类似，只是素材主题不同。核心都是**多素材混剪 + 深度驱虫处理 = 原创度高的二次创作作品**。视频发布后获得播放量，就能产生收益。

# 视频号创作分成计划保姆级教程：P5：核心注意事项

上节课我们介绍了收益赛道的作品制作方法，本节是最后一课，我们将总结在整个操作过程中必须严格遵守的几点核心注意事项，这些细节直接关系到账号的成败。

## 必须遵守的规则

以下是操作视频号创作分成计划时需要牢记的关键点：

1.  **禁止重复发布**：同一个视频绝对不能发布第二次。如果某个视频流量好，想再次发布以获得更多收益，这种行为会被平台直接判定为搬运，导致处罚。
2.  **禁止互粉刷粉**：100个粉丝必须通过发布内容自然增长而来。不要让亲朋好友互相关注，也不要参与任何互粉活动。这类无效粉丝会对开通创作分成计划造成严重阻碍。
3.  **开通后清理旧视频**：成功开通创作分成计划后，**第一件事就是删除之前为了涨粉和认证而发布的所有视频**（包括花草视频和涨粉视频）。清理旧内容，保持账号内容与未来收益赛道的一致性。
4.  **开通前绝不点“原创”**（重中之重）：在成功开通创作分成计划之前，发布任何视频时，**绝对不要勾选“原创”声明**。如果提前勾选，很可能导致永远无法开通分成计划。这一点必须牢记。

这些注意事项，特别是第3点和第4点，是很多新手容易忽略却至关重要的步骤。严格遵守可以避免账号走弯路。
