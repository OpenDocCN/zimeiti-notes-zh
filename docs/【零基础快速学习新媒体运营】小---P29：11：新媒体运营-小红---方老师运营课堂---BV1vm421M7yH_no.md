# 新媒体运营：P29：11：小红书平台用户标签及流量入口 📊

![](img/b618d292913e8a0dd9c82d7bb153aa3e_1.png)

在本节课中，我们将深入探讨小红书平台如何为用户打上标签，以及平台内主要的流量入口是如何运作的。理解这些底层逻辑，能帮助我们更好地规划内容，让笔记精准触达目标用户。

上一节我们介绍了小红书如何判定内容标签，本节中我们来看看平台如何判定用户标签。这个过程能帮助我们反向推导，理解自己的笔记是如何被系统推荐给潜在用户的。

## 用户标签判定逻辑 🔍

小红书系统为用户打标签，主要依据是用户的**浏览行为**。用户注册时选择的兴趣范围，仅作为初期内容引导，参考价值不大，无法进入系统深层的用户标签池。

系统通过分析用户的具体行为来判定其兴趣分类。以下是几个核心的行为指标：

### 1. 点击率
**点击率**指系统在发现页向用户推送多条内容后，用户点击不同类型内容的概率。例如，系统推送了明星、美妆、家居、母婴等内容，用户点击美妆类内容三次，家居类两次，母婴类一次或未点击。系统便能判定该用户对母婴内容不感兴趣，可能不是宝妈，并对美妆内容更感兴趣。随后，系统会持续推送更多美妆内容进行试探，从而为用户打上**强关联**或**弱关联**的标签。

### 2. 停留时长
**停留时长**是另一个关键行为指标。即使用户点击了某类内容（如美妆），但如果很快划走，说明该笔记并非其真正喜好。系统会继续推送同类目下的其他细分内容（如上妆、护肤）进行观察。反之，如果用户对平时较少点击的类目（如家居）某篇笔记停留很久，反复浏览图片和评论，系统可能判断用户有潜在需求（如装修、购买餐具），从而调整推荐策略。

### 3. 页面浏览量（PV）
**PV** 指用户浏览页面的深度。例如，用户不仅看了一篇装修笔记，还点进博主主页，并连续看了第二、第三篇笔记。这提升了该用户在你主页的浏览页面数。提升用户在你账号内的浏览深度，能显著增加账号的健康度评分，对账号成长有巨大帮助。

### 4. 互动行为
互动行为包括**点赞、收藏、评论、关注**。这些数据直接关系到笔记能否进入更大的推荐流量池。互动数据高的笔记，获得推荐的机会更大，也能带动账号内其他笔记的流量，并有助于积累粉丝，为后续变现（如接单）打下基础。

## 平台流量分发入口 🚪

了解用户标签后，我们还需知道流量从何而来。小红书主要有四个流量分发入口：

以下是各入口的详细说明：

*   **发现页**：这是最主要的流量来源。内容包含：
    *   用户已关注的达人发布的内容。
    *   系统根据用户标签（点击率、停留时长等）推荐的、其可能感兴趣的内容。
    *   少量系统试探性推荐的、用户未接触过的多样化内容，以丰富平台生态。
*   **关注页**：这里集中展示用户所有关注博主的最新笔记。当关注列表过长时，用户可能会直接在此页寻找特定博主。
*   **附近页**：基于手机定位，推送同城或附近的内容，尤其适合具有强地域特征的需求，如**美食、本地旅游、线下店推广（如美容院）**。做线下实体店推广时，应重点关注此入口。
*   **购物页**：这是小红书向电商化发展的重要入口，直接匹配用户的购物需求。目前用户习惯仍在培养中，许多交易仍通过笔记“种草”完成。

## 总结 📝

本节课中，我们一起学习了小红书平台判定用户标签的核心逻辑，包括**点击率、停留时长、页面浏览深度（PV）和互动行为**。同时，我们也梳理了平台四大流量入口：**发现页、关注页、附近页和购物页**。
