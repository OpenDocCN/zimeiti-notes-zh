# 小红书运营教程：P35：33.蒙版 🎬

![](img/eabfc9ac0a6b799d4d5f89d8bb1627f1_0.png)

![](img/eabfc9ac0a6b799d4d5f89d8bb1627f1_2.png)

在本节课中，我们将要学习视频剪辑软件中一个非常实用的功能——蒙版。蒙版功能可以帮助我们以特定的形状来展示视频或图片素材，实现分屏、聚焦、创意遮罩等视觉效果，是提升视频表现力的重要工具。

![](img/eabfc9ac0a6b799d4d5f89d8bb1627f1_4.png)

## 蒙版的基本概念

![](img/eabfc9ac0a6b799d4d5f89d8bb1627f1_6.png)

![](img/eabfc9ac0a6b799d4d5f89d8bb1627f1_8.png)

蒙版功能允许我们选中轨道上的素材，然后为其添加一个特定形状的“窗口”。只有窗口内的部分会被显示出来，窗口外的部分则会被隐藏。大部分蒙版样式都以几何形状的方式呈现。

![](img/eabfc9ac0a6b799d4d5f89d8bb1627f1_10.png)

![](img/eabfc9ac0a6b799d4d5f89d8bb1627f1_12.png)

上一节我们介绍了基础的剪辑操作，本节中我们来看看如何利用蒙版来创造更丰富的视觉效果。

![](img/eabfc9ac0a6b799d4d5f89d8bb1627f1_14.png)

## 蒙版的类型与获取

![](img/eabfc9ac0a6b799d4d5f89d8bb1627f1_16.png)

在剪映等视频编辑软件中，蒙版通常分为预设形状和自定义形状两种。

![](img/eabfc9ac0a6b799d4d5f89d8bb1627f1_18.png)

![](img/eabfc9ac0a6b799d4d5f89d8bb1627f1_20.png)

*   **预设形状蒙版**：软件内置了多种固定形状，方便用户快速调用。
*   **自定义形状蒙版**：在一些更专业的软件中，用户可以自由绘制或选择复杂形状来创建蒙版效果。

![](img/eabfc9ac0a6b799d4d5f89d8bb1627f1_22.png)

![](img/eabfc9ac0a6b799d4d5f89d8bb1627f1_24.png)

剪映软件为用户提供了七种默认的预设蒙版形状。

![](img/eabfc9ac0a6b799d4d5f89d8bb1627f1_26.png)

## 如何在剪映中添加与调整蒙版

![](img/eabfc9ac0a6b799d4d5f89d8bb1627f1_28.png)

以下是添加和调整蒙版的具体操作步骤。

![](img/eabfc9ac0a6b799d4d5f89d8bb1627f1_30.png)

![](img/eabfc9ac0a6b799d4d5f89d8bb1627f1_32.png)

1.  **选中素材**：在时间轴上，点击你想要添加蒙版的视频或图片素材。
2.  **打开蒙版面板**：在右侧的参数调节面板中，找到并点击“蒙版”选项。
3.  **选择蒙版形状**：在弹出的蒙版列表中，选择你想要的形状，例如：线形、镜面、圆形、矩形、爱心或星星。
    *   选择“线形”蒙版后，画面会以一条线为界分成两部分，类似于分屏效果。
4.  **调整蒙版属性**：添加蒙版后，你可以对它的各项属性进行精细调整。
    *   **位置**：可以直接在预览窗口中拖拽蒙版来移动它的位置，也可以在参数面板中输入精确的坐标数值。
    *   **旋转**：可以输入旋转角度，或直接拖拽蒙版上的旋转控制点。
    *   **大小**：可以通过拖拽蒙版边缘的白色控制点来改变大小，也可以在参数面板中输入数值或使用微调按钮。
    *   **羽化**：羽化功能可以使蒙版边缘产生柔和的渐变透明效果，让蒙版与背景的过渡更自然。拖动羽化滑块即可调整羽化程度。
5.  **应用动画与效果**：你还可以为蒙版本身添加入场、出场等动画效果，或者对应用了蒙版的素材进行调色等进一步处理。

![](img/eabfc9ac0a6b799d4d5f89d8bb1627f1_34.png)

![](img/eabfc9ac0a6b799d4d5f89d8bb1627f1_36.png)

## 核心操作要点

![](img/eabfc9ac0a6b799d4d5f89d8bb1627f1_38.png)

为了让初学者更好地掌握，这里总结几个核心操作要点。

![](img/eabfc9ac0a6b799d4d5f89d8bb1627f1_40.png)

![](img/eabfc9ac0a6b799d4d5f89d8bb1627f1_42.png)

*   **蒙版的本质是“形状选择器”**，它从原始素材中“框选”出一个特定形状的区域进行展示。
*   **关键帧**：蒙版的位置、大小、旋转等属性都可以添加关键帧，从而制作出蒙版移动、缩放、旋转的动画效果。
*   **灵活运用羽化**：合理使用羽化值，可以避免蒙版边缘生硬，让合成效果更加真实、有层次感。

![](img/eabfc9ac0a6b799d4d5f89d8bb1627f1_44.png)

本节课中我们一起学习了蒙版功能的基础概念和在剪映中的使用方法。你学会了如何为素材添加各种形状的蒙版，并调整其位置、大小、羽化等属性。掌握蒙版技巧，能让你轻松实现画面分割、局部突出、创意转场等效果，是提升视频专业度的关键一步。
