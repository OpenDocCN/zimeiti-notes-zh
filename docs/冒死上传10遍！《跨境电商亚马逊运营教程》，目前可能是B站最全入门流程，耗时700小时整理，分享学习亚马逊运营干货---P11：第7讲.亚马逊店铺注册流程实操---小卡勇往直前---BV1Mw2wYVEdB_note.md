上一节我们介绍了注册前的准备工作，本节中我们来看看如何使用模拟器进行实际操作。

### 访问注册模拟器

![](img/d237f8ec37e63dcc66f3cc2af2492ca3_7.png)

首先，我们需要访问亚马逊店铺注册模拟器。在搜索引擎中搜索“亚马逊店铺注册模拟”，即可找到相关工具。注册流程在全球各站点基本一致，本教程以美国站为例进行演示。

![](img/d237f8ec37e63dcc66f3cc2af2492ca3_9.png)

![](img/d237f8ec37e63dcc66f3cc2af2492ca3_11.png)

**注意**：模拟注册时，切勿使用真实个人信息，否则可能导致与现有或未来账号产生关联风险。

![](img/d237f8ec37e63dcc66f3cc2af2492ca3_13.png)

### 创建账户与基础信息填写

以下是创建亚马逊卖家账户的第一步，需要填写姓名、邮箱和密码。

1.  **姓名**：填写模拟提供的示例姓名，如“田丽琴”。
2.  **邮箱地址**：使用模拟器提供的虚拟邮箱地址。
3.  **密码**：设置一个密码，例如 `123`。
4.  点击“下一步”后，系统会向邮箱发送验证码。在模拟页面右下角查看并输入验证码（如 `231416`）以完成邮箱验证。

完成邮箱验证后，系统会要求绑定手机号。

### 绑定手机与公司信息录入

接下来，需要添加中国大陆手机号以接收短信验证码。验证成功后，便进入正式的店铺注册页面。

以下是填写公司详细信息的步骤。

![](img/d237f8ec37e63dcc66f3cc2af2492ca3_15.png)

![](img/d237f8ec37e63dcc66f3cc2af2492ca3_17.png)

1.  **营业执照国家**：选择“中国”。
2.  **主体类型**：选择“私有企业”。
3.  **公司名称**：填写公司全称，例如“杭州博辰网络科技有限公司”。注意英文名称需全部小写且无空格，如 `hangzhoubochenwangluokejiyouxiangongsi`。
4.  **统一社会信用代码**：准确填写营业执照上的18位代码，例如 `9111010117822640812P`。
5.  **公司地址与邮编**：填写营业执照上的注册地址。可通过网络查询对应邮编，例如杭州市江干区的邮编为 `310016`。

**关键点**：此步骤信息提交后无法修改，务必仔细核对，确保与营业执照完全一致。

### 法人身份信息与居住地址

完成公司信息填写后，需要验证法人身份。

1.  **法人姓名**：填写法人姓名，如“田丽琴”。
2.  **国籍与出生信息**：根据身份证信息选择国籍、出生地及出生年月日。
3.  **身份证号码与有效期**：准确填写身份证号码及有效期截止日期。
4.  **居住地址**：此地址至关重要。建议填写法人身份证地址或信用卡账单地址。若账号触发审核（如二审或QIC审核），提供与此地址一致的账单有助于提高通过率。

### 企业受益人确认与收款设置

接下来需要确认企业受益人和设置收款方式。

1.  **企业受益人**：根据公司股权结构如实勾选。若为个人独资，可同时勾选“企业受益所有人”和“企业法人代表”。若有持股超过25%的其他股东，则需额外添加其信息（注意：这可能导致账号关联风险）。
2.  **收款账户**：选择已准备好的收款工具（如连连支付），并填写账户持有人姓名及9位数的识别代码。这些信息可在收款平台后台或联系客服获取。

### 信用卡信息与店铺设置

店铺运营需要绑定一张国际信用卡。

1.  **信用卡资料**：填写信用卡卡号、持卡人姓名（拼音）、有效期（如 `09/2024`）。
2.  **账单地址**：将收费地址修改为信用卡账单地址，并确保与此前填写的“居住地址”策略保持一致。

完成信用卡绑定后，进行店铺基础设置。

1.  **店铺名称**：可先设定一个名称，店铺注册成功后仍可修改。
2.  **UPC与品牌问题**：若无产品编码和品牌备案，对于“是否拥有UPC编码”和“是否是品牌所有者”等问题，通常选择“否”。

### 文件上传与视频认证

这是注册流程的最后关键步骤。

![](img/d237f8ec37e63dcc66f3cc2af2492ca3_19.png)

以下是需要准备并上传的审核文件清单。

1.  **法人身份证**：正反面高清照片，需完整显示四个边角。
2.  **营业执照**：高清扫描件或照片。

![](img/d237f8ec37e63dcc66f3cc2af2492ca3_21.png)

上传文件后，仔细复核所有已填信息，确认无误后点击提交。系统将提示进行视频认证。

视频认证是验证法人身份的重要环节，以下是注意事项。

1.  **预约时间**：在系统提供的时段内选择预约时间。
2.  **设备准备**：确保电脑或手机的摄像头、麦克风功能正常。
3.  **材料准备**：法人需亲自到场，并准备好营业执照原件。
4.  **网络环境**：**强烈建议使用手机4G/5G流量进行视频通话，避免使用Wi-Fi，以防IP关联风险。**
5.  **认证过程**：通话时长约5-15分钟，亚马逊工作人员会询问简单问题以确认法人身份和经营意图。

视频认证结束后，保存好注册用的邮箱、密码及预约页面网址。等待审核结果通知即可。
