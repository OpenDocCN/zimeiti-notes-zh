# 抖音快速起号教程：P6：人物设定 🎭

![](img/10054b12bd65d7cdada4366ba58145ed_0.png)

在本节课中，我们将要学习抖音账号运营中至关重要的一环——人物设定。一个清晰、真实且富有吸引力的人设，是解决粉丝信任、实现裂变传播和高效变现的基础。

![](img/10054b12bd65d7cdada4366ba58145ed_2.png)

上一节我们介绍了如何测试爆款内容，本节中我们来看看如何为你的账号注入灵魂，即策划一个成功的人物设定。

![](img/10054b12bd65d7cdada4366ba58145ed_4.png)

## 什么是人设策划？

![](img/10054b12bd65d7cdada4366ba58145ed_6.png)

在抖音中做人设，就像在游戏中选角色一样，需要提前对人物角色进行设定。例如，设定其称呼、职业、拥有的技能点等。

![](img/10054b12bd65d7cdada4366ba58145ed_8.png)

但与游戏角色所不同的是，我们在抖音中设定的人物角色是“人生角色”，所以需要更具有真实性。

![](img/10054b12bd65d7cdada4366ba58145ed_10.png)

## 人设的重要性

![](img/10054b12bd65d7cdada4366ba58145ed_12.png)

不管是短视频还是直播，一个成功的IP能够解决粉丝信任的问题。它能够起到裂变传播的作用，当然也具备很强的变现能力。

![](img/10054b12bd65d7cdada4366ba58145ed_14.png)

所以我们做账号，不管是创作内容还是策划选题，目的就是在获取流量的同时打造IP。打造IP的核心，就是在一定的场景或者情境中，进行账号人格化的过程。

![](img/10054b12bd65d7cdada4366ba58145ed_16.png)

## 如何策划账号人设？

![](img/10054b12bd65d7cdada4366ba58145ed_18.png)

账号人格化，也就是我们通常所说的账号人设。对于账号人设的策划，我们可以根据下图从以下几个方面着手。

![](img/10054b12bd65d7cdada4366ba58145ed_0.png)

以下是策划人设时需要明确的六个核心方面：

1.  **名字/昵称**：一个IP首先得有一个叫法，这是IP传播的“名片”。所以要明确这个IP叫什么，就像一个品牌的品牌名或商标名一样重要。
2.  **职业/身份**：人格化必然要具备身份。这是用户认识这个人或者这个IP的联想入口，比如老师、教授、博士、律师等。
3.  **称呼/社会关系**：一个IP有了称呼，能快速拉近与用户的距离，起到“移情”的作用。如“XX哥”、“XX大爷”、“XX姥姥”。
4.  **形象/第一印象**：以用户的视角来看待IP，使IP形象化更容易占据用户心智。如大叔形象、萌萌哒、铁憨憨、甜美可爱、光头等。
5.  **性格/爱好/特点**：一个具有鲜明特点的IP，除了具有很强的记忆点之外，还具有很强的真实性。太完美的形象容易脱离大众。如暴躁、娘炮、爱吃、爱喝酒等。
6.  **穿着打扮/财富地位**：这个是属于社会的刻板印象，大众的普遍认知。借助用户的固有思维为IP，快速在用户心智中定位。比如有钱代表会赚钱，开豪车代表成功人士，打扮土代表着老实等。

## 从人设到账号搭建

以上内容策划好之后，就可以结合你自身的产品优势、服务优势等，将其实施到账号搭建中，即抖音账号的“五件套”：**头像、昵称、背景图、简介、视频封面**。

## 提炼你的“超级符号”

除了对账号IP进行人格化的策划之外，还要通过选题策划、内容策划，为您的人设提炼出记忆点，即通常大家所说的“超级符号”。

以下是提炼超级符号可以着手的几个方向：

*   **画风（视觉）**：如场景、道具、服饰等。
*   **听觉**：如BGM、特效音、配音、声线特点等。
*   **文案**：能够反应优势且容易记忆的提炼词，如“XX简单粗暴”。
*   **表情/情绪**：如笑、惊恐、无奈等表情包。
*   **原特色**：如方言、口音、错发音、网络语、慢条斯理、抑扬顿挫等。
*   **动作**：习惯性的手/肢体动作，比如撩头发、扶眼镜等。

超级符号提炼好之后，在你进行短视频创作时，就可以不断地在视频中呈现，从而强化用户记忆，形成独特的品牌标识。

本节课中我们一起学习了如何为抖音账号策划一个成功的人物设定。我们从理解人设的重要性开始，逐步拆解了策划人设的六个维度，并学习了如何将人设落地到账号搭建和内容创作中，最终通过提炼“超级符号”来强化IP记忆点。记住，一个真实、鲜明且一致的人设，是你账号在抖音海洋中脱颖而出的关键。
