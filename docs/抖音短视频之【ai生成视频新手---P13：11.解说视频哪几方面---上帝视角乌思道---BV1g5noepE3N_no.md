# AI短视频创作教程：P13：11.解说视频的组成要素

![](img/2f5a270b4bb710f3555d8f05466b0cae_1.png)

在本节课中，我们将要学习解说类视频的核心构成要素。了解这些基础组成部分，是后续进行AI自动化创作的关键第一步。

上一节我们探讨了AI视频创作的潜力，本节中我们来看看一个标准的解说视频具体由哪几方面组成。

![](img/2f5a270b4bb710f3555d8f05466b0cae_3.png)

首先，我们需要明确什么样的视频属于解说类视频。请看下图示例：

![](img/2f5a270b4bb710f3555d8f05466b0cae_1.png)

这个视频是典型的解说类视频。它展示了“胡子哥的胜利来之不易”等内容，并配有“比赛竞技大家请勿模仿，如果给你一个机会，你愿意参加这种比赛吗”等文案。这类视频通常围绕一个核心事件或内容展开叙述和评论。

通过观察，我们可以总结出解说类视频由三个核心方面组成。以下是其具体构成：

1.  **画面/视频**：这是视频的视觉主体，可以是影视剪辑、综艺片段、实拍画面或动画等。
2.  **文案**：指视频中讲述或评论的文本内容，它决定了视频的核心信息和观点走向。
3.  **配音**：将文案转化为语音，为画面提供声音解说，是连接画面与观众理解的关键桥梁。

这三者共同作用，构成了一个完整的解说视频。例如，下图所示的国外综艺解说视频，就是运用了国外综艺的画面，搭配原创的解说文案和配音制作而成。

![](img/2f5a270b4bb710f3555d8f05466b0cae_3.png)

这类以国外开放、猎奇综艺为素材的视频，因其内容抓人眼球，在平台上颇受欢迎。对于想尝试此类题材的创作者，需要注意素材来源的合规性。

我们刚才讲到了解说视频由画面、文案、配音三方面组成。对于不擅长或不想自己撰写文案的创作者，寻找合适的素材是一个解决方案。

说到找素材，这里有一个关键问题：我们本身制作的是视频解说，能否直接借用其他视频的文案来使用？

**答案是不能。** 直接使用其他视频的文案属于“洗稿”行为，存在内容重复和版权风险。平台的内容审核机制会将视频与视频进行比对。

因此，我们需要转换思路，采用逆向思维来寻找素材。既然我们做的是视频，就不要从视频中找文案。我们可以从其他形式的内容中获取灵感。

具体而言，我们可以去寻找**文章**或**音频**作为文案素材来源。因为大多数内容平台中，文章、音频和视频是分开审核的，它们之间的内容重复度检查相对独立。这是一个非常实用的小技巧。

这个技巧源于实践。早期运营自媒体时，我曾通过提取喜马拉雅、企鹅FM等音频平台的音频内容，并将其转写成文字，来作为文章发布的素材。这种方法有效提升了内容产出数量，且避免了重复度问题。

将这种思路迁移到视频创作上同样有效。通过将文章或音频内容转化为视频解说文案，可以高效地批量生产原创度较高的视频内容。如下图所示，通过多账号、堆数量的方式，可以获得不错的收益。

本节课中我们一起学习了解说视频的三个基本组成要素：**画面**、**文案**和**配音**。同时，掌握了一个重要的素材寻找原则：通过逆向思维，从**文章或音频**中获取文案灵感，而非直接复制其他视频的内容。这是实现高效、合规创作的基础。下一节，我们将开始学习如何利用AI工具来具体生成这些要素。
