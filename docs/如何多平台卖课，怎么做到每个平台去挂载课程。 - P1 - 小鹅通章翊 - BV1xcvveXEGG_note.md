# 多平台卖课指南：P1：小鹅通核心功能解析 🎯

在本节课中，我们将要学习如何利用“小鹅通”这一工具，实现课程在多平台的挂载与销售。我们将详细拆解小鹅通的核心功能，了解它如何帮助老师解决线上教学与售卖的关键问题。

---

小鹅通是一个集多平台卖课、私域直播与私域运营于一体的综合解决方案。它旨在帮助老师简化线上课程的分发与管理流程。

## 多平台课程挂载与售卖 📈

上一节我们概述了小鹅通的核心定位，本节中我们来看看其实现多平台卖课的具体方式。

线上卖课的一个关键环节是能够在各大平台挂载并销售课程。小鹅通支持在短视频下方或直播间内直接挂载课程链接，从而实现课程在多平台的曝光与销售。

## 丰富的课程形式支持 🎬

了解了销售渠道，接下来我们看看小鹅通支持哪些课程形式来满足不同的教学需求。

小鹅通为老师提供了多样化的课程形式选择。以下是主要的课程类型：

*   **录播课**：预先录制好的视频课程。
*   **私域直播**：在自有平台或社群内进行的直播教学。
*   **训练营**：带有社群服务与督学环节的系列课程。
*   **随堂检测**：在课程中嵌入的测验题目。
*   **AI互动课**：融入人工智能技术的交互式课程。

这些形式全面覆盖了老师制作与售卖课程的需求。

## 零抽成政策与学员管理 👥

除了课程形式，平台的收费政策和学员管理能力也是老师关心的重点。

特别值得一提的是，小鹅通**不对老师售卖的课程进行抽成**，所有收入归老师所有。这为老师提供了重要的利润保障。

同时，小鹅通具备强大的学员管理能力。系统会详细记录每位学员的消费记录、学习进度、完成任务数以及个人标签等信息。这些数据至关重要，能帮助老师更好地进行学员分层、制定学习跟进策略以及规划课程复购方案。

## 其他功能与个性化需求 💡

我们已经介绍了小鹅通在卖课、课程形式和学员管理方面的核心优势。

除了上述高频功能，小鹅通还拥有许多其他特色功能，等待老师们亲自探索。虽然每位老师的需求都是独特的，但小鹅通的设计足以满足大多数老师的线上教学与运营需求。
