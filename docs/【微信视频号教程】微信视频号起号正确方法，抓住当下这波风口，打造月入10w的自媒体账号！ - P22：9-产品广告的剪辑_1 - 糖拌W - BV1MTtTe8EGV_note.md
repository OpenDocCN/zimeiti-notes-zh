# 微信视频号教程：P22：9-产品广告的剪辑_1

## 概述
在本节课中，我们将学习如何为产品广告剪辑视频素材。我们将从寻找素材开始，逐步完成下载、筛选和初步剪辑，最终拼接成一个完整的广告视频。整个过程旨在为后续添加广告内容做好准备。

## 寻找与下载素材
上一节我们介绍了广告视频的构思，本节中我们来看看如何获取具体的剪辑素材。首先需要在抖音等平台寻找与产品相关的口播视频。

1.  打开抖音应用。
2.  在搜索框输入产品关键词，例如“红糖红枣黑芝麻糕口播”。
3.  在搜索结果中浏览，找到一个合适的博主主页。
4.  进入该博主主页，挑选多个（3~5个）不同角度（如小孩吃、老人吃、制作过程）的视频。

以下是下载素材的步骤：

1.  点开选中的视频，点击“复制链接”。
2.  打开“创作猫”等去水印工具。
3.  在工具内选择“去水印”功能，粘贴复制的链接。
4.  点击“免费去水印”并“保存视频”到本地。
5.  重复以上步骤，下载3~5个视频素材。

## 素材检查与初步剪辑
素材下载完成后，在开始正式剪辑前，必须对每个视频进行检查，确保内容符合要求。

打开剪映，开始创作，导入下载好的所有视频素材。

在剪辑前，需要完整播放每个视频，仔细检查以下内容：
*   **价格信息**：视频中提及的产品价格（如“九块九三包”）是否与你计划销售的价格一致。如果不一致，需要将相关片段剪掉。
*   **引导话术**：视频中是否有“点击下方小黄车”等引导至其他购物渠道的语句。如有，需要将这部分分割并删除。

如果视频中存在上述问题，处理方法是：使用剪辑软件的分割功能，将问题片段删除。对于新手，如果视频问题较多，建议直接更换素材，重新寻找和下载。

完成检查和初步修剪后，将处理好的视频片段按逻辑顺序拼接起来，即可导出一个初步成型的广告视频。

## 总结
本节课中我们一起学习了产品广告视频的素材准备与初步剪辑流程。我们掌握了从抖音寻找口播素材、使用工具下载去水印视频、以及导入剪映进行内容检查和基础剪辑的方法。关键点在于确保素材中**没有矛盾的价格信息和外部购物引导**。这样，我们就得到了一个干净的、可用于后续加工的广告视频雏形。下一节，我们将在此基础上，学习如何为这些视频润色并巧妙地加入广告信息。
