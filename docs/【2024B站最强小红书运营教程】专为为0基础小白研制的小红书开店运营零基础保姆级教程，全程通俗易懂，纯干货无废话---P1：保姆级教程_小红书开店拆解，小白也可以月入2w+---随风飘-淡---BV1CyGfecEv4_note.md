## 开店流程 📝
上一节我们概述了课程内容，本节中我们来看看如何迈出第一步：开设你的小红书店铺。

![](img/6a2b74da905e7a07b880809267ef1951_4.png)

开店流程本身非常简单，遵循官方指引即可完成。

以下是具体的操作步骤：
1.  打开小红书App。
2.  点击左上角的菜单图标。
3.  选择“创作中心”。
4.  点击“更多服务”选项。
5.  选择“店铺”，点击“立即入住”。

你只需按要求提交资料即可完成入住。

![](img/6a2b74da905e7a07b880809267ef1951_6.png)

### 店铺类型选择
小红书店铺主要分为**个人店**和**企业店**。

![](img/6a2b74da905e7a07b880809267ef1951_8.png)

对于新手，我们建议从**个人店**开始。个人店有两种形式：
*   **个人店**：仅需身份证和**1000元**保证金。
*   **个体工商户店**：需要身份证、营业执照和保证金。

![](img/6a2b74da905e7a07b880809267ef1951_10.png)

企业店则需要营业执照，并需缴纳**600元**年费。根据我们的实测，平台目前并未给企业店明显的流量倾斜。个人店与企业店的主要区别在于，一个身份证只能开一家个人店，而一份营业执照可以开设三家店铺。

![](img/6a2b74da905e7a07b880809267ef1951_12.png)

**核心建议**：初期运营，选择个人店足矣。在单店尚未运营成功时，开设多个店铺并无意义。

### 运营成本分析
了解成本是开始任何项目的前提。小红书电商的主要成本构成如下：
*   **保证金**：1000元（关店可退）。
*   **平台扣点**：约为营业额的 **5%**。
*   **支付手续费**：每单约 **0.7%**。
*   **流动资金**：初期可采用无货源模式，准备2000-3000元即可启动。

## 选品策略 🛍️
成功开店后，下一步就是决定“卖什么”。选品是电商成功的关键，但方法可以很直接。

![](img/6a2b74da905e7a07b880809267ef1951_14.png)

核心逻辑是：**利用市场验证来选品，而非凭空猜测**。其他电商平台的爆款产品，有很大概率在小红书也能成功。

以下是三个高效的选品渠道：
1.  **淘宝/京东**：寻找符合小红书平台“调性”（如高颜值、新奇特、生活方式类）的爆款产品。身边有电商朋友将淘宝爆款搬到小红书，第二篇笔记单量即破百。
2.  **抖音精选联盟**：查看抖音精选联盟的排行榜，了解当前热卖商品。筛选出符合小红书用户喜好的产品，直接进行测试。
3.  **拼多多销量榜**：在拼多多的类目销量排行榜中，选择适合小红书的品类，搬运其中的热销品。

**选品公式**可以概括为：
`成功选品 = 其他平台已验证的爆款 × 与小红书用户画像的匹配度`

你只需要判断排行榜上的商品是否可能受到小红书用户喜欢，然后搬运测试即可。这相当于借助了大卖家的选品能力，是最快捷稳妥的方式。

## 流量获取与内容创作 📈
选品上架后，就进入了最核心的环节：获取流量。小红书本质是“种草”社区，店铺超过90%的流量都来源于发布的笔记。

因此，与其铺多个店铺（店群），不如专注于**创建账号矩阵，大量发布优质笔记**来获取流量。用户路径是：看到笔记 -> 对产品产生兴趣 -> 进入店铺购买。

### 如何创作有效的种草笔记
直接发布硬广效果很差，会破坏信任感。正确的做法是**以普通用户（素人）的视角进行分享和种草**。

创作高质量种草笔记的方法很简单：**借鉴各大平台的买家秀和用户评价**。

具体操作步骤如下：
1.  **寻找素材**：去淘宝、京东、抖音等平台的商品评论区，寻找好看、真实的买家秀图片或视频。
2.  **改编文案**：基于买家的真实使用评价进行修改和润色。这些评价本身就是最真实的使用体验，非常适合作为种草文案。

**代码式总结**这个流程：
```plaintext
有效笔记 = 真实买家秀素材 + 用户评价改编的文案
```

当你的单个账号通过这套方法（SOP）跑通并稳定出单后，就可以通过**批量注册账号、批量发布笔记**的矩阵模式来放大流量，实现增长。
