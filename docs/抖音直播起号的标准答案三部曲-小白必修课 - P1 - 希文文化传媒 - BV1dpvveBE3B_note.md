在本节课中，我们将学习抖音直播起号的核心逻辑与前期准备工作。我们将拆解直播流量的两大来源，明确起号阶段需要关注的关键数据，并介绍必要的设备与后台操作。

![](img/0b0cc440d5d7ff084e25e62662e43bbc_11.png)

直播流量主要分为两个部分：**人气流量**和**电商流量**。人气流量主要通过浅层数据（如停留、互动、关注、点赞）来提升。电商流量则依赖于精准的人群标签和良好的电商数据表现。

电商数据的核心指标包括：
*   **GMV**：商品交易总额。
*   **成交密度**：单位时间内的成交订单数。
*   **千次观看成交金额**：每千次观看产生的成交额。
*   **人群标签精准度**：直播间吸引的用户是否符合目标客群。

## 设备与后台准备 💻

![](img/0b0cc440d5d7ff084e25e62662e43bbc_13.png)

![](img/0b0cc440d5d7ff084e25e62662e43bbc_15.png)

以下是直播前需要准备的设备与后台查看方法。

*   **直播设备**：电脑或手机均可用于直播。
*   **后台查看**：需要学会使用巨量百应的“直播中控台”。
    *   **专业屏**：用于详细分析流量来源。路径为：直播中控台 -> 点击“大屏监控” -> 进入专业屏 -> 点击“流量分析”。其中，**黄线代表直播推荐流量**，蓝线通常代表付费流量。
    *   **基础屏**：用于监控实时核心数据面板，如下单人数、成交金额等。

## “鱼”资源的注意事项 🐟

![](img/0b0cc440d5d7ff084e25e62662e43bbc_21.png)

![](img/0b0cc440d5d7ff084e25e62662e43bbc_23.png)

“鱼”（即用于辅助数据操作的账号）的使用非常关键，以下是几个核心要点。

*   **网络要求**：“鱼”在执行任务时必须使用手机流量（三大运营商），禁止使用物联网卡。
*   **账号状态**：“鱼”账号在任务前4-5天内不能有切换账号的行为，且需要保持正常的日常活跃度。
*   **养号流程**：在正式任务前，需要让“鱼”提前2天做准备工作，包括：
    *   观看同行品类直播间。
    *   点击商品、查看详情页。
    *   在商城搜索目标商品。
    *   进行加购、收藏等行为。
*   **人群限制**：必须根据商品属性限制“鱼”的性别和年龄标签。例如，卖女装必须使用女性标签的账号。

## 常见断流原因排查 🔍

当直播间出现流量中断或下滑时，需要按照以下顺序逐一排查问题。

![](img/0b0cc440d5d7ff084e25e62662e43bbc_25.png)

![](img/0b0cc440d5d7ff084e25e62662e43bbc_27.png)

1.  **店铺与商品问题**：
    *   所带商品的商家店铺评分过低或有严重违规。
    *   直播账号关联的主店铺（如果是子账号）有违规或低分。
    *   商家存在大量投诉、差评，或“挂羊头卖狗肉”（货不对板）的情况。
    *   商品成分造假、缺少质检报告等。
    *   店铺本身权重低，起号困难。
2.  **直播账号问题**：
    *   账号有效粉丝不足，存在大量互关粉或天窗粉（非真人粉）。
    *   直播前15天内有切换账号的行为。
    *   账号活跃度不足。
3.  **直播设备与环境问题**：
    *   设备（手机/电脑）因历史违规被平台监控。
    *   网络环境关联：必须遵循“一机一卡一IP”原则，避免同一网络下多账号操作。MAC地址被记录也可能导致问题。
4.  **直播过程违规**：
    *   **话术违规**：如虚假宣传导致信用分扣除。
    *   **画面违规**：内容低质、暴露或疑似违规。
    *   **直播间素材/摄像头违规**。
    *   注意：一场直播内收到两次低质弹窗警告，流量会严重下滑。
5.  **账号数据长期不佳**：
    *   流量结构不健康。健康的结构通常是 **“721”** 或 **“631”**，即直播推荐流量占60-70%，付费流量占20-30%，其他来源（搜索、关注等）占10%。
    *   口碑分持续下降。
    *   信用分被扣除。
