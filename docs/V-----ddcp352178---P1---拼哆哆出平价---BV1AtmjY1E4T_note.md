# 机器学习入门：1：线性回归基础

## 概述
在本节课中，我们将要学习机器学习中最基础且重要的算法之一：线性回归。我们将了解它的核心思想、数学原理以及如何用简单的代码实现它。

线性回归的目标是找到一条直线（或一个超平面），使得这条直线能够最好地“拟合”我们已有的数据点。这里的“最好”通常意味着所有数据点到这条直线的垂直距离（即误差）的平方和最小。

## 核心概念：假设函数与损失函数

上一节我们介绍了线性回归的目标，本节中我们来看看如何用数学语言来描述它。这涉及到两个核心概念：假设函数和损失函数。

**假设函数** 是我们用来预测的模型。对于一元线性回归，它是一条直线的方程：
`hθ(x) = θ₀ + θ₁x`
其中，`θ₀` 是截距，`θ₁` 是斜率，`x` 是输入特征。

**损失函数** 用于衡量我们的预测值与真实值之间的差距。最常用的损失函数是均方误差：
`J(θ₀, θ₁) = (1/2m) * Σ (hθ(x⁽ⁱ⁾) - y⁽ⁱ⁾)²`
其中，`m` 是样本数量，`hθ(x⁽ⁱ⁾)` 是对第 `i` 个样本的预测值，`y⁽ⁱ⁾` 是第 `i` 个样本的真实值。我们的目标就是找到一组参数 `(θ₀, θ₁)`，使得损失函数 `J` 的值最小。

## 参数求解：梯度下降法

知道了目标是最小化损失函数，我们该如何找到那组最优的参数呢？一个广泛应用的方法是梯度下降法。

梯度下降法的思想非常直观：想象你站在一座山上，想要最快地下到山脚（即损失函数的最小值点）。你会环顾四周，选择最陡峭的下坡方向迈出一步，然后重复这个过程，直到到达谷底。

在数学上，我们通过不断迭代更新参数来实现这个过程。以下是参数更新的公式：
`θⱼ := θⱼ - α * (∂J(θ)/∂θⱼ)`
其中，`α` 称为学习率，它控制着我们每一步迈出的距离。`∂J(θ)/∂θⱼ` 是损失函数 `J` 对参数 `θⱼ` 的偏导数，它指明了当前点的最陡下降方向。

对于我们的线性回归模型，偏导数的具体计算如下：
`∂J/∂θ₀ = (1/m) * Σ (hθ(x⁽ⁱ⁾) - y⁽ⁱ⁾)`
`∂J/∂θ₁ = (1/m) * Σ (hθ(x⁽ⁱ⁾) - y⁽ⁱ⁾) * x⁽ⁱ⁾`

## 实现步骤

理解了原理之后，我们可以将线性回归的实现分解为几个清晰的步骤。以下是使用梯度下降法训练线性回归模型的主要流程：

1.  **初始化参数**：通常将参数 `θ₀` 和 `θ₁` 初始化为0。
2.  **计算预测值**：使用当前的参数和假设函数计算所有样本的预测值 `hθ(x)`。
3.  **计算损失**：根据损失函数公式计算当前参数下的总误差。
4.  **计算梯度**：计算损失函数关于每个参数的偏导数（即梯度）。
5.  **更新参数**：按照梯度下降公式，用梯度和学习率更新参数。
6.  **重复迭代**：重复步骤2到5，直到损失函数的值收敛（变化非常小）或达到预设的迭代次数。

## 代码示例

让我们通过一个简单的Python代码片段，将上述步骤具体化。这里我们使用NumPy库来进行数值计算。

```python
import numpy as np

# 生成一些模拟数据
np.random.seed(0)
m = 100  # 样本数量
X = 2 * np.random.rand(m, 1)  # 特征
y = 4 + 3 * X + np.random.randn(m, 1)  # 标签，包含一些噪声

# 初始化参数
theta = np.random.randn(2, 1)  # [theta0, theta1]
learning_rate = 0.1
n_iterations = 1000

# 为特征矩阵X添加一列1，以便与theta0相乘
X_b = np.c_[np.ones((m, 1)), X]

# 梯度下降迭代
for iteration in range(n_iterations):
    # 计算梯度
    gradients = (2/m) * X_b.T.dot(X_b.dot(theta) - y)
    # 更新参数
    theta = theta - learning_rate * gradients

print(f"训练得到的参数：截距 theta0 = {theta[0][0]:.3f}, 斜率 theta1 = {theta[1][0]:.3f}")
```

这段代码首先生成了一些线性数据，然后通过梯度下降法迭代更新参数 `theta`，最终输出学习到的直线参数。

## 总结
本节课中我们一起学习了线性回归的基础知识。我们首先了解了线性回归的目标是拟合数据直线。然后，我们引入了**假设函数** `hθ(x) = θ₀ + θ₁x` 和**损失函数**（均方误差）来形式化这个问题。接着，我们学习了如何使用**梯度下降法**及其更新公式 `θⱼ := θⱼ - α * (∂J(θ)/∂θⱼ)` 来寻找最优参数。最后，我们梳理了实现的步骤，并通过一个简单的代码示例进行了演示。线性回归是通往更复杂模型的基石，理解它对于后续的学习至关重要。
