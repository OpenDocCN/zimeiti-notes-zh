# 抖音运营速成指南：P6：4.用户定位 🎯

在本节课中，我们将要学习抖音运营中至关重要的一环：用户定位。理解并精准定位你的目标用户，是创作出受欢迎内容、实现账号增长和商业变现的基础。

## 用户定位的核心逻辑

用户定位的核心是思考：我们准备给什么样的人，发什么样的作品。要理解这一点，我们需要先了解抖音的内容推荐机制。

首先，创作者发布内容。抖音平台会综合分析账号和作品，并结合用户在浏览时表现出的兴趣相关性，将视频推送给可能感兴趣的用户。用户在刷到视频后，会根据观看体验产生一系列行为反馈，例如**观看、点赞、关注、评论或转发**。

![](img/8e45bcb7ba4591d430958c26b47c5f48_1.png)

抖音算法会依据这些行为反馈，决定是否将视频推送给更多相似的用户。这就是一个视频能够成为热门视频的基本逻辑。因此，想要创作出热门视频，首先必须对你的目标用户进行深入分析。

当然，我们运营抖音的终极目的是为了变现，而非单纯追求视频热度。但视频播放量是变现的前提。因此，在分析目标用户时，必须将核心用户的商业价值考虑在内。你需要针对核心用户，将视频内容的价值进行拆解，并在不同维度上进行精细化的策划与设计。这样才能最大限度地保证，你发布的视频所吸引到的核心人群是精准的。

## 抖音平台用户现状

在开始定位前，我们先了解一下抖音平台用户的整体特征。抖音日活跃用户已突破7亿，人均每日使用时长超过2小时。这意味着在任何时间、任何地点、任何行业，都有海量用户在抖音上活跃。

从整体用户画像来看，抖音用户具备以下特征：
*   **性别**：女性用户占比约55%。
*   **年龄**：18至30岁的用户占比高达71%。
*   **城市分布**：超过60%的用户分布在一、二线城市。
*   **兴趣偏好**：教育、美食、旅游、搞笑和娱乐是总体兴趣偏好的前五名。其中，教育和美食的占比最高。

![](img/8e45bcb7ba4591d430958c26b47c5f48_3.png)

此外，抖音上的年轻群体（特别是18-30岁）的网上购物动机，有91.8%是“因为自己喜欢或需要买给自己”。他们喜爱的人物和形象通常具备以下几种特质：
1.  **愉悦/搞笑型**：能带来快乐和轻松感。
2.  **颜值控**：外形好看、可爱、漂亮。
3.  **正能量型**：具备爱国情怀和社会正能量。
4.  **认真生活型**：展现出认真处事、热爱生活的态度。

因此，在创作视频或打造个人IP时，参考这些内容特质会更容易获得用户青睐。

## 如何定位目标用户

定位目标用户时，可以参考行业竞争对手或对标账号的数据，也可以分析线下或过往的企业消费者数据。对于目标用户画像的基础分析，应从以下几个维度展开：

以下是用户基础画像的分析维度：
*   **性别**：男、女。
*   **年龄**：18岁及以下、18-28岁、28-38岁、38-48岁、48-58岁、58岁及以上。
*   **地域**：一线城市、二线城市、三线城市、四线城市等。
*   **人群特征**：日常白领、熬夜人群、家庭主妇、学生等。
*   **消费水平**：经济型商品购买者、豪华型商品购买者。

对于电商类账号，还需考虑更多维度：

![](img/8e45bcb7ba4591d430958c26b47c5f48_7.png)

以下是电商用户需考虑的附加维度：
*   **购买时间偏好**：上午、中午、下午、晚上。
*   **价格敏感度**：高、中、低。
*   **囤货偏好**：高、中、低。

从兴趣标签来看，抖音平台将视频观看者的兴趣划分为17个大类，包括：金融、汽车、科技、娱乐、服饰、教育、母婴、法律、体育、游戏、美食、生活、家装、美妆、旅游、房产、商务服务。

## 用户的商业价值分层（5A模型）

对于品牌商家而言，抖音采用5A模型对用户进行商业价值分层：

以下是抖音5A用户分层模型：
*   **A1（了解感知）**：即“我知道”阶段。用户被动接收信息，是品牌知名度的主要来源。
*   **A2（吸引兴趣）**：即“我喜欢”阶段。旨在增加用户对品牌的兴趣，创造短期记忆或强化长期记忆。
*   **A3（主动问询）**：即“我相信”阶段。适度引发用户好奇，促使其主动收集品牌信息，是转化率最高、最接近成交的高价值人群。
*   **A4（购买行动）**：即“我购买”阶段。目标是让用户完成购买，并涉及后续的使用和售后服务。
*   **A5（拥护推荐）**：即“我推荐”阶段。用户对品牌有强烈忠诚度，表现为复购、留存和向他人推荐。

此外，还有规模最大的 **O人群（潜在人群）**，指在平台内多个兴趣领域活跃、等待被品牌吸引注意力的用户。

## 进阶用户描述维度

![](img/8e45bcb7ba4591d430958c26b47c5f48_9.png)

除了上述维度，我们还可以从婚姻状况、家庭结构、职业、收入、资产等方面对用户进行更精细的描述。如果某些行业获取这些数据有难度，也可以从用户人格类型入手，这对内容产出具有直接的指导意义。

例如，可以参考九型人格等分析工具：

以下是针对不同人格类型的内容策略举例：
*   **忠诚型人格**：小心谨慎，多疑虑。视频内容应强调**真实感**和**接地气**，更容易获得其信任。
*   **理想主义者**：偏感性。视频内容应多以提供**情绪价值**为主。
*   **艺术家型人格**：乐于接受新体验。可以发布具有**创意**或**挑战体验型**的内容。
*   **学者型人格**：对知识类内容更感兴趣，且互动意愿较高。
