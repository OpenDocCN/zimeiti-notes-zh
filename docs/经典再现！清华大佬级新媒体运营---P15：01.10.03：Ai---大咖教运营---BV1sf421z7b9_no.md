# 新媒体运营：P15：01.10.03：AI：从分析爆款到生成爆款笔记 🧠

在本节课中，我们将学习如何利用AI工具（如ChatGPT）分析平台上的爆款笔记，并基于分析结果，指导AI生成具备爆款潜质的新笔记。我们将通过一个考研领域的实例，完整演示从“喂资料”到“出文案”的全过程。

上一节我们介绍了通过“喂资料”让AI学习特定平台文案风格的方法。虽然上一节的旅游攻略案例效果不理想，但方法是正确的。本节中，我们来看看如何深化这一过程，通过分析具体的爆款内容，让AI学习其成功要素并用于创作。

## 分析爆款笔记的核心要点

![](img/f96a52b96180c53f95839b3c9294bdcf_1.png)

我们的目标是让AI深入理解一篇笔记为何能成为爆款。这需要从以下几个关键维度进行分析：

![](img/f96a52b96180c53f95839b3c9294bdcf_3.png)

![](img/f96a52b96180c53f95839b3c9294bdcf_5.png)

![](img/f96a52b96180c53f95839b3c9294bdcf_7.png)

以下是需要AI重点分析的几个方面：

![](img/f96a52b96180c53f95839b3c9294bdcf_9.png)

![](img/f96a52b96180c53f95839b3c9294bdcf_11.png)

![](img/f96a52b96180c53f95839b3c9294bdcf_13.png)

1.  **爆款因素**：分析内容中有哪些点可能促使其成为爆款（例如：情感共鸣、实用干货、逆袭故事等）。
2.  **整体结构**：拆解文案的叙事逻辑。
    *   **开篇**：用什么方式（如提问、制造焦虑、引用名人）吸引用户继续阅读。
    *   **中间**：提供了哪些内容来戳中用户痛点，提升用户停留时长。
    *   **结尾**：如何设置“钩子”（如鼓励互动、提供联系方式）引导用户私信或关注。
3.  **关键词策略**：分析笔记中是否刻意植入了特定关键词。
    *   如果有，关键词是什么？
    *   关键词的出现频率和分布结构是怎样的？

## 实战演练：分析考研领域爆款笔记

我们选择小红书平台上点赞数较高的考研类笔记作为分析样本。

### 第一步：向AI下达分析指令

![](img/f96a52b96180c53f95839b3c9294bdcf_21.png)

![](img/f96a52b96180c53f95839b3c9294bdcf_23.png)

我们需要给AI一个清晰的分析指令。指令模板如下：

![](img/f96a52b96180c53f95839b3c9294bdcf_25.png)

![](img/f96a52b96180c53f95839b3c9294bdcf_27.png)

```
我给你提供一篇小红书平台上的爆款笔记文案，请你分析：
1. 爆款因素：例如情感共鸣、实用建议等。
2. 整体结构：
   - 开篇用什么吸引用户？
   - 中间用什么内容戳中用户痛点？
   - 结尾用什么方式引导用户互动或关注？
3. 关键词分析：分析文案是否结合中文分词法设计了关键词。如果有，请指出关键词及其出现频率；如果没有，请说明。
```

### 第二步：处理AI的分析结果

![](img/f96a52b96180c53f95839b3c9294bdcf_38.png)

AI会对每篇笔记给出分析。例如，对于一篇以“备考焦虑”开头的笔记，AI可能给出如下分析：

![](img/f96a52b96180c53f95839b3c9294bdcf_40.png)

*   **爆款因素**：情感共鸣（焦虑）、逆袭故事、实用备考方法、使用表情符号增加亲和力、结尾正能量鼓励。
*   **结构分析**：
    *   开篇：用“政治选择题错很多”的焦虑场景吸引考生。
    *   中间：分享英语和专业课的具体备考经验，解决痛点。
    *   结尾：鼓励读者，传递积极信号。
*   **关键词分析**：本篇没有明显刻意设计的关键词，词汇多为直接常用词。

## 指导AI生成爆款笔记大纲

掌握了爆款要素后，下一步是指引AI根据这些要素创作新笔记。首先，从生成大纲开始。

向AI发出指令：
```
我现在需要写一篇关于考研的小红书笔记。请结合你刚才总结的爆款要素，为我规划一个具备爆款潜质的笔记大纲。大纲需要细分到具体的小节点。
```

AI生成的大纲可能如下：
1.  **开篇：吸引注意**
    *   用引人入胜的问题或场景切入。
    *   简要介绍笔记主题。
2.  **个人逆袭故事引入**
    *   简短讲述自己的备考逆袭经历，引发共鸣。
3.  **核心干货分享**
    *   专业选择建议。
    *   总体时间规划（如月度计划）。
    *   政治备考具体方法。
    *   英语备考具体方法。
    *   专业课备考策略。
4.  **时间管理与心态调整**
    *   强调效率的重要性。
    *   提供克服焦虑的建议。
5.  **结尾：互动引导**
    *   用正能量话语鼓励读者。
    *   引导提问、私信或关注。

## 生成完整的爆款笔记文案

![](img/f96a52b96180c53f95839b3c9294bdcf_73.png)

大纲确认无误后，即可让AI根据大纲和爆款要素生成完整文案。

![](img/f96a52b96180c53f95839b3c9294bdcf_75.png)

向AI发出最终指令：
```
请根据上述大纲，撰写一篇完整的小红书笔记文案。要求如下：
1. 结构严格遵循爆款笔记的要素。
2. 包含一个简短的逆袭小故事来引入情绪。
3. 文案中要自然融入“考研”、“备考”等关键词。
4. 使用适量的emoji表情来增加亲和力。
```

**AI生成的文案示例：**
```
当你曾经迷茫，觉得时间不够用，做阅读一篇错三个，是不是很崩溃？我懂那种感觉！😣 但我也亲身经历了从错题连篇到顺利上岸的逆袭。今天，就把我的**考研**全流程攻略分享给你！

**逆袭小故事**：大三下学期，我的**备考**状态一塌糊涂...（此处省略具体故事）

![](img/f96a52b96180c53f95839b3c9294bdcf_77.png)

![](img/f96a52b96180c53f95839b3c9294bdcf_78.png)

**📅 超详细时间规划**
现在（11月）到考前，这样安排：
- 11-12月：政治一轮+英语单词...
- 1-2月：专业课一轮...
（具体计划）

**📚 分科攻克秘籍**
- **政治**：...
- **英语**：...
（具体方法）

最重要的永远是时间管理和心态！合理规划，坚持每日学习，别让焦虑打败你。只要坚持，你也能逆袭！💪

有任何**考研**问题，随时留言问我！一起加油！
```

这篇文案已经具备了爆款笔记的基本框架：情感开头、故事共鸣、干货结构、关键词植入、表情运用和互动结尾。运营人员可以此为基础进行优化和调整。

## 进阶提示：使用定制化AI助手

![](img/f96a52b96180c53f95839b3c9294bdcf_80.png)

上述流程使用通用ChatGPT（如3.5版本）可能遇到遗忘上下文等问题。为了获得更稳定、更符合个人风格的效果，可以利用高级功能（如ChatGPT的GPTs）创建定制助手。

![](img/f96a52b96180c53f95839b3c9294bdcf_82.png)

你可以：
1.  在助手配置中上传多篇优秀的小红书文案样本作为学习资料。
2.  在指令中永久设定好文案风格、常用emoji库、关键词偏好等要求。
3.  打造一个专属于你的“小红书爆款文案生成器”，从而获得质量更高、更稳定的输出。

![](img/f96a52b96180c53f95839b3c9294bdcf_84.png)

本节课中，我们一起学习了如何系统性地训练AI分析爆款笔记，并引导其生成具备爆款潜质的新文案。关键步骤在于：**提供清晰指令、喂养多篇样本、总结爆款要素、先大纲后文案**。通过这个方法，即使是初学者，也能借助AI快速产出符合平台调性的优质内容草稿，极大提升运营效率。
