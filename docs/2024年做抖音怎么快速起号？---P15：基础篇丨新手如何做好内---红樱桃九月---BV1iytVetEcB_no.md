# 抖音运营指南：P15：新手如何做好内容定位 🎯

在本节课中，我们将学习一个高效的内容选题公式，帮助新手创作者快速找到创作方向，解决“不知道发什么”的难题，从而更精准地吸引目标粉丝。

![](img/935b9f90a3989b0403f925ffb4ec5e8c_1.png)

你现在为抖音内容流的泪，都是在确定选题时，脑子里进的水。你是否为想选题想到头秃，不知道明天该发布什么内容？今天教你一套公式，一分钟帮你解决这个苦恼。

会找选题的人，能有更多时间专注于视频制作。每天发布三条内容，一个月就是九十条。依靠堆量都能形成竞争优势，根本不需要投入大量资金推广。而且，擅长选题的人能比你更精准地找到用户的痛点，也更容易吸引精准粉丝驻足观看，从而快速为账号打上标签。

![](img/935b9f90a3989b0403f925ffb4ec5e8c_3.png)

那么，有什么办法可以解决选题难题呢？

![](img/935b9f90a3989b0403f925ffb4ec5e8c_5.png)

当然有。这是一套百万粉丝博主常用的选题技巧。为了方便理解，我将其总结成了一个公式：**内容定位 + 用户痛点 + 借势热点**。

用好这个公式，你的内容方向一定不会跑偏。例如，如果你想和我一样成为一名知识博主，就需要去挖掘行业内的关键词，例如：播放量、粉丝量、定位、选题等。这些是客观关键词，不包含主观描述。

然后，在这个基础上寻找用户的痛点，例如：便捷、高效、技巧教学、内容输出等。最后，去结合近期的热点话题。将这三个部分进行任意组合即可。

接下来，我们把这个公式套用在其他行业里试试。以美妆行业为例，内容定位可以是：美白、除皱、口红。用户痛点则是：平价好用、明星同款、避坑扫雷。最后，借势热点进行排列组合，就能产生诸如“迪丽热巴同款美白面膜推荐”、“七夕千万别送女朋友口红”等具体选题。

![](img/935b9f90a3989b0403f925ffb4ec5e8c_7.png)

以上，我已经举一反二了，至于举一反三就交给你了。你可以根据自己行业的特性，将你的思考过程实践出来。

![](img/935b9f90a3989b0403f925ffb4ec5e8c_9.png)

本节课中，我们一起学习了内容选题的核心公式：**内容定位 + 用户痛点 + 借势热点**。掌握这个公式，你可以系统性地生成大量精准选题，节省思考时间，将精力更多地投入到内容制作中，从而更高效地打造高权重抖音账号。
