# 亚马逊跨境电商入门到精通：P14：2.2、我要开店的运作原理

![](img/69c8d66440dfa452e04ca8a13dbb19ab_0.png)

在本节课中，我们将详细拆解在亚马逊平台开设店铺并运营的核心流程。整个过程可以概括为四个关键步骤：注册账户、上架商品、处理订单与收款。我们将逐一解析每个步骤的细节，帮助你清晰理解亚马逊店铺的运作原理。

## 🏢 第一步：创建亚马逊卖家中心账户

![](img/69c8d66440dfa452e04ca8a13dbb19ab_2.png)

在开始销售之前，你需要在亚马逊卖家中心注册账户。注册前，你需要根据自身业务情况，在“个人销售计划”和“职业销售计划”之间做出选择。

![](img/69c8d66440dfa452e04ca8a13dbb19ab_4.png)

以下是两种计划的核心区别对比：

*   **职业销售计划**：每月需支付 **$39.99** 的固定订阅费。无论销售多少商品，此费用不变。此计划适合计划大量销售商品的卖家。
*   **个人销售计划**：无需支付月费，但每售出一件商品需支付 **$0.99** 的费用。此计划适合每月仅销售少量商品的卖家。

两种计划均可使用亚马逊物流（FBA）服务。然而，职业销售计划额外提供以下专属功能：自定义运费、批量上架工具、广告优惠和优惠券、自动定价工具、销售指导以及市场网络服务等。

![](img/69c8d66440dfa452e04ca8a13dbb19ab_6.png)

## 📦 第二步：将商品上架

创建账户后，下一步是将你的商品上架到亚马逊平台。这个过程通常可以在5分钟内完成。

![](img/69c8d66440dfa452e04ca8a13dbb19ab_8.png)

商品上架主要有两种方式：

1.  **匹配现有商品页面**：如果你销售的商品与亚马逊目录中已有的商品完全相同，你可以直接匹配到该商品的现有页面进行销售。
2.  **创建新商品页面**：如果你是某商品的唯一卖家、销售自有品牌商品或向某个类别添加新品，则需要创建一个全新的商品详细信息页面。

创建新商品页面需要一个商品标识符（即条形码），常见的类型包括UPC、GTIN、EAN或ISBN。你可以在商品包装上找到这些条形码：**12位数字的是UPC，13位数字的是EAN**。如果你销售的商品没有条形码，可以申请 **GTIN豁免**，从而无需条形码也能上架商品。

以下是上架商品的具体操作方法：

![](img/69c8d66440dfa452e04ca8a13dbb19ab_10.png)

*   使用 **“添加新产品”** 选项，可以逐个添加商品。
*   使用 **“通过上传批量添加商品”** 选项，可以一次性上传整个库存。
*   对于手工制作或可定制商品的卖家，平台还提供了特殊的商品上架选项以满足其独特需求。

## 🚚 第三步：配送订单与客户服务

商品成功上架后，你需要决定如何高效地配送订单并提供客户服务。亚马逊为此提供了两种主要方案。

![](img/69c8d66440dfa452e04ca8a13dbb19ab_12.png)

![](img/69c8d66440dfa452e04ca8a13dbb19ab_13.png)

**方案一：亚马逊物流（FBA）**
使用FBA的卖家将商品发送至亚马逊的运营中心。此后，亚马逊将负责商品的存储、打包、配送、客户服务及退货处理。使用FBA的商品有资格享受 **Prime** 配送服务，并会显示 **Prime徽章**，这有助于提升商品的吸引力和销量。

**方案二：商家自发货（FBM）**
选择自行配送的卖家需要自己负责订单的打包、按时发货、物流跟踪、客户咨询以及处理退货等所有环节。

![](img/69c8d66440dfa452e04ca8a13dbb19ab_15.png)

## 💰 第四步：收取销售收入

![](img/69c8d66440dfa452e04ca8a13dbb19ab_17.png)

订单完成后，你将通过亚马逊获得销售收入。亚马逊会每两周通过 **自动清算中心（ACH）** 将符合条件的销售收入结算至你绑定的银行账户。

为了准确计算利润，卖家必须熟悉亚马逊的各项收费。你可以在卖家中心主页的搜索栏中输入 **“亚马逊收费表”**，该帮助页面会列出所有与销售相关的费用，如佣金、FBA费用等。

最后，我们强烈建议你定期访问卖家中心 **“绩效”** 选项卡下的 **“账户状况”** 页面，持续监控你的卖家绩效指标，确保账户健康运营。

![](img/69c8d66440dfa452e04ca8a13dbb19ab_19.png)

## 📝 总结

![](img/69c8d66440dfa452e04ca8a13dbb19ab_20.png)

本节课我们一起学习了在亚马逊开店的完整运作流程。我们将其分解为四个核心步骤：**注册并选择销售计划**、**上架商品**、**选择物流与客服模式（FBA/FBM）** 以及 **收款与费用管理**。理解并掌握每一步的操作与选择，是成功开启亚马逊跨境电商之旅的基础。
