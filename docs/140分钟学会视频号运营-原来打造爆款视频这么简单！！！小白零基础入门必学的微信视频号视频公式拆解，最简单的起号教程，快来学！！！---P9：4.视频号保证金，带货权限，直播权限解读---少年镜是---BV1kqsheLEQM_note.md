# 视频号运营：P9：4. 视频号保证金、带货权限与直播权限解读 🔍

在本节课中，我们将详细解读视频号的带货身份、视频带货、橱窗带货以及直播带货的具体权限与开通条件。理解这些规则是开启视频号电商变现的基础。

## 带货身份与经营资质

上一节我们介绍了视频号的基础设置，本节中我们来看看如何查看和设置带货身份。

在创作者中心板块，点击右上角的个人头像，即可查看你的带货身份。身份主要分为两种：**达人**和**商家**。这个身份后期可以修改，无需担心初始选择错误。

经营资质部分通常无需过多关注。

## 带货权限详解

了解身份后，我们来看看具体的带货权限。带货权限主要分为三个板块：视频带货、橱窗带货和直播带货。

以下是三种带货方式的具体说明：

*   **视频带货**
    *   **含义**：指在发布的视频中挂载购物车，类似抖音的“小黄车”。用户观看视频时，屏幕会出现橙色购物车标识，点击即可购买商品。
    *   **开通条件**：要求账号拥有**1000个有效粉丝**，且视频内容必须为**原创**。对于搬运内容的账号，开通此权限较为困难。
    *   **注意事项**：此权限非常严格，一旦发生违规，权限会被立即关闭。即使权限被关闭，仍可通过橱窗进行带货。

*   **橱窗带货**
    *   **含义**：在视频号主页展示商品橱窗，用户点击橱窗即可浏览和购买商品。这是最常用、最稳定的带货方式，对粉丝数无硬性要求（但开通橱窗本身可能有其他条件，如缴纳保证金）。
    *   **优势**：权限相对稳定，是搬运类或非原创账号的主要带货途径。

![](img/9f4565e0ffeb0e6c423e26a86836091d_3.png)

*   **直播带货**
    *   **含义**：在直播过程中销售商品。这是互动性强、转化率高的带货方式。
    *   **开通条件（挂商品）**：目前规则要求视频号拥有**1000个有效粉丝**，方可在直播间挂载购物车。未满足此条件只能进行纯内容直播，无法销售商品。
    *   **实名认证限制**：当前规则规定，一个自然人仅能实名认证**2个**视频号（此前可认证5个）。超过数量则无法发起带货直播。

## 直播权限实操与解决方案

![](img/9f4565e0ffeb0e6c423e26a86836091d_7.png)

![](img/9f4565e0ffeb0e6c423e26a86836091d_9.png)

了解了直播带货的条件后，我们通过实际操作来看看权限的具体表现。

![](img/9f4565e0ffeb0e6c423e26a86836091d_11.png)

一个满足千粉条件的账号，在发起直播时，可以顺利进入“商品”页面添加购物车，从而开始带货直播。

![](img/9f4565e0ffeb0e6c423e26a86836091d_13.png)

而一个未满足千粉条件的账号，在尝试直播添加商品时，系统会提示“**未达到直播条件，视频号有效关注1000人才可以进行直播带货**”，从而无法在直播间挂载商品。

那么，对于想快速开启直播带货但粉丝不足的创作者，有什么解决方案呢？

![](img/9f4565e0ffeb0e6c423e26a86836091d_15.png)

![](img/9f4565e0ffeb0e6c423e26a86836091d_17.png)

**核心解决方案是：开通视频号小店。** 开通视频号小店后，即使粉丝数不足1000，也可以在直播间挂载商品。开通小店需要准备营业执照等相关资质。具体的开店流程和资质办理方法，我们将在后续课程中详细讲解。

---

本节课中我们一起学习了视频号的三种带货权限及其开通条件。总结如下：
1.  **视频带货**：要求高（千粉+原创），权限易丢失。
2.  **橱窗带货**：最常用、最稳定的带货方式。
3.  **直播带货**：需满足**千粉**或已**开通视频号小店**，且受实名认证数量限制。

![](img/9f4565e0ffeb0e6c423e26a86836091d_19.png)

对于初学者，建议从经营橱窗和积累粉丝开始，同时了解视频号小店作为开通直播带货的备选路径。
