# 新媒体运营教程：P15：01.10.03：🤖 训练AI从分析到产出爆款笔记

在本节课中，我们将学习如何训练AI（以ChatGPT为例）分析爆款笔记，并基于分析结果生成具备爆款潜质的新笔记。整个过程分为两个核心步骤：**分析学习**与**模仿创作**。

上一节我们介绍了通过“喂资料”的方式让AI理解特定平台的文案风格。虽然上一节的旅游攻略案例效果不理想，但方法是正确的。关键在于让AI透彻分析你提供的优质内容，从中学习平台的文案写法。

本节中，我们来看看如何具体操作，让AI分析爆款内容并产出新笔记。

## 第一步：让AI分析爆款笔记

![](img/180a114c8f06c1bc9626420f8ee5402e_1.png)

我们的目标是让AI成为我们的“内容分析师”。为此，我们需要向它提供真实的爆款笔记，并要求它从以下几个维度进行分析：

以下是需要AI分析的核心要点：

1.  **爆款因素**：分析内容为何能成为爆款，例如是否引发了情感共鸣、提供了实用价值、讲述了逆袭故事等。
2.  **内容结构**：分析文案的整体框架。
    *   **开篇**：用什么方式（如提问、制造焦虑、引用名人）吸引用户继续阅读。
    *   **中间**：提供了哪些内容来戳中用户痛点，锁住用户停留时间。
    *   **结尾**：设置了什么“钩子”（如鼓励互动、引导私信、提供福利）来促进用户行动。
3.  **关键词策略**：分析笔记中是否刻意植入了关键词（符合中文分词法），以及关键词的出现频率和分布方式。

![](img/180a114c8f06c1bc9626420f8ee5402e_3.png)

![](img/180a114c8f06c1bc9626420f8ee5402e_5.png)

### 操作演示：以“考研”领域为例

我们选取一篇小红书上的考研爆款笔记（3000+点赞）作为分析样本。

首先，我们需要给AI清晰的指令。以下是一个有效的指令模板：

```
请你分析以下提供的小红书平台上的考研爆款笔记文案。
你需要分析：
1. 文案的爆款因素（例如：情感共鸣、实用建议等）。
2. 文案的整体结构（开篇如何吸引用户，中间如何阐述痛点，结尾如何引导互动）。
3. 该笔记是否结合中文分词法设计了关键词。如果有，请指出关键词是什么、出现频率如何；如果没有，请说明。
```

接着，将复制的爆款笔记文案粘贴给AI。AI会给出类似下表的分析结果：

| 分析维度 | AI反馈示例 |
| :--- | :--- |
| **爆款因素** | 情感共鸣（开篇描述备考焦虑）、逆袭故事、提供明确备考计划、使用表情符号增加亲和力、结尾正能量鼓励。 |
| **内容结构** | 开篇：用“阅读一篇错三个”的焦虑场景吸引同类考生。<br>中间：分享个人英语、专业课的备考方法和时间规划。<br>结尾：鼓励读者，并暗示可提供进一步帮助。 |
| **关键词分析** | 整篇文案没有明显为SEO设计的关键词堆砌，词汇均为直接、常用的备考相关词汇。 |

重复此过程，为AI提供多篇（建议3-5篇）同领域的爆款笔记进行分析，让它积累足够的“经验”。

## 第二步：整合分析与生成新笔记

在AI分析了多篇笔记后，我们需要让它总结出共性规律，并据此创作新内容。

![](img/180a114c8f06c1bc9626420f8ee5402e_14.png)

![](img/180a114c8f06c1bc9626420f8ee5402e_16.png)

### 1. 总结爆款因素

请AI对之前分析过的所有笔记进行整合与去重，提炼出该领域爆款笔记的通用要素。

**指令示例**：
```
请将你前面分析过的几篇考研爆款笔记的爆款因素总结一下，整合成一个完整的列表。
```

AI可能给出的总结列表：
*   情感共鸣与逆袭故事引入
*   提供明确、阶段性的备考计划
*   给出针对各科目的具体建议和资源推荐
*   强调时间管理和效率提升
*   使用表情符号和亲切口吻
*   结尾进行正能量鼓励并引导互动

### 2. 生成内容大纲

基于总结的爆款因素，让AI为你要写的新笔记规划一个结构。

**指令示例**：
```
现在，我需要写一篇关于考研的小红书笔记。请结合你刚才总结的爆款因素，为我规划一个具备爆款潜质的笔记大纲。大纲需要细分到每个部分的核心要点。
```

AI生成的大纲可能如下：
1.  **开篇引入**
    *   用焦虑/共鸣场景或逆袭小故事吸引读者。
    *   简要表明笔记主题（如：24考研全流程规划）。
2.  **核心干货**
    *   专业选择建议。
    *   分科目（政治、英语、数学、专业课）备考策略与时间节点。
    *   时间管理与心态调整方法。
3.  **结尾互动**
    *   正能量总结与鼓励。
    *   引导读者评论、提问或关注。

### 3. 根据大纲撰写完整文案

确认大纲后，让AI根据大纲和爆款因素，填充具体内容，生成完整笔记。

**指令示例**（这是一个需要精细调整的环节）：
```
请按照你提供的大纲，撰写完整的笔记文案。要求如下：
1. 结构严格遵循爆款笔记的要素。
2. 开头需要融入一个简短的“逆袭”故事来引发读者共鸣。
3. 文中需要自然使用一定数量的emoji表情。
4. 需要以符合中文分词法的方式，自然融入“考研”、“政治备考”等关键词。
```

**生成的文案示例**：
> “曾经我也在深夜刷题到崩溃，觉得考研是座翻不过的山……但最终我成功逆袭上岸！把我的全流程规划分享给你，相信你也可以！📚
>
> **🔍 专业选择**：如果你的专业有深造价值，别犹豫！
> **🗓️ 全年规划**：
> - 现在-12月：基础阶段，主攻英语单词和专业课课本。
> - 明年1-6月：强化阶段，政治+英语真题+专业课背诵。
> …（具体计划）
>
> **💡 心态调整**：每天按计划进行，拒绝焦虑！你走过的每一步都算数！
>
> 有任何问题都可以在评论区问我，看到都会回~一起加油，明年岸上见！💪”

![](img/180a114c8f06c1bc9626420f8ee5402e_18.png)

**文案评估与优化**：AI生成的初稿在结构和基础信息上通常合格，但可能缺乏真正的“网感”和细节亮点。运营人员可以在此基础上进行优化，比如调整语气、强化痛点描述、增加更具吸引力的“钩子”等。使用更强大的模型（如GPT-4）或创建专属的GPTs助手，通过提供更优质的样本和更细致的指令，能获得更高质量的初稿。

![](img/180a114c8f06c1bc9626420f8ee5402e_20.png)

本节课中我们一起学习了利用AI辅助内容创作的核心流程：**提供爆款样本 -> 指令化分析 -> 总结规律 -> 生成大纲 -> 产出文案**。这个方法能极大提升内容创作的效率，并为新手运营提供了清晰的学习和模仿路径。记住，AI是强大的辅助工具，但最终的内容把关和人性化润色，仍需依靠运营者的专业判断。
