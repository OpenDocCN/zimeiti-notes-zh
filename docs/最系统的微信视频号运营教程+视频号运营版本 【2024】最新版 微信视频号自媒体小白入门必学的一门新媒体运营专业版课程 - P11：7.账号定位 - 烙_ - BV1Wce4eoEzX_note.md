# 微信视频号运营教程：P11：7.账号定位

在本节课中，我们将要学习如何解决视频号运营初期的两个核心问题：快速获取初始粉丝以及获取原创视频素材。我们将通过具体的方法和资源支持，帮助你顺利迈出第一步。

上一节我们介绍了账号定位的基础概念，本节中我们来看看如何解决运营初期的实际困难。

## 解决初始粉丝问题

![](img/3865e476ab9be6ee617219bf28a6e441_1.png)

有同学询问是否需要隐藏账号之前的作品。答案是肯定的，需要隐藏。这是为了保持账号内容的新鲜与统一。

![](img/3865e476ab9be6ee617219bf28a6e441_3.png)

获取初始粉丝的方法很简单：坚持每天发布一条内容。不要急于求成，先从积累100个粉丝开始。素材问题我们稍后解决，请按步骤一步步来。

## 解决原创视频素材问题

接下来解决原创视频的问题。我知道很多同学面临“不会拍摄、不会剪辑、没有专业素材”的困境。

![](img/3865e476ab9be6ee617219bf28a6e441_5.png)

![](img/3865e476ab9be6ee617219bf28a6e441_6.png)

以下是同学们普遍面临的情况：
*   不会拍摄视频。
*   不会剪辑视频。
*   没有现成的优质素材。

![](img/3865e476ab9be6ee617219bf28a6e441_7.png)

因此，我为大家准备了一份重磅福利：每一位认真听课的同学，都可以从我这里领取**30个已经剪辑好的视频素材**。

这些素材来自我投入近200万元建立的独家素材库，由专业的剪辑团队制作完成，质量很高。素材库涵盖多种类型，例如：
*   **搞笑娱乐**类视频。
*   **生活妙招**类视频。
*   其他多种热门领域的视频。

每个素材都是独立成品，领取后链接即失效，无需担心与他人重复。

## 如何领取素材

领取素材需要满足几个简单的条件。

![](img/3865e476ab9be6ee617219bf28a6e441_9.png)

以下是领取素材的三个步骤：
1.  **课堂互动**：在直播间公屏上打出“想”字。
2.  **积极互动**：全程保持积极的课堂互动。
3.  **完成听课**：我们的后台会统计听课时长与互动数据。

满足条件后，请按以下方式操作：
找到向你发送听课链接的助理老师，并对他说：“是张老师让我来领取素材的。” 这样助理老师就会为你办理领取。请注意礼貌，直接索要通常无法获得帮助。
