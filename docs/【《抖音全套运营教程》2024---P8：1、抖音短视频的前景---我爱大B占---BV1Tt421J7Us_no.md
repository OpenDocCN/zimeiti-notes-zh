# 抖音运营教程：P8：抖音短视频的前景 📈

![](img/45881f49472eba97360cf6279cf7d26e_1.png)

在本节课中，我们将要学习抖音短视频平台的整体前景，分析其市场潜力、用户构成以及为何它是我们运营起步的理想选择。

了解抖音短视频的前景，需要先通过一张图片来认识其市场规模。

![](img/45881f49472eba97360cf6279cf7d26e_3.png)

![](img/45881f49472eba97360cf6279cf7d26e_1.png)

这张图片来自抖音创作者大会的数据报告。报告显示，有超过2200万人在抖音的合计收入超过了417亿元。抖音希望将这个数字翻一番，让创作者的总收入达到800亿元。

![](img/45881f49472eba97360cf6279cf7d26e_3.png)

这体现了平台巨大的流量扶持力度。同时，我们需要明确第一个知识点：抖音用户主要分为两类。

以下是两类用户的定义：
*   **玩家**：指单纯在抖音上娱乐，没有计划通过平台赚钱的用户。
*   **创作者**：指希望通过自己制作抖音内容来获取收益的用户。本课程的学员都属于这一类。

上一节我们区分了用户类型，本节中我们来看看为何要选择抖音平台。在众多短视频平台中，为何要专注于抖音？

选择平台就像开设实体店铺选址。你一定会选择一个流量大、适合你业务的地方。做短视频同理，必须选择一个流量巨大的平台。如果平台自身流量不足，就无法分配给我们，我们也难以赚到钱。

抖音的流量数据如下：
*   **日活跃用户**：超过8亿。
*   **月活跃用户**：超过40亿人次。

这是一个极其庞大的流量池。流量大，竞争压力是否也很大？答案是不会。因为平台上的创作者数量相对较少，大约只有5000万。

这里的“创作者”定义很宽泛。并非只有网红达人才算创作者。例如，邻居家的老太太用手机拍下自己做的饭菜并上传到抖音，她也算创作者。因此，你90%的竞争压力，实际上来自于像邻里乡亲这样的普通用户。

接着，做抖音短视频的门槛很低。它不需要购买电脑或专业摄像机。只需要一部智能手机和一个手机号，就可以注册抖音账号。抖音倡导全民通过短视频分享和记录生活，操作非常简单。

那么，当前有什么具体的风口值得我们把握呢？首先是“抖音计划”。

抖音计划指的是，平台将帮助1000名优质的艺术创作者实现粉丝增长超过100万。有的同学可能会担心自己不会创作。

不必担心。如果你什么都会，就不需要来听课了。正是因为我们是初学者，才需要学习。老师曾指导过许多学员，包括年长者。他们虚心请教的态度值得学习：**“虚心请教厉害的人不丢人。只要你在我不懂的方面比我有本事，而我又想学，你就是我的老师。”** 这种态度能让人少走很多弯路。

因此，不以年龄定高低，不以现状断未来。老师希望学员都是谦虚好学的。如果你认为自己已经全会了，那么这门课可能不适合你。

市场庞大、竞争相对温和、内容创作门槛低，这些都是抖音的优势。我们现在身处大数据时代。在这个时代，平台会根据你的喜好推荐内容。

**算法逻辑**：`用户偏好 -> 平台推荐 -> 内容曝光`
你喜欢看什么，抖音就会给你推荐什么。这可能导致我们看到的内容比较片面。

对于内容创作，许多同学有疑问：
*   宝妈问：可以拍宝宝或做的饭菜吗？**可以。**
*   实体店主问：可以拍产品来引流吗？**可以。**
*   上班族问：可以分享普通生活吗？**可以。**

每月40亿的活跃用户中，只要你发布的内容不触犯法律和道德底线，不是低俗内容，总有一部分人会喜欢你的风格。一定会有人喜欢你的内容。

那么，为什么你觉得自己的内容没人喜欢，粉丝只有几十个呢？这是因为“喜欢你内容的那部分人”还没有刷到过你的作品。他们无法看到，自然无法成为你的粉丝。

所以，对于新手小白而言，大家都迫切想涨粉的心情可以理解。但说句实在话，不要把“涨粉”作为第一个目标。

我们的首要目标是：**先把流量（播放量）做起来。** 当你的视频播放量上涨后，粉丝自然会随之增长。`播放量 ↑ -> 曝光度 ↑ -> 潜在粉丝 ↑ -> 总粉丝数 ↑`

本节课中我们一起学习了抖音短视频的广阔前景、用户分类、平台选择的理由以及新手起步的核心思路。关键在于认识到平台的巨大流量和低创作门槛，并调整心态，将提升内容流量而非单纯涨粉作为初期运营的重点。
