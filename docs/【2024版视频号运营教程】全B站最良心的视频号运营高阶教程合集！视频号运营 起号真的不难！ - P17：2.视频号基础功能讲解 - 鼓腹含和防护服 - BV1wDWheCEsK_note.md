# 视频号运营教程：P17：视频号基础功能讲解 🎬

![](img/e49ec2ec16430b2ac7c9e7ce12e547c3_1.png)

在本节课中，我们将系统学习视频号的基础功能设置与页面布局。掌握这些基础操作是开启视频号带货之旅的第一步。

![](img/e49ec2ec16430b2ac7c9e7ce12e547c3_3.png)

## 视频号入口与开通

![](img/e49ec2ec16430b2ac7c9e7ce12e547c3_5.png)

![](img/e49ec2ec16430b2ac7c9e7ce12e547c3_7.png)

视频号的主要入口位于微信“发现”页的“朋友圈”下方。如果你的朋友圈下方没有显示“视频号”入口，可以按照以下步骤开启。

![](img/e49ec2ec16430b2ac7c9e7ce12e547c3_9.png)

以下是开启入口的步骤：
1.  打开微信，点击右下角“我”。
2.  进入“设置”。
3.  选择“通用”。
4.  点击“发现页管理”。
5.  找到“视频号”选项，并开启其右侧的开关。

![](img/e49ec2ec16430b2ac7c9e7ce12e547c3_11.png)

![](img/e49ec2ec16430b2ac7c9e7ce12e547c3_13.png)

开启后，返回“发现”页即可看到视频号入口。点击进入，如果尚未开通视频号，页面会提示你“发表视频”。按照指引设置账号名称，即可成功开通视频号。

## 视频号主页面布局

进入视频号后，我们会看到几个主要的功能分区。

上一节我们介绍了如何找到并进入视频号，本节中我们来看看它的主页面构成。

*   **推荐**：系统根据你的观看喜好，智能推送的视频内容。
*   **朋友点赞**：展示你的微信好友点赞过的视频。这是基于社交关系的推荐。
*   **关注**：集中展示你已关注的视频号发布的最新内容。
*   **搜索（放大镜图标）**：用于搜索特定的视频号账号或关键词。例如，搜索“保鲜膜”可以找到相关领域的带货账号，这对我们寻找“对标账号”（即学习模仿的对象）非常有帮助。

## 个人主页与基础设置

点击右下角的个人头像，可以进入个人中心。这里包含了“赞”、“收藏”、“关注”和“消息”（私信）等标签页。

其中，“隐私”设置非常重要。在这里，你可以管理“点赞”信息的可见性。例如，你可以设置不让特定好友看到你点赞过的视频，这在运营带货账号时能保护个人隐私。

![](img/e49ec2ec16430b2ac7c9e7ce12e547c3_15.png)

![](img/e49ec2ec16430b2ac7c9e7ce12e547c3_17.png)

现在，让我们进入最关键的部分——个人视频号主页的后台设置。点击个人中心页面的“视频号”入口，即可进入你自己的视频号主页。

在主页右上角点击“...”菜单，选择“资料设置”，可以进行以下基础信息修改：
*   **头像、名字、性别、地区**：按需设置即可。账号名称一年内可修改**5**次。
*   **简介**：建议填写引导购物的文案，例如：“视频同款购买方式，请点击下方商品橱窗”。可以直接参考优秀对标账号的写法。
*   **视频号ID**：长按可复制。在与商家洽谈“定向佣金”合作时，需要将此ID提供给对方。
*   **隐私与权限**：
    *   **在个人名片展示视频号**：开启后，你的微信好友能在你的个人信息页看到你的视频号；关闭则可隐藏。
    *   **作品权限**：可以设置作品“公开”或“公开但不推荐给朋友”。你也可以在发布时或发布后，手动选择“不让特定好友看见”你的某个作品。

## 商品橱窗与作品管理

在你的视频号主页，找到并点击“管理橱窗”，即可进入带货后台。在浏览他人视频号时，点击其主页的“商品橱窗”，可以查看并购买其推荐的商品。

这里需要注意两个关键数据：
*   **账号“已售”数**：指该视频号博主所有带货商品的总销量。
*   **单个商品“已售”数**：指该商品在所有带货博主那里的总销量。

这两个数据含义不同，在分析账号和选品时需要区分清楚。

![](img/e49ec2ec16430b2ac7c9e7ce12e547c3_19.png)

对于自己发布的视频作品，点击作品右下角的“...”菜单，可以进行多项管理操作：

![](img/e49ec2ec16430b2ac7c9e7ce12e547c3_21.png)

以下是作品管理的常用功能：
*   **置顶**：将重要作品固定在主页顶部。
*   **上热门（加热）**：付费推广作品。
*   **关闭/开启评论**：管理评论区。
*   **仅自己可见/删除**：隐藏或删除作品。

![](img/e49ec2ec16430b2ac7c9e7ce12e547c3_23.png)

![](img/e49ec2ec16430b2ac7c9e7ce12e547c3_25.png)

## 评论区运营技巧

![](img/e49ec2ec16430b2ac7c9e7ce12e547c3_27.png)

![](img/e49ec2ec16430b2ac7c9e7ce12e547c3_29.png)

评论区是与观众互动、促进转化的关键区域。当作品流量较大时，可能会收到负面评论，你可以**长按该评论**并选择删除。

![](img/e49ec2ec16430b2ac7c9e7ce12e547c3_31.png)

为了引导用户下单，我们可以在评论区发布指引性评论，例如：“点击下方小黄车即可购买同款”。虽然视频号评论没有官方“置顶”功能，但**点赞数最高的评论会默认靠前显示**。因此，可以通过自己或请他人为这条指引评论点赞，使其保持在评论区顶部，方便新用户看到。

![](img/e49ec2ec16430b2ac7c9e7ce12e547c3_33.png)
