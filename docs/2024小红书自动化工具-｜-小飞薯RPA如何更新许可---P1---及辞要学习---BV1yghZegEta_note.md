# 小飞薯RPA教程：P1：如何更新软件许可 🔄

在本节课中，我们将学习如何为“小飞薯RPA”自动化工具更新软件许可。更新是确保软件功能正常、适应平台最新规则的关键步骤。

## 概述

当软件发布新版本时，用户会通过QQ邮箱收到更新通知邮件。邮件中包含一个许可文件的压缩包附件。本教程将指导你完成下载、解压并替换旧许可文件的全过程，以确保软件成功更新。

## 更新流程详解

上一节我们概述了更新的重要性，本节中我们来看看具体的操作步骤。整个过程主要分为三个环节：接收邮件、下载替换、验证更新。

![](img/b642b2eca6b828583c8d6dfd447fe4cb_1.png)

### 1. 接收更新通知

软件发布更新时，系统会向你的QQ邮箱发送一封通知邮件。即使你使用其他邮箱服务，只要绑定了QQ邮箱，同样可以收到通知。你可以通过访问QQ邮箱网页版、电脑端已登录的QQ，或手机上的QQ邮箱App来查收这封邮件。

### 2. 下载并替换许可文件

以下是下载和替换许可文件的具体步骤：

1.  **打开邮件并下载附件**：登录QQ邮箱，找到更新通知邮件，点击邮件中的附件进行下载。该附件是一个压缩包（如 `update_license.zip`），内含新的软件许可文件。
2.  **解压压缩包**：将下载的压缩包解压到任意位置，你会得到一个独立的许可文件（如 `license.key`）。
3.  **定位软件目录**：找到你电脑上“小飞薯RPA”软件的安装目录。
4.  **替换文件**：将解压出的新许可文件**剪切**，然后**粘贴**到软件目录中。系统会提示“是否替换目标中的文件”，请选择“替换”或“是”以确认操作。
5.  **清理文件**：替换完成后，之前的压缩包和解压出的旧文件即可删除。

![](img/b642b2eca6b828583c8d6dfd447fe4cb_3.png)

> **核心操作**：替换文件的关键在于用新文件覆盖旧文件。在代码或命令行中，这类似于一个覆盖写入的操作：`cp -f new_license.key /path/to/software/license.key`。

![](img/b642b2eca6b828583c8d6dfd447fe4cb_1.png)

### 3. 验证更新是否成功

![](img/b642b2eca6b828583c8d6dfd447fe4cb_5.png)

文件替换完成后，需要启动软件来验证更新是否成功。以下是验证方法：

![](img/b642b2eca6b828583c8d6dfd447fe4cb_7.png)

1.  **启动软件**：进入“小飞薯RPA”软件目录，运行主程序（如 `XiaoFeiShu.exe`）。
2.  **观察启动界面**：软件成功启动后，请注意启动界面或软件内的Logo。每次重大更新，我们都会更换新的Logo（例如本次更新为火箭图标）。**Logo的变更即代表软件更新成功**。
3.  **测试核心功能**：使用软件扫码登录，并尝试执行一项自动化任务（如点赞、发布）。若能正常操作，则表明程序已完全更新并可以正常工作。

## 重要提醒

我们的每一次更新都至关重要，并非随意发布。更新内容通常涉及：

*   **平台规则适配**：跟进小红书平台的最新政策和规则。
*   **功能优化与Bug修复**：修复重大漏洞或增加免检测等核心功能。
*   **稳定性提升**：确保自动化流程的稳定运行。

因此，**强烈建议你在收到更新邮件后，尽快完成软件更新**。使用旧版本可能导致软件失效、账号操作异常，甚至给账号安全带来风险。保持软件为最新版本，是安全、高效使用“小飞薯RPA”的前提。
