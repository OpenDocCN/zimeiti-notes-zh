# 跨境电商FBM运营：P1：客户索赔处理指南

在本节课中，我们将学习如何处理FBM（卖家自配送）模式下遇到的客户索赔问题。我们将了解索赔的类型、处理流程以及如何维护店铺的健康指标，确保你的店铺能够稳定运营。

## 客户索赔的类型与影响

在运营FBM时，你可能会遇到客户提出索赔的情况。这类索赔通常表现为客户要求退货，并索要退货地址。你可以直接同意退货，并告知客户问题已处理完毕，无需他实际将货物退回。

但是，你必须及时处理这些索赔。如果未能及时处理，你的店铺后续会累积风险率。不过，此时你的店铺通常不会被直接封禁。

## 亚马逊的店铺绩效指标

上一节我们提到了风险率，本节中我们来看看亚马逊具体考核哪些指标。亚马逊会考核你的店铺缺陷率、准时送达率以及有效追踪率。

当你的这些指标不达标时，亚马逊会向你发出风险提示。请放心，在这个阶段，你的店铺不会被封停。

## 新店铺的考察期与长期风险

这是一段周期的考核，对于新店铺尤其如此。如果你刚出现此类情况，店铺不会被封停。

但如果你持续出现大量此类问题，你的店铺最终是会被封掉的。

---

**本节课总结**

本节课中，我们一起学习了FBM模式下客户索赔的处理方法。核心要点是：对于简单的退货索赔，可以快速同意以关闭案例；同时，必须关注亚马逊的绩效指标（缺陷率、准时送达率、有效追踪率），及时处理问题以避免指标恶化。新店铺有一定容错空间，但长期或大量的问题将导致店铺被封停。保持及时响应和良好的绩效是FBM运营的关键。
