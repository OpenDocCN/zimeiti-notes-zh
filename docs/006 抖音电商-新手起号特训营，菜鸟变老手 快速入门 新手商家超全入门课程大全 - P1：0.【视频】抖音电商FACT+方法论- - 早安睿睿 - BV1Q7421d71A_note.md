## 第一节：FACT经营方法论

![](img/899b5295f6edbd3f46eb48b6a5e276de_4.png)

![](img/899b5295f6edbd3f46eb48b6a5e276de_5.png)

上一节我们概述了课程目标，本节中我们来看看抖音电商的基础经营框架——FACT方法论。抖音是兴趣电商平台，官方将其定义为：满足用户对美好生活的向往、挖掘潜在购物兴趣、提升生活品质的电商模式。

![](img/899b5295f6edbd3f46eb48b6a5e276de_6.png)

![](img/899b5295f6edbd3f46eb48b6a5e276de_8.png)

基于兴趣电商的特性，**优质内容**至关重要。内容是连接用户与商品的桥梁，是重要的触达和互动渠道，也是商品场景化、生动化呈现的方式。

FACT经营方法论包含四个核心组成部分，共同构成了内容经营场。

以下是FACT方法论的四个要点：

1.  **阵地自营 (Field)**：这是商家的自有阵地和持续内容生产基地。商家需要做好短视频和直播。
    *   **短视频**：用于种草、引流和转化，打通“货找人”的消费链路。
    *   **直播**：用于承接流量，实现高效转化，是核心交易场。
    *   短视频与直播需协同共振，以提升店铺整体转化效率。

![](img/899b5295f6edbd3f46eb48b6a5e276de_10.png)

2.  **达人矩阵 (Alliance)**：商家或品牌需结合目标人群画像筛选达人，并与多层级的达人进行合作。
    *   与不同层级达人配合，可以承接不同层级的粉丝流量。
    *   可与达人建立长期合作关系，实现共同成长。

3.  **主题活动 (Campaign)**：平台会举办大量主题活动，商家应积极参与以获得更多曝光。
    *   例如：平台大促（双十一、618、520）。
    *   例如：单项IP活动（超级品牌日）。
    *   例如：行业专属活动（白象微、宝贝计划等）。

4.  **头部大V (Top KOL)**：联合具有专业影响力的头部达人（如董先生、李佳琦）或明星进行合作。
    *   头部大V能为品牌提供强力背书，带动销量爆发式增长。
    *   他们能撬动庞大的粉丝群体，制造品牌或行业热点，帮助品牌快速破圈。例如，花西子品牌就曾借助李佳琦的影响力迅速崛起。

![](img/899b5295f6edbd3f46eb48b6a5e276de_12.png)

FACT方法论的核心是实现 **“货找人”** 的动作，即通过内容主动触达和激发用户的潜在兴趣。

## 第二节：从FACT到全域兴趣电商 (FACT+)

上一节我们介绍了实现“货找人”的FACT方法论，本节中我们来看看它的升级版——全域兴趣电商（FACT+）。FACT+在原有基础上，打通了 **“人找货”** 的链路，从而提升了基于明确消费兴趣的高效转化。

FACT+主要新增了两个经营场景：

![](img/899b5295f6edbd3f46eb48b6a5e276de_14.png)

1.  **中心场**：主要包括**抖音商城**和**搜索**。
    *   **抖音商城**：是生意经营稳定的中心化交易场。
    *   **搜索**：是精准匹配用户需求的流量枢纽。用户主动搜索行为日益增多，是重要的流量来源。

![](img/899b5295f6edbd3f46eb48b6a5e276de_15.png)

2.  **营销场**：强调**广告与自然流量的协同**。
    *   需要通过投流广告（如巨量千川）结合优质内容，加速生意增长。
    *   需提升广告投放能力，优化电商经营各环节的协同效率。

新增中心场和营销场后，与原有的内容场（FACT）结合，便构成了完整的全域兴趣电商经营体系。

通过下图可以更完整地理解全域兴趣电商的构成：

![](img/899b5295f6edbd3f46eb48b6a5e276de_17.png)

![](img/899b5295f6edbd3f46eb48b6a5e276de_14.png)

![](img/899b5295f6edbd3f46eb48b6a5e276de_19.png)

它包括三大策略：

*   **人群策略**：做好种草拉新与用户复购。
*   **场域策略**：即**内容场**（FACT）、**中心场**（商城、搜索）和**营销场**（广告协同）。
*   **货品策略**：提供优价好物，推动商品成长，开发新品。
*   **基础体验优化**：这是经营的基石，对应店铺的**体验分**，包含：
    *   商品体验
    *   物流体验
    *   服务体验
    维护好店铺口碑和体验分，是其他场域经营能够更上一层楼的前提。

## 第三节：案例拆解——科沃斯如何玩转全域兴趣电商

理论学习后，我们通过科沃斯的案例，具体看它如何在全域兴趣电商环境下实现GMV 8000万、曝光6.5亿的成绩。其策略覆盖了三个核心场域。

![](img/899b5295f6edbd3f46eb48b6a5e276de_21.png)

![](img/899b5295f6edbd3f46eb48b6a5e276de_22.png)

以下是科沃斯在各场域的具体打法：

*   **内容场 (FACT) 应用**：
    *   **短视频**：发起话题挑战赛，参与平台热点，联动科技类达人和虚拟IP，实现品牌破圈和高效种草。
    *   **直播**：与《变形金刚》IP跨界联名，强化品牌科技感；官方直播间与多达人直播间联动直播，转化种草用户，实现销量爆发。
    *   **货品与服务**：主推旗舰商品与潜力新品，并全程保障商品质量、导购、售后与物流服务，提升用户体验。

*   **营销场应用**：
    *   发起的话题挑战赛获得超1.5亿次曝光。
    *   通过IP联名活动进行流量聚合与转化，强化“科沃斯=高科技”的用户心智。

![](img/899b5295f6edbd3f46eb48b6a5e276de_24.png)

*   **中心场应用**：
    *   **抖音商城**：通过参与品牌馆置顶展示、居家好物品牌置顶、商城定向展示等方式，扩大曝光，获取转化新增量。
    *   **搜索**：
        *   **启发式搜索**：在用户搜索相关品类（如“餐饮”）时，提前置顶科沃斯品牌。
        *   **联想截流**：用户搜索“扫地”时，直接置顶展示科沃斯热门活动型号，提前拦截精准流量。

![](img/899b5295f6edbd3f46eb48b6a5e276de_25.png)

## 总结与回顾 🎯

![](img/899b5295f6edbd3f46eb48b6a5e276de_27.png)

本节课我们一起学习了抖音电商从FACT到FACT+（全域兴趣电商）的经营方法论。

![](img/899b5295f6edbd3f46eb48b6a5e276de_28.png)

*   **FACT方法论**是内容场的核心，通过**阵地自营、达人矩阵、主题活动、头部大V**实现 **“货找人”**。
*   **全域兴趣电商 (FACT+)** 在FACT基础上，新增了**中心场（商城、搜索）** 和**营销场（广告协同）**，打通了 **“人找货”** 链路，实现了全域流量分发与转化。
*   **经营要点**：商家需要协同运营**内容场**（短视频、直播）、**中心场**（商城、搜索）和**营销场**（付费流量、平台活动），并夯实**店铺体验分**这一经营基石。
