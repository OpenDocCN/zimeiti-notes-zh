# 微信视频号运营：P21：19.实操步骤 - 开通与上架商品

在本节课中，我们将学习如何在微信视频号中开通商品橱窗功能并上架商品。这是通过视频号进行带货变现的关键一步。

上一节我们介绍了视频号的潜力，本节中我们来看看具体的操作流程。

## 第一步：打开微信
首先，需要打开手机上的微信应用。

![](img/95d62d9cefa463619a343bb457d73140_1.png)

## 第二步：进入“发现”页
在微信主界面，点击底部的“发现”选项卡。

![](img/95d62d9cefa463619a343bb457d73140_2.png)

## 第三步：找到视频号入口
在“发现”页面中，将视线向上滑动。你会看到“视频号”的入口，其图标是一个黄色的蝴蝶形状。点击进入视频号。

![](img/95d62d9cefa463619a343bb457d73140_1.png)
![](img/95d62d9cefa463619a343bb457d73140_2.png)

![](img/95d62d9cefa463619a343bb457d73140_4.png)

![](img/95d62d9cefa463619a343bb457d73140_6.png)

## 第四步：进入个人中心
进入视频号后，点击页面右上角的人形头像图标，进入个人中心。

![](img/95d62d9cefa463619a343bb457d73140_4.png)
![](img/95d62d9cefa463619a343bb457d73140_6.png)

## 第五步：进入创作者中心
在个人中心页面，向下滑动，在最下方找到并点击“创作者中心”。如果未看到此选项，可能是因为账号是新注册的且未发布过作品。发布一条作品后即可出现。

![](img/95d62d9cefa463619a343bb457d73140_8.png)

![](img/95d62d9cefa463619a343bb457d73140_9.png)

## 第六步：找到带货中心
进入创作者中心后，在页面中找到“带货中心”的入口，其图标类似一个小卖部。

![](img/95d62d9cefa463619a343bb457d73140_8.png)
![](img/95d62d9cefa463619a343bb457d73140_9.png)

接下来，系统会根据你的账号状态显示不同内容。以下是两种情况的处理方式。

### 情况一：未开通橱窗权限
如果尚未开通商品橱窗功能，点击“带货中心”后会跳转到开通界面。

1.  **选择带货身份**：界面会要求你选择带货身份。选项通常包括“我是商家”、“我是达人”、“我是商家且已有公众号”。**请选择“我是达人”**。
    ![](img/95d62d9cefa463619a343bb457d73140_11.png)

![](img/95d62d9cefa463619a343bb457d73140_11.png)

2.  **交纳保证金**：选择身份后，会跳转到交纳保证金的界面。点击“去交纳”按钮，按照提示通过微信支付交纳100元保证金即可完成开通。

    ![](img/95d62d9cefa463619a343bb457d73140_13.png)

### 情况二：已开通橱窗权限
如果已经开通了商品橱窗权限，点击“带货中心”后会直接进入商品选品和上架界面。

## 商品上架操作
开通权限后，即可开始上架商品。以下是核心操作路径：

1.  在“带货中心”内，你可以浏览和搜索来自视频号选品库的商品。
2.  找到想要推广的商品后，通常有“加橱窗”或类似按钮，点击即可将商品添加到自己的橱窗中。
3.  你可以在个人主页的橱窗中管理已上架的商品。

![](img/95d62d9cefa463619a343bb457d73140_13.png)

整个操作流程的路径可以总结为以下代码结构：
```plaintext
微信 -> 发现 -> 视频号 -> 个人中心（头像）-> 创作者中心 -> 带货中心 -> (开通/选品上架)
```

![](img/95d62d9cefa463619a343bb457d73140_15.png)

![](img/95d62d9cefa463619a343bb457d73140_15.png)

本节课中我们一起学习了开通微信视频号商品橱窗和上架商品的具体步骤。关键点在于通过“创作者中心”进入“带货中心”，根据账号状态完成身份选择与保证金交纳，之后便可以在选品库中添加商品进行推广。掌握这个流程是开始视频号带货的第一步。
