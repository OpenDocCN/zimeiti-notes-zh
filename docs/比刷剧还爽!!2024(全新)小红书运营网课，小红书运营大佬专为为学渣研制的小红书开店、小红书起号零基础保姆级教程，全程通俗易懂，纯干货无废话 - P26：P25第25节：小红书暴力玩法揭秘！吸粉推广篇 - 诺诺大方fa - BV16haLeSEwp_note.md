# 小红书运营：P26：小红书暴力玩法揭秘！吸粉推广篇 🔍

在本节课中，我们将要学习小红书运营中的两种核心推广方式：自建账号矩阵推广与官方广告投放。我们将理清两者的关系，并详细介绍各自的运作模式与适用场景，帮助你根据自身情况选择最合适的推广策略。

上一节我们介绍了小红书内容创作的核心技巧，本节中我们来看看如何通过不同的账号策略和官方工具来放大推广效果。

## 账号类型解析

首先，我们和大家介绍一下小红书的账号，到底可以分成哪三类。

以下是三种主要的账号类型及其特点：

1.  **企业号**
    *   大部分作用是树立官方品牌形象。
    *   可以拥有官方认证标志，开展官方特权活动。
    *   可以展示品牌名称、地址等信息，方便开展业务。

2.  **个人号**
    *   大部分用户运营的是个人号。
    *   特点是灵活性高，可以是品牌员工或消费者。
    *   同时能塑造个人品牌特色，品牌相关人物和员工出镜也显得自然。

3.  **引流号**
    *   在之前的课程中讲过，俗称“炮灰号”。
    *   数量可以是多个。
    *   作用是用完即可舍弃，或用于紧急情况充当小号。
    *   一般没有太多个性化规划。

通常，我们看到的品牌推广或推广渠道，基本上都是以企业号、个人号或引流号自行发布笔记进行引流。

## 企业号矩阵运营建议

那么，我们给大家今天这节课的建议是，打造企业号矩阵有一些建议。

以下是构建有效账号矩阵的几个关键点：

1.  **品牌活动积极参与**
    *   品牌号发起活动时，个人号（小号）应积极逐个参与。
    *   例如，婚纱、服装美妆类企业号发起的试用或团购活动。

2.  **个人号展示品牌属性**
    *   在个人号的笔记内容中，多暗示品牌的定位属性。
    *   例如，品牌定位是城市年轻人还是职场年轻人。

3.  **个人号发布产品笔记的频率控制**
    *   发布的频率不宜过多。
    *   一般采用 **三七分** 原则：**30%** 用于营销，**70%** 用于种草或科普类内容。

4.  **企业号引流技巧**
    *   企业号若想引流，最好使用引流小号。
    *   在笔记下方通过评论的方式进行相关引流。

跟大家讲了企业号矩阵的提议之外，之后我们大家还知道有哪些跟矩阵号之外还有哪些推广方式吗？

### 广告类型

信息流广告主要分为两大类：

1.  **信息流广告**
    *   用户浏览小红书首页信息流时，不定期间隔出现的广告。

2.  **搜索广告**
    *   用户进行搜索时出现的广告。
    *   例如，用户搜索“帽子”时，系统可能推荐大衣、外套等搭配商品。
    *   或用户搜索“考研资料”时，可能推荐补充能量的商品。
    *   这与小红书的官方算法有关。

我们投的搜索竞价类广告，即关键词搜索广告，最具有代表性的是百度。小红书借鉴了此类模式，通过对特定关键词出价，实现不同的广告排名。

**公式**：**广告排名 ≠ 出价高低**，但出价是影响排名的重要因素。
