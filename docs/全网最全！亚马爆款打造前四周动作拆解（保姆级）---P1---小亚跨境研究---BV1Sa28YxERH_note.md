# 亚马逊爆款打造：P1：新品上架前准备与推广前四周动作拆解 🚀

在本节课中，我们将学习如何系统性地为亚马逊新品上架做准备，并拆解推广前四周的核心操作步骤。课程内容分为新品上架前的准备工作与上架后四周的推广执行两大部分，共涵盖9个关键环节。

## 新品上架前的准备工作

上一节我们概述了课程的整体结构，本节中我们来看看新品上架前需要完成的五个核心准备工作。

### 1. 流量入口布局

亚马逊平台拥有多个流量入口，提前布局是获取曝光的基础。以下是需要关注的主要流量入口：

*   **搜索流量**：通过关键词优化获取。
*   **类目流量**：确保产品放入正确的分类节点。
*   **广告流量**：包括Sponsored Products, Sponsored Brands等。
*   **关联流量**：如“看了又看”、“买了也买”。
*   **站外流量**：从社交媒体、红人等渠道引流。

### 2. 理解A9算法逻辑

A9是亚马逊的核心搜索排名算法。理解其底层逻辑有助于优化Listing。核心公式可简化为：

**相关性 + 转化率 + 满意度 = 更高排名**

*   **相关性**：关键词与产品、用户搜索意图的匹配度。
*   **转化率**：流量转化为订单的效率。
*   **满意度**：由客户评价、退货率、复购率等指标体现。

### 3. Listing优化技巧

一个优秀的Listing是转化的前提。优化需覆盖以下要素：

*   **标题**：核心关键词前置，包含品牌、主要特性。
*   **图片与视频**：高清主图、场景图、功能图、视频演示。
*   **五点描述**：突出产品卖点与解决痛点，嵌入关键词。
*   **产品描述**：补充详细信息，增强信任感。
*   **后台搜索词**：填入未在前台显示的关联关键词。

### 4. 竞品调研分析

通过分析优秀竞品，可以获取宝贵市场信息。调研应包含：

*   **价格策略**：竞品定价、优惠券使用情况。
*   **Review分析**：研究好评关注点和差评痛点。
*   **Listing质量**：学习其图片、文案布局。
*   **流量来源**：初步判断其主要的广告或关键词。

### 5. 早期Review获取

早期评价对建立信任、提升转化至关重要。合规获取方式包括：

*   **亚马逊Vine计划**：官方合规渠道，获取高质量评价。
*   **早期评论人计划**（若仍适用）：激励已购客户留评。
*   **邮件跟进**：通过合规的售后邮件请求反馈。

## 推广前四周动作拆解

![](img/9c0695fc93c8437198d2c23c41b1a462_2.png)

完成了充分的上架前准备后，我们进入实战推广阶段。接下来，我们将分四周详细拆解每一步推广动作。

### 第一周：广告基础搭建

![](img/9c0695fc93c8437198d2c23c41b1a462_3.png)

本周核心目标是启动广告，为产品引入初始流量。重点是学会布局自动与手动广告。

以下是第一周的关键动作列表：

1.  **开启自动广告**：广泛匹配，用于探测客户搜索词和ASIN。
2.  **开启手动广告（广泛/词组）**：针对核心大词，进行初步测试。
3.  **设置合理的预算与竞价**：根据类目竞争程度设定。
4.  **监控广告数据**：重点关注曝光量、点击率和转化率。

![](img/9c0695fc93c8437198d2c23c41b1a462_4.png)

### 第二周：调整推广节奏

基于第一周的数据反馈，本周需要调整策略，明确推广方向。

![](img/9c0695fc93c8437198d2c23c41b1a462_5.png)

以下是第二周需要执行的操作：

1.  **分析自动广告报告**：将高绩效搜索词添加为手动广告关键词。
2.  **优化手动广告结构**：区分核心词、长尾词，并建立不同的广告活动。
3.  **调整竞价策略**：对表现好的关键词提高竞价，表现差的降低或否定。
4.  **考虑开启商品定位广告**：针对互补品或竞品ASIN进行投放。

![](img/9c0695fc93c8437198d2c23c41b1a462_6.png)

### 第三周：广告深度优化

产品经过两周推广，已积累一定数据。本周需进行精细化运营，提升广告效率。

优化工作主要围绕以下几点展开：

![](img/9c0695fc93c8437198d2c23c41b1a462_7.png)

1.  **否定无效流量**：在自动和手动广告中否定不相关或表现极差的关键词。
2.  **拓展优质关键词**：利用亚马逊品牌分析、第三方工具拓展新的高相关词。
3.  **优化广告架构**：将表现稳定的关键词转移到单独的“精准匹配”活动中，以便更好控制。
4.  **评估ACOS**：分析广告投入产出比，在销量增长和成本间找到平衡点。

### 第四周：打造高转化条件，拉动排名

![](img/9c0695fc93c8437198d2c23c41b1a462_8.png)

本周目标是集中资源打造高转化环境，冲击更高销售排名。

为实现目标，可以采取以下组合策略：

![](img/9c0695fc93c8437198d2c23c41b1a462_9.png)

1.  **促销组合**：合理使用Coupon、Promotion（如买一赠一、百分比折扣）刺激购买。
2.  **站外引流**：通过社交媒体、Deal站等渠道引流，短期内提升销量和排名。
3.  **广告聚焦**：加大核心出单词的广告投入，抢占头部流量。
4.  **优化Listing**：根据前三周的客户反馈和广告数据，微调Listing细节，提升转化率。
