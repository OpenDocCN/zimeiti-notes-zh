# 抖音起号教程：P11：3 抖音选题思路技巧 - 上

## 📝 概述
在本节课中，我们将学习一套高效的抖音选题公式。这套公式能帮助你快速找到创作方向，解决“不知道发什么”的难题，并让你的内容更精准地吸引目标用户。

---

## 🧠 选题的重要性
你现在做抖音时遇到的困难，往往源于选题阶段思考不足。不会找选题会导致创作效率低下。

![](img/4b398ff1dc30191810cbacb9fe57b483_1.png)

会找选题的人，能将更多时间用于视频制作。每天发布三条内容，一个月就能积累九十条。依靠数量优势就能在竞争中脱颖而出，无需过度依赖付费推广。

此外，擅长选题的人能更精准地找到用户的痛点。这更容易吸引精准粉丝观看，从而帮助账号快速打上标签。

![](img/4b398ff1dc30191810cbacb9fe57b483_3.png)

## 🔑 核心选题公式
那么，有什么方法可以解决选题难题呢？答案是存在的。这是一套百万粉丝博主常用的选题技巧。

为了方便理解，我们将其总结为一个公式：
**选题 = 内容定位 + 用户痛点 + 借势热点**

用好这个公式，你的内容方向就不会偏离。

![](img/4b398ff1dc30191810cbacb9fe57b483_5.png)

## 📚 公式应用详解
上一节我们介绍了核心公式，本节中我们来看看如何具体应用。以下是应用该公式的三个步骤：

1.  **确定内容定位**：首先，明确你的账号定位。例如，如果你想成为知识博主，就需要挖掘行业内的关键词，如“播放量”、“粉丝量”、“定位”、“选题”等。这些关键词应是客观的，不包含主观描述。

2.  **寻找用户痛点**：在内容定位的基础上，寻找目标用户的真实需求和困扰。例如，对于知识类内容，用户的痛点可能包括“如何便捷高效地学习”、“需要实用技巧教学”、“缺乏持续的内容输出灵感”等。

3.  **结合时事热点**：最后，关注并借用近期的热点事件或话题。将前两步的元素与热点进行任意组合，就能生成新颖的选题。

## 💡 实战案例演示
让我们通过具体案例来加深理解。以下是两个行业的应用示例：

*   **知识博主案例**：
    *   **内容定位**：播放量、粉丝量、定位、选题。
    *   **用户痛点**：便捷高效、技巧教学、内容输出。
    *   **借势热点**：春节。
    *   **组合选题示例**：
        *   如何用拜年话术让视频播放量翻倍？（春节+播放量+技巧）
        *   春节假期必火的三个选题方向。（春节+选题+内容输出）
        *   专属于新同学的快速涨粉三个秘诀。（高效+粉丝量+教学）

*   **美妆行业案例**：
    *   **内容定位**：美白、除皱、口红。
    *   **用户痛点**：平价、好用、明星同款、避坑扫雷。
    *   **借势热点**：迪丽热巴同款、七夕节。
    *   **组合选题示例**：
        *   迪丽热巴同款美白面膜推荐。（明星同款+美白+推荐）
        *   七夕千万别送女朋友这几支口红！（七夕+口红+避坑）

## 🚀 总结与练习
本节课我们一起学习了抖音选题的核心公式：**选题 = 内容定位 + 用户痛点 + 借势热点**。我们通过知识博主和美妆行业两个案例进行了“举一反二”的演示。

至于“反三”，就需要你来完成了。你可以根据自己所在行业的特性，运用这个公式进行思考，并将你的思考过程分享出来。
