# 新媒体运营自学教程：P16：02.1：工具篇-全平台短视频文案一键提取 📝

在本节课中，我们将学习一个非常实用的运营工具，它能帮助我们从抖音、小红书、哔哩哔哩等多个平台的短视频中，一键在线提取出完整的文案内容。掌握这个工具，可以极大提升我们收集素材和创作内容的效率。

## 工具介绍与优势

上一节我们介绍了文案提取的需求，本节中我们来看看这个强大的在线工具。传统提取文案的方法通常比较繁琐，例如使用“轻斗”小程序，或者先将视频下载到本地，再导入剪映等软件进行识别。这些方法不仅步骤多，而且难以覆盖所有平台。

这个在线工具的核心优势在于其**跨平台**和**在线操作**的便捷性。你只需要获得目标短视频的分享链接，无需下载视频文件，即可在网页上快速提取出文案。

其核心操作可以概括为以下伪代码：
```
输入：短视频平台链接（如抖音、B站、小红书等）
过程：工具在线解析链接并识别视频语音
输出：格式化后的纯文本文案
```

## 工具使用演示

![](img/828f574e5416c29394bc41476b60f912_3.png)

以下是使用该工具提取文案的具体操作步骤。

![](img/828f574e5416c29394bc41476b60f912_4.png)

1.  **打开工具网页**：首先，在浏览器中打开该在线工具的界面。
    
    ![](img/828f574e5416c29394bc41476b60f912_1.png)
    

2.  **复制视频链接**：在抖音、B站等平台，找到你想要提取文案的视频，点击“分享”按钮复制其链接。
    
    ![](img/828f574e5416c29394bc41476b60f912_3.png)
    ![](img/828f574e5416c29394bc41476b60f912_4.png)
    

![](img/828f574e5416c29394bc41476b60f912_6.png)

3.  **粘贴并提取**：将复制的链接粘贴到工具页面的输入框中，点击“提取文案”按钮。
    
    ![](img/828f574e5416c29394bc41476b60f912_6.png)
    

![](img/828f574e5416c29394bc41476b60f912_8.png)

4.  **获取与处理文案**：稍等片刻，工具就会识别并显示出视频的完整文案。你可以直接“一键复制”使用，也可以点击“格式化”，让文案按照字幕段落的形式排列，更便于阅读和修改。
    
    ![](img/828f574e5416c29394bc41476b60f912_8.png)
    

## 多平台实测

了解了基本操作后，我们来看看它在不同平台上的实际效果。除了抖音，该工具对哔哩哔哩（B站）等平台的支持同样出色。

我们将一个B站视频的链接（通常以 `BV` 号开头）粘贴到工具中并提取。结果显示，工具能够准确识别出视频中的语音内容，并转换成文字，例如：“新手做直播电商，怎么才能打造出直播爆品。在直播销售的不同阶段...”。这证明了其跨平台文案提取的准确性和实用性。

## 应用场景与价值

这个工具的强大之处在于它能无缝对接多个内容平台。无论是抖音、快手、小红书，还是视频号、哔哩哔哩，只要你能获得视频链接，就可以快速提取文案。

这对于运营工作具有重要价值。提取出的优质文案，可以作为我们进行二次创作、撰写口播稿、分析内容结构的宝贵素材。它节省了传统方法中下载、上传、转码的大量时间，将多步操作简化为一步，显著提升了工作效率。
