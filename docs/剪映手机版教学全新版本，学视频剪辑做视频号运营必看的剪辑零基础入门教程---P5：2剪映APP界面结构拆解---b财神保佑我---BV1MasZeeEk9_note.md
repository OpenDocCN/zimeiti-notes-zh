# 剪映手机版教学：P5：剪映APP界面结构拆解 📱

在本节课中，我们将要学习剪映APP的整体界面结构。了解每个模块的功能和位置，是熟练使用这款软件进行视频剪辑的第一步。

![](img/931f941c7218202715699dbb16da29aa_1.png)

## 概述

剪映APP的界面设计非常简洁，主要包含四个核心模块。熟悉这四个模块是我们开启视频剪辑之路的正确方式。

## 界面四大模块介绍

以下是剪映APP底部的四个主要功能模块，它们构成了软件的基础操作框架。

1.  **剪辑**
    *   启动剪映APP后，默认进入的首页就是“剪辑”页面。
    *   要开始剪辑视频，需要点击顶部的 **`开始创作`** 按钮，创建新项目并导入素材。
    *   在首页界面中，还可以查看之前剪辑过的项目草稿。

2.  **剪同款**
    *   剪映APP内置了大量短视频模板。
    *   对于新手，如果看到喜欢的爆款视频但不会制作，可以使用“剪同款”功能。
    *   用户只需选择模板，并按要求导入自己的视频或图片素材，即可一键生成风格相似的作品。
    *   模板按类别划分，如卡点视频、毕业季、节日主题等，方便用户查找。

3.  **消息**
    *   剪映APP与抖音账号互通。
    *   授权登录抖音账号后，可以在此模块接收官方消息、点赞通知和评论提醒。

![](img/931f941c7218202715699dbb16da29aa_3.png)

4.  **我的**
    *   这是个人账号信息页。
    *   登录后，可以查看和编辑头像、名称、抖音号、粉丝量、获赞量等信息。
    *   也可以从此处直接跳转到自己的抖音账号主页。

### 1. “剪同款”功能演示

![](img/931f941c7218202715699dbb16da29aa_9.png)

启动剪映APP，首先进入的是“剪同款”界面。这里展示了官方推荐的大量视频模板。

例如，选择一个“旅行Vlog”模板，点击预览。如果喜欢，可以点击右下角的 **`剪同款`** 按钮。

根据提示导入所需数量的素材片段（例如8个），点击 **`下一步`**。

![](img/931f941c7218202715699dbb16da29aa_11.png)

导入素材后，可以直接预览成片效果。在此界面，你还可以进行简单编辑：
*   **编辑视频**：对单个素材片段进行替换、裁剪或单独拍摄。
*   **编辑文本**：点击模板中的文字，可以修改其内容。

![](img/931f941c7218202715699dbb16da29aa_13.png)

预览无误后，点击右上角的 **`导出`**。建议将分辨率设置为 **`1080P`**，这是抖音平台的推荐规格。导出时请耐心等待，不要锁屏或切换程序。

### 2. “剪辑”功能界面

回到首页，点击底部的 **`剪辑`** 模块，这里是进行自由创作的核心区域。

![](img/931f941c7218202715699dbb16da29aa_15.png)

点击顶部的 **`开始创作`**，选择素材并 **`添加到项目`**，即可进入剪辑主界面。

主界面分为三个主要部分：

![](img/931f941c7218202715699dbb16da29aa_17.png)

*   **视频预览窗口**（顶部）：在此区域预览视频的实时效果。
*   **视频轨道/时间轴**（中部）：所有剪辑软件的核心。视频按时间顺序排列在此轨道上。中间有一条白色的 **`时间线`**，它标识着当前操作的位置，用于素材的拆分、定位和转场。
*   **工具栏**（底部）：包含各种编辑工具的一级菜单，如：
    *   `剪辑`（分割、变速、动画等）
    *   `音频`
    *   `文字`
    *   `画中画`
    *   `特效`
    *   `滤镜`等
    要对视频进行操作，需要先选中轨道上的素材，然后在工具栏中选择相应功能。

![](img/931f941c7218202715699dbb16da29aa_19.png)

## 总结

![](img/931f941c7218202715699dbb16da29aa_27.png)
