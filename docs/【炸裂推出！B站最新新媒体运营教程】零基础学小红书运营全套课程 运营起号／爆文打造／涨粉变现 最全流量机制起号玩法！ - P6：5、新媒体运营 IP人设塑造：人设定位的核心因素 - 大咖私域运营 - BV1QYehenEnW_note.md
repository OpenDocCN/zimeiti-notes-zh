在本节课中，我们将要学习新媒体运营中一个关键且常令人困惑的环节：如何塑造IP人设。许多人觉得人设定位很复杂，不知从何下手。实际上，人设的打造并非一蹴而就，它会随着你的内容方向和账号成长逐步凸显。前期所做的人设定位工作，主要是为了固化后期账号需要融入的特征点。如果你在运营账号时特别纠结人设问题，那么本节课将为你清晰地解答人设到底是什么，以及我们通常如何打造它。

角色，简单来说，就是你在特定赛道中希望呈现的专业身份。你选择什么赛道、准备输出什么方向的内容，就需要让别人知道你在这个领域具备一定的专业性。

以下是不同赛道的角色定位示例：

*   **好物分享/测评**：你可以定位为“产品测评师”或“生活好物发现家”。
*   **装修博主**：你需要展示自己拥有正在装修或已装修好的房子，记录整个过程，分享遇到的坑和最终成果，定位为“家居改造达人”或“避坑指南提供者”。
*   **母婴赛道**：最常见的角色是“宝妈”。但也可以定位为“育儿护理专家”，这个身份不一定要求自身是宝妈，可以是相关领域的专业人员。
*   **情感赛道**：可以定位为“情感顾问”、“心理学导师”，分享情感、成长、励志话题。也可以打造“成长型人设”，例如分享自己作为30岁女性，如何处理家庭、职场、创业中遇到的问题并走出来的经历。

### 2. 内容 📝

内容定位我们之前讨论过，关键在于内容是否吸引用户、是否有价值、是否有平台特色。在人设定位的语境下，内容需要特别强调**垂直与持续**。

![](img/bf1df8a62517e97efb401ab8cfb19dfe_66.png)

![](img/bf1df8a62517e97efb401ab8cfb19dfe_67.png)

![](img/bf1df8a62517e97efb401ab8cfb19dfe_68.png)

![](img/bf1df8a62517e97efb401ab8cfb19dfe_69.png)

![](img/bf1df8a62517e97efb401ab8cfb19dfe_70.png)

![](img/bf1df8a62517e97efb401ab8cfb19dfe_71.png)

![](img/bf1df8a62517e97efb401ab8cfb19dfe_72.png)

![](img/bf1df8a62517e97efb401ab8cfb19dfe_73.png)

![](img/bf1df8a62517e97efb401ab8cfb19dfe_74.png)

你必须持续输出与所选赛道紧密相关的内容。这关系到账号的垂直度和标签化，直接影响账号成长和平台的流量分发。

![](img/bf1df8a62517e97efb401ab8cfb19dfe_75.png)

![](img/bf1df8a62517e97efb401ab8cfb19dfe_76.png)

**逻辑示例：**
如果你的第一篇内容关于儿童教育，第二篇却分享宠物日常，那么之前因教育内容关注你的用户，可能对宠物内容不感兴趣而划走。这会向平台反馈“该视频不受欢迎”，从而影响后续推流效果。

### 3. 形式 🎬

形式指的是内容呈现的具体方式，主要包括以下几个方面：

![](img/bf1df8a62517e97efb401ab8cfb19dfe_96.png)

![](img/bf1df8a62517e97efb401ab8cfb19dfe_97.png)

![](img/bf1df8a62517e97efb401ab8cfb19dfe_98.png)

![](img/bf1df8a62517e97efb401ab8cfb19dfe_99.png)

![](img/bf1df8a62517e97efb401ab8cfb19dfe_100.png)

![](img/bf1df8a62517e97efb401ab8cfb19dfe_101.png)

![](img/bf1df8a62517e97efb401ab8cfb19dfe_102.png)

![](img/bf1df8a62517e97efb401ab8cfb19dfe_103.png)

![](img/bf1df8a62517e97efb401ab8cfb19dfe_104.png)

*   **内容产出场景**：你的内容是在固定室内场景拍摄，还是在室外不同场景切换？例如，总是在上班路上、车内，或特定工作场景中分享心得。拍摄场景可以预先设定，并参考对标账号的做法。
*   **声音呈现**：是本人出镜讲解，还是使用配音？例如，有些视频人物出镜但声音是后期配音。这取决于你能否流畅地进行口头表达，技术上门槛不高（如使用剪映等工具）。
*   **内容形式**：内容以何种体裁呈现？主要有以下几种：
    *   **剧情型**：创作难度较高，需要剧本和表演。
    *   **干货/经验分享型**：需要准备丰富的文案和脚本。
    *   **生活记录型**：需要积累大量的日常拍摄素材。

你需要根据自身资源和能力，选择合适的内容形式。

记忆点是让用户记住你的关键，需要通过精心设计的固定环节来强化。以下是常见的记忆点设计：

![](img/bf1df8a62517e97efb401ab8cfb19dfe_122.png)

![](img/bf1df8a62517e97efb401ab8cfb19dfe_123.png)

![](img/bf1df8a62517e97efb401ab8cfb19dfe_125.png)

![](img/bf1df8a62517e97efb401ab8cfb19dfe_126.png)

![](img/bf1df8a62517e97efb401ab8cfb19dfe_128.png)

*   **视频前几秒的固定动作或特效**
*   **统一的开场白**
*   **贯穿视频的固定背景音乐（BGM）**
*   **视频中统一出现的标志性物品**
*   **统一的服装风格**

![](img/bf1df8a62517e97efb401ab8cfb19dfe_130.png)

这些记忆点需要反复出现，刻入用户心智。当用户刷到你的视频，看到某个瞬间就能联想到你和你提供的内容。

**记忆点设计策略：**
*   **前置强化**：固定动作、特效、开场白等应放在**视频开头**。当用户浏览你的主页多个视频时，每次开头都看到相似元素，记忆会得到极大强化。
*   **全局贯穿**：统一的BGM、标志性物品或服装风格，则应**贯穿整个视频或所有内容**。例如，一听到某段BGM就知道是你的视频，或者看到某个物品就联想到你的账号。

![](img/bf1df8a62517e97efb401ab8cfb19dfe_141.png)

![](img/bf1df8a62517e97efb401ab8cfb19dfe_142.png)
