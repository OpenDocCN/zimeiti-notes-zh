# 服装行业流量爆发营：P63：画中画功能详解

在本节课中，我们将学习剪映App中“画中画”功能的核心操作。你将掌握如何添加、排列和编辑画中画素材，并学会制作三分屏视频以及为视频添加光斑特效。

![](img/5a64ac7f791ceff5e5e7c09153ca9e80_0.png)

## 添加与编辑画中画

上一节我们介绍了课程概述，本节中我们来看看如何添加和编辑画中画。

打开剪映App并导入素材后，在功能栏中找到“画中画”选项。点击“新增画中画”，即可在主视频轨道下方添加新的视频或图片素材。

以下是关于画中画素材的基本操作：
*   **移动**：在预览区用手指按住并拖动素材。
*   **缩放**：在预览区用双指捏合或分开。

画中画素材可以在预览窗口中自由缩放和移动位置。

## 层级管理与轨道数量

了解了基础操作后，我们来看看如何管理多个画中画素材。

剪映App最多支持六个画中画轨道，加上一个主轨道，总计七个轨道。当添加第二个画中画素材时，可以将其移动到画面右侧。

以下是层级管理的核心概念：
*   **素材层级越高，在画面中显示的位置越靠前**。
*   点击“层级”按钮可以调整画中画素材的上下顺序。
*   点击“切主轨”可以将画中画素材移动到主轨道上。

## 实战案例一：制作三分屏视频

掌握了层级管理，现在我们来应用这些知识制作一个三分屏视频。

点击“开始创作”，导入一段视频素材。将视频比例设置为 **9:16**。然后返回，点击“画中画”，新增两个画中画素材并导入。

以下是制作三分屏的关键步骤：
1.  将三个视频素材（一个主轨，两个画中画轨）在画面中并排排列。
2.  调整每个素材的大小和位置，使它们各占据画面的三分之一。

![](img/5a64ac7f791ceff5e5e7c09153ca9e80_2.png)

预览效果，一个三分屏视频就制作完成了。

## 实战案例二：添加光斑特效

除了布局，画中画还能实现丰富的视觉效果。接下来，我们学习如何使用画中画为视频添加光斑特效。

同上，先导入一段主视频。然后点击“画中画”，新增画中画。在素材库中搜索“光斑”，选择一段合适的光斑素材添加进来。

以下是添加光斑特效的操作：
*   选中光斑素材轨道。
*   点击底部工具栏中的“混合模式”。
*   在混合模式选项中选择“绿色”。

这样，光斑特效就自然地叠加到主视频上了。
