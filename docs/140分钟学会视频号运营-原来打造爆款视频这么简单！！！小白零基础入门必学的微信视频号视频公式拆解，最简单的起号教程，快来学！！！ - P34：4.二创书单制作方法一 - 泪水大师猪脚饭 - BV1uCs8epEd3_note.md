# 视频号运营：P34：二创书单制作方法一

## 📚 概述
在本节课中，我们将学习如何通过二次创作（二创）的方式制作书单类视频。这是一种比直接混剪原创度更高、效果更好的方法，尤其适合新手入门。我们将从寻找素材、提取文案、替换片段到最终成片，一步步拆解整个操作流程。

---

## 🎬 剪辑是必备技能
上一节我们介绍了混剪的基础概念，本节中我们来看看如何通过二次创作提升视频的原创度。对于新手而言，剪辑是视频制作的必备技能。许多初学者觉得混剪和剪辑很难，根源在于缺乏基础练习。

**核心建议**：先拿十几个视频练习剪辑，熟悉每个功能的作用。这个过程大约需要一两个小时。熟能生巧后，再学习课程会事半功倍。

## 📖 为何选择书单带货
在众多视频带货类型中，书单类内容一直是“常青树”。无论是俄罗斯带货、益智玩具还是老照片修复，书单的变现效果都相对稳定。只要学员认真操作，基本都能出单。

书单视频的逻辑是通过一段故事、影视片段或道理阐述，最终引出一本书，激发观众的购买欲望。这种软性推广比直接展示产品功能的“硬广”更容易被接受。

## 🛠️ 二次创作实操步骤
掌握了基本逻辑后，我们进入实操环节。二次创作的核心是：**提取原视频文案和结构，替换其中的视频素材，生成新的配音和字幕**。

![](img/b6cd95912c23860b7ff4636ac97cc0e3_17.png)

我们以书籍《女孩要懂得保护自己》的带货视频为例进行拆解。

### 第一步：提取原视频素材与文案
1.  在抖音找到目标带货视频，通过“分享”按钮复制链接。
2.  使用去水印工具（如“配音神器”）保存视频到相册。
3.  使用文案提取工具，将视频的解说文案提取出来。

![](img/b6cd95912c23860b7ff4636ac97cc0e3_19.png)

### 第二步：生成新配音
1.  检查提取的文案是否有错别字，并进行修正。
2.  使用配音工具（如剪映或第三方AI配音），选择合适的声音生成新配音。
3.  试听配音，如果某段语速过快，可以添加停顿（例如1.5秒），然后重新合成。
4.  将最终配音导出保存。

![](img/b6cd95912c23860b7ff4636ac97cc0e3_21.png)

![](img/b6cd95912c23860b7ff4636ac97cc0e3_23.png)

### 第三步：在剪辑软件中替换素材
1.  将去水印的原视频导入剪映。
2.  **遮盖原字幕**：使用“蒙版-矩形”功能，框选原字幕区域，选择“反转”，并设置“画布模糊”作为背景，从而遮盖原字幕。
3.  **关闭视频原声**。
4.  **替换视频片段**：这是二创的核心。分析原视频由哪些片段组成（例如：女讲师演讲、学生在操场、几个女生玩耍、书本特写）。
    *   **方法A（替换）**：分割出要替换的片段，点击“替换”，从素材库或本地导入新素材。
    *   **方法B（删除后添加）**：删除原片段，在相应位置添加新找到的素材。确保新素材时长与原片段大致相同。
5.  **寻找新素材的途径**：
    *   **剪映素材库**：搜索关键词（如“女讲师培训”）。
    *   **抖音直接搜索**：搜索更具体的关键词（如“女生台上演讲培训视频”），找到合适片段后保存使用（尽量去水印）。
6.  为所有新添加的片段**关闭原声**。

### 第四步：合成最终视频
1.  将第二步生成的新配音导入剪辑软件，作为视频的主音频。
2.  根据画面内容，微调音频的位置（分割、移动），使音画同步。
3.  添加合适的背景音乐。
4.  通过“文本-识别字幕”功能，为新的配音生成字幕。
5.  调整新字幕的样式和位置，确保美观。
6.  最后，将帧率设置为60，导出视频。

**核心操作公式**：
```
二次创作视频 = 原视频结构 + 新配音 + 新素材片段 + 新字幕 + 背景音乐
```
**关键代码逻辑（伪代码）**：
```python
原视频 = 下载(对标视频链接)
原文案 = 提取文案(原视频)
新音频 = 生成配音(清洗后的原文案)
新素材列表 = []
for 片段 in 分析(原视频片段):
    新片段 = 搜索替换素材(片段.关键词)
    新素材列表.添加(新片段)
新视频 = 合成(新素材列表, 新音频)
添加字幕(新视频, 新音频)
导出视频(新视频)
```

## ✅ 总结
本节课我们一起学习了书单类视频的二次创作方法。关键在于**解构优质视频，保留其文案内核与结构，并替换视觉素材**。通过提取文案、生成新配音、在抖音或素材库寻找替代片段、重新组合并添加字幕，你就能制作出一个原创度更高的新视频。

![](img/b6cd95912c23860b7ff4636ac97cc0e3_33.png)

![](img/b6cd95912c23860b7ff4636ac97cc0e3_35.png)

记住，这个方法的核心优势在于**原创度提升**，这比单纯混剪一条视频更能获得平台推荐。请务必重视实操，多练习几次，你就能熟练掌握整个流程。
