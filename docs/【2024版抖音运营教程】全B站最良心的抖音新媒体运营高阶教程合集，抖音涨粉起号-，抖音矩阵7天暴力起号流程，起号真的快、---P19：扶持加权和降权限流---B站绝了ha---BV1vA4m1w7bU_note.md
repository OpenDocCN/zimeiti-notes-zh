# 抖音运营教程：P19：扶持加权与降权限流

在本节课中，我们将学习抖音平台对视频内容的两种核心处理机制：扶持加权与降权限流。理解这些规则，是确保你的视频获得良好推荐、避免违规的关键。

上一节我们介绍了账号权重的分类，本节中我们来看看具体哪些内容会受到平台的扶持，哪些内容又会导致被降权限流。

## 扶持加权的内容特征 🚀

![](img/04f7fe489fb0bc9f59d685f8e76536b9_3.png)

平台会对符合以下特征的优质原创视频进行流量扶持和权重加成。

以下是获得扶持加权的视频需要满足的具体条件：

![](img/04f7fe489fb0bc9f59d685f8e76536b9_5.png)

*   **视频格式**：必须为竖版的原生视频，即 **9:16** 的移动端格式。
*   **画面质量**：拍摄画面需保持平稳，作品画质清晰。
*   **音频质量**：视频中无过多的杂音和噪音。

这些要求旨在鼓励创作者生产适合手机端观看的高质量内容。关于此部分的详细说明，我已整理成资料页，供大家课后查阅。

![](img/04f7fe489fb0bc9f59d685f8e76536b9_1.png)

## 导致降权限流的内容 🚫

了解完受鼓励的内容，我们再来看看哪些“雷区”会导致视频被降权甚至限流。避开这些内容至关重要。

以下是明确会导致降权限流的内容类型列表：

1.  **违规信息**：视频内容涉及吸烟、暴力、炫富（金钱）等不良信息。
    ![](img/04f7fe489fb0bc9f59d685f8e76536b9_3.png)
2.  **外部引流**：出现或引导至微信、微博、淘宝等站外联系方式。长时间展示商品Logo、手机号、二维码（二维码出现时间建议不超过3秒，但审核严格时瞬间出现也可能被驳回）。
    ![](img/04f7fe489fb0bc9f59d685f8e76536b9_5.png)
3.  **硬性广告**：进行明显的营销广告、推销或引流卖货行为。
4.  **无口播的好物推荐**：这是需要特别注意的一点。如果视频是拆箱或好物推荐类内容，**没有口播解说**，仅展示产品，极易被判定为广告而导致小黄车被下架。

针对第4点“无口播的好物推荐”，我结合实例详细说明。有些同学制作好物分享视频来带货，但发布后小黄车立即被下架，常见原因有两个：

*   **案例一**：视频中直接出现“**九块九**”等明确标价信息，属于直接销售行为。
*   **案例二**：视频只是沉默地展示玩具玩法，全程无任何口播介绍。

正确的做法是：为视频添加口播解说。即使只是简单描述使用感受，例如：“我儿子玩这个玩具，专注了四个小时。” 这样的口播能有效将视频性质从“广告”转化为“内容分享”，从而通过审核。请务必避免不说话、直接展示产品的形式。

其他如直接口播广告、念联系方式等硬广行为，在抖音平台是行不通的。
