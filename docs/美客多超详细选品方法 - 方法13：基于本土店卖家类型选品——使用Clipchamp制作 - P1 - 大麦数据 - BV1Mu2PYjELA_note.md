# 美客多选品教程：方法13：基于本土店卖家类型选品

在本节课中，我们将学习一种高效的选品方法：通过分析美客多本土店卖家的热销产品来寻找商机。本土卖家更了解当地市场需求，他们的成功选品对我们有重要参考价值。

![](img/e6ed39de4e54276aad0e1626d30f7165_0.png)

## 🎯 核心思路

美客多本土店卖家是最了解本土市场需求的群体。本土卖家选择的产品，通常更符合当地消费者的实际需求。通过观察相近品类下本土店铺内的热销产品，并结合我们自身的供应链或成本优势，以更有竞争力的价格切入市场，就有可能获得可观的平台流量。

那么，具体如何操作呢？

## 🔍 操作步骤

以下是查看本土店卖家热销产品的具体操作流程。

1.  **进入分析工具**
    打开“大麦数据”工具，进入其“行业纵览”或“商品CBT分析”功能模块。

2.  **筛选目标类目**
    在模块中选择你计划进入或正在经营的目标商品类目。

![](img/e6ed39de4e54276aad0e1626d30f7165_2.png)

3.  **设置关键筛选条件**
    为了找到近期有潜力的产品，需要设置以下两个核心筛选条件：
    *   **卖家类型**：选择“本土店”。
    *   **上架时间**：设置为“近3个月”。
    *   **销售表现**：将“月销量”条件设置为大于100。

4.  **获取结果并分析**
    应用以上筛选条件后，系统将列出所有符合条件的本土店热销产品列表。接下来，你需要仔细分析这些产品的特点、价格区间和卖家情况。

![](img/e6ed39de4e54276aad0e1626d30f7165_3.png)

通过以上步骤，我们就能高效地筛选出近期在本地市场表现强劲的产品。大麦数据是拉美地区卖家广泛使用的选品与运营分析工具。

![](img/e6ed39de4e54276aad0e1626d30f7165_4.png)

## 📝 总结

本节课我们一起学习了基于本土店卖家类型进行选品的方法。我们了解到，本土卖家的选品是市场需求的直接反映。关键在于使用专业工具（如大麦数据），通过筛选“本土店”、“近3个月上架”和“月销量>100”等条件，快速定位近期热销品，从而为我们的定价和销售策略提供可靠的数据支持。
