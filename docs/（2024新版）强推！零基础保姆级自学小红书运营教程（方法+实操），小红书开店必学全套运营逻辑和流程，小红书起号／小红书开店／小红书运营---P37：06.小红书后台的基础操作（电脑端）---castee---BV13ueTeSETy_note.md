# 小红书运营教程：P37：06.小红书后台的基础操作（电脑端） 💻

在本节课中，我们将学习如何通过电脑端操作小红书商家后台，涵盖保证金、支付方式、运费保和地址设置等核心功能。这些是店铺运营的基础，掌握它们能让你的店铺管理更高效。

![](img/d59cdffa1870e0ce3efb9769544b3d8f_1.png)

上一节我们介绍了手机端的操作，本节中我们来看看电脑端如何完成这些关键设置。

![](img/d59cdffa1870e0ce3efb9769544b3d8f_3.png)

## 保证金 💰

保证金是开店时需要向平台缴纳的一笔信用担保资金。前期可以暂不缴纳，但若想长期经营或开通更多支付方式，则建议缴纳。

以下是关于保证金的几个要点：

![](img/d59cdffa1870e0ce3efb9769544b3d8f_5.png)

*   **金额**：个人店铺保证金通常在 **1000元** 左右，具体金额根据所选经营类目而定。企业店保证金在2000元至数万元不等。
*   **缴纳入口**：登录小红书千帆商家后台，在“资金”模块中找到“店铺保证金管理”进行充值。
*   **可退还性**：与多数平台规则一致，当店铺不再经营时，保证金在无违规扣款的情况下可以全额退还。
*   **扣款规则**：若发生损害消费者权益的行为（如承诺退款但未执行），平台会从保证金中扣款。正常合规经营则不会被扣。

![](img/d59cdffa1870e0ce3efb9769544b3d8f_7.png)

> **核心公式**：`应缴保证金 = f(店铺类型， 经营类目)`
> 其中，店铺类型包括个人店、个体店、企业店；经营类目决定了具体金额。

![](img/d59cdffa1870e0ce3efb9769544b3d8f_9.png)

如果你不确定自己类目的保证金金额，可以在后台的保证金管理页面，通过选择相应类目进行查询。

![](img/d59cdffa1870e0ce3efb9769544b3d8f_11.png)

## 支付方式开通 💳

开通多元化的支付方式可以方便更多消费者下单，提升转化率。主要支持支付宝和微信支付。

![](img/d59cdffa1870e0ce3efb9769544b3d8f_13.png)

支付方式的开通同样在“资金”模块中操作。

以下是开通两种支付方式的步骤与注意事项：

![](img/d59cdffa1870e0ce3efb9769544b3d8f_15.png)

![](img/d59cdffa1870e0ce3efb9769544b3d8f_17.png)

1.  **支付宝支付**：提交资料后，通常可以当天完成开通。
2.  **微信支付**：需要先缴纳店铺保证金，开通流程可能需要**第二天**才能完成设置。提现金额将进入注册小店时使用的法人账户。

无论是绑定支付宝还是微信，都需要按照平台要求，填写对应的身份证或营业执照信息进行验证。

## 运费保设置 🛡️

运费保是一项平台服务，旨在为消费者提供退货时的运费保障，类似于其他电商平台的“退货运费险”。

![](img/d59cdffa1870e0ce3efb9769544b3d8f_19.png)

开通运费保能打消消费者对运费的顾虑，从而可能提升订单转化率并降低交易纠纷。

![](img/d59cdffa1870e0ce3efb9769544b3d8f_21.png)

它的核心作用是：当消费者退货退款成功后，**系统会自动补贴运费**。这项服务在后台的“售后”模块中，找到“运费保”选项即可点击开通。

![](img/d59cdffa1870e0ce3efb9769544b3d8f_23.png)

## 地址设置 📍

![](img/d59cdffa1870e0ce3efb9769544b3d8f_25.png)

正确设置发货和退货地址对于订单处理和客户沟通至关重要。此设置与手机端操作逻辑类似。

![](img/d59cdffa1870e0ce3efb9769544b3d8f_27.png)

地址设置在后台的“订单物流工具”中完成。

![](img/d59cdffa1870e0ce3efb9769544b3d8f_29.png)

以下是地址设置的具体建议：

*   **新建地址**：在“物流工具”中点击“新建地址”，填写准确的发货或退货地址。
*   **无货源商家**：若你是一件代发（无货源）模式，建议将退货地址填写为供应商的地址，并**在地址后备注“退货前请联系客服”**，以便协调处理。
*   **运费模板**：根据你的发货地（或供应商所在地）设置合理的运费模板，特别是对于新疆、西藏等偏远地区。

![](img/d59cdffa1870e0ce3efb9769544b3d8f_31.png)

本节课中我们一起学习了小红书电脑端后台的四个基础操作：保证金的查询与缴纳、支付方式的开通、运费保服务的启用以及地址与运费模板的设置。掌握这些是店铺顺畅运营的第一步，建议你根据自身店铺类型和经营模式，逐一在后台进行配置。
