小红书运营教程：P3：第2节：小红书平台检测规则详解 🔍

在本节课中，我们将要学习小红书平台的核心检测规则。了解这些规则是账号安全运营、避免违规限流的基础。我们将从平台检测的四个主要方面入手，并介绍一个实用的工具来辅助内容创作。

上一节我们介绍了账号定位，本节中我们来看看平台如何监管内容。小红书的检测系统主要覆盖以下四个关键区域。

以下是平台重点检测的四个方面：
1.  **个人资料、笔记内容、私信及评论**：这些位置不得含有微信号、微信二维码、代购、水印、推广淘宝店铺等明显的营销词汇。
2.  **手机设备**：同一台设备最多建议只登录三个账号，过多账号可能引发风险提示。
3.  **笔记内容规范**：内容中不得诱导用户，例如使用“关注领奖”、“点赞抽奖”等话术。这是平台明令禁止的诱导行为。
4.  **内容原创度**：小红书对原创内容的保护程度较高。切勿将同一篇文章在多个账号上重复发布。原则是：**一篇文章，一个账号**。

除了以上四个方面，敏感词检测是另一项重要规则。平台对某些特定词汇非常敏感。

以下是常见的敏感词类别示例，在创作时应尽量避免触及：
*   手机卡
*   金融
*   电子烟
*   代购
*   医疗用品

![](img/7bbfbe6351f778b5479ed3b4e16467db_1.png)

那么，如果在写作时不慎加入了未知的敏感词该怎么办？这就需要借助外部工具进行检测。

这里介绍两个可用于检测小红书敏感词的网站。我们以第一个网站为例进行演示。

![](img/7bbfbe6351f778b5479ed3b4e16467db_3.png)

假设我们撰写了一段文案，其中包含了“代购”这个词。操作步骤如下：
1.  打开检测网站。
2.  将文案内容粘贴到网站的输入框中。
3.  点击“立即检测”按钮。

![](img/7bbfbe6351f778b5479ed3b4e16467db_5.png)

检测完成后，网站会高亮或列出文案中存在的敏感词。例如，它会提示“代购”属于敏感词。这样，我们就能在发布前及时发现并修改问题。

![](img/7bbfbe6351f778b5479ed3b4e16467db_7.png)

![](img/7bbfbe6351f778b5479ed3b4e16467db_9.png)

第二个网站的使用逻辑类似，虽然可能需要注册，但功能基本一致。通常，使用第一个网站进行检测已经足够满足日常需求。
