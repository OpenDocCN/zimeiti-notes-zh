# 小红书运营：P8：店铺营销工具的使用和活动任务 📈

![](img/f58cb63583450e02a92f62a56d4ccfe4_0.png)

在本节课中，我们将学习小红书店铺后台的营销工具与活动任务。掌握这些工具的使用方法，可以有效提升店铺的转化率，并利用平台对新店的扶持政策获取更多流量。

![](img/f58cb63583450e02a92f62a56d4ccfe4_2.png)

上一节我们介绍了选品与上架流程，本节中我们来看看如何利用营销工具促进销售。

![](img/f58cb63583450e02a92f62a56d4ccfe4_6.png)

营销工具位于小红书商家后台。在后台点击“营销”即可进入。

![](img/f58cb63583450e02a92f62a56d4ccfe4_8.png)

![](img/f58cb63583450e02a92f62a56d4ccfe4_0.png)

![](img/f58cb63583450e02a92f62a56d4ccfe4_10.png)

后台主要有“活动报名”和“营销管理”两大板块。“活动报名”使用频率不高，我们主要关注“营销管理”。

![](img/f58cb63583450e02a92f62a56d4ccfe4_4.png)

![](img/f58cb63583450e02a92f62a56d4ccfe4_12.png)

在“营销管理”中，核心工具是“商品促销”和“属卷运营中心”。“属卷”即优惠券。

![](img/f58cb63583450e02a92f62a56d4ccfe4_14.png)

![](img/f58cb63583450e02a92f62a56d4ccfe4_8.png)

![](img/f58cb63583450e02a92f62a56d4ccfe4_16.png)

小红书的营销工具相对简洁，主要分为**打折**和**优惠券**两种形式。

![](img/f58cb63583450e02a92f62a56d4ccfe4_18.png)

---

![](img/f58cb63583450e02a92f62a56d4ccfe4_20.png)

了解了营销工具的入口后，我们先来看商品促销的设置。

商品上架后，建议设置促销。单品促销操作较繁琐，推荐使用“多品”或“全店”促销。

![](img/f58cb63583450e02a92f62a56d4ccfe4_22.png)

以下是创建全店促销的步骤：
1.  选择“全店活动”。
2.  设置活动时间。
3.  选择“全部用户”和“全部商品”。
4.  选择促销类型，如“满件打折”、“满额立减”或“满减打折”。
5.  设置具体规则，例如“满1件打7折”。
6.  点击保存。

![](img/f58cb63583450e02a92f62a56d4ccfe4_24.png)

![](img/f58cb63583450e02a92f62a56d4ccfe4_20.png)

![](img/f58cb63583450e02a92f62a56d4ccfe4_26.png)

这样，一个全店范围的打折活动就创建完成了。

![](img/f58cb63583450e02a92f62a56d4ccfe4_28.png)

设置完商品促销，我们再来看看如何利用优惠券进一步刺激消费。

![](img/f58cb63583450e02a92f62a56d4ccfe4_30.png)

优惠券可以显著提高转化率。页面显示的促销工具越多，顾客的下单欲望通常越强。

在“属卷运营中心”可以创建优惠券。例如，如果你的定价策略是“两倍加五”，可以额外设置一张“满10元减5元”的优惠券。

优惠券能提升“客单价”感知。例如，一个标价50元的产品，使用10元优惠券后实付40元，顾客会认为以优惠价买到了价值50元的产品，从而促进购买。

![](img/f58cb63583450e02a92f62a56d4ccfe4_32.png)

以下是创建优惠券的建议：
*   **类型**：可以创建“满减券”或“无门槛券”。
*   **范围**：设置为“全店通用”即可。
*   **示例**：若定价为“两倍加五”，可设置“满10减5”的优惠券。也可以根据客单价设置，如“满19减10”。

![](img/f58cb63583450e02a92f62a56d4ccfe4_32.png)

![](img/f58cb63583450e02a92f62a56d4ccfe4_34.png)

![](img/f58cb63583450e02a92f62a56d4ccfe4_36.png)

除了基础的打折和优惠券，平台还提供了一些互动营销工具。

![](img/f58cb63583450e02a92f62a56d4ccfe4_38.png)

在“营销管理”中，还有“晒单有礼”和“点评有礼”等功能。

“晒单有礼”是鼓励顾客发布带图评价，奖励通常是优惠券，但对顾客吸引力有限，效果一般。

“点评有礼”是鼓励顾客进行文字评价，奖励是**无门槛优惠券**（如5元或10元）。这对顾客的吸引力更大，商家可以设置发行张数和适用商品，用以测试效果。

![](img/f58cb63583450e02a92f62a56d4ccfe4_40.png)

**注意**：这类活动的成本（优惠券面额）需由商家承担。

![](img/f58cb63583450e02a92f62a56d4ccfe4_42.png)

---

最后，对于新店来说，充分利用平台扶持至关重要。

![](img/f58cb63583450e02a92f62a56d4ccfe4_44.png)

在手机端“小红书商家版”APP的首页，有一个“成长任务”板块。这是平台对新店的流量扶持计划。

建议在上架第一批商品后，积极参与这些任务。任务通常包括：
*   发布一定数量的商品笔记。
*   笔记流量达到某个目标（如超过1000）。
*   达成一定的成交额。

完成不同阶段的任务，可以获得相应的流量奖励和功能解锁，对新店起步非常友好。

![](img/f58cb63583450e02a92f62a56d4ccfe4_46.png)

![](img/f58cb63583450e02a92f62a56d4ccfe4_46.png)

本节课中我们一起学习了小红书店铺的核心营销工具与活动。
1.  **商品促销**：通过设置全店或单品打折（如`满1件打7折`）来吸引顾客。
2.  **优惠券**：创建满减券或无门槛券（如`满10减5`），提升转化率和客单价感知。
3.  **互动活动**：可尝试“点评有礼”发放无门槛券，激励好评。
4.  **新店任务**：务必参与手机端APP的“成长任务”，获取平台流量扶持。

![](img/f58cb63583450e02a92f62a56d4ccfe4_50.png)

合理搭配使用这些工具和活动，能为你的店铺运营打下坚实基础。
