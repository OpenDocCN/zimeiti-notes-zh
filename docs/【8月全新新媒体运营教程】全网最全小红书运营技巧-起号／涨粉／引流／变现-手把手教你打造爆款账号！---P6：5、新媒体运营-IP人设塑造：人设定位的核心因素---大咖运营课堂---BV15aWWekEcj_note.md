1.  **角色**
    *   **定义**：即你在特定赛道中以何种身份提供内容。你需要让用户相信你提供的内容具有专业性。
    *   **示例**：
        *   **好物分享/测评**：专业测评师。
        *   **装修博主**：真实记录自家装修过程、分享避坑经验的屋主。
        *   **母婴赛道**：可以是分享育儿经验的**宝妈**，也可以是提供专业知识的**育儿护理专家**。
        *   **情感赛道**：可以是**心理学顾问**，也可以是分享自身成长经历的**成长型人设**（例如：分享职场、创业经历的30岁女性）。

2.  **内容**
    *   **定义**：你持续提供的、与所选赛道高度相关的具体信息。
    *   **核心要求**：内容需吸引用户、有价值，并在平台上有一定特色。这直接关系到账号的垂直度和标签。
    *   **重要性公式**：`内容垂直度 = 平台精准标签 = 更有效的流量分发`
    *   **反面示例**：如果首篇内容讲儿童教育，第二篇讲宠物Vlog，平台会将内容推给不同兴趣的用户，可能导致数据不佳，影响推荐效果。

3.  **形式**
    *   **定义**：内容呈现的具体方式，主要包括产出场景、声音和内容形式。
    *   **产出场景**：室内固定场景、室外多变场景、车内、上班路上等。可参考对标账号。
    *   **声音**：本人出镜讲解，或不出镜仅使用配音。
    *   **内容形式**：
        *   **剧情型**：创作难度较高。
        *   **干货/经验分享型**：需要精心准备文案脚本。
        *   **生活记录型**：需要积累大量拍摄素材。

4.  **记忆点**
    *   **定义**：能让用户快速记住并识别你的独特元素。
    *   **设计原则**：需要刻意设计并反复强化，植入用户心智。
    *   **记忆点清单**：
        *   **视频前几秒的固定元素**（需前置）：固定动作、特效、开场白。
        *   **贯穿视频全程的元素**：统一的BGM（背景音乐）。
        *   **贯穿内容全局的元素**：统一出现的物品、统一风格的服装。
