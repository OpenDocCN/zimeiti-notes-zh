# 亚马逊运营：P1：竞品站外推广渠道反查与分析教程

在本节课中，我们将学习如何快速反查竞争对手在亚马逊站外做了哪些推广活动，并分析其推广效果。掌握这一技能，有助于你了解市场动态，优化自己的营销策略。

上一节我们介绍了课程目标，本节中我们来看看具体的操作步骤与工具。

## 🎼 第一步：使用工具查询竞品站外记录

首先，你需要获取一个免费查询工具。复制竞品亚马逊商品的ASIN码，打开该工具。

![](img/90257cb47cb12f7023dd7f9d1461c9f7_1.png)

以下是具体操作步骤：
1.  将复制的ASIN粘贴到工具的指定输入框中。
2.  选择该商品对应的亚马逊站点（如美国站、英国站）。
3.  点击“折扣查询”按钮。
4.  系统将展示该商品的历史站外推广记录。

![](img/90257cb47cb12f7023dd7f9d1461c9f7_1.png)

![](img/90257cb47cb12f7023dd7f9d1461c9f7_2.png)

![](img/90257cb47cb12f7023dd7f9d1461c9f7_2.png)

## 🎼 第二步：分析站外推广详情

查询到记录后，你可以点击“查看详情”来深入分析每一次推广的具体情况。

详情页面通常包含以下核心信息：
*   **折扣力度**：推广时提供的具体折扣比例或金额。
*   **发帖日期**：推广活动发布的具体时间。
*   **排名变化**：推广前后商品在亚马逊BSR榜单上的排名波动情况。

![](img/90257cb47cb12f7023dd7f9d1461c9f7_4.png)

分析这些数据可以帮助你判断何种折扣力度和推广时机效果更佳。

![](img/90257cb47cb12f7023dd7f9d1461c9f7_4.png)

## 🎼 第三步：获取站外推广链接

除了查看汇总数据，该工具还提供“外链查询”功能，能帮你找到原始的推广内容。

以下是操作流程：
1.  在工具的“外链查询”区域输入商品ASIN。
2.  点击查询，系统会列出该商品所有的站外推广帖子地址。
3.  直接点击列表中的链接，即可跳转到对应的站外推广页面（如Deal站、社交媒体帖子等），观察其推广文案和用户互动情况。

![](img/90257cb47cb12f7023dd7f9d1461c9f7_6.png)

![](img/90257cb47cb12f7023dd7f9d1461c9f7_6.png)

本节课中我们一起学习了如何利用免费工具，通过竞品的ASIN码反查其站外推广渠道、分析折扣力度与排名变化，并直接访问推广原帖。这套方法能有效帮助你洞察竞争对手的营销策略，为自己的站外推广计划提供数据参考。
