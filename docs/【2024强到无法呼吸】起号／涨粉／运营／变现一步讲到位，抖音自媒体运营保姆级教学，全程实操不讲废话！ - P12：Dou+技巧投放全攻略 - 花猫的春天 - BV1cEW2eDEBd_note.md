## 第二阶段：3000粉丝以内（定向投放）

*   **进阶关键**：根据自己账号的内容受众，添加**年龄、性别、地域**等定向条件。
    *   **示例**：如果你是“家居好物”类账号，主要受众可能是**女性，30-40岁**。
    *   **公式**：`投放受众 = 达人相似粉丝 + 自定义定向条件（如：性别=女，年龄=30-40）`
*   **目的**：这样操作能吸引更精准的粉丝，快速为账号打上内容标签，加速向1万粉丝的目标冲刺。这是许多付费课程不会透露的关键步骤。
