# 抖音运营教程：P45：如何衡量选题质量

在本节课中，我们将学习如何判断一个短视频选题是否优秀。我们将介绍五个核心的衡量标准，帮助你从众多想法中筛选出最具潜力的选题，从而提升视频的吸引力和传播效果。

上一节我们介绍了如何寻找和拓展选题。本节中，我们来看看如何评估这些选题的质量。

以下是衡量选题好坏的五个核心标准：

![](img/e919add17d7eb96a34d40d04876ad790_1.png)

1.  **低成本**：用户希望以最小的代价（如时间、精力、金钱）获得期望的效果。例如，“玩手机躺着也能减肥”就比“50种高效减肥法”更具吸引力。公式可表示为：**用户采纳意愿 ∝ 1 / 感知成本**。
2.  **戳中痛点**：选题需要直击用户最迫切、最强烈的需求或烦恼。例如，对于已经看中某款口红的用户，“如何领取大额优惠券”就比“口红色号推荐”更能打动她。
3.  **场景熟悉**：选题基于用户日常生活中常见、高频率出现的场景，容易引发共鸣。例如，“电影院里的那些烦人行为”或“妈妈常说的几句话”。
4.  **受众精准**：选题内容明确指向特定人群的兴趣和需求。例如，“新手必备化妆流程”就精准定位了化妆初学者。
5.  **需求高频**：选题涉及的需求在目标用户生活中出现频率较高。例如，“厨房收纳”比“婚纱挑选”对大多数人而言需求更频繁。

在实际操作中，**低成本**和**戳中痛点**是最常用且有效的两个衡量维度。一个选题若能同时满足这两点，往往能获得很好的反馈。

---

我们如何应用这些标准呢？当你通过九宫格等方法产生一系列选题后，可以逐一用这五个标准进行检验。

以下是具体的筛选步骤：

*   将每个选题对照上述五个标准进行打分或勾选。
*   通常，一个选题若能符合其中两到三点，就可以被视为一个合格的、值得尝试的好选题。
*   如果所有选题看起来都不够理想，可以回到文案脚本阶段进行优化，尝试从不同角度切入，使其更符合“低成本”或“痛点”原则。

本节课中我们一起学习了衡量短视频选题质量的五个核心标准：**低成本**、**戳中痛点**、**场景熟悉**、**受众精准**和**需求高频**。掌握这些标准，能帮助你在创作初期有效筛选出更具爆款潜力的选题，避免盲目拍摄，提升运营效率。记住，好的开始是成功的一半，一个优秀的选题是视频获得流量的重要基石。
