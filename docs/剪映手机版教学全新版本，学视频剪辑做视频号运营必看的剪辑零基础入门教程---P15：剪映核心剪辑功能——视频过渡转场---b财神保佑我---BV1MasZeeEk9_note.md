# 剪映手机版教学：P15：视频过渡转场核心功能 🎬

在本节课中，我们将要学习剪映的核心剪辑功能之一——视频的过渡与转场。通过学习如何运用这些技巧，可以让你的视频镜头衔接更加自然流畅，甚至制作出炫酷的卡点视频。

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_1.png)

视频过渡转场是指在两个视频镜头之间，为了使衔接更自然、避免生硬感而进行的视频处理技巧。在短视频制作中，巧妙运用转场可以极大提升视频的观赏性和专业感。接下来，我们将通过一个卡点视频的实例，详细介绍如何在剪映中操作。

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_3.png)

## 准备工作：创建项目与导入素材

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_5.png)

首先，我们创建一个新项目并导入制作卡点视频所需的照片素材。



![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_7.png)

紧接着，我们需要为视频搭配一个节奏感强的背景音乐。卡点视频的精髓在于画面切换与音乐节拍同步。



![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_9.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_10.png)

以下是选择并添加音乐的步骤：
1.  点击底部工具栏的 **`音频`** 按钮。
2.  选择 **`抖音收藏`**，导入你提前收藏好的动感音乐。
3.  点击 **`使用`** 按钮，将音乐添加到时间轴。

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_12.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_13.png)

## 核心步骤：音乐踩点与素材对齐

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_15.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_16.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_17.png)

音乐导入后，下一步是进行“踩点”，即标记出音乐中的重拍节奏，以便让照片的切换对准这些节拍。



![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_19.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_21.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_23.png)

为音频轨道设置踩点的具体操作如下：
1.  选中时间轴上的音频轨道。
2.  点击底部工具栏的 **`踩点`** 按钮。
3.  开启 **`自动踩点`** 功能。剪映提供“节拍1”和“节拍2”两种模式，“节拍2”的节奏点更密集，适合制作快速炫酷的卡点效果。
4.  系统会自动在音轨上生成黄色标记点，点击对号确认。

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_25.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_27.png)

踩点完成后，时间轴的音频轨道上会出现一系列黄色小点，这些就是我们需要对齐的节奏点。



![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_29.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_31.png)

现在，我们需要调整每张照片的时长，使其结尾与这些节奏点对齐。
1.  选中第一张照片，拖动其右边缘，与第一个节奏点的时间线对齐。
2.  依次调整后续每张照片的时长，使其都与下一个节奏点对齐。

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_33.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_35.png)

## 核心技巧：添加转场效果

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_37.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_38.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_40.png)

调整好素材时长后，两个镜头之间的衔接点会显示为一个小方块。这就是我们添加转场效果的位置。如果不添加转场，视频的切换会显得平淡且缺乏节奏感。



![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_42.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_43.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_45.png)

为视频添加转场的步骤如下：
1.  点击两个素材之间的小方块。
2.  底部会弹出转场效果库。剪映提供了丰富的转场类型，包括：
    *   **基础转场**：如叠化、闪黑。
    *   **运镜转场**：如推进、拉远、旋转，能模拟摄像机运动效果。
    *   **幻灯片**：如翻页、立方体。
    *   **特效转场**：如模糊、闪光。
    *   **遮罩转场**：如圆形、星形划像。
3.  选择你喜欢的转场效果（例如“运镜转场”中的“拉远”），并可为每个衔接点设置不同的转场，让视频更富变化。

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_46.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_48.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_49.png)

添加转场后，预览视频，你会发现镜头切换变得动感十足。

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_51.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_52.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_54.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_56.png)

## 优化调整：统一画幅与添加特效

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_57.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_59.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_60.png)

为了让视频视觉效果更统一、更专业，我们还需要进行两项优化。

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_62.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_63.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_64.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_65.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_66.png)

首先，调整照片的画幅比例。由于导入的照片尺寸可能不一，我们需要将其统一。
1.  点击底部工具栏的 **`比例`** 按钮。
2.  选择 **`9:16`** 的竖屏比例（适用于抖音等平台）。
3.  双指在预览窗口中缩放和移动每张照片，使其适配画布，避免出现黑边或裁剪不当。



![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_67.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_69.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_70.png)

其次，可以添加特效来进一步提升视频的炫酷程度。
1.  点击底部工具栏的 **`特效`** 按钮。
2.  在 **`画面特效`** 中，选择适合的动态特效，如“变清晰”、“抖动”或“镜片”。
3.  将特效拖拽到时间轴的照片片段上方，并调整其时长，使其与对应的画面节奏匹配。

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_72.png)

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_74.png)

## 课程总结 🎯

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_76.png)

本节课中，我们一起学习了剪映中视频过渡与转场的核心功能。
*   **转场的作用**：使视频镜头间的衔接更自然流畅，提升视频节奏感和观赏性。
*   **关键操作**：通过点击素材间的衔接方块，可以调用丰富的转场效果库。
*   **制作流程**：制作卡点视频需遵循“导入素材 -> 添加节奏音乐 -> 音乐踩点 -> 对齐素材 -> 添加转场 -> 优化画幅与特效”的流程。
*   **效果库**：剪映提供了**基础转场**、**运镜转场**、**幻灯片**、**特效转场**等多种类型，足以满足大部分视频的创作需求。

![](img/c7b6b2439f6f0b71a4bd156f4f98efb7_78.png)

熟练掌握转场技巧，能让你轻松制作出吸引眼球的卡点视频和运镜视频，为你的视频号运营增添亮点。
