## 变换音频 🎵

![](img/c1d81eb5c4163a433dd633d52c7df4fe_20.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_22.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_24.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_26.png)

变换音频是指将一个视频的画面与另一个视频的声音结合到一起的操作。例如，使用视频A的画面，但配上视频B的声音。

![](img/c1d81eb5c4163a433dd633d52c7df4fe_28.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_30.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_32.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_33.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_35.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_37.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_39.png)

以下是操作步骤：

![](img/c1d81eb5c4163a433dd633d52c7df4fe_41.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_43.png)

1.  **导入视频素材**：打开剪映，点击“开始创作”，导入你想要的视频画面（例如视频A）。
2.  **关闭原声**：选中视频轨道，点击左下角的“关闭原声”按钮（小喇叭图标），将视频A的原始声音关闭。
3.  **添加新音频**：点击底部工具栏的“音频”。
4.  **提取音乐**：在音频选项中，选择“提取音乐”。系统会跳转到你的相册，选择包含你想要声音的视频（例如视频B）。注意，这里选择的是“仅导入视频的声音”。
5.  **对齐音频与视频**：导入音频后，检查音频轨道和视频轨道的长度。如果音频比视频长，选中音频轨道，将播放指针移动到视频结束的位置，点击“分割”，然后删除多余的部分。确保两者长度一致。
6.  **导出并重新导入以识别字幕**：点击右上角“导出”。完成导出后，再次点击“开始创作”，将刚导出的视频重新导入。这样，音频和视频就合并为一个文件了。
7.  **识别字幕**：点击底部“文字”，选择“识别字幕”，系统会自动识别音频中的语音并生成字幕。识别出的字幕可以批量调整样式、大小和位置。

![](img/c1d81eb5c4163a433dd633d52c7df4fe_45.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_47.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_49.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_51.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_53.png)

**核心概念**：`新视频 = 视频A.画面 + 视频B.声音`

![](img/c1d81eb5c4163a433dd633d52c7df4fe_55.png)

## 三屏视频制作 🖼️

![](img/c1d81eb5c4163a433dd633d52c7df4fe_57.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_59.png)

三屏视频是指一个画面被平均分割成三个相同部分并列播放的效果。

![](img/c1d81eb5c4163a433dd633d52c7df4fe_61.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_63.png)

以下是制作三屏视频的步骤：

![](img/c1d81eb5c4163a433dd633d52c7df4fe_65.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_67.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_69.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_71.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_73.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_75.png)

1.  **准备横屏素材**：确保你的视频素材是横屏（16:9比例）。竖屏素材无法做出理想的三屏效果。
2.  **导入并调整比例**：将横屏素材导入剪映。在底部工具栏找到“比例”，选择“9:16”（竖屏比例）。此时视频会居中显示，上下出现黑边。
3.  **首次导出**：直接点击右上角“导出”。这一步是为了固定视频的竖屏格式。
4.  **重新导入并添加特效**：将导出的视频再次导入剪映。点击底部“特效”，选择“画面特效”。
5.  **应用分屏特效**：在特效分类中找到“分屏”，然后选择“三屏”特效。
6.  **调整特效时长**：添加特效后，特效轨道默认可能只覆盖视频开头部分。选中特效轨道，拖动其尾部的白色方块，使其长度与视频轨道完全一致，这样整个视频都会应用三屏效果。

![](img/c1d81eb5c4163a433dd633d52c7df4fe_77.png)

**核心概念**：`三屏效果 = 横屏素材 -> 调整比例为9:16 -> 导出 -> 重新导入 -> 添加“三屏”特效`

![](img/c1d81eb5c4163a433dd633d52c7df4fe_79.png)

## 抠像功能 ✂️

![](img/c1d81eb5c4163a433dd633d52c7df4fe_81.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_91.png)

抠像功能可以帮助你将视频中的主体（如人物）或特定颜色背景分离出来。主要有两种方式：

![](img/c1d81eb5c4163a433dd633d52c7df4fe_93.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_95.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_97.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_99.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_101.png)

*   **智能抠像**：适用于人物、动物或动画主体。选中视频轨道，点击底部“剪辑”，找到“智能抠像”，点击后系统会自动识别人物并去除背景。
*   **色度抠图**：适用于纯色背景（如绿幕、蓝幕）的视频或图片。选中素材，点击“剪辑”->“色度抠图”。移动取色器（圆圈中心的小方块）对准要抠除的颜色（如绿色），然后调整“强度”滑块，直到背景被干净地去除。

![](img/c1d81eb5c4163a433dd633d52c7df4fe_103.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_105.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_107.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_109.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_111.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_113.png)

**核心概念**：
*   智能抠像：`输出 = 智能抠像(人物视频)`
*   色度抠图：`输出 = 色度抠图(绿幕素材， 取色绿色， 强度=50)`

## 封面制作 🖼️✨

![](img/c1d81eb5c4163a433dd633d52c7df4fe_125.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_127.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_129.png)

一个统一、整洁的作品封面能提升账号主页的专业度。

![](img/c1d81eb5c4163a433dd633d52c7df4fe_131.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_133.png)

在剪映中设置封面的方法如下：

![](img/c1d81eb5c4163a433dd633d52c7df4fe_135.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_137.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_139.png)

1.  视频编辑界面，点击右上角的“设置封面”。
2.  你有两个选择：
    *   **视频帧**：从当前视频中滑动选取一帧画面作为封面。
    *   **相册导入**：从手机相册中选择一张图片作为封面。
3.  选择好画面后，**务必点击右上角的“保存”**，封面才会生效。

![](img/c1d81eb5c4163a433dd633d52c7df4fe_141.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_143.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_145.png)

**注意**：在抖音发布时，请确保在发布页关闭“动态封面”选项，否则抖音可能会自动生成一个动态封面，覆盖你在剪映中设置的静态封面。

## 背景修改 🎨

![](img/c1d81eb5c4163a433dd633d52c7df4fe_147.png)

此功能可以为横屏视频在调整为竖屏后，填充上下黑边的区域。

![](img/c1d81eb5c4163a433dd633d52c7df4fe_149.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_151.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_153.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_155.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_157.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_159.png)

**前提**：必须使用横屏（16:9）素材。

![](img/c1d81eb5c4163a433dd633d52c7df4fe_161.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_163.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_165.png)

操作步骤：

![](img/c1d81eb5c4163a433dd633d52c7df4fe_167.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_169.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_171.png)

1.  导入横屏素材，在“比例”中选择“9:16”。
2.  点击底部“背景”。
3.  有三种背景模式可选：
    *   **画布颜色**：选择纯色填充背景。
    *   **画布样式**：使用剪映提供的图案或从相册导入图片作为背景。
    *   **画布模糊**：将视频画面进行模糊处理后作为背景，有四种模糊程度可选。这是制作“中间清晰、上下模糊”效果的方法。

![](img/c1d81eb5c4163a433dd633d52c7df4fe_173.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_175.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_177.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_179.png)

**核心概念**：`新背景 = 画布模糊(原视频画面， 模糊度)`

![](img/c1d81eb5c4163a433dd633d52c7df4fe_183.png)

## 课程总结 🎉

![](img/c1d81eb5c4163a433dd633d52c7df4fe_189.png)

本节课我们一起学习了手机剪映的五个实用功能：
1.  **变换音频**：将不同视频的画面和声音进行组合。
2.  **三屏制作**：利用特效将横屏视频制作成流行的三屏竖版效果。
3.  **抠像功能**：使用智能抠像和色度抠图分离视频主体。
4.  **封面制作**：为视频设置统一、美观的静态封面。
5.  **背景修改**：为横屏视频转换比例后填充或美化背景区域。

![](img/c1d81eb5c4163a433dd633d52c7df4fe_191.png)

![](img/c1d81eb5c4163a433dd633d52c7df4fe_193.png)

请大家课后多加练习，特别是三屏视频的制作，这将是我们本节课的实践作业。熟练掌握这些功能，能让你的视频剪辑更加得心应手。
