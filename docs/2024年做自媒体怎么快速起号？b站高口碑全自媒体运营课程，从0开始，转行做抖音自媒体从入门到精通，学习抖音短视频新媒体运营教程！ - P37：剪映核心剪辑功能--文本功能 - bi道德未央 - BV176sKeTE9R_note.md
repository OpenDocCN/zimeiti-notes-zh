# 自媒体运营课程：P37：剪映核心剪辑功能--文本功能 📝

在本节课中，我们将要学习剪映App中一个至关重要的功能——文本功能。文本是视频中传递信息、突出主题的关键元素。我们将从如何添加文本开始，逐步掌握字幕的样式设置、自动识别以及贴纸应用等核心技巧，帮助你高效地为视频添加专业的字幕和文字效果。

---

## 文本功能概述

![](img/de30efc3be770061462dd76b6888dc27_1.png)

一条完整的视频，除了画面和背景音乐，另一个非常重要的元素就是文本或字幕。通过添加文本或字幕，可以更准确地传递视频想要表达的主题、思想及相关信息。这能让观众更明确、更快速地理解视频内容。

本节课将讲解如何通过剪映为视频添加文本，并介绍剪映中文本功能的使用方法和技巧。



![](img/de30efc3be770061462dd76b6888dc27_1.png)

在剪映中，我们可以自由地使用文本功能为视频添加字幕。添加的字幕可以单独设置样式，包括字体、字号、颜色、间距以及字幕动画等。

此外，剪映中有两个非常高效的功能：**字幕的自动识别**和**歌词识别**功能。添加文本信息可以是手动的。例如，在配置好画外音或音频后，可以通过“识别字幕”或“识别歌词”的方式，一键为视频快速添加上相应的字幕。这大大降低了操作难度和时间成本。

最后，在文本功能中还可以添加贴纸。关于文本功能的基本介绍就到这里，接下来进入功能实操演练环节。

![](img/de30efc3be770061462dd76b6888dc27_3.png)

文本功能同样位于添加视频后的一级菜单中。点击“文字”功能，会出现四个选项：**新建文本**、**识别字幕**、**识别歌词**和**贴纸**。下面进行详细操作演示。



## 功能实操演练

### 1. 新建文本（手动添加字幕）

我们以一条已添加音频的视频为例进行演示。首先，我们需要为这条视频植入字幕。

回到一级菜单，点击底部菜单栏的“文字”选项。然后选择“新建文本”，直接输入文字内容。例如，输入“剪映视频剪辑教程”。

输入完成后，时间线上会添加一个文字轨道。我们可以对文字的样式进行设置。点击底部的“样式”，可以对文字的**字体**、**描边**、**标签**、**阴影**、**字间距**、**对齐方式**等进行单独设置。

以下是可调整的部分样式参数示例：
*   **字体**：例如选择“新青年体”。
*   **描边**：可以设置描边的颜色（如红色、蓝色）和粗细。
*   **花字**：可以使用抖音官方推出的各种特色“花字”样式。
*   **气泡**：可以选择对话形式的气泡样式来包装文字。
*   **动画**：可以为文字添加入场动画（如螺旋上升）和出场动画，并设置动画时长。

设置完成后保存即可预览效果。这是手动输入字幕的方式。



![](img/de30efc3be770061462dd76b6888dc27_5.png)

### 2. 识别字幕（为语音自动添加字幕）

![](img/de30efc3be770061462dd76b6888dc27_5.png)

有些视频本身带有语音对话。这种情况下，我们可以使用“识别字幕”功能来快速添加字幕。

例如，我们为视频录制一段画外音：“剪映视频剪辑教程”。添加这段音频后，回到文本界面，点击“文字”，然后选择“识别字幕”。



![](img/de30efc3be770061462dd76b6888dc27_7.png)

![](img/de30efc3be770061462dd76b6888dc27_8.png)

![](img/de30efc3be770061462dd76b6888dc27_7.png)

点击“开始识别”，系统会自动识别音频内容并生成字幕轨道。预览视频，即可看到字幕已自动添加。这是通过识别形式一键快速添加字幕的方法。

![](img/de30efc3be770061462dd76b6888dc27_8.png)

### 3. 识别歌词（为歌曲添加歌词字幕）

如果视频中添加了带有歌词的歌曲，可以使用“识别歌词”功能快速植入歌词字幕。

删除之前的文字，为视频添加一首歌曲（例如从抖音收藏中选择）。确保歌曲有歌词。



![](img/de30efc3be770061462dd76b6888dc27_10.png)

![](img/de30efc3be770061462dd76b6888dc27_10.png)

![](img/de30efc3be770061462dd76b6888dc27_12.png)

导入并对齐音频后，回到文本界面，选择“文字”下的“歌词识别”。



![](img/de30efc3be770061462dd76b6888dc27_14.png)

![](img/de30efc3be770061462dd76b6888dc27_14.png)

![](img/de30efc3be770061462dd76b6888dc27_16.png)

![](img/de30efc3be770061462dd76b6888dc27_16.png)

点击“开始识别”。识别完成后，视频轨道下方会出现歌词文字轨道，画面中也会显示歌词。同样，我们可以对歌词的样式进行个性化设置，如调整位置、更换字体、添加描边和动画，以提升视频的意境和观感。



![](img/de30efc3be770061462dd76b6888dc27_18.png)

![](img/de30efc3be770061462dd76b6888dc27_19.png)

视频中的所有文本信息添加并设置完成后，即可导出视频，完成制作。



![](img/de30efc3be770061462dd76b6888dc27_18.png)
