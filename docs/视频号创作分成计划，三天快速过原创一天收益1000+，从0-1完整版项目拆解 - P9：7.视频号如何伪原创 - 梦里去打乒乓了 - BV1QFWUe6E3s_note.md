# 视频号创作分成计划：P9：7.视频号如何伪原创 🎬

在本节课中，我们将学习视频号平台对原创内容的扶持政策，并重点拆解如何通过“伪原创”的方式，高效创作内容以获取流量。对于没有原创能力的新手，掌握“伪原创”的流程至关重要。

上一节我们介绍了视频号发布作品的基础，本节中我们来看看平台对原创内容的激励措施。

## 平台原创激励政策 📈

平台支持原创，并设有创作者激励计划，对原创内容提供流量倾斜。该计划包括流量扶持，目标是让1000万原创作者获得流量，让百万优质创作者获得收入。

![](img/48a29c9ca376031a4bd024fdc2418aa4_1.png)

在零粉丝阶段，平台会赠送2500到5万点的流量券，相当于直接推送观众。达到千粉阶段后，可加入重点作者扶持计划，优质视频可以申报专项奖金。达到万粉乃至百万粉阶段，申报奖金的机会更多。

因此，原创内容能获得平台一级一级的流量池推进，效果较好。但这需要个人具备相应的技术能力，例如会拍摄、会编写剧本。

## 伪原创的可行性与案例 📊

![](img/48a29c9ca376031a4bd024fdc2418aa4_6.png)

许多同学表示没有做过原创，现在开始做会比较困难。大部分学员是新手。因此，今天的重点讲解搬运与二次创作。

如果没有内容，不知道如何运营账号，可以借鉴优质账号的爆款内容进行二次创作。这样能产生新内容，带来新的流量。我们追求的就是流量。

![](img/48a29c9ca376031a4bd024fdc2418aa4_8.png)

以下是搬运可行的案例证明：
*   **案例背景**：一位名叫“大学生中娜”的同学，在疫情期间封校，课余时间尝试操作视频号。
*   **操作过程**：在指导下，通过搬运内容并蹭当时的热点（如刘畊宏健身操）来发布作品。
*   **成果展示**：其账号作品数据如下（见原文图片）：
    *   一个搬运的视频获得9.4万点赞。
    *   另一个搬运的视频获得4.1万点赞。
    *   其他作品数据有高有低，证明搬运也能获得可观的流量。

这个案例说明，通过搬运并进行适当处理，同样可以获得流量。

## 伪原创的核心流程 🔄

搬运并非直接下载发布。平台支持原创，直接搬运带水印的内容会导致账号违规。因此，“伪原创”需要进行处理。

![](img/48a29c9ca376031a4bd024fdc2418aa4_10.png)

以下是伪原创的标准操作流程：

![](img/48a29c9ca376031a4bd024fdc2418aa4_11.png)

**第一步：选品与定位**
首先确定你要售卖的商品。确定货品后，再寻找相关的视频内容。可以在视频号创作者中心、阿里巴巴1688、拼多多等平台选品。
例如，若想做音乐类账号，可以卖车载U盘或接音乐广告；做影视剧账号，可以接广告或引流卖货。根据要卖的商品来确定账号内容类型。

**第二步：寻找灵感与素材**
可以借助一些数据平台寻找灵感和优质账号。例如：
*   在视频号内查看各类目（如影视、音乐）的排行榜。
*   使用“西瓜数据”、“有赚数据”、“微信指数”等工具。
*   进入排行榜中的优质账号，学习其内容优点，借鉴思路。

**第三步：验证与获取素材**
在抖音等其他平台发现优质账号和作品后，**关键一步**是验证该内容是否已在视频号被搬运过。如果已有人搬运，再次发布容易造成内容重复。
确认无人搬运后，即可确定该素材。获取素材时，**首要步骤是去除原视频的水印**。

**第四步：二次剪辑**
去除水印后，需要对素材进行二次剪辑，使其看起来像是新创作的“原创”内容。这包括但不限于：调整片段顺序、添加新的字幕或特效、修改背景音乐、混剪多个素材等。公式可以概括为：
`伪原创作品 = 去水印的优质素材 + 个性化的二次剪辑`

![](img/48a29c9ca376031a4bd024fdc2418aa4_13.png)

本节课中我们一起学习了视频号平台的原创激励政策，并通过具体案例了解了“伪原创”的可行性。核心内容是掌握伪原创的标准四步流程：**选品定位 -> 寻找素材 -> 验证并去水印 -> 二次剪辑**。遵循这个流程，即使新手也能高效地产出内容，获取平台流量。
