# 微信视频号运营从0~1全攻略：P25：短视频变声与配音的两种方法 🎬

在本节课中，我们将学习在视频混剪时，如何对视频的声音进行去重处理。我们将介绍两种实用方法：第一种是直接在剪映App内使用变声功能；第二种是通过配音神器提取文案并生成全新配音。这两种方法都能有效帮助你的视频通过平台审核。

## 方法一：使用剪映变声功能

上一节我们介绍了视频剪辑的基础操作，本节中我们来看看如何利用剪映自带的工具修改人声。这种方法操作简单，能快速改变原视频的人声效果。

首先，将需要处理的视频导入剪映。选中视频轨道，在左侧功能栏中找到“声音效果”选项。

以下是具体操作步骤：
1.  点击“声音效果”，进入变声菜单。
2.  菜单中提供了多种免费变声选项，如“圣诞老人”、“台湾小哥”等。
3.  选择你喜欢的音效，例如“男生-大叔”。
4.  部分音效支持参数微调，你可以根据预览效果调整音调等参数。

![](img/b05e3ef0ceec8742494a6ebab7a0c3e9_1.png)

通过以上步骤，原视频的人声就被改变了，这能起到一定的去重作用。但需要注意的是，这种方法**只改变了人声，视频的背景音乐保持不变**。

![](img/b05e3ef0ceec8742494a6ebab7a0c3e9_1.png)

## 方法二：使用配音神器生成全新配音

如果希望声音去重效果更彻底，我们可以采用第二种方法。这种方法的核心是**提取原视频文案 -> 修改文案 -> 生成全新配音**，从而完全替换掉原视频的所有音频。

首先，需要使用配音神器这类工具。打开工具，从手机相册导入目标视频，使用“文案提取”功能获取视频中的解说文字。

获取文案后，建议对文案进行适当修改，这能进一步增强去重效果。

以下是修改文案的注意事项：
*   修改时尽量保持原意，避免过多增删字数。
*   可以替换部分词语，例如将“老人”改为“大人”。
*   可以调整句式，例如将“怎么办呢？”改为“咋办呢？”。
*   目标是让文案表达相同意思，但用词和语序有所不同。

文案修改完成后，将其复制到配音神器的“制作配音”页面。接下来，选择一个新的配音员声音，例如选择一个不同的男声，然后点击“合成配音”。合成完成后，导出并下载这段全新的音频文件。

![](img/b05e3ef0ceec8742494a6ebab7a0c3e9_3.png)

![](img/b05e3ef0ceec8742494a6bab7a0c3e9_3.png)

现在，我们回到剪映进行最终合成。新建项目，导入原视频，然后**关闭视频的原声**。接着，点击“添加音频 -> 提取音乐”，导入我们刚刚生成的全新配音文件。

![](img/b05e3ef0ceec8742494a6ebab7a0c3e9_5.png)

![](img/b05e3ef0ceec8742494a6bab7a0c3e9_5.png)

至此，原视频的人声和背景音乐都已被替换。你还可以在此基础上，添加新的背景音乐，并调整音乐音量，使其不影响主配音。

![](img/b05e3ef0ceec8742494a6ebab7a0c3e9_7.png)

![](img/b05e3ef0ceec8742494a6bab7a0c3e9_7.png)

最后，你可以利用剪映的“识别字幕”功能，为新的配音生成字幕。之后，便可以按照之前所学的混剪方法，继续完成视频的后续制作。

本节课中我们一起学习了两种为短视频变声和配音的方法。第一种方法在剪映内直接变声，快捷但保留背景音乐；第二种方法通过外部工具生成全新配音，去重更彻底。你可以根据实际需要灵活选择使用，让制作的视频内容更具独特性。
