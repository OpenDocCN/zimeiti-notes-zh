在本节课中，我们将学习如何制作一个专业且吸引人的小红书商品详情页。详情页是用户了解商品、建立信任并最终决定购买的关键环节，其设计至关重要。我们将逐一拆解详情页的各个组成部分，并提供清晰的操作指导。

## 概述

商品详情页的制作包含多个要素，如主图、标题、描述、SKU、视频和售后服务说明。本节课程将详细讲解每个部分的设计要点和操作技巧，帮助你打造高转化率的商品页面。

## 主图设计 📸

上一节我们介绍了详情页的整体构成，本节中我们首先来看看最重要的部分——主图设计。商品详情的第一张主图是用户对产品的第一印象，其重要性远超后面几张图片。

以下是主图设计的核心要点：

*   **核心地位**：第一张主图必须在3秒内吸引用户眼球，决定用户是否会继续浏览。
*   **设计原则**：
    *   **图片质量**：选择**高清、具有代表性**的图片，清晰展示商品整体外观和细节。
    *   **文案策略**：
        *   对于**批发、小商品**类目，可以适当增加文案描述产品优势。
        *   对于**服装、家居、饰品**等类目，文案应**精简**，只添加核心卖点文字，或**不加文字**，保持图片干净整洁。
    *   **文案占比**：整体文字面积**不应超过图片的五分之一**。

**核心公式**：**吸引力 = 高清图片 + 精准的文案策略（或无文案）**

主图是店铺流量转化的第一环，务必精心设计。

## 标题与副标题

![](img/13de6cb7f15ee774778723c9f841ca59_6.png)

了解了主图的重要性后，我们来看看如何通过文字进一步吸引用户。标题和副标题承担着不同的功能。

![](img/13de6cb7f15ee774778723c9f841ca59_8.png)

以下是两者的具体区别与制作要点：

*   **产品标题**：
    *   **作用**：用于**用户搜索**。虽然小红书以内容推荐为主，但仍有约30%的流量来自商品关键词搜索。
    *   **制作方法**：标题应包含**1-2个核心关键词**，其余部分用长尾词补充。注意**避免关键词堆砌**，确保60个字符内没有过多重复字眼。
    *   **示例代码（伪代码）**：
        ```
        标题 = 核心关键词1 + 核心关键词2 + 修饰性长尾词
        # 例如：“夏季新款 碎花连衣裙 女 收腰显瘦 度假风”
        ```

*   **副标题**：
    *   **作用**：**没有搜索权重**，主要用于**详细描述产品特点**，提升用户信任感和购买意愿。
    *   **制作方法**：根据产品特性自由编辑，突出卖点、优惠或品牌故事。

## 商品描述

在设置了吸引人的标题后，我们需要用详细的描述来建立信任并规避风险。商品描述是确保信息准确、避免违规的关键。

以下是商品描述的注意事项：

*   **必须详细准确**：清晰介绍商品的**规格、材质、颜色、尺寸**等信息，确保顾客能全面了解商品。
*   **规避风险**：描述与实物不符可能导致**虚假宣传投诉**，致使商品被平台强制下架，所有努力白费。

因此，商品描述务必实事求是，详尽清晰。

## 主图SKU与商品视频

接下来，我们看看如何设置商品选项和利用视频进行展示。这部分相对直观，但也有一些细节需要注意。

以下是相关操作要点：

*   **主图SKU**：即商品的可选项目，如**颜色、款式、尺寸、价格**等。根据你实际拥有的商品选项进行设置即可。
*   **商品视频**：
    *   **时长建议**：控制在**15秒左右**为宜。视频过大可能因加载缓慢导致用户流失。
    *   **内容**：动态展示商品外观、功能或使用场景。

## 详情页与售后服务说明

最后，我们整合所有细节，完成详情页的整体布局，并附上重要的售后保障。详情页是对主图信息的扩展，而售后服务说明则是成交的临门一脚。

以下是这两部分的制作指南：

*   **详情页制作**：
    *   **内容**：用于多角度、分模块展示商品的**细节、特征和优点**。
    *   **数量建议**：图片数量建议在**9到12张**左右。如果图片较多，可适当缩短视频时长。
    *   **整体体验**：确保用户浏览完所有详情（图片+视频）的**总停留时间在40秒左右**，体验最佳。

*   **售后服务说明**：
    *   **位置**：放在详情页的**最后一张**。
    *   **内容**：必须单独制作一张图，清晰写明**发货时间、物流方式、退换货规则（如7天无理由）** 等。
    *   **重要性**：
        1.  **对用户**：提供购物保障，增加安全感，**提高成交率**。
        2.  **对商家**：是明确的交易约定。一旦发生售后纠纷，此说明可作为凭证与平台沟通，**避免不必要的差评和投诉**。

**核心公式**：**用户信任 = 详细商品描述 + 明确的售后服务条款**
