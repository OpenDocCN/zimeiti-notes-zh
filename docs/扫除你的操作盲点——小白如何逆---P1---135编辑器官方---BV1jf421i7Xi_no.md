# 135编辑器入门教程：第1章：掌握核心操作面板与工具栏

![](img/49131f5dd97c117484332ddbfaedd169_1.png)

在本节课中，我们将系统学习135编辑器的核心操作界面与工具栏功能。通过掌握新旧工具栏的切换、文字与背景设置、表格操作以及样式编辑等关键技能，你将能够解决排版中90%的基础操作问题，实现从新手到熟练用户的跨越。

![](img/49131f5dd97c117484332ddbfaedd169_3.png)

## 1.1：认识135编辑器操作界面

在深入学习具体功能前，我们首先需要熟悉135编辑器的整体操作界面布局。

以下是编辑器的主要功能区域划分：

*   **左侧工具栏**：包含“我的文章”、“我的图片”等核心资源管理入口。
*   **样式展示区**：位于左侧工具栏右侧，用于搜索和选择各类排版样式。
*   **素材编辑区域**：中央的蓝色框区域，是进行文章内容编辑和样式调整的主要工作区。
*   **右侧功能区域**：提供“复制全文”、“导入文章”等常用操作功能。

## 1.2：切换与掌握新旧工具栏

上一节我们认识了编辑器的整体布局，本节中我们来看看如何操作核心的编辑工具栏。135编辑器提供了新旧两版工具栏，以适应不同用户的操作习惯。

*   **老版工具栏**：功能图标一目了然，所有操作按钮直接展示在顶部。
*   **新版工具栏**：设计简洁，将部分功能收纳在“插入”等下拉菜单中，界面更清爽。

![](img/49131f5dd97c117484332ddbfaedd169_3.png)

![](img/49131f5dd97c117484332ddbfaedd169_5.png)

以下是切换新旧工具栏的操作步骤：

1.  点击编辑器右上角的“功能设置”图标。
2.  在弹出的菜单中，勾选或取消勾选“工具栏”选项。
3.  勾选时为老版工具栏，取消勾选则为新版工具栏。

**注意**：新版工具栏支持自定义，点击最右侧的“+”号，最多可添加23个常用功能按钮。

## 1.3：文字格式的全面设置

工具栏的核心功能之一是设置文字格式。我们将其分为基础操作和进阶操作两部分来学习。

**文字基础操作**主要包括加粗、倾斜、颜色、对齐、字间距、下划线等。操作方式为选中文字后，在工具栏上点击对应按钮。

**文字进阶操作**则提供了更多美化选项，以下是具体功能的操作方法：

*   **文字阴影**：点击工具栏上的字体阴影按钮（图标为“T”带阴影），可选择预设阴影或自定义阴影的偏移量和模糊度。
*   **文字背景色**：点击“背景色”按钮（图标为“AB”），可为选中文字添加底色。为使效果更美观，可在文字前后添加空格。
*   **文字边框**：选中文字后，会自动弹出辅助工具栏，点击“A在边框内”的图标即可添加边框，并可修改边框颜色。
*   **文字上标/下标**：选中需要设置的文字（如“B”），在弹出辅助工具栏中点击“…”更多选项，即可找到上标和下标按钮。

此外，**全文文字样式**功能可以一键统一整篇文章的文字格式。点击工具栏上的“全文文字样式”按钮，可统一设置默认字体、字号、行高、字间距、对齐方式等。**慎用此功能**，因为它会应用到全文所有文字。

关于**对齐方式**，有几点需要注意：
*   `两端对齐`配合`首行缩进`是长段落排版最常用的方式，能使版面整齐美观。
*   调整`段前距`和`段后距`可以控制段落之间的间距。
*   排版时应遵循`行间距 > 字间距`的原则，通常字间距设为1.5，行间距设为1.75或2。
*   如果文章包含英文，在“全文文字样式”中将`英文断词`改为`不断词`，可以避免单词在行尾被错误断开。

## 1.4：背景设置的三种类型

![](img/49131f5dd97c117484332ddbfaedd169_9.png)

上一节我们详细讲解了文字处理，本节我们来看看如何为文章设置背景。135编辑器支持三种背景类型：图片背景、颜色背景和双背景。

**图片背景设置是关键**，以下是确保背景显示完美的推荐参数：
*   **上下左右边距**：建议均设置为 **`0`**。这便于后续自由调整正文区域的宽度。
*   **背景图尺寸**：选择 **`水平拉伸等比`**。
*   **对齐方式**：选择 **`居中`**。
*   **重复方式**：选择 **`垂直重复`**。
*   **图片大小**：宽度建议680px，高度控制在2000px左右，文件大小不超过4M，以确保在微信公众号正常显示。

**颜色背景**分为纯色和渐变色。设置纯色背景很简单，在背景设置中选择颜色即可。设置渐变色背景有两种方法：
1.  在背景设置中直接勾选“渐变色”，但此方法效果不明显。
2.  使用“配色方案”工具：在功能设置中确保“配色方案”可见，在排版前先应用一个带有渐变色的样式，或通过“全文换色”中的“渐变色”来设置。

**双背景**指“图片+颜色”的组合，可以营造更丰富的视觉效果。
*   **静态双背景**：背景图片需为PNG格式，参数要求与普通图片背景相同。
*   **动态双背景**：可使用GIF动图作为背景图，注意动画帧数不超过300帧，其他参数与静态图一致。

![](img/49131f5dd97c117484332ddbfaedd169_11.png)

**多种颜色背景**的实现方法是：在编辑区使用“后空行”插入多个独立区域，为每个区域分别设置不同的背景色。

## 1.5：图片与表格的进阶操作

掌握了背景设置后，我们来看看如何高效处理图片和表格。

**全文图片样式**可以批量管理所有图片：
*   `全文居中`：一键将所有图片设置为居中对齐。
*   `无缝设置`：让多张图片之间没有间隙。
*   `圆角`/`边框`：为全文图片统一添加圆角或边框效果。

**表格工具**是排版中的难点，以下是核心操作点：
1.  **调出表格工具**：插入表格后，点击表格区域，在“区域操作”中开启“表格操作”面板，或直接右键点击表格。
2.  **增删行列**：右键菜单或“表格操作”面板中有“插入行/列”和“删除当前行/列”选项。
3.  **移动行列**：使用“表格行上移/下移”功能。
4.  **设置单元格**：
    *   **背景色**：选中单元格，在“区域操作”的“背景”中设置，或右键选择“单元格属性”设置。
    *   **边框**：右键选择“表格属性”，在边框设置中调整颜色和样式。将颜色透明度（A值）设为 **`0`** 可使边框透明。
    *   **对齐方式**：选中单元格，在“表格操作”中选择“单元格对齐方式”。
5.  **调整行高列宽**：
    *   **行高**：按回车键增加行高，或通过“区域操作”中的“间距”调整内边距。
    *   **列宽**：在“区域操作”的“宽度”中，拖动百分比滑块进行调整。不推荐直接拖动表格线调整，因为不易操作。

## 1.6：其他实用工具详解

除了主要功能，工具栏还包含一些提升效率的实用工具。

**格式刷与提取样式**：两者都用于快速复制格式，但有区别。
*   `格式刷`：只能复制文字的加粗、颜色、字号等基础格式，无法复制字间距、行间距和对齐方式。
*   `提取样式`：可以复制文字的颜色、字号、行高、字间距，但无法复制文字的倾斜效果和对齐方式。
*   **共同限制**：两者均不能复制文字的对齐方式。

**特殊字符与表情**：通过工具栏的“特殊字符”按钮，可以插入各种符号、数字、箭头乃至表情，增加排版趣味性。

**清除格式**：在从Word等外部来源复制内容后，务必先使用“清除格式”功能，去除自带的样式（如白底），以免造成排版混乱。

**查找替换**：与Word功能类似，可以快速查找文中特定内容并替换。

![](img/49131f5dd97c117484332ddbfaedd169_13.png)

**全文自动排版**：点击“全文自动排版”（魔术棒图标），可一键清除全文的格式、字号和字体，让内容回归最简洁状态，是开始排版前的好习惯。

## 1.7：样式操作面板（全局样式）

![](img/49131f5dd97c117484332ddbfaedd169_15.png)

上一节我们完成了对工具栏的学习，本节开始，我们将深入功能更强大的样式操作面板。样式操作分为“样式操作”（控制整个样式）和“区域操作”（可进行局部修改）。首先学习“样式操作”，即全局样式面板。

如何调出样式操作面板？
1.  将鼠标移至编辑器右上角“功能设置”。
2.  取消勾选“切换到基础旧版”，即可使用新版面板。
3.  在新版面板中，点击“开启区域样式”，即可看到完整的操作选项。

![](img/49131f5dd97c117484332ddbfaedd169_17.png)

以下是样式操作的核心功能：

**基础操作**：
*   `锁定面板`：可将操作面板拖动并锁定在屏幕任意位置。
*   `样式ID`：点击可复制当前样式的唯一ID，用于快速搜索同款样式或向客服反馈问题。
*   `修改颜色`：鼠标悬停在样式颜色圈上，闪烁的部分即为可修改的颜色。
*   `复制/剪切/删除`：对样式进行基本管理。“复制并插入”能快速生成样式副本。
*   `变换样式`：将当前样式替换为其他样式，但可能保留原有文字。
*   `上下移动`：拖动样式或使用箭头按钮调整样式在文中的位置。
*   `转图片`：将样式（特别是含英文的）转为图片，避免在不同设备上显示错乱。
*   `样式代码`：高级功能，可直接修改样式HTML/CSS代码，如删除背景代码：`background-image: url(...)`。

**进阶操作**：
*   `背景`：可为单个样式设置“图片+颜色”的双背景，方法与全文背景设置类似。
*   `前后空行`：在样式前后插入空行，**这是避免“俄罗斯套娃”（在样式内直接回车嵌套内容导致排版错乱）的关键操作**，务必多用。
*   `传递`：将当前样式的**颜色和宽比度**快速应用到文中其他**同款样式**上。
*   `层叠顺序`：类似PS中的图层顺序，**数值越大，层级越高**。用于调整重叠元素的上下显示关系。
*   `段落`：对当前样式内部的段落进行精细设置，如字号、行距、对齐等，不影响其他样式。
*   `定位`：通过调整“左右偏移”（正数向右）、“前间距”（负数向上）、“后间距”（负数向上）的数值，精确控制样式的位置。常与“层叠顺序”配合用于创意排版。
*   `加一个/减一个`与`自动编号`：仅对特定的“正文段落样式”生效，可快速增减段落或为带编号的样式自动排序。
*   `宽比度`：调整样式本身的宽度比例，**不会影响全文两侧边距**，常用于创意排版，如将多行样式改为标题样式。
*   `转角度`：让样式360度旋转，配合“层叠顺序”和“定位”制作创意布局。
*   `左右翻转`：制作镜像样式。注意，整体翻转后，内部的文字和图片也需要**分别选中**，在“区域操作”中再次翻转以校正方向。
*   `倒影`：为样式添加倒影效果。“高度”值越小，渐变效果越明显；“偏移”值控制倒影与原文的距离。

## 1.8：区域操作面板（层级编辑）

![](img/49131f5dd97c117484332ddbfaedd169_21.png)

本节我们学习样式操作的另一个核心——“区域操作”面板。它与“样式操作”最大的区别在于拥有**层级选择**这一秘密武器，允许你对样式的局部进行编辑。

**层级选择**是区域操作的基础。将鼠标移至样式不同部分，闪烁的区域即代表当前选中的层级。所有操作都需先选中正确层级。

以下是区域操作的关键功能：

*   `宽度`与`间距`：通过选中不同层级，可以调整样式内某个特定部分（如图片、文字块）的宽度，或设置其内外边距。这是修改老模板两侧间距的常用方法。
*   `背景`：可以为样式内的特定文字区域添加背景色。也常用于为一些特殊图片样式换图（当无法直接双击换图时）。
*   `边框`：为选中层级添加边框，可自定义颜色、粗细、线型（实线/虚线）和圆角。通过设置上下左右不同的边框值，可以创造多样的装饰线条。
*   `阴影`：分为内阴影和外阴影。通过调整“偏移”（决定方向）和“模糊”半径，为元素添加立体感。
*   `前后空行`：在样式内部的特定区域（如标题与正文之间）插入空行，以便添加内容，是避免内部嵌套混乱的重要操作。
*   `复制`：选中某个层级（如一个标题模块）进行复制，可以快速在样式内部或外部生成相同结构。
*   `保存`：将修改后的样式保存到“我的样式库”中，方便下次使用。免费用户最多保存3个。
*   `区域代码`：高级功能。例如，当删除样式内多余空行导致排版错乱时，可在此面板中找到对应的 `<p>` 标签代码并删除，以安全移除空行。

区域操作中的`转角度`、`透明度`、`左右翻转`等功能，其逻辑与样式操作中的一致，只是操作对象变成了你通过层级选择选中的局部元素。

## 课程总结

本节课中，我们一起系统学习了135编辑器的核心操作。
1.  我们认识了编辑器的界面布局，并学会了切换新旧工具栏。
2.  深入掌握了文字格式的基础与进阶设置方法。
3.  学习了图片背景、颜色背景及双背景的设置技巧与关键参数。
4.  了解了图片的批量处理与表格的插入、编辑和美化。
5.  掌握了“格式刷”、“清除格式”等提升效率的实用工具。
6.  重点攻克了“样式操作”与“区域操作”两大面板，学会了如何全局修改样式，以及通过层级选择对样式进行局部精细编辑和创意组合。
