# 剪映手机版教学：P26：5.照片多样卡点教学 📸

在本节课中，我们将学习如何使用剪映手机版为照片制作多样化的卡点效果。这是一种让静态照片跟随音乐节奏动态呈现的技巧，非常适合用于制作视频号的动态相册或精彩集锦。

## 概述
我们将通过导入素材、添加音乐、自动踩点、添加转场与动画等步骤，完成一个富有节奏感的照片卡点视频。

## 教程步骤

### 1. 导入素材与音频
首先，我们需要准备视频素材和背景音乐。

打开剪映应用，导入你的照片素材。接着，点击“添加音频”，从你的收藏中选择一首适合的BGM（背景音乐）并点击使用。

### 2. 进行音乐踩点
为了让照片切换精准匹配音乐节奏，我们需要进行踩点操作。

点击已添加的音乐轨道，选择“踩点”功能。在踩点面板中，点击“自动踩点”按钮。此时，音频轨道上会出现一系列**小黄点**，它们代表了音乐的节拍点。

### 3. 对齐图片与删除多余音频
接下来，我们将每张照片的起始位置与一个小黄点对齐。

将时间线移动到第一个小黄点位置，将第一张照片的起始边缘对准它。重复此操作，让所有照片都起始于不同的小黄点。对齐完成后，将音乐轨道上超出最后一张照片的部分**音频删除**，使音画同步结束。

### 4. 添加转场效果
为了使照片之间的切换更自然流畅，我们可以添加转场。

点击两张照片之间的**小白块**图标，进入转场菜单。选择一个你喜欢的转场效果（例如“散白”），并勾选“应用到全部”，这样所有照片之间都会应用相同的转场。

### 5. 为照片添加动画
静态照片添加动画后，动态感会更强。以下是添加动画的步骤。

首先，选中第一张照片，点击“动画”按钮。在“入场动画”或“组合动画”中，选择一个效果（例如“荡秋千”）。然后，将此动画效果应用到序列中的所有照片上。

### 6. 使用关键帧制作放大效果
除了预设动画，我们还可以使用关键帧来自定义运动效果，例如让照片有节奏地微微放大。

选中第一张照片，在时间线的起始、中间和结束位置分别**添加一个关键帧**（共三个）。将第一个和第三个关键帧处的图片**稍微放大**，第二个关键帧处的图片保持原大小或略向下移动。这样就会形成一个“放大-复位-放大”的跳动效果。使用同样的方法，为后面的照片也添加上关键帧动画。

## 总结
本节课我们一起学习了制作照片卡点视频的全流程。从导入对齐素材、利用自动踩点匹配节奏，到添加转场、动画和自定义关键帧效果，这些步骤能让你轻松将静态照片转化为动态感十足的短视频。多加练习，你就能熟练运用这些技巧，制作出更精彩的卡点作品。
