# 手机剪辑教程：P41：19. 人物重影跑进画面

在本节课中，我们将学习如何制作一种人物从画面外跑进屏幕，并带有重影视觉效果的方法。这个效果常用于短视频，能增加动感和趣味性。

## 概述

我们将通过复制视频图层、使用智能抠像和入场动画功能，来实现人物从白色背景中“跑”出来的视觉效果。整个过程在手机剪辑软件中即可完成。

## 详细步骤

以下是实现该效果的具体操作流程。

**第一步：准备视频与背景**

首先，在剪辑软件中导入一段人物奔跑的视频素材。使用双指在预览界面 pinch 操作，将视频画面适当缩小一些。



**第二步：设置画布背景**

点击功能区的“背景”选项，选择“画布颜色”。从颜色面板中选取白色作为背景。这样，缩小的视频就会位于白色画布中央。



**第三步：复制并创建画中画**

选中已导入的视频图层，使用“复制”功能，为其创建一个副本。接着，选中这个副本图层，点击“切画中画”功能，将其转换为画中画图层。

将新创建的画中画图层向左拖动，使其起始时间点与主视频图层的开头完全对齐。



**第四步：应用抠像与动画**

选中画中画图层，点击“智能抠像”功能，将人物从背景中分离出来。

然后，点击“动画”选项，选择“入场动画”分类。在入场动画列表中，找到并选择“动感缩小”效果。将动画的持续时间调整为大约 **2.8秒**。



**第五步：完成与导出**

至此，核心效果已经制作完成。你可以为视频添加合适的背景音乐。最后预览成品，确认无误后即可导出。

## 总结

本节课我们一起学习了制作“人物重影跑进画面”效果的方法。关键步骤在于**复制图层创建画中画**，并对画中画图层应用**智能抠像**和**动感缩小入场动画**。通过调整动画时长，可以控制人物“跑入”的速度和节奏。你可以尝试将此效果应用于不同的运动场景，创造出属于自己的动态短片。
