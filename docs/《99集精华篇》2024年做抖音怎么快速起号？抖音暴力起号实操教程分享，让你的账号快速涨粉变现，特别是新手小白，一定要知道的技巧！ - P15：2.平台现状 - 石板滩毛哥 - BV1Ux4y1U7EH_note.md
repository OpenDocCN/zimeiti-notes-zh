# 抖音暴力起号实操教程：P15：2.平台现状 📊

在本节课中，我们将要学习当前自媒体行业的平台现状。了解行业的宏观环境，是判断是否入局以及如何规划的第一步。

上一节我们介绍了自媒体的基本概念，本节中我们来看看这个行业所处的环境究竟如何。

## 行业合法性：国家支持 ✅

![](img/6baa30a237af18588303cc12556a0d91_3.png)

首先需要明确，自媒体行业是受到国家政策支持的合法行业。这与一些采取非法手段牟利的活动有本质区别。国家已明令禁止非法行为，而自媒体则属于被鼓励发展的范畴。

![](img/6baa30a237af18588303cc12556a0d91_5.png)

![](img/6baa30a237af18588303cc12556a0d91_7.png)

以下是国家支持自媒体行业的具体证据：

![](img/6baa30a237af18588303cc12556a0d91_8.png)

![](img/6baa30a237af18588303cc12556a0d91_10.png)

*   **地方政府文件**：例如，荆州市人社局于今年6月2日发布了“六项措施”，明确指出要“加大政策宣传力度，有力支持自媒体以及新媒体行业的蓬勃发展”。
*   **多地政策跟进**：不仅是荆州，广西壮族自治区等许多地方政府都发布了支持新媒体行业发展的政策性文件。
*   **创作导向**：相关文件强调“坚持正确导向，内容为王”。这意味着，未来自媒体创作者的核心收益将来自于其创作的**文章和视频内容**。

## 平台激励：真金白银的补贴 💰

在政策鼓励下，各大自媒体平台也积极响应，推出了丰厚的创作者激励计划。

![](img/6baa30a237af18588303cc12556a0d91_12.png)

一个关键案例是西瓜视频（抖音的中视频平台）宣布的计划：未来一年将额外拿出**20亿元**人民币补贴中视频创作者。

这笔资金有两个核心特点：
1.  **纯补贴**：这20亿元是**不包含任何商单、直播和电商收入**的纯现金补贴。
2.  **保底上不封顶**：20亿元仅是保底数额，实际补贴金额没有上限。

平台出钱鼓励创作，这为创作者提供了直接的经济动力和保障。

## 市场潜力：巨大的供需缺口 📈

![](img/6baa30a237af18588303cc12556a0d91_14.png)

![](img/6baa30a237af18588303cc12556a0d91_16.png)

现阶段自媒体行业拥有一个显著优势：极其庞大的用户基数和相对较少的创作者数量，形成了巨大的市场机会。

![](img/6baa30a237af18588303cc12556a0d91_18.png)

以下是相关的数据对比：

![](img/6baa30a237af18588303cc12556a0d91_20.png)

*   **用户规模（需求侧）**：根据2022年的市场调查报告，中国网民规模已达**9.4亿人**。这些人是短视频和内容的消费者。
*   **创作者规模（供给侧）**：截至去年，中国全职从事自媒体的人约有370万，兼职约600万，合计约**1000万人**。

我们可以用一个简单的比例来理解这个市场：
`创作者数量 : 用户数量 ≈ 1000万 : 9.4亿 ≈ 1 : 94`
这意味着，平均每94个看视频的人，才对应1个做视频的人。单个创作者产出的内容速度，远远无法满足海量用户的观看需求。这种供需不平衡，正是行业充满机会、并被国家视为未来发展趋势的原因之一。

从电视的发展历程也能看出趋势：从黑白电视到彩色电视，再到如今能连接WIFI的智能电视，内容传播的渠道和终端在不断进化，未来内容的展示场景将无处不在，用户数量只会更加庞大。

本节课中我们一起学习了自媒体行业的平台现状。我们了解到，这是一个**受国家政策支持**的合法行业，平台方也提供了**巨额的资金补贴**来激励创作。更重要的是，市场存在着**海量用户与少量创作者**之间的巨大缺口，这为新人入局提供了广阔的空间和机会。认清这个积极向好的现状，是坚定信心、迈出第一步的重要基础。
