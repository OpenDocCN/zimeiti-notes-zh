# 新媒体运营：P12：01.10.0：ChatGPT 4.0 的几种使用渠道 🚀

## 概述
在本节课中，我们将学习如何获取和使用当前最强大的AI对话模型——ChatGPT 4.0。对于新媒体运营人员而言，掌握AI工具能极大提升工作效率，例如撰写脚本、制作PPT和生成报告。我们将介绍多种体验和使用ChatGPT 4.0的渠道，包括免费和付费选项。

## 微软 Edge 浏览器 Copilot
上一节我们介绍了AI对运营工作的重要性，本节中我们来看看第一个推荐渠道：微软Edge浏览器的Copilot功能。它是微软基于ChatGPT 4.0模型打造的AI助手，集成在Windows系统中，智能程度较高。

![](img/0e3b273512e23099594ce3dad0a79b97_1.png)

![](img/0e3b273512e23099594ce3dad0a79b97_3.png)

使用该功能有两个前提条件：
1.  登录微软账号（可使用Outlook邮箱注册）。
2.  需要使用“梯子”（VPN）才能正常访问。

Copilot提供三种对话模式：
*   **更有创造力**：AI的回答更具开放性和创意。
*   **平衡**：介于精确和创造力之间，适合一般用途。
*   **更精确**：回答更为严谨和准确，适合专业内容。

例如，我们可以使用“更有创造力”模式，输入指令：
```
请帮我写一个关于装修的短视频脚本，时长控制在3分钟左右。
```
AI便会生成相应的脚本文案。此外，Copilot还能基于你当前打开的网页内容进行分析，功能实用。

然而，Copilot也存在一些缺点：
1.  **精准度稍逊**：相较于官方ChatGPT 4.0，经过微软调教的Copilot在某些回答上可能不够精准。
2.  **连续对话能力受限**：在当前对话中持续提问的次数有限，达到限制后需开启新对话。
3.  **记忆功能有所缺失**：在利用历史对话优化后续回答方面能力较弱，且缺乏插件功能。

如果你的Edge浏览器没有Copilot图标，可以尝试下载最新版的Edge Dev（测试版）。你可以在Edge设置中检查更新，或访问微软官网的Edge Dev频道下载页面获取。

## 第三方平台与共享账号
除了官方集成工具，我们还可以借助第三方平台来使用ChatGPT 4.0。以下是几个可用的渠道：

### 1. 灵感AI
这是一个集成了多种AI模型的平台，包括GPT-3.5、GPT-4.0（含联网版）以及国内主流模型。

该平台的优势如下：
*   **模型丰富**：集成市面上多数主流AI模型。
*   **GPT-4免费额度**：新注册用户在一定时间内可免费使用GPT-4，有次数限制。
*   **无需梯子**：在国内可直接访问。

其缺点是响应速度可能较慢。平台也提供付费套餐，如19.9元包月，适合有高频需求的用户。

### 2. Vee AI
这个平台同样提供GPT-3.5和GPT-4.0模型。

其特点如下：
*   **响应速度快**：达到正常GPT使用水准。
*   **附带功能**：支持语音回答和绘画功能。
*   **免费额度有限**：初始赠送的Token（计算单位）通常仅适用于GPT-3.5。

![](img/0e3b273512e23099594ce3dad0a79b97_7.png)

需要注意的是，使用此平台需要梯子，且其Token消耗计算方式可能不易理解。

![](img/0e3b273512e23099594ce3dad0a79b97_9.png)

![](img/0e3b273512e23099594ce3dad0a79b97_11.png)

### 3. ChatGPT共享账号平台
这是一个网络共享账号平台，用户可以临时使用他人分享的ChatGPT账号。

平台使用说明如下：
*   **账号状态颜色标识**：
    *   **绿色**：账号空闲。
    *   **橙色**：账号近期被使用过。
    *   **红色**：账号正在被使用或使用频繁。
    *   **金色边框**：代表是付费的ChatGPT Plus（GPT-4）账号。
*   **使用方法**：找到状态良好的账号，点击后按提示设置密码即可登录使用。

这个平台的优势是可以免费体验GPT-4.0，但缺点是可用且状态良好的Plus账号比较稀缺，同样需要梯子才能访问。

## 国内AI大模型替代方案
如果无法使用上述需要梯子的渠道，国内也有一些优秀的AI大模型可供选择，它们无需梯子即可访问。

![](img/0e3b273512e23099594ce3dad0a79b97_13.png)

![](img/0e3b273512e23099594ce3dad0a79b97_15.png)

以下是几个主要的国内AI模型：
1.  **文心一言（百度）**：百度推出的AI对话模型，有免费版和付费的“4.0”版本，能满足基础的文案创作需求。
2.  **豆包AI（字节跳动）**：字节跳动出品的AI助手，在生成短视频脚本等运营相关内容上表现不错。
3.  **通义千问（阿里巴巴）**：阿里云推出的AI模型。
4.  **天工AI**：另一个可用的国内AI大模型。

这些国内模型在基础功能上足以辅助新媒体运营工作，例如生成灵感、撰写初稿等。虽然在某些复杂或专业任务上可能与顶尖模型存在差距，但作为提升效率的工具完全合格。建议选择一款用起来顺手的模型加以熟练使用。

为了直观对比，我们可以用同一个指令测试不同模型：
```
请帮我写一个关于装修的短视频脚本文案，要求镜头丰富，语言吸引人。
```
通过对比生成结果，可以更好地感受不同模型的特点和风格。
