# AI工具应用：P18：02.3：第二款AI生成PPT工具详解 🚀

在本节课中，我们将学习第二款强大的AI生成PPT工具。这款工具在生成速度、稳定性和模板丰富度方面具有优势，能与上一节介绍的工具形成互补，共同提升我们的PPT制作效率。

上一节我们介绍了一款AI生成PPT工具，本节中我们来看看另一款表现优异的工具。这款工具的主要优势在于生成速度快、运行稳定，并且提供了海量的模板供用户选择。要获得最佳生成效果，需要用户提供详细且结构清晰的节点内容。

## 工具使用方式

该工具主要提供两种生成方式：**AI智能生成**和**导入本地大纲**。通常，导入本地大纲的方式能获得更可控、更符合预期的结果。

![](img/90a15c54418b7675a466a10280eacf3a_1.png)

![](img/90a15c54418b7675a466a10280eacf3a_3.png)

以下是两种方式的具体说明：
*   **AI智能生成**：用户直接输入PPT主题，工具会自动构思大纲并填充内容，最终生成完整的PPT。
*   **导入本地大纲**：用户需提前准备好结构化的文档大纲，导入后工具会基于此大纲生成PPT，这种方式对内容的把控度更高。

## 实战演示：导入本地大纲

首先，我们需要准备一份详细的大纲。可以使用ChatGPT等AI助手来辅助完成。

**步骤一：生成大纲内容**
我们向ChatGPT发出指令，例如：“请为我撰写一份关于‘装修行业在小红书平台的运营推广计划’的详细大纲，并完善每一个小节点的具体内容信息，保持好结构和格式排版。”

**步骤二：导入并生成PPT**
1.  在工具界面选择“导入本地大纲”功能。
2.  将准备好的大纲文档上传。
3.  工具会快速解析文档结构，提取出大纲。
4.  从丰富的模板库中，选择一个符合行业风格（如装修行业）的模板。
5.  点击“生成PPT”，稍作等待即可获得初稿。

**步骤三：编辑与优化**
生成后的PPT可以在线编辑。如果发现某些页面内容缺失或排版不理想，可以进行以下操作：
*   在“大纲编辑”区域补充详细内容。
*   单击特定页面，使用“更换页面版式”功能一键调整布局。
*   通过右侧的工具栏调整文字、形状和背景。
*   如果对整体风格不满意，可以使用“模板替换”功能快速更换整套模板。
