## 1. 笔记收录 📥

![](img/cde25bb83a012130d455b5c3eef5015f_35.png)

![](img/cde25bb83a012130d455b5c3eef5015f_37.png)

![](img/cde25bb83a012130d455b5c3eef5015f_39.png)

![](img/cde25bb83a012130d455b5c3eef5015f_41.png)

![](img/cde25bb83a012130d455b5c3eef5015f_43.png)

笔记收录是笔记获得流量的基础。它指的是笔记发布后，能否通过搜索完整标题被找到。如果搜索不到，说明笔记暂时未被收录。收录状态并非一成不变，初期未收录的笔记后期仍有可能被收录。

![](img/cde25bb83a012130d455b5c3eef5015f_45.png)

![](img/cde25bb83a012130d455b5c3eef5015f_47.png)

![](img/cde25bb83a012130d455b5c3eef5015f_49.png)

![](img/cde25bb83a012130d455b5c3eef5015f_51.png)

笔记被收录意味着平台算法认可其质量，会将其分发至公域流量池。反之，如果笔记在初始流量池中表现不佳，则可能无法被收录，后续流量也会受限。

![](img/cde25bb83a012130d455b5c3eef5015f_53.png)

![](img/cde25bb83a012130d455b5c3eef5015f_55.png)

![](img/cde25bb83a012130d455b5c3eef5015f_57.png)

![](img/cde25bb83a012130d455b5c3eef5015f_59.png)

![](img/cde25bb83a012130d455b5c3eef5015f_61.png)

以下是影响笔记收录的几个关键点：

![](img/cde25bb83a012130d455b5c3eef5015f_63.png)

![](img/cde25bb83a012130d455b5c3eef5015f_65.png)

![](img/cde25bb83a012130d455b5c3eef5015f_67.png)

![](img/cde25bb83a012130d455b5c3eef5015f_69.png)

*   **原创度**：平台会检测笔记内容与站内其他内容的**重合度**。笔记需要有自己的独特价值和观点。
*   **规避违规词汇**：小红书平台管理严格。笔记中若包含**违规词汇**（如涉及玄学、伪科学、封建迷信等），可能导致笔记被限流或仅自己可见。需积累经验，避免使用已知的敏感词汇。
*   **初始数据反馈**：笔记发布后的初始互动数据直接影响收录判断。主要包括：
    *   **点击率**：公式可简化为 `点击率 = 点击量 / 曝光量`。低点击率说明封面或标题吸引力不足。
    *   **停留时长**：用户点进笔记后的观看时长。停留时间过短（如仅3-5秒）通常意味着内容不吸引人；停留30秒以上则说明内容有一定吸引力。
    *   **互动率**：包括点赞、收藏、评论、关注、分享等行为。积极的互动数据会向平台传递内容优质的信号。

## 2. 笔记权重分 ⚖️

![](img/cde25bb83a012130d455b5c3eef5015f_79.png)

![](img/cde25bb83a012130d455b5c3eef5015f_81.png)

![](img/cde25bb83a012130d455b5c3eef5015f_83.png)

![](img/cde25bb83a012130d455b5c3eef5015f_85.png)

除了收录，每篇笔记还有一个隐形的“**权重分**”，它决定了笔记能获得多少初始曝光。这个分数受两方面影响：

![](img/cde25bb83a012130d455b5c3eef5015f_87.png)

![](img/cde25bb83a012130d455b5c3eef5015f_89.png)

![](img/cde25bb83a012130d455b5c3eef5015f_91.png)

![](img/cde25bb83a012130d455b5c3eef5015f_93.png)

*   **账号权重**：运营良好、历史数据优秀的账号，其新发布笔记的初始权重分更高。这就是为什么粉丝量大的博主，笔记基础阅读量通常也更高。
*   **单篇笔记数据反馈**：如上文所述，笔记发布后的点击率、停留时长、互动率等数据，会动态影响其权重分，进而决定平台是否给予更多流量推荐。

![](img/cde25bb83a012130d455b5c3eef5015f_95.png)

![](img/cde25bb83a012130d455b5c3eef5015f_97.png)

![](img/cde25bb83a012130d455b5c3eef5015f_99.png)

![](img/cde25bb83a012130d455b5c3eef5015f_101.png)

![](img/cde25bb83a012130d455b5c3eef5015f_103.png)

因此，持续运营好账号，并确保每篇笔记都有良好的数据表现，是提升权重的根本。

![](img/cde25bb83a012130d455b5c3eef5015f_105.png)

![](img/cde25bb83a012130d455b5c3eef5015f_107.png)

封面是笔记的“门面”，直接决定了**点击率**。一个优秀的封面需要做到：

![](img/cde25bb83a012130d455b5c3eef5015f_121.png)

![](img/cde25bb83a012130d455b5c3eef5015f_123.png)

![](img/cde25bb83a012130d455b5c3eef5015f_125.png)

![](img/cde25bb83a012130d455b5c3eef5015f_127.png)

![](img/cde25bb83a012130d455b5c3eef5015f_129.png)

*   **视觉冲击**：使用惊艳、令人向往的场景图。
*   **价值直给**：在封面上直接展示用户需要的核心信息或解决方案，让用户一眼看到价值点。

![](img/cde25bb83a012130d455b5c3eef5015f_131.png)

## 4. 需求匹配度 🔗

![](img/cde25bb83a012130d455b5c3eef5015f_149.png)

![](img/cde25bb83a012130d455b5c3eef5015f_151.png)

![](img/cde25bb83a012130d455b5c3eef5015f_153.png)

![](img/cde25bb83a012130d455b5c3eef5015f_155.png)

![](img/cde25bb83a012130d455b5c3eef5015f_157.png)

笔记内容必须与目标用户的搜索意图和痛点高度匹配。这包括：

![](img/cde25bb83a012130d455b5c3eef5015f_159.png)

![](img/cde25bb83a012130d455b5c3eef5015f_161.png)

![](img/cde25bb83a012130d455b5c3eef5015f_163.png)

![](img/cde25bb83a012130d455b5c3eef5015f_165.png)

*   **精准选题**：基于关键词调研，优先创作用户当前最关心、搜索热度最高的话题内容。例如，在装修领域，“小户型装修避坑指南”比“装修风格大全”更能击中特定用户的即时需求。
*   **内容直给**：避免冗长铺垫。应在文案、封面或首图中直接抛出用户最关心的几个痛点解决方案。例如，直接列出“小户型装修必须注意的3个电路走向问题”。
*   **结构清晰**：内容组织要有逻辑，方便用户快速获取信息。配合文末的“钩子”（如引导评论、关注），可以有效提升互动和停留时长。

![](img/cde25bb83a012130d455b5c3eef5015f_167.png)

![](img/cde25bb83a012130d455b5c3eef5015f_169.png)

![](img/cde25bb83a012130d455b5c3eef5015f_171.png)

![](img/cde25bb83a012130d455b5c3eef5015f_173.png)

当笔记完美匹配用户需求时，点击率、停留时长和互动数据自然会提升。
