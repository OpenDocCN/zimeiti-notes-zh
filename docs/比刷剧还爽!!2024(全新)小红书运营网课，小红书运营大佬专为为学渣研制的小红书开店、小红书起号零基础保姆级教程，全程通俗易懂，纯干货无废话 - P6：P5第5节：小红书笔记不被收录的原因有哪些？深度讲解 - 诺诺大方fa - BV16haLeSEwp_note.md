# 小红书运营：P6：小红书笔记不被收录的原因深度解析 🔍

在本节课中，我们将深入探讨小红书笔记发布后未被平台收录的常见原因。理解这些原因对于确保您的内容能够正常曝光和传播至关重要。

上一节我们介绍了笔记发布的基础流程，本节中我们来看看哪些因素会导致笔记无法被系统收录。以下是导致小红书笔记不被收录的五点核心原因。

## 1. 存在违规推广行为
平台禁止直接的硬性广告。若笔记内容包含未经处理的品牌名称、明显的商品推销信息或直接的购买引导，会被判定为违规推广，从而不被收录。

## 2. 触犯敏感词检测规则
平台设有敏感词过滤机制。您的笔记或账号若包含违规敏感词，即使暂时被收录，后续审核发现后也会被剔除或限流。**核心操作**是发布前自行检查或使用工具排查敏感词。

## 3. 内容本身违规（图片与文字）
内容违规涵盖图片和文字两方面。
*   **图片违规**：避免使用带有其他平台水印或二维码的图片。此类跨平台引流方式已不被小红书允许。
*   **文字违规**：严禁出现不文明词汇、诱导性信息及任何平台明令禁止的敏感内容。
    *   **特别注意**：严禁涉及政治、破坏民族团结等原则性问题的词汇，包括其谐音。我们应专注于合法合规的网络营销。

## 4. 文章发送频率异常
短时间内高频次发布笔记，会导致系统判定账号行为异常，疑似机器人操作，从而不收录新发布的笔记。需保持合理、人性化的发布节奏。

## 5. 发布设备已被平台标记
平台能够识别设备硬件信息（如IMEI码）。若某台设备曾有过严重违规行为（如封号），即使卸载重装APP或更换新账号，该设备仍可能被平台标记，导致其上发布的内容不被收录。

本节课中我们一起学习了导致小红书笔记不被收录的五点主要原因：**违规推广、敏感词、内容违规、发布频率异常以及设备被标记**。理解并规避这些问题，是保障笔记正常流通、获得曝光的基础。在后续课程中，我们将学习如何创作符合规范、易于被收录的优质内容。
