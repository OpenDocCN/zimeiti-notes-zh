# 抖音运营教程：P4：3.账号定位 🎯

![](img/91267c791ab9da3c890622abdcd4df8f_0.png)

在本节课中，我们将要学习抖音账号定位的核心方法。一个清晰的定位是账号快速起号、涨粉和变现的基础。我们将通过反思三个关键问题来构建你的账号定位。



![](img/91267c791ab9da3c890622abdcd4df8f_2.png)

## 你是谁？—— 确定账号类型与主题

上一节我们介绍了账号定位的重要性，本节中我们来看看如何确定“你是谁”。这需要明确你的账号类型和呈现给观众的主题人物。

首先，我们要确定接下来要做的账号类型。除了之前章节提到的大类赛道（即抖音定义的一级垂类），我们还需思考在一级垂类下细分的具体赛道。

**公式：账号垂直类型 = 一级垂类 + 二级垂类**

这好比参加运动会，你报的项目“田径”是一级垂类，而你最擅长且有优势的“百米赛跑”就是二级垂类。通过定位账号的细分类目，可以确定账号的垂直类型。

在抖音中，无论任何行业，只要定位足够清晰，都有机会成功。因为抖音流量巨大，一旦做起来，增长和裂变速度会非常快。

其次，我们要确定账号主题是谁。账号主题是账号的拥有人，即最终呈现给粉丝的账号主人公。这能让粉丝清楚知道在看谁的作品。

在抖音商业类账号中，账号主题主要分为两类：
*   **商家账号**：主题是公司。内容需偏商业化，并不断通过内容拉近与粉丝的距离，避免严肃。这能建立信任感和好感，符合抖音的娱乐属性。
*   **达人账号**：主题是个人。首要解决信任问题，可通过IP打造的方式，为账号树立一个积极正面的人设。



## 你是干什么的？—— 明确账号内容与价值

明确了“你是谁”之后，接下来我们需要定义“你是干什么的”，即你的账号提供什么内容与价值。

最直接的方式是通过抖音认证身份（即职业）。达人可认证黄V，公司可认证蓝V。认证后，粉丝能清晰看到认证标签，从而明白你的职业和账号内容。同时，认证也能获取平台背书，增加信任感。

在未认证前，也可通过账号设置向粉丝明确传达你的业务。以下是可进行统一设置的要素：
*   头像
*   背景图
*   简介
*   视频封面

通过对自身商业模式和账号类型的确定，将这些元素进行与业务关联的统一设置。

最后，需要花时间思考：你将为粉丝提供什么样的价值？在抖音中，粉丝对博主的价值需求主要有以下几种类型：

以下是抖音中常见的粉丝价值需求类型：

1.  **知识**
    *   **认知型**：如人生思考、思维方式、读书分享、行业内幕、成功经验、个人提升。
    *   **干货型**：如技能提升、经验总结、操作步骤、冷知识。
    *   **常识普及型**：如生活、健康、产品、科学、历史、地区等常识的解读与普及。
    *   **提供价值**：认同感（如三观、观点认同）、减少知识盲区的获得感、进而产生自我认同。

2.  **情绪**
    *   **类型**：开心愉悦（搞笑、夸张表情）、共鸣（情侣、婆媳、家庭、职场等关系）、情绪波动（争议话题、反转反差内容引发）、怀旧、emo、震惊、好奇等。
    *   **提供价值**：共鸣。利用表情、文字、画面、声音、肢体动作准确抒发情绪，直接感染观众。

3.  **感觉**
    *   **类型**：视频画面美感、构图、治愈感、视觉冲击、对作者人格魅力的仰慕；视频声音带来的节奏感、心跳加速或听觉享受。常见于颜值、萌宠、美景、才艺、明星、文化类账号。
    *   **提供价值**：共情。因表情、文字、画面、声音、肢体动作的自然表现，让人产生联想，并在情绪、认知等层面与作者产生连接，间接引发共鸣。

4.  **体验**
    *   **类型**：展示不一样的生活方式、到不了的地方、未曾体验的事物与感觉。常见于美食（享用/制作）、旅行、运动、角色体验类账号。
    *   **提供价值**：满足好奇心，带来紧张、激动、感动等体验和获得感。

5.  **期待**
    *   当用户被内容吸引力、观看体验或对作者本人产生好奇时，会在看完视频后意犹未尽，从而点击关注，期待下次更新。许多博主会利用这点，不将所有内容在一个视频中呈现完，以制造持续吸引力。



## 为什么要关注你？—— 创造关注动机

“为什么要关注你”是定位的最终落脚点，它源于你为粉丝提供的独特价值。综合以上两点，当你能清晰定义自己并提供明确价值时，关注你的理由自然显现。

你可以通过提供持续的知识收获、稳定的情绪共鸣、美好的感官体验、新奇的虚拟经历，或制造持续的更新期待，来创造粉丝关注你的核心动机。



本节课中我们一起学习了抖音账号定位的完整框架。你需要依次回答“你是谁”（确定类型与主题）、“你是干什么的”（明确内容与价值），并最终整合出“为什么要关注你”的答案。一个精准、垂直且能提供明确价值的账号定位，是你在抖音快速起号、实现涨粉与变现的第一步，也是最关键的一步。
