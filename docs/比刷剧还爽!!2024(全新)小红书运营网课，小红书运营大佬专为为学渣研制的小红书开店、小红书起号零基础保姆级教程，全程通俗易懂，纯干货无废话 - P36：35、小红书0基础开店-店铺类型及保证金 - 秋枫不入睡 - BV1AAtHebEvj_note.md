## 店铺类型体系

小红书的店铺体系主要分为以下几种类型，每种类型对应不同的经营主体和资质要求。

以下是小红书的主要店铺类型列表：

*   **个人店**
*   **个体工商户店**
*   **普通企业店**
*   **专卖店**
*   **旗舰店**
*   **卖场型旗舰店**

这些类型本质上代表了不同层级的品牌或商家身份。接下来，我们将逐一详细解析。

## 各类店铺详解

### 个人店

个人店是最基础的店铺类型，类似于线下摆地摊。其核心特点是申请和经营门槛低。

![](img/35a58b3042de113d626554acea2642d0_6.png)

*   **申请主体**：个人用户，仅需身份证。
*   **经营特点**：适合经营零星、小额的商品，如首饰、服装、手机壳等。可以销售多个品牌，如同一个“杂货铺”。
*   **店铺命名**：格式为“专业号名称 + 店名”，通常由系统自动匹配。
*   **保证金**：涉及**类目保证金**和**浮动保证金**，二者取金额高者。个人店的起始保证金通常为1000元。

### 个体工商户店

![](img/35a58b3042de113d626554acea2642d0_8.png)

个体工商户店与个人店的主要区别在于需要营业执照。

*   **与个人店的区别**：入驻要求中多了一项**营业执照**，对品牌授权无要求。
*   **其他方面**：如经营特点、保证金规则等，与个人店基本一致。

![](img/35a58b3042de113d626554acea2642d0_10.png)

![](img/35a58b3042de113d626554acea2642d0_12.png)

### 普通企业店

普通企业店及后续的专卖店、旗舰店等，主要差异体现在入驻要求的复杂度上。通常，入驻要求越复杂，店铺在平台获得的权重可能越高。

*   **申请主体**：企业。
*   **入驻要求**：必须持有企业的营业执照。

对于刚入门的新手，建议从个人店开始测试市场潜力和产品可行性，无需初期投入过高成本。

### 专卖店、旗舰店与卖场型旗舰店

![](img/35a58b3042de113d626554acea2642d0_14.png)

这三种店铺类型主要服务于品牌方或品牌授权经销商。

*   **专卖店**：要求营业执照及品牌认可（自有品牌或非独占授权品牌）。
*   **旗舰店**：要求营业执照及**品牌授权书**（销售他人品牌）。
*   **卖场型旗舰店**：要求具备零售资质，并可经营多个品牌。但在小红书以女性用户为主的生态中，此类大型店铺较少，反而为中小卖家留下了发展空间。

## 保证金规则详解 💰

![](img/35a58b3042de113d626554acea2642d0_17.png)

![](img/35a58b3042de113d626554acea2642d0_19.png)

了解店铺类型后，最关键的是理解保证金规则。保证金分为**基础保证金**、**类目保证金**和**浮动保证金**。

![](img/35a58b3042de113d626554acea2642d0_20.png)

### 类目保证金

![](img/35a58b3042de113d626554acea2642d0_22.png)

类目保证金根据您经营的商品类目固定收取。平台将类目划分为普通企业类目、特殊类目和跨境类目等，不同类目对应不同的保证金金额。具体金额需参考平台官方发布的最新类目保证金表。

### 浮动保证金（重点）

浮动保证金是根据店铺的月成交金额动态计算的，这是需要特别关注的规则。

以下是浮动保证金的计算阶梯：

*   **月成交额 ≤ 5万元**：无需额外浮动保证金（但需已缴纳基础保证金）。
*   **5万元 < 月成交额 ≤ 10万元**：需保证金 **3000元**。
*   **10万元 < 月成交额 ≤ 50万元**：需保证金 **5000元**。
*   **月成交额 > 50万元**：需保证金 **20000元**。

**核心机制**：平台会检查您保证金账户的总额是否满足当前月成交额对应的要求。如果余额不足，您将无法提现店铺资金。

**公式表示**：
```
所需总保证金 = max(类目保证金， 基础保证金 + 浮动保证金)
```
其中，浮动保证金根据上述月成交额阶梯确定。

![](img/35a58b3042de113d626554acea2642d0_24.png)

**重要提示**：例如，您只缴纳了1000元基础保证金，当某月成交额超过5万元时，由于保证金总额不足3000元，您将无法提现。因此，务必随时关注成交额并确保保证金充足。

![](img/35a58b3042de113d626554acea2642d0_25.png)
