# 抖音运营教程：P12：产品与用户兴趣标签 🧩

在本节课中，我们将学习如何运用“九宫格选题法”，通过分析产品属性和用户兴趣标签，为抖音账号创作源源不断的优质选题。我们将以健身软件Keep为例，详细拆解这一方法。

上一节我们介绍了选题的重要性，本节中我们来看看如何系统地挖掘选题方向。

## 产品标签分析

首先，我们需要分析产品本身的核心特点。以Keep为例，我们将产品名称放在中心，围绕它联想八个关键的产品属性标签。

以下是分析产品标签的步骤：

1.  **确定核心产品**：将“Keep”置于九宫格中心。
2.  **联想产品特点**：围绕Keep，思考其作为一款产品的核心功能和属性。
3.  **填充周边八个格子**：将联想到的所有关键词填入周围的空格。

例如，对于Keep，其产品标签可能包括：
*   线上健身软件
*   专业健身教练
*   饮食搭配建议
*   减肥攻略
*   健身干货知识
*   科学运动方法
*   打卡运动制度
*   主要用户为一二线城市人群

> 产品标签的分析类似于市场定位，例如抖音用户多集中于一、二线城市，而快手用户则更多分布在三、四线城市。这类数据可以在百度指数或艾瑞网等平台查询。

完成产品标签填充后，我们暂时将其放在一边。

## 用户兴趣标签分析

接下来，我们分析目标用户的兴趣点。将核心用户群体置于另一个九宫格的中心，然后进行发散联想。

以下是分析用户兴趣标签的步骤：

1.  **确定核心用户**：将“健身爱好者”或“想要减肥/塑形的人”置于九宫格中心。
2.  **联想用户兴趣**：思考这类用户除了健身，还可能对哪些领域感兴趣。
3.  **填充兴趣标签**：将联想到的兴趣点填入周围的格子。

例如，健身爱好者的兴趣标签可能包括：
*   线下健身房
*   健身装备（如紧身衣、运动鞋）
*   轻脂美食/减肥餐
*   旅行
*   星座
*   明星动态
*   心灵鸡汤
*   时尚穿搭

通过这个过程，我们将用户群体从“健身爱好者”扩展到了更广泛的“热爱生活者”。对于不同的目标用户（如“网易云用户”），则需要填充完全不同的兴趣标签。

![](img/d1aff5656ee5e9154ac2520c38f199e5_1.png)

现在，我们得到了两个填满的九宫格：一个是产品标签，一个是用户兴趣标签。

## 运用“连连看”生成选题

有了这两组标签后，如何生成具体的视频选题呢？答案是进行“连连看”组合。

**核心方法**：从“产品标签”和“用户兴趣标签”中分别选取一个关键词，将它们进行创意结合，形成一个完整的视频选题方向。

以下是几个组合示例：

*   **组合1：饮食搭配 + 旅行**
    *   生成的选题方向：`外出旅行时，如何享受美食又能控制体重？`

*   **组合2：运动方法 + 明星**
    *   生成的选题方向：`赵丽颖都在练的5个高效健身方法，你一定要学起来！`

*   **组合3：科学 + 健身房**
    *   生成的选题方向：`在健身房如何科学训练，避免受伤并高效增肌？`

通过这种交叉组合，理论上可以衍生出无数个选题（8 x 8 = 64种基础组合），从而建立起庞大的内容素材库，解决“不知道拍什么”的难题。

## 优化选题与寻找灵感

然而，并非所有组合产生的想法都足够优质或容易展开。当你对某个组合没有灵感时，可以借助搜索工具。

**操作技巧**：将组合出的关键词（如“健身 科学”）输入百度等搜索引擎，查看搜索联想词和相关长句。这些大众搜索的短语往往能给你带来新的灵感，帮助你完善选题，使其更贴近用户真实的搜索和关心的问题。
