# 小红书开店运营教程：P1：抄作业 📝

## 概述
在本节课中，我们将学习一种通过“抄作业”方式，利用抖音中视频计划获取收益的简单方法。该方法核心在于借鉴爆款内容，通过简单加工形成自己的视频，适合新手操作。

---

## 第一步：寻找爆款选题 🔍
上一节我们概述了方法，本节中我们来看看具体如何操作。第一步是寻找已经被市场验证的爆款视频选题。

1.  打开抖音，在搜索框中输入“温情故事”。
2.  点击顶部的“视频”分类板块。
3.  在筛选条件中选择“最多点赞”。
4.  此时，页面会展示大量高点赞量的爆款温情故事视频。

---

## 第二步：提取并处理文案 ✂️
找到目标视频后，我们需要获取其核心内容。以下是提取和处理视频文案的步骤。

1.  随意点击一个高赞视频进入播放页。
2.  点击视频右下角的“分享”按钮。
3.  在弹出的菜单底部，找到并点击“复制链接”。
4.  将复制的链接粘贴到文案提取工具中，获取视频的原始文案。

**注意**：获取的文案不能直接使用。必须通读全文，理解故事脉络后，用自己的语言重新讲述一遍。这样可以有效避免违规风险。

---

## 第三步：利用工具自动生成视频 🎬
文案准备就绪后，我们可以借助高效工具快速完成视频制作。以下是具体操作。

1.  将重新讲述后的文案导入“剪映”APP。
2.  使用“一键成片”功能。
3.  该功能会根据文案内容自动匹配画面素材并生成配音。
4.  至此，一个完整的视频就制作完成了，无需复杂的剪辑操作。

---

## 第四步：发布与获取收益 💰
视频制作完成后，需要通过正确的渠道发布才能获得收益。以下是发布流程。

1.  在抖音顶部搜索“中视频计划”。
2.  进入“中视频伙伴计划”官方入口。
3.  通过该入口发布制作好的视频。
4.  发布时，务必勾选“同步到西瓜视频和今日头条”。

![](img/7eb38fcd21ff90c866f0da1f37e5e218_1.png)

**核心机制**：视频在这三个平台（抖音、西瓜、头条）产生的播放量，都会为您带来收益。

---

## 方法总结与核心要点
本节课我们一起学习了“抄作业”获取收益的全流程。这个方法看似简单，但关键在于它精准地抓住了两个爆款核心要素：**爆款文案**和**爆款选题**。

**公式**：成功视频 ≈ 爆款选题 + 爆款文案 + 个人化处理

![](img/7eb38fcd21ff90c866f0da1f37e5e218_3.png)

所需的工具，如文案提取器、视频下载器等，可以通过特定方式获取。请记住，简单的方法重在执行。只要能够坚持操作，就能在平台上获得收益。
