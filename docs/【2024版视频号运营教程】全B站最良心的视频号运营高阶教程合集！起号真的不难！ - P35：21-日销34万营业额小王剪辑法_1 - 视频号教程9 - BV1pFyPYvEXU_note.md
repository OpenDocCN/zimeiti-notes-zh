# 视频号运营教程：P35：21-日销34万营业额小王剪辑法_1 🎬

在本节课中，我们将学习一种高效的视频剪辑方法，即“小王剪辑法”。该方法通过一系列步骤，将下载的原始视频素材进行二次创作，替换掉原有的广告内容，并加入自己的产品信息，最终制作出适合在视频号平台发布的带货视频。我们将从视频导入开始，逐步讲解放大、裁剪、变速、广告定位与替换、画中画应用以及最终效果调整等核心操作。

---

## 视频预处理：基础调整 🔧

上一节我们介绍了课程概述，本节中我们来看看视频剪辑的第一步——基础预处理。这个阶段的目标是准备好原始素材，为后续的广告替换打下基础。

首先，将下载好的视频导入剪辑软件。导入后，对视频画面进行轻微放大。操作方法是使用双指在屏幕上向外拉伸，将画面放大一点点。这个操作可以消除视频边缘可能存在的平台水印或无关元素。

接着，进行“掐头去尾”。这意味着需要裁剪掉视频开头和结尾不必要的部分。例如，视频开头可能有录制者的开场白，结尾可能有结束语或黑屏，这些部分都需要删除，使视频内容更加紧凑。

最后，对视频进行变速处理。通常将视频速度调整为1.1倍。轻微的加速可以使视频节奏更快，更符合短视频平台的观看习惯，同时也能在一定程度上避免平台查重。

以下是预处理的核心步骤列表：
1.  **导入视频**：将素材导入剪辑软件。
2.  **轻微放大画面**：使用双指手势放大视频，消除边缘干扰。
3.  **掐头去尾**：裁剪掉视频开头和结尾的无效片段。
4.  **调整速度**：将视频速度设置为 **1.1倍**。

完成这些基础调整后，视频的主体框架就准备好了。

## 定位与移除原广告 🎯

在完成基础调整后，下一步是处理视频中的原有广告。我们的目标是将别人视频中用于带货的产品，替换成我们自己的产品。

首先，需要仔细观察原视频，找到其中插入广告的片段。通常，广告会出现在剧情发展的某个特定环节，例如人物开始介绍或展示某个产品的时候。你需要明确原广告从哪一秒开始，到哪一秒结束。

找到原广告片段后，需要将其从视频中移除。操作方法是通过“分割”功能，将广告片段单独裁剪出来，然后删除。有时，原广告可能不是独立片段，而是与剧情画面重叠。这时，可以通过“放大”画面局部的方式，将包含广告产品的画面区域裁剪掉，只保留干净的剧情背景。

例如，原视频在介绍“沙琪玛”，而你要卖的是“榨菜”。那么，所有出现“沙琪玛”产品画面或口播的片段，都需要被移除或遮盖。

以下是定位与移除原广告的关键点：
*   **观察原视频**：找到广告插入的起止时间点。
*   **分割与删除**：使用分割工具切除独立的广告片段。
*   **局部放大裁剪**：对于重叠的广告画面，放大画面以裁剪掉产品部分。
*   **确保画面干净**：反复检查，确保原广告产品没有任何残留。

处理干净原广告后，视频中就留下了可以嵌入我们自己产品信息的“空位”。

## 添加与融合新广告 ✨

清除了原广告，现在就可以加入我们自己的产品广告了。核心方法是使用“画中画”功能，将自己的产品图片或视频素材叠加到原视频的特定位置。

![](img/ba9f1b341c59f7f30c797ceb5e2e771d_3.png)

首先，准备好自己产品的图片或短视频素材。然后，在剪辑软件中点击“画中画”功能，新增一个轨道，并将产品素材导入。

![](img/ba9f1b341c59f7f30c797ceb5e2e771d_5.png)

接下来是关键步骤：将产品素材移动到原视频中广告应该出现的位置。例如，原剧情中人物手拿“沙琪玛”的地方，现在就需要用“榨菜”的图片覆盖上去。你需要调整画中画素材的大小、位置和出现时间，使其与人物动作和剧情自然衔接，完全遮盖住原产品。

为了让添加的广告更自然，可以为其添加简单的动画效果，比如“关键帧”。通过设置起始和结束的关键帧，可以让产品图片随着镜头或人物动作有轻微的移动，模拟真实手持的效果。

此外，你可能会添加多段产品介绍片段。这些片段可以从其他电商广告中获取，但需要注意替换成与自己产品相符的文案和口播。保持总视频时长在1分30秒到1分40秒之间为宜。

以下是添加新广告的步骤列表：
1.  **准备素材**：准备好清晰的产品图片或视频。
2.  **添加画中画**：导入素材到新的画中画轨道。
3.  **对位与覆盖**：调整素材位置和大小，完全覆盖原广告位。
4.  **添加动态效果**：使用**关键帧**功能让素材动起来，增强真实感。
5.  **插入口播广告**：在合适位置添加产品介绍片段，控制总时长。

通过细致的调整，让新广告看起来就像是原视频的一部分。

## 最终效果优化与导出 🚀

所有广告内容添加完毕后，需要对整个视频进行最终的效果优化，使其更具吸引力和专业感，然后导出成品。

首先，可以再次对视频的某些片段进行缩放和位移。例如，将剧情片段向左或向右轻微移动，并适当放大，这种变化可以增加视频的动感，避免画面一成不变。

其次，添加一些装饰元素。在视频的四个角落添加半透明的“贴纸”（如Logo、装饰图案），并将它们的透明度调至最低，使其几乎看不见。这样做的主要目的是为了提升视频的原创度，防止平台判定为搬运。

![](img/ba9f1b341c59f7f30c797ceb5e2e771d_11.png)

然后，可以添加一个全局的“画中画”图层。这个图层可以使用自己拍摄的一段生活素材（如房间一角、桌面），将其覆盖全屏，并将混合模式设置为“叠加”，不透明度调到很低（5%以内）。这层淡淡的纹理也能有效提升视频的原创性。

最后，进行调色。为视频添加一个“滤镜”，并简单调整“调节”参数（如亮度、对比度、饱和度等），将所有数值的调整控制在5以内。微调可以统一视频色调，使其更美观。

完成所有调整后，将视频比例设置为 **9:16**（竖屏），选择背景样式，并勾选“全局应用”。最后，点击导出，选择高清分辨率，等待渲染完成即可。

![](img/ba9f1b341c59f7f30c797ceb5e2e771d_13.png)

以下是最终优化与导出的清单：
*   **画面微动**：对部分片段做缩放和位移。
*   **添加角落贴纸**：放置低透明度贴纸提升原创度。
*   **叠加纹理图层**：使用**混合模式（如叠加）** 添加全屏画中画。
*   **统一色调**：添加滤镜，微调**调节参数（各项≤5）**。
*   **设置比例与导出**：设定比例为 **9:16**，导出高清视频。

本节课中我们一起学习了“小王剪辑法”的全流程。从视频的预处理、定位并移除原广告，到添加融合自己的新广告，最后进行整体的效果优化与导出。这套方法的核心在于**细致地遮盖原产品，并自然地将自己的产品嵌入剧情**，同时通过多种技巧（如关键帧动画、贴纸、纹理叠加、微调色）提升视频的原创性和观感。掌握这个方法，你可以高效地将热门剧情素材转化为自己的带货视频。
