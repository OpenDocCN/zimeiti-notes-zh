## 概述：什么是人群定位？

小红书店铺的人群定位，其逻辑与账号定位相似，但更侧重于**商业转化**。它关乎你的店铺内容会被推送给谁，以及最终谁会购买你的产品。理解并做好人群定位，是为后续所有店铺运营操作（如内容推广、产品定价）所做的必要技术准备。

## 人群定位的五个核心维度

以下是构成小红书店铺人群定位的五个关键要素，我们将逐一进行解析。

### 1. 个人属性

个人属性是用户的基础标签，包括**性别、年龄范围和喜欢的内容**。这与我们之前选择店铺类型时的考量是相通的。

在操作店铺时，需注意店铺属性应与账号属性相匹配。若只做了账号定位而忽略店铺的人群属性，引流来的用户可能与你的产品不匹配。因此，我们需要根据所售产品来反向确定目标用户的个人属性。例如，销售高端护肤品，目标用户可能定位为**25-40岁、女性、对美妆护肤内容感兴趣**的人群。这种方式比单纯做账号时选择的年龄范围更为聚焦。

### 2. 消费能力

用户的消费能力并非主观感觉，而是小红书系统通过数据计算得出的。其公式大致为：

**用户月均消费能力 ≈ 用户年总消费额 / 12**

系统会根据用户历史购买产品的**类型、价格和搜索行为**，综合判断其消费层级，形成一个“隐藏的会员等级”。因此，在为产品定价时，必须参考目标人群的消费能力。定价过高或过低，都难以吸引到核心消费群体。

### 3. 购买类型

小红书系统会分析用户的购买行为，并通常只为单个用户推广**最多三个**产品大类。这些大类可包括日用品、生活用品、户外用品、家具玩具等。

由于“千人千面”的推荐机制，系统会根据用户最近的购买行为来动态调整展示的产品类型。通常，用户最近频繁浏览或购买的类型会被优先展示。这意味着，如果你的产品恰好属于目标用户近期关注的类型，那么被推荐的概率就会大大增加。

### 4. 购买地域

系统会根据物流信息等数据，分析不同地域用户的购买偏好。例如，系统可能发现北京用户购买服装较多，而上海用户购买食品较多。

随后，系统会将店铺与地域进行匹配。如果你的店铺产品属性（如风格、价位）恰好符合北京用户的普遍偏好，那么你的内容被推送给北京地区用户的几率就会显著提高。这是一个**将对应产品分享给对应地域人群**的自动化过程。

### 5. 购买习惯

![](img/246e34232541f7af8e24d15f9b81614a_8.png)

购买习惯指用户典型的购物决策模式，例如是否喜欢“货比三家”、浏览多久后下单等。系统通过分析用户的**搜索和浏览兴趣**来判断其习惯。

![](img/246e34232541f7af8e24d15f9b81614a_10.png)

![](img/246e34232541f7af8e24d15f9b81614a_12.png)

这里需要注意一个运营细节：如果用户在你的店铺浏览后，却最终在竞争对手的店铺下单，那么系统可能会判定竞争对手的店铺更符合该用户需求，从而在后续推荐中给予对手更高的权重。虽然过去那种通过恶意浏览干扰对手流量的手段现已不适用，但这种用户行为对系统推荐的自然影响依然存在。

## 综合应用与前期策略

总结来说，店铺运营的流程通常是：**店铺定位 → 选品 → 初步定价 → 人群定位**。

对于人群定位中的五个维度（个人属性、消费能力、购买类型、购买地域、购买习惯），新手卖家需要采取阶段性策略：

*   **如果你想快速扩大曝光**：前期可以暂不精细操作后三个维度（购买类型、地域、习惯）。因为系统在匹配更精准人群的同时，展示量可能会下降（例如从3000降至1000）。前期应优先完成**个人属性**和**消费能力**的定位，以保证一定的流量基础。
*   **如果你追求高精准转化**：则可以全面定义这五个维度。这样虽然前期流量会减少，但引入的用户质量更高，后续的数据转化率和提升速度会更好。

此外，后续要讲的**价格定位**，与人群定位这五个维度（尤其是消费能力、购买类型和习惯）是相互影响、共同匹配的。你可以先自行定价，再观察人群数据；也可以先明确人群，再确定价格。

## 总结

![](img/246e34232541f7af8e24d15f9b81614a_14.png)

本节课中我们一起学习了小红书店铺**人群定位**的完整框架。我们了解到，人群定位包含**个人属性、消费能力、购买类型、购买地域和购买习惯**五个核心维度。正确的人群定位能帮助系统将你的店铺推荐给最有可能购买的用户。作为新手，你可以根据自身是追求“流量规模”还是“转化精度”来选择不同的执行策略，逐步优化你的店铺运营。
