# 微信视频号运营教程：P13：11.如何填写邀请函 📝

在本节课中，我们将学习如何正确填写视频号创作者计划的邀请函。这是加入官方计划、获取流量扶持和变现机会的关键一步。整个过程简单明了，我们将一步步进行拆解。

![](img/ef619b31f4e79bef1ab889f4da92e388_1.png)

上一节我们介绍了如何注册视频号，本节中我们来看看如何找到并填写关键的邀请函信息。

## 第一步：打开微信
首先，你需要打开手机上的微信应用。



![](img/ef619b31f4e79bef1ab889f4da92e388_1.png)

![](img/ef619b31f4e79bef1ab889f4da92e388_3.png)

![](img/ef619b31f4e79bef1ab889f4da92e388_4.png)

## 第二步：进入“发现”页
打开微信后，点击底部导航栏的“发现”选项。



![](img/ef619b31f4e79bef1ab889f4da92e388_3.png)

![](img/ef619b31f4e79bef1ab889f4da92e388_6.png)

## 第三步：找到“视频号”入口
在“发现”页面中，向上滑动屏幕。你可以找到“视频号”的文字入口，或者一个黄色的蝴蝶图标，点击它即可进入视频号。



![](img/ef619b31f4e79bef1ab889f4da92e388_4.png)

![](img/ef619b31f4e79bef1ab889f4da92e388_8.png)

## 第四步：进入个人主页
进入视频号后，点击页面右上角的个人头像图标，进入个人主页。



![](img/ef619b31f4e79bef1ab889f4da92e388_9.png)

![](img/ef619b31f4e79bef1ab889f4da92e388_6.png)

## 第五步：查看账号信息
在个人主页中，向下滑动屏幕。你可以看到“我的视频号”区域，这里会清晰显示你的**账号昵称**和**粉丝数量**。
例如，显示为“天道酬勤，39917个粉丝”。



![](img/ef619b31f4e79bef1ab889f4da92e388_8.png)

![](img/ef619b31f4e79bef1ab889f4da92e388_9.png)

现在我们已经找到了填写邀请函所需的前两项信息。接下来，我们看看邀请函上其他栏目如何填写。

以下是邀请函各栏目的填写指南：
1.  **视频号账号昵称**：填写你在上一步中看到的昵称，例如 `天道酬勤`。
2.  **粉丝数量**：填写你账号当前的粉丝数，例如 `39917`。
3.  **活动邀请人**：由于是通过特定渠道加入，此处填写指定邀请人的名字，例如 `张良老师`。
4.  **实名信息**：填写你本人的真实姓名，例如 `周建国`。
5.  **结算手机号**：填写你用于接收验证码和后续结算的手机号码，例如 `13812345678`。注意号码间不要有空格。



![](img/ef619b31f4e79bef1ab889f4da92e388_11.png)

![](img/ef619b31f4e79bef1ab889f4da92e388_11.png)

![](img/ef619b31f4e79bef1ab889f4da92e388_12.png)

![](img/ef619b31f4e79bef1ab889f4da92e388_12.png)

![](img/ef619b31f4e79bef1ab889f4da92e388_13.png)

## 第六步：提交并确认
所有信息填写完毕后，仔细检查一遍。确认无误后，点击页面上的蓝色“提交”按钮。
系统会弹出“确定要提交吗？”的确认框，点击“确认”即可完成提交。



![](img/ef619b31f4e79bef1ab889f4da92e388_13.png)

提交成功后，页面会提示“已成功加入创作者分成计划”。之后，你就可以开始准备发布视频内容了。

![](img/ef619b31f4e79bef1ab889f4da92e388_15.png)

例如，素材库中包含《还珠格格》、《我不是药神》、《大话西游》等经典影视剧的授权片段。

本节课中我们一起学习了填写视频号创作者计划邀请函的完整流程。从打开微信、找到账号信息，到准确填写昵称、粉丝数、邀请人、实名信息和手机号，最后提交确认。完成这一步，你就正式加入了计划，可以开始使用提供的正版素材进行内容创作，迈出视频号运营的第一步。
