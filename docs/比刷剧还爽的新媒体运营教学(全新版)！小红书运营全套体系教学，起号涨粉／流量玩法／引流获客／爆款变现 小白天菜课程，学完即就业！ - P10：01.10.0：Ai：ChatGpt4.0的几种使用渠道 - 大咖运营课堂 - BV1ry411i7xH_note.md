# 新媒体运营：P10：01.10.0：ChatGPT 4.0 的几种使用渠道 🚀

在本节课中，我们将学习如何获取和使用目前最强大的AI对话模型——ChatGPT 4.0。对于新媒体运营人员而言，掌握AI工具是提升工作效率、拓展能力边界的关键。我们将介绍几种不同的渠道，包括免费和付费选项，帮助你找到最适合自己的方式。

## 概述：AI对运营的重要性

在之前的课程中，我们提到过，在当前时代，最需要接触和使用AI智能对话模型的两个行业是程序员和新媒体运营。

程序员可以利用AI编写代码、调试程序。而作为运营人员，AI可以帮助我们撰写短视频脚本、制作PPT、生成表格和报告等。

掌握AI工具，意味着你拥有了一个私人的知识专家。你无需具备特定领域的深厚专业知识，也能借助AI接触和完成相关工作。这极大地降低了学习和工作的门槛。

## 渠道一：微软 Edge 浏览器 Copilot 🌐

首先，我们来看第一个推荐的渠道——微软Edge浏览器内置的Copilot功能。这是微软基于ChatGPT技术打造、集成在Windows系统中的AI助手。

### 如何使用 Copilot

使用Copilot有两个前提条件：
1.  登录微软账号（可使用Outlook邮箱注册）。
2.  需要“梯子”（VPN）才能访问该服务。

Copilot通常集成在浏览器右侧工具栏。在最新版Win11系统中，也可能在任务栏直接出现图标。打开后，界面会提供三种对话模式：
*   **更有创造力**：AI的回答更具开放性和创意。
*   **平衡**：介于创造力和精确性之间，适合一般用途。
*   **更精确**：回答更为严谨和准确。

**示例：生成短视频脚本**
我们可以输入指令：“请帮我写一个关于装修的短视频脚本，时长控制在3分钟左右。” AI会快速生成包含场景、旁白、镜头语言的完整脚本。

此外，Copilot的优势在于它能分析你当前打开的网页内容，提供相关总结或建议，这对信息搜集和内容创作很有帮助。

### Copilot 的优缺点分析

上一节我们介绍了Copilot的基本用法，本节中我们来看看它的优缺点。

**优点：**
*   免费使用，与系统深度集成。
*   可分析当前网页内容。

**缺点：**
*   **回答精准度稍逊**：虽基于GPT-4，但经过微软调教，在某些专业回答上可能与官方版本有细微差距。
*   **连续对话能力受限**：无法在同一对话窗口中进行过多次连续提问，有时需开启新对话。
*   **记忆功能有所缺失**：对历史对话内容的记忆和分析能力不如官方ChatGPT完整，可能影响后续回答的连贯性和个性化。

尽管有这些限制，但对于日常运营工作，Copilot的功能已完全足够。如果你的Edge浏览器没有Copilot图标，可以尝试下载最新的Edge Dev（开发者）测试版。

## 渠道二：第三方集成平台与共享账号 🛠️

除了官方和浏览器集成渠道，网络上还有许多开发者利用ChatGPT的API搭建的第三方平台。

以下是几个可用的第三方平台示例：

**1. 灵感AI**
*   **特点**：集成了GPT-3.5、GPT-4、GPT-4 Turbo以及国内主流大模型（如讯飞星火、文心一言等）。
*   **费用**：新注册用户有一定免费额度（如7天内无限次使用，但有限制次数）。后续有付费套餐（如19.9元/月）。
*   **缺点**：生成速度可能较慢。
*   **访问**：无需“梯子”。

**2. Wee AI**
*   **特点**：提供GPT-3.5和GPT-4模型，响应速度较快。
*   **费用**：注册赠送一定额度的Token（计算单位），但免费额度通常仅适用于GPT-3.5。尝试使用GPT-4可能需要付费。
*   **缺点**：免费额度有限，需要“梯子”。

**3. ChatGPT共享账号平台**
*   **特点**：一个分享ChatGPT账号（包括Plus付费账号）的网站。账号状态用颜色标示（绿色-空闲，橙色-近期使用，红色-使用中，金色-可能为GPT-4账号）。
*   **使用**：找到“绿色”或“金色”账号，点击后设置密码即可登录使用。
*   **注意**：GPT-4账号因使用人数多，可能遇到“对话次数达到上限”的提示（官方限制为每3小时约30条消息）。GPT-3.5账号通常更易获得。
*   **访问**：需要“梯子”。

## 渠道三：国内AI大模型替代品 🇨🇳

对于不希望使用“梯子”的运营者，国内多家科技公司也推出了优秀的AI大模型，它们是ChatGPT的可行替代品。

以下是几个主流的国内AI大模型：

*   **百度·文心一言**：百度推出的对话模型，有免费版和付费的“4.0”版本。在文案生成、创意构思方面能满足基本运营需求。
*   **字节跳动·豆包AI**：字节跳动出品的AI助手，同样可以生成短视频脚本、文案等内容，响应速度快。
*   **阿里巴巴·通义千问**：阿里云的大模型，功能全面。
*   **昆仑万维·天工AI**：另一个知名的国内大模型。

**示例对比：**
输入相同指令“请帮我写一个关于装修的短视频脚本文案，要求镜头丰富，语言吸引人”，文心一言和豆包AI都能生成结构完整、包含景别、画面、台词、时长的脚本。虽然在创意深度和语言灵活性上可能与GPT-4存在差距，但对于日常运营辅助、激发灵感来说完全足够。

![](img/10b0ada04bf2b4f171b6cc9f481d51b9_17.png)

![](img/10b0ada04bf2b4f171b6cc9f481d51b9_19.png)

选择哪一个，取决于你的具体需求和使用习惯。建议都尝试一下，选择用起来最顺手、最能提升你工作效率的那一个。

## 总结与对比

本节课我们一起学习了体验和使用ChatGPT 4.0的几种主要渠道。

我们来简单总结和对比一下：
*   **微软 Copilot**：方便、免费，但需“梯子”，能力略有阉割。
*   **第三方平台/共享账号**：提供了访问GPT-4的途径（免费或低价），但稳定性、速度和安全性需注意，多数需“梯子”。
*   **国内AI大模型**：访问便捷，完全免费或成本极低，无需“梯子”，是大多数国内运营人员的务实选择。

作为运营人员，工具的核心目的是提升效率。**“什么东西好用，我们就去用什么”**。如果有条件，鼓励你体验一下ChatGPT 4.0的强大能力；如果条件有限，国内的AI工具也足以让你的运营工作如虎添翼。
