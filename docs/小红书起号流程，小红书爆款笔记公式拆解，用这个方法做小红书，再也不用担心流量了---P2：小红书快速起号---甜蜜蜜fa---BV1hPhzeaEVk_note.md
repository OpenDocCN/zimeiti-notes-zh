一个新手从零开始做小红书，30天之后月入5000元是有可能的。

![](img/bf50df43cec411b84f4ec4043234fe3b_11.png)

在小红书认真发布帖子的博主中，月收入超过1000元的占到了57%，月收入超过5000元的占到了28%。

![](img/bf50df43cec411b84f4ec4043234fe3b_13.png)

![](img/bf50df43cec411b84f4ec4043234fe3b_15.png)

也就是说，只要你开始做小红书，就有28%的概率纯靠小红书养活自己。接下来的内容将帮助你获得这28%的入场券。首先，我们需要了解小红书是一个什么样的平台。

![](img/bf50df43cec411b84f4ec4043234fe3b_17.png)

它的本质是一个社交分享型平台。我们要注意“社交”和“分享”这两个关键词，这意味着我们必须为他人提供有价值的内容。

![](img/bf50df43cec411b84f4ec4043234fe3b_19.png)

这是做起来一个小红书号的核心。因此，有四类人不适合做小红书。

![](img/bf50df43cec411b84f4ec4043234fe3b_21.png)

以下是这四类人的具体描述：
*   第一，不喜欢创作和表达，没有持续输出内容能力的人。
*   第二，不善于检索、阅读和总结信息的人。
*   第三，心理脆弱，容易放弃的人。
*   第四，没有执行力的人。

这四条只要你占了一条，后面的内容你都不用再听了。如果你一条都不占，那么恭喜你，基础能力没有问题。接下来，我们会手把手教大家解决三个问题。

![](img/bf50df43cec411b84f4ec4043234fe3b_23.png)

以下是三个核心问题：
*   第一，如何从零开始起一个号。
*   第二，如何保证持续有高质量内容的输出。
*   第三，如何确保内容能够变成收益。

![](img/bf50df43cec411b84f4ec4043234fe3b_25.png)

![](img/bf50df43cec411b84f4ec4043234fe3b_27.png)

![](img/bf50df43cec411b84f4ec4043234fe3b_29.png)

这三个问题我会分为“起号篇”和“变现篇”两期讲给大家。本期是“起号篇”。

上一节我们明确了学习目标，本节中我们来看看起号的第一步。起号第一件事情不是起名，不是搞头像，搞简介，而是确定自己到底要卖什么赚钱。

它决定了整个号的方向。我们做小红书不是为了做慈善、做分享，我们做小红书的唯一目的就是赚钱。有些人辛辛苦苦运营了半天，有了几千几万粉丝，然后发现自己的方向赚不来钱。

比如有些人喜欢做每日好剧分享、每日读书打卡的号，热度也很高，但没有什么商业价值。这种核心问题就在于，一开始就没有想清楚自己这个号到底要卖什么东西。

所以，第一步你要确定的是自己到底要卖什么产品，然后在这个方向上确定号怎么做。你要卖衣服，你就做穿搭号；你要卖付费课程，你就做培训号；你要卖吃的，那就做美食号。

![](img/bf50df43cec411b84f4ec4043234fe3b_31.png)

你自己擅长不擅长某个领域，并不是你选择方向的依据。选择方向的依据是：你要在这个方向上，第一能够持续创作内容，第二有产品可以交付，那就可以去做。至于擅长不擅长，不擅长你可以学。所有人刚起号的时候，都不可能比这个领域的大V更加擅长。

![](img/bf50df43cec411b84f4ec4043234fe3b_33.png)

能力都是做着做着就起来了。就算你刚开始不懂穿搭，你做上200个穿搭的帖子出来，你还能不会吗？

如果实在不知道该做什么，这里可以给你一个参考。小红书里面最赚钱的领域是家居、护肤，其次是情感、健身健康、母婴和美食。其他领域的收入普遍都没有这些高。所以你在这里面随便挑一个你可以做的就可以了。

确定方向之后，是一些基础的工作。以下是基础工作的具体步骤：

![](img/bf50df43cec411b84f4ec4043234fe3b_35.png)

![](img/bf50df43cec411b84f4ec4043234fe3b_37.png)

*   **第一个是起名**。给你一个万能模板：**要做的事情 + 标签 + 个人昵称**。要做的事情和标签之间要有一定的反差感。比如：卖猪肉的转行医学生于宇、卖烧烤的下岗基金经理曾格、抑郁的心理医生刘六、不会穿搭的女装店主、不会做饭的美食博主、不会骑自行车的女神旅行达人。这种反差太好找了。
*   **第二，头像随便弄，别经常换就行**。没有什么讲究。
*   **第三，把默认小红书改成自己的微信号**。这样后面有人私信你，你就可以直接告诉他加我小红书号就可以了，这样可以合规引流。
*   **第四，简介还是万能模板**：**身份（我是谁）+ 我在做什么 + 这个号会发什么**。比如说这个人身份是九零后自媒体创业者，做什么：帮人搞流量、搞变现，号里面会发什么：会发小红书运营、自媒体干货、IP打造。
*   **第五，熟悉平台规则**。以下是关键规则：
    *   禁止炫富，不要带给人焦虑。制造焦虑可以短期有流量，但长期来讲既会被平台打压，又不利于人设塑造。
    *   禁止直接引流。帖子中间不能出现任何微信和公众号等信息，更不能出现二维码。合规的引流方式以后我们会讲。
    *   不要出现不文明言论、不良言论。
    *   不要过度擦边。
    *   不要抄袭大V的内容。可以借鉴选题，但不能抄写内容。
    *   可以发泄负能量，但不要宣传负能量。比如你可以说自己心情不好，但不要宣传“人就应该抑郁，不抑郁不是正常人”这样的观点。

![](img/bf50df43cec411b84f4ec4043234fe3b_39.png)

到这里，所有基础工作都做完了。接下来是最重点的部分：如何做出好内容，如何把账号给做起来。

![](img/bf50df43cec411b84f4ec4043234fe3b_41.png)

做出好内容、做起来账号总共有五个方面。以下是五个核心要点：

![](img/bf50df43cec411b84f4ec4043234fe3b_43.png)

*   **第一，找到一个稳定的创作模式，进而找到一个稳定的素材来源**。比如这个博主，他的创作模式就是每次做一道新菜分享给大家，菜品的做法显然是无穷无尽的，所以他这个模式可以保证长期持续稳定的更新。再比如这个人，你可以看到他的内容就是不同类型的衣服穿搭，衣服的类型款式、穿搭的方式是年年都不同、无穷无尽的，所以它也可以保证自己源源不断地有输出的内容。每一个领域都有类似的创作模式，稍微研究一下对标账号，你就可以把它搞清楚。如果自己只是有感而发，或者根据自己的经验、人生经历、知识去创作，你写个一二十篇恐怕就没东西写了。所以你必须要找到一个稳定的创作模式和素材的来源。
*   **第二，内容必须有价值**。要么能解决需求痛点，要么能引起情绪波动。如果别人想做红烧肉但是不知道怎么做，你教会了他，这就是解决了需求。如果他晚上刷小红书的时候肚子正在饿着，你给他提供了一个美食视频，这就能引起情绪波动。这二者同时都有，算是满足底线要求的合格内容。只满足了一条属于低质量的内容。都不满足，那就是毫无价值的垃圾内容，没有发出来的必要。
*   **第三，封面既要引发用户兴趣，又要解决用户问题**。内容的情绪价值和痛点价值，全部都要体现在封面上。具体的操作因领域而异，比如说搞穿搭的照片要有个性、亮眼，而且要养眼；搞美食的图片拍出来有食欲就可以了；搞知识的要展示出最引人注目的观点，让别人能关注到你。如果你灵感迸发，想到了一个天才文案写在封面上，靠着封面说不定就能爆一个帖子。这是一种感觉，只能多去揣摩同领域的大号，然后自己去试，做的多了自然就会了。
*   **第四，账号要做出个人风格**。这个牵扯到个人IP的问题，后面我们会详细讲。这里简单说就是让你“懒”一点：不要瞎折腾封面，不要乱换，一直用一套模板就可以了；视频的片头音乐不要乱换，一直用一套就行了；说话的风格不要刻意去学别人的，按自己的习惯坚持表达下去就可以了。比如这个人，你看他所有的封面都是一个风格，时间长了就记住他了。
*   **第五，宁可减少数量，也要保证质量**。如果能三天更一篇牛逼的，就不要一天更三篇垃圾的。但是你要保证更新频率稳定，你不要两个月更一篇。一个月之内一定要更一篇，如果一个月之内没有更新，平台就会觉得你断更了，流量会少很多。

![](img/bf50df43cec411b84f4ec4043234fe3b_45.png)

到这里，如果今天讲的东西全部都搞定，坚持做下去，起号绝对不是问题。

![](img/bf50df43cec411b84f4ec4043234fe3b_47.png)

变现相关有五个后续问题：
*   第一，如何让同样的内容阅读量跳跃式地增长。
*   第二，如何大规模跳跃式地涨粉。
*   第三，如何安全高效地引流和变现。
*   第四，如何让帖子发布一年后也能继续带来收入。
*   第五，小红书群体的赚钱逻辑是什么。

![](img/bf50df43cec411b84f4ec4043234fe3b_49.png)
