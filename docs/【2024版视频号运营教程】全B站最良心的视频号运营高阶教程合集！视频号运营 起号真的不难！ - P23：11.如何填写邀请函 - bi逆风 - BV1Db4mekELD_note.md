# 视频号运营教程：P23：11.如何填写邀请函

在本节课中，我们将学习如何正确填写视频号创作者计划的邀请函。我们将一步步演示从查找个人视频号信息到最终提交表单的完整流程，确保信息准确无误。

## 查找视频号信息

![](img/1a14dedbd62b44541950646b6b5abbbd_1.png)

上一节我们介绍了如何注册视频号，本节中我们来看看如何找到自己的账号昵称和粉丝数，这是填写邀请函的第一步。

**第一步：打开微信应用**
打开手机上的微信应用。



![](img/1a14dedbd62b44541950646b6b5abbbd_1.png)

**第二步：进入“发现”页面**
在微信底部导航栏中，点击进入“发现”页面。



![](img/1a14dedbd62b44541950646b6b5abbbd_3.png)

![](img/1a14dedbd62b44541950646b6b5abbbd_4.png)

![](img/1a14dedbd62b44541950646b6b5abbbd_3.png)

**第三步：找到“视频号”入口**
在“发现”页面中，向上滑动屏幕。找到并点击“视频号”入口（图标为黄色蝴蝶形状或直接显示“视频号”文字）。



![](img/1a14dedbd62b44541950646b6b5abbbd_4.png)

**第四步：进入个人中心**
进入视频号页面后，点击右上角的个人头像图标，进入个人中心。



![](img/1a14dedbd62b44541950646b6b5abbbd_6.png)

![](img/1a14dedbd62b44541950646b6b5abbbd_6.png)

![](img/1a14dedbd62b44541950646b6b5abbbd_7.png)

**第五步：查看账号信息**
在个人中心页面下滑，找到“我的视频号”栏目。这里会显示你的**账号昵称**和**粉丝数量**。
例如：`昵称：天道酬勤，粉丝：39917`



![](img/1a14dedbd62b44541950646b6b5abbbd_7.png)

## 填写邀请函表单

找到账号信息后，接下来我们开始填写邀请函。以下是需要填写的具体项目：

**第一项：视频号账号昵称**
填写你在上一步查看到的视频号昵称，例如：`天道酬勤`。

**第二项：视频号粉丝数量**
填写你在上一步查看到的粉丝数量，例如：`39917`。

**第三项：活动邀请人**
由于是通过特定渠道加入，此处需填写邀请人姓名，例如：`张良` 或 `张良老师`。

**第四项：实名信息**
填写你本人的真实姓名，例如：`周建国`。

**第五项：结算手机号**
填写用于接收验证码和后续结算的手机号码，例如：`13812345678`。注意号码中间不能有空格。



![](img/1a14dedbd62b44541950646b6b5abbbd_9.png)

![](img/1a14dedbd62b44541950646b6b5abbbd_9.png)

![](img/1a14dedbd62b44541950646b6b5abbbd_10.png)

![](img/1a14dedbd62b44541950646b6b5abbbd_11.png)

所有信息填写完毕后，仔细检查一遍。



![](img/1a14dedbd62b44541950646b6b5abbbd_10.png)

确认无误后，点击蓝色的 **“提交”** 按钮。系统会弹出确认提示，点击 **“确认”** 即可完成提交。



![](img/1a14dedbd62b44541950646b6b5abbbd_11.png)

提交成功后，页面会显示“已成功加入创作者分成计划”的提示。

![](img/1a14dedbd62b44541950646b6b5abbbd_13.png)

## 后续操作与素材支持

![](img/1a14dedbd62b44541950646b6b5abbbd_14.png)

成功提交邀请函后，导师通常会提供起步帮助。例如，为账号设置合适的昵称（如“建国影视”），并直接提供可发布的视频素材。

这些素材来源于有版权的正规素材库，涵盖了影视、动漫等多种类型，例如《还珠格格》、《我不是药神》、《大话西游》等，确保创作者可以安全使用。
