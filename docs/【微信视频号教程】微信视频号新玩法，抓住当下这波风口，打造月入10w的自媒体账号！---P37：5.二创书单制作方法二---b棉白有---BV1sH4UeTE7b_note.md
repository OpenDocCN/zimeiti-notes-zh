# 微信视频号教程：P37：5.二创书单制作方法二

在本节课中，我们将学习如何通过“二创”的方式制作书单视频。我们将详细讲解如何获取素材、提取文案、替换画面以及最终合成一个全新的视频。这种方法能帮助你高效地产出内容，抓住平台流量。

![](img/c8720c0b6528e4559b4983de7bee986a_1.png)

## 素材获取与初步处理

![](img/c8720c0b6528e4559b4983de7bee986a_3.png)

上一节我们介绍了二创的基本思路，本节中我们来看看具体的操作方法。首先，我们需要找到合适的原始视频素材。

![](img/c8720c0b6528e4559b4983de7bee986a_5.png)

![](img/c8720c0b6528e4559b4983de7bee986a_7.png)

以下是获取素材的步骤：
1.  在抖音搜索目标书籍名称，例如“走进大学城”。
2.  在搜索结果中，点击“商品”标签。
3.  选择按“销量”排序，找到相关的带货视频。
4.  进入达人主页，在其视频列表中选择一个合适的视频作为素材。

![](img/c8720c0b6528e4559b4983de7bee986a_9.png)

![](img/c8720c0b6528e4559b4983de7bee986a_11.png)

例如，我们找到一个时长1分58秒的视频，其内容是关于大学介绍的。这类视频前后声音可能不一致，但这并不影响我们使用。

接着，我们需要对原始视频进行处理。
1.  复制视频链接，使用去水印工具去除视频水印。
2.  打开剪映，将去水印后的视频导入。

## 视频分割与文案提取

![](img/c8720c0b6528e4559b4983de7bee986a_13.png)

获取素材后，下一步是分割视频并提取其中的文案。

操作步骤如下：
1.  在剪映中，将时间轴放大，便于精确操作。
2.  找到视频中声音发生变化的位置（例如53秒处），使用分割工具将视频剪开。
3.  保留我们需要的片段（例如前半部分），删除不需要的部分。
4.  将处理好的片段导出为一个新视频。

接下来，我们需要提取这段视频的文案。
1.  打开“配音神器”这类工具。
2.  使用“视频提取文案”功能，导入刚才导出的视频片段。
3.  系统会自动识别并生成文字文案。
4.  （提示：如果工具需要金币，可以通过每日签到获取。）

## 音频制作与画面替换

文案提取完成后，我们就可以制作新的音频，并开始替换原始视频的画面，这是二创的核心环节。

首先进行音频制作：
1.  在“配音神器”中，将提取的文案粘贴到配音框。
2.  选择一个合适的声音角色进行配音。
3.  生成配音音频并导出。

接下来进行画面替换，以下是具体步骤：
1.  回到剪映，打开原始视频工程。
2.  定位到需要替换的画面片段（例如一段3秒的毕业典礼镜头）。
3.  在抖音搜索相关关键词（如“高校毕业典礼”），找到新的素材视频，下载并去水印。
4.  在剪映中，删除原视频的这3秒片段，将新找到的素材视频导入并放置在相同位置。
5.  将新素材的时长精确裁剪为3秒，确保与前后画面衔接。
6.  在新旧片段之间添加一个转场效果（如“叠化”）。
7.  将新添加片段的原声音量调整为0。

我们可以重复此过程，替换多个片段。例如：
*   将“打高尔夫”的画面替换为搜索到的“美女打高尔夫”视频。
*   将“宿舍单间”的画面替换为搜索到的“精装修室内”视频。
替换时需注意，新画面的内容与格调应与原视频描述的场景大致相符。

## 音画合成与最终调整

所有素材替换完毕后，我们需要将新的音频与画面进行合成，并做最后的精细化调整。

操作流程如下：
1.  关闭原始视频中所有片段的原声。
2.  将之前通过“配音神器”生成的新配音音频导入剪映，并拖到音频轨道。
3.  检查音频与视频的长度是否匹配。如果不匹配，可以：
    *   **视频过长**：对视频进行“常规变速”，适当加快。
    *   **视频过短**：添加更多相关画面素材，或对视频进行减速。
    *   目标是使画面时长与音频时长基本一致，多余部分可删除。
4.  对于视频中保留的、带有原声的片段（如人物采访），可以保留其声音，但建议替换其画面，并缩短人物正面特写的时长，以避免潜在的人脸版权风险。
5.  根据配音内容，在适当位置添加对应的画面（如提到“游戏”时插入游戏画面），增强匹配度。

## 添加包装与导出视频

核心内容制作完成后，还需要添加字幕、背景音乐等包装元素，让视频更完整。

以下是收尾步骤：
1.  **添加背景音乐**：在音频轨道添加一段合适的背景音乐，调节音量使其不干扰配音。
2.  **生成字幕**：使用剪映的“识别字幕”功能，为新的配音自动生成字幕。
3.  **遮盖原字幕**：将新生成的字幕样式进行调整（如改变颜色、大小、位置），以完全遮盖视频中可能残留的原始字幕。
4.  **检查水印**：确保所有使用的素材都已去除水印。如有遗漏，可将画面放大以遮盖水印。
5.  **导出设置**：选择分辨率为1080P，帧率为60帧，使用推荐码率，然后导出视频。

![](img/c8720c0b6528e4559b4983de7bee986a_15.png)

本节课中我们一起学习了二创书单视频的完整制作流程。从获取素材、提取文案，到替换画面、合成音效，再到最后添加字幕和背景音乐。关键在于多练习，初期制作可能较慢，但熟练后效率会大幅提升。记住，剪辑技能重在实操，通过不断练习，你也能快速制作出高质量的原创视频。
