上一节我们概述了账号定位的重要性，本节中我们首先来看看人设定位。人设定位的目的是让用户清楚地知道内容创作者是谁，有什么样的背景和专业性。一个清晰的人设能建立信任感，并让用户更容易记住你。

![](img/049b1f7ce606231cf5662c1541d0b081_56.png)

人设定位的核心目标有两个：
1.  **建立专业性**：让用户相信你是在特定领域长期深耕的专家，你提供的内容具有参考价值。
2.  **增强记忆点**：通过一系列统一的视觉和内容元素，让用户能在海量信息中快速识别并记住你。

![](img/049b1f7ce606231cf5662c1541d0b081_58.png)

![](img/049b1f7ce606231cf5662c1541d0b081_68.png)

以下是打造人设的关键元素：
*   **基础形象（账号五件套）**：包括头像、名称、简介、背景图和封面。这些是用户对你的第一印象。
*   **内容展现**：包括视频中的标准动作、固定的开场白或口头禅、统一的视频节奏或特效风格。这些元素能形成强烈的个人风格。

**关于人设的注意事项**：
*   **出镜的重要性**：在短视频平台，真人出镜、进行口播分享是打造人设最有效的方式。图文类账号（如某些知识分享号）若不出镜，则较难建立鲜明的人设，用户可能只记住内容而记不住创作者。
*   **人设的必要性**：并非所有账号都必须有强个人色彩的人设。例如，一些影视解说类账号，用户关注的可能是内容本身（如解说的电影）而非解说者个人。你需要根据赛道特点决定。

内容定位主要围绕以下三个核心点展开：

1.  **表达核心**：你的每一条内容想表达的中心思想是什么？是分享知识、吐槽现象、记录生活还是提供解决方案？内容主题必须清晰。
2.  **吸引策略**：你的内容如何吸引用户？在同质化竞争中，你的差异化和创意点在哪里？例如，家居博主可以分享“平价好物改造”，穿搭博主可以聚焦“小个子通勤穿搭”，从大领域中切出更精准、更有吸引力的细分方向。
3.  **流量获取**：你的内容如何获得平台更多的推荐流量？这取决于两点：
    *   **用户反馈**：内容发布后，用户的点赞、评论、分享、完播率等数据。
    *   **赛道热度**：在你这赛道内，当前平台上的用户更关心哪些子话题？选择用户基数更大、更热门的内容方向，有助于获得更好的初始流量。

**核心公式**：`优质内容 = 清晰主题 + 差异化创意 + 契合平台热点`

**方案A：育儿经验分享型**
*   **人设定位**：一位拥有3岁宝宝的妈妈，分享亲身养娃、带娃的真实经历。
*   **商业定位**：围绕宝宝用品进行变现，如推荐纸尿裤、奶瓶、童装、食品等。
*   **内容定位**：解决宝妈们的具体痛点，例如“宝宝不爱吃奶粉怎么办？”、“如何调整宝宝睡眠规律？”、“孩子挑食如何处理？”。

**方案B：产后恢复指导型**
*   **人设定位**：一位产后体重增加30斤，但成功恢复甚至优化了身材的宝妈。
*   **商业定位**：推荐产后恢复相关产品，如塑身衣、护肤品、身材管理计划等。
*   **内容定位**：分享产后恢复知识，如“产后修复标准动作”、“妊娠纹科学处理”、“缓解产后情绪焦虑”。

本节课中我们一起学习了新媒体账号定位的三个核心组成部分：**人设定位**、**商业定位**和**内容定位**。
*   **人设定位**决定了账号的个性和信任度。
*   **商业定位**规划了账号的变现路径和最终目标。
*   **内容定位**指导着具体的创作方向和策略。
