# 抖音运营实战：P25：口播类视频拍摄方法-7

在本节课中，我们将学习口播类视频的具体拍摄方法。我们将通过一个真实案例，详细拆解其构图、内容定位和拍摄技巧，帮助你掌握制作高质量口播视频的核心要点。

---

上一节我们介绍了视频拍摄的通用注意事项，本节中我们来看看口播类视频的具体拍摄方法。

![](img/ce9c9a99ced17728e00c582a32292bb3_1.png)

口播类视频，无论内容主题是金融、装修、养生还是情感，其核心拍摄方法都是相通的。同样，记录日常生活、拍摄段子或美食视频，其拍摄逻辑也基本一致。因此，掌握以下两种类型的拍摄方法，几乎可以覆盖所有场景。

首先，我们来看第一个类型：口播视频。我将以我们的一位学员“种小草”为例进行讲解。

![](img/ce9c9a99ced17728e00c582a32292bb3_1.png)

这位学员是一名40岁、带着三个孩子的宝妈，她的账号名为“种小草”，主要做情感口播内容。她曾有一条视频播放量超过8000万，并凭借该视频涨粉30万。为什么她的一条视频能取得如此效果，而我们涨粉却很难？

关键在于她的内容吸引了非常精准的群体。你看她的视频，一眼就能明白她想吸引的是谁。例如，如果我的目标群体是20多岁的学生，那么我拍摄的所有内容都围绕学生感兴趣的话题，这样涨粉就会既快又精准。

我们先从构图讲起。抖音是**九比十六**的竖屏平台。关键在于视频内容本身是采用横屏还是竖屏构图。观察案例，她使用的是**四比三**的横屏构图。这里需要强调：无论你做哪种类型的视频，构图风格需要统一。如果你选择横屏，就全部使用横屏；如果选择竖屏，就全部使用竖屏。保持稳定是需要注意的细节。

在构图中，人物主体应占据画面中央约**70%** 的面积。不要怼脸拍摄，那样会让人观感不适。确保画面的左、右、上方留有适当空白，这样会让画面看起来清晰、舒适。

如果你认为自己外貌并不出众，是否还能做口播？答案是肯定的。抖音的成功并非完全依赖颜值。口播的核心在于传达内容和知识。你只需要让他人觉得你清晰、体面即可。如果对自己外貌不自信，可以开启美颜滤镜。在这个时代，合理使用美颜功能能让每个人都呈现更好的状态。请放心使用，只要不过于夸张即可。

接下来，我们直接观看她那条涨粉30万的视频。我的要求是：看完后请直接回答，这个视频想要吸引哪个特定群体？

![](img/ce9c9a99ced17728e00c582a32292bb3_3.png)

（视频台词示例：“跟婆婆在一起住是什么样的感受？我跟我婆婆在一起住了16年了…我最大的感受就是这十多年以来，我娘家人很少在我家住上几天…”）

![](img/ce9c9a99ced17728e00c582a32292bb3_3.png)

有同学可能回答“婆媳关系”。但这是一种关系，具体是哪个群体呢？实际上，对婆媳关系感触最深的，绝非男性群体，也非未结婚的年轻女性。最能产生共鸣的，是**已经结婚的女性群体**。因此，这个视频精准吸引的是已婚女性。我们稍后可以通过数据来验证这一点。

同时注意，案例中的博主也使用了美颜。如果你对自己的外貌不自信，可以适当开启美颜功能，让自己看起来更得体。她的构图也是**四比三**的横屏。

有同学会问：老师，口播视频一定要在户外拍吗？可以在家拍吗？当然可以。拍摄地点不限，关键在于背景要整洁美观，避免过于杂乱。

接着是另一个常见问题：

![](img/ce9c9a99ced17728e00c582a32292bb3_5.png)

“老师，拍口播需要把稿子背下来吗？我记性不好背不下来怎么办？” 完全不需要背诵。在剪辑环节，我会推荐一个实用工具，让你可以边看稿子边录制，这会让过程简单得多。

此外，做口播视频最关键的一点是**声音必须清晰**。为此，我推荐你购买一个手机收声麦克风（领夹麦），价格很便宜，只需十几元。在淘宝、京东等平台都能买到。它能极大提升你视频的音频质量。

![](img/ce9c9a99ced17728e00c582a32292bb3_5.png)

以下是口播视频拍摄的核心要点总结：

1.  **构图统一**：选择横屏（如**4:3**）或竖屏（**9:16**）后，保持所有视频构图风格一致。
2.  **主体突出**：人物占据画面约**70%**，避免怼脸，注意留白。
3.  **内容精准**：明确你的视频想要吸引**哪个具体群体**，并围绕该群体的兴趣和痛点创作内容。
4.  **形象管理**：合理使用美颜滤镜，保持画面中形象清晰、体面。
5.  **背景简洁**：无论室内外，确保拍摄背景整洁、美观。
6.  **提词工具**：使用提词器或相关APP实现**边看边录**，无需背诵稿子。
7.  **收音清晰**：建议使用**外接麦克风**（如领夹麦）来确保音频质量。

---

本节课中，我们一起学习了口播类视频的拍摄方法。核心在于**统一的构图、精准的受众定位、清晰的声音**以及利用工具（如提词器、麦克风）来降低拍摄难度。记住，口播的核心是价值传递，而非颜值。掌握这些技巧，你也能制作出吸引目标观众的高质量口播视频。
