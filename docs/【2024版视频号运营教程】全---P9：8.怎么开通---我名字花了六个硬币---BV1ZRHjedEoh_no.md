## 注册流程详解
对于不识字或操作不熟悉的学员，关键在于将步骤极致简化。我们采用的方法是：**用箭头图示代替文字说明**。

具体操作逻辑如下：
```
如果学员不识字 -> 则提供带箭头的截图 -> 箭头指向哪里就点击哪里
```

当时，我向学员发送了一份注册流程表。流程表将步骤分解得非常清晰。

![](img/d3fa52d07c9c06367ce67b8910b07d2c_4.png)

该流程表的核心结构如下：

第一步：打开微信，点击右下角“我”。
第二步：进入“设置”页面。
第三步：找到并点击“通用”选项。
第四步：选择“发现页管理”，开启视频号入口。

由于该学员不识字，仅认识数字，下方的文字说明对他无效。因此，我采用了上述箭头指示法，引导他逐步点击，最终成功完成注册。

这个方法证明了，只要教学方法得当，任何人都能学会。没有学不会的学生，只有不会教的老师。
