## 开店流程 📝

上一节我们介绍了课程的整体框架，本节中我们来看看如何开设小红书店铺。开店是启动小红书电商的第一步，流程非常简单直接。

以下是开店的具体步骤：
1.  打开小红书应用。
2.  点击左上角的菜单图标。
3.  选择“创作中心”。
4.  点击“更多服务”选项。
5.  选择“店铺”，点击“立即入住”。
6.  按照官方要求提交资料完成入住。

![](img/73def4f30000e7f09f13ae8d51fd7c42_5.png)

小红书店铺主要分为个人店和企业店两种形式。

**个人店**包含两种类型：
*   **个人店**：仅需身份证和**1000元保证金**即可开通。
*   **个体工商户店**：需要身份证、营业执照和保证金。

**企业店**包含三种类型（普通企业店、专卖店、旗舰店），均需要营业执照，并需缴纳**600元年费**。

对于新手，建议直接开通个人店。经过实测，平台并未对企业店有流量倾斜。唯一的区别在于，一份营业执照可以开设三家店铺，而个人身份只能开设一家。但初期不建议盲目开设多个店铺，应集中精力做好一个。

![](img/73def4f30000e7f09f13ae8d51fd7c42_7.png)

开店的主要成本构成如下：
*   **保证金**：1000元（可退）。
*   **平台扣点**：约为营业额的**5%**。
*   **支付渠道费**：每单约**0.7%**。
*   **流动资金**：初期可采用无货源模式，准备2000-3000元即可启动。

## 选品策略 🎯

成功开店后，下一步就是决定卖什么。选品是电商成功的关键，其核心逻辑是利用市场数据，而非凭空想象。

选品的原则是：在其他电商平台上的爆款产品，有很大概率在小红书上也能成为爆款。我们应借助成熟平台的选品能力来弥补自身不足。

以下是三个推荐的选品渠道：
1.  **某宝/某东**：寻找符合小红书平台调性（如高颜值、新奇特、生活方式类）的爆款产品。
2.  **某音精选联盟**：查看精选联盟排行榜，筛选出符合小红书用户调性的热卖商品。
3.  **某多多销量排行榜**：选择适合小红书的类目，从其销量排行榜中挑选潜在爆品。

整个选品过程可以概括为：**从各大平台榜单筛选爆品 -> 判断其与小红书用户画像的匹配度 -> 搬运至小红书店铺进行测试**。这种方式高效且稳妥。

## 流量获取与内容创作 📈

上一节我们掌握了如何借助市场力量选品，本节中我们来看看如何为商品获取流量。小红书本质是种草平台，店铺超过90%的流量都来自于发布的笔记。因此，获取流量的核心是多发优质笔记，而非开设多个店铺（店群模式效果不佳）。

用户购买的完整路径是：**看到笔记 -> 对产品产生兴趣 -> 进入店铺 -> 完成购买**。所以，如何发布笔记是小红书电商成败的关键。

![](img/73def4f30000e7f09f13ae8d51fd7c42_11.png)

直接发布硬广效果很差，因为小红书用户认知度高，难以建立信任。正确的方法是**以素人（真实用户）的视角进行种草分享**。

![](img/73def4f30000e7f09f13ae8d51fd7c42_13.png)

![](img/73def4f30000e7f09f13ae8d51fd7c42_15.png)

制作种草笔记的方法很简单：
1.  **素材来源**：去某音、某宝、某东等平台的商品评论区或买家秀中，寻找美观、真实的图片或视频素材。
2.  **文案撰写**：基于买家真实的评价和使用体验进行修改和润色。这些评价本身就是最自然的种草文案。

![](img/73def4f30000e7f09f13ae8d51fd7c42_17.png)

当单个账号的运营模式（SOP）跑通并稳定出单后，可以考虑通过**账号矩阵**来放大流量。即批量运营多个小红书账号，同时发布笔记为同一个店铺引流，从而扩大曝光和订单规模。
