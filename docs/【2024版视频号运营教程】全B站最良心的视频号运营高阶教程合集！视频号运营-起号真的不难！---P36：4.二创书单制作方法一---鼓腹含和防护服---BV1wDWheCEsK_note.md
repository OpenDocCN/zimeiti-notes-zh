# 视频号运营教程：P36：4.二创书单制作方法一 📚

在本节课中，我们将学习一种高效的书单视频制作方法——二次创作。我们将通过拆解一个具体的案例，详细讲解如何利用现有素材进行重组和替换，从而制作出原创度更高的书单带货视频。

---

## 剪辑是必备技能

上一节我们介绍了视频号运营的基础，本节中我们来看看视频制作的核心技能——剪辑。剪辑是制作任何类型视频的必备技能。许多新手刚开始接触时，会觉得混剪和剪辑操作很困难。这通常源于对剪辑软件的不熟悉。

解决方法是**熟能生巧**。建议先自行练习剪辑十几个视频，花一两个小时熟悉每个功能的作用。当你对软件有了基本了解后，再学习课程内容会更容易理解和掌握。

## 书单带货的优势

除了之前课程中提到的混剪方法，我们还可以进行二次创作来制作书单视频。书单类视频一直是内容领域的“常青树”，只要认真执行，通常都能获得不错的销售转化。

书单账号的底层逻辑是：通过一段影视剧片段、故事或道理阐述，最终引出一本书的推荐，从而激发观众的购买欲望。这种软性推荐方式，比直接展示产品功能的“硬广”更容易被观众接受。

3.  **分析并借鉴**：研究这些高销量商品下的热门视频，理解其内容结构和推荐逻辑。你不仅可以参考我们推荐的书籍，也可以直接借鉴这些对标账号橱窗里销量高的任何书籍。

## 二次创作实操步骤

![](img/9c1dbaf460184303a7c1a8f05045bf66_10.png)

![](img/9c1dbaf460184303a7c1a8f05045bf66_12.png)

![](img/9c1dbaf460184303a7c1a8f05045bf66_14.png)

本节我们将通过一个具体案例，一步步演示如何进行书单视频的二次创作。我们以书籍《女孩要懂得保护自己》的相关视频为例。

### 第一步：获取原始素材

1.  在抖音搜索目标书籍，例如“女孩要懂得保护自己”。
2.  进入商品页，选择“销量”排序，找到高销量商品下的“达人秀”视频。
3.  选择一个点赞量较高的视频作为参考模板。
4.  复制视频链接，使用去水印工具（如“配音神器”）保存视频到本地。
5.  同时，利用工具提取出视频的完整文案。

### 第二步：处理音频与文案

1.  **检查文案**：仔细检查提取出的文案，纠正可能的错别字。
2.  **生成新配音**：使用配音软件（如剪映内的配音功能或第三方工具），为文案生成新的音频。可以选择与视频风格匹配的音色。
    *   代码示例（概念性）：`audio = generate_voiceover(text=cleaned_script, voice_type="emotional_female")`
3.  **调整语速**：如果觉得某段配音语速过快，可以在配音软件中插入停顿（例如1.5秒），然后重新合成。
4.  将生成的新配音导出并保存。

![](img/9c1dbaf460184303a7c1a8f05045bf66_16.png)

### 第三步：在剪辑软件中重组视频

![](img/9c1dbaf460184303a7c1a8f05045bf66_18.png)

![](img/9c1dbaf460184303a7c1a8f05045bf66_20.png)

1.  **导入与准备**：将去水印后的原视频导入剪映。
2.  **去除原片元素**：
    *   关闭视频原声。
    *   使用**蒙版**（矩形）功能，将原视频的字幕遮盖掉。操作：选择片段 -> 蒙版 -> 矩形 -> 调整覆盖字幕 -> 勾选“反转” -> 确认。
    *   将背景设置为**画布模糊**，使遮盖区域更自然。
3.  **替换视频片段（核心操作）**：
    *   **方法A：直接替换**：选中想要替换的片段，点击“替换”，从素材库或本地选择新素材。新素材时长会自动匹配原片段时长。
    *   **方法B：删除后添加**（更推荐）：
        1.  分割出想要替换的片段，记录其时长（例如5.5秒）。
        2.  删除该片段。
        3.  在相同时间点，导入你找到的新素材。
        4.  将新素材裁剪至与记录时长大致相同（如5.5秒）。
        5.  关闭新素材的原声，并调整画面至合适大小。
    *   **寻找新素材的途径**：
        *   **剪映素材库**：搜索关键词，如“老师讲课”、“女生演讲”。
        *   **抖音直接搜索**：搜索更具体的关键词，如“女讲师培训”，找到合适片段后保存并使用去水印工具处理。
4.  **重复替换**：按照上述方法，将视频中其他可替换的片段（如学生操场场景、学习场景等）逐一进行替换。可以从同一个商品的其他“达人秀”视频中获取不同片段进行组合。

### 第四步：合成与导出

1.  **添加新音频**：将第二步生成的新配音音频导入剪辑软件，与视频对齐。
2.  **微调对齐**：如果音频与某个画面切换点不匹配，可以**分割音频**，进行前后微调，确保声画同步。
3.  **添加背景音乐**：选择一首风格相符的背景音乐，音量调至低于配音音量，作为衬托。
4.  **生成新字幕**：
    *   使用“文本” -> “识别字幕”功能，基于新配音自动生成字幕。
    *   为字幕选择合适的样式，并**应用到全部**。
5.  **最终检查**：确保所有原视频字幕已被遮盖，新字幕位置正确，音频清晰无冲突。
6.  **导出视频**：设置帧率为60帧，导出高清视频。

![](img/9c1dbaf460184303a7c1a8f05045bf66_22.png)

**公式总结二创核心**：`新视频 = (原视频结构 - 原片段A/B/C + 新片段A‘/B’/C‘) + 新配音 + 新字幕 + 背景音乐`

本节课中我们一起学习了书单视频的二次创作全流程。关键点在于：**解构优质模板视频，保留其文案逻辑和结构，但替换其中的画面素材和音频，从而组合成一个原创度更高的新视频**。这种方法比单纯混剪一条视频更能提升原创性，是制作书单带货内容的有效技巧。请记住，多练习、多分析对标账号，是掌握这一方法的最佳途径。
