# 微信视频号教程：P26：如何制作直播间贴图 🎨

在本节课中，我们将学习如何为微信视频号直播间制作一个透明的商品推荐贴图。这种贴图可以清晰地展示产品信息和价格，提升直播间的专业度和信息传达效率。

---

上一节我们介绍了直播的基础设置，本节中我们来看看如何制作一个实用的直播间贴图。

制作这种贴图，我们可以使用一款名为“微麦水印相机”（原“微商水印相机”）的手机应用。该应用提供了简单易用的模板功能。

以下是具体的操作步骤：

1.  **下载应用**：在手机应用商店搜索并下载“微麦水印相机”。
2.  **选择模板**：打开应用，无需开通会员。找到“水印库”或“更多水印”入口，进入“水印模板”。在模板分类中找到“横竖文字”类目，并选择一个喜欢的模板，例如第一个模板。
3.  **编辑内容**：点击模板进入编辑界面。将示例文字替换为你自己的商品信息。例如：
    *   第一行：`好物推荐`
    *   第二行：`老国货品牌`
    *   后续行：`9.9元/1盒`、`19.9元/3盒`、`29.9元/5盒`
4.  **调整样式**：你可以自由调整文字颜色、字体。但最关键的一步是设置背景。
5.  **设置透明背景**：找到背景色设置选项。**必须选择“无背景”或带有斜杠（/）标识的透明选项**。确保背景透明度为100%，否则贴图在直播间会带有不透明的底色，影响效果。
6.  **生成图片**：编辑完成后，点击“生成”或“保存”按钮，将制作好的透明贴图保存到手机相册。

---

现在，我们已经制作好了贴图，接下来看看如何在直播中使用它。

![](img/5c77be8afd184ded7815be32cd660884_1.png)

以下是添加到直播间的步骤：

![](img/5c77be8afd184ded7815be32cd660884_3.png)

1.  **发起直播**：在微信视频号中发起直播。
2.  **添加贴图**：在直播界面找到添加贴图或画面的功能按钮（通常是一个“+”或图片图标）。
3.  **选择图片**：从弹出的选项中选择“手机相册”或“从相册选择”。
4.  **导入并调整**：从相册中找到并选择刚才保存的透明贴图。贴图会出现在直播画面中，你可以用手指拖动来调整其位置，双指缩放来调整大小，直到满意为止。

![](img/5c77be8afd184ded7815be32cd660884_5.png)

如果在制作时发现文字大小或间距不合适，你可以随时返回微麦水印相机进行修改。

以下是调整贴图细节的方法：

![](img/5c77be8afd184ded7815be32cd660884_11.png)

*   重新打开已保存的贴图项目或使用相同模板新建。
*   在编辑界面，找到调整`字体`、`文字大小`、`行距`、`字间距`的选项。
*   根据直播画面的实际预览效果进行微调，确保所有信息清晰易读。
*   调整完毕后，再次生成并保存新图片，然后在直播间替换即可。

![](img/5c77be8afd184ded7815be32cd660884_12.png)
