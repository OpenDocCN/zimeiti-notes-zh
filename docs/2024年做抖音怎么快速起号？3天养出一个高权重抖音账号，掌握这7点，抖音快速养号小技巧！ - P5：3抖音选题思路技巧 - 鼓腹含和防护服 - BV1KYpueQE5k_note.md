# 抖音运营指南：P5：抖音选题思路技巧 🎯

在本节课中，我们将学习一套高效的抖音选题公式。这套方法能帮助你快速找到创作方向，解决“不知道发什么”的难题，从而更精准地吸引目标用户，为账号打上标签。

---

## 选题的重要性

![](img/6e8f6e6e9343da8ced46a6746ddb203e_1.png)

会找选题的创作者，能将更多时间投入视频制作。通过每日发布多条内容，例如一天三条，一个月就能积累90条作品。依靠数量优势，就能在竞争中占据主动，甚至无需依赖付费推广。

更重要的是，擅长选题的人能更精准地定位用户痛点。这有助于吸引精准粉丝观看，从而快速为账号打上垂直标签。

---

## 核心选题公式

那么，如何解决选题困难呢？这里有一套百万粉丝博主常用的技巧。为了方便理解，我们将其总结为一个核心公式：

![](img/6e8f6e6e9343da8ced46a6746ddb203e_3.png)

**选题 = 内容定位 + 用户痛点 + 借势热点**

用好这个公式，你的内容方向就不会跑偏。

---

![](img/6e8f6e6e9343da8ced46a6746ddb203e_5.png)

## 公式应用详解

以下是应用该公式的具体步骤：

**第一步：确定内容定位**
你需要挖掘所在行业的关键词。这些关键词应是客观、中性的，不包含主观描述。
*   例如，对于知识博主，关键词可以是：`播放量`、`粉丝量`、`定位`、`选题`。

**第二步：寻找用户痛点**
在定位的基础上，思考目标用户最关心、最困扰的问题是什么。
*   例如，用户的痛点可能是：`便捷`、`高效`、`技巧教学`、`内容输出`。

**第三步：借势热点**
结合近期发生的热点事件、节日或流行趋势。
*   例如，临近春节，热点就是“拜年”、“春节内容”。

**第四步：排列组合**
将以上三个步骤得出的元素进行任意组合，即可生成具体选题。

## 行业应用案例

上一节我们介绍了公式的组成部分，本节中我们来看看如何在不同行业里应用它。

**案例一：知识博主**
*   **内容定位**：`播放量`、`粉丝量`、`选题`
*   **用户痛点**：`高效`、`技巧`
*   **借势热点**：`春节`
*   **生成选题**：
    *   如何用拜年话术提升视频播放量？
    *   春节必火的内容选题技巧。
    *   “张同学”快速涨粉的三个秘诀解析。

![](img/6e8f6e6e9343da8ced46a6746ddb203e_7.png)

**案例二：美妆行业**
*   **内容定位**：`美白`、`除皱`、`口红`
*   **用户痛点**：`平价`、`明星同款`、`避坑`
*   **借势热点**：`迪丽热巴同款`、`七夕节`
*   **生成选题**：
    *   迪丽热巴同款美白面膜平价推荐。
    *   七夕节千万别送女朋友这几支口红（避坑指南）。
