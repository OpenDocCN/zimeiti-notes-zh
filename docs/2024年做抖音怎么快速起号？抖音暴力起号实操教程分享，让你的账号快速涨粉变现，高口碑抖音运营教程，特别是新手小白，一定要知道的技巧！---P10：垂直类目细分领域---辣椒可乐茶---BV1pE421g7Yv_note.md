# 抖音运营教程：P10：垂直类目的细分领域 🎯

在本节课中，我们将深入探讨抖音账号运营中“垂直度”的具体划分。理解垂直类目的层级，能帮助你更精准地定位内容，吸引目标粉丝。

上一节我们介绍了内容标签的重要性，本节中我们来看看如何将“垂直”这个概念进行更细致的划分。

我们常说做账号要垂直。很多同学会想到，做美食就一直发美食，做游戏就一直发游戏。这没错，但这仅属于最宽泛的**一级垂直领域**。

在一级垂直领域之下，还存在更精细的**二级**和**三级垂直领域**。我们可以通过一个公式来理解这种层级关系：
**垂直领域层级 = 一级类目 > 二级类目 > 三级类目 > ...**

以下我们以“游戏”这个一级类目为例，具体说明各级细分：

*   **一级垂直领域**：游戏
*   **二级垂直领域**：在“游戏”之下，可以细分为手游、电竞、网游等。
*   **三级垂直领域**：在二级类目下继续细分。例如：
    *   手游 -> 王者荣耀
    *   电竞 -> DOTA 2
    *   网游 -> 英雄联盟

三级垂直领域还可以继续向下细分，达到更极致的垂直。例如：
*   王者荣耀 -> 专玩“安琪拉”这个英雄的教学和视频。
*   绝地求生（吃鸡）-> 只玩“雨林”地图的攻略。
*   各类游戏 -> 只制作“游戏剧情解说”类视频。

这种细分意味着，你的全部视频内容都围绕一个极其具体的点展开。大领域和小领域都可以作为选择，这取决于你自己的兴趣和规划。

因为之前有同学对“垂直领域”的概念不太清楚，所以今天在此结合实例进行了细化讲解。本节课后续的一些内容也会涉及到这部分知识。
