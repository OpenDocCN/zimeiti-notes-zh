## 付费起号的核心逻辑

上一段我们提到了新号启动面临的挑战。本节中我们来看看付费起号模式的基本表现。

一个新号开播即开始大力投放付费流量。在起号的前五天左右，其付费流量占比高达90%甚至100%。但在大约六七天后，直播间的自然流量会开始显著增长，付费流量占比则逐渐降低。不过，无论自然流量多高，直播间总会维持约6%到10%的付费流量占比。若不熟悉此模式，可能无法察觉这部分付费流量的存在。

这套付费起号方法主要围绕三个核心点展开。理解这三个关键链条，操作千川加速付费流就会变得简单。

## 核心一：直投直播间画面

第一个核心点是：只有采用**直投直播间画面**的千川计划，才能启动自然流量。

平台将流量渠道划分得非常清晰。只有当直接投放直播间画面时，产生的互动、成交等数据才会累积在**直播间画面本身**这个载体上。此流程不涉及短视频。我们的目的是为直播间画面打上标签。

如果投放的是短视频素材，那么所有数据（包括成交）都会归属于该素材本身，而非直播间。因此，直投对应的是直播间的流量通道，这是加速自然流量的前提。

**核心操作**：
```plaintext
广告投放目标：直播间画面直投
数据归属：直播间载体
目的：积累直播间标签
```

## 核心二：利用付费打上精准标签

上一节我们明确了数据积累的载体。本节中我们来看看如何为这个载体打上正确的“标签”。

标签是告诉平台你需要什么样流量的信号。标签精准，平台才会根据你直播间的历史成交人群模型进行推流。如果标签不精准，系统推送的流量也不精准，导致数据反馈差，自然流量便难以实现递增。

因此，打上精准标签后，自然流量也会变得精准。打标签应使用特定的千川计划。请看一组数据对比：

在不设置任何定向的情况下，千川的覆盖人群高达21亿。但中国总人口仅约14亿，多出的部分包含了大量站外及其他流量。而将“八大人群”等核心标签包叠加，覆盖人数约为14亿。

这里需要思考：在**21亿**的宽泛范围里定向，还是在**14亿**的精准人群包（如DMP人群包）里定向，哪个打上的标签更精准？

**但必须提醒**：达摩盘（DMP）的流量价格昂贵。同样的“人货场”配置，投莱卡定向出价30元可能跑得动，但投达摩盘人群包，出价可能需要提高到90元才能启动。因此其投产比较低，因为它本质是**为精准打标签而设计的工具**，并非追求直接高投产。

**核心概念**：
- **标签** = 直播间画像
- **打标签工具**：优先使用**达摩盘(DMP)**等精准人群包计划（尽管成本高）。
- **目的**：告知系统你的精准目标人群。

## 核心三：成为“优质卖家”撬动自然流

打完标签后，并不意味着平台会自动推送大量自然流量。平台只会向同层级中**数据表现更优的直播间**倾斜流量。

如何定义“优质卖家”？举例说明：
- A卖家：场观1000，销售额1000。
- B卖家：场观1000，销售额2000。

平台若推送自然流量，会给谁？电商部门作为字节跳动的一部分，有KPI考核。作为负责人，必然将流量分配给**流量利用率更高**的直播间，即B卖家。

由此引申出关键选择：如果系统推荐计划的投产是3，而精准人群包计划的投产是2，你会如何选？90%的人会选择系统推荐，因为流量便宜，投产更高。

但若从**平台视角的“流量有效利用率”** 评估：
- **系统推荐计划**：消耗300元，场观1000，销售额1000，投产≈3.3。
- **人群包计划**：消耗1000元，场观1000，销售额2000，投产=2。

对于平台而言，虽然人群包计划投产低，但它创造了更高的**销售额**和**UV价值**。平台考核的是整体GMV和流量利用效率，而非广告主的投产比。

**因此记住核心**：付费起号阶段，**不看付费计划自身的投产高低**。付费计划的投产与你是否盈利，和平台没有直接关系。平台只关注你的直播间能否产生更优的**数据**（如销售额、UV价值）。即使付费投产低，但整体销售额高、流量利用率高，平台就更愿意给你推送自然流量。

**核心公式**：
```
平台推流优先级 ∝ (销售额， UV价值)  // 与广告计划自身投产比关系不大
```

## 总结

本节课我们一起学习了直播带货新号付费起号的三个核心要点：

理解并执行这三点，是2024年快速启动直播带货新号的有效路径。
