# 小红书炮机V4.0教程：P1：四种核心功能详解与操作指南 🚀

在本节课中，我们将详细学习小红书炮机V4.0版本的五种主流打法。我们将重点介绍其中四种核心功能的具体操作步骤，包括“先搜索后推荐关注”、“首页导航刷新关注”、“协议UID跳转”以及“普通私信功能”。通过学习，你将能够利用这些工具实现跨行业的精准曝光与引流。

---

## 小红书炮机V4.0：1：先搜索后推荐关注 🔍

上一节我们概述了课程内容，本节中我们来看看第一种核心功能“先搜索后推荐关注”的具体操作方法。此功能通过搜索指定关键词，然后推荐并关注相关用户。

**操作步骤如下：**

1.  **准备搜索关键词**：
    *   打开MT管理器，找到并打开名为 `搜索词点TXT` 的文件。
    *   在该文件中填写你的行业关键词。**注意：只能填写一个关键词**。
    *   填写完成后，保存并关闭文件。
    *   **示例代码（文件内容）**：
        ```
        小说代打
        ```

2.  **配置软件参数**：
    *   打开炮机软件，在“功能选择”中，选中 **`先搜索后推荐关注`**。
    *   “巢穴”（行业分类）选项保持默认，无需改动。
    *   在“版本号”处，选择与你当前小红书APP版本一致的号。例如，小红书是8.28版本，这里就选8.28。
    *   “评论时间”的数值控制操作速度：**数值越小，速度越快；数值越大，速度越慢**。通常设置在88到299之间（约1-3分钟一个操作）。
    *   其他选项如评论数量、笔记收藏等均保持默认，无需修改。

3.  **启动运行**：
    *   点击“保存参数”。
    *   打开悬浮窗，启动脚本。脚本将自动搜索你设置的关键词（如“小说代打”），并执行推荐、关注等操作。

**核心概念**：此功能的核心是 **`关键词驱动`** 的自动化搜索与交互。

## 小红书炮机V4.0：2：首页导航刷新关注 📱

了解了基于搜索的引流后，我们来看第二种更便捷的方法：“首页导航刷新关注”。此功能直接利用小红书首页的导航栏进行引流，无需修改外部文本文件。

**操作步骤如下：**

1.  **配置软件**：
    *   在炮机软件的“功能选择”中，修改为 **`首页导航刷新关注`**。
    *   在“巢穴”选项中，选择你需要的行业分类（如：潮鞋、穿搭、母婴等）。
    *   选择与小红书APP对应的“版本号”。

2.  **设置小红书首页导航**：
    *   打开小红书APP，进入首页。
    *   找到首页顶部的导航栏列表，点击“编辑”。
    *   在列表中找到你刚才在软件中选择的行业分类（例如“潮鞋”）。
    *   **长按**该分类，将其拖动到“直播”选项的后面，然后点击完成。

![](img/b3b643112b96a6cbc8e11c7d18ac300c_1.png)

3.  **调整速度并运行**：
    *   回到炮机软件，设置“评论时间”（规则同上：数值小则快）。
    *   确保其他所有选项均为默认值。
    *   保存参数后，直接启动脚本。脚本将自动刷新你设置的首页导航栏目（如“潮鞋”），并执行关注操作。

**核心概念**：此功能实现了 **`界面元素定位`** 与自动化操作的结合。

## 小红书炮机V4.0：3：协议UID跳转功能 ⚙️

前面两种方法侧重于公开内容引流，本节我们学习更精准的“协议UID跳转功能”。此功能需要先采集目标用户的UID，然后进行定向跳转与互动。

**操作步骤如下：**

1.  **配置主软件**：
    *   在炮机软件中，将“功能选择”修改为 **`协议UID跳转`**。
    *   “首页导航”保持默认。
    *   选择对应的小红书“版本号”（要求8.18以上）。
    *   设置“评论时间”，其他选项默认。

2.  **采集目标用户UID**：
    *   打开“小红书采集协议”工具，登录一个小红书账号。
    *   搜索你的行业关键词（如“小说代打”），进入相关笔记。
    *   在评论区设置采集关键词（如“求带”），然后点击“评论采集”。
    *   工具将筛选并列出符合条件的用户UID，将其复制。

3.  **填写UID到指定文件**（两种方法）：
    *   **方法一（电脑操作）**：在电脑上创建 `用户ID.txt` 和 `用户UID.txt` 两个文本文档，将复制的UID分别粘贴进去并保存。然后将这两个文件传输到手机，替换MT管理器中原有的对应文件。
    *   **方法二（手机操作）**：直接在手机的MT管理器中找到 `用户ID.txt` 和 `用户UID.txt` 文件，打开并将复制的UID粘贴进去保存。

4.  **执行定向引流**：
    *   完成UID填写后，回到炮机软件，保存参数并启动脚本。
    *   脚本将自动跳转到你指定的用户主页，执行关注、点赞等操作。

**核心概念**：此功能的核心是 **`UID定向`** 与 **`协议跳转`**，实现精准用户触达。

## 小红书炮机V4.0：4：普通私信功能 💬

最后，我们来学习最直接的触达方式——“普通私信功能”。此功能可以向指定用户自动发送私信。

**操作步骤如下：**

1.  **配置软件**：
    *   在炮机软件中，将“功能选择”修改为 **`普通私信功能`**。
    *   **注意**：此功能要求小红书版本必须在 **8.28以上**。
    *   版本号对应选择8.28或更高。

2.  **准备私信话术与目标UID**：
    *   **准备话术**：在MT管理器中，找到 `私信话术.txt` 和 `话术.txt` 文件，在其中填写你想要发送的私信内容（两个文件内容一致），例如“渡鸦科技社”。
    *   **准备UID**：使用“采集协议”工具（步骤同上节）采集精准用户的UID，并填写到 `用户ID.txt` 和 `用户UID.txt` 文件中。**确保这些UID不是你已经关注过的用户**。

3.  **执行私信发送**：
    *   所有文件配置完成后，在炮机软件中保存参数。
    *   启动脚本，软件将自动跳转到目标用户主页并发送私信。
    *   **重要提醒**：每个账号每日主动私信建议不超过10条，以防违规。

**核心概念**：此功能实现了 **`自动化私信`** 发送，是直接沟通的利器。

## 功能使用注意事项与总结 📝

在本节课中，我们一起学习了小红书炮机V4.0的四种核心引流功能：

1.  **先搜索后推荐关注**：通过关键词搜索进行自动化引流。
2.  **首页导航刷新关注**：利用APP首页导航进行便捷引流。
3.  **协议UID跳转功能**：通过采集UID实现精准用户定向互动。
4.  **普通私信功能**：向指定用户自动发送私信，直接触达。

**重要注意事项**：
*   目前仅以上四种功能稳定可用，其他功能可能存在Bug，请勿使用。
*   使用任何功能时，务必确保小红书APP版本与软件中设置的版本号一致。
*   操作速度（评论时间）需合理设置，过快可能增加风险。
*   私信功能需谨慎使用，控制频率，遵守平台规则。
