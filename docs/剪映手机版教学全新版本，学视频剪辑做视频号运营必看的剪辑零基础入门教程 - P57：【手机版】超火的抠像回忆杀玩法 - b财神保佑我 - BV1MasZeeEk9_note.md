# 剪映手机版教学：P57：超火的抠像回忆杀玩法 🎬

![](img/fe4b23ccd44da798c1763040a5560173_1.png)

在本节课中，我们将学习如何制作一个流行的“抠像回忆杀”视频。我们将重点复习卡点技巧，并学习滤镜与滤镜图层的区别、智能抠像功能以及特效图层的应用。通过本教程，你将能够独立完成一个具有电影感回忆效果的短视频。

---

## 概述

本节课我们将制作一个“抠像回忆杀”视频。核心步骤包括：导入并卡点剪辑背景视频、添加音乐、使用画中画功能导入人物视频、应用智能抠像、以及通过滤镜和特效图层来营造氛围。我们将重点理解**滤镜**与**滤镜图层**的区别，这是本课的关键知识点。

---

## 开始制作

首先，打开剪映App，点击主界面上的 **`开始创作`** 按钮。

![](img/fe4b23ccd44da798c1763040a5560173_1.png)

### 1. 导入素材与音乐

选中准备好的背景视频素材，点击 **`添加`** 按钮，将其导入到时间线。

导入后，先关闭视频的原声。点击时间线左侧的 **`关闭原音`** 按钮。

接下来为视频添加音乐。点击底部工具栏的 **`添加音频`** -> **`提取音乐`**，然后选择包含目标音乐的视频文件，点击 **`仅导入视频的声音`**。这样，音乐就被单独导入到时间线中。

### 2. 复习卡点技巧

我们的背景画面已经根据音乐节拍预先剪辑好了。如果你需要自己剪辑，核心原则是：**在音乐重音（如“噔”的一声）处切换画面**。

你可以返回主菜单查看音乐的波形图，通过放大时间线来精确找到节拍点，并在对应位置分割视频，实现画面与节奏同步。

上一节我们导入了素材，接下来我们为视频添加核心的人物元素。

### 3. 添加画中画人物视频

点击时间线后方的 **`+`** 添加按钮，导入人物回头的视频。

选中这个新添加的人物视频，在底部工具栏点击 **`切画中画`**。这样，人物视频就会移动到下方轨道，与上方的背景视频形成画中画效果。

我们需要根据音乐起始的节奏点，将人物视频分割。找到音乐开始起伏的位置，选中画中画轨道，点击 **`分割`** 按钮，将视频一分为二。

### 4. 应用智能抠像

选中分割后的**后半段**人物视频，点击底部工具栏的 **`抠像`** -> **`智能抠像`**。软件会自动处理，抠除人物背景。处理完成后，记得勾选确认。

**效果对比**：
*   **前半段**：不应用抠像，人物与背景共存。
*   **后半段**：应用智能抠像，只保留人物，背景变为透明，从而露出主轨道的视频。

现在，我们有了一个从完整场景过渡到人物特写的画面。接下来，我们需要通过调色来强化回忆感。

### 5. 理解并应用滤镜

这里我们将学习一个核心概念：**单独滤镜** 与 **滤镜图层** 的区别。

**第一种方法：为单个视频添加滤镜**
1.  选中**背景视频**片段。
2.  点击底部工具栏的 **`滤镜`**。
3.  在滤镜列表中选择 **`黑白`** 分类下的 **`赫本`** 滤镜。
4.  将强度调整为 **`100%`** 并勾选确认。
**效果**：此滤镜**仅作用于**当前选中的背景视频片段。

**第二种方法：通过滤镜图层添加滤镜**
1.  点击左上角返回箭头，退回到**主菜单界面**（能看到所有轨道）。
2.  点击底部工具栏的 **`滤镜`**。
3.  同样选择 **`黑白-赫本`** 滤镜，强度 **`100%`**，勾选确认。
**效果**：时间线上会生成一个独立的**蓝色滤镜图层**。这个图层会覆盖其长度范围内的所有视频轨道（包括主轨道和画中画轨道）。

**核心区别总结**：
*   **选中视频后加滤镜**：效果仅应用于该单一视频片段。
*   **主菜单加滤镜**：会生成一个**滤镜图层**，效果可覆盖多个轨道和片段，影响范围更广。

在本例中，我们只需要为背景调色。因此，删除滤镜图层，采用第一种方法，仅为背景视频片段添加“赫本”滤镜。

### 6. 添加画面特效

为了增加复古胶片感，我们可以添加特效。特效是以**图层**形式工作的，理解这一点很重要。

返回主菜单界面，点击底部工具栏的 **`特效`** -> **`画面特效`**。

以下是需要添加的特效列表：
*   **复古** 分类下的 **`黑色噪点`**。
*   **投影** 分类下的 **`车窗影`**。

添加后，时间线上会出现对应的特效图层（如“黑色噪点”）。**必须将特效图层的长度拖动至与整个视频时长一致**，否则未被图层覆盖的部分将没有特效效果。

### 7. 微调与导出

如果画中画人物视频长度不足，可以对其进行变速以延长。
选中人物视频，点击 **`变速`** -> **`常规变速`**，将速度值调整为小于1（例如 **`0.8`**），即可实现慢放效果，然后裁剪至与主视频对齐。

最后，点击右上角的 **`导出`** 按钮，即可完成视频制作。

## 总结

本节课我们一起学习了“抠像回忆杀”视频的制作全流程。我们重点掌握了三个知识点：
1.  **滤镜的两种应用方式**：直接为片段添加，或通过**滤镜图层**全局添加，并理解了两者的区别。
2.  **智能抠像功能**：用于分离人物与背景。
3.  **特效图层的使用**：特效需要通过图层来覆盖和管理作用时长。

![](img/fe4b23ccd44da798c1763040a5560173_3.png)

同时，我们也复习了卡点剪辑和画中画功能。熟练掌握这些技巧，你就能轻松制作出情感丰富的回忆风格短片了。
