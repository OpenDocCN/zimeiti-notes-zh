# 微信视频号教程：P44：直播间透明文字制作

![](img/ec2299b05072588933605f1b8d6bca6f_0.png)

![](img/ec2299b05072588933605f1b8d6bca6f_2.png)

在本节课中，我们将学习如何为微信视频号直播间制作带有透明背景的文字贴图。这种贴图可以叠加在直播画面上，用于展示商品信息、促销活动或关键提示，从而提升直播间的专业度和信息传达效率。

![](img/ec2299b05072588933605f1b8d6bca6f_4.png)

上一节我们介绍了直播间的基础设置，本节中我们来看看如何制作美观实用的透明文字贴图。

![](img/ec2299b05072588933605f1b8d6bca6f_6.png)

## 准备与提取文案

![](img/ec2299b05072588933605f1b8d6bca6f_8.png)

首先，需要准备好要在直播间展示的文字内容。为了节省时间，可以直接从已有的文档或聊天记录中提取所需文案。

![](img/ec2299b05072588933605f1b8d6bca6f_10.png)

以下是操作步骤：
1.  从源文档中复制所需的文字内容。
2.  将复制的内容发送到另一个微信账号上，以便在手机端访问。
3.  在手机微信中，对文案进行初步的排版整理，例如分行，使其更清晰。

![](img/ec2299b05072588933605f1b8d6bca6f_12.png)

## 使用模板创建文字样式

![](img/ec2299b05072588933605f1b8d6bca6f_14.png)

文案准备就绪后，接下来使用设计工具为其添加样式。我们将使用“搞定设计”等工具中丰富的模板。

![](img/ec2299b05072588933605f1b8d6bca6f_16.png)

![](img/ec2299b05072588933605f1b8d6bca6f_18.png)

以下是操作步骤：
1.  打开设计工具，选择“新建画布”或“方形海报”等合适尺寸的模板。
2.  在模板库中，可以看到各式各样的文字样式模板，如标题、气泡、价格标签等。
3.  选择一个喜欢的模板，例如一个标题样式。
4.  双击模板中的文字框，清除原有的示例文字。
5.  粘贴之前准备好的文案内容。

![](img/ec2299b05072588933605f1b8d6bca6f_20.png)

## 调整文字细节

![](img/ec2299b05072588933605f1b8d6bca6f_22.png)

![](img/ec2299b05072588933605f1b8d6bca6f_24.png)

初步粘贴后，文字效果可能不理想，需要进行细节调整，使其更美观。

![](img/ec2299b05072588933605f1b8d6bca6f_26.png)

以下是操作步骤：
1.  如果感觉文字间距过密，可以双击文字框进入编辑模式。
2.  找到排版设置，通常在底部有带 **A** 图标的选项。
3.  调整“字间距”参数，使文字排列更舒展。
4.  调整完成后收起设置面板，并将文字框放置在画布合适的位置。

![](img/ec2299b05072588933605f1b8d6bca6f_28.png)

## 添加辅助信息元素

![](img/ec2299b05072588933605f1b8d6bca6f_30.png)

主标题设置好后，可以继续添加其他信息，如副标题、价格、促销标签等，形成完整的信息组合。

![](img/ec2299b05072588933605f1b8d6bca6f_32.png)

![](img/ec2299b05072588933605f1b8d6bca6f_34.png)

以下是操作步骤：
1.  准备好下一段文案，例如“给我爸爸”。
2.  再次选择“文字”工具，添加一个新的文字图层。
3.  双击新文字框，粘贴文案。
4.  可以在此处更改字体，选择与主标题搭配的样式。
5.  在“样式”选项中，修改文字颜色，例如选择一个醒目的颜色。
6.  调整文字大小至合适尺寸。
7.  对于促销信息（如“拍60包更划算”），可以寻找更活泼的元素模板。
8.  在“气泡”、“价格”、“爆炸贴”等分类中寻找合适的样式模板。
9.  选择“爆炸贴”模板，双击编辑，粘贴促销文案并调整大小。

![](img/ec2299b05072588933605f1b8d6bca6f_36.png)

## 抠图生成透明背景

![](img/ec2299b05072588933605f1b8d6bca6f_38.png)

所有文字元素都设计并排版完成后，最后一步是去除图片的白色背景，生成透明的PNG图片，以便在直播软件中叠加。

![](img/ec2299b05072588933605f1b8d6bca6f_40.png)

![](img/ec2299b05072588933605f1b8d6bca6f_42.png)

以下是操作步骤：
1.  将设计好的图片保存到手机相册。
2.  在设计工具中找到“智能抠图”功能。
3.  点击“选择图片”，导入刚才保存的图片。
4.  系统会自动识别并去除背景，点击“对号”确认抠图效果。
5.  最后，将抠好的透明背景图片保存下来。

![](img/ec2299b05072588933605f1b8d6bca6f_44.png)

本节课中我们一起学习了制作直播间透明文字贴图的完整流程。从文案准备、模板套用、细节调整到最终抠图，每一步都是提升直播间视觉效果的关键。掌握这个方法，你可以快速为每次直播制作个性化的提示信息，有效吸引观众注意力并促进转化。
