# 小红书运营教程：P22：如何在抖音选品 🛍️

在本节课中，我们将学习一种高效的选品方法：利用抖音平台为小红书店铺寻找高客单价、优质的货源。这种方法特别适合希望节省时间、不想在多个电商平台间反复对比的运营者。

上一节我们介绍了赛道选择的重要性，本节中我们来看看如何在一个平台上完成选品和素材收集。

## 账号选品法

首先，我们介绍第一种方法：通过抖音上的优质账号进行选品。此方法的核心是找到目标赛道中表现良好的抖音账号，并分析其产品。

1.  打开抖音应用。
2.  点击右上角的搜索图标。
3.  在搜索框输入目标产品关键词（例如“耳环”），并选择“视频”分类进行搜索。
4.  浏览搜索结果，寻找那些视频风格统一、场景固定、明显为实拍的账号。这类账号通常是专业卖家。
5.  避免选择使用真人头像或频繁真人出镜的账号，以降低潜在的侵权风险。

以下是分析目标账号的步骤：

*   **检查商品橱窗**：进入目标账号主页，查看其商品橱窗，了解产品定价和销量。例如，价格在29.8元、36.8元且销量可观的产品，通常具备市场潜力。
*   **验证小红书存在**：复制该抖音账号名称，在小红书内搜索。如果该账号未在小红书运营，则其素材可被安全借鉴。
*   **规避直接抄袭**：借鉴时，务必更改账号名称和头像，仅参考其产品链路与内容风格。

## 获取产品与定价

确定对标账号后，下一步是获取具体产品并制定价格。

1.  在目标账号橱窗中，选择近期销量较高的商品，对其主图进行截图。
2.  打开拼多多（或其他货源平台），使用“以图搜图”功能查找同款产品，对比进货价格。
3.  参考对标账号在抖音的售价（例如29.8元），为小红书店铺定价。你可以选择相同价格，或稍低的价格（如24.9元）以增加竞争力。
4.  计算利润空间。如果单件利润过低（仅1-2元），则放弃该产品；若利润可观（如20-30元），则适合上架。

**利润计算公式**：`利润 = 小红书售价 - 货源平台成本价`

这种方法能有效解决素材问题。优质抖音账号会持续更新内容，你只需对其视频进行简单二次创作（如去重）即可发布，无需自己从头创作。

## 精选联盟选品法

第二种方法是通过抖音的“精选联盟”功能选品。此方法仅适用于已开通抖音商品橱窗的用户。

1.  进入抖音商品橱窗，点击“选品广场”。
2.  选品主要有两个方向：
    *   **查看排行榜**：关注“爆款榜”，榜单按品类（如服饰内衣、钟表配饰）实时更新热销商品。在抖音畅销的商品，通常在小红书也有市场。
    *   **关键词搜索**：在精选联盟内直接搜索产品关键词（如“饰品”），系统会展示相关热销商品及其排名。
3.  点开感兴趣的商品，你会发现“相关视频”和“话题”板块。这里聚集了大量带货达人创作的视频素材，可供你学习、二次创作或直接参考。

精选联盟选品的优点是：所选商品经过市场验证，自带热度，且拥有丰富多样的现成推广素材。

本节课中我们一起学习了两种在抖音选品的方法：账号对标法和精选联盟选品法。这两种方法能帮助你快速找到有潜力的高客单价产品，并解决内容素材来源问题，让你能更专注于小红书店铺的运营与增长。
