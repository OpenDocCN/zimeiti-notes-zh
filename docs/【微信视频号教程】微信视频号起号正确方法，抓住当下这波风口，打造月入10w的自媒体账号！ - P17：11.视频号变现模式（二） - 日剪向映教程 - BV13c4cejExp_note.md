## 视频与直播带货

![](img/c412ae2650e56396220cff053778e86a_7.png)

如果没有自己的产品，可以通过视频带货或直播带货赚取佣金。

![](img/c412ae2650e56396220cff053778e86a_9.png)

### 开通微信小商店

微信小商店是带货的核心工具，开通完全免费。

以下是开通微信小商店的步骤：
1.  在微信搜索“微信小商店”公众号。
2.  关注后，点击菜单栏的“免费开店”。
3.  按照提示完成“快速开店”流程。

**优势对比**：
*   **微信小商店**：0元入驻、0粉丝要求、支持多场景（视频号、公众号、社群、小程序）卖货。
*   **抖音橱窗**：需500元押金、发布10条以上视频、拥有1000粉丝。

### 有货源：销售自有商品

![](img/c412ae2650e56396220cff053778e86a_17.png)

如果你有自己的产品，可以在小商店上架销售。

操作路径：进入微信小商店后台 -> 点击“新增商品” -> 上传商品图片和编辑信息。

可以使用美图秀秀、创客贴、搞定设计等工具优化商品图片。

![](img/c412ae2650e56396220cff053778e86a_19.png)

### 无货源：赚取推广佣金

如果没有货源，可以成为“推广者”，通过帮别人卖货赚取佣金。

操作路径：进入视频号“创作者中心” -> 点击“带货中心” -> 进入“选品中心”。

在选品中心，你可以看到各类商品及其佣金比例。例如：
*   某图书售价59.8元，佣金27.44元。
*   佣金率可高达50%，相比其他平台（如抖音约25%-30%）更有优势。

**佣金计算公式**：
```
单笔订单收入 = 商品售价 × 佣金比例
```

### 扩展链接带货

![](img/c412ae2650e56396220cff053778e86a_21.png)

![](img/c412ae2650e56396220cff053778e86a_22.png)

![](img/c412ae2650e56396220cff053778e86a_23.png)

除了小商店，还可以通过“扩展链接”小程序，带货来自其他电商平台（如1688、淘宝、京东、拼多多）的商品。

![](img/c412ae2650e56396220cff053778e86a_25.png)

操作流程：
1.  在其他平台（如1688）找到想推广的商品，复制商品链接。
2.  打开“扩展链接”小程序，将链接粘贴进去。
3.  生成带货链接，可添加到视频下方。

这种方式相当于“中间商赚差价”，无需囤货。

![](img/c412ae2650e56396220cff053778e86a_27.png)

## 收益测算与案例

了解方法后，我们来测算一下可能的收益。

![](img/c412ae2650e56396220cff053778e86a_29.png)

![](img/c412ae2650e56396220cff053778e86a_30.png)

**关键指标**：转化率（观看者中购买商品的比例）。
*   较差水平：约1%
*   中等水平：约5%-6%
*   优秀账号：可达10%-20%

![](img/c412ae2650e56396220cff053778e86a_31.png)

**收益模拟计算**：
假设一个视频获得1万点赞（即约1万次店铺曝光），取最低转化率1%，商品平均佣金50元。
```
购买人数 = 10000 × 1% = 100人
预计收入 = 100人 × 50元 = 5000元
```
这仅是一个视频的潜在收入。如果坚持日更，月收入潜力可观。

**真实案例**：
*   **学员A**：新账号发布第4个作品即获10万点赞，单视频实现可观收益。
*   **学员B（胡明飞）**：从零开始的物业保安，学习两个月后，通过视频号带货，单日收入过万，月收入远超本职工作。

## 总结

![](img/c412ae2650e56396220cff053778e86a_47.png)

本节课我们一起学习了微信视频号的核心变现模式之二：
1.  可以通过**导流至公众号**进行知识付费或服务变现。
2.  **带货**是核心变现手段，无论有无货源，均可通过**微信小商店**或**扩展链接**销售商品、赚取佣金。
3.  微信小商店具有**零成本、零粉丝门槛、高佣金率**的优势。
4.  通过实际**收益测算**和**学员案例**，我们看到了在视频号认真运营所能带来的潜在收入。
