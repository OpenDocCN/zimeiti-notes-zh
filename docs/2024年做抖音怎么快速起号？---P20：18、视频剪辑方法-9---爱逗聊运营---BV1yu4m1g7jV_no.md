# 抖音运营实操教程：P20：18、视频剪辑方法-9 ✂️

在本节课中，我们将学习使用剪映App进行视频剪辑的完整流程，并介绍一个辅助口播拍摄的实用工具。课程内容将涵盖从导入素材、基础剪辑、添加音乐字幕，到最终导出的每一步操作，确保初学者也能轻松上手。

![](img/4cee47ff79333f7cc134085cefcd4f5d_3.png)

上一节我们介绍了视频拍摄的要点，本节中我们来看看如何将拍摄好的素材剪辑成完整的作品。

![](img/4cee47ff79333f7cc134085cefcd4f5d_5.png)

剪辑环节采用直接实操的方式进行。请准备好笔记，跟随步骤操作。

我们使用**剪映**（黑色图标）进行剪辑。这是抖音官方出品的软件，提供手机版和电脑版。我们先讲解手机版的操作。

首先，打开剪映App，点击“开始创作”。

![](img/4cee47ff79333f7cc134085cefcd4f5d_7.png)

![](img/4cee47ff79333f7cc134085cefcd4f5d_9.png)

![](img/4cee47ff79333f7cc134085cefcd4f5d_1.png)

![](img/4cee47ff79333f7cc134085cefcd4f5d_11.png)

接着，从相册中选择需要剪辑的视频素材，点击“添加”导入。如果您的作品由多个片段（如开头、中间、结尾）组成，建议先导入并剪辑好第一个片段，再点击轨道尾部的“+”号添加下一个片段。这样可以保持工程文件清晰，不易混乱。

![](img/4cee47ff79333f7cc134085cefcd4f5d_3.png)

我们将以一条22秒的视频为例进行剪辑。导入后，首先检查并调整视频的构图与比例。

![](img/4cee47ff79333f7cc134085cefcd4f5d_13.png)

以下是调整视频构图与比例的具体步骤：

![](img/4cee47ff79333f7cc134085cefcd4f5d_15.png)

1.  **选中视频片段**：在下方轨道中，点击需要调整的视频片段。
2.  **进入编辑菜单**：向右滑动底部工具栏，找到并点击“编辑”图标（一个方框形状）。
3.  **旋转画面**：在编辑菜单中，点击左侧的“旋转”功能，将画面调整到正确的方向。
4.  **裁剪比例**：如需进一步调整画面比例，可点击第三个“裁剪”功能。这里提供了多种比例选项，如 **1:1**、**4:3**、**16:9** 等，可根据平台要求或个人喜好选择。

![](img/4cee47ff79333f7cc134085cefcd4f5d_7.png)

![](img/4cee47ff79333f7cc134085cefcd4f5d_17.png)

调整完毕后，点击对勾确认，并返回主编辑界面。

![](img/4cee47ff79333f7cc134085cefcd4f5d_19.png)

![](img/4cee47ff79333f7cc134085cefcd4f5d_9.png)

![](img/4cee47ff79333f7cc134085cefcd4f5d_21.png)

接下来，我们需要修剪掉视频中无用的部分，例如开场的沉默或结尾的冗余动作。

![](img/4cee47ff79333f7cc134085cefcd4f5d_23.png)

点击播放按钮预览视频。时间轴上的**白色竖线**即代表当前播放位置，也充当“剪刀”的角色。

![](img/4cee47ff79333f7cc134085cefcd4f5d_25.png)

![](img/4cee47ff79333f7cc134085cefcd4f5d_27.png)

以下是剪辑视频片段的操作流程：

![](img/4cee47ff79333f7cc134085cefcd4f5d_29.png)

1.  **定位分割点**：将白色竖线移动到需要裁剪的位置（例如，说话开始前）。
2.  **分割视频**：点击选中视频片段，在底部工具栏中找到并点击“分割”按钮。视频将在竖线位置被切成两段。
3.  **删除片段**：点击选中需要删除的片段（如前一段静默部分），然后点击“删除”按钮。

![](img/4cee47ff79333f7cc134085cefcd4f5d_31.png)

重复此操作，即可流畅地剪掉开头、中间或结尾任何不需要的部分。核心操作可总结为：**选中 -> 分割 -> 选中 -> 删除**。

![](img/4cee47ff79333f7cc134085cefcd4f5d_33.png)

![](img/4cee47ff79333f7cc134085cefcd4f5d_13.png)

![](img/4cee47ff79333f7cc134085cefcd4f5d_35.png)

视频内容修剪完毕后，下一步是为视频添加背景音乐。

![](img/4cee47ff79333f7cc134085cefcd4f5d_37.png)

点击底部工具栏的“音频”按钮。如果您已使用“淘金阁”等工具生成了配音文件，可以点击“提取音乐”，直接从视频中提取音频。

![](img/4cee47ff79333f7cc134085cefcd4f5d_39.png)

![](img/4cee47ff79333f7cc134085cefcd4f5d_17.png)

如果没有，则点击“添加音乐”。剪映内置了丰富的音乐库，可按“美食”、“美妆”、“萌宠”等分类查找，也可以直接搜索歌曲名。点击歌曲旁边的播放图标可以预览，选定后点击“使用”即可添加到轨道中。

![](img/4cee47ff79333f7cc134085cefcd4f5d_41.png)

![](img/4cee47ff79333f7cc134085cefcd4f5d_19.png)

![](img/4cee47ff79333f7cc134085cefcd4f5d_43.png)

如果添加的音乐过长，其剪辑方法与视频相同：将竖线移至音乐结尾处，**选中音乐轨道 -> 点击“分割” -> 选中多余部分 -> 点击“删除”**。

![](img/4cee47ff79333f7cc134085cefcd4f5d_45.png)

![](img/4cee47ff79333f7cc134085cefcd4f5d_29.png)

![](img/4cee47ff79333f7cc134085cefcd4f5d_47.png)

您还可以调整音乐音量：选中音乐轨道，点击左侧出现的“音量”按钮，拖动滑块即可调节。

![](img/4cee47ff79333f7cc134085cefcd4f5d_31.png)

![](img/4cee47ff79333f7cc134085cefcd4f5d_49.png)

视频和音乐都处理好之后，我们需要为口播内容添加字幕。

![](img/4cee47ff79333f7cc134085cefcd4f5d_51.png)

点击底部工具栏的“文字”按钮。无需手动输入，我们可以使用智能识别功能。

![](img/4cee47ff79333f7cc134085cefcd4f5d_53.png)

以下是添加和编辑字幕的步骤：

![](img/4cee47ff79333f7cc134085cefcd4f5d_55.png)

1.  **识别字幕**：点击“识别字幕”，然后选择“仅视频”。建议不要选“全部”，以免将背景音乐里的歌词也识别出来。
2.  **检查修改**：系统会自动生成字幕并匹配时间点。检查是否有错别字，双击字幕条即可进行修改。
3.  **美化字幕**：您可以点击“编辑”，为字幕更换字体、颜色（例如，根据衣服颜色搭配字幕色）、添加动画等，让字幕更美观。

![](img/4cee47ff79333f7cc134085cefcd4f5d_57.png)

![](img/4cee47ff79333f7cc134085cefcd4f5d_35.png)

至此，一个完整的视频已基本制作完成：比例已调整，内容已剪辑，音乐已添加，字幕已生成。

![](img/4cee47ff79333f7cc134085cefcd4f5d_45.png)

![](img/4cee47ff79333f7cc134085cefcd4f5d_59.png)

在导出最终视频前，有一个关键设置需要注意，它直接影响视频的清晰度。

![](img/4cee47ff79333f7cc134085cefcd4f5d_61.png)

点击右上角的“导出”按钮。在导出设置中，请务必调整两个参数：
*   **帧率 (FPS)**：设置为 **30**。
*   **分辨率**：拉至最高（如1080p或4K）。

这样可以在符合抖音平台标准的同时，尽可能保证视频的高清画质。设置完成后，点击“导出”即可。导出过程中请勿锁屏或切换程序，耐心等待完成。

![](img/4cee47ff79333f7cc134085cefcd4f5d_63.png)

![](img/4cee47ff79333f7cc134085cefcd4f5d_47.png)

视频导出后，可以直接通过剪映分享到抖音。

除了剪辑，对于需要出镜口播的创作者，记忆台词是一大难题。接下来，我为大家推荐一个能有效解决这个问题的辅助工具。

![](img/4cee47ff79333f7cc134085cefcd4f5d_69.png)

这款软件叫做 **“爱提词”**。它的核心功能是将台词以提词板的形式悬浮在手机屏幕上，让您可以在拍摄时照着读，无需背诵。

![](img/4cee47ff79333f7cc134085cefcd4f5d_71.png)

以下是使用爱提词进行口播拍摄的基本方法：

![](img/4cee47ff79333f7cc134085cefcd4f5d_73.png)

1.  **添加台词**：打开App，点击右下角“+”号，新建一个提词稿。输入或粘贴您的台词内容，并保存。
2.  **开始提词**：打开刚创建的稿子，点击“开始提词”，选择“录制模式”。台词会以悬浮窗形式显示在拍摄界面上。
3.  **自定义设置**：您可以点击中间区域进行设置，调整字体大小、滚动速度等，方便阅读。
4.  **使用内置美颜**：在拍摄界面，点击左下角的“美颜”功能，可以适度调整瘦脸、大眼、磨皮、美白等效果。与一些美颜过度的相机相比，它的效果更自然。
5.  **获取灵感**：App内还设有“爆款文案”板块，涵盖情感、职场、生活等多个领域，可供您参考和学习。

![](img/4cee47ff79333f7cc134085cefcd4f5d_75.png)

![](img/4cee47ff79333f7cc134085cefcd4f5d_63.png)
