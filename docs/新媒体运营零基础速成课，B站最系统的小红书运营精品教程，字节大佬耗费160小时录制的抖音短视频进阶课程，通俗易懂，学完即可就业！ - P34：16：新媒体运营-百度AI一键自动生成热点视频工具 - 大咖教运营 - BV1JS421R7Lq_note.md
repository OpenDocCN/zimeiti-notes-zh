## 工具概述与界面介绍

上一节我们介绍了AIGC在内容创作中的潜力，本节中我们来看看一个具体的落地工具。百度AI创作平台提供了两大核心功能：**AI一键成片**和**AI文本生成与润色**。我们将重点讲解AI一键成片功能。

进入平台首页，可以看到“AI成片”和“AI笔记”两个主要入口。导航栏位于页面左侧。



## AI一键成片功能详解

AI成片功能主要分为两个部分。

### 1. 外部文案导入生成
在“输入文案”区域，你可以将预先准备好的视频脚本或文案粘贴进去。系统会自动根据文案内容搜索匹配的素材，并合成带有配音的完整视频。

![](img/6c2cc5bacfa697c3202b9d0a29d41bc6_3.png)

![](img/6c2cc5bacfa697c3202b9d0a29d41bc6_5.png)

### 2. 热点文章生成（需百家号）
如果你拥有百家号并发布过原创文章，可以选择“选择文章成片”。系统会列出你的文章，选择后即可一键生成视频。

### 3. 实时热点生成（核心功能）
这是工具最强大的功能。在界面右侧，系统会展示当前的热点事件列表。

![](img/6c2cc5bacfa697c3202b9d0a29d41bc6_7.png)

![](img/6c2cc5bacfa697c3202b9d0a29d41bc6_8.png)

以下是利用热点生成视频的步骤：
1.  **选择热点**：从右侧热点列表中点击一个你感兴趣的话题。
2.  **AI生成文案**：系统会自动根据该热点生成一段视频文案。
3.  **检查与优化文案**：生成后务必人工检查文案的准确性、合规性，并可将最吸引人的核心点调整至开头，以提升视频点击率。
4.  **选择关键素材**：系统会推荐与话题相关的图片或视频片段。你可以勾选认为合适的素材，这些素材会被优先插入到最终视频中。
5.  **一键生成视频**：点击“AI一键成片”，系统开始自动匹配素材、配音和背景音乐，合成视频。



## 视频的编辑与优化

AI生成的初版视频可能需要在细节上进行优化，以确保质量。

### 素材替换
如果AI匹配的素材不准确或缺失，可以进行手动替换。
*   **搜索替换**：在素材库中，使用关键词搜索更合适的素材。
*   **本地上传**：可以上传本地符合内容的素材文件。
*   **免责声明**：对于新闻类内容，若使用非原始素材，建议在视频中添加声明，例如：“素材来源于网络，非事件原视频”。

![](img/6c2cc5bacfa697c3202b9d0a29d41bc6_10.png)

### 模板与样式
系统提供横版和竖版等多种视频模板，可根据发布平台（如抖音用竖版）进行选择。

![](img/6c2cc5bacfa697c3202b9d0a29d41bc6_12.png)

### 配音与背景音乐
*   **朗读音**：可以更换不同的AI配音音色，并调整语速。例如，紧张的社会新闻可以调快语速以营造氛围。
    ```python
    # 概念示例：调整语音参数
    voice_settings = {
        “speaker”: “female_fast”, # 选择音色
        “speed”: 1.5 # 调整语速倍率
    }
    ```
*   **背景音乐**：可以在内置音乐库中选择，但目前不支持自定义上传。你可以在视频下载后，用专业剪辑软件替换背景音乐。

## 视频的下载与发布

![](img/6c2cc5bacfa697c3202b9d0a29d41bc6_19.png)

视频编辑完成后，按以下步骤操作：
1.  点击右上角的“发布视频”按钮。
2.  系统会检查素材是否全部匹配。如有缺失提示，需补充相应素材。
3.  素材补充完毕后，点击“生成视频”。
4.  等待几分钟，视频渲染完成后，页面会出现“下载”按钮。
5.  将视频下载到本地后，可进行最终调整，然后发布到任意平台。

![](img/6c2cc5bacfa697c3202b9d0a29d41bc6_21.png)

![](img/6c2cc5bacfa697c3202b9d0a29d41bc6_23.png)

**重要提示**：使用工具时，必须严格遵守法律法规和平台规则，不得制作或传播谣言等非法内容。



## AI笔记与文案辅助工具

![](img/6c2cc5bacfa697c3202b9d0a29d41bc6_25.png)

除了视频生成，该平台的“AI笔记”功能也是一个强大的文案助手，类似于ChatGPT，适用于新媒体运营的多种场景。

AI笔记能帮助我们完成以下工作：
*   **撰写文案**：根据给定主题（如“地球还有多少年寿命”）自动生成文章初稿。
*   **文笔润色**：对已有的文案进行优化，使其更流畅、更具吸引力。
*   **辅助创作**：可用于设计直播话术、产品宣传文案、脚本等。
