# 剪映教程：P10：画中画功能详解 🎬

在本节课中，我们将要学习剪映的核心剪辑功能之一——画中画。这个功能允许你将多个视频或图片叠加在同一个时间线上播放，从而创造出内容更丰富、更有趣的视频效果。

![](img/e07f7981d286fc8dd9ef2553816f038b_1.png)

## 什么是画中画？

![](img/e07f7981d286fc8dd9ef2553816f038b_3.png)

上一节我们介绍了基础的剪辑功能，本节中我们来看看画中画。画中画功能可以实现多个画面同时播放的效果。这些画面可能不在同一个时间线上，但通过叠加，它们可以出现在同一视频中。许多特效视频都借助了画中画功能，将两个或多个视频素材叠加在同一个轨道上同步播放。

![](img/e07f7981d286fc8dd9ef2553816f038b_5.png)

![](img/e07f7981d286fc8dd9ef2553816f038b_7.png)

**核心概念**：画中画本质上是创建了一个新的视频轨道（`轨道N`）叠加在主轨道（`轨道0`）之上。其基本操作可表示为：
```
最终画面 = 叠加(轨道0, 轨道1, ..., 轨道N)
```
使用画中画能使视频内容变得更加丰富有趣。

![](img/e07f7981d286fc8dd9ef2553816f038b_9.png)

## 画中画实操指南

![](img/e07f7981d286fc8dd9ef2553816f038b_11.png)

![](img/e07f7981d286fc8dd9ef2553816f038b_12.png)

了解了基本概念后，我们通过一个具体案例来学习如何操作。

![](img/e07f7981d286fc8dd9ef2553816f038b_14.png)

![](img/e07f7981d286fc8dd9ef2553816f038b_16.png)

**第一步：导入并设置主素材**
1.  打开剪映，点击“开始创作”。
2.  导入一张照片作为背景素材。
3.  进入编辑界面后，首先调整视频比例。点击底部工具栏的“比例”，选择“9:16”。

![](img/e07f7981d286fc8dd9ef2553816f038b_17.png)

![](img/e07f7981d286fc8dd9ef2553816f038b_18.png)

**第二步：添加画中画素材**
以下是添加画中画的具体步骤：
1.  返回一级菜单，点击“画中画”功能。
2.  选择“新增画中画”。
3.  从相册导入第二个素材，例如一段星空特效视频。

**第三步：剪辑与对齐画中画素材**
1.  如果导入的星空素材开头有多余部分，将时间线光标移动到需要保留的开始位置。
2.  点击“分割”工具，然后删除前半部分不需要的片段。
3.  将保留的星空片段拖动，与主照片素材的起始位置对齐。

**第四步：混合模式抠图**
此时，画中画的星空素材可能带有黑色背景。我们需要将其去除，只保留星星。
1.  选中时间线上的画中画（星空）轨道。
2.  在底部工具栏中找到并点击“混合模式”。
3.  在混合模式选项中，选择“绿色”。这个模式会自动滤除黑色背景，使星空叠加在照片上。

![](img/e07f7981d286fc8dd9ef2553816f038b_20.png)

**第五步：调整与预览**
1.  可以拖动、缩放或旋转画中画素材，将其调整到合适的位置。
2.  点击播放预览最终效果。现在，照片背景上已经叠加了动态的星空特效。

通过以上步骤，我们就利用画中画功能完成了一个简单的特效视频制作。

## 功能总结与应用

本节课中我们一起学习了剪映的画中画功能。我们来总结一下：
*   **功能核心**：画中画允许在原视频轨道上新增独立的视频轨道，实现多轨道素材的同步播放。
*   **主要作用**：它将单一视频画面变为两个甚至多个画面的组合，极大地丰富了视频内容。
*   **应用场景**：掌握画中画是制作多种特效的基础。除了本节课的叠加特效案例，未来制作三连屏、分屏效果或人影分身等高级技巧，都需要用到画中画功能。

![](img/e07f7981d286fc8dd9ef2553816f038b_22.png)

理解并熟练运用画中画，能为你的视频创作打开更多可能性。
