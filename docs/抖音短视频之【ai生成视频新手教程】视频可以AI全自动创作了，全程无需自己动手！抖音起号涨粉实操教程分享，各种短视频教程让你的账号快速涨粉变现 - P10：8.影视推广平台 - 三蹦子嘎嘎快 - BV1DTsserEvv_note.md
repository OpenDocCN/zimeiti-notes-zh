# AI短视频创作：P10：8.影视推广平台 💰

![](img/f69b05d472ffb61d9e9a662068d690f7_1.png)

在本节课中，我们将要学习如何通过“影视推广平台”为你的AI创作视频结算收益。我们将详细讲解哪些平台可以带来直接收入，以及如何通过完成推广任务来获得比单纯播放量分成更高的奖金。

![](img/f69b05d472ffb61d9e9a662068d690f7_3.png)

---

上一节我们介绍了视频的多平台分发，本节中我们来看看如何通过特定平台将播放量转化为实际收益。

![](img/f69b05d472ffb61d9e9a662068d690f7_5.png)

很多同学问到，如果想把视频发布到抖音或快手，该如何结算收益。需要明确的是，我们制作完成的视频，除了抖音和快手，发布到其他30多个自媒体平台，都可以依靠视频本身的播放量来结算收益。

![](img/f69b05d472ffb61d9e9a662068d690f7_7.png)

但是，抖音和快手这两个平台，视频发布后主要的作用是增长粉丝。这是需要强调的一点。

那么，有没有同学只想专注于做抖音和快手这两个熟悉的平台，同时又不仅想涨粉，还想依靠播放量来结算收益呢？答案是肯定的。

对于这类需求，视频可以发布到四个平台：**抖音、快手、小红书以及微信视频号**。这些都是我们日常会使用到的平台。小红书在年轻用户群体中尤为流行，而微信视频号则内置于我们每个人都在使用的微信中。

在这几个平台发布视频后，我们通过“影视推广平台”来接取任务，并通过完成任务来结算奖金。平台上的素材非常多，包括爱奇艺、搜狐等平台的任务，这些任务都对应着具体的影视剧名称。

以下是关于影视推广的核心说明：

*   **影视推广**：指的是为电影、电视剧或综艺节目进行线上宣传。由于线下宣传受限，片方会通过推广平台发布任务，创作者制作推广视频并发布，平台则根据视频的播放量等数据给创作者结算奖金。
*   **任务提交流程**：完成视频制作后，需要到平台提交任务回填链接。需要填写的信息包括：微信号、账号昵称、发布平台ID、对应的影视剧名称、作品链接、发布时间以及视频播放量的截图。提交完整信息后，方可结算收益。

影视推广的收益，通常比单纯依靠平台播放量分成的收益更高。平台上有大量的短剧推广任务，这些短剧由传媒公司或网红自制，他们同样需要推广。

发布平台主要是**抖音、小红书、视频号**。结算时，取一部剧最高播放量的三个作品，其播放量总和作为结算依据。对创作者账号的粉丝量没有要求。

以下是各平台推广任务的收益计算方式：

*   **抖音**：每1万播放量，奖金为50元。
*   **微信视频号**：每1万播放量，奖金为10元。（因用户基数大，单价较低）
*   **小红书**：每1000播放量，奖金为30元。（单价最高）
*   **抖音+快手（联合任务）**：按点赞数结算，无粉丝要求。例如，单个作品5000赞奖励100元；抖音1万赞奖励200元；快手1万赞奖励180元。

此外，像搜狐短剧等任务，虽然其平台热度可能不如前几个，但提供的推广单价往往是最高的。

本节课中我们一起学习了“影视推广平台”的运作模式。关键点在于：通过接取具体的影视推广任务，并在指定的社交平台（如抖音、快手、小红书、视频号）发布视频，可以根据播放量或点赞数获得比普通流量分成更高的奖金。这为我们在专注平台涨粉的同时，开辟了一条有效的变现途径。
