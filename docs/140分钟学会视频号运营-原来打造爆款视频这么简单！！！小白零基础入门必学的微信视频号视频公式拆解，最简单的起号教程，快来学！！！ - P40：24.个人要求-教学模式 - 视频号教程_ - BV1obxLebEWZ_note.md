# 视频号运营：P40：个人要求与教学模式 📋

在本节课中，我们将明确学习视频号运营的个人要求与教学模式。了解这些要求是确保学习效果和后续实践成功的基础。

上一节我们介绍了视频号运营的潜力，本节中我们来看看参与学习的具体要求。

我的要求比较多，首先需要说明的是时间。时间是最重要的因素。一天24小时对每个人都是公平的。无论你是什么身份，例如宝妈或宝奶奶，我都理解你们的不易。这类学员通常只有在特定时间段才有空，例如早上7点半之前、中午一两点钟或晚上10点半以后。

![](img/da52c5a5d30dc09d50f3f8ca83338ba3_3.png)

因此，我的核心要求是：**每天能抽出至少30分钟**。只要满足这个时间条件，我就可以带你操作一个具体的玩法——创作者分成计划。坚持操作，一个账号一天发三条内容，一个月可获得约7200元收益。如果你有两个账号，收益则可能翻倍。

对于想做全职的学员，要求是**每天能投入一小时以上**。这样我们可以操作更多玩法，例如创作者分成计划、视频带货和无人直播。创作者分成计划的收益如前所述。视频带货方面，参与商家联盟活动，31天收益可达约9000元。请注意，该活动即将截止。一个账号收益约16000元，两个账号收益则可能超过30000元。

![](img/da52c5a5d30dc09d50f3f8ca83338ba3_5.png)

以下是关于时间要求的具体说明：
*   **基础要求**：每天至少30分钟，用于操作创作者分成计划。
*   **进阶要求**：每天一小时以上，可操作创作者分成、视频带货、无人直播等多玩法。

明确了时间要求后，我们来看看对学员个人品质的要求。

我的第二个要求是：**不能拖拉，不能偷懒**。我出身部队，做事讲究“快、准、狠”。我最讨厌有执行力却拖拖拉拉、影响效率的人。积极互动、有行动力的学员往往学得更好。

![](img/da52c5a5d30dc09d50f3f8ca83338ba3_9.png)

第三个要求是：**必须告知真实情况**。这样我才能为你量身定制学习方案。例如，宝妈时间零散，老人学习可能较慢，上班族晚上才有时间，或有辞职打算的学员，情况都不同。甚至存在年纪大、文化水平低、不识字等特殊情况。

我有一份学员情况记录表，里面记录了各种背景的学员，例如56岁在家带孙子的吴春兰、63岁不识字的周建国、84岁的周梅阿姨，以及破产的宝妈、大学生等。我都会根据具体情况指导。如果你担心自己学不会，不必忧虑。对于学习困难的学员，我会采用“笨办法”，别人练一遍，你就练三遍。从陌生到熟练需要一个过程，我的目标是让大家在最短时间内拿到结果。

![](img/da52c5a5d30dc09d50f3f8ca83338ba3_11.png)

综合以上几点，我的核心教学要求可以总结为：**听话照做，全力以赴**。

![](img/da52c5a5d30dc09d50f3f8ca83338ba3_13.png)

为什么要求“听话照做，全力以赴”？因为只有这样，我才能确保你**百分之百学会、百分之百出单、百分之百赚到钱**。或许你会质疑是否有百分之百成功的事。我出身军人家庭，深知中国人创造过许多“不可能”。我的办公室里一直挂着毛主席的画像，它代表一种责任和信念。我相信，只要学员能做到这八个字，就一定能达成目标。
