# 剪映教程：P13：剪映核心剪辑功能——色度抠图功能 ✂️🎨

在本节课中，我们将要学习剪映手机版的一项核心且强大的剪辑功能——色度抠图。这项功能可以帮助我们轻松地将绿幕背景的特效素材抠出，并合成到其他视频中，是制作特效类视频的利器。

色度抠图功能是剪映APP近期新增的一项功能，它极大地方便了用户在手机端进行视频合成创作。过去，这类操作通常需要在PR或AE等专业软件中完成。现在，通过剪映的色度抠图，我们可以便捷地处理绿幕素材，实现丰富的视觉效果。

上一节我们介绍了色度抠图的基本概念，本节中我们来看看如何具体操作。接下来，我们将通过一个简单的实例，演示如何使用色度抠图功能。

---

## 操作演示：合成飞机飞过天空的特效

![](img/1cb6bed3ec001b17f72483e17e7ac071_1.png)

以下是使用色度抠图合成视频的具体步骤：

1.  **导入主视频素材**  
    首先，在剪映中导入一段作为背景的主视频素材，例如一段天空的视频。

2.  **添加绿幕素材**  
    点击 **画中画** 功能，导入第二段带有绿幕背景的特效视频素材（例如一架飞机的绿幕素材）。此时，绿幕素材会叠加在天空视频之上，背景仍是绿色。

    ![](img/1cb6bed3ec001b17f72483e17e7ac071_1.png)

![](img/1cb6bed3ec001b17f72483e17e7ac071_3.png)

3.  **使用色度抠图**  
    选中画中画轨道上的绿幕素材，在底部工具栏中找到 **色度抠图** 选项（位于蒙版功能旁边）。

![](img/1cb6bed3ec001b17f72483e17e7ac071_5.png)

![](img/1cb6bed3ec001b17f72483e17e7ac071_7.png)

4.  **取色与调整**  
    *   使用取色器（一个吸管图标）点击画面中的绿色背景。
    *   拖动 **强度** 滑块，逐步将绿色背景扣除干净。调整时需观察画面，确保主体被完整保留，边缘处理干净。
    *   如果抠图后主体边缘有残留的绿色杂边，可以适当调整 **阴影** 参数，使合成效果更自然柔和。

    ![](img/1cb6bed3ec001b17f72483e17e7ac071_3.png)

    ![](img/1cb6bed3ec001b17f72483e17e7ac071_5.png)

5.  **裁剪与编辑**  
    *   如果素材带有水印，可以使用裁剪功能将其去掉。
    *   调整绿幕素材的位置和大小，使其与背景视频协调。例如，将飞机调整到合适的飞行轨迹上。
    *   裁剪掉素材多余的部分，使合成节奏紧凑。

    ![](img/1cb6bed3ec001b17f72483e17e7ac071_7.png)

![](img/1cb6bed3ec001b17f72483e17e7ac071_9.png)

![](img/1cb6bed3ec001b17f72483e17e7ac071_10.png)

6.  **添加动画与调整**  
    *   可以为飞机的入场添加关键帧动画，使其飞行过程更平滑。
    *   调整素材的入场时间点，删除前面不必要的静止片段。

    ![](img/1cb6bed3ec001b17f72483e17e7ac071_9.png)
    ![](img/1cb6bed3ec001b17f72483e17e7ac071_10.png)

7.  **最终预览**  
    完成所有调整后，预览最终效果。飞机将从天空飞过，与背景视频完美融合。

    ![](img/1cb6bed3ec001b17f72483e17e7ac071_12.png)

本节课中我们一起学习了剪映的色度抠图功能。总结来说，**使用色度抠图功能，可以很方便地将绿幕素材中的特效主体抠出来，再结合画中画功能进行合成**。这为我们制作抖音、短视频中常见的各种特效视频提供了极大的便利。
