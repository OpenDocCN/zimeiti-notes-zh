## 核心概念：核心关键词与长尾关键词
上一节我们介绍了内容策划的基础，本节中我们来看看关键词的分类。关键词主要分为两类：**核心关键词**和**长尾关键词**。

*   **核心关键词**：指代一个行业或产品的通用、概括性词汇。例如：`女装`、`奶粉`、`压缩机`。
*   **长尾关键词**：由核心关键词拓展而来的、更具体、更长的短语。它通常反映了用户更细分、更具体化的需求。例如，由核心词“女装”拓展出的`女装图片`、`女装店`、`女装穿搭`。

![](img/17b21cab7ad16dfb3235fc64865a7bba_13.png)

理解这两个概念是进行关键词分析和内容布局的基础。

## 关键词分析工具：需求图谱与拓词逻辑
为了找到用户的需求，我们需要借助关键词分析工具。这类工具能帮助我们洞察用户搜索行为，从而指导内容创作。

![](img/17b21cab7ad16dfb3235fc64865a7bba_15.png)

以下是使用关键词分析工具的核心步骤：

1.  **输入核心词**：在工具中输入你的行业或产品核心词，例如`眉笔`。
2.  **分析需求图谱**：工具会生成一个“词云图”或“需求图谱”。图中字体越大，代表该关联词被搜索的频率越高。例如，搜索“眉笔”的用户，可能高频关联搜索`颜色`、`图片`、`品牌`等词。这揭示了用户的关注点。
3.  **进行关键词拓展**：点击图谱中的高频关联词（如`颜色`），工具会进一步推荐更具体的词，例如`眉笔颜色怎么选`、`眉笔什么颜色好看`。这就是**长尾关键词**的生成逻辑。

![](img/17b21cab7ad16dfb3235fc64865a7bba_17.png)

![](img/17b21cab7ad16dfb3235fc64865a7bba_19.png)

![](img/17b21cab7ad16dfb3235fc64865a7bba_21.png)

**核心公式**：`核心关键词` -> `高频关联需求词` -> `具体长尾关键词`

![](img/17b21cab7ad16dfb3235fc64865a7bba_23.png)

这个工具的价值在于，它能帮你发现那些并非平台热点，但却是你目标用户真实存在的“小众需求”。覆盖这些需求，能让你吸引到更精准、粘性更高的粉丝。

## 小红书平台的关键词高级玩法
了解了关键词的基础和分析方法后，我们来看看在小红书平台上的一些实战技巧。小红书是一个极度依赖关键词匹配的图文内容平台。

### 小红书笔记的三要素
一篇完整的小红书笔记包含三个关键部分，都与关键词息息相关：
1.  **标题**
2.  **文案**
3.  **标签**

![](img/17b21cab7ad16dfb3235fc64865a7bba_29.png)

### 关键词堆砌玩法
目前，平台上存在一种“关键词堆砌”的玩法。具体操作是：在笔记正文结束后（标签之后），额外添加一段看似与标签相关、但实际由大量重复关键词组成的文字。

![](img/17b21cab7ad16dfb3235fc64865a7bba_31.png)

![](img/17b21cab7ad16dfb3235fc64865a7bba_33.png)

**示例代码（模拟堆砌文案）**：
```
#瑞幸拿铁 #咖啡推荐
...（正文结束）...
瑞幸拿铁 瑞幸 瑞幸咖啡 微信校园 拿铁推荐 瑞幸点单攻略 瑞幸隐藏菜单 咖啡 学生党咖啡
```

![](img/17b21cab7ad16dfb3235fc64865a7bba_35.png)

**玩法原理**：
这种做法的目的是增加目标关键词（如“瑞幸”）在整篇笔记中的出现频率。当小红书的**Spider**（网络爬虫程序）抓取和识别笔记内容时，会统计词汇频率。更高的关键词频率可能提示系统，该笔记与某个主题高度相关，从而可能在用户搜索该关键词时获得更好的排名展示。

![](img/17b21cab7ad16dfb3235fc64865a7bba_37.png)

**注意事项**：
*   **效果有限**：这种方法的作用因产品、类目和平台算法调整而异，并非绝对有效。
*   **影响体验**：对用户而言，这段堆砌的文字阅读体验不友好。
*   **历史渊源**：这其实是传统网站**SEO**中一种初级策略的变体。

### （拓展知识）传统SEO的隐藏玩法
在网站SEO时代，有过更极端的做法：通过修改文字颜色（如将字体颜色设置为与背景色相同），使关键词对用户“不可见”，但**Spider**依然能抓取到。例如：
```html
<span style="color: #ffffff;">仓鼠 仓鼠饲养 仓鼠品种</span>
```
*   *用户看到的是空白，但爬虫能读取到“仓鼠”等关键词。*
**请注意**：小红书平台目前不支持随意修改字体颜色，且此类隐藏文本的行为在大多数平台都被视为违规，存在风险，不推荐使用。
