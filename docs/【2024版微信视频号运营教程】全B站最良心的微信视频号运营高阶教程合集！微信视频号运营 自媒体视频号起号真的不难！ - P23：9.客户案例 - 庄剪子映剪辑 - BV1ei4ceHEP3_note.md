# 微信视频号运营教程：P23：9.客户案例

## 概述
在本节课中，我们将通过一个具体的客户案例——周建国（周叔）的学习经历，来了解视频号运营从零开始的实际操作过程。我们将看到如何克服学习障碍、利用特殊通道快速参与平台计划，并理解其中的收益规则。

上一节我们介绍了视频号运营的基础框架，本节中我们来看看一个真实的学员是如何从零起步的。

![](img/839b94f3e5758f9ae76d671c451af64a_4.png)

![](img/839b94f3e5758f9ae76d671c451af64a_5.png)

当时我也是这样一步一步去教我们周建国周叔的。我告诉他按照这个步骤流程去操作，他说他去试一下。

我是去年的6月20号中午12点49分给他发的消息。到了12点58分，过了大概10分钟。

我看到周建国周叔没有回复我，我猜测他遇到了问题。我就问他是否出了问题。

他回复说，他记性不太好，总是搞不好。这个情况很常见。我发现老一辈的同学，包括有些同学，总是不好意思，觉得会给老师添麻烦，或者有一点自卑。他不好意思，总觉得老师会不会觉得他太笨，或者学得太慢。

所以我从来不会在群里讲今天谁挣了多少，明天谁进步了多少。因为我知道有些同学自己一个人搞得挺好，但看到别人好像挣得多或进步快，就会着急。我跟大家讲，不要跟别人比，你跟自己比就可以了。

然后他不会操作。我说周叔没关系，接下来我给你打视频，我点哪里你就点哪里。我给他打视频，他拒绝了。我就问他是否不方便。

![](img/839b94f3e5758f9ae76d671c451af64a_7.png)

他回复说他点错了。接通之后，我给他打了一个30分钟的视频电话。我记得很清楚，我教了他四遍。

![](img/839b94f3e5758f9ae76d671c451af64a_7.png)

为什么教了四遍？不是他不肯学。当时的情况是，我跟周建国周叔说，让他按照我箭头所指的，找到那个视频号。因为他不识字，我说你找到这个黄黄的，像只狐狸一样的图标。他说他找不到。

我说怎么可能找不到呢，咱们的手机难道长得不一样吗？我让他截图发过来。截图之后发现还真不一样。

![](img/839b94f3e5758f9ae76d671c451af64a_9.png)

![](img/839b94f3e5758f9ae76d671c451af64a_10.png)

比如这是周建国周叔的手机，他的手机字体比我的要大。后来我想明白了，他使用的是老年模式。这就导致原来在这个位置的东西，跑到别的地方去了。

![](img/839b94f3e5758f9ae76d671c451af64a_9.png)
![](img/839b94f3e5758f9ae76d671c451af64a_10.png)

所以我想明白了原因。要解决这个问题很简单，我想了一个办法。

我用我的一台手机给周建国周叔打视频电话，对着我的另一台手机。我把另一台手机的显示设置也调成老年模式，跟他的一模一样，然后我点哪里他就点哪里。这样教年纪大的同学，他们就能学会。

![](img/839b94f3e5758f9ae76d671c451af64a_12.png)

还是那句话，没有教不会的学生，只有不会教的老师。有的时候笨办法、死办法，但效果挺好。

注册完之后，我给大家发了一个海报。

![](img/839b94f3e5758f9ae76d671c451af64a_14.png)

![](img/839b94f3e5758f9ae76d671c451af64a_12.png)

为什么要发海报呢？因为刚才我们说了，正常流程需要100个粉丝且有原创视频才能参加创作者分成计划。周建国周叔刚刚注册视频号，按正规流程肯定不行。

但是，今天再给大家发放一个福利。各位同学可以跟周建国一样，直接走我独家的内测通道。哪怕你是零粉丝、零作品、零点赞，也可以直接加入创作者分成计划。这就是我给大家发放的第二个福利。

![](img/839b94f3e5758f9ae76d671c451af64a_16.png)

![](img/839b94f3e5758f9ae76d671c451af64a_17.png)

那怎么进入我的内测通道呢？请看清楚，这个就是我们内测通道独家的海报。

![](img/839b94f3e5758f9ae76d671c451af64a_16.png)
![](img/839b94f3e5758f9ae76d671c451af64a_17.png)

海报上写着“视频号官方内购创作者分成计划，张院长唯一指定邀请函，零粉丝也可加入，有效期至2024年12月”。

怎么进行添加呢？在海报的下方，有一个专属的二维码。你只要扫描即可添加腾讯集团旗下微信视频号官方进行策划。

![](img/839b94f3e5758f9ae76d671c451af64a_19.png)

这个二维码现在不用扫，因为扫了没有用。这个二维码是每人一码的。扫描完成后，你还需要把信息录入到后台系统，这样收益才能打到你的账户上。

![](img/839b94f3e5758f9ae76d671c451af64a_19.png)

接下来给大家讲一下收益部分，这也是大家最关心的。

首先，我们有一个提成规则，大家应该有印象。这个提成之前讲得很详细：**1000个播放量是10块钱，1万个是100块钱**。

最关键的是，我之所以会带着一位63岁的大叔去做这个玩法，就是因为它有保底收益。他这么大年纪，如果没有一个兜底的东西，作为一个新手，很难拿到很高的收益。

![](img/839b94f3e5758f9ae76d671c451af64a_21.png)

请看清楚这个规则：**你只要发布一条视频，即可获得80块钱的保底收益**。也就是说，你只要把视频发出去了，不管数据如何，哪怕点赞和播放都是零，也可以获得80元。

80块钱可以买包烟，可以买个菜。而且，经过我的申请，你一天可以发3条视频。**3 * 80 = 240块钱**。

一个账号最多发3条。当然，理论上你可以有多个账号。但是，目前一个身份证只能绑定两个账号。并且现在是一人一卡一手机，防止封号受限。经过测试，一个人操作三四个账号就差不多了，做太多账号（比如10个）需要10台手机、5张身份证，根本忙不过来。

## 总结
本节课中，我们一起学习了客户周叔的实操案例。我们看到了老年学员可能遇到的界面问题及其解决方案，了解了通过独家内测通道让零基础账号快速加入创作者计划的方法，并重点分析了视频号计划的保底收益规则。记住，关键点在于**利用特殊通道快速启动**，以及理解**80元/条的保底收益**和**单账号日发3条**的规则。对于初学者，从简单可执行的步骤开始，与自己比较，逐步进步即可。
