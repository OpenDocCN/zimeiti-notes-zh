## 第二步：直播间“分享员”裂变玩法

完成账号准备后，下一步的核心是直播间的产品推广与用户裂变。这一步的关键在于设计一个极具吸引力的用户分享机制，让观众自发成为你的推广员。

在直播过程中，我们设置了一个简单的规则：**凡是下单购买任意商品的用户，都可以自动成为“分享员”**。

以下是“分享员”机制的核心优势：

1.  **自购返佣**：分享员自己下单购买商品，可以获得返佣。例如，一件售价20元的商品，分享员购买后可获得4元返佣。公式表示为：`自购返佣 = 商品售价 × 返佣比例`。
2.  **分享获利**：分享员将直播间分享到朋友圈或社群，好友通过其分享链接下单，分享员同样能获得佣金。
3.  **门槛极低**：成为分享员无需像内容达人那样创作视频或直播，唯一动作就是“分享”。这极大降低了参与门槛，代码逻辑上可以简化为：`if (用户下单) { 用户角色 = “分享员”; }`。
4.  **收益可观**：尽管门槛低，但通过自购和分享带来的双重收益，对用户有很强的吸引力。

## 玩法带来的三重收益联动

这套分享玩法的作用不仅仅是直接促进销售。它像一台发动机，启动了销量、私域流量和公域流量的联动增长。

*   **销量提升**：直接的返佣激励刺激了直播间内的成交。
*   **私域积累**：每一位分享员都将自己的社交关系（朋友圈、社群）引入了你的直播间，这为你沉淀了宝贵的私域流量。
*   **公域助推**：大量私域用户的涌入，显著提升了直播间的互动、停留、转化等数据。良好的数据表现会触发视频号的公域流量推荐机制，为直播间带来更多自然流量。

最终，**销量、私域流量、公域流量**三者形成正向循环，互相促进，从而实现销售额的成倍增长。
