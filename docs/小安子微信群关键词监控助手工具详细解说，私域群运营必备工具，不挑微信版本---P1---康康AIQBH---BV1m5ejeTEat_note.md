## 工具概述与核心优势 🔍

小安子微信监控助手是一款基于云电脑和RPA（机器人流程自动化）技术实现的工具。它能够自动监控微信（包括群聊和私聊）中的聊天内容，并根据设定的关键词进行记录与提取。

![](img/db1db180e788139d281b35e3364c0fa3_2.png)

它的主要优点体现在以下三个方面：
*   **安全稳定**：采用RPA技术模拟人工操作监控微信程序，**不支持多PC端同时登录**，有效避免了封号风险。
*   **广泛兼容**：支持所有版本的微信电脑客户端，微信版本升级也不受影响。
*   **高效持续**：提供7x24小时不间断的监控服务。

## 常见问题解答 ❓

了解了应用场景后，您可能对工具的使用存在一些疑问。以下是用户最常关心的几个问题及其解答：

*   **问：对微信版本有要求吗？**
    **答**：没有要求。工具支持微信所有电脑版本，且版本升级不受影响。
*   **问：使用工具有封号危险吗？**
    **答**：风险很低。监控在云端RPA环境中运行，**仅需在PC端扫码登录一次**，不涉及微信协议破解，因此安全性高。
*   **问：监控前需要做什么准备？**
    **答**：需要准备一个可用于在电脑端登录的微信号。请注意，微信**不支持**同一账号在多个PC端同时登录，开启监控时，其他电脑端的微信会被迫下线。
*   **问：工具是如何实现监控的？**
    **答**：监控机器人会模拟人工点击未读消息，提取最新聊天内容，并将`群名称`、`发言人`、`发言内容`发送到后台。您可以在后台查看所有记录。
*   **问：是实时监控吗？**
    **答**：基本上是实时的。通常一秒内即可捕获信息，但在信息量极大的活跃群中，可能会有轻微延迟。
*   **问：支持监控图片和语音吗？**
    **答**：目前仅支持文本内容监控。图片、语音监控属于定制开发需求。
*   **问：是否提供本地部署？**
    **答**：提供。可将整个平台部署在客户自己的服务器上，实现数据本地化，具体价格需咨询客服。

## 后台操作指南 🖥️

上一个视频对小安子平台做了介绍，现在我们一起进入用户后台，学习具体的操作步骤。首先，访问小安子官方网站并登录。请注意，账号需联系客服开通授权，自行注册的账号无法使用。

登录后，您将看到主面板，这里展示了账号下的微信号数量、昨日监控情况等统计信息。“我的微信号”区域管理所有监控账号，“最新日志”区域则显示最近的监控记录。

### 微信号与关键词管理

点击左侧“微信号”菜单，可以添加需要监控的微信。但添加后需客服授权才能启用。在此页面，您可以设置监控关键词。
**操作代码示例（添加关键词）**：
```
1. 在关键词输入框中，输入一个关键词。
2. 按下回车键，即可添加该关键词。
3. 重复上述步骤，可添加多个关键词。
4. 所有关键词输入完毕后，点击“提交”按钮。
```

![](img/db1db180e788139d281b35e3364c0fa3_6.png)

### 启动监控流程

当微信号获得客服授权后，您可以点击“开始”按钮。此时系统会弹出一个微信PC端的登录二维码。请使用您的手机微信扫描此二维码，并按照提示（通常需确认“在新设备登录”）完成授权登录。

手机授权成功后，返回小安子后台，点击“我已经扫码登录成功”的确认按钮，监控任务便正式启动。此时，原“开始”按钮会变为“暂停”按钮，表示监控正在运行。
