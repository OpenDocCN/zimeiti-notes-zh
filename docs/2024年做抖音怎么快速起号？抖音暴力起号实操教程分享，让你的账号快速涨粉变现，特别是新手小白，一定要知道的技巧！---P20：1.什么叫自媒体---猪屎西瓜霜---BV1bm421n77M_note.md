# 抖音运营入门：P20：1. 什么叫自媒体

在本节课中，我们将要学习自媒体的基本概念、现状与前景，以及如何从零开始打造一个能够变现的账号。课程内容分为四个主要板块，旨在帮助初学者快速理解并入门自媒体领域。

## 课程内容板块划分

以下是本课程涵盖的四个核心板块。

1.  **自媒体现状与前景分析**：了解行业的当前状况和未来潜力，明确从事自媒体能带来的好处。
2.  **零基础打造高收益账号**：学习如何选择平台、领域和渠道，解决视频零播放或无法变现的问题。
3.  **亿万流量的多种变现玩法**：探索除播放量收益外，其他适合新手的变现方法。
4.  **自媒体变现的正确规划**：学习如何系统性地规划账号运营，将收益稳定地赚到手。

## 什么是自媒体？

上一节我们介绍了课程的整体框架，本节中我们来看看最基础的概念：到底什么是自媒体？

![](img/63882ade7b8ebf186182d7244a4fe79a_3.png)

我用最直白的话来解释：**自媒体就是个人媒体**。

具体操作可以概括为以下步骤：

![](img/63882ade7b8ebf186182d7244a4fe79a_5.png)

1.  打开手机或电脑。
2.  在搜索引擎（如百度）上搜索“自媒体平台”。
3.  选择一个平台并注册一个属于你自己的账号。
4.  在你的账号上发布你自己创作的文章或视频。
5.  当有人观看你的内容时，你就可以依靠播放量或阅读量来获得平台结算的收益。

![](img/63882ade7b8ebf186182d7244a4fe79a_7.png)

这个过程，就叫做做自媒体。其核心公式可以简化为：

![](img/63882ade7b8ebf186182d7244a4fe79a_9.png)

**自媒体 = 个人账号 + 自主创作 + 平台发布 + 流量变现**
