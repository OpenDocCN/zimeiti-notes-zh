# 巨量千川投放教程：P1：新户直播间起号计划搭建技巧 🚀

![](img/1d373233fafd4515c178fb4b76c00ffd_0.png)

![](img/1d373233fafd4515c178fb4b76c00ffd_2.png)

在本节课中，我们将学习如何为巨量千川的新直播间搭建有效的起号投放计划。我们将从计划类型、数量、定向、创意到优化策略，系统地讲解核心技巧，帮助初学者快速上手。

![](img/1d373233fafd4515c178fb4b76c00ffd_4.png)

## 计划类型与数量 📊

新直播间起号，应以投放“成交”为目标，并主要使用“控成本”计划类型。

![](img/1d373233fafd4515c178fb4b76c00ffd_6.png)

以下是关于计划数量的建议：
*   若预算不高，前期建议搭建5至10条计划。
*   若预算有限，不应建立过多计划，可适当提高出价。

![](img/1d373233fafd4515c178fb4b76c00ffd_8.png)

## 人群定向策略 🎯

![](img/1d373233fafd4515c178fb4b76c00ffd_10.png)

上一节我们介绍了计划的基础设置，本节中我们来看看核心的人群定向策略。前期搭建的第一批计划，定向必须精准。因为后续我们将从这些计划中进行“赛马”筛选，并复制优化。

![](img/1d373233fafd4515c178fb4b76c00ffd_12.png)

如果第一批计划定向不准，例如莱卡计划设置不当，可能第一天还能消耗，第二天就会跑不动。因此，人群定向应遵循“先精准，后放开”的原则。

![](img/1d373233fafd4515c178fb4b76c00ffd_14.png)

以下是关于人群包使用的具体建议：
*   除了“店铺历史成交人群包”和“老账户人群包”，其他情况不建议使用人群包。

![](img/1d373233fafd4515c178fb4b76c00ffd_16.png)

## 创意形式选择 ✨

![](img/1d373233fafd4515c178fb4b76c00ffd_18.png)

创意方面，建议同时尝试“直投直播间”和“投放视频素材”两种形式，以测试哪种效果更好。

![](img/1d373233fafd4515c178fb4b76c00ffd_20.png)

如果视频素材无法通过审核，则应以直投为主。

![](img/1d373233fafd4515c178fb4b76c00ffd_22.png)

## 投放初期表现与优化 🔍

![](img/1d373233fafd4515c178fb4b76c00ffd_24.png)

在起号阶段，计划初始的“千次展现成本”可能较高。这是因为系统前期需要为你推送高质量人群，否则新账户难以转化。当计划开始产生成交后，千次展现成本会逐渐下降。

![](img/1d373233fafd4515c178fb4b76c00ffd_26.png)

因此，我们需要做的是筛选出优质计划，并及时关停效果不佳的计划。

![](img/1d373233fafd4515c178fb4b76c00ffd_28.png)

那么，如何判断一个计划是否为好计划呢？以下是评估标准：
*   **最低标准**：计划在半小时内能消耗掉相当于出价金额的预算。
*   若这半小时内有转化，则让该计划继续运行，以获取更多成交。
*   若计划没有转化，但千次展现成本在正常范围内，可让其继续多消耗10%至20%的预算，观察是否能出单。

![](img/1d373233fafd4515c178fb4b76c00ffd_30.png)

请记住，计划的第一笔订单非常关键。一旦产生首单，后续投放会顺畅很多。这也是为什么强调前期需要为账户“建立转化模型”。有了模型，后续的探索就会容易得多。

## 成本控制与计划调整 ⚖️

![](img/1d373233fafd4515c178fb4b76c00ffd_32.png)

在投放过程中，如果计划转化成本过高，该怎么办？是否需要压价？

![](img/1d373233fafd4515c178fb4b76c00ffd_34.png)

操作前需确保：计划有稳定的消耗和流量速度，同时转化成本低于出价，这样才有压价空间。否则，贸然压价可能导致计划失效。

![](img/1d373233fafd4515c178fb4b76c00ffd_36.png)

如果高出价计划跑出的投资回报率无法接受，其目的仅为快速获取一波流量，那么可以直接关停该计划。

![](img/1d373233fafd4515c178fb4b76c00ffd_38.png)

## 计划模式演进 🔄

![](img/1d373233fafd4515c178fb4b76c00ffd_40.png)

最后，在起号阶段，前期应以“自定义”计划为主。当你跑出一定的数据模型后，例如自定义计划的投资回报率能接近盈亏平衡线，再去尝试“托管”计划，效果会更好。

![](img/1d373233fafd4515c178fb4b76c00ffd_42.png)
