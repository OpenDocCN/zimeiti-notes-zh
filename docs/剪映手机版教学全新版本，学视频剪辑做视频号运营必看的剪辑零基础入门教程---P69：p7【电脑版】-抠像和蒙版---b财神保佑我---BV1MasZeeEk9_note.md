# 剪映PC版教程：P7：抠像与蒙版

![](img/694ce0bfb1d7917636c6929497cf0a82_0.png)

![](img/694ce0bfb1d7917636c6929497cf0a82_0.png)

在本节课中，我们将学习剪映中两个非常有趣且强大的工具：**抠像**和**蒙版**。我们将通过制作一系列富有创意和视觉冲击力的视频效果，来掌握这两个工具的核心用法。

首先，让我们预览一下本节课将要制作的效果。视频中包含了几个看似不可思议的场景：一个人从电视机里爬出、一个人无实物托举锤子、以及人物在天空中驾驶汽车等。这些效果看似复杂，但在剪映中，只需运用**抠像**和**蒙版**两个工具，就能轻松实现。

## 抠像功能详解

上一节我们概述了本节课的目标，本节中我们来看看第一个核心工具：**抠像**。抠像，顾名思义，就是将画面中的主体（如人物）从背景中分离出来。

剪映提供了三种主要的抠像方法，适用于不同的场景。

以下是三种抠像方法的具体介绍：

![](img/694ce0bfb1d7917636c6929497cf0a82_2.png)

*   **色度抠像**：这是最快捷的方法，适用于主体处于纯色背景（通常是绿色或蓝色幕布）的情况。其原理是吸取并移除指定的背景颜色。
    *   **操作公式**：`选中素材 -> 进入“抠像”面板 -> 选择“色度抠像” -> 用吸管工具吸取背景色 -> 调整“强度”和“阴影”参数`
*   **自定义抠像**：当背景复杂、非纯色时，可以使用此功能。它允许用户使用智能画笔手动描绘需要保留的主体部分。
    *   **操作代码**（描述性）：
        1.  选择“自定义抠像”。
        2.  使用“智能画笔”在主体上涂抹。
        3.  如有误选，使用“智能橡皮擦”或“橡皮擦”修正。
        4.  点击“应用效果”。
*   **智能抠像**：此功能由软件自动识别并抠出画面中的人物。**请注意**：它主要针对人物图像，对于动物、物体等非人物主体的识别效果可能不佳。

通过以上三种方法，我们可以将任何主体从原背景中分离，并放置到新的场景中，这是合成创意视频的基础。

## 蒙版功能详解

学会了如何抠出主体后，我们来看看如何将不同的画面自然地融合在一起，这就需要用到**蒙版**功能。蒙版就像一块带有形状的挡板，可以控制图层哪些部分显示、哪些部分隐藏。

剪映提供了多种形状的蒙版，如线性、镜面、圆形、矩形、爱心和星形等，以满足不同的合成需求。

以下是蒙版的核心操作与参数：

*   **基础形状**：选择线性、圆形、矩形等不同形状的蒙版。
*   **位置与大小**：可以拖动蒙版控制点或输入数值，调整蒙版的位置和覆盖范围。
*   **旋转**：旋转蒙版的角度。
*   **羽化**：**这是让合成效果自然的关键**。增加羽化值可以使蒙版边缘产生柔和的渐变过渡，避免生硬的分界线。
*   **圆角**（矩形蒙版特有）：可以调整矩形四个角的圆润程度。

例如，使用**线性蒙版**可以将两个视频的天空部分进行替换；使用**圆形蒙版**可以实现局部聚焦或画中画效果。蒙版与关键帧结合，还能制作出复杂的动态遮罩动画。

## 综合案例实战

前面我们分别学习了抠像和蒙版的核心操作，本节我们将综合运用这两个工具，一步步还原课程开始时看到的创意效果。

我们将制作四个小片段：人物出电视、隔空控锤、空中驾车和人物微缩。以下是核心步骤的梳理：

1.  **人物出电视**：
    *   对绿幕人物素材使用**色度抠像**，去除绿色背景。
    *   将抠好的人物拖到电视机画面上方。
    *   使用**矩形蒙版**处理人物底部，使其与电视机屏幕边缘融合，避免穿帮。
    *   最后在**调节**功能中，降低画面中残留绿色的饱和度，使合成更干净。

2.  **隔空控锤**：
    *   对悬挂的锤子素材使用**自定义抠像**，仔细描绘锤子主体。
    *   将抠出的锤子图层放在人物手部动作的画面上。
    *   为锤子图层添加**位置关键帧**，使其运动轨迹与人物抬手、放下的动作同步，形成“隔空操控”的错觉。

3.  **空中驾车**：
    *   对车内绿幕素材使用**自定义抠像**结合**色度抠像**，确保人物和车内饰被完整抠出。
    *   同样通过**调节**功能去除残留的绿色杂边。
    *   将抠好的画面叠加到天空飞行的背景视频上，即可合成出汽车在空中行驶的效果。

4.  **人物微缩**：
    *   对行走的人物素材使用**自定义抠像**。
    *   将抠出的人物缩小，放置到桌面场景的图片上，调整位置和大小，形成“小人国”的视觉效果。

## 课程总结

本节课中，我们一起学习了剪映中**抠像**与**蒙版**两大核心合成工具。

*   **抠像**负责将主体从原背景中分离，主要有色度抠像（用于纯色背景）、自定义抠像（用于复杂背景）和智能抠像（用于人物）三种方法。
*   **蒙版**负责控制图层的显示与隐藏区域，并通过羽化等功能让合成效果更加自然流畅。

![](img/694ce0bfb1d7917636c6929497cf0a82_4.png)

通过将这两个工具结合使用，并辅以关键帧动画和调色技巧，我们就能打破现实限制，创作出从电视机中爬出、隔空取物、空中驾驶等各种充满想象力的视频效果。掌握它们是进阶视频剪辑、进行创意合成的必经之路。请务必多加练习，尝试用它们创作你自己的作品。
