# 亚马逊图片搜索新功能详解：P1：五大功能助力精准选品

在本节课中，我们将学习亚马逊近期推出的五项图片搜索新功能。这些功能旨在提升搜索的直观性和精准度，帮助卖家更高效地浏览和定位产品，同时也对卖家的图片和视频素材质量提出了更高要求。

上一节我们介绍了课程主题，本节中我们将逐一详细解析这五项新功能。

以下是五项核心图片搜索功能的详细介绍：

1.  **搜索框图片建议**
    当卖家在搜索框输入关键词（如 `Mal shirt`）时，平台会显示相关的图片建议。这些建议包含了产品的**颜色**、**款式**、**材质**等多种属性。卖家可以根据自身需求，直接点击图片浏览相关产品。

2.  **在图片搜索中添加关键词**
    当卖家上传一张产品图片（例如三人沙发）进行搜索时，可以直接通过输入额外的**关键词**来缩小搜索范围，使结果更加精确。

3.  **类似产品推荐**
    当卖家搜索某一产品（如连衣裙）时，可以点击“类似推荐”功能。系统会基于搜索词，推荐款式类似但细节（如**裙子长度**或**袖子设计**）不同的产品，帮助卖家找到更心仪的款式。

![](img/90e1871693b7402a5d80a6fd80c60449_1.png)

4.  **搜索结果中的视频展示**
    卖家现在可以在**家居用品**、**电器**、**玩具**、**电子产品**等类目的搜索结果中，直接观看产品视频。这能大幅提高产品的**点击率**和**转化率**。

![](img/90e1871693b7402a5d80a6fd80c60449_2.png)

5.  **圈选搜索**
    买家在浏览某张图片时，可以直接**圈定图片中的特定产品区域**进行二次检索。此功能能帮助买家精准定位其更感兴趣的具体产品。

这些功能的更新，无疑对产品主图、场景图及视频的内容与质量提出了更高要求。它们既为卖家带来了更丰富的展示机会和流量入口，也带来了优化视觉素材以适应新搜索逻辑的挑战。
