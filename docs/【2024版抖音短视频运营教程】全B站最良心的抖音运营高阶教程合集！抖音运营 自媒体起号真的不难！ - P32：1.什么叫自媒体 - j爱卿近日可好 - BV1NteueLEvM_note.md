# 抖音短视频运营教程：P32：1. 什么叫自媒体 🎬

在本节课中，我们将要学习自媒体的核心概念、课程的整体结构，并深入理解“自媒体”一词的通俗定义。

课程内容主要分为四个板块。以下是这四个板块的简要介绍：

1.  **自媒体现状与前景**：分析行业的当前情况与未来潜力，帮助判断是否值得投入。
2.  **零基础打造高收益账号**：解决视频零播放、无收益等问题，指导平台、领域与渠道的选择。
3.  **亿万流量的多种变现玩法**：除了播放量收益，介绍更多适合新手的变现方法。
4.  **自媒体变现的正确打开方式**：为决定投入的同学提供账号规划与系统性赚钱的路径。

上一节我们介绍了课程的整体框架，本节中我们来看看第一个核心问题：到底什么是自媒体？

那么，首先来了解一下到底什么是自媒体。

用最通俗的话来解释：**自媒体**就是个人或小团队利用互联网平台，创建并发布内容，从而吸引关注、产生影响或获得收益的行为。

![](img/ecf369dded20d8cb1d313ba22f33ffbf_5.png)

具体操作可以概括为一个简单的流程：
1.  在手机或电脑上，通过百度等搜索引擎查找“自媒体平台”（如抖音、快手、B站、微信公众号等）。
2.  在你选择的平台上，注册一个属于你自己的账号。
3.  在你的账号上，发布你自己创作的文章或视频。
4.  当有人观看你的内容时，你就可以依靠播放量等因素来结算收益。

![](img/ecf369dded20d8cb1d313ba22f33ffbf_7.png)

这个过程，就叫做做自媒体。其核心公式可以简化为：

![](img/ecf369dded20d8cb1d313ba22f33ffbf_9.png)

**自媒体 = 个人账号 + 自主创作 + 平台发布 + 获取流量（关注/收益）**
