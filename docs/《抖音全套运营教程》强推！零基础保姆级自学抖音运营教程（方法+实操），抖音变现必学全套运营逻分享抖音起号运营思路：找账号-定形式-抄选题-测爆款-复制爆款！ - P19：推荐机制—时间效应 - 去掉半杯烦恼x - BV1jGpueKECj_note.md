# 抖音全套运营教程：P19：推荐机制—时间效应 🕰️

在本节课中，我们将要学习抖音推荐机制中的第三个重要概念——**时间效应**。上一节我们介绍了基础推荐和叠加推荐，本节中我们来看看视频发布后，其流量可能随时间推移而变化的奇妙现象。

![](img/21f548ec55cb643410432c5f7aa737f6_1.png)

## 什么是时间效应？

时间效应是指，一个视频在发布后，其流量可能并非在第一时间达到峰值，而是在未来某个时间点（例如一周或一个月后）突然获得大量推荐和播放量增长的现象。

## 时间效应如何运作？

为了理解时间效应，我们需要了解抖音的视频库机制。

![](img/21f548ec55cb643410432c5f7aa737f6_1.png)

抖音平台拥有一个庞大的**视频库**，所有用户发布的视频都会存储在其中。当你的视频首次发布时，如果内容未蹭到热点或质量不够突出，它可能只是安静地存放在视频库中。

然而，当未来出现新的热点事件时，抖音的算法会重新扫描视频库，寻找与当前热点相匹配的“库存”视频，并将其进行**二次推荐**，重新走一遍推荐流程。

以下是时间效应发生的一个典型场景：
*   你发布了一个关于明星A的视频，但当时并未获得太多流量。
*   一段时间后，明星A因为某事件（如发布新作品、出现绯闻）热度飙升。
*   此时，抖音算法从视频库中找到了你之前发布的、与明星A相关的视频，并将其重新推荐给对此热点感兴趣的用户，从而使你的旧视频播放量大幅上涨。

这个概念最初源自贴吧的“挖坟”机制，即通过算法发现热点后，重新挖掘并推荐历史内容。

## 时间效应的实践意义

如果你按照教程认真制作了视频，即使初期数据不佳，也无需气馁。你的视频始终存在于抖音的视频库中，未来仍有很大机会借助热点事件被重新推荐。你可以通过积极互动（如回复评论）来增加视频被再次推荐的几率。

## 与快手机制的对比

需要特别强调的是，时间效应是**抖音平台特有**的推荐机制。

以下是抖音与另一大短视频平台快手在推荐机制上的核心区别：
*   **抖音**：采用**时间效应**机制。旧视频有机会因热点被重新挖掘和推荐。
*   **快手**：采用**择新去旧**机制。平台主要推荐新发布的视频，旧视频（如发布一周或一个月后）通常不会再获得推荐，无论是否有新的热点关联。
