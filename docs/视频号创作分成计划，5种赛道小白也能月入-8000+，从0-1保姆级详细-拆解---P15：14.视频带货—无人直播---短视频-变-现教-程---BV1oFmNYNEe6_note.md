## 核心玩法：视频带货 + 无人直播
刚刚提到的创作分成计划仅是“开胃小菜”，适合兼职操作。而“翻身暴富玩法”才是今天的核心与重点。

**翻身暴富玩法**的本质是：**视频带货** 加上 **无人直播**。
用公式表示即：
**核心收益 = 视频带货收益 + 无人直播收益**

以下是该玩法的两个核心组成部分：
1.  **视频带货**：通过制作短视频，展示和推荐商品，引导用户购买并赚取佣金。
2.  **无人直播**：一种自动化或半自动化的直播形式，可以24小时不间断展示商品，实现被动收入。
