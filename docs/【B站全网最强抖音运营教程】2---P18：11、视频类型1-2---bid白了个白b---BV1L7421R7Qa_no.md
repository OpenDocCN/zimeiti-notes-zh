# 抖音运营高阶教程：P18：11、视频类型1-2

在本节课中，我们将要学习抖音账号运营中一个至关重要的中间环节——**人设**的建立。我们将详细区分强人设与弱人设，并介绍一种核心的视频制作形式：口播。

上一节我们解决了文案问题，本节中我们来看看内容创作的下一个关键环节。许多教程会直接谈论涨粉与变现，但在这两者之间，还有一个必须建立的桥梁，那就是**人设**。建立人设的目的并非为了炫耀，而是为了在众多同类创作者中脱颖而出。例如，有十个人都做装修内容，如果其中一位创作者（比如小王同学）树立了一个幽默搞笑的人设，观众就更容易记住他，而忽略其他九位。当观众因为喜欢你这个人而关注你时，涨粉和后续的变现便水到渠成。

这就是人设的作用。接下来，我们来学习这个知识点，请边听边记。

## 🧑‍🤝‍🧑 人设的两种类型：强与弱

人设主要分为两种：**强人设**与**弱人设**。区分清楚这两种人设，你就能快速判断任何视频的类型。

首先，**强人设**是指以创作者**个人**为切入点的内容。它必须与创作者自身密切相关。

以下是强人设常见的切入点：

![](img/6d835811a13570fde1fd19e6614021ce_1.png)

*   **个人经验与专业**：分享你所在行业的经验。例如：装修、办公设备、房产、汽车、餐饮、婚介、法律咨询、电商等领域。
*   **个人兴趣与生活**：分享你的兴趣爱好与日常生活。例如：美食、旅行、手工、宠物、星座占卜、美妆、护肤、服装穿搭、游戏、摄影等。

**公式：强人设 = 以“我”（创作者本人）为中心 + 分享“我的”经验/生活/爱好**

![](img/6d835811a13570fde1fd19e6614021ce_3.png)

这类内容大部分需要**真人出镜**，因为它与“你”这个人直接相关。

与之相对的是**弱人设**。弱人设的内容与创作者**本人关系不大**，核心目的是运营好账号本身。

以下是弱人设的例子：

*   **情感语录账号**：发布情感文案配音乐，无真人出镜或配音。
*   **影视解说账号**：虽然有人声解说电影，但观众关注的是电影内容和解说，而非解说者本人。

**公式：弱人设 = 以“内容”（主题/素材）为中心 + 创作者个人属性弱化**

只要能区分强弱人设，你就能更清晰地规划自己的账号方向。

## 🎙️ 核心内容形式：口播

明确了人设，我们来看具体的内容制作。只要内容与你个人相关，都可以作为视频素材。我们首先讲解一种非常高效且通用的形式：**讲故事**。

讲故事可以分享客户故事、从业经历或人生感悟。故事本身不要求100%真实，可以从文案平台寻找有趣的故事进行分享。而讲故事最直接的方式，就是通过**口播**。

口播，即用嘴巴播讲内容。它有很多分类：

*   养生口播
*   情感口播
*   育儿口播
*   知识类口播（如房产、汽车、金融财经、装修等）

**代码示例（口播视频核心结构）：**
```
视频开始 -> 吸引人的开头（提出问题/抛出观点） -> 核心内容讲述（故事或知识点） -> 结尾总结或引导互动 -> 视频结束
```

**讲故事类口播**的目的是让观众觉得内容有趣、吸引人。
**教知识类口播**的目的是让观众在几十秒内获得“收获感”，觉得学到了东西。

无论你从事法律、保险、金融、装修还是汽车领域，都可以采用口播形式。它的优势在于制作简单：一部手机、一个支架，准备好文案即可开拍。

关于口播，有两个常见问题：
1.  **口才不好能做吗？** 可以。口播是**提前准备好稿子**再录制，并非临场发挥。
2.  **普通话不标准能做吗？** 可以。如果你的目标客户是同城或有地域属性，使用方言可能更亲切、更有趣。观众可以通过字幕理解内容。

本节课中我们一起学习了抖音账号运营中“人设”的重要性，清晰地区分了**强人设**与**弱人设**，并深入介绍了**口播**这种核心视频形式。记住，建立鲜明的人设是连接内容与粉丝的关键，而口播是实践强人设、输出个人化内容的有效工具。在接下来的课程中，我们将继续学习其他内容类型和拍摄技巧。
