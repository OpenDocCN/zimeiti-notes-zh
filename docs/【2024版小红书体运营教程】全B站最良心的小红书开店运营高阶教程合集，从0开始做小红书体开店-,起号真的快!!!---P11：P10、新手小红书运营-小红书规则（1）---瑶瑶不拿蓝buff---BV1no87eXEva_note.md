# 小红书运营教程：P10：小红书规则（1） 📚

在本节课中，我们将要学习小红书平台的核心规则。了解这些规则是成功运营账号、避免违规和获得推荐流量的基础。前面我们已经探讨了平台本质、市场分析和起号方法，本节将正式进入核心内容，详细解析小红书的推荐规则。

## 什么是小红书规则？ 🤔

![](img/26e08962d6fbe825388838d1acdf901d_1.png)

![](img/26e08962d6fbe825388838d1acdf901d_3.png)

小红书的规则是平台为维护内容生态、管理用户行为而设立的一系列准则。它决定了你的内容能否被推荐、账号是否安全。如果不了解这些规则，即使账号已经建立，也可能因违规而无法获得流量，甚至被封禁。

![](img/26e08962d6fbe825388838d1acdf901d_5.png)

## 推荐规则的四大要点 📋

![](img/26e08962d6fbe825388838d1acdf901d_7.png)

![](img/26e08962d6fbe825388838d1acdf901d_9.png)

小红书的推荐规则内容非常丰富，我将其归纳为四个核心要点。这些要点看似不多，但每个都包含大量细节，需要我们仔细理解和掌握。

![](img/26e08962d6fbe825388838d1acdf901d_11.png)

![](img/26e08962d6fbe825388838d1acdf901d_13.png)

以下是推荐规则的四个组成部分：

![](img/26e08962d6fbe825388838d1acdf901d_15.png)

1.  **用户行为**
2.  **自序商品**
3.  **图文阴影**

![](img/26e08962d6fbe825388838d1acdf901d_17.png)

接下来，我们将逐一进行解析。

### 1. 用户行为 👤

用户行为规则主要规范账号在平台上的日常操作，特别是涉及营销和引流的行为。理解哪些能做、哪些不能做至关重要。

以下是用户行为规则中**严禁**的内容：

*   **微信号**：不得在笔记、评论、私信中直接留下微信号。
*   **二维码**：禁止发布任何形式的引流二维码。
*   **代购水印**：笔记图片上不能带有代购联系方式的水印。
*   **某某店铺**：不能直接宣传或提及自己的店铺名称进行导流。

**核心逻辑**：小红书严格禁止任何将平台公域流量直接导向其他私域（如微信、其他店铺）的赤裸裸的营销行为。平台希望交易和互动留在站内。

**应对技巧**：虽然直接引流被禁止，但可以通过一些间接方式操作。例如，你可以使用一个小号，在主号的笔记评论区以“官方”口吻置顶评论，并@你的小号，将用户引导至小号，再由小号进行下一步引导。这需要一套账号矩阵配合。

以下是用户行为规则中**允许**的替代方式：

![](img/26e08962d6fbe825388838d1acdf901d_19.png)

*   **个人邮箱**：可以留下邮箱地址。
*   **微博**：可以提及或引导至微博（但需注意尺度）。
*   **谐音/话术引导**：可以使用谐音字、特定话术来间接暗示联系方式，但需巧妙设计，且不宜频繁操作，每天最多给1-2个人发送，否则仍有风险。

### 2. 自序商品 💊

“自序商品”规则主要针对内容涉及的领域进行限制，目的是防范风险。

以下是自序商品规则中**必须避免**的领域：

*   **医疗建议**：不能提供具体的医疗诊断、治疗建议或推荐医院。例如，“去XX医院看这个病很好”这类内容是不允许的。
*   **投资建议**：严禁提供股票、基金、房产等具体的投资建议或涨跌预测。例如，“这只股票会涨，赶紧买”、“那个楼盘肯定升值”等内容均属违规。

![](img/26e08962d6fbe825388838d1acdf901d_21.png)

**核心逻辑**：这些领域专业性强、风险高，容易引发纠纷或诈骗，平台对此类内容的管控极其严格。

![](img/26e08962d6fbe825388838d1acdf901d_23.png)

![](img/26e08962d6fbe825388838d1acdf901d_25.png)

### 3. 图文阴影 📝

![](img/26e08962d6fbe825388838d1acdf901d_27.png)

“图文阴影”规则主要规范笔记内容本身的创作与发布。

以下是图文阴影规则中**需要注意**的要点：

*   **禁止诱导互动**：不得在笔记中通过利益诱导用户关注、点赞、评论。例如，“关注我抽奖送礼物”、“点赞领取资料”等都属于违规诱导行为。
*   **设备与账号管理**：在电脑端操作时，**一台设备同时登录的账号最好不要超过3个**。如果使用模拟器（如雷电模拟器）多开，更需注意IP和数据隔离，超过3个账号容易被系统检测为异常。
*   **内容原创度**：笔记内容的原创度越高，初期获得的基础流量推荐就越好。直接搬运他人的图片和文案，账号权重会很低，甚至被判违规。如果是二次创作（如使用他人图片但配自己文案），原创度可能只有50%，虽有一定流量，但效果远不如高原创内容。

![](img/26e08962d6fbe825388838d1acdf901d_29.png)

**核心逻辑**：平台鼓励**原创、真实、非功利性**的分享。通过高质量内容自然吸引用户，而非通过技术手段或利益诱导。

## 被推荐的核心逻辑 🧠

了解规则是“避坑”，理解推荐逻辑则是“加油”。系统如何判断内容好坏并分配流量？

**核心公式**：`内容质量 + 用户互动 + 账号健康度 = 推荐权重`

1.  **系统判定**：当你发布笔记后，小红书系统会基于规则对内容进行审核，并分析其标签、关键词和质量。
2.  **初始流量池**：审核通过后，系统会将笔记推送给一小部分可能感兴趣的用户（初始流量池）。
3.  **数据反馈**：系统密切关注这批用户的**互动数据**，包括：点击率、阅读完成率、点赞、收藏、评论、转发。
4.  **流量分配**：如果数据反馈良好（高于同类内容平均水平），系统会判定笔记为“优质内容”，并将其推送给更大的流量池，形成**阶梯式推荐**。反之，则停止推荐。
5.  **账号权重累积**：持续产出受推荐的内容，会提升你的**账号权重**。高权重账号发布新笔记时，获得的初始流量池会更大，形成良性循环。

**简单来说**：规则是门槛，决定了你能不能上场；而推荐逻辑是比赛规则，决定了你能跑多快、走多远。你需要用合规的内容（遵守规则）去争取好的用户数据（理解逻辑），从而获得更多流量。
