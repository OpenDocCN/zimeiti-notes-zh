# 视频号运营教程：P23：3. 如何给短视频配音 🎙️

在本节课中，我们将学习如何为短视频进行配音。当原视频的配音效果不佳时，我们可以通过替换配音来提升视频质量。本节将介绍配音工具的选择与使用方法，并演示如何将新配音应用到视频中。

## 配音工具的选择 🛠️

上一节我们介绍了视频剪辑的基本概念，本节中我们来看看如何获取和制作配音。配音工具有很多，例如“配音神器”就是一个功能强大的选择。

除了配音功能，这类工具通常还集成了其他实用功能。

以下是“配音神器”小程序提供的主要功能列表：
*   **文案提取**：从视频中提取文字内容。
*   **视频去水印**：移除视频中的水印标识。
*   **修改视频MD5值**：通过改变视频文件的唯一标识码，辅助视频去重。

**注意**：仅修改MD5值并不足以应对平台查重，后续仍需进行视频混剪操作。关于混剪的具体方法，我们将在下节课详细讲解。

## 如何搜索与使用工具 🔍

无论是去水印、修改MD5还是配音，我们都可以在微信中搜索对应的小程序来完成任务。

![](img/dd3dfa65f1409901cefdb0669a407c34_1.png)

以下是搜索和使用工具的步骤：
1.  打开微信，点击顶部搜索框。
2.  输入关键词，如“去水印”、“配音”或“修改MD5”。
3.  在搜索结果中选择“小程序”分类，即可找到大量相关工具。

![](img/dd3dfa65f1409901cefdb0669a407c34_1.png)

![](img/dd3dfa65f1409901cefdb0669a407c34_3.png)

推荐使用“配音神器”或“青斗”小程序。若某个工具暂时无法使用，可随时更换其他同类工具。

![](img/dd3dfa65f1409901cefdb0669a407c34_3.png)

“配音神器”的配音功能需要消耗金币（收费），但其文案提取、去水印等功能是免费的。用户可以通过每日签到领取金币。合成一段配音通常消耗3个金币。

其会员开通选项如下：
*   **包月会员**：29元/月
*   **终身会员**：99元

## 制作配音的步骤 🎬

了解工具后，我们开始学习制作配音的具体流程。

以下是使用“配音神器”制作配音的详细步骤：
1.  打开“配音神器”小程序，点击配音功能。
2.  清除默认文字，输入你准备好的文案脚本。
3.  在声音库中选择喜欢的音色（如女声、男声等），可以点击试听。
    *   `试听代码示例（伪代码）：audio.play_preview(voice_id='female_01')`
4.  选择好声音后，点击“合成配音”。
5.  合成完成后，选择导出方式。通常可选择“保存配音视频到相册”或“保存字幕视频到相册”，两者生成的视频样式不同。

![](img/dd3dfa65f1409901cefdb0669a407c34_5.png)

导出过程中请勿退出小程序。导出成功后，配音文件将保存至手机相册。

![](img/dd3dfa65f1409901cefdb0669a407c34_7.png)

![](img/dd3dfa65f1409901cefdb0669a407c34_5.png)

## 在剪映中替换视频原声 ✂️

获得配音文件后，下一步就是在剪辑软件中替换掉视频的原声。

操作步骤如下：
1.  打开“剪映”APP，导入需要处理的视频素材。
2.  选中视频轨道，点击“关闭原声”。
3.  点击“添加音频” -> “提取音乐”。
4.  从手机相册中选择刚刚导出的配音视频文件。
5.  导入后，新的配音就会替换掉视频原有的音频轨道。

![](img/dd3dfa65f1409901cefdb0669a407c34_7.png)

**公式表示替换过程**：`最终视频 = 原视频画面 + 新配音音频`

通过以上操作，即可完成视频配音的替换。本节演示仅说明方法，实际应用中请确保配音内容与视频画面相匹配。
