## 渠道一：微软Edge浏览器（Copilot）💻

首先推荐的是微软Edge浏览器内置的Copilot功能。它是微软基于ChatGPT技术打造、集成在Windows系统中的AI助手，其智能程度在一定程度上与4.0模型相匹配，是普通用户可便捷使用的渠道之一。

使用该功能有两个前提条件：
1.  登录微软账号（可使用Outlook邮箱注册）。
2.  需要“梯子”（VPN）才能访问该服务。

Copilot通常集成在浏览器右侧工具栏。更新至Win11最新版本后，也可能在系统任务栏出现。它提供“更有创造力”、“平衡”和“更精确”三种对话模式，以适应不同需求。

以下是使用Copilot请求撰写短视频脚本的示例：
```
请帮我写一个关于装修的短视频脚本，时长控制在3分钟左右。
```
Copilot的优势在于，它可以分析你当前打开的网页内容。但其功能也存在一些限制。

上一节我们介绍了Edge Copilot的基本使用，本节中我们来看看它的主要优缺点。

**优点：**
*   集成在系统中，使用方便。
*   基于强大的GPT-4模型，能力不俗。

**缺点：**
1.  **精准度稍逊**：相较于官方ChatGPT 4.0，Copilot是经过调教的版本，在某些方面存在限制，回答可能不够精准。
2.  **连续对话能力受限**：在当前对话中持续提问多次后，可能被要求开启新对话窗口。
3.  **记忆功能有所缺失**：在历史对话记录的读取和基于上下文优化回答方面，能力不如官方版本完整。

对于大部分运营工作而言，Copilot的功能已足够使用。如果你的Edge浏览器没有Copilot图标，可以尝试下载最新的Edge Dev（开发者）测试版，其中必定包含此功能。

## 渠道二：第三方集成平台 🌐

我们要推荐的第二种渠道是一些第三方开发者搭建的平台。这些平台通过调用ChatGPT的API，让国内用户也能访问和使用，其中部分支持GPT-4模型。

以下是几个经过测试可用的平台介绍：

![](img/9dba1032fc1ac9e73d6e68eb22754ab2_5.png)

![](img/9dba1032fc1ac9e73d6e68eb22754ab2_7.png)

**1. 灵感AI**
*   **特点**：集成了多种模型，包括GPT-3.5、GPT-4（含联网版）、Claude等，以及国内主流大模型。
*   **费用**：提供新用户7天内免费无限次使用GPT-4的额度（具体次数限制需以平台说明为准）。
*   **缺点**：生成速度可能较慢。
*   **访问要求**：无需“梯子”。

![](img/9dba1032fc1ac9e73d6e68eb22754ab2_9.png)

![](img/9dba1032fc1ac9e73d6e68eb22754ab2_11.png)

![](img/9dba1032fc1ac9e73d6e68eb22754ab2_13.png)

**2. WeeChat**
*   **特点**：支持GPT-3.5和GPT-4，响应速度快，具备语音回答功能。
*   **费用**：注册赠送一定额度的Token（计算单位），可用于GPT-3.5。使用GPT-4可能需要付费。
*   **缺点**：免费额度有限，连续对话能力一般。
*   **访问要求**：需要“梯子”。

![](img/9dba1032fc1ac9e73d6e68eb22754ab2_15.png)

![](img/9dba1032fc1ac9e73d6e68eb22754ab2_17.png)

![](img/9dba1032fc1ac9e73d6e68eb22754ab2_19.png)

**3. ChatGPT账号共享平台**
*   **特点**：一个共享ChatGPT账号的网站。绿色代表账号空闲，橙色代表近期使用过，红色代表正在使用。带有“金光”标识的通常是付费的Plus账号（支持GPT-4）。
*   **使用**：可以尝试使用这些共享的账号直接登录官方ChatGPT界面。GPT-3.5账号通常可用，GPT-4 Plus账号因使用人数多，可能遇到“使用上限”提示。
*   **优点**：可直接体验官方界面和功能。
*   **缺点**：账号状态不稳定，需要“梯子”。

## 渠道三：国内AI大模型 🇨🇳

第三个推荐渠道是国内公司开发的AI大模型。它们无需“梯子”即可访问，虽然并非ChatGPT，但在许多基础任务上也能提供有效帮助。

以下是几个主要的国内AI模型：
*   **百度文心一言**：百度推出的AI对话模型，有免费版和付费的“4.0”版本。在文案生成、灵感激发等运营辅助工作中表现合格。
*   **字节豆包**：字节跳动推出的AI助手。同样可以完成脚本撰写、内容规划等任务。
*   **阿里通义千问**：阿里巴巴的AI模型。
*   **天工AI**：昆仑万维推出的AI模型。

你可以根据使用习惯和具体需求，选择其中一个进行深度使用和“调教”，使其更适应你的工作流程。

![](img/9dba1032fc1ac9e73d6e68eb22754ab2_26.png)

![](img/9dba1032fc1ac9e73d6e68eb22754ab2_28.png)

为了直观对比，我们可以用同一指令测试不同模型：
```
指令：请帮我写一个关于装修的短视频脚本文案，要求镜头丰富，语言吸引人。
```
*   **国内模型（如文心一言）**：通常会提供结构清晰的脚本框架，包括景别、画面内容、台词、时长备注等。
*   **ChatGPT 4.0**：生成的脚本在场景设计、叙事感和语言感染力上往往更胜一筹，更具“电影感”。

选择工具时，应本着提升效率的原则，哪个好用、哪个能更好地辅助工作，就选择哪个。

## 课程总结

本节课我们一起学习了体验和使用ChatGPT 4.0的三种主要渠道：
1.  **微软Edge Copilot**：系统集成，方便但略有功能限制。
2.  **第三方平台**：如灵感AI、WeeChat等，是折中体验GPT-4的途径，需注意速度、费用和访问条件。
3.  **国内AI大模型**：如文心一言、豆包等，访问无障碍，适合作为日常运营的辅助工具。
