# 跨境电商亚马逊运营教程：P35：创建新商品：报价和图片

![](img/03bcd9cfc94216c53318ac1d6fdb2f57_0.png)

在本节课中，我们将继续学习在亚马逊后台创建新商品的流程，重点讲解“报价”和“图片”两个关键部分的填写要点。这是商品成功上架并吸引买家的核心环节。

上一节我们介绍了商品的基本信息填写，本节中我们来看看如何设置商品的价格、库存以及上传商品图片。

## 报价设置

报价部分决定了商品的销售价格、库存状态和发货时效，直接影响买家的购买决策。

**处理时间**：指买家下单后，卖家需要多少天来准备并发货。通常以天数计算，例如 `处理时间 = 2天`。

**补货日期**：根据您的库存情况预估商品可以再次上架销售的日期。请根据实际库存计算。

![](img/03bcd9cfc94216c53318ac1d6fdb2f57_2.png)

**礼品包装与信息**：此功能主要面向**专业销售计划**的卖家。如果您是个人卖家，无需选择此项，选择后也无实际意义。

![](img/03bcd9cfc94216c53318ac1d6fdb2f57_3.png)

**订单商品最大数量**：限制单个买家在您店铺购买此商品的最大件数。例如，可设置为 `最大数量 = 5`。

**开始销售时间**：设置商品在亚马逊上正式发布并开始销售的具体时间。

![](img/03bcd9cfc94216c53318ac1d6fdb2f57_5.png)

## 合规信息

对于大多数普通商品（如本节示例的服装），若无特殊成分（如电池、液体等航空禁运品），此部分通常无需填写。主要针对电子产品等有特殊运输要求的品类。

![](img/03bcd9cfc94216c53318ac1d6fdb2f57_7.png)

## 商品图片上传

![](img/03bcd9cfc94216c53318ac1d6fdb2f57_9.png)

商品图片是吸引买家点击和购买的第一要素。高质量的图片至关重要。

以下是上传主图的步骤：
1.  点击图片上传区域。
2.  从您的电脑中选择已准备好的商品主图文件。
3.  确认图片清晰、符合亚马逊要求（如纯白背景、展示商品全貌等）。

![](img/03bcd9cfc94216c53318ac1d6fdb2f57_11.png)

**关于商品变体与UPC码的补充说明**：如果您的商品有多个变体（例如，2种颜色 x 2种尺码），则总共需要4个独立的UPC码，每个变体对应一个唯一的UPC。

![](img/03bcd9cfc94216c53318ac1d6fdb2f57_13.png)

确保所有信息填写无误后，即可保存并完成商品创建。

![](img/03bcd9cfc94216c53318ac1d6fdb2f57_15.png)

本节课中我们一起学习了创建亚马逊新商品时“报价”和“图片”部分的详细设置。重点包括处理时间、最大购买数量等报价策略，以及上传高质量主图的操作。正确填写这些信息，是商品顺利上架并获得良好展示的基础。
