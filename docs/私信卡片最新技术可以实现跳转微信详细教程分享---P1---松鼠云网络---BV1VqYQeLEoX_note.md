# 私信卡片跳转微信教程：P1：配置自动回复卡片实现微信引流

在本节课中，我们将学习如何为抖音企业号配置私信自动回复卡片，实现用户点击卡片后直接跳转至微信添加好友的功能。我们将从所需资质、配置平台到具体操作步骤进行详细说明。

## 所需资质与平台

上一节我们了解了课程目标，本节中我们来看看实现此功能需要满足哪些前提条件。

![](img/406e11980bd6622143b329e12f5dbb2a_1.png)

首先，您的抖音账号需要是企业身份。具体资质要求如下：

*   抖音企业蓝V认证号。
*   抖音普通企业号。

> **注意**：个人抖音号理论上也可以配置此功能，但效果和稳定性不如企业号，因此不建议使用。

其次，您需要一个获客平台。该平台负责对接抖音企业号的私信自动回复官方功能，确保操作的安全性与合规性。

## 配置步骤详解

了解了基本要求后，接下来我们进入核心的配置环节。以下是配置私信自动回复卡片的完整流程。

1.  **登录后台**：首先，登录您所使用的获客平台管理后台。
2.  **绑定账号**：在后台中找到“自动回复”或类似功能模块，点击“绑定企业号”，添加您要配置此功能的抖音企业号。
3.  **创建卡片**：成功绑定账号后，在该账号的操作选项中，选择“创建跳转卡片”或类似按钮。
4.  **提交审核**：根据页面指引，填写卡片信息（如欢迎语、跳转链接等）并提交。提交后等待平台审核通过即可生效。
