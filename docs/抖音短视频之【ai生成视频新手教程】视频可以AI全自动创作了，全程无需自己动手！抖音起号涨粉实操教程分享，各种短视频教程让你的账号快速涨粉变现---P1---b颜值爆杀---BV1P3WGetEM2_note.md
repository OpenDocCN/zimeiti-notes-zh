# AI短视频创作：P1：全自动AI视频创作教程 🚀

在本节课中，我们将学习如何利用AI工具实现中视频的全自动创作。这种方法无需手动拍摄或剪辑，通过组合文案生成、AI配音和智能画面匹配，即可高效地产出高质量视频内容。我们将分步拆解核心工具与操作流程，让初学者也能轻松上手。

![](img/7e195bb2b5d02df945bc898d4d7a945c_1.png)

## 平台偏好与核心思路

上一节我们介绍了课程目标，本节中我们来看看平台喜欢什么样的视频作品。平台青睐的中视频作品通常具备两个核心要素：**优秀的文案**和**高度相关且精彩的画面**。满足这两点，视频才更容易获得推荐和播放量。

基于此，我们的全自动创作思路是：利用三个AI工具分别解决文案、配音和画面问题。

以下是实现全自动创作所需的三个核心工具：

1.  **文案创作工具**：`ChatGPT`，用于生成高度原创且优质的文案脚本。
2.  **AI配音工具**：例如`配音神器`，用于将文案转化为自然的人声配音。
3.  **视频素材匹配工具**：`三三搜帧`，用于根据文案和配音，智能匹配高质量、无版权的动态视频画面。

## 第一步：使用ChatGPT生成原创文案

有了核心思路，我们首先来解决文案问题。ChatGPT是一款强大的人工智能对话模型，其文案创作能力尤为突出。它不仅能生成逻辑通顺的文本，其产出内容甚至带有情感和思维痕迹，非常适合用于创作故事性内容。

操作步骤如下：

1.  向ChatGPT提出明确的文案需求。例如：“请帮我写一个约1000字的中国历史故事。”
2.  ChatGPT会生成初步文案。如果对结果不满意，可以追加更精细的条件。例如：“请在刚才的故事中加入更多人物，并保持总字数不变。”
3.  通过多次交互和优化，最终获得一份高度原创且符合要求的视频文案。

> **核心提示**：提问越精细，得到的答案就越符合预期。你可以指定故事类型、地区、人物或情节走向。

## 第二步：使用AI工具为文案配音

获得优质文案后，下一步是将其转化为语音。这一步相对简单，我们使用专业的AI配音工具来完成。

以下是具体操作流程：

1.  打开配音工具（如配音神器），点击“制作配音”功能。
2.  将上一步从ChatGPT获得的文案完整粘贴到输入框中。
3.  在“主播选择”或“更多主播”中，挑选一个符合视频风格和语调的AI声音。
4.  试听并调整语速、语调等参数，确认无误后，保存生成的配音音频文件。

## 第三步：使用三三搜帧匹配并合成视频

最后一步，也是最关键的一步，是为配音匹配合适的画面。我们使用“三三搜帧”工具，它能根据文案语义智能搜索并推荐无版权的动态视频素材。

具体实现方法如下：

1.  打开三三搜帧工具，找到“AI合成视频”或“合成视频”功能。
2.  导入第二步中生成的配音音频文件。
3.  工具会自动分析音频台词，并为每一句话推荐多个高度吻合的动态视频画面供你选择。
4.  逐一确认或替换每个台词对应的画面，确保画面与内容高度相关。
5.  所有画面选择完毕后，点击“生成视频”，工具会自动将配音与画面合成为一个完整的视频。

**最终效果示例**：
生成的视频将拥有流畅的配音和精准匹配的精彩画面，例如讲述历史故事时，画面会随之出现相关的场景、人物动态或史料影像。
