# 拼多多运营教程：P1：推广费返还政策详解与查询指南 💰

在本节课中，我们将要学习拼多多平台关于推广费返还的最新政策，并掌握如何查询返还明细。这对于控制运营成本至关重要。

做过拼多多推广的商家都清楚，推广费用占据了运营成本的大部分。对于竞争激烈的产品，推广成本占比可能超过90%。只要订单成交，平台就会扣除相应的推广费用。

过去，对于发生仅退款甚至秒退的订单，其对应的推广费用不会返还给商家。今年大约8月底到9月初，拼多多平台在这方面做出了调整。符合特定条件的退款订单，其推广费会退还给商家。

需要注意的是，退回的费用并非以现金形式返还到商家的银行账户。而是以“推广红包”的形式发放到商家的推广账户中，默认会继续用于推广商品。这种方式对商家而言是有利的。

## 如何查询推广费返还明细

![](img/f2d80f2e26375841954f22dece4848d1_1.png)

上一节我们介绍了推广费返还政策的基本情况，本节中我们来看看具体的查询步骤。以下是详细的查询路径：

![](img/f2d80f2e26375841954f22dece4848d1_2.png)

1.  登录拼多多商家后台，找到“推广管理”模块。
2.  在店铺名称下方，点击进入“账户管理”。

![](img/f2d80f2e26375841954f22dece4848d1_1.png)

![](img/f2d80f2e26375841954f22dece4848d1_2.png)

3.  进入账户管理后，点击“财务管理”选项。在此页面，你可以看到推广花费概况，例如“今日花费：2246元”、“昨日花费：4991元”。
4.  在财务数据下方，找到“红包”栏目，例如显示“29个已完29个”。点击“已完”按钮。

点击后，页面将展示每日返还的推广红包明细。例如，某个店铺每日可获得100多元至200多元不等的推广返还红包。

![](img/f2d80f2e26375841954f22dece4848d1_4.png)

![](img/f2d80f2e26375841954f22dece4848d1_4.png)

![](img/f2d80f2e26375841954f22dece4848d1_5.png)

![](img/f2d80f2e26375841954f1_5.png)

累积下来，一个月可能有数千元的返还金额。另一个例子是，一个日推广费约1000元的店铺，每日也能获得几十元到一百多元的返还。

![](img/f2d80f2e26375841954f22dece4848d1_7.png)

![](img/f2d80f2e26375841954f22dece4848d1_7.png)

## 手机端查询方式

除了电脑端，在拼多多商家版手机App上同样可以查询。操作路径也是在推广账户的“财务管理”界面查看返还红包的明细。

这项政策改变了以往推广费有去无回的状况，将部分费用返还给商家，是一个积极的改进。它切实减轻了商家的推广压力。例如，一个店铺每天节省100元左右，长期下来就是一笔可观的成本节约。

本节课中我们一起学习了拼多多推广费返还政策的核心要点与查询方法。总结来说，平台对符合条件的退款订单推广费进行了返还，以推广红包形式发放，商家可通过后台“推广管理 > 账户管理 > 财务管理”路径查询明细。这项政策有助于降低商家的运营成本。
