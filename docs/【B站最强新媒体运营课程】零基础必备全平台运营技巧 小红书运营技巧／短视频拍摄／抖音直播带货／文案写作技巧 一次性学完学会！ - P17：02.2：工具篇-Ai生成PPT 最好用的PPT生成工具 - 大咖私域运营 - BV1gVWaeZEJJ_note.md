# 新媒体运营课程：P17：02.2：工具篇-AI生成PPT 🚀

在本节课中，我们将学习如何使用一款高效的AI工具来生成PPT。这款工具能根据你提供的主题或现有文档，自动梳理大纲并生成详细的演示文稿内容，非常适合快速制作初稿。

## 工具简介与模式

上一节我们介绍了多种新媒体运营工具，本节中我们来看看这款AI生成PPT工具。它主要提供两种工作模式：

1.  **主题生成模式**：输入一个主题，工具会结合类似ChatGPT的生成式AI能力，自动创作完整的PPT内容。
2.  **文档生成模式**：上传现有的文档（如TXT、PDF文件），AI助手会自动总结文档大纲，并基于此大纲生成PPT的详细内容。

## 编辑与优化生成结果

生成完毕后，我们可以进入编辑界面查看和调整PPT。此工具的一大优势在于，它能将文档内容智能归类到不同的大纲节点下，并生成对应的详细阐述，而不仅仅是进行格式排版。

在编辑界面右侧，提供了多种调整选项：

*   **更换模板**：工具内置了多种设计模板，可以一键切换，快速改变整体风格。
*   **编辑内容**：可以直接修改PPT中的文字、形状等元素。
*   **本地化调整**：您也可以将生成的PPT下载到本地，使用PowerPoint或Keynote进行更精细的调整。
