# 小红书电商运营：P3：小红书入驻流程与注意事项 🛒

在本节课中，我们将详细讲解如何在小红书开设店铺，包括核心概念、注册流程、店铺类型选择以及关键的注意事项。通过学习，你将能够独立完成店铺的开设，并避开常见的注册陷阱。

## 核心概念区分

在开始注册之前，需要明确两个核心概念：**专业号**与**小红书店铺**。

*   **小红书店铺**：类似于大家熟知的“抖店”，是进行商品销售的平台。
*   **专业号**：类似于抖音的“蓝V认证”，是经过平台认证的账号身份。

**关系说明**：如果你打算直接用现有小红书账号注册店铺，需要先完成该账号的认证。

## 店铺主体类型选择

上一节我们区分了核心概念，本节中我们来看看如何选择店铺的认证主体。这决定了你能开设店铺的数量和类型。

以下是三种主要的店铺主体类型：

1.  **个人店**
    *   **认证主体**：个人身份证。
    *   **开店数量**：**仅能开设1家**。
    *   **建议**：对于大多数个人创业者，建议首选个人店。目前平台对个人店管控相对宽松，注册审核快（通常当天秒过），且能避免一些职业打假风险。

2.  **个体工商户店**
    *   **认证主体**：个体工商户营业执照。
    *   **开店数量**：**最多可开设3家**。
    *   **重要规则**：一旦选择以个体工商户为主体认证，你的小红书账号认证主体即升级为企业性质。此时，**你将无法再开设个人店**。

3.  **企业店（含旗舰店、专卖店、普通企业店）**
    *   **认证主体**：公司营业执照。
    *   **建议**：**个人创业者不建议注册**。此类店铺更适合品牌方或供应链商家。个人身份注册企业店容易面临更严格的审核和潜在的职业打假风险。

**总结选择建议**：**个人操作，无品牌供应链，优先开个人店。**

## 关键注册流程与避坑指南

了解了店铺类型后，我们进入具体的注册环节。以下是注册过程中必须注意的关键步骤和常见问题。

### 店铺命名规则与避坑

![](img/e10618414385b6547cf446bc22457b1e_1.png)

这是一个极易出错的环节，请务必仔细阅读。

*   **规则**：通过小红书账号直接注册店铺时，**店铺名称将自动生成为“{小红书账号名}的店铺”**。
*   **示例**：如果你的小红书账号名叫“张三”，那么注册下来的店铺名就是 **“张三的店铺”**。
*   **核心问题**：店铺名具有唯一性。如果“张三的店铺”这个名字已被占用，你在最终确认时会提示重复，导致注册失败。
*   **避坑操作**：
    1.  在修改小红书账号名前，**先搜索确认是否有同名用户**。
    2.  如果已有同名用户，建议先修改你的小红书账号名，确保其唯一性。
    3.  然后再去进行店铺注册，这样系统抓取的名字就不会冲突。
*   **原因**：小红书账号名**7天内只能修改一次**。如果因店铺名重复导致注册失败，你将需要等待7天后才能再次尝试，非常浪费时间。

### 保证金与平台扣点

注册时还需了解相关费用政策。

*   **保证金**：大部分类目为 **1000元**。目前平台对营业执照经营范围的审核相对宽松，八成类目都能通过。
*   **平台佣金（扣点）**：比例为 **5%**，但有免佣政策。
    *   **免佣规则**：店铺单月结算额 ≤ 1万元时，平台免收佣金。
    *   **收费规则**：结算额 > 1万元时，**仅对超出1万元的部分收取5%佣金**。
    *   **计算公式**：`应缴佣金 = (当月结算总额 - 10000) * 5%`
    *   **结算示例**：若当月销售额为15000元，则佣金为 `(15000 - 10000) * 5% = 250元`，而非 `15000 * 5% = 750元`。
*   **重要提示**：系统会“先收后退”。即使当月销售额未达1万，也可能显示扣费，但平台会在后续进行返还。

### 经营信息公示设置

在店铺注册完成后，需要设置公示信息才能上传商品。这里有一个官方未明确提示但必须操作的选项。

在设置“经营类型”时，你会看到多个选项。**请务必将列出的所有经营类型选项（通常为3-4个）全部勾选**。这一步很多人会忽略，导致后续功能受限。

## 总结与资源

![](img/e10618414385b6547cf446bc22457b1e_5.png)

本节课中我们一起学习了小红书店铺入驻的全流程。

1.  **明确概念**：区分了“专业号”与“店铺”的关系。
2.  **选择主体**：根据自身情况（个人/个体户/企业）选择合适的店铺类型，个人创业者建议从个人店开始。
3.  **避开陷阱**：重点掌握了**店铺命名唯一性**的规则及提前验证的方法，这是顺利注册的关键。
4.  **了解费用**：清楚了1000元左右的保证金及“月销1万内免佣金”的扣点政策。
5.  **完成设置**：记得在后台将所有的“经营类型”选项全部勾选。
