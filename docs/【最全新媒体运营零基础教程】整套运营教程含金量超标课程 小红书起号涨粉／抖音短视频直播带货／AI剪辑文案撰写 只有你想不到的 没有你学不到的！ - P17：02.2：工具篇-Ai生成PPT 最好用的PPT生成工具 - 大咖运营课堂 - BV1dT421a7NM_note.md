# 新媒体运营零基础教程：P17：02.2：工具篇-AI生成PPT 🚀

在本节课中，我们将要学习一款非常实用的AI工具，它能帮助我们快速生成演示文稿。我们将了解它的两种核心工作模式，并通过一个实际案例演示其操作流程。

## 概述

AI生成PPT工具可以根据用户提供的主题或现有文档，自动梳理内容大纲，并基于此大纲生成详细的PPT页面内容。这极大地提升了制作演示文稿的效率。

## 工具核心功能

![](img/06e5a4f5065462a3297e05f05e69468e_1.png)

![](img/06e5a4f5065462a3297e05f05e69468e_3.png)

该工具主要提供两种生成模式，适应不同的创作起点。

以下是两种核心工作模式：

![](img/06e5a4f5065462a3297e05f05e69468e_5.png)

1.  **主题生成模式**：用户输入一个主题，工具会像ChatGPT一样，根据主题生成完整的PPT内容。
2.  **文档生成模式**：用户上传一篇文档（如TXT、PDF），AI助手会自动总结文档大纲，并基于大纲生成PPT内容。

![](img/06e5a4f5065462a3297e05f05e69468e_7.png)

## 实战演示：基于文档生成PPT

上一节我们介绍了工具的两种模式，本节中我们来看看如何利用现有文档快速生成PPT。我们将通过上传一篇关于新媒体运营的文章进行演示。

![](img/06e5a4f5065462a3297e05f05e69468e_9.png)

![](img/06e5a4f5065462a3297e05f05e69468e_11.png)

操作步骤如下：

![](img/06e5a4f5065462a3297e05f05e69468e_13.png)

1.  准备一篇文档并上传。
2.  工具会自动解读文档，并在界面下方列出分析后的大纲要点。
3.  大纲确认后，点击“开始制作”。
4.  工具会基于对内容的理解，自动生成PPT的详细内容。
5.  生成完成后，可以进入编辑界面。

在演示中，上传一篇关于“新媒体运营流量与创作标签”的文章后，工具分析出了包括“新媒体运营的本质”、“账号运营的策略”等在内的多个大纲节点。

## 编辑与优化

生成初步PPT后，我们可以对其进行调整和美化，使其更符合个人需求。

以下是可进行的优化操作：

![](img/06e5a4f5065462a3297e05f05e69468e_15.png)

*   **更换模板**：工具提供多种模板可供快速切换，虽然当前模板库数量有限，但切换速度很快。
*   **调整内容**：在编辑界面右侧，可以对文字、形状等元素进行修改和替换。
*   **本地深度编辑**：将生成的PPT下载到本地，使用PowerPoint或Keynote等软件进行更细致的调整。

![](img/06e5a4f5065462a3297e05f05e69468e_17.png)

这个工具最关键的优势在于，它能基于简单的提纲或文档，自动拓展出详细的页面内容，而不仅仅是进行格式排版。
