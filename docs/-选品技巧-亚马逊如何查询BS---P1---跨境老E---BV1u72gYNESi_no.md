# 亚马逊选品技巧：P1：如何查询BS榜单100名之后的产品 🔍

在本节课中，我们将学习一种无需借助付费工具，即可查询亚马逊Best Sellers榜单100名之后产品的方法。这对于深入分析市场竞品、进行精细化选品至关重要。

## 概述

![](img/78175c7c4c79228f4e7b8840c8850f89_1.png)

进行亚马逊选品时，分析Best Sellers榜单是常规操作。榜单前100名的产品固然具有参考价值，但为了获得更全面的市场洞察，我们常常需要查看前300名甚至更靠后的产品。过去可以通过修改URL参数（如将`P2`改为`P3`）来实现，但该方法现已失效。本节将介绍一种新的、有效的方法。

## 操作步骤详解

以下是查询BS榜单100名之后产品的具体步骤。

### 第一步：定位目标市场与产品

![](img/78175c7c4c79228f4e7b8840c8850f89_3.png)

首先，我们需要打开一个与您想分析的市场相同的产品页面。例如，如果您想分析“瑜伽垫”市场，就打开一个该类别下的任意产品详情页。

### 第二步：查找产品信息与类目ID

![](img/78175c7c4c79228f4e7b8840c8850f89_5.png)

在产品详情页中，将页面滚动至最底部。您会看到一个名为“产品信息”的折叠区域。点击“产品详情”将其展开。

![](img/78175c7c4c79228f4e7b8840c8850f89_7.png)

在展开的详情中，您可以找到该产品在BS榜单中的当前排名（例如“#19 in Sports & Outdoors”）。同时，请注意浏览器地址栏中的URL。

在URL中，找到一串以`B0`开头、由字母和数字组成的代码，这串代码就是该产品所在类目的**节点ID**。例如，URL中可能包含类似`node=1234567011`的部分，其中的`1234567011`就是类目ID。请复制这串数字。

### 第三步：替换URL以获取完整榜单

我们提供了一个基础查询模板。您需要将模板URL中的特定部分替换为上一步复制的类目ID。

基础模板URL结构如下：
```
https://www.amazon.com/Best-Sellers/zgbs/【此处替换为你的类目ID】
```

![](img/78175c7c4c79228f4e7b8840c8850f89_9.png)

操作方法是：将模板中的 `【此处替换为你的类目ID】` 这串占位文字，直接替换为您刚刚复制的数字ID。替换后，完整的URL应类似于：
```
https://www.amazon.com/Best-Sellers/zgbs/1234567011
```

### 第四步：访问与排序

在浏览器中访问替换后的新URL。页面加载后，点击页面上方的“**Sort by: Featured**”下拉菜单，并选择“**Best Sellers**”选项。

完成此操作后，页面将展示该类目下**排名前1000名**的所有产品，而不仅仅是前100名。您现在可以系统地分析这些竞品的各项数据，如价格、评论、上架时间等，为选品决策提供支持。

## 资源与提示

*   为方便使用，本节教程的**基础模板链接**已附在文末。您只需按上述步骤替换类目ID即可。
*   请确保**只替换**指定的占位文字，URL的其他部分不要做任何修改。
*   通过此方法获得的数据，结合其他市场分析维度（如需求趋势、竞争热度），能帮助您更准确地评估市场机会。

## 总结

本节课我们一起学习了如何手动查询亚马逊BS榜单100名之后的产品。核心方法是：**通过任意产品页获取其类目节点ID，并利用特定URL模板进行替换访问**。这个方法简单有效，无需额外成本，能帮助您突破前100名的限制，对市场进行更深层、更全面的竞争分析，是选品过程中一个非常实用的技巧。
