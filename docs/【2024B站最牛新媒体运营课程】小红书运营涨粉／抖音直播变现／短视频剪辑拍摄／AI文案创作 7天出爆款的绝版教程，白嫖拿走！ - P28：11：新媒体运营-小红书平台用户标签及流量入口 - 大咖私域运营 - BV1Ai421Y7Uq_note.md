# 新媒体运营小红书课程：P28：11：小红书平台用户标签及流量入口 📊

在本节课中，我们将学习小红书平台如何为用户打上标签，以及笔记是如何通过不同的流量入口被分发给用户的。理解这些底层逻辑，能帮助我们更好地创作内容，并让笔记精准触达目标受众。

## 用户标签判定逻辑

![](img/7f944538313caaa2f21b23a5b6ae3f4e_10.png)

小红书系统为用户打标签，主要依据是用户的**浏览行为**。用户刚注册时选择的兴趣范围，参考价值不大，它仅作为平台对用户的内容方向引导和初步筛选，无法进入系统最深层的用户标签池。

![](img/7f944538313caaa2f21b23a5b6ae3f4e_11.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_12.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_13.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_14.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_16.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_17.png)

系统判定用户的兴趣分类，完全基于其浏览行为。以下是系统分析用户行为的主要维度：

![](img/7f944538313caaa2f21b23a5b6ae3f4e_19.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_21.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_22.png)

### 1. 点击率
**点击率**是指系统在发现页向用户推送多个内容（如明星、美妆、家居、母婴、装修等）后，用户点击各类内容的频率。例如，用户点击了10条内容，其中美妆类3次，家居类2次，母婴类1次或0次。系统通过此数据，能判断用户对母婴内容不感兴趣，可能不是宝妈；而对美妆内容感兴趣，则会持续推送更多美妆内容进行试探。

![](img/7f944538313caaa2f21b23a5b6ae3f4e_24.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_25.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_26.png)

通过点击率的逻辑，系统能为用户打上**强关联**和**弱关联**的标签。

![](img/7f944538313caaa2f21b23a5b6ae3f4e_27.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_28.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_29.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_30.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_31.png)

为了便于理解，我们可以想象一个标签云图。假设一个用户的标签云中，“美妆”这个词最大，而“母婴”、“装修”等词较小，这代表该用户是美妆领域的重度兴趣用户。系统后续会主要为其推送美妆相关内容，减少或不推送其不感兴趣的内容，以提升用户的获得感与停留时长。

![](img/7f944538313caaa2f21b23a5b6ae3f4e_33.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_34.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_35.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_36.png)

这就是系统、平台、用户和博主之间的关联。作为博主，需要平衡发布的内容，既要为用户提供价值，也要为平台提供价值。

![](img/7f944538313caaa2f21b23a5b6ae3f4e_37.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_38.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_40.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_41.png)

### 2. 停留时长
**停留时长**是另一个关键分析行为。例如，用户喜欢美妆内容，但很多笔记点开就划走，说明这些具体笔记不符合其喜好。系统会继续推送美妆领域下的其他细分内容（如上妆、护肤等）来观察用户行为。

![](img/7f944538313caaa2f21b23a5b6ae3f4e_43.png)

如果用户对某类平时点击较少的内容（如家居）产生了长停留（仔细阅读图片、查看评论），系统则会判断用户可能有相关需求（如装修、购买餐具），并考虑推送更多相关笔记或商品。

![](img/7f944538313caaa2f21b23a5b6ae3f4e_44.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_45.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_46.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_47.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_48.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_50.png)

### 3. 页面浏览量 (PV)
**PV** 可以理解为用户浏览页面的深度。例如，用户看完一篇装修笔记后，还点进博主主页，并连续看了第二、三、四篇笔记。这提升了用户在该页面的浏览数。

![](img/7f944538313caaa2f21b23a5b6ae3f4e_51.png)

如果账号主页内容精致，就能有效提升用户的浏览深度。这个数值对账号健康度评分有很大帮助。浏览深度足够时，还可能引发**关注**行为，这是永久提升账号权重和粉丝数的关键。

![](img/7f944538313caaa2f21b23a5b6ae3f4e_53.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_54.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_55.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_56.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_57.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_58.png)

### 4. 互动行为
以下是用户的主要互动行为，每一项都是我们在创作笔记时需要努力提升的数据：
*   **点赞**
*   **收藏**
*   **评论**
*   **关注**

![](img/7f944538313caaa2f21b23a5b6ae3f4e_59.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_60.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_61.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_62.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_63.png)

互动数据高，不仅能提升笔记进入更大推荐流量池的几率，还可能带动账号内其他笔记的流量，对后续接单、变现也更有帮助。互动行为是提升笔记下一层级推荐流量的核心。

![](img/7f944538313caaa2f21b23a5b6ae3f4e_65.png)

了解用户标签后，我们来看笔记是如何通过不同入口分发给用户的。小红书主要有四个流量分发入口：

![](img/7f944538313caaa2f21b23a5b6ae3f4e_74.png)

### 1. 发现页
发现页呈现的内容主要包含以下几点：
*   **已关注的达人**发布的内容。
*   根据用户**点击率**等行为标签，系统认为用户可能喜欢的**兴趣匹配内容**。
*   系统为了增加平台内容多样性，也会推送一些用户可能未接触过的**发散性内容**。

### 2. 关注页
关注页的内容非常简单：
*   仅展示用户**已关注博主**发布的笔记。

![](img/7f944538313caaa2f21b23a5b6ae3f4e_91.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_93.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_94.png)

### 3. 附近页
附近页的功能类似于早期微信的“附近的人”，其逻辑是：
*   根据用户手机定位，推送**附近及同城相关的内容**。

![](img/7f944538313caaa2f21b23a5b6ae3f4e_96.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_97.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_98.png)

![](img/7f944538313caaa2f21b23a5b6ae3f4e_100.png)

该板块以美食、本地旅游等强地域特征的内容为主。如果你是线下实体店（如美容院、餐厅）的运营者，在做信息流广告时需要特别注意附近人群的推送。

![](img/7f944538313caaa2f21b23a5b6ae3f4e_102.png)

### 4. 购物页
购物页是小红书向电商方向发展的重要入口，主要匹配用户的购物需求。目前，部分用户尚未形成在小红书直接购物的强习惯，更多人仍倾向于搜索笔记获取推荐。该入口与小红书的直播带货、商品笔记等变现方式紧密相关。
