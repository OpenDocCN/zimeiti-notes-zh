# 视频号运营教程：P27：13-视频画中画如何添加_1 🎬

在本节课中，我们将要学习如何在视频中添加画中画效果。画中画是一种将多个视频或图片层叠在一起的技术，可以丰富视频内容，增加视觉层次感。我们将详细介绍添加画中画的三种主要素材类型及其具体操作步骤。

## 概述

画中画功能允许你在主视频之上叠加额外的视频或图片。添加画中画主要分为三个步骤：导入素材、调整声音、设置混合模式。接下来，我们将逐一讲解这些步骤。

## 画中画的三种素材类型

画中画素材主要分为以下三种类型：

1.  **自拍图/视频**：使用自己拍摄的照片或视频。
2.  **剧情素材**：使用手机中存储的影视剧、短片等剧情类视频片段。
3.  **矢量/素材库内容**：使用软件内置素材库中的纯色背景、矢量图形或视频模板。

## 添加自拍视频作为画中画

上一节我们介绍了画中画的素材类型，本节中我们来看看如何添加一段自拍视频作为画中画。

以下是添加自拍视频的具体步骤：

1.  点击编辑界面中的“画中画”按钮，然后选择“新增画中画”。
2.  从相册中选择一段自拍视频并点击“添加”。添加后，该视频会覆盖在主视频之上。
3.  **关闭画中画声音**：由于主视频本身可能有声音，为避免干扰，需要关闭画中画的音频。点击选中画中画轨道，在出现的工具栏中找到“音量”图标，将音量滑块拖到最左侧（数值为0）。
4.  点击“混合模式”选项。在弹出的模式中，选择“绿色”或“叠加”效果。
5.  将“不透明度”参数调整到 **2** 左右（范围在1-3之间），使画中画变为半透明状态。
6.  点击确认（对勾图标）。此时，画中画视频将变为透明，叠加在主视频上。
7.  最后，务必在时间轴上将画中画素材的尾部与主视频的尾部对齐，拖动其末端直至与主视频等长。

## 添加个人照片作为画中画

学会了添加视频后，添加静态图片的操作也类似。照片本身没有声音，因此无需进行静音操作。

以下是添加个人照片的具体步骤：

1.  同样通过“新增画中画”功能，这次选择一张个人照片。
2.  照片添加后也会覆盖主画面。点击选中该照片轨道，然后点击“混合模式”。
3.  选择“叠加”或其他效果，并将不透明度调整至 **2** 左右。照片会变为半透明。
4.  此时，你可以自由缩放或移动这张半透明的照片，将其放置在屏幕任意位置，因为它已不会完全遮挡主视频。

## 添加剧情类视频作为画中画

如果你的手机里有下载的短视频或影视片段，也可以将它们作为画中画添加进来，操作逻辑是相同的。

以下是添加剧情视频的具体步骤：

1.  通过“新增画中画”导入一段剧情视频。
2.  **首要步骤是关闭其原声**。
3.  接着，点击“混合模式”，选择“绿色”等效果，并将不透明度调至 **2**。
4.  如果导入的剧情视频时长**短于**主视频，你可以通过**复制**功能来延长它。选中该画中画片段，在工具栏末尾找到“复制”按钮并点击，即可生成一段相同的片段，将其拖拽衔接在原片段之后。
5.  如果画中画视频时长**超过**了主视频，你可以直接拖拽其末端进行缩短，或者使用“分割”工具将多余部分删除。

## 使用素材库内容作为画中画

最后，我们来看看如何使用软件自带的素材库。库内包含大量纯色背景、矢量动画等素材，非常适合用作装饰性画中画。

以下是使用素材库的具体步骤：

1.  点击“新增画中画”，然后选择“素材库”。
2.  你可以在搜索框中输入关键词（如“矢量”）来查找相关素材，也可以直接浏览分类。
3.  选择一段合适的素材（例如一段3分钟的动画）并添加。
4.  添加后，重复标准操作：**关闭原声** -> 进入“混合模式” -> 选择“绿色”效果 -> 将不透明度调整为 **2**。
5.  同样，如果素材时长不够，可以使用“复制”功能；如果太长，则拖拽调整或使用“分割”删除。

## 核心技巧与目的总结

本节课中我们一起学习了画中画的添加与处理。整个过程的核心操作可以总结为以下流程：

**添加素材 -> 关闭画中画声音 -> 设置混合模式（绿色/叠加） -> 调整不透明度（至2左右）-> 调整时长与位置**

添加画中画的主要目的是在不影响主画面观感的前提下，叠加额外的视觉元素。通过将其设置为半透明，这些元素既能丰富视频内容，又能避免喧宾夺主。平台系统在识别视频时，能够检测到这些“看不见”的图层，从而可能增加视频被推荐的机会，这也就是所谓的“触众”策略。

如果你在后续操作中忘记了某个步骤，可以随时回看本教程中对应的章节进行复习。
