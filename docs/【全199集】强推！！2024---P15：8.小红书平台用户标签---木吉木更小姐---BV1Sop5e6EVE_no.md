# 小红书运营教程：P15：小红书平台用户标签及流量入口 📊

在本节课中，我们将要学习小红书平台如何为用户打上标签，以及平台主要的流量入口有哪些。理解这些底层逻辑，能帮助我们更好地分析笔记如何被推荐给用户，并指导我们如何优化内容以获得更多精准曝光。

上一节我们介绍了系统如何判定内容标签，本节中我们来看看平台如何判定用户标签。这个过程能帮助我们分析和理解笔记是如何被系统推荐给用户的，并能从背后的逻辑反推，如何让笔记被我们的精准用户看到。

## 用户标签判定逻辑

系统为用户打标签的主要依据是用户的**浏览行为**。用户刚注册账号时所选择的兴趣范围，参考价值不大。它只是平台在初期对用户进行内容方向的引导和初步筛选，这个筛选还无法进入系统最深层的用户标签池。

![](img/53f7941c81138db74b90078048b1bd1c_3.png)

![](img/53f7941c81138db74b90078048b1bd1c_5.png)

系统判定用户的兴趣分类，是根据用户持续的浏览行为来决定的。以下是系统判断用户喜好的几个核心行为指标：

以下是几个核心的用户行为指标：

![](img/53f7941c81138db74b90078048b1bd1c_7.png)

![](img/53f7941c81138db74b90078048b1bd1c_8.png)

![](img/53f7941c81138db74b90078048b1bd1c_9.png)

![](img/53f7941c81138db74b90078048b1bd1c_11.png)

1.  **点击率**
    *   **定义**：系统在发现页向用户推送多个内容（例如美妆、家居、母婴等），用户点击哪一类内容更多，系统就能通过算法得出用户的兴趣偏好。
    *   **公式**：`点击率 = (对某类内容的点击次数) / (系统推送的该类内容总数)`
    *   **作用**：系统通过点击率判断用户的强关联和弱关联兴趣标签，并持续推送相关领域内容进行试探。

2.  **停留时长**
    *   **定义**：用户点开一篇笔记后观看的持续时间。如果用户对某篇笔记停留时间短，说明内容可能不感兴趣；如果停留时间长，甚至反复观看，则说明兴趣浓厚。
    *   **作用**：帮助系统更精细地判断用户在某个大标签下（如美妆）的具体偏好（如护肤或彩妆），甚至能发现用户的潜在新需求（如突然对家居内容产生长时停留）。

![](img/53f7941c81138db74b90078048b1bd1c_13.png)

![](img/53f7941c81138db74b90078048b1bd1c_14.png)

![](img/53f7941c81138db74b90078048b1bd1c_15.png)

![](img/53f7941c81138db74b90078048b1bd1c_16.png)

3.  **页面浏览量**
    *   **定义**：用户浏览页面的深度。例如，用户不仅看了一篇笔记，还点进博主主页，并观看了第二、第三篇笔记。
    *   **代码/概念**：这可以理解为用户的 **PV** 值。PV越高，说明用户对账号内容越感兴趣。
    *   **作用**：提升用户在我们主页的PV值，能为账号带来非常健康的评分，有利于账号成长。

4.  **互动行为**
    *   **定义**：用户对笔记的点赞、收藏、评论、关注等行为。
    *   **作用**：这些互动数据直接决定了笔记能否进入下一层更大的流量池。数据越好，笔记获得的推荐流量越多，同时也能带动账号其他笔记的曝光，并提升粉丝数。

为了更直观地理解，我们可以想象一个**标签云图**。系统会根据用户行为，为其打上大小不一的兴趣标签（如美妆、母婴、装修）。其中最大的那个标签，就代表了用户当前最主要的兴趣方向，系统会据此主要推送相关内容。

平台、用户和博主三者之间形成一个价值闭环：博主创作内容为用户提供价值；平台通过算法将内容匹配给用户，也为用户提供价值；优质内容和活跃用户反过来为平台创造价值。

## 平台流量分发入口

了解用户标签后，我们来看小红书笔记主要的曝光渠道，即流量分发入口。小红书目前主要有四个流量入口：

![](img/53f7941c81138db74b90078048b1bd1c_30.png)

![](img/53f7941c81138db74b90078048b1bd1c_31.png)

![](img/53f7941c81138db74b90078048b1bd1c_32.png)

![](img/53f7941c81138db74b90078048b1bd1c_33.png)

![](img/53f7941c81138db74b90078048b1bd1c_34.png)

以下是四个主要的流量入口：

*   **发现页**
    *   **内容构成**：
        1.  用户已关注的达人发布的内容。
        2.  系统根据用户标签（点击率、停留时长等）推荐的、用户可能感兴趣的内容。
        3.  系统为了平台内容多样化而推荐的一些用户可能未接触过的内容。
    *   **核心价值**：这是博主获取流量的**最主要来源**。我们的笔记和账号后期能获得的流量，主要依赖于在发现页的推荐。

![](img/53f7941c81138db74b90078048b1bd1c_36.png)

![](img/53f7941c81138db74b90078048b1bd1c_37.png)

![](img/53f7941c81138db74b90078048b1bd1c_39.png)

*   **关注页**
    *   **内容构成**：纯粹展示用户所有已关注博主发布的最新笔记。
    *   **特点**：流量较为稳定，但依赖于粉丝的主动查看习惯。

![](img/53f7941c81138db74b90078048b1bd1c_40.png)

![](img/53f7941c81138db74b90078048b1bd1c_42.png)

![](img/53f7941c81138db74b90078048b1bd1c_44.png)

*   **附近页**
    *   **推送逻辑**：根据手机定位，推送附近及同城的相关内容。
    *   **内容特点**：以美食、本地旅游、线下店推广等强地域性需求的内容为主。
    *   **运营提示**：如果你是线下实体店（如美容院、餐厅），在做信息流广告或内容时，需要特别注意对附近人群的覆盖。

![](img/53f7941c81138db74b90078048b1bd1c_46.png)

![](img/53f7941c81138db74b90078048b1bd1c_48.png)

*   **购物页**
    *   **定位**：小红书向电商化发展的重要入口，直接匹配用户的购物需求。
    *   **现状**：部分用户已养成在此搜索购物的习惯，但仍有大量用户习惯先搜索相关笔记做攻略。笔记与电商的结合日益紧密。

![](img/53f7941c81138db74b90078048b1bd1c_50.png)

![](img/53f7941c81138db74b90078048b1bd1c_52.png)

## 总结

![](img/53f7941c81138db74b90078048b1bd1c_54.png)

![](img/53f7941c81138db74b90078048b1bd1c_55.png)

本节课中我们一起学习了小红书平台用户标签的判定逻辑和四大流量入口。

![](img/53f7941c81138db74b90078048b1bd1c_56.png)

![](img/53f7941c81138db74b90078048b1bd1c_57.png)

1.  **用户标签**：系统通过分析用户的**点击率、停留时长、页面浏览量和互动行为**等浏览行为，为用户打上动态的兴趣标签，从而实现内容的精准推荐。
2.  **流量入口**：笔记主要通过**发现页、关注页、附近页和购物页**四个渠道曝光。其中，**发现页**是流量的核心来源，其推荐机制与用户标签系统紧密相连。
