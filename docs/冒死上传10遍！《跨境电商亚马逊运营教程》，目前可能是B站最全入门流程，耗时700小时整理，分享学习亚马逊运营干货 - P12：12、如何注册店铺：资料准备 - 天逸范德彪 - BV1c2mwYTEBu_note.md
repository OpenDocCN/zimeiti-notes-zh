# 跨境电商亚马逊运营教程：P12：如何注册店铺：资料准备 📝

![](img/7b5593b03b811c965fbc4a9cb64cc29f_0.png)

在本节课中，我们将要学习注册亚马逊店铺前需要准备哪些核心资料。这是开启亚马逊运营的第一步，准备好齐全且正确的资料至关重要。

上一节我们介绍了如何选择亚马逊站点，本节中我们来看看注册店铺前需要准备哪些具体材料。

## 资料清单概述

以下是注册亚马逊店铺必须准备的六项核心资料，我们将逐一详细说明。

1.  **营业执照彩色扫描件**
    *   这是指营业执照的彩色照片或扫描件。
    *   执照类型可以是企业执照，也可以是个体工商户执照。
    *   执照可在当地工商局免费办理。
    *   如果没有执照，可以借用家人或朋友的。例如，借用开超市或服装店的亲友的执照。
    *   如果无法借用，可以寻找代办机构办理，费用通常在几百到一千元人民币左右，也可以联系课程助理老师协助办理。

2.  **法人身份证正反面照片**
    *   这是指营业执照上对应法人的身份证照片。
    *   **核心要求**：营业执照和法人身份证必须是同一个人的信息。
    *   请注意，除了这两份资料，后续其他资料可以与法人信息不同。

3.  **付款信用卡**
    *   这张信用卡的唯一作用是支付亚马逊店铺的月租费。
    *   信用卡必须是全币种信用卡，首选带有 **Visa** 或 **Mastercard** 标志的卡。
    *   各大银行（如工商银行、建设银行、广发银行、中信银行等）均可办理。
    *   如果没有信用卡，可以借用他人的，或者花费约100元购买一张虚拟信用卡。只需每月按时向虚拟卡内充值月租金额（39.99美元）即可。

4.  **海外收款账户**
    *   这个账户的功能类似于国际版的支付宝。
    *   当你的亚马逊店铺产生销售额后，资金会先进入这个海外收款账户，然后你再从此账户提现到自己的国内银行储蓄卡。
    *   注册过程简单快捷，只需在线填写资料并绑定个人身份证和银行卡即可完成，通常只需几分钟。

5.  **电话号码**
    *   需要一个有效的中国大陆手机号码，用于接收验证码和官方通知。

6.  **邮箱地址**
    *   需要一个常用的电子邮箱，例如163邮箱、126邮箱或QQ邮箱均可。
    *   个人推荐使用163邮箱进行注册。

## 重要注意事项

![](img/7b5593b03b811c965fbc4a9cb64cc29f_4.png)

在准备上述资料时，有几个关键点需要特别注意。

*   **关于营业执照的要求**：
    *   对注册资金和注册年限没有硬性要求。
    *   执照经营范围需带有销售性质即可（如零售、批发）。
    *   **重要警告**：**切勿**在经营范围中主动添加“进出口贸易”字样。添加后，按规定需额外办理进出口手续，属于画蛇添足，会增加不必要的麻烦。
    *   执照的经营范围与你未来在亚马逊店铺销售的产品无需一致。例如，用超市的执照注册的店铺，完全可以销售服装。

*   **关于资料归属**：
    *   重申：**营业执照**和**法人身份证**必须属于同一个人。
    *   其他资料（信用卡、收款账户、手机号、邮箱）可以是运营者本人的，与法人信息不同也没有关系。

*   **关于信用卡的作用**：
    *   它主要用于店铺激活时扣除首月月租。当店铺开始运营并有销售额后，后续月租会优先从店铺账户余额中扣除，并非每次都从信用卡扣款。
