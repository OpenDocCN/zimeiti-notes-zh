# 知识付费行业观察：1：平台关停潮解析

## 概述
在本节课中，我们将要学习近期知识付费行业的重要变化。我们将分析多个主流平台关停其知识付费业务板块的原因，并探讨这一趋势对内容创作者和行业未来的影响。

## 平台关停现象
近期知识付费行业风波不断，很多博主都陷入恐慌。

抖音宣布在7月31号关闭支付费平台学量。

很早以前，腾讯课堂已经宣布8月1号关闭课程反问，10月1号即将会关停服务。

腾讯大学也宣布了不再发布新课程。

## 关停原因分析
上一节我们介绍了各大平台的关停动作，本节中我们来分析一下这些平台为什么要关停这个板块。

难道这些业务来讲都不挣钱吗？

我们先来说一下抖音，说实话抖音的支付场子足足可以让一家人花几十倍，但即便这样这些GM模营要比起其他类目呢在抖音是微不足道。

而且利用抖音割韭菜人太多了，造成了大量的课数。

像腾讯课堂和淘宝大学吧，腾讯课堂和淘宝大学其实做的都是比较不错的。

上面的课程几块几百块几千块的都有，而且销量非常的精，毕竟是大平台有足够的流量也有课程。

但即使是这个样子，整个产业在整个腾讯阿里的生态体经微不足道，可以说是事多客数多，挣钱少。

这让服务器资源大，课程储存的流量成本的压力也很大。

既然不挣钱，还留着干嘛，还不如卸掉把我一身情楚。

## 行业监管与IP影响
分析了经济原因后，我们再来看看监管与内容生态的影响。

今年开始有很多的IP大佬又不断报了，某达某州、某省等IP大佬，后台都给抖音带来不少麻烦。

你这次抖音不再默默的帮领导们擦屁股，所以直接选择一刀切。

## 未来趋势：私域流量的崛起
平台政策收紧后，知识付费的未来在哪里？本节我们来看看流量的新去向。

你试想一下，一旦公寓没有大IP大量的流量。

在哪里都会去向私域，因为人的求知欲是不会变的。

最终私欲变成自付费的冲天，没有了公寓影响，在私域里搞致付费依旧大有可。

## 总结
本节课中我们一起学习了近期知识付费平台关停潮的现象。我们分析了其背后的经济原因，即**投入产出比（ROI）过低**，公式可表示为 `ROI = (收益 - 成本) / 成本`。同时，我们也探讨了监管压力和头部IP带来的风险。最后，我们展望了行业未来向私域流量转移的趋势。对于知识创作者而言，构建自己的私域阵地将变得愈发重要。
