1.  **角色**
    *   **定义**：你在特定赛道中扮演的专业身份。这直接关联到你选择的内容方向。
    *   **示例**：
        *   **好物分享/测评赛道**：你需要展现选品的专业眼光和真实体验。
        *   **装修博主**：你需要证明自己拥有正在装修或已装修完成的真实案例，通过记录过程、分享避坑经验、展示前后对比来建立可信度。
        *   **母婴赛道**：常见的角色是“宝妈”，但也可以是“育儿护理专家”，后者不一定亲身是母亲，可以是相关领域的专业人士。
        *   **情感赛道**：可以是“心理学顾问”、“情感大师”，也可以是“成长型人设”。成长型人设通过分享自身经历（如女性在家庭、职场、创业中遇到的问题及应对方法）来与观众共鸣。
    *   **核心公式**：`人设角色 = 内容赛道 + 专家身份`

2.  **内容**
    *   **定义**：你持续提供的、与所选赛道高度相关的信息。
    *   **要求**：内容需要吸引用户、提供价值，并最好具备一定平台特色。这直接关系到账号的垂直度和标签化。
    *   **重要性**：平台根据账号标签进行流量分发。如果内容不垂直（例如第一篇讲儿童教育，第二篇讲宠物），会扰乱账号标签，导致推送给不感兴趣的用户，数据表现差，进而影响后续流量。
    *   **核心原则**：坚持输出**垂直领域**的**统一内容**，以强化账号标签。

3.  **形式**
    *   **定义**：内容呈现的具体方式，包括场景、声音和内容类型。
    *   **构成要素**：
        *   **产出场景**：室内固定场景、室外多变场景、车内、上班路上等。可以参考对标账号的常用场景。
        *   **声音呈现**：本人原声出镜讲解，或不出镜仅使用配音（可用`剪映`等工具实现）。需评估自己能否流畅表达。
        *   **内容类型**：剧情演绎、干货分享、生活记录等。不同类型各有优劣：
            *   `剧情型`：创作难度较高。
            *   `干货型`：需要精心准备文案脚本。
            *   `生活记录型`：需要积累大量拍摄素材。

4.  **记忆点**
    *   **定义**：让用户能记住并识别你的独特设计。
    *   **设计方法**：在内容中固定植入某些元素。
    *   **以下是常见的记忆点设计，可分为两类**：
        *   **前置强化型（视频开头出现）**：
            *   前几秒的固定动作或手势。
            *   固定的开场特效。
            *   统一的开场白或口号。
            *   **设计逻辑**：将记忆点置于视频开头，能在用户连续刷到你多个视频时，形成强烈的重复冲击，加深印象。
        *   **全局贯穿型（贯穿整个视频）**：
            *   统一的背景音乐（BGM）。
            *   固定出现的物品（如特定的台灯、背景装饰）。
            *   统一风格的服装。
            *   **设计逻辑**：这些元素伴随内容全程，能营造独特的氛围感，使用户在听到音乐或看到物品时就能联想到你。

本节课中我们一起学习了新媒体IP人设定位的四大核心因素：**角色**、**内容**、**形式**和**记忆点**。塑造人设的关键在于，根据你的内容赛道明确专业身份，并持续输出垂直、有价值的内容。同时，通过精心设计的内容形式（场景、声音、类型）和强化记忆点（前置动作、统一BGM等），让你的账号在用户心中留下深刻、统一的印象。记住，人设是在长期的内容运营中自然生长和巩固的，无需在起步阶段过度焦虑。
