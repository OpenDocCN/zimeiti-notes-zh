# 抖音运营教程：P11：扶持加权与降权限流

在本节课中，我们将学习抖音平台的内容审核规则，了解哪些内容会获得平台的扶持与加权，哪些内容则可能导致账号被降权甚至限流。掌握这些规则是避免违规、让账号健康发展的关键。

![](img/b57a8b8e06e92dd789f62fcfd48a1e05_1.png)

上一节我们介绍了账号权重的分类，本节中我们来看看具体的内容发布规范。

![](img/b57a8b8e06e92dd789f62fcfd48a1e05_3.png)

## 获得扶持加权的内容特征

平台会优先推荐符合以下特征的内容，这有助于提升视频的初始流量。

![](img/b57a8b8e06e92dd789f62fcfd48a1e05_5.png)

以下是获得扶持加权的内容要点：
*   **视频格式**：发布竖版的原生视频，即移动端适配的 **9:16** 格式。
*   **画面质量**：确保画面平稳，作品清晰度高。
*   **音频质量**：视频中无过多的杂音或噪音。

![](img/b57a8b8e06e92dd789f62fcfd48a1e05_1.png)

这些要点较为基础，此处不再详细展开。如需深入了解，后续整理的资料页中会包含此部分内容供大家课后查阅。

## 导致降权限流的内容禁区

发布以下类型的内容，极有可能触发平台的审核机制，导致视频流量受限或账号权重降低。

以下是会导致降权限流的内容禁区：
1.  **违规信息**：内容涉及吸烟、暴力、炫富（过度展示金钱）等不良信息。
    ![](img/b57a8b8e06e92dd789f62fcfd48a1e05_3.png)
2.  **外部引流**：在视频中留下微信、微博、淘宝等外部联系方式。长时间展示商品Logo、手机号或二维码也属于违规行为。
    > **注意**：二维码并非绝对禁止出现，但展示时间**最长不要超过3秒**。有时审核严格，即使出现1秒也可能导致视频不通过。
    ![](img/b57a8b8e06e92dd789f62fcfd48a1e05_5.png)
3.  **硬性广告**：进行明显的营销广告、推销或引流卖货行为。
4.  **无声好物推荐**：进行无口播的单纯拆箱或物品展示。这是许多新手容易踩坑的地方。
    > **案例说明**：有学员发布好物推荐视频后，小黄车被下架。原因一是视频中直接出现了“9块9”等价格信息，属于硬性推销；二是视频仅有画面和背景音乐，没有任何口播介绍。
    > **正确做法**：即使是好物推荐，也应通过口播简单介绍产品特点或使用体验，例如：“这个玩具孩子玩了四个小时都没停”，将推荐融入真实的分享场景中，而非单纯展示。
5.  **口播广告**：在口播中直接念出广告语或联系方式进行推销。这种方式在抖音行不通，更适合传统电商场景。
