# 剪映教程：P10：剪映核心剪辑功能讲解--画中画功能 🎬

在本节课中，我们将要学习剪映的核心剪辑功能之一：画中画。这个功能允许你在一个视频画面上叠加另一个视频或图片，从而创造出内容更丰富、更有趣的多画面效果。

## 概述

画中画功能是视频剪辑中实现多画面同步播放的关键技术。我们经常看到视频中有多个画面同时进行，这些画面可能并不在同一时间线上，这种效果就是通过画中画实现的。在剪映中，许多特效视频也借用了这个功能，通过将两个或多个视频素材叠加在同一时间轨道上同时播放，来增强视频的表现力。

![](img/ca3a95c33e3e11bcb76c8152062ad0a2_1.png)

使用画中画的一个核心特点是能使视频内容变得更加丰富和有趣。

## 画中画功能实操

![](img/ca3a95c33e3e11bcb76c8152062ad0a2_3.png)

上一节我们介绍了画中画的概念，本节中我们来看看如何在剪映中实际操作。

![](img/ca3a95c33e3e11bcb76c8152062ad0a2_5.png)

首先，打开剪映APP，点击“开始创作”，导入一张照片作为背景素材。

![](img/ca3a95c33e3e11bcb76c8152062ad0a2_1.png)

![](img/ca3a95c33e3e11bcb76c8152062ad0a2_7.png)

导入照片素材后，我们需要先调整视频画面的比例。点击底部工具栏的“比例”，选择 **9:16** 的竖屏比例，并调整照片位置以适应画面。

![](img/ca3a95c33e3e11bcb76c8152062ad0a2_9.png)

![](img/ca3a95c33e3e11bcb76c8152062ad0a2_7.png)

![](img/ca3a95c33e3e11bcb76c8152062ad0a2_11.png)

接着，我们将为这个背景添加画中画效果。回到一级菜单，点击“画中画”功能，然后选择“新增画中画”。

![](img/ca3a95c33e3e11bcb76c8152062ad0a2_12.png)

![](img/ca3a95c33e3e11bcb76c8152062ad0a2_11.png)

![](img/ca3a95c33e3e11bcb76c8152062ad0a2_14.png)

在素材库中，我们选择一个星空特效视频作为画中画素材导入。

![](img/ca3a95c33e3e11bcb76c8152062ad0a2_16.png)

![](img/ca3a95c33e3e11bcb76c8152062ad0a2_14.png)

![](img/ca3a95c33e3e11bcb76c8152062ad0a2_17.png)

导入后，时间轴上会出现两条轨道：主轨道（背景照片）和画中画轨道（星空特效）。此时，两个画面已经在同一时间线上重叠。

![](img/ca3a95c33e3e11bcb76c8152062ad0a2_18.png)

![](img/ca3a95c33e3e11bcb76c8152062ad0a2_18.png)

以下是处理画中画素材的基本步骤：

1.  **修剪素材**：选中画中画轨道，将时间线光标移动到需要开始的位置，点击“分割”，删除前面多余的部分。
2.  **调整时长与位置**：拖动画中画素材的尾部以调整其持续时间，并拖动整个片段以调整它在时间线上的位置。

现在，我们想要将星空特效融合到背景照片上，但特效视频带有黑色背景。以下是去除黑色背景的方法：

1.  选中画中画轨道（星空特效）。
2.  在底部工具栏中找到并点击“混合模式”。
3.  在混合模式选项中，选择 **绿色** 模式。这个模式会过滤掉黑色部分，只保留发光的星星特效。

![](img/ca3a95c33e3e11bcb76c8152062ad0a2_20.png)

![](img/ca3a95c33e3e11bcb76c8152062ad0a2_20.png)

操作完成后，黑色背景消失，星空特效便完美叠加在照片之上。你可以通过双指在预览窗口缩放和移动，来调整画中画画面的大小和位置。

最后，播放预览整个视频效果。这样，我们就通过画中画功能，为一张静态照片添加了动态的星空特效。

## 功能总结与延伸

本节课中我们一起学习了剪映的画中画功能。总结其核心操作如下：

*   **功能入口**：在一级菜单点击 **画中画** -> **新增画中画**。
*   **核心效果**：在原视频轨道上新增独立的视频轨道，实现多轨道视频的同步播放。
*   **关键处理**：使用 **混合模式**（如绿色）可以去除画中画素材的纯色背景，实现更好的融合效果。

理解和掌握画中画功能，对我们后期制作更复杂的视频效果大有帮助。就像本节课的案例，我们实现了简单的特效叠加。未来在制作三连屏、多屏互动甚至人影分身等高级效果时，其核心技术都依赖于画中画功能的多轨道叠加与处理能力。
