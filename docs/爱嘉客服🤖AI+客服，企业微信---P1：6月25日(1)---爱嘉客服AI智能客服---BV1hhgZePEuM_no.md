# 爱嘉客服AI智能客服教程：P1：企业微信群聊中触发AI客服回复的两种模式 🛠️

在本节课中，我们将学习如何在企业微信的外部群聊中配置爱嘉客服AI，使其能够智能地识别并回复消息。核心在于理解AI客服的两种触发模式：**“@后回答”** 与 **“全部回答”**。

我们将通过模拟客户（手机端）与客服（电脑端）的交互场景，清晰地展示这两种模式的工作机制与设置方法。

## 模式一：需“@”触发的回复模式 @

现在，我们测试AI客服在企业微信群聊中的回复逻辑。

AI客服系统检测到群聊中出现了新消息。

此时，AI客服发现这条消息**并没有“@”提及它**。因此，AI客服不会做出任何回复。这是一种避免群聊信息干扰的精准响应模式。

![](img/cf1e41de325d20be07ce0a2f59a01674_10.png)

那么，如何让AI客服在群聊中回答呢？我们尝试发送一条内容相同、但特意“@”了AI客服的消息。

使用相同的问题去提问，但这次“@”了AI客服。可以看到，AI客服立即对客户的问题进行了回答。

## 模式二与模式切换：全局响应与灵活配置 ⚙️

除了被“@”后才回复，AI客服也支持另一种模式：回答群聊中的所有问题。

系统提供了一个类似开关的配置选项，允许您为每个群聊选择工作模式。
其逻辑可以简化为：
```python
if message.is_mentioned(@AI_customer_service):
    reply()
elif group.setting == “reply_all”:
    reply()
```
您可以根据需要，为不同群聊设置是“全部回答”还是“艾特后再回答”。

此外，客服系统的运行**不需要**将鼠标光标固定在窗口上。您可以随意移动鼠标，处理其他工作，AI客服会在后台自动运行，不影响您的多任务操作。

## 总结 📝

本节课中，我们一起学习了爱嘉客服AI在企业微信群聊中的两种核心响应模式：
1.  **“@后回答”模式**：仅在消息中“@”AI客服时，才会触发回复，确保回复的精准性。
2.  **“全部回答”模式**：AI客服将响应群聊中的所有消息。

您可以通过灵活的开关为每个群聊单独配置模式。该系统在后台自动运行，不占用您的操作焦点。如果您需要为您的企业定制专属的AI客服解决方案，欢迎联系我们。
