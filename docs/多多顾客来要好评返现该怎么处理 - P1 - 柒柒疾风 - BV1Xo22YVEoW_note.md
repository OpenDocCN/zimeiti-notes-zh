# 拼多多运营教程：P1：如何处理顾客索要好返现

在本节课中，我们将学习在拼多多平台上，当顾客主动索要“好评返现”时，商家应如何合规、有效地进行应对。我们将重点了解平台规则、标准回复话术以及处理恶意差评的方法。

## 📜 平台规则解读

上一节我们明确了本课的学习目标，本节中我们来看看拼多多平台关于“好评返现”的具体规定。

听说现在拼多多好评返现会违规。从9月1号起，这个规定就开始执行了，明确规定了电商法实施好评返现是不正当竞争行为。如果查到了，处罚金额是20万到200万的罚款。很多买家可能还不知道这个规定。

## 💬 标准客服应对流程

了解了平台的严格规定后，我们来看看当顾客主动询问时，客服应该如何进行标准回复。

当买家找客服问有没有好评返现的时候，客服不要急着回复有或没有。客服应该先询问买家对产品是否满意。

以下是针对买家不同反馈的标准回复策略：

*   **如果买家表示满意**：客服可以回复“不好意思，亲，我们产品利润低，没有这种活动。”
*   **如果买家表示不满意**：客服可以回复“小店只有‘评价有礼’活动，可以申请给您补差价。”请注意，不要说多余的话。

## ⚠️ 应对恶意差评与售后问题

掌握了标准话术后，我们还需要知道如果买家因未获得返现而给出差评，该如何处理。

如果买家直接给差评，商家可以去平台举报该评价。举报时选择“利用评价要挟”的选项，并把相关的聊天记录作为证据上传，平台审核后就有可能删除该差评。

此外，对于其他处理不了的售后问题，平台可能会对商家进行扣分等处罚。
