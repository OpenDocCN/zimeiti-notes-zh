# 新媒体运营系统课：P8：用户定位 🎯

在本节课中，我们将要学习新媒体运营中至关重要的一环——用户定位。理解并精准定位你的目标用户，是创作出受欢迎内容、实现账号增长与商业变现的基础。

## 理解用户定位与内容分发机制

用户定位的核心是思考：我们准备给什么样的人，创作什么样的作品。

要理解这一点，首先需要了解抖音平台的内容分发过程。这个过程可以概括为一个循环：

1.  **创作与发布**：你创作内容并在抖音发布。
2.  **平台分析与初次推荐**：抖音平台会对你的账号和作品进行综合分析，并结合用户的历史兴趣与行为，将你的视频**初次推送给可能感兴趣的用户**。
3.  **用户行为反馈**：用户看到视频后，会根据观看体验产生**点赞、评论、转发、关注**等行为反馈。
4.  **扩大推荐**：抖音系统根据这些正向的行为反馈，判断视频内容质量，进而将视频**推送给更多具有相似特征的用户**。

这个循环持续进行，便可能诞生一个热门视频。因此，想要创作出有潜力的热门内容，**首先必须对你的目标用户进行深入分析**。

![](img/5dc13b3c1f1ac2392852cc99b6a4f1d6_1.png)

当然，运营的终极目的是变现，而非单纯追求播放量。但视频播放量是变现的前提。因此，在分析目标用户时，必须将**核心用户的商业价值**考虑进去。你需要针对核心用户，将视频内容的价值进行拆解，并在不同维度上进行精细化策划与设计，以确保吸引到的人群足够精准。

## 抖音平台用户画像概览

在深入定位前，我们先了解抖音平台的整体用户现状。抖音日活跃用户已突破7亿，人均每日使用时长超过2小时，这意味着在任何时间、任何领域都有海量用户在活跃。

从整体特征来看，抖音用户画像具有以下特点：

*   **性别与年龄**：女性用户占比约55%，18-30岁的用户占比高达71%。
*   **城市分布**：超过60%的用户分布在一、二线城市。
*   **兴趣偏好**：教育、美食、旅游、搞笑、娱乐是总体兴趣偏好的前五名，其中教育和美食占比最高。
*   **消费动机**：年轻群体（特别是18-30岁）的线上购物动机中，91.8%是“因为自己喜欢或需要而购买”。
*   **内容偏好**：这部分年轻群体偏好的内容或人物形象特质主要包括：
    1.  令人心情愉悦或搞笑幽默的。
    2.  高颜值、可爱、漂亮的。
    3.  具备爱国情怀与社会正能量的。
    4.  展现认真生活与处事态度的。

因此，在创作视频或打造个人IP时，可以参考以上内容特质进行设计。

## 如何构建目标用户画像

![](img/5dc13b3c1f1ac2392852cc99b6a4f1d6_5.png)

明确了平台大盘情况后，我们来学习如何定位自己的目标用户。用户数据可以参考行业竞争对手、对标账号，或来自线下及过往的企业消费者数据。

构建目标用户画像，应从以下几个基础维度进行分析：

以下是用户画像的基础分析维度列表：

*   **性别**：男、女。
*   **年龄**：18岁及以下、18-28岁、28-38岁、38-48岁、48-58岁、58岁及以上。
*   **地域**：一线城市、二线城市、三线城市、四线城市等。
*   **人群特征**：日常白领、熬夜人群、家庭主妇、学生等。
*   **消费水平**：经济型商品购买者、豪华型商品购买者。
*   **兴趣标签（抖音平台）**：抖音将用户兴趣划分为17个大类，包括：金融、汽车、科技、娱乐、服饰、教育、母婴、法律、体育、游戏、美食、生活、家装、美妆、旅游、房产、商务服务。

对于电商类账号，还需额外考虑以下维度：

以下是电商类用户需考虑的附加维度列表：

*   **购买时间偏好**：上午、中午、下午、晚上。
*   **价格敏感度**：高、中、低。
*   **囤货偏好**：高、中、低。

## 理解用户的消费旅程：5A人群模型

对于品牌商家而言，抖音平台还提供了更精细的“5A人群分层”模型，用于描述用户与品牌关系的不同阶段：

以下是抖音5A人群模型详解列表：

*   **A1（了解感知）**：即“我知道”阶段。用户被动接收品牌信息，是品牌知名度的主要来源。
*   **A2（吸引）**：即“我喜欢”阶段。用户对品牌产生兴趣，品牌需要创造短期记忆或强化长期记忆。
*   **A3（问询）**：即“我相信”阶段。用户被引发好奇，主动收集品牌相关信息，是**最接近成交转化、转化率最高**的环节。
*   **A4（行动）**：即“我购买”阶段。用户完成购买行为，并进入使用、售后等环节。
*   **A5（拥趸）**：即“我推荐”阶段。用户成为品牌忠实粉丝，表现为复购、留存率高，并乐于向他人推荐品牌。

此外，还有两类重要人群：

*   **O人群（机会人群）**：在平台内多个相关兴趣领域活跃，但尚未与品牌产生联系的**潜在人群**，等待被吸引和转化。
*   **A1-A3人群关系**：A1是被动感知人群，经多次触达可转化为A2（轻交互人群）；A2经立体化内容触达，可转化为A3（深度交互人群）；A5则是品牌的私域用户，会关注账号、发表正面评论等。

## 进阶：从人格特质角度洞察用户

![](img/5dc13b3c1f1ac2392852cc99b6a4f1d6_9.png)

除了以上客观维度，我们还可以从更主观的“用户人格”角度进行描述，这对内容创作具有直接的指导意义。当某些行业数据难以获取时，这种方法尤为有效。

可以参考人格分析理论，针对不同类型的人格特点产出相应内容：

以下是针对不同人格特质的内容创作建议列表：

*   **忠诚型人格**：这类人小心谨慎、偏保守。视频内容应**强调真实感、接地气**，更容易获得他们的信任。
*   **理想主义者**：这类人偏感性。视频内容应**侧重提供情绪价值**，引发情感共鸣。
*   **艺术家型人格**：这类人乐于接受新体验。可以发布**富有创意或挑战体验型**的内容。
*   **学者型人格**：这类人对知识类内容更感兴趣，并且**互动意愿较高**。可提供干货知识、深度分析等内容。
