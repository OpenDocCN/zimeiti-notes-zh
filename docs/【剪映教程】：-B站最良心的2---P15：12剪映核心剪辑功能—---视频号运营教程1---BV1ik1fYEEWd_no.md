# 剪映教程：P15：视频过渡转场功能详解 🎬

在本节课中，我们将学习剪映的核心剪辑功能之一——视频过渡转场。转场是连接两个视频镜头的重要技巧，能让视频衔接更自然流畅，避免生硬感。通过学习使用剪映内置的转场效果，即使是初学者也能制作出节奏感强、视觉效果出众的短视频。

![](img/8430d33da1ad9cd19373bf987b7fad0f_1.png)

## 什么是视频过渡转场？ 🔄

![](img/8430d33da1ad9cd19373bf987b7fad0f_3.png)

视频过渡转场是指在两个视频镜头之间，为了使其衔接更顺畅而采用的视频处理技巧或剪辑方法。在短视频制作中，技术流创作者常运用各种转场和运镜技巧，制作出炫酷的视觉效果。在剪映中，我们可以利用软件提供的多种转场功能，轻松提升视频的专业度和吸引力。

![](img/8430d33da1ad9cd19373bf987b7fad0f_5.png)

## 实战操作：制作卡点视频 🎵

![](img/8430d33da1ad9cd19373bf987b7fad0f_7.png)

上一节我们介绍了转场的基本概念，本节我们将通过制作一个卡点视频的实例，来学习如何具体应用这些功能。

![](img/8430d33da1ad9cd19373bf987b7fad0f_9.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_10.png)

### 第一步：导入素材与音乐

首先，创建一个新项目并导入几张照片作为视频素材。接着，为视频添加背景音乐。卡点视频通常需要节奏感强的音乐。

![](img/8430d33da1ad9cd19373bf987b7fad0f_12.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_13.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_15.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_16.png)

以下是操作步骤：
1.  点击 **`音频`** 按钮。
2.  选择 **`抖音收藏`**，导入一首节奏感强的音乐。
3.  点击 **`使用`** 按钮，将音乐添加到时间轴。

![](img/8430d33da1ad9cd19373bf987b7fad0f_17.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_19.png)

### 第二步：为音乐踩点

![](img/8430d33da1ad9cd19373bf987b7fad0f_21.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_23.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_25.png)

卡点视频的核心是让画面切换与音乐节拍同步。我们需要为音乐标记出节奏点。

![](img/8430d33da1ad9cd19373bf987b7fad0f_27.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_29.png)

以下是操作步骤：
1.  选中音频轨道。
2.  点击底部工具栏的 **`踩点`** 按钮。
3.  开启 **`自动踩点`** 功能，选择节拍更密集的 **`节拍2`** 选项。
4.  系统会自动在节奏点上生成标记点，点击 **`√`** 确认。

![](img/8430d33da1ad9cd19373bf987b7fad0f_31.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_33.png)

### 第三步：调整素材时长

![](img/8430d33da1ad9cd19373bf987b7fad0f_35.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_37.png)

我们需要让每张照片的持续时间与一个音乐节拍对齐。

![](img/8430d33da1ad9cd19373bf987b7fad0f_38.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_40.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_42.png)

以下是操作步骤：
1.  选中第一张照片，拖动其右边缘，使其结束时间线与第一个节奏标记点对齐。
2.  重复此操作，依次调整后续所有照片的时长，使它们各自对应一个节拍。

![](img/8430d33da1ad9cd19373bf987b7fad0f_43.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_45.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_46.png)

### 第四步：添加转场效果

![](img/8430d33da1ad9cd19373bf987b7fad0f_48.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_49.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_51.png)

这是让视频变得炫酷的关键步骤。我们在每两张照片之间的衔接处添加转场效果。

![](img/8430d33da1ad9cd19373bf987b7fad0f_52.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_54.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_56.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_57.png)

以下是操作步骤：
1.  放大时间轴，以便更清晰地看到照片之间的衔接点（显示为小方块）。
2.  点击两个素材之间的小方块，进入转场选择界面。
3.  剪映提供了丰富的转场类型，如 **`基础转场`**、**`运镜转场`**、**`幻灯片`**、**`特效`** 等。
4.  本例中，我们选择 **`运镜转场`** 类别，并依次为每个衔接点选择不同的效果（如“推进”、“拉远”等）。

![](img/8430d33da1ad9cd19373bf987b7fad0f_59.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_60.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_62.png)

### 第五步：统一画幅与添加特效

![](img/8430d33da1ad9cd19373bf987b7fad0f_63.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_64.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_65.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_66.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_67.png)

为了使视频观感更统一、更丰富，我们还需要进行两项优化。

![](img/8430d33da1ad9cd19373bf987b7fad0f_69.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_70.png)

以下是操作步骤：
1.  **调整画幅比例**：点击 **`比例`**，选择 **`9:16`** 的竖屏比例。然后逐一选中每张照片，在预览窗口中调整其大小和位置，使其适配画布。
2.  **添加视频特效**：点击 **`特效`** 按钮，选择喜欢的特效（如“变清晰”、“抖动”等），将其拖动到时间轴的特效轨道上，并调整其时长和位置，以增强视觉冲击力。

![](img/8430d33da1ad9cd19373bf987b7fad0f_72.png)

![](img/8430d33da1ad9cd19373bf987b7fad0f_74.png)

## 效果预览与总结 🎉

![](img/8430d33da1ad9cd19373bf987b7fad0f_76.png)

完成以上所有步骤后，再次预览视频。你会发现，添加了转场和特效的卡点视频，其节奏感和视觉流畅度相比最初有了巨大提升。

![](img/8430d33da1ad9cd19373bf987b7fad0f_78.png)

本节课中，我们一起学习了剪映的视频过渡转场功能。**其核心操作是：点击时间轴上两个素材之间的衔接点（小方块），即可从丰富的效果库中选择并应用转场。** 通过巧妙地运用基础、运镜、幻灯片等各类转场，并结合音乐踩点，我们可以显著提升视频的观赏性和专业度，这对于制作流行的卡点视频尤为重要。
