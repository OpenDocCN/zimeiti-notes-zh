# 微信视频号运营从0~1全攻略：P13：如何填写邀请函

在本节课中，我们将学习如何正确填写视频号创作者计划的邀请函。这是一个关键步骤，关系到您能否成功加入计划并开始运营。整个过程简单明了，只需按照步骤操作即可。

![](img/89e8c2bdfbbb1f273c515116c5747ac3_1.png)

上一节我们介绍了视频号的基础注册，本节中我们来看看如何填写后续的邀请信息。

## 第一步：查找视频号账号与粉丝数

![](img/89e8c2bdfbbb1f273c515116c5747ac3_3.png)

![](img/89e8c2bdfbbb1f273c515116c5747ac3_4.png)

首先，您需要找到自己视频号的账号昵称和粉丝数量。以下是具体操作步骤。

![](img/89e8c2bdfbbb1f273c515116c5747ac3_6.png)

1.  **打开微信应用**：在您的手机上找到并打开微信软件。

2.  **进入“发现”页面**：在微信底部导航栏中，点击“发现”选项。

![](img/89e8c2bdfbbb1f273c515116c5747ac3_8.png)

![](img/89e8c2bdfbbb1f273c515116c5747ac3_9.png)

3.  **找到“视频号”入口**：在“发现”页面中向上滑动，找到并点击“视频号”入口（图标为黄色蝴蝶形状）。

4.  **进入个人主页**：点击视频号页面右上角的个人头像图标，进入个人主页。

5.  **查看账号信息**：在个人主页中向下滑动，找到“我的视频号”区域。此处会明确显示您的**账号昵称**和**粉丝数量**。
    *   **账号昵称**：例如 `天道酬勤`
    *   **粉丝数量**：例如 `39917`

## 第二步：填写邀请函信息

找到账号信息后，即可开始填写邀请函。邀请函通常包含以下几个必填项，请确保信息准确无误。

以下是需要填写的核心信息项：

![](img/89e8c2bdfbbb1f273c515116c5747ac3_11.png)

![](img/89e8c2bdfbbb1f273c515116c5747ac3_12.png)

![](img/89e8c2bdfbbb1f273c515116c5747ac3_13.png)

1.  **视频号账号昵称**：填写您在上一步中查看到的昵称，例如 `天道酬勤`。
2.  **粉丝数量**：填写您当前的粉丝总数，例如 `39917`。
3.  **活动邀请人**：根据您的加入渠道，填写对应的邀请人姓名，例如 `张良老师`。
4.  **实名信息**：此处必须填写您本人的真实姓名，例如 `周建国`。
5.  **结算手机号**：填写用于接收验证码和后续结算的手机号码，例如 `13812345678`。**注意**：号码中间不能有空格。

## 第三步：提交与确认

所有信息填写完毕后，请进行最后一步操作。

![](img/89e8c2bdfbbb1f273c515116c5747ac3_15.png)

![](img/89e8c2bdfbbb1f273c515116c5747ac3_16.png)

1.  **点击提交**：检查所有信息无误后，点击页面上的蓝色“提交”按钮。
2.  **确认提交**：系统会弹出“确定要提交吗？”的确认框，请务必点击“确认”完成提交。
3.  **提交成功**：成功后，页面会显示“已成功加入创作者分成计划”的提示。

**关键提醒**：提交过程就像出门锁门，必须点击“确认”才算最终完成，切勿遗漏此步骤。
