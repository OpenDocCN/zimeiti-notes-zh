## 分析爆款笔记的核心要点
以下是分析一篇爆款笔记时，需要让AI重点关注的几个方面：

![](img/a0b5aa4a0c2765b8b212dd9536e7b16f_18.png)

![](img/a0b5aa4a0c2765b8b212dd9536e7b16f_20.png)

![](img/a0b5aa4a0c2765b8b212dd9536e7b16f_22.png)

1.  **爆款因素分析**：分析内容中可能导致其成为爆款的点。例如，是否引发了情感共鸣、提供了实用价值或讲述了逆袭故事。
2.  **内容结构分析**：拆解文案的框架。
    *   **开篇**：用什么方式（如提问、制造焦虑、引用名人）吸引用户继续阅读。
    *   **中间**：提供了哪些内容来戳中用户痛点，提升用户停留时长。
    *   **结尾**：设置了什么“钩子”（如鼓励互动、提供联系方式）来促使用户私信或关注。
3.  **关键词分析**：检查笔记中是否刻意植入了特定关键词。
    *   如果有，关键词是什么？
    *   关键词的出现频率和分布结构是怎样的？

### 第一步：收集并输入爆款文案
首先，需要找到目标领域的爆款笔记，并将其全文复制下来。例如，我们找到一篇点赞数3000+的考研备考笔记。

![](img/a0b5aa4a0c2765b8b212dd9536e7b16f_36.png)

![](img/a0b5aa4a0c2765b8b212dd9536e7b16f_38.png)

![](img/a0b5aa4a0c2765b8b212dd9536e7b16f_40.png)

**提示词示例**：
```
我给你提供一篇小红书平台上关于考研的爆款笔记文案，你需要分析：
1. 爆款因素（例如：情感共鸣、实用建议、逆袭故事等）。
2. 文案的整体结构（开篇如何吸引用户、中间如何阐述痛点、结尾如何引导互动）。
3. 分析该笔记是否有结合中文分词法设计关键词。如果有，请指出关键词及其出现频率；如果没有，请说明。
```

### 第二步：解读AI的初步分析
AI会返回类似以下的分析结果：
*   **爆款因素**：情感共鸣（开篇制造焦虑）、逆袭故事、提供详细的备考规划、使用表情符号增加亲和力、结尾给予正能量鼓励。
*   **结构分析**：开篇用自身经历引发共鸣，中间分科目给出具体方法和时间节点，结尾鼓励并引导互动。
*   **关键词分析**：本篇文案没有明显刻意设计的关键词，词汇均为常用词。

**注意**：AI（特别是3.5版本）可能在分析多篇内容后“遗忘”最初的任务指令。如果发现分析偏离方向，需要重新发送或强调最初的指令要求。

### 第三步：整合爆款要素
在让AI分析多篇爆款笔记后，需要指令它将这些分析结果进行整合去重，总结出该领域爆款笔记的通用要素列表。

**提示词示例**：
```
请将你前面分析过的几篇考研爆款笔记的爆款因素总结一下，整合成一个去重后的列表。
```

AI可能会总结出如下要点：
*   情感共鸣与逆袭故事引入
*   明确的时间规划和备考节点
*   分科目（政治、英语、专业课）的针对性建议
*   具体的资源与技巧分享
*   正能量鼓励与互动引导

![](img/a0b5aa4a0c2765b8b212dd9536e7b16f_53.png)

### 第四步：生成新笔记大纲与文案
基于总结的爆款要素，现在可以指导AI生成新的笔记。

![](img/a0b5aa4a0c2765b8b212dd9536e7b16f_54.png)

![](img/a0b5aa4a0c2765b8b212dd9536e7b16f_56.png)

**首先，生成大纲**：
```
请结合你总结的考研爆款笔记因素，为我规划一篇新的小红书笔记大纲。大纲需要细分到具体的小节点，待我确认后再生成完整文案。
```
AI生成的大纲可能包括：吸引人的开篇、主题介绍、专业选择分析、各科目备考规划、时间管理、心态调整、互动引导等部分。

**接着，生成完整文案**：
```
请按照上面确认的大纲，撰写完整的文案。要求：
1. 严格遵循爆款笔记的结构和要素。
2. 在开篇融入一个简单的“逆袭”小故事，引发读者共鸣。
3. 文案中要包含适量的emoji表情。
4. 尝试以中文分词法的思维，自然融入“考研全流程”、“考研政治”等关键词。
```

### 第五步：优化与进阶
AI生成的初稿可能比较基础，对于运营人员来说可能足够使用，但仍有优化空间。你可以：
*   手动调整文案，使其更精炼或更丰富。
*   使用更强大的模型（如GPT-4）获得更优效果。
*   利用**自定义GPTs**功能，创建一个专属的“小红书文案助手”。你可以上传平台规范、优质案例、常用表情包等资料让它学习，从而更稳定地产出符合你要求的文案。

## 总结
本节课我们一起学习了利用AI辅助内容创作的核心流程：
1.  **喂资料与分析**：提供爆款内容让AI学习其风格和成功要素。
2.  **要素提炼**：指令AI总结出可复用的爆款公式。
3.  **内容生成**：基于提炼的公式，让AI生成具备爆款潜质的新内容大纲和文案。
4.  **迭代优化**：结合人工判断进行优化，或利用高级工具（如GPTs）打造专属助手。
