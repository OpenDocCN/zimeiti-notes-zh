# 小红书运营：P45：43、新手小红书运营-小红书店铺规划（2）

![](img/5b51ae51555c8616c584fa599095d9b7_1.png)

在本节课中，我们将要学习小红书店铺运营的中期与后期规划。我们将了解中期需要关注的各种排名机制，以及后期发展的两个主要方向。这些知识将帮助你理解如何系统性地提升店铺表现。

![](img/5b51ae51555c8616c584fa599095d9b7_3.png)

![](img/5b51ae51555c8616c584fa599095d9b7_5.png)

上一节我们介绍了店铺运营的前期规划，本节中我们来看看中期和后期我们应该如何操作。整体来说，中期规划的重点非常多，内容也较为复杂。

![](img/5b51ae51555c8616c584fa599095d9b7_7.png)

![](img/5b51ae51555c8616c584fa599095d9b7_1.png)

![](img/5b51ae51555c8616c584fa599095d9b7_9.png)

中期规划的核心是提升店铺在各种排名中的位置，从而获得更多平台给予的20%的基础流量曝光。如果你的前期数据属性没有做好，中期规划的操作会非常困难。

我们先来了解一下中期规划包含哪些内容。

![](img/5b51ae51555c8616c584fa599095d9b7_7.png)

中期规划的内容体系较大，主要包括：综合排名分类、价格排名、销量排名、关键词排名以及个性化搜索。此外，还涉及动态PK与静态PK的选项。

![](img/5b51ae51555c8616c584fa599095d9b7_11.png)

这些内容都与店铺整体表现密切相关。由于每个部分都可以展开成独立课程，这里我们主要进行概括性讲解。

![](img/5b51ae51555c8616c584fa599095d9b7_13.png)

首先是**综合排名分类**。这是平台默认的宝贝排名状态，主要根据店铺的综合因素对商品进行排序。

它的考核维度非常多，包括：
*   商品的描述质量
*   商品主图的质量
*   单品的综合数据
*   店铺的整体交易情况
*   是否能满足个性化搜索趋势下的客户需求

![](img/5b51ae51555c8616c584fa599095d9b7_15.png)

综合排名具体细分为16项（目前为15项），你的店铺在这些项目中的排名越高，能抢到的20%基础流量的概率就越大。

![](img/5b51ae51555c8616c584fa599095d9b7_17.png)

以下是综合排名涉及的16项关键因素：
1.  人气
2.  销量
3.  信誉
4.  价格
5.  新店与新品扶持
6.  商城优先级
7.  下架时间
8.  标题优化
9.  橱窗推荐
10. 转化率
11. 收藏率
12. 回头客比例
13. 好评率
14. 交易数量
（注：有一项已被删除，现为15项）

![](img/5b51ae51555c8616c584fa599095d9b7_19.png)

![](img/5b51ae51555c8616c584fa599095d9b7_15.png)

接下来是**价格排名**和**销量排名**。

价格排名是根据商品价格进行排序的机制。同价格商品的排名会受到宝贝关键词匹配度和类目相关性的影响。

价格排名系统主要通过以下几点来确定产品排名：
*   **价格竞争力**：你的定价在同类商品中是否有优势。
*   **价格真实性**：是否频繁、大幅度修改价格。
*   **价格稳定性**：价格是否保持稳定。
*   **价格与销售量的平衡**：定价是否与销量水平相匹配。

![](img/5b51ae51555c8616c584fa599095d9b7_21.png)

![](img/5b51ae51555c8616c584fa599095d9b7_23.png)

如果这几个点把握不好，排名系统就可能失效。

![](img/5b51ae51555c8616c584fa599095d9b7_25.png)

![](img/5b51ae51555c8616c584fa599095d9b7_21.png)

然后是**销量排名**。买家经常通过商品销量来排序，这能在一定程度上反映宝贝的人气。

![](img/5b51ae51555c8616c584fa599095d9b7_27.png)

销量排名高的商品如果排名靠后，可能是由于类目不精准导致的。影响销量排名的因素包括：
*   销量的增长趋势
*   类目的相关性
*   商品的转化率
*   商品的评价内容

![](img/5b51ae51555c8616c584fa599095d9b7_29.png)

![](img/5b51ae51555c8616c584fa599095d7_27.png)

![](img/5b51ae51555c8616c584fa599095d9b7_31.png)

最后是**关键词排名**和**个性化搜索**。

关键词排名非常细致。例如，搜索“T恤”、“男士T恤”、“女士T恤”都是不同的关键词，各自有独立的排名系统。你需要优化特定关键词的属性，才能获得该关键词下的曝光。

个性化搜索则是平台根据用户属性与店铺属性的匹配度，将你的商品推荐给更精准的人群。个性化搜索做得越好，用户越精准，后续的数据表现也会越好。

关于**动态PK与静态PK**，这部分内容较为深入，通常在小红书店铺流量达到第四或第五层级时才需要考虑，前期效果有限，此处暂不展开。

由于中期规划内容实在太多，无法在此详细拆解每一步。后期会单独开设大课时进行系统分享。

了解了中期规划后，我们来看后期规划。后期规划主要围绕店铺发展到一定阶段后的两个发展方向。

![](img/5b51ae51555c8616c584fa599095d9b7_33.png)

![](img/5b51ae51555c8616c584fa599095d9b7_33.png)

后期规划主要有两个节点：**加大力度投入**和**开新店**。

**1. 加大力度投入**
当你认为店铺还有提升空间时，可以从以下五个方面进行突破，从小红书中层用户向大用户发展：
*   数据分析
*   店铺分析
*   数据掌控
*   店铺优化
*   数据维护与瓶颈突破

这个阶段通常意味着要开始进行付费推广，以实现从第五阶段到第六阶段的跨越。

**2. 开新店**
如果你觉得当前店铺已达到产品上限，不想冒险加大投入，则可以考虑开新店。
*   吸取第一个店铺的运营经验和思路。
*   经营类似商品，并按照已验证的模式操作。
*   第二个店铺的效果通常会比第一个更好。
*   在拥有成熟的运营思路、店铺属性和一定资金后，再考虑是否对新店加大推广力度。

做好基础规划后，对类似商品进行小范围调整即可。这就是中期和后期的整体规划思路。

![](img/5b51ae51555c8616c584fa599095d9b7_35.png)

本节课中我们一起学习了小红书店铺的中期与后期规划。中期规划的核心是理解并优化**综合排名**、**价格排名**、**销量排名**、**关键词排名**和**个性化搜索**，以提升在平台流量池中的竞争力。后期规划则提供了**加大投入**或**开设新店**两种发展路径。这些内容较为体系化，建议在打好前期基础后，再针对性地深入研究。下一节，我们将讲解小红书运营的其他相关内容。
