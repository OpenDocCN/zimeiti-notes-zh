# 拼多多商家教程：P1：技术服务费退还政策解读 🛒

在本节课中，我们将详细解读拼多多平台关于“退款订单自动退还技术服务费”的最新政策。我们将了解政策的具体内容、适用范围、生效时间以及相关的限制条件。

## 政策核心内容

上一节我们介绍了课程概述，本节中我们来看看政策的核心变动。拼多多平台已将“0.6%基础服务费自动退款”的适用范围，从原来的活动商品扩大到了平台上的全部商品。

## 生效时间与重要例外

了解了核心内容后，我们需要注意政策的生效时间和关键例外情况。该政策已于9月24日凌晨起全面生效。

但是，存在以下两个重要的不支持情况：
*   多多国际的商品订单暂不支持此政策。
*   由买家原因导致的订单退款，也暂不支持退回服务费。

## 近期其他利好政策

除了技术服务费退还政策，拼多多平台近期还推出了一系列对商家利好的调整。以下是部分主要政策列表：
*   商家贷款提现门槛由100元调至1元。
*   全部类目的基础保证金由1000元调至500元。
*   “先用后付”订单的基础服务费从1%调至0.6%。
*   偏远地区免费中转，中转费由平台承担。
*   售后申诉不再限制次数。
*   自动返还推广红包等功能。

## 总结

本节课中我们一起学习了拼多多最新的技术服务费退还政策。我们明确了该政策已覆盖全平台商品，并已于9月24日生效，但同时需注意多多国际订单和买家原因退款不支持的例外情况。此外，我们还了解了平台近期推出的一系列降低经营成本、优化体验的利好措施。
