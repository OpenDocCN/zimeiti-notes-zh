# 小红书运营：P45：2.小红书店铺分类 🏪

![](img/457a410e3f1751c1da1a8b2575d24300_1.png)

![](img/457a410e3f1751c1da1a8b2575d24300_3.png)

![](img/457a410e3f1751c1da1a8b2575d24300_5.png)

在本节课中，我们将要学习小红书平台上的店铺分类体系。了解不同类型的店铺及其特点、入驻要求和保证金规则，是开设店铺前必须掌握的基础知识。

上一节我们介绍了小红书运营的基础概念，本节中我们来看看平台具体的店铺类型划分。

小红书平台上的店铺类型主要可以分为以下几种。

以下是小红书主要的店铺类型列表：
*   **个人店**
*   **个体工商户店**
*   **普通企业店**
*   **专卖店**
*   **旗舰店**
*   **卖场型旗舰店**

接下来，我们将逐一详细解析每种店铺类型。

### 个人店
**个人店**的申请主体是个人用户。这适合刚毕业、辞职或想尝试小规模经营的个体，类似于现实中的“摆地摊”。

![](img/457a410e3f1751c1da1a8b2575d24300_7.png)

![](img/457a410e3f1751c1da1a8b2575d24300_9.png)

其经营特点是限定经营零星、小额的商品，例如首饰、服装、包包、手机壳等。它类似于一个“杂货铺”，可以销售多个品牌的产品。

个人店的入驻要求非常简单，核心公式为：
**入驻要求 = 个人身份证**

![](img/457a410e3f1751c1da1a8b2575d24300_11.png)

![](img/457a410e3f1751c1da1a8b2575d24300_13.png)

店铺命名格式由系统自动生成，通常为“专业号名称 + 店名”。关于保证金，个人店涉及**基础保证金**和**浮动保证金**，二者取高值。个人店的基础保证金通常为1000元。

### 个体工商户店
**个体工商户店**与个人店的主要区别在于入驻要求。它需要申请人提供**营业执照**，但对品牌授权没有要求。

除了多出一个营业执照的要求，个体工商户店在其他方面与个人店基本一致。

### 普通企业店
**普通企业店**以及后续更高级的店铺类型，其核心区别都体现在**入驻要求**上。通常，入驻要求越简单，店铺的初始权重可能越低；入驻要求越复杂，店铺的权重和权限可能越高。对于运营新手，建议从要求较低的个人店开始测试。

![](img/457a410e3f1751c1da1a8b2575d24300_15.png)

普通企业店的入驻要求核心是：
**入驻要求 = 企业营业执照**
申请主体必须为企业，即拥有国家认证的公司实体。

![](img/457a410e3f1751c1da1a8b2575d24300_17.png)

![](img/457a410e3f1751c1da1a8b2575d24300_19.png)

![](img/457a410e3f1751c1da1a8b2575d24300_21.png)

### 专卖店、旗舰店与卖场型旗舰店
这三种店铺类型可以放在一起理解，它们主要是名称和授权模式不同。

![](img/457a410e3f1751c1da1a8b2575d24300_23.png)

*   **专卖店**：要求持有营业执照和**品牌认可**（自有品牌或非自有品牌均可）。核心是销售特定品牌的产品。
*   **旗舰店**：要求持有营业执照和**品牌授权书**。通常是品牌方或获得品牌一级授权的经销商开设。
*   **卖场型旗舰店**：要求具备零售资质，并可经营多个品牌的商品。但在小红书平台，这类店铺较少见，因为小红书的用户和流量以女性为主，与大型综合卖场的定位不完全匹配。这反而为中小卖家提供了更友好的竞争环境。

![](img/457a410e3f1751c1da1a8b2575d24300_25.png)

### 保证金规则详解
了解保证金规则对店铺资金管理至关重要。保证金主要包括**基础保证金**和**浮动保证金**。

所有店铺类型的基础保证金通常为**1000元**。不缴纳基础保证金，店铺将无法正常运营。

浮动保证金则根据店铺的月成交金额动态计算。以下是浮动保证金的计算规则：

| 月成交金额区间 | 所需浮动保证金 |
| :--- | :--- |
| ≤ 50,000元 | 0元 (但需已交1000元基础保证金) |
| 50,001元 ~ 100,000元 | 3,000元 |
| 100,001元 ~ 500,000元 | 5,000元 |
| > 500,000元 | 20,000元 |

![](img/457a410e3f1751c1da1a8b2575d24300_27.png)

![](img/457a410e3f1751c1da1a8b2575d24300_29.png)

**重要提示**：平台会在提现时检查保证金账户余额是否充足。如果店铺月成交额超过5万元，但保证金总额（基础+浮动）未达到对应标准，则无法成功提现。务必定期关注保证金状态。

![](img/457a410e3f1751c1da1a8b2575d24300_31.png)

本节课中我们一起学习了小红书的六大店铺类型：个人店、个体工商户店、普通企业店、专卖店、旗舰店和卖场型旗舰店，并重点了解了它们的区别、入驻要求以及关键的保证金计算规则。掌握这些知识，能帮助你根据自身情况选择合适的店铺类型，并做好资金规划。下一节课，我们将探讨如何在小红书进行选品与寻找货源。
