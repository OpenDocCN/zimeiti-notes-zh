# 抖音全套运营教程：P21：10、如何寻找文案-1

在本节课中，我们将要学习如何解决抖音内容创作中的核心难题之一——文案创作。对于初学者而言，写出吸引人的文案可能很困难。本节将介绍几个实用的在线工具，帮助你轻松找到并生成优质文案，甚至解决配音问题，让你的视频内容更具吸引力。

---

上一节我们讨论了优化作品以提升吸粉能力的重要性。本节中，我们来看看如何具体解决文案创作这个首要问题。

很多同学表示不会写文案。没有关系，我们可以借助一些现成的工具和平台。以下是五个实用的文案素材网站，它们都是网页版，无需下载APP，在手机或电脑浏览器中搜索即可使用。

*   **淘金阁**

接下来，我们以“淘金阁”为例，详细讲解如何使用。

这是淘金阁网站的首页，上方是它的网址。如果你直接搜索“淘金阁”广告太多，可以直接输入这个网址访问。

![](img/fcf4887952493315f0d0c44165445a03_1.png)

![](img/fcf4887952493315f0d0c44165445a03_3.png)

我们可以根据自己账号的定位，选择相应的分类。例如，点击“情感”类型。

![](img/fcf4887952493315f0d0c44165445a03_3.png)

进入后，你会看到大量情感类的话题文案，并且通常有多页内容。之前提到的五个平台，足够你日常使用了。

![](img/fcf4887952493315f0d0c44165445a03_5.png)

当你面对众多文案不知如何选择时，可以套用之前教过的公式：**一个群体 + 一个核心话题 + 疑问**。觉得合适的文案就可以选用。

![](img/fcf4887952493315f0d0c44165445a03_5.png)

![](img/fcf4887952493315f0d0c44165445a03_7.png)

通过这个方法，你基本不会再缺文案素材。

2.  使用页面下方的“输入多个关键词”进行免费搜索。

例如，我们搜索“宝妈”这个关键词。

解决了文案来源，还有一个常见问题是配音。很多同学担心自己的声音不好听。

淘金阁网站还提供了一个非常实用的工具：在线配音生成。点击网站右上角的“小工具”图标。

![](img/fcf4887952493315f0d0c44165445a03_15.png)

![](img/fcf4887952493315f0d0c44165445a03_11.png)

![](img/fcf4887952493315f0d0c44165445a03_17.png)

在工具页面中，选择第四个选项“配音在线生成”。

![](img/fcf4887952493315f0d0c44165445a03_19.png)

![](img/fcf4887952493315f0d0c44165445a03_13.png)

这个功能是免费的。进入后，你会看到极其丰富的配音音色选择。

以下是部分音色类型举例：
*   **亲和女生、温柔女生、甜美女生**
*   **自然女声、卖场广播**
*   **亲切女声、活力女声**（直播配音）
*   **标准男声、标准女声**
*   **湖南口音、童声**
*   **英语配音**
*   **方言配音**：四川话、粤语、东北话、台湾话
*   **精品男配**：情感解说、影视解说、娱乐解说等

![](img/fcf4887952493315f0d0c44165445a03_15.png)

使用方法很简单：
1.  在左侧“配音文本”框中输入或粘贴你准备好的文案。
    ```text
    示例文案：欢迎来到我的抖音课堂，今天你学习了吗？
    ```
    ![](img/fcf4887952493315f0d0c44165445a03_17.png)
    ![](img/fcf4887952493315f0d0c44165445a03_19.png)

![](img/fcf4887952493315f0d0c44165445a03_21.png)

2.  在右侧选择你喜欢的音色，例如选择“东北话”。
3.  调整下方的参数：**音量**、**语速**、**语调**。
    ![](img/fcf4887952493315f0d0c44165445a03_21.png)
    ![](img/fcf4887952493315f0d0c44165445a03_23.png)

![](img/fcf4887952493315f0d0c44165445a03_23.png)

4.  点击“立即生成”。
5.  如果是手机端使用，可能需要用微信扫码登录一下，即可免费使用。
    ![](img/fcf4887952493315f0c44165445a03_25.png)

![](img/fcf4887952493315f0d0c44165445a03_25.png)

6.  生成后，在“配音生成列表”中，找到你的作品，点击右下角的下载按钮，即可将音频文件保存到本地。
    ![](img/fcf4887952493315f0d0c44165445a03_27.png)

下载好的配音文件，可以在后续的视频剪辑环节直接使用。

![](img/fcf4887952493315f0d0c44165445a03_27.png)
