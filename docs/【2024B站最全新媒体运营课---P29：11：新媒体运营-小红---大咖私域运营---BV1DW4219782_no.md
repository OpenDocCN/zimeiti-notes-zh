# 新媒体运营：P29：小红书平台用户标签及流量入口 📊

![](img/871133917d41a785808a7b6065da88b8_1.png)

在本节课中，我们将学习小红书平台如何为用户打上标签，以及平台有哪些主要的流量入口。理解这些机制，有助于我们分析笔记如何被系统推荐给用户，并反推如何让笔记被更多精准用户看到。

上一节我们介绍了系统如何判定内容标签，本节中我们来看看平台如何判定用户标签。

## 用户标签判定机制

平台主要通过分析用户的**浏览行为**来判定其标签。用户注册时选择的兴趣范围参考价值不大，仅作为初期内容方向的引导，无法进入系统深层的用户标签池。

以下是系统判定用户兴趣分类的主要行为依据：

### 1. 点击率
**点击率**指系统在发现页向用户推送多个内容后，用户点击各类内容的概率。例如，系统推送了明星、美妆、家居、母婴等内容，用户点击美妆内容三次，家居两次，母婴一次或未点击。系统会判定该用户对母婴内容不感兴趣，可能不是宝妈，并对美妆内容更感兴趣。随后，系统会持续推送更多美妆内容进行试探，通过点击率逻辑为用户打上强关联或弱关联标签。

我们可以将用户标签理解为一个**标签云图**。假设一个用户的标签云中，“美妆”相关标签最大，代表该用户是美妆内容的偏好者。系统会持续为其推送美妆内容，减少或不推送其不感兴趣的内容（如母婴），以提升用户的获得感与停留时长。

### 2. 停留时长
**停留时长**是分析用户喜好的另一关键行为。例如，用户喜欢美妆内容，但点开某些美妆笔记后很快划走，说明这些笔记不符合其喜好。系统会尝试推送美妆领域内其他细分内容（如上粉、护肤）进行观察。

如果用户对平常点击较少的家居类内容，某次点开后停留很久，仔细浏览图片和评论，系统可能判定该用户有装修需求或想购买相关商品（如盘子），进而考虑推送更多相关笔记或商品。

### 3. 页面浏览量 (PV)
**PV** 指用户浏览页面的深度。例如，用户点开一篇装修笔记后，又进入博主主页，并连续浏览了第二、三、四篇笔记。这提升了该用户在该页面的浏览数。

如果账号主页内容精致，能有效提升用户的浏览深度。这个数值对账号健康度评分有非常大的帮助。

### 4. 互动行为
互动行为包括**点赞、收藏、评论、关注**。这些数据是我们在创作每篇笔记时需要努力提升的目标。点赞、收藏数据高，笔记进入更大推荐流量池的几率更高，也可能带动账号内其他笔记流量攀升，并提升粉丝数，为后续接单、变现打下基础。

互动行为能够提升笔记进入下一层推荐流量池的几率。

## 平台流量分发入口

了解用户标签后，我们来看小红书平台的主要流量分发入口。目前小红书有四个主要流量入口。

以下是各流量入口的详细介绍：

### 1. 发现页
发现页呈现的内容主要包含：
*   **已关注的达人**发布的内容。
*   根据**用户标签**（点击率、浏览行为）推荐的、系统判定用户可能喜欢的内容。
*   少量用户未刷到过或不感兴趣的内容，以增加平台内容的多样化，满足用户攻略、教程、分享等多元化需求。

对于博主而言，笔记和账号后期能获得的主要流量来源于发现页。

### 2. 关注页
关注页内容相对简单，主要展示用户**已关注的所有博主**发布的笔记。用户通常从这里直接查找强关注的博主内容。

### 3. 附近页
附近页根据**手机定位**，向用户推送附近及同城的相关内容。该板块以具有强地域特征的内容为主，如美食、当地旅游、景点等。

对于运营线下实体店（如美容院、餐厅）的博主，在做小红书信息流广告时，需要特别注意对附近人群的推送。

### 4. 购物页
购物页是小红书向电商方向发展的重要入口，主要匹配用户的购物需求。目前，部分用户尚未在小红书形成强购物习惯，更多人仍倾向于在传统电商平台或抖音购物。许多用户仍习惯通过搜索笔记来获取购物推荐。
