## 第一节：店铺官方账号 🏪

上一节我们介绍了课程概述，本节中我们来看看什么是店铺官方账号。

店铺官方账号是店铺选定的唯一抖音号，并与店铺进行绑定。绑定后，该抖音号主页会显示“进入店铺”字样。用户通过此抖音号可以直接进入所经营的店铺。这个功能实现了抖音账号与店铺的一体化关联。

![](img/5e8a5c7e575a6cf2694029da63c4932f_5.png)

对于消费者而言，这增强了店铺的品牌概念。用户无论是搜索店铺名称还是该抖音账号，都能实现购买链路。因此，店铺官方账号的核心功能是实现店铺与抖音号的一体化。

以下是店铺官方账号的绑定流程与特点：
*   **绑定流程**：店铺缴纳保证金并成功入驻后，通过平台弹窗引导即可完成绑定。
*   **强制绑定**：若不绑定官方账号，店铺的其他功能将无法使用。

### 官方账号的权益与注意事项

作为店铺官方账号，平台赋予了一些特定权益，同时也存在严格的限制。

以下是官方账号的主要权益：
*   **免费企业认证**：自动免费获得每年价值600元的企业蓝V认证。
*   **培养用户习惯**：有助于培养用户通过账号进入店铺的习惯。
*   **双重曝光**：用户搜索店铺名或账号名，都能为店铺和账号带来曝光。

以下是绑定官方账号时必须注意的关键事项：
*   **永久绑定**：官方账号一旦与店铺绑定，**无法解绑**。绑定前务必慎重选择抖音号。
*   **绑定限制**：目前仅支持绑定**抖音账号**，头条、西瓜等平台账号暂不支持。
*   **特殊入驻路径**：通过抖音H5页面入驻的商家，入驻成功时会自动将当前登录的抖音号绑定为官方账号。
*   **资质与结算**：绑定后，该抖音号与店铺共享店铺资质和收款资质。通过该抖音号产生的所有收入，都会结算到店铺账户中。

## 第二节：店铺授权号 🔑

在了解了官方账号之后，本节我们来探讨功能更为灵活的店铺授权号。

店铺授权号的核心功能是实现“零粉丝开播”，并允许商品在抖音、今日头条、抖音火山版、西瓜视频等多个平台进行分发和售卖。图中显示，“馋妈妈旗舰店”的商品可以通过绑定不同平台的账号进行销售。

### 授权号的开通资质与规则

以下是开通店铺授权号需要满足的资质与规则：
*   **店铺类型要求**：官方旗舰店、旗舰店、专卖店、专营店对开通授权号无特殊要求。它们在抖音、西瓜、头条、火山版四个平台可分别绑定最多4个账号。
*   **账号唯一性**：同一个抖音号不能同时既作为店铺授权号，又作为该店铺的官方账号。
*   **单店绑定**：一个抖音号仅支持被**一家店铺**绑定为授权号。
*   **新增类型（企业店/个体店）**：企业店和个体店也可开通授权号，但门槛较高：
    *   店铺体验分需大于4.0。
    *   店铺官方账号粉丝数需≥1000。
    *   被绑定的授权号账号粉丝数也需≥1000。
    *   每个渠道（平台）最多可开设3个授权号。

### 授权号的两种类型：形象号与人设号

一个店铺可以同时拥有店铺形象号和店铺人设号，也可以只开通其中一种。选择依据在于账号的定位。

以下是两种类型的定义与选择建议：
*   **店铺形象号**：以品牌或店铺形象为核心。若你希望账号昵称带有店铺类型或表达店铺整体形象，应选择此类。
*   **店铺人设号**：以个人或团队人设形象为核心。其他情况通常选择此类。

![](img/5e8a5c7e575a6cf2694029da63c4932f_7.png)

以下是两种类型的具体规则对比：

![](img/5e8a5c7e575a6cf2694029da63c4932f_9.png)

**店铺形象号规则**：
*   名称不可重复。
*   名称一旦设置，**无法修改**。
*   账号注销后，绑定关系自动解除。
*   可自行选择是否认证蓝V（需符合命名规范）。

**店铺人设号规则**：
*   只能绑定**普通抖音号**（已认证蓝V的账号需先取消认证才能绑定）。
*   账号名称**可以随意修改**。
*   账号注销后，绑定关系自动解除。
*   **不允许认证蓝V**。

### 授权号的命名规范 📝

![](img/5e8a5c7e575a6cf2694029da63c4932f_11.png)

店铺授权号的命名需遵循固定公式，根据店铺类型和账号类型（形象号/人设号）有所不同。

以下是各类店铺的命名公式：
*   **官方旗舰店**：品牌名 + 官方旗舰店 + [形象号/人设名]
    *   *形象号示例*：阿牛官方旗舰店
    *   *人设号示例*：阿牛官方旗舰店小王
*   **旗舰店**：品牌名 + 旗舰店 + [形象号/人设名]
    *   *形象号示例*：阿牛旗舰店
    *   *人设号示例*：阿牛旗舰店小李
*   **专卖店**：品牌名 + 企业商号 + 专卖店 + [形象号/人设名]
    *   *形象号示例*：阿牛常盛专卖店
    *   *人设号示例*：阿牛常盛专卖店小张
*   **专营店**：企业商号 + 专营店 + [形象号/人设名]
    *   *形象号示例*：常盛专营店
    *   *人设号示例*：常盛专营店主播

> **案例说明**：品牌“阿牛”授权给“常盛有限公司”。该公司开设了“阿牛官方旗舰店”、“阿牛旗舰店”、“阿牛常盛专卖店”、“常盛专营店”四家店。其授权号可按上述公式命名，例如“阿牛官方旗舰店售后号”（形象号）或“阿牛官方旗舰店小王”（人设号）。

### 绑定授权号的流程与注意事项

![](img/5e8a5c7e575a6cf2694029da63c4932f_13.png)

![](img/5e8a5c7e575a6cf2694029da63c4932f_14.png)

绑定店铺授权号的操作非常简单。

![](img/5e8a5c7e575a6cf2694029da63c4932f_15.png)

![](img/5e8a5c7e575a6cf2694029da63c4932f_16.png)

以下是绑定步骤：
1.  进入抖店后台。
2.  选择「经营」-「账号管理」-「店铺授权账号」。
3.  点击「新增授权账号」。
4.  选择对应平台，并使用该平台账号扫码完成绑定。

![](img/5e8a5c7e575a6cf2694029da63c4932f_18.png)

以下是绑定授权号的重要注意事项：
*   **长期绑定**：账号绑定成为授权号后的**180天内，无法解绑或更换**。绑定前务必确认该账号是否符合要求。
*   **形象号材料**：官方旗舰店、旗舰店、专卖店开通店铺形象号时，需上传对应的品牌授权书。专营店则可以使用企业商号，无需品牌授权书。
*   **人设号蓝V限制**：店铺人设号不允许认证蓝V。若想将已认证蓝V的账号绑为人设号，需先关闭蓝V认证。

## 总结 🎯

本节课中，我们一起学习了抖音电商中官方账号与授权号的核心知识。

我们来回顾一下三种账号的核心区别：
*   **店铺官方账号**：与店铺永久绑定的唯一抖音号，实现店号一体。**权益**包括免费企业认证、提供稳定入口、获得双重曝光。**注意**绑定后不可解绑，且仅限抖音账号。
*   **店铺形象号**：以品牌/店铺形象呈现的授权号。**特点**是名称不可修改，可自选是否认证蓝V。**注意**名称不可重复，需符合命名规范。
*   **店铺人设号**：以个人/团队人设形象呈现的授权号。**特点**是名称可修改，不可认证蓝V。**注意**绑定前需关闭已有蓝V，需符合命名规范。

本节重点讲解了官方账号的定义与注意事项，以及授权号的类型、命名规范与绑定规则。建议你在学习后，可以尝试在抖店后台进行实际操作，以加深理解。
