# 自媒体运营教程：P42：视频剪辑找素材 🎬

在本节课中，我们将要学习视频剪辑类内容创作的第一步：如何高效、合法地寻找素材。这是制作高质量视频的基础，也是避免版权问题和平台违规的关键。

![](img/eddd986ecdbb7c9a3f1b86659165232a_1.png)

## 视频剪辑的类型

![](img/eddd986ecdbb7c9a3f1b86659165232a_3.png)

视频剪辑类内容涵盖多种形式。以下是常见的几种类型：
*   **电影剪辑**：对电影内容进行二次创作。
*   **电视剧剪辑**：对电视剧集进行剪辑和重组。
*   **综艺剪辑**：剪辑综艺节目中的精彩片段。

上一节我们介绍了视频剪辑的几种类型，本节中我们来看看如何为这些类型的视频寻找素材。

## 素材来源平台

寻找素材是视频剪辑的第一步。以下是三个主要的素材来源渠道：

**1. 视频平台**
所有主流的视频平台都是潜在的素材库。例如，下图列出的30多个自媒体平台均可用于寻找素材。
![](img/eddd986ecdbb7c9a3f1b86659165232a_1.png)
![](img/eddd986ecdbb7c9a3f1b86659165232a_3.png)

**2. 专业影视/素材网站**
除了通用视频平台，还有一些专业的网站提供无版权或可商用的素材。
*   **影视类网站**：例如豆瓣电影，以及讲师整理的50多个无版权要求的影视网站。
*   **综合素材网站**：讲师还整理了近60个网站，涵盖旅游、图片、音效、体育、美食、动漫、搞笑、国内外视频等多种类型的素材。

**3. 影视推广平台**
在抖音、快手等平台，单纯发布视频可能主要起到涨粉作用。但有一些专门的影视推广平台，使用其提供的影视素材创作视频，可以依靠播放量直接获得奖金。这部分内容后续会详细讲解。

## 如何查找素材

正确的找素材方式不是直接下载完整视频（即“搬运”），这会导致95%的审核无法通过。正确的方法是：从大量视频中，提取符合自己视频主题的片段。

![](img/eddd986ecdbb7c9a3f1b86659165232a_9.png)

**核心操作公式**：
`搜索关键词` -> `浏览结果` -> `截取10秒左右片段`

![](img/eddd986ecdbb7c9a3f1b86659165232a_11.png)

以下是具体步骤：

**步骤一：确定主题与关键词**
首先，你需要确定视频的主题。例如，我们决定制作一个“沈腾搞笑名场面”的合集。那么，核心人物就是“沈腾”，主题围绕“搞笑”和“名场面”。

**步骤二：在视频平台搜索**
打开任意视频平台（如西瓜视频），在搜索框中输入确定好的关键词。
![](img/eddd986ecdbb7c9a3f1b86659165232a_5.png)
![](img/eddd986ecdbb7c9a3f1b86659165232a_7.png)

**步骤三：筛选与截取片段**
搜索后，你会得到大量相关视频。此时，你需要浏览这些视频，找出其中符合“沈腾搞笑”主题的精彩部分。
![](img/eddd986ecdbb7c9a3f1b86659165232a_13.png)
**关键点**：每个素材片段的时长应控制在**10秒钟左右**。这样截取的片段在平台审核时更容易通过，能有效避免违规。

![](img/eddd986ecdbb7c9a3f1b86659165232a_13.png)

**操作示例**：
如果你想找“大杨哥”、“小杨哥”的直播搞笑片段，同样只需在平台搜索他们的名字即可。
![](img/eddd986ecdbb7c9a3f1b86659165232a_9.png)
![](img/eddd986ecdbb7c9a3f1b86659165232a_11.png)

## 素材整理与后续制作

找到所需素材片段后，下一步就是进行视频剪辑。目前最常用的剪辑工具是**剪映**软件。根据学员反馈，手机端的使用更为普遍，后续实操部分将主要以手机剪映为例进行演示。

确定主题（如“沈腾搞笑名场面”）并找到相应素材片段后，即可开始剪辑创作。
![](img/eddd986ecdbb7c9a3f1b86659165232a_15.png)

![](img/eddd986ecdbb7c9a3f1b86659165232a_15.png)
