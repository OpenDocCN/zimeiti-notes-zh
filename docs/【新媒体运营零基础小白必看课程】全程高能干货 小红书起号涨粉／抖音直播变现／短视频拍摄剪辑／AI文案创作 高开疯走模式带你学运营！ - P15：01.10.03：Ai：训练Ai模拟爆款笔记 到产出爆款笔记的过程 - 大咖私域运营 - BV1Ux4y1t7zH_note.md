## 概述
在本节课中，我们将学习如何利用AI工具（如ChatGPT）分析和学习平台上的爆款内容，并最终指导AI生成具备爆款潜力的笔记。整个过程包括：提供爆款文案让AI分析、提炼爆款因素、整合学习要点，并最终生成符合平台风格的新文案。

## 第一步：明确分析指令与要点
要让AI有效分析爆款笔记，我们需要给出清晰、具体的指令。核心是让AI分析内容为何能成为爆款，并学习其结构。

![](img/202ceb3d94bea68a9d689a07350ebf75_15.png)

以下是需要AI重点分析的几个方面：
1.  **爆款因素**：找出内容中可能使其受欢迎的关键点。
2.  **整体结构**：
    *   **开篇**：用什么方式吸引用户继续阅读。
    *   **中间**：提供了哪些内容来戳中用户痛点，提升停留时长。
    *   **结尾**：设置了什么“钩子”来引导用户互动（如私信、关注）。
3.  **关键词策略**：分析笔记是否植入了特定关键词，以及关键词的出现频率和铺设方式。

![](img/202ceb3d94bea68a9d689a07350ebf75_17.png)

![](img/202ceb3d94bea68a9d689a07350ebf75_19.png)

![](img/202ceb3d94bea68a9d689a07350ebf75_21.png)

我们可以将上述要求整合成给AI的指令，例如：
```
请你分析提供的小红书爆款笔记文案。需要分析：
1. 文案的爆款因素。
2. 文案的整体结构（开篇如何吸引用户、中间如何戳中痛点、结尾如何引导互动）。
3. 笔记是否设计了关键词。如有，请指出关键词、出现频率及铺设方式。
```

## 第三步：整合AI分析出的爆款因素
AI在分析多篇笔记后，会给出每篇的爆款因素。我们需要让AI对这些分析结果进行去重和整合，提炼出该领域爆款笔记的共性。

![](img/202ceb3d94bea68a9d689a07350ebf75_57.png)

可以给AI如下指令：
```
请将你之前分析过的所有考研爆款笔记的爆款因素总结一下，整合成一个完整的列表。
```

AI可能总结出的爆款因素列表如下：
*   **情感共鸣**：开篇常用焦虑、担忧、迷茫等情绪吸引目标读者。
*   **逆袭故事**：分享个人从困境到成功的经历，增加可信度和代入感。
*   **明确的备考计划**：提供清晰、阶段性的复习时间表和目标。
*   **针对性建议**：针对政治、英语、数学、专业课等具体科目给出方法。
*   **具体资源和技巧**：推荐实用的书籍、课程、APP或学习技巧。
*   **时间节点明确**：强调关键的时间点，增强紧迫感和规划性。
*   **正能量鼓励**：在结尾处给予读者鼓励和信心。
*   **使用表情符号**：在文案中穿插使用emoji，增加亲和力和视觉节奏。

![](img/202ceb3d94bea68a9d689a07350ebf75_59.png)

![](img/202ceb3d94bea68a9d689a07350ebf75_61.png)

**注意**：AI有时会“遗忘”或遗漏之前分析过的某些点（如“逆袭故事”）。如果发现总结不全面，可以提醒AI：“你一共分析了四篇笔记，请重新回顾并整合所有因素。”

## 第四步：指导AI生成爆款笔记大纲与文案
获得爆款因素列表后，就可以指导AI进行创作了。这个过程应分步进行，先产出大纲，确认后再生成完整文案。

![](img/202ceb3d94bea68a9d689a07350ebf75_63.png)

![](img/202ceb3d94bea68a9d689a07350ebf75_64.png)

1.  **生成笔记大纲**：
    给AI指令：
    ```
    请结合你总结的考研爆款笔记因素，为我规划一篇小红书爆款笔记的大纲。大纲需要细分到具体的小节点。
    ```
    AI生成的大纲可能包括：吸引人的开篇、主题介绍、专业选择分析、各科备考规划、时间管理与心态调整、互动引导等部分。

2.  **根据大纲生成完整文案**：
    确认大纲无误后，给AI更详细的指令来生成文案：
    ```
    请按照上述大纲，撰写一篇完整的小红书笔记文案。要求：
    1. 结构严格遵循爆款笔记因素。
    2. 开篇融入一个简单的“逆袭小故事”引发共鸣。
    3. 文案中要包含适量的emoji表情。
    4. 尝试以中文分词法的思维，自然融入“考研”、“备考”等核心关键词。
    ```

    AI根据指令生成的文案示例开头：
    > “当你曾经迷茫，觉得时间不够用，考研压力巨大……我理解那种感觉！但我亲身经历了逆袭。只要你下定决心，制定合理计划，考研绝非遥不可及！🔥”

    正文会按照大纲，展开各科备考建议、时间规划等，并在结尾鼓励互动。

**提示**：与AI对话时，措辞会影响产出。例如，“包含丰富的emoji”和“包含适量的emoji”会导致表情符号的数量不同。需要根据平台风格调整指令。

![](img/202ceb3d94bea68a9d689a07350ebf75_66.png)

## 总结
本节课我们一起学习了利用AI辅助内容创作的全流程：
1.  **明确指令**：告诉AI需要分析爆款笔记的哪些维度（因素、结构、关键词）。
2.  **投喂资料**：提供真实的、数据好的爆款文案给AI学习。
3.  **整合要点**：让AI总结并提炼出公共的爆款因素列表。
4.  **分步创作**：先让AI根据爆款因素列出大纲，审核后再生成融合了所有要点的完整文案。

![](img/202ceb3d94bea68a9d689a07350ebf75_68.png)

![](img/202ceb3d94bea68a9d689a07350ebf75_70.png)

通过这个方法，即使是运营新手，也能借助AI快速产出符合平台调性、具备爆款潜力的内容草稿。当然，AI的初稿可能需要人工进行优化和润色，以使其更加精炼和出色。使用更强大的模型（如GPT-4）或定制化的AI助手（如GPTs），能够获得更优质、更符合个人风格的产出。
