# 拼多多运营：P17：发布商品时如何填写商品基本信息 🛒

在本节课中，我们将学习在拼多多平台发布商品时，如何正确填写商品的基本信息，特别是如何选择商品类目。这是商品成功上架并获取流量的关键第一步。

前面的课程我们已经介绍了拼多多店铺的基础操作。本节中，我们来看看发布商品的具体流程，核心在于理解并正确选择商品类目。

## 一、选择主营类目

![](img/ca919976bc144a16a0a1527047638823_1.png)

![](img/ca919976bc144a16a0a1527047638823_2.png)

首先需要明确的是，店铺的主营类目决定了你能发布哪些商品。商家在入驻时已选择主营类目，后续发布的商品必须属于该主营类目下的子类目。

![](img/ca919976bc144a16a0a1527047638823_3.png)

![](img/ca919976bc144a16a0a1527047638823_4.png)

*   **查看与修改**：主营类目信息可在商家后台的“店铺信息”中查看。每个店铺有一次修改主营类目的机会。
*   **发布入口**：在商家后台左侧菜单栏，点击 **`商品管理 -> 发布新商品`** 即可开始发布流程。
*   **企业店铺注意**：企业店铺发布的商品必须在营业执照的经营范围之内，否则可能无法发布或被限制流量。

## 二、区分海淘与非海淘店铺

对于不同类型的店铺，类目选择有特殊要求。

*   **海淘店铺**：入驻时选择“非海淘进口”作为主营类目，可售卖国内商品及国内发货的进口商品。
*   **非海淘店铺**：发布商品时，标题、主图或详情页中**不要出现“海淘”字样**，并且必须选择“普通商品”或“国内发货的进口商品”。

## 三、处理特殊类目资质

部分特殊类目（如食品、医药）的商品在发布时，需要商家提前上传相应的资质证明。

操作路径为：商家后台 -> **`店铺信息 -> 店铺经营许可`**，在此处添加对应的许可证（如食品经营许可证、医疗器械生产许可证等）。未上传相应资质，则无法发布对应类目的商品。

## 四、正确选择商品分类层级

![](img/ca919976bc144a16a0a1527047638823_6.png)

![](img/ca919976bc144a16a0a1527047638823_7.png)

发布商品时，需要逐级选择类目，通常包括一级、二级、三级，甚至四级类目。正确选择至关重要。

以下是选择类目时的三个核心要点：

1.  **流量匹配**：搜索展示页推荐的类目是基于商家选择的主营类目。如果商品未放在主营类目下的正确细分分类中，可能无法获得类目流量，甚至被屏蔽。
2.  **子类目归属**：商品必须发布在主营类目下的子类目中。例如，“包包”可能分属“儿童背包”和“女包”等不同子类目，放错类目会影响流量。
3.  **类目准确性**：系统推荐的一、二、三级分类下可能存在相似的类目，务必根据商品实际属性选择最准确的一个。
