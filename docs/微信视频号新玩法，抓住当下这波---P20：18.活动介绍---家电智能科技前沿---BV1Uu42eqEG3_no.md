# 微信视频号新玩法：P20：18.活动介绍

在本节课中，我们将要学习微信视频号“商家联盟”活动的具体介绍。本次活动旨在为2024年中国行业新发展奠定视频化带货电商的技术基础，由短视频教研部负责人张良老师全程对接。活动秉持公平公正的理念展开。

## 活动负责人与背景

![](img/646f0ce38d7b5e9c150077073b87b03e_1.png)

上一节我们介绍了活动的宗旨，本节中我们来看看活动的负责人与独特优势。

张老师作为本次活动的全国总负责人，拥有独家对接权限。这意味着，只有通过张老师才能参与此次活动。

![](img/646f0ce38d7b5e9c150077073b87b03e_3.png)

## 官方提供的核心支持

![](img/646f0ce38d7b5e9c150077073b87b03e_5.png)

了解了负责人背景后，我们来看看本次活动为参与者提供的核心支持。以下是官方提供的两大关键资源：

1.  **高质量视频素材**：官方为每个参与账号准备了30条高质量产品视频素材。参与者每天发布一条即可。
    ![](img/646f0ce38d7b5e9c150077073b87b03e_1.png)

![](img/646f0ce38d7b5e9c150077073b87b03e_7.png)

2.  **巨额流量扶持**：每个参与账号可获得90万的流量扶持，帮助视频获得更多曝光。

![](img/646f0ce38d7b5e9c150077073b87b03e_9.png)

## 活动激励政策

![](img/646f0ce38d7b5e9c150077073b87b03e_14.png)

除了素材和流量，本次活动还提供了直接的现金激励。以下是激励政策详情：

![](img/646f0ce38d7b5e9c150077073b87b03e_15.png)

*   **单条视频奖励**：每发布一条符合要求的带货视频，即可获得300元奖励。
*   **奖励结算方式**：奖励实行“日结”制度，今天发布视频，明天奖励到账。
*   **总收益计算**：若完成全部30条视频的发布，总计可获得奖励金额为：
    `总收益 = 30条 × 300元/条 = 9000元`
    ![](img/646f0ce38d7b5e9c150077073b87b03e_14.png)
    ![](img/646f0ce38d7b5e9c150077073b87b03e_15.png)
