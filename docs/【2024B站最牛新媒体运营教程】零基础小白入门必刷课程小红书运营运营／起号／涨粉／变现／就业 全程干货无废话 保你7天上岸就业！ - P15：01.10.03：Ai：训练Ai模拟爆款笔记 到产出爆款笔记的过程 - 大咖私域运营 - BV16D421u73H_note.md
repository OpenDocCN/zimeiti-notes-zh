# 新媒体运营AI实战：01.10.03：AI训练与爆款笔记生成 🚀

在本节课中，我们将学习如何训练AI（如ChatGPT）分析爆款笔记，并最终生成具备爆款潜质的小红书笔记。整个过程分为两步：首先让AI学习并总结爆款笔记的成功要素，然后指导它基于这些要素创作新内容。

上一节我们介绍了通过“喂资料”的方式，让AI理解特定平台的用语和风格。虽然以旅游攻略为例的成果可能不尽理想，但方法是正确的。关键在于为AI提供你所在赛道的优质内容，让它进行分析和学习。

## 分析爆款笔记的核心要素

![](img/5560e04ec756dc6002f8b73f1dfbfa73_11.png)

![](img/5560e04ec756dc6002f8b73f1dfbfa73_13.png)

要让AI学会创作爆款，首先需要教会它识别什么是“爆款”。我们需要引导AI从多个维度分析我们提供的爆款笔记样本。

以下是分析时需要关注的几个核心点：

![](img/5560e04ec756dc6002f8b73f1dfbfa73_15.png)

![](img/5560e04ec756dc6002f8b73f1dfbfa73_17.png)

1.  **爆款因素**：分析内容为何能成为爆款。例如，是否引发了情感共鸣、提供了实用价值、讲述逆袭故事等。
2.  **整体结构**：拆解文案的叙事框架。
    *   **开篇**：用什么方式（如提问、金句、痛点描述）吸引用户继续阅读。
    *   **中间**：如何通过内容（如解决方案、经验分享）戳中用户痛点，提升停留时长。
    *   **结尾**：如何设置“钩子”（如鼓励互动、提供福利）引导用户关注、私信或评论。
3.  **关键词策略**：分析笔记中是否刻意植入了关键词。如果有，需分析是哪个关键词、其出现频率以及在全文中是如何分布的。

### 第一步：提供样本并下达分析指令

![](img/5560e04ec756dc6002f8b73f1dfbfa73_30.png)

![](img/5560e04ec756dc6002f8b73f1dfbfa73_32.png)

![](img/5560e04ec756dc6002f8b73f1dfbfa73_34.png)

![](img/5560e04ec756dc6002f8b73f1dfbfa73_36.png)

首先，我们需要找到几篇点赞量较高的考研相关笔记，将文案内容复制下来。接着，向AI（如ChatGPT 3.5）下达明确的指令。

![](img/5560e04ec756dc6002f8b73f1dfbfa73_38.png)

![](img/5560e04ec756dc6002f8b73f1dfbfa73_39.png)

![](img/5560e04ec756dc6002f8b73f1dfbfa73_41.png)

**给AI的指令示例：**
```
我给你提供几篇小红书平台上关于考研的爆款笔记文案。请你分析：
1.  它们的爆款因素有哪些？
2.  分析文案的整体结构：开篇用什么吸引用户？中间用什么内容戳中用户痛点？结尾用什么方式引导用户互动或关注？
3.  分析每篇笔记的关键词：是否结合中文分词法设计了关键词？如果有，请指出关键词是什么、出现频率如何。
```

### 第二步：整合分析结果

![](img/5560e04ec756dc6002f8b73f1dfbfa73_57.png)

在AI分析了多篇样本（例如4篇）后，我们需要让它对分析结果进行整合和去重，总结出该领域爆款笔记的共性。

**给AI的后续指令：**
```
请将你前面分析过的几篇考研爆款笔记的爆款因素总结一下，整合成一个清晰的列表。
```

![](img/5560e04ec756dc6002f8b73f1dfbfa73_59.png)

![](img/5560e04ec756dc6002f8b73f1dfbfa73_61.png)

AI可能会总结出如下要点：
*   **情感共鸣**：开篇描述焦虑、迷茫等真实状态，让读者产生“这就是我”的感觉。
*   **逆袭故事**：分享个人从低谷到成功的备考经历，提供希望。
*   **明确规划**：提供清晰的备考时间线、各科目复习计划。
*   **实用建议**：推荐具体的学习方法、资源（书籍、老师、APP）。
*   **时间管理**：强调效率，给出具体的时间分配建议。
*   **正能量鼓励**：在结尾给予读者信心和激励。
*   **表情符号运用**：适当地使用emoji增加亲和力和阅读节奏感。

![](img/5560e04ec756dc6002f8b73f1dfbfa73_63.png)

### 第三步：基于爆款要素生成新内容

![](img/5560e04ec756dc6002f8b73f1dfbfa73_65.png)

获得总结出的爆款要素后，就可以指导AI进行创作了。这个过程最好分步进行。

**1. 先生成内容大纲**
```
我现在要写一篇关于考研的小红书笔记。请结合你刚才总结的爆款笔记因素，为我规划一个具备爆款潜质的笔记大纲。大纲需要细分到具体的小节点。
```
AI生成的大纲可能包括：吸引人的标题、引发共鸣的开篇、专业选择建议、分科目备考规划、时间管理技巧、心态调整方法、互动引导结尾等部分。

**2. 根据大纲撰写完整文案**
在确认大纲无误后，下达最终创作指令。
```
请按照上面确定的大纲，撰写完整的笔记文案。要求：
1.  文案结构严格遵循爆款笔记的因素和结构。
2.  开篇需要融入一个简单的“逆袭”小故事来引入情绪。
3.  文中需要包含适量的emoji表情。
4.  需要以“考研”、“备考计划”等作为关键词，自然地融入文案中。
```

## 成果评估与优化方向

![](img/5560e04ec756dc6002f8b73f1dfbfa73_72.png)

AI生成的初稿文案在结构、专业度和爆款要素融合上可能已具备一定基础，足以满足一般运营需求。例如，它可能以“我理解你备考时的迷茫……”开头，中间提供分月规划表，结尾鼓励互动。

然而，对于更高要求，此初稿可能仍有优化空间：
*   **语言更精炼**：避免啰嗦，直接切中痛点。
*   **个人特色更鲜明**：让“逆袭故事”更具细节和感染力。
*   **关键词更自然**：避免生硬堆砌，让关键词与内容浑然一体。
*   **互动钩子更强**：设计更具吸引力的引导话术。

## 进阶工具：定制专属AI助手

在实战中，可能会遇到AI遗忘前文指令的情况。为了获得更稳定、更个性化的输出，可以利用像ChatGPT的“GPTs”功能创建定制助手。

你可以：
*   **上传专属文件**：将你收集的优质小红书文案范例、专用表情包列表等作为知识库上传。
*   **设定固定指令**：在助手创建时，就写入“请用小红书风格写作”、“开头要抓人眼球”等核心要求。
*   这样打造出的私人AI助手，能更稳定地输出符合你特定风格和需求的内容，提升创作效率。
