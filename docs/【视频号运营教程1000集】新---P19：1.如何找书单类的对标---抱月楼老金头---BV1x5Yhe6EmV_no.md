# 视频号运营教程：P19：如何寻找书单类对标账号 📚

在本节课中，我们将学习在短视频平台（如抖音）上，如何高效地寻找与自身定位相符的对标账号。我们将以“书单”类账号为例，介绍两种核心方法，帮助你快速发现并关注可模仿的优质账号。

上一节我们介绍了寻找对标账号的重要性，本节中我们来看看具体如何操作。

## 方法一：利用“关注”推荐功能

首先，在平台内搜索你计划从事领域的关键词，例如“书单”。从搜索结果中找到一个你感兴趣的非真人出镜账号。

![](img/9733586b97394fe1299f959c9a696a42_1.png)

点开该账号的头像进入主页，先不要点击关注。浏览其视频内容，确认其类型与你想要模仿的风格相符。

如果内容符合你的要求，点击“关注”按钮。此时，平台会立刻在页面下方弹出一个“你可能感兴趣”的账号列表。这些账号通常与当前账号的官方标签相似，属于同一类型。

![](img/9733586b97394fe1299f959c9a696a42_3.png)

![](img/9733586b97394fe1299f959c9a696a42_4.png)

以下是操作步骤：
1.  进入目标账号主页，确认内容类型。
2.  点击“关注”按钮。
3.  查看平台推荐的“你可能感兴趣”账号列表。
4.  逐一进入推荐账号的主页，检查其视频内容和橱窗商品。
5.  若内容符合你的要求，则点击关注该账号。

通过这种方式，你可以在两三分钟内关注数十个同类账号。此后，平台算法会持续向你推荐更多相似账号。

![](img/9733586b97394fe1299f959c9a696a42_6.png)

## 方法二：通过商品“达人秀”寻找

![](img/9733586b97394fe1299f959c9a696a42_8.png)

第二种方法是通过具体商品来寻找相关账号。例如，如果你计划销售某本书，可以直接在平台搜索该书名。

在搜索结果页，切换到“商品”分类，并按“销量”排序。这样可以筛选出销售该商品最成功的店铺。

进入销量较高的商品详情页，通常会有“达人推荐”或“买家秀”板块。这个板块汇集了大量推广该商品的短视频账号。

以下是操作步骤：
1.  在平台搜索框输入具体商品名（如“墨菲定律漫画版”）。
2.  在搜索结果页点击“商品”分类，并按“销量”排序。
3.  选择销量较高的商品，进入其详情页。
4.  找到“达人推荐”或“买家秀”板块。
5.  浏览该板块下的所有推荐视频，这些视频的发布账号就是你的潜在对标账号。

![](img/9733586b97394fe1299f959c9a696a42_10.png)

这些账号的视频类型通常高度一致，多为影视混剪、电视剧片段或名人语录剪辑，且带货销量可观，非常适合新手模仿。一个热门商品下可能关联数百个推广账号，为你提供了丰富的学习资源。

本节课中我们一起学习了寻找对标账号的两种核心方法：一是利用关注后的系统推荐功能，二是通过商品详情页的达人推荐板块。掌握这两种方法，你可以快速、高效地建立起自己的学习库，为后续的内容创作和账号运营打下坚实基础。
