# 拼多多运营教程：P1：无价格优势店铺如何利用直通车起量扩量？🚀

在本节课中，我们将学习拼多多新店与老店在没有价格优势的情况下，如何通过“商品推广”（直通车）工具来启动流量并扩大销售规模。我们将分别探讨新店和老店的不同操作思路与具体步骤。

---

## 新店起量实操指南

上一节我们概述了课程目标，本节中我们来看看新店铺的具体操作流程。对于新店新链接，想通过商品推广快速起爆，需要按顺序完成以下关键步骤。

以下是新店起量必须做好的四件事：

1.  **做好基础营销氛围**：利用平台提供的营销工具，如限时限量购、大促活动页面、优惠券等，营造促销感。
2.  **链接上架后执行破零动作**：商品上架后，需要通过一些方式实现初始销售。可以是搜索渠道的补单，也可以是分享链接的递增式破零。
3.  **创建并测试推广计划**：在商品推广中，可以创建“成交出价”或“投产比（ROI）”模式的计划。建议先创建一个以保本为目标的计划，尝试将单个链接的日订单量快速提升到20至30单。如果连保本的计划都消耗不动预算，则考虑更换链接。
4.  **稳定后扩展与升级**：当单链接能稳定产出几十单时，可以进行**单品裂变**（创建相似链接）或上架新品。随后，将运营重心转向“全店托管”模式。例如，若保本投产比为`3`，则在全店托管中可将目标投产比设置为`5`。日限额可设为`300`元。这个思路追求稳定且有利润。如果其中有链接数据表现特别突出，可以再将其单独拉出来，使用“成交出价”或“投产比”模式进行重点投放。

## 老店扩量核心策略

![](img/0d9a95481fc0ff809b9bc9f266c2c34e_3.png)

了解了新店的启动方法后，我们接着分析老店铺如何突破瓶颈。对于那些订单量停滞不前的老店，若想利用商品推广扩大体量，核心策略与新店不同。

![](img/0d9a95481fc0ff809b9bc9f266c2c34e_4.png)

老店通常已具备稳定的用户标签，因此扩量的关键在于利用“全店托管”功能广泛拉取流量。

![](img/0d9a95481fc0ff809b9bc9f266c2c34e_5.png)

具体操作如下：将全店托管的**投产比**设置为略高于保本点的水平，目的是每天引入更多流量。流量进来后，通过**单品多链接上架**或**多品多链接上架**的策略来承接。

这里有一个定价技巧：新品上架时，定价可以比同行高出`10%`到`15%`。由老品拉来的流量会自然导向新品进行承接。

对于新链接，曝光越多，成长越快。一旦新链接数据跑起来，就可以将其单独拉出，使用商品推广的“成交出价”或“投产比”模式进行稳定成本投放。无论是哪种模式，初期可以设置`100`到`200`元的日限额，预算消耗完即可。

重复以上操作，利用多个链接矩阵来冲击市场规模，体量就一定能够提升。

## 课程总结

本节课中，我们一起学习了拼多多新店与老店在没有价格优势时，如何差异化地运用直通车工具。

![](img/0d9a95481fc0ff809b9bc9f266c2c34e_7.png)

*   **对于新店**，核心路径是：**营造氛围 -> 破零启动 -> 测试保本计划 -> 稳定后转全店托管并裂变**。
*   **对于老店**，核心路径是：**利用全店托管拉流量 -> 通过多链接上架（新品可提价）承接流量 -> 表现好的链接单独精准投放**。

![](img/0d9a95481fc0ff809b9bc9f266c2c34e_8.png)

关键在于理解不同店铺阶段的目标差异，并灵活组合使用“成交出价”、“投产比”及“全店托管”等推广模式。通过多链接、矩阵化的操作，即使没有价格优势，也能有效提升店铺流量与销量。
