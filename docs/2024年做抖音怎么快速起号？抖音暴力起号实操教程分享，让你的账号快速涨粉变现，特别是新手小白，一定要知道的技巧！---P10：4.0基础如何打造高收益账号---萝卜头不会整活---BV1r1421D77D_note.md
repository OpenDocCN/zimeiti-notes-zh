# 抖音运营：P10：4.0基础如何打造高收益账号

在本节课中，我们将要学习如何解决视频播放量低的问题，并了解如何根据平台特性选择正确的领域和内容类型，从而让平台愿意为你的账号推送流量。

## 播放量低迷的原因分析

![](img/5bc09dc3918aa007dcaa5c35b30cedd9_1.png)

上一节我们介绍了账号运营的基本心态，本节中我们来看看为什么视频发布后播放量总是很低。很多同学会遇到视频发布后始终是零播放，或者只有几百到一千多播放的情况。这并非平台针对你或不给你推送流量。

根本原因在于，自媒体平台众多，每个平台都有其独特的规则和偏好。

![](img/5bc09dc3918aa007dcaa5c35b30cedd9_3.png)

## 平台与领域的匹配原则

目前主流的自媒体平台有30多个。这些平台未来都可能成为你的运营阵地。

![](img/5bc09dc3918aa007dcaa5c35b30cedd9_1.png)

以下是这些平台的一个关键特点：每个平台的收益结算方式和适合的内容领域各不相同。很多同学对此并不熟悉。

![](img/5bc09dc3918aa007dcaa5c35b30cedd9_3.png)

每个平台适合的领域确实不同。举个例子，如果你注册了“今日头条”账号，却选择了一个适合B站的舞蹈领域内容，你认为能做起来吗？你做的平台和选择的领域不匹配，这就是很多自媒体账号未能做起来的原因。平台不会把流量浪费在不适合其用户的内容上，它会优先扶持符合平台调性和用户喜好的领域。

![](img/5bc09dc3918aa007dcaa5c35b30cedd9_5.png)

关于各平台领域的具体区分，后续可以提供给大家参考。

## 如何选择内容赛道

除了选对平台，在这30多个平台里，我们该选择制作什么类型的内容呢？

![](img/5bc09dc3918aa007dcaa5c35b30cedd9_5.png)

首先思考一下，平时刷视频时，你看得最多的是哪类内容？是影视解说、游戏、综艺、动漫、搞笑，还是其他？例如，影视类内容在所有平台上的扩散量通常都很大。需要注意的是，像军事这类内容，在国内平台可以做，但在一些海外平台（如TikTok）可能受到限制。

![](img/5bc09dc3918aa007dcaa5c35b30cedd9_7.png)

以下是当前大数据显示用户点击量、阅读量和完播率最高的三个领域：
1.  **视频剪辑类**
2.  **解说视频类**
3.  **混剪视频类**

## 理解“大数据”推荐逻辑

什么是“大数据”？简单来说，平台会分析你的行为。例如，在抖音上，如果你今天看美女视频的时间更长，从明天开始，平台给你推送美女相关的视频就会变多。如果你看美食内容最多，那么平台推送的美食视频就会增加。这就是大数据推荐算法，它可能比我们自己更了解我们的喜好。

**公式表示其核心逻辑可简化为：**
`推荐权重 = f(用户历史行为， 视频内容特征)`

跟着大数据揭示的用户偏好去做视频，成功率会更高。

![](img/5bc09dc3918aa007dcaa5c35b30cedd9_9.png)

本节课中我们一起学习了解决播放量问题的关键：**平台与领域匹配**以及**内容类型选择**。核心在于，不要盲目发布内容，而要先研究平台特性，选择与之匹配的领域（如影视、解说、混剪），并理解平台的推荐机制，让大数据成为你的助力，从而获得平台的流量扶持。
