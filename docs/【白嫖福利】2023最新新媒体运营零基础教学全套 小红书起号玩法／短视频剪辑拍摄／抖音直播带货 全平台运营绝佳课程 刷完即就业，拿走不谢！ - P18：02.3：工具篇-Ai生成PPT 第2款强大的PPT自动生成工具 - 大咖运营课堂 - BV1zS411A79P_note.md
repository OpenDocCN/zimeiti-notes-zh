# 新媒体运营：P18：02.3：工具篇-AI生成PPT（第2款）🚀

在本节课中，我们将学习第二款强大的AI生成PPT工具。这款工具在生成速度、模板丰富度和编辑灵活性方面具有优势，能有效辅助我们快速制作演示文稿。

上一节我们介绍了第一款AI生成PPT工具，本节中我们来看看另一款功能互补的工具。

## 工具核心优势✨

这款工具主要有两个优势。第一是生成速度快且运行稳定。第二是它提供了数量庞大的模板库，用户可以根据需要选择各种样式并调整细节。

然而，要获得理想效果，需要用户提供较为详细和丰富的大纲节点内容，工具才能分析并生成优质的模板。

## 两种生成方式🔧

该工具提供两种生成PPT的方式：AI智能生成和导入本地大纲。通常，导入本地大纲生成的方式使用频率更高，可控性更强。

以下是具体的操作步骤：

1.  **准备大纲内容**：首先，使用ChatGPT等AI工具撰写一个主题大纲。例如，输入指令“帮我完善每一个小节点的具体内容信息，保持好结构和格式排版”，以获取结构清晰的详细大纲。
2.  **导入大纲生成**：在工具中选择“导入本地大纲”功能，将准备好的大纲文档上传。工具会快速梳理文档结构。
3.  **选择模板**：从丰富的模板库中，挑选一个与内容主题（如装修行业）风格相符的模板。
4.  **生成与编辑**：点击生成PPT。生成后，可以在右侧的编辑面板中对文字、形状、背景进行详细设置。如果对某页排版不满意，可以单击该页面，一键更换预设的版式。

## 内容优化技巧💡

有时生成的内容可能不完整，例如大纲中的某些细节未被加载到PPT中。这时，可以通过“大纲编辑”功能进行补充。

以下是内容填充的方法：

*   在“大纲编辑”界面的对应章节下，直接回车添加详细内容。
*   确保内容节点的格式清晰，必要时通过换行来区分标题和正文，这有助于AI更好地理解结构，从而生成更准确的页面。

## AI智能生成体验🤖

除了导入大纲，还可以尝试“AI智能生成”功能。直接输入主题（例如：“装修行业在小红书平台的运营推广计划”），工具会自动构思大纲并生成完整的PPT文案。

生成完成后，同样可以进入编辑界面，自由调整字体、样式或更换整套模板，直至满意。
