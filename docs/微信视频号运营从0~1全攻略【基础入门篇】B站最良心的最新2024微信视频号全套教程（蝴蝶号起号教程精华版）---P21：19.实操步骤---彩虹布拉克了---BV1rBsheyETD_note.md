# 微信视频号运营从0~1全攻略：P21：19.实操步骤 - 开通与上架商品

在本节课中，我们将学习微信视频号带货功能的核心操作：如何开通商品橱窗权限以及如何上架商品。这是将你的视频号内容转化为收入的关键一步，流程简单明了，适合所有初学者。

上一节我们介绍了视频号的基础设置，本节中我们来看看如何开启带货功能。

## 第一步：进入视频号

以下是进入视频号界面的具体步骤：

1.  **打开微信**：在手机主屏幕找到并点击微信应用图标。
2.  **点击“发现”**：进入微信后，点击底部导航栏的“发现”选项。
3.  **进入“视频号”**：在“发现”页面，将视线向上滑动，找到并点击“视频号”入口（图标为黄色的蝴蝶形状）。

![](img/3a6d8d038dc6e31b8943b26452417ec3_1.png)

## 第二步：找到创作者中心

成功进入视频号后，下一步是找到管理后台。

![](img/3a6d8d038dc6e31b8943b26452417ec3_4.png)

![](img/3a6d8d038dc6e31b8943b26452417ec3_6.png)

4.  **点击个人头像**：在视频号主页的右上角，点击人形头像图标。

![](img/3a6d8d038dc6e31b8943b26452417ec3_4.png)

5.  **进入“创作者中心”**：点击头像后进入个人页面，向下滑动屏幕，在最底部找到并点击“创作者中心”。（注：若未发布过作品，可能无法看到此入口，发布一条视频后即可显示。）

![](img/3a6d8d038dc6e31b8943b26452417ec3_6.png)

![](img/3a6d8d038dc6e31b8943b26452417ec3_8.png)

## 第三步：开通带货权限

![](img/3a6d8d038dc6e31b8943b26452417ec3_9.png)

进入创作者中心后，我们就可以进行带货功能的设置了。

6.  **点击“带货中心”**：在创作者中心页面内，找到“带货中心”选项并点击，其图标类似一个小卖部。

![](img/3a6d8d038dc6e31b8943b26452417ec3_8.png)

![](img/3a6d8d038dc6e31b8943b26452417ec3_9.png)

点击后，系统会根据你的账号状态进行跳转。如果尚未开通权限，则会进入开通流程；如果已开通，则会直接进入选品上架界面。

以下是开通带货权限的详细流程：

![](img/3a6d8d038dc6e31b8943b26452417ec3_11.png)

7.  **选择带货身份**：页面会提示你选择带货身份。通常有三个选项：`我是商家`、`我是达人`、`我是商家且已有视频号`。**这里请选择“我是达人”**。即使你是厂家，也建议先用达人身份开通，因为该身份门槛较低，产品由平台供应。

![](img/3a6d8d038dc6e31b8943b26452417ec3_11.png)

8.  **交纳保证金**：选择身份后，会跳转到保证金交纳界面。你需要点击“去交纳”按钮，通过微信支付**100元**的保证金即可完成开通。

![](img/3a6d8d038dc6e31b8943b26452417ec3_13.png)

## 第四步：上架商品（选品）

![](img/3a6d8d038dc6e31b8943b26452417ec3_13.png)

开通权限后，再次点击“带货中心”，界面就会变为商品选品广场。

9.  **浏览与选择商品**：你可以在商品广场中浏览各类商品，平台会根据你的视频内容推荐相关品类。
10. **上架商品**：找到想推广的商品后，点击“加橱窗”或类似的按钮，该商品就会上架到你的视频号橱窗中。

![](img/3a6d8d038dc6e31b8943b26452417ec3_15.png)

**核心操作总结**：
*   开通权限：`点击“带货中心” -> 选择“我是达人” -> 交纳100元保证金`
*   上架商品：`在“带货中心”选品广场 -> 点击“加橱窗”`

![](img/3a6d8d038dc6e31b8943b26452417ec3_15.png)

本节课中我们一起学习了微信视频号开通商品橱窗和上架商品的完整流程。关键点在于通过“创作者中心”进入“带货中心”，以达人身份交纳保证金完成开通，之后便可以在选品广场自由添加商品。这个功能无需粉丝门槛，是视频号变现的快速通道。
