**体制内生存指南：P1：识别“老实人”特质**

在本节课中，我们将一起探讨体制内环境中“老实人”的典型特征。理解这些特征有助于我们进行自我定位和职场认知。

上一段我们明确了课程目标，接下来，我们将具体拆解“老实人”的几个核心表现。

![](img/38b94e6a9733bef1ba874dfb498e7492_1.png)

以下是“老实人”常见的几个行为模式：

*   **过度重视规则与流程**：他们严格遵循所有明文规定和既定流程，即使在某些情境下变通更为高效，也极少主动提出或采用灵活方法。其行为逻辑可以概括为：**行为 = 严格遵循(规章制度)**。
*   **回避冲突与竞争**：在面临利益争夺或观点分歧时，他们通常选择退让或沉默，不善于为自己争取权益，避免成为焦点。
*   **不擅长汇报与沟通**：他们专注于埋头完成具体事务，但往往不重视或不善于向上级主动、定期地汇报工作进展、突出个人贡献和遇到的困难。
*   **边界感模糊，不善拒绝**：经常承担职责范围外或本应属于他人的工作，由于不善于说“不”，导致自身工作负荷过重。这可以用一个简单的条件逻辑描述：
    ```python
    if 他人请求帮助：
        即使超出本职或繁忙，也通常回答：“好的，交给我吧。”
    ```
*   **对人际关系敏感度低**：更关注事情本身的对错，而非其中涉及的人际关系和权力动态，有时会无意中得罪人或错过关键信息。

![](img/38b94e6a9733bef1ba874dfb498e7492_2.png)

通过以上分析，我们了解了“老实人”在体制内环境中的主要行为特点。

本节课中，我们一起学习了体制内“老实人”的五大特征，包括恪守成规、回避竞争、疏于汇报、不善拒绝以及人际敏感度不足。认识这些特征是进行自我审视和职场导航的第一步。
