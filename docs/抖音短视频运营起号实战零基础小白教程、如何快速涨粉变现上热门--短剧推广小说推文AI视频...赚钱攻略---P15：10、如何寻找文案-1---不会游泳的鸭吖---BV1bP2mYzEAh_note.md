# 抖音短视频运营起号实战零基础小白教程：P15：10、如何寻找文案-1 📝

在本节课中，我们将要学习如何解决短视频创作中的核心难题之一：文案获取。对于初学者而言，写出吸引人的文案是提升视频质量和吸引粉丝的关键第一步。我们将介绍实用的工具和方法，帮助你轻松找到并处理文案，甚至解决配音问题。

---

上一节我们讨论了优化作品对于吸引粉丝的重要性，本节中我们来看看如何具体解决文案来源的问题。很多同学在起步阶段不会自己创作文案，这没有关系，我们可以借助一些高效的网络工具。

以下是五个实用的文案素材网站，它们都是网页版，无需下载APP，在手机或电脑浏览器中即可直接使用。

现在，让我们以“淘金阁”为例，详细讲解如何使用这些工具。

进入淘金阁网站后，你会看到主页有非常丰富的分类，例如财经、军事、动漫、科技、美食、美文、星座、体育、旅游、育儿、汽车、社会、娱乐、时尚、房产、国际、宠物等。

在选择文案时，你可以参考一个简单的公式：**一个特定群体 + 一个核心话题 + 疑问形式**。用这个模式去套用筛选出的文案，觉得合适就可以采用。

解决了文案来源，我们再来解决另一个常见问题：有的同学担心自己的声音不好听，不适合配音。

![](img/6cb267748de49bdd49ecfb0635358efb_12.png)

这同样有办法解决。在淘金阁网站上，找到并点击右上角的“小工具”选项。



![](img/6cb267748de49bdd49ecfb0635358efb_13.png)

![](img/6cb267748de49bdd49ecfb0635358efb_8.png)


![](img/6cb267748de49bdd49ecfb0635358efb_14.png)

![](img/6cb267748de49bdd49ecfb0635358efb_15.png)

在工具页面中，选择第四个选项：“配音在线生成”。



![](img/6cb267748de49bdd49ecfb0635358efb_10.png)


这个功能可以免费将文字转换成语音，且配音种类非常丰富。包括：

*   **亲和女生、温柔女声、甜美女声**
*   **自然女声、卖场广播**
*   **直播配音、亲切女声、活力女生**
*   **通用场景（标准男声、标准女声）**
*   **方言场景（四川话、粤语、东北话、台湾话等）**
*   **外语配音**
*   **精品男配（情感解说、影视解说、娱乐解说等）**



![](img/6cb267748de49bdd49ecfb0635358efb_12.png)
![](img/6cb267748de49bdd49ecfb0635358efb_13.png)
![](img/6cb267748de49bdd49ecfb0635358efb_14.png)
![](img/6cb267748de49bdd49ecfb0635358efb_15.png)


使用方法很简单：
1.  在左侧的“配音文本”框中输入或粘贴你准备好的文案。
2.  在右侧选择你喜欢的声音类型，例如“东北话”。
3.  调整下方的**音量**、**语速**和**语调**参数。
4.  点击“立即生成”。
