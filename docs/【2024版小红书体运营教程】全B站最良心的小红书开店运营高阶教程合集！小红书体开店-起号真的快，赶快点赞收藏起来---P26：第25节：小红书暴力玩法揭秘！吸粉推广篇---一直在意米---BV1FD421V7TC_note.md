## 账号类型与矩阵策略

小红书的账号主要分为三类：企业号、个人号和引流号。

*   **企业号**：主要用于树立官方品牌形象。其核心功能包括获得官方认证标识、展示品牌名称与地址，以及使用平台赋予的特权来开展业务。
*   **个人号**：这是目前大多数用户运营的账号类型，其特点是**灵活性高**。个人号可以是品牌员工或真实消费者，便于塑造个人品牌特色，让品牌相关人物自然出镜。
*   **引流号**：这类账号在之前的课程中被称为“炮灰号”。其作用通常是辅助推广，用完即弃或作为备用小号。引流号一般没有精细的个性规划。

通常，品牌的推广会通过企业号、个人号或引流号发布笔记来实现引流。以下是关于打造企业号矩阵的一些具体建议：

以下是构建有效账号矩阵的四个关键行动：

1.  **积极参与品牌活动**：企业号发起的试用、团购等活动，应动用多个小号（个人号或引流号）积极参与，以增加活动热度。
2.  **个人号强化品牌属性**：在个人号的笔记内容中，应多次暗示品牌的定位。例如，品牌若定位为“都市职场青年”，则个人号内容需围绕此场景展开。
3.  **控制营销内容比例**：个人号发布品牌产品内容的频率不宜过高。一个较好的比例是 **营销内容 : 种草/科普内容 = 3 : 7**。即30%用于直接推广，70%用于提供价值、建立信任。
4.  **利用小号评论引流**：企业号发布内容后，可使用引流号在评论区进行互动或引导，以实现更隐蔽和有效的引流。

## 官方付费广告：小红书CPC竞价广告

![](img/7f011bdfb75763168d8a990bfcfa02c9_11.png)

![](img/7f011bdfb75763168d8a990bfcfa02c9_13.png)

小红书官方的广告系统是**CPC（Cost Per Click，按点击付费）竞价广告**，即我们常说的竞价广告。

![](img/7f011bdfb75763168d8a990bfcfa02c9_15.png)

![](img/7f011bdfb75763168d8a990bfcfa02c9_17.png)

其申请与使用流程均在小红书官方网站的“业务合作”板块中完成，具体路径为：**官网 -> 业务合作 -> 推广合作**。

![](img/7f011bdfb75763168d8a990bfcfa02c9_19.png)

不过，使用官方广告服务有一个重要前提：**必须先认证企业号**。

![](img/7f011bdfb75763168d8a990bfcfa02c9_21.png)

企业号的入驻流程同样在官网完成，需要提交相关资质文件并支付审核费用。认证成功后，普通账号即升级为官方企业号，从而获得广告投放资格。

官方广告主要分为两大类：

1.  **信息流广告**：当用户浏览小红书首页推荐内容时，系统会不定期间隔插入广告。这种广告形式与内容流融合，曝光量大。
2.  **搜索广告**：当用户使用关键词进行搜索时，系统会根据算法展示相关的广告。例如，用户搜索“考研资料”，可能会看到“营养补充品”或“学习工具”的广告。其核心逻辑是**关键词竞价排名**，即通过对特定关键词出价来竞争广告位。

目前，信息流广告位主要出现在搜索结果页的第3位、第13位和第23位。出价不同，获得的广告位也不同。

![](img/7f011bdfb75763168d8a990bfcfa02c9_23.png)

关于广告投放的更多常见问题，可以在小红书官方平台的问答页面找到详细解答。

![](img/7f011bdfb75763168d8a990bfcfa02c9_25.png)

需要注意的是，官方广告投放的费用相对较高。因此，在资金允许的情况下，建议考虑与官方合作投放竞价广告，因为它能直接获取由平台算法推荐的精准流量。

本节课中我们一起学习了小红书的两大核心推广路径。首先，我们梳理了企业号、个人号和引流号的特点，并给出了构建账号矩阵的具体建议，核心在于**内容比例（3:7）** 与**多账号协同**。接着，我们介绍了官方的CPC竞价广告，包括信息流与搜索广告两种形式，其本质是**通过关键词出价竞争广告位**。对于有预算的品牌，官方广告是获取精准流量的高效途径。
