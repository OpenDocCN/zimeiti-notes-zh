### 1. 抖音短视频运营 🎬

![](img/7fdf36ac26417a7f74bcf3d049be6e1b_21.png)

抖音是当前主流的短视频平台，掌握其运营方法对理解其他平台（如视频号、快手、西瓜视频）也大有裨益。

![](img/7fdf36ac26417a7f74bcf3d049be6e1b_23.png)

以下是抖音运营板块需要学习的内容体系：

![](img/7fdf36ac26417a7f74bcf3d049be6e1b_25.png)

*   **平台基础与官方资源**：学习抖音平台的基本概念、官方账号提供的流量扶持活动等。
*   **流量算法机制**：理解账号冷启动、数据加权、延后爆发、流量触顶等核心算法逻辑。
*   **核心数据指标**：掌握五个最重要的数据指标及其优化方法。
*   **十大运营要素**：了解运营短视频账号必须关注的十个关键点。
*   **账号定位与布局**：学习账号的全局定位、布局策略及流量分发机制。
*   **电商与权重管理**：了解抖音小店运营、账号权重提升及养号方法。
*   **垂直领域与变现**：学习打造垂直领域内容、个人账号变现模式及矩阵运营方法。
*   **快速起号赛道**：掌握在抖音快速启动账号的赛道选择。

### 2. 小红书图文笔记运营 📝

![](img/7fdf36ac26417a7f74bcf3d049be6e1b_31.png)

小红书是经典的图文笔记运营平台，其运营逻辑与短视频平台有所不同。

![](img/7fdf36ac26417a7f74bcf3d049be6e1b_33.png)

![](img/7fdf36ac26417a7f74bcf3d049be6e1b_35.png)

以下是小红书运营板块的核心学习点：

![](img/7fdf36ac26417a7f74bcf3d049be6e1b_37.png)

*   **系统化起号思路**：学习从零开始运营小红书的步骤，包括定位、垂直领域打造及差异化内容组合。
*   **官方号与素人号运营**：掌握如何玩转平台机制，协同运营品牌官方号与素人号，进行品牌铺设。
*   **隐藏流量入口**：了解六个能带来显著流量的隐藏流量值，对前期起号帮助巨大。
*   **私域引流与店铺运营**：学习小红书独特的私域引流方法及店铺运营技巧。
*   **达人种草与商业合作**：掌握KOC/KOL合作、广告报价与返点谈判等商业化变现方法。

### 4. 直播带货运营 🛒

![](img/7fdf36ac26417a7f74bcf3d049be6e1b_63.png)

直播带货是当下最火的电商营销模式，即使不亲自操作，也需要深入了解。

以下是直播带货板块的核心知识：

![](img/7fdf36ac26417a7f74bcf3d049be6e1b_65.png)

![](img/7fdf36ac26417a7f74bcf3d049be6e1b_67.png)

*   **“人、货、场”搭建**：学习直播带货中人物、产品、场景的搭配逻辑。
*   **全流程策划**：掌握直播前、中、后期需要准备的内容，确保直播效果。
*   **专业术语解读**：理解各类直播带货名词术语的含义及其带来的结果。
*   **脚本撰写与策划**：学习如何撰写吸引人群的直播脚本和完整的策划方案。
*   **全步骤实操**：了解从起号、选品、视频拍摄、场景布置到直播投流、数据复盘的全部流程。
*   **账号系统化运营**：学习如何从零启动直播带货账号，突破新手期，结合自然流与付费流。
*   **流量玩法与工具**：掌握直播间道具玩法、自然流权重提升以及付费投流工具（如千川、小店随心推）的使用。

![](img/7fdf36ac26417a7f74bcf3d049be6e1b_75.png)

优质的内容离不开专业的拍摄与剪辑，这是运营者的基本功。

![](img/7fdf36ac26417a7f74bcf3d049be6e1b_77.png)

以下是需要掌握的内容创作技能：

![](img/7fdf36ac26417a7f74bcf3d049be6e1b_79.png)

![](img/7fdf36ac26417a7f74bcf3d049be6e1b_81.png)

*   **视频剪辑核心**：学习配合短视频打法的核心剪辑技巧，如转场、特效、定格、关键帧等。
    ```markdown
    **核心剪辑要素** = 转场 + 特效 + 定格 + 关键帧
    ```
*   **短视频构思与创作**：学习短视频标题、脚本的撰写方法，构思选题方向与文案。
*   **结构设计与完播率**：掌握通过优化视频前几秒内容来吸引目标人群、提升完播率的技巧。这考验的不仅是拍摄手法，更是对短视频内容结构的理解。

### 6. AI智能辅助运营 🤖

在当今的互联网运营中，利用AI工具提升效率已成为必备技能。本课程将独家教授如何利用ChatGPT等AI进行智能辅助运营。

![](img/7fdf36ac26417a7f74bcf3d049be6e1b_83.png)

以下是AI运营板块的学习重点：

![](img/7fdf36ac26417a7f74bcf3d049be6e1b_85.png)

*   **AI与内容生产**：理解新媒体运营以内容串联用户关系，而AI作为强大的文字生成模型，能辅助生成爆款内容、打造人设、撰写脚本。
*   **高效使用ChatGPT**：学习如何快速上手ChatGPT，并进行进阶应用。
*   **对话模型与指令调教**：了解GPT-3.5与GPT-4.0的区别，学习通过优化Prompt（指令）来训练AI，使其更准确地理解产品与需求。
*   **定制化工具搭建**：学习搭建专为特定行业服务的GPTS小程序，让AI真正成为得力的工作助手。
    ```markdown
    **高效AI应用** = 优质Prompt + 针对性训练 + 定制化工具
    ```
