# 抖音运营教程：P5：3.账号定位 🎯

![](img/44aff79cb29879bee02e11870e6310ff_0.png)

在本节课中，我们将要学习抖音账号运营中至关重要的一步：**账号定位**。清晰的定位是账号快速起号、获得高权重的基础。我们将通过反思三个核心问题来构建你的账号定位。

![](img/44aff79cb29879bee02e11870e6310ff_2.png)

---

上一节我们介绍了抖音的赛道选择，本节中我们来看看如何为你的账号进行精准定位。对账号定位的理解，需要反思三个问题：**你是谁**、**你是干什么的**、**为什么要关注你**。

## 确定账号类型与细分赛道

首先，我们要确定接下来要做的账号类型。除了之前章节提到的大类赛道（即抖音定义的一级垂类），我们还需思考在一级垂类下细分的二级赛道是什么。

这好比参加运动会，你报的项目是“田径”，这是你的一级垂类。而你最擅长且具有优势的项目是“百米赛跑”，这就是你的二级垂类。因此，我们可以通过定位账号的细分类目，来确定账号的垂直类型。

![](img/44aff79cb29879bee02e11870e6310ff_0.png)

在这张图中可以看到，在抖音中，无论任何行业，只要定位足够清晰，都有机会成功。因为抖音流量足够大，一旦做起来，增长和裂变的速度将非常快。

## 明确账号主体

其次，我们要确定账号主体是谁。账号主体是该账号的拥有人，并最终呈现给粉丝的账号主人公。即，要让粉丝明确知道是在看谁的作品。

在抖音商业类账号中，账号主体主要分为两种：
*   **商家账号**：如果账号主体是公司，即为商家账号。那么账号内容就要偏向商业化，并需不断通过内容拉近与粉丝的距离，避免严肃。这样既能建立信任感，也能带来好感，比较符合抖音的娱乐属性。
*   **达人账号**：如果账号主体是个人，即为达人账号。那么首先应解决信任问题。可以通过IP打造的方式，为账号树立一个积极正面的人设。

## 明确账号内容与职业

然后，我们要明确这个账号是做什么的。最直接的方式是通过抖音认证身份，也就是我们的职业。

*   如果是达人，可以通过认证**黄V**。
*   如果是公司，可以通过认证**蓝V**。

认证职业之后，粉丝可以清晰看到我们的认证标签，从而明白我们的职业以及这个账号大概做什么。同时，商家认证蓝V或达人认证黄V，也可以获取平台背书，增加信任感。

在未认证之前，也可以通过账号设置向粉丝明确传达你是做什么的。以下是可进行业务关联统一设置的几个方面：
*   头像
*   背景图
*   简介
*   视频封面

## 思考为粉丝提供的价值

最后，我们需要花时间思考一个问题：**你将为粉丝提供什么样的价值？** 在抖音中，粉丝对于博主的价值需求主要分为以下几类：

以下是粉丝价值需求的几种主要类型：

1.  **知识**
    *   **认知型**：如人生思考、思维方式、读书分享和感悟、行业内幕、成功经验、个人提升等。
    *   **干货型**：如技能提升、经验总结、过程步骤、冷知识等。
    *   **常识普及型**：如生活常识、健康常识、产品常识、科学常识、历史常识、地区常识的解读和普及。
    *   **提供价值**：认同（如三观认同、观点认同）、减少知识盲区的获得感，进而产生对作者的认同。

2.  **情绪**
    *   **类型**：开心愉悦（如搞笑类、夸张表情）、人物关系共鸣（如情侣、婆媳、夫妻、家庭、上下级、师生关系）、情绪波动（如争议话题、反转反差内容、反常识开场激发）、怀旧、emo、震惊、好奇、颠覆三观等。
    *   **提供价值**：**共鸣**。利用表情、文字、画面、声音、肢体动作等抒发准确到位的情绪，直接感染观众。

3.  **感觉**
    *   **类型**：视频的画面美感、构图；观看产生的喜欢、治愈、视觉冲击；对作者人格魅力的仰慕；视频声音带来的节奏感、心跳加速或听觉享受。例如颜值类、萌宠类、美景类、才艺类、明星大咖、文化类账号。
    *   **提供价值**：**共情**。因表情、文字、画面、声音、肢体动作等的自然表现，让人产生联想，并与作者在情绪、认知和观念等意识层面产生连接，从而间接引发共鸣。

4.  **体验**
    *   **类型**：抖音让用户看到不一样的生活方式，了解去不了的地方，体验未曾体验的事物。例如流量最多的美食账号（享用体验、制作体验）、各式旅行体验、运动体验、角色体验等。
    *   **提供价值**：主要是好奇心的满足，以及紧张感、激动、感动、体验获得感。

5.  **期待**
    *   当用户被你的短视频内容吸引，或对作者本人产生好奇与期待时，看完视频会产生意犹未尽的感觉，从而点击关注，期待下次更新。很多博主会利用这一点，不将所有内容在一个视频中呈现完，以制造悬念。

---

本节课中我们一起学习了抖音账号定位的核心方法。我们通过反思“你是谁、做什么、为什么关注你”这三个问题，逐步明确了账号的细分赛道、主体身份、内容方向以及能为粉丝提供的核心价值。清晰的定位是账号运营的基石，希望你能根据这些要点，打造出属于自己的高权重抖音账号。
