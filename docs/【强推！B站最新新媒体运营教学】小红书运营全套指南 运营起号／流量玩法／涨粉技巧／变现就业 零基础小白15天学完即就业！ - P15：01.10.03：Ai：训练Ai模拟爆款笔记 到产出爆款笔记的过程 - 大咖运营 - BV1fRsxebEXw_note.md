## 分析爆款笔记的核心要点

![](img/beac4bc19af1d273cc58eacdc0122524_7.png)

![](img/beac4bc19af1d273cc58eacdc0122524_9.png)

要让AI学习爆款，我们需要引导它分析几个关键维度。以下是分析时需要关注的几个核心点：

1.  **爆款因素分析**：分析内容为何能成为爆款，例如是否引发了情感共鸣、提供了实用价值或讲述了逆袭故事。
2.  **文案结构拆解**：分析文案的起承转合。
    *   **开篇**：用什么方式（如提问、制造焦虑、引用名人）吸引用户继续阅读。
    *   **中间**：提供了哪些内容来戳中用户痛点，满足需求，提升用户停留时长。
    *   **结尾**：如何设置“钩子”（如鼓励互动、引导私信、提供福利）促使用户行动。
3.  **关键词策略分析**：分析笔记是否刻意植入了关键词。
    *   如果有，关键词是什么？
    *   关键词的出现频率和分布结构是怎样的？

### 第一步：收集与分析爆款笔记

![](img/beac4bc19af1d273cc58eacdc0122524_27.png)

![](img/beac4bc19af1d273cc58eacdc0122524_29.png)

首先，我们需要找到目标领域（如“考研”）的爆款笔记，并将文案提供给AI进行分析。

![](img/beac4bc19af1d273cc58eacdc0122524_31.png)

![](img/beac4bc19af1d273cc58eacdc0122524_33.png)

![](img/beac4bc19af1d273cc58eacdc0122524_35.png)

![](img/beac4bc19af1d273cc58eacdc0122524_37.png)

我们给AI的指令可以如下：
```
你是一名小红书爆款文案分析师。我将提供一篇小红书上的爆款笔记文案，请你分析：
1. 爆款因素：为什么这篇笔记能火？请列出关键点。
2. 文案结构：
   - 开篇用什么吸引了用户？
   - 中间用什么内容戳中了用户痛点？
   - 结尾用什么方式引导用户互动（如评论、私信、关注）？
3. 关键词分析：该笔记是否有结合中文分词法设计关键词？如果有，请指出关键词及其出现频率。
```

![](img/beac4bc19af1d273cc58eacdc0122524_39.png)

![](img/beac4bc19af1d273cc58eacdc0122524_41.png)

![](img/beac4bc19af1d273cc58eacdc0122524_43.png)

![](img/beac4bc19af1d273cc58eacdc0122524_45.png)

将多篇爆款笔记文案依次输入，让AI进行分析。AI会反馈类似这样的分析结果：
*   **爆款因素**：情感共鸣、提供逆袭故事、给出明确备考计划、时间节点清晰、正能量鼓励。
*   **文案结构**：开篇用焦虑情绪吸引→中间分享详细备考方法和资源→结尾鼓励并引导提问。
*   **关键词分析**：本篇无明显刻意设计的关键词。

**注意**：AI（特别是3.5版本）可能在分析多篇内容后“遗忘”最初指令，需要适时提醒或重新输入指令。

### 第二步：整合爆款要素

在AI分析完数篇笔记后，指令它将这些分析结果进行整合、去重，总结出该领域爆款笔记的通用要素列表。

![](img/beac4bc19af1d273cc58eacdc0122524_56.png)

指令示例：
```
请将你之前分析过的几篇考研爆款笔记的爆款因素总结一下，整合成一个清晰的列表。
```

![](img/beac4bc19af1d273cc58eacdc0122524_58.png)

AI可能会生成如下列表：
*   情感共鸣（焦虑、逆袭）
*   提供明确的阶段性备考计划
*   给出针对性强的实用建议
*   强调时间管理与重要节点
*   推荐具体资源与学习技巧
*   使用表情符号增加亲和力
*   结尾进行正能量鼓励并引导互动

### 第三步：生成爆款笔记大纲与文案

最后，我们指令AI运用学习到的爆款要素，创作一篇新的笔记。

![](img/beac4bc19af1d273cc58eacdc0122524_60.png)

![](img/beac4bc19af1d273cc58eacdc0122524_61.png)

![](img/beac4bc19af1d273cc58eacdc0122524_63.png)

**首先，生成大纲：**
```
请结合你总结的考研爆款笔记因素，为我规划一篇新的小红书笔记大纲。大纲需要结构清晰，每个主要部分下要有细分的要点。
```
AI生成的大纲可能包括：吸引人的标题、引发共鸣的开篇、专业选择建议、分科目备考规划、时间管理技巧、心态调整方法、结尾互动引导。

**接着，根据大纲生成完整文案：**
```
请根据上述大纲，撰写完整的笔记文案。要求：
1. 严格遵循爆款笔记的结构和要素。
2. 在开篇融入一个简短的“逆袭故事”引入，引发读者情绪。
3. 文案中需包含适量的emoji表情。
4. 尝试将“考研规划”、“政治备考”等关键词自然融入文中。
```
AI将输出一篇融合了爆款要素的完整笔记。这篇文案可能已经具备不错的基础，但通常还需要人工进行优化和调整，使其更精炼、更生动或更符合个人风格。

## 总结与进阶提示

本节课中，我们一起学习了训练AI产出爆款笔记的三步法：**分析爆款**、**整合要素**、**生成内容**。这个方法的核心在于通过精准的指令，让AI从优秀案例中学习规律，并应用于新内容的创作。

![](img/beac4bc19af1d273cc58eacdc0122524_65.png)

**进阶提示**：
1.  使用更强大的模型（如GPT-4）或创建自定义的GPTs，可以获得更高质量、更稳定的输出。
2.  在自定义GPTs中，你可以上传更多样本、设定固定的文案风格、甚至提供专属表情包库，打造更懂你的私人AI助手。
3.  AI生成的文案是初稿，运营人员需要在此基础上进行润色、优化和个性化调整，才能达到最佳效果。
