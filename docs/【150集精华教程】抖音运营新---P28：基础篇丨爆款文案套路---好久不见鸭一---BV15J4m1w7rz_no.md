# 抖音运营新手起号：P28：基础篇丨爆款文案套路

## 概述

![](img/66b73deb7f3bdbdad24e25ad574be041_0.png)

在本节课中，我们将要学习一个适用于各行各业的爆款文案创作套路。在内容为王的时代，掌握高效的脚本创作能力是运营者的必备技能。我们将通过一个简单的万能公式，帮助你快速写出能引起共鸣、提升视频数据的高质量文案。

## 什么是好文案

上一节我们提到了脚本创作能力的重要性，本节中我们来看看如何定义一篇好文案。一篇好文案的核心只有一点：**要让对方认为与自己有关**。

这即是所谓的“引起共鸣”。如果文案内容与观众无关，观众自然不会停留观看。

![](img/66b73deb7f3bdbdad24e25ad574be041_2.png)

## 万能文案公式

明白了核心目标后，我们来看看具体怎么做。这里提供一个创作爆款文案的万能公式：

**现象 + 危害 + 原因 + 解决办法**

以下是这个公式中每个部分的作用解析：

*   **现象**：用于筛选目标用户，让观众感觉“与我有关”，从而愿意停留。
*   **危害**：负责放大观众的焦虑或痛点情绪，防止观众在5秒内划走视频。
*   **原因**：引导观众继续观看，寻找答案，从而提高视频的完播率。
*   **解决办法**：给出具体方案或价值，鼓励观众点赞、收藏，提升互动率。

## 公式应用案例

为了帮助你更好地理解，我们来看两个将公式应用于不同行业的例子。

**案例一：服装行业**
*   **现象**：梨形身材的女孩太难买裤子了。
*   **危害**：挑错了样式显得腿又粗又短。
*   **原因**：但梨形身材是天生的，很难改变。
*   **解决办法**：而这三种裤子不仅显瘦，还显腿长，穿上第二天就被小哥哥搭讪了。姐妹们赶快收藏起来吧！

这段文案开头就锁定了“梨形身材”的目标客户，并通过公式的层层推进，吸引观众看完，最终引导互动。积累这样的精准粉丝后，后续变现也会更容易。

**案例二：运营行业**
*   **现象**：每天用心制作短视频，但发完只有100播放。
*   **危害**：继续下去只是浪费时间，徒增焦虑。
*   **原因**：这跟你没有系统的运营底层逻辑有关。
*   **解决办法**：现在去看完我的前三条视频，不花一分钱也能学会运营知识。还不快快行动起来！

## 公式的优势与你的任务

![](img/66b73deb7f3bdbdad24e25ad574be041_4.png)

用好这个套路，不管什么行业都可以直接套用，生成不足30秒的短文案。这种文案创作起来轻松，尤其在进行账号矩阵运营时，能极大提升内容生产效率。

以上我们已经举了两个例子（举一反二）。至于“反三”，就交给你了。你可以根据自己行业的特性，运用这个公式进行创作练习，并将你的思考过程分享出来。

## 总结

本节课中，我们一起学习了创作爆款文案的核心与万能公式。我们明确了好文案的标准是 **“让观众认为与自己有关”**，并掌握了 **“现象 + 危害 + 原因 + 解决办法”** 这一结构化写作框架。通过套用这个公式，你可以快速为任何行业创作出吸引目标用户、提升视频关键数据（停留、完播、互动）的优质文案。现在，就尝试用这个公式为你自己的领域写一段文案吧。
