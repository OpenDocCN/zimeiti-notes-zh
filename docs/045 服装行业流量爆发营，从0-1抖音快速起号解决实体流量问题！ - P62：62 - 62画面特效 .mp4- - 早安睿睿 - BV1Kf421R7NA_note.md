# 服装行业流量爆发营：P62：画面特效

在本节课中，我们将学习如何使用剪映软件为视频画面添加特效，从而提升视频的视觉吸引力和表现力，解决画面平淡、缺乏生气的问题。

![](img/e051f601b4c16767dc88e9a334567058_0.png)

---

上一节我们介绍了视频剪辑的基础操作，本节中我们来看看如何为画面增添活力。你的视频画面是否总感觉平淡，没有生气？本教程将帮助你解决这个问题。今天我们将讲解如何给画面增加特效。

首先，打开剪映软件，并导入一段包含人物的视频素材。

在软件的功能栏中，找到并点击“特效”功能。进入后，你会看到两个主要功能选项：**画面特效**和**人物特效**。

我们先来讲解**画面特效**的使用。

以下是使用画面特效的步骤：

1.  **选择特效**：在“画面特效”分类下，软件提供了多种自带的特效模板可供选择。
2.  **预览效果**：点击任意一个特效，即可在预览窗口中看到该特效应用到视频上的效果。
3.  **调整参数**：再次点击已选中的特效，可以进入参数调整界面，对特效的强度、范围等属性进行微调。
4.  **确认应用**：调整到满意的效果后，点击确认。该特效会作为一个独立的**效果图层**添加到视频轨道上。
5.  **管理特效图层**：
    *   点击轨道上的效果图层，拖动其首尾两端，可以改变特效作用的时长。
    *   选中效果图层后，在功能栏中还可以直接将其**替换**为其他特效素材，操作非常方便。

接下来，我们看看**人物特效**。

人物特效的使用方法与画面特效**完全一样**。你只需切换到“人物特效”分类，选择喜欢的特效（如大头、流汗等），然后通过上述相同的步骤进行应用和调整即可。

---

本节课中，我们一起学习了如何利用剪映的“画面特效”和“人物特效”功能来增强视频的视觉效果。通过添加和调整特效图层，你可以轻松地让视频画面变得更生动、更有趣，从而更好地吸引观众的注意力。
