# 剪映教程：P10：剪映核心剪辑功能讲解--画中画功能 🎬

在本节课中，我们将要学习剪映的核心剪辑功能之一：画中画。这个功能允许你在一个视频画面上叠加播放另一个或多个视频，从而创造出内容更丰富、更有趣的视频效果。

## 概述

画中画功能是视频剪辑中常用的技巧。我们经常看到一些视频中有多个画面同时播放，这些画面可能并不在同一个时间线上，这种效果就是通过画中画功能实现的。在剪映中，许多特效视频也借用了画中画功能，将两个或多个视频素材叠加在同一个时间轨道上同步播放。

![](img/33ff840c77d071dc88a99ebe9e80e1b2_1.png)

使用画中画的一个核心特点是能让视频内容变得更加丰富和有趣。

上一节我们介绍了其他核心功能，本节中我们来看看如何具体使用画中画功能。

## 画中画功能实操

![](img/33ff840c77d071dc88a99ebe9e80e1b2_3.png)

接下来，我们将通过一个具体的例子，一步步演示如何在剪映中使用画中画功能。

![](img/33ff840c77d071dc88a99ebe9e80e1b2_5.png)

首先，打开剪映软件，点击“开始创作”，导入一张照片作为背景素材。

![](img/33ff840c77d071dc88a99ebe9e80e1b2_7.png)

![](img/33ff840c77d071dc88a99ebe9e80e1b2_1.png)

![](img/33ff840c77d071dc88a99ebe9e80e1b2_9.png)

导入照片素材后，我们需要先调整视频画面的比例。点击底部工具栏的“比例”，选择 **9:16** 的竖屏比例。

![](img/33ff840c77d071dc88a99ebe9e80e1b2_7.png)

![](img/33ff840c77d071dc88a99ebe9e80e1b2_11.png)

调整好比例后，可以适当拉长照片的显示时长。接着，我们需要添加画中画。回到一级菜单，点击“画中画”功能，然后选择“新增画中画”。

![](img/33ff840c77d071dc88a99ebe9e80e1b2_12.png)

![](img/33ff840c77d071dc88a99ebe9e80e1b2_11.png)

![](img/33ff840c77d071dc88a99ebe9e80e1b2_14.png)

在素材库中，我们添加一个“星空特效”的视频素材。

![](img/33ff840c77d071dc88a99ebe9e80e1b2_16.png)

![](img/33ff840c77d071dc88a99ebe9e80e1b2_14.png)

![](img/33ff840c77d071dc88a99ebe9e80e1b2_17.png)

导入星空特效素材后，你可能会发现素材开头有多余的部分。我们需要将其删除。将时间线光标移动到需要保留的起始位置，点击“分割”工具，然后选中前面多余的部分，点击“删除”。

![](img/33ff840c77d071dc88a99ebe9e80e1b2_18.png)

![](img/33ff840c77d071dc88a99ebe9e80e1b2_18.png)

删除多余部分后，将剩下的特效素材拖动到时间线开头，与背景照片对齐。现在，我们就在主视频轨道（背景照片）上叠加了一个画中画轨道（星空特效）。

此时播放预览，你会看到星空特效带有黑色背景，覆盖在照片之上。为了让特效融合到背景中，我们需要去除黑色背景。

以下是处理步骤：
1.  选中时间线上的**画中画轨道**（星空特效）。
2.  在底部工具栏中找到并点击“**混合模式**”。
3.  在混合模式选项中，选择“**绿色**”模式。

![](img/33ff840c77d071dc88a99ebe9e80e1b2_20.png)

选择“绿色”模式后，黑色背景就被去除了，只留下发光的星星特效叠加在照片上。你还可以调整画中画的位置和大小，将其放置在合适的位置。

![](img/33ff840c77d071dc88a99ebe9e80e1b2_20.png)

最后，播放预览整个视频，你就得到了一个带有星空特效的动态照片效果。这个效果就是通过画中画功能实现的。

## 功能小结与总结

本节课中我们一起学习了剪映的画中画功能。

在剪映中，点击“画中画”功能，可以直接在主视频轨道上新增一个独立的视频轨道。这实现了多轨道视频的同步运行，让单一视频画面可以变成两个甚至多个画面的同步播放，极大地丰富了视频内容。

理解和掌握画中画功能，对我们后期制作特效视频非常有帮助。就像本节课的案例操作，我们通过画中画和混合模式，为一个静态照片添加了动态的星空特效。

**核心操作公式可以概括为：**
`主视频轨道 + 画中画轨道 + 混合模式（如绿色） = 多层复合特效视频`

未来在制作三连屏、多屏互动或人影分身等高级效果时，其技术基础都是画中画功能。
