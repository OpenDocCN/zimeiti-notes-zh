# 抖音运营：P15：内容的三有原则

在本节课中，我们将学习抖音内容创作的核心原则——“三有原则”。我们将通过分析具体案例，理解如何让内容对用户产生吸引力，并探讨为什么有些看似不错的账号却难以增长粉丝。

在探讨内容定位时，一个核心问题是：用户究竟喜欢什么样的作品？理解这一点，是创作出受欢迎内容的关键。

![](img/b2469e30d9595c36a7e7ab738ff8bb2f_11.png)

以下是几个视频案例，它们展示了不同类型的内容如何吸引观众。

![](img/b2469e30d9595c36a7e7ab738ff8bb2f_13.png)

*   **案例一：钢琴简谱教学**
    ![](img/b2469e30d9595c36a7e7ab738ff8bb2f_1.png)
    这是一个提供《卡农》简易版钢琴简谱的教学视频。对于钢琴爱好者或初学者而言，这类内容提供了实际的学习价值。

![](img/b2469e30d9595c36a7e7ab738ff8bb2f_15.png)

*   **案例二：宠物日常**
    ![](img/b2469e30d9595c36a7e7ab738ff8bb2f_7.png)
    这是一个关于宠物“小汪”的趣味视频。喜欢宠物的人会对这类内容产生天然的兴趣，并可能因此进行点赞互动。

![](img/b2469e30d9595c36a7e7ab738ff8bb2f_17.png)

*   **案例三：情感共鸣内容**
    ![](img/b2469e30d9595c36a7e7ab738ff8bb2f_13.png)
    这段视频讲述了男性成长中的沉默与担当，容易引发特定观众群体的情感共鸣，促使他们关注发布者。

通过以上案例，我们可以总结出一个全网通用的内容受欢迎原则。

这个原则被称为“三有原则”，即内容需要具备以下至少一个特点：**有用处**、**有兴趣**、**有共鸣**。

为了更清晰地理解，我们可以将常见的抖音内容类型归类到这三个原则下：

![](img/b2469e30d9595c36a7e7ab738ff8bb2f_19.png)

![](img/b2469e30d9595c36a7e7ab738ff8bb2f_21.png)

*   **有用处**：教程类、推荐类等干货内容，能为用户提供实际价值。
*   **有兴趣**：美好的画面、剧情、颜值、萌宠、炫技、恶搞情景剧等，能激发用户的观看兴趣。
*   **有共鸣**：励志类、鸡汤类（包括反鸡汤、毒鸡汤）等内容，能引发用户的情感共鸣。

![](img/b2469e30d9595c36a7e7ab738ff8bb2f_23.png)

理解“三有原则”后，创作者在构思内容时，可以首先从这三个方向进行延伸。然而，面对众多内容类型，新手可能会感到困惑：究竟该选择哪一类？

在确定具体内容类型之前，我们还需要理解另一个关键概念：账号垂直度。让我们通过一个案例来探讨。

请看这个唱歌视频：
![](img/b2469e30d9595c36a7e7ab738ff8bb2f_19.png)
视频的拍摄和演唱质量都不错，也获得了一定的点赞（5.8万），但该账号的粉丝数却很低（仅5173）。这引出了一个常见问题：**为什么作品不差，却不涨粉丝？**

计算一个简单的比例可以反映问题：
`粉丝转化率 = 粉丝数 / 获赞总数`
该账号的粉丝转化率明显偏低。可能的原因并非内容不垂直（该账号主页显示其持续发布唱歌内容），而是存在更深层的问题。

**垂直度**指的是账号持续专注于某一特定领域（如只做美食、只做美妆）。对于大多数普通创作者而言，保持垂直是必要的，因为这能让用户对你的账号产生明确预期，从而有理由关注你。除非创作者个人魅力极强，否则随意更换领域很难留住粉丝。

那么，上述案例中账号不涨粉的核心问题可能在于：**主题IP不够鲜明**或**账号整体搭建（如主页、简介）存在问题**。我们将在后续章节中详细分析这些因素。

本节课中我们一起学习了抖音内容的“三有原则”（有用处、有兴趣、有共鸣），这是创作受欢迎内容的基础框架。同时，我们通过案例分析了“垂直度”的重要性，并初步探讨了影响粉丝增长的其他关键因素。理解这些概念，能帮助你在内容创作初期找准方向。
