接下来是考验环节。请看以下视频内容，并判断它属于“强人设”还是“弱人设”视频。

![](img/41447f3a162544c31bd874cb8c91eb7d_13.png)

![](img/41447f3a162544c31bd874cb8c91eb7d_15.png)

视频文案核心观点如下：
穷人改变命运最大的难点在于其父母和原生阶层。如果生活在父母身边并深受其影响，则很难改变命运。因为如果父母的观念正确，他们自己就不会是穷人。穷人翻身通常需要离开家乡、远离父母、独自打拼。其次，穷人的朋友往往也是穷人，他们的思维方式可能同样存在问题。第三，穷人常常缺乏抓住机会所需的关键本金，且不敢承担风险。而许多富人敢于冒险和借贷。最后，长期处于社会边缘可能导致不自信和逃避责任，难以团结他人。因此，改变命运需要：离开家乡、结识新朋友、敢于在能力范围内承担风险、以及建立自信。

## 人设类型判断与原因

![](img/41447f3a162544c31bd874cb8c91eb7d_21.png)

![](img/41447f3a162544c31bd874cb8c91eb7d_23.png)

看完视频后，我们可以得出结论：这是一个典型的**弱人设**视频。

![](img/41447f3a162544c31bd874cb8c91eb7d_25.png)

![](img/41447f3a162544c31bd874cb8c91eb7d_27.png)

判断依据如下：
*   **内容与讲述者无关**：视频讨论的是“穷人”与“富人”的普遍现象，但并未涉及视频创作者自身的经历、观点或故事。
*   **形式与内容分离**：视频的画面（可能是指治愈系的空镜头或动画）与所讲述的严肃社会观点没有直接关联，是“两张皮”。

## 爆款视频的成功秘诀 🔑

![](img/41447f3a162544c31bd874cb8c91eb7d_33.png)

![](img/41447f3a162544c31bd874cb8c91eb7d_35.png)

那么，为什么这样一个“弱人设”视频能够火爆呢？关键在于它实现了 **“双向吸引”**。

![](img/41447f3a162544c31bd874cb8c91eb7d_37.png)

![](img/41447f3a162544c31bd874cb8c91eb7d_39.png)

以下是其吸引流量的核心逻辑：
*   **吸引群体A**：一部分观众被视频的**画面形式**（如治愈系画面）所吸引。
*   **吸引群体B**：另一部分观众被视频的**观点内容**所触动，产生了强烈共鸣。
*   **效果叠加**：两种吸引点并存，扩大了视频的受众覆盖面，让不同喜好的人都能找到停留的理由。

## 方法论迁移与应用 ✨

![](img/41447f3a162544c31bd874cb8c91eb7d_45.png)

上一节我们分析了案例成功的原理，本节中我们来看看如何将“双向吸引”法则应用到自己的视频创作中。

![](img/41447f3a162544c31bd874cb8c91eb7d_47.png)

![](img/41447f3a162544c31bd874cb8c91eb7d_49.png)

![](img/41447f3a162544c31bd874cb8c91eb7d_51.png)

其核心公式可概括为：
**普通画面/场景 + 能引发共鸣的文案/故事 = 提升完播率与吸引力**

![](img/41447f3a162544c31bd874cb8c91eb7d_53.png)

以下是具体的应用思路：
*   **宝妈博主**：如果单纯拍摄宝宝或美食无人观看，可以尝试为画面配上**有趣的育儿故事**或**家庭情感话题**。
*   **实体店商家**：如果单纯拍摄产品或工厂无人观看，可以尝试讲述**你的创业故事**、**行业洞察**或**客户服务经历**。

这种方法既能通过故事或观点吸引与你产生共鸣的潜在客户，又能让他们在倾听的过程中看完你的产品展示。

![](img/41447f3a162544c31bd874cb8c91eb7d_55.png)

## 课程总结

本节课我们一起学习了一个爆款视频案例。我们首先判断了其“弱人设”的属性，然后揭示了它通过 **“形式吸引”** 与 **“内容共鸣”** 实现“双向吸引”的底层逻辑。最后，我们探讨了如何将这一逻辑迁移到不同领域的账号运营中，为普通画面赋予能打动人心的故事或观点，从而提升视频的完播率和吸引力。掌握这一小技巧，是优化视频数据、突破流量瓶颈的关键之一。
