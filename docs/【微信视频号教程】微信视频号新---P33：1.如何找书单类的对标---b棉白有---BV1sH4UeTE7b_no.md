# 短视频运营教程：P33：如何寻找书单类对标账号 📚

在本节课中，我们将学习在短视频平台（如抖音）上，如何高效地找到与您想做的账号类型（例如书单类）相似的对标账号。掌握这个方法，可以帮助您快速了解市场，为后续的内容创作和账号运营提供参考。

上一节我们介绍了寻找对标账号的重要性，本节中我们来看看两种具体、高效的实操方法。

## 方法一：利用“关注”推荐功能

此方法的核心是利用平台的推荐算法。当您关注一个账号后，平台会基于该账号的标签，为您推荐一批相似账号。

![](img/c6e7b5757946f283b4d80811e6dee5fc_1.png)

操作步骤如下：

1.  **搜索与筛选**：在抖音搜索与您目标领域相关的关键词（例如“书单”）。从搜索结果中找到一个视频类型符合您预期、且非真人出镜的账号。
2.  **初步考察**：点击该账号头像进入主页，浏览其视频内容，确认是否与您想做的类型相符。
3.  **触发推荐**：若符合，点击“关注”按钮。关注后，页面下方会立即弹出“你可能感兴趣”的账号列表。这些账号通常与您刚关注的账号标签相似。
4.  **批量筛选**：逐个点击弹出的推荐账号进行查看。如果其作品是您想模仿的类型，则继续点击关注。如果不是，则返回并滑动查看下一个推荐账号。

![](img/c6e7b5757946f283b4d80811e6dee5fc_1.png)

![](img/c6e7b5757946f283b4d80811e6dee5fc_3.png)

通过此流程，您可以在两三分钟内关注数十个同类账号。此后，平台的信息流会为您推荐更多相似账号。

![](img/c6e7b5757946f283b4d80811e6dee5fc_4.png)

## 方法二：通过商品“达人秀”寻找

此方法适用于已有明确商品（如特定书籍）目标的用户。通过搜索商品并查看其销售页面，可以找到推广该商品的大量达人账号。

![](img/c6e7b5757946f283b4d80811e6dee5fc_6.png)

![](img/c6e7b5757946f283b4d80811e6dee5fc_8.png)

以下是具体操作步骤：

1.  **搜索目标商品**：在抖音右上角点击搜索，输入具体的商品名称，例如“墨菲定律漫画版”。
2.  **筛选商品结果**：在搜索结果页，点击“商品”标签，然后选择按“销量”排序，以找到销量最高的商品链接。
3.  **进入商品详情页**：点击销量高的商品，进入其商品详情页。
4.  **查找达人视频**：在商品详情页中，找到“达人推荐”或“买家秀”板块（通常标注为“达人秀”）。这里汇集了众多推广该商品的短视频账号。

![](img/c6e7b5757946f283b4d80811e6dee5fc_8.png)

![](img/c6e7b5757946f283b4d80811e6dee5fc_10.png)

5.  **分析与关注**：浏览“达人秀”中的视频，点击您感兴趣的视频进入达人主页。考察其账号内容、视频形式和销售数据。如果符合您的对标要求，即可关注。

例如，一个销售2万册以上的书籍，其“达人秀”中可能包含数百个推广视频，这意味着有数百个同领域账号可供您参考和学习。这些账号的视频类型通常高度相似，如影视混剪、名人语录剪辑等，非常适合模仿。

![](img/c6e7b5757946f283b4d80811e6dee5fc_10.png)

---

本节课中我们一起学习了两种在抖音寻找对标账号的实用方法：**利用“关注”后的系统推荐**和**通过商品“达人秀”挖掘**。掌握这两种方法，您就能快速建立起一个丰富的同领域账号参考库，为您的短视频创作之路打下坚实基础。
