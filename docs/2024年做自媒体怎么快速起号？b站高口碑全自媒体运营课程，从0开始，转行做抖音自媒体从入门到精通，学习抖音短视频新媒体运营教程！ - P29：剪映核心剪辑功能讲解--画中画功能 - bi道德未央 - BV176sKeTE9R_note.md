# 自媒体运营课程：P29：剪映核心剪辑功能讲解--画中画功能 🎬

在本节课中，我们将要学习剪映的核心剪辑功能之一——画中画。这个功能允许我们在一个视频画面上叠加播放另一个或多个视频，从而创造出内容更丰富、更具视觉冲击力的视频效果。

![](img/548152e49176921d8ca33641de01d8d4_1.png)

## 画中画功能概述

上一节我们介绍了其他剪辑功能，本节中我们来看看画中画。我们经常看到一些视频中，有多个画面同时进行，但这些画面并不在同一个时间线上。这种效果就是通过画中画功能实现的。

![](img/548152e49176921d8ca33641de01d8d4_3.png)

![](img/548152e49176921d8ca33641de01d8d4_5.png)

在剪映中，许多特效视频也借用了画中画功能。其原理是将两个或多个视频素材叠加在同一个时间轨道上同步播放。使用画中画的一个重要特点是能让视频内容变得更加丰富和有趣。

![](img/548152e49176921d8ca33641de01d8d4_7.png)

关于画中画的介绍就到这里，接下来我们将带大家进行实际操作。

![](img/548152e49176921d8ca33641de01d8d4_9.png)

## 画中画功能实操演练

![](img/548152e49176921d8ca33641de01d8d4_11.png)

![](img/548152e49176921d8ca33641de01d8d4_13.png)

以下是使用剪映添加和调整画中画的具体步骤。

![](img/548152e49176921d8ca33641de01d8d4_14.png)

![](img/548152e49176921d8ca33641de01d8d4_16.png)

1.  **开始创作并导入素材**
    首先，点击“开始创作”按钮，导入一张照片作为主视频的背景素材。

    ![](img/548152e49176921d8ca33641de01d8d4_5.png)

![](img/548152e49176921d8ca33641de01d8d4_18.png)

2.  **调整主视频比例**
    导入照片后，先对主视频的比例进行调整。点击底部工具栏的“比例”，选择“9:16”以适应手机竖屏播放。

    ![](img/548152e49176921d8ca33641de01d8d4_9.png)

![](img/548152e49176921d8ca33641de01d8d4_19.png)

![](img/548152e49176921d8ca33641de01d8d4_20.png)

3.  **新增画中画轨道**
    适当拉长主照片的时长。然后返回一级菜单，点击“画中画”功能，选择“新增画中画”。

    ![](img/548152e49176921d8ca33641de01d8d4_11.png)

4.  **导入画中画素材**
    从素材库中再添加一个其他素材，例如我们选择一个星空粒子特效素材。

    ![](img/548152e49176921d8ca33641de01d8d4_13.png)
    ![](img/548152e49176921d8ca33641de01d8d4_14.png)

![](img/548152e49176921d8ca33641de01d8d4_22.png)

![](img/548152e49176921d8ca33641de01d8d4_23.png)

5.  **剪辑画中画素材**
    导入特效素材后，如果前面有多余部分（如文字），需要将其删除。将时间线光标移动到合适位置，点击“分割”，然后删除前面的片段。将剩余部分调整到视频开头。

    ![](img/548152e49176921d8ca33641d8ca33641de01d8d4_22.png)

6.  **使用混合模式抠除背景**
    此时，我们看到画中画素材带有黑色背景。我们希望粒子特效叠加在照片上，需要去掉黑色背景。选中画中画轨道，在底部工具栏选择“混合模式”，然后点击“绿色”模式。这样黑色背景就被去除了。

    ![](img/548152e49176921d8ca33641de01d8d4_25.png)

![](img/548152e49176921d8ca33641de01d8d4_25.png)

7.  **调整位置并预览效果**
    最后，调整画中画素材的位置和大小，然后播放预览整体效果。这样，我们就通过画中画功能，为一张静态照片添加了动态的星空粒子特效。

    ![](img/548152e49176921d8ca33641de01d8d4_27.png)

## 课程总结

![](img/548152e49176921d8ca33641de01d8d4_27.png)

本节课中，我们一起学习了剪映中的画中画功能。在剪映中，点击画中画功能可以直接在原视频轨道上新增一个独立的视频轨道，实现多轨道视频的同步运行。这能让单一视频画面变成两个甚至多个画面同时播放，极大地丰富了视频的内容。

理解和掌握画中画功能，对我们后期制作特效视频有很大帮助。正如本节课的案例操作所示，我们通过画中画实现了一个简单的特效叠加。后续在制作三分屏、人影分身等高级效果时，都会用到画中画这一核心功能。
