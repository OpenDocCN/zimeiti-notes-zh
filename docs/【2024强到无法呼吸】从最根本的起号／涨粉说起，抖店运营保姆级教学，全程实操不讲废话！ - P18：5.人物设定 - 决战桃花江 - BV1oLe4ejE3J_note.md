# 抖音运营实战教程：P18：5.人物设定

![](img/32428fb2fd1f42640d43c24fdddd03e2_0.png)

在本节课中，我们将要学习抖音账号运营中一个至关重要的环节——人物设定（简称“人设”）。我们将探讨人设的重要性、如何系统性地策划人设，以及如何通过“超级符号”强化人设记忆点。

![](img/32428fb2fd1f42640d43c24fdddd03e2_2.png)

## 人设策划的核心概念

![](img/32428fb2fd1f42640d43c24fdddd03e2_4.png)

上一节我们介绍了账号定位的基础，本节中我们来看看如何将定位具体化为一个生动的人物形象。

![](img/32428fb2fd1f42640d43c24fdddd03e2_6.png)

在抖音中建立人设，就像在游戏中创建角色一样，需要提前对人物角色进行设定。例如，设定角色的**称呼**、**职业**、**拥有的技能点**等。

![](img/32428fb2fd1f42640d43c24fdddd03e2_8.png)

但与游戏角色不同的是，我们在抖音中设定的人物角色是**人生角色**，因此需要更具**真实性**。

![](img/32428fb2fd1f42640d43c24fdddd03e2_10.png)

## 人设的重要性与目的

![](img/32428fb2fd1f42640d43c24fdddd03e2_12.png)

无论是短视频还是直播，一个成功的IP能够解决**粉丝信任**的问题。它能够起到**裂变传播**的作用，同时也具备很强的**变现能力**。

![](img/32428fb2fd1f42640d43c24fdddd03e2_14.png)

因此，我们运营账号，无论是创作内容还是策划选题，目的都是在获取流量的同时**打造IP**。打造IP的核心，就是在一定的场景或情境中，进行**账号人格化**的过程。

![](img/32428fb2fd1f42640d43c24fdddd03e2_16.png)

## 如何系统策划人设

![](img/32428fb2fd1f42640d43c24fdddd03e2_18.png)

账号人格化，也就是我们通常所说的账号人设。对于账号人设的策划，我们可以从以下几个方面着手。

以下是策划人设的六个核心维度：

1.  **名字/昵称**：一个IP首先得有一个叫法，这是IP传播的名片。这就像一个品牌的品牌名或商标名一样重要。
2.  **职业/职务**：人格化必然要具备身份。这是用户认识这个人或这个IP的联想入口，例如：`老师`、`教授`、`律师`。
3.  **称呼/社会关系**：一个IP有了亲切的称呼，能快速拉近与用户的距离，起到移情的作用。例如：`XX哥`、`XX大爷`、`XX姥姥`。
4.  **形象/第一印象**：以用户的视角来看待IP，使IP形象化，更容易占据用户心智。例如：`大叔形象`、`萌萌哒`、`铁憨憨`。
5.  **性格/爱好**：一个具有鲜明特点的IP，除了具有很强的记忆点，还具有很强的真实性。太完美的形象容易脱离大众。例如：`暴躁`、`娘炮`、`爱吃爱喝酒`。
6.  **穿着打扮/财富地位**：这属于社会的刻板印象和大众的普遍认知。借助用户的固有思维，可以为IP快速在用户心智中定位。例如：`开豪车`代表成功人士，`打扮土`可能代表老师。

## 从人设到账号搭建

以上内容策划好之后，就可以结合你自身的**产品优势**、**服务优势**等，将其实施到**账号搭建**中，即抖音账号的**五件套**：`头像`、`昵称`、`背景图`、`简介`、`视频封面`。

## 提炼人设的“超级符号”

除了对账号IP进行人格化的策划之外，还要通过选题和内容策划，为你的人设提炼出**记忆点**，即通常所说的**超级符号**。

以下是提炼超级符号的几个方向：

*   **画风（视觉）**：例如特定的`场景`、`道具`、`服饰`。
*   **听觉**：例如标志性的`BGM`、`特效音`、`配音声线`。
*   **文案**：能够反映优势且容易记忆的提炼词。例如：`“XX简单粗暴”`。
*   **表情/情绪**：例如标志性的`笑`、`惊恐`、`无奈`等表情。
*   **语言特色**：例如`方言`、`口音`、`错发音`、`慢条斯理的语调`。
*   **动作/习惯**：例如习惯性的`撩头发`、`扶眼镜`等肢体动作。

超级符号提炼好之后，在你进行短视频创作时，就可以不断地在视频中呈现，从而强化用户记忆，形成独特的账号标签。

## 课程总结

本节课中，我们一起学习了抖音账号**人物设定（人设）** 的完整流程。我们明白了人设的本质是**账号人格化**，其目的是建立信任、便于传播和变现。我们系统地学习了从六个维度策划人设的方法，并掌握了通过提炼**超级符号**来强化人设记忆点的技巧。将策划好的人设与账号五件套结合，并贯穿于内容创作中，是打造成功IP的关键一步。
