在本节课中，我们将学习小红书平台如何为用户打上标签，以及平台主要的流量分发入口。理解这些机制，有助于我们分析笔记如何被推荐给用户，并反向推导出如何让内容被更多精准用户看到。

上一节我们介绍了系统如何判定内容标签，本节中我们来看看平台如何判定用户标签。

## 用户标签判定机制

平台主要通过分析用户的**浏览行为**来为其打上标签。用户注册时选择的兴趣范围参考价值不大，仅作为初期内容方向的引导，无法进入系统深层的用户标签池。

系统根据以下用户行为数据来判定其兴趣分类：

以下是判定用户标签的核心行为指标：

1.  **点击率**
    *   **定义**：系统在发现页向用户推送多个领域的内容（如明星、美妆、家居、母婴、装修），用户点击哪一类内容更多，系统便能计算出其偏好。
    *   **作用**：通过点击率的逻辑，系统能为用户打上强关联或弱关联的标签。例如，用户频繁点击美妆内容，系统会判定其对美妆感兴趣，并持续推送相关笔记进行试探。

2.  **停留时长**
    *   **定义**：用户点开一篇笔记后观看的持续时间。
    *   **作用**：即使同属美妆领域，若用户快速划走某些笔记，说明内容不吸引他。系统会尝试推送该领域下其他细分内容（如上妆、护肤）或其他领域内容（如家居），并通过观察用户在新内容上的停留时长，进一步判断其即时兴趣或潜在需求（如最近想买盘子）。

3.  **页面浏览量**
    *   **定义**：用户浏览页面的深度，例如不仅看了单篇笔记，还点进博主主页并查看了多篇其他笔记。
    *   **作用**：提升用户在你主页的浏览深度，能为账号带来非常健康的评分，对账号成长有巨大帮助。

4.  **互动行为**
    *   **定义**：包括点赞、收藏、评论、关注等用户主动操作。
    *   **作用**：这些数据直接决定了笔记能否进入下一层更大的推荐流量池。高的互动数据不仅有助于笔记获得更多推荐，还能带动账号内其他笔记的曝光，提升粉丝数，是账号变现的基础。

通过以上行为，系统会为用户构建一个动态的“兴趣云图”，其中较大的标签代表用户的核心兴趣，系统会据此主要推送相关内容，以提升用户的获得感与平台停留时长。

## 平台流量分发入口

了解用户标签后，我们来看看小红书主要的流量分发入口。平台通过不同入口将内容匹配给不同需求的用户。

小红书目前主要有四个流量入口：

以下是四个核心流量入口及其特点：

1.  **发现页**
    *   **内容构成**：
        *   已关注达人发布的新内容。
        *   根据用户标签（点击率、停留时长等）推荐的、系统判定你可能喜欢的内容。
        *   少量系统推荐的、你可能未接触过的多样化内容，以丰富平台生态。
    *   **价值**：这是笔记和账号获取流量的最主要来源。

2.  **关注页**
    *   **内容构成**：纯粹展示所有已关注博主发布的内容。
    *   **价值**：用户主动订阅的内容流，流量精准但相对稳定。

3.  **附近页**
    *   **内容构成**：根据手机定位，推送附近及同城的相关内容。
    *   **价值**：以美食、本地旅游、线下店铺等强地域特征内容为主。对于推广线下实体店（如美容院、餐厅）的运营者，需要重点关注此入口，并在投放信息流广告时注意附近人群的定向。

![](img/8cf1b3572f2e52c87ad3b22e11638a70_4.png)

4.  **购物页**
    *   **内容构成**：小红书电商商城入口，展示商品及直播带货等内容。
    *   **价值**：平台向电商化发展的重要板块，直接匹配用户的购物需求。但目前用户习惯仍在培养中，许多人仍倾向于先搜索笔记“种草”。
