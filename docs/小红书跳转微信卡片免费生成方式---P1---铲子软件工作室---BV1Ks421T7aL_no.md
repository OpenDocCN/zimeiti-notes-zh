# 小红书私域运营教程：P1：跳转微信卡片生成与自动回复设置 🚀

在本节课中，我们将学习如何免费生成小红书跳转至微信的卡片，并设置自动回复功能，以实现私信的批量管理与私域流量的高效引导。

上一节我们概述了课程目标，本节中我们来看看具体的操作步骤。以下是生成跳转卡片与设置自动回复的核心流程。

**第一步：绑定小红书账号**
首先，你需要登录管理后台，找到一个用于绑定的账号入口。点击“账号管理”，选择“新建账号”。

![](img/73b5967f324038fe1522dcf5679f1053_5.png)

**第二步：填写账号信息**
在新建账号的界面中，输入你的小红书账号与密码。同时，你需要准备一张用于跳转的微信卡片。信息填写完毕后，点击“确定”，账号即成功绑定至后台。

**第三步：配置自动回复规则**
账号绑定后，点击“管理设置”或“回复信息”选项。在此处，你可以设置自动回复的具体内容。系统通常支持三种回复类型：
*   **文本回复**：直接回复预设的文字内容。
*   **笔记回复**：自动回复指定的小红书笔记链接。
*   **卡片回复**：自动发送跳转到微信的引导卡片。

![](img/73b5967f324038fe1522dcf5679f1053_7.png)

**第四步：启用自动回复功能**
所有回复规则设置完毕并确认后，返回账号管理列表，找到对应账号并点击“启动”按钮。至此，针对该账号的自动回复功能便已全部设置完成。
