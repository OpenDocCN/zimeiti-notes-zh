1.  **角色**
    *   角色即你在特定赛道中扮演的专业身份。你需要让用户相信，你提供的内容具有专业性。
    *   **公式：角色 = 赛道方向 + 专业身份**
    *   例如：
        *   **装修博主**：展示自己真实的装修过程、前后对比、避坑经验。
        *   **母婴赛道**：可以是分享亲身经验的**宝妈**，也可以是提供专业知识的**育儿护理专家**。
        *   **情感赛道**：可以是提供建议的**心理学顾问**，也可以是分享自身成长经历的**成长型人设**（如分享职场、创业心路的女性）。

2.  **内容**
    *   内容必须与你选择的赛道高度垂直且持续。这关系到账号的垂直度和平台标签，直接影响账号成长和流量分发。
    *   **核心原则**：你做什么赛道，就持续提供该赛道相关的内容。避免内容混杂，导致平台无法为你打上精准标签，影响推荐效果。
    *   例如：如果你的账号定位是儿童教育，那么内容应始终围绕此主题，而不是穿插宠物或旅行Vlog。

3.  **形式**
    *   形式指的是你呈现内容的具体方式。确定形式有助于统一账号风格，降低创作难度。
    *   以下是需要考虑的形式要素：
        *   **产出场景**：室内、室外、车内等固定或变化的拍摄地点。
        *   **声音呈现**：本人出镜讲解、使用配音、或朋友对话配音等形式。
        *   **内容类型**：剧情演绎、干货分享、生活记录等。不同类型对脚本、素材的要求不同。

4.  **记忆点**
    *   记忆点是让用户记住你的关键。它通过重复出现的固定元素，在用户心中形成深刻印象。
    *   记忆点主要分为两类：
        *   **前置型记忆点**：在视频开头几秒出现，冲击力强，用于快速建立认知。
            *   例如：固定的开场动作、特效、口头禅。
        *   **贯穿型记忆点**：在整个视频中持续存在，潜移默化地加深印象。
            *   例如：统一的背景音乐（BGM）、标志性的物品、特定风格的服装。
