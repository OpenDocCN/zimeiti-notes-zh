## 工具安装与文件准备

![](img/d1f04b6774f8f29c5672b390debc634e_8.png)

![](img/d1f04b6774f8f29c5672b390debc634e_9.png)

在开始实操前，我们需要完成工具的安装和基础文件配置。

![](img/d1f04b6774f8f29c5672b390debc634e_11.png)

首先，下载工具安装包。下载完成后，需要将特定文件放入手机指定文件夹。
以下是具体操作步骤：
1.  通过数据线连接手机与电脑。
2.  将名为“自定义”的文件夹（内含图片、`艾特.txt`和`私信.txt`文件）复制到手机存储根目录。
3.  也可使用手机端的MT管理器或文件管理器手动创建“自定义”文件夹并放入相应文件。

## 基础参数设置与权限开启

文件准备就绪后，我们需要在手机上开启必要权限并设置基础参数。

首先，在手机系统设置中开启“无障碍权限”。开启后，工具界面会显示“启停自动化”，请手动将其打开，显示“开启成功”。
接着，开启“悬浮窗权限”，以便随时查看和控制工具运行状态。

![](img/d1f04b6774f8f29c5672b390debc634e_13.png)

## 关键词精准曝光功能实操

权限设置完成后，我们来学习第一个核心功能：关键词精准曝光。

此功能用于自动关注并曝光精准用户。操作前，请确保已在抖音账号的头像、昵称、简介中设置好“钩子”（即吸引目标用户回关的内容）。
以下是操作流程：
1.  在工具中进入“关键词精准曝光”功能。
2.  选择筛选“一天内”的用户。
3.  可输入关键词进行筛选（如“创业”），关键词间用英文逗号 `,` 分隔。
4.  勾选要执行的操作，如“评论点赞”和“曝光关注”。建议选择1-2项，以降低检测风险。
5.  设置任务延迟时间。**演示时可设为2-4秒，但实际使用时建议设置为1-2分钟**，以保证账号安全。
6.  点击“保存参数”。
7.  打开抖音，点击工具上的“立即启动”，脚本将自动执行点赞和关注操作。

运行后，被关注的用户会在消息列表看到你，若你的“钩子”对其有吸引力，便会回关，从而实现引流。

![](img/d1f04b6774f8f29c5672b390debc634e_15.png)

## 全自动评论截流功能

![](img/d1f04b6774f8f29c5672b390debc634e_17.png)

曝光功能可以吸引用户主动回关，而评论截流则能让我们主动出现在潜在客户的视线中。

![](img/d1f04b6774f8f29c5672b390debc634e_19.png)

此功能用于在目标作品的评论区自动发布引流评论。
以下是操作步骤：
1.  在工具中找到“关键词评论”功能。
2.  在评论内容框中输入你的评论话术，不同话术用英文逗号 `,` 分隔，系统会随机选择发布以避免违规。
3.  点击“保存参数”。
4.  点击“立即启动”，工具将自动在目标作品下发布评论。

![](img/d1f04b6774f8f29c5672b390debc634e_21.png)

优质的评论钩子能带来持续的精准流量。此功能还可结合“回复首评”功能，通过回复作品下第一个评论来提升自己评论的曝光位置。

![](img/d1f04b6774f8f29c5672b390debc634e_23.png)

## 精准采集与自动私信功能

除了公开区域的曝光和评论，我们还可以通过私信进行更直接的触达。这需要先采集精准用户。

![](img/d1f04b6774f8f29c5672b390debc634e_27.png)

此功能分为两步：首先采集目标用户ID，然后进行自动私信。
以下是采集步骤：
1.  启动工具内的“评论采集”协议，并登录抖音网页版。
2.  设置采集条件，如“一天内”和关键词“创业”。
3.  点击“开始搜索”，工具会自动爬取符合条件的帖子。
4.  点击“获取评论”，采集所有评论用户数据。
5.  点击“导出数据”，将用户UID列表复制出来。

采集完成后，进行私信操作：
1.  将复制的UID列表，**每行一个**粘贴到手机“自定义”文件夹内的 `私信.txt` 文件中并保存。
2.  在工具的“私信功能”中，设置好私信话术（可设置多条）。
3.  启动私信功能，工具将自动向列表中的用户发送私信。**务必设置较慢的发送频率（如2分钟一条）**，以防触发平台限制。

## 导流与自动回复设置

用户被吸引后，我们需要将其引导至私域流量池（如微信）。本节介绍如何高效、安全地导流。

![](img/d1f04b6774f8f29c5672b390debc634e_29.png)

我们使用经过官方审核的跳转链接或二维码进行导流，安全无风险。
1.  将导流话术（如“长按识别免费领取”）与跳转码一起发送给用户。
2.  用户识别二维码后，可直接跳转至微信。

对于更高阶的自动回复，需使用企业号或蓝V账号：
1.  在社群提供的“外联系统”后台绑定你的抖音蓝V账号。
2.  绑定后，当用户访问你的抖音主页或发送消息时，系统可自动回复预设的导流外链，实现全自动导流。
