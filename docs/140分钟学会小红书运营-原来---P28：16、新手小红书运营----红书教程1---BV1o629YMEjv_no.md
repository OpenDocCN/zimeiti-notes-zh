![](img/0eb06d6924b98fb1679b1888956cc4f3_11.png)

上一节我们介绍了如何选择要复制的产品、引流及分享内容。本节中，我们来看看如何对这些选定的内容进行具体修改。

## 内容修改的三种方式

以下是三种核心的内容修改方式，它们能有效降低笔记的重复率。

### 1. 完全复制
完全复制是指对从其他平台（如1688、抖音）或小红书直接获取的完整笔记（包括视频、图片、文案）进行**三次小修改**。目的是让系统判定该内容并非完全重复，从而获得基础权重。

*   **视频修改**：使用剪辑软件（如剪映）将完整视频**分割成三段**，然后重新组合发布。此操作可排除90%-95%的重复数据量。
    *   `操作公式：完整视频 -> 分割为三段 -> 重新组合发布`
*   **图片修改**：使用图片编辑软件，在**不显眼处添加微小元素**（如小黑点、小白点），或直接截图后使用。同样进行2-3次操作。
*   **文字修改**：更改文字的**字体、大小、简繁体**，或调整排版（如横排改斜排）。文字修改的效果通常较好。

> **注意**：完全复制操作简单，适合新手或用于引流，但因同类内容较多，权重普遍不高。直接复制粘贴不修改，极易被系统屏蔽，失去基础曝光。

![](img/0eb06d6924b98fb1679b1888956cc4f3_15.png)

### 2. 主体复制
主体复制指**复制产品的整体框架（如图片/视频），但文案部分由自己重新编写**。需要对产品的宣传点、标题、折扣信息等进行修改，保留对自己有用的部分，替换掉无用的部分。

*   `操作逻辑：复制他人内容框架 + 替换/重写文案 = 新笔记`
*   此方法更适用于**图片和文字**类笔记。视频因其连贯性，较难进行主体复制。
*   如果想蹭原笔记流量，可使用“完全复制”；如果只想借鉴其形式，则使用“主体复制”。

### 3. 格式复制
格式复制指**仅复制笔记的文案格式和推广结构，但更换其中的产品主图和核心关键词**。适用于原笔记文案很精彩，但图片质量一般或与自家产品不符的情况。

*   `操作逻辑：复制他人文案格式 -> 替换产品主图 -> 替换核心产品词 = 新笔记`

![](img/0eb06d6924b98fb1679b1888956cc4f3_17.png)

无论采用哪种复制方式，**都必须进行修改**。不修改的笔记几乎100%会与海量现有内容重复。通过修改，我们可以规避掉平台上大量由“店群”模式产生的重复、低质内容，为自己争取到有效的竞争空间。

> **背景知识：“店群”模式**：指运营者开设大量店铺，批量上架成千上万款商品，并从全网采集商品图片和资料进行发布，赚取差价。这导致了平台内存在大量高度重复的内容。我们的修改操作，正是为了与这部分低质内容区分开来。

本节课中，我们一起学习了小红书笔记的三种内容修改方式：**完全复制**、**主体复制**和**格式复制**。核心在于理解任何借鉴来的内容都必须经过有效修改，以降低系统判定的重复率，从而获得更好的曝光基础。
