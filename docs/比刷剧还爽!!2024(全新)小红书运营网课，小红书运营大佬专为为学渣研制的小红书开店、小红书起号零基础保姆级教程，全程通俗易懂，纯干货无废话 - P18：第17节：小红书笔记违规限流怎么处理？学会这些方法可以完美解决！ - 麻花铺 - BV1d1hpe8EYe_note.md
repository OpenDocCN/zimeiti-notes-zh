## 核心原则回顾

上一节我们介绍了小红书平台的基本逻辑，本节中我们来看看具体的违规场景。首先，我们必须再次强化一个核心概念：小红书的主旋律是**种草**和**分享**。平台不鼓励直接的硬性广告营销，除非是官方合作的大品牌。因此，我们所有笔记的创作思想都应围绕这两个核心展开。

## 常见的笔记违规类型

以下是小红书官方通常会判定为违规的几种情况，理解它们有助于我们提前规避风险。

1.  **广告与销售笔记**
    笔记内容过于明显地推销产品，让用户一眼就能看出是广告。

2.  **不合规地留下联系方式**
    在笔记或个性签名中直接留下微信、电话等联系方式。目前相对安全的方式是留下邮箱。另一种历史玩法是使用“大小号联动”：大号发布笔记时@一个专门留联系方式的小号，但此方法风险较高，小号易被封禁。

3.  **非原创内容**
    小红书判断原创的初级标准是：内容是否首次在小红书平台发布。直接从平台内搬运或拼凑的内容，会被判定为非原创。公式可以理解为：
    `原创判定 = 内容首次发布平台 == 小红书`

4.  **伪造数据**
    通过非正常手段刷高粉丝数、阅读量、点赞、收藏等数据。平台有算法监控数据异常，例如点击率或关注转化率远高于行业正常水平（如3%）时，即会被判定为作弊。示例逻辑：
    ```python
    # 伪代码示例：异常数据检测逻辑
    if (笔记点击率 > 行业平均点击率 * 异常阈值):
        标记为“数据异常”
    ```

5.  **不友善行为**
    在笔记或评论中无确凿证据地诽谤、恶意抨击他人或同行。所有批评性结论都应有相关证据（如新闻报道、检测报告）支持。

6.  **诱导分享**
    以利益（如礼品、红包）诱导用户分享笔记，此类营销行为目前不被平台提倡。

7.  **侵权、盗版、虚假或无价值内容**
    特别是“无价值内容”，指那些信息量稀疏、逻辑混乱或纯粹自嗨的长篇大论。内容价值不在于字数多少，而在于是否对用户有实质帮助。

8.  **色情低俗及封建迷信内容**
    发布涉黄或宣扬没有科学依据的封建迷信活动（如摆放某物即可改运）的内容，平台会严厉打击。

## 违规问题的解决方案

了解了违规类型后，我们来看看对应的预防和解决方法。以下是几个关键的操作准则。

1.  **发布多产品对比**
    不要只宣传单一产品。可以通过对比多个同类产品的特性，或引用权威信息进行优劣分析，这样更符合“分享”和“种草”的基调，减少硬广嫌疑。

2.  **杜绝数据造假**
    坚决不要刷粉丝、刷阅读、刷点赞收藏。不真实的数据极易被系统识别，导致账号被降权。

3.  **避免恶意贬低**
    不要在没有确凿证据的情况下贬低其他产品或品牌。客观的对比可以，但恶意的攻击会引来举报，损害账号信誉。

4.  **远离诱导与低质内容**
    不进行诱导分享，不发布色情、低俗或毫无逻辑与价值的“水文”。

5.  **坚持发布原创内容**
    这是最根本的解决方法。原创不仅是避免违规的保障，也是获得流量的基础。对于“不知如何原创”的问题，其核心在于方法而非文笔。即使你不是专业文案，只要掌握了特定技巧，也能写出100%原创的优质笔记。我们将在下一节课（第15节）对此进行详细展开。
