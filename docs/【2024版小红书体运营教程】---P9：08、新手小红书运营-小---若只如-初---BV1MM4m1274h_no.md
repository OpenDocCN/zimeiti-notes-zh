# 小红书运营教程：P9：08、新手小红书运营-小红书规则（1）📚

在本节课中，我们将要学习小红书平台的核心规则。了解这些规则是成功运营账号、避免违规和获取流量的基础。前面我们已经介绍了小红书的本质、市场分析和起号方法，本节将正式进入平台规则的学习。

## 什么是小红书的规则

![](img/9ebf08ddb44ef96a9d6fc14396bb004d_3.png)

![](img/9ebf08ddb44ef96a9d6fc14396bb004d_4.png)

小红书的规则是平台为了维护社区秩序和用户体验而制定的一系列准则。它决定了你的内容能否被推荐、账号是否安全，以及流量获取的效率。了解并遵守这些规则至关重要。

## 小红书规则的四大要点

![](img/9ebf08ddb44ef96a9d6fc14396bb004d_9.png)

![](img/9ebf08ddb44ef96a9d6fc14396bb004d_11.png)

我将小红书的规则总结为四个核心要点。虽然只有四点，但每个要点都包含丰富且重要的内容，需要我们仔细理解和掌握。

![](img/9ebf08ddb44ef96a9d6fc14396bb004d_13.png)

![](img/9ebf08ddb44ef96a9d6fc14396bb004d_15.png)

以下是四个核心要点：
1.  **推荐规则**
2.  **用户行为规范**
3.  **内容发布准则**
4.  **账号安全与登录限制**

上一节我们介绍了规则的框架，本节中我们重点来看第一个要点：**推荐规则**。

## 推荐规则详解

推荐规则是小红书系统决定是否将你的内容展示给更多用户的核心机制。系统会分析你的账号信息、发布内容，从而对你的账号进行定位，并据此分配初始的流量曝光和权重。

推荐规则主要围绕以下三个方面展开：

以下是推荐规则下的三个关键子项：
*   **用户行为**：指账号的日常操作，哪些行为被允许，哪些被禁止。
*   **自序商品**：指在内容中涉及商品推荐或销售时的规范。
*   **图文阴影**：指发布笔记（图文/视频）时需要遵守的具体规定。

### 1. 用户行为规范

用户行为规范明确了在账号运营中，哪些行为是平台禁止的，哪些是在一定限制下允许的。遵守这些规范是账号安全的基础。

![](img/9ebf08ddb44ef96a9d6fc14396bb004d_17.png)

以下是用户行为中**严格禁止**的核心内容（最敏感的五点）：
*   **微信号**
*   **二维码**
*   **代购信息**
*   **水印**
*   **某某店铺**（直接宣传其他店铺名称）

**核心逻辑**：平台禁止任何直接的营销和导流行为。例如，你不能在笔记中直接宣传自己的主店，试图将用户引导至微信或其他平台。

**规避技巧示例**：如果你想引导用户，可以采用间接方式。例如，用主账号在评论区以官方口吻@你的小号，或者使用谐音、话术来暗示联系方式。但需注意，私聊引导也要控制频率，每天最多向1-2人发送，否则仍有风险。

以下是用户行为中**在限制下允许**的内容：
*   个人资料中允许存在**邮箱**。
*   可以使用**微博谐音**或特定**话术**进行间接引导。

### 2. 自序商品规范

自序商品规范主要针对内容中涉及商品推荐或销售的行为。小红书对此类内容的管控相对宽松，但仍有明确红线。

以下是自序商品中**必须避免**的两点：
1.  **不能轻易提供医疗建议**（例如，“这个病去某某医院看最好”）。
2.  **不能轻易提供投资建议**（例如，“这只股票必涨，赶紧买”）。

![](img/9ebf08ddb44ef96a9d6fc14396bb004d_26.png)

**核心逻辑**：平台严禁任何可能涉及诈骗、误导，或造成用户财产损失的内容。其他类型的商品推荐则相对自由。

### 3. 图文阴影（笔记发布）规范

图文阴影规范指的是在发布笔记时必须遵守的规则，关乎内容能否获得推荐。

![](img/9ebf08ddb44ef96a9d6fc14396bb004d_28.png)

以下是图文阴影中**不能做**的事情：
*   **不得诱导用户**：例如，“关注我就抽奖”、“私信我领资料”等直接的利益诱导行为。
*   **内容原创度低**：直接搬运他人的图片和文案，账号权重会很低，甚至作废。

以下是图文阴影中**应该做**的事情：
*   **保证内容原创度**：原创度越高，获得的基础流量扶持就越多。
*   **合理规避规则**：例如，通过店铺活动标题等间接方式进行推广（但此方式搜索权重较低）。

**关于账号登录的重要补充**：
*   **公式/规则**：`单台电脑最多同时登录 ≤ 3 个小红书账号`
*   建议使用“雷电模拟器”等多开工具进行分流，并注意修改IP等数据。
*   如果需要登录第4个账号，请先退出其中一个。超过3个账号同时在线容易被系统检测，增加风险。
