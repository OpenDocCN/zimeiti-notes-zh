# 抖音运营速成课：P7：人物设定策划

![](img/d892d0a24ca73580cac80981aed981f7_0.png)

在本节课中，我们将要学习如何在抖音上策划一个成功的人物设定，即“人设”。这是打造高辨识度、高信任度账号的核心步骤。

![](img/d892d0a24ca73580cac80981aed981f7_2.png)

上一节我们介绍了账号定位的重要性，本节中我们来看看如何将定位具体化为一个生动的人物角色。

![](img/d892d0a24ca73580cac80981aed981f7_4.png)

在抖音中建立人设，如同在游戏中创建角色，需要预先设定其基本属性。但与虚拟游戏角色不同，抖音人设是面向真实人生的角色，因此必须更具真实感。

![](img/d892d0a24ca73580cac80981aed981f7_6.png)

无论是短视频还是直播，一个成功的人设IP能够有效解决粉丝的信任问题，促进内容的裂变传播，并具备强大的商业变现潜力。因此，我们运营账号、创作内容和策划选题的核心目标，不仅在于获取流量，更在于打造IP。打造IP的本质，是在特定场景或情境中，完成账号的**人格化**过程。

![](img/d892d0a24ca73580cac80981aed981f7_8.png)

这个过程需要注入情感、塑造故事并贴上鲜明的标签。账号人格化的结果，就是我们通常所说的“人设”。

![](img/d892d0a24ca73580cac80981aed981f7_10.png)

对于人设的具体策划，我们可以从以下几个维度着手。以下是构建人设的核心要素列表：

![](img/d892d0a24ca73580cac80981aed981f7_12.png)

**1. 名称**
一个IP必须有一个易于传播的称呼，这是其名片。它应像品牌名一样明确且易记。

![](img/d892d0a24ca73580cac80981aed981f7_14.png)

**2. 职业/身份**
人格化角色必须具备社会身份，这是用户认识并联想该IP的入口。例如：老师、律师、创业者。

**3. 社会称谓**
一个恰当的称呼能快速拉近与用户的距离，产生情感共鸣。例如：“XX哥”、“XX姐”、“XX叔”。

![](img/d892d0a24ca73580cac80981aed981f7_16.png)

**4. 形象印象**
从用户视角塑造IP的外在形象，使其更容易被记住和识别。例如：大叔形象、甜美可爱、时尚达人。

![](img/d892d0a24ca73580cac80981aed981f7_18.png)

**5. 性格与特点**
赋予IP鲜明的性格、爱好或优缺点，能增强其真实感和记忆点。过于完美的形象反而显得不真实。例如：暴躁但热心、吃货属性。

**6. 外在象征**
利用社会普遍认知（刻板印象）来快速定位IP。例如：特定的穿着代表某种职业，开豪车象征成功。

策划好以上内容后，便可结合自身的产品或服务优势，将其落实到抖音账号的“五件套”中：**头像、昵称、背景图、简介、视频封面**。

除了静态的人格化策划，还需通过持续的**内容创作**来强化人设，提炼出让人过目不忘的“超级符号”。

超级符号可以从以下几个层面进行提炼。以下是提炼超级符号的关键方向列表：

**1. 视觉画风**
固定的场景、标志性道具、特色服饰等。

**2. 听觉标识**
独特的背景音乐（BGM）、特效音、配音声线或口头禅。

**3. 文案标签**
能够反映核心优势且易于记忆的提炼词。例如：“简单粗暴教XX”。

**4. 情绪表达**
标志性的表情或情绪，如夸张的笑、无奈的耸肩。

**5. 语言特色**
方言、口音、独特的说话节奏（如慢条斯理、抑扬顿挫）。

**6. 行为习惯**
标志性的肢体动作，如撩头发、扶眼镜、特定的手势。

提炼出超级符号后，在后续的短视频创作中，应有意识地在多个视频中重复呈现这些元素，不断强化用户记忆。

本节课中我们一起学习了抖音人设的完整策划流程。从设定基础身份信息到通过内容提炼超级符号，每一步都是为了将账号塑造成一个真实、立体、有记忆点的“人”，从而在激烈的竞争中脱颖而出，赢得粉丝的信任与喜爱。
