## 视频制作全流程拆解
了解了市场前景后，我们进入核心实操部分。本节将详细讲解如何使用AI辅助制作英文磨耳朵视频。

![](img/af84570b45b677052d94c7c807b68751_17.png)

### 第一步：准备素材与提取歌词
以下是使用剪映软件提取视频歌词的步骤。

![](img/af84570b45b677052d94c7c807b68751_19.png)

1.  打开剪映软件，将准备好的英文歌曲或视频素材拖入媒体库。
2.  将素材拖到时间轴上。
3.  在时间轴上右键点击视频片段，选择“音频分离”。这样，音频就被单独分离出来了。
4.  点击顶部菜单栏的“文本”选项，然后选择“识别歌词”。
5.  点击“开始识别”，软件会自动生成字幕。

![](img/af84570b45b677052d94c7c807b68751_21.png)

**注意**：新版剪映的歌词识别功能可能需要会员。如果需要无需会员即可识别的老版本，请在关注后，于评论区发送“已关666”，我会私信发送给你。

![](img/af84570b45b677052d94c7c807b68751_23.png)

![](img/af84570b45b677052d94c7c807b68751_24.png)

![](img/af84570b45b677052d94c7c807b68751_25.png)

6.  识别完成后，点击右上角的“导出”按钮。
7.  在导出设置中，务必勾选“字幕导出”选项，并选择“TXT”格式。这样就能将歌词文本文件保存到本地。

![](img/af84570b45b677052d94c7c807b68751_27.png)

### 第二步：使用AI生成视频画面
歌词提取完成后，我们需要为歌曲配上相应的画面。本节我们使用字节跳动的“豆包”AI工具来生成图片。

![](img/af84570b45b677052d94c7c807b68751_29.png)

![](img/af84570b45b677052d94c7c807b68751_31.png)

原视频画面通常比较简单。我们可以根据歌词含义，手写提示词让AI生成画面，如果歌词本身具有画面感且简单，也可以直接使用。

![](img/af84570b45b677052d94c7c807b68751_32.png)

操作步骤如下：

1.  打开豆包AI（或类似AI绘画工具），找到“图像生成”功能。
2.  设置图片比例为 **9:16**，这是适合手机竖屏观看的比例。
3.  选择“动漫”风格，这种风格通常更受孩子喜欢。
4.  在输入框中描述你想要的画面。例如，输入：“两个小孩坐在金色的稻田中”。
5.  点击发送，AI会生成一组图片。你可以选择最满意的一张，点击“下载原图”保存。
6.  重复此过程，为歌曲的不同段落生成一系列图片。可以复制修改提示词以提高效率，例如将场景改为“傍晚，两个小孩坐在金色的稻田中”。

### 第三步：剪辑与合成视频
所有图片生成完毕后，我们回到剪映，将音频、图片和字幕合成一个完整的视频。

1.  将生成的所有图片导入剪映的媒体库。
2.  全选所有图片，将它们拖到时间轴上。此时，图片会覆盖在原有视频轨上方，你可以删除原视频只保留图片和音频。
3.  根据音乐长度，对图片序列和音频进行修剪，使其长度匹配。
4.  播放音乐，调整图片出现的顺序和时长，尽量让图片切换卡在音乐的节奏点上，这样观感更自然。
5.  如果某些图片尺寸不符，可以选中该图片，在预览窗口中调整缩放和位置，使其覆盖整个画面。
6.  删除之前导出的旧字幕轨道。点击“文本”->“识别歌词”，为修剪后的音频重新识别字幕。
7.  调整字幕样式：更换字体（如“得意黑”）、添加底色、调整位置和大小，使其清晰美观。

### 第四步：添加动画与特效
静态图片切换可能略显单调，我们可以为图片添加简单的关键帧动画，让视频更生动。

![](img/af84570b45b677052d94c7c807b68751_34.png)

![](img/af84570b45b677052d94c7c807b68751_36.png)

1.  **推进动画**：选中一张图片，在视频开始和结束的位置，分别为“缩放”参数添加关键帧（点击参数前的菱形图标）。将结束帧的缩放比例调大，即可模拟镜头推进的效果。
    *代码/操作逻辑示意*：`起始关键帧（缩放100%） -> 结束关键帧（缩放120%）`
2.  **移动动画**：选中另一张图片，在开始和结束位置，为“位置”的Y轴参数添加关键帧。调整Y轴数值，可以模拟镜头上下摇动的效果。
    *代码/操作逻辑示意*：`起始关键帧（Y轴 0） -> 结束关键帧（Y轴 -50）`
3.  **添加转场**：在图片之间的衔接处，点击“转场”特效，推荐使用“叠化”类别的转场（如基础叠化），效果会更柔和自然。
4.  **最终导出**：所有调整完成后，点击右上角“导出”。确保只勾选“视频导出”，设置好分辨率和帧率，点击“导出”即可生成最终视频文件。
