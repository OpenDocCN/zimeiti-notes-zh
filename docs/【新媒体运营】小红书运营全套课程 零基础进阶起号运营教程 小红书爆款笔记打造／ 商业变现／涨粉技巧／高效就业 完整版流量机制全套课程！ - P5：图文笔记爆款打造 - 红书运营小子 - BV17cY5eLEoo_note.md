## 什么是商品笔记？

![](img/fb572b2d054e098bc32861a12b72a80d_5.png)

商品笔记是一种强有力的带货形式。商家在图文笔记中添加商品链接，用户可以直接在小红书平台完成从“种草”（产生兴趣）到“拔草”（完成购买）的全过程。

![](img/fb572b2d054e098bc32861a12b72a80d_7.png)

## 商品笔记的分发逻辑

![](img/fb572b2d054e098bc32861a12b72a80d_9.png)

商品笔记的分发逻辑以**公域流量**为主，最终目标是转化为**私域流量**。其完整路径可分为以下三个部分：

![](img/fb572b2d054e098bc32861a12b72a80d_11.png)

以下是分发逻辑的三个关键环节：

![](img/fb572b2d054e098bc32861a12b72a80d_13.png)

1.  **推荐浏览**：用户通过搜索或在主页推荐流中看到笔记。
2.  **笔记转化**：用户对商品感兴趣，点击进入笔记页面，并通过笔记中的商品链接直接购买商品。
3.  **主页转化**：用户在阅读商品笔记后，进入商家主页并关注商家，成为其粉丝。

![](img/fb572b2d054e098bc32861a12b72a80d_15.png)

上一节我们介绍了商品笔记的基本概念和分发路径，本节中我们来看看打造爆款笔记需要关注哪些核心指标。

![](img/fb572b2d054e098bc32861a12b72a80d_17.png)

## 爆款图文笔记的核心指标

![](img/fb572b2d054e098bc32861a12b72a80d_19.png)

打造爆款图文笔记需要掌握两个核心指标：**互动价值**与**交易价值**。

![](img/fb572b2d054e098bc32861a12b72a80d_21.png)

### 1. 互动价值

![](img/fb572b2d054e098bc32861a12b72a80d_23.png)

互动价值衡量笔记吸引用户参与互动的能力，主要包括笔记的点击率、点赞、收藏、分享、评论等数据。其中，**点击率**是一个关键公式：

![](img/fb572b2d054e098bc32861a12b72a80d_25.png)

**点击率 = 点击次数 / 展现次数**

![](img/fb572b2d054e098bc32861a12b72a80d_27.png)

笔记的展现主要来自发现页、关注页、附近页和搜索页，其中**发现页是主要的流量来源**。为了让图文笔记更容易在发现页被推荐，需要掌握以下三个要素：

![](img/fb572b2d054e098bc32861a12b72a80d_29.png)

以下是提升发现页推荐的三个要素：

*   **标题**：关键词要醒目，并加入长尾关键词。恰当地运用关键词能让用户快速了解笔记内容，激发好奇心。同时，将标题数字化或借助热点话题，能有效吸引点击。
*   **封面**：封面设计需紧密围绕笔记内容。可以使用对比图、步骤图、效果图等形式，直观捕捉用户需求。封面需要突出产品优点和用户下单的理由。
*   **内容**：笔记内容需要具备吸引力，促使用户从“种草”转向“购买”，最终提升产品销量。

### 2. 交易价值

![](img/fb572b2d054e098bc32861a12b72a80d_31.png)

![](img/fb572b2d054e098bc32861a12b72a80d_33.png)

交易价值直接衡量笔记的带货能力，主要包括**商品点击率、加购和购买行为**。商品点击率能直接体现笔记内容的质量。

为了提升交易价值，笔记内容需要突出商品的功能、卖点与价值，帮助消费者做出决策，缩短其购物考虑链路。同时，在**评论区**内积极回复用户问题，并置顶购买路径及当前可享受的购买福利，能有效刺激用户下单。

![](img/fb572b2d054e098bc32861a12b72a80d_35.png)

想要持续做好商品笔记，以上技巧需要综合运用。单一方法可能带来偶然的爆点，但整体能力的提升才能让商品笔记的流量和转化更加稳定。

![](img/fb572b2d054e098bc32861a12b72a80d_37.png)
