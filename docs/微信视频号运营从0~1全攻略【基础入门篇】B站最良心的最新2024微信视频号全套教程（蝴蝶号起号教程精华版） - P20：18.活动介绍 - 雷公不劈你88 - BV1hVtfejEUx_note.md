# 微信视频号运营从0到1全攻略：P20：18.活动介绍 🎬

在本节课中，我们将要学习一个由官方发起的“商家联盟”活动。这个活动旨在为视频号带货提供强力支持，我们将详细介绍活动的背景、核心福利以及参与步骤。

## 活动背景与负责人

![](img/61eae59ea1ff1b30c9882400f2ef4e0a_1.png)

为促进2024年中国电商行业的新发展，为视频化带货电商奠定技术基础，特推出“商家联盟”活动。本次活动由短视频教研部负责人张良院长全程对接。

![](img/61eae59ea1ff1b30c9882400f2ef4e0a_2.png)

本次活动以公平公正为核心理念展开。张老师作为本次活动的全国总负责人，拥有独家对接权限。

![](img/61eae59ea1ff1b30c9882400f2ef4e0a_4.png)

## 活动核心福利介绍

![](img/61eae59ea1ff1b30c9882400f2ef4e0a_6.png)

以下是本次“商家联盟”活动提供的几项核心扶持政策，力度空前。

1.  **官方视频素材支持**：针对商家缺乏优质拍摄素材的痛点，官方已准备好大量高质量视频素材。每个参与活动的账号可获得**30条**现成视频素材，满足每日发布需求。
    ![](img/61eae59ea1ff1b30c9882400f2ef4e0a_1.png)
    ![](img/61eae59ea1ff1b30c9882400f2ef4e0a_2.png)

![](img/61eae59ea1ff1b30c9882400f2ef4e0a_8.png)

2.  **官方流量扶持**：每个参与账号可获得**90万**的流量扶持，解决视频发布后没有曝光的问题。

![](img/61eae59ea1ff1b30c9882400f2ef4e0a_10.png)

3.  **素材库丰富易用**：官方素材库产品种类齐全，例如中秋节礼品、餐巾纸、牛奶、火锅底料等，并且支持分类浏览和关键词搜索功能，方便商家快速找到所需素材。

## 活动奖励与结算方式

![](img/61eae59ea1ff1b30c9882400f2ef4e0a_13.png)

除了素材和流量，本次活动还提供了直接的现金奖励，以激励商家积极发布内容。

*   **奖励标准**：每发布一条带货视频，即可获得**300元**奖励。
*   **结算周期**：奖励实行“日结”制度，今天发布视频，明天奖励到账。
*   **总收益计算**：如果发满30条素材视频，总奖励金额为 `30 * 300 = 9000元`。
    ![](img/61eae59ea1ff1b30c9882400f2ef4e0a_15.png)

## 活动期限与参与条件

![](img/61eae59ea1ff1b30c9882400f2ef4e0a_15.png)

上一节我们了解了活动的丰厚福利，本节中我们来看看活动的时效性与参与要求。

本次活动由微信视频号官方发起，张良院长为唯一指定对接人。活动有明确的起止日期，并非长期开放。

*   **活动期限**：活动于2024年1月13日正式开始，并计划于2024年8月10日结束。
    ![](img/61eae59ea1ff1b30c9882400f2ef4e0a_17.png)
    ![](img/61eae59ea1ff1b30c9882400f2ef4e0a_18.png)

*   **参与条件**：参与活动并获取奖励，必须完成一个核心动作：将指定的带货视频发布到视频号，并正确挂上商品链接。这需要参与者进行简单的实操学习。

![](img/61eae59ea1ff1b30c9882400f2ef4e0a_17.png)

![](img/61eae59ea1ff1b30c9882400f2ef4e0a_18.png)
