# 小红书运营教程：P2：小红书店铺运营与变现方式

![](img/78161ad412db09869604bc0856b904ef_0.png)

![](img/78161ad412db09869604bc0856b904ef_2.png)

在本节课中，我们将要学习在小红书平台进行店铺运营与流量变现的几种核心方式。我们将详细介绍如何通过官方合作、自营小店以及企业号营销来实现商业目标。

![](img/78161ad412db09869604bc0856b904ef_4.png)

![](img/78161ad412db09869604bc0856b904ef_6.png)

上一节我们介绍了小红书的市场现状，本节中我们来看看具体的店铺运营与变现路径。

![](img/78161ad412db09869604bc0856b904ef_8.png)

## 官方平台合作

![](img/78161ad412db09869604bc0856b904ef_10.png)

![](img/78161ad412db09869604bc0856b904ef_12.png)

第一种方式是借助小红书的官方平台进行合作。这需要你登录小红书的电脑端。

![](img/78161ad412db09869604bc0856b904ef_14.png)

在登录小红书电脑端之后，你可以在界面中找到“创作者服务”板块。

![](img/78161ad412db09869604bc0856b904ef_16.png)

在该服务板块中，你能看到“小红书合作”的相关功能。如果你是一个内容创作者账号，希望通过为他人带货来实现引流和流量变现，就可以通过这个官方渠道来实现。因为小红书的官方平台构建了一个连接多方的生态系统。

![](img/78161ad412db09869604bc0856b904ef_18.png)

![](img/78161ad412db09869604bc0856b904ef_20.png)

这个平台内既包含了大量的内容创作者，也聚集了许多专业的账号运营机构。

![](img/78161ad412db09869604bc0856b904ef_22.png)

例如，一些MCN机构就活跃于此。同时，平台上也少不了大量寻求推广的商家。

![](img/78161ad412db09869604bc0856b904ef_24.png)

![](img/78161ad412db09869604bc0856b904ef_26.png)

也就是说，当你加入小红书的这个合作平台后，就相当于进入了一个包含专业MCN机构、需要带货的商家以及像你这样的创作者的生态圈。

![](img/78161ad412db09869604bc0856b904ef_28.png)

那么你来到这个平台之后，商家可能会主动与你寻求合作。

![](img/78161ad412db09869604bc0856b904ef_30.png)

MCN机构也可能会与你合作，因为它们的主要业务就是进行账号孵化和运营。

![](img/78161ad412db09869604bc0856b904ef_32.png)

如果你的账号表现出色，你很有机会进入它们的孵化阵营。

![](img/78161ad412db09869604bc0856b904ef_34.png)

![](img/78161ad412db09869604bc0856b904ef_36.png)

这意味着你的账号将来有很大概率能成长为一个头部账号。大家可以登录小红书电脑端去具体查看这个功能。

![](img/78161ad412db09869604bc0856b904ef_38.png)

## 自营小红书店铺

![](img/78161ad412db09869604bc0856b904ef_40.png)

第二种方式叫做自开小店。你可以通过小红书平台自己开设一个小店。

![](img/78161ad412db09869604bc0856b904ef_42.png)

![](img/78161ad412db09869604bc0856b904ef_44.png)

无论你是个人、商家还是企业，无论你是否拥有营业执照，都可以申请开通。

![](img/78161ad412db09869604bc0856b904ef_46.png)

具体操作是，在你的手机小红书App中，点击右上角的“三道杠”菜单。

![](img/78161ad412db09869604bc0856b904ef_48.png)

![](img/78161ad412db09869604bc0856b904ef_50.png)

点进去之后，找到“创作者服务中心”，进入该中心。

![](img/78161ad412db09869604bc0856b904ef_52.png)

你会发现“开通店铺”或“成为专业号”的选项。在成为专业号的过程中，就包含开通小店这一步骤。

![](img/78161ad412db09869604bc0856b904ef_54.png)

个人可以开小店，商家和企业也可以开小店。

![](img/78161ad412db09869604bc0856b904ef_56.png)

![](img/78161ad412db09869604bc0856b904ef_58.png)

大家可以打开手机上的小红书App，按照上述路径操作一下。

![](img/78161ad412db09869604bc0856b904ef_60.png)

那么在你成功开设小店之后，你就可以自己上架商品。

![](img/78161ad412db09869604bc0856b904ef_62.png)

![](img/78161ad412db09869604bc0856b904ef_64.png)

企业可以上架自己所生产或经营的商品。

![](img/78161ad412db09869604bc0856b904ef_66.png)

这样就可以实现带货变现。

![](img/78161ad412db09869604bc0856b904ef_68.png)

## 企业号营销合作

![](img/78161ad412db09869604bc0856b904ef_70.png)

![](img/78161ad412db09869604bc0856b904ef_72.png)

最后一种方式叫做企业号营销。企业号营销的核心在于，如果你本身就是一个企业或商家。

![](img/78161ad412db09869604bc0856b904ef_74.png)

你可以选择直接与小红书商城进行合作。你可以将你的货物供应给小红书。

![](img/78161ad412db09869604bc0856b904ef_76.png)

从而成为小红书的供货商或自营商品的一部分。

![](img/78161ad412db09869604bc0856b904ef_78.png)

![](img/78161ad412db09869604bc0856b904ef_80.png)

你只需要负责供货，然后通过你自己的小红书企业号，为这些商品进行内容引流。这种方式就叫做企业号的营销。

![](img/78161ad412db09869604bc0856b904ef_82.png)

企业号为自己店铺的商品，或者为通过小红书渠道经营的商品进行引流。

![](img/78161ad412db09869604bc0856b904ef_84.png)

![](img/78161ad412db09869604bc0856b904ef_86.png)

以上这些，就是当你自己拥有小红书店铺，或者计划与专业方合作时，进行流量变现的几种主要方式。

![](img/78161ad412db09869604bc0856b904ef_88.png)
