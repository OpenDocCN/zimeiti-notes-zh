# 微信视频号运营教程：P13：11.如何填写邀请函

![](img/90c604a79f47b97c5b70eaedc8b2281a_1.png)

在本节课中，我们将学习如何正确填写视频号创作者计划的邀请函。这是加入计划、获取官方支持的关键一步，流程简单，但需要确保信息准确无误。

上一节我们介绍了如何注册视频号，本节中我们来看看如何填写与之关联的邀请函。

![](img/90c604a79f47b97c5b70eaedc8b2281a_3.png)

![](img/90c604a79f47b97c5b70eaedc8b2281a_4.png)

## 第一步：找到你的视频号信息

![](img/90c604a79f47b97c5b70eaedc8b2281a_6.png)

首先，你需要找到自己视频号的昵称和粉丝数量。以下是具体操作步骤。

![](img/90c604a79f47b97c5b70eaedc8b2281a_8.png)

以下是详细的操作步骤列表：

![](img/90c604a79f47b97c5b70eaedc8b2281a_9.png)

1.  **打开微信**：在手机上启动微信应用。
2.  **进入“发现”页**：点击微信底部导航栏的“发现”选项。
3.  **找到“视频号”入口**：在“发现”页面中，向上滑动，找到并点击“视频号”入口（图标为黄色蝴蝶形状）。
4.  **访问个人主页**：进入视频号后，点击页面右上角的个人头像图标，进入个人主页。
5.  **查看账号信息**：在个人主页中下滑，找到“我的视频号”区域。这里会清晰显示你的**账号昵称**和**粉丝数量**。

例如，你的信息可能显示为：`天道酬勤，39917个粉丝`。那么，账号昵称为 **“天道酬勤”**，粉丝数为 **“39917”**。

## 第二步：填写邀请函

获取到账号信息后，即可开始填写邀请函。邀请函通常包含以下几个核心字段。

![](img/90c604a79f47b97c5b70eaedc8b2281a_11.png)

![](img/90c604a79f47b97c5b70eaedc8b2281a_12.png)

以下是需要填写的关键信息列表：

![](img/90c604a79f47b97c5b70eaedc8b2281a_13.png)

1.  **视频号昵称**：填写你在上一步中查看到的账号昵称，例如 **“天道酬勤”**。
2.  **粉丝数量**：填写你当前的粉丝数，例如 **“39917”**。
3.  **活动邀请人**：根据你的加入渠道，填写指定的邀请人姓名，例如 **“张良老师”**。
4.  **实名信息**：此处必须填写你本人的真实姓名，例如 **“周建国”**。
5.  **结算手机号**：填写用于接收验证码和后续结算的手机号码，格式为连续数字，例如 **“13812345678”**。

## 第三步：提交与确认

![](img/90c604a79f47b97c5b70eaedc8b2281a_15.png)

![](img/90c604a79f47b97c5b70eaedc8b2281a_16.png)

所有信息填写完毕后，请务必仔细检查一遍，确保无误。

最后，点击页面上的 **“提交”** 按钮。系统可能会弹出确认提示，点击 **“确认”** 即可完成提交。成功提交后，页面会显示“已成功加入创作者计划”之类的提示。

![](img/90c604a79f47b97c5b70eaedc8b2281a_18.png)

## 关于素材的补充说明

![](img/90c604a79f47b97c5b70eaedc8b2281a_20.png)

![](img/90c604a79f47b97c5b70eaedc8b2281a_22.png)

成功加入计划后，即可开始创作。对于不熟悉操作或寻找素材的用户，导师可能会提供帮助。例如，导师可能从已购版权的素材库中（如包含《还珠格格》、《我不是药神》等正版影视片段的素材库）挑选内容，整理好顺序后发给学员，学员直接发布即可。这能确保内容合规，提高起步效率。

![](img/90c604a79f47b97c5b70eaedc8b2281a_24.png)

本节课中我们一起学习了填写视频号创作者计划邀请函的完整流程，从查找账号信息到填写各项内容并最终提交。记住，准确填写信息是成功加入计划的基础。
