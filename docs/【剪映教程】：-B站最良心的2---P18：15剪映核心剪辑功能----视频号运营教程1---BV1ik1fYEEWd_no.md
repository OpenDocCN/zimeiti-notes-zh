# 剪映核心剪辑功能：P18：文本功能详解 📝

在本节课中，我们将要学习剪映的核心功能之一——文本功能。一个完整的视频除了画面和音频，文本或字幕也是传递信息、突出主题的关键元素。通过添加文本，可以更准确、更快速地让观众理解视频内容。

上一节我们介绍了音频功能，本节中我们来看看如何为视频添加和设置文本。

![](img/f030e424ae187e59f7561c6de5a940e6_1.png)

## 文本功能概述

在剪映中，我们可以自由地为视频添加字幕。添加的字幕可以单独设置样式，包括字体、字号、颜色、间距以及动画效果。

此外，剪映还提供了两个高效的功能：**字幕自动识别**和**歌词识别**。如果你已经为视频配好了旁白或添加了带歌词的音乐，可以使用这两个功能一键快速生成字幕，极大地降低了操作难度和时间成本。

最后，文本功能模块中还包含**贴纸**的添加选项。

关于文本功能的基本介绍就到这里，接下来我们进入功能实操环节。

## 功能实操演练

文本功能位于一级菜单栏。点击底部菜单栏的 **`文字`** 按钮，即可看到四个主要选项：`新建文本`、`识别字幕`、`识别歌词`和`贴纸`。

我们将以一条已添加音频的视频为例，详细演示每个功能。

### 1. 新建文本（手动添加）

手动添加文本是最基础的方式。

以下是操作步骤：
1.  点击底部菜单栏的 **`文字`**。
2.  选择 **`新建文本`**。
3.  在输入框中输入文字内容，例如“剪映视频剪辑教程”。
4.  输入完成后，时间轴上会新增一个文本轨道，画面上会显示输入的文字。

输入文字后，我们可以对其样式进行个性化设置。点击底部的 **`样式`** 选项，可以进行以下调整：
*   **字体**：更换字体，如选择“新青年体”。
*   **样式**：设置描边、标签、阴影等效果。例如，可以设置描边的颜色（红、灰、黄、蓝）和粗细。
*   **花字**：直接应用抖音官方提供的各种特色艺术字体。
*   **气泡**：选择对话气泡样式的文本框。
*   **动画**：为文字添加入场、出场或循环动画。例如，添加一个“螺旋上升”的入场动画，并可以设置动画的持续时间。

![](img/f030e424ae187e59f7561c6de5a940e6_3.png)

设置完成后，点击对勾保存。预览视频，即可看到添加了动画和样式的字幕效果。

### 2. 识别字幕（语音转文字）

如果视频中包含人声对话或旁白，可以使用“识别字幕”功能自动生成字幕。

![](img/f030e424ae187e59f7561c6de5a940e6_5.png)

![](img/f030e424ae187e59f7561c6de5a940e6_6.png)

以下是操作步骤：
1.  确保视频轨道上已添加包含人声的音频。
2.  点击 **`文字`** -> **`识别字幕`**。
3.  点击 **`开始识别`**。
4.  系统会自动识别音频中的语音，并生成对应的文字轨道。

识别完成后，预览视频，字幕会自动匹配语音出现。你可以像编辑“新建文本”一样，对这些自动识别的字幕进行样式修改。

### 3. 识别歌词（音乐歌词转文字）

![](img/f030e424ae187e59f7561c6de5a940e6_8.png)

如果为视频添加了带歌词的歌曲，可以使用“识别歌词”功能快速生成歌词字幕。

![](img/f030e424ae187e59f7561c6de5a940e6_10.png)

![](img/f030e424ae187e59f7561c6de5a940e6_12.png)

以下是操作步骤：
1.  为视频添加一首带歌词的歌曲（如从“抖音收藏”中添加）。
2.  点击 **`文字`** -> **`识别歌词`**。
3.  点击 **`开始识别`**。
4.  系统会自动识别音乐中的歌词，并生成对应的文字轨道。

![](img/f030e424ae187e59f7561c6de5a940e6_14.png)

识别完成后，歌词会显示在视频画面上。同样，你可以对这些歌词字幕进行样式调整，例如更换字体、添加描边、设置动画，以提升视频的意境和观感。

### 4. 贴纸

在`文字`功能模块中，还可以选择添加`贴纸`，为视频增加一些趣味性的图形元素。

当所有文本信息添加并设置完毕后，即可导出完成视频制作。

![](img/f030e424ae187e59f7561c6de5a940e6_16.png)

## 课程总结

![](img/f030e424ae187e59f7561c6de5a940e6_19.png)

本节课中我们一起学习了剪映的文本功能。我们了解到，该功能可以快速为视频添加字幕，并能对字幕的颜色、字体、字号、动画、间距等进行一系列个性化调整。

通过手动新建、识别字幕和识别歌词这三种主要方式，我们可以高效地完成视频字幕的添加工作。合理使用字幕，能显著提升视频内容的信息传达效率和整体质量。
