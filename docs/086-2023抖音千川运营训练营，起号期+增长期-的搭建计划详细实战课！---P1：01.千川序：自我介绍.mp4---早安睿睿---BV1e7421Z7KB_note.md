# 抖音千川运营训练营：01：课程序言与讲师介绍 🎬

在本节课中，我们将学习本课程的概述，并了解讲师的基本背景与行业经验。课程将首先进行自我介绍，并初步探讨千川投放的核心前提与不同类目的投放效果差异。

大家好，我是本课程的讲师小北。我在电商行业已有十年经验。我曾是淘宝大学、万堂书院以及抖音巨量学的认证讲师。与其他讲师不同之处在于，我本人也运营过个人IP账号，在抖音拥有76万粉丝，在快手拥有300万粉丝。同时，我也是内蒙古爱心扶贫大使。在过去两年，我曾担任一家上市公司的副总裁，负责抖音直播基地业务，并操盘过多个知名品牌。我的座右铭是“慎独、刚毅、谦卑、坚忍、勤奋”，这源自曾国藩的《挺经》一书，对我启发很大。

本节课是千川投流实战课程的序言部分。在课程开篇，我需要强调一个关键前提：若想做好千川投放，你的口碑分和小店分必须大于4.6。如果尚未达到此标准，你需要先着手提升这两个分数。我会在后续每节课中都提醒这一点。

接下来，我们通过几个实际案例来了解千川投放的效果。首先是一个零食类目的案例，我们实现了23.4万的销售额，投放成本为1.04万，投资回报率（ROI）为5.32。第二个案例是数码产品类目，我们投放了83万，实现了203万的销售额，七日ROI为2.5。由此可见，不同类目的投放策略和效果存在显著差异。能够达成这样的投放金额，说明投流本身是一门学问。对于大盘账户或客单价较高的数码类产品，投放难度会更大。有人可能会质疑，如此高的投放费用和相对较低的ROI是否还有利润。请相信，如果没有利润，没有人会每月投入80多万进行投放。

第三个案例是女装类目，其投放效果更为显著。我们投放了6.8万，实现了220万的七日累计销售额，ROI高达31.86。看到这里，你可能会认为这个女装账号的投手非常厉害。但事实上，以上三个不同类目的账号都是由我本人操盘的。这说明，在投手经验一致的情况下，不同类目所能产出的ROI和效果天然不同。投手的经验固然重要，但并非决定性因素。有些类目天然缺乏自然流量，完全依赖千川付费推广，这类目ROI通常相对偏低。而对于有自然流量的直播间，其成交数据有时也会被计入千川的成交中，这一点需要大家理解。

从下节课开始，我们将进入更细致、更实战的讲解环节。
