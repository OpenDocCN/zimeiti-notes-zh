# 抖音全套运营教程：P7：人物设定 🎭

![](img/d0bbfeaf11e2cb8929d74b648cbca6e8_0.png)

在本节课中，我们将要学习抖音账号运营中一个至关重要的环节：**人物设定**。我们将探讨什么是人设、为何需要人设，以及如何系统地策划和打造一个具有吸引力和辨识度的账号人格。

![](img/d0bbfeaf11e2cb8929d74b648cbca6e8_2.png)

---

![](img/d0bbfeaf11e2cb8929d74b648cbca6e8_4.png)

上一节我们介绍了如何寻找对标账号和确定内容形式，本节中我们来看看如何为你的账号注入灵魂——**人物设定**。

![](img/d0bbfeaf11e2cb8929d74b648cbca6e8_6.png)

在抖音中策划人设，就像在游戏中创建角色一样，需要提前对人物角色进行设定。例如，角色的称呼、职业、拥有的技能点等。但与游戏角色不同的是，我们在抖音中设定的是**人生角色**，因此需要更具真实性。

![](img/d0bbfeaf11e2cb8929d74b648cbca6e8_8.png)

无论是短视频还是直播，一个成功的IP能够解决粉丝的信任问题。它能够起到裂变传播的作用，同时也具备很强的变现能力。因此，我们运营账号，无论是创作内容还是策划选题，其目的都是在获取流量的同时**打造IP**。打造IP的核心，就是在特定的场景或情境中，完成**账号人格化**的过程。

![](img/d0bbfeaf11e2cb8929d74b648cbca6e8_10.png)

这个过程需要注入情感、塑造故事，并贴上具有辨识度的标签。账号人格化，也就是我们通常所说的**账号人设**。

![](img/d0bbfeaf11e2cb8929d74b648cbca6e8_12.png)

对于账号人设的策划，我们可以从以下几个方面着手。以下是具体的人设策划维度：

![](img/d0bbfeaf11e2cb8929d74b648cbca6e8_14.png)

1.  **名字/姓氏/昵称**
    一个IP首先得有一个叫法，这是IP传播的名片。就像品牌的商标名一样重要，需要明确这个IP叫什么。

2.  **职业/职务/职位**
    人格化必然要具备身份。这是用户认识这个人或这个IP的联想入口，例如：老师、教授、博士、律师等。

![](img/d0bbfeaf11e2cb8929d74b648cbca6e8_16.png)

![](img/d0bbfeaf11e2cb8929d74b648cbca6e8_18.png)

3.  **称呼/称谓/社会关系**
    一个IP有了特定的称呼，能快速拉近与用户的距离，起到移情的作用。例如：XX哥、XX大爷、XX姥姥。

4.  **形象印象/第一印象**
    以用户的视角来看待IP，使IP形象化，更容易占据用户心智。例如：大叔形象、萌萌哒、铁憨憨、甜美可爱、光头等。

5.  **性格/爱好/特点**
    一个具有鲜明特点的IP，除了具有很强的记忆点，还具有很强的真实性。过于完美的形象容易脱离大众。例如：暴躁、娘炮、爱吃、爱喝酒等。

6.  **穿着打扮/财富地位**
    这属于社会的刻板印象和大众的普遍认知。借助用户的固有思维，可以为IP在用户心智中快速定位。例如：有钱代表会赚钱，开豪车代表成功人士，打扮土气可能代表老师。

以上内容策划好之后，就可以结合你自身的产品优势、服务优势等，将其落实到**账号搭建**中，即抖音账号的“五件套”：头像、昵称、背景图、简介、视频封面。

---

除了对账号IP进行人格化的基础策划，还需要通过持续的**内容输出**，为你的人设提炼出深刻的**记忆点**，即通常所说的**超级符号**。

超级符号的提炼可以从以下几个方面着手。以下是提炼超级符号的具体方向：

*   **画风（视觉）**：例如独特的场景、标志性道具、固定服饰等。
*   **听觉**：例如固定的BGM、特效音、配音的声线特点等。
*   **文案**：能够反映优势且容易记忆的提炼词，例如：“XX界天花板”、“简单粗暴”。
*   **表情/情绪传递**：例如标志性的笑、惊恐、无奈等表情包。
*   **语言特色**：例如方言、口音、错发音、网络语、慢条斯理或抑扬顿挫的语调等。
*   **动作/习惯**：例如习惯性的撩头发、扶眼镜等肢体动作。

超级符号提炼好之后，在你进行短视频创作时，就可以不断地在视频中重复呈现这些元素，从而强化用户记忆，形成独特的品牌资产。

---

本节课中，我们一起学习了抖音账号**人物设定**的核心逻辑。我们了解到，人设的本质是**账号人格化**，它通过赋予账号身份、性格、形象等元素来建立真实感和信任感。策划人设需要从名称、职业、形象、性格等多维度入手，并最终落实到账号的视觉呈现上。更重要的是，要通过提炼**超级符号**（如固定的视觉、听觉、语言或动作习惯），在内容中不断强化人设，使其深入人心。

一个清晰、独特且一致的人设，是你在抖音海量内容中脱颖而出、连接粉丝并实现长期价值的关键。
