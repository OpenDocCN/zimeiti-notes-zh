# 自媒体运营入门：P5：如何通过播放量变现

## 概述
在本节课中，我们将要学习自媒体行业区别于抖音、快手等平台的核心变现模式——通过视频播放量直接结算收益。我们将了解其运作原理、收益计算方式，并解答新手常见的关于播放量和收益的疑问。

上一节我们介绍了自媒体行业的入门优势，本节中我们来看看具体的赚钱模式。

## 为何选择靠播放量变现的行业
我选择这个行业有一个最基础的原因，它对于零基础的小白来说是最友好的。我当初选择自媒体也是因为它不需要我们有任何粉丝基础。我不是网红，也不是明星。现阶段我去做自媒体，没有粉丝也能做起来吗？很多人会有这样的疑惑和担心。

现在我告诉你们，自媒体平台和自媒体行业不需要我们有任何粉丝基础。我们去发视频，靠播放量依然可以去结算收益。

## 自媒体平台与抖音/快手的区别
很多同学了解最多的自媒体平台是抖音。我们平常用刷视频最多的平台，不是抖音就是快手。这两个平台不靠播放量结算收益。这两个平台靠什么去变现和赚钱呢？

![](img/b34655dd4b1b8c4ce879cc7d317b1253_1.png)

因为现阶段抖音和快手几乎已经变成了商业化模式，全部是靠带货去变现的。你们有没有在抖音上面买过东西？不管有没有用，都去买一点。大家都看见了，带货能赚钱。

但是我们自媒体平台跟抖音有一个不同的区别。我们在自媒体平台上面去发视频，靠播放量结算收益。

![](img/b34655dd4b1b8c4ce879cc7d317b1253_3.png)

## 收益如何计算：单价与提现门槛
我们需要了解的是，1万的播放量到底能够拿到多少钱。为什么是以万为单位呢？很多同学会有这样的疑问。当然有以千为单位的单价，但是只有达到1万播放量时产生的收益，我们才能够提现。

以下是收益计算的核心概念：

*   **千次阅读单价**：`1000次播放量收益 = 4.2元`
*   **万次播放量收益**：`10000次播放量收益 = 4.2元 * 10 = 42元`

![](img/b34655dd4b1b8c4ce879cc7d317b1253_5.png)

这42元就是未来我们能够提现到银行卡里的钱。我们想要去赚钱，首先得把这个单价记清楚。

## 行业前景与行动号召
我们知道这个单价了。这个行业是国家支持的，目前也没有饱和。我们纯靠播放量都可以去变现，不需要任何粉丝基础。这样的行业值得咱们去做吗？

![](img/b34655dd4b1b8c4ce879cc7d317b1253_7.png)

有句话说得非常好，顺风奔跑猪都能起飞。就看我们自己能不能去做一只风口上的猪了。

## 解答常见问题：播放量低的原因
很多同学说觉得可以做，因为赚钱。但是我发过视频，视频发出去为什么永远都是零播放量呢？为什么播放量有时候有100，甚至多的时候有1000的播放量，但提不了现，就只有几块钱？

是不是现在这个平台已经不需要拉新用户了？新用户注册对于我们来说没有任何扶持？平台给我限流了？其实不是这样子的。
