# 抖音运营教程：P51：文案素材与视频制作

在本节课中，我们将学习如何为已确定的视频创意寻找文案素材，并了解一种将文字素材转化为视频的实用方法。课程将重点介绍一个名为“淘金阁”的工具，并讲解其使用方法。

## 文案素材的获取

上一节我们讨论了视频创意的确定。当创意确定后，下一步就是寻找相应的文案素材。

当然，你也可以选择直接搬运他人的视频素材并进行二次处理。但如果你想根据特定创意搜索文案，并用自己的形式制作视频，那么寻找文字素材是更原创的途径。

## 从文案到视频的制作

![](img/92845e81150a510993467cab73c8e537_37.png)

找到文案后，接下来就是视频制作环节。

![](img/92845e81150a510993467cab73c8e537_39.png)

![](img/92845e81150a5110993467cab73c8e537_37.png)

本节将介绍制作视频的另一种形式。首先，请确保你已经掌握之前课程中提到的创意工具（如5118热词搜索）和本节的文案搜索工具（淘金阁），并在课后多加练习。

![](img/92845e81150a510993467cab73c8e537_39.png)

视频制作有多种方式，例如真人出镜拍摄，或者音乐影视混剪。对于混剪，你可以使用专门的网站获取素材，也可以采用录屏的方式。

之前课程中分享过的台词混剪网站、以及如“mv cat”这类提供电影下载的网站，都可以作为影视混剪的素材来源。

![](img/92845e81150a510993467cab73c8e537_41.png)

**本节课总结：**
我们一起学习了如何利用“淘金阁”工具为视频创意搜索文案素材，明确了将文字素材转化为视频不属于搬运行为。同时，我们回顾了视频制作的几种常见途径，包括真人拍摄和利用各类网站进行影视混剪。关键在于将找到的文案通过自己的创意和形式呈现出来。
