# 剪映电脑版教程：P2：制作文字出现效果 ✨

在本节课中，我们将学习如何在剪映PC版中制作一种动态的文字出现效果。这种效果能让文字像被“擦除”或“扫过”一样逐渐显现，为视频增添视觉吸引力。

---

![](img/5a0ce5a1b8a2550d990662cb351b2859_0.png)

许多朋友询问类似下图的文字出现效果是否能在剪映中实现。答案是肯定的。本节将分享在剪映PC版中实现这种效果的具体步骤。

![](img/5a0ce5a1b8a2550d990662cb351b2859_0.png)

## 第一步：添加与调整基础文本

首先，将文本素材拖拽到时间轴上，并输入你想要的文案内容。

![](img/5a0ce5a1b8a2550d990662cb351b2859_2.png)

接着，调整文字的字体和大小，然后将其导出为一个视频片段。这个片段将作为我们制作效果的基础素材。

![](img/5a0ce5a1b8a2550d990662cb351b2859_2.png)

## 第二步：应用混合模式与蒙版

现在，将刚才导出的文字视频素材，拖拽到时间轴中，并放置在你希望文字出现效果生效的原始视频素材的上方轨道。

以下是实现核心效果的关键操作列表：

1.  **选择混合模式**：在右侧检查器中，找到“混合模式”选项，并选择 **“绿色”** 模式。这会使文字素材的黑色背景变为透明，只留下白色文字。
2.  **添加线性蒙版**：在检查器中找到“蒙版”工具，选择 **“线性蒙版”**。画面上会出现一条可旋转和移动的分界线。
3.  **设置关键帧动画**：
    *   将播放指针移动到文字**结束出现**的时间点。
    *   在此位置，点击蒙版“位置”参数前的菱形图标，**打上一个关键帧**。此时，蒙版线应位于文字完全显露的位置（例如最右侧）。
    *   然后将播放指针移动到文字**开始出现**的时间点（更早一些）。
    *   在此位置，直接拖动画面中的线性蒙版，将其移动到文字起始的位置（例如最左侧）。剪映会自动在此处生成第二个关键帧。

这样，我们就创建了一个蒙版从右向左移动的动画，从而实现了文字逐渐显露的效果。

## 第三步：完善与制作第二个文字效果

完成上述步骤后，一个基本的文字出现效果就制作完成了。

![](img/5a0ce5a1b8a2550d990662cb351b2859_4.png)

如果你想制作第二个文字出现效果，可以重复此流程。拖入新的文本，调整其角度和颜色。然后，参照第一个文字素材的关键帧设置方法，为第二个文字设置关键帧动画。例如，可以让第二个文字从右向左移动出现。

**本节课总结**

在本节课中，我们一起学习了在剪映PC版中制作动态文字出现效果的方法。核心流程分为三步：**准备文本素材** -> **应用“绿色”混合模式与“线性蒙版”** -> **通过设置位置关键帧创建动画**。掌握这个技巧后，你可以轻松为视频标题或字幕添加吸引人的入场效果。
