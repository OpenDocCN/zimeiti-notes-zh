# 自媒体运营课程：P34：剪映核心剪辑功能——视频过渡转场 🎬

在本节课中，我们将要学习剪映软件中一个至关重要的剪辑功能——视频过渡转场。掌握这一技巧，能让你的视频片段衔接更加自然流畅，并显著提升视频的观赏性和专业感。

## 什么是视频过渡转场？

视频过渡转场，指的是在两个视频镜头之间，为了使衔接更自然、不生硬或不突兀，而采用的一系列视频处理技巧或剪辑方法。

在短视频制作领域，有一个“技术流派”。我们经常在短视频中看到的那些炫酷的运镜和转场效果，就运用了大量的视频拍摄转场技巧和运镜技巧。

![](img/08c51450e32a1d942f50c6b1cf2e77ab_1.png)

上一节我们介绍了基础的剪辑操作，本节中我们来看看如何利用剪映内置的转场功能，来提升我们整个视频的制作效果和视觉冲击力。

![](img/08c51450e32a1d942f50c6b1cf2e77ab_3.png)

## 转场的重要性与应用

![](img/08c51450e32a1d942f50c6b1cf2e77ab_5.png)

视频过渡转场是为了让两个镜头更好地衔接而进行的处理。在抖音等平台，我们经常能看到爆款的卡点视频，这些视频都借鉴了大量的转场和过渡技巧。

在剪映中，如果能巧妙利用这些特效转场，我们也可以快速制作出吸引人的卡点视频或高逼格的运镜视频。

![](img/08c51450e32a1d942f50c6b1cf2e77ab_7.png)

关于视频过渡转场的概念就介绍到这里。接下来，我们同样进入实操环节，详细讲解视频过渡和转场功能的具体使用方法。

![](img/08c51450e32a1d942f50c6b1cf2e77ab_9.png)

## 实操：制作一个卡点视频

![](img/08c51450e32a1d942f50c6b1cf2e77ab_11.png)

我们将以制作一个卡点视频为例，进行简单的练习操作，演示如何使用视频的过渡转场功能。

![](img/08c51450e32a1d942f50c6b1cf2e77ab_13.png)

### 第一步：创建项目与导入素材

![](img/08c51450e32a1d942f50c6b1cf2e77ab_15.png)

首先，我们创建一个新项目，并导入几张照片作为素材。

![](img/08c51450e32a1d942f50c6b1cf2e77ab_17.png)

![](img/08c51450e32a1d942f50c6b1cf2e77ab_18.png)

![](img/08c51450e32a1d942f50c6b1cf2e77ab_19.png)

导入素材后，要制作卡点视频，我们需要选择一首动感十足或平台热门的背景音乐（BGM）。

![](img/08c51450e32a1d942f50c6b1cf2e77ab_21.png)

### 第二步：添加并设置音乐

![](img/08c51450e32a1d942f50c6b1cf2e77ab_23.png)

在“音频”选项中，选择“抖音收藏”，导入我们事先收藏好的音乐。

![](img/08c51450e32a1d942f50c6b1cf2e77ab_25.png)

试听音乐，可以明显听到其中有“动词打次”的节奏点，这种音乐非常适合制作卡点视频。点击“使用”导入音乐。

![](img/08c51450e32a1d942f50c6b1cf2e77ab_27.png)

![](img/08c51450e32a1d942f50c6b1cf2e77ab_29.png)

音乐导入后，接下来要进行“卡点”操作，即“踩点”。我们需要将音乐中的重音节拍标记出来，以便后续根据这些节点来设置照片的切换和转场。

以下是设置音乐踩点的步骤：
1.  点击音频轨道。
2.  在底部工具栏选择“踩点”。
3.  打开“自动踩点”功能。

![](img/08c51450e32a1d942f50c6b1cf2e77ab_31.png)

![](img/08c51450e32a1d942f50c6b1cf2e77ab_33.png)

可以看到有两个节拍选项：“节拍1”和“节拍2”。“节拍2”划分的踩分点更密集，适合制作炫酷的卡点视频；“节拍1”则更适合制作柔和型、带特效的卡点视频。

![](img/08c51450e32a1d942f50c6b1cf2e77ab_35.png)

我们以“节拍2”为例，点击对号确认。此时，音频轨道上会出现一系列黄色小点，这就是我们分好的节奏节点。

![](img/08c51450e32a1d942f50c6b1cf2e77ab_37.png)

![](img/08c51450e32a1d942f50c6b1cf2e77ab_38.png)

### 第三步：调整素材时长与对齐节点

![](img/08c51450e32a1d942f50c6b1cf2e77ab_40.png)

![](img/08c51450e32a1d942f50c6b1cf2e77ab_42.png)

现在，我们需要根据踩好的点来调整每张照片的时长，使其与节奏点对齐。

![](img/08c51450e32a1d942f50c6b1cf2e77ab_43.png)

![](img/08c51450e32a1d942f50c6b1cf2e77ab_45.png)

操作方法是：选中第一张照片，拖动其右侧边缘，使其结束时间线与第一个节奏点对齐。第二张、第三张照片同理操作，依次与后续的节奏点对齐。

![](img/08c51450e32a1d942f50c6b1cf2e77ab_46.png)

![](img/08c51450e32a1d942f50c6b1cf2e77ab_48.png)

调整完成后，照片的切换点就与音乐的节拍同步了。

![](img/08c51450e32a1d942f50c6b1cf2e77ab_49.png)

![](img/08c51450e32a1d942f50c6b1cf2e77ab_51.png)

### 第四步：添加转场效果

![](img/08c51450e32a1d942f50c6b1cf2e77ab_53.png)

![](img/08c51450e32a1d942f50c6b1cf2e77ab_55.png)

如果不添加任何转场，直接预览视频，会发现镜头切换非常生硬，缺乏动感和节奏感。现在，我们来为每个衔接点添加转场效果。

![](img/08c51450e32a1d942f50c6b1cf2e77ab_57.png)

![](img/08c51450e32a1d942f50c6b1cf2e77ab_58.png)

以下是添加转场的具体步骤：
1.  将时间轴放大，以便更清晰地看到两个视频片段之间的衔接点（一个小方块图标）。
2.  点击这个小方块。
3.  系统会弹出转场效果库，里面包含**基础转场**、**运镜转场**、**幻灯片**、**特效**、**遮罩**等多种类别。
4.  我们可以尝试使用“运镜转场”类别下的效果，例如“推进”或“拉远”。
5.  为每一个照片之间的衔接点都添加一个转场效果。

![](img/08c51450e32a1d942f50c6b1cf2e77ab_60.png)

![](img/08c51450e32a1d942f50c6b1cf2e77ab_62.png)

添加完成后，再次预览视频，你会发现镜头的切换变得流畅且富有动感，视觉效果提升显著。

![](img/08c51450e32a1d942f50c6b1cf2e77ab_63.png)

![](img/08c51450e32a1d942f50c6b1cf2e77ab_64.png)

![](img/08c51450e32a1d942f50c6b1cf2e77ab_65.png)

### 第五步：优化画面与添加特效

![](img/08c51450e32a1d942f50c6b1cf2e77ab_66.png)

![](img/08c51450e32a1d942f50c6b1cf2e77ab_67.png)

在制作过程中，可能会发现导入的照片画幅比例不一致。为了使画面看起来更匀称，我们需要统一调整。

![](img/08c51450e32a1d942f50c6b1cf2e77ab_68.png)

![](img/08c51450e32a1d942f50c6b1cf2e77ab_69.png)

![](img/08c51450e32a1d942f50c6b1cf2e77ab_71.png)

操作方法是：选中每张照片，在编辑面板中调整其缩放和位置，使其适配统一的画布比例（如9:16）。

![](img/08c51450e32a1d942f50c6b1cf2e77ab_73.png)

![](img/08c51450e32a1d942f50c6b1cf2e77ab_75.png)

如果觉得只有转场效果还比较单调，为了进一步提升视频的绚丽程度，我们还可以添加一些特效。

![](img/08c51450e32a1d942f50c6b1cf2e77ab_77.png)

例如，可以在视频开头添加一个“变清晰”特效，或在某些节奏点添加“动感”类特效（如“抖动”、“金片”等）。添加后，同样需要调整特效的时长和位置，使其与视频节奏匹配。

![](img/08c51450e32a1d942f50c6b1cf2e77ab_79.png)

![](img/08c51450e32a1d942f50c6b1cf2e77ab_81.png)

添加特效后再次预览，视频的整体效果会更加丰富多彩。

![](img/08c51450e32a1d942f50c6b1cf2e77ab_82.png)

## 课程总结 🎯

![](img/08c51450e32a1d942f50c6b1cf2e77ab_84.png)

![](img/08c51450e32a1d942f50c6b1cf2e77ab_86.png)

本节课中我们一起学习了剪映中的视频过渡和转场功能。

![](img/08c51450e32a1d942f50c6b1cf2e77ab_88.png)

其核心操作是：**点击视频轨道上两个片段之间的衔接点（小方块），即可从效果库中选择并添加转场效果**。剪映提供了大量的转场素材，包括基础转场、运镜转场、幻灯片、遮罩转场等。

利用这些转场方式，可以极大地提升我们视频制作的流畅度、节奏感和丰富性，对于后期制作，特别是卡点类视频，有非常大的帮助。
