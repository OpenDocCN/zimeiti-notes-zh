# 短视频运营教程：P31：如何寻找书单类对标账号 📚

在本节课中，我们将学习在短视频平台（以抖音为例）上，如何高效地寻找与自身定位相符的对标账号。我们将重点讲解两种实用的方法，帮助你快速锁定模仿对象，为后续的内容创作和账号运营打下基础。

上一节我们介绍了寻找对标账号的重要性，本节中我们来看看具体如何操作。

## 方法一：利用“关注”推荐功能

![](img/5fa912023b326e4d9d41bb935916f41d_1.png)

此方法的核心是利用平台的推荐算法。当你关注一个目标账号时，平台会基于标签相似度，为你推荐更多同类型的账号。

操作步骤如下：

![](img/5fa912023b326e4d9d41bb935916f41d_3.png)

![](img/5fa912023b326e4d9d41bb935916f41d_4.png)

1.  **搜索并筛选目标账号**：在抖音搜索框输入你计划运营领域的关键词（例如“书单”）。从搜索结果中，找到一个视频类型符合你预期、且非真人出镜的账号。
2.  **进入账号主页**：点击该账号的头像进入其主页，先浏览其视频内容，确认是否与你想做的类型相符。
3.  **触发关联推荐**：如果内容相符，点击“关注”按钮。点击后，页面下方会立即弹出一个“你可能感兴趣”的账号列表。这些账号通常与当前账号的官方标签高度相似。
4.  **批量筛选与关注**：逐一查看推荐账号的作品，判断是否为你想要模仿的类型。如果是，则点击关注；如果不是，则返回继续查看下一个。通过此方法，可在两三分钟内关注数十个同类账号。此后，平台的信息流会为你推荐更多相似账号。

## 方法二：通过商品“达人秀”寻找

![](img/5fa912023b326e4d9d41bb935916f41d_6.png)

![](img/5fa912023b326e4d9d41bb935916f41d_8.png)

此方法适用于已有明确商品（如特定书籍）目标的运营者。通过搜索商品并查看其销售页面的“达人秀”，可以找到大量推广该商品的账号。

操作步骤如下：

1.  **搜索目标商品**：在抖音右上角搜索框，直接输入具体的商品名称（例如“墨菲定律漫画版”）。
2.  **筛选商品结果**：搜索结果默认显示“综合”标签，需手动点击切换到“商品”标签。
3.  **按销量排序**：进入商品页面后，点击“销量”进行排序，筛选出销量最高的商品链接。
4.  **进入商品详情页**：点击销量高的商品，进入其商品详情页。对于销量较高（例如2万以上）的商品，页面通常设有“达人秀”或“买家秀”板块。
5.  **发现对标账号**：“达人秀”板块内展示了大量推广该商品的短视频及发布账号。这些账号的内容形式（如影视混剪、语录剪辑）和定位（销售书籍）高度相似，是极佳的对标和模仿对象。一个商品链接下可能汇聚数百个推广账号，提供了丰富的参考样本。

以下是两种方法的对比总结：

![](img/5fa912023b326e4d9d41bb935916f41d_10.png)

*   **适用场景**：方法一适用于领域宽泛的初步探索；方法二适用于有具体商品目标的精准寻找。
*   **核心逻辑**：方法一依赖平台算法推荐；方法二依赖商品关联的创作者数据。
*   **效率**：两种方法均能高效、批量地发现对标账号。

本节课中我们一起学习了在抖音寻找书单类对标账号的两种核心方法：通过关注触发平台推荐，以及通过商品“达人秀”发掘推广账号。掌握这些方法，能帮助你快速建立自己的参考账号库，明确内容方向。
