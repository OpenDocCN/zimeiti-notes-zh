# 视频号运营：P26：二创书单制作方法一 📚

在本节课中，我们将学习如何通过二次创作（二创）来制作书单类视频。这是一种比简单混剪原创度更高、效果更好的方法，尤其适合新手入门。我们将从寻找素材、提取文案、替换片段到最终成片，一步步拆解整个流程。

---

## 剪辑是必备技能

上一节我们介绍了混剪的基础概念，本节中我们来看看如何通过二次创作提升视频的原创度。对于新手而言，剪辑是视频制作的必备技能。许多初学者觉得混剪和剪辑很难，根源在于缺乏实际操作经验。

解决方法是动手练习。你可以先尝试剪辑十几个视频，熟悉每个功能的作用。花费一两个小时练习后，再回头看教程，会发现一切变得简单易懂。

## 为何选择书单带货

对于想做纯视频带货的学员，我建议尝试书单类视频。书单是内容领域的常青树，只要认真执行，通常都能获得订单。

除了之前课程中讲解的混剪方法，我们还可以进行二次创作。你会发现，许多成功的书单视频，其素材都来源于网络，并非实拍。二次创作的本质，是将各种视频素材打乱顺序后重新组合。这种方式比单一视频混剪的原创度更高。

## 理解书单账号的底层逻辑

书单账号的运营逻辑很清晰：通过一段影视剧片段或小故事阐述一个道理，最后引出一本书进行推荐，从而激发观众的购买欲望。

这种通过内容引发共鸣，再自然推荐产品的方式，比直接展示产品功能的“硬广”效果要好得多。建议大家先按照上述书单搜索，并通过“达人秀”功能多刷一些对标账号，感受其内容逻辑。

## 二次创作实操步骤

![](img/09bcf826b2a37f16a3fc5680c54fbc8f_7.png)

接下来，我们进入核心环节，一步步演示如何进行书单视频的二次创作。

![](img/09bcf826b2a37f16a3fc5680c54fbc8f_9.png)

![](img/09bcf826b2a37f16a3fc5680c54fbc8f_11.png)

![](img/09bcf826b2a37f16a3fc5680c54fbc8f_13.png)

### 第一步：寻找并分析对标视频

![](img/09bcf826b2a37f16a3fc5680c54fbc8f_15.png)

1.  打开抖音，搜索目标书籍，例如“女孩要懂得保护自己”。
2.  点击“商品”标签，按“销量”排序。
3.  选择销量高的商品，进入其“达人秀”页面。这里汇集了众多卖家的推广视频。
4.  分析一个高赞视频（例如点赞数7.8万的）。你会发现，这类视频通常由网上素材拼接而成，通过一段配音文案将道理讲清楚，最后展示书籍。

### 第二步：提取核心素材与文案

1.  复制视频链接，使用去水印工具（如“配音神器”）保存视频到相册。
2.  使用文案提取工具，将视频的配音文案提取出来。

### 第三步：生成新的配音

![](img/09bcf826b2a37f16a3fc5680c54fbc8f_17.png)

1.  检查提取出的文案，修正错别字。
2.  使用配音工具（如剪映或第三方APP）生成新的配音。可以选择不同的音色。
3.  试听配音，如果某段语速过快，可以添加停顿（例如1.5秒），然后重新合成。
4.  将最终生成的配音文件导出保存。

### 第四步：在剪辑软件中重组素材

![](img/09bcf826b2a37f16a3fc5680c54fbc8f_19.png)

1.  打开剪映，导入去水印后的原视频。
2.  关闭视频原声。
3.  使用**蒙版**功能遮盖原视频的字幕。
    *   点击“特效”->“基础”->“矩形蒙版”。
    *   调整蒙版大小覆盖字幕，选择“反转”。
    *   点击“背景”->“画布模糊”，使遮盖区域与背景融合。
4.  开始替换视频片段。以原视频第一个“女老师演讲”片段为例：
    *   **方法A（替换）**：分割出该片段，点击“替换”，在素材库搜索“女讲师”、“培训”等关键词，选择合适的无字幕素材进行替换。
    *   **方法B（删除并新增）**：删除原片段，在对应位置添加自己找到的新素材。确保新片段时长与原片段大致相同，并关闭其原声。
5.  重复以上步骤，替换其他你想更改的片段（如“学生在操场”、“夜晚学习”等场景）。素材可以从抖音直接搜索相关关键词保存，或使用剪映素材库。
6.  视频结尾展示书籍的片段，也可以从其他达人秀视频中截取替换，进一步增加原创度。

### 第五步：合成音频与添加字幕

![](img/09bcf826b2a37f16a3fc5680c54fbc8f_21.png)

![](img/09bcf826b2a37f16a3fc5680c54fbc8f_23.png)

1.  将第三步生成的配音音频导入剪映，拖入时间轴。
2.  根据配音节奏，微调视频片段的长度和位置，确保音画同步。
3.  添加背景音乐，选择符合视频基调的音乐，并调低音量，避免掩盖配音。
4.  使用“文本”->“识别字幕”功能，为新的配音生成字幕。
5.  为字幕选择合适的样式，并调整位置，确保不会与未遮盖干净的原字幕重叠。

### 第六步：导出视频

检查所有片段原声已关闭，音画同步无误后，点击导出。建议将帧率设置为60帧，以获得更流畅的播放效果。

## 总结与核心要点

![](img/09bcf826b2a37f16a3fc5680c54fbc8f_25.png)

![](img/09bcf826b2a37f16a3fc5680c54fbc8f_27.png)

本节课我们一起学习了书单视频二次创作的全流程。核心在于**替换、重组、重配音**。

![](img/09bcf826b2a37f16a3fc5680c54fbc8f_29.png)

![](img/09bcf826b2a37f16a3fc5680c54fbc8f_31.png)

**核心公式可归纳为：**
`二创书单视频 = 对标视频框架 + 替换部分视觉素材 + 全新配音/字幕 + 背景音乐`

**操作要点回顾：**
*   **重在实操**：多练习是掌握剪辑的唯一途径。
*   **素材来源**：抖音搜索、剪映素材库是两大主要素材来源。
*   **原创关键**：替换核心片段、使用全新配音和字幕，能有效提升视频原创度。
*   **细节处理**：务必关闭所有替换片段的原声，并处理好原视频的字幕。

![](img/09bcf826b2a37f16a3fc5680c54fbc8f_33.png)

![](img/09bcf826b2a37f16a3fc5680c54fbc8f_35.png)

通过这种方法，你就能制作出原创度更高、更具个人特色的书单推荐视频了。
