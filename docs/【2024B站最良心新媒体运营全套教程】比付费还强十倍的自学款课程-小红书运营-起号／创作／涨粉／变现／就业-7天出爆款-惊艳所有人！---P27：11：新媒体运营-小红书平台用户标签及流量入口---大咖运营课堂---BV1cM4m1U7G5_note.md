小红书系统为用户打标签的主要依据是用户的**浏览行为**。用户注册时选择的兴趣范围参考价值不大，它仅作为初期内容方向的引导，无法进入系统深层的用户标签池。

### 1. 点击率
**点击率** 是指系统在发现页向用户推送多个内容后，用户点击各类内容的频率。例如，系统推送了明星、美妆、家居、母婴等内容，用户点击美妆内容三次，家居内容两次，母婴内容一次或未点击。系统通过算法判定该用户对母婴内容不感兴趣，可能不是宝妈，而对美妆内容更感兴趣。随后，系统会持续推送更多美妆内容进行试探，从而为用户打上强关联或弱关联的标签。

**公式示例**：
```
用户点击率分析 = 统计(用户点击某类内容的次数 / 系统推送该类内容的总次数)
```

### 2. 用户标签云图示例
为了更直观地理解，我们可以想象一个标签云图。假设一个用户的标签云图中，“美妆”标签最大，其次是“美甲”、“美发”，而“母婴”、“装修”等标签很小。这代表该用户是美妆领域的重度兴趣用户。系统会主要为其推送美妆相关内容，减少或不推送其不感兴趣的内容，以提升用户的平台停留时长和获得感。

这体现了平台、用户和博主三方的关系：
*   **博主** 创作内容，为用户和平台提供价值。
*   **平台** 通过算法为内容匹配标签相符的用户，也为用户提供价值。
*   **用户** 消费内容，其行为数据反馈给平台，优化推荐系统。

### 3. 停留时长
**停留时长** 是分析用户喜好的另一关键指标。例如，一个常看美妆的用户，如果对某篇美妆笔记很快划走，说明内容不吸引他。系统会尝试推送美妆领域下的其他细分内容（如腮红、护肤）。如果系统试探性地推送了一篇家居内容，用户不仅点击，还停留了很长时间，仔细浏览图片和评论，系统就可能判定该用户有装修或购买家居用品（如盘子）的潜在需求，从而考虑推送相关商品或笔记。

### 4. 页面浏览量（PV）
**PV** 可以理解为用户浏览页面的深度。例如，用户看完一篇装修笔记后，还点进博主主页，连续看了第二、三、四篇笔记。这个浏览页面数就是PV。如果博主主页内容精致，就能有效提升用户的PV。高PV对账号健康度有很大帮助，并可能引导用户产生**关注**行为。关注是永久提升账号权重和粉丝基数的关键。

### 5. 互动行为
互动行为包括点赞、收藏、评论、关注。这些数据直接关系到笔记的传播效果和账号价值：
*   高互动数据能帮助笔记进入更大的推荐流量池。
*   有利于博主后续接单、变现。
*   能带动账号内其他笔记的流量，提升粉丝数。

## 小红书流量分发入口 🚀
了解用户标签后，我们来看笔记是如何通过不同入口分发给用户的。小红书主要有四个流量分发入口：

1.  **发现页**
    *   **内容来源**：主要包含用户已关注的达人发布的内容，以及系统根据用户标签（点击率、停留时长等）匹配的“可能喜欢”的内容。
    *   **平台价值**：为了内容多样化，发现页也会少量推荐用户可能不感兴趣或未接触过的内容，以满足用户探索、看攻略、看教程等多维需求。
    *   **对博主的意义**：这是笔记和账号获取流量的**主要来源**。

2.  **关注页**
    *   **内容来源**：纯粹展示用户所关注博主发布的内容。
    *   **用户习惯**：部分用户会直接在此页查看强关注博主的内容。

![](img/32ae0acfc3a9b9319c96390d2e71fe96_114.png)

3.  **附近页**
    *   **推送逻辑**：根据手机定位，推送附近及同城的相关内容。
    *   **内容特征**：以美食、本地旅游、线下店推广等强地域性内容为主。
    *   **对博主的意义**：如果你是线下实体店（如美容院、餐厅），在做信息流广告或内容时，需重点关注附近人群的推送。

4.  **购物页**
    *   **平台定位**：这是小红书向电商化发展、实现变现的核心入口。
    *   **用户习惯**：目前部分用户尚未形成在此直接购物的强习惯，更多人仍倾向于先搜索相关笔记。该入口主要匹配用户的购物需求。

本节课中我们一起学习了小红书用户标签的判定机制和四大流量分发入口。
*   系统通过分析用户的**点击率、停留时长、页面浏览深度（PV）和互动行为**来为其打上兴趣标签。
*   笔记主要通过**发现页、关注页、附近页和购物页**四个入口被推荐，其中**发现页**是流量的主要来源。
*   理解“平台-用户-博主”三者的价值关系，有助于我们创作出既能满足用户需求、又符合平台推荐逻辑的内容，从而有效提升账号的曝光与增长。
