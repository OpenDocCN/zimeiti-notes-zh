# 自媒体运营教程：P12：4 抖音新的流量推送机制 📊

在本节课中，我们将要学习抖音平台最新的流量推送机制——“铁粉机制”。了解这一机制，将帮助你理解近期流量变化的原因，并找到在新规则下获得更好推荐的方法。

![](img/2159208ae382ac30b428f2a4cc539535_1.png)

最近许多创作者的流量表现不佳，这是因为抖音的算法发生了重大变动。

![](img/2159208ae382ac30b428f2a4cc539535_1.png)

许多账号的流量出现了严重下滑，例如知名网红毛毛姐一夜掉粉80万。然而，这对中小型博主而言，却可能是一次实现弯道超车的好机会。坚持看到视频最后，你一定会有所收获。造成这一现象的主要原因是：

抖音的算法新增了一个“铁粉机制”。所谓“铁粉”，指的是那些长期支持你的粉丝。在以前，我们只能看到粉丝的总数，这导致了“强者恒强”的局面——粉丝多的人播放量就越多。即便只是发布一段毫无意义的吃水果视频，也能吸引走几十万甚至上百万的流量。

![](img/2159208ae382ac30b428f2a4cc539535_3.png)

![](img/2159208ae382ac30b428f2a4cc539535_3.png)

![](img/2159208ae382ac30b428f2a4cc539535_5.png)

对于普通创作者来说，这是一种不公平；对于平台生态的健康发展而言，这也是不健康的。

![](img/2159208ae382ac30b428f2a4cc539535_5.png)

因此，这个“铁粉系统”将在一定程度上打破这种局面，使流量的分配更加均衡，让用心做好内容的创作者能获得更多的流量推荐。无论一个账号拥有几百万粉丝，在其作品发布的初期，平台都会优先将内容推送给他的“铁粉”进行观看和评价。

只有通过了“铁粉”这一关的考核后，作品才会被进行大范围的推荐。通常，在10万播放量以内，“铁粉”贡献的播放量要占到60%到70%的比例。例如，我是一名运营教学博主，如果我因为近期流量不佳，就去发布一些炫富、博眼球的话题，我的老粉丝看到后就会感到不满。

他们会觉得我这个干货博主改变了初心，因此我的流量也不见得会因此变好。

![](img/2159208ae382ac30b428f2a4cc539535_7.png)

![](img/2159208ae382ac30b428f2a4cc539535_7.png)

接下来，我们解释一下为什么这对中小博主来说是一次机会。首先，你需要明白一个基本概念：不同行业或赛道的粉丝体量天花板是不同的。你做娱乐号，粉丝涨到1000万都有可能；但如果你做的是技能培训，比如教开挖掘机，粉丝体量的上限可能就只有10万左右。

原因主要有以下两点：
1.  **学习内容相对枯燥**：用户观看短视频主要还是以娱乐消遣为主要目的。
2.  **目标受众基数小**：学习开挖掘机的人本身在短视频平台上的数量就比较有限。

本节课中，我们一起学习了抖音新推出的“铁粉机制”。其核心在于**作品发布初期优先推送给核心粉丝（铁粉）**，通过他们的反馈来决定是否进行更大范围的推荐。这要求创作者必须**持续输出符合账号定位、满足老粉丝期待的内容**，而不是盲目追逐热点。对于中小博主而言，这正是一个凭借**垂直、优质内容**深耕细分领域，从而获得稳定流量的好时机。
