## 对账规则与逻辑

上一节我们介绍了抖音来客的基本操作，本节中我们来看看财务对账的核心规则。理解对账逻辑是确保收入准确无误的基础。

每张团购券核销后的结算，都包含**收入**和**支出**两大部分。

*   **订单实收**：这是商家实际收到的、用于计算各项费用的基准金额。
    *   **计算公式**：`订单实收 = 券售卖金额 - 商家优惠`
    *   **另一种理解**：`订单实收 = 用户实付 + 抖音支付优惠 + 平台补贴金额`

*   **支出汇总**：这是平台从“订单实收”中扣除的各项佣金和费用。所有佣金均以“订单实收”为基数计算。
    *   **计算公式**：`支出汇总 = 软件服务费 + 达人佣金 + 服务商佣金 + 撮合经济抽佣 + 保险费用`
    *   **软件服务费说明**：例如美食品类总费率为2.5%，它包含0.6%的支付服务费和1.9%的剩余部分。具体标准请参考官方《2022年生活服务软件服务费标准说明》。
    *   **计算规则**：每一笔支出的系统计算规则是**四舍五入，保留两位小数**。

*   **商家最终提现金额**：扣除所有费用后，商家实际能提现的金额。
    *   **计算公式**：`提现金额 = 订单实收 - 支出汇总`

订单核销后，款项不会立即到账。系统会根据设定的“账期”自动发起提现。
*   **示例**：如果商户账期为 **T+5**（即5天），那么8月2日核销的订单，系统会在8月7日自动发起提现。

## 核销与分账明细查询步骤

理解了规则后，接下来我们进入实操环节，学习如何在抖音来客后台查询和核对明细。

![](img/8e464a54a1dd7814827ed0cce122fd7e_10.png)

### 第一步：查看账期

![](img/8e464a54a1dd7814827ed0cce122fd7e_12.png)

首先，你需要确认自己账户的结算账期。
1.  登录抖音来客电脑端。
2.  点击进入 **「财务管理」->「提现记录」** 或 **「财务管理」->「收款账户」**。
3.  在页面最上方可以找到账期信息。
4.  你也可以在「收款账户」的 **「账户详情」** 中查看具体的收款账期。
    *   如果显示 **T+5**，即表示订单核销后第5天，款项会自动打款至你的收款账户。

### 第二步：查询核销与分账信息

确认账期后，即可下载对账所需的明细数据。
以下是查询和导出数据的路径：

![](img/8e464a54a1dd7814827ed0cce122fd7e_14.png)

1.  **查询核销明细**：
    *   路径：**「订单管理」->「核销明细」**。
    *   在此页面，你可以查看筛选日期内的所有核销记录。
    *   点击 **「导出数据」**，即可下载核销明细的Excel表格。

2.  **查询提现与分账明细**：
    *   路径：**「财务管理」->「提现记录」**。
    *   在此页面，可以查看每个收款账户的提现记录及对应的分账明细。
    *   点击 **「导出提现记录」** 和 **「导出分账明细」**，即可下载相应数据。

![](img/8e464a54a1dd7814827ed0cce122fd7e_16.png)

**数据下载时间范围示例**：
假设账期为T+5，你需要核对8月1日至8月15日的核销与提现情况。
*   **核销明细**：应下载 **8月1日 - 8月15日** 的数据。
*   **分账明细**：由于有5天账期，8月1日核销的订单会在8月6日提现，因此应下载 **8月6日 - 8月20日** 的数据。

### 第三步：核对订单数量

![](img/8e464a54a1dd7814827ed0cce122fd7e_18.png)

下载完明细表后，我们首先核对订单数量是否一致。
以下是核对步骤：

![](img/8e464a54a1dd7814827ed0cce122fd7e_20.png)

1.  在 **「核销明细」** 表格中，筛选出 **“已核销”且“提现成功”** 的记录，并统计其总条数。
2.  在 **「分账明细」** 表格中，筛选出 **“提现成功”** 的记录，并统计其总条数。
3.  对比两个数量。**如果条数一致，则说明核销和提现的订单数量匹配**。

![](img/8e464a54a1dd7814827ed0cce122fd7e_22.png)

### 第四步：核对提现金额

![](img/8e464a54a1dd7814827ed0cce122fd7e_24.png)

订单数量核对无误后，接下来进行最关键的一步：核对提现金额。
以下是金额核对的步骤：

1.  基于第三步的筛选结果，在 **「核销明细」** 表中，汇总 **“商家应得（提现金额）”** 这一列的总和。
2.  在 **「分账明细」** 表中，汇总 **“提现金额”** 这一列的总和。
3.  对比这两个总和。**在账目正确的情况下，这两个金额应该完全一致**。

**补充说明**：
在核销明细中，“商家应得”即系统计算出的你可提现的金额，其逻辑为：
`商家应得 = 订单实收 - 支出汇总`
其中，每一笔佣金支出均遵循 **四舍五入，保留两位小数** 的规则计算，然后加总得出总支出。

![](img/8e464a54a1dd7814827ed0cce122fd7e_26.png)

### 第五步：完成对账

![](img/8e464a54a1dd7814827ed0cce122fd7e_28.png)

当核销与提现的订单数量、以及提现金额都核对无误后，本次对账操作就全部完成了。

![](img/8e464a54a1dd7814827ed0cce122fd7e_30.png)

## 问题咨询

如果在上述对账过程中发现任何数量或金额不符的问题，你可以通过以下途径寻求帮助：
*   在抖音来客电脑端页面右上角，找到 **「小助手」** 图标。
*   点击发起在线咨询，向官方客服反馈你遇到的问题。
