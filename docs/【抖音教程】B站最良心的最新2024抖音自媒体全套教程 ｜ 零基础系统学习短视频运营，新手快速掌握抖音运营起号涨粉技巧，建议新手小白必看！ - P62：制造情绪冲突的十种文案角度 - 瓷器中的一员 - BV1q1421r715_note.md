# 抖音短视频运营教程：P62：制造情绪冲突的十种文案角度

在本节课中，我们将要学习如何通过文案制造情绪冲突，从而提升短视频内容的吸引力和说服力。文案的表达方式直接影响用户的感受和行动意愿，掌握制造情绪冲突的技巧至关重要。

## 文案表达方式的重要性

上一节我们介绍了选题方法，本节中我们来看看文案的具体表达。同样一个选题，用不同的方式说出来，其表达效果截然不同。

直接陈述观点往往效果平淡。例如，劝说用户健身，如果说“来健身房变成更美的自己”，这句话就像一句普通的广告语，难以触动用户。

如果换一种表达，利用“失去心理”或“恐吓心理”来制造紧迫感，效果会显著不同。例如，“如果你不去健身房，你就失去了变美的机会”。这种表述会在用户心中形成一种紧迫感。

另一个例子是销售面膜。直接说“敷面膜永远保持20岁”同样干巴巴。如果说“不尽早护肤，你会长出难以去除的鱼尾纹和色斑”，用户可能会更重视。这说明，同样的选题，不同的文案表达给用户的心理暗示是不同的。

这类似于促销活动。告知“商品打五折”是一种刺激；若加上“仅剩最后两个名额”，则会制造更强的紧迫感，促使用户立即行动。两种刺激造成的效果不同。

## 制造情绪冲突的文案角度

因此，在文案中，我们需要刻意制造情绪的冲突。以下是制造情绪冲突的十种文案角度模板，可供参考和使用：

![](img/6d26c53a96e4c22cee1a5a3c561285dc_1.png)

1.  **直接陈述法**：平铺直叙地说明好处。例如：`来健身房变成更美的自己`。
2.  **失去心理法**：强调不行动会失去什么。例如：`不去健身，你将失去健康与活力`。
3.  **恐惧心理法**：描绘不行动的严重后果。例如：`不科学健身，可能导致运动损伤`。
4.  **稀缺性法**：制造机会或名额的稀缺感。例如：`限时优惠，仅剩最后X个名额`。
5.  **对比法**：将行动前后的状态进行对比。
6.  **故事共鸣法**：通过讲述故事引发用户情感共鸣。
7.  **挑战权威法**：提出与常识不同的观点，引发好奇。
8.  **利益承诺法**：明确给出可衡量的利益承诺。
9.  **群体认同法**：暗示这是大多数人的选择。例如：`90%的健身达人都在这样做`。
10. **指令行动法**：给出清晰、简单的行动指令。

**请注意**：前几种模板（如直接陈述法）效果相对普通，但某些行业仍可使用。恐惧心理和失去心理是较常使用且有效的方法，但不宜过于频繁，以免引起用户反感。

这个技巧可以运用到你所有的视频内容中。结合之前学习的九宫格选题法，你可以将不同的产品与用户需求填入九宫格，然后利用“连连看”的方式组合出选题，再运用上述制造情绪冲突的文案角度进行表达，从而创作出更具吸引力的内容。

例如，针对“健身”选题，使用九宫格延伸出“办公室人群”、“缓解腰痛”等子方向，再运用“恐惧心理法”创作文案：“长期久坐不锻炼？小心腰间盘突出找上门！”
