# 亚马逊开店教程：P12：如何注册店铺：资料准备 📋

![](img/40c2c7c4d8f69a4d2488d055bfaf58d0_0.png)

在本节课中，我们将学习注册亚马逊店铺前需要准备的所有核心资料。了解并备齐这些资料是成功开店的第一步。

上一节我们介绍了如何选择亚马逊站点，本节中我们来看看注册店铺具体需要准备哪些材料。

## 资料清单总览

注册亚马逊店铺需要准备六项核心资料。以下是每一项资料的详细说明与要求。

### 1. 营业执照彩色扫描件

第一份需要准备的资料是**营业执照的彩色扫描件**（即清晰的照片）。

*   **执照类型**：可以是**企业营业执照**，也可以是**个体工商户营业执照**。
*   **办理途径**：可在当地工商局免费办理。若没有执照，可借用家人或朋友的，或寻找代办机构协助办理，费用通常在几百至一千元人民币左右。
*   **重要要求**：
    *   对**注册资本**和**注册年限**无硬性要求。
    *   执照**经营范围**需带有销售性质（如零售、批发等）。
    *   **特别注意**：**切勿**在经营范围中添加“进出口贸易”字样。添加后需额外办理进出口手续，属于画蛇添足。
    *   执照的经营范围与您计划在亚马逊销售的产品**无需一致**。

### 2. 法人身份证正反面照片

第二份资料是**法人身份证的正反面清晰照片**。

*   **核心关联**：此处的“法人”指营业执照上的法定代表人。**营业执照与法人身份证必须是同一个人的**。
*   **后续资料关联**：除这两份资料必须一致外，其余注册资料（如下文将介绍的信用卡、收款账户等）可以与法人不是同一个人。

### 3. 付款信用卡

第三份资料是用于支付亚马逊平台月租的**付款信用卡**。

*   **唯一用途**：此卡**仅用于支付亚马逊每月39.99美元的店铺租金**。
*   **卡片类型**：必须是**全币种信用卡**，卡面上需有 **Visa** 或 **Mastercard** 标识。国内各大银行（如工商、建设、招商、中信等）均可办理。
*   **备选方案**：若无信用卡，可借用他人的，或花费约100元购买一张虚拟信用卡，每月按时存入月租金额即可。店铺产生销售额后，亚马逊会优先从店铺余额扣款，故此卡作用较为单一。

### 4. 海外收款账户

第四份资料是用于接收亚马逊销售货款的**海外收款账户**。

*   **核心功能**：它相当于**国际版的支付宝**。当您的亚马逊店铺产生销售额后，资金会先到达此账户，再由您提现至本人的国内银行卡。
*   **获取方式**：可通过第三方支付服务商（如Payoneer、PingPong、万里汇等）在线免费注册，流程类似注册支付宝，只需绑定本人身份证和国内银行卡即可。

### 5. 联系电话

第五份资料是一个有效的**国内手机号码**，用于接收验证码及官方通知。

### 6. 电子邮箱

第六份资料是一个**电子邮箱地址**，用于注册账号和接收重要邮件。

*   **邮箱推荐**：163邮箱、126邮箱、QQ邮箱等均可使用，其中**163邮箱**是较为推荐的选择。

本节课中我们一起学习了注册亚马逊店铺必须准备的六项核心资料：营业执照、法人身份证、付款信用卡、海外收款账户、联系电话和电子邮箱。确保这些资料真实、有效且符合要求，是顺利完成店铺注册的基础。下一节，我们将开始具体的注册流程操作。
