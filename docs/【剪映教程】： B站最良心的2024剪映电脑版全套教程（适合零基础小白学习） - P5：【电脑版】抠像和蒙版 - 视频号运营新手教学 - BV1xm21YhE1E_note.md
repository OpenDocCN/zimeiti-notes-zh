# 剪映PC教程：P5：抠像与蒙版 🎬

![](img/8a9af538b5d4fb594bf9a7c42aa1b035_0.png)

在本节课中，我们将学习剪映中两个非常强大且有趣的功能：**抠像**和**蒙版**。通过它们，我们可以轻松实现人物从电视中爬出、无实物托举等充满创意和视觉冲击力的视频效果。这些操作看似复杂，实则非常简单。

## 概述：两个核心工具

要实现本节课的案例效果，我们主要依赖两个工具：**抠像**和**蒙版**。抠像负责将主体从原始背景中分离出来，而蒙版则用于将不同的画面元素进行无缝融合。

现在，让我们打开剪映，将素材拖入时间线，并选中素材。在右侧的控制面板中，找到“基础”面板旁边的“抠像”和“蒙版”选项，这就是我们今天要学习的核心。

## 第一部分：抠像功能详解 🧑‍🎨

抠像，顾名思义，就是将画面中的特定主体“抠”出来。在剪映中，主要有三种抠像方式，适用于不同的场景。

### 1. 色度抠像

色度抠像是利用纯色背景（通常是绿色或蓝色）进行快速抠像的方法。其原理是吸取并移除指定的颜色，从而保留前景主体。

**操作公式**：`吸取背景色 -> 调整强度与阴影 -> 完成抠像`

以下是具体操作步骤：
1.  导入带有纯色背景（如绿幕）的素材。
2.  在“抠像”面板中选择“色度抠像”。
3.  使用“吸管”工具点击画面中的背景颜色。
4.  调整“强度”和“阴影”滑块，直到主体被干净地抠出，背景消失。

例如，要将一个绿幕恐龙放入电视画面，只需用吸管吸取绿色，调整参数，即可将恐龙完美抠出并放置到电视屏幕上。

### 2. 自定义抠像

当背景不是纯色，或者需要更精细地控制抠取范围时，就需要使用“自定义抠像”。它允许我们手动描绘出需要保留的部分。

**核心操作**：使用智能画笔涂抹要保留的区域，使用智能橡皮擦修正错误。

以下是自定义抠像的工作流程：
*   **智能画笔**：在主体上涂抹，软件会自动识别并选中相连区域。
*   **应用效果**：涂抹完成后，点击“应用效果”即可完成抠像。
*   **智能橡皮擦**：如果不小心选中了多余部分，可以用此工具擦除。
*   **橡皮擦**：用于更精细地擦除特定区域。

此功能非常适合背景复杂、主体明确的素材，如从街景中抠出人物。

### 3. 智能抠像

智能抠像是剪映提供的自动化工具，但它有一个明确的使用条件。

**重要提示**：智能抠像**仅支持对人物图像进行智能识别**。它的工作原理是识别人物的五官和轮廓。

因此，对于动物、物品或无人物的场景，智能抠像可能无法工作或效果不佳，此时应选择“自定义抠像”。

上一节我们详细介绍了三种抠像方法，它们能帮助我们将主体从原背景中分离。接下来，我们看看如何利用“蒙版”工具，将这些抠出的主体或其他视频片段进行创造性的合成。

## 第二部分：蒙版功能详解 🖼️

蒙版就像一个“形状遮罩”，可以隐藏或显示图层的某些部分，从而实现画面叠加、过渡和无缝拼接的效果。剪映提供了多种形状的蒙版。

**核心概念**：为图层添加一个蒙版形状，形状内的区域显示，形状外的区域隐藏。通过调整“羽化”值可以使边缘过渡更自然。

以下是几种常用蒙版及其应用：
*   **线性蒙版**：产生一条直线的分割过渡。常用于拼接两个画面的天空与地面。
*   **镜面蒙版**：产生两条平行线，显示中间部分。适合制作电影感的宽屏开场或特殊分屏。
*   **圆形蒙版**：显示圆形区域。常用于制作聚焦、窥视效果。
*   **矩形蒙版**：显示矩形区域。其独有的“圆角”功能可以让矩形边角变圆润。
*   **爱心/星形蒙版**：显示爱心或星形区域。用于制作创意图形效果。

所有蒙版都可以进行**位置**、**旋转**、**大小**和**羽化**的调整，你可以直接在预览窗口拖动控制点，也可以在参数面板输入精确数值。

掌握了抠像和蒙版的基本原理后，我们就可以将它们结合起来，动手复现本节课开头看到的那些神奇效果了。

## 第三部分：案例实战演练 ✨

我们将使用“抠像”结合“蒙版”或“调色”，完成三个核心案例。

### 案例一：人物从电视中爬出

这个效果需要将绿幕前的人物与电视背景合成。
1.  **抠出人物**：使用“色度抠像”吸取绿色，初步抠出人物。残留的绿边可能用蒙版处理更佳。
2.  **处理边缘**：添加一个“矩形蒙版”框住电视屏幕区域，让蒙版边缘与电视边框吻合，隐藏屏幕外多余的部分。
3.  **颜色修正**：进入“调节”功能，选择“HSL”工具，降低画面中**绿色**的饱和度和亮度，彻底消除可能的绿边。
4.  **调整位置**：将抠好的人物图层移动到电视屏幕上，调整其位置和大小，使其“爬出”的动作与电视边框自然衔接。

### 案例二：无实物托举锤子

这个效果需要将吊线锤子与人物表演合成。
1.  **抠出锤子**：使用“自定义抠像”的智能画笔仔细描绘锤子主体，用橡皮擦修整多余部分。
2.  **制作动画**：利用**关键帧**制作锤子的位移动画。在人物手部开始上抬的位置给锤子打一个位置关键帧；在手抬到最高处时，将锤子移动到相应位置，软件会自动生成上升动画。可以添加更多关键帧模拟锤子的晃动。
3.  **颜色融合**：通过“调节”功能降低锤子的亮度，使其光影与整体场景更匹配。

### 案例三：车内飞行特效

这个效果需要将绿幕车内景与天空背景合成。
1.  **抠出车与人**：综合使用“自定义抠像”描绘车内主体和“色度抠像”去除大块绿色背景。
2.  **去除绿边**：同样使用“HSL”工具，选择绿色，大幅降低其饱和度和亮度，消除抠像后人物和车体边缘的绿色杂边。
3.  **合成背景**：将抠干净的车内景图层，叠加到天空飞行的背景视频上方，即可完成“车内飞行”的错觉。

## 总结与练习 💪

本节课中，我们一起学习了剪映两大核心合成功能：
1.  **抠像**：包括**色度抠像**（用于纯色背景）、**自定义抠像**（用于复杂背景和精细控制）和**智能抠像**（用于人物镜头）。
2.  **蒙版**：包括线性、镜面、圆形、矩形等多种形状，用于控制图层的显示区域，实现画面拼接与融合。
