# 新媒体运营零基础入门：P12：01.10.0：AI：ChatGPT 4.0的几种使用渠道 🚀

## 概述
在本节课中，我们将学习如何获取和使用目前最强大的AI对话模型——ChatGPT 4.0。对于新媒体运营人员而言，掌握AI工具能极大提升工作效率，例如撰写脚本、制作PPT和生成报告。我们将介绍多种体验和使用ChatGPT 4.0的渠道，包括免费和付费选项。

## 微软Edge浏览器：Copilot
上一节我们介绍了AI对运营工作的重要性，本节中我们来看看第一个推荐渠道：微软Edge浏览器内置的Copilot功能。它是微软基于ChatGPT技术开发的AI助手，集成在Windows系统中，智能程度与4.0模型相近。

![](img/e0b4eff2e9621dea2f583a84643d636a_1.png)

![](img/e0b4eff2e9621dea2f583a84643d636a_3.png)

使用Copilot有两个前提条件：
1.  登录微软账号（可使用Outlook邮箱注册）。
2.  需要“梯子”（VPN）才能访问该服务。

Copilot提供三种对话模式：
*   **更有创造力**：AI的回答更具开放性和创意。
*   **平衡**：介于精确和创造力之间，适合一般用途。
*   **更精确**：回答更为严谨和准确，适合专业内容。

**示例：生成短视频脚本**
我们可以向Copilot提问，例如：
```
请帮我写一个关于装修的短视频脚本，时长控制在3分钟左右。
```
AI会生成包含场景、画面和台词的详细脚本。此外，Copilot还能基于你当前浏览的网页内容进行分析，辅助办公和电脑设置。

然而，Copilot也存在一些缺点：
1.  **精准度稍逊**：相较于官方ChatGPT 4.0，经过调校的Copilot在某些回答上可能不够精准。
2.  **连续对话能力受限**：在当前对话中持续提问次数有限，可能需要开启新对话窗口。
3.  **记忆功能有所缺失**：历史对话记录和分析能力不如官方版本强大，且缺乏插件功能。

如果您的Edge浏览器没有Copilot图标，可以尝试下载最新版的Edge Dev（测试版）。您可以在Edge设置中检查更新，或访问微软官网的Dev频道下载。

## 第三方平台与共享账号
了解了系统自带的工具后，我们来看看一些由开发者搭建的第三方平台。这些平台通常集成了多种AI模型，包括GPT-4，有些在国内网络环境下可直接访问。

以下是几个可用的第三方平台：

**1. 灵感AI**
*   **优势**：集成多种模型（如GPT-3.5、GPT-4、文心一言等）；新用户有一定免费使用额度。
*   **缺点**：生成速度可能较慢。
*   **费用**：提供免费试用，后续有包月等付费选项。

**2. Wee（示例名称）**
*   **优势**：响应速度快；支持GPT-3.5和GPT-4。
*   **缺点**：免费额度通常仅限GPT-3.5；需要“梯子”访问。
*   **功能**：可能包含语音交互等附加功能。

**3. ChatGPT共享账号平台**
*   **优势**：可以免费使用他人分享的ChatGPT账号（包括Plus会员账号）。
*   **使用方式**：平台会显示账号状态（绿色/空闲，橙色/使用中，红色/繁忙，金色/Plus会员）。找到可用账号后，输入密码即可登录使用。
*   **注意**：免费账号，尤其是GPT-4账号，可能因使用人数多而出现对话次数上限或响应慢的情况。

**重要提示**：使用第三方平台和共享账号时，请注意信息安全，避免输入个人敏感数据。部分平台需要“梯子”才能稳定访问。

如果访问上述渠道存在困难，我们还可以考虑国内公司开发的AI大模型。它们无需“梯子”，在国内网络环境下可直接使用，虽然能力与ChatGPT 4.0存在差距，但对于基础的运营辅助工作已足够。

**1. 百度文心一言**

## 官方ChatGPT 4.0体验
最后，我们来直观感受一下官方ChatGPT 4.0的能力。通过合适的渠道使用官方版本，通常能获得最优质、最连贯的AI交互体验。

![](img/e0b4eff2e9621dea2f583a84643d636a_9.png)

![](img/e0b4eff2e9621dea2f583a84643d636a_11.png)

**示例对比**
我们使用同样的指令测试：
```
请帮我写一份关于装修的短视频脚本，要求镜头丰富，语言吸引人。
```
官方ChatGPT 4.0生成的脚本在场景构建、镜头语言设计和叙事节奏上往往更细致、更有创意，体现出其在复杂任务处理上的优势。

选择工具的核心原则是：**什么工具能最高效地提升你的工作效果，就用什么工具。** 对于有条件的新媒体运营者，深入学习和使用ChatGPT 4.0将带来显著的工作提效。

## 总结
本节课我们一起学习了体验和使用ChatGPT 4.0的多种渠道：
1.  **微软Edge Copilot**：系统集成，方便但略有功能限制。
2.  **第三方平台与共享账号**：提供了更多选择，部分可免费体验GPT-4，但需注意速度和稳定性。
3.  **国内AI大模型**：如文心一言、豆包等，访问便捷，适合基础运营辅助工作。
4.  **官方ChatGPT 4.0**：能力最强，是追求高效工作的优选。
