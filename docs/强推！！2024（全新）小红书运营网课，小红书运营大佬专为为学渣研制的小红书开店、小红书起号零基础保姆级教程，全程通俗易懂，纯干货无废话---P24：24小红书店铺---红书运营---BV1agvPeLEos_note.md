上一节我们介绍了主体定位的重要性，本节中我们来看看构成主体定位的第一个核心要素：店铺定位。

## 店铺定位 🏪

店铺定位是确定店铺运营模式和销售类目的过程。它要求我们明确方向，有规划地运营，而非盲目尝试。

以下是店铺定位需要确定的三个关键点：

1.  **模式**：确定店铺的经营模式。主要分为**单店模式**和**多店模式**。对于新手，通常建议从**单店模式**开始。
2.  **类目**：选择要销售的产品类别。分为**大类目**（竞争激烈，需有好产品）、**小类目**（需求大且自身有推广优势）和**冷门类目**。选择需基于自身产品资源。
3.  **产品分析**：选定类目后，需分析产品的**价格**、**类型**和**时效性**。
    *   **价格**：需调研产品在小红书平台的定价范围及市场接受度。
    *   **类型**：产品的具体种类，与所选类目相关。
    *   **时效性**：产品的销售周期。对于**单店模式**，产品时效性应**长于6个月**，避免选择季节性过强的产品（如应季服装），否则店铺难以持续运营。

明确了“卖什么”和“怎么卖”之后，接下来我们需要确定“卖给谁”，这就是人群定位。

## 人群定位 👥

人群定位是指确定我们目标客户的特征范围，这与小红书账号运营中的人群定位概念一致，但此处更侧重于店铺的消费者。

以下是人群定位的五个主要维度，可用于前期构建店铺基础数据：

![](img/c24378f0aabe6f07990780c8291c644c_10.png)

1.  **个人属性**：包括性别、年龄范围、兴趣偏好等。
2.  **消费能力**：指用户的历史消费水平，由平台系统根据用户行为计算得出。
3.  **购买类型**：用户偏好购买的产品类型。
4.  **购买地域**：用户所在的地理区域。
5.  **购买习惯**：用户的购物行为习惯。

![](img/c24378f0aabe6f07990780c8291c644c_12.png)

![](img/c24378f0aabe6f07990780c8291c644c_14.png)

**注意**：人群定位并非强制步骤。进行精准定位可以帮助店铺更快起步；若资源有限，也可在运营后期逐步调整优化。

确定了目标人群，我们还需要制定与之匹配的产品定价策略，即价格定位。

## 价格定位 💰

价格定位决定了店铺吸引的用户类型，并与人群定位中的“消费能力”维度联动。系统会根据价格区间为店铺匹配相应的潜在客户。

价格定位主要分为三个级别，对应不同的店铺操作模式：

1.  **低端产品**：折扣力度大，通常定价为原价的**3折及以下**。主要吸引价格敏感型用户。
2.  **中端产品**：折扣适中，通常定价为原价的**4折至6折**。面向消费能力中等的用户。
3.  **高端产品**：折扣较小，通常定价为原价的**7折及以上**。面向消费能力较强、注重品质的用户，回头客比例可能更高。

**基础定价公式**：
产品售价 ≈ （产品成本 + 运费 + 预期利润）
在小红书平台，最终售价通常为产品出厂成本的 **1至3倍**。需结合计划给出的折扣来反推定价。例如：
`产品成本50元 + 运费10元 + 利润50元 = 基础价110元`。若计划打8折，则标价应为 `110 / 0.8 ≈ 138元`。

**核心原则**：定价需实在，与产品价值匹配。虚高定价难以在小红书获得用户认可，不利于店铺长期发展。折扣是一种营销手段，最终目的是在合理利润和吸引用户之间找到平衡。
