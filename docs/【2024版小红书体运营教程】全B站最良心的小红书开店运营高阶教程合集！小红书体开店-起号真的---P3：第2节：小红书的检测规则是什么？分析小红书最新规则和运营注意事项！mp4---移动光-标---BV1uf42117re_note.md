# 小红书运营教程：P3-2：小红书平台检测规则详解 🕵️

在本节课中，我们将要学习小红书平台的核心检测规则。了解这些规则是账号安全运营、避免违规限流的基础。我们将从多个维度分析平台禁止的行为，并提供实用的自查工具与方法。

上一节我们介绍了账号定位，本节中我们来看看平台有哪些具体的规则需要我们遵守。小红书的检测系统主要监控以下几个方面。

## 个人资料与内容审查

小红书平台会系统检测用户的**个人资料**、**笔记内容**、**私信**以及**评论**。

以下是这些区域严禁出现的内容：
*   微信号、微信二维码。
*   “代购”等直接营销词汇。
*   推广淘宝等第三方店铺的信息。
*   非平台许可的水印。

平台管理相对宽松的方面是：**个人资料中允许填写邮箱**。用户也可以使用微博ID或谐音词进行间接引流，再通过话术引导至微信。具体的引流话术技巧将在后续课程中详细讲解。

## 设备与行为规范

除了内容，平台也会检测运营行为本身。

以下是相关的设备与行为准则：
*   **设备检测**：同一台移动设备最多建议登录三个小红书账号，不宜过多。
*   **禁止诱导互动**：笔记内容中不得出现诱导用户关注、点赞或评论以抽奖领奖的词汇。例如“关注领奖”、“点赞抽奖”等均被明令禁止。
*   **保护原创度**：小红书对原创内容保护程度较高。切勿将同一篇文章用多个账号在平台内重复发布。原则是：**一篇文章，一个账号发布一次**。

## 敏感词检测机制

一个核心的检测规则是关于**敏感词**的过滤。平台对某些领域和词汇有严格限制。

以下是一些常见的违禁领域示例：
*   手机卡
*   金融理财
*   电子烟
*   医疗用品
*   代购

在撰写笔记时，若不小心加入了违禁词，笔记可能会被限流或无法通过审核。为了解决这个问题，我们可以借助外部工具进行自查。

## 实用敏感词检测工具

![](img/328e318eb604c697d2626207e773fda4_1.png)

以下是两个可用于检测小红书违禁词的在线工具，使用流程基本相同。

**工具一：即时检测网站**
1.  访问检测网站。
2.  将写好的笔记正文粘贴到输入框中。
3.  点击“立即检测”按钮。
4.  系统会高亮或列出文中包含的所有敏感词。

例如，输入一段包含“代购”的文本，检测结果会明确提示“代购”为敏感词。
```
检测前文本：“分享一次海外代购经历...”
检测结果：提示“代购”为敏感词。
```

**工具二：需注册的检测平台**
另一个网站功能类似，但需要简单注册后才能使用。其操作逻辑与第一个工具一致：粘贴文本，执行检测，查看过滤结果。

对于大多数用户而言，使用第一个免费工具进行日常检测已经足够。它可以帮助我们在发布前有效过滤掉可能违规的词汇。

本节课中我们一起学习了小红书平台的四大检测规则：个人资料与内容规范、设备登录限制、禁止诱导互动行为以及敏感词过滤机制。掌握这些规则并善用检测工具，是账号健康、稳定运营的前提。关于更全面的违禁词列表，我们正在整理中，将在后续及时提供给大家。
