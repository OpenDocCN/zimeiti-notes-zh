## 用户标签：平台如何“认识”你

![](img/a11b2ef16dc76b2c2019a8ce5c072693_38.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_40.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_42.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_44.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_46.png)

平台（如抖音、小红书）会给每一位用户建立详细的“数据画像”，即用户标签。这个过程基于用户的行为数据，目的是为了向用户精准推送其感兴趣的内容，从而提升用户粘性和平台价值。

![](img/a11b2ef16dc76b2c2019a8ce5c072693_48.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_50.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_52.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_54.png)

以下是平台判断用户兴趣并打标签的核心数据维度：

![](img/a11b2ef16dc76b2c2019a8ce5c072693_56.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_58.png)

1.  **点击率**：用户点击某类视频的概率。例如，平台推送了5条美食视频，用户全部点击观看，这表明用户对美食内容兴趣浓厚。
2.  **停留时长（完播率）**：用户点开视频后观看了多久。观看时间越长，代表内容越吸引用户。
3.  **互动行为**：用户对内容进行的点赞、评论、收藏等操作。这些行为直接反映了用户的参与度和喜好。

![](img/a11b2ef16dc76b2c2019a8ce5c072693_60.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_62.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_64.png)

我们可以用一个简单的公式来理解平台如何量化用户的兴趣偏好：

![](img/a11b2ef16dc76b2c2019a8ce5c072693_66.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_68.png)

```
用户对某领域兴趣值 = (点击率权重 * 点击次数) + (完播率权重 * 平均观看时长) + (互动行为权重 * 互动次数)
```

![](img/a11b2ef16dc76b2c2019a8ce5c072693_70.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_72.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_74.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_76.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_78.png)

平台通过持续分析这些数据，为每个用户生成一个动态的“兴趣词云”。例如，一个用户的标签画像可能显示：**美食(60%)**、**科技(25%)**、**母婴(15%)**。平台便会优先向其推荐更多美食相关内容，减少或不推荐母婴内容。

![](img/a11b2ef16dc76b2c2019a8ce5c072693_82.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_84.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_86.png)

## 创作者标签：如何让平台“记住”你

![](img/a11b2ef16dc76b2c2019a8ce5c072693_88.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_90.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_92.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_94.png)

作为内容创作者，我们最关心的是如何让平台为我们的账号打上准确的“创作者标签”。这决定了我们的内容会被推送给哪一类用户。

![](img/a11b2ef16dc76b2c2019a8ce5c072693_96.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_98.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_100.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_102.png)

许多人误以为只要多发某一领域的内容，平台自然就会识别。实际上，平台是通过分析你账号的整体数据表现来打标签的，其逻辑与用户标签类似，但视角不同。

![](img/a11b2ef16dc76b2c2019a8ce5c072693_104.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_106.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_108.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_110.png)

假设有两个母婴类创作者：
*   **创作者A**：内容混杂，既发母婴辅食，也发女装穿搭和日常生活打卡。
*   **创作者B**：内容高度垂直，只发布母婴相关的各类内容。

![](img/a11b2ef16dc76b2c2019a8ce5c072693_112.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_114.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_116.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_118.png)

当创作者A发布一条女装视频时，平台会根据其历史数据（曾发过母婴内容），将这条视频同时推送给“母婴用户”和“女装用户”。这会导致一个严重问题：占比较大的母婴用户可能对女装内容不感兴趣，直接划走，造成视频的**点击率、完播率、互动数据**全面下滑。平台会因此判定这条视频质量不佳。

![](img/a11b2ef16dc76b2c2019a8ce5c072693_120.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_122.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_124.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_126.png)

以下是内容不垂直对账号数据的负面影响：

![](img/a11b2ef16dc76b2c2019a8ce5c072693_128.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_130.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_132.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_134.png)

*   **推送人群错位**：内容被推送给不感兴趣的历史粉丝，导致基础数据变差。
*   **账号定位模糊**：平台无法将你识别为某个领域的“专家”，从而降低推送权重。
*   **粉丝价值降低**：用户因某一类内容关注你，却无法预期你未来发布的内容，关注价值下降。

![](img/a11b2ef16dc76b2c2019a8ce5c072693_136.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_138.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_140.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_142.png)

因此，垂直定位的根本意义在于：**将自己包装成某个特定领域的专家**，让平台能清晰地将你的账号与精准的用户群体进行匹配。

![](img/a11b2ef16dc76b2c2019a8ce5c072693_144.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_146.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_148.png)

## 关注行为的特殊意义

![](img/a11b2ef16dc76b2c2019a8ce5c072693_150.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_152.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_154.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_156.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_158.png)

在用户互动行为中，“关注”需要被单独强调。关注不是一个简单的互动，它代表了用户对账号**长期价值**的认可。

![](img/a11b2ef16dc76b2c2019a8ce5c072693_160.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_162.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_164.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_166.png)

用户关注一个账号，是因为期待获得持续、稳定的特定价值。例如，关注一个母婴专家，是为了持续获得育儿知识或产品推荐。如果一个账号内容杂乱，用户就无法确认其未来能否提供预期价值，从而降低了关注的意愿。这进一步印证了垂直定位对于积累高价值粉丝的重要性。

本节课中我们一起学习了：
1.  **用户标签**是平台通过分析用户的点击率、完播率和互动行为，为其建立的动态兴趣画像，用于实现精准内容推荐。
2.  **创作者标签**是平台通过分析账号内容的整体数据表现，为其确定的领域身份。内容垂直是获得精准标签的关键。
3.  **垂直定位**的核心目的是将自己打造成某个领域的“专家”，以便平台能将你的内容高效地分发给最感兴趣的用户，避免因内容混杂导致的数据内耗和粉丝价值流失。

![](img/a11b2ef16dc76b2c2019a8ce5c072693_186.png)

![](img/a11b2ef16dc76b2c2019a8ce5c072693_188.png)
