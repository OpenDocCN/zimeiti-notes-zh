# 抖音运营教程：P14：7.视频剪辑实操（下）

## 概述
在本节课中，我们将继续学习视频剪辑的实操部分，重点讲解如何为视频片段添加转场效果，以及如何为视频添加和调整字幕。这些步骤对于提升视频的观看流畅度和专业性至关重要。

## 添加视频转场
接下来，我们来看看如何为视频添加转场。我们的视频通常由多个约10秒的片段组成，并非一个完整的原始视频。

![](img/a5dcb984b2f6c7cbbe358ac0403d0c1c_5.png)

这些视频片段之间需要有衔接，因此必须添加转场。转场的作用是让用户明确感知到不同片段内容的切换。

![](img/a5dcb984b2f6c7cbbe358ac0403d0c1c_7.png)

如果不添加转场，当用户看到下一个片段内容突然变化时，可能会感到困惑，甚至退出视频查看详情。用户一旦退出，视频的**完播率**就会直线下降。**完播率**直接影响视频的推荐权重和最终收益。

![](img/a5dcb984b2f6c7cbbe358ac0403d0c1c_9.png)

![](img/a5dcb984b2f6c7cbbe358ac0403d0c1c_11.png)

以下是添加转场的具体步骤：

![](img/a5dcb984b2f6c7cbbe358ac0403d0c1c_13.png)

1.  在视频轨道右侧，找到一个加号按钮。这个按钮功能很多，点击它。
2.  点击后，选择“素材库”选项。
3.  在素材库的“热门”分类中，可以找到大量转场素材，例如各种动态效果。目前很多视频使用的转场都来源于此，可以放心使用。
4.  选择转场时，注意时长。避免选择过长的转场（如16秒），因为观众通常没有耐心观看不重要的内容。尽量选择**3~5秒**的转场，**3秒**左右最为合适。
5.  选择好转场素材后，点击添加。

![](img/a5dcb984b2f6c7cbbe358ac0403d0c1c_15.png)

## 统一转场比例
转场视频添加完成后，必须确保其比例与主视频保持一致，即**16:9**。

![](img/a5dcb984b2f6c7cbbe358ac0403d0c1c_17.png)

添加的转场素材下方，通常会有一个“画布模糊”的选项。点击该选项，并选择第一个“清晰”的画面。这样，转场的背景就会自动适配为16:9的横屏比例。

设置完成后，同样点击右下角的对勾进行确认。

![](img/a5dcb984b2f6c7cbbe358ac0403d0c1c_19.png)

## 为视频添加字幕
视频和转场都处理完毕后，下一步是为视频添加文案，也就是字幕。

![](img/a5dcb984b2f6c7cbbe358ac0403d0c1c_21.png)

以下是添加字幕的步骤：

1.  找到手机剪辑界面左下角的“文字”选项，点击进入。
2.  选择“新建文本”右侧的“识别字幕”功能。
3.  点击“开始匹配”，软件会自动识别视频或音频中的人物对话，并生成对应的字幕文本。
4.  字幕生成后，可以进行微调。例如，可以拖动字幕文本框右下角，将字幕放大。
5.  将放大后的字幕文本框，拖动到视频画面最下方的位置，这样观看体验最舒适。调整好第一句字幕后，后续字幕会自动应用相同的样式和位置，非常方便。

![](img/a5dcb984b2f6c7cbbe358ac0403d0c1c_23.png)

## 核心要点总结
本节课中我们一起学习了视频剪辑后半部分的核心操作。

![](img/a5dcb984b2f6c7cbbe358ac0403d0c1c_25.png)

1.  **添加转场**：在视频片段间添加**3~5秒**的转场，以提升观看流畅度，保障**完播率**。
2.  **统一比例**：确保所有素材（包括转场）的比例均为**16:9**，并使用“画布模糊”功能进行调整。
3.  **添加字幕**：使用“识别字幕”功能自动生成字幕，并调整其大小和位置至视频底部。

![](img/a5dcb984b2f6c7cbbe358ac0403d0c1c_27.png)

制作视频时，务必严格按照上述步骤和比例要求执行。在选择素材时，也应避免使用过长的视频，以免在平台审核时遇到问题。只要按部就班地操作，视频剪辑并不困难。
