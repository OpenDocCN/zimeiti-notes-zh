### 第一步：调整画面构图与比例

![](img/733216888b641bfd0eb50d0f972f1ce8_21.png)

首先检查视频的构图和比例。如果画面方向或比例不正确，需要进行调整。

![](img/733216888b641bfd0eb50d0f972f1ce8_23.png)

1.  点击时间轴上的视频片段。
2.  在底部工具栏中向右滑动，找到并点击 **`编辑`** 按钮（图标通常为方块）。
3.  在编辑菜单中，点击 **`旋转`** 功能，将画面调整到正确的方向。

![](img/733216888b641bfd0eb50d0f972f1ce8_25.png)

![](img/733216888b641bfd0eb50d0f972f1ce8_9.png)

![](img/733216888b641bfd0eb50d0f972f1ce8_27.png)

4.  如需裁剪画面比例，可点击 **`裁剪`** 功能。
5.  在裁剪界面中，可以选择预设比例，如 **`1:1`**（正方形）、**`4:3`**、**`16:9`**（横屏）或 **`9:16`**（竖屏，抖音常用）。选择合适比例并确认。

![](img/733216888b641bfd0eb50d0f972f1ce8_29.png)

![](img/733216888b641bfd0eb50d0f972f1ce8_11.png)
![](img/733216888b641bfd0eb50d0f972f1ce8_13.png)

![](img/733216888b641bfd0eb50d0f972f1ce8_31.png)

调整完成后，点击返回箭头回到主编辑界面。

![](img/733216888b641bfd0eb50d0f972f1ce8_33.png)

![](img/733216888b641bfd0eb50d0f972f1ce8_35.png)

### 第二步：剪辑视频内容

![](img/733216888b641bfd0eb50d0f972f1ce8_37.png)

接下来，我们需要剪掉视频中多余、无效的部分，例如开头沉默、结尾无关动作等。

![](img/733216888b641bfd0eb50d0f972f1ce8_39.png)

核心操作是 **`选中` -> `分割` -> `删除`**。

![](img/733216888b641bfd0eb50d0f972f1ce8_41.png)

1.  播放视频，找到需要剪掉的起始位置，将时间线（白色竖线）移动至该处。
2.  点击选中视频片段（片段会显示白框）。
3.  点击底部工具栏中的 **`分割`** 按钮。视频将在时间线位置被切成两段。
4.  点击选中需要删除的那一段视频。
5.  点击底部工具栏中的 **`删除`** 按钮。

![](img/733216888b641bfd0eb50d0f972f1ce8_43.png)

![](img/733216888b641bfd0eb50d0f972f1ce8_15.png)

![](img/733216888b641bfd0eb50d0f972f1ce8_45.png)

重复以上步骤，剪掉所有不需要的片段。例如，结尾处无关的动作也可以同样操作删除。

![](img/733216888b641bfd0eb50d0f972f1ce8_47.png)

![](img/733216888b641bfd0eb50d0f972f1ce8_17.png)

![](img/733216888b641bfd0eb50d0f972f1ce8_49.png)

请仔细播放检查，确保剪辑后的视频流畅连贯。

![](img/733216888b641bfd0eb50d0f972f1ce8_51.png)

### 第三步：添加与编辑背景音乐

![](img/733216888b641bfd0eb50d0f972f1ce8_53.png)

视频内容剪辑好后，需要添加背景音乐。

![](img/733216888b641bfd0eb50d0f972f1ce8_55.png)

1.  点击底部工具栏中的 **`音频`** 按钮。
2.  添加音乐有两种常用方式：
    *   **提取音乐**：如果你已经用其他软件（如淘金阁）为视频配好了音，可以点击 **`提取音乐`**，直接导入视频文件中的音频。
        ![](img/733216888b641bfd0eb50d0f972f1ce8_19.png)
    *   **添加音乐**：点击 **`音乐`**，进入剪映自带的音乐库。你可以根据分类（如美食、美妆、情感）选择，或直接搜索歌曲。点击歌曲旁边的播放按钮可以试听。
        ![](img/733216888b641bfd0eb50d0f972f1ce8_21.png)
        ![](img/733216888b641bfd0f972f1ce8_23.png)
        ![](img/733216888b641bfd0eb50d0f972f1ce8_25.png)
3.  选择好音乐后，点击 **`使用`**，音乐就会被添加到视频轨道下方。
    ![](img/733216888b641bfd0eb50d0f972f1ce8_27.png)

**剪辑与调整音乐：**
*   **剪辑长度**：如果音乐过长，同样使用 **`选中` -> `分割` -> `删除`** 的方法，将多余部分剪掉。
    ![](img/733216888b641bfd0eb50d0f972f1ce8_29.png)
*   **调整音量**：点击选中音频轨道，在底部工具栏点击 **`音量`**，拖动滑块即可调整音乐声音大小，避免盖过人声。
    ![](img/733216888b641bfd0eb50d0f972f1ce8_31.png)
    ![](img/733216888b641bfd0eb50d0f972f1ce8_33.png)
    ![](img/733216888b641bfd0eb50d0f972f1ce8_35.png)

![](img/733216888b641bfd0eb50d0f972f1ce8_57.png)

### 第四步：添加字幕

![](img/733216888b641bfd0eb50d0f972f1ce8_61.png)

为视频添加字幕可以提升观看体验。剪映提供了自动识别字幕的功能，非常高效。

1.  点击底部工具栏中的 **`文本`** 按钮。
2.  点击 **`识别字幕`**。
    ![](img/733216888b641bfd0eb50d0f972f1ce8_37.png)
3.  在弹出的菜单中，选择 **`仅视频`**（如果视频包含人声），这样可以避免将背景音乐的歌词也识别出来。然后点击 **`开始识别`**。
    ![](img/733216888b641bfd0eb50d0f972f1ce8_39.png)
4.  系统会自动识别视频中的语音并生成字幕，显示在时间轴上。
    ![](img/733216888b641bfd0eb50d0f972f1ce8_41.png)
    ![](img/733216888b641bfd0eb50d0f972f1ce8_43.png)

![](img/733216888b641bfd0eb50d0f972f1ce8_63.png)

**编辑与美化字幕：**
1.  检查字幕是否有错别字。双击字幕条，即可进入编辑框修改文本。
    ![](img/733216888b641bfd0eb50d0f972f1ce8_45.png)
2.  点击 **`样式`**，可以修改字幕的字体、颜色、大小等。例如，为了使字幕更醒目，可以选择与画面主色调搭配的颜色。
    ![](img/733216888b641bfd0eb50d0f972f1ce8_47.png)

![](img/733216888b641bfd0eb50d0f972f1ce8_65.png)

至此，视频的**画面、音频、字幕**都已处理完毕。
![](img/733216888b641bfd0eb50d0f972f1ce8_49.png)

![](img/733216888b641bfd0eb50d0f972f1ce8_67.png)

### 第五步：导出高清视频

![](img/733216888b641bfd0eb50d0f972f1ce8_69.png)

在导出前，需要进行关键设置以保证视频清晰度。

![](img/733216888b641bfd0eb50d0f972f1ce8_71.png)

1.  点击右上角的 **`导出`** 按钮。
2.  在导出设置界面，调整参数：
    *   **分辨率**：建议拉至最高（如1080p或2K、4K）。分辨率越高，视频越清晰。
    *   **帧率**：设置为 **`30`** 帧/秒。这是抖音平台最适配的帧率，能保证流畅度且文件大小适中。
3.  设置完成后，点击 **`导出`** 按钮。
    ![](img/733216888b641bfd0eb50d0f972f1ce8_51.png)
    ![](img/733216888b641bfd0eb50d0f972f1ce8_53.png)

**重要提示**：导出过程中请勿锁屏或切换手机应用程序，耐心等待导出完成，以免中断导致导出失败。
![](img/733216888b641bfd0eb50d0f972f1ce8_55.png)

![](img/733216888b641bfd0eb50d0f972f1ce8_73.png)

导出完成后，可以直接通过剪映的分享按钮，将视频发布到抖音。
![](img/733216888b641bfd0eb50d0f972f1ce8_57.png)

![](img/733216888b641bfd0eb50d0f972f1ce8_75.png)
