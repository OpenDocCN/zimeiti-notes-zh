# 剪映手机版教学：P54：蒙版原理讲解与三分屏片头制作 🎬

在本节课中，我们将学习剪映中“蒙版”功能的核心原理，并运用它制作一个经典的三分屏片头效果。通过本教程，你将理解蒙版如何限制画面显示区域，并掌握组合多个视频与动画来创作动态开场的方法。

## 蒙版原理讲解 🧩

![](img/4d112e948b3cf8091c797b5cd491b564_1.png)

上一节我们开启了蒙版篇章的学习。本节中，我们来看看蒙版究竟是什么。

在剪映中，蒙版是一个图形工具，其作用是将视频画面的显示范围限制在该图形内部，图形之外的区域则不显示。

![](img/4d112e948b3cf8091c797b5cd491b564_3.png)

![](img/4d112e948b3cf8091c797b5cd491b564_4.png)

以下是添加蒙版的基本步骤：
1.  点击「开始创作」，导入素材视频。
2.  在时间轴上选中一个视频片段。
3.  在底部工具栏中找到并点击「蒙版」选项。

点击后，你会看到多种蒙版形状可供选择：
*   线性蒙版
*   镜面蒙版
*   圆形蒙版
*   矩形蒙版
*   爱心蒙版
*   星形蒙版

![](img/4d112e948b3cf8091c797b5cd491b564_1.png)

通过演示可以清楚地看到，蒙版就是添加一个图形，将视频画面限制在这个图形内部，外部则不显示。

## 蒙版的控制参数 ⚙️

了解了蒙版的基本概念后，我们进一步学习如何调整它。以矩形蒙版为例，选中后可以看到其边缘有多个控制点。

![](img/4d112e948b3cf8091c797b5cd491b564_3.png)
![](img/4d112e948b3cf8091c797b5cd491b564_4.png)

这些控制点功能如下：
*   **左右箭头**：拖动可水平缩放蒙版大小。
*   **上下箭头**：拖动可垂直缩放蒙版大小。
*   **圆角图标**：拖动可调整矩形边角的圆润程度。拉大变圆角，拉小变尖角。
*   **羽化图标**：拖动可调整边缘羽化效果，实现从不透明到透明的渐变过渡。

蒙版添加后，你可以用手指在屏幕上拖动来移动它的位置。也可以用双指进行缩放和旋转操作。旋转时，屏幕上会显示具体的旋转度数。

## 三分屏片头制作实践 ✂️

掌握了蒙版的调整方法，现在我们将理论应用于实践，制作一个三分屏片头。本节课我们将使用“镜面蒙版”。

**第一步：准备与排列素材**
1.  导入三段颜色对比鲜明的视频素材（例如：沙漠、森林、冰山）。
2.  将主轨道上的视频（如森林）添加一个「镜面蒙版」。
3.  旋转蒙版至 **35度**，并调整其大小和位置。
4.  将另外两段视频（沙漠、冰山）通过「切画中画」功能，切入到画中画轨道。
5.  调整三段视频的时长，使其保持一致。

**第二步：为画中画添加蒙版**
现在，我们需要让三个画面平行排列。
1.  选中画中画轨道的冰山视频，为其添加「镜面蒙版」。
2.  同样将蒙版旋转 **35度**，确保与主轨道视频的蒙版边缘平行。
3.  在蒙版界面内拖动，可以选取冰山视频中想要展示的部分；在蒙版界面外拖动，则是移动整个视频。将调整好的冰山画面放置在森林画面的左上方。
4.  对沙漠视频重复上述操作：添加镜面蒙版、旋转35度，并将其放置在森林画面的右下方。

至此，三个画面的静态拼贴已完成。接下来为其添加动画效果。

**第三步：添加入场动画**
动画能让片头更生动。以下是添加动画的步骤：
1.  **主轨道视频（森林）**：为其添加「入场动画」-「动感缩小」，并将动画时长拉长至约4秒。
2.  **画中画视频（冰山）**：添加「入场动画」-「向右滑动」，时长同样设置为4秒左右。
3.  **画中画视频（沙漠）**：添加「入场动画」-「向左滑动」，时长与其他动画保持一致。

**第四步：添加音乐与文字**
1.  点击「音频」，添加一首符合意境的背景音乐。
2.  点击「贴纸」，在搜索框中输入关键词（如“山海”），选择合适的文字贴纸添加到画面中。
3.  将文字贴纸调整至合适大小，并放置在画面中央。让其出现的时间点与三分屏动画完成的时间点对齐。

**第五步：导出与后续处理**
片头制作完成后，建议先单独导出。
1.  点击右上角「导出」，将制作好的片头保存到手机。
2.  之后，你可以重新创作，将导出的片头作为新视频的开场，再衔接后续的主视频内容。这样做可以避免在复杂时间线上直接添加转场时可能遇到的问题。

## 课程总结 📝

本节课中，我们一起学习了剪映蒙版的核心原理与操作。我们了解到蒙版是一个图形遮罩，用于限制视频的显示区域。通过动手实践，我们掌握了使用镜面蒙版旋转、排列多个视频，并配合入场动画来制作一个动态三分屏片头的完整流程。关键点在于保持各蒙版旋转角度一致，并合理安排动画的时序与方向。请多加练习，熟练掌握蒙版这一强大工具的使用。
