# 小红书运营教程：P7：第5节：小红书笔记不被收录的原因有哪些？深度讲解 📝

在本节课中，我们将要学习小红书笔记不被平台收录的常见原因。理解这些原因有助于我们规避问题，确保内容能够正常展示和传播。

上一节我们介绍了笔记发布的基本流程，本节中我们来看看哪些因素会导致笔记无法被收录。

## 笔记不被收录的五大原因

以下是导致小红书笔记不被收录的五个主要原因。

1.  **违规推广**
    笔记内容包含硬性广告，例如直接提及品牌名称或进行明显的商品推销。平台不鼓励未经处理的直接商业推广。

2.  **敏感词违规**
    笔记文本或账号历史中包含平台禁止的敏感词汇。即使笔记暂时被收录，系统在后续检测中若发现违规词，也会将笔记剔除或限流。核心检测逻辑可理解为：
    ```
    if 笔记内容 contains 敏感词列表中的词汇:
        执行 不收录 或 限流操作
    ```

3.  **内容违规（图片与文字）**
    此原因涵盖图片和文字两方面的违规。
    *   **图片违规**：避免使用带有其他平台水印或二维码的图片，此类引流方式已被平台禁止。
    *   **文字违规**：严禁出现不文明词汇、诱导性内容以及政治敏感、破坏民族团结等法律法规明令禁止的词汇及其谐音。我们运营网络项目的原则是合法合规。

4.  **发布频率异常**
    短时间内发送笔记的频率过高、过快，可能导致系统判定账号行为异常，疑似机器人操作，从而不收录笔记。

5.  **设备被拉黑**
    如果当前使用的移动设备曾有过严重违规记录，即使更换了新账号，平台也可能通过读取设备的硬件参数（如IMEI码）识别并限制该设备发布的内容。这可以理解为设备层面的封禁。

本节课中我们一起学习了导致小红书笔记不被收录的五类核心原因，包括违规推广、敏感词、内容违规、发布频率异常以及设备问题。理解并避免这些问题，是保证笔记正常曝光的基础。
