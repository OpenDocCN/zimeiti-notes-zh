# 抖音私信卡片跳转微信教程：P1：配置私信自动回复卡片

在本节课中，我们将学习如何配置抖音企业号的私信自动回复卡片，实现用户点击卡片后直接跳转至微信添加好友的功能。整个过程基于官方接口，安全可靠。

## 功能与资质要求

上一段我们了解了课程目标，接下来看看实现此功能需要满足哪些前提条件。

以下是配置私信自动回复卡片所需的两项核心资质：
1.  抖音账号需为**抖音企业号蓝V**或**抖音普通企业号半蓝**。
2.  需要使用指定的**获客平台助手**。该平台对接了抖音企业号的官方私信自动回复接口。

## 配置步骤详解

了解了基本要求后，本节中我们来看看具体的配置流程。整个过程在获客平台的后台完成。

![](img/407b01b49beb1cb3e147e11c1811a771_1.png)

以下是配置私信卡片的详细步骤：
1.  **登录后台**：首先，登录您的获客平台管理后台。
2.  **新增账号**：点击右上角的“新增”按钮，添加您希望用户跳转的目标微信账号。
3.  **进入配置**：账号添加成功后，在对应账号后找到“配置卡片”选项并点击进入。
4.  **设置卡片内容**：在配置页面中，您可以详细设置卡片的各项属性。

    ![](img/407b01b49beb1cb3e147e11c1811a771_1.png)

    ![](img/407b01b49beb1cb3e147e11c1811a771_2.png)

![](img/407b01b49beb1cb3e147e11c1811a771_2.png)

5.  **配置回复规则**：除了外观，您还可以设置功能规则，例如：
    *   是否启用“自动撤回”功能。
    *   触发方式：可选择“用户进入私信自动回复”或“用户发送消息后再回复卡片”。
6.  **保存并启用**：所有设置完成后，点击“确定”保存。最后，务必打开该卡片的“开关”以激活功能。
