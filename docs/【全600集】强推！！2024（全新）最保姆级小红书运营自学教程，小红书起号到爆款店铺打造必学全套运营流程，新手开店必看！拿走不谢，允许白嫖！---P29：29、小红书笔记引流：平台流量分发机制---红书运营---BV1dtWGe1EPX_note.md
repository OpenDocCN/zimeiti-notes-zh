# 小红书运营教程：P29：小红书笔记引流：平台流量分发机制 🚀

![](img/856453a9e21433bc6dbe742e43de198e_1.png)

在本节课中，我们将要学习小红书平台的核心流量分发机制。了解这套机制，是提升笔记曝光、获得更高排名的关键。我们将拆解其算法逻辑，并解释如何通过优化内容来提升笔记的流量表现。

## 概述：流量分发的核心逻辑

小红书平台通过一套综合评分系统来决定笔记的展示排名。这套系统的核心是考察用户与笔记的交互行为，并根据这些行为计算出一个综合分数，以此决定笔记在信息流中的位置。

## 基础评分系统：S1CCES

上一节我们介绍了流量分发的重要性，本节中我们来看看其核心的计算系统。小红书流量分发机制可以简称为 **S1CCES**，即点赞、收藏、转发、评论和关注。它们各自拥有独立的评分权重。

以下是各项交互行为的具体分值：
*   **点赞**：1分
*   **收藏**：1分
*   **转发**：4分
*   **评论**：10分
*   **关注**：8分

当一个用户完成从点赞到关注的全部行为时，该笔记将获得 `1 + 1 + 4 + 10 + 8 = 24` 分。系统会综合计算笔记获得的平均分数，并以此作为排名的基础依据。

## 展示与排名机制

了解了基础评分后，我们需要明白笔记是如何被展示和排序的。小红书的信息流页面与抖音不同，它一次展示20篇笔记，刷新后才会加载新的20篇。

在这个展示列表中，排名从第1名到第20名。用户向下滑动，系统会加载第21到40名，以此类推。我们的目标就是让笔记进入前20名，并尽可能获得靠前的位置。排名的高低，正是由上文提到的综合评分（CES）来决定的。

## 流量分发的完整流程

那么，一篇笔记从发布到获得流量，具体会经历哪些步骤呢？以下是其完整的流程：

1.  **内容打标与初次曝光**：笔记发布后，系统会根据笔记内容（图片、文案）和账号预设的标签，为其打上多个分类标签。随后，笔记会获得一个基础的曝光量（例如1000次），并被推送给可能对这些标签感兴趣的用户。

2.  **核心指标考核**：在基础曝光阶段，系统会重点考核两个核心指标：
    *   **点击率**：用户是否被你的封面和标题吸引并点击进入。这是所有后续互动的前提。
    *   **CES互动评分**：用户点击进入后，是否产生了点赞、收藏、评论、转发、关注等行为。这些行为将累积成CES分数。

3.  **阶梯式推流**：平台的流量推送并非一次性完成，而是分波次进行的。例如，首次推送400曝光，第二次300曝光，第三次500曝光。
    *   如果笔记在某一波曝光中，点击率和互动评分高于同期推送的其他笔记，系统就会在下一波推送中给予更多流量扶持。
    *   反之，如果数据不达标，后续的推送流量可能会减少甚至停止。一篇笔记的推广周期通常约为三天。

## 其他重要指标

除了点击率和CES评分，系统还会关注以下指标，它们共同反映了笔记的内容质量和用户粘性：

*   **互动率**：笔记下评论、@好友等用户间或用户与作者间互动行为的频率。高互动率直接反映了内容的吸引力，即使是粉丝数不多的账号，优质互动也能带来更多推荐。
*   **完播率**（针对视频笔记）：用户完整观看视频的比例。完播率高说明内容能留住用户。
*   **互动值**：一个综合性的衡量指标，涵盖了用户与笔记交互的深度和广度。

## 总结与要点回顾

本节课中我们一起学习了小红书平台的流量分发核心机制。总结来说，无论算法如何微调，其核心始终是**考察笔记带来的粉丝交互行为**。

**核心公式可以概括为：**
**高排名 = 高点击率 + 高CES互动评分 + 良好的互动率/完播率**

你的笔记能排到第几名，完全取决于内容能否激发用户的点击和互动欲望。对于原创内容，即使是从零开始，通常也能获得150-200次的基础曝光。理解并优化这个流程中的每一个环节，是提升小红书笔记流量的关键。
