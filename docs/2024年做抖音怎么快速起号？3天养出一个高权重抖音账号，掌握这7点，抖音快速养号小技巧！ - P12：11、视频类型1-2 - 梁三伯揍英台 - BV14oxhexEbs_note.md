# 抖音运营实战：P12：11、视频类型1-2 - 人设与口播

在本节课中，我们将要学习抖音账号运营中两个至关重要的概念：**人设**的建立与区分，以及**口播**这种视频形式的具体应用。理解并掌握这些知识，是制作吸引人内容、实现账号增长的基础。

上一节我们解决了文案问题，本节中我们来看看内容创作的下一个关键环节。从涨粉到变现并非一条直线，中间有一个核心环节，那就是建立你的**人设**。建立人设的目的并非为了炫耀，而是为了在众多同类创作者中脱颖而出。例如，有十个人都做装修内容，如果其中一位创作者建立了幽默搞笑的人设，观众就更容易记住他，从而忽略其他九位。当观众喜欢你这个人时，他们自然就成为你的粉丝。

这就是人设的作用。接下来，我们来详细讲解这个知识点。

## 人设的两种类型：强与弱

人设主要分为两种类型：**强人设**和**弱人设**。学会区分它们，你就能快速判断任何视频的人设类型。

首先，**强人设**是指以创作者**个人**为切入点的内容。它必须与创作者自身密切相关。

以下是强人设常见的切入点：

![](img/935ad1aa6e7db90ba43ed27ee3110b44_1.png)

*   **个人职业与经验**：例如分享装修、办公、房产、汽车、餐饮、婚介、法律咨询、电商等领域的行业经验。
*   **个人兴趣与生活**：例如分享美食、旅行、手工、宠物、星座占卜、美妆护肤、服装穿搭、游戏、摄影等爱好。

强人设的内容核心是分享创作者自己的工作、生活和爱好，并且大部分会采用**真人出镜**的形式，因为内容与本人直接相关。

与之相对的是**弱人设**。弱人设的内容与创作者**个人**关系不大，主要目的是为了运营好账号本身。

以下是弱人设的例子：

*   **情感语录账号**：发布配乐的情感文字图片或视频，无真人出镜或配音。
*   **影视解说账号**：虽然有人声解说电影，但焦点是电影内容，而非解说者本人。

弱人设的核心是内容本身，创作者个人色彩较淡。

## 内容创作的核心：从自身出发

只要内容与你个人相关，都可以作为视频素材。我们接下来逐一分析几种常见的内容形式。首先，我们从“讲故事”开始。

### 形式一：讲故事（口播）

讲故事，就是分享客户故事、从业经历或人生感悟。故事本身不要求完全真实，可以从文案平台寻找有趣的故事进行分享。讲故事最主要的形式就是**口播**。

口播是指面对镜头进行讲述的视频形式。它有很多分类：

*   养生口播
*   情感口播
*   育儿口播
*   知识类口播（如房产、汽车、金融财经、装修等）

口播视频主要实现两种效果：
1.  **讲故事类口播**：让观众觉得内容有趣、有吸引力。
2.  **教知识类口播**：让观众在几十秒内获得“收获感”，觉得学到了东西。

#### 关于口播的常见问题

以下是关于口播的几个关键解答：

*   **口才不好能做口播吗？**  
    可以。口播是**提前准备好稿子**再录制，并非临场发挥。很多口播者的流畅表达都源于充分准备。
*   **普通话不好能做口播吗？**  
    可以。如果你的目标客户是同城或有地域属性，使用**方言**可能效果更好，能增加亲切感和趣味性。同时，可以配上字幕确保观众理解。
*   **口播怎么拍？**  
    非常简单。用手机架好，对着摄像头念出准备好的文案即可。这是一种易于起步的创作形式。

本节课中我们一起学习了如何通过建立**强人设**在竞争中凸显个人特色，并深入了解了**口播**这种核心视频形式的特点与制作要点。记住，从你自身出发的内容最具独特性，而口播是将你专业知识或个人故事有效传递出去的高效方式。
