# 自媒体运营教程：P7：6.视频剪辑实操（下） 🎬

![](img/b4f9b7bcd5a1dcc8dfc28657b18fab79_0.png)

在本节课中，我们将学习如何调整视频比例、裁剪素材、添加背景、设置转场以及识别字幕，从而将一个不符合标准的素材，处理成符合平台要求的16:9横屏视频。这些操作是视频剪辑的基础，能让你的素材应用范围更广，视频观感更专业。

上一节我们介绍了视频剪辑的基础操作，本节中我们来看看如何处理比例不匹配的素材，并为其添加必要的过渡元素。

## 调整视频比例与裁剪素材

在寻找素材的过程中，你找到的素材不可能每一个都是标准的16:9横屏比例。就像示例中的视频，它本身并不是16:9横屏。如果你仍然想使用这个素材，可以按照以下步骤操作。

![](img/b4f9b7bcd5a1dcc8dfc28657b18fab79_2.png)

首先，在剪辑界面右下角找到“比例”选项。点击后，选择“16:9”这一横屏比例。选择完成后，观察视频画面是否自动适配。如果视频画面没有变化（即视频主体在画面中静止不动），则需要进一步手动调整。

这时，我们需要裁剪掉视频自带的元素（如黑边或原有字幕），以放大核心画面。操作步骤如下：

![](img/b4f9b7bcd5a1dcc8dfc28657b18fab79_4.png)

1.  点击视频轨道左侧的“剪辑”选项。
2.  在弹出的菜单中选择“编辑”，然后选择“裁剪”功能。
3.  在裁剪界面中，仔细调整框选范围，确保移除所有不需要的黑色边框和原有字幕，使主体画面最大化。
4.  裁剪完成后，点击右下角的对勾确认。

完成裁剪后，视频主体画面会变得更大、更清晰，但它可能仍然不是一个完整的16:9横屏。

## 添加背景填充画面

为了让视频充满16:9的画布，我们需要使用背景填充功能。以下是具体步骤：

1.  在编辑界面找到右下角的“背景”选项。
2.  点击进入后，选择“画布模糊”功能。
3.  在模糊样式中，选择第一个（最清晰的）模糊效果作为背景。

经过这一步操作，视频就变成了一个完整的16:9横屏画面。背景会自动填充视频主体之外的区域，使画面看起来完整且专业。

## 为视频添加转场

![](img/b4f9b7bcd5a1dcc8dfc28657b18fab79_6.png)

![](img/b4f9b7bcd5a1dcc8dfc28657b18fab79_8.png)

转场是连接不同视频片段的过渡效果，例如在两个不同的场景或内容之间插入一个短暂的搞笑片段。添加转场的目的，是明确告知用户一个片段已经结束，另一个全新的片段即将开始。这能提升观看的流畅度，避免用户因画面突然切换而感到困惑甚至退出，从而保护视频的完播率。完播率是影响视频推荐和收益的重要因素。

![](img/b4f9b7bcd5a1dcc8dfc28657b18fab79_10.png)

以下是添加转场的步骤：

1.  在视频轨道上，点击片段之间的“+”号。
2.  在弹出的“素材库”中，选择“热门”分类，里面提供了大量常用的转场素材。
3.  在选择转场时，**务必注意时长**。应避免使用过长（如6秒、17秒）的转场，观众没有耐心观看冗长的过渡。**保持转场时长在3到5秒最为合适**。
4.  选择一个约3秒时长的转场素材，点击添加。

添加转场后，同样需要确保其比例与主视频一致（16:9）。选中转场片段，在编辑选项中为其设置“画布模糊”背景，选择第一个清晰的模糊效果即可。

![](img/b4f9b7bcd5a1dcc8dfc28657b18fab79_12.png)

## 识别与添加字幕

为视频添加字幕可以极大提升观看体验。剪映提供了便捷的自动识别字幕功能。操作如下：

1.  点击剪辑界面左下角的“文字”选项。
2.  点击“新建文本”右侧的“识别字幕”按钮。
3.  点击“开始匹配”，软件会自动识别视频中的人声并生成字幕轨道。

字幕识别完成后，通常只需要调整其中一个字幕块的样式，后续字幕会自动应用相同样式。

1.  选中一个字幕块，拖动右下角的方框将其放大到合适尺寸。
2.  将这个字幕块拖动到视频画面底部合适的位置。

![](img/b4f9b7bcd5a1dcc8dfc28657b18fab79_14.png)

调整好一个字幕后，整个视频的所有字幕都会自动统一为这个样式和位置，无需逐个调整。

![](img/b4f9b7bcd5a1dcc8dfc28657b18fab79_16.png)
