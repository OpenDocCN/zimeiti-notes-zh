# 亚马逊欧洲站GPSR合规指南：P1：GPSR核心要求与紧急应对

在本节课中，我们将学习亚马逊欧洲站即将强制执行的《通用产品安全法规》（GPSR）的核心要求。该法规涉及几乎所有品类，卖家需在截止日期前完成合规调整，否则商品将被下架。

## 📅 法规生效时间与影响范围

GPSR法规的合规截止日期为**12月13日**。在此日期之前，所有在欧洲站销售的商品链接必须符合GPSR要求，否则将面临被平台强制下架的风险。

该法规覆盖面极广，基本上**除食品类目外**的所有产品都需要进行合规操作。

## 📝 GPSR合规两大核心操作

上一节我们了解了法规的紧迫性与范围，本节中我们来看看具体需要完成的两项核心操作。以下是卖家必须完成的两个关键步骤：

1.  **信息上传**：在亚马逊卖家后台，为每一条商品链接上传指定的“负责人”和“制造商”信息。
2.  **标签张贴**：在商品实物或其包装上，张贴包含负责人信息的合规标签。

未能完成以上操作的商品将无法继续销售。

## ⚠️ 特别提醒与资源获取

对于尚未注册欧洲负责人（欧代）的卖家，当前首要任务是立即着手办理注册，时间已经非常紧迫。

如果对GPSR的具体细节、标签格式或后台操作步骤存在疑问，可以参考一份详细的指导文档。该文档内容涵盖：

*   合规标签的具体要求。
*   操作过程中遇到的常见问题解答。
*   后台信息上传的详细步骤与注意事项。
