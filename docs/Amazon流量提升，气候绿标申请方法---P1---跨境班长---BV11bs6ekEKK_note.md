# 亚马逊流量提升：P1：气候友好绿标申请教程

在本节课中，我们将学习如何为亚马逊商品申请“气候友好承诺”绿标。这个认证能显著提升商品在亚马逊平台的曝光率和转化率。

---

## 🌍 什么是气候友好承诺绿标？

这种环保绿标在亚马逊的页面中会有非常大量的流量扶持，涵盖了22个的流量入口的支持。像大疆、安克、JBL这些头部的产品页面都有这种碳中和的认证。这是亚马逊在2020年联合政府和非营利性组织推出的气候友好计划。

对于获得这个认证的商品，就会有这种气候友好的标识，呈现在你的产品推荐页，还有专属的BSR节点，能够获得大量的额外流量，对于转化率也会大大的提升。很多海外的消费者其实还是非常认可这类认证的。

因为他们购买一个商品，还能对社会和自然环境做出贡献。这点在海外的消费者认知教育中还是非常普遍的。

---

## 🔍 如何获得认证？

获得这个认证的方案有三种。

上一节我们介绍了绿标的价值，本节中我们来看看具体的申请路径。以下是三种主要的认证方式：

1.  **亚马逊官方认证**
    大家可以直接在后台直接申请。需要满足的需求会比较多，流程也相对比较复杂。

2.  **碳中和认证**
    通过计算产品的碳排放量，去证明你的产品在其生命周期中实现了碳中和状态和认证。它就更加适用于全部品类，而且流程相对比较简单。

3.  **直接验厂**
    通过第三方的一些认证机构，去验证你的碳中和行动是否真实、是否可持续有效。这种会更加适合一些自有工厂型的卖家。它的时间周期也会比较长，而且流程会相对比较复杂。

---

## 🚀 推荐方案与流程

所以总结下来，是直接用**碳中和认证**是最简单省事儿的。它关键是快，而且适用于全部的品类。目前这些头部的大卖认证，都是由这家公司做的，中文名叫“绿洲”。

整个的流程从数据的采集，再到碳足迹的数据计算，完成认证后，两周左右的时间，你的产品就能够呈现气候认证的标识，也大概两个月左右就能完成所有的流程。

以下是基于“绿洲”公司官网信息整理的全流程认证介绍：

*   **第一步：数据采集**。向认证机构提供产品生产、运输、使用等环节的相关数据。
*   **第二步：碳足迹计算**。认证机构根据提供的数据，使用专业模型计算产品的碳排放总量。公式可简化为：`产品碳足迹 = 原材料生产排放 + 制造过程排放 + 运输排放 + 使用阶段排放 + 废弃处理排放`。
*   **第三步：碳中和与认证**。通过购买碳信用额等方式抵消计算出的碳排放，实现“碳中和”，最终获得认证证书。
*   **第四步：标识上线**。将获得的认证信息提交至亚马逊后台，通过审核后，商品页面即可展示气候友好标识。

---

本节课中我们一起学习了亚马逊气候友好承诺绿标的重要性、三种申请途径以及最高效的“碳中和认证”全流程。对于希望快速提升流量和转化的卖家而言，通过第三方机构进行碳中和认证是目前最推荐的选择。
