# 视频号运营教程：P13：7.如何给视频抽帧补帧 🎬

在本节课中，我们将学习视频剪辑中的一个重要技巧——抽帧与补帧。这个操作可以帮助我们微调视频时长，同时避免因简单掐头去尾导致的画面跳跃感，是提升视频流畅度的基础方法。

上一节我们介绍了视频剪辑中“掐头去尾”的基本操作，本节中我们来看看如何通过“抽帧补帧”进行更精细的调整。

## 核心概念与准备工作

首先，理解帧的概念是关键。视频由连续的静态图片（帧）快速播放而成。在大多数平台，视频的帧率是**30帧每秒（30 FPS）**。这意味着，一秒的视频由30张连续的画面组成。

**公式：1秒视频 = 30帧 (30 FPS)**

在进行操作前，需要将视频轨道在时间轴上**拉到最长的刻度**，以便我们能精确地对每一帧进行操作。

## 如何进行抽帧

抽帧，指的是从视频中删除某一帧或几帧画面。以下是具体步骤：

1.  **定位帧**：将时间轴放大至最大，找到你想要删除的某一帧。例如，在时间码 `00:00:00` 到 `00:00:01` 之间，有30个帧（F）。
2.  **分割帧**：在目标帧的位置（例如第20帧）进行分割，紧接着在下一帧（第21帧）的位置再次分割。
3.  **删除帧**：选中这被分割出来的、长度极短（即一帧）的片段，将其删除。

这个操作就完成了一次抽帧。通常，在整个视频中抽掉2-3帧，对视频的流畅度几乎没有影响。

## 如何进行补帧

补帧，与抽帧相反，是通过复制已有的某一帧来增加视频长度。以下是具体步骤：

1.  **选择帧**：在视频中选中一个你希望复制的极短片段（通常是一帧或几帧）。
2.  **复制帧**：使用复制功能（快捷键通常是 `Ctrl+C` / `Cmd+C`），然后粘贴（`Ctrl+V` / `Cmd+V`）到视频的其他位置。

这个复制的操作就属于补帧。补帧可以平衡因抽帧减少的时长，或单纯用于微调视频节奏。

## 操作流程与注意事项

一个完整的“抽帧补帧”流程通常遵循“抽几帧，补几帧”的原则，以保持视频原时长大致不变。

以下是操作时的关键点列表：
*   **适度原则**：整个视频抽帧和补帧总计3-5次即可，过度操作可能影响观感。
*   **选择位置**：尽量在画面动作变化不大、相对静止的片段进行抽补帧，这样更不易被察觉。
*   **检查效果**：操作完成后，务必完整播放一遍，检查视频是否依然流畅自然。

![](img/4bdb651d02e1fd1788045e459dec217f_2.png)

通过结合使用抽帧与补帧，我们可以对视频进行非常精细的时长调整，而不会像直接裁剪那样丢失完整的画面内容。

本节课中我们一起学习了视频剪辑中的抽帧与补帧技术。我们了解了帧的概念，掌握了如何通过分割删除来抽帧，以及通过复制粘贴来补帧。记住，这是一个用于微调视频、提升流畅度的辅助技巧，使用时需遵循适度原则。掌握它，能让你的视频剪辑更加精细和专业。
