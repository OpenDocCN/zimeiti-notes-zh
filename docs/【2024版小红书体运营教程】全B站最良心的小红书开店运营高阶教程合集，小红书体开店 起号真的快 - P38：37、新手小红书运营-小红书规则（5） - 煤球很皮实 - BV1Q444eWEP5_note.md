上一节我们介绍了小红书平台的整体推荐机制与流量来源。本节中，我们来看看笔记如何通过平台的“考核”获得更多推荐，即“被推的核心逻辑”。

## 被推的核心逻辑

笔记发布后，平台会给予一个初始的展示量（例如新号可能获得约1000次展示）。笔记能否获得更多流量，取决于它能否通过平台的两层核心数据考核：**打开率**与**互动率**。这个过程可以理解为笔记的“升级”或“破圈”。

### 第一层考核：打开率

当用户打开小红书App，推荐页每次会展示约4-6篇笔记，下拉刷新约出现20篇同类型笔记。你的笔记会在这批展示中接受第一轮测试。

**核心公式**：`打开率 = 笔记点击量 / 笔记展示量`

平台要求笔记的**平均打开率需达到11%以上**，才能进入下一流量池。
*   例如，若初始展示量为1000次，则需要获得至少 `1000 * 11% = 110` 次点击（小眼睛）。
*   达标后，平台可能会将展示量提升至2000-5000次。
*   若打开率低于11%，则该笔记通常只会获得初始的一波推荐，后续流量有限。

### 第二层考核：互动率

通过第一层考核后，笔记进入更大的流量池。此时，平台会考核笔记的互动数据。

**核心公式**：`互动率 = （评论+收藏+点赞+分享等互动数） / 笔记点击量`

![](img/cea384f6c19930a9e0e0e750dd72097b_9.png)

![](img/cea384f6c19930a9e0e0e750dd72097b_11.png)

平台要求笔记的**平均互动率需达到3%以上**，才有机会继续升级。
*   例如，若笔记点击量为110次，则需要获得至少 `110 * 3% ≈ 4` 次互动（如评论）。
*   互动率达标，笔记展示量可能跃升至1万-2万甚至更高。

![](img/cea384f6c19930a9e0e0e750dd72097b_13.png)

![](img/cea384f6c19930a9e0e0e750dd72097b_15.png)

**总结流程**：`初始展示 -> 考核打开率（≥11%）-> 进入更大流量池 -> 考核互动率（≥3%）-> 获得更多推荐`。新号通常有机会升级2-3次，而高权重老号的起点更高。

![](img/cea384f6c19930a9e0e0e750dd72097b_17.png)

### 互动权重评分

![](img/cea384f6c19930a9e0e0e750dd72097b_19.png)

除了基础达标线，不同的互动行为对笔记的权重加分也不同。平台会对以下行为进行评分（权重通常由高到低）：
1.  **分享**
2.  **收藏**
3.  **评论**
4.  **点赞**
5.  **关注用户**

**请注意**：必须先满足 **11%的打开率** 和 **3%的互动率** 这两个基础属性，后续的互动权重加分才能更有效地推动流量增长。

了解了如何获得推荐后，我们更需要知道哪些行为会阻碍账号发展。以下是运营中必须规避的常见错误。

## 必须避免的运营误区

以下列举了新手运营小红书时常犯的错误，请务必注意。

![](img/cea384f6c19930a9e0e0e750dd72097b_21.png)

![](img/cea384f6c19930a9e0e0e750dd72097b_23.png)

1.  **新号切勿立即发笔记**：新建账号后，需要一段时间（俗称“养号”）来模拟正常用户行为，浏览、互动，让平台建立账号标签。立即发笔记会消耗初始权重，导致后续流量低迷。
2.  **违规必然导致降权**：任何违规行为（如导流、抄袭、违禁内容）都会导致账号被降权，笔记推荐和搜索排名大幅下降，恢复难度极大。
3.  **标签切忌反复修改**：账号、店铺或笔记的标签一旦确定，不要频繁修改。尤其是在账号已有稳定流量后修改核心标签，极易引发限流。
4.  **避免一机多号登录**：尽量保证“一机一卡一号”的登录环境。同一设备登录多个账号可能被判定为营销号，影响权重。
5.  **谨慎处理引流内容**：在笔记或私信中直接发布微信号、二维码等引流信息，是平台重点打击的行为，会导致严厉处罚。
6.  **谨慎发布推广内容**：过早或过多地发布硬广内容，会影响账号内容质量评分，不利于长期运营。
