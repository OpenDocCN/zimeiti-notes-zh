# 抖音运营教程：P15：8.影视推广平台 🎬

在本节课中，我们将要学习如何通过“影视推广平台”在抖音、快手等平台发布视频并结算收益。我们将详细解释其运作模式、收益计算方式以及具体的操作步骤。

![](img/3d4650a005fbcffe94b0adeb10fd0446_3.png)

许多同学询问，如果想把视频发布到抖音或快手平台，应该如何结算收益。本节内容将解答这个问题。

我们剪辑完成的视频，除了可以发布到抖音和快手，还能分发到超过30个其他自媒体平台。在这些其他平台上，我们可以依靠视频的**播放量**直接结算收益。

![](img/3d4650a005fbcffe94b0adeb10fd0446_5.png)

然而，抖音和快手这两个平台的情况有所不同。正如之前提到的，视频发布到这两个平台，主要的作用是**增长粉丝**。视频本身通常不能直接通过播放量带来收益。

那么，有没有同学只想专注于抖音和快手这两个熟悉的平台，但又希望不仅能涨粉，还能通过播放量获得收益呢？答案是肯定的。

以下是为您介绍的解决方案所涉及的平台。

![](img/3d4650a005fbcffe94b0adeb10fd0446_7.png)

视频可以发布到以下四个主要平台：
*   **抖音**
*   **快手**
*   **小红书**
*   **微信视频号**

这些平台都是大家日常会使用到的。小红书在年轻用户，特别是女性用户中较为流行。微信视频号则内嵌于微信应用中，几乎人人可用。

发布视频后，我们通过“影视推广平台”来接取任务，并依据任务要求结算奖金。平台上的素材非常丰富。

以下是影视推广任务的示例：
*   **爱奇艺的任务**
*   **搜狐的任务**
*   各种**影视剧的推广任务**

“影视推广”是指为电影、电视剧或综艺节目进行线上宣传。由于线下宣传受限，片方会通过此类平台发布任务，创作者制作推广视频并发布到指定平台，即可根据视频数据（如播放量）获得奖金。

若要参与任务并获得收益，需要完成一个关键的提交步骤。

每个任务都需要提交“回填链接”以确认参与和结算。需要填写的信息包括：
1.  个人微信号和账号昵称。
2.  发布视频的抖音或快手ID。
3.  所推广的具体电视剧/电影名称。
4.  已发布视频的作品链接。
5.  视频的发布时间。
6.  视频播放量的截图。

提交完整的回填信息后，平台方会据此结算并发放收益。

影视推广的收益通常比单纯依靠平台播放量分成的收益更高。目前平台上也有大量的短剧推广任务。

短剧通常由传媒公司或网红自制，他们同样需要推广。我们可以制作相关视频并发布到指定平台（如抖音、小红书、视频号），根据播放量要求结算收益。**参与这类任务对账号粉丝数没有要求**。

收益的计算基于单部剧集下，**播放量最高的三个作品的数据总和**。不同平台的结算标准不同。

以下是各平台的收益计算标准：
*   **抖音**：每1万播放量，奖金为 **50元**。
*   **微信视频号**：每1万播放量，奖金为 **10元**（因用户基数大，单价较低）。
*   **小红书**：每1000播放量，奖金为 **30元**（单价最高）。

此外，当视频播放量超过基础要求后，每增加1万播放量通常还会有额外的奖金。

除了上述平台，还有例如搜狐短剧等任务。虽然搜狐视频目前的用户活跃度可能不如前几个平台，但其提供的任务单价有时很有竞争力。

对于搜狐短剧任务，收益结算方式有所不同。

部分任务要求视频同时在抖音和快手发布，收益按**点赞数**计算，同样对粉丝数无要求。结算标准如下：
*   **抖音**：单条作品获5000赞奖励100元；获1万赞奖励200元。
*   **快手**：单条作品获1万赞奖励180元。
*   **搜狐短剧**：在此类任务中，其提供的单价往往是最高的。

本节课中，我们一起学习了“影视推广平台”的运作模式。关键点在于：通过接取特定影视剧的推广任务，将视频发布到抖音、快手、小红书、视频号等平台，并根据不同平台规则（按播放量或点赞数）结算收益。这种方式不仅能帮助账号涨粉，还能获得可观的推广奖金，且对账号的初始粉丝量没有要求。
