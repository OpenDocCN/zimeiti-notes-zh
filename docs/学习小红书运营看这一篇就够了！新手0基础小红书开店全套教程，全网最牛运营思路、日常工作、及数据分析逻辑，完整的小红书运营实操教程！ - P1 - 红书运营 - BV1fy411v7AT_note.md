# 小红书运营：P1：零基础开店与店铺类型详解

## 概述
在本节课中，我们将学习小红书零基础开店的完整流程，包括开店前的准备工作、店铺注册的具体步骤，以及小红书平台上的各种店铺类型及其区别。掌握这些基础知识，是开启小红书电商之旅的第一步。

---

## 第一课：开店准备 📝

上一节我们介绍了课程的整体框架，本节中我们来看看开店前需要准备哪些具体事项。

![](img/ff5cb676c0c9862fc71f7742680ab805_1.png)

小红书开店准备相对简单，主要涉及以下几个核心部分：

![](img/ff5cb676c0c9862fc71f7742680ab805_3.png)

1.  **账号与身份信息**：您需要注册一个小红书账号，并准备好本人的身份证正反面照片。
2.  **支付工具**：您需要拥有并绑定银行卡的微信支付和支付宝账户。这两个工具用于客户付款和商家收款。**注意**：请使用已实名认证且使用超过3年的稳定账户，新注册的账户在收款量较大时存在被封风险。
3.  **资金准备**：开店需要准备两部分资金。
    *   **保证金与手续费**：基础保证金为1000元，同时平台会收取约7元的手续费。这是与其他平台（如淘宝、拼多多）仅收保证金的不同之处。
    *   **店铺流动资金**：建议至少准备2000元以上，用于店铺前期运营、产品上架、促销活动等，是店铺能够启动并产生数据的关键。

**核心概念**：`开店成本 = 保证金(1000元) + 手续费(约7元) + 流动资金(≥2000元)`

完成以上准备后，您的店铺基础搭建工作就基本完成了。

## 第二课：店铺注册 📱

上一节我们准备好了开店所需的材料，本节中我们进入实操环节，看看如何一步步注册店铺。

以下是详细的店铺注册步骤：

1.  **设备与登录**：使用手机下载小红书APP。用准备好的手机号登录账号。
2.  **进入创作中心**：点击右下角“我”进入个人中心，然后点击页面右上角或左上角的“三条杠”菜单图标，选择“创作中心”。
3.  **找到开店入口**：在创作中心页面，点击“更多服务”，在服务列表中找到并点击“开通店铺”（通常位于第三排左右）。
4.  **开始认证**：页面会弹出店铺开通引导，点击“立即认证”按钮。
5.  **提交材料与选择类型**：按照系统提示，上传已准备好的身份证照片，并选择店铺类型。**对于新手，强烈建议先选择“个人店”**进行练手和熟悉流程。
6.  **填写店铺信息**：填写店铺名称、简介等信息。这些信息后期可以修改，但建议一次性填写准确。
7.  **等待审核**：提交所有信息后，等待平台审核。审核时间通常为5分钟至1天，成功后即可正式开通店铺。
8.  **下载客服工具**：店铺开通后，务必在手机上下载“小红书客服工作台”APP。这是处理客户咨询和售后问题的独立工具，无法在普通小红书APP内完成。

注册流程完成后，您的店铺就正式上线了。

## 第三课：店铺类型详解 🏪

上一节我们完成了店铺注册，本节中我们来深入了解小红书平台上有哪些店铺类型，以及它们之间的区别，帮助您明确自己的定位。

小红书店铺体系主要分为以下几类，其核心差异体现在**入驻要求**和**保证金规则**上：

![](img/ff5cb676c0c9862fc71f7742680ab805_9.png)

以下是各类店铺的简要说明：

![](img/ff5cb676c0c9862fc71f7742680ab805_11.png)

*   **个人店**
    *   **申请主体**：个人用户。
    *   **经营特点**：适合经营小额、多品类商品，如首饰、服装、手机壳等，类似于“杂货铺”。
    *   **入驻要求**：仅需本人身份证。最简单，适合零基础新手入门体验。
    *   **保证金**：基础保证金为1000元。

![](img/ff5cb676c0c9862fc71f7742680ab805_13.png)

*   **个体工商店**
    *   **申请主体**：个体工商户。
    *   **经营特点**：与个人店类似。
    *   **入驻要求**：需要个体工商户营业执照。比个人店要求稍高。
    *   **保证金**：基础保证金为1000元。

*   **普通企业店**
    *   **申请主体**：企业。
    *   **经营特点**：经营企业自有产品或授权产品。
    *   **入驻要求**：需要企业营业执照。
    *   **保证金**：基础保证金为1000元。

![](img/ff5cb676c0c9862fc71f7742680ab805_15.png)

*   **专卖店/旗舰店/卖场型旗舰店**
    *   **申请主体**：品牌方、大型零售商。
    *   **经营特点**：经营单一或多个品牌商品，要求有品牌资质或授权。
    *   **入驻要求**：需要营业执照+品牌商标注册证或品牌授权书。要求最高。
    *   **保证金**：基础保证金为1000元，但可能涉及特定类目保证金。

![](img/ff5cb676c0c9862fc71f7742680ab805_17.png)

![](img/ff5cb676c0c9862fc71f7742680ab805_19.png)

![](img/ff5cb676c0c9862fc71f7742680ab805_21.png)

**重要概念：浮动保证金**
除了基础保证金，小红书还有一个**浮动保证金**规则，它与店铺月成交额挂钩：

![](img/ff5cb676c0c9862fc71f7742680ab805_25.png)

```
如果 月成交额 < 5万元：浮动保证金为0（但仍需缴纳1000元基础保证金）。
如果 5万元 ≤ 月成交额 < 10万元：需补足保证金至3000元。
如果 10万元 ≤ 月成交额 < 50万元：需补足保证金至5000元。
如果 月成交额 ≥ 50万元：需补足保证金至20000元。
```

**注意**：当您提现时，系统会检查保证金账户总额是否达到当前成交额对应的标准。如果不足，将无法提现。例如，只交了1000元保证金，但月成交额达到6万，则账户资金无法提出，需补足2000元使总保证金达到3000元。

对于新手，从**个人店**开始尝试是最稳妥的选择。

## 总结
本节课我们一起学习了小红书开店的三大基础模块：**开店准备**、**店铺注册**和**店铺类型**。您需要准备好身份信息、支付工具和启动资金；按照步骤指引完成店铺注册；并根据自身情况，理解不同店铺类型的区别，新手建议从个人店起步。掌握了这些，您就为后续的选品、运营打下了坚实的基础。下节课，我们将进入实战环节，学习如何为您的店铺挑选产品和寻找货源。
