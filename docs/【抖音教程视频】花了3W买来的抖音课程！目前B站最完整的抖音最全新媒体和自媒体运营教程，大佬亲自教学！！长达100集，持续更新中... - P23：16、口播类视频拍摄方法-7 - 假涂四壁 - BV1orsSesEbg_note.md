# 抖音运营教程：P23：16、口播类视频拍摄方法-7 🎬

## 概述
在本节课中，我们将学习口播类视频的具体拍摄方法。我们将通过分析一个成功案例，详细讲解拍摄时需要注意的构图、内容定位、设备使用等核心要点，帮助你掌握制作高质量口播视频的基础技巧。

---

上一节我们介绍了视频拍摄的通用注意事项，本节中我们来看看口播类视频的具体拍摄方法。

拍摄环节需要注意以下几点：构图比例、主题突出、事物清晰、光线色彩和内容连贯。这些描述可能比较抽象，老师将通过举例来具体说明。主要举两种类型的例子：口播类型和日常生活类型。

![](img/bb39e474b25774af47eae5a6a7a9ce2d_1.png)

因为无论你做什么领域的口播，例如金融、装修、养生、情感等，拍摄方法都是相同的。同样，无论你是拍段子、记录生活、拍搞笑内容还是日常美食，拍摄方法也都一样。

所以老师只举这两种例子，它们几乎涵盖了所有的拍摄方法。首先，我们看第一个口播类型的例子。我举一个咱们学员的例子，名字会打在公屏上。

![](img/bb39e474b25774af47eae5a6a7a9ce2d_1.png)

你可以搜索到他的名字叫“种小草”。他做的是情感口播。他是一位40岁带三个娃的宝妈，单条视频播放量破8000多万，一个视频涨了30万粉丝。为什么他一条视频就能涨30万粉丝，而我们涨粉却那么难？

这是因为他吸引的都是针对性很强的群体。稍后你看他的视频，一眼就能看出他吸引的是谁。就像如果我吸引的群体是20多岁的学生党，我拍的所有内容都是学生感兴趣的，那么我的涨粉速度就会非常快，而且非常精准。

先说一下构图。抖音是一个**九比十六**的竖屏平台。我们每个人用的都是这个比例。关键在于画面内部，你想拍成横屏还是竖屏。你看他拍的是横屏，一个**四比三**的横屏。

强调一下，无论你做哪种类型的视频，不要一会儿横屏，一会儿竖屏，风格不统一。如果你选择横屏，就全部拍横屏；选择竖屏，就全部拍竖屏。要保持稳定，这些细节都需要注意。

接着，人物主体应占据画面中间约**70%** 的面积。不要怼着大脸拍，一张大脸占满屏幕会让人看不下去。一定要在左边、右边和上方留出适当的空白，这样画面看起来才清晰、舒适。

如果你觉得自己不是特别漂亮或帅气，能不能做口播、拍抖音？答案是：可以。抖音并不是只有特别好看的人才能火。口播主要传达的是内容和知识。你只需要让别人看起来清晰、体面就行了。觉得自己不好看的，可以开启美颜滤镜。在这个时代，美颜滤镜一开，就没有丑人。你可以放心大胆地开美颜，只要别太夸张就行。

接着，我先带你看一下他这条涨粉30万的视频。我的小要求是：看完之后直接回答我，这个视频想吸引的是哪个群体的眼球。

![](img/bb39e474b25774af47eae5a6a7a9ce2d_3.png)

![](img/bb39e474b25774af47eae5a6a7a9ce2d_3.png)

（视频内容示例：“跟婆婆在一起住是什么样的感受？我跟我婆婆在一起住了16年了，平时磕磕碰碰有点小不愉快，那都是正常的。买了衣服不敢光明正大拿回家，新衣服都是放几天再穿，也都不是事儿哈。我最大的感受就是这十多年以来，我娘家人很少在我家住上几天，很少，这对于我内心来说其实是一个很大的缺失。”）

有同学说“婆媳关系”。那么具体是哪个群体呢？婆媳关系是一种关系，但具体吸引哪类人？你看，我们班有同学说是“做媳妇的”，非常好。谁对婆媳关系最有感触？其实男性群体对婆媳关系的感触绝对没有女性深刻。而且，什么样的人能感受婆媳关系？没结婚的十几岁、二十几岁小姑娘能感受到吗？她感受不到。一定是结了婚的女性群体，才能真真切切地感受到婆媳关系，才能和“种小草”产生共鸣。因此，它吸引的一定是已婚群体。我们稍后可以通过数据具体看看是哪个群体。

先讲一下，你看他是不是开了美颜？路线有点歪了。所以你们到时候如果觉得自己不是特别漂亮，可以稍微开一下美颜，让自己看起来体面一点就行。然后你看，他的构图也是四比三。

有同学问：“老师，做口播我不想在户外拍，能在家里拍吗？” 可以，你在哪里拍都行。只要把背景整理得好看一点就行，别让背景太难看。

本节课中，我们一起学习了口播类视频的拍摄方法。我们通过“种小草”的案例，分析了如何通过精准的**内容定位**吸引目标群体，掌握了**九比十六竖屏**中采用**四比三横屏构图**、人物主体占比约**70%** 并适当留白的技巧，并了解了使用**美颜滤镜**和**手机收声麦克风**来提升画面观感和音频质量的重要性。记住，稳定的风格和清晰的内容传达是口播视频成功的关键。
