# 新媒体运营入门：P15：01.10.03：🤖 训练AI从分析到产出爆款笔记

## 概述
在本节课中，我们将学习如何训练AI（以ChatGPT为例）分析爆款笔记，并最终引导它生成具备爆款潜质的新笔记。我们将通过具体的步骤，从“喂资料”到“提要求”，完整地走一遍流程。

上一节我们介绍了通过提供资料让AI理解平台风格和用语的方法。虽然以旅游攻略为例的结果可能不理想，但方法是正确的。关键在于让AI通过分析你提供的优质内容，学习特定平台的文案风格和写作方式。

## 第一步：明确分析指令
要让AI有效分析，我们需要给出清晰、具体的指令。以下是分析爆款笔记时需要关注的几个核心点，我们需要在指令中明确告诉AI：

*   **分析爆款因素**：找出内容中可能使其成为爆款的关键点。
*   **分析内容结构**：拆解文案的框架。
    *   **开篇**：用什么方式吸引用户继续阅读。
    *   **中间**：提供了哪些内容来戳中用户痛点，提升停留时长。
*   **分析互动设计**：结尾如何设置“钩子”，引导用户私信、关注或评论。
*   **分析关键词策略**：检查笔记是否植入了特定关键词，以及其出现频率和铺设方式。

我们可以将指令整合如下：
```
请你分析提供的小红书爆款笔记文案。需要分析：
1. 爆款因素：列出可能使其成为爆款的关键点。
2. 文案结构：分析开篇如何吸引用户，中间如何切入痛点，结尾如何引导互动。
3. 关键词：分析是否结合中文分词法设计了关键词。如果有，指出关键词是什么、出现频率如何。
```

## 第二步：提供素材并获取分析
我们以“考研”领域的爆款笔记为例进行演示。

1.  **寻找爆款笔记**：在小红书平台找到点赞数据较高（例如3000+）的考研相关笔记。
2.  **复制文案内容**：将选中的笔记全文复制下来。
3.  **提交给AI分析**：将文案连同上面的分析指令，提交给ChatGPT。

![](img/12215f4dc146c6afcd16ab1d84d3ae83_9.png)

![](img/12215f4dc146c6afcd16ab1d84d3ae83_10.png)

**示例分析结果反馈**：
*   **爆款因素**：情感共鸣（开篇营造焦虑感）、逆袭故事、提供详细备考计划、时间节点明确、正能量鼓励、使用表情符号增加亲和力。
*   **文案结构**：开篇用焦虑情绪吸引同类考生 -> 中间分享个人逆袭经验和具体方法 -> 结尾鼓励并引导互动。
*   **关键词分析**：多数笔记没有明显刻意设计的关键词，词汇均为常用词汇。

我们可以重复此过程，为AI提供多篇（例如3-4篇）不同风格的爆款笔记进行分析，让它积累更丰富的认知。

![](img/12215f4dc146c6afcd16ab1d84d3ae83_12.png)

## 第三步：整合分析结果并生成大纲
在AI分析了多篇笔记后，我们需要让它对分析结果进行整合与去重，总结出该领域的通用爆款要素。

![](img/12215f4dc146c6afcd16ab1d84d3ae83_14.png)

**指令示例**：
```
请将你前面分析过的几篇考研爆款笔记的爆款因素总结一下，整合成一个清晰的列表。
```

![](img/12215f4dc146c6afcd16ab1d84d3ae83_16.png)

获得总结列表后（例如：情感共鸣、逆袭故事、明确计划、资源建议、时间管理、正能量鼓励等），即可进入创作阶段。首先，让AI根据这些爆款要素生成一个笔记大纲。

**指令示例**：
```
我要写一篇关于考研的小红书笔记。请结合你总结的爆款笔记因素，为我规划一个具备爆款潜质的笔记大纲。大纲需要细分到具体的小节点。
```

**AI生成的大纲可能包括**：
*   开篇：用问题或故事吸引注意，引发情感共鸣。
*   主体：分点介绍专业选择、各科（政治/英语/数学/专业课）备考规划、时间管理方法。
*   结尾：心态调整建议、正能量鼓励、引导互动关注。

![](img/12215f4dc146c6afcd16ab1d84d3ae83_18.png)

## 第四步：根据大纲生成完整文案
大纲确认无误后，即可指令AI根据大纲和爆款要素撰写完整文案。此时可以提出更细致的要求。

**指令示例**：
```
请按照上述大纲，撰写完整的笔记文案。要求：
1. 结构严格遵循爆款笔记因素。
2. 包含一个简短的“逆袭小故事”引入，调动读者情绪。
3. 文案中要自然融入“考研”、“备考”等关键词。
4. 使用适量的emoji表情。
```

生成的文案将融合之前分析的爆款元素，例如以共鸣性开头引入逆袭故事，提供结构化建议，并以鼓励和引导互动结尾。虽然对于高级运营者可能还需优化，但对于入门者或基础内容生产，这已是一个合格的起点。

## 总结
本节课我们一起学习了利用AI辅助创作爆款笔记的完整流程。

![](img/12215f4dc146c6afcd16ab1d84d3ae83_20.png)

1.  **明确指令**：指导AI从爆款因素、结构、关键词等维度分析内容。
2.  **投喂素材**：提供多篇目标领域的真实爆款笔记给AI学习。
3.  **总结要素**：让AI整合分析结果，提炼共性爆款因子。
4.  **生成大纲**：基于爆款要素，规划笔记内容框架。
5.  **产出文案**：依据大纲和细节要求，生成融合爆款元素的完整笔记。

![](img/12215f4dc146c6afcd16ab1d84d3ae83_22.png)

![](img/12215f4dc146c6afcd16ab1d84d3ae83_24.png)

通过这个流程，我们可以系统化地利用AI学习平台调性，并批量生产内容。对于追求更高质量或个性化风格的创作者，可以进一步利用ChatGPT的“自定义GPT”功能，上传更详细的风格指南和样例，打造专属的创作助手。
