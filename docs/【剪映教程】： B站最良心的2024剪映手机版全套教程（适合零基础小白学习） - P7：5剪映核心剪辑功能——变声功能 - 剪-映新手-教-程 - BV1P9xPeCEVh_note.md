# 剪映教程：P7：5 剪映核心剪辑功能——变声功能 🎤

在本节课中，我们将要学习剪映App中的一个核心功能——变声功能。这个功能可以帮助你改变视频中的人声，无论是为了优化音质、创造趣味效果，还是模拟多人对话场景，都非常实用。

很多初学者在制作短视频时，可能会觉得自己的原声不够理想。此外，在制作包含多人对话或搞笑内容的视频时，也需要对声音进行处理。剪映的变声功能可以轻松将男声变为女声、女声变为男声，或者转换成大叔、萝莉等特色音效，极大地丰富了视频的创作可能性。

上一节我们介绍了其他剪辑工具，本节中我们来看看如何具体使用变声功能。

![](img/b2e9e691fead692c0cc199b457dc81e4_1.png)

## 实操步骤

![](img/b2e9e691fead692c0cc199b457dc81e4_3.png)

以下是使用剪映变声功能的详细步骤。

1.  **导入素材**：首先，打开剪映App，导入一段包含人声的视频素材。

    

2.  **选择素材**：在时间轴上选中需要变声的视频片段。

    

3.  **找到变声功能**：在底部出现的二级菜单中，找到并点击 **“变声”** 功能按钮。

    

![](img/b2e9e691fead692c0cc199b457dc81e4_5.png)

4.  **选择音效**：点击后，界面会弹出多种预设的音效选项，例如“大叔”、“萝莉”、“女生”、“男生”、“怪物”等。你可以点击不同的音效进行试听，并选择最适合视频效果的一个。

    

5.  **完成变声**：选择音效后，视频的原声就会被实时替换。你可以播放预览，检查变声效果是否满意。

    

## 功能应用与总结

使用变声功能，不仅可以解决对原声不满意的尴尬，还能轻松实现视频的多人对话互动效果。例如，你可以通过变声，让一个人配音扮演父亲、母亲、孩子等多个角色，从而制作出有趣的家庭对话或情侣聊天场景。这大大降低了创作门槛，帮助新手用户提升视频的音频质感和互动趣味性。
