# 抖音运营教程：P33：11、视频类型1-2

在本节课中，我们将要学习抖音账号运营中关于“人设”的核心概念，并深入探讨两种主要的人设类型：强人设与弱人设。理解并选择适合自己的人设，是连接内容创作与粉丝增长、最终实现变现的关键一环。

上一节我们介绍了文案的重要性，本节中我们来看看内容创作的下一个核心环节：建立人设。

从单纯追求涨粉到最终实现变现，并非一条直线。中间存在一个至关重要的环节，即建立你的“人设”。建立人设的目的并非为了炫耀，而是为了在众多同类创作者中脱颖而出。例如，有十个人都做装修内容，如果其中一位创作者（比如小王同学）树立了一个鲜明有趣的人设，观众就更容易记住他，而忽略其他九位。当观众因为喜欢你这个人而关注你时，他们就成为了你的粉丝。这就是人设的价值。

接下来，我们来学习这个知识点。请边听边理解，人设主要分为两种类型：**强人设**与**弱人设**。区分清楚这两种人设后，你就能轻松判断任何视频所属的类型。

## 强人设：以“我”为核心

首先，第一种是**强人设**。强人设的核心是以创作者个人为切入点，内容与创作者自身密切相关。

强人设的内容必须与“你自己”相关。以下是强人设常见的切入点：

![](img/b8398d7dc61f6f793f782b77f50f9046_1.png)

*   **分享行业经验**：例如从事装修、办公设备、房产、汽车、餐饮、婚介、法律咨询、电商等领域的工作者，分享专业知识和经验。
*   **展示兴趣爱好**：例如制作美食、旅行vlog、手工艺品、宠物日常、星座占卜、美妆教程、护肤心得、服装穿搭、游戏攻略、摄影技巧等。

简而言之，以你个人为切入点，去分享你的生活、工作或爱好。大部分强人设视频会采用**真人出镜**的形式，因为内容与创作者本人直接相关。

![](img/b8398d7dc61f6f793f782b77f50f9046_3.png)

## 弱人设：以“内容”为核心

与强人设相对应的，是**弱人设**。弱人设的核心是内容本身，与创作者个人的关联度较弱，主要目的是运营好账号。

以下是弱人设的典型例子：

*   **情感语录类账号**：发布配乐的情感文字图片或视频，通常无真人出镜或配音。
*   **影视解说类账号**：虽然可能有解说者的声音，但焦点是电影内容和解说本身，观众关注的是“解说了什么电影”，而非“谁在解说”。

弱人设视频的重点在于传递内容价值，创作者个人形象并非吸引观众的主要因素。

理解并能够区分强人设与弱人设，是规划账号方向的基础。只有明确了这一点，才能更好地决定自己适合创作何种类型的内容。

## 内容创作方向：从人设到视频

明确了人设类型后，任何与你个人（强人设）或你选定领域（弱人设）相关的内容，都可以成为视频素材。下面我们逐一讲解几种常见的内容形式，首先从“讲故事”开始。

### 1. 讲故事（口播）

讲故事，即分享客户故事、从业经历或人生感悟。故事本身不要求绝对真实，可以从文案平台寻找有趣的故事进行分享。

讲故事主要采用**口播**形式。口播即面对镜头进行讲述。

口播涵盖多种领域，例如：
*   养生知识口播
*   情感话题口播
*   育儿经验口播
*   房产、汽车、金融、装修等专业知识口播

**讲故事类口播**的目的是让观众觉得内容有趣、有共鸣。**知识分享类口播**的目的是让观众在几十秒内获得“收获感”，觉得学到了东西。

无论你身处哪个领域，只要你有知识或故事可分享，都可以尝试口播。口播拍摄相对简单，只需准备好文案，用手机架好即可录制。

**常见问题解答：**
*   **口才不好能做口播吗？** 可以。口播是提前准备好稿子再录制，并非临场发挥。
*   **普通话不好能做口播吗？** 可以。如果你的目标受众是同城用户，使用方言可能更具亲和力和趣味性。同时，可以通过添加字幕确保观众理解。

口播是一种非常实用且易于上手的内容形式，在后续课程中我们会详细讲解拍摄技巧。

本节课中我们一起学习了抖音账号运营中“人设”的概念与分类。我们了解到，**强人设**以创作者个人为核心，**弱人设**以内容本身为核心。正确建立人设是吸引精准粉丝的关键。此外，我们还介绍了基于人设的一种基础内容形式——**口播**，并解答了相关的常见疑问。理解这些基础概念，将帮助你更清晰地规划自己的账号内容方向。
