上一节我们介绍了文案提取的需求，本节中我们来看看传统方法的不足。以往，当我们想获取一个短视频的文案时，通常需要借助一些比较原始或平台特定的方法。

以下是几种常见的传统方法：
*   使用“轻斗”等小程序进行提取。

这些方法虽然可行，但存在效率低下或平台限制的问题。例如，针对抖音的方法可能无法直接用于小红书或哔哩哔哩。
