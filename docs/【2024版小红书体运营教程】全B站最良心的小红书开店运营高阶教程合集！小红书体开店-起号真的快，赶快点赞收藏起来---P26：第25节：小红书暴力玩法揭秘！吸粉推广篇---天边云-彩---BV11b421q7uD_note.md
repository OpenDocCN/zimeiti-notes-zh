## 账号类型解析

上一节我们介绍了小红书运营的基础，本节中我们来看看平台上的账号类型。小红书的账号主要可以分为三类。

以下是三类账号的详细说明：

*   **企业号**：主要作用是树立官方品牌形象。企业号拥有官方认证标志，可以展示品牌名称、地址等信息，并享有官方特权，便于开展业务。
*   **个人号**：这是目前大多数用户使用的账号类型。个人号的特点是**灵活**，可以是品牌员工或消费者身份，更容易塑造个人品牌特色，也适合品牌相关人物出镜。
*   **引流号**：这类账号在之前的课程中被称为“炮灰号”。引流号可以创建多个，通常没有精细的运营规划，主要用于特定引流目的，使用后可以舍弃，或在紧急情况下作为备用小号。

## 企业号矩阵运营建议

了解了账号分类后，接下来我们探讨如何有效利用这些账号进行推广。目前常见的推广方式是企业号、个人号或引流号自行发布笔记进行引流。

以下是打造企业号矩阵的四点具体建议：

![](img/750b7b6857c3a58dc69610a824ca7b21_2.png)

1.  **个人号参与品牌活动**：品牌企业号发起的试用、团购等活动，应使用个人小号积极参与，增加活动热度。
2.  **个人号展示品牌属性**：在个人号的笔记内容中，应多次暗示品牌的定位，例如目标用户是城市年轻人还是职场新人。
3.  **个人号控制产品发布频率**：个人号发布品牌产品内容的频率不宜过高。建议采用 **`营销内容 : 种草/科普内容 = 3 : 7`** 的比例。
4.  **利用引流号进行评论引流**：企业号发布内容后，可以使用引流号在评论区进行互动和引流，提升笔记活跃度。

## 官方广告投放：CPC竞价广告

![](img/750b7b6857c3a58dc69610a824ca7b21_8.png)

除了运营自有的账号矩阵，还有另一种高效的推广方式，即小红书的官方广告。

![](img/750b7b6857c3a58dc69610a824ca7b21_10.png)

![](img/750b7b6857c3a58dc69610a824ca7b21_12.png)

![](img/750b7b6857c3a58dc69610a824ca7b21_14.png)

小红书官方广告主要指 **CPC（Cost Per Click）竞价广告**。其申请与使用流程可在小红书官方网站的“业务合作” -> “推广合作”板块中找到。但请注意，投放官方广告有一个前提：**必须先认证企业号**。

企业号认证流程同样在官网进行，需要提交相关资质并支付审核费用，通过后普通账号即可升级为官方认证的企业号。

获得企业号资质后，便可以投放信息流广告。官方信息流广告主要分为两大类：

*   **信息流广告**：用户浏览小红书首页信息时，系统会在信息流中不定期间隔插入的广告。
*   **搜索广告**：当用户使用关键词进行搜索时，出现的与关键词相关的广告。其原理类似于百度的关键词竞价排名，通过对关键词出价来竞争不同的广告位。

目前，信息流广告位主要出现在搜索结果页的第3位、第13位和第23位。关于广告投放的更多常见问题，可以在官方平台的问答区找到详细解答。

官方广告投放的费用相对较高。因此，在资金允许的情况下，建议考虑与官方合作投放竞价广告，因为它能直接获得平台推荐的精准流量。
