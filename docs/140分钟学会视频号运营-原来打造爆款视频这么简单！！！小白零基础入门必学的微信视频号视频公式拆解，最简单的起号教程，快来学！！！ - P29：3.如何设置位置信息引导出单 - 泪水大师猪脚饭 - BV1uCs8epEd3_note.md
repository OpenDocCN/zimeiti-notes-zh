# 视频号运营：P29：如何设置位置信息引导出单 📍

在本节课中，我们将学习一个提升视频号转化率的关键技巧：如何正确设置视频的位置信息。这个看似简单的操作，能有效引导同城或特定区域的用户下单，是本地商家和希望吸引线下流量的创作者必须掌握的功能。

上一节我们介绍了视频内容策划的核心思路，本节中我们来看看如何利用平台工具为你的视频增加“地利”优势。

## 位置信息的作用

在视频号中为你的视频添加位置信息，主要有三个核心作用：
1.  **提升同城曝光**：系统会将你的视频推荐给定位地点附近的用户，增加本地流量。
2.  **建立信任与场景感**：显示具体门店或地点，让用户感觉更真实、更可信。
3.  **直接引导消费**：用户点击位置标签后，可以查看地点详情、导航或联系商家，缩短决策路径。

## 设置位置信息的步骤

![](img/db6ef5a9a73fb017708a30fa1c8cd4ff_0.png)

以下是具体的操作流程，请确保在发布视频前完成设置。

**第一步：进入发布页面**
在微信中打开视频号，点击底部中间的“+”号按钮，选择“拍摄”或“从相册选择”你的视频素材，进入视频编辑与发布页面。

**第二步：找到位置设置选项**
在发布页面中，向下滑动，在描述文案输入框下方，找到并点击“你在哪里？”或“添加位置”的选项。

![](img/db6ef5a9a73fb017708a30fa1c8cd4ff_1.png)

**第三步：搜索并选择位置**
系统会弹出位置搜索界面。你有两种选择方式：
*   **精确选择**：直接搜索你的店铺或目标地点的**全称**（例如“泪水大师猪脚饭深圳南山店”），并从结果中选择最准确的一个。
*   **模糊选择**：如果不想暴露具体地址，可以选择一个更大的地标或区域（例如“深圳南山区”、“万象天地”）。

**第四步：确认并发布**
选中位置后，它会出现在你的发布页面。此时，你可以继续编辑视频描述、封面等，最后点击“发表”即可。发布成功后，视频下方就会显示你设置的位置标签。

## 最佳实践与注意事项

![](img/db6ef5a9a73fb017708a30fa1c8cd4ff_2.png)

为了让位置信息发挥最大效果，请遵循以下建议：

**1. 优先选择精确地址**
对于实体店铺，务必使用营业执照上的准确名称和地址。这有助于积累地点的人气与评价，形成长期的流量入口。公式可以表示为：
**有效位置信息 = 准确名称 + 真实地址**

**2. 描述文案与位置联动**
在视频描述中，可以巧妙提及位置信息。例如：
> “今天带大家来看看我们**南山旗舰店**的后厨，新鲜猪脚每天现卤！📍点击下方地址，可以直接导航过来哦~”

![](img/db6ef5a9a73fb017708a30fa1c8cd4ff_3.png)

**3. 规避风险**
如果发布的内容与特定地点无关，或涉及隐私，不建议添加位置信息，以免造成误导或打扰。
