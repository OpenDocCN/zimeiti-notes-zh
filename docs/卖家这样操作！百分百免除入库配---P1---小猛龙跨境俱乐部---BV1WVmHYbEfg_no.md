## 亚马逊入库配置费规则解析

自2024年亚马逊入库配置费更新后，许多卖家为此感到困扰。根据亚马逊的规则，若想免去入库配置服务费，您的货件需要被分到**大于4个**不同的仓库。如果不分仓，则意味着需要支付高额费用。

![](img/bffddd20eda5dfa377ae96878dd3a8a8_1.png)

亚马逊的分仓逻辑旨在保证一件产品在多个仓库都有库存，从而让产品离客户更近，提升配送效率。

![](img/bffddd20eda5dfa377ae96878dd3a8a8_2.png)

以下是影响分仓的几个关键点：

*   **货量少于4箱**：基本不会被分仓，无法免除配置费。
*   **货量大于4箱且为单一产品**：箱数越多，被分仓的可能性越大。
*   **货量大于4箱且为多个不同产品**：例如发5箱货，是5个完全不同的产品，大多情况下也不会被分仓。

对于销售标准尺寸商品的卖家而言，如果能将货物分到4到5个仓库，通常是成本最优的选择。因此，我们的核心目标就是找到方法实现分仓，从而避免这笔费用。

## 100%免除入库配置费的方法

上一节我们了解了分仓的基本规则，本节中我们来看看亚马逊官方发布的、可以100%免除入库配置费的具体方法。

只要您的货件同时满足以下两个条件，即可获得“亚马逊优化的入库选项”，且无需支付入库配置服务费：

1.  **每个商品SKU至少包含5个相同的纸箱或托盘**。
2.  **每个纸箱或托盘必须包含相同的商品组合，且每件商品在每个纸箱或托盘中的数量也必须相同**。

**举个例子**：
假设卖家要发 **产品A 100个**， **产品B 100个**。如果想要免除配置费，可以这样设置装箱单：
*   准备5个相同的箱子。
*   每个箱子都装入：产品A 20个 + 产品B 20个。
*   这样，每个SKU（产品A和产品B）都恰好有5个箱子，且每个箱子的商品组合和数量完全一致。

## 灵活调整的装箱策略

如果您的产品不止两个，或者各产品数量不一致，则需要先进行计算。核心原则是：让每个SKU都能均匀地分配到至少5个箱子里。

![](img/bffddd20eda5dfa377ae96878dd3a8a8_8.png)

**计算公式如下**：
1.  确定您计划使用的总箱数 **X**（X ≥ 5）。
2.  将每个产品的总数量除以 **X**，得到每个箱子中该产品的数量。
    *   产品A每箱数量 = 产品A总数量 / X
    *   产品B每箱数量 = 产品B总数量 / X
    *   （以此类推）

**例如**：需要发产品A 150个，产品B 90个。如果我们决定用6个箱子（X=6）。
*   则每箱应装：产品A 25个（150/6），产品B 15个（90/6）。

## 其他分仓技巧与成本权衡

有时，系统可能仍然不给分仓。此时可以尝试以下技巧：
*   **添加一个无关的SKU**：尝试在每一箱里额外放入一个小件产品（例如一个赠品或配件）。这样就从“原箱发货”变成了“混装发货”，从而可能触发分仓。

![](img/bffddd20eda5dfa377ae96878dd3a8a8_8.png)

是否选择分仓，需要根据具体情况权衡。主要考虑以下因素：

**1. 产品特性与配置费**
*   配置费按商品数量收取。
*   对于**轻但数量多**的产品，总配置费可能很高，免除它更为划算。
*   对于**重但数量少**的产品，支付配置费后发往运费更低的**美西**仓库，可能整体更划算。

**2. 运输时效与成本**
*   **不着急**：可以接受货物被分配到距离较远的**美东或美中**仓库，慢速运输，以换取免配置费。
*   **考虑头程运费**：如果不支付配置费，FBA系统通常会将货物分配至美东和美中仓库，这可能增加头程运输成本。
*   **主动分仓策略**：有时，主动选择分仓并支付配置费，将货物集中发往**美西**仓库，再结合**卡车派送**，反而能因为更低的尾程配送成本和更高的物流效率，实现总成本最优。
