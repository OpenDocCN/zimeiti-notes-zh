# 新媒体运营速成指南：P19：02.4：工具篇-运营必备营销日历 📅

在本节课中，我们将要学习一个对运营工作至关重要的工具——营销日历。掌握它，你将学会如何通过追踪和结合热点事件，高效地规划内容，从而为你的账号获取更多流量。

---

上一节我们介绍了内容规划的重要性，本节中我们来看看如何借助工具系统化地捕捉热点。这个工具就是**营销日历**。

作为运营人员，无论是拍摄短视频还是创作图文内容，获取流量最快的方式之一就是结合热点。我们需要了解每天、每周或特定时间段内有哪些节日、纪念日或社会事件，这些都可以成为我们内容创作的切入点。



![](img/168def841f487ce762d9f8e42679b1da_1.png)

![](img/168def841f487ce762d9f8e42679b1da_1.png)

那么，营销日历具体是什么？我们一起来看它的界面。它能够按日期展示当天相关的历史事件、名人诞辰、纪念日等。例如，某天可能是“中国物理学之父逝世日”或“宝马公司成立日”。

如果你的内容能够巧妙地与这些热点关联，你的内容就拥有了一个基础的流量池。无论是汽车领域的内容结合“宝马公司成立”，还是文化领域的内容结合“《茶花女》首演”，或是结合“国际妇女节”、“马航MH370事件”等社会话题，都可以根据日历提前规划。



![](img/168def841f487ce762d9f8e42679b1da_3.png)

除了列出热点事件，这个工具的功能更为强大。例如，点击“宝马汽车公司成立”这个事件，它还能提供其他品牌或创作者围绕此热点所做的优秀案例。

![](img/168def841f487ce762d9f8e42679b1da_3.png)
