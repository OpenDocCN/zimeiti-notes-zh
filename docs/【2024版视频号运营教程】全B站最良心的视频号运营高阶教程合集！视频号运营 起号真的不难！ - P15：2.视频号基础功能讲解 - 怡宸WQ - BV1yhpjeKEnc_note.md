# 视频号运营教程：P15：2. 视频号基础功能讲解 🎬

![](img/844423739b3fdbab14e1b36d169300db_1.png)

在本节课中，我们将系统学习视频号的基础功能设置与界面操作。掌握这些是开启视频号带货之旅的第一步。

![](img/844423739b3fdbab14e1b36d169300db_3.png)

## 视频号入口与开通

![](img/844423739b3fdbab14e1b36d169300db_5.png)

![](img/844423739b3fdbab14e1b36d169300db_7.png)

视频号主要有两个进入端口。第一个端口位于微信“发现”页的“朋友圈”下方。第二个端口在微信“我”->“设置”->“通用”->“发现页管理”中，找到“视频号”并开启显示。

![](img/844423739b3fdbab14e1b36d169300db_9.png)

如果朋友圈下方没有视频号入口，可按以下步骤操作：
1.  点击微信右下角“我”。
2.  进入“设置”。
3.  选择“通用”。
4.  点击“发现页管理”。
5.  找到“视频号”选项，并开启其开关。

![](img/844423739b3fdbab14e1b36d169300db_11.png)

![](img/844423739b3fdbab14e1b36d169300db_13.png)

开通视频号非常简单。进入视频号页面后，点击“发表视频”，根据提示设置账号名称，即可完成开通。未开通也不影响观看视频。

## 视频号主界面功能

进入视频号后，主界面顶部有三个主要标签页：“推荐”、“朋友点赞”和“关注”。

*   **推荐**：系统根据你的观看喜好智能推送的视频流。
*   **朋友点赞**：展示你的微信好友点赞过的视频。哪位好友点赞了某个作品，会在此处显示。
*   **关注**：集中展示你已关注的所有视频号发布的最新内容。

界面右上角的放大镜图标是搜索功能。你可以直接搜索**视频号账号名称**来找到特定账号，也可以搜索**商品关键词**（如“保鲜膜”）来寻找相关的带货账号，这对后期寻找对标账号很有帮助。

## 个人主页与隐私设置

点击底部导航栏的“我”，即可进入个人视频号主页。点击右上角的三个点，进入“资料设置”。

![](img/844423739b3fdbab14e1b36d169300db_15.png)

以下是个人资料设置的关键项：
*   **头像、名字、性别、地区、简介**：这些信息可以自由填写，建议参考优秀对标账号的样式。简介可以多次修改，名字一年内可修改**5**次。
*   **视频号ID**：在资料页面可以找到。长按即可复制。**视频号ID**在与商家设置定向佣金时会用到。
*   **视频号二维码**：你的视频号专属二维码，可用于分享。

![](img/844423739b3fdbab14e1b36d169300db_16.png)

接下来，我们看看重要的隐私与权限设置。点击“隐私与权限”：
*   **在个人名片展示视频号**：开启后，你的微信好友在其通讯录中点击你的名片时，能看到你的视频号入口。若不想让熟人看到，可关闭此选项。
*   **作品公开范围**：可以设置为“公开”或“公开但不推荐给朋友”。你还可以在“不让他（她）看”列表中，添加特定的微信好友。设置后，你发布的视频将对这些人不可见。
*   **点赞隐私**：在“隐私”->“点赞”中，可以设置“不让他（她）看我的点赞”以及“不看他（她）的点赞”。例如，为自己发布的视频点赞后，若不想让某些人看到，可在此处设置。

其他如“黑名单”、“更换管理员”、“注销视频号”等功能，可按界面提示操作。注销账号通常需要满足一定条件，如完成提现、解除公众号绑定等。

## 商品橱窗与作品管理

在个人主页，点击“管理橱窗”可直接进入带货后台。点击他人视频号主页的“商品橱窗”，可查看其带货情况。

需要区分两个“已售”数据：
*   **账号“已售”数**：指该视频号作者所有带货商品的历史总销量之和。
*   **单个商品“已售”数**：指该商品被**所有带货视频号**销售出的总量。

![](img/844423739b3fdbab14e1b36d169300db_18.png)

进入自己发布的任一视频作品页面，点击右上角的“...”菜单，可以进行多项管理操作：
*   **置顶**：将该视频固定在个人主页顶部。
*   **上热门**：付费为视频增加曝光。
*   **关闭评论**：禁止他人对此视频评论。
*   **仅自己可见** / **删除**：隐藏或删除该视频。
*   **添加提示**：为视频添加文字说明（通常带货视频可不加）。

![](img/844423739b3fdbab14e1b36d169300db_20.png)

## 评论区互动管理

![](img/844423739b3fdbab14e1b36d169300db_22.png)

评论区是与观众互动的重要区域。对于不当评论，可以**长按该条评论**，选择删除。

![](img/844423739b3fdbab14e1b36d169300db_23.png)

![](img/844423739b3fdbab14e1b36d169300db_25.png)

![](img/844423739b3fdbab14e1b36d169300db_27.png)

视频号评论**没有官方置顶功能**。但评论的排列顺序会受点赞数影响。因此，如果需要将引导下单的评论（如“商品在下方橱窗”）让更多人看到，可以发动小号或朋友为该条评论**点赞**，点赞数越高，评论位置越靠前，越容易被新观众看到。
