# 亚马逊物流教程：P1：发货重量计算方式详解

在本节课中，我们将要学习亚马逊物流中一个非常关键的概念：发货重量是如何计算的。理解这一点对于控制物流成本至关重要。

## 概述

发货到亚马逊仓库或通过亚马逊物流（FBA）发货时，运费的计算并非简单地依据货物的实际重量。物流服务商通常会采用一种称为“体积重”或“计费重”的概念来计算运费，并取**实际重量**与**体积重量**中数值较大的一个作为计费依据。

上一节我们介绍了计费重的基本概念，本节中我们来看看具体的计算规则和实际应用。

## 核心计算规则

物流费用的计算遵循一个核心原则：比较货物的实际重量和体积重量，取较大值作为计费重量。其公式如下：

**计费重量 = MAX(实际重量， 体积重量)**

其中，**体积重量**的计算公式为：

**体积重量 (kg) = 长 (cm) × 宽 (cm) × 高 (cm) / 抛重系数**

> 注：国际物流中常见的“抛重系数”为5000或6000，具体数值需向您的物流服务商确认。例如，系数为5000时，公式为：`体积重量 = 长 × 宽 × 高 / 5000`。

## 如何判断使用哪种重量？

以下是判断您的货物将按何种方式计费的步骤：

1.  **测量与计算**：首先，准确测量货物的外箱尺寸（长、宽、高，单位：厘米）和实际重量（单位：公斤）。
2.  **计算体积重**：使用上述公式计算出货物的体积重量。
3.  **比较与取大**：将计算出的体积重量与实际重量进行比较。
4.  **确定计费重**：数值较大的那个重量，就是物流商最终用来计算运费的“计费重量”。

## 空运与海运的常见计费标准

不同的运输方式，其计费重量的计算标准也略有不同。

*   **空运**：通常对计费重量有最低要求。例如，即使您的货物实际重量只有12公斤，如果计算出的体积重低于21公斤，许多空运渠道也会按**最低计费重量21公斤**来收费。这就是所谓的“重量分界点”或“最低收费重量”。
*   **海运**：同样遵循“取大原则”，但通常没有像空运那样高的最低计费重量门槛。计费更直接地取决于实际重量与体积重量的比较结果。

## 实例分析与成本考量

以您提到的“12公斤，尺寸未超过60厘米”的货物为例，价格是否合理取决于计费方式：

*   如果物流商按**实际重量12公斤**计费，那么1080元的价格需要参考当前市场空运或快船海运的单价来判断。
*   如果因尺寸较大，按**体积重量**计费，且计算结果远高于12公斤（例如达到21公斤或更高），那么1080元对应的单价就会降低。
*   此外，还需明确是**空运**还是**海运**。空运单价高但时效快；海运（快船/慢船）单价低但时效慢。1080元对于12公斤空运和12公斤快船海运而言，意义完全不同。

因此，要评估价格，您必须向服务商确认两个关键信息：**运输方式** 以及 **最终的计费重量是多少**。

## 总结

本节课中我们一起学习了亚马逊物流中发货重量的核心计算规则。请务必记住，运费基于**实际重量**与**体积重量**的较大值计算。在发货前，主动测量尺寸、计算体积重，并向物流服务商确认其采用的抛重系数和最低计费标准，是精准预估和控制物流成本的关键步骤。
