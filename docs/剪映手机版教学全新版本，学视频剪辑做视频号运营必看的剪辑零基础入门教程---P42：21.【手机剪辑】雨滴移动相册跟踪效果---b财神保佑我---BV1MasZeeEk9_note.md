# 剪映手机版教学：P42：21. 雨滴移动相册跟踪效果 🌧️📸

在本节课中，我们将学习如何制作一个动态的“雨滴移动相册跟踪效果”。这个效果能让多张照片像雨滴一样依次滑入画面，并带有边框装饰，最终组合成一个动态相册，非常适合用于视频号等短视频内容创作。

## 概述
我们将分步完成效果制作。首先，需要准备并排列多张照片，为它们添加统一的边框特效并导出。然后，重新导入这个合成好的视频，通过关键帧动画让照片依次移动，最后加入雨滴音效和背景音乐。

## 素材准备与初步合成
首先，我们需要将多张照片排列成一个网格，并为其添加统一的视觉效果。

1.  导入第一张图片到剪映主轨道。
2.  将这张图片的时长拉长一些。
3.  将图片缩小，并移动到画面的左上角位置。
4.  点击“新增画中画”，导入第二张图片。
5.  将第二张图片摆放在第一张图片的右侧，紧挨着它。
6.  调整第二张图片的时长，使其与第一张图片的结尾对齐。
7.  重复步骤4至6，继续导入更多图片，将它们依次排列成一个整齐的网格。

以下是排列图片的核心操作，通常通过拖拽实现：
```
[将图片A拖拽至左上角] -> [导入图片B为画中画] -> [将图片B与图片A右对齐并时长对齐]
```

完成图片排列后，我们来设置统一的背景和边框。

1.  点击“背景”选项。
2.  选择“画布模糊”功能，并应用第四个模糊程度最高的样式。
3.  点击“特效”，选择“画面特效”。
4.  在“边框”分类下，找到并选择“粉黄渐变”特效。
5.  将这个特效的时长拉长，覆盖整个视频片段。
6.  为第二张图片（画中画）同样添加“粉黄渐变”特效：添加特效后，在参数设置中将“作用对象”选为对应的画中画图片。
7.  为所有剩余图片重复步骤6，确保每一张都加上了相同的边框特效。

当所有图片都添加了边框后，点击“导出”按钮，将这个合成好的视频保存到手机相册备用。

## 关键帧动画制作
上一节我们完成了静态相册的合成，本节中我们来看看如何让它动起来。我们将通过关键帧技术，让照片依次滑入画面。

1.  新建一个项目，导入刚才导出的那个合成视频。
2.  将视频放大，使其左上角与画面左上角对齐。
3.  在视频开头的位置，添加一个**关键帧**。这个关键帧记录了视频的初始位置。
4.  将时间线移动到**1秒**的位置。
5.  在此处，将视频画面向左下角移动，直到**第二张图片**完全出现在画面中央。
6.  移动操作会自动在1秒处生成一个新的**关键帧**。这样，剪映就会自动创建从第一个关键帧到第二个关键帧的移动动画。
7.  将时间线向后移动**10帧**，在此处添加一个**关键帧**。这个关键帧的目的是“定格”当前画面，为下一次移动做准备。
8.  将时间线再向后移动**1秒**。
9.  在此处，继续向左下角移动视频画面，让**第三张图片**出现在画面中央。
10. 重复步骤7到9，即“移动10帧 -> 添加关键帧 -> 移动1秒 -> 移动画面”这个循环，直到所有照片都依次滑过画面。
11. 在视频的结尾处，最后移动画面，让**所有图片**都移出画面之外。

通过以上步骤，我们就完成了照片依次滑入又滑出的动态效果。

## 添加音效与总结
动态效果完成后，最后一步是添加氛围音效和背景音乐，让视频更加生动。

1.  点击“音频”，选择“音效”。
2.  在音效库中搜索并添加“雨滴”声音效果。
3.  将雨滴音效的时长拉长，与整个视频的时长对齐。
4.  再次点击“音频”，选择“抖音收藏”或其他方式，添加你喜欢的背景歌曲。

至此，整个“雨滴移动相册跟踪效果”就制作完成了。你可以点击播放预览最终效果。

本节课中我们一起学习了如何利用画中画、边框特效、关键帧动画以及音效，制作一个富有动感的雨滴移动相册。关键步骤在于先合成静态网格相册并导出，再通过关键帧控制其移动路径，从而创造出照片依次跟踪出现的视觉效果。
