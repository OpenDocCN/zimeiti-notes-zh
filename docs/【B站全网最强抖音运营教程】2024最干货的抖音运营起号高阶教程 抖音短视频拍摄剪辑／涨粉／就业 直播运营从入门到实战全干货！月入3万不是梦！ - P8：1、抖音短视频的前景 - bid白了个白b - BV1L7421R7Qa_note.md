# 抖音运营教程：P8：1、抖音短视频的前景 📈

![](img/e04b7277a0512c913569f63d37f2dc8b_1.png)

在本节课中，我们将要学习抖音短视频平台的发展前景，分析其市场潜力与机遇，并明确作为创作者入局的核心优势。

![](img/e04b7277a0512c913569f63d37f2dc8b_3.png)

了解抖音短视频的前景，需要先通过一张图片来认识。

![](img/e04b7277a0512c913569f63d37f2dc8b_1.png)

这张图片是抖音创作者大会的一份数据报告。报告显示，有超过2200万人在抖音的合计收入超过了417亿元。抖音希望将这个数字翻一番，让创作者的总收入达到800亿元。

![](img/e04b7277a0512c913569f63d37f2dc8b_3.png)

这体现了平台巨大的流量扶持力度。同时，我们需要明确第一个知识点：抖音用户主要分为两类。以下是这两类人的定义：

*   **玩家**：指单纯在抖音上娱乐，没有计划通过平台赚钱的用户。
*   **创作者**：指计划通过自己制作抖音内容来获取收益的用户。本课程的学员都属于这一类。

上一节我们区分了用户类型，本节中我们来看看为何选择抖音平台。在众多短视频平台中，为何要专注于抖音？这类似于开设实体店铺。选择店面位置时，一定会寻找**流量大**且**适合经营**的地点。运营短视频同理，必须选择一个流量庞大的平台。平台自身缺乏流量，便无法将流量分配给我们，我们也难以获得收益。

抖音的日活跃用户超过8亿，月活跃用户更是达到40亿人次。这是一个极其庞大的流量池。流量巨大，竞争压力是否也会很大？答案是否定的。平台上的创作者数量约为5000万。这里的“创作者”定义广泛，并非特指网红达人。例如，邻居家的老太太用手机拍摄自己做饭的视频并上传到抖音，她也算作创作者。因此，你90%的竞争压力，实际上来自于身边的普通用户。

接着，我们来探讨做抖音短视频的硬件门槛。做抖音短视频是否需要购买电脑或专业摄像机？答案是不需要。你只需要一部**智能手机**和一个**手机号**，就可以注册抖音账号。抖音倡导全民通过短视频的形式分享和记录生活，入门非常简单。

那么，当前平台有哪些值得我们关注的风口呢？首先，抖音官方有一项计划：**抖音将帮助1000名优质的艺术创作者，实现粉丝增长超过100万**。有的同学可能会担心自己不会创作。这完全不必担心，正因为我们有所欠缺，才需要系统学习。虚心向比自己专业的人请教并不可耻，这能让我们少走很多弯路。学习的态度至关重要。

市场空间巨大，竞争环境相对宽松，行业内容产出门槛较低。我们正处于大数据时代。在这个时代，平台会根据用户的喜好推荐内容。你喜欢看什么，抖音就会持续给你推荐什么。这意味着，只要我们发布的内容不触犯法律和道德底线，不是低俗信息，**总会有一部分人喜欢你的内容**。一定会有人欣赏你。

为什么老师肯定地说会有人喜欢你，而你却只有几十个粉丝呢？原因在于，喜欢你内容的那部分人，可能还没有刷到过你的作品。他们无法看到，自然无法成为你的粉丝。

因此，对于新手小白而言，大家都迫切希望涨粉的心情可以理解。但说句实在话，不要把“涨粉”作为首要目标。我们的第一个目标应该是：**先把流量（播放量）做起来**。当你的视频播放量提升后，粉丝自然会随之增长。
