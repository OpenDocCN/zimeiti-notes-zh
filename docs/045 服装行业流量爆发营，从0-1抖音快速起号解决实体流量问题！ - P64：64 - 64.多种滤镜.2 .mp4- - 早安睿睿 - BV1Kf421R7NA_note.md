# 剪映教程：045：64 - 多种滤镜应用指南

在本节课中，我们将学习如何使用剪映软件中的滤镜功能，为视频画面增添丰富的视觉效果，从而提升视频的整体质感。

![](img/7bf9447f1dc762e46fb9287c20782fac_0.png)

## 滤镜功能概述

滤镜是视频剪辑中常用的美化工具，它能快速改变画面的色调、风格和氛围。上一节我们介绍了视频的基础剪辑，本节中我们来看看如何通过滤镜为视频增色。

## 滤镜的添加与调整

首先，在剪映中导入一段视频素材。点击下方工具栏中的“滤镜”按钮，软件会展示一个丰富的滤镜库，你可以根据视频内容自由选择适合的风格。

选定滤镜后，可以通过滑动条来**控制滤镜的强度**，数值范围通常为0到100。例如，你可以将强度设置为 `强度: 75`，以获得适中的效果。点击“确定”应用。

## 滤镜的范围与叠加

剪映允许你精确控制滤镜的作用范围。在时间线上，拖动滤镜效果条的两端，可以**调整其应用的起始和结束时间**。

更强大的是，软件支持添加多个滤镜。点击轨道空白处，选择“新增滤镜”，即可叠加另一个滤镜效果，并分别设置各自的强度。如果对某个滤镜不满意，可以选中后点击“删除”。

## 滤镜的适用场景

![](img/7bf9447f1dc762e46fb9287c20782fac_2.png)

那么，在什么情况下应该使用滤镜呢？以下是几个典型的应用场景：

*   **区分回忆片段**：为闪回或回忆的片段添加特定滤镜（如黑白、褪色），能很好地将它与当前时间线的视频区分开来。
*   **丰富素材呈现**：对同一段素材应用不同的滤镜，可以实现**多种风格的呈现方式**，增加视频的变化性。
*   **美化瑕疵画面**：对于光线不足、色彩平淡或存在轻微瑕疵的视频，合适的滤镜能起到显著的**画面美化与修正效果**。
