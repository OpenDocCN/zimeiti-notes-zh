# 小红书运营：P35：最新视频剪辑2-2

## 概述
在本节课中，我们将学习一个完整的小红书视频剪辑流程。我们将从素材编辑开始，逐步完成调色、添加特效、抽帧、添加画中画与贴纸，最后为视频添加动态文字和配音。这套方法旨在通过精细处理，提升视频的原创度和平台推荐概率。

## 视频片段编辑

上一节我们完成了视频素材的导入与初步分割。本节中，我们来看看如何对每个片段进行精细化编辑，使其更具原创性。

编辑操作位于选中视频片段后，向右滑动出现的菜单中。以下是核心的编辑方法：

*   **镜像翻转**：在编辑菜单中，使用“左右镜像”功能可以翻转画面。需注意翻转后画面是否美观，若影响观感则不必使用。
*   **画面缩放**：使用双指在预览画面直接 pinch（捏合）操作，可以放大或缩小视频画面，改变原有构图。
*   **角度旋转**：在编辑菜单中，使用“旋转”功能，可以将画面向左或向右轻微扭转1-2度。

**编辑的核心原则是组合使用上述功能**。每个片段都应进行以下三种组合之一的操作：
1.  画面放大 + 角度旋转。
2.  画面放大 + 镜像翻转。
3.  角度旋转 + 镜像翻转。

**注意事项**：如果片段中包含文字，镜像翻转可能导致文字反向。此时可以不使用镜像，转而采用“放大+旋转”的组合。若文字区域影响不大，也可直接裁切掉该部分画面。

完成所有片段的上述编辑后，视频的视觉呈现将与原素材产生显著差异。

## 视频调色

编辑完画面后，下一步是为视频统一调色，增强质感。点击底部工具栏的“调节”进入调色面板。

调色时，应优先调整影响画面清晰度和色彩层次的核心参数。以下是推荐的参数调整范围：

*   **亮度**：通常保持不动，原素材亮度一般已合适。
*   **对比度**：增加 **3~5**。提高对比度能让色彩更鲜明。
*   **饱和度**：增加 **1~3**。轻微提升让色彩更饱满。
*   **锐化**：增加 **20~30**。这是关键步骤，能显著提升画面清晰度和细节。
*   **颗粒**：增加 **20~30**。添加适量胶片颗粒感，能增加视频质感。
*   **色温/色调**：可轻微调整 **1~2**，或不调整。

**重要提示**：调节完成后，务必在时间轴上将调节效果的长度拉满，使其与整个视频时长一致。

关于滤镜，非必要不使用。如需使用，建议选择“德古拉”等风格化滤镜，并将强度控制在 **10%** 左右，避免过度影响原色调。

## 添加转场特效

调色完成后，可以在各个视频片段之间添加转场特效，使过渡更自然。点击两个片段之间的白色竖杠（转场按钮）即可添加。

以下是添加特效的要点：

*   **特效选择**：建议只使用“闪白”或“闪黑”这类简洁的特效，避免花哨效果影响观感。
*   **时长控制**：特效时长通常设置为 **0.2秒**。如果前后视频片段本身很长（十几秒以上），可适当延长至1-2秒。
*   **应用范围**：
    *   若想所有转场使用同一种特效，点击“全局应用”。
    *   若想为不同转场设置不同特效，则分别选择并“确认”即可。

## 视频抽帧处理

添加特效后，需要进行抽帧操作。这是提升视频系统识别原创性的重要技巧。

**抽帧规则**：视频片段时长每满 **2秒**，则抽掉1帧；满 **4秒**，则抽掉2帧。时长低于2秒的片段无需抽帧。

**操作步骤**：
1.  选中一个需要抽帧的片段。
2.  双指在时间轴上将其放大到最大刻度，以便精细操作。
3.  在片段时间线的任意位置，点击“分割”。
4.  将分割点前的一小段视频向左微移，或将分割点后的一小段视频向右微移，使中间产生一帧的微小空隙。
5.  最后将所有片段顺序拉紧，消除因抽帧产生的间隙。

**务必重视**：抽帧是防止视频被平台判定为重复搬运的关键步骤，需要耐心为每个符合条件的片段执行此操作。

## 添加画中画

抽帧完成后，我们通过添加画中画来进一步增加视频的复杂性和原创信息。

需要添加两个画中画图层：

1.  **现场实拍图片**：用手机现场随意拍摄一张照片导入。
2.  **个人元素**：可以是一段自拍视频或一张个人照片。

**操作步骤**：
*   点击底部工具栏“画中画”，新增画中画图层，导入相应素材。
*   将画中画素材放大至与主视频画面一致，并将其时长拉至与主视频结尾对齐。
*   选中画中画图层，点击“混合模式”，将“不透明度”调整为 **1%**。这样人眼几乎看不见，但系统能检测到该图层信息。

## 添加贴纸

添加画中画后，继续添加贴纸元素。点击底部工具栏“贴纸”，添加4个不同的贴纸。

**操作要点**：
1.  将每个贴纸的时长都拉满，与主视频结尾对齐。
2.  将4个贴纸分别移动到视频画面的四个角。
3.  使用双指将每个贴纸**缩小到肉眼几乎看不见的程度**。
4.  如果贴纸太小无法选中，可以点击时间轴上该贴纸的轨道进行选中和调整。

## 添加动态文字与关键帧

这是最后一步，也是制造动态信息差的关键。点击底部工具栏“文本”，新建文本。

**操作步骤**：
1.  输入你想添加的文字，例如产品卖点、店铺名或任意句子。
2.  点击“样式”，找到“不透明度”，将其设置为 **1%-2%**，使文字在画面上不可见。
3.  开始添加关键帧：
    *   **手动打关键帧**：在文本轨道开头点击“添加关键帧”（菱形加号图标），然后移动时间线，稍微拖动文本改变其位置，系统会自动生成新的关键帧。重复此过程，在视频中设置多个关键帧。
    *   **快速打关键帧技巧**：一只手按住文本并随意拖动，另一只手同时不断点击“添加关键帧”按钮，可以快速生成大量运动轨迹。
4.  文字会随着关键帧的设置而隐形移动，为视频注入独特的动态信息。

## 字幕识别与配音

如果原视频有台词或你想添加配音，可以按以下方法操作：

*   **识别字幕**：如果素材本身有说话内容，点击“文本”->“识别字幕”，系统可自动生成字幕，你只需调整样式。
*   **添加配音**：
    1.  点击“文本”->“新建文本”，输入你想配音的文案。
    2.  选中该文本，点击“文本朗读”。
    3.  在出现的音色列表中选择一个喜欢的声音（如“四郎”），系统便会自动生成该段文字的语音。将生成的音频与原视频对齐即可。

## 导出视频

完成所有步骤后，即可导出视频。点击右上角“导出”按钮，在设置中选择分辨率 **1080P**，帧率 **30帧**，然后导出。

## 总结

本节课我们一起学习了小红书视频混剪的完整流程。我们首先对每个视频片段进行了编辑（镜像、缩放、旋转），然后进行了统一调色（侧重锐化、对比度、颗粒）。接着，我们添加了转场特效，并执行了重要的抽帧操作。之后，通过添加隐形的画中画、贴纸和动态关键帧文字，大幅提升了视频的原创复杂度。最后，我们还学习了如何添加字幕和配音。请记住，细节决定成败，严格按照步骤操作并保持耐心，是制作出能获得平台推荐视频的关键。
