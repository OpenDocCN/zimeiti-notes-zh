# 微信视频号运营教程：P2：视频号分成计划 🎬

在本节课中，我们将学习如何快速开通微信视频号的“分成计划”并获取收益。课程将涵盖开通条件、具体操作流程、收益计算方式，并重点介绍两个高潜力赛道。

![](img/e6ac1abb8c3c440daa98ba430916e15d_1.png)

## 概述

视频号分成计划是创作者通过发布原创视频，在视频评论区展示广告并获得分成收益的机制。本节教程将提供一个实测有效的快速开通方法，并分享两种当前流量巨大的内容赛道。

![](img/e6ac1abb8c3c440daa98ba430916e15d_3.png)

## 开通条件与快速起号方法

上一节我们介绍了视频号的基础注册流程。本节中我们来看看如何快速开通分成计划。

开通视频号分成计划需要满足两个核心条件：
1.  **粉丝数达到100以上**。
2.  **内容符合平台规范**，即优质原创、画风精美。

过去，新号发布10条风景类视频即可通过原创审核。但现在，风景类内容已很难通过。以下是经过实测的快速起号方法：

**具体操作步骤如下：**
1.  注册新视频号后，直接进行“怼脸拍”（即人物正面出镜拍摄）。
2.  确保每条视频时长超过9秒。
3.  每天发布5条此类视频。
4.  持续操作到第三天，即可申请开通分成计划。

![](img/e6ac1abb8c3c440daa98ba430916e15d_1.png)

通过此方法，账号前期收益可能较低，但坚持发布，收益会逐步提升。如下图所示，一个账号在16号后收益显著增长。

![](img/e6ac1abb8c3c440daa98ba430916e15d_3.png)

## 分成计划开通与收益核心要点

![](img/e6ac1abb8c3c440daa98ba430916e15d_5.png)

成功起号后，下一步是正式开通分成计划并理解收益规则。

![](img/e6ac1abb8c3c440daa98ba430916e15d_6.png)

**开通流程如下：**
1.  在视频号中找到“分成计划”入口。
2.  在发布满10条“怼脸拍”视频后的第三天，点击申请加入。
3.  **关键步骤**：此后发布的每一条原创视频，都必须勾选“原创声明”。若不开启此声明，即使视频获得千万播放，也无法获得任何收益。

![](img/e6ac1abb8c3c440daa98ba430916e15d_5.png)
![](img/e6ac1abb8c3c440daa98ba430916e15d_6.png)

**关于收益计算：**
平台未公布明确公式，但根据经验，收益大致范围可参考：
`收益 ≈ 视频播放量 × (1~3元/万播放)`
实际收益主要取决于**广告点击率**。即使播放量高，若无人点击视频下方的广告，收益也不会理想。广告分成受视频质量、播放量、发布数量和广告行业选择等多因素影响。

**因此，制作视频时需想方设法提升评论区活跃度，激励用户互动和点击广告。**

## 推荐内容赛道（一）：人性语录类

了解基础操作后，我们来看看具体做什么内容。以下是两个经过验证的高流量赛道。

首先介绍“人性语录类”。目前入局视频号，不推荐从竞争激烈的“祝福类”内容开始。人性领域是除祝福类外最火的领域之一。

它与文案号不同，核心在于用一句话或一段话直指人性本质。这类内容吸引的粉丝质量较高，年龄多在30岁以上，因此收益潜力也较大。

以下是制作人性语录类视频的完整流程框架，按此步骤操作可复现成功案例：

1.  **获取文案与素材**：从特定渠道收集直击人心的语录文本和配套画面素材。
2.  **文案配音**：使用配音软件或工具，将文案转化为音频。
3.  **视频制作**：将音频、素材画面、文字字幕进行剪辑合成。

![](img/e6ac1abb8c3c440daa98ba430916e15d_9.png)

![](img/e6ac1abb8c3c440daa98ba430916e15d_8.png)

**重要执行建议：**
*   **坚持发布**：开通原创后，切勿因前几天收益低而放弃。如上图收益记录所示，需要持续发布一段时间（如7-10天）才能迎来收益增长。
*   **控制频率**：每个账号每天发布3-5个视频为宜，**切忌超过5个**。发布过多可能导致账号权重下降。

## 推荐内容赛道（二）：文案鸡汤类

接下来看第二个赛道：“文案鸡汤类”。这类内容在所有内容平台都经久不衰。

其制作流程与人性语录类类似，核心环节包括：
1.  **获取素材**：收集励志、情感类文案和匹配的图片/视频素材。
2.  **搭配音乐与配音**：为文案配上合适的背景音乐或人工配音。
3.  **选择内容模板**：使用固定的图文或短视频模板进行制作。
4.  **合成输出**：生成完整的视频内容。

## （附加）电视剧解说搬运玩法

除了原创赛道，还有一种基于工具的效率化玩法：电视剧解说视频搬运。

该玩法的逻辑是：利用“分成计划”，通过高播放视频带动广告点击获利。电视剧解说类视频通常传播率和互动率都很高。

**操作步骤如下：**
1.  **寻找素材**：从其他视频平台（如某瓜、某音）下载热门的电视剧解说视频。
2.  **批量去重**：使用专门的去重软件对下载的视频进行批量处理，以通过平台原创审核。
3.  **发布变现**：将处理后的视频发布到视频号，依靠创作分成变现。
