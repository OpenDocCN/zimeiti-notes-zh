# 抖音短视频运营教程：P5：3 抖音选题思路技巧 🎯

在本节课中，我们将学习一套高效的抖音选题公式。这套方法能帮助你快速确定内容方向，解决“不知道发什么”的难题，从而更高效地产出吸引目标用户的视频。

## 选题的重要性 📈

![](img/dd7f3968d81b140ddcfb07bf3b6112a6_1.png)

上一节我们介绍了账号定位，本节我们来看看选题的具体思路。不会选题会导致内容创作效率低下。

会找选题的人，能有更多时间专注于视频制作。每天发布三条视频，一个月就能积累90条内容。依靠数量优势就能在竞争中脱颖而出，甚至无需依赖付费推广。此外，擅长选题的人能更精准地找到用户痛点，也更容易吸引精准粉丝观看，从而快速为账号打上标签。

## 核心选题公式 🔑

那么，有什么方法可以解决选题困难呢？答案是有的。这是一套百万粉丝博主常用的选题技巧。为了方便理解，我们将其总结为一个公式：

![](img/dd7f3968d81b140ddcfb07bf3b6112a6_3.png)

**选题 = 内容定位 + 用户痛点 + 借势热点**

用好这个公式，你的内容方向就不会跑偏。

## 公式应用详解 🛠️

![](img/dd7f3968d81b140ddcfb07bf3b6112a6_5.png)

下面，我们将通过具体例子来拆解这个公式的每个部分。

### 1. 内容定位
内容定位是你的账号所属领域或行业的核心关键词。这些关键词是客观的，不包含主观描述。
*   对于知识博主：关键词可能是 **`播放量`**、**`粉丝量`**、**`定位`**、**`选题`**。
*   对于美妆博主：关键词可能是 **`美白`**、**`除皱`**、**`口红`**。

### 2. 用户痛点
用户痛点是你的目标受众在该领域最关心、最亟待解决的问题或需求。
*   知识领域用户痛点：**`便捷`**、**`高效`**、**`技巧教学`**、**`内容输出`**。
*   美妆领域用户痛点：**`平价好用`**、**`明星同款`**、**`避坑扫雷`**。

### 3. 借势热点
借势热点是指结合当前的社会话题、节日、流行趋势或明星事件。
*   例如，临近春节，热点就是“**`春节`**”。
*   其他热点可能包括“**`七夕`**”、“**`迪丽热巴同款`**”等。

## 公式组合实战 💡

现在，我们将以上三个部分进行排列组合，就能生成具体的选题。以下是应用示例：

**案例一：知识博主**
*   **组合示例1**：`选题`（定位） + `高效`（痛点） + `春节`（热点） = 《如何用拜年的方式，高效策划春节必火选题》
*   **组合示例2**：`涨粉`（定位） + `技巧教学`（痛点） + `张同学`（热点） = 《张同学快速涨粉的三个秘诀》

**案例二：美妆博主**
*   **组合示例1**：`美白面膜`（定位） + `明星同款`（痛点） + `迪丽热巴`（热点） = 《迪丽热巴同款美白面膜推荐》
*   **组合示例2**：`口红`（定位） + `避坑扫雷`（痛点） + `七夕`（热点） = 《七夕千万别送女朋友这几支口红！》
