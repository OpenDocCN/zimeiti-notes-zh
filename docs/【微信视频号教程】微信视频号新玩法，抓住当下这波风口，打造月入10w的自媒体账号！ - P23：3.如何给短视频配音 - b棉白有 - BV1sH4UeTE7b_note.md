# 微信视频号教程：P23：如何给短视频配音 🎙️

在本节课中，我们将学习如何为短视频进行配音。有时，原始视频的配音效果不佳，我们可以将其替换。配音过程并不复杂，我们将介绍实用的工具和具体操作步骤。

## 配音工具介绍

上一节我们介绍了视频处理的基本概念，本节中我们来看看配音需要用到的核心工具。配音工具种类繁多，其中“配音神器”是一个功能全面的选择。

![](img/f9b71dee60ca1528cccb1ecd52172982_1.png)

以下是“配音神器”小程序的主要功能：
*   **配音**：输入文案脚本，即可生成语音。
*   **文案提取**：从视频中提取文字内容。
*   **视频去水印**：去除视频中的水印标识。
*   **修改视频MD5**：改变视频文件的唯一识别码，增加视频的原创度。

> **核心概念**：**MD5** 是视频文件的“数字指纹”。修改它可以使平台系统将其识别为一个“新”文件，有助于通过原创审核。公式表示为：`新视频文件 = 修改MD5(原视频文件)`。

![](img/f9b71dee60ca1528cccb1ecd52172982_3.png)

除了“配音神器”，在微信中直接搜索关键词，也能找到大量类似工具。

![](img/f9b71dee60ca1528cccb1ecd52172982_5.png)

## 如何搜索与选择工具

![](img/f9b71dee60ca1528cccb1ecd52172982_7.png)

了解了核心工具后，我们来看看如何寻找和选择它们。所有相关工具几乎都能在微信内找到。

以下是搜索与使用工具的步骤：
1.  打开微信，点击右上角的搜索图标。
2.  输入关键词，如“去水印”、“修改MD5”或“配音”。
3.  在搜索结果中选择“小程序”分类。
4.  从列表中选择一个工具使用即可。

我推荐使用“配音神器”或“青抖”。如果某个工具暂时无法使用，更换另一个即可。“配音神器”的优点是，除配音功能外，其他如去水印、文案提取等功能均免费。

## 使用“配音神器”进行配音

现在，我们进入实际操作环节，学习如何使用“配音神器”生成配音。

以下是使用“配音神器”生成配音的详细步骤：
1.  打开“配音神器”小程序，点击“配音”功能。
2.  清除默认文字，将你准备好的文案脚本粘贴或输入到文本框中。
3.  在下方选择你喜欢的声音类型（如女声、男声等），可以点击试听。
4.  选择好声音后，点击“合成配音”。
5.  合成完成后，选择“保存配音视频到相册”或“保存字幕视频到相册”。两者区别在于后者会生成带字幕的视频。
6.  等待保存完成，不要中途退出。

![](img/f9b71dee60ca1528cccb1ecd52172982_9.png)

## 在剪映中替换视频原声

生成配音文件后，最后一步是在视频剪辑软件中替换原声。这里以“剪映”为例进行说明。

![](img/f9b71dee60ca1528cccb1ecd52172982_11.png)

以下是在剪映中替换视频原声的步骤：
1.  打开剪映，导入需要替换声音的视频素材。
2.  选中视频轨道，点击“关闭原声”。
3.  点击底部工具栏的“音频”，然后选择“提取音乐”。
4.  从手机相册中选择刚刚保存好的配音视频文件。
5.  导入后，配音音频就会添加到时间轴上，完成原声替换。

> **操作演示**：关闭原声后，通过 `音频 -> 提取音乐 -> 选择配音文件` 的路径，即可用新配音覆盖视频原有声音。

本节课中我们一起学习了为短视频配音的全流程。我们介绍了“配音神器”这个核心工具，演示了如何搜索并使用它来生成配音，最后讲解了如何在剪映中完成音频替换。掌握这些步骤，你就能自由地为视频更换更合适的配音了。
