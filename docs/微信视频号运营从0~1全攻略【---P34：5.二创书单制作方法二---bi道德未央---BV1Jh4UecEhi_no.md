# 微信视频号运营从0~1全攻略：P34：5.二创书单制作方法二

在本节课中，我们将学习如何通过“二创”的方式制作书单视频。我们将详细讲解如何提取原视频素材、替换画面、重新配音，最终生成一个符合平台规则的原创视频。

![](img/d3e3320f847b6e1f97680437e5081d0a_1.png)

## 概述
上一节我们介绍了二创书单的基本思路，本节中我们来看看具体的操作步骤。我们将以《走进大学城》这本书为例，演示如何从抖音寻找素材，并通过剪辑软件进行二次创作。

## 操作步骤详解

![](img/d3e3320f847b6e1f97680437e5081d0a_7.png)

![](img/d3e3320f847b6e1f97680437e5081d0a_9.png)

### 1. 寻找并下载原始素材
首先，我们需要找到合适的原始视频素材。以下是具体步骤。

![](img/d3e3320f847b6e1f97680437e5081d0a_11.png)

*   在抖音搜索目标书籍名称，例如“走进大学城”。
*   在搜索结果中，选择“商品”标签，并按“销量”排序。
*   从商品关联的达人视频中，寻找内容优质、时长合适的视频作为素材源。

### 2. 提取所需视频片段
找到目标视频后，我们需要将其下载并导入剪辑软件，提取我们需要的部分。

![](img/d3e3320f847b6e1f97680437e5081d0a_13.png)

*   复制视频链接，使用去水印工具下载无水印版本。
*   打开剪映，导入下载好的视频。
*   如果原视频前后声音不一致（例如前半段是旁白，后半段是真人出镜讲解），可以在声音变化的时间点进行分割。例如，在53秒处分割，保留前半部分的文案内容。
*   将分割后的前半部分视频导出，备用。

### 3. 提取文案并重新配音
接下来，我们需要提取视频中的文案，并生成新的配音，这是实现“二创”的关键。

*   使用“配音神器”等工具，通过“视频提取文案”功能，从刚才导出的视频片段中提取出文字内容。
*   在配音神器中选择一个适合的声音（如成熟稳重的女声），将提取的文案生成新的音频文件并导出。
*   **小提示**：如果账号可用次数有限，记得每日签到领取免费额度。

### 4. 替换视频画面素材
现在，我们要对原视频的画面进行替换，只保留其内容结构和节奏。以下是替换画面的方法。

*   回到剪映中原来的工程文件，关闭原视频的音量。
*   逐帧分析原视频，将其中具有明显特征或可能涉及版权的画面片段分割出来。例如，一段3秒的“大学生毕业典礼”画面。
*   在抖音搜索与分割片段内容相关的关键词，如“高校 毕业典礼”，下载新的无水印视频素材。
*   在剪映中，删除原视频中分割出来的那3秒片段，将新下载的3秒素材导入并放置在同一位置。
*   在新旧片段之间添加一个转场效果（如“叠化”），使过渡更自然。
*   确保新素材的时长与原片段完全一致，以保持音频同步。
*   重复此过程，替换1-2个核心画面。例如，将“打高尔夫”的画面替换为搜索到的“美女打高尔夫”视频；将“宿舍单间”画面替换为其他“精装修室内”视频。

### 5. 合成音频与调整对齐
画面替换完成后，导入新生成的配音，并进行音画对齐的精细调整。

*   将之前用配音神器生成的新音频导入剪映，添加到时间轴。
*   播放视频，检查新的音频是否与替换后的画面长度匹配。
*   **如果音频长，视频短**：可以适当延长某些画面片段，或为音频对应的部分添加更多相关素材。
*   **如果音频短，视频长**：可以适当缩短画面，或对视频进行“常规变速”处理，加快播放速度。
*   目标是让音频结束时，视频画面也基本结束，多余的部分可以删除。

### 6. 处理真人出镜片段
对于原视频中真人出镜讲解的部分，可以采用“保留音频，替换画面”的策略。

*   保留原视频后半段真人讲解的**音频**。
*   将其对应的**画面**替换为从抖音搜索到的、内容意境相似的其他视频素材（如“成熟女性讲话”素材库）。
*   **注意**：尽量减少同一人物正面特写镜头的时长，以避免可能的人脸识别风险。

### 7. 添加字幕与背景音乐
最后，为视频添加新的字幕和背景音乐，完成最终包装。

*   使用剪映的“识别字幕”功能，为新的配音自动生成字幕。
*   调整新字幕的样式和位置，确保其完全覆盖掉原视频可能残留的旧字幕。
*   在“音频”中添加一段合适的背景音乐，并调低音量，使其不干扰主配音。
*   **小提示**：剪映会员能提供更多字体和功能，建议开通。

### 8. 最终检查与导出
在导出前，进行最后的质量检查。

*   检查视频中是否完全去除了“抖音”水印。如有残留，可通过放大画面或添加贴纸进行遮盖。
*   确认所有替换素材的衔接是否自然，音画是否同步。
*   导出设置选择：分辨率1080P，帧率60帧，码率推荐即可。

## 总结
本节课我们一起学习了二创书单视频的完整制作流程。核心在于**提取原视频文案，生成新配音**，并**替换关键画面素材**。通过这种方法，我们可以高效地产出符合平台原创要求的视频。操作本身并不复杂，关键在于多练习。从第一个视频可能需要一两小时，到熟练后十分钟完成一个，全凭熟能生巧。请务必动手实操，积累经验。
