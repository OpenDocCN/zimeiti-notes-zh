在本节课中，我们将要学习剪映电脑版中“滤镜”和“调节”面板的核心功能。我们将从认识调色面板开始，逐步学习一级调色和二级调色的概念与操作方法，最终完成一个具有电影感蓝灰色调的案例。

## 认识调色面板

在开始调色之前，我们首先需要了解剪映中的调色工具。调色主要分为两个步骤：一级调色（颜色校正与还原）和二级调色（风格化处理）。我们将使用“滤镜”和“自定义调节”功能来完成这些操作。

### 滤镜功能

滤镜是预设的调色模板，适用于对色彩敏感度不高的用户，可以快速为视频应用不同的风格。

以下是滤镜的主要分类：

*   **电影级**：模拟好莱坞电影色调，如青橙色、蓝灰调、黑金调。
*   **人像**：专门优化人物肤色，可能包含磨皮、肤色提亮等效果。
*   **基础**：适用于各种场景的通用滤镜，如旷野蓝、室内、黑白等。
*   **复古胶片**：增加颗粒感和怀旧色彩。
*   **美食/风景/夜景**：针对特定主题优化的色彩风格。

**注意**：直接套用滤镜有时效果不佳，通常需要先进行一级调色来校正视频原片的色彩问题。

![](img/784aae209c662e1da1f13fb23e59d90f_4.png)

### 自定义调节

如果滤镜效果不理想，或者想要更精细地控制颜色，就需要使用“自定义调节”功能。将“自定义调节”效果拖到视频轨道上，即可在右侧打开调节面板。

调节面板主要包含四个核心板块：

1.  **基础**：调整画面整体的明暗、对比和色彩倾向。
2.  **HSL**：针对画面中特定的颜色（红、橙、黄、绿…）进行独立调整。
3.  **曲线**：通过曲线工具精细控制画面的亮度和色彩通道。
4.  **色轮**：分别调整画面暗部、中间调、亮部的色彩和饱和度。

上一节我们介绍了调色的基本工具，本节中我们来看看如何进行具体的一级调色操作。

## 一级调色：基础校正

一级调色的目标是校正视频素材的原始色彩问题，例如偏色、灰暗或过曝，使画面色彩恢复正常，为后续风格化调色打下良好基础。

![](img/784aae209c662e1da1f13fb23e59d90f_6.png)

我们将通过“基础”调节面板中的参数来实现。以下是各核心参数的作用：

*   **色温**：控制画面整体偏蓝（冷调）或偏黄（暖调）。
    *   **公式**：`色温值 < 0` 偏蓝， `色温值 > 0` 偏黄。
*   **色调**：平衡色温，控制画面偏绿或偏品红（紫），常用于修正肤色。
*   **饱和度**：控制颜色的鲜艳程度。数值为0时，画面变为灰色，而非黑白。
    *   **公式**：`饱和度 = 颜色浓度`
*   **亮度**：如同为整个场景开灯，整体提亮或压暗画面。
*   **对比度**：调整画面最亮和最暗部分的差异。降低对比度会使画面变灰。
*   **高光**：仅调整画面中最亮的部分（如天空、反光面）。
*   **阴影**：仅调整画面中较暗的部分（如背光面、暗部细节）。
*   **光感**：智能调整画面中主体（如人物）的明亮感，效果更自然。
*   **锐化**：提高画面边缘清晰度，让细节更突出。但不宜过度，否则会产生噪点。
*   **颗粒**：为画面添加胶片颗粒感，营造复古氛围。
*   **褪色**：降低整体色彩浓度，营造清淡、复古的色调。
*   **暗角**：使画面四角变暗或变亮，从而突出视觉中心的主体。

**操作示例**：针对一个略显灰暗、肤色不透亮的人物镜头，我们可以进行如下校正：
1.  适当增加**对比度**，让画面更清晰。
2.  微调**色温**，校正可能存在的偏色。
3.  提高**光感**和**高光**，让人物皮肤更白皙透亮。
4.  降低**阴影**，增强人物轮廓的立体感。

完成基础校正后，画面色彩会更加干净、正常。接下来，我们可以进入更精细的颜色调整。

## 二级调色：风格化调整

二级调色是在一级校正的基础上，为视频赋予特定的艺术风格或情感色彩。我们将使用HSL、曲线和色轮工具。

### HSL调整

HSL允许你单独调整画面中某一种颜色的属性。

*   **H（色相）**：改变颜色本身（例如，将红色变为橙色或紫色）。
*   **S（饱和度）**：改变该颜色的鲜艳程度。
*   **L（亮度）**：改变该颜色的明暗程度。

**操作示例**：想要人物皮肤更白皙红润。
1.  选择**橙色**（影响肤色）。
2.  将**饱和度**略微降低，肤色会更白皙。
3.  将**亮度**适当提高，肤色会更透亮。
4.  选择**红色**（影响唇色）。
5.  适当增加**饱和度**，让唇色更鲜艳。

### 曲线调整

曲线工具提供了更强大的控制力。RGB曲线控制明暗，而红、绿、蓝曲线则基于互补色原理来调整色彩。

*   **互补色关系**：
    *   红 ←→ 青（Cyan）
    *   绿 ←→ 品红（Magenta）
    *   蓝 ←→ 黄（Yellow）
*   **操作逻辑**：将红色曲线向上拉，画面偏红；向下拉，画面偏青（红色的互补色）。其他颜色曲线同理。

**常用技巧**：在RGB曲线上拉出一个轻微的“S”型，可以同时增加对比度，让画面更通透。

### 色轮调整

色轮可以直观地分别对画面的**阴影**、**中间调**、**高光**三个区域进行色彩偏移和饱和度调整。

*   **阴影/中间调/高光**：分别控制对应明暗区域的色彩。
*   **全局（偏移）**：统一调整整个画面的色彩倾向。
*   **强度**：控制当前色轮调整效果的强弱程度。

**操作示例**：营造电影感色调。
1.  **阴影**色轮：向蓝色轻微偏移，并降低一点亮度，让暗部更沉稳。
2.  **中间调**色轮：向橙色或黄色轻微偏移，增加温暖感或复古感。
3.  **高光**色轮：向青色或蓝色轻微偏移，让亮部更清澈。

完成细致的风格化调整后，我们还可以回到滤镜面板，为视频叠加一个合适的预设滤镜，使风格更突出、效果更统一。

## 案例实战：电影感蓝灰色调

现在，我们将综合运用所学知识，制作课程开头的电影感蓝灰色调案例。

**步骤一：添加调节层并执行一级调色**
1.  在轨道上添加一个“自定义调节”效果，并覆盖需要调色的片段。
2.  在“基础”面板中调整以下参数，校正画面：
    *   亮度：-15
    *   对比度：-40
    *   高光：+30
    *   光感：-10
    *   饱和度：+15
    *   锐化：+40

**步骤二：应用风格化滤镜**
1.  在“滤镜”面板中，找到“电影级”分类。
2.  选择“蓝灰色”滤镜，将其拖拽到视频轨道上。
3.  如果觉得色调浓度不够，可以复制叠加一层“蓝灰色”滤镜，直到达到理想的蓝灰电影质感。

通过对比调色前后的画面，你可以看到色彩从普通变得充满高级感和电影叙事氛围。

## 总结

本节课中我们一起学习了剪映电脑版调色的核心流程：
1.  **认识工具**：了解了“滤镜”和“自定义调节”两大功能模块。
2.  **一级调色**：使用“基础”面板校正画面的曝光、对比度和色彩平衡，为后续调色打好基础。
3.  **二级调色**：深入使用**HSL**、**曲线**和**色轮**工具，对特定颜色和画面区域进行风格化处理，创造独特的视觉风格。
4.  **综合应用**：通过“电影感蓝灰色调”的案例，实践了从校正到风格化的完整调色流程。
