在本节课中，我们将要学习小红书平台的核心推荐逻辑，以及运营过程中必须避免的违规行为。理解这些规则是账号健康发展和获得流量的基础。

上一节我们介绍了小红书整体的推荐与流量规则，本节中我们来看看笔记被推荐的具体核心逻辑，以及需要规避的“坑”。

### 被推荐的核心逻辑

笔记发布后，平台会基于一套算法决定是否给予更多流量。核心逻辑在于通过两个关键数据指标进行层级跨越：**打开率**和**互动率**。

**1. 打开率 (CTR)**
打开率是笔记获得初次曝光后，用户点击进入查看的比例。这是流量升级的第一道门槛。
*   **计算方式**：`打开率 = (笔记点击量 / 笔记展示量) * 100%`
*   **核心阈值**：在初次推荐的20条同类型笔记中，你的笔记打开率需要**超过11%**，才能进入下一流量池。
*   **示例**：如果笔记获得1000次展示，至少需要110次点击（即110个小眼睛），平台才会考虑给予2000-5000次的下一级展示量。若低于此阈值，流量将停留在初始水平。

**2. 互动率 (Engagement Rate)**
互动率是用户点击进入笔记后，进行点赞、评论、收藏、分享等行为的比例。这是流量再次升级的关键。
*   **计算方式**：`互动率 = (互动总量 / 笔记点击量) * 100%`
*   **核心阈值**：平均互动率需要**超过3%**，才能推动流量向更高层级跃进。
*   **示例**：如果笔记有110次点击，那么至少需要4次以上的互动（评论、赞、藏等），才能满足3%的互动率要求，从而有机会从2000-5000展示量级，跃升至1万-2万甚至更高的展示量级。

**流量升级路径总结**：新笔记发布 → 平台给予基础展示量 → **打开率 > 11%** → 进入更大流量池 → **互动率 > 3%** → 进入更高层级流量池。对于新账号，通常有机会升级2-3个层级；老账号因基础权重高，起始流量池更大，有潜力冲击更高层级。

**3. 互动行为权重**
除了打开率和互动率，具体的用户行为也被平台赋予不同权重，影响笔记的最终评分。以下是主要互动行为的权重关系：
`笔记综合评分 = f(点赞, 收藏, 评论, 分享, 关注)`
其中，**分享**和**关注**行为的权重通常高于简单的点赞和收藏。但这一切的前提是笔记的打开率和互动率这两个基础属性必须首先达标。

### 必须避免的运营“坑”

![](img/0c4963b2452729447b863dee59e65e0b_19.png)

了解推荐逻辑后，还需规避以下常见错误，以防账号被降权或限流。

以下是运营中必须注意的禁忌列表：

1.  **新号切勿立即发笔记**：新注册账号需要“养号”，模拟正常用户行为（浏览、互动）一段时间，建立初始权重。立即发笔记会消耗平台给予的初始测试流量，导致后续流量低迷。
2.  **违规必被降权**：任何违规行为（如导流、抄袭、发布违禁内容）都会导致笔记推荐减少、搜索排名下降，甚至账号被限流。一次违规可能导致流量锐减，恢复难度极大。
3.  **标签切忌反复修改**：账号、店铺或笔记的标签一旦确定并产生流量后，避免频繁修改。大幅修改标签可能导致系统重新识别，造成流量中断或下滑。
4.  **设备与账号需一一对应**：尽量避免一台手机登录多个小红书账号，或一个账号在多台设备频繁切换。建议遵循“一机一卡一号”的原则，使用电脑登录也最好不要超过3个账号。
5.  **谨慎处理推广与引流**：笔记内容和私信、评论中，避免出现明显的微信号、电话号码、二维码等站外引流信息。此类行为极易被判定违规。
6.  **谨慎处理用户互动**：对于用户的留言和私信，应积极妥善回复。但需注意，在回复中直接进行硬性广告推销或引流，同样存在风险。
