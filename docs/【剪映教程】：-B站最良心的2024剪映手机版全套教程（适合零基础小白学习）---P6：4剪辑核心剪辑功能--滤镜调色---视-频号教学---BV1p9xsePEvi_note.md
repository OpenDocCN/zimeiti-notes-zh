# 剪映教程：P6：剪辑核心功能——滤镜与调色 🎨

在本节课中，我们将学习剪映的核心剪辑功能之一：滤镜与调色。许多初学者拍摄的视频常常色彩平淡、不够鲜亮。通过本课的学习，你将掌握如何利用剪映的滤镜和调色工具，快速提升视频画面的色彩表现力与整体质感。

## 为什么需要调色？✨

上一节我们介绍了基础的剪辑操作，本节中我们来看看如何优化视频的视觉观感。提升视频质感的一个关键方法就是调色。在专业软件如PR或达芬奇中，调色功能非常复杂。但对于新手而言，剪映提供了简单易用的滤镜和调色工具，足以满足日常的调色需求。

调色主要分为两个部分：
1.  **滤镜**：即预设好的色彩模板，可以一键套用，快速改变视频风格。
2.  **调节**：允许你对视频的色彩参数进行个性化、精细化的调整。

## 实战：使用滤镜快速调色 🎬

![](img/645ee35657b9e1f6cbe27580377da746_4.png)

了解了基本概念后，我们进入实战环节。首先，我们来看如何使用滤镜功能。

1.  在剪映中导入一段素材。观察原素材，你可能会发现天空不够蓝，草木不够绿。
2.  点击底部工具栏的 **“滤镜”** 按钮。
3.  剪映提供了多种风格的滤镜模板，例如“鲜亮”、“复古”、“电影”等。
4.  尝试点击“鲜亮”滤镜，视频的色彩和亮度会立即发生显著变化。你可以通过对比原图，直观感受滤镜的效果。

**代码示例：应用滤镜**
（在剪映中，此过程通过点击界面完成，其逻辑可理解为：）
`video.applyFilter(filterName: “鲜亮”)`

通过套用合适的滤镜，可以快速统一视频风格，这是最便捷的调色方法。

## 进阶：使用调节功能精细调色 ⚙️

如果套用滤镜后，你对某些局部色彩仍不满意，就需要使用“调节”功能进行个性化调整。以下是调节功能的核心参数及其作用：

*   **亮度**：控制画面整体明暗。
*   **对比度**：增加可使亮部更亮、暗部更暗，画面更通透。
*   **饱和度**：控制色彩鲜艳程度。提升饱和度能让天空更蓝、草木更绿。
*   **锐化**：增加画面细节的清晰度，使影像更细腻。
*   **高光**：调整画面中最亮部分的明暗。
*   **阴影**：调整画面中最暗部分的明暗。
*   **色温**：调整画面冷暖色调。偏蓝为冷，偏黄为暖。
*   **色调**：调整画面整体色彩倾向。

**操作步骤：**
1.  点击底部工具栏的 **“调节”** 按钮。
2.  根据你的需求，逐一调整上述参数。例如，想让天空更蓝、树木更绿，可以适当**提升饱和度**和**对比度**，并微调**色温**向冷色调偏移。
3.  调整过程中，可以随时播放预览，对比调整前后的效果。

**公式示例：目标色彩调整**
`最终效果 = 基础画面 + 滤镜(风格) + 调节(亮度, 对比度, 饱和度, ...)`

通过“调节”功能，你可以针对视频的特定色彩问题进行精准优化，无论是风景、人物还是美食视频，都能找到合适的调整方案。
