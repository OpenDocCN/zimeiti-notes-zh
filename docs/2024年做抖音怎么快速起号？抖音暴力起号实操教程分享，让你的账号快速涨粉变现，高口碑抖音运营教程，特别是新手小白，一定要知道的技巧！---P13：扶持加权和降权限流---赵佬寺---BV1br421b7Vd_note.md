# 抖音运营教程：P13：扶持加权与降权限流

在本节课中，我们将学习抖音平台对视频内容的两种核心处理机制：扶持加权与降权限流。理解这些规则，能帮助你有效提升视频曝光，避免违规导致流量受限。

上一节我们介绍了账号权重的分类，本节中我们来看看具体哪些内容会受到平台的扶持，哪些内容又会被降权。

![](img/549dcfb13b86378bf42508a42682deab_1.png)

## 🚀 扶持加权的内容标准

平台会对符合以下标准的优质原创视频给予流量扶持和权重加成。

![](img/549dcfb13b86378bf42508a42682deab_3.png)

以下是获得扶持加权的关键要求：

*   **视频格式**：必须为竖版移动端原生视频，即 **9:16** 的格式。
*   **画面质量**：画面需保持平稳，作品清晰度高。
*   **音频质量**：无过多的杂音或背景噪音。

![](img/549dcfb13b86378bf42508a42682deab_5.png)

![](img/549dcfb13b86378bf42508a42682deab_1.png)

这些标准较为基础，平台希望鼓励的是高质量的原创内容。关于此部分的详细说明，我会整理成资料页供大家课后查阅。

## ⚠️ 会被降权限流的内容

相反，如果视频内容涉及以下违规点，则会被系统识别并实施降权或限流处理。

以下是主要的违规内容类型：

1.  **不良导向**：内容涉及吸烟、暴力、炫富（金钱）等信息。
    ![](img/549dcfb13b86378bf42508a42682deab_3.png)
2.  **违规引流**：出现或引导至微信、微博、淘宝等外部平台联系方式；长时间展示商品Logo、手机号；二维码展示时间**最长不建议超过3秒**（严格时，出现1秒也可能被驳回）。
3.  **硬性广告**：进行明显的营销广告、推销行为，直接引流卖货。
4.  **无口播的好物推荐**：这是需要特别注意的一点。许多同学在做好物推荐或带货视频时容易触犯。**仅展示产品拆箱或使用过程，而没有口播介绍，极易被判定为营销广告导致小黄车被下架**。

关于第4点，我们展开说明。如果你推荐一款产品（如玩具、纸巾），正确的做法是配合口播进行介绍。例如：

```text
“我儿子玩这个玩具，整整四个小时都没停下来。”
“这张纸沾水后也不会破，可以反复使用。”
```

而错误示范包括：在画面上直接标注“9.9元”，或全程无声仅用背景音乐展示产品。前者直接涉及金钱营销，后者因缺乏口播解读容易被系统判定为纯广告内容。

5.  **直接口播广告与联系方式**：在口播中直接念出广告语或联系方式，这在抖音是行不通的。

本节课中我们一起学习了抖音平台的“扶持加权”与“降权限流”规则。核心在于：制作**9:16竖版高清、无杂音的原生视频**以获得扶持；同时务必避免**硬广、不良信息、无口播带货及违规引流**，以防账号被降权。掌握这些，你的起号之路将更加顺畅。
