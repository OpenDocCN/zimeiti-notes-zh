# 抖音自媒体运营：P29：垂直类目细分领域 🎯

在本节课中，我们将深入探讨账号垂直度的概念，并学习如何通过细分领域来提升内容的精准度和吸引力。我们将从一级垂直领域开始，逐步深入到更精细的三级、四级领域，帮助你理解如何选择并深耕自己的内容方向。

上一节我们介绍了内容标签的重要性，本节中我们来看看如何将“垂直”这个概念进行更细致的划分。

我们经常强调做账号要垂直。很多初学者可能会简单地认为，选择“美食”或“游戏”这样的大类目一直做下去，就是垂直。这个理解没错，但这仅仅是垂直的起点，属于**一级垂直领域**。

为了更精准地吸引目标观众，我们可以在大领域下进行更细致的划分。以下是垂直领域的层级划分示例：

*   **一级垂直领域**：最宽泛的内容大类。
    *   例如：**游戏**、美食、美妆、教育。
*   **二级垂直领域**：在一级领域下的主要分支。
    *   例如（在“游戏”下）：**手游**、电竞、网游、单机游戏。
*   **三级垂直领域**：在二级领域下的具体项目或主题。
    *   例如（在“手游”下）：**《王者荣耀》**、**《和平精英》**、**《原神》**。
*   **四级垂直领域（可选项）**：在三级领域下的极致细分，如特定模式、角色或风格。
    *   例如（在“《王者荣耀》”下）：**专玩“安琪拉”这个英雄**的教学、只做“5V5竞技”模式攻略、专注“游戏剧情”解说。

通过这个结构可以看出，垂直是一个可以不断深入的过程。大领域和小领域都可以作为你的选择，这完全取决于你自己的兴趣和定位。

我记得在之前的课程中，有同学对“垂直”的概念还不太清晰。所以今天特地结合实例进行了细化讲解，因为后续的很多运营策略都与这个细分思维紧密相关。
