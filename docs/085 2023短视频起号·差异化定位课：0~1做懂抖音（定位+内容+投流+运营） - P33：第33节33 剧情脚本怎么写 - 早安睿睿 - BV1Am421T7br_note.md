# 短视频起号·差异化定位课：P33：剧情脚本怎么写 📝

在本节课中，我们将学习如何为剧情类短视频撰写脚本。脚本是视频能否成为爆款的核心关键，提前设计好脚本，可以确保视频的节奏、话题点和画面质量都达到理想状态。

![](img/1c14a1db48b08a7249418677b01bc57a_1.png)

许多新手起号时容易忽略脚本的重要性，导致视频完成后总觉得缺少亮点。实际上，爆款作品往往是“脚本前置”的，即在拍摄前就规划好每一秒的文案、字幕、灯光和构图。成熟的MCN机构之所以能持续产出爆款，正是因为他们掌握了成熟的脚本模板和套路，并将其复制应用。

![](img/1c14a1db48b08a7249418677b01bc57a_3.png)

![](img/1c14a1db48b08a7249418677b01bc57a_5.png)

上一节我们介绍了脚本的重要性，本节中我们来看看如何具体撰写剧情类账号的脚本。

## 剧情脚本的核心：反转 🔄

![](img/1c14a1db48b08a7249418677b01bc57a_7.png)

![](img/1c14a1db48b08a7249418677b01bc57a_9.png)

![](img/1c14a1db48b08a7249418677b01bc57a_11.png)

剧情类脚本的核心在于**反转**。反转能够持续调动观众的情绪，使其在观看时经历“波荡起伏”的心理状态。观众的情绪会随着剧情发展不断被拉高和压低，从而被牢牢吸引。

## 案例分析：温情剧情号

![](img/1c14a1db48b08a7249418677b01bc57a_27.png)

![](img/1c14a1db48b08a7249418677b01bc57a_29.png)

让我们通过一个具体案例来拆解反转的运用。这个账号名为“这是他的故事”，视频讲述了一对夫妻的日常。

**视频内容回顾：**
1.  妻子交了房租，夫妻二人骑电瓶车外出。
2.  **（第2秒，第一次反转）** 他们的电瓶车被一位车主驱赶，指责其占用了汽车车位，制造了第一个冲突点。
3.  **（第6-7秒，矛盾升级）** 车主进一步言语冲突，让观众好奇剧情如何发展。
4.  夫妻二人来到常去的路边摊吃饭，回忆往昔。
5.  **（第三次反转铺垫）** 妻子感叹“现在咱俩都结婚了，还是陈冉冉（可能指生活依旧）”，让观众和丈夫误以为她在抱怨现状。
6.  **（第三次反转揭晓）** 丈夫安慰说“别人有的，咱以后也会有”，妻子却反问“你说什么呢？”，随后解释自己说的是“这菜这么多年了，还是这么好吃”，误会解除，温情收尾。

![](img/1c14a1db48b08a7249418677b01bc57a_31.png)

![](img/1c14a1db48b08a7249418677b01bc57a_33.png)

![](img/1c14a1db48b08a7249418677b01bc57a_35.png)

在这个约50秒的视频中，至少安排了3-4次情绪转折和反转，不断吸引观众看下去。

## 案例分析：搞笑剧情号

![](img/1c14a1db48b08a7249418677b01bc57a_41.png)

搞笑类剧情号对反转的运用更加密集和极限。我们以账号“派小轩”的一条视频为例。

![](img/1c14a1db48b08a7249418677b01bc57a_43.png)

![](img/1c14a1db48b08a7249418677b01bc57a_45.png)

**视频内容拆解：**
以下是该视频中高频反转的节奏：
*   **（第3秒）** 期待看日出 vs 男方关注电线杆。
*   **（约第14秒）** 期待美照 vs 拍照翻车。
*   **（第三次反转）** 告知还有视频 vs 视频效果糟糕。
*   **（第24秒）** 以为是视频 vs 其实是动图。
*   **（又一次反转）** 叫住女方以为重拍 vs 其实是男方自己想拍照。

![](img/1c14a1db48b08a7249418677b01bc57a_47.png)

![](img/1c14a1db48b08a7249418677b01bc57a_49.png)

这条约30秒的视频，实现了近**6次反转**，平均每5秒一次，这正是其获得近2000万播放量的关键。

## 如何撰写剧情脚本：核心公式

![](img/1c14a1db48b08a7249418677b01bc57a_55.png)

![](img/1c14a1db48b08a7249418677b01bc57a_57.png)

基于以上分析，我们可以总结出撰写剧情类短视频脚本的核心方法。

![](img/1c14a1db48b08a7249418677b01bc57a_59.png)

![](img/1c14a1db48b08a7249418677b01bc57a_61.png)

![](img/1c14a1db48b08a7249418677b01bc57a_63.png)

**核心公式：高频反转 = 前5秒首次反转 + 后续每10秒一个反转**

![](img/1c14a1db48b08a7249418677b01bc57a_65.png)

![](img/1c14a1db48b08a7249418677b01bc57a_67.png)

以下是具体操作步骤：
1.  **前5秒，必须出现第一次反转。** 迅速打破平静，制造冲突、意外或笑点，抓住观众注意力。例如，温馨开场突然被外人打断，或期待场景出现滑稽失误。
2.  **后续节奏，保持每10秒左右一个反转。** 通过不断制造新的冲突、误会或意外，并在短时间内解开或转化，使观众情绪持续起伏。这就像相声中的“丢包袱”和“解包袱”，让故事充满张力。

![](img/1c14a1db48b08a7249418677b01bc57a_69.png)

![](img/1c14a1db48b08a7249418677b01bc57a_73.png)

![](img/1c14a1db48b08a7249418677b01bc57a_71.png)

## 总结与下期预告

![](img/1c14a1db48b08a7249418677b01bc57a_73.png)

本节课中我们一起学习了剧情类短视频脚本的撰写核心：**反转**，而且是**高频次的反转**。通过“前5秒出反转，后续每10秒一个反转”的节奏，可以有效牵引观众情绪，打造爆款视频。
