# AI短视频制作教程：P1：揭秘抖音名人语录视频制作全流程 🎬

![](img/23d66ff8c5d093b6a092157ce0cca99a_1.png)

在本节课中，我们将学习如何制作在抖音上流行的“名人语录”类短视频。这类视频通常使用AI技术克隆名人声音，并生成对口型视频，结合认知类语录内容，最终通过带货等方式实现变现。整个流程清晰简单，适合新手操作。

![](img/23d66ff8c5d093b6a092157ce0cca99a_3.png)

## 声音克隆 🔊

上一节我们概述了制作流程，本节中我们来看看第一个核心步骤：声音克隆。其核心原理是利用AI模型，通过分析一段短样本音频，学习并模仿该声音的声纹特征，从而生成新的语音。公式可简化为：`AI模型(样本音频) -> 克隆声纹模型`。

![](img/23d66ff8c5d093b6a092157ce0cca99a_5.png)

具体操作步骤如下：

![](img/23d66ff8c5d093b6a092157ce0cca99a_7.png)

以下是声音克隆的详细步骤：

1.  **准备声音样本**：寻找并下载一段目标名人（如视频中的“鸣人”）的讲话视频，时长约25秒。这是克隆声音的基础材料。
2.  **提取纯净音频**：将下载的视频导入“剪映”等视频编辑软件。使用软件的“音频分离”功能，将人声与背景音乐分离，并导出为独立的音频文件（如MP3格式）。
3.  **进行声音克隆**：打开网页版AI声音克隆工具。点击“声音构建”或类似功能，上传上一步准备好的纯净音频样本。系统会自动处理并生成一个克隆后的声音模型。

![](img/23d66ff8c5d093b6a092157ce0cca99a_9.png)

完成以上步骤后，你就拥有了一个可以生成该名人音色配音的AI模型。

## 生成对口型视频 🗣️

![](img/23d66ff8c5d093b6a092157ce0cca99a_13.png)

掌握了声音克隆技术后，接下来我们需要让“名人”开口说出我们准备的文案。本节将介绍如何制作逼真的对口型视频。

其技术核心是让静态图片根据音频驱动，生成口型、面部表情同步的动态视频。这个过程可以理解为：`AI工具(静态图片 + 音频文件) -> 动态对口型视频`。

![](img/23d66ff8c5d093b6a092157ce0cca99a_15.png)

以下是制作对口型视频的步骤：

1.  **准备名人图片**：在百度等搜索引擎中，搜索一张目标名人的高清正面照。图片质量越高，生成效果越好。
2.  **生成配音音频**：使用上一步克隆好的声音模型，将你撰写的“认知语录”文案输入，生成对应的配音音频文件。
3.  **合成动态视频**：打开AI对口型视频生成工具。分别上传准备好的名人正面照和生成的配音音频。点击“生成”按钮，等待系统处理。通常，生成一段10秒左右的视频片段，非常适合作为视频的开头素材。

![](img/23d66ff8c5d093b6a092157ce0cca99a_17.png)

## 视频剪辑与变现 💰

![](img/23d66ff8c5d093b6a092157ce0cca99a_19.png)

我们已经得到了核心的音频和视频素材，本节将讲解如何将它们整合成完整的视频，并探讨此类内容的变现方式。

剪辑的核心在于将AI生成的素材与其他相关素材进行有机组合，通过混剪增强视频的丰富度和吸引力。

![](img/23d66ff8c5d093b6a092157ce0cca99a_21.png)

以下是视频剪辑与变现的步骤：

![](img/23d66ff8c5d093b6a092157ce0cca99a_23.png)

1.  **整合素材进行剪辑**：将AI生成的对口型视频片段和配音音频，一同导入“剪映”软件。根据语录内容，寻找相关的影视片段、动画或图片素材进行混剪，丰富视频画面。
2.  **完成视频导出**：对剪辑好的视频进行调色、添加字幕、背景音乐等后期处理，最后导出成片即可发布。
3.  **主要变现途径**：此类“认知语录”视频常见的变现方式是“带货”，尤其是书籍。因为内容主题与认知、成长相关，在视频结尾或评论区植入相关书籍的推广链接，转化效果较好。

本节课中我们一起学习了制作抖音名人语录视频的三个关键阶段：首先通过AI工具克隆名人声音；然后利用克隆声音和名人照片生成对口型视频；最后通过剪辑整合素材，并通过带货实现内容变现。整个过程依托于易用的AI工具，大大降低了制作门槛。
