# 创业风向标：P1：私域引流素材制作教程 🚀

在本节课中，我们将学习如何从零开始制作私域引流素材。课程将涵盖常见的引流内容类型、制作工具选择以及具体的制作步骤，旨在帮助初学者快速上手。

## 概述

私域运营的核心在于吸引目标用户。制作高质量的引流素材是第一步。本节将拆解视频类、图文类和矢量图类三种主流素材的制作方法，并提供实用的工具和技巧。

---

## 引流素材的常见类型 🎬

上一节我们概述了课程内容，本节中我们来看看具体的引流素材类型。常见的引流内容大致分为三类：视频类、图文类和矢量图类。制作以上任何类型的内容，都需要先寻找对标账号作为参考。

![](img/3d343bceb5077747793130e7062643c8_1.png)

以下是三种类型的简要说明：

![](img/3d343bceb5077747793130e7062643c8_2.png)

*   **视频类**：需要一定的剪辑基础。如果新人没有基础，需要逐步学习。
*   **图文类**：制作门槛相对较低，适合新手起步。
*   **矢量图类**：同样需要剪辑基础，且素材需要自行寻找。这类视频流量潜力大，但对能力有一定要求。

## 视频类素材制作详解 🎥

视频是当前非常有效的引流形式。对于新人而言，掌握基础的剪辑技能是关键。

### 剪辑基础学习

如果不会剪辑，可以从学习常用功能开始。剪辑视频比较常用的功能包括：**关键帧**、**音效**、**画中画**、**特效**、**滤镜**、**文字模板**和**文字动画效果**。这些功能可以逐个搜索教程进行学习。

### 视频制作的两种方法

![](img/3d343bceb5077747793130e7062643c8_6.png)

了解了基础功能后，我们来看看具体的制作方法。视频类账号的剪辑主要有两种实操方法。

![](img/3d343bceb5077747793130e7062643c8_7.png)

以下是两种方法的介绍：

1.  **真人出镜拍摄**：直接拿手机对着所要呈现的内容拍摄。这种方法简单粗暴，很接地气，也不需要额外配音，直接加背景音乐或者一边拍一边读文字即可。
    ![](img/3d343bceb5077747793130e7062643c8_5.png)
    ![](img/3d343bceb5077747793130e7062643c8_6.png)

![](img/3d343bceb5077747793130e7062643c8_8.png)

2.  **画中画加录屏（不出镜账号）**：这种方法需要一定的剪辑基础。部分素材可以直接使用剪映App内的资源。新手可以慢慢进行优化，找找感觉。
    ![](img/3d343bceb5077747793130e7062643c8_7.png)
    ![](img/3d343bceb5077747793130e7062643c8_8.png)

> **工具提示**：如果你完全不会剪辑，可以寻找相关的剪映教程（包含手机版和电脑版）进行系统学习。

## 图文类素材制作详解 📝

图文类素材制作门槛较低，非常适合没有设计基础的新手。如果你不会使用专业的PS软件，有以下工具推荐。

以下是三款推荐的作图工具：

*   **美图秀秀**：操作简单，功能全面。
*   **搞定设计**：包含了大量的素材模板，可直接套用。
*   **Canva（刻画）**：在线设计平台，拥有丰富的模板和素材库。

### 图文制作步骤

![](img/3d343bceb5077747793130e7062643c8_10.png)

那么，图文类内容具体如何制作呢？方法非常直接。

1.  先寻找对标账号，例如搜索“图文类工具分享”来寻找你要参考的账号风格。
2.  将你的文案内容，放入“搞定设计”这类工具的合适模板中。
3.  调整字体、颜色和布局，一份原创图文素材就产生了。

## 矢量图类素材制作详解 ✨

![](img/3d343bceb5077747793130e7062643c8_13.png)

矢量图类视频通常用于传递观点和认知，视觉冲击力强。这类内容制作需要一些技巧。

### 素材获取与处理

首先，需要找到可用的矢量图素材。这里分享一个网站：**阿里巴巴矢量图标库**。里面包含了海量的素材可以使用。

找到需要的矢量图后，直接导入到剪映等剪辑软件中。接下来的步骤就是对画面添加动画、跟踪效果、音效，并处理文字模板。

![](img/3d343bceb5077747793130e7062643c8_10.png)
![](img/3d343bceb5077747793130e7062643c8_11.png)

### 内容创作流程

由于这类账号主要讲述认知和观点，所以“二创”是关键。具体操作流程如下。

以下是制作一个原创矢量图认知视频的步骤：

1.  **寻找素材**：搜索关键词，如“让你醍醐灌顶的一句话”，会联想出很多相关视频。
2.  **提取文案**：选择一个视频，将其链接复制到“创作猫”这类软件中，使用“链接转文字”功能，将视频文案复制下来。
3.  **文案二创**：对复制下来的文案进行二次创作。如果无法使用高级AI工具，可以使用“秘塔写作猫”等免费AI写作工具辅助。**切记，开头第一句话最好和原视频一样**，例如“最近看到的醍醐灌顶的一段话”，然后手动修改后面的内容。
4.  **合成视频**：将创作好的文案进行配音，然后将配音和下载好的矢量图素材导入剪映，进行剪辑合成。

![](img/3d343bceb5077747793130e7062643c8_15.png)

![](img/3d343bceb5077747793130e7062643c8_15.png)
![](img/3d343bceb5077747793130e7062643c8_16.png)

![](img/3d343bceb5077747793130e7062643c8_16.png)

这样就完成了一个原创的矢量图认知视频。
