# AI短视频创作教程：P9：7.视频剪辑实操（下）

## 概述
在本节课中，我们将继续学习视频剪辑的实操部分，重点讲解如何为拼接的视频片段添加转场效果，以及如何为视频添加字幕。这些步骤对于提升视频的流畅度和专业性至关重要。

![](img/e4d51b08af2dec3caa565c0f187b1a3e_1.png)

## 调整视频比例
上一节我们介绍了如何导入和裁剪素材。在完成素材的裁剪后，需要统一所有视频片段的画面比例。

![](img/e4d51b08af2dec3caa565c0f187b1a3e_3.png)

![](img/e4d51b08af2dec3caa565c0f187b1a3e_5.png)

以下是调整视频比例的步骤：
1.  选中视频轨道上的任意一个片段。
2.  在编辑界面中找到并点击“比例”选项。
3.  在弹出的比例选项中，选择 **16:9** 的横屏比例。
4.  点击右下角的对勾确认操作。

![](img/e4d51b08af2dec3caa565c0f187b1a3e_7.png)

![](img/e4d51b08af2dec3caa565c0f187b1a3e_9.png)

完成此操作后，所有视频片段都将被统一为16:9的横屏比例。

![](img/e4d51b08af2dec3caa565c0f187b1a3e_11.png)

![](img/e4d51b08af2dec3caa565c0f187b1a3e_13.png)

## 添加视频转场
视频由多个片段拼接而成，每个片段时长大约在十秒左右。片段之间需要有自然的衔接，因此必须添加转场效果。

![](img/e4d51b08af2dec3caa565c0f187b1a3e_15.png)

转场的作用是明确提示用户，当前内容已切换到另一个不同的片段。如果不添加转场，观众在观看时可能会因为画面内容的突然切换而感到困惑，甚至退出视频。这会直接导致视频的**完播率**下降，而完播率是影响视频推荐量和收益的关键指标之一。

![](img/e4d51b08af2dec3caa565c0f187b1a3e_17.png)

因此，添加转场是必要的步骤。

![](img/e4d51b08af2dec3caa565c0f187b1a3e_19.png)

以下是添加转场的具体方法：
1.  在视频轨道上，找到两个片段之间的位置，点击右侧的 **“+”** 号按钮。
2.  点击后，会进入素材添加界面，选择 **“素材库”**。
3.  在素材库的 **“热门”** 分类中，可以找到大量转场素材，例如各种动态切换效果。
4.  选择转场素材时，注意时长不宜过长，尽量选择 **3到5秒** 的转场，以免影响观看体验。最合适的是3秒钟的转场。
5.  选中合适的转场素材，点击添加。

![](img/e4d51b08af2dec3caa565c0f187b1a3e_21.png)

添加转场后，同样需要确保其比例与主视频一致（16:9）。转场素材添加后，编辑界面通常会提供 **“画布模糊”** 或背景填充选项。点击“画布模糊”，选择第一个清晰的画面背景，即可将转场背景调整为16:9的横屏。最后，记得点击右下角的对勾确认。

## 添加视频字幕
完成转场添加后，下一步是为视频添加字幕（文案），以提升视频的可看性和信息传达效率。

![](img/e4d51b08af2dec3caa565c0f187b1a3e_23.png)

以下是添加字幕的步骤：
1.  在手机剪辑软件界面左下角，找到并点击 **“文字”** 选项。
2.  点击“新建文本”右侧的 **“识别字幕”** 功能。
3.  点击“开始匹配”，软件会自动识别视频中的语音，并生成对应的字幕文本。
4.  字幕生成后，可以对其进行调整。拖动字幕文本框右下角，可以放大字幕。
5.  将放大后的字幕文本框，拖动到视频画面最下方的位置，这样观看最舒适。
6.  调整好第一句字幕后，后续的字幕样式通常会同步更新，无需逐一调整。

![](img/e4d51b08af2dec3caa565c0f187b1a3e_25.png)

![](img/e4d51b08af2dec3caa565c0f187b1a3e_27.png)

## 总结
本节课中，我们一起学习了视频剪辑后半部分的核心操作：统一视频比例、添加转场效果以及识别添加字幕。请记住，在制作过程中，务必严格按照步骤操作，选择时长合适的素材（避免过长），并保持所有元素的风格比例统一，这样才能制作出平台易于通过审核、观众体验良好的视频。整个流程并不复杂，关键在于细心和规范。
