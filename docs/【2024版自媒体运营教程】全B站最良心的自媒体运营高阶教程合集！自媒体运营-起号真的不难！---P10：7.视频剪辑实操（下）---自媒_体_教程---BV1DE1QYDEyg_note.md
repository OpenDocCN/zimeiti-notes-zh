## 添加视频转场

![](img/3359a427fd1bf9252c73e221596fbe37_3.png)

一个完整的视频通常由多个片段组成，每个片段时长可能在10秒左右。这些片段之间需要有自然的衔接，这就需要使用“转场”效果。转场能让用户明确感知到视频内容进入了下一个部分。

如果不添加转场，观众在观看时可能会感到突兀，甚至退出视频查看内容是否出错。这种行为会直接导致视频的**完播率**下降。

**完播率**是影响视频单价和最终收益的关键指标之一。因此，添加转场是必要的步骤。

![](img/3359a427fd1bf9252c73e221596fbe37_5.png)

以下是添加转场的具体操作：

![](img/3359a427fd1bf9252c73e221596fbe37_6.png)

![](img/3359a427fd1bf9252c73e221596fbe37_7.png)

![](img/3359a427fd1bf9252c73e221596fbe37_8.png)

1.  在视频编辑轨道右侧，找到一个“加号”按钮，点击它。
2.  在弹出的菜单中，选择“素材库”。
3.  在素材库的“热门”分类中，可以找到多种转场素材，例如搞笑片段或动态效果。这些转场在各类视频中都很常见。
4.  选择转场时，需要注意时长。避免选择过长的转场（例如16秒），因为观众通常没有耐心观看与主要内容无关的长片段。**建议选择时长在3到5秒之间的转场**。
5.  选中合适的转场后，点击添加。

![](img/3359a427fd1bf9252c73e221596fbe37_9.png)

转场视频添加后，必须确保其画面比例与主视频保持一致，即**16:9**的横屏比例。

以下是调整转场视频比例的步骤：

1.  选中已添加的转场片段。
2.  在编辑菜单中找到“画布模糊”或“背景”选项。
3.  选择第一个清晰的画面模式，这通常会将背景填充为符合16:9比例的模糊效果。
4.  调整完毕后，点击右下角的对勾确认。

![](img/3359a427fd1bf9252c73e221596fbe37_11.png)

![](img/3359a427fd1bf9252c73e221596fbe37_12.png)

至此，视频的比例调整和转场添加步骤就完成了。

## 添加与调整字幕

视频内容制作完成后，下一步是添加字幕，方便观众理解。我们可以使用软件的自动识别功能来高效完成。

![](img/3359a427fd1bf9252c73e221596fbe37_14.png)

以下是添加字幕的操作流程：

![](img/3359a427fd1bf9252c73e221596fbe37_15.png)

1.  在手机剪辑软件界面左下角，找到“文字”选项并点击。
2.  选择“新建文本”，然后在右侧功能中找到“识别字幕”按钮。
3.  点击“识别字幕”，并选择“开始匹配”。软件会自动识别视频或音频中的人物对话，并生成对应的文字轨道。
4.  字幕生成后，通常需要调整其大小和位置以获得最佳观看体验。选中字幕文本框，拖动其右下角可以放大字体。
5.  将放大后的字幕文本框拖动到视频画面的最下方，这个位置通常最为舒适。**调整好第一个字幕后，后续的字幕会自动应用相同的样式和位置**，非常方便。

## 核心要点与总结

本节课中我们一起学习了视频剪辑后半部分的核心操作。

![](img/3359a427fd1bf9252c73e221596fbe37_17.png)

![](img/3359a427fd1bf9252c73e221596fbe37_18.png)

我们来总结一下制作一个合格视频的关键步骤：
*   **严格遵循步骤**：从找素材到剪辑，每一步都需按要求操作。
*   **控制素材时长**：避免使用过长的视频片段或转场，以保证内容紧凑。
*   **保持比例统一**：确保所有视频和转场素材均为**16:9**的横屏比例。
*   **利用自动工具**：善用“识别字幕”等功能提升剪辑效率。

![](img/3359a427fd1bf9252c73e221596fbe37_19.png)

![](img/3359a427fd1bf9252c73e221596fbe37_20.png)

视频剪辑本身并不复杂，关键在于细心和遵循正确的流程。按照上述步骤操作，你就能制作出符合平台要求、观感流畅的视频内容。
