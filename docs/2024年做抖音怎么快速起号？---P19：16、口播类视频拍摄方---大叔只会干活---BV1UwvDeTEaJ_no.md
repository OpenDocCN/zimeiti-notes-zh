上一节我们介绍了视频拍摄的通用注意事项，本节中我们来看看如何将这些原则应用到具体的视频类型中。口播类视频是抖音上非常常见且有效的内容形式，其拍摄方法具有通用性。

![](img/d991bfd6544d2e38b61da1ddfb582788_1.png)

无论你从事金融、装修、养生还是情感领域，口播视频的拍摄核心方法都是一致的。同样，记录日常生活、拍摄段子或美食视频，其拍摄逻辑也相通。因此，掌握口播视频的拍摄，就能触类旁通。

首先，我们分析一个成功案例。这位名为“种小草”的学员，是一位40岁带三个孩子的宝妈，她专注于情感口播领域。她的一条视频播放量超过8000万，并凭借该视频增长了30万粉丝。

她成功的关键在于精准吸引目标群体。她的所有内容都围绕特定群体的兴趣点展开，这使得涨粉速度快且粉丝精准。例如，如果目标群体是20多岁的学生，那么所有内容都应围绕学生感兴趣的话题展开。

以下是口播视频拍摄需要关注的几个核心要点：

1.  **构图比例**：抖音是9:16的竖屏平台。你需要在其中选择横屏或竖屏构图。案例中的视频采用了4:3的横屏构图。**关键点在于保持统一**：如果选择横屏，则所有视频都应使用横屏；选择竖屏，则全部使用竖屏。

2.  **人物主体**：人物应占据画面中央约70%的面积。避免将脸部怼满屏幕。画面左、右、上方应适当留白，使画面看起来清晰、舒适。

3.  **外貌与表现**：口播视频的核心是内容与知识的传达，并非依赖出众的外貌。只要让观众感觉清晰、体面即可。如果对自己的外貌不自信，可以开启美颜滤镜进行调整，但注意不要过度夸张。

4.  **内容与共鸣**：视频内容需要针对特定群体，引发共鸣。我们通过案例视频来具体分析。

![](img/d991bfd6544d2e38b61da1ddfb582788_3.png)

现在，我们一起观看“种小草”这条涨粉30万的视频片段，并思考：这个视频旨在吸引哪个特定群体？

> “跟婆婆在一起住是什么样的感受？我跟我婆婆在一起住了16年了，平时磕磕碰碰有点小不愉快，那都是正常的……我最大的感受就是这十多年以来，我娘家人很少在我家住上几天……这对于我内心来说其实是一个很大的缺失。”

观看后，你认为她吸引的是哪个群体？是泛指“婆媳关系”吗？实际上，对婆媳关系感触最深的，并非男性群体，也非未婚的年轻女性。**最能产生共鸣的群体，是已经结婚、亲身经历婆媳关系的女性**。她的内容精准地吸引了已婚女性群体。

从视频中我们还可以看到，她使用了美颜功能，使人物形象更得体。构图保持4:3横屏，背景简洁。

接着，解答两个常见问题：

*   **问题一：口播必须背稿吗？**
    不需要完全背下来。在后续的剪辑环节，可以使用提词器工具，实现边看边读边录制，这大大降低了拍摄难度。
    ```python
    # 核心工具：提词器
    工具功能 = “辅助录制，避免背稿”
    使用场景 = “口播视频拍摄”
    ```

*   **问题二：如何保证声音清晰？**
    声音清晰是口播视频的关键。建议购买一个手机收声麦克风（领夹麦），价格低廉（约十几元），能在淘宝、京东等平台轻松购得，它能显著提升音频质量。
    ```python
    # 核心设备：收声麦克风
    设备 = “手机收声麦克风（领夹麦）”
    作用 = “提升音频清晰度，减少环境噪音”
    获取方式 = “电商平台（如淘宝、京东）”
    ```

## 总结
本节课我们一起学习了口播类视频的拍摄方法。我们通过案例了解到，成功的口播视频需要**统一的构图比例**、**得体的人物呈现**，最重要的是**精准的内容定位**以引发目标群体共鸣。同时，利用**提词器**和**收声麦克风**等工具，可以有效降低拍摄难度并提升成品质量。记住，内容的价值和针对性远比外貌更重要。
