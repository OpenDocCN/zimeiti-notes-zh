# 小红书运营：P26：账号类型与官方广告解析

在本节课中，我们将学习小红书账号的三种主要类型，并对比自建账号推广与官方广告投放的区别。我们将理清企业号、个人号和引流号各自的特点与用途，并介绍小红书官方广告（CBC竞价广告）的运作方式。

## 账号类型解析

上一节我们介绍了账号定位的重要性，本节中我们来看看小红书平台上有哪几种主要的账号类型。了解这些类型有助于我们选择最适合自己的运营策略。

小红书账号主要分为三类：企业号、个人号和引流号。

以下是三类账号的详细说明：

*   **企业号**
    *   企业号的主要作用是树立官方品牌形象。
    *   它拥有官方认证标志和特权，例如展示品牌名称、地址等，便于开展业务。

*   **个人号**
    *   个人号是目前大多数用户运营的账号类型，其特点是灵活性高。
    *   运营者可以是品牌员工或消费者，账号内容更具个人品牌特色，品牌相关人物出镜也显得更自然。

*   **引流号**
    *   引流号在之前的课程中提及过，俗称“炮灰号”，可以创建多个。
    *   这类账号通常无需精细运营，可用于临时推广或在紧急情况下作为备用小号使用，一般没有长期的内容规划。

通常，品牌的推广渠道会通过企业号、个人号或引流号发布笔记进行引流。接下来，我们将探讨如何有效组合这些账号。

## 企业号矩阵运营建议

了解了三种账号类型后，我们可以思考如何将它们协同运作。以下是关于打造企业号矩阵的一些具体建议：

*   **积极参与品牌活动**：品牌的企业号发起活动（如试用、团购）时，可以动用多个小号去参与，增加活动热度。
*   **个人号展示品牌属性**：在个人号的笔记内容中，应多次暗示品牌的定位属性（例如，目标用户是城市年轻人还是职场新人）。
*   **控制产品发布频率**：个人号发布品牌产品内容的频率不宜过高。建议采用 **三七分** 的比例：**30%** 的内容用于营销，**70%** 的内容用于种草或科普。
*   **利用引流号评论引流**：企业号如需直接引流，最好使用引流号在其笔记评论区进行引导，这样更为安全。

除了自建账号矩阵进行推广，还有另一种重要的推广方式，那就是官方广告。

## 小红书官方广告（CBC）介绍

上一部分我们讨论了自运营账号的策略，本节中我们来看看另一种推广方式：小红书官方广告。

小红书官方广告即CBC竞价广告。其申请流程和使用场景可以在小红书官方网站的“业务合作” -> “推广合作”板块找到。

认证成功后，即可投放信息流广告。官方广告主要分为两大类：

1.  **信息流广告**：用户浏览小红书首页时，信息流中会不定期出现的广告。
2.  **搜索广告**：当用户使用关键词搜索时出现的广告。系统会根据算法进行相关推荐（例如，搜索“帽子”可能推荐大衣；搜索“考研资料”可能推荐营养品）。

其中，搜索广告是典型的竞价广告，其原理与百度关键词竞价类似：通过对关键词出价，竞争不同的广告排名位置。

当前信息流广告位常见的有第3位、第13位和第23位等。
