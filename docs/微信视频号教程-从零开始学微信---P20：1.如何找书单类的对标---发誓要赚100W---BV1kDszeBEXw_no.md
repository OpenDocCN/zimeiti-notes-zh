# 微信视频号教程：P20：如何寻找书单类对标账号 📚

在本节课中，我们将学习如何在短视频平台（以抖音为例）上，高效地寻找与你想要运营的账号类型（特别是书单类）相似的对标账号。掌握这个方法，能帮助你快速了解市场，明确模仿和学习的对象。

## 核心思路与准备工作

上一节我们介绍了寻找对标账号的重要性，本节中我们来看看具体的操作方法。核心思路是：**通过关键词搜索或已发现的优质账号，利用平台的推荐机制，批量找到同类型的账号**。

在开始之前，你需要明确自己想做的账号类型，例如“书单”、“渔具”或“玩具”，并准备好相应的关键词。

![](img/dff6ee72bd67028a4dd28b2efef4c703_1.png)

## 方法一：利用“关注”推荐功能

这种方法适用于你已经找到了一个不错的参考账号。以下是具体操作步骤：

![](img/dff6ee72bd67028a4dd28b2efef4c703_3.png)

1.  **初步评估账号**：在抖音搜索关键词（如“书单”），找到一个非真人出镜、视频类型符合你预期的账号。点击其头像进入主页，查看其视频内容是否与你想做的类型相符。
2.  **触发推荐机制**：如果内容相符，点击“关注”按钮。关注后，页面下方会立即弹出“你可能感兴趣”的用户列表。这些账号通常与刚关注的账号标签相似，类型相近。
3.  **批量筛选与关注**：逐一查看推荐列表中的账号主页及其作品。如果也是你想模仿的类型（例如，同样是卖书），就点击关注。如果不是，则返回继续浏览下一个。通过这种方式，可以在两三分钟内关注几十个同类账号。此后，平台算法会为你推荐更多相似账号。

## 方法二：通过商品“达人秀”寻找

![](img/dff6ee72bd67028a4dd28b2efef4c703_6.png)

![](img/dff6ee72bd67028a4dd28b2efef4c703_8.png)

如果你想销售特定商品（如某本书），这个方法更为直接。以下是具体操作步骤：

1.  **搜索目标商品**：在抖音右上角点击搜索，输入具体的商品名，例如“墨菲定律漫画版”。
2.  **筛选高销量商品**：搜索结果默认是“综合”页，需切换到“商品”标签页，然后按“销量”排序，找到销量最高的商品链接。
3.  **进入商品详情页**：点击进入该高销量商品页面（注意避开直播入口）。通常，销量在2万以上的商品，其详情页会包含“达人秀”或“买家秀”板块。
4.  **发现大量对标账号**：在“达人秀”板块中，会展示大量推广过该商品的达人视频。这些账号的视频类型（如影视混剪、名人语录剪辑）和商业模式（卖书）高度一致，是绝佳的对标学习对象。你可以直接点击进入这些账号主页进行关注和模仿。

## 总结

![](img/dff6ee72bd67028a4dd28b2efef4c703_10.png)

本节课中我们一起学习了两种在抖音上寻找书单类（或其他品类）对标账号的实用方法：
*   **方法一**：通过关注目标账号，利用平台推荐算法批量获取同类账号。
*   **方法二**：通过搜索具体商品，进入高销量商品的“达人秀”板块，一次性发现大量同类型推广账号。

这两种方法能帮助你快速建立自己的对标账号库，为后续的内容创作和运营提供清晰的参考方向。
