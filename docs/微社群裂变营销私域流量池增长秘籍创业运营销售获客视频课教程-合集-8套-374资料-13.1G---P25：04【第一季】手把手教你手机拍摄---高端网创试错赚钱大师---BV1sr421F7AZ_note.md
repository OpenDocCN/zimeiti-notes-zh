# 手机拍摄实战：P25：04：基础静物拍摄案例

在本节课中，我们将通过一系列具体的静物拍摄案例，手把手演示如何运用之前学到的简单步骤，拍出清晰、专业的商品照片。你将看到，即使是生活中常见的物品，也能通过标准化的流程获得出色的拍摄效果。

## 案例一：书籍与支撑物

这张照片是上一节课的示例。书本能够立起来，是因为背后有一个盒子作为支撑。我们现在就来拍摄这个作为支撑物的盒子。

![](img/57cb5c6be9120e7ce9b1dd9b83ae3fee_3.png)

![](img/57cb5c6be9120e7ce9b1dd9b83ae3fee_1.png)

拍摄步骤如下：
1.  打开灯光，照亮被摄物体。
2.  将盒子放置在画面中央。
3.  按下快门进行拍摄。
4.  将手机镜头放大，再次拍摄一张。

![](img/57cb5c6be9120e7ce9b1dd9b83ae3fee_4.png)

无论你将物体拍得大还是小，最终发布到网络时，通常都会使用放大后的效果。我们来对比一下哪种方式拍出的照片更清晰。

![](img/57cb5c6be9120e7ce9b1dd9b83ae3fee_3.png)
![](img/57cb5c6be9120e7ce9b1dd9b83ae3fee_4.png)

在初始状态下可能难以分辨，但经过多次放大对比后，可以发现**直接在相机中放大拍摄的照片**，其清晰度会略胜一筹。

接着，我们按照上节课的方法，适当提亮照片后再进行拍摄。

至此，这个盒子的拍摄就完成了。整个过程非常简单，只需四个步骤。以下是标准操作流程：
1.  **布光**：用灯照亮物体。
2.  **构图与对焦**：摆好物体，置于画面中央并放大。
3.  **调整曝光**：根据画面调整亮度。
4.  **拍摄**：按下快门。

前期的拍摄虽然完成，但别忘了我们还需要进行后期修饰。

## 核心总结与流程回顾

![](img/57cb5c6be9120e7ce9b1dd9b83ae3fee_8.png)

本节课我们拍摄了多个案例，它们的拍摄步骤完全一致，可以总结为以下标准化流程：

**拍摄四步法公式**：
```
1. 布光照明
2. 居中放大
3. 调整亮度
4. 按下快门
```

这个流程特别简单。在了解这个方法之前，你是否觉得摄影是一件很难的事情？事实上，它并没有想象中那么复杂，甚至可以说非常简单。
