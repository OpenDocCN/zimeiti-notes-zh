## 人设策划的核心概念

![](img/79c1842cb63bc67c7155cacbe0f2487b_6.png)

在抖音中策划人设，如同在游戏中创建角色，需要预先设定角色的各项属性。但与虚拟游戏角色不同，抖音中的人物角色是“人生角色”，因此必须更具真实感。

![](img/79c1842cb63bc67c7155cacbe0f2487b_8.png)

无论是短视频还是直播，一个成功的IP（个人品牌）能够解决粉丝的信任问题，起到裂变传播的作用，并具备强大的变现能力。

因此，我们运营账号，无论是创作内容还是策划选题，其核心目的都是在获取流量的同时打造IP。打造IP的本质，就是在特定场景或情境中，完成账号的**人格化**过程。

![](img/79c1842cb63bc67c7155cacbe0f2487b_10.png)

**账号人格化公式**：
`账号人格化 = 情感注入 + 故事塑造 + 符号化标签`

![](img/79c1842cb63bc67c7155cacbe0f2487b_12.png)

账号人格化，即我们通常所说的“账号人设”。

![](img/79c1842cb63bc67c7155cacbe0f2487b_14.png)

## 人设策划的六个维度

![](img/79c1842cb63bc67c7155cacbe0f2487b_16.png)

对于账号人设的策划，我们可以从以下六个具体方面着手构建。以下是构建一个立体人设的关键要素：

![](img/79c1842cb63bc67c7155cacbe0f2487b_18.png)

1.  **名字/昵称**
    一个IP首先需要一个称呼，这是IP传播的名片。它如同一个品牌的名称或商标，至关重要。

2.  **职业/职务**
    人格化必然具备身份。职业是用户认识这个人或IP的联想入口，例如：老师、教授、律师、程序员等。

3.  **称呼/社会关系**
    一个恰当的称呼能快速拉近与用户的距离，产生情感共鸣。例如：“XX哥”、“XX姐”、“XX大爷”、“XX姥姥”。

4.  **形象/第一印象**
    从用户视角塑造IP形象，使其更容易占据用户心智。例如：大叔形象、萌萌哒、铁憨憨、甜美可爱、光头等。

5.  **性格/爱好**
    一个具有鲜明特点的IP，除了记忆点强，也更具真实性。过于完美的形象反而容易脱离大众。例如：暴躁、幽默、爱吃、爱喝酒、宅等。

6.  **穿着打扮/社会标签**
    借助社会普遍认知和刻板印象，可以快速在用户心智中为IP定位。例如：穿着精致代表时尚达人，开豪车代表成功人士，打扮朴素可能代表专业人士。

## 从人设到账号搭建

以上内容策划完成后，即可结合你自身的产品优势、服务优势等，将其落实到抖音账号的“五件套”中：**头像、昵称、背景图、简介、视频封面**。

除了对账号进行人格化策划，还需通过持续的选题与内容策划，为你的人设提炼出独特的“记忆点”，即所谓的**超级符号**。

## 超级符号的提炼方向

超级符号是让人过目不忘的鲜明特征，可以从以下几个维度进行提炼：

*   **画风（视觉）**：独特的场景、道具、服饰搭配。
*   **听觉**：标志性的BGM、特效音、配音声线。
*   **文案**：能够反映优势且易于记忆的提炼词，例如：“简单粗暴”、“YYDS”。
*   **表情/情绪**：标志性的情绪传递，如大笑、惊恐、无奈等表情包。
*   **语言特色**：方言、口音、独特的说话节奏（如慢条斯理、抑扬顿挫）。
*   **动作/习惯**：习惯性的肢体动作，例如：撩头发、扶眼镜、标志性的手势。

超级符号提炼好后，在进行短视频创作时，就应不断在视频中重复呈现这些元素，强化用户的记忆。

本节课中我们一起学习了抖音账号“人物设定”的完整策划方法。我们从理解人设的重要性开始，逐步拆解了构建人设的六个具体维度，并学习了如何将人设落地到账号搭建中，最后掌握了提炼“超级符号”以强化人设记忆点的关键技巧。一个清晰、一致且富有魅力的人设，是你抖音账号从众多内容中脱颖而出的核心武器。
