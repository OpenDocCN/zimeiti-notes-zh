# 跨境电商运营：P1：彻底解决跟卖问题

在本节课中，我们将学习如何利用亚马逊的“透明计划”来有效且彻底地防止产品被跟卖。我们将从商标的基本应用讲起，逐步深入到透明计划的申请与实施步骤。

---

商标注册成功后，上架的产品是否有必要打上自己的商标？如果不打标，是否容易被他人跟卖？如果你使用已完成品牌备案的商标名上架产品，产品链接的标题前会自动显示你的商标名。这种情况下，并不强制要求在产品实体上印制商标。

当然，如果你的产品是定制款，也可以在实体产品上打上商标。即使你拥有**R标**（已注册的商标），他人仍然可能跟卖你的产品，但你可以据此发起投诉。若想从根本上防止跟卖，最佳方案是加入亚马逊的“透明计划”。

上一节我们提到了透明计划是解决方案，本节中我们来看看它的具体细节。透明计划目前主要适用于**FBA**（亚马逊物流）产品，对于**FBM**（卖家自发货）产品是否能参与，其资格尚不明确。

以下是申请透明计划的具体步骤：
1.  联系亚马逊的透明计划招商经理。
2.  从招商经理处获取申请表格。
3.  在表格中填写你的**商标信息**、**店铺信息**以及**产品信息**并提交。
4.  审核通过后，亚马逊会为你提供专属的**二维码标签**。

获得二维码标签后，你需要将其粘贴在每一个对应的产品实体上。完成此操作后，你的产品将受到保护，无法再被其他卖家跟卖。

综上所述，本节课我们一起学习了从品牌标识到透明计划的全流程。透明计划通过为每个产品赋予独一无二的二维码，是目前唯一能够安全、有效且彻底解决亚马逊跟卖问题的方法。
