# 新媒体运营：P29：11：小红书平台用户标签及流量入口 📊

![](img/1fd5ab768b0e708f44809cc19b38b6b0_1.png)

在本节课中，我们将学习小红书平台如何为用户打上标签，以及平台主要的流量入口有哪些。理解这些机制，有助于我们分析笔记如何被系统推荐给精准用户，并指导我们优化内容策略。

上一节我们介绍了系统如何判定内容标签，本节中我们来看看平台如何判定用户标签。这个过程能帮助我们理解笔记是如何被推荐给用户的，并可以从底层逻辑反推，如何让笔记被更多精准用户看到。

## 用户标签判定机制 🔍

平台为用户打标签的主要依据是用户的**浏览行为**。用户刚注册时选择的兴趣范围参考价值不大，它仅作为初期内容方向的引导，无法进入系统深层的用户标签池。

系统根据用户的一系列浏览行为来判定其兴趣分类。以下是主要的判定维度：

以下是判定用户标签的几个核心行为指标：

1.  **点击率**
    *   **定义**：系统在发现页向用户推送多个内容（如美妆、家居、母婴等），用户点击某一类内容的频率。
    *   **作用**：系统通过计算用户点击不同类别内容的概率，为其打上强关联或弱关联的标签。例如，用户点击10条内容中，有3条是美妆，2条是家居，1条是母婴，系统会判定该用户对美妆最感兴趣，对母婴兴趣较弱。

2.  **停留时长**
    *   **定义**：用户点开一篇笔记后观看的持续时间。
    *   **作用**：即使同属一个大类（如美妆），用户对不同细分内容（如上妆、护肤）的停留时长也不同。系统通过此数据进一步细化用户偏好。例如，用户对多数美妆内容快速划走，但对某篇家居内容停留很久，系统可能判断其近期有装修或购买家居用品的需求。

3.  **页面浏览量（PV）**
    *   **定义**：用户浏览页面的深度，例如不仅看了单篇笔记，还进入了博主主页并查看了其他多篇笔记。
    *   **作用**：PV值能反映用户对账号内容的兴趣浓度。提升用户在你主页的浏览深度，能为账号健康度带来积极评分，即使粉丝基数不大。

4.  **互动行为**
    *   **定义**：用户对笔记的点赞、收藏、评论、关注等操作。
    *   **作用**：这些数据直接提升笔记进入下一级推荐流量池的几率。高互动数据不仅有利于笔记获得更多推荐，也能带动账号内其他笔记的流量，并增加粉丝数，是内容运营的核心目标。

## 平台流量入口分布 🚪

了解用户标签后，我们来看小红书主要的流量分发入口。平台目前主要有四个流量入口，各自承载不同的内容推荐逻辑。

以下是小红书平台的四个主要流量入口及其特点：

1.  **发现页**
    *   **内容构成**：
        *   已关注达人发布的新内容。
        *   根据用户标签（点击率、停留时长等）匹配的“可能感兴趣”的内容。
        *   少量系统推荐的、用户未接触过的“发散性内容”，以保持平台内容的多样性。
    *   **核心价值**：这是笔记和账号获取流量的最主要来源。

2.  **关注页**
    *   **内容构成**：纯粹展示用户所有已关注博主发布的内容。
    *   **核心价值**：方便用户集中查看所关注账号的更新，是粉丝流量的直接体现。

3.  **附近页**
    *   **内容构成**：根据手机定位，推送同城或附近地区的相关内容。
    *   **核心价值**：以具有强地域特征的内容为主，如本地美食、旅游、线下店铺推广等。对于运营线下实体店的账号，此入口尤为重要。

4.  **购物页**
    *   **内容构成**：小红书电商生态的入口，包含商品列表、直播带货等内容。
    *   **核心价值**：满足用户的直接购物需求，是平台实现商业变现和向电商方向发展的重要板块。但目前用户习惯仍在培养中，许多用户仍倾向于先搜索相关笔记（种草）而非直接进入商城购物。
