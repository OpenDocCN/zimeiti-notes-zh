在本节课中，我们将要学习小红书店铺运营中至关重要的一环——人群定位。我们将详细拆解如何精准定义你的目标用户，并理解这与店铺定位的区别与联系。

上一节我们介绍了店铺定位，它决定了店铺的整体发展方向。本节中我们来看看人群定位，它关注的是我们具体要面向哪一类用户进行操作。

人群定位与店铺定位存在差别。店铺定位定位的是整个店铺的发展方向，而人群定位定位的是我们在操作时对应的用户人群。我们已经完成了产品选择、价格设定以及产品发展方向的选择。接下来，我们需要选择对应的用户范围。

因为这是通过产品来判别用户，或者通过用户来选择产品，两者是相互的，缺一不可。如果你先选择了产品，那么你就要选择对应的人群。如果你没有产品，那么你可以先选择人群再选择产品。这是整体网络运营的基本技巧。两者相互依存，缺一不可。有产品就要选人，没有产品就要通过人选产品。这些就是人群定位的整体方向。

在操作店铺时，不要小瞧小红书的整体搜索系统。我们所了解的所有内容，都是为店铺定位做的基础准备。如果这些数据做不好，后期店铺很难做大，可能一天只卖几单或十几单。因此，必须把基础技术做好，才能把店铺做起来。

在个人定位方面，我将其分为五个类型。以下是这五个用户人群类型的具体分析：

1.  **个人属性**：包括用户的性别、年龄范围以及喜欢的内容类型（如文旅、食品、服装、游戏、美妆、电影电视、体育运动）。这对于店铺非常重要。例如，如果你是卖运动类产品的，那么你的目标用户前期选择喜欢的内容就会向这些类型偏移。这就是第一个要点。
2.  **消费能力**：根据用户每年购买的消费能力来平均计算。小红书的运营系统很大程度上借鉴了其他平台（如淘宝）。系统通过用户12个月内的消费金额、订单数等数据进行等级划分。这些是隐藏属性，包括收藏、加购、关注等行为。用户隐藏属性越高，其购买行为对你店铺的商品排名提升帮助越大。
3.  **购买类型**：系统分析用户的购买行为，主要基于其在小红书上搜索或浏览过的产品。系统通常只会给用户推荐三个类型的产品。我们可以将其分为日用品、生活用品、户外用品、家具玩具等大类，也包括一些冷门产品。购买类型确认了用户的喜好，这正是我们店铺需要做的人群定位。小红书通过个人属性、消费能力、购买类型、购买地域和购买习惯这几个点，将庞大的用户群体划分成不同区域。
4.  **购买地域**：分析用户的地域性购买习惯。但这对我们影响不大，因为产品通常是面向全国销售的。小红书系统目前在这方面还不够强大，购买地域更多是一种记录，用于稍微提升人群推荐的精准性。
5.  **购买习惯**：指用户购买的搜索习惯。例如，用户喜欢先看第一家、再看第二家、最后在第三家购买；或者用户偏爱母婴、化妆品、服装等特定类目。系统通常只记录用户最关注的三个类目，并根据其喜好进行产品推荐。

这五个人群类型是小红书用户的人群画像，而非我们店铺的人群。我们需要结合小红书用户的人群行为习惯，再根据自己店铺的情况，最终确定店铺的目标人群。因此，必须先了解小红书用户的人群行为习惯。

只有了解了对应的人群，我们才好去运营店铺。也就是说，通过我们之前店铺定位中确定的模式、内部产品来分析人群。人群确定以后，后续操作店铺就会方便很多。这就是主体定位中的人群定位部分。
