# 抖音全套运营教程：P13：抖音短视频的前景 📈

![](img/5d9a8e24b6ce33b24d6cf9fe1867f0a3_1.png)

在本节课中，我们将要学习抖音短视频平台的整体前景，分析其市场潜力、用户构成以及为何它适合新手创作者起步。了解这些背景知识，是规划后续运营策略的重要基础。

## 平台数据与收入前景

首先，我们通过一张来自抖音创作者大会的数据报告图片来了解平台潜力。

![](img/5d9a8e24b6ce33b24d6cf9fe1867f0a3_3.png)

![](img/5d9a8e24b6ce33b24d6cf9fe1867f0a3_1.png)

该报告显示，已有超过2200万人在抖音获得收入，总收入超过417亿元。抖音平台计划将这个数字翻倍，目标是让创作者总收入达到800亿元。

![](img/5d9a8e24b6ce33b24d6cf9fe1867f0a3_3.png)

这意味平台将提供巨大的流量扶持。这里引出第一个知识点：抖音用户主要分为两类。

以下是这两类用户的定义：
*   **玩家**：指单纯在抖音娱乐、消费内容，未考虑通过平台赚钱的用户。
*   **创作者**：指主动制作并发布内容，意图通过抖音获取收益的用户。本课程的学员都属于此类。

## 为何选择抖音平台

市面上存在多个短视频平台，例如快手等。我们为何要专注于抖音？

这个选择逻辑类似于开设实体店：你需要寻找一个人流量大、位置合适的店面。做短视频同样需要选择一个流量庞大的平台。如果平台自身流量不足，便无法分给创作者，创作者也就难以获利。

抖音的日活跃用户超过8亿，月活跃用户更是达到40亿人次。这是一个极其庞大的流量池。

## 竞争压力与创作门槛

面对如此大的流量，竞争压力是否也会很大？答案是否定的。

平台上的创作者数量约为5000万。这里的“创作者”定义广泛，并非特指网红或达人。例如，一位邻居老太太用手机拍摄自己做饭的视频并上传到抖音，她也算作创作者。因此，90%的竞争压力实际上来源于像你我一样的普通用户。

接着看创作门槛。做抖音短视频是否需要购买专业设备？

**答案**：不需要。你只需要一部智能手机和一个手机号，即可注册抖音账号。抖音倡导全民通过短视频分享和记录生活，入门非常简单。

## 当前风口与内容方向

那么，当前有哪些值得关注的风口？

首先，抖音推出了“抖音计划”，旨在帮助1000名优质艺术创作者实现粉丝增长超过100万。可能有同学担心自己不会创作。这无需忧虑，正因为存在知识盲区，才需要系统学习。

市场空间巨大，内容需求旺盛，各行各业都容易产出内容。我们正处于大数据时代，平台会根据用户的喜好推荐内容。例如，你喜欢看帅哥，刷到的就多是相关内容。

这导致每个人看到的内容都相对片面。因此，无论你是什么身份，都有创作空间：

以下是不同身份的可创作方向举例：
*   **宝妈**：可以拍摄宝宝日常或烹饪过程。
*   **实体店主**：可以拍摄产品进行引流。
*   **普通上班族**：可以分享日常生活。

每月40亿的活跃用户中，只要你的内容不触犯法律和道德底线，不是低俗内容，总会有一部分人喜欢并关注你。之所以有些同学粉丝很少，是因为喜欢你的人还没有刷到你的作品。

对于新手而言，心情迫切可以理解，但不应将“涨粉”作为首要目标。

**正确的首要目标**：先把视频的播放量做起来。流量增长后，粉丝自然会随之而来。
