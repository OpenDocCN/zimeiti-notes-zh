# 小红书运营教程：P26：小红书账号矩阵与官方广告投放指南

在本节课中，我们将深入探讨小红书运营中的两个核心策略：如何构建有效的账号矩阵，以及如何利用小红书的官方广告系统进行推广。我们将理清自建账号引流与官方广告投放之间的关系，帮助你根据自身情况选择最合适的推广路径。

## 账号类型解析

上一节我们介绍了小红书的基本玩法，本节中我们来看看小红书平台上的账号主要分为哪几类。理解不同类型的账号及其用途，是制定运营策略的基础。

小红书账号主要可分为以下三类：

1.  **企业号**
    *   主要作用是树立官方品牌形象。
    *   拥有官方认证标识，可展示品牌名称、地址等信息。
    *   享有平台赋予的特权，便于开展官方业务。

2.  **个人号**
    *   目前大多数用户运营的账号类型，灵活性高。
    *   运营者身份可以是品牌员工、产品消费者等，利于塑造个人品牌特色。
    *   适合品牌相关人物或员工出镜，进行内容创作。

3.  **引流号**
    *   在以往课程中提及过，俗称“炮灰号”。
    *   数量可以是多个，作用相对次要，通常无需精细运营。
    *   可用于测试内容、参与活动，或在紧急情况下作为备用账号使用。
    *   这类账号通常没有长期的人格化规划。

一般而言，品牌的推广渠道主要是通过上述类型账号自行发布笔记进行引流。

## 构建企业账号矩阵的建议

了解了账号分类后，我们来看看如何系统地运用这些账号，构建一个协同作战的“账号矩阵”。以下是打造企业号矩阵的具体建议：

1.  **积极参与品牌号活动**：品牌企业号发起的试用、团购等活动，应安排多个小号（个人号或引流号）积极参与，营造热度。
2.  **个人号强化品牌属性**：运营个人号时，应在笔记内容中多次暗示品牌的定位。例如，品牌目标用户是都市青年还是职场新人，内容风格就应与之契合。
3.  **控制个人号的营销比例**：个人号发布品牌产品内容的频率不宜过高。一个较好的比例是 **营销内容：非营销内容 = 3：7**。即30%的内容用于产品推广，70%的内容用于种草分享或知识科普。
4.  **利用引流号进行评论引流**：当企业号发布内容后，可以使用引流号在评论区进行互动或引导，实现更隐蔽的引流效果。

## 官方广告投放：小红书CPC竞价广告

除了运营自有账号矩阵，另一种重要的推广方式是借助小红书平台官方的广告系统。接下来，我们将目光转向小红书的官方广告投放。

小红书官方广告主要指 **CPC（Cost Per Click）竞价广告**，即按点击付费的广告模式。其申请入口位于小红书官方网站的“业务合作”板块内，具体路径为：**业务合作 > 推广合作**。

## 官方广告的主要形式

![](img/e805a85fa7362455f19e4601b3928a1d_13.png)

拥有企业号后，即可投放信息流广告。官方广告主要分为两大类：

1.  **信息流广告**：用户浏览小红书首页“发现”页时，会穿插在正常笔记中的广告。形式原生，不易引起用户反感。
2.  **搜索广告**：当用户使用小红书搜索功能时，出现的与关键词相关的广告。其原理类似于百度的关键词竞价排名，通过对特定关键词出价，竞争不同的广告展示位。

搜索广告的核心逻辑是**关键词竞价**，公式可以简化为：
**广告排名 = f(出价， 广告质量)**  
广告主通过调整对关键词的出价，来影响广告在搜索结果中的展示位置。

目前，信息流广告常见的固定展示位主要是第3位、第13位和第23位等。

关于广告投放的更多常见问题，可以在小红书广告平台的官方帮助文档中查找。

![](img/e805a85fa7362455f19e4601b3928a1d_17.png)

需要指出的是，官方广告投放涉及一定的费用，且门槛和成本相对较高。

**建议**：在资金预算允许的情况下，可以考虑与官方合作投放竞价广告，因为它能直接获取由平台算法推荐的精准流量。
