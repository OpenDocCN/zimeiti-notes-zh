# 抖音运营教程：P33：扶持加权与降权限流

在本节课中，我们将学习抖音平台对视频内容的审核与流量分配规则。具体来说，我们将了解哪些内容会获得平台的“扶持加权”，从而获得更多推荐流量；以及哪些内容会导致“降权限流”，影响视频的传播效果。掌握这些规则，是做好抖音运营的基础。

上一节我们介绍了账号权重的分类，本节中我们来看看具体影响权重的视频内容规范。

## 🚀 获得扶持加权的内容特征

![](img/297be8299fcc02da26dbf1c55b1f9bdf_1.png)

平台会优先推荐符合以下标准的视频内容，这被称为“扶持加权”。

![](img/297be8299fcc02da26dbf1c55b1f9bdf_3.png)

以下是获得扶持加权的关键要素：

*   **视频格式**：必须为竖版移动端原生视频，即 **9:16** 的格式。这是抖音平台的原生视频格式。
*   **画面质量**：画面需保持平稳，作品清晰度高。
*   **音频质量**：视频中无过多的杂音或噪音。

![](img/297be8299fcc02da26dbf1c55b1f9bdf_1.png)

![](img/297be8299fcc02da26dbf1c55b1f9bdf_5.png)

这些要求较为基础，旨在保证基本的观看体验。关于此部分的详细资料，后续会整理成资料页供大家课后查阅。

## ⚠️ 导致降权限流的内容禁区

相反，如果视频内容触及以下禁区，则很可能被平台“降权限流”，即减少推荐甚至不予通过。

以下是会导致降权限流的具体内容类型：

*   **违规信息**：视频内容涉及吸烟、暴力、炫富（金钱）等不良信息。
    ![](img/297be8299fcc02da26dbf1c55b1f9bdf_3.png)
*   **外部引流**：在视频中留下微信、微博、淘宝等外部平台的联系方式。长时间出现商品Logo、手机号、二维码也属于此类。**注意**：二维码并非绝对禁止，但出现时间最长不要超过3秒。不过，在审核严格时，即使出现1秒也可能被驳回。
    ![](img/297be8299fcc02da26dbf1c55b1f9bdf_5.png)
*   **硬性广告**：进行明显的营销广告、推销或引流卖货行为。
*   **无口播的好物推荐**：这是需要特别注意的一点。许多同学在做好物推荐类账号时，容易踩坑。例如，仅展示产品拆箱过程或单纯展示物品，没有任何口头介绍（口播）。这种形式风险很高。

### 关于“无口播好物推荐”的详细说明

我们以几位同学的实际案例来理解这个问题。

**案例一**：一位同学在视频画面中直接标注了“**九块九**”的价格信息。这种直接涉及金钱和卖货的行为，导致其视频中的“小黄车”（购物车）功能被立即下架。

**案例二**：另一位同学推荐玩具，但全程没有说一句话，仅使用了背景音乐。这种“**无口播拆箱**”的形式同样容易被系统判定为硬性广告。

**正确的做法**：即便是推荐好物，也需要加入简单的口播介绍。例如，你可以说：“我儿子玩这个玩具，四个小时都没停下来。”通过这种带有场景和体验描述的口播，将推荐软化，视频就更容易通过审核。

总结来说，**千万不要不说话，直接展示产品、拆箱或做推荐**，这非常危险。至于直接口播广告、念联系方式等更明显的卖货行为，在抖音平台是行不通的，更适合其他电商平台。

本节课中我们一起学习了抖音平台“扶持加权”与“降权限流”的核心规则。关键在于制作符合平台格式（9:16竖屏）和质量要求的原生内容，同时严格规避违规信息、外部引流和硬性广告。特别是对于好物推荐类内容，务必加入自然的口播描述，避免单纯的无声展示。理解并应用这些规则，能有效帮助你避开雷区，让视频获得更好的初始流量。关于内容优化的更多技巧，我们将在后续课程中详细展开。
