# 亚马逊开店教程：P8：04、对跨境的态度 🇨🇳

在本节课中，我们将要学习国家对于跨境电商行业的态度与相关政策，理解为何当前是进入该领域的有利时机。

上一节我们介绍了跨境电商的基本概念，本节中我们来看看国家层面对此持何种态度。

## 国家的支持态度

我们国家是否支持我们从事跨境电商呢？要理解这一点，可以先思考一个简单的问题：出口和进口这两种贸易形式，哪一种对国家的经济更有利？

以下是两种贸易形式的简要对比：
*   **出口**：将本国产品卖给外国人。
*   **进口**：将外国产品买进本国。

回顾历史，在90年代至00年代，我国制造业尚不发达，许多日常用品如“洋火”（火柴）、“洋车”（自行车）都需要进口。长期处于贸易逆差（进口大于出口）状态对国家经济不利。相比之下，**出口能够为国家创造外汇收入，促进产业发展和就业，对经济更为有利**。

![](img/3a759d681bf8e2c596e621f34809cc0d_3.png)

因此，为了鼓励普通人参与到出口贸易中来，将事业做大做强，国家推出了一项关键政策。

![](img/3a759d681bf8e2c596e621f34809cc0d_5.png)

## 核心扶持政策：无票免税

为了鼓励大众参与跨境电商，国家提出了 **“无票免税”** 政策。

我们可以通过搜索“跨境电商 无票免税政策”来验证。根据官方新闻（如新闻联播）的报道，李克强总理曾明确指出：对跨境电商零售出口实施无票免税政策。

![](img/3a759d681bf8e2c596e621f34809cc0d_7.png)

**政策核心含义**：`你将中国产品通过跨境电商平台（如亚马逊）卖给外国人，所获得的收入，目前无需向国家缴纳相关税款。`

这与其他行业形成鲜明对比：
*   开设实体店（如超市、服装店）需要缴税。
*   经营国内网店，收入达到一定标准也需要缴税。
*   而现阶段经营跨境电商零售出口，享受**无票免税**优惠。

## 总结

![](img/3a759d681bf8e2c596e621f34809cc0d_9.png)

本节课中我们一起学习了国家对于跨境电商的态度。**国家现阶段非常鼓励跨境电商发展**，其核心体现便是为跨境电商零售出口量身定制的 **“无票免税”** 政策。这项政策降低了创业者的税务负担和运营门槛，是当前进入该领域的一大红利。理解这一宏观背景，能帮助我们更坚定地迈出跨境电商的第一步。
