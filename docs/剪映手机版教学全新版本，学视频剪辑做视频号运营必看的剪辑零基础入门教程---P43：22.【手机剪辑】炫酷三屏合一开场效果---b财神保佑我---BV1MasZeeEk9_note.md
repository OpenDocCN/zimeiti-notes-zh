# 视频剪辑入门：P43：22. 炫酷三屏合一开场效果制作教程 🎬

在本节课中，我们将学习如何使用剪映手机版制作一个动感的三屏合一视频开场效果。这个效果非常适合用于视频号等短视频平台的开场，能瞬间吸引观众注意力。

## 概述
我们将通过导入多段视频、运用画中画、蒙版及关键帧动画，配合特效，逐步构建出一个炫酷的三屏开场。整个过程简单明了，即使是零基础的初学者也能轻松跟上。

## 详细步骤

上一节概述了我们的学习目标，本节中我们来看看具体的操作流程。

### 第一步：素材导入与基础布局
首先，在剪映中导入四条视频素材。将时间轴上后添加的三段视频切入画中画轨道。调整这三段画中画视频的起始时间，使它们之间大约间隔0.5秒依次出现。

### 第二步：应用镜面蒙版
接下来，我们需要对前三段画中画视频分别应用蒙版来创造分屏效果。
以下是操作要点：
*   选中第一段画中画视频。
*   点击底部工具栏中的 **“蒙版”** 按钮。
*   在蒙版类型中选择 **“镜面蒙版”**。
*   将蒙版旋转至大约 **-70度** 的角度。
*   对第二段和第三段画中画视频重复上述操作，应用相同的镜面蒙版。

### 第三步：调整分屏位置
现在，屏幕上应该出现了三个带有倾斜角度的视频条。在预览窗口中，手动拖动调整这三段视频的位置，将它们并排排列，为后续的主视频留出显示空间。

### 第四步：主视频关键帧动画
关键帧动画是让视频“动”起来的核心。我们为主视频（即第四条，在底层主轨道的视频）制作一个蒙版展开动画。
1.  选中主轨道上的第四段视频。
2.  在其开始处添加一个**关键帧**。
3.  点击 **“蒙版”**，再次选择 **“镜面蒙版”**。
4.  在第一个关键帧处，将蒙版压缩成一条细线，并移动到屏幕一侧。
5.  将时间线拖动到大约第5秒的位置。
6.  在此处，将镜面蒙版向反方向旋转（例如从-70度调到70度），并将蒙版放大至完全显示视频内容。系统会自动在此处生成一个新的关键帧。

### 第五步：添加入场特效
为了让三个分屏的出现更有冲击力，我们需要为它们添加不同的入场动画。
以下是特效添加方案：
*   **左侧视频**：添加 **“右下甩入”** 入场动画，时长设置为1秒。
*   **中间视频**：添加 **“上下抖动”** 入场动画。
*   **右侧视频**：添加 **“向左下甩入”** 入场动画。

### 第六步：添加氛围特效
最后，为主视频增添动感氛围。在特效面板中，为主视频（第四段）添加 **“蹦迪光”** 和 **“灵魂出窍”** 特效。添加后，记得在特效参数中调整“作用对象”，确保特效仅应用于主视频。

## 总结
本节课中，我们一起学习了制作炫酷三屏合一开场效果的全过程。我们先后完成了**素材布局**、使用**镜面蒙版**创造分屏、利用**关键帧**制作蒙版展开动画，并为各分屏添加了**入场特效**和**氛围特效**。通过组合这些基础功能，你便能创造出专业且吸引眼球的开场效果。现在，你可以尝试使用自己的视频素材进行练习了。
