# 亚马逊FBM新手选品教程：P1：核心选品方法与店铺实操

## 概述
在本节课中，我们将学习亚马逊FBM（自发货）模式下的核心选品方法。课程将结合真实学员案例，演示如何通过数据化工具筛选潜力产品，并讲解多站点（如日本站）运营及跟卖策略的实操流程。我们将重点关注如何低成本、高效率地启动，并规避常见风险。

## 章节二：多站点运营与账户问题处理

在实际运营中，可能会同时操作多个亚马逊站点，并遇到如KYC审核等账户问题。正确处理这些问题至关重要。

上一节我们确立了运营基础，本节中我们来看看如何处理多站点和账户审核。

**欧洲站KYC审核要点**
KYC（Know Your Customer）是欧洲站的必要审核。需准备并提交准确的资料。
*   **所需材料**：公司营业执照、法人身份证、户口本页面、公司对公账单或法人个人账单（如水电煤、信用卡账单）。
*   **关键提示**：
    1.  确保提交的账单地址与身份证或营业执照地址一致。
    2.  不要随意修改后台已提交或待审核的信息，以免触发更严格的审核。
    3.  若审核要求提供公司章程等文件，可能需要专业美工处理。

**日本站开通与策略**
日本站距离中国近，物流时效相对较好，适合作为补充站点。
*   **开通**：开通日本站通常无需像欧洲站那样进行复杂的税务设置（如JCT），流程相对简单。
*   **策略分离**：建议不同运营模式使用不同站点。例如，北美站专注于FBM精铺，日本站则可尝试跟卖策略。确保账户活动分离，降低风险。

## 章节三：FBM跟卖选品实操详解

![](img/0f6d136ad283207f961bebc68de205f3_6.png)

跟卖是亚马逊上一种快速的出单方式，但需要严谨的选品来保证利润和规避风险。以下是利用工具进行数据化选品的核心步骤。

上一节解决了账户层面的问题，本节中我们进入核心环节：如何为跟卖筛选合适的产品。

### 3.1 选品工具与数据抓取
首先，我们需要借助工具从亚马逊前台抓取潜在的产品数据。

**操作流程如下：**
1.  在选品工具中设置目标站点（如日本站）。
2.  输入搜索关键词。
3.  设置关键筛选条件：
    *   **价格区间**：过滤掉过低或过高价格的产品，聚焦于有利润空间的区间。
    *   **卖家类型**：可筛选“非FBA”卖家，或根据物流情况灵活选择。
    *   **卖家数量**：卖家数量较少的产品，竞争压力相对较小。
    *   **近期销量**：关注上架时间短但销量增长快的产品。
4.  启动抓取，工具会自动收集符合条件的产品列表。

### 3.2 核心数据筛选与计算
抓取到原始数据后，需要通过关键指标进行人工筛选。

![](img/0f6d136ad283207f961bebc68de205f3_8.png)

![](img/0f6d136ad283207f961bebc68de205f3_10.png)

![](img/0f6d136ad283207f961bebc68de205f3_11.png)

**需要计算的核心指标是产品的日均销量估算：**
`日均销量 ≈ (近24小时销量 + 近7天销量 + 近30天销量) / 3 / 30`
这个公式帮助估算产品稳定的日销售能力，避免被单日爆发数据误导。

![](img/0f6d136ad283207f961bebc68de205f3_13.png)

**筛选逻辑如下：**
*   **剔除虚高数据**：对于销量数据波动巨大或看似不真实的产品，直接排除。
*   **关注新品**：上架时间短（如几天内）但已产生销量的产品，可能是市场热点。
*   **检查品牌与商标**：使用商标查询工具检查产品是否注册了品牌。优先选择**纯英文**或**纯日文**且无注册商标的品牌，警惕中英文混合的“钓鱼”品牌。
*   **评估利润空间**：即使产品单价低，但如果销量足够大，也能积累可观利润。毛利率达到30%-40%即可考虑。

### 3.3 货源与利润核算
确定潜力产品后，下一步是寻找货源并计算利润。

**操作步骤如下：**
1.  在1688等采购网站找到相同或类似产品的货源。
2.  使用利润核算表格，填入亚马逊售价、采购成本、头程运费、平台佣金等。
3.  计算最终利润。跟卖时，售价应对标当前**购物车持有者的价格**，而不是最高标价。
4.  确保最终利润符合预期。避免选择运费是采购成本数倍的大件产品。

### 3.4 跟卖上架与调价策略
使用跟卖辅助软件进行上架和价格管理。

**上架设置：**
*   在软件中创建跟卖任务，填入目标产品的ASIN。
*   库存数量设置为`1`，采取“卖完再补”的谨慎策略。
*   填写找到的货源采购链接。

**智能调价规则：**
*   设置“同步购物车”价格，确保自己的价格始终跟随当前购物车持有者。
*   可以设置“定时跟卖”规则，在特定时间段（如竞争对手休息时）自动上架跟卖，出单后即下架，减少被发现的概率。
*   设置价格上下限，防止系统调价异常。

## 章节四：风险控制与注意事项

跟卖模式伴随较高风险，必须提前建立风控意识。

上一节我们学会了如何操作跟卖，本节中我们来看看如何保护自己，安全运营。

![](img/0f6d136ad283207f961bebc68de205f3_15.png)

**以下是跟卖必须注意的风险点：**
*   **知识产权侵权**：绝对远离有商标、版权（如动漫形象、球队Logo、知名品牌）的产品。这是导致账户被封的最主要原因。
*   **恶意投诉应对**：同行竞争可能导致恶意投诉。核心在于**知道如何申诉**：保留好采购发票、与供应商的沟通记录等，作为申诉证据。
*   **切忌贪婪**：不要大量跟卖多个产品。精选1-2个能稳定出单的产品即可，做到“见好就收”。
*   **物流与成本**：避免销售体积大、重量重的产品，防止利润被高昂运费吞噬。

**数据维护建议：**
*   选品工具抓取的数据，如果无效或不符合要求，及时清空，重新设置条件进行下一轮抓取。
*   保持数据精准，而非数量庞大。
