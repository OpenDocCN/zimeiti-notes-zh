## 创作者标签与用户标签的概念

上一节我们介绍了课程的整体框架，本节中我们来看看平台算法的基石——标签系统。

创作者标签是指平台为内容发布者（即创作者）定义的属性，用于标识其账号主要发布哪一类内容。用户标签是指平台为浏览内容的用户定义的属性，用于标识其兴趣偏好。

平台无法直接精准分析视频或笔记的具体内容。当账号发布第一个内容后，平台会识别你为“创作者+用户”的双重身份，但此时你的创作者标签是空白的。平台需要通过观察用户对你内容的互动行为，来逐步判定并为你打上标签。

## 创作者标签是如何形成的

理解了基本概念后，我们来看看创作者标签具体是如何被平台赋予的。

这个过程的核心是：**平台通过观测有标签的用户对你内容的反馈数据，来反向推导你的内容标签**。你的作品发布后，平台会先给予一小部分基础流量进行测试。这部分流量会分发给带有各种标签的用户（如旅游、搞笑、美食、母婴等）。

![](img/df53611d9d083773e0c55b63be49afd3_5.png)

以下是平台判定创作者标签的简化流程：

![](img/df53611d9d083773e0c55b63be49afd3_7.png)

1.  **内容发布**：创作者发布一条内容（例如，一条关于美食的笔记）。
2.  **初始分发**：平台将内容推送给一小批具有不同标签的用户。
3.  **用户反馈**：不同标签的用户产生不同的互动行为。
    *   对内容不感兴趣的用户（如标签为“旅游”、“舞蹈”的用户）可能会快速划走。
    *   对内容感兴趣的用户（如标签为“美食”的用户）可能会停留更长时间、点赞或收藏。
4.  **数据积累与判定**：平台收集这些互动数据。如果数据显示，标签为“美食”的用户群体在你的内容上产生了显著的正面互动（如长停留、点赞），而其他标签的用户互动较弱，平台就会逐步为你的账号打上“美食”相关的创作者标签。
5.  **标签强化**：随着你持续发布同类优质内容，吸引更多“美食”标签用户互动，你的“美食”创作者标签会越来越清晰和牢固。

**公式表示**：
`创作者标签 = f(∑(用户标签_i × 互动权重_i))`
其中，`i` 代表观看你内容的第 `i` 个用户，`互动权重` 包括停留时长、完播率、点赞、收藏等行为的量化值。

**关键点**：平台判定内容类型，主要依赖的是**用户行为数据**，而非直接分析你的标题、文案或视频画面。虽然文案和标题有一定辅助作用，但核心逻辑是“千人千面”的用户兴趣匹配。

## 用户标签的特征与形成

在了解了创作者标签的来源后，我们再来看看与之协作的用户标签有何特点。

用户标签具有多样性、交叉性和可变性。一旦你开始使用小红书，平台就会立即开始为你建立初始标签。你的每一个互动行为都在塑造和更新你的用户标签。

以下是影响用户标签的主要行为维度：

*   **内容浏览时长与完播率**：在某篇笔记停留的时间越长，系统越认为你对该内容感兴趣。
*   **互动行为（点赞、收藏、评论）**：这是最直接的兴趣信号。你点赞和收藏了哪些内容，平台就会强化你相关的标签。
*   **主动反馈（点击“不感兴趣”）**：这会削弱你与该类内容相关的标签。
*   **用户画像信息**：包括地区、年龄、设备型号等后台数据，也会作为标签的参考维度。

用户标签是一个**交叉集合**。例如，一个用户的标签池中可能同时存在“美食”、“女装”、“考研”、“追星”等。平台通过分析这些标签的权重和组合，可以勾勒出清晰的用户画像（例如：“一位喜爱美食与时尚的女大学生”）。

**代码逻辑示意**：
```python
# 简化示例：更新用户标签
def update_user_tags(user, content_category, interaction_type):
    if interaction_type == "like" or interaction_type == "long_view":
        user.tags[content_category] += 1  # 强化该内容标签
    elif interaction_type == "dislike" or interaction_type == "quick_skip":
        user.tags[content_category] -= 1  # 弱化该内容标签
    # 根据所有标签的权重，为用户推荐内容
    recommend_content(user.tags)
```

**标签的可变性**：用户标签并非一成不变。如果你近期频繁浏览某一类新内容（如“露营”），相关标签的权重就会上升，你收到此类推荐的比例也会增加。反之，通过“不感兴趣”或快速划走，可以降低某些标签的权重。

## 标签系统的协同与意义

![](img/df53611d9d083773e0c55b63be49afd3_13.png)

![](img/df53611d9d083773e0c55b63be49afd3_15.png)

最后，让我们将创作者标签和用户标签联系起来，看它们如何协同工作，构成平台流量分发的核心。

整个系统形成了一个智能的“内容-用户”匹配循环：
1.  **平台通过用户标签理解用户喜欢什么**。
2.  **平台通过创作者标签理解内容属于什么类别**。
3.  **平台将具有特定创作者标签的内容，分发给具有匹配用户标签的人群**。
4.  **用户的互动行为数据，反过来验证并修正创作者标签的准确性**。

这个循环使得平台能够高效地将合适的内容推荐给感兴趣的人，从而提升用户粘性和内容分发效率。这也解释了为什么坚持垂直领域创作、吸引精准用户互动，对于账号成长至关重要。

![](img/df53611d9d083773e0c55b63be49afd3_17.png)

![](img/df53611d9d083773e0c55b63be49afd3_19.png)

同时，这也揭示了所谓“大数据”和“信息透明”的底层逻辑：我们在平台上的每一个行为都在被分析，并转化为定义我们数字身份的标签。

## 总结

本节课中我们一起学习了小红书平台流量分发的核心算法逻辑。

*   **创作者标签**：是平台根据**用户互动数据**反向推导出的账号内容属性，而非直接分析内容本身。它决定了你的内容会被推荐给哪类用户。
*   **用户标签**：是平台根据用户的**浏览时长、互动行为、画像信息**等综合形成的兴趣偏好集合。它具有多样性、交叉性和可变性，决定了用户会看到什么内容。
*   **协同作用**：两者共同构成了“千人千面”推荐系统的基础。理解这一点，就能明白为何“垂直定位”和“吸引精准用户互动”是起号阶段最关键的任务。
