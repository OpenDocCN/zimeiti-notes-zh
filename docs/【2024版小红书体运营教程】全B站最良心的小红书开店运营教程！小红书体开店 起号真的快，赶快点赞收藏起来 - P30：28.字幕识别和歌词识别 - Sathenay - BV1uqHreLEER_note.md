# 小红书运营教程：P30：28.字幕识别和歌词识别 📝

在本节课中，我们将学习剪映软件中文本工具的两个核心功能：智能字幕（包括字幕识别与文稿匹配）和歌词识别。这些功能能极大提升视频制作的效率，帮助你快速为视频添加准确的字幕。

上一节我们介绍了文本工具的基础操作，本节中我们来看看如何利用智能功能高效生成字幕。

![](img/9b35c8f45be499dc8b4a373e21cb413a_1.png)

## 智能字幕功能详解

![](img/9b35c8f45be499dc8b4a373e21cb413a_3.png)

智能字幕功能主要包含两种模式：**识别字幕**和**文稿匹配**。识别字幕能自动识别视频中的人声并生成字幕；文稿匹配则允许你输入已有文稿，软件会自动将其与视频画面进行匹配。

以下是使用“识别字幕”功能的步骤：
1.  将带有人声的视频素材拖入时间轴。
2.  在左侧工具栏选择“文本”工具。
3.  点击“智能字幕”下的“开始识别”。

![](img/9b35c8f45be499dc8b4a373e21cb413a_1.png)

识别过程中，可以勾选“同时清空已有字幕”。此选项适用于重新识别字幕，以修正错别字或补充遗漏内容，新字幕会完全替换旧字幕。

![](img/9b35c8f45be499dc8b4a373e21cb413a_3.png)

识别完成后，字幕会出现在时间轴上。默认的白色字体在白色背景下可能不清晰，此时可以更改预设样式。

## 字幕的检查与批量修改

识别出的字幕通常比较准确，但仍需检查。你可以单击时间轴上的字幕块，在右侧编辑面板中直接修改文字。

![](img/9b35c8f45be499dc8b4a373e21cb413a_5.png)

对于通过“识别字幕”功能生成的字幕，软件提供了一个非常便捷的特性：**批量修改样式**。你只需选中其中任意一个字幕，修改其字体、颜色、大小、气泡或花字等基础属性，所有通过识别生成的字幕样式都会同步更新。这确保了整个视频字幕风格的一致性。

![](img/9b35c8f45be499dc8b4a373e21cb413a_7.png)

**公式表示**：`修改单个识别字幕的属性 => 所有识别字幕的属性同步更新`

## 使用文稿匹配功能

如果你已有视频对应的完整文字稿，使用“文稿匹配”功能效率更高。

以下是操作流程：
1.  在“智能字幕”区域点击“开始匹配”。
2.  在弹出的窗口中粘贴或输入你的文稿。
3.  点击“开始匹配”，软件会自动将文稿分段并与视频时间轴对齐。

## 为字幕添加动画效果

字幕不仅可以静态显示，还可以添加动画效果以增强观感。在文本的编辑面板中，你可以找到“动画”选项。

动画分为三类：
*   **入场动画**：字幕出现时的效果。
*   **出场动画**：字幕消失时的效果。
*   **循环动画**：字幕持续显示期间的效果。

**需要注意的是**：为字幕添加动画效果是**针对单个字幕块**的操作，不会像修改基础样式那样批量应用。你可以根据每个字幕的呈现需求单独设置。

## 文本朗读功能

“朗读”功能可以将文字转换成AI语音。选中一个字幕或文本块，点击“朗读”，可以选择多种音色（如“甜美女生”、“新闻男声”等）。

**代码示例**（操作逻辑）：
```plaintext
1. 选择文本轨道 -> 2. 点击“朗读” -> 3. 选择音色 -> 4. 点击“开始朗读”
```

生成后，会新增一条AI配音的音频轨道。若想为多个文本批量生成朗读，可以框选所有文本轨道后，再执行朗读操作。

使用此功能时，**务必关闭或调低视频原声**，以避免与AI配音冲突。

## 歌词识别与本地字幕

对于音乐类视频，应使用“识别歌词”功能。它与“识别字幕”的核心区别在于算法优化：**识别歌词**专门针对有旋律、有伴奏的歌曲音频；**识别字幕**则针对清晰的人声解说。

你还可以导入本地字幕文件。剪映支持 **SRT、LRC、ASS** 等格式的字幕文件。同样，剪映导出的视频也可以选择单独导出 **SRT** 或 **TXT** 格式的字幕文件。
