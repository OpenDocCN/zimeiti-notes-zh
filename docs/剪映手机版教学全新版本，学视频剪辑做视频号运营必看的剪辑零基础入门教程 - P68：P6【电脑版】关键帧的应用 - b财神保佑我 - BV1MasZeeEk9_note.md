# 剪映PC版教程：P6：美颜美体与关键帧应用 🎬

![](img/dce6a762049a7afbd4e7d05276b3941f_0.png)

在本节课中，我们将学习剪映PC版的两个核心功能：**美颜美体**和**关键帧动画**。美颜美体功能可以智能优化人物形象，而关键帧则是制作动画效果的基础。掌握这两项技能，能让你的视频更具表现力。

## 美颜美体功能详解 💄

上一节我们介绍了调节与调色面板，本节中我们来看看剪映PC版另一个强大的功能——美颜美体。此功能专为优化视频中的人物画面设计。

美颜美体功能通过识别视频中人物的五官，进行针对性的美化处理。请注意，此功能仅对包含清晰人脸的画面有效，对于风景、物体特写等非人物画面则无法识别。

以下是美颜美体的主要操作步骤：

1.  **开启与识别**：在视频控制面板的“基础”选项中，点击“美颜美体”。软件会自动追踪并识别人物脸部。
2.  **基础美颜**：在“美颜”选项卡下，可以进行肤色调整、磨皮、去法令纹、去眼袋、美白等操作，使皮肤看起来更均匀、光滑、通透。
3.  **肤色调整**：可以改变整体肤色倾向，例如调整为冷白皮或暖黄皮，并通过滑块控制调整程度。
4.  **面部重塑（美形）**：在“美形”选项卡下，可以精细调整面部轮廓。例如：
    *   **瘦脸**：打造小V脸效果。
    *   **面部比例**：调整下颚骨、颧骨宽度，以及上庭、中庭、下庭的长度。
    *   **五官调整**：单独调整眼睛大小、眼距、鼻子大小与形状、嘴巴大小与形状（可制作微笑唇），以及眉毛间距与形状。
5.  **手动微调**：可使用“手动瘦脸”功能，通过拖拽控制点自定义调整脸部轮廓，但需谨慎使用以避免失真。
6.  **智能美妆**：软件提供多种妆容“套装”，一键应用即可自动为人脸添加合适的口红、眉毛、眼影等妆容，并可单独调整各妆效的强度与颜色。
7.  **美体功能**：对于全身画面，可以使用“媒体”功能调整身高、瘦身、肩宽、腰型、天鹅颈等，使人物体态更优美。

完成调整后，可以对比处理前后的效果。该功能能智能追踪人脸，即使人物有动作，妆容也能自然跟随，效果逼真。

## 关键帧动画原理与应用 🎞️

看完美颜美体功能，本节我们来学习视频动画的核心——关键帧。要理解关键帧，首先需要了解动画的基本原理。

**帧**是视频的基本单位。视频由一系列连续的静止画面（即帧）快速播放而成。**帧速率**表示每秒播放的帧数，例如25帧/秒。**关键帧**则是用来记录画面属性（如位置、大小、透明度）在特定时间点状态的标记点。

![](img/dce6a762049a7afbd4e7d05276b3941f_2.png)

**关键帧的工作原理**是：在动画的起始点和结束点分别打上关键帧，并设置不同的参数（如不同的位置），软件会自动计算并生成中间帧，从而形成平滑的动画过程。公式可以简单理解为：

`起始关键帧（状态A）` + `结束关键帧（状态B）` = `软件自动生成的中间动画`

![](img/dce6a762049a7afbd4e7d05276b3941f_6.png)

理解了关键帧的底层逻辑后，我们就可以利用它制作各种动画效果，如位移、缩放、旋转、透明度变化等。

![](img/dce6a762049a7afbd4e7d05276b3941f_8.png)

在剪映PC版中，几乎所有素材（视频、图片、贴纸、文本、特效）的属性，只要后面带有一个**菱形图标** `◇`，都可以添加关键帧。

以下是关键帧的基本操作演示：

1.  **缩放动画**：选中一个素材（如一个小猪贴纸）。在时间线起始处，点击缩放属性后的菱形图标添加关键帧，记录当前大小。将时间线滑块拖到结尾，修改缩放数值（如调小）。软件会自动生成从小到大的缩小动画。
2.  **位移动画**：在起始处添加关键帧，将素材移动到画面左侧。在结束处添加关键帧，将素材移动到画面右侧。这样就生成了从左到右的移动动画。可以结合缩放关键帧，制作一边移动一边缩小的复合动画。
3.  **旋转动画**：在起始处为旋转属性添加关键帧。在结束处，将旋转数值设置为360度。素材就会产生旋转一周的动画效果。
4.  **透明度动画（淡入淡出）**：常用于视频转场。在视频片段开头，为不透明度添加关键帧并设为0%。在稍后的时间点，添加第二个关键帧并将不透明度恢复为100%，这就形成了淡入效果。在片段结尾处反向操作，即可实现淡出效果。
5.  **控制动画速度**：两个关键帧之间的时间距离越长，动画变化速度就越慢；距离越短，变化速度就越快。

关键帧的应用非常广泛，除了上述基础效果，还可用于制作抖动特效、复杂的小动画以及高级的视频特效合成。
