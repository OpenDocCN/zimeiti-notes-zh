### 第一步：内容审核

![](img/71ea42ec45b609b274ed4984aa17bbdd_8.png)

笔记发布后，系统首先会进行内容审核。审核主要关注以下几点：

*   **敏感词与违规内容**：检查笔记中是否包含平台禁止的敏感词汇、限制准入的类目或其他非法信息。
*   **原创性与重复度**：判断内容是否为抄袭，以及素材是否高度重复或非原创。

内容审核是笔记能否进入下一流程的门槛。

### 第二步：笔记收录

内容审核通过后，笔记进入收录环节。收录与否由两个核心因素决定：

![](img/71ea42ec45b609b274ed4984aa17bbdd_10.png)

![](img/71ea42ec45b609b274ed4984aa17bbdd_11.png)

![](img/71ea42ec45b609b274ed4984aa17bbdd_12.png)

1.  **账号权重**：账号在系统流量层级中的位置。权重越高，初始曝光基础通常越好。
2.  **笔记质量**：由用户的行为数据决定。例如，在一定的初始曝光量下，笔记的点击率（阅读数/曝光量）是关键的衡量指标。

**公式示例**：
`笔记点击率 = 笔记阅读数 / 笔记曝光量`

即使账号权重高，如果笔记点击率远低于同层级平均水平，也可能不被收录。反之，新账号的笔记若在有限曝光下获得不错的点击率，被收录的可能性则会增加。

### 第三步：特征提取与打标签

笔记被收录的同时，系统会对其内容进行分析，提取特征并打上标签。标签来源包括：

*   **用户标签**：浏览、互动该笔记的用户身上带有的兴趣标签。
*   **账号历史数据**：账号以往内容被系统认定的领域标签。
*   **内容关键词**：从标题、正文中拆分和提炼出的核心词汇。

系统会综合这些信息，判断笔记所属的领域（如旅游、美妆、数码等），并为笔记打上相应的标签。这是后续参与关键词排名的基础。

### 第四步：内容索引与动态排名

这是最核心的环节。系统会根据提取的标签，将笔记放入不同关键词对应的“索引库”中进行初步排序。

![](img/71ea42ec45b609b274ed4984aa17bbdd_14.png)

![](img/71ea42ec45b609b274ed4984aa17bbdd_15.png)

**我们可以通过一个例子来模拟这个过程：**

假设我们发布了一篇标题为 **《8月去云南旅游的完整攻略，学生党必看完整避坑攻略》** 的笔记。

![](img/71ea42ec45b609b274ed4984aa17bbdd_17.png)

1.  **标签提取**：系统可能从中提取出“云南”、“旅游”、“云南旅游”、“学生”、“避坑”等标签。
2.  **初始索引**：笔记会被分别放入这些标签下的索引库，并有一个初始排名。例如：
    *   在“云南”标签下，排名第7位。
    *   在“旅游”标签下，排名第6位（假设是F位）。
    *   在“云南旅游”标签下，排名第4位（D位）。
3.  **动态调整**：索引库中的排名并非固定不变，它会根据后续的用户行为数据动态调整。
    *   **排名上升**：如果在“云南旅游”这个关键词下，用户点击率高、互动好，笔记在该标签下的排名就会上升（例如从D位升到B位）。
    *   **排名下降或移除**：如果在“学生”这个标签下，因为内容不垂直，长期没有相关搜索用户点击，笔记在这个标签下的排名会下降，甚至最终被移出该标签的索引库。

**影响动态调整的核心因素**，即我们上节课讲到的：搜索点击率、搜索场景下的用户互动行为（点赞、收藏、评论）等。

### 第五步：动态展示与结果呈现

现代新媒体平台非常智能，为了避免内容同质化，并将流量更合理地分发给不同创作者，搜索排名引入了“动态展示”机制。

这意味着，**即使是同一个关键词，不同用户搜索时，看到的搜索结果排序也可能不同**。系统会根据该用户的历史兴趣和行为，在多个备选索引库中挑选最合适的排序结果进行展示。

**代码逻辑示意（伪代码）**：
```python
# 当用户搜索关键词时
def get_search_results(keyword, user_profile):
    # 1. 获取该关键词下的多个可能排序列表（索引库）
    possible_rankings = fetch_index_libraries(keyword)
    # 2. 根据用户画像，选择最匹配的一个排序结果
    best_ranking = select_best_ranking(possible_rankings, user_profile)
    # 3. 返回最终的搜索结果
    return best_ranking
```
因此，笔记的搜索排名是“因词而异”和“因人而异”的动态结果。
