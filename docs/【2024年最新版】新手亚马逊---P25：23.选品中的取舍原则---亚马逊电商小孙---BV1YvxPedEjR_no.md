# 亚马逊跨境电商运营教程：P25：23.选品中的取舍原则 🎯

在本节课中，我们将要学习亚马逊选品过程中一个至关重要的环节——取舍原则。我们将探讨为何并非所有精心挑选的产品都能成为爆款，以及当产品表现不及预期时，应如何科学、果断地进行取舍，以优化资源分配和运营效率。

在进行了前面的市场与数据分析后，我们正式进入选品环节。然而，选品过程中还有一个关键点，即需要进行取舍。

并非所有通过前期分析选出的产品，都能在实际运营中成功打造成爆款并取得良好销量。产品上线后，需要真正接受市场的考验。

尽管我们可能借助了各种数据和工具，但当产品实际投入运营时，结果常常与预期存在偏差。面对这种偏差，一方面我们需要尽力而为，但最终并非每款产品都能成功。

对于那些经过努力仍无法打造成功的产品，我们需要采取行动：保留好的产品，果断舍弃不好的产品。通常，选择10款产品可能产生的结果是：有2-3款能成为爆款，甚至成为Best Seller；另有4-5款保持平稳销售；最后2-3款则可能销量极低，与预期严重不符。

对于最后这类产品，即便经过多角度、全方位的推广，销量依然无法提升。此时，我们只能选择舍弃。这是因为我们需要将有限的精力与可调动的资源，集中应用到更有潜力的产品上。

那么，应依据何种原则对产品进行舍弃呢？我的建议是：在自身能力可接受的范围内，果断舍弃。纠结于守着没有起色的产品没有意义。

当然，有些卖家会采取另一种较好的处理方式：以低价清仓来舍弃产品。在整个销售过程中，价格是最敏感的因素。因此，当产品在预期价格下无法售出，并且占用了库存、影响了运营节奏或耗费了大量精力时，可以尝试降价清仓以清除库存。清仓后，该产品便不再作为实际销售的产品。

此外，还有另一种情况：某些产品可能客单价不高，但由于前期选品失误，在后期运营中才发现问题巨大。此时产品销量极低，库存数量大，不仅占用了亚马逊仓储空间（FBA仓位），还会影响整体的运营思路。

如果你的精力允许，可以果断选择完全舍弃这些产品。众所周知，货物存放在亚马逊仓库会产生仓储费。与其让货物无法销售地存放在那里，不如果断舍弃更为有效。

在此阶段，我们可能会面临一部分成本损失。但相较于当前损失的成本与后续整体运营所能产生的效益，经过权衡后，舍弃可能是更明智的选择。

这里我给大家举一个简单的例子。我曾接触过一位卖家，初期他规模较小，采用自发货模式，每天约有10-20单。经过交流并分析其店铺后，我发现其产品在选品上存在问题，并且他使用了跟卖等方式，优化也不到位。

整体分析后，这位卖家果断舍弃了原有的自发货产品，进行了重新选品。同时，他将发货方式全部调整为FBA。在起步阶段，他直接为每款产品发了约1000个库存到FBA仓库。众所周知，新卖家的FBA标准尺寸库存容量上限是5000个。他发了5款产品后，仓位就已占满。

在销售过程中，他运用了一些营销手段，例如联系买家刷评、赠送样品获取测评、使用站内广告等。但实际运营结果并未达到他的预期：5款产品中，有3款销售得不错，但另外2款基本没有销量。经过一个多月，这两款滞销产品各自只售出了十几个。

此时，经过一个多月的反思和再次选品，他已经发现了更具潜力的产品。他急于处理当前仓位以补进新货，但已无法补仓。于是，他果断采取了一个方式：直接让亚马逊销毁处理这两款滞销产品。

当我得知时，与他进行了简单沟通。我问他，这样舍弃意味着损失了约3万元人民币的成本，是否代价太高？他的回答是：如果不舍弃，他将需要耗费大量精力在这些滞销产品上。与其纠缠于此，不如将更多的精力和资源投入到有潜力销售更好的潜在优质产品上。因此他做出了这个决定。

当然，我并非主张每位卖家都对产品做如此处理。如果你有更好的方式，例如在美国或欧洲仓库有固定客户，当然可以将FBA仓的产品撤仓，送给客户或通过其他渠道销售。但如果你希望尽快在亚马逊平台上发展，就必须采取积极主动的方式进行处理。

结果是，刚才案例中的那位卖家，经过三四个月的打造，逐步培养起了自己的优势产品。大约4个月后，他的日均销售额达到了3000美金左右。他高兴地告诉我，当前这些选品的利润率基本都在40%左右。回顾之前舍弃产品的举动，他仍然觉得非常有效，至少对他后续整体销量的提升起到了不小的促进作用。

这个案例分析告诉我们，面对具体情况时，必须采取积极有效的方式，当然，这需要结合个人的实际情况。

接下来，我们看看下一页PPT。在选品中，我提到了“选十成三”的大致比例。那么，我们为何要在选品中进行这种取舍呢？原因主要有以下三点。

以下是三个核心原因：

1.  **降低仓储成本，腾出仓位空间**：正如案例中提到的，对于新卖家，FBA仓位只有5000个。虽然随着销量增长，亚马逊会扩展我们的仓位，或者在特定情况下我们可以主动联系客服申请扩容，但及时清理滞销品是管理仓储成本、为新品腾出空间的基础。
2.  **节省精力，专注于热销产品的开发和打造**：如果我们将精力平均倾注到所有产品上，那么用于有潜力产品的精力就会不足。果断舍弃无法做好的产品，意味着我们可以将更多精力专注地投入到新品的开发上。我认为，新品开发是贯穿整个亚马逊运营始终的行为，只有不断开发新品、不断打造，整个运营才能进入良性循环。

本节课中，我们一起学习了亚马逊选品的取舍原则。我们明白了市场检验的重要性，认识到并非所有产品都能成功，并学习了在面对滞销品时，应基于仓储成本、精力分配和运营效率进行果断舍弃。通过案例，我们看到了合理取舍如何帮助卖家集中资源，最终推动整体业务向更健康、更盈利的方向发展。记住，有舍才有得，在亚马逊运营中，科学的取舍是走向成功的关键一步。
