# 抖音运营教程：P18：10、如何寻找文案-1 📝

在本节课中，我们将要学习如何为抖音视频寻找和准备优质的文案。文案是视频内容的核心，好的文案能有效吸引观众，提升视频的完播率和互动率，从而帮助账号获得更多流量。对于初学者而言，写文案可能是一个难题，本节将介绍几个实用的在线工具，帮助你轻松获取和生成文案，甚至解决配音问题。

上一节我们讨论了优化作品的重要性，本节中我们来看看如何具体解决文案创作这个首要难题。

## 如何使用淘金阁寻找文案

进入淘金阁网站后，你会看到主页有非常多的内容分类。

![](img/15405f4504025781a4a421a9f876d7b2_3.png)

![](img/15405f4504025781a4a421a9f876d7b2_3.png)

以下是主要的分类方向：

*   财经
*   军事
*   动漫
*   科技
*   美食
*   美文
*   星座
*   体育
*   旅游
*   育儿
*   汽车
*   社会
*   娱乐
*   时尚
*   房产
*   国际
*   宠物

你可以根据自己账号的定位，点击相应的分类。例如，点击“情感”类型，页面中会展示大量相关话题文案，并且通常有多页内容可供浏览。

![](img/15405f4504025781a4a421a9f876d7b2_5.png)

![](img/15405f4504025781a4a421a9f876d7b2_5.png)

在挑选文案时，可以套用一个简单的公式：**一个群体 + 一个核心话题 + 疑问**。例如，“宝妈如何应对孩子的叛逆期？”。运用这个思路，你就能从海量文案中筛选出合适的内容。

![](img/15405f4504025781a4a421a9f876d7b2_7.png)

如果分类列表中没有你想要的特定内容，可以使用网站的搜索功能。

![](img/15405f4504025781a4a421a9f876d7b2_7.png)

在搜索框输入关键词，例如“宝妈”，然后点击“开始抓取”。系统会为你找出所有包含该关键词的文案，这些结果通常足够使用。

![](img/15405f4504025781a4a421a9f876d7b2_9.png)

![](img/15405f4504025781a4a421a9f876d7b2_9.png)

手机端和电脑端的操作方法完全一致，无需担心设备差异。

![](img/15405f4504025781a4a421a9f876d7b2_11.png)

## 利用工具生成配音

![](img/15405f4504025781a4a421a9f876d7b2_13.png)

解决了文案问题后，部分同学可能担心自己的声音不够好听或不适合出镜配音。淘金阁等工具也提供了在线配音生成功能。

点击网站右上角的“小工具”，然后选择“配音在线生成”功能。

![](img/15405f4504025781a4a421a9f876d7b2_11.png)

进入配音界面后，你会发现这里有非常丰富的音色可供选择。

![](img/15405f4504025781a4a421a9f876d7b2_15.png)

![](img/15405f4504025781a4a421a9f876d7b2_13.png)

![](img/15405f4504025781a4a421a9f876d7b2_17.png)

以下是部分音色类型：

*   亲和女生
*   温柔女生
*   甜美女生
*   自然女声
*   卖场广播
*   亲切女声（直播）
*   标准男声/女声
*   童声
*   各地方言（如四川话、粤语、东北话）
*   外语配音
*   情感解说
*   影视解说

![](img/15405f4504025781a4a421a9f876d7b2_19.png)

生成配音的操作非常简单。在左侧的“配音文本”框中输入或粘贴你准备好的文案。

![](img/15405f4504025781a4a421a9f876d7b2_17.png)

例如，输入：“欢迎来到小月老师的抖音直播课堂”。

接着，在右侧选择你喜欢的音色，例如“东北话（大虎）”。你还可以调整音量、语速和语调，使配音效果更符合你的视频风格。

![](img/15405f4504025781a4a421a9f876d7b2_21.png)

![](img/15405f4504025781a4a421a9f876d7b2_21.png)

![](img/15405f4504025781a4a421a9f876d7b2_23.png)

调整完毕后，点击“立即生成”。如果是手机端使用，通常需要扫码登录一下即可免费使用。

![](img/15405f4504025781a4a421a9f876d7b2_25.png)

![](img/15405f4504025781a4a421a9f876d7b2_25.png)

生成后，在配音列表中找到你的作品，点击下载即可将音频文件保存到本地。

![](img/15405f4504025781a4a421a9f876d7b2_27.png)

![](img/15405f4504025781a4a421a9f876d7b2_27.png)

这个音频文件可以在后续的视频剪辑环节直接使用。
