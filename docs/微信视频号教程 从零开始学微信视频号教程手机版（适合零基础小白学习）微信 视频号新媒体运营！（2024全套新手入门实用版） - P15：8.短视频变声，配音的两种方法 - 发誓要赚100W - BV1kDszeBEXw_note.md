# 微信视频号教程：P15：短视频变声与配音的两种方法 🎬

在本节课中，我们将学习为混剪视频进行声音去重的两种实用方法。这两种方法能有效改变视频的原声，帮助你的内容通过平台审核，同时保持视频的吸引力。

上一节我们介绍了视频剪辑的基础操作，本节中我们来看看如何具体处理视频中的声音。

## 方法一：使用剪映内置变声功能

![](img/fecdf52e4ba40608e96bc08762e3388c_1.png)

第一种方法直接在剪映App内完成，操作简单快捷。此方法通过改变原视频人声的音色来实现去重，但会保留视频原有的背景音乐。

以下是具体操作步骤：

1.  在剪映中导入需要处理的视频素材。
2.  选中时间轴上的视频片段。
3.  在屏幕下方功能栏向左滑动，找到并点击 **“声音效果”** 选项。
4.  在变声效果列表中，选择你喜欢的音效，例如“男生”、“大叔”或“卡通”等。
5.  部分音效支持调节参数，你可以进一步微调声音效果。

通过以上步骤，视频的人声部分就完成了变声处理。这是一种快速的声音去重方式。

## 方法二：使用配音神器重新生成配音

接下来我们看看第二种方法。这种方法更为彻底，它需要先将视频文案提取出来，修改后使用配音软件生成全新的音频文件，再替换掉原视频的声音。

以下是具体操作流程：

![](img/fecdf52e4ba40608e96bc08762e3388c_3.png)

1.  **提取与修改文案**：
    *   使用配音神器或其他工具的“文案提取”功能，从原视频中提取出文字内容。
    *   对提取出的文案进行适当修改。例如，将“老人”改为“大人”，将“怎么办呢”改为“咋办呢”。修改时注意不要过多增减字数，以保持语意连贯。

![](img/fecdf52e4ba40608e96bc08762e3388c_5.png)

2.  **生成新配音**：
    *   将修改后的文案复制到配音神器的“制作配音”页面。
    *   选择一个你喜欢的声音角色（如“男生”）。
    *   点击 **“合成配音”** 并导出生成的音频文件。

![](img/fecdf52e4ba40608e96bc08762e3388c_7.png)

3.  **在剪映中替换音频**：
    *   重新在剪映中导入原视频，并关闭视频的 **“原声”**。
    *   点击 **“添加音频”** -> **“提取音乐”**，导入刚才生成的新配音文件。
    *   此时，视频的配音和背景音乐都已改变。你可以根据需要，再添加新的背景音乐，并调整音量大小。

这种方法实现了声音和背景音乐的完全替换，去重效果更显著。

本节课中我们一起学习了两种为短视频进行声音去重的方法：一是利用剪映的变声功能快速改变音色；二是通过提取文案、修改并重新配音来彻底替换原声。你可以根据视频素材和需求选择合适的方法，结合之前学习的混剪技巧，制作出更具原创性的视频内容。
