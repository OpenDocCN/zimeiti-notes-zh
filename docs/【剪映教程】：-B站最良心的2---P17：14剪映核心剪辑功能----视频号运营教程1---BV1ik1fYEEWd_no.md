# 剪映核心剪辑功能教程：P17：14 剪映核心剪辑功能--贴纸 🎨

在本节课中，我们将要学习剪映的核心剪辑功能之一——贴纸。贴纸功能可以为视频增添趣味性、提升画面质量，并有效传递信息。我们将从贴纸的作用讲起，然后进入实际操作环节，学习如何添加、选择和设置贴纸。

## 贴纸功能概述

上一节我们介绍了其他剪辑功能，本节中我们来看看贴纸。在日常制作Vlog、美食或旅拍视频时，我们经常看到视频中插入各种好看的贴纸。过去，贴纸常用于照片修图。如今，在视频制作中添加贴纸，同样能显著提升视频的画面质量和信息传递效果。

![](img/4d53fe581a953b65b90e252ca2c7a1c0_1.png)

![](img/4d53fe581a953b65b90e252ca2c7a1c0_2.png)

在剪映中运用贴纸添加功能，可以从丰富的贴纸库中选择元素。添加平台热门的贴纸能提升视频的炫酷程度。例如，在制作换装或变身类视频时，添加贴纸能大大增强画面感。对于日常Vlog，添加贴纸也能提升视频的形象和趣味性。

关于贴纸的介绍就到这里。接下来，我们进入贴纸的实操环节。

## 贴纸添加与设置实操

以下是贴纸添加与设置的具体步骤。

![](img/4d53fe581a953b65b90e252ca2c7a1c0_4.png)

1.  **新建项目并导入素材**：首先，在剪映中新建一个项目，并导入需要编辑的视频素材。
2.  **进入贴纸菜单**：在下方的一级菜单栏中，找到并点击 **“贴纸”** 按钮。
3.  **选择贴纸**：点击后，会进入贴纸素材库。库中包含大量分类贴纸，例如热门表情、日常Vlog类、文字类、边框、动植物等。可以根据视频题材类型，选择合适的贴纸。
    *   **代码示例（操作路径）**：`主界面 -> 底部菜单 -> 点击“贴纸”图标`
4.  **添加贴纸**：点击选中的贴纸，它就会被添加到视频轨道上方的贴纸轨道中。
5.  **调整贴纸时长**：添加后，可以拖动贴纸轨道两端的白边，来设置贴纸显示的时长。可以让它贯穿整个视频，也可以只出现在特定片段。
    *   **公式描述（时长设置）**：`贴纸显示时长 = 拖动轨道端点进行设置`
6.  **预览效果**：设置完成后，可以点击播放按钮预览贴纸在视频中的效果。
7.  **添加动画（可选）**：选中贴纸轨道，还可以为贴纸设置入场、出场等动画效果，让它的出现形式更生动。

![](img/4d53fe581a953b65b90e252ca2c7a1c0_6.png)
