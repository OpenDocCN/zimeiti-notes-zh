# 抖音运营教程：P14：8.影视推广平台

![](img/6e38e67d6b18a63c99cdb7ca32a36a76_1.png)

在本节课中，我们将要学习如何通过“影视推广”平台，在抖音、快手等平台发布视频并获得收益。我们将详细讲解其运作模式、收益结算方式以及具体操作步骤。

![](img/6e38e67d6b18a63c99cdb7ca32a36a76_3.png)

---

许多同学会问，如果想把视频发布到抖音或快手平台，应该如何结算收益。接下来，我将为大家详细讲解。

![](img/6e38e67d6b18a63c99cdb7ca32a36a76_5.png)

正如之前提到的视频案例，我们剪辑完成后可以发布到30多个自媒体平台。除了抖音和快手，其他平台通常可以依靠视频本身的播放量来结算收益。

然而，抖音和快手这两个平台有所不同。就像我讲到的，视频发布到这两个平台，最主要的作用可能是增长粉丝。这就是我要强调的重点。

![](img/6e38e67d6b18a63c99cdb7ca32a36a76_7.png)

有没有同学只想专注于抖音和快手这两个熟悉的平台？如果你想同时在这两个平台运营，不仅希望涨粉，还希望通过播放量获得收益，答案是肯定的。

所有这类视频可以发布的平台包括：抖音、快手、小红书以及微信视频号。这四个平台大家应该都很熟悉，是日常生活中会经常使用的。小红书在年轻用户，特别是女性用户中较为流行。微信视频号则内置于微信应用中，每个人都可以发布内容。

在这几个平台发布视频后，我们可以通过“影视推广”平台来接取任务，并通过完成任务来结算奖金。平台提供的素材非常丰富。

以下是影视推广的核心概念与操作流程：

**影视推广**是指为电视剧、电影或综艺节目进行线上宣传。许多制片方或广告商会通过此类平台发布任务，创作者制作并发布推广视频，即可根据视频数据获得奖金。

**操作流程**通常遵循以下步骤：
1.  **接取任务**：在影视推广平台上选择想要推广的剧集或短剧任务。
2.  **制作视频**：使用平台提供的素材或自行剪辑创作推广视频。
3.  **发布视频**：将成品视频发布到指定的平台，如抖音、快手、小红书或视频号。
4.  **提交回填链接**：在平台任务页面提交信息，通常包括：
    *   微信号或账号昵称
    *   发布平台的ID（如抖音号）
    *   所推广剧集名称
    *   已发布视频的链接
    *   视频发布时间
    *   视频播放量/点赞量的截图
5.  **结算收益**：平台审核提交的信息后，会根据既定规则结算奖金。

影视推广的收益通常比单纯依靠平台播放量分成的收益更高。平台上有大量短剧推广任务，这些短剧多由传媒公司或网红自制，他们需要创作者帮助宣传。

这类任务对账号粉丝数没有要求，收益结算主要看视频数据。通常，一部剧的收益会取你发布的、播放量最高的三个作品的数据总和进行计算。

不同平台的收益计算规则有所不同：

*   **抖音**：每1万播放量奖金约为50元。
*   **微信视频号**：每1万播放量奖金约为10元。单价较低是因为用户基数大，触达容易。
*   **小红书**：每1000播放量奖金约为30元，单价相对较高。
*   **抖音+快手（按点赞结算）**：部分任务按点赞数结算，无粉丝要求。例如，单条作品获5000赞奖金约100元，1万赞奖金约200元（快手1万赞约180元）。

此外，像搜狐视频等平台也有短剧推广任务。虽然其平台热度可能不如前几位，但提供的任务单价有时颇具竞争力。

![](img/6e38e67d6b18a63c99cdb7ca32a36a76_9.png)

本节课中，我们一起学习了影视推广的盈利模式。关键点在于：通过接取特定推广任务，在抖音、快手、小红书、视频号等平台发布视频，并根据播放量或点赞量来结算奖金。这种方式对账号粉丝数没有门槛要求，是新手起步并获得收益的有效途径之一。记住核心操作流程：接任务、做视频、发平台、填链接、等收益。
