# 小红书运营教程：P32：30.关键帧 🎬

![](img/3d60dceec7422c12756cc314a4913f60_1.png)

在本节课中，我们将要学习视频剪辑中一个非常核心的功能——关键帧。通过关键帧，我们可以为文字、图片、特效等元素创建平滑的动画效果，让内容更具动感和吸引力。

---

![](img/3d60dceec7422c12756cc314a4913f60_3.png)

上一节我们介绍了基础剪辑操作，本节中我们来看看如何为元素添加动画。

关键帧的原理与Flash动画中的关键帧相似。它的作用是记录元素在某个时间点的属性状态（如位置、大小、颜色），并在不同关键帧之间自动生成平滑的过渡动画，从而实现动态效果。

以下是添加关键帧的基本步骤，我们以调整一个星星图形的缩放动画为例进行说明：

1.  **选择起始帧并设置初始状态**：在时间轴上，将播放头移动到动画的起点（例如0秒处）。在右侧的属性面板中找到“缩放”选项，将其数值修改为初始大小，例如`50%`。
2.  **添加第一个关键帧**：点击“缩放”属性旁边的菱形图标或“添加关键帧”按钮。此时，时间轴上会生成一个菱形标记，表示已记录下当前时刻缩放为`50%`的状态。
3.  **设置中间状态并添加关键帧**：将播放头移动到下一个时间点（例如第2秒）。然后，修改“缩放”的数值，例如调整为`80%`。系统通常会自动在此刻添加一个新的关键帧。
4.  **设置结束状态并添加关键帧**：再次移动播放头（例如到第4秒），将“缩放”数值改回`50%`，系统会再次记录一个关键帧。
5.  **预览动画效果**：点击播放，你会看到星星图形会从50%大小逐渐平滑放大到80%，再逐渐缩小回50%。关键帧的作用就是自动计算并生成这些帧与帧之间的过渡动画。

---

除了缩放，关键帧还可以应用于多种属性以创造丰富效果。

其操作逻辑与缩放动画完全一致。以下是其他几种常用的关键帧类型：

*   **位置关键帧**：可以让元素在画面中沿设定路径移动。代码逻辑可理解为随时间变化其坐标值，例如 `position: (x1, y1) -> (x2, y2)`。
*   **旋转关键帧**：可以让元素进行旋转。例如，在0秒时旋转角度为`0deg`，在2秒时变为`360deg`，元素就会做一周旋转。
*   **不透明度/颜色关键帧**：可以实现元素的淡入淡出或颜色渐变效果。

---

本节课中我们一起学习了关键帧的核心概念与基础操作方法。关键帧通过记录不同时间点的属性状态，并自动生成平滑过渡，是制作视频动画效果的基石。掌握位置、缩放、旋转等关键帧的应用，能显著提升你内容的视觉表现力。
