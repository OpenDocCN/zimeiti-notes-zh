# 跨境电商全球开店：P37：第18讲.品牌旗舰店制作 🏬

在本节课中，我们将学习如何制作一个亚马逊品牌旗舰店。品牌旗舰店是展示品牌形象、集中引流和提升转化的重要工具。我们将从基础设置开始，逐步讲解页面布局、模块选择以及设计策略，帮助你打造一个专业且有效的品牌店面。

## 品牌旗舰店入口与价值

![](img/fbbb85f5acb414fa89b063870f03aace_1.png)

上一节我们介绍了品牌建设的基础，本节中我们来看看品牌旗舰店的具体制作方法。

首先，打开亚马逊网站并搜索一个关键词。在搜索结果页的顶部，广告位的上方，你会看到一个特殊的广告位置。这个位置就是品牌广告位。品牌广告可以展示品牌Logo、品牌标语以及产品信息。但请注意，只有成功创建了品牌旗舰店后，才能投放此类广告。

![](img/fbbb85f5acb414fa89b063870f03aace_3.png)

此外，你可以通过该广告位上的按钮，直接进入自己或竞争对手的品牌旗舰店。

![](img/fbbb85f5acb414fa89b063870f03aace_1.png)

上图展示了一个竞争对手的品牌旗舰店示例。接下来，我们将进入编辑模式，学习如何制作自己的品牌旗舰店。

## 店面基础设置

了解了品牌旗舰店的价值后，我们开始进行基础设置。点击“编辑品牌旗舰店”后，首先需要配置店面设置。

以下是店面设置的核心步骤：
*   **品牌信息**：填写店铺的品牌名称和上传品牌Logo。这个Logo将显示在店铺的显著位置。
*   **商品网格布局**：选择商品显示的网格样式，可选“正方形”或“立方体”。请根据你店铺页面的整体设计风格来决定。

![](img/fbbb85f5acb414fa89b063870f03aace_3.png)

![](img/fbbb85f5acb414fa89b063870f03aace_5.png)

## 创建页面与结构规划

![](img/fbbb85f5acb414fa89b063870f03aace_7.png)

完成基础设置后，我们需要为店铺创建内容页面。合理的页面结构能帮助顾客更好地浏览和查找商品。

点击“添加页面”可以创建一个全新的页面。例如，如果你销售睫毛产品，可以创建一个名为“睫毛”的页面。在页面描述中，可以简短介绍该页面的产品，并适当埋入关键词。

创建页面时，有四种内容模块类型可供选择：商品网格、标注、选取框等。创建完成后，你可以通过拖拽的方式，将新页面设置为另一个页面的子页面，从而构建出一级栏目和二级栏目的树状结构。例如，主页面是“键盘”，其子页面可以是“机械键盘”、“键盘垫”等。

## 首页内容模块布局

规划好页面结构后，我们来重点设计旗舰店的首页。首页是顾客的第一印象，布局至关重要。

建议先研究竞争对手的首页设计。通常，一个优秀的首页会包含以下模块：

![](img/fbbb85f5acb414fa89b063870f03aace_5.png)

以下是构建首页时推荐使用的模块及其作用：
*   **视频模块**：在店铺开头放置产品视频，能动态展示产品特色，吸引顾客注意力。
*   **图片展板（轮播图）**：可以展示3到8张图片，建议像素至少为 **1500px * 700px**。每张图片都应链接到店铺内的具体产品，实现视觉引流。例如，同时销售键盘、鼠标和键盘垫，可以用三张轮播图分别展示并链接。
*   **可购物图片**：这是一种特殊的图片模块，顾客可以直接点击图片上的“立即购买”等按钮，跳转到产品链接页面，购物路径非常短。
*   **自由组合模块（拆分部分）**：如果你想在一个版块内同时展示视频、图片和商品，可以使用“拆分部分”模块。例如，代码示意如下，表示左右分栏布局，左侧放视频，右侧放商品列表：
    ```html
    <div class="split-section">
        <div class="left-column">[视频模块]</div>
        <div class="right-column">[商品网格模块]</div>
    </div>
    ```
*   **系统推荐模块**：包括“精选促销”、“畅销商品”和“推荐商品”。这些模块由亚马逊系统自动填充，无需手动链接商品。“畅销商品”展示店铺内最热卖的产品；“推荐商品”则会根据访客的进入路径和浏览行为，智能推荐相关产品。

![](img/fbbb85f5acb414fa89b063870f03aace_7.png)

## 设计策略与总结

掌握了各个模块的功能后，最后我们来探讨一下总体的设计策略。

在设计品牌旗舰店时，一个非常有效的方法是参考和学习本品类内领先品牌的店铺布局。分析他们如何排列视频、轮播图、商品矩阵和推荐模块，然后结合自身产品的特点进行模仿和创新。
