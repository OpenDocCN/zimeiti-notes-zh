# 抖音运营入门：P2：1、抖音短视频的前景 📈

![](img/50425ccefd0d4cc31020c31c5356670c_1.png)

在本节课中，我们将要学习抖音短视频平台的发展前景，了解其市场规模、用户构成以及为何选择抖音作为创作起点的核心原因。

![](img/50425ccefd0d4cc31020c31c5356670c_3.png)

了解抖音短视频的前景，需要通过一张图片来了解。

![](img/50425ccefd0d4cc31020c31c5356670c_1.png)

这张图片是抖音创作者大会的一份数据报告。报告显示，有超过2200万人在抖音的合计收入超过了417亿元。抖音希望将这个数字翻一番，让创作者的总收入达到800亿元。

![](img/50425ccefd0d4cc31020c31c5356670c_3.png)

这代表了巨大的流量扶持。同时，这里介绍第一个知识点：抖音上面有两类人。第一类人叫做“玩家”，他们单纯地玩抖音，没想过从抖音上赚钱。另一类人，可以称作“创作者”，因为你们想通过制作抖音内容来获取收益。这个概念稍后还会提到。

那么，为什么我们选择抖音短视频？在众多短视频平台中，为何选择抖音？可以类比开实体店铺。选择店面位置时，肯定会找一个流量好、适合开店的地方。做短视频也一样，一定要选一个流量大的平台。如果平台自身流量小，就无法分给创作者，创作者也就赚不到钱。

抖音每天的日活跃用户达到了8亿以上，月活跃用户更是达到了40亿人次以上。这是非常庞大的流量。

流量大，竞争压力会不会很大？答案是不会。因为创作者只有约5000万。创作者并非必须是网红达人。例如，邻居家的老太太用手机拍下自己做的饭传到抖音上，这位老太太也算创作者。所以，90%的竞争压力都来自于普通用户。

接着，做抖音短视频需不需要买电脑或摄像机？答案是不需要。只需要一部智能手机和一个手机号，就可以注册抖音账号。抖音倡导全民通过短视频的形式分享和记录生活，非常简单。

第三个问题，有什么样的风口值得我们去做？首先，抖音计划“抖音将帮助1000名优质的艺术创作者涨粉超过100万”。有的同学说不会创作，这不用担心。如果什么都会，就不需要来听课了。正是因为不会，才需要学习。

之前带过很多学员，包括年纪较大的。他们选择学习的原因是：虚心请教厉害的人不丢人。只要在不懂的方面对方有本事，而自己又想学，那么对方就是老师。虚心请教能让人少走很多弯路。不以年龄定高低，不以穷富断今后。

市场大、竞争相对温和、行业容易产出内容。现在是大数据时代，人人无隐私。你喜欢看什么，抖音就推荐什么。例如：
*   喜欢看帅哥，刷到的就是有腹肌的帅哥。
*   喜欢看美女，刷到的就是长腿美女。

这导致每个人看到的内容可能非常片面。对于不知道拍什么的小白，以下是可行的方向：
*   宝妈可以拍自己的宝宝或做的饭菜。
*   开实体店的可以拍自己的产品用于引流。
*   普通上班族可以分享自己的生活。

为什么可以？因为一个月有40亿的活跃用户。只要内容不触犯法律和道德红线，不是低俗内容，总有一部分人会喜欢。之所以只有几十个粉丝，是因为喜欢你的人还没有刷到过你的作品。

对于小白来说，大家都想涨粉，心情迫切。但不要把涨粉作为第一个目标。第一个目标应该是先把流量和播放量做起来。粉丝自然会随之而来。

本节课中我们一起学习了抖音平台的巨大潜力、用户分类、低门槛特性以及为初学者指明的创作方向。核心在于理解平台流量逻辑，并明确初期应将提升内容曝光度（播放量）作为首要目标，而非单纯追求粉丝数量。
