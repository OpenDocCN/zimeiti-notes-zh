# 抖音运营：P2：抄作业法 🚀

在本节课中，我们将学习一种通过“抄作业”方式参与中视频计划获取收益的实操方法。该方法核心在于借鉴爆款内容，并通过简单工具进行二次创作，适合新手操作。我们将分步拆解从选题、提取文案到最终发布的完整流程。

## 概述
本节教程将详细介绍“抄作业”法的具体步骤。该方法通过复制爆款视频的文案框架，结合自动化剪辑工具，快速生成符合中视频计划要求的视频内容，从而实现收益。

## 第一步：寻找爆款选题 🔍
上一节我们介绍了中视频计划的潜力，本节中我们来看看如何找到可靠的创作起点。寻找爆款选题是成功的第一步。

操作步骤如下：
1.  打开抖音，在搜索框内输入“温情故事”。
2.  点击顶部的“视频”板块。
3.  将筛选条件设置为“最多点赞”。

执行以上操作后，你将看到大量高点赞量的爆款温情故事视频。这类内容容易引发观众情感共鸣，爆款概率较高。

![](img/ec705ffb6be6cf8711885a6c6d1af226_1.png)

---

## 第二步：提取视频文案 📝
找到目标视频后，下一步是提取其核心文案。以下是提取文案的具体流程。

![](img/ec705ffb6be6cf8711885a6c6d1af226_3.png)

操作步骤如下：
1.  进入选定的爆款视频播放页。
2.  点击右下角“分享”按钮。
3.  在弹出的菜单中找到并点击“复制链接”。
4.  将复制的链接粘贴到文案提取工具中（如“轻抖”等工具）。
5.  使用工具的“链接转文案”功能，获取视频的完整文案。

**核心工具公式**：`目标视频链接` + `文案提取工具` = `原始文案`

![](img/ec705ffb6be6cf8711885a6c6d1af226_3.png)

---

## 第三步：文案处理与视频生成 🎬
获取原始文案后，不能直接使用，必须进行二次处理以避免违规。处理完成后，即可利用工具快速生成视频。

以下是关键操作点：
*   **文案处理**：通读提取的文案，理解其故事脉络与情感核心。然后，完全用自己的语言和表达方式，将故事重新讲述一遍。这是避免抄袭和违规的关键。
*   **视频生成**：将处理后的文案粘贴到“剪映”APP的创作页面。使用其“一键成片”功能，软件会自动为文案匹配画面素材并生成配音。

**核心操作代码**（概念描述）：
```plaintext
输入：二次创作后的文案
工具：剪映APP - “一键成片”功能
输出：一个配有画面和声音的完整视频
```

---

## 第四步：发布与获取收益 💰
视频制作完成后，需要通过正确的入口发布，才能计入中视频计划并获得收益。

发布流程如下：
1.  在抖音顶部搜索“中视频计划”。
2.  点击进入“中视频伙伴计划”官方入口。
3.  通过该入口发布制作好的视频。
4.  发布时，务必勾选“同步至西瓜视频和今日头条”。

完成以上步骤后，你的视频在抖音、西瓜视频、今日头条三个平台产生的播放量，都将为你带来收益。

## 总结
本节课我们一起学习了“抄作业”法的完整操作流程：
1.  **寻找爆款**：在抖音搜索“温情故事”并按点赞排序筛选。
