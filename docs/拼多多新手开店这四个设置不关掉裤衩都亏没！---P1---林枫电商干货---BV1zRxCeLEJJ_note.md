# 拼多多开店避坑指南：01：必须关闭的四个扣费设置 💸

在本节课中，我们将学习拼多多新店运营初期必须关闭的四个平台设置。这些设置如果保持开启，可能导致店铺被平台自动扣费或产生不必要的损失。掌握关闭方法，是保护店铺利润的第一步。

许多新手商家在运营拼多多店铺时，常会遇到被平台莫名扣费的情况。这通常是因为平台内几个默认开启的扣费或营销功能未被关闭。

上一节我们介绍了课程目标，本节中我们来看看具体需要关闭哪四个设置。

以下是四个必须检查并关闭的关键设置：
1.  **消费者体验计划**
2.  **智能营销**
3.  **自动跟价**
4.  **仅退款（策略优化）**

---

## 1. 关闭消费者体验计划 🛡️

消费者体验计划是平台的一项服务，前期可能只扣取小额费用，但后期每笔订单都可能扣除数元，长期累积是一笔不小的开支。

**关闭路径公式**为：
`商家后台` -> `售后管理` -> `消费者体验` -> `更多` -> `停用`

在商家后台找到“售后管理”模块，进入后即可看到“消费者体验”选项。打开该页面，点击右上角的“更多”按钮，选择“停用”即可关闭此计划。

---

## 2. 关闭智能营销工具 🤖

![](img/91ba9655a3ba177578f4ab0e4bc656f1_1.png)

智能营销工具可能会自动创建一些付费推广活动，在不经意间消耗您的推广预算。

![](img/91ba9655a3ba177578f4ab0e4bc656f1_2.png)

**关闭操作流程**如下：
进入“营销工具”下的“全部营销工具”，找到“设置记录”。点击对应活动链接后的“详情”，选择“暂停”并“结束”该活动。

![](img/91ba9655a3ba177578f4ab0e4bc656f1_3.png)

---

## 3. 关闭自动跟价功能 ⚖️

![](img/91ba9655a3ba177578f4ab0e4bc656f1_4.png)

自动跟价功能开启后，系统会根据竞争对手的价格自动调整您的商品售价，可能导致利润大幅下降甚至亏损。

![](img/91ba9655a3ba177578f4ab0e4bc656f1_5.png)

**关闭方法**：
在“营销活动”相关页面，找到带有“问号”图标的“自动跟价说明”。点击进入说明页面，您会看到一个“关闭”按钮，直接点击即可禁用此功能。

![](img/91ba9655a3ba177578f4ab0e4bc656f1_6.png)

## 4. 优化仅退款策略 📉

![](img/91ba9655a3ba177578f4ab0e4bc656f1_8.png)

平台“仅退款”功能无法直接关闭，但通过配置策略，可以大幅减少因“仅退款”造成的损失。

![](img/91ba9655a3ba177578f4ab0e4bc656f1_9.png)

**核心策略**是创建以下三个自动化规则：
*   **ERP取消发货自动退**：当订单在发货前被取消时，系统自动处理退款。
*   **发货仅退款，退货退款自动转拒收策略**：针对已发货的仅退款申请，自动转为要求退货退款的流程。
*   **小金额自动退**：为小额订单设置自动退款阈值，避免纠纷消耗精力。

![](img/91ba9655a3ba177578f4ab0e4bc656f1_10.png)

创建并启用这些策略，可以有效拦截和减少不合理的“仅退款”申请。

![](img/91ba9655a3ba177578f4ab0e4bc656f1_11.png)

本节课中我们一起学习了拼多多新店必须关注的四个扣费设置：消费者体验计划、智能营销、自动跟价以及仅退款策略的优化方法。及时关闭或优化这些设置，是保障店铺基础利润、避免不必要损失的关键步骤。请根据上述路径，尽快检查您的店铺后台并进行调整。
