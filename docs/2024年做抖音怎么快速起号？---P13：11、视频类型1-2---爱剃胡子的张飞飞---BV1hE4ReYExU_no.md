# 抖音运营实战：P13：11、视频类型1-2

在本节课中，我们将要学习抖音账号运营中一个至关重要的环节：**人设的建立**。我们将深入探讨强人设与弱人设的区别，并详细介绍一种核心的视频制作形式——口播，帮助你明确内容方向，为账号的快速成长打下坚实基础。

上一节我们解决了文案问题，本节中我们来看看内容创作的下一个关键环节。许多课程会直接教你涨粉或变现，但从吸引粉丝到实现变现并非一条直线，中间还有一个核心环节：**建立人设**。

建立人设的目的并非为了炫耀，而是为了在众多同类创作者中脱颖而出。例如，有十个人都做装修内容，如果其中一位创作者（比如小王同学）树立了一个幽默搞笑的人设，观众就更容易记住他，而忽略其他九位。当观众因为喜欢你这个人而关注你时，他们就成为了你的粉丝。这就是人设的价值。

接下来，我们来学习这个核心知识点。人设主要分为两种类型：**强人设**与**弱人设**。理解并区分这两种人设后，你就能快速判断任何视频的人设类型。

**强人设**是指内容以创作者**个人**为切入点，与创作者自身密切相关。其核心公式可以概括为：

**强人设 = 以“我”（个人/经历/爱好）为中心的内容**

![](img/ca315832dd479f9691b22ad189392faa_1.png)

具体可以分为以下两类：

1.  **以个人职业或经验为切入点**：例如分享装修、办公、房产、汽车、餐饮、婚介、法律咨询、电商等领域的行业经验。
2.  **以个人兴趣爱好为切入点**：例如制作美食、旅行、手工、宠物、星座占卜、美妆护肤、服装穿搭、游戏、摄影等内容。

强人设内容通常需要分享创作者的生活、工作或爱好，并且**大部分需要真人出镜**，因为内容与“我”紧密相关。

**弱人设**则与创作者本人关系不大，核心目的是为了运营好账号本身。例如：

*   纯文案类账号：发布情感语录配音乐，无真人出镜或配音。
*   影视解说类账号：虽然有人声解说，但焦点是电影内容，而非解说者本人。

弱人设的公式可以理解为：

**弱人设 = 以“内容/主题”为中心，弱化创作者个人**

区分清楚人设类型后，我们才能更好地规划内容。只要内容与你个人相关，都可以作为视频素材。下面，我们来详细介绍第一种强人设的常见内容形式：**讲故事**。

讲故事是口播视频的一种重要形式。你可以分享客户故事、从业经历或人生感悟。故事本身不要求完全真实，可以从文案平台寻找有趣的故事进行分享。

**口播**，即用嘴巴讲述内容，是强人设账号最常用、也最容易上手的视频形式之一。口播主要分为两大类：

1.  **故事型口播**：如情感口播、养生故事等，目的是让观众觉得有趣、有共鸣。
2.  **知识型口播**：如育儿、金融、装修、房产、汽车等领域的知识分享，目的是让观众在几十秒内获得“收获感”，觉得学到了东西。

以下是适合做口播的领域示例，几乎涵盖所有垂直行业：
*   法律知识
*   保险金融
*   装修知识
*   汽车知识
*   房产知识
*   ……

口播的优势在于**拍摄简单**。你只需要准备好文案，用手机支架固定好手机即可录制。这里有两个常见问题：

*   **口才不好能做口播吗？**  
    **可以**。口播是提前准备好稿子再录制，并非临场发挥。大部分口播者的“好口才”都源于充分的准备。
*   **普通话不好能做口播吗？**  
    **可以**。如果你的目标客户是同城或有地域属性，使用方言可能效果更佳，听起来更亲切、有趣。观众如果听不懂，可以通过字幕来理解内容。

本节课中我们一起学习了抖音账号运营的核心概念**人设**，明确了**强人设**与**弱人设**的区别，并深入了解了**口播**这种基础且高效的内容形式。掌握这些，你就能更清晰地规划自己的内容方向，为制作吸引人的视频迈出关键一步。
