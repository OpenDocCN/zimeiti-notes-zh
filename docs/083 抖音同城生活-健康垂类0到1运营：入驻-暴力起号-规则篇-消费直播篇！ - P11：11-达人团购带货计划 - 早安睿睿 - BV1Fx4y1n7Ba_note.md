## 什么是通用计划？🤔

通用计划是商家与达人合作的一种计划类型，它是原“通用佣金”功能的升级版。

商家可以为自己的团购商品创建通用计划。创建后，所有符合条件的生活服务达人都能以**视频**或**直播**的形式，推广该商家的团购商品。当用户购买并到店核销后，平台会将核销金额按商家设定的比例分成给推广的达人。

![](img/81aac55994de1d44b506f1841f98b9b8_10.png)

其核心流程可以概括为以下公式：
**商家创建通用计划 -> 达人自由推广 -> 用户核销 -> 按比例分佣给达人**

## 如何发布通用计划？📱

![](img/81aac55994de1d44b506f1841f98b9b8_16.png)

上一节我们介绍了通用计划的概念，本节中我们来看看如何在抖音来客App中具体发布一个通用计划。

以下是发布通用计划的具体步骤：

![](img/81aac55994de1d44b506f1841f98b9b8_18.png)

1.  **打开抖音来客App**，进入首页。
2.  选择 **“达人带货”** 模块。
3.  点击 **“发计划”** 按钮。
4.  在计划类型中选择 **“通用计划”**，进入发布页面。
5.  **选择内容形式**：你可以勾选“短视频”和/或“直播”，决定达人通过何种方式带货。
6.  **计费方式**：通用计划仅支持 **“按核销量结算”**。
7.  **添加商品**：点击“下一步”，选择你想要设置通用计划的团购商品。
    *   **注意**：已经发布过通用计划的商品无法再次发布。
8.  **设置佣金比例**：选好商品后，需要为该商品设置佣金比例，可设置范围为 **`1%` 至 `80%`**。
9.  **开启通知**：建议打开 **“达人推广时通知我”** 的开关。这样当有达人开始推广你的商品时，你会收到消息提醒。

![](img/81aac55994de1d44b506f1841f98b9b8_20.png)

![](img/81aac55994de1d44b506f1841f98b9b8_22.png)

完成以上步骤并提交，通用计划即发布成功。之后，达人带货售出并核销后，即可按你设置的比例获得佣金。

![](img/81aac55994de1d44b506f1841f98b9b8_24.png)

## 什么是招募计划？👥

通用计划面向所有达人，而招募计划则允许商家进行更精准的筛选。招募计划是商家针对特定等级或粉丝量级的达人，主动发布合作邀请的一种计划类型。

![](img/81aac55994de1d44b506f1841f98b9b8_26.png)

商家发布招募计划后，达人在平台的“合作广场”中可以查看并报名。商家审核确认合作后，达人再开始进行内容创作和推广。目前招募计划支持 **“按核销量结算”** 的计费方式。

![](img/81aac55994de1d44b506f1841f98b9b8_28.png)

## 如何发布招募计划？🎯

![](img/81aac55994de1d44b506f1841f98b9b8_32.png)

了解了招募计划的定向性后，我们进入实操环节，学习发布招募计划的完整流程。

![](img/81aac55994de1d44b506f1841f98b9b8_34.png)

以下是发布招募计划的具体步骤，主要分为三步：

![](img/81aac55994de1d44b506f1841f98b9b8_36.png)

**第一步：填写计划基本信息**
*   **计划名称**：自由填写，建议便于商家和达人查找。
*   **招募周期**：可选择“长期招募”或“短期招募”。若选短期，需设置具体的截止日期。
*   **期望探店时间**：填写你希望达人前来探店的具体日期或每周时间段，建议选择店内不忙时。
*   **希望视频发布时间**：设置达人发布推广视频的时间范围。**这个时间段建议设置得长一些**，以给达人留出制作时间。超期发布将无法获得佣金。
*   **可接待门店与对接人电话**：务必确认所选门店和对接人知晓此事，并保持电话畅通。

![](img/81aac55994de1d44b506f1841f98b9b8_38.png)

**第二步：选择商品并设置佣金**
*   选择要推广的团购商品，并可为不同商品设置不同的佣金比例（`1%-80%`）。
*   设置**佣金有效期**，超出有效期后达人的推广将不再分佣。

![](img/81aac55994de1d44b506f1841f98b9b8_40.png)

**第三步：填写对达人的要求**
*   **招募达人数**：最多可招募 `200` 人。
*   **达人要求**：可设置达人等级要求和/或粉丝量要求。
*   **达人报名是否需要确认**：
    *   选择“需要”，则达人报名后需经你同意才能合作。
    *   选择“不需要”，则达人报名后可直接开始合作。
*   **额外支持**：设置是否提供接待餐食、是否为达人视频投放Dou+（加热）。提供的支持越多，通常越能吸引达人报名。

![](img/81aac55994de1d44b506f1841f98b9b8_42.png)

全部填写完成后，点击提交即可发布计划。你可以在“管理计划”页面查看招募进度、确认报名达人以及查看相关推广数据。

## 什么是一口价定向计划？💰

![](img/81aac55994de1d44b506f1841f98b9b8_44.png)

定向计划适用于商家已有明确合作意向的达人。其中，**一口价定向计划**是指：商家针对指定的商品，向指定的达人支付一笔固定的佣金（一口价），邀请该达人在短视频中进行推广。

![](img/81aac55994de1d44b506f1841f98b9b8_46.png)

![](img/81aac55994de1d44b506f1841f98b9b8_48.png)

该计划需要商家提前在平台充值，资金由平台担保，合作完成后结算给达人。其合作模式可以简化为：
**商家充值 -> 创建计划并指派达人 -> 达人接受任务并发布视频 -> 平台支付固定佣金**

## 如何发布一口价定向计划？🔧

![](img/81aac55994de1d44b506f1841f98b9b8_52.png)

在发布一口价计划前，需确保已开通并充值佣金钱包。

![](img/81aac55994de1d44b506f1841f98b9b8_54.png)

**1. 开通充值账户：**
打开抖音来客App，点击“营销推广”右侧的“更多”，进入“佣金钱包”，按页面提示完成开户和充值。

**2. 发布一口价定向计划：**
*   在“达人带货”页面点击“发计划”，选择“定向计划”。
*   目前一口价计划**仅支持短视频带货**。在计费方式中选择“一口价”。
*   **填写计划要求**：
    *   **营销目标**：选择“团购推广”或“门店曝光”。
    *   **指派达人**：输入目标达人的抖音号或用户名，可同时指派多名。
    *   **一口价佣金金额**：**务必提前与达人协商好金额**，再在线填写，避免达人因金额不符而拒绝合作。
*   确认所有信息无误后，点击“下一步”完成支付（若余额不足需先充值）。
*   计划发布后，可在“管理计划”页面查看达人是否接受任务及其履约进度。

## 如何发布按核销量结算的定向计划？🎬

![](img/81aac55994de1d44b506f1841f98b9b8_60.png)

![](img/81aac55994de1d44b506f1841f98b9b8_62.png)

除了固定佣金，定向计划也支持更常见的按核销量比例分佣模式。这种模式适合与多位达人进行定制化合作，例如提供专属套餐或佣金比例。

以下是发布步骤：

![](img/81aac55994de1d44b506f1841f98b9b8_64.png)

1.  在“发计划”中选择“定向计划”。
2.  **选择内容形式**：短视频或直播。
3.  **选择计费方式**：**“按核销量结算”**。
4.  **填写计划信息**：
    *   **计划名称**：便于识别的名称。
    *   **添加团购商品并设置佣金**：可批量选择商品并设置佣金比例（`0%-80%`）。注意：审核未通过或“仅直播间可见”的商品有特殊限制。
    *   **佣金有效期**：设置视频发布后多少天内产生的订单可计佣（直播计划无需设置）。
    *   **期待视频发布时间**：**务必与达人约定准确**，超期发布将无法获得佣金。
    *   **指定带货达人**：通过搜索昵称或从“达人广场”添加，最多可指定 `200` 个达人。
    *   **商家联系电话**：填写有效的联系方式。
5.  **直播计划简化版**：若创建直播计划，只需填写名称、团购商品、佣金比例、指定达人和对接人电话即可发布。

## 如何管理已发布的定向计划？📊

![](img/81aac55994de1d44b506f1841f98b9b8_68.png)

计划发布后，需要进行有效的管理和维护。

*   **查看计划**：在“达人带货”主页右上角点击“定向计划”进行查看。
*   **查看详情与数据**：点击计划卡片可进入详情页，若达人已发布内容，点击其头像可查看详细数据。
*   **修改计划**：在计划详情页，可以修改联系方式、增加团购商品或达人、延长视频发布时间。
*   **停止合作**：
    *   **停止整个计划**：点击详情页右上角“停止合作”。
    *   **停止单个达人**：点击该达人的卡片，选择“取消合作”。
*   **重要注意事项**：
    *   **直播计划**：可随时停止。停止后，未开播的达人合作取消；已开播的达人正常结算。
    *   **短视频计划**：**仅在达人未发布视频（未履约）时支持取消合作**。

## 总结 📝

![](img/81aac55994de1d44b506f1841f98b9b8_74.png)

![](img/81aac55994de1d44b506f1841f98b9b8_76.png)

本节课中我们一起学习了抖音来客平台上三种主要的达人带货计划：
1.  **通用计划**：面向全体达人，设置佣金比例后即可等待达人自由推广，操作简单。
2.  **招募计划**：可设定达人门槛（如等级、粉丝量），主动招募并筛选合作达人，适合精准拓客。
3.  **定向计划**：包括“一口价”和“按核销量结算”两种模式，用于与指定达人进行定制化合作，灵活度高。

![](img/81aac55994de1d44b506f1841f98b9b8_78.png)

掌握这些计划的特性和发布流程，能帮助你更有效地利用达人资源，提升店铺的线上曝光与销量。更多相关的达人带货教学课程将在之后陆续发布。
