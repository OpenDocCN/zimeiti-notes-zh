# 新媒体运营：P32：14：用户标签与创作者标签的完整诠释 📊

在本节课中，我们将要学习新媒体平台（如抖音、小红书）中“垂直定位”与“账号打标签”的核心概念。我们将通过对比用户标签和创作者标签，帮助你理解平台如何识别内容与用户，以及为何保持内容垂直对账号成功至关重要。

## 用户标签：平台如何“认识”你

![](img/8033027e2549e08afea21832c410722a_3.png)

上一节我们提到了垂直定位的重要性，本节中我们来看看平台是如何通过“标签”来理解用户的。用户标签是平台为每一位用户建立的兴趣画像，它决定了用户会看到什么样的内容。

平台通过分析用户的行为数据来生成标签，主要依据以下三点：

1.  **点击率**：用户点击进入视频的概率。
2.  **停留时长/完播率**：用户观看一个视频的时长或是否看完。
3.  **互动行为**：包括点赞、评论、收藏、转发等。

我们可以用一个简单的公式来模拟平台的计算逻辑：

```
用户对某领域兴趣值 = (点击该领域视频数 / 总推送视频数) * 权重1 + (该领域视频平均停留时长 / 总平均停留时长) * 权重2 + (该领域互动数 / 总互动数) * 权重3
```

假设平台给一位用户推送了15条视频，分布如下：
*   美食类视频5条，用户点击了5条。
*   美女类视频5条，用户点击了2条。
*   科技类视频5条，用户点击了1条。

此时，平台通过计算会发现该用户对“美食”领域的兴趣值最高。为了留住用户，平台后续会倾向于为他推送更多美食相关的内容。反之，如果推送了某个领域（如母婴）的视频但用户从不点击，平台则会减少此类内容的推送。

![](img/8033027e2549e08afea21832c410722a_5.png)

这就是所谓的“信息茧房”。例如，一对夫妻吵架后各自发视频，丈夫的视频评论区多是为丈夫说话的男性用户，妻子的视频评论区则多是支持妻子的女性用户。这正是平台根据用户标签进行内容分发的体现。

![](img/8033027e2549e08afea21832c410722a_7.png)

## 创作者标签：平台如何“定义”你

理解了平台如何为用户打标签后，我们来看看作为内容创作者，平台是如何定义我们的。创作者标签决定了你的内容会被推荐给谁。

很多人认为，只要多发某个领域的视频，平台就会自动认定你是该领域的博主。但实际上，平台判断账号属性的方式与判断用户类似，也是通过分析账号所发内容吸引来的用户行为数据。

![](img/8033027e2549e08afea21832c410722a_9.png)

假设有两个博主：
*   **博主A**：内容混杂，既发母婴，也发女装和日常生活打卡。
*   **博主B**：内容垂直，只发母婴相关。

![](img/8033027e2549e08afea21832c410722a_11.png)

当博主A发布一条女装视频时，平台会主要将这条视频推送给之前对他**母婴内容**感兴趣的用户（因为历史数据如此），同时也会尝试匹配部分对女装感兴趣的新用户。

这时就会出现问题：那些原本因为母婴内容关注他的用户，对这条女装视频不感兴趣，会直接划走，导致该视频的**完播率、互动数据**变差。平台会因此判定这条视频内容不佳，从而减少推荐。

长此以往，账号标签会变得混乱。平台无法确认这个账号的“专长”是什么，也就无法将其内容精准地推荐给可能感兴趣的人群，导致流量不稳定。

## 垂直定位的核心：成为“专家”

平台拥有海量用户，其核心竞争力就是通过**兴趣标签**进行精准的内容推送。因此，平台需要的是在某个领域持续产出优质内容的“专家”，而不是内容杂乱的“杂家”。

在用户互动行为中，“关注”是一个需要单独强调的特殊行为。用户关注一个账号，意味着他认为这个账号能提供**长期价值**。例如，一个母婴专家账号能持续提供育儿知识和好物推荐，用户因此愿意关注。

如果你的账号内容杂乱，用户无法预期你下次会更新什么，那么关注你的价值点就消失了。因此，保持内容垂直的核心目的有两个：
1.  **让平台给你打上清晰、单一的创作者标签**，从而将你的内容精准推荐给目标用户。
2.  **为用户提供明确的关注价值**，建立稳定的粉丝群体。

## 总结与下节预告

本节课中我们一起学习了：
1.  **用户标签**是平台根据用户的行为数据（点击、停留、互动）生成的兴趣画像，决定了用户看到的内容。
2.  **创作者标签**是平台根据你的内容所吸引的用户行为数据，为你的账号定义的属性，决定了你的内容被推送给谁。
3.  保持**内容垂直**的根本意义在于，让平台将你识别为某个领域的“专家”，从而获得更精准的流量推荐，并为用户提供明确的关注价值。
