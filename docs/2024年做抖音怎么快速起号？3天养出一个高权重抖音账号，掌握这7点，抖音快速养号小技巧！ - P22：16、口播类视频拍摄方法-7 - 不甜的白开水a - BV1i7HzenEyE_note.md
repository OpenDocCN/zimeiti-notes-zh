# 抖音运营教程：P22：16、口播类视频拍摄方法-7

## 概述
在本节课中，我们将学习口播类视频的具体拍摄方法。我们将通过一个成功案例，详细拆解拍摄时需要注意的构图、内容定位、设备使用等核心要点，帮助你掌握制作高质量口播视频的技巧。

---

上一节我们介绍了视频拍摄的通用注意事项，本节中我们来看看如何将这些要点应用到具体的口播视频拍摄中。

![](img/ca4822a61e859e7dceb505510468848c_1.png)

口播类视频内容多样，包括金融、装修、养生、情感等。虽然主题不同，但其拍摄方法是相通的。同样，记录日常生活、拍摄段子或美食视频，其拍摄方法也基本一致。因此，掌握口播和日常记录这两种类型的拍摄方法，几乎就能应对所有拍摄场景。

首先，我们来看第一个类型：口播视频。我将以一位学员“种小草”的成功案例进行讲解。

![](img/ca4822a61e859e7dceb505510468848c_1.png)

这位学员是一名40岁、带着三个孩子的宝妈，她创作的是情感口播内容。她的一条视频播放量超过8000万，并凭借该视频涨粉30万。为什么她的视频能如此成功？核心在于她精准吸引了目标群体。她的视频内容能让观众一眼就看出她想吸引的是谁。例如，如果我的目标群体是20多岁的学生，那么我拍摄的所有内容都围绕学生感兴趣的话题，这样涨粉就会既快又精准。

接下来，我们具体分析她的拍摄技巧。以下是拍摄口播视频时需要关注的几个核心要点：

**1. 构图与画幅比例**
抖音是九比十六的竖屏平台。在拍摄时，你需要决定视频主体是横屏还是竖屏。案例中的视频采用了四比三的横屏构图。关键点在于：风格要统一。如果你选择横屏，就应始终保持横屏拍摄；如果选择竖屏，则全部使用竖屏。保持一致性是重要的细节。

**2. 人物主体与画面布局**
人物应占据画面中央约70%的面积。避免将脸部怼得过近拍摄，这会让观众感到不适。确保画面的左、右、上方留有适当空白，这样会使画面看起来更清晰、舒适。

**3. 外貌与表现力**
如果你认为自己外貌不够出众，是否还能做口播？答案是肯定的。抖音的成功并非完全依赖颜值。口播的核心在于内容与知识的传达。只要让观众觉得你清晰、体面即可。可以利用美颜滤镜适当修饰，但注意不要过度夸张。

![](img/ca4822a61e859e7dceb505510468848c_3.png)

**4. 内容定位与受众共鸣**
视频内容必须精准指向特定群体，并引发他们的共鸣。现在，我们来看一下“种小草”那条涨粉30万的视频片段，请你思考：这个视频想要吸引哪个群体？

![](img/ca4822a61e859e7dceb505510468848c_3.png)

（视频台词示例：“跟婆婆在一起住是什么样的感受……我最大的感受就是这十多年以来，我娘家人很少在我家住上几天……”）

这个视频讨论的是婆媳关系。那么，哪个群体对此最有感触？通常是已婚女性群体。未婚或年轻的女性对此感受不深，而男性群体的共鸣感也相对较弱。因此，这个视频成功吸引了已婚女性这一精准受众。通过后台数据也可以验证这一点。

**5. 拍摄场景与背景**
有同学问，口播是否必须在户外拍摄？并非如此。你可以在任何地方拍摄，居家环境完全可以。重点是背景要整洁、美观，避免过于杂乱难看。

**6. 台词与提词技巧**
对于记忆力不好、担心背不下台词的同学，在剪辑环节有一个实用工具可以解决这个问题。你不需要背诵，可以边看边读进行录制，这会让拍摄变得更简单。

**7. 收音设备**
口播视频中，声音清晰至关重要。为了保证音质，建议你购买一个手机收声麦克风。这种设备价格便宜，只需十几元，在淘宝、京东等平台都能买到。

## 总结
本节课中，我们一起学习了口播类视频的拍摄方法。我们通过案例拆解，明确了**构图比例统一**、**人物布局合理**、**内容精准定位**以及**使用辅助工具**（如提词器和收声麦）的重要性。记住，成功的口播不在于主讲人多漂亮，而在于内容能否清晰传达并击中目标受众的内心。掌握这些要点，你就能迈出制作高质量口播视频的第一步。
