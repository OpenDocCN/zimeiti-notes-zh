# 新媒体运营：P15：01.10.03：🤖 AI：从分析爆款到生成爆款笔记

在本节课中，我们将学习如何利用AI工具（如ChatGPT）分析平台上的爆款内容，并基于分析结果，指导AI生成具备爆款潜质的新笔记。整个过程分为两个核心步骤：**喂食与分析**、**整合与生成**。

上一节我们介绍了通过“喂资料”的方式，让AI学习特定平台的文案风格和用语习惯。虽然上一节的旅游攻略案例效果不理想，但方法是正确的。本节中，我们来看看如何通过分析具体的爆款笔记，让AI掌握其成功要素，并最终产出新的爆款内容。

## 第一步：喂食与分析爆款笔记

![](img/19854b10b527279adf05b3db6db30af1_7.png)

![](img/19854b10b527279adf05b3db6db30af1_9.png)

要让AI学会创作爆款，首先需要让它理解什么是“爆款”。我们需要提供真实的爆款笔记文案给AI，并指令它进行结构化分析。分析应聚焦于以下几个核心维度：

![](img/19854b10b527279adf05b3db6db30af1_11.png)

![](img/19854b10b527279adf05b3db6db30af1_13.png)

![](img/19854b10b527279adf05b3db6db30af1_15.png)

以下是要求AI分析的具体指令点：

![](img/19854b10b527279adf05b3db6db30af1_17.png)

![](img/19854b10b527279adf05b3db6db30af1_19.png)

1.  **爆款因素**：分析文案中可能使其成为爆款的关键点。
2.  **整体结构**：
    *   **开篇**：用什么方式吸引用户继续阅读。
    *   **中间**：提供了哪些内容来戳中用户痛点，提升停留时长。
    *   **结尾**：用什么“钩子”促使用户互动（如私信、关注）。
3.  **关键词策略**：分析笔记是否结合了中文分词法植入特定关键词。如果有，需指出关键词、出现频率及铺设结构。

![](img/19854b10b527279adf05b3db6db30af1_21.png)

![](img/19854b10b527279adf05b3db6db30af1_23.png)

![](img/19854b10b527279adf05b3db6db30af1_25.png)

**操作示例**：
我们以“考研”领域的一篇爆款笔记为例。将笔记文案复制后，向AI（如ChatGPT 3.5）发出如下指令：
```
你是一名小红书平台的内容分析师。我将提供一篇考研领域的爆款笔记文案，请你分析：
1. 它的爆款因素有哪些？
2. 分析其整体结构：开篇如何吸引用户？中间如何戳中痛点？结尾如何引导互动？
3. 分析该笔记是否设计了关键词。如有，请指出关键词、出现频率及铺设方式。
```
AI分析后，可能给出如下反馈（示例）：
*   **爆款因素**：情感共鸣（开篇营造焦虑感）、逆袭故事、提供详细备考计划、使用表情符号增加亲和力、结尾正能量鼓励。
*   **结构分析**：开篇用“阅读一篇错三个”引发共鸣；中间分享个人备考经验和具体方法；结尾鼓励读者并引导提问。
*   **关键词分析**：本篇未发现明显基于中文分词法设计的关键词，词汇多为直接常用词。

![](img/19854b10b527279adf05b3db6db30af1_27.png)

重复此过程，为AI提供多篇（建议3-5篇）同领域的爆款笔记进行分析，让它积累足够的“爆款认知”。

## 第二步：整合要素与生成新笔记

在AI分析了多篇爆款笔记后，我们需要让它将分析出的成功要素进行整合、去重，形成一份该领域的“爆款公式”。

**操作示例**：
首先，指令AI总结之前所有分析过的爆款笔记的共同点：
```
请将你之前分析过的几篇考研爆款笔记的爆款因素总结一下，整合成一个完整的列表。
```
AI可能总结出如下列表：
1.  **情感共鸣**：开篇用焦虑、担忧等情绪切入。
2.  **逆袭故事**：分享个人从迷茫到成功的经历。
3.  **明确规划**：提供清晰的各科目备考时间线和计划。
4.  **实用建议**：给出具体的学习方法、资源推荐。
5.  **时间管理**：强调效率和时间节点。
6.  **正能量鼓励**：结尾给予读者信心和激励。
7.  **表情符号**：适量使用emoji增强亲和力。
8.  **互动引导**：结尾鼓励评论、私信。

接下来，基于这份“爆款公式”，指令AI生成新笔记的大纲和具体文案。

![](img/19854b10b527279adf05b3db6db30af1_29.png)

![](img/19854b10b527279adf05b3db6db30af1_31.png)

![](img/19854b10b527279adf05b3db6db30af1_32.png)

![](img/19854b10b527279adf05b3db6db30af1_33.png)

1.  **生成大纲**：
    ```
    请根据你总结的考研爆款笔记因素，为我规划一篇新的小红书笔记大纲。大纲需要细分到具体的小节点。
    ```
    AI生成的大纲可能包括：吸引人的标题、共鸣开篇、逆袭故事引入、分科目备考规划、时间管理技巧、心态调整方法、结尾互动引导。

2.  **生成完整文案**：
    ```
    请根据上述大纲，撰写完整的笔记文案。要求：
    - 严格遵循爆款笔记的结构和因素。
    - 包含一个简短的逆袭小故事来引入情绪。
    - 适量使用emoji表情。
    - 尝试在文案中融入“考研”、“备考规划”等关键词。
    ```
    随后，AI将生成一篇融合了多种爆款元素的完整笔记。你可以根据实际需求，对生成的内容进行微调和优化。

## 进阶技巧与注意事项

*   **指令的精确性**：AI对指令非常敏感。例如，“包含丰富的emoji”和“包含emoji”可能导致生成结果中表情符号的数量差异很大。需仔细斟酌用词。
*   **模型的局限性**：使用中可能会遇到AI遗忘前文指令的情况（在3.5版本中较常见）。此时需要重新发送或强调核心要求。
*   **利用高级功能（如GPTs）**：对于长期需求，可以创建自定义的GPTs。通过上传大量小红书风格范文、专用表情包列表等资料，打造一个更懂你领域和风格的专属AI助手，从而获得更稳定、优质的内容产出。
