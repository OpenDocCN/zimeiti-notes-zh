# TikTok+亚马逊跨境电商运营全套零基础视频教程：P6：4.准备工作-邮箱注册（电脑端） 📧

在本节课中，我们将学习如何注册一个海外邮箱，这是后续注册TikTok账号、苹果ID以及其他跨境电商相关服务的基础准备工作。

上一节我们介绍了准备工作的重要性，本节中我们来看看如何具体操作，完成Outlook邮箱的电脑端注册。

## 为什么选择Outlook邮箱？

注册邮箱是运营跨境电商账号的第一步。这个邮箱将用于注册TikTok账号、苹果ID以及其他必要的海外服务。

![](img/482dd8829511008b2cc057464b1edebb_1.png)

以下是推荐使用Outlook邮箱的几个原因：
*   **注册简便**：流程简单，操作方便。
*   **无需手机号**：注册时不需要绑定手机号码。
*   **注册数量多**：在同一个IP地址下，每天可以免费注册**3个**邮箱账号。
*   **可循环注册**：如果配合更换IP地址（科学上网），理论上可以无限循环注册，邮箱资源充足。

![](img/482dd8829511008b2cc057464b1edebb_2.png)

接下来，我将为大家详细演示电脑端的注册流程。考虑到部分学员可能没有电脑，后续也会有手机端的注册教程。

## 电脑端Outlook邮箱注册实操演示 🖥️

以下是电脑端注册Outlook邮箱的完整步骤。

1.  **打开浏览器并搜索**
    首先，在电脑的浏览器（如微软Edge、谷歌Chrome、360浏览器等）中搜索关键词 **“outlook”**。

    ![](img/482dd8829511008b2cc057464b1edebb_1.png)

2.  **进入官网并创建账户**
    在搜索结果中，点击进入Outlook的官方网站。在页面中找到并点击 **“创建免费账户”** 按钮。

    ![](img/482dd8829511008b2cc057464b1edebb_2.png)

3.  **设置邮箱用户名**
    页面加载后，进入创建账户界面。在“电子邮件”栏位输入你想要的邮箱前缀（例如：`ca22857857ttok`）。如果提示用户名已被占用，需要尝试其他组合，直到出现“下一步”按钮。

4.  **创建密码**
    用户名可用后，点击“下一步”。在新页面中，设置你的账户密码，并点击“下一步”。

5.  **填写姓名与出生日期**
    接下来填写姓名（使用英文或拼音即可，例如：`Ca Car`）。然后填写出生日期，**务必确保年龄大于18周岁**（例如：1996年）。地区选择“中国”即可，点击“下一步”。

6.  **完成人机验证**
    系统可能会要求进行人机验证，例如识别图片中的特定物体（如骆驼）。按照提示完成验证即可。

7.  **登录与确认**
    验证通过后，系统会询问是否保持登录状态，可以选择“否”。随后页面会跳转，进入新注册的Outlook邮箱界面，这表示注册成功。

    ![](img/482dd8829511008b2cc057464b1edebb_4.png)

8.  **查看邮箱地址**
    要查看完整的邮箱地址，可以点击页面右上角的账户头像或名称，在弹出的信息中即可看到你的完整Outlook邮箱地址（格式为：`用户名@outlook.com`）。

![](img/482dd8829511008b2cc057464b1edebb_4.png)
