# 亚马逊北美站风险提示：P1：W-8税务信息更新指南 🔍

在本节课中，我们将学习如何处理亚马逊北美站卖家收到的“更新美国纳税信息”风险提示。这不是店铺审核，而是常规的税务信息更新流程。我们将一步步了解其本质并完成操作。

---

## 风险提示的本质

近期，许多北美站卖家收到了亚马逊要求更新美国纳税信息的风险提示，截止日期通常为12月3日。无需担心，这并非店铺审核，仅代表您店铺的W-8税务登记已到期。

上一节我们明确了该通知的性质，本节中我们来看看具体的操作步骤。

## 信息更新操作流程

您只需根据最新的店铺注册信息重新登记即可。流程非常简单。

以下是具体的操作步骤列表：

1.  打开亚马逊卖家后台的该通知。
2.  根据店铺的实际信息，逐步选择并填写各项选项。
3.  输入您的纳税身份信息（如W-8BEN表格内容）等资料。
4.  检查无误后，直接提交。

提交完成后，系统会提示提交成功。此后三年内，您无需再处理此项事务。
