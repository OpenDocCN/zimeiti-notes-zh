# 小红书运营：P26：小红书暴力玩法揭秘！吸粉推广篇 🚀

![](img/99d527ee9cd18da796012e9bbfd35578_1.png)

在本节课中，我们将探讨小红书运营中的一个核心问题：是自建账号矩阵进行推广更好，还是直接与官方合作投放广告更有效。我们将详细解析这两种方式的区别、适用场景及具体操作步骤，帮助你理清思路，找到最适合自己的推广路径。

---

上一节我们介绍了小红书运营的基础概念，本节中我们来看看账号类型与官方广告这两种核心推广方式的关系。

首先，我们介绍小红书的账号类型。小红书账号主要可以分为以下三类：

**1. 企业号**
企业号的主要作用是树立官方品牌形象。它拥有官方认证标志，并享有特定特权，例如展示品牌名称、品牌地址等。这有助于更便捷地开展相关业务。

**2. 个人号**
目前大部分用户运营的是个人号。个人号的特点是灵活性高，运营者可以是品牌员工，也可以是品牌消费者。这使账号更具个人品牌特色，品牌相关人物或员工出镜也显得更自然。

**3. 引流号**
引流号在之前的课程中已提及，俗称“炮灰号”。这类账号可以批量创建，主要用于特定引流目的，用后即可舍弃，或在紧急情况下作为备用小号使用。引流号通常没有精细的个性规划。

通常，品牌推广会通过企业号、个人号或引流号自行发布笔记进行引流。接下来，我们提供一些关于打造企业号矩阵的建议。

以下是打造企业号矩阵的四点建议：

1.  **积极参与品牌活动**：品牌号发起的活动，如试用、团购等，应使用小号积极参与，增加活动热度。
2.  **个人号展示品牌属性**：在个人号的笔记内容中，应多次暗示品牌的定位，例如目标人群是城市年轻人还是职场新人。
3.  **控制产品发布频率**：个人号发布品牌产品内容的频率不宜过高。建议采用 **营销内容:种草/科普内容 = 3:7** 的比例。
4.  **利用引流号评论引流**：企业号如需直接引流，最好使用引流号在其笔记评论区进行，这样更为安全。

在了解了账号矩阵推广之后，我们来看看另一种重要的推广方式：小红书官方广告。

![](img/99d527ee9cd18da796012e9bbfd35578_7.png)

![](img/99d527ee9cd18da796012e9bbfd35578_9.png)

小红书官方广告主要指竞价广告。它主要分为两大类：

![](img/99d527ee9cd18da796012e9bbfd35578_11.png)

**1. 信息流广告**
当用户浏览小红书首页时，广告会不定期间隔出现在信息流中。

![](img/99d527ee9cd18da796012e9bbfd35578_13.png)

**2. 搜索广告**
当用户使用关键词进行搜索时，系统会根据算法展示相关的广告。例如，用户搜索“帽子”，可能看到与之搭配的“大衣”或“外套”的广告。

![](img/99d527ee9cd18da796012e9bbfd35578_15.png)

![](img/99d527ee9cd18da796012e9bbfd35578_17.png)

搜索竞价广告的核心原理是关键词竞价排名，这与百度搜索广告类似。广告主对特定关键词出价，出价不同，获得的广告位排名也不同。

![](img/99d527ee9cd18da796012e9bbfd35578_19.png)

目前，信息流广告位主要出现在以下位置：

![](img/99d527ee9cd18da796012e9bbfd35578_21.png)

*   第3位
*   第13位
*   第23位

关于广告投放的常见问题，可以在小红书官方网站的推广合作板块查看详细问答。

---

要投放官方广告，首先需要认证企业号。以下是企业号入驻的基本流程：

![](img/99d527ee9cd18da796012e9bbfd35578_23.png)

1.  访问小红书官方网站。
2.  找到“业务合作”板块，进入“推广合作”。
3.  按照官方指引，提交相关企业资质并支付审核认证费用。
4.  审核通过后，普通账号即升级为官方认证的企业号。

获得企业号资质后，即可在后台创建并投放信息流或搜索竞价广告。

![](img/99d527ee9cd18da796012e9bbfd35578_25.png)

![](img/99d527ee9cd18da796012e9bbfd35578_27.png)

官方广告投放费用较高。因此，在资金允许的情况下，建议考虑与官方合作投放竞价广告，因为它能直接获得平台推荐的精准流量。

![](img/99d527ee9cd18da796012e9bbfd35578_29.png)
