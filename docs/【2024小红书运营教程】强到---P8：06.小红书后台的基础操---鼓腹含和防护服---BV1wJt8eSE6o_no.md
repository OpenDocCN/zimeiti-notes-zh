# 小红书运营教程：P8：06.小红书后台的基础操作（电脑端） 💻

在本节课中，我们将学习如何通过电脑端操作小红书商家后台，重点掌握保证金、支付方式、运费宝和地址设置这四个核心功能的基础操作。

上一节我们介绍了手机端的相关操作，本节中我们来看看电脑端如何完成这些关键的后台设置。

![](img/f053165cce9ad646958455ad09d81048_1.png)

## 保证金开通与管理 💰

![](img/f053165cce9ad646958455ad09d81048_2.png)

保证金是开设店铺的基础。前期可以暂不缴纳，但若想长期经营或开通更多支付方式，则建议缴纳。

以下是保证金的几个要点：

*   **金额**：个人店保证金通常在 **1000元** 左右，具体金额因类目而异。企业店保证金在2000元至数万元不等。
*   **缴纳路径**：登录小红书商家后台，在首页找到 **`资金`** -> **`店铺保证金管理`**。
*   **可退还**：与主流电商平台规则一致，店铺关闭后，保证金在无违规扣款的情况下可以退还。
*   **扣款规则**：若发生损害消费者权益的行为（如未履行退款承诺），平台会从保证金中扣款。正常经营则不会扣款。

![](img/f053165cce9ad646958455ad09d81048_4.png)

![](img/f053165cce9ad646958455ad09d81048_5.png)

**核心概念**：保证金的缴纳是开通微信支付等功能的必要条件。公式表示为：
**开通微信支付 = 缴纳店铺保证金**

![](img/f053165cce9ad646958455ad09d81048_6.png)

一些特殊类目（如二手奢侈品、珠宝玉石）不支持个人店开通，需注册个体店或企业店。若要查询具体类目的保证金金额，在后台选择相应类目即可查看。

## 支付方式开通 🔗

![](img/f053165cce9ad646958455ad09d81048_8.png)

开通支付方式能让顾客使用微信或支付宝付款，对提升成交率很重要。

![](img/f053165cce9ad646958455ad09d81048_9.png)

以下是开通步骤：

1.  在后台首页，点击 **`资金`** 模块。
2.  找到 **`支付宝`** 与 **`微信`** 开通入口。
3.  **支付宝**：通常可当天完成注册绑定。
4.  **微信**：需先缴纳保证金，次日方可设置。提现金额将进入店铺注册法人的账户。

开通时，根据要求填写身份证信息。若开通的是个体店或企业店，则还需提供营业执照信息。

![](img/f053165cce9ad646958455ad09d81048_11.png)

![](img/f053165cce9ad646958455ad09d81048_12.png)

## 运费宝设置 🛡️

运费宝是一项推荐开通的服务，类似于其他平台的“运费险”，能为买卖双方的退换货运费提供保障。

以下是运费宝的作用与开通：

*   **作用**：买家发起“7天无理由退货”等售后请求时，系统会自动补贴运费。这能打消买家下单顾虑，**提升订单转化率**，并降低交易纠纷率。
*   **开通路径**：在后台的 **`售后`** 管理板块中，找到 **`运费宝`** 服务，点击开通即可。

![](img/f053165cce9ad646958455ad09d81048_14.png)

![](img/f053165cce9ad646958455ad09d81048_15.png)

开通运费宝，相当于为店铺增加了一层服务保障，对促进销售有积极作用。

![](img/f053165cce9ad646958455ad09d81048_16.png)

![](img/f053165cce9ad646958455ad09d81048_17.png)

## 地址设置 📍

正确设置退货地址对于处理售后订单至关重要，尤其对于一件代发的无货源模式。

以下是地址设置的关键点：

![](img/f053165cce9ad646958455ad09d81048_19.png)

*   **设置路径**：在后台进入 **`订单`** -> **`物流工具`**，点击 **`新建地址`**。
*   **填写策略**：
    *   **无货源商家**：建议将退货地址填写为偏远地区（如新疆），并在地址后附加“**联系客服**”字样。这样能引导买家在退货前先与您沟通，方便您同步信息给上游供应商。
    *   **有货源商家**：直接填写您的真实仓库或办公地址即可。
*   **关联设置**：此处的地址需与“7天无理由退货”等售后规则中的地址保持一致。

![](img/f053165cce9ad646958455ad09d81048_20.png)

![](img/f053165cce9ad646958455ad09d81048_21.png)

本节课中我们一起学习了小红书电脑端后台的四个基础操作：保证金的缴纳与管理、支付方式的开通、运费宝服务的设置以及退货地址的填写。掌握这些是店铺正常运营和提供基础服务保障的前提。请根据您的店铺类型和经营模式，在后台完成相应配置。
