# 视频号运营教程：P50：营业执照与视频号实名不一致时开通小店教程 📝

在本节课中，我们将学习当微信视频号的个人实名信息与营业执照上的法人信息不一致时，如何成功开通视频号小店。这是一个常见但关键的实操步骤。

## 概述
开通视频号小店是进行商品带货的前提。当运营者使用的微信号实名信息与准备入驻的营业执照法人信息不一致时，流程会略有不同。本节教程将一步步演示如何以“经办人”身份完成小店开通。

## 操作步骤详解

以下是具体的操作流程，请确保每一步都操作准确。

### 第一步：进入入驻页面
首先，打开微信，进入“视频号”页面。点击右上角的个人头像，进入“创作者中心”。在创作者中心内，找到并点击“带货中心”。系统将跳转至视频号带货官方页面，在此页面点击“入驻”按钮。

### 第二步：填写店铺主体信息
点击“开店”并同意相关协议后，进入信息填写页面。你需要在此上传营业执照照片。如果从相册选择提示文件过大，建议直接使用“拍照”功能现场拍摄营业执照并上传。在“主体类型”中，选择“个体工商户”。

**主体类型选择代码示例：**
```
主体类型：个体工商户
```

随后，填写店铺名称，其他如“行业”、“材料”等信息可暂时不填，点击“下一步”。

### 第三步：设置结算账户
进入“选择账户类型”页面。这里**必须选择“对私账户”**，而非对公账户。然后填写用于收款的银行卡号、开户行及开户行所在地。填写完毕后，点击“下一步”。

### 第四步：上传身份证信息
接下来需要上传身份证照片。点击“上传居民身份证”，通过拍照或相册上传身份证的正反面照片。上传成功后，点击“下一步”。

### 第五步：设置超级管理员
这是关键步骤。在“超级管理员身份”页面，你会看到“经办人”、“经营者”、“法人”等选项。

**核心逻辑公式：**
> **超级管理员选择规则** = **微信实名认证人**
> 当微信实名 ≠ 营业执照法人时，此处应选择 **“经办人”**。

因此，请选择“经办人”并点击确定。随后，系统会要求上传这位经办人（即当前微信的实名者）的身份证信息。按要求拍摄并上传身份证照片。

接着，填写身份证生效日期，并绑定一个手机号及邮箱。邮箱格式可参考 `任意数字@qq.com`。填写完毕后，点击“下一步”。

### 第六步：提交审核与关注公众号
根据提示，关注指定的公众号以完成授权。关注后返回原页面，点击“提交商户信息”。提交后，通常等待几分钟即可获得初步审核通过的结果。

### 第七步：完成账户验证
审核通过后，需要进行账户验证。系统会提供两种验证方式：
1.  **法人验证**：要求使用与法人信息一致的微信扫码验证。由于我们实名信息不一致，**无法使用此方法**。
2.  **汇款验证**：向填写的银行卡号汇入一笔指定金额（如0.09元）。

选择“汇款验证”。使用任意银行卡，向步骤三中填写的收款卡号转账0.09元。汇款成功后，返回验证页面点击“我已汇款”。系统会查询汇款结果，此过程可能需要等待数分钟。

### 第八步：签署协议并完成开店
汇款验证通过后，页面会提示进入“签署”环节。点击“签署”，获取手机验证码并确认。随后，在《开户意愿书》等文件上确认签署。

最后，在店铺类型中选择“个体店”，无需填写品牌等信息，直接点击“开店”。再次等待最终审核，通常15-30分钟内即可完成。

审核全部通过后，阅读并同意相关协议，你的视频号小店就正式开通成功了。

## 总结
本节课我们一起学习了在微信实名与营业执照法人信息不一致的情况下开通视频号小店的完整流程。核心在于理解并正确选择“经办人”作为超级管理员，并通过“汇款验证”方式完成最终账户认证。掌握这个方法，即使账号实名信息与营业执照不符，也能顺利开启你的带货之旅。
