## 人设定位的四大核心因素

![](img/3ed38238ecb07af639f72e626989d9e3_52.png)

![](img/3ed38238ecb07af639f72e626989d9e3_54.png)

我将人设定位的核心总结为四个关键点：**角色**、**内容**、**形式**和**记忆点**。只要在账号的整个运营生命周期中，持续地将这四个要素融入进去，你的人设定位基本上就不会出现大的问题。

![](img/3ed38238ecb07af639f72e626989d9e3_56.png)

![](img/3ed38238ecb07af639f72e626989d9e3_58.png)

![](img/3ed38238ecb07af639f72e626989d9e3_60.png)

### 1. 角色：明确你的专业身份

![](img/3ed38238ecb07af639f72e626989d9e3_62.png)

![](img/3ed38238ecb07af639f72e626989d9e3_64.png)

![](img/3ed38238ecb07af639f72e626989d9e3_66.png)

![](img/3ed38238ecb07af639f72e626989d9e3_68.png)

角色，简单来说，就是你在特定赛道中以什么身份提供内容。你需要让用户相信你提供的内容具有专业性。

![](img/3ed38238ecb07af639f72e626989d9e3_70.png)

![](img/3ed38238ecb07af639f72e626989d9e3_72.png)

![](img/3ed38238ecb07af639f72e626989d9e3_74.png)

以下是不同赛道的角色定位示例：

![](img/3ed38238ecb07af639f72e626989d9e3_76.png)

![](img/3ed38238ecb07af639f72e626989d9e3_78.png)

![](img/3ed38238ecb07af639f72e626989d9e3_80.png)

![](img/3ed38238ecb07af639f72e626989d9e3_82.png)

*   **好物分享/测评博主**：你需要展示你确实使用并评测过这些产品。
*   **装修博主**：你需要告诉用户你拥有正在装修或已装修好的房子，并记录整个过程，分享遇到的坑和最终成果，展示前后对比。
*   **母婴赛道**：
    *   **宝妈**：这是最常见的角色。
    *   **护理专家**：你可以是某机构或某专业领域的人员，以专家身份进行分享。
*   **情感赛道**：
    *   **心理学顾问/情感大师**：可以讲解情感、成长、励志等话题。
    *   **成长型人设**：分享个人经历，例如30岁女性如何应对家庭、创业、职场中的问题，如何从失败中走出。

![](img/3ed38238ecb07af639f72e626989d9e3_84.png)

![](img/3ed38238ecb07af639f72e626989d9e3_86.png)

![](img/3ed38238ecb07af639f72e626989d9e3_88.png)

**核心公式：角色 = 赛道方向 + 专家身份**

![](img/3ed38238ecb07af639f72e626989d9e3_90.png)

![](img/3ed38238ecb07af639f72e626989d9e3_92.png)

![](img/3ed38238ecb07af639f72e626989d9e3_94.png)

你只需要根据自己选择的赛道和内容方向，将自己塑造成该领域的“专家”身份即可。

![](img/3ed38238ecb07af639f72e626989d9e3_98.png)

![](img/3ed38238ecb07af639f72e626989d9e3_100.png)

### 2. 内容：保持垂直与价值

![](img/3ed38238ecb07af639f72e626989d9e3_104.png)

![](img/3ed38238ecb07af639f72e626989d9e3_106.png)

![](img/3ed38238ecb07af639f72e626989d9e3_108.png)

![](img/3ed38238ecb07af639f72e626989d9e3_110.png)

内容定位我们之前已经讨论过，关键在于内容是否吸引用户、是否有价值、是否有平台特色。在人设定位中，内容还需要特别注意一点：**垂直与一致性**。

![](img/3ed38238ecb07af639f72e626989d9e3_112.png)

![](img/3ed38238ecb07af639f72e626989d9e3_114.png)

你做什么赛道，就持续提供该赛道相关的内容。这关系到账号的垂直度和标签化，直接影响账号的成长和平台的流量分发。

![](img/3ed38238ecb07af639f72e626989d9e3_116.png)

![](img/3ed38238ecb07af639f72e626989d9e3_118.png)

![](img/3ed38238ecb07af639f72e626989d9e3_120.png)

![](img/3ed38238ecb07af639f72e626989d9e3_122.png)

**逻辑示例：**
假设你的第一篇内容是关于儿童教育的，获得了部分用户的点赞。当平台第二次将你的内容（比如宠物相关）推送给同一批用户时，他们可能不感兴趣并划走。这会向平台反馈“用户不喜欢此内容”，从而影响后续的推荐流量。

![](img/3ed38238ecb07af639f72e626989d9e3_124.png)

![](img/3ed38238ecb07af639f72e626989d9e3_126.png)

![](img/3ed38238ecb07af639f72e626989d9e3_128.png)

**核心要点：内容方向必须与赛道高度一致，持续输出，以强化账号标签。**

![](img/3ed38238ecb07af639f72e626989d9e3_130.png)

![](img/3ed38238ecb07af639f72e626989d9e3_132.png)

![](img/3ed38238ecb07af639f72e626989d9e3_134.png)

### 3. 形式：确定内容的呈现方式

![](img/3ed38238ecb07af639f72e626989d9e3_138.png)

![](img/3ed38238ecb07af639f72e626989d9e3_140.png)

![](img/3ed38238ecb07af639f72e626989d9e3_142.png)

形式指的是你通过何种具体方式将内容呈现给用户。主要包括以下几个方面：

![](img/3ed38238ecb07af639f72e626989d9e3_144.png)

![](img/3ed38238ecb07af639f72e626989d9e3_146.png)

![](img/3ed38238ecb07af639f72e626989d9e3_148.png)

![](img/3ed38238ecb07af639f72e626989d9e3_150.png)

以下是形式需要考虑的几个方面：

![](img/3ed38238ecb07af639f72e626989d9e3_152.png)

![](img/3ed38238ecb07af639f72e626989d9e3_154.png)

![](img/3ed38238ecb07af639f72e626989d9e3_156.png)

1.  **内容产出场景**：你的内容是在固定室内场景拍摄，还是在多变的室外场景？例如，固定在家中的书房，或是在上班路上、面试途中、车内等。场景的选择也可以参考对标账号。
2.  **声音呈现**：你是自己出镜讲解，还是不出镜仅使用配音？现在通过“剪映”等工具实现配音非常方便。你需要评估自己的表达能力和声音条件。
3.  **内容形式**：你的内容是以剧情、干货分享还是生活记录的形式呈现？
    *   **剧情型**：创作难度相对较高。
    *   **干货型**：需要精心准备文案和脚本。
    *   **生活记录型**：需要积累大量的拍摄素材。

![](img/3ed38238ecb07af639f72e626989d9e3_158.png)

![](img/3ed38238ecb07af639f72e626989d9e3_160.png)

![](img/3ed38238ecb07af639f72e626989d9e3_162.png)

![](img/3ed38238ecb07af639f72e626989d9e3_164.png)

你需要根据自身条件和赛道特点，选择最适合的呈现形式。

![](img/3ed38238ecb07af639f72e626989d9e3_166.png)

![](img/3ed38238ecb07af639f72e626989d9e3_168.png)

![](img/3ed38238ecb07af639f72e626989d9e3_170.png)

### 4. 记忆点：让用户牢牢记住你

![](img/3ed38238ecb07af639f72e626989d9e3_172.png)

![](img/3ed38238ecb07af639f72e626989d9e3_174.png)

![](img/3ed38238ecb07af639f72e626989d9e3_176.png)

记忆点是让人设深入人心、形成品牌识别的关键。你需要设计一些固定元素，反复刻入用户的脑海。

![](img/3ed38238ecb07af639f72e626989d9e3_178.png)

![](img/3ed38238ecb07af639f72e626989d9e3_180.png)

以下是打造记忆点的几种方法：

![](img/3ed38238ecb07af639f72e626989d9e3_182.png)

![](img/3ed38238ecb07af639f72e626989d9e3_184.png)

![](img/3ed38238ecb07af639f72e626989d9e3_186.png)

![](img/3ed38238ecb07af639f72e626989d9e3_188.png)

*   **视频前几秒的固定元素**：固定动作、特效、开场白。这些元素需要**前置**，在视频开头就呈现给用户。当用户刷到你多个视频时，开头的统一格式能强烈加深印象。
*   **统一的BGM（背景音乐）**：贯穿视频全程的标志性音乐。当音乐响起，用户即使还没看到画面，也能联想到是你。
*   **统一出现的物品**：在视频中始终出现的某个特定物品，如背景里的一个台灯、一个挂饰。
*   **统一的服装风格**：符合你人设的固定着装风格，例如农村题材的质朴服装、职场题材的职业装。

![](img/3ed38238ecb07af639f72e626989d9e3_190.png)

![](img/3ed38238ecb07af639f72e626989d9e3_192.png)

![](img/3ed38238ecb07af639f72e626989d9e3_194.png)

**记忆点设计逻辑：**
*   **前置强化**：固定动作、特效、开场白放在视频开头，冲击力最强。
*   **全局贯穿**：BGM、固定物品、服装风格贯穿整个视频或所有视频，形成持续的印象。

![](img/3ed38238ecb07af639f72e626989d9e3_196.png)

![](img/3ed38238ecb07af639f72e626989d9e3_198.png)

![](img/3ed38238ecb07af639f72e626989d9e3_199.png)

![](img/3ed38238ecb07af639f72e626989d9e3_201.png)

在设计记忆点时，要有意识地将该前置的元素放在开头，将该贯穿全局的元素融入内容的方方面面。

![](img/3ed38238ecb07af639f72e626989d9e3_203.png)

## 本节总结

![](img/3ed38238ecb07af639f72e626989d9e3_213.png)

本节课我们一起学习了人设定位的四大核心因素：
1.  **角色**：明确你在赛道中的专业身份。
2.  **内容**：保持内容垂直，提供持续有价值的输出。
3.  **形式**：确定适合你的内容呈现场景、声音和类型。
4.  **记忆点**：设计并强化固定元素，让用户快速识别并记住你。

![](img/3ed38238ecb07af639f72e626989d9e3_221.png)

![](img/3ed38238ecb07af639f72e626989d9e3_223.png)

记住，人设是在长期的内容运营中逐渐丰满起来的。前期做好这四点的规划和固化，能为账号的长期发展打下坚实的基础。下一节课，我们将继续探讨人设定位的配套设置，看看如何将这些核心因素落实到账号的各个细节中。
