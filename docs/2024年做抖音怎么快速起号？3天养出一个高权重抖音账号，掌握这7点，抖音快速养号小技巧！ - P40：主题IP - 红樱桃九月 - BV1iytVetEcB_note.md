## 什么是主题IP？
上一节我们介绍了账号定位的基础，本节中我们来看看如何通过**主题IP**让内容更具吸引力。

![](img/4846a72d22d78351c0f71cd1540fee83_8.png)

![](img/4846a72d22d78351c0f71cd1540fee83_10.png)

主题IP是指你的视频内容围绕一个**核心主题或系列**展开。这个主题像一根线，将你分散的视频串联起来，让观众产生持续关注的欲望。

接下来，我们通过两个视频案例来具体感受。

**案例一：家庭生活系列**
视频内容是一对夫妻在雪地中嬉戏打闹。如果只看画面，这只是一个普通的日常记录。
然而，视频标题是 **“挑战惹老婆生气的第21天”**。
这个标题赋予了视频一个明确的**系列主题**。观众会好奇：前20天发生了什么？之后还会有什么故事？这就成功引导了用户的持续关注。

**案例二：情感教学系列**
视频内容是一位“小姨”在黑板前讲解恋爱技巧，标题为 **“小姨恋爱课”**。
这个账号围绕 **“恋爱教学”** 这一垂直领域，策划了系列内容。它满足了特定人群（如不擅长恋爱的男性）的需求，容易引发共鸣，因此能快速积累粉丝。

![](img/4846a72d22d78351c0f71cd1540fee83_12.png)

通过对比可以发现，一个没有明确主题的唱歌账号，用户听完可能就划走了。而一个有**主题IP**的账号，能像连续剧一样吸引用户期待下一集。

## 如何策划你的主题IP？
理解了主题IP的重要性后，我们来看看如何为自己的账号策划一个主题IP。

![](img/4846a72d22d78351c0f71cd1540fee83_14.png)

以下是四种常见且有效的主题IP命名方法，每种方法都结合了**个人/场景标识**与**垂直领域**。

**1. 名字 + 垂直领域**
这种方法将创作者的个人品牌与专业领域结合。
*   `老韩摄影教学第12期`
*   `猪猪小姐姐的撩汉指南`

**2. 人群 + 垂直领域**
这种方法直接瞄准目标受众，内容针对性极强。
*   `别人家男朋友的拍照技巧第三期`
*   `宝妈育儿小技巧之辅食添加`

**3. 地域 + 垂直领域**
这种方法利用地域特色或经历作为内容主线。
*   `西藏自驾之旅第50天`
*   `广州美食之火锅篇`

**4. 场景 + 垂直领域**
这种方法将内容限定在特定场景下，具有高度辨识度。
*   `办公室迷惑行为之为什么总有人脚臭`
*   `婚礼现场的创意小心思`

## 总结
本节课我们一起学习了**主题IP**的策划方法。核心要点是：不要发布零散的内容，而要用一个明确的**系列主题**将你的视频串联起来。你可以通过 **“名字/人群/地域/场景 + 垂直领域”** 的公式来构思你的主题IP。一个好的主题IP能显著提升账号的权重，让用户记住你并期待你的下一次更新。
