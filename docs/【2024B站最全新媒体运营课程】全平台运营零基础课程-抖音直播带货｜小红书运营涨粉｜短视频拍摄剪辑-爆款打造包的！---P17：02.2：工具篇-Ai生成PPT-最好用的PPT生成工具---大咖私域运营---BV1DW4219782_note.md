# 新媒体运营课程：P17：02.2：工具篇-AI生成PPT 🚀

在本节课中，我们将学习如何使用一款强大的AI工具来快速生成PPT。这款工具能根据你提供的主题或现有文档，自动梳理大纲并生成详细的演示文稿内容，非常适合需要快速制作演示材料的场景。

## 工具简介与核心功能

上一节我们介绍了多种新媒体运营工具，本节中我们来看看这款专为PPT设计打造的AI助手。

这款工具的核心功能是**智能生成**。它主要提供两种工作模式：
1.  **主题生成模式**：输入一个主题，AI会像ChatGPT一样，基于该主题生成完整的PPT内容和结构。
2.  **文档分析模式**：上传一篇现有的文档（如TXT、PDF），AI会自动总结文档大纲，并基于此大纲生成详细的PPT内容。

我试用过许多AI生成PPT的工具，这款在**基于大纲生成详细内容**方面表现尤为出色。

## 编辑与优化生成结果

![](img/de708f28e471e299b4e648beb9ae1878_11.png)

虽然AI生成了初稿，但我们通常还需要进行一些调整和美化，使其更符合我们的需求。

![](img/de708f28e471e299b4e648beb9ae1878_13.png)

### 更换演示模板

工具右侧提供了“更换模板”的选项。虽然当前内置的模板数量不算非常多，但切换速度非常快。我们可以选择一个最符合内容风格的模板。

![](img/de708f28e471e299b4e648beb9ae1878_13.png)

### 调整内容细节

更重要的是，我们可以对每一页的内容进行微调。

![](img/de708f28e471e299b4e648beb9ae1878_15.png)

在编辑界面右侧，我们可以：
*   修改文字内容、字体和大小。
*   调整形状、颜色和布局。
*   替换或添加图片。

如果对模板有更高要求，我们也可以将生成好的PPT下载到本地，使用PowerPoint或Keynote等专业软件进行深度编辑。
