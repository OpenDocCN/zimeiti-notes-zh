# 抖音运营实战：P19：18、视频剪辑方法-9 🎬

在本节课中，我们将学习使用剪映App进行视频剪辑的完整流程，以及一个辅助口播拍摄的提词器软件的使用方法。课程内容从基础操作到导出设置，旨在帮助初学者快速掌握制作抖音视频的核心剪辑技巧。

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_1.png)

## 剪辑软件选择与导入

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_3.png)

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_5.png)

上一节我们介绍了视频拍摄的要点，本节中我们来看看如何进行后期剪辑。我们选择使用**剪映**进行剪辑，这是抖音官方的剪辑工具，有手机版和电脑版。本教程以手机版为例进行讲解。

首先，打开剪映App，点击“开始创作”按钮。

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_7.png)

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_1.png)

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_9.png)

接着，从相册中选择需要剪辑的视频素材，点击“添加”将其导入到剪辑轨道中。如果你的视频由多个片段组成，可以依次导入并分别处理。

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_11.png)

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_3.png)

## 基础剪辑：构图与裁剪

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_13.png)

视频导入后，第一步是调整画面构图。以下是调整步骤：

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_15.png)

1.  **旋转画面**：点击轨道上的视频片段，向右滑动底部工具栏，找到并点击“编辑”图标（方块形状）。在编辑界面中，点击左侧的“旋转”功能，将画面调整到正确的方向。
    ![](img/30b78b1d95fdf20ee9a89d5ded14fd44_7.png)

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_17.png)

2.  **裁剪比例**：在同一个编辑界面，点击“裁剪”功能。这里提供了多种比例选项，如**1:1**、**4:3**、**16:9**等。选择适合抖音的竖屏比例（如9:16）或你需要的其他比例进行裁剪。
    ![](img/30b78b1d95fdf20ee9a89d5ded14fd44_11.png)

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_19.png)

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_21.png)

调整完毕后，点击对勾返回主剪辑界面。

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_23.png)

## 核心剪辑：分割与删除

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_25.png)

接下来，我们需要剪掉视频中无用的部分，例如开头沉默、结尾冗余的画面。操作的核心是“选中-分割-删除”。

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_27.png)

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_29.png)

1.  **定位分割点**：播放视频，找到需要剪掉的片段的起始位置，将时间轴（白色竖线）移动到此位置。
2.  **执行分割**：点击选中视频轨道，然后点击底部工具栏的“分割”按钮。视频会被从此处切成两段。
    ![](img/30b78b1d5fdf20ee9a89d5ded14fd44_13.png)
3.  **删除片段**：点击选中需要删除的片段，然后点击“删除”按钮即可。
    ![](img/30b78b1d95fdf20ee9a89d5ded14fd44_15.png)

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_31.png)

重复以上步骤，去除所有不必要的部分，只保留精华内容。

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_33.png)

## 添加与编辑背景音乐

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_35.png)

视频内容修剪好后，需要添加背景音乐（BGM）。点击底部工具栏的“音频”按钮。

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_37.png)

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_39.png)

以下是添加音乐的两种方法：

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_41.png)

*   **提取音乐**：如果你已经用其他工具（如淘金阁）生成了配音文件，可以点击“提取音乐”，从手机视频中提取音频。
    ![](img/30b78b1d95fdf20ee9a89d5ded14fd44_17.png)
*   **添加音乐**：点击“添加音乐”，可以使用剪映自带的音乐库。你可以按分类查找，或直接搜索歌曲名称。试听后点击“使用”即可添加。
    ![](img/30b78b1d95fdf20ee9a89d5ded14fd44_19.png)

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_43.png)

音乐添加后，如果长度不合适，同样可以使用“分割”和“删除”功能进行剪辑。选中音乐轨道，移动时间轴到目标位置，分割后删除多余部分。
![](img/30b78b1d95fdf20ee9a89d5ded14fd44_29.png)

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_45.png)

你还可以调整音乐音量：选中音乐轨道，点击“音量”按钮，拖动滑块即可调节。
![](img/30b78b1d95fdf20ee9a89d5ded14fd44_31.png)

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_47.png)

## 快速添加与修改字幕

手动添加字幕效率很低。剪映提供了高效的“识别字幕”功能。

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_49.png)

1.  点击底部工具栏的“文本”按钮。
2.  选择“识别字幕”。
3.  在弹出的菜单中，选择“仅识别视频原声”，然后点击“开始识别”。系统会自动为视频中的人声生成字幕。
    ![](img/30b78b1d95fdf20ee9a89d5ded14fd44_35.png)

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_51.png)

识别完成后，字幕会出现在视频上。你可以进行以下修改：

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_53.png)

*   **修改错别字**：双击字幕条，即可进入编辑界面修改文本。
    ![](img/30b78b1d95fdf20ee9a89d5ded14fd44_41.png)
*   **调整样式**：选中字幕条，在底部可以更改字体、颜色、大小、动画等。例如，让字幕颜色与画面主体颜色呼应。
    ![](img/30b78b1d95fdf20ee9a89d5ded14fd44_43.png)

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_55.png)

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_57.png)

## 导出设置与发布

所有剪辑步骤完成后，在导出前需要进行关键设置，以保证视频清晰度。

点击右上角的“导出”按钮。在导出设置界面：
1.  将**帧率**设置为 **30 fps**。
2.  将**分辨率**拉到最高（如1080p或4K）。

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_59.png)

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_61.png)

```plaintext
导出设置公式：帧率 = 30fps，分辨率 = 最高可用
```

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_47.png)

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_63.png)

设置完成后，点击“导出”。导出过程中请勿锁屏或切换应用程序，等待导出完成。完成后，可以直接通过剪映分享到抖音。
![](img/30b78b1d95fdf20ee9a89d5ded14fd44_53.png)

## 口播辅助工具推荐

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_65.png)

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_67.png)

对于需要出镜进行口播讲解的同学，背诵稿子是一大难题。这里推荐使用“爱提词”App。

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_69.png)

这款软件的核心功能是悬浮提词器，可以让你在拍摄时看着台词念，无需背诵。

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_71.png)

以下是基本使用方法：
1.  打开App，点击右下角“+”号，新建一个提词板。
2.  输入或粘贴你的口播文案，并保存。
    ![](img/30b78b1d95fdf20ee9a89d5ded14fd44_63.png)
3.  点击“开始提词”，选择“录制模式”。台词会以悬浮窗形式显示在相机画面上，你可以边看边读。
4.  在设置中，可以调整台词字体大小、滚动速度等，方便阅读。
    ![](img/30b78b1d95fdf20ee9a89d5ded14fd44_65.png)
5.  软件内置美颜功能，可以调整瘦脸、大眼、磨皮等参数，让拍摄效果更自然美观。
    ![](img/30b78b1d95fdf20ee9a89d5ded14fd44_69.png)

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_73.png)

![](img/30b78b1d95fdf20ee9a89d5ded14fd44_75.png)

此外，App内还提供了“爆款文案”板块，涵盖情感、职场、生活等多个领域，可以作为内容创作的参考。
![](img/30b78b1d95fdf20ee9a89d5ded14fd44_77.png)
