# 手机剪辑教程：P43：雨滴移动相册跟踪效果 🌧️📱

在本节课中，我们将学习如何使用手机剪辑软件制作一个“雨滴移动相册跟踪效果”。这个效果能让多张照片像雨滴一样依次滑入画面，并带有边框和背景模糊，最终组合成一个动态的电子相册。整个过程分为素材准备、特效添加和关键帧动画三个主要部分。

## 素材准备与初步排列 🖼️

首先，我们需要导入并排列好所有的基础照片素材。

1.  导入第一张图片到剪辑轨道。
2.  将这张图片的时长拉长。
3.  缩小这张图片，并将其移动到画面的左上角。
4.  通过“新增画中画”功能，导入第二张图片。
5.  将第二张图片摆放在画面中第二个预设的位置上。
6.  调整第二张图片的时长，使其与第一张图片的时长对齐。
7.  重复步骤4至6，依次导入并摆放好所有图片，直到所有照片都在画面中排列整齐。

在完成所有图片的初步排列后，我们需要为它们设置一个统一的背景。

1.  打开“背景”设置选项。
2.  选择“画布模糊”功能。
3.  在模糊程度选项中，选择第四个（通常是模糊程度最高的选项）。

## 添加边框与导出中间素材 🎨

上一节我们排列好了照片并设置了背景，本节中我们为每张照片添加统一的装饰边框。

以下是添加边框特效的步骤：
*   打开“特效”菜单，选择“画面特效”。
*   在“边框”分类中，找到并选择“粉黄渐变”特效。
*   将这个特效的时长拉长，使其与主视频的时长对齐。
*   为第二张图片（画中画图层）添加相同的“粉黄渐变”边框特效，并将其时长对齐。
*   关键的一步是：将这个特效的“作用对象”设置为对应的画中画图片。
*   重复此过程，为轨道上的所有图片都加上相同的边框特效。

完成以上步骤后，将当前工程**导出为一个视频文件**，并备用。这个文件是我们制作最终动画效果的基础素材。

## 创建关键帧移动动画 ⏩

现在，我们利用刚才导出的视频素材来创建照片依次滑入的动画效果。

1.  重新导入上一步导出的那个视频文件。
2.  将视频画面放大，并放置到画面的左上角（作为动画的起始位置）。
3.  在视频开头处**添加一个关键帧**，记录下当前位置。
4.  将时间线移动到**1秒后**的位置，**再次添加一个关键帧**。
5.  在这个1秒处的关键帧上，将画面**移动并放大**，使第二张照片完整地显示在画面中（即移出第一个画面，进入第二个画面）。
6.  将时间线再**向后移动10帧**，**添加一个关键帧**（此帧用于保持画面稳定）。
7.  之后，在**每秒的间隔处**，都添加一个关键帧，并将画面移动到下一张照片的位置。
8.  每次移动画面后，同样间隔10帧添加一个关键帧用于定格。
9.  重复此过程，直到时间线来到视频结尾处，此时应将所有照片画面都移出显示区域，完成整个滑动动画。

## 添加音效与最终合成 🎵

动画部分已经完成，本节我们为视频添加雨滴音效和背景音乐，完成最终合成。

1.  在音效素材库中，搜索并加入“雨滴”声音特效。
2.  将雨滴音效的时长拉长，使其与整个视频的时长对齐。
3.  通过“添加音频”功能，导入你在抖音收藏的歌曲作为背景音乐。
4.  预览并调整音效与音乐的音量平衡，确保效果和谐。

本节课中我们一起学习了制作“雨滴移动相册跟踪效果”的全过程。我们首先排列照片并设置模糊背景，然后为每张照片添加边框并导出中间素材。接着，通过设置一系列关键帧，制作了照片依次滑入的平滑动画。最后，我们为动画配上了应景的雨滴音效和背景音乐。掌握这个流程，你就能轻松制作出富有动感和氛围感的电子相册视频。
