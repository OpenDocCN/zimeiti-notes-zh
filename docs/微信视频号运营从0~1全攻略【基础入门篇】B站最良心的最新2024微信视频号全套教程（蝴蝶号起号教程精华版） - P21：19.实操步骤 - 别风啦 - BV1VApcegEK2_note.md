# 微信视频号运营从0~1全攻略：P21：19.实操步骤

在本节课中，我们将要学习如何开通微信视频号的商品橱窗功能，这是进行视频号带货的基础操作。整个过程分为几个清晰的步骤，我们将逐一进行讲解。

## 概述
本节教程将详细演示从打开微信到成功开通视频号商品橱窗权限的全过程。核心操作包括进入视频号创作者中心、找到带货功能入口、选择身份并缴纳保证金。我们将确保每一步都简单明了，便于初学者跟随操作。

## 操作步骤详解
以下是开通视频号商品橱窗权限的具体步骤，请按照顺序进行操作。

![](img/f759513a264daf37fa38cddfc270a519_4.png)

### 第一步：打开微信
首先，我们需要在手机上启动微信应用。这是所有后续操作的基础。

![](img/f759513a264daf37fa38cddfc270a519_6.png)

### 第二步：进入“发现”页
打开微信后，点击底部导航栏的 **“发现”** 选项卡。

### 第三步：找到视频号入口
在“发现”页面中，将视线向上滑动。您会看到 **“视频号”** 的入口，其图标是一个黄色的蝴蝶标志。点击该入口进入视频号。

![](img/f759513a264daf37fa38cddfc270a519_8.png)

![](img/f759513a264daf37fa38cddfc270a519_9.png)

### 第四步：进入个人中心
进入视频号界面后，在页面右上角找到一个人形头像图标，点击它进入个人中心。

### 第五步：进入创作者中心
在个人中心页面，向下滑动，直到页面最底部。找到并点击 **“创作者中心”** 选项。
> **注意**：如果您的账号是全新注册且未发布过任何作品，可能暂时看不到此选项。发布一条视频后，该功能便会出现。

### 第六步：找到带货中心
进入创作者中心后，在页面中找到 **“带货中心”** 的选项，其图标类似一个小卖部。点击进入。

![](img/f759513a264daf37fa38cddfc270a519_11.png)

### 第七步：开通橱窗权限
点击“带货中心”后，系统会根据您的账号状态进行跳转。
*   **如果未开通**：页面将引导您开通橱窗权限。
*   **如果已开通**：页面将直接进入商品上架管理界面。

对于需要开通权限的账号，系统会要求您选择带货身份。通常会出现三个选项：
1.  我是商家
2.  我是达人
3.  我是商家且已有视频号

![](img/f759513a264daf37fa38cddfc270a519_13.png)

**请选择“我是达人”**。即使您是厂家或拥有自有品牌，也建议先用这个身份开通，因为其门槛相对较低。

### 第八步：缴纳保证金
选择“我是达人”后，页面会跳转到保证金缴纳界面。您需要点击 **“去交纳”** 按钮，系统会引导您在微信支付中完成100元保证金的缴纳。缴纳成功后，您的商品橱窗权限即开通。

![](img/f759513a264daf37fa38cddfc270a519_15.png)

## 总结
本节课中，我们一起学习了开通微信视频号商品橱窗功能的完整流程。关键步骤可以总结为：**打开微信 -> 进入发现页 -> 点击视频号 -> 进入个人中心 -> 打开创作者中心 -> 进入带货中心 -> 选择“达人”身份 -> 缴纳保证金**。完成这些操作后，您就具备了在视频号中展示和销售商品的基础资格，为后续的带货运营做好了准备。
