# 新媒体运营课程：P15：01.10.03：🤖 训练AI从分析到产出爆款笔记

在本节课中，我们将学习如何训练AI（以ChatGPT为例）分析爆款笔记，并基于分析结果生成具备爆款潜质的新内容。我们将通过一个考研领域的案例，完整演示从提供素材、分析爆款因素到最终生成文案的全过程。

上一节我们介绍了通过提供资料让ChatGPT理解特定平台风格和用语的方法。虽然上一节的旅游攻略案例结果不理想，但方法是正确的。本节中，我们来看看如何通过分析具体的爆款内容，让AI学习并产出高质量的笔记。

## 分析爆款笔记的核心要点

![](img/7b5ac2fc6b8541dbf7d1bf26c753151a_3.png)

![](img/7b5ac2fc6b8541dbf7d1bf26c753151a_5.png)

要让AI学习爆款笔记，我们需要引导它分析几个关键点。以下是需要关注的核心要素：

![](img/7b5ac2fc6b8541dbf7d1bf26c753151a_7.png)

![](img/7b5ac2fc6b8541dbf7d1bf26c753151a_9.png)

![](img/7b5ac2fc6b8541dbf7d1bf26c753151a_11.png)

![](img/7b5ac2fc6b8541dbf7d1bf26c753151a_13.png)

*   **爆款因素分析**：分析内容为何能成为爆款，例如是否引发了情感共鸣、提供了实用价值等。
*   **内容结构拆解**：分析文案的整体结构，包括：
    *   **开篇**：用什么方式吸引用户继续阅读。
    *   **中间**：提供了哪些内容来戳中用户痛点，提升用户停留时长。
*   **结尾**：设置了什么“钩子”来引导用户互动（如私信、关注）。
*   **关键词策略**：分析笔记中是否植入了特定关键词，以及关键词的出现频率和铺设方式。

### 第一步：提供爆款笔记并下达分析指令

![](img/7b5ac2fc6b8541dbf7d1bf26c753151a_23.png)

![](img/7b5ac2fc6b8541dbf7d1bf26c753151a_25.png)

![](img/7b5ac2fc6b8541dbf7d1bf26c753151a_27.png)

首先，我们需要找到目标领域的爆款笔记内容。找到后，向ChatGPT（此处使用3.5版本示例）下达清晰的分析指令。

![](img/7b5ac2fc6b8541dbf7d1bf26c753151a_29.png)

**提供给AI的指令示例：**
```
我给你提供一些小红书平台上关于考研的爆款笔记文案。你需要分析这些文案的爆款因素，比如文案的整体结构（开篇如何吸引用户、中间如何戳中痛点、结尾如何引导互动）。同时，分析每篇笔记是否设计了关键词。如果有，请告诉我关键词是什么、出现频率如何。
```

![](img/7b5ac2fc6b8541dbf7d1bf26c753151a_31.png)

随后，将复制的爆款笔记文案粘贴给AI。

### 第二步：查看并评估AI的分析结果

![](img/7b5ac2fc6b8541dbf7d1bf26c753151a_40.png)

![](img/7b5ac2fc6b8541dbf7d1bf26c753151a_42.png)

AI会对每篇笔记进行反馈。以下是一个典型分析结果的组成部分：

![](img/7b5ac2fc6b8541dbf7d1bf26c753151a_44.png)

![](img/7b5ac2fc6b8541dbf7d1bf26c753151a_46.png)

![](img/7b5ac2fc6b8541dbf7d1bf26c753151a_48.png)

![](img/7b5ac2fc6b8541dbf7d1bf26c753151a_50.png)

1.  **情感共鸣**：文案开篇用焦虑或担忧的情感吸引考生产生共鸣。
2.  **逆袭故事**：分享个人备考经验和逆袭故事，增加可信度。
3.  **实用建议**：提供英语、政治、专业课等科目的具体备考方法和时间规划。
4.  **表现形式**：使用表情符号（emoji）增加亲和力与情感表达。
5.  **正能量鼓励**：文末鼓励读者，传递正能量。
6.  **关键词分析**：判断文案是否明确植入了设计好的关键词。在本案例中，AI指出几篇笔记均未明显使用“中文分词法”设计关键词，多为直接常用词汇。

**注意**：AI有时会“遗忘”之前的指令或分析过的内容。如果发现分析跑偏，需要重新强调核心指令，或提醒它回顾已分析过的笔记数量与内容。

![](img/7b5ac2fc6b8541dbf7d1bf26c753151a_59.png)

### 第三步：整合爆款因素并生成大纲

在提供多篇笔记进行分析后，指令AI对分析出的所有爆款因素进行去重和整合。

**整合指令示例：**
```
请将你前面分析过的几篇考研爆款笔记的爆款因素总结一下，整合成一个清晰的列表。
```

获得整合后的爆款因素列表（如：情感共鸣、明确备考计划、时间节点、实用资源推荐、正能量鼓励等）后，即可进入创作阶段。首先，让AI根据这些因素生成内容大纲。

![](img/7b5ac2fc6b8541dbf7d1bf26c753151a_62.png)

![](img/7b5ac2fc6b8541dbf7d1bf26c753151a_64.png)

**生成大纲指令示例：**
```
我要写一篇关于考研的小红书笔记。请结合你刚才总结的爆款因素，为我规划一个具备爆款潜质的笔记大纲。大纲需要细分到具体的小节点。
```

AI生成的大纲可能包含：吸引人的开篇、主题介绍、各科备考规划、时间管理、心态调整、互动引导等部分。

### 第四步：根据大纲生成完整文案

![](img/7b5ac2fc6b8541dbf7d1bf26c753151a_66.png)

确认大纲后，指令AI根据大纲和爆款因素撰写完整文案，并提出更细致的要求。

![](img/7b5ac2fc6b8541dbf7d1bf26c753151a_68.png)

![](img/7b5ac2fc6b8541dbf7d1bf26c753151a_69.png)

![](img/7b5ac2fc6b8541dbf7d1bf26c753151a_70.png)

![](img/7b5ac2fc6b8541dbf7d1bf26c753151a_72.png)

**生成文案指令示例：**
```
请按照你提供的大纲，撰写完整的文案。要求如下：
1. 文案结构严格遵循爆款笔记的特点。
2. 包含一个简单的“逆袭小故事”来引入读者情绪。
3. 文中要使用适量的emoji表情。
4. 尝试结合中文分词法的思路，融入“考研”、“备考规划”等关键词。
```

## 成果评估与优化方向

AI生成的文案在结构、专业度和情感元素上已具备基础框架，例如：
*   开篇使用“我理解那种感觉…”引发共鸣。
*   中间部分提供清晰的科目备考建议和时间线。
*   结尾进行鼓励并引导互动。

对于一般运营需求，此文案已可使用。若要进一步优化，可以考虑：
1.  **提供更丰富的学习资料**：将更多优秀笔记的排版、固定句式或专属表情包作为资料喂给AI，让它学习更独特的风格。
2.  **使用更强大的模型**：如ChatGPT-4.0，通常能产生更优质、更智能的内容。
3.  **创建定制化AI助手**：利用GPTs等功能，打造一个熟知你个人风格和需求的专属内容生成助手，避免反复提醒指令。

## 总结

本节课我们一起学习了训练AI产出爆款笔记的完整流程。关键步骤包括：**提供爆款素材→下达精准分析指令→整合爆款因素→生成内容大纲→撰写并优化文案**。通过这种方法，我们可以系统化地利用AI学习平台爆款逻辑，并高效生成符合要求的初稿，为内容创作提供有力支持。
