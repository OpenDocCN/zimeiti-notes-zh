# 抖音运营教程：P19：11、视频类型1-2 📚

在本节课中，我们将要学习抖音账号运营中一个至关重要的环节：**建立人设**。我们将深入探讨强人设与弱人设的区别，并重点讲解第一种核心视频类型——**口播**，帮助你理解如何通过内容塑造个人形象，吸引目标粉丝。

上一节我们介绍了文案的重要性，本节中我们来看看如何通过建立人设来连接“涨粉”与“变现”。人设不是用来“装”的，而是为了让用户在众多同类账号中记住你。例如，有十个人都做装修内容，但其中一位建立了幽默搞笑的人设，用户就更容易记住他。当用户喜欢你这个人时，自然就会成为你的粉丝。

## 人设的两种类型：强与弱

要建立有效的人设，首先需要理解其两种基本类型。

以下是两种人设的核心定义：

![](img/8336e2753269a4c4499143c799da03e8_1.png)

*   **强人设**：以**你个人**为切入点。内容与你自己密切相关，分享你的行业经验、工作生活或个人兴趣爱好。这类账号通常需要真人出镜。
    *   **公式**：强人设 = 个人经验/生活/爱好 + 真人出镜
    *   **示例领域**：装修、餐饮、法律咨询、电商、美食探店、旅行vlog、美妆护肤、宠物日常等。



![](img/8336e2753269a4c4499143c799da03e8_3.png)

![](img/8336e2753269a4c4499143c799da03e8_1.png)

*   **弱人设**：内容与**你本人**关系不大，核心目的是运营好账号本身。内容主体是素材、知识或观点，而非塑造创作者个人形象。
    *   **公式**：弱人设 = 主题内容（如文案、影视） > 创作者本人
    *   **示例类型**：情感语录配图/音乐、影视解说、科普动画、书单分享等。



![](img/8336e2753269a4c4499143c799da03e8_3.png)

理解了人设的分类，我们就能更好地规划内容。对于强人设账号，所有与你个人相关的内容都可以成为视频素材。

## 核心视频类型一：口播 🎤

接下来，我们详细讲解第一种适用于强人设的、非常高效的内容形式：口播。

口播，即面对镜头用语言进行表达的视频形式。它非常适合用来“讲故事”或“分享知识”。

以下是口播的两大主要方向：

1.  **讲故事型口播**：分享客户故事、从业经历或人生感悟。故事本身不一定完全真实，可以从文案平台寻找有趣素材。目的是让观众觉得**有意思**，愿意听下去。
2.  **教知识型口播**：分享特定领域的专业知识或实用技巧。例如育儿、金融、养生、装修、房产知识等。目的是让观众在几十秒内获得**收获感**，觉得学到了东西。

口播的优势在于制作相对简单，且适用领域广泛。

以下是关于口播常见问题的解答：

*   **问：口才不好能做口播吗？**
    *   **答：可以。** 口播是提前准备好稿子再录制，并非临场发挥。准备好文案是关键。
*   **问：普通话不标准能做口播吗？**
    *   **答：可以，有时效果更好。** 如果你的目标客户是同城或有地域属性，使用方言会显得更亲切、有趣。观众可以通过字幕理解内容。

本节课中我们一起学习了抖音账号运营中“人设”的核心概念，明确了**强人设**与**弱人设**的区别。我们重点剖析了**口播**这种视频类型，了解了它通过“讲故事”和“教知识”来吸引用户的逻辑，并解答了关于口才和普通话的常见疑虑。掌握这些基础，是规划你账号内容方向的重要第一步。
