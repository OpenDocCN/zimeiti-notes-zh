# 新媒体运营精品教程：P15：01.10.03：AI：训练AI模拟爆款笔记到产出爆款笔记的过程 🤖

在本节课中，我们将学习如何利用AI工具（如ChatGPT）分析平台上的爆款内容，并基于分析结果，指导AI生成具备爆款潜质的新笔记。这是一个从“学习”到“创作”的完整流程。

上一节我们介绍了通过“喂资料”的方式，让AI理解特定平台的用语和风格。虽然以旅游攻略为例的最终内容可能不够理想，但方法是正确的。本节中，我们来看看如何深化这一过程，通过分析具体的爆款笔记，让AI学习其成功要素并用于创作。

## 核心分析要点

![](img/2febe4d844de1b6ccc954121165228b7_1.png)

![](img/2febe4d844de1b6ccc954121165228b7_3.png)

![](img/2febe4d844de1b6ccc954121165228b7_5.png)

要让AI学习爆款笔记，我们需要引导它分析以下几个关键点：

![](img/2febe4d844de1b6ccc954121165228b7_7.png)

![](img/2febe4d844de1b6ccc954121165228b7_9.png)

![](img/2febe4d844de1b6ccc954121165228b7_11.png)

![](img/2febe4d844de1b6ccc954121165228b7_13.png)

以下是需要AI重点分析的几个方面：

1.  **爆款因素**：分析内容为何能成为爆款，例如是否引发了情感共鸣、提供了实用价值、讲述了逆袭故事等。
2.  **内容结构**：
    *   **开篇**：用什么方式（如提问、制造焦虑、引用名人）吸引用户继续阅读。
    *   **中间**：提供了哪些内容来戳中用户痛点，锁住用户停留时间。
    *   **结尾**：如何设置“钩子”（如鼓励互动、引导私信、提供福利）促使用户行动。
3.  **关键词策略**：分析笔记中是否刻意植入了关键词，以及关键词的出现频率和分布方式。

接下来，我们需要给AI清晰的指令，让它按照我们的要求进行分析。指令范例如下：

![](img/2febe4d844de1b6ccc954121165228b7_35.png)

![](img/2febe4d844de1b6ccc954121165228b7_36.png)

```
你是一名小红书爆款文案分析师。我将提供一篇小红书上的爆款笔记文案，请你分析：
1.  它的爆款因素有哪些？（例如：情感共鸣、实用干货、逆袭故事等）
2.  它的整体结构是什么？（开篇如何吸引用户？中间如何阐述内容？结尾如何引导互动？）
3.  它是否设计了关键词？如果有，是哪个词？出现频率如何？
```

### 第三步：解析AI的反馈

![](img/2febe4d844de1b6ccc954121165228b7_53.png)

![](img/2febe4d844de1b6ccc954121165228b7_55.png)

![](img/2febe4d844de1b6ccc954121165228b7_56.png)

AI会对文案进行分析并给出反馈。例如，对于一篇考研笔记，AI可能给出如下分析：

*   **爆款因素**：情感共鸣（开头描述备考焦虑）、逆袭故事（分享个人成功经验）、实用建议（提供具体备考方法）、使用表情符号增加亲和力。
*   **内容结构**：开篇以焦虑情绪吸引同类考生；中间分享英语、专业课备考方法；结尾给予鼓励，引导关注。
*   **关键词分析**：本篇未发现明显按中文分词法设计的关键词，词汇多为常用词。

重复以上步骤，为AI提供多篇（建议3-5篇）同领域的爆款笔记进行分析，让它积累足够的“经验”。

![](img/2febe4d844de1b6ccc954121165228b7_58.png)

![](img/2febe4d844de1b6ccc954121165228b7_60.png)

### 第四步：整合爆款要素并生成大纲

在AI分析了多篇笔记后，我们需要让它总结共通的爆款要素。

![](img/2febe4d844de1b6ccc954121165228b7_62.png)

可以下达指令：“请将你之前分析过的几篇考研爆款笔记的爆款因素总结一下，去重后整合成一个清晰的列表。”

AI可能会总结出：情感共鸣、明确的备考计划、时间节点清晰、提供具体资源/技巧、正能量鼓励等要素。

接着，基于这些要素，让AI生成一篇新笔记的大纲。
指令如下：“请结合你总结的爆款笔记因素，为我规划一篇关于考研的小红书笔记大纲。大纲需要细分到具体的小节点。”

![](img/2febe4d844de1b6ccc954121165228b7_64.png)

![](img/2febe4d844de1b6ccc954121165228b7_66.png)

![](img/2febe4d844de1b6ccc954121165228b7_67.png)

### 第五步：根据大纲生成完整文案

![](img/2febe4d844de1b6ccc954121165228b7_68.png)

最后，我们让AI根据确定好的大纲，生成完整的笔记文案。此时可以提出更细致的要求。

指令范例如下：
```
请按照你提供的大纲，撰写完整的文案。
要求：
1.  结构严格遵循爆款笔记的要素。
2.  开头融入一个简短的“逆袭小故事”引入情绪。
3.  文中适当使用emoji表情。
4.  尝试自然融入“考研规划”、“政治备考”等关键词。
```

AI会根据指令生成一篇融合了爆款要素的完整笔记初稿。这篇稿子已经具备了不错的基础，运营人员可以在此基础上进行微调和优化，使其更符合个人风格或具体需求。

## 进阶技巧与工具

![](img/2febe4d844de1b6ccc954121165228b7_70.png)

在操作中可能会遇到AI遗忘前文指令的情况，这是当前模型的局限。解决方法是在关键步骤前重新强调核心要求。

此外，利用ChatGPT的“自定义GPT”（GPTs）功能，可以创建专属的助手。你可以上传小红书风格的文案范例、专用表情包列表等资料供其学习，从而打造一个更懂你需求的私人内容创作助手，提升输出内容的稳定性和质量。

![](img/2febe4d844de1b6ccc954121165228b7_72.png)

![](img/2febe4d844de1b6ccc954121165228b7_74.png)

本节课中我们一起学习了如何分步骤地训练AI分析爆款笔记，并最终产出具备爆款潜质的新内容。从“提供素材”到“分析要素”，再到“整合生成”，这个过程能有效提升利用AI进行内容创作的效率和质量。后续课程将继续分享更多AI在小红书、抖音等平台运营中的高级应用技巧。
