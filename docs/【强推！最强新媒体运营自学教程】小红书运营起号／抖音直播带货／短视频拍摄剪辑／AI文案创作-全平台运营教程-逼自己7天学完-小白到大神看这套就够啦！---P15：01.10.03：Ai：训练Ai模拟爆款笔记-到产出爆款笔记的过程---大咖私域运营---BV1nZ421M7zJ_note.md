在本节课中，我们将学习如何训练AI（以ChatGPT为例）分析现有爆款笔记，并基于分析结果，生成符合平台风格和爆款逻辑的新笔记。整个过程分为两个核心步骤：**分析学习**与**模仿创作**。

## 分析爆款笔记：教会AI识别成功要素 🧐

![](img/45f0294684789659e0aecd2432d54c4c_9.png)

![](img/45f0294684789659e0aecd2432d54c4c_11.png)

以下是分析爆款笔记时需要让AI重点关注的几个方面。我们需要通过清晰的指令引导AI进行拆解：

1.  **爆款因素分析**：分析内容为何能成为爆款。例如，是否引发了情感共鸣、提供了实用价值、讲述了逆袭故事等。
2.  **文案结构分析**：
    *   **开篇**：用什么方式（如提问、制造焦虑、引用名人）吸引用户继续阅读。
    *   **中间**：用什么内容（如痛点分析、详细攻略、资源推荐）留住用户，提升停留时长。
    *   **结尾**：用什么“钩子”（如鼓励互动、提供联系方式、引导私信）促使用户行动。
3.  **关键词策略分析**：分析笔记中是否刻意植入了特定关键词，以及这些关键词的出现频率和分布方式。

### 第一步：收集并输入样本

![](img/45f0294684789659e0aecd2432d54c4c_25.png)

![](img/45f0294684789659e0aecd2432d54c4c_27.png)

首先，需要找到多篇点赞量高的“考研”爆款笔记，将它们的全文内容复制下来。

![](img/45f0294684789659e0aecd2432d54c4c_29.png)

![](img/45f0294684789659e0aecd2432d54c4c_31.png)

![](img/45f0294684789659e0aecd2432d54c4c_33.png)

**操作指令示例**：
```
我给你提供几篇小红书平台上关于“考研”的爆款笔记文案。请你分析这些文案的爆款因素，例如：
1. 文案的整体结构：开篇如何吸引用户？中间如何戳中用户痛点？结尾如何引导用户互动或关注？
2. 分析每篇笔记是否有结合中文分词法设计关键词。如果有，请指出是哪个关键词，以及它的出现频率。
```

### 第二步：让AI进行分析并总结规律

![](img/45f0294684789659e0aecd2432d54c4c_50.png)

![](img/45f0294684789659e0aecd2432d54c4c_51.png)

![](img/45f0294684789659e0aecd2432d54c4c_53.png)

![](img/45f0294684789659e0aecd2432d54c4c_54.png)

AI在分析每篇笔记后，会给出各自的爆款因素和结构特点。例如，它可能总结出：
*   开篇常用“情感共鸣”（如备考焦虑）或“知名人物引用”（如罗翔老师的话）吸引读者。
*   中间部分核心是提供“明确的备考计划”、“针对性建议”、“具体资源推荐”。
*   结尾注重“正能量鼓励”和“引导互动”。
*   多数笔记没有严格设计关键词，更多使用直接、常用的词汇。

接着，我们需要让AI对所有分析结果进行整合去重，提炼出通用的爆款要素列表。

**操作指令示例**：
```
请将你前面分析过的几篇考研笔记的爆款因素总结一下，整合成一个完整的列表。
```

### 第三步：基于分析结果指导AI创作

![](img/45f0294684789659e0aecd2432d54c4c_56.png)

![](img/45f0294684789659e0aecd2432d54c4c_58.png)

获得爆款要素列表后，即可指导AI进行创作。这个过程建议分两步走：先列大纲，再写正文。

1.  **生成大纲**：要求AI根据总结的爆款因素，规划一篇新笔记的框架。
    **操作指令示例**：
    ```
    请结合你刚才分析的爆款笔记因素，为我规划一篇关于考研的小红书笔记大纲。大纲节点要细分，在我确认无误后再生成正文。
    ```

2.  **生成正文**：基于确认后的大纲，让AI填充内容，并再次强调需融入爆款要素。
    **操作指令示例**：
    ```
    请按照上面确认的大纲，撰写具体的笔记文案。要求：
    *   结构严格遵循爆款笔记因素。
    *   包含一个简单的“逆袭”小故事来引入情绪。
    *   使用适量的emoji表情。
    *   尝试在文案中自然融入“考研”等相关关键词。
    ```

### 第四步：优化与进阶

AI生成的初稿可能不尽完美，但已具备可用框架。此时可以：
*   **人工优化**：对文案的流畅度、细节和感染力进行手动调整。
*   **使用高级工具**：如ChatGPT 4.0或自定义GPTs（GPT商店中的定制机器人），它们具有更强的理解力、记忆力和定制化能力。你可以上传更多样本、设定固定的文案风格模板，打造专属的“爆款笔记生成助手”，从而获得更稳定、优质的结果。

## 核心要点与公式总结 📊

本节课中我们一起学习了利用AI辅助创作爆款笔记的完整工作流。其核心逻辑可以概括为以下公式：

![](img/45f0294684789659e0aecd2432d54c4c_63.png)

**爆款笔记生成 = 爆款样本分析 + 要素规律总结 + 结构化模仿创作**

1.  **输入**：提供多篇目标领域的真实爆款笔记。
2.  **处理**：通过精准指令让AI分析**结构**、**爆点**和**关键词**，并总结规律。
3.  **输出**：分步指导AI（先大纲后正文）生成融合了这些规律的新内容。
4.  **迭代**：对初稿进行人工润色，或利用更先进的AI工具提升效果。

![](img/45f0294684789659e0aecd2432d54c4c_65.png)

![](img/45f0294684789659e0aecd2432d54c4c_67.png)

通过这个系统化的方法，即使是没有经验的运营者，也能借助AI快速产出符合平台调性、具备爆款潜质的内容草稿，极大提升创作效率。
