# 抖音运营教程：P19：11、视频类型1-2 - 人设与口播 📚

在本节课中，我们将要学习抖音内容创作中两个至关重要的概念：**人设**的建立与**口播**视频的制作。理解并应用这些知识，是连接内容创作与粉丝增长、最终实现变现的关键环节。

上一节我们解决了文案问题，本节中我们来看看内容的核心——人设。许多教程直接谈论涨粉与变现，但在这两者之间，有一个不可或缺的环节：**建立你的人设**。人设不是为了炫耀，而是为了在众多同类创作者中脱颖而出。例如，有十个人都做装修内容，但其中一位建立了幽默搞笑的人设，观众就更容易记住他。当观众喜欢你这个人时，他们自然就成为你的粉丝。

现在我们来讲解这个知识点，请边听边记。人设主要分为两种类型。

以下是两种主要的人设类型：

![](img/deadff0c86d239ab5be4c151f8a8a419_1.png)

*   **强人设**：以创作者**个人**为切入点。内容与创作者自身密切相关，例如分享个人行业经验、工作生活或兴趣爱好。这类内容通常需要真人出镜。
    *   **公式**：**强人设 = 个人经验/生活/爱好 + 真人出镜**
    *   **示例领域**：装修、餐饮、法律咨询、电商、美食探店、旅行vlog、美妆护肤、服装穿搭等。
*   **弱人设**：内容与创作者**个人**关联度较低，核心目的是运营好账号本身。创作者个人并非内容的焦点。
    *   **公式**：**弱人设 = 主题内容（如文案、影视） > 创作者个人**
    *   **示例类型**：情感语录混剪、影视解说、音乐分享、非真人出镜的知识科普等。

![](img/deadff0c86d239ab5be4c151f8a8a419_3.png)

区分清楚强弱人设后，我们才能更好地规划内容。接下来，我们看看如何基于人设来生产视频内容。只要内容与你个人相关，都可以作为视频素材。

以下是几种基于人设的常见视频内容方向，我们先从“讲故事”开始：

1.  **讲故事**：你可以分享客户故事、从业经历或人生感悟。故事本身不要求完全真实，可以从文案平台寻找有趣素材进行分享。讲述故事最主要的形式就是**口播**。

“口播”即用嘴巴播讲，是内容创作的一种高效形式。刚才我们提到了讲故事的**口播**，其实它的分类非常广泛。

以下是常见的口播分类：

*   **故事型口播**：讲述各类故事，目的是让观众觉得有趣、有共鸣。
*   **知识型口播**：分享实用知识，目的是让观众在几十秒内获得“收获感”，觉得学到了东西。
    *   **示例领域**：育儿知识、金融财经、养生技巧、装修避坑、房产汽车知识等。

无论你身处哪个领域，例如法律、保险、金融，或是装修、汽车行业，都可以采用口播形式。口播的优势在于拍摄简单，一部手机即可完成，核心在于准备好文案。

针对口播常见的两个顾虑，这里给出明确解答：

*   **口才不好能做口播吗？**  
    **可以。** 口播是依据提前准备好的稿子进行录制，并非临场发挥。你看到的口播达人，大多也是经过充分准备的。
*   **普通话不好能做口播吗？**  
    **可以。** 如果你的目标客户是同城或有地域属性，使用方言可能效果更佳，显得亲切且有趣。同时，你可以通过添加字幕来确保信息传递无障碍。

本节课中我们一起学习了抖音运营中“人设”的建立与“口播”视频的制作方法。我们明确了强人设与弱人设的区别，并了解了口播作为一种高效内容形式的具体应用与优势。掌握这些，你就为创作吸引人、有辨识度的内容打下了坚实基础。在接下来的课程中，我们将继续深入，学习如何具体拍摄这些内容。
