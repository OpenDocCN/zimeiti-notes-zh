## 人设定位的四大核心因素

### 1. 角色：明确你的专业身份

![](img/36ed2937e3a679473f5b4e9823049005_50.png)

![](img/36ed2937e3a679473f5b4e9823049005_52.png)

![](img/36ed2937e3a679473f5b4e9823049005_54.png)

![](img/36ed2937e3a679473f5b4e9823049005_56.png)

![](img/36ed2937e3a679473f5b4e9823049005_58.png)

角色，简单来说，就是你在特定赛道中希望呈现的专业身份。你选择什么赛道、准备输出什么方向的内容，就需要让别人知道你提供的内容具备专业性。

![](img/36ed2937e3a679473f5b4e9823049005_60.png)

![](img/36ed2937e3a679473f5b4e9823049005_62.png)

![](img/36ed2937e3a679473f5b4e9823049005_64.png)

以下是不同赛道的角色定位示例：

![](img/36ed2937e3a679473f5b4e9823049005_66.png)

![](img/36ed2937e3a679473f5b4e9823049005_68.png)

![](img/36ed2937e3a679473f5b4e9823049005_70.png)

![](img/36ed2937e3a679473f5b4e9823049005_72.png)

![](img/36ed2937e3a679473f5b4e9823049005_74.png)

*   **好物分享/测评博主**：你需要以亲身使用者的身份，提供真实、客观的体验报告。
*   **装修博主**：你可以作为正在装修或已装修完成的房主，记录整个过程，分享遇到的坑和最终成果，通过前后对比展示你的专业选择。
*   **母婴赛道**：最常见的角色是“宝妈”。但根据内容方向，你也可以定位为“育儿护理专家”，这个身份不一定要求自身是宝妈，可以是相关领域的专业人员。
*   **情感赛道**：你可以是“情感顾问”或“心理学导师”，分享关于情感、成长、励志的话题。此外，还可以打造“成长型人设”，例如分享一位30岁女性如何应对家庭、职场问题，以及创业失败后如何走出来的真实经历。

![](img/36ed2937e3a679473f5b4e9823049005_76.png)

![](img/36ed2937e3a679473f5b4e9823049005_78.png)

![](img/36ed2937e3a679473f5b4e9823049005_80.png)

![](img/36ed2937e3a679473f5b4e9823049005_82.png)

### 2. 内容：保持垂直与价值

![](img/36ed2937e3a679473f5b4e9823049005_102.png)

![](img/36ed2937e3a679473f5b4e9823049005_104.png)

![](img/36ed2937e3a679473f5b4e9823049005_106.png)

![](img/36ed2937e3a679473f5b4e9823049005_108.png)

关于内容定位，我们之前课程已经强调过几个关键点：内容要吸引用户、要有价值、要在平台上有特色。在人设定位的语境下，内容还需要特别注意垂直度。

![](img/36ed2937e3a679473f5b4e9823049005_110.png)

![](img/36ed2937e3a679473f5b4e9823049005_112.png)

![](img/36ed2937e3a679473f5b4e9823049005_114.png)

![](img/36ed2937e3a679473f5b4e9823049005_116.png)

![](img/36ed2937e3a679473f5b4e9823049005_118.png)

你必须持续输出与你所选赛道强相关的内容。这涉及到账号的垂直度和标签化问题。我们一直强调要做垂直领域，因为这是打牢账号标签、影响账号成长和后期流量分发的关键。

![](img/36ed2937e3a679473f5b4e9823049005_120.png)

![](img/36ed2937e3a679473f5b4e9823049005_122.png)

![](img/36ed2937e3a679473f5b4e9823049005_124.png)

![](img/36ed2937e3a679473f5b4e9823049005_126.png)

![](img/36ed2937e3a679473f5b4e9823049005_128.png)

试想，如果你的第一篇内容是关于儿童教育，第二篇却变成了宠物分享。那么，之前因为教育内容而关注你的用户，很可能对宠物内容不感兴趣并划走。这个行为会向平台反馈“用户不喜欢此内容”，从而影响视频的推荐效果。

### 3. 形式：确定内容的呈现方式

![](img/36ed2937e3a679473f5b4e9823049005_150.png)

![](img/36ed2937e3a679473f5b4e9823049005_152.png)

![](img/36ed2937e3a679473f5b4e9823049005_154.png)

![](img/36ed2937e3a679473f5b4e9823049005_156.png)

形式指的是你通过何种方式将内容呈现给用户。主要包括以下几个方面，你需要提前规划：

![](img/36ed2937e3a679473f5b4e9823049005_158.png)

![](img/36ed2937e3a679473f5b4e9823049005_160.png)

![](img/36ed2937e3a679473f5b4e9823049005_162.png)

![](img/36ed2937e3a679473f5b4e9823049005_164.png)

![](img/36ed2937e3a679473f5b4e9823049005_166.png)

*   **内容产出场景**：你的视频主要在什么场景下拍摄？是固定的室内场景，还是多变的室外场景？例如，固定在车内分享、在上班路上记录等。你的拍摄场景可以参照对标账号的呈现方式。
*   **声音呈现**：你是自己出镜讲解，还是不想出镜，仅通过后期配音来完成？现在很多账号采用真人出镜搭配配音的形式，技术上用“剪映”等工具就能轻松实现。你需要评估自己的表达能力和声音条件。
*   **内容形式**：你的内容主要以哪种形式呈现？常见的有：
    *   **剧情型**：创作难度较高，需要剧本和表演。
    *   **干货/经验分享型**：需要准备丰富的文案和脚本。
    *   **生活记录型**：需要积累大量的日常拍摄素材。

![](img/36ed2937e3a679473f5b4e9823049005_168.png)

![](img/36ed2937e3a679473f5b4e9823049005_170.png)

![](img/36ed2937e3a679473f5b4e9823049005_172.png)

![](img/36ed2937e3a679473f5b4e9823049005_174.png)

每种形式各有优劣势，你需要根据自身情况选择。

![](img/36ed2937e3a679473f5b4e9823049005_176.png)

![](img/36ed2937e3a679473f5b4e9823049005_178.png)

### 4. 记忆点：让用户牢牢记住你

![](img/36ed2937e3a679473f5b4e9823049005_186.png)

![](img/36ed2937e3a679473f5b4e9823049005_188.png)

![](img/36ed2937e3a679473f5b4e9823049005_190.png)

记忆点是让人设脱颖而出的关键。你需要设计一些固定元素，反复植入内容，加深用户在观众心中的印象。

![](img/36ed2937e3a679473f5b4e9823049005_192.png)

![](img/36ed2937e3a679473f5b4e9823049005_194.png)

![](img/36ed2937e3a679473f5b4e9823049005_196.png)

![](img/36ed2937e3a679473f5b4e9823049005_198.png)

![](img/36ed2937e3a679473f5b4e9823049005_200.png)

以下是常见的记忆点设计，可分为两类：

![](img/36ed2937e3a679473f5b4e9823049005_202.png)

![](img/36ed2937e3a679473f5b4e9823049005_204.png)

![](img/36ed2937e3a679473f5b4e9823049005_206.png)

**A. 前置型记忆点（在视频开头呈现）**
这类记忆点能在用户刷到视频的瞬间形成强烈冲击，加深印象。
*   **固定动作**：视频前几秒的招牌动作。
*   **固定特效**：统一的开场特效。
*   **固定开场白**：每期视频使用的统一开场语。

![](img/36ed2937e3a679473f5b4e9823049005_208.png)

![](img/36ed2937e3a679473f5b4e9823049005_210.png)

![](img/36ed2937e3a679473f5b4e9823049005_212.png)

![](img/36ed2937e3a679473f5b4e9823049005_214.png)

![](img/36ed2937e3a679473f5b4e9823049005_216.png)

**B. 贯穿型记忆点（在视频全程或多次出现）**
这类记忆点能在用户观看过程中持续强化认知。
*   **统一BGM**：使用标志性的背景音乐，音乐响起就能联想到你。
*   **标志性物品**：视频中反复出现的特定物品，如特定的台灯、摆设等。
*   **统一风格服装**：符合你人设定位的、具有辨识度的着装风格。

![](img/36ed2937e3a679473f5b4e9823049005_218.png)

![](img/36ed2937e3a679473f5b4e9823049005_220.png)

![](img/36ed2937e3a679473f5b4e9823049005_222.png)

![](img/36ed2937e3a679473f5b4e9823049005_224.png)

![](img/36ed2937e3a679473f5b4e9823049005_226.png)

**设计逻辑**：将具有冲击力的元素（动作、特效、开场白）放在视频开头；将能营造氛围和风格的元素（BGM、物品、服装）贯穿视频始终。

![](img/36ed2937e3a679473f5b4e9823049005_228.png)

![](img/36ed2937e3a679473f5b4e9823049005_230.png)

![](img/36ed2937e3a679473f5b4e9823049005_232.png)

本节课我们一起学习了人设定位的四大核心因素：**角色、内容、形式和记忆点**。

![](img/36ed2937e3a679473f5b4e9823049005_244.png)

![](img/36ed2937e3a679473f5b4e9823049005_246.png)

![](img/36ed2937e3a679473f5b4e9823049005_250.png)

![](img/36ed2937e3a679473f5b4e9823049005_252.png)

1.  **角色**是你在赛道中的专业身份。
2.  **内容**需要保持垂直，提供持续价值。
3.  **形式**包括场景、声音和内容类型，需要提前规划。
4.  **记忆点**需要通过设计固定的前置元素和贯穿元素来强化用户印象。

![](img/36ed2937e3a679473f5b4e9823049005_254.png)

![](img/36ed2937e3a679473f5b4e9823049005_256.png)

掌握这四个因素，并将其系统性地融入你的账号运营中，就能构建出一个清晰、立体且令人难忘的IP人设。下一节课，我们将继续探讨人设定位的配套设置，例如账号昵称、头像、简介等如何与人设配合。
