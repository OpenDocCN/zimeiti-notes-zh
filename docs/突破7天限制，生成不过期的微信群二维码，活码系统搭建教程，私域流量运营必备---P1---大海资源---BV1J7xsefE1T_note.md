# 私域引流宝搭建教程：P1：系统原理与服务器环境搭建

在本节课中，我们将学习如何搭建一个名为“私域引流宝”的开源活码系统。该系统能有效解决微信群二维码7天过期的问题，并具备客服码、渠道码等高级功能，是私域流量运营的得力工具。我们将从系统原理讲起，并完成从购买服务器到安装系统环境的全过程。

## 系统核心原理

上一节我们介绍了课程目标，本节中我们来看看私域引流宝是如何解决微信群二维码过期问题的。

### 活码的基本原理

微信群二维码本身有7天有效期限制。活码系统通过在用户和真实的微信群二维码之间增加一个中间层来解决此问题。

1.  **生成系统活码**：用户扫描的并非直接的微信群二维码，而是由系统生成的一个永久有效的活码。
2.  **关联真实二维码**：在系统后台，管理员可以将这个系统活码与一个真实的、7天有效的微信群二维码关联起来。
3.  **动态替换**：当后台关联的微信群二维码即将过期时，管理员只需在后台将其替换为新的二维码。用户扫描的始终是同一个系统活码，因此可以持续访问到有效的微信群。

**核心流程公式**：
```
用户扫描 -> 系统活码（永久有效） -> 后台关联的真实微信群二维码（可动态替换）
```

### 防封禁的多域名策略

![](img/98f2919b324a8ce06245f2e5cd914988_1.png)

![](img/98f2919b324a8ce06245f2e5cd914988_2.png)

单纯的活码仍可能因落地页被举报而失效。私域引流宝采用了更巧妙的三域名架构来降低风险。

以下是系统后台配置中涉及的三种域名：

![](img/98f2919b324a8ce06245f2e5cd914988_4.png)

*   **短链域名**：用户实际扫描的二维码所对应的域名。
*   **入口域名**：系统内部用于跳转的中间域名。
*   **落地域名**：最终展示微信群二维码的页面所在的域名。

![](img/98f2919b324a8ce06245f2e5cd914988_6.png)

**防封禁流程**：
当用户举报时，通常举报的是**落地域名**。即使该域名被封禁，**短链域名**和**入口域名**依然有效。管理员只需在后台更换一个新的**落地域名**，整个活码即可恢复使用，大大提升了系统的稳定性。

**流程代码描述**：
```python
# 用户访问流程
用户访问(短链域名) -> 系统跳转(入口域名) -> 最终展示(落地域名)

# 封禁处理流程
落地域名被封 -> 后台更换新落地域名 -> 活码恢复可用
```

### 其他高级功能简介

除了核心的活码功能，该系统还包含以下实用功能：

*   **客服码**：对外只推广一个统一的客服码，系统可轮询分配访客给多个后台客服，提升接待效率。
*   **渠道码**：通过一个活码，系统可以记录扫码用户的设备、IP等信息，用于渠道分析和用户画像。

## 服务器准备与注册

理解了系统原理后，我们需要一个服务器来承载它。本节我们将使用亚马逊云（AWS）的免费套餐来获取一台12个月免费的云服务器。

### 注册亚马逊云账户

首先，访问亚马逊云官网并开始注册。

以下是注册账户的关键步骤：

1.  使用电子邮箱地址创建新账户。
2.  验证电子邮箱，输入收到的验证码。
3.  设置一个符合复杂度要求的账户密码。
4.  填写个人联系信息，地址需使用拼音填写。
5.  绑定信用卡以完成身份验证（会有一笔小于1美元的临时预授权，不会实际扣费）。
6.  通过短信验证身份，输入收到的PIN码。
7.  选择“基本支持（免费）”计划，完成注册。

### 启动免费EC2实例

账户注册并激活后，登录控制台，开始创建服务器实例。

以下是创建EC2实例的步骤：

1.  在控制台找到并进入“EC2”服务。
2.  将页面右上角的地区更改为离你较近的区域，例如“亚太地区（首尔）”。
3.  点击“启动实例”。
4.  为实例命名，例如“dahai”。
5.  选择操作系统镜像。教程中使用的是“Debian”系统。
6.  在“密钥对”部分，创建新的密钥对，命名为“dahai”，密钥格式选择“.ppk”（用于PuTTY连接），下载并妥善保存密钥文件。
7.  在网络设置中，配置安全组，放行SSH（22端口）、HTTP（80端口）和HTTPS（443端口）的流量。
8.  配置存储大小，免费套餐为30GB，此处设置为29GB以避免超额。
9.  检查所有配置，点击“启动实例”。

## 连接服务器与安装面板

![](img/98f2919b324a8ce06245f2e5cd914988_8.png)

![](img/98f2919b324a8ce06245f2e5cd914988_10.png)

服务器创建成功后，我们需要连接到它并安装一个可视化的管理面板。

### 使用PuTTY连接服务器

我们需要使用PuTTY这个工具通过SSH协议连接服务器。

1.  下载并安装PuTTY客户端。
2.  打开PuTTY，在“Session”页面输入服务器的公有IP地址。
3.  在“Connection -> Data”中，设置登录用户名（对于Debian系统，默认为`admin`）。
4.  在“Connection -> SSH -> Auth”中，加载之前下载的.ppk格式密钥文件。
5.  返回“Session”，为此连接设置一个名称（如“dahai”）并保存。
6.  双击保存的连接，首次连接时点击“Accept”接受密钥，即可成功登录服务器。

### 安装宝塔面板

成功连接服务器后，我们通过命令安装宝塔面板，以便后续通过网页轻松管理服务器和网站。

1.  在PuTTY终端中，切换到root用户。输入命令 `sudo su`，然后为root用户设置一个新密码（命令：`passwd root`）。
2.  执行宝塔面板针对Debian系统的安装命令。安装脚本可从宝塔官网获取。
3.  安装完成后，命令行会显示面板的访问地址、用户名和密码。请复制保存这些信息。
4.  回到AWS控制台，为实例的安全组添加规则，放行宝塔面板的默认端口（通常是77892，请以安装完成时显示的端口为准）和数据库端口（3306）。
5.  在浏览器中访问上一步获取的宝塔面板外网地址，使用用户名和密码登录。

## 配置网站环境

登录宝塔面板后，我们需要安装运行网站所必需的软件环境。

### 安装系统组件

为了提高服务器性能，我们首先安装“Linux工具箱”插件，并为其添加虚拟内存（Swap）。

![](img/98f2919b324a8ce06245f2e5cd914988_12.png)

![](img/98f2919b324a8ce06245f2e5cd914988_14.png)

1.  在宝塔面板的“软件商店”中搜索并安装“Linux工具箱”。
2.  安装后，进入其设置页面，添加2048MB的虚拟内存。

![](img/98f2919b324a8ce06245f2e5cd914988_16.png)

![](img/98f2919b324a8ce06245f2e5cd914988_18.png)

### 安装Web运行环境

接下来，安装Nginx、MySQL和PHP。

![](img/98f2919b324a8ce06245f2e5cd914988_20.png)

以下是需要安装的软件及其版本：

*   **Nginx**：选择最新稳定版，使用“极速安装”。
*   **MySQL**：选择5.7或更高版本，使用“极速安装”。
*   **PHP**：选择7.4版本（私域引流宝要求PHP 7.0-7.4），使用“极速安装”。

等待所有软件安装完毕。

## 域名解析与网站创建

![](img/98f2919b324a8ce06245f2e5cd914988_22.png)

为了更规范地访问我们的系统，我们使用域名来创建网站。

### 域名解析

如果你有一个域名（教程中以Gname为例），需要将其解析到你的服务器IP。

![](img/98f2919b324a8ce06245f2e5cd914988_24.png)

![](img/98f2919b324a8ce06245f2e5cd914988_25.png)

1.  登录你的域名注册商管理后台。
2.  添加两条A记录解析：
    *   记录类型：A，主机记录：`@`，记录值：你的服务器公网IP。
    *   记录类型：A，主机记录：`www`，记录值：你的服务器公网IP。

### 在宝塔面板创建网站

![](img/98f2919b324a8ce06245f2e5cd914988_27.png)

域名解析生效后（通常几分钟），在宝塔面板中创建网站。

![](img/98f2919b324a8ce06245f2e5cd914988_29.png)

1.  在宝塔面板“网站”页面，点击“添加站点”。
2.  输入你的域名（可以带www或不带）。
3.  同时创建一个MySQL数据库，记录下数据库名、用户名和密码。
4.  点击“提交”，网站即创建成功。

![](img/98f2919b324a8ce06245f2e5cd914988_31.png)

## 部署私域引流宝程序

![](img/98f2919b324a8ce06245f2e5cd914988_33.png)

最后一步，我们将私域引流宝的源代码上传并安装到刚刚创建的网站中。

### 上传程序文件

1.  进入宝塔面板中该网站的“根目录”。
2.  删除目录下默认生成的文件。
3.  从私域引流宝的GitHub发布页面下载最新版的程序压缩包。
4.  在网站根目录点击“上传”，将压缩包上传至服务器。
5.  上传完成后，解压压缩包，并将解压出的所有文件移动到网站根目录。

### 安装向导

文件部署完成后，通过浏览器访问你的域名，开始安装。

1.  访问你的域名，将自动跳转到安装页面。
2.  点击“安装”。
3.  环境检测通过后，点击“全新安装”。
4.  在数据库配置页面，填写之前创建数据库时记录的信息：
    *   数据库地址：`localhost`
    *   数据库名、用户名、密码：填写宝塔面板创建时生成的。
5.  设置管理员账号、邮箱和密码。
6.  点击“开始安装”，等待安装完成。
7.  安装成功后，点击“前往管理后台”，使用刚才设置的管理员账号密码登录。
