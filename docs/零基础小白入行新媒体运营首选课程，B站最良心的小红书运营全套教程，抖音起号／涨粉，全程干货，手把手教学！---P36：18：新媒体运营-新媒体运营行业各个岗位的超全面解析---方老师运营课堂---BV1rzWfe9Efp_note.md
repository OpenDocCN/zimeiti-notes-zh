## 内容运营 📝

内容运营是新媒体运营领域最基础的岗位之一。其主要职责围绕内容的策划、编辑与发布展开。

具体工作内容包括撰写文章、剪辑视频等。完成内容制作后，将其发布到各个平台。这些内容不一定需要完全原创，也可能涉及对现有内容的搜集、整合与二次创作。这是一个入门级的运营岗位。

![](img/eb55d267cc338f9e81e85c8e48237e90_14.png)

![](img/eb55d267cc338f9e81e85c8e48237e90_16.png)

该岗位有时也被称为“新媒体运营专员”。以下是内容运营的核心工作内容：

*   **内容策划**：规划要生产什么类型的内容。
*   **内容编辑**：进行文字撰写、视频剪辑等具体制作。
*   **内容发布**：将成品发布到指定的新媒体平台。

![](img/eb55d267cc338f9e81e85c8e48237e90_18.png)

这个岗位的薪资通常不会太高，工作逻辑相对简单。只要掌握一些基础的运营思路，例如关键词布局，即可胜任。即使是零基础的应届生，有人带领学习一到两周也能快速上手。

![](img/eb55d267cc338f9e81e85c8e48237e90_20.png)

## 短视频运营 🎬

短视频运营是一个更为复杂、深入且核心的岗位。它通常由一个团队协作完成，岗位构成如下：

*   **策划/运营**：整体负责人，确定选题和方向。
*   **编导**：根据策划的主题，设计具体的拍摄脚本和分镜。
*   **拍摄**：负责现场选景、镜头设计、设备调试等拍摄工作。
*   **演员**：视频内容的出镜表演者（多为公司内部人员兼任）。

![](img/eb55d267cc338f9e81e85c8e48237e90_22.png)

![](img/eb55d267cc338f9e81e85c8e48237e90_24.png)

其中，运营岗位的工作最为综合和关键。以下是短视频运营的核心工作内容：

![](img/eb55d267cc338f9e81e85c8e48237e90_26.png)

![](img/eb55d267cc338f9e81e85c8e48237e90_28.png)

![](img/eb55d267cc338f9e81e85c8e48237e90_30.png)

1.  **分析数据，确定选题**：通过数据分析，决定拍摄什么内容能吸引目标用户或成为热点。公式可以表示为：**热门选题 = 数据分析(用户偏好 + 平台趋势)**。
2.  **追踪热点**：培养“网感”，密切关注互联网和行业动态，将热点事件与自身产品结合，创作相关内容。例如，每日查看 `百度热搜`、`知乎热榜`、`抖音热榜` 等平台。
3.  **数据优化**：对视频的最终效果（如点赞、评论、涨粉数）负责，通过优化各项数据指标来完成绩效目标（如月涨粉1000）。
4.  **封面与标题规划**：设计和优化视频封面与标题，这是提升点击率的关键。运营需要思考如何通过封面和标题吸引用户点击。
5.  **负责内容制作全流程**：涉及视频脚本、布景选景、内容选题、视频创作及特效剪辑等环节。

![](img/eb55d267cc338f9e81e85c8e48237e90_32.png)

## 用户运营 👥

用户运营专注于将公域平台的用户引流至私域，并进行维护和转化，核心目标是提升用户活跃度与价值。

以下是用户运营的两个主要方向：

![](img/eb55d267cc338f9e81e85c8e48237e90_34.png)

*   **社群运营**：维护粉丝群或业务群，通过活动、内容等方式保持群内用户的活跃度，确保商业信息能够有效触达。例如抖音、小红书的官方粉丝群运营。
*   **私域运营**：涉及用户成交体系，通常与销售部门协同。工作包括将用户引流至微信等私域，并通过活动分发、一对一沟通等方式促进消费。其核心是 **私域引流** 与 **销售转化** 的闭环。

## 直播运营 📺

![](img/eb55d267cc338f9e81e85c8e48237e90_36.png)

![](img/eb55d267cc338f9e81e85c8e48237e90_38.png)

直播运营是一个体系化的部门，包含多个细分岗位，协同完成一场直播。

![](img/eb55d267cc338f9e81e85c8e48237e90_40.png)

以下是直播团队常见的岗位构成：

![](img/eb55d267cc338f9e81e85c8e48237e90_42.png)

*   **总导演**：直播流程的总负责人。
*   **主播与副播**：主播负责核心讲解，副播协助或顶替。
*   **主播助手**：在幕后配合主播，响应其需求。
*   **投手**：负责广告投放（如千川、抖加），控制广告开支与投放效果，核心目标是追求高 **ROI（投资回报率）**。公式为：**ROI = 销售额 / 广告消耗**。
*   **上架专员**：负责直播中产品的上架、库存修改、卖点信息更新等。
*   **直播场控**：监控直播间氛围，管理评论等。
*   **剪辑**：实时或后期剪辑直播中的精彩片段，用于二次传播。

![](img/eb55d267cc338f9e81e85c8e48237e90_44.png)

![](img/eb55d267cc338f9e81e85c8e48237e90_46.png)

## 电商运营 🛒

![](img/eb55d267cc338f9e81e85c8e48237e90_48.png)

电商运营在传统电商平台（如淘宝、京东）中扮演总控角色，负责店铺的整体运营。

![](img/eb55d267cc338f9e81e85c8e48237e90_50.png)

其工作内容繁杂，需要多年经验才能精通，主要包括：
*   店铺销售运营与策略制定。
*   店铺评分管理与风险控制。
*   节假日活动策划。
*   店铺广告投放（如直通车）。
*   发货、物流等供应链环节管控。

![](img/eb55d267cc338f9e81e85c8e48237e90_52.png)

## 广告投放（投手） 🎯

![](img/eb55d267cc338f9e81e85c8e48237e90_54.png)

![](img/eb55d267cc338f9e81e85c8e48237e90_56.png)

广告投放专员，也称投手或广告优化师，专门负责在各平台进行付费推广。

![](img/eb55d267cc338f9e81e85c8e48237e90_58.png)

![](img/eb55d267cc338f9e81e85c8e48237e90_60.png)

投手需要精通不同平台的广告投放工具与逻辑，例如：
*   **抖音**：需熟悉千川、抖加、小店随心推等工具。
*   **B站**：需了解B站信息流广告和“起飞”推广。
*   **小红书**：主要是小红书信息流广告。

![](img/eb55d267cc338f9e81e85c8e48237e90_62.png)

![](img/eb55d267cc338f9e81e85c8e48237e90_64.png)

此外，必须理解基本的广告计费模式，如：
*   **CPC (Cost Per Click)**：按点击计费。
*   **CPM (Cost Per Mille)**：按千次展示计费。
*   **CPA (Cost Per Action)**：按行动（如下载、注册）计费。

![](img/eb55d267cc338f9e81e85c8e48237e90_66.png)
