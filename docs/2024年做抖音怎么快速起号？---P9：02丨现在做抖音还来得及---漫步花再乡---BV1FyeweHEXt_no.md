## 市场机会分析

上一节我们介绍了课程的整体框架，本节中我们来看看当前抖音市场的机会。很多人担心各行业已有大量头部账号，自己难以立足。他们认为头部账号赚得盆满钵满，自己可能连汤都喝不到。

实际上，这种担忧大可不必。短视频的爆发期仅三年左右，市场规模达数百亿，这还不足以垄断任何一个行业。

![](img/e28eb1c9715a1de72aa0cc34c4304fbc_3.png)

**公式：市场饱和度 = 现有竞争者数量 / 市场总规模**
当前市场总规模巨大，该值远小于1，意味着新进入者仍有广阔空间。

## 个人IP的价值

而且，个人IP一旦建立起来，你就会拥有一大批无条件信任你的用户。从此，对你而言，天下没有难做的生意。对于你的用户来说，你可能就是这个行业里，他唯一接触到的一个专业的老师，或者说最适合的老师。

![](img/e28eb1c9715a1de72aa0cc34c4304fbc_5.png)

有时你的用户并不是因为你最牛而选择你，而是因为他信任你，且“跳一跳就能够得着”才选择你。

以下是关于用户选择的逻辑：
*   **信任感**：用户因为认可你的专业性和真诚而选择你。
*   **可接近性**：用户觉得你的水平或产品是他通过努力可以触及的，而非高不可攀。

## 如何理解目标用户

![](img/e28eb1c9715a1de72aa0cc34c4304fbc_7.png)

怎么理解“跳一跳就能够得着”呢？举个例子，假如你要填报高考志愿，你会咨询以下哪个学长或学姐？
1.  清华北大的学霸
2.  985/211院校的优秀学生
3.  普通一本院校的学长
4.  二本院校的过来人

我相信答案一定是越往下人数越多。因为对于“学校”的需求规模就是越往下走，人数越多。一个平时考试最多上二本的人，有可能去问清华北大的学生吗？就算去问，对方也可能不了解二本院校的情况，毕竟圈层差距太大。

所以，不用担心自己没有用户。你总是能找到属于自己的那部分用户。

![](img/e28eb1c9715a1de72aa0cc34c4304fbc_9.png)

## 获取精准流量的核心

第二个问题，如何获取精准流量？现在流量成本越来越高，无论是之前的图文时代还是现在的短视频时代，流量几乎成为了商业的核心。但无论任何时代，投产比最高的流量渠道依然是**内容**，尤其是**垂直领域**的内容。

无论你是卖什么产品的，垂直内容都是沉淀付费用户的最佳路径。

以下是操作示例：
```python
# 假设你是卖脐橙的
content_topic = “展示新鲜脐橙”
action = “拍摄刚摘下的脐橙，一刀切下去展示汁多饱满”
goal = “让观众一眼就知道你是干什么的，能提供什么价值”
result = “时间长了，一定会有人为你买单”
```

![](img/e28eb1c9715a1de72aa0cc34c4304fbc_11.png)

当然，你也得保持品质。万古不变的套路就是**真诚**，你的真诚往往会在你的内容里流露出来。

## 创作什么样的内容

第三个问题，要做什么样的内容？我认为好内容的核心在于**利他**，尤其是“成人达己”的内容，人人都爱。

就拿我来举例，我是教大家如何做好短视频的。我把内容分为了三个板块：
1.  **爆款模仿**：帮大家了解爆款账号背后的逻辑，把其中可借鉴复制的点拿出来为我们自己所用。
2.  **运营技巧**：通过我的经验，系统完整地分享运营账号的逻辑，包括一些可能需要规避的坑。
3.  **项目拆解**：拆解一些当下热门的项目，比如“中视频计划”、“抖音小店”的玩法等。

![](img/e28eb1c9715a1de72aa0cc34c4304fbc_13.png)

内容时长一般在3分钟以上。这些内容对于新手来说一定是有价值的，不像30秒的视频那么信息稀碎。

另外，最重要的是，做这件事对我自身也有很大的帮助。在给你们拆解分析的时候，也是我自我学习迭代的一个过程。只不过我把这个过程和结果记录了下来，分享给了你们，让大家和我一起进步。

这就是“成人达己”的好内容。就算最后没有留下一个粉丝，可拥有这些知识，本身就是巨大的收获。
