# 视频号剪辑教程：P35：21-日销34万营业额小王剪辑法_1

在本节课中，我们将学习一种高效的手机剪辑方法，用于替换视频中的广告产品。这种方法的核心在于通过“画中画”和关键帧功能，精准覆盖原视频中的广告，并替换为自己的产品。我们将从视频的基础处理开始，逐步讲解如何定位、遮挡和替换广告，最终完成一个可用于视频号发布的成品。

## 视频基础处理

上一节我们介绍了课程概述，本节中我们来看看视频剪辑的第一步：基础处理。基础处理包括调整画面、裁剪无用片段和调整播放速度，目的是为后续的广告替换做好准备。

以下是基础处理的三个核心步骤：

1.  **放大画面**：将导入的视频画面略微放大。这有助于在后续步骤中更精确地进行裁剪和遮挡操作。操作公式可理解为：`画面比例 = 原始比例 * 1.0X` （轻微放大）。
2.  **掐头去尾**：裁剪掉视频开头和结尾不必要的部分，使内容更紧凑。这一步与第一步的顺序可以互换。
3.  **调整变速**：将视频速度调整为 **1.1倍**。轻微的加速可以使视频节奏更快，更符合短视频平台的观看习惯。

完成以上三步后，视频的主体框架便处理完毕。

## 定位原广告位置

在开始替换广告前，我们必须先准确找到原视频中广告出现的位置和时间点。

原视频的广告剧情通常来源于直播广场等渠道。我们需要仔细观察原视频，识别出广告产品的展示部分。例如，原视频可能在推销“沙琪玛”，而我们需要将其替换为“榨菜”。

定位广告的方法是：仔细观看原视频，记录下产品首次出现、特写展示和口播介绍的精确时间点。如果不确定，可以用手机录屏功能反复观看确认。

## 遮挡与裁剪原广告

![](img/47ff4658704a394b0cbc57a1cc3910dc_1.png)

找到原广告后，下一步是将其从视频中移除。我们通过分割、放大画面并裁剪来实现。

![](img/47ff4658704a394b0cbc57a1cc3910dc_3.png)

首先，在广告开始的位置使用**分割**功能，将广告片段独立出来。
接着，将需要移除的广告片段画面**放大**，直到原产品图像移出画面可视区域，然后裁剪掉放大后的多余部分。这个过程可能需要反复调整，确保原广告产品完全被裁剪掉，不留下任何痕迹。

![](img/47ff4658704a394b0cbc57a1cc3910dc_5.png)

例如，原视频中人物在介绍“沙琪玛”的部分，我们需要通过放大画面，将“沙琪玛”移出画面后，再裁剪掉该时间段内的画面。

## 添加新广告（画中画与关键帧）

清除原广告后，现在添加我们自己的产品广告。这里主要使用“画中画”和“关键帧”功能。

![](img/47ff4658704a394b0cbc57a1cc3910dc_7.png)

以下是添加新广告的具体步骤：

![](img/47ff4658704a394b0cbc57a1cc3910dc_9.png)

1.  **导入产品素材**：提前准备好产品（如榨菜）的图片或视频素材。通过“画中画”功能导入这些素材。
2.  **定位与缩放**：将产品素材移动到原广告产品出现的位置，并调整其大小，使其能完全覆盖住原产品。
3.  **使用关键帧实现动态跟随**：为了让静态图片能跟随视频画面移动，需要使用**关键帧**功能。在素材开始处添加一个关键帧，随着视频播放，移动产品素材的位置，系统会自动在移动后的位置生成新的关键帧，从而实现素材的动态跟踪效果。
4.  **精细调整**：反复播放检查，确保自己的产品素材在每一帧都完全遮挡住原广告产品，没有“穿帮”镜头。

## 整体优化与导出

![](img/47ff4658704a394b0cbc57a1cc3910dc_11.png)

所有广告替换完成后，需要对视频进行整体优化，提升观感，然后导出成品。

以下是整体优化的几个方面：

![](img/47ff4658704a394b0cbc57a1cc3910dc_13.png)

*   **二次画面调整**：可以再次轻微放大或左右移动某些片段，增加画面的动态感和剪辑痕迹，这属于一种流行的剪辑风格。
*   **添加装饰元素**：
    *   **贴纸**：在视频四个角落添加半透明的贴纸，并降低其透明度，起到装饰和防搬运的作用。
    *   **画中画叠加**：可以添加一段自己的实拍素材（如家居场景），使用“混合模式”（如正片叠底）并将其透明度调至 **5%** 以内，覆盖全屏，增加视频的独特质感。
*   **应用滤镜与调节**：为视频添加一个合适的滤镜，并在“调节”选项中微调参数（如亮度、对比度、饱和度），所有数值建议控制在 **±5** 以内。
*   **设置比例与背景**：将视频比例设置为 **9:16**（竖屏），并选择纯色或模糊背景填充黑边，记得勾选“全局应用”。
*   **最终导出**：检查无误后，将视频导出，时长建议控制在 **1分30秒至1分40秒** 左右。

本节课中我们一起学习了“小王剪辑法”的核心流程：从视频的基础处理，到精准定位并裁剪原广告，再到利用画中画和关键帧技术无缝替换为自己的产品广告，最后进行整体的美化与导出。掌握这个方法，你就能高效地制作出适合视频号发布的带货视频。关键在于耐心和细致，尤其是在遮挡原广告和添加新广告的环节，需要反复检查以确保效果完美。
