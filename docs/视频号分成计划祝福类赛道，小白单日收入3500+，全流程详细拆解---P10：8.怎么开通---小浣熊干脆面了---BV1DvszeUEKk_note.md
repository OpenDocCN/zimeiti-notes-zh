## 案例背景与教学理念
上一节我们介绍了视频号分成计划的基本概念，本节中我们来看看如何具体操作开通。教学将以实操为主，通过展示真实的手机操作画面来进行讲解。

![](img/a5758c8d290d45fe407478e180866af6_1.png)

这位学员是张老师曾指导过的一名同学。在讲解案例和操作时，都会使用老师自己的手机界面进行演示，以确保真实性。

## 学员情况与针对性教学
这位学员年纪较大，自称没读过什么书，看不清很多字，并对互联网操作感到陌生。针对这种情况，老师调整了沟通和教学方式。

*   **沟通方式调整**：由于学员不识字，老师将文字沟通改为发送语音消息，以确保信息有效传达。
*   **年龄不是障碍**：老师指导过的学员中，年龄最大的有84岁。只要成年，都有机会学会。
*   **学习动机**：这位63岁的周叔想通过视频号赚钱养活自己，不拖累子女，同时满足个人爱好（如抽烟喝酒）的开销，保持经济独立和尊严。

## 过往教训与正确选择
周叔曾花费约2万元报名多个抖音课程，但最终一无所获。这个经历让老师感到气愤。老师指出，抖音的红利期已过（如17、18年），目前并不适合新手小白入场。相比之下，视频号是更值得尝试的新平台。

## 开通视频号全流程拆解
在明确了学员想学习“保底解析”玩法后，老师首先带领他注册视频号。以下是详细的注册步骤。

老师为学员提供了一份清晰的注册流程表。

![](img/a5758c8d290d45fe407478e180866af6_4.png)

### 注册流程步骤
以下是注册视频号的具体步骤：

1.  **打开微信**：在手机主屏幕找到并点击微信图标。
2.  **进入“发现”页**：点击微信底部的“发现”选项卡。
3.  **找到“视频号”入口**：在“发现”页面，找到并点击“视频号”。
4.  **创建视频号**：点击页面右上角的“人像”图标或“创建视频号”按钮，按提示完成身份验证和信息填写。

### 针对特殊情况的解决方案
由于周叔不识字，看不懂流程表上的文字说明，老师采用了更直观的方法：

**解决方案**：在流程表截图上用箭头明确标注每一步需要点击的位置。教学口诀是：“**箭头指哪里，你就点哪里**”。通过这种“指哪打哪”的笨办法，确保学员能够完成操作。

这个案例证明，**没有学不会的学生，只有不会教的老师**。只要教学方法得当，任何人都能学会。
