# 自媒体入门指南：P9：2.自媒体平台现状 📊

在本节课中，我们将要学习自媒体行业的现状，分析其可行性与发展前景，并了解国家政策与平台支持，帮助你判断是否值得进入这个领域。

上一节我们介绍了自媒体的基本概念，本节中我们来看看这个行业是否值得投入。

## 行业可行性分析

我们了解了什么是自媒体，接下来需要分析这个行业是否可以做，以及它能带来什么好处。在此之前，很多同学可能或多或少都有所了解。

许多人可能想找兼职，或者考虑跨行业转换工作。在这个过程中，大家都会有疑问：这个行业到底能不能做？近几年由于疫情原因，很多上班族没有赚到钱，因此都想寻找副业。

但需要警惕的是，很多人抱有暴富心态，想一夜之间赚很多钱，从而采取了一些不合法的手段。例如，前往缅甸等地最终受害的新闻，大家应该有所了解。这类行业是国家明令禁止的、不合法的，对我们没有任何好处，甚至危害健康。

现阶段，自媒体行业是属于国家支持的。国家已经将支持文件下达到了各个地方政府。

## 平台支持与市场前景

![](img/a2f944e29eb5478743abd7a837813e89_7.png)

有句话说得好：跟着国家的脚步走，一定能赚到钱。在政策支持之外，平台也为我们提供了实质激励。

首先，我们通过一个视频了解平台支持。西瓜视频曾宣布，未来一年将额外拿出20亿元补贴中视频创作人。

![](img/a2f944e29eb5478743abd7a837813e89_7.png)

从这个视频中，我们可以获取两个核心信息：
1.  **20亿元是真金白银的纯补贴**，不包含任何商单、直播和电商收入。
2.  这20亿元是保底，上不封顶。

这说明了什么？它说明作为自媒体创作者，赚钱不只有“带货”一条路。通过制作视频或文章，依靠播放量同样可以变现。这20亿元，就是未来创作者可能赚到的收益。

除了平台补贴，自媒体行业本身拥有庞大的流量基础。根据相关报告，现阶段中国网民规模已达 **9.4亿人**，接近10亿。

看视频的群体覆盖极广，从小学生到成年人，各个年龄段的人都在刷视频。有看的人，就有做的人。那么，创作者有多少呢？

以下是自媒体从业者数据：
*   全职从业者：**370万**
*   兼职从业者：**600万**
*   总计：约 **970万**（可估算为1000万）

这意味着，大约每100个看视频的人中，只有1个人在制作视频。一个人制作视频的速度，远远赶不上大众消费视频的速度。因此，内容供给是**不足**的。

这也正是国家支持自媒体行业发展的原因之一。很多同学担心行业是否已经饱和。

根据自媒体发展趋势分析，在大数据时代背景下，自媒体行业**远远没有饱和**。

![](img/a2f944e29eb5478743abd7a837813e89_9.png)

“远远没有饱和”这个判断意味着，在未来十年内，你都可以将自媒体作为兼职或全职来发展，并以此赚钱。

![](img/a2f944e29eb5478743abd7a837813e89_9.png)

本节课中我们一起学习了自媒体行业的现状。我们了解到，该行业受到国家政策明确支持，各大平台也提供巨额补贴激励创作者。同时，庞大的用户基数和相对较少的创作者数量表明，市场远未饱和，发展前景广阔。这为我们进入自媒体领域提供了充分的信心和依据。
