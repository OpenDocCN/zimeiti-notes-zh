# 抖音短视频运营教程：P40：9.收益结算 💰

在本节课中，我们将详细学习影视推广类短视频的收益结算规则。我们将了解结算周期、计算方式、注意事项，并通过一个实际案例来具体说明收益是如何产生的。

![](img/afff9c856c6bdfda0732c7b3e069bc75_1.png)

## 收益结算概述

![](img/afff9c856c6bdfda0732c7b3e069bc75_3.png)

影视推广的收益结算按月进行，每月月底结算一次。收益主要依据视频的播放量来计算，但不同的推广任务（如不同的短剧）会有各自的具体要求。

![](img/afff9c856c6bdfda0732c7b3e069bc75_5.png)

## 结算规则详解

![](img/afff9c856c6bdfda0732c7b3e069bc75_7.png)

![](img/afff9c856c6bdfda0732c7b3e069bc75_9.png)

![](img/afff9c856c6bdfda0732c7b3e069bc75_11.png)

上一节我们介绍了影视推广的基本概念，本节中我们来看看具体的收益结算规则。

![](img/afff9c856c6bdfda0732c7b3e069bc75_13.png)

**核心结算公式**为：`收益 = 基础播放量收益 + 额外播放量奖励`。每个任务会设定一个基础播放量目标和对应的基础收益，超过目标的部分会按一定规则（如每万次播放奖励10元）追加奖励。

![](img/afff9c856c6bdfda0732c7b3e069bc75_15.png)

![](img/afff9c856c6bdfda0732c7b3e069bc75_17.png)

以下是结算过程中的几个关键点：

![](img/afff9c856c6bdfda0732c7b3e069bc75_19.png)

![](img/afff9c856c6bdfda0732c7b3e069bc75_21.png)

*   **结算周期**：收益按月结算，每月底进行一次。
*   **计算依据**：收益按照视频的有效播放量进行计算。
*   **任务要求**：每个短剧推广任务都有不同的具体要求，例如需要发布的视频数量、指定发布的平台等。必须严格按照要求执行，否则作品可能被下架，无法获得收益。
*   **双份收益可能**：除了指定的四个主要推广平台，如果你愿意为视频添加解说，还可以将作品发布到其他平台（如影视解说类平台），从而获得额外的版权收益。这意味着一个视频可以带来两份收入。

![](img/afff9c856c6bdfda0732c7b3e069bc75_23.png)

## 实际案例分析

![](img/afff9c856c6bdfda0732c7b3e069bc75_25.png)

![](img/afff9c856c6bdfda0732c7b3e069bc75_27.png)

为了让大家更直观地理解，我们来看一个实际案例。我的一位伙伴最初希望通过抖音积累粉丝，为日后带货做准备，因此选择了影视推广作为起点。

![](img/afff9c856c6bdfda0732c7b3e069bc75_29.png)

![](img/afff9c856c6bdfda0732c7b3e069bc75_31.png)

![](img/afff9c856c6bdfda0732c7b3e069bc75_33.png)

他首先完成了一个短剧推广任务。该任务要求发布三个视频。他发布的视频获得了186万的高播放量。这是因为短剧内容贴近现实，容易引发用户共鸣。

![](img/afff9c856c6bdfda0732c7b3e069bc75_35.png)

![](img/afff9c856c6bdfda0732c7b3e069bc75_37.png)

根据结算规则，他这次任务的基础收益约为500元。并且，规则注明后续每增加1万播放量，会额外奖励10元。

![](img/afff9c856c6bdfda0732c7b3e069bc75_39.png)

随后，他又尝试了另一个短剧任务。这个任务的视频播放量仅为1.4万，但因为它本身的话题度不同，平台给出的单价更高，所以仍有150元的收益。这说明**播放量高低并非决定收益的唯一因素，任务本身的单价也非常关键**。

![](img/afff9c856c6bdfda0732c7b3e069bc75_41.png)

在为期一周的时间里，他总共完成了四个短剧任务。到了月底结算时，收益明细如下：

![](img/afff9c856c6bdfda0732c7b3e069bc75_43.png)

1.  **第一个任务**：基础播放量收益500元 + 后续新增6万播放量的奖励60元 = **580元**。
2.  **第二个任务**：播放量一般，但单价高，结算 **150元**。
3.  **第三个任务**：因未按要求同时发布到快手平台，只发了抖音，收益被扣除一半，结算 **120元**。这强调了**严格遵守任务要求的重要性**。
4.  **第四个任务**：任务单价较低，结算 **45元**。

![](img/afff9c856c6bdfda0732c7b3e069bc75_45.png)

**一周总计收益：895元**。这个案例表明，投入时间越多，完成的任务越多，收益也就越可观。结算流程是：完成任务后，需填写“回填链接”提交作品ID，经审核通过后，收益会在月底统一结算。初期可能由个人转账，后期均会通过对公账户进行结算。

![](img/afff9c856c6bdfda0732c7b3e069bc75_47.png)

## 常见问题解答

![](img/afff9c856c6bdfda0732c7b3e069bc75_49.png)

许多同学可能会有疑问：为什么制片方不自己打广告，而要让我们来做推广呢？

这涉及到一个成本效益问题。制片方如果自行投放广告，费用极其高昂，单个广告的投入通常在10万至100万元以上。相比之下，将推广任务分发给我们海量的创作者，利用我们的力量进行扩散，其成本要低得多，而覆盖面和传播效果却可能更好。尤其是在当前环境下，线上推广的优势更加明显。因此，这对我们而言是一个很好的机会。

本节课中我们一起学习了影视推广的收益结算方式。我们明确了它是**按月结算、按播放量计算**的，并且**必须严格遵守每个任务的具体要求**（如发布平台、视频数量）。通过实际案例，我们看到了收益的构成（基础收益+额外奖励），也理解了遵守规则（如全平台发布）对保障收益至关重要。最后，我们从商业角度明白了这种推广模式对双方的价值所在。
