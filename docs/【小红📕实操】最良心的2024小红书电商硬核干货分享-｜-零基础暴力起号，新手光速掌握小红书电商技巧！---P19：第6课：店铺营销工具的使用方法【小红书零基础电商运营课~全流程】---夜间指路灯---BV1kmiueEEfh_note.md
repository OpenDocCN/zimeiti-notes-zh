# 小红书电商运营课：P19：店铺营销工具的使用方法

在本节课中，我们将学习小红书店铺后台营销工具的核心使用方法。掌握这些工具，可以帮助你为商品设置促销活动，吸引顾客下单，从而提升店铺销量。

上一节我们介绍了商品上架，本节中我们来看看如何利用营销工具为商品增添吸引力。

## 营销活动报名

首先，进入小红书店铺后台，点击“营销”板块。第一个功能是“营销活动”，其中的“活动报名”会展示店铺当前可以参加的官方活动。

以下是查看和报名活动的步骤：
1.  点击“可报名的活动”，系统会列出近期可参与的活动，例如“三八节”活动。
2.  点击具体活动进入详情页，查看报名条件，例如店铺评分需高于4分，或商品价格需满足特定要求。
3.  仔细阅读活动规则，例如“满300减40”。你需要评估自己是否能承担此折扣成本。
4.  决定是否报名。对于“双11”、“双12”等大型活动，建议报名以获取流量。对于日常小型活动，可将精力更多投入在笔记内容创作上。

## 商品促销设置

“营销管理”下的“商品促销”功能，是设置店铺日常优惠的核心。其中包含“单品促销”和“多品促销”，我们主要使用“多品促销”。

在多品促销中，最常用的两个工具是“满额立减”和“满件打折”。中间的几个工具基本可以忽略。

### 满额立减

“满额立减”是指顾客订单金额达到指定数额后，可立即减免一定金额。

以下是创建“满额立减”活动的步骤：
1.  点击“创建活动”，为活动起一个名称（例如“123”）。
2.  设置活动时间，建议持续时间在三个月以上。
3.  选择活动商品范围。通常选择“全部商品”进行全店促销，这样管理起来最方便。
4.  设置优惠条件。公式为：**订单金额 ≥ X元，立减Y元**。
    *   `X`：通常设置为店铺的平均客单价。例如，店铺商品多在50元以上，可设为50。
    *   `Y`：设置减免金额，例如5元。
5.  点击“完成”即可创建成功。如需取消，可随时暂停或删除该活动。

### 满件打折

“满件打折”是指顾客购买任意商品（或指定商品）即可享受折扣，更适合新店快速出单。

![](img/32c6d0f9fb6b556f1271aa5e60d8d0eb_1.png)

以下是创建“满件打折”活动的步骤：
1.  点击“创建活动”，设置活动名称与时间（同样建议三个月以上）。
2.  选择“全部商品”。
3.  设置优惠条件。核心规则是：**购买任意1件商品，即可享受Z折优惠**。
    *   折扣`Z`：建议新店设置为8折或9折。代码逻辑可理解为 `最终售价 = 商品原价 * 0.8`。
4.  点击“完成”。

![](img/32c6d0f9fb6b556f1271aa5e60d8d0eb_3.png)

**重要提示**：“满额立减”和“满件打折”只需选择一个即可，两者不能同时生效。对于新店，更推荐使用“满件打折”，因为它能直接降低顾客的决策门槛，有助于快速积累初始销量和店铺权重。

当店铺运营稳定、权重提升后，你可以取消折扣活动，恢复原价（即之前课程提到的“两倍上传”价格）销售。

## 其他工具说明

![](img/32c6d0f9fb6b556f1271aa5e60d8d0eb_7.png)

店铺后台可能还有其他营销工具，如“善单有礼”等。对于初学者而言，只需掌握本节课讲解的“活动报名”、“满额立减”和“满件打折”这几个核心工具即可，无需在其他复杂功能上花费过多精力。

![](img/32c6d0f9fb6b556f1271aa5e60d8d0eb_9.png)

本节课中我们一起学习了小红书店铺营销工具的使用方法。核心在于通过“满件打折”帮助新店快速启动，并适时参与平台大型活动获取流量。记住，营销是辅助，扎实的选品和优质的笔记内容才是店铺长期发展的根本。
