# 小红书运营教程：P29：小红书笔记引流：平台流量分发机制 🚀

![](img/c6bc36abe9b495bce8f94818b704d2f4_1.png)

在本节课中，我们将要学习小红书平台的核心流量分发机制。了解这套算法，是提升笔记曝光、获得更多流量的关键。我们将拆解其评分系统、分发流程以及影响排名的核心因素。

## 概述

小红书平台通过一套复杂的算法来决定笔记的展示顺序和流量分配。这套机制的核心目标是筛选出高质量、受用户欢迎的内容。理解这套机制，有助于我们优化笔记，从而在平台竞争中脱颖而出。

## 流量分发机制的核心：CES评分系统

小红书笔记的初始排名基于一套名为 **CES** 的综合评分系统。CES代表**点赞（C）**、**收藏（S）**、**转发/评论（E）** 以及**关注**。系统会为这些用户行为赋予不同的权重分值。

以下是CES系统的具体计分规则：
*   **点赞**：计 **1** 分
*   **收藏**：计 **1** 分
*   **转发**：计 **4** 分
*   **评论**：计 **10** 分
*   **关注**：计 **8** 分

当一个用户完成从点击到关注的全部互动行为时，该笔记将获得总计 **18** 分的最高单次互动分值。系统会计算笔记的平均CES得分，并以此作为基础排名的重要依据。

## 笔记的展示与排名规则

上一节我们介绍了CES评分系统，本节中我们来看看笔记是如何被展示和排序的。

小红书的信息流页面与抖音不同，它一次展示20篇笔记。这20个位置存在明确的排名，从第1名到第20名。用户下滑后，会刷新出下一页的20篇笔记（即第21-40名）。

笔记发布后，系统会为其打上内容标签，并推送给可能感兴趣的用户。初始阶段，平台会给予笔记一个基础的曝光量（例如1000次）。笔记能否进入更好的展示位置（如前20名），取决于它在初始曝光中获得的CES评分。

## 流量的分层推送机制

笔记的流量并非一次性发放完毕，而是分批次、阶梯式推送的。这个过程可以概括为“测试-放大”。

笔记发布后，系统会进行第一波小流量测试（例如400次曝光）。在这一波曝光中，如果笔记的**点击率**和**CES互动数据**表现良好，超过了同期测试的其他笔记，系统便会判定其为优质内容，从而在第二波推送中给予更多流量（可能翻倍）。反之，如果数据不佳，后续流量则会减少甚至停止。

因此，**点击率**是获得第二波流量的首要门槛。没有点击，后续的点赞、评论等互动就无从谈起，笔记的流量生命周期也会大大缩短。

## 影响流量分发的其他关键指标

除了CES评分，平台还会综合考察其他用户行为数据，以全面评估笔记质量。以下是几个核心指标：

*   **互动率**：指笔记评论区作者与粉丝的互动频率。高互动率直接反映了内容能引发讨论，是高质量的重要标志。
*   **完播率**：对于视频笔记，用户观看完成的比率至关重要。
*   **互动值**：这是一个综合性的用户粘性指标。

即便是一个粉丝数不多的新账号，只要笔记能引发上述优质的交互行为，获得较高的CES评分和互动数据，同样有机会被系统推荐给更多用户，实现流量的阶梯式增长。

## 总结

![](img/c6bc36abe9b495bce8f94818b704d2f4_3.png)

本节课中我们一起学习了小红书平台的流量分发机制。其核心在于一套基于用户互动行为（CES评分）的算法。笔记通过**分层测试**的方式获得流量，**点击率**是获得后续流量的钥匙，而**点赞、收藏、评论、转发、关注**等深度互动则决定了流量能攀升到多高。理解并优化这些关键指标，是提升小红书笔记排名的根本方法。
