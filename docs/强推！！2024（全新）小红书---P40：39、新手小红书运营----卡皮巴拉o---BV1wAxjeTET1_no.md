# 小红书运营：P40：39、新手小红书运营-小红书主体定位（2） 🎯

![](img/db3cc3534f35ad413c253673fe1c4a57_1.png)

在本节课中，我们将要学习小红书店铺运营中至关重要的“人群定位”。上一节我们介绍了店铺定位的基础，本节中我们来看看如何精准地定位你的目标客户群体，并理解系统如何根据这些信息进行流量分配。

![](img/db3cc3534f35ad413c253673fe1c4a57_2.png)

![](img/db3cc3534f35ad413c253673fe1c4a57_4.png)

人群定位与账号定位类似，但它是针对店铺整体运营的。不要小看小红书的搜索体系，了解人群定位是为后续店铺操作打下的基础准备。

![](img/db3cc3534f35ad413c253673fe1c4a57_6.png)

## 1. 个人属性 👤

个人属性指用户的性别、年龄范围和喜欢的内容。这与选择店铺类型是相互关联的。

小红书开店与做个人账号不同。账号起号后，店铺运营也需要匹配账号的属性，才能有效推广。如果只做了账号定位而忽略了店铺的人群属性，引流来的客户可能并不精准。因此，需要将两者结合。

选择方式是根据你的产品来确定目标人群。例如，做账号时可能选择2-3个年龄阶段，但为店铺选品时，这个范围需要进一步缩小，因为通过产品搜索来的用户更为精准。

**核心公式**：
```
目标人群 = 产品属性 ∩ (性别， 年龄， 兴趣)
```

## 2. 消费能力 💰

用户的消费能力是根据其年度总购买金额，分12个月计算出的月平均花费来评估的。

小红书系统会通过用户的购买行为（如产品类型、价格、搜索方式）来隐性地判断其消费层级。因此，在为产品定价时，必须考虑目标人群的消费能力，才能吸引到他们。

**核心概念**：
```
用户消费层级 ≈ (年度总消费额 / 12)
系统通过购买记录与行为模式进行估算
```

## 3. 购买类型 🛍️

系统分析用户购买行为后，通常只会给用户推荐最多三个类型的产品。

大类目可包括日用品、生活用品、户外用品、家具、玩具等。在自然搜索时，基本只显示三个类型的产品，且根据用户最近的购买行为进行区分。这体现了小红书“千人千面”的推荐机制。

用户经常浏览什么内容，系统就会展示什么内容。这种偏好基本2-3天就可以定型，后续若要更改，可能需要4天左右，并且需要持续搜索新的品类。

## 4. 购买地域 🌍

小红书系统会根据用户的收货地址来分析不同地域的购买偏好。

例如，系统会分析在北上广深等城市，哪些产品（如服装、食品）购买得更多。如果你的店铺产品属性恰好符合北京用户的喜好，那么系统就会将你的店铺更多地推荐给在北京浏览的相应用户。

系统会将对应目标的产品分享给对应目标的人群。

## 5. 购买习惯 🔄

用户的购买习惯也有划分，例如“货比三家”或长时间浏览后再购买。

系统根据个人用户的搜索兴趣来判断。你搜索某个兴趣越多，相关产品的推广就越广。但最终的成交是关键。如果用户在你的店铺浏览后，却去别人的店铺购买，那么下一步系统推广别人店铺属性的几率就会比你高。

过去在其他平台，存在一种竞争手段：先让人浏览同行店铺，再引导至自己店铺成交，从而“抢夺”流量。虽然现在不完全适用，但这种用户行为的潜意识影响仍然存在，需要注意。

以下是人群定位的五个核心维度总结列表：

![](img/db3cc3534f35ad413c253673fe1c4a57_12.png)

*   **个人属性**：根据产品确定用户的性别、年龄和兴趣。
*   **消费能力**：依据用户月均消费能力来制定产品价格。
*   **购买类型**：系统通常为用户推荐不超过三个产品类型。
*   **购买地域**：系统根据地域消费偏好进行定向推荐。
*   **购买习惯**：注意用户“货比三家”等行为对流量分配的影响。

---

## 综合应用与前期建议 📝

整体来说，店铺运营的流程是：先进行**店铺定位**和**选品**，初步了解定价，保证产品时效性，然后进行**人群定位**，即处理好个人属性、消费能力、购买类型、地域和习惯。

*   **如果你想做大**：需要深入理解并应用这五点，让推广更精准。
*   **如果你是新手想先熟悉节奏**：可以暂时只了解这五点，但不急于立刻严格照做。因为前期就过度限定人群，虽然流量更精准，但总曝光量可能会减少（例如从2000展示降到1000左右），不利于初期推广。可以先将重点放在店铺定位和选品上。

价格定位是上述内容的延伸，它决定了购买类型、地域和习惯的匹配。如果前期不想做精细的人群定位，可以自主定价，这并不影响操作，但需要知道，精细定位前期会牺牲部分流量，却能为后期数据提升打下更好基础。
