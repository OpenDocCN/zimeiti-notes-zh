# 剪映手机版教学：P52：人物遮挡文字的片头效果 🎬

在本节课中，我们将学习如何制作一个“人物遮挡文字”的片头效果。这种效果让人物看起来像是从文字前方走过，常用于视频开场，能有效提升视觉吸引力。我们将通过使用画中画、智能抠像、层级关系和混合模式等功能来实现它。

---

## 第一步：导入并复制视频素材

首先，我们需要导入人物走路的视频素材。

1.  打开剪映，点击 **`开始创作`**。
2.  在相册中选择人物走路的视频，点击 **`添加`**。
3.  在时间轴上选中这段视频。
4.  滑动下方菜单栏，找到并点击 **`复制`** 功能。现在时间轴上就有了两段完全相同的视频。

![](img/69a04bf89322c9d89d1819242088e3dd_1.png)

**操作代码示例：**
```
选中视频 -> 菜单栏 -> 复制
```

---

## 第二步：创建画中画并智能抠像

上一节我们复制了视频，本节中我们来看看如何利用画中画功能分离出人物。

1.  选中时间轴上的第二段视频。
2.  滑动菜单栏，点击 **`切画中画`**。
3.  长按这个新生成的画中画图层，将其拖动到时间轴最前端，与上方的视频起点对齐。
4.  选中这个画中画图层。
5.  点击菜单栏中的 **`抠像`**，然后选择 **`智能抠像`**。软件会自动处理，抠出人物。
6.  抠像完成后，点击对勾确认。

现在，我们得到了一个独立的人物图层。

---

## 第三步：添加文字并调整层级关系

人物已经抠出，接下来我们需要添加文字，并调整图层顺序，让人物位于文字上方。

1.  点击 **`新增画中画`**。
2.  导入一个带有文字的纯黑背景视频（制作方法见第五步）。
3.  选中这个文字画中画图层。
4.  点击菜单栏中的 **`层级`**。
5.  在层级面板中，将 **文字图层** 拖动到 **人物画中画图层** 的下方。这意味着人物图层将覆盖在文字图层之上。

**核心概念：**
*   **层级关系**：决定了多个画中画图层的上下覆盖顺序。数字越大或位置越靠后，图层就越在上方。

---

## 第四步：使用混合模式去除黑底

现在文字被人物挡住了，但文字本身还有黑色背景。我们需要去除黑底，只保留白色文字。

1.  确保选中文字画中画图层。
2.  在菜单栏中找到并点击 **`混合模式`**。
3.  在混合模式选项中选择 **`绿色`**。
4.  点击对勾确认。

**核心概念：**
*   **混合模式-绿色**：其作用是过滤掉黑色，保留白色或其他亮色。公式可以简单理解为：`最终显示 = 去除黑色部分`。**请注意，此方法仅适用于“黑底白字”的文字素材**。如果文字是其他颜色（如黄色），使用绿色模式会导致颜色失真。

![](img/69a04bf89322c9d89d1819242088e3dd_3.png)

应用后，黑色背景消失，只留下白色文字，且人物在文字上方移动的效果就初步显现了。

![](img/69a04bf89322c9d89d1819242088e3dd_5.png)

## 第五步：制作黑底白字文字素材

![](img/69a04bf89322c9d89d1819242088e3dd_9.png)

以下是制作“黑底白字”文字素材的步骤：

1.  在剪映首页，点击 **`开始创作`**。
2.  点击 **`素材库`**，在搜索框或热门分类中找到并选择 **纯黑色背景** 素材，点击添加。
3.  点击底部 **`文本`** -> **`新建文本`**，输入你想要的文字（例如：从广州路过）。
4.  点击 **`字体`**，为文字选择一个样式（例如：创意字体中的苏打体）。
5.  调整文字大小和位置。
6.  （可选）复制文本图层，修改内容为拼音或英文，调整大小后放在主文字下方。
7.  制作完成后，点击右上角 **`导出`** 保存视频。这个视频即可作为第三步中导入的文字素材。

![](img/69a04bf89322c9d89d1819242088e3dd_11.png)

![](img/69a04bf89322c9d89d1819242088e3dd_12.png)

---

## 第六步：为文字添加移动动画

![](img/69a04bf89322c9d89d1819242088e3dd_14.png)

最后，我们为文字添加一个从右向左移动的动画，使效果更生动。

![](img/69a04bf89322c9d89d1819242088e3dd_16.png)

1.  将时间线指针移动到文字素材的起始位置。
2.  选中文字画中画图层，在预览窗口中将其拖动到画面右侧外部。
3.  点击菜单栏中的 **`关键帧`** 按钮，添加一个起始关键帧。
4.  将时间线指针移动到文字素材的结束位置。
5.  在预览窗口中，将文字水平拖动到画面左侧外部。此时会自动生成一个结束关键帧。

现在播放预览，你会看到文字从右向左移动，而抠出的人物始终在文字前方行走，形成了人物遮挡文字的动态片头效果。

---

## 总结

本节课中我们一起学习了“人物遮挡文字”片头效果的制作方法。我们回顾一下核心步骤：
1.  **复制视频并抠像**：通过复制和智能抠像功能分离出人物主体。
2.  **理解层级关系**：利用层级功能，将人物图层置于文字图层之上。
3.  **应用混合模式**：使用 **`绿色`** 混合模式去除文字素材的黑色背景，此方法仅适用于黑底白字。
4.  **添加关键帧动画**：为文字设置位移动画，增强动态感。

掌握这个效果的关键在于理解图层的上下顺序以及混合模式的正确应用。你可以尝试使用不同的文字和人物素材，创造出属于自己的创意片头。
