### 微软 Edge 浏览器 Copilot

首先推荐的是微软 Edge 浏览器及其内置的 Copilot 功能。Copilot 是微软基于 ChatGPT 技术打造、供 Windows 系统用户使用的功能。它基于 4.0 模型开发，在智能程度上与 4.0 有一定匹配度，是普通用户可以使用的一种方式。

使用 Copilot 有两个前提条件：
1.  需要登录微软账号。你可以注册一个 Outlook 邮箱来登录。
2.  需要“梯子”（科学上网工具）才能访问该服务。

Copilot 通常集成在浏览器的右侧边栏。在最新版的 Win11 系统中，它也可能出现在系统任务栏。如果未使用“梯子”，打开 Copilot 会提示“该服务在你所在的地区不可用”。启用“梯子”后，即可正常显示 Copilot 界面。

Copilot 提供三种交互模式：**更有创造力**、**平衡**和**更精确**。
*   **更有创造力**模式：AI 会根据对话内容，以更开放的方式回答问题。
*   **平衡**模式：介于精确和创造力之间。
*   **更精确**模式：适用于需要严谨、专业回答的场景。

对于非专业内容，使用“更有创造力”或“平衡”模式通常效果较好。如果觉得回答不够理想，可以尝试“更精确”模式。

例如，我们可以使用“更有创造力”模式，让它撰写一个关于装修的短视频脚本。输入指令：`请帮我写一个关于装修的短视频脚本，时长控制在3分钟左右`。AI 会开始生成内容，响应速度取决于网络连接状况。

Edge 浏览器版 Copilot 还有一个优势：它可以基于你当前打开的网页内容进行分析。在浏览器右上角的 Copilot 图标处点击，即可使用此功能。

此外，Copilot 还能执行一些简单的电脑操作指令，例如设置暗黑模式、打开记事本或特定网页等。

然而，Edge Copilot 也存在一些缺点：
1.  **精准度稍逊**：虽然基于 4.0 模型，但经过微软调教后，在某些方面与官方 ChatGPT 4.0 相比存在细微差距。
2.  **连续对话能力受限**：在当前对话中持续提问的次数有限，达到一定限制后会提示开启新对话。
3.  **记忆功能有所缺失**：在读取和分析历史对话记录、优化后续回答方面，能力不如官方版本完整。同时，它也缺乏插件功能。

尽管如此，对于日常运营工作，Copilot 的功能已基本足够。如果你的 Edge 浏览器没有 Copilot 图标，可以尝试下载最新版的 Edge Dev（开发者）或 Beta（测试）频道版本，这些版本通常包含该功能。你也可以在浏览器设置中检查并更新至最新版本。

以下是几个可用的第三方平台介绍：

*   **灵感 AI**：这个网站非常强大，提供一定的免费使用额度。它集成了当前可用的多种模型，如 GPT-3.5、GPT-4.0、GPT-4.0 联网版、GPT-4.0 Turbo 最新版，以及讯飞星火、文心一言、通义千问等国内模型。其优点是 GPT-4.0 可免费使用一定次数，新注册用户可能有 7 天免费无限量试用期。缺点是响应速度可能较慢。如果觉得好用，也有包月付费选项。
*   **WEE**：这个网站提供 GPT-3.5 和 GPT-4.0 模型，响应速度很快。初始注册会赠送一定量的免费 Token（计算单位），但通常仅适用于 GPT-3.5 模型。想试用 GPT-4.0 可能需要付费。它还具有语音回答功能。缺点是也需要“梯子”，且免费额度有限。
*   **ChatGPT 账号共享平台**：这是一个网络共享平台，提供他人分享的 ChatGPT 账号。账号状态通常用颜色标示：绿色表示空闲，橙色表示近期使用过，红色表示正在使用或使用频繁。带有“金光”标识的通常是可用的 GPT-4.0 Plus 付费账号。你可以尝试使用这些共享账号登录官方 ChatGPT 界面。GPT-3.5 账号通常较多且免费可用，而 GPT-4.0 账号由于官方限制（如每 3小时约30条消息的限额）可能比较稀缺或需要等待。这也是一个可以免费体验 GPT-4.0 的渠道，但同样需要“梯子”。

以下是几个国内 AI 大模型的介绍：
