## 什么是自媒体？

上一节我们介绍了课程的整体框架，本节中我们来看看最核心的概念：到底什么是自媒体？

![](img/97a2f8776c77fa973b65c390295d2670_3.png)

我用最直白的话来解释：自媒体就是**个人通过互联网平台发布内容，并以此获得关注或收益的行为**。

具体操作可以概括为一个简单的公式：

**自媒体 = 注册平台账号 + 发布原创内容（文章/视频） + 获取观看流量 → 实现收益**

![](img/97a2f8776c77fa973b65c390295d2670_5.png)

这个过程可以分解为以下几步：

![](img/97a2f8776c77fa973b65c390295d2670_7.png)

1.  打开手机，在百度等搜索引擎上搜索“自媒体平台”。
2.  选择一个平台，注册一个属于自己的账号。
3.  在自己的账号上发布自己创作的文章或视频。
4.  当内容有人观看时，便可以依靠播放量等因素来结算收益。

![](img/97a2f8776c77fa973b65c390295d2670_9.png)

这个完整的流程就叫做自媒体。
