## 平台资金支持 💰

![](img/5a1e106f0528941b3fcc74b594950535_7.png)

除了政策支持，自媒体平台也积极响应号召，投入真金白银激励创作者。

通过一个平台官方视频可以了解到关键信息：**西瓜视频决定未来一年额外拿出20亿元补贴中视频创作人**。

![](img/5a1e106f0528941b3fcc74b594950535_7.png)

这20亿元有两个重点：
1.  这是**纯补贴**，不包含任何商单、直播和电商的收入。
2.  20亿元是**保底数额，上不封顶**。

![](img/5a1e106f0528941b3fcc74b594950535_9.png)

![](img/5a1e106f0528941b3fcc74b594950535_9.png)

这意味着创作者通过制作视频，可以直接从这20亿的补贴池中获得收益。

## 市场供需分析 📈

现阶段自媒体行业还有一个显著优势：庞大的用户基数和悬殊的创作者比例。

根据2022年的一份行业报告，我国相关数据如下：

*   **用户规模（需求端）**：中国网民规模约 **9.4亿**。这些是潜在的视频观众。
*   **创作者规模（供给端）**：全职与兼职从事自媒体的人数总和约 **970万**，可近似看作 **1000万**。

由此可以得出一个简单的供需模型：

```
观众人数 / 创作者人数 ≈ 9.4亿 / 0.1亿 ≈ 94 : 1
```

这相当于约每100名观众，才对应1名内容创作者。显然，内容生产的速度难以完全满足消费需求。这种供需不平衡，正是行业被持续看好的内在原因，也印证了从传统电视到互联网视频的时代发展趋势。
