# 拼多多新手开店教程：P1：必须关闭的五个危险设置 🛡️

在本节课中，我们将学习拼多多店铺后台五个必须关闭或调整的危险设置。这些设置如果保持默认开启，可能会在您不知情的情况下导致利润损失、自动降价或增加不必要的退款。关闭它们能有效保护您的店铺利润，减少运营风险。

上一节我们介绍了课程目标，本节中我们来看看具体的操作步骤。

## 第一步：停用消费者体验计划 💸

首先，您需要找到并停用“消费者体验计划”。该计划会从每笔订单中扣除额外费用。

以下是具体操作路径：
1.  进入店铺后台，找到 **售后管理**。
2.  点击 **消费者体验**。
3.  在消费者体验页面，找到并点击 **更多** 选项。
4.  选择 **停用消费者体验计划** 并确认。

**核心概念**：`消费者体验计划扣费 = 每笔订单扣款5-10元`。若不暂停，此费用将持续产生。

## 第二步：关闭店铺满减活动 🛑

接下来，需要检查并关闭可能被默认开启的“店铺满减”活动，以避免非预期的折扣。

![](img/08e782dad80982e4486f2654727aded8_1.png)

以下是关闭步骤：
1.  进入后台，点击 **店铺营销**。
2.  找到 **营销工具**，进入后找到 **店铺满减**。
3.  在店铺满减活动页面，找到 **更多操作** 按钮。
4.  选择 **结束活动** 即可将其关闭。

![](img/08e782dad80982e4486f2654727aded8_2.png)

## 第三步：关闭自动跟价功能 ⚔️

![](img/08e782dad80982e4486f2654727aded8_3.png)

“自动跟价”功能会参考同行价格自动调整您的商品售价，可能导致无底线的价格战，严重侵蚀利润。

![](img/08e782dad80982e4486f2654727aded8_4.png)

以下是关闭方法：
1.  进入 **营销活动** 页面。
2.  找到 **自动跟价说明** 旁边的 **问号（?）** 图标并点击。
3.  在弹出的说明窗口中，会出现 **关闭** 按钮，点击即可关闭此功能。

**核心概念**：当您与竞争对手都开启此功能时，系统会触发 `自动降价循环`，可能导致亏损。

## 第四步：设置退款策略以减少损失 🔄

虽然无法直接关闭退款功能，但通过设置智能策略，可以大幅减少“仅退款”订单，并挽回部分损失。

以下是推荐的策略设置思路：
1.  **创建ERP策略**：设置“取消发货自动退”和“已发货仅退款自动转拒收退款”。这能减少大部分恶意仅退款。
2.  **创建小金额自动退策略**：为小额订单设置自动退款，但为超过预期金额的订单设置门槛，防止平台自动同意大额退款。
3.  **利用售后申诉**：对于不合理的退款订单，通过后台的 **订单申诉 > 售后申诉** 路径进行申诉，尝试追回款项。

**核心公式**：`有效退款策略 = 减少90%的恶意仅退款 + 拦截大额自动退款 + 申诉追回损失`

## 第五步：关闭全局优选起量 🚫

对于新手商家，系统可能默认开启“全局优选起量”或“极速起量”功能，这可能在不经意间消耗大量推广预算。

![](img/08e782dad80982e4486f2654727aded8_6.png)

以下是关闭“全局优选起量”的步骤：
1.  进入 **商品推广** 页面。
2.  选择 **稳定成本推广** 计划。
3.  在计划列表最右侧，找到并点击 **起量设置**。
4.  在弹出的窗口中，选择 **确定关闭**，然后点击 **保存并生效**。

![](img/08e782dad80982e4486f2654727aded8_7.png)

本节课中我们一起学习了拼多多店铺后台五个关键的危险设置：消费者体验计划、店铺满减、自动跟价、退款策略以及全局优选起量。通过关闭或调整这些设置，您可以有效避免隐形扣费、非必要降价和预算浪费，从而保护店铺利润，为后续稳定经营打下坚实基础。请务必根据教程检查您的店铺设置。
