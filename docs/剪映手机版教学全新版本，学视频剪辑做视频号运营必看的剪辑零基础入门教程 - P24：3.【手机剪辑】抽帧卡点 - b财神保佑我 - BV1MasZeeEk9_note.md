# 剪映手机版教学：P24：3.【手机剪辑】抽帧卡点

在本节课中，我们将要学习一种流行的视频剪辑技巧——抽帧卡点。这种技巧能让你的视频节奏与音乐完美同步，创造出富有动感和冲击力的视觉效果。我们将详细介绍两种实现抽帧卡点的方法。

## 🎵 什么是抽帧卡点？

抽帧卡点是一种通过删除视频中特定帧来匹配音乐节奏的剪辑手法。其核心原理是：在音乐的节奏点上，将视频分割并删除中间片段，使画面产生“跳跃”或“卡顿”感，从而强化节奏。

## 🎼 第一种方法：自动踩点

当背景音乐的节奏点非常明显时，我们可以使用剪映的“自动踩点”功能来快速完成操作。

以下是具体操作步骤：

1.  **导入音乐**：在时间线上添加你喜欢的音乐。
2.  **选中音乐**：点击选中音频轨道。
3.  **自动踩点**：点击下方工具栏中的 **“踩点”** 按钮，然后选择 **“自动踩点”**。
4.  **分割视频**：在音乐节奏点（波形高点）出现的位置，将视频轨道分割。
5.  **删除片段**：在两个相邻的节奏点之间，将分割出来的视频片段删除。
6.  **重复操作**：继续在下一个节奏点分割，并删除中间片段，如此重复直到完成。

**核心操作公式**可概括为：`选中音乐 -> 自动踩点 -> 在标记点分割视频 -> 删除中间片段 -> 重复`。

## 🎧 第二种方法：手动踩点

如果音乐的节奏不那么规整，或者你想要更个性化的卡点效果，就需要使用手动踩点。

上一节我们介绍了自动踩点，本节中我们来看看如何手动操作。

以下是具体操作步骤：

1.  **手动添加点**：选中音乐轨道后，在 **“踩点”** 模式下，手动在每一个节奏波形的高峰位置点击添加标记点。
2.  **分割与删除**：标记点添加完毕后，像第一种方法一样，在这些手动添加的标记点位置将视频分割，并删除中间不需要的片段。

这样，即使面对节奏复杂的音乐，你也能精准地完成抽帧卡点。

## 📝 总结

本节课中我们一起学习了抽帧卡点的两种实现方法。第一种是依靠软件的 **`自动踩点`** 功能处理节奏明显的音乐；第二种是通过 **`手动添加标记点`** 来应对节奏自由的音乐。掌握这两种方法，你就能轻松为视频添加富有节奏感的抽帧效果，让作品更具吸引力。
