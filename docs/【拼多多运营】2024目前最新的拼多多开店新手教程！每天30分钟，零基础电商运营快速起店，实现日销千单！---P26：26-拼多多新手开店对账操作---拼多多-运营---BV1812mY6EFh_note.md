# 拼多多运营：P26：拼多多新手开店对账操作 💰

在本节课中，我们将学习拼多多店铺后台的核心对账操作。了解如何提现、查看收益、管理保证金以及查询扣款明细，是每位新手商家必须掌握的基本功。清晰的账目管理是店铺健康运营的基础。

## 一、店铺资金提现操作 💳

上一节我们介绍了课程概述，本节中我们来看看如何将店铺收入提现到个人银行卡。这是商家获取经营回报的直接方式。

操作路径为：打开商家管理后台 -> 点击“账户资金” -> 选择“贷款账户” -> 点击“提现”。

![](img/e3e13d5b28e98cae64287ead57a4a5ad_1.png)

![](img/e3e13d5b28e98cae64287ead57a4a5ad_2.png)

![](img/e3e13d5b28e98cae64287ead57a4a5ad_3.png)

以下是具体的操作规则和注意事项：
*   **提现频率与金额**：每日仅可提现一次，最小提现金额为100元。
*   **审核与到账时间**：提现申请需1-2个工作日审核，审核通过后，资金将在3个工作日内到账。
*   **延迟情况**：若遇法定节假日或周末，到账时间可能顺延。

![](img/e3e13d5b28e98cae64287ead57a4a5ad_4.png)

## 二、查看店铺经营收益 📊

了解如何提现后，我们需要知道店铺究竟赚了多少钱。接下来，我们将学习如何通过后台账单查看详细的收入与支出明细。

你需要进入“贷款对账单”页面进行查询。路径为：商家后台 -> 账户资金 -> 贷款对账单。

以下是账单查询的功能介绍：
*   **查看明细**：支持按日或按月查看交易明细，包括每一笔收入与支出。
*   **高级筛选**：可通过“高级选项”筛选不同的账单类型，如货款、技术服务费等。
*   **数据导出**：可直接点击“导出”按钮，将账单数据下载为表格文件，方便离线分析。
*   **查询限制**：单次最长可查询31天内的数据，最多显示50万条记录。

## 三、理解各类保证金 🛡️

![](img/e3e13d5b28e98cae64287ead57a4a5ad_6.png)

在经营过程中，我们可能会缴纳多种保证金。本节将解析店铺保证金、活动保证金和营销保证金的作用与区别。

保证金主要用于保障平台交易和营销活动的正常进行，并非“保护费”。若无违规，保证金金额保持不变。

![](img/e3e13d5b28e98cae64287ead57a4a5ad_8.png)

以下是三种常见保证金的说明：
1.  **店铺保证金**：开店时必须缴纳。个人店铺通常为2000元，企业店铺为1000元。
2.  **活动保证金**：报名特定平台活动（如“秒杀”）时需缴纳。例如，秒杀活动通常要求5000元保证金。若活动结束或无违规，可申请提现。
3.  **营销保证金**：使用某些营销工具（如“拼单返现”、“关注店铺优惠券”）前需充值。例如，创建预算为76元的返现活动，需先充值76元作为保证金。

## 四、查询扣款明细 🔍

![](img/e3e13d5b28e98cae64287ead57a4a5ad_10.png)

![](img/e3e13d5b28e98cae64287ead57a4a5ad_11.png)

有时我们会发现账户资金被扣除，却不清楚原因。最后，我们来学习如何查询具体的扣款明细，做到每一笔支出都心中有数。

扣款主要发生在以下几种情况：延迟发货罚款、缺货罚款以及通过“小额打款”功能向消费者退款。

查询路径为：商家后台 -> 账户资金 -> 贷款扣款明细。

在扣款明细页面，你可以：
*   通过**商品ID、扣款时间、订单编号**等信息精准查询。
*   查看“最近24小时”或“最近30天”等时间段的扣款记录。
*   若记录为空，则表明店铺在该时段内经营规范，无违规扣款。
