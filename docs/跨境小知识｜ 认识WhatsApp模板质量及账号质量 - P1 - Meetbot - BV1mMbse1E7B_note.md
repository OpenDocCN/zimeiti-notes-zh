# WhatsApp营销入门：P1：认识WhatsApp模板质量与账号质量 📱

在本节课中，我们将要学习WhatsApp商业账号的两个核心风控概念：**模板质量**和**账号质量**。理解这两个指标对于安全、有效地进行WhatsApp营销至关重要。

---

## 什么是模板质量？

上一节我们介绍了课程目标，本节中我们来看看什么是模板质量。模板质量是WhatsApp官方对企业账号所创建的消息模板进行的一项评级，它直接反映了该模板在发送时的安全性与风险等级。

企业账号的模板质量共有9种具体等级，但为了便于理解和操作，WhatsApp使用4种颜色对其进行直观评分。

以下是四种模板质量颜色评级的含义：

*   **绿色**：高质量模板。表示该模板内容安全合规，发送风险极低，是发送消息时的**首选**。
*   **黄色**：中质量模板。表示模板存在一定风险，可能需要优化或谨慎使用。
*   **红色**：低质量模板。表示模板风险较高，使用此类模板发送消息容易导致送达率下降或账号受限。
*   **灰色**：通常表示模板正在审核中或已被拒绝。

在使用模板发送营销或通知消息时，应**首选绿色评分的模板**。使用黄色或红色评分的模板都会带来不同程度的风险，可能导致消息无法送达或影响账号健康。

## 什么是账号质量？

了解了模板质量后，我们再来看看与之密切相关的账号质量。账号质量是对您的整个WhatsApp商业账号健康状况的综合评估。

企业账号的账号质量主要分为三种状态，并通过两种颜色标识其安全等级。

以下是账号质量的三种状态与颜色标识：

*   **绿色（高质量）**：账号状态**安全**。可以正常开展营销活动。
*   **黄色**：账号存在风险。建议暂停营销活动，检查并优化操作，否则可能收到官方警告。
*   **红色 / 已标记**：账号风险很高或已被系统标记。**建议立即停止发送消息**，并排查问题，否则极有可能导致账号被封禁。

![](img/909a5e75086f9ac7164d8fdf79eb2606_1.png)

对于黄色、红色及已标记的账号状态，最稳妥的做法是暂停营销甚至完全停止主动发送消息，以避免账号收到警告乃至被封号。

## 总结

本节课中我们一起学习了WhatsApp风控体系中的两个关键指标。

1.  **模板质量**：分为绿、黄、红等颜色等级，**发送消息应首选绿色高质量模板**。
2.  **账号质量**：分为绿色（安全）、黄色（风险）、红色/已标记（高危）状态，**非绿色状态需谨慎操作，必要时暂停营销**。
