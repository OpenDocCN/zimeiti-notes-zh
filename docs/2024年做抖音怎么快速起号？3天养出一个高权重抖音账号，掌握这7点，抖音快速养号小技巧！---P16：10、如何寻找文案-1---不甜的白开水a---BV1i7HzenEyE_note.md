# 抖音运营教程：P16：10、如何寻找文案-1

在本节课中，我们将要学习如何为抖音视频寻找和准备文案。这是优化作品、吸引粉丝的关键一步。我们将介绍实用的工具和方法，帮助你解决“不知道写什么”和“声音不好听”的难题。

上一节我们讨论了提升作品质量的重要性，本节中我们来看看如何具体解决文案创作的问题。很多初学者面临不会写文案的困境，这没有关系，我们可以借助一些高效的工具。

以下是五个可以直接在网页端使用的文案素材平台，无需下载APP，在手机或电脑浏览器中搜索即可使用。

*   淘金阁
*   （其他四个平台名称，根据原文“五个是文案的网页”提示，此处应列出，但原文未明确给出，故保留概括）

![](img/b0c157b0863f398c301b1f2cfdf46aa6_1.png)

我以“淘金阁”为例进行讲解。这是该网站的主页，上面有非常多的内容分类。

![](img/b0c157b0863f398c301b1f2cfdf46aa6_1.png)

![](img/b0c157b0863f398c301b1f2cfdf46aa6_3.png)

你可以看到财经、军事、动漫、科技、美食、情感、三农等众多类别。我们点开“情感”类型，页面会展示大量相关话题文案，通常有多页内容可供选择。

![](img/b0c157b0863f398c301b1f2cfdf46aa6_3.png)

选择文案时，可以套用一个简单的公式：**一个群体 + 一个核心话题 + 疑问**。例如，“宝妈如何平衡工作与家庭？”。你觉得合适的文案就可以使用。

![](img/b0c157b0863f398c301b1f2cfdf46aa6_5.png)

如果分类里没有你想要的特定内容，可以使用网站的搜索功能。在搜索框输入关键词，例如“宝妈”，然后点击抓取。

![](img/b0c157b0863f398c301b1f2cfdf46aa6_7.png)

![](img/b0c157b0863f398c301b1f2cfdf46aa6_7.png)

搜索结果会显示所有包含该关键词的文案，这通常足够满足你的需求。手机和电脑的操作方法是一样的。

![](img/b0c157b0863f398c301b1f2cfdf46aa6_9.png)

解决了文案来源，接下来解决配音问题。很多同学担心自己的声音不够好听，我们可以使用工具进行配音。

![](img/b0c157b0863f398c301b1f2cfdf46aa6_11.png)

在淘金阁网站右上角找到“小工具”入口，点击进入后选择“配音在线生成”功能。

![](img/b0c157b0863f398c301b1f2cfdf46aa6_13.png)

![](img/b0c157b0863f398c301b1f2cfdf46aa6_11.png)

这个功能是免费的。进入后，你会看到非常丰富的音色选择。

![](img/b0c157b0863f398c301b1f2cfdf46aa6_13.png)

![](img/b0c157b0863f398c301b1f2cfdf46aa6_15.png)

以下是部分音色类型：

![](img/b0c157b0863f398c301b1f2cfdf46aa6_17.png)

*   亲和女生、温柔女生、甜美女生
*   自然女声、卖场广播
*   标准男声、标准女声
*   方言配音（如四川话、粤语、东北话）
*   童声、英文配音
*   情感解说、影视解说音色

![](img/b0c157b0863f398c301b1f2cfdf46aa6_19.png)

使用方法很简单。在左侧“配音文本”框中输入或粘贴你准备好的文案。

![](img/b0c157b0863f398c301b1f2cfdf46aa6_17.png)

在右侧选择你喜欢的音色，例如“东北话”。你还可以调整音量、语速和语调。

![](img/b0c157b0863f398c301b1f2cfdf46aa6_21.png)

![](img/b0c157b0863f398c301b1f2cfdf46aa6_21.png)

![](img/b0c157b0863f398c301b1f2cfdf46aa6_23.png)

设置完成后，点击“立即生成”。在手机上使用时，可以扫码登录。生成后，在配音列表中可以下载音频文件到手机。

![](img/b0c157b0863f398c301b1f2cfdf46aa6_25.png)

![](img/b0c157b0863f398c301b1f2cfdf46aa6_25.png)

![](img/b0c157b0863f398c301b1f2cfdf46aa6_27.png)

下载的音频文件可以在后续的视频剪辑环节中使用。这样，你不仅解决了文案问题，也一并解决了配音的难题。

本节课中我们一起学习了如何利用在线工具寻找文案素材，并使用免费的配音功能为视频生成音频。掌握这两个技巧，能有效提升你视频内容的吸引力和制作效率，为优化作品、吸引粉丝打下坚实基础。
