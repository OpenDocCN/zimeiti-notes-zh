# 跨境电商入门教程：P4：03、跨境电商的发展预测 📈

![](img/02b032d065013911eb08851fa29f9053_1.png)

在本节课中，我们将探讨全球疫情对跨境电商市场的影响，并分析由此带来的市场趋势变化。我们将了解特定品类需求的升降，以及这些变化背后的原因。

---

上一节我们介绍了跨境电商的基础概念，本节中我们来看看在特殊时期，跨境电商的未来走势如何。很多同学关心疫情造成的影响。通过新闻可以了解到，中国是全球第一个爆发疫情的国家，同时也是最先控制住疫情的国家。

因此，许多外国国家开始效仿中国的防控措施，例如居家隔离。然而，由于文化差异等因素，部分国家的疫情至今仍未得到有效控制。例如，美国每日仍有大量新增感染和死亡病例。

这次疫情的影响非常广泛。许多遵守规定的国外商场开始实施限流措施。商场作为人口聚集地，如果不限流，很容易导致交叉感染。商场限流和居家隔离政策，导致整个线下交易受到巨大冲击，线上交易则开始活跃起来。

此外，许多国外工厂也处于关门状态。工人出于安全考虑不敢上班。中国工厂能够复工是因为疫情已受控，而许多外国工厂则难以承担复工的风险和潜在赔偿。外国工厂关门导致了供需不平衡，即本国无法生产出所需的产品，于是许多人转向跨境电商平台进行购买。

因此，整个市场因疫情发生了显著变化。以下是基于统计观察到的具体变化：

受疫情影响，许多居家类型的产品需求量上升。
*   **居家生活类产品**：例如日常家居用品。
*   **健身器材**：例如哑铃、瑜伽服、瑜伽垫、瑜伽球、跳绳、跑步机等室内健身产品。
*   **厨房用品**：居家烹饪相关产品的需求也在增加。

相反，一些非居家类产品的需求则出现下降。
*   **户外运动设备**：例如篮球、足球等需要外出使用的各类设备。

![](img/02b032d065013911eb08851fa29f9053_3.png)

这就是疫情对市场造成的具体影响，这个变化点需要大家清楚。

本节课中，我们一起学习了疫情如何重塑跨境电商市场。核心结论是：**线下受阻，线上活跃；工厂停产，跨境补缺；居家品类需求上升，户外品类需求下降**。理解这些趋势变化，有助于我们在选品和运营时做出更明智的决策。
