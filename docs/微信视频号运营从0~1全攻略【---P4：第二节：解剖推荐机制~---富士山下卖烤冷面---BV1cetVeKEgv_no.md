# 微信视频号运营从0~1全攻略：P2：解剖推荐机制 🔍

在本节课中，我们将要学习微信视频号的核心推荐机制。理解这套机制是做好视频号运营的基础，它与抖音等平台有本质区别，掌握其原理才能避免走弯路。

上一节我们介绍了视频号的平台属性，本节中我们来看看它的核心推荐逻辑是如何运作的。

![](img/42cad14b67c3e9ab7058fbf4f90f12bb_1.png)

## 推荐机制的核心：社交链启动

![](img/42cad14b67c3e9ab7058fbf4f90f12bb_2.png)

![](img/42cad14b67c3e9ab7058fbf4f90f12bb_3.png)

视频号的推荐机制与抖音不同，其核心是基于社交链进行“强启动”。

![](img/42cad14b67c3e9ab7058fbf4f90f12bb_4.png)

**社交链**是指：当用户A点赞了某个视频，该视频会出现在用户A的所有好友的“朋友点赞”信息流中。这与抖音不同，在抖音中，用户可以关闭“朋友喜欢”的展示，点赞行为相对私密。而在视频号，点赞行为具有公开的社交传播属性。

我们可以用一个简单的模型来理解这个启动过程：

1.  **初始用户冷启动**：假设你是我的粉丝（初始用户），你看到并点赞了我的视频。
2.  **辐射一环社交圈**：你的点赞行为，导致我的视频被推送给你的所有好友（你的社交圈）。
3.  **无限延伸传播**：如果你的好友也觉得视频不错并点赞，那么视频将进一步推送给他们的好友，从而实现社交链的裂变式传播。

![](img/42cad14b67c3e9ab7058fbf4f90f12bb_6.png)

这个过程的前提是，内容本身需要获得用户（尤其是初始用户）的认可，即拥有较高的**点赞率**。

## 从私域到公域的流量跃迁

在第一节我们提到，视频号是一个偏私域属性的平台。如果内容仅在私域社交圈内传播，意义有限。因此，平台设计了一套从私域启动到公域分发的机制。

![](img/42cad14b67c3e9ab7058fbf4f90f12bb_8.png)

当内容通过社交链传播，其用户互动数据（如点赞、转发量）达到某个**阈值**后，平台会判定该内容为优质内容，并将其投入更大的**公域流量池**，推荐给更多陌生人。

![](img/42cad14b67c3e9ab7058fbf4f90f12bb_9.png)

**阈值**的数值因领域而异。例如，对于知识类博主，阈值可能是获得200个点赞，并触达2000次观看。

因此，你在刷视频号时，既会看到好友点赞的视频（社交推荐），也会看到完全不认识的人发布的视频（纯公域推荐）。视频号的推荐机制正是**社交推荐**与**公域推荐**两者的结合与关联。

## 流量分级：一级与二级传播

为了更直观地理解，我们可以将流量分为一级流量和二级流量。

*   **一级流量**：指你的粉丝。当你发布内容后，系统会首先将内容推送给你的粉丝。
*   **二级流量**：指通过粉丝的社交行为（点赞、转发）所触达的、粉丝的好友及其更外层的社交网络。

传播路径可以表示为：
`你的发布 -> 推送给一级流量（你的粉丝）-> 粉丝互动 -> 辐射至二级流量（粉丝的好友）-> 二级流量互动 -> 辐射至更外层...`

通过一级流量的“辐射”加上二级流量的“传播”，内容得以在社交网络中蔓延开来。

## 如何触发推荐：关键的用户反馈指标

从运营层面看，要想做出爆款，关键在于促使用户产生特定的互动行为。平台主要依据以下几项用户反馈数据来决定是否将内容推入更大的公域流量池：

以下是核心的用户反馈指标：

1.  **点赞/转发率**：这是视频号权重最高的指标。点赞直接触发社交链传播，是冷启动的关键。
2.  **完播率**：用户观看视频的完整程度，反映了内容吸引力。
3.  **互动率**：主要指评论率和关注率，反映了内容的讨论度和账号吸引力。

![](img/42cad14b67c3e9ab7058fbf4f90f12bb_11.png)

其中，**高点赞和高转发是触发流量跃迁阈值的最直接动力**。

## 催生爆款的内容关键：以“情”动人

那么，什么样的内容更容易获得高点赞和高互动呢？根据运营经验，一个非常关键的点是：**动情促进高互动，更易催生出爆款**。

“情”包含多种维度：

*   **感动之情**：如公益善举、亲情、爱情故事，容易击破心理防线。
*   **骄傲之情**：如家国情怀、民族自豪感相关的内容，让人振奋。
*   **感性之情**：能引发特定群体（如女性、父母）强烈共鸣的内容，让人感同身受。

情感内容最容易引发用户强烈的情绪反应，从而促使其进行点赞、评论等互动行为。这也解释了为什么在视频号的头部爆款和热门榜单中，**情感类账号占据了相当大的比例**。

## 平台的流量倾斜策略

了解平台对不同账号的流量扶持策略，有助于我们定位自身。平台的流量倾斜大致分为三级：

![](img/42cad14b67c3e9ab7058fbf4f90f12bb_13.png)

以下是流量倾斜的三个级别：

*   **一级倾斜（最多）**：面向**头部大号**（如顶级情感号）。平台会给予其巨大的公域流量扶持，使其长期占据榜单前列。
*   **二级倾斜（其次）**：面向**MCN机构旗下账号**。平台会给予一定的初始流量包或倾斜，然后根据内容反馈数据决定是否追加流量。
*   **三级倾斜（较少）**：面向**素人白号**。这类账号没有先天流量优势，必须依靠内容本身获得**高互动、高反馈**，才能突破阈值，进入更大的流量池。

## 核心问题与运营起号策略

基于以上机制，一个核心问题浮现出来：**对于一个没有初始粉丝（冷启动用户）的新号，该如何运营起来？**

正确的策略不是只在视频号内闭门造车。建议采用“从公域引流，在视频号内转化与裂变”的路径：

1.  **公域引流**：首先在抖音、小红书等公域平台进行内容运营，积累初始粉丝和影响力。
2.  **引导迁移**：将这些公域流量引导至你的视频号，成为你的**初始启动用户**。
3.  **视频号内深耕**：在视频号内，专注于生产能引发“高互动、高点赞”的内容，利用这批初始用户启动社交链传播。
4.  **冲击公域**：当互动数据达到阈值，内容便有機會进入视频号的公域流量池，获得爆发式增长。
