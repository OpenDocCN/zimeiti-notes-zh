### 1. 角色：明确你的专业身份

![](img/693dec5f44b8d6f7d2d1da65cade660b_80.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_82.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_84.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_86.png)

角色，简单来说，就是你基于所选赛道和内容方向，需要向用户展示的专业身份。你需要让别人相信，你提供的内容具有专业性。

![](img/693dec5f44b8d6f7d2d1da65cade660b_88.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_90.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_92.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_94.png)

以下是不同赛道的角色示例：

![](img/693dec5f44b8d6f7d2d1da65cade660b_96.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_98.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_100.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_102.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_104.png)

*   **好物分享/测评**：你可以是资深的产品体验官或挑剔的消费者。
*   **装修博主**：你需要展示自己真实拥有并正在装修的房屋，以记录者和避坑分享者的身份出现。公式可以概括为：**角色 = 赛道领域 + 专家/经验者身份**。
*   **母婴赛道**：最常见的角色是“宝妈”。但根据内容方向，你也可以扮演“育儿护理专家”或“儿童心理顾问”等角色。
*   **情感赛道**：你可以是“情感咨询师”、“心理学顾问”，也可以是分享自身经历的“成长型人设”。例如，一位30岁女性分享自己如何处理家庭、职场、创业中的问题。

### 2. 内容：保持垂直与价值

![](img/693dec5f44b8d6f7d2d1da65cade660b_122.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_124.png)

内容是人设的基石。关于内容定位，我们之前强调过几个要点：内容要吸引用户、要有价值、要有平台特色。

![](img/693dec5f44b8d6f7d2d1da65cade660b_130.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_132.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_138.png)

在人设定位的语境下，内容的关键在于 **垂直和持续**。这直接关系到账号标签和平台流量分发。

![](img/693dec5f44b8d6f7d2d1da65cade660b_140.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_142.png)

```python
# 假设平台根据用户历史行为（如点赞）推荐内容
user_interests = [“育儿”, “装修”] # 用户标签
your_video_topics = [“育儿”, “宠物”] # 你的视频主题列表

![](img/693dec5f44b8d6f7d2d1da65cade660b_148.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_150.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_152.png)

# 如果内容不垂直，用户可能因不感兴趣而划走
if current_video_topic not in user_interests:
    user_skip() # 用户划走
    platform_feedback(“内容不相关”) # 向平台反馈负面信号
```

### 3. 形式：确定内容的呈现方式

![](img/693dec5f44b8d6f7d2d1da65cade660b_172.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_174.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_176.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_178.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_180.png)

形式指的是你通过何种具体方式将内容呈现给用户。它包含以下几个需要确定的方面：

![](img/693dec5f44b8d6f7d2d1da65cade660b_182.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_184.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_186.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_188.png)

*   **产出场景**：你的内容是在固定的室内场景拍摄，还是在多变的室外场景（如上班路上、面试途中、车内）录制？拍摄场景可以参照对标账号的设置。
*   **出镜与声音**：你是否亲自出镜？你的声音是原声呈现，还是使用配音工具（如剪映）处理？这取决于你能否流畅、自信地进行表达。
*   **内容形式**：你的内容主要采用哪种体裁？
    *   **剧情型**：创作难度较高，需要剧本和表演。
    *   **干货/经验分享型**：需要精心准备文案和脚本。
    *   **生活记录型**：需要积累大量的日常拍摄素材。

### 4. 记忆点：打造深刻的用户印象

![](img/693dec5f44b8d6f7d2d1da65cade660b_208.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_210.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_212.png)

记忆点是让人设脱颖而出、被用户牢牢记住的关键。它需要通过设计一些固定元素，反复刻入用户心智。

![](img/693dec5f44b8d6f7d2d1da65cade660b_214.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_216.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_218.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_220.png)

以下是常见的记忆点设计，可分为两类：

![](img/693dec5f44b8d6f7d2d1da65cade660b_222.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_224.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_226.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_228.png)

**（1）视频开头前置（强化冲击力）**
*   **固定动作**：每期视频开始时的标志性动作。
*   **固定特效**：独特的开场特效或转场。
*   **固定开场白**：一句标志性的开场语。

![](img/693dec5f44b8d6f7d2d1da65cade660b_230.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_232.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_234.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_236.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_238.png)

将这些元素放在视频前几秒，当用户浏览你的主页多个视频时，重复的强刺激会极大加深印象。公式为：**记忆点强化 = 固定元素 × 出现位置（前置）**。

![](img/693dec5f44b8d6f7d2d1da65cade660b_240.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_242.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_244.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_246.png)

**（2）贯穿内容全局（营造氛围感）**
*   **统一BGM**：使用同一首或同一风格的音乐作为背景。当音乐响起，用户即使没看到画面，也能联想到你。
*   **标志性物品**：视频中始终出现的某个物品，如特定的台灯、玩偶或背景装饰。
*   **统一服装风格**：符合你人设的固定着装风格，如田园风、职场精英风等。

![](img/693dec5f44b8d6f7d2d1da65cade660b_248.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_250.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_252.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_254.png)

![](img/693dec5f44b8d6f7d2d1da65cade660b_256.png)

这些元素贯穿视频始终，共同营造出独特的氛围和辨识度。
