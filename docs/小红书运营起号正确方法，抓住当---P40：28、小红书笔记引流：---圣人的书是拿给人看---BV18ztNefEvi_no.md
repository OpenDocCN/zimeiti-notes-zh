# 小红书运营起号正确方法：P40：28、小红书笔记引流：平台流量结构 📊

![](img/b017db5787d7992f9cbad01ace3998d2_1.png)

在本节课中，我们将要学习小红书平台的流量结构。了解流量从哪里来，以及如何针对不同来源优化笔记，是成功运营账号的第一步。

上一节我们介绍了笔记制作的基础，本节中我们来看看小红书笔记引流的整体流量结构。流量结构指的是平台流量的分布和来源。理解这一点，能帮助我们明确笔记的投放方向，以获得最佳效果。

小红书笔记的整体流量结构主要分为三个部分。

以下是三个核心流量入口：

1.  **发现页**
2.  **关注页**
3.  **搜索页**

打开小红书App，主页面通常显示“发现”和“关注”两个标签页，旁边还有一个放大镜图标代表“搜索”。这是三个最主要的流量入口。此外，还有一个“本地”页面，但此页面对于大多数运营目的并非首选。

“本地”页面主要展示基于地理位置的内容。它适用于特定类型的博主，例如户外、美食探店（如介绍北京、上海、杭州的美食）等。如果你运营的是商品推广或引流账号，应尽量避免使用地域性标签进行推广。

因为基于地域的推荐逻辑是：距离用户越近、发布时间越短、互动数据越好的内容，展示越多。这会打乱你账号的整体标签，吸引来的可能并非你的目标客户。例如，你卖食品，但“本地”页可能推荐给来看美女的用户，转化率很低。

除非你的商业模式就是本地化服务（例如同城首饰店，支持送货上门），否则若想将商品卖向全国，应专注于全国性的流量入口，避免使用本地推广。本地市场的用户属性较为混杂，不利于账号标签的精准养成。

因此，笔记引流的核心流量结构就是上述三个重点：发现页、关注页和搜索页。

首先，我们了解一下什么是发现页。发现页是小红书**最重要**的流量来源，约占平台总流量的 **70%**。它也是平台测试笔记内容表现的主阵地。

一篇笔记最终能获得多大规模的曝光，很大程度上取决于它在发现页获得的用户反馈（如点赞、收藏、评论）。之前课程提到过，账号权重会影响笔记的基础曝光量。一篇新笔记发布后，平台根据账号权重，可能会给予一个初始曝光（例如1000或2000次）。这**初始曝光的100%** 以及后续主要的流量增长，都发生在发现页。

![](img/b017db5787d7992f9cbad01ace3998d2_3.png)

对于新账号或新笔记，流量几乎全部来自发现页。关注页和搜索页的流量，通常属于那些已有稳定粉丝群或高权重内容的成熟账号。发现页占据了总流量的70%，剩下的30%则分布在关注页和搜索页。

![](img/b017db5787d7992f9cbad01ace3998d2_3.png)

*（图示为发现页界面）*

接下来，我们看看搜索页。搜索页的流量占比仅次于发现页，大约为 **20%**，并且是小红书平台内**增长最快**的流量板块。

小红书目前拥有约3亿用户，增长已进入平稳期。平台未来的增长动力，将更多来自于现有用户购物需求的提升。搜索页流量的增长，正反映了用户“主动寻找商品”的意图在加强。

与淘宝、拼多多（主打商品）和抖音（主打内容）不同，小红书是内容和商品的综合体。用户通过搜索框主动寻找商品，其购买意图非常明确。因此，虽然搜索页总流量占20%，但其**成交转化率却可能占据全平台的70%以上**。

用户在其他页面（发现页、关注页）可能是被内容吸引，进而产生兴趣购买。而在搜索页，用户是带着明确需求来的，转化路径更短，效率更高。然而，搜索页的流量并非新账号初期就能轻易获取。

搜索流量依赖于笔记的**关键词排名**，而这需要账号权重、笔记质量、历史成交、用户评价等多维度数据积累。对于新账号，首要任务不是争夺搜索流量，而是通过优质内容在发现页获得曝光和互动，提升账号基础权重。

最后，我们来了解关注页。关注页的流量占比最小，大约只有 **10%**，甚至可能不到（实际约7%左右）。但是，这部分流量是**最精准**的。

关注页位于App最左侧。用户一旦关注了你，他们发布的新笔记就会出现在这个页面。这部分用户是你的粉丝，对你的内容或商品已经建立了初步的信任和兴趣。因此，针对他们进行营销或引流，成功率会更高。

不过，有一个关键点需要注意：来自关注页的流量**不计入影响笔记能否成为“爆文”的核心权重数据**。例如，你有一个笔记获得了1万次浏览，但如果这1万次全部来自你的1万个粉丝在关注页的点击，这篇笔记很可能不会进入更大的公域流量池（发现页）进行推荐。

平台判断笔记是否优质，主要看它在**公域流量（发现页）** 中的表现。只有当笔记在发现页的初始曝光中获得了良好的互动率（点击率、点赞率等），平台才会给它更多曝光。单纯依靠粉丝在关注页的点击，很难“撬动”平台的推荐算法。

因此，运营的顺序很关键：前期重点做好**发现页**，积累权重和口碑；中期优化**搜索页**关键词，承接主动需求；长期维护**关注页**粉丝，实现精准转化。这就是小红书平台的流量结构钥匙。

本节课中我们一起学习了小红书流量的三大来源：占比70%的发现页（主阵地）、占比20%的搜索页（高转化）以及占比约10%的关注页（最精准）。理解这个结构，能帮助我们有策略地分配运营精力，实现账号的稳步成长。
