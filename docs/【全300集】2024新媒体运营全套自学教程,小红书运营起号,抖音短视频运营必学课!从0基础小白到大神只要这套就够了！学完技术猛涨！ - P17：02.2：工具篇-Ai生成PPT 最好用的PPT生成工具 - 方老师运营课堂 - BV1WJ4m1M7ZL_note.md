# 新媒体运营自学教程：P17：02.2：工具篇-AI生成PPT 🚀

在本节课中，我们将要学习一款非常实用的AI生成PPT工具。这款工具能够根据你提供的主题或现有文档，自动梳理大纲并生成详细的PPT内容，是提升工作效率的利器。

上一节我们介绍了新媒体运营的基础概念，本节中我们来看看如何利用AI工具辅助内容创作。

## 工具简介与模式

这款AI工具主要提供两种生成模式，满足不同场景下的需求。

以下是两种核心工作模式：
1.  **主题生成模式**：输入一个主题，工具会结合生成式AI（如ChatGPT）的能力，自动生成完整的PPT全文。
2.  **文档生成模式**：上传一篇现有的文档（如TXT、PDF），AI助手会自动总结文档大纲，并基于此大纲生成PPT的详细内容。

## 核心优势与内容编辑

这款工具的核心优势在于，它不仅能处理完整的文档，更能基于**零散的大纲或要点**，自动扩充生成详细的页面内容。这与许多仅能美化已有全文内容的AI工具有本质区别。

生成PPT后，我们可以进行多项调整，使其更符合需求。

以下是主要的后期编辑功能：
*   **更换模板**：工具提供多种设计模板，可以一键切换，快速改变整体风格。
*   **编辑内容**：在右侧编辑面板中，可以直接修改每页的文字、调整形状、替换图片等。
*   **本地深度调整**：可以将生成的PPT下载到本地（如PowerPoint或Keynote中），进行更精细的个性化设计。
