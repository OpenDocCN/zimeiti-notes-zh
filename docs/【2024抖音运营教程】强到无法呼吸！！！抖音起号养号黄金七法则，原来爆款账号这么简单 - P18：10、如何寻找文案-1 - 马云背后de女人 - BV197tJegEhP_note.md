# 抖音运营教程：P18：10、如何寻找文案-1 📝

在本节课中，我们将要学习如何为抖音视频寻找和准备文案。对于初学者而言，文案是视频内容的核心，好的文案能有效提升视频吸引力，帮助你将浏览用户转化为粉丝。我们将介绍几个实用的在线工具，并详细讲解如何利用它们来获取和生成文案，甚至解决配音问题。

## 优化作品以提升吸粉能力

上一节我们讨论了提升浏览量，本节中我们来看看如何将浏览量转化为粉丝。关键在于优化作品质量。即使每天发布多条视频，如果内容质量不佳，平台也不会给予更多流量，效果反而会差。因此，我们的策略是：**宁可减少发布频率，也要确保每条视频都足够吸引人**。接下来，我们将一步步讲解如何优化作品。

首先需要解决的是文案问题。许多同学不擅长撰写文案，这没有关系。

## 利用在线工具寻找文案

以下是五个可用于寻找文案的网页平台（非App，无需下载）：

1.  **淘金阁**
2.  **易撰**
3.  **乐观号**
4.  **微小宝**
5.  **新媒体管家**

![](img/72cc3aa23ecf464b5ecec893837649a6_1.png)

你只需在手机或电脑的浏览器（如百度、搜狗、360浏览器）中搜索这些名称即可直接使用。

我们以“淘金阁”为例进行讲解。如果你在搜索时发现广告太多，可以直接输入其网址访问。

![](img/72cc3aa23ecf464b5ecec893837649a6_1.png)

![](img/72cc3aa23ecf464b5ecec893837649a6_3.png)

进入淘金阁主页后，你会看到非常多的内容分类，例如财经、军事、动漫、科技、美食、美文、星座、体育、旅游、育儿、汽车、社会、娱乐、时尚、房产、国际、宠物等。

我们可以任意点开一个分类，例如“情感”类型。页面上会展示大量该类别下的话题文案，并且通常有多页内容可供浏览。

![](img/72cc3aa23ecf464b5ecec893837649a6_3.png)

![](img/72cc3aa23ecf464b5ecec893837649a6_5.png)

这五个平台提供的文案资源足够日常使用。在选择文案时，如果你感到困惑，可以套用一个简单的公式：**`一个特定群体 + 一个核心话题 + 一个疑问句式`**。例如，“宝妈们，你们是如何平衡工作和带娃的？”。觉得合适的文案就可以选用。

![](img/72cc3aa23ecf464b5ecec893837649a6_5.png)

通过这种方式，你将不再缺乏文案素材。

![](img/72cc3aa23ecf464b5ecec893837649a6_7.png)

## 使用关键词精准搜索文案

如果分类列表中没有你想要的特定内容，有两种解决方法：
*   有条件可以开通平台会员，使用更高级的搜索功能。
*   不想开通会员的同学，可以留意页面下方的“输入多个关键词”搜索框。

![](img/72cc3aa23ecf464b5ecec893837649a6_9.png)

例如，我们可以搜索“宝妈”这个关键词。

![](img/72cc3aa23ecf464b5ecec893837649a6_7.png)

![](img/72cc3aa23ecf464b5ecec893837649a6_11.png)

点击“开始抓取”后，页面会显示所有包含“宝妈”关键词的文案。这些结果通常足以满足需求。手机和电脑的操作方法完全相同，无需担心设备差异。

## 利用工具生成配音解决声音问题

解决了文案来源，我们再来解决另一个常见问题：担心自己声音不好听。淘金阁等工具提供了在线配音生成功能。

点击页面右上角的“小工具”图标，然后选择工具列表中的第四个选项：“配音在线生成”。

![](img/72cc3aa23ecf464b5ecec893837649a6_11.png)

![](img/72cc3aa23ecf464b5ecec893837649a6_15.png)

这个功能完全免费，且提供丰富的配音音色选择。

![](img/72cc3aa23ecf464b5ecec893837649a6_17.png)

![](img/72cc3aa23ecf464b5ecec893837649a6_13.png)

![](img/72cc3aa23ecf464b5ecec893837649a6_19.png)

以下是可供选择的音色类型示例：
*   **亲和女生、温柔女生、甜美女生**
*   **自然女声、卖场广播**
*   **亲切女声、活力女声（直播配音）**
*   **标准男声、标准女声（通用场景）**
*   **湖南口音、童声**
*   **英文配音**
*   **方言配音**：四川话、粤语、东北话、台湾话
*   **外语配音**
*   **精品男配**：情感解说、影视解说、娱乐解说等

![](img/72cc3aa23ecf464b5ecec893837649a6_15.png)

使用方法非常简单：
1.  在左侧“配音文本”框中输入或粘贴你准备好的文案。
2.  在右侧选择你喜欢的音色，例如选择“东北话-大虎”。
3.  调整音量、语速和语调参数。
4.  点击“立即生成”。

![](img/72cc3aa23ecf464b5ecec893837649a6_21.png)

在手机上使用时，可能需要用微信扫码登录一下，之后便能免费使用所有配音功能。

![](img/72cc3aa23ecf464b5ecec893837649a6_21.png)

![](img/72cc3aa23ecf464b5ecec893837649a6_23.png)

![](img/72cc3aa23ecf464b5ecec893837649a6_23.png)

生成完成后，在“配音生成列表”中，你可以直接下载生成的音频文件到手机。

![](img/72cc3aa23ecf464b5ecec893837649a6_25.png)

![](img/72cc3aa23ecf464b5ecec893837649a6_25.png)

在后续的剪辑环节中，你就可以直接使用这个配音文件。至此，你不仅解决了文案问题，连配音的难题也一并解决了。
