1.  **寻找目标视频**：在抖音中搜索一个包含文案的视频，例如“情感语录”。

2.  **复制视频链接**：选择一个合适的视频，复制其分享链接。
3.  **粘贴并提取**：将复制的链接粘贴到工具界面的输入框中，点击“提取文案”按钮。

4.  **获取结果**：稍等片刻，提取结果就会显示出来。例如，可能得到如下文案：

提取出的文案可以一键复制。工具还提供“格式化”功能，能将文案按照字幕的段落形式排列整齐，便于阅读和修改。此外，该工具也支持上传本地视频文件来提取文案。
