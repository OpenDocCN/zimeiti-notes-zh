# 自媒体运营教程：P5：2.平台现状 📊

在本节课中，我们将要学习自媒体行业的平台现状，了解其发展环境、政策支持、市场规模以及从业者与用户的比例，帮助你判断进入这个行业的可行性。

上一节我们介绍了自媒体的基本概念，本节中我们来看看这个行业所处的宏观环境。

## 行业环境与政策支持

现阶段整个社会非常浮躁，很多人都想赚钱。但之前有人采取不合法的手段（例如去缅甸）赚钱，这类行为是国家明令禁止的，对我们没有任何好处。

现阶段自媒体行业属于国家支持的行业。国家已将支持文件下达到各个地方政府。

以下是部分地方政府发布的政策性文件示例：
*   荆州市发布了“六项措施助力自媒体及新媒体高质量发展”的文件。
*   该文件由人社局于今年6月2日发布。
*   文件中指出要“加大政策宣传力度，有力支持自媒体以及新媒体行业的蓬勃发展”。

以下是其他地区发布的类似文件：
*   广西壮族自治区人民政府
*   四川省人民政府
*   浙江省人民政府
*   杭州市人民政府
*   江西省赣州市人民政府

这些文件都强调了“坚持正确导向，内容为王”。这意味着，未来作为自媒体创作者，我们创作的文章和视频是结算收益最重要的方式和途径。

既然国家已经下达了明确的政府文件支持，那么自媒体行业是完全可以来做的。

## 平台激励与资金投入

除了政策支持，自媒体平台也响应国家号召，出资鼓励创作者制作视频。

![](img/20a277ac3a19420db6e1f370bf35d6ca_6.png)

平台投入了巨额资金进行补贴。例如，西瓜视频曾宣布“未来一年将额外拿出**20亿元**补贴中视频创作人”。

平台创始人强调：
1.  这是真金白银的20亿元，不包含任何商单、直播和电商的收入，是**纯补贴**。
2.  平台会探索“保底+分成”模式，助力创作人职业化。
3.  20亿元是保底金额，上不封顶。

这意味着，创作者通过制作视频，可以直接从这20亿元的补贴池中获得收益。

## 市场规模与供需关系

现阶段自媒体行业有一个显著优势：用户数量非常庞大。

![](img/20a277ac3a19420db6e1f370bf35d6ca_8.png)

根据2022年中国研究生网站发布的市场调查与发展前景分析报告（发布于今年9月9日）：
*   我国现阶段**中国网民规模为9.4亿人**，接近10亿。
*   这9.4亿人就是观看视频的潜在用户。

那么，制作视频的人有多少呢？
*   截止去年，我国**全职**从事自媒体的人数为 **370万人**。
*   **兼职**从事自媒体的人数为 **600万人**。
*   两者相加，从事视频创作的总人数约为 **970万人**，可近似看作 **1000万人**。

由此我们可以得出一个简单的比例关系：
`视频创作者 : 视频观众 ≈ 1000万 : 9.4亿 ≈ 1 : 94`

这相当于每100个看视频的人中，只有大约1个人在做视频。一个人制作视频的速度，远远赶不上100个人观看视频的速度，**内容供给是严重不足的**。

这正是国家支持自媒体行业发展的重要原因。从我们观看设备的变迁（黑白电视 → 彩电 → 可联网的智能电视）也能看出，互联网视频是未来的发展趋势，用户规模还将持续扩张。

本节课中我们一起学习了自媒体行业的平台现状。我们了解到，该行业拥有坚实的国家政策支持、平台真金白银的投入、庞大的用户市场以及内容供不应求的广阔前景。这些因素共同构成了当前进入自媒体领域的有利环境。
