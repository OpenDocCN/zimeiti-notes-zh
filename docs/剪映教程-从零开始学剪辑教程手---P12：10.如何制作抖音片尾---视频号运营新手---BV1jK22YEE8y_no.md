# 剪映教程：P12：10. 如何制作抖音片尾 🎬

![](img/78468e7da57b70cd81414d61ee146784_3.png)

在本节课中，我们将学习如何使用剪映手机版制作一个抖音风格的专属片尾。这种片尾通常会在视频最后出现，展示你的头像并引导观众关注，是提升账号辨识度和互动率的有效方式。

![](img/78468e7da57b70cd81414d61ee146784_7.png)

![](img/78468e7da57b70cd81414d61ee146784_9.png)

上一节我们介绍了视频剪辑的基础操作，本节中我们来看看如何利用画中画和混合模式功能，组合素材制作出吸引人的片尾效果。

![](img/78468e7da57b70cd81414d61ee146784_10.png)

![](img/78468e7da57b70cd81414d61ee146784_12.png)

## 操作步骤详解

![](img/78468e7da57b70cd81414d61ee146784_14.png)

![](img/78468e7da57b70cd81414d61ee146784_15.png)

![](img/78468e7da57b70cd81414d61ee146784_17.png)

以下是制作抖音专属片尾的具体步骤。

![](img/78468e7da57b70cd81414d61ee146784_19.png)

![](img/78468e7da57b70cd81414d61ee146784_20.png)

1.  **创建项目并导入背景**
    *   打开剪映，点击“创作”开始一个新项目。
    *   在素材库中选择一个**白底素材**作为背景导入到主轨道。

![](img/78468e7da57b70cd81414d61ee146784_22.png)

![](img/78468e7da57b70cd81414d61ee146784_24.png)

![](img/78468e7da57b70cd81414d61ee146784_26.png)

2.  **添加人物素材（第一层画中画）**
    *   点击下方工具栏的“画中画”功能，选择“新增画中画”。
    *   从相册中选择一张人物照片或一段视频导入。这里以一位小姐姐的素材为例。
    *   将画中画素材放大，并移动到画面中合适的位置。

![](img/78468e7da57b70cd81414d61ee146784_28.png)

![](img/78468e7da57b70cd81414d61ee146784_30.png)

3.  **剪辑与调整人物素材**
    *   如果素材是视频，我们只需要其中人物微笑的部分。使用分割工具，将需要的片段裁剪出来。
    *   删除前后不需要的部分。
    *   再次调整人物素材的位置和大小。

![](img/78468e7da57b70cd81414d61ee146784_32.png)

![](img/78468e7da57b70cd81414d61ee146784_34.png)

4.  **设置人物素材的混合模式**
    *   选中人物画中画轨道，在下方工具栏中找到“混合模式”选项。
    *   将混合模式设置为 **`变暗`**。
    *   此时，人物素材会与白色背景融合，只留下人物主体轮廓。根据效果，可再次微调其大小和位置。

![](img/78468e7da57b70cd81414d61ee146784_36.png)

![](img/78468e7da57b70cd81414d61ee146784_38.png)

5.  **添加片尾模板素材（第二层画中画）**
    *   再次点击“画中画”，新增一个画中画轨道。
    *   导入准备好的**黑底片尾模板素材**（通常包含“关注”箭头、文字等元素）。
    *   将模板素材拖放到人物素材上方合适的位置。

![](img/78468e7da57b70cd81414d61ee146784_40.png)

![](img/78468e7da57b70cd81414d61ee146784_42.png)

6.  **设置模板素材的混合模式**
    *   选中模板素材轨道，进入“混合模式”设置。
    *   将混合模式改为 **`绿色`**。此模式会过滤掉黑色背景，只保留白色或彩色的图形元素。
    *   调整模板素材的大小，使其与人物头像协调匹配。

    > **提示**：如果模板尺寸不合适，可以尝试先将其导入主轨道，然后使用“切画中画”功能将其转为画中画，再进行混合模式（`绿色`）和大小调整。

![](img/78468e7da57b70cd81414d61ee146784_44.png)

![](img/78468e7da57b70cd81414d61ee146784_45.png)

![](img/78468e7da57b70cd81414d61ee146784_46.png)

7.  **最终预览与调整**
    *   播放预览整个片尾效果，检查人物与模板元素的位置、大小是否和谐。
    *   进行最后的细微调整，确保视觉效果整洁美观。

![](img/78468e7da57b70cd81414d61ee146784_48.png)

![](img/78468e7da57b70cd81414d61ee146784_50.png)

## 课程总结与作业

![](img/78468e7da57b70cd81414d61ee146784_52.png)

![](img/78468e7da57b70cd81414d61ee146784_53.png)

本节课我们一起学习了制作抖音专属片尾的全过程。核心是利用**两层画中画**和不同的**混合模式**（`变暗` 与 `绿色`）来合成素材，从而创造出引导关注的动态效果。

**课后作业**：请根据本课所学，使用你自己的头像照片和提供的片尾模板素材，制作一个个性化的“求关注”抖音片尾。你可以在课程配套的素材库中下载所需的模板进行练习。

![](img/78468e7da57b70cd81414d61ee146784_55.png)

![](img/78468e7da57b70cd81414d61ee146784_57.png)

通过这个简单的制作，你就能为你的视频增加一个专业的结尾，有效提升粉丝的关注度。
