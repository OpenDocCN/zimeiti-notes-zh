# 小红书开店教程：P6：4.小红书店铺账号测试 🧪

在本节课中，我们将要学习如何为小红书店铺测试账号权重。这是开店前至关重要的一步，能帮助我们筛选出有潜力的账号，避免在无权重账号上浪费时间和精力。

![](img/e084e67d350d83122b9c3decfa250244_1.png)

上一节我们介绍了开店的基础准备，本节中我们来看看如何科学地测试账号。

![](img/e084e67d350d83122b9c3decfa250244_3.png)

![](img/e084e67d350d83122b9c3decfa250244_5.png)

小红书的店铺账号与其他平台不同，并非所有新创建的账号都具备基础权重。无论是使用新手机号、新身份证注册，还是使用注册多年的老号，都存在一定概率（约5%-10%）的账号本身没有权重。如果使用无权重账号运营店铺，后续所有努力都可能无效。因此，在正式开店前，必须对账号进行测试。

![](img/e084e67d350d83122b9c3decfa250244_7.png)

## 测试核心原理与时机

账号测试的核心是观察笔记的自然流量数据。测试必须在**开店注册之前**进行。我们通过发布笔记，观察其初始流量表现，来判断账号是否具备基础权重，而不是等店铺所有信息都完善后再测试。

操作方式比较简单，准备2-3个账号即可。测试周期约为3到5天。

以下是具体的操作步骤：

1.  **发布测试笔记**：使用每个待测试的账号，发布一篇原创内容（视频或图文均可）。内容无需复杂，可以是产品介绍或日常分享。
2.  **观察初始流量**：笔记发布后约2分钟内，系统会给予少量“强推流量”，此时“小眼睛”（阅读量）可能达到20以上，但这并非有效判断依据。
3.  **分析持续数据**：我们需要观察笔记在**3天以内**的持续自然流量。这是判断账号权重的关键。

## 账号测试合格标准 📊

判断账号是否有权重的数据标准如下：
*   **每日标准**：笔记每日自然增长的小眼睛数量需超过 `50`。
*   **周期标准**：3天后，该笔记累计的小眼睛总数需超过 `150`。

满足以上标准，则说明该账号具备基础权重，适合用于开店。

## 测试必须注意的要点 ⚠️

以下是进行账号测试时必须严格遵守的几个关键点，任何一点疏忽都可能导致测试失败：

1.  **内容必须原创**：测试笔记的内容**必须是原创**。如果是视频，时长需超过15秒（建议20秒左右）。直接复制他人内容，即使账号有权重，该笔记也可能无法获得流量。
2.  **禁止同设备ID测试**：不能在**同一部手机**上通过换SIM卡来测试多个账号。小红书系统识别的是设备ID，而非手机卡。必须使用不同的手机设备进行测试。
3.  **禁用同一Wi-Fi网络**：测试期间，所有测试手机**不能连接同一个Wi-Fi网络**，必须使用各自的手机移动数据流量。如果测试手机曾连接过同一个Wi-Fi，需断开该Wi-Fi并等待至少一周后，再使用流量进行测试，否则账号数据会关联，导致测试无效。
