## 概述
在本节课中，我们将学习如何为TikTok等海外平台手动搬运并二次剪辑视频素材。这是跨境电商内容创作的基础步骤，核心在于获取无水印素材并进行处理，使其适合海外发布。

## 搬运方式简介
上一节我们介绍了跨境电商的内容策略，本节中我们来看看如何具体获取视频素材。搬运视频主要有两种方式：手动搬运和使用工具批量搬运。工具搬运效率更高，而手动搬运更适合初学者，只需一部手机即可操作。

## 手动去水印教程
接下来，我们以“茶杯犬”相关视频为例，演示完整的手动去水印流程。

**第一步：复制视频链接**
1.  在抖音找到目标视频。
2.  点击视频右侧的**分享**按钮。
3.  在弹出菜单中，向左滑动找到并点击 **“复制链接”**。

![](img/dc2f23ce29f6c2784cabe477e34b23d2_17.png)

**关键操作**：`点击分享按钮 -> 滑动找到“复制链接”`

![](img/dc2f23ce29f6c2784cabe477e34b23d2_19.png)

![](img/dc2f23ce29f6c2784cabe477e34b23d2_20.png)

**第二步：使用工具解析**
1.  打开微信，在搜索框内搜索小程序 **“青抖”**。
2.  进入“青抖”小程序后，找到并点击 **“去水印”** 功能。
3.  系统通常会自动识别剪贴板中的链接，点击 **“确定”** 开始解析。
4.  解析完成后，你将获得一个去除了平台水印（如左上角/右下角图标）的高清视频。

![](img/dc2f23ce29f6c2784cabe477e34b23d2_22.png)

![](img/dc2f23ce29f6c2784cabe477e34b23d2_23.png)

![](img/dc2f23ce29f6c2784cabe477e34b23d2_25.png)

**关键操作**：`微信搜索“青抖” -> 使用“去水印”功能 -> 粘贴链接解析`

**第三步：保存与后续处理**
1.  在“青抖”小程序中，点击 **“保存到相册”**，将处理好的视频下载到手机本地。
2.  视频保存后，即可使用剪映等剪辑软件进行二次创作，为发布到海外平台做准备。
