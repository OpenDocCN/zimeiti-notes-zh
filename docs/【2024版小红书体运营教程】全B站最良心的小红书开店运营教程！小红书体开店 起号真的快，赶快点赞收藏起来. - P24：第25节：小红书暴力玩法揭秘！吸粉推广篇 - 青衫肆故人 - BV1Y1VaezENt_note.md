# 小红书运营教程：P24：小红书暴力玩法揭秘！吸粉推广篇 💥

在本节课中，我们将要学习小红书账号矩阵的构建与官方广告投放的两种核心推广方式，帮助你理清自建账号推广与官方广告投放的关系。

## 账号类型解析

上一节我们介绍了推广的整体思路，本节中我们来看看小红书平台上的账号具体有哪些类型。小红书的账号主要可以分为以下三类。

以下是三类账号的详细说明：

1.  **企业号**：主要作用是树立官方品牌形象。企业号拥有官方认证标志，可以展示品牌名称、地址等信息，并享有平台赋予的特权，便于开展业务。
2.  **个人号**：这是目前大多数用户运营的账号类型。个人号的特点是**灵活性高**，可以是品牌员工或消费者身份，更容易塑造个人品牌特色，也适合品牌相关人物出镜。
3.  **引流号**：这类账号在之前的课程中也有提及，俗称“炮灰号”。其作用通常是辅助推广，用完即可舍弃，或用于应对紧急情况。引流号一般没有精细的个性规划。

通常，品牌的推广渠道依赖于企业号、个人号或引流号自行发布笔记进行引流。

## 企业号矩阵运营建议

了解了账号分类后，我们来看看如何有效运营它们。对于希望系统化运营的品牌，构建企业号矩阵是一个高效策略。

![](img/67a86ed507efe51e105e43eaf29b73c5_1.png)

以下是打造企业号矩阵的四点具体建议：

![](img/67a86ed507efe51e105e43eaf29b73c5_3.png)

![](img/67a86ed507efe51e105e43eaf29b73c5_5.png)

1.  **积极参与品牌号活动**：例如，婚纱、服装、美妆类企业号发起的试用或团购活动，可以动用多个小号（引流号或个人号）去参与，增加活动热度。
2.  **个人号展示品牌属性**：在个人号的笔记内容中，应多次暗示品牌的定位。例如，品牌目标用户是都市年轻人还是职场新人，需要在内容中清晰体现。
3.  **控制个人号营销比例**：个人号发布品牌产品内容的频率不宜过高。建议采用 **“三七分”原则**：即30%的内容用于营销推广，70%的内容用于种草分享或知识科普。
4.  **利用引流号进行评论引流**：当企业号发布内容后，可以使用引流号在评论区进行互动和引导，以实现更隐蔽和有效的引流。

## 官方广告投放介绍

![](img/67a86ed507efe51e105e43eaf29b73c5_9.png)

![](img/67a86ed507efe51e105e43eaf29b73c5_11.png)

除了运营自有的账号矩阵，另一种重要的推广方式是借助平台力量。接下来，我们将目光转向小红书官方的广告投放系统。

![](img/67a86ed507efe51e105e43eaf29b73c5_13.png)

小红书官方广告，即竞价广告，是平台提供的付费推广服务。要使用此服务，首先需要认证一个企业号。

企业号认证流程如下：
1.  访问小红书官方网站。
2.  找到“业务合作”板块。
3.  进入“推广合作”页面，按照指引提交企业资质并支付审核费用。
4.  审核通过后，普通账号即升级为官方认证的企业号。

完成认证后，便可投放信息流广告。官方广告主要分为两大类：

1.  **信息流广告**：当用户浏览小红书首页信息时，系统会不定期间隔插入推广内容。
2.  **搜索广告**：当用户使用关键词进行搜索时，与关键词相关的推广内容会出现在搜索结果中。其原理类似于百度的关键词竞价排名，通过**调整出价**来竞争不同的广告展示位。

目前，信息流广告常见的展示位置有第3位、第13位和第23位等。关于广告投放的更多常见问题，可以在小红书官方平台的帮助文档中查询。

![](img/67a86ed507efe51e105e43eaf29b73c5_15.png)

需要注意的是，官方广告投放的费用相对较高。因此，在资金允许的情况下，建议考虑与官方合作投放竞价广告，因为它能直接获得平台推荐的精准流量。

![](img/67a86ed507efe51e105e43eaf29b73c5_17.png)
