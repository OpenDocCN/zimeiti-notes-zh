# 视频号运营教程：P27：7.如何给视频抽帧补帧 🎬

在本节课中，我们将学习视频剪辑中的一个实用技巧——抽帧与补帧。这个技巧可以帮助我们微调视频时长，同时保持视频内容的流畅性，是视频二次创作（如混剪）时常用的方法。

上一节我们介绍了视频剪辑的基础操作，本节中我们来看看如何通过抽帧和补帧来精细调整视频。

## 操作步骤详解

首先，将视频素材导入剪辑软件，并将其时间线拉伸到最精细的刻度，以便进行帧级别的操作。通常，视频的每一秒包含**30帧**（30F）。

以下是抽帧的具体步骤：

1.  在时间线上，定位到你想要删除的某一帧。例如，找到第20帧（20F）的位置。
2.  在此帧处进行**分割**操作。
3.  紧接着，在第21帧（21F）的位置再次进行**分割**。这样，你就单独选中了这一帧的画面。
4.  将这一小段（即这一帧）**删除**。这个删除某一帧画面的过程，就叫做“抽帧”。

对于一个完整的视频，通常抽取**3帧**左右即可，这样对视频整体的观感影响最小。

完成抽帧后，我们来看看如何进行补帧。补帧可以平衡因抽帧而减少的时长，或用于其他创意目的。

以下是补帧的具体步骤：

1.  在视频的其他部分，选择一段时长为一帧的画面。
2.  对该片段进行**复制**操作。
3.  将复制出来的帧片段，粘贴到视频中你希望添加的位置。这个复制并添加帧的过程，就是“补帧”。

同样，补帧的数量一般与抽帧数量对应，例如补**3帧**。通过这样的抽帧与补帧操作，视频的总时长和主体内容不会发生明显变化，但能实现我们调整节奏或规避重复度的目的。

本节课中我们一起学习了视频剪辑中的抽帧与补帧技巧。关键要点是：将视频放大至帧级别（**30帧/秒**），通过**分割**与**删除**来抽帧，并通过**复制**片段来补帧。合理运用这个技巧，可以在不影响视频流畅度的前提下，对视频进行微调。请在你的视频剪辑实践中尝试应用这个方法。
