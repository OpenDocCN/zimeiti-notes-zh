## 平台对原创内容的支持

上一节我们介绍了发布作品的基础，本节中我们来看看平台对内容的态度。平台本身支持原创，并设有创作者激励计划，为原创内容提供流量倾斜。

以下是创作者激励计划的具体扶持措施：
*   **流量扶持**：平台计划让1000万原创作者获得流量。
*   **收入激励**：让百万优质创作者获得收入。
*   **零粉阶段**：赠送2500到5万点的流量券，即直接推送观众。
*   **千粉阶段**：可加入重点作者扶持计划，优质视频可申报专项奖金。
*   **万粉及百万粉阶段**：可申报的专项奖金更多。

因此，原创内容能获得平台一级一级的流量池推进。但这需要制作者具备相应的技术能力，如拍摄、编写台本等。

## 伪原创的核心流程

伪原创并非直接下载并发布，平台支持原创，直接搬运带水印的内容可能导致账号违规。因此，我们需要进行“伪”创作。

以下是进行伪原创的核心步骤：

**第一步：选品与定位**
首先，你需要确定要售卖的商品或账号的定位。可以通过以下方式寻找灵感：
*   进入视频号“创作者中心”。
*   参考其他电商平台，如阿里巴巴1688、拼多多等。
*   利用数据平台（如西瓜数据、友望数据、微信指数）查看视频号排行榜，分析各类目（如影视、音乐）下的优质账号，借鉴其优点。

![](img/aea3a5f1f319e1bad90a43f2dca9ea04_8.png)

**公式：账号定位 = 目标商品/服务 + 内容类型**

![](img/aea3a5f1f319e1bad90a43f2dca9ea04_9.png)

例如，若计划售卖宠物用品，则定位为宠物类账号；若计划售卖车载U盘，则可定位为音乐分享账号。

**第二步：寻找并验证素材**
确定定位后，去其他平台（如抖音）寻找相关的优质账号和视频素材。关键一步是验证该素材在视频号平台是否已被大量搬运。如果已有大量重复内容，则应避免再次使用。

**第三步：下载与去水印**
找到合适的素材后，不能直接下载发布。首先需要使用工具去除原视频的水印。

**第四步：二次剪辑**
去水印后，必须对视频进行二次剪辑，以规避平台查重。这包括但不限于：调整片段顺序、添加转场特效、重新配乐、添加字幕或画中画、改变播放速度等。

**代码示例（伪代码）：**
```
# 伪原创处理流程
1. 定位 = 选择目标领域(例如：宠物)
2. 素材 = 在外部平台搜索优质视频(定位)
3. if 检查视频号重复度(素材) == 低：
4.     去水印视频 = 下载并去除水印(素材)
5.     最终作品 = 二次剪辑(去水印视频) # 包括裁剪、变速、加滤镜等操作
6.     发布作品(最终作品)
7. else：
8.     重新寻找素材
```
