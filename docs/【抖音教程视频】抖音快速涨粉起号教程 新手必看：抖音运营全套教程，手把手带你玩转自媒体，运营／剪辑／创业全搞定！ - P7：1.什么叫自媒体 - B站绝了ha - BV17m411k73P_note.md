# 自媒体运营入门：P1：什么是自媒体

在本节课中，我们将要学习自媒体的核心概念、课程的整体结构，并清晰地理解“自媒体”究竟是什么。课程内容将分为四个主要板块，为后续的深入学习打下基础。

## 课程内容概述

我将今天的课程内容划分为四个核心板块。

以下是四个板块的具体介绍：

![](img/6c29a8eb4706f8f0ef05be89191dccc1_1.png)

1.  **自媒体现状与前景**：无论从事任何行业，全职或兼职，都需要清楚该行业的现状与前景。了解清楚后，我们再着手操作会更放心。
2.  **零基础打造高收益账号**：很多同学可能遇到过视频发布后零播放，或在抖音、快手等平台无法通过播放量结算收益的情况。本板块将教授如何选择平台、领域和渠道，从而将收益赚到手。
3.  **亿万流量的多种变现玩法**：除了依靠视频或文章的播放量结算收益，自媒体还有许多其他变现方式。本板块将介绍这些适合零基础同学上手的玩法。
4.  **自媒体变现的正确打开方式**：如果你听完课程后觉得自媒体适合自己并有时间操作，本板块将教你如何规划个人账号，并系统性地将收益赚到手。

以上是今天课程内容的整体划分。

## 什么是自媒体？

上一节我们介绍了课程的整体框架，本节中我们来看看最核心的概念：到底什么是自媒体？

![](img/6c29a8eb4706f8f0ef05be89191dccc1_5.png)

我用最通俗的话来解释：自媒体就是，我们今天打开手机，在百度上搜索一个自媒体平台，然后去该平台注册一个属于我们自己的账号。在我们自己的这个账号上，发布我们自己创作的文章或视频。当有人观看我们的内容时，我们就可以依靠这个播放量来结算收益。

![](img/6c29a8eb4706f8f0ef05be89191dccc1_7.png)

这个过程，就叫做**自媒体**。

![](img/6c29a8eb4706f8f0ef05be89191dccc1_9.png)

用简单的公式来描述其核心流程就是：
**注册个人账号 -> 发布原创内容 -> 获取观看/阅读 -> 产生收益**
