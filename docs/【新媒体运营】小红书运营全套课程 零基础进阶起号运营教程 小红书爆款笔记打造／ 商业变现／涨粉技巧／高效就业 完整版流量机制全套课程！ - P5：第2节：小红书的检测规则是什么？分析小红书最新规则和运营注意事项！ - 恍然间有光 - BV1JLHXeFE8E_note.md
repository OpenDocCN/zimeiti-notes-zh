# 小红书运营课程：P5：第2节：小红书的检测规则是什么？🔍

在本节课中，我们将详细解析小红书的平台检测规则与运营注意事项。了解这些规则是避免账号违规、确保内容安全发布的基础。

## 平台检测的核心范围

小红书系统主要对以下四个方面的内容进行检测。

以下是具体的检测项目列表：
*   **个人资料**：账号简介、头像等个人信息。
*   **笔记内容**：发布的图文或视频笔记正文。
*   **私信与评论**：用户之间的私信沟通及笔记下的公开评论。

在这些范围内，平台严格禁止出现以下信息：微信号、微信二维码、代购、水印以及推广淘宝店铺等营销词汇。

不过，平台管理相对宽松的方面是：官方允许在个人资料中留下邮箱地址。同时，运营者可以使用“微博”等词汇进行谐音替代来间接引流，再通过话术引导用户添加微信。具体的引流方法与话术技巧，我们将在后续课程中深入讲解。

## 设备与行为规范

上一节我们介绍了内容层面的检测，本节中我们来看看账号与行为相关的规则。

首先，平台会检测**手机设备**。建议同一台设备最多只登录三个小红书账号，这一点需要特别注意。

其次，在笔记内容中，**不得诱导用户**。例如，“关注领奖”、“点赞抽奖”这类活动是小红书明令禁止的，它们属于违规的诱导行为。

## 内容原创度要求

小红书平台对内容的**原创度保护程度较高**。因此，在推广时需要注意：一篇文章仅在一个账号发布即可，切勿将同一篇文章在多个账号上重复发布。

## 敏感词检测与处理

除了上述规则，小红书还有一个重要的检测机制：**敏感词检测**。涉及手机卡、金融、电子烟、代购、医疗用品等领域的词汇，应尽量避免触碰。

![](img/cb2f135190af46e6067cd034823d677f_1.png)

那么，在撰写笔记时，如果不慎加入了未知的敏感词该怎么办？这时就需要借助敏感词检测工具。

![](img/cb2f135190af46e6067cd034823d677f_3.png)

以下是两个实用的敏感词检测网站：
1.  第一个网站（无需登录）：将写好的文案粘贴进去，点击“立即检测”，即可高亮显示敏感词。
    ![](img/cb2f135190af46e6067cd034823d677f_1.png)
2.  第二个网站（需注册）：使用方式类似，注册后即可使用，会给出过滤结果。
    ![](img/cb2f135190af46e6067cd034823d677f_3.png)

我们通常使用第一个网站进行检测就足够了。只需将文章内容粘贴到检测框中，网站便会自动识别出文中的敏感词。
