## 什么是搜索排名？

上一节我们介绍了课程的整体框架，本节中我们来看看什么是搜索排名。搜索排名指的是当用户在平台（如B站、小红书、百度等）的搜索框中输入特定词汇后，系统根据一系列规则，将相关内容按顺序展示给用户的结果列表。

例如，在B站搜索框输入“英语口语”并点击搜索后，页面会展示一系列相关内容。

![](img/8926fd86b84c908e2ac3c13c11d04882_1.png)

这些展示的结果就是搜索排名。我们需要先排除几类特殊内容：

![](img/8926fd86b84c908e2ac3c13c11d04882_3.png)

以下是需要排除的几类内容：
1.  **平台官方课程**：例如图中第一项，这是B站上架的付费课程，不属于普通博主可竞争的自然流量排名。
2.  **广告推广内容**：例如图中带有“创作推广”标识的内容，这是通过付费广告投放获得的位置，不属于自然排名。一旦停止付费，排名就会消失。
3.  **直播内容**：我们本节课主要讨论自然视频内容的排名。

排除以上内容后，剩下的视频列表（如“YOUTUBE口语精选”、“easy english系列”等）就是基于平台算法得出的**自然搜索排名**。排名越靠前的内容，获得用户点击和观看的概率就越大。

**核心结论**：在搜索场景下，内容排名越靠前，获得的流量占比就越高。这个原理适用于所有具备搜索功能的平台，包括图文、短视频及搜索引擎。

## 关键词：用户需求的入口

理解了搜索排名的场景，接下来必须掌握的核心概念是**关键词**。关键词是用户表达确切搜索需求的词汇。

例如，我们刚才搜索的“英语口语”就是一个关键词。用户只有在有明确目的时才会进行搜索，这与基于兴趣的随机浏览行为有本质区别。

为了更有效地运营，我们需要对关键词进行分类。以下是主要的分类方式：

### 核心关键词与长尾关键词

行业内的关键词通常分为**核心关键词**（核心词）和**长尾关键词**（长尾词）。

*   **核心关键词**：通常是搜索量较大、范围较广的词汇，能代表一个主要领域，但用户意图不够精确。
    *   **示例公式**：`核心词 ≈ 行业或领域宽泛名称`
*   **长尾关键词**：由核心词衍生而来，通常更长、更具体，能精准反映用户的细分需求。
    *   **示例公式**：`长尾词 = 核心词 + 修饰词（如：怎么练、零基础、方法、课程）`

我们以“英语培训”行业为例进行分析。假设有两个词：“英语”和“英语口语”。

*   **“英语”**：这是一个非常宽泛的学科名词。搜索“英语”的用户意图很多元（如了解考试、专业、音标），无法判断其是否有学习提升的明确需求。因此，**“英语”不适合作为培训机构的精准核心词**。
*   **“英语口语”**：这个词范围收窄，指向了“英语”的一个具体方面。搜索它的用户，大概率有提升口语能力的需求。因此，**“英语口语”可以作为该机构的一个核心关键词**。

然而，仅凭“英语口语”这个核心词，仍无法完全洞察用户的具体痛点。这时就需要借助长尾词。

## 长尾关键词：洞察用户具体需求

长尾词是理解用户真实需求的关键。它们由核心词衍生，描述了更具体的场景、问题或解决方案。

以下是基于核心词“英语口语”可能衍生出的部分长尾词示例：
*   英语口语怎么练
*   英语口语练习方法
*   英语口语零基础
*   英语口语课程

每个长尾词都对应着一类用户的明确需求：
1.  搜索“**英语口语怎么练**”的用户，需要练习方法和指导。
2.  搜索“**英语口语零基础**”的用户，需要针对初学者的学习路径。
3.  搜索“**英语口语课程**”的用户，可能有较强的付费学习意向。

我们可以根据不同的长尾词，创作更具针对性的内容标题：
*   针对“练习方法”：`最高效的英语口语练习方法，一般人不知道的3个技巧`
*   针对“零基础”：`零基础30天快速提升英语口语能力的黄金法则`

长尾词还可以进一步细化。例如，从“英语口语练习”这个长尾词，又能衍生出更具体的词汇：

## 关键词体系搭建示例

我们可以通过一个结构图来理解关键词之间的关系：

```
核心关键词1：英语口语
├─ 长尾关键词1.1：英语口语怎么练
│  └─ 衍生词：英语口语练习软件、英语口语练习素材...
├─ 长尾关键词1.2：英语口语零基础
│  └─ 衍生词：零基础自学、零基础速成、零基础入门...
└─ 长尾关键词1.3：英语口语课程

核心关键词2：英语语法
├─ 长尾关键词2.1：英语语法零基础
└─ 长尾关键词2.2：英语语法填空技巧
```

**核心逻辑**：通过核心词定位赛道，通过长尾词洞察用户的具体问题，从而创作出能够精准匹配需求、解决痛点的内容。这是决定新媒体运营方向的基础。

## 本节课总结

本节课中我们一起学习了：
1.  **搜索排名的定义**：用户在平台搜索后，内容按规则展示的顺序列表，排名直接影响流量获取。
2.  **关键词的概念**：用户表达搜索意图的词汇，是连接需求与内容的桥梁。
3.  **关键词的分类**：重点区分了**核心关键词**（宽泛、代表主领域）和**长尾关键词**（具体、反映细分需求）。
4.  **关键词体系的作用**：核心词用于定位内容方向，长尾词用于精准洞察用户痛点，指导具体内容创作。

![](img/8926fd86b84c908e2ac3c13c11d04882_9.png)

理解并掌握关键词的划分，是后续进行内容优化、提升搜索排名的第一步。
