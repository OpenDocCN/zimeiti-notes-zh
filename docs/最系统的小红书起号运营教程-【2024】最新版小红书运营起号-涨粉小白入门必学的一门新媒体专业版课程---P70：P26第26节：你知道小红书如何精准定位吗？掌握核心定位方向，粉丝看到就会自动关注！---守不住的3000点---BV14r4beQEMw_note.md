# 小红书运营教程：P26：小红书账号类型与官方广告解析

在本节课中，我们将学习小红书账号的三种主要类型，并对比自建账号运营与投放官方广告两种推广方式的区别与联系。课程将帮助你理清思路，为你的运营策略提供明确方向。

## 账号类型解析

上一节我们介绍了账号定位的重要性，本节中我们来看看小红书平台具体有哪些账号类型。了解不同类型的账号及其用途，是制定有效运营策略的基础。

小红书账号主要分为以下三类：

**1. 企业号**
企业号的核心作用是树立官方品牌形象。它拥有官方认证标志，并享有特定特权，例如展示品牌名称、地址等信息，便于开展商业活动。

**2. 个人号**
个人号是目前大多数用户运营的账号类型。其特点是灵活性高，运营者可以是品牌员工或消费者。个人号能更好地塑造个人品牌特色，适合品牌相关人物出镜进行内容创作。

**3. 引流号**
引流号，俗称“炮灰号”，其作用是在特定推广场景下辅助引流。这类账号通常没有精细的个性规划，可以随时创建或弃用，常用于应对紧急情况或作为辅助推广渠道。

通常，品牌的推广内容会通过企业号、个人号或引流号发布笔记来实现引流。

## 企业号矩阵运营建议

在明确了账号分类后，如何协同使用它们呢？以下是打造企业号矩阵的一些具体建议：

**1. 积极参与品牌活动**
品牌号发起的活动（如试用、团购），应使用个人小号积极参与，以增加活动热度与真实性。

**2. 个人号展示品牌属性**
个人号应在笔记内容中，巧妙暗示品牌的定位与属性。例如，品牌若定位为“都市职场年轻人”，个人号内容就应围绕相关场景展开。

**3. 控制营销内容比例**
个人号发布品牌产品相关内容时，需注意频率与比例。建议遵循 **营销内容 : 种草/科普内容 = 3 : 7** 的原则，即30%为营销内容，70%为提供价值的干货内容。

**4. 利用引流号进行评论互动**
企业号若需直接引流，可配合使用引流号，在相关笔记的评论区进行引导，这种方式更为隐蔽和自然。

## 官方广告（CBC竞价广告）介绍

![](img/8f73b2b9aa8737b4d765fc1f4604ac86_7.png)

除了运营自建账号矩阵，另一种重要的推广方式是投放小红书官方广告。下面我们来详细了解这种付费推广模式。

![](img/8f73b2b9aa8737b4d765fc1f4604ac86_9.png)

小红书官方广告即CBC（Cost By Click）竞价广告。广告主通过竞价购买广告位，当用户点击广告时产生费用。

![](img/8f73b2b9aa8737b4d765fc1f4604ac86_11.png)

### 广告投放前提与流程

![](img/8f73b2b9aa8737b4d765fc1f4604ac86_13.png)

投放官方广告有一个必要前提：**必须先认证企业号**。

企业号认证流程如下：
1.  访问小红书官网。
2.  找到“业务合作”板块。
3.  进入“推广合作”或“企业号入驻”流程。
4.  按照指引提交企业资质并支付审核费用。
5.  通过认证后，普通账号即升级为官方企业号。

### 广告类型与展示位

完成企业号认证后，即可投放信息流广告。主要分为两大类：

**1. 信息流广告**
当用户浏览小红书首页“发现”页时，系统会在信息流中不定时插入广告。这是最常见的广告形式。

**2. 搜索广告**
当用户使用小红书搜索功能时，在搜索结果中展示的广告。其原理类似于百度的关键词竞价排名。

搜索广告的展示逻辑由平台算法决定，系统会根据用户搜索的关键词，智能匹配并推荐相关商品或内容。例如，用户搜索“考研资料”，系统可能推荐能量补充类商品。

![](img/8f73b2b9aa8737b4d765fc1f4604ac86_15.png)

目前，信息流广告的主要展示位有：**第3位、第13位、第23位**。广告主通过竞价争夺这些广告位，出价越高，获得优质展示位的机会越大。

![](img/8f73b2b9aa8737b4d765fc1f4604ac86_17.png)

> 注：关于广告投放的详细规则、费用及常见问题，可在小红书官网的“推广合作”板块查阅官方文档与问答。官方广告费用较高，建议在资金允许的情况下考虑。

![](img/8f73b2b9aa8737b4d765fc1f4604ac86_19.png)

## 课程总结

本节课中我们一起学习了小红书账号的三大类型（企业号、个人号、引流号）及其核心用途，并探讨了构建企业号矩阵的运营建议。同时，我们介绍了另一种重要的推广方式——小红书官方CBC竞价广告，包括其投放前提、类型及展示逻辑。理解自运营与付费推广的区别与联系，能帮助你更全面地规划小红书营销策略。
