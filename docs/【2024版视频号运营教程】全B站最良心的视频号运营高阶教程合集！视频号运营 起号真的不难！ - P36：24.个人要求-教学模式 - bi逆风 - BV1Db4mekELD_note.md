## 核心要求详解

以下是成为学员必须满足的三个核心要求。

### 1. 时间要求 ⏰

![](img/4f92ea2f0656ab87fa787f8e22c138b3_3.png)

时间是学习与操作的基础。无论身份如何，每天都需要保证固定的学习与操作时间。

*   **最低要求**：每天至少**30分钟**。这个时间足以学习并操作“创作者分成计划”这一种玩法。
*   **全职或进阶要求**：每天**1小时以上**。这样可以同时操作多种玩法，例如“创作者分成计划”结合“视频带货”或“无人直播”，以实现更高收益。

**收益公式参考**：
*   创作者分成计划：`日收益 ≈ 80元` → `月收益 ≈ 80元/天 * 30天 = 2400元`
*   视频带货（参与活动）：`日收益 ≈ 300元` → `月收益 ≈ 300元/天 * 30天 = 9000元`
*   （注：以上为示例数据，实际收益因人而异。）

### 2. 执行力要求 ⚡

执行力是取得成果的关键。老师出身部队，作风强调“快、狠、准”，最忌讳拖拉与懒惰。

![](img/4f92ea2f0656ab87fa787f8e22c138b3_5.png)

*   **必须做到**：不拖延、不偷懒、行动迅速。
*   **互动体现**：积极的课堂互动（如按要求在公屏打字）是执行力最直接的体现。无法完成简单互动指令的学员，很难谈得上有执行力。

### 3. 坦诚沟通要求 🤝

为了让教学方案最适合你，必须向老师如实说明个人情况。

![](img/4f92ea2f0656ab87fa787f8e22c138b3_7.png)

*   **需要说明的信息包括**：
    *   身份（如：宝妈、上班族、学生、退休人员）。
    *   每日可支配的时间段。
    *   学习能力（如：是否觉得学得慢、是否识字）。
    *   账号状态（新号/老号）。
    *   特殊目标（如：想全职投入）。
*   **教学承诺**：老师会根据你的具体情况（如零散时间、学习速度慢、不识字等）量身定制教学方案，甚至采用“笨办法”、“死办法”确保每个人都能学会。从陌生到熟练，核心在于**反复练习**。

**练习量示例**：
```python
# 如果别人每天练习1遍
others_practice_per_day = 1

# 你每天多练习2遍
your_extra_practice = 2
your_practice_per_day = others_practice_per_day + your_extra_practice # 每天3遍

# 一个月（30天）的总练习量
monthly_practice = your_practice_per_day * 30 # 一个月练习90遍
```
通过超额练习，可以加速从陌生到熟练的过程。

## 教学模式与承诺

上一节我们明确了学员需要满足的要求，本节中我们来看看老师基于这些要求所提供的教学模式与保障。

### 教学模式：听话照做，全力以赴

![](img/4f92ea2f0656ab87fa787f8e22c138b3_11.png)

老师的核心教学要求是：**听话照做，全力以赴**。这并非简单的服从，而是基于成功路径的方法论。

*   **原因**：只有学员严格按照被验证有效的方法步骤去执行，老师才能把控教学进程，排除因个人随意发挥导致的失败风险。
*   **目标**：确保学员能够 **`100%学会`**、 **`100%出单`**、 **`100%赚到钱`**。
*   **精神支撑**：老师以“三个不相信”的军队精神（不相信有完不成的任务，不相信有克服不了的困难，不相信有战胜不了的敌人）和毛主席的奋斗历程为榜样，强调在看似“不可能”的任务面前，坚定的信念与执行力能创造奇迹。

### 教学保障：个性化跟踪记录

老师会为正式学员建立个人学习档案进行跟踪记录，确保教学计划的落实。过往学员案例包括56岁宝妈、63岁不识字老人、84岁高龄学员、大学生、负债创业者等，均通过定制化教学取得了成果。
