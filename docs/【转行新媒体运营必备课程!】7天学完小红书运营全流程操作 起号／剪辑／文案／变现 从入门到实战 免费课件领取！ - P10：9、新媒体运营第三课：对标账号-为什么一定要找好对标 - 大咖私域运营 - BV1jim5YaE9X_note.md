首先，我们能知道目标用户喜欢什么内容。这里有一个关键问题：我们应该找什么样的账号作为对标？是数据差的，还是数据好的？答案是有一定数据量，但并非赛道顶级的账号。例如，在某图文笔记平台，对于一个刚起号的新手，粉丝量在30万、赞藏数据近百万的顶级账号参考意义不大。

这个阶段的账号，其内容能清晰地告诉我们，在该领域内，精准用户群体更喜欢什么。具体通过分析对标账号最近创作的内容方向来体现。例如，他近期发布了5篇笔记，其中2篇数据较好，3篇数据较差。我们就重点拆解那2篇数据好的笔记，了解其主题和形式，并进行模仿和复刻。这就是我们确定内容方向的方法。

![](img/85fd80fad073d5e31274305c425bee2f_37.png)

![](img/85fd80fad073d5e31274305c425bee2f_38.png)

其次，我们能发现平台喜欢推荐什么内容。从流量分发的底层逻辑来看，内容数据从少到多的过程，体现了平台的推荐机制。例如，一篇笔记的浏览量从200增长到1000甚至1万，是平台根据用户反馈不断加大推荐力度的结果。

我们可以通过分析多个对标账号近期数据表现好的内容题材，来判断平台当前倾向于推荐什么。如果某些内容被平台推荐后，用户反馈数据（如点赞、评论、收藏）很好，就说明这些内容是平台和用户共同喜欢的。反之，数据差的内容则不受青睐。因此，对标账号内容流量的升降趋势，能帮助我们把握平台的内容偏好。

了解了意义之后，我们来看看具体需要分析对标账号的哪些数据。以下是需要重点关注的几个维度：

![](img/85fd80fad073d5e31274305c425bee2f_59.png)

![](img/85fd80fad073d5e31274305c425bee2f_60.png)

![](img/85fd80fad073d5e31274305c425bee2f_62.png)

![](img/85fd80fad073d5e31274305c425bee2f_64.png)

![](img/85fd80fad073d5e31274305c425bee2f_65.png)

**1. 内容结构**
*   **人设与出镜**：账号是否打造了鲜明的人设？是否有人物出镜？
*   **文案节奏**：文案的开头、中间、结尾分别讲什么？整体结构如何？节奏是快是慢？
*   **场景**：内容在什么场景下拍摄？（室内/室外，固定场景/多变场景）
*   **人物**：讲解人物是固定的还是变换的？塑造了怎样的IP？

![](img/85fd80fad073d5e31274305c425bee2f_66.png)

![](img/85fd80fad073d5e31274305c425bee2f_67.png)

![](img/85fd80fad073d5e31274305c425bee2f_69.png)

![](img/85fd80fad073d5e31274305c425bee2f_71.png)

**2. 视觉与文案元素**
*   **封面**：封面的设计风格、视觉重点是什么。
*   **标题**：标题的撰写技巧和吸引点。
*   **内容选题**：近期主要创作哪些方向的内容？哪些选题获得了更好的数据？

分析完数据，我们最终要落实到行动上。对标账号主要能为我们提供两大核心作用：

![](img/85fd80fad073d5e31274305c425bee2f_86.png)

**1. 提供方向的指引**
很多同学在运营账号时，虽然确定了赛道和人设，但在具体内容形式上容易陷入迷茫。例如，纠结封面该如何设计才能获得高点击率。在没有足够运营经验时，不要凭空想象。最有效的方法是参照对标账号已经验证成功的模式。他们的封面、内容形式就是你可以直接模仿的起点。这能解决运营前期最大的迷茫问题，为你的内容创作提供明确、可行的路径。

![](img/85fd80fad073d5e31274305c425bee2f_88.png)

![](img/85fd80fad073d5e31274305c425bee2f_89.png)

![](img/85fd80fad073d5e31274305c425bee2f_90.png)

![](img/85fd80fad073d5e31274305c425bee2f_91.png)

![](img/85fd80fad073d5e31274305c425bee2f_92.png)

![](img/85fd80fad073d5e31274305c425bee2f_93.png)

![](img/85fd80fad073d5e31274305c425bee2f_94.png)

![](img/85fd80fad073d5e31274305c425bee2f_95.png)

**2. 支持后期选题的复刻**
在后续的内容规划中，我们可以直接参照对标账号成功的内容形式，包括其标题、文案和风格。然后，运用我们上节课讲到的“差异化”方法，将这些优秀元素与我们自身的特色相结合，复刻并打造属于自己的爆款选题。

![](img/85fd80fad073d5e31274305c425bee2f_96.png)

![](img/85fd80fad073d5e31274305c425bee2f_97.png)

![](img/85fd80fad073d5e31274305c425bee2f_98.png)

```
我的新选题 = 对标账号的爆款选题框架 + 我自身的独特观点/经历/表现形式
```
