# 抖音自媒体运营：P13：策划内容定位 🎯

在本节课中，我们将要学习如何为你的抖音账号策划一个清晰、具体且具有竞争力的内容定位。仅仅选择一个主题是不够的，我们还需要深入规划身份、场景、道具、态度等细节，才能让你的账号在众多同质化内容中脱颖而出。

---

上一节我们讨论了如何选择主题IP，但很多同学会担心：这个主题已经有很多人在做了，我还能火起来吗？

![](img/69a908f6703280456ce50c1737dbcd42_1.png)

这个问题很关键。当我们选定一个主题IP后，例如“老韩摄影教学”，会发现市场上已有“小韩摄影教学”等类似账号。此时，单纯模仿可能无法成功。

![](img/69a908f6703280456ce50c1737dbcd42_1.png)

那么，如何让我们的主题IP活起来并火起来呢？答案在于更具体、更差异化的核心内容策划。

---

## 从主题到具体策划

选定主题IP只是第一步。接下来，我们需要构思更细致的内容框架。例如，选择了“摄影教学”主题，但具体教什么？拍人还是拍景？用手机还是相机？这些都需要确定。

因此，除了主题IP，我们还需要规划以下核心要素：
*   **身份**：你以什么角色出现？（如：专业摄影师、摄影爱好者）
*   **场景**：在哪里拍摄？（如：工作室、户外、街头）
*   **道具与动作**：使用什么工具？展现什么行为？
*   **态度**：传达怎样的情绪或价值观？（如：专业严谨、幽默搞笑、暴躁直接）
*   **机位与视频结构**：如何拍摄和编排内容？

这些要素环环相扣。策划得越丰富、越具体，你的内容定位就越明确，后续的内容生产也越容易。

例如，账号“暴躁的小风”就在“态度”上选择了“暴躁”作为鲜明标签，从而获得了大量粉丝。其内容脚本模式与我们后续将讲到的技巧是相通的。

*   **身份**：设定为一位“公主”或“仙女”。
*   **场景**：选择在具有外国古堡风格的场景中。
*   **道具与服装**：华丽的欧洲宫廷风裙装。

这种独特的策划组合，构建了一个极具辨识度的“人物IP”，使得即使内容普通，也能在视觉和概念上抓住观众，从而脱颖而出。

## 总结

本节课中，我们一起学习了如何深化内容定位。

![](img/69a908f6703280456ce50c1737dbcd42_11.png)

1.  **明确核心**：仅有一个宽泛的主题IP（如“摄影教学”）是不够的，必须进行具体策划。
2.  **规划要素**：需要系统性地构思身份、场景、道具、动作、态度、机位等细节。公式可以概括为：
    **内容竞争力 = 主题IP + (身份 + 场景 + 道具 + 态度 + ...)**
    括号内的要素越独特、组合越巧妙，竞争力越强。
3.  **借鉴思路**：通过“仙女酵母”的案例我们看到，通过打造独特的**人物IP**（如特定身份、场景），即使内容本身不新奇，也能成功吸引大量粉丝。

![](img/69a908f6703280456ce50c1737dbcd42_12.png)

记住，在内容同质化的环境中，细节和整体风格的差异化策划，是让你账号脱颖而出的关键。下一节，我们将进入更实操的部分，学习如何基于定位生产内容。
