# 小红书开店运营教程：P5：5.小红书的3种店铺类型 🏪

![](img/437ba4057f3416d95e82ebfde3c70eea_1.png)

在本节课中，我们将要学习小红书平台的三种主要店铺类型：个人店、个体店和企业店。了解它们之间的区别，是选择合适开店方式、规划运营策略的第一步。

![](img/437ba4057f3416d95e82ebfde3c70eea_3.png)

上一节我们介绍了开店的基础准备，本节中我们来看看具体的店铺分类。小红书店铺主要分为三种：个人店、个体店和企业店。企业店中又包含旗舰店和专卖店等细分类型，但本节我们主要聚焦于这三种基础类型的核心区别。

以下是三种店铺类型的详细对比与说明：

![](img/437ba4057f3416d95e82ebfde3c70eea_5.png)

## 个人店

![](img/437ba4057f3416d95e82ebfde3c70eea_7.png)

个人店是门槛最低的店铺类型。
*   **保证金**：**1000元**。支持先销售产品，后补交保证金。
*   **可售类目**：受限，只能上架平台规定的部分类目（非电器等特殊品类）。具体需参考《小红书经营大类一览表》。
*   **开店门槛**：仅需实名认证。**零粉丝**即可开店，无需营业执照。
*   **店铺设置**：店铺名字和头像可以**随意更改**。

![](img/437ba4057f3416d95e82ebfde3c70eea_9.png)

> **注意**：虽然可以后交保证金，但不缴纳保证金将无法参与平台活动，且用户不能使用微信支付。由于保证金可退，建议开店时立即缴纳。

## 个体店

个体店适合拥有个体工商户资质的卖家。
*   **保证金**：**1000元**。必须**先缴纳保证金**，才能上架销售产品。
*   **可售类目**：比个人店更广泛，可以销售带有品牌授权的产品（如代理的品牌电器）。
*   **开店门槛**：需要**个体工商户营业执照**，并认证**个体企业号**（认证费600元/年）。**零粉丝**可开店。
*   **店铺设置**：店铺名字和认证类目**不可随意更改**。如需修改店名，需重新缴纳600元认证费。

![](img/437ba4057f3416d95e82ebfde3c70eea_11.png)

> **规则提示**：一个营业执照最多可认证**两个**个体/企业店。

## 企业店

企业店适合公司或品牌方，权限最大。
*   **保证金**：**2000元起**。若升级为专卖店或旗舰店，保证金可能高达**20000元**。必须先缴纳。
*   **可售类目**：最广泛，几乎涵盖所有品类（仅限平台邀约的类目除外）。可申请开设**品牌旗舰店**或**品牌专卖店**。
*   **开店门槛**：需要**企业营业执照**，并认证**企业号**（认证费600元/年）。**零粉丝**可开店。
*   **店铺设置**：店铺名字和认证类目**不可随意更改**。但类目可以**新增**（如从卖杯子新增卖项链类目），新增类目需提交资质并通过平台审核，此过程不额外收费。修改店名则需重新缴纳认证费。

### 小红书电商的核心逻辑

![](img/437ba4057f3416d95e82ebfde3c70eea_13.png)

总结完店铺类型，我们需要理解小红书电商与传统电商（如淘宝、天猫）的本质区别，这主要体现在流量来源上：

![](img/437ba4057f3416d95e82ebfde3c70eea_15.png)

![](img/437ba4057f3416d95e82ebfde3c70eea_17.png)

**小红书店铺流量公式**：`优质内容 -> 用户种草 -> 消费决策 -> 购买成交`

![](img/437ba4057f3416d95e82ebfde3c70eea_19.png)

其消费链路是：用户通过**搜索**或平台**自然推荐**刷到您的笔记，被笔记内容**种草**，从而对产品产生兴趣，经过决策后最终完成购买。因此，运营的核心在于创造能引发共鸣和种草的优质内容。

![](img/437ba4057f3416d95e82ebfde3c70eea_21.png)

平台对“无货源代拍”等行为的打击，对于致力于通过优质内容做好电商的卖家而言，实际是利好，因为这有助于净化环境，让流量更精准地分配给用心经营的商家。

![](img/437ba4057f3416d95e82ebfde3c70eea_23.png)
