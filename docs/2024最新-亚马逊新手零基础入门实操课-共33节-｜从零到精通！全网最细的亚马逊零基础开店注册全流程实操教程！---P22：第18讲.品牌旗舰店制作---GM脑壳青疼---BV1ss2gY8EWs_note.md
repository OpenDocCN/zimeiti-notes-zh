# 亚马逊新手零基础入门实操课：P22：第18讲.品牌旗舰店制作 🏬

在本节课中，我们将要学习如何从零开始制作一个亚马逊品牌旗舰店。品牌旗舰店是展示品牌形象、聚合产品并提升客户购物体验的重要工具。我们将详细介绍其入口、设置步骤、页面布局以及核心模块的运用。

## 品牌旗舰店的入口与价值

![](img/f4439bccb61b60f2751304e89adebc8d_1.png)

上一节我们介绍了品牌的基本概念，本节中我们来看看如何创建品牌旗舰店。首先，我们需要知道品牌旗舰店在哪里以及它有何作用。

打开亚马逊前台页面，在搜索框中输入一个关键词进行搜索。在搜索结果页的顶部，位于普通广告位上方，有一个特殊的广告位置。

![](img/f4439bccb61b60f2751304e89adebc8d_3.png)

![](img/f4439bccb61b60f2751304e89adebc8d_1.png)

这个位置就是品牌广告位。只有成功创建了品牌旗舰店的卖家，才能投放此类广告。品牌广告可以直接展示品牌Logo、品牌标语和产品信息。此外，客户可以通过广告中的按钮直接进入品牌旗舰店页面，无论是你自己的店铺还是竞争对手的店铺都可以访问。

## 店面基础设置

了解了入口后，我们开始学习具体的创建步骤。首先从基础的店面设置开始。

进入卖家后台的“品牌旗舰店”编辑界面。第一个需要设置的部分是“店面设置”。

![](img/f4439bccb61b60f2751304e89adebc8d_3.png)

以下是店面设置的核心内容：
*   **品牌信息**：填写你的品牌名称和上传品牌Logo。这个Logo将显示在店铺页面的显著位置。
*   **商品网格布局**：选择商品在网格中的显示样式，有“正方形”和“立方体”两种选项。你可以根据店铺的整体设计风格来选择更合适的布局。

![](img/f4439bccb61b60f2751304e89adebc8d_5.png)

## 创建页面与结构规划

![](img/f4439bccb61b60f2751304e89adebc8d_7.png)

完成基础设置后，接下来需要构建店铺的内容页面。合理的页面结构能帮助客户快速找到所需商品。

在编辑器中，点击“添加页面”可以创建一个全新的页面。创建页面时，需要填写页面名称和URL关键词。例如，如果你销售睫毛产品，页面名称可以设为“睫毛”，URL关键词可设为“eyelashes”。

在页面描述中，可以用简短的一句话介绍该页面销售的产品或主要功能，同时也可以巧妙地埋入一些核心关键词。创建页面后，你可以通过拖拽的方式，将一个页面设置为另一个页面的子页面，从而形成“一级栏目”和“二级栏目”的树状结构。

例如，主营“键盘”的店铺，可以设置一级页面为“键盘”，其下再设置“机械键盘”、“键盘垫”、“键盘套”等子页面，使店铺结构清晰，便于导航。

## 首页内容模块布局与设计

页面结构搭建好后，最重要的就是设计首页的内容。首页是客户的第一印象，需要精心布局。

我们可以参考优秀竞争对手的店铺首页设计。通常，一个专业的品牌旗舰店首页会包含多个模块。

![](img/f4439bccb61b60f2751304e89adebc8d_5.png)

以下是首页可以添加的核心模块及其作用：
*   **品牌视频/图片展板**：建议在首页顶部放置一个品牌宣传视频或一个轮播图展板。轮播图可以展示3-8张图片，每张图片建议尺寸为**1500像素 x 700像素**。每张图片都可以链接到店铺内的特定产品，实现直接引流。
*   **可购物图片**：这是一种高效的模块，客户可以直接点击图片上的“立即购买”按钮，跳转到产品详情页完成购买。
*   **自定义组合模块**：编辑器提供了“拆分部分”的布局选项，允许你自由组合不同的内容。例如，可以在一个模块的左侧添加视频，右侧展示产品；或者在上方放图，下方添加商品列表。这为个性化设计提供了很大空间。
*   **自动推荐模块**：有两个非常重要的自动化模块：“精选促销/畅销商品”和“推荐商品”。添加这些模块后，亚马逊系统会自动根据店铺销售数据或客户的访问路径，向客户展示最畅销、正在促销或可能感兴趣的商品，无需手动关联产品。

## 总结与实操建议

本节课中我们一起学习了亚马逊品牌旗舰店的完整制作流程。

总结一下关键步骤：首先找到品牌旗舰店的入口并理解其价值；然后进行品牌Logo和布局的基础设置；接着规划清晰的页面层级结构；最后，通过组合视频、轮播图、可购物图片及自动推荐模块来设计一个吸引人且转化率高的首页。

对于初学者，一个非常有效的实操建议是：模仿你所在品类中顶尖品牌的旗舰店布局。分析他们的首页使用了哪些模块、如何排列、如何引导客户，并借鉴其思路应用到自己的店铺设计中。
