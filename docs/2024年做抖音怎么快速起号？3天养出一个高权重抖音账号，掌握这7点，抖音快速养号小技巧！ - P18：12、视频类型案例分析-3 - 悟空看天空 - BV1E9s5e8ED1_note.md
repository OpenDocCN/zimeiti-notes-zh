上一节我们探讨了视频内容的基本构成，本节中我们来看看一个具体的案例分析，以理解“强人设”与“弱人设”的区别。

![](img/110c31356a706848bb7287861c9dcf7a_11.png)

![](img/110c31356a706848bb7287861c9dcf7a_13.png)

现在进入老师考验你们的环节。请看以下视频内容，并立即回答：这是一个**强人设**视频，还是**弱人设**视频？

![](img/110c31356a706848bb7287861c9dcf7a_15.png)

> 我觉得穷人改变命运最大的难点就是他的父母，还有他的原生阶层。如果他生活在父母身边，父母对他施加很大的影响力，基本上他这辈子就没机会改变命运了。因为你想一个问题，他爹妈如果观点是对的，那他爹妈怎么会把他生成一个穷人？他爹妈在那么多年的社会实践中，最终变成了一个穷人，一定是因为他的观点是有很多问题的。所以他深受他父母影响，他就绝对完蛋。

![](img/110c31356a706848bb7287861c9dcf7a_17.png)

![](img/110c31356a706848bb7287861c9dcf7a_19.png)

> 所以你会发现一个现象，那就是穷人翻身必定不是在老家，他一定是去别的城市，远离父母，自己一个人独自打拼。父母管不到他了，那这个时候他就有机会改变自己思维了。

![](img/110c31356a706848bb7287861c9dcf7a_21.png)

![](img/110c31356a706848bb7287861c9dcf7a_23.png)

以下是视频中阐述的穷人难以翻身的几个核心原因：

![](img/110c31356a706848bb7287861c9dcf7a_25.png)

1.  **父母与原生阶层的影响**：如果父母的观念正确，他们自己就不会是穷人。长期受其影响，子女难以突破思维局限。
2.  **社交圈层的固化**：“物以类聚，人以群分”。穷人的朋友通常也是穷人，他们的思维和解决问题的方式可能存在相似局限。公式可以表示为：`个人认知 ≈ 所处圈层的平均认知`。
3.  **缺乏本金与风险承担能力**：当机会来临时，往往因缺少关键资金而无法抓住。同时，由于对风险的恐惧，不敢通过借贷等方式“All in”（全部投入）。核心逻辑是：`机会 - (本金 + 风险承受力) = 错失`。
4.  **内心的虚弱与不自信**：长期处于社会边缘导致习惯性不自信、逃避责任，从而无法团结他人，难以成就事业。

![](img/110c31356a706848bb7287861c9dcf7a_27.png)

![](img/110c31356a706848bb7287861c9dcf7a_29.png)

基于以上分析，视频给出了改变命运的建议：

![](img/110c31356a706848bb7287861c9dcf7a_31.png)

![](img/110c31356a706848bb7287861c9dcf7a_33.png)

1.  离开家乡，远离父母。
2.  结识新朋友，突破原有社交圈层。
3.  在能力范围内，敢于承担风险，有“All in”的勇气。
4.  建立自强和自信。

看完了视频内容，我们班的同学几乎都答对了。这是一个典型的**弱人设**视频。

![](img/110c31356a706848bb7287861c9dcf7a_41.png)

![](img/110c31356a706848bb7287861c9dcf7a_43.png)

**为什么它是弱人设？**
一方面，视频内容虽然讨论了“穷人”和“富人”，但与讲述者**本人**没有直接关系。视频的画面（可能是治愈系的空镜或动画）、声音和所讲述的观点是分离的，观众无法通过内容构建出讲述者清晰的个人形象或故事。

![](img/110c31356a706848bb7287861c9dcf7a_45.png)

**为什么这个视频能火？**
它运用了“双向吸引”的策略：
1.  **形式吸引**：一部分观众会被其画面风格（如治愈强迫症的画面）所吸引。
2.  **内容共鸣**：另一部分观众则对其观点产生强烈共鸣，认为“说得太好了”。

![](img/110c31356a706848bb7287861c9dcf7a_47.png)

![](img/110c31356a706848bb7287861c9dcf7a_49.png)

这种将**吸引人的形式**与**引发共鸣的内容**相结合的方法，就是“双向吸引”。

![](img/110c31356a706848bb7287861c9dcf7a_51.png)

![](img/110c31356a706848bb7287861c9dcf7a_53.png)

我们可以将这个技巧应用到自己的视频创作中：
*   **宝妈**拍宝宝日常没人看？可以尝试配上搞笑的故事或趣事。
*   **实体店老板**拍产品、工厂没人看？可以分享你的从业经历、创业故事或人生感悟。

代码式的创作思路可以概括为：`爆款潜力 ≈ 吸引人的形式 + 引发共鸣的内容`。

前期只要账号定位和标题准确，后期就可以灵活运用这种“双向吸引”的方法。很多视频完播率不高，往往只是缺少这样的小技巧而已。抖音运营本身就是一个需要不断学习和实践的行业。

**本节课总结**
本节课我们一起分析了一个具体的视频案例。我们学会了如何区分**强人设**与**弱人设**，并掌握了“双向吸引”这一核心内容策略。关键在于，将视觉或形式上的吸引力与能够引发目标观众深度共鸣的观点或故事相结合，从而有效提升视频的完播率和互动率。记住，好的内容需要巧妙的包装和精准的触达。
