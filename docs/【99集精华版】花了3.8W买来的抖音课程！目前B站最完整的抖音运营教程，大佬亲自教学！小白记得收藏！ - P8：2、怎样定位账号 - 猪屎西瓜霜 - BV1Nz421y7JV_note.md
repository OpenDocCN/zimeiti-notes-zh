# 抖音运营教程：P8：2、怎样定位账号

在本节课中，我们将学习抖音账号定位的核心方法。清晰的定位是获取流量的基础，能帮助你的内容吸引目标观众，而不是像发朋友圈一样随意。

上一节我们了解了抖音运营的前景，本节中我们来看看如何为账号进行精准定位。

## 账号定位的重要性
不要把抖音账号当成朋友圈来运营。今天发自拍，明天发游玩，后天发日常，这种内容模式只有朋友可能感兴趣，但无法吸引广泛的陌生观众。为了获得流量，账号必须有一个明确的定位。

## 如何进行初步定位
账号定位应围绕一个特定领域持续创作内容。抖音内容大致可分为17个主要标签领域。

![](img/4ffb3954fb62f629914c2bcb77e34b78_1.png)

以下是抖音内容的17个主要领域标签：
*   美食
*   生活
*   房产
*   游戏
*   金融
*   宠物
*   美妆
*   口播（如情感、养生、育儿、知识科普等）
*   家装
*   服装
*   娱乐
*   （其他领域）

你需要基于自己的兴趣、擅长或想发展的方向，从中选择一个主要领域。专注于一个领域是实现内容垂直的关键，试图覆盖所有领域反而效果不佳。

![](img/4ffb3954fb62f629914c2bcb77e34b78_3.png)

## 如何进行二度细分定位
拥有初步定位只是第一步。接下来需要进行领域细分，即二度定位。一个大类下包含多种具体的内容方向。

![](img/4ffb3954fb62f629914c2bcb77e34b78_5.png)

以下是几个领域的细分示例：
*   **美食领域**可细分为：美食教程、美食探店、吃播、美食产品测评。
*   **游戏领域**可细分为：真人出镜游戏、游戏实况、游戏剧情解说、手游、端游。
*   **情感领域**可细分为：两性情感、婚姻情感、亲情。

确定具体的细分方向，能使你的账号内容更聚焦，更容易吸引精准粉丝。

![](img/4ffb3954fb62f629914c2bcb77e34b78_7.png)

## 定位的后续步骤
在课堂上，老师会与积极互动的同学进行一对一沟通，帮助完成更精准的二度定位。因为有效的定位需要结合个人具体情况进行分析，无法在课堂上为所有人泛泛而谈。

本节课中我们一起学习了抖音账号定位的两步法：首先在17个大类中选择一个初步领域，然后在该领域内进行二次细分，确定具体的内容方向。清晰的定位是账号运营成功的起点，请围绕你选择的领域开始规划内容吧。
