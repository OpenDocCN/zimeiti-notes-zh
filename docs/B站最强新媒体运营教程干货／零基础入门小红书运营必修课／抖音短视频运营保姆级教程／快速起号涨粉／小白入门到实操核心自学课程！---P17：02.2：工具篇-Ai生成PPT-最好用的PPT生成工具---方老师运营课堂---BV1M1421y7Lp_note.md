# 新媒体运营教程：P17：02.2：工具篇-AI生成PPT 🚀

在本节课中，我们将学习如何使用一款强大的AI工具来快速生成PPT。这款工具能根据你提供的主题或现有文档，自动梳理大纲并生成详细的演示文稿内容，非常适合需要快速制作演示文稿的运营人员。

上一节我们介绍了新媒体运营中的其他工具，本节中我们来看看如何利用AI提升PPT制作效率。

## 工具核心功能

这款AI生成PPT工具主要提供两种模式：

1.  **主题生成模式**：你只需输入一个主题，工具会结合类似ChatGPT的生成式AI能力，自动创作全文并生成PPT。
2.  **文档转换模式**：你可以上传一篇已有的文档（如TXT、PDF），AI助手会自动总结文档大纲，并基于此大纲生成PPT内容。

我试用过许多AI生成PPT工具，这款在基于大纲生成详细内容方面表现尤为出色。

### 手动调整内容

这款工具最大的优势在于：它不仅能根据全文生成PPT，更能**基于大纲中的每个小节点，自动生成对应的详细内容**。

```text
传统AI生成PPT：需要用户提供全部完整内容 -> 工具负责排版。
本工具的优势：用户只需提供主题或文档 -> AI自动分析并生成大纲 -> 为每个大纲节点生成详细内容。
```

这一点至关重要，因为它解决了我们最常见的痛点——我们往往只有一个思路或框架，需要AI来帮助充实内容，而不是我们已经写好了所有细节再让它简单排版。

![](img/07aa6e876fe124ae98abf1393266d093_23.png)

在编辑界面右侧，我们可以对文字、形状等元素进行细致的调整和替换。

![](img/07aa6e876fe124ae98abf1393266d093_23.png)

![](img/07aa6e876fe124ae98abf1393266d093_25.png)
