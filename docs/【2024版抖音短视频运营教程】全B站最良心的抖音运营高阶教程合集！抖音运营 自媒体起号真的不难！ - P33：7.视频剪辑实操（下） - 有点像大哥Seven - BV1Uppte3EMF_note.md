# 抖音短视频运营教程：P33：7.视频剪辑实操（下）

在本节课中，我们将继续学习视频剪辑的后续关键步骤，包括如何为视频片段添加转场、如何添加字幕，以及一些确保视频质量和平台审核通过率的注意事项。

上一节我们介绍了视频素材的导入和基础比例调整，本节中我们来看看如何让多个视频片段流畅衔接，并完善视频的字幕信息。

## 添加转场

![](img/52d55bda8a9a2eaa63eabff7968ce06c_3.png)

![](img/52d55bda8a9a2eaa63eabff7968ce06c_5.png)

视频通常由多个短片段组成，片段之间需要有明确的衔接。如果不添加转场，观众在切换到不同内容的片段时可能会感到突兀，甚至退出视频，这会严重影响视频的**完播率**，进而影响账号的收益。

![](img/52d55bda8a9a2eaa63eabff7968ce06c_7.png)

**完播率**是衡量视频质量的关键指标，公式可以理解为：
`完播率 = (完整观看视频的用户数) / (观看视频的总用户数)`

![](img/52d55bda8a9a2eaa63eabff7968ce06c_9.png)

![](img/52d55bda8a9a2eaa63eabff7968ce06c_11.png)

以下是添加转场的具体步骤：

![](img/52d55bda8a9a2eaa63eabff7968ce06c_13.png)

1.  在视频轨道上，找到并点击右侧的“**+**”号按钮。
2.  在弹出的菜单中，选择“**素材库**”。
3.  在素材库的“热门”分类中，可以找到大量转场素材，例如各种动态效果。
4.  选择转场时，注意时长。过长的转场（如16秒）会让观众失去耐心。建议选择**3~5秒**的转场素材，3秒最为合适。
5.  选中合适的转场素材，点击添加。

![](img/52d55bda8a9a2eaa63eabff7968ce06c_15.png)

转场素材添加后，必须确保其画面比例与主视频保持一致（例如16:9）。通常，添加后编辑界面会提供“**画布模糊**”选项，选择第一个清晰的模糊样式，即可将转场背景适配为正确的横屏比例，最后点击右下角的对勾确认。

![](img/52d55bda8a9a2eaa63eabff7968ce06c_17.png)

## 添加字幕（文案）

字幕能提升视频的可看性和理解度，方便观众在无声环境下观看。剪映提供了自动识别字幕的功能，非常便捷。

![](img/52d55bda8a9a2eaa63eabff7968ce06c_19.png)

以下是添加字幕的步骤：

![](img/52d55bda8a9a2eaa63eabff7968ce06c_21.png)

1.  点击手机屏幕左下角的“**文字**”选项。
2.  点击“**新建文本**”右侧的“**识别字幕**”功能。
3.  点击“开始匹配”，软件会自动识别视频或音频中的人声，并生成字幕文本。
4.  对生成的字幕进行微调。拖动字幕文本框右下角，可以放大字幕。
5.  将调整好的字幕文本框拖动到视频画面底部合适的位置。调整好第一句字幕后，后续字幕会自动应用相同的样式和位置。

## 核心注意事项总结

![](img/52d55bda8a9a2eaa63eabff7968ce06c_23.png)

在剪辑过程中，请牢记以下几点，以确保视频质量并通过平台审核：

![](img/52d55bda8a9a2eaa63eabff7968ce06c_25.png)

*   **素材时长**：避免使用过长的视频素材或转场。
*   **步骤规范**：严格遵循素材选择、比例调整、添加转场和字幕的步骤。
*   **平台规则**：不符合规范的视频（如比例错误、内容过长）可能在审核阶段无法通过。

![](img/52d55bda8a9a2eaa63eabff7968ce06c_27.png)

本节课中我们一起学习了视频剪辑的收尾工作：通过添加转场来提升片段衔接的流畅度与完播率，利用自动识别功能快速添加字幕，并总结了确保视频合格的核心注意事项。掌握这些步骤，你的视频剪辑流程就基本完整了。
