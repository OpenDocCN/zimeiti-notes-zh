# 微信视频号运营从0~1全攻略：P31：19.实操步骤

在本节课中，我们将学习如何开通微信视频号的商品橱窗功能，并完成商品上架前的必要设置。这是开启视频号带货之旅的关键一步。

上一节我们介绍了视频号的基础概念，本节中我们来看看如何具体操作，开通带货权限。

## 第一步：进入视频号

![](img/49092f6d4732767b6327db25831b6fcd_1.png)

![](img/49092f6d4732767b6327db25831b6fcd_2.png)

以下是进入视频号界面的具体步骤：

![](img/49092f6d4732767b6327db25831b6fcd_4.png)

1.  **打开微信**：在手机主屏幕找到并点击微信应用图标。
2.  **点击“发现”**：进入微信后，点击底部导航栏的“发现”选项。
3.  **进入“视频号”**：在“发现”页面，将视线向上滑动，找到并点击“视频号”入口（图标为黄色的蝴蝶形状）。

![](img/49092f6d4732767b6327db25831b6fcd_6.png)

## 第二步：找到创作者中心

进入视频号后，需要找到功能设置入口。

![](img/49092f6d4732767b6327db25831b6fcd_8.png)

![](img/49092f6d4732767b6327db25831b6fcd_9.png)

1.  在视频号主页，点击右上角的**人形头像**图标。
2.  进入个人页面后，向下滑动屏幕，直到页面最底部。
3.  找到并点击 **“创作者中心”**。如果未看到此选项，可能是因为账号为新注册且未发布过作品，发布一条作品后即可显示。

## 第三步：开通带货权限

在创作者中心，我们可以进行带货功能的开通与管理。

1.  在创作者中心页面，找到并点击 **“带货中心”**（图标类似一个小卖部）。
2.  点击后，系统会判断你的账号状态。如果未开通橱窗权限，页面将跳转至开通界面。

![](img/49092f6d4732767b6327db25831b6fcd_11.png)

以下是开通带货权限的核心操作：

*   **选择带货身份**：在开通界面，系统会要求你选择带货身份。通常提供“我是商家”、“我是达人”等选项。
*   **关键选择**：对于大多数个人运营者，请选择 **“我是达人”** 这个身份。
*   **缴纳保证金**：选择身份后，会进入保证金缴纳界面。点击“去交纳”按钮，按照提示通过微信支付缴纳100元保证金即可完成开通。

**核心操作公式**：
`点击“带货中心” -> 选择“我是达人” -> 缴纳保证金 -> 开通成功`

![](img/49092f6d4732767b6327db25831b6fcd_13.png)

## 第四步：商品上架准备

开通权限后，再次点击“带货中心”，界面就会变为商品选品、上架管理的功能面板，你可以从这里开始选择商品并关联到你的视频或直播中。

![](img/49092f6d4732767b6327db25831b6fcd_15.png)

本节课中我们一起学习了从打开微信到成功开通视频号商品橱窗的完整流程。关键点在于通过“创作者中心”进入“带货中心”，并正确选择“达人”身份以完成开通。掌握这个步骤，你就具备了在视频号进行带货的基础条件。下一节，我们将学习如何在这个中心里挑选和上架商品。
