## 什么是ICP备案？🔍

上一节我们介绍了课程主题，本节中我们来看看核心概念。ICP备案是互联网信息服务提供者备案的简称。其核心作用是**防止网站违规**。对于知识博主而言，如果你想直接在抖音小店或视频号小店等平台销售课程，通常需要办理此项备案。

## ICP备案全流程详解📝

了解了ICP备案的基本概念后，接下来我们看看其完整的办理步骤。以下是办理ICP备案的六个关键步骤：

1.  **访问官网**：打开阿里云官方网站。
2.  **购买域名**：在平台上购买一个网站域名。
3.  **域名认证**：完成域名认证，此步骤需要提供营业执照。
4.  **提交备案**：开始正式备案流程。系统会提示购买服务器，并需要在线提交营业执照、法人身份证等相关证件。
5.  **电话核实**：等待审核电话。通常在一周内，会有工作人员致电核实备案信息。
6.  **备案成功**：电话核实后，等待约3到5天，会收到提示备案成功的短信。
