# 抖音运营教程：P22：16、口播类视频拍摄方法-7 🎬

上一节我们介绍了拍摄的通用注意事项，本节中我们来看看如何将这些要点应用到具体的视频类型中。老师将以“口播类”视频为例进行讲解。

口播类视频无论内容主题是金融、装修、养生还是情感，其拍摄方法都基本一致。同样，记录日常生活、拍摄段子或美食视频，其拍摄方法也相通。因此，掌握以下两种类型的拍摄方法，几乎就能应对所有场景。

首先，我们分析第一个类型：口播视频。老师会以一位学员“种小草”的成功案例进行说明。

![](img/faf2a7b03202931096cf352e887de857_1.png)

这位学员是一名40岁、带着三个孩子的宝妈，她创作情感口播内容。其中一条视频播放量超过8000万，并借此涨粉30万。她成功的关键在于内容精准吸引了目标群体。例如，如果你的内容针对20多岁的学生党，并持续创作他们感兴趣的主题，涨粉速度就会很快。

以下是拍摄口播视频时需要关注的几个核心要点：

**1. 构图与比例**
抖音平台采用 **9:16** 的竖屏比例。在拍摄时，你需要决定画面内部是横屏还是竖屏。案例中的视频采用了 **4:3** 的横屏构图。关键是要保持统一：如果你选择横屏，那么所有视频都应保持横屏；选择竖屏，则全部使用竖屏。这能保证账号内容的视觉一致性。

**2. 人物主体**
人物应占据画面中央约 **70%** 的面积。避免将脸部怼得过近拍摄，画面左、右、上方应适当留白。这样能使画面看起来更清晰、舒适。

![](img/faf2a7b03202931096cf352e887de857_3.png)

**3. 外貌与表现**
你不需要特别漂亮或帅气才能做口播。口播的核心是传递内容和知识。只要让观众觉得你清晰、体面即可。如果对自己的外貌不自信，可以开启美颜滤镜，但注意不要过度夸张。

**4. 内容与目标群体**
视频内容应具有针对性，吸引特定群体。我们通过案例视频来分析其目标受众。

![](img/faf2a7b03202931096cf352e887de857_3.png)

视频主题是“跟婆婆在一起住是什么样的感受”。它讨论的是婆媳关系。最能对此产生共鸣的群体，是已婚女性。因此，这条视频精准吸引了已婚女性群体。通过数据分析也能验证这一点。

**5. 拍摄环境与准备**
口播视频不限于户外拍摄，在家也可以。关键是确保背景整洁美观。此外，如果担心记不住台词，可以在剪辑环节使用提词工具，实现边看边录，降低拍摄难度。

**6. 收音设备**
清晰的声音对口播视频至关重要。老师推荐使用一个实惠的工具：手机收声麦克风。价格通常在十几元左右，在淘宝、京东等平台都能买到，能显著提升音频质量。
