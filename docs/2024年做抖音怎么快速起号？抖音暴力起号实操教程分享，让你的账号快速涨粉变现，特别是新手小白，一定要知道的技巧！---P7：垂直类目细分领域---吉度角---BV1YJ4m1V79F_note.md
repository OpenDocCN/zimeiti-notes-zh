# 抖音运营教程：P7：垂直类目的细分领域 🎯

在本节课中，我们将深入探讨抖音账号运营中“垂直度”的核心概念。我们将学习如何将一个大领域层层细分，从而找到更精准、更具竞争力的内容定位，这对于新手快速起号至关重要。

上一节我们介绍了内容标签的重要性，本节中我们来看看如何将“垂直”这个概念具体化、层级化。

我们常说做账号要垂直。很多同学会简单地认为，选择“美食”或“游戏”这样的大类目一直做下去，就是垂直。这个理解没有错，但这仅仅是垂直的起点，属于**一级垂直领域**。

为了更清晰地理解垂直度的层级划分，我们以“游戏”领域为例进行拆解。

以下是垂直类目的分级示例：

*   **一级垂直领域**：这是一个最宽泛的分类。例如，你决定做“游戏”内容。
*   **二级垂直领域**：这是一级领域下的子分类。在“游戏”之下，可以细分为“手游”、“电竞”、“网游”等。
*   **三级垂直领域**：这是二级领域下的进一步细分，通常具体到某个游戏或玩法。例如，在“手游”下的“王者荣耀”，或在“电竞”下的“DOTA2”。
*   **四级垂直领域（可选项）**：这是更极致的细分，可以具体到游戏中的某个模式、英雄、地图或剧情线。例如，在“王者荣耀”中只玩“安琪拉”这个英雄，或在“吃鸡”游戏中只玩“雨林”地图。

通过这个例子可以看出，垂直的层级可以不断深入。大领域和小领域都可以作为你的选择，这完全取决于你自己的兴趣和规划。

之前有同学对“垂直领域”的概念不太清楚，希望通过今天的细化讲解，大家能够完全理解。这个概念对于后续的内容策划和账号定位非常关键。
