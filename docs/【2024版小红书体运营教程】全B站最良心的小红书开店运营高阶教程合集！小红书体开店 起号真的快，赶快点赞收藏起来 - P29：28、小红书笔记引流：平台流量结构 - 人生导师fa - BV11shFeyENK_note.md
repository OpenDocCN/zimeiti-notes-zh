# 小红书运营教程：P29：28、小红书笔记引流：平台流量结构 📊

![](img/c86f7382f4c3ddc57fa28f0bf935337d_1.png)

在本节课中，我们将要学习小红书平台的流量结构。了解流量从哪里来，以及不同流量入口的特点，是制定有效运营策略的基础。我们将重点分析发现页、关注页和搜索页这三个核心流量来源，并探讨如何根据它们的特点来优化我们的笔记发布策略。

上一节我们介绍了笔记制作的基础，本节中我们来看看笔记发布后，流量是如何在小红书平台上分布的。

## 什么是平台流量结构

流量结构指的是小红书平台整体流量的分布和来源。理解流量结构的目的，是让我们明白不同类型的笔记，投放到不同的流量入口，会获得怎样的效果。这有助于我们更精准地进行内容规划和推广。

小红书App的整体流量结构主要分为三个核心入口。

以下是三个核心流量入口的列表：

1.  **发现页**：这是小红书首页默认展示的“发现”板块。
2.  **关注页**：这是用户关注博主后，查看其更新的专属页面。
3.  **搜索页**：即App内的搜索功能，用户通过关键词主动寻找内容。

除了这三个主页面，App内还有一个“本地”页面。该页面主要展示基于地理位置的内容，对于大多数做商品引流和推广的账号来说，适用性有限。它更适合户外、美食探店等具有强烈地域属性的博主。如果我们的目标是面向全国推广商品，应尽量避免过度依赖本地流量，因为这可能会打乱账号的整体标签，吸引非目标人群。

因此，我们笔记引流的整体流量结构，核心就是**发现页、关注页和搜索页**这三个重点。

## 发现页：最主要的流量来源 🚀

发现页是整个小红书**最重要**的流量来源，其流量约占平台总流量的 **70%**。它也是平台进行流量层级测试的主阵地。一篇笔记最终能获得多大规模的曝光，很大程度上取决于它在发现页获得的用户初始反馈（如点击、点赞、收藏、评论）。

![](img/c86f7382f4c3ddc57fa28f0bf935337d_3.png)

当我们发布一篇笔记后，平台会根据账号权重给予一个初始的基础曝光量（例如1000或2000次）。这**基础曝光量的100%** 都会投放到发现页。用户只有通过持续互动（关注、搜索）后，笔记才会逐渐出现在关注页和搜索页。所以，对于新账号或新笔记，初期几乎所有的流量都来自于发现页。

## 搜索页：转化率最高的流量来源 🔍

搜索页的流量占比约为 **20%**，是仅次于发现页的流量入口。同时，它也是目前小红书流量增长最快的板块，并且是**转化率最高**的流量来源。

搜索流量的用户意图非常明确，他们是带着具体需求（如“春季连衣裙”、“油皮粉底液推荐”）来主动寻找解决方案或商品的。因此，通过搜索页进入笔记的用户，其购买或深度互动的意愿远高于其他渠道。可以说，小红书平台内直接成交的订单，有超过 **70%** 可能来自于搜索流量。

然而，搜索流量并非初期就能轻易获取。它依赖于笔记包含有效的**关键词**，以及商品或账号本身具有一定的**权重**（如历史成交、评价、互动数据）。对于新账号，首要任务是在发现页打好基础，积累权重，后期才能更好地承接搜索流量。

## 关注页：最精准但影响有限的流量来源 👥

关注页的流量占比最小，大约在 **7%-10%** 左右。这部分流量虽然量小，但用户**最精准**。因为他们已经关注了你的账号，对你的内容有明确的兴趣。

关注页的流量非常稳定，只要用户关注了你，你发布的新笔记就会出现在他们的关注流中。但是，这部分流量有一个重要特点：它**不计入影响笔记爆款权重的核心数据**。也就是说，即使你有1万粉丝，笔记发布后这1万人都看到了，但如果这篇笔记在发现页的初始反馈（点击率等）很差，它依然无法获得更大的平台推荐。

因此，关注页的作用在于维护核心粉丝、进行深度互动和引导私域转化，但它不能替代发现页作为获取流量的主战场。

## 运营策略与总结

本节课中我们一起学习了小红书平台的三大流量结构：**发现页（70%）、搜索页（20%）、关注页（<10%）**。

基于此，我们可以得出清晰的运营顺序和策略：

1.  **前期主攻发现页**：通过优化封面、标题和内容，提升笔记在发现页的点击率和互动率，这是冷启动和积累账号权重的关键。
2.  **中期布局搜索页**：在笔记中合理布局行业关键词，优化商品信息。待账号有一定权重和成交后，搜索流量会成为高转化的稳定来源。
3.  **长期维护关注页**：通过持续输出优质内容积累精准粉丝，将公域流量沉淀为私域资产，用于深度运营和引导。

理解这个流量结构，就是掌握了小红书流量密码的钥匙。它告诉我们笔记做好后，应该按什么顺序、在什么阶段、重点向哪个流量池发力。
