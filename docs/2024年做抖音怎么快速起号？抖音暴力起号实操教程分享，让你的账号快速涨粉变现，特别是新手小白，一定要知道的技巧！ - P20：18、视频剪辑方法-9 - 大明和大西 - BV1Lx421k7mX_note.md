上一节我们介绍了视频拍摄的要点，本节中我们来看看如何将拍摄好的素材剪辑成完整的视频。剪辑环节将直接进行实操演示，请准备好你的笔记，跟随步骤一起操作。

我们使用**剪映**（黑色图标版本）进行剪辑。这是抖音官方的剪辑工具，有手机版和电脑版。我们先讲解手机版的操作，方便没有电脑的同学学习。

打开剪映App，点击“开始创作”按钮。

![](img/599dfbbbe89d1fd30f9c93dc6cd43b96_15.png)

![](img/599dfbbbe89d1fd30f9c93dc6cd43b96_17.png)

接着，从相册中选择需要剪辑的视频素材，点击“添加”导入。如果你的视频由多个片段组成（例如生活Vlog的开头、中间、结尾），建议先导入并剪辑好第一个片段，再点击轨道尾部的“+”号添加第二个片段。这样可以保持工程文件清晰，不易混乱。

以下是视频剪辑的核心步骤：

![](img/599dfbbbe89d1fd30f9c93dc6cd43b96_25.png)

![](img/599dfbbbe89d1fd30f9c93dc6cd43b96_27.png)

**第一步：调整画面构图与比例**
导入视频后，首先检查画面。如果构图或方向不对，可以进行调整。
1.  点击选中时间轴上的视频片段。
2.  在底部工具栏中向右滑动，找到并点击“编辑”按钮（图标通常是一个方块）。
3.  在编辑菜单中，点击“旋转”来调整画面到正确的方向。
4.  如果需要裁剪画面比例，可以点击“裁剪”选项。这里提供了多种比例，如**1:1**、**4:3**、**16:9**等，选择适合抖音的竖屏比例（如9:16）即可。

![](img/599dfbbbe89d1fd30f9c93dc6cd43b96_29.png)

![](img/599dfbbbe89d1fd30f9c93dc6cd43b96_31.png)

![](img/599dfbbbe89d1fd30f9c93dc6cd43b96_7.png)

![](img/599dfbbbe89d1fd30f9c93dc6cd43b96_33.png)

**第二步：剪辑视频内容**
调整好画面后，需要剪掉视频中多余、无效的部分。
1.  播放视频，找到需要裁剪的起始位置。
2.  将时间轴上的白色竖线（播放指示器）移动到此位置。
3.  点击底部工具栏的“分割”按钮，视频会被切成两段。
4.  选中需要删除的片段，点击“删除”按钮。
重复此操作，剪掉开头沉默、中间失误或结尾无关的部分。核心操作可总结为：**选中 -> 分割 -> 选中 -> 删除**。

**第三步：添加背景音乐**
视频内容剪辑完成后，需要添加合适的背景音乐。
1.  点击底部工具栏的“音频”按钮。
2.  如果你有预先下载好的配音或音乐文件，可以点击“提取音乐”，从手机视频中提取音频。
3.  如果没有，则点击“添加音乐”。剪映提供了丰富的音乐库，你可以根据分类（如美食、美妆、萌宠）查找，或直接搜索歌曲名。点击音乐可以预览，选定后点击“使用”即可添加。

![](img/599dfbbbe89d1fd30f9c93dc6cd43b96_43.png)

![](img/599dfbbbe89d1fd30f9c93dc6cd43b96_45.png)

![](img/599dfbbbe89d1fd30f9c93dc6cd43b96_17.png)

![](img/599dfbbbe89d1fd30f9c93dc6cd43b96_47.png)

添加音乐后，如果音乐过长，同样可以使用**分割**和**删除**功能进行裁剪。你还可以选中音乐轨道，点击“音量”来调整背景音乐的大小，使其不会盖过人声。

![](img/599dfbbbe89d1fd30f9c93dc6cd43b96_49.png)

**第四步：添加与编辑字幕**
为视频添加字幕能提升观看体验。以下是高效添加字幕的方法：
1.  点击底部工具栏的“文本”按钮。
2.  选择“识别字幕”功能。
3.  在弹出的菜单中，选择“仅识别视频原声”（如果你添加了背景音乐且不需要歌词字幕，就不要选“全部”）。
4.  点击“开始识别”，系统会自动为视频中的人声生成字幕。

![](img/599dfbbbe89d1fd30f9c93dc6cd43b96_51.png)

![](img/599dfbbbe89d1fd30f9c93dc6cd43b96_53.png)

![](img/599dfbbbe89d1fd30f9c93dc6cd43b96_35.png)

![](img/599dfbbbe89d1fd30f9c93dc6cd43b96_55.png)

![](img/599dfbbbe89d1fd30f9c93dc6cd43b96_57.png)

识别完成后，检查字幕是否有错别字。双击字幕条即可进行修改。你还可以在右侧的菜单中调整字幕的字体、样式、颜色等，使其与视频风格更搭配。

**第五步：导出高清视频**
在导出视频前，需要进行关键设置以保证视频清晰度。
1.  点击右上角的“导出”按钮。
2.  在导出设置中，调整参数：
    *   **分辨率**：尽可能拉满（如1080P或4K），分辨率越高越清晰。
    *   **帧率**：设置为**30帧**，这是抖音平台最合适的帧率。
3.  设置完成后，点击“导出”。导出过程中请勿锁屏或切换App，耐心等待完成。
4.  导出后，可以直接通过剪映分享到抖音。

掌握了基础剪辑后，我们为口播类创作者推荐一个实用工具。如果你在录制时记不住台词，可以尝试使用“爱提词”这款App。

![](img/599dfbbbe89d1fd30f9c93dc6cd43b96_65.png)

![](img/599dfbbbe89d1fd30f9c93dc6cd43b96_67.png)

以下是使用“爱提词”的基本方法：
1.  下载并打开“爱提词”App。
2.  点击右下角的“+”号，新建一个提词板。
3.  输入或粘贴你的口播文案，并设置一个标题，然后保存。
4.  点击“开始提词”，选择“录制模式”。此时，文案会以悬浮窗形式显示在屏幕上，你可以一边看词一边录制，无需背诵。
5.  在设置中，可以调整字体大小、滚动速度等，方便阅读。
6.  该App内置了基础的美颜功能（如瘦脸、大眼、磨皮），可以在录制时开启，让画面更美观自然，且效果比一些美颜相机更真实。
7.  App内还提供了“爆款文案”板块，涵盖情感、职场、生活等多个领域，可以作为内容创作的参考。
