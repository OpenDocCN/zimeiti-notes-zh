在本节课中，我们将要学习《亚马逊运营手册》这个项目的起源、核心目标以及它能为你提供的价值。我们将了解这个手册是如何诞生的，它旨在解决什么问题，以及你如何能从中获益并参与其中。

大家好，我是36。在这个假期，我没有出门，而是填补了一个之前未完成的项目。这个项目就是我们所说的《亚马逊运营手册》。它最初于2020年8月25日开始整理，之后陆陆续续进行了一些编写。

经过四年时间，2.0版本现已上线。我重新编写了大纲，并补充了一些已失效的链接。目前手册的主要内容就是这些项目。以下这些以表情符号开头的章节，是尚未完成编写的部分。

我会陆续将这些章节更新上去。创建这个手册的初衷，是为了记录我个人的一些感悟。同时，也是为了随时记录亚马逊平台的一些变化。最重要的目的是帮助新手卖家更快地了解如何入手亚马逊，或者获得一些基础认知。

因为我当初刚入行亚马逊时，也是去网上购买各种书籍。但我发现书籍的内容很容易过时。无论是平台政策还是一些运营玩法，都会更新迭代，而书籍无法及时反映这些变化。

因此，我萌生了制作一本电子书的想法，这样可以快速进行内容更新。例如，当某些政策改变，或者近期关于产品五点描述的要求出现新变化时，我都能及时更新上去。这是本手册最核心的初衷。

如果你觉得观看视频更方便，也可以去我们的“泼辣有课”观看免费的视频教学，这样学习速度可能更快。但视频教学存在一个不足之处，那就是无法快速搜索到你遇到的特定问题。

我们的电子书则比较方便，你可以在右上角直接进行快速搜索。如果你在阅读过程中遇到任何问题，或者发现手册中有未提及的内容，欢迎在社区留言给我们。我们会将其补充上去。
