# AI新媒体运营教程：P15：01.10.03：AI：从分析爆款笔记到生成爆款笔记 🚀

## 概述
在本节课中，我们将学习如何利用AI工具（如ChatGPT）分析现有平台上的爆款内容，并基于分析结果，指导AI生成具备爆款潜质的新笔记。整个过程分为两个核心步骤：**分析学习**与**模仿创作**。

上一节我们介绍了通过“喂资料”的方式，让AI理解特定平台的用语和风格。虽然示例结果可能不完美，但方法是正确的。本节中，我们来看看如何更深入地分析爆款内容，并让AI学习其成功要素，最终产出新内容。

## 第一步：分析爆款笔记 📊
要让AI学会创作，首先需要让它学会分析。我们需要指导AI对提供的爆款笔记进行多维度拆解。

以下是分析时需要关注的几个核心点：
1.  **爆款因素**：分析内容为何能成为爆款，例如是否引发了情感共鸣、提供了实用价值等。
2.  **内容结构**：拆解文案的起承转合。
    *   **开篇**：用什么方式（如提问、制造焦虑、引用名人）吸引用户继续阅读。
    *   **中间**：提供了哪些解决用户痛点的内容，如何维持用户的阅读兴趣和停留时长。
    *   **结尾**：如何设置“钩子”（如鼓励互动、引导私信、提供福利）促使用户产生下一步行为。
3.  **关键词策略**：分析笔记中是否刻意植入了关键词，以及关键词的出现频率和分布方式。

![](img/bba1e4d80021455cd7626f65b5e72c71_8.png)

### 操作演示：向AI下达分析指令
我们可以向AI（如ChatGPT 3.5）提供明确的指令和爆款笔记原文。

![](img/bba1e4d80021455cd7626f65b5e72c71_10.png)

**指令示例**：
```
请你分析以下提供的小红书平台上的考研相关爆款笔记文案。
你需要分析：
1. 文案的爆款因素（例如：情感共鸣、实用建议等）。
2. 文案的整体结构（开篇如何吸引用户，中间如何戳中痛点，结尾如何引导互动或关注）。
3. 分析该笔记是否有结合中文分词法设计关键词。如果有，请指出是哪个关键词及其出现频率；如果没有，请说明。
```

## 第二步：整合爆款要素并生成大纲 📝
在AI分析了多篇爆款笔记后，我们需要让它将分析出的成功要素进行整合、去重，形成一个清晰的“爆款公式”。

### 操作演示：让AI总结与规划
1.  **总结爆款因素**：
    ```
    请将你前面分析过的几篇考研爆款笔记的爆款因素总结一下，整合成一个清晰的列表。
    ```
    **AI可能总结的要素**：情感共鸣、逆袭故事、明确的备考计划、时间节点、针对性建议、资源推荐、正能量鼓励、使用表情符号等。

![](img/bba1e4d80021455cd7626f65b5e72c71_26.png)

![](img/bba1e4d80021455cd7626f65b5e72c71_28.png)

2.  **生成内容大纲**：
    基于总结的爆款要素，指令AI为新的笔记规划一个具备爆款潜质的大纲。
    ```
    现在，我需要写一篇关于考研的小红书笔记。请结合你刚才总结的爆款笔记因素，为我规划一个具备爆款潜质的笔记大纲。大纲需要细分到每个部分的要点，待我确认后，再生成完整文案。
    ```
    **AI生成的大纲示例**：
    *   1. 开篇：用引人入胜的话语或问题吸引注意，介绍主题。
    *   2. 专业选择分析。
    *   3. 全年时间规划表。
    *   4. 各科（政治/英语/数学/专业课）备考攻略。
    *   5. 时间管理与心态调整建议。
    *   6. 结尾：鼓励互动，引导关注或私信。

---

## 第三步：依据大纲生成完整文案 ✍️
大纲确认后，即可指令AI根据大纲和爆款要素，生成完整的笔记文案。此时可以提出更细致的要求。

### 操作演示：生成最终文案
```
请按照你提供的大纲，撰写一篇小红书考研笔记文案。
要求：
1. 文案结构严格遵循爆款笔记的结构。
2. 在开篇融入一个简单的“逆袭小故事”，引发读者共鸣。
3. 文案中要包含适量的emoji表情。
4. 尝试结合中文分词法的思路，自然地融入“考研”、“备考”等关键词。
```

![](img/bba1e4d80021455cd7626f65b5e72c71_30.png)

**AI生成的文案示例**：
> “曾经我也在深夜刷题到崩溃，觉得考研上岸是个遥不可及的梦…但通过合理的规划，我最终成功逆袭！把我的经验分享给你，考研全流程规划看这一篇就够了！📚
> **一、时间规划是灵魂** ⏰
> 从现在起，你的备考时间线应该是：11月-2月基础阶段…（具体规划）
> **二、政治备考**：核心是… #考研 #备考
> …
> 坚持就是胜利！有任何问题都可以问我，一起加油上岸！💪”

> **注意**：AI生成的初稿可能不尽完美，例如故事不够生动、关键词植入生硬、表情符号使用不当等。但这为运营人员提供了一个高质量的基础草稿，可以在此基础上进行人工优化和调整，效率远高于从零开始。

## 进阶技巧与工具 🛠️
*   **使用更强大的模型**：ChatGPT 4.0或更高版本在理解指令、内容创造性和连贯性上通常表现更好。
*   **创建自定义GPTs**：利用ChatGPT的GPTs功能，可以创建一个专属的“小红书文案助手”。你可以上传平台规范、优质案例、专用表情包列表等资料供其学习，使其产出更符合你特定要求的文案。
*   **指令需精确**：与AI对话时，措辞的细微差别可能导致结果不同。例如，“包含丰富的表情”和“包含适量表情”带来的效果就不一样，需要在实践中不断调整和优化你的指令。

---

## 总结
本节课我们一起学习了利用AI辅助新媒体内容创作的核心流程：
1.  **分析学习**：提供爆款笔记给AI，让其分析**爆款因素**、**内容结构**和**关键词策略**。
2.  **整合规划**：让AI总结共性，并基于此生成一个符合爆款逻辑的**内容大纲**。
3.  **模仿创作**：根据大纲和详细指令，让AI生成**完整文案初稿**，运营人员可在此基础上进行优化。
