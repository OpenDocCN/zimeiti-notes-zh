# 拼多多运营实操教程：P31：拼多多开店-搜索推广规则 🚀

在本节课中，我们将要学习拼多多搜索推广的核心规则。我们将了解排名机制、扣费方式以及影响推广效果的关键因素——质量分。掌握这些基础知识，是有效进行搜索推广的第一步。

## 排名机制解析

![](img/36d8a9c85a180b09830196de053f3368_1.png)

上一节我们介绍了搜索推广的重要性，本节中我们来看看排名是如何决定的。

拼多多搜索推广的综合排名由以下公式决定：
**综合排名 = 关键词出价 × 关键词质量得分**

![](img/36d8a9c85a180b09830196de053f3368_3.png)

这意味着，在关键词质量得分相近的情况下，出价越高，商品的排名就会越靠前。



![](img/36d8a9c85a180b09830196de053f3368_4.png)

![](img/36d8a9c85a180b09830196de053f3368_1.png)

![](img/36d8a9c85a180b09830196de053f3368_6.png)

具体操作时，商家可以在拼多多商家后台的推广工具中，查看市场平均出价，并设置自定义出价。为了获得更精准的流量，系统还提供了“精确匹配”功能，通常建议设置30%的溢价。

## 扣费规则详解

了解了排名机制后，接下来我们需要清楚推广是如何扣费的，这直接关系到我们的推广成本。

拼多多搜索推广的扣费规则主要有以下三点：



![](img/36d8a9c85a180b09830196de053f3368_6.png)
![](img/36d8a9c85a180b09830196de053f3368_7.png)
![](img/36d8a9c85a180b09830196de053f3368_8.png)

以下是具体的扣费规则说明：
1.  **核心扣费公式**：您的实际扣费 = (下一位的出价 × 下一位的质量得分) / 您的质量得分 + 0.01元。这个公式确保了您的出价只需比下一位竞争者高出0.01元，就能获得更靠前的排名。
2.  **最高扣费限制**：系统按此公式计算出的金额，如果高于您设置的关键词出价，则按您设置的实际出价扣费。
3.  **最低扣费限制**：每次点击的扣费最低为0.1元。如果公式计算出的金额低于0.1元，也会按0.1元扣费。

![](img/36d8a9c85a180b09830196de053f3368_10.png)

平台规则已经过调整，现在很难做到几分钱一个点击，目前最低点击单价通常在0.1元左右。



![](img/36d8a9c85a180b09830196de053f3368_10.png)

![](img/36d8a9c85a180b09830196de053f3368_12.png)

## 质量分核心概念

![](img/36d8a9c85a180b09830196de053f3368_13.png)

![](img/36d8a9c85a180b09830196de053f3368_14.png)

理解了扣费，我们还需要明白公式中的另一个关键变量——质量分。它决定了您用同样的出价能获得多少流量。

质量分是衡量关键词、商品信息与用户搜索意图相关性的综合指标，可以简单理解为推广权重。它以10分制呈现，分数越高，意味着您能获得更多流量、更高的权重和排名。



![](img/36d8a9c85a180b09830196de053f3368_12.png)
![](img/36d8a9c85a180b09830196de053f3368_13.png)
![](img/36d8a9c85a180b09830196de053f3368_14.png)

![](img/36d8a9c85a180b09830196de053f3368_16.png)

那么，哪些因素会影响质量分的高低呢？



![](img/36d8a9c85a180b09830196de053f3368_16.png)

以下是影响质量分的三个主要因素：
1.  **关键词相关性**：指推广关键词与商品标题的文本匹配度。两者越相符，相关性得分越高。
2.  **类目相关性**：商品发布的类目必须与关键词所属类目一致。类目错误会导致质量分很差。
3.  **推广商品质量**：商品本身的点击率、转化率、销量等数据表现，会直接影响质量分。

![](img/36d8a9c85a180b09830196de053f3368_18.png)

本节课中我们一起学习了拼多多搜索推广的三大基础规则：**排名由“出价×质量分”决定**；**扣费遵循特定公式并有上下限**；**质量分受相关性、类目和商品数据影响**。理解这些是优化推广、控制成本的前提。下节课我们将分享更深入的推广实操技巧。
