# 抖音运营教程：P17：影视推广平台详解 🎬

在本节课中，我们将详细学习如何通过“影视推广平台”在抖音、快手等平台获取收益。我们将了解其运作模式、收益结算方式以及具体的操作步骤。

许多同学询问，如果想把视频发布到抖音或快手平台，应该如何结算收益。本节内容将解答这个问题。

我们剪辑完成的视频，除了可以发布到抖音和快手，还可以分发到超过30个其他自媒体平台。在这些平台上，我们可以直接依靠视频的播放量来结算收益。

![](img/62a681c3993f11f6f04c57422a59ed34_5.png)

然而，抖音和快手这两个平台的情况有所不同。正如之前提到的，视频发布在这两个平台上，主要的作用是增长粉丝。

那么，有没有同学只想专注于抖音和快手这两个熟悉的平台，但又希望不仅能涨粉，还能通过播放量获得收益呢？答案是肯定的。

![](img/62a681c3993f11f6f04c57422a59ed34_7.png)

以下是为您介绍的解决方案：影视推广平台。

---

视频可以发布在抖音、快手、小红书以及微信视频号这四个我们常用的平台上。小红书在年轻用户，特别是95后女性群体中较为流行。微信视频号则是微信内置的一个功能，每个人都可以使用。

在这几个平台发布视频后，我们需要通过“影视推广平台”来接取任务，并通过完成任务来结算奖金。平台上的素材非常丰富。

“影视推广”是指为电视剧、电影或综艺节目进行线上宣传。如今，许多制片方或广告商会通过这类平台，招募创作者制作推广视频，以替代传统的线下宣传活动。我们通过制作和发布视频来帮助他们推广，并根据视频的播放量等数据获得奖金。

每个视频任务完成后，都需要提交一个“回填链接”进行报名。在链接中，我们需要填写自己的微信号、账号昵称、发布平台的ID、所推广的剧集名称、作品链接、发布时间以及视频播放量的截图。提交完整信息后，平台方会据此为我们结算收益。

影视推广的收益，通常比单纯依靠平台播放量分成的收益更高。

影视推广的收益计算方式根据平台不同而有所区别，并且对账号粉丝数没有要求。平台会选取你为同一部剧集发布的、播放量最高的三个作品，将其播放量相加作为结算依据。

以下是各平台具体的收益计算标准：

*   **抖音**：每1万次播放，奖金为50元。
*   **微信视频号**：每1万次播放，奖金为10元。（因用户基数庞大，单价较低）
*   **小红书**：每1000次播放，奖金为30元。（单价最高）

此外，当播放量超过基础门槛后，每增加1万次播放，通常还会有额外的奖金。

除了上述按播放量结算的平台，部分任务（如某些搜狐短剧）会在抖音和快手平台按点赞数结算收益，同样对粉丝数无要求。

以下是按点赞数结算的示例：

*   **抖音**：单个作品获得5000个赞，奖金100元；1万个赞，奖金200元。
*   **快手**：单个作品获得1万个赞，奖金180元。（在所示案例中单价最高）

本节课中，我们一起学习了影视推广平台的运作模式。关键点在于，通过接取平台任务，我们可以在抖音、快手等平台实现“涨粉”与“收益”的双重目标。核心步骤是：**接任务 -> 制作发布视频 -> 提交回填链接 -> 结算奖金**。记住，不同平台的结算规则（按播放量或点赞数）和单价各不相同，选择适合自己的任务至关重要。
