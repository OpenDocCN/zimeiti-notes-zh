## 内容确立的五个步骤

上一节我们介绍了笔记写作的基础框架，本节中我们来看看如何具体确立一篇笔记的内容。以下是确立内容的五个关键步骤。

### 1. 确定运营领域

首先，必须明确你的账号专注于哪个领域。例如，是美妆、护肤、母婴还是其他类别。在之前的课程中已多次强调，领域不明确会导致账号垂直度降低。

**账号垂直度公式**：`账号价值 = 内容质量 × 垂直度`

垂直度降低会引发一系列问题：平台推荐减少、粉丝流失，并可能陷入“内容不受欢迎 → 不被推荐”的恶性循环。

### 2. 梳理行业关键词

确定领域后，需要梳理与该领域相关的核心关键词。在之前的课程中，我们介绍过利用百度工具、爱站网、5118、百度指数等工具进行关键词查询。

本节课不再赘述具体工具用法。大家可以根据自身行业，利用这些工具查询并整理出关键词排行榜。用好这些关键词，能极大提升内容的搜索可见度和吸引力。

### 3. 搜集与分析同行案例

接下来，需要搜集并分析优秀同行的笔记案例。有人可能会问，在小红书内无法直接看到同行的详细数据，该怎么办？

### 4. 使用工具进行数据筛选

![](img/238ff5ead3dc21155ad9416c77615d83_7.png)

那么，如何具体使用千瓜数据呢？下面进行详细说明。

在千瓜数据网站左侧的功能栏中，可以看到热词榜、笔记收录查询等多种功能。本节课主要讲解“笔记收录查询”功能。

![](img/238ff5ead3dc21155ad9416c77615d83_7.png)

发布笔记后，若担心笔记未被平台收录，即可使用此功能进行查询。此外，“达人搜索”功能也极为实用，可按昵称或笔记内容搜索目标达人。

例如，搜索“美妆”领域达人，结果会按彩妆、护肤、母婴等属性分类。你可以根据自身需求筛选，并查看达人属性（如官方号、品牌号、头部达人）、排名及详细数据。

点击任意达人进入详情页，可以查看其发布笔记的数据、分类、详情，甚至笔记评论中的高频词云。这些数据能帮助我们深度掌握小红书的内容参数。

![](img/238ff5ead3dc21155ad9416c77615d83_9.png)

**工具使用核心**：`数据驱动决策 = 搜集案例 + 分析参数 + 筛选优化`

### 5. 整理与发布自有内容

搜集好同行案例后，即可对这些内容进行筛选和借鉴，最终整理成自己的笔记并发布。

关于内容的具体写作结构，在之前的课程中已详细讲解，主要采用“总-分-总”的结构。如果觉得写作困难，建议回顾之前的课程内容。

## 其他注意事项

除了上述步骤，在撰写笔记标题时还需注意以下几点：

以下是标题优化的三个要点：

1.  **控制字数**：标题尽量控制在12到18个字之间。
2.  **添加符号美化**：小红书用户以女性居多，在标题中添加适当的符号（如✨、❗、🔥）可以更好地吸引眼球。
3.  **加入具体数据**：在标题中加入具体数字能增加真实感和冲击力。例如，“百万销量”、“30天瘦10斤”等，能让内容显得更可信。
