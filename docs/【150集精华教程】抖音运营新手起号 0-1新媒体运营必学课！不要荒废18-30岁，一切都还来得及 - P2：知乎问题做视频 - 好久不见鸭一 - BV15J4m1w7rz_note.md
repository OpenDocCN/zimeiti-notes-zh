在本节课中，我们将学习如何将知乎平台上的精彩问答内容，制作成短视频并发布到自媒体平台，从而获取创作者收益。这是一种可以长期操作的内容创作方法。

最近，我在操作一个相对冷门的项目。这个项目的核心是将知乎上的一些精彩回答片段录制下来。

然后，将这些视频发布到抖音等自媒体平台，以获取平台给予创作者的流量收益。

![](img/d7a4fd28cb2c60e525313f654637acb1_3.png)

这是一个可以长久操作的项目。如果你能吃苦，也能熬夜，那么可以跟随我一起操作这个“复制粘贴”式的项目。在过去两年里，我坚持了18个月，总共获得了超过100万的收益。仅昨天一天，就为我带来了可观的收入。我们制作的视频，形式如下例所示。

**视频标题示例**：今日话题，不小心跟男闺蜜睡了怎么办。

![](img/d7a4fd28cb2c60e525313f654637acb1_9.png)

![](img/d7a4fd28cb2c60e525313f654637acb1_11.png)

**视频内容示例**：一位网友是这样回答的，哪有什么不小心，其实都是内心里早已经期待发生的事情，只是借助酒精这个噱头付诸于行动罢了。



上一节我们介绍了项目的基本形式，本节中我们来看看它的优势。这类视频自带热度，因为问题的标题本身就具有很强的点击吸引力。

![](img/d7a4fd28cb2c60e525313f654637acb1_13.png)

![](img/d7a4fd28cb2c60e525313f654637acb1_11.png)

视频内容往往充满话题性和冲击力，容易引发观众兴趣。例如，上面展示的视频为我带来了超过20万的播放量。有播放量就能产生收益，我的账号每1万次播放量大约能带来**20元**的收益。因此，那个视频的收益约为 **`20万播放 / 1万 * 20元 = 400元`**。



这种模式的另一个优势是易于规模化操作。账号完全可以进行批量注册，并在多个平台同步发布内容。知乎上的素材可以说是取之不尽，用之不竭，完全不用担心内容枯竭的问题。
