## 推荐机制的核心逻辑
上一节我们介绍了小红书整体的推荐规则与流量来源。本节中，我们来看看笔记被推荐的具体核心逻辑，即笔记如何从基础流量池跃升到更高层级。

平台通过一套分级机制来决定是否给笔记更多流量。其核心考核两个指标：**打开率**与**互动率**。

### 1. 第一层级：打开率
当用户打开小红书首页，推荐流每次会展示约20条笔记。你的笔记会出现在其中。平台首先考核的是**打开率**。

**打开率**的计算公式为：
`打开率 = (笔记点击量 / 笔记展示量) * 100%`

平台要求笔记的打开率**必须超过11%的平均水平**，才能进入下一流量池。例如：
*   若笔记获得1000次展示，则至少需要110次点击（小眼睛）。
*   达标后，平台可能将展示量提升至2000-5000次。
*   若打开率低于11%，笔记将停留在初始流量池，推荐很快结束。

### 2. 第二层级：互动率
当笔记通过打开率考核，进入更大流量池后，平台开始考核**互动率**。

**互动率**主要指评论率，计算公式可简化为：
`互动率 ≈ (评论数 / 笔记点击量) * 100%`

平台要求平均互动率**需达到3%以上**。例如：
*   若笔记有110次点击，则至少需要约4条评论。
*   互动率达标后，笔记有资格继续跃升，展示量可能达到1万至2万甚至更高。

**总结核心流程**：笔记发布后，先靠**高打开率**（>11%）进入下一级，再靠**高互动率**（>3%）争取再次升级。对于新账号，通常有机会跃升2-3个层级；老账号因基础权重高，起点更高。

## 互动行为权重
除了打开率和互动率，用户的其他互动行为也为笔记增加“评分”，影响长期推荐。以下是主要互动行为的权重排序（从高到低）：
1.  **分享**
2.  **收藏**
3.  **评论**
4.  **点赞**
5.  **关注**

![](img/6e263f96b0cbb6caac75af244daca88f_19.png)

**请注意**：这些加权行为建立在打开率与互动率达标的基础上。如果基础数据不达标，互动权重再高也难以获得大量推荐。

## 必须避免的运营误区
了解规则后，我们更需要避开平台红线。以下是新手运营中常见的坑，务必避免。

以下是小红书运营中必须规避的几点：
*   **新号忌立即发笔记**：新注册账号需“养号”，模仿正常用户浏览、互动几天，建立标签。立即发笔记会消耗初始权重，导致后续流量低迷。
*   **违规必然导致降权**：任何违规（如导流、抄袭、违禁内容）都会导致笔记推荐减少、搜索排名下降，账号可能被限流长达7天，恢复难度极大。
*   **标签切忌反复修改**：账号、店铺定位一旦确定，不要频繁更改。尤其是在已有稳定流量后修改标签，极易引发限流。
*   **禁止一机多号**：原则上应保证一部手机只登录一个账号，最多不超过3个。频繁切换或多个账号使用同一设备易被判定为营销号。
*   **谨慎处理引流信息**：避免在笔记、评论、私信中直接留下联系方式或诱导用户跳转至其他平台。此类行为风险极高。
*   **谨慎处理用户互动**：回复评论或私信时需遵守社区规范，避免争吵、发布敏感信息或过度营销。

## 总结
本节课我们一起学习了小红书笔记推荐的核心逻辑与关键禁忌。
1.  **核心逻辑**：平台通过**打开率（>11%）**和**互动率（>3%）**两级指标，决定是否将笔记推入更高流量池。
2.  **避免误区**：新号养号、避免违规、稳定标签、一机一号、谨慎引流是保证账号健康度的基本要求。
