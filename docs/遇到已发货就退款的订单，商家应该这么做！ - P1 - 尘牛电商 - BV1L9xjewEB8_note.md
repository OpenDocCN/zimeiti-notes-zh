# 电商售后处理：P1：已发货订单退款处理教程 🚚

在本节课中，我们将要学习如何处理买家在商品发货后申请仅退款的情况。这种情况如果处理不当，可能导致商家“钱货两空”。我们将分情况讨论具体的操作步骤和核心原则，确保您的权益得到保障。

---

## 概述

当商品已经发出，买家却申请仅退款时，商家需要根据快递的不同状态采取相应措施。核心原则是：**确保货物成功拦截或退回后，再操作退款**。切勿立即同意退款。

---

## 情况一：快递运输途中订单

上一节我们概述了核心原则，本节中我们来看看第一种具体情况：商品仍在快递运输途中。

此时，商家应按顺序执行以下步骤：

以下是处理运输途中退款订单的具体步骤列表：
1.  **联系买家沟通**：告知买家，如果不需要商品，可以协助拦截快递。
2.  **请求买家配合**：说明若拦截失败，需要买家在收货时拒收快递。
3.  **联系快递拦截**：立即联系对应的快递公司，发起拦截申请。
4.  **确认拦截成功**：等待并确认快递公司反馈拦截成功的信息。
5.  **操作退款**：仅在收到明确的拦截成功信息后，再在平台同意买家的退款申请。

**核心操作逻辑**可以用以下流程表示：
```
if (快递状态 == “运输中”) {
    联系买家沟通并请求配合；
    向快递公司发起拦截；
    while (未收到拦截成功反馈) {
        等待；
    }
    同意退款；
}
```

## 情况二：快递正在派送订单

处理完运输途中的订单，我们接下来看看更紧急的一种情况：商品已经到达买家所在地，正在派送。

![](img/21e011f4ec85390c6b1b80cabd3435a8_2.png)

此时，关键点在于必须确保货物被**拒收**。具体步骤如下：

![](img/21e011f4ec85390c6b1b80cabd3435a8_3.png)

以下是处理派送中退款订单的具体步骤列表：
1.  **联系买家拒收**：第一时间联系买家，请其当面拒收该快递包裹。
2.  **跟进买家操作**：若买家未能及时拒收，应立即联系负责派送的快递员，请求其协助拦截并退回包裹。
3.  **确认退回结果**：等待快递方确认包裹已成功拦截并安排退回。
4.  **操作退款**：仅在确认货物已成功退回后，再处理退款。

**核心注意事项**是：
> 无论哪种情况，商家都必须在**确认货物成功退回**的物流信息后，才能进行退款操作。提前退款可能导致财物两失。

![](img/21e011f4ec85390c6b1b80cabd3435a8_4.png)

![](img/21e011f4ec85390c6b1b80cabd3435a8_5.png)

本节课中我们一起学习了如何处理“已发货仅退款”的订单。我们将其分为**快递运输途中**和**快递正在派送**两种情况，并分别给出了具体的处理步骤。请牢记最核心的原则：**见退货，再退款**。遵循这个流程，可以有效避免不必要的损失。
