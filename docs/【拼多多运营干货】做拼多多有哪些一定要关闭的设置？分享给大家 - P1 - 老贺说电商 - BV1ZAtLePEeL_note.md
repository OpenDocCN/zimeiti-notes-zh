# 拼多多运营教程：P1：必须关闭的店铺设置 🛠️

在本节课中，我们将学习拼多多店铺运营中几个必须关闭或谨慎设置的选项。这些设置如果处理不当，可能会导致不必要的成本支出或店铺违规风险。

## 必须关闭的设置项

上一节我们介绍了本课的核心目标，本节中我们来看看具体有哪些关键设置需要调整。

以下是三个必须重点关注并关闭的设置项：

1.  **客服设置中的“新疆西藏直邮”**
    *   此设置若开启，可能导致订单被随意分配至新疆、西藏等偏远地区，产生高昂的物流成本。

2.  **营销工具中的“智能营销”和“短信营销”**
    *   这些自动化营销工具若不经仔细配置，可能在不合适的时机向客户发送推广信息，引起反感或造成不必要的营销费用。

3.  **消费者体验计划中的“退货包运费”和“零元退货”**
    *   盲目开启这些服务会增加你的售后成本。务必根据商品利润和类目情况谨慎评估后再决定是否开启。

**核心建议**：不去设置或错误设置上述选项，可能导致直接的经济损失。

## 近期运营注意事项

了解了必须关闭的设置后，我们再来看看近期需要特别注意的运营规则。

平台近期正在严格查处以下违规行为：
*   **码本图**：即商品主图包含联系方式、二维码等导流信息。
*   **虚假宣传**：夸大或虚构商品功效。
*   **模糊SKU名称**：SKU信息描述不清，存在误导。

**违规后果**：一旦被查实，店铺将面临直接限流。因此，新建商品链接时一定要规避这些行为。

关于更详细的违规案例解读，可参考4月16日的相关专题视频。许多卖家被处罚，往往是因为忽略了这些基础但重要的规则。

## 偏远地区物流设置详解

最后，我们来详细解决“新疆西藏直邮”的具体关闭方法。

如果你不慎开启了该选项，可以通过以下路径关闭：
**操作路径**：店铺后台 -> 物流工具 -> 偏远地区设置。

以下是具体的设置建议：

*   **推荐方案**：将“包邮比例”修改为 **1%**，并取消勾选“支持配送”选项。
*   **彻底不发方案**：保持不勾选状态。若产生此类订单，可选择承担约**8元/单**的缺货罚款。
*   **折中方案**：选择发货至中转仓，通常每单需额外增加约**3元**成本。

本节课中我们一起学习了拼多多店铺中必须关闭的几项设置、近期需规避的违规行为，以及处理偏远地区订单的具体操作方法。正确配置这些项目，是控制成本、避免违规、实现稳定运营的基础。
