# 抖音运营教程：P13：Dou+技巧投放全攻略 🚀

在本节课中，我们将要学习专业玩家如何高效使用抖音的“Dou+”功能进行付费推广。许多教程只公开了80%的内容，而将关键的20%保留在付费课程中，这常常导致初学者即使按教程操作也效果不佳。本节将为你揭示这20%的核心秘密，提供一套从0到1万粉丝的完整Dou+投放策略。

上一节我们介绍了Dou+投放的基本概念，本节中我们来看看如何分阶段进行精细化操作。

![](img/696438beadb3c183101a263b4b19ad79_1.png)

## 第一阶段：1000粉丝以内（混搭投放）

在账号粉丝数低于1000时，应采用“混搭投放”策略。具体操作步骤如下：

以下是第一阶段的详细操作流程：
1.  **等待自然流量**：视频发布后，先让其自然推荐流量跑完。
2.  **首次投放（智能推荐）**：选择**100元**预算，投放目标选择“**点赞评论量**”，投放方式选择“**智能推荐**”。
3.  **二次投放（达人相似）**：完成首次投放后，再选择**100元**预算，投放目标同样为“点赞评论量”，但投放方式改为“**达人相似**”，并手动选择**5个对标账号**。

![](img/696438beadb3c183101a263b4b19ad79_3.png)

**选择对标账号的核心标准**：优先选择那些**短期内粉丝数刚突破1万**的账号。因为这类账号的粉丝粘性相对较弱，且对新账号的包容性更高，有利于你的视频获得初始互动。

![](img/696438beadb3c183101a263b4b19ad79_5.png)

**第一步先投100元智能推荐的原因**：此举的目的是让系统为你的视频数据“美化”一下，即通过付费带来更多点赞和评论。数据更好的视频会利用用户的“从众心理”，吸引更多自然用户停留和观看。

## 第二阶段：3000粉丝以内（定向投放）

当粉丝数增长到1000至3000之间时，策略应转向更精准的“定向投放”。

![](img/696438beadb3c183101a263b4b19ad79_7.png)

以下是第二阶段的投放方法：
1.  **等待自然流量**：同样，先让视频的自然流量跑完。
2.  **核心投放**：直接选择**200元**预算，投放目标为“点赞评论量”，投放方式选择“**达人相似**”。对标账号的选择标准与第一阶段相同。
3.  **附加定向条件（关键步骤）**：在达人相似的基础上，必须根据你的账号定位，额外设置**年龄、性别、地域**等筛选条件。
    *   **示例**：如果你是“家居好物”类账号，主要受众是**30-40岁的女性**，投放时就应勾选这些条件。
    *   **示例**：如果你是“同城”类账号，则必须在地域条件中限定你所在的城市。

**这样操作的目的**：附加定向条件能让你吸引的粉丝更加精准，从而帮助抖音系统更快地为你的账号打上正确标签，加速向1万粉丝的目标冲刺。这是许多付费课程中不会明确告知的关键技巧。

## 第三阶段：1万粉丝以上（智能投放）

当粉丝数超过1万后，账号标签已相对稳定，此时应采用更高效的“智能投放”策略。

![](img/696438beadb3c183101a263b4b19ad79_9.png)

以下是第三阶段的操作指南：
1.  **筛选视频**：发布一系列视频后，观察其自然流量数据，选择其中**播放表现最好**的一条。
2.  **执行投放**：为该优质视频投放**200元**预算，投放目标选择“**点赞评论量**”，投放方式选择“**智能推荐**”。

**此阶段不投“达人相似”的原因**：当账号粉丝过万后，系统已经能够较为准确地识别你的内容受众。此时使用“智能推荐”，系统会基于你账号的现有标签和视频内容，自动将广告展示给最可能感兴趣的用户，投放效率更高，成本控制更优。
