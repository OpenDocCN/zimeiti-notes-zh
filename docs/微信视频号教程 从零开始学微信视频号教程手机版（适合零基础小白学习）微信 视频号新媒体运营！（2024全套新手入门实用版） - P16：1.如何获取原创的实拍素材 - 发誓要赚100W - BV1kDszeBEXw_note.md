# 微信视频号教程：P16：如何获取原创的实拍素材 📸

在本节课中，我们将学习如何为视频号内容获取原创的实拍素材。原创素材能有效提升视频的原创度，降低被平台判定违规的风险，是进行视频混剪和提升内容质量的关键步骤。

## 概述

视频号运营中，除了搬运和混剪，许多团队会选择实拍原创内容。但自行购买样品实拍成本较高。因此，我们将学习两种低成本获取原创素材的方法，以便在混剪时加入，增加视频的原创性。

## 方法一：使用“产选原创素材”小程序

![](img/9dae3e52c6cd425cff49a46512ad16bd_1.png)

第一种获取原创素材的方式是通过微信小程序。以下是具体操作步骤：

![](img/9dae3e52c6cd425cff49a46512ad16bd_3.png)

1.  **打开微信**，在搜索框中输入“**产选原创素材**”。
2.  进入该小程序，注册并登录账号。
3.  根据提示绑定你的抖音账号。
4.  绑定后，你可以在小程序内浏览不同类目的视频素材，例如食品饮料、日用百货、美妆护肤、母婴用品等。
5.  选择你需要的视频，点击“**下载视频**”即可保存到手机。

**优点**：视频由专业达人拍摄，画质通常比较**高清**。
**缺点**：素材库的**视频数量相对有限**。

![](img/9dae3e52c6cd425cff49a46512ad16bd_5.png)

![](img/9dae3e52c6cd425cff49a46512ad16bd_6.png)

上一节我们介绍了通过专业小程序获取素材，虽然质量高但选择有限。接下来，我们看看第二种更灵活、素材来源更广泛的方法。

## 方法二：从电商平台买家秀获取

第二种方法是从淘宝或拼多多等电商平台的商品评论区，下载买家发布的实拍视频作为素材。以下是具体操作流程。

### 第一步：找到目标商品

首先，你需要确定要带货的商品。可以参考视频号选品中心的“**爆款榜**”或“**短视频热卖**”榜单。例如，假设我们选择“**村庄农品蒜香椒盐粉**”这个商品。

### 第二步：在淘宝获取素材（需录屏）

![](img/9dae3e52c6cd425cff49a46512ad16bd_8.png)

在淘宝App中搜索目标商品，进入商品详情页。

1.  下滑至**评论区**，找到带有“**视频**”标签的买家秀。
2.  点击播放视频。
3.  轻点屏幕，进入**全屏（清屏）播放模式**，并确保打开声音。
4.  使用手机的**录屏功能**，将这段买家秀视频录制下来。

![](img/9dae3e52c6cd425cff49a46512ad16bd_10.png)

**注意**：手机版淘宝的买家秀视频通常没有直接下载选项，因此录屏是目前可行的方法。

### 第三步：在拼多多获取素材（可直接下载）

在拼多多App中，同样搜索并进入目标商品的详情页。

1.  在评论区找到带视频的买家秀。
2.  点击视频右下角的“**分享**”按钮。
3.  选择“**发送给朋友**”，将其分享到微信（可以发给自己或文件传输助手）。
4.  在微信中打开这个链接。
5.  此时视频可以进入全屏模式，你可以选择**录屏**，或者使用更便捷的下载方法。

**更高效的下载方法**：
1.  复制拼多多买家秀视频的链接。
2.  打开 **QQ浏览器**（其他浏览器可能不支持），将链接粘贴到地址栏并打开。
3.  播放视频，点击右下角的“**...**”更多选项。
4.  选择“**下载**”，视频便会保存到你的手机相册。

**权限设置提示**：如果使用QQ浏览器下载后，视频无法在相册或剪辑软件中找到，请检查手机的应用权限设置。进入手机 **设置 > 应用管理 > QQ浏览器 > 权限管理**，确保其拥有**存储（或相册）访问权限**。

## 总结

本节课我们一起学习了两种为视频号获取原创实拍素材的方法：
1.  通过 **“产选原创素材”微信小程序** 获取高清专业素材。
2.  从 **淘宝、拼多多等电商平台的买家秀** 中，通过录屏或借助QQ浏览器下载的方式获取真实用户实拍素材。

掌握这些方法后，你就能在后续的混剪视频中加入原创片段，有效提升视频的原创度。下节课，我们将学习如何发布一条完整的带货视频，并演示如何将这些原创素材剪辑到视频中。
