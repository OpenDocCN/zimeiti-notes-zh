# 自媒体运营从0到1全攻略：P23：垂直类目细分领域 🎯

在本节课中，我们将要学习垂直类目的细分领域。理解这个概念，能帮助你更精准地定位自己的账号内容，从而吸引更明确的受众群体。

上一节我们介绍了内容标签的重要性，本节中我们来看看如何将“垂直”这个概念进行更细致的划分。

我们常说做账号要“垂直”。很多同学会简单地认为，选择“美食”或“游戏”作为领域，并一直发布相关内容，就是垂直了。这种理解没有错，但这仅仅是垂直分类的起点，属于**一级垂直领域**。

除了**一级垂直领域**，垂直度还有更细致的级别划分。以下是垂直类目的分级结构：

*   **一级垂直领域**：最宽泛的大类。例如：**游戏**、美食、美妆。
*   **二级垂直领域**：在一级领域下的具体分支。例如，在“游戏”这个一级领域下，可以细分为：**手游**、电竞、网游。
*   **三级垂直领域**：在二级领域下的进一步聚焦。例如，在“手游”这个二级领域下，可以细分为：**《王者荣耀》**、**《和平精英》**、**《原神》**。
*   **四级垂直领域（或更细）**：在三级领域下的深度垂直。例如，在“《王者荣耀》”这个三级领域下，可以只做：**安琪拉英雄教学**、打野位攻略、或特定游戏模式的娱乐剧情。

为了更清晰地理解，我们可以用一个简单的公式来描述这种层级关系：
**垂直深度 = 一级领域 → 二级领域 → 三级领域 → …**

我们以“游戏”领域为例，来具体说明各级划分：
*   **一级垂直领域**：**游戏**。在这个层面，你可以制作任何与游戏相关的内容。
*   **二级垂直领域**：例如**手游**。此时，你的内容范围聚焦在手机游戏上。
*   **三级垂直领域**：例如**《王者荣耀》**。你的账号将全部围绕这一款游戏展开。
*   **四级垂直领域**：例如**《王者荣耀》中的安琪拉教学**。你的内容会极度专精，只讨论这一个英雄的玩法、出装和技巧。

这种细分可以无限进行下去。例如，玩《和平精英》只玩“雨林”地图，或只制作游戏内的“剧情模式”内容。你的账号内容越细分，吸引的受众就越精准，粉丝粘性也可能越高。

所以，垂直类目的细分领域有一级、二级、三级甚至更多级别。大领域和小领域都可以作为你的选择，这完全取决于你自己的兴趣和规划。

![](img/078dcee9a5e6a86d605500f6681f8f9d_3.png)

本节课中我们一起学习了垂直类目的细分领域。核心在于理解，垂直不是一个固定不变的大类，而是一个可以层层深入、不断聚焦的体系。从宽泛的“游戏”到具体的“某款游戏的某个英雄”，选择哪个层级进行深耕，将决定你账号的独特定位和内容方向。
