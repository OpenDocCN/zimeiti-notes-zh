# 拼多多店铺优化教程：P1：必须关闭的四个设置 🛑

在本节课中，我们将学习如何关闭拼多多店铺后台中四个可能增加不必要成本或影响利润的设置。这些设置若保持开启，可能导致利润被侵蚀或广告花费失控。

## 概述

拼多多后台的部分功能旨在提升消费者体验或帮助商家快速起量，但有时会以牺牲商家的利润为代价。对于希望精细化运营、控制成本的商家而言，关闭这些自动功能是必要的。下面我们将逐一介绍这些设置及其关闭方法。

## 必须关闭的设置列表

以下是四个需要重点检查并关闭的设置项，我们将详细说明每个设置的影响与关闭步骤。

### 1. 消费者体验计划（原“退货包运费”）

“退货包运费”功能已升级为“消费者体验计划”。该计划可能按订单扣除高额费用，严重影响利润。

**关闭步骤：**
1.  进入拼多多商家后台。
2.  在顶部的搜索框中输入“消费者体验”并搜索。
3.  进入相关页面后，在页面最右侧找到“更多”选项。
4.  点击“停用消费者体验计划”即可。

> **核心影响**：此设置可能导致**每成交一单，就被扣除近一半的利润**，使得经营无利可图。

### 2. 短信营销计划

“短信营销计划”中的智能提醒服务会向消费者发送营销短信，可能产生额外费用。

**关闭步骤：**
1.  在商家后台搜索框输入“短信营销”并搜索。
2.  找到“短信营销计划”入口。
3.  在“智能提醒短信服务”板块，点击“查看更多数据”。
4.  在页面右侧找到“停用服务”按钮，点击关闭。

### 3. 自动跟价

“自动跟价”功能位于营销活动中，开启后系统会自动调整商品价格以参与竞争，可能引发不必要的价格战。

**关闭步骤：**
1.  进入后台的“营销活动”页面。
2.  在相关活动区域右侧，找到显示为“自动跟价已开启”的状态提示。
3.  将鼠标悬停在该状态旁的问号（`?`）图标上。
4.  在弹出的说明或操作区域中，找到并点击“直接关闭”按钮。

### 4. 全局优选起量与极速起量

在创建直通车推广计划时，“全局优选起量”和“极速起量”是系统默认勾选的选项。它们会为了快速获取流量而大幅提高出价，导致广告花费急剧升高。

**关闭步骤：**
1.  在新建直通车推广计划的过程中，仔细查看计划设置页面。
2.  手动找到并**取消勾选**“全局优选起量”选项。
3.  同时，检查并**取消勾选**“极速起量”选项。
4.  确认无误后，再完成计划的创建。

> **核心提示**：若您发现直通车花费异常高昂，首先应检查是否开启了此设置。关闭方法是**在创建每一个新计划时，手动取消勾选**。

## 总结

本节课我们一起学习了拼多多商家后台中四个关键设置的关闭方法。回顾一下，它们分别是：**消费者体验计划**、**短信营销计划**、**自动跟价**以及**直通车的全局优选/极速起量**。定期检查并关闭这些自动功能，能帮助您更有效地控制成本、保护利润，实现店铺的精细化运营。
