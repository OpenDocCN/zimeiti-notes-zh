# 亚马逊运营教程：第四课：常见促销与优惠券设置方法

![](img/5390de87783e0f5c136740cca2aee86e_1.png)

在本节课中，我们将学习亚马逊平台上三种核心的促销工具：Prime会员专享折扣、全店通用优惠券以及品牌定制促销。我们将逐一讲解它们的设置要求、操作步骤以及各自的优缺点，帮助你有效提升店铺转化率。

![](img/5390de87783e0f5c136740cca2aee86e_3.png)

## 第一部分：Prime会员专享折扣 🎯

上一节我们介绍了课程概述，本节中我们来看看第一种促销工具——Prime会员专享折扣。这是一种针对亚马逊付费会员的专属促销，通常在特定大促日（如Prime Day）期间效果显著。

### 设置资格要求

以下是创建Prime专享折扣前，你的账户和商品必须满足的条件：

1.  **商品评价**：商品必须至少拥有一条评论，且平均星级评分不低于**4星**。
2.  **折扣力度**：提供给Prime会员的折扣价，必须比面向非会员的当前价格至少优惠**20%**。
    *   **公式**：`Prime会员价 ≤ 非会员价 × 80%`
3.  **历史最低价**：本次专享折扣的价格，必须比该商品**历史最低价**再低**5%**。
    *   **公式**：`本次专享折扣价 ≤ 历史最低价 × 95%`
4.  **店铺绩效（针对自发货卖家）**：
    *   迟发率不得超过 **4%**。
    *   订单缺陷率不得超过 **1%**。
    *   配送前取消率不得超过 **0.5%**。
    *   商品必须设置为**免运费**。

### 创建步骤

满足上述条件后，你可以按以下步骤创建折扣：

![](img/5390de87783e0f5c136740cca2aee86e_5.png)

![](img/5390de87783e0f5c136740cca2aee86e_7.png)

1.  进入卖家后台，在 **“广告”** 菜单下找到 **“Prime专享折扣”**。
2.  点击 **“创建折扣”**。
3.  填写促销活动名称、日期和时间。
4.  通过上传表格的方式添加商品（最多500个SKU），并设置折扣力度。
5.  提交并等待审核。

![](img/5390de87783e0f5c136740cca2aee86e_8.png)

![](img/5390de87783e0f5c136740cca2aee86e_10.png)

**注意**：成功创建并产生销售后，亚马逊会收取**50美元**的活动费用。若活动期间无销售，则不收费。

## 第二部分：全店通用优惠券 🛒

上一节我们介绍了针对特定会员的折扣，本节中我们来看看受众更广的工具——优惠券。它可以设置给所有买家或仅Prime会员，是日常促销的常用手段。

### 设置资格要求

![](img/5390de87783e0f5c136740cca2aee86e_12.png)

![](img/5390de87783e0f5c136740cca2aee86e_13.png)

![](img/5390de87783e0f5c136740cca2aee86e_14.png)

以下是卖家与商品参与优惠券活动的基本要求：

![](img/5390de87783e0f5c136740cca2aee86e_16.png)

*   **对卖家的要求**：
    *   必须是专业卖家。
    *   店铺总体反馈评级不低于 **3.5星**。**新店铺（无评级）也可设置**。
*   **对商品的要求**：
    *   **无评论的新品**：可以设置。
    *   **有1-4条评论的商品**：平均评分需 ≥ **2.5星**。
    *   **有5条以上评论的商品**：平均评分需 ≥ **3星**。

### 创建方法

你可以选择单个创建或批量创建优惠券。

**单个创建**：
1.  进入卖家后台 **“广告” > “优惠券”**。
2.  点击 **“创建优惠券”**。
3.  搜索并选择要设置的商品ASIN。
4.  设置优惠券类型（折扣或满减）、额度、预算、使用日期等。
5.  设置“目标买家”（所有买家或仅Prime会员）及是否可与促销叠加（通常选“否”）。

![](img/5390de87783e0f5c136740cca2aee86e_18.png)

**批量创建（适用于商品较多时）**：
1.  在“优惠券”页面，点击 **“批量创建优惠券”** 并下载模板文件。
2.  填写模板。以下是关键字段的填写说明：
    *   `sku`：填入商品SKU，多个SKU用分号隔开。
    *   `discount_type`：折扣类型，填 `Percentage` 表示百分比折扣。
    *   `discount_value`：折扣数值，例如填 `5` 表示5%折扣，系统会自动添加“%”符号。
    *   `budget`：预算金额，例如 `800`。
    *   `start_date` / `end_date`：开始/结束日期，格式需严格匹配，如 `2024-09-25T00:00:00`。
    *   `customer_eligibility`：客户资格，`All` 表示所有买家，`Prime` 表示仅Prime会员。
3.  填写完毕后，回到后台批量上传页面，提交文件即可。

![](img/5390de87783e0f5c136740cca2aee86e_19.png)

![](img/5390de87783e0f5c136740cca2aee86e_21.png)

![](img/5390de87783e0f5c136740cca2aee86e_23.png)

**常见错误**：
*   ASIN未用分号正确分隔。
*   在`discount_value`等字段中自行添加了“%”或“$”符号。
*   日期格式填写错误。
*   必填项（红色字段）未填写。

**建议**：优惠券折扣力度设置在 **5%-10%** 即可，优先为有销售记录、热卖或符合平台活动的商品设置。

## 第三部分：品牌定制促销 🔥

上一节我们介绍了面向大众的优惠券，本节我们来看看更高级、更精准的促销方式——品牌定制促销。它允许你向特定的客户群体推送专属折扣。

### 设置资格与特点

品牌定制促销并非所有卖家都能使用，它有较高门槛：

1.  **品牌备案**：店铺必须完成品牌备案。
2.  **客户群体规模**：你的品牌必须拥有一定规模的受众群体（通常指在过去90天内，有超过1000名客户浏览过你的品牌店铺或商品）。
3.  **商品评级要求**：与优惠券要求相同（无评论商品可参与；有1-4条评论需≥2.5星；有5条以上评论需≥3星）。

![](img/5390de87783e0f5c136740cca2aee86e_25.png)

**优势**：
*   **转化率高**：推送非常精准，面向已对你品牌产生兴趣的客户。
*   **受众面大**：能覆盖浏览过、加购过或购买过你品牌的客户。

![](img/5390de87783e0f5c136740cca2aee86e_27.png)

**劣势**：
*   **折扣门槛高**：最低折扣必须为 **10%**，最高50%。
*   **商品选择有限**：虽然可以排除部分商品不参与，但最多只能排除 **100个** SKU。或者，你只能选择最多 **100个** 特定SKU参与活动。

### 创建步骤

1.  进入卖家后台 **“广告” > “品牌定制促销”**。
2.  选择你的品牌，点击 **“创建定制促销”**。
3.  选择促销目标客户类型（例如：新客户、现有客户、回访客户等）。
4.  选择参与商品：
    *   **添加所有商品并排除特定ASIN**（最多排除100个）。
    *   或 **仅添加特定ASIN**（最多添加100个）。
5.  选择具体的客户细分群体（例如：“过去90天内加购未购买的客户”）。
6.  设置促销名称、折扣力度（10%-50%）、预算和活动日期。
7.  提交创建。

**操作建议**：优先从品牌分析报告中，找出那些被频繁加购但未下单的商品ASIN，将它们加入品牌定制促销，转化效果通常最好。

## 总结 📝

本节课我们一起学习了亚马逊三种主要的促销设置方法：
1.  **Prime会员专享折扣**：适合在大促日提升爆款销量，要求高，费用高，但转化效果好。
2.  **全店通用优惠券**：日常促销利器，设置灵活，适合用于提升单品或全店流量。
3.  **品牌定制促销**：高阶精准营销工具，能有效提升品牌客户复购和转化，但门槛和折扣力度要求较高。

![](img/5390de87783e0f5c136740cca2aee86e_29.png)

请根据你的店铺发展阶段、商品情况和营销目标，灵活选择和搭配使用这些工具。建议新手卖家从设置优惠券开始，逐步尝试其他促销方式。
