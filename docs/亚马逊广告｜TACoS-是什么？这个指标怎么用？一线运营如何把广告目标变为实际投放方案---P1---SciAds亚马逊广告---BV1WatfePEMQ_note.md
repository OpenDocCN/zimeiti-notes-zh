![](img/a3d67fc002d2f94414f439dfb1bf3831_0.png)

在本节课中，我们将要学习亚马逊广告中的一个核心业务指标——TACoS。我们将探讨它的定义、管理意义，以及一线运营如何将其转化为可执行的广告投放方案。课程内容将帮助你理解如何平衡广告支出与销售利润，实现高效管理。

## 什么是TACoS？🤔

![](img/a3d67fc002d2f94414f439dfb1bf3831_2.png)

TACoS是亚马逊卖家社群中广泛使用的一个业务指标，它没有官方的定义。通常，它指的是在特定时间段内，针对某一款产品（ASIN），其广告总花费占总销售额的比例。

![](img/a3d67fc002d2f94414f439dfb1bf3831_3.png)

![](img/a3d67fc002d2f94414f439dfb1bf3831_4.png)

其核心计算公式为：
**TACoS = (广告总花费 / 总销售额) × 100%**

这里的“总销售额”不限于广告带来的销售额，而是指卖家后台仪表盘上该产品的总销售额。管理者之所以关注这个指标，是因为在扣除采购、运费、佣金和退货等成本后，产品的毛利润通常是清晰且稳定的。由于运营目标通常同时包含利润率和销售额，许多公司会通过设定TACoS的上限，来防止一线运营人员过度投放广告导致产品亏损。

总而言之，TACoS是管理者为了提高管理效率、控制利润而设计的一个业务指标。它的出现，也反映了卖家对利润率的日益重视，以及广告在业务中扮演着越来越重要的角色，标志着行业走向成熟。

## 广告投放应以CPS为抓手 🎯

上一节我们介绍了管理者视角下的TACoS及其设计初衷。本节中我们来看看一线运营人员应该如何操作。事实上，一切广告投放都应以**CPS**为抓手。

![](img/a3d67fc002d2f94414f439dfb1bf3831_10.png)

相比ACoS（广告成本销售比），我们更推荐使用CPS来衡量广告效果。这里需要引入一个相关概念：**TCPS**。TCPS是指总花费除以总订单数（包括自然订单）得出的每个订单的成本，即总出单成本。

![](img/a3d67fc002d2f94414f439dfb1bf3831_11.png)

CPS的定义是：
**CPS = 广告花费 / 广告带来的订单数**
它代表每个广告订单的成本。

CPS相比ACoS具有更好的通用性，因为它可以统一衡量广告、折扣等多种营销活动的成本。CPS与ACoS可以直接换算，公式如下：
**ACoS = (CPS / 产品售价) × 100%**

例如，如果CPS是10美元，产品售价是40美元，那么ACoS就是25%。

![](img/a3d67fc002d2f94414f439dfb1bf3831_13.png)

![](img/a3d67fc002d2f94414f439dfb1bf3831_14.png)

管理层面为了统筹销售额和利润率，使用TCPS来度量总体投放成本是合理的。但在具体广告投放执行层面，CPS是一个比TCPS更直接、更好的操作抓手。管理者设定了一个高层目标（O），但一线运营不能直接使用这个不可操作的目标，他们需要一个具体的“抓手”来实现它。

以下是CPS与TCPS的关系说明：
*   通常情况下，CPS越高，广告订单在总订单中的占比就越高。
*   结合广告中更高的CPS，可以清楚地知道CPS和TCPS是**正相关**的。即广告的CPS越高，总的TCPS也倾向于越高。

因此，广告投放在实操中应当以CPS为抓手。运营人员需要根据工作经验和业务数据，结合管理层对TCPS的限制，推导并设定出当前阶段广告活动中合适的CPS目标。运营有其节奏，例如追求三个月内的TCPS可控，那么某个月略微超出目标也是可以接受的。这个过程非常个性化，依赖于运营人员的判断，无法被固定规则或机器完全替代。

## CPS目标如何转化为广告方案？⚙️

![](img/a3d67fc002d2f94414f439dfb1bf3831_18.png)

上一节我们谈到，运营的核心工作之一是制定CPS目标。本节我们来看这个目标如何落地。管理层制定TCPS或TACoS目标，一线运营将其转化为CPS或ACoS目标。那么，接下来的广告投放、关键词筛选和出价设置等工作由谁来完成呢？

这部分执行工作可以借助专业的SaaS广告软件来完成。从管理角度看，这实现了专业化分工：管理者负责战略目标，运营者负责战术目标，而具体的广告优化执行则交给软件工具。这样能提升广告活动的效果和效率。

![](img/a3d67fc002d2f94414f439dfb1bf3831_20.png)

例如，在某些SaaS软件的广告活动创建表单中，你可以直接填写CPS目标。其他设置（如广告活动名称、SKU、预算等）与在亚马逊后台创建广告活动类似。将选词和出价工作“外包”给这类软件，相当于你拥有了一位不知疲倦的优秀广告优化师助手。

![](img/a3d67fc002d2f94414f439dfb1bf3831_21.png)

## 总结 📝

本节课中我们一起学习了TACoS指标及其在亚马逊广告管理中的应用。

![](img/a3d67fc002d2f94414f439dfb1bf3831_23.png)

![](img/a3d67fc002d2f94414f439dfb1bf3831_24.png)

随着跨境电商行业逐渐成熟，广告的作用和花费日益增长。卖家管理者提出TACoS概念，以适应业务发展，合理管理利润并控制广告成本。

![](img/a3d67fc002d2f94414f439dfb1bf3831_25.png)

![](img/a3d67fc002d2f94414f439dfb1bf3831_26.png)

作为管理概念的TACoS或TCPS，在实际操作中需要由一线运营人员转化为可执行的CPS目标来使用。确定了CPS目标后，运营人员可以通过使用SaaS广告软件，自动生成与目标相匹配的完整广告方案，包括关键词和出价策略。

![](img/a3d67fc002d2f94414f439dfb1bf3831_28.png)

![](img/a3d67fc002d2f94414f439dfb1bf3831_29.png)

这一过程实现了业务目标的层层分解与专业化协作，让管理者、运营者和工具各司其职，最终达到运营效率和广告效果的最大化。
