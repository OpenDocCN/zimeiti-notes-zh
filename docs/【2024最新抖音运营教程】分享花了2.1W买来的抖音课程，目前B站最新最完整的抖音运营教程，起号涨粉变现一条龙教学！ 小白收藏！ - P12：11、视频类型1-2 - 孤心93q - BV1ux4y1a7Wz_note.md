# 抖音运营教程：P12：11、视频类型1-2

## 概述
在本节课中，我们将要学习抖音内容创作中一个至关重要的环节：**人设**的建立。我们将区分强人设与弱人设，并重点讲解一种基于人设的、非常实用的视频类型——**口播**。理解这些概念，将帮助你更清晰地规划自己的账号内容。

---

## 人设：连接涨粉与变现的桥梁
上一节我们解决了文案问题，本节中我们来看看下一个环节：内容。很多课程直接教授涨粉或变现，但从涨粉到变现并非直线，中间还有一个关键环节，那就是**建立人设**。

建立人设的目的并非为了炫耀，而是为了在众多同类创作者中脱颖而出。例如，有十个人都做装修内容，其中一位创作者（比如小王）通过建立“特别爱搞笑”的人设，就能让观众迅速记住他，从而淘汰其他九位。当观众喜欢你、记住你，他们自然就成为你的粉丝。这就是人设的作用。

### 强人设与弱人设
以下是两种主要的人设类型，理解它们有助于你分析和规划自己的内容。

![](img/a3a266e5b94519872d978c14a17cfeef_1.png)

**强人设**：以创作者**个人**为切入点，内容与自身密切相关。
*   **核心公式**：**内容主题 = 个人经验/生活/爱好**
*   **表现形式**：通常为真人出镜，分享行业经验、工作日常或个人兴趣爱好。
*   **常见领域举例**：
    *   行业经验：装修、办公、房产、汽车、餐饮、婚介、法律咨询、电商等。
    *   兴趣爱好：美食、旅行、手工、宠物、星座、美妆、护肤、服装穿搭、游戏、摄影等。

**弱人设**：内容与创作者**个人**关联度较低，核心目的是运营好账号。
*   **核心特征**：内容主体并非创作者本人。
*   **常见形式举例**：
    *   情感语录视频（配音乐和文字，无真人出镜或配音）。
    *   影视解说（核心是电影内容和解说，与解说者本人特质关系不大）。

![](img/a3a266e5b94519872d978c14a17cfeef_3.png)

区分清楚强人设与弱人设，是规划后续内容方向的基础。

## 内容创作：从人设到视频
只要内容与你个人相关，都可以作为视频素材。接下来，我们讲解第一种基于人设的视频类型：**讲故事**。

### 讲故事与口播
讲故事，即分享客户故事、从业经历或人生感悟。故事本身不要求完全真实，可以从文案平台寻找有趣的内容进行分享。

讲故事主要依靠“说”，这种形式被称为**口播**。

口播的分类非常广泛，几乎涵盖所有知识或经验分享领域：
*   **讲故事类口播**：目的是让观众觉得有趣、有共鸣。
*   **教知识类口播**：目的是让观众在几十秒内获得“收获感”，觉得学到了东西。
*   **常见领域**：养生、情感、育儿、房产、汽车、金融财经、装修、法律、保险等。

**代码示例（口播视频结构）**：
```
视频开始 -> 提出痛点/问题/故事开头 -> 讲解知识/讲述故事 -> 总结观点/给出建议 -> 视频结束
```

### 关于口播的常见问题
以下是关于口播拍摄的几个关键解答：

*   **口才不好能做口播吗？**
    **可以。** 口播是依据提前准备好的稿子进行录制，并非临场发挥。大部分口播者的流畅表达都源于充分准备。

*   **普通话不好能做口播吗？**
    **可以，有时甚至更有优势。**
    *   如果你的目标客户在同城，使用方言会增加亲切感，效果可能优于普通话。
    *   某些方言本身具有趣味性，能吸引观众。
    *   可以通过添加字幕来解决观众可能听不懂的问题。

口播是一种易于上手的视频形式，只需准备好文案，用手机即可拍摄。在后续课程中，我们会详细讲解具体的拍摄方法。

## 总结
本节课中，我们一起学习了抖音运营的核心概念之一：**人设**。我们明确了**强人设**（以个人为中心）与**弱人设**（以内容为中心）的区别，并深入探讨了基于人设的一种高效视频形式——**口播**。口播通过讲述故事或分享知识，能够有效吸引观众并建立信任，是连接内容创作与粉丝增长的重要工具。记住，清晰的个人定位加上合适的表达形式，是账号脱颖而出的关键。
