# 抖音运营与剪辑：P24：18、视频剪辑方法-9 ✂️

上一节我们介绍了视频制作的前期准备，本节中我们来看看具体的剪辑操作。剪辑环节将进行直接实操演示，请准备好笔记跟随操作。

![](img/358ecdc1dc165ae7e6237b2515038273_7.png)

我们使用**剪映**（黑色图标版本）进行剪辑。这是抖音官方出品的软件，有手机版和电脑版。本节以手机版为例进行讲解。

首先，打开剪映App，点击 **`开始创作`** 按钮。

![](img/358ecdc1dc165ae7e6237b2515038273_9.png)

![](img/358ecdc1dc165ae7e6237b2515038273_11.png)

![](img/358ecdc1dc165ae7e6237b2515038273_1.png)

![](img/358ecdc1dc165ae7e6237b2515038273_13.png)

接着，从相册中选择需要剪辑的视频素材，点击 **`添加`** 导入。如果你的视频由多个片段（如开头、中间、结尾）组成，建议先导入并剪辑好第一个片段，再点击轨道上的 **`+`** 号添加下一个片段，依次处理，这样逻辑更清晰。

![](img/358ecdc1dc165ae7e6237b2515038273_3.png)
![](img/358ecdc1dc165ae7e6237b2515038273_5.png)
![](img/358ecdc1dc165ae7e6237b2515038273_7.png)

![](img/358ecdc1dc165ae7e6237b2515038273_15.png)

导入视频后，第一步是调整画面构图与比例。以下是操作步骤：

![](img/358ecdc1dc165ae7e6237b2515038273_17.png)

1.  点击轨道上的视频片段。
2.  向右滑动底部工具栏，找到并点击 **`编辑`**。
3.  点击 **`旋转`** 调整画面至正确方向。
4.  如需裁剪比例，点击 **`裁剪`**，可选择 **`9:16`**（抖音竖屏）、**`1:1`**、**`16:9`** 等比例。

![](img/358ecdc1dc165ae7e6237b2515038273_19.png)

![](img/358ecdc1dc165ae7e6237b2515038273_9.png)
![](img/358ecdc1dc165ae7e6237b2515038273_11.png)
![](img/358ecdc1dc165ae7e6237b2515038273_13.png)

![](img/358ecdc1dc165ae7e6237b2515038273_21.png)

调整好画面后，接下来进行视频内容的裁剪。播放视频，找到需要删除的冗余部分（如开头沉默、结尾无关动作）。时间轴上的白色竖线即切割点。

![](img/358ecdc1dc165ae7e6237b2515038273_23.png)

以下是裁剪无用片段的方法：
*   **选中**视频片段，将白色竖线移动到切割点。
*  点击底部工具栏的 **`分割`**，视频将被分成两段。
*   **选中**需要删除的片段，点击 **`删除`**。

![](img/358ecdc1dc165ae7e6237b2515038273_25.png)

重复 **`选中 -> 分割 -> 选中 -> 删除`** 的流程，即可去除所有不需要的部分。

![](img/358ecdc1dc165ae7e6237b2515038273_27.png)

![](img/358ecdc1dc165ae7e6237b2515038273_29.png)

![](img/358ecdc1dc165ae7e6237b2515038273_15.png)
![](img/358ecdc1dc165ae7e6237b2515038273_17.png)

![](img/358ecdc1dc165ae7e6237b2515038273_31.png)

视频裁剪完毕后，需要添加背景音乐。点击底部工具栏的 **`音频`**。

![](img/358ecdc1dc165ae7e6237b2515038273_33.png)

![](img/358ecdc1dc165ae7e6237b2515038273_35.png)

添加音乐有以下两种常用方式：
*   **提取音乐**：如果你已用其他工具（如淘金阁）生成配音，可点击 **`提取音乐`**，直接导入视频文件中的音频。
*   **添加音乐**：点击 **`音乐`**，在曲库中按分类（美食、美妆、萌宠等）寻找，或直接搜索。点击歌曲旁的 **`预览`** 图标试听，选择后点击 **`使用`** 即可导入。

![](img/358ecdc1dc165ae7e6237b2515038273_37.png)

![](img/358ecdc1dc165ae7e6237b2515038273_19.png)
![](img/358ecdc1dc165ae7e6237b2515038273_21.png)
![](img/358ecdc1dc165ae7e6237b2515038273_23.png)
![](img/358ecdc1dc165ae7e6237b2515038273_25.png)
![](img/358ecdc1dc165ae7e6237b2515038273_27.png)

![](img/358ecdc1dc165ae7e6237b2515038273_39.png)

如果添加的音乐过长，同样可以使用 **`分割`** 与 **`删除`** 进行裁剪。选中音频轨道，移动白色竖线到结束点，分割后删除多余部分。

![](img/358ecdc1dc165ae7e6237b2515038273_41.png)

![](img/358ecdc1dc165ae7e6237b2515038273_29.png)

你还可以调整音乐音量。选中音频片段，点击 **`音量`**，拖动滑块即可调节。

![](img/358ecdc1dc165ae7e6237b2515038273_47.png)

![](img/358ecdc1dc165ae7e6237b2515038273_33.png)
![](img/358ecdc1dc165ae7e6237b2515038273_35.png)

![](img/358ecdc1dc165ae7e6237b2515038273_49.png)

视频和音频都处理好后，接下来为视频添加字幕。点击底部工具栏的 **`文字`**，再点击 **`识别字幕`**。

![](img/358ecdc1dc165ae7e6237b2515038273_51.png)

为了只识别人声，建议选择 **`仅视频`** 模式，然后点击 **`开始识别`**。系统会自动生成字幕，并匹配时间点。

![](img/358ecdc1dc165ae7e6237b2515038273_53.png)

![](img/358ecdc1dc165ae7e6237b2515038273_37.png)
![](img/358ecdc1dc165ae7e6237b2515038273_39.png)
![](img/358ecdc1dc165ae7e6237b2515038273_41.png)

![](img/358ecdc1dc165ae7e6237b2515038273_55.png)

识别完成后，检查字幕是否有错别字。**双击**字幕条即可进行修改。你还可以在右侧菜单中调整字体、颜色、样式等，让字幕更美观。

![](img/358ecdc1dc165ae7e6237b2515038273_57.png)

至此，视频的**画面比例、内容剪辑、背景音乐、字幕**都已处理完毕，一个完整的视频就制作好了。

![](img/358ecdc1dc165ae7e6237b2515038273_61.png)

![](img/358ecdc1dc165ae7e6237b2515038273_51.png)

![](img/358ecdc1dc165ae7e6237b2515038273_63.png)

在导出视频前，有一个关键设置能影响视频清晰度。点击右上角 **`导出`** 按钮。

在导出设置中，请按以下参数调整：
*   **帧率 (fps)**：设置为 **`30`**。
*   **分辨率**：拉至最高（如 **`1080p`** 或 **`4K`**）。

![](img/358ecdc1dc165ae7e6237b2515038273_65.png)

**`高分辨率 + 30帧率`** 是保证抖音视频清晰且兼容性好的常用设置。设置完成后，点击导出，等待渲染完成即可。导出过程中请勿锁屏或切换应用。

![](img/358ecdc1dc165ae7e6237b2515038273_67.png)

![](img/358ecdc1dc165ae7e6237b2515038273_69.png)

![](img/358ecdc1dc165ae7e6237b2515038273_55.png)
![](img/358ecdc1dc165ae7e6237b2515038273_57.png)

除了剪辑，对于需要出镜口播的创作者，这里推荐一个提词器软件：**爱提词**。它可以帮助你无需背诵稿子，流畅完成录制。

![](img/358ecdc1dc165ae7e6237b2515038273_77.png)

以下是爱提词的基本使用方法：
1.  打开软件，点击右下角 **`+`** 号。
2.  **添加**你的口播文案，可以打字或粘贴。
3.  输入标题后**保存**。
4.  点击 **`开始提词`**，选择 **`录制模式`**，即可边看稿边拍摄。

软件还提供实用功能：
*   **调整字体**：在设置中可放大字体，方便阅读。
*   **内置美颜**：点击左下角 **`美颜`**，可调节瘦脸、大眼、磨皮等参数，效果比原相机更佳，且比一些美颜App更自然。
*   **爆款文案参考**：软件内的“爆款文案”板块提供了多个领域的文案范例，可供学习和灵感参考。
