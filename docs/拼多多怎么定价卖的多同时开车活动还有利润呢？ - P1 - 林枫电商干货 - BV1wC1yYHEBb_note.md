# 拼多多运营教程：P1：定价与推广策略详解

在本节课中，我们将学习拼多多新品如何科学定价，以及如何设置推广投入以实现销量与利润的平衡。核心内容包括定价公式、活动策略及推广投入产出比的计算方法。

## 定价策略与公式

上一节我们介绍了课程目标，本节中我们来看看如何为商品制定一个有竞争力的价格。

拼多多新品定价可直接参考同行价格。若无直接同行，可参考跨领域相似商品。同行售价可作为定价基准，这能节省市场调研成本。

但若想实现销量与利润兼得，需采用更精确的计算方法。请遵循以下定价公式：

**定价公式：拼单价 = (成本 + 目标利润) / 0.7**

假设单件商品成本为20元，希望每单利润为10元，则计算如下：
`拼单价 = (20 + 10) / 0.7 = 30 / 0.7 ≈ 42.86元`

此价格为参与平台活动（通常要求7折）前的基准价。按此定价，在参与7折活动时，实际售价约为30元，正好覆盖成本并达成目标利润。

## 活动与优惠券设置

![](img/8a787cea120ecce7b543c8161405430a_1.png)

理解了基础定价后，我们来看看如何通过活动与优惠券设置来维护利润空间。

![](img/8a787cea120ecce7b543c8161405430a_2.png)

为链接设置价格弹性空间至关重要。以下是具体操作步骤：

*   **活动提报**：以7折价格提报平台活动。
*   **优惠券设置**：若无法上活动，可设置一张12元的店铺券。这样，平台记录的历史最低价会始终高于30元（即42.86元减12元），有效保护了利润底线。

## 推广投入产出比计算

![](img/8a787cea120ecce7b543c8161405430a_4.png)

商品上架并完成基础定价后，推广是获取流量的关键。除了优化广告数据，设置合理的投入产出比目标同样重要。

以下是计算推广投入产出比的步骤与公式：

1.  **计算保本投产比**：这是推广不亏不赚的临界点。
    **公式：保本投产比 = 客单价 / 单件利润**
    沿用前例，客单价30元，单件利润10元，则 `保本投产比 = 30 / 10 = 3`。初期推广可按此投产比起步，运行数日以积累数据。

2.  **计算优化后投产比**：需考虑退货、平台服务费等其他成本。
    *   **计算退货率**：例如，3天销售100单，退货1单，则退货率为1%。
    *   **估算其他成本**：将平台服务费、罚款等杂项估算为总成本的5%。
    *   **优化公式**：**优化后投产比 = 保本投产比 / (1 - 退货率) * (1 + 其他成本率)**
        代入数值：`优化后投产比 = 3 / (1 - 0.01) * (1 + 0.05) ≈ 3 / 0.99 * 1.05 ≈ 3.18`

![](img/8a787cea120ecce7b543c8161405430a_6.png)

3.  **计算最佳投产比**：若要推广产生最高利润，实际投产比应达到优化后值的1.3倍以上。
    **公式：最佳投产比 = 优化后投产比 * 1.3**
    代入数值：`最佳投产比 ≈ 3.18 * 1.3 ≈ 4.13`

![](img/8a787cea120ecce7b543c8161405430a_7.png)

为方便计算，可使用自动计算表格，只需填入成本、利润等基础数值，即可自动生成保本、优化后及最佳投产比。

本节课中我们一起学习了拼多多的核心定价策略与推广投入计算方法。关键点在于：通过公式 `拼单价 = (成本 + 利润) / 0.7` 确定基础价格；利用活动与优惠券维护价格体系；并通过分步计算保本、优化后及最佳投产比来指导推广投放，从而实现销量增长与利润保障的双重目标。
