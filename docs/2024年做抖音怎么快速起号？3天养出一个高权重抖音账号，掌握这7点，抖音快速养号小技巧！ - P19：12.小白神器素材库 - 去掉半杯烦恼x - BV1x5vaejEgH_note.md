# 抖音运营指南：P19：12.小白神器素材库

![](img/cee5ac807caaf9a7181fa3d34fbdd093_1.png)

在本节课中，我们将学习如何使用一个名为“一搜猫”的工具来高效地寻找和制作视频素材。这对于想要制作原创内容，特别是影视解说类视频的初学者来说，是一个非常实用的技巧。

上一节我们介绍了内容创作的基本思路，本节中我们来看看如何利用工具快速获取素材。

今天寻找素材，我们将使用一个被称为“小白神器”的工具。

## 视频资源搜索

第一个功能是视频资源搜索，它可以找到所有平台上的视频。

所有我们想要制作的视频主题，都能通过它找到。例如，如果我们想找关于“沈腾”的视频。

![](img/cee5ac807caaf9a7181fa3d34fbdd093_3.png)

我们直接搜索“沈腾”这个关键词。

![](img/cee5ac807caaf9a7181fa3d34fbdd093_9.png)

![](img/cee5ac807caaf9a7181fa3d34fbdd093_5.png)

所有平台上关于沈腾的视频就都能找到了。这就是视频资源功能。

## 文章素材获取

![](img/cee5ac807caaf9a7181fa3d34fbdd093_11.png)

我们今天找素材，主要会用到“自媒体资源”功能，这里提供的是文章素材。

![](img/cee5ac807caaf9a7181fa3d34fbdd093_7.png)

![](img/cee5ac807caaf9a7181fa3d34fbdd093_13.png)

我们点开“自媒体资源”。顶部的“领域”选项，可以根据自己做的领域随机选择。选择领域后，下方有“来源”选项，列出了所有的自媒体平台，我们可以自行选择。例如，我选择“百家号”来看一下。

这里展示了所有的文章，这就是我们寻找素材的来源。具体该怎么找呢？我来找一个最近大家比较熟悉的例子。

文章非常多，我们都可以查看。比如，我们看这篇关于胡歌主演的电视剧《县委大院》的文章，这部剧最近比较火。

![](img/cee5ac807caaf9a7181fa3d34fbdd093_9.png)

## 视频素材匹配

如果我们要制作解说视频，第一步是寻找视频素材。视频素材可以根据文章主题来寻找。例如，针对热播剧《县委大院》，我们直接复制这个名字。

然后，打开我们手机上的“西瓜视频”应用。

![](img/cee5ac807caaf9a7181fa3d34fbdd093_11.png)

在搜索栏中，直接搜索电视剧的名字“县委大院”。

![](img/cee5ac807caaf9a7181fa3d34fbdd093_13.png)

直接搜索即可。现在，视频素材就能找到了。在找素材的过程中，我再次强调一点：**不要只盯着一个人的素材用，不要直接下载完整视频。每个视频片段最好保持在十秒钟左右**，这才是我们能用的素材。现在，视频素材找到了。

## 文案提取与配音

下一步，文案从哪里来呢？

![](img/cee5cac807caaf9a7181fa3d34fbdd093_15.png)

文案就是从我们刚才找到的文章中提取出来的文字。有了文案，接下来需要配音。

以下是几种配音方案：

**方案一：自己录音**
自己会配音的同学，可以用手机自带的录音机。打开录音机，自己把文字念一遍进行配音。这样制作的内容是纯原创的，别人很难搬运。

**方案二：使用方言**
如果担心普通话不标准，可以使用方言。目前平台上方言内容较少，更有创新性和亲切感。例如，用四川话说“宝贝”、“乖乖”，听起来会更亲切，能增加视频的吸引力。

**方案三：使用智能配音**
如果既不想自己配，也不想用方言，可以直接将复制的文字粘贴回“一搜猫”首页。在首页左下角有一个“智能配音”选项。

点开它，把文字粘贴进去。右边有非常多的声音可以选择，例如“邻家小姐姐”、“可爱宝贝”、“萝莉音”、“御姐音”等。我们还可以调整语速和音量。选择完毕后，点击左下角的“试听配音”。

例如，合成后的配音可能是：“林志为张新城市到光明县参加工作后的第一天，没有人理会他的时候，他就想方设法的与同事进行沟通...” 这个声音效果是可以接受的。

这样，文案不用自己写，视频素材容易找，配音也能一键生成，非常方便。

**方案四：使用其他配音工具**
如果不想用“一搜猫”的配音功能，微信小程序里还有一个叫“配音神器”的工具。它也是专门的配音软件，会员费大约每月29元。记住一句话：**小投资，大回报**。一万播放量的收益通常不止29元。如果不想付费，也可以继续使用“一搜猫”的功能。
