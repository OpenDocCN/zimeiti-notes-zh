## 第一步：产品基础搭建

上一节我们概述了课程目标，本节中我们来看看如何为你的产品打下坚实的基础。产品基础决定了初始的点击率和转化率，是后续所有操作的前提。

### 主图设计
拼多多消费者是视觉动物，他们先看图片，再决定是否点击。产品主图必须突出核心卖点，并使用营销词精准戳中买家痛点，从而吸引点击。

![](img/2a4c1f682dedca39582f5c8e761cd003_5.png)

**示例**：一款美妆蛋的主图使用了“官方正品”、“次达品牌职工”等营销词来提升点击率。

![](img/2a4c1f682dedca39582f5c8e761cd003_7.png)

### 标题优化
标题是获取自然搜索流量的关键。可以参考头部同行的选词逻辑，但切忌直接抄袭。因为搜索排名依赖于关键词的权重，直接抄袭等于为他人做嫁衣。

**操作建议**：
1.  在标题前添加括号，突出2到4个字的**核心卖点**，例如“疯抢中”、“低价清库存”。
2.  根据产品自身卖点进行添加。

![](img/2a4c1f682dedca39582f5c8e761cd003_9.png)

### 定价策略
合理的定价策略能提升权重并促进转化。

**操作步骤**：
1.  先设置较高的**拼单价**。
2.  后续通过**优惠券**或**限时限量购**活动降低实际售价。
3.  这样既能提升产品权重，又能让买家感受到实惠，提高转化率。

### SKU设置
拼多多外漏价显示的是链接内最低价的SKU。利用这一点可以设置引流SKU。

**操作步骤**：
1.  设置一个**低价SKU**，价格最好低于同行均价。
2.  为避免低价SKU被大量购买造成亏损，可采用以下方法：
    *   用**形容词**描述，降低被购买的可能性。
    *   用**试用装**或**小样**代替正装。
    *   **示例**：销售蓝牙耳机，低价SKU可以设置为“一根数据线”，通常消费者不会只买数据线，从而实现了截流和防亏损。

### 防比价
做好防比价能避免平台系统强制降价。具体方法可参考相关专题视频。

## 第二步：拉升产品权重

完成了产品的基础搭建后，我们需要通过各种手段来提升产品在平台内的权重，以获得更多曝光机会。

以下是拉升权重的几个关键操作：

*   **基础销量与评价**：销量可通过“改销量”工具设置（例如改为1万）。评价则需要精心维护，为主推款准备100-200条带图评价，可以从优质同行的买家秀中挑选参考。90%的消费者会参考评价再决定是否下单。
*   **DSR评分**：30天内需积累至少50条有效好评，将DSR评分维持在行业平均水平的60%以上，以稳定店铺流量。若低于50%，流量有被同行抢占的风险。
*   **店铺层级**：新店前期应力争达到第3或第4层级。每个层级的流量上限不同。可以通过“拍大单”的方式，快速突破达到下一层级所需的营业额门槛。

## 第三步：付费推广引流

当产品具备一定基础后，我们可以通过付费推广来精准引流，加速爆款的成长。

拼多多直通车主要有两种出价方式：**成交出价**和**投产比（ROI）出价**。

### 成交出价设置
*   不要完全按照系统推荐出价。
*   若产品客单价在50元以内，可按**热卖SKU的价格**设置成交出价。
*   若客单价在50元以上，则根据**利润空间**设置。例如客单价137元，利润34元，成交出价最高可设为34元。

### 投产比（ROI）出价设置
理解ROI的计算公式是设置出价的基础。

**核心公式**：
1.  **保本ROI** = 售价 / 利润
    *   **示例**：产品成本20元，售价40元，利润20元，则保本ROI = 40 / 20 = 2。
2.  前期先按**保本ROI出价**，启动直通车。
3.  积累一定数据后，根据实际**退货率**计算**净ROI**。
    *   **公式**：净ROI = 保本ROI / (1 - 退货率) * 1.1
4.  最终，需要将ROI优化到**最佳区间**（通常高于净ROI），例如4.35到5.58之间，推广才算是有效数据。

### 竞品对标法
本节的核心玩法是：对标同行优秀数据，设定明确的追赶目标。

**操作步骤**：
1.  打开商家后台，查看“店铺访客数据”和“成交UV价值”。
2.  找到三条参考线：自己店铺数据、同行同层级平均数据、**同行同层级优秀数据**。
3.  运用公式计算目标：
    *   **销售额** = 访客数 × 成交UV价值
    *   **目标订单量** = (同行优秀访客数 × 同行优秀UV价值) / 自己产品的客单价
4.  通过优化关键词，促使进店访客成交，从而提升店铺自然流量。

## 第四步：活动推广与报名

掌握了付费推广后，我们可以借助平台活动进一步放大流量，为冲刺大促做好准备。

新手商家前期适合参加一些门槛较低的活动，例如“21365”或“21856”。活动期间需特别注意维护店铺的整体转化率，这既能提升当前活动的权重，也能为未来报名“双十一”等大促活动积累良好的数据基础。

![](img/2a4c1f682dedca39582f5c8e761cd003_11.png)

在报名大促活动前，必须维持好店铺的日销售额和转化率。稳定的数据表现会让平台在审核时减少“压价”的幅度，避免虽然报上了活动却无利可图甚至亏损的情况。
