# 手机剪辑教程：P29：电影感镂空字体片头制作 🎬

在本节课中，我们将学习如何使用蒙版功能，制作一个具有电影感的镂空文字片头开场效果。整个过程分为准备文字素材和组合动画两个主要部分，步骤清晰，适合初学者跟随操作。

## 概述
我们将通过两个剪辑项目来完成这个效果。首先，我们需要创建一段纯文字视频作为素材。然后，在第二个项目中，通过画中画、混合模式以及关键帧动画，让文字以镂空扫光的形式显现出来。

## 文字素材准备
首先，我们需要制作一段作为蒙版素材的文字视频。

1.  新建一个剪辑项目。
2.  导入一段纯黑色背景。
3.  在黑色背景上添加文字。文字内容自定，建议将文字位置调整得稍微靠下一些。
4.  将这段带有文字的黑色背景视频导出备用。

**核心操作**：`新建项目 -> 添加黑色背景 -> 添加文字 -> 导出视频`

## 组合动画制作
上一节我们准备好了文字素材，本节中我们来看看如何利用它制作最终的片头动画。以下是具体的操作步骤。

1.  再次新建一个剪辑项目。
2.  通过“画中画”功能，导入上一步导出的文字视频。
3.  将画中画图层调整到合适的位置。
4.  选中画中画图层，点击“混合模式”，选择“变暗”模式。这样，黑色部分将变为透明，只留下文字部分。
5.  将画中画图层中多余的部分分割并删除，使其长度与主视频匹配。

**核心概念**：混合模式中的 **`变暗`** 模式，可以使画中画图层的深色区域（黑色）变透明。

## 创建文字显现动画
接下来，我们为文字图层添加一个从上向下逐渐显现的动画效果。

1.  选中画中画图层，点击“蒙版”，选择“线性蒙版”。
2.  将画面中出现的黄色蒙版线拖动到画面上方，与顶部对齐。
3.  在画中画图层时间线的起始位置，添加一个关键帧。
4.  将时间线移动到画中画图层的末尾。
5.  再次点击“蒙版”，将黄色蒙版线从画面顶部拖动到画面底部。
6.  系统会自动在末尾生成一个关键帧，从而形成一个蒙版从上向下移动的动画，文字也随之逐渐显现。

**核心概念**：**关键帧**用于记录某个时间点图层属性的状态（如蒙版位置），两个关键帧之间会自动生成平滑的动画。

## 添加黑色背景扫光
为了使效果更有层次感，我们将在文字下方添加一个同步运动的黑色背景。

1.  再次通过“画中画”，导入一段纯黑色背景素材。
2.  将黑色背景放大至全屏，并将其轨道拖动到文字图层的下方。
3.  在黑色背景图层的开头添加一个关键帧。
4.  将时间线移动到该图层的末尾。
5.  为黑色背景同样添加一个“线性蒙版”。
6.  将蒙版线从画面底部向上拖动，直到其位置与上方文字图层的蒙版线大致重合。
7.  这样，黑色背景会形成一个从下向上的扫光动画，与文字的显现动画相呼应。

## 总结
本节课中我们一起学习了制作电影感镂空字体片头的完整流程。我们首先创建了文字素材，然后利用**画中画**、**混合模式（变暗）**、**线性蒙版**和**关键帧动画**这些核心功能，组合出了文字逐渐扫光显现的视觉效果。通过为底层黑色背景添加反向动画，增强了整体的电影感。你可以尝试更换不同的背景视频，应用此片头效果。
