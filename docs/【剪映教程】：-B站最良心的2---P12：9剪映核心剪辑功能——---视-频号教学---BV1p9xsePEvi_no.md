# 剪映教程：P12：9 剪映核心剪辑功能——蒙版功能 🎬

在本节课中，我们将要学习剪映的核心剪辑功能之一：蒙版。蒙版是一种遮罩工具，用于隐藏或显示视频画面的特定部分，从而创造出各种转场特效和视觉效果，让视频内容更加精致。

## 什么是蒙版？

蒙版是视频剪辑中常用的遮罩功能。它的核心作用是**隐藏**视频画面中的某些元素，不让观众看到。在剪映中，我们通过蒙版功能来实现这一效果。

![](img/e35b4e44f9c1ad8806c09041000959c1_1.png)

许多我们看到的转场特效视频，都大量应用了蒙版功能。

## 蒙版效果案例展示

以下是两个运用了蒙版功能的视频案例。

![](img/e35b4e44f9c1ad8806c09041000959c1_3.png)

*   **案例一：双重人影转场**
    在某个镜头中，画面里出现了两个人影重叠结合的效果。这种效果就是通过运用蒙版功能，并将边缘进行羽化处理来实现的。

*   **案例二：图形拼图转场**
    在另一个视频中，可以看到五角星等图形的拼图效果。这是使用了图形蒙版，让视频画面以圆形、方形、五角星等特定形状显示，从而创造出绚丽的转场。

上一节我们了解了蒙版的概念和效果，本节中我们来看看如何在剪映中实际操作蒙版功能。

![](img/e35b4e44f9c1ad8806c09041000959c1_5.png)

## 蒙版功能实战操作

![](img/e35b4e44f9c1ad8806c09041000959c1_7.png)

以下是使用蒙版功能的具体步骤。

1.  **导入素材并选中**
    点击“开始创作”，导入几张图片或视频素材。导入后，视频会按顺序平铺播放，缺乏特效。首先选中第一段素材。

![](img/e35b4e44f9c1ad8806c09041000959c1_9.png)

2.  **找到并添加蒙版**
    在底部二级菜单中找到并点击 **`蒙版`** 按钮。系统会默认添加一个线性蒙版。

3.  **线性蒙版**
    添加线性蒙版后，视频画面会呈现渐变隐藏的效果。拖动蒙版线可以调整显示与隐藏的区域。**公式：`显示区域 = 原始画面 - 蒙版遮挡区域`**。之前流行的“遇见当年”视频就使用了线性蒙版来擦除中间多余部分。

![](img/e35b4e44f9c1ad8806c09041000959c1_11.png)

4.  **镜像蒙版**
    选择镜像蒙版，画面会向两侧对称地隐藏。拖动箭头可以调整羽化值和范围，让边缘过渡更自然。

![](img/e35b4e44f9c1ad8806c09041000959c1_13.png)

5.  **图形蒙版（圆形、星形等）**
    选择圆形、星形等图形蒙版，画面会被约束在相应的图形内显示。通过调整图形的位置、大小和羽化值，可以制作出圆形、五角星等形状的转场效果。

![](img/e35b4e44f9c1ad8806c09041000959c1_15.png)

了解了静态蒙版的添加方法后，我们来看看如何让蒙版效果动起来，制作更炫酷的转场。

![](img/e35b4e44f9c1ad8806c09041000959c1_17.png)

## 制作动态蒙版转场

![](img/e35b4e44f9c1ad8806c09041000959c1_18.png)

![](img/e35b4e44f9c1ad8806c09041000959c1_20.png)

要制作动态的蒙版转场（如案例中的双重人影效果），需要结合画中画和动画功能。

![](img/e35b4e44f9c1ad8806c09041000959c1_21.png)

以下是制作动态蒙版转场的步骤。

1.  **添加画中画**
    复制主视频轨道上的素材，并将其粘贴到画中画轨道。

2.  **为画中画添加蒙版**
    选中画中画素材，点击底部 **`蒙版`** 按钮。例如，选择一个星形蒙版，调整其位置和羽化值。

![](img/e35b4e44f9c1ad8806c09041000959c1_23.png)

3.  **添加动画效果**
    为添加了蒙版的画中画素材设置入场动画，例如“向右甩入”。可以为多个画中画图层设置相同的动画，以增强效果。

4.  **预览最终效果**
    播放时间轴，即可看到带有动态蒙版转场（如双重叠影）的炫丽效果。

![](img/e35b4e44f9c1ad8806c09041000959c1_25.png)

![](img/e35b4e44f9c1ad8806c09041000959c1_26.png)
