# 视频号运营：P28：直播时如何发布草稿箱视频

![](img/df90dfcfc3ba33e3def492348663247f_0.png)

在本节课中，我们将学习一个实用的直播技巧：如何在直播过程中，将提前准备好的草稿箱视频发布出去，以吸引更多观众进入直播间。

## 概述

![](img/df90dfcfc3ba33e3def492348663247f_2.png)

直播开始时，如果发现直播间观众稀少，我们可以通过发布预先准备好的视频来吸引流量。本节将详细介绍具体的操作步骤和注意事项。

![](img/df90dfcfc3ba33e3def492348663247f_4.png)

## 操作步骤

以下是直播时发布草稿箱视频的具体流程。

![](img/df90dfcfc3ba33e3def492348663247f_6.png)

1.  点击直播间右上角的三个点图标。
2.  选择“最小化”直播间，将其暂时隐藏。
3.  返回视频号主页，找到“草稿箱”。
4.  选择提前准备好的视频，点击“发表”。

![](img/df90dfcfc3ba33e3def492348663247f_8.png)

> **注意**：发布时系统可能会提示“正在直播中，请稍后”，但视频依然可以成功发布。请勿直接在直播界面点击“发表视频”按钮，该路径在直播中无法使用。

![](img/df90dfcfc3ba33e3def492348663247f_10.png)

## 提升直播人气的其他方法

![](img/df90dfcfc3ba33e3def492348663247f_12.png)

上一节我们介绍了通过发布视频吸引观众，本节中我们来看看其他辅助提升直播人气的方法。

除了发布视频，主动分享直播间是另一个有效途径。

*   **分享至私域**：将直播间链接分享到微信朋友圈或直接转发给好友。
*   **利用社群**：提前准备一些微信群（如业主群、兴趣群、品牌用户群等），在直播时将链接分享到这些群中。
*   **使用小号**：如果不便使用主号分享，可以注册小号加入相关社群，通过小号进行分享引流。

![](img/df90dfcfc3ba33e3def492348663247f_14.png)

> **平台规则提示**：视频号官方有相应的流量扶持机制。通常，从私域（如微信群、朋友圈）引导进入直播间的观众数量，会影响平台向公域推荐流量的数量。例如，私域引导10人，公域可能再推荐10人。

## 自查与优化

如果尝试了上述方法后，直播间依然留不住人，观众进入后很快离开，那么可能需要从自身寻找原因。

直播是一项需要练习的技能，观众留存率低可能与以下因素有关：

*   **直播话术**：话术是否吸引人、有互动性。
*   **表现状态**：是否精神饱满、富有感染力。死气沉沉的状态容易让观众流失。
*   **内容与产品**：在排除外部因素后，需反思直播内容或所推荐产品本身是否具有吸引力。

## 总结

本节课中我们一起学习了直播时提升人气的核心方法。

1.  **核心操作**：通过**最小化直播间 -> 进入主页草稿箱 -> 发布视频**的路径，在直播中发布预存视频。
2.  **辅助引流**：积极将直播间分享到私域朋友圈和各种微信群，可利用小号进行操作以保护主号隐私。
3.  **内在优化**：关注直播话术、个人表现状态等内在因素，持续练习和优化。

掌握这些方法，并结合反复实践，能有效帮助你在直播初期吸引并留住更多观众。
