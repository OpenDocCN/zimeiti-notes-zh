# 视频号运营教程：P25：4. 如何扒别人直播间话术 🎤

在本节课中，我们将学习如何高效地借鉴和复制其他成功直播间的话术。通过使用特定工具，你可以快速获取并分析他人的直播脚本，从而优化自己的直播内容，无需额外付费学习。

---

上一节我们介绍了如何复制他人的视频和选品策略，本节中我们来看看如何获取并学习他人直播间的话术。话术是直播带货的核心，直接影响转化率。掌握优秀的话术结构，能让你快速上手开播。

你需要准备的工具是“飞书”应用。具体操作步骤如下：

以下是使用飞书妙计录制直播间话术的步骤：

![](img/14a02353568f70610fff0f4d9bd0abf4_1.png)

1.  **下载并打开飞书应用**。
2.  在应用首页，点击底部导航栏的 **“工作台”**。
3.  在工作台页面，找到并点击功能 **“飞书妙计”**。
4.  进入飞书妙计后，点击右下角的 **语音录制按钮**，即可开始录音。



![](img/14a02353568f70610fff0f4d9bd0abf4_3.png)

![](img/14a02353568f70610fff0f4d9bd0abf4_1.png)

此时，你可以打开任意一个你想学习的直播间。飞书妙计会实时录制直播间内的所有声音，包括主播的介绍、互动和逼单话术。



![](img/14a02353568f70610fff0f4d9bd0abf4_3.png)

通常，录制**15分钟**左右的直播内容，就足以分析出该主播介绍产品、营造氛围和促进下单的核心话术逻辑。

录制完成后，点击完成按钮。在生成的录音文件中，进行以下操作：

以下是导出和整理话术的步骤：

1.  点击录制好的文件。
2.  选择 **“分享”** 按钮。
3.  在分享选项中，选择生成 **“可编辑”** 或 **“可阅读”** 的链接。
4.  **复制** 该链接。
5.  将链接发送到微信，并在微信中打开。

在微信中打开链接后，页面会显示录制内容转换成的**文字稿**。你可以直接复制这些文本。

接下来，你需要对复制下来的原始话术进行提炼和编辑，去除无关的口水话，保留核心的产品介绍、卖点阐述和逼单逻辑，最终形成适合你自己风格和产品的直播脚本。

你可以将最终版话术写在纸上，或存入手机备忘录。开播时如果紧张或遗忘，可以随时查看，确保直播流程顺畅。



![](img/14a02353568f70610fff0f4d9bd0abf4_5.png)

本节课中我们一起学习了如何利用“飞书妙计”工具来获取和分析他人直播间的话术。关键操作是：**打开飞书 -> 进入工作台 -> 使用飞书妙计录制 -> 导出文字稿 -> 提炼编辑成自己的脚本**。通过这个方法，你可以快速学习优秀同行的经验，构建自己的直播话术体系，从而更自信地开启直播。
