### 核心概念：核心关键词与长尾关键词

![](img/0c6b9d4ce88da2408cfd4ebd1f5afc76_13.png)

首先，我们需要理解两个基础概念：**核心关键词**与**长尾关键词**。

*   **核心关键词**：指代一个行业或产品的通用、概括性词汇。例如：`女装`、`奶粉`、`压缩机`。
*   **长尾关键词**：由核心关键词拓展而来的、更具体、更长的短语。它通常反映了用户更细分、更具体的问题或需求。例如，由核心词“女装”拓展出的`女装图片`、`女装店`、`女装穿搭`等。

这两个概念是平台**SEO（搜索引擎优化）** 的基础，对于依赖搜索流量的小红书平台尤为重要。

#### 1. 需求图谱分析

![](img/0c6b9d4ce88da2408cfd4ebd1f5afc76_28.png)

![](img/0c6b9d4ce88da2408cfd4ebd1f5afc76_30.png)

通过工具生成的需求图谱（词云），可以直观看到与核心词相关的长尾词及其热度。

![](img/0c6b9d4ce88da2408cfd4ebd1f5afc76_34.png)

![](img/0c6b9d4ce88da2408cfd4ebd1f5afc76_35.png)

*   **操作**：在工具中输入你的核心关键词（如“眉笔”）。
*   **解读**：图中字体越大，代表该长尾词的搜索热度越高。例如，“眉笔”的需求图谱中，“颜色”、“图片”、“品牌”等词可能非常突出。
*   **应用**：这揭示了用户当前最关心的价值点。你可以据此规划内容方向，优先覆盖这些高需求点。

#### 2. 关键词拓词逻辑

![](img/0c6b9d4ce88da2408cfd4ebd1f5afc76_37.png)

![](img/0c6b9d4ce88da2408cfd4ebd1f5afc76_39.png)

拓词的完整逻辑是一个从核心词到具体问题的层层深入过程。

![](img/0c6b9d4ce88da2408cfd4ebd1f5afc76_41.png)

![](img/0c6b9d4ce88da2408cfd4ebd1f5afc76_43.png)

![](img/0c6b9d4ce88da2408cfd4ebd1f5afc76_45.png)

以下是关键词拓展的路径示例：
```
核心词：眉笔
  ├─ 长尾方向1：颜色
  │     ├─ 拓展词1：眉笔颜色怎么选
  │     └─ 拓展词2：眉笔什么颜色好看
  │
  └─ 长尾方向2：色号
        ├─ 拓展词1：眉笔色号推荐
        └─ 拓展词2：植村秀眉笔色号
```
通过这种方式，你可以挖掘出大量用户可能搜索的具体问题，从而生成丰富的内容选题库，覆盖更广泛的潜在用户需求。

### 小红书平台的关键词“黑科技”玩法

![](img/0c6b9d4ce88da2408cfd4ebd1f5afc76_53.png)

在小红书，有一种利用平台机制提升关键词权重的玩法，即**关键词堆砌**。

![](img/0c6b9d4ce88da2408cfd4ebd1f5afc76_55.png)

*   **现象**：有些笔记会在正文结束后，额外添加一段看似与标签相关、实则重复堆砌核心关键词的文案（例如，重复“瑞幸咖啡”等词）。
*   **原理**：平台通过名为 **`Spider`（蜘蛛）** 的程序爬取和识别笔记内容。堆砌关键词可以提高该词在内容中的出现频率，从而“暗示”系统这篇笔记与该关键词高度相关，可能提升其在搜索结果中的排名。
*   **注意**：这是一种取巧方法，对用户体验不友好。其效果因产品类目和平台算法调整而异，需谨慎使用。

更极端的传统SEO手法甚至通过改变文字颜色（如设为白色），让关键词对用户不可见但能被`Spider`抓取。不过，小红书目前不支持用户自定义字体颜色。
