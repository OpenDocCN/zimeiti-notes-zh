# 小红书运营：P11：小红书的3种热门笔记类型

在本节课中，我们将学习三种能有效提升转化率的小红书热门笔记类型。掌握这些笔记的写作方法，可以帮助你更高效地创作内容，吸引目标用户。

上一节我们介绍了账号定位与基础设置，本节中我们来看看如何通过具体的笔记形式来提升内容效果。

## 教育类笔记 📚

教育类笔记的核心是传授知识或技能，在帮助用户的过程中自然植入产品信息。这种笔记适合美妆、瘦身、美食、P图软件、美容、摄影等行业。

以下是教育类笔记首图的三种常见构成形式：

*   **对比图**：展示使用产品前后的差异。例如，口红涂在嘴上与拿在手上的颜色对比，或使用某产品前后的皮肤状态对比。
*   **效果图**：直接展示使用后的最终效果。例如，眼影上妆后的整体妆容效果。
*   **步骤图**：分解展示操作过程。例如，“五步告别圆肩驼背”的教程，过程中可能会展示辅助产品（如矫正带）。

接下来，我们看看教育类笔记的正文应如何撰写。正文的目的是系统性地指导用户，通常可遵循以下五步结构：

1.  **使用工具或材料**：列出所需的产品及辅助工具。
2.  **操作步骤**：详细说明每一步的具体做法。
3.  **注意事项**：提示操作中需要避免的误区或关键点。
4.  **产品安利**：自然地推荐你觉得好用的产品。**注意**：介绍产品的篇幅建议不超过全文的30%，以免被判定为硬广。
5.  **常见问题解答**：针对用户操作中可能遇到的“失败”问题（如“为什么贴不好双眼皮贴？”）进行解答，这比展示成功案例更能吸引用户阅读。

掌握了教育类笔记的写法后，我们来看另一种适用性更广的笔记类型。

## 合集类笔记 📦

合集类笔记通过集合多个相关产品，为用户提供一站式解决方案或参考。它几乎适用于所有行业，例如针对特定场景（旅行、健身）、人群（学生党、上班族）或需求进行推荐。

写作合集类笔记时，常见问题是角度单一。为了避免这个问题，你可以从以下七种不同角度切入创作：

*   **针对人群**：例如，《学生党开学彩妆攻略》、《上班族代步小电动车推荐》。
*   **针对成本**：例如，《100元内平价护肤品合集》、《五块钱一个的网红早餐包》。
*   **针对效果**：例如，《让男朋友合不拢嘴的夏季穿搭》、《瘦身必备零食清单》。
*   **针对地区**：例如，《香港平价美食推荐》、《南京本地网红店打卡》。
*   **针对场景**：例如，《旅游必备清单》、《居家清洁好物指南》。
*   **针对产品属性**：例如，《微电流美容仪测评》、《某某色系口红合集》。
*   **针对品牌**：例如，《体察内部员工隐藏菜单》、《星巴克夏季新品测评》。

合集类笔记的另一个挑战是容易显得广告意图明显。解决方法是在介绍多个产品时，**均匀分布篇幅**，并客观提及产品的**优点和缺点**，这样能有效降低推广嫌疑，增加可信度。

以下是合集类笔记的两种常见首图形式：

*   **产品拼图**：使用美图秀秀、黄油相机等软件将多个产品图片拼接在一起。
*   **产品摆拍**：将相关产品（如旅行所需物品）整齐地摆放在一个平面（如地板、桌面）上进行拍摄。

总结一下，合集类笔记的标题应从**展现的角度**切入，正文则应包含**推荐理由**和**产品特点说明**。如果配图已有详细文字说明，正文可简单提及产品名称，这有助于小红书系统抓取关键词标签。

了解了图文笔记的两种主要形式后，我们来看看当前平台更推崇的内容形式。

## 视频类笔记 🎥

视频类笔记是目前各平台流量扶持较大的内容形式，如果你有条件，非常推荐尝试。视频内容更生动，信息承载量也更大。

制作视频笔记需要准备以下工具：

*   **拍摄工具**：手机、专业相机、三脚架、补光灯。预算允许可购置简易摄影棚（约两三百元）。
*   **剪辑工具**：
    *   **手机APP**：剪映（抖音旗下）、必剪等。
    *   **电脑软件**：Adobe Premiere (PR) 等专业软件。

关于视频笔记，有几个关键参数需要注意：

*   **视频时长**：小红书默认视频时长为**1分钟**。持续发布优质视频，达到一定标准后，平台会自动将时长权限提升至5分钟。若想发布超过10分钟的视频，需在获得5分钟权限后向官方邮件申请。
*   **视频尺寸**：推荐两种比例：
    *   **3:4（竖屏）或 4:3（横屏）**：即9:16或16:9。这是手机屏幕的主流比例，**3:4的竖屏大尺寸视觉上更震撼**。
    *   **1:1（正方形）**：适合需要展示产品细节的账号，如美妆、护肤品类，画面显得更精致。

在内容策划上，建议你多研究所在行业内成功的视频博主。分析他们的**视频时长、构图、选品、脚本结构**（如开头多少秒吸引注意力，中间如何展开，结尾如何收尾），总结并模仿其成功经验。视频内容可以围绕**教程、测评、私人分享、生活VLOG**等方向展开。

本节课中我们一起学习了提升小红书笔记转化率的三种核心类型：**教育类笔记**、**合集类笔记**和**视频类笔记**。每种类型都有其适用的场景和创作要点，掌握它们并能灵活运用，将帮助你创作出更受用户欢迎、转化效果更佳的内容。
