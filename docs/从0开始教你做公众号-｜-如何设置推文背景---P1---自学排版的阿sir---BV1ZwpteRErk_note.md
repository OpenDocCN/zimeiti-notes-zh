# 公众号排版教程：P1：如何设置推文背景 🎨

在本节课中，我们将要学习如何为公众号推文设置背景颜色。背景颜色是排版的基础元素之一，合适的背景色能让你的推文更具美感和阅读舒适度。

推文背景颜色指的是文字内容下方的底色。通过观察不同风格的推文，我们可以发现背景色的运用非常灵活。例如，活泼的内容可能搭配黄色或蓝色背景，正式的内容则常用白色，而护眼的豆绿色也颇受欢迎。选择合适的背景颜色，可以让推文风格更统一，视觉效果更佳。

那么，具体应该如何操作呢？接下来，我们将以135编辑器为例，一步步进行讲解。



## 设置全文背景颜色

![](img/41e1b0d978b4ac60885d41733ba82145_1.png)

上一节我们了解了背景颜色的概念，本节中我们来看看如何在编辑器中设置全文背景颜色。

首先，打开135编辑器。如果你对这款编辑器还不熟悉，可以查阅相关的入门教程。

在编辑器的上方工具栏中，找到用于设置背景颜色的图标。点击后，会弹出一个颜色选择面板。

以下是设置全文背景的步骤：
1.  在颜色选择面板中，你可以看到大量预设的颜色。
2.  点击任意颜色，整个编辑区域的背景就会立即更换为该颜色。
3.  此后，在任意位置添加的文字，其底色都将是你所选择的颜色。

即使没有开通会员，也可以使用丰富的免费颜色，完全不必担心选项单调。

除了使用预设颜色，你还可以进行自定义。点击“我的上传”选项，可以将本地图片上传并设置为推文背景，这提供了更大的创作自由度。

我们刚才操作的区域是135编辑器的工具栏，大部分排版设置功能都集中在这里。因此，当你在排版过程中遇到问题时，可以多尝试点击工具栏中的各个选项，自行探索解决方法。当然，也欢迎你在教程下方留言提问。



## 设置局部背景颜色

我们已经学会了如何设置全文背景，现在来看看如何为推文的特定部分设置背景颜色。

首先，在编辑器中输入一些文字，模拟正常的排版过程。

如果你只想为某一段文字或某一个模块添加背景色，操作也非常简单。只需点击该区域的空白处，当该区域被红色虚线框选中后，在编辑器右侧的工具栏中找到“背景”选项并点击。

接下来的步骤与设置全文背景完全相同。以下是操作流程：
1.  点击目标区域，使其被红色虚线框选中。
2.  点击右侧工具栏中的“背景”按钮。
3.  在弹出的颜色面板中选择你想要的颜色，点击确定即可。

这个过程非常便捷，可以让你灵活地控制不同区域的视觉效果。
