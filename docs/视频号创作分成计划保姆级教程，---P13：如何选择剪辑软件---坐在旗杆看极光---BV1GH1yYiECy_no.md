# 视频号创作分成计划保姆级教程：P13：如何选择剪辑软件 🎬

在本节课中，我们将学习如何根据自身需求，从三款主流剪辑软件中做出选择。我们将逐一分析每款软件的特点、适用场景及优缺点，帮助你找到最适合自己的工具。

上一节我们介绍了视频创作的整体流程，本节中我们来看看具体执行剪辑任务时，软件该如何选择。

## 软件概览

以下是三款市面上常用且作者推荐的视频剪辑软件：

1.  **Adobe Premiere Pro (PR)**：专业且应用广泛的视频剪辑软件。
2.  **Edius (ED)**：稳定高效，适合批量处理的视频剪辑软件。
3.  **剪映**：操作友好、功能人性化的国产视频剪辑软件。

## 软件详细分析

### Adobe Premiere Pro (PR)

PR是目前市面上应用最广的专业视频剪辑软件。无论是为了求职，还是制作专业水准的视频，选择PR都是正确的决定。但其缺点也很明显。

![](img/c95a6c96ac33a0423729622838d0ac10_1.png)

PR对电脑配置要求高，运行不稳定。剪辑少量视频（如三五个）没有问题，但如果需要同时剪辑上百个视频，则不建议使用PR。PR的优势在于效果出色、功能全面且细致。

*   **调色与抠像**：表现非常优秀。
*   **音频处理**：功能强大。

![](img/c95a6c96ac33a0423729622838d0ac10_1.png)

然而，它对新手并不友好。新手若想学习PR，需要投入相当的时间和精力。

### Edius (ED)

Edius是一款对电脑配置要求低、运行非常稳定的视频剪辑软件，也是作者此前在电视台工作时专用的软件。

它在调色和抠像方面的表现并非顶尖。如果仅需进行最基本的调色，它可以满足需求。作者目前主要使用它来制作微课、精品课以及进行批量剪辑。

当需要同时处理上百个视频，甚至中间涉及素材互相调用时，Edius也能稳定运行，不易出错。如果需要制作微课、婚礼摄影或个人工作视频，可以选择Edius。若为求职或制作专业视频，则建议选择PR。

### 剪映

剪映作为国产软件，其很多功能设计非常人性化，极大地降低了视频剪辑的门槛。

许多新手通过简单学习（例如学习本套课程），也能利用剪映制作出相当专业的视频。如果你只是个人娱乐，并非追求极致的专业效果，那么直接选择剪映即可。
