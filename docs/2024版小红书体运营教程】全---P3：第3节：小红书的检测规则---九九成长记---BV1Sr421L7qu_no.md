# 小红书运营教程：P3：小红书平台检测规则详解 🕵️

在本节课中，我们将要学习小红书平台的核心检测规则。了解这些规则是避免账号违规、确保内容安全发布的基础。我们将从多个维度分析小红书的审查机制，并提供实用的规避方法。

## 概述

小红书平台会通过系统自动检测与人工审核相结合的方式，对用户发布的内容进行管理。主要检测范围包括个人资料、笔记内容、互动信息以及发布设备。遵守这些规则是账号健康运营的第一步。

## 平台主要检测内容

上一节我们介绍了账号搭建的基础，本节中我们来看看平台具体会检测哪些内容。以下是小红书重点审查的四个方面：

1.  **资料与内容审查**：小红书会检测用户的个人资料、笔记正文、私信以及评论。这些地方不得包含以下营销信息：
    *   直接联系方式，如微信号、微信二维码。
    *   营销词汇，如“代购”、“淘宝店铺推广”。
    *   第三方平台水印。

2.  **设备关联检测**：平台会检测账号登录的设备。为了降低风险，建议同一台设备最多只登录三个小红书账号。

3.  **禁止诱导互动**：笔记内容不得诱导用户进行特定操作。以下行为是被明令禁止的：
    *   引导关注后领奖。
    *   要求点赞后抽奖。

4.  **原创度保护**：小红书对原创内容的保护程度较高。因此，在推广时，切勿将同一篇文章通过多个账号在平台内重复发布。原则是：**一篇文章，一个账号**。

## 敏感词检测与处理

除了上述规则，内容中的敏感词是触发平台审核的常见原因。接下来，我们重点学习如何应对敏感词问题。

## 总结

本节课中我们一起学习了小红书平台的检测规则。核心要点包括：避免在资料和内容中留下直接营销信息；注意设备登录数量；禁止在内容中诱导用户互动；坚持原创，不一稿多投；并学会使用工具检测和规避敏感词。掌握这些规则，能为你的账号运营打下坚实的安全基础。
