# 小红书运营教程：P19：17.新工具介绍：向左向右全选和裁剪 ✂️

![](img/a87e0b38c3be698a5ea58d7f5cd3862c_0.png)

![](img/a87e0b38c3be698a5ea58d7f5cd3862c_2.png)

在本节课中，我们将要学习剪映软件的两个新功能：“向左/向右全选”和“向左/向右裁剪”。这些功能能极大提升多轨道视频剪辑的效率，帮助你更快速地处理素材。

![](img/a87e0b38c3be698a5ea58d7f5cd3862c_4.png)

## 向左全选与向右全选

![](img/a87e0b38c3be698a5ea58d7f5cd3862c_6.png)

![](img/a87e0b38c3be698a5ea58d7f5cd3862c_8.png)

上一节我们介绍了基础剪辑操作，本节中我们来看看如何高效地选择多个轨道上的素材。当你的视频工程包含多个音频、视频或文字轨道时，逐一手动选择会非常繁琐。

![](img/a87e0b38c3be698a5ea58d7f5cd3862c_10.png)

![](img/a87e0b38c3be698a5ea58d7f5cd3862c_12.png)

“向左全选”和“向右全选”功能正是为了解决这个问题而设计的。它们允许你以播放头（时间轴上的竖线光标）为基准，快速选中其左侧或右侧所有轨道上的素材。

![](img/a87e0b38c3be698a5ea58d7f5cd3862c_14.png)

以下是这两个功能的具体说明：

![](img/a87e0b38c3be698a5ea58d7f5cd3862c_16.png)

![](img/a87e0b38c3be698a5ea58d7f5cd3862c_18.png)

*   **向左全选**：点击此功能，会选中**播放头左侧**所有轨道上的素材片段。
*   **向右全选**：点击此功能，会选中**播放头右侧**所有轨道上的素材片段。

![](img/a87e0b38c3be698a5ea58d7f5cd3862c_20.png)

选中后，你可以使用选择工具（快捷键 `V`）整体拖拽这些被选中的素材，进行移动或删除操作。

![](img/a87e0b38c3be698a5ea58d7f5cd3862c_22.png)

![](img/a87e0b38c3be698a5ea58d7f5cd3862c_24.png)

## 向左裁剪与向右裁剪

![](img/a87e0b38c3be698a5ea58d7f5cd3862c_26.png)

了解了高效选择素材的方法后，我们再来看看如何高效地裁剪和删除素材。在传统剪辑中，我们通常先用分割工具（快捷键 `Ctrl + B`）切断视频，再切换回选择工具删除不需要的部分。这个过程需要多次点击，略显繁琐。

![](img/a87e0b38c3be698a5ea58d7f5cd3862c_28.png)

![](img/a87e0b38c3be698a5ea58d7f5cd3862c_30.png)

新推出的“向左裁剪”和“向右裁剪”功能将**分割**和**删除**两步合并为一步，让剪辑更加流畅。

![](img/a87e0b38c3be698a5ea58d7f5cd3862c_32.png)

以下是这两个功能的核心作用：

![](img/a87e0b38c3be698a5ea58d7f5cd3862c_34.png)

![](img/a87e0b38c3be698a5ea58d7f5cd3862c_36.png)

*   **向左裁剪**：此功能会以播放头为界，**删除播放头左侧**的所有内容，并保留右侧部分。效果等同于“分割+删除左侧片段”。
*   **向右裁剪**：此功能会以播放头为界，**删除播放头右侧**的所有内容，并保留左侧部分。效果等同于“分割+删除右侧片段”。

![](img/a87e0b38c3be698a5ea58d7f5cd3862c_38.png)

![](img/a87e0b38c3be698a5ea58d7f5cd3862c_40.png)

例如，当你将播放头移动到视频的第9秒并点击“向左裁剪”，那么0到9秒的内容会被直接删除，时间轴上只剩下9秒之后的视频。这比先按 `Ctrl + B` 分割，再手动选中前半段并按 `Delete` 删除要快捷得多。

![](img/a87e0b38c3be698a5ea58d7f5cd3862c_42.png)

本节课中我们一起学习了剪映的四个效率工具：“向左/向右全选”用于快速选中多轨道素材，“向左/向右裁剪”用于一键完成分割与删除。熟练掌握这些功能，能让你在剪辑小红书视频时更加得心应手，节省大量时间。
