# 小红书运营：P19：小红书后台的基础操作（电脑端） 💻

在本节课中，我们将学习如何通过电脑端操作小红书商家后台。主要内容包括保证金的缴纳与管理、支付方式的开通、运费保服务的设置以及退货地址的配置。这些是店铺运营的基础，掌握它们能让你的店铺管理更高效。

![](img/43d29aa5ffcd28831503a2ec1aa376ab_1.png)

![](img/43d29aa5ffcd28831503a2ec1aa376ab_3.png)

上一节我们介绍了手机端的操作，本节中我们来看看电脑端如何完成这些核心设置。

## 保证金管理 💰

保证金是开店的基础环节。前期可以不缴纳，但若想长期经营或开通更多支付方式，则建议缴纳。

![](img/43d29aa5ffcd28831503a2ec1aa376ab_5.png)

**保证金的几个要点：**
*   **金额**：个人店铺保证金通常在 **1000元** 左右，具体金额取决于所选商品类目。
*   **缴纳入口**：登录 **小红书千帆系统** -> 点击“资金” -> 进入“店铺保证金管理”页面进行充值。
*   **可退还**：与主流电商平台规则一致，当店铺不再经营时，保证金可以申请退还。
*   **可能扣款**：为保证消费者权益，若发生交易纠纷且商家未妥善处理（如承诺退款但未执行），平台会从保证金中扣款。正常合规经营则不会被扣。

![](img/43d29aa5ffcd28831503a2ec1aa376ab_7.png)

**公式/代码示例：**
`个人店基础保证金 ≈ 1000元`

![](img/43d29aa5ffcd28831503a2ec1aa376ab_9.png)

以下是如何查看和缴纳保证金的步骤：
1.  打开小红书千帆系统。
2.  在首页找到并点击“资金”选项。
3.  选择“店铺保证金管理”。
4.  根据自己店铺的经营类目，查看需缴纳的金额并进行充值。

![](img/43d29aa5ffcd28831503a2ec1aa376ab_11.png)

需要注意的是，部分特殊类目（如二手奢侈品、珠宝玉石、家用电器等）不支持个人店开通，需升级为个体店或企业店。企业店的保证金通常在2000元至数万元不等。

![](img/43d29aa5ffcd28831503a2ec1aa376ab_13.png)

## 支付方式开通 💳

开通多样化的支付方式可以方便更多消费者下单，提升转化率。主要支持支付宝和微信支付。

![](img/43d29aa5ffcd28831503a2ec1aa376ab_15.png)

**核心规则：**
*   **微信支付**：**必须缴纳保证金后才能开通**。提现金额将进入店铺注册法人的账户。
*   **支付宝支付**：缴纳保证金后即可开通。

![](img/43d29aa5ffcd28831503a2ec1aa376ab_17.png)

**开通路径：**
在千帆系统“资金”板块下，可以找到“支付宝”和“微信”的开通入口，按页面指引完成绑定即可。

## 运费保设置 🛡️

运费保是一项推荐开通的服务，它能有效降低消费者的下单顾虑，从而提升订单转化率。

![](img/43d29aa5ffcd28831503a2ec1aa376ab_19.png)

**运费保的作用：**
当消费者发起“七天无理由退货”并成功后，系统会自动补贴运费。这能打消买家对运费的担忧，提升下单意愿，同时降低店铺的纠纷率。

![](img/43d29aa5ffcd28831503a2ec1aa376ab_21.png)

**开通路径：**
在后台的“售后”管理板块中，找到“运费保”服务，点击开通即可。

![](img/43d29aa5ffcd28831503a2ec1aa376ab_23.png)

![](img/43d29aa5ffcd28831503a2ec1aa376ab_25.png)

## 地址设置 📍

![](img/43d29aa5ffcd28831503a2ec1aa376ab_27.png)

正确设置退货地址对于处理售后订单至关重要，尤其对于一件代发的无货源模式。

![](img/43d29aa5ffcd28831503a2ec1aa376ab_29.png)

**设置技巧：**
*   **地址填写**：在“订单物流工具”中“新建地址”。
*   **无货源商家**：可以填写供应商所在地或一个通用地址（如新疆），并在地址后附加“**联系客服**”字样。这样当有退货时，买家会先联系你，方便你进行后续处理。
*   **有货源商家**：直接填写自己的真实发货地址即可。

**操作路径：**
登录后台 -> 进入“订单物流工具” -> 在“物流/运费”板块下新建或管理退货地址。

![](img/43d29aa5ffcd28831503a2ec1aa376ab_31.png)

本节课中我们一起学习了小红书电脑端后台的四个基础操作：保证金的查看与缴纳、支付方式的绑定、运费保服务的开通以及退货地址的设置。掌握这些是店铺顺畅运营的第一步，建议你根据自己店铺的实际情况逐一完成配置。
