## 概述

![](img/504b6be0cac96e45f168b03692f0e633_4.png)

商单是小红书创作者通过发布内容为品牌方产品打广告，从而获得收入的变现方式。它与开店卖货不同，对粉丝数量的初始要求相对较低，是一种可以快速启动的盈利途径。

![](img/504b6be0cac96e45f168b03692f0e633_8.png)

## 什么是商单？

![](img/504b6be0cac96e45f168b03692f0e633_10.png)

上一节我们介绍了其他运营工具，本节中我们来看看商单。

![](img/504b6be0cac96e45f168b03692f0e633_12.png)

![](img/504b6be0cac96e45f168b03692f0e633_14.png)

商单是区别于开店的一种变现模式。开店属于细水长流型，初期订单可能较少，随着粉丝增长，订单和收入才会逐步增加。接商单则与粉丝数、浏览量的直接关系不大，通常只要有1000粉丝就能开始接单挣钱。如果操作得当，第一个月的收入可能不低于5000元。

![](img/504b6be0cac96e45f168b03692f0e633_16.png)

商单的本质是一个广告。你发布作品不是为了卖自己的产品，而是帮别人的品牌或产品打广告。

## 商单的运作平台与流程

那么，商单具体是如何运作的呢？

商单主要通过小红书的官方平台“蒲公英”进行。其基本流程如下：
1.  品牌方（例如“蓝天牌牙膏”）在蒲公英平台发布推广任务，并设定一个任务价格（例如600元）。
2.  小红书平台会从中抽取一部分作为平台服务费（例如300元）。
3.  创作者实际获得的报酬是剩余部分（例如300元）。

要在这个平台接单，创作者需要成为“蒲公英”认证的写手（或称为编辑）。这个身份并非对所有用户自动开放，平台会像招聘一样进行审核，通常要求申请者具备相关工作经验（如传媒公司编辑、品牌运营等）。不过，这个问题可以通过以合作公司员工身份上报等方式解决。

![](img/504b6be0cac96e45f168b03692f0e633_18.png)

![](img/504b6be0cac96e45f168b03692f0e633_20.png)

当你的账号粉丝达到1000并成功认证后，就可以在蒲公英平台看到大量订单，并主动选择想合作的商家。同时，认证后商家也可能主动私聊你寻求合作。

![](img/504b6be0cac96e45f168b03692f0e633_22.png)

## 商单的常见形式

![](img/504b6be0cac96e45f168b03692f0e633_24.png)

了解了商单的运作平台，我们来看看它在内容上是如何体现的。

你在小红书上看到的许多“测评”、“盘点”类内容，很可能就是商单。以下是一些具体例子：

![](img/504b6be0cac96e45f168b03692f0e633_26.png)

*   **测评盘点类**：例如一篇名为“曝光黑名单，热门美容仪别踩坑了”的笔记，看似客观地对比多个品牌，实则重点突出某个特定品牌（如“陪驰”），将其与知名品牌并列，提升其形象。
*   **合集推荐类**：例如一篇盘点多个香水的笔记。创作者通常不会自费购买所有产品进行测评，这往往是品牌方付费的推广内容。
*   **软性植入类**：在一些以互动、知识分享为主的账号（如“情商号”）中，会自然地穿插广告。例如，在讨论人际交往的话题中，植入烫发护理、防晒喷雾、面膜等产品的推广。这些广告内容并非创作者的亲身经历分享。

**核心概念**：商单 = 品牌付费 + 创作者发布广告内容。其代码逻辑可简化为：
```python
if 用户粉丝数 >= 1000 and 通过蒲公英认证:
    可接商单 = True
    收入 = 商单报价 - 平台佣金
```

## 总结

本节课中我们一起学习了小红书商单变现的基础知识。我们明确了商单是为品牌方做广告的变现方式，了解了其通过“蒲公英”平台运作的流程和分佣模式，并认识了商单在内容上常见的几种“软性”表现形式。理解商单是拓宽小红书运营收入渠道的重要一步。
