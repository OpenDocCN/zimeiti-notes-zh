# 小红书开店教程：P2-2：小红书平台检测规则详解 🕵️

在本节课中，我们将要学习小红书平台的核心检测规则。了解并遵守这些规则，是账号安全运营、避免违规处罚的基础。我们将从多个维度分析平台的最新规则和关键注意事项。

上一节我们介绍了开店的基础准备，本节中我们来看看平台具体会检测哪些内容，以及如何规避风险。

## 平台主要检测内容

小红书平台主要通过以下四个方面对账号和内容进行监测与管理。

以下是四个核心检测维度：

1.  **账号资料与互动内容检测**
    *   **检测范围**：个人资料、笔记正文、私信及评论。
    *   **禁止行为**：不得含有直接引流信息，例如：微信号、微信二维码、“代购”字样、第三方平台水印（如淘宝）、以及明显的营销词汇。
    *   **允许的替代方式**：个人资料中允许填写邮箱。可以使用“微博”或谐音词进行间接引流，再通过话术引导至微信。具体方法将在后续课程中详细讲解。

2.  **登录设备检测**
    *   **核心规则**：平台会监测账号的登录设备。
    *   **重要建议**：同一台移动设备最多只登录3个小红书账号。超出此数量可能被系统判定为营销号，存在风险。

3.  **笔记内容合规性检测**
    *   **禁止行为**：笔记内容不得诱导用户互动。例如，“关注领奖”、“点赞抽奖”等利诱性词汇是小红书明令禁止的，这属于违规的诱导用户行为。

4.  **内容原创度检测**
    *   **平台政策**：小红书对原创内容的保护程度较高。
    *   **重要建议**：切勿将同一篇文章在多个账号上重复发布。推广时，应遵循“一篇文章对应一个账号”的原则，避免被系统判定为搬运或 spam（垃圾信息）。

## 敏感词检测与处理方法

除了上述规则，平台另一项重要的检测机制是**敏感词过滤**。许多领域和词汇在小红书属于受限或违禁内容。

以下是常见的违禁领域示例，请尽量避免触及：
*   手机卡
*   金融理财
*   电子烟
*   医疗用品
*   代购

在创作笔记时，可能会无意中使用到一些未知的敏感词。为此，我们可以借助外部工具进行预先检测。

![](img/5e282495b9117adb79e4ed36d2341c3d_1.png)

这里介绍一个实用的在线检测网站。其使用方法非常简单：
1.  将写好的笔记正文粘贴到网站的输入框中。
2.  点击“立即检测”按钮。
3.  网站会自动高亮或列出正文中存在的所有小红书平台敏感词。

例如，检测句子 `“分享一款好用的代购渠道。”`，工具会标记出“**代购**”为敏感词。这样，我们就能在发布前及时修改，确保内容安全。

> **提示**：部分检测网站可能需要注册，但通常免费功能已足够日常使用。请选择可靠的工具进行检测。
