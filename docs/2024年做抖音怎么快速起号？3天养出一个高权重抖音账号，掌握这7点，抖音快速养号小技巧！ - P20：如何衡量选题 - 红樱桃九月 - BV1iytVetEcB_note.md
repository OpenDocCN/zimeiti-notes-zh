# 抖音运营：P20：如何衡量选题 📊

在本节课中，我们将学习如何判断一个抖音选题是否优秀。我们将介绍五个核心标准，帮助你从众多想法中筛选出最具潜力的内容方向，从而提升视频的吸引力和传播效果。

上一节我们介绍了如何寻找和拓展选题，本节中我们来看看如何评估这些选题的质量。

以下是衡量选题好坏的五个核心标准：

![](img/71770fd5bf446077704e62c16437cb87_1.png)

1.  **低成本**：用户希望以最小的代价（如时间、精力、金钱）获得期望的效果。例如，“玩手机躺着也能减肥”就比“50种减肥方法”更具吸引力。
2.  **精准**：内容需要明确指向特定人群的特定需求。例如，“新手必备的化妆全流程”就比泛泛的“化妆教程”更精准。
3.  **熟悉**：利用用户熟悉的场景或共同记忆引发共鸣。例如，“你有没有在电影院里遇到过这种人？”。
4.  **高频**：选题涉及的需求或场景在用户生活中出现频率高。例如，“厨房收纳”、“日常穿搭”。
5.  **痛点**：直击用户最迫切、最强烈的需求或烦恼。例如，对于已确定喜欢某款口红的用户，“如何获得大额优惠券”就是其价格痛点。

在这些标准中，**低成本**和**痛点**是最常被用到的两个关键点。

“低成本”意味着为用户提供“捷径”。其核心逻辑可以概括为：**效果 / 投入 > 用户心理预期**。当用户认为达成目标的性价比极高时，选题的吸引力就越大。

“痛点”则是用户最敏感、最希望解决的问题。例如，当用户已经认可某商品价值但嫌其太贵时，“价格”就是核心痛点。能解决这个痛点的内容（如优惠信息）就极具吸引力。

“熟悉”这个标准也经常被使用，尤其在情景剧类内容中。它通过唤起用户的共同经历来建立情感连接，例如“放假回家，妈妈总说的三句话”。

当你产生一个选题想法时，可以将其代入以上五个标准进行衡量。通常，一个选题若能符合其中两到三点，就是一个值得尝试的好选题。

当然，并非每个想法都能成为爆款，筛选和删减是必要的过程。如果你觉得所有选题都不够好，可以回到文案本身进行优化和迭代。

![](img/71770fd5bf446077704e62c16437cb87_3.png)

本节课中我们一起学习了衡量抖音选题质量的五个标准：低成本、精准、熟悉、高频和痛点。掌握这些标准，能帮助你有方向地筛选和优化选题，为创作出更具吸引力的视频内容打下坚实基础。
