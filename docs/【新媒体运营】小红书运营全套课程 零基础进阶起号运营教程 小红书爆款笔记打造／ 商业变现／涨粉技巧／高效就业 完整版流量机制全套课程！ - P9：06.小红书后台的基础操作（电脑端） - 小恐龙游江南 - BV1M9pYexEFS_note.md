# 新媒体运营：P9：06.小红书后台的基础操作（电脑端） 💻

在本节课中，我们将学习如何通过电脑端操作小红书商家后台，涵盖保证金、支付方式、运费保和地址设置等核心功能。这些是店铺运营的基础，掌握它们能帮助你更高效地管理店铺。

![](img/75e9e3f629116bd6351f8825891a1a7e_1.png)

上一节我们介绍了手机端的操作，本节中我们来看看电脑端后台的具体设置。

![](img/75e9e3f629116bd6351f8825891a1a7e_3.png)

## 一、保证金管理 💰

保证金是开设店铺时可能需要缴纳的一笔资金，用于保障消费者权益。

*   **前期可暂不缴纳**：店铺开通初期，可以不立即缴纳保证金。
*   **个人店保证金**：金额通常在 **1000元** 左右，具体数额取决于你选择的商品类目。
*   **保证金可退还**：与抖音等平台类似，当店铺不再经营时，保证金可以申请退还。
*   **保证金可能被扣除**：如果店铺发生违规行为或未履行对消费者的承诺（如未按时退款），平台会从保证金中扣款。正常合规经营则不会被扣。

以下是查询和缴纳保证金的步骤：

![](img/75e9e3f629116bd6351f8825891a1a7e_5.png)

1.  登录 **小红书千帆** 商家后台系统。
2.  在首页找到并点击 **“资金”** 模块。
3.  在资金管理页面，选择 **“店铺保证金管理”**。
4.  在此页面，你可以查看需要缴纳的保证金金额，并进行充值操作。
5.  选择你经营的商品类目，系统会显示对应的应缴金额。

![](img/75e9e3f629116bd6351f8825891a1a7e_7.png)

**公式/代码示例**：
`应缴保证金 = f(所选商品类目)`
（函数 `f` 表示保证金金额由所选类目决定）

![](img/75e9e3f629116bd6351f8825891a1a7e_9.png)

**注意**：部分特殊类目（如二手奢侈品、珠宝玉石、家用电器等）不支持个人店开通，需要升级为个体店或企业店。企业店的保证金通常在2000元至数万元不等。

![](img/75e9e3f629116bd6351f8825891a1a7e_11.png)

如果想要长远经营店铺，或者希望开通更丰富的支付方式，建议缴纳保证金。

## 二、支付方式开通 💳

![](img/75e9e3f629116bd6351f8825891a1a7e_13.png)

开通支付方式能让顾客使用更便捷的付款工具，提升下单转化率。主要包括微信支付和支付宝支付。

*   **微信支付**：**必须缴纳保证金后才能开通**。开通后，提现金额将进入注册小店时使用的法人账户。
*   **支付宝支付**：开通流程相对简单，通常可以当天完成设置。

![](img/75e9e3f629116bd6351f8825891a1a7e_15.png)

以下是开通支付方式的路径：

![](img/75e9e3f629116bd6351f8825891a1a7e_17.png)

1.  在千帆后台首页，点击进入 **“资金”** 模块。
2.  在资金页面找到 **“支付设置”** 或类似选项。
3.  分别按照指引开通 **“微信支付”** 和 **“支付宝支付”**。
4.  根据要求填写相关信息（如身份证、营业执照等）并完成绑定。

开通微信支付可以覆盖更多使用微信支付的年轻用户群体。

## 三、运费保设置 🛡️

运费保是一项增值服务，旨在降低消费者的购物顾虑，可能提升订单转化率。

*   **作用**：当消费者发起“七天无理由退货”时，系统会自动补贴退货运费，卖家无需承担。
*   **好处**：打消顾客对运费的担忧，提升下单意愿，同时减少因运费产生的交易纠纷。

![](img/75e9e3f629116bd6351f8825891a1a7e_19.png)

![](img/75e9e3f629116bd6351f8825891a1a7e_21.png)

以下是开通运费保服务的步骤：

![](img/75e9e3f629116bd6351f8825891a1a7e_23.png)

1.  在千帆后台，找到并进入 **“售后”** 管理模块。
2.  在售后设置中，找到 **“运费保”** 服务选项。
3.  阅读服务说明后，点击 **“开通”** 即可。

![](img/75e9e3f629116bd6351f8825891a1a7e_25.png)

推荐开通此服务，尤其对于无货源代发模式的商家，能有效简化售后流程。

## 四、地址设置 📍

![](img/75e9e3f629116bd6351f8825891a1a7e_27.png)

![](img/75e9e3f629116bd6351f8825891a1a7e_29.png)

正确设置发货和退货地址对于订单处理和售后服务至关重要。

*   **发货地址**：根据你的货源情况填写。如果是有货源商家，填写真实仓库地址；如果是无货源代发，可填写供应商或同步的地址。
*   **退货地址**：建议进行策略性设置。例如，可以填写一个偏远地区（如新疆）的地址，并备注 **“退货前请联系客服”**。这样可以在消费者申请退货时，争取沟通机会，避免直接退货。
*   **设置路径**：在后台找到 **“订单”** 或 **“物流工具”** 模块，选择 **“地址管理”** 或 **“新建地址”** 进行设置。

**代码示例**（地址设置逻辑）：
```plaintext
if (订单类型 == “退货”) {
    显示地址: “新疆省XX市XX区 (退货前请联系客服获取正确地址)”;
} else {
    显示地址: 真实发货仓库地址;
}
```

本节课中我们一起学习了小红书电脑端后台的四个基础操作：保证金的查询与缴纳、支付方式的开通、运费保服务的设置以及发货/退货地址的管理。这些设置是店铺顺畅运营的基石，请务必根据自身情况妥善配置。熟练掌握后台操作，能为你的店铺运营打下坚实基础。
