## 昵称设置：简单易记是关键 🏷️

![](img/0eb98d6de0b275acc92865028831122a_30.png)

昵称的作用是告诉别人“我是谁”。许多人会纠结于昵称是否足够亮眼或好记，但过度思考并无必要。人设的建立依赖于长期的运营、内容呈现和角色塑造，而非仅仅一个昵称。你只需注意以下两点，昵称就不会有太大问题。

以下是设置昵称的两个核心要点：

![](img/0eb98d6de0b275acc92865028831122a_45.png)

1.  **避免使用生僻字或复杂文字**：当用户想搜索你时，生僻字会增加他们的输入难度和搜索时间，不利于传播。这类似于网站域名的选择原则：**域名要短、易读、易记**。昵称过长或杂乱，同样难以形成有效记忆和人设。
2.  **尽量与赛道关联（非强制，但有加分）**：如果你的昵称能体现你所处的领域，会更有辨识度。例如：
    *   `萌萌老师产后修复`：明确指向产后修复领域。
    *   `某某旅行日记`：与旅行内容挂钩。
    *   `爱健身的某某某`：表明健身达人的身份。

    *   **专业类（教师、律师、医生、金融从业者等）**：必须突出专业性。应使用能体现职业身份的正装或工作场景照片，以建立信任感。

## 简介撰写：清晰明了的个人说明书 📝

一份合格的简介应做到以下几点：

![](img/0eb98d6de0b275acc92865028831122a_118.png)

![](img/0eb98d6de0b275acc92865028831122a_119.png)

![](img/0eb98d6de0b275acc92865028831122a_120.png)

![](img/0eb98d6de0b275acc92865028831122a_121.png)

![](img/0eb98d6de0b275acc92865028831122a_122.png)

![](img/0eb98d6de0b275acc92865028831122a_123.png)

1.  **讲清核心信息**：用简短的语句说明你的角色、背景、背书、成就或参与过的重点项目。无需罗列全部。
2.  **段落有序，易于阅读**：避免大段文字堆砌。合理分段，甚至可以使用表情符号进行区隔，让信息层次分明。
3.  **前期不放广告**：账号初期权重低，简介中放入联系方式或营销信息易违规。待账号运营一段时间，流量和权重提升后，再考虑添加如“私信咨询”等引导信息。
4.  **完善职业标签等资料**：许多平台（如图文笔记平台）的个人资料栏（如职业标签）会显示在主页，这是塑造人设的简单加分项，请如实或适当包装后填写。
5.  **封面与背景图风格统一**：所有内容封面的视觉风格需要提前设计（后续课程会详述）。背景图的设置原则与头像一致：专业赛道突出专业感，生活赛道突出真实感。
