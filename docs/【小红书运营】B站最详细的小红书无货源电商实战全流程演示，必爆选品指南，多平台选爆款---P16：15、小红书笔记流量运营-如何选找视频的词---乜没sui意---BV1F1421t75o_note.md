# 小红书运营：P16：15、小红书笔记流量运营-如何用词找视频 🔍

在本节课中，我们将学习如何通过关键词来寻找视频素材。这是小红书无货源电商运营中，高效获取对标视频内容的核心方法。我们将重点讲解如何选择范围更广、数量更多的关键词，并演示具体的操作流程。

上一节我们介绍了选品的基本思路，本节中我们来看看如何为选定的方向寻找合适的视频内容。寻找视频的第一步是确定搜索关键词，关键词的选择范围直接决定了你能找到的视频素材的广度与数量。

## 关键词选择原则

![](img/943bc450eb879be4665d69b6f751ed3a_1.png)

关键词不能过于狭隘。如果词太具体，相当于直接锁定了某个特定产品，这不利于我们广泛地寻找视频素材。因此，我们需要选择范围更大的词。

![](img/943bc450eb879be4665d69b6f751ed3a_3.png)

![](img/943bc450eb879be4665d69b6f751ed3a_5.png)

正确的做法是选择**一级、二级或三级类目的大词**。例如，在“童装”这个类目下：
*   **一级类目词**：`童装`
*   **二级类目词**：`童装卫衣`
*   **三级类目词**：`童装卫衣辣妹装`

![](img/943bc450eb879be4665d69b6f751ed3a_7.png)

![](img/943bc450eb879be4665d69b6f751ed3a_9.png)

你可以直接使用这些类目词进行搜索。在发布笔记时，平台也会有类目选择，你可以按顺序选择一级、二级类目作为参考。

![](img/943bc450eb879be4665d69b6f751ed3a_11.png)

![](img/943bc450eb879be4665d69b6f751ed3a_13.png)

以下是具体操作步骤：
1.  确定你的产品方向（如：母婴 -> 童装）。
2.  提取对应的类目关键词，例如 `童装`、`童装卫衣`、`儿童羽绒服`。
3.  复制这些关键词，直接去视频平台（如抖音）搜索，即可找到大量相关视频。

![](img/943bc450eb879be4665d69b6f751ed3a_15.png)

## 关键词扩展方法

在确定了一级或二级类目词之后，我们可以进行二次扩展，以发现更多样、更精准的搜索方向。一个有效的方法是使用**小红书下拉词**进行扩展。

![](img/943bc450eb879be4665d69b6f751ed3a_17.png)

![](img/943bc450eb879be4665d69b6f751ed3a_19.png)

以下是利用小红书下拉词扩展关键词的步骤：
1.  打开小红书APP，点击搜索框。
2.  输入你选定的一级或二级类目词，例如 `童装`。
3.  观察搜索框下方自动弹出的推荐关键词，如“童装发圈文案”、“童装穿搭”、“童装裙”等。
4.  选择其中你认为有潜力的推荐词，或者换一个二级词（如 `童装卫衣`）进行搜索，会得到更丰富的下拉推荐，例如“童装卫衣辣妹装”、“童装卫衣下衣失踪”等。
5.  将这些扩展后的关键词（如 `童装卫衣辣妹装`）复制下来，前往抖音等视频平台搜索，就能找到对标的视频素材。

![](img/943bc450eb879be4665d69b6f751ed3a_21.png)

![](img/943bc450eb879be4665d69b6f751ed3a_23.png)

**核心概念**：`搜索关键词 = 一级/二级类目词 + 下拉推荐扩展词`

![](img/943bc450eb879be4665d69b6f751ed3a_25.png)

## 操作要点与技巧

![](img/943bc450eb879be4665d69b6f751ed3a_27.png)

![](img/943bc450eb879be4665d69b6f751ed3a_29.png)

通过词找视频，与我们之前“确定产品后再找词”的逻辑不同。这里是为了直接寻找视频素材，所以需要先选定一些范围较大的词。

![](img/943bc450eb879be4665d69b6f751ed3a_31.png)

你同样可以在抖音的搜索框中进行关键词的二次扩展。例如，搜索 `童装卫衣辣妹装` 后，观察抖音给出的相关搜索推荐，可能会发现“女卫衣童装”等新方向。因为热门趋势在抖音和小红书平台通常具有相似性，这能帮助我们拓宽选材方向。

![](img/943bc450eb879be4665d69b6f751ed3a_33.png)

![](img/943bc450eb879be4665d69b6f751ed3a_35.png)

在选择下拉扩展词时，有一个重要技巧：**尽量选择二级或三级词**。
*   `童装`（一级词）范围太宽，不够精准。
*   `童装卫衣`（二级词）或 `童装卫衣辣妹装`（三级词）则更为具体，能帮助我们找到风格和内容更聚焦的视频，提高选材效率。

![](img/943bc450eb879be4665d69b6f751ed3a_37.png)

本节课中我们一起学习了如何通过关键词寻找视频素材。我们掌握了选择一、二、三级类目大词的原则，并学会了利用小红书下拉词进行关键词扩展的方法。记住，选择范围适中的二级或三级词进行搜索，是高效找到优质对标视频的关键。在接下来的课程中，我们将学习如何从找到的这些视频中筛选出可用的爆款素材。
