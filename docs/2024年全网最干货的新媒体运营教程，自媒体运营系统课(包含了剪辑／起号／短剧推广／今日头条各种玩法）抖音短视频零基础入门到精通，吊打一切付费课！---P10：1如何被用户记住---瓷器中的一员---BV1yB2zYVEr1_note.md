# 新媒体运营系统课：P10：如何被用户记住

在本节课中，我们将要学习一个关键概念：如何让你的账号在众多竞争者中脱颖而出，被用户牢牢记住。上一节我们介绍了账户搭建与内容定位，本节中我们来看看如何通过“超级符号”建立独特的用户记忆点。

## 什么是超级符号？

![](img/dc97e95e51bef267995317d5828abb00_1.png)

![](img/dc97e95e51bef267995317d5828abb00_2.png)

![](img/dc97e95e51bef267995317d5828abb00_3.png)

当你的账号定位清晰后，面临的下一个挑战是：如何让用户记住你、爱上你并关注你。这便涉及到“超级符号”的运用。超级符号是一个独特的、可重复的标识，它可以是视觉形象、行为动作、口头禅或声音，能帮助用户在众多信息中快速识别并记住你。

![](img/dc97e95e51bef267995317d5828abb00_4.png)

### 其他形式的超级符号

![](img/dc97e95e51bef267995317d5828abb00_26.png)

![](img/dc97e95e51bef267995317d5828abb00_28.png)

除了行为和话语，超级符号还可以有多种形式：

*   **视觉符号**：例如，一个总是戴着特定款式帽子的人，帽子就成了他的视觉超级符号。
*   **听觉符号**：特定的背景音乐、音效或说话语调（如“OK不？”、“宝贝”）都可以成为听觉符号。

这些符号的核心目的，是降低用户的记忆成本，让他们能通过一个简单的元素快速想起你。

## 如何打造你的超级符号？

理解了超级符号的概念后，我们来看看如何为自己的账号设计一个有效的超级符号。

以下是设计超级符号时可以遵循的几个步骤：

1.  **关联定位**：超级符号必须与你的账号定位（商业定位、内容定位）强相关。木鱼医生的木鱼契合其“科普”中带点“禅意”或“警醒”的调性。
2.  **简单易记**：符号必须足够简单、独特且易于重复。一个复杂的动作或拗口的句子很难传播。
3.  **保持一致性**：一旦确定，就要在所有内容中反复、一致地使用这个符号，强化用户印象。
4.  **多感官结合**：效果最佳的超级符号往往能同时调动多种感官。例如，木鱼医生结合了视觉（动作）和听觉（音效）；李佳琦的话语符号也配合了其夸张的视觉表情。
