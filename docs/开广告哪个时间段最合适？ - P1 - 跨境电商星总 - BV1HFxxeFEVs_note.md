# 跨境电商广告投放教程：P1：如何选择最佳广告时段

在本节课中，我们将学习如何为跨境电商广告选择最合适的投放时间段。这是优化广告效果、控制预算的关键一步。

上一节我们介绍了广告投放的基本概念，本节中我们来看看如何确定具体的投放时间。

对于新用户，建议全天开启广告。这样做的目的是为了收集足够的数据，了解广告在不同时间段的表现。

当广告运行一段时间后，你需要分析数据，找出点击发生最集中的时间段。

以下是分析广告数据后可以采取的优化步骤：
*   查看广告报告，识别点击量最高的时间段。
*   根据识别出的高峰时段，调整广告投放设置。
*   在效果好的时间段，可以适当增加预算以获得更多曝光。

通过以上步骤，你可以将预算集中在效果最佳的时间段，从而提升广告的整体投资回报率。

本节课中我们一起学习了如何通过数据分析来优化广告的投放时间。核心方法是：初期**全天投放**以收集数据，后期根据**点击集中的时间段**进行聚焦投放。
