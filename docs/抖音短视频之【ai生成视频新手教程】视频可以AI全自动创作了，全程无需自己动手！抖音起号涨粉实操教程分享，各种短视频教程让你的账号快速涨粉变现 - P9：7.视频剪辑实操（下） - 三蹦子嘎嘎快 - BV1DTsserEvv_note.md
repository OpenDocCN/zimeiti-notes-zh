# AI短视频创作教程：P9：视频剪辑实操（下）🎬

在本节课中，我们将继续学习视频剪辑的后续关键步骤，重点是如何为拼接的视频片段添加转场效果，以及如何为视频添加字幕。这些操作能有效提升视频的观感和完播率。

上一节我们介绍了如何调整视频比例并添加背景，本节中我们来看看如何让多个视频片段之间的衔接更自然。

![](img/6baea12571548d2c72c238378c175292_1.png)

## 添加视频转场

我们的视频并非一个完整的原有视频，而是由多个约十秒钟的片段拼接而成。因此，视频片段之间必须有衔接，我们需要添加转场视频。

![](img/6baea12571548d2c72c238378c175292_3.png)

为什么要添加转场？目的是让用户明确感知到每个片段的内容是不同的。如果不添加转场，用户在观看时切换到下一个片段，可能会因为内容突然变化而感到困惑，甚至退出视频查看。这个退出动作会导致视频的**完播率**直线下降，而完播率直接影响视频的单价和最终收益。所以，添加转场是必要的。

![](img/6baea12571548d2c72c238378c175292_5.png)

![](img/6baea12571548d2c72c238378c175292_7.png)

以下是添加转场的具体步骤：

![](img/6baea12571548d2c72c238378c175292_9.png)

1.  在视频编辑轨道上，找到右侧的加号（`+`）按钮并点击。
2.  在弹出的菜单中选择“素材库”。
3.  在素材库的“热门”分类中，可以找到大量转场素材，例如各种动态切换效果。这些转场素材在各类短视频中非常常见，可以放心使用。

![](img/6baea12571548d2c72c238378c175292_11.png)

在选择转场时，请注意一点：不要选择时长过长的转场，例如16秒的。因为观众通常没有耐心看完不重要的内容。我们应尽量选择时长在**3到5秒**之间的转场，其中3秒左右最为合适。

![](img/6baea12571548d2c72c238378c175292_13.png)

选择好转场素材并添加后，还需要确保转场视频的比例与主视频保持一致，即**16:9**的横屏比例。添加后，在编辑界面右下角可以找到“画布模糊”选项，点击并选择第一个清晰的画面，即可将转场背景调整为正确的比例。最后，记得点击右下角的对勾（`√`）确认操作。

![](img/6baea12571548d2c72c238378c175292_15.png)

## 为视频添加字幕

![](img/6baea12571548d2c72c238378c175292_17.png)

完成转场添加后，下一步是为视频添加文案，也就是字幕。

以下是添加字幕的具体步骤：

1.  找到手机编辑界面左下角的“文字”选项并点击。
2.  在文字功能中，选择“新建文本”右侧的“识别字幕”功能。
3.  点击“开始匹配”，系统会自动识别视频或音频中人物所说的话，并生成对应的字幕。

![](img/6baea12571548d2c72c238378c175292_19.png)

字幕生成后，通常需要稍作调整以达到最佳视觉效果：

![](img/6baea12571548d2c72c238378c175292_21.png)

*   **放大字幕**：拖动字幕文本框右下角，向右拖动即可放大文字。
*   **调整位置**：将放大后的字幕文本框拖动到视频画面最下方的位置，这样观看起来最舒服。

调整好第一句字幕后，后续的字幕都会自动应用相同的样式和位置，非常方便。

![](img/6baea12571548d2c72c238378c175292_23.png)

## 核心要点回顾

本节课中我们一起学习了视频剪辑后半部分的核心操作：

![](img/6baea12571548d2c72c238378c175292_25.png)

1.  **添加转场**：在视频片段间插入3-5秒的转场素材，以明确区分内容、提升观感、保障完播率。操作路径为：轨道加号 (`+`) → 素材库 → 选择热门转场。
2.  **统一比例**：确保添加的转场视频也调整为**16:9**的横屏比例，使用“画布模糊”功能实现。
3.  **添加字幕**：使用“识别字幕”功能自动生成字幕，并调整其大小和位置至视频底部，以增强视频的可读性和专业性。

![](img/6baea12571548d2c72c238378c175292_27.png)

请严格按照这些步骤操作，在寻找素材时也避免使用过长的视频，以免影响平台审核。整个流程并不复杂，但每一步都对最终视频的质量至关重要。
