# 微信视频号运营教程：P33：19.实操步骤 - 开通与上架商品

在本节课中，我们将学习如何在微信视频号中开通商品橱窗功能并上架商品。这是进行视频号带货的基础操作，步骤清晰简单，适合初学者跟随操作。

上一节我们介绍了视频号运营的整体思路，本节中我们来看看如何具体开通带货功能。

## 第一步：打开微信
首先，我们需要启动微信应用。这是所有后续操作的基础。

## 第二步：进入“发现”页
在微信主界面底部导航栏中，找到并点击 **“发现”** 按钮。

![](img/0e56b76a43c0ad78c9a174fdf2014aea_1.png)

## 第三步：找到视频号入口
在“发现”页面中，将视线向上移动。您会看到 **“视频号”** 的入口，其图标是一个黄色的蝴蝶形状。点击该入口进入视频号。

![](img/0e56b76a43c0ad78c9a174fdf2014aea_2.png)

![](img/0e56b76a43c0ad78c9a174fdf2014aea_1.png)
![](img/0e56b76a43c0ad78c9a174fdf2014aea_2.png)

![](img/0e56b76a43c0ad78c9a174fdf2014aea_4.png)

## 第四步：进入个人中心
进入视频号后，在界面右上角找到一个 **人形头像** 图标。点击该图标进入个人中心。

![](img/0e56b76a43c0ad78c9a174fdf2014aea_6.png)

![](img/0e56b76a43c0ad78c9a174fdf2014aea_4.png)
![](img/0e56b76a43c0ad78c9a174fdf2014aea_6.png)

## 第五步：进入创作者中心
在个人中心页面，向下滑动屏幕，直到页面最底部。找到并点击 **“创作者中心”**。

> 注意：如果您的账号是全新注册且未发布过任何作品，可能暂时看不到此入口。发布一条作品后即可出现。

![](img/0e56b76a43c0ad78c9a174fdf2014aea_8.png)

![](img/0e56b76a43c0ad78c9a174fdf2014aea_9.png)

## 第六步：找到带货中心
进入创作者中心后，您会看到数据面板。在其中找到名为 **“带货中心”** 的模块，其图标类似一个小卖部。点击进入。

![](img/0e56b76a43c0ad78c9a174fdf2014aea_8.png)
![](img/0e56b76a43c0ad78c9a174fdf2014aea_9.png)

接下来，系统会根据您的账号状态引导您进行不同操作。

## 情况一：开通橱窗权限（未开通用户）
如果您尚未开通商品橱窗功能，点击“带货中心”后会跳转到开通权限的界面。

以下是开通权限的具体步骤：

![](img/0e56b76a43c0ad78c9a174fdf2014aea_11.png)

1.  **选择带货身份**：界面会要求您选择带货身份。通常有三个选项：
    *   我是商家
    *   **我是达人**
    *   我是商家且已有视频号
    对于绝大多数个人创作者，请选择 **“我是达人”**。

2.  **交纳保证金**：选择身份后，会跳转到交纳保证金的页面。您需要点击 **“去交纳”** 按钮。
3.  **支付费用**：系统将引导您通过微信支付交纳 **100元** 的保证金。完成支付后，权限即开通成功。

![](img/0e56b76a43c0ad78c9a174fdf2014aea_11.png)
![](img/0e56b76a43c0ad78c9a174fdf2014aea_13.png)

> 提示：一个身份证可以认证两个账号。建议第一个账号先选择“达人”身份，使用平台的选品中心（产品是0成本的）。第二个账号再根据实际情况选择“商家”身份。

![](img/0e56b76a43c0ad78c9a174fdf2014aea_13.png)

## 情况二：上架商品（已开通用户）
如果您已经开通了橱窗权限，点击“带货中心”后，将直接进入商品管理界面。您可以在此从平台的选品中心选择商品，并将其添加到自己的橱窗中。

整个操作流程的核心逻辑如下：
```
点击“带货中心” -> 判断权限状态 -> (未开通)引导开通 -> (已开通)进入商品上架界面
```

![](img/0e56b76a43c0ad78c9a174fdf2014aea_15.png)

![](img/0e56b76a43c0ad78c9a174fdf2014aea_15.png)

本节课中我们一起学习了微信视频号开通商品橱窗和上架商品的全流程。关键点在于通过“创作者中心”进入“带货中心”，并根据系统指引完成身份选择（推荐“达人”）和保证金交纳。完成这些步骤后，您就可以开始为您的视频号添加商品，开启带货之旅了。
