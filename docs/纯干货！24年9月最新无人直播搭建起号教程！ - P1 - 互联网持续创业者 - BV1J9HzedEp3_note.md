# 无人直播搭建教程：P1：电脑端无人直播搭建起号全流程 🚀

## 概述
在本节课中，我们将学习如何在电脑端搭建无人或半无人直播间。课程将涵盖三种主流直播类型（纯无人、半无人出镜、半无人不出镜）的搭建流程、所需软件工具、素材处理方法以及规避平台违规的核心技巧。

---

## 一、电脑端直播的三种类型

上一节我们概述了课程内容，本节中我们来看看电脑端直播的三种主要搭建类型。

**1. 纯无人直播**
*   **定义**：直播间画面完全由预先录制或剪辑的视频素材播放构成，声音由音频素材播放。
*   **核心**：通过调用视频和音频素材，构建虚拟的“人、货、场”场景来实现自动开播。

**2. 半无人直播（出镜）**
*   **定义**：采用“背景层 + 人像层”的两层结构。
*   **背景层**：剪辑好的视频素材。
*   **人像层**：真实主播在绿幕前被摄像头拍摄，并通过抠图技术合成到背景层上。
*   **优势**：无需搭建实体商品场景，成本低，通过更换背景视频即可轻松换品，场景多样化。

**3. 半无人直播（不出镜）**
*   **定义**：主播不露脸，但直播间画面包含实景内容。
*   **构成**：采用特定实景视频素材（如店铺、工厂）作为背景，同时配合样品展示、手部出镜等实时操作进行直播。
*   **优势**：适合不希望露脸的主播，可将账号打造为“店播”或“厂播”账号，增强带货可信度。

## 二、搭建前的准备工作

了解了直播类型后，在开始动手搭建前，我们需要做好以下两项核心准备。

### （一）硬件：电脑配置要求
稳定的直播体验依赖于合格的硬件。以下是基础的下限配置标准：
*   **CPU**：英特尔 i5 十代或以上
*   **显卡**：显存 4GB 或以上
*   **内存**：16GB
*   **系统**：Windows 10 或 11（64位）
*   **说明**：此为标准下限，可根据预算选择更高配置。配置不足会导致直播卡顿、用户体验差，并增加违规风险。

### （二）软件：账号与权限
电脑端直播的核心软件是抖音官方的“直播伴侣”。
*   **开播权限**：正常申请需要账号拥有 **1000粉丝** 且近期无违规记录。
*   **无1000粉丝的解决方案**：
    1.  通过其他内容积累粉丝至1000。
    2.  通过电商平台（如某多多、某宝）搜索“直播伴侣开播权”服务，付费获取开通帮助（适用于不开通带货功能，仅做电脑直播的用户）。
*   **注意**：如需在直播中挂载小黄车（带货）、小房子（团购）、小雪花（课程）等链接，则必须满足1000粉丝的条件。

## 三、核心软件：直播伴侣基础操作

准备工作就绪后，本节我们来熟悉最重要的开播软件——直播伴侣的基本界面和功能。

以下是直播伴侣主界面各功能区介绍：
1.  **模式切换**：可在“常规模式”（视频直播）和“语音模式”之间切换。
2.  **导播功能**：用于多场景直播时的画面切换（预览画面与输出画面分离）。
3.  **画面比例**：务必切换为 **竖屏（9:16）**，分辨率应为 **1080x1920**，以适配手机观看。
4.  **添加素材**：构建直播画面的核心区域。可以添加摄像头、视频、图片、窗口采集等。
5.  **直播工具**：
    *   **直播设置**：关键设置项，包含视频（清晰度）、音频（麦克风与扬声器）、直播间权限（如是否允许观众互相关注）。
    *   **链接挂载**：可挂载“抖音电商”（带货）、“生活服务”（团购）、“小程序”（课程）等链接。
    *   **直播标题/内容**：设置直播标题、选择直播内容分类、填写直播介绍。
6.  **开始直播**：所有设置完成后，点击此按钮开始推流。

**开播流程公式**：
`登录账号` -> `设置竖屏画布` -> `添加并布置素材` -> `进行直播设置（视频、音频、权限）` -> `挂载所需链接` -> `填写直播信息` -> `点击开始直播`

## 四、进阶方案：为何及如何使用 OBS

上一节我们介绍了单独使用直播伴侣的方法，但为了更灵活、更安全地搭建素材型直播间，我们需要引入一款强大的辅助工具——OBS。

### （一）为什么需要 OBS？
单纯在直播伴侣中添加视频素材循环播放，极易被平台判定为“录播”违规。引入OBS的核心目的是：
1.  **规避录播检测**：将视频素材放在OBS中处理，再以“虚拟摄像头”的形式提供给直播伴侣，可以降低直接判罚录播的风险。
2.  **处理复杂场景**：OBS支持更丰富的图层、源和滤镜效果，便于构建多层画面。

### （二）OBS 与直播伴侣的衔接流程
以下是两者协同工作的核心步骤：
1.  **在OBS中设置**：
    *   创建竖版画布（1080x1920）。
    *   `添加来源` -> `媒体源`，导入视频文件并勾选“循环播放”。
    *   启动 **虚拟摄像机** 功能。
    ```bash
    # 操作路径示意
    OBS：文件 -> 设置 -> 视频 -> 基础分辨率设为 1080x1920
    OBS：来源面板 -> “+” -> 媒体源 -> 浏览选择视频 -> 勾选“循环” -> 确定
    OBS：右下角控件 -> “启动虚拟摄像机”
    ```
2.  **在直播伴侣中调用**：
    *   `添加素材` -> `摄像头`。
    *   在摄像头设备列表中，选择 **OBS Virtual Camera**。
    *   此时，OBS中的画面即被接入直播伴侣。

## 五、关键问题：规避“非实时直播”违规

通过OBS接入素材后，可能面临新的违规类型——“非实时直播”。这是因为画面仍是循环素材，缺乏实时互动元素。

**解决方案：叠加实时实景层**
在直播伴侣中，**在OBS虚拟摄像头图层之上**，再添加一个真实的摄像头。
*   **操作方法**：`添加素材` -> `摄像头`，选择你的物理摄像头（如罗技C920）。
*   **场景应用**：
    *   **半无人出镜**：将此摄像头对准绿幕前的主播，进行抠图，将真人合成到画面中。
    *   **半无人不出镜/纯无人优化**：将此摄像头对准绿幕前的样品或手写牌。通过抠图，将实时移动的**样品**、**手部动作**或**文字提示牌**叠加到直播画面中。
*   **核心作用**：这些实时、动态的实景元素能有效“调节画面”，向平台证明直播间的实时性，从而规避“非实时”判罚。

**规避违规核心公式**：
`循环素材背景` + `实时动态实景层（真人/样品/手部）` + `实时口播或互动` = `降低违规风险`

## 六、声音与清晰度设置

画面问题解决后，声音和清晰度是影响观看体验的另外两个关键。

### （一）声音传递（OBS -> 直播伴侣）
默认情况下，OBS中的视频声音不会自动传到直播伴侣。需按以下步骤设置：
1.  在OBS中，点击 **音频混合器** 中对应媒体源旁边的 **齿轮图标**。
2.  选择 **“高级音频属性”**。
3.  将该媒体源的 **“音频监听”** 模式由“关闭监听”改为 **“监听并输出”**。
4.  此时，直播伴侣的“扬声器”设备才能接收到OBS播放的音频。

### （二）画面清晰度设置
1.  **OBS内清晰度**：
    *   确保输出画布分辨率是1080x1920。
    *   在直播伴侣的摄像头设置中，将OBS虚拟摄像头的“视频格式”设为 **NV12** 或 **YUY2**，“色彩空间”设为 **709**，“色彩范围”设为 **局部**。
2.  **直播推流清晰度**：
    *   在直播伴侣的 `直播设置` -> `视频` 中，选择“蓝光”或“自定义”。
    *   建议分辨率 **1920x1080**，码率 **8000-10000 kbps**，帧率 **30**（根据电脑性能可尝试60）。
    *   视频编码优先选择 **硬件编码（如NVIDIA NVENC）**。

## 八、半无人直播的合规搭建要点

本节我们综合运用前面所学，重点讲解如何搭建一个能“放心起号”的半无人直播间。

### （一）半无人出镜直播间
*   **结构**：OBS播放背景视频 + 真实摄像头拍摄绿幕前主播。
*   **合规核心**：
    1.  **主播出镜比例**：主播画面应占屏幕 **三分之一以上**，避免过小或仅露侧脸。
    2.  **展示实物**：主播应手持或口播介绍**实物商品**，避免“未展示实物商品”违规。
    3.  **真人口播**：必须由主播**实时口播**，严禁播放预录音频对口型。

### （二）半无人不出镜（手播）直播间
*   **结构**：OBS播放实景背景视频（如工厂流水线）+ 真实摄像头拍摄绿幕前的样品和手部操作。
*   **合规核心**：
    1.  **实景背景**：使用真实的店铺、工厂等场景视频，增强可信度。
    2.  **实时操作**：通过手部展示样品细节、更换产品、举起提示牌（如“点关注”）等动作，提供充足的实时动态。
    3.  **真人口播**：同样需要实时讲解，可与手部动作配合。
