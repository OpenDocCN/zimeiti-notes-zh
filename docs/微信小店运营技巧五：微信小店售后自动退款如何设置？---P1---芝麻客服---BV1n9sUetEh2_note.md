## 大促期间的售后挑战

上一节我们介绍了微信小店的基本运营玩法。本节中，我们来看看大促期间可能遇到的售后问题。

在双十一、双十二、618等电商促销活动期间，由于商品热销和促销活动的吸引，店铺订单量会急剧增加。与此同时，申请退款的顾客数量也可能随之上升。

面对这种情况，商家需要高效处理仅退款请求，以确保顾客满意度。部分商家会选择在大促前增加兼职客服人员来处理退款等售后问题，但这会导致人工成本上升。

## 解决方案：售后自动退款功能

为了解决上述问题，我们可以使用“芝麻视频号助手”的售后自动退款功能。通过设置自动审核规则，可以简化退款流程。对于符合特定条件的退款请求，系统能够自动通过并立即完成退款。

芝麻视频号助手支持同时管理多个微信小店。企业可以为不同的店铺或商品设置不同的自动退款规则。

## 如何设置自动退款规则

以下是设置自动退款规则的核心步骤与可配置条件：

**核心规则逻辑**：当订单满足预设条件时，系统自动同意用户的仅退款申请。

![](img/1732221a7739cc5874d56e0bab0d58c9_1.png)

可配置的条件主要包括：
*   **用户下单时间**：例如，可设置为下单后24小时内。
*   **订单金额**：例如，可设置为金额在200元以内的订单。
*   **申请售后时间**：指用户提交退款申请的时间点。
*   **订单状态**：例如，可限定为“尚未发货”的订单。

![](img/1732221a7739cc5874d56e0bab0d58c9_2.png)

当订单同时满足所有设置的条件时，用户申请仅退款，系统就会自动同意并退款。

**规则示例**：
`如果 订单状态 == “未发货” 且 下单时间 <= 24小时 且 订单金额 <= 200元，则 自动同意退款。`
