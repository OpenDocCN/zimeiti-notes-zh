# AI小红书运营教程：P14：01.10.2：训练AI写出符合平台风格的文案

## 概述
在本节课中，我们将学习如何训练AI（如ChatGPT）来撰写符合小红书平台风格的文案。上一节我们介绍了如何让AI生成爆款标题，本节我们将重点探讨如何让AI理解并创作出符合平台调性的完整内容。

---

## 从标题到内容：训练AI的核心思路
上一节我们介绍了如何让AI写出符合平台风格的标题。这个方法不仅适用于小红书，也适用于抖音、哔哩哔哩等其他平台。每个平台都有其独特的关键词和文化氛围，以及大众认可的标题形式。关键在于提取这些重点关键词，整理成你自己的“标题库”。每次需要写标题时，将库中的关键词和你的主题结合，指令AI生成，就能得到符合平台风格的标题。

关于AI写标题的内容就介绍到这里。接下来，我们需要重点研究的是**内容本身**。这就是本节要讲解的核心：如何用AI写出符合爆款结构或平台风格的内容体系。

---

## 训练AI理解平台文案风格
大部分内容ChatGPT都能撰写，但如何让它写出符合特定平台风格的内容呢？这需要讲究几个要点。

第一个要点是**文案风格**。为了让AI理解你需要的风格，你必须提供案例。你需要告诉AI：“我提供这些内容作为参考，我希望你写出类似风格的内容。” 这意味着你需要给AI“喂”材料——提供一定量的、其他人的爆款文章。

![](img/87e3d8d528c4e8669e558a0536845d08_1.png)

![](img/87e3d8d528c4e8669e558a0536845d08_2.png)

![](img/87e3d8d528c4e8669e558a0536845d08_3.png)

这些文章能让AI分析出平台特有的风格，例如：
*   女性化、少女般活泼的用语。
*   平台专用语，如小红书上用户间常称呼“宝子”。

我们的目标是让ChatGPT通过分析提供的材料，自行总结出这些风格特点，并记忆下来，以便后续根据这些特点进行创作。

## 实操演示：训练AI分析小红书文案
以下是训练AI分析小红书文案风格的具体操作步骤。

首先，我们需要给AI清晰的指令，告诉它接下来要做什么。

![](img/87e3d8d528c4e8669e558a0536845d08_7.png)

![](img/87e3d8d528c4e8669e558a0536845d08_8.png)

**指令示例：**
```
接下来我会给你提供一些小红书上的爆款笔记文案。你需要根据这些文案，分析小红书平台的文案撰写风格、特定的平台用语、词汇、语气等。
每次接收文案后，请告诉我你已经学习理解，并给出你对这篇文案的风格、语气、类型等相关分析。
```

![](img/87e3d8d528c4e8669e558a0536845d08_10.png)

![](img/87e3d8d528c4e8669e558a0536845d08_12.png)

这个指令明确了AI的任务：1) 学习我们提供的资料库；2) 每次学习后给出反馈和分析，确保我们知道它已理解。

![](img/87e3d8d528c4e8669e558a0536845d08_14.png)

我们以“张家界旅游攻略”为例进行演示。找到一篇小红书笔记，将标题和正文内容提供给AI。

**提供给AI的文案示例（已清理格式）：**
```
标题：张家界5天4晚超全攻略！血泪教训，不踩坑建议！
文案：刚和闺蜜从张家界回来……（此处为具体文案内容）……希望姐妹们都能玩得开心！
```

AI在接收这篇文案后，给出了分析反馈，例如：
*   **风格分析**：以个人游玩经历为主线，增加了真实感和亲近感。
*   **标题分析**：以“血泪教训”和“不踩坑建议”为标题，强调了作者的经验教训，传达了一种亲切的分享心态。
*   **语气分析**：表达了担忧、焦虑，同时送上祝愿。

这个分析过程表明AI正在尝试理解文案的特点。接着，我们需要持续为AI提供更多不同风格的文案材料。

**持续“喂”材料的注意事项：**
1.  这项工作不是一次性的。在ChatGPT的同一对话中持续提供资料，AI会读取并记忆前面的内容。
2.  提供的资料需要具有**代表性**和**多样性**。目标是让AI理解小红书平台分享类内容的表述方式（如第一人称叙述）、语气以及平台用语。
3.  避免只提供通用性内容（如单纯罗列“第一天、第二天”的路线）。应寻找具有鲜明小红书风格的文案，例如包含强烈个人感受、吐槽或种草安利的笔记。

通过多次提供不同风格的文案，我们就在逐步构建AI对小红书平台风格的认知。

## 指令AI进行创作与效果评估
在提供一定量的学习材料后，我们可以指令AI进行创作。

**创作指令示例：**
```
请结合你的学习，帮我写一篇具有爆款潜质的张家界5日游攻略小红书笔记。笔记需涵盖丰富的小红书风格的emoji表情，同时配上十个符合小红书旅游赛道风格的爆款标题。
```

AI根据指令生成了一篇笔记和十个标题。然而，评估发现其生成的**旅游攻略内容比较空泛**，更像景点罗列，缺乏深度分析和实用建议（如哪些景点必去/可省略、具体路线规划、避坑要点）。

**效果不理想的原因分析：**
1.  **训练材料不足或不够典型**：我们提供的旅游攻略类文案可能深度分析较少，导致AI学到的风格有限。
2.  **创作指令可以更细化**：对于旅游攻略这类需要具体信息的内容，可以分步指令。例如，先让AI列出张家界所有主要景点，再针对每个景点规划路线、分析特色与避坑点。
3.  **AI模型版本影响**：使用ChatGPT 3.5与更先进的4.0版本，在理解复杂指令和生成内容质量上会有显著差异。4.0版本通常表现更聪明、更符合要求。

同时，AI生成的标题也可能不符合预期。这时，我们需要**结合上一节的标题训练方法**。可以在一个新的对话中，先提供固定的标题生成指令和爆款关键词库，再让它针对“张家界5日游攻略”这个主题生成标题，这样效果会更好。

**优化后的标题生成流程：**
1.  新建对话。
2.  输入预设的标题生成指令和关键词库。
3.  要求AI生成特定主题的标题。

经过优化，AI生成的标题质量（如“明星同款打卡地点”、“氛围感满满”、“超低价机票发现人间烟火美景”等）会得到提升，更贴近小红书的爆款风格。

## 核心要点总结与下节预告
本节课我们一起学习了训练AI撰写小红书风格文案的核心方法。

**核心要点总结：**
1.  **明确训练目标**：让AI理解特定平台的文案风格、语气和用语。
2.  **提供高质量材料**：持续为AI“喂”入多样化的、具有平台特色的爆款文案作为学习资料。
3.  **给予清晰指令**：在学习和创作阶段，都需要给AI明确、具体的指令。
4.  **分步优化**：对于复杂内容（如深度攻略），可采用分步指令的方式引导AI创作。
5.  **结合专项训练**：将内容风格训练与标题生成等专项训练结合使用，效果更佳。

训练AI的本质，是让它从一个知识丰富的通用模型，转变为理解你特定领域要求的“专家”。我们需要通过各种指令和材料，教会它理解我们的真实需求。
