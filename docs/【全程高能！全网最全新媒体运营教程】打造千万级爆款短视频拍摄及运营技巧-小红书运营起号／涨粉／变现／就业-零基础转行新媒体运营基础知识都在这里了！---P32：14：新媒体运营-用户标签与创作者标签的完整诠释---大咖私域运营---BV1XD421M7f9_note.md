# 新媒体运营：P32：用户标签与创作者标签的完整诠释 🏷️

![](img/a60efb2c9bc26942218545ab3bca0636_0.png)

![](img/a60efb2c9bc26942218545ab3bca0636_2.png)

在本节课中，我们将要学习新媒体平台（如抖音、小红书）中两个核心概念：**用户标签**与**创作者标签**。理解这两个标签的运作机制，是做好账号垂直定位、实现精准内容分发的关键。我们将通过简单的数据和逻辑，帮助你清晰掌握平台如何识别用户喜好以及如何为创作者打标签。

![](img/a60efb2c9bc26942218545ab3bca0636_4.png)

---

上一节我们介绍了新媒体运营的基础概念，本节中我们来看看平台如何通过标签系统来理解用户和创作者。

首先，我们通过一个模拟数据图来理解**用户标签**是如何产生的。假设你是一名平台用户，平台会监测你的所有行为。

![](img/a60efb2c9bc26942218545ab3bca0636_6.png)

![](img/a60efb2c9bc26942218545ab3bca0636_8.png)

![](img/a60efb2c9bc26942218545ab3bca0636_0.png)

![](img/a60efb2c9bc26942218545ab3bca0636_10.png)

![](img/a60efb2c9bc26942218545ab3bca0636_12.png)

平台会根据你的行为数据，为你绘制一个兴趣画像。这个过程主要依据以下三个核心数据指标：

![](img/a60efb2c9bc26942218545ab3bca0636_14.png)

以下是判断用户兴趣的三个关键行为指标：

![](img/a60efb2c9bc26942218545ab3bca0636_16.png)

1.  **点击率**：用户点击进入视频的概率。
2.  **完播率**：用户观看视频的时长比例，计算公式可以简化为：
    `完播率 = 视频播放完成次数 / 视频总播放次数`
3.  **互动行为**：包括点赞、评论、收藏、转发等（**关注**行为较为特殊，我们稍后会单独说明）。

![](img/a60efb2c9bc26942218545ab3bca0636_18.png)

例如，平台向你均匀推荐了15条视频：5条美食、5条美女、5条科技。如果你点击了全部5条美食视频、2条美女视频和1条科技视频，平台通过算法分析后，会认为你对“美食”内容最感兴趣。为了留住你，平台后续会为你推荐更多美食相关的内容，从而让你在平台上停留更久，创造消费或互动的可能性。

![](img/a60efb2c9bc26942218545ab3bca0636_20.png)

反之，如果向你推荐了5条母婴视频，你一次都没有点击，平台则会判断你不喜欢此类内容，并减少相关推送。这就是所谓的“信息茧房”效应，平台通过精细化标签实现个性化内容分发。

![](img/a60efb2c9bc26942218545ab3bca0636_22.png)

![](img/a60efb2c9bc26942218545ab3bca0636_24.png)

![](img/a60efb2c9bc26942218545ab3bca0636_26.png)

理解了平台如何为用户打标签后，我们作为内容创作者，最关心的是**创作者标签**。

![](img/a60efb2c9bc26942218545ab3bca0636_30.png)

![](img/a60efb2c9bc26942218545ab3bca0636_32.png)

很多人误以为，只要多发某个领域的视频，平台就会自动认定你是该领域的博主。实际上，平台判断创作者领域的依据，同样是基于**账号标签**。这个标签的形成，依赖于你发布内容所吸引来的用户群体的**整体数据反馈**。

![](img/a60efb2c9bc26942218545ab3bca0636_34.png)

![](img/a60efb2c9bc26942218545ab3bca0636_36.png)

这里存在一个关键矛盾：**账号内容垂直度与平台推荐精准度的矛盾**。

![](img/a60efb2c9bc26942218545ab3bca0636_38.png)

![](img/a60efb2c9bc26942218545ab3bca0636_40.png)

举个例子：假设你是一个母婴博主，但你的账号内容混杂，既发母婴内容，也发女装和日常生活视频。

![](img/a60efb2c9bc26942218545ab3bca0636_42.png)

![](img/a60efb2c9bc26942218545ab3bca0636_44.png)

当你发布一条女装视频时，平台的推荐逻辑是：
1.  将视频主要推送给之前与你母婴内容互动过的用户（因为他们是你账号的初始标签受众）。
2.  同时，尝试匹配一部分对“女装”感兴趣的新用户。

![](img/a60efb2c9bc26942218545ab3bca0636_46.png)

这时就会出现数据“拉扯”：原有的母婴用户对女装视频不感兴趣，导致**点击率、完播率、互动数据**大幅下降。即便有少数女装用户喜欢你的视频，但负面数据占比更高，平台算法会整体判定这条视频内容不佳，从而减少推荐。

![](img/a60efb2c9bc26942218545ab3bca0636_48.png)

长此以往，平台无法将你的账号精准归类，会认为你的账号内容杂乱、不够专业，从而影响所有内容的初始流量推送。

![](img/a60efb2c9bc26942218545ab3bca0636_50.png)

![](img/a60efb2c9bc26942218545ab3bca0636_52.png)

此外，我们需要单独理解 **“关注”** 这个行为。关注不同于普通的点赞评论，它代表用户认为你的账号有**长期价值**。

![](img/a60efb2c9bc26942218545ab3bca0636_56.png)

![](img/a60efb2c9bc26942218545ab3bca0636_58.png)

用户关注一个母婴博主，是期望持续获得专业的母婴知识和好物推荐。如果你的账号内容垂直度低，用户无法预测你下次会更新什么，那么关注你的价值点就消失了，这直接影响粉丝的忠诚度和增长。

![](img/a60efb2c9bc26942218545ab3bca0636_60.png)

因此，作为创作者，我们必须明白：

![](img/a60efb2c9bc26942218545ab3bca0636_62.png)

![](img/a60efb2c9bc26942218545ab3bca0636_64.png)

**平台标签分为“用户标签”和“创作者标签”。我们必须保证自己的“创作者标签”清晰、单一且始终如一。**

![](img/a60efb2c9bc26942218545ab3bca0636_66.png)

本节课中我们一起学习了用户标签与创作者标签的核心逻辑。简单总结如下：

![](img/a60efb2c9bc26942218545ab3bca0636_72.png)

1.  **用户标签**是平台通过分析用户的点击率、完播率和互动行为，为其绘制的兴趣画像，用于实现个性化内容推荐。
2.  **创作者标签**是平台通过分析创作者内容所吸引用户的整体数据反馈，为账号打上的领域标识。
3.  创作者必须保持内容**垂直**，确保账号标签清晰唯一，才能让平台将你识别为该领域的“专家”，从而将你的内容精准推送给感兴趣的用户，获得更好的流量和粉丝增长。

![](img/a60efb2c9bc26942218545ab3bca0636_74.png)

下一节，我们将探讨如何通过策划关键词、标题和内容文案，让系统在短时间内快速识别并确认我们的“专家”身份，为新账号成功打上精准的创作者标签。
