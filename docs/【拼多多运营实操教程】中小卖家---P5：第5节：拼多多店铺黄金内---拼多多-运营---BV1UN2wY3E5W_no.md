## 三、标题关键词的选取与组合 🔑

![](img/5bd1c4c4ffb5efb0203465da78a14ff7_29.png)

标题的核心在于关键词的选取。这里介绍两种选词方法。

![](img/5bd1c4c4ffb5efb0203465da78a14ff7_30.png)

**方法一：利用买家版搜索下拉框**
1.  打开拼多多买家版APP。
2.  在搜索框输入产品的**核心关键词**。
3.  查看搜索下拉框推荐的关联词，这些都是近期热搜词。
4.  选取与产品匹配的关键词备用。

![](img/5bd1c4c4ffb5efb0203465da78a14ff7_32.png)

![](img/5bd1c4c4ffb5efb0203465da78a14ff7_33.png)

**方法二：利用商家后台“搜索词分析”工具**
1.  登录拼多多商家后台。
2.  进入 **推广平台 > 推广工具 > 搜索词分析**。
3.  选择与产品对应的**一级、二级、三级类目**。
4.  查看系统推荐的**飙升词**，或主动搜索关键词，查看其**搜索指数、点击指数、竞争指数**。
5.  选取数据高且与产品**风格、属性完全匹配**的关键词。这类词转化能力更强，流量更精准。

**标题组合公式**
将选取的关键词组合成通顺的30字标题（不含违禁词）。可参考以下公式：
*   `品牌/型号 + 商品核心词`
*   `促销词/特点形容词 + 商品核心词`
*   `地域特点词 + 品牌词 + 商品核心词`

## 四、产品属性的填写 📝

![](img/5bd1c4c4ffb5efb0203465da78a14ff7_39.png)

![](img/5bd1c4c4ffb5efb0203465da78a14ff7_40.png)

属性填写关乎商品在搜索和推荐页的曝光。

1.  带`*`号的为**重要属性**，必须准确填写。
2.  不带`*`号的属性，也建议**尽可能填写准确、详细**。
3.  填写越准确、越详细，越有利于商品获得搜索和推荐流量。

## 五、商品视频的上传 🎬

视频能极大提升转化率，建议尽可能上传。

![](img/5bd1c4c4ffb5efb0203465da78a14ff7_44.png)

**商品视频**：展示商品整体，时长60秒内，比例1:1、16:9或3:4。
**商品讲解视频**：详细讲解商品，建议时长1分钟内，比例9:16。
**商详视频**：展示在商品详情页顶部，比例建议3:4。

## 七、白底图与SKU布局 💎

![](img/5bd1c4c4ffb5efb0203465da78a14ff7_51.png)

**白底图**：必须上传，影响活动报名和类目流量。

![](img/5bd1c4c4ffb5efb0203465da78a14ff7_52.png)

![](img/5bd1c4c4ffb5efb0203465da78a14ff7_54.png)

**SKU布局**是内功重点，涉及规格命名和定价技巧。

**SKU规格命名**：目的是防止平台与低价同款比价。命名应详细、独特，可参考以下格式：
*   `数量 + 规格`
*   `颜色 + 款式 + 功能`
*   `尺码 + 数量组合 + 颜色`
*   `颜色 + 款式 + 组合推荐词`
*   `款式 + 详情 + 组合内容`

![](img/5bd1c4c4ffb5efb0203465da78a14ff7_56.png)

**SKU图片处理**：可采用“图中图”方式防比价。即用制图软件将产品抠成透明图，再叠加到新的背景图上。

![](img/5bd1c4c4ffb5efb0203465da78a14ff7_57.png)

**SKU定价公式**：
```
售卖价 = 拼单价 × 2 + (5到10元优惠)
```
此公式预留了利润空间，附加的5-10元优惠可用于设置优惠券等营销活动，为后期调价和促销留有余地。

## 八、其他重要设置 ⚙️

![](img/5bd1c4c4ffb5efb0203465da78a14ff7_60.png)

![](img/5bd1c4c4ffb5efb0203465da78a14ff7_61.png)

**满减折扣**：这是平台强制功能，无法关闭。发布时建议将其设置为**9.9折**等高折扣，避免因默认低折扣导致亏损。

![](img/5bd1c4c4ffb5efb0203465da78a14ff7_62.png)

![](img/5bd1c4c4ffb5efb0203465da78a14ff7_63.png)

**发货设置**：发货时间建议设置为**48小时**，并设置好运费模板，避免因延迟发货被平台处罚。
