# 抖音运营实战：P16：制造情绪冲突的十种文案角度

在本节课中，我们将学习如何通过制造情绪冲突来提升抖音文案的吸引力。我们将探讨不同的文案表达方式如何影响用户心理，并介绍一种实用的选题方法，帮助你的内容脱颖而出。

## 选题与表达方式的重要性

上一节我们介绍了选题的重要性。选好题之后，我们才能进行脚本结构设计和拍摄。本节中我们来看看，同样一句话，用不同的方式表达，效果会截然不同。

例如，如果我们想让一个人去健身瘦身，直接说“来健身房，变成更美的自己”，这句话就显得无关痛痒，像一句普通的广告词。

但如果我们换一种说法：“如果你不去健身房，你就失去了变美的机会”。这种利用“失去心理”和“恐吓心理”的表述，会在用户心中形成一种紧迫感。

这个原理同样适用于其他产品。卖面膜时说“敷面膜，永远保持20岁”，也是一句干巴巴的广告语。如果说“不护肤、不敷面膜，你就会长鱼尾纹、长斑，这些斑是永久性的”，用这种心理去讲述，用户可能会更重视。

**核心概念**：同样的选题，用不同的话术或文案表达，给用户的心理暗示是不同的。

## 制造情绪冲突的心理学原理

这种差异的根源在于制造了“情绪冲突”。它利用了人类的心理弱点。

例如，双十一抢购时，商家说“今天买我的东西打五折”。如果再加上一个刺激：“这个五折优惠仅剩两个名额了”，你就会立刻下单。这两种刺激造成的紧迫感完全不同。

主播带货也是同样的道理。通过制造“马上要没了”、“最后机会”的冲突感，可以极大地促进销售。

因此，回想一下普通的文案，比如“如何科学地健身”，它同样是无关痛痒的。用户可能想看，但缺乏去实操的动力。

如果我们换一种说法：“如果你不科学健身，会造成XX后果（如因长期久坐导致腰间盘突出）”。虽然例子可能有些夸张，但道理是相通的。

**核心结论**：同样的选题，用不同的方式和心理战术说出来，给用户造成的情感冲突和情绪冲击是不一样的。我们的目标就是**制造情绪的冲突**。

## 情绪冲突的模板与注意事项

制造情绪冲突有一些模板可以使用。需要说明的是，前面几种模板效果可能不是特别好，但某些行业仍可适用。而“恐惧心理”和“失去心理”是经常使用的有效手段。

**重要提示**：即使是有效的心理战术，也不宜过于频繁使用。如果每一条视频都制造恐惧，会给用户带来不好的心理感受，甚至引起反感。

这个方法可以应用在你的任何视频中。例如，健身领域可以塑造“一时身体的疼痛”与“终身的疼痛”之间的冲突。

大家可以学习优秀广告公司（如奥美）的文案，它们深谙此道。

## 九宫格选题法实践

大家需要记录和学会的是“九宫格选题法”。

![](img/5089632914bb9de7d995e174f3c7e8dd_3.png)

以下是使用该方法的具体步骤：
1.  你可以将任何不同的产品和目标用户代入九宫格。
2.  围绕中心主题，将相关元素延伸开来。
3.  利用“连连看”的方式，将不同格子的元素进行组合，从而生成一个独特的选题。
