# 视频号运营教程：P21：1. 如何自学剪映剪辑 🎬

在本节课中，我们将学习如何从零开始，利用官方和第三方资源高效自学剪映软件，掌握视频剪辑的基础功能。

短视频普及多年，许多人已具备一定的剪辑基础。对于从未使用过剪映的初学者，首先需要在手机应用商店下载“剪映”App。它是抖音官方的剪辑软件，功能强大且大部分免费，足以满足日常创作需求，无需开通会员。

上一节我们提到了获取软件，本节中我们来看看如何找到并利用官方教程进行自学。

## 利用官方“创作者课堂”

打开剪映App后，在首页最下方可以找到“创作者课堂”入口。



![](img/8e5b67057451720b2f8a418889c20fe0_1.png)

![](img/8e5b67057451720b2f8a418889c20fe0_1.png)

进入后，在搜索框中输入“剪映小助手”进行搜索。在搜索结果中，选择“用户”标签，排在第一位的便是剪映官方账号。



![](img/8e5b67057451720b2f8a418889c20fe0_3.png)

![](img/8e5b67057451720b2f8a418889c20fe0_3.png)

关注这个官方账号。其主页提供模板、课程和素材。我们重点学习“课程”部分。



![](img/8e5b67057451720b2f8a418889c20fe0_5.png)

![](img/8e5b67057451720b2f8a418889c20fe0_5.png)

以下是推荐学习的免费课程：
*   **剪映功能大全**：共87节课。
*   **新手常见问题**：共10节课。

这些课程每节时长很短，多为几十秒到两分钟左右。完整学完所有课程大约需要两小时，即可实现从零基础到掌握专业操作。对于没有剪辑基础的同学，务必先系统学完这些基础功能再进行实操。



![](img/8e5b67057451720b2f8a418889c20fe0_7.png)

![](img/8e5b67057451720b2f8a418889c20fe0_7.png)

点开任意课程视频，在播放界面下方可以看到相关的二级工具菜单。



![](img/8e5b67057451720b2f8a418889c20fe0_9.png)

![](img/8e5b67057451720b2f8a418889c20fe0_9.png)

![](img/8e5b67057451720b2f8a418889c20fe0_11.png)

课程视频时长确实很短，便于利用碎片时间学习。



![](img/8e5b67057451720b2f8a418889c20fe0_11.png)

通过点击“加载更多”可以观看全部课程。学完这些，你就能掌握剪映的基本使用方法。



![](img/8e5b67057451720b2f8a418889c20fe0_13.png)

![](img/8e5b67057451720b2f8a418889c20fe0_13.png)

掌握了官方学习路径后，我们还可以在抖音上寻找更多优质、精炼的教程作为补充。

例如，可以关注一位名为“妮妮”的创作者。她的视频教程比较简短系统。



![](img/8e5b67057451720b2f8a418889c20fe0_17.png)

![](img/8e5b67057451720b2f8a418889c20fe0_19.png)
