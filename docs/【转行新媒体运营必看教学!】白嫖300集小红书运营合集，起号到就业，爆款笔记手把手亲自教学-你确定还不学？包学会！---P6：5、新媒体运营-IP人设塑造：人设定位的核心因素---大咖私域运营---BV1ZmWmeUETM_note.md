### 1. 角色：明确你的专业身份

![](img/61356e700d732006cd57d1f8f917105a_68.png)

角色，简单来说，就是你在特定赛道中希望呈现的专业身份。这与你选择的内容方向紧密相关。

![](img/61356e700d732006cd57d1f8f917105a_78.png)

![](img/61356e700d732006cd57d1f8f917105a_80.png)

![](img/61356e700d732006cd57d1f8f917105a_82.png)

![](img/61356e700d732006cd57d1f8f917105a_84.png)

![](img/61356e700d732006cd57d1f8f917105a_86.png)

*   **核心公式**：**角色 = 赛道方向 + 专业身份**
*   **举例说明**：
    *   **装修博主**：你需要展示自己确实有房子在装修或已装修完成，通过记录过程、分享避坑经验、展示前后对比来建立专业性和可信度。
    *   **母婴赛道**：你的角色可以是“宝妈”，也可以是“育儿专家”或“儿童护理顾问”。后者不一定需要亲身经历，但需要具备该领域的专业知识。
    *   **情感赛道**：你可以是“情感导师”、“心理学顾问”，也可以是分享自身成长经历的“成长型人设”。例如，一位30岁女性分享自己如何处理家庭、职场、创业中的问题，并最终走出困境的经历。

### 2. 内容：保持垂直与价值

![](img/61356e700d732006cd57d1f8f917105a_108.png)

内容是人设的基石。关于内容定位，我们之前强调过要吸引用户、提供价值并具备特色。在人设塑造中，内容的关键在于**垂直和持续**。

![](img/61356e700d732006cd57d1f8f917105a_118.png)

![](img/61356e700d732006cd57d1f8f917105a_120.png)

![](img/61356e700d732006cd57d1f8f917105a_122.png)

![](img/61356e700d732006cd57d1f8f917105a_124.png)

![](img/61356e700d732006cd57d1f8f917105a_126.png)

*   **核心逻辑**：**内容垂直度 → 账号标签 → 平台流量分发**
*   **重要性**：平台通过你的内容为你打上“标签”，并将内容分发给可能感兴趣的用户。如果你的内容今天讲育儿，明天讲宠物，后天发Vlog，系统就无法为你建立清晰的标签。更糟糕的是，之前对你A内容感兴趣的用户，可能对你B内容不感兴趣而划走，这会向平台传递“内容不受欢迎”的负面信号，从而影响后续流量。
*   **行动指南**：确定赛道后，持续提供该领域的相关内容，这是建立稳固人设和获得稳定流量的基础。

以下是形式需要考虑的几个方面：

![](img/61356e700d732006cd57d1f8f917105a_160.png)

![](img/61356e700d732006cd57d1f8f917105a_162.png)

![](img/61356e700d732006cd57d1f8f917105a_164.png)

![](img/61356e700d732006cd57d1f8f917105a_166.png)

![](img/61356e700d732006cd57d1f8f917105a_168.png)

*   **内容产出场景**：你是在固定的室内场景（如书房、工作室）拍摄，还是在多变的室外场景（如通勤路上、咖啡厅）分享？场景的选择可以强化人设氛围。
*   **出镜与声音**：
    *   **真人出镜+原声**：最具亲和力和真实感。
    *   **不出镜+配音**：适合不想露脸或声音条件不佳的创作者，可用剪辑软件（如**剪映**）的配音功能实现。
    *   **多人出镜+配音**：例如朋友对话场景，实际用配音呈现，增加戏剧效果。
*   **内容形式**：你的内容属于以下哪种类型？
    *   **剧情型**：创作难度较高，需要剧本和表演。
    *   **干货/经验分享型**：需要精心准备文案脚本，输出有价值的信息。
    *   **生活记录型**：需要积累大量日常拍摄素材，真实感强。

### 4. 记忆点：让用户记住你的关键

![](img/61356e700d732006cd57d1f8f917105a_192.png)

![](img/61356e700d732006cd57d1f8f917105a_194.png)

记忆点是让人设脱颖而出、被用户深刻记住的“钩子”。它需要通过设计，并在内容中反复强化。

![](img/61356e700d732006cd57d1f8f917105a_202.png)

![](img/61356e700d732006cd57d1f8f917105a_204.png)

以下是打造记忆点的常见方法：

![](img/61356e700d732006cd57d1f8f917105a_212.png)

![](img/61356e700d732006cd57d1f8f917105a_214.png)

![](img/61356e700d732006cd57d1f8f917105a_216.png)

![](img/61356e700d732006cd57d1f8f917105a_218.png)

*   **视频开头固定化**（前置强化）：
    *   **固定动作**：如一个标志性的手势或开场姿势。
    *   **固定特效**：每次视频开头使用的相同视觉特效。
    *   **固定开场白**：一句标志性的口头禅或问候语。
    *   **设计原理**：将记忆点集中在视频前几秒，能在用户每次刷到你时形成强烈的重复冲击，快速加深印象。
*   **贯穿视频始终**（全局强化）：
    *   **统一BGM**：使用同一首或同一风格的音乐作为背景，音乐响起即能唤起用户对你的记忆。
    *   **标志性物品/服装**：在视频中始终出现的某个物品（如特定的台灯、玩偶），或统一风格的着装（如职场精英的西装、田园博主的特定服饰）。

![](img/61356e700d732006cd57d1f8f917105a_220.png)

![](img/61356e700d732006cd57d1f8f917105a_222.png)

![](img/61356e700d732006cd57d1f8f917105a_224.png)

![](img/61356e700d732006cd57d1f8f917105a_226.png)

![](img/61356e700d732006cd57d1f8f917105a_228.png)

**记忆点设计公式**：**强记忆人设 = 前置固定元素（动作/特效/开场白） + 全局统一元素（BGM/物品/服装）**

本节课我们一起学习了人设定位的四大核心因素：**角色**、**内容**、**形式**和**记忆点**。

![](img/61356e700d732006cd57d1f8f917105a_262.png)

![](img/61356e700d732006cd57d1f8f917105a_264.png)

1.  **角色**是你在赛道中的专业身份，需与内容方向匹配。
2.  **内容**必须保持垂直，持续输出同一领域有价值的信息，以建立清晰的账号标签。
3.  **形式**决定了内容的呈现方式，需根据自身情况和内容类型选择最合适的出镜、声音和内容形式。
4.  **记忆点**是让人设被记住的关键，需要通过设计固定的开头元素和贯穿全局的标识来反复强化。
