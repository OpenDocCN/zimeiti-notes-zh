# 小红书电商运营：P10：06.小红书后台的基础操作（电脑端） 💻

在本节课中，我们将学习如何通过电脑端操作小红书商家后台，涵盖保证金、支付方式开通、运费保设置及地址管理等核心功能。这些是店铺正常运营的基础设置。

## 保证金管理 💰

上一节我们介绍了店铺定位，本节中我们来看看店铺的基础财务设置。首先讲解保证金。

保证金是开店时向平台缴纳的一笔信用担保资金。前期可以不缴纳，但若想长期经营或开通更多支付方式，则建议缴纳。

![](img/1c86715732b95a130b40b64e097b2a4d_5.png)

**保证金金额公式**：
```
个人店保证金 ≈ 1000元
企业店保证金 ≈ 2000元 - 20000元
```
具体金额根据所选经营类目而定，特殊类目（如二手奢侈品、珠宝玉石）的保证金可能更高，且个人店可能无法经营。

![](img/1c86715732b95a130b40b64e097b2a4d_7.png)

操作路径为：登录 **小红书千帆后台** -> 点击“资金” -> 进入“店铺保证金管理”。在此页面可以充值、查询待补缴金额或申请退款。

![](img/1c86715732b95a130b40b64e097b2a4d_9.png)

保证金在店铺关闭且无违规、无售后纠纷的情况下可全额退还。若存在未处理的客户退款等问题，平台会从中扣除相应款项。

![](img/1c86715732b95a130b40b64e097b2a4d_11.png)

## 支付方式开通 🔗

了解保证金后，接下来需要设置收款渠道，即开通支付方式。主要支持支付宝和微信支付。

![](img/1c86715732b95a130b40b64e097b2a4d_13.png)

**重要规则**：
*   缴纳保证金是开通**微信支付**的前提条件。
*   **支付宝**支付开通后立即生效。
*   **微信支付**需在缴纳保证金的**次日**才能设置。
*   提现金额将进入店铺注册时使用的法人账户。

操作路径为：在“资金”板块下，找到“支付设置”，分别按指引开通支付宝与微信支付。开通时需按要求绑定对应账户（个人店为身份证，个体/企业店需营业执照）。

![](img/1c86715732b95a130b40b64e097b2a4d_15.png)

![](img/1c86715732b95a130b40b64e097b2a4d_17.png)

## 运费保设置 🛡️

开通支付方式能让顾客付款，而设置运费保则能提升顾客下单意愿，降低购物顾虑。

运费保是一项平台服务，当消费者发起“七天无理由退货”时，系统会自动补贴退货运费。这能有效提升订单转化率并减少售后纠纷。

**开通建议**：推荐所有商家开通此服务。

![](img/1c86715732b95a130b40b64e097b2a4d_19.png)

操作路径为：进入“售后”管理板块 -> 找到“运费保” -> 点击“开通全店运费保服务”。

![](img/1c86715732b95a130b40b64e097b2a4d_21.png)

![](img/1c86715732b95a130b40b64e097b2a4d_23.png)

## 地址与物流设置 📦

![](img/1c86715732b95a130b40b64e097b2a4d_25.png)

最后，我们来设置发货与退货地址，这是完成订单履约的关键一步。

![](img/1c86715732b95a130b40b64e097b2a4d_27.png)

对于采用“一件代发”模式的无货源商家，退货地址建议填写供应商所在地，或在地址后备注“退货前请联系客服”。对于有自有仓库的商家，则填写真实发货地址即可。

![](img/1c86715732b95a130b40b64e097b2a4d_29.png)

操作路径为：进入“订单” -> “物流工具” -> “地址库”，点击“新建地址”进行设置。在此还可以设置不同区域的运费模板。

**本节课总结**：我们一起学习了小红书电脑端后台的四个基础操作：**保证金**的缴纳与查询、**支付方式**的开通、**运费保**服务的启用，以及**发货/退货地址**的设置。完成这些配置，你的店铺就具备了接收订单、处理资金与完成发货的基础能力。
