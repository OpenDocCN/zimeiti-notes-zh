# 新媒体运营系统课：P12：7.视频剪辑实操（下）

在本节课中，我们将继续学习视频剪辑的核心操作，重点掌握如何为视频片段添加转场、调整比例以及添加字幕。这些步骤对于提升视频的流畅度、观感和完播率至关重要。

上一节我们介绍了视频素材的导入与基础剪辑，本节中我们来看看如何通过转场和字幕让视频变得更专业。

## 添加视频转场

![](img/89e870abcc7b4ccbf166ed4739eff579_1.png)

我们的视频通常由多个10秒左右的片段组成。这些片段之间需要有自然的衔接，否则观众在切换内容时会感到突兀，可能导致他们退出视频，从而**严重影响视频的完播率**。

![](img/89e870abcc7b4ccbf166ed4739eff579_3.png)

**核心公式**：`高完播率 = 更高的平台推荐权重与收益`

因此，为每个片段之间添加转场是必要的步骤。转场能明确提示用户内容已切换，保持观看的连贯性。

以下是添加转场的具体步骤：

![](img/89e870abcc7b4ccbf166ed4739eff579_5.png)

![](img/89e870abcc7b4ccbf166ed4739eff579_6.png)

![](img/89e870abcc7b4ccbf166ed4739eff579_7.png)

1.  在视频轨道右侧，找到并点击 **“+”** 号按钮。
2.  在弹出的菜单中，选择 **“素材库”**。
3.  在素材库的 **“热门”** 分类中，可以找到大量现成的转场素材（例如搞笑的动画或过渡效果）。
4.  **关键点**：选择转场时，时长应控制在**3到5秒**。过长的转场（如16秒）会让观众失去耐心。
5.  选中一个合适的转场（例如3秒的），点击添加至轨道。

![](img/89e870abcc7b4ccbf166ed4739eff579_8.png)

![](img/89e870abcc7b4ccbf166ed4739eff579_9.png)

## 统一视频比例

添加的转场素材，必须与主视频保持相同的画面比例，即**16:9**的横屏比例，以确保视觉统一。

以下是统一比例的步骤：

1.  选中已添加的转场片段。
2.  在编辑菜单中找到 **“画布模糊”** 或 **“背景”** 选项。
3.  选择第一个清晰的模糊样式，系统会自动将转场背景填充为16:9的横屏。
4.  最后，点击右下角的 **“√”** 确认修改。

![](img/89e870abcc7b4ccbf166ed4739eff579_11.png)

![](img/89e870abcc7b4ccbf166ed4739eff579_12.png)

至此，视频的比例和转场设置就完成了。

## 添加视频字幕（文案）

字幕能帮助观众更好地理解内容，尤其在有旁白或对话的视频中必不可少。我们可以利用软件的自动识别功能快速添加。

![](img/89e870abcc7b4ccbf166ed4739eff579_14.png)

以下是添加字幕的步骤：

![](img/89e870abcc7b4ccbf166ed4739eff579_15.png)

1.  点击编辑界面左下角的 **“文字”** 选项。
2.  选择 **“新建文本”**，然后点击右侧的 **“识别字幕”** 功能。
3.  点击 **“开始匹配”**，软件会自动识别视频中的语音并生成字幕文本。
4.  字幕生成后，需要进行调整：
    *   拖动文本框右下角，将字幕**放大**到合适的尺寸。
    *   将整个字幕文本框拖动到视频画面的**最下方**，这是最舒适观看的位置。
5.  调整好第一句字幕后，后续字幕会自动应用相同的样式，非常方便。

## 核心要点总结

本节课中我们一起学习了视频剪辑后半部分的核心操作：

![](img/89e870abcc7b4ccbf166ed4739eff579_17.png)

![](img/89e870abcc7b4ccbf166ed4739eff579_18.png)

![](img/89e870abcc7b4ccbf166ed4739eff579_19.png)

1.  **添加转场**：在片段间插入3-5秒的转场素材，提升流畅度，保障完播率。
2.  **统一比例**：确保所有元素（包括转场）均为**16:9**横屏比例，使用“画布模糊”功能进行调整。
3.  **添加字幕**：使用“识别字幕”功能自动生成，并调整其大小和位置至画面底部。

![](img/89e870abcc7b4ccbf166ed4739eff579_20.png)

记住，剪辑时需严格遵循步骤：选择合适时长的素材、统一比例、添加转场与字幕。避免使用过长或比例不符的素材，这会影响平台的审核通过率。整个流程并不复杂，按部就班操作即可完成一个基础的视频作品。
