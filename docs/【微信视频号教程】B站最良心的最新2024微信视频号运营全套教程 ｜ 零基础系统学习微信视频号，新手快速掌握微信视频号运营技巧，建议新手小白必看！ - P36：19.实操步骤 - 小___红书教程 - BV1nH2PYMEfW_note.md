# 微信视频号运营教程：P36：19. 实操步骤 - 开通与上架商品橱窗

在本节课中，我们将学习如何为微信视频号开通商品橱窗功能，并完成商品上架。这是进行视频号带货的基础操作，步骤清晰，适合新手跟随操作。

上一节我们介绍了视频号运营的基础概念，本节中我们来看看如何具体开通带货功能。

## 第一步：进入微信视频号

以下是进入视频号界面的具体步骤。

1.  **打开微信**：在手机主屏幕找到并点击微信应用图标。
2.  **点击“发现”**：进入微信后，点击底部导航栏的“发现”选项。
3.  **进入“视频号”**：在“发现”页面，找到并点击“视频号”入口（图标为黄色蝴蝶形状）。

![](img/531f9039f7cf42e2970f34a783813792_1.png)

![](img/531f9039f7cf42e2970f34a783813792_1.png)

![](img/531f9039f7cf42e2970f34a783813792_2.png)

![](img/531f9039f7cf42e2970f34a783813792_2.png)

## 第二步：找到创作者中心

![](img/531f9039f7cf42e2970f34a783813792_4.png)

成功进入视频号后，下一步是找到管理后台。

![](img/531f9039f7cf42e2970f34a783813792_6.png)

4.  **点击个人头像**：在视频号主页的右上角，点击个人头像图标。
5.  **进入“创作者中心”**：在打开的页面中向下滑动，找到并点击“创作者中心”。（注：新注册账号需发布至少一条视频后才会显示此入口）。

![](img/531f9039f7cf42e2970f34a783813792_4.png)

![](img/531f9039f7cf42e2970f34a783813792_6.png)

![](img/531f9039f7cf42e2970f34a783813792_8.png)

## 第三步：开通商品橱窗功能

![](img/531f9039f7cf42e2970f34a783813792_9.png)

在创作者中心，我们可以进行带货相关的设置。

6.  **点击“带货中心”**：在创作者中心页面，找到类似小卖部图标的“带货中心”并点击。

![](img/531f9039f7cf42e2970f34a783813792_8.png)

![](img/531f9039f7cf42e2970f34a783813792_9.png)

接下来，系统会根据你的账号状态引导操作。如果未开通，点击后会进入开通流程；如果已开通，则会进入商品上架界面。

以下是开通橱窗权限的流程：

![](img/531f9039f7cf42e2970f34a783813792_11.png)

7.  **选择带货身份**：在开通界面，系统会要求选择身份。请选择 **“我是达人”** 选项。即使你是厂家，初期也建议先以达人身份开通。
8.  **缴纳保证金**：选择身份后，会跳转到保证金缴纳页面。点击 **“去交纳”**，通过微信支付100元保证金即可完成开通。

![](img/531f9039f7cf42e2970f34a783813792_11.png)

![](img/531f9039f7cf42e2970f34a783813792_13.png)

**核心操作总结**：
```plaintext
路径：微信 -> 发现 -> 视频号 -> 个人头像 -> 创作者中心 -> 带货中心
关键选择：身份选择 -> “我是达人”
关键动作：缴纳保证金 -> 100元
```

## 第四步：上架商品（功能开通后）

![](img/531f9039f7cf42e2970f34a783813792_13.png)

开通橱窗权限后，再次点击“带货中心”，界面将变为商品选品和上架页面。

![](img/531f9039f7cf42e2970f34a783813792_15.png)

在此页面，你可以：
*   从视频号官方商品库（选品中心）挑选商品。
*   添加自有商品（需具备相应资质）。
*   对已添加的商品进行管理，并关联到你的视频或直播中进行推广。

![](img/531f9039f7cf42e2970f34a783813792_15.png)

本节课中我们一起学习了微信视频号商品橱窗从开通到上架的完整实操流程。我们首先进入了视频号的创作者中心，接着完成了以“达人”身份开通橱窗并缴纳保证金的关键步骤，最后了解了开通后如何上架商品。掌握这些步骤，你就具备了在视频号进行带货的基础能力。
