# 小红书运营教程：P5：小红书笔记不被收录的原因分析 🔍

在本节课中，我们将要学习小红书笔记发布后不被平台收录的常见原因。了解这些原因，可以帮助你有效规避问题，确保你的内容能够顺利进入平台流量池。

上一节我们介绍了笔记发布的基本流程，本节中我们来看看哪些因素会导致笔记无法被收录。以下是五个核心原因。

## 1. 存在违规推广行为
平台禁止硬性广告。笔记内容若包含直接的品牌名称、产品链接或过于明显的推销话术，会被判定为违规推广，从而不被收录。

## 2. 含有平台敏感词
你的笔记或账号历史中如果存在违规内容，即使笔记暂时被收录，后续也可能被系统剔除。平台会对内容进行敏感词检测。

以下是需要注意的敏感词类型：
*   **政治敏感类**：例如破坏民族团结、谈论敏感政治话题等。
*   **通用违规类**：包括不文明词汇、诱导性言论、色情暴力等。
*   **谐音变体类**：使用谐音、拼音等方式规避检测的违禁词。

**核心原则**：我们通过网络项目进行合法营销以获取收入，必须严格遵守法律法规，绝不触碰原则性违禁内容。

## 3. 内容本身违规（图片与文字）
内容违规可能直接导致笔记不被收录，也可能在收录后因被举报而限流。这分为图片和文字两部分。

内容违规主要包括：
*   **图片违规**：避免带有其他平台的水印或二维码。这种引流方式已不适合小红书。
*   **文字违规**：严禁出现前文所述的各类敏感词、违规词。

## 4. 笔记发布频率异常
发送文章频率过快、过高，可能导致账号行为异常。系统会判定该账号可能是机器人操作，从而不收录其发布的笔记。

## 5. 发布设备已被平台拉黑
如果你使用的手机设备曾有过严重违规记录，即使更换了新账号，在该设备上发布的内容也可能不被收录。

**原理说明**：当APP安装到手机时，会读取设备的硬件参数（如IMEI码、手机号等）。如果该设备曾被标记为违规，那么后续在此设备上登录的任何账号都可能受到连带影响。公式可以简单理解为：
`设备黑名单状态 = 关联(设备硬件信息， 历史违规记录)`

---

本节课中我们一起学习了导致小红书笔记不被收录的五大主要原因：**违规推广**、**敏感词**、**内容违规**、**发布频率异常**以及**设备被拉黑**。检查并规避这些问题，能显著提高你笔记的收录率，为获得更多曝光打下基础。
