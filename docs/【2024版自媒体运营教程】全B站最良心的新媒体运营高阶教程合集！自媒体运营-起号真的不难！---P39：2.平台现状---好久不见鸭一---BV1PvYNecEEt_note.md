# 自媒体运营教程：P39：2.平台现状 📊

在本节课中，我们将要学习自媒体行业的平台现状。我们将了解国家政策对行业的支持、平台提供的资金补贴以及行业的用户规模，从而判断进入自媒体行业的可行性。

上一节我们介绍了什么是自媒体，本节中我们来看看这个行业的平台现状如何，以及它是否值得去做。

## 国家政策支持 📜

现阶段社会氛围较为浮躁，许多人都想赚钱。过去有人采取非法手段牟利，例如前往缅甸等地的新闻，这些行为已被国家明令禁止，对我们毫无益处。

现阶段，自媒体行业属于国家支持的范畴。国家已将支持文件下达到各个地方政府。

![](img/dea7c1106b2a1d2dd1d269ccd4478e36_1.png)

这些文件都强调了要坚持正确导向，内容为王。这意味着，未来作为自媒体创作者，我们创作的**文章和视频**才是结算收益最重要的方式和途径。

听到这里，你认为在国家的明确支持下，自媒体行业可以进入吗？答案是肯定的。

## 平台资金补贴 💰

既然有了政策性文件支持我们创作，自媒体平台也需要响应国家号召，出资鼓励视频创作。那么平台会出多少钱呢？

通过以下视频可以了解平台的具体补贴计划。

![](img/dea7c1106b2a1d2dd1d269ccd4478e36_5.png)

视频中，平台创始人宣布：**未来一年将额外拿出20亿元补贴中视频创作人**。需要强调两点：
1.  这是**真金白银的20亿元**，不包含任何商单、直播和电商收入，是纯补贴。
2.  平台将探索保底加分成的模式，助力创作人职业化，并且**20亿元只是保底，上不封顶**。

![](img/dea7c1106b2a1d2dd1d269ccd4478e36_7.png)

从视频中我们了解到，平台提供了巨额资金支持。这20亿元纯补贴，就是我们未来通过创作视频可以直接赚取的收益之一。

## 行业用户规模 👥

除了政策和资金，现阶段自媒体行业还有一个显著优势：庞大的用户数量。

根据2022年9月9日中国研究生网站发布的行业报告，我国现阶段**网民规模已达9.4亿人**，接近10亿。这9.4亿人就是拥有手机和视频平台、每天都会刷视频的潜在观众。

有如此多的看视频的人，做视频的人有多少呢？报告显示，截止去年：
*   全职从事自媒体的人数为 **370万**
*   兼职从事自媒体的人数为 **600万**

两者相加，从事视频创作的人数约为**970万**，我们可简化为1000万。

这就形成了一个比例：**大约1000万人制作视频，供9.4亿人观看**。相当于每100个观众中，只有1个内容创作者。试想一下，一个人制作的视频，足够100个人看吗？一个人创作视频的速度，能赶上100个人观看视频的速度吗？答案显然是不够的。

这正是国家支持自媒体行业发展的重要原因。我们国家未来的发展趋势在互联网领域显而易见：从我们小时候看的黑白电视，到长大后的彩电，再到如今家家户户的电视都能连接WIFI。未来，我们制作的视频甚至可以在家庭电视上播放，可以想象其用户规模将是多么庞大。

![](img/dea7c1106b2a1d2dd1d269ccd4478e36_9.png)

本节课中我们一起学习了自媒体行业的平台现状。我们了解到：**国家已出台政策明确支持行业发展**；**主流平台提供巨额真金白银的补贴**以鼓励创作；并且行业拥有**近10亿的庞大用户基础**，而创作者相对稀缺，市场前景广阔。这些现状共同表明，自媒体是一个值得进入且充满机会的领域。
