# 视频号运营教程：P31：19.实操步骤 - 开通与上架商品

在本节课中，我们将学习视频号运营中一个非常关键的实操环节：如何开通视频号的商品橱窗功能，并完成商品上架。这是实现视频号带货变现的基础步骤，操作过程清晰简单，我们将一步步进行拆解。

上一节我们介绍了视频号运营的整体框架，本节中我们来看看如何具体开通带货权限。

## 第一步：进入视频号入口

![](img/5d52b365268b25975bffb5f27c36b13c_1.png)

首先，我们需要打开手机微信应用。



![](img/5d52b365268b25975bffb5f27c36b13c_2.png)

接下来，在微信的“发现”页面，找到并点击“视频号”入口。该入口通常位于页面顶部，图标为黄色的蝴蝶形状。



![](img/5d52b365268b25975bffb5f27c36b13c_4.png)

## 第二步：进入个人中心

![](img/5d52b365268b25975bffb5f27c36b13c_6.png)

点击进入视频号后，在页面的右上角可以找到一个“人形头像”图标，点击此处即可进入个人中心。



## 第三步：找到创作者中心

![](img/5d52b365268b25975bffb5f27c36b13c_8.png)

进入个人中心后，向下滑动页面，在最底部找到“创作者中心”选项并点击进入。如果未找到此选项，可能是因为账号尚未发布过作品，发布一条作品后即可出现。



![](img/5d52b365268b25975bffb5f27c36b13c_9.png)

## 第四步：进入带货中心

在创作者中心页面内，我们需要找到“带货中心”的入口。该入口图标类似一个小卖部，点击即可进入。



## 第五步：开通橱窗权限（或上架商品）

点击“带货中心”后，系统会根据你的账号状态跳转到不同页面。核心操作如下：

![](img/5d52b365268b25975bffb5f27c36b13c_11.png)

*   **若未开通**：页面将引导你开通橱窗权限。
*   **若已开通**：页面将直接进入商品上架管理界面。

以下是开通橱窗权限的具体步骤：

1.  **选择带货身份**：系统会要求你选择带货身份，通常有“我是商家”和“我是达人”等选项。**对于大多数个人创作者，建议选择“我是达人”**。
2.  **缴纳保证金**：选择身份后，会跳转到缴纳保证金的界面。目前视频号达人开通橱窗通常需要缴纳**100元**的保证金。点击“去交纳”，按照提示在微信内完成支付即可。



完成以上步骤后，你的视频号商品橱窗功能就成功开通了。

![](img/5d52b365268b25975bffb5f27c36b13c_13.png)

## 第六步：上架商品

开通权限后，再次点击“带货中心”，即可进入选品、上架商品的界面。你可以在这里从视频号官方选品库中选择商品，将其添加到自己的橱窗中，以便在视频或直播中进行推广。

![](img/5d52b365268b25975bffb5f27c36b13c_15.png)

本节课中我们一起学习了开通视频号商品橱窗和上架商品的完整流程。总结起来，关键步骤是：**进入创作者中心 -> 找到带货中心 -> 选择达人身份并缴纳保证金 -> 开通后在选品库上架商品**。掌握这个基础操作，你就迈出了视频号带货变现的第一步。
