## 视频号基础与目标设定
话不多说，我们从最基础开始。视频号的入口位于微信朋友圈下方。它可以发布短视频、长视频（目前最长可达1小时）以及图片（最多9张）。我们主要操作的是短视频。

![](img/437cc5c5da54c8e7e45131cdd27102a2_8.png)

![](img/437cc5c5da54c8e7e45131cdd27102a2_9.png)

开始之前，你需要为自己制定一个计划和小目标。明确你运营账号的目的是什么。

![](img/437cc5c5da54c8e7e45131cdd27102a2_10.png)

*   **如果只是随意玩玩**：你可以将视频号当作朋友圈，随意发布自拍、文字或搞笑视频。
*   **如果希望通过视频号赚钱或做成事业**：你需要认真地为账号进行定位。

**定位是第一步**。它决定了你后期带什么货、吸引什么样的人群，因此必须重视。

## 定位策略与内容垂直
对于在其他平台（如抖音、快手）已有作品的创作者，可以直接将内容搬运到视频号。有些内容在其他平台不火，但在视频号可能会突然爆冷，获得肉眼可见的播放量增长。

![](img/437cc5c5da54c8e7e45131cdd27102a2_12.png)

![](img/437cc5c5da54c8e7e45131cdd27102a2_13.png)

以下是定位的类型，你可以根据自己的兴趣爱好或特长来选择：
1.  **音乐类**
2.  **好物分享（生活好物）类**
3.  **美食制作类**
4.  **精彩生活分享类**
5.  **其他特长展示类**（如唱歌、特殊技能等）

**确定定位后，不要随意更改**。之后发布作品必须保持**垂直**。

**垂直**的意思是：如果你确定做搞笑类内容，就应持续发布搞笑类视频，而不能今天发搞笑、明天发影视剪辑、后天发音乐表演。内容杂乱无法吸引精准的粉丝。
