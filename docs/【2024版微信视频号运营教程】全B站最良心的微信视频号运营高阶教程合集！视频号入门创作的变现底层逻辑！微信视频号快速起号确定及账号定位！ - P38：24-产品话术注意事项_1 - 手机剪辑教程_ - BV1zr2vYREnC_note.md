# 视频号运营教程：P38：新人开播的话术准备与避坑指南 🎤

在本节课中，我们将学习新人开播前最关键的一步：如何准备合规且有效的直播话术。我们将重点讲解如何获取话术素材、如何修改以避免违规，以及如何利用工具高效整理话术。

## 开播前的核心准备

上一节我们介绍了账号搭建与内容剪辑的基础，本节中我们来看看开播前需要做的具体准备工作。当橱窗开通、账号养好、视频会剪之后，开播便提上日程。开播前，核心工作是准备话术、选品以及设计直播流程。我们首先从最重要的话术准备开始讲解。

## 话术的来源与风险

通常，供应链或厂家会提供产品话术。例如，供应链老板可能会直接发给你一套现成的话术文本。**但是，切勿直接照搬使用这些原始话术。** 因为厂家提供的这些话术，往往是从小红书、抖音等平台“套用”过来的，他们本身可能缺乏直播经验。

厂家可能不清楚直播过程中有许多**违禁词**和**违规表述**。一旦触犯，会导致严重的后果。

## 违规话术的严重后果

以下是必须避免的违规话术类型及其后果：

*   **功效性虚假宣传**：这是最严重的违规行为。例如，销售木耳时，宣称其可以“抗癌”、“降三高”；销售山药面时，过度夸大其药用“作用”。这类关于治疗疾病或夸大保健功效的表述绝对禁止。
*   **处罚结果**：一旦违规，直播间可能被中断，更严重的是商品橱窗会被**封禁3到5天**。如果账号正处于流量上升期，这样的处罚会造成巨大损失。

因此，从任何渠道获取的话术都必须经过严格修改和“消毒”后才能使用。

## 如何修改与优化话术

拿到原始话术后，我们需要进行合规化改造。核心原则是：**将绝对化的“功效”描述，转化为温和的“改善”或“使用场景”描述。**

例如，对于一款“樟脑丸”产品：
*   **错误表述（违规）**：“本产品能**除湿**、**防螨**。”（这属于功效宣称）
*   **正确表述（合规）**：“这款产品可以帮助**改善**家里的潮湿环境。”“把它放在衣柜里，可以让衣物保持干爽。”

产品的**使用方法、规格参数（如一盒多少个、一次用多少）** 等信息可以直接引用，这些是客观描述，没有风险。

## 万能话术框架与工具使用

为了高效准备话术，可以借助一个万能框架和实用工具。

**1. 万能话术框架**
这是一个适用于多种产品的结构，你可以根据产品特性填充内容：
```
1. 产品展示：大家好，今天给大家带来这款XX。
2. 核心卖点：它最大的特点是……（结合场景，如：轻松解决小白鞋发黄问题）。
3. 规格与用法：一盒里面有X个，每次用X个，XX天用一次就行。
4. 价格与促销：今天直播间价格是XX，只有今天有这个优惠。
5. 保障与号召：大家收到货就是所见即所得，放心购买。X号链接，直接去拍！
```

**2. 利用工具提取文案**
我们经常看到优秀的带货视频，其文案可以借鉴。这时可以使用“创头猫”等工具（或其他视频文案提取工具）。
操作步骤如下：

**请注意**：提取的文案同样需要修改，去除其中的违规功效词，再与我们自己的万能框架结合。

## 话术的灵活运用与总结

对于视频号中老年用户较多的特点，话术需要更接地气。可以采用一些他们熟悉的“暗语”。例如，将“糖尿病”人群委婉地称为“小糖人”，这样既避免了敏感的医疗词汇，又能让目标客户心领神会。

**话术整理得越充分，你在直播时就会越流畅、越自信。** 不断整理和优化话术，是新人主播必须掌握的技能。

本节课中我们一起学习了新人开播的话术准备全流程：从识别供应链提供话术的风险，到修改违规词的核心技巧，再到利用万能框架和工具高效组织话术。记住，合规是底线，清晰流畅的表述是目标。下一节课，我们将开始讲解直播中的其他重要环节。
