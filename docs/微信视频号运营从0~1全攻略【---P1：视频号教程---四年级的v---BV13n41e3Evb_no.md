## 开通分成计划的核心条件

注册完视频号后，如何快速开通分成计划来领取收益？起号是关键。如果起号不到位，就无法快速开通。

目前，开通视频号分成计划需要满足以下两个条件：
1.  **粉丝数达到100以上**。
2.  **内容符合平台规范**，即优质、原创、画风精美。

前段时间，新注册的视频号拍摄10条风景视频就可能通过原创认证。但现在，仅拍摄风景类内容很难通过。

## 3天快速开通分成计划实测方法

这里我分享一个本人实测有效的方法。

申请注册新的视频号之后，直接进行“怼脸拍”（即人物正面出镜讲话）。每条视频时长需超过9秒。每天发布5条。

![](img/ed67e96588b2ef3c6fe7fb1d4f921cdf_9.png)

**操作流程公式**：
```
新号注册 → 每日“怼脸拍”视频5条（>9秒/条）→ 持续2天 → 第3天申请开通分成计划
```

![](img/ed67e96588b2ef3c6fe7fb1d4f921cdf_10.png)

第三天，你就可以找到入口申请加入分成计划。前几天收益可能较低，但坚持发布优质内容，收益会逐步提升。

## 开通流程与关键注意事项

以下是开通分成计划的具体操作步骤和必须注意的细节。

**第一步：找到视频号分成计划入口**。
在发布10条“怼脸拍”视频后的第三天，在视频号后台找到“分成计划”入口并申请加入。

**关键注意事项**：
通过原创认证后，发布视频时**务必勾选“原创声明”**。如果不点开，即使视频获得几千万播放，也不会产生任何收益。

## 收益计算与提升策略

那么，收益是如何计算的呢？腾讯官方没有给出非常明确的规则。根据我的运营经验，大致范围是：
```
每万次播放量收益 ≈ 1 到 3 元
```
具体收益仍取决于广告点击率。有时即使播放量很高，但如果无人点击视频下方或评论区的广告，收益也不会高。

广告分成收入与作者视频质量、播放量、投稿数量、广告行业选择等多方面因素有关。

因此，我们在制作原创视频时，要想尽一切办法**让评论区活跃起来**，这有助于提升互动和广告曝光。

## 两大推荐赛道：人性语录与文案鸡汤

在众多内容形式中，我推荐两个目前竞争相对较小但流量巨大的赛道。

**1. 人性语录类**
*   **特点**：通过一句话或一段话，直指人性本质。与文案号不同，其粉丝质量通常较高，年龄段多在30岁以上。
*   **优势**：该领域收益潜力很高。

**2. 文案鸡汤类**
*   **特点**：这类内容在任何平台都非常火爆，通过优美的文字和配音传递情感与道理。

对于想入局视频号的新手，我不推荐一开始就做“祝福类”内容，因为该领域过于饱和，竞争非常激烈。

## 内容制作与运营心法

![](img/ed67e96588b2ef3c6fe7fb1d4f921cdf_12.png)

关于“人性语录”和“文案鸡汤”这两个赛道，从获取文案、素材、配音到视频制作的完整流程，我已为大家整理好详细的文档。由于步骤讲解起来较为枯燥，大家可以直接参照文档一步步操作。

![](img/ed67e96588b2ef3c6fe7fb1d4f921cdf_13.png)

在运营时，请注意以下核心心法：

![](img/ed67e96588b2ef3c6fe7fb1d4f921cdf_14.png)

**坚持发布**：
开通原创后，不要因为前几天收益不高就停止更新。以我的账号为例，收益也是从每天几元、几十元逐步积累起来的。

**发布频率**：
每个账号每天最好发布3-5条视频，**切记不要超过5条**。当前平台规则下，日更数量过多可能导致账号权重下降。

![](img/ed67e96588b2ef3c6fe7fb1d4f921cdf_16.png)

**持久运营**：
通过原创认证后，至少要坚持更新7天。我的账号在持续更新10天后，才迎来了显著的收益增长。
