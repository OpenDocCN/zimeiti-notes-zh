## 核心分析思路：从转化率入手 🔍

![](img/09d8bf1c6bb9bbcdbea815b9dbfd303b_5.png)

在整个排名考核中，转化率是重中之重。产品需要有良好的产出和转化，平台才会判定用户喜欢该产品，从而给予更多推荐。因此，转化率是我们持续关注的核心指标。

![](img/09d8bf1c6bb9bbcdbea815b9dbfd303b_7.png)

分析的第一步是观察转化率是否稳定。

![](img/09d8bf1c6bb9bbcdbea815b9dbfd303b_9.png)

**转化率公式**：`转化率 = 成交订单数 / 访客数`

![](img/09d8bf1c6bb9bbcdbea815b9dbfd303b_11.png)

![](img/09d8bf1c6bb9bbcdbea815b9dbfd303b_12.png)

如果店铺转化率维持稳定，但访客量发生了变化，则说明访客变化与转化率无关。

![](img/09d8bf1c6bb9bbcdbea815b9dbfd303b_14.png)

当访客变化而转化率稳定时，我们需要进一步观察和分析其他数据。以下是需要关注的关键指标：

*   **客单价**：`客单价 = 成交总额 / 成交订单数`
*   **店铺成交额**：`成交额 = 客单价 * 成交订单数`
*   **UV价值**：`UV价值 = 成交总额 / 访客数`

![](img/09d8bf1c6bb9bbcdbea815b9dbfd303b_16.png)

如果这些数据发生了变化，而其他数据（如转化率）保持稳定，那么就可以判定访客的变化是由这些数据（客单价、成交额、UV价值）的变化所导致的。

我们通过这种方式进行问题排查和定位，然后针对发现的问题调整运营策略。

![](img/09d8bf1c6bb9bbcdbea815b9dbfd303b_18.png)

## 案例分析：数据联动与问题排查 📉

举个例子：
假设店铺转化率一直稳定在10%，没有太大波动。但在早期，用户倾向于购买高客单价的SKU，而现在客单价下降了。

*   在单量和访客不变的情况下，客单价下降会导致**成交额下降**。
*   成交额下降，则**UV价值**也会随之下降。

![](img/09d8bf1c6bb9bbcdbea815b9dbfd303b_20.png)

当除了转化率稳定外，客单价、成交额、UV价值都在下降时，平台可能会判定该店铺对访客的价值在降低，产品可能过季或存在其他问题。于是，平台会开始**缩减访客推荐**。

![](img/09d8bf1c6bb9bbcdbea815b9dbfd303b_21.png)

如果访客减少后，相关数据未能维持稳定，平台可能进一步判定产品不受欢迎，导致流量大幅下滑，店铺数据崩盘。

![](img/09d8bf1c6bb9bbcdbea815b9dbfd303b_23.png)

因此，当我们发现访客呈下降趋势时，应进行如下排查：
1.  检查转化率是否有问题。如果没问题，则转向其他方面。
2.  如果问题出在客单价下降，就需要分析下降原因（例如：高客单SKU吸引力不足？）。
3.  找出原因后，制定策略引导用户完成更高客单价的成交，从而提升整体成交额和UV价值，最终稳定访客数据。

![](img/09d8bf1c6bb9bbcdbea815b9dbfd303b_25.png)

同理，UV价值和成交额的下降，也无非是转化率波动或客单价下降导致的。

![](img/09d8bf1c6bb9bbcdbea815b9dbfd303b_27.png)

![](img/09d8bf1c6bb9bbcdbea815b9dbfd303b_28.png)

## 当转化率不稳定时 ⚠️

如果转化率本身不稳定、波动、下降或处于较低水平，那么我们的核心目标就是**提升转化率**。

![](img/09d8bf1c6bb9bbcdbea815b9dbfd303b_30.png)

关于如何提升转化率，我们在之前的课程中已经分享过相关优化方法，例如通过优化店铺详情页、主图、评价等来提高转化，此处不再赘述。

![](img/09d8bf1c6bb9bbcdbea815b9dbfd303b_31.png)

![](img/09d8bf1c6bb9bbcdbea815b9dbfd303b_32.png)

同时，在分析转化率下降的原因时，我们不仅要看内部数据，还要关注外部因素，做好对比分析。

以下是可能的外部原因及应对思路：

![](img/09d8bf1c6bb9bbcdbea815b9dbfd303b_34.png)

*   **同行竞争**：是否有同行在做促销活动？他们的活动价格是否更低，抢走了你的转化？或者他们通过活动抢占了你的资源位，导致你访客下降？
    *   **应对**：考虑是否需要进行竞品分析、调整定价策略或参与平台活动来应对挑战。
*   **产品生命周期**：有些产品具有明显的生命周期。如果产品生命周期已到末期，是否应该强行拉升？还是应该通过其他方式（如上新、清仓）完成产品迭代和过渡？

![](img/09d8bf1c6bb9bbcdbea815b9dbfd303b_35.png)
