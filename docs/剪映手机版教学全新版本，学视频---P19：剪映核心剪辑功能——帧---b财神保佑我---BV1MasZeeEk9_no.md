# 剪映手机版教学：P19：剪映核心剪辑功能——帧定格 🎬

在本节课中，我们将要学习剪映App中的一个核心剪辑功能——**帧定格**。这个功能能够将视频的某一瞬间画面静止，是制作卡点视频和特效视频的必备技能。

## 什么是帧定格？ 🖼️

上一节我们介绍了基础的剪辑操作，本节中我们来看看帧定格功能。帧定格指的是在连续播放的视频画面中，突然将某一帧画面静止并持续一段时间。这个功能常用于制作转场特效或强调某个精彩瞬间，能让视频更具视觉冲击力。

## 帧定格效果案例 📽️

![](img/720d260afcba311f491958db6dfad66e_3.png)

讲了这么多理论，可能很多小伙伴还不清楚定格功能的具体表现。以下是几个典型的应用场景：

*   **卡点视频**：在音乐节奏点（如鼓点）上使用定格，画面瞬间静止，与音乐形成强烈呼应。
*   **特效转场**：从一个动态场景突然定格，然后切换到下一个场景，营造酷炫的转场效果。
*   **强调画面**：突出视频中的某个关键动作或表情，通过定格让观众有更长的停留时间观看。

## 如何添加帧定格？ ✂️

理解了概念和效果后，我们进入实操环节，看看如何在剪映中添加帧定格。

1.  **导入并选中素材**：打开剪映App，点击“创作”导入一段视频素材。在时间轴上选中你想要添加定格的那段视频。

2.  **找到定格功能**：在底部出现的二级菜单中，滑动找到并点击 **“定格”** 按钮。

    ```python
    # 操作路径示意
    选中视频片段 -> 底部菜单 -> 滑动 -> 点击【定格】
    ```

3.  **生成定格画面**：点击后，剪映会自动截取选中片段**最后一帧**的画面，并生成一段默认时长为3秒的静态图片片段。

4.  **编辑定格片段**：生成的这个静态片段，你可以像处理普通图片或视频一样对它进行编辑。以下是你可以进行的操作：

    *   **调整时长**：拖动定格片段的两端，可以自由缩短或延长其静止显示的时间。
    *   **添加特效**：为定格画面添加“牛皮纸”、“褪色”等特效，增强风格。
    *   **添加滤镜**：为定格画面更换不同的滤镜，改变色调和氛围。
    *   **进行裁剪或删除**：如果不需要，可以像删除普通片段一样删除它。

## 核心要点与技巧 💡

![](img/720d260afcba311f491958db6dfad66e_5.png)

使用“定格”功能后，剪映会自动生成一个3秒的静态画面。这个时长是可以自由调整的，你可以根据音乐节奏或视频节奏，将其设置为1秒、2秒或其他任何时长。

除了利用视频片段的最后一帧，你还可以先将播放指针（时间轴）移动到视频的任意一帧，然后分割视频，再对分割出来的那一小段视频使用“定格”功能，这样就能定格你想要的任意瞬间画面。
