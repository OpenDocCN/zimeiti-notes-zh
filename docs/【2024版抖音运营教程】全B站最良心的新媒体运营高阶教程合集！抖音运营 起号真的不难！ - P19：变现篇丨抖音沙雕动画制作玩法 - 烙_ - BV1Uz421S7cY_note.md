# 抖音运营高阶教程：P19：变现篇丨沙雕动画制作玩法 🎬

在本节课中，我们将学习一种被称为“懒人做法”的抖音沙雕动画制作方法。这种方法无需复杂技术，也无需自己编写段子和配音，就能制作出吸引流量的作品。我们将从软件操作到内容创作，一步步拆解这个流程。

## 概述：沙雕动画的“懒人”制作法

这种沙雕动画的制作方法只有少数人知道。它不需要你掌握任何技术，也不需要你编写段子和配音，就能让作品自带流量。虽然它看起来复杂，似乎不是一个人能完成的，但实际上，一部手机就能制作。

## 第一步：软件设置与场景选择

![](img/16aee9f389e7adffed4b0f436d98dd78_3.png)

在制作上，你只需要打开指定的软件。首先，选择合适的视频比例。对于抖音，选择 **9:16** 的比例；对于西瓜视频，选择 **16:9** 的比例。分辨率选择 **1080P** 会让视频更清晰。



![](img/16aee9f389e7adffed4b0f436d98dd78_3.png)

![](img/16aee9f389e7adffed4b0f436d98dd78_5.png)

帧率保持默认设置不动。接下来，在场景库中进行选择，例如选择一个古代场景。



![](img/16aee9f389e7adffed4b0f436d98dd78_5.png)

## 第二步：选择角色与模型

![](img/16aee9f389e7adffed4b0f436d98dd78_7.png)

将界面向右滑动，就能看到该分类下的全部场景。选择角色的方式与此相同。分类中有许多角色模型，包括沙雕风格、武侠风格、斗罗大陆风格、奥特曼风格等，非常丰富。



![](img/16aee9f389e7adffed4b0f436d98dd78_7.png)

## 第三步：动画制作与动作编辑

当我们把场景和角色都选择好后，就可以开始制作动画了。它的操作方式与手机版剪映软件相似。我们点击角色，选择“添加动作”，然后点击“播放”预览，人物就会动起来。如果你不喜欢某个动作，可以随时更换。你还可以添加更多角色来丰富情节。最后，导入音频文件，就可以合成出完整的动画。这个过程非常简单。



## 第四步：配音的“懒人”玩法

配音是沙雕动画的灵魂。下面教你一个无需自己编段子的“懒人”玩法，这个方法还能带来流量和收益。这个方法就是“拍同款”。一些聪明的博主已经开始使用这个方法。具体操作是，你去找一些搞笑博主的视频，直接使用他的原声来为你的沙雕动画配音。



![](img/16aee9f389e7adffed4b0f436d98dd78_9.png)

![](img/16aee9f389e7adffed4b0f436d98dd78_9.png)

## “拍同款”方法的三大优势

以下是使用“拍同款”方法为沙雕动画配音的三个主要好处：

1.  **原创度高**：作品原创度极高，平台不会因此限制你的视频播放量和收益。
2.  **蹭取流量**：你可以蹭到大博主的流量。有些用户喜欢到原视频下观看其他人的模仿作品，这为你带来了额外的曝光机会。
3.  **节省精力**：这省去了你编写段子和配音的精力，提高了效率。节省下来的时间，你可以用来运营更多账号，获取更多流量。

所以，有时候你缺少的不是资源，而是发现资源的眼光。
