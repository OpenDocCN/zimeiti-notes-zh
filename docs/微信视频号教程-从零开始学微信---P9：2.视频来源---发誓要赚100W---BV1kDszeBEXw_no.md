# 微信视频号教程：P9：2.视频来源

## 概述
在本节课中，我们将学习如何为视频号带货获取视频素材。核心内容包括：如何从其他平台（如抖音）搬运视频、去除视频水印，以及提取视频中的文案脚本。这些是进行视频混剪和二次创作的基础步骤。

---

## 视频搬运与去水印方法
上一节我们介绍了视频号运营的基本思路。本节中，我们来看看如何具体获取无水印的视频素材。我们通常不需要自己实拍，而是通过搬运和混剪来制作内容。

![](img/a71c2d9be779da2708f64d01980e0f75_1.png)

操作的核心是使用微信小程序工具。以下是具体步骤：

1.  **在抖音找到目标视频**
    打开抖音，搜索你需要的产品关键词（例如“擦丝器”），找到合适的带货视频。

2.  **复制视频链接**
    点击视频分享箭头，选择“复制链接”。

3.  **使用去水印工具**
    返回微信，在搜索框输入“去水印”，可以找到许多免费工具。课程中演示了“青抖去水印”小程序。

4.  **去除并保存视频**
    在工具中粘贴复制的链接，点击“一键去水印”。完成后，根据提示观看广告，即可将无水印视频保存到手机相册。

![](img/a71c2d9be779da2708f64d01980e0f75_3.png)

> **核心概念**：去水印的本质是通过工具解析视频源文件地址，避开平台添加的水印层进行下载。代码逻辑可简化为：`解析(视频链接) -> 获取纯净视频流 -> 下载`。

需要注意的是，直接通过抖音保存的视频会带有平台水印（通常在左上角或右下角），因此必须经过去水印步骤才能使用。

该工具不仅支持抖音、快手等平台的链接，也支持提取你手机相册中本地视频的文案，非常实用。
