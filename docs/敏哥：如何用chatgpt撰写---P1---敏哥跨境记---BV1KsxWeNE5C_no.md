# 敏哥跨境记：P1：如何用ChatGPT高效撰写亚马逊Listing

在本节课中，我们将学习如何通过训练和提问两个核心步骤，使用ChatGPT高效、专业地撰写亚马逊产品Listing。掌握此方法能显著提升文案质量和创作效率。

## 🎯 概述：训练与提问

用ChatGPT撰写文案，最重要的就是两个步骤：**训练**和**提问**。

上一节我们概述了核心流程，本节中我们来看看具体如何操作。

## 📚 第一步：角色训练与数据输入

首先，你需要对ChatGPT进行角色设定和专项训练。

你可以这样开始训练：“下面请你扮演一名亚马逊的Listing撰写专家。我会以‘请学习’为开头，传授你撰写亚马逊Listing标题的技巧。如果学会了，那就回复‘学会了’。”

以下是训练的关键操作：

你需要找出大量优质的标题案例，至少10个以上，提供给ChatGPT，让它通过学习快速掌握Listing标题的撰写技巧。

## 💡 第二步：构建有效的提问指令

完成训练后，接下来的核心就是如何提问。

例如，一个有效的提问指令第一段可以这样写：“我是一名亚马逊卖家，请你帮我根据以下信息，生成我的Listing标题内容。” 后面附上你产品的所有相关信息。

接着，在指令中提出具体要求。以下是构建指令的关键要点：

*   **提取信息**：先提取产品卖点和特性。
*   **融入关键词**：加上产品所在品类的关键词。
*   **优化表达**：优化表达，让北美客户看起来更舒服、更自然。
*   **严格围绕关键词**：请严谨地围绕关键词展开创作，必须达到100%关键词相关性以及埋词率。
*   **输出格式**：请给我英文和中文两个版本。

## ✨ 效果总结

基本按照以上两个步骤操作下来，ChatGPT写出来的文案就会非常专业和地道，其质量至少能超越市面上80%的运营人员。

本节课中，我们一起学习了利用ChatGPT撰写亚马逊Listing的高效方法，核心在于通过**角色训练**和**结构化提问**来引导AI产出专业文案。掌握“训练”与“提问”这两个关键步骤，你就能大幅提升Listing撰写的效率与质量。
