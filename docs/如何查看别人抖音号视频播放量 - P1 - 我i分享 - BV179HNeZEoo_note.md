# 抖音数据分析教程：P1：如何查看他人账号视频播放量 📊

在本节课中，我们将学习如何通过官方工具“巨量百应”查看其他抖音账号的视频播放量等数据。这个方法主要适用于已开通电商权限的账号。

![](img/7108d2eeaf47f5f7bedc830a3f84817b_1.png)

## 操作步骤详解

上一节我们介绍了课程目标，本节中我们来看看具体的操作流程。

![](img/7108d2eeaf47f5f7bedc830a3f84817b_3.png)

### 1. 登录与入口
首先，打开“巨量百应”并登录任意一个账号（通常需要该账号已开通电商权限）。登录后，在界面中找到并点击“成长”选项。



### 2. 添加目标作者
在“成长”页面下方，找到“收藏作者”功能。



以下是添加作者的具体步骤：
1.  点击“添加作者”。
2.  在弹出的输入框中，**需要输入对方的抖音号（一串数字或字母组合），而不是抖音昵称**。
3.  从抖音复制目标账号的抖音号，粘贴至此并点击添加。



### 3. 查看基础数据
添加成功后，点击该作者即可进入其数据概览页。在此页面，你可以直接看到其部分视频的播放量。但默认展示有限，最多只显示4页，每页5个视频，共计20个。

### 4. 查看详细数据
为了获取更全面的数据，我们需要查看详情页。在作者数据页面的最上方，找到并点击“视频详情”。



进入“视频详情”页面后，向下滑动，即可查看该账号**近30天**内发布的所有视频的详细数据，包括：
*   **播放量**
*   **点赞数**
*   **评论数**
*   **分享数**

30天的数据对于大多数分析需求已经足够。

## 常见问题与解决方案

有时在添加作者时，系统可能会提示“没有权限”。本节中我们来看看如何解决这个问题。

问题的核心在于数据查看权限是相互的。如果你想查看他人的数据，通常也需要开放自己的数据供他人查询。



以下是解决权限问题的步骤：
1.  点击“巨量百应”右上角的“设置”图标。
2.  进入“个人信息管理”。
3.  你会看到三个选项，请确保将下方两个关于数据共享的选项**打开**。
    *   **逻辑公式**：`你的数据公开状态` 与 `查看他人数据的权限` 相关联。
4.  开启后，通常即可正常查看大部分账号的数据。如果开启后仍无权限，则可能是对方账号关闭了其数据共享设置。

![](img/7108d2eeaf47f5f7bedc830a3f84817b_5.png)

本节课中我们一起学习了通过“巨量百应”平台查看他人抖音视频数据的方法。关键点在于使用正确的“抖音号”添加作者，并在权限受阻时检查并打开自身的数据共享设置。掌握这个方法，可以帮助你更好地进行账号分析和市场调研。
