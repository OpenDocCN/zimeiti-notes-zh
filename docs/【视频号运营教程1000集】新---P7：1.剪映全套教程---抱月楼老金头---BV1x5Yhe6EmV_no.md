# 视频号运营教程：P7：剪映自学指南

在本节课中，我们将学习如何从零开始自学剪映软件。剪映是抖音官方出品的视频剪辑工具，功能强大且大部分免费，非常适合新手入门。掌握其基础功能是进行视频创作的第一步。

## 软件获取与简介

上一节我们介绍了课程背景，本节中我们来看看如何获取并开始学习剪映。

首先，如果你从未使用过剪映，需要在手机应用商店搜索并下载“剪映”App。它是目前使用最广泛的剪辑软件之一，与“必剪”等工具功能相似。作为抖音官方软件，剪映功能全面，基本可以满足绝大部分免费剪辑需求，无需开通会员。

## 总结与建议

本节课中我们一起学习了如何自学剪映。主要有两种途径：
1.  **系统学习**：通过剪映App内的“创作者课堂”，跟随“剪映官方”账号的免费课程（《剪映功能大全》和《新手常见问题》）进行系统学习。
2.  **碎片化学习**：在抖音平台搜索教学账号（如“妮妮6868618”）或关键词，观看短视频教程作为补充。

剪映学习并不困难，将上述推荐的课程看完，你就能熟练运用其基础功能，这些功能对于日常视频创作来说完全足够。
