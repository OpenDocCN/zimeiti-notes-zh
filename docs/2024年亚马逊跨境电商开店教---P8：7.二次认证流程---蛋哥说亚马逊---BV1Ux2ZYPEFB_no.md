# 亚马逊跨境电商入门到精通：P8：二次认证与账户信息完善

## 🔐 登录卖家后台与二次认证

注册成功后，您会收到确认邮件。此时，您需要登录卖家中心后台。

**登录地址**：在您最初注册的亚马逊卖家平台页面进行登录。例如，北美站卖家平台的登录入口通常位于页面右侧。

登录成功后，您将进入卖家后台界面。此时，系统通常会提示您完成一项重要的安全设置：**启动两步验证**。

两步验证是亚马逊为保护账户安全而设置的额外验证步骤。您需要按照以下流程操作：

![](img/71c21210da29a6f5bd1e0bd04c064531_11.png)

1.  在卖家后台，点击右上角的 **“设置”**，然后进入 **“登录设置”** 或直接找到 **“二次认证”/“两步验证”** 选项。
2.  点击 **“启动两步验证”**。
3.  系统会提示您添加一个手机号码用于接收短信验证码。请确保使用一个**未在亚马逊账户中绑定过**的手机号码。
4.  选择 **“短信”** 方式，输入手机号，点击继续。

![](img/71c21210da29a6f5bd1e0bd04c064531_13.png)

完成手机号验证后，系统会引导您设置认证器应用。

![](img/71c21210da29a6f5bd1e0bd04c064531_15.png)

以下是设置认证器应用的步骤：

1.  系统会提示您需要在手机上安装一个认证器应用程序。点击下拉菜单，它会推荐几个应用，例如 **Google Authenticator（谷歌身份验证器）** 或 **Microsoft Authenticator**。
2.  前往您的手机应用商店（如苹果App Store或安卓应用市场），搜索并下载推荐的认证器应用（例如“Google Authenticator”）。
3.  打开认证器应用，扫描亚马逊后台提供的二维码。
4.  扫描成功后，认证器应用会生成一个6位数的动态验证码。将此验证码输入到亚马逊后台的指定位置并提交。

![](img/71c21210da29a6f5bd1e0bd04c064531_17.png)

完成以上步骤，两步验证即设置成功。此后登录账户时，除了输入密码，还需要输入认证器应用生成的动态码，安全性大大提高。

![](img/71c21210da29a6f5bd1e0bd04c064531_19.png)

![](img/71c21210da29a6f5bd1e0bd04c064531_20.png)

## 📝 完善账户与税务信息

完成两步验证后，您的账户可能仍未完全激活。卖家后台首页或设置页面常会有一个醒目的提示，要求您 **“提供账户信息”** 或完善 **“税务信息”**。

您需要按以下步骤操作：

1.  点击提示信息，或通过 **“设置” > “账户信息”** 进入相关页面。
2.  在“账户信息”页面，找到显示为红色或带有警告标识的 **“税务信息”** 部分并点击。
3.  系统会引导您填写一份税务调查表格（例如W-8BEN表格，用于非美国纳税人）。
4.  填写时，请根据您的实际情况选择“个人”或“企业”。
5.  在需要填写姓名、地址等信息时，**若表格没有明确要求填写中文，则使用拼音填写**。例如，姓名和地址均填写拼音形式。
6.  仔细核对所有信息，确认无误后提交表格。

提交成功后，您的账户信息便完善完毕，店铺注册流程基本完成。

## ⚠️ 关于账户审核（二审）的说明与应对

部分卖家在注册后或运营中可能会遇到“二审”（即账户审核），收到账户被冻结并要求提交额外材料进行申诉的通知。

如果遇到二审，请严格遵循以下原则：

![](img/71c21210da29a6f5bd1e0bd04c064531_22.png)

*   **按提示提交材料**：亚马逊会明确列出所需文件，例如**水电费账单、网络宽带费账单、信用卡账单**等。要求什么就提交什么。
*   **确保信息一致**：所有账单上的姓名和地址必须与您注册亚马逊账户时使用的信息（即营业执照或身份证信息）**完全一致**。
*   **提高通过率**：提交的材料越齐全、越清晰，通过审核的概率越大。

**常见问题解决方案**：
如果账单（如水电费）的缴费人不是注册人本人（例如是配偶），您可以准备一份**正式的租房合同**，证明注册人是该地址的租客。将租房合同与房东名下的账单一起拍照提交，以证明消费关联性。

![](img/71c21210da29a6f5bd1e0bd04c064531_24.png)

**重要提醒**：
遇到二审，很大程度上是因为注册时信息填写有误或材料准备不充分。因此，注册前务必仔细准备，操作时认真核对。若申诉多次仍不通过，可能需要更换全新的信息（包括电脑、网络IP等）重新注册。

![](img/71c21210da29a6f5bd1e0bd04c064531_26.png)

## 💎 课程总结

![](img/71c21210da29a6f5bd1e0bd04c064531_28.png)

本节课中我们一起学习了亚马逊店铺注册后的收尾工作：
1.  **设置二次认证**：通过短信和认证器应用启用两步验证，保障账户安全。
2.  **完善税务信息**：根据引导填写税务表格，使用拼音完成信息确认。
3.  **了解账户审核**：认识到二审的存在，并掌握了遇到时应遵循的“按提示提交一致材料”的基本原则。
