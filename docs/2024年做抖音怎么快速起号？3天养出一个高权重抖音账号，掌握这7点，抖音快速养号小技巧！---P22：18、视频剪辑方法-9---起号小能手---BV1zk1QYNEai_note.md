# 抖音运营：P22：18、视频剪辑方法-9 ✂️

![](img/388b515495b551683bd3f7cfaf031042_1.png)

在本节课中，我们将通过实际操作，学习如何使用剪映App完成一条抖音视频的完整剪辑流程。课程将涵盖从素材导入、基础剪辑、添加音乐字幕，到最终导出的所有关键步骤，并额外介绍一款实用的提词器软件，帮助口播创作者提升效率。

上一节我们介绍了视频拍摄的要点，本节中我们来看看如何将拍摄好的素材剪辑成一条完整的视频。我们将使用抖音官方的剪辑软件——剪映。

首先，请确保你下载的是黑色的“剪映”App。它提供手机版和电脑版，我们先从手机版开始讲解。

![](img/388b515495b551683bd3f7cfaf031042_9.png)

![](img/388b515495b551683bd3f7cfaf031042_11.png)

打开剪映App，点击“开始创作”按钮。

![](img/388b515495b551683bd3f7cfaf031042_13.png)

![](img/388b515495b551683bd3f7cfaf031042_1.png)

以下是导入素材的步骤：
1.  从相册中选择你需要剪辑的视频片段。
2.  点击“添加”按钮，将视频导入到剪辑轨道中。
3.  如果你有多个片段（如生活Vlog的开头、中间、结尾），建议先导入并剪辑好第一条，再点击轨道尾部的“+”号导入第二条，以此类推。这样可以保持剪辑过程清晰有序。

![](img/388b515495b551683bd3f7cfaf031042_15.png)

---

![](img/388b515495b551683bd3f7cfaf031042_17.png)

现在，我们已经将一条22秒的视频素材添加到了剪辑轨道。接下来，我们开始进行基础调整。

![](img/388b515495b551683bd3f7cfaf031042_19.png)

首先检查视频的构图和比例。如果视频方向或比例不合适，可以进行以下操作：
1.  选中轨道上的视频片段。
2.  向右滑动底部工具栏，找到并点击“编辑”按钮（图标通常是一个方块）。
3.  点击“旋转”功能，将视频调整到正确的方向。
4.  如果需要裁剪画面比例，可以点击“裁剪”功能，选择如 **1:1**、**4:3**、**16:9** 等预设比例，或手动调整。

![](img/388b515495b551683bd3f7cfaf031042_21.png)

![](img/388b515495b551683bd3f7cfaf031042_23.png)

调整完毕后，点击播放预览视频。你会发现视频开头有一段没有说话的空镜头，这部分需要剪掉。

![](img/388b515495b551683bd3f7cfaf031042_25.png)

以下是剪辑无用片段的方法：
1.  将时间轴（那条竖线）移动到需要裁剪的位置。
2.  选中视频片段，点击底部工具栏的“分割”按钮。视频会被分成两段。
3.  选中不需要的前一段，点击“删除”按钮。
4.  重复此操作，删除视频中间或结尾不需要的部分（如关手机的动作）。

![](img/388b515495b551683bd3f7cfaf031042_27.png)

![](img/388b515495b551683bd3f7cfaf031042_29.png)

**核心操作可以总结为：选中 -> 分割 -> 选中 -> 删除。**

![](img/388b515495b551683bd3f7cfaf031042_31.png)

![](img/388b515495b551683bd3f7cfaf031042_33.png)

![](img/388b515495b551683bd3f7cfaf031042_15.png)

视频内容修剪完毕后，我们需要为其添加背景音乐。音乐能极大地提升视频的观感和氛围。

![](img/388b515495b551683bd3f7cfaf031042_39.png)

![](img/388b515495b551683bd3f7cfaf031042_41.png)

点击底部工具栏的“音频”按钮。

![](img/388b515495b551683bd3f7cfaf031042_43.png)

以下是添加音乐的两种方法：
1.  **提取音乐**：如果你已经在其他软件（如淘金阁）中准备好了配音或音乐文件，可以点击“提取音乐”，直接从手机视频中提取音频。
2.  **添加音乐**：点击“音乐”，进入剪映自带的音乐库。你可以根据分类（如美食、美妆、萌宠）寻找，或直接搜索歌曲名称。点击歌曲旁边的播放图标可以预览，选定后点击“使用”即可添加。

![](img/388b515495b551683bd3f7cfaf031042_45.png)

![](img/388b515495b551683bd3f7cfaf031042_47.png)

![](img/388b515495b551683bd3f7cfaf031042_21.png)

![](img/388b515495b551683bd3f7cfaf031042_49.png)

添加的音乐如果过长，同样可以使用 **选中 -> 分割 -> 删除** 的方法进行裁剪。
如果需要调整音乐音量，选中音频轨道，点击“音量”按钮，拖动滑块即可调节大小。

![](img/388b515495b551683bd3f7cfaf031042_51.png)

---

![](img/388b515495b551683bd3f7cfaf031042_53.png)

画面和音乐都准备好后，下一步是为视频添加字幕。手动输入字幕非常耗时，剪映提供了高效的自动识别功能。

![](img/388b515495b551683bd3f7cfaf031042_55.png)

点击底部工具栏的“文本”按钮，然后选择“识别字幕”。

![](img/388b515495b551683bd3f7cfaf031042_57.png)

以下是生成和调整字幕的步骤：
1.  在弹出窗口中，选择“仅视频”，然后点击“开始识别”。系统会自动识别视频中的语音并生成字幕。
2.  识别完成后，字幕会出现在视频的相应位置。你可以拖动字幕条调整其出现的时间点。
3.  检查是否有错别字，双击字幕即可进行修改。
4.  你还可以统一修改字幕的样式，如字体、颜色、大小等，使其与视频风格更搭配。

![](img/388b515495b551683bd3f7cfaf031042_59.png)

![](img/388b515495b551683bd3f7cfaf031042_43.png)

![](img/388b515495b551683bd3f7cfaf031042_61.png)

至此，视频的比例、内容、音乐和字幕都已处理完毕，一条完整的视频就初具雏形了。

在导出最终成品前，有一个关键设置会影响视频的清晰度，务必进行调整。

![](img/388b515495b551683bd3f7cfaf031042_67.png)

点击右上角的“导出”按钮。在导出设置页面，请进行以下调整：
*   **帧率 (FPS)**：设置为 **30**。这是抖音平台最合适的帧率。
*   **分辨率**：尽可能拉到最高（如1080P或4K）。分辨率越高，视频越清晰。

![](img/388b515495b551683bd3f7cfaf031042_69.png)

**导出设置公式：帧率 = 30，分辨率 = 最高。**

![](img/388b515495b551683bd3f7cfaf031042_71.png)

设置完成后，点击“导出”。请注意，导出过程中不要锁屏或切换其他应用程序，耐心等待导出完成。导出后，你可以直接通过剪映分享到抖音。

![](img/388b515495b551683bd3f7cfaf031042_73.png)

最后，为进行口播视频创作或直播的同学推荐一个实用工具——**爱提词**。它可以解决记不住台词、需要反复背诵的难题。

![](img/388b515495b551683bd3f7cfaf031042_79.png)

以下是爱提词App的基本使用方法：
1.  打开App，点击右下角的“+”号，新建一个提词板。
2.  输入或粘贴你的口播文案，并保存。
3.  点击“开始提词”，选择“录制模式”。此时，文案会以悬浮窗形式显示在手机屏幕上，你可以一边看词一边录制，无需背诵。
4.  在设置中，可以调整字体大小、滚动速度等，方便阅读。
5.  该App内置了基础的美颜功能（如瘦脸、大眼、磨皮），拍摄效果比原相机更佳，且比一些美颜相机更自然。
6.  软件内还提供“爆款文案”库，涵盖情感、职场、生活等多个领域，可供创作参考。
