# 微信视频号运营从0到1全攻略：P20：18.活动介绍 🎬

在本节课中，我们将要学习一个由官方发起的、旨在助力商家进行视频化带货的专项活动。我们将详细介绍该活动的背景、核心福利、参与方式以及具体的操作步骤。

## 活动背景与目标

![](img/a16d560fd2eecbe323fa3da5847fa7ed_1.png)

![](img/a16d560fd2eecbe323fa3da5847fa7ed_2.png)

为促进2024年中国行业的新发展，并为视频化带货电商奠定技术基础，特推出面向商家的专项活动。本次活动由短视频教研部负责人张良老师全程对接。活动秉持公平公正的理念展开。

张老师作为本次活动的全国总负责人，拥有独家对接权限。这意味着，只有通过张老师这里才能对接本次活动资源。

![](img/a16d560fd2eecbe323fa3da5847fa7ed_4.png)

## 活动核心福利介绍

![](img/a16d560fd2eecbe323fa3da5847fa7ed_6.png)

上一节我们介绍了活动的背景，本节中我们来看看本次活动为参与者提供的具体支持。以下是本次活动的三大核心福利：

![](img/a16d560fd2eecbe323fa3da5847fa7ed_8.png)

1.  **官方提供高质量视频素材**：许多同学担心没有素材。官方已为大家准备好了素材。每个参与账号将获得30条视频素材，每天发布一条即可。
    ![](img/a16d560fd2eecbe323fa3da5847fa7ed_1.png)
    ![](img/a16d560fd2eecbe323fa3da5847fa7ed_2.png)

![](img/a16d560fd2eecbe323fa3da5847fa7ed_10.png)

2.  **提供巨额流量扶持**：针对“发了视频卖不出去”的担忧，活动为每个账号提供90万的流量扶持。
3.  **发放视频发布保底奖励**：本次活动力度巨大。参与者每发布一条视频，即可获得300元保底奖励。奖励次日到账。发布30条视频总计可获得9000元奖励。
    ![](img/a16d560fd2eecbe323fa3da5847fa7ed_15.png)

![](img/a16d560fd2eecbe323fa3da5847fa7ed_12.png)

## 素材库与使用方式

![](img/a16d560fd2eecbe323fa3da5847fa7ed_13.png)

了解了核心福利后，我们来看看官方素材库的具体内容和使用方法。官方素材库品类丰富，质量上乘，操作便捷。

以下是素材库的特点：

*   **品类齐全**：素材库包含中秋节礼品、餐巾纸、牛奶、火锅底料、洗漱手套、不锈钢杯等多种商品视频。
    ![](img/a16d560fd2eecbe323fa3da5847fa7ed_4.png)
    ![](img/a16d560fd2eecbe323fa3da5847fa7ed_6.png)
    ![](img/a16d560fd2eecbe323fa3da5847fa7ed_8.png)
    ![](img/a16d560fd2eecbe323fa3da5847fa7ed_10.png)
*   **支持分类与搜索**：素材库支持按分类筛选，也支持关键词搜索功能，方便快速找到所需素材。例如，搜索“书籍”即可找到相关视频素材。
    ![](img/a16d560fd2eecbe323fa3da5847fa7ed_12.png)
    ![](img/a16d560fd2eecbe323fa3da5847fa7ed_13.png)
*   **直接下载使用**：选中素材后，可直接点击下载，方便快捷。

![](img/a16d560fd2eecbe323fa3da5847fa7ed_15.png)

本次活动由微信视频号官方发起，张良院长为唯一指定对接人。活动具有明确的时效性。

活动于1月13日正式开始，并定于8月10日结束。今天是8月9日，是参与活动的最后机会。
