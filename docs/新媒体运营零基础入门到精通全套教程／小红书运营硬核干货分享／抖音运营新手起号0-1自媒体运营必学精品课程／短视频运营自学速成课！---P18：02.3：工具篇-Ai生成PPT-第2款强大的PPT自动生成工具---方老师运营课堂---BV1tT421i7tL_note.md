在本节课中，我们将学习第二款强大的AI生成PPT工具。这款工具在生成速度、稳定性和模板丰富度方面具有优势，能与上一节课分享的工具形成互补，共同提升我们的PPT制作效率。

上一节我们介绍了第一款AI生成PPT工具，本节中我们来看看另一款同样高效的工具。它的核心优势在于**生成速度快、运行稳定**，并且提供了**海量的模板**供用户选择。不过，要获得理想效果，需要用户提供**结构清晰、内容详尽的大纲**。

## 两种生成方式

该工具主要提供两种创建PPT的方式：**AI智能生成**和**导入本地大纲**。以下是具体操作步骤。

### 1. 导入本地大纲生成

![](img/61d150baddff14671e44c8ffc870d224_1.png)

![](img/61d150baddff14671e44c8ffc870d224_3.png)

![](img/61d150baddff14671e44c8ffc870d224_5.png)

我通常更倾向于使用“导入本地大纲”的方式，因为它能更好地控制最终内容。

![](img/61d150baddff14671e44c8ffc870d224_6.png)

首先，我们需要准备一份详细的大纲。可以使用ChatGPT等AI工具辅助生成。例如，输入指令：
```
帮我撰写一份关于“装修行业在小红书平台的运营推广计划”的详细大纲，并完善每一个小节点的具体内容信息，保持好结构和格式排版。
```

获得大纲文本后，即可开始操作：
1.  在工具界面选择“导入本地大纲”。
2.  上传包含大纲的本地文件。
3.  工具会快速解析并提取文档结构。
4.  接下来，从丰富的模板库中挑选一个适合装修行业的风格。
5.  点击“生成PPT”，稍等片刻即可完成。

生成后，如果发现某些详细内容（如“用户群体分析”）未被加载到页面中，可以在编辑器的“大纲编辑”区域进行补充。在该区域，可以编辑目录和每个章节下的具体内容。

### 2. AI智能生成

如果你希望从零开始快速生成，可以使用“AI智能生成”功能。
1.  在工具界面选择“AI智能生成”。
2.  在输入框中直接填写主题，例如“装修行业在小红书平台的运营推广计划”。
3.  点击生成，AI会自动构思并生成详细的大纲和内容。
4.  同样，选择一个合适的模板风格。
5.  点击“生成PPT”，AI会合成完整的文案和页面。

## 编辑与调整

无论采用哪种方式生成，都可以在编辑界面进行深度调整，以满足个性化需求。
*   **编辑内容**：在右侧面板可以调整文字、形状和背景。
*   **更换版式**：如果对某一页的排版不满意，可以单击该页面，从推荐的多种版式中一键更换。
*   **整体换肤**：通过“模板替换”功能，可以快速为整个PPT更换另一套设计模板，且响应速度很快。
*   **在线替换**：支持在线更换图片等素材。
