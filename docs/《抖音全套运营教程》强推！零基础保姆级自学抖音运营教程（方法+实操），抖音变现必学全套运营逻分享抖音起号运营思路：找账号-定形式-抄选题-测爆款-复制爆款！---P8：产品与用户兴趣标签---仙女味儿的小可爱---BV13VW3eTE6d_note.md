# 抖音运营教程：P8：产品与用户兴趣标签 🧩

在本节课中，我们将学习如何利用“九宫格选题法”，结合产品属性和用户兴趣标签，为你的抖音账号源源不断地生成创意选题。我们将以健身软件Keep为例，进行详细拆解。

上一节我们介绍了选题的重要性，本节中我们来看看如何系统地挖掘选题方向。

## 产品标签挖掘

首先，我们需要分析产品本身的特点。将你的核心产品（例如Keep）放在九宫格的中心，然后围绕它联想产品的八个关键属性或特点。

以下是围绕Keep健身软件可能想到的产品标签：

1.  **线上健身软件**：产品的核心形态。
2.  **专业健身教练**：产品提供的核心服务或资源。
3.  **饮食搭配建议**：产品的附加功能或特色内容。
4.  **减肥/塑形攻略**：产品解决的主要用户痛点。
5.  **健身干货知识**：产品提供的教育价值。
6.  **科学运动方法**：产品的专业性和可信度支撑。
7.  **打卡运动制度**：产品的运营机制或用户激励方式。
8.  **一二线城市用户为主**：产品的主要用户画像或市场定位。

这个过程的关键是**发散思维**，将你能想到的所有与产品相关的关键词填入周围的八个格子。完成这一步后，产品相关的标签就暂时确定了。

## 用户兴趣标签拓展

接下来，我们转向用户。将你的目标用户（例如“健身爱好者”）放在另一个九宫格的中心，然后围绕他们联想其可能感兴趣的八个领域。

以下是围绕“健身爱好者”可能拓展的用户兴趣标签：

![](img/b305ae343858a03b07c239b8328cabf2_1.png)

1.  **健身房**：线下场景的延伸兴趣。
2.  **健身装备**：如运动服装、器材等相关消费。
3.  **轻脂美食/减肥餐**：与健康生活方式相关的饮食兴趣。
4.  **旅行**：作为热爱生活的一种表现。
5.  **星座/明星八卦**：普遍的娱乐消遣兴趣。
6.  **心灵鸡汤/个人成长**：情感或精神层面的需求。
7.  **时尚穿搭**：对外形和美的普遍追求。
8.  **生活技巧/窍门**：提升生活效率或品质的兴趣。

这个步骤的目的是**扩大用户群的外延**，从核心的“健身爱好者”扩展到更广泛的“热爱生活者”，从而发现更多可结合的选题角度。

## 创意连连看：生成具体选题

现在，我们手上有两个填满的九宫格：一个是产品标签，一个是用户兴趣标签。接下来就是最关键的创意环节——连连看。

**方法**：从产品标签和用户兴趣标签中各选一个词，将它们进行组合，尝试构思出一个完整的视频选题或文案标题。

以下是几个组合示例：

*   **组合1**：产品标签`饮食搭配` + 用户标签`旅行`
    *   **生成选题**：《外出旅行时，如何享受美食又能控制体重？》
*   **组合2**：产品标签`运动方法` + 用户标签`明星`
    *   **生成选题**：《赵丽颖都在偷偷练的3个居家燃脂动作，简单易学！》
*   **组合3**：产品标签`科学健身` + 用户标签`健身房`
    *   **生成选题**：《健身房最被低估的5个器械，用对了效果翻倍！》

**技巧**：如果自己联想困难，可以借助搜索引擎。例如，搜索“科学 健身”等关键词组合，查看相关的长尾词或热门话题，从中获得灵感。

通过这种“产品x兴趣”的交叉组合，理论上可以衍生出**无数个选题方向**（8x8=64种基础组合，每种组合又可细化），从而建立起庞大的选题素材库，彻底解决“不知道拍什么”的难题。

## 总结

本节课中我们一起学习了“九宫格选题法”的核心操作流程：
1.  **挖掘产品标签**：围绕核心产品，发散思维列出其所有特点。
2.  **拓展用户兴趣标签**：围绕目标用户，将其兴趣范围从核心向周边生活扩展。
3.  **创意组合**：将产品标签与用户兴趣标签进行两两配对，像玩“连连看”一样，碰撞出具体的视频选题或文案标题。

![](img/b305ae343858a03b07c239b8328cabf2_3.png)

这个方法能系统性地帮你打开思路，将产品卖点与用户感兴趣的内容相结合，创作出既垂直又具有破圈潜力的优质内容。记住，好的选题是成功的一半。
