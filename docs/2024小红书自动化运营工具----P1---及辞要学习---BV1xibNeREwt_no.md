# 小飞薯RPA软件教程：P1：如何更新软件 🔄

在本节课中，我们将要学习如何更新“小飞薯RPA”软件。更新软件是确保您能使用最新功能、优化和修复的关键步骤。整个过程简单快捷，只需几个步骤即可完成。

## 概述

软件更新涉及新功能添加、用户规则优化和错误修复。为了获得最佳体验，请务必使用最新版本。更新通知会通过QQ群、邮件或私信等多种方式发布，但用户也需自行留意群文件更新。

## 获取最新版本

首先，您需要获取最新版本的软件安装包。

*   **来源**：最新版软件位于每一个“小飞鼠优质用户QQ群”的群文件中。
*   **命名规则**：软件包名称遵循 **`小飞鼠{日期版本号}`** 的格式。例如，`小飞鼠240709` 代表2024年7月9日的更新版本。
*   **检查更新**：群文件中的软件始终是最新版。如果您当前使用的是旧版本（如240709），当群文件出现更新的版本（如240713）时，直接下载即可。

## 更新步骤详解

上一节我们介绍了如何获取新软件，本节中我们来看看具体的更新操作方法。整个过程的核心是 **将旧软件中的许可证文件移动到新软件中**。

以下是详细的图文操作步骤：

1.  **下载并解压新软件**：从QQ群文件下载最新版本的软件压缩包，并将其解压到电脑上的任意位置。

    

2.  **转移许可证文件**：
    *   打开您**旧版本**小飞薯软件所在的文件夹。
    *   找到名为 **`许可证`** 的文件（或类似名称的关键文件）。
    *   对该文件执行 **`剪切`** 操作。

    

![](img/8c0a1ba19df8e8e8052c4887fad2e858_3.png)

3.  **粘贴许可证并替换**：
    *   打开您刚解压的**新版本**小飞薯软件文件夹。
    *   在空白处执行 **`粘贴`** 操作，将许可证文件移入新文件夹。
    *   至此，旧版本软件可以安全删除。

    

![](img/8c0a1ba19df8e8e8052c4887fad2e858_5.png)

4.  **启动验证**：
    *   像往常一样，从系统菜单（如MIN）启动新版本的小飞薯软件。
    *   如果软件能正常启动，显示新版本的Logo，并弹出登录窗口，则说明更新已成功完成。
