# 小红书电商运营教程：P11：小红书店铺类型详解 🏪

![](img/b417ae3fa2231efdea84029df24a9d17_0.png)

在本节课中，我们将系统学习小红书电商平台上的店铺类型。了解不同类型的店铺及其特点，是商家进行平台入驻和制定运营策略的第一步。

## 概述

小红书平台上的店铺主要分为两大类：商家入驻的店铺和平台自营的店铺。每种类型下又有更细致的划分，它们在入驻门槛、权重和运营模式上各有不同。

## 商家店铺类型

商家自行申请入驻开设的店铺统称为商家店。根据品牌资质和经营模式，主要可分为以下五种类型。

以下是五种商家店的具体介绍：

1.  **官方旗舰店**
    *   这是权重最高的店铺类型。
    *   申请门槛高，通常需要被小红书平台认定为具有一定知名度的品牌。

2.  **旗舰店**
    *   权重仅次于官方旗舰店。
    *   同样需要品牌具备一定的知名度才能申请。

3.  **品牌店**
    *   这是最普遍、最常见的商家店铺类型。
    *   普通商家入驻小红书，默认开通的就是品牌店，其名称通常为“某某某品牌店”。

4.  **专卖店**
    *   与品牌店类似，是较为常见的店铺形式。

5.  **集合店**
    *   此类店铺允许在一个店铺内销售多个品牌的产品，类似于天猫的专营店。
    *   **重要提示**：目前小红书平台已暂停集合店的新入驻申请。未来是否会重新开放，需关注平台政策。

## 店铺权重与升级

上一节我们介绍了店铺的基本类型，本节中我们来看看店铺间的权重关系以及升级路径。

不同店铺类型之间存在明确的权重高低之分，其排序为：**官方旗舰店 > 旗舰店 > 品牌店/专卖店/集合店**。更高的权重意味着在笔记和商品搜索排名中可能获得一定的优先展示优势，但这并非绝对因素。

对于已开店的商家，店铺类型并非一成不变。品牌店在满足一定条件后，可以申请升级为旗舰店或官方旗舰店。

以下是店铺升级的核心条件：

*   **主要依据**：店铺的营业额是关键的考核指标。
*   **标准差异**：不同商品类目对营业额的要求不同。通常，大类目的要求更高，小类目可能月销售额达到几十万即可申请。
*   **申请流程**：当营业额达到平台要求后，商家可向平台提交升级申请。

## 平台自营店铺

除了商家入驻的店铺，小红书平台自身也运营着自营店铺，即“小红书福利社”。这类似于京东的京东自营模式。

小红书福利社会主动筛选在平台上销售火爆、受欢迎的商品，邀请其品牌方以供应商身份入驻。在这种合作模式下，商家作为供货商，以较低的供货价（例如约为市场价的六折）向小红书福利社供货，由平台方负责销售和运营。

## 总结

![](img/b417ae3fa2231efdea84029df24a9d17_2.png)

本节课中我们一起学习了小红书电商的店铺体系。我们了解到店铺主要分为**商家店**和**平台自营店**。商家店中，**官方旗舰店**和**旗舰店**权重最高但申请门槛也高；**品牌店**是最常见的入驻形式；而**集合店**目前已暂停入驻。店铺可根据**营业额**表现进行升级。此外，平台自营的**小红书福利社**会邀请畅销品牌以供应商模式合作。理解这些店铺类型，有助于你根据自身品牌定位和发展阶段，做出最合适的开店选择。
