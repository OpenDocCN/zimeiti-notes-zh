# 抖音运营指南：P11：策划内容定位

## 概述
在本节课中，我们将要学习如何为抖音账号策划一个清晰、具体且有竞争力的内容定位。仅仅选择一个热门主题是不够的，我们需要从多个维度进行细化，才能让账号在众多同质化内容中脱颖而出。

![](img/0257ebf9f273d9a0a7fd634fa87980d9_1.png)

上一节我们讨论了如何选择主题IP，本节中我们来看看如何将这个主题深化为完整的、可执行的内容定位。

## 主题IP的局限性
许多同学在选择主题IP后会产生疑问：这个主题似乎很多人都在做，例如“老韩摄影教学”火了之后，又出现了“小韩摄影教学”。当重复内容特别多时，自己选择的主题还能火起来吗？

![](img/0257ebf9f273d9a0a7fd634fa87980d9_1.png)

模仿热门主题是人之常情，但能否成功，关键在于我们后续的**核心内容**策划。仅仅确定“摄影教学”这个主题是不够的，我们还需要思考更具体的问题。

## 深化内容定位的要素
在确定主题IP之后，我们需要构思更具体的内容框架。以下是需要进一步明确的核心要素：

*   **身份**：以什么角色出现？（例如：专业摄影师、摄影爱好者、器材测评师）
*   **场景**：在什么环境下拍摄？（例如：影棚、户外、街头、家中）
*   **道具与动作**：使用什么工具？展现什么行为？（例如：使用特定相机、演示调参数、展示成片）
*   **机位**：采用什么拍摄角度和镜头语言？（例如：第一人称视角、固定机位、多机位切换）
*   **态度**：传达什么样的情绪或价值观？（例如：严谨专业、轻松幽默、暴躁直接）
*   **视频结构**：内容如何编排？（例如：提出问题->演示过程->展示结果）
*   **核心内容**：具体教授或表达什么？（例如：手机人像技巧、单反风光参数、后期修图流程）

内容定位是一个环环相扣的过程。构思越丰富、越具体，定位就越明确，后续的内容产出也会越容易。

## 案例分析：差异化定位的力量
为了帮助理解，我们来看两个通过精细化定位成功的案例。

![](img/0257ebf9f273d9a0a7fd634fa87980d9_3.png)

![](img/0257ebf9f273d9a0a7fd634fa87980d9_5.png)

### 案例一：暴躁的小峰
这个账号在“态度”这个要素上选择了“暴躁”作为核心特色。即使其脚本模式与他人相似，但鲜明的个人态度使其积累了大量粉丝。这说明了在通用框架下，一个突出的差异化要素就能带来巨大优势。

![](img/0257ebf9f273d9a0a7fd634fa87980d9_7.png)

### 案例二：仙女酵母
这个案例更能说明全面策划的重要性。我们通过一个具体视频来分析。

![](img/0257ebf9f273d9a0a7fd634fa87980d9_9.png)

![](img/0257ebf9f273d9a0a7fd634fa87980d9_3.png)
（视频对话内容：寻找遗忘的包和电话，等待被遗忘的闺蜜。）

单看这个段子内容，它并不新颖，甚至有些普通。但这个视频获得了数百万点赞，账号“仙女酵母”拥有超过525万粉丝。其成功的关键在于精心的整体策划：

*   **身份**：塑造了“公主”或“仙女”的角色。
*   **场景**：设定在具有外国古欧洲风格的场景中。
*   **道具与服装**：使用了华丽的复古服饰和道具。

正是这种在**身份、场景、道具**上的独特打造，构成了其鲜明的人物IP，使得即使内容本身普通，也能在众多账号中脱颖而出。

## 总结
本节课我们一起学习了如何深化抖音账号的内容定位。我们认识到，仅有一个热门主题IP是远远不够的。要想在竞争激烈的环境中胜出，必须对**身份、场景、动作、道具、机位、态度**等多个维度进行具体策划。通过案例“暴躁的小峰”和“仙女酵母”，我们看到了差异化定位和全面人物IP打造的巨大价值。记住，定位越具体，你的账号形象就越清晰，内容生产也越有方向。
