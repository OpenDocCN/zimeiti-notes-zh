## 开店篇 🏪

上一节我们概述了课程内容，本节中我们来看看如何开设小红书店铺。开店是启动小红书电商的第一步，流程非常简单。

![](img/cc42a6d18d4a362ead566257617be014_5.png)

以下是开店的具体步骤：
1.  打开小红书应用。
2.  点击左上角的菜单图标。
3.  选择“创作中心”。
4.  点击“更多服务”选项。
5.  选择“店铺”，点击“立即入住”。
6.  按照官方要求提交资料完成入住。

小红书店铺主要分为个人店和企业店。
*   **个人店**：包括仅需身份证和1000元保证金的“个人店”，以及需要额外提供营业执照的“个体工商户店”。
*   **企业店**：包括普通企业店、专卖店和旗舰店，均需营业执照并缴纳600元年费。

对于初学者，建议直接开通**个人店**。实测表明，平台目前并未对企业店有流量倾斜。个人店与企业店的主要区别在于，一份营业执照可开设三个企业店，而个人身份只能开一家个人店。

开店成本主要包括以下几项：
*   **保证金**：1000元（不做可退）。
*   **平台扣点**：营业额的5%。
*   **支付渠道费**：约每单0.7%。
*   **流动资金**：初期可采用无货源模式，准备2000-3000元即可启动。

## 选品篇 🛍️

成功开店后，下一步就是决定卖什么。选品是电商成功的关键，但方法并不复杂。核心逻辑是：利用其他成熟电商平台的爆款数据来辅助决策，而非凭空想象。

以下是三个推荐的选品渠道：
1.  **某宝/某东**：寻找符合小红书平台调性（如高颜值、新奇特、生活方式类）的爆款产品。已有案例显示，将其他平台的爆款直接搬运至小红书，首篇笔记即可出单。
2.  **某音精选联盟**：查看精选联盟的排行榜，了解当前热卖商品。筛选出符合小红书用户画像的产品进行测试。
3.  **某多多销量排行榜**：选择适合小红书的类目（如家居、饰品、服装），从排行榜中挑选爆品进行测试。

选品的核心公式是：**爆品潜力 = 其他平台验证的爆款 × 与小红书用户的重合度**。我们只需判断排行榜上的商品是否可能匹配小红书用户喜好，然后直接搬运测试。这相当于借助了大卖家的选品能力，是最快捷稳妥的方式。

## 流量获取篇 📈

![](img/cc42a6d18d4a362ead566257617be014_9.png)

选品上架后，便进入最重要的环节——获取流量。小红书本质是种草社区，店铺超过90%的流量都来源于发布的笔记。因此，运营的核心在于内容，而非店铺数量。

小红书不适合“店群”模式（开设大量店铺），而适合“账号矩阵”模式（运营多个账号发布大量笔记）。因为平台的电商自然流量占比低，铺货意义不大。流量通路是：**用户看到笔记 → 对产品产生兴趣 → 进入店铺购买**。多发高质量笔记远比多开店铺有效。

制作有效的种草笔记是关键。直接发布硬广效果很差，必须从**素人（真实用户）视角**切入，进行真实分享才能建立信任、促进转化。

![](img/cc42a6d18d4a362ead566257617be014_11.png)

![](img/cc42a6d18d4a362ead566257617be014_13.png)

制作种草笔记的简单方法是：
1.  **找素材**：去某音、某宝、某东等平台的商品评论区或买家秀中，寻找美观、真实的图片或视频素材。
2.  **改文案**：基于买家的真实评价进行修改和润色。这些评价本身就是用户的使用体验，非常适合作为种草文案。

![](img/cc42a6d18d4a362ead566257617be014_15.png)

以下是制作种草笔记的代码式流程：
```
素材来源 = [某音买家秀， 某宝评价， 某东晒图]
笔记内容 = 精选(素材来源) + 润色(用户评价文案)
发布笔记(笔记内容)
```

![](img/cc42a6d18d4a362ead566257617be014_17.png)

当单个账号的“选品-笔记-出单”流程跑通后，即可通过**批量运营多个小红书账号、批量发布笔记**的方式，为店铺引流，放大收益。
