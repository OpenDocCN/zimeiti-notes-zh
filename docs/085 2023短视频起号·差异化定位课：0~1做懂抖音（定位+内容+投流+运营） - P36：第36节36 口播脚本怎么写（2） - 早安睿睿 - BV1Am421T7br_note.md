在本节课中，我们将学习如何撰写口播类文案的核心部分——分论点。分论点写得好，能让你的内容既简单易懂，又显得有深度和说服力，从而有效提升视频的完播率和转粉率。

上一节我们介绍了口播文案的整体结构，本节中我们来看看如何具体构建有吸引力的分论点。

## 分论点撰写的六大要素

以下是撰写分论点时必须注意的六个核心要素，掌握它们能让你的论述逻辑清晰、引人入胜。

1.  **观点必须配合案例**
    每提出一个分论点，都应配合一个具体案例进行解释。这在专业赛道（如法律、理财、家居）中尤为重要。案例能让抽象的专业观点变得通俗易懂。
    *   **公式**：`分论点 + 案例 = 易于理解的观点`

2.  **观点排序：从猎奇到常规**
    观点的排列顺序应打破用户固有认知。将最新奇、最反常识的观点放在前面，将大众普遍知晓的观点放在后面。这样可以在一开始就抓住用户的好奇心。

3.  **观点排序：悬念置后**
    另一种策略是将用户最好奇的点放在最后说。这在分析热点事件时尤其有效，能制造悬念，吸引用户为了听到最终结论而看完视频，从而提升完播率。

4.  **案例来源：自身经历与大众认知**
    所举案例最好来自两方面：
    *   **自身经历**：能增强IP的专业性和可信度。
    *   **大众认知**：引用历史典故、知名品牌或人物故事，确保观众能快速理解并产生共鸣。

![](img/ce9b24c718a202a12896e8c337cbc4d1_6.png)

![](img/ce9b24c718a202a12896e8c337cbc4d1_8.png)

5.  **学会观点留白**
    无需面面俱到，把所有分论点都说全。可以故意省略一两个大众已知的观点，留给观众在评论区补充。这既能避免内容啰嗦，又能增加互动。
    *   **代码逻辑**：`if (观点 == 大众已知) { 可选择省略 }`

![](img/ce9b24c718a202a12896e8c337cbc4d1_10.png)

![](img/ce9b24c718a202a12896e8c337cbc4d1_12.png)

6.  **避免阐述大众已知观点**
    切忌说“正确的废话”。重复用户已经知道的信息，会显得内容肤浅、啰嗦，导致用户失去兴趣并划走视频。

## 案例解析：高盖伦视频拆解

![](img/ce9b24c718a202a12896e8c337cbc4d1_22.png)

![](img/ce9b24c718a202a12896e8c337cbc4d1_24.png)

我们以博主“高盖伦”一条分析瑞幸咖啡的视频为例，看看上述要素如何应用。

![](img/ce9b24c718a202a12896e8c337cbc4d1_26.png)

![](img/ce9b24c718a202a12896e8c337cbc4d1_28.png)

**视频核心问题**：瑞幸如何将“瑰夏”等名贵咖啡的价格打下来？

![](img/ce9b24c718a202a12896e8c337cbc4d1_30.png)

![](img/ce9b24c718a202a12896e8c337cbc4d1_32.png)

**他的三个分论点**：
1.  **规模优势（药量大）**：瑞幸门店超7000家，通过大宗采购降低咖啡豆成本。
2.  **成本优化**：91%门店为自提店，节省装修与房租成本，不让消费者为环境买单。
3.  **行业本质**：咖啡本是街头饮品，国外品牌抬高了溢价，瑞幸作为国货品牌主动让利。

![](img/ce9b24c718a202a12896e8c337cbc4d1_34.png)

![](img/ce9b24c718a202a12896e8c337cbc4d1_36.png)

**要素应用分析**：
*   **观点+案例**：提出“规模优势”后，立刻用“在埃塞俄比亚采购2000吨咖啡豆”的案例具体说明。
*   **排序从猎奇到常规**：前两点（规模采购、成本控制）是多数用户不知道的“内部信息”，放在前面。第三点（国货让利）是大众相对熟悉的品牌叙事，放在最后。
*   **观点留白**：视频并未提及瑞幸所有的成功因素（如营销策略、数字化系统等），留给懂行的观众讨论，增加了评论区互动空间。
*   **避免正确废话**：没有赘述“瑞幸咖啡便宜”这个人尽皆知的现象，直接切入“为什么便宜”的深度分析。

![](img/ce9b24c718a202a12896e8c337cbc4d1_38.png)

## 实操演练：构建分论点

假设我们的选题是：**为什么中国缺乏百年品牌？**

我们初步构思三个分论点：
1.  历史大环境原因（新中国成立时间尚短，经历社会变革）。
2.  企业家急功近利，注重短期利益。
3.  家族企业继承模式与西方不同。

![](img/ce9b24c718a202a12896e8c337cbc4d1_40.png)

**如何运用六大要素组织内容**：

1.  **为观点配备案例**：
    *   阐述“历史大环境”时，可对比日本、欧美老牌资本主义国家，说明其社会性质未经历彻底变革，企业得以延续。
2.  **优化观点排序**：
    *   将最反常识的“历史大环境”原因放在第一位。
    *   将大众普遍认同的“企业家急功近利”放在第二位或最后。
    *   将“继承模式”作为新的视角放在最后，或作为结尾升华。
3.  **主动留白**：
    *   不提及所有原因（如市场竞争、消费习惯变化等），激发观众在评论区补充讨论。

## 如何精彩结尾与升华 🎯

![](img/ce9b24c718a202a12896e8c337cbc4d1_44.png)

![](img/ce9b24c718a202a12896e8c337cbc4d1_46.png)

总分总结构的“总结”部分容易重复观点，导致用户流失。一个精彩的结尾应能提升完播率或增加互动。

![](img/ce9b24c718a202a12896e8c337cbc4d1_48.png)

![](img/ce9b24c718a202a12896e8c337cbc4d1_50.png)

以下是两种高效的结尾方法：

![](img/ce9b24c718a202a12896e8c337cbc4d1_52.png)

![](img/ce9b24c718a202a12896e8c337cbc4d1_54.png)

1.  **观点后置法**
    将一个有价值的分论点保留，不在正文中展开，而是放在结尾作为升华抛出。
    *   **实操**：在“百年品牌”案例中，将“继承机制”这个点放在结尾详细阐述，给观众“听到新知识”的满足感。

![](img/ce9b24c718a202a12896e8c337cbc4d1_56.png)

2.  **抛出问题法**
    在结尾向观众提出一个开放性问题，引导互动。
    *   **实操**：讲完分论点后，提问：“那么，你们看好目前哪个中国品牌，最有可能成为百年老店呢？” 这能吸引用户看完视频并参与评论。

![](img/ce9b24c718a202a12896e8c337cbc4d1_58.png)
