# 新媒体运营教程：P24：不用会员也可以识别字幕 🎵

![](img/547e0ab651168bb0f9c31985d59630e3_1.png)

在本节课中，我们将学习如何制作一个带有动态歌词效果的音乐视频。这种视频因其沉浸式的体验，能有效吸引观众停留观看，是提升视频互动和收益的有效方法。我们将从导入素材开始，一步步完成歌词识别、动态效果添加以及最终成片的制作。

![](img/547e0ab651168bb0f9c31985d59630e3_3.png)

## 准备工作 🛠️

上一节我们介绍了视频剪辑的基础概念，本节中我们来看看如何具体操作。首先，我们需要准备好剪辑软件和一段视频素材。

1.  打开你的视频剪辑软件，点击“开始创作”。
2.  在素材库中，勾选你准备好的视频文件，然后点击右下角的“添加”按钮，将视频导入到时间轴。
3.  为了专注于背景音乐，我们需要关闭视频自带的原声。点击视频轨道左侧的“关闭原声”按钮。

## 调整视频画面比例 📐

视频添加进来后，我们需要调整其比例以适应平台要求，并优化观看体验。

1.  在下方工具栏中向左滑动，找到并点击“比例”工具。
2.  在弹出的选项中，选择“3:4”的比例。此时，视频画面两侧可能会出现黑边。
3.  使用双指在预览屏幕上 pinch（捏合）来放大视频，直到画面撑满整个屏幕。
4.  你可以轻微拖动视频，调整其显示位置，然后点击对勾（√）确认。

## 添加背景音乐 🎶

画面调整好后，接下来我们为视频配上合适的背景音乐。

1.  在下方工具栏中向右滑动，找到并点击“音频”工具。
2.  点击“音乐”，你可以从推荐歌单中选择，或通过顶部的搜索框查找特定歌曲。例如，搜索“彩云之南”。
3.  试听歌曲，找到想要的版本后，点击右侧的“使用”按钮，音乐就会被添加到时间轴。
4.  将时间轴光标移动到视频结尾处，选中音乐轨道，点击工具栏中的“分割”工具，然后删除视频之后多余的音乐片段。
5.  最后，将时间轴光标移回视频开头。

## 免费识别与整理歌词 ✍️

这是本节课的核心技巧，无需会员即可获得歌词文本。

1.  点击下方工具栏的“返回”按钮，然后找到并点击“文本”工具。
2.  点击“识别歌词”，软件会开始自动匹配。**注意**：直接使用此功能可能需要会员。
3.  识别完成后，点击任意一句歌词，在工具栏中找到“编辑字幕”。
4.  进入编辑界面后，将光标移动到**第二句歌词的开头**，点击键盘或工具栏上的“删除”（×）键，将第二句歌词合并到第一句后面。
5.  重复此操作，**将所有后续歌词的开头都删除**，使所有歌词合并成一大段文本。
6.  点击“全选”，然后选择“拷贝”（或“复制”）。
7.  拷贝完成后，点击右上角的“删除”按钮，将识别出来的原始歌词轨道删除，并点击对勾（√）确认。

## 创建与美化歌词文本 🎨

现在，我们将复制好的大段歌词整理成美观的、分行的样式。

1.  确保时间轴在视频开头，点击“文本” -> “新建文本”。
2.  在输入框内长按，选择“粘贴”，将刚才拷贝的整段歌词粘贴进来。
3.  手动为歌词分行。在每句歌词的末尾（例如“彩云之南”后）点击，然后点击输入键盘右下角的“换行”键。
4.  重复操作，直到所有歌词都单独成行。
5.  点击“字体”，可以选择一个喜欢的字体，例如“江湖体”。
6.  点击“样式”，向上滑动找到“排列”工具。
7.  在排列选项中，调整“行间距”的滑块，让每行歌词之间的间隔看起来舒适。
8.  点击对勾（√）确认。然后，在时间轴上拖动歌词轨道两端的白色边框，使其长度与视频主轨道对齐。

## 制作歌词滚动动画 🎬

静态歌词缺乏动感，接下来我们让它随着音乐滚动起来。

1.  将时间轴光标移动到歌词轨道的**最开头**。
2.  选中歌词轨道，点击工具栏上的“关键帧”按钮（通常是一个菱形图标），添加一个起始关键帧。
3.  在添加了关键帧的这一刻，在预览屏幕上将歌词**向下拖动**，直到画面上只显示最后四行歌词。
4.  将时间轴光标移动到**歌词轨道的末尾**。
5.  在预览屏幕上将歌词**向上拖动**，直到第一行歌词出现在画面顶部附近。此时，软件会自动在末尾生成一个新的关键帧。
6.  现在播放预览，可以看到歌词随着时间从下向上平滑滚动了。

## 添加高级动画效果 ✨

为了让效果更专业，我们可以为歌词添加颜色填充动画。

1.  选中歌词轨道，点击工具栏中的“样式”。
2.  点击“动画”，在入场动画或循环动画中找到“卡拉OK”效果并选择。
3.  选择后，歌词会随着音乐播放，从左到右逐渐填充上颜色。你可以在下方选择喜欢的填充颜色。
4.  点击对勾（√）确认。

## 制作歌词渐隐效果 🌫️

若想实现歌词在滚动时，唱过的部分逐渐淡出的效果，需要用到画中画和蒙版功能。

1.  选中主视频轨道，点击工具栏中的“复制”。
2.  选中复制出来的视频片段，点击工具栏中的“切画中画”。它会自动对齐到下方轨道。
3.  长按这个画中画片段，将其拖动到与主视频开头对齐。
4.  选中这个画中画图层，在工具栏中找到“蒙版”，点击进入。
5.  选择“线性”蒙版。屏幕上会出现一条可旋转的黄线。
6.  将黄线**向上拖动**到靠近画面顶部的位置，只留出一点空间。
7.  拖动黄线旁边的“羽化”（双箭头）图标，轻微拉一下，使边缘过渡更自然。
8.  点击对勾（√）确认蒙版设置。
9.  此时，画中画图层会遮挡住一部分歌词。点击画中画图层，在工具栏中找到“层级”，将其设置为“置顶”。这样，歌词就会在滚动到蒙版上方时逐渐“消失”。

## 添加歌曲标题 🏷️

最后，为视频添加一个漂亮的歌曲标题。

1.  点击两次“返回”，回到主工具栏，点击“文本” -> “新建文本”。
2.  输入歌名，例如“彩云之南”。
3.  点击“样式”，可以更改字体颜色（如黄色）。
4.  点击“动画”，为标题选择一个入场动画，例如“渐显”。
5.  调整动画时长（通常3秒左右），点击对勾（√）确认。
6.  在时间轴上拉长标题轨道，使其与视频等长。
7.  在预览屏幕上，可以双指放大标题，并将其拖放到合适的位置（如画面中央）。

## 总结与成片 🎉

本节课中，我们一起学习了制作动态歌词音乐视频的全流程。从**导入素材、调整比例、添加音乐**，到核心的**免费识别与整理歌词**技巧，再到**创建滚动动画、添加卡拉OK效果**，以及利用**画中画和线性蒙版**制作渐隐高级效果，最后**添加标题**完成制作。

通过这一系列操作，你便能制作出专业且吸引人的音乐视频。完成后，点击导出即可获得成片。不断练习这些步骤，你就能熟练掌握这种流行的视频制作方法。
