# 抖音运营教程：P21：产品与用户兴趣标签 🧩

在本节课中，我们将学习如何利用“九宫格选题法”，结合产品属性与用户兴趣标签，为抖音账号创作源源不断的视频选题。我们将以健身软件Keep为例，详细拆解这一方法。

上一节我们介绍了选题的重要性，本节中我们来看看如何系统地生成优质选题。

## 产品标签分析

首先，我们需要分析产品本身的核心特点。将目标产品（例如Keep）置于中心，围绕它联想其产品属性。

以下是围绕Keep健身软件可以联想到的产品标签：

1.  **线上健身软件**：产品的核心形态。
2.  **专业健身教练**：产品提供的核心服务或资源。
3.  **饮食搭配建议**：产品提供的附加价值。
4.  **减肥/塑形攻略**：产品解决的主要用户需求。
5.  **健身干货知识**：产品提供的教育内容。
6.  **科学运动方法**：产品强调的专业性。
7.  **打卡运动制度**：产品的用户激励与互动机制。
8.  **一二线城市用户为主**：产品的主要用户画像。

这些标签代表了产品自身的功能与定位。分析完成后，我们可以暂时将其放在一边。

## 用户兴趣标签分析

接下来，我们分析使用该产品的用户可能还对哪些内容感兴趣。将核心用户（例如“健身爱好者”）置于另一个九宫格的中心。

![](img/ad3ee8aa7a12ad251f4a3bdf0dbcaaff_1.png)

围绕“健身爱好者”这一核心用户，我们可以联想其关联兴趣：

1.  **健身房**：线下健身场景。
2.  **健身装备**：如运动服、运动鞋等。
3.  **轻脂美食/减肥餐**：与健身目标相关的饮食。
4.  **旅行**：作为热爱生活的一种表现。
5.  **星座/明星**：普遍的娱乐兴趣点。
6.  **鸡汤文**：激励、情感类内容。
7.  **时尚穿搭**：与外形、形象管理相关。
8.  **生活技巧**：提升生活品质的内容。

通过这种方式，我们将用户标签从“健身爱好者”扩展到了更广泛的“热爱生活者”，从而覆盖更潜在的兴趣圈层。

## 运用“连连看”生成选题

现在，我们手头有两组标签：**产品标签**和**用户兴趣标签**。生成选题的关键在于将这两组标签进行创意连接。

以下是进行“连连看”组合的示例：

*   **组合1：饮食搭配 + 旅行**
    *   生成选题示例：《外出旅行时，如何享受美食又能控制体重？》
*   **组合2：运动方法 + 明星**
    *   生成选题示例：《赵丽颖都在坚持的3个居家健身动作，简单又有效！》
*   **组合3：科学 + 健身房**
    *   生成选题示例：《别再瞎练了！用这5个科学方法，让你的健身房效率翻倍。》

通过不同标签的交叉组合，可以衍生出大量选题方向。如果创意枯竭，可以借助搜索引擎，输入组合关键词（如“健身 科学”），参考相关的长尾词或热门话题来激发灵感。

## 建立选题与素材库

这种“连连看”的方法能帮助我们生成无数个选题方向。每一个有效的选题背后，都可以规划成一个具体的视频内容，进而建立起你的视频素材库。这意味着你将拥有持续不断的内容可发布，有效解决“不知道拍什么”的难题。

![](img/ad3ee8aa7a12ad251f4a3bdf0dbcaaff_3.png)

本节课中我们一起学习了“九宫格选题法”。我们首先分别梳理了**产品自身标签**和**目标用户兴趣标签**，然后通过将两类标签进行创意组合（连连看），来批量生成视频选题。掌握这个方法，能帮助你系统化地挖掘内容方向，为运营抖音账号打下坚实的内容基础。
