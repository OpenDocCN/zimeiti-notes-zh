# 抖音运营教程：P12：5.视频剪辑找素材 🎬

在本节课中，我们将学习如何为抖音视频剪辑寻找合适的素材。掌握正确的找素材方法，是制作原创内容、避免违规的关键。

上一节我们介绍了账号定位，本节中我们来看看如何为视频剪辑寻找素材。

![](img/323850f5efde2db3c2a3170168cc4375_1.png)

## 素材来源平台

![](img/323850f5efde2db3c2a3170168cc4375_3.png)

视频剪辑内容广泛，包括电影、电视剧、综艺节目等。制作这类视频，首先需要找到素材。以下是寻找素材的主要途径：

### 1. 主流视频平台
所有主流视频平台都是素材库。下图列出了30多个可用的自媒体平台，均可作为素材来源。

![](img/323850f5efde2db3c2a3170168cc4375_1.png)

### 2. 专业影视网站
第二个途径是专业的影视网站，例如豆瓣电影。这类网站通常提供大量无版权要求的影视素材。我整理了近50个此类网站，后续可提供给大家。

### 3. 综合性素材网站
如果不想只做影视类，还可以寻找其他类型的素材。我整理了60多个网站，涵盖旅游、图片、音效、体育、美食、动漫、综艺、搞笑以及国内外各类视频素材。

### 4. 影视推广平台
第三个是影视推广平台。在抖音或快手发布普通视频主要作用是涨粉，难以直接变现。但使用这些推广平台提供的影视素材，可以通过播放量获得奖金。这个平台稍后会重点讲解。

## 关键注意事项：避免“搬运”

找到素材后，有一个至关重要的原则必须遵守：**不能直接下载整个视频使用**。

很多教程提到的“搬运”行为，即直接下载他人视频发布，现在平台审核非常严格，**95%** 的此类操作都无法通过审核。

正确的找素材方式是：
**从大量视频中，提取符合自己视频主题的片段，每个片段时长控制在 `10秒` 左右。**

![](img/323850f5efde2db3c2a3170168cc4375_13.png)

然后，将这些片段通过剪辑软件重新组合、加工，形成自己的原创内容。这样操作才能有效通过平台审核，避免违规。

素材找到后，就需要开始制作视频。制作视频需要用到剪辑软件。
