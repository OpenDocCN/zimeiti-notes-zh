# 新媒体运营：P29：11：小红书平台用户标签及流量入口 📊

![](img/f458c26a1b628dd4e43e6e759c64a145_1.png)

在本节课中，我们将要学习小红书平台如何为用户打上标签，以及平台主要的流量入口有哪些。理解这些机制，有助于我们分析笔记如何被系统推荐给精准用户，并指导我们优化内容策略。

上一节我们介绍了系统如何判定内容标签，本节中我们来看看平台如何判定用户标签。这个过程能帮助我们理解笔记是如何被系统推荐给用户的，并可以从背后的逻辑反推，如何让笔记被我们的精准用户看到。

## 用户标签判定机制 👤

系统为用户打标签的主要依据是用户的**浏览行为**。用户注册时选择的兴趣范围参考价值不大，它仅作为初期内容方向的引导，无法进入系统深层的用户标签池。系统判定用户兴趣分类的核心依据是动态的浏览行为。

以下是系统分析用户行为并打标签的几个关键维度：

*   **点击率**：系统在发现页向用户推送多种内容（如美妆、家居、母婴）。用户点击哪类内容更多，系统就能计算出其偏好。例如，用户点击10条内容中，美妆类3次，家居类2次，母婴类0次，系统会判定该用户对母婴内容不感兴趣，并持续推送更多美妆内容进行试探，从而打上**强关联**或**弱关联**的标签。
*   **停留时长**：用户对某类内容的停留时间反映了其兴趣深度。例如，用户常看美妆内容但划走得快，说明推送的笔记不精准。若某天系统推送了一条家居内容，用户不仅点击，还仔细阅读图片和评论，停留很久，系统就可能判断该用户有装修或购买家居用品的潜在需求。
*   **页面浏览量（PV）**：指用户浏览页面的深度。例如，用户看完一篇笔记后，又点进博主主页浏览了多篇其他笔记。这个数值（PV）对账号健康度有积极影响。在粉丝基数较小时，提升用户在你主页的浏览深度，能为账号带来非常健康的评分。
*   **互动行为**：包括点赞、收藏、评论、关注。这些数据直接关系到笔记的互动率，是提升笔记进入下一级推荐流量池的关键。互动数据好，不仅笔记能获得更多推荐，也有助于账号接单、变现，并可能带动账号内其他笔记的流量。

## 平台流量分发入口 🚀

了解用户标签后，我们来看看小红书主要的流量分发入口。作为用户可能经常使用，但作为运营者需要分析其区别。

小红书平台主要有四个流量入口：

*   **发现页**：这是笔记和账号获取流量的主要来源。发现页呈现的内容包括：
    1.  已关注达人发布的内容。
    2.  根据用户标签（点击率、停留时长等）匹配的、系统认为用户可能喜欢的**兴趣内容**。
    3.  系统为了平台内容多样化而推送的、用户可能未接触过的**发散性内容**。
*   **关注页**：这里集中展示用户所有已关注博主发布的最新笔记。当关注列表很长时，用户也可能从这里寻找特定博主的内容。
*   **附近页**：基于手机定位，推送同城或附近区域的内容。此入口强地域特征，以美食、本地旅游、线下店铺（如美容院、景点）等内容为主。做线下实体店推广的运营者，需要重点关注此入口及相关的信息流广告设置。
*   **购物页**：这是小红书向电商方向发展的重要入口，直接匹配用户的购物需求。用户习惯在此搜索商品或通过笔记种草后跳转购买。目前，小红书的电商生态仍在发展中，许多用户的购物行为仍与笔记内容紧密关联。
