# 小红书运营教程：P50：05.小红书后台的基础操作（手机端） 📱

![](img/8f268245a649dacd6ec6db6e5393a9e0_0.png)

在本节课中，我们将学习如何使用手机端的小红书商家后台“千帆”系统，完成店铺的基础设置，包括地址管理、运费模板、支付开通和优惠券创建等核心操作。

## 系统入口与界面概览

上一节我们介绍了小红书运营的整体框架，本节中我们来看看如何通过手机进行后台的具体操作。

首先，需要在手机应用商店搜索并下载“小红书千帆”App，这是官方的商家后台管理系统。登录后，您将看到主界面。

主界面通常包含以下模块：
*   **消息**：用于接收客服咨询、平台通知和违规提醒。
*   **首页**：显示店铺状态和待办任务（如“去初始化”）。
*   **全部工具**：集成了店铺运营的所有核心功能入口。

## 核心功能详解

以下是店铺初始化必须完成的几项关键设置。

### 1. 地址与物流设置

填写正确的发货地址和设置合理的运费模板，是店铺运营的第一步。

1.  在“首页”或“全部工具”中找到“地址管理”入口。
2.  点击“新建地址”，填写联系人、所在地区及详细地址。
    *   **公式**：`发货地址 = 联系人信息 + 地区信息 + 详细街道信息`
    *   对于一件代发模式，此地址可填写为您联系的供应商或代发仓库的地址。
3.  关键点：在“留给买家留言”一栏，**务必填写有效的客服联系方式**，以便处理退货问题。若填写“请联系客服”等无效信息，可能导致平台审核不通过。
4.  随后，进入“物流管理”或“运费模板”设置。
    *   您可以创建不同的模板，例如：
        *   **全国包邮模板**：指定地区免运费。
        *   **偏远地区不配送模板**：例如，设置新疆、西藏等地不支持购买。
        *   **买家付邮费模板**：指定地区需由买家承担运费。
    *   系统有默认模板，但建议根据您的商品成本重新配置。

### 2. 支付方式开通

为确保顾客能顺利下单，必须开通店铺的收款功能。

1.  根据平台指引，完成店铺主体认证（如个人店、企业店）。
2.  在支付设置中，依次开通 **微信支付** 和 **支付宝支付**。
3.  按照流程绑定您的实名认证身份证及对应银行卡。
4.  操作完成后，顾客在您店铺即可使用微信或支付宝付款。

### 3. 保证金缴纳

若计划长期运营，缴纳保证金是必要的，它能增加店铺信誉并解锁更多功能。

*   保证金金额根据您经营的**商品类目**不同而有所差异。
*   在后台找到保证金缴纳入口，进行补交即可。

### 4. 营销工具：创建优惠券

优惠券是吸引顾客、提升转化的有效营销工具。

1.  在“全部工具”中找到“营销”或“优惠券”模块。
2.  点击“创建优惠券”或“优惠券活动报名”。
3.  您可以选择创建不同类型：
    *   `店铺通用券`：全店商品可用。
    *   `指定商品券`：仅限特定商品使用。
    *   `店铺新人券`：针对首次进店用户。
4.  创建后可管理“正在生效”、“已失效”的优惠券活动。

## 总结

![](img/8f268245a649dacd6ec6db6e5393a9e0_2.png)

本节课中我们一起学习了小红书千帆后台（手机端）的基础操作。核心步骤包括：下载千帆App、设置发货地址与运费模板、开通微信与支付宝支付、缴纳类目保证金以及创建营销优惠券。这些是店铺能够正常接待顾客、完成交易的基础，请务必逐一完成并熟悉各功能位置。多操作、多摸索，便能快速掌握。
