# 抖音运营实战：P17：16、口播类视频拍摄方法-7 🎬

在本节课中，我们将学习口播类视频的具体拍摄方法。我们将通过一个成功案例，详细拆解其构图、内容定位和拍摄技巧，帮助你掌握制作高质量口播视频的核心要点。

上一节我们介绍了视频拍摄的通用注意事项，本节中我们来看看如何将这些原则应用到具体的口播视频拍摄中。

口播视频的拍摄方法具有通用性。无论你从事金融、装修、养生还是情感领域，其核心拍摄逻辑是相同的。同样，记录日常生活、拍摄段子或美食视频，其方法也基本一致。因此，掌握口播和日常记录这两种类型的拍摄，几乎就能覆盖所有场景。

首先，我们分析第一个类型：口播视频。这里以一位学员“种小草”为例进行说明。

![](img/6f136caf6552b96fe03bf957d0f9292a_1.png)

她的账号名为“种小草”，主打情感口播内容。作为一名40岁、带着三个孩子的宝妈，她的一条视频播放量超过8000万，并借此涨粉30万。为什么她的视频能如此成功？关键在于她精准吸引了目标群体。观看她的视频，你能立刻明白她想吸引的是哪类观众。例如，如果我的目标是20多岁的学生，那么我拍摄的所有内容都应围绕学生的兴趣点，这样涨粉才会又快又精准。

以下是关于构图的具体要点：

*   **平台比例**：抖音是**9:16**的竖屏平台。
*   **内容比例选择**：你需要在竖屏框架内，决定拍摄横屏还是竖屏内容。案例中“种小草”使用的是**4:3**的横屏构图。
*   **比例一致性**：无论选择哪种比例，必须保持统一。不能时而横屏，时而竖屏。选择横屏就全部拍横屏，选择竖屏就全部拍竖屏。保持稳定是重要细节。
*   **人物构图**：人物主体应占据画面中央约**70%** 的面积。避免怼脸拍摄，那样会让观众感到不适。确保画面左、右、上方留有适当空白，使整体看起来清晰、舒适。

如果你认为自己外貌不够出众，是否还能做口播？答案是肯定的。抖音的成功并非完全依赖颜值。口播的核心在于**内容**和**知识**的传递。只要让观众觉得你清晰、体面即可。对于外貌不自信的创作者，可以开启美颜滤镜。在这个时代，合理使用美颜功能能让每个人都呈现更好的状态。请放心使用，只要不过度夸张即可。

接下来，我们直接观看她那条涨粉30万的视频。观看时请思考：这个视频旨在吸引哪个特定群体？

![](img/6f136caf6552b96fe03bf957d0f9292a_3.png)

![](img/6f136caf6552b96fe03bf957d0f9292a_3.png)

视频文案示例：
> “跟婆婆在一起住是什么样的感受？我跟我婆婆在一起住了16年了，平时磕磕碰碰有点小不愉快，那都是正常的……我最大的感受就是这十多年以来，我娘家人很少在我家住上几天……这对于我内心来说，其实是一个很大的缺失。”

有观众提到“婆媳关系”。但这具体指向哪个群体呢？婆媳关系是一种关系，那么谁对此最有感触？男性群体的感受通常不如女性深刻。而未结婚的年轻女孩也难以真正体会。因此，最能产生共鸣的群体是**已婚女性**。她吸引的正是这个精准的已婚群体。我们后续可以通过数据来验证这一点。

从拍摄上看，她使用了美颜功能。如果你对自己的外貌不够满意，也可以适度使用美颜，让自己看起来更得体。她的构图同样是4:3的横屏。

关于拍摄地点，有同学问是否必须户外拍摄。答案是否定的，在家也可以拍摄。关键在于**背景要整洁美观**，避免杂乱难看的背景。

![](img/6f136caf6552b96fe03bf957d0f9292a_5.png)

另一个常见问题是：口播是否需要背稿？虽然视频很短，但记忆力不好背不下来怎么办？解决方案在剪辑环节。你可以借助提词器工具，实现**边看边读边录制**，这比背诵全文要简单得多。

![](img/6f136caf6552b96fe03bf957d0f9292a_5.png)

此外，口播视频中，声音清晰度至关重要。为此，推荐你购买一个**手机收声麦克风**。这种设备价格便宜，仅需十几元，在淘宝、京东等平台都能买到。它能显著提升音频质量。
