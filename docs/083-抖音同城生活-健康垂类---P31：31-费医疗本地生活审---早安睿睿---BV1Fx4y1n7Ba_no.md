# 抖音本地生活运营：083：医疗健康类目审核规则详解

![](img/306e3d2966449997c52ba0cd059a5eb9_0.png)

在本节课中，我们将要学习抖音平台针对“本地生活-医疗健康”类目的核心审核规则。理解这些规则是账号安全运营、避免违规限流或封禁的基础。我们将从平台禁止的行为、内容创作边界以及不同账号的权限差异几个方面进行系统讲解。

## 平台禁止的医疗内容行为

上一节我们概述了课程目标，本节中我们来看看抖音平台明确禁止的几类医疗相关行为。这些规则是所有涉及医疗健康内容的账号都必须遵守的底线。

以下是抖音本地生活医疗创作规则中明确禁止的核心要点：

![](img/306e3d2966449997c52ba0cd059a5eb9_2.png)

1.  **禁止使用医护人员形象进行专业科普**：商家账号（如医美、口腔机构）不得使用自有医生、护士等专业人员的形象进行疾病科普、知识普及或变相问诊。平台允许销售特定产品（如洁牙套餐），但禁止以专业身份进行医疗内容输出。
2.  **禁止涉及具体医疗知识**：内容中不得出现关于疾病预防、诊断、改善、治疗或所谓“小妙招”的知识。例如，推广洁牙服务时，不能通过描述“不洁牙会导致牙周炎”等具体病症来制造恐惧，进而推销产品。
3.  **禁止暗示产品具有治疗功能**：不能将所售商品（如牙膏、仪器）与疾病治疗功能不当关联。平台审核的关键并非某个特定词汇，而是直播或内容中整体的**行为**和**倾向**。

## 内容创作中的敏感红线

了解了基本禁令后，我们需要进一步关注内容创作中容易触碰的敏感红线。这些规则旨在维护健康的平台环境，避免对用户造成误导。

以下是本地生活医疗创作规则中需要特别注意的几点：

1.  **禁止制造容貌与健康焦虑**：内容不得将容貌不佳、身体不适与负面评价关联，也不得将容貌出众、身体强健与积极评价不当关联。例如，医美、口腔正畸类账号，应避免使用“术前丑”和“术后美”的强烈对比图来制造焦虑，即使账号已报备“白名单”，此类行为仍可能导致处罚。
2.  **必须明确“运动不能替代诊疗”**：若内容涉及康复、健体等专业领域，必须明确告知消费者“健身运动不可代替医院诊疗，身体不适请及时就医”。这在运动康复、产后修复等领域尤为重要。
3.  **禁止贬低同行进行对比**：不得在内容中与其他商家、商品或医疗方案进行对比，并通过贬低他人来抬高自己。这在整形等医疗方案推广中是隐形的违规行为。

## 医疗科普与商品推广的权限边界

![](img/306e3d2966449997c52ba0cd059a5eb9_4.png)

上一节我们介绍了内容创作的禁区，本节中我们来看看不同账号的核心权限差异，这是理解抖音医疗健康内容审核的**关键**。

平台对内容的管控核心在于区分 **“医疗科普”** 与 **“商品推广”**。

*   **医疗科普权限**：仅授予具备医疗豁免权的**黄V（个人专业认证）** 或**蓝V（机构认证）** 账号。这些账号可以基于权威医学指南进行疾病知识科普和健康建议。
*   **商品推广权限**：本地生活报备“白名单”的商家账号，**仅获得推广特定备案商品（如洁牙、热玛吉）的权限，并未获得医疗科普权限**。

**公式：商家账号权限 = 商品A + 商品B + ... ≠ 医疗科普**

这意味着，作为商家，你的账号可以讲解“洁牙服务是什么、流程如何、体验怎样”，但不能以医生口吻诊断用户“你有牙周炎，需要来治疗”。你可以做“健康科普”（如“定期洗牙的好处”），但不能做“医疗科普”（如“地包天必须做正畸”）。理解这一底层逻辑，是规划合规内容、筛选合作达人的核心。
