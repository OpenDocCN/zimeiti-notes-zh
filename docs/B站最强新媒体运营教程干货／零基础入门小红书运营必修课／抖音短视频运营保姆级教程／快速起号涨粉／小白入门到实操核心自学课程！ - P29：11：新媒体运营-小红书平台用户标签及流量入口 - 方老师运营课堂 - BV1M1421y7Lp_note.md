## 用户标签判定机制

系统为用户打标签的主要依据是用户的**浏览行为**。用户刚注册时选择的兴趣范围，参考价值不大，它仅作为平台对用户内容方向的初步引导，无法进入系统最深层的用户标签池。

系统判定用户兴趣分类的核心依据是用户的浏览行为。以下是系统分析用户行为的主要维度：

以下是判定用户标签的几个关键行为指标：

1.  **点击率**
    *   **公式**：`点击率 = (点击某类内容的次数) / (系统推送的该类内容总次数)`
    *   系统在发现页为用户推送多种内容（如美妆、家居、母婴）。用户点击哪类内容更多，系统就能计算出其偏好。例如，用户点击10条内容，其中美妆3次、家居2次、母婴0次，系统会判定该用户对母婴内容不感兴趣，并持续推送更多美妆内容进行试探。

2.  **停留时长**
    *   用户点开笔记后的观看时长。例如，用户通常快速划走美妆笔记，但某天对一条家居笔记停留了很久，仔细阅读图片和评论，系统可能推断该用户近期有装修或购买家居用品的需求。

3.  **页面浏览量（PV）**
    *   用户浏览的页面深度。例如，用户不仅看了某篇笔记，还点进博主主页，并连续看了第二、第三篇笔记。提升用户在自己主页的浏览深度，对账号健康度有积极影响。

4.  **互动行为**
    *   包括点赞、收藏、评论、关注。这些数据直接提升笔记的互动率，有助于笔记进入更大的推荐流量池，并可能带动账号内其他笔记的流量。

通过以上行为，系统会为用户建立一个标签云图，其中关联性强的标签（如“美妆”）代表用户的核心兴趣，系统会据此主要推送相关领域的内容。

## 平台流量分发入口

了解用户标签后，我们来看小红书主要的流量分发入口。平台通过不同入口满足用户多样化的需求。

小红书目前主要有四个流量分发入口：

以下是四个核心流量入口及其特点：

1.  **发现页**
    *   **主要包含**：已关注达人的内容 + 系统根据用户标签推荐的兴趣内容 + 少量平台推荐的发散性内容（用于探索用户潜在兴趣，增加内容多样性）。
    *   **核心价值**：这是笔记和账号获取流量的最主要来源。

2.  **关注页**
    *   **主要包含**：用户所关注的所有达人发布的笔记。
    *   用户通常直接从此处查看强关注对象的内容。

3.  **附近页**
    *   **主要包含**：根据手机定位推送的附近及同城内容，以美食、旅游、本地服务等强地域特征内容为主。
    *   **适用场景**：非常适合线下实体店（如美容院、餐厅）进行本地推广。在做信息流广告时，可侧重附近人群的定向投放。

4.  **购物页**
    *   **主要包含**：小红书商城及电商相关内容，是平台变现和电商发展的核心入口。
    *   **现状**：部分用户尚未形成在此直接购物的强习惯，更多仍倾向于先搜索相关种草笔记。
