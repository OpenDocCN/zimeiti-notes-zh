![](img/ccea4f57888bb68955eb70e5424282b4_1.png)

在TikTok上销售商品并产生订单后，发货是核心环节。目前，主要的物流模式可以根据销售渠道进行划分。

![](img/ccea4f57888bb68955eb70e5424282b4_3.png)

1.  **TikTok小店官方物流**：适用于在TikTok平台内开设小店并出单的卖家。
2.  **独立站/私域物流**：适用于将流量引导至独立网站或私域进行成交的卖家。此模式又可细分为：
    *   国内小包直发
    *   海外仓发货

## TikTok小店官方物流

上一节我们概述了主要物流模式，本节中我们来看看第一种——TikTok小店官方物流。

如果你在TikTok小店出单，平台一般推荐使用其官方物流服务。

**运作流程**：
1.  卖家在TikTok小店产生订单。
2.  使用TikTok官方物流发货。
3.  官方物流合作商在指定城市（如上海、杭州、义乌、深圳、东莞）提供上门揽收服务。
4.  物流商将包裹直接配送至海外买家。

**核心优势**：
*   **便捷**：支持多个城市上门揽收。
*   **有补贴**：平台会对订单提供物流补贴，具体金额取决于用户的订单金额。

**公式/代码示例**：
```
物流总成本 = 标准运费 - 平台补贴金额
```

## 独立站/私域物流方案

对于通过独立站或私域流量出单的卖家，物流需要自行解决。主要有以下两种方案。

### 方案一：国内小包直发

国内小包直发，通常通过货代公司完成，适合订单量不大或刚起步的卖家。

以下是其标准操作流程：

1.  **导出订单**：从你的独立站后台导出海外买家的订单信息。
2.  **贴单打包**：将订单信息贴在对应的商品包裹上。
3.  **国内段运输**：使用任意国内快递，将包裹寄送至深圳等地的货代公司。
4.  **国际段运输**：货代公司收到包裹后，负责完成清关并发送至目的地国家。

**特点**：
*   **灵活**：无需囤货，一单一发。
*   **成本**：运费通常在几十到几百元人民币不等，取决于重量和目的地。

![](img/ccea4f57888bb68955eb70e5424282b4_5.png)

### 方案二：海外仓模式

如果你的业务稳定、订单量大，海外仓模式可以显著提升物流效率与客户体验。

![](img/ccea4f57888bb68955eb70e5424282b4_7.png)

**运作流程**：
1.  **预存货品**：将一批商品提前通过海运/空运等方式，运送至目标国家的第三方仓库。
2.  **接收订单**：独立站或私域产生销售订单。
3.  **本地发货**：海外仓服务商根据订单，直接从当地仓库完成拣货、打包和配送。

**核心优势**：
*   **时效快**：从买家所在国本地发货，配送时间短。
*   **物流成本低**：后段本地配送费用远低于跨国直邮。

**注意事项**：
*   **适合对象**：适合跨境经验丰富、出单率稳定的卖家。
*   **资金压力**：需要提前备货，占用较多资金，不建议新人贸然尝试。

## 其他特殊物流模式

除了上述主流方案，还有两种特殊的物流模式可供选择。

### 模式一：云仓

云仓是一种全球分布式仓储网络解决方案。

**运作原理**：
卖家将商品存储在云仓服务商的全球多个仓库中。一旦产生订单，系统会自动指派离买家最近的仓库进行发货。

**优势**：
*   **免自建仓**：无需自己管理和租赁海外仓库。
*   **智能化**：实现自动分仓、就近配送。
*   **成本**：通常每单会收取几毛到几块钱的服务费。

### 模式二：一件代发

一件代发是零库存、低风险的入门方式。

**运作流程**：
1.  卖家自身无货源。
2.  从支持一件代发的供应商网站选择商品。
3.  将商品信息上架到自己的TikTok或独立站。
4.  产生订单后，在代发网站下单，并填写最终客户的收货信息。
5.  供应商直接发货给终端客户。

**本质**：赚取**零售价**与**代发成本价**之间的差价。

![](img/ccea4f57888bb68955eb70e5424282b4_9.png)

**公式/代码示例**：
```
单笔利润 = 顾客支付零售价 - 代发网站采购价 - 平台费用
```
