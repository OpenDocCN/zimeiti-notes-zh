# 抖音全套运营教程：P4：账号定位 🎯

![](img/ca0baf1175e624132e17a825b2d7733c_0.png)

在本节课中，我们将要学习抖音账号定位的核心方法。账号定位是运营的基石，它决定了内容的方向和吸引的粉丝群体。一个清晰的定位能帮助你在庞大的抖音生态中脱颖而出。

![](img/ca0baf1175e624132e17a825b2d7733c_2.png)

## 你是谁？—— 确定账号类型与主题

上一节我们介绍了赛道选择，本节中我们来看看如何进一步细化账号定位。首先需要反思三个核心问题：你是谁，你是干什么的，以及为什么要关注你。

首先，我们要确定所做账号的类型。除了选择一级垂类（大类赛道），还需思考其下的细分赛道。这就像参加运动会，你报的项目是“田径”（一级垂类），而你最擅长且有优势的是“百米赛跑”（二级垂类）。通过定位账号的细分类目，可以确定账号的垂直类型。

在抖音中，无论任何行业，只要定位足够清晰，都有机会成功。因为抖音流量巨大，一旦做起来，增长和裂变速度会非常快。

其次，要确定账号主题是谁。账号主题即账号的拥有人和最终呈现给粉丝的主人公，目的是让粉丝明确知道在看谁的作品。在抖音商业类账号中，账号主题主要分为两类：

以下是两种主要的账号主题类型：
*   **商家账号**：主题是公司。内容需偏商业化，并不断通过内容拉近与粉丝的距离，避免严肃。这样既能建立信任感，也能带来好感，更符合抖音的娱乐属性。
*   **达人账号**：主题是个人。首要解决的是信任问题，可以通过IP打造的方式，为账号树立一个积极正面的人设。

## 你是干什么的？—— 明确账号内容与价值

明确了“是谁”之后，接下来要解决“干什么”的问题。最直接的方式是通过抖音认证身份，即表明职业。

以下是两种认证方式及其作用：
*   **黄V认证**：适用于达人。认证后粉丝可以清晰看到认证标签，明白你的职业和账号内容方向。
*   **蓝V认证**：适用于公司。同样能为账号提供清晰的职业标识。

无论是商家认证蓝V还是达人认证黄V，都能获取平台背书，增加信任感。在未认证前，也可以通过账号设置向粉丝明确传达业务信息。

以下是传达业务信息的设置要点：
*   头像
*   背景图
*   简介
*   视频封面

应对这些元素进行与业务关联的统一设置。

## 为什么要关注你？—— 提供核心价值

最后，需要花时间思考：你将为粉丝提供什么样的价值？在抖音中，粉丝对博主的价值需求主要分为以下几类。

以下是粉丝关注的五类核心价值：

**A. 知识价值**
抖音中的知识一般分为几种类型。认知型如人生思考、思维方式、读书分享、行业内幕、成功经验、个人提升等。干货型如技能提升、经验总结、过程步骤、冷知识等。常识普及型如生活、健康、产品、科学、历史、地区等常识的解读和普及。
这类账号提供的价值是 **认同**，如三观认同、观点认同，减少知识盲区带来的获得感，进而产生自我认同。

**B. 情绪价值**
情绪是影响观众的重要因素。作品中观众容易接受的情绪价值类型包括：开心愉悦（如搞笑、夸张表情）、感染共鸣（如情侣、婆媳、家庭、职场等人物关系）、情绪波动（由争议话题、反转反差内容激发），以及怀旧、emo、震惊、好奇等。
这类账号提供的价值是 **共鸣**，即利用表情、文字、画面、声音、肢体动作等准确到位地抒发情绪，直接感染观众。

**C. 感觉价值**
博主通过短视频的画面、文字和声音传递信息。其提供的价值还包括视频的画面美感、构图，以及给观众带来的喜欢、治愈、视觉冲击感，或个人魅力带来的仰慕感。声音方面则能带来节奏感、心跳加速或听觉享受。例如颜值、萌宠、美景、才艺、明星、文化类账号。
这类账号提供的价值是 **共情**，即因表情、文字、画面、声音、肢体动作的自然表现，让人产生联想，并在情绪、认知等意识层面与作者产生连接，从而间接引发共鸣。

**D. 体验价值**
抖音让用户看到了不一样的生活方式，了解到去不了的地方，体验到未曾经历的事物。例如流量很大的美食账号（享用与制作体验）、旅行、运动、角色体验等。
这类账号提供的价值主要是 **好奇心的满足**，以及紧张感、激动、感动、体验获得感。

**E. 期待价值**
当用户被你的短视频内容、观看体验或对作者本人的好奇所吸引，看完后会产生意犹未尽的感觉，从而点击关注，期待下次更新。许多博主会利用这一点，不将所有内容在一个视频中呈现完，以制造持续吸引力。

---

本节课中我们一起学习了抖音账号定位的完整框架。你需要依次明确“你是谁”（账号类型与主题）、“你是干什么的”（内容与认证）以及“为什么要关注你”（提供的核心价值）。一个精准的定位是账号成功的起点，它能帮助你更高效地创作内容、吸引目标粉丝并实现持续增长。
