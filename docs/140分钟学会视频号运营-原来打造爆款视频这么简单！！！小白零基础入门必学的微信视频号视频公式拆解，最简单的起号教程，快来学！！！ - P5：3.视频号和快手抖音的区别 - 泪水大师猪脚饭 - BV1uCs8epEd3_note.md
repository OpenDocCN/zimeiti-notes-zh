# 140分钟学会视频号运营：P5：3. 视频号和快手抖音的区别 🆚

在本节课中，我们将要学习视频号与抖音、快手这两个主流短视频平台的核心区别。我们将从用户数据、创作者生态、平台扶持和未来机遇等多个维度进行对比分析，帮助你理解为何视频号是当前值得投入的新兴平台。

上一节我们介绍了视频号的基础概念，本节中我们来看看它与抖音、快手的核心差异。

## 平台数据对比：用户规模与发展速度

我们既然将视频号与抖音和快手对标，那么它有哪些优势？我们为何不选择抖音和快手，而选择视频号这个平台？

![](img/725be500599c6ae3d4c4ce8c0e0e759c_1.png)

我们从三个维度进行分析。第一个是日活跃用户数（DNU）。视频号目前的日活用户是5亿，抖音是6.5亿，快手是3.2亿。抖音达到这个数值，整整花了5到6年时间。它从2016年开始发展，至今日活用户达到这个规模用了数年。但视频号在短短一年半的时间里，日活用户就远超快手，直追抖音。这说明其发展速度非常快。

![](img/725be500599c6ae3d4c4ce8c0e0e759c_2.png)

## 创作者生态：竞争环境的差异

接下来，我们看一下创作者数量，这个对比非常重要。以下是三个平台的创作者情况对比：

*   **视频号**：创作者数量相对较少。
*   **抖音 & 快手**：创作者数量庞大，接近2亿。

![](img/725be500599c6ae3d4c4ce8c0e0e759c_1.png)
![](img/725be500599c6ae3d4c4ce8c0e0e759c_2.png)

正如数据所示，视频号的创作者竞争远没有抖音、快手激烈。在抖音和快手，人才众多，如果你没有独特的技术或突出优势，很难与他们竞争。同时，视频号目前的商业化程度也相对较低，但未来必然会向商业化程度高的方向发展。

![](img/725be500599c6ae3d4c4ce8c0e0e759c_4.png)

## 平台扶持力度：流量红利期

![](img/725be500599c6ae3d4c4ce8c0e0e759c_5.png)

腾讯内部有相关消息称，计划在2022年下半年开启视频号的商业化进程。这意味着平台目前的扶持力度非常高。

如果你经常刷视频号，可能会发现一个现象：一些曾经在抖音或其他平台火过的段子，在视频号上再发一遍，有可能再次爆火，甚至获得比原平台更高的播放量。这就是平台流量扶持的体现。

## 选择视频号的核心原因

基于以上分析，我们现在总结为何要选择做视频号。

![](img/725be500599c6ae3d4c4ce8c0e0e759c_13.png)

**第一，平台有明确的流量扶持。** 并且现在正是入局的好时机。官方发布了“创作者激励计划”，推行“三有政策”扶持创作者。

![](img/725be500599c6ae3d4c4ce8c0e0e759c_7.png)
![](img/725be500599c6ae3d4c4ce8c0e0e759c_8.png)
![](img/725be500599c6ae3d4c4ce8c0e0e759c_9.png)
![](img/725be500599c6ae3d4c4ce8c0e0e759c_10.png)
![](img/725be500599c6ae3d4c4ce8c0e0e759c_11.png)

“三有政策”是指：未来一年，平台将通过**流量扶持**、**专项奖励**和**全生命周期成长权益体系**，让创作者实现**有流量**、**有收入**、**有成长**。这为零基础的新人提供了极大的支持。

**第二，视频号具备先天优势。** 它背靠微信生态，天生自带流量，可以视为“生下来就自带buff加成”。而抖音和快手是从零开始发展起来的。

![](img/725be500599c6ae3d4c4ce8c0e0e759c_13.png)

**第三，当前是入局的黄金窗口期。** 抖音和快手的内容生态已相对成熟，可见天花板。而视频号即将在下半年开启商业化，意味着将有大资本涌入。目前，视频号上少有顶级流量账号，几十、几百播放量的内容很常见。这意味着现在起号、做号更加容易。一旦你的账号做起来，掌握了流量密码，自然就能与品牌、资本合作，通过推广广告赚取红利。

## 历史机遇与未来展望

现在做视频号，相当于5年前做抖音。回想早期通过抖音短视频赚钱的人非常多。例如2018年双十一，薇娅单场带货销售额近2.67亿。虽然她现在已退出舞台，但这个数字足以说明早期红利的力量。单场直播，薇娅能赚200多万，李佳琦能赚300多万。他们固然优秀，但更重要的是踩中了时代的红利。

那么，视频号到底值不值得做？它的未来发展趋势肉眼可见。
