# 自媒体教程：P10：9.收益结算 💰

在本节课中，我们将要学习影视推广项目的收益结算方式，包括结算周期、计算规则以及如何通过多平台发布获取双份收益。我们还将通过一个实际案例，了解具体的收益构成和操作中的注意事项。

## 结算周期与规则

![](img/e0184819ab967b9127a0e9d0ef010ce5_3.png)

![](img/e0184819ab967b9127a0e9d0ef010ce5_5.png)

上一节我们介绍了影视推广的平台选择，本节中我们来看看收益是如何结算的。影视推广的收益结算为**按月结算**，即每月月底进行一次结算。

![](img/e0184819ab967b9127a0e9d0ef010ce5_7.png)

收益的计算基础是**播放量**。每个短剧推广任务的具体要求不同，必须严格按照任务方的要求执行。

![](img/e0184819ab967b9127a0e9d0ef010ce5_9.png)

![](img/e0184819ab967b9127a0e9d0ef010ce5_11.png)

## 获取双份收益的方法

![](img/e0184819ab967b9127a0e9d0ef010ce5_13.png)

除了在指定的四个平台（如抖音、快手等）发布视频获取播放量收益外，创作者还可以通过添加解说，将同一个视频素材发布到其他平台，从而获取**双份收益**。

![](img/e0184819ab967b9127a0e9d0ef010ce5_15.png)

![](img/e0184819ab967b9127a0e9d0ef010ce5_17.png)

![](img/e0184819ab967b9127a0e9d0ef010ce5_19.png)

这意味着，在完成基础推广任务的同时，你还可以利用视频的**版权**，制作影视解说内容，在其他渠道获得额外收入。

![](img/e0184819ab967b9127a0e9d0ef010ce5_21.png)

## 实战案例解析

![](img/e0184819ab967b9127a0e9d0ef010ce5_23.png)

![](img/e0184819ab967b9127a0e9d0ef010ce5_25.png)

以下是一个学员的实际操作案例，通过这个案例我们可以更直观地理解收益构成。

![](img/e0184819ab967b9127a0e9d0ef010ce5_27.png)

![](img/e0184819ab967b9127a0e9d0ef010ce5_29.png)

这位学员最初希望通过抖音积累粉丝，为日后带货做准备，因此选择了影视推广作为起点。

![](img/e0184819ab967b9127a0e9d0ef010ce5_31.png)

![](img/e0184819ab967b9127a0e9d0ef010ce5_33.png)

![](img/e0184819ab967b9127a0e9d0ef010ce5_35.png)

**操作流程如下：**
1.  每次制作视频前，必须先通过官方渠道报名并填写相关信息。
2.  视频制作完成后，必须及时回填链接，并准确填写发布视频的**账号ID**。这是结算的前提，信息错误将无法获得收益。
3.  必须严格遵守每个短剧任务对发布视频数量的要求。例如，任务要求发布3个视频，就必须发布3个；要求10个，就必须发布10个。未按要求操作可能导致视频被下架处理。

![](img/e0184819ab967b9127a0e9d0ef010ce5_37.png)

![](img/e0184819ab967b9127a0e9d0ef010ce5_39.png)

**该学员的收益详情：**
*   **短剧A**：视频播放量达到186万。基础结算约为500元。此外，规则规定每增加1万播放量，额外奖励10元。
*   **短剧B**：播放量为1.4万。虽然播放量较低，但该任务的单价较高，结算了150元。
*   **短剧C**：任务要求同时在抖音和快手发布，但该学员只发布了抖音，未发布快手。因此，最终收益被扣除一半，仅结算120元。
*   **短剧D**：任务单价较低，最终结算45元。

![](img/e0184819ab967b9127a0e9d0ef010ce5_41.png)

**总结**：在一周时间内，该学员共完成了4个短剧任务，总收益为 `500 + 60 + 150 + 120 + 45 = 895` 元。收益的多少取决于**任务单价**和**实际完成的播放量**。时间充裕的创作者可以同时进行多个任务。

![](img/e0184819ab967b9127a0e9d0ef010ce5_43.png)

**重要提示**：所有收益后期均通过对公账户结算。并且，视频发布前必须经过审核，审核通过后方可发布。未经审核发布的视频将被下架，且无法获得收益。

![](img/e0184819ab967b9127a0e9d0ef010ce5_45.png)

## 常见问题解答

![](img/e0184819ab967b9127a0e9d0ef010ce5_47.png)

很多同学会有疑问：为什么制片方不自己发广告，而要让我们来做推广？

![](img/e0184819ab967b9127a0e9d0ef010ce5_49.png)

这涉及到一个成本问题。制片方如果自行投放广告，费用极其高昂。

以下是传统广告与影视推广的成本对比：
*   **传统广告**：单次投放费用，低则10万元左右，高则100万元起步。
*   **影视推广**：将任务分发给大量创作者，按播放量结算奖金。总成本远低于传统广告。

因此，制片方选择影视推广模式，是因为可以利用创作者群体的力量，实现更广的扩散度和更高的性价比。尤其在当前环境下，线上推广已成为主流选择。这对我们创作者而言，是一个很好的机会。

![](img/e0184819ab967b9127a0e9d0ef010ce5_51.png)
