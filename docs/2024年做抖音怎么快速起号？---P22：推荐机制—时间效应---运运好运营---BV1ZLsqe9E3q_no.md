# 抖音运营教程：P22：推荐机制—时间效应

![](img/126786b486163ed0eaded94f924eee9a_1.png)

在本节课中，我们将要学习抖音推荐机制中的第三个重要概念——时间效应。理解这个概念，有助于你明白为何有些视频会在发布很久之后突然获得流量。

上一节我们介绍了基础推荐和叠加推荐，本节中我们来看看推荐机制中的“时间效应”。

![](img/126786b486163ed0eaded94f924eee9a_1.png)

## 什么是时间效应？

时间效应是指，一个视频在发布后，可能不会立即获得大量推荐或播放量，但在未来某个时间点（例如一周、一个月甚至更久之后），其播放量会突然上涨。

这种现象的原因在于抖音的算法机制。抖音平台拥有一个庞大的**视频库**，所有用户发布的视频都会被存储在其中。

## 时间效应如何运作？

以下是时间效应的运作流程：

1.  **视频入库**：当你发布一个视频后，它就被存入抖音的**视频库**中。
2.  **初始沉寂**：如果视频发布时没有蹭到热点，内容也不算特别突出，它可能就在视频库中“沉寂”下来，初始流量不高。
3.  **热点触发**：当未来出现新的热点事件或话题时，抖音的算法会主动从**视频库**中搜索与当前热点相匹配的“旧视频”。
4.  **二次推荐**：算法将这些匹配上的“旧视频”重新提取出来，投入新一轮的推荐流程（即再次经历基础推荐和叠加推荐），从而使其播放量在发布很久后出现增长。

这个过程类似于网络论坛中的“挖坟”或“顶帖”行为，即通过算法发现热点，重新推荐历史内容。

## 一个简单的例子

假设你发布了一个关于明星A的视频，但发布当天反响平平。一周后，明星A因为某事件（如发布新作品、出现新闻）热度飙升。此时，抖音算法可能会从**视频库**中找到你之前发布的那个相关视频，并将其推荐给对“明星A”当前热点感兴趣的用户，从而使你的旧视频播放量上涨。

## 与其他平台的区别

需要特别强调的是，时间效应是**抖音平台特有**的推荐机制。

以下是抖音与另一主流短视频平台快手的核心区别：

*   **抖音**：采用 **`时间效应`** 机制。旧视频有机会因新的热点被重新推荐。
*   **快手**：采用 **`择新去旧`** 机制。平台主要推荐新发布的视频，旧视频（如发布一周或一个月后）通常不会再获得系统推荐，即使出现相关热点。
