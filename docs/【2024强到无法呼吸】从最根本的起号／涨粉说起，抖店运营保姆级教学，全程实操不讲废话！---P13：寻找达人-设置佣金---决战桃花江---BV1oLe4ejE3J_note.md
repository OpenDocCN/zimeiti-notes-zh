# 抖店运营：P13：寻找达人-设置佣金 🎯

![](img/682b7a7a94ef36c2d7fa3fee9937d0f0_1.png)

在本节课中，我们将学习如何主动寻找适合带货的抖音达人，以及如何通过设置合理的佣金来吸引达人合作。我们将介绍三种主要的寻找达人方法，并详细讲解如何筛选达人、如何与达人沟通以及如何设置佣金策略。

![](img/682b7a7a94ef36c2d7fa3fee9937d0f0_3.png)

## 主动寻找达人的三种方法

上一节我们介绍了达人合作的重要性，本节中我们来看看如何主动出击，寻找合适的带货达人。主要有以下三种途径。

以下是三种寻找达人的核心方法：

1.  **抖音平台搜索**：通过抖音创作者服务中心的商品橱窗，可以找到基于昨日商品分享榜排名的达人排行榜。
2.  **抖店后台达人广场**：在抖店营销中心的精选联盟里，系统会根据你的店铺经营范围，自动匹配约1000位潜在合作达人。
3.  **第三方数据平台**：利用如飞瓜数据、蝉妈妈等工具，可以查看更详细的达人榜单、带货数据和粉丝画像。

![](img/682b7a7a94ef36c2d7fa3fee9937d0f0_5.png)

## 方法一：抖音平台搜索

![](img/682b7a7a94ef36c2d7fa3fee9937d0f0_7.png)

首先，我们来看如何在抖音App内直接寻找达人。

具体操作路径为：打开抖音 -> 进入“创作者服务中心” -> 点击“商品橱窗” -> 向下滑动找到“排行榜”。这个榜单展示了昨日商品分享表现优异的达人，你可以根据品类相关性进行初步筛选。

![](img/682b7a7a94ef36c2d7fa3fee9937d0f0_9.png)

![](img/682b7a7a94ef36c2d7fa3fee9937d0f0_11.png)

## 方法二：抖店后台达人广场

![](img/682b7a7a94ef36c2d7fa3fee9937d0f0_13.png)

接下来，我们进入更精准的官方渠道——抖店后台的达人广场。

操作路径为：抖店后台 -> 营销中心 -> 精选联盟 -> 达人广场。进入后，你可以看到系统推荐的达人列表。这里支持手动筛选，例如选择你店铺所属的**一级品类**，并区分**直播达人**与**短视频达人**。

![](img/682b7a7a94ef36c2d7fa3fee9937d0f0_15.png)

选中一位达人后，点击进入详情页，需要重点查看以下数据：
*   **带货数据**：带货场次、直播天数、推广商品数、合作店铺数、场均销售额、转化率。
*   **系统匹配度**：系统根据**粉丝匹配度**、**商品匹配度**、**历史合作匹配度**三个维度给出的综合评分，分数越高越匹配。
*   **粉丝分析**：粉丝的性别、年龄、地域、消费偏好、客单价区间。
*   **带货分析**：历史带货的品类和品牌。
*   **历史作品**：了解其内容风格。
*   **相似达人**：系统推荐的同类型达人。

若确定合作意向，可以点击“发送邀约”或通过抖音后台私信联系达人。

![](img/682b7a7a94ef36c2d7fa3fee9937d0f0_17.png)

## 方法三：第三方数据平台（以飞瓜、蝉妈妈为例）

最后，我们借助更强大的第三方工具进行深度筛选。这里以飞瓜数据为例进行演示。

在飞瓜数据后台，找到“小店达人榜”，这里可以看到更详细的昨日商品分享榜和直播分享榜。除了看榜单，还可以通过“商品库”功能，搜索与你产品类似的爆款商品，查看是哪些达人在带货，从而找到潜在合作对象。

## 如何筛选合适的达人

找到大量达人后，我们需要通过三个核心维度进行筛选，以确保合作效果。

![](img/682b7a7a94ef36c2d7fa3fee9937d0f0_19.png)

以下是筛选达人时必须考察的三个维度：

![](img/682b7a7a94ef36c2d7fa3fee9937d0f0_21.png)

1.  **达人带货类型与类目相关性**：确保达人历史带货品类与你的产品类目高度相关。
2.  **粉丝画像与产品匹配度**：分析达人粉丝的性别、年龄、消费偏好和价格承受区间，是否与你的目标客户群一致。
3.  **带货能力（GMV）**：查看达人近期的场均销售额、总销量等数据，评估其真实的带货实力。

![](img/682b7a7a94ef36c2d7fa3fee9937d0f0_23.png)

**以飞瓜数据为例，查看带货类型与粉丝画像：**
在“播主搜索”中，可按行业或带货品类筛选。进入达人详情页后，在“粉丝分析”部分查看“粉丝消费偏好”和“价格区间分布”。在“带货分析”部分查看“近30天预估销售额”等数据。

## 与达人合作及跟进流程

![](img/682b7a7a94ef36c2d7fa3fee9937d0f0_25.png)

与达人初步接洽后，合作流程通常分为以下几个步骤：

1.  **寄送样品**：将产品样品寄给达人，并提醒其查收。
2.  **跟进内容发布**：通过数据平台（如飞瓜）或抖店后台的“电商罗盘”，监测达人是否发布带货短视频或开设带货直播。
3.  **检查内容质量**：亲自进入直播间观察带货氛围，或查看短视频的数据表现（如播放量、转化率）。

## 与达人沟通合作的三个步骤（自我包装）

![](img/682b7a7a94ef36c2d7fa3fee9937d0f0_27.png)

很多新手不知道如何与达人沟通。成功接洽的关键在于专业的自我包装。

以下是提升合作成功率的三个沟通步骤：

![](img/682b7a7a94ef36c2d7fa3fee9937d0f0_29.png)

1.  **自我包装**：塑造专业的商家形象。
    *   **账号包装**：微信/抖音账号使用专业头像（如公司Logo），昵称注明身份（如“XX品牌供应链对接”）。
    *   **实力展示**：准备一份公司或产品介绍的PPT/资料，展示品牌实力、生产实力或爆款历史数据。
    *   **私信话术**：例如：“您好，关注到您橱窗的XX品类商品，我们工厂可提供更高佣金和更低价格，期待合作。”
2.  **体现产品专业度**：提供清晰专业的选品资料。
    *   制作一份包含**产品编号、图片、零售价、直播价、优惠券信息、佣金比例**的详细表格（Excel）。这能极大提升沟通效率，方便达人将你的产品存入其选品库。
3.  **提供有竞争力的价格与佣金**：灵活运用价格策略。
    *   首次可发送“通版”报价。对于重点想合作的优质达人，可在后续沟通中提供更具竞争力的“底价”和更高佣金。

## 终极秘诀：直播间互动破冰法

一个亲测有效的破冰方法是：**蹲守目标达人的直播间并进行小额打赏**。

**操作建议：**
*   **选择直播间**：优先选择在线人数较少（如50-100人）的带货直播间，你的发言更容易被看到。
*   **互动方式**：先刷价值5-10元的小礼物（如“棒棒糖”）吸引主播注意，然后在公屏留言：“主播下播后私聊，谈一下产品合作。”
*   **效果**：通过小额投入成为“榜一大哥”，能有效打破陌生感，大幅提升达人回复私信和通过好友申请的概率。

## 如何吸引达人主动找你？

除了主动寻找，优化店铺也能吸引达人主动联系你。

达人通常会通过数据平台筛选优质商品。吸引他们的核心条件如下：
*   **店铺体验分**：不能低于 **4.8分**。
*   **产品质量与销量**：产品有优势，且有不错的销售基础。
*   **佣金设置**：佣金比例具有吸引力。

![](img/682b7a7a94ef36c2d7fa3fee9937d0f0_31.png)

## 如何设置佣金？

![](img/682b7a7a94ef36c2d7fa3fee9937d0f0_33.png)

设置佣金时，可参考以下两个策略：

1.  **对标同行**：查看同类商品中优秀竞争对手设置的佣金比例，在此基础上适当提高。例如，同行连衣裙开20%佣金，你可设置为25%-30%。
2.  **计算利润空间**：根据你的进货成本和预期利润，将商品的**毛利率控制在30%-50%之间**，为支付达人佣金留出充足空间。

**核心公式：**
`商品售价 = 成本价 + 平台扣点 + 预期利润 + 达人佣金`
你需要在这个公式框架内，平衡各方利益，设置最终售价和佣金。
