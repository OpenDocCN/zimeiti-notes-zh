以下是三个主要的内容创作领域供你选择：
*   **混剪**：将多个视频素材进行创意拼接。
*   **解说**：对影视、游戏、知识等内容进行口播讲解。
*   **影视推广**：剪辑并推荐影视作品。

你不能每一个领域都想做。最开始必须选择一个领域进行专攻。获得收益和成果后，再慢慢拓展其他领域。没有人能一口气吃成胖子。

![](img/fbffdb0275ca7674c79dfe54aec06ef8_7.png)

你必须对自己要做的领域或视频类型有明确的想法。

**创作内容 -> 吸引流量 -> 实现变现**
