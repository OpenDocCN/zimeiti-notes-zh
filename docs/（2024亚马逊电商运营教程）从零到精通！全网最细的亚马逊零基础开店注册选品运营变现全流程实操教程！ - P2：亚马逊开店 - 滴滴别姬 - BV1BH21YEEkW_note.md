完成准备工作后，即可开始注册流程。

1.  打开浏览器，访问百度搜索引擎。
2.  搜索“亚马逊全球开店”。
3.  进入官网后，点击页面右上角的“前往站点注册”。
4.  在站点选择中，点击“北美站”进行注册。

**核心操作** 可以概括为访问官网并选择北美站点：`前往亚马逊全球开店官网 -> 选择“北美站”注册`。

北美站包含了美国、加拿大和墨西哥市场。开通北美站后，你可以在三个国家同时进行销售，市场潜力较大。如果你是新手，建议从北美站开始运营。

## 寻找与上架产品

上一节我们介绍了账号注册，本节中我们来看看如何寻找适合销售的产品并完成上架。

产品不会找，可以参考以下国内货源网站。这些平台上通常有专门为跨境卖家提供的资源。

*   **1688**
*   **中国制造网**
*   **义乌购**

![](img/54f4a5d73cfbbd203cf5f08262dec042_1.png)

在选择产品时，应尽量寻找国内外**差价比较大**的产品。例如，一款在国内很火的大肚杯，国内拿货价约为10元人民币。通过亚马逊将其卖到美国，售价可达**19.9美金**（约合人民币100多元）。除去所有成本，一单的利润可以达到80多元。

![](img/54f4a5d73cfbbd203cf5f08262dec042_2.png)

你无需囤货，也无需押资金。可以直接到上述货源网站寻找支持**一件代发**的供应商。找到合适的产品后，复制产品的图片、标题和详情页信息。通过适当加价，将这些信息上传到你注册好的亚马逊店铺中即可。一旦有海外顾客下单，你只需在货源网站选择一件代发，由供应商直接发货给顾客。
