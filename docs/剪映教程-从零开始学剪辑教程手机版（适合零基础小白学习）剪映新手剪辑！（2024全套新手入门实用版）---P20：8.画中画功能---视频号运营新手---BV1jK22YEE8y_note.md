# 剪映教程：P20：画中画功能 🎬

在本节课中，我们将学习剪映App中一个非常核心的功能——**画中画**。你将了解它的基本概念，并通过两个实际案例，掌握其最常见的两种用法。本节课的重点和难点在于理解视频的**层级排序**。

## 什么是画中画？

![](img/8e267bdd293a8010098dc4d8284244d0_1.png)

画中画功能允许你在主视频轨道之上，叠加额外的视频或图片素材。这些叠加的素材可以自由缩放、移动，从而创造出丰富的视觉效果。

## 如何添加画中画？

以下是添加画中画的具体步骤。

1.  打开剪映App，点击“开始创作”并导入你的主视频素材。
2.  进入编辑界面后，在底部工具栏中找到并点击 **“画中画”** 按钮。
3.  点击 **“新增画中画”**，从相册中选择你想要叠加的视频或图片。
4.  添加成功后，该素材会出现在主轨道下方的一个独立轨道上。你可以在预览窗口中用手指对其进行任意的**缩放**和**移动**。

剪映最多支持同时添加**6个**画中画素材，它们分别位于不同的轨道上。

## 理解层级关系

当有多个画中画素材重叠时，理解它们的层级关系至关重要。

*   **默认规则**：后添加的画中画素材**层级更高**。在重叠区域，高层级的素材会覆盖低层级的素材。
*   **调整层级**：选中任意画中画素材，在底部工具栏中找到 **“层级”** 按钮。在弹出的窗口中，你可以通过拖动来改变所有画中画（包括主轨道）素材的上下覆盖关系。调整完毕后点击“确定”。
*   **轨道切换**：选中画中画素材，点击 **“切主轨”**，可以将其移动到主视频轨道。反之，选中主轨道素材，点击 **“切画中画”**，可以将其移动到画中画轨道。

掌握了画中画的基础操作后，我们通过两个案例来巩固学习。

## 案例一：制作三分屏效果

我们将使用画中画功能制作一个经典的三分屏视频。

1.  点击“开始创作”，导入一段视频作为背景。
2.  点击底部工具栏的 **“比例”**，设置为 **`16:9`**，然后点击“返回”。
3.  点击 **“画中画” -> “新增画中画”**，导入第二段视频。
4.  用同样的方法，再次新增画中画，导入第三段视频。
5.  现在屏幕上有三个视频层。分别移动和缩放它们，使每个视频各占屏幕约三分之一面积。
6.  调整完毕后，点击播放预览效果。

一个简单的三分屏视频就制作完成了。

## 案例二：混合模式应用（光斑叠加）

接下来，我们结合画中画和混合模式功能，为视频添加光斑特效。

1.  新建一个项目，导入你的主视频素材。
2.  点击 **“画中画” -> “新增画中画”**。
3.  点击 **“素材库”**，在搜索框中输入“光斑”，选择一个黑色背景的光斑素材并添加。
4.  将添加的光斑素材放大至铺满屏幕。
5.  选中光斑素材，在底部工具栏中找到并点击 **“混合模式”**。
6.  在混合模式选项中选择 **“绿色”**。这个模式会过滤掉黑色部分，只保留亮色的光斑。
7.  预览视频，你会发现光斑已经完美叠加在主视频之上了。
