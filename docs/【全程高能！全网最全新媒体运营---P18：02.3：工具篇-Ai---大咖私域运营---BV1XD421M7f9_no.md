## 概述
在本节课中，我们将学习第二款强大的AI生成PPT工具。上一节我们介绍了一款AI生成PPT工具，本节中我们来看看另一款互补的工具，它能弥补前一款工具的某些不足，并提供更快的速度和更丰富的模板选择。

## 工具核心优势
这款工具主要有两个优势。第一，生成速度快且稳定。第二，模板库非常庞大，提供了丰富的样式供选择和调整细节。它与之前分享的工具相辅相成，能更好地满足PPT生成的工作需求。

## 两种生成方式
该工具提供了两种核心的PPT生成方式。

以下是两种方式的简要说明：
1.  **AI智能生成**：直接输入主题，由AI自动构思大纲并填充内容。
2.  **导入本地大纲**：用户自行准备详细的大纲和内容，导入后由工具套用模板生成PPT。

## 方法一：导入本地大纲生成
我通常使用“导入本地大纲”的方式，因为它能更好地控制最终内容。

首先，我们需要准备一份详细的大纲。可以使用ChatGPT等AI助手来辅助完成。

例如，向ChatGPT提问：“请为我撰写一份关于‘装修行业在小红书平台的运营推广计划’的详细大纲，并完善每一个小节点的具体内容信息，保持好结构和格式排版。”

获得AI生成的大纲后，我们将其内容复制到一个文本文档中。**关键点在于**：需要确保大纲结构清晰，节点内容详细，这样工具才能更好地分析和套用模板。

准备好内容后，进入工具操作界面。

以下是操作步骤：
1.  选择“导入本地大纲”功能。
2.  上传或粘贴准备好的大纲文档。
3.  工具会快速解析文档结构。
4.  在丰富的模板库中，选择一个符合行业风格（如装修行业）的模板。
5.  点击“生成PPT”。

生成完成后，会进入PPT编辑界面。如果发现某些大纲节点内容未被正确加载，可以在编辑器的“大纲编辑”区域进行手动补充和调整。

在编辑界面，我们可以进行全面的自定义。

以下是可调整的选项：
*   **文字设置**：修改字体、大小、颜色。
*   **形状设置**：调整图形样式。
*   **背景设置**：更换页面背景。
*   **页面排版**：单击页面，可一键更换当前页的布局模板。
*   **整体模板**：支持随时更换整个PPT的模板样式，响应速度很快。

## 方法二：AI智能生成
接下来，我们看看“AI智能生成”方式。这种方式更适合快速搭建一个PPT框架。

![](img/59a96843205cec3340ea097d836f304f_1.png)

![](img/59a96843205cec3340ea097d836f304f_3.png)

操作非常简单。

以下是操作步骤：
1.  选择“AI智能生成”功能。
2.  在输入框中直接填写主题，例如“装修行业在小红书平台的运营推广计划”。
3.  点击生成，AI会自动构思大纲并填充详细内容。
4.  选择模板并生成PPT。

这种方式由AI全权负责内容创作，生成的PPT文字内容比较完整。但内容可能不完全符合用户的精确预期，更适合作为初稿或灵感来源。
