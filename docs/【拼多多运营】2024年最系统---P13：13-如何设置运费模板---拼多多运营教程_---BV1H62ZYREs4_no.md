# 拼多多运营教程：P13：如何设置运费模板 🚚

在本节课中，我们将学习拼多多店铺运营中一个非常实用的功能——运费模板的设置。运费模板能帮助我们高效管理不同商品的运费规则，是店铺物流设置的基础。

## 什么是运费模板？

上一节我们介绍了运费模板的基本概念，本节我们来详细看看它的定义和作用。

![](img/e250f7f7431761fa81bbf309ec319ada_1.png)

![](img/e250f7f7431761fa81bbf309ec319ada_2.png)

![](img/e250f7f7431761fa81bbf309ec319ada_3.png)

运费模板是为店铺中同一批商品设置统一运费规则的模板。当你需要修改配送地区或运费时，只需修改该运费模板，所有关联的商品运费信息就会同步更新。

![](img/e250f7f7431761fa81bbf309ec319ada_4.png)

![](img/e250f7f7431761fa81bbf309ec319ada_5.png)

运费模板支持为不同地区设置不同的配送和运费规则。

## 设置路径在哪里？

![](img/e250f7f7431761fa81bbf309ec319ada_9.png)

了解概念后，接下来我们看看在哪里进行设置。

![](img/e250f7f7431761fa81bbf309ec319ada_10.png)

以下是设置运费模板的具体路径：
1.  登录拼多多**商家管理后台**。
2.  进入 **发货管理** 页面。
3.  点击 **物流工具**。
4.  在物流工具页面，即可找到 **运费模板** 并点击 **新建**。

## 运费模板的构成与设置步骤

掌握了路径，现在我们来实际操作，看看如何一步步创建运费模板。运费模板的设置主要包含三个部分：模板基本信息、包邮配送区域和买家付邮费区域。

### 1. 填写模板基本信息

首先，点击“新建运费模板”后，我们需要填写基础信息。

*   **模板名称**：可随意填写，用于自己识别，例如“测试运费模板”。
*   **发货地址**：填写真实的商品发货地。如果是一件代发，则填写供货商的发货地址。填写不实地址可能导致物流投诉。
*   **模板类型**：选择 **自定义模板**。

### 2. 设置包邮配送区域

基本信息填好后，我们来设置哪些地区可以享受包邮。

包邮区域指买家无需承担运费的地区。通常，由于物流成本较高，**新疆、西藏、香港、澳门、台湾** 等偏远地区默认不勾选为包邮区。如果你的快递合作支持且利润允许，可根据实际情况勾选。

如果发货快递包含顺丰，可以勾选“顺丰包邮”，平台对顺丰快递有流量加权。

### 3. 设置买家付邮费区域

对于不包邮的地区，我们可以设置让买家承担运费，这样既能覆盖更多客户，又能控制成本。

在“指定区域运费”中，添加上一步未勾选的地区（如新疆）。

运费计算方式通常有两种：
*   **按件数**：适用于普通商品。公式为：`总运费 = 首件运费 + (购买件数 - 1) * 续件运费`。例如，首件10元，续件8元。
*   **按重量**：适用于大件或重货商品。

此外，可以设置 **指定条件包邮**，例如“满5件包邮”或“满100元包邮”，以刺激消费。

### 4. 设置不配送区域

对于完全无法发货的地区（如某些海外地区），可以在“不配送区域”中添加。通常选择“合作的快递不配送此区域”作为理由。

所有信息设置完毕后，点击 **提交**，运费模板即创建成功。

## 如何应用运费模板？

模板创建好后，我们来看看如何在发布商品时使用它。

在商家后台发布商品时，填写完商品信息后，在页面底部会看到“运费模板”选项。点击“其他模板”，即可从列表中选择刚刚创建好的运费模板。同一批商品可以关联同一个模板，实现运费规则的统一管理。
