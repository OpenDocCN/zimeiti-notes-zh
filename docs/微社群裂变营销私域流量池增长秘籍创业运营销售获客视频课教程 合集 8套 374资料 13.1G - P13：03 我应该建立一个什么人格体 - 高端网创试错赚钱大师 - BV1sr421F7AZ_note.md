# 微社群裂变营销私域流量池增长秘籍：P13：03 我应该建立一个什么人格体 🧑💼

在本节课中，我们将学习如何基于定位，进一步丰富和构建一个吸引目标客户的人格体形象。我们将探讨如何通过分析目标人群，来决定人格体的具体特征和输出内容，从而有效提升客户满意度和信任度。

---

在上一节我们讨论了定位的重要性，有了清晰的定位，人格体的雏形便已显现。然而，仅仅有雏形是不够的，我们需要对人格体进行丰富和细化。如果定位与输出的内容不匹配，将直接导致客户满意度下降。

![](img/7008867436a3a1ce932dcb0a54408df1_1.png)

那么，内容应如何丰富呢？内容的丰富完全由你的目标人群决定。选品决定了人格体的雏形，而人群则决定了你需要输出什么样的内容。为了方便理解，我们来看一个简单的例子。

例如，如果你是做幼教的。若想让你的学生家长满意，你需要体现出自己的**细心、耐心和责任心**。通常，年轻、有亲和力的女性角色在这一领域会更受欢迎。

但同样是做老师，如果你教的是高三学生。家长的核心需求是提升孩子的学习成绩，而非生活照顾。因此，他们可能更倾向于选择**教学水平高、责任心强、有多年经验的老教师**。至于外貌等因素，则不是他们关心的重点。

由此可见，在丰富人格体形象时，我们需要从以下三个方面入手：

以下是构建人格体的三个核心维度：
1.  **什么人**：你的目标客户是谁？
2.  **关注点**：他们对你的产品或服务最关心什么？
3.  **爱好**：他们的兴趣爱好是什么？

![](img/7008867436a3a1ce932dcb0a54408df1_3.png)

我们以化妆品代购为例进行说明。

![](img/7008867436a3a1ce932dcb0a54408df1_4.png)

**什么人**：做化妆品代购，主要客户群体通常是**90后的白领女性**。

**关注点**：客户的关注点必然围绕产品本身，例如：产品**是否为正品**、**通过什么渠道进货**、**能否提供购物小票**等。

**爱好**：这些客户的爱好可能包括**化妆、购物、追星**等。

将这些信息整理出来后，如何用来丰富自己的人格体呢？

首先，你的形象定位更加明确。既然客户是90后白领，你的年龄段最好与之相符，性别也最好一致（除非你是非常懂产品的异性专家）。

其次，你知道了他们的疑虑，就能通过文案轻松打消他们的顾虑。例如，如果你是**人肉代购，每月去一趟韩国**，你就可以每月进行一次采购直播，日常再分享一些直播花絮。

最后，知道了他们的爱好，你的文案素材就源源不断。发布一些关于追星、购物心得、化妆技巧的内容，能有效建立亲和力与喜爱度。

关于具体如何撰写和运用这些文案，我们将在后续课程中详细讲解。

在这里，给大家布置一个作业：

请按照上述方法，列出你的目标客户的几条核心关注点与兴趣爱好。然后，写出几项你自身可以匹配这些关注点和爱好的优势。

这个练习将帮助你厘清当前定位和文案可能存在的问题，并指明具体的优化方向。这个作业并非强制，但如果你有兴趣优化自己的人格体，可以将其整理成Word文档。只要发给我，我都会认真查看并提供解答。

本节课中，我们一起学习了如何通过分析目标人群的**身份、关注点和爱好**这三个维度，来系统性地丰富我们的人格体形象。明确这些要素，能让我们输出的内容更精准，从而高效地建立信任、提升客户满意度。
