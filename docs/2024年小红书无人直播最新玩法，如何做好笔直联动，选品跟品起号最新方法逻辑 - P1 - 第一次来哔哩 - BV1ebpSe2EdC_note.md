# 小红书无人直播教程：P1：电脑端无人直播搭建与笔直联动起号方法

在本节课中，我们将学习如何搭建电脑端无人直播间，并掌握通过“笔直联动”策略快速起号的核心方法。课程将分为两个主要部分：第一部分讲解抖音（DY）无人直播的搭建技巧，第二部分深入剖析小红书（红薯）的快速起号逻辑。

## 一、 开播前准备工作

在开始搭建直播间之前，我们需要准备好必要的硬件和软件。以下是具体的准备工作清单。

### 1. 硬件配置要求
首先，你需要一台符合以下最低配置的台式电脑：
*   **CPU**：英特尔 i5 十代或以上。
*   **显卡**：NVIDIA GTX 1060 或以上。
*   **内存**：16GB。
*   **操作系统**：Windows 10 64位系统。

请检查你的电脑是否符合此标准。如果符合，即可进行下一步。

### 2. 账号与店铺准备
根据你要操作的平台，准备相应的账号：
*   **抖音（DY）**：使用一个活跃的、经常刷视频的账号即可开播。
*   **小红书/抖音（红薯/阿多）**：
    *   可以新开一个个人店，要求较低。
    *   或者使用已有的商家店铺。
确保在开播前完成店铺的注册或准备。

### 3. 必备软件安装
以下是需要提前下载安装的软件：
*   **直播伴侣**：抖音官方的开播软件。
*   **小红书直播PC端**：请确保安装版本为 **2.38.2**，版本错误可能导致无法开播。
*   **OBS Studio**：一款免费开源的视频录制和直播推流软件，用于场景搭建。
请提前下载并安装好这些软件。

上一节我们列出了开播所需的准备工作，本节中我们将详细讲解第一个核心软件——直播伴侣的设置方法。

## 二、 直播伴侣参数设置详解

直播伴侣是开播的主体软件，正确的参数设置是保证直播画面清晰、流程不卡顿的关键。以下是关键设置步骤：

1.  **登录与模式选择**：
    *   登录账号后，若需申请开播权限，可按提示操作。
    *   在软件左侧选择 **“常规模式”**。

2.  **添加素材丰富直播间**：
    *   点击中间的“+”号，可以添加摄像头、视频、图片、窗口捕获等素材，以构建直播间内容。

3.  **核心：直播设置优化**（点击软件下方的“直播设置”）：
    *   **画布选择**：切换为 **竖屏（9:16）**，对齐方式选“左右裁剪”。
    *   **网络测速**：检查“当前上行速度”，稳定在50Mbps以上方可保证流畅。
    *   **清晰度设置**：不要使用“智能推荐”。
        *   在“视频”选项中，选择 **“蓝光”** 或 **“自定义”**。
        *   分辨率设置为 **1920 x 1080**。
        *   视频码率设置为 **6000 或 8000**。
    *   **帧率设置**：根据电脑配置选择。
        *   配置高：可选45或60帧。
        *   配置一般：选择30帧即可。
    *   **视频编码**：选择 **“硬编”**。
    *   **高级设置**：
        *   “预设”值越高越清晰（如P6、P7），根据电脑性能选择。
        *   “配置”选择第一个。
        *   “速率控制”选择 **“CBR”**（恒定码率）。
        *   “最大B帧”根据配置选择2或3。
        *   开启“心理视觉调优”、“高速编码”等选项。
        *   “色彩空间”选“709”，“色彩范围”选“局部”。

正确设置后，直播画面的清晰度和流畅度将得到显著提升。

掌握了直播伴侣的基础设置后，我们来看看如何利用OBS软件实现更复杂的无人直播场景搭建。本节重点介绍抖音的“视频+图片”语音模式搭建。

## 三、 抖音（DY）无人直播搭建：视频+图片语音模式

这种模式的核心是使用OBS构建直播场景，并通过直播伴侣的“语音模式”推流。

### 1. OBS基础设置
首先，打开OBS进行初始设置：
*   进入 **“设置” -> “视频”**。
*   将基础画布分辨率设置为 **1080 x 1920**（竖屏9:16），输出分辨率保持一致。
*   帧率根据电脑配置设置为30或60。

### 2. 构建语音模式场景
在直播伴侣中操作：
*   将开播模式从“常规模式”切换为 **“语音模式”**。
*   在语音模式界面，选择“自定义”背景，而非默认大头像。
*   此时你可以像在OBS中一样，添加图片作为背景，添加商品贴片，并添加循环播放的商品讲解视频。注意保持各元素比例协调（建议1:1）。

### 3. OBS内搭建“真人主播”画面
在OBS中新建场景：
*   添加 **“视频捕获设备”** 来源，选择你的摄像头。
*   使用 **“色度键”** 滤镜扣除绿幕，将真人主播抠出。
*   调整主播画面大小，使其约占屏幕三分之一。

### 4. 实现多视频轮播与调用
为了实现多个讲解视频自动轮播，我们需要使用一个插件：
*   在OBS来源中添加 **“VLC视频源”**。
*   在属性中，勾选“循环播放”，播放行为选“随机播放”。
*   点击“浏览”，**按住Shift键**多选你提前准备好的多个视频文件（总时长建议不低于1.5小时）。
*   这样，OBS就会随机、循环播放这些视频。

### 5. 将OBS画面接入直播伴侣
这是关键步骤，不使用OBS虚拟摄像头，而是：
*   确保OBS中“真人主播”和“轮播视频”场景已搭建好。
*   在直播伴侣（语音模式下）中，点击“添加素材” -> **“摄像头”**。
*   在摄像头列表中，选择 **“OBS虚拟摄像头”** 或类似名称（这需要OBS安装虚拟摄像头插件）。
*   此时，OBS中构建的包含真人主播和轮播视频的完整画面，就会接入到直播伴侣的语音模式中。

通过以上步骤，你就搭建了一个看似真人实时讲解，实则自动轮播的“语音模式”直播间。

## 四、 小红书（红薯）无人直播搭建与笔直联动起号

小红书的核心在于“笔记”与“直播”的联动，即 **“笔直联动”** 。其逻辑是：通过发布优质笔记为直播间预热、引流，再通过直播放大流量并转化。

### 1. 笔直联动核心逻辑
*   **定义**：前期发布爆款笔记（不挂商品链接），吸引用户在评论区“种草”。当用户对你的内容产生兴趣后，你开启直播，这些潜在用户就会点击你的头像进入直播间，从而完成下单转化。
*   **优势**：一篇爆款笔记积累数月的流量，可能通过一场直播就能集中爆发，起号速度非常快。

### 2. OBS画面深度处理（增强原创性）
为了提升直播画面的“原创性”和防检测能力，在OBS中需要对素材进行多层处理：
以下是处理素材的四个关键步骤：
*   **步骤一：水平翻转**。右键点击视频源 -> 变换 -> 水平翻转。
*   **步骤二：微调旋转**。右键点击视频源 -> 变换 -> 编辑变换。将旋转角度调整为1-5度之间（如2.3度），使画面轻微倾斜。
*   **步骤三：色彩校正**。为视频源添加“色彩校正”滤镜，微调色调、饱和度等参数，轻微改变颜色。
*   **步骤四：动态抖动**。添加“抖动”插件（如Shake插件），并将其绑定到音频上，让画面随背景音乐有极其微弱的震动感。

### 3. 多层画面叠加
增加直播画面的复杂度和信息量：
*   在OBS中，除了主商品视频，还可以通过 **“浏览器”** 来源添加一些动态背景（如星空、粒子效果、滚动字幕等网站的链接）。
*   将这些背景层叠加在主画面之下，并调整其 **“不透明度”** 至一个较低的值（例如10%-30%），使其作为若隐若现的背景。

### 4. 音频层处理
*   **添加环境噪音**：使用“VLC视频源”随机循环播放一段环境白噪音或商场背景音音频文件，音量调至极低，营造真实感。
*   **添加互动音频**：添加另一个“媒体源”，导入提前录好的、循环播放的欢迎语和互动话术音频（如“欢迎新进直播间的宝宝”），设置间隔播放。

### 5. 画面调用到小红书直播端
有三种主流方式将OBS处理好的画面推送到小红书直播软件：
*   **方法一：虚拟摄像头插件**。在OBS中安装虚拟摄像头插件，然后在小红书直播软件中添加摄像头，并选择对应的OBS虚拟摄像头。
*   **方法二：使用“三个方框”类采集卡模拟软件**。这类软件可以在电脑上模拟出虚拟采集卡。将OBS输出到该软件，再在小红书直播软件中选择该虚拟采集卡作为视频源。
*   **方法三：使用“投屏”类软件**。将OBS窗口投射到一个虚拟显示器上，然后在小红书直播软件中使用“窗口捕获”或“屏幕捕获”功能，捕获这个虚拟显示器上的画面。

### 6. 音频输出设置
在小红书直播软件的“直播设置”中，将麦克风和扬声器都设置为 **“VB-Audio Virtual Cable”** 之类的虚拟音频线（需提前安装）。这样，OBS中处理好的所有音频（商品原声、背景音、互动话术）就能统一传输到直播中。

### 7. 选品与笔记发布建议
*   **垂直赛道**：专注于一个细分领域（如只做连衣裙，或只做项链）。
*   **爆款笔记**：精心制作封面和标题，这是提高点击率和曝光的关键。可以参考对标账号的爆款。
*   **货源**：初期可通过1688等批发网站寻找货源，一件代发。
