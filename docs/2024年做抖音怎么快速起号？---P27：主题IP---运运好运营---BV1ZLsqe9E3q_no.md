## 什么是主题IP？
上一节我们介绍了内容定位的重要性，本节中我们来看看如何为内容找到一个核心的“钩子”，即主题IP。

![](img/0c86062e82c042dfe2a0d16bfc828638_10.png)

首先，我们分析两个视频案例。

第一个视频标题为“挑战惹老婆生气的第21天”。视频内容是一对夫妻在打雪仗的日常互动。如果去掉“挑战惹老婆生气的第21天”这个标题，视频看起来只是一个普通的日常记录。但加上这个主题标题后，它引导观众产生好奇心：前20天发生了什么？后续还会有什么故事？这个主题将一系列视频串联起来，形成了一个可持续更新的系列。

第二个视频来自账号“小姨恋爱课”。视频内容是以小黑板为背景，讲解恋爱技巧。它的主题非常明确——“小姨恋爱课”。这个主题吸引了特定人群（如对恋爱技巧感兴趣的观众），并且可以通过“第X期”的形式持续产出内容，建立账号的辨识度和观众期待。

![](img/0c86062e82c042dfe2a0d16bfc828638_12.png)

通过对比可以发现，一个没有明确主题的账号（例如单纯唱歌），观众可能因为单条视频有趣而点赞，但很难形成持续关注的动力。而拥有**主题IP**的账号，能更好地吸引观众留存，期待后续更新。

## 如何策划主题IP？
明确了主题IP的价值后，我们来看看如何为自己的账号策划一个合适的主题IP。

![](img/0c86062e82c042dfe2a0d16bfc828638_14.png)

以下是策划主题IP的几种常见方法，每种方法都遵循 **`[前缀] + [垂直领域]`** 的基本公式。

**1. 名字 + 垂直领域**
这种方法将个人名称与专业领域结合，突出个人品牌。
*   `老韩摄影教学第12期`
*   `猪猪小姐姐的撩汉指南`

**2. 人群 + 垂直领域**
这种方法明确目标受众，直接吸引特定群体。
*   `别人家男朋友的拍照技巧 第三期`
*   `宝妈育儿小技巧之辅食添加`
*   `新手开店避坑指南`

**3. 地域 + 垂直领域**
这种方法结合地理位置特色，增加内容的独特性和代入感。
*   `西藏自驾之旅的第50天`
*   `环球旅居日本站`
*   `广州美食之火锅篇`

**4. 场景 + 垂直领域**
这种方法限定一个具体的生活或工作场景，内容更具象、易引发共鸣。
*   `办公室迷惑行为之为什么总有人脚臭`
*   `儿童房风格装修指南`
*   `婚礼现场的创意小心思`

## 总结
本节课中我们一起学习了**主题IP**的策划。主题IP是你账号内容的系列化“包装”，它能够有效提升账号的辨识度，培养观众的观看习惯和期待感。在确定商业定位后，不要急于拍摄单条视频，应先运用 **`[前缀] + [垂直领域]`** 的公式，为自己的账号规划一个清晰、可持续的主题IP，这是打造高权重账号的重要一步。
