# 新媒体运营零基础教程：P17：02.2：工具篇-AI生成PPT 🚀

在本节课中，我们将学习一个非常实用的AI工具，它能帮助我们快速生成PPT演示文稿。我们将了解它的两种核心模式，并通过一个实际案例演示其工作流程。

## 概述

本节课将介绍一款高效的AI生成PPT工具。该工具可以根据用户提供的主题或现有文档，自动梳理内容大纲，并基于大纲生成详细的PPT页面内容。这是众多同类工具中表现非常出色的一款。

## 工具核心模式

该工具主要提供两种生成模式，以适应不同的创作起点。

以下是两种核心模式：

1.  **主题生成模式**：用户输入一个主题，AI将结合PPT的生成逻辑，自动创作出完整的PPT全文。
2.  **文档生成模式**：用户上传一篇已有的文档（如TXT、PDF文件），AI助手会自动总结文档大纲，并基于此大纲生成PPT内容。

## 编辑与优化

生成初稿后，我们可以对PPT进行进一步的调整和美化。

在编辑界面右侧，找到“更换模板”功能。当前工具提供的模板数量虽然有限，但切换速度非常快。



![](img/a3ca62d2cc4ddce20a6546999b1f031f_15.png)



此工具一个显著的优点是：它能够**基于大纲节点自动生成详细内容**。这与许多其他工具不同，那些工具通常要求用户提供全部完整内容，它只是进行格式排版。而本工具在你仅提供要点或文章时，就能填充具体阐述，这大大提升了创作效率。

选定模板后，你还可以在右侧面板对文字、形状等元素进行细节调整。当然，你也可以将生成的PPT下载到本地，使用PowerPoint或Keynote进行更深入的定制。
