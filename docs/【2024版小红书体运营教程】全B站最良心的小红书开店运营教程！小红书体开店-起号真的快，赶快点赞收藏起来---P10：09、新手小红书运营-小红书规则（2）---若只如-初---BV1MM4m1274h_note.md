在本节课中，我们将要学习小红书平台规则的第二部分，重点解析“被推核心逻辑”与“推广规则”。理解这些规则是避免违规、获取平台额外流量支持的关键。

上一节我们介绍了小红书的基础规则，本节中我们来看看更具体的流量分配逻辑。

## 被推核心逻辑 🤔

“被推核心逻辑”指的是小红书系统为笔记提供额外推广和权重加成的内在规则。简单来说，符合特定条件的笔记会获得系统更多的流量扶持。

以下是获得额外流量加成的四种主要途径：

![](img/bf6b458eedce353269be998faa2be2dc_5.png)

1.  **与平台存在利益相关的笔记**
    如果笔记涉及与小红书有合作的品牌或相关方（例如品牌赠送、权威第三方认证），账号会获得基础权重加成。普通账号起号的基础展现可能在100-300，而有认证的账号展现量可能翻倍至2000-6000。这取决于品牌影响力和平台认可度。公式可理解为：**基础流量 × 认证系数 = 初始展现量**。

![](img/bf6b458eedce353269be998faa2be2dc_7.png)

2.  **付费推广系统**
    小红书平台提供付费推广工具（如薯条、信息流广告），使用这些工具确实能为笔记增加权重。**但不建议新手在运营前期使用**。因为商品笔记在信息流中展示比例较低（约5%-10%），前期投入竞争大、效果有限。付费推广更适合中后期店铺突破流量瓶颈时使用。

3.  **积极参与官方活动**
    参加小红书官方发起的各类话题活动或品牌合作活动。平台会对积极参与的账号产生好感，并可能给予一定的流量倾斜。

4.  **晒单与抽奖互动**
    发布真实的用户晒单内容或举办抽奖活动，也属于平台鼓励的互动行为，有助于提升账号活跃度和信任度。

了解被推逻辑后，我们需要掌握具体的推广行为准则。

## 推广规则详解 ⚙️

推广规则明确了在发布和运营过程中必须遵守与规避的事项，以确保笔记能正常获得流量。

以下是推广过程中必须注意的三个核心要点：

*   **发布须知**
    笔记内容必须符合《小红书社区规范》。触犯规范的内容将无法通过审核或获得展示。尤其需要避开宣传“长期使用才体现功效”的产品，例如声称“一个月瘦身”、“三个月美白”等功能性产品，这类内容极易被限流。

*   **注意事项**
    这部分包含两个关键禁令：
    1.  **禁止数据造假**：严禁通过人工手段干预笔记数据，例如刷粉丝、刷点赞、刷搜索点击等。小红书以“内容为王”，人为造假容易被系统识别，导致权重降低甚至封号，得不偿失。
    2.  **控制发布频率**：一个账号每天发布推广笔记不宜过多。**建议每日发布1-2篇精品笔记足矣**。发布超过5篇，权重会显著下降，前两篇的流量也会受到影响。代码逻辑可类比为：`if (每日发布笔记数 > 5) { 权重衰减系数 = 大幅降低 }`。

*   **违规处罚机制**
    小红书通过“管家”系统（包含机器与人工审核）监控违规行为。一旦收到警告，笔记曝光将受到严重限制。平台还会监控账号数据的异常波动，**如果粉丝数、互动量等数据月环比增长超过20%**，系统会重点检测其真实性，若判定为造假则将实施惩罚。因此，稳定、自然的增长比短期暴涨更安全。

本节课中我们一起学习了小红书流量分配的“被推核心逻辑”以及必须严格遵守的“推广规则”。掌握这些要点，可以帮助我们有效规避风险，并利用规则为账号成长加速。
