# 微信视频号运营教程：P33：19.实操步骤 - 开通与上架商品橱窗 🛒

在本节课中，我们将学习如何在微信视频号中开通商品橱窗功能并上架商品。这是进行视频号带货变现的关键一步，操作流程清晰简单，适合初学者跟随操作。

上一节我们介绍了视频号运营的基础概念，本节中我们来看看如何具体开通带货功能。

## 第一步：进入微信视频号

以下是进入视频号界面的具体步骤：

1.  **打开微信**：在您的手机上启动微信应用。
2.  **点击“发现”**：在微信底部导航栏中，找到并点击“发现”选项卡。
3.  **进入“视频号”**：在“发现”页面顶部，找到“视频号”入口（图标为黄色的蝴蝶形状），点击进入。

![](img/ce446665663bf0f8decc6d9a5ed1e169_1.png)

![](img/ce446665663bf0f8decc6d9a5ed1e169_1.png)

![](img/ce446665663bf0f8decc6d9a5ed1e169_2.png)

![](img/ce446665663bf0f8decc6d9a5ed1e169_2.png)

## 第二步：找到创作者中心

进入视频号后，请按以下步骤操作：

![](img/ce446665663bf0f8decc6d9a5ed1e169_4.png)

4.  **点击个人头像**：在视频号页面的右上角，点击人形头像图标，进入个人主页。

![](img/ce446665663bf0f8decc6d9a5ed1e169_6.png)

![](img/ce446665663bf0f8decc6d9a5ed1e169_4.png)

5.  **进入“创作者中心”**：在个人主页中向下滑动，在页面底部找到并点击“创作者中心”。（注：若未发布过作品，可能无法看到此入口，需先发布一条视频。）

## 第三步：进入带货中心

在创作者中心页面，您可以查看账号数据。接下来需要找到带货功能入口。

![](img/ce446665663bf0f8decc6d9a5ed1e169_8.png)

![](img/ce446665663bf0f8decc6d9a5ed1e169_9.png)

6.  **点击“带货中心”**：在创作者中心页面内，找到“带货中心”的入口（图标类似一个小卖部），并点击它。

![](img/ce446665663bf0f8decc6d9a5ed1e169_8.png)

![](img/ce446665663bf0f8decc6d9a5ed1e169_9.png)

## 第四步：开通橱窗权限

点击“带货中心”后，系统会根据您的账号状态引导下一步操作。如果您尚未开通橱窗权限，将进入开通流程。

以下是开通橱窗权限的核心步骤：

7.  **选择带货身份**：系统会要求您选择带货身份。通常提供“我是商家”、“我是达人”等选项。**对于大多数个人创作者，请选择“我是达人”**。

![](img/ce446665663bf0f8decc6d9a5ed1e169_11.png)

![](img/ce446665663bf0f8decc6d9a5ed1e169_11.png)

8.  **交纳保证金**：选择身份后，会跳转到交纳保证金的界面。您需要点击“去交纳”按钮，通过微信支付交纳**100元**的保证金即可完成开通。

![](img/ce446665663bf0f8decc6d9a5ed1e169_13.png)

**核心操作公式**：
```
开通橱窗 = 进入带货中心 -> 选择“我是达人” -> 交纳100元保证金
```

## 第五步：上架商品（已开通权限后操作）

![](img/ce446665663bf0f8decc6d9a5ed1e169_13.png)

如果您已经开通了橱窗权限，再次点击“带货中心”将直接进入商品选品和上架界面。

9.  **选品与上架**：在带货中心，您可以浏览并选择来自视频号选品联盟的商品，将其添加到自己的橱窗中。

![](img/ce446665663bf0f8decc6d9a5ed1e169_15.png)

**操作逻辑说明**：
-   同一个身份证可以注册两个视频号账号。建议第一个账号先选择“达人”身份，使用平台的选品库（O品）进行带货。
-   开通权限后，“带货中心”的功能会从“开通”变为“管理/上架”。

![](img/ce446665663bf0f8decc6d9a5ed1e169_15.png)

本节课中我们一起学习了微信视频号开通商品橱窗和上架商品的完整流程。关键点在于通过“创作者中心”进入“带货中心”，根据引导选择达人身份并交纳保证金即可开通。开通后，您就可以在选品中心挑选商品，开始您的视频号带货之旅了。整个过程无需粉丝门槛，操作简便。
