## 核心操作原理
上一节我们介绍了这个副业的潜力，本节中我们来看看具体的操作原理。其核心是将拍摄的真人视频，一键生成动漫视频，然后发布到TikTok上。一条这样的视频可能带来五位数的浏览量。目前掌握此方法的人不多。

以下是实现这一效果的具体工具和步骤。

### 视频转动漫工具
今天分享一个视频转动漫的工具。它操作简单，只需几步就能生成原创动漫视频。该工具无需付费，且转换速度非常快。

### 操作步骤
以下是使用该工具的具体步骤：
1.  打开软件，点击加号（`+`）。
2.  选择“动漫转视频”功能。
3.  在效果中选择“日漫”。
4.  点击“下一步”，导入一段自己的视频。
5.  点击“一键生成动漫”，等待处理完成。
6.  保存生成的作品。

### 视频后期处理
生成动漫视频后，可以进一步优化其视觉效果。操作步骤如下：
1.  打开“剪映”软件。
2.  导入制作好的动漫视频。
3.  点击“调节”功能。
4.  将**对比度**、**饱和度**、**色温**的数值调高，使色彩更加鲜艳。

完成后即可观看成品。

如果你不想自己出镜拍摄，也可以通过其他渠道获取素材。

以下是获取各类素材的途径：
*   **视频素材**：可以从“电影天堂”、“80S”、“人人影视”等网站寻找高清无水印的影视片段。
*   **音乐素材**：可以在“爱给网”、“耳聆网”等平台寻找合适的背景音乐。
*   **剪辑教程**：不熟悉剪辑的朋友，可以在“哔哩哔哩”上搜索免费的视频剪辑教程，有很多专业人士分享。

关于剪辑软件，手机用户可以使用“剪映”。我们将获取的素材进行二次剪辑后，即可发布到自己的TikTok账号。
