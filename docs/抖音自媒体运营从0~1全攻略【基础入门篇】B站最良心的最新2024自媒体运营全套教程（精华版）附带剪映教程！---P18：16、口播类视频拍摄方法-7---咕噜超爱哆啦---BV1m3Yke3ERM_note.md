# 抖音自媒体运营从0~1全攻略：P18：16、口播类视频拍摄方法-7 🎬

在本节课中，我们将要学习口播类视频的具体拍摄方法。我们将通过一个成功案例，详细拆解其构图、内容定位、拍摄技巧等核心要素，帮助你掌握制作高质量口播视频的关键步骤。

上一节我们介绍了视频拍摄的通用注意事项，本节中我们来看看如何将这些要点应用到具体的视频类型中。口播类视频是抖音上非常常见且有效的内容形式，无论你的领域是金融、装修、养生还是情感，其核心拍摄逻辑是相通的。

首先，我们分析一个名为“种小草”的学员案例。她是一位40岁带三个娃的宝妈，专注于情感口播领域。她的一条视频播放量超过8000万，并借此涨粉30万。她的成功关键在于精准吸引了目标群体。

以下是其视频拍摄的核心要点分析：

**1. 构图与画幅**
抖音是**9:16**的竖屏平台。但视频内容本身可以选择横屏或竖屏拍摄。“种小草”的视频采用了**4:3**的横屏构图。关键点在于：一旦选定一种画幅，应保持统一，不要横竖屏混用，以保证账号风格的稳定性。人物主体应占据画面中央约**70%**的比例，画面上下左右需适当留白，使观感清晰舒适。

![](img/22265c448a6e7d52da4ba6400292b14a_3.png)

**2. 人物与表现**
口播视频的核心是内容传达，而非颜值。拍摄时无需“怼脸”，保持得体清晰即可。如果对自己的外貌不自信，可以开启美颜滤镜进行适度调整，原则是“清晰体面”，不过度夸张。

**3. 内容与受众定位**
“种小草”的视频标题为《跟婆婆在一起住是什么样的感受》，内容讲述了与婆婆同居16年的真实感受。这条视频精准吸引了**已婚女性群体**，尤其是对婆媳关系有深切体会的用户。这说明了内容与目标受众高度匹配的重要性。

**4. 拍摄环境与设备**
口播视频对拍摄场景要求灵活，室内外均可。关键在于背景要整洁美观。为保证声音清晰，建议使用一个**手机收声麦克风**，这是提升音频质量性价比极高的设备。

**5. 台词与录制技巧**
对于担心记不住台词的同学，无需死记硬背。可以在剪辑环节，采用“边读边录”或使用提词器工具来完成录制，这能大大降低拍摄门槛。
