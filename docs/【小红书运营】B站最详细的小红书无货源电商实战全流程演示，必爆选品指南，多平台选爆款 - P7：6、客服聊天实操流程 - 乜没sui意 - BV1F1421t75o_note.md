# 小红书运营：P7：客服聊天实操流程 💬

在本节课中，我们将学习小红书店铺客服工作的具体操作流程与注意事项，包括自主客服与官方客服的区别、设置方法以及日常管理要点。

上一节我们介绍了选品与上架，本节中我们来看看如何通过专业的客服服务来承接流量、促进转化并维护店铺形象。

## 自主客服设置与管理

小红书的客服分为自主客服和官方客服。自主客服即商家自己的人工客服。

以下是自主客服的几个核心注意事项：

1.  **服务时间计算**：自主客服的考核计算时间是**0点到24点**，即全天24小时。但实际操作中影响不大，因为小红书的用户群体与淘宝用户重叠度高，购物经验丰富，咨询量本身不大。平台要求的人工回复率大于**80%**，通常都能达到。

2.  **用户质量**：小红书用户优质度较高，不属于下沉市场。实操中很少遇到恶意扯皮、仅退款或骗取运费险的情况，客服压力相对较小。

3.  **自动回复工具**：可以设置自动欢迎语等机器人回复功能。后台还有跟单助手等工具，商家可根据自身运营需求选择开启，这些工具有助于促进转化。

4.  **工单处理**：当消费者投诉至平台时，平台会生成工单。商家只需在工单要求的时间内，按照流程处理并提交处理结果即可。该流程与抖音电商类似。

## 子账号创建与管理

如果招聘了客服人员，不应让其直接登录主账号，需要创建子账号。

以下是创建子账号的步骤：
*   子账号通过**邮箱**注册，填写名称、邮箱账号和验证码。
*   为子账号勾选并开通所需的功能权限（如客服）。
*   创建后，需在后台确认该客服账号是否处于可服务状态。如果客服不在线，不要为其分配咨询流量，以免造成客户无人应答。
*   同一个邮箱可以绑定多个店铺的子账号，方便多店统一管理。

## 官方客服服务说明

小红书平台为鼓励更多商家（包括散户、宝妈等兼职人群）开店，提供了官方客服服务。该服务为收费项目，不建议所有商家使用。

以下是关于官方客服的详细信息：
*   **适用场景**：适合无法保证24小时在线回复的兼职或个人卖家。
*   **收费模式**：费用由“基础服务费”加“咨询转化金额的**3%** 提成”构成。基础服务费根据月咨询量阶梯定价，例如14-60元、120元、600元等。
*   **开通路径**：在客服后台，点击“团队工具”，找到“申请关闭客服”选项，其页面内可申请开通官方客服。
*   **服务保障**：如果官方代运营客服未达到服务标准，费用会打九折。

## 客服工具使用指南

高效的客服工作离不开便捷的工具。建议同时配置电脑端和手机端。

以下是各端工具的设置与使用方法：

1.  **手机端**：下载“小红书商家版”APP。在APP内可以处理客服聊天，建议将消息设置为**强提醒**，确保及时响应。

![](img/4bab6f41f8a849533be297af328d8007_5.png)

2.  **电脑端**：在商家后台主页，点击“客服”模块，找到并下载PC客户端。下载安装后，其操作界面与主流电商平台相似，包含聊天窗口、3分钟回复率监控、排队用户管理、数据统计及强提醒设置等功能。

![](img/4bab6f41f8a849533be297af328d8007_7.png)

建议大家将电脑端设置为自动登录并常开，同时手机端也保持登录状态。这样即使外出，也能通过手机及时处理客户咨询。
