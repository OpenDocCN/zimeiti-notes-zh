# 短视频起号·差异化定位课：01：抖音到底是什么 📱

![](img/a73d9340d74173374d37301b72c889f7_1.png)

在本节课中，我们将要学习抖音平台的本质及其发展历程。理解抖音是什么以及它想要什么，是我们在平台上成功运营账号的第一步。

## 大众认知中的抖音

在深入探讨之前，我们先来看看大众普遍认为的抖音是什么样的平台。

![](img/a73d9340d74173374d37301b72c889f7_3.png)

很多人知道抖音是一个短视频平台，拥有约8亿用户。我们每天打开抖音主要有两个原因：一是观看有趣的内容，二是获取特定领域的知识。例如，男生可能关注美女内容，女生可能关注美食内容。

有些人认为这个平台趣味性较低，甚至选择卸载。然而，对另一些人而言，抖音是消磨时间的有效工具，例如在用餐时观看，既能放松也能发现新内容。

![](img/a73d9340d74173374d37301b72c889f7_5.png)

## 抖音的进化历程

你以为的抖音仅止于此吗？要真正了解抖音，我们需要回顾它的发展史。只有理清其进化脉络，才能明白这个平台的本质。

### 第一阶段：娱乐平台 (2016-2017)

抖音在2016-2017年刚上线时，是一个偏娱乐属性的平台。

![](img/a73d9340d74173374d37301b72c889f7_7.png)

当时平台上流行的内容主要是两类：一类是跳舞视频，另一类是卡点特效视频。作为新平台，抖音的首要目标是积累第一批用户，包括创作者和观看者。而吸引用户最快的方式就是提供娱乐内容，这与当时其他平台上流行的颜值主播、游戏直播等内容类似。

### 第二阶段：内容平台 (2018-2019)

到了2018-2019年，抖音开始构建内容的广度，转变为一个内容平台。

![](img/a73d9340d74173374d37301b72c889f7_9.png)

在此之前，抖音在人们心中可能仍是一个只有娱乐内容的平台。因此，从2018年开始，抖音开始扶持各个垂直领域的内容流量。许多当前各领域的头部账号都是在这个时期起步的。

![](img/a73d9340d74173374d37301b72c889f7_11.png)

以下是两个典型例子：
*   **噗噗唧唧**：美食赛道的头部账号。
*   **某美妆达人**：曾经的直播一哥。

他们的第一条作品都始于2018年11月。这个阶段，抖音的核心诉求是“内容至上”，它需要补充各个细分赛道的垂类内容，培养用户在抖音上获取知识（如美食教程、数码测评）的新习惯。

### 第三阶段：电商平台 (2020年)

在积累了近4亿用户后，抖音在2020年进入了转化阶段，最快的方式就是发展电商。

![](img/a73d9340d74173374d37301b72c889f7_13.png)

无论是直播还是卖货，抖音在2020年从一个内容平台转变为一个具有电商属性的平台。从第一季度开始，罗永浩、薇娅等头部主播开始在抖音进行直播带货。

这个阶段，平台的核心诉求是“变现”和“卖货”。它培养了用户在抖音上进行消费的第三个习惯，利用其庞大的用户基础，满足了用户的购物需求。

### 第四阶段：争夺用户时长 (2021年至今)

从去年开始，抖音的核心战略转向增加用户的停留时长。

![](img/a73d9340d74173374d37301b72c889f7_15.png)

抖音通过一系列战略部署，旨在稀释用户在其他APP上的使用时间。以下是其主要举措：

![](img/a73d9340d74173374d37301b72c889f7_17.png)

*   **本地生活**：满足用户查找本地消费目的地的需求。
*   **新农人计划**：与快手竞争，吸引三农内容创作者和观众。
*   **中视频伙伴计划**：扶持口播类知识博主，发布横屏长视频，对标B站。
*   **短剧计划**：对标爱奇艺、优酷、腾讯视频等长视频平台。
*   **图文计划**：对标小红书。
*   **搜索功能**：对标百度。
*   **世界杯转播权**：争夺用户在电视端的观看时间。

![](img/a73d9340d74173374d37301b72c889f7_19.png)

## 抖音当前的核心诉求与我们的策略

![](img/a73d9340d74173374d37301b72c889f7_21.png)

总结其发展历程后，我们明白了抖音当前的核心诉求。在用户增长见顶的背景下，它的目标不再是单纯拉新，而是以下两点：

1.  **突破用户停留时长**：让用户花更多时间在抖音上，满足原本在其他APP上的内容与消费需求。
    *   **结论**：**谁的内容能让用户贡献更长的观看时间，谁的流量就会更好。** 具体表现为视频的**播放时长**和**完播率**更高。
    *   **互动值**（如转粉、访问主页、分享、收藏）同样重要，因为这些行为背后也增加了用户的停留时长。

2.  **丰富平台变现**：让平台赚到更多钱。
    *   **结论**：**谁的变现模式更丰富、更高效，能让平台获益更多，谁的账号获得的扶持就会更大。**

因此，我们今年要在抖音上运营账号，必须关注以下两点：
*   **创作优质内容**，提升用户停留时长。
*   **设计丰富高效的变现模式**，为平台创造价值。
