# 剪映教程：P19：剪映核心剪辑功能——帧定格 🎬

在本节课中，我们将要学习剪映的一个核心剪辑功能——**定格**。这个功能能够将视频的某一瞬间画面静止，是制作卡点、转场等特效视频的常用技巧。

## 概述

![](img/066a8a12042039147739e623a930a7b3_1.png)

定格功能指的是在视频的动态画面中，突然将某一帧画面静止下来。这个功能常用于制作卡点视频或特殊转场效果，能让视频更具视觉冲击力。

上一节我们介绍了基础的剪辑操作，本节中我们来看看如何利用“定格”功能为视频增添创意。

![](img/066a8a12042039147739e623a930a7b3_3.png)

## 功能理解与案例展示

单纯的理论可能难以理解定格功能的效果。我们可以通过一个案例来直观感受。

以下是“万有引力”卡点视频的案例解析：
*   该视频在每一个镜头切换的最后一帧，都使用了定格功能。
*   例如，在听到“砰”的枪声卡点后，视频画面会静止片刻，然后再切换到下一个场景。
*   这种静止的画像效果，就像拍摄了一张照片，让视频看起来非常炫酷。

## 实操步骤：如何添加定格

理解了定格的效果后，我们进入实操环节，学习如何在剪映中添加定格效果。

1.  **导入与准备素材**
    打开剪映，点击“创作”导入一段视频素材。为了演示更清晰，可以先将过长的视频进行裁剪，保留需要的片段。

2.  **应用定格功能**
    选中视频素材，在底部二级菜单中找到并点击 **`定格`** 按钮。软件会自动截取当前时间指针所在位置的画面，并生成一段时长为3秒的静态图片片段。

3.  **编辑定格画面**
    生成的定格片段可以像普通图片一样进行编辑。以下是常见的编辑操作：
    *   **添加特效**：可以为定格画面添加滤镜（如牛皮纸、黑白褪色）或动效，改变其风格。
    *   **调整时长**：可以拖动定格片段的两端，自由调整其静止的持续时间，不限于默认的3秒。
    *   **删除**：如果效果不满意，可以选中定格片段后直接删除。

![](img/066a8a12042039147739e623a930a7b3_5.png)

## 效果预览与总结

编辑完成后，预览整个视频。你会看到，在应用了定格的瞬间，动态视频会变为静态画面，之后再继续播放或转场，从而创造出强烈的节奏感和视觉停顿效果。

本节课中我们一起学习了剪映的**定格**功能。关键点在于：该功能能一键生成静态画面片段，用于制作卡点视频。你可以自由调整这段静态画面的时长，并为其添加特效，使其更好地融入视频的整体风格。
