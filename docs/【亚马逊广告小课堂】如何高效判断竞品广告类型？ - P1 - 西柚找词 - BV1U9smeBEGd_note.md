# 亚马逊广告小课堂：P1：高效判断竞品广告类型 🎯

在本节课中，我们将学习如何快速、准确地判断竞争对手在亚马逊上投放的广告类型及其占比。这对于优化我们自身的广告策略至关重要。

![](img/cffc7c868181d2af0d4ab3435686e283_1.png)

上一节我们介绍了课程目标，本节中我们来看看具体的操作步骤。

## 第一步：点击广告分析 🔍

首先，在亚马逊广告平台或相关分析工具中找到并点击“广告分析”功能入口。

![](img/cffc7c868181d2af0d4ab3435686e283_3.png)

![](img/cffc7c868181d2af0d4ab3435686e283_1.png)

## 第二步：查看广告类型与占比 📊

进入广告分析页面后，系统会展示指定竞品的广告活动详情。

以下是您将看到的核心信息：
*   **广告类型**：明确显示竞品投放的是**商品推广（SP）**、**品牌推广（SB）**还是**展示型推广（SD）**广告。
*   **广告占比**：以百分比或图表形式，直观展示各类广告在竞品总广告活动中的分布情况。

![](img/cffc7c868181d2af0d4ab3435686e283_5.png)

![](img/cffc7c868181d2af0d4ab3435686e283_5.png)

本节课中我们一起学习了判断竞品广告类型的两个核心步骤：访问广告分析功能，并解读其中的广告类型与占比数据。掌握这个方法，能帮助你快速了解市场广告策略，为制定自己的广告计划提供数据支持。
