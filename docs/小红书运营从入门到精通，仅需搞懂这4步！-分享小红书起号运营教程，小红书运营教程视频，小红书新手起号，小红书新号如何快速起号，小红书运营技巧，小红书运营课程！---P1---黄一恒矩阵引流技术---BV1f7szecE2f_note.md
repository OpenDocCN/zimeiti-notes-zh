## 推流机制：点击为王，互动为后

上一节我们介绍了课程的整体框架，本节中我们来看看小红书的推流机制。理解平台的流量分发逻辑，是做好运营的第一步。

小红书是一个“点击为王，互动为后”的平台。**点击率**直接决定了笔记观看量的下限。例如，你发布的笔记流量可能是200、500、10000或35000，这主要取决于点击率的高低。而点击率主要由**封面**质量决定。如果封面足够吸引人，小红书的流量潜力巨大。点击率过低时，甚至连薯条广告都无法正常投放。

![](img/bb8f63bacaba543a55affd6c1977e1ba_4.png)

而**互动量**决定了笔记观看量的上限。笔记发布后，初始流量可能只有一两千。后续能否突破五万、十万，则取决于小红书的用户反馈算法，即**CES得分**。

CES得分计算公式如下：
```
CES = 点赞数 * 1 + 收藏数 * 2 + 评论数 * 4 + 转发数 * 4 + 关注数 * 8
```

接下来，系统会对笔记进行**打标签**。标签分为大标签和细分标签。大标签约有30多个，且不断更新。例如，在首页点击“小三角”，可以看到“露营”、“美食”等大标签。每个大标签下又细分为小标签，如“露营”下可能有“露营装备”、“露营地”、“露营穿搭”等。

内容的分发机制首先基于**兴趣标签**。系统根据用户近期的点击、互动、搜索行为，决定其首页推荐内容。因此，用户首页刷到的内容，反映了其近期的兴趣爱好。

其次，小红书有**重定向机制**。笔记会优先推送给曾与之产生过互动的用户（如点赞、收藏、评论、关注过该笔记的用户）。这是重定向机制的第一层。

最后是**初始分发曝光量**。这由账号的**流量层级**决定，而流量层级取决于账号最近30天的观看及互动数据。如果账号近期整体流量较高，新笔记发布后初始流量可能达到两三千。反之，如果近期流量未超过500，新笔记想过500也会比较困难。流量层级影响同质化内容获得的曝光量，导致不同账号发布相似内容时，流量差异巨大。

小红书的**流量池推荐机制**基于**千次曝光得分**。该得分计算公式为：
```
千次曝光得分 = (CES得分 / 曝光量) * 1000
```

当千次曝光得分较高时，笔记会进入更大的流量池。小红书共有约8个流量池：
*   初始流量池：约200曝光
*   基础流量池：200-500曝光
*   三级流量池：500-5000曝光
*   小型流量池：5000-20000曝光
*   潜力流量池：2万-20万曝光
*   热门流量池：20万-500万曝光（大热门）
*   爆款流量池：500万-1000万+曝光
*   全站流量池：1200万-3000万+曝光（覆盖大部分活跃用户）

千次曝光得分越高，进入的流量池越大。若得分不足，笔记通常会卡在200曝光左右，无法扩散。此时需要回头优化内容。

## 起号原理：观察、识别与入池

理解了流量机制后，我们来看看一个新账号的成长路径。小红书的起号过程可分为三个阶段。

第一个阶段是**观察期**。新账号注册后，系统不确定其内容质量，会限制流量。通常表现为新号发布笔记流量很低，甚至没有流量。系统此举旨在识别风险，避免违规内容传播。对于敏感行业，发布违规内容可能导致账号被限流甚至封禁。

操作建议：前期内容需要“脱敏”，避免营销敏感内容，尽量发布利他性的干货内容，帮助账号快速度过观察期。

第二个阶段是**识别期**。度过观察期（快则一周，慢则一个月）后，账号进入识别期。系统会识别你的内容与哪些标签相关，并尝试推荐给对应人群。内容好坏，取决于推送给谁。例如，将露营内容推给露营爱好者就是好内容，推给不感兴趣的人就是差内容。

度过识别期最有效的方法是：发布用户经常搜索的内容。这能帮助系统快速识别账号定位。例如，做净水器，就发布搜索下拉框中推荐的相关选题。

此阶段涉及两种标签：
1.  **内容标签**：笔记本身对应的标签。
2.  **用户反馈标签**：用户互动行为（点赞、收藏等）产生的标签。

系统判定内容与用户标签匹配度越高，推送力度越大。操作方向是专注**搜索流量**，帮助账号快速度过识别期。

第三个阶段是**入池期**。笔记发布前几天流量可能只有两三百，但隔了几天或一周后，流量突然增长到三五千并持续增加，这代表笔记已进入更大的流量池。

加速入池有一个有效方法：**投薯条**。薯条投放可以强制触发人工审核，帮助快速度过观察期。操作步骤分为两次投放：
1.  第一次投放：选择“点赞收藏”目标，拉高CES得分。
2.  第二次投放：选择“增加浏览量/播放量”目标。

投放时长设为24小时，调整后投放两次即可。投薯条的主要意义在于：强制过审、加速度过观察期、提高初始流量、快速拉高流量增长、帮助账号和内容打上精准标签。

## 起号方法：搜索与互动

掌握了起号原理，本节我们来看看两种具体、高效的起号方法。

第一种是**搜索起号法**，这是目前行业内最简单易行的方法。其核心目的是让系统快速识别账号定位，将内容推送给相关人群，从而快速获得用户反馈标签。搜索流量在小红书总流量中占比超过30%，因为超过60%的用户打开小红书会首先搜索解决问题。

操作步骤如下：直接在赛道内搜索核心关键词，将搜索下拉框中的长尾词全部整理下来，围绕这些选题发布笔记，即可快速起号。

以下是操作时的几个关键细节：
*   **选题**：优先选择收藏数大于点赞数的笔记作为对标，这类内容更容易卡住关键词排名。
*   **内容**：如果借鉴爆款，内容替换需超过30%，避免被判定抄袭。
*   **标题**：必须包含核心关键词。标题权重排序为：标题 > 内容 > 标签 > 话题。
*   **封面**：可以直接模仿对标笔记的封面，已验证点击率。
*   **视频笔记**：将视频中的文案复制到笔记正文中，方便系统识别。
*   **避免营销**：内容中不要带有明显的营销广告，否则可能连薯条审核都无法通过。

第二种方法是**骗评论起号法**。这个方法利用评论来快速拉高CES得分。因为在CES公式中，评论价值4分，相当于4个点赞或2个收藏，是获取流量最快的方法之一。

核心思路是降低用户评论成本，引导用户轻松参与。最简单的方式是使用 **“极限二选一”** 模板。让用户在A和B之间做选择，例如“房间问题，衣服沙发项链选哪个？”、“两个人口味相近，极限二选一”。几乎所有赛道都可以套用此模板。

另一种方式是**简答题**，直接提问，例如“说出你最喜欢的城市”、“说出你家装的最失败的电器”。核心在于降低用户思考成本，答案越简单直接越好，类似于做一个投票。

## 内容运营：流量与转化的平衡

最后，我们来探讨小红书的内容运营策略。运营内容时，可以分为两种类型，需要搭配使用。

**流量型内容**的特点是流量大，但转化率较低，属于“宽口”内容。例如，做装修的账号发布“2024年最流行的装修风格”，这种选题覆盖面广，无论用户房型大小或装修阶段，都可能感兴趣。

**营销型内容**的特点是转化率高，但流量相对较小，属于“窄口”内容。例如，同样是装修账号，发布“大平层、差户型怎么装修”，就限定了目标人群（大户型业主），流量虽小但意向精准。

正确的策略是两者配合：用流量型内容获取曝光，拉高账号基础权重；用营销型内容实现转化，成交订单。

具体应用可采用 **“九浅一深”** 策略：
*   **“浅”**（流量内容）：触达潜在用户。例如做西藏旅游，发布“西藏美食攻略”、“西藏拍照教学”，吸引对西藏感兴趣的大众用户，获取大流量。
*   **“深”**（营销内容）：转化目标客户。例如发布“国庆定制胡杨林小包团”等高客单价产品笔记，只转化已决定出游的精准客户。

“九浅”是为了获得广泛曝光，“一深”是为了获取目标客户进行转化。需注意两种错误倾向：
*   **只浅不深**：有流量无成交，账号越做越泛，与行业脱节。
*   **只深不浅**：只能依赖老客户，无法获取新客户。
