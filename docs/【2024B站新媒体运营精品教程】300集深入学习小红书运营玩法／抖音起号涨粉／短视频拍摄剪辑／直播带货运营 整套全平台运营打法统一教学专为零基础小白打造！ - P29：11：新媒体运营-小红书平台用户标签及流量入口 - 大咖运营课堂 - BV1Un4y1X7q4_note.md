在本节课中，我们将深入学习小红书平台如何为用户打标签，以及平台主要的流量入口有哪些。理解这些机制，有助于我们反向推导，让自己的笔记更精准地触达目标用户。

上一节我们介绍了系统如何判定内容标签，本节中我们来看看平台如何判定用户标签。这个过程能帮助我们分析和理解自己的笔记是如何被系统推荐给用户的，从而找到让笔记被精准用户看到的方法。

## 用户标签判定机制

小红书系统为用户打标签，主要依据用户的**浏览行为**。用户注册时选择的兴趣范围参考价值不大，它仅作为初期内容方向的引导，无法进入系统深层的用户标签池。

系统根据以下用户行为来判定其兴趣分类：

以下是判定用户标签的核心行为指标：

1.  **点击率**
    *   **定义**：系统在发现页向用户推送多种内容（如美妆、家居、母婴等），用户点击哪一类内容更多，系统就能计算出其偏好。
    *   **公式**：`用户对某类内容的点击率 = 点击该类内容的次数 / 系统推送的总次数`
    *   **作用**：通过点击率，系统能为用户打上强关联或弱关联的标签。例如，用户频繁点击美妆内容，系统会判定其为美妆兴趣用户，并持续推送相关笔记进行试探。

2.  **停留时长**
    *   **定义**：用户点开一篇笔记后观看的持续时间。
    *   **作用**：停留时长能更精细地判断用户喜好。例如，用户虽然常看美妆，但对某类护肤笔记停留时间特别长，系统就会强化其“护肤”子标签。如果用户对偶然推送的家居内容停留很久，系统可能判断其有装修或购买家居用品的潜在需求。

3.  **页面浏览量（PV）**
    *   **定义**：用户浏览页面的深度，例如不仅看了单篇笔记，还进入了博主主页并浏览了其他多篇笔记。
    *   **作用**：PV值能提升账号的健康度和质量评分。用户在你主页浏览的笔记越多，系统越认为你的账号内容优质、有吸引力，这对账号成长有极大帮助。

4.  **互动行为**
    *   **定义**：包括点赞、收藏、评论、关注等用户主动操作。
    *   **作用**：这些数据直接决定笔记能否进入更大的推荐流量池。高互动数据不仅能带来更多曝光，还能提升账号权重，为后续接单、变现奠定基础，同时可能带动账号内其他笔记的流量。

简单来说，平台通过持续观察用户的**点击、停留、深度浏览和互动**行为，像绘制一幅兴趣云图一样，为用户打上动态变化的标签，目的是让用户持续刷到喜欢的内容，获得满足感和更长的平台停留时间。

## 平台流量分发入口

理解了用户标签，我们再来看看小红书主要的流量分发入口。作为用户你可能经常使用，但作为运营者需要分析其区别。

小红书目前主要有四个流量入口：

以下是四个核心流量入口及其特点：

1.  **发现页**
    *   **内容构成**：
        *   已关注达人发布的新内容。
        *   系统根据用户标签（点击率、兴趣偏好）推荐的“猜你喜欢”内容。
        *   少量系统试探性推荐的多样化内容，以保持平台内容生态的丰富性。
    *   **核心价值**：这是笔记和账号获取流量的**最主要来源**。我们的运营目标就是让笔记进入发现页的推荐流。

2.  **关注页**
    *   **内容构成**：纯粹展示用户所有已关注博主发布的笔记。
    *   **特点**：流量精准但相对有限。是维护铁杆粉丝、进行深度互动的重要阵地。

3.  **附近页**
    *   **内容构成**：根据手机定位，推送同城或附近地区的笔记。
    *   **内容类型**：以美食、本地旅游、线下店铺等强地域性内容为主。
    *   **运营提示**：如果你是线下实体店（如美容院、餐厅），在做信息流广告或内容运营时，需要特别重视“附近”页的流量和人群定位。

4.  **购物页**
    *   **内容构成**：小红书电商和直播带货的核心入口，直接链接商品与商城。
    *   **现状**：部分用户尚未在此形成强购物习惯，更多人仍习惯先搜索笔记“种草”。但这是平台商业化的重点方向。
    *   **注意**：用户对笔记中硬性广告的容忍度在降低，“假种草”容易引起反感。

通过以上三节课（含前两节），我们完整梳理了小红书流量推荐与分发的底层逻辑。这些知识是理解平台运行机制、制定有效运营策略的基础。
