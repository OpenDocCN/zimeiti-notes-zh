# 手机剪辑教程：P36：镜头跟踪人物效果 🎬

在本节课中，我们将学习如何使用手机剪辑软件制作一个“镜头跟踪人物”的视觉效果。这个效果能让画面焦点在合影中的不同人物之间平滑移动，最终展现出完整画面，非常适合用于照片回忆或开场介绍。

## 概述
镜头跟踪效果的核心是**通过关键帧控制画面位置与蒙版动画**，模拟镜头移动并聚焦于不同主体的过程。我们将分步完成从设置关键帧到添加蒙版动画的全流程。

上一节我们介绍了关键帧的基础概念，本节中我们来看看如何将其与蒙版结合，实现动态跟踪效果。

## 步骤详解
以下是实现镜头跟踪人物效果的具体操作步骤。

1.  **导入并设置照片**
    准备一张合影照片，将其导入剪辑轨道。首先将照片的时长拉长，为后续的关键帧动画预留足够时间。

2.  **为第一个人物设置关键帧**
    来到时间轴起始处（0秒），为照片的“位置”和“缩放”参数打上第一个关键帧。此时画面应聚焦于第一个人物。

3.  **移动焦点至第二个人物**
    将时间指针移动到大约1秒的位置。调整画面的位置和缩放，使焦点移动到第二个人物上。软件会自动在此刻打上新的关键帧。

4.  **移动焦点至第三个人物**
    继续将时间指针向后移动（例如到2秒处）。再次调整画面，使焦点对准第三个人物。同样，此操作会生成新的关键帧。

5.  **展现完整照片**
    将时间指针移动到照片的结尾处。调整画面，露出整张合影的全貌。这将在结尾处创建最后一个关键帧。

6.  **添加黑色遮罩层**
    通过“画中画”功能，在素材库中导入一张黑色图片，并将其放大至全屏覆盖画面。

7.  **应用圆形蒙版**
    选中黑色画中画图层，找到“蒙版”工具，选择“圆形”蒙版。然后点击“翻转”按钮，使蒙版区域反转。

8.  **同步蒙版与镜头焦点**
    将时间指针移回第一个关键帧的位置（0秒）。调整圆形蒙版的大小和位置，使其仅框住第一个人物，并在此为蒙版的“大小”和“位置”参数打上关键帧。
    接着，将时间指针移动到倒数第二个关键帧处（即聚焦第三个人物的时刻），同样调整蒙版框住该人物，并打上关键帧。

9.  **完成蒙版动画**
    将时间指针移动到最后（展现全图的）关键帧位置。在此处，将圆形蒙版放大至完全露出整个画面。
    最后，回到时间轴开头，将蒙版向下拖动，制造一个从画面外移入的**入场动画效果**。

至此，镜头跟踪人物的核心效果已经完成。

## 收尾工作
最后，为整个片段配上合适的背景音乐，预览并导出视频即可。

## 总结
本节课我们一起学习了制作“镜头跟踪人物”效果的方法。其核心流程可总结为：**为背景图片设置移动缩放的`关键帧动画`，再通过画中画图层的`蒙版关键帧动画`与之同步，控制视觉焦点的转移**。关键在于让两个动画在时间点上保持同步，从而创造出镜头自然移动跟踪的感觉。你可以尝试将此效果应用于不同的照片或场景中。
