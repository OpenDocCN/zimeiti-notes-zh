## 起号前的准备工作

![](img/f099f09197029a35c0f3ec61341a24e7_7.png)

上一节我们概述了课程内容，本节中我们来看看起号前必须完成的准备工作。这些步骤是保障直播稳定、避免违规的基础。

以下是开播前必须完成的七项设置：

1.  **开启“抖音小安”防搜索功能**：在抖音“更多功能”中找到“抖音小安”，开启“陌生人不可以搜索我”选项，防止同行干扰。
2.  **关闭同城流量**：在直播设置中关闭同城推荐，避免不精准的同城流量扰乱初期模型。
3.  **禁用Wi-Fi，使用手机流量开播**：网络不稳定可购买信号放大器，确保使用稳定的手机蜂窝数据。
4.  **遵循“一机一卡一账号”原则**：尽量保证一个手机、一张手机卡只登录一个直播账号，降低连锁封控风险。
5.  **避免账号多设备同时登录**：直播期间，不要在另一台设备上同时登录该账号。
6.  **切勿使用虚拟手机号注册账号**：例如166、190、199等号段，此类账号稳定性差，易被风控。
7.  **禁止异地登录**：账号在A城市登录后，短时间内不要在B城市登录，否则易触发二次人脸验证或限流。

此外，需制定严格的直播计划：**24小时内只播一场**，且要**准时准点开播**（建议避开整点，如8:15、9:30），误差不超过2分钟。若需更换直播时间段，请**停播24小时**后再在新时间开播。新注册的账号建议**养号3天**后再开播。

## 起号核心逻辑与流量应对策略

完成基础设置后，我们进入核心环节。起号的本质是建立精准的“成交模型”。模型的核心权重排序是：**订单人群 > 人气数据**。因此，一切操作都应围绕“获得精准成交”展开。下面根据不同的流量情况，我们采取不同策略。

### 场景一：直播间流量少（个位数在线）时如何起号

当直播间流速慢、在线人数低时，目标是**拉流速、做留存、建信任**，而非强求成交。

*   **核心策略**：以**人性话术**为主，建立信任，拉近与观众的距离。
*   **话术要点**：通过卖惨、哭穷、讲故事、聊家常等方式，引发观众同情与共鸣。例如：“哥哥姐姐们，小妹第一天创业直播，没人气，能陪我说说话吗？”
*   **关键动作**：
    1.  **主动拉黑**：即使只有两三人，也要拉走长时间停留但无互动、非目标人群的观众，以净化流量标签。
    2.  **点对点互动**：目的不是卖货，而是**留存**。与每一位进入的观众真诚交流，塑造人设（如创业宝妈、大学生）。
    3.  **自然植入产品**：在聊天中自然提及产品，而非硬性推销。例如聊到家乡时，可问：“哥哥，你们那边用这种XX的人多吗？”
*   **直播目标与时长**：本场核心目标是提升流速和停留时长，为下场铺垫。必须保证**有效直播时长大于2小时**。
*   **关于“鱼塘”补单**：若有老客资源，建议在**第二场**进行补单，因为第一场后人群画像会更精准。

### 场景二：直播间流量大（急速流）时如何起号

新号开播常遇到急速流量，来得快、去得快、难留存。核心要认清：这波流量多是**真人娱乐粉**，购物意图弱，我们的目标是**筛选**和**互动**。

*   **核心策略**：**不强憋、不强留**，利用流量做互动数据，筛选出潜在兴趣用户。
*   **错误做法**：用钩子话术强留所有人，结果往往是流量迅速跌至谷底。
*   **正确流程（以15分钟极速流为例）**：
    1.  **速品**：快速介绍产品，过滤掉完全没兴趣的人。
    2.  **福利/互动**：抛出“小惊喜”（明钩子），要互动。例如：“想吃真醋、不想吃勾兑醋的扣1！”
    3.  **再速品**：进一步讲解，继续筛选。
    4.  **拉新**：利用留下的人做裂变，吸引新流量。
    5.  **点对点沟通**：与最后留下的少数人深入交流，尝试逼单。
*   **成功标准**：只要这场直播的**尾部流量不为零**，有人留下互动，即算合格。

## 三场起号流程的具体规划

理解了单场策略后，我们将其串联成一个完整的、为期三场的起号流程。

### 第一场：做流速记忆，筛选人群
*   **目标**：不求成交，只求**稳住尾部流量**，做出流速记忆。
*   **操作**：按上述“场景二”的流程操作，重点做好**速品-互动-筛选**的循环。
*   **话术结构重点**：`速品 -> 福利 -> 速品 -> 拉新 -> 速品 -> 福利 -> 点对点 -> 保障 -> 逼单`

### 第二场：必须实现成交
*   **目标**：必须**出单**，强化成交模型。
*   **操作**：沿用第一场节奏，但在尾部流量阶段，利用**人性话术**和**痛点话术**，对留下的精准用户进行逼单。
*   **逼单四要素**：**痛点话术**（戳中需求）、**保障话术**（消除顾虑）、**产品对比话术**（突出优势）、**价格拆分话术**（塑造性价比）。例如：“您可能玩了一辈子核桃，但没玩过这种品质的…今天您痛快我也痛快，拍一发二！”

### 第三场：微憋单，促递增
*   **目标**：实现**订单量递增**。
*   **操作**：开始尝试微憋单。话术结构可升级为：`拉新+福利 -> 互动 -> 福利 -> 速品 -> 拉新 -> 场景话术 -> 保障话术 -> 逼单（统计+上车）`。
*   **“福利”的灵活运用**：福利不一定是低价，而是你塑造的“价值感”。例如：“今天给您上的是意大利设计师3年打磨的款式，这本身就是福利！”

## 高级技巧：点对点与反向逼单

在人数不多的场景下，点对点逼单是最高效的成交方式。关键在于**自信、专业、果断**。

**点对点逼单话术公式**：
`确认需求 + 塑造价值 + 给出震撼福利 + 稀缺性 + 指令行动`

**示例**：“这位‘张三’哥哥，你是不是想找一双爬山不累脚的鞋？（确认需求）这双鞋用了XX科技，内置按摩颗粒（塑造价值）…今天你痛快我也痛快，一包烟的钱让你带走！（震撼福利）但只有5单，因为我是来破价做品宣的（稀缺性）。拍了回来打‘已拍’，我让运营直接下链接（指令行动）。”

**反向逼单技巧**：当有人犹豫时，果断下架链接，制造稀缺感和紧迫感。“运营，把链接下了！今天不是谁都能拍，是我让谁拍谁才能拍。”此举常能促使观望者下单。

## 常见问题答疑（Q&A）

本节我们汇总一些直播中的常见问题，并提供解决思路。

1.  **Q：个位数直播间没人说话怎么办？**
    A：用**人性话术**主动破冰，拉近关系。即使只有一个人，也要像朋友一样聊天，建立信任。

2.  **Q：总是被同行举报怎么办？**
    A：这是常态。坚持合规直播，开启防搜索功能。重心应放在提升自身直播内容和吸引力上。

3.  **Q：“其他流量”占比很高怎么办？**
    A：头条、西瓜等“其他流量”是抖音正在引导的电商流量。不要试图关闭，而应努力**转化它们**。成功转化后，账号流速会非常稳定。

4.  **Q：一个人能完成直播吗？**
    A：完全可以。主播需兼任中控（控节奏）和运营（上链接、看数据）的职责。关键在于提前准备好话术、流程，并勤加练习。

5.  **Q：老账号流量停滞怎么办？**
    A：很可能是流量被“榨干”或直播节奏固化。需要**改变话术结构、调整货品**，或重新审视流量曲线，在合适节点“炸流量”。
