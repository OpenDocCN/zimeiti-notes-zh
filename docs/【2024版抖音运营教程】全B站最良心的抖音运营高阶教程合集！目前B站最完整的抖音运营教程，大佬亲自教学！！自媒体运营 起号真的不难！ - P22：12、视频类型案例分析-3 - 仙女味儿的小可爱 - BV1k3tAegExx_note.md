## 案例分析：强人设还是弱人设？🤔

![](img/2d022a2a80726e8e05b54fff1388f835_5.png)

![](img/2d022a2a80726e8e05b54fff1388f835_7.png)

上一节我们介绍了人设的概念，本节中我们来看看一个具体的视频案例，并判断它属于强人设还是弱人设。

![](img/2d022a2a80726e8e05b54fff1388f835_9.png)

请看以下视频内容摘要，并做出判断：

![](img/2d022a2a80726e8e05b54fff1388f835_11.png)

![](img/2d022a2a80726e8e05b54fff1388f835_13.png)

> 穷人改变命运最大的难点就是他的父母，还有他的原生阶层。如果他生活在父母身边，深受其影响，基本上就没机会改变命运了。因为如果父母的观念是正确的，他们自己就不会是穷人。在多年的社会实践中成为穷人，说明其观念存在很多问题。

![](img/2d022a2a80726e8e05b54fff1388f835_15.png)

> 所以，穷人翻身通常需要离开家乡，远离父母，独自去其他城市打拼。当父母无法管束时，他才有机会改变自己的思维。

![](img/2d022a2a80726e8e05b54fff1388f835_17.png)

以下是视频中总结的穷人改变命运需要克服的几点障碍：

![](img/2d022a2a80726e8e05b54fff1388f835_19.png)

![](img/2d022a2a80726e8e05b54fff1388f835_21.png)

1.  **原生家庭与阶层**：受父母错误观念的影响。
2.  **社交圈子**：身边的朋友通常也是穷人，其解决问题的方法往往也是错误的。
3.  **缺乏本金与机会**：当机会来临时，可能因缺少关键资金而无法抓住，且缺乏承担风险的胆量。
4.  **内心虚弱与不自信**：长期处于社会边缘导致习惯性不自信、逃避责任，难以团结他人。

![](img/2d022a2a80726e8e05b54fff1388f835_23.png)

因此，想要改变命运需要做到以下几点：

![](img/2d022a2a80726e8e05b54fff1388f835_25.png)

1.  离开家乡，远离父母。
2.  结识新朋友，跳出原有阶层。
3.  敢于在能力范围内承担风险，有“all in”的胆识。
4.  建立自强和自信。

![](img/2d022a2a80726e8e05b54fff1388f835_27.png)

![](img/2d022a2a80726e8e05b54fff1388f835_29.png)

## 案例解析：为何这是“弱人设”视频？🔍

![](img/2d022a2a80726e8e05b54fff1388f835_31.png)

看完视频内容，我们可以得出结论：这是一个典型的**弱人设**视频。

![](img/2d022a2a80726e8e05b54fff1388f835_33.png)

![](img/2d022a2a80726e8e05b54fff1388f835_35.png)

一方面，视频内容虽然讨论了“穷人”和“富人”，但与发布者“我”自身没有直接关系。视频中的画面、声音与所讲述的观点内容是分离的，并未塑造出一个鲜明、具体的个人形象。

![](img/2d022a2a80726e8e05b54fff1388f835_37.png)

那么，为什么这样的“弱人设”视频能够火爆呢？关键在于它实现了**双向吸引**：

![](img/2d022a2a80726e8e05b54fff1388f835_39.png)

![](img/2d022a2a80726e8e05b54fff1388f835_41.png)

*   **一部分用户**被视频的**画面**（如治愈系的风格）所吸引。
*   **另一部分用户**则与视频的**观点**产生强烈共鸣，认为说得很好。

![](img/2d022a2a80726e8e05b54fff1388f835_43.png)

这种“画面”与“观点”各自吸引不同人群的策略，使得视频的受众面更广。

![](img/2d022a2a80726e8e05b54fff1388f835_45.png)

## 创作启示：如何应用“双向吸引”策略？💡

![](img/2d022a2a80726e8e05b54fff1388f835_47.png)

![](img/2d022a2a80726e8e05b54fff1388f835_49.png)

这种“弱人设”+“双向吸引”的策略可以广泛应用于各种类型的账号运营中。以下是几个应用示例：

![](img/2d022a2a80726e8e05b54fff1388f835_51.png)

*   **宝妈账号**：如果单纯拍摄宝宝日常或做饭无人观看，可以尝试为画面**配上**一个搞笑的家庭趣事或育儿故事。
    *   **代码示例**：`视频内容 = 宝宝可爱画面 + 配音（搞笑故事/育儿心得）`
*   **实体店/工厂账号**：如果单纯展示产品或车间无人观看，可以尝试为画面**配上**创业经历、行业故事或人生感悟。
    *   **代码示例**：`视频内容 = 产品/工厂画面 + 配音（创业故事/行业洞察）`

![](img/2d022a2a80726e8e05b54fff1388f835_53.png)

通过这种方式，你既能用画面吸引一部分用户，又能用内容（故事、观点）吸引另一部分产生共鸣的用户，让他们既能听下去，又能看完你的产品展示。

## 总结与要点回顾 📝

![](img/2d022a2a80726e8e05b54fff1388f835_55.png)

本节课中我们一起学习了如何通过案例分析来理解“弱人设”视频。

![](img/2d022a2a80726e8e05b54fff1388f835_57.png)

1.  **核心判断**：我们分析的案例是一个**弱人设**视频，因其内容与创作者本人关联弱，且画面与观点分离。
2.  **成功关键**：该视频的成功在于运用了**双向吸引**策略，即分别通过视觉元素和观点内容吸引不同偏好的用户。
3.  **实用技巧**：我们可以将这种策略迁移到自己的内容创作中。**公式**可以概括为：`吸引力 = 视觉吸引力 + 内容共鸣力`。为你的常规画面配上能引发共鸣的文案、故事或观点，是提升视频完播率和吸引目标受众的有效小技巧。

许多视频缺乏传播力，并非内容不好，而仅仅是缺少这样巧妙的结合技巧。抖音运营本身就是一个需要不断学习和优化方法的行业。
