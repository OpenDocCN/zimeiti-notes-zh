# 新媒体运营全平台高阶教程：P16：02.1：工具篇-全平台短视频文案一键提取 📝

在本节课中，我们将学习一个非常实用的运营工具，它能帮助我们从多个平台的短视频中，一键提取出完整的文案内容。掌握这个工具，可以极大地提升内容收集和创作的效率。

## 工具介绍与背景

在运营工作中，我们常常会遇到优秀的短视频，并希望借鉴或分析其文案。传统的方法可能包括使用“轻抖”小程序，或者将视频下载后导入“剪映”等软件进行提取。这些方法虽然可行，但操作相对繁琐，并且难以覆盖所有平台。

上一节我们介绍了文案提取的需求，本节中我们来看看一个更高效的解决方案。这个工具的核心优势在于，它支持**在线操作**，无需下载视频，只需提供视频链接，即可跨平台提取文案。

## 工具核心功能与操作步骤

![](img/2fd7d45fd7d6aeb2a81e7fc7b8a8d582_3.png)

![](img/2fd7d45fd7d6aeb2a81e7fc7b8a8d582_5.png)

该工具的核心功能是**在线解析视频链接并提取字幕/文案**。其操作逻辑可以简化为以下公式：

![](img/2fd7d45fd7d6aeb2a81e7fc7b8a8d582_7.png)

**`输出文案 = 工具.提取( 平台视频链接 )`**

![](img/2fd7d45fd7d6aeb2a81e7fc7b8a8d582_9.png)

以下是使用该工具的具体操作步骤：

![](img/2fd7d45fd7d6aeb2a81e7fc7b8a8d582_11.png)

1.  **复制视频链接**：在抖音、B站、小红书等平台，找到目标短视频并复制其分享链接。
2.  **粘贴链接**：打开该工具的网页界面，将复制的链接粘贴到指定的输入框中。
3.  **一键提取**：点击“提取文案”按钮，系统会自动解析视频并处理。
4.  **获取与处理文案**：提取完成后，可以直接复制纯文本，或使用“格式化”功能，将文案整理成带时间轴的字幕格式。

![](img/2fd7d45fd7d6aeb2a81e7fc7b8a8d582_13.png)

此外，该工具也支持上传本地视频文件进行文案提取，增加了使用的灵活性。

## 跨平台实操演示

为了让大家更直观地理解，我们进行两个平台的实操演示。

**首先，我们演示抖音平台。**
我们找到一个情感语录类短视频，复制其链接并粘贴到工具中。点击提取后，很快便得到了准确的文案内容：“听完你就释怀了，村上春树说，我动了离开你的念头，不是因为你不好，也不是因为不爱了……”。提取出的文案可以一键复制，方便后续使用。

**接下来，我们测试哔哩哔哩平台。**
我们准备了一个B站视频的BV号链接，同样粘贴到工具中进行提取。结果显示，工具成功提取出了“新手做直播电商，怎么才能打造出直播爆品？在直播销售的不同阶段……”等文案，准确性非常高。

这个演示表明，该工具能有效覆盖主流短视频平台。提取出的文案，我们可以进行修改和调整，用于自己的口播视频脚本或内容分析，这对许多行业都非常有用。
