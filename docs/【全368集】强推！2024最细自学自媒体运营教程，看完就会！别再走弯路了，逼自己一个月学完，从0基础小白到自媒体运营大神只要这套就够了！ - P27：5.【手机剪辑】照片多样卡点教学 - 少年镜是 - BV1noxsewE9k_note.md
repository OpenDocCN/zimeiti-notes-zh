# 自媒体运营教程：P27：手机剪辑：照片多样卡点教学 📸

在本节课中，我们将学习如何使用剪映App为照片制作多样化的卡点视频。整个过程从导入素材开始，到添加音乐、踩点、调整动画，最终完成一个动态的视频作品。

## 概述
我们将通过一系列步骤，将静态照片与音乐节奏结合，制作出富有动感的卡点视频。核心操作包括音乐踩点、图片对齐节奏点以及添加关键帧动画。

## 教程步骤

上一节我们介绍了视频剪辑的基础，本节中我们来看看如何制作照片卡点视频。

首先，打开剪映App，导入你准备好的照片素材。

接着，为视频添加背景音乐。点击“添加音频”，从你的收藏中找到合适的BGM并点击使用。

为了让照片切换与音乐节奏同步，需要进行踩点操作。点击音乐轨道，选择“踩点”功能，然后使用“自动踩点”选项。

以下是图片对齐节奏点的操作：
*   将每一张图片的起始位置，与音频轨道上出现的“小黄点”（节奏点）精确对齐。
*   对齐完成后，将音频多余的部分删除。

接下来，为图片之间的切换添加转场效果。点击两张图片之间的“小白块”，选择“闪白”转场效果。

为了统一风格，可以点击“应用到全部”按钮，将此转场效果应用于所有图片之间。

现在为图片添加动画。选中一张图片，点击“动画”选项，选择“荡秋千”效果。

重复上述操作，为后面的所有图片都添加上“荡秋千”动画。

为了让视频动感更强，我们还可以添加关键帧动画。首先，将所有图片稍微放大。

然后，为一张图片打上三个关键帧。在时间轴起始位置添加第一个关键帧（图片向上移动），在中间位置添加第二个关键帧（图片向下移动），在结束位置添加第三个关键帧（图片再次向上移动）。

使用同样的方法，为后面的每一张图片都加上这三个位置的关键帧动画。

完成以上所有步骤后，你的照片卡点视频就制作完毕了。

## 总结
本节课中我们一起学习了制作照片卡点视频的全流程。核心在于利用**自动踩点**功能对齐音乐节奏，并通过**关键帧**（在时间轴上标记图片位置变化的点）和**动画**为静态照片注入动态效果。反复练习这些步骤，你就能熟练制作出节奏感强的视频作品。
