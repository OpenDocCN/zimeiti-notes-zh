1.  **角色**
    角色是指你在特定赛道中希望呈现的专业身份。你提供什么方向的内容，就需要塑造相应的专家形象。
    *   **公式**：`人设角色 = 赛道方向 + 专业身份`
    *   **示例**：
        *   **装修博主**：展示真实装修过程、分享避坑经验、呈现前后对比。
        *   **母婴赛道**：可以是分享亲身经验的“宝妈”，也可以是提供专业知识的“护理专家”。
        *   **情感赛道**：可以是提供建议的“心理学顾问”，也可以是分享自身经历的“成长型人设”（如：分享职场、创业心路的30岁女性）。

2.  **内容**
    内容是人设的基石，必须与你选择的赛道高度垂直且持续。
    *   **核心要求**：内容需吸引用户、提供价值、并具备一定平台特色。
    *   **重要性**：垂直、持续的内容输出有助于打上精准的**账号标签**，直接影响平台的流量分发。内容杂乱会混淆用户和算法，导致推荐效果下降。
    *   **原则**：`做什么赛道，就持续提供什么赛道的内容`。

3.  **形式**
    形式是指内容呈现的具体方式，它影响着用户的观感和记忆。
    *   **产出场景**：室内固定场景、室外多变场景、车内、特定地点等。
    *   **声音呈现**：本人原声出镜、使用配音（可用`剪映`等工具实现）。
    *   **内容类型**：
        *   **剧情型**：创作难度较高，但故事性强。
        *   **干货/经验分享型**：需要精心准备文案脚本。
        *   **生活记录型**：需要积累大量日常拍摄素材。
    *   **建议**：可以参考对标账号的呈现形式。

4.  **记忆点** 🧠
    记忆点是让用户快速记住并识别你的关键，需要精心设计并反复强化。
    *   **设计方法**：在视频中植入固定、重复的元素。
    *   **记忆点清单**：
        *   **视频开头前置**：固定动作、固定特效、固定开场白。这些元素在视频前几秒出现，能形成强烈的初始印象。
        *   **贯穿视频全程**：统一的BGM（背景音乐）。音乐一响，用户就能联想到你。
        *   **贯穿内容全局**：统一出现的物品（如特定摆件）、统一风格的服装。这些元素持续强化视觉印象。
