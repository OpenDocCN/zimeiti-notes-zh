# 亚马逊运营在家一天做什么：P1：居家办公与副业探索

在本节课中，我们将跟随一位亚马逊运营的日常，了解居家办公的工作模式，并探索如何利用零成本副业增加收入。课程将涵盖居家办公的优势、副业的具体操作步骤以及如何平衡主业与副业。

## 🏠 居家办公的开始

早上好。我去健身房活动了一下，练了背，结束后回家，随便吃了饺子，就开始在家办公了。

今天主要告诉大家一个好消息。裸辞不到一个月，我已经找到新工作，依然是亚马逊运营。这份工作既不用打卡，也不用坐班。

## ⏰ 灵活的时间安排

我可以自己安排工作和休息时间。实行四天工作、三天休息的双休制，工作日不办公。除了亚马逊运营的工作，我平时还会做一些小副业。

## 💡 零成本副业机会

之前有个客户给我介绍了“浩利商城”，做带货达人。这项副业零成本、零投资，不需要拿货，就是帮别人推广商品以获取佣金，效果挺不错。

以下是实现带货达人身份的5个具体步骤：

1.  注册成为平台的推广员。
2.  在商城中选择你想推广的商品。
3.  获取专属推广链接或二维码。
4.  将链接分享到你的社交圈（如朋友圈、社群）。
5.  通过你的链接产生购买后，即可获得佣金。

## 🔄 主业与副业的平衡

如果你现在没有找到工作，感到焦虑，也可以考虑自己发展副业。关于要不要找新工作，我更倾向于先进行一段时间的休息和自我提升。但是，居家办公的工作机会真的很难拒绝。

现在我负责的亚马逊站点，除了欧洲站，美国站是最主要的。对于涉足这样全新的领域，我的内心感到非常欣喜。同时，经营副业也让我很开心。
