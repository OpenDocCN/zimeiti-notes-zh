# 剪映入门教程：P6：剪映APP界面结构拆解 📱

在本节课中，我们将要学习剪映APP的界面结构。了解软件的各个功能模块是学习剪辑的第一步。我们将逐一拆解“剪辑”、“剪同款”、“消息”和“我的”这四个核心部分，帮助你快速熟悉剪映的操作环境。

![](img/abeea094a8325430f7b563e02cbd7352_1.png)

上一节我们介绍了剪映的基本概念，本节中我们来看看它的具体界面布局。

## 界面四大模块概览

剪映APP的界面非常简洁，主要包含以下四个模块：
*   **剪辑**
*   **剪同款**
*   **消息**
*   **我的**

熟悉这四个模块，是我们开启剪辑之路的正确方式。

接下来，我们先简要介绍每个模块的功能，然后进入软件实际操作界面进行详细讲解。

### 1. 剪辑 ✂️

![](img/abeea094a8325430f7b563e02cbd7352_3.png)

启动剪映APP后，默认进入的首页就是“剪辑”页面。这是我们进行视频创作的核心区域。

![](img/abeea094a8325430f7b563e02cbd7352_5.png)

以下是“剪辑”页面的主要功能：
*   **开始创作**：点击顶部的“开始创作”按钮，可以创建新的剪辑项目并导入素材。
*   **草稿箱**：在首页界面中，可以看到之前剪辑过的项目草稿，方便继续编辑。

### 2. 剪同款 🎬

![](img/abeea094a8325430f7b563e02cbd7352_7.png)

“剪同款”是剪映为新手提供的特色功能。APP内自带大量热门短视频模板。

![](img/abeea094a8325430f7b563e02cbd7352_8.png)

![](img/abeea094a8325430f7b563e02cbd7352_9.png)

![](img/abeea094a8325430f7b563e02cbd7352_11.png)

以下是使用“剪同款”的步骤：
1.  在“剪同款”页面浏览并选择喜欢的视频模板类别（如卡点、VLOG、节日主题等）。
2.  点击“剪同款”，根据提示导入相应数量的个人视频或图片素材。
3.  导入后，可以对每个素材片段进行简单编辑，如替换、裁剪。
4.  也可以编辑模板中的预设文字。
5.  预览效果无误后，点击“导出”即可生成同款视频。导出时建议选择 **1080P** 分辨率。

通过这个功能，即使没有剪辑基础，也能快速制作出效果不错的视频。

### 3. 消息 🔔

“消息”模块与抖音平台打通。授权抖音账号登录后，可以在这里查看到抖音官方的通知、作品收到的点赞、评论等互动消息。

![](img/abeea094a8325430f7b563e02cbd7352_13.png)

### 4. 我的 👤

![](img/abeea094a8325430f7b563e02cbd7352_15.png)

“我的”是个人中心页面。同样因为与抖音账号互通，这里会显示你的抖音账号信息，包括头像、名称、粉丝量、获赞数等。你也可以从这里直接跳转到自己的抖音主页。此外，这里还会展示你在抖音上点赞和收藏过的视频。

## 剪辑界面详解

![](img/abeea094a8325430f7b563e02cbd7352_17.png)

现在，让我们回到最核心的“剪辑”界面，看看开始创作后的工作区布局。

当我们点击“开始创作”并导入素材后，会进入以下主要工作界面：

![](img/abeea094a8325430f7b563e02cbd7352_19.png)

![](img/abeea094a8325430f7b563e02cbd7352_17.png)

![](img/abeea094a8325430f7b563e02cbd7352_21.png)

![](img/abeea094a8325430f7b563e02cbd7352_23.png)

工作界面主要分为三个区域：
1.  **视频预览窗口**：顶部区域，用于实时预览剪辑效果。
2.  **时间轴轨道**：中间区域，这是所有剪辑软件的核心。视频是由连续画面组成的，时间轴以时间为顺序展示所有的视频、音频、文字等素材轨道。中间贯穿的白色竖线称为 **时间线**，它是进行裁剪、添加转场等操作的时间基准点。
3.  **底部工具栏**：底部区域，包含各种剪辑工具。首次看到的是**一级工具栏**，包括：
    *   `剪辑`、`音频`、`文字`、`画中画`、`特效`、`滤镜`、`比例`、`调节` 等。
    *   选中时间轴上的素材后，可以使用`分割`、`变速`、`动画`、`蒙版`等工具进行详细编辑。

![](img/abeea094a8325430f7b563e02cbd7352_25.png)

![](img/abeea094a8325430f7b563e02cbd7352_27.png)
