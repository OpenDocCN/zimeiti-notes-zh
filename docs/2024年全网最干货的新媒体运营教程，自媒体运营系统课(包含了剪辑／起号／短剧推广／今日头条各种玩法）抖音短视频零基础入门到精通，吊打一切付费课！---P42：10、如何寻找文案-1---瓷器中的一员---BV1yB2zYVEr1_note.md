# 新媒体运营系统课：P42：10、如何寻找文案-1

在本节课中，我们将要学习如何为短视频寻找和准备文案。这是优化作品、提升吸粉能力的关键第一步。我们将介绍实用的文案素材网站，并讲解如何利用工具解决配音问题，让你即使不擅长写作或对自己的声音不自信，也能轻松制作出吸引人的视频内容。

## 优化作品以提升吸粉能力

上一节我们讨论了提升浏览量，本节中我们来看看如何将观众转化为粉丝。将观众转化为粉丝的核心在于优化作品质量。即使每天发布多条视频，如果内容质量不佳，平台也不会给予更多流量，效果反而更差。正确的做法是：**保证发布频率的同时，确保每一条视频都具有吸引力**。这样效果才会显现。
