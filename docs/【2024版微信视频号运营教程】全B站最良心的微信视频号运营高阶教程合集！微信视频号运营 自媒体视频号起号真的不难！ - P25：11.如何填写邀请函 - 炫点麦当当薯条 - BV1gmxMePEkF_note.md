# 微信视频号运营教程：P25：11.如何填写邀请函 📝

在本节课中，我们将学习如何正确填写视频号创作者计划的邀请函。这是一个关键步骤，关系到您能否成功加入计划并开始运营。我们将一步步拆解，确保您能清晰、准确地完成所有信息的填写。

![](img/529191223b5e3d476a2364dd476c0409_1.png)

---

## 第一步：找到您的视频号账号信息

![](img/529191223b5e3d476a2364dd476c0409_3.png)

上一节我们介绍了如何注册视频号，本节中我们来看看如何找到账号昵称和粉丝数量。这是填写邀请函的第一步。

![](img/529191223b5e3d476a2364dd476c0409_4.png)

![](img/529191223b5e3d476a2364dd476c0409_5.png)

以下是具体操作步骤：

1.  **打开微信应用**：在您的手机上找到并打开微信软件。

2.  **进入“发现”页面**：在微信底部导航栏中，点击“发现”选项。

![](img/529191223b5e3d476a2364dd476c0409_7.png)

![](img/529191223b5e3d476a2364dd476c0409_8.png)

3.  **找到“视频号”入口**：在“发现”页面中，向上滑动，找到并点击“视频号”入口（图标为黄色蝴蝶形状）。

4.  **访问个人主页**：进入视频号页面后，点击右上角的个人头像图标，进入个人中心。

5.  **查看账号信息**：在个人中心页面下滑，找到“我的视频号”区域。这里会明确显示您的**账号昵称**和**粉丝数量**。
    *   **账号昵称**：例如“天道酬勤”。
    *   **粉丝数量**：例如“39917”。

请将这两个信息记录下来，用于填写邀请函。

---

## 第二步：填写邀请函的具体内容

![](img/529191223b5e3d476a2364dd476c0409_10.png)

找到账号信息后，我们就可以开始填写邀请函了。邀请函通常包含多个必填项，每一项都至关重要。

![](img/529191223b5e3d476a2364dd476c0409_11.png)

![](img/529191223b5e3d476a2364dd476c0409_12.png)

以下是需要填写的项目及说明：

*   **视频号账号昵称**：填写您在上一步中查看到的昵称，例如 `天道酬勤`。
*   **粉丝数量**：填写您当前的粉丝数，例如 `39917`。
*   **活动邀请人**：由于您是通过特定渠道加入，此处应填写邀请人的名称，例如 `张良` 或 `张良老师`。
*   **实名信息**：此处必须填写您本人的真实姓名，例如 `周建国`。这关系到后续的身份验证与收益结算。
*   **结算手机号**：填写您本人用于接收验证码和后续结算的手机号码，例如 `13812345678`。请注意，号码中间不应有空格。

所有信息填写完毕后，找到并点击页面上的 **“提交”** 按钮。

![](img/529191223b5e3d476a2364dd476c0409_14.png)

---

![](img/529191223b5e3d476a2364dd476c0409_15.png)

## 第三步：确认提交与后续操作

信息填写完成并点击提交后，系统通常会弹出确认提示。

![](img/529191223b5e3d476a2364dd476c0409_17.png)

*   **务必点击确认**：在弹出的“确定要提交吗？”或类似提示框中，选择“确认”或“确定”。这一步就像出门后确认锁门一样重要，确保信息已成功提交。
*   **提交成功反馈**：成功提交后，页面会显示“已成功加入创作者分成计划”等提示信息。

![](img/529191223b5e3d476a2364dd476c0409_19.png)

![](img/529191223b5e3d476a2364dd476c0409_21.png)

提交成功后，您的导师或邀请人可能会为您提供起步素材，例如剪辑好的视频文件，并指导您进行首次发布。这些素材可能来自拥有正规版权的素材库，确保了内容发布的合规性。

本节课中我们一起学习了填写视频号邀请函的完整流程：从查找账号信息，到逐一填写昵称、粉丝数、邀请人、实名信息和手机号，最后确认提交。请严格按照步骤操作，确保信息准确无误，为您的视频号运营之路打下坚实基础。
