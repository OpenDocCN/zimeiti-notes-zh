# 抖店运营保姆级教学：P17：4.用户定位 🎯

在本节课中，我们将要学习抖音运营中一个至关重要的环节：用户定位。理解并精准定位你的目标用户，是创作出热门内容、实现商业变现的基础。

## 用户定位的核心逻辑

上一节我们介绍了账号搭建的基础，本节中我们来看看如何找到你的核心观众。用户定位的核心是思考：我们准备给什么样的人，发什么样的作品。

我们可以将抖音的推荐逻辑简化为一个流程：

![](img/9c9aeafdfa683e88865ca62e2169d198_1.png)

1.  **创作者发布视频**。
2.  **抖音平台**通过分析账号、作品内容，并结合**用户的历史兴趣**，将视频推送给可能感兴趣的用户。
3.  **用户**看到视频后，产生**观看、点赞、评论、转发**等行为反馈。
4.  **抖音平台**根据这些正向反馈，将视频推送给**更多具有相似特征的用户**，从而形成热门传播。

因此，想要视频获得高播放量，必须从源头开始，对你的目标用户进行深入分析。虽然我们的终极目标是变现，但**视频播放量是变现的前提**。在分析用户时，必须将核心用户的商业价值考虑在内，并针对他们来设计和拆解视频内容的价值。

## 抖音平台用户现状

在开始定位前，我们需要了解抖音平台的整体用户画像。抖音日活跃用户已超7亿，人均使用时长超过2小时，这意味着任何时间、地点和行业都有海量活跃用户。

以下是抖音用户的主要特征：

![](img/9c9aeafdfa683e88865ca62e2169d198_3.png)

![](img/9c9aeafdfa683e88865ca62e2169d198_4.png)

*   **性别比例**：女性用户占比约55%。
*   **年龄分布**：18-30岁用户是主力军，占比高达71%。
*   **城市分布**：超过60%的用户分布在一、二线城市。
*   **兴趣偏好**：排名前五的兴趣领域依次是：教育、美食、旅游、搞笑、娱乐。其中教育和美食占比最高。
*   **购物动机**：年轻群体（18-30岁）中，91.8%的网购动机是“自己喜欢”或“需要买给自己”。
*   **内容偏好**：这部分年轻用户喜欢的人物和内容特质包括：
    1.  令人心情愉悦或搞笑幽默的。
    2.  高颜值、可爱、漂亮的。
    3.  具有爱国情怀和社会正能量的。
    4.  展现认真生活、处事态度的。

![](img/9c9aeafdfa683e88865ca62e2169d198_5.png)

在创作视频或打造人设时，需要参考这些普遍的内容特质。

## 如何定位目标用户

定位目标用户时，数据来源可以是对标账号分析、行业报告，或是企业自身的线下消费者数据。

以下是构建目标用户画像的基础分析维度：

*   **性别**：男 / 女
*   **年龄**：18岁及以下、18-28岁、28-38岁、38-48岁、48-58岁、58岁及以上
*   **地域**：一线城市、二线城市、三线城市、四线城市等
*   **人群特征**：日常白领、熬夜人群、家庭主妇、学生等
*   **消费水平**：经济型商品购买者、豪华型商品购买者
*   **电商相关维度**（如适用）：
    *   **购买时间偏好**：上午、中午、下午、晚上
    *   **价格敏感度**：高、中、低
    *   **囤货偏好**：高、中、低
*   **用户兴趣**：抖音平台将兴趣划分为17个大类，包括金融、汽车、科技、娱乐、服饰、教育、母婴、法律、体育、游戏、美食、生活、家装、美妆、旅游、房产、商务服务。

## 用户的商业价值分层（5A模型）

对于品牌商家而言，抖音采用5A模型对用户进行分层，以衡量其商业价值：

1.  **A1（了解感知）**：即“我知道”阶段。用户被动接收信息，是品牌知名度的来源。
2.  **A2（吸引兴趣）**：即“我喜欢”阶段。增加用户对品牌的好感，创造短期记忆。
3.  **A3（主动问询）**：即“我相信”阶段。引发用户好奇，促使其主动搜索品牌信息，是**最接近成交转化**的高价值人群。
4.  **A4（购买行动）**：即“我购买”阶段。用户完成购买及后续的消费、使用行为。
5.  **A5（拥护推荐）**：即“我推荐”阶段。用户成为品牌忠实粉丝，会复购并主动向他人推荐。

此外，还有规模最大的 **O人群（潜在人群）**，指在平台内活跃但尚未与品牌产生互动，等待被吸引的广大用户。

## 进阶用户描述维度

除了以上基础维度，还可以从更立体的角度描述用户：

![](img/9c9aeafdfa683e88865ca62e2169d198_9.png)

*   **社会属性**：婚否、家庭结构、职业、收入、资产等。
*   **人格特质**：如果上述数据难以获取，从人格类型出发分析用户，对内容创作同样具有指导意义。

以下是依据人格类型进行内容策划的示例：

*   **忠诚型人格**：性格谨慎、多疑、保守。针对他们，视频内容应**强调真实感和接地气**。
*   **理想主义者**：偏感性。视频内容应**侧重提供情绪价值**。
*   **艺术家型人格**：乐于接受新体验。可以发布**创意十足或挑战体验型**的内容。
*   **学者型人格**：对知识类内容更感兴趣，且**互动意愿较高**。
