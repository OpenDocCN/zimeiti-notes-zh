在本节课中，我们将学习小红书笔记的三种内容修改方式：完全复制、主体复制和格式复制。掌握这些方法，可以有效降低笔记内容的重复率，提升笔记在平台内的权重和曝光机会。

上一节我们介绍了如何选择用于复制的产品、引流内容和分享内容。本节中，我们来看看在选定内容后，如何进行具体的修改操作。

以下是三种核心的修改方式：

**1. 完全复制**
完全复制是指对从其他平台（如1688、抖音）或小红书内直接获取的完整笔记（包括视频、图片、文案），进行三次小修改。目的是降低系统判定的重复率，避免因内容高度重复而被屏蔽或降低权重。
*   **视频修改**：使用剪辑软件（如剪映）将完整视频分割成三段，然后重新组合导出。公式可表示为：`修改后视频 = 分割(原视频, 3段) + 重组`。此操作可排除约90%-95%的重复数据。
*   **图片修改**：使用图片编辑工具，在不显眼处添加细微修改（如添加小色点）。同样进行2-3次操作，以达到降低重复率的目的。
*   **文字修改**：更改文字的字体、大小、简繁体格式，或调整排版方式（如横排变斜排）。

完全复制操作简单，适合新手或用于引流，但因其内容同质化较高，笔记本身的权重提升有限。

**2. 主体复制**
主体复制指只复制产品的核心展示部分（如主图、视频演示），但文案部分需要自己重新编写。适用于图片和文字类笔记。
*   **操作核心**：保留原笔记的视觉框架或产品演示逻辑，但替换掉所有描述性文字，融入自身产品的卖点和活动信息。代码逻辑可类比为：`新笔记 = 原笔记.视觉框架 + 自定义文案`。

这种方式比完全复制更费时，但能更好地传递自身产品信息，减少文案层面的直接竞争。

**3. 格式复制**
格式复制指保留原笔记优秀的文案结构和表达格式，但替换掉其中的产品图片和核心关键词。
*   **操作核心**：当遇到文案精彩但图片普通或产品不符的笔记时，可采用此方法。保留其文案逻辑和格式，仅更换主图并将产品关键词替换为自己的。例如：`新笔记 = 原笔记.文案格式 + 自定义产品图 + 替换关键词(原产品, 新产品)`。

![](img/b1d5153f91839525ae508e86c65ae5e0_16.png)

这种方式能借鉴爆款文案的吸引力，同时展示自己的产品。

需要注意的是，无论采用哪种复制方式，都必须进行修改。不修改直接发布，笔记几乎100%会被系统判定为重复内容，难以获得流量。进行修改后，可以规避掉平台上大量由“店群”模式产生的重复商品和笔记，为自己争取到有效的竞争空间。
