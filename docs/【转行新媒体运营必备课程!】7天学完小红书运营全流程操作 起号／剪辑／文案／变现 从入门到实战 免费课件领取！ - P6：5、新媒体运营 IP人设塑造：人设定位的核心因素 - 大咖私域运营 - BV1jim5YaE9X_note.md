1.  **角色**
    角色是指你在特定赛道中扮演的专业身份。你提供什么方向的内容，就需要让别人知道你在这个领域的专业性。例如：
    *   如果你是**装修博主**，你需要展示自己真实的装修过程、遇到的坑以及最终成果。
    *   如果你是**母婴赛道**的创作者，你可以是分享亲身经验的**宝妈**，也可以是提供专业知识的**护理专家**。
    *   如果你是**情感赛道**的创作者，你可以是**心理学顾问**，也可以是分享自身蜕变经历的**成长型人设**（例如，分享如何走出家庭或职场困境）。

    公式可以概括为：**人设角色 = 赛道方向 + 专业身份**

2.  **内容**
    内容是你人设的基石。你需要持续提供与你所选赛道垂直相关、对用户有价值且有特色的内容。内容的垂直性至关重要，它影响账号标签和平台的流量分发。
    *   如果内容杂乱（例如，第一篇讲儿童教育，第二篇讲宠物），会打乱账号标签，导致推荐不精准，影响数据表现。
    *   核心原则：**你做什么赛道，就持续提供什么赛道的内容。**

3.  **形式**
    形式是指你呈现内容的具体方式，主要包括：
    *   **产出场景**：是在固定的室内场景，还是多变的室外、车内等场景？可以参考对标账号的常用场景。
    *   **呈现方式**：是真人出镜口播，还是仅使用配音（可通过`剪映`等工具实现）？这取决于你能否流畅地表达。
    *   **内容类型**：是**剧情演绎型**、**干货分享型**，还是**生活记录型**？每种类型各有优劣：干货型需要精心准备文案脚本；生活记录型需要大量拍摄素材；剧情型创作难度相对较高。

4.  **记忆点** 🧠
    记忆点是让用户记住你的关键，需要通过设计固定的环节来强化。以下是几种常见的记忆点设计方法：
    *   **视频开头前置**：包括**固定动作**、**特效**、**开场白**。将这些元素放在视频前几秒，能使用户在刷到你多个视频时快速形成深刻记忆。
    *   **贯穿全程**：包括**统一的BGM（背景音乐）** 和**统一出现的物品**（如特定的台灯、装饰品）。它们伴随整个视频，能营造独特的氛围和识别度。
    *   **视觉统一**：**固定风格的服装**也能成为记忆点，帮助用户建立视觉联想。

    设计记忆点的核心思路是：**将强化识别的元素（动作、开场）前置，将营造氛围的元素（BGM、物品、服装）贯穿全局。**
