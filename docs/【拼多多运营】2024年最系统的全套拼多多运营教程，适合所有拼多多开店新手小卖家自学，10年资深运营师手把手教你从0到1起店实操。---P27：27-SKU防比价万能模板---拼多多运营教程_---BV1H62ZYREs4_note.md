# 拼多多运营教程：P27：SKU防比价万能模板 📝

### 1. 百货通用型 🛒

![](img/e9ebc0e36827535dc924d2d35506767a_8.png)

百货类产品的SKU命名核心是**营造紧迫感和超值感**。其特点是结合**倒计时、件数和赠品**来撰写名称。

![](img/e9ebc0e36827535dc924d2d35506767a_9.png)

**命名公式**：`活动倒计时 + 特惠件数 + 送XX赠品`

例如，销售蚊香架产品：
*   **SKU1（引流款）**：`【活动倒计时1小时】特惠56件！送蚊香一盒 | 金色鸟笼蚊香架单装` - 价格可设为较低的5.9元，用于吸引点击。
*   **SKU2（主推款）**：`【活动倒计时1小时】特惠56件！送蚊香一盒 | 金色鸟笼蚊香架两装` - 价格设为9.9元，提供更高性价比选择。

这种结构通过倒计时制造紧迫感，通过赠品提升价值感，引导消费者购买主推SKU。

### 2. 食品类目型 🍰

食品类买家最关心**口味和性价比**。SKU命名需要巧妙组合这些元素，突出分量和优惠。

![](img/e9ebc0e36827535dc924d2d35506767a_16.png)

![](img/e9ebc0e36827535dc924d2d35506767a_17.png)

**命名思路**：`突出口味选择 + 强调“加量不加价”`

例如，销售奶糕产品：
*   **规格（口味）**：`芒果味`、`草莓味`、`芒果草莓混合味`
*   **SKU1**：`【一大袋20包】送4包，共24包 | [价格]`
*   **SKU2**：`【一大袋20包】送8包，共28包 | [价格]`

通过明确写出“送X包，共计X包”，比单纯写“28包”更能给消费者带来划算的心理感受，实现差异化。

### 3. 3C数码型 📱

3C数码类产品SKU要营造“**技术先进、功能强大**”的印象。特点是**用好话术、强调发货快、并写满18个字的卖点**。

![](img/e9ebc0e36827535dc924d2d35506767a_21.png)

**命名公式**：`年份/新款 + 核心功能卖点（堆砌） + 价格`

![](img/e9ebc0e36827535dc924d2d35506767a_22.png)

例如，销售多功能闹钟：
*   **SKU1**：`2024新款急速发货 | 13种灯+音箱+闹钟+无线充+大尺寸 | [价格]`
*   **SKU2**：`2024新款急速发货 | 闹钟+无线充+9种灯+音箱+小身材 | [价格]`

通过调整卖点顺序和组合，即使产品相似，也能创造出不同的SKU名称。对于配件类产品，可采用以下模式：
*   **SKU1（引流款）**：`【今年新款】拍一发五超级划算限时抢 | 3.3元`
*   **SKU2**：`防折防断充电线头保镖加厚10个装 | [价格]`

### 4. 汽配型 🚗

![](img/e9ebc0e36827535dc924d2d35506767a_27.png)

汽配买家注重**质量、售后和时效**。SKU命名要传递“**工厂直销、保障齐全、买到即赚**”的信息。

**命名公式**：`质保承诺 + 工厂冲量/亏本 + 加急发货 + 具体规格`

例如，销售洗车水枪：
*   **规格**：`质保3年 工厂冲榜亏量 加急发货`
*   **SKU1**：`15米防爆管+七档模式水枪+万能接头 | [价格]`
*   **SKU2**：`20米防爆管+七档模式水枪+万能接头 | [价格]`

![](img/e9ebc0e36827535dc924d2d35506767a_29.png)

通过强调质保和工厂直销，打消买家疑虑，促进下单。

### 5. 电器型 🔌

![](img/e9ebc0e36827535dc924d2d35506767a_33.png)

![](img/e9ebc0e36827535dc924d2d35506767a_35.png)

电器类产品需要突出**技术背景和正品保障**。特点是**强调外国技术、国产价格、销量榜单和高功率**。

**命名公式**：`技术背景 + 榜单排名 + 官方正品 + 具体参数`

![](img/e9ebc0e36827535dc924d2d35506767a_37.png)

例如，销售电机产品：
*   **规格**：`德国无刷纯铜电机 榜单TOP1`
*   **SKU1**：`官方原装正品 690W 深邃黑 | [价格]`

“官方原装正品”等字眼能极大提升电器产品的可信度和转化率。

![](img/e9ebc0e36827535dc924d2d35506767a_39.png)

### 6. 洗护型 🧴

![](img/e9ebc0e36827535dc924d2d35506767a_42.png)

![](img/e9ebc0e36827535dc924d2d35506767a_43.png)

洗护类产品（如香薰、洗发水）要塑造**高端、持久、专业的形象**。特点是使用**先进词汇、强调效果天数和高端的形容词**。

![](img/e9ebc0e36827535dc924d2d35506767a_45.png)

**命名公式**：`科技感前缀 + 持久天数 + 高端场景/香型`

例如，销售香薰机：
*   **SKU1（主机）**：`AI自控喷香机 持续留香60天 | [价格]`
*   **SKU2（引流精油）**：`专用香薰精油（无香薰机） | 低价`
*   **SKU3（精油套餐）**：`聚划算！5款香水各一瓶共5瓶 | [价格]`
*   **SKU4（高端精油）**：`人气推荐·希尔顿高端酒店香型3瓶装 | [价格]`

关联高端酒店等场景，能有效提升产品格调和购买欲望。

### 7. 服装型 👕

![](img/e9ebc0e36827535dc924d2d35506767a_48.png)

服装类SKU防比价的关键在于**在款式名称中加入自定义编码**。这既能**防止比价**，又便于**后台库存管理**。

![](img/e9ebc0e36827535dc924d2d35506767a_50.png)

![](img/e9ebc0e36827535dc924d2d35506767a_51.png)

**命名公式**：`件数 + 自定义字母/数字码 + 颜色 + 另类尺码描述`

例如，销售T恤：
*   **款式名称**：`三件套 CH黑卡其SE鸟声色XXL`
    *   “CH”、“SE”、“XXK”等为自定义编码，无实际含义，主要用于防比价和库存管理。
*   **尺码**：按常规标注，如`M`、`L`、`XL`、`XXL（适合150-170斤）`。

在款式名中加入复杂编码，使同行难以直接复制，从而实现有效防比价。
