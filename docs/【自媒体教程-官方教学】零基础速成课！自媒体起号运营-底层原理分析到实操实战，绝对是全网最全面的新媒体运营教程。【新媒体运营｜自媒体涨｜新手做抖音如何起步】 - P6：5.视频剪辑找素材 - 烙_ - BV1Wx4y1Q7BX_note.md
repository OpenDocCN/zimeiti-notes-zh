# 自媒体运营：P6：视频剪辑找素材 🎬

在本节课中，我们将要学习视频剪辑类内容创作的第一步：如何高效、合规地寻找素材。这是制作视频的基础，也是避免违规的关键。

![](img/5834cda42c8c88fb173c030a9a31fa17_1.png)

## 素材来源概览

![](img/5834cda42c8c88fb173c030a9a31fa17_3.png)

视频剪辑类内容涵盖电影、电视剧、综艺等多种形式。要制作这类视频，首先需要找到合适的素材。寻找素材主要可以从以下三个渠道入手。

以下是主要的素材来源渠道：

1.  **视频平台**：所有主流的自媒体平台都是潜在的素材库。
2.  **影视网站**：专注于影视内容的网站，通常提供无版权或可商用的素材。
3.  **影视推广平台**：这类平台不仅提供素材，还可能通过播放量提供收益。

## 渠道一：视频平台

上一节我们介绍了素材的三大来源，本节中我们来看看如何从视频平台寻找素材。

视频平台是指常见的自媒体内容平台。我们可以直接在这些平台上搜索关键词来寻找所需片段。

以下是具体操作步骤：

![](img/5834cda42c8c88fb173c030a9a31fa17_5.png)

![](img/5834cda42c8c88fb173c030a9a31fa17_7.png)

*   打开任意搜索引擎（如百度）。
*   搜索目标视频平台（例如“西瓜视频”）。
*   进入该平台的网页版。
*   在平台内使用关键词进行搜索。

例如，想寻找综艺或网红直播片段（如“大杨哥”、“小杨哥”），直接在平台搜索栏输入这些名字即可找到大量相关视频。

![](img/5834cda42c8c88fb173c030a9a31fa17_9.png)

## 正确获取素材的方法

![](img/5834cda42c8c88fb173c030a9a31fa17_11.png)

找到素材后，关键在于如何正确获取和使用。直接下载完整视频并上传，属于“搬运”行为，目前平台审核严格，通过率极低。

正确的做法是：从大量视频中，选取与自身视频主题相符的片段。每个片段的时长应控制在**10秒钟**左右。这样处理后的素材在平台审核时更容易通过，能有效避免违规。

我们可以再举一个例子。如果想制作关于明星沈腾的内容，同样在视频平台搜索“沈腾”，就能找到丰富的素材。核心是围绕主题（如“沈腾的搞笑名场面”）来选取多个短片段，而非使用长视频。

## 剪辑工具准备

![](img/5834cda42c8c88fb173c030a9a31fa17_13.png)

素材问题解决后，我们需要使用工具进行剪辑。在众多软件中，剪映因其易用性和功能全面，是较为推荐的选择。

接下来，我们将确定一个视频主题。例如，制作一个“沈腾搞笑名场面”的合集。视频中的所有片段都应包含沈腾本人，并围绕“搞笑”这一核心主题展开。
