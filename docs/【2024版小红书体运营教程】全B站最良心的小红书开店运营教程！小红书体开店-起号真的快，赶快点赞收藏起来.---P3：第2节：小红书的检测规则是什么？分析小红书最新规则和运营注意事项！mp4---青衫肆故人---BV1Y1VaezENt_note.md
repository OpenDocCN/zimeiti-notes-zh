# 小红书运营教程：P3：第2节：小红书平台检测规则详解 🔍

在本节课中，我们将要学习小红书平台的核心检测规则。了解这些规则是安全、有效运营账号的基础，能帮助我们避免违规，确保内容顺利发布和传播。

上一节我们介绍了小红书运营的基本概念，本节中我们来看看平台具体如何检测内容，以及我们需要特别注意哪些方面。

---

小红书平台主要通过以下四个方面对账号和内容进行检测与管理。

以下是具体的检测维度与注意事项：

1.  **账号与内容检测**
    *   平台会检测**个人资料、笔记内容、私信以及评论**。
    *   这些位置不得含有微信号、微信二维码、代购、水印、推广淘宝店铺等明显的营销词汇。
    *   平台管理相对宽松的部分是：个人资料中允许填写邮箱；可以使用微博名称或谐音进行间接引流，再通过话术引导至微信。

2.  **设备关联检测**
    *   小红书会检测登录设备。建议**同一台设备最多只登录三个账号**，超出此数量可能增加账号风险。

3.  **内容规范检测**
    *   笔记内容不得诱导用户。例如，“关注领奖”、“点赞抽奖”等属于小红书明令禁止的诱导行为。

4.  **原创度检测**
    *   平台对原创内容的保护程度较高。
    *   切勿将同一篇文章在多个小红书账号上重复发布。推广时，应遵循**一篇文章只由一个账号发布**的原则。

除了以上四点，还有一个至关重要的规则是敏感词检测。在日常写作中，我们可能会无意中使用到平台禁止的词汇。

以下是需要警惕的敏感词类别与检测方法：

*   **常见敏感领域**：例如“手机卡”、“金融”、“电子烟”、“代购”、“医疗用品”等领域的词汇，应尽量避免触碰。
*   **敏感词检测工具**：若不确定文章是否包含敏感词，可以使用专门的检测网站进行排查。

其操作逻辑如下：
1.  将写好的文章内容粘贴到检测网站的输入框中。
2.  点击检测按钮。
3.  网站会标记出内容中存在的敏感词。

例如，在一个检测网站中输入包含“代购”的句子，检测结果会明确提示该词为敏感词。
