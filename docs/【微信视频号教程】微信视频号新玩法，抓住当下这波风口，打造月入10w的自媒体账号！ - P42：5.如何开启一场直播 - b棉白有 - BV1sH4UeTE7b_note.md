# 微信视频号教程：P42：如何开启一场直播 🎥

在本节课中，我们将学习如何在微信视频号中开启一场直播。我们将从直播前的准备工作开始，逐步讲解直播的发起、设置、进行中的操作，以及直播结束后的处理。

---

## 直播前的准备工作

![](img/8310b0702803ccf65a43afd944c1fd6f_1.png)

![](img/8310b0702803ccf65a43afd944c1fd6f_2.png)

上一节我们介绍了视频发布与预热，本节中我们来看看直播前需要做哪些具体准备。

![](img/8310b0702803ccf65a43afd944c1fd6f_3.png)

![](img/8310b0702803ccf65a43afd944c1fd6f_4.png)

1.  **确定商品与发布视频**：在决定直播后，首先需要确定你要推广或销售的商品。围绕该商品发布相关视频内容。
2.  **等待视频流量**：发布的视频需要获得一定的初始流量，建议等待视频播放量达到 **1000** 左右，这有助于为直播引流。
3.  **准备直播话术**：提前准备好直播过程中要讲解的内容和话术。
4.  **准备备用视频**：为防止直播开始时观众进入缓慢，可以提前准备几个用于暖场的视频，并保存到草稿箱，以便在直播中随时发布。
5.  **创建直播预告**：如果你无法立即开播，或者希望提前锁定观众，务必创建一个直播预告。这能让对你的直播感兴趣的观众提前预约。当你开播时，预约过的观众会第一时间收到开播提醒。

## 发起与设置直播

准备工作就绪后，我们就可以进入直播的发起与设置环节。

![](img/8310b0702803ccf65a43afd944c1fd6f_8.png)

以下是发起直播的具体步骤：

![](img/8310b0702803ccf65a43afd944c1fd6f_10.png)

1.  在视频号中点击“发起直播”按钮，然后选择“直播”选项。
2.  **设置直播主题**：直播主题通常可以写为“**视频同款热卖中**”。你也可以参考其他成功账号的写法进行设置。
3.  **添加商品**：在商品设置区域，提前添加你计划在直播中销售的商品。可以添加单品或多个商品。
    ```markdown
    操作路径：直播设置 -> 添加商品
    ```
4.  **美化直播画面**：在画面设置中，可以使用美颜、滤镜、人物特效等功能来优化直播画面效果。
5.  **设置直播贴图**：可以使用如“微微商水印相机”等工具制作简单的介绍贴图。具体方法可参考对标账号的做法。
6.  **设置直播模式**：如果是教学或测试，可以将直播模式设置为“**部分可见**”，以避免无关用户进入影响流程。对于卖货直播，通常选择公开模式。
7.  **标记地理位置**：在卖货直播中，需要标记一个地理位置，完成后即可点击“开始直播”。

## 直播中的关键操作

![](img/8310b0702803ccf65a43afd944c1fd6f_16.png)

直播开始后，为了提升观看体验和转化效果，有一些关键操作需要注意。

![](img/8310b0702803ccf65a43afd944c1fd6f_18.png)

以下是直播进行中的几个要点：

![](img/8310b0702803ccf65a43afd944c1fd6f_20.png)

1.  **讲解并弹出商品**：在讲解商品时，务必点击直播界面中的购物车，将对应商品弹出。这样观众才能看到商品链接和详情，从而**提升转化率**。
    ```markdown
    核心操作：点击购物车 -> 选择讲解的商品 -> 弹出
    ```
2.  **使用背景音乐**：直播时可以添加背景音乐，这不会影响主要功能，还能活跃气氛。
3.  **管理评论与上墙**：如果某条观众评论特别有价值或对你有利，可以长按该评论，选择“**点上墙**”。这样该评论会固定在直播间醒目位置，让新进入的观众也能看到。
4.  **遵守直播规范**：直播过程中，注意不要使用违规词汇，确保直播内容符合平台规定。

## 结束直播与后续处理

![](img/8310b0702803ccf65a43afd944c1fd6f_22.png)

当直播内容完成后，需要正确地结束直播并处理后续事宜。

以下是结束直播的步骤：

1.  点击直播间右上角的“**×**”号按钮。
2.  在弹出的确认框中，选择“结束直播”。
3.  结束直播后，系统会提示你是否“**生成直播回放**”。你可以根据需要选择生成，回放视频可供观众回顾或你进行复盘。
4.  此外，在直播过程中，你可以随时将直播间链接分享给他人，邀请他们观看。
