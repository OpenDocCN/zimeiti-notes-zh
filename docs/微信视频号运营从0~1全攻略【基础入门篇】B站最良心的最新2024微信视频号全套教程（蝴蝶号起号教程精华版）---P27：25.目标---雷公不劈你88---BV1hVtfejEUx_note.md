## 不留退路，抓住风口

因为有退路，你会错过很多风口。回顾历史，每一次重大机遇出现时，许多人因为“有退路”、“觉得难”、“不相信”而错失良机。

*   **改革开放（80年代）**：是机会，但很多人觉得下海经商太远、太冒险。
*   **房地产**：是机会，但很多人已背负房贷或觉得门槛高。
*   **淘宝**：是机会，但早期很多人认为网上购物是骗人的。
*   **抖音**：是机会，但很多人觉得自己学不会。

![](img/510b45dad756fc5f1acc7353cc87c24d_6.png)

**核心规律**：中国大约每5到10年就会出现一次重大的时代机遇，即所谓的 **`风口`** 或 **`红利期`**。

![](img/510b45dad756fc5f1acc7353cc87c24d_3.png)
![](img/510b45dad756fc5f1acc7353cc87c24d_4.png)

当前，**视频号就是明确的风口**。这一次，我们需要以不留退路的决心去把握它。
