# 剪映专业版教程：P10：菜单命令详解 🎬

在本节课中，我们将详细学习剪映专业版界面顶部的菜单命令。了解这些菜单功能，能帮助你更高效地管理项目、调整软件设置，并掌握基础编辑操作。

## 项目名称管理

创建剪映项目后，会进入视频编辑界面。界面顶部区域默认以日期方式自动保存项目名称。

*   **重命名项目**：你可以单击选中该名称，然后按键盘上的 **`Delete`** 键或 **`Backspace`** 键将其删除，之后即可输入新的名称进行重命名。
*   **按键位置**：`Delete` 键通常位于键盘右侧区域。

了解如何管理项目名称是进行有序创作的第一步。

## 菜单命令总览

紧接着名称区域右侧的是菜单命令栏。单击界面顶部的菜单按钮，可以展开下拉菜单，其中包含多个功能模块。

以下是各主要菜单项的功能介绍：

*   **文件**：执行新建草稿、导入、导出等项目级操作。
*   **编辑**：进行撤销、恢复、复制、剪切、粘贴、删除等基础编辑操作。
*   **布局模式**：设置软件界面布局，包括默认布局、媒体素材优先、播放器优先、属性调节优先及时间线优先模式。
*   **更多操作**：查看用户协议、隐私条款、第三方版权信息，检测当前软件运行环境，以及查看软件版本号以确认是否需要更新。
*   **帮助**：访问帮助中心，提交使用中遇到的故障或问题意见反馈，查看软件快捷键列表（后续课程会详细讲解）。
*   **设置**：单击后可对软件进行全局配置。

## 全局设置与界面导航

在“设置”菜单中，你可以对软件进行个性化配置。

![](img/58450dc1b3229c736701c8d08e180cd1_1.png)

设置界面主要包括：

*   **草稿设置**：管理项目草稿的相关参数。
*   **剪辑设置**：调整与视频剪辑相关的各项偏好。

![](img/58450dc1b3229c736701c8d08e180cd1_1.png)

![](img/58450dc1b3229c736701c8d08e180cd1_3.png)

*   **性能设置**：优化软件运行性能，可根据电脑配置进行调整。

有兴趣的同学可以自行探索各项设置的具体内容。

![](img/58450dc1b3229c736701c8d08e180cd1_3.png)

除了菜单，界面中还有便捷的导航控制：

*   **返回首页**：你可以单击左上角的“返回首页”按钮，或者单击界面右上角的关闭叉号（`×`），两者都可以返回到剪映的启动首页。

![](img/58450dc1b3229c736701c8d08e180cd1_5.png)

*   **退出剪映**：选择相应选项即可完全退出剪映专业版应用程序。

![](img/58450dc1b3229c736701c8d08e180cd1_5.png)
