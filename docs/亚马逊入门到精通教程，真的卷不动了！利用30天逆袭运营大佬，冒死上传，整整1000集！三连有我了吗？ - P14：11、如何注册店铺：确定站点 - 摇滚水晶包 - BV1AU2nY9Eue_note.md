# 亚马逊入门到精通教程：P14：11、如何注册店铺：确定站点 🎯

在本节课中，我们将要学习注册亚马逊店铺的第一步：如何选择合适的销售站点。了解各个站点的特点与规则，是开启亚马逊之旅的重要基础。

---

上一节我们介绍了亚马逊的运营模式，本节中我们来看看注册店铺的具体流程。注册店铺的第一个环节，是了解并确定你要在哪个亚马逊站点进行销售。

亚马逊有四大主流站点，也有一些规模较小的站点。以下是几个主要站点的介绍：

*   **北美站**：包含美国、加拿大和墨西哥三个国家。北美是亚马逊的主要消费区域，拥有近5亿人口。该站点的月租费用为 **39.99美元**。
*   **欧洲站**：包含英国、德国、法国、西班牙、意大利、荷兰、瑞典、波兰共8个国家。该地区拥有约3.4亿网上购物用户。欧洲站的月租费用为 **25英镑**。
*   **日本站**：亚马逊在日本电商平台的地位很高，类似于淘宝在中国。日本站的月租费用为 **4900日元**。
*   **澳洲站（澳大利亚站）**：该地区互联网渗透率达88%，人均GDP位居全球前十，跨境消费率全球第一。澳洲站的月租费用为 **49.95澳元**。

此外，像印度站、中东站和新加坡站等属于规模较小的站点。

站点简单了解完之后，我们面临一个实际问题：做亚马逊时，应该从哪个站点开始？

首先需要明确一个规则：根据亚马逊的规定，每个卖家原则上只能开通一个店铺，但这个店铺可以同时激活并运营多个站点。例如，开通一个店铺后，可以同时激活北美、欧洲、日本和澳洲这四大站点。

那么，我们建议从哪个站点开始呢？我们的建议是**先做北美站**。

选择北美站主要有两个原因：
1.  **规则与福利最完善**：亚马逊是美国创办的公司，因此北美站是其规则最健全、卖家福利最多的一个站点。
2.  **享受“全球开店”计划**：这是亚马逊的一项重要福利。当你开通了北美站后，再去开通欧洲站、日本站和澳洲站时，这些**额外站点的月租可以全部免除**。这意味着，如果你的店铺同时开通了北美、欧洲、日本、澳洲四个站点，每月只需支付**一份北美站的月租（39.99美元）**即可。

公式表示即：`总月租 = 北美站月租（39.99美元） + 0（其他站点月租）`

因此，为了最大化利用平台规则并节省成本，我们建议在注册店铺时，优先开通北美站。
