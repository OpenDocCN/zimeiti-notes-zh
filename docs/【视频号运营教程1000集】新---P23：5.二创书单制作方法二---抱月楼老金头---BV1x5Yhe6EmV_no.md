## 概述
在本节课中，我们将学习制作二创书单视频的第二种方法。我们将通过一个具体案例，详细讲解如何提取原视频素材、替换画面、重新配音并最终合成一个全新的原创视频。整个过程将使用剪映等工具，步骤清晰，适合初学者上手。

## 核心操作流程

上一节我们介绍了二创书单的基本概念，本节中我们来看看如何通过替换素材和重新配音来制作一个二创视频。

![](img/544afa1afba8af78a0338558a9e33318_13.png)

### 第一步：寻找并处理原视频
首先，我们需要找到一个合适的原视频作为创作基础。

以下是具体操作步骤：
1.  在抖音搜索目标书籍（例如“走进大学城”），选择“商品”并按“销量”排序，可以找到相关的带货视频。
2.  进入达人主页，寻找合适的视频素材。例如，一个关于“汕头大学”的1分58秒视频。
3.  复制视频链接，使用去水印工具去除视频水印。
4.  将去水印后的视频导入剪映。

### 第二步：分割视频与提取文案
原视频可能前后声音不一致，我们需要截取需要的部分并提取文案。

以下是具体操作步骤：
1.  在剪映中，将视频轨道拉长，便于精确到每一帧进行操作。
2.  在声音发生变化的位置（例如53秒处）使用**分割**功能，将视频分成前后两部分。本例中，我们保留前半部分。
3.  将分割出的前半部分视频导出，备用。
4.  打开“配音神器”APP，通过“视频提取文案”功能，导入刚导出的视频，将画面中的文案文字提取出来。

### 第三步：生成新配音与替换画面素材
获得文案后，我们需要生成新的配音，并替换原视频中的部分画面，这是二创的核心。

以下是具体操作步骤：
1.  在“配音神器”中，为提取的文案选择合适的配音音色，生成新的音频文件并导出。
2.  回到剪映，处理原视频。对于需要替换的画面片段（例如一段3秒的“毕业典礼”镜头），先将其分割出来。
3.  在抖音搜索与片段内容匹配的新素材（如搜索“高校 毕业典礼”），下载并去水印。
4.  在剪映中，删除原视频的3秒片段，将下载的新素材导入并裁剪至同样时长。
5.  在片段衔接处添加转场效果（如“叠化”），并将新素材片段的**原声音量调整为0**。
6.  重复此过程，替换视频中其他可更换的画面（如“打高尔夫”、“宿舍环境”等片段），通常替换1-2个核心素材即可。

### 第四步：合成与调整
将所有新元素组合在一起，并进行最终调整，使视频完整流畅。

以下是具体操作步骤：
1.  关闭原视频的所有原声。
2.  导入第三步中生成的新配音音频。
3.  **对齐音频与画面**：检查新配音时长与视频画面是否匹配。如果不匹配：
    *   **音频过长**：可对视频进行“常规变速”，适当加快播放速度；或分割删除部分冗余画面。
    *   **音频过短**：可对视频进行减速；或添加相关空镜头、图片素材以延长画面。
4.  对于保留原声的片段（如视频后半部分人物口播），可以保留声音，但**替换画面**为其他相似场景，以减少原出镜人物的面部曝光，避免潜在版权或肖像权风险。
5.  为整个视频添加一个合适的背景音乐。
6.  使用剪映的“识别字幕”功能，为新的配音生成字幕。通过调整字幕样式和位置，**覆盖掉原视频的旧字幕**。
7.  最后检查视频，确保没有明显的其他平台水印（可通过放大画面遮盖或去水印工具处理）。
8.  导出视频，设置参数为：分辨率1080P，帧率60帧，码率选择推荐值。

## 总结
本节课我们一起学习了二创书单视频的制作方法。其核心思路是：**提取优质视频文案 -> 利用配音工具生成新音频 -> 替换原视频部分画面素材 -> 重新组合成新视频**。关键在于素材的替换与声画的对齐。初学者前期操作可能会较慢，但通过多次练习，熟练后可在很短时间内完成一个视频的制作。记住，剪辑技能的核心在于**熟能生巧**，大胆尝试和反复练习是进步最快的方式。
