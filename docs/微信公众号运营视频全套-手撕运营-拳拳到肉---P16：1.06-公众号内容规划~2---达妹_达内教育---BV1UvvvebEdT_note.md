## 宣传性账号的内容规划

![](img/bf260d8540aff5d7746c25cd88291137_12.png)

![](img/bf260d8540aff5d7746c25cd88291137_14.png)

宣传性账号是指企业运营公众号的主要目的是提高自身影响力，向受众宣传自己，但不会直接销售商品。运营此类公众号的核心目标是增粉。

对于宣传性账号，发布的内容可以分为两大类：**企业自身需要的信息**和**用户需要的信息**。

![](img/bf260d8540aff5d7746c25cd88291137_16.png)

![](img/bf260d8540aff5d7746c25cd88291137_17.png)

以下是企业自身需要发布的信息类型：

![](img/bf260d8540aff5d7746c25cd88291137_18.png)

![](img/bf260d8540aff5d7746c25cd88291137_19.png)

![](img/bf260d8540aff5d7746c25cd88291137_20.png)

*   **产品信息**：如新产品发布、产品更新等。
*   **公司信息**：如公司动态、重大决策等。
*   **企业新闻**：如公司获奖、参与行业大会等。
*   **公司活动**：指公司内部或对外举办的大型活动。
*   **福利活动**：指专门为用户设计、提供好处的互动活动。

![](img/bf260d8540aff5d7746c25cd88291137_21.png)

这些内容对企业自身很重要，但用户可能并不十分关心。

相比之下，用户需要的信息更能吸引关注和阅读。以下是用户需要的信息类型：

*   **行业资讯**：提供所在领域的动态、趋势和分析。
*   **干货知识**：分享实用的技巧、方法、案例或专业知识。
*   **福利活动**：用户可以直接参与并获得好处的活动。
*   **互动问答**：解答用户疑问，提供专业建议，增强互动。

发布用户需要的信息，能有效提升阅读量和粉丝粘性。

![](img/bf260d8540aff5d7746c25cd88291137_23.png)

因此，对于宣传性账号，建议遵循以下内容发布比例公式：

![](img/bf260d8540aff5d7746c25cd88291137_24.png)

**自身信息 : 用户信息 ≈ 3 : 7**

![](img/bf260d8540aff5d7746c25cd88291137_26.png)

多发布对用户有价值的信息，才能吸引并留住粉丝。如果发布自身广告过多，会导致**掉粉**和**降低账号活跃度**。

![](img/bf260d8540aff5d7746c25cd88291137_28.png)

在文章排序上，也有一个通用建议：

*   **头条**：通常发布对用户有价值的干货信息。
*   **其他条（如二条、三条）**：发布公司自身的信息。
*   只有当公司信息**特别重要**（例如公司上市、举办大型活动）时，才可将其放在头条发布，但需注意控制此类情况的频率。

![](img/bf260d8540aff5d7746c25cd88291137_30.png)

## 电商类账号的内容规划

![](img/bf260d8540aff5d7746c25cd88291137_32.png)

电商类账号是指以销售产品为直接目标的公众号，例如京东、拼多多等平台的官方账号。运营此类公众号的核心目的就是促进销售。

![](img/bf260d8540aff5d7746c25cd88291137_33.png)

![](img/bf260d8540aff5d7746c25cd88291137_34.png)

![](img/bf260d8540aff5d7746c25cd88291137_35.png)

电商类账号必须发布商品信息，但发布策略有讲究。公众号的推送不应是商品列表的简单罗列，而应发布**精选商品**和**活动信息**。

![](img/bf260d8540aff5d7746c25cd88291137_36.png)

![](img/bf260d8540aff5d7746c25cd88291137_38.png)

以下是电商类公众号适合发布的内容类型：

![](img/bf260d8540aff5d7746c25cd88291137_40.png)

*   **精选最低折扣**：整理当期最优惠的商品。
*   **应季/最新款式**：推荐符合季节潮流或刚上市的新品。
*   **专题整理**：围绕特定主题（如节日、场景）整理商品合集，例如“七夕送礼指南”。
*   **好评内容梳理**：汇总用户好评或打造爆款商品清单。
*   **抽奖/促销活动**：直接吸引用户参与的营销活动。

![](img/bf260d8540aff5d7746c25cd88291137_42.png)

这意味着，虽然公众号和电商网站都在卖货，但公众号内容应承担“**编辑筛选**”的角色，将海量商品中**最值得推荐、最有话题性、优惠力度最大**的部分，以内容的形式包装并推送给用户，为用户提供价值，而不仅仅是广告。

![](img/bf260d8540aff5d7746c25cd88291137_44.png)

本节课中我们一起学习了宣传性账号和电商类账号的内容规划方法。总结核心要点：

1.  **宣传性账号**：核心是平衡企业宣传与用户价值，遵循 **3:7** 的发布比例，多提供行业资讯、干货知识等用户需要的信息。
2.  **电商类账号**：核心是促进销售，但内容需经过精选和包装，以专题、折扣、新品等形式为用户提供购物指南，而不仅是商品罗列。

![](img/bf260d8540aff5d7746c25cd88291137_50.png)

理解不同账号的定位和目标，是制定有效内容策略的第一步。
