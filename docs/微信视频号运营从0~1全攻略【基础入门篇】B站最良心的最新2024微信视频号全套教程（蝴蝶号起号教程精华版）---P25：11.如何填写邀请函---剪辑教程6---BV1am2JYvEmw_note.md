# 微信视频号运营从0~1全攻略：P25：11.如何填写邀请函

在本节课中，我们将学习如何正确填写视频号创作者计划的邀请函。我们将一步步演示如何找到并填写你的视频号昵称、粉丝数、邀请人、实名信息及结算手机号，确保信息准确无误，以便成功提交申请。

![](img/6f3697bc16cc12ee012e3023bdc3969e_1.png)

## 第一步：打开微信应用

首先，你需要打开手机上的微信应用。这是所有后续操作的基础。

![](img/6f3697bc16cc12ee012e3023bdc3969e_3.png)

![](img/6f3697bc16cc12ee012e3023bdc3969e_4.png)

## 第二步：进入“发现”页面

打开微信后，点击底部导航栏的“发现”选项。

## 第三步：找到并进入“视频号”

![](img/6f3697bc16cc12ee012e3023bdc3969e_6.png)

![](img/6f3697bc16cc12ee012e3023bdc3969e_7.png)

在“发现”页面中，向上滑动屏幕。你会看到“视频号”的入口，其图标是一个黄色的蝴蝶标志。点击它即可进入视频号。

## 第四步：访问个人主页

进入视频号界面后，点击屏幕右上角的个人头像图标。这将进入你的个人主页。

## 第五步：查看账号信息

在个人主页中，向下滑动屏幕。你可以看到“我的视频号”区域，这里清晰地显示了你的**账号昵称**和**粉丝数量**。

![](img/6f3697bc16cc12ee012e3023bdc3969e_9.png)

例如，显示为：
```
昵称：天道酬勤
粉丝：39917
```
在填写邀请函时，账号昵称就填写“天道酬勤”，粉丝数填写“39917”。

![](img/6f3697bc16cc12ee012e3023bdc3969e_10.png)

![](img/6f3697bc16cc12ee012e3023bdc3969e_11.png)

## 第六步：填写邀请函详细信息

找到账号信息后，接下来需要填写邀请函的其他部分。以下是需要填写的项目列表：

*   **活动邀请人**：根据你加入计划的渠道，此处应填写“张良”或“张良老师”。
*   **实名信息**：此处必须填写你本人的真实姓名，例如“周建国”。
*   **结算手机号**：填写你用于接收验证码和后续结算的手机号码，例如 `13812345678`。注意号码中间不要有空格。

![](img/6f3697bc16cc12ee012e3023bdc3969e_13.png)

![](img/6f3697bc16cc12ee012e3023bdc3969e_14.png)

## 第七步：提交并确认

所有信息填写完毕后，点击页面底部的蓝色“提交”按钮。系统会弹出确认提示，请务必点击“确认”以完成提交。提交成功后，页面会显示“已成功加入创作者分成计划”的提示。
