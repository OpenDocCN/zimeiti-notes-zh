# 视频号运营：P8：视频号的制作者分成讲解 🎬

在本节课中，我们将详细讲解视频号创作者中心的核心功能，特别是与带货、佣金分成相关的操作。你将学习如何开通商品橱窗、添加商品、管理佣金以及处理账号申诉等关键步骤。

## 创作者中心概览

上一节我们介绍了视频号的基础设置，本节中我们来看看创作者中心的具体功能。进入创作者中心后，首先看到的是数据概览板块。这里可以查看账户的关注数据、粉丝画像和视频播放量等核心指标。

数据概览下方有热门课程和创意风向标模块，对于初学者而言，这些内容并非重点，可以暂时忽略。

## 加热工具与账号诊断

接下来，我们重点看一下“加热工具”。这个功能类似于抖音的“抖加”，是通过付费推广视频的方式。通常，我们运营自然流量时不需要使用此工具。

然而，加热工具可以帮助我们诊断账号状态。点击“账号诊断”，如果账号状态显示为绿色“正常”，则表明系统判定账号无异常。但需注意，有时系统显示正常，视频却无法加热，这可能意味着账号存在“隐性限流”问题。

如果遇到隐性限流，可以进行申诉。申诉入口有两个：
1.  在账号诊断页面，点击“点击此处申诉”的链接。
2.  在视频号私信中，查找“视频号团队”发送的消息，其中也可能包含申诉入口。

提交申诉时，需要填写申诉内容并上传相关证据。关于申诉的更多细节，我们将在后续课程中详细讲解。

## 开通与使用商品橱窗

![](img/024cf6fac4da07d40eebc183da758025_1.png)

![](img/024cf6fac4da07d40eebc183da758025_2.png)

现在，我们进入本节课的核心部分——带货功能。点击“带货中心”，可以查看今日的总销售额（GMV）、订单数、具体商品及销售来源等数据。

要开始带货，首先需要开通“商品橱窗”。如果你的账号尚未开通，界面会显示“暂无带货权限”。

![](img/024cf6fac4da07d40eebc183da758025_4.png)

以下是开通商品橱窗的步骤：
1.  点击“橱窗”或“前往缴纳”按钮。
2.  点击“充值缴纳”，然后选择“去认证”。
3.  选择“个人”类型，并上传与当前微信实名认证信息一致的身份证正反面照片。
4.  填写姓名、身份证号等信息，勾选同意协议后提交。
5.  认证审核通常需要10分钟至数小时。审核通过后，再次进入该页面，缴纳100元保证金即可成功开通。

![](img/024cf6fac4da07d40eebc183da758025_6.png)

开通后，你的橱窗界面将可以添加商品。

## 如何选品与添加商品

开通橱窗后，下一步是添加商品。你可以通过以下两种方式选品：
1.  在“选品中心”直接浏览或搜索商品。
2.  在“爆款商品榜”中，按分类（如食品生鲜、宠物绿植）寻找热销商品。

选择商品时，建议关注佣金比例。通常，我们选择佣金率在 **30%** 以上的商品，以保证收益。找到心仪商品后，直接点击“加橱窗”即可。

![](img/024cf6fac4da07d40eebc183da758025_8.png)

![](img/024cf6fac4da07d40eebc183da758025_9.png)

如果你想与特定商家合作，获取更高佣金，可以：
1.  在商品详情页找到并复制商家的联系方式（微信号或手机号）。
2.  通过微信联系商家，洽谈“定向佣金”合作。
3.  商家开通定向佣金后，你可以在“合作管理”或“带货邀约”板块查看并确认合作。

## 橱窗管理与佣金提现

![](img/024cf6fac4da07d40eebc183da758025_11.png)

对于已添加的商品，可以进行管理。若想移除某个商品，操作如下：
1.  点击“管理”。
2.  选择要移除的商品，点击“移除”。
3.  **关键步骤**：操作后务必点击右上角的“完成”按钮，否则移除不会生效。

在“带货中心”还可以查看“带货评分”和“保证金”状态。满足条件后，保证金可以在此申请提现。

![](img/024cf6fac4da07d40eebc183da758025_13.png)

![](img/024cf6fac4da07d40eebc183da758025_15.png)

关于佣金提现，系统有时会提示开通“联盟账户”。请注意，开通联盟账户提现会扣除 **6%** 的手续费。你可以选择不开通，以节省这部分费用。
