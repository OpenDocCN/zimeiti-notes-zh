## 笔记被推荐的核心逻辑

笔记发布后，平台会根据一系列数据指标决定是否给予更多流量。这个过程可以理解为“流量升级”。例如，一个新账号的笔记初始可能只有约1000次展示（曝光），如何让它升级到2000-5000次甚至更高的展示量？关键在于以下两个核心指标。

### 1. 打开率 (Click-Through Rate, CTR)

打开率是决定笔记能否进入下一级流量池的首要指标。

当用户打开小红书App，首页（推荐页）会展示4-6篇笔记。系统会将你的笔记与同类笔记一起，放入一个约20篇笔记的“初级流量池”中进行测试。

**核心公式**：
`打开率 = (笔记点击次数 / 笔记展示次数) * 100%`

平台要求笔记的打开率必须超过同批测试笔记的**平均11%**，才能晋级到更大的流量池。

*   **举例**：如果你的笔记获得了1000次展示，那么至少需要110次点击（即“小眼睛”数达到110），打开率达到11%，系统才会将其推荐给2000-5000人。如果达不到，流量就会停滞在初始水平。
*   **影响因素**：封面图片的吸引力、标题的文案、前3秒的视频内容。

### 2. 互动率 (Engagement Rate)

互动率是笔记在通过第一级测试后，能否继续升级的关键。

当笔记满足了11%的打开率后，平台会考核用户的互动行为。

**核心公式**：
`互动率 = (点赞、收藏、评论、转发等互动总数 / 笔记点击次数) * 100%`

![](img/def4fe60e358fda1ce1445a73ee94b27_9.png)

平台要求平均互动率需达到**3%以上**。

![](img/def4fe60e358fda1ce1445a73ee94b27_11.png)

![](img/def4fe60e358fda1ce1445a73ee94b27_13.png)

*   **举例**：承接上例，如果有110人点击了笔记，那么至少需要约4个互动（110 * 3% ≈ 3.3，向上取整），互动率才算达标。互动数据越高，获得下一级（如2万-10万展示）推荐的机会越大。

![](img/def4fe60e358fda1ce1445a73ee94b27_15.png)

**流量升级路径总结**：
笔记发布 → 进入初级流量池（约20篇笔记） → **考核打开率（>11%）** → 晋级至中级流量池 → **考核互动率（>3%）** → 晋级至更大流量池。

![](img/def4fe60e358fda1ce1445a73ee94b27_17.png)

对于新账号，通常有机会升级2-3级。老账号或高权重账号，起点更高，有机会冲击更高级别的流量。但通常第5级之后，需要结合付费推广才能获得爆发式增长。

![](img/def4fe60e358fda1ce1445a73ee94b27_19.png)

### 3. 互动行为权重

除了率值，具体的互动行为也有不同的权重评分。平台对用户行为的价值评估大致排序如下：

**分享 > 收藏 > 评论 > 关注 > 点赞**

这意味着，一篇笔记被用户分享或收藏，比仅仅获得点赞，对笔记的推荐助力更大。但请注意，**这一切的前提是打开率和互动率这两个基础指标必须首先达标**。

## 必须避免的运营“坑” 🚫

了解规则后，以下是新手运营小红书时必须规避的常见错误，以避免账号被降权或限流。

![](img/def4fe60e358fda1ce1445a73ee94b27_21.png)

以下是运营中需要特别注意的要点列表：

![](img/def4fe60e358fda1ce1445a73ee94b27_23.png)

![](img/def4fe60e358fda1ce1445a73ee94b27_25.png)

1.  **新号不要立刻发布笔记**：新注册的账号需要“养号”。立即发笔记会消耗平台给予的初始测试流量，可能导致后续流量始终低迷。建议先正常浏览、互动几天再发布内容。
2.  **违规会降权**：任何违规行为（如导流、抄袭、虚假宣传）都会导致笔记推荐和搜索排名大幅下降，通常伴随长达7天的流量限制，恢复难度极大。
3.  **标签忌反复修改**：账号、店铺或笔记的标签一旦确定，不要频繁修改。尤其是在账号已有稳定流量后修改标签，极易引发系统识别混乱，导致限流。
4.  **一机多号风险高**：尽量避免一台手机频繁切换登录多个小红书账号。建议遵循“一机一卡一号”的原则，使用电脑端最多也不要超过3个账号。
5.  **谨慎处理引流内容**：在笔记、评论或私信中直接留联系方式、引导至其他平台等硬广行为，是平台重点打击对象，会直接导致违规。
6.  **谨慎处理留言私信**：对于用户的咨询或留言，应积极合规互动。避免使用敏感词或发生争吵，负面互动也可能影响账号健康度。

![](img/def4fe60e358fda1ce1445a73ee94b27_27.png)
