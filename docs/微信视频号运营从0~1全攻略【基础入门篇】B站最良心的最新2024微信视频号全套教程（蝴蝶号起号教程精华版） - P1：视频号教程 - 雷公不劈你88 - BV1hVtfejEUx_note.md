## 2. 开通分成计划的核心条件与技巧

注册完视频号后，如何快速开通分成计划并领取收益？起号环节非常关键，若操作不当，将无法快速开通。

目前，开通视频号分成计划需要满足以下两个条件：
1.  粉丝数达到100以上。
2.  发布的内容符合平台规范，即优质原创、画风精美。

可能在前段时间，新注册的视频号拍摄10条风景视频就能通过原创认证。但现在，仅拍摄风景类视频很难通过。

![](img/60d6e03eb0f660448fbef781975e5011_5.png)

这里告诉大家一个方法：申请注册新视频号后，直接进行“怼脸拍”（人物正面镜头）。每条视频时长需超过9秒。每天发布5条。实测**第三天**即可开通分成计划。

![](img/60d6e03eb0f660448fbef781975e5011_6.png)

前几天的收益可能较低，但从数据来看，收益在持续运营后会显著提升。

## 4. 收益计算逻辑与提升策略

成功开通分成计划后，我们自然关心收益如何计算以及如何提升。

目前腾讯官方未给出非常明确的收益计算规则。根据实操经验，每万次播放的收益大约在**1到3元**之间。具体收益很大程度上取决于广告的点击情况。有时即使播放量很高，但如果无人点击视频下方或评论区的广告，收益依然不高。

广告分成收入与作者视频质量、播放量、投稿数量、广告行业选择等多方面因素相关。

因此，在制作原创视频时，我们需要想尽一切办法**让视频的评论区活跃起来**，这有助于提升广告曝光和点击。



## 5. 两大推荐赛道详解：人性语录与文案鸡汤

对于现在想入局视频号的新手，不推荐一开始就做“祝福类”内容，因为该领域参与者太多，竞争过于激烈。
