我们使用**剪映**（黑色图标版本）进行剪辑。这是抖音官方的剪辑工具，请放心使用。它提供手机版和电脑版，本节课我们先讲解手机版的操作。

以下是使用手机版剪映剪辑视频的详细步骤：

![](img/cc9020e6354235d4765a598b4f77930f_29.png)

![](img/cc9020e6354235d4765a598b4f77930f_31.png)

![](img/cc9020e6354235d4765a598b4f77930f_33.png)

**第一步：导入素材**
1.  打开剪映App，点击“开始创作”。
2.  从相册中选择需要剪辑的视频片段，点击“添加”。如果视频由多个片段组成，建议先导入第一条进行剪辑，完成后再通过点击“+”号导入下一条，依次处理，避免混乱。

![](img/cc9020e6354235d4765a598b4f77930f_35.png)

![](img/cc9020e6354235d4765a598b4f77930f_37.png)

**第二步：调整画面构图与比例**
导入视频后，首先检查画面构图。
1.  点击视频轨道，向右滑动底部工具栏，找到并点击“编辑”图标（方块形状）。
2.  点击“旋转”功能，将画面调整到正确的方向。
3.  如需裁剪画面比例，可点击“裁剪”功能，选择如1:1、4:3、16:9等合适比例。调整完毕后，点击确认。

![](img/cc9020e6354235d4765a598b4f77930f_39.png)

![](img/cc9020e6354235d4765a598b4f77930f_41.png)

![](img/cc9020e6354235d4765a598b4f77930f_43.png)

**第三步：剪辑视频内容**
调整好画面后，需要修剪掉视频中无效的部分。
1.  播放视频，找到需要裁剪的起始位置。
2.  将时间轴（白色竖线）移动至该位置，点击视频轨道，选择“分割”功能。视频将被分成两段。
3.  选中需要删除的片段，点击“删除”。
    *   **核心操作可总结为：`选中 -> 分割 -> 选中 -> 删除`**。
4.  重复此操作，修剪掉视频开头、中间不必要的停顿以及结尾的冗余部分。

![](img/cc9020e6354235d4765a598b4f77930f_45.png)

![](img/cc9020e6354235d4765a598b4f77930f_47.png)

**第四步：添加背景音乐**
视频内容修剪完毕后，需要添加合适的背景音乐。
1.  点击底部工具栏的“音频”。
2.  如果你已有准备好的配音文件（如从其他软件生成），可以点击“提取音乐”，从视频中提取音频。
3.  如果没有，则点击“添加音乐”。剪映提供了丰富的音乐库，你可以根据分类（如美食、美妆、情感）浏览选择，或直接搜索歌曲。点击歌曲旁边的播放图标可以预览，选定后点击“使用”。
4.  **剪辑音乐**：如果音乐过长，操作与剪辑视频相同。将时间轴移至需要裁剪的位置，选中音乐轨道，点击“分割”，然后删除多余部分。
5.  **调整音量**：选中音乐轨道，点击“音量”，可以调节背景音乐的音量大小，使其不掩盖人声。

![](img/cc9020e6354235d4765a598b4f77930f_49.png)

![](img/cc9020e6354235d4765a598b4f77930f_51.png)

**第五步：添加与编辑字幕**
为视频添加字幕可以提升观看体验，剪映能自动识别语音生成字幕。
1.  点击底部工具栏的“文字”。
2.  点击“识别字幕”，然后选择“仅视频”。（如果选择“全部”，可能会同时识别出背景音乐的歌词）。
3.  系统将自动识别视频中的语音并生成字幕条。
4.  生成后，检查是否有错别字。双击字幕条即可进行修改。
5.  你还可以点击“编辑”，为字幕更换字体、颜色、样式等，使其与视频风格更搭配。

![](img/cc9020e6354235d4765a598b4f77930f_53.png)

![](img/cc9020e6354235d4765a598b4f77930f_55.png)

**第六步：导出高清视频**
所有编辑完成后，在导出前需注意一个关键设置，以保证视频清晰度。
1.  点击右上角的“导出”按钮。
2.  在导出设置中，将“帧率”调整为 **30**，将“分辨率”拉到最高（如1080P或4K）。**公式化记忆：导出设置 = 帧率30 + 最高分辨率**。
3.  设置完成后，点击“导出”。导出过程中请勿锁屏或切换应用程序，耐心等待导出完成。
4.  导出后，可以直接通过剪映分享到抖音。

![](img/cc9020e6354235d4765a598b4f77930f_57.png)

![](img/cc9020e6354235d4765a598b4f77930f_59.png)

接下来，为制作口播视频的同学推荐一个辅助工具，它可以解决记不住台词的问题。

![](img/cc9020e6354235d4765a598b4f77930f_65.png)

![](img/cc9020e6354235d4765a598b4f77930f_67.png)

![](img/cc9020e6354235d4765a598b4f77930f_69.png)

这个软件叫做 **“爱提词”**。它的作用是让你在拍摄时能看着台词念，无需背诵。

![](img/cc9020e6354235d4765a598b4f77930f_71.png)

![](img/cc9020e6354235d4765a598b4f77930f_73.png)

以下是爱提词的使用方法：

![](img/cc9020e6354235d4765a598b4f77930f_75.png)

1.  **添加台词**：打开App，点击右下角“+”号，创建新台词稿。输入或粘贴你的文案内容，并设置一个标题（如“情感语录”），然后保存。
2.  **提词拍摄**：保存后，点击“开始提词”，选择“录制模式”。此时，台词会以悬浮窗形式显示在手机屏幕上，你可以一边看着台词，一边进行录制。
3.  **个性化设置**：
    *   **调整字体**：点击屏幕中央的“设置”图标，可以放大或缩小字体，方便阅读。
    *   **开启美颜**：点击左下角的“美颜”图标，可以调节瘦脸、大眼、磨皮、美白等参数，让拍摄效果更自然美观，不同于一些滤镜过重的相机App。
4.  **灵感参考**：App内还提供“爆款文案”板块，涵盖职场、情感、生活等多个领域，可以作为内容创作的参考。
