# 高客单的小众类目：P1：避开价格战的选品策略 🎯

在本节课中，我们将要学习为什么许多店铺难以盈利，以及如何通过选择高客单价的小众类目来有效避开激烈的价格竞争，从而提升店铺的利润空间。

许多店铺不赚钱，根本原因在于他们始终没有做能赚钱的事情。具体来说，很多人本身并非生产厂商，不具备价格优势，却总在纠结自己的产品为什么没有价格优势。

既然没有价格优势，就不应该执着于做低价产品。也不应该将竞争焦点放在价格上，而应该考虑如何避开价格战。

那么，在拼多多这样的平台上能避开价格战吗？答案是肯定的。许多高客单价的小众类目就可以有效避开价格战，因为这类产品的市场竞争通常不那么激烈。

## 什么是高客单的小众类目？🤔

上一节我们提到了高客单小众类目可以避开竞争，本节中我们来看看具体有哪些产品属于这个范畴。

高客单价的小众类目指的是那些单价较高、受众相对特定、竞争程度较低的细分产品领域。以下是几个典型的例子：

*   **大型五金机械与设备**：例如专业工具或工业设备。
*   **个性化定制礼品**：例如刻字服务或专属设计产品。
*   **高端数码配件**：例如高品质的耳机、键盘或专业摄影配件。
*   **小众手机的配件**：例如特定型号的手机壳、贴膜。
*   **创意家居用品**：例如设计独特、功能新颖的家居装饰或实用物品。

这些产品的共同特点是利润空间相当不错，并且市场竞争没有大众商品那么激烈。
