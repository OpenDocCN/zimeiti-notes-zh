# 新媒体运营：P29：11：小红书平台用户标签及流量入口 📊

![](img/cd261d890443c1ce8b35b67450aa622b_1.png)

在本节课中，我们将学习小红书平台如何为用户打上标签，以及平台主要的流量分发入口。理解这些机制，有助于我们分析笔记如何被系统推荐给精准用户，并指导我们优化内容策略。

上一节我们介绍了系统如何判定内容标签，本节中我们来看看平台如何判定用户标签。这个过程能帮助我们理解笔记是如何被推荐给用户的，并可以从底层逻辑反推，如何让笔记被更多精准用户看到。

## 用户标签判定机制 🔍

系统为用户打标签的主要依据是用户的**浏览行为**。用户注册时选择的兴趣范围参考价值不大，它仅作为平台对内容方向的初步引导和筛选，无法进入系统最深层的用户标签池。

系统根据用户的浏览行为来判定其兴趣分类。以下是判定用户标签的几个核心行为指标：

以下是判定用户标签的几个核心行为指标：

1.  **点击率**
    *   **公式**：`点击率 = (点击某类内容的次数) / (系统推送的该类内容总次数)`
    *   系统在发现页向用户推送多种内容（如美妆、家居、母婴）。用户点击哪类内容更多，系统就能计算出其偏好。例如，用户点击10条内容，其中美妆3次、家居2次、母婴0次，系统会判定该用户对母婴内容不感兴趣，并持续推送更多美妆内容进行试探，从而打上强关联或弱关联的标签。

2.  **停留时长**
    *   即用户在某篇笔记页面的观看时间。即使同属美妆领域，如果用户点开笔记后很快划走，说明内容不吸引他。系统会继续推送美妆下的其他细分内容（如上妆、护肤）来观察。反之，如果用户对一篇平常很少点击的家居笔记停留很久，仔细阅读图片和评论，系统可能判断他有装修或购买家居用品的需求。

3.  **页面浏览量（PV）**
    *   指用户浏览页面的深度。例如，用户不仅看了某篇装修笔记，还点进博主主页，并连续看了第二、第三篇笔记。这个数值（浏览页面数）对账号健康度有积极影响。在粉丝基数较小时，提升用户在你主页的浏览深度，能为账号带来非常健康的评分。

4.  **互动行为**
    *   包括点赞、收藏、评论、关注。这些数据直接关系到笔记能否进入更大的推荐流量池。互动数据高，不仅有利于笔记获得更多推荐，还可能带动账号内其他笔记的流量，并提升粉丝数，是实现接单、变现等目标的基础。

## 平台流量分发入口 🚀

了解用户标签后，我们来看看小红书主要的流量分发入口。作为用户可能经常使用，但未必分析过其区别。

小红书平台主要有四个流量分发入口：

以下是四个主要的流量分发入口：

1.  **发现页**
    *   这是笔记获取流量的主要来源。包含三部分内容：
        *   **关注达人**：你已关注的博主发布的内容。
        *   **兴趣内容匹配**：系统根据你的点击率等用户标签，推荐你可能喜欢的内容。
        *   **发散性内容匹配**：系统也会推荐一些你可能不感兴趣或未刷到过的内容，以增加平台内容的多样性，满足用户潜在的多方位需求。

2.  **关注页**
    *   这里集中展示所有你关注的博主发布的最新笔记。用户通常从这里直接查看强关联博主的动态。

3.  **附近页**
    *   根据手机定位，推送同城或附近的内容。此板块以具有强地域特征的内容为主，如美食、本地旅游、景点等。对于推广线下实体店（如美容院、餐厅）的运营者，需要重点关注此入口，并在后期考虑信息流广告时，注意对附近人群的推送。

4.  **购物页**
    *   这是小红书向电商方向发展的重要入口，直接匹配用户的购物需求。目前，部分用户尚未形成在小红书直接购物的强习惯，更多人在传统电商平台或抖音完成购物。许多用户仍习惯通过搜索笔记来“种草”。该入口是平台实现变现的关键路径。
