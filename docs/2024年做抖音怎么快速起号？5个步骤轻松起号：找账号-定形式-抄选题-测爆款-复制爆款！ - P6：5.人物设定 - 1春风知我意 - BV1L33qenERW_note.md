# 抖音运营教程：P6：人物设定

![](img/c7813b841294fbdd8531009317da0414_0.png)

在本节课中，我们将要学习如何在抖音上策划和建立有效的人物设定（人设）。一个清晰、真实且具有吸引力的人设，是账号获取流量、建立信任并最终实现变现的核心基础。

![](img/c7813b841294fbdd8531009317da0414_2.png)

## 人设策划的核心概念

![](img/c7813b841294fbdd8531009317da0414_4.png)

上一节我们介绍了账号定位与选题，本节中我们来看看如何为账号注入灵魂——即人物设定。

![](img/c7813b841294fbdd8531009317da0414_6.png)

在抖音中建立人设，类似于在游戏中创建角色，需要预先对人物的各项特征进行规划和设定。例如，角色的称呼、职业、技能等。但与游戏角色不同的是，我们在抖音中设定的是“人生角色”，因此需要具备更高的真实性和可信度。

![](img/c7813b841294fbdd8531009317da0414_8.png)

无论是通过短视频还是直播，一个成功的IP（人设）能够有效解决粉丝的信任问题，促进内容的裂变传播，并具备强大的商业变现潜力。因此，我们运营账号、创作内容或策划选题的核心目的之一，就是在获取流量的同时打造IP。

![](img/c7813b841294fbdd8531009317da0414_10.png)

打造IP的核心过程，是在特定的场景或情境中，完成账号的“人格化”。这个过程需要注入情感、塑造故事，并贴上具有辨识度的符号标签。账号人格化，就是我们通常所说的“建立人设”。

![](img/c7813b841294fbdd8531009317da0414_12.png)

## 人设策划的六个维度

![](img/c7813b841294fbdd8531009317da0414_14.png)

对于人设的具体策划，我们可以从以下六个关键维度着手。以下是每个维度的详细说明：

1.  **名字/姓氏/昵称**
    一个IP首先需要一个称呼，这是其传播的名片。必须明确这个IP叫什么，其重要性如同一个品牌的名称或商标。

![](img/c7813b841294fbdd8531009317da0414_16.png)

2.  **职业/职务/职称**
    人格化的角色必然具备社会身份。这是用户认识并联想该IP的入口。例如：老师、教授、博士、律师等。

![](img/c7813b841294fbdd8531009317da0414_18.png)

3.  **称呼/称谓/社会关系**
    一个恰当的称呼能快速拉近与用户的距离，产生情感共鸣。例如：“XX哥”、“XX大爷”、“XX姥姥”。

4.  **形象/第一印象**
    从用户视角塑造IP形象，使其更容易被记住并占据用户心智。例如：大叔形象、萌萌哒、铁憨憨、甜美可爱、光头等。

5.  **性格/爱好/特点**
    一个具有鲜明特点的IP，不仅记忆点强，也显得更真实。过于完美的形象反而容易脱离大众。例如：暴躁、娘炮、爱吃、爱喝酒等。

6.  **穿着打扮/财富地位**
    借助社会普遍认知和刻板印象，可以快速在用户心中为IP定位。例如：有钱代表会赚钱、开豪车代表成功人士、打扮土气可能代表老师傅。

## 从人设到账号搭建

以上六个维度的内容策划完成后，就可以结合你自身的产品或服务优势，将其落实到抖音账号的“五件套”搭建中。**账号五件套**包括：
*   头像
*   昵称
*   背景图
*   简介
*   视频封面

## 提炼人设的“超级符号”

除了对IP进行人格化策划，还需要通过持续的选题和内容创作，为人设提炼出易于记忆的“超级符号”。超级符号可以从以下几个方向提炼：

以下是提炼超级符号的几个具体方向：

*   **画风（视觉）**：独特的场景、道具、服饰搭配。
*   **听觉**：标志性的BGM、特效音、配音声线。
*   **文案**：能够反映优势且朗朗上口的提炼词，例如：“XX简单粗暴”。
*   **表情与情绪**：标志性的笑容、惊恐、无奈等表情包式情绪传递。
*   **语言特色**：方言、口音、错误发音、习惯性网络用语、特定的说话节奏（慢条斯理或抑扬顿挫）。
*   **动作习惯**：标志性的肢体动作，例如：撩头发、扶眼镜、特定手势。

超级符号提炼好后，在进行短视频创作时，就应不断地在视频中重复呈现这些元素，从而强化用户记忆，形成独特的账号标签。

## 课程总结

本节课中，我们一起学习了抖音人设策划的全过程。我们首先明确了人设对于账号成功的重要性，它关乎信任、传播与变现。接着，我们详细拆解了策划人设的六个核心维度：名称、职业、称呼、形象、性格和外在标志。然后，我们探讨了如何将人设落地到账号的五件套设置中。最后，我们介绍了如何通过提炼“超级符号”——包括视觉、听觉、文案、表情、语言和动作习惯——来不断强化人设，使其深入人心。记住，一个成功的人设是真实、独特且贯穿于所有内容之中的。
