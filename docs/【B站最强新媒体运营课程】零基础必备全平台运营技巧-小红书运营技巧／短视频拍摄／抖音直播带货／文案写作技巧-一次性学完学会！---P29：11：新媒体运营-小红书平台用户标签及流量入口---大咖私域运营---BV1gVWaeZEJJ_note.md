# 新媒体运营：P29：11：小红书平台用户标签及流量入口 📊

![](img/ea3d08985f179f940a8cac07d73d5a87_1.png)

在本节课中，我们将要学习小红书平台如何判定用户标签，以及平台主要的流量分发入口。理解这些机制，有助于我们分析笔记如何被系统推荐给精准用户，从而优化运营策略。

上一节我们介绍了系统如何判定内容标签，本节中我们来看看平台如何判定用户标签。这个过程能帮助我们反向推导，如何让笔记被更多精准用户看到。

## 用户标签判定机制

系统为用户打标签的主要依据是用户的**浏览行为**。用户注册时选择的兴趣范围参考价值不大，仅作为初期内容方向的引导，无法进入系统深层的用户标签池。

系统根据以下用户行为来判定其兴趣分类：

以下是判定用户标签的核心行为指标：

1.  **点击率**
    *   **定义**：系统在发现页向用户推送多种内容（如美妆、家居、母婴等），用户点击哪一类内容更多，系统就能计算出其偏好。
    *   **作用**：通过点击概率，系统为用户打上**强关联**或**弱关联**的标签。例如，用户频繁点击美妆内容，系统会判定其为美妆兴趣用户，并持续推送相关笔记进行试探。

2.  **停留时长**
    *   **定义**：用户点开一篇笔记后观看的持续时间。
    *   **作用**：停留时长能进一步细化用户兴趣。例如，用户虽常点美妆笔记，但划走得很快，说明内容不匹配。若某篇家居笔记停留很久，系统可能判定其有装修或购买家居用品的潜在需求。

3.  **页面浏览量（PV）**
    *   **定义**：用户浏览页面的深度，例如不仅看了单篇笔记，还进入了博主主页并浏览了其他多篇笔记。
    *   **作用**：PV值能提升账号的健康度评分。用户在你主页浏览的笔记越多，对账号的权重提升帮助越大。

4.  **互动行为**
    *   **定义**：包括点赞、收藏、评论、关注等用户主动行为。
    *   **作用**：互动数据直接提升笔记进入下一级推荐流量池的几率。高互动笔记能获得更多推荐，同时带动账号内其他笔记的流量，并有助于粉丝增长和后续变现。

## 平台流量分发入口

了解用户标签后，我们来看看小红书主要的流量来自哪里。平台主要通过以下四个入口进行内容分发：

以下是小红书四大核心流量入口：

1.  **发现页**
    *   **内容构成**：
        *   已关注达人发布的内容。
        *   根据用户标签（点击率、停留时长等）匹配的**兴趣内容**。
        *   系统为了内容多样化而推荐的**发散性内容**。
    *   **重要性**：这是笔记和账号获取流量的**最主要来源**。

2.  **关注页**
    *   **内容构成**：纯粹展示用户**已关注博主**发布的笔记。
    *   **特点**：流量较为稳定，但依赖于粉丝基数和粉丝活跃度。

3.  **附近页**
    *   **分发逻辑**：根据手机定位，推送**附近及同城**的相关内容。
    *   **内容类型**：以美食、本地旅游、线下店铺等强地域特征的内容为主。
    *   **适用场景**：非常适合**线下实体店**（如美容院、餐厅）的本地推广。在投放信息流广告时，可重点考虑此入口。

4.  **购物页**
    *   **定位**：小红书向电商平台发展的重要入口，侧重于商品购买和直播带货。
    *   **用户习惯**：目前部分用户尚未形成在此直接购物的强习惯，更多人仍倾向于先搜索相关种草笔记。
