在本节课中，我们将要学习自媒体的核心概念，并了解本系列教程的整体内容框架。我们将从最基础的定义开始，帮助你清晰地理解自媒体是什么，以及后续课程将如何指导你从零开始打造一个能够变现的账号。

首先，第一个板块是自媒体的现状与前景。无论我们从事任何行业，无论是全职还是兼职，都需要清楚该行业的现状和前景。了解清楚这些信息，我们才能判断这个行业是否值得投入，以及它能为我们带来哪些好处。在充分了解之后，我们再开始操作，心里才会更有底。

接下来，第二个板块将针对零基础的同学，讲解如何打造一个高收益的账号。许多对自媒体有所了解，或者曾经尝试过的同学可能都遇到过这样的问题：发布的视频要么是零播放量，要么是在某些平台（如抖音、快手）上无法通过播放量获得收益。这是为什么呢？在这个板块中，我会教大家如何选择平台、领域和渠道，从而真正将收益赚到手。

第三个板块将介绍拥有亿万流量的多种变现玩法。很多同学了解的自媒体，可能仅限于制作视频或文章发布到平台，依靠播放量来结算收益。但实际上，除了这种方式，还有很多其他的变现方法。这些玩法都适合零基础的同学上手操作，我将在这一板块为大家详细讲解。

![](img/83d09afd45d06fbeddebe2baa93c5fea_3.png)

那么，首先让我们来了解一下，到底什么是自媒体呢？

![](img/83d09afd45d06fbeddebe2baa93c5fea_9.png)

我用最通俗的话来解释：自媒体就是，今天我们打开自己的手机，在百度上搜索一个自媒体平台（例如：`头条号`、`百家号`），然后去该平台注册一个属于我们自己的账号。在我们自己的这个账号上，发布我们自己创作的文章或视频。当有人观看我们的内容时，我们就可以依靠播放量来结算收益。这个过程，就叫做自媒体。
