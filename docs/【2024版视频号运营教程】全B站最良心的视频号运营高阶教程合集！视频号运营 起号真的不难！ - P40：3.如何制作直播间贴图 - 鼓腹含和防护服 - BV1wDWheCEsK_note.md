# 视频号运营教程：P40：如何制作直播间贴图 🎨

在本节课中，我们将学习如何为直播间制作一个透明背景的促销贴图。这种贴图常用于展示“好物推荐”等信息，能有效吸引观众注意并促进转化。整个过程无需付费，使用一款手机应用即可轻松完成。

## 工具准备 📱

上一节我们介绍了贴图的作用，本节中我们来看看制作它需要什么工具。制作此类贴图，我们需要使用一款名为“微脉水印相机”的手机应用。该应用前身为“微商水印相机”，在各大应用商店均可下载。

## 制作步骤详解 🛠️

准备好工具后，接下来我们进入核心的制作环节。我们将分步讲解如何创建并导出透明背景的贴图。

以下是详细的操作步骤列表：

1.  **打开应用并选择模板**
    启动“微脉水印相机”应用，无需开通会员。在主界面找到并点击“水印库”或“更多水印”，进入模板选择页。在“水印模板”中，找到“横竖文字”分类，并选择第一个基础模板。

2.  **编辑文字内容**
    点击模板进入编辑界面。在文本框中输入你需要在直播间展示的信息，例如：
    `好物推荐 老国货黄氏霜 九块九一盒 19块九三盒 29块九五盒`
    你可以根据自己直播的商品，模仿其他直播间的文案进行修改。

3.  **调整文字样式**
    编辑界面提供了丰富的样式调整选项：
    *   **文字颜色**：点击调色板图标，可以更换文字颜色，例如彩色、粉色等。
    *   **字体与大小**：点击“字体”或“文字大小”选项，可以调整字体类型和字号。
    *   **行距与间距**：通过“行距”和“字间距”选项微调排版。

![](img/c3c923842ed8dd39513fdb1dc509901d_1.png)

4.  **关键：设置透明背景**
    这是制作直播间贴图最重要的一步。找到“背景色”或“背景”设置选项。
    **核心操作**：必须将背景设置为完全透明。通常该选项显示为一个带斜杠的方框 `□` 或“无背景”、“透明”字样。确保背景不透明度调整为 `0`。
    **错误示例**：如果选择了纯色背景（如白色、黑色），贴图在直播间会带有不透明的底色框，影响效果。

![](img/c3c923842ed8dd39513fdb1dc509901d_3.png)

5.  **生成并保存图片**
    所有设置调整完毕后，点击“生成”或“保存”按钮。在保存选项中，务必选择“保存到手机相册”或“存入相册”，以确保图片本地存储成功。

![](img/c3c923842ed8dd39513fdb1dc509901d_5.png)

## 在直播间添加贴图 📺

![](img/c3c923842ed8dd39513fdb1dc509901d_7.png)

贴图制作完成后，我们来看看如何在直播中实际使用它。

![](img/c3c923842ed8dd39513fdb1dc509901d_9.png)

以下是添加贴图到直播间的流程：

1.  在直播平台（如视频号）发起直播。
2.  在直播间的功能菜单中，找到“画面贴图”、“添加贴纸”或类似功能的按钮。
3.  选择“从手机相册选择”或“本地图片”。
4.  从相册中选中刚刚制作并保存好的透明背景贴图。
5.  贴图会出现在直播画面上，你可以用手指拖动来调整其位置，双指缩放来调整其大小，直到满意为止。

![](img/c3c923842ed8dd39513fdb1dc509901d_11.png)

![](img/c3c923842ed8dd39513fdb1dc509901d_12.png)

如果添加后觉得文字大小或位置不合适，你可以返回“微脉水印相机”应用，重新编辑原文件，调整字号、布局后再次生成并覆盖保存，然后在直播间更新贴图即可。

![](img/c3c923842ed8dd39513fdb1dc509901d_14.png)

本节课中我们一起学习了直播间促销贴图的完整制作流程。我们首先准备了“微脉水印相机”这个工具，然后逐步完成了**选择模板、编辑文案、调整样式、设置透明背景**以及**保存导出**等关键步骤。最后，我们演示了如何将做好的透明贴图添加到直播画面中。记住，制作的核心在于**背景透明化**，这样贴图才能完美融入直播场景，不显突兀。掌握这个技能，你就能轻松为自己的直播间制作各种风格的提示贴图了。
