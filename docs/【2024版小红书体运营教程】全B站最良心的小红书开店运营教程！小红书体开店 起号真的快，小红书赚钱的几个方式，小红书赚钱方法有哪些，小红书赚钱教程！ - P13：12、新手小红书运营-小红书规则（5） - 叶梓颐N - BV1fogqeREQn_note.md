## 被推荐的核心逻辑 🔄

笔记发布后，平台会根据一系列数据指标决定是否给予更多流量。这个过程可以理解为“流量升级”。核心逻辑主要围绕两个关键数据指标：**打开率**和**互动率**。

### 1. 打开率 (Click-Through Rate)

当用户打开小红书App，首页推荐流会展示4-6篇笔记。平台会将新笔记放入一个包含约20篇同类型笔记的“流量池”进行测试。

**打开率**是指看到笔记的用户中，有多少人点击进入了笔记详情页。这个数据是笔记能否进入下一级更大流量池的第一个门槛。

**核心公式**：
```
打开率 = (笔记点击量 / 笔记展示量) * 100%
```
平台要求笔记的**平均打开率需达到11%以上**，才有机会获得翻倍的展示量。

**举例说明**：
*   如果笔记获得了1000次展示，那么至少需要110次点击（即110个“小眼睛”）。
*   达到此标准，平台可能会将展示量提升至2000-5000次。
*   若低于11%，则笔记通常只会获得最初的一波基础流量，后续推荐乏力。

### 2. 互动率 (Engagement Rate)

当笔记通过“打开率”测试进入更大流量池后，**互动率**将决定它能否继续升级。

**互动率**是指点击进入笔记的用户中，产生点赞、收藏、评论、转发等互动行为的比例。

**核心公式**：
```
互动率 = (互动行为总数 / 笔记点击量) * 100%
```
平台要求笔记的**平均互动率需达到3%以上**。

**举例说明**：
*   假设笔记有110次点击，那么至少需要约4次互动（评论、点赞等）。
*   互动率达标，笔记展示量可能从几千升级到上万次。
*   互动数据越好，获得高额展示（如10万+）的机会越大。

![](img/97e2e5596728cfce0d2aad82816c1b17_9.png)

**流量升级路径总结**：笔记发布 → 考核**打开率**（≥11%）→ 进入更大流量池 → 考核**互动率**（≥3%）→ 获得更高层级推荐。

![](img/97e2e5596728cfce0d2aad82816c1b17_11.png)

![](img/97e2e5596728cfce0d2aad82816c1b17_13.png)

对于新账号，通常有机会升级2-3个层级。老账号因基础权重高，起步层级更高，有潜力冲击更高级别的流量。但顶级流量（如百万级曝光）通常需要结合优质内容与商业推广。

![](img/97e2e5596728cfce0d2aad82816c1b17_15.png)

### 3. 互动行为权重

![](img/97e2e5596728cfce0d2aad82816c1b17_17.png)

除了率，互动行为的类型也有不同权重。平台会对以下行为进行综合评分：
*   **点赞**
*   **关注**
*   **收藏**
*   **分享**

![](img/97e2e5596728cfce0d2aad82816c1b17_19.png)

这些行为能进一步提升笔记的评分。但**前提是打开率和互动率这两个基础指标必须首先达标**。

## 必须避免的运营“坑” ⚠️

了解规则后，在运营中应主动避免以下常见错误，以防账号被降权或限流。

以下是运营中需要特别注意的要点列表：

1.  **新号不要立即发笔记**：新注册的账号需要“养号”，即先模拟正常用户浏览、互动几天，再发布内容。立即发笔记会消耗平台给予的初始测试流量，导致后续流量不佳。
2.  **违规必会降权**：任何违规行为（如导流、抄袭、发布违禁内容）都会导致账号被降权，笔记推荐和搜索排名大幅下降，恢复难度极大。
3.  **标签切忌反复修改**：账号、店铺或笔记的标签一旦确定，不要频繁修改。尤其是在账号已有稳定流量后修改核心标签，极易引发限流。
4.  **避免一机多号**：尽量保证一部手机只登录一个小红书账号，使用独立手机卡。同一设备登录多个账号容易被系统判定为营销号。
5.  **谨慎处理引流内容**：在笔记或私信中直接留联系方式、引导至其他平台需非常谨慎，这很容易触发违规。
6.  **谨慎处理留言私信**：避免在私信或评论中出现违规词汇或过度营销行为。
