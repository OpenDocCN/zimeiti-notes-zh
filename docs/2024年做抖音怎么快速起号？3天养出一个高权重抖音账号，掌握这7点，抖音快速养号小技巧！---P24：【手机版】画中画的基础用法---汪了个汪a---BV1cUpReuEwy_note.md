# 视频剪辑教程：P24：【手机版】画中画的基础用法 🎬

![](img/701192c3673171cd7de9680776fdf11b_1.png)

![](img/701192c3673171cd7de9680776fdf11b_3.png)

在本节课中，我们将学习视频剪辑中“画中画”功能的基础概念和多种应用方法。画中画是制作多图层、创意视频效果的核心工具。

![](img/701192c3673171cd7de9680776fdf11b_5.png)

## 什么是画中画？🤔

画中画功能允许你在主视频画面上叠加播放另一个或多个视频，形成“画中有画”的效果。这是实现复杂视频合成的基础。

## 如何添加画中画？➕

以下是添加画中画的基本步骤：

![](img/701192c3673171cd7de9680776fdf11b_7.png)

1.  **导入主视频**：点击“添加”按钮，从相册中选择一个视频导入到主轨道。
2.  **进入画中画功能**：点击底部工具栏中的“画中画”选项。
3.  **新增画中画**：点击“新增画中画”，选择另一个视频导入。这个新视频会以图层形式叠加在主视频之上。

![](img/701192c3673171cd7de9680776fdf11b_9.png)

![](img/701192c3673171cd7de9680776fdf11b_11.png)

**核心操作代码**：`主轨道视频 -> 工具栏 -> 画中画 -> 新增画中画 -> 选择视频`

![](img/701192c3673171cd7de9680776fdf11b_13.png)

## 画中画的核心操作 🔧

![](img/701192c3673171cd7de9680776fdf11b_15.png)

![](img/701192c3673171cd7de9680776fdf11b_17.png)

上一节我们介绍了如何添加画中画，本节中我们来看看如何管理和调整这些图层。

以下是画中画图层管理的几个关键操作：

![](img/701192c3673171cd7de9680776fdf11b_19.png)

*   **调整层级**：选中一个画中画图层，点击“层级”按钮。你可以通过拖动调整图层的上下顺序。层级高的图层会遮挡层级低的图层。**主轨道视频永远处于最底层**。
*   **切换轨道**：
    *   **切画中画**：选中主轨道上的视频片段，点击“切画中画”，该片段会移动到画中画轨道。
    *   **切主轨**：选中画中画轨道上的视频，点击“切主轨”，该片段会移动回主轨道。
*   **添加多个画中画**：你可以重复“新增画中画”的步骤，最多可以添加**6个**画中画图层。

## 画中画的创意应用实例 🎨

![](img/701192c3673171cd7de9680776fdf11b_21.png)

掌握了基础操作后，我们来看看画中画功能在实际创作中的几种应用。

![](img/701192c3673171cd7de9680776fdf11b_23.png)

![](img/701192c3673171cd7de9680776fdf11b_25.png)

### 应用一：添加电影遮幅

![](img/701192c3673171cd7de9680776fdf11b_27.png)

![](img/701192c3673171cd7de9680776fdf11b_29.png)

通过添加黑色画中画，可以轻松为视频制作上下黑边，营造电影感。

![](img/701192c3673171cd7de9680776fdf11b_31.png)

![](img/701192c3673171cd7de9680776fdf11b_33.png)

1.  新增画中画，从素材库导入一个“黑场”视频。
2.  将黑色画中画拖动到视频上方，调整大小以覆盖画面顶部。
3.  复制这个黑色画中画，将其拖动到视频下方。这样，一个带有黑色遮幅的电影感视频就完成了。

![](img/701192c3673171cd7de9680776fdf11b_35.png)

![](img/701192c3673171cd7de9680776fdf11b_37.png)

### 应用二：制作圆形头像框（如游戏直播）

![](img/701192c3673171cd7de9680776fdf11b_39.png)

![](img/701192c3673171cd7de9680776fdf11b_41.png)

画中画结合蒙版功能，可以制作出圆形头像框效果。

![](img/701192c3673171cd7de9680776fdf11b_43.png)

![](img/701192c3673171cd7de9680776fdf11b_45.png)

1.  新增画中画，导入人物视频。
2.  选中该画中画，点击“蒙版”，选择“圆形”蒙版。
3.  调整圆形蒙版的大小和位置，框住人物头部。
4.  （可选）为了添加白色边框，可以再新增一个“白场”视频作为画中画。
5.  同样为白场视频添加圆形蒙版，并调整得比头像蒙版稍大一圈。
6.  通过调整层级，将白色圆形置于人物头像图层之下，即可形成白色边框。

![](img/701192c3673171cd7de9680776fdf11b_47.png)

![](img/701192c3673171cd7de9680776fdf11b_49.png)

### 应用三：为视频添加底部字幕背景

![](img/701192c3673171cd7de9680776fdf11b_51.png)

![](img/701192c3673171cd7de9680776fdf11b_53.png)

利用画中画可以为字幕创建一个纯净的背景条，使文字更清晰。

![](img/701192c3673171cd7de9680776fdf11b_55.png)

![](img/701192c3673171cd7de9680776fdf11b_57.png)

![](img/701192c3673171cd7de9680776fdf11b_59.png)

1.  新增画中画，导入一个“黑场”视频。
2.  将其拖动并放大至覆盖视频底部区域。
3.  返回主工具栏，点击“文本”添加字幕，将文字放置在黑色背景条上。

![](img/701192c3673171cd7de9680776fdf11b_61.png)

![](img/701192c3673171cd7de9680776fdf11b_63.png)

### 应用四：制作三分屏效果

![](img/701192c3673171cd7de9680776fdf11b_65.png)

![](img/701192c3673171cd7de9680776fdf11b_67.png)

使用多个画中画可以分割画面，制作流行的三分屏效果。

![](img/701192c3673171cd7de9680776fdf11b_69.png)

![](img/701192c3673171cd7de9680776fdf11b_70.png)

![](img/701192c3673171cd7de9680776fdf11b_72.png)

1.  将一段视频导入主轨道。
2.  点击“比例”，将画布比例设置为 **9:16**（竖屏）。
3.  将同一段视频通过“新增画中画”添加两次，这样你总共拥有1个主视频和2个画中画视频。
4.  分别调整这三个视频的位置和大小，让它们均匀地排列在竖屏画布中。

![](img/701192c3673171cd7de9680776fdf11b_74.png)

![](img/701192c3673171cd7de9680776fdf11b_76.png)

### 应用五：替换视频边框中的内容

![](img/701192c3673171cd7de9680776fdf11b_78.png)

![](img/701192c3673171cd7de9680776fdf11b_80.png)

你可以利用画中画，将自己的视频内容嵌入到下载的边框素材中。

![](img/701192c3673171cd7de9680776fdf11b_82.png)

![](img/701192c3673171cd7de9680776fdf11b_84.png)

![](img/701192c3673171cd7de9680776fdf11b_86.png)

1.  新增画中画，导入一个带有透明区域的边框素材。
2.  再次新增画中画，导入你自己的视频。
3.  调整自己视频的大小和位置，使其嵌入边框的透明区域内。
4.  （可选）如果边框上有原有文字需要遮盖，可以新增一个“白场”画中画，使用“矩形”蒙版遮盖原文字，然后在上面添加自己的新文本。

![](img/701192c3673171cd7de9680776fdf11b_88.png)

![](img/701192c3673171cd7de9680776fdf11b_89.png)
