# 视频号运营：P36：直播时如何发布草稿箱视频 📤

在本节课中，我们将学习一个实用的直播技巧：如何在直播过程中，将提前准备好的草稿箱视频发布出去，以吸引更多观众进入直播间。

## 概述

![](img/fb9e29c00d461addaf38fc842391e2e3_2.png)

直播开始时，如果发现直播间观众稀少，我们可以通过发布预先制作好的视频来吸引流量。本节教程将详细拆解这一操作的具体步骤和注意事项。

![](img/fb9e29c00d461addaf38fc842391e2e3_4.png)

## 直播中发布草稿箱视频的步骤

上一节我们介绍了直播前的准备工作，本节中我们来看看如何在直播中操作发布视频。

![](img/fb9e29c00d461addaf38fc842391e2e3_6.png)

以下是具体的操作流程：

![](img/fb9e29c00d461addaf38fc842391e2e3_8.png)

1.  **点击右上角菜单**：在直播界面中，点击屏幕右上角的三个点（`...`）菜单按钮。
2.  **最小化直播间**：在弹出的菜单中，选择将直播间最小化。
3.  **进入视频号主页**：此时，点击进入你的视频号个人主页。
4.  **选择草稿箱视频**：在主页中，找到并选择你提前准备好的草稿箱视频。
5.  **点击发表**：点击“发表”按钮发布视频。系统可能会提示“正在直播中，请稍后”，但视频依然可以成功发布出去。

> **核心操作公式**：`点击菜单 -> 最小化直播 -> 进入主页 -> 选择草稿 -> 点击发表`

![](img/fb9e29c00d461addaf38fc842391e2e3_10.png)

**重要提示**：必须通过上述路径，即从视频号主页进入草稿箱发布。如果直接在直播界面点击“发表视频”的入口，是无法成功发布的。

![](img/fb9e29c00d461addaf38fc842391e2e3_12.png)

## 提升直播人气的辅助方法

成功发布视频可以带来初始流量，但为了进一步提升直播间的热度，我们还可以采取以下措施。

以下是几种有效的辅助引流方法：

![](img/fb9e29c00d461addaf38fc842391e2e3_14.png)

*   **分享直播间**：将你的直播间链接分享到微信朋友圈，或直接转发给微信好友。
*   **利用社群导流**：提前准备一些微信群（如业主群、兴趣群、粉丝群等），在直播时将链接分享到这些群里。
*   **使用小号操作**：如果不想用主号在个人社交圈分享，可以注册小号，用小号加入各类社群并进行分享，这是一种更灵活的导流方式。

官方对于私域流量导入有扶持政策。简单来说，如果你能从自己的私域（如微信群、朋友圈）引导一定数量的用户进入直播间，平台会相应地为你从公域推荐更多流量。

## 直播内容与状态的自我检查

如果完成了视频发布和分享引流后，直播间依然留不住人，观众进入后很快离开，那么问题可能出在直播内容本身。

这时，你需要从以下几个方面进行自我检查：

*   **直播话术**：你的讲解是否吸引人？是否清晰流畅？
*   **表现力**：你是否精神饱满、充满热情？死气沉沉的状态会立刻“赶走”观众。
*   **控场能力**：你能否与观众有效互动，并掌控直播节奏？

直播是一项需要不断练习的技能。当遇到问题时，应首先反思并优化自身的表现，而不是单纯归咎于产品或外部因素。

## 总结

本节课中我们一起学习了直播时发布草稿箱视频的关键技巧。核心要点是：通过“最小化直播->进入主页”的路径来发布视频。同时，我们介绍了通过分享直播间到私域社群进行引流的方法，以及当流量不佳时如何从话术、状态等方面进行自我优化。掌握这些方法，能帮助你在直播初期更有效地吸引和留住观众。
