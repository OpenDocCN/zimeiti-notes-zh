# 跨境电商亚马逊运营教程：P38：创建新商品：描述

在本节课中，我们将学习如何为亚马逊商品创建有效的“描述”和“商品特性”（五点描述）。这是优化商品详情页、提升转化率的关键步骤。

上一节我们介绍了商品标题的撰写，本节中我们来看看如何完善描述部分。

## 商品描述与五点描述概述

商品描述是详细展示产品信息的区域。五点描述（Bullet Points）则是提炼产品核心卖点的5条特性列表，在手机端通常只显示前3条。

## 如何撰写五点描述

撰写五点描述前，需要提炼产品的核心特性。我们可以参考产品供应商（如1688）提供的信息，并结合产品详情页中未充分展示的卖点。

以下是撰写五点描述的步骤：

1.  **收集产品特性**：从产品源头（如1688页面）和自身观察中，收集所有卖点。例如：大毛领、防风口袋、不钻绒、含绒量46%-50%、螺纹袖口等。
2.  **筛选并排序**：由于手机端只显示前三条，必须将最重要的三个特性（如含绒量、大毛领、不钻绒）放在最前面。
3.  **优化文案格式**：每条描述的首字母需要大写，避免全部使用大写字母，并确保语句通顺、无重复。

## 如何撰写商品描述

商品描述可以视为对五点描述的补充，用于放置无法纳入五点列表的详细信息，或对产品进行更全面的介绍。这有助于增加页面的搜索关键词密度。

撰写时，应将产品详细介绍清楚。可以借鉴国内电商平台中“副标题”的写法。

*   **填写商品描述**：在描述框中，输入经过优化和翻译的详细文案。
*   **填写五点描述**：在“商品特性”区域，按重要性顺序填写5条特性，并确保前三条最具吸引力。
*   **格式检查**：务必检查所有英文文案的首字母大写情况，避免直接复制粘贴导致格式错误。

**核心操作提醒**：
```text
- 五点描述前三条最重要。
- 所有英文要点首字母大写（例如：`Large fur collar`）。
- 描述内容可补充搜索关键词。
```
