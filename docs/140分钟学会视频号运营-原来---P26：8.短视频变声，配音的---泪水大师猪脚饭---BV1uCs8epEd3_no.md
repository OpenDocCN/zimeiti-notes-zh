# 视频号运营：P26：短视频变声与配音的两种方法 🎬

在本节课中，我们将学习在视频剪辑中实现声音去重的两种核心方法。通过改变视频的原声，可以有效提升作品的原创度，这对于视频混剪和内容创作至关重要。

上一节我们介绍了视频剪辑的基础操作，本节中我们来看看如何具体处理视频中的声音元素。

## 方法一：使用剪映内置变声功能

![](img/b4479d864f5ddfac6382ba0f82e130b3_1.png)

第一种方法是在剪映App内部直接对视频原声进行变声处理。这种方法操作快捷，适合快速修改。

以下是具体操作步骤：

1.  在剪映中导入需要处理的视频素材。
2.  选中视频轨道，在左侧功能栏中找到并点击 **`声音效果`**。
3.  在变声效果列表中，选择喜欢的音效，例如“男生”、“圣诞老人”或“台湾小哥”等。
4.  部分音效支持调节参数，可以进一步微调声音特质。

通过这种方式，视频的人声部分会发生改变，但背景音乐保持不变。

## 方法二：使用配音工具重新生成配音

![](img/b4479d864f5ddfac6382ba0f82e130b3_3.png)

第二种方法是借助外部配音工具，提取视频文案并生成全新的配音。这种方法能彻底改变声音，并有机会一并更换背景音乐。

![](img/b4479d864f5ddfac6382ba0f82e130b3_5.png)

以下是具体操作步骤：

1.  使用配音工具（如“配音神器”）的 **`文案提取`** 功能，从视频中提取出文字内容。
2.  对提取出的文案进行适当修改。修改原则是保持原意，但调整部分用词或语序，例如将“老人”改为“大人”，“怎么办呢”改为“咋办呢”。这能进一步增强去重效果。
3.  将修改后的文案复制到配音工具中，选择一个喜欢的声音角色（如“男生”），然后合成新的配音文件并导出。
4.  回到剪映，导入原视频并 **`关闭原声`**。
5.  通过 **`添加音频` -> `提取音乐`** 功能，导入刚刚生成的新配音文件。
6.  此时，可以进一步添加新的背景音乐，并调整音量平衡。

![](img/b4479d864f5ddfac6382ba0f82e130b3_7.png)

这种方法不仅改变了人声，也完全替换了原有的音频环境。

本节课中我们一起学习了两种为短视频变声和配音的方法：利用剪映内置功能快速变声，以及通过配音工具彻底重制音频。掌握这些技巧，能有效提升视频的原创性，是进行高质量视频混剪的关键步骤。
