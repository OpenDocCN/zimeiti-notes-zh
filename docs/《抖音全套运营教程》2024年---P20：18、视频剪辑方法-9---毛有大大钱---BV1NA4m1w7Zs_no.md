# 《抖音全套运营教程》：P20：18、视频剪辑方法-9 🎬

在本节课中，我们将通过实际操作，学习如何使用剪映App进行视频剪辑。课程将涵盖从导入素材、基础剪辑、添加音乐与字幕，到最终导出的完整流程。同时，我们也会介绍一个辅助口播拍摄的实用工具。

![](img/4141e6eddf742f23c85417ed1e4384ee_1.png)

---

![](img/4141e6eddf742f23c85417ed1e4384ee_3.png)

上一节我们介绍了视频剪辑的整体思路，本节中我们来看看具体的操作步骤。我们将使用剪映App进行手机端剪辑教学。

![](img/4141e6eddf742f23c85417ed1e4384ee_5.png)

首先，请确保下载的是黑色图标的“剪映”App。打开App后，点击“开始创作”按钮。

![](img/4141e6eddf742f23c85417ed1e4384ee_1.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_7.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_9.png)

接着，从相册中选择需要剪辑的视频素材，点击“添加”将其导入到剪辑轨道中。如果你的视频由多个片段组成，建议先剪辑好第一个片段，再点击轨道尾部的“+”号导入并剪辑下一个片段，这样可以保持工作条理清晰。

![](img/4141e6eddf742f23c85417ed1e4384ee_11.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_3.png)

导入视频后，第一步是调整画面构图与比例。选中视频片段，向右滑动工具栏，找到并点击“编辑”按钮（图标通常为方块）。

![](img/4141e6eddf742f23c85417ed1e4384ee_7.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_13.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_15.png)

在编辑菜单中，点击“旋转”将画面调整到合适的方向。如果需要裁剪画面比例，可以点击“裁剪”，然后选择如1:1、4:3、16:9等预设比例或自由调整。

![](img/4141e6eddf742f23c85417ed1e4384ee_9.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_17.png)

调整好画面后，接下来进行内容剪辑。播放视频，找到需要删除的冗余部分（如开场沉默、结尾无关动作）。将时间轴指针（竖线）移动到需要分割的位置，选中视频片段，点击工具栏中的“分割”按钮。

![](img/4141e6eddf742f23c85417ed1e4384ee_19.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_21.png)

**操作逻辑**：`选中视频 -> 点击“分割” -> 选中要删除的片段 -> 点击“删除”`

![](img/4141e6eddf742f23c85417ed1e4384ee_23.png)

重复此操作，即可流畅地剪掉所有不需要的部分。

![](img/4141e6eddf742f23c85417ed1e4384ee_25.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_13.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_27.png)

视频内容剪辑完毕后，需要添加背景音乐。点击底部工具栏的“音频”按钮。

![](img/4141e6eddf742f23c85417ed1e4384ee_29.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_31.png)

如果你有预先准备好的配音文件（例如从其他工具生成），可以点击“提取音乐”，直接从视频中提取音轨。如果没有，则点击“添加音乐”，在曲库中寻找。

![](img/4141e6eddf742f23c85417ed1e4384ee_33.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_17.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_35.png)

音乐库中有丰富的分类，如美食、美妆、萌宠等。你可以点击歌曲旁边的播放图标进行预览，选择合适音乐后点击“使用”。

![](img/4141e6eddf742f23c85417ed1e4384ee_37.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_19.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_39.png)

添加音乐后，如果音乐过长，其剪辑方法与视频相同：`选中音频轨道 -> 分割 -> 删除多余部分`。你还可以选中音频，点击“音量”来调整音乐大小，使其不掩盖人声。

![](img/4141e6eddf742f23c85417ed1e4384ee_41.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_31.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_43.png)

接下来是添加字幕。点击底部工具栏的“文字”按钮，然后选择“识别字幕”。为了更精准，建议选择“仅识别视频原声”，避免将背景音乐歌词也识别出来。

![](img/4141e6eddf742f23c85417ed1e4384ee_45.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_35.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_47.png)

系统会自动生成字幕并匹配时间点。生成后，检查是否有错别字，双击字幕即可进行修改。你还可以调整字幕的字体、颜色和样式，使其与视频风格更搭配。

![](img/4141e6eddf742f23c85417ed1e4384ee_41.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_49.png)

至此，一个包含画面、音乐和字幕的完整视频就基本制作完成了。在导出前，有一个关键设置需要注意。

![](img/4141e6eddf742f23c85417ed1e4384ee_51.png)

点击右上角的“导出”按钮，在导出设置中，将“帧率”调整为**30**，将“分辨率”拉到允许的最高值（如1080P）。这样可以保证视频在抖音平台上的清晰度，避免被过度压缩。

![](img/4141e6eddf742f23c85417ed1e4384ee_53.png)

**导出设置公式**：`帧率 = 30`， `分辨率 = 最高可用值`

![](img/4141e6eddf742f23c85417ed1e4384ee_55.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_57.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_47.png)

设置完成后，点击“导出”。导出过程中请勿锁屏或切换App，耐心等待完成。导出后，可以直接通过剪映分享到抖音。

![](img/4141e6eddf742f23c85417ed1e4384ee_59.png)

以上是视频剪辑的核心流程。接下来，针对拍摄口播视频的同学，推荐一个能提升效率的辅助工具：“爱提词”App。

![](img/4141e6eddf742f23c85417ed1e4384ee_61.png)

这个工具能解决台词记不住的问题。打开App后，点击右下角的“+”号，将你的口播文案粘贴或输入进去。

![](img/4141e6eddf742f23c85417ed1e4384ee_59.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_63.png)

保存文案后，点击“开始提词”并选择“录制模式”。此时，文案会以悬浮窗形式显示在手机屏幕上，你可以一边看词一边录制，无需背诵。

![](img/4141e6eddf742f23c85417ed1e4384ee_65.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_63.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_67.png)

在提词器设置中，可以调整字体大小，方便阅读。该App还内置了拍摄功能，点击左下角的“美颜”可以开启瘦脸、大眼、磨皮等效果，让拍摄效果更自然美观。

![](img/4141e6eddf742f23c85417ed1e4384ee_69.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_67.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_71.png)

此外，“爱提词”的“爆款文案”板块提供了职场、情感、生活等多个领域的文案参考，值得花时间浏览学习。

![](img/4141e6eddf742f23c85417ed1e4384ee_73.png)

![](img/4141e6eddf742f23c85417ed1e4384ee_75.png)
