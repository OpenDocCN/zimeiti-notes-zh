上一节我们完成了身份验证，点击“下一页”后，系统会提示设置收款方式。

![](img/dca1d9723bdb54ac8aadfe2114d0a794_11.png)

### 设置收款方式与VISA卡信息
以下是设置收款方式的具体步骤，核心是填写正确的VISA信用卡信息。

![](img/dca1d9723bdb54ac8aadfe2114d0a794_13.png)

![](img/dca1d9723bdb54ac8aadfe2114d0a794_14.png)

![](img/dca1d9723bdb54ac8aadfe2114d0a794_15.png)

1.  **准备已激活的VISA卡**：你需要一张带有 **VISA** 标志的信用卡。卡片从银行寄出后，**必须通过银行官方APP或根据寄卡附带的指引完成激活**，未激活的卡片无法使用。
2.  **填写卡片信息**：在收款方式设置页面，填写以下信息：
    *   **卡号**：VISA卡正面的卡号。
    *   **有效期**：卡片上的到期日期，格式为“月/年”。例如，卡上印有 `10/23`，则在对应位置填写 `10` 和 `2023`。
    *   **持卡人姓名**：填写你姓名的拼音，需与卡片印刻的姓名一致。
    *   **账单地址**：这是一个关键信息，**不一定是你身份证上的地址**。你需要通过以下两种方式之一确认准确的账单地址：
        *   **致电银行客服**：查询你信用卡的账单邮寄地址。
        *   **登录银行手机APP**：在信用卡账单或账户详情页面查找账单地址。
    *   如果查询到的账单地址与系统默认（身份证地址）不同，请点击“添加不同地址”，并按照填写地址的标准格式（省、市、街道等）重新填写。

![](img/dca1d9723bdb54ac8aadfe2114d0a794_17.png)

![](img/dca1d9723bdb54ac8aadfe2114d0a794_19.png)

> **核心提示**：办理信用卡时，请明确要求办理带有 **VISA** 标志的双币信用卡，并向工作人员出示相关图片以确保无误。

![](img/dca1d9723bdb54ac8aadfe2114d0a794_21.png)

### 设置收款账户
收款方式设置的下半部分，是关联你的第三方收款账户（如Payoneer、PingPong等）。

1.  **银行地址**：此处**必须选择“美国”**，不要选择中国。
2.  **收款人姓名**：填写你姓名的拼音。
3.  **收款账号**：登录你的第三方收款平台，找到“9位数汇款路径代码”（ABA Routing Number）和“银行账号”，将其分别填入对应位置。银行账号通常需要填写两遍以作确认。

> **操作公式**：`收款账户信息 = 银行地址(美国) + 收款人姓名拼音 + 9位汇款路径码 + 银行账号`

### 完成店铺信息问卷
设置好收款信息后，点击下一页，会进入一个店铺信息问卷环节。

以下是各问题的建议选项，请根据箭头指示或自身情况选择：
*   你是否在亚马逊其他站点有店铺？ → **选择“是”或“否”均可，按指引操作。**
*   你是否拥有自己的品牌？ → **若无，则选择“否”。**
*   你的商品是否由其他公司生产？ → **通常选择“否”。**
*   预计每月销售额？ → **可选择“小于 $100” 或 “$100 - $500”。**

### 选择卖家类型与身份验证
完成问卷后，进入卖家类型选择。

![](img/dca1d9723bdb54ac8aadfe2114d0a794_29.png)

1.  **选择卖家类型**：因为是个人注册，此处选择“**个人卖家**”。
2.  **填写身份信息**：接下来需要填写法人的身份证信息。
    *   **身份证号**：填写你的身份证号码。
    *   **有效期**：注意格式为 **月-日-年**。例如身份证到期日为2027年2月9日，则填写 `2-9-2027`。
    *   **姓名**：“姓”填写拼音姓氏，“名”填写拼音名字。
    *   **出生日期**：同样按照月-日-年的格式填写。
    *   **地址**：填写你身份证上的居住地址。
3.  **上传证明文件**：根据页面提示，上传**身份证人像面照片**、**身份证国徽面照片**。此外，还需上传一份地址证明，如：
    *   **信用卡账单**：选择“银行卡”类别，上传近期的信用卡账单截图或照片。
    *   或其它可证明地址的文件。

![](img/dca1d9723bdb54ac8aadfe2114d0a794_31.png)

![](img/dca1d9723bdb54ac8aadfe2114d0a794_33.png)

![](img/dca1d9723bdb54ac8aadfe2114d0a794_35.png)

### 提交并等待审核
所有信息填写并文件上传完毕后，提交注册申请。页面会提示“注册正在进行中”，审核结果通常会在**两个工作日内**通过你注册时填写的邮箱通知你。

## 总结
本节课我们一起学习了亚马逊个人店铺注册流程的后半部分。核心操作包括：正确填写并激活VISA信用卡、核实并填写准确的账单地址、关联第三方收款账户、完成店铺问卷、选择个人卖家类型以及提交身份证件进行最终验证。请确保每一步的信息都准确无误，这是成功开通店铺的基础。完成提交后，请耐心等待亚马逊的审核邮件。
