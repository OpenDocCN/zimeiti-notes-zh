# 跨境电商TRO应对指南：1：预警、介入与决策

在本节课中，我们将学习如何应对美国法院的临时限制令。课程将涵盖提前预警的重要性、收到TRO后的决策思路，并通过实际案例帮助理解。

---

上一段我们提到了TRO预警的价值，本节中我们来看看如何具体获取和利用预警信息。

对于跨境卖家而言，关注TRO预警信息至关重要。美国约90%的专利侵权诉讼在TRO阶段便告结束，后续诉讼成本极高。因此，提前预警能有效规避风险。

目前，市场上有许多服务商提供TRO提前预警服务。我们自身也在知识星球中每周更新涉及TRO的产品和专利列表。

卖家可以对照列表，评估自身产品是否有可能面临被告风险。例如，卖家“跨境老张”曾发现其产品出现在预警列表中，这提示他可能面临诉讼。

“跨境老张”随即做出了一个非常正确的决定：在尚未收到法院TRO通知时，便提前介入处理。最终，他成功避免了损失，安然无恙。

所以，请务必关注相关预警信息，并敏感地判断其与自身业务的关联。此前，我曾预警过一个鼠标专利。一位销售鼠标的卖家得知后，认为该专利与自己的产品完全无关。

然而，两个月后，这位卖家联系我，确认他确实因那项专利被发起了TRO。

因此，及早关注信息并提前采取行动是关键。

---

上一节我们讨论了预警与提前介入，本节中我们来看看收到TRO后该如何决策。

如果收到了TRO，应选择应诉还是和解？这主要取决于涉案金额。

如果你的涉案金额很高，可以考虑聘请律师积极应诉。如果你的涉案金额很低，那么选择和解可能是更经济的做法。

此外，如果你希望未来继续销售该产品，通常也需要选择应诉。因为通过和解方案，对方很可能不会允许你继续销售。

---

本节课中我们一起学习了应对TRO的核心思路。首先，通过预警服务提前发现风险并介入，是成本最低的防御策略。其次，在收到TRO后，决策应基于涉案金额和是否希望继续销售产品：金额高或想继续销售，则考虑应诉；金额低则可选择和解。保持对行业动态的敏感并及时行动，是保护店铺安全的关键。
