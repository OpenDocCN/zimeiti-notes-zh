# 视频号运营：P31：如何寻找书单类对标账号 📚

在本节课中，我们将学习在短视频平台（如抖音）上，如何高效地寻找与自身定位相符的对标账号。我们将以“书单类”账号为例，介绍两种简单实用的方法，帮助你快速建立自己的参考账号库。

上一节我们介绍了寻找对标账号的重要性，本节中我们来看看具体如何操作。

## 方法一：利用“关注”推荐功能

此方法的核心是利用平台的推荐算法。当你关注一个账号时，平台会基于标签相似度，为你推荐其他可能感兴趣的账号。

![](img/afdad3ea67936309787e8e5cb6227731_1.png)

具体操作步骤如下：

1.  在平台搜索框内，输入你计划运营领域的关键词（例如“书单”）。
2.  从搜索结果中，找到一个视频类型符合你预期的账号。
3.  点击进入该账号主页，浏览其视频内容，确认是否为你想要模仿的类型。
4.  如果符合，点击“关注”按钮。
5.  点击关注后，页面下方通常会弹出一个“你可能还喜欢”或“相似账号推荐”的列表。
6.  列表中的账号大多与初始账号类型相似。你可以逐个点开查看，若内容合适，则继续关注。

![](img/afdad3ea67936309787e8e5cb6227731_3.png)

![](img/afdad3ea67936309787e8e5cb6227731_4.png)

通过这种方式，你可以在两三分钟内关注数十个同类账号。此后，平台的信息流会持续为你推荐更多相似账号。

## 方法二：通过商品“达人秀”寻找

![](img/afdad3ea67936309787e8e5cb6227731_6.png)

此方法通过具体商品链接，寻找推广该商品的大量账号，效率更高。

![](img/afdad3ea67936309787e8e5cb6227731_8.png)

具体操作步骤如下：

1.  确定一本你想推广或参考的书籍（例如《墨菲定律漫画版》）。
2.  在平台搜索框内，直接搜索该书籍名称。
3.  在搜索结果页，切换到“商品”分类，并按“销量”排序。
4.  找到销量较高的商品链接，点击进入商品详情页。
5.  在商品详情页中，寻找“达人推荐”或“买家秀”板块。
6.  该板块会展示大量推广过此商品的达人视频。点击任意视频，即可进入对应账号主页。

以下是操作流程的简化表示：
```
搜索具体商品 -> 进入高销量商品页 -> 查找“达人推荐”板块 -> 获取大量对标账号
```

这个板块汇聚了所有推广该商品的账号，可能多达数百个。这些账号的内容形式和目标受众高度相似，是绝佳的模仿和学习对象。

![](img/afdad3ea67936309787e8e5cb6227731_10.png)

---

本节课中我们一起学习了两种寻找对标账号的方法：一是通过关注账号触发平台推荐，二是通过商品页的达人推荐板块批量获取。掌握这两种方法，你可以快速建立起自己领域的账号参考体系，为后续的内容创作打下坚实基础。
