# 新媒体运营零基础入门教程：P16：02.3：工具篇-AI生成PPT 🛠️

## 概述
在本节课中，我们将学习第二款强大的AI生成PPT工具。这款工具在生成速度、稳定性和模板丰富度方面具有优势，我们将通过实际操作演示其两种核心使用方法。

---

上一节我们介绍了一款AI生成PPT工具，本节中我们来看看另一款功能互补的工具。

这款工具主要有两个优势。第一是生成速度快且稳定。第二是模板库非常庞大，提供了丰富的样式选择。结合上一节分享的工具，两者可以相辅相成，为PPT制作提供更多便利。

要使用这款工具，需要将PPT或大纲节点的内容准备得比较详细和丰富，这样工具才能分析并生成出质量较高的模板。

## 两种生成方式
该工具提供了两种生成PPT的方式。

以下是两种方式的具体说明：
1.  **AI智能生成**：直接输入主题，由AI自动构思大纲和内容。
2.  **导入本地大纲**：导入已准备好的详细文档大纲，由AI基于内容生成PPT。

![](img/357bebd54f72cf9b3cff59ae39fa7ed1_1.png)

我个人更常使用“导入本地大纲”的方式。

## 实践：导入本地大纲生成
首先，我们需要准备一份详细的大纲。可以使用ChatGPT等AI工具辅助生成。

例如，向ChatGPT 4.0提出请求：
```
请以“装修行业在小红书平台的运营推广计划”为主题，撰写一份详细的大纲。请完善每一个小节点的具体内容信息，并保持清晰的结构和格式排版。
```

AI生成的大纲可能在某些细节上不够详尽，例如“教育内容制作”部分可能缺少拓展。我们可以先基于这个大纲进行后续操作。

准备好大纲文档后，在工具中选择“导入本地大纲”功能，并上传文件。

工具会快速梳理文档的大纲结构。接着，我们需要从海量的模板库中，挑选一个适合装修行业的颜色与风格。

选择模板后，点击“生成PPT”。生成完成后，我们进入编辑界面。

在编辑界面中，如果发现某些页面内容缺失（例如“内容趋势”下方无内容），我们可以通过“大纲编辑”功能进行补充。在大纲编辑器中，可以详细填写章节背景信息。

如果对某一页的排版不满意，可以单击该页面，工具会提供多种备选页面布局供一键更换。

此外，整个PPT的模板也可以随时更换。在编辑器中找到“模板替换”功能，选择新模板并应用，所有内容会自动适配新样式，响应速度很快。

这个工具提供了完整的在线编辑功能，包括文字设置、形状设置和背景设置，可以满足基础的PPT编辑需求。

## 实践：AI智能生成
接下来，我们尝试第二种方式：AI智能生成。

在工具中选择“AI智能生成”，在输入框中直接填写主题，例如：“装修行业在小红书平台的运营推广计划”。

点击生成后，AI会先自动构思一份详细的大纲。大纲生成后，同样需要选择一个合适的模板。

选择模板并点击“生成PPT”，AI不仅会生成幻灯片框架，还会自动为每一页填充详细的文案内容。

生成完毕后，进入编辑界面。在此，我们可以像之前一样，调整字体、样式、更换模板，或者通过“大纲编辑”功能微调具体内容。

这种方式适合快速生成一个内容完整的PPT初稿。
