# 抖音运营新手起号：P21：扶持加权与降权限流 📊

在本节课中，我们将学习抖音平台对视频内容的两种核心处理机制：扶持加权与降权限流。理解这些规则，是避免踩坑、让作品获得更多推荐的基础。

## 概述
本节课将详细解析抖音平台鼓励的视频格式（扶持加权）和禁止的内容行为（降权限流）。掌握这些要点，能帮助你创作出更符合平台规范、更容易获得流量的作品。

## 扶持加权的内容标准 ✅
上一节我们介绍了账号权重的概念，本节中我们来看看哪些内容特征会受到平台的扶持与加权。平台倾向于推荐符合其原生体验的视频。

以下是会受到扶持加权的视频特征列表：
*   **视频格式**：竖版（9:16）的移动端原生视频。这是抖音的标准格式。
*   **画面质量**：画面平稳，作品清晰。
*   **音频质量**：无过多杂音或噪音。

## 会被降权限流的内容 🚫
了解平台鼓励什么之后，我们更需要知道哪些是红线。发布以下内容，轻则导致视频推荐量降低，重则导致视频无法通过审核或账号受到处罚。

以下是会被降权限流的内容类型列表：
1.  **违规信息**：内容涉及吸烟、暴力、炫富（金钱）等不良导向。
2.  **外部引流**：出现微信、微博、淘宝等站外联系方式。长时间出现商品Logo、手机号、二维码（二维码出现时间最好不超过3秒，但审核严格时，出现即可能被驳回）。
3.  **硬性广告**：进行明显的营销广告、推销或引流卖货行为。

![](img/7d9e9322ca13d9257fa58a3f9afe42d2_3.png)
![](img/7d9e9322ca13d9257fa58a3f9afe42d2_5.png)

关于好物推荐类内容，这里有一个需要特别注意的案例。很多同学在做分享好物、挂小黄车带货的账号，但经常遇到小黄车被下架的问题。

以下是两个导致失败的常见操作及其正确做法：
*   **错误操作1**：在视频画面中直接标注价格（如“九块九”）。这属于直接的营销行为。
*   **正确做法1**：应避免在视频中直接出现标价信息。
*   **错误操作2**：单纯展示产品，进行无口播的拆箱或使用演示，全程只有背景音乐。
*   **正确做法2**：必须加上口播进行简单介绍。例如：“我儿子玩这个玩具，四个小时都没停下来。”通过情景化描述来软性推荐产品。

核心要点是：**不能单纯地、直接地展示产品，需要结合情景或口播将其转化为内容分享**。直接的口播广告或展示联系方式在抖音是行不通的。

## 总结
本节课我们一起学习了抖音平台的流量调控规则。关键在于创作**竖版高清的原生视频**以获得扶持，同时严格**避免违规信息、硬性广告和无情景的单纯产品展示**以防止被降权。理解并应用这些规则，你的内容运营之路会更加顺畅。
