# 抖音运营新手起号：P27：扶持加权与降权限流规则详解 🎬

在本节课中，我们将要学习抖音平台对于视频内容的扶持加权与降权限流规则。了解哪些内容能获得平台推荐，哪些内容会触发惩罚，是做好账号运营的基础。

上一节我们介绍了账号权重的分类，本节中我们来看看具体影响权重的视频内容规范。

## 获得扶持加权的内容特征 ✅

平台会优先推荐符合以下技术标准和内容质量要求的视频。

![](img/5dd491e5636b8223628bd27d06a6055d_3.png)

以下是获得扶持加权的核心要求列表：
*   **视频格式**：竖版移动端原生视频，即 **9:16** 的格式。
*   **画面质量**：画面平稳，作品清晰。
*   **声音质量**：无过多杂音或噪音。

这些要求较为基础，平台资料页有更详细的说明。如果对此有特别需求，后续整理的资料页会包含这部分内容，可供课后自行查阅。

## 导致降权限流的内容禁区 ❌

了解完加分项，我们再来看看哪些内容是绝对的红线，会导致账号被降权限流。

以下是明确会导致降权限流的内容类型列表：
1.  **违规信息**：视频内容涉及吸烟、暴力、炫富（金钱）等信息。
2.  **外部引流**：出现微信、微博、淘宝等外部平台联系方式。
3.  **商业标识**：长时间出现商品Logo、手机号。二维码的出现时间**最长不要超过3秒**，但在审核严格时，即使出现1秒也可能被驳回。
4.  **硬性广告**：进行明显的营销广告、推销或引流卖货行为。

## 好物推荐类内容的注意事项 🛍️

对于想做**好物推荐**或**带货**的同学，需要特别注意一个常见误区：**无口播的单纯产品展示极易被判定为广告而限流**。

以下是两个具体的反面案例与分析：
*   **案例一（直接标价）**：视频画面中直接出现“**9.9元**”等价格信息。这属于直接的销售行为，挂载的小黄车商品通常无法通过审核。
*   **案例二（无声展示）**：视频仅展示玩具玩法或产品功能，全程无任何口播介绍，仅使用背景音乐。这种“哑剧”式的展示同样危险。

正确的做法是采用**“剧情类软广告”**或加入**口播介绍**。例如，在展示产品时，可以口播说：“我儿子玩这个玩具，专注了四个小时都没停下来。”通过分享使用体验和感受来软性介绍产品，这样既能传递信息，又更符合平台的内容推荐逻辑。

请务必注意，不要不说话直接拆箱或展示产品。

本节课中我们一起学习了抖音平台关于内容扶持与限流的核心规则。关键点在于：制作**9:16格式的优质原生视频**可以获得加权；而涉及**违规信息、外部引流和硬性广告**的内容则会受到惩罚。特别是对于带货账号，务必通过**加入口播或剧情**的方式进行软性推荐，避免单纯的商品展示。这些规则页将被整理成资料供大家复习。接下来，我们将进入更正式的运营技巧讲解。
