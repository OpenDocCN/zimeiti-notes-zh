# 微信视频号教程：P33：18.活动介绍 🎬

在本节课中，我们将详细介绍一个由微信视频号官方发起的“商家联盟”活动。该活动旨在为视频带货提供强力支持，我们将逐一解析活动的核心内容、提供的资源以及参与步骤。

## 活动背景与目标

为促进2024年中国电商行业的新发展，并为视频化带货奠定技术基础，微信视频号特别推出了“商家联盟”活动。本次活动由短视频教研部负责人张良老师全程对接，秉持公平、公正、公开的理念展开。

![](img/403c98d6f9a6e3a9b229c960552bfa9d_1.png)

## 活动负责人与独家资源

张老师作为本次活动的全国总负责人，拥有对接活动的独家权限。这意味着，只有通过张老师这里才能成功对接本次活动，这体现了其独特的实力与背景。

![](img/403c98d6f9a6e3a9b229c960552bfa9d_3.png)

## 活动核心扶持内容

![](img/403c98d6f9a6e3a9b229c960552bfa9d_5.png)

以下是本次活动为参与者提供的核心资源与扶持政策。

### 1. 官方提供视频素材
许多同学担心没有高质量的视频素材。本次活动力度空前，官方已为大家准备好了现成的素材。每位参与活动的账号，官方将直接提供 **30条** 高质量视频。参与者每天只需发布一条即可。

![](img/403c98d6f9a6e3a9b229c960552bfa9d_7.png)

![](img/403c98d6f9a6e3a9b229c960552bfa9d_1.png)

### 2. 官方素材库展示
官方准备的素材质量很高。素材库中包含了各类产品的拍摄视频，例如中秋节礼品、餐巾纸、纯牛奶、火锅底料、洗漱手套、不锈钢杯等，品类丰富。

### 3. 巨额流量扶持
针对“发了视频卖不出去”的担忧，活动为每个参与账号提供 **90万** 的流量扶持。这能极大提升视频的曝光率和成交机会。

### 4. 保底收益激励
本次活动提供了强有力的保底收益激励。参与者每发布一条符合要求的带货视频，即可获得 **300元** 的奖励，并且结算方式是“今天发布，明天到账”。

如果完成全部30条视频的发布，总计可获得：
```
总收益 = 30条 × 300元/条 = 9000元
```

## 活动发起方与时效性

本次活动的发起方是 **微信视频号**，张良老师是官方指定的全国总负责人。活动具有明确的时效性，并非长期开放。活动于1月13日正式开始，并计划于8月10日结束。
