# 微信视频号教程：P20：5.如何制作引导下单的视频 🛒

在本节课中，我们将学习如何在微信视频号中制作引导用户下单的视频。引导视频是提升转化率的关键，它能清晰地向观众展示如何找到并购买你推荐的商品。我们将介绍三种主要方法：简介引导、评论区引导以及制作专门的置顶引导视频。

![](img/51ff214ae130ce62ee267efd5cd277fc_1.png)

## 简介与评论区引导

![](img/51ff214ae130ce62ee267efd5cd277fc_2.png)

![](img/51ff214ae130ce62ee267efd5cd277fc_4.png)

上一节我们概述了引导下单的重要性，本节我们先来看两种最直接的引导方式：个人简介和视频评论区。

在视频号中，引导用户下单的信息可以直接留在个人简介里。用户点开你的主页后就能看到。具体操作是，将引导语复制到个人简介的编辑框中。

![](img/51ff214ae130ce62ee267efd5cd277fc_6.png)

**示例引导语**：
```
欢迎来到我的视频号！所有推荐好物均可在主页商品橱窗中找到，搜索关键词即可购买。
```

另一种方式是在视频的评论区进行引导。你可以在发布视频后，在自己的视频评论区发布引导信息。模板可以参考常见的样式。

![](img/51ff214ae130ce62ee267efd5cd277fc_8.png)

以下是操作步骤：
1.  发布视频后，找到该视频的评论区。
2.  点击评论输入框，输入引导文案。
3.  如果对评论不满意，可以将其删除并重新编辑。

![](img/51ff214ae130ce62ee267efd5cd277fc_10.png)

## 制作置顶引导视频

![](img/51ff214ae130ce62ee267efd5cd277fc_12.png)

![](img/51ff214ae130ce62ee267efd5cd277fc_14.png)

除了简介和评论，制作一个置顶的引导视频是更专业和有效的方法。这种视频会固定显示在你主页的顶部，专门指导用户如何购买。

首先，你需要准备素材。打开任意一个你自己的视频，将其转发给你的另一个微信账号。接着，从接收消息的账号点开这个视频，并开始手机录屏。这样录制下来的是观众视角的界面，更具指导性。

![](img/51ff214ae130ce62ee267efd5cd277fc_16.png)

录屏完成后，我们将使用剪映App进行后期加工，添加指引箭头、文字说明等元素。

## 使用剪映添加指引元素

![](img/51ff214ae130ce62ee267efd5cd277fc_18.png)

![](img/51ff214ae130ce62ee267efd5cd277fc_20.png)

现在，我们进入具体的视频制作环节。将录制好的素材导入剪映后，通过添加贴纸和文本来制作清晰的购买指引。

打开剪映，导入刚才录制的视频素材。我们将通过以下步骤添加指引：

**第一步：添加箭头指引**
1.  点击底部工具栏的 **“贴纸”** 按钮。
2.  在搜索框中输入关键词 **“箭头”**。
3.  从结果中选择一个合适的箭头贴纸，添加到视频轨道上。
4.  调整箭头的位置、大小和方向，使其指向你的头像或相关按钮。

![](img/51ff214ae130ce62ee267efd5cd277fc_22.png)

![](img/51ff214ae130ce62ee267efd5cd277fc_24.png)

**第二步：添加文字说明**
1.  点击底部工具栏的 **“文字”** 按钮。
2.  选择 **“新建文本”** 或 **“文字模板”**。
3.  输入引导文字，例如 **“点击我头像”**。
4.  调整文字的样式、颜色和位置，使其与箭头对齐，清晰可读。

![](img/51ff214ae130ce62ee267efd5cd277fc_26.png)

**第三步：标注核心操作区域**
1.  再次点击 **“贴纸”** 按钮。
2.  搜索 **“圈”** 或类似形状的贴纸。
3.  添加一个圆形或框形贴纸，将其调整到合适大小，圈出“商品橱窗”或“搜索栏”等关键按钮。
4.  如果贴纸持续时间过长，可以在轨道上拖动其两端，调整其显示时长。

![](img/51ff214ae130ce62ee267efd5cd277fc_28.png)

![](img/51ff214ae130ce62ee267efd5cd277fc_30.png)

**第四步：添加最终操作提示**
1.  在圈出按钮的位置附近，再次添加文字。
2.  输入如 **“点击商品橱窗”**、**“搜索关键词下单”** 等最终操作提示。
3.  确保所有指引元素在时间轴上对齐，逻辑连贯。

![](img/51ff214ae130ce62ee267efd5cd277fc_31.png)

![](img/51ff214ae130ce62ee267efd5cd277fc_33.png)

制作完成后，导出视频，并将其发布到你的视频号，并设置为置顶视频。

## 课程总结

![](img/51ff214ae130ce62ee267efd5cd277fc_35.png)

本节课我们一起学习了在微信视频号中引导用户下单的三种方法。我们介绍了如何在个人简介和评论区进行简单直接的文字引导。接着，我们重点讲解了如何制作一个专业的置顶引导视频，包括录制观众视角的素材，以及使用剪映App添加箭头、圆圈和文字说明等指引元素。掌握这些方法，你可以更有效地将视频流量转化为实际购买，提升账号的变现能力。
