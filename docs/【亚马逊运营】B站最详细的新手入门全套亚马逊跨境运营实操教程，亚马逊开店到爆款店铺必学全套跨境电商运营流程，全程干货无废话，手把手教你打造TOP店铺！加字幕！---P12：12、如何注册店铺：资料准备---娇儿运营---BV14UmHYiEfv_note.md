# 亚马逊运营：P12：如何注册店铺：资料准备 📝

![](img/95840f23e0b37cc783bdbef30718a808_0.png)

在本节课中，我们将详细学习注册亚马逊店铺前需要准备的所有核心资料。了解并备齐这些资料是成功开店的第一步。

上一节我们介绍了如何选择亚马逊站点，本节中我们来看看注册店铺具体需要准备哪些材料。

## 资料清单总览

注册亚马逊店铺需要准备六项核心资料。以下是每一项资料的详细说明与注意事项。

### 1. 营业执照彩色扫描件

第一份需要准备的资料是**营业执照的彩色扫描件**（即清晰的照片）。

*   **执照类型**：可以是企业性质的执照，也可以是个体工商户性质的执照。
*   **办理途径**：可在当地工商局免费办理。若没有执照，可借用家人或朋友的，或寻找代办机构协助办理（费用通常在几百至一千元人民币左右）。
*   **重要要求**：
    *   **注册资本与年限**：无硬性要求，无论是1元还是1000万，新办或老执照均可。
    *   **经营范围**：需带有销售性质（如零售、批发等）。但**切忌**在经营范围中添加“进出口贸易”字样，否则需额外办理复杂的进出口手续，属于画蛇添足。
    *   **与店铺产品的关联**：执照的经营范围与您计划在亚马逊上销售的产品**无需一致**。例如，用超市的执照注册店铺后，完全可以销售服装。

### 2. 法人身份证正反面照片

第二份资料是**法人身份证的正反面清晰照片**。

这里有一个核心原则：**营业执照与对应的法人身份证必须是同一个人**。而后续的其他资料则可以与这两份资料的所有人不同。

### 3. 付款信用卡

第三份资料是用于支付亚马逊平台月租的**付款信用卡**。

*   **唯一作用**：支付每月39.99美元的店铺月租。
*   **卡片类型**：必须是**全币种信用卡**，卡面上需有 **Visa** 或 **Mastercard** 标识。国内各大银行（如工商、建设、招商、中信等）均可办理。
*   **备选方案**：如果没有信用卡，可以借用他人的，或花费约100元购买一张虚拟信用卡，每月按时存入足额月租即可。实际上，在店铺产生营业额后，亚马逊会优先从店铺余额中扣款，因此信用卡的作用相对单一。

### 4. 海外收款账户

第四份资料是**海外收款账户**，其功能类似于“国际版的支付宝”。

![](img/95840f23e0b37cc783bdbef30718a808_2.png)

![](img/95840f23e0b37cc783bdbef30718a808_3.png)

*   **核心作用**：用于接收并提现您在亚马逊店铺的销售所得。
*   **获取方式**：可通过第三方支付服务商（如Payoneer、PingPong、万里汇等）在线免费注册。注册过程简单，只需绑定您的身份证和国内银行卡即可。
*   **简单理解**：就像淘宝卖家通过支付宝提现，亚马逊卖家则通过这个海外收款账户将美金货款提现至个人银行账户。

![](img/95840f23e0b37cc783bdbef30718a808_4.png)

### 5. 联系电话

第五份资料是一个有效的**国内手机号码**，用于接收验证码和官方通知。

### 6. 电子邮箱

第六份资料是一个**电子邮箱地址**，用于注册和接收亚马逊官方邮件。

*   **邮箱类型**：163、126、QQ邮箱等均可使用。
*   **个人推荐**：建议使用**163邮箱**，在稳定性和兼容性上表现较好。

本节课中我们一起学习了注册亚马逊店铺必须准备的六项资料：营业执照、法人身份证、付款信用卡、海外收款账户、手机号和邮箱。确保这些资料真实、有效且符合要求，是顺利完成店铺注册流程的坚实基础。下一节，我们将进入具体的店铺注册操作环节。
