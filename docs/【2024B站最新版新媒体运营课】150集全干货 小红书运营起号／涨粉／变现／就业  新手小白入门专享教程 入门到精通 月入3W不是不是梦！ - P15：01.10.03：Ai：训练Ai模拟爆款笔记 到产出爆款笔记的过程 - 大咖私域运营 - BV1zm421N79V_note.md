# 新媒体运营AI实战课：P15：01.10.03：AI训练与爆款笔记生成全流程 🚀

在本节课中，我们将学习如何训练AI（以ChatGPT为例）分析爆款笔记，并最终生成具备爆款潜质的新笔记。整个过程分为两步：**喂食资料让AI学习平台风格**，以及**引导AI分析并模仿爆款结构**。

## 核心步骤概述

![](img/b418a7224b18aead6e85afb92bc7f7d7_11.png)

![](img/b418a7224b18aead6e85afb92bc7f7d7_13.png)

![](img/b418a7224b18aead6e85afb92bc7f7d7_15.png)

我们将引导AI完成以下任务：
1.  **分析爆款因素**：找出内容为何能成为爆款的关键点。
2.  **解构文案框架**：分析笔记的开头、中间、结尾分别如何吸引和留住用户。
3.  **识别关键词策略**：检查笔记是否植入了特定关键词及其频率。
4.  **整合学习与生成**：让AI基于分析结果，生成一份融合了爆款元素的新笔记大纲和文案。

## 第一步：准备与分析爆款笔记

![](img/b418a7224b18aead6e85afb92bc7f7d7_23.png)

![](img/b418a7224b18aead6e85afb92bc7f7d7_25.png)

![](img/b418a7224b18aead6e85afb92bc7f7d7_27.png)

首先，我们需要找到目标领域的爆款笔记作为分析样本。这里以“考研”领域为例。

![](img/b418a7224b18aead6e85afb92bc7f7d7_29.png)

![](img/b418a7224b18aead6e85afb92bc7f7d7_31.png)

1.  **寻找样本**：在小红书等平台搜索“考研”，筛选点赞量高（例如3000以上）的笔记。
2.  **复制文案**：将选中的爆款笔记全文内容复制下来。
3.  **向AI下达分析指令**：向ChatGPT提供一个清晰的分析指令模板。

![](img/b418a7224b18aead6e85afb92bc7f7d7_33.png)

![](img/b418a7224b18aead6e85afb92bc7f7d7_35.png)

![](img/b418a7224b18aead6e85afb92bc7f7d7_37.png)

![](img/b418a7224b18aead6e85afb92bc7f7d7_39.png)

![](img/b418a7224b18aead6e85afb92bc7f7d7_41.png)

![](img/b418a7224b18aead6e85afb92bc7f7d7_43.png)

**核心指令代码示例**：
```
你是一名小红书爆款文案分析师。我将提供一篇小红书上的爆款笔记文案，请你分析：
1.  **爆款因素**：总结该文案能成为爆款的关键原因。
2.  **整体结构**：
    - 开篇用什么方法吸引用户？
    - 中间用什么内容戳中用户痛点？
    - 结尾用什么方式引导用户互动（如评论、私信、关注）？
3.  **关键词分析**：根据整篇文案，分析作者是否结合中文分词法设计了特定关键词。如果有，请指出是哪个关键词，并说明其出现频率和比率；如果没有，请说明。
```
将复制的文案粘贴在指令下方，发送给AI。

## 第二步：整合爆款因素

![](img/b418a7224b18aead6e85afb92bc7f7d7_51.png)

![](img/b418a7224b18aead6e85afb92bc7f7d7_53.png)

![](img/b418a7224b18aead6e85afb92bc7f7d7_54.png)

![](img/b418a7224b18aead6e85afb92bc7f7d7_56.png)

![](img/b418a7224b18aead6e85afb92bc7f7d7_57.png)

向AI提供多篇（建议3-5篇）爆款笔记进行分析后，需要让AI对分析结果进行总结和去重。

**整合指令示例**：
```
请将你前面分析过的所有关于考研的爆款笔记（共X篇）的爆款因素总结一下，整合成一个去重后的完整列表。
```
AI会输出一个包含“情感共鸣”、“逆袭故事”、“明确时间规划”、“实用资源建议”、“正能量鼓励”等常见爆款因素的列表。这个列表就是我们希望新笔记具备的核心要素。

## 第三步：生成爆款笔记大纲与文案

有了爆款因素列表后，就可以指导AI进行创作了。

1.  **生成大纲**：首先让AI根据爆款因素，规划一篇新笔记的框架。

    **指令示例**：
    ```
    我需要写一篇关于考研的小红书笔记。请结合你刚才总结的爆款因素，为我规划一个具备爆款潜质的笔记大纲。大纲需要细分到具体的小节点。
    ```
    AI会生成一个包含“开篇引入”、“专业选择分析”、“各科备考规划”、“时间管理”、“心态调整”、“互动引导”等部分的大纲。

![](img/b418a7224b18aead6e85afb92bc7f7d7_59.png)

2.  **生成完整文案**：在确认大纲无误后，让AI根据大纲写出完整文案，并加入更多具体要求。

    **最终生成指令示例**：
    ```
    请根据上面确定的大纲，撰写完整的笔记文案。要求如下：
    - 文案结构严格遵循爆款笔记的要素和结构。
    - 开篇需要包含一个简单的“逆袭小故事”来引发读者共鸣。
    - 合理使用emoji表情。
    - 尝试以“考研全流程”、“考研政治”等作为关键词，自然地融入文案中。
    ```

## 关键技巧与注意事项

在操作过程中，有几个要点需要注意：

*   **指令的明确性**：给AI的指令必须清晰、具体。例如，“包含emoji表情”和“包含丰富的emoji表情”会导致不同的输出结果。
*   **AI的“记忆”问题**：使用ChatGPT 3.5等模型时，它可能忘记之前的对话上下文。如果发现AI跑题或遗忘，需要重新发送或强调核心指令。
*   **结果的优化**：AI生成的初稿可能比较生硬或普通。对于运营人员来说，它提供了一个优质的基础稿，可以在此基础上进行人工优化和润色，使其更生动、更有个性。
*   **进阶工具**：可以考虑使用ChatGPT 4.0或创建自定义的GPTs。通过向GPTs上传更多小红书风格范例、专用表情包等资料，可以打造出更懂你、产出更符合平台调性的私人AI助手。
