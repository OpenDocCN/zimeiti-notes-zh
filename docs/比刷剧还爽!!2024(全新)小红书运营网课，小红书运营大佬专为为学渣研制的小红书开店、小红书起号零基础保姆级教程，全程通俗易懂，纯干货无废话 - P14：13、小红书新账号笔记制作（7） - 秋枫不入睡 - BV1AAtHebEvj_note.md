# 小红书运营：P14：13、小红书新账号笔记制作（7）📝

在本节课中，我们将要学习小红书笔记制作的第三种核心类型——内容笔记。我们将详细解析内容笔记的定义、作用、制作要点，并厘清它与商品笔记、引流笔记的区别与联系。

![](img/408e41699a60224f46802862fb6189e6_3.png)

上一节我们介绍了商品笔记和引流笔记，本节中我们来看看内容笔记。

![](img/408e41699a60224f46802862fb6189e6_5.png)

内容笔记与引流笔记的相似度超过70%，但与商品笔记的差异较大。内容笔记的实际效果是在特定领域内进行大规模的人群辐射。如果你计划成为网红或博主，内容笔记是你的重要渠道。

简单来说：
*   **引流笔记**：核心目标是获取流量。
*   **商品笔记**：核心目标是直接促进商品销售。
*   **内容笔记**：核心目标是建立个人或品牌影响力，为成为网红或博主铺路。

![](img/408e41699a60224f46802862fb6189e6_7.png)

因此，内容笔记在技术属性上与商品笔记类似，但在内容宣传的属性和目标上，与另外两种笔记完全不同。我们需要重点关注这些不同的地方。

## 内容笔记的定位与作用 🎯

内容笔记的选题定位，应选择你熟悉且能对目标受众产生价值的主题。这可以是个人经验分享、知识讲解、产品测评等。

因为你的目标是成为网红或博主，所以你必须对所选领域的知识点有清晰认知。如果认知不清，以“盲猜”形式制作笔记，将无法提供实质性价值，难以吸引用户。

本质上，作为网红或博主，你是一个“情报系统”，负责搜集、整合信息并结合自身经验，分享给感兴趣的用户。

在确定定位时，需明确笔记的最终作用，即想清楚如何变现。主要方式包括：
1.  **推广商品**：通过影响力带货。
2.  **接广告**：为品牌进行内容营销。
3.  **推广分享内容/品牌**：将账号打造成网红账号，进而宣传自己公司的产品或品牌，提升知名度。

对于个人博主，前期应专注于一个方向进行发展，这样更容易成功。账号成长起来后，可以再考虑拓展其他变现方式。内容笔记的核心作用是**提升账号整体权重**，其目标性不像商品笔记那样直接明确，更具灵活性。

## 内容笔记的制作要点 ✍️

内容笔记的发送渠道与引流笔记相同，都位于小红书的“推荐”页面中。这类笔记通常提供较多的实用信息。

其标题吸引力和内容价值的要求与商品笔记类似，但宣传方式更间接、更深入。你需要像讲述故事一样，介绍产品的背景、用法等，每一篇笔记都是对个人品牌或产品理念的一次宣传。

以下是内容笔记的三种核心形式，你可以根据自身情况选择：
*   **结构清晰**：确保宣传内容有严密的逻辑性，经得起推敲。
*   **引用数据**：在可能的情况下，使用数据和研究来支撑你的观点，增加内容的权威性和说服力。例如，可以进行产品使用前后的效果对比，但需注意方式方法。
*   **情感链接**：尝试与用户建立情感上的连接。这种方式属于“慢热型”，适合长期、持久的品牌建设或情感类博主。

## 三种笔记的关系与选择 🤔

内容笔记整体上与商品笔记差距较大，但与引流笔记有70%的相似度，只是侧重点不同：
*   **引流笔记**：受众广泛，属性不精准，“广撒网”式地获取流量。
*   **内容笔记**：受众精准，针对对专业内容有需求的人群，旨在成为特定领域的“领路人”。

在实际操作中，笔记制作主要围绕三个点两两组合进行：
1.  **引流笔记 + 商品笔记**
2.  **引流笔记 + 内容笔记**
3.  **内容笔记 + 商品笔记**

**商品笔记永远处于链条的最后端**。你可以只做引流笔记，而不做商品笔记或内容笔记。具体选择哪种组合，完全取决于你在小红书平台想要发展的渠道和目标。
