## 账号权重的决定因素

![](img/308d62192a45019d00ab7352cd931acc_6.png)

账号权重主要围绕两个核心维度进行评估：**账号分值**和**视频分值**。理解这两个维度是提升权重的基础。

![](img/308d62192a45019d00ab7352cd931acc_8.png)

### 1. 账号分值

账号分值由系统根据以下三个方面进行综合评分：

![](img/308d62192a45019d00ab7352cd931acc_10.png)

*   **实名认证**：完成账号的实名认证。
*   **资料完整度**：完善头像、昵称、签名等所有资料。
*   **违规记录**：账号是否存在违规或限流历史。

![](img/308d62192a45019d00ab7352cd931acc_12.png)

**核心公式**：`账号分值 = f(实名认证, 资料完整度, 违规记录)`

一个资料完整、无违规记录的账号，更容易被系统判定为“优质账号”，从而获得基础流量推荐。反之，违规操作会导致限流甚至封号。

---

上一节我们介绍了账号的基础分值，接下来我们看看更动态、更重要的部分——视频分值。

### 2. 视频分值

视频分值决定了单个内容乃至账号整体的流量表现。系统主要从以下五个方面进行考量：

以下是评估视频分值的五个关键指标：

1.  **播放量**：视频的绝对播放数据，需达到同类内容的平均水平线以上。
2.  **互动指数**：综合评论、点赞、收藏、分享和新增关注等互动行为的数据。
3.  **粉丝净增长量**：考察周期（通常为7天）内粉丝的净增长数量。
4.  **投稿频率**：账号持续、稳定发布视频的频率。
5.  **完播率**：用户完整观看视频的比例，是内容吸引力的核心指标。

**核心逻辑**：`高权重 = 稳定投稿 + 高播放量 + 高互动 + 高完播率 → 粉丝正增长`

这五个指标相互关联，形成循环。投稿是流量的源泉，稳定的更新（如日更或隔日更）才能为其他指标的提升提供可能。高互动和高完播率能推动播放量增长，进而带来粉丝正增长，最终全面提升账号权重。

---

## 提升权重的核心策略

基于以上分析，提升账号权重的唯一方法是建立并维持一个正向循环。这要求创作者具备双重能力。

以下是实现权重提升的两个关键行动准则：

*   **保证稳定的更新频率**：这是维持账号活跃度、获取流量基础的底线。如果无法日更，建议至少保持三天一更，避免长时间断更。
*   **持续输出优质内容**：在稳定更新的前提下，尽可能提高内容质量，以获取高播放、高互动和高完播率，驱动粉丝增长。

**代码式描述**：
```python
while 运营账号:
    发布视频(保证频率, 提升质量)
    数据 = 获取反馈(播放量, 互动, 完播率)
    if 数据达标:
        权重 += 1
        粉丝数 += 增长
    else:
        权重 -= 1
```

对于绝大多数创作者，必须兼顾“量”与“质”。仅有关注频率而内容粗糙，或内容优质但更新缓慢，都难以持续提升权重。只有两者结合，才能让账号进入“发布视频 → 获得正反馈 → 权重提升 → 获得更多流量”的良性循环。
