在本节课中，我们将深入探讨小红书平台如何为用户打标签，以及平台主要的流量入口有哪些。理解这些底层逻辑，有助于我们分析自己的笔记如何被系统推荐给精准用户，从而优化运营策略。

上一节我们介绍了系统如何判定内容标签，本节中我们来看看平台如何判定用户标签。这个过程能帮助我们反推笔记被推荐给用户的逻辑，从而找到让精准用户看到我们内容的方法。

## 用户标签判定逻辑 🔍

小红书系统主要通过分析用户的**浏览行为**来为其打标签。用户注册时选择的兴趣范围参考价值不大，仅作为初期内容方向的引导，无法进入系统深层的用户标签池。

系统根据以下几种用户行为来判定其兴趣分类：

以下是判定用户标签的核心行为指标：

1.  **点击率**
    *   **定义**：系统在发现页向用户推送多种内容（如美妆、家居、母婴等），用户点击某一类内容的频率。
    *   **作用**：系统通过计算用户点击不同类别内容的概率，为其打上强关联或弱关联的标签。例如，用户频繁点击美妆内容，系统会判定其对美妆感兴趣，并持续推送相关笔记进行试探。

2.  **停留时长**
    *   **定义**：用户点开一篇笔记后观看的持续时间。
    *   **作用**：即使同属一个大类，用户对不同具体内容的停留时长也能反映其细分偏好。例如，一个美妆用户对多数美妆笔记划走得很快，但对某篇家居笔记停留很久，系统可能判断其近期有装修或购买家居用品的需求。

3.  **页面浏览量（PV）**
    *   **定义**：用户浏览页面的深度，例如不仅看了单篇笔记，还进入了博主主页并浏览了其他多篇内容。
    *   **作用**：PV值高，说明用户对账号内容兴趣浓厚。这能显著提升账号的健康度评分，尤其在新账号粉丝基数较小时，是重要的正向指标。

4.  **互动行为**
    *   **定义**：包括点赞、收藏、评论、关注等用户主动操作。
    *   **作用**：这些数据直接决定笔记能否进入下一层级的推荐流量池。高互动数据不仅有利于笔记获得更多推荐，也能带动账号整体权重提升，为后续接单、变现打下基础。

简单来说，系统通过持续观察和分析用户的这些行为数据，像绘制一幅兴趣云图一样，为其打上动态变化的标签，从而确保推送的内容能最大程度满足用户喜好，提升其在平台的停留时间和获得感。

## 平台流量分发入口 🚪

了解用户标签后，我们来看看小红书主要的流量分发入口。作为运营者，需要清楚不同入口的特点和推荐逻辑。

小红书目前主要有四个流量入口：

以下是四个核心流量入口及其特点：

1.  **发现页**
    *   **内容构成**：
        *   已关注达人发布的新内容。
        *   系统根据用户标签（点击率、兴趣等）匹配的推荐内容。
        *   少量平台为丰富内容多样性而推送的试探性内容。
    *   **核心价值**：这是笔记和账号获取流量的**最主要来源**。运营的终极目标就是让内容进入发现页的推荐流。

2.  **关注页**
    *   **内容构成**：纯粹展示用户所有已关注博主发布的内容。
    *   **特点**：流量精准但相对有限，取决于粉丝的活跃度和打开习惯。

3.  **附近页**
    *   **内容构成**：基于手机定位，推送同城或附近地区的相关内容。
    *   **适用内容**：具有强地域特征的内容，如本地美食、旅游攻略、线下店推广（美容院、餐厅等）。
    *   **运营提示**：做线下实体店推广的运营者，需要特别重视这个入口，并在考虑信息流广告时，可侧重附近人群的定向投放。

4.  **购物页**
    *   **内容构成**：小红书电商生态的入口，直接导向商品和带货直播。
    *   **现状**：部分用户尚未形成在此直接购物的强习惯，更多人仍习惯先搜索相关笔记（种草）再做决策。该入口是平台商业变现的重要方向。
