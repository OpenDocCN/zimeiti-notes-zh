# 小红书运营教程：P28：小红书爆款笔记写作方法（二）

在本节课中，我们将学习如何确立小红书笔记的内容框架。我们将分步骤讲解从确定领域到最终成稿的完整流程，并介绍一个实用的数据分析工具来辅助创作。

上一节我们介绍了爆款笔记的基础写作思路，本节中我们来看看如何系统性地确立笔记内容。

## 内容确立的五个步骤

很多人在写小红书内容时，不知道如何确立主题和方向。以下是确立内容的五个核心步骤。

### 1. 确定内容领域

首先，必须明确你的账号专注于哪个领域。例如，是美妆、护肤、母婴还是其他类别。在之前的课程中已多次强调，领域不明确会导致账号垂直度降低。

账号垂直度降低会引发一系列问题：官方推荐减少、粉丝流失，并形成恶性循环——粉丝越不喜欢你的内容，官方就越不推荐。

### 2. 梳理行业关键词

确定领域后，需要梳理与该领域相关的关键词。在之前的课程中，我们介绍了使用百度工具、爱站网、5118、百度指数等工具进行关键词查询的方法。

本节课不再赘述具体操作。大家可以根据自身行业，利用这些工具梳理出关键词及其热度排行。用好这些关键词能极大提升内容效果。

### 4. 筛选与整理内容

搜集好同行案例后，即可对这些案例内容进行筛选。筛选完成后，将精华部分整理成自己的内容，然后进行发布。

以下用牵瓜数据演示操作。在工具左侧栏目中，可以看到诸多功能，如热词排行榜、热词搜索榜、笔记收录查询等。

本节课主要讲解“笔记收录查询”功能。发布笔记后，若担心笔记未被官方收录，即可利用此功能进行查询。热词排行榜等功能此处暂不展开。

另一个核心功能是“达人搜索”。可以按昵称搜索，或按笔记内容搜索（例如搜索“美妆”，但部分高级功能需付费）。按昵称搜索的结果，会展示达人的属性分类，如彩妆、护肤、母婴、美式饮品、服饰穿搭等。

大家可根据自身需求进行筛选。结果页会显示达人是否为官方号、品牌号或头部达人，并包含丰富的数据，如达人对比、排名等。

![](img/8f8ece6e760088babb00ce54623762af_11.png)

如果需要使用此网站，可根据实际需求在淘宝寻找开通渠道，费用通常比较划算。

![](img/8f8ece6e760088babb00ce54623762af_11.png)

利用此功能，点击任意达人的详情页，就能查看其发布的所有数据，包括笔记分类、详情，以及笔记评论的词云分析（展示高频词汇）。用好这个工具，能帮助我们更好地掌握小红书的数据参数。

### 5. 整理成稿与发布

内容筛选完毕后，最后一步是整理成自己的稿件并发布。关于内容结构，在之前的课程中已讲过，主要采用“总-分-总”的结构来撰写。

这种结构并不难掌握。如果觉得困难，建议回顾之前的课程内容，本节课不再重复。

## 其他注意事项

除了上述步骤，在撰写标题时还需注意以下几点。

以下是标题优化的三个要点：

1.  **控制字数**：标题字数尽量控制在 **12~18个字** 之内。
2.  **添加符号美化**：标题中可多带一些符号（如✨、❗、🔥）进行美化。小红书女性用户居多，美化后的标题更能吸引眼球。
3.  **加入具体数据**：在标题中加入具体数据可以增加真实感。例如，像“百万销量”、“3天瘦5斤”这类包含具体数字的标题，更容易让人信服。因此，在拟定标题时，请尝试加入数字。
