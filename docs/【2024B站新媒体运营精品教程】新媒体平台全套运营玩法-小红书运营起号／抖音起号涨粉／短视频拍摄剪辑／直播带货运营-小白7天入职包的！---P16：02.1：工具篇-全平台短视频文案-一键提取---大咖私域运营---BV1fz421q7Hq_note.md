# 新媒体运营工具教程：P16：02.1：工具篇-全平台短视频文案一键提取 📝

在本节课中，我们将学习一个非常实用的运营工具，它能帮助我们从抖音、小红书、哔哩哔哩等多个平台的短视频中，一键提取出文案内容。这个方法能极大提升我们收集素材和创作内容的效率。

上一节我们介绍了文案提取的需求，本节中我们来看看如何用一个在线工具实现跨平台文案提取。

## 工具介绍与优势

以往提取短视频文案，可能需要使用“轻斗”小程序，或者将视频下载后导入“剪映”等剪辑软件来提取字幕。这些方法较为繁琐，且难以覆盖所有平台。

今天分享的工具是一个在线网页工具。它的核心优势在于：**你只需将目标短视频的链接复制到工具中，即可在线一键提取出该视频的文案**。它支持抖音、快手、小红书、哔哩哔哩、视频号等多个主流平台。

## 工具使用演示

![](img/16bb276c24593cc3142ca7389d5eb36e_5.png)

以下是使用该工具提取文案的具体操作步骤。

![](img/16bb276c24593cc3142ca7389d5eb36e_7.png)

### 第一步：准备视频链接
首先，在你想要提取文案的平台（如抖音）上，找到目标视频并复制其分享链接。

### 第二步：打开工具并粘贴链接
打开该在线工具的网页界面，将复制的视频链接粘贴到指定的输入框中。

### 第三步：提取与处理文案
点击“提取文案”按钮。稍等片刻，工具就会将视频中的语音内容转换成文字并显示出来。

提取出的文案，你可以进行两项关键操作：
1.  **一键复制**：直接复制全部文本。
2.  **一键格式化**：点击此按钮，工具会将文案按照视频中的语句分段排列，使其更清晰易读，类似于字幕格式。

此外，该工具也支持**上传本地视频文件**来提取文案。

## 跨平台提取实例

为了验证工具的效果，我们分别在抖音和哔哩哔哩平台进行了测试。

### 实例一：提取抖音视频文案
我们选择了一个抖音的情感语录视频，将其链接粘贴到工具中。提取结果准确输出了视频中的全部口播文案，例如：“听完你就释怀了，村上春树说，我动了离开你的念头，不是因为你不好，也不是因为不爱了……”。

### 实例二：提取哔哩哔哩视频文案
我们同样测试了一个B站的运营教学视频。将视频的BV号链接粘贴后，工具成功提取出了视频文案，例如：“新手做直播电商，怎么才能打造出直播爆品？在直播销售的不同阶段……”。提取准确率非常高。
