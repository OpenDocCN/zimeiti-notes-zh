# 抖音运营指南：P26：推荐机制—时间效应 📅

![](img/8e235bf0c24041e3f148d10441c4a574_1.png)

在本节课中，我们将要学习抖音推荐机制中的第三个重要概念——时间效应。理解了时间效应，你就能明白为何有些视频会在发布很久之后突然获得流量。

上一节我们介绍了基础推荐和叠加推荐，本节中我们来看看时间效应。

## 什么是时间效应？

时间效应是指，一个视频在发布后，即使初期数据不佳，也可能在未来某个时间点因为算法或热点事件而被重新推荐，从而获得新的播放量增长。

## 时间效应如何运作？

以下是其运作原理的简要说明：

1.  **视频入库**：用户发布的每一个视频都会被存入抖音庞大的视频库中。
2.  **静默期**：如果视频发布初期内容不够优质或未蹭上热点，它可能暂时“沉睡”在视频库中。
3.  **二次触发**：当未来出现新的热点事件或话题时，抖音的算法会从视频库中搜索与当前热点相匹配的“库存”视频。
4.  **重新推荐**：被算法选中的旧视频会被重新提取，进入新一轮的推荐流程（即再次经历基础推荐和叠加推荐），从而获得“二次生命”。

## 一个具体例子

假设你发布了一个关于明星A的视频，但发布当天反响平平。一周后，明星A因为某新闻事件突然成为热点。此时，抖音算法可能会从视频库中找到你之前发布的那个相关视频，并将其重新推荐给对明星A感兴趣的用户，从而使该视频的播放量大幅上涨。

这个概念源自早期网络社区的“挖坟”机制，即通过算法挖掘旧内容以匹配新热点。

## 与快手机制的对比

这里需要强调，时间效应是抖音平台的特色机制。另一个主流短视频平台——快手，采用的则是“**择新去旧**”机制。

以下是两者的核心区别：

*   **抖音（时间效应）**：算法会持续从整个视频库中匹配内容，旧视频有机会因新热点被重新推荐。
*   **快手（择新去旧）**：算法主要推荐新发布的视频。一个视频发布一周或一个月后，通常就不再获得系统推荐，无论之后出现何种热点。

因此，在抖音平台，用心制作每一个视频是有长期价值的，因为它可能在未来的任何时候被重新激活。
