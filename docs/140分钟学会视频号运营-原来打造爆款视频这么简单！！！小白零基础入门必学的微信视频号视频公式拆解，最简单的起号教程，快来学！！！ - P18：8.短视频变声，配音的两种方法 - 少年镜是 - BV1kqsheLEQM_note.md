# 视频号运营：P18：短视频变声与配音的两种方法 🎬

在本节课中，我们将学习为混剪视频进行声音去重的两种核心方法。通过改变视频的原声，可以有效提升作品的原创度，这对于视频号运营至关重要。

上一节我们介绍了视频混剪的基本技巧，本节中我们来看看如何具体处理视频中的音频，以实现去重。

## 方法一：使用剪映内置变声功能 🎚️

第一种方法直接在剪映App内完成，操作简单快捷。

![](img/e053d377f05e60288e6aaa5080b83e16_1.png)

以下是具体操作步骤：

1.  将需要处理的视频导入剪映。
2.  选中视频轨道，在编辑面板中向左滑动，找到“声音效果”选项。
3.  点击进入，即可看到多种变声效果，如“圣诞老人”、“台湾小哥”等。
4.  选择喜欢的音效（例如“男生-大叔”），视频的原声就会立即改变。
5.  部分音效支持微调参数，可以点击“参数”按钮进行个性化设置。

**核心操作**：`选中视频 -> 声音效果 -> 选择变声`

![](img/e053d377f05e60288e6aaa5080b83e16_1.png)

这种方法能快速改变人声，但背景音乐保持不变。接下来，我们学习一种更彻底的音频替换方法。

## 方法二：使用配音神器重新生成配音 🗣️

第二种方法通过第三方工具提取文案并生成全新配音，能同时替换人声和背景，去重效果更佳。

以下是完整的操作流程：

![](img/e053d377f05e60288e6aaa5080b83e16_3.png)

1.  **提取视频文案**：打开“配音神器”等工具，从手机相册导入目标视频，使用“文案提取”功能获取视频内的所有文字。
2.  **修改文案**：对提取出的文案进行适当修改。这是关键的去重步骤。
    *   **修改原则**：保持原意，替换部分词语或调整句式。例如，将“老人”改为“大人”，将“怎么办呢？”改为“咋办呢？”。
    *   **注意事项**：避免大幅度增删文字，以防与视频口型对不上。
3.  **生成新配音**：将修改后的文案复制到配音神器的“制作配音”页面。
    *   选择你喜欢的声音类型（如“男生-阳光”）。
    *   点击“合成配音”，试听满意后“导出”并“下载”到本地。
4.  **替换视频原声**：打开剪映，导入原视频。
    *   关闭视频的“原声”。
    *   点击“添加音频” -> “提取音乐”，导入刚才生成的新配音文件。

![](img/e053d377f05e60288e6aaa5080b83e16_5.png)

![](img/e053d377f05e60288e6aaa5080b83e16_3.png)
![](img/e053d377f05e60288e6aaa5080b83e16_5.png)

5.  **添加背景音乐（可选）**：此时视频已无背景音乐，你可以点击“音频”->“音乐”，添加新的背景乐，并调低音量，避免掩盖配音。

![](img/e053d377f05e60288e6aaa5080b83e16_7.png)

![](img/e053d377f05e60288e6aaa5080b83e16_7.png)

6.  **完成混剪**：最后，可以使用剪映的“识别字幕”功能为新的配音生成字幕，再结合之前学习的混剪技巧完成作品。

本节课中我们一起学习了为短视频进行声音去重的两种实用方法：利用剪映的**变声功能**快速修改，以及通过**配音神器提取文案并重新生成**来彻底替换音频。掌握这些技巧，能有效提升你视频的原创性和独特性，是运营视频号的重要一环。
