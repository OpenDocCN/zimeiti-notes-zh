## 课程概述

## 小红书笔记违规的常见类型

小红书平台的主旋律是**种草**与**分享**，而非硬性营销。平台对直接广告行为管控严格。以下是几种常见的违规类型：

1.  **广告与销售笔记**
    笔记内容推销意图过于明显，让用户一眼就能看出是在直接推销产品。

2.  **不合规地留下联系方式**
    在笔记或评论区直接留下微信、电话等联系方式。目前相对安全的方式是在个人简介中留邮箱，或采用“大小号联动”的间接方法（大号笔记中@只留有联系方式的小号）。

3.  **非原创内容**
    小红书判断原创的初级标准是：内容是否首次在小红书平台发布。直接从平台内搬运或拼凑的内容，会被判定为非原创。
    *伪代码示例：*
    ```python
    if 内容 in 小红书历史数据库:
        判定为“非原创”
    else:
        判定为“原创”（初步）
    ```

4.  **伪造数据**
    通过非正常手段刷高粉丝数、阅读量、点赞、收藏等数据。平台有监测模型，异常数据（如点击率或关注转化率远高于正常范围）极易被识别为作弊。
    *示例逻辑：*
    ```
    正常广告点击率 ≈ 0.3% - 3%
    若你的笔记：阅读量=100，新增粉丝=50
    则粉丝转化率 = 50%，显著异常 → 判定为数据造假
    ```

5.  **不友善行为**
    在笔记或评论中无确凿证据地诽谤、恶意抨击他人或竞品。发表观点需有依据，避免情绪化的攻击言论。

6.  **诱导分享**
    以利益（如礼品、红包）诱导用户分享笔记，此类营销行为不被允许。

7.  **侵权、盗版、虚假或无价值内容**
    特别是“无价值内容”，指那些缺乏实质信息、纯粹自嗨或没有逻辑支撑的文章（例如，宣称在房间某处放盆水就能改运，却无任何科学依据）。

8.  **色情低俗及封建迷信内容**
    发布涉黄或宣扬封建迷信（如算命、看风水）且无法用逻辑论证的内容。

## 违规问题的解决方案

了解违规类型后，以下是针对性的解决方法：

1.  **采用多产品对比**
    不要只宣传单一产品。可以通过对比同类多个产品的特性来突出自身优势，这样更符合“分享测评”的基调，而非硬广。
    *示例公式：*
    **产品A优势** = (特性1对比) + (特性2对比) + (新闻/数据佐证)

2.  **杜绝数据造假**
    坚决不要刷量。平台的反作弊系统能有效识别异常数据，造假会导致账号降权甚至封禁。

3.  **客观评价，不恶意贬低**
    评价竞品时应有理有据，避免无证据的贬低，以防被举报。

4.  **避免诱导与低质内容**
    不进行诱导分享，不发布色情、低俗或无逻辑的虚假内容。

5.  **坚持发布原创内容**
    这是最核心的解决之道。即使你不是专业文案，也可以通过掌握方法创作出高质量的原创笔记。关于如何零基础写出原创爆文，我们将在下一节课（第15节）中详细展开。
