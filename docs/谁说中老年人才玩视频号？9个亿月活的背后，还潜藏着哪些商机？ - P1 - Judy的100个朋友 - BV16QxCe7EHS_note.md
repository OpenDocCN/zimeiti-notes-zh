# 视频号商业机会解析：P1：视频号生态与核心逻辑

在本节课中，我们将深入探讨视频号的商业生态、核心逻辑以及其中潜藏的机遇。我们将分析视频号与其他平台的区别，理解其“人的生意”本质，并探讨普通人如何在这一生态中找到自己的位置。

## 视频号：最后的流量洼地

上一节我们介绍了课程主题，本节中我们来看看视频号的基本面。视频号的月活跃用户已达到9亿。与品牌和CMO圈子的交流显示，各平台定位不同：抖音是卖货的，小红书是种草的，而视频号是专门给老板和公关同事看的平台。

从去年开始，市场普遍认为流量红利消失，大家都在寻找下一个流量洼地，而视频号在2024年仍存在许多新机会。三年前这个说法或许不准确，但如今情况已变。当时抖音日活约7亿，而微信日活约10-11亿。两者相减，约有3亿用户在使用微信但未接触短视频和直播。这部分用户虽然是社会存量用户，但很可能是整个短视频和直播市场的增量用户。这就是视频号被称为“最后洼地”的原因。

如今，几乎每个手机用户都知道微信有视频号功能。许多电商从业者来到视频号，初期起量相对容易，但商品似乎特别容易过时。这是因为很容易触及品类的天花板。因此，在视频号里，需要关注**LTV**，即用户生命周期的价值，不能仅靠一个爆品反复消耗用户。

视频号虽然观看人数众多，但其中形成稳定消费习惯的用户仍在增长过程中，需要商家去培育。这本质上是信任厚度的问题。因此，某些行业在视频号中表现尤为突出。

以下是表现突出的行业类别：
*   教培类目：包括儿童教育、成人教育、职场技能、个人IP打造及直播教学，甚至针对老年人的课程，凡是知识密集型的领域都表现良好。
*   特定电商品类：例如珠宝玉石、滋补类目等，在视频号中也跑得不错。

当你更深入地观察视频号，会发现其当前的红利并非普涨普跌、大开大合的洼地概念，而是平台官方在帮助提升整体渗透率。

## 视频号生意的核心：人大于货

上一节我们分析了视频号的用户基本盘，本节中我们来看看其核心商业逻辑。你解读了许多未来趋势。首先，现在是商家进入视频号的好时机，因为商家拥有自己的产品和服务。其次，在所有商家中，特别是那些注重客户服务、交易并非一次性结束、需要大量私域维护以实现复购和升单的产品，特别适合视频号。

人们经常问视频号与其他平台的区别。视频号内的生意强烈围绕“人”进行，即**人的生意远大于货的生意**。视频号在微信生态中的底层逻辑，是构建在熟人社交关系网络之上的。它有公众号、朋友圈这样的“兄弟”功能，只不过有些属于私域，有些属于公域。短视频和直播的加入，只是加快了信息流动，从而加速了人设的建立。

我们仔细思考会把什么内容发到朋友圈？通常是能代表自身价值主张、自己高度认同的内容。归纳来说，就是那些你认为高于你自身、你所向往和追求的内容。它一定是抬头看的目标，我们都希望扮演一个更好的自己。记住，是“扮演”。人就是这样，不是把欲望摆在台面，而是把梦想摆在台面上去分享。

基于这个逻辑，你会发现视频号里大部分能传播开的人设，都是“抬头看”的人设。例如，视频号里“好老师”人设特别火，带货能力也很强。当然，在某个领域成为能解决问题的“好专家”也可以。一旦这种被仰望的人设建立起来，用户就特别容易与你产生深度信任。在小红书，大家认可的是“我”的品味；在视频号，大家认可的是“我”的人品。这是非常深刻的区别。

前两年，人们对视频号的印象可能是很多商品卖给中老年人，或者知识付费等虚拟产品在视频号成交特别好。这一两年有所变化吗？对于新的、想要入局视频号的人来说，还存在哪些潜在的细分品类机会？不要误会它是一个中老年平台。随着渗透率加深，到明年后年，用户群体一定会逐步下移。

例如，今年其消费数据已从老年人群体转移到“姐姐”群体——即中年女性，她们是家庭中有娃有老公的消费中枢。这类用户有两个特点：
1.  消费能力更强。
2.  更关注产品的内涵，无论是内容内涵、品牌故事内涵还是情绪价值内涵。

当这些因素结合在一起，你会发现从去年到今年，有几类品类增长很快，例如香云纱、沉香、马面裙等。同时，市场上流行一个词叫“创始人IP”。无论是新中式服饰，还是沉香等疗愈类品牌，往往都由一个IP在前端引流，后端成交产品。近一两年，各种企业家IP下场，他们带着高信任度、高背书，例如雷军、周鸿祎。这对普通人的冲击和挑战是什么？

## 创始人IP的本质与普通人的机会

上一节我们探讨了视频号“人设”的重要性，本节中我们聚焦于高维度的“创始人IP”。我认为IP一旦下场，本质上并非在存量中做争夺。今天市场对IP的一个定义是用于获取流量。流量是什么？流量就是认知增量。创始人IP对于一个行业或品牌来说，最大的能力在于能否作为品牌的“老师”去培训市场，其培训方法就是构建一种认知增量。它其实是更高一个维度，在引领用户迁移到下一个认知上。

这有点像江南春所说的“抢占用户心智”，而且是一种新的心智，是用新的认知替代旧的。因为用户在认知上有一个天性——喜新厌旧。即使你今天做的不是教培，营销的本质就是教培，即把一个概念植入用户脑海。只不过教培交付的就是这个概念本身，而营销在植入概念后，还需要有一个产品来交付。例如卖车，就是先告诉你需要有一个“移动的家”。所有关于新主张、新概念、增量理念的营销，都是如此。

我们回到最初的问题。创始人IP是做什么的？创始人IP就是那个“老师”，或者说得更极致一点，就是那个“教主”。我感觉很多老板做创始人IP的思路完全错了。老板或品牌，必须占据用户认知的制高点。那个点是网红无法替代、平台投放也无法替代的。你要去那个点扎根、立旗，告诉大家“我为什么是我，是不一样的烟火”，然后把“烟火”放出去。大家抬头看“烟火”时，注意力就是你的流量。占据了那个高点，然后向下辐射时，注意力自然能转换成流量，也许大也许小，但这已不是最重要的事了。

因此，老板做创始人IP的目的不是为了卖货，但最终结果一定能卖掉货，是这个逻辑。否则，作为老板天天在直播间喊“321上链接”，和员工抢工作，员工去干什么呢？没事可做了。

过去这一年，IP似乎成了很多老板的心病，觉得做不起来IP公司就完蛋，把创始人IP或做内容当作拯救公司的唯一稻草。我认为，本质上老板的这种焦虑都源于市场内卷。大部分能做成公司的，通常不是因为专业技能有多强，而是因为在专业领域里，你的主张是人群中最坚定的。因为你坚定，所以你能看得最远，能承受中途的艰险。你成了“唐僧”，所以才能招来孙悟空（能力比你强）、猪八戒（比你滑头）、沙僧（比你忠诚）。但他们的老板是你——唐僧，因为你看到了远方。

在没有“流量”这个概念的时代，创业的基本法就是这样：谁坚定谁是赢家，谁看得远谁是赢家，谁有信仰谁是赢家。但为什么到了流量时代，就变成了谁“胸口碎大石”、谁演技好谁是赢家？我特别反对前几年一些抖音博主提出的“流量是一切生意的根本”的说法。

**价值才是一切生意的根本**。价值再往下延伸，**你对未来的定义才是一切生意的根本**。大家说我们应该向流量下跪，贵什么贵？最终，流量一定会流向对社会有价值的资产和人。你刚刚说的那段话很像创业者的使命和初心。应该是使命感内容化，内容才是创始人IP该做的，这才是核心和基石，才具有感召力。

我今天早上听了你的播客，觉得讲得特别好：创业者天生就是表达者。我特别赞同。我创业15年，如果你对这个世界没有强烈的表达欲，是不会去创业的。你可以测试自己：是否对这个世界有强烈的表达欲？这种表达是“我要在这个世界的牌桌上，抢几个席位坐下来，不管你们喜不喜欢我”。这可能是创业的原点。

本节课中我们一起学习了视频号的商业生态。我们了解到视频号拥有9亿月活的庞大基础，其核心逻辑是“人的生意远大于货的生意”，建立在微信的熟人社交网络之上，适合需要深度信任和长期维护的品类。我们探讨了“抬头看”的人设建立方法，并深入分析了“创始人IP”的本质——不是争夺存量流量，而是创造认知增量，占据用户心智的制高点。最后，我们重申了价值和对未来的定义才是生意的根本，流量终将流向有价值的资产。对于创业者而言，将使命感转化为内容，才是打造IP的正道。
