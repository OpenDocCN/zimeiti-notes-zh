# 小红书运营教程：P23：21.快捷键的使用 🚀

在本节课中，我们将要学习如何设置和使用小红书运营工具的快捷键。掌握快捷键能显著提升工作效率，让你在内容创作和店铺管理中更加得心应手。

![](img/c096ad616c70943661aee0dde9ceddb3_1.png)

## 概述

上一节我们介绍了工具的基础操作，本节中我们来看看如何通过快捷键来优化工作流程。熟练使用快捷键可以避免频繁使用鼠标点击，从而节省时间，提高操作速度。

![](img/c096ad616c70943661aee0dde9ceddb3_3.png)

## 如何找到快捷键设置

有两种主要方法可以打开快捷键设置面板。

![](img/c096ad616c70943661aee0dde9ceddb3_5.png)

**方法一：通过菜单栏**
首先，可以在软件顶部的菜单栏中找到“帮助”选项。

![](img/c096ad616c70943661aee0dde9ceddb3_1.png)

![](img/c096ad616c70943661aee0dde9ceddb3_7.png)

在“帮助”菜单中，能够找到“快捷键”选项，单击即可打开快捷键的设置界面。

![](img/c096ad616c70943661aee0dde9ceddb3_3.png)

![](img/c096ad616c70943661aee0dde9ceddb3_9.png)

**方法二：通过界面按钮**
当然，也可以在软件界面的右上角位置，找到一个类似小键盘的图标按钮。

![](img/c096ad616c70943661aee0dde9ceddb3_5.png)

单击这个按钮之后，同样会打开快捷键的设置面板。

## 快捷键的分类

快捷键设置面板中包含了多种类型的快捷键，以适应不同的操作场景。

以下是主要的快捷键分类：
*   **时间线快捷键**：用于在编辑时间线时快速操作，如剪切、复制、粘贴片段。
*   **播放器快捷键**：控制视频或音频的播放、暂停、前进、后退等。
*   **基础快捷键**：适用于软件全局的通用操作，例如保存文件、撤销、重做。
*   **其他快捷键**：涵盖更多特定功能或工具的快速访问。

![](img/c096ad616c70943661aee0dde9ceddb3_11.png)

## 自定义与修改快捷键

每个人都有自己的操作习惯，软件允许你自定义快捷键以满足个人需求。

**修改单个快捷键**
如果你更喜欢其他按键组合，可以直接单击现有快捷键进行修改。

![](img/c096ad616c70943661aee0dde9ceddb3_13.png)

![](img/c096ad616c70943661aee0dde9ceddb3_11.png)

**同步设置**
如果需要同步你的快捷键设置，可以登录账号。这样，当你更换电脑或其他设备时，就不需要重新进行设置了。

**操作步骤**
1.  在快捷键设置界面，直接按下你想要设置的新按键组合。
2.  如果新快捷键与其他功能冲突，系统会给出提示，此时需要重新修改。
3.  修改成功后，单击“保存”按钮以应用更改。
4.  如果你想放弃所有自定义，恢复成初始状态，可以单击“恢复默认值”按钮。

软件还提供了灵活性，允许你借鉴其他流行软件的快捷键布局，或者创建自己的模板。

以下是相关的设置选项：
*   你可以选择像 **Premiere Pro** 这类专业软件的快捷键预设，如果你熟悉它们的话。
*   你也可以完全自定义一套符合自己工作习惯的快捷键方案，并保存为模板。
