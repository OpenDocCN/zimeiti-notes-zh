# 新媒体运营从0到1：P17：02.2：工具篇-AI生成PPT 🚀

在本节课中，我们将学习一个非常实用的AI工具，它能帮助我们根据主题或现有文档，自动生成PPT大纲和详细内容。这个工具能极大提升我们制作演示文稿的效率。

上一节我们介绍了AI在文案撰写中的应用，本节中我们来看看如何利用AI辅助我们完成PPT制作。

## 工具核心功能

![](img/24b0a80da5aeb896320c89656ad67df3_1.png)

![](img/24b0a80da5aeb896320c89656ad67df3_3.png)

这个AI工具主要提供两种生成模式。

![](img/24b0a80da5aeb896320c89656ad67df3_5.png)

![](img/24b0a80da5aeb896320c89656ad67df3_7.png)

以下是两种核心模式的具体说明：
*   **主题生成模式**：输入一个主题，AI将结合类似ChatGPT的生成式能力，自动创作出完整的PPT全文内容。
*   **文档生成模式**：上传一篇现有的文档（如TXT、PDF文件），AI智能助手会自动总结文档大纲，并基于此大纲生成详细的PPT内容。

![](img/24b0a80da5aeb896320c89656ad67df3_9.png)

## 实战演示：基于文档生成PPT

![](img/24b0a80da5aeb896320c89656ad67df3_11.png)

接下来，我们通过一个实际案例来演示如何使用该工具的文档生成模式。

我准备了一篇关于“新媒体运营流量与创作标签”的文章作为示例。我们将上传这篇文章，让AI工具进行处理。

![](img/24b0a80da5aeb896320c89656ad67df3_13.png)

![](img/24b0a80da5aeb896320c89656ad67df3_15.png)

首先，在工具界面点击“继续”并上传文件。AI会解读文档内容，并自动梳理出大纲。例如，它可能总结出以下几个部分：
1.  新媒体运营的本质
2.  平台运营的角度
3.  账号运营的策略
4.  用户行为的分析与应用
5.  创作激励与流量分配
6.  数据分析在运营中的作用
7.  案例分析

![](img/24b0a80da5aeb896320c89656ad67df3_17.png)

大纲生成后，点击“开始制作”，工具会进入PPT生成页面。稍作等待，AI便会基于对内容的理解和大纲，自动填充每一页PPT的详细内容。

![](img/24b0a80da5aeb896320c89656ad67df3_19.png)

![](img/24b0a80da5aeb896320c89656ad67df3_21.png)

## 生成后编辑与优化

PPT内容生成完毕后，我们可以对其进行进一步的调整和美化。

工具右侧提供了编辑面板，我们可以进行以下操作：
*   **更换模板**：虽然当前提供的模板数量有限，但切换速度非常快。我们可以选择一个合适的模板来统一PPT风格。
*   **调整内容**：可以直接在页面中修改文字，或使用右侧的编辑工具调整形状、布局等元素。
*   **本地深度编辑**：如果对AI生成的基础框架满意，我们可以将PPT下载到本地，再使用PowerPoint或Keynote等软件进行更精细的调整。

![](img/24b0a80da5aeb896320c89656ad67df3_23.png)

这个工具最突出的优点是，它能够基于我们提供的简单大纲或文档，自动扩展出详实的讲解内容。这比仅仅将现成内容排版成幻灯片格式要有用得多，真正解决了“从思路到成稿”的痛点。
