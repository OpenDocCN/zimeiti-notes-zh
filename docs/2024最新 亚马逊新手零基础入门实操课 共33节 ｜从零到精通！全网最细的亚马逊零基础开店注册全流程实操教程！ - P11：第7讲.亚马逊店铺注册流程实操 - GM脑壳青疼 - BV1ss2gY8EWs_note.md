## 注册模拟器准备

上一节我们介绍了注册前的准备工作，本节中我们来看看如何找到并使用注册模拟器进行练习。

![](img/2bb342091eee14310e33fc8110cbb1fc_7.png)

首先，在搜索引擎中搜索“亚马逊店铺注册模拟”，即可找到官方的店铺注册模拟器。通常，卖家会优先注册美国站。

![](img/2bb342091eee14310e33fc8110cbb1fc_9.png)

**核心提示**：注册美国站时，其他12个站点（如欧洲站、日本站）也会同步开通，但处于未激活状态。后续只需在后台激活对应站点即可开始运营。



![](img/2bb342091eee14310e33fc8110cbb1fc_11.png)

![](img/2bb342091eee14310e33fc8110cbb1fc_13.png)

![](img/2bb342091eee14310e33fc8110cbb1fc_1.png)



进入模拟器后，选择“新卖家模拟注册”，并点击“开始模拟”。**请务必注意**：模拟过程中**不要使用任何真实信息**，否则可能导致与你现有或未来的真实账号产生关联，引发封号风险。



![](img/2bb342091eee14310e33fc8110cbb1fc_3.png)



## 账户基础信息设置

以下是设置登录账户所需的基础信息步骤。

1.  **姓名**：填写模拟器提供的虚拟姓名，例如“田丽琴”。
2.  **邮箱与密码**：使用模拟器提供的虚拟邮箱地址。密码可以随意设置，例如 `123`。
3.  **邮箱验证**：点击“下一步”后，模拟器会发送验证码。注意查看网页右下角的弹窗获取验证码（例如 `231416`），并输入以完成验证。



![](img/2bb342091eee14310e33fc8110cbb1fc_7.png)



4.  **手机号验证**：添加一个中国手机号，接收短信验证码并完成验证。



![](img/2bb342091eee14310e33fc8110cbb1fc_15.png)

![](img/2bb342091eee14310e33fc8110cbb1fc_11.png)

## 公司信息填写

完成个人账户设置后，接下来需要填写公司相关的详细信息。

1.  **选择国家与主体类型**：营业执照所在国家选择“中国”。主体类型选择“私有企业”。
2.  **公司名称**：填写公司全称，例如“杭州博辰网络科技有限公司”。注意，英文名称需全部小写且无空格，例如 `hangzhoubochenwangluokejiyouxiangongsi`。
3.  **统一社会信用代码**：准确填写营业执照上的“统一社会信用代码”，例如 `31101178226408125P`。
4.  **公司地址**：填写营业执照上的注册地址，并确保邮编准确无误（可通过网络查询，例如 `310016`）。

**关键核对点**：公司名称、信用代码和地址必须与营业执照完全一致，提交后无法返回修改。



## 法人身份信息验证

公司信息确认无误后，需要进一步验证店铺法人的身份信息。

1.  **法人姓名与国籍**：填写法人的中文姓名、拼音、国籍及出生地。
2.  **身份证信息**：准确填写身份证号码、有效期截止日期。
3.  **居住地址**：这是一个重要字段。建议填写**法人身份证地址**或**信用卡账单地址**。若账号触发审核（如二审），提供与此地址一致的账单有助于提高通过率。
4.  **企业关系声明**：根据公司股权结构如实勾选。如果是个人独资公司，可同时勾选“企业受益人”和“法人代表”；如有其他持股超过25%的股东，则需额外添加股东信息，注意这可能引发账号关联风险。



![](img/2bb342091eee14310e33fc8110cbb1fc_17.png)



## 收款与支付信息绑定

身份验证完成后，需要设置收款和支付方式，这是店铺能够正常回款和扣费的关键。

以下是绑定收款账户的步骤。

1.  **选择收款服务商**：例如选择“连连支付”。
2.  **填写账户信息**：输入账户持有人姓名（法人姓名）及9位数的银行识别代码。这些信息可在你的收款账户后台找到，或咨询客服获取。



接下来是绑定信用卡的步骤。

1.  **填写信用卡资料**：包括卡号、持卡人姓名、有效期（例如 `09/2024`）。
2.  **账单地址**：将收费地址修改为你的**信用卡账单地址**。如果与之前填写的居住地址不同，需点击“添加新地址”重新填写。



## 店铺信息与身份文件上传

绑定支付方式后，即可设置店铺名称并上传验证文件。

![](img/2bb342091eee14310e33fc8110cbb1fc_19.png)

1.  **店铺名称**：可先设定一个名称，店铺注册成功后仍可修改。
2.  **产品信息**：关于UPC编码和品牌问题，若无则选择“否”。
3.  **文件上传**：这是关键步骤，需要上传高清文件。
    *   **身份证**：正反面照片。
    *   **营业执照**：原件照片。
    **要求**：文件必须清晰，四个边角完整露出，无反光或遮挡。



![](img/2bb342091eee14310e33fc8110cbb1fc_19.png)



![](img/2bb342091eee14310e33fc8110cbb1fc_21.png)

上传后，仔细复核所有已填信息，确认无误后点击“提交”。



## 视频认证与完成注册

信息提交后，最后一步是完成视频认证。

1.  **预约时间**：系统会引导你预约一个视频通话时间（通常为工作日）。选择合适的时间段，例如“周五下午1:30”。
2.  **保存入口**：务必保存好视频认证的网址链接。届时需使用之前注册的邮箱和密码登录。
3.  **认证准备**：
    *   由法人本人参加。
    *   提前准备好**营业执照原件**和**身份证原件**。
    *   确保电脑或手机的摄像头、麦克风正常工作。
    *   **重要**：如使用手机，请使用**手机流量（4G/5G）**，切勿连接Wi-Fi，以防IP关联风险。
4.  **认证过程**：通话时间约5-15分钟。亚马逊工作人员会核实你的身份，并询问一些简单问题，如确认是否为本人、计划销售什么产品等。
5.  **等待审核**：视频认证结束后，等待亚马逊的审核结果通知即可。
