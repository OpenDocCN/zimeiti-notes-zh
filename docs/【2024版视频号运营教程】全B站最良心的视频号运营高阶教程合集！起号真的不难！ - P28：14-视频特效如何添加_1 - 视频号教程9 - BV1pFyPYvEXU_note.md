# 视频号运营教程：P28：14-视频特效如何添加_1

在本节课中，我们将学习如何在视频剪辑中添加特效。特效能够为视频增添氛围和趣味性，但使用不当也可能影响观感。我们将重点介绍如何正确地为视频主轨道和画中画轨道添加特效，并理解“作用对象”这一核心概念。

上一节我们介绍了画中画功能，本节中我们来看看如何为视频添加特效。

## 添加画面特效

在剪辑界面，点击“画中画”功能右侧的“特效”按钮，即可进入特效面板。特效主要分为“画面特效”和“人物特效”两类。

以下是添加一个简单画面特效的步骤：
1.  点击“画面特效”。
2.  在特效列表中选择一个效果，例如“星星灯”或“星星坠落”。
3.  点击特效将其添加到时间轴。
4.  调整特效的持续时长，使其与视频片段匹配。
5.  点击“对勾”确认添加。

此时，该特效默认作用于整个视频画面（即主视频轨道）。像“星星”这类氛围特效，添加在主画面上通常不会造成问题。

## 理解“作用对象”

然而，并非所有特效都适合直接添加在主视频上。例如，如果添加“模糊”特效到主视频，会导致整个画面变模糊，这通常不是我们想要的效果。

这时，就需要使用“作用对象”功能。其核心逻辑是：**指定特效生效的特定轨道**。

操作步骤如下：
1.  选中已添加的特效片段（如“模糊”特效）。
2.  在下方参数中找到“作用对象”选项。
3.  点击并选择“画中画”。
4.  点击“对勾”确认。

完成上述操作后，“模糊”特效将仅作用于画中画视频，而主视频画面则保持清晰。这个功能是精确控制特效范围的关键。

## 添加人物特效

人物特效的使用逻辑与画面特效完全一致。例如，添加“闪白”特效时，如果直接作用于主视频，人物会频繁闪烁，影响观看。

正确的做法是：
1.  添加“闪白”特效。
2.  通过“作用对象”选项，将其指定到“画中画”轨道。
3.  确认后，特效将仅影响画中画中的人物，主视频不受干扰。

对于新手，一个安全的做法是：**将添加的“画面特效”或“人物特效”的作用对象都设置为“画中画”**。这样可以确保主视频画面的干净与稳定。

## 谨慎使用氛围特效

特效库中还有“气氛”分类，例如“爱心飘荡”、“彩带”、“放烟花”等，适合用于视频结尾营造温馨或庆祝的氛围。

但是，这些特效通常比较显眼，如果使用不当或强度过高，会严重干扰视频主体内容。因此，对于初学者，建议在熟练掌握基础剪辑和“作用对象”功能前，暂时不要使用这类复杂的氛围特效。如果添加后效果不理想，可以选中特效片段直接删除。

本节课中我们一起学习了视频特效的添加方法。核心在于使用“作用对象”功能将特效精确应用到目标轨道（通常是画中画），以避免影响主视频画面。记住，对于初学者，从简单的特效开始，并确保主画面清晰不受干扰，是制作出整洁视频的关键。
