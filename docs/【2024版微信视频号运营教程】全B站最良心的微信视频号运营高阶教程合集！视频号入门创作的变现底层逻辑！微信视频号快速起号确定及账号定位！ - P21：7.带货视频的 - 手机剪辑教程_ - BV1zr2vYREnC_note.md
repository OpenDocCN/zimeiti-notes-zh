# 微信视频号运营教程：P21：7.带货视频的素材寻找与剪辑准备

## 概述
在本节课中，我们将学习如何为带货视频寻找合适的剧情素材和产品广告素材。这是制作带货视频前至关重要的准备步骤。

## 寻找带货视频的剧情模式
上一节我们介绍了直播前的准备工作，本节中我们来看看如何为带货视频准备素材。带货视频的核心模式是在特定剧情中嵌入产品广告。以下是三种主流且高效的剧情模式：

1.  **短剧解说加产品广告**：模式为 `短剧解说 + 产品广告`。选取一段短剧（如霸道总裁、古装等题材），在剧情合适的位置插入产品广告。
2.  **故事会加产品广告**：模式为 `故事会 + 产品广告`。采用讲述家庭、情感等生活故事的视频形式，同样在故事中嵌入产品广告。
3.  **情感语录对话框加产品广告**：模式为 `情感语录对话框 + 产品广告`。使用情感对话或语录的文本对话框形式作为视频主体，并加入产品广告。

这些模式可以适配现代、古装等多种题材，关键在于将产品自然融入剧情。

## 如何寻找剧情素材
掌握了视频模式后，接下来需要寻找具体的剧情素材。您可以通过以下平台和步骤进行搜索：

*   **搜索平台**：抖音、快手、小红书等短视频平台。
*   **搜索方法**：根据您想制作的视频类型，直接搜索对应的关键词。
    *   例如，想制作“故事会”类型，就搜索“故事会”。
    *   想制作“短剧解说”类型，就搜索“短剧解说”或“霸道总裁”、“婆媳”等更具体的关键词。
    *   想制作“情感语录”类型，就搜索“情感语录对话框”。

**重要补充：筛选优质剧情素材的要点**
在搜索时，建议应用以下筛选条件，以找到更具潜力的素材：
*   在搜索结果中，优先选择“最新发布”（例如一天内）的视频。
*   选择您自己未曾看过、且内容能吸引您愿意看下去的剧情。
*   如果发现同一个剧情被多个账号发布，可能意味着该内容正在热门，可以考虑选用。

## 如何寻找产品广告素材
找到剧情后，下一步是为您要带货的产品寻找广告素材。方法非常简单直接：

*   **搜索方法**：直接在短视频平台搜索您的产品名称。
    *   例如，如果您销售“保鲜膜套”，就直接搜索“保鲜膜套”。
    *   如果您销售“木耳”，就直接搜索“木耳”。
*   在搜索结果中选择“视频”分类，即可找到大量可供参考或使用的产品展示、功能介绍等广告素材。

## 总结
本节课我们一起学习了制作带货视频前的素材准备。核心内容包括：认识三种主流的带货视频模式（短剧解说、故事会、情感语录对话框加广告），掌握在抖音、快手等平台搜索剧情素材和产品广告素材的具体方法。记住，选择新颖、吸引人的剧情和清晰的产品素材是成功的第一步。下一节，我们将开始学习如何将这些素材剪辑成一个完整的带货视频。
