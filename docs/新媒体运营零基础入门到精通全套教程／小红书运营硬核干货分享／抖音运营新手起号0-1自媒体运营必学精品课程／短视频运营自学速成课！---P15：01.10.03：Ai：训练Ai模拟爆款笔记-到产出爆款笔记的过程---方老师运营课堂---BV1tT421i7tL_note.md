# AI辅助新媒体运营：P15：01.10.03：训练AI从分析到产出爆款笔记 📝

在本节课中，我们将学习如何训练AI（以ChatGPT为例）分析现有爆款笔记，并基于分析结果生成具备爆款潜质的新内容。我们将通过一个考研领域的实例，完整演示从“喂资料”到“出文案”的全过程。

上一节我们介绍了通过“喂资料”让AI理解特定平台风格和用语的方法。虽然上一节的旅游攻略案例效果不理想，但方法是正确的。本节中，我们来看看如何通过分析具体的爆款内容，让AI学习其成功要素，并最终产出新笔记。

## 分析爆款笔记的核心要点

![](img/0f9626e50538cf31ab2ad02fdfdd952d_3.png)

![](img/0f9626e50538cf31ab2ad02fdfdd952d_5.png)

![](img/0f9626e50538cf31ab2ad02fdfdd952d_7.png)

要让AI学习爆款，我们需要引导它分析几个关键维度。以下是需要AI重点分析的几个方面：

![](img/0f9626e50538cf31ab2ad02fdfdd952d_9.png)

![](img/0f9626e50538cf31ab2ad02fdfdd952d_11.png)

![](img/0f9626e50538cf31ab2ad02fdfdd952d_12.png)

![](img/0f9626e50538cf31ab2ad02fdfdd952d_14.png)

1.  **爆款因素分析**：分析内容为何能成为爆款，例如是否引发了情感共鸣、提供了实用价值、讲述了逆袭故事等。
2.  **内容结构分析**：拆解文案的框架。
    *   **开篇**：用什么方式（如提问、制造焦虑、引用名人）吸引用户继续阅读。
    *   **中间**：提供了哪些内容来戳中用户痛点，满足用户需求，从而提升用户停留时长。
    *   **结尾**：设置了什么“钩子”（如鼓励互动、引导私信、提供福利）来促使用户产生关注、评论或私信等行为。
3.  **关键词策略分析**：分析笔记中是否刻意植入了特定关键词。
    *   如果有，关键词是什么？
    *   关键词的出现频率和分布情况如何？
    *   是如何结合中文分词法来设计关键词的？

![](img/0f9626e50538cf31ab2ad02fdfdd952d_16.png)

## 实战演练：以考研领域为例

接下来，我们进入实战环节，以小红书平台的考研领域爆款笔记为例。

![](img/0f9626e50538cf31ab2ad02fdfdd952d_20.png)

### 第一步：收集并输入爆款笔记原文

![](img/0f9626e50538cf31ab2ad02fdfdd952d_22.png)

![](img/0f9626e50538cf31ab2ad02fdfdd952d_24.png)

首先，我们需要找到目标领域的爆款笔记原文。我们选取了几篇点赞量较高的考研备考笔记，将原文内容复制下来。

### 第二步：向AI下达分析指令

![](img/0f9626e50538cf31ab2ad02fdfdd952d_26.png)

![](img/0f9626e50538cf31ab2ad02fdfdd952d_28.png)

我们开启一个新的ChatGPT（3.5版本）对话，并给出明确的指令。指令模板如下：

![](img/0f9626e50538cf31ab2ad02fdfdd952d_30.png)

![](img/0f9626e50538cf31ab2ad02fdfdd952d_32.png)

![](img/0f9626e50538cf31ab2ad02fdfdd952d_34.png)

```
我给你提供一篇小红书平台上关于考研的爆款笔记文案，请你分析：
1.  这篇文案的爆款因素有哪些？（例如：情感共鸣、实用建议、逆袭故事等）
2.  分析文案的整体结构：
    *   开篇用什么吸引了用户？
    *   中间用什么内容戳中了用户痛点？
    *   结尾用什么方式促使用户互动或关注？
3.  分析该笔记是否结合中文分词法设计了关键词。
    *   如果没有，请告诉我“没有明显关键词”。
    *   如果有，请指出是哪个关键词，并说明其出现频率和分布情况。

这是文案内容：[这里粘贴爆款笔记原文]
```

### 第三步：整合分析结果，提炼爆款要素

在AI分析了多篇笔记后，我们需要让它对分析结果进行整合与去重，总结出该领域爆款笔记的共性要素。

![](img/0f9626e50538cf31ab2ad02fdfdd952d_53.png)

我们可以对AI说：“请将你前面分析过的几篇考研爆款笔记的爆款因素总结一下，整合成一个清晰的列表。”

![](img/0f9626e50538cf31ab2ad02fdfdd952d_55.png)

AI总结出的常见爆款因素包括：
*   情感共鸣（如备考焦虑、逆袭故事）
*   明确的备考计划与时间线
*   针对性的科目备考建议
*   具体的资源与技巧分享
*   时间管理与效率提升强调
*   正能量鼓励与心态调整
*   使用表情符号增强感染力

### 第四步：基于爆款要素，生成内容大纲

![](img/0f9626e50538cf31ab2ad02fdfdd952d_57.png)

有了爆款要素后，我们可以指令AI根据这些要素，为我们规划一篇新笔记的框架。

指令如下：“请结合你前面总结的爆款笔记因素，为我规划一篇关于考研的小红书笔记大纲。大纲需要细分到具体的小节点，在我确认大纲后，再生成完整文案。”

AI生成的大纲可能包含：
1.  **吸引人的开篇**：用问题或共鸣感引入。
2.  **主题介绍**：简要说明笔记价值。
3.  **逆袭故事引入**：分享个人经历，建立信任。
4.  **核心干货部分**：
    *   专业选择建议
    *   全年备考时间规划
    *   政治/英语/数学/专业课分科备考策略
5.  **心态与时间管理**：提供调整建议。
6.  **互动引导结尾**：鼓励提问、引导关注。

![](img/0f9626e50538cf31ab2ad02fdfdd952d_59.png)

![](img/0f9626e50538cf31ab2ad02fdfdd952d_60.png)

### 第五步：根据大纲，生成完整文案

确认大纲后，我们给出最终生成指令：“请按照你提供的大纲，撰写一篇完整的小红书文案。要求：严格遵循爆款笔记的结构；包含一个简短的逆袭小故事引发共鸣；文案中要使用emoji表情；并尝试结合中文分词法，自然地融入‘考研’、‘备考’等关键词。”

AI根据指令生成的文案，开头会营造共鸣感（如“我理解那种焦虑…”），中间会分点提供详细的备考建议和时间节点，结尾会进行鼓励并引导互动。虽然文案在细节和文采上可能还有优化空间，但其结构和要素已具备爆款笔记的基本特征，对于运营工作来说是一个高效的初稿。

## 进阶技巧与工具提示

![](img/0f9626e50538cf31ab2ad02fdfdd952d_62.png)

在操作中可能会遇到AI遗忘之前指令的情况（尤其在长对话中），此时需要重新强调核心要求。此外，为了获得更稳定、更个性化的输出，可以考虑使用高级工具：
*   **ChatGPT 4.0**：比3.5版本理解能力更强，生成的文案通常更优质。
*   **自定义GPTs**：可以创建专属的AI助手，通过上传小红书风格范例、专用表情包列表等资料，让它更深入地学习你的特定风格要求，实现“私人订制”般的文案生成。

本节课中我们一起学习了如何通过“分析爆款-总结要素-生成内容”的流程，训练AI辅助我们产出具备爆款潜质的小红书笔记。关键在于提供清晰的指令和高质量的“饲料”（爆款原文）。通过不断迭代和优化指令，你将能越来越高效地利用AI成为你的内容创作助手。
