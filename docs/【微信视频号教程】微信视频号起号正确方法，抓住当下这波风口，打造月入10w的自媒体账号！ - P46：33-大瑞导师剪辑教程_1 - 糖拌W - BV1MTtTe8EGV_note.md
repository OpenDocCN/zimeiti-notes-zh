# 视频剪辑教程：P46：33：基础剪辑流程与关键帧应用

在本节课中，我们将学习一个完整的短视频剪辑流程。我们将从广告片段衔接开始，逐步添加画中画、特效、滤镜、调节、背景、贴纸，并重点讲解如何为贴纸添加关键帧动画，最后完成视频的导出设置。整个过程旨在提升视频的丰富度与原创性。

## 广告片段衔接检查

首先，检查广告片段与前后视频内容的衔接是否流畅。具体操作是预览广告片段前后的画面与声音，确保过渡自然，没有突兀的跳转或音画不同步问题。

## 添加画中画

接下来，我们为视频添加画中画图层以丰富画面。

以下是添加和调整画中画的步骤：
1.  导入一个视频或图片作为画中画素材。
2.  将画中画素材放大至合适尺寸，并将其时长拖动至与主视频结尾对齐。
3.  调整画中画的**不透明度**，通常设置为 `1` 或 `2`，使其半透明地叠加在主画面上。
4.  如果画中画素材时长过短，可以通过**降低播放速度**来延长。例如，将速度调整为 `0.1`。
5.  延长后，记得在需要的位置对画中画进行**分割**，并删除多余部分。

## 添加视频特效

为视频添加特效可以增强视觉吸引力。

以下是添加特效的步骤：
1.  在特效菜单中选择一个非VIP的免费特效（例如“氛围”类特效）。
2.  点击确认后，将特效时长**铺满**整个视频。
3.  将特效的**作用对象**设置为**画中画**图层。
4.  用同样的方法，可以再添加一个**人物特效**（如“涂鸦”效果），并同样将其作用到画中画上。

## 调整滤镜与画面参数

为了使画面色调统一且更清晰，需要进行调色和细节调整。

以下是调节画面参数的步骤：
1.  添加一个**滤镜**，例如“自然”滤镜，并将强度调整到 `34` 左右，然后铺满全程。
2.  进入**调节**功能，对以下参数进行微调（数值变化尽量控制在 `1-2` 之间，避免画面失真）：
    *   **锐化**：稍微增加，让边缘更清晰。
    *   **曝光**：微调 `-1` 或 `+1`。
    *   **高光**：稍微增加 `+1` 或 `+2`。
    *   **色温**：降低 `-1` 或 `-2`，让画面偏冷。
    *   **色调**：同样降低 `-1`。

## 添加背景与贴纸

背景和贴纸能进一步装饰画面，增加趣味性。

以下是添加背景和贴纸的步骤：
1.  为视频**添加背景**，选择“模糊”或“纯色”样式，并将其不透明度调低，以免影响主画面。
2.  添加**贴纸**。选择与视频风格（如唯美、简约）相符的贴纸，例如小心心、小星星。
3.  将所有贴纸的时长都**铺满**整个视频。
4.  将每个贴纸放置在画面角落或边缘，确保它们**不影响**主体内容的观看。

## 应用关键帧动画

上一节我们添加了静态贴纸，本节中我们来看看如何让贴纸动起来，这就是关键帧动画。

关键帧可以记录贴纸在某个时间点的位置、大小等信息，通过在不同时间点设置不同的关键帧，软件会自动生成贴纸移动的动画。

以下是添加关键帧动画的步骤：
1.  新增一个贴纸（例如一朵小云彩），并将其铺满全程。
2.  将贴纸缩小，并放置在视频起始位置的屏幕一角。
3.  选中该贴纸，在时间轴上点击**添加关键帧**按钮。
4.  将时间线向后拖动一段距离，然后**移动贴纸的位置**，系统会自动在此处生成一个新的关键帧。
5.  重复步骤4，在不同的时间点继续移动贴纸位置，创建多个关键帧。
6.  让贴纸沿着屏幕边缘缓慢移动，避免在画面中央快速晃动影响观感。

通过预览，你可以看到贴纸按照你设定的路径慢慢移动。这就是利用关键帧创建简单动画的过程。

## 最终检查与导出

完成所有剪辑后，需要进行最终检查，确保每一项调整都已到位。

以下是最终检查与导出的步骤：
1.  按照从右到左（或你的习惯顺序）依次检查**调节**、**滤镜**、**贴纸**、**背景**、**特效**、**画中画**等所有项目是否已添加。
2.  确认无误后，进入导出设置。
3.  将**帧率**设置为 `60` fps，以获得更流畅的画面。
4.  将**分辨率**设置为 `1080P`。
5.  点击导出，等待渲染完成。

本节课中我们一起学习了从片段衔接到最终导出的完整剪辑流程。核心步骤包括添加画中画、特效、进行调色、装饰画面，并重点掌握了使用**关键帧**为贴纸添加动画效果的方法。记住，初期操作可能较慢，多加练习即可熟练。
