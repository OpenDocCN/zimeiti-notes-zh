# 抖音运营教程：P18：内容的三有原则

![](img/db7a7525b3a34e5ba90bd1b25fd38e8c_1.png)

在本节课中，我们将学习如何确定内容定位，并深入理解一个核心原则——“三有原则”。这个原则能帮助你判断什么样的内容更受用户喜爱，从而指导你创作出更具吸引力的作品。

在上一节探讨了内容定位的重要性后，我们回到一个核心问题：如何确定用户究竟喜欢什么样的作品？本节将通过分析具体案例，为你揭示其中的规律。

![](img/db7a7525b3a34e5ba90bd1b25fd38e8c_9.png)

以下是几个不同领域的视频案例，我们将逐一分析它们吸引人的关键点。

![](img/db7a7525b3a34e5ba90bd1b25fd38e8c_11.png)

![](img/db7a7525b3a34e5ba90bd1b25fd38e8c_13.png)

*   **案例一：钢琴教学**
    ![](img/db7a7525b3a7525b3a34e5ba90bd1b25fd38e8c_1.png)
    这是一个提供简易版《卡农》钢琴谱的教学视频。对于钢琴爱好者或初学者而言，这类内容提供了明确的**实用价值**。

![](img/db7a7525b3a34e5ba90bd1b25fd38e8c_15.png)

*   **案例二：宠物日常**
    ![](img/db7a7525b3a34e5ba90bd1b25fd38e8c_7.png)
    这是一只可爱小猫的日常视频。宠物内容天然能吸引**喜爱动物的人群**，激发他们的**兴趣**和情感互动。

*   **案例三：情感共鸣**
    ![](img/db7a7525b3a34e5ba90bd1b25fd38e8c_13.png)
    这个视频讲述了成年男性沉默背后的心理原因。其内容引发了特定观众群体的**情感共鸣**，让他们觉得“说到了心坎里”。

![](img/db7a7525b3a34e5ba90bd1b25fd38e8c_17.png)

通过以上案例，我们可以总结出一个全网通用的内容创作核心原则。

这个原则被称为 **“三有原则”** 。它指出，受欢迎的内容通常至少满足以下三点之一：
1.  **有用处**：提供知识、技能、攻略等实用价值。例如：教程、测评、推荐。
2.  **有兴趣**：内容本身具有吸引力，能引发观众的好奇或喜爱。例如：高颜值、萌宠、炫技、搞笑剧情。
3.  **有共鸣**：触动观众的情绪，引发情感上的认同感。例如：励志故事、情感鸡汤、生活感悟。

用公式可以表示为：
**受欢迎内容 ≈ 有用处 ∪ 有兴趣 ∪ 有共鸣**

理解“三有原则”后，你可能会思考：内容类型如此之多，我该如何选择？别急，在确定具体方向前，我们还需要理解另一个关键概念——“垂直度”。

为了说明“垂直度”的重要性，我们来看一个案例。

![](img/db7a7525b3a34e5ba90bd1b25fd38e8c_19.png)
这个唱歌视频的拍摄和演绎质量都不错，也获得了一定的点赞（5.8万）。但其账号粉丝数却很低（仅5173），**点赞粉丝比（赞粉比）异常**。

**赞粉比 = 总获赞数 / 总粉丝数**
这个比值过高，通常意味着视频内容虽然偶有亮点吸引点赞，但不足以让用户产生“持续关注”的欲望。

可能的原因是什么？账号主页显示其内容均为唱歌，领域似乎是垂直的。这里就引出了两个更深层的问题：**主题IP的吸引力和账号整体搭建**。

“垂直”指的是账号内容长期聚焦于**同一特定领域**。例如，只做美食教程、只做游戏解说、只做美妆分享。这样做的好处是，当用户因为某个视频对你产生兴趣时，他能明确预期你的其他内容也符合他的喜好，从而更有关注的动力。

除非创作者本人具有极强的个人魅力（IP），否则对于大多数普通账号而言，保持内容垂直是积累精准粉丝的基础。下一节，我们将进一步探讨如何解决“有播放、不涨粉”的问题，即如何打造有吸引力的主题IP和进行有效的账号搭建。

---

本节课中我们一起学习了：
1.  判断内容吸引力的 **“三有原则”**：**有用处、有兴趣、有共鸣**。
2.  内容**垂直度**的概念及其对涨粉的重要性。
3.  通过**赞粉比**可以初步诊断账号的粉丝转化效率。

![](img/db7a7525b3a34e5ba90bd1b25fd38e8c_25.png)

掌握“三有原则”是创作优质内容的起点，而保持垂直则是让内容价值持续积累、吸引精准粉丝的关键。在接下来的课程中，我们将具体学习如何应用这些原则来规划和优化你的账号内容。
