### 1. 角色：明确你的专业身份

![](img/f47dcdfb603643c3f755671fc94173a5_70.png)

![](img/f47dcdfb603643c3f755671fc94173a5_72.png)

![](img/f47dcdfb603643c3f755671fc94173a5_74.png)

![](img/f47dcdfb603643c3f755671fc94173a5_76.png)

角色定位的核心在于，你需要根据自己选择的赛道和内容方向，塑造一个具有专业性的身份。这个身份能让用户相信你提供的内容是有价值的。

![](img/f47dcdfb603643c3f755671fc94173a5_78.png)

![](img/f47dcdfb603643c3f755671fc94173a5_80.png)

![](img/f47dcdfb603643c3f755671fc94173a5_82.png)

![](img/f47dcdfb603643c3f755671fc94173a5_84.png)

以下是不同赛道的角色定位示例：

![](img/f47dcdfb603643c3f755671fc94173a5_86.png)

![](img/f47dcdfb603643c3f755671fc94173a5_88.png)

![](img/f47dcdfb603643c3f755671fc94173a5_90.png)

![](img/f47dcdfb603643c3f755671fc94173a5_92.png)

*   **装修博主**：你需要展示自己真实的装修过程、遇到的坑以及最终成果。公式可以概括为：**真实经历 + 过程记录 + 成果对比**。
*   **母婴赛道**：最常见的角色是“宝妈”。但你也可以定位为“护理专家”，不一定必须是宝妈本人，可以是相关领域的专业人员。角色公式为：**专业身份（如宝妈/专家） + 细分领域知识**。
*   **情感赛道**：可以定位为“心理学顾问”或“情感大师”，分享情感、成长、励志类内容。另一种是“成长型人设”，分享个人在家庭、职场、创业中的真实经历和蜕变。角色公式为：**专业头衔/个人经历 + 解决方案/成长故事**。

![](img/f47dcdfb603643c3f755671fc94173a5_94.png)

![](img/f47dcdfb603643c3f755671fc94173a5_96.png)

![](img/f47dcdfb603643c3f755671fc94173a5_98.png)

![](img/f47dcdfb603643c3f755671fc94173a5_100.png)

**总结**：你提供什么赛道的内容，就把自己塑造成该领域的专家或深度体验者。

### 2. 内容：保持垂直与价值

![](img/f47dcdfb603643c3f755671fc94173a5_112.png)

![](img/f47dcdfb603643c3f755671fc94173a5_114.png)

![](img/f47dcdfb603643c3f755671fc94173a5_116.png)

![](img/f47dcdfb603643c3f755671fc94173a5_118.png)

内容是人设的基石。关于内容定位，我们需要确保内容对用户有吸引力、有价值，并且在平台上有特色。在人设定位中，内容必须与你的赛道高度相关，并持续输出。

![](img/f47dcdfb603643c3f755671fc94173a5_120.png)

![](img/f47dcdfb603643c3f755671fc94173a5_122.png)

![](img/f47dcdfb603643c3f755671fc94173a5_124.png)

![](img/f47dcdfb603643c3f755671fc94173a5_126.png)

这一点至关重要，因为它关系到账号的**垂直度**和**标签**。平台算法会根据你的内容标签将视频推荐给可能感兴趣的用户。如果你的内容杂乱无章（例如第一篇讲育儿，第二篇讲宠物），会导致推荐不精准，用户可能因不感兴趣而划走，从而影响视频的完播率、点赞等数据，最终损害账号的流量推荐。

![](img/f47dcdfb603643c3f755671fc94173a5_128.png)

![](img/f47dcdfb603643c3f755671fc94173a5_130.png)

![](img/f47dcdfb603643c3f755671fc94173a5_132.png)

![](img/f47dcdfb603643c3f755671fc94173a5_134.png)

**核心原则**：**`内容方向 = 赛道领域`**。你选择什么赛道，就持续提供该领域的相关内容。

### 3. 形式：确定内容的呈现方式

![](img/f47dcdfb603643c3f755671fc94173a5_144.png)

![](img/f47dcdfb603643c3f755671fc94173a5_146.png)

![](img/f47dcdfb603643c3f755671fc94173a5_148.png)

![](img/f47dcdfb603643c3f755671fc94173a5_150.png)

形式指的是你通过何种具体方式将内容呈现给用户。主要包括以下几个方面：

![](img/f47dcdfb603643c3f755671fc94173a5_152.png)

![](img/f47dcdfb603643c3f755671fc94173a5_154.png)

![](img/f47dcdfb603643c3f755671fc94173a5_156.png)

![](img/f47dcdfb603643c3f755671fc94173a5_158.png)

![](img/f47dcdfb603643c3f755671fc94173a5_160.png)

*   **产出场景**：你的拍摄场景是固定的（如室内、车内）还是多变的（如室外、不同生活工作场景）？固定的场景有助于形成视觉记忆点。
*   **出镜与声音**：
    *   **真人出镜**：是否本人出镜讲解？
    *   **声音处理**：使用原声还是配音？现在利用`剪映`等工具进行配音已非常方便。
*   **内容形式**：你的内容主要以哪种形式呈现？
    *   **剧情型**：有故事线和表演，创作难度较高。
    *   **干货/经验分享型**：需要精心准备文案脚本，输出有价值的信息。
    *   **生活记录型**：需要积累大量的日常拍摄素材。

![](img/f47dcdfb603643c3f755671fc94173a5_162.png)

![](img/f47dcdfb603643c3f755671fc94173a5_164.png)

![](img/f47dcdfb603643c3f755671fc94173a5_168.png)

### 4. 记忆点：让用户牢牢记住你

![](img/f47dcdfb603643c3f755671fc94173a5_178.png)

![](img/f47dcdfb603643c3f755671fc94173a5_180.png)

![](img/f47dcdfb603643c3f755671fc94173a5_182.png)

![](img/f47dcdfb603643c3f755671fc94173a5_184.png)

记忆点是让人设脱颖而出的关键。你需要设计一些固定元素，反复强化，刻入用户心智。当用户看到这些元素时，能立刻联想到你。

![](img/f47dcdfb603643c3f755671fc94173a5_186.png)

![](img/f47dcdfb603643c3f755671fc94173a5_188.png)

![](img/f47dcdfb603643c3f755671fc94173a5_190.png)

![](img/f47dcdfb603643c3f755671fc94173a5_192.png)

以下是常见的记忆点设计，可分为两类：

![](img/f47dcdfb603643c3f755671fc94173a5_194.png)

![](img/f47dcdfb603643c3f755671fc94173a5_196.png)

![](img/f47dcdfb603643c3f755671fc94173a5_198.png)

![](img/f47dcdfb603643c3f755671fc94173a5_200.png)

**A. 前置强化型（用于视频开头）**：
*   **固定动作**：视频前几秒的特定手势或行为。
*   **特色特效**：开场的固定视觉特效。
*   **标志性开场白**：每期视频使用的第一句话。

![](img/f47dcdfb603643c3f755671fc94173a5_202.png)

![](img/f47dcdfb603643c3f755671fc94173a5_204.png)

![](img/f47dcdfb603643c3f755671fc94173a5_206.png)

![](img/f47dcdfb603643c3f755671fc94173a5_208.png)

**设计逻辑**：将这些记忆点放在视频开头，当用户浏览你的主页或连续刷到你的视频时，能形成强烈的重复冲击，快速加深印象。公式为：**`记忆点植入 = 视频开头固定动作/特效/台词`**。

![](img/f47dcdfb603643c3f755671fc94173a5_210.png)

![](img/f47dcdfb603643c3f755671fc94173a5_212.png)

![](img/f47dcdfb603643c3f755671fc94173a5_214.png)

![](img/f47dcdfb603643c3f755671fc94173a5_216.png)

![](img/f47dcdfb603643c3f755671fc94173a5_218.png)

**B. 全局贯穿型（用于整个视频）**：
*   **统一BGM**：使用固定的背景音乐，音乐一响，用户就知道是你。
*   **标志性物品**：画面中始终出现的某个物品（如特定的台灯、摆设）。
*   **风格化服装**：符合账号调性的统一着装风格（如田园风、职场风）。

![](img/f47dcdfb603643c3f755671fc94173a5_220.png)

![](img/f47dcdfb603643c3f755671fc94173a5_222.png)

![](img/f47dcdfb603643c3f755671fc94173a5_224.png)

![](img/f47dcdfb603643c3f755671fc94173a5_226.png)

**设计逻辑**：这些元素贯穿视频始终，营造整体氛围和辨识度。公式为：**`记忆点强化 = 统一BGM/物品/服装风格`**。

本节课我们一起学习了人设定位的四大核心因素：
1.  **角色**：根据赛道确立专业或体验者身份。
2.  **内容**：坚持垂直领域，输出有价值的内容以打牢账号标签。
3.  **形式**：确定拍摄场景、出镜方式及内容表现形式（剧情、干货、记录）。
4.  **记忆点**：通过设计前置（如固定开场）和全局（如统一BGM）的固定元素，强化用户印象。
