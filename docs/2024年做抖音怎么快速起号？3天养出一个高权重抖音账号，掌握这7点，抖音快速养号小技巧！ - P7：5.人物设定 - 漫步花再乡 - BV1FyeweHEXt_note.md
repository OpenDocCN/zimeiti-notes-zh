# 抖音运营入门：P7：人物设定策划 🎭

![](img/358c02ec481d0d568fd2d990a349f997_0.png)

在本节课中，我们将要学习抖音账号运营中一个至关重要的环节——**人物设定策划**。一个清晰、真实且有吸引力的人设，是账号获取流量、建立信任并最终实现变现的基石。

![](img/358c02ec481d0d568fd2d990a349f997_2.png)

上一节我们介绍了账号定位，本节中我们来看看如何将定位具体化为一个鲜活的人物角色。

![](img/358c02ec481d0d568fd2d990a349f997_4.png)

## 人设策划的核心概念

![](img/358c02ec481d0d568fd2d990a349f997_6.png)

在抖音中策划人设，就像在游戏中创建角色一样，需要提前对人物进行设定。例如，设定其称呼、职业、技能点等。但与游戏角色不同的是，我们在抖音中设定的是**人生角色**，因此需要更具真实性。

![](img/358c02ec481d0d568fd2d990a349f997_8.png)

无论是短视频还是直播，一个成功的IP能够解决粉丝信任问题，起到裂变传播的作用，并具备很强的变现能力。因此，我们运营账号，无论是创作内容还是策划选题，目的都是在获取流量的同时**打造IP**。

![](img/358c02ec481d0d568fd2d990a349f997_10.png)

打造IP的核心，是在特定的场景或情境中，进行**账号人格化**的过程。这个过程需要注入情感、塑造故事、并贴上具有辨识度的标签。

![](img/358c02ec481d0d568fd2d990a349f997_12.png)

## 账号人格化：人设策划的六个维度

![](img/358c02ec481d0d568fd2d990a349f997_14.png)

账号人格化，即我们通常所说的“账号人设”。对于人设的策划，可以从以下六个方面着手。以下是具体维度：

![](img/358c02ec481d0d568fd2d990a349f997_16.png)

1.  **名字/姓氏/昵称**
    *   一个IP首先需要一个叫法，这是其传播的名片。它就像一个品牌的商标名一样重要，需要明确IP叫什么。

![](img/358c02ec481d0d568fd2d990a349f997_18.png)

2.  **职业/职务/职称**
    *   人格化必然要具备身份。这是用户认识这个IP的联想入口，例如：老师、教授、博士、律师等。

3.  **称呼/社会关系**
    *   一个IP有了特定的称呼，能快速拉近与用户的距离，起到移情作用。例如：XX哥、XX大爷、XX姥姥。

4.  **形象/第一印象**
    *   以用户的视角看待IP，使IP形象化，更容易占据用户心智。例如：大叔形象、萌萌哒、铁憨憨、甜美可爱、光头等。

5.  **性格/爱好/特点**
    *   一个具有鲜明特点的IP，除了记忆点强，也更具真实性。过于完美的形象容易脱离大众。例如：暴躁、娘炮、爱吃、爱喝酒等。

6.  **穿着打扮/财富地位**
    *   这属于社会刻板印象和大众普遍认知。借助用户的固有思维，可以快速为IP在用户心智中定位。例如：有钱代表会赚钱，开豪车代表成功人士，打扮土气可能代表老师。

## 从策划到落地

以上六个维度的内容策划好后，就可以结合你自身的产品优势、服务优势等，将其落实到**账号搭建**中，即抖音账号的“五件套”：**头像、昵称、背景图、简介、视频封面**。

除了对账号IP进行人格化策划，还需要通过选题和内容策划，为你的人设提炼出**记忆点**，即通常所说的“超级符号”。

## 提炼你的“超级符号”

超级符号是让人设脱颖而出的关键记忆点。以下是提炼超级符号的几个方向：

*   **画风（视觉）**：如独特的场景、道具、服饰。
*   **听觉**：如标志性的BGM、特效音、配音声线特点。
*   **文案**：能够反映优势且容易记忆的提炼词，例如：“XX界天花板”、“简单粗暴”。
*   **表情/情绪**：如标志性的笑、惊恐、无奈等表情包式情绪传递。
*   **语言特色**：如方言、口音、错误发音、网络用语、慢条斯理或抑扬顿挫的说话方式。
*   **动作/习惯**：如习惯性的撩头发、扶眼镜等肢体动作。

超级符号提炼好之后，在进行短视频创作时，就可以不断地在视频中重复呈现这些元素，强化用户记忆。

本节课中我们一起学习了抖音账号**人物设定策划**的完整流程。我们从理解人设的重要性开始，学习了从六个维度构建人格化IP的方法，并探讨了如何将策划落地到账号搭建中，最后介绍了通过提炼“超级符号”来强化人设记忆点的技巧。一个精心策划且真实可信的人设，是你在抖音海洋中扬帆起航的坚固船帆。
