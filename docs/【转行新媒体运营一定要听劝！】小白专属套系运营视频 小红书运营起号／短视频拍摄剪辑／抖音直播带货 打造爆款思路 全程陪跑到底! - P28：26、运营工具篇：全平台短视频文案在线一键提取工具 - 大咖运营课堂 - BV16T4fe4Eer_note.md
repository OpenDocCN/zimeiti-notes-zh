## 实战演示：提取抖音文案

让我们通过一个实际例子来演示如何使用。首先，我们在抖音找到一个目标视频，例如一个关于“情感语录”的短视频。

![](img/0c80406008636cf55bd963cf98c34e34_11.png)

1.  在抖音APP中，点击视频分享按钮，选择“复制链接”。
2.  打开文案提取工具的网页界面。
3.  将复制好的链接粘贴到工具的输入框中。
4.  点击“提取文案”按钮。

![](img/0c80406008636cf55bd963cf98c34e34_13.png)

稍等片刻（通常只需几秒钟），工具就会将视频中的语音识别并转换成文字显示出来。例如，可能得到如下文案：
> “听完你就释怀了，村上春树说，我动了离开你的念头，不是因为你不好，也不是因为不爱了……”

我们可以直接复制这段文案，或使用“格式化”功能让其更易读。
