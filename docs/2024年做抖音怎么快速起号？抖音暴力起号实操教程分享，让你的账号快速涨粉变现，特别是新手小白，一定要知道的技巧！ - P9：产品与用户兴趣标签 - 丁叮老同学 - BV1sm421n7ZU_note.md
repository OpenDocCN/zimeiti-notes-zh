# 抖音运营实战：P9：产品与用户兴趣标签

在本节课中，我们将学习一种名为“九宫格选题法”的实用技巧。该方法通过系统性地分析产品属性和用户兴趣，帮助创作者高效生成源源不断的视频选题，解决“不知道拍什么”的难题。

上一节我们介绍了选题的重要性，本节中我们来看看如何具体操作。我们将以健身软件“Keep”为例，详细拆解如何运用九宫格进行选题。

## 第一步：构建产品标签九宫格

首先，我们需要分析产品本身的特点。将核心产品（例如Keep）置于九宫格的中心，然后围绕它，在周围的八个格子中填入你能想到的所有产品相关标签。

以下是具体操作步骤：

1.  **确定核心**：将你的核心产品或服务（本例为“Keep”）放在九宫格中央。
2.  **联想填充**：围绕中心，发散思维，填写该产品的所有相关属性和特点。

以Keep为例，我们可以填充以下产品标签：

*   **线上健身软件**：这是产品的核心形态。
*   **专业健身教练**：产品提供的重要服务或资源。
*   **饮食搭配建议**：产品提供的附加价值。
*   **减肥/塑形攻略**：产品解决的核心用户需求。
*   **健身干货知识**：产品提供的教育内容。
*   **科学运动方法**：产品的专业性和可信度支撑。
*   **打卡运动制度**：产品的用户激励和互动机制。
*   **一二线城市用户为主**：产品的核心用户画像或市场定位。

完成这一步后，产品侧的标签就填充完毕了。这个步骤帮助我们全面梳理了产品能提供什么。

## 第二步：构建用户兴趣标签九宫格

接下来，我们转向用户分析。将核心用户画像（例如“健身爱好者”）置于另一个九宫格的中心，然后围绕它，联想这类用户可能感兴趣的一切事物。

以下是具体操作步骤：

1.  **确定用户**：将你的目标用户群体（本例为“健身爱好者”）放在九宫格中央。
2.  **兴趣发散**：尽可能广泛地联想该用户群体可能拥有的其他兴趣爱好。

![](img/4fe68f1ec985c673c4b4ada19c7d255d_1.png)

对于“健身爱好者”，其兴趣标签可能包括：

*   **健身房**：线下健身的场所，是兴趣的自然延伸。
*   **健身装备**：如运动鞋、紧身衣等，与核心活动强相关。
*   **轻脂美食/减肥餐**：对饮食有管理需求，关注健康饮食。
*   **旅行**：热爱生活的人可能喜欢的活动。
*   **星座/明星八卦**：大众化的娱乐兴趣点。
*   **励志鸡汤**：追求自我提升的人可能关注的内容。
*   **时尚穿搭**：对自身形象有要求。
*   **生活热爱者**：将用户画像从单一标签扩展为更立体的人格。

完成这两个九宫格后，我们就拥有了两大素材库：一个是**产品特性库**，一个是**用户兴趣库**。

![](img/4fe68f1ec985c673c4b4ada19c7d255d_1.png)

## 第三步：进行“连连看”创意组合

现在，我们进入最关键的创意生成阶段。将第一个九宫格（产品标签）和第二个九宫格（用户兴趣标签）中的词汇进行随机或有意地连接组合，从而碰撞出新的选题灵感。

连接的基本公式是：
**选题 ≈ 产品标签A + 用户兴趣标签B**

以下是几个组合示例：

*   **示例1**：`饮食搭配`（产品标签） + `旅行`（用户兴趣标签）
    *   **生成选题**：《外出旅行时，如何享受美食又能控制体重？》
*   **示例2**：`运动方法`（产品标签） + `明星`（用户兴趣标签）
    *   **生成选题**：《赵丽颖都在练的5个高效燃脂动作，赶紧学起来！》
*   **示例3**：`科学`（产品标签） + `健身房`（用户兴趣标签）
    *   **生成选题**：《别再瞎练了！用这3个科学方法，让健身房效率翻倍。》

通过这种“连连看”的方式，理论上可以生成海量选题（8x8=64种基础组合，每种组合还可衍生不同角度）。这能有效解决灵感枯竭的问题。

## 第四步：优化与拓展选题

然而，并非所有组合都能直接产出优质选题。当思维受限时，可以借助搜索工具进行拓展。

具体方法是：将组合出的关键词（如“健身 科学”）输入搜索引擎或内容平台，查看相关的搜索联想和热门内容，从中获得句式、角度上的启发，进一步优化你的选题。

本节课中我们一起学习了“九宫格选题法”。我们首先分别围绕**产品属性**和**用户兴趣**构建了两个九宫格标签库，然后通过将两者词汇进行“连连看”式的交叉组合，批量生成视频选题创意。这个方法的核心在于**打破思维定式**，将产品卖点与用户泛兴趣结合，从而创造出既有相关性又能破圈吸引目标用户的丰富内容。坚持使用此法，你将能建立起自己的选题素材库，实现内容的持续产出。
