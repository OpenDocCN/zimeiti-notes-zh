# 小红书运营教程：P34：抠像处理 🎬

在本节课中，我们将学习如何使用剪映软件对视频进行抠像处理。抠像是视频剪辑中一项重要的技术，能够帮助我们去除视频的背景，将主体分离出来，以便进行更复杂的合成与创作。

## 抠像前的准备

![](img/ee3ac9b52eff8c7c1c36fa4628039771_1.png)

进行抠像处理前，首先需要确保视频素材的背景是纯色或近似纯色的。常见的背景色有绿色（绿幕）或蓝色（蓝幕），这有助于软件更精确地识别和分离背景与主体。

选中需要处理的视频素材后，在右侧的**参数调节面板**中找到“抠像”选项。剪映提供了三种抠像工具：**色度抠图**、**自定义抠像**和**智能抠像**。

![](img/ee3ac9b52eff8c7c1c36fa4628039771_3.png)

## 色度抠图

![](img/ee3ac9b52eff8c7c1c36fa4628039771_5.png)

![](img/ee3ac9b52eff8c7c1c36fa4628039771_7.png)

色度抠图适用于背景颜色单一且与主体颜色差异较大的情况。以下是使用色度抠图的步骤：

![](img/ee3ac9b52eff8c7c1c36fa4628039771_9.png)

1.  在“抠像”选项中选择“色度抠图”。
2.  点击**取色器**（吸管工具），然后在视频预览画面中点击你想要去除的背景颜色。例如，如果背景是绿色，就点击绿色区域。
3.  选取颜色后，调整**强度**滑块。强度值越大，被抠除的背景范围就越大。调整时需要观察预览画面，避免强度过高导致主体部分也被误删。
    ![](img/ee3ac9b52eff8c7c1c36fa4628039771_1.png)
4.  接着可以调整**阴影**滑块。这个参数可以为抠出的人物边缘添加阴影效果，使其与新的背景融合得更自然、更具立体感。
    ![](img/ee3ac9b52eff8c7c1c36fa4628039771_3.png)

为了更好地演示，我们可以从剪映的“素材库”中导入一个绿幕素材。
![](img/ee3ac9b52eff8c7c1c36fa4628039771_5.png)
![](img/ee3ac9b52eff8c7c1c36fa4628039771_7.png)

选中绿幕素材，使用色度抠图工具吸取绿色背景，并微调强度和阴影参数，直到人物被干净地分离出来，边缘过渡自然。
![](img/ee3ac9b52eff8c7c1c36fa4628039771_9.png)

## 智能抠像

![](img/ee3ac9b52eff8c7c1c36fa4628039771_11.png)

上一节我们介绍了手动选取颜色的色度抠图，本节中我们来看看更自动化的“智能抠像”。

智能抠像由剪映的算法自动识别并抠出画面中的主体。需要注意的是，此功能目前**仅支持对人物图像**进行智能识别，并且需要保持网络连接才能使用。

![](img/ee3ac9b52eff8c7c1c36fa4628039771_13.png)

操作非常简单：选中人物素材后，在“抠像”选项中选择“智能抠像”，软件便会自动处理。
![](img/ee3ac9b52eff8c7c1c36fa4628039771_11.png)

对于人像素材，智能抠像通常能获得比较干净的效果，效率很高。

## 自定义抠像

当素材背景复杂，或者需要抠取非人物对象（如物品、动物）时，色度抠图和智能抠像可能无法满足需求。这时就需要使用“自定义抠像”工具进行精细的手动处理。

以下是自定义抠像的操作流程：

1.  在“抠像”选项中选择“自定义抠像”。
    ![](img/ee3ac9b52eff8c7c1c36fa4628039771_13.png)
2.  你会看到**智能画笔**、**智能橡皮**和**橡皮擦**工具。
3.  首先使用**智能画笔**涂抹你想要保留的主体部分。软件会尝试自动识别并选中你涂抹的区域，被选中的区域会显示为墨绿色。
4.  如果软件自动选中的范围不准确，有多余或缺失的部分，可以使用**智能橡皮**擦除多余选区，或再次使用**智能画笔**补全缺失部分。
5.  你可以通过按住 `Ctrl` 键并滚动鼠标滚轮来放大画面，以便进行更精细的涂抹和擦除。
6.  处理毛发、镂空等复杂边缘时需要格外耐心和仔细。
7.  初步勾勒完成后，点击“应用效果”。如果发现仍有瑕疵，可以继续使用智能橡皮和智能画笔进行微调，直到满意为止。
    ![](img/ee3ac9b52eff8c7c1c36fa4628039771_15.png)

![](img/ee3ac9b52eff8c7c1c36fa4628039771_15.png)

自定义抠像虽然操作上比前两种方法更复杂、耗时，但它提供了最高的灵活性和控制精度，是处理复杂抠像任务的必备工具。

---

本节课中我们一起学习了剪映的三种抠像方法：**色度抠图**、**智能抠像**和**自定义抠像**。色度抠图适合纯色背景，智能抠图能快速处理人像，而自定义抠像则提供了最大的灵活性来处理复杂场景。在实际操作中，你可以根据素材的具体情况选择最合适的工具，或结合使用以达到最佳效果。
