## 支付方式设置 💳

![](img/d126e75abc8a1389ec8cd49a1408453e_5.png)

上一节我们介绍了开通小店，本节中我们来看看开通后首先要配置的支付功能。

![](img/d126e75abc8a1389ec8cd49a1408453e_7.png)

进入小店后台，找到“店铺的支付方式”设置项。以下是必须完成的步骤：

*   **操作**：将页面中展示的四个支付功能全部开启。
*   **目的**：只有全部开通后，后期才能正常将货款提现到您的账户（无论是对公账户还是个人账户）。

![](img/d126e75abc8a1389ec8cd49a1408453e_9.png)

## 商家保障中心配置 🛡️

完成支付设置后，我们进入“商家保障中心”进行关键功能配置。

![](img/d126e75abc8a1389ec8cd49a1408453e_13.png)

### 1. 开通运费险

![](img/d126e75abc8a1389ec8cd49a1408453e_15.png)

运费险是一项重要的消费者保障服务，开通后平台会给予一定的流量倾斜。其费用通常很低，仅需几毛钱。

开通运费险主要有以下好处：

*   **增加曝光**：可提升商品约 **19%** 的曝光量。
*   **自动理赔**：消费者退货时，系统会自动完成理赔流程。
*   **促进转化**：打消顾客下单顾虑，提升转化率。
*   **减少纠纷**：降低因退货问题产生的客户投诉。

### 2. 设置急速退款

在商家保障中心首页下滑，找到“急速退款的助手”并点击进入。

*   **操作**：开启“未发货仅退款”功能。
*   **规则**：系统会自动识别同时满足以下两个条件的订单：
    1.  订单金额 **< 300元**。
    2.  下单时间在 **6小时** 内且尚未发货。
    对于此类订单，若有客户申请退款，系统将执行急速退款。“退货退款”选项此处无需勾选。

![](img/d126e75abc8a1389ec8cd49a1408453e_17.png)

### 3. 加入安心购

继续在“商家保障中心”下滑，找到“安心购”服务。

![](img/d126e75abc8a1389ec8cd49a1408453e_19.png)

*   **前提**：需先开通其要求的各项子功能（页面会显示“已开通”或“未开通”状态）。
*   **操作**：逐一开通所有要求的功能，然后加入安心购。
*   **好处**：商品详情页会显示“安心购”标识，有助于**提高订单转化率、成交率**，并能**提升商家体验分**。

## 物流与地址管理 📦

保障服务配置妥当后，接下来设置物流相关的核心信息。

### 1. 管理地址库

进入“地址库管理”，添加默认的发货地址和退货地址。这对于做无货源的商家尤为重要。

*   **背景**：平台近期推出“闪电退货”规则，用户退货时，系统会自动发送地址库中设置的**默认退货地址**。
*   **操作**：
    1.  提前与您的上家（供应商）对接，获取准确的退货地址并填写在此处。
    2.  若供应商不提供退货地址，可采取变通方法：
        *   随意填写一个地址。
        *   在“详细地址”栏中，编辑提示话术，例如：
        ```
        退货请联系客服。
        该地址不作为退货地址，请勿直接退回。
        ```
    3.  将该地址同时设为“默认发货地址”和“默认退货地址”。

![](img/d126e75abc8a1389ec8cd49a1408453e_21.png)

### 2. 创建运费模板

新店铺只有一个“全国包邮”的默认模板且不可修改。我们需要新建自定义模板。

![](img/d126e75abc8a1389ec8cd49a1408453e_23.png)

点击“新建模板”，按以下步骤操作：

*   **模板名称**：输入自定义名称，如“部分地区包邮”。
*   **发货地址**：选择或填写您的实际发货地。
*   **运费设置**：有三种计费方式，建议选择 **阶梯计价**。
    *   **阶梯计价**：按件数或重量分段计算运费。
        *   **示例（按件数）**：
        ```
        首件（1件内）：0元
        每增加1件：增加0元
        ```
    *   **固定运费**：无论购买多少，运费固定为一个金额。
    *   **卖家包邮**：所有订单均由卖家承担运费。
*   **指定地区运费**：对于新疆、西藏、内蒙古等偏远地区，可以单独设置较高的运费。
    1.  点击“添加地区”，选择所需省份/地区。
    2.  为其设置独立的运费规则，例如：
        ```
        首件（1件内）：30元
        每增加1件：增加10元
        ```
*   **其他设置**：
    *   **指定条件包邮**：可设置满X件或满X元包邮（如果基础设置已是包邮，此项通常不设）。
    *   **限制下单区域**：可将因疫情等原因无法配送的地区添加进来，这些地区的用户将无法下单。
*   **保存**：完成所有设置后，点击保存。之后在运费模板列表即可查看新建的模板及其被多少商品使用。

## 总结 📝

本节课我们一起学习了抖音小店的基础搭建流程。我们依次完成了：
1.  **支付方式**的全面开通，确保资金正常提现。
2.  **商家保障中心**的核心功能配置，包括开通**运费险**获取流量、设置**急速退款**提升体验、加入**安心购**促进转化。
3.  **物流与地址管理**，设置默认地址以应对“闪电退货”规则，并创建了符合自身需求的**自定义运费模板**。

![](img/d126e75abc8a1389ec8cd49a1408453e_25.png)

这些步骤是店铺正常运营和获得平台基础流量的基石，请务必认真完成。下节课，我们将深入讲解“闪电退货”策略，学习如何创建多地址退货方案。
