# 抖音运营实战：P15：5.视频剪辑实操（上） 📱✂️

![](img/ec985ce96982e571fadbc9588919f33d_0.png)

在本节课中，我们将学习如何使用手机版剪映App进行视频剪辑的基础操作。我们将从确定视频主题开始，逐步完成添加素材、制作标题背景、添加文字与配音，以及调整视频比例等核心步骤。本教程专为初学者设计，力求简单明了。

---

![](img/ec985ce96982e571fadbc9588919f33d_2.png)

上一节我们介绍了视频策划的思路，本节中我们来看看如何动手将想法变成实际的视频。我们将使用手机上的剪映App进行操作，因为手机剪辑更加便捷，适合随时随地创作。

首先，我们需要在手机上下载并安装剪映App。它的电脑端和手机端图标一致。

![](img/ec985ce96982e571fadbc9588919f33d_4.png)

![](img/ec985ce96982e571fadbc9588919f33d_0.png)

安装完成后，打开剪映App。

![](img/ec985ce96982e571fadbc9588919f33d_6.png)

![](img/ec985ce96982e571fadbc9588919f33d_4.png)

进入应用后，点击首页的“**+ 开始创作**”按钮。

![](img/ec985ce96982e571fadbc9588919f33d_8.png)

---

![](img/ec985ce96982e571fadbc9588919f33d_10.png)

在开始剪辑前，必须先确定视频的主题。这决定了后续素材的寻找方向。

![](img/ec985ce96982e571fadbc9588919f33d_12.png)

例如，我们本次选择制作一个“**沈腾综艺爆笑名场面**”的主题视频。这意味着我们寻找的所有素材片段都必须包含沈腾这个人。

![](img/ec985ce96982e571fadbc9588919f33d_14.png)

![](img/ec985ce96982e571fadbc9588919f33d_6.png)

![](img/ec985ce96982e571fadbc9588919f33d_16.png)

主题确定后，点击“开始创作”界面的加号，从手机相册中选择已准备好的、符合主题的素材视频，将其导入到剪辑轨道中。

![](img/ec985ce96982e571fadbc9588919f33d_18.png)

![](img/ec985ce96982e571fadbc9588919f33d_10.png)

![](img/ec985ce96982e571fadbc9588919f33d_20.png)

---

![](img/ec985ce96982e571fadbc9588919f33d_22.png)

素材添加完成后，我们需要为视频制作一个标题页。标题页能清晰地向观众传达视频主题。

在视频轨道上，找到并点击“**添加**”按钮（通常显示为“+”号）。

![](img/ec985ce96982e571fadbc9588919f33d_24.png)

![](img/ec985ce96982e571fadbc9588919f33d_16.png)

![](img/ec985ce96982e571fadbc9588919f33d_26.png)

在弹出的菜单中，选择“**素材库**”。

![](img/ec985ce96982e571fadbc9588919f33d_28.png)

![](img/ec985ce96982e571fadbc9588919f33d_18.png)

剪映自带的素材库提供了多种背景。对于初学者，建议选择简洁的纯色背景（如黑色或白色），这样画面看起来更工整，不会显得杂乱。

![](img/ec985ce96982e571fadbc9588919f33d_24.png)

![](img/ec985ce96982e571fadbc9588919f33d_30.png)

选择黑色背景，点击“添加”将其放入视频轨道，通常它会自动插入到所有素材的最前方。

---

背景有了，接下来需要添加主题文字。

点击底部工具栏中的“**文字**”选项。

![](img/ec985ce96982e571fadbc9588919f33d_32.png)

![](img/ec985ce96982e571fadbc9588919f33d_30.png)

然后点击“**新建文本**”，输入我们的主题，例如：“沈腾综艺爆笑名场面”。

![](img/ec985ce96982e571fadbc9588919f33d_34.png)

为了让文字更美观，我们可以直接使用“**文字模板**”。模板区分为免费和付费，选择一款合适的免费模板应用即可。

![](img/ec985ce96982e571fadbc9588919f33d_32.png)

应用模板后，点击右下角的“**文本朗读**”功能。为文字添加语音解说可以极大提升视频的吸引力，尤其适合自己不会或不想配音的创作者。

以下是选择文本朗读功能的代码式描述：
```
点击【文字轨道】 -> 选择【文本朗读】 -> 从【音色库】中挑选喜欢的声音
```

在音色库中，有各种方言、萝莉音、播音腔等可选。目前使用率较低的“**方言音**”更具特色和创新性，可能更容易吸引观众注意。

选择音色（如“广西话”）后，系统会自动生成语音音频。调整黑色背景素材的时长，使其与语音长度一致，实现“音画同步”。

---

标题页制作完成后，我们回到主视频素材部分。

首先需要调整视频的**比例**，这是适应不同平台发布要求的关键一步。

![](img/ec985ce96982e571fadbc9588919f33d_36.png)

大多数视频平台（如B站、西瓜视频）要求视频比例为 **16:9**（横屏）。

而抖音平台则要求视频比例为 **9:16**（竖屏）。

![](img/ec985ce96982e571fadbc9588919f33d_38.png)

操作路径如下：
```
点击底部工具栏【比例】 -> 选择【16:9】或【9:16】
```

请根据你的目标发布平台选择正确的比例。

![](img/ec985ce96982e571fadbc9588919f33d_40.png)

本节课中我们一起学习了视频剪辑的初始流程：从确定主题、导入素材，到制作标题页（添加背景、文字和配音），最后调整视频比例。这些是完成一个视频作品最基础且必要的步骤。下节课我们将继续讲解如何对主视频进行裁剪、分割、添加转场和背景音乐等进阶操作。
