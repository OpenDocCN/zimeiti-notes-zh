## 第一部分：抖音剪辑神器——全面了解剪映APP 📱

![](img/4c8caebb9342ab3e12fad96458e9f8a9_7.png)

在短视频时代，许多用户看到朋友圈刷屏的爆款视频时，常常感到困惑：这些视频是如何制作的？对于没有剪辑技术、不会拍摄或剪辑的用户来说，这是一个难题。

学习本课程并了解剪映APP后，你将能够自己设计和制作出类似的爆款视频。

### 第一小节：认识剪映APP
上一节我们概述了课程目标，本节中我们来看看剪映APP本身。第一小节将带领大家快速认识这款工具。
*   **剪映APP是什么**：它是抖音官方推出的一款视频剪辑软件。
*   **通过剪映制作短视频的优势**：操作简便，功能强大，与抖音平台深度集成。
*   **哪些人适合用剪映做短视频**：适合短视频爱好者、自媒体新手、电商从业者等所有想制作短视频的用户。

### 第二小节：剪映APP界面结构拆解
认识了剪映APP是什么之后，我们需要熟悉它的操作环境。本节将详细讲解剪映APP的整体界面结构。

![](img/4c8caebb9342ab3e12fad96458e9f8a9_13.png)

剪映APP的界面结构比较简单，主要划分为四个部分：

![](img/4c8caebb9342ab3e12fad96458e9f8a9_15.png)

1.  **剪辑**：这是软件的核心功能区。所有视频剪辑操作基本上都在这里完成。
2.  **剪同款**：这是一个非常实用的功能。剪映将市面上流行的爆款视频做成了模板，用户只需将自己的素材填充到模板中，即可一键生成同款视频。其核心逻辑可以表示为：**用户素材 + 视频模板 = 爆款视频**。
3.  **消息**：当用户授权登录自己的抖音账号后，可以在这里同步查看抖音的官方消息、收到的点赞、粉丝留言等。
4.  **我的**：展示个人账号信息。将抖音号与剪映号关联后，抖音账号的相关信息会在这里展示。

![](img/4c8caebb9342ab3e12fad96458e9f8a9_17.png)

![](img/4c8caebb9342ab3e12fad96458e9f8a9_19.png)

### 第三小节：剪映核心操作功能详解
了解了界面结构，接下来我们进入最核心的部分——功能学习。本节是第一部分课程的重中之重，将详细拆解剪映的15大核心功能。

掌握这些功能的使用方法和技巧，才能理解视频制作的逻辑与原理，在今后创作时灵活运用。如果不了解这些功能，即使在后续的实战案例中，也可能无法理解其设计原理。

![](img/4c8caebb9342ab3e12fad96458e9f8a9_21.png)

![](img/4c8caebb9342ab3e12fad96458e9f8a9_23.png)

以下是剪映剪辑视频过程中最常用的15项功能列表：
*   分割
*   音频
*   文本
*   贴纸
*   画中画
*   特效
*   滤镜
*   比例调整和背景设置
*   视频过渡（转场）
*   定格
*   倒放
*   蒙版
*   变声
*   色度抠图
*   抖音常用素材片段的寻找

![](img/4c8caebb9342ab3e12fad96458e9f8a9_25.png)

掌握了这15项功能，在制作各类视频时，你都能得心应手地使用它们来完成创作。
