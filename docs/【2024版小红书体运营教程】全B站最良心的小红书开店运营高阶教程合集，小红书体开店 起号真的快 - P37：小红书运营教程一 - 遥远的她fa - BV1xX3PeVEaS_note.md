## 认识小红书平台

上一节我们介绍了课程的整体框架，本节中我们来看看小红书平台本身。

小红书App的核心界面主要由几个流量板块构成。了解这些板块有助于我们精准地投放内容。

以下是小红书的主要流量入口：

1.  **发现页**：这是最主要的流量来源。平台会根据用户的注册信息、选择的兴趣标签（如宠物、美食、VLOG）和浏览行为，通过算法进行特征匹配，为用户推荐相关的笔记和视频。
2.  **位置页**：基于用户的地理位置信息，推荐当地的景点、餐厅、展览等相关笔记，具有很强的地域属性。
3.  **关注页**：展示用户所关注账号发布的内容。对于新账号，系统也会根据算法推荐一些可能感兴趣的账号供用户关注。优质的账号有机会被系统推荐到其他用户的关注信息流中。
4.  **搜索框**：用户通过关键词主动搜索内容。搜索框内的热门搜索词是重要的流量入口。通过搜索而来的用户通常目的明确，转化质量较高。

整个小红书平台，流量较大的板块主要就是以上四个：发现、位置、关注和搜索。

## 平台检测规则详解

了解了流量入口后，我们需要明白平台允许和禁止什么。本节将详细介绍小红书的检测规则。

![](img/1839bd3780185a86fabcde31674c3182_1.png)

关于小红书的检测规则，主要从以下四个方面进行：

1.  **内容检测范围**：平台会检测**个人资料、笔记内容、私信以及评论**。在这些地方，**不得出现**微信号、微信二维码、代购、其他平台水印、推广淘宝店铺等明显的营销词汇。但平台对**邮箱**相对宽松，个人资料中允许填写。也可以用“微博”等谐音或替代词进行间接引流，再通过话术引导至微信（具体方法后续课程会详述）。
2.  **设备检测**：平台会检测登录设备。**建议同一台设备最多只登录3个账号**，过多账号可能导致被判定为营销号。
3.  **内容行为规范**：笔记内容**不得诱导用户**，例如“关注领奖”、“点赞抽奖”等。这类行为是小红书明令禁止的。
4.  **原创度要求**：小红书对原创内容的保护程度较高。**切勿将同一篇文章在多个账号上重复发布**。一篇文章只在一个账号发布即可，多发会被判定为搬运或低质内容。

此外，还有一个重要的规则是**敏感词检测**。平台对某些领域词汇比较敏感，例如手机卡、金融、电子烟、医疗用品等，相关内容需格外谨慎。

![](img/1839bd3780185a86fabcde31674c3182_3.png)

如果你不确定自己的文案是否包含敏感词，可以使用敏感词检测工具。例如，将写好的文案粘贴到检测网站中，点击检测，工具会标出可能的敏感词。

```
示例：在检测工具中输入“代购最新款手机”，点击检测后，工具可能会提示“代购”属于敏感词。
```

建议在发布前，都使用此类工具进行一遍检查，以避免违规。

## 内容推荐机制解析

明白了规则，我们再来看看小红书是如何决定把你的内容推荐给更多人的。本节将解析其核心的推荐机制。

小红书的推荐机制与抖音、头条等平台逻辑相似，都基于**标签**和**算法**。

1.  **打标签与审核**：用户和内容都会被系统打上相应的标签。每篇笔记发布后，首先会经过**机器审核**，检测是否有违规内容。
2.  **分级流量池推荐**：如果机器审核无违规，笔记会进入一个**基础流量池**（例如，推荐给200-500人）。系统会根据这批初始读者的**互动数据**（点赞、收藏、评论、转发）来判断笔记质量。
3.  **数据驱动的晋级**：如果互动数据达到某个阈值，笔记就会被推荐到下一个更大的**二级流量池**（例如，1000人）。在新的流量池中，如果互动数据依然优秀，则有机会进入**三级流量池**（例如，5000人），以此类推。
4.  **人工审核介入**：如果机器审核无法判定是否违规，笔记会进入**人工审核**环节。若人工也判定违规，笔记将被限流或放入“小黑屋”。

这个机制可以总结为以下流程：
**发布笔记 → 机器审核 → 进入基础流量池 → 根据互动数据（点赞/收藏/评论/转发）决定是否晋级至更大流量池 → 可能的人工复核。**

因此，笔记的流量是逐步积累的，而非一蹴而就。核心在于内容能否在每一个流量池中引发良好的互动。

## 搜索排名规则

除了推荐流量，搜索流量也非常重要。本节我们来看看如何让笔记在搜索结果中获得更好的排名。

需要注意的是，小红书的**内容推荐**和**搜索收录**是两个相对独立的系统。被推荐的笔记不一定能被搜索到。

想让笔记获得搜索排名，**前提是笔记必须被小红书平台收录**。这也是为什么声称“100%包收录”的服务基本不可信。

影响搜索排名的主要因素有以下几点：

1.  **关键词**：这是搜索引擎的核心。关键词可分为三类：
    *   **核心关键词**：描述产品/主题最基础的词语，搜索量高。（例如：“祛痘”）
    *   **相关关键词**：核心关键词的扩展。（例如：“食疗祛痘”）
    *   **长尾关键词**：由多个词组成的短语，更具体，意图更明确。（例如：“有什么办法可以快速祛痘”）
2.  **账号与笔记权重**：这是一个综合体系，包括发布账号的权重、互动用户账号的权重等。权重越高，排名通常越靠前。
3.  **互动数据**：笔记的点赞、收藏、评论数量直接影响其在搜索结果中的排名。数据越好，排名越靠前。
4.  **用户行为**：账号的日常行为是否像“正常用户”。例如，定期浏览、互动，而不是集中、机械地点赞或发布内容。异常行为会降低权重。

## 笔记不被收录的常见原因

很多新手会遇到笔记发布后没有流量的问题，很可能是因为笔记未被收录。本节总结了几点常见原因。

笔记不被收录，通常可以对照检查以下五点原因：

![](img/1839bd3780185a86fabcde31674c3182_5.png)

![](img/1839bd3780185a86fabcde31674c3182_7.png)

1.  **违规推广**：笔记中包含硬广信息，如直接出现品牌名、促销信息等。
2.  **触犯敏感词**：文案或图片中含有平台禁止的敏感词汇，即使暂时被收录，后期也可能被系统剔除或限流。
3.  **内容违规**：
    *   **图片违规**：带有其他平台的水印、二维码等。
    *   **文字违规**：含有违禁词、敏感词、不文明用语或诱导信息。
    *   **特别注意**：严禁涉及政治、破坏民族团结等法律法规禁止的内容。
4.  **发布频率异常**：短时间内高频发布笔记，容易被系统判定为营销机器人，导致账号异常，笔记不被收录。
5.  **设备被拉黑**：如果当前使用的手机设备曾有过严重违规记录，其硬件信息（如IMEI码）可能已被平台记录。即使更换账号，在该设备上发布的内容也较难被收录。

## 种草笔记速成四大步骤

![](img/1839bd3780185a86fabcde31674c3182_11.png)

掌握了基础规则，我们就可以开始创作内容了。本节将手把手教你写出高质量的小红书种草笔记。

![](img/1839bd3780185a86fabcde31674c3182_13.png)

小红书的核心内容是**种草笔记**。写好种草笔记，后续的引流、变现都会事半功倍。关键是要避免写成“硬广”。

![](img/1839bd3780185a86fabcde31674c3182_15.png)

一篇优秀的种草笔记，可以通过以下四个步骤完成：

### 第一步：准备素材图片
根据产品特点，准备4类图片：
*   **产品图 (2-3张)**：展示商品全貌。
*   **效果图 (1-2张)**：展示使用效果。
*   **细节图 (1-2张)**：展示工艺、材质等细节。
*   **使用图 (1-4张)**：展示真实使用场景。

**素材来源**：可以直接参考淘宝、天猫等电商平台的商品主图和买家秀（评价带图）。

### 第二步：撰写文案
文案包括标题和正文。

![](img/1839bd3780185a86fabcde31674c3182_17.png)

*   **标题优化**：在标题中突出产品的核心卖点或特色。
    *   示例：原标题《男朋友挑选的圣诞礼物：蒸汽足浴盆》
    *   优化后《**无需倒水**的蒸汽足浴盆，男朋友挑的礼物太贴心！》
*   **正文结构**：推荐按以下逻辑顺序书写，能让文章读起来更自然、可信：
    1.  **从个人经历切入**：用“我”开头，讲述自己的困扰或需求，增加代入感。（例如：“作为一个冬天手脚冰凉的女生，我一直在寻找解决办法…”）
    2.  **指出痛点/竞品缺点**：详细描述现有产品或普遍存在的不足。（例如：“传统的泡脚桶又重又难倒水，加热也不均匀…”）
    3.  **引入产品，给出解决方案**：自然引出推荐的产品，并说明它是如何解决上述痛点的，列出其优势。（例如：“直到我发现了这款XX足浴盆，它一键启动、水电分离、恒温加热，完美解决了我的问题…”）
    4.  **补充个人感受与注意事项**：分享使用后的真实体验，并补充一些保养或使用小贴士。

### 第三步：制作图片
*   **封面图**：需精心设计，最好包含产品图，并使用“黄油相机”或“美图秀秀”等APP添加吸引人的文字和滤镜。
*   **配图**：准备5-8张，与正文内容配合，可以添加标签或简短文字说明。

### 第四步：发布前检查
完成以上步骤后，务必进行最终检查：
*   检查错别字和敏感词。
*   确保文案排版清晰，适当换行，不要过于密集。
*   多使用表情符号，让文案更生动。
*   确认所有图片无水印、无违规二维码。
