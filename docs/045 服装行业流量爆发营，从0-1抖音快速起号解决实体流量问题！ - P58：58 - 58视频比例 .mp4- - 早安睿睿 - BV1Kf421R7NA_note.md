# 服装行业流量爆发营：P58：短视频拍摄的视频比例调整 📐

![](img/2e67bb9ebb7e7ef44c6edf2185e61b90_0.png)

在本节课中，我们将要学习短视频制作中一个非常基础但至关重要的概念：**视频比例**。我们将了解不同平台的比例要求，并掌握如何使用剪映App来调整视频比例，特别是解决横屏视频上传到竖屏平台（如抖音）时画面留白的问题。

## 什么是视频比例？

视频比例是指视频画面的宽度与高度之比。在短视频流行之前，我们常见的视频比例是 **4:3**（如传统电视）或 **16:9**（如宽屏电影和大多数视频平台）。随着抖音、快手等平台的兴起，**9:16** 的竖屏视频比例开始普及。

![](img/2e67bb9ebb7e7ef44c6edf2185e61b90_2.png)

## 如何设置视频比例？

我们将使用“剪映”App进行实操。以下是具体步骤：

1.  **导入素材**：打开剪映，点击“开始创作”，导入一段已拍摄好的视频素材。导入后，视频会保持其原始拍摄比例。
2.  **选择比例**：将底部功能栏滑到最右侧，点击“比例”按钮（通常是倒数第三个图标）。
3.  **匹配平台**：在弹出的选项中，根据目标平台选择相应比例：
    *   **抖音/快手**：选择 **9:16**（竖屏）。
    *   **西瓜/头条**：选择 **16:9**（横屏）。
    *   **淘宝**：选择 **3:4**（竖屏）。

## 横屏视频如何适配竖屏平台？

如果你拍摄的是横屏（16:9）视频，但想上传到抖音（9:16），直接选择9:16比例后，视频会居中显示，上下会出现黑色留白区域。以下是三种解决此问题的方法：

### 方法一：添加背景图片

![](img/2e67bb9ebb7e7ef44c6edf2185e61b90_4.png)

返回到功能栏，滑到最右侧，点击“背景”选项（通常是倒数第二个图标）。然后选择“画布样式”，点击图片标志，从相册中选择一张合适的图片作为背景。这样就能用自定义图片填充上下空白区域。

![](img/2e67bb9ebb7e7ef44c6edf2185e61b90_6.png)

### 方法二：使用画布模糊

在“背景”功能中，选择“画布模糊”。这里有四个模糊度选项，从左到右模糊程度递增。选择后，App会自动将你的主视频画面放大并模糊处理，作为背景填充上下区域，使画面整体更协调。

### 方法三：更改画布颜色

如果你不想要复杂背景，也不想要黑色留白，可以在“背景”功能中选择“画布颜色”。然后选择一个纯色（如白色）作为背景，这样上下区域就会变成你选择的纯色。

## 总结

![](img/2e67bb9ebb7e7ef44c6edf2185e61b90_8.png)

本节课中，我们一起学习了视频比例的核心概念，掌握了如何为不同平台（如抖音、西瓜视频）设置正确的视频比例。更重要的是，我们通过三种实用方法——**添加背景图片**、**使用画布模糊**和**更改画布颜色**——解决了横屏视频适配竖屏平台时的画面留白问题。这些技巧能帮助你制作出更专业、更符合平台要求的短视频内容。
