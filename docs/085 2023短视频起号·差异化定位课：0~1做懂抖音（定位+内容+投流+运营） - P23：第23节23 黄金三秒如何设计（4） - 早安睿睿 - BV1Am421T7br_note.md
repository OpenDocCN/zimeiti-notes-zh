# 短视频起号·差异化定位课：23：黄金三秒设计（4）——戳痛点法 📌

![](img/ffeaf2897f68828bbcc4695b8098eacb_0.png)

在本节课中，我们将学习设计短视频“黄金三秒”的第四种核心方法：**戳痛点法**。这种方法通过快速引发观众共鸣，激发其解决问题的欲望，从而有效提升视频的完播率与转化效果。

上一节我们介绍了利用悬念或提问来吸引观众，本节中我们来看看如何通过直接触及用户痛点来抓住他们的注意力。

![](img/ffeaf2897f68828bbcc4695b8098eacb_2.png)

## 概述：什么是戳痛点法？

![](img/ffeaf2897f68828bbcc4695b8098eacb_4.png)

戳痛点法的核心逻辑分为两步：
1.  **引发共鸣**：视频开头直接陈述一个目标用户普遍会遇到的问题或困境，让用户产生“这正是我的情况”的共鸣感。
2.  **激发期待**：用户因共鸣而产生解决问题的迫切需求，从而对视频后续提供的解决方案充满期待，愿意继续观看。

这种方法在好物种草、知识分享、问题解决类视频中尤为常见。

## 方法解析与案例

![](img/ffeaf2897f68828bbcc4695b8098eacb_6.png)

以下是戳痛点法的具体应用方式，我们通过几个案例来理解。

**案例一：亲子教育账号**
*   **视频开头**：“入园那天，害怕宝宝哭，把宝宝扔给老师就跑？还不如提前几天多做这样的准备…”
*   **痛点分析**：精准戳中了宝妈对于孩子“入园焦虑”的普遍痛点。
*   **方法应用**：第一句话直接点出问题，并暗示有解决方案（“多做准备”），吸引有同样焦虑的家长继续观看。

![](img/ffeaf2897f68828bbcc4695b8098eacb_8.png)

**案例二：好物种草账号**
*   **视频开头**：“能随意调节茶水浓度的杯子，你见过吗？”
*   **痛点分析**：戳中了喝茶人士“茶汤浓度难以控制，时浓时淡”的痛点。
*   **方法应用**：首句提出一个能解决痛点的产品功能，立即引发目标用户的兴趣和对产品的期待。

**案例三：美妆护肤口播账号**
*   **视频开头**：“哈喽，今天分享一下怎么搞定这个法令纹…”
*   **痛点分析**：直接点出“法令纹”这一具体的美容痛点。
*   **方法应用**：开篇明义，告诉观众本视频将解决这个明确的问题，吸引有相同困扰的用户。

## 如何设计你的戳痛点开场

掌握了核心逻辑后，你可以按以下步骤为自己的视频设计开场。

1.  **梳理选题痛点**：每个选题都对应着用户的一个细分需求或痛点。例如：
    *   **美食赛道（回锅肉）**：痛点可能是“油腻”。开场可设计为：“觉得回锅肉腻？是你方法不对，这样做一点也不腻。”
    *   **美妆赛道（油性皮肤）**：痛点可能是“皮肤出油”。开场可设计为：“油皮夏天不敢化妆？三招教你轻松告别油光。”
    *   **汽车/电动车赛道**：痛点可能是“油价高”。开场可设计为：“油价太贵加不起？这款电动车一天成本不到一块钱。”

2.  **开场句式模板**：一个有效的戳痛点开场通常遵循这个公式：
    `【痛点描述/提问】 + 【解决方案暗示/引导】`
    例如：“`如果你也遇到[XX问题]，试试这[数字]个方法…`”

3.  **衔接视频主体**：开场之后，视频后半部分应清晰、有条理地展示解决问题的步骤、方法或产品功能，满足观众的期待。

## 总结与预告

本节课我们一起学习了“黄金三秒”的第四种设计方法——**戳痛点法**。其关键在于**快速引发共鸣**，并**激发解决问题的期待**。这种方法适用性广，尤其在带货和知识分享类视频中效果显著。
