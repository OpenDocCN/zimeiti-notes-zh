# 抖音自媒体运营：P19：扶持加权与降权限流 📊

在本节课中，我们将学习抖音平台对内容的具体评判标准，了解哪些内容会获得平台的“扶持加权”，哪些内容则可能导致“降权限流”。掌握这些规则是账号健康运营的基础。

上一节我们介绍了账号权重的分类，本节中我们来看看平台具体鼓励和限制的内容类型。

## 获得扶持加权的内容 ✅

![](img/63649281d34bcbdcea1b6a3fbaf6b738_3.png)

平台会优先推荐符合以下标准的视频内容。

以下是获得扶持加权的核心要求列表：

![](img/63649281d34bcbdcea1b6a3fbaf6b738_5.png)

*   **视频格式**：竖版移动端原生视频。具体指 **9:16** 的竖屏格式，这是抖音的原生视频格式。
*   **画面质量**：画面平稳，作品清晰。
*   **音频质量**：无过多杂音或噪音。

这些要求较为基础，旨在保证基本的观看体验。相关详细说明会整理在课后资料中供大家查阅。

## 导致降权限流的内容 ❌

接下来，我们看看哪些内容会被平台限制流量甚至处罚。这是本节课的重点，需要特别注意。

以下是明确会导致降权限流的内容类型列表：

1.  **违规信息**：视频内容涉及吸烟、暴力、炫富（金钱）等信息。
2.  **外部引流**：出现微信、微博、淘宝等站外联系方式。长时间出现商品Logo、手机号、二维码（二维码出现时间建议不超过3秒，但审核严格时瞬时出现也可能被驳回）。
3.  **硬性广告**：进行明显的营销广告、推销或引流卖货行为。

关于第3点“硬性广告”，需要特别展开说明。平台尤其警惕**无口播的拆箱或好物推荐视频**。例如，有同学制作玩具或日用品推荐视频时，仅展示产品，没有口播解说，导致挂载的小黄车被下架。

**正确的做法是结合口播进行“软性推荐”**。例如：
*   **错误示范**：视频中只出现产品，或打上“9.9元”的价格标签。
*   **正确示范**：通过口播介绍产品体验，如“我儿子玩这个玩具玩了4个小时都没停”或“这款纸巾沾水后依然不易破”。

核心原则是：避免单纯展示产品，要通过情景或口播赋予内容价值，避免被系统判定为纯商业广告。

本节课中我们一起学习了抖音平台的“扶持加权”与“降权限流”规则。关键在于制作**竖屏、高清、无杂音**的原生视频，并避免**硬广、引流和违规内容**。对于带货视频，务必加入口播进行软性讲解。这些规则资料将整理下发，供大家随时查阅。
