# 微信视频号教程：P26：11.如何填写邀请函

在本节课中，我们将学习如何正确填写视频号创作者计划的邀请函。这是加入计划、开启变现的关键一步，我们将一步步拆解每个填写项，确保信息准确无误。

## 📝 邀请函填写步骤详解

![](img/404fd9efadabe490b54de3216c4bb869_1.png)

上一节我们介绍了如何注册视频号，本节中我们来看看如何填写与之关联的邀请函。请准备好你的微信，跟随以下步骤操作。

### 第一步：打开微信
以下是第一步操作：
*   在手机上找到并点击微信应用图标，启动微信。

### 第二步：进入“发现”页
打开微信后，我们需要进入特定页面。以下是第二步操作：
*   在微信主界面底部，找到并点击 **“发现”** 选项卡。



![](img/404fd9efadabe490b54de3216c4bb869_3.png)

![](img/404fd9efadabe490b54de3216c4bb869_1.png)

![](img/404fd9efadabe490b54de3216c4bb869_4.png)

### 第三步：找到视频号入口
进入“发现”页后，下一步是定位视频号。以下是第三步操作：
*   在“发现”页面中向上滑动，找到 **“视频号”** 入口（文字标识或蝴蝶图标），点击进入。



![](img/404fd9efadabe490b54de3216c4bb869_3.png)

![](img/404fd9efadabe490b54de3216c4bb869_4.png)

### 第四步：进入个人主页
进入视频号后，需要找到个人账户信息。以下是第四步操作：
*   在视频号界面右上角，点击 **个人头像** 图标，进入个人主页。



![](img/404fd9efadabe490b54de3216c4bb869_6.png)

![](img/404fd9efadabe490b54de3216c4bb869_6.png)

![](img/404fd9efadabe490b54de3216c4bb869_7.png)

### 第五步：查看账号信息
在个人主页，我们可以找到填写邀请函所需的核心信息。以下是第五步操作：
*   在个人主页中向下滑动，找到 **“我的视频号”** 部分。这里会显示你的**账号昵称**和**粉丝数量**。
    *   例如：昵称显示为“天道酬勤”，粉丝量显示为“39917”。



![](img/404fd9efadabe490b54de3216c4bb869_7.png)

## ✍️ 邀请函各栏目填写指南

了解了如何查找信息后，我们开始正式填写邀请函的各个栏目。

### 1. 视频号账号昵称与粉丝量
根据上一步找到的信息，在邀请函对应位置填写：
*   **账号昵称**：填写你的视频号昵称，例如“天道酬勤”。
*   **粉丝数量**：填写你当前的粉丝数，例如“39917”。

### 2. 活动邀请人
由于是通过特定渠道加入，邀请人栏目需固定填写：
*   **邀请人**：填写 **“张良”** 或 **“张良老师”**。

### 3. 实名信息
此处需要填写与微信支付实名一致的姓名：
*   **实名信息**：填写你的真实姓名，例如“周建国”。

### 4. 结算手机号
这是用于接收验证码和后续结算的重要信息：
*   **手机号**：填写你本人常用的手机号码，例如 `13812345678`。注意数字间**不要有空格**。



![](img/404fd9efadabe490b54de3216c4bb869_9.png)

![](img/404fd9efadabe490b54de3216c4bb869_9.png)

![](img/404fd9efadabe490b54de3216c4bb869_10.png)

![](img/404fd9efadabe490b54de3216c4bb869_10.png)

![](img/404fd9efadabe490b54de3216c4bb869_11.png)

### 5. 提交确认
所有信息填写完毕后，进行最终提交：
*   点击页面底部的 **蓝色“提交”按钮**。
*   系统会弹出“确定要提交吗？”的确认框，点击 **“确认”** 完成提交。



![](img/404fd9efadabe490b54de3216c4bb869_11.png)

提交成功后，页面会提示“已成功加入创作者分成计划”。
