# 抖音运营教程：P4：3.账号定位 🎯

![](img/3516a4652aa12c580e6bedba75bf291b_0.png)

在本节课中，我们将要学习抖音账号定位的核心方法。账号定位是运营的基石，它决定了内容的方向、吸引的粉丝类型以及最终的变现潜力。一个清晰的定位能帮助你在海量内容中脱颖而出。

上一节我们介绍了抖音运营的整体思路，本节中我们来看看如何具体地为你的账号进行精准定位。

![](img/3516a4652aa12c580e6bedba75bf291b_2.png)

## 理解账号定位：三个核心问题

对账号定位的理解，需要反思三个问题：你是谁，你是干什么的，为什么要关注你。

![](img/3516a4652aa12c580e6bedba75bf291b_0.png)

## 第一步：确定账号类型与细分赛道

首先，我们要确定接下来要做的账号类型。除了我们之前章节讲过的大类赛道（即抖音定义的一级垂类），我们还需思考在一级垂类下细分的具体赛道是什么。

这好比参加运动会，你报的项目是“田径”，这是你的一级垂类。而你最擅长且具有优势的项目是“百米赛跑”，那么“百米赛跑”就是你的二级垂类。通过定位账号的细分类目，可以确定账号的垂直类型。

![](img/3516a4652aa12c580e6bedba75bf291b_2.png)

在这张图中可以看到，在抖音中，无论任何行业，只要定位足够清晰，都有机会成功。因为抖音流量巨大，一旦做起来，增长和裂变的速度会非常快。

## 第二步：明确账号主体

其次，我们要确定账号主体是谁。账号主体是该账号的拥有人，并最终呈现给粉丝的账号主人公。目的是让粉丝清楚地知道是在看谁的作品。

在抖音商业类账号中，账号主体主要分为两类：
*   **商家账号**：如果账号主体是公司，即为商家账号。账号内容应偏向商业化，并需不断通过内容拉近与粉丝的距离，避免过于严肃。这样既能建立信任感，也能带来好感，更符合抖音的娱乐属性。
*   **达人账号**：如果账号主体是个人，即为达人账号。这类账号首先应解决信任问题。可以通过IP打造的方式，为账号树立一个积极、正面的人设。

## 第三步：明确账号内容与职业

我们还要明确这个账号是做什么的。最直接的方式是通过抖音认证身份，也就是我们的职业。

*   如果是**达人**，可以通过认证**黄V**。
*   如果是**公司**，可以通过认证**蓝V**。

认证职业后，粉丝可以清晰地看到我们的认证标签，从而明白我们的职业以及这个账号大概做什么。同时，商家认证蓝V或达人认证黄V，也可以获取平台背书，增加信任感。

在未认证之前，也可以通过账号设置向粉丝明确传达你是做什么的。以下是可进行业务关联统一设置的元素：
*   头像
*   背景图
*   简介
*   视频封面

## 第四步：思考为粉丝提供的价值

最后，我们需要花时间思考一个问题：你将为粉丝提供什么样的价值？在抖音中，粉丝对博主的价值需求主要有以下几种类型：

以下是粉丝价值需求的几种主要类型：

1.  **知识价值**
    *   **认知型**：如人生思考、思维方式、读书分享、行业内幕、成功经验、个人提升等。
    *   **干货型**：如技能提升、经验总结、过程步骤、冷知识等。
    *   **常识普及型**：如生活、健康、产品、科学、历史、地区等常识的解读和普及。
    *   **提供给粉丝的价值**：**认同感**（如三观认同、观点认同），减少知识盲区的获得感，进而产生对作者的认同。

2.  **情绪价值**
    *   情绪是影响观众的重要因素。作品中观众容易接受的情绪类型包括：开心愉悦（如搞笑、夸张表情）、共鸣（如情侣、婆媳、家庭、职场等人物关系）、情绪波动（如争议话题、反转反差内容、反常识开场激发）、怀旧、emo、震惊、好奇、颠覆三观等。
    *   **提供给粉丝的价值**：**共鸣**。利用表情、文字、画面、声音、肢体动作等准确到位地抒发情绪，直接感染观众。

3.  **感觉价值**
    *   博主通过短视频的画面、文字和声音传递信息。提供的价值包括：视频的画面美感、构图，以及给观众带来的喜欢、治愈、视觉冲击感，或个人魅力带来的仰慕感；视频声音带来的节奏感、心跳加速或听觉享受。
    *   常见类型：颜值类、萌宠类、美景类、才艺类、明星大咖、文化类等。
    *   **提供给粉丝的价值**：**共情**。因表情、文字、画面、声音、肢体动作等的自然表现，让人产生联想，并在情绪、认知等意识层面与作者产生连接，从而间接引发共鸣。

4.  **体验价值**
    *   抖音让用户看到了不一样的生活方式，了解了自己到不了的地方，体验了未曾体验过的事物。这就是抖音为用户提供的体验价值。
    *   常见类型：流量最多的美食账号（享用体验、制作体验）、各类旅行体验、运动体验、角色体验等。
    *   **提供给粉丝的价值**：主要是**好奇心的满足**，以及紧张感、激动、感动、体验获得感。

5.  **期待价值**
    *   当用户被你的短视频内容吸引，或对作者本人产生好奇与期待时，在看完视频后可能会意犹未尽，从而点击关注，期待下次更新。
    *   许多博主会利用这一点，不将所有内容在一个视频中呈现完，给人留下悬念和期待。

本节课中我们一起学习了抖音账号定位的完整步骤。我们从理解定位的三个核心问题出发，逐步确定了账号的细分赛道、主体身份、内容方向，并深入分析了能为粉丝提供的五大核心价值。一个精准、清晰的账号定位，是你在抖音平台获得持续关注和成功变现的第一步。下一节，我们将探讨如何基于定位进行内容策划。
