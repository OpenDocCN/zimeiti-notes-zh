# 微信视频号运营：P13：11.如何填写邀请函

在本节课中，我们将学习如何正确填写视频号创作者计划的邀请函。这是加入计划、获取官方素材支持的关键一步，流程简单，但需要确保信息准确无误。

上一节我们介绍了如何注册视频号，本节中我们来看看如何完成后续的邀请函填写，以成功加入创作者计划。

![](img/f87606c5339f70fb5df1c6b97a341498_1.png)

## 第一步：打开微信
首先，在您的手机上找到并打开微信应用。

## 第二步：进入“发现”页
打开微信后，点击底部导航栏的 **“发现”** 选项卡。

![](img/f87606c5339f70fb5df1c6b97a341498_3.png)

## 第三步：找到“视频号”入口
在“发现”页面中，向上滑动屏幕。您会看到 **“视频号”** 的入口（文字标识或蝴蝶图标），点击进入。



![](img/f87606c5339f70fb5df1c6b97a341498_4.png)

![](img/f87606c5339f70fb5df1c6b97a341498_5.png)

![](img/f87606c5339f70fb5df1c6b97a341498_3.png)



![](img/f87606c5339f70fb5df1c6b97a341498_4.png)



![](img/f87606c5339f70fb5df1c6b97a341498_5.png)

## 第四步：进入个人主页
进入视频号页面后，点击屏幕右上角的 **个人头像** 图标，进入个人主页。



![](img/f87606c5339f70fb5df1c6b97a341498_7.png)

![](img/f87606c5339f70fb5df1c6b97a341498_7.png)

![](img/f87606c5339f70fb5df1c6b97a341498_8.png)

## 第五步：查看账号信息
在个人主页中向下滑动，找到 **“我的视频号”** 部分。这里会清晰显示您的**账号昵称**和**粉丝数量**。

例如，显示为：
```
昵称：天道酬勤
粉丝：39917
```



![](img/f87606c5339f70fb5df1c6b97a341498_8.png)

接下来，我们开始填写邀请函。以下是需要填写的五项核心信息，请逐一核对。

**第一项：视频号昵称**
填写您在第五步中看到的账号昵称，例如：`天道酬勤`。

**第二项：粉丝数量**
填写您在第五步中看到的粉丝数，例如：`39917`。

**第三项：活动邀请人**
由于是通过特定渠道加入，此处需填写邀请人姓名，例如：`张良` 或 `张良老师`。

![](img/f87606c5339f70fb5df1c6b97a341498_10.png)

**第四项：实名信息**
此处必须填写您本人的真实姓名，例如：`周建国`。

![](img/f87606c5339f70fb5df1c6b97a341498_11.png)

![](img/f87606c5339f70fb5df1c6b97a341498_12.png)

**第五项：结算手机号**
填写您用于接收验证码和后续结算的本人手机号码，格式为连续数字，例如：`13812345678`。



![](img/f87606c5339f70fb5df1c6b97a341498_10.png)



![](img/f87606c5339f70fb5df1c6b97a341498_11.png)



![](img/f87606c5339f70fb5df1c6b97a341498_12.png)

## 第六步：提交并确认
所有信息填写完毕后，点击页面底部的 **蓝色提交按钮**。
系统会弹出确认提示（如“确定要提交吗？”），请务必点击 **“确认”** 完成提交。

![](img/f87606c5339f70fb5df1c6b97a341498_14.png)

![](img/f87606c5339f70fb5df1c6b97a341498_15.png)

成功提交后，页面会显示“已成功加入创作者分成计划”的提示。



![](img/f87606c5339f70fb5df1c6b97a341498_14.png)

成功加入计划后，您将获得官方正版素材库的使用权限。例如，素材库中包含大量拥有版权的影视动漫素材，如《还珠格格》、《我不是药神》、《大话西游》等，可供您安全地用于视频创作。

本节课中我们一起学习了填写视频号创作者计划邀请函的完整流程。从查找账号信息到填写昵称、粉丝数、邀请人、实名信息及手机号，最后提交确认。每一步都至关重要，确保信息准确方能成功加入计划，从而获得正版素材库等官方支持，为您的视频号运营打下坚实基础。
