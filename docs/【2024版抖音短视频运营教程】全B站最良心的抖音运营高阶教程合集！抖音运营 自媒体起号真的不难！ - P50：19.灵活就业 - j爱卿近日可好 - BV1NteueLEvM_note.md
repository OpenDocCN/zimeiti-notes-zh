# 抖音短视频运营教程：P50：19.灵活就业 💼

在本节课中，我们将探讨自媒体运营作为一种灵活就业的选择，分析其适用人群、行业前景以及相关的职业发展机会。

上一节我们介绍了自媒体运营的多种变现路径，本节中我们来看看如何将自媒体技能转化为一份正式的职业，实现灵活就业。

## 自媒体作为灵活就业的选择

从事自媒体运营，可以考虑将其作为一种灵活的就业方式。这种选择比较适合两类人群。

以下是两类适合考虑自媒体就业的人群：

1.  **临近毕业的大学生**：现阶段临近大学毕业，即将进入社会就业，但不确定哪个行业有发展前景，可以考虑自媒体。
2.  **寻求职业转型的职场人**：现阶段本职工作已没有很好的升值或涨薪空间，想要转行进入互联网行业，也可以考虑自媒体。

![](img/c68bc5b6d5cf3d7e8da2c164fd1fa596_1.png)

因为当前自媒体的教学体系已很成熟，专业的自媒体运营师在各行各业都有广泛的应用。如今每个行业都离不开“宣传”二字。

以下是需要宣传的行业举例：

![](img/c68bc5b6d5cf3d7e8da2c164fd1fa596_3.png)

*   餐饮行业
*   传媒行业
*   教育行业

如今，我们已很少在大街上看到发传单、发小卡片了。这并不意味着他们不做宣传，而是将阵地转移到了互联网平台。这正是未来自媒体运营师的主要工作范围之一。

## 全媒体运营师的职业前景

![](img/c68bc5b6d5cf3d7e8da2c164fd1fa596_5.png)

目前，国家已将“全媒体运营师”纳入国家职业分类。该职业于去年2月份被正式纳入，是一个合法的职业。

在招聘平台上，该职位的起薪也相当可观。以下是招聘平台（如Boss直聘）上为零经验者提供的部分薪资范围：

*   5000 ~ 8000元
*   5000 ~ 7000元
*   7000 ~ 8000元

如果你有转行或进入互联网行业的打算，可以考虑这个方向。

## 团队与工作环境介绍

如果你未来确实考虑转行进入互联网行业，可以考虑加入我们的团队。

我们公司位于江西省赣州市的恒客产业园，是受政府园区邀请入驻办公的。公司设有多个业务部门以支持整体运营。

以下是公司主要部门的职能介绍：

*   **业务支持中心**：这是前端部门，为后端所有部门提供支持。
*   **课程研发中心**：该部门由我主要负责，全权把控平台与课程内容的更新。
*   **流量中心**：现阶段主要负责为内部学员对接官方流量，也负责影视宣传推广、抖音直播切片、挂载小黄车等与千万级网红视频的对接工作。
*   **VIP服务中心**：这是学员加入后的后端扶持部门。当学员遇到素材不会找或视频不知如何制作等问题时，我们的老师会通过语音、视频或远程操控等方式提供帮助。我们后端的VIP老师都非常专业。

除了工作，我们也注重员工的内部成长与休闲生活。

![](img/c68bc5b6d5cf3d7e8da2c164fd1fa596_7.png)

以下是公司提供的部分文化与休闲设施：

*   定期举办读书会
*   拥有咖啡厅
*   设有影音室
*   配备健身房

我们希望所有伙伴都能在工作中感到开心，并快乐地获得收入。
