# 剪映入门教程：P41：剪辑篇丨基础界面全解析 🎬

在本节课中，我们将系统学习剪映App的基础界面与核心操作。通过一个简单的案例，你将了解从创建项目到导出的完整流程，掌握各功能区域的作用，为后续深入学习剪辑技巧打下坚实基础。

## 界面概览与项目创建

打开剪映App后，首先注意两个核心区域：
*   **开始创作**：这是创建新作品的按钮。
*   **草稿箱**：所有已完成或未完成的作品都会自动保存在这里。

接下来，我们通过一个案例来熟悉基础界面。点击“开始创作”进入编辑页面。

## 素材导入与比例设置

导入素材时，请注意区分两个标签页：
*   **视频**：用于导入视频文件。
*   **照片**：用于导入图片文件。**请勿在“视频”标签中寻找照片**。

剪映内置了丰富的“素材库”，包含各种类型的素材可供使用。本节课，我们选择一张黑色背景图进行讲解。

添加素材后，我们进入编辑主界面。首先看到的是**预览框**。需要记住的最基本概念是视频比例：
*   初始的横向比例是 **16:9**。
*   抖音平台支持竖屏视频，因此我们需要将比例改为 **9:16**。点击工具栏中的“比例”，选择 **9:16** 即可。

## 主工具栏功能详解

完成基础设置后，应习惯点击左上角“撤销”箭头或空白处，**返回主工具栏**进行后续操作。主工具栏包含以下核心功能：

以下是主工具栏的核心功能列表：
*   **文本**：点击后可添加和编辑字幕。
*   **音频**：用于添加背景音乐或音效。
*   **剪辑**：选中某段素材后，可对其进行详细编辑。
*   **贴纸**：为视频添加各种装饰性贴纸元素。
*   **画中画**：添加额外的视频或图片图层，与主轨道素材叠加。

**画中画**功能需要特别说明：它是在现有素材之上叠加新图层。例如，点击“新增画中画”并选择一段素材，该素材便会以图层形式覆盖在主素材上。可通过**双指外拉**的手势放大画中画图层。此时，主素材并未消失，只是被上层画面遮挡。

## 素材编辑与效果处理

上一节我们介绍了核心添加功能，本节中我们来看看如何对素材进行编辑和美化。

对素材的基础编辑常使用“分割”功能。如果一段素材过长，可以选中它，点击“分割”将其切断。不需要的部分可以选中后**删除**。此外，直接拖动素材边缘也可进行长短的微调。

返回主工具栏，我们继续查看其他功能：
*   **特效**：为视频添加视觉特效。
*   **滤镜**：为视频快速应用不同的色彩风格，类似于照片美颜中的滤镜功能。
*   **背景**：更改视频画布的背景样式或颜色。**注意**：添加背景后，若被主素材完全遮挡，可能无法立即看到效果，需要拉长主素材或调整图层才能显现。

## 界面导航与成品导出

新手常遇到的一个问题是：添加了多段素材或画中画后，在时间轴上找不到它们了。解决方法有两种：
1.  点击时间轴左上角的**折叠/展开小圆圈**。
2.  直接点击工具栏中的 **“画中画”** 按钮，相关图层会单独显示。

如果需要全屏预览作品，可以点击预览窗右下角的**放大镜图标**。

作品制作完成后，最后一步是导出。点击右上角的 **“导出”** 按钮。在导出设置中，通常选择 **1080P** 分辨率与 **60帧** 的帧率，以获得更清晰流畅的视频画质。

## 课程总结

本节课中，我们一起学习了剪映的基础操作界面。我们从创建项目、设置比例开始，逐步了解了文本、音频、贴纸、画中画等核心工具的用途，并掌握了分割素材、添加特效滤镜、更改背景等基本编辑方法。最后，我们还学会了如何管理多轨道素材以及导出最终作品。掌握这些基础知识，是迈向短视频剪辑高手的第一步。
