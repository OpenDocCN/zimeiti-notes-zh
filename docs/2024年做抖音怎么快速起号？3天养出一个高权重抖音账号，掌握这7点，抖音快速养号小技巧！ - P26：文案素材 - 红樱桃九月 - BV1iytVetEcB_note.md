# 抖音运营教程：P26：文案素材 📝

在本节课中，我们将学习如何为你的抖音视频寻找和获取文案素材。这是视频创作中至关重要的一步，好的文案是视频内容的核心。

上一节我们讨论了视频创意的来源，本节中我们来看看如何为已确定的创意寻找合适的文案。

![](img/79d935ef53e100fc105499e5ef5ada3e_1.png)

![](img/79d935ef53e100fc105499e5ef5ada3e_3.png)

## 文案素材的来源

![](img/79d935ef53e100fc105499e5ef5ada3e_5.png)

![](img/79d935ef53e100fc105499e5ef5ada3e_7.png)

确定了视频创意后，就需要寻找相应的文案。你可以选择直接搬运他人的视频素材并进行二次处理。但如果你想基于自己的创意，以另一种形式（如口播、图文等）制作视频，就需要主动搜索文案。

![](img/79d935ef53e100fc105499e5ef5ada3e_9.png)

以下是获取文案素材的一个高效工具介绍。

### 工具推荐：淘金阁

![](img/79d935ef53e100fc105499e5ef5ada3e_11.png)

![](img/79d935ef53e100fc105499e5ef5ada3e_13.png)

一个非常好用的文案素材工具是“淘金阁”网站。这是一个自媒体全网搜索引擎。

![](img/79d935ef53e100fc105499e5ef5ada3e_15.png)

打开淘金阁网站后，你会看到其主界面。点击进入后，可以发现网站内汇集了来自众多平台的庞大文章素材库。

![](img/79d935ef53e100fc105499e5ef5ada3e_17.png)

它搜集的渠道包括：
*   今日头条
*   百家号
*   企鹅号
*   趣头条
*   一点资讯
*   QQ看点
*   凤凰网
*   网易资讯
*   搜狐网
*   东方网
*   简书
*   快资讯

![](img/79d935ef53e100fc105499e5ef5ada3e_19.png)

## 如何使用淘金阁

![](img/79d935ef53e100fc105499e5ef5ada3e_21.png)

在网站的搜索框中，输入你想要创作内容的关键词。例如，搜索“星座”。

![](img/79d935ef53e100fc105499e5ef5ada3e_23.png)

![](img/79d935ef53e100fc105499e5ef5ada3e_25.png)

按下回车键后，点击“开始抓取”按钮。

![](img/79d935ef53e100fc105499e5ef5ada3e_27.png)

下方就会展示出工具抓取到的所有关于“星座”的文章。请注意，这里找到的是**文章素材**，而非视频素材。

![](img/79d935ef53e100fc105499e5ef5ada3e_29.png)

![](img/79d935ef53e100fc105499e5ef5ada3e_31.png)

将别人的文章素材转化为自己的视频内容，这与直接搬运他人视频有本质区别，不属于违规搬运的范畴。

![](img/79d935ef53e100fc105499e5ef5ada3e_33.png)

你可以随意点开一篇文章进行查看。例如，打开一篇名为《分手后也能挽回，最放不下前任的三大星座》的文章。

![](img/79d935ef53e100fc105499e5ef5ada3e_35.png)

点击后，你就能立即看到完整的文章内容，也就是我们想要的文案。例如，关于巨蟹座、双鱼座等的描述。

![](img/79d935ef53e100fc105499e5ef5ada3e_37.png)

![](img/79d935ef53e100fc105499e5ef5ada3e_39.png)

找到文案后，你可以直接采用复制粘贴的方式，将其内容转化为己用。

## 从文案到视频

现在文案已经找到，接下来就是视频制作环节。

视频制作有多种形式，例如真人出镜拍摄，或者音乐影视混剪。关于混剪，之前课程分享过的台词混剪网站、以及如“mv cat”这类提供电影下载的网站，都可以作为素材来源，供你下载后进行影视混剪创作。

![](img/79d935ef53e100fc105499e5ef5ada3e_41.png)

本节课中我们一起学习了如何利用“淘金阁”工具高效地搜索文案素材，并理解了将文章转化为视频内容与直接搬运视频的区别。掌握这个技巧，能帮助你更自主、更合规地创作抖音视频内容。
