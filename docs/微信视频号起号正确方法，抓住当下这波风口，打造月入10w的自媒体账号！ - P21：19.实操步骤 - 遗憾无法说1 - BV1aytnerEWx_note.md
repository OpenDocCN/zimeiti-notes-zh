# 微信视频号运营教程：P21：19.实操步骤

## 概述
在本节课中，我们将学习如何开通微信视频号的商品橱窗功能，并完成商品上架。这是将视频号内容变现的关键一步，操作流程清晰简单。

上一节我们介绍了视频号的基础概念，本节中我们来看看如何具体开通带货功能。

## 开通视频号商品橱窗权限
以下是开通商品橱窗权限的具体步骤。

![](img/99d0b186ee4873cd23f39813da68a3ce_4.png)

![](img/99d0b186ee4873cd23f39813da68a3ce_6.png)

1.  第一步是打开手机上的微信应用。
2.  第二步是点击微信底部的“发现”选项卡。
3.  第三步是在“发现”页面找到“视频号”入口。该入口有文字标识，图标是一个黄色的蝴蝶形状。
4.  第四步是进入视频号后，点击页面右上角的人形头像图标，进入个人主页。
5.  第五步是在个人主页向下滑动，找到并点击“创作者中心”。如果未找到此选项，需要先发布一条视频作品。
6.  第六步是在创作者中心页面，找到并点击“带货中心”。其图标类似一个小卖部。

完成以上步骤后，系统会判断你的账号状态。如果尚未开通橱窗权限，点击后将进入开通流程；如果已开通，则会直接进入商品上架管理界面。

![](img/99d0b186ee4873cd23f39813da68a3ce_8.png)

![](img/99d0b186ee4873cd23f39813da68a3ce_9.png)

## 选择带货身份与缴纳保证金
点击“带货中心”后，会跳转到身份选择界面。你需要根据自身情况选择带货身份。

以下是三种身份选项：
*   **我是商家**：适用于拥有自己品牌或货源的账号。
*   **我是达人**：适用于通过推广他人商品赚取佣金的账号。
*   **我是商家授权号**：适用于获得品牌方授权进行推广的账号。

对于大多数初学者，建议选择 **`我是达人`** 身份。这样可以直接从视频号官方商品库（选品中心）选择商品进行推广，无需自己备货。

![](img/99d0b186ee4873cd23f39813da68a3ce_11.png)

选择“我是达人”后，会进入保证金缴纳界面。你需要点击“去交纳”按钮，通过微信支付缴纳100元保证金。缴纳成功后，即开通了视频号带货权限。

## 商品上架操作
开通权限后，再次点击“带货中心”，即可进入商品管理后台。

以下是核心操作区域：
*   **选品中心**：在这里可以浏览和搜索海量商品，选择你想要推广的产品。
*   **我的橱窗**：管理你已经添加的商品，可以对其进行上架、下架或排序。
*   **佣金结算**：查看通过推广商品获得的收入明细。

![](img/99d0b186ee4873cd23f39813da68a3ce_13.png)

商品上架的通用流程可以概括为：**`在选品中心找到商品 -> 点击“加橱窗” -> 在我的橱窗中点击“上架”`**。上架后，商品就会出现在你的视频号主页橱窗中，观众可以直接点击购买。

![](img/99d0b186ee4873cd23f39813da68a3ce_15.png)

## 总结
本节课我们一起学习了微信视频号开通商品橱窗功能的完整流程。关键步骤包括：进入创作者中心、开通带货权限、选择达人身份、缴纳保证金，最后在选品中心完成商品上架。掌握这些操作，你就为通过视频号实现内容变现打下了坚实的基础。
