# 小红书运营系统课：P81：18 文字蒙版效果 ✨

在本节课中，我们将学习如何利用蒙版和关键帧功能，制作一个动态的文字片头效果。这个技巧能显著提升视频的视觉吸引力，非常适合用于小红书短视频的开场。

## 概述 📋

我们将通过几个步骤，逐步创建出文字从无到有、颜色变化并最终以蒙版形式展开的动画效果。整个过程涉及素材准备、关键帧设置、混合模式应用以及蒙版动画制作。

## 第一步：准备基础文字动画 🎬

![](img/165c9c1cbfd7061824310d40bbdf2af6_3.png)

首先，我们需要创建一段会移动的基础文字动画。

1.  在剪辑软件中，从素材库添加一个黑色背景素材，并将其时长拉长至大约7秒。
2.  在黑色背景上输入你喜欢的文字，并将文字放大。
3.  将时间轴移动到视频开始（0秒）的位置，选中文字图层，并添加一个**关键帧**。此时，将文字拖动到画面左侧，使其第一个字母对齐画面边缘。
4.  将时间轴移动到视频结尾（约7秒）的位置，将文字水平拖动到画面右侧，使其最后一个字母对齐画面边缘。软件会自动在此处生成第二个关键帧，从而创建出文字从左向右移动的动画。
5.  将这段动画导出为视频文件，备用。我们将其称为“白色移动文字”。

## 第二步：创建彩色文字素材 🎨

接下来，我们将制作一个带有纯色背景的彩色文字素材，用于后续的抠像处理。

![](img/165c9c1cbfd7061824310d40bbdf2af6_8.png)

1.  新建一个项目，导入一张纯绿色的图片作为背景。
2.  导入上一步导出的“白色移动文字”视频，将其放置在绿色背景上方。
3.  将这段文字的颜色修改为红色。
4.  将合成后的视频（即红色文字在绿色背景上移动）导出备用。我们将其称为“红字绿底素材”。

## 第三步：生成镂空文字效果 ✂️

![](img/165c9c1cbfd7061824310d40bbdf2af6_12.png)

现在，我们要利用混合模式，制作出镂空文字的效果。

1.  新建一个项目，导入你准备好的主视频素材。
2.  将第一步中导出的“白色移动文字”视频，以画中画形式导入，并覆盖在主视频上方。
3.  调整画中画图层的位置和大小，使其与主视频画面完全重合。
4.  选中画中画图层，在工具栏中找到并点击 **`混合模式`** 选项。
5.  在混合模式列表中，选择 **`变暗`** 模式。此时，白色文字部分将变为透明，透出下方的视频画面，形成镂空效果。
6.  将这段合成后的视频导出备用。我们将其称为“镂空文字效果”。

## 第四步：合成最终蒙版动画 🎞️

最后一步，我们将把所有元素组合起来，并应用蒙版动画，完成片头效果。

![](img/165c9c1cbfd7061824310d40bbdf2af6_18.png)

1.  再次新建一个项目，重新导入你的主视频素材。
2.  将第二步导出的“红字绿底素材”以画中画形式导入，并调整其位置与主视频对齐。
3.  选中“红字绿底”画中画图层，在工具栏中找到 **`色度抠图`** 功能。
4.  使用取色器选取绿色背景部分，调整强度等参数，将绿色背景完全去除，只保留红色的移动文字。
5.  导入第三步导出的“镂空文字效果”视频，也以画中画形式添加，并确保其与主视频对齐。此时，你应该能看到三层画面：最底层是主视频，中间是红色文字，最上层是镂空文字。
6.  选中顶层的“镂空文字效果”图层，将时间轴移动到第1秒的位置，在此处添加一个**关键帧**。
7.  在同一个时间点（第1秒），找到并点击 **`蒙版`** 工具，选择 **`镜面蒙版`**。
8.  在画面中，将镜面蒙版调整到最小，缩成一条细线，并放置在画面左侧。
9.  将时间轴移动到第3秒的位置，软件会自动记录蒙版状态。此时，将镜面蒙版旋转**180度**，并向外拖动放大，直到完全移出画面之外，覆盖所有文字。
10. 将时间轴移动到第5秒的位置，再次调整蒙版，将其缩小回一条细线（可以放置在画面右侧，与开头呼应）。
11. 将“镂空文字效果”图层在5秒之后多余的部分裁剪删除。
12. 最后，可以为“镂空文字效果”图层添加一个“出场抖动”或“震动”类的动画特效，增强动感。

## 总结 🏁

本节课我们一起学习了制作动态文字蒙版片头的完整流程。我们首先创建了移动的文字动画，然后通过混合模式制作镂空效果，再利用色度抠图处理彩色文字，最后通过**关键帧**控制**镜面蒙版**的缩放与移动，实现了文字逐渐显现又变化的视觉动画。

![](img/165c9c1cbfd7061824310d40bbdf2af6_23.png)

掌握这个技巧后，你可以灵活运用蒙版和关键帧，为你的小红书视频设计出各种独具创意的开场效果。
