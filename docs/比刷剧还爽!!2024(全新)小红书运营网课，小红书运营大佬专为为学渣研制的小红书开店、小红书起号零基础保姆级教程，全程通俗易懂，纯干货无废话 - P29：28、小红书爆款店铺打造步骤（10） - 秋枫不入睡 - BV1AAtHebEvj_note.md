店铺进入中期，意味着前期的基础规划、笔记发布和数据积累已经完成，店铺开始获得稳定的流量。此时，如果店铺增长停滞不前，就需要启动中期规划，其核心在于理解和优化小红书的商品排名系统。

小红书店铺的中期排名优化主要涉及六个方面，它们共同决定了商品在平台内的曝光和排序。

以下是中期规划需要关注的六大排名维度：

1.  **综合排名**
2.  **价格排名**
3.  **销量排名**
4.  **关键词排名**
5.  **个性化搜索**
6.  **动态与静态PK**

### 综合排名分类 🏆
综合排名是小红书默认的商品排序依据，它根据店铺、账号和商品的综合表现进行加权计算。

**店铺综合权重 = 账号权重 + 店铺权重 + 商品权重**

*   **账号权重**：影响笔记的基础曝光量。
*   **店铺权重**：影响用户搜索时店铺的排名位置。
*   **商品权重**：决定商品在商品池内的展示机会。

综合排名具体由12项指标构成，它们直接关联你的账号、店铺和商品。

以下是综合排名的12项具体指标：

*   人气
*   销量
*   信誉
*   价格
*   下架时间
*   标题优化
*   转化率
*   收藏率
*   回头客比率
*   好评率
*   交易笔数
*   物流与服务体验

![](img/4aef9bd187ccfe4cd939d18cdfd3b3b2_10.png)

### 价格排名 💰
价格排名并非指价格越低排名越高，而是一个复杂的评分体系。每个细分项做到位可以加分，做得不好则可能扣分。

![](img/4aef9bd187ccfe4cd939d18cdfd3b3b2_13.png)

价格排名包含多个评估小项，每一项都关乎中期运营的细节策略。

![](img/4aef9bd187ccfe4cd939d18cdfd3b3b2_15.png)

以下是价格排名涉及的主要维度：

*   价格竞争力
*   价格真实性
*   价格稳定性
*   价格与销量的平衡
*   消费者心理
*   促销活动
*   价格策略
*   产品测款

### 销量排名 📈
销量排名、价格排名和关键词排名三者没有绝对的先后顺序，都是综合计算的一部分。但通常，**关键词排名带来的权重增益最大，销量排名次之，价格排名再次之**。

然而，销量排名虽然权重高，但提升难度最大；价格排名相对容易优化；关键词排名则是运营的核心与难点。

在销量之外，精准地获取流量离不开关键词优化。

### 个性化搜索与动态/静态PK ⚔️
个性化搜索以及动态PK、静态PK属于中期规划里的进阶内容，涉及到与同行店铺的直接竞争。

**个性化搜索**：指平台根据用户的历史行为（如浏览、点赞、收藏）为其推荐个性化商品列表。
**动态/静态PK**：可以理解为与竞争对手在特定关键词或流量池中的排名角逐。

如果对前四项基础排名没有深入理解，将很难看懂后两者的玩法和策略。只有打好基础，才能进行有效的市场竞争。

## 中期规划操作路径总结
本节分享旨在让大家明确小红书店铺中期运营的重点和路径。

中期规划的操作应遵循以下步骤：
1.  **理解并优化综合排名**的12项指标。
2.  依次优化**价格排名**、**销量排名**和**关键词排名**。
3.  在前三项稳固的基础上，研究**个性化搜索**的推荐机制。
4.  最后，介入**动态与静态PK**，与同行展开竞争。

完成这一整套中期规划，意味着你的产品达到了当前运营模式下的“上限”。此后，店铺的发展方向将指向两个选择：**开设新店（多店模式）** 或 **加大付费推广力度（单店模式）**。

当店铺通过中期规划进入高层级（如第五、第六层级）后，必须通过付费推广来突破流量天花板，抢占市场份额，并为店铺购买“保险”（即通过付费稳定流量和排名）。
