上一节我们介绍了视频剪辑的整体流程，本节中我们来看看具体的操作。首先，需要在手机上找到并打开剪映应用。手机端和电脑端的剪映图标是一致的。

![](img/d0f1604788a451a148ece28a5d2ee469_6.png)

![](img/d0f1604788a451a148ece28a5d2ee469_7.png)

打开软件后，会看到主界面有一个“开始创作”的加号按钮。点击这个加号，从手机相册中选择一个准备好的视频素材，将其添加到项目中。

![](img/d0f1604788a451a148ece28a5d2ee469_9.png)

## 添加视频背景

![](img/d0f1604788a451a148ece28a5d2ee469_10.png)

![](img/d0f1604788a451a148ece28a5d2ee469_11.png)

视频素材本身通常不包含合适的主题背景，因此需要手动制作。在视频轨道右侧，可以找到一个加号按钮。

以下是添加背景的步骤：
1.  点击轨道右侧的加号。
2.  在出现的菜单中，选择“素材库”。这是剪映软件自带的资源库。
3.  在“热门”分类中，可以看到多种风格的背景素材。
4.  建议选择简单明了、能清晰传达视频主题的背景。例如，可以选择一个纯黑色的背景并添加。

## 制作视频标题

背景添加完成后，需要为视频添加标题。在手机界面左下角找到“文字”选项。

![](img/d0f1604788a451a148ece28a5d2ee469_16.png)

以下是添加标题的步骤：
1.  点击“文字”，然后选择“新建文本”，会出现一个文本框。
2.  在文本框中输入视频的主题。例如，如果制作关于“沈腾”的搞笑视频，标题可以是“沈腾综艺爆笑名场面”。请确保视频素材中包含主题人物。
3.  初始的白色文字可能不够醒目。可以点击“文字模板”，直接套用预设的样式，这样更快捷方便。模板有免费和付费之分，可根据需要选择。
4.  选择满意的模板后，点击右上角的对勾确认。
5.  最后，手动拖动文本框右下角，将标题放大到合适的尺寸，使其更清晰。

## 为标题添加配音

![](img/d0f1604788a451a148ece28a5d2ee469_20.png)

![](img/d0f1604788a451a148ece28a5d2ee469_21.png)

视频需要有声音来吸引观众。如果不会自己配音，可以使用剪映的“文本朗读”功能。

![](img/d0f1604788a451a148ece28a5d2ee469_22.png)

以下是添加配音的步骤：
1.  选中标题文本，在下方工具栏中找到“文本朗读”功能。
2.  系统提供多种音色，如方言、萝莉音、播音腔等。目前方言类配音比较有特色，能吸引更多注意。
3.  试听并选择一个喜欢的声音，例如“广西话”方言。
4.  选择后点击对勾，系统会自动生成音频并下载。
5.  生成后，可以播放试听效果。接着，选中黑色背景轨道，手动将其时长缩短，使其与配音的时长保持一致，实现“音画同步”。

## 调整视频比例

视频比例是平台审核的重要标准。除了抖音平台使用 **9:16** 的竖屏比例，其他主流平台（如B站、西瓜视频）均要求 **16:9** 的横屏比例。

![](img/d0f1604788a451a148ece28a5d2ee469_26.png)

如果导入的视频素材不是16:9的横屏，需要进行调整。

**方法一：直接调整比例**
1.  点击下方工具栏中的“比例”选项。
2.  选择“16:9”的横屏比例。
3.  观察视频画面是否自动适配。如果能适配，则调整完成。

**方法二：裁剪与模糊背景（适用于方法一无效时）**
如果选择16:9比例后视频画面没有变化或出现黑边，则需要手动裁剪并填充背景。

![](img/d0f1604788a451a148ece28a5d2ee469_28.png)

![](img/d0f1604788a451a148ece28a5d2ee469_29.png)

以下是手动调整的步骤：
1.  选中视频片段，点击下方“剪辑”选项，进入“编辑”菜单，选择“裁剪”。
2.  在裁剪时，建议裁掉视频自带的原始字幕（后续可以自己添加），并且边框不要留太多，以免产生难看的黑边。
3.  裁剪完成后点击对勾。此时视频可能仍不是完美的16:9。
4.  点击下方工具栏的“背景”选项。
5.  选择“画布模糊”功能，并应用第一个模糊度选项。
6.  模糊的背景会自动填充视频两侧的空白区域，从而形成一个完整的16:9横屏画面。
