# AI新媒体运营课程：P15：01.10.03：AI训练与爆款笔记产出 🚀

## 概述
在本节课中，我们将学习如何训练AI（以ChatGPT为例）分析爆款笔记，并最终生成具备爆款潜质的新笔记。整个过程包括：向AI“投喂”资料、引导其分析爆款因素、整合学习成果，并最终指令其产出符合平台风格和用户需求的文案。

上一节我们介绍了通过提供资料让AI学习特定平台风格的方法。虽然以旅游攻略为例的成果可能不完美，但方法是正确的。本节中，我们来看看如何深化这一过程，通过分析具体的爆款内容，让AI掌握创作爆款笔记的核心要素。

## 第一步：明确分析指令与要点
要让AI有效分析，我们需要给出清晰、具体的指令。核心分析要点应包括以下几个方面：

![](img/e0ba9e8cbad757f5a24e04b7158a2fa2_5.png)

![](img/e0ba9e8cbad757f5a24e04b7158a2fa2_7.png)

以下是需要AI重点分析的几个维度：
1.  **爆款因素**：分析内容为何能成为爆款，例如是否引发了情感共鸣、提供了实用价值等。
2.  **整体结构**：
    *   **开篇**：用什么方式（如提问、痛点描述、名人引言）吸引用户继续阅读。
    *   **中间**：提供了哪些内容来戳中用户痛点，满足需求，提升用户停留时长。
    *   **结尾**：如何设置“钩子”（如鼓励互动、提供联系方式）促使用户私信、关注或评论。
3.  **关键词策略**：分析笔记中是否刻意植入了特定关键词。如果有，需指出关键词是什么、出现频率如何，以及是如何在文案中布局的。

![](img/e0ba9e8cbad757f5a24e04b7158a2fa2_9.png)

![](img/e0ba9e8cbad757f5a24e04b7158a2fa2_11.png)

## 第二步：提供爆款笔记作为分析样本
我们需要寻找目标领域（例如“考研”）的真实爆款笔记，将文案内容提供给AI进行分析。

![](img/e0ba9e8cbad757f5a24e04b7158a2fa2_15.png)

![](img/e0ba9e8cbad757f5a24e04b7158a2fa2_17.png)

![](img/e0ba9e8cbad757f5a24e04b7158a2fa2_19.png)

我们以一篇小红书平台上点赞量3000+的考研笔记为例。将文案内容复制后，向ChatGPT发出整合了上述要点的分析指令。

![](img/e0ba9e8cbad757f5a24e04b7158a2fa2_21.png)

![](img/e0ba9e8cbad757f5a24e04b7158a2fa2_23.png)

![](img/e0ba9e8cbad757f5a24e04b7158a2fa2_24.png)

**示例指令如下：**
```
我给你提供一篇小红书平台上的考研爆款笔记文案，请你分析：
1.  它的爆款因素有哪些？
2.  文案的整体结构：开篇如何吸引用户？中间用什么内容戳中痛点？结尾如何引导互动？
3.  分析该笔记是否有结合中文分词法设计关键词。如果有，请指出关键词及其出现频率。
```
AI给出了分析结果，指出该笔记的爆款因素包括：以“焦虑情感”引发共鸣、分享“逆袭故事”和经验、使用表情符号增加亲和力、结尾传递正能量等。同时，AI判断该笔记没有明显设计的关键词。

![](img/e0ba9e8cbad757f5a24e04b7158a2fa2_26.png)

![](img/e0ba9e8cbad757f5a24e04b7158a2fa2_28.png)

为了让AI学习更全面，我们应提供多篇不同特点的爆款笔记进行重复分析。例如，另一篇引用“罗翔老师”的笔记，AI分析出其利用了“名人效应”和“情感认知转折”作为爆款点。

## 第三步：整合与总结爆款因素
在提供了足够多的样本后，我们需要引导AI对分析过的所有爆款因素进行整合、去重，形成一个清晰的列表。

![](img/e0ba9e8cbad757f5a24e04b7158a2fa2_30.png)

![](img/e0ba9e8cbad757f5a24e04b7158a2fa2_32.png)

我们可以向AI发出如下指令：
```
请将你前面分析过的几篇考研爆款笔记的爆款因素总结一下，整合成一个清晰的列表。
```
AI反馈的总结可能包括：情感共鸣、逆袭故事、明确的备考计划、针对性建议、具体资源推荐、时间节点管理、正能量鼓励等。需要注意的是，AI有时会“遗忘”或遗漏之前分析过的某些点（如“使用表情符号”），这时可能需要我们提醒它回顾上下文。

## 第四步：基于学习成果生成新笔记大纲
当AI掌握了爆款要素后，我们就可以指令它开始创作。首先，让它根据学习到的爆款因素，规划一篇新笔记的大纲。

**指令示例：**
```
请结合你前面分析的爆款笔记因素，为我规划一篇关于考研的小红书笔记大纲。大纲需要细分到每个部分的要点。
```
AI生成的大纲可能包含：吸引人的开篇、主题介绍、分科目的备考规划（政治、英语、数学、专业课）、时间与心态管理、互动引导等部分。这个大纲融合了之前分析的爆款结构。

## 第五步：根据大纲生成完整文案
最后，我们要求AI根据确定的大纲，撰写完整的文案，并再次强调需融入爆款元素。

**最终生成指令：**
```
请按照上述大纲，撰写完整的文案。要求：
1.  结构严格遵循爆款笔记因素。
2.  包含一个简单的“逆袭小故事”引入，以引发读者共鸣。
3.  文案中要自然地使用一些emoji表情。
4.  尝试以中文分词法的思维融入“考研”等相关关键词。
```
AI根据指令生成的文案，会包含情感化的开篇、具体的备考建议、时间规划以及鼓励互动的结尾。虽然文案在细节上可能仍有优化空间，但对于一般运营需求来说，它已经是一个结构完整、要素齐全的合格初稿。

## 进阶提示：使用GPTs定制专属助手
在对话过程中，ChatGPT（特别是3.5版本）可能会遗忘之前的指令或上下文。为了获得更稳定、个性化的输出，可以利用OpenAI的**GPTs**功能创建专属助手。

你可以：
*   在GPTs的“知识库”中上传多篇符合你要求的小红书文案范例、表情包使用规范等文件。
*   在“指令”栏中永久性地设定好分析要点、文案风格、关键词要求等。
*   这样就能打造一个更懂你需求的私人AI运营助手，避免在每次对话中重复指令。

## 总结
本节课我们一起学习了利用AI辅助创作爆款笔记的完整流程：从**下达明确的分析指令**，到**投喂多篇爆款样本**，再到**整合爆款因素**并**生成大纲与文案**。关键在于通过精准的指令，引导AI学习并模仿爆款内容的结构、情感和策略。虽然当前结果可能需要人工微调，但这套方法能极大提升内容创作的效率。后续课程将继续分享AI在短视频、直播等领域的运营技巧。
