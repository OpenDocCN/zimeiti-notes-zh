看完视频后，请问这是**强人设**还是**弱人设**视频？

![](img/f1788f24453c20adbe4a2542cb3315e7_31.png)

答案是：**弱人设**。

![](img/f1788f24453c20adbe4a2542cb3315e7_33.png)

![](img/f1788f24453c20adbe4a2542cb3315e7_35.png)

为什么它是弱人设呢？主要有两个原因：
1.  视频内容（关于阶层和命运的讨论）与发布者本人（画面中的剃胡子过程）没有直接关系。视频讲述的是普遍观点，而非个人故事或经历。
2.  视频的画面（剃胡子）和声音（讲述观点）是分离的，讲述者并未出镜表达这些观点，人设感很弱。

![](img/f1788f24453c20adbe4a2542cb3315e7_37.png)

那么，为什么这样一个“弱人设”视频能火呢？关键在于它实现了 **“双向吸引”** 。

![](img/f1788f24453c20adbe4a2542cb3315e7_39.png)

![](img/f1788f24453c20adbe4a2542cb3315e7_41.png)

以下是该视频吸引用户的两个核心点：

![](img/f1788f24453c20adbe4a2542cb3315e7_43.png)

1.  **吸引视觉兴趣点**：一部分用户被“剃胡子”这个解压、治愈的画面所吸引，单纯为了观看这个过程而停留。
2.  **吸引内容共鸣点**：另一部分用户被视频中关于“穷人如何改变命运”的观点所触动，产生了强烈的共鸣，认为说得很好。

![](img/f1788f24453c20adbe4a2542cb3315e7_45.png)

这种“画面吸引一类人，内容吸引另一类人”的策略，大大拓宽了视频的受众面，从而获得了更多的流量。

![](img/f1788f24453c20adbe4a2542cb3315e7_47.png)

![](img/f1788f24453c20adbe4a2542cb3315e7_49.png)

理解了“双向吸引”的原理后，我们可以将其应用到自己的视频创作中。以下是具体的应用思路：

![](img/f1788f24453c20adbe4a2542cb3315e7_53.png)

*   **对于宝妈博主**：如果你拍摄宝宝日常或做饭视频没人看，可以尝试为画面配上有趣的育儿故事或家庭趣事。
    *   **公式**：`日常画面 (吸引视觉) + 共鸣故事/观点 (吸引思想) = 双向吸引`
*   **对于实体店商家**：如果你拍摄产品或工厂视频没人看，可以尝试配上你的创业经历、行业故事或人生感悟。
    *   **代码逻辑**：
        ```python
        if 视频内容 == “单纯展示产品”:
            吸引力可能不足
        elif 视频内容 == “产品展示 + 个人故事/专业观点”:
            实现双向吸引，覆盖更广人群
        ```

这种方法的核心在于，让**视觉元素**和**内容观点**分别服务于不同兴趣的用户，从而提升视频的完播率和互动率。

![](img/f1788f24453c20adbe4a2542cb3315e7_55.png)

本节课中我们一起学习了一个典型的“弱人设”视频案例。我们分析了其“弱”在何处，更重要的是，我们掌握了它之所以能成功的秘诀——**“双向吸引”** 策略。

总结来说，制作一个具有传播力的视频，不一定非要依赖强烈的个人形象。通过将**具有吸引力的画面**与**能引发共鸣的观点或故事**相结合，你可以同时吸引两类观众，有效提升视频的流量。许多视频完播率不高，可能只是缺少这样一个小技巧。抖音运营本身就是一个需要不断学习和测试的行业。
