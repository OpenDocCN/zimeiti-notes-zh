## 工具界面与核心功能概览

进入工具界面后，首页主要提供两大核心功能：**AI成片**和**AI笔记**。导航栏位于左侧。

上一节我们介绍了工具的整体情况，本节中我们来看看“AI成片”功能的具体使用。

### AI成片功能详解

AI成片功能主要分为两个部分：

1.  **输入外部文案成片**：你可以将其他地方准备好的视频脚本或文案粘贴到输入框，工具会自动搜索素材并匹配配音，生成完整视频。
2.  **选择文章成片**：此功能需要你拥有一个已发布原创文章的百家号。关联后，你可以直接选择已发布的文章内容来生成视频。

除了上述两种方式，该工具还有一个非常强大的功能，即基于实时热点自动生成视频。

---

## 基于热点一键生成视频

在AI成片界面的右侧，工具会展示当前的热点话题列表。例如“伴娘被多人拿灭火器喷倒”、“13岁女孩被逼喝一瓶白酒”等。

以下是使用热点生成视频的步骤：

![](img/7a79bb528ddb008283b0121e79be0986_3.png)

1.  **选择热点**：点击你感兴趣的热点话题。
2.  **AI生成文案**：系统会自动根据该热点生成一段视频文案。这个过程通常只需几十秒。
3.  **检查与优化文案**：生成文案后，**务必仔细检查**内容是否合法合规、有无事实错误。这是运营者必须承担的责任。你可以将视频中最吸引人的核心爆点调整到文案开头，以提升视频点击率。
4.  **选择关键素材**：文案下方有“关键素材推荐”，这是AI从网络抓取的与热点相关的图片或视频片段。勾选合适的素材，能让最终视频更真实、丰富。
5.  **一键成片**：点击“AI一键成片”按钮，系统将根据文案和你选择的素材，自动合成视频。

![](img/7a79bb528ddb008283b0121e79be0986_5.png)

**代码示例：核心操作流程**
```plaintext
选择热点 -> AI生成文案 -> 人工检查优化 -> 选择推荐素材 -> 点击“AI一键成片”
```

## 视频的编辑与优化

![](img/7a79bb528ddb008283b0121e79be0986_7.png)

![](img/7a79bb528ddb008283b0121e79be0986_9.png)

AI自动生成的视频初稿可能在某些细节上不够完美，我们需要进行微调。

上一节我们完成了视频的自动生成，本节中我们来看看如何对视频进行编辑优化。

### 素材替换

如果AI匹配的某个视频片段不合适或缺失，可以进行手动替换。

*   **查看与搜索**：在视频预览时间轴上点击需要替换的片段，右侧会显示当前素材。你可以使用工具内置的素材搜索功能，寻找更贴切的替代内容。
*   **上传本地素材**：你也可以上传自己电脑中的本地素材进行替换。
*   **免责声明**：对于新闻热点类视频，如果使用的不是原始画面，建议在视频中添加文字声明，例如“素材来源于网络，非事件原视频”。

### 调整视频样式与音频

*   **模板**：工具提供横版、竖版等多种视频模板，可根据发布平台选择。
*   **朗读音**：可以更换配音员的声音，并调整语速。例如，对于紧张的社会新闻，可以使用语速较快、情绪更饱满的配音。
*   **背景音乐**：工具内置了分类背景音乐库，但选择有限。目前不支持上传自定义音乐。

![](img/7a79bb528ddb008283b0121e79be0986_11.png)

**一个实用的技巧**：你可以先将AI生成的视频下载到本地，然后使用专业的剪辑软件（如剪映）进行深度加工，包括替换背景音乐、去除水印、精细化剪辑等，这能让你对成品有完全的控制权。

![](img/7a79bb528ddb008283b0121e79be0986_13.png)

## 视频的下载与发布

视频编辑满意后，需要将其导出。

![](img/7a79bb528ddb008283b0121e79be0986_15.png)

![](img/7a79bb528ddb008283b0121e79be0986_17.png)

1.  点击右上角的 **“发布视频”** 按钮。
2.  系统会检查素材是否全部就位，如有缺失会提示补充。
3.  补充完毕后，点击 **“生成视频”**。
4.  等待几分钟，视频渲染完成后，页面会出现 **“下载”** 按钮。
5.  点击下载，即可将视频保存到本地。之后，你可以将其发布到任何短视频平台。

**重要提醒**：使用任何AI工具创作内容，都必须严格遵守国家法律法规和平台规则，不造谣、不传播违法信息。

![](img/7a79bb528ddb008283b0121e79be0986_19.png)

## 辅助工具：AI笔记（文案创作与润色）

除了视频生成，该工具还集成了“AI笔记”功能，这相当于一个内置的AI写作助手。

以下是AI笔记的主要用途：

*   **文案创作**：你可以输入一个主题或问题（如“地球还有多少年寿命，如何保护地球”），AI会为你生成相应的文章或观点文案。
*   **文笔润色**：将你已经写好的文案粘贴进去，AI可以帮你优化语句，使其更流畅、更有吸引力。
*   **配图建议**：生成的文案还可以让AI推荐配图。

![](img/7a79bb528ddb008283b0121e79be0986_21.png)

这个AI模型基于百度的“文心一言”开发，更符合中文使用习惯，是新媒体运营人员在文案创作方面的得力助手。
