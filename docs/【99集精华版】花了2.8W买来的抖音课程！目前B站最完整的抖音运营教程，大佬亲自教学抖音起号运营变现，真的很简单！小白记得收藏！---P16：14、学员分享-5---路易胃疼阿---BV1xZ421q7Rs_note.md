上一节我们看了学员的成功案例，本节中我们来看看一种常见的内容形式：拍段子。

想做对口型、搞笑剧情、沙雕段子或顾客趣闻这类段子内容的同学，可以在公屏上扣“1”。

拍段子并不难。例如，我有一位在工地辛苦上班的学员。他最初自己拍的搞笑视频播放量只有两三千。但经过学习后，他给我反馈说：“老师，我的播放量快要破500万了，评论和涨粉还在增加，我都不知道该怎么回复了。” 我们都希望体验这种播放量暴涨的心情。

拍段子的核心是让别人哈哈一笑，觉得有趣。这需要一定的表演成分。如果你本身特别腼腆害羞，可能不太适合。不过不用担心，后期如果你不知道拍什么，老师会提供**段子素材**和**拍摄脚本**。

**核心操作流程可以概括为：**
```
确定段子类型 -> 获取素材/脚本 -> 进行表演拍摄 -> 后期剪辑 -> 发布并互动
```
