# 抖音运营教程：P22：产品与用户兴趣标签 🏷️

在本节课中，我们将学习一种实用的选题方法——九宫格选题法。我们将以健身软件Keep为例，详细拆解如何结合产品属性和用户兴趣，生成源源不断的视频创作灵感。

上一节我们探讨了选题的重要性，本节中我们来看看如何具体操作。

## 产品标签分析 🔍

首先，我们需要分析产品本身的特点。以Keep为例，我们将其放在九宫格的中心，然后围绕它联想其产品属性。

以下是Keep可能具备的产品特点列表：
*   **线上健身软件**：这是其核心功能。
*   **专业健身教练**：平台提供专业指导。
*   **饮食搭配**：提供与健身配套的饮食建议。
*   **减肥攻略**：针对减重人群的专项内容。
*   **健身干货知识**：传授科学的健身方法与理论。
*   **打卡运动制度**：通过打卡机制鼓励用户坚持。
*   **一二线城市用户为主**：这是其主要的用户画像特征，类似于抖音与快手的用户地域分布差异，这些数据可以在百度指数等平台查到。

将以上产品相关标签填入九宫格后，我们暂时将其放在一边。

## 用户兴趣标签分析 👥

接下来，我们分析使用该产品的用户群体及其兴趣。使用健身软件的用户通常是健身爱好者、想要减肥或塑身的人。我们将“健身爱好者”放在另一个九宫格的中心。

现在，开始联想这类用户可能还对什么感兴趣。

![](img/d30ed875ca1c0e25a0eec08df08210d8_1.png)

以下是健身爱好者可能关联的兴趣标签列表：
*   **健身房**：线下健身场所。
*   **健身装备**：如紧身衣、运动鞋等。
*   **轻脂美食**：与健康、减肥相关的饮食。
*   **旅行**：热爱生活的一种表现。
*   **星座、明星、鸡汤文**：大众化的兴趣点，用于扩大用户群标签。

核心思路是：从核心的“健身爱好者”标签出发，逐步扩展到“热爱生活者”，从而覆盖更广泛的兴趣点，为选题提供更多可能性。

## 生成选题：连连看大法 ✨

现在，我们有了两个填满的九宫格：一个是产品标签，一个是用户兴趣标签。接下来就是生成具体选题的关键步骤。

我们将两个九宫格中的标签进行随机“连连看”，尝试将它们组合成一个完整的句子或主题。

例如：
*   **组合1**：饮食搭配 + 旅行
    *   可能的选题：`外出旅行时，如何享受美食又能控制体重？`
*   **组合2**：运动方法 + 明星
    *   可能的选题：`赵丽颖都在坚持的5个居家健身法，你一定要学！`
*   **组合3**：科学 + 健身房
    *   可能的选题：`别再瞎练了！用这3个科学方法，让健身房效率翻倍。`

通过这种交叉联想的方法，理论上可以衍生出无数个选题方向，从而建立庞大的素材库，解决“不知道拍什么”的难题。

## 如何优化选题？ ⚡

但是，并非所有联想出来的选题都是好选题。例如“在健身房如何科学撩妹”这类话题，虽然有趣，但可能与账号的核心价值偏离。

判断一个选题好坏的标准，需要结合目标受众的精准度、内容的实用性与吸引力，以及是否与你的账号定位相符。我们将在后续课程中详细讲解如何筛选和优化选题。
