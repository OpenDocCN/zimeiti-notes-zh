# OZON选品教程：P1：数据驱动的热销品挖掘方法 🔍

在本节课中，我们将学习一种被OZON平台大卖家广泛使用的选品方法。该方法通过分析类目数据，帮助卖家快速识别出当前的热销商品以及具有增长潜力的产品，从而为您的店铺运营提供科学的决策依据。

## 概述：数据化选品流程

上一节我们介绍了数据化选品的重要性，本节中我们将具体看看如何操作。整个方法的核心是利用工具采集并分析特定类目下的商品数据，从而做出判断。以下是完整的操作步骤。

## 第一步：确定目标类目并采集数据

首先，您需要确定一个想要分析的商品类目。我们以“女士上衣”这个类目为例进行演示。

1.  打开您选定的数据分析工具。
2.  将目标类目在OZON上的页面链接复制到工具的指定输入框中。
3.  启动工具的数据采集功能。

执行上述操作后，工具会自动爬取并整理该类目下所有商品的详细信息。

## 第二步：解读与分析数据表格

工具采集完成后，会生成一个结构化的数据表格。这个表格是选品分析的核心，它通常包含以下关键字段：

*   **商品名称**：产品的具体名称。
*   **评价**：商品获得的用户评分。
*   **30天内销量**：该商品在过去30天内的销售数量。公式可表示为：`近期销量 = 过去30天销售件数总和`
*   **销售件数**：可能指总销量或特定周期的销量。
*   **访问量**：商品页面的浏览次数。
*   **销售排名**：商品在该类目下的销售排名位置。

通过这个表格，我们可以获得该类目市场的全景视图。

## 第三步：筛选与识别机会产品

![](img/46683de67823a16d079ee2a62de959c9_1.png)

在获得数据表格后，我们就可以开始进行筛选分析，以发现机会。以下是两个主要的分析方向：

**1. 识别当前热销产品**
观察“30天内销量”和“销售排名”这两列数据。销量高、排名靠前（数字小）的商品，就是当前市场上已经验证过的热销品。

![](img/46683de67823a16d079ee2a62de959c9_2.png)

**2. 发现潜力上升产品**
寻找那些“访问量”较高，但“30天内销量”相对处于中等或上升趋势的商品。这类产品可能正处于市场需求的增长期，是值得关注的潜力股。

本节课中我们一起学习了如何利用数据工具进行OZON平台选品。我们首先确定了目标类目并采集数据，然后解读了包含销量、排名、访问量等关键信息的数据表格，最后通过分析这些数据，学会了如何筛选出当前的热销品和具有增长潜力的产品。掌握这一方法，能让您的选品决策更加精准和高效。
