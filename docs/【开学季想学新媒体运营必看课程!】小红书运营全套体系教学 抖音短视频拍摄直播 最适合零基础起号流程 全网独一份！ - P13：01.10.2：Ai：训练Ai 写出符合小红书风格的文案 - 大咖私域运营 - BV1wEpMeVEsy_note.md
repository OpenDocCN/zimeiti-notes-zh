# 小红书运营全套体系教学：P13：01.10.2：AI：训练AI写出符合小红书风格的文案 📝

## 概述
在本节课中，我们将学习如何训练AI（如ChatGPT）来撰写符合小红书平台风格的文案内容。上一节我们介绍了如何让AI生成爆款标题，本节我们将重点探讨如何让AI理解并创作出符合平台调性的正文内容。

## 从标题到内容：训练AI的核心思路
上一节我们介绍了如何让AI写出符合心意的爆款标题。这个方法不仅适用于小红书，也适用于抖音、哔哩哔哩等其他平台。每个平台都有其独特的关键词和文化，标题也有大众认可的形式。如果能将重点关键词提炼出来，整理成自己的标题库，每次需要写标题时，让AI结合你的主题和这个库来创作，就能生成符合平台风格的标题。

关于AI写标题的内容就介绍到这里。接下来，我们需要重点研究的是如何用AI写出符合爆款结构或平台风格的内容体系。

## 训练AI理解平台文案风格
大部分内容ChatGPT都能写出来，但如何写出符合平台风格的内容，需要讲究几个关键点。

![](img/4aa36507b71c3fda488161c816756499_1.png)

![](img/4aa36507b71c3fda488161c816756499_3.png)

第一个关键点是文案风格。要让AI了解你需要的风格，你必须提供案例。你需要告诉AI，你写的是这类内容，并提供一定量的他人爆款文章作为材料。AI需要从这些文章中分析出平台专用的风格化用语，例如女性风格用语、活泼少女气息的词语，或平台专用语（如小红书上常用的“宝子”等）。这些内容需要让ChatGPT通过分析材料自行学习并记忆，以便后续根据此进行创作。

以下是训练AI理解平台风格的具体步骤：

1.  **明确指令**：首先，需要明确告知AI接下来要做什么。例如，你可以输入：“接下来我会给你提供一些小红书上的爆款笔记文案。你需要根据这些文案，分析小红书平台的文案撰写风格、特定平台用语、爆款词汇、语气等。每次接收文案后，请告诉我你学习到了，并给出你对这篇文案的风格、语气、类型等相关分析。”
2.  **提供学习材料**：向AI发送精选的小红书爆款文案。最好将标题和正文分开标注（例如，“标题：”、“文案：”），并注意格式清晰。
3.  **获取分析反馈**：AI在接收每篇文案后，应给出分析反馈，说明它从中学到了什么。这能确保AI确实在学习，并给你正向的确认。
4.  **持续喂养资料**：这个训练过程不是一次性的。在同一个对话中持续提供多样化的文案资料，AI会读取并记忆前面的内容，相当于在不断扩充它的“知识库”。

![](img/4aa36507b71c3fda488161c816756499_5.png)

**核心指令示例**：
```text
指令：分析以下小红书爆款文案，学习其风格、语气、平台专用词汇和内容结构。
材料：[这里粘贴文案标题和正文]
要求：学习后，请回复“已学习”，并简要分析该文案的特点。
```

## 实操演示：以旅游攻略为例
我们以“张家界旅游攻略”为例进行简单实操。

首先，我们向AI发送指令，明确学习任务。然后，我们开始提供第一篇小红书旅游攻略文案。AI在接收后，给出了分析反馈，例如指出文案“以个人游玩经历为主线，增加了真实感和亲近感”，并分析了其语气和类型。

![](img/4aa36507b71c3fda488161c816756499_10.png)

接着，我们继续提供第二篇、第三篇不同风格或侧重点的文案（例如避坑指南、吐槽体验等），让AI持续学习。

**注意要点**：在喂养资料时，需要注意选择有特色的文案。我们的目标是让AI理解小红书平台上，种草类或分享类内容是用什么样的语气和方式表述的（如第一人称分享），以及平台特有的用语。应避免只提供通用的、流程化的内容（如单纯的“第一天、第二天”路线罗列），而要寻找能体现平台独特风格的文案。

## 让AI进行内容创作
在提供一定量的学习材料后，就可以尝试让AI进行创作。

例如，我们发出指令：“请结合你的学习，帮我写一篇具有爆款潜质的张家界5日游攻略小红书笔记。要涵盖丰富的小红书风格的emoji表情，同时配上十个符合小红书旅游赛道的爆款标题。”

AI生成的内容可能包含景点介绍、体验描述和标题建议。然而，初次生成的结果可能不尽如人意。例如，内容可能比较空泛，像诗句一样罗列景点，但缺乏具体的推荐理由、避坑建议或路线细节，这对于真正寻找攻略的用户来说参考价值不足。

**问题分析**：这通常有两个原因。一是我们提供的学习材料本身可能不够典型或深入；二是使用的AI模型能力有限（例如使用GPT-3.5与GPT-4.0会有显著差异）。对于旅游攻略这类需要具体信息的题材，更好的方法是先让AI列出所有相关景点，再针对每个景点要求其规划路线、分析特色和避坑点。

## 优化创作结果
如果对AI初次生成的标题不满意，可以结合上一节课的标题指令进行优化。你需要创建一个新的对话，或明确给出标题创作指令，例如让其结合“爆款关键词”来生成标题。

经过优化后，AI生成的标题会更贴近小红书的风格，例如出现“明星同款”、“C位夜景”、“出片指南”、“氛围感”等平台高频词汇。

**核心要点**：训练AI的关键在于通过各种形式让它理解你的具体要求。很多AI知识体系丰富，但未必是你所在领域的“专家”。我们需要做的是让它理解我们专业领域内的表达方式和受众喜好。

## 总结
本节课我们一起学习了如何训练AI撰写符合小红书风格的文案。关键在于：
1.  **提供明确指令**：告诉AI需要学习和分析什么。
2.  **喂养优质材料**：提供多样化的、具有平台特色的爆款文案作为学习样本。
3.  **分步引导创作**：先让AI学习风格，再让其进行创作，并根据结果优化指令。
4.  **区分使用场景**：创作标题和正文时，可能需要结合不同的指令集或开启新的对话以获得最佳效果。
