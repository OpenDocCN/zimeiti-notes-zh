# 抖音自媒体运营：P26：推荐机制—时间效应 ⏰

在本节课中，我们将要学习抖音推荐机制中的第三个重要概念——时间效应。上一节我们介绍了基础推荐和叠加推荐，本节中我们来看看时间效应如何影响视频的长期表现。

![](img/0cbc02a623e5aafb19e7328c58d2f031_1.png)

除了基础推荐和叠加推荐之外，抖音的推荐机制里还有一个叫做时间效应。

![](img/0cbc02a623e5aafb19e7328c58d2f031_1.png)

上图展示了推荐机制的整体流程，其中第三部分就是我们今天要讲的“时间效应”。那么，什么是时间效应呢？

不知道同学们有没有遇到过这种情况：发布了一个视频之后，在当天或第二天，它没有被推荐或推荐量很少，播放量也比较低。但是可能过了一周之后，你发现它的播放量开始上涨，甚至能持续一个月。到了一个月或两个月之后，原先发过的视频播放量又上涨了。这是为什么呢？这其实就是时间效应在起作用。

为什么会出现这种情况？我们来看一下背后的逻辑。

抖音平台有一个庞大的视频库。只要你把视频发布出去，这个视频就会存在于抖音的视频库中。也就是说，所有用户发布的视频都在这个库里。

如果第一天发布后，视频没有蹭到热点，内容也不算特别优质，它可能就静静地待在视频库里。但是，等到某一天，平台上出现了一些新的热点事件。

如果你的视频内容恰好与这个新热点比较符合，抖音的算法就会从它的视频库中，寻找原有的、符合热点趋势的库存视频，进行二次推荐。

我举个例子。比如你特别喜欢明星A，发布了一个关于明星A的视频，但今天发布后没有火。过了一段时间，明星A可能因为某件事（例如出新作品、有新闻）热度上去了。这时，你之前发布的那个关于明星A的视频，就可能因为契合了这个新热点，而被算法重新发现并进行二次推荐，重新走一遍推荐流程。

像这种时间效应，源自于贴吧等社区早期的一个概念，叫做“挖坟”。其核心逻辑是：**算法通过识别当前热点，重新去视频库中寻找历史内容，将合适的内容再次纳入流量池进行推荐**。这个过程可以用一个简单的伪代码表示：

```python
if 当前有热点事件发生:
    从视频库中搜索 与热点关键词匹配的历史视频
    if 找到匹配视频:
        将该视频重新加入推荐流程
```

所以说，同学们如果非常用心地制作视频，并且按照正确的方法操作，你的视频在未来被再次推荐的几率是比较大的。

这里我也要再强调一点：时间效应这种机制，是专属于抖音平台的。

除了抖音，快手也是一个重要的短视频平台，但两者在这一点上区别较大。快手的推荐机制叫做“择新去旧”机制。

什么是“择新去旧”机制？简单来说，就是**平台主要推荐新的视频。对于旧的视频，在发布一周甚至一个月之后，通常就不再给予推荐**。不论此时发生了什么相关的热点，平台一般都不会再去推荐旧的视频。

所以，这是抖音和快手在推荐机制上一个比较大的差别。抖音的叫“时间效应”，快手的叫“择新去旧”。大家可以稍微记一下这个区别。

本节课中我们一起学习了抖音推荐机制的第三个部分——时间效应。我们了解了它的定义：算法会根据新热点，从历史视频库中挖掘并重新推荐相关内容。同时，我们也对比了抖音的“时间效应”与快手的“择新去旧”机制的核心区别。理解这一点，有助于我们以更长远的眼光看待视频的生命周期和潜在价值。
