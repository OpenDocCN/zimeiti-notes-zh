# 小红书手机端自动化工具：P1：安装与基础设置 🚀

在本节课中，我们将学习如何安装和设置用于小红书手机端的自动化工具，包括“薯摸鱼”、“薯探长”和“薯爆单”。我们将从准备工作开始，逐步完成软件的下载、安装和基础配置。

---

## 准备工作

在开始安装之前，需要确保你的电脑环境满足基本要求。以下是安装前需要完成的步骤。

1.  **下载安装包**：首先，你需要获取自动化工具的安装文件。
2.  **安装雷电模拟器**：这些工具需要在安卓模拟器环境中运行，推荐使用雷电模拟器。
3.  **安装Python**：部分工具或脚本依赖Python环境。
4.  **配置ADB**：ADB工具用于连接电脑和模拟器，是自动化操作的基础。

---

## 安装步骤详解

上一节我们介绍了准备工作，本节中我们来看看具体的安装流程。

![](img/ed2bdceee49737e07e6c801752ce398a_0.png)

### 1. 安装雷电模拟器

雷电模拟器将提供一个虚拟的安卓手机环境。

以下是安装雷电模拟器的步骤：
*   访问雷电模拟器官网，下载最新版本的安装程序。
*   运行安装程序，按照提示完成安装。建议使用默认安装路径。
*   安装完成后，启动雷电模拟器。

### 2. 在模拟器中安装小红书

自动化工具需要针对小红书应用进行操作。

以下是在模拟器中安装小红书的步骤：
*   在雷电模拟器内置的“应用中心”或浏览器中搜索“小红书”。
*   找到官方应用并点击安装。
*   安装完成后，在模拟器桌面找到小红书图标。

### 3. 配置ADB连接

![](img/ed2bdceee49737e07e6c801752ce398a_1.png)

ADB是连接电脑和安卓设备的桥梁，必须正确配置。

以下是配置ADB连接的步骤：
*   确保雷电模拟器正在运行。
*   打开电脑的命令行工具。
*   输入命令 `adb connect 127.0.0.1:5555` 来连接模拟器。
*   连接成功后，会显示 `connected to 127.0.0.1:5555`。

### 4. 安装自动化工具

核心的自动化功能由“薯摸鱼”等工具提供。

以下是安装自动化工具的步骤：
*   解压你下载的自动化工具安装包。
*   根据工具提供的说明文档，运行主程序或安装脚本。
*   首次运行时，工具可能会请求必要的权限，请根据提示允许。

## 基础设置与连接测试

![](img/ed2bdceee49737e07e6c801752ce398a_2.png)

软件安装完成后，需要进行基础设置以确保工具能正常工作。

### 工具初始化设置

每个工具都有其特定的配置界面。

以下是常见的初始化设置项：
*   **选择设备**：在工具中确保已正确识别并连接到雷电模拟器。
*   **设置间隔时间**：为自动化操作（如点赞、评论）设置合理的延迟，模拟真人操作，避免被封禁。
*   **配置任务流程**：根据你的需求（如养号、引流、爆文监测），选择或编辑对应的自动化任务脚本。

### 连接与权限测试

在开始正式使用前，务必进行测试。

以下是测试步骤：
*   在自动化工具中点击“连接设备”或类似按钮。
*   工具应成功捕获模拟器屏幕画面。
*   尝试运行一个简单的测试任务，例如“自动刷新首页”，观察模拟器是否能够正确响应。

## 核心概念与代码示意

理解以下核心概念有助于你更好地使用和排查问题。

*   **ADB命令**：自动化工具底层通过发送ADB命令来控制设备。例如，模拟点击屏幕坐标(x, y)的命令是：
    ```bash
    adb shell input tap x y
    ```
*   **屏幕坐标**：自动化操作依赖于屏幕上的特定坐标位置，这些坐标需要在模拟器分辨率固定的情况下事先获取。
*   **操作间隔**：在自动化脚本中，两次操作之间需要加入等待时间，这通常通过代码中的 `sleep()` 函数实现。例如，等待2秒：
    ```python
    import time
    time.sleep(2)
    ```
