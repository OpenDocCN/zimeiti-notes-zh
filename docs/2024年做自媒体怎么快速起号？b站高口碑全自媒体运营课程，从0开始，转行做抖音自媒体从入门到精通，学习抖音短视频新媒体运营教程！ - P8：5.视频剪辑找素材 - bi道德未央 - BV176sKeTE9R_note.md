# 自媒体运营课程：P8：5.视频剪辑找素材 🎬

在本节课中，我们将学习如何为视频剪辑寻找素材。这是制作视频的第一步，也是至关重要的一步。我们将介绍几种主要的素材来源，并演示如何高效、合规地找到你需要的视频片段。

## 视频剪辑的类型

![](img/252a21d10cb11478ec29a8ed75806abf_1.png)

视频剪辑涵盖多种类型。以下是常见的几种：
*   **电影剪辑**：基于电影内容进行剪辑。
*   **电视剧剪辑**：基于电视剧内容进行剪辑。
*   **综艺剪辑**：基于综艺节目内容进行剪辑。

![](img/252a21d10cb11478ec29a8ed75806abf_3.png)

## 素材来源平台

上一节我们介绍了视频剪辑的类型，本节中我们来看看去哪里寻找这些素材。以下是主要的素材来源渠道：

1.  **视频平台**：所有主流的自媒体视频平台都是素材库。例如，我之前展示过的30多个自媒体平台列表中的平台，都可以用来寻找素材。
    ![](img/252a21d10cb11478ec29a8ed75806abf_1.png)
    ![](img/252a21d10cb11478ec29a8ed75806abf_3.png)

2.  **影视网站**：专门的影视网站是另一个重要来源。例如豆瓣电影这类网站。我整理了近50个无版权要求的影视素材网站，涵盖电影、电视剧等，后续可以分享给大家。

3.  **综合素材网站**：除了影视类，很多同学可能需要动漫、综艺、搞笑、旅游、美食等各类素材。我也整理了超过60个网站，覆盖图片、音效、体育、各国视频等多元素材，同样可以后续分享。

4.  **影视推广平台**：这是在抖音、快手等平台进行影视内容创作的特殊渠道。与普通视频不同，在这些平台上发布符合要求的影视剪辑内容，可以依靠播放量直接获得平台奖励。这个平台稍后会重点讲解。

## 核心：合规使用素材

找到素材后，最关键的一步是如何合规使用。**绝对不要直接下载整个视频并上传**，这种做法属于“搬运”，目前平台审核严格，95%的搬运内容都无法通过。

正确的素材使用方式是：
**从大量视频中，提取符合自己视频主题的片段，每个片段的时长建议控制在10秒左右。**
按照这个标准准备素材，在后续剪辑和平台审核时，才能最大程度避免违规。

## 确定主题与准备工具

![](img/252a21d10cb11478ec29a8ed75806abf_13.png)

素材问题解决后，就可以开始制作视频了。首先需要确定视频主题。

例如，我们决定制作一个主题为 **“沈腾搞笑名场面”** 的视频。那么所有寻找的素材片段都必须包含沈腾这个人，并围绕“搞笑”这一核心。
![](img/252a21d10cb11478ec29a8ed75806abf_15.png)

确定主题后，就需要使用剪辑软件。最常用、最易上手的工具是 **剪映** 软件。你可以根据习惯选择手机版或电脑版进行操作。
