# 亚马逊运营：P1：如何利用直评提升新品销量 🚀

在本节课中，我们将学习关于亚马逊新品推广中“直评”的核心知识。内容包括直评的作用、安全操作数量、常见问题解答以及推广策略，旨在帮助初学者理解并安全地使用直评工具。

## 概述

直评是亚马逊上一种由买家直接留下的评价，无需购买产品。它本身不增加商品权重，但能有效提升链接的视觉可信度，从而帮助提高转化率。正确使用直评是新品冷启动阶段的重要策略之一。

## 视频认证与账号安全

如果您的亚马逊账号因视频认证失败而被封禁，目前没有解决办法。视频认证必须由账号法人本人露脸完成，无法由他人代替。

如果您让他人代替进行视频认证，则必须能证明代替者与账号法人的关系。在认证过程中，回答不够流畅、犹豫或眼神飘忽都可能引起亚马逊怀疑，导致认证失败。

## 断货期间的广告策略

当主推产品断货时，不应完全关闭广告。

正确的做法是：**不要关闭广告**，但应将广告预算调到最低。主推产品仍需维持一定的推广力度。

如果产品断货时间非常长，且通过站外引流、站内广告都无法重新推起销量，建议直接放弃该产品，将其移除。

## 直评的常见问题与操作指南

以下是关于直评的多个核心问题的解答。

### 关于直评的同步与数量
*   **变体评价同步**：不推荐将日本站的变体评价同步到美国站，因为语言不同。常见的做法是在英国、加拿大等英语国家间同步评价至美国站。
*   **直评数量限制**：从安全性角度，建议每个ASIN上**4个**直评。从稳定性（即不易被删除）角度，一个父体下所有子体共享的直评总数建议在**20个以内**。具体数量需根据个人需求和风险承受能力决定。

### 关于买家号与直评质量
*   **买家号要求**：用于上直评的买家号有很高要求。一个稳定的直评号，其留下的直评总数通常不会超过5个。之后该买家号便不再使用，因为继续使用可能导致其之前留下的所有直评被连带删除。
*   **直评的作用**：直评**没有权重**，不会提升搜索排名。它的核心作用是**提升转化率**，通过增加评价数量来增强潜在买家的信任感。

### 直评的实操细节
*   **内容形式**：直评可以上传图片和视频。安全的方法是：先上文字直评，等待其稳定显示后，再通过“追评”功能补上图片或视频。
*   **价格参考**：直评市场价格差异较大。根据市场调研，常见价格在15至70美元之间，超过100美元的报价通常不合理。
*   **“质保”陷阱**：不要迷信“质保”服务。直评是否被删除取决于亚马逊系统，与服务商无关。一条直评若能稳定超过3天，通常能维持较长时间；若频繁被删，即使有质保也无济于事。
*   **操作主体**：直评只能上在具体的**子体（ASIN）** 上，不能上在父体上。父体下显示的直评数量是其下所有子体直评的共享总和。

### 直评稳定性与风险
*   **直评掉落原因**：主要取决于**买家号质量**。如果一个买家号留下的直评数量过多，会触发亚马逊审核，导致该号留下的所有直评都被删除。
*   **安全建议**：一个ASIN上4个直评是相对安全的。一个父体下的直评总数控制在20个以内，稳定性较高。超过20个，掉落风险会显著增加。

## 广告与推广策略

本节我们来看看与广告相关的常见问题。

### 广告投放问题
*   **手动精准广告无曝光**：如果手动精准广告没有曝光，首先尝试将关键词出价（CPC）拉到最高。若仍无曝光，则放弃该精准词，转为使用短语匹配模式进行测试。
*   **广告开关时间**：**不要**在白天开启广告，晚上关闭。美国时间晚上是下单高峰期，此时关闭广告会错失订单。广告应保持持续开启，可以通过调整预算来控制花费，但避免频繁开关。

### 关键词与排名
*   **关键词已自然排名靠前**：如果某个关键词已经能在搜索结果前三页找到，说明该词权重不错。此时应该为其单独开设精准广告，进一步助推其排名上升至首页。
*   **广告结构调整**：如果同一个关键词在词组匹配广告中能出单，但在精准匹配广告中只有点击没有转化，可以考虑关闭精准匹配广告，让该词在效果更好的词组匹配广告中积累权重。

## 变体合并与违规

上一节我们介绍了广告策略，本节中我们来看看变体操作的相关风险。

*   **变体违规处罚**：如果亚马逊判定您的变体合并违规（如合并僵尸链接），会**立即**拆分您的变体，不会提前通知。
*   **当前风险**：目前，普通的合规变体合并仍然可以进行，亚马逊并非100%查处。但需要**绝对避免合并僵尸链接**，此类操作目前处罚严厉，可能导致账号被封。
*   **合并成功不等于安全**：收到变体合并成功的邮件，仅代表操作成功，不意味着后续不会被系统复查和处罚。

## 新品推广与运营思路

### 新品期操作
*   **直评使用时机**：直评主要针对**新品链接**，用于帮助其度过无评价的冷启动期。老链接通常不需要上直评。
*   **不出单与直评**：新品在没有订单时，正是使用直评提升页面转化率的好时机。不要等到有订单了才考虑上直评。
*   **推广组合**：直评可以与“绿标”（Amazon‘s Choice）等标志同时存在，二者不冲突。

### 价格与竞争
*   **应对价格战**：如果所在类目发生普遍的价格战，且您的排名因此下滑，短期内可能需要选择“同流合污”，适当降价以维持排名和流量。逆势坚持高价可能导致市场份额丢失。

### 其他推广方式
*   **站外推广**：新品做站外推广（如折扣网站）并不违规，但通常效果有限，且可能打乱亚马逊对您新品期的流量分配逻辑，需谨慎使用。
*   **免评单**：免评单主要用于推动关键词排名。下单时必须通过搜索关键词进入产品页面，以模拟真实购买行为，否则异常订单比例过高会引发审核。

## 总结

本节课中我们一起学习了亚马逊直评的全面应用指南。我们明确了直评的核心作用是提升转化率而非权重，掌握了每个ASIN上4个、每个父体下总数不超过20个的安全操作范围。我们还探讨了与直评配合的广告策略、变体合并的风险，以及在新品期如何综合运用各种手段进行推广。记住，所有操作都应优先考虑账号安全，并保持对平台规则变化的敏感度。
