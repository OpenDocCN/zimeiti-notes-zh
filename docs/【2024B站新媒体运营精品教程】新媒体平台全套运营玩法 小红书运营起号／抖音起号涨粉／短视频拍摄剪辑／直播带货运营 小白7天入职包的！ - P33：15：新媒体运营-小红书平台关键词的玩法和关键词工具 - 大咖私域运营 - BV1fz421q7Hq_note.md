## 核心概念：核心关键词与长尾关键词
上一节我们介绍了内容策划的基础，本节中我们来看看关键词的分类。理解这两种关键词是进行有效内容布局的第一步。

![](img/e99ad8e27be821c77c994ffdf22b078c_13.png)

*   **核心关键词**：指代行业或产品的通用、宽泛的统称词。例如：`女装`、`奶粉`、`压缩机`。
*   **长尾关键词**：由核心关键词拓展而来的、更具体、更长、搜索意图更明确的词组。例如：`女装图片`、`女装穿搭`、`女装店`。

**核心公式**：
```
长尾关键词 = 核心关键词 + 修饰词/场景词/问题词
```
（例如：`女装` + `图片` = `女装图片`）

![](img/e99ad8e27be821c77c994ffdf22b078c_15.png)

## 关键词分析工具：洞察用户需求 🔍
理解了关键词类型后，我们需要工具来发现它们。本节将介绍如何使用关键词工具进行需求分析。

这里展示的是一个关键词工具的“需求图谱”或“词云图”功能。当我们输入一个核心词（如`眉笔`），工具会展示与之相关的所有高频长尾词，并通过字体大小直观体现搜索热度。

**工具作用分析**：
1.  **发现内容方向**：通过云图，可以清晰看到用户最关心的点。例如，搜索`眉笔`的用户，可能最关心`颜色`、`图片`、`品牌`。这为我们制作内容（如“眉笔颜色怎么选”、“眉笔效果图对比”）提供了明确方向。
2.  **覆盖小众需求**：平台热点往往集中，但通过关键词工具，我们可以挖掘那些虽不集中但真实存在的用户需求，制作内容覆盖这部分用户，获得精准关注。

![](img/e99ad8e27be821c77c994ffdf22b078c_17.png)

**核心逻辑**：工具帮助我们分析“当用户搜索A时，他们真正关心B、C、D”，从而指导我们生产B、C、D相关的内容。

## 关键词拓展方法：构建内容库 📚
掌握了分析工具，接下来我们学习如何系统地拓展关键词，为内容创作提供源源不断的选题。

![](img/e99ad8e27be821c77c994ffdf22b078c_26.png)

![](img/e99ad8e27be821c77c994ffdf22b078c_28.png)

以下是关键词拓展的完整逻辑链条：

![](img/e99ad8e27be821c77c994ffdf22b078c_30.png)

1.  **确定核心词**：从你的行业或产品出发，确定一个核心词，例如 `眉笔`。
2.  **挖掘一级长尾词**：使用工具，找到与核心词直接关联的高频需求词，如 `眉笔颜色`、`眉笔色号`、`眉笔防水`。
3.  **拓展二级长尾词**：针对每一个一级长尾词，进一步拓展。例如，针对 `眉笔颜色`，可以拓展出 `眉笔颜色怎么选`、`眉笔什么颜色好看`、`黄皮适合的眉笔颜色`。
4.  **形成选题库**：将这些拓展出的长尾关键词整理起来，每一个都可以转化为一个具体的内容选题或笔记标题。

![](img/e99ad8e27be821c77c994ffdf22b078c_32.png)

![](img/e99ad8e27be821c77c994ffdf22b078c_34.png)

**代码示例（伪代码）**：
```python
核心词 = “眉笔”
一级长尾词列表 = 工具分析(核心词) # 返回 [‘颜色’， ‘色号’， ‘防水’， …]
选题库 = []

![](img/e99ad8e27be821c77c994ffdf22b078c_36.png)

for 词 in 一级长尾词列表:
    二级长尾词列表 = 工具分析(核心词 + 词) # 如分析“眉笔颜色”
    选题库.extend(二级长尾词列表) # 将拓展结果加入选题库

print(选题库) # 输出所有可创作的内容方向
```

## 小红书关键词高级玩法 🎯
对于小红书平台，关键词的运用尤为关键。本节将揭示一个在平台内流传的特定关键词策略。

![](img/e99ad8e27be821c77c994ffdf22b078c_42.png)

![](img/e99ad8e27be821c77c994ffdf22b078c_44.png)

小红书系统通过一个叫做 **Spider（蜘蛛）** 的程序来抓取和识别笔记内容，从而判断笔记的主题并将其推荐给相关搜索用户。

**玩法：关键词堆砌**
部分运营者会在笔记的标题、正文尤其是文末，大量重复目标关键词。例如，一篇关于“瑞幸咖啡”的笔记，可能在文末不起眼处重复写入“瑞幸 瑞幸咖啡 瑞幸拿铁 瑞幸校园……”等词组。

![](img/e99ad8e27be821c77c994ffdf22b078c_46.png)

![](img/e99ad8e27be821c77c994ffdf22b078c_48.png)

*   **目的**：提高目标关键词在笔记中的“词频”，让系统Spider更容易判定该笔记与“瑞幸咖啡”高度相关，从而提升在相关搜索结果的排名。
*   **注意**：此方法可能影响用户体验，平台也不鼓励。其效果因类目而异，对于知识分享、工具测评等类目可能有一定作用，但需谨慎使用。

**历史玩法参考**：在传统网站SEO中，有过更极端的做法，如将堆砌的关键词字体颜色设置为与背景色相同，使人眼不可见但Spider可抓取。小红书目前不支持修改字体颜色，但文末堆砌的做法是其变体。
