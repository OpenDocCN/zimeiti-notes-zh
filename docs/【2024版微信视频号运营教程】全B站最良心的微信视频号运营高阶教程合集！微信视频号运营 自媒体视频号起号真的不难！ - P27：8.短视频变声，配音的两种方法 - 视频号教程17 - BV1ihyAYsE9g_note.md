# 微信视频号运营教程：P27：8.短视频变声与配音的两种方法

在本节课中，我们将学习在视频混剪时，为视频声音进行去重的两种实用方法。这两种方法能有效提升视频的原创度，是运营视频号的重要技巧。

上一节我们介绍了视频剪辑的基础操作，本节中我们来看看如何对视频的音频部分进行处理，以实现去重。

## 方法一：使用剪映内置变声功能

![](img/85a80fbcebf8021b0ba7c4f3d35977c7_1.png)

第一种方法是在剪映App内部直接对原视频的声音进行变声处理。这种方法操作简单快捷。

以下是具体操作步骤：

1.  将需要处理的视频导入剪映。
2.  选中视频轨道，在左侧功能栏中找到并点击“声音效果”。
3.  在变声效果列表中，选择你喜欢的效果，例如“男生-大叔”或“男生-小帅”。
4.  部分效果支持调节参数，你可以根据需求微调，使声音更自然。

通过这种方式，视频的人声部分会发生改变，但背景音乐保持不变。这是一种基础的音频去重手段。

## 方法二：使用配音工具重新生成配音

![](img/85a80fbcebf8021b0ba7c4f3d35977c7_3.png)

第二种方法是借助第三方配音工具，提取原视频文案并重新生成全新的配音。这种方法去重效果更彻底。

![](img/85a80fbcebf8021b0ba7c4f3d35977c7_5.png)

以下是具体操作步骤：

1.  **提取文案**：使用配音工具（如“配音神器”）的文案提取功能，从原视频中提取出文字内容。
2.  **修改文案**：对提取出的文案进行适当修改。这是关键步骤，能进一步提升原创度。
    *   **修改逻辑**：替换部分词语（如“老人”改为“大人”），调整句式（如“怎么办呢？”改为“咋办呢？”），但注意不要过多增减字数，保持原意。
    *   **示例修改**：
        *   原文：“老人给小孩削皮的时候，我们还不放心。怎么办呢？”
        *   修改后：“大人给小孩削皮的时候，我们还不放心。咋办呢？”
3.  **生成新配音**：将修改后的文案复制到配音工具中，选择一个你喜欢的声音角色（如“男生-阳光”），然后合成并导出新的音频文件。
4.  **替换音频**：回到剪映，导入原视频，关闭原声。通过“添加音频”->“提取音乐”功能，导入新生成的配音文件。
5.  **添加背景音乐**：此时视频原有的背景音乐已被移除，你可以添加新的背景音乐，并调整音量大小，使配音更突出。

![](img/85a80fbcebf8021b0ba7c4f3d35977c7_7.png)

这种方法不仅改变了人声，也替换了背景音乐，实现了更全面的音频去重。

本节课中我们一起学习了短视频音频去重的两种核心方法：**剪映内置变声**与**配音工具重新生成**。第一种方法便捷，适合快速处理；第二种方法效果更彻底，能显著提升视频原创度。在实际操作中，你可以根据视频需求灵活选择或结合使用这两种方法。掌握这些技巧，能让你的混剪视频更具独特性。
