# 抖音运营教程：P6：人物设定

![](img/a2dae5612335d1218da1e513d7282d62_0.png)

在本节课中，我们将要学习抖音账号运营中一个至关重要的环节——人物设定。一个清晰、真实且富有吸引力的人设，是账号获取流量、建立信任并最终实现变现的基础。

![](img/a2dae5612335d1218da1e513d7282d62_2.png)

## 人设策划的核心概念

![](img/a2dae5612335d1218da1e513d7282d62_4.png)

上一节我们介绍了账号定位，本节中我们来看看如何将定位具体化为一个鲜活的人物角色。

![](img/a2dae5612335d1218da1e513d7282d62_6.png)

在抖音中建立人设，类似于在游戏中创建角色，需要预先进行详细的设定，包括称呼、职业、技能等。但与游戏角色不同的是，抖音中的人物设定是**人生角色**，因此需要具备更强的**真实性**。

![](img/a2dae5612335d1218da1e513d7282d62_8.png)

无论是短视频还是直播，一个成功的IP能够解决粉丝的信任问题，起到裂变传播的作用，并具备强大的变现能力。因此，我们运营账号，无论是创作内容还是策划选题，其核心目的都是在获取流量的同时**打造IP**。

打造IP的核心，是在特定的场景或情境中，完成**账号人格化**的过程。这个过程需要注入情感、塑造故事，并贴上具有辨识度的标签。

![](img/a2dae5612335d1218da1e513d7282d62_10.png)

## 账号人格化策划框架

![](img/a2dae5612335d1218da1e513d7282d62_12.png)

账号人格化，即我们通常所说的“人设”。对于人设的策划，可以从以下六个方面系统性地着手构建。

![](img/a2dae5612335d1218da1e513d7282d62_14.png)

以下是构建人设的六个核心维度：

![](img/a2dae5612335d1218da1e513d7282d62_16.png)

1.  **名字/姓氏/昵称**
    *   一个IP首先需要一个称呼，这是IP传播的名片。它如同一个品牌的名称或商标，必须明确且易于记忆。

![](img/a2dae5612335d1218da1e513d7282d62_18.png)

2.  **职业/职务/职称**
    *   人格化必然伴随身份。这是用户认识并联想这个IP的入口，例如：老师、教授、博士、律师。

3.  **称呼/社会关系**
    *   一个恰当的称呼能快速拉近与用户的距离，产生情感共鸣。例如：XX哥、XX大爷、XX姥姥。

4.  **形象/第一印象**
    *   从用户视角塑造IP形象，使其更容易占据用户心智。例如：大叔形象、萌萌哒、铁憨憨、甜美可爱、光头。

5.  **性格/爱好/特点**
    *   一个具有鲜明特点的IP，不仅记忆点强，也更显真实。过于完美的形象反而容易脱离大众。例如：暴躁、娘炮、爱吃、爱喝酒。

6.  **穿着打扮/财富地位**
    *   借助社会普遍认知和刻板印象，可以快速在用户心智中为IP定位。例如：有钱代表会赚钱，开豪车代表成功人士，打扮土气可能代表老实人。

## 从人设到超级符号

将以上内容策划好后，即可结合自身的产品或服务优势，将其落实到账号搭建中，即完善抖音账号的“五件套”：**头像、昵称、背景图、简介、视频封面**。

除了对账号进行人格化策划，还需要通过持续的选题与内容策划，为人设提炼出独特的**记忆点**，即通常所说的“超级符号”。

超级符号的提炼可以从以下几个维度进行：

以下是提炼超级符号的几个关键方向：

*   **画风（视觉）**：如独特的场景、道具、服饰风格。
*   **听觉**：如标志性的BGM、特效音、配音或声线特点。
*   **文案**：能够反映优势且易于记忆的提炼词，例如：“简单粗暴”、“XX天花板”。
*   **表情/情绪**：如标志性的笑、惊恐、无奈等表情或情绪传递方式。
*   **原特色**：如方言、口音、错误发音、特定的说话节奏（慢条斯理、抑扬顿挫）。
*   **动作/习惯**：如习惯性的肢体动作，例如：撩头发、扶眼镜、标志性手势。

超级符号提炼完成后，在进行短视频创作时，就应不断地在视频中重复呈现这些元素，从而强化用户记忆，形成深刻的品牌认知。

## 课程总结

本节课中我们一起学习了抖音账号运营中“人物设定”的完整方法论。我们首先明确了人设的本质是**真实的人生角色**，其核心目标是实现**账号人格化**。接着，我们通过一个六维框架系统地学习了如何构建人设，并进一步探讨了如何将人设升级为具有强传播力的**超级符号**。记住，一个成功的人设是连接内容、用户与商业价值的桥梁。
