# 自媒体运营教程：P9：影视推广平台 💰

![](img/bc4beef10c12875f7a4408a57ddfd54c_1.png)

在本节课中，我们将要学习一个重要的变现渠道——影视推广平台。我们将了解如何将制作好的视频发布到抖音、快手等平台，并通过完成特定任务来结算收益，这对于希望在这些平台获得直接收入的创作者至关重要。

![](img/bc4beef10c12875f7a4408a57ddfd54c_3.png)

上一节我们介绍了视频的多平台分发策略，本节中我们来看看如何在抖音、快手等平台通过“任务”模式获得收益。

![](img/bc4beef10c12875f7a4408a57ddfd54c_5.png)

很多同学问到，如果想把视频发到抖音或快手，该如何结算收益。需要明确的是，我们剪辑完成的视频可以发布到30多个自媒体平台。除抖音和快手外，其他平台通常可以依靠视频本身的播放量直接结算收益。

但是，抖音和快手这两个平台情况特殊。就像之前提到的，视频发到这两个平台，主要作用可能仅限于增长粉丝。

![](img/bc4beef10c12875f7a4408a57ddfd54c_7.png)

是否有同学只想专注于做抖音和快手这两个自己熟悉的平台？同时，是否不仅想涨粉，还想依靠播放量来结算收益？答案是肯定的。

对于这类需求，视频可以发布到以下四个平台：抖音、快手、小红书以及微信视频号。这四个都是大家日常可能频繁使用的平台。小红书在年轻用户，特别是女性用户中较为流行。微信视频号则内嵌于微信应用中，人手一个微信即可发布。

在这几个平台发布视频后，我们需要通过“影视推广平台”来接取任务，并通过完成任务来结算奖金。平台上的任务素材非常丰富。

以下是影视推广的核心概念：
*   **影视推广**：指为电视剧、电影或综艺节目进行线上宣传。如今，许多制片方或广告商会利用推广平台，让创作者制作视频帮助宣传，并依据视频效果（如播放量）结算奖金。

每个任务都需要提交一个“回填链接”进行报名。在链接中，需要填写以下信息：
*   个人微信号和账号昵称。
*   所使用的抖音或快手ID。
*   所推广的具体电视剧/电影名称。
*   发布作品的平台链接。
*   作品发布时间。
*   作品播放量的截图。

提交完整的回填链接后，方可等待收益结算。影视推广的收益通常比单纯依靠平台播放量分成的收益更高。

我们可以看一下不同平台的收益计算规则：

以下是各平台影视推广任务收益计算示例：
*   **抖音**：每1万播放量，奖金为 **50元**。（对比：头条号每万次播放收益约为34元）
*   **微信视频号**：每1万播放量，奖金为 **10元**。单价较低是因为其用户基数极大，触达容易。
*   **小红书**：每1000播放量，奖金为 **30元**。折算下来单价最高。
*   **额外奖励**：当视频播放量超过基础任务要求后，每增加1万播放量通常还会有额外奖金。

此外，对于搜狐短剧等任务，结算方式可能不同。例如，有些任务要求抖音和快手两个平台都发布，并按**点赞数**结算，且对账号粉丝数无要求。

以下是按点赞结算的示例：
*   **抖音**：单条作品获5000赞奖励 **100元**，1万赞奖励 **200元**。
*   **快手**：单条作品获1万赞奖励 **180元**。

![](img/bc4beef10c12875f7a4408a57ddfd54c_9.png)

本节课中我们一起学习了影视推广平台的运作模式。我们了解到，在抖音、快手等平台，可以通过接取特定的影视推广任务，并依据播放量或点赞数来获得比普通流量分成更高的收益。关键在于正确提交任务回填链接，并理解不同平台的结算规则。这为创作者在这些平台实现直接变现提供了清晰路径。
