# 抖音运营教程：P21：新手如何做好内容定位 📝

在本节课中，我们将学习一套高效的内容选题公式，帮助你快速确定视频方向，解决“不知道发什么”的难题，从而更精准地吸引目标粉丝。

![](img/af001bc48062d467cc1eb244c1b392b6_1.png)

---

你现在为抖音流量发愁，往往是因为选题时思路不清晰。是否经常为想选题而烦恼，甚至不知道明天该发布什么内容？本节将教你一套实用公式，一分钟内帮你解决这个困扰。

会寻找选题的人，能将更多时间投入视频制作。每天发布三条视频，一个月就能积累九十条内容。依靠数量优势就能在竞争中脱颖而出，甚至无需额外投放广告。此外，擅长选题的人能更精准地把握用户痛点，也更容易吸引精准粉丝观看。

![](img/af001bc48062d467cc1eb244c1b392b6_3.png)

这有助于账号快速被打上正确标签。那么，有什么方法可以解决选题难题呢？

当然有。这是一套百万粉丝博主常用的选题技巧。为方便理解，我们将其总结为一个公式：

![](img/af001bc48062d467cc1eb244c1b392b6_5.png)

**内容定位 + 用户痛点 + 借势热点**

用好这个公式，你的内容方向就不会偏离。例如，如果你想和我一样成为一名知识博主，就需要挖掘行业内的关键词，例如：播放量、粉丝量、定位、选题等。这些关键词本身是客观的，不包含主观描述。

然后，在此基础上寻找用户的痛点，例如：便捷、高效、技巧教学、内容输出等。最后，结合近期的热点话题，将这些元素进行任意组合即可。

让我们将这个公式套用到其他行业试试。以美妆行业为例，内容定位可以是：美白、除皱、口红。

用户痛点可能是：平价好用、明星同款、避坑扫雷。最后，借势热点进行排列组合，就能产生诸如“迪丽热巴同款美白面膜推荐”、“七夕千万别送女朋友口红”等具体选题。

![](img/af001bc48062d467cc1eb244c1b392b6_7.png)

以上我们已经举了两个例子。至于举一反三，就交给你了。你可以根据自己所在行业的特性，将你的思考过程实践出来。
