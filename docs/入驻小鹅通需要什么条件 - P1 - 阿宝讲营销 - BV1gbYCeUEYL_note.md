# 小鹅通入驻指南：P1：入驻条件详解

在本节课中，我们将详细学习入驻小鹅通平台所需的具体条件。无论您是个人、企业还是其他类型的机构，只要具备内容输出能力，都可以利用小鹅通来销售知识类商品，过程非常便捷。

## 🧑‍💼 主体类型与资质要求

上一节我们概述了小鹅通平台的通用性，本节中我们来看看不同入驻主体需要准备的具体材料。根据您的身份，所需条件有所不同。

以下是不同主体类型对应的资质要求列表：

*   **个人入驻**：仅需提供**本人身份证**。
*   **公司/企业入驻**：需要提供**企业营业执照**以及**法人身份证信息**。
*   **教育机构等其他组织入驻**：需要提供**机构负责人信息**以及**相关行业的资质证明**。

## 📦 特殊商品类目补充资质

如果您计划销售知识类商品以外的实物商品，则可能需要根据商品类目补充特定的行业资质。

例如，如果您销售的是食品，根据规定就必须补充**食品安全许可证**。在准备过程中，请务必根据您所售商品的具体类型，提前咨询并备齐所有必要的证明文件。
