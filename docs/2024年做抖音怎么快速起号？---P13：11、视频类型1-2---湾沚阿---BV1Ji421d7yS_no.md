# 抖音运营实战：P13：11、视频类型1-2 - 人设与口播详解

在本节课中，我们将深入探讨抖音账号运营中至关重要的一环——**人设**的建立，并详细介绍一种核心的视频内容形式：**口播**。理解并掌握这些概念，是连接内容创作与粉丝增长、实现变现的关键步骤。

上一节我们解决了文案问题，本节中我们来看看如何通过建立人设来提升账号辨识度，并学习如何制作有效的口播视频。

## 人设：账号的灵魂与记忆点

从涨粉到变现并非一条直线，中间存在关键一环：**建立人设**。人设的目的并非炫耀，而是为了在众多同类账号中脱颖而出。例如，有十个人都做装修内容，如果其中一位（比如“小王同学”）建立了幽默搞笑的人设，观众就更容易记住他，从而忽略其他九位。当观众喜欢你、认可你时，自然就会成为你的粉丝。

以下是两种主要的人设类型，理解它们的区别有助于你定位自己的账号。

### 强人设

**强人设**的核心是**以创作者个人为切入点**，内容与创作者自身密切相关。其公式可以概括为：

![](img/62598f6d557c57fa85e058ee17848f29_1.png)

**强人设 = 个人经历/专业/兴趣 + 真人出镜（通常）**

![](img/62598f6d557c57fa85e058ee17848f29_3.png)

具体表现形式包括：
*   **分享行业经验**：例如装修、办公、房产、汽车、餐饮、婚介、法律咨询、电商等领域的从业者分享专业知识。
*   **分享兴趣爱好**：例如美食、旅行、手工、宠物、星座占卜、美妆护肤、服装穿搭、游戏、摄影等爱好者分享相关生活。

这类内容通常需要真人出镜，因为核心是展示“你”这个人和你的世界。

### 弱人设

**弱人设**则与创作者本人关系不大，核心目标是**运营好账号本身**。其公式可以概括为：

**弱人设 = 主题内容（如文案、影视） + 非个人化呈现**

具体例子包括：
*   **情感语录账号**：发布配乐的情感文字图片或视频，无真人出镜或配音。
*   **影视解说账号**：虽然有人声解说电影，但观众关注的是电影内容和解说本身，而非解说者个人。

区分清楚强人设与弱人设，是规划账号内容方向的基础。

## 内容创作的核心形式：口播详解

只要内容与你个人相关，都可以制作成视频。我们首先详细讲解一种高效且易操作的形式：**口播**。

**口播**是指面对镜头进行讲述的视频形式。其核心代码结构如下：
```plaintext
口播视频 = 精心准备的文案 + 面对镜头的讲述（可配字幕）
```

口播主要分为两大类，目的各不相同：

1.  **讲故事类口播**：分享客户故事、从业经历、人生感悟等。目的是让观众觉得**有趣、有共鸣**。故事本身不要求完全真实，可以从文案平台寻找素材。
2.  **教知识类口播**：分享育儿、金融、养生、装修、房产、汽车等领域的专业知识。目的是让观众在几十秒内获得**收获感**，觉得学到了东西。

无论你从事哪个领域，如法律、保险、金融、装修等，都可以采用口播形式。它的优势在于拍摄简单（一部手机即可），且内容依赖提前准备的文案，而非临场发挥。

以下是针对常见顾虑的解答：

*   **口才不好能做口播吗？**  
    **可以。** 口播使用的是提前准备好的稿子，对着镜头朗读或讲述即可，并非即兴演讲。
*   **普通话不标准能做口播吗？**  
    **可以，有时效果更佳。** 如果你的目标客户是同城或有地域属性，使用方言会显得更亲切、有趣，反而能增强吸引力。同时，可以通过添加字幕来解决听众的理解问题。

本节课中我们一起学习了抖音账号运营中“人设”的重要性及其两种类型（强人设与弱人设），并深入剖析了“口播”这种核心视频形式的定义、分类与实操要点。建立清晰的人设是让观众记住你的关键，而口播则是将你的专业知识或个人特色有效传达出去的有力工具。掌握这些基础，你的内容创作将更有方向。
