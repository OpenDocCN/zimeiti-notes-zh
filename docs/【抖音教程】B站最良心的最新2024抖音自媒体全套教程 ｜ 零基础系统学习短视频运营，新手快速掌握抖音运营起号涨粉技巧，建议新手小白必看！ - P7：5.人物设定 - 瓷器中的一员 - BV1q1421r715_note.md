# 抖音运营教程：P7：人物设定

![](img/57b834854b1abd46d920dedea9a5b1f3_0.png)

在本节课中，我们将要学习抖音账号运营中一个至关重要的环节——**人物设定**。我们将探讨如何像设计游戏角色一样，为你或你的账号打造一个真实、立体且富有吸引力的人格化形象，从而解决粉丝信任、促进传播并增强变现能力。

![](img/57b834854b1abd46d920dedea9a5b1f3_2.png)

## 概述：什么是账号人设？

![](img/57b834854b1abd46d920dedea9a5b1f3_4.png)

上一节我们介绍了账号定位的基础，本节中我们来看看如何将定位具体化为一个鲜活的“人”。在抖音中策划人设，就像在游戏中预先设定角色一样，需要规划其称呼、职业与技能。

![](img/57b834854b1abd46d920dedea9a5b1f3_6.png)

但与游戏角色不同的是，我们在抖音设定的是**人生角色**，因此必须更具真实感。无论是短视频还是直播，一个成功的IP能有效解决粉丝信任问题，起到裂变传播的作用，并具备强大的变现潜力。

![](img/57b834854b1abd46d920dedea9a5b1f3_8.png)

因此，我们运营账号，无论是创作内容还是策划选题，核心目的之一就是在获取流量的同时**打造IP**。打造IP的核心，就是在特定场景或情境中，完成**账号人格化**的过程。

![](img/57b834854b1abd46d920dedea9a5b1f3_10.png)

## 如何进行账号人格化策划？

![](img/57b834854b1abd46d920dedea9a5b1f3_12.png)

账号人格化，即我们通常所说的“人设”。这个过程需要注入情感、塑造故事并贴上鲜明的标签。对于人设的策划，我们可以从以下几个具体方面着手。

![](img/57b834854b1abd46d920dedea9a5b1f3_14.png)

以下是策划账号人设的六个核心维度：

![](img/57b834854b1abd46d920dedea9a5b1f3_16.png)

1.  **名字/昵称**
    一个IP首先需要一个叫法，这是IP传播的名片。必须明确这个IP叫什么，其重要性如同一个品牌的商标名。

![](img/57b834854b1abd46d920dedea9a5b1f3_18.png)

2.  **职业/职务**
    人格化必然要具备身份。职业是用户认识这个人或IP的联想入口，例如：老师、教授、博士、律师。

3.  **称呼/社会关系**
    一个IP有了亲切的称呼，能快速拉近与用户的距离，起到情感共鸣的作用。例如：XX哥、XX大爷、XX姥姥。

4.  **形象/第一印象**
    以用户视角看待IP，使IP形象化，更容易占据用户心智。例如：大叔形象、萌萌哒、铁憨憨、甜美可爱、光头。

5.  **性格/爱好/特点**
    一个具有鲜明特点的IP，除了记忆点强，也更具真实性。过于完美的形象容易脱离大众。例如：暴躁、娘炮、爱吃、爱喝酒。

6.  **穿着打扮/财富地位**
    这属于社会刻板印象和大众普遍认知。借助用户的固有思维，可以快速为IP在用户心智中定位。例如：有钱代表会赚钱，开豪车代表成功人士，打扮土气代表老师。

以上内容策划完成后，即可结合你自身的产品或服务优势，将其落实到**抖音账号的五件套**（头像、昵称、背景图、简介、视频封面）的搭建中。

## 如何提炼人设的“超级符号”？

除了对账号进行基础的人格化策划，还需要通过持续的选题与内容策划，为你的人设提炼出易于记忆的**超级符号**。

超级符号的提炼可以从以下几个感官维度着手：

*   **画风（视觉）**
    例如：独特的场景、标志性道具、固定服饰风格。
*   **听觉**
    例如：专属BGM、标志性特效音、独特的配音或声线特点。
*   **文案**
    能够反映优势且易于记忆的提炼词。例如：“简单粗暴”、“XX天花板”。
*   **表情/情绪传递**
    例如：标志性的笑、惊恐、无奈等表情或情绪表达方式。
*   **语言特色**
    例如：方言、口音、特色发音、惯用网络语、慢条斯理或抑扬顿挫的讲话节奏。
*   **动作/习惯**
    例如：习惯性的肢体动作，如撩头发、扶眼镜、特定手势。

超级符号提炼好后，在进行短视频创作时，就应不断在视频中重复呈现这些元素，强化用户记忆。

## 总结

本节课中，我们一起学习了抖音账号**人物设定**的核心方法。我们首先明确了人设的本质是**账号人格化**，接着从六个维度详细拆解了如何策划一个立体的人设，最后探讨了如何通过多感官维度提炼人设的**超级符号**，并在内容中持续强化。记住，一个真实、鲜明且一致的人设，是你在抖音海洋中脱颖而出、连接粉丝并实现长期价值的关键基石。
