# 抖音运营实战：P21：15、视频类型3-6：低温度剧情与三农内容

在本节课中，我们将要学习抖音内容创作中的“低温度”剧情类型，并了解当前平台重要的流量扶持方向——“三农”内容。我们将通过具体案例拆解这类视频的核心要素，并为你提供清晰的内容创作方向与质量评估标准。

上一节我们介绍了多种视频类型，本节中我们来看看“低温度”剧情和“三农”内容这两个具体方向。

“低温度”剧情同样是剧情类视频，但其核心是传递温暖、正能量。这类视频的主题通常围绕同事友谊、父母亲情、社会温情等展开。

目前，抖音平台除了年底的流量高峰，还有一个重要的流量扶持方向，即“三农”流量扶持。如果你身处农村或对农村题材感兴趣，可以重点关注这个领域。

![](img/5b7455b72a54d914a9d5aea5520797dc_1.png)

如果你目前不知道拍摄什么内容，并且身处农村，建议可以尝试拍摄农村美食或记录农村生活，这能获得可观的三农流量扶持。

这类视频的目的是触动观众的情感，引发共鸣，而不是单纯追求有趣。

![](img/5b7455b72a54d914a9d5aea5520797dc_42.png)

可能有同学会想：“我去拍帮助老爷爷这样的事件，会有人看吗？”答案在于视频的**结构设计**。上述案例的看点在于**反转**——情理之中，意料之外。这种设计能让观众产生“原来如此”的感叹，或者即使猜到结局，仍能被细节打动。

如果你感到创作思路混乱，请遵循以下原则来简化决策。

![](img/5b7455b72a54d914a9d5aea5520797dc_44.png)

以下是明确创作方向的关键建议：

*   **聚焦单一领域**：从课程提到的多种类型中，只选择**一个**方向深入去做。
*   **保持领域垂直**：不要同时涉猎多个差异很大的领域，专注才能让账号定位清晰，避免混乱。

确定方向后，你需要判断作品质量。我们可以使用三个关键词来评估。

以下是评估作品质量的三个核心维度：

1.  **有趣**：作品能让观众开怀大笑。
2.  **有用**：作品能让观众获得知识、技能或信息上的收获。
3.  **有感**：作品能引发观众的情感触动或思考。

![](img/5b7455b72a54d914a9d5aea5520797dc_46.png)

你的作品只要满足以上**任意一个**标准，就可以算作是好作品。如果三个标准一个都不符合，那么就需要反思并优化视频内容。

内容创作是一个需要不断学习和实践的过程，不可能在短时间内完全掌握。本节课初步介绍了内容环节的框架和思路，帮助你建立认知并找到备选方向。

本节课中我们一起学习了“低温度”正能量剧情视频的特点，了解了“三农”内容的流量机会，掌握了聚焦单一领域进行创作的方法，并学会了用“有趣、有用、有感”三个标准来初步评估视频质量。希望这些内容能为你开启抖音创作提供清晰的思路。
