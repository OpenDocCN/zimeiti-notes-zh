# 抖音运营教程：P48：文案素材与视频制作 🎬

在本节课中，我们将学习如何为视频创意寻找文案素材，并了解一种将文字内容转化为视频的创作方法。我们将介绍一个实用的工具，并讲解如何合法、高效地利用素材进行二次创作。

![](img/8e78c37cb83272e7eec4d55133a530f3_1.png)

上一节我们介绍了视频创意的确定方法，本节中我们来看看如何为创意寻找合适的文案。

![](img/8e78c37cb83272e7eec4d55133a530f3_3.png)

视频创意确定后，需要搜索相应的文案。直接搬运他人视频素材并进行处理是一种方式。若希望根据确定的创意搜索文案，并以另一种形式制作视频，则需要寻找文字素材。

![](img/8e78c37cb83272e7eec4d55133a530f3_5.png)

![](img/8e78c37cb83272e7eec4d55133a530f3_7.png)

## 文案素材搜索工具 🛠️

![](img/8e78c37cb83272e7eec4d55133a530f3_9.png)

文案素材方面，推荐一个名为“淘金阁”的网站工具。



![](img/8e78c37cb83272e7eec4d55133a530f3_11.png)

淘金阁是一个“自媒体全网搜索引擎”。



![](img/8e78c37cb83272e7eec4d55133a530f3_13.png)

点击进入后，可以看到该工具聚合了大量文章。



![](img/8e78c37cb83272e7eec4d55133a530f3_15.png)

以下是淘金阁搜集文章的主要渠道列表：
*   今日头条
*   百家号
*   企鹅号
*   趣头条
*   一点资讯
*   QQ看点
*   凤凰网
*   网易资讯
*   搜狐网
*   东方网
*   简书
*   快资讯



![](img/8e78c37cb83272e7eec4d55133a530f3_17.png)

因此，淘金阁拥有一个庞大的文案素材库。



![](img/8e78c37cb83272e7eec4d55133a530f3_19.png)

![](img/8e78c37cb83272e7eec4d55133a530f3_21.png)

在搜索框中输入目标关键词，例如“星座”。



![](img/8e78c37cb83272e7eec4d55133a530f3_23.png)

点击“开始抓取”后，下方会列出所有抓取到的相关文章。这是寻找文案素材，而非视频素材。



![](img/8e78c37cb83272e7eec4d55133a530f3_25.png)

![](img/8e78c37cb83272e7eec4d55133a530f3_27.png)

将别人的文章素材转化为自己的视频，不属于直接搬运视频的范畴。这与直接使用他人视频有本质区别。



![](img/8e78c37cb83272e7eec4d55133a530f3_29.png)

我们随机打开一篇文章，例如“分手后也能挽回最放不下前任的三大星座”。



![](img/8e78c37cb83272e7eec4d55133a530f3_31.png)

![](img/8e78c37cb83272e7eec4d55133a530f3_33.png)

点击后即可直接查看文章全文。



![](img/8e78c37cb83272e7eec4d55133a530f3_35.png)

这样就能获得我们想要的文案内容，例如关于巨蟹座、双鱼座等的描述。



![](img/8e78c37cb83272e7eec4d55133a530f3_37.png)

## 从文案到视频 🎥

![](img/8e78c37cb83272e7eec4d55133a530f3_39.png)

找到文案后，接下来需要制作视频。本节介绍一种视频制作形式。

关于视频创意，之前提到的5118热词搜索以及本节课的淘金阁文案搜索，都是有效的工具，课后可以多加练习使用。

接下来进入视频制作环节。视频制作方式多样，例如真人出镜拍摄，或者音乐影视混剪。音乐影视混剪可以使用特定网站或录屏方式获取素材。之前分享过的台词混剪网站，以及如 `mvcat.com` 这类提供电影下载的网站，都可以作为影视混剪的素材来源。

![](img/8e78c37cb83272e7eec4d55133a530f3_41.png)

本节课中我们一起学习了如何利用“淘金阁”工具搜索文案素材，并理解了将文章合法转化为视频内容的创作思路，同时回顾了多种视频制作的形式。
