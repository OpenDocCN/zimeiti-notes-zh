# 视频号运营：P35：19.实操步骤 - 开通与上架商品橱窗

在本节课中，我们将学习如何为微信视频号开通商品橱窗功能，并完成商品上架。这是实现视频号带货变现的关键一步，操作简单，适合零基础用户。

上一节我们介绍了视频号运营的核心逻辑，本节中我们来看看如何具体操作，开通你的带货权限。

![](img/036cf333ad2fc32113ea9f997a23fac7_1.png)

![](img/036cf333ad2fc32113ea9f997a23fac7_2.png)

## 第一步：进入视频号创作者中心

![](img/036cf333ad2fc32113ea9f997a23fac7_4.png)

以下是进入创作者中心的具体步骤：

![](img/036cf333ad2fc32113ea9f997a23fac7_6.png)

1.  **打开微信**：在手机主屏幕找到并点击微信应用图标。
2.  **点击“发现”**：进入微信后，点击底部导航栏的“发现”选项。
3.  **进入“视频号”**：在“发现”页面，找到并点击“视频号”入口（图标为黄色蝴蝶形状）。
4.  **点击个人头像**：进入视频号后，点击页面右上角的个人头像图标。
5.  **进入“创作者中心”**：在个人页面中，向下滑动，找到并点击“创作者中心”。（若未显示，需先发布一条视频作品。）

## 第二步：开通商品橱窗功能

![](img/036cf333ad2fc32113ea9f997a23fac7_8.png)

![](img/036cf333ad2fc32113ea9f997a23fac7_9.png)

进入创作者中心后，我们需要找到带货入口并开通权限。

6.  **找到“带货中心”**：在创作者中心页面内，找到名为“带货中心”的模块（图标类似小卖部）。
7.  **选择带货身份**：点击“带货中心”后，系统会跳转到身份选择页面。这里提供“我是商家”、“我是达人”等选项。**请选择“我是达人”**。
8.  **缴纳保证金**：选择身份后，会进入保证金缴纳界面。点击“去交纳”，通过微信支付100元保证金即可完成开通。

**核心操作公式**：
```
开通橱窗 = 进入创作者中心 -> 点击带货中心 -> 选择“我是达人” -> 缴纳100元保证金
```

![](img/036cf333ad2fc32113ea9f997a23fac7_11.png)

## 第三步：上架商品到橱窗

开通权限后，原来的“带货中心”入口会变为商品管理界面。

9.  **管理商品**：再次点击“带货中心”，即可进入选品广场或你的商品管理后台。
10. **选择商品**：在选品广场中，你可以浏览并选择想要推广的商品。
11. **上架商品**：点击“加橱窗”或类似按钮，即可将商品添加至你的视频号橱窗。

![](img/036cf333ad2fc32113ea9f997a23fac7_13.png)

至此，你已经完成了视频号商品橱窗的开通与基础商品上架操作。

![](img/036cf333ad2fc32113ea9f997a23fac7_15.png)

本节课中我们一起学习了从零开始开通视频号商品橱窗的完整流程。关键点在于通过“创作者中心”找到“带货中心”，选择达人身份并缴纳保证金。开通后，你就可以在选品广场挑选商品进行推广，为你的视频号内容增添变现渠道。
