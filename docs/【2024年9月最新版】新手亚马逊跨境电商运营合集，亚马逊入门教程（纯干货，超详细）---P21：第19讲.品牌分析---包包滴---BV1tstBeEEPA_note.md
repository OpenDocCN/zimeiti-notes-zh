在本节课中，我们将学习亚马逊后台“品牌分析”功能的核心板块与使用方法。品牌分析是亚马逊为品牌卖家提供的重要数据工具，能帮助我们洞察市场、优化关键词和了解客户。

![](img/722d1118ca9a98fed8fcbc02328ee0cd_5.png)

上一节我们介绍了品牌旗舰店的展示功能，本节中我们来看看如何利用“品牌分析”工具获取关键市场数据。

## 🔍 亚马逊关键词搜索

品牌分析的第一个核心功能是“亚马逊关键词搜索”。此功能类似于卖家精灵等选品工具的关键词反查与拓展，也类似于广告投放中的广泛匹配逻辑。

以下是使用该功能的具体步骤与解读：

1.  **搜索与结果**：在搜索框输入一个种子关键词（例如 “lashes”），点击“应用”。系统会返回一系列相关的关键词及其数据。
2.  **搜索排名**：结果中的“搜索排名”数字越小，代表该关键词的搜索量越大。但亚马逊不显示具体的搜索量数值。
3.  **热门商品**：每个关键词下方会展示点击率和转化率综合排名前三的商品（ASIN）。这有助于我们了解在该关键词下，哪些链接表现最好，其竞争格局如何。
    *   **公式解读**：`关键词流量份额 = 某ASIN点击份额 * 转化份额`。如果一个商品在某关键词下的点击份额高达77%，意味着它垄断了大部分流量，新卖家进入难度较大。
4.  **ASIN反查**：此功能亦可作为ASIN反查工具使用。输入竞争对手的ASIN，可以查询到该商品正在获得流量的所有关键词，以及每个关键词下Top 3商品的点击份额。

通过以上数据，我们可以在打关键词排名时，优先选择竞争机会较大（即Top 3商品点击份额总和较低）的关键词，避开已被头部商品垄断的流量词。

## 📊 其他分析板块

![](img/722d1118ca9a98fed8fcbc02328ee0cd_7.png)

除了关键词搜索，品牌分析还包含其他几个有价值的板块。

![](img/722d1118ca9a98fed8fcbc02328ee0cd_9.png)

以下是各板块的简要介绍：

![](img/722d1118ca9a98fed8fcbc02328ee0cd_11.png)

1.  **重复购买行为**：此板块帮助分析顾客的复购情况。可以查看哪些商品产生了复购订单，以及不同顾客群体（如新客 vs 老客）的购买占比。
2.  **市场篮子分析**：该功能展示了顾客同时购买哪些商品的比例较高。对于规划产品组合或进行捆绑销售有参考意义，但日常使用频率相对较低。
3.  **人数统计**：这是非常有用的客户画像工具。可以清晰地看到购买我们店铺商品的客户群体特征。
    *   **分析维度**：包括年龄分布、家庭收入分布、教育程度分布、性别分布以及婚姻状况分布。
    *   **代码示例**（数据示意）：
        ```json
        “客户画像”： {
          “年龄”： [“25-34岁： 40%”， “35-44岁： 30%”],
          “收入”： [“$60K-$100K： 35%”， “$100K+： 25%”],
          “性别”： [“女性： 70%”， “男性： 30%”]
        }
        ```

这些数据能帮助我们更精准地定位目标客户，并指导产品开发、文案撰写和广告投放策略。
