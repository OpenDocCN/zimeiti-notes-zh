# 新媒体运营：P10：小红书平台笔记审核规则 📝

在本节课中，我们将要学习小红书平台笔记的流量推荐与分发机制。我们将详细拆解一篇笔记从发布到推荐给用户的完整流程，并重点解析其中的审核规则与账号权重概念。

## 笔记发布后的审核流程

上一节我们介绍了小红书“千人千面”的推荐逻辑。本节中，我们来看看一篇笔记发布后，需要经历哪些审核阶段才能被推荐。

一篇笔记发布后，会依次经历三个主要审核阶段：**机器审核**、**去重检测**和**用户匹配**。正常的笔记在机器审核阶段即可通过。

### 1. 机器审核

笔记首先会被上传到审核系统。该系统会根据预设的基本规则进行初步审核。

以下是机器审核的主要规则：

*   **敏感内容检测**：系统会识别笔记中的敏感词、敏感图片等违规内容。
*   **初步质量评估**：系统会评估笔记的基础质量，例如字数、广告词出现频率、图片清晰度、文字可识别性等。

如果笔记存在明确违规内容，可能会被直接封禁。如果内容存在疑似违规或机器难以判断的情况，笔记会被标记并转入**人工审核**阶段。如果笔记没有问题，则会进入下一阶段。

![](img/ab59eafa0c0f653228d431278c0bd4ff_3.png)

### 2. 去重检测

![](img/ab59eafa0c0f653228d431278c0bd4ff_5.png)

![](img/ab59eafa0c0f653228d431278c0bd4ff_7.png)

通过机器审核后，笔记会进入去重检测阶段。系统会将你的笔记与平台数据库内的其他数据进行比对，检查相似度。

![](img/ab59eafa0c0f653228d431278c0bd4ff_9.png)

![](img/ab59eafa0c0f653228d431278c0bd4ff_11.png)

去重检测主要比对以下几个方面：

![](img/ab59eafa0c0f653228d431278c0bd4ff_13.png)

![](img/ab59eafa0c0f653228d431278c0bd4ff_15.png)

*   **图片相似度**：检查图片是否为直接抄袭或复制他人。
*   **文案重复率**：计算文案内容在数据库中的重复比例。
*   **段落与整体相似度**：分析段落结构及整体内容的相似程度。

![](img/ab59eafa0c0f653228d431278c0bd4ff_17.png)

![](img/ab59eafa0c0f653228d431278c0bd4ff_19.png)

如果笔记是原封不动的抄袭，重复率检测结果为100%，笔记会被直接封禁。如果对图片（如裁剪、加滤镜）或文案（如调整段落、扩充内容）进行了修改，则有可能通过检测。

![](img/ab59eafa0c0f653228d431278c0bd4ff_21.png)

### 3. 用户匹配

![](img/ab59eafa0c0f653228d431278c0bd4ff_23.png)

![](img/ab59eafa0c0f653228d431278c0bd4ff_25.png)

通过去重检测后，笔记会进入系统进行用户匹配。系统会分析笔记的特征（如标签、关键词），以便后续将其推荐给可能感兴趣的用户。这是流量分发的开始。

![](img/ab59eafa0c0f653228d431278c0bd4ff_27.png)

![](img/ab59eafa0c0f653228d431278c0bd4ff_29.png)

## 账号权重与内容分发

![](img/ab59eafa0c0f653228d431278c0bd4ff_31.png)

![](img/ab59eafa0c0f653228d431278c0bd4ff_33.png)

在机器审核阶段，系统除了检查违规，还会为笔记打一个“基础质量分”。这个分数会结合**账号权重**，共同影响笔记的初始曝光量。

![](img/ab59eafa0c0f653228d431278c0bd4ff_35.png)

![](img/ab59eafa0c0f653228d431278c0bd4ff_37.png)

账号权重是一个抽象概念，代表平台系统根据账号历史表现（如内容垂直度、违规记录、更新频率等）给出的综合评分。我们可以用一个简单的模型来理解：

![](img/ab59eafa0c0f653228d431278c0bd4ff_39.png)

![](img/ab59eafa0c0f653228d431278c0bd4ff_41.png)

**账号权重模型（假设）:**
`初始权重 = 0`
`最终权重 = f(内容质量, 违规次数, 更新频率, 用户互动, ...)`

![](img/ab59eafa0c0f653228d431278c0bd4ff_43.png)

![](img/ab59eafa0c0f653228d431278c0bd4ff_45.png)

权重高低直接影响笔记的初始推荐流量池大小。

![](img/ab59eafa0c0f653228d431278c0bd4ff_47.png)

### 新号阶段（低权重期）

![](img/ab59eafa0c0f653228d431278c0bd4ff_49.png)

![](img/ab59eafa0c0f653228d431278c0bd4ff_51.png)

新注册的账号，初始权重可以理解为 **0**。这个阶段通常被称为“养号期”。

![](img/ab59eafa0c0f653228d431278c0bd4ff_53.png)

![](img/ab59eafa0c0f653228d431278c0bd4ff_55.png)

在新号阶段，必须严格遵守平台规则，为账号打下良好基础。以下是需要注意的要点：

![](img/ab59eafa0c0f653228d431278c0bd4ff_57.png)

![](img/ab59eafa0c0f653228d431278c0bd4ff_59.png)

*   **严禁违规**：绝对不要发布广告词、违禁内容。
*   **保持活跃**：注意内容发布的时间与周期，尽量保持稳定更新。
*   **内容垂直**：持续发布同一领域的内容，帮助系统为账号打上标签。

![](img/ab59eafa0c0f653228d431278c0bd4ff_61.png)

此阶段笔记的初始推荐量通常很低（如30-40次曝光），因为系统无法精准判断内容质量和目标受众。即使内容优质，也很难突破流量瓶颈，除非极少数笔记在少量曝光中获得了异常高的互动率。

![](img/ab59eafa0c0f653228d431278c0bd4ff_63.png)

![](img/ab59eafa0c0f653228d431278c0bd4ff_65.png)

### 成熟账号阶段（高权重期）

![](img/ab59eafa0c0f653228d431278c0bd4ff_67.png)

![](img/ab59eafa0c0f653228d431278c0bd4ff_69.png)

当账号持续发布优质、垂直的内容后，系统会逐渐提高其在该领域的权重。

![](img/ab59eafa0c0f653228d431278c0bd4ff_71.png)

![](img/ab59eafa0c0f653228d431278c0bd4ff_73.png)

例如，一个专注“彩妆”领域的账号，其权重分会随着优质内容的积累而提升。高权重账号发布的笔记，会获得更大的初始推荐流量池和更精准的用户匹配。

![](img/ab59eafa0c0f653228d431278c0bd4ff_75.png)

**核心逻辑**：权重提升依赖于账号的长期、稳定、合规的优质运营。在低权重期，应专注于“养号”而非追求爆文。

![](img/ab59eafa0c0f653228d431278c0bd4ff_77.png)

![](img/ab59eafa0c0f653228d431278c0bd4ff_79.png)

## 关于人工审核的补充说明

![](img/ab59eafa0c0f653228d431278c0bd4ff_81.png)

![](img/ab59eafa0c0f653228d431278c0bd4ff_83.png)

人工审核主要针对机器审核标记的“疑似违规”内容进行二次复核。但由于内容量巨大，成本高昂，并非所有笔记都会进入人工审核。

人工审核在特定领域（如政治、色情擦边、法律严禁内容）力度较大。目前，平台机器审核算法已相当完善，触发明确红线的笔记通常会被直接封禁，可能不再经过人工复核。

**给新手的建议**：如果认为笔记被误判，可以通过官方渠道申诉。但在账号前期，应极力避免任何违规行为，因为即使是少数几次违规，也可能对账号产生隐形的限流等负面影响。

## 总结

本节课我们一起学习了小红书笔记的审核与分发机制。

1.  **审核流程**：笔记需依次通过**机器审核**（查违规、评质量）、**去重检测**（查抄袭）和**用户匹配**（打标签）三个阶段。
2.  **账号权重**：这是一个基于账号长期表现的系统综合评分，直接影响笔记的初始曝光量。新号权重低，需通过“养号”提升权重。
3.  **运营要点**：初期避免违规、保持内容垂直与稳定更新是提升账号权重、获得推荐流量的基础。

理解这些规则，能帮助我们更有效地运营小红书账号，为创作出爆款笔记奠定基础。
