## 第一部分：如何寻找小红书对标账号

![](img/1181728b9f7ca090ab06dc892212d062_21.png)

以下是寻找小红书对标账号的具体步骤和标准，我们以“童装”类目为例进行演示。

![](img/1181728b9f7ca090ab06dc892212d062_27.png)

![](img/1181728b9f7ca090ab06dc892212d062_28.png)

1.  **确定搜索关键词**：在搜索框中输入你的类目关键词，例如“童装”。
2.  **筛选潜在账号**：浏览搜索结果，寻找与你定位（如女童穿搭、童装套装等）相似的账号。
3.  **分析账号数据**：点开潜在账号的主页，重点观察以下几个核心指标：
    *   **粉丝量**：选择粉丝量适中（既不是头部巨无霸，也不是刚起步）的账号。这类账号的成功路径更有可能被我们模仿和触及。
    *   **笔记点赞数据**：这是判断账号内容质量的关键。优秀的对标账号应该“时不时有爆款笔记出现”。如果一个账号的笔记点赞量普遍很低，且长时间没有出现过百或更高的爆文，那么模仿它的意义就不大。
    *   **内容一致性**：观察账号的封面风格、标题套路、内容主题是否统一。统一且具有辨识度的风格是账号成功的重要因素。

![](img/1181728b9f7ca090ab06dc892212d062_30.png)

![](img/1181728b9f7ca090ab06dc892212d062_31.png)

![](img/1181728b9f7ca090ab06dc892212d062_32.png)

![](img/1181728b9f7ca090ab06dc892212d062_33.png)

![](img/1181728b9f7ca090ab06dc892212d062_34.png)

**核心筛选公式**：
```
优质对标账号 ≈ 适中粉丝量 + 稳定的平均互动数据 + 间歇性的爆款笔记产出
```

![](img/1181728b9f7ca090ab06dc892212d062_35.png)

![](img/1181728b9f7ca090ab06dc892212d062_37.png)

![](img/1181728b9f7ca090ab06dc892212d062_38.png)

让我们通过一个实例来分析。在搜索“童装”后，我们找到了一个名为“富养女最美穿搭”的账号。这个账号具有以下特点：
*   粉丝量22万，属于可模仿的范畴。
*   笔记点赞数据优秀，频繁出现点赞数过百甚至数百的爆款笔记。
*   封面标题高度统一，均以“富养女”开头，形成了强烈的个人风格和记忆点。
*   内容聚焦于“潮款套装”，定位清晰。

## 第二部分：如何在抖音寻找素材对标账号

![](img/1181728b9f7ca090ab06dc892212d062_51.png)

![](img/1181728b9f7ca090ab06dc892212d062_52.png)

![](img/1181728b9f7ca090ab06dc892212d062_53.png)

找到小红书对标账号，解决了“账号方向和形式”的问题。接下来，我们需要解决“内容素材从哪里来”的问题。这时，我们需要在抖音上寻找素材对标账号。

![](img/1181728b9f7ca090ab06dc892212d062_54.png)

![](img/1181728b9f7ca090ab06dc892212d062_55.png)

![](img/1181728b9f7ca090ab06dc892212d062_56.png)

**为什么是抖音？** 因为抖音是当前最大的短视频内容平台，每天产生海量的视频素材。这些素材是我们进行二次创作或混剪的宝贵来源。

**两个平台分工明确**：
*   **小红书对标账号**：决定你的**账号定位、人设和呈现形式**。
*   **抖音对标账号**：为你提供源源不断的**视频内容素材**。

![](img/1181728b9f7ca090ab06dc892212d062_58.png)

![](img/1181728b9f7ca090ab06dc892212d062_60.png)

以下是寻找抖音素材对标账号的步骤：

![](img/1181728b9f7ca090ab06dc892212d062_61.png)

![](img/1181728b9f7ca090ab06dc892212d062_62.png)

![](img/1181728b9f7ca090ab06dc892212d062_63.png)

1.  **打开抖音，搜索关键词**：根据你在小红书确定的定位（如“童装”），在抖音搜索框输入相同关键词。
2.  **切换到“视频”标签**：在搜索结果页点击“视频”，浏览相关视频。
3.  **筛选优质素材账号**：点击进入那些点赞量较高（特别是近期发布的）的视频主页。筛选标准与小红书类似：
    *   观察该账号的视频数据是否“**普遍不错，且时不时有远超平均水平的爆款**”。
    *   优先选择**非真人出镜**或非头部网红的账号，以降低侵权风险。
    *   关注那些**更新频繁、爆款率较高**的中小型账号。
4.  **建立专属素材库**：建议使用一个**单独的抖音号**来执行这个任务。例如，你做童装，就用这个号只刷和关注童装内容。这样，抖音的推荐算法会不断为你推送同领域的优质视频，让你更高效地发现素材。

![](img/1181728b9f7ca090ab06dc892212d062_65.png)

**操作建议**：
```text
1. 准备一个干净的抖音小号。
2. 用该小号大量搜索、浏览、点赞和关注你所在领域的视频。
3. 最终筛选出5-10个高质量的素材来源账号并关注。
4. 日常运营中，只需定期查看这些账号的新视频，发现数据快速上升的潜力爆款，及时进行二次创作并发布到小红书。
```

![](img/1181728b9f7ca090ab06dc892212d062_67.png)

![](img/1181728b9f7ca090ab06dc892212d062_68.png)

![](img/1181728b9f7ca090ab06dc892212d062_69.png)

![](img/1181728b9f7ca090ab06dc892212d062_70.png)

![](img/1181728b9f7ca090ab06dc892212d062_71.png)

通过这种方式，你无需全网盲目寻找素材，只需维护好这个小小的“素材监控池”，就能保证内容创作的效率和质量。

## 总结

本节课中，我们一起学习了寻找对标账号的双平台策略。

![](img/1181728b9f7ca090ab06dc892212d062_75.png)

*   在**小红书**，我们寻找的是**形式与方向的对标**，用于确定账号的整体调性、封面标题风格和内容定位。关键是要找到那些数据健康、有爆款潜力、风格统一的账号进行学习。
*   在**抖音**，我们寻找的是**内容素材的对标**，用于解决“拍什么”的问题。关键是建立专属的素材号，持续关注一批产出稳定、爆款率高的同行，将其作为灵感与素材的来源。

![](img/1181728b9f7ca090ab06dc892212d062_77.png)

做好这两步，你的小红书运营就有了清晰的学习目标和稳定的内容供应链，为账号的快速起步奠定了坚实的基础。下一节，我们将开始学习如何设置你的小红书账号主页。
