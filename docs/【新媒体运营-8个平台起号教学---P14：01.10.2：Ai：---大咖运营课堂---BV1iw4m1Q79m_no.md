# 新媒体运营：P14：01.10.2：🤖 训练AI写出符合小红书风格的文案

在本节课中，我们将学习如何训练AI（如ChatGPT）来撰写符合小红书平台风格的文案。上一节我们介绍了如何让AI生成爆款标题，本节中我们来看看如何让AI理解并创作出符合平台调性的完整内容。

## 概述：训练AI理解平台风格

要让AI写出符合特定平台（如小红书）风格的文案，关键在于“投喂”资料并引导其学习。AI需要理解该平台独特的语气、常用词汇、内容结构和用户偏好。这个过程不仅仅是下达指令，更是通过提供范例让AI进行模仿和学习。

## 核心步骤：提供资料与分析

以下是训练AI的核心步骤，我们需要为AI提供学习材料并设定明确的学习目标。

![](img/d41c1d8ad52b470b984f7c3e3f5cc7f9_5.png)

1.  **明确指令**：首先，你需要清晰地告诉AI它的任务。例如，你可以输入以下指令：
    ```
    接下来我会给你提供一些小红书上的爆款笔记文案。你需要根据这些文案，分析小红书的文案撰写风格、特定的平台用语、词汇、语气等。每接收一篇文案后，请告诉我你理解并学习了，并给出你对这篇文案风格、语气、类型等的分析。
    ```
    这段指令明确了AI的角色（分析者）和学习目标（平台风格）。

![](img/d41c1d8ad52b470b984f7c3e3f5cc7f9_7.png)

![](img/d41c1d8ad52b470b984f7c3e3f5cc7f9_9.png)

2.  **投喂范例**：接着，你需要寻找并复制粘贴真正具有小红书特色的爆款文案给AI。注意，应选择那些能体现平台独特语气（如亲切、分享感强）、常用词汇（如“宝子”、“姐妹”、“绝绝子”）和内容结构（如个人经历为主线、包含“避坑”建议）的笔记。

![](img/d41c1d8ad52b470b984f7c3e3f5cc7f9_11.png)

![](img/d41c1d8ad52b470b984f7c3e3f5cc7f9_12.png)

3.  **获取反馈**：每提供一篇文案后，要求AI给出分析反馈。这不仅能确认AI是否在认真学习，也能让你了解它从范例中提取了哪些关键特征。一个典型的AI反馈可能如下：
    > “学习完毕。这篇文案以个人游玩经历为主线，增加了真实感和亲近感。标题使用了‘血泪教训’和‘不踩坑建议’等词汇，强调了经验分享和实用价值。语气亲切，像朋友间的安利。”

## 关键注意事项

在投喂资料时，有几个要点需要特别注意，以确保训练效果。

*   **多样性**：提供的文案范例应涵盖不同风格和类型（如种草、攻略、吐槽、好物分享），避免全是同质化的内容（如单一的旅游路线清单）。
*   **质量**：选择那些真正具有“网感”、能体现平台语言特色的文案，而不是通用、平淡的说明性文字。
*   **持续性**：训练不是一次性的。在同一个对话窗口中持续投喂资料，AI会积累“记忆”，从而更深入地理解平台风格。

## 实践：生成小红书风格文案

经过一定量的资料投喂后，即可尝试让AI进行创作。你可以下达具体的创作指令。

例如，结合之前的学习，请AI创作一篇文案：
```
请结合你的学习，帮我写一篇具有爆款潜质的张家界5日游攻略小红书笔记。内容需涵盖丰富的小红书风格Emoji表情，同时生成10个符合小红书旅游赛道风格的爆款标题。
```
AI会根据之前学习的风格进行创作。初期成果可能不尽完美，这时可以通过以下方式优化：
*   **补充信息**：如果AI对景点细节描述不足，你可以先让它列出张家界的主要景点及特色，再让其基于此撰写攻略。
*   **细化指令**：要求文案包含“避坑指南”、“必打卡机位”、“行程路线规划”等具体模块。
*   **结合标题指令**：在需要生成标题时，可以新建一个对话窗口，结合上一节课的“爆款标题生成指令”来获得更佳效果。

## 总结与展望

本节课中我们一起学习了训练AI撰写平台化文案的方法。核心在于**通过提供高质量的范例，让AI分析并内化平台的独特风格**，再指令其进行模仿创作。这个过程适用于小红书、抖音、B站等多个内容平台。

需要注意的是，AI工具（如GPT-3.5与GPT-4.0）的能力不同，生成的文案质量也会有差异。同时，你提供的“饲料”（范例）质量直接决定了AI的“产出”水平。
