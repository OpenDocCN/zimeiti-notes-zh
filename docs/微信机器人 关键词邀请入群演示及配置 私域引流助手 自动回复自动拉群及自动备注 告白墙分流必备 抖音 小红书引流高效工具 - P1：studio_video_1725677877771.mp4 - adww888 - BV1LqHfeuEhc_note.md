# 微信机器人教程：P1：关键词邀请入群功能详解 🚀

在本节课中，我们将学习如何配置微信机器人的“关键词邀请入群”功能。该功能可实现客户添加机器人好友后，通过回复特定关键词，自动收到指定群的邀请链接，从而实现自动化引流与社群管理。

## 功能演示 🎬

上一节我们概述了功能目标，现在通过一个演示来直观了解其工作流程。

我现在向机器人发送一个关键词，例如“**新生群**”。机器人随即自动回复了包含群邀请链接的提示信息。

我们再次测试，回复另一个关键词。机器人同样回复了对应社群的邀请链接。

用户点击此链接，即可直接加入目标群聊。

---

## 功能配置 ⚙️

看完演示效果后，本节我们来看看具体的配置方法。如果您使用过基础版机器人，对此界面会感到熟悉。

配置入口位于“**业务参数设置**”中的“**智能应答**”模块。此处新增了一个“**邀请入群**”的选项。

以下是配置时需要填写的关键项：

*   **关键词**：用户需要回复的触发词，例如“新生群”。
*   **匹配方式**：通常为“完全匹配”，确保回复精准。
*   **消息来源**：与基础版设置一致。
*   **邀请入群**：**这是核心新增选项**。您可以从白名单内的群聊列表中，选择该关键词对应的目标群。

配置完成后保存即可生效。
