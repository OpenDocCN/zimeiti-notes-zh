# 微信视频号运营从0~1全攻略：P21：19.实操步骤

在本节课中，我们将学习如何开通微信视频号的商品橱窗功能，这是进行视频号带货的基础操作。整个过程分为两大步：开通橱窗权限和上架商品。我们将通过清晰的步骤演示，确保每位初学者都能轻松掌握。

上一节我们介绍了视频号运营的基础概念，本节中我们来看看如何具体操作，开启你的带货之旅。

## 第一步：进入视频号创作者中心

![](img/9c40c16a560066ae6c218891c2a7dd56_4.png)

以下是进入视频号后台管理界面的具体步骤。

![](img/9c40c16a560066ae6c218891c2a7dd56_6.png)

1.  **打开微信**：在手机主屏幕找到并点击微信应用图标。
2.  **点击“发现”**：进入微信后，点击底部导航栏的“发现”选项卡。
3.  **进入“视频号”**：在“发现”页面，将视线向上滑动，找到并点击“视频号”入口（图标为黄色的蝴蝶形状）。
4.  **点击个人头像**：进入视频号页面后，点击右上角的个人头像图标。
5.  **进入“创作者中心”**：在个人页面中，向下滑动，找到并点击最下方的“创作者中心”。（若未显示，需先发布一条视频作品。）
6.  **找到“带货中心”**：在创作者中心页面内，找到“带货中心”的入口（图标类似一个小卖部）。

## 第二步：开通橱窗与上架商品

![](img/9c40c16a560066ae6c218891c2a7dd56_8.png)

![](img/9c40c16a560066ae6c218891c2a7dd56_9.png)

进入“带货中心”后，系统会根据你的账号状态引导你进行下一步操作。核心逻辑如下：

**操作逻辑**：`点击“带货中心” -> 若未开通，则进入开通流程；若已开通，则直接进入选品上架界面。`

### 情况一：开通橱窗权限

![](img/9c40c16a560066ae6c218891c2a7dd56_11.png)

如果你从未开通商品橱窗，点击“带货中心”后会进入身份选择页面。

1.  **选择带货身份**：页面会提供“我是商家”、“我是达人”、“我是商家且已有视频号”三个选项。**请选择“我是达人”**。
2.  **缴纳保证金**：选择身份后，会跳转到保证金缴纳页面。点击“去交纳”，通过微信支付100元保证金即可完成开通。

![](img/9c40c16a560066ae6c218891c2a7dd56_13.png)

### 情况二：上架商品

如果你已经开通了橱窗权限，点击“带货中心”后将直接进入商品选品库，你可以在此浏览并选择商品添加到自己的橱窗中。

![](img/9c40c16a560066ae6c218891c2a7dd56_15.png)

本节课中我们一起学习了开通微信视频号商品橱窗的完整流程。关键操作是进入“创作者中心”的“带货中心”，根据引导选择“达人”身份并缴纳保证金。完成这些步骤后，你就具备了在视频号进行带货的基础条件，接下来就可以开始挑选商品，为你的视频内容添加销售链接了。
