# AI新媒体运营教程：P15：01.10.03：🤖 训练AI从分析到产出爆款笔记

## 概述
在本节课中，我们将学习如何利用AI工具（如ChatGPT）分析现有爆款笔记，并基于分析结果，指导AI生成具备爆款潜质的新笔记。整个过程分为两个核心步骤：**分析学习**与**模仿创作**。

上一节我们介绍了通过“喂资料”的方式让AI理解特定平台的用语风格。虽然上一节的旅游攻略案例结果不理想，但方法是正确的。只要提供足够且优质的行业内容给AI分析，它就能学习到该平台的文案风格和写作要点。

本节中，我们来看看如何将这个方法深化，通过分析具体的爆款笔记，提炼出可复制的成功要素，并最终让AI生成新的内容。

## 第一步：分析爆款笔记
要让AI学会创作，首先需要教会它“鉴赏”。我们需要提供爆款笔记作为样本，并指令AI进行结构化分析。

![](img/b9c88a3ccabf31733e57762153b154a1_7.png)

以下是分析时需要关注的几个核心点，我们需要明确地告诉AI：
1.  **爆款因素**：分析内容为何能成为爆款（例如：情感共鸣、实用价值、逆袭故事等）。
2.  **整体结构**：分析文案的起承转合。
    *   **开篇**：用什么方式（如提问、共鸣痛点）吸引用户。
    *   **中间**：用什么内容（如具体方法、资源推荐）戳中用户痛点，提升停留时长。
    *   **结尾**：用什么“钩子”（如鼓励、邀请互动）促使用户私信或关注。
3.  **关键词策略**：分析笔记是否刻意植入了特定关键词，以及关键词的出现频率和分布方式。

![](img/b9c88a3ccabf31733e57762153b154a1_9.png)

### 操作演示：以“考研”领域为例
我们选取一篇小红书上的考研爆款笔记（3000+点赞）作为分析样本。

![](img/b9c88a3ccabf31733e57762153b154a1_11.png)

![](img/b9c88a3ccabf31733e57762153b154a1_13.png)

**给AI的指令如下：**
```
我给你提供一篇小红书平台上的考研爆款笔记文案。
请你分析：
1. 这篇文案的爆款因素有哪些？
2. 分析文案的整体结构：开篇如何吸引用户？中间如何戳中痛点？结尾如何引导互动？
3. 分析该笔记是否有结合中文分词法设计关键词。如果有，请指出关键词是什么及其出现频率；如果没有，请说明。
```

![](img/b9c88a3ccabf31733e57762153b154a1_15.png)

![](img/b9c88a3ccabf31733e57762153b154a1_17.png)

![](img/b9c88a3ccabf31733e57762153b154a1_19.png)

**AI分析结果示例：**
*   **爆款因素**：情感共鸣（开篇描述备考焦虑）、逆袭故事分享、提供实用备考建议、使用表情符号增加亲和力、结尾传递正能量鼓励。
*   **整体结构**：开篇以焦虑共鸣引入 -> 中间分享个人经验和具体方法 -> 结尾鼓励并暗示可提供更多帮助。
*   **关键词分析**：本篇未发现明显刻意设计的关键词，多为“考研”、“备考”等常规词汇。

![](img/b9c88a3ccabf31733e57762153b154a1_21.png)

重复此过程，为AI提供多篇（建议3-5篇）同领域的爆款笔记进行分析，让它积累足够的“经验”。

## 第二步：整合爆款要素并生成大纲
在AI分析了多篇样本后，我们需要让它将分析出的爆款要素进行整合、去重，形成一个清晰的“爆款公式”。

**给AI的指令如下：**
```
请将你前面分析过的几篇考研爆款笔记的爆款因素总结一下，整合成一个清晰的列表。
```

![](img/b9c88a3ccabf31733e57762153b154a1_23.png)

![](img/b9c88a3ccabf31733e57762153b154a1_25.png)

**AI生成的爆款要素列表可能包括：**
*   情感共鸣（焦虑、迷茫）
*   逆袭故事引入
*   明确的备考计划与时间节点
*   针对性的学科建议（政治、英语、专业课）
*   具体的资源与技巧分享
*   时间管理与效率提升强调
*   正能量鼓励与心态调整
*   使用表情符号增强感染力
*   结尾引导互动（提问、私信）

获得要素列表后，即可指令AI根据这些要素，规划一篇新笔记的框架。

**给AI的指令如下：**
```
现在，请结合你总结的爆款笔记因素，为我规划一篇关于考研的小红书笔记大纲。
大纲需要结构完整，并细化到每个部分的要点。在我确认大纲后，再生成完整文案。
```

**AI生成的大纲示例：**
1.  **开篇：引人入胜的引入**
    *   使用情感共鸣话语或提问。
    *   简要介绍主题（考研备考）。
2.  **主体：干货内容分享**
    *   专业选择分析。
    *   全年时间规划表。
    *   政治/英语/数学/专业课分科备考策略。
    *   时间管理与心态调整方法。
3.  **结尾：互动引导**
    *   鼓励读者，强化正能量。
    *   提醒读者可以评论提问或私信。

## 第三步：根据大纲生成完整文案
大纲确认无误后，即可让AI填充内容，生成完整的笔记文案。此时需要给出更具体的写作要求。

**给AI的指令如下：**
```
请按照上面确认的大纲，撰写完整的文案。
要求：
1. 文案结构严格遵循爆款笔记因素和结构。
2. 开篇需要包含一个简单的“逆袭小故事”来引入情绪。
3. 文中需要包含一定量的emoji表情。
4. 需要结合中文分词法，自然地融入核心关键词，例如：“考研全流程”、“考研政治”。
```

**AI生成的文案示例（节选）：**
> 曾经我也在深夜刷题到崩溃，觉得考研上岸是个遥不可及的梦……但最终我成功逆袭了！只要你用对方法，一切皆有可能！💪
>
> **🔑 考研全流程规划：**
> *   **现阶段（X月-X月）**：主攻**考研政治**选择题与英语单词……
> *   **冲刺期（X月-X月）**：各科真题模拟，查漏补缺……
>
> 记住，合理的**时间管理**是逆袭的关键！保持好心态，你也能行！有任何问题，欢迎评论区留言或私信我~ 📚

这篇文案已经具备了爆款笔记的基本要素：情感带入、干货结构、关键词植入和互动引导。对于运营工作来说，这已是一个可用的基础稿。当然，我们还可以在此基础上进行人工优化和调整，使其更精炼、更出彩。

## 总结
本节课我们一起学习了利用AI辅助创作爆款笔记的完整流程：
1.  **分析学习**：提供多篇爆款笔记给AI，指令其分析爆款因素、结构和关键词策略。
2.  **整合要素**：让AI总结分析结果，提炼出可复制的爆款要素列表。
3.  **模仿创作**：基于爆款要素，让AI先产出内容大纲，再根据大纲和要求生成完整文案。
