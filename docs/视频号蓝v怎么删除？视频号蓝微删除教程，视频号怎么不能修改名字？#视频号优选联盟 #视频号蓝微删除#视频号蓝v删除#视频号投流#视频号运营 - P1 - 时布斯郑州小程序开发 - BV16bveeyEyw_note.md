# 视频号运营：01：视频号蓝V认证删除与名称修改教程

在本节课中，我们将学习如何删除视频号的蓝V认证，以及了解为何在拥有蓝V认证时无法修改账号名称。掌握此操作后，您将能够重新设定账号信息。

## 概述：为何需要删除蓝V认证？

![](img/5012e15ec3c3dfe346456cd2c0c1545e_0.png)

视频号在完成蓝V认证后，账号名称将被锁定，无法直接修改。若您需要更改名称，必须先删除现有的蓝V认证。本节将详细介绍删除认证的完整步骤。

## 核心操作：逐步删除蓝V认证

以下是删除视频号蓝V认证的具体操作流程。请确保您已登录需要操作的视频号账号。

1.  **进入视频号个人中心**
    打开微信，点击底部“发现”选项卡，然后进入“视频号”。在视频号页面，点击右上角的个人头像图标。

2.  **访问创作与认证管理**
    进入个人主页后，点击“创作中心”。随后，在页面中找到“带货中心”或相关功能区，点击其旁边的“更多”按钮（通常以“...”或类似图标表示）。

3.  **找到认证管理入口**
    在展开的菜单中，选择“认证”选项。页面将展示您当前视频号的认证状态。

![](img/5012e15ec3c3dfe346456cd2c0c1545e_2.png)

4.  **执行删除操作**
    在认证信息页面，找到并点击右上角的“三个点”菜单图标。在弹出的选项中，选择“删除认证”。

    > **关键提示**：系统会提示“删除后可以重新发起认证”，并告知您本年剩余的认证次数。此操作需谨慎，因为每年的认证机会有限。

## 操作后续：删除认证之后

上一节我们完成了蓝V认证的删除操作，本节中我们来看看后续可以进行的操作。

删除蓝V认证后，您的视频号账号名称的修改限制即被解除。此时，您可以返回账号设置页面，按照常规流程修改您的视频号名称。修改完成后，如果您仍需要蓝V认证，可以重新发起认证申请。

![](img/5012e15ec3c3dfe346456cd2c0c1545e_4.png)

## 总结

本节课中我们一起学习了视频号蓝V认证的删除流程。核心要点在于：**蓝V认证存在 → 名称不可修改 → 需先删除认证 → 再修改名称**。请记住，认证删除操作每年有次数限制，操作前请务必确认您的需求。完成名称修改后，您可以根据需要决定是否再次进行认证。
