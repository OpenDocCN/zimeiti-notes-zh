# 微信视频号运营教程：P21：7.账号定位

在本节课中，我们将要学习微信视频号运营中关于账号定位的两个核心问题：如何处理旧作品以及如何解决原创视频素材的难题。我们将提供清晰、可执行的步骤，帮助初学者快速上手。

## 旧作品处理与发布频率

上一节我们讨论了账号定位的重要性，本节中我们来看看如何为账号的初期运营打好基础。首先，需要处理账号中已有的旧作品。

![](img/ec876311f92fd89f4d79f511ba47cebc_1.png)

对于新定位的账号，需要将之前发布的、不符合新定位的作品隐藏起来。这是为了保持账号内容风格的统一性，让新关注的粉丝对你的账号有清晰的认知。

关于内容发布，初期应保持稳定的更新节奏。建议每天发布一条视频内容。这样既能保持账号的活跃度，又不会因为内容准备不足而影响质量。

![](img/ec876311f92fd89f4d79f511ba47cebc_3.png)

**核心操作**：
*   **隐藏旧作品**：进入视频号后台，将不符合新定位的历史作品设置为“仅自己可见”。
*   **发布频率**：`发布频率 = 1条/天`

遵循以上步骤，可以首先解决账号内容纯净度的问题，并为后续的内容发布建立规律。

## 解决原创视频素材问题

![](img/ec876311f92fd89f4d79f511ba47cebc_5.png)

![](img/ec876311f92fd89f4d79f511ba47cebc_6.png)

解决了基础设置后，接下来我们面对一个核心挑战：如何获得高质量的原创视频素材。许多初学者面临不会拍摄、不会剪辑的困境。

![](img/ec876311f92fd89f4d79f511ba47cebc_7.png)

以下是学员常见的三种情况：
1.  不具备专业的拍摄设备和条件。
2.  缺乏视频剪辑的技能和经验。
3.  没有现成的、适合发布的视频素材。

针对这些普遍问题，本教程提供一项支持：为认真学习的学员提供已剪辑好的视频素材。这些素材来自一个投入大量资金建设的独家素材库，涵盖了多种热门领域。

**素材库示例领域**：
*   搞笑娱乐
*   生活妙招
*   情感故事
*   影视混剪

这些素材均由专业剪辑师制作完成，用户可以直接使用，极大地降低了内容创作的门槛。
