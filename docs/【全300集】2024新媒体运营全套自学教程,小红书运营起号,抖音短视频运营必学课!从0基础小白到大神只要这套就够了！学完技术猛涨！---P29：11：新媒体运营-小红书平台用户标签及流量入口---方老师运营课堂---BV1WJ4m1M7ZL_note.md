## 用户标签判定机制 🔍

系统为用户打标签的主要依据是用户的**浏览行为**。用户注册时选择的兴趣范围参考价值不大，仅作为初期内容方向的引导，无法进入系统深层的用户标签池。

以下是判定用户标签的几个核心行为指标：

1.  **点击率**
    *   **含义**：系统在发现页向用户推送多种内容（如美妆、家居、母婴等），用户点击哪类内容更多，系统就能计算出其偏好。
    *   **逻辑**：例如，用户点击10条内容，其中美妆3次，家居2次，母婴0次。系统会判定该用户对母婴内容不感兴趣，并持续推送更多美妆内容进行试探。

2.  **停留时长**
    *   **含义**：用户点开笔记后的观看时长。
    *   **逻辑**：即使用户常点击美妆内容，但如果每次都快速划走，说明具体内容不吸引他。系统会尝试推送美妆下的其他细分内容（如上妆、护肤）。反之，若用户对偶然推送的家居内容停留很久，系统可能判断其有装修或购买家居用品的需求。

3.  **页面浏览量（PV）**
    *   **含义**：用户浏览页面的深度。
    *   **逻辑**：用户不仅看了某篇笔记，还进入了博主主页，并浏览了多篇其他笔记。这个行为能提升账号的健康度评分，对账号成长有积极帮助。

4.  **互动行为**
    *   **含义**：用户的点赞、收藏、评论、关注等操作。
    *   **逻辑**：这些数据直接提升笔记的热度和权重，有助于笔记进入更大的推荐流量池，同时也能带动账号内其他笔记的曝光，并增加粉丝。

## 平台流量分发入口 🚪

了解用户标签后，我们来看笔记会被分发给用户的哪些入口。小红书平台主要有四个流量分发入口。

以下是四个主要的流量分发入口：

1.  **发现页**
    *   **内容构成**：
        *   已关注达人发布的内容。
        *   根据用户标签（点击率、兴趣）匹配的推荐内容。
        *   部分平台为增加内容多样性而推送的试探性内容。
    *   **价值**：这是笔记和账号获取流量的**主要来源**。

2.  **关注页**
    *   **内容构成**：纯粹展示用户所关注博主发布的最新笔记。
    *   **价值**：用于查看固定关注对象的内容。

3.  **附近页**
    *   **内容构成**：根据手机定位，推送同城或附近的内容，以美食、旅游、本地生活等强地域性内容为主。
    *   **价值**：对于推广线下实体店（如美容院、餐厅）非常有价值。在做信息流广告时，可重点考虑此入口。

4.  **购物页**
    *   **内容构成**：小红书电商和变现的主要入口，包含商品笔记、直播带货和商城。
    *   **价值**：直接匹配用户的购物需求，是商业转化的关键场景。
