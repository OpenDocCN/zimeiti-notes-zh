## 内容审核机制 🔍

上一节我们介绍了内容定位，本节中我们来看看内容发布后的第一道关卡：审核机制。所有平台的内容审核主要分为两大块：**机器审核**和**人工审核**。绝大部分内容的前期审核由机器完成。

![](img/3053a6d5103df0af7900795dc2710a87_9.png)

### 机器审核

![](img/3053a6d5103df0af7900795dc2710a87_11.png)

![](img/3053a6d5103df0af7900795dc2710a87_13.png)

笔记或视频发布后，首先由机器审核内容是否合规。合规性主要包含以下两点：

1.  **敏感词检测**：系统会检测内容（包括图文、口播文案、页面元素）是否包含平台禁止的敏感词。敏感词范围广泛，例如违反广告法的词汇、法律禁止的内容、政治敏感话题以及平台明确禁止的“黑五类”赛道内容。如果存在敏感词，内容可能被直接限流、审核不通过甚至被打回。
2.  **营销广告检测**：平台（尤其是基于兴趣分发的短视频平台和基于种草分享的图文平台）不鼓励直接的硬广内容。如果笔记中包含直接的产品介绍、服务推广、价格优势等营销信息，很可能被判定为营销广告，导致前期不投流或非蓝V账号的内容被限流，损害账号权重。

![](img/3053a6d5103df0af7900795dc2710a87_15.png)

![](img/3053a6d5103df0af7900795dc2710a87_17.png)

![](img/3053a6d5103df0af7900795dc2710a87_19.png)

![](img/3053a6d5103df0af7900795dc2710a87_21.png)

以下是机器审核的核心逻辑伪代码：
```python
def machine_review(content):
    if contains_sensitive_keywords(content):
        return “限流或打回”
    elif is_hard_advertisement(content):
        return “判定为营销广告，限流”
    else:
        return “进入下一步检测”
```

### 查重检测

![](img/3053a6d5103df0af7900795dc2710a87_23.png)

机器审核的另一个重要环节是查重检测。每个平台都有自己的内容数据库。新内容发布后，平台会提取其特征，并与数据库中的历史内容进行比对。

*   如果发现高度重合或完全抄袭的内容，平台会判定为抄袭。
*   后果通常是**打回**或**限流**。

![](img/3053a6d5103df0af7900795dc2710a87_25.png)

因此，创作时应借鉴优秀内容的观点，但必须进行**调整、优化和重新表述**，形成自己的内容。AI工具可以辅助进行内容改写，但需谨慎使用。

![](img/3053a6d5103df0af7900795dc2710a87_29.png)

### 人工审核的介入

![](img/3053a6d5103df0af7900795dc2710a87_31.png)

![](img/3053a6d5103df0af7900795dc2710a87_33.png)

![](img/3053a6d5103df0af7900795dc2710a87_35.png)

![](img/3053a6d5103df0af7900795dc2710a87_37.png)

那么，人工审核在何时介入呢？主要有两种情况：

![](img/3053a6d5103df0af7900795dc2710a87_39.png)

![](img/3053a6d5103df0af7900795dc2710a87_41.png)

![](img/3053a6d5103df0af7900795dc2710a87_43.png)

1.  **机器审核存疑时**：当机器审核发现内容可能存在敏感词或违规风险，但无法完全确定时，会拉长审核时间，并将内容提交给人工审核员进行最终判断。通常，机器审核未通过的内容，在人工审核阶段也很难通过。
2.  **日常内容排查**：平台会有审核人员日常巡查站内内容。你可能遇到之前发布正常的笔记，在后期被突然打回或提示违规。这可能是由于用户举报，也可能是人工排查时发现了问题。对于此类情况，若无重大问题，通常选择申诉、删除或隐藏内容。

人工审核的存在是因为任何自动化规则都存在漏洞。有些内容可能通过特殊方式绕过机器审核，但会在人工排查或用户举报环节暴露风险。作为运营者，应在遵守规则的前提下寻找优化空间。

## 流量分发机制 🚀

![](img/3053a6d5103df0af7900795dc2710a87_45.png)

![](img/3053a6d5103df0af7900795dc2710a87_47.png)

内容审核通过后，下一步就是进入流量分发阶段。系统会决定将内容分发给哪些用户。这个过程主要分为两步：**特征提取**和**数据测试**。

### 第一步：特征提取与初步分类

![](img/3053a6d5103df0af7900795dc2710a87_49.png)

系统首先需要理解你的内容是什么，以及可能适合推送给谁。它会从以下几个维度提取特征并做初步分类：

![](img/3053a6d5103df0af7900795dc2710a87_51.png)

![](img/3053a6d5103df0af7900795dc2710a87_53.png)

![](img/3053a6d5103df0af7900795dc2710a87_55.png)

1.  **关键词分析**：系统通过语义分析、中文分词等技术，从标题、正文、语音识别出的字幕、封面及图片文案中提取核心主题关键词。
2.  **Tag标签**：发布时手动添加的话题或标签为系统提供参考，但并非决定性因素。系统会参考这些标签来辅助理解内容方向。
3.  **账号历史数据**：系统会分析该账号以往发布的内容及互动数据。如果历史内容垂直度高（例如都是护肤主题），且吸引过相关兴趣用户互动，那么新发布的同类内容在分类时会获得更高的权重。

**核心概念**：账号垂直度影响系统对你新内容的分类判断，公式可以简单理解为：
`内容分类权重 = f(关键词匹配度， 标签相关性， 账号历史垂直数据)`

![](img/3053a6d5103df0af7900795dc2710a87_57.png)

### 第二步：数据测试与内容评估

![](img/3053a6d5103df0af7900795dc2710a87_59.png)

![](img/3053a6d5103df0af7900795dc2710a87_61.png)

![](img/3053a6d5103df0af7900795dc2710a87_63.png)

完成初步分类后，系统会将内容先推送给一小部分符合特征的用户进行“数据测试”。根据这部分用户的**行为反馈**，系统判断内容质量，并决定是否扩大推荐范围。主要的评估维度包括：

![](img/3053a6d5103df0af7900795dc2710a87_65.png)

![](img/3053a6d5103df0af7900795dc2710a87_67.png)

以下是评估内容质量的四个核心用户行为指标：

![](img/3053a6d5103df0af7900795dc2710a87_69.png)

![](img/3053a6d5103df0af7900795dc2710a87_71.png)

![](img/3053a6d5103df0af7900795dc2710a87_73.png)

![](img/3053a6d5103df0af7900795dc2710a87_75.png)

1.  **点击率 (CTR)**：`点击率 = 点击量 / 曝光量`。高点击率意味着封面和标题足够吸引人。
2.  **停留时长/完播率**：
    *   视频平台：关注**平均播放时长**和**完播率**。
    *   图文平台：关注用户**平均浏览时长**，包括阅读图文、查看评论、展开更多信息等交互行为所花费的总时间。
3.  **页面浏览深度 (PV)**：指用户在一次访问中浏览的页面数量。例如，用户从推荐页点击笔记，然后进入博主主页，再查看其他笔记。浏览的页面越多，说明内容吸引力越强，对账号整体的数据增益越高。
4.  **互动行为**：包括点赞、收藏、评论、分享、关注等。这些互动行为是内容价值最直接的体现。用户完成其中任何一项，都会为内容数据带来增益。

![](img/3053a6d5103df0af7900795dc2710a87_77.png)

![](img/3053a6d5103df0af7900795dc2710a87_79.png)

![](img/3053a6d5103df0af7900795dc2710a87_81.png)

**核心概念**：内容质量评估是一个综合打分过程，可以概括为：
`内容质量分数 = w1*点击率 + w2*停留时长 + w3*页面深度 + w4*互动率`
（其中w为各维度的权重系数）

## 主要流量入口 🚪

![](img/3053a6d5103df0af7900795dc2710a87_91.png)

![](img/3053a6d5103df0af7900795dc2710a87_93.png)

![](img/3053a6d5103df0af7900795dc2710a87_95.png)

了解内容如何被分发后，我们还需要知道用户从哪里看到内容。以某图文平台为例，主要流量入口包括：

![](img/3053a6d5103df0af7900795dc2710a87_97.png)

以下是平台的几个核心流量来源：

![](img/3053a6d5103df0af7900795dc2710a87_99.png)

![](img/3053a6d5103df0af7900795dc2710a87_101.png)

1.  **发现页/首页**：这是最大的公域流量池。系统根据用户标签和实时热度，混合推荐各类内容。用户在此浏览多基于兴趣探索。
2.  **关注页**：用户在此查看已关注博主的新内容。这部分流量目的性非常强，用户粘性高。
3.  **同城页**：满足用户基于地理位置的需求，如寻找本地美食、住宿、娱乐信息。流量需求明确。
4.  **搜索入口**：用户主动搜索关键词，意图极其明确，通常是解决问题的终点站。
5.  **购物入口**：直接对接电商需求，是转化变现的重要场景。

## 总结 📝

本节课中我们一起学习了新媒体平台的流量分发与审核机制。我们首先剖析了**内容审核**的两道防线：机器审核（敏感词、营销广告、查重）和人工审核（存疑复核、日常排查）。随后，我们深入了解了**流量分发**的两步流程：系统通过提取关键词、标签和账号历史数据进行**特征提取与分类**；然后通过小范围**数据测试**，依据点击率、停留时长、页面浏览深度和互动行为等指标评估内容质量，决定后续流量分配。最后，我们回顾了内容分发的几个主要**流量入口**。
