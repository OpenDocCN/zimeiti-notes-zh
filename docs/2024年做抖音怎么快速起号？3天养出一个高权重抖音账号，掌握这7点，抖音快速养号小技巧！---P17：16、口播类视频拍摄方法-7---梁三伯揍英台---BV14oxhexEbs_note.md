# 抖音运营教程：P17：口播类视频拍摄方法

## 概述
在本节课中，我们将学习口播类视频的具体拍摄方法。掌握正确的拍摄技巧，能让你的视频内容更专业、更具吸引力，从而有效提升账号权重和视频传播效果。

上一节我们介绍了视频拍摄的通用注意事项，本节中我们来看看如何将这些要点应用到具体的视频类型中。拍摄环节至关重要，不会拍摄的同学需要集中注意力学习。拍摄主要需注意以下几点：构图比例、主题突出、画面清晰、光线色彩以及内容连贯。为了便于理解，我们将以两种最常见的视频类型为例进行讲解：口播类和日常生活记录类。这两种类型几乎涵盖了所有拍摄场景。

首先，我们来看第一种类型：口播视频。无论你从事金融、装修、养生还是情感等领域，口播视频的拍摄方法都是相通的。

为了让讲解更直观，我将以一位学员的成功案例为例。你可以在抖音搜索“种小草”，她是一位40岁、带着三个孩子的宝妈，专注于情感口播领域。她曾创作出一条播放量超8000万、单条视频涨粉30万的爆款内容。为什么她能取得如此成绩，而我们涨粉却如此困难？

核心原因在于她的内容能够精准吸引目标群体。看完她的视频，你就能立刻明白她想要吸引的是哪类观众。例如，如果我的目标群体是20多岁的学生，那么我拍摄的所有内容都应围绕学生感兴趣的话题，这样涨粉才会又快又精准。

### 构图与画幅比例
我们来分析一下她的视频构图。抖音平台采用**9:16**的竖屏显示比例，这是所有创作者的基础画布。关键在于，你需要在其中选择是拍摄横屏还是竖屏内容。观察案例视频，她采用的是**4:3**的横屏构图。

这里有一个重要原则：无论你制作哪种类型的视频，都应保持画幅统一。如果你选择横屏，就应全部使用横屏；如果选择竖屏，就应全部使用竖屏。保持一致性是提升账号专业度的细节之一。

### 人物主体与画面布局
在画面布局上，人物主体应占据画面中央约**70%**的面积。切忌将脸部怼满屏幕拍摄，这样会让观众感到不适。确保画面的左、右、上方留有适当空白，这样能使画面看起来更清晰、舒适。

如果你对自己的外貌不够自信，是否还能做口播、拍抖音？答案是肯定的。抖音的成功并非完全依赖于颜值。口播视频的核心在于传递内容和知识。你只需要让观众觉得你形象清晰、得体即可。对于外貌不自信的创作者，可以开启美颜滤镜功能。在这个时代，合理使用美颜功能能让每个人都以更好的状态出镜。请放心使用，只要不过度夸张即可。

### 案例分析：目标群体定位
接下来，我将展示她那条涨粉30万的视频片段。观看时请思考：这个视频旨在吸引哪个特定群体的关注？

> **视频文案示例：**
> “跟婆婆在一起住是什么样的感受？我跟我婆婆在一起住了16年了，平时磕磕碰碰有点小不愉快，那都是正常的。买了衣服不敢光明正大拿回家，新衣服都是放几天再穿，也都不是事儿哈。我最大的感受就是这十多年以来，我娘家人很少在我家住上几天，很少，这对于我内心来说其实是一个很大的缺失。”

有同学可能认为是关于“婆媳关系”。但这是一种关系，具体是哪个群体呢？实际上，对“婆媳关系”最有感触的，绝非男性群体，而是已婚女性群体。未婚或年轻的女性很难对此产生深刻共鸣。因此，这个视频精准吸引的是**已婚女性**观众，使她们能与创作者“种小草”产生强烈共鸣。我们后续可以通过视频数据来验证这一判断。

从案例中我们还可以看到，创作者使用了美颜功能（尽管有些失真），这印证了美颜的实用性。同时，她的构图始终保持**4:3**的横屏比例。

### 拍摄场景与提词技巧
有同学可能会问：口播视频是否必须在户外拍摄？能否在家拍摄？答案是可以的。你可以在任何地方拍摄，只要确保背景整洁、美观即可，避免背景过于杂乱。

另一个常见问题是：口播内容需要背诵吗？我记忆力不好，背不下来怎么办？解决方案在剪辑环节。你可以利用提词器工具，实现边看边读边录制，这会让拍摄过程简单得多。

### 收音清晰度
对于口播视频而言，声音清晰度至关重要。因此，我推荐大家购置一个**手机收声麦克风**。这类设备价格便宜，仅需十几元，在淘宝、京东等平台均可轻松购得。它能显著提升音频质量，确保你的内容传递清晰。

## 总结
本节课我们一起学习了口播类视频的拍摄全流程。我们明确了保持**9:16**竖屏平台下的**4:3**横屏构图一致性，学会了人物应占据画面约**70%** 并留有空白。我们理解了内容应精准定位像**已婚女性**这样的目标群体，并通过案例分析了其重要性。同时，我们解决了拍摄场景、利用提词器避免背诵以及使用**手机收声麦**保证音质等实际问题。掌握这些要点，你就能更专业地拍摄出口播视频，为账号增长打下坚实基础。
