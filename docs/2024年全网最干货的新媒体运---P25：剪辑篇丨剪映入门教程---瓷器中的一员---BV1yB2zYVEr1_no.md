# 新媒体运营系统课：P25：剪辑篇丨剪映入门教程 🎬

在本节课中，我们将要学习视频剪辑软件“剪映”的基础界面和核心操作。通过一个简单的案例，你将了解如何开始创作、导入素材、使用主要工具并完成视频导出。

## 概述

我们将从打开剪映开始，逐步介绍其主界面布局、核心功能区域如预览窗口、工具栏（文本、音频、剪辑等），以及“画中画”、“分割”、“特效”等关键操作。课程旨在帮助初学者快速熟悉剪映的工作环境。

## 基础界面介绍

打开剪映应用后，首先注意两个主要部分。

*   **开始创作**：这是创建新作品的按钮。
*   **草稿箱**：所有已完成或未完成的作品都会自动保存在这里。

接下来，我们通过一个简单案例来了解基础界面。

## 素材导入与预览

点击“开始创作”后，进入素材选择界面。这里需要注意两个分类。

*   **视频**：用于导入视频文件。
*   **照片**：用于导入图片文件。请注意，照片需在此处导入，在“视频”分类中无法找到照片。

剪映内置了丰富的“素材库”，包含各种类型的视频素材，可供下载使用。本节中，我们选择一段黑色背景素材进行讲解。

添加素材后，界面中央是“预览框”，用于预览视频效果。旁边是“比例”调整选项。最基本的横屏比例是 **16:9**。若需制作抖音竖屏视频，则需将比例改为 **9:16**。

调整比例后，我们需要返回“主工具栏”进行后续操作。养成使用后返回主工具栏的习惯很重要。

## 主工具栏功能详解

以下是主工具栏中各核心功能的介绍。

*   **文本**：点击后可为视频添加字幕或文字说明。
*   **音频**：用于为视频添加背景音乐或音效。
*   **剪辑**：选中某段素材后，可使用此功能对该素材进行详细编辑。
*   **贴纸**：为视频添加各种装饰性贴纸元素，类似于现实中粘贴贴纸。
*   **画中画**：此功能可在当前视频上方叠加另一段视频素材。新增的画中画素材会出现在下层轨道，可通过双指捏合进行放大或缩小。上层原有素材可能被覆盖，但并未消失。
*   **分割**：用于裁剪视频素材。选中素材后，使用此功能可将一段长素材切割成两部分。不需要的部分可以删除。此外，拖动素材边缘也可进行精细的长度调整。
*   **特效**：为视频添加视觉特效，使画面更美观。
*   **滤镜**：为视频画面应用不同的色彩预设，快速调整视频色调，类似于照片美颜中的滤镜功能。
*   **背景**：用于更改视频画布的背景样式或颜色。更改后，若被主视频素材覆盖，可通过拉长背景层或调整主素材位置来显示。
*   **导出**：视频制作完成后，使用此功能将作品保存到设备。导出时建议选择 **1080P** 分辨率和 **60帧** 的配置，以获得较高清流畅的画质。

## 界面导航技巧

操作过程中，可能会遇到素材轨道“消失”或找不到的情况。这里有两个实用技巧。

1.  注意轨道左侧的“折叠/展开”小圆圈，点击它可以展开或折叠嵌套的轨道（如画中画轨道）。
2.  直接点击底部工具栏的“画中画”选项，可以快速定位并显示画中画素材。

此外，预览框右上角有放大镜图标，点击可全屏预览视频效果，再次点击则缩小返回。
