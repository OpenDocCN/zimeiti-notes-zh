# 微信视频号运营从0到1全攻略：P9：7. 账号定位与素材获取 🎯

在本节课中，我们将学习账号定位后的两个关键实操步骤：如何处理旧作品以及如何解决原创视频素材问题。这是启动视频号运营前必须完成的基础工作。

上一节我们介绍了账号定位的核心思路，本节中我们来看看定位明确后，具体需要执行哪些操作。

![](img/7e1709e19809230b77e2fef581f51101_1.png)

## 处理旧作品与发布频率

![](img/7e1709e19809230b77e2fef581f51101_3.png)

账号定位确定后，需要将之前与定位不符的作品隐藏。这是为了保持账号内容垂直度，给新观众清晰的第一印象。

发布视频时，素材无需着急。保持一天发布一条视频的频率即可。

![](img/7e1709e19809230b77e2fef581f51101_5.png)

我们的首要目标是积累最初的100个粉丝。先解决粉丝基础问题，再逐步优化素材质量。请按顺序逐个解决问题。

![](img/7e1709e19809230b77e2fef581f51101_6.png)

![](img/7e1709e19809230b77e2fef581f51101_7.png)

## 解决原创视频素材问题

接下来，我们解决原创视频素材的来源问题。许多同学面临不会拍摄、不会剪辑、没有专业素材的困境。

以下是张老师为学员提供的重磅福利：每位同学都可以领取30个已经剪辑好的视频素材。

这些素材来自张老师投入近200万元搭建的独家素材库。库内素材均由专业团队制作，例如搞笑娱乐类视频，都已完成精剪，可以直接使用。

![](img/7e1709e19809230b77e2fef581f51101_9.png)

为了确保素材分发的有序与公平，领取需要满足三个条件：

1.  在直播间公屏上打出“想”字。
2.  在课程中积极互动。
3.  完整听完课程。（后台会统计听课时长与互动数据）

满足条件后，请按以下流程领取：

![](img/7e1709e19809230b77e2fef581f51101_11.png)

*   **联系对象**：找到向你发送听课链接的助理老师。
*   **领取话术**：向助理老师说明“是张老师让我来领取素材的”。这样助理老师会知道这是经过授意的安排。

![](img/7e1709e19809230b77e2fef581f51101_13.png)

![](img/7e1709e19809230b77e2fef581f51101_14.png)

本节课中我们一起学习了账号启动阶段的两个核心操作：隐藏旧作品以明确账号定位，以及通过领取现成素材解决初期视频制作难题。遵循“先解决100粉丝，再优化素材”的顺序，可以更高效地跨过启动门槛。准备好这些基础工作后，我们就可以进入具体的实操教学阶段了。
