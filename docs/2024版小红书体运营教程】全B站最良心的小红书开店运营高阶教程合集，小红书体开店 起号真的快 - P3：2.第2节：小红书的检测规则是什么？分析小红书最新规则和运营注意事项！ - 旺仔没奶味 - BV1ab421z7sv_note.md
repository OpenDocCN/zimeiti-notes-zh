# 小红书运营教程：P3：2.2：小红书平台检测规则详解 📋

在本节课中，我们将要学习小红书平台的核心检测规则。了解这些规则是避免账号违规、确保内容安全发布的基础。我们将从多个维度分析小红书的检测机制，并提供实用的自查工具与方法。

## 平台内容检测范围

上一节我们介绍了小红书的基本运营框架，本节中我们来看看平台具体会检测哪些内容。小红书主要检测以下四个方面的信息：

以下是平台重点检测的四个具体方面：

1.  **个人资料、笔记内容、私信及评论**：这些位置不得含有微信号、微信二维码、代购信息、水印，以及推广淘宝店铺等营销词汇。
2.  **允许的引流方式**：平台官方允许在个人资料中留下邮箱。也可以使用微博账号或谐音词进行间接引流，再通过话术引导至微信。
3.  **设备登录限制**：同一台移动设备最多建议只登录三个小红书账号，超出可能增加风险。
4.  **禁止诱导用户行为**：笔记内容中不得出现诱导用户关注领奖、点赞抽奖等明令禁止的营销行为。

## 内容原创度与发布规范

除了上述明确禁止的内容，平台对内容的原创性保护程度也很高。因此，在推广时需要注意发布规范。

核心原则是：**一篇文章只用一个账号发布**。切勿将同一篇文章在多个小红书账号上重复发布，这种行为容易被系统判定为违规或低质内容。

## 敏感词检测与处理

另一个至关重要的规则是敏感词检测。在创作笔记时，如果无意中使用了违禁词汇，可能导致内容被限流甚至违规。

以下是常见的违禁词类型示例，请尽量避免触及：

*   手机卡
*   金融
*   电子烟
*   代购
*   医疗用品
