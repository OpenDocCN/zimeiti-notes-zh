# 新媒体运营：P28：小红书暴力玩法揭秘！吸粉推广篇 🔍

在本节课中，我们将要学习小红书运营中一个核心的推广策略选择：是自建账号矩阵进行引流，还是与官方合作投放广告。我们将理清这两种方式的关系，并详细介绍各自的运作模式与适用场景。

## 账号类型解析

上一节我们介绍了课程背景，本节中我们来看看小红书平台上的账号主要分为哪几类。理解账号类型是制定推广策略的基础。

以下是小红书平台上的三种主要账号类型：

1.  **企业号**：主要作用是树立官方品牌形象。企业号拥有官方认证标志，可以展示品牌名称、地址等信息，并享有官方特权，便于开展业务。
2.  **个人号**：这是目前大多数用户运营的账号类型。个人号的特点是**灵活性高**，运营者可以是品牌员工或消费者。个人号便于塑造个人品牌特色，也适合品牌相关人物出镜。
3.  **引流号**：这类账号在之前的课程中被称为“炮灰号”。引流号**数量可以多个**，作用通常是辅助推广，用完即可舍弃，或在紧急情况下作为备用小号使用。这类账号通常没有精细的个性规划。

## 企业号矩阵运营建议

了解了账号分类后，我们来看看如何有效利用这些账号进行推广。对于希望建立品牌影响力的运营者，构建一个企业号矩阵是常见策略。

以下是打造企业号矩阵的四点具体建议：

![](img/62f992dc872d130d608a72216d0fe5db_1.png)

1.  **积极参与品牌号活动**：例如，婚纱、服装、美妆等品类的企业号发起试用或团购活动时，可以动用多个小号去参与，增加活动热度。
2.  **个人号展示品牌属性**：在个人号的笔记内容中，应多次暗示品牌的定位。例如，品牌目标用户是城市年轻人还是职场新人，需要在内容中有所体现。
3.  **控制个人号营销内容比例**：个人号发布品牌产品内容的频率不宜过高。一个较好的比例是 **营销内容：种草/科普内容 = 3：7**。即30%的内容用于直接营销，70%的内容用于建立信任和提供价值。
4.  **利用引流号进行评论引流**：企业号发布笔记后，可以使用引流号在评论区进行互动或引导，实现更自然的引流效果。

## 官方广告投放介绍

![](img/62f992dc872d130d608a72216d0fe5db_7.png)

除了自建账号矩阵，另一种重要的推广方式是直接与小红书官方合作投放广告。接下来，我们来看看官方广告的具体形式。

![](img/62f992dc872d130d608a72216d0fe5db_9.png)

![](img/62f992dc872d130d608a72216d0fe5db_11.png)

小红书官方广告主要指 **CBC（Cost Per Click）竞价广告**，即按点击付费的广告。其申请入口位于小红书官网的“业务合作”板块下的“推广合作”。但请注意，**申请官方广告投放的前提是必须已认证企业号**。

![](img/62f992dc872d130d608a72216d0fe5db_13.png)

企业号认证流程同样在官网完成，需要提交相关资质并支付审核费用，通过后普通账号即可升级为官方认证的企业号。

拥有企业号后，即可投放信息流广告。官方广告主要分为两大类：

1.  **信息流广告**：当用户浏览小红书首页信息时，系统会不定期间隔插入广告内容。
2.  **搜索广告**：当用户使用关键词进行搜索时，系统会根据算法匹配并展示相关的广告。例如，用户搜索“帽子”，可能看到搭配推荐的“大衣”广告；搜索“考研资料”，可能看到“营养补充品”广告。其原理类似于百度的关键词竞价排名，通过**对不同关键词出价**来竞争不同的广告展示位。

目前，信息流广告常见的展示位置主要是笔记信息流的第3位、第13位和第23位。

关于广告投放的更多常见问题，可以在小红书广告平台的官方帮助文档中查看详细问答。

![](img/62f992dc872d130d608a72216d0fe5db_15.png)

需要指出的是，官方广告投放的费用相对较高。因此，在资金允许的情况下，建议考虑与官方合作投放竞价广告，因为它能直接获得由平台算法推荐的精准流量。
