# 小红书运营教程：P2：第2节：小红书检测规则详解 📋

在本节课中，我们将系统学习小红书平台的检测规则。了解这些规则是账号安全运营、避免违规限流的基础。我们将从平台检测的多个维度进行分析，并提供实用的自查工具与方法。

## 平台检测的核心维度

上一节我们介绍了小红书运营的基础概念，本节中我们来看看平台具体会从哪些方面进行审核。小红书的检测系统主要覆盖以下四个关键领域。

以下是平台重点检测的四个方向：

1.  **账户与内容信息**：平台会检测个人资料、笔记正文、私信及评论。这些位置**不得含有**微信号、微信二维码、“代购”信息、第三方平台水印，以及推广淘宝等店铺的营销词汇。
2.  **设备与登录环境**：平台会检测手机设备。建议**同一台设备最多只登录三个账号**，过多账号可能被判定为营销号，影响权重。
3.  **用户互动行为**：笔记内容**不得诱导用户**。例如，“关注领奖”、“点赞抽奖”等属于小红书明令禁止的诱导互动行为。
4.  **内容原创度**：小红书对原创内容的保护程度较高。**切勿将同一篇文章在多个账号上发布**。应遵循“一篇文章，一个账号”的原则。

## 敏感词检测与规避方法

除了上述通用规则，内容中的敏感词是触发审核的常见原因。接下来，我们重点学习如何检测和规避敏感词。

小红书对某些领域和词汇较为敏感，例如“手机卡”、“金融”、“电子烟”、“医疗用品”等。在撰写笔记时，应尽量避免涉及。

若不确定内容是否包含敏感词，可以使用专门的检测工具进行自查。以下是两个实用的在线检测网站：

*   **网站一（无需注册）**：打开网站后，将准备好的笔记正文粘贴进检测框，点击“立即检测”。系统会快速标出内容中存在的小红书平台敏感词。
    ![](img/2cf0f7a160400b43bad03dabd6cecd9c_1.png)
*   **网站二（需注册）**：此网站功能类似，注册登录后，同样将内容粘贴入内，即可获得过滤后的结果和敏感词提示。
    ![](img/2cf0f7a160400b43bad03dabd6cecd9c_3.png)

**操作核心**：在发布笔记前，先将文案放入上述任一网站进行检测，根据提示修改或替换敏感词，能有效降低笔记违规风险。

## 官方许可的引流方式

了解禁止项的同时，也需要知道平台许可的运营方式。虽然直接留微信号被禁止，但小红书官方**允许在个人资料中留下邮箱地址**。

![](img/2cf0f7a160400b43bad03dabd6cecd9c_3.png)

此外，一种常见的合规引流策略是：使用“微博”等平台的谐音或代称（如“围脖”）进行初步引导，再通过话术将用户引导至微信。这种间接方式相对更安全，具体的话术技巧将在后续课程中详细展开。

![](img/2cf0f7a160400b43bad03dabd6cecd9c_5.png)
