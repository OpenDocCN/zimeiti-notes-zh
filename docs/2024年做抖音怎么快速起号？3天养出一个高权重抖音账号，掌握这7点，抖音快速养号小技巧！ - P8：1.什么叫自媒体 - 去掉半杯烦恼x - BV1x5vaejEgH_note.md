# 抖音运营入门：P8：1：什么是自媒体

在本节课中，我们将要学习自媒体的基本概念、现状与前景，以及课程的整体框架。这将帮助你理解自媒体的本质，并为后续的账号打造与变现学习打下基础。

首先，课程内容分为四个主要板块。

以下是课程板块的详细介绍：

![](img/d5c1b605d9a1d4beb25982638d5ef320_1.png)

1.  **自媒体现状与前景**：无论全职或兼职进入一个行业，都需要先了解其现状与发展前景。明确这些信息有助于判断该行业是否值得投入，以及能带来哪些益处。了解清楚后再开始操作，会更有信心。
2.  **零基础打造高收益账号**：许多接触过自媒体的同学可能遇到过视频发布后零播放，或在抖音、快手等平台无法通过播放量获得收益的情况。本板块将讲解如何选择平台、领域和渠道，从而真正将收益赚到手。
3.  **亿万流量的多种变现玩法**：多数人了解的自媒体，是通过发布视频或文章获取播放量来结算收益。实际上，除此之外还存在许多其他变现方式。本板块将介绍这些适合零基础同学上手的多种玩法。
4.  **自媒体变现的正确打开方式**：如果你听完课程后，认为自媒体适合自己并有时间操作，本板块将教你如何规划个人账号，并系统性地将收益赚到手。

那么，到底什么是自媒体呢？

![](img/d5c1b605d9a1d4beb25982638d5ef320_9.png)

用通俗的话来解释，自媒体就是：今天我们打开手机，在百度上搜索一个自媒体平台，然后在该平台注册一个属于自己的账号。接着，在自己的账号上发布自己制作的**文章**或**视频**。当内容获得观看时，就可以依靠**播放量**来结算收益。这个过程就叫做自媒体。
