# 抖音运营高阶教程：P23：16、口播类视频拍摄方法-7 🎬

## 概述
在本节课中，我们将学习口播类视频的具体拍摄方法。我们将通过一个成功案例，详细拆解拍摄时需要注意的构图、主题、光线等核心要素，并解释如何通过内容精准吸引目标观众。

---

上一节我们介绍了视频拍摄的通用注意事项，本节中我们来看看口播类视频的具体拍摄方法。

拍摄环节需要注意以下几点：构图比例、主题突出、事物清晰、光线色彩和内容连贯。这些描述可能比较抽象，下面将通过举例来具体说明。主要举两种类型：口播类型和日常生活类型。

![](img/e9f0ab41f00709d1368f4a2acc027c91_1.png)

因为无论你做什么领域的口播，例如金融、装修、养生、情感等，拍摄方法都是相同的。同样，无论你是拍段子、记录生活、拍搞笑内容还是制作美食日常，拍摄方法也都一样。

所以，通过这两种类型，几乎可以囊括所有的拍摄方法。首先，我们来看第一个口播类型的案例。

以下是老师展示的一个学员案例，其账号名为“种小草”。

![](img/e9f0ab41f00709d1368f4a2acc027c91_1.png)

该账号专注于情感口播。账号主是一位40岁、带着三个孩子的宝妈。她的一条视频播放量超过8000万，并通过这一个视频涨粉30万。为什么她一条视频就能涨粉30万，而我们涨粉却如此困难？

原因在于她的内容吸引了非常精准的群体。稍后观看她的视频，你一眼就能看出她的目标观众是谁。例如，如果我的目标群体是20多岁的学生，我拍摄的所有内容都围绕学生感兴趣的话题，那么我的涨粉速度也会非常快，并且粉丝非常精准。

首先说一下构图。抖音是**九比十六**的竖屏平台。关键在于，在竖屏框架内，你选择拍摄横屏还是竖屏内容。案例中的博主拍摄的是**四比三**的横屏画面。

制作视频时，无论哪种类型，不要时而横屏、时而竖屏。如果选择横屏，就全部使用横屏；如果选择竖屏，就全部使用竖屏。保持稳定，注意这些细节。

接着，人物主体应占据画面中央约**70%** 的面积。不要怼脸拍摄，一张大脸充满画面会让人看不下去。确保画面的左、右、上方适当留白，这样看起来画面清晰、舒适。

如果你认为自己不是特别漂亮或帅气，是否还能做口播、拍抖音？答案是肯定的。抖音的成功并非完全依赖颜值。口播的核心在于传达内容和知识。只要让他人觉得你清晰、体面即可。如果对自己外貌不自信，可以开启美颜滤镜。在这个时代，开启美颜滤镜后，就没有丑人。请放心使用美颜，只要别太夸张即可。

![](img/e9f0ab41f00709d1368f4a2acc027c91_3.png)

接下来，我将带你看一下那条涨粉30万的视频。观看后请直接回答：这个视频想要吸引哪个群体的眼球？

![](img/e9f0ab41f00709d1368f4a2acc027c91_3.png)

视频文案示例：
“跟婆婆在一起住是什么样的感受？我跟我婆婆在一起住了16年了，平时磕磕碰碰有点小不愉快，那都是正常的。买了衣服不敢光明正大拿回家，新衣服都是放几天再穿，也都不是事儿哈。我最大的感受就是这十多年以来，我娘家人很少在我家住上几天，很少。这对于我内心来说其实是一个很大的缺失。”

有同学提到“婆媳关系”。那么具体是哪个群体呢？婆媳关系是一种关系，但究竟是哪个群体对此最有感触？实际上，男性群体对婆媳关系的感触绝对没有女性深刻。而且，什么样的人能感受婆媳关系？十几岁、二十几岁未结婚的小姑娘能感受到吗？她们感受不到。一定是已婚的女性群体，才能真真切切地感受到婆媳关系，才能与“种小草”产生共鸣。因此，她吸引的必定是已婚群体。稍后我们可以通过数据进一步验证。

另外，请注意视频中的人物是否开启了美颜。画面有些歪斜，所以如果你觉得自己不是特别上镜，可以适当开启美颜，让自己看起来更体面一些。她的构图也是四比三的比例。

有同学问：“老师，做口播我不想在户外拍，可以在家拍吗？”当然可以，在哪里拍摄都行。只要背景布置得美观一些，不要过于杂乱难看即可。

接着，另一个常见问题是：“老师，我做口播是不是得把稿子背下来？虽然一条视频很短，但我记忆力不好，背不下来怎么办？”

如果背不下来，在剪辑环节，我可以推荐一个好工具。做口播不需要背诵，可以边读边录，这样更简单。

此外，做口播最关键的是声音必须清晰。因此，我推荐大家购买一个便宜好用的设备：手机收声麦克风。价格大约十几块钱，可以在淘宝、京东等平台自行购买。

## 总结
本节课我们一起学习了口播类视频的拍摄方法。核心要点包括：保持统一的横屏或竖屏构图比例；人物主体居中并留有适当留白；内容应精准针对目标群体（如案例中的已婚女性）；合理使用美颜滤镜提升画面观感；如果记不住文案，可以借助工具边读边录；并推荐使用手机收声麦克风来确保音频清晰。掌握这些方法，你也能拍出吸引人的口播视频。
