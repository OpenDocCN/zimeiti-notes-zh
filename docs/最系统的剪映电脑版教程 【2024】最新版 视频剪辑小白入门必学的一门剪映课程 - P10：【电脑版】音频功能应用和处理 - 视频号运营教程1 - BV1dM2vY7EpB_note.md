# 剪映电脑版教程：P10：音频功能应用和处理

![](img/2a0a40f299f5df994648a8a8bd14f602_0.png)

在本节课中，我们将要学习剪映电脑版中音频功能的应用和处理。我们将详细介绍音频面板的各个模块，并通过一个完整的卡点视频案例，学习如何利用这些功能高效地制作出富有节奏感的视频。

## 音频功能模块详解

上一节我们介绍了视频剪辑的基础操作，本节中我们来看看音频功能的具体应用。选中音频素材后，下方会出现五个主要模块。

### 2. 音效库

音效库为视频添加各种声音效果，使画面更生动。以下是音效库的部分分类：

*   **常用音效**：如“叮”、“What”等，适用于日常视频和短视频。
*   **分类音效**：如笑声、综艺、机械、BGM、人声转场、打斗、动物叫声等。

![](img/2a0a40f299f5df994648a8a8bd14f602_16.png)

![](img/2a0a40f299f5df994648a8a8bd14f602_17.png)

### 3. 音频提取

此功能可以从已有视频文件中提取音频。操作方法是导入视频文件，软件会自动分离并提取其中的音频轨道。

### 4. 抖音收藏

登录抖音账号后，在此处可以同步并使用你在抖音App中收藏的音乐。

![](img/2a0a40f299f5df994648a8a8bd14f602_19.png)

### 5. 链接下载

![](img/2a0a40f299f5df994648a8a8bd14f602_21.png)

如果你在抖音上发现了喜欢的音乐但无法收藏，可以将其视频分享链接粘贴到此处，软件会自动下载该音乐到本地库中。

## 卡点视频案例实战

了解了音频的基础功能后，本节我们将通过一个具体的卡点视频案例，学习如何综合运用这些工具。

### 第一步：导入素材与音乐

首先，将案例所需的所有图片素材导入到媒体池。接着，导入背景音乐。我们可以使用“音频提取”功能，从一个已有的示例视频中提取音乐，并拖拽到时间线上。

![](img/2a0a40f299f5df994648a8a8bd14f602_23.png)

### 第二步：自动踩点

卡点视频的关键是让画面切换对准音乐的鼓点。剪映提供了“自动踩点”功能来简化这一过程。

1.  选中时间线上的音频素材。
2.  在工具栏找到并点击 **`自动踩点`** 按钮。
3.  选择 **`节拍2`**，软件会在音频波形上生成密集的鼓点标记。

![](img/2a0a40f299f5df994648a8a8bd14f602_25.png)

这些标记点就是我们需要对齐画面切换的位置。

### 第三步：制作开场动画

开场部分是一个多层叠加的动效文字。

1.  **添加文字模板**：在“文本”->“文字模板”中，找到“怦然心动”模板，拖到时间线开头，并调整其长度至第一个重音鼓点处。修改文字内容为“I think”。
2.  **添加图片与蒙版**：
    *   导入第一张图片（如花朵），在“画面”->“基础”中将其比例调整为 **`3:4`**。
    *   将项目比例也设置为 **`3:4`**。
    *   为图片添加“复古胶片”分类下的“美高”滤镜。
    *   在“特效”->“画面特效”中，找到“矩形”模板，添加到图片上。通过调整模板参数，制作出上下错位排列的三层画面效果。
    *   复制两层矩形模板，分别调整其位置和大小，形成递进出现的动画，并让它们的出现时间对准后续的鼓点。
3.  **添加中心文字**：在画面完全展开的鼓点位置，添加“夏日”文字模板，修改内容为“think”，并放置于画面中央。可以适当降低中间图片层的不透明度，以突出文字。

![](img/2a0a40f299f5df994648a8a8bd14f602_27.png)

### 第四步：快速剪辑图片序列

![](img/2a0a40f299f5df994648a8a8bd14f602_29.png)

在音乐的中段，我们需要快速切换一系列图片来强化节奏感。

![](img/2a0a40f299f5df994648a8a8bd14f602_31.png)

![](img/2a0a40f299f5df994648a8a8bd14f602_33.png)

1.  将多张图片（如小猫、城市风景等）依次拖到时间线上。
2.  根据音频上的鼓点标记，裁剪每张图片的时长，使其切换与鼓点同步。
3.  为每张图片统一应用“美高”滤镜，并确保图片比例为 **`3:4`**。

![](img/2a0a40f299f5df994648a8a8bd14f602_35.png)

### 第五步：制作彩色边框效果

接下来制作一个具有彩色边框的分屏效果。

1.  导入一张图片（如脚部特写），调整比例为 **`3:4`**。
2.  添加“矩形”模板，并复制出多个，竖向排列在画面中。
3.  为所有图片层统一添加“美高”滤镜。
4.  为每一个矩形图层单独添加“负片偏移”特效（位于“特效”->“画面特效”->“热门”或“复古”分类）。
5.  选中每个“负片偏移”特效，在“基础”属性中点击调色板图标，为每个边框更改不同的颜色。
6.  在对应的框内添加文字“You”、“Are”、“Dope”。

### 第六步：制作人物聚焦与分屏效果

![](img/2a0a40f299f5df994648a8a8bd14f602_37.png)

1.  **人物聚焦**：
    *   导入一张人物图片，调整比例。
    *   复制一层，为上层图片添加“矩形”蒙版（在“画面”->“蒙版”中），框出人物主体。
    *   为上层图片添加“负片偏移”特效并调整颜色，形成人物彩色、背景黑白的对比效果。
2.  **上下分屏**：
    *   导入另一张图片，调整比例。
    *   复制一层，分别为上下两层图片添加“矩形”蒙版，各占一半画面。
    *   为上层图片添加“负片偏移”和“黑白胶片”特效，营造风格化对比。

### 第七步：结尾制作与导出

![](img/2a0a40f299f5df994648a8a8bd14f602_39.png)

![](img/2a0a40f299f5df994648a8a8bd14f602_41.png)

1.  在视频结尾添加最后几张图片，并统一色调。
2.  添加结尾文字“like really dope”，使用“花字”样式，并调整描边颜色，制作错落有致的排版。
3.  **导出视频**：
    *   点击右上角 **`导出`** 按钮。
    *   设置文件名称、选择导出文件夹。
    *   参数建议：分辨率 **`1920x1080`**，码率 **`自定义`**，编码 **`H.264`**，格式 **`MP4`**，帧率 **`30`**。
    *   点击“导出”等待完成。
