# 剪映手机版教学：P12：蒙版功能详解 🎬

在本节课中，我们将要学习剪映的核心剪辑功能之一——蒙版。蒙版是一种遮罩功能，常用于隐藏视频中的特定部分或创建转场特效，让视频看起来更加精致和专业。

上一节我们介绍了剪映的基础剪辑功能，本节中我们来看看如何使用蒙版功能。

## 什么是蒙版？ 🎭

蒙版是视频剪辑中常用的遮罩功能。在处理视频时，有时需要隐藏某些内容，不让观众看到。蒙版就是实现这种隐藏效果的工具。在剪映中，蒙版功能可以创造出各种转场特效。

## 蒙版功能的应用案例 📽️

![](img/5d0592a71383c4c5fedb53dc33c58764_1.png)

以下是两个运用蒙版功能的视频案例。

第一个案例展示了双重人影的转场效果。通过蒙版功能，将两个人影结合，并羽化边缘，创造出梦幻般的重叠效果。

![](img/5d0592a71383c4c5fedb53dc33c58764_1.png)

第二个案例展示了五角星拼图效果。这是通过图形蒙版实现的，让视频画面以特定形状（如五角星）显示，形成绚丽的转场。

## 如何在剪映中使用蒙版 🛠️

接下来，我们展示如何在视频中添加和使用蒙版功能。

![](img/5d0592a71383c4c5fedb53dc33c58764_3.png)

首先，点击“开始创作”，导入几张图片素材。正常情况下，图片会按顺序平铺播放，效果较为平淡。

![](img/5d0592a71383c4c5fedb53dc33c58764_3.png)

选中第一张图片，在底部二级菜单中找到“蒙版”选项并点击。剪映提供了多种蒙版类型。

### 线性蒙版 📏

![](img/5d0592a71383c4c5fedb53dc33c58764_5.png)

线性蒙版是最基础的蒙版类型。添加后，画面会被一条线分割，拖动可以调整显示与隐藏的区域。向上拉动越多，遮盖的部分也越多。

**公式表示蒙版作用**：`显示区域 = 原始画面 - 蒙版遮盖区域`

![](img/5d0592a71383c4c5fedb53dc33c58764_7.png)

![](img/5d0592a71383c4c5fedb53dc33c58764_5.png)

![](img/5d0592a71383c4c5fedb53dc33c58764_7.png)

![](img/5d0592a71383c4c5fedb53dc33c58764_9.png)

一些流行的视频，如“遇见当年”效果，就使用了线性蒙版来擦除中间多余部分。

![](img/5d0592a71383c4c5fedb53dc33c58764_9.png)

### 镜像蒙版 🔄

镜像蒙版会向两侧同时应用效果。拖动箭头可以调整羽化值和范围大小，使过渡更加自然。

![](img/5d0592a71383c4c5fedb53dc33c58764_11.png)

### 图形蒙版 ⭐

图形蒙版包括圆形、矩形、星形等。例如，选择星形蒙版，画面就会以星形轮廓显示。

![](img/5d0592a71383c4c5fedb53dc33c58764_13.png)

以下是添加星形蒙版的步骤：
1.  选中素材。
2.  点击底部“蒙版”。
3.  选择“星形”蒙版。
4.  调整位置和羽化值。

![](img/5d0592a71383c4c5fedb53dc33c58764_15.png)

![](img/5d0592a71383c4c5fedb53dc33c58764_11.png)

添加蒙版后，视频就具备了特定的形状转场效果。

![](img/5d0592a71383c4c5fedb53dc33c58764_17.png)

![](img/5d0592a71383c4c5fedb53dc33c58764_13.png)

![](img/5d0592a71383c4c5fedb53dc33c58764_18.png)

## 制作动态蒙版特效 ✨

![](img/5d0592a71383c4c5fedb53dc33c58764_20.png)

为了让蒙版效果动起来，可以结合转场或动画特效，或者使用画中画功能。

![](img/5d0592a71383c4c5fedb53dc33c58764_21.png)

以下是制作动态双重影效果的步骤：
1.  复制视频图层，添加为画中画。
2.  选中画中画图层，点击“蒙版”。
3.  选择一种蒙版类型（如星形），调整位置和羽化值。
4.  为画中画图层添加入场动画（如“向右甩入”）。

![](img/5d0592a71383c4c5fedb53dc33c58764_15.png)

![](img/5d0592a71383c4c5fedb53dc33c58764_17.png)
![](img/5d0592a71383c4c5fedb53dc33c58764_18.png)

![](img/5d0592a71383c4c5fedb53dc33c58764_20.png)
![](img/5d0592a71383c4c5fedb53dc33c58764_21.png)

完成设置后，即可预览到带有动态蒙版的双重叠影效果。

![](img/5d0592a71383c4c5fedb53dc33c58764_23.png)

![](img/5d0592a71383c4c5fedb53dc33c58764_23.png)
