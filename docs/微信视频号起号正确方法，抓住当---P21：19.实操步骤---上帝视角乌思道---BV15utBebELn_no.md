# 微信视频号运营教程：P21：19.实操步骤 - 开通与上架商品

在本节课中，我们将学习微信视频号开通商品橱窗权限以及上架商品的具体操作步骤。整个过程简单直接，即使是初学者也能轻松跟上。

上一节我们介绍了视频号的基础概念，本节中我们来看看如何实际操作，迈出视频号带货的第一步。

![](img/0307eec5fa28de5c5cfeb2d5b8f817d5_1.png)

## 第一步：打开微信
操作的第一步是启动你的微信应用。

![](img/0307eec5fa28de5c5cfeb2d5b8f817d5_2.png)

## 第二步：进入“发现”页
在微信主界面，点击底部的 **“发现”** 选项卡。

![](img/0307eec5fa28de5c5cfeb2d5b8f817d5_4.png)

![](img/0307eec5fa28de5c5cfeb2d5b8f817d5_6.png)

## 第三步：找到视频号入口
在“发现”页面，将视线向上移动。你会看到 **“视频号”** 的入口，其图标是一个黄色的蝴蝶形状。点击进入。

## 第四步：进入个人中心
进入视频号后，点击页面右上角的 **人形头像** 图标，进入个人中心。

![](img/0307eec5fa28de5c5cfeb2d5b8f817d5_8.png)

## 第五步：进入创作者中心
在个人中心页面，向下滑动，直到页面最底部。找到并点击 **“创作者中心”**。

![](img/0307eec5fa28de5c5cfeb2d5b8f817d5_9.png)

**注意**：如果你的账号是全新注册且未发布任何作品，可能暂时看不到此入口。发布一条视频后即可出现。

## 第六步：找到带货中心
进入创作者中心后，在页面中找到 **“带货中心”** 的入口（图标类似一个小卖部），并点击进入。

## 第七步：开通橱窗权限或上架商品
点击“带货中心”后，系统会根据你的账号状态跳转到不同页面。

![](img/0307eec5fa28de5c5cfeb2d5b8f817d5_11.png)

### 情况一：未开通权限，需先开通
如果你的账号尚未开通商品橱窗功能，点击后会进入开通流程。

以下是开通步骤：
1.  **选择带货身份**：页面会提示你选择身份，通常有“我是商家”和“我是达人”等选项。
    *   **核心操作**：无论你是个人还是厂家，初次开通都建议选择 **“我是达人”**。
2.  **缴纳保证金**：选择身份后，会跳转到缴纳保证金的界面。
    *   **核心操作**：点击 **“去交纳”**，系统会引导你在微信支付中完成100元保证金的缴纳。

**公式/代码表示关键选择**：
```
if (用户身份 == 个人 || 用户身份 == 厂家) {
    推荐选择 = “我是达人”;
}
```

![](img/0307eec5fa28de5c5cfeb2d5b8f817d5_13.png)

完成以上两步，即成功开通视频号商品橱窗权限。

### 情况二：已开通权限，直接上架商品
如果你的账号已经开通了橱窗权限，点击“带货中心”后将直接进入商品选品或管理页面，你可以在此浏览、选择并上架你想要推广的商品。

![](img/0307eec5fa28de5c5cfeb2d5b8f817d5_15.png)

## 总结
本节课中我们一起学习了微信视频号开通商品橱窗和上架商品的完整流程。我们首先从打开微信开始，逐步进入视频号的创作者中心，最终到达带货中心。关键在于根据账号状态，完成身份选择（建议选“达人”）和保证金缴纳，即可获得带货资格。开通后，你就可以在带货中心选择商品进行推广了。整个过程清晰明了，现在你就可以动手尝试了。
