# 抖音自媒体运营：P6：3抖音选题思路技巧

在本节课中，我们将学习一套高效的抖音选题公式。这套公式能帮助你快速找到创作方向，解决“不知道发什么”的难题，并吸引精准粉丝，为账号快速打上标签。

## 选题的重要性

你现在做抖音流的泪，都是定选题时脑子里进的水。很多人想选题想得头都秃了，不知道明天该发什么内容。

会找选题的人，会有更多时间去制作视频。如果每天发布三条内容，一个月就是90条。依靠堆量都能形成竞争力，根本不需要额外投入大量推广费用。

![](img/8f9dd3aff204301031a2d23fbf142cad_1.png)

而且，会做选题的人，能比你更精准地找到用户的痛点所在。这更容易得到精准粉丝的驻足观看，从而为账号快速打上标签。

## 核心选题公式

那么，有什么办法解决选题难题吗？当然有。这是一套百万粉丝博主常用的选题技巧。

为了方便理解，我把它总结成了一个公式：

![](img/8f9dd3aff204301031a2d23fbf142cad_3.png)

**选题 = 内容定位 + 用户痛点 + 借势热点**

用好这个公式，你的内容一定不会跑偏。

## 公式应用详解

上一节我们介绍了核心公式，本节中我们来看看如何具体应用。

![](img/8f9dd3aff204301031a2d23fbf142cad_5.png)

以下是公式的三个组成部分及其应用方法：

1.  **内容定位**
    *   这是你账号的垂直领域。你需要挖掘行业内的核心关键词。
    *   例如，知识博主的关键词可以是：播放量、粉丝量、定位、选题等。这些关键词是客观的，不包含主观描述。

2.  **用户痛点**
    *   在内容定位的基础上，寻找目标用户最关心、最困扰的问题。
    *   例如，对于知识博主，用户痛点可能是：如何便捷高效地学习、需要实用技巧、渴望系统教学、不知道如何持续输出内容等。

3.  **借势热点**
    *   结合近期的社会热点、节日、流行话题，与你的内容定位和用户痛点进行组合。
    *   这能有效提升内容的时效性和关注度。

## 实战案例解析

我们已经了解了公式的构成，现在通过具体案例来看看它是如何运作的。

**案例一：知识博主**

*   **内容定位**：知识分享、自媒体运营。
*   **用户痛点**：快速涨粉、高效学习、节日热点内容。
*   **借势热点**：春节。
*   **组合选题示例**：
    *   如何用拜年话术让播放量翻倍？（春节+播放量+技巧）
    *   春节必火的三个选题内容。（春节+选题+热点）
    *   专为同学设计的快速涨粉三个秘诀。（用户群体+涨粉+技巧）

**案例二：美妆行业**

*   **内容定位**：美白、除皱、口红。
*   **用户痛点**：产品平价好用、想要明星同款、需要避坑指南。
*   **借势热点**：明星动态、七夕情人节。
*   **组合选题示例**：
    *   迪丽热巴同款美白面膜推荐。（明星同款+美白+产品推荐）
    *   七夕千万别送女朋友这几支口红。（节日+口红+避坑）
