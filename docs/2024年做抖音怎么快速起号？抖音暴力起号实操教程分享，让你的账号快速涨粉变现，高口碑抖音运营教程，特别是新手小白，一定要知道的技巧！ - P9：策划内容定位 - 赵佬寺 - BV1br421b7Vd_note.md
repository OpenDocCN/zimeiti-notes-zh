# 抖音运营教程：P9：策划内容定位

在本节课中，我们将学习如何为抖音账号策划一个清晰、具体且有吸引力的内容定位。这是账号能否脱颖而出的关键一步。

上一节我们讨论了如何选择主题IP。本节中，我们来看看在确定主题IP之后，如何进一步细化内容定位，使其更具竞争力。

## 主题IP的局限性

![](img/3793a70047ab8f133a914109f813d756_1.png)

许多人在确定主题IP后会产生疑问：这个主题已经有很多人在做了，内容似乎很重复，我的账号还能火起来吗？例如，“老韩摄影教学”最初很火，但后来出现了很多类似的“小韩摄影教学”，新账号就很难再火起来。

![](img/3793a70047ab8f133a914109f813d756_1.png)

这个问题的关键在于，仅仅选择一个热门主题IP是不够的。模仿他人是人之常情，但能否成功取决于更核心的内容策划。

## 细化核心内容要素

在选定主题IP之后，我们需要构思更具体、更细致的核心内容。例如，选择了“摄影教学”这个主题，但具体教什么、拍人还是拍景、用手机还是相机教，这些都需要确定。

除了主题IP，我们需要规划以下具体要素：
*   **身份**：以什么角色出现？
*   **场景**：在什么环境下拍摄？
*   **道具与动作**：使用什么物品？做什么动作？
*   **态度**：表达什么样的情绪或观点？
*   **机位**：采用什么拍摄角度？
*   **视频结构与内容**：视频如何编排？具体讲什么？

内容定位是一个环环相扣的过程。构思得越丰富、越具体，定位就越明确，后续的内容产出也越容易。

## 案例分析：差异化策划的力量

以下是两个通过精细化策划获得成功的案例，它们说明了具体要素的重要性。

### 案例一：暴躁的小风

![](img/3793a70047ab8f133a914109f813d756_3.png)

账号“暴躁的小风”在“态度”这个要素上选择了“暴躁”作为鲜明特点。这种明确的态度定位，结合固定的脚本模式，使其积累了大量粉丝。这属于人物IP打造的一部分。

![](img/3793a70047ab8f133a914109f813d756_5.png)

### 案例二：仙女酵母

![](img/3793a70047ab8f133a914109f813d756_7.png)

让我们详细分析一个更具体的案例：账号“仙女酵母”。她的视频段子内容本身可能并不新颖，但策划非常出色。

![](img/3793a70047ab8f133a914109f813d756_9.png)

![](img/3793a70047ab8f133a914109f813d756_3.png)

（视频脚本示例：在咖啡馆寻找遗忘的包和手机，最后发现闺蜜把自己也忘在了咖啡馆。）

这个段子内容比较常见，但该视频获得了数百万点赞。其成功的关键在于对以下要素的独特打造：
*   **身份**：设定为“公主”或“仙女”角色。
*   **场景与服装**：选择外国、古欧洲风格的场景和华丽服装。
*   **道具**：使用符合角色设定的精致道具。

这种独特的整体策划（身份+场景+道具+服装），即使内容不新颖，也能让账号在众多同类内容中脱颖而出。这也属于人物IP的深度打造。

![](img/3793a70047ab8f133a914109f813d756_11.png)

## 总结

本节课中，我们一起学习了如何策划抖音账号的内容定位。

![](img/3793a70047ab8f133a914109f813d756_11.png)

核心要点是：**内容定位 = 主题IP + 具体要素**。仅仅选择一个热门主题（如“摄影教学”、“美食探店”）是不够的。你必须进一步定义你的**身份、场景、道具、动作、态度和机位**等具体要素。策划得越细致、越有特色，你的账号就越容易在众多竞争者中被记住和关注。

![](img/3793a70047ab8f133a914109f813d756_13.png)

通过“暴躁的小风”和“仙女酵母”的案例，我们看到，即使内容梗本身不新奇，但通过一套独特且一致的要素组合（人物IP打造），也能成功吸引大量粉丝。下一节，我们将学习如何基于这个定位，开始制作你的视频内容。
