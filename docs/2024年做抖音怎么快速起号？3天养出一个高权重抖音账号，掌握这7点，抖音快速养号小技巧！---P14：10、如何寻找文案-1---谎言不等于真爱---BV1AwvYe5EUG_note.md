# 抖音运营实战：P14：10、如何寻找文案-1

在本节课中，我们将要学习如何解决抖音内容创作中的核心难题之一：文案获取。我们将介绍几个实用的在线工具，帮助你快速找到并生成吸引人的文案，甚至解决配音问题，从而提升视频质量，有效吸引粉丝。

上一节我们讨论了优化作品对于吸引粉丝的重要性，本节中我们来看看如何具体解决文案创作这个首要问题。

解决了文案来源，另一个常见问题是配音。很多创作者担心自己的声音不够好听。淘金阁平台内置了“配音在线生成”工具，可以免费解决这个问题。

点击右上角“小工具”，选择第四项“配音在线生成”即可进入功能页。

![](img/a4011b5e6bc05f1f5e70293d1bdc8e53_11.png)

该工具提供了极其丰富的音色选择。

![](img/a4011b5e6bc05f1f5e70293d1bdc8e53_15.png)

![](img/a4011b5e6bc05f1f5e70293d1bdc8e53_13.png)

![](img/a4011b5e6bc05f1f5e70293d1bdc8e53_17.png)

以下是主要的音色类型：

![](img/a4011b5e6bc05f1f5e70293d1bdc8e53_19.png)

*   **亲和女生**、**温柔女生**、**甜美女生**
*   **自然女声**、**卖场广播**
*   **直播配音**（亲切女声、活力女声）
*   **通用场景**（标准男声、标准女声）
*   **方言配音**（四川话、粤语、东北话、台湾话）
*   **外语配音**（英文）
*   **精品男配**（情感解说、影视解说、娱乐解说）

使用方法非常简单。在左侧“配音文本”区域输入或粘贴你的文案。

![](img/a4011b5e6bc05f1f5e70293d1bdc8e53_17.png)

![](img/a4011b5e6bc05f1f5e70293d1bdc8e53_21.png)

在右侧选择喜欢的音色，并调整音量、语速和语调等参数。

![](img/a4011b5e6bc05f1f5e70293d1bdc8e53_23.png)

![](img/a4011b5e6bc05f1f5e70293d1bdc8e53_21.png)
