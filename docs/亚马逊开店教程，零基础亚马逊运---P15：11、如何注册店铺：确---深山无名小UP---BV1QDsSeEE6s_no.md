# 亚马逊开店教程：P15：11、如何注册店铺：确定站点 🌍

![](img/5d77519fea2db1ec9ae5d560beda364e_0.png)

在本节课中，我们将要学习注册亚马逊店铺的第一步：如何选择并确定你要运营的站点。了解各个站点的特点与规则，是开启亚马逊之旅的重要基础。

---

上一节我们介绍了亚马逊的运营模式，本节中我们来看看注册店铺的具体流程。注册店铺的第一个环节，是了解亚马逊的全球站点。

亚马逊有四大主流站点，同时也存在一些小站点。以下是几个主要站点的介绍：

*   **北美站**：包含美国、加拿大和墨西哥三个国家。北美是亚马逊的主要消费区，囊括了将近5亿人口。北美站的月租费用是 **39.99美金**。
*   **欧洲站**：包含英国、德国、法国、西班牙、意大利、荷兰、瑞典、波兰共8个国家。它拥有3.4亿网上购物用户。欧洲站的月租是 **25英镑**。
*   **日本站**：亚马逊在日本电商平台的地位类似于淘宝在中国国内电商平台的地位，相对较高。日本站的月租是 **4900日元**。
*   **澳洲站（澳大利亚站）**：澳大利亚站拥有88%的互联网渗透率，人均GDP位居全球前十，跨境消费率全球第一。中国是澳大利亚的第一出口市场和第一进口来源地。澳洲站的月租是 **49.95澳元**。

此外，像印度站、中东站和新加坡站等，属于规模较小的站点。

在简单了解各个站点之后，我们需要决定从哪个站点开始运营。

首先，亚马逊规定每个卖家只能开通一个店铺，但这个店铺可以同时包含多个站点。例如，开通一个亚马逊店铺后，可以同时激活北美、欧洲、日本和澳洲这四大站点。这意味着 **一个店铺可以运营多个站点**。

那么，我们具体应该从哪个站点开始呢？建议优先选择**北美站**。

选择北美站的原因有以下几点：
1.  **规则与福利**：亚马逊是美国创办的项目，因此北美站是亚马逊规则最健全、福利最多的一个站点。
2.  **全球开店计划**：这是亚马逊的一项重要政策。当你开通了北美站之后，再去开通欧洲站、日本站和澳洲站时，其他站点的月租可以全部免除。公式可以表示为：`总月租 = 北美站月租（39.99 USD）`。这意味着，如果你的店铺同时开通了四大站点，每月只需支付一份北美站的月租即可。

因此，在注册店铺时，第一个要确定的步骤就是：**优先开通一个北美站点**。
