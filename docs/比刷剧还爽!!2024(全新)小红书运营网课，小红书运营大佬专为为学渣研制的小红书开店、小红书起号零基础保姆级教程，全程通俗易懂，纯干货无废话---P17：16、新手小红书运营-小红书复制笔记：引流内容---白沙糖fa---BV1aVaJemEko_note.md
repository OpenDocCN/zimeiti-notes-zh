在本节课中，我们将学习小红书笔记的三种核心修改方式：完全复制、主体复制和格式复制。掌握这些方法，可以有效降低笔记内容的重复率，提升在平台上的权重和曝光机会。

上一节我们介绍了如何选择用于复制的产品、引流内容和分享内容。本节中，我们来看看在选定内容后，如何进行具体的修改操作。

以下是三种核心的笔记修改方式：

**1. 完全复制**
完全复制是指对从其他平台（如1688、抖音）或小红书内整体搬运过来的笔记，进行三次小修改。目的是提升笔记的权重属性，让系统判定其重复内容不多。
*   **视频修改**：使用剪辑软件（如剪映）将完整视频分割成三段，然后重新组合发布。公式可表示为：`修改后视频 = 分割(原视频, 3段) + 重组`。此操作可排除约90%-95%的重复数据量。
*   **图片修改**：使用图片软件，在不显眼处添加两三个小黑点或白点。肉眼难以察觉，但能改变系统识别的图像特征。
*   **文字修改**：更改文字的字体、大小、简繁体，或调整排版（如横排改斜排）。这些改动能使系统判定为接近全新的内容。

![](img/4afa37a6b8d72dac38e5c1eca94e598a_15.png)

完全复制操作简单，适合新用户或用于引流，但因此类内容本身同质化高，权重普遍较低。

**2. 主体复制**
主体复制与完全复制不同，它不直接复制原文案，而是仿写。具体操作是复制产品的整体框架（如图片、视频结构），但文案部分需要自己重新编写，并对产品宣传点、标题等进行修改。
*   **适用场景**：主要适用于图片和文字笔记。视频因其成体系性，较难进行主体复制。
*   **核心操作**：保留有用的框架，替换或修改无用部分，加入自己产品的信息。若想蹭原笔记流量，建议使用完全复制；若只想借鉴格式，则使用主体复制。

![](img/4afa37a6b8d72dac38e5c1eca94e598a_17.png)

**3. 格式复制**
格式复制是指保留原笔记精彩的文案和推广格式，但更换产品的主图图片。适用于原文案很好，但图片质量一般或与自身产品不符的情况。
*   **核心操作**：`新笔记 = 原笔记文案 + 新产品主图`。同时，需将文案中的核心关键词（如产品名称）替换为自己的。

无论是哪种复制方式，都必须对内容进行修改。不修改的笔记几乎100%会与海量现有内容重复，导致权重极低甚至被屏蔽。进行修改后，可以有效规避平台上大量由“店群”模式产生的重复内容，为自己争取到新的竞争空间。
