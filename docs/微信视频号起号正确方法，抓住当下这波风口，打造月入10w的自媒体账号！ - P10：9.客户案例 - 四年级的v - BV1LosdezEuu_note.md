我看到周建国周叔没有回复我，猜测他可能遇到了问题。于是我主动询问他是否遇到了困难。

他回复说，自己记性不太好，总是操作不好。这种情况很常见，尤其是一些年长的学员，他们有时会不好意思提问，担心给老师添麻烦，或者觉得自己学得慢、不够聪明。

如果你也有类似的情况，这很正常。作为老师，我的职责就是帮助大家解决问题。我从不刻意在群里宣扬谁赚了多少钱或进步有多快，因为我知道有些学员自己默默做得很好，但一比较就容易着急。

**核心原则是：不要跟别人比，跟自己比。** 只要今天比昨天有进步，就可以了。

他表示还是不会操作。我说没关系，接下来我通过视频电话一步步指导他。

![](img/250b9a6992e27c0b0c7c5871edfeb3f7_7.png)

我给他打视频电话，他一开始拒绝了。我问他是否不方便，后来接通后，我给他打了大约30分钟的视频电话。我记得很清楚，我教了他四遍。

为什么教了四遍？不是他不肯学，而是遇到了一个具体的技术问题。我让周叔按照我箭头所指的位置，找到视频号的图标。他说找不到那个“黄黄的，像只狐狸”的图标。

我觉得不可能，于是让他截图发过来。截图后我发现，我们的手机界面确实不一样。原因是周叔的手机开启了**老年模式**，导致图标的位置和大小发生了变化，所以他找不到。

要解决这个问题，我想了一个办法。我用一台手机给周叔打视频电话，然后用我的另一台手机，将界面也调成和他一模一样的老年模式。这样，我点哪里，他就跟着点哪里。

![](img/250b9a6992e27c0b0c7c5871edfeb3f7_9.png)

这个办法虽然看起来“笨”，但效果很好。这印证了一个道理：没有教不会的学生，只有不会教的老师。

因为按照正常流程，参与“创作者分成计划”需要100个粉丝并有原创视频。周叔刚注册，显然不符合条件。

但今天有一个福利：各位学员可以通过我的独家内测通道，即使是**零粉丝、零作品、零点赞**的新号，也可以直接加入创作者分成计划。

以下是参与内测通道的步骤：

![](img/250b9a6992e27c0b0c7c5871edfeb3f7_13.png)

1.  获取专属海报：海报上写有“视频号官方内测创作者分成计划”和“院长唯一指定邀请函，0粉丝也可加入”等信息。
2.  扫描专属二维码：海报下方有一个专属二维码，扫描即可添加官方企微。
3.  注意：这个二维码是**一人一码**，扫描后需要将个人信息录入后台系统，这样后续收益才能正确打到你的账户。

接下来讲解大家最关心的收益部分。

![](img/250b9a6992e27c0b0c7c5871edfeb3f7_16.png)

![](img/250b9a6992e27c0b0c7c5871edfeb3f7_17.png)

收益主要由两部分构成：
1.  **流量分成**：按照视频播放量计算，公式大致为 `收益 = 播放量 / 1000 * 10元`。例如，1万播放量约100元。
2.  **保底收益**：这是本计划的核心优势。**每发布一条符合条件的视频，即可获得80元的保底收益**，无论视频数据如何（即使是0播放、0点赞）。

代码示例（收益计算逻辑）：
```python
# 假设保底收益为80元/条，流量分成单价为10元/千次播放
base_income_per_video = 80
cpm_rate = 10 # 每千次播放收益

def calculate_income(play_count, video_count):
    traffic_income = (play_count / 1000) * cpm_rate
    guaranteed_income = video_count * base_income_per_video
    total_income = traffic_income + guaranteed_income
    return total_income

# 示例：发布3条视频，总播放量5万
total = calculate_income(50000, 3)
print(f"预计总收益：{total} 元")
```

经过我的申请，一个账号**每天最多可以发布3条视频**。那么每日保底收益为：
`3条/天 * 80元/条 = 240元/天`

一个账号最多发3条。虽然理论上可以注册多个账号操作，但目前平台规则是**一个身份证只能绑定两个账号**，并且需要遵循“一人一卡一手机”的原则，以防封号。经过测试，一个人操作3-4个账号是比较合理且能忙得过来的上限。

总结关键点：
1.  **因材施教**：面对不同学员（如年长学员）需灵活调整教学方法，解决具体困难（如老年模式）。
2.  **利用特殊通道**：通过内测邀请，可以实现零门槛快速参与分成计划。
3.  **收益结构清晰**：收益由“流量分成”和“保底收益”两部分组成，其中保底收益提供了稳定的收入托底。
4.  **操作有上限**：合理规划账号数量（建议3-4个），在平台规则内稳健操作。

这个案例表明，只要方法得当并利用好平台资源，任何人都可以尝试在视频号上起步并获得收益。
