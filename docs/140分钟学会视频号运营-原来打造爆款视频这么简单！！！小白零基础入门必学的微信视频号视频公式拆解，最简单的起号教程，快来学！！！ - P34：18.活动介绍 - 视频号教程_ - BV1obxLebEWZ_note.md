# 视频号运营：P34：18.活动介绍

在本节课中，我们将详细介绍一个官方推出的视频号商家扶持活动。我们将了解活动的背景、核心负责人、提供的具体资源以及参与活动的步骤。对于初学者而言，这是一个获得官方支持、快速起步的绝佳机会。

## 活动背景与负责人

![](img/3f5616170a4e4d02000545650ef8376c_1.png)

上一节我们介绍了视频号运营的基础，本节中我们来看看一个具体的官方扶持活动。

为促进2024年中国电商行业的新发展，并为视频化带货电商奠定技术基础，微信视频号官方特推出“商家联盟”活动。本次活动由短视频教研部负责人张良老师全程对接。活动秉持公平、公正、公开的理念展开。

![](img/3f5616170a4e4d02000545650ef8376c_3.png)

张老师作为本次活动的全国总负责人，拥有唯一的官方对接权限。这意味着，只有通过张老师这里才能对接参与到本次活动中。

![](img/3f5616170a4e4d02000545650ef8376c_5.png)

## 活动核心扶持资源

了解了活动背景后，我们来看看本次活动为参与者提供了哪些实实在在的扶持资源。以下是活动提供的三大核心支持：

![](img/3f5616170a4e4d02000545650ef8376c_7.png)

1.  **官方视频素材**：针对新手没有拍摄素材的难题，官方已提前准备好高质量产品视频素材。每个参与账号可获得30条视频素材，满足一个月每天发布一条的需求。
    
    ![](img/3f5616170a4e4d02000545650ef8376c_1.png)
    
2.  **专属流量扶持**：为帮助视频获得曝光，每个参与账号将获得90万的流量扶持。这能极大解决视频发布后没有流量、卖不出去货的担忧。
    
3.  **素材库与搜索功能**：官方素材库品类丰富，涵盖中秋节礼品、餐巾纸、牛奶、火锅底料、洗漱手套、不锈钢杯等多种商品。

## 活动奖励与结算方式

![](img/3f5616170a4e4d02000545650ef8376c_11.png)

除了素材和流量，本次活动还设置了极具吸引力的现金奖励政策。

![](img/3f5616170a4e4d02000545650ef8376c_12.png)

活动提供保底收益：参与者每按要求发布一条带货视频，即可获得300元奖励。奖励结算方式为“日结”，即今天发布视频，明天奖励到账。

如果发布完30条素材视频，总计可获得奖励为：
`300元/条 * 30条 = 9000元`

## 活动期限与参与条件

![](img/3f5616170a4e4d02000545650ef8376c_15.png)

需要注意的是，任何活动都有期限。本次活动已于1月13日正式开始，并将于8月10日截止。

![](img/3f5616170a4e4d02000545650ef8376c_17.png)
![](img/3f5616170a4e4d02000545650ef8376c_18.png)
![](img/3f5616170a4e4d02000545650ef8376c_19.png)

参与活动并获得奖励有一个核心前提：必须将官方提供的视频素材发布出去，并正确挂上商品链接。这意味着参与者仍需学习基本的视频发布与带货操作。

![](img/3f5616170a4e4d02000545650ef8376c_17.png)

## 活动参与实操步骤

![](img/3f5616170a4e4d02000545650ef8376c_18.png)

![](img/3f5616170a4e4d02000545650ef8376c_19.png)

最后，我们来了解如何一步步参与这个活动。以下是参与活动的简要操作流程：

1.  通过指定渠道联系并确认参与本次活动。
2.  从官方素材库中获取30条视频素材。
3.  学习如何发布视频并挂载商品链接。
4.  每日发布一条视频，等待奖励结算。
