# 小红书运营：P51：06.小红书后台的基础操作（电脑端）

在本节课中，我们将学习如何通过电脑端操作小红书商家后台，涵盖保证金、支付方式、运费宝和地址设置等核心功能。这些基础设置是店铺正常运营的前提，掌握它们能帮助你更高效地管理店铺。

上一节我们介绍了手机端的操作，本节中我们来看看电脑端后台的具体设置。

## 保证金开通方式

保证金是开店时需要缴纳的一笔押金，用于保障平台和消费者的权益。

*   **前期可暂不缴纳**：对于个人店铺，前期可以暂时不缴纳保证金。
*   **金额与类目相关**：保证金的金额与您选择的商品类目有关，个人店通常在**1000元人民币**左右。不同类目（如二手奢侈品、珠宝玉石等）的保证金可能不同，部分类目个人店无法开通。
*   **缴纳与退还**：保证金在店铺关闭且无违规纠纷的情况下可以退还。但如果店铺存在违规行为（如未履行退款承诺），平台会从中扣款。
*   **长期经营建议缴纳**：若计划长期经营，或希望开通微信支付等功能，则必须缴纳保证金。

![](img/c299967cfaba2308a825153feeaf9824_4.png)

以下是缴纳保证金的步骤：
1.  登录小红书商家后台（千帆系统）。
2.  在首页找到并点击 **“资金”** 模块。
3.  进入 **“店铺保证金管理”** 页面。
4.  选择您的经营类目，系统会显示需缴纳的金额，按提示完成充值即可。

## 支付方式开通

开通支付方式能让顾客使用更便捷的付款工具，提升下单体验。主要有两种方式：支付宝支付和微信支付。

*   **开通位置**：同样在后台的 **“资金”** 模块下，可以找到支付宝和微信支付的开通入口。
*   **开通条件**：**缴纳店铺保证金是开通微信支付的前提**。如果不缴纳，则无法使用微信支付。
*   **开通流程**：
    *   **支付宝**：通常可以立即申请开通。
    *   **微信支付**：需要先缴纳保证金，一般在**第二天**才能进行设置。
*   **提现账户**：店铺的提现金额将进入注册小店时使用的**法人身份证对应的银行账户**。
*   **所需材料**：根据提示绑定相应账户。个体店需要提供营业执照，个人店提供身份证信息即可。

## 运费宝设置

运费宝是一项增值服务，类似于其他平台的“运费险”，能有效降低消费者的购物顾虑。

*   **作用**：当消费者发生退货退款时，系统会自动补贴运费。这能**提升订单转化率**，并**降低因运费产生的纠纷率**。
*   **推荐开通**：建议商家开通此服务，尤其对于无货源代发模式，能避免在运费问题上与顾客产生分歧。
*   **开通路径**：在后台的 **“售后”** 模块中，找到 **“运费宝”** 选项，点击开通即可。

## 地址设置

正确设置发货和退货地址是订单处理的基础，对于无货源（一件代发）模式尤为重要。

以下是地址设置的核心要点：
1.  **设置路径**：进入后台的 **“订单”** -> **“物流工具”**，点击 **“新建地址”**。
2.  **退货地址技巧（针对无货源商家）**：可以将退货地址设置为一个**偏远地区**（如新疆），并在地址后**备注“退货前请联系客服”**。这样能有效减少不必要的退货直接寄回，方便你与供应商协调。
3.  **运费模板**：在物流工具中，还需要设置相应的**运费模板**，根据商品重量、发货地、目的地来配置运费规则。
4.  **有货源商家**：直接填写真实的仓库发货地址即可。
