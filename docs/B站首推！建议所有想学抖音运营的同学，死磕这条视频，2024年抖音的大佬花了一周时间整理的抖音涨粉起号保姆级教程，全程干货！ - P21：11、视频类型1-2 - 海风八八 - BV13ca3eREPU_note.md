# 抖音运营教程：P21：11、视频类型1-2 - 人设与口播详解 📚

在本节课中，我们将要学习抖音账号运营中至关重要的两个概念：**人设**的建立与**口播**视频的制作。理解并应用这些知识，是连接内容创作与粉丝增长、最终实现变现的关键环节。

上一节我们解决了文案问题，本节中我们来看看内容创作的下一个核心环节。很多教程会直接谈论涨粉与变现，但在这两者之间，还有一个必不可少的步骤：**建立你的人设**。

建立人设的目的并非为了炫耀，而是为了在众多同类创作者中脱颖而出。例如，有十个人都做装修内容，如果其中一位“小王同学”树立了一个幽默搞笑的人设，观众就更容易记住他，从而忽略其他九位。当观众因为喜欢你这个人而关注你时，他们就成为了你的粉丝。这就是人设的价值。

现在我们来讲解这个知识点，请边听边记。人设主要分为两种类型。

![](img/48f86c8bee072d014ce208ce91b378b7_1.png)

以下是两种主要的人设类型：

![](img/48f86c8bee072d014ce208ce91b378b7_3.png)

*   **强人设**：以创作者**个人**为内容核心切入点。内容与创作者自身密切相关，例如分享个人行业经验、专业知识或兴趣爱好。这类账号通常需要真人出镜。
    *   **行业经验示例**：装修、办公、房产、汽车、餐饮、婚介、法律咨询、电商等。
    *   **兴趣爱好示例**：美食、旅行、手工、宠物、星座、美妆、护肤、服装穿搭、游戏、摄影等。

*   **弱人设**：内容与创作者**个人**关联度较低，核心目的是运营好账号本身。内容形式大于个人特质。
    *   **典型示例**：
        1.  情感语录类账号（配乐图文，无真人出镜或配音）。
        2.  影视解说类账号（核心是电影内容与解说，解说者的个人特质并非焦点）。

区分清楚强人设与弱人设，是规划账号内容方向的基础。接下来，我们将探讨如何基于人设来生产视频内容。

内容创作环节中，任何与你个人相关的事物都可以成为视频素材。我们首先详细讲解第一种极具潜力的视频形式：**讲故事**。

讲故事可以分享客户案例、从业经历或人生感悟。故事本身不要求完全真实，可以从文案平台寻找有趣的故事进行分享。而讲故事最直接的方式，就是通过**口播**。

口播视频的分类非常广泛。以下是常见的口播类型：

*   **讲故事型口播**：目的是让观众觉得内容有趣、吸引人。
*   **知识分享型口播**：例如育儿、金融、养生、装修、房产、汽车等领域的知识科普。目的是让观众在几十秒内获得“收获感”，觉得学到了东西。

口播形式适合众多领域，如法律、保险、金融、装修知识科普等。它的优势在于**易于拍摄**：只需准备好文案，用手机支架固定机位即可录制。

针对常见顾虑，这里有两个关键解答：
1.  **口才不好能否做口播？**  
    **可以**。口播是依据提前准备好的稿子进行录制，并非临场发挥。大部分口播者的流畅表达都源于充分准备。
2.  **普通话不标准能否做口播？**  
    **可以**。如果你的目标客户是本地或同城人群，使用方言可能更具亲和力和辨识度。同时，可以通过添加字幕来确保信息传达无障碍。

口播是一种后期易于操作和起步的内容形式。在后续课程中，我们会详细教授具体的拍摄技巧。

本节课中我们一起学习了抖音账号运营的核心概念。我们明确了**建立人设**（分为强人设与弱人设）是连接内容与粉丝的关键，并深入探讨了**口播**这一高效的内容形式，包括其类型、优势以及如何克服口才与普通话方面的顾虑。掌握这些基础知识，将为你后续的内容创作打下坚实的根基。
