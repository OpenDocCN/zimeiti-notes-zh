# 亚马逊入门到精通教程：P15：12、如何注册店铺：资料准备 📋

![](img/c5cf5f6ac7f4f3dac765b6b5e611c842_0.png)

在本节课中，我们将要学习注册亚马逊店铺前必须准备的各项资料。了解并备齐这些资料是成功开店的第一步。

## 概述
注册亚马逊店铺需要准备一系列特定的文件和账户。本节将详细介绍每一份资料的具体要求、作用以及获取方式，确保你能顺利完成注册前的准备工作。

## 资料清单与详解
以下是注册亚马逊店铺需要准备的六份核心资料。

### 1. 营业执照彩色扫描件
第一份需要准备的资料是营业执照的彩色扫描件，即营业执照的照片。

*   **执照类型**：可以是企业性质的执照，也可以是个体工商户性质的执照。
*   **办理方式**：可在当地工商局免费办理。
*   **解决方案**：如果没有营业执照，可以借用家人或朋友的执照。如果无法借用，可以寻找代办机构办理，或联系推荐你学习的助理老师协助办理，费用通常在几百到一千元左右。
*   **执照要求**：
    *   **注册资本与年限**：没有硬性要求。无论是1元还是1000万的注册资本，无论是新注册的还是已注册多年的执照，均可使用。
    *   **经营范围**：需要带有销售性质即可，例如“百货零售”、“服装销售”等。
    *   **重要提醒**：**切勿**在经营范围中添加“进出口贸易”字样。添加后，你将必须去相关部门办理进出口手续，这属于画蛇添足。
    *   **经营范围与销售产品**：执照的经营范围与你计划在亚马逊上销售的产品**不需要一致**。例如，使用超市的营业执照注册的店铺，完全可以销售服装。

### 2. 法人身份证正反面照片
第二份需要指明的资料是法人身份证的正反面照片。

*   **关联性要求**：**营业执照上的法人与身份证必须是同一个人**。
*   **与其他资料的关系**：剩余的注册资料可以与这两份资料不是同一个人。例如，你可以借用朋友的营业执照和身份证，而其他资料（如信用卡、收款账户）可以绑定你自己的信息。

### 3. 付款信用卡 💳
第三份需要准备的资料是用于付款的信用卡。

*   **唯一作用**：支付亚马逊的月租费。
*   **卡片类型**：需要全币种信用卡，首选带有 **Visa** 或 **Mastercard** 标志的卡片。
*   **办理银行**：国内各大银行（如工商银行、农业银行、建设银行、招商银行、浦发银行、中信银行等）均可办理。
*   **解决方案**：如果没有信用卡，可以借用他人的，或者花费约100元购买一张虚拟信用卡。只需每月向虚拟卡中充值39.99美元（月租费）即可。
*   **作用说明**：该信用卡主要作用是在店铺尚无销售额时扣除月租。当店铺产生营业额后，亚马逊会优先从店铺余额中扣款，不会频繁动用信用卡。

### 4. 海外收款账户 🌐
第四份需要准备的资料是海外收款账户。

![](img/c5cf5f6ac7f4f3dac765b6b5e611c842_2.png)

![](img/c5cf5f6ac7f4f3dac765b6b5e611c842_3.png)

*   **核心功能**：此账户相当于**国际版的支付宝**。它的作用是接收你在亚马逊店铺的销售款，并提现至你的国内银行储蓄卡。
*   **工作原理**：就像淘宝店的收入通过支付宝提现一样，亚马逊店铺的收入需要通过海外收款账户提现。
*   **获取方式**：可以通过服务商提供的链接进行注册，过程类似于注册支付宝，只需绑定个人身份证和国内银行卡，几分钟即可完成。

![](img/c5cf5f6ac7f4f3dac765b6b5e611c842_4.png)

### 5. 电话号码 📱
第五份需要准备的资料是电话号码。

*   **要求**：使用国内的手机号码即可。

### 6. 邮箱号码 📧
第六份需要准备的资料是邮箱号码。

*   **可用类型**：163邮箱、126邮箱、QQ邮箱等均可使用。
*   **个人推荐**：建议使用**163邮箱**。

![](img/c5cf5f6ac7f4f3dac765b6b5e611c842_6.png)

## 总结
本节课我们一起学习了注册亚马逊店铺前必须准备的六项核心资料：**营业执照扫描件**、**法人身份证**、**付款信用卡**、**海外收款账户**、**电话号码**和**邮箱**。重点在于理解营业执照与身份证的关联性要求，以及各项资料（如信用卡、收款账户）的具体作用和获取方式。备齐这些资料后，你就可以着手进行店铺注册了。
