# 抖音运营入门：P5：抖音短视频的前景

![](img/0116b369e6e7b8ecb58facd831c6a07c_1.png)

在本节课中，我们将要学习抖音短视频平台的整体前景，分析其市场潜力、用户构成以及为何它是适合新手创作者起步的平台。

![](img/0116b369e6e7b8ecb58facd831c6a07c_3.png)

了解抖音短视频的前景，需要先通过一张图片来认识。

![](img/0116b369e6e7b8ecb58facd831c6a07c_1.png)

这张图片来自抖音创作者大会的数据报告。报告显示，有超过2200万人在抖音的合计收入超过了417亿元。并且抖音希望将这个数字翻一番，让创作者的总收入达到800亿元。

![](img/0116b369e6e7b8ecb58facd831c6a07c_3.png)

这体现了平台巨大的流量扶持力度。同时，这里需要明确第一个知识点：抖音用户主要分为两类。第一类是“玩家”，他们单纯地使用抖音娱乐，没有考虑通过平台赚钱。第二类是“创作者”，就像本课程的学员，你们希望通过制作抖音内容来获取收益。区分这两类人很重要，后续内容中还会提及。

那么，为什么我们要选择抖音短视频呢？市面上有快手、火山等多个短视频平台，为何偏偏要做抖音？这类似于开实体店选址。做实体店生意的同学在选址时，肯定会寻找流量大、位置好的地方。做短视频同理，必须选择一个流量庞大的平台。如果平台自身流量不足，就无法分配流量给创作者，创作者也就难以赚钱。

抖音的日活跃用户超过8亿，月活跃用户更是达到了40亿人次。这是一个极其庞大的流量池。流量如此之大，竞争压力会不会也很大？答案是不会。因为平台上的“创作者”数量相对较少，大约只有5000万。这里的“创作者”并非特指网红或达人。例如，邻居家的老太太用手机拍摄自己做饭的视频并上传到抖音，她也算创作者。因此，你90%的竞争压力，实际上来自于身边的普通人。

接着，做抖音短视频是否需要购买电脑或摄像机？答案是不需要。你只需要一部智能手机和一个手机号，就可以注册抖音账号。抖音倡导全民通过短视频的形式分享和记录生活，门槛很低。

那么，当前有什么样的风口值得我们去做呢？首先，抖音平台有明确的扶持计划，例如“抖音将帮助1000名优质的艺术创作者涨粉超过100万”。有的同学可能会说：“老师，我不会创作。”这完全不用担心。如果你什么都会，就不需要来听课了。正是因为大家是新手，才需要学习。老师曾指导过许多学员，包括年纪较大的长辈。他们虚心请教的态度让人印象深刻。一位学员曾说：“虚心请教厉害的人不丢人。只要你在我不懂的方面比我有本事，而我又想学，你就是我的老师。”这种谦虚好学的精神，能让人少走很多弯路。因此，不以年龄定高低，不以现状断未来。老师希望学员都是谦虚好学的，这样老师才愿意倾囊相授。

市场庞大、竞争相对温和、内容创作门槛低，这些都是抖音的优势。我们现在身处大数据时代。在这个时代，平台会根据你的喜好推荐内容。你喜欢看帅哥，刷到的就多是有腹肌的帅哥；你喜欢看美女，刷到的就多是长腿美女。这导致我们每个人看到的内容圈层比较固定。

因此，很多同学会困惑自己该拍什么。以下是几种常见情况：
*   **宝妈**：可以拍摄宝宝的生活片段或自己做的饭菜。
*   **实体店主**：可以拍摄产品介绍或店铺日常，用于引流。
*   **普通上班族**：可以分享日常生活，即使你觉得平淡。

为什么这些内容都有机会？因为每月40亿的用户中，只要你的内容不触犯法律和道德底线，不是低俗内容，总会有一部分人喜欢你的风格。你可能会问：“为什么老师说一定会有人喜欢我，我却只有几十个粉丝？”这是因为喜欢你内容的那部分人，还没有刷到过你的作品。他们无法看到，自然无法成为你的粉丝。

所以，对于新手小白而言，大家迫切想涨粉的心情可以理解。但说句实在话，不要把“涨粉”作为第一个目标。我们的首要目标应该是：**先把流量（播放量）做起来**。当你的视频播放量提升后，粉丝自然就会到来。

本节课中我们一起学习了抖音平台的巨大前景、用户分类、低门槛特性以及新手起步的核心思路。关键在于认识到平台的流量优势，并明确初期应以提升内容曝光（播放量）为首要任务，粉丝增长会随之而来。
