在本节课中，我们将要学习小红书平台的流量结构。了解流量从哪里来，以及如何针对不同流量来源优化笔记，是打造爆款内容、实现有效引流的基础。

上一节我们介绍了笔记制作的整体框架，本节中我们来看看小红书平台流量的具体构成。

## 概述：什么是流量结构？

流量结构指的是小红书平台整体流量的分布和来源。理解流量结构，能帮助我们明确不同类型的笔记应该投放到什么位置，以及它们能获得怎样的效果。

## 小红书流量结构的三大板块

小红书的整体流量结构主要分为三个部分：**发现页**、**搜索页**和**关注页**。以下是这三个板块的详细解析。

### 1. 发现页：流量的主阵地 🔥

发现页是整个小红书最重要的流量来源，占据了平台约**70%**的流量。它也是平台进行流量层级测试的主要场所。一篇笔记最终能获得多大规模的曝光，很大程度上取决于它在发现页获得的用户反馈。

**核心机制**：当你发布一篇笔记后，平台会根据你的账号权重，给予一个初始的基础曝光量（例如1000或2000次）。这**70%**的基础流量几乎全部会出现在发现页。新账号或新笔记的初始曝光，基本都依赖于发现页。

**公式表示**：
`笔记总曝光潜力 ≈ 发现页用户反馈数据 × 平台推荐系数`

因此，优化笔记在发现页的点击率、互动率（点赞、收藏、评论），是获得更大流量的关键。

### 2. 搜索页：精准流量的增长引擎 🔍

搜索页的流量占比约为**20%**，是仅次于发现页的流量来源，并且是小红书平台内增长最快的流量板块。

**为什么重要**：搜索流量代表着用户的**主动意图**。当用户通过关键词搜索商品或内容时，其购买或深入了解的意愿非常强烈。因此，虽然搜索页总流量占比为20%，但其带来的**成交转化率可能占据全平台的70%以上**。

![](img/6c60d89ccaa3ccf58ebb7715ebcef41a_3.png)

**核心概念**：搜索页流量依赖于**关键词**。笔记能否在搜索结果中获得展示，取决于标题、正文、标签中是否包含用户搜索的关键词，以及该笔记的历史权重（点击率、转化率等）。

**注意**：对于新账号而言，直接抢占搜索页流量比较困难，因为这需要笔记具备一定的权重和关键词排名。前期工作应聚焦于发现页，后期再通过优化关键词和积累数据来获取搜索流量。

### 3. 关注页：高粘性的私域流量 ❤️

关注页的流量占比最小，大约在**7%-10%**左右。这部分流量虽然少，但用户最为精准。

**特点**：关注你的用户是对你已发布内容感兴趣的人。向他们推送新内容，互动率和转化率通常较高。这部分流量适合用于维护核心粉丝、发布专属活动或引导至其他私域平台（如微信群）。

**重要限制**：关注页的流量**不计入笔记的基础权重计算**。也就是说，即使你有1万粉丝，笔记发布后全部粉丝都阅读了，这篇笔记也不会因此被系统判定为“爆款”而推向更大的发现页流量池。它的传播范围基本局限于已关注的用户。

**代码逻辑类比**：
```python
if 流量来源 == “关注页”:
    不影响_发现页推荐算法权重()
else:
    计入_平台流量分发模型()
```

## 关于“本地页”的特别说明

在APP中，除了上述三个主页面，你还会看到“本地”页面。该页面主要基于地理位置推荐内容。

**适用场景**：适合本地生活类博主（如美食探店、同城活动）或提供本地即时服务（如上门维修、同城闪送）的账号用于寻找素材或吸引同城客户。

**注意事项**：对于大多数希望将商品卖向全国的账号，**不建议**主要使用本地页进行引流。因为：
1.  它会打乱账号的垂直标签。
2.  流量地域属性强，限制发展空间。
3.  内容混杂，目标用户不精准。

除非你的商业模式本身就是地域性的，否则应谨慎使用。

## 流量利用策略与顺序

理解了三大流量板块的特点后，我们可以制定清晰的运营策略：

1.  **前期主攻发现页**：通过优化封面、标题和内容，提升在发现页的互动数据，撬动平台推荐，获得基础流量和账号权重。
2.  **中期布局搜索页**：在笔记中合理布局行业关键词，积累优质内容。随着账号权重提升，笔记在搜索结果中的排名会逐渐靠前，带来持续、精准的被动流量。
3.  **长期维护关注页**：通过持续输出有价值的内容，积累忠实粉丝。将关注页作为稳定互动和高效转化的私域池。

**流量密码钥匙**：`做好发现页 -> 优化搜索页 -> 维护关注页`

## 总结

![](img/6c60d89ccaa3ccf58ebb7715ebcef41a_5.png)

本节课中我们一起学习了小红书平台的流量结构。我们了解到：
*   平台流量主要由**发现页（约70%）**、**搜索页（约20%）** 和**关注页（约10%）** 构成。
*   **发现页**是流量的起点和爆款的主战场。
*   **搜索页**是转化率最高的精准流量引擎。
*   **关注页**是高粘性用户的私域阵地，但不影响公域推荐权重。
*   运营中应遵循 **“先公域，后私域；先泛流量，后精准流量”** 的顺序，依次在发现页、搜索页和关注页发力。
