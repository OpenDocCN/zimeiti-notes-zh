# 拼多多运营教程：P5：如何在拼多多开店 🛒

在本节课中，我们将要学习如何在拼多多平台完成店铺的入驻流程。无论你是个人还是企业，本教程都将以个人店铺为例，为你详细拆解电脑端和手机端的开店步骤，帮助你从零开始，成功开设自己的拼多多店铺。

## 店铺类型与准备

上一节我们介绍了课程的整体框架，本节中我们来看看开店的具体操作。首先，拼多多店铺主要分为个人店铺和企业店铺。虽然入驻方式基本相同，但需要提交的资质材料有所不同。

![](img/2b82463ed7c7586c087603b927c4647e_1.png)

以下是开店前需要明确的核心概念：
*   **店铺类型**：根据经营主体，选择**个人店铺**或**企业店铺**。
*   **主营类目**：指店铺计划主要销售的商品类别，分为**普通商品**和**医药健康**两大类。

## 电脑端入驻流程

电脑端操作界面更大，适合详细填写信息。以下是电脑端开店的完整步骤：

1.  **访问官网**：在浏览器中搜索并进入拼多多官方网站。
2.  **进入入驻页面**：在官网首页顶部的菜单栏中，找到并点击“**商家入驻**”。
3.  **开始注册**：在打开的页面中，输入一个**从未在拼多多开过店的手机号码**，设置店铺登录密码，输入收到的短信验证码，然后点击“**0元入驻**”。
4.  **选择店铺类型**：根据自身情况，选择个人店或企业店。
5.  **填写店铺信息**：设置店铺名称、登录密码，并选择主营类目。**注意**：如果尚未确定具体经营方向，建议先选择“**普通商品**”，此类目后期可申请变更为“医药健康”（需补充资质），但反之则不行。
6.  **提交资质材料**：按照页面提示，上传开店人的身份证正反面照片等个人信息。如果是企业店，则需上传营业执照等相关材料。
7.  **完成实名认证**：资料提交后，系统会弹出二维码。请使用微信扫描该二维码，根据指引完成人脸识别实名认证。
8.  **等待审核**：所有信息提交完毕后，等待平台审核。个人店铺审核通常较快，一般在**2个工作日**内完成。

![](img/2b82463ed7c7586c087603b927c4647e_3.png)

## 手机端入驻流程

手机端操作便捷，适合随时随地申请。以下是手机端开店的步骤：

1.  **下载APP**：在手机应用商城搜索并下载“**拼多多商家版**”APP。
2.  **开始注册**：打开APP，点击“**0元开店**”。同样输入未注册过的手机号，填写验证码，点击“立即开店”。
3.  **选择与填写**：选择店铺类型（个人店/个体工商户/企业店等），然后根据指引上传身份证照片、设置店铺名称。
4.  **完成验证**：进行人脸识别验证，完成实名认证。
5.  **等待审核**：提交所有信息后，耐心等待平台审核，审核时效与电脑端一致。

## 总结与注意事项

![](img/2b82463ed7c7586c087603b927c4647e_5.png)

本节课中我们一起学习了在拼多多开店的完整流程。核心要点总结如下：

![](img/2b82463ed7c7586c087603b927c4647e_6.png)

*   **流程统一**：无论个人店还是企业店，其**核心入驻流程**（注册->选类型->填信息->传资料->做认证->等审核）是相同的。
*   **类目选择要谨慎**：主营类目首选“**普通商品**”，以便保留后期调整的灵活性。
*   **材料提前备好**：开店前请准备好身份证、手机号等必要材料，企业店铺还需备好营业执照。
*   **审核需要时间**：提交后请耐心等待，个人店审核约需**2个工作日**，企业店（如旗舰店）可能需**3个工作日**。
