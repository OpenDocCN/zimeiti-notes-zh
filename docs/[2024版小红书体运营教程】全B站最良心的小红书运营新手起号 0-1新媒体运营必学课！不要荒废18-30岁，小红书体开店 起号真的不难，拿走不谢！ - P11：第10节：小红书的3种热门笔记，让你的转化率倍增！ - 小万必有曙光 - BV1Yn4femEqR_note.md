# 小红书运营教程：P11：第10节：小红书的3种热门笔记，让你的转化率倍增！ 📈

在本节课中，我们将详细讲解小红书的三种热门笔记类型。掌握这些笔记的创作方法，可以有效提升内容的转化率。建议在学习时做好笔记，以便更好地理解和应用。

![](img/0a32b1bc4ba5d615e3f5e7675ef30ca4_3.png)

上一节我们介绍了账号定位与内容规划，本节中我们来看看具体如何创作高转化率的笔记。

## 教育类笔记 📚

教育类笔记的核心是传授知识或技能，在过程中自然植入产品。这类笔记适合美妆、瘦身、美食、P图软件、美容摄影等行业。

以下是教育类笔记首图的三种常见构成方式：

*   **对比图**：展示使用产品前后的效果对比。例如，口红涂在手上与嘴上的颜色对比，或使用某产品前后的皮肤状态对比。
*   **效果图**：直接展示产品的最终使用效果。例如，眼影上妆后的整体妆容效果。
*   **步骤图**：分解展示操作过程。例如，“五步告别圆肩驼背”教程，过程中可能会展示辅助产品。

![](img/0a32b1bc4ba5d615e3f5e7675ef30ca4_5.png)

关于正文内容，教育类笔记通常遵循一个清晰的框架。以下是撰写正文时可以包含的五个部分：

1.  **使用工具或材料**：列出所需的产品和辅助工具。
2.  **操作步骤**：详细说明每一步的具体做法。
3.  **注意事项**：提示操作中需要避免的问题或关键点。
4.  **产品安利**：在分享经验时，自然地推荐你觉得好用的产品。**注意**：介绍产品的文字篇幅建议不超过全文的30%，以免被判定为硬广。
5.  **常见问题解答**：针对操作中可能出现的失败情况（如“为什么贴不好双眼皮贴”）进行解答，这更能吸引读者。

## 合集类笔记 🗂️

合集类笔记通过汇总推荐多种产品，满足用户特定场景或需求。这类笔记几乎适用于所有行业。

为了避免写作角度单一，我们可以从多个维度切入来创作合集。以下是七种不同的写作角度：

1.  **针对人群**：例如，《学生党开学彩妆攻略》、《上班族代步小电动车推荐》。
2.  **针对成本**：例如，《100元内平价护肤品合集》、《五块钱网红早餐包》。
3.  **针对效果**：例如，《让男朋友合不拢嘴的夏季穿搭》、《瘦身必备零食清单》。
4.  **针对地区**：例如，《香港平价美食推荐》、《南京本地网红店打卡》。
5.  **针对场景**：例如，《旅游必备清单》、《居家清洁好物指南》。
6.  **针对产品属性**：例如，《微电流美容仪测评》、《某某色系口红合集》。
7.  **针对品牌**：例如，《喜茶内部员工隐藏菜单》、《星巴克夏季新品测评》。

合集类笔记容易显得推广意图过重。解决方法是在介绍产品时**均匀分布篇幅**，并**客观说明产品的优点和缺点**，这样能增加可信度，减少推广嫌疑。

合集类笔记的首图主要有两种形式：

*   **产品拼图**：使用美图秀秀、黄油相机等软件将多个产品图片拼接在一起。
*   **产品摆拍图**：将相关产品有序地摆放在一起进行拍摄，例如铺满地面的旅行必备品。

总结一下，合集类笔记的标题应从特定角度切入，正文则应包含**推荐理由**和**产品特点说明**。如果配图已有详细文字说明，正文可简要提及产品名称，这有助于平台搜索引擎抓取关键词。

## 视频类笔记 🎥

视频类笔记是目前平台重点扶持的内容形式，能获得更多流量推荐。如果你有条件，非常建议尝试。

制作视频笔记需要一些基础准备。以下是所需的工具和注意事项：

*   **拍摄工具**：手机、专业相机、三脚架、补光灯。预算充足可考虑简易摄影棚。
*   **剪辑工具**：手机APP推荐**剪映**，电脑专业软件可使用 **Adobe Premiere (PR)**。
*   **视频时长**：小红书默认视频时长为1分钟。发布优质内容达到一定标准后，可申请开通5分钟甚至更长视频权限。
*   **视频尺寸**：推荐两种比例：
    *   **3:4 (竖屏)**：大尺寸，视觉冲击力强，适合大多数内容。
    *   **4:3 (横屏)**：画面更紧凑，适合展示化妆品等精致物品。

![](img/0a32b1bc4ba5d615e3f5e7675ef30ca4_7.png)

关于视频内容，建议你多研究所在行业中成功的视频博主。分析他们的**视频时长、构图、选品和脚本结构**（如开头、中间、结尾如何安排）。模仿并总结成功经验是快速进步的有效方法。

常见的视频内容方向包括教程、测评、好物分享、生活VLOG等。结合你的行业，选择适合的形式进行创作。
