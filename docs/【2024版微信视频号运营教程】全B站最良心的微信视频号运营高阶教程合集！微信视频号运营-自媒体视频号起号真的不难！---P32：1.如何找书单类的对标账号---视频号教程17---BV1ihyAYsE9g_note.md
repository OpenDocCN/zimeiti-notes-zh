# 微信视频号运营教程：P32：如何寻找书单类对标账号

在本节课中，我们将学习在短视频平台（以抖音为例）上，如何高效地寻找与自身定位相符的对标账号。掌握这一方法，能帮助你快速了解目标领域的视频风格和运营模式，为后续的内容创作提供明确参考。

上一节我们介绍了寻找对标账号的重要性，本节中我们来看看具体的操作方法。寻找对标账号的核心思路是：**利用平台算法和商品关联数据**，发现同类型的高质量账号。

以下是两种核心的寻找方法。

![](img/8977e6b7f9fbbe1dc91de8388263d0c1_1.png)

## 方法一：利用“关注”推荐功能

此方法利用抖音的推荐算法，通过关注一个种子账号，来发现更多同类型账号。

![](img/8977e6b7f9fbbe1dc91de8388263d0c1_3.png)

![](img/8977e6b7f9fbbe1dc91de8388263d0c1_4.png)

1.  **搜索关键词**：在抖音搜索框输入你计划运营领域的关键词，例如“书单”、“渔具”或“玩具”。
2.  **筛选种子账号**：从搜索结果中，找到一个视频类型符合你预期、且非真人出镜的账号作为“种子”。
3.  **触发推荐**：进入该账号主页，点击“关注”按钮。
4.  **发现同类账号**：点击关注后，页面下方会立即弹出“你可能感兴趣”的账号列表。这些账号通常与种子账号的标签和类型高度相似。
5.  **批量筛选**：逐一查看推荐账号的作品，若符合你的模仿需求，则点击关注。如此反复，可在短时间内关注大量同类账号。

![](img/8977e6b7f9fbbe1dc91de8388263d0c1_6.png)

完成以上步骤后，平台算法会持续向你推荐类似账号，形成一个高效的学习流。

![](img/8977e6b7f9fbbe1dc91de8388263d0c1_8.png)

## 方法二：利用商品“达人秀”功能

此方法通过具体商品链接，找到推广该商品的大量账号，这些账号的内容形式高度统一，是绝佳的对标样本。

1.  **搜索目标商品**：在抖音搜索框直接输入你计划销售或相关领域的书籍名称，例如“墨菲定律漫画版”。
2.  **切换至商品筛选**：在搜索结果页，将顶部标签从“综合”切换为“**商品**”。
3.  **按销量排序**：进入商品页面后，选择按“**销量**”从高到低排序，找到热销商品。
4.  **进入商品详情页**：点击销量较高的商品，进入其详情页面。
5.  **查找“达人秀”**：在商品详情页中，找到“**达人秀**”或“买家秀”板块。
6.  **发掘对标账号**：“达人秀”板块会展示大量推广该商品的短视频及发布账号。这些账号的内容形式（如影视混剪、语录剪辑）和变现目的高度一致，是极佳的对标学习对象。

![](img/8977e6b7f9fbbe1dc91de8388263d0c1_10.png)

通过“达人秀”，你可以一次性发现数百个同类型账号，彻底解决“找不到参考”的问题。

本节课中我们一起学习了两种在抖音寻找对标账号的高效方法：一是利用关注后的算法推荐，二是通过热销商品的“达人秀”进行挖掘。掌握这些技巧，能让你快速定位学习目标，为后续的视频号内容创作打下坚实基础。
