分析完视频内容后，答案是：这是一个**弱人设**视频。

![](img/286dfa3c614276bb906c79b90d3211bb_33.png)

![](img/286dfa3c614276bb906c79b90d3211bb_35.png)

以下是判断依据：
1.  **内容与发布者无关**：视频讨论的是“穷人”和“富人”的普遍现象，但并未涉及视频创作者本人的经历、观点或故事。人设没有建立起来。
2.  **声画分离**：根据描述，视频画面（可能是治愈系画面）与讲述的严肃社会观点完全无关，进一步削弱了人设的强度。

![](img/286dfa3c614276bb906c79b90d3211bb_37.png)

![](img/286dfa3c614276bb906c79b90d3211bb_39.png)

那么，为什么这样一个弱人设的视频能火呢？原因在于它巧妙地运用了“双向吸引”策略。

![](img/286dfa3c614276bb906c79b90d3211bb_41.png)

![](img/286dfa3c614276bb906c79b90d3211bb_43.png)

以下是该视频成功的关键点：
*   **画面吸引一批用户**：治愈系的画面能满足一部分用户的视觉偏好，吸引他们停留。
*   **观点吸引另一批用户**：尖锐的社会观点能引发大量用户的共鸣和思考，促使他们点赞、评论和转发。

![](img/286dfa3c614276bb906c79b90d3211bb_45.png)

![](img/286dfa3c614276bb906c79b90d3211bb_47.png)

这种“双向吸引”策略可以总结为一个简单的公式：
**视频吸引力 = 视觉元素 + 共鸣观点**

![](img/286dfa3c614276bb906c79b90d3211bb_49.png)

![](img/286dfa3c614276bb906c79b90d3211bb_51.png)

![](img/286dfa3c614276bb906c79b90d3211bb_53.png)

这种思路可以迁移到各种类型的账号运营中。以下是具体应用示例：
*   **宝妈账号**：如果单纯拍摄宝宝或美食没人看，可以尝试为画面配上有趣的育儿故事或家庭趣事。
*   **实体店账号**：如果单纯拍摄产品或工厂没人看，可以尝试讲述你的创业经历、行业故事或客户案例。

我们前期发布的视频如果数据不佳，往往不是因为内容不好，而是缺少这样的小技巧。抖音运营本身就是一个需要不断学习和优化方法的行业。

本节课中我们一起学习了如何通过案例分析区分强人设与弱人设。我们了解到，一个成功的视频不一定需要强人设，通过“双向吸引”的策略——即用视觉元素吸引一波用户，同时用有共鸣的内容吸引另一波用户——同样可以获得高传播度。掌握这一思路，能有效提升视频的完播率和互动数据。
