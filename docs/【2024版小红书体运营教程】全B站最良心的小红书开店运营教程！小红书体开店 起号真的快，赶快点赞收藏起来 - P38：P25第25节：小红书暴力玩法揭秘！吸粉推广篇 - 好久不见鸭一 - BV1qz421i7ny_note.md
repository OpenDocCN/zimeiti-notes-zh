## 账号类型解析

上一节我们介绍了小红书运营的基础，本节中我们来看看小红书平台上的账号主要分为哪几类。理解账号类型是选择推广方式的第一步。

小红书账号主要分为三类：企业号、个人号和引流号。

以下是三类账号的详细说明：

*   **企业号**：主要作用是树立官方品牌形象。企业号拥有官方认证标志，可以展示品牌名称、地址等信息，并享有平台赋予的特权，便于开展官方业务。
*   **个人号**：这是目前大多数用户使用的账号类型。个人号运营灵活，可以是品牌员工或消费者身份。它便于塑造个人品牌特色，也适合品牌相关人物出镜进行内容创作。
*   **引流号**：这类账号在之前的课程中被称为“炮灰号”。引流号可以创建多个，其作用相对次要，通常用于辅助推广，用完即可舍弃，或在紧急情况下作为备用小号使用。这类账号通常没有精细的个性规划。

通常，品牌的推广渠道会通过企业号、个人号或引流号自行发布笔记进行引流。

## 企业号矩阵运营建议

了解了账号分类后，我们来看看如何有效利用这些账号进行协同作战。构建一个有效的账号矩阵能极大提升推广效率。

![](img/7de7f040567f9322caeb70feba7bff7a_1.png)

以下是打造企业号矩阵的四点建议：

![](img/7de7f040567f9322caeb70feba7bff7a_3.png)

![](img/7de7f040567f9322caeb70feba7bff7a_5.png)

1.  **积极参与品牌活动**：品牌的企业号发起试用、团购等活动时，可以动用多个小号（个人号或引流号）积极参与，增加活动热度与真实性。
2.  **个人号展示品牌属性**：在个人号的笔记内容中，应多次暗示品牌的定位属性。例如，品牌定位是城市年轻人还是职场新人，需在内容中清晰体现。
3.  **控制营销内容比例**：个人号发布品牌产品相关内容时，需注意发布频率。建议遵循 **营销内容：种草/科普内容 = 3：7** 的比例。即30%的内容用于直接营销，70%的内容用于种草分享或知识科普。
4.  **利用小号评论引流**：当企业号发布内容希望引流时，可以安排引流小号在评论区进行互动引导，从而实现更自然的用户转化。

## 官方广告投放介绍

![](img/7de7f040567f9322caeb70feba7bff7a_9.png)

除了自建账号矩阵，另一种重要的推广方式是借助平台力量。本节中我们来看看小红书官方的付费广告系统。

![](img/7de7f040567f9322caeb70feba7bff7a_11.png)

小红书官方广告主要指 **小红书CPC广告**，即按点击付费的竞价广告。要投放此类广告，首先需要认证一个企业号。

![](img/7de7f040567f9322caeb70feba7bff7a_13.png)

企业号认证流程为：访问小红书官网 -> 进入“业务合作”板块 -> 选择“推广合作” -> 按照指引提交企业资质并支付审核费用 -> 完成认证。

完成企业号认证后，即可投放信息流广告。官方广告主要分为两大类：

*   **信息流广告**：用户浏览小红书首页信息流时，系统会不定期插入推荐的广告内容。
*   **搜索广告**：当用户使用关键词进行搜索时，在搜索结果中出现的广告。其原理类似于百度竞价排名，通过对关键词出价竞争来获得不同的广告展示位。

目前，信息流广告常见的展示位序是第3位、第13位和第23位。广告位的具体排名由出价和平台算法共同决定。

关于广告投放的常见问题与详细规则，可以在小红书官方网站的“推广合作”相关页面查看官方问答与说明。需要注意的是，官方广告投放需要一定的资质，且费用相对较高。

![](img/7de7f040567f9322caeb70feba7bff7a_15.png)

在资金允许的情况下，建议考虑与官方合作投放竞价广告，因为它能直接获得由平台算法推荐的精准流量。
