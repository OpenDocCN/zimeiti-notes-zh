# 微信视频号运营教程：P38：直播时如何发布草稿箱视频并提升人气

![](img/792652e2a6c59e048b4efc3660046d19_0.png)

在本节课中，我们将学习一个实用的直播技巧：如何在直播过程中发布预先准备好的草稿箱视频，以及如何通过分享直播间来有效提升直播间的观众人数。

## 直播中发布草稿箱视频

![](img/792652e2a6c59e048b4efc3660046d19_2.png)

上一节我们介绍了直播前的准备工作，本节中我们来看看如何在直播进行时，巧妙地发布视频来吸引观众。

![](img/792652e2a6c59e048b4efc3660046d19_4.png)

当您开启直播后，如果发现直播间观众进入缓慢，可以按照以下步骤操作，发布您提前准备好的视频。

![](img/792652e2a6c59e048b4efc3660046d19_6.png)

以下是详细的操作步骤：

![](img/792652e2a6c59e048b4efc3660046d19_8.png)

1.  点击直播间界面右上角的“三个点”菜单按钮。
2.  在弹出的菜单中，选择“最小化”直播窗口。
3.  此时，您可以返回视频号的主页。
4.  在主页中，找到并进入“草稿箱”。
5.  选择您提前准备好的视频，点击“发表”。

**核心操作**可以概括为：**最小化直播窗口 -> 进入主页草稿箱 -> 选择视频发表**。

![](img/792652e2a6c59e048b4efc3660046d19_10.png)

需要注意的是，在直播过程中，如果直接点击直播界面下方的“发表视频”按钮，系统会提示无法操作。必须通过上述“返回主页”的路径才能成功发布。

![](img/792652e2a6c59e048b4efc3660046d19_12.png)

## 分享直播间以增加观众

成功发布视频可以吸引外部流量，同时，主动分享直播间也是提升人气的关键方法。

以下是几种有效的分享方式：

![](img/792652e2a6c59e048b4efc3660046d19_14.png)

*   **分享至朋友圈**：将您的直播间链接分享到微信朋友圈。
*   **转发给好友**：直接将直播间分享给您的微信好友。
*   **分享至微信群**：将直播间链接发布到相关的微信群里。

微信视频号官方对于从私域（如微信群、朋友圈）引入的流量有扶持政策。简单来说，**私域引流人数与平台公域推荐人数存在正相关关系**。例如，您从私域引入10位观众，平台可能会相应地从公域再为您推荐10位潜在观众。

如果您个人不愿意将直播间分享到家人群或同事群，可以尝试使用备用微信号（小号）进行操作。您可以提前让备用号加入一些目标用户可能存在的群组，例如小区业主群、兴趣社群或品牌用户群等。

## 直播表现自查

如果尝试了以上所有方法后，直播间人气仍然不高，或观众进入后迅速离开，那么可能需要审视自身的直播表现。

直播是一项需要练习的技能。请检查以下几点：

*   **话术是否吸引人**：您的讲解是否清晰、有感染力？
*   **状态是否饱满**：您是否精神饱满、充满热情？死气沉沉的状态很容易让观众流失。
*   **内容与产品**：多从自身表达和互动方式上找原因，持续优化。

本节课中我们一起学习了两个核心技巧：第一，通过“最小化直播-主页发稿”的路径在直播中发布草稿箱视频；第二，通过多渠道分享直播间并利用平台的流量扶持规则来提升观众人数。同时，也提醒大家要不断打磨自己的直播状态与话术，这是留住观众的根本。
