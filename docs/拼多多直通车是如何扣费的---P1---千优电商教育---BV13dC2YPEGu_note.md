# 拼多多直通车教程：P1：扣费机制详解与高额扣费规避指南 💰

在本节课中，我们将要学习拼多多直通车广告的扣费核心规则，特别是如何理解并避免因“大单”交易而产生意料之外的高额推广费用。

很多商家被扣除了高额的推广费却不明白原因。例如，自己完成了一笔大额订单，结果被扣了好几百元推广费。

其根本原因在于，商家必须了解拼多多直通车的一条重要规则：**在14天内，任何点击过直通车广告后产生的成交，都会被计入广告扣费**。因此，如果你计划进行大额订单交易（即“做大单”），就需要特别注意。

为了避免在“做大单”时被扣除高额推广费，你可以采取以下两种策略：

以下是两种有效的规避方法：

1.  **使用全新账号**：寻找一个在最近14天内没有访问过你的店铺、且与你没有任何关联的账号来进行这笔大额交易。
2.  **调整出价方式**：将广告出价模式改为“成交出价”。因为按“投产比”出价扣费金额可能较高，而“成交出价”模式是相对固定的。在此模式下再进行大额交易更为稳妥。
