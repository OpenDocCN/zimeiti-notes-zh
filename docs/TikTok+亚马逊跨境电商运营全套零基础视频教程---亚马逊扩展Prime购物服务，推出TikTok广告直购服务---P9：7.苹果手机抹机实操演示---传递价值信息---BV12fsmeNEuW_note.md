在本节课中，我们将学习如何对苹果手机进行“抹机”操作。这是一个基础但关键的步骤，旨在将手机恢复到一个纯净、未激活的状态，并按照目标市场（如美国）的要求进行初始设置，为后续的运营工作做好准备。

![](img/0dbb904e7b17f8d765f4246cf2dc3e5b_2.png)

上一节我们介绍了运营前的准备工作，本节中我们来看看如何具体操作苹果手机的抹机流程。

## 抹机流程概述

苹果手机的抹机流程在不同型号间基本一致。我们将以苹果6S为例进行演示。如果操作其他型号时遇到问题，可以截图或录屏向解答老师咨询。

## 详细操作步骤

以下是苹果手机抹机的完整步骤。

1.  **进入还原选项**
    确保手机处于未激活或已重置状态，未插入SIM卡，仅连接Wi-Fi。打开手机的“设置”应用，向下滑动找到“通用”选项并点击进入。

2.  **执行抹除操作**
    在“通用”设置页面中，继续向下滑动，找到“还原”选项。点击进入后，选择“抹掉所有内容和设置”。系统会弹出确认窗口，点击“抹掉iPhone”以继续。

    ![](img/0dbb904e7b17f8d765f4246cf2dc3e5b_1.png)

3.  **等待手机重启**
    确认后，手机会开始抹除数据并自动重启。这个过程通常需要1到2分钟，请耐心等待。

    ![](img/0dbb904e7b17f8d765f4246cf2dc3e5b_2.png)

4.  **进行初始设置（关键步骤）**
    手机重启后，会进入激活引导界面。请按以下顺序进行设置：
    *   **语言**：按Home键（圆形按钮）开始。选择目标市场语言，例如做美区选择“English”。
    *   **国家/地区**：选择目标市场，例如美国“United States”。
    *   **连接网络**：连接可用的Wi-Fi网络。
    *   **Apple ID**：在询问Apple ID的步骤，选择“忘记密码或没有Apple ID？” -> “稍后在设置中设置” -> “不使用”。
    *   **条款与定位**：点击同意（Agree）条款。在后续关于定位服务、Siri、数据分析等设置中，**全部选择“禁用”或“稍后设置”**。
    *   **外观**：选择默认的显示外观即可。

5.  **完成抹机后的关键设置**
    进入系统主界面后，抹机尚未完全结束，还需进行以下设置：
    *   **时区设置**：进入“设置” -> “通用” -> “日期与时间”，关闭“自动设置”，然后手动将时区设置为目标市场时区，例如美国洛杉矶“Los Angeles”。
    *   **隐私设置**：进入“设置” -> “隐私与安全性”，确保“定位服务”已关闭。同时检查“跟踪”选项（位于定位服务下方），如果存在“允许App请求跟踪”的选项，也将其关闭。
    *   **网络权限**：如果主屏幕上的系统应用显示为等待下载状态，点击它们，在弹出的网络访问请求中，选择“允许使用无线局域网与蜂窝数据”，应用便会开始下载。

完成以上所有步骤后，手机的抹机操作就全部完成了。此时手机已恢复纯净状态，并按照目标市场的基础要求完成了设置。
