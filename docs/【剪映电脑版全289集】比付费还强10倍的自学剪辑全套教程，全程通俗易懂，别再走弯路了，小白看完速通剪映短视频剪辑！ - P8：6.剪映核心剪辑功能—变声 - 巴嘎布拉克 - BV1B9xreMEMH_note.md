# 剪映电脑版教程：P8：6. 剪映核心剪辑功能—变声 🎤

在本节课中，我们将要学习剪映软件中的一个核心功能——**变声**。许多初学者在制作短视频时，可能会觉得自己的原声或配音效果不够理想。此外，在制作包含多人对话或需要特定音效的视频时，变声功能就显得尤为重要。本节教程将详细介绍如何使用剪映的变声功能，将声音转换为女生、男生、萝莉等多种效果，从而丰富视频的听觉体验。

---

上一节我们介绍了其他剪辑功能，本节中我们来看看如何改变视频中的人声。

## 功能简介与用途

变声功能可以改变视频或音频轨道中原始人声的音色。它的主要用途包括：
*   **优化声音**：改善用户对自身原声不满意的情况。
*   **创造角色**：通过将单一声音变为不同音色，模拟视频中的多人对话效果，例如一人分饰父亲、母亲等多个角色。
*   **增加趣味**：在搞笑、玩梗类视频中，使用特殊音效（如怪物声、大叔声）来增强娱乐效果。

## 操作步骤详解

![](img/21b60f1aed87752d90be3672c512751b_3.png)

以下是使用变声功能的具体步骤。

1.  **导入并选中素材**
    打开剪映软件，导入一段包含人声的视频素材。在时间轴上单击该视频片段，使其处于选中状态。

    ![](img/21b60f1aed87752d90be3672c512751b_1.png)

2.  **找到变声功能**
    选中视频后，软件界面底部会弹出二级编辑菜单。在该菜单中，找到并点击 **“变声”** 功能按钮。

    ![](img/21b60f1aed87752d90be3672c512751b_3.png)

3.  **选择变声效果**
    点击“变声”后，界面会显示多种预设的音色效果。软件提供了例如 **“大叔声”**、**“萝莉声”**、**“女生”**、**“男生”**、**“怪物”** 等选项。直接点击任意效果（如“女生”），即可实时应用到当前视频的人声上。

4.  **预览与调整**
    应用效果后，可以播放视频预览变声后的效果。您可以尝试切换不同的音效，直到找到最符合视频需求的那一个。

    ![](img/21b60f1aed87752d90be3672c512751b_5.png)

![](img/21b60f1aed87752d90be3672c512751b_5.png)

## 功能总结与优势

变声功能的使用非常简单，核心操作可概括为：**选中视频 -> 点击底部‘变声’ -> 选择预设效果**。

此功能不仅解决了用户对声音不自信的尴尬，更重要的是，它能高效地帮助新手用户实现复杂的音频效果。通过一人演绎多种声音，可以轻松制作出多人互动的对话场景，极大提升了视频的趣味性和专业质感。
