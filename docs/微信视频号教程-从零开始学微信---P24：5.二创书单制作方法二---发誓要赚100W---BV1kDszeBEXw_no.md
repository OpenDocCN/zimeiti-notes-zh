# 微信视频号教程：P24：5. 二创书单制作方法二

在本节课中，我们将学习如何通过“二创”的方式制作书单视频。我们将详细讲解如何提取原视频素材、替换画面、重新配音并最终合成一个全新的视频。这种方法能帮助你高效地产出原创内容。

## 概述

![](img/efb5647a1aaffef310e81898a819c02c_3.png)

上一节我们介绍了二创的基本概念，本节中我们来看看具体的操作步骤。我们将以《走进大学城》这本书为例，演示如何从抖音寻找素材，并进行剪辑与重组，最终生成一个属于你自己的书单推荐视频。

## 素材寻找与初步处理

![](img/efb5647a1aaffef310e81898a819c02c_9.png)

![](img/efb5647a1aaffef310e81898a819c02c_11.png)

首先，我们需要找到合适的原始视频作为素材基础。

以下是寻找素材的步骤：
1.  在抖音搜索目标书籍的名称，例如“走进大学城”。
2.  在搜索结果中，点击“商品”标签。
3.  选择按“销量”排序，找到相关的带货视频。
4.  进入达人主页，在其视频列表中寻找可用的内容。

例如，我们找到了一个关于“汕头大学”的视频，总时长为1分58秒。这个视频前后部分的声音不一致，我们可以选择只使用前半部分，或者使用整个视频。

![](img/efb5647a1aaffef310e81898a819c02c_13.png)

## 视频剪辑与文案提取

找到素材后，下一步是进行初步剪辑并提取文案。

我们使用剪映APP进行操作。将下载并去除了水印的原视频导入剪映。在声音发生变化的位置（例如53秒处），将视频分割。我们保留前半部分用于提取文案。

操作公式可以概括为：
```
原视频 -> 导入剪映 -> 在音频转折点分割 -> 导出前半部分
```

导出视频片段后，我们使用“配音神器”这类工具来提取文案。在配音神器中，通过“相册视频提取文案”功能，即可将视频中的语音转换成文字。

## 音频替换与素材重组

提取文案后，我们需要为这段文案生成新的配音，并替换原视频中的画面。

以下是核心操作流程：
1.  **生成新配音**：在配音神器中选择合适的声音角色，将提取的文案生成新的音频文件并导出。
2.  **替换视频画面**：回到剪映，处理原视频。我们的目标是替换其中1到2个画面素材，以实现二创。
    *   **定位替换点**：例如，原视频中有一段3秒钟的“毕业典礼”画面。
    *   **寻找新素材**：在抖音搜索“高校毕业典礼”等关键词，下载合适的视频片段。
    *   **执行替换**：在剪映中，删除原视频的3秒片段，将新找到的3秒素材导入并插入。注意关闭新素材的原声，并在衔接处添加“叠化”等转场效果。
3.  **重复操作**：用同样的方法，替换其他可以更换的画面，例如“打高尔夫”、“宿舍内景”等片段。替换的原则是，新画面的内容和格调需与原片段大致相符。

## 音画对齐与最终调整

所有素材替换完成后，我们将新生成的配音音频导入剪映，并关闭原视频的所有声音。

此时需要仔细调整，确保新的音频长度与视频画面总长度匹配。

**音画同步策略**：
*   **音频过长**：可以适当加快视频播放速度（常规变速），或分割删除部分视频片段。
*   **音频过短**：可以适当减慢视频播放速度，或添加一些相关的空镜头、图片素材来延长视频。

调整至音画基本同步后，进行最后几步操作：
1.  **添加背景音乐**：为视频添加一个合适的背景音乐，调节音量使其不干扰配音。
2.  **生成新字幕**：使用剪映的“识别字幕”功能，为新的配音生成字幕。通过调整字幕样式（如背景、字体颜色），覆盖掉原视频可能残留的旧字幕。
3.  **检查水印**：确保最终视频中没有任何抖音或其他平台的原始水印。可以通过放大画面或使用去水印工具处理。
4.  **导出视频**：选择60帧率、1080P分辨率，使用推荐码率，导出最终视频。

## 总结

![](img/efb5647a1aaffef310e81898a819c02c_15.png)

本节课我们一起学习了二创书单视频的完整制作流程。关键步骤包括：**寻找素材 -> 提取文案 -> 生成新音频 -> 替换画面 -> 音画对齐 -> 添加背景音乐与字幕 -> 导出成品**。初学者在前期操作可能会比较慢，但通过多次练习，熟能生巧，制作效率会大幅提升。记住，剪辑的核心在于大胆尝试和不断优化，亲手制作几个视频后，你就能掌握其中的要领。
