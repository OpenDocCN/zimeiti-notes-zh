## 剪辑软件选择与导入

上一节我们介绍了视频制作的前期准备，本节中我们来看看具体的剪辑操作。

![](img/5eda48882fb396428a8772585eb14d3a_9.png)

![](img/5eda48882fb396428a8772585eb14d3a_11.png)

首先，你需要下载并使用正确的剪辑软件。我们推荐使用**剪映**，这是抖音官方的剪辑工具，有手机版和电脑版。请确保下载的是黑色图标的版本。

![](img/5eda48882fb396428a8772585eb14d3a_13.png)

以下是开始剪辑的第一步：

1.  打开手机版剪映App。
2.  点击主界面中的 **`开始创作`** 按钮。
3.  从相册中选择你需要剪辑的视频素材，点击 **`添加`** 将其导入项目。
4.  如果你的视频由多个片段组成（如生活片段），建议先导入并剪辑好第一个片段，然后点击时间轴上的 **`+`** 号，继续导入并剪辑下一个片段。这样可以保持工程清晰，避免混乱。

![](img/5eda48882fb396428a8772585eb14d3a_15.png)

![](img/5eda48882fb396428a8772585eb14d3a_17.png)

## 基础剪辑操作

![](img/5eda48882fb396428a8772585eb14d3a_19.png)

导入素材后，我们开始进行基础的剪辑处理。

![](img/5eda48882fb396428a8772585eb14d3a_21.png)

![](img/5eda48882fb396428a8772585eb14d3a_23.png)

### 调整画面比例与构图

![](img/5eda48882fb396428a8772585eb14d3a_25.png)

![](img/5eda48882fb396428a8772585eb14d3a_27.png)

首先检查视频的构图和比例。如果画面方向或比例不合适，可以进行以下调整：

![](img/5eda48882fb396428a8772585eb14d3a_29.png)

![](img/5eda48882fb396428a8772585eb14d3a_31.png)

1.  在时间轴上选中需要调整的视频片段。
2.  在底部工具栏中向右滑动，找到并点击 **`编辑`** 按钮（图标通常是一个方块）。
3.  在编辑菜单中，点击 **`旋转`** 来调整画面至正确的方向。
4.  如果需要裁剪画面比例，可以点击 **`裁剪`** 选项。这里提供了多种预设比例，如 **`1:1`**、**`4:3`**、**`16:9`** 等，你可以根据发布平台的要求进行选择。

![](img/5eda48882fb396428a8772585eb14d3a_33.png)

![](img/5eda48882fb396428a8772585eb14d3a_35.png)

### 分割与删除无用片段

![](img/5eda48882fb396428a8772585eb14d3a_37.png)

![](img/5eda48882fb396428a8772585eb14d3a_39.png)

调整好画面后，需要修剪掉视频中无效的部分（如开场沉默、结尾冗余动作）。

![](img/5eda48882fb396428a8772585eb14d3a_41.png)

![](img/5eda48882fb396428a8772585eb14d3a_43.png)

1.  播放视频，找到需要裁剪的起始位置，将时间轴上的 **`白色竖线（播放指示器）`** 移动至该处。这根竖线也起到“剪刀”的作用。
2.  选中视频片段，点击底部工具栏中的 **`分割`** 按钮。视频会在此处被切成两段。
3.  选中需要删除的片段（如前一段或后一段），点击 **`删除`** 按钮将其移除。
4.  重复此操作，删除所有不需要的片段。核心操作可以总结为：**`选中 -> 分割 -> 选中 -> 删除`**。

![](img/5eda48882fb396428a8772585eb14d3a_45.png)

## 添加音频与字幕

![](img/5eda48882fb396428a8772585eb14d3a_47.png)

![](img/5eda48882fb396428a8772585eb14d3a_49.png)

视频内容修剪完毕后，下一步是为其添加背景音乐和字幕。

![](img/5eda48882fb396428a8772585eb14d3a_51.png)

### 添加与剪辑背景音乐

![](img/5eda48882fb396428a8772585eb14d3a_53.png)

1.  点击底部工具栏中的 **`音频`** 按钮。
2.  如果你已有准备好的配音文件（例如从其他工具生成），可以点击 **`提取音乐`**，直接从视频文件中导入音轨。
3.  如果没有，则点击 **`添加音乐`**。剪映提供了丰富的音乐库，你可以根据分类（如美食、美妆、萌宠）查找，或直接搜索歌曲。点击歌曲旁边的播放图标可以预览，选定后点击 **`使用`** 即可添加。
4.  如果添加的音乐过长，其剪辑方法与视频相同：将白色竖线移至需要裁剪的位置，**`选中音乐轨道 -> 分割 -> 选中多余部分 -> 删除`**。
5.  可以调整音乐音量：选中音乐轨道，点击 **`音量`** 按钮，拖动滑块即可调节。

![](img/5eda48882fb396428a8772585eb14d3a_55.png)

### 自动识别并添加字幕

![](img/5eda48882fb396428a8772585eb14d3a_57.png)

![](img/5eda48882fb396428a8772585eb14d3a_59.png)

手动添加字幕效率较低，剪映的自动识别功能可以极大提升效率。

![](img/5eda48882fb396428a8772585eb14d3a_61.png)

1.  点击底部工具栏中的 **`文本`** 按钮。
2.  选择 **`识别字幕`** 功能。
3.  在弹出的菜单中，建议选择 **`仅视频`** 进行识别，以避免同时识别出背景音乐里的歌词，造成干扰。
4.  点击开始识别，系统会自动为视频中的人声生成字幕。
5.  识别完成后，检查字幕是否有错别字。双击字幕条即可进入编辑模式进行修改。
6.  你还可以统一或单独修改字幕的**字体**、**样式**（如颜色、大小）等，使其与视频风格更搭配。

![](img/5eda48882fb396428a8772585eb14d3a_63.png)

## 导出设置与发布

![](img/5eda48882fb396428a8772585eb14d3a_65.png)

![](img/5eda48882fb396428a8772585eb14d3a_67.png)

完成所有剪辑后，在导出视频前，有一个关键设置需要注意，它直接影响视频的清晰度。

![](img/5eda48882fb396428a8772585eb14d3a_69.png)

1.  点击右上角的 **`导出`** 按钮。
2.  在导出设置中，调整两个关键参数：
    *   **帧率 (FPS)**：设置为 **`30`**。这是抖音平台最适配的帧率，能保证视频流畅。
    *   **分辨率**：尽可能选择最高选项（如1080P或4K）。分辨率越高，视频越清晰。
3.  设置完成后，点击导出。**在导出过程中，请勿锁屏或切换至其他应用程序**，以免中断导出进程。
4.  导出完成后，你可以直接通过剪映的分享功能，将视频发布到抖音。

![](img/5eda48882fb396428a8772585eb14d3a_71.png)

## 口播拍摄辅助工具推荐

![](img/5eda48882fb396428a8772585eb14d3a_73.png)

对于需要录制口播视频的同学，背诵台词可能是一个难题。这里推荐一个实用的辅助工具：**爱提词**。

![](img/5eda48882fb396428a8772585eb14d3a_75.png)

![](img/5eda48882fb396428a8772585eb14d3a_77.png)

这款App的核心功能是提词器，可以让你在拍摄时看着台词念，无需背诵。

![](img/5eda48882fb396428a8772585eb14d3a_79.png)

![](img/5eda48882fb396428a8772585eb14d3a_81.png)

以下是其主要使用方法：

![](img/5eda48882fb396428a8772585eb14d3a_83.png)

![](img/5eda48882fb396428a8772585eb14d3a_85.png)

1.  **添加台词**：打开App，点击右下角的 **`+`** 号，创建新文稿。你可以直接输入或粘贴提前准备好的台词内容，并为其设置一个标题。
2.  **使用提词器**：保存文稿后，点击 **`开始提词`**，选择 **`录制模式`**。台词会以悬浮窗形式显示在手机屏幕上，你可以一边看词一边录制。
3.  **个性化设置**：
    *   **调整字体**：点击屏幕中央的设置按钮，可以调整台词文字的字体、大小和滚动速度，方便阅读。
    *   **开启美颜**：在拍摄界面，左下角有 **`美颜`** 功能。你可以调整瘦脸、大眼、磨皮、美白等参数，让拍摄效果更自然美观，不同于一些滤镜过重的美颜相机。
4.  **获取灵感**：爱提词App内还提供了一个 **`爆款文案`** 板块，分类整理了职场、情感、生活、科普等多个领域的文案素材，可以作为你创作内容的参考。

![](img/5eda48882fb396428a8772585eb14d3a_87.png)

本节课中我们一起学习了使用剪映进行视频剪辑的全流程，包括素材导入、画面调整、分割删除、添加音效字幕以及正确的导出设置。同时，还了解了“爱提词”这款能极大提升口播视频制作效率的辅助工具。掌握这些基础操作和工具，你就能独立完成一个完整视频的剪辑与制作。
