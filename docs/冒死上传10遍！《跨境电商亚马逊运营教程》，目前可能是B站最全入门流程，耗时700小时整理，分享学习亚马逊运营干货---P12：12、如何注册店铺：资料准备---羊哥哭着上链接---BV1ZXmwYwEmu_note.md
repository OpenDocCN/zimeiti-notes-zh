# 跨境电商亚马逊运营教程：P12：如何注册店铺：资料准备 📝

![](img/82d718ca31ee2ab1bc3aebab97de7b3b_0.png)

在本节课中，我们将要学习注册亚马逊店铺前必须准备的各项资料。了解并备齐这些资料是成功开店的第一步。

上一节我们介绍了如何选择亚马逊站点，本节中我们来看看注册店铺需要准备哪些具体材料。

## 资料清单总览

注册亚马逊店铺需要准备以下六项核心资料。以下是每一项资料的详细说明与注意事项。

1.  **营业执照彩色扫描件**
    *   此文件为营业执照的彩色照片或扫描件。
    *   执照类型可以是企业执照，也可以是个体工商户执照。
    *   执照可在当地工商局免费办理。
    *   若没有执照，可借用家人或朋友的执照，或寻找代办机构协助办理，费用通常在几百至一千元人民币左右。
    *   **执照要求**：
        *   对注册资本和注册年限无硬性要求。
        *   经营范围需带有销售性质（如零售、批发）。
        *   **重要提示**：请勿在经营范围中添加“进出口贸易”字样，否则需额外办理进出口手续，增加不必要的复杂度。
        *   执照的经营范围与店铺实际销售的产品无需一致。

2.  **法人身份证正反面照片**
    *   此文件为营业执照上对应法人的身份证正反面照片。
    *   **核心要求**：营业执照与法人身份证必须是同一个人的信息。

3.  **付款信用卡**
    *   此信用卡的唯一作用是支付亚马逊的月租费。
    *   信用卡需为全币种信用卡，卡面需带有 **Visa** 或 **Mastercard** 标识。
    *   可在各大银行（如工商银行、建设银行、招商银行等）申请办理。
    *   若无信用卡，可借用他人的，或花费约100元购买一张虚拟信用卡，每月按时存入月租费（39.99美元）即可。
    *   **补充说明**：店铺产生营业额后，亚马逊会优先从店铺余额中扣除月租，信用卡并非主要扣款渠道。

4.  **海外收款账户**
    *   此账户用于接收亚马逊店铺的销售货款，并将其提现至个人储蓄卡。
    *   其功能类似于国际版的支付宝。
    *   可通过服务商链接快速注册，绑定个人身份证和银行卡即可使用。

![](img/82d718ca31ee2ab1bc3aebab97de7b3b_4.png)

5.  **电话号码**
    *   需要一个有效的中国大陆手机号码，用于接收验证信息。

6.  **电子邮箱**
    *   需要一个常用的电子邮箱地址，用于注册和接收官方通知。
    *   推荐使用163邮箱，126邮箱或QQ邮箱也可。

本节课中我们一起学习了注册亚马逊店铺所需的六项基本资料：营业执照、法人身份证、付款信用卡、海外收款账户、电话号码和电子邮箱。确保这些资料齐全且符合要求，是顺利完成店铺注册流程的关键。下一节，我们将进入具体的注册操作步骤。
