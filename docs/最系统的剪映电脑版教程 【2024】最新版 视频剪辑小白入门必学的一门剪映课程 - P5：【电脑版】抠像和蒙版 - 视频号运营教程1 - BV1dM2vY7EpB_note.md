# 剪映PC版教程：P5：抠像与蒙版

![](img/33490d6b1bf4c085834b33f7159eb67e_0.png)

在本节课中，我们将学习剪映中两个非常强大且有趣的功能：**抠像**和**蒙版**。通过它们，我们可以轻松实现人物从电视中爬出、无实物托举等充满创意和视觉冲击力的合成效果。课程将从核心概念讲起，并通过具体案例演示操作步骤。

## 概述

上一节我们介绍了关键帧动画，本节我们将深入探讨视频合成的核心技术。**抠像**功能允许我们将视频中的特定主体（如人物、物体）从原始背景中分离出来。**蒙版**功能则能让我们以各种形状来显示或隐藏视频的特定部分，实现画面的无缝融合。掌握这两个工具，是进行高级视频创作的基础。

## 抠像功能详解

抠像，顾名思义，就是将画面中的“像”（主体）从背景中“抠”出来。在剪映中，主要有三种抠像方法，适用于不同的场景。

以下是三种抠像工具的介绍：

1.  **色度抠像**
    *   **核心原理**：通过吸取并移除画面中的纯色背景（如绿幕、蓝幕）来抠出主体。其操作可概括为公式：`最终画面 = 原始画面 - 选取的色块`。
    *   **适用场景**：主体处于纯色背景前的视频素材。
    *   **操作方法**：在“抠像”面板选择“色度抠像”，使用吸管工具吸取背景颜色，然后调整“强度”和“阴影”参数，直到背景被干净地移除。

2.  **自定义抠像**
    *   **核心原理**：使用智能画笔手动描绘需要保留的主体区域，软件会自动识别边缘完成抠像。其逻辑类似于代码中的区域选择：`select_area(画笔描绘的区域)`。
    *   **适用场景**：背景复杂、无法使用色度抠像的素材。
    *   **操作方法**：选择“自定义抠像”，用“智能画笔”涂抹主体。如有误选，可用“智能橡皮擦”或“橡皮擦”进行修正，最后点击“应用效果”。

3.  **智能抠像**
    *   **核心原理**：软件自动识别人物轮廓进行抠像。其内部算法可理解为：`if 检测到人物特征: 则进行抠像`。
    *   **适用场景**：画面主体为人的视频素材。
    *   **注意事项**：此功能仅针对人物有效。对于动物、物体等，需使用“自定义抠像”。

## 蒙版功能详解

了解了如何分离主体后，我们来看看如何将不同的画面元素优雅地融合在一起。蒙版功能就像一块有形状的挡板，可以控制图层哪些部分显示、哪些部分隐藏。

剪映提供了多种形状的蒙版，以下是其主要类型和应用：

*   **线性蒙版**：创建一条直线分割线，常用于制作两个画面的上下或左右拼接过渡。
*   **镜面蒙版**：创建两条平行线，显示中间部分，隐藏两侧，适合制作电影感的宽荧幕效果或特殊聚焦。
*   **圆形蒙版**：显示圆形区域内的画面，可用于制作镜头光晕、局部特写或圆形画中画。
*   **矩形蒙版**：显示矩形区域内的画面。其独特之处在于可以调整“圆角”参数，让矩形变成圆角矩形，使边缘过渡更柔和。
*   **爱心与星形蒙版**：显示爱心或星形区域内的画面，用于添加创意图形效果。

**蒙版通用参数**：
*   **位置/大小**：可拖动蒙版控制点或输入数值调整。
*   **旋转**：控制蒙版的角度。
*   **羽化**：`羽化值越高，边缘过渡越模糊、越自然`。这是实现无缝融合的关键参数。

## 案例实战：制作创意合成效果

现在，我们将综合运用抠像和蒙版，还原课程开头展示的几个创意效果。

### 案例一：人物从电视机中爬出

这个效果需要将绿幕前拍摄的人物合成到电视画面中。

1.  **抠出人物**：将绿幕人物素材拖入时间轴。在“抠像”面板使用**色度抠像**，吸取绿色背景并调整强度，初步抠除绿幕。
2.  **精细处理**：由于人物动作复杂，可能仍有残留或穿帮。此时添加一个**矩形蒙版**，框选住电视屏幕区域，使人物仅从屏幕内显示。调整蒙版羽化使边缘自然。
3.  **颜色校正**：人物边缘可能仍有绿色反光。在“调节”功能的“HSL”选项中，选择绿色，降低其饱和度和亮度，即可消除绿边。
4.  **合成定位**：将处理好的人物图层置于电视背景素材之上，调整位置和大小，让人物的手部等关键部位与电视机边缘自然衔接。

### 案例二：无实物托举锤子

这个效果需要将用线吊起的锤子，合成到人物手中，并制作跟随手部运动的动画。

1.  **抠出锤子**：将吊线锤子素材拖入时间轴。使用**自定义抠像**的智能画笔仔细描绘锤子主体，用橡皮擦修掉多余的线或背景，完成抠像。
2.  **初步调色**：用色度抠像辅助去除可能存在的杂色，并通过调节工具让锤子的光影与环境匹配。
3.  **制作关键帧动画**：将抠好的锤子图层放在人物图层上方。在时间轴上，根据人物手部抬起、移动的动作，为锤子图层的“位置”属性打上关键帧，让锤子跟随手部运动。关键帧的添加逻辑是：`在时间点A设置位置关键帧 -> 在时间点B移动锤子到新位置（自动生成关键帧）`。

### 案例三：车内飞行特效

此效果将绿幕车内拍摄的人物，与动态的天空背景合成。

1.  **抠出人物与车内景**：使用**自定义抠像**，仔细描绘人物、座椅、方向盘等所有需要保留的前景元素。
2.  **去除绿幕杂色**：同样使用色度抠像和HSL颜色调节（降低绿色饱和度/亮度），彻底清理绿色背景和边缘反光。
3.  **背景合成**：将抠出的人物图层置于动态天空背景素材之上。根据背景的运动幅度（如晃动、上升），可以适当为人物图层添加微小的位置或旋转关键帧，模拟随车一起运动的颠簸感，增强真实感。

## 总结

本节课我们一起学习了剪映中**抠像**与**蒙版**两大核心合成工具。
*   **抠像**负责将主体从原背景中分离，我们掌握了针对纯色背景的**色度抠像**、针对复杂背景的**自定义抠像**以及便捷的**智能抠像**。
*   **蒙版**负责控制图层的显示区域，我们熟悉了线性、圆形、矩形等多种蒙版形状，并理解了**羽化**参数对于边缘融合的重要性。

通过“电视爬出”、“无实物托举”和“车内飞行”三个综合案例的实战，我们演练了如何将这两个工具结合使用，并辅以**关键帧动画**和**颜色调节**，最终创造出富有想象力的视频效果。请务必下载练习素材进行实际操作，熟练掌握这些技巧，你将能开启视频创作的无限可能。
