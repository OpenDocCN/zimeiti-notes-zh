## 推荐规则详解
推荐规则是指小红书系统如何判定账号质量，并据此决定给予多少流量曝光和推荐的机制。系统会分析你的账号信息、发布内容，为你的账号打上标签，然后根据这些标签将你的内容推荐给可能感兴趣的用户。

推荐规则主要围绕三个维度展开：**用户行为**、**自序商品**和**图文阴影**。

### 1. 用户行为规范
用户行为规范定义了在账号日常运营中，哪些行为是禁止的，哪些是允许的。遵守这些规范是账号存活的前提。

以下是用户行为中**严格禁止**的内容：
*   **微信号**
*   **二维码**
*   **代购水印**
*   **某某店铺**（直接宣传其他店铺名称）

![](img/ffad8718e2d53904283af90032c0c278_19.png)

**核心逻辑**：平台严禁任何形式的直接营销和导流行为。例如，你不能在笔记中直接宣传自己的主店，引导用户去其他平台购买。

**允许的变通方法**：虽然不能直接宣传，但可以通过一些技巧间接引导。例如，在笔记评论区，用自己的账号“官方置顶”一条评论，并@你的另一个小号，通过小号进行引导。这是一种常见的规避策略，但需谨慎使用。

以下是用户行为中**允许**的内容：
*   **个人邮箱**
*   **微博账号**
*   **使用谐音或话术间接引导微信**（例如，将“微信”写作“V星”）

![](img/ffad8718e2d53904283af90032c0c278_21.png)

![](img/ffad8718e2d53904283af90032c0c278_23.png)

**重要提示**：即使是允许的间接引导，也需严格控制频率（如每日私聊不超过2-6人），否则仍有被封号的风险。关于更安全的引流和变现方法，我们将在后续的“小红书变现”章节详细讲解。

### 2. 自序商品规范
“自序商品”主要指涉及专业建议和账号操作安全方面的规则。小红书在此方面的管控相对宽松，但仍有红线不可触碰。

以下是自序商品中**不能做**的事情：
*   **提供医疗建议**（如推荐医院、药品）。
*   **提供投资建议**（如推荐股票、房产、理财产品）。
    **公式**：`发布内容 ≠ 医疗/投资建议`
*   **一台电脑同时登录超过3个账号**。
    **代码逻辑**：`if (同一IP登录账号数 > 3) { 触发风险检测 }`

![](img/ffad8718e2d53904283af90032c0c278_29.png)

**操作建议**：对于多账号运营，建议使用“雷电模拟器”等工具进行IP分流，且单台设备最好不超过3个账号。登录第4个账号前，请先退出一个。

### 3. 图文阴影规范
“图文阴影”特指在发布笔记（图文/视频）时需要遵守的规范，核心是内容质量和互动引导方式。

![](img/ffad8718e2d53904283af90032c0c278_31.png)

以下是图文阴影中**不能做**的事情：
*   **诱导用户互动**：例如，“关注领奖”、“评论抽奖”、“私信送资料”等直接诱导关注、点赞、评论的行为。
*   **内容抄袭**：直接搬运他人的图片和文案。这会严重损害账号权重，甚至导致废号。
    **代码逻辑**：`if (内容原创度 ≈ 0%) { 账号权重 = 极低 }`

以下是图文阴影中**鼓励做**的事情：
*   **提高内容原创度**：原创度越高，获得的基础流量扶持就越大。
    **代码逻辑**：`if (内容原创度 > 70%) { 基础流量扶持 = 高 }`
*   **二次创作**：如果使用网络素材，需进行深度加工和整合，形成新的观点或内容，保证至少50%以上的原创度。

**新手优势**：即使是从二次创作开始（难度约3/10），其起步也远比完全原创（可能感觉难度10/10）要容易得多，因此小红书对新人依然友好。

## 总结
本节课我们一起学习了小红书运营的第一个核心规则——**推荐规则**。我们了解到，系统通过分析**用户行为**、**自序商品**信息和**图文阴影**质量来判定账号，并决定流量分配。关键点在于：避免直接导流、不提供专业领域建议、保证设备登录安全，以及全力提升内容原创度。理解并应用这些规则，是为账号打下坚实流量基础的第一步。
