# 微信视频号运营教程：P21：19.实操步骤 - 剪辑教程分享a

在本节课中，我们将学习如何开通微信视频号的商品橱窗功能，这是通过视频号带货变现的关键一步。整个过程简单直接，我们将分步拆解，确保每位初学者都能轻松跟上。

上一节我们介绍了视频号的基础概念，本节中我们来看看如何具体开通带货权限。

## 开通视频号商品橱窗

![](img/bef33523a3d6eb2f3d2103e6c43f7a20_1.png)

![](img/bef33523a3d6eb2f3d2103e6c43f7a20_2.png)

以下是开通商品橱窗功能的具体操作步骤。

1.  **第一步：打开微信**
    在您的手机上启动微信应用程序。

![](img/bef33523a3d6eb2f3d2103e6c43f7a20_4.png)

![](img/bef33523a3d6eb2f3d2103e6c43f7a20_6.png)

2.  **第二步：点击“发现”**
    进入微信主界面后，点击底部导航栏的“发现”选项。

3.  **第三步：进入“视频号”**
    在“发现”页面中，将视线向上滑动，找到并点击“视频号”入口。该入口旁有一个黄色的蝴蝶图标作为标识。

4.  **第四步：点击个人头像**
    进入视频号页面后，点击页面右上角的个人头像图标。

![](img/bef33523a3d6eb2f3d2103e6c43f7a20_8.png)

![](img/bef33523a3d6eb2f3d2103e6c43f7a20_9.png)

5.  **第五步：进入“创作者中心”**
    点击个人头像后，进入个人主页。在此页面中向下滑动，直到页面最底部，找到并点击“创作者中心”。（注：若未发布过作品，可能无法看到此入口，请先发布一条视频。）

6.  **第六步：点击“带货中心”**
    进入“创作者中心”后，在页面中找到“带货中心”的选项并点击。其图标类似一个小卖部。

完成以上步骤后，系统会根据您的账号状态跳转到不同页面。如果您尚未开通橱窗权限，将进入开通流程；如果已开通，则会直接进入商品上架管理页面。

## 如何开通橱窗权限

![](img/bef33523a3d6eb2f3d2103e6c43f7a20_11.png)

如果您点击“带货中心”后进入的是开通权限界面，请按以下步骤操作。

1.  **选择带货身份**
    界面会要求您选择带货身份，通常有三个选项：`我是商家`、`我是达人`、`我是商家且已有视频号`。**请选择“我是达人”**。

2.  **交纳保证金**
    选择“我是达人”后，系统会跳转到交纳保证金的页面。您需要点击页面上的“去交纳”按钮，按照提示通过微信支付交纳**100元**的保证金。支付成功后，即开通了视频号带货权限。

核心操作可以总结为以下流程：
```
打开微信 -> 发现 -> 视频号 -> 个人头像 -> 创作者中心 -> 带货中心 -> 选择“我是达人” -> 交纳100元保证金
```

![](img/bef33523a3d6eb2f3d2103e6c43f7a20_13.png)

## 如何上架商品

对于已经开通橱窗权限的账号，点击“带货中心”后将直接进入商品选品和上架的管理后台。您可以在此浏览、选择联盟商品，并将其添加到自己的橱窗中。

![](img/bef33523a3d6eb2f3d2103e6c43f7a20_15.png)

本节课中我们一起学习了开通微信视频号商品橱窗的完整流程。关键点在于通过“创作者中心”找到“带货中心”，选择达人身份，并交纳100元保证金即可开通。开通后，您就可以在带货中心选择商品进行推广，迈出视频号变现的第一步。
