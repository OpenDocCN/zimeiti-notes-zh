# AI中视频创作教程：P1：利用AI工具全自动创作中视频 🚀

在本节课中，我们将学习如何利用AI工具实现中视频的全自动创作。这种方法可以高效地产出符合平台要求的优质内容。

![](img/7d92609a3c1bd0669dbd98e6ad0b2e84_1.png)

接下来分享的内容，可能颠覆你对中视频的认知。我们已经验证了一个方法，即中视频可以用AI工具全自动创作。验证成功后，第一时间来分享。

这相当于中视频可以帮助你实现全自动收益。先别质疑，看完这个视频，你一定会被震撼。有的朋友可能会疑问，凭借AI全自动创作的中视频，能通过中视频计划吗？能有很好的播放量吗？能通过平台审核吗？答案是肯定的。

在分享具体操作方法之前，先问一个问题：平台喜欢什么样的中视频作品？毫无疑问，首先视频的文案要足够优秀。其次，画面要与文案高度吻合、高度相关，并且要有一定的精彩程度。这样的视频才能获得平台青睐。

知道这些之后，我们要用到三个工具。

以下是实现全自动创作所需的三个核心工具：

1.  **文案创作神器：ChatGPT**
    最近火遍全球的ChatGPT，是一款人工智能创作工具。简单来说，它虽然不是专门用于文案创作，但其文案创作能力目前是最强的。ChatGPT的推出甚至引起了很多人的恐慌，因为很多专家学者深度考察后发现，它不仅仅是一个人工智能工具，甚至似乎拥有了人的思维和情感。它回答问题和写出的内容，明显不像机器人写的，里边有丰富的情感和可感知的情绪。这非常惊人，但我们做中视频计划，恰恰需要这样一款神奇的工具。

2.  **AI配音工具：例如“配音神器”**
    有了高度原创且优质的文案后，需要将文案进行配音。这一步比较简单。我们使用AI配音工具，比如“配音神器”。这样就能根据文案生成配音。

3.  **视频素材匹配工具：“三三搜帧”**
    最后一步是根据配音和文案，匹配高度相关的动态视频画面。画面质量不能太低，不能使用剪映“一键成片”功能匹配出的静态图片轮播形式。这里推荐“三三搜帧”工具。它会根据每一句文案，匹配高度吻合的动态视频画面，非常给力。

结合这三个工具，我们就能实现中视频的AI全自动创作。接下来演示具体操作方法。建议点赞收藏，干货非常多。

上一节我们介绍了三个核心工具，本节中我们来看看如何具体操作。

首先是使用ChatGPT生成文案。有的朋友可能不会用ChatGPT，也没有账号。这没关系，视频结尾会一并分享。先了解怎么用即可。

例如，我们以历史故事为例。在ChatGPT上提问：“我需要一份1000字左右的中国历史故事，你帮我写一下。” 右边截图是它给出的答案，约1000字，写的是曹操和华佗的故事。这只是演示，大家可以发挥想象，让它写任何故事。比如，可以要求写一个民间故事，还可以加上定语，如“写一个湖南地区的民间故事”或“写一个偏门冷门的民间故事”，它都能写出来。因为是演示，所以提问比较笼统。你提问的问题越精细，它给出的答案也越精细。此外，如果对答案不满意，还可以追加条件。

例如，在刚才曹操和华佗的故事里，想再加入一个人物。可以追加说：“你说的故事挺好，但我还想要更多人物加入，故事字数不要变。” 它就把荀彧这个人物加进去了，故事看起来就更独特了，可能真有很多人不知道他们三人发生的故事。

有了高度原创的文案后，接下来要给文案进行配音。

我们打开配音工具，点击“制作配音”，把刚才得到的高度原创文案粘贴进去。然后点击“更多主播”，在里面选择喜欢的主播风格。

![](img/7d92609a3c1bd0669dbd98e6ad0b2e84_3.png)

选择后让他进行配音，最后选择“保存作品”即可。

得到配音文件后，最后一步是找到高度匹配的视频画面。

我们用到的工具是“三三搜帧”。这个工具能力特别强大。只要输入你想找的视频画面的大概意思，点击搜索，它就会自动找出很多相关画面，而且都没有版权，可以直接使用。基于这个功能，它还有一个特别强大的功能叫“AI合成视频”。只需要导入配音文件，它就会自动帮你匹配画面。

![](img/7d92609a3c1bd0669dbd98e6ad0b2e84_5.png)

![](img/7d92609a3c1bd0669dbd98e6ad0b2e84_3.png)

回到“三三搜帧”首页，点击“合成视频”。点击后，它会根据你的台词，匹配多个相符的画面。我们选择好匹配的画面后，拉到最底下点击“生成视频”，就能生成高质量的原创视频。
