# 亚马逊运营教程：P37：创建新商品：分类

![](img/37e513ecd2c77a8866b296a07490074d_0.png)

在本节课中，我们将学习在亚马逊卖家后台创建新商品时，如何正确地为商品选择分类。这是商品上架流程中至关重要的一步。

![](img/37e513ecd2c77a8866b296a07490074d_2.png)

## 概述

![](img/37e513ecd2c77a8866b296a07490074d_4.png)

![](img/37e513ecd2c77a8866b296a07490074d_5.png)

正确选择商品分类是发布新商品的第一步。分类决定了你的商品会被展示在哪个类目下，如果选择错误，不仅会影响商品曝光，还可能违反亚马逊平台规则。本节将详细介绍如何通过亚马逊前台目录找到并选择正确的商品分类。

## 进入创建商品页面

首先，我们需要进入创建新商品的页面。以下是具体步骤。

1.  登录亚马逊卖家后台。
2.  将鼠标悬停在顶部导航栏的“目录”选项上。
3.  在下拉菜单中，点击“添加商品”。
4.  在打开的页面中，点击“创建新商品信息”按钮。

完成以上步骤后，你将进入商品创建页面。此时，系统会要求你为商品选择一个分类，这类似于在淘宝开店时选择商品类目。

## 理解分类选择的重要性

选择分类时，必须确保所选类目与你的商品完全匹配。例如，销售羽绒服，就必须将其放在“服装”下的“夹克与羽绒服”类目中。如果将其错误地放置在“卫衣”或“衬衫”类目下，就构成了“错放类目”，这是违反亚马逊平台政策的违规行为。

因此，发布商品前，务必确认你的产品所属的正确分类。

## 如何查找并选择分类

![](img/37e513ecd2c77a8866b296a07490074d_7.png)

在创建新商品页面，你会看到一个搜索框和“所有商品分类”的列表。列表中的大类别是一级类目，点击后展开的则是二级类目。

以发布一款“男士羽绒服”为例，操作流程如下：

1.  在一级类目列表中，找到并点击“服装、鞋靴和珠宝首饰”。
2.  在展开的二级类目中，选择“男士”。
3.  接下来，在出现的选项中点击“服装”。
4.  最后，在服装细分中点击“夹克与羽绒服”下的“羽绒服”。

![](img/37e513ecd2c77a8866b296a07490074d_9.png)

至此，你就为商品选定了正确的分类。

![](img/37e513ecd2c77a8866b296a07490074d_11.png)

## 解决语言障碍：使用浏览器翻译工具

对于初学者，面对全英文的类目名称可能会感到困惑。一个简单的解决方法是利用浏览器的翻译功能。

以下是使用360安全浏览器内置翻译功能的步骤：

1.  确保你使用的是360安全浏览器。
2.  点击浏览器右上角的“扩展”按钮（通常是一个拼图块图标）。
3.  在扩展中心找到并启用自带的“翻译”插件。
4.  启用后，当你访问亚马逊英文页面时，点击浏览器地址栏附近的翻译图标（如一个小地球图标），即可将整个页面翻译成中文。

![](img/37e513ecd2c77a8866b296a07490074d_13.png)

![](img/37e513ecd2c77a8866b296a07490074d_14.png)

启用翻译功能后，之前复杂的英文类目名称都会变成清晰的中文，方便你快速准确地找到目标分类。其他浏览器（如搜狗浏览器）的翻译功能位置和操作方式也类似。

选择好分类后，点击确认，即可进入下一步的商品信息编辑页面，开始填写商品标题、图片、描述等详细信息。

![](img/37e513ecd2c77a8866b296a07490074d_16.png)
