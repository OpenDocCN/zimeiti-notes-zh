## 从销售思维到用户思维

![](img/0075beeecac04e080f6cd7161712f70b_18.png)

![](img/0075beeecac04e080f6cd7161712f70b_20.png)

![](img/0075beeecac04e080f6cd7161712f70b_22.png)

上一节我们介绍了商业定位的重要性。本节中我们来看看，在确定了自身资源后，如何有效地吸引用户。

![](img/0075beeecac04e080f6cd7161712f70b_24.png)

![](img/0075beeecac04e080f6cd7161712f70b_26.png)

![](img/0075beeecac04e080f6cd7161712f70b_28.png)

![](img/0075beeecac04e080f6cd7161712f70b_30.png)

单纯以销售思维去制作内容，往往会直接强调产品优势、价格优惠或专业权威。例如：

![](img/0075beeecac04e080f6cd7161712f70b_32.png)

![](img/0075beeecac04e080f6cd7161712f70b_34.png)

![](img/0075beeecac04e080f6cd7161712f70b_36.png)

*   **医美机构**：强调“我能让你变美”、“医生专业权威”。
*   **卖书**：强调“这本书特别好”、“原价59元，七夕特惠49元”。

![](img/0075beeecac04e080f6cd7161712f70b_38.png)

![](img/0075beeecac04e080f6cd7161712f70b_40.png)

![](img/0075beeecac04e080f6cd7161712f70b_42.png)

这种直接推销的方式在**淘宝**这类**电商平台**是可行的，因为用户打开淘宝时，本身就带有明确的**购物需求**和**消费预期**。

![](img/0075beeecac04e080f6cd7161712f70b_44.png)

![](img/0075beeecac04e080f6cd7161712f70b_46.png)

![](img/0075beeecac04e080f6cd7161712f70b_48.png)

![](img/0075beeecac04e080f6cd7161712f70b_50.png)

然而，**抖音**本质上是一个**娱乐平台**。用户打开抖音的主要目的是**寻找乐趣和放松**。如果此时直接进行硬性推销，很容易引起用户的反感。即使价格再优惠，用户也可能觉得“关我什么事？我是来开心的”。

![](img/0075beeecac04e080f6cd7161712f70b_52.png)

![](img/0075beeecac04e080f6cd7161712f70b_54.png)

![](img/0075beeecac04e080f6cd7161712f70b_56.png)

因此，我们必须转换思维，从“我想卖什么”转变为“用户想看什么”。

![](img/0075beeecac04e080f6cd7161712f70b_58.png)

![](img/0075beeecac04e080f6cd7161712f70b_60.png)

![](img/0075beeecac04e080f6cd7161712f70b_62.png)

## 理解平台差异：电商 vs. 娱乐

![](img/0075beeecac04e080f6cd7161712f70b_64.png)

![](img/0075beeecac04e080f6cd7161712f70b_66.png)

![](img/0075beeecac04e080f6cd7161712f70b_68.png)

以下是电商平台与娱乐平台的核心区别：

![](img/0075beeecac04e080f6cd7161712f70b_70.png)

![](img/0075beeecac04e080f6cd7161712f70b_72.png)

![](img/0075beeecac04e080f6cd7161712f70b_74.png)

![](img/0075beeecac04e080f6cd7161712f70b_76.png)

*   **电商平台（如淘宝）**：
    *   **用户意图**：**购物需求**、**消费准备**。
    *   **内容策略**：直接展示产品卖点、价格优势、促销信息是有效的。
    *   **公式**：`用户意图 = 购买` -> `内容策略 = 直接推销`

![](img/0075beeecac04e080f6cd7161712f70b_78.png)

![](img/0075beeecac04e080f6cd7161712f70b_80.png)

![](img/0075beeecac04e080f6cd7161712f70b_82.png)

*   **娱乐平台（如抖音）**：
    *   **用户意图**：**娱乐放松**、**获取资讯**、**消磨时间**。
    *   **内容策略**：需要将商业信息巧妙融入有趣、有用、有共鸣的内容中。
    *   **公式**：`用户意图 = 娱乐` -> `内容策略 = 提供价值（趣味/知识/情感）+ 软性植入`

![](img/0075beeecac04e080f6cd7161712f70b_84.png)

![](img/0075beeecac04e080f6cd7161712f70b_86.png)

![](img/0075beeecac04e080f6cd7161712f70b_88.png)

## 用户思维的核心：提供“利益获得感”

![](img/0075beeecac04e080f6cd7161712f70b_90.png)

![](img/0075beeecac04e080f6cd7161712f70b_92.png)

![](img/0075beeecac04e080f6cd7161712f70b_94.png)

互联网思维有很多种，但**用户思维**是首要且核心的。它要求我们始终思考：**用户喜欢什么？用户需要什么？**

![](img/0075beeecac04e080f6cd7161712f70b_96.png)

![](img/0075beeecac04e080f6cd7161712f70b_98.png)

![](img/0075beeecac04e080f6cd7161712f70b_100.png)

![](img/0075beeecac04e080f6cd7161712f70b_102.png)

这需要具体到视频创作的每一秒。即使是7-8秒的短视频，也要预判用户看完每一秒后的反应。

![](img/0075beeecac04e080f6cd7161712f70b_104.png)

![](img/0075beeecac04e080f6cd7161712f70b_106.png)

![](img/0075beeecac04e080f6cd7161712f70b_108.png)

让我们通过一个例子来理解。假设你学会了拍摄打光技巧，想在抖音上分享知识并实现变现。你的视频计划讲解以下几种光：

![](img/0075beeecac04e080f6cd7161712f70b_110.png)

![](img/0075beeecac04e080f6cd7161712f70b_112.png)

![](img/0075beeecac04e080f6cd7161712f70b_114.png)

1.  **测光**：让视频更清晰。
2.  **底光**：制造恐怖氛围。
3.  **蝴蝶光**：让人物脸部更美丽、轮廓更立体。

![](img/0075beeecac04e080f6cd7161712f70b_116.png)

![](img/0075beeecac04e080f6cd7161712f70b_118.png)

![](img/0075beeecac04e080f6cd7161712f70b_120.png)

![](img/0075beeecac04e080f6cd7161712f70b_122.png)

**如果缺乏用户思维，你可能会：**

![](img/0075beeecac04e080f6cd7161712f70b_124.png)

![](img/0075beeecac04e080f6cd7161712f70b_126.png)

![](img/0075beeecac04e080f6cd7161712f70b_128.png)

*   **内容顺序**：想到哪个讲哪个，随机排列。
*   **视频标题**：“卷心菜教你三种打光方式，非常有用，赶紧收藏！”

![](img/0075beeecac04e080f6cd7161712f70b_130.png)

![](img/0075beeecac04e080f6cd7161712f70b_132.png)

![](img/0075beeecac04e080f6cd7161712f70b_134.png)

**运用用户思维，你需要思考：**

![](img/0075beeecac04e080f6cd7161712f70b_136.png)

![](img/0075beeecac04e080f6cd7161712f70b_138.png)

![](img/0075beeecac04e080f6cd7161712f70b_140.png)

![](img/0075beeecac04e080f6cd7161712f70b_142.png)

1.  **内容排序**：哪种光的效果对用户最有吸引力？**“蝴蝶光让人变美”** 直接提供了 **“利益获得感”**（变漂亮），而“底光制造恐怖”只是一种效果演示。因此，应该把最具吸引力的“蝴蝶光”放在视频最前面，第一时间抓住用户兴趣。
2.  **标题优化**：“教你三种打光方式”是一个平淡的陈述。可以改为：**“卷心菜教你一招，把姑娘拍得更美的打光技巧！”** 这个标题直接点明了用户能获得的利益（把姑娘/自己拍美），点击欲会强得多。

![](img/0075beeecac04e080f6cd7161712f70b_144.png)

![](img/0075beeecac04e080f6cd7161712f70b_146.png)

![](img/0075beeecac04e080f6cd7161712f70b_148.png)

**核心操作公式：**
`内容价值排序 = 按“用户利益获得感”强弱降序排列`
`标题策略 = 明确点出用户能获得的核心利益（而非罗列功能）`

![](img/0075beeecac04e080f6cd7161712f70b_150.png)

![](img/0075beeecac04e080f6cd7161712f70b_152.png)

![](img/0075beeecac04e080f6cd7161712f70b_154.png)

## 实践步骤总结

![](img/0075beeecac04e080f6cd7161712f70b_156.png)

![](img/0075beeecac04e080f6cd7161712f70b_158.png)

![](img/0075beeecac04e080f6cd7161712f70b_160.png)

以下是运用用户思维创作内容的简单步骤：

![](img/0075beeecac04e080f6cd7161712f70b_162.png)

![](img/0075beeecac04e080f6cd7161712f70b_164.png)

![](img/0075beeecac04e080f6cd7161712f70b_166.png)

![](img/0075beeecac04e080f6cd7161712f70b_168.png)

1.  **明确自身资源**：你有什么产品、技能或知识？（已完成商业定位）
2.  **分析用户场景**：你的目标用户在抖音上是来干什么的？（娱乐、学习、种草）
3.  **挖掘用户需求**：结合你的资源，用户**真正需要**和**感兴趣**的点是什么？是变美、省钱、解压还是学到技巧？
4.  **设计内容与话术**：将你的销售目标（如卖课、引流）包装成能满足用户需求的内容。把最大的“利益点”放在最显眼的位置（如视频开头、标题）。
5.  **预判用户反应**：审视每一帧画面、每一句文案，思考用户是否会觉得“这关我什么事？”，并将其替换为“这对我有用/有趣”。

本节课中我们一起学习了抖音运营中的**用户思维**。关键点在于认识到抖音是娱乐平台，必须摒弃直接的销售话术，转而思考如何为用户提供**价值**和**利益获得感**。通过优化内容顺序和标题，将你的商业目标无缝嵌入到用户喜欢看的内容中，才能实现有效的引流与转化。记住，始终站在用户的视角去创作，是做好抖音内容的第一步。
