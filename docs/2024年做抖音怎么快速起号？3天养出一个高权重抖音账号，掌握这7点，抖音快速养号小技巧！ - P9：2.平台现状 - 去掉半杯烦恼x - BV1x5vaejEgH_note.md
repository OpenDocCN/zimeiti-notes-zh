# 抖音运营入门：P9：2.平台现状 📊

在本节课中，我们将要学习当前自媒体行业的平台现状，分析其政策背景、市场支持以及用户规模，帮助你判断进入这个领域的可行性。

上一节我们介绍了自媒体的基本概念，本节中我们来看看这个行业所处的平台环境到底如何。

## 政策支持 📜

现阶段，整个社会氛围比较浮躁，许多人都想快速赚钱。但需要注意的是，过去有人采取不合法的手段牟利，例如前往缅甸等地的非法活动，这些行为已被国家明令禁止，对我们没有任何好处。

![](img/78b6655ae4d9d96abeaa4704496e8ae6_1.png)

与之相反，自媒体行业目前属于国家支持的范畴。国家已将支持文件下达到各个地方政府。

![](img/78b6655ae4d9d96abeaa4704496e8ae6_3.png)

以下是相关政策文件的例证：

![](img/78b6655ae4d9d96abeaa4704496e8ae6_5.png)

*   **荆州市六项措施**：该文件由人社局于今年6月2日发布，明确指出要“加大政策宣传力度，有力支持自媒体以及新媒体行业的蓬勃发展”。
*   **广西壮族自治区文件**：同样发布了支持新媒体行业发展的政策性文件。

这些文件都强调了“坚持正确导向，内容为王”的原则。这意味着，未来作为自媒体创作者，我们创作的**文章和视频内容**，将是获得收益的核心与关键。

因此，从政策层面看，自媒体行业是国家鼓励并可以从事的领域。

![](img/78b6655ae4d9d96abeaa4704496e8ae6_7.png)

## 平台激励 💰

既然有了政策文件支持我们创作，那么自媒体平台也需要响应号召，提供资金激励。

以下是西瓜视频（字节跳动旗下平台）的一个案例：

![](img/78b6655ae4d9d96abeaa4704496e8ae6_9.png)

> 西瓜视频决定将额外拿出**20亿元**，补贴中视频创作人。这**20亿元**是真金白银的纯补贴，不包含任何商单、直播和电商收入。平台还会探索保底加分成的模式，并且这**20亿元**只是保底，上不封顶。

从这个案例可以了解到，平台愿意投入巨额资金，纯粹用于补贴视频创作者。这意味着，未来通过制作视频，我们有机会从这类补贴计划中获得收益。

## 市场与用户规模 👥

现阶段，自媒体行业还有一个显著优势：庞大的用户基础。

根据2022年《中国研究生网站》发布的自媒体行业市场调查与发展前景分析报告（发布于今年9月9日），我国目前的网民规模已达到**9.4亿人**，接近10亿。这些网民就是潜在的视频观众。

那么，与之对应的视频创作者有多少呢？

以下是相关数据：

*   **全职自媒体从业者**：370万人
*   **兼职自媒体从业者**：600万人
*   **总计**：约970万人，可近似看作**1000万人**

这形成了一个**100:1**的供需关系，即平均每100个看视频的人，对应只有1个制作视频的人。显然，创作者的产出速度很难完全满足观众的消费需求。

这种巨大的供需缺口，正是国家支持自媒体行业发展的重要原因。从我们观看媒介的演变——从黑白电视到彩色电视，再到如今能连接WiFi的智能电视——可以看出，互联网视频内容是未来的发展趋势，其覆盖的用户规模将无比庞大。

![](img/78b6655ae4d9d96abeaa4704496e8ae6_11.png)

本节课中我们一起学习了自媒体行业的平台现状。我们了解到该行业拥有坚实的**政策支持**，平台方也提供了真金白银的**资金激励**（例如**20亿元**补贴计划），并且面对着一个由**9.4亿**观众和仅约**1000万**创作者构成的、存在巨大缺口的**市场**。这些因素共同表明，自媒体是一个值得关注和尝试的领域。
