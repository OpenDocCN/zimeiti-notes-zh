# 抖音小店运营实战：P13：商品改销量 📈

## 概述
在本节课中，我们将学习如何通过“改销量”的方法，快速为抖音小店积累30个有效评价订单，从而快速提升店铺的“体验分”。这是一种低成本、高效率的运营技巧。

---

## 核心目标与概念
抖音小店想要快速获得“体验分”，必须积累**30个有效订单**。有效订单是指用户收货后给出评价的订单。

上一节我们介绍了店铺基础设置，本节中我们来看看如何通过营销工具快速完成这个目标。

## 方法一：通过“限时限量购”改销量

![](img/5a27f57aa5fb09fd9ebcbdc2fb4c88fb_4.png)

首先，我们进入抖店后台，点击“营销中心”。

![](img/5a27f57aa5fb09fd9ebcbdc2fb4c88fb_5.png)

在营销中心内，找到“限时限量购”功能，通过创建一个限时限量购活动，并设置一个极低价格的SKU来完成下单。

以下是具体操作步骤：

1.  **新建活动**：点击“新建”按钮，创建一个新的限时限量购活动。
2.  **设置活动信息**：
    *   **活动名称**：可随意填写。
    *   **活动时间**：根据需求设置，例如设置为1天。
    *   **订单取消时间**：建议设置为“3分钟”。
    *   **活动结束后**：选择“恢复为原价，继续售卖”。
    *   **优惠方式**：选择“一口价”。
3.  **添加并设置商品**：
    *   点击“添加商品”，选择你需要改销量的商品（例如一件外套）。
    *   在SKU设置区域，先为所有正常SKU设置一个你能接受的优惠价（例如原价129元，设置为110元），并设置一个较大的库存量（例如200件）。
    *   **关键步骤**：找到列表最下方的一个SKU，将其价格修改为 **`0.01`** 元，库存量设置为你需要完成的订单数（例如30件）。
    *   使用“批量填充”功能，快速应用价格和库存设置。
4.  **提交前的准备**：
    *   在点击“提交”按钮**之前**，必须提前联系好帮你下单的人。
    *   沟通好活动上架时间，让他们提前关注店铺并进入小店准备。
    *   这样可以避免活动被无关用户抢拍，造成损失。
5.  **成本计算**：
    *   以30单为例，每单 `0.01` 元，总成本仅为 **`0.3`** 元。
    *   活动结束后商品恢复原价，不影响商品原有权重。

## 方法二：通过“拼团”改销量

除了限时限量购，我们还可以使用“拼团”功能达到相同目的。操作逻辑基本一致。

以下是具体操作步骤：

1.  **创建拼团活动**：在营销中心点击“拼团”，创建新活动。
2.  **设置活动信息**：
    *   **活动名称**：可随意填写。
    *   **活动时间**：拼团活动有效期通常为1天，请设置准确的时间范围。
    *   **成团人数**：设置为最低要求（例如5人）。
    *   **开团自动成团**：勾选此项。
    *   **订单取消时间**：同样建议设置为“3分钟”。
3.  **添加并设置商品**：
    *   添加目标商品。
    *   设置方法与限时限量购完全相同：正常SKU设优惠价，最后一个SKU价格改为 **`0.01`** 元，库存设为目标单数（例如50件）。
4.  **重要提醒**：
    *   同样需要**提前联系好下单人员**，避免被“薅羊毛”。
    *   建议选择**客单价较低**的商品（如十几元、二十几元）进行操作，即使发生意外，损失也较小。

---

## 核心注意事项 ⚠️

在操作过程中，有一个必须严格遵守的规则：

**绝对禁止**直接在商品编辑页修改SKU价格为超低价。这种做法会严重影响商品权重和店铺正常运营。

正确的操作路径只能是：**营销中心 -> 限时限量购 / 拼团**。

## 总结
本节课中我们一起学习了快速提升抖音小店“体验分”的两种实操方法：
1.  利用 **“限时限量购”** 活动，设置超低价SKU进行下单。
2.  利用 **“拼团”** 活动，实现同样的改销量目的。
