在本节课中，我们将学习如何在内容选题上实现“先泛后垂”的策略。上节课我们明确了新手起号需要先泛后垂，本节将具体讲解操作方法。这主要涉及两个层面的逻辑：一是视频内容的整体铺排，二是单条视频的内部结构。本节课我们先聚焦于内容选题层面。

## 理解“两个钩子” 🎣

在抖音内容创作中，每个视频和内容选题都需要设置“两个钩子”。

第一个是**泛逻辑钩子**，用于吸引广泛的用户群体，让大众愿意看完视频。第二个是**垂逻辑钩子**，用于吸引精准用户，让他们深度理解内容并可能转化为粉丝。

泛钩子的核心目的是**获取流量**，支撑账号在起号阶段产出爆款，突破流量池，为账号打上标签并积累首批精准用户。垂钩子的核心目的是**深度吸引用户**，建立账号的专业性和唯一性，促进用户关注。

![](img/60f2aec46f234d06c776303c69f93a67_4.png)

简单来说，在账号前期（例如发布10条作品），可以用7-8条泛内容来支撑流量，让账号快速启动；用2-3条垂内容来展示专业深度，吸引用户成为粉丝。

![](img/60f2aec46f234d06c776303c69f93a67_6.png)

## 案例分析：泛与垂的选题实践 📊

我们通过一个医美/审美赛道的账号“一本正经”来具体分析。

![](img/60f2aec46f234d06c776303c69f93a67_8.png)

### 泛内容钩子案例

**视频主题**：分析为什么某些在韩务工的中国女爱豆在韩国更受欢迎，探讨中韩审美差异。
**核心策略**：开篇以“为什么……”的疑问句式，切入大众感兴趣的娱乐八卦话题，吸引包括吃瓜群众在内的广泛用户。在讨论审美差异的过程中，自然引出账号核心领域“审美”。
**数据表现**：播放量达200万。
**效果**：成功利用泛话题获取巨大流量，让更多用户接触到账号。

### 垂内容钩子案例

**视频主题**：详细解析如何模仿女星于文文，打造“拽酷”风格。
**核心策略**：直接切入“如何变美”的垂直细分领域（酷飒风格），目标受众非常明确——想变酷的女生。
**数据表现**：播放量20万。
**效果**：虽然流量相对较小，但能深度吸引对口的目标用户，展示账号在特定领域的专业度，促进精准粉丝的关注和转化。

![](img/60f2aec46f234d06c776303c69f93a67_10.png)

这个拥有83万粉丝的成熟账号，其内容选题依然遵循“有泛有垂”的原则，泛内容负责拉新和扩大流量，垂内容负责沉淀和转化核心粉丝。

## 如何设计泛与垂的选题？ ✍️

以下是针对不同赛道设计泛、垂选题的思路举例：

![](img/60f2aec46f234d06c776303c69f93a67_12.png)

| 赛道 | 泛选题思路（吸引流量） | 垂选题思路（吸引粉丝） |
| :--- | :--- | :--- |
| **幼儿教育** | 为什么小孩都那么皮？（共性痛点） | 如何培养3-6岁小孩的自律性？（专业细分） |
| **汽车维修** | 为什么宝马/理想事故率讨论度高？（热点车型） | 小鹏P7日常保养的3个关键步骤。（具体车型保养） |
| **美妆** | 为什么这些网红能快速走红？（热点现象） | 新手如何画出干净服帖的底妆？（具体技巧） |

通过以上例子可以看出，泛选题通常覆盖面更广，垂选题则更专业、更细分。

### 设计泛选题的三个要点

1.  **多说共性问题**：聚焦目标赛道内用户普遍存在的痛点或关注点。例如，“小孩调皮”比“培养小孩自律”更具普遍性。
2.  **结合热点事件**：热点自带流量。例如，讨论热门车型的事故、当红明星的妆容，能有效提升内容曝光。
3.  **多用“为什么”，少用“怎么做”**：以疑问句开场，满足用户的八卦和猎奇心理，引导他们以“吃瓜”心态看完视频，再在其中植入专业观点。公式为：**泛流量 = 热点/共性问题 + “为什么”句式**。

![](img/60f2aec46f234d06c776303c69f93a67_14.png)

## 泛与垂的权重如何分配？ ⚖️

泛内容和垂内容的比例并非固定不变，而应根据账号的不同发展阶段动态调整。

*   **起号初期（前5-10条作品）**：
    *   **目标**：突破流量池，获取数据反馈，让系统建立初步用户画像。
    *   **建议比例**：**泛80% : 垂20%**。重心放在做泛内容获取流量上，用少量垂内容展示专业性。

![](img/60f2aec46f234d06c776303c69f93a67_16.png)

*   **标签建立期（账号有初步数据后）**：
    *   **目标**：巩固账号标签，吸引并沉淀第一批精准粉丝。
    *   **建议比例**：**泛60% : 垂40%**。逐步增加垂内容的比重，强化账号人设。

![](img/60f2aec46f234d06c776303c69f93a67_18.png)

*   **稳定发展期（账号标签稳定后）**：
    *   **目标**：平衡流量获取与粉丝维护，实现稳定增长。
    *   **建议比例**：**泛50% : 垂50%**。如案例中的账号，可能调整为泛40%、垂60%，根据实际运营情况灵活变动。

## 总结 📝

本节课我们一起学习了在内容选题上实现“先泛后垂”的具体方法。关键在于理解并设置好“两个钩子”：用**泛选题**（讨论共性问题、结合热点、多用“为什么”）来获取广泛流量；用**垂选题**（深入专业细分领域）来吸引精准用户并促进转化。同时，需要根据账号发展的不同阶段（初期、成长期、稳定期），动态调整泛内容和垂内容的发布比例。掌握这一策略，能帮助你在起号阶段更清晰、更有效地进行内容规划。
