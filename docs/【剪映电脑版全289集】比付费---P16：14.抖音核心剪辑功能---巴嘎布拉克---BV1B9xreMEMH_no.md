# 剪映电脑版教程：P16：14. 抖音核心剪辑功能—视频过渡转场 🎬

在本节课中，我们将要学习剪映的核心剪辑功能之一：视频的过渡与转场。转场是连接两个视频镜头的重要技巧，能让视频切换更自然流畅，避免生硬感。通过学习并运用剪映提供的丰富转场效果，你可以显著提升视频的观赏性和专业度，轻松制作出吸引人的卡点视频或炫酷的运镜视频。

## 什么是视频过渡转场？ 🔄

视频过渡转场是指两个视频镜头之间，为了使衔接更顺畅而采用的视频处理技巧或剪辑方法。在抖音等平台，许多爆款的卡点视频都大量运用了转场技巧。巧妙利用剪映中的转场和过渡特效，可以帮助我们快速制作出节奏感强、视觉冲击力大的视频。

上一节我们介绍了转场的基本概念，本节中我们来看看如何在剪映中实际操作。

## 实战演练：制作卡点视频 🎵

接下来，我们通过一个制作卡点视频的实例，来详细讲解视频过渡和转场的具体应用方法。

### 第一步：创建项目与导入素材

首先，我们创建一个新项目，并导入几张照片作为视频素材。

### 第二步：添加背景音乐

要制作卡点视频，需要选择一首节奏感强的背景音乐。我们可以在“音频”选项卡中选择“抖音收藏”，导入一首收藏好的动感音乐。



![](img/3e22a86e22f60203999d770f300d912a_3.png)



![](img/3e22a86e22f60203999d770f300d912a_3.png)

试听音乐，确认其有明显的节奏点（如“动词打次”），这非常适合制作卡点视频。然后点击“使用”按钮，将音乐导入到时间轴。

![](img/3e22a86e22f60203999d770f300d912a_5.png)



![](img/3e22a86e22f60203999d770f300d912a_5.png)

### 第三步：为音乐踩点

音乐导入后，下一步是“踩点”，即标记出音乐的节拍点，以便后续让画面切换与节拍同步。

以下是踩点的具体步骤：
1.  点击时间轴上的音频轨道。
2.  在底部工具栏找到并点击“踩点”。
3.  打开“自动踩点”功能。
4.  系统提供“节拍一”和“节拍二”两种选项。“节拍二”的踩分点更密集，适合制作炫酷的卡点视频；“节拍一”则更适合制作柔和、带特效的卡点视频。
5.  我们选择“节拍二”，然后点击对号确认。

![](img/3e22a86e22f60203999d770f300d912a_7.png)



![](img/3e22a86e22f60203999d770f300d912a_9.png)



踩点完成后，音频轨道上会出现一系列黄色小点，这些就是音乐的节奏点。

### 第四步：调整照片时长

现在，我们需要根据踩好的节奏点来调整每张照片的时长，使每张照片的结束点对齐一个节奏点。

操作方法是：选中第一张照片，拖动其右侧边缘，与时间轴上的第一个节奏点对齐。然后依次调整后续每张照片的时长，使其分别对齐后续的节奏点。

### 第五步：添加转场效果

![](img/3e22a86e22f60203999d770f300d912a_23.png)

这是让视频“活”起来的关键。我们需要在每两张照片之间的衔接点（即时间轴上显示的小方块）添加转场效果。

![](img/3e22a86e22f60203999d770f300d912a_25.png)

以下是添加转场的步骤：
1.  将时间轴放大，以便更清晰地看到照片之间的衔接点。
2.  点击两个素材之间的小方块。
3.  右侧会弹出转场效果库。剪映提供了多种类型的转场，包括：
    *   **基础转场**：如淡入淡出。
    *   **运镜转场**：如推进、拉远、旋转等，能模拟摄像机运动。
    *   **幻灯片转场**：如百叶窗、立方体等。
    *   **特效转场**：如模糊、闪光等。
    *   **遮罩转场**：使用图形进行过渡。
4.  我们可以选择“运镜转场”类别，然后为每个衔接点选择不同的效果，例如“推进”、“拉远”等。

## 课程总结 📝

本节课中，我们一起学习了剪映中视频过渡与转场功能的核心应用。要添加转场，只需点击视频轨道上两个素材之间的衔接点（小方块），即可从丰富的效果库中选择合适的转场效果。剪映提供了基础转场、运镜转场、幻灯片、遮罩转场等多种类型，合理利用这些效果可以极大地提升视频的流畅度、节奏感和视觉丰富度，对于制作卡点类等需要强节奏感的视频尤其有帮助。
