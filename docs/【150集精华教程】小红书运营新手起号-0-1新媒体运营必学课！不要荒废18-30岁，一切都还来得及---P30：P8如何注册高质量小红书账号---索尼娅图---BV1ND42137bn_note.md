## 账号主页构成与设置

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_20.png)

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_22.png)

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_24.png)

注册账号后，我们会进入账号主页。主页是用户对你的第一印象，也是平台识别账号定位的重要依据。主页的设置主要包括头像、昵称和简介。

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_26.png)

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_28.png)

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_30.png)

以下是设置账号主页的三个核心要素：

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_32.png)

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_34.png)

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_36.png)

1.  **头像**
    *   头像需要与你的内容领域相匹配，具有代表性。
    *   **公式**：`头像 ≈ 内容领域视觉符号`
    *   例如，分享美妆内容，可以使用公司代言人、有颜值的员工肖像或自己的代表性照片。

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_38.png)

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_40.png)

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_42.png)

2.  **昵称**
    *   昵称应让人一眼就能大致了解你的账号是做什么的。
    *   **公式**：`昵称 ≈ 领域关键词 + 个人/品牌标识`
    *   例如，“穿搭日记”、“美妆小课堂”等。

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_44.png)

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_46.png)

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_48.png)

3.  **简介**
    *   简介需要清晰说明你的身份或账号价值。
    *   如果未开通小红书店铺，需要将流量引导至其他平台进行转化，可以在简介中标注联系方式。
    *   **注意**：简介中的任何引导信息都不得违反小红书的社区公约，需注意用词。

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_52.png)

## 兴趣类目选择策略

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_56.png)

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_58.png)

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_60.png)

首次使用账号时，系统会要求选择感兴趣的类目。虽然最多可以选择四个，但对于新账号，需要有策略地进行选择。

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_62.png)

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_64.png)

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_66.png)

以下是选择兴趣类目的核心原则：

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_68.png)

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_70.png)

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_72.png)

*   **重点突出**：四个类目中，应重点经营**第一类**，其他三类作为辅助。
*   **关联性强**：所选的四个类目之间应具有自然的关联性，而不是毫不相干。
*   **示例**：如果主做“男士服装搭配”（时尚类），辅助类目可以选择“搞笑”、“情感”、“运动”。因为关注时尚的男性用户，很可能也对幽默、情感话题和运动健身感兴趣。而“科技数码”与“男士时尚”的关联性则相对较弱。

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_74.png)

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_76.png)

**核心定位公式**：
```
账号核心内容 = 主选兴趣类目
```
对于个人账号，初期（冷启动阶段）的笔记内容应紧紧围绕这个核心类目展开。对于企业账号，则必须基于自身的产品和业务类型来选择主攻类目。

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_78.png)

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_80.png)

## 让系统识别你的账号定位

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_84.png)

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_86.png)

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_88.png)

小红书依靠智能算法系统来识别内容标签和用户兴趣标签。一个新账号，如何让系统快速了解你的定位呢？

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_90.png)

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_92.png)

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_94.png)

我们无法直接告诉系统，但可以通过“笨办法”来达成：**持续、垂直地发布同一领域的高质量内容**。系统会通过分析你发布的笔记内容、关键词、互动数据等，逐渐为你的账号打上相应的标签，从而将你的内容推荐给感兴趣的用户。

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_96.png)

![](img/fb53fdaa4a0cb64da3d16fc722c9e94d_98.png)

本节课中我们一起学习了注册高质量小红书账号的关键步骤。我们明确了账号主页（头像、昵称、简介）的设置要点，掌握了兴趣类目的选择策略，并理解了通过持续发布垂直内容来让系统识别账号定位的原理。一个好的开始是成功的一半，扎实完成账号初始化设置，将为后续的内容运营打下坚实基础。
