# 自媒体运营从0~1全攻略：P4：2.自媒体平台现状 📊

在本节课中，我们将要学习自媒体行业的现状，分析其是否值得进入，并了解国家政策与平台支持，帮助你判断这个行业的可行性与前景。

## 概述

上一节我们了解了自媒体的定义，本节中我们来看看这个行业本身。我们需要分析自媒体行业是否可以做，以及从事这个行业能带来哪些好处。

## 行业可行性分析

![](img/f44fe22d377804cccf4cda02394e9e9d_1.png)

很多同学在考虑兼职、转行或寻找副业时，都会产生疑问：这个行业到底能不能做？

近几年由于疫情等原因，实体经济受到冲击，许多上班族希望寻找副业。但在这个过程中，部分人抱有暴富心态，甚至采取了不合法的手段，最终导致严重后果。这类行业是国家明令禁止的，对我们没有任何好处。

现阶段，自媒体行业属于国家支持的范畴。国家已将支持文件下达到各个地方政府。

![](img/f44fe22d377804cccf4cda02394e9e9d_3.png)

![](img/f44fe22d377804cccf4cda02394e9e9d_4.png)

以下是部分地方政府发布的支持文件示例：

*   **荆州市人社局**：于去年6月发布了《六项措施助力自媒体及新媒体高质量发展》的文件。
*   **广西壮族自治区**：发布了有关新媒体的支持政策。
*   **四川省人民政府**：发布了相关文件。
*   **浙江省人民政府、杭州市人民政府、江西省赣州市人民政府**：均发布了相应的政策性文件。

这些文件指出，要坚持正确导向，并且通过创作视频或文章来获取收益，是重要的方式。由此可见，这个行业是可以放心去做的。但必须强调一点：国内自媒体平台管理严格，**黄、赌、毒、擦边**等内容绝对不能做。

例如，海外版抖音TikTok上的一些擦边视频，就不能放到国内平台。这一点必须首先明确。

## 平台支持与市场前景

国家有政策支持，平台也积极响应号召，支持视频创作者。那么，平台具体给予了哪些支持呢？

首先，通过一个视频来了解。西瓜视频曾宣布，未来一年将额外拿出**20亿元**补贴中视频创作者。创始人强调：第一，这是真金白银的20亿元，不包含任何商单、直播和电商收入，是纯补贴；第二，平台会探索保底加分成的模式，助力创作人职业化；第三，20亿元只是保底，上不封顶。

这意味着，作为自媒体创作者，并非只有带货一条变现路径。通过制作视频或文章，依靠播放量同样可以变现。这20亿元，就是未来可以赚取的收益之一。

除了平台补贴，自媒体行业本身拥有庞大的流量基础。根据相关市场调查报告显示：

*   **中国网民规模**：约9.4亿人（接近10亿）。这是观看视频的庞大用户群体，覆盖各年龄段。
*   **从业者规模**：截止去年，全职从事自媒体者约370万，兼职者约600万，合计约970万（可视为1000万）。

做一个简单的对比：**1000万人制作视频，10亿人观看视频**。这相当于每100个看视频的人中，只有1个人在生产内容。一个人制作视频的速度，很难赶上大众消费视频的速度。因此，内容供给是远远不足的。

这也解释了为什么国家要支持自媒体行业发展。很多同学担心行业是否已经饱和。根据行业趋势分析，在大数据时代背景下，**自媒体行业远远没有饱和**。“远远”这个词表明，在未来十年内，你都可以选择将自媒体作为兼职或全职来发展。

## 总结

![](img/f44fe22d377804cccf4cda02394e9e9d_8.png)

本节课中我们一起学习了自媒体行业的现状。我们了解到，自媒体是国家政策支持、平台真金白银补贴的朝阳行业。目前行业从业者与用户规模对比悬殊，内容供给远未饱和，市场前景广阔。但必须牢记，创作需遵守平台规则，杜绝一切违规内容。在认清这些现状的基础上，我们可以更有信心地迈出第一步。
