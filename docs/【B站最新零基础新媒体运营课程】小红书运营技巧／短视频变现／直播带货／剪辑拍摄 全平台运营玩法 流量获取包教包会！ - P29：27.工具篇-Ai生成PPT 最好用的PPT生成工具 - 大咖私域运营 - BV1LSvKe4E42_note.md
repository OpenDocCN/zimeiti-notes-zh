# 新媒体运营课程：P29：27.工具篇-AI生成PPT 🚀

在本节课中，我们将学习一款非常实用的AI工具，它能帮助我们根据主题或现有文档，自动生成结构清晰、内容详实的PPT演示文稿。

上一节我们介绍了多种新媒体运营工具，本节中我们来看看如何利用AI技术高效地制作PPT。

## 工具核心功能

这款AI工具主要提供两种生成模式，以满足不同的创作起点。

![](img/67fdb5ab40509fe88d2d468e61f3358f_1.png)

以下是两种核心模式的具体说明：

![](img/67fdb5ab40509fe88d2d468e61f3358f_3.png)

*   **主题生成模式**：用户只需输入一个核心主题，工具便会结合类似ChatGPT的生成式AI能力，自动创作出完整的PPT全文内容。
*   **文档生成模式**：用户可以上传已有的文档（如TXT、PDF文件），AI助手将自动分析文档内容，总结出核心大纲，并基于此大纲生成详细的PPT页面内容。

![](img/67fdb5ab40509fe88d2d468e61f3358f_5.png)

## 实战演示：基于文档生成PPT

![](img/67fdb5ab40509fe88d2d468e61f3358f_7.png)

接下来，我们通过一个实际案例来演示“文档生成模式”的工作流程。我们将上传一篇关于新媒体运营的文章，观察工具如何进行处理。

**步骤一：上传文档**
我们准备了一篇关于“新媒体运营流量与创作标签”的文章。在工具界面点击“继续”后，上传该文件。

![](img/67fdb5ab40509fe88d2d468e61f3358f_9.png)

![](img/67fdb5ab40509fe88d2d468e61f3358f_1.png)

![](img/67fdb5ab40509fe88d2d468e61f3358f_11.png)

**步骤二：AI分析并生成大纲**
工具会快速解读上传的文档，并自动提炼出内容大纲。从下图可以看到，它为我们列出了多个核心章节。

![](img/67fdb5ab40509fe88d2d468e61f3358f_3.png)

![](img/67fdb5ab40509fe88d2d468e61f3358f_13.png)

生成的大纲示例如下：
1.  新媒体运营的本质
2.  平台运营的角度
3.  账号运营的策略
4.  用户行为的分析与应用
5.  创作激励与流量分配

7.  案例分析

![](img/67fdb5ab40509fe88d2d468e61f3358f_15.png)

**步骤三：开始制作PPT**
大纲确认后，点击“开始制作”。工具会基于对文档的理解和已生成的大纲，自动创作每一页PPT的详细内容。这个过程通常很快。

![](img/67fdb5ab40509fe88d2d468e61f3358f_17.png)

## 工具优势与编辑调整

生成完成后，我们可以进入编辑界面查看和调整PPT。这款工具的核心优势在于其深度内容理解和结构化生成能力。

以下是该工具的主要优点和后续操作：

*   **深度内容生成**：与许多只能将现有内容简单排版到幻灯片上的工具不同，此工具能**分析文档逻辑，将内容归类到不同的大纲节点下，并自动扩充出详细的解说文本**。这对于只有要点或大纲，需要充实内容的场景尤为有用。
*   **模板更换**：在编辑界面右侧，可以使用“更换模板”功能。虽然当前可用模板数量不算非常多，但切换速度很快，能满足基础的美化需求。
*   **局部编辑**：生成的内容和版式均可手动调整，包括文字、形状、图片等元素。
*   **灵活的工作流**：用户可以选择在工具内直接编辑，也可以将生成的PPT下载到本地，使用PowerPoint或Keynote等软件进行深度加工。
