# 抖音运营教程：P11：垂直类目细分领域 🎯

在本节课中，我们将深入探讨抖音账号运营中“垂直度”的核心概念。我们将学习如何将宽泛的领域逐级细分，从而打造出更具吸引力和竞争力的账号定位。理解这一点，对于新手规划内容方向至关重要。

上一节我们介绍了内容标签的重要性，本节中我们来看看如何将“垂直”这个概念进行更精细的划分。

我们常说做账号要垂直。很多同学会简单地认为，选择“美食”或“游戏”作为领域就是垂直。这个理解没有错，但这仅仅是垂直度的第一个层级。

垂直领域是有明确的级别划分的。从宽泛到具体，我们可以将其分为一级、二级、三级甚至更多层级。

以下是垂直领域分级的具体说明：

*   **一级垂直领域**：这是最宽泛的大类。例如：**游戏**、美食、美妆、教育等。选择这个级别意味着你的内容范围非常广。

*   **二级垂直领域**：这是一级领域下的子分类。以“游戏”为例，其二级领域包括：**手游**、电竞、网游、单机游戏等。选择二级领域，你的内容范围开始聚焦。

*   **三级垂直领域**：这是二级领域下的进一步细分。它通常指向某个具体的产品、主题或风格。例如：
    *   在“手游”下，可以是：**《王者荣耀》**、**《和平精英》**、**《原神》** 等具体游戏。
    *   在“电竞”下，可以是：**《DOTA2》**、**《CS:GO》** 等。

理解了三级划分后，我们甚至可以进行四级、五级的深度垂直。这取决于你将内容聚焦到何种程度。

以下是深度垂直的举例：

*   如果你做 **《王者荣耀》**（三级领域），你可以进一步只做：
    *   **安琪拉** 这个英雄的攻略、玩法视频。
    *   只玩 **“无限乱斗”** 这一个游戏模式。
*   如果你做 **《和平精英》**（三级领域），你可以进一步只做：
    *   全部在 **“雨林”** 地图进行的游戏视频。
    *   只创作 **剧情类** 的游戏短视频。

**核心概念可以总结为公式：垂直深度 = 领域范围的具体化程度。**

从一级到多级的细分过程，就是内容范围不断收缩、定位不断清晰的过程。大领域流量广但竞争激烈，小领域竞争小但受众精准。具体选择哪个层级的垂直度，需要根据你自身的资源、兴趣和目标来决定。

![](img/545fc2c4cf3daaa89ef9580bcd19636a_3.png)

本节课中我们一起学习了垂直类目的细分体系。记住，垂直不是一个模糊的概念，而是一个可以从一级到多级逐层细分的具体策略。通过选择适合自己的垂直深度，你可以更有效地吸引目标粉丝，在抖音运营中建立起独特的竞争优势。
