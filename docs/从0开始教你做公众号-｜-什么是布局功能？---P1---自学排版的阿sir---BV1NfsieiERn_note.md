# 公众号排版教程：P1：布局功能详解 🧩

在本节课中，我们将要学习公众号推文排版中的“布局”功能。布局功能可以将整个编辑区域划分为多个部分，从而极大地提升排版的自由度和灵活性，实现类似分屏的效果。掌握它，能让你的推文结构更丰富、更具吸引力。

## 什么是布局功能？

上一节我们介绍了课程目标，本节中我们来看看布局功能的具体定义。布局功能的核心，是将一整块推文编辑屏幕分割成若干独立的区域。

**公式描述**：`完整编辑区域 = 区域1 + 区域2 + ... + 区域N`

每个区域都可以独立进行编辑操作，互不干扰。这类似于手机上的分屏功能，允许你在同一屏幕内并行处理不同内容。

## 如何使用基础布局？

了解了布局的概念后，我们进入实践环节，看看如何在编辑器中应用基础布局。

以下是进入并使用基础布局的步骤列表：

1.  打开135编辑器，进入图文编辑界面。
2.  点击编辑器左侧工具栏的“样式”选项。
3.  在样式面板中，找到并点击“布局”分类。
4.  在布局分类下，选择“基础布局”。

编辑器提供了8种常用的基础布局模板。你可以根据内容展示的需要，选择合适的布局结构。例如，可以选择左右两栏、左中右三栏或上下结构等。

## 布局功能实战演示

理论结合实践，本节我们将通过一个具体例子，演示布局功能如何提升排版自由度。

我选择了一个“两份布局”（即左右分栏）进行演示。选择该布局后，编辑区域会被分为左右两个独立的部分。

此时，你可以在左侧区域放置一张图片，同时在右侧区域放置一个文字样式模板。这种并排展示的方式，使得图文结合更加紧密和美观。

如果不使用任何布局功能，整个编辑区域将是一个整体，你只能逐个添加和调整样式模板，灵活性较低。布局功能的核心优势正在于此——它提供了更高的排版自由度。

## 探索进阶布局组合

基础布局已经能满足大部分需求，但布局的玩法不止于此。我们可以尝试组合不同的布局，创造出更丰富的效果。

首先，我们插入一个基础的“两份布局”。接着，在其中一侧的布局区域内，再次点击“布局”按钮，这次我们选择一个“上下滑动”的布局样式。

然后，我们找到一个多文字的样式模板，将其放入这个新创建的滑动布局中。这个过程，本质上是用布局功能模拟了复杂样式模板的效果，你可以自由替换其中的图片和文字。

**代码描述滑动效果**：
```html
<div class="scrollable-layout">
  <!-- 这里可以放置长图或多段内容 -->
</div>
```

实现滑动效果后，你可以在里面放入一张长图。例如，当你想在推文中展示往期文章时，可以将文章生成长图，然后通过此滑动布局嵌入。相比之下，如果直接将长图放在普通布局中，则无法实现滑动浏览的效果。

需要注意的是，这种方法是一种利用现有布局工具实现滑动效果的简便方式。若要实现更多样、更专业的交互效果，通常需要进入更高级的FVG编辑器进行设计。
