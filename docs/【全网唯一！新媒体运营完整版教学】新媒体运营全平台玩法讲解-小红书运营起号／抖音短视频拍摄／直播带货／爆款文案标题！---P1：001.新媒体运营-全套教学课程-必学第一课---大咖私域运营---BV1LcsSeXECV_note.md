# 新媒体运营：001：课程概述与讲师介绍 🚀

在本节课中，我们将要学习这套新媒体运营课程的总体框架、涵盖的核心内容，并了解讲师的专业背景与经验。

这是一套迄今为止内容最全面、最新颖且深入的新媒体运营教程。本套教程的制作耗时6个月，教学总时长超过1000分钟，并且仍在持续更新中。

教程内容覆盖了三大互联网主流运营渠道：**短视频运营**、**图文笔记运营**和**直播运营**。基于讲师超过12年的互联网运营操盘经验，累计运营超过300个主流及冷门赛道，共计超过1200个新媒体账号。

教程总结出了一套适合缺乏互联网运营思维、希望从零开始学习的同学的方法。内容涵盖**玩法内容运营**、**短视频运营起号涨粉变现**、**私域流量矩阵运营**以及**关键词与AI应用**等超过200个运营核心技巧。

我是方宇老师。我从2012年开始进入互联网运营领域，至今已拥有超过12年的互联网运营操盘经验。自2023年起，我在B站分享的新媒体运营公开课视频累计播放量已超过130万，并且这个数据未来将持续增长。

在此期间，我帮助了许多希望从事或转行新媒体运营的同学，成功获得了具备实操运营经验的工作岗位。

本套课程将结合我12年的运营经验，针对当前互联网的四个主要方向进行深度讲解，分别是：**图文笔记运营**、**短视频运营**、**直播运营**以及时下的互联网新兴产物**AI的实战应用**。

本教程将以通俗易懂的方式，深入讲解目前各大互联网平台的底层流量推送逻辑、账号的流量层级增长规则。我们将搞清楚流量从何而来、如何提升流量、如何运营以增加账号权重。

新手将学习如何从零到一打造账号的爆款内容，无论是短视频还是图文笔记。通过运营手法和技巧，我们将把新手眼中难以把控的运营问题变得可控，例如实现**可复制的模式**等。

这套教程涉及的干货知识点非常丰富。当你开始学习时，我相信你会沉浸其中，并且每一节课都可能需要反复观看。
