# 抖音运营指南：P12：3 抖音选题思路技巧 📝

在本节课中，我们将学习一套高效的抖音选题公式。这套方法能帮助你快速找到创作方向，解决“不知道发什么”的难题，从而更精准地吸引目标用户，为账号打上正确标签。

---

上一节我们探讨了账号定位的重要性，本节中我们来看看如何具体生成可持续的优质选题。

那么，有什么具体方法可以解决选题难题呢？当然有。这是一套百万粉丝博主常用的选题技巧。为了方便理解，我将其总结为一个核心公式。

**选题公式 = 内容定位 + 用户痛点 + 借势热点**

用好这个公式，你的内容方向就不会跑偏。

![](img/f0482444687989cf283a8646d6267fa4_5.png)

![](img/f0482444687989cf283a8646d6267fa4_5.png)

以下是应用该公式的具体步骤：

首先，明确你的**内容定位**。例如，如果你想成为一名知识博主，就需要挖掘行业内的关键词，例如：
*   播放量
*   粉丝量
*   账号定位
*   内容选题

这些关键词都是客观的领域核心，不包含主观描述。

接着，在定位的基础上寻找**用户痛点**。对于知识领域，用户痛点可能包括：
*   如何便捷地操作
*   如何高效地学习
*   需要哪些实用技巧
*   希望获得教学内容

最后，结合近期的**热点**进行排列组合。例如，临近春节时，可以创作如下选题：
*   “如何用拜年话术提升视频播放量”
*   “春节必火的内容选题盘点”
*   “解析‘张同学’快速涨粉的三个秘诀”

为了加深理解，我们再将这个公式套用到另一个行业试试看。以美妆行业为例：

*   **内容定位**关键词：美白、除皱、口红。
*   **用户痛点**关键词：平价好用、明星同款、避坑指南、扫雷测评。
*   **借势热点**进行组合，可以产生如下选题：
    *   “迪丽热巴同款美白面膜推荐”
    *   “七夕节千万别送女朋友这几种口红”
