# 微信视频号运营教程：P9：7.账号定位与素材获取

在本节课中，我们将学习微信视频号账号运营的两个核心环节：账号定位的初期处理与原创视频素材的获取方法。我们将解决从零开始积累粉丝和获取高质量视频素材的实际问题。

## 账号初期处理与粉丝积累

![](img/1a7be72d54516f5318d4987bd5b04b6d_1.png)

上一节我们讨论了账号定位的重要性，本节中我们来看看账号启动初期的具体操作。

首先，需要处理账号的过往内容。如果账号之前发布过作品，建议将其隐藏。这个操作是为了让账号以全新的定位面向观众。

![](img/1a7be72d54516f5318d4987bd5b04b6d_3.png)

其次，粉丝积累需要循序渐进。初期目标应设定为获取**100个粉丝**。达成这个目标后，再专注于解决视频素材问题。发布频率建议为**一天一条**，保持稳定更新。

以下是达成初期目标的步骤概要：
1.  隐藏账号之前的作品。
2.  将积累100个粉丝作为首要目标。
3.  保持每日一条视频的更新频率。

## 解决原创视频素材问题

![](img/1a7be72d54516f5318d4987bd5b04b6d_6.png)

![](img/1a7be72d54516f5318d4987bd5b04b6d_7.png)

解决了粉丝基础目标后，接下来我们面对的核心难题是如何获得优质的原创视频素材。许多初学者面临不会拍摄、不会剪辑的困境。

为此，我们提供了一个解决方案：向学员免费提供已剪辑好的视频素材。这些素材来自一个投入重金打造的独家素材库，涵盖了多种类型，例如搞笑娱乐、生活妙招等。每个素材都经过专业剪辑师处理，直接可用。

以下是素材库的部分特点：
*   **高质量**：由专业团队剪辑，素材精良。
*   **多品类**：包含搞笑、生活、情感等多种类型。
*   **独家性**：提供给学员后链接即失效，避免重复。
*   **可持续**：有庞大的剪辑团队持续生产新素材。

## 如何领取免费素材

了解素材来源后，我们来看看具体的领取规则。领取资格与课堂互动挂钩。

![](img/1a7be72d54516f5318d4987bd5b04b6d_9.png)

领取素材需要满足以下三个条件：
1.  在直播间公屏上发送“想”字。
2.  在课程中积极互动。
3.  完整听完课程（后台会统计听课时长与互动数据）。

满足条件后，请按以下流程操作：
1.  联系向你发送听课链接的助理老师。
2.  向助理老师表明是“张老师”让来领取素材的。

请注意沟通方式，礼貌地提出请求有助于顺利获得素材。
