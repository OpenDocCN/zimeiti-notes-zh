# 抖音自媒体短视频运营：P12：精选联盟

## 概述
在本节课中，我们将要学习抖音电商体系中的“精选联盟”。我们将了解它的定义、适合人群、加入条件、后台操作设置以及核心玩法。通过本课，你将掌握如何利用精选联盟平台，让达人帮你销售商品，从而实现快速起量和变现。

---

## 什么是精选联盟？
精选联盟是一个撮合商品与达人的CPS平台。**CPS** 是一种按实际销售效果支付推广佣金的结算模式。平台统一展示商品库，达人自行筛选推广。简单来说，它是一个连接商家和带货达人的中介平台。

商家负责将商品上架至平台，达人则负责挑选商品并进行推广销售。

## 精选联盟适合什么样的人？
精选联盟本质上是一个分销平台。那么，什么样的人适合作为供货方（即商家）加入呢？主要有以下三类人群：

以下是三种适合的人群：
1.  **拥有工厂的生产制造者**：这类商家产能充足，需要通过更多渠道消化库存。
2.  **供应链整合能力强的商家**：他们资源广泛，能整合多个品牌的产品，统一进行分销，赚取中间差价。
3.  **资金实力雄厚的商家**：因为供货需要垫资。例如，合作达人可能单日创造出数百万甚至上千万的销售额，商家需要有足够的资金来应对订单、发货及结算。

总结来说，**有工厂、供应链强、资金足**的商家，操作精选联盟会更为轻松。

## 如何加入精选联盟？
上一节我们介绍了适合的人群，本节中我们来看看加入的具体条件。商家需要满足平台设定的要求才能开通精选联盟功能。

![](img/02c0211b3199f757f3de07e5a45a9534_4.png)

以下是加入精选联盟的主要条件：
*   **权限要求**：店铺的精选联盟权限未被关闭过，或关闭次数小于3次。如果因违规导致权限被关闭超过3次，该店铺将永久无法再入驻。
*   **体验分要求**：店铺的“商家体验分”需大于等于4.0分（此分数为示例，具体数值可能随平台规则调整，请以开通时后台显示为准）。若体验分低于要求，则无法开通。

## 精选联盟后台操作指南
了解了加入条件后，我们进入抖店后台，看看具体如何设置。

首先，打开抖店后台，进入 **“营销中心”**。
在左侧功能栏找到并点击 **“精选联盟”**，然后进入 **“计划管理”**。

![](img/02c0211b3199f757f3de07e5a45a9534_4.png)

![](img/02c0211b3199f757f3de07e5a45a9534_8.png)

![](img/02c0211b3199f757f3de07e5a45a9534_9.png)

在这里，你可以创建三种推广计划：
1.  **普通计划**：面向所有达人的公开计划。
2.  **定向计划**：针对特定合作达人设置的计划。
3.  **专属计划**：同样是针对特定达人的计划，与定向计划类似，但适用场景略有不同。

这三种计划的佣金设置范围不同：

| 计划类型 | 佣金设置范围 |
| :--- | :--- |
| 普通计划 | 1% - 50% |
| 定向计划 | 0% - 80% |
| 专属计划 | 0% - 50% |

> **提示**：如果后期需要进行补单操作，可以创建普通计划，并将佣金设置为最低的1%。

### 如何设置“普通计划”？
我们以最常用的“普通计划”为例，演示设置步骤。

1.  在“计划管理”页面，点击 **“添加商品”**。
2.  从商品库中选择你想要推广的商品。
3.  设置佣金比例，范围在1%到50%之间。可根据产品利润空间决定，通常给达人设置在20%-30%之间较为常见。
    ![](img/02c0211b3199f757f3de07e5a45a9534_8.png)
4.  设置“寄样方式”，可以选择“免费寄样”或“不支持寄样”。如果产品库存充足且希望达人重点推广，可以选择免费寄样。
    ![](img/02c0211b3199f757f3de07e5a45a9534_11.png)
5.  点击 **“确定”**，计划即创建成功。之后可以在列表中查看和修改。
    ![](img/02c0211b3199f757f3de07e5a45a9534_14.png)

---

### 如何设置“定向计划”？
“定向计划”用于给特定合作达人更高的佣金。设置流程如下：

1.  点击 **“新建定向计划”**。
2.  选择商品，点击下一步。
3.  设置你与该达人约定好的专属佣金率（例如，普通计划20%，定向计划可设为30%）。
4.  设置计划的推广时间。
5.  输入合作达人的抖音UID，添加后提交即可。
    ![](img/02c0211b3199f757f3de07e5a45a9534_16.png)

“专属计划”的设置方法与“定向计划”类似。

## 精选联盟的核心玩法与突破策略
前面我们介绍了三种计划，其核心区别在于面向的达人群体和佣金比例不同。商家需要平衡两端：**消费者能接受的价格**和**达人期望获得的佣金**。

![](img/02c0211b3199f757f3de07e5a45a9534_18.png)

那么，如何快速突破，让更多达人为你带货呢？以下是关键策略：

以下是快速突破的策略步骤：
1.  **启动阶段：寻找关键达人**：在初期没有达人主动带货时，可以投入一些资金，寻找一个**中大型主播**合作，支付一定的坑位费。只要他能帮你把销量和热度带起来，其他中小主播就会主动来找你合作。
2.  **增长阶段：利用流量效应**：大主播借助平台流量，小主播则会跟随大主播的流量和选品。销量起来后，产品会获得更多曝光。
3.  **成熟阶段：开展自播**：当产品通过达人带货建立起一定销量和知名度后，你可以自己开设直播间进行销售。此时，你可以给出比达人更低的**厂家直售价**（例如，达人售价99元，自播价79元）。这样能吸引更多消费者直接购买，节省佣金支出，利润更高。

![](img/02c0211b3199f757f3de07e5a45a9534_20.png)

**核心建议**：只要你的产品上架，并且符合开通条件，就应立即开通精选联盟并设置合理的佣金计划。这是获取免费推广流量、启动销售的重要一步。

## 总结
本节课中，我们一起学习了抖音电商的“精选联盟”功能。我们明确了它是一个CPS分销平台，适合有供应链、资金实力的商家。我们掌握了开通的条件（体验分、权限），并详细学习了后台三种推广计划（普通、定向、专属）的设置方法。最后，我们探讨了通过“寻找关键达人启动 -> 利用流量效应增长 -> 开展自播变现”的策略路径，来实现销售的快速突破。记住，开通并利用好精选联盟，是商品在抖音生态获得曝光和销量的关键环节。
