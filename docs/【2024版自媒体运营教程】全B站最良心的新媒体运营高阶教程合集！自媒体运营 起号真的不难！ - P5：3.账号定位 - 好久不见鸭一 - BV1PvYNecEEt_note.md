# 自媒体运营教程：P5：3.账号定位 🎯

![](img/adac41b5c7be563ffdaa27b687687e23_0.png)

在本节课中，我们将要学习账号定位的核心方法。账号定位是自媒体运营的基石，它决定了你的内容方向、目标受众和长期发展潜力。一个清晰的定位能帮助你在海量内容中脱颖而出。

![](img/adac41b5c7be563ffdaa27b687687e23_2.png)

## 你是谁？—— 确定账号类型与主体

上一节我们介绍了赛道选择，本节中我们来看看如何进一步细化定位。首先需要反思三个核心问题：你是谁，你是干什么的，为什么要关注你。

首先，要确定接下来要做的账号类型。除了大类赛道（即平台定义的一级垂类），还需思考在一级垂类下细分的二级赛道。

**公式：账号垂直类型 = 一级垂类 + 二级垂类**

例如，参加运动会时，你报的项目“田径”是一级垂类，而你最擅长的“百米赛跑”则是二级垂类。通过定位账号的细分类目，可以确定账号的垂直类型。

在抖音等平台，只要定位足够清晰，任何行业都有机会。平台流量巨大，一旦做起来，增长和裂变速度会非常快。

其次，要确定账号主体是谁。账号主体是账号的拥有人和最终呈现给粉丝的主人公，需要让粉丝明确知道在看谁的作品。

在抖音商业类账号中，账号主体主要分为两类：

以下是两种账号主体的区别：

*   **商家账号**：主体是公司。内容需偏向商业化，并不断通过内容拉近与粉丝的距离，避免严肃。这既能建立信任感，也能带来好感，更符合平台的娱乐属性。
*   **达人账号**：主体是个人。首先应解决信任问题，可以通过IP打造的方式，为账号树立一个积极正面的人设。

## 你是干什么的？—— 明确内容与价值

明确了身份后，接下来需要明确这个账号是做什么的，以及能为粉丝提供什么价值。

最直接的方式是通过平台认证身份（即职业）。达人可认证黄V，公司可认证蓝V。认证后，粉丝可以清晰看到认证标签，从而明白账号的职业和大致内容方向。同时，认证也能获取平台背书，增加信任感。

在未认证前，也可以通过账号设置向粉丝明确传达业务信息。

以下是传达业务信息的几种方式：

*   头像
*   背景图
*   个人简介
*   视频封面

通过对自身商业模式和账号类型的确定，进行上述元素的业务关联统一设置。

最后，需要花时间思考：你将为粉丝提供什么样的价值？在抖音中，粉丝对博主的价值需求主要分为以下几类：

以下是粉丝关注的五种核心价值类型：

1.  **知识**：减少知识盲区，获得认同感（如三观、观点认同）。
    *   **认知型**：人生思考、思维方式、读书分享、行业内幕、成功经验、个人提升。
    *   **干货型**：技能提升、经验总结、过程步骤、冷知识。
    *   **常识普及型**：生活、健康、产品、科学、历史、地区等常识的解读与普及。
2.  **情绪**：引发观众共鸣，直接感染观众。
    *   类型包括：开心愉悦（搞笑、夸张）、关系共鸣（情侣、婆媳、家庭等）、情绪波动（争议、反转、反差、反常识内容引发的情绪），以及怀旧、emo、震惊、好奇等。
3.  **感觉**：通过视听元素引发共情，让观众在意识层面与作者产生连接。
    *   价值体现：视频的画面美感、构图、视觉冲击；个人魅力带来的仰慕感；声音带来的节奏感或听觉享受。常见于颜值、萌宠、美景、才艺、文化类账号。
4.  **体验**：满足好奇心，提供获得感。
    *   价值体现：让用户看到不同的生活方式、了解未知领域、体验未曾经历的事物。常见于美食（制作与享用）、旅行、运动、角色体验等账号。
5.  **期待**：制造意犹未尽的感觉，促使用户点击关注。
    *   方法：利用内容的吸引力、观看体验或对作者本人的好奇，不将所有内容在一个视频中呈现完，留下悬念，让用户期待下次更新。

## 总结

本节课中我们一起学习了账号定位的系统方法。我们首先探讨了如何通过确定账号类型（一级与二级垂类）和主体（商家或达人）来回答“你是谁”。接着，我们学习了如何通过认证和账号设置明确“你是干什么的”。最后，我们详细分析了能为粉丝提供的五种核心价值：**知识**、**情绪**、**感觉**、**体验**和**期待**。一个成功的账号定位，需要清晰、独特且能持续提供明确价值，这是吸引并留住粉丝的关键。
