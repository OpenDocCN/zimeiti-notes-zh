在本节课中，我们将学习抖音千川广告在增长期的计划搭建方法。增长期是账号已建立初步模型后的阶段，我们将重点学习如何利用已有标签，进行更精准、更高效的人群投放，以实现账户的稳定增长。

上一节我们介绍了模型期的投放策略，本节中我们来看看增长期如何操作。进入增长期后，账户已拥有基础标签，这使得投放比模型期更为简单。因为我们已经能够预估出价，知道大概出多少钱能让计划跑动起来。

## 如何确定出价 💰

在增长期，我们可以根据历史数据来设定合理的出价。查看你之前跑动的计划数据是关键。

![](img/a4e8c92ebfea383c5c1cd3655d4847fe_4.png)

![](img/a4e8c92ebfea383c5c1cd3655d4847fe_6.png)

![](img/a4e8c92ebfea383c5c1cd3655d4847fe_8.png)

![](img/a4e8c92ebfea383c5c1cd3655d4847fe_10.png)

例如，一个计划消耗了665元，成交金额为9000元。其中，平均千次展现成本（CPM）为86元，转化成本为101元。这意味着，平均每获得一个成交订单，需要花费101元。客单价为1500元。

**核心公式**：
*   `转化成本 = 总消耗 / 成交订单数`
*   示例：`101元 ≈ 665元消耗 / 成交订单数`

![](img/a4e8c92ebfea383c5c1cd3655d4847fe_12.png)

![](img/a4e8c92ebfea383c5c1cd3655d4847fe_14.png)

通过这个数据，你就能知道，出价大约在转化成本的80%-100%左右，计划就有可能跑动。例如，转化成本是101元，那么出价设为88元就可能跑得动。这帮助你明确了账户的转化成本水平，为后续出价提供依据。

## 增长期投放策略：精耕细作 🎯

![](img/a4e8c92ebfea383c5c1cd3655d4847fe_16.png)

![](img/a4e8c92ebfea383c5c1cd3655d4847fe_18.png)

在增长期，投放策略可以从“泛投圈人”转向“精耕细作”。因为系统已经了解你需要什么样的人群，所以我们可以更精准地定位。

以下是增长期搭建计划的核心步骤：

**第一步：缩小人群包范围**
在模型期，我们可能使用5000万左右的大人群包来探索。进入增长期后，我们可以将人群包范围缩小到2000万左右。这样做的原因是，账户已有标签，系统能更准确地在这个缩小范围内找到目标用户，计划更容易起量。

**第二步：利用人群洞察进行精准交集**
这是增长期提升投放效率的关键。我们不再只是投放宽泛的人群标签，而是将直播间的高转化人群与行业高潜人群进行交集，打造超精准人群包。

![](img/a4e8c92ebfea383c5c1cd3655d4847fe_20.png)

![](img/a4e8c92ebfea383c5c1cd3655d4847fe_22.png)

1.  **确定高转化人群**：在人群洞察中，分析你直播间成交金额或转化率最高的人群标签，例如“小镇中老年”、“资深中产”。
2.  **确定高潜行业人群**：分析行业偏好中，与你的产品（如智能家居）相关度最高的潜在、核心、高购买力人群。
3.  **进行人群交集**：将你的高转化人群（如“小镇中老年”）与行业高潜人群（如“智能家居”人群）进行交集运算。这样得到的人群，既是你的目标客户画像，又对相关产品有高兴趣。

**操作示例（代码逻辑描述）**：
```python
# 假设我们有两个人群包
高转化人群包 = [‘小镇中老年’, ‘资深中产’]
行业高潜人群包 = [‘智能家居潜在人群’, ‘智能家居核心人群’, ‘智能家居高购买力人群’]

![](img/a4e8c92ebfea383c5c1cd3655d4847fe_24.png)

# 进行交集运算，得到精准投放包
精准投放包 = 交集(高转化人群包[0], 行业高潜人群包) # 例如：“小镇中老年” ∩ “智能家居各类人群”
```
通过此方法，你可以创建多个精准人群包（例如：小镇中老年x智能家居、资深中产x服饰鞋包等）。每个包的人数可能只有200-400万，但由于极度精准，在增长期同样能跑动。

![](img/a4e8c92ebfea383c5c1cd3655d4847fe_26.png)

**第三步：大小计划组合投放**
增长期不应只依赖一种计划类型，而应采用组合策略：

![](img/a4e8c92ebfea383c5c1cd3655d4847fe_28.png)

![](img/a4e8c92ebfea383c5c1cd3655d4847fe_30.png)

![](img/a4e8c92ebfea383c5c1cd3655d4847fe_32.png)

*   **大包计划（如2000万人群）**：用于保障稳定的流量供给和探索更深度的转化（如ROI目标）。因为范围大，系统有空间寻找更优质用户。
*   **小包计划（如300-400万精准交集包）**：用于提升投放效率和ROI。人群精准，转化路径短，适合设置成交目标。

这种“大包保量，小包保效”的组合，可以让账户在增长期更稳健地发展。

## 重要补充与技巧 ⚠️

在搭建计划时，还需要注意以下几点：

*   **极速推广需谨慎**：即使账户进入增长期或成熟期，也不应过度依赖极速推广。极速流量过大可能导致模型跑偏，影响ROI。正确的做法是主投专业计划，用极速计划作为辅助。
*   **搜索广告的搭建**：搜索广告是增长期获取精准流量的有效补充。其搭建相对简单：
    1.  在关键词推荐中，下载行业高搜索量词表（如月搜索量>5000）。
    2.  筛选出与产品相关的核心词（如“连衣裙”）。
    3.  采用“短语匹配”方式上传关键词。
    4.  定向设置上，依然可以叠加你的核心人群标签（如年龄、性别、消费能力）。
    5.  可以分别建立“短语匹配”和“精准匹配”计划来测试效果。

**核心操作**：通过人群交集功能，将你的直播间粉丝模型与行业兴趣标签结合，创建出多个200-400万规模的超精准人群包进行测试投放。
