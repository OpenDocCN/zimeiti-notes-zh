# 无人直播入门：P1：素材循环与直播广场推荐

在本节课中，我们将学习如何通过循环播放素材实现无人直播，并成功获得直播平台的推荐，从而吸引大量观众进入直播间。

上一节我们介绍了课程的整体目标，本节中我们来看看实现无人直播并提升人气的具体步骤。

## 核心步骤

以下是实现无人直播并提升在线人数的几个关键操作。

1.  **准备直播素材**：预先录制或制作好一段高质量的视频或画面内容。
2.  **设置循环播放**：使用直播软件（如OBS）将准备好的素材设置为循环播放模式。核心操作是在来源中添加媒体源，并勾选“循环”选项。
    ```obs
    [来源] -> 添加 -> 媒体源 -> 选择文件 -> 勾选“循环”
    ```
3.  **开启直播推流**：在直播软件中正确配置直播平台的推流地址和串流密钥，然后开始推流。
4.  **触发平台推荐机制**：通过稳定的直播内容、合适的直播标签和分类，吸引平台算法将直播间推荐至“直播广场”等流量入口。
5.  **观察数据提升**：成功被推荐后，直播间在线人数会显著上升，案例中达到了两千多人在线。

## 注意事项

在操作过程中，有以下几点需要特别留意。

*   确保素材内容符合平台规定，避免违规。
*   循环播放的素材时长不宜过短，以避免观众察觉明显的重复感。
*   保持网络连接稳定，防止直播中断影响推荐。

本节课中我们一起学习了无人直播的基本设置流程，从准备循环素材到获得平台推荐的关键步骤。掌握这些方法，可以有效搭建一个自动运行的直播间并吸引初始流量。
