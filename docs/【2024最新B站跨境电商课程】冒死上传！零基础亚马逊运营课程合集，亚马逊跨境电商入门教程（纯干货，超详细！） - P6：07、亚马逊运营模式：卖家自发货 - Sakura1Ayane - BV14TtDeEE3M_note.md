# 亚马逊跨境电商入门教程：P6：07、亚马逊运营模式：卖家自发货 🚚

![](img/6e19c8315102e07b06356ca5873a4384_1.png)

在本节课中，我们将要学习亚马逊卖家自发货模式。这是一种非常适合新手的起步方式，其核心在于无需提前囤货，可以极大地降低初始运营风险。

## 模式概述

卖家自发货模式，英文简称 **FBM**。其核心流程是：卖家在亚马逊店铺上架商品，当有海外顾客下单后，卖家再从国内采购该商品，并直接发货给海外顾客。

**核心流程公式**：
`选品 -> 上架亚马逊 -> 接收订单 -> 国内采购 -> 直发海外`

上一节我们介绍了FBA模式，本节中我们来看看与之相对的FBM模式。

## 运营思路详解

![](img/6e19c8315102e07b06356ca5873a4384_6.png)

以下是卖家自发货模式的具体操作思路：

1.  **选品**：从国内电商平台（如1688、淘宝等）选择你认为有潜力的商品。
2.  **上架**：将该商品的标题、图片、描述等信息发布到你的亚马逊店铺中。
3.  **销售**：在亚马逊店铺进行销售，等待海外顾客下单。
4.  **采购与发货**：一旦产生订单，你立即从国内货源处采购该商品，并安排直接发往海外顾客地址。

这种模式常被称为“无货源模式”，因为你并不需要事先拥有所售商品的库存。

## 模式优势与劣势

了解了基本思路后，我们来分析一下这种模式的优缺点。

### 优势：低风险启动

以下是FBM模式的主要优势点：

*   **轻资产运营**：无需提前囤货和压货，启动资金门槛低，通常几千到一万元即可开始。
*   **风险可控**：由于没有库存压力，可进可退。如果某款产品销量不佳，可以直接下架，几乎没有经济损失。
*   **灵活测试**：非常适合新手测试市场，寻找潜力商品。如果发现某款FBM商品销量稳定，可以随时转为FBA模式，批量发货到亚马逊仓库。

### 劣势：物流体验

与FBA模式相比，FBM模式也存在明显的短板：

*   **物流耗时较长**：从国内发货到海外，物流时间通常比FBA的本地配送要长，可能影响顾客购买体验和店铺绩效指标。

## 模式组合策略

一个常见的问题是：FBA和FBM模式能否在同一个店铺中并存？

答案是肯定的。你完全可以同时运营FBA和FBM商品。例如，你可以将经过市场验证、销量稳定的商品转为FBA，以提升物流速度；同时，继续用FBM模式测试新的商品或销售长尾商品。这种组合策略能让你更灵活地管理店铺和库存。
