# 微信视频号运营教程：P13：11.如何填写邀请函 📝

在本节课中，我们将学习如何正确填写视频号创作者计划的邀请函。这是加入计划、获取官方扶持的关键一步。整个过程简单明了，我们将一步步拆解，确保每位初学者都能轻松掌握。

## 第一步：找到你的视频号账号信息 🔍

![](img/7a99b757986f46c88e6fb4a521ffe8c7_1.png)

上一节我们介绍了如何注册视频号，本节中我们来看看如何找到账号昵称和粉丝数量。这是填写邀请函的基础信息。

首先，你需要打开手机上的微信应用。



![](img/7a99b757986f46c88e6fb4a521ffe8c7_1.png)

![](img/7a99b757986f46c88e6fb4a521ffe8c7_3.png)

其次，在微信底部导航栏找到并点击“发现”页面。



![](img/7a99b757986f46c88e6fb4a521ffe8c7_4.png)

![](img/7a99b757986f46c88e6fb4a521ffe8c7_3.png)

第三步，在“发现”页面中向上滑动，找到“视频号”入口（图标类似一只黄色的蝴蝶）。



![](img/7a99b757986f46c88e6fb4a521ffe8c7_4.png)

第四步，进入视频号后，点击页面右上角的个人头像图标。



![](img/7a99b757986f46c88e6fb4a521ffe8c7_6.png)

![](img/7a99b757986f46c88e6fb4a521ffe8c7_6.png)

![](img/7a99b757986f46c88e6fb4a521ffe8c7_7.png)

点击进入后，向下滑动页面，找到“我的视频号”部分。这里会清晰显示你的**账号昵称**和**粉丝数量**。

例如，显示为：
```
昵称：天道酬勤
粉丝：39917
```

![](img/7a99b757986f46c88e6fb4a521ffe8c7_7.png)

## 第二步：填写邀请函具体内容 ✍️

现在我们已经找到了账号信息，接下来开始正式填写邀请函。邀请函通常包含以下几个关键字段。

以下是需要填写的项目列表：

1.  **视频号账号昵称**：填写上一步中找到的昵称，例如“天道酬勤”。
2.  **粉丝数量**：填写当前账号的粉丝数，例如“39917”。
3.  **活动邀请人**：由于是通过特定渠道加入，此处需填写邀请人姓名，例如“张良”或“张良老师”。
4.  **实名信息**：此处必须填写你本人的真实姓名，例如“周建国”。
5.  **结算手机号**：填写用于接收验证码和后续结算的手机号码，例如“13812345678”。注意号码中**不能有空格**。



![](img/7a99b757986f46c88e6fb4a521ffe8c7_9.png)

## 第三步：提交与确认 ✅

![](img/7a99b757986f46c88e6fb4a521ffe8c7_9.png)

![](img/7a99b757986f46c88e6fb4a521ffe8c7_10.png)

所有信息填写完毕后，最后一步是提交并确认。

![](img/7a99b757986f46c88e6fb4a521ffe8c7_11.png)

请仔细检查所填信息无误，然后点击页面上的**蓝色“提交”按钮**。



![](img/7a99b757986f46c88e6fb4a521ffe8c7_10.png)

系统会弹出确认框，提示“确定要提交吗？”，点击**确认**即可完成。



![](img/7a99b757986f46c88e6fb4a521ffe8c7_11.png)

![](img/7a99b757986f46c88e6fb4a521ffe8c7_13.png)

提交成功后，页面会显示“已成功加入创作者分成计划”的提示。

![](img/7a99b757986f46c88e6fb4a521ffe8c7_14.png)
