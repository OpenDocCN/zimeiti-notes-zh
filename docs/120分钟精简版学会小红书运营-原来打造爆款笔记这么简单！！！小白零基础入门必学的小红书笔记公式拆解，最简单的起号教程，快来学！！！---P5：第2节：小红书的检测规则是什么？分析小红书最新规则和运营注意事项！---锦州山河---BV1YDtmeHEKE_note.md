# 小红书运营入门：P5：第2节：小红书平台检测规则详解 🔍

在本节课中，我们将要学习小红书平台的核心检测规则。了解这些规则是避免账号违规、确保内容安全发布的基础。我们将从平台检测的多个维度进行分析，并提供实用的工具和方法来辅助你的运营工作。

## 平台检测的四个核心维度

上一节我们介绍了小红书的基本概念，本节中我们来看看平台具体在哪些方面进行内容审核。小红书的检测系统主要针对以下四个维度进行审查。

以下是平台重点检测的四个具体方面：

1.  **账户资料与内容**：平台会检测个人资料、笔记正文、私信以及评论。这些地方不得出现微信号、微信二维码、代购、水印、推广淘宝店铺等明显的营销词汇。
2.  **设备与账号关联**：平台会检测登录设备。建议同一台设备最多只登录三个账号，超出可能增加风险。
3.  **用户互动规范**：笔记内容不得诱导用户。例如，“关注领奖”、“点赞抽奖”等引导互动行为是平台明令禁止的。
4.  **内容原创度**：小红书对原创内容的保护程度较高。切勿将同一篇文章在多个账号上重复发布，应遵循“一篇文章对应一个账号”的原则。

## 敏感词检测与规避方法

除了上述通用规则，内容中的敏感词是另一个审查重点。平台对某些特定领域词汇（如手机卡、金融、电子烟、医疗用品等）管控严格。

如果你在撰写笔记时，不确定是否包含了违规词汇，可以使用专门的敏感词检测工具进行排查。以下是两个实用的在线检测网站及其使用方法。

**第一个网站（无需注册）**：
1.  访问检测网站。
2.  将写好的笔记内容粘贴到输入框中。
3.  点击“立即检测”按钮。
4.  系统会高亮或列出内容中存在的小红书平台敏感词。

例如，输入一段包含“代购”的文本，检测结果会明确提示该词为敏感词。

1.  注册并登录该网站。
2.  同样将内容粘贴至检测框。
3.  执行检测后，网站会返回过滤后的结果，并列出发现的敏感词。
