# 小红书运营：P21：如何制作素材

## 概述
在本节课中，我们将学习如何为小红书内容制作获取无水印的图片与视频素材。整个过程将使用一款名为“龙猫工具大师”的免费工具，操作简单，适合零基础学员。

## 准备工作
在开始制作素材前，需要先准备好工具。我们将使用“龙猫工具大师”这款应用程序。

以下是具体准备步骤：
1.  在手机应用商店搜索并下载“龙猫工具大师”App。
2.  打开App，使用微信账号进行登录。
3.  该工具完全免费，但初始加载可能较慢，请耐心等待。

## 获取使用次数
上一节我们准备好了工具，本节中我们来看看如何确保有足够的使用次数。工具虽免费，但部分功能有次数限制。

“我的”页面内设有“任务中心”，如果提取素材的次数不足，可以通过完成任务来获取更多次数。

以下是增加次数的具体方法：
*   点击“签到”领取每日奖励。
*   完成页面内显示的其他简单任务。
*   完成任务后，即可在相应位置领取提取次数。

## 提取小红书素材
掌握了基础操作后，现在我们进入核心环节：如何从小红书提取无水印素材。

整个过程无需复杂操作，只需复制链接并粘贴即可。

以下是提取小红书素材的步骤：
1.  在小红书App中找到想要的笔记，点击“分享”按钮并选择“复制链接”。
2.  打开“龙猫工具大师”App，它会自动识别剪贴板内容，弹出“允许粘贴”提示，点击“允许”。
3.  在弹出的“提取已复制的链接吗？”对话框中，点击“提取”。
4.  稍等片刻，素材加载完成后，点击“保存”按钮，即可将无水印的图片或视频保存到手机相册。

## 提取抖音素材
除了小红书，我们同样可以获取抖音平台的视频素材。其操作逻辑与提取小红书素材完全一致。

以下是提取抖音素材的步骤：
1.  在抖音App中找到目标视频，点击“分享”按钮后选择“复制链接”。
2.  切换到“龙猫工具大师”App，允许粘贴后点击“提取”。
3.  视频素材解析出来后，点击“保存到相册”即可获得无水印视频。

## 其他备用工具
除了“龙猫工具大师”，市面上还有其他类似工具可供选择，作为备用方案。

这些工具多为小程序，使用时通常需要观看一段广告来解锁功能。

以下是两个可选的备用工具：
*   **蜜趣**：在微信小程序中搜索“蜜趣”即可找到。
*   **青INDLE**：同样在微信小程序中搜索“青INDLE”即可使用。

## 总结
本节课我们一起学习了如何为小红书内容制作获取无水印素材。核心流程是：**复制链接 -> 打开工具 -> 提取素材 -> 保存到相册**。我们主要使用了“龙猫工具大师”这款免费工具，并介绍了增加使用次数的方法以及“蜜趣”、“青INDLE”等备用工具。掌握这些方法，你就能轻松获取各类平台的素材用于内容创作。
