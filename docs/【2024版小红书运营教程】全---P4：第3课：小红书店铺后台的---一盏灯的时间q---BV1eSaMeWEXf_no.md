# 小红书电商运营课：P4：第3课：小红书店铺后台的基础设置 🛠️

在本节课中，我们将学习如何完成小红书店铺后台的基础设置。这是店铺开通后，正式开始运营和销售商品前必须完成的关键步骤。我们将逐一讲解保证金、支付方式、运费险、物流设置和店铺公示这五个核心环节。

上一节我们介绍了如何开通店铺，本节中我们来看看店铺开通后需要完成哪些基础配置。

## 一、 保证金 💰

保证金是开店的基础要求之一。对于个人店铺，保证金金额通常是1000元。这笔费用在店铺关闭后可以申请退还。

在店铺运营初期，你可以选择暂不缴纳保证金，这不会影响店铺的基本功能。但为了提升店铺的完整度和可信度，建议在资金允许的情况下先行缴纳。

**操作路径**：进入小红书商家后台 -> 点击“资金” -> 选择“保证金”。

以下是关于保证金的详细说明：
*   **金额**：个人店铺的保证金通常为 **1000元**。
*   **缴纳时机**：初期可交可不交，建议有收入后再缴纳。
*   **可退还性**：保证金在关店后可申请全额退还。
*   **类目查询**：如需确认具体类目的保证金金额，可在后台“课堂”中搜索“保证金”查看官方规则。

## 二、 支付方式 💳

支付方式的设置关系到顾客的付款渠道和你的货款提现。需要绑定支付宝和微信支付账户。

**操作路径**：进入小红书商家后台 -> 点击“资金” -> 选择“货款资金”。

以下是支付方式设置的要点：
*   **支付宝账户**：按提示操作即可，通常当天可以完成认证和审核。
*   **微信支付账户**：若不缴纳保证金，则无法开通微信支付，但这基本不影响订单转化。开通审核通常需要一天时间。
*   **账户一致性**：绑定的支付宝、微信支付及银行卡的实名信息，必须与开店认证的身份证信息一致。

## 三、 运费险 📦

开通运费险可以降低消费者的购物顾虑，对提升转化率，尤其是服装鞋包等品类的转化率有积极作用。

**操作路径**：进入小红书商家后台 -> 点击“交易” -> 选择“运费保/运费险”。

以下是关于运费险的核心信息：
*   **建议开通**：特别是对于服装、鞋包等需要试穿、易产生退换货需求的品类。
*   **费用**：每成交一单，系统会扣除一小笔费用（例如几毛钱）。
*   **作用**：当顾客发生退货时，平台会向顾客补贴运费（例如7元），从而减少其下单顾虑。

## 四、 物流设置 🚚

物流设置包括地址库管理、运费模板等，是确保订单正常发货和退货的关键。

**操作路径**：进入小红书商家后台 -> 点击“交易” -> 选择“物流工具”。

### 1. 地址库管理

![](img/dff0270470b870d5dd62fa20484701ca_1.png)

地址库需要设置发货地址和退货地址。

![](img/dff0270470b870d5dd62fa20484701ca_3.png)

*   **发货地址**：建议填写你自己的真实地址和手机号。若你所在地快递不发达，可填写杭州、上海、广州等一线城市地址，这样前台会显示从该城市发货，有利于吸引顾客。
*   **退货地址**：这是一个重要技巧。由于你可能有多个供货商，他们的退货地址各不相同，而系统默认只有一个退货地址。
    *   **技巧**：将默认退货地址设置为一个“无法寄达”的地址，例如中国港澳台地区，并在详细地址栏填写“**已咨询客服获取**”。这样能避免顾客因系统自动同意退货而将商品寄到错误地址。
    *   **多地址退货**：小红书支持为不同商品设置独立的退货地址。你可以在地址库中为每个热销商品新建一个退货地址，并关联对应的商品链接，实现精准退货。

### 2. 运费模板

![](img/dff0270470b870d5dd62fa20484701ca_5.png)

系统会提供默认的运费模板，上架商品时必须正确选择。

![](img/dff0270470b870d5dd62fa20484701ca_6.png)

*   **关键选择**：务必选择“**默认非偏远地区包邮**”模板。
*   **原因**：该模板默认对新疆、西藏、青海、宁夏、内蒙古、海南等偏远地区不包邮。这样可以避免因向这些地区包邮而产生额外的运费损失，因为你的上游供货商（如拼多多）通常也不对这些地区包邮。

## 五、 店铺公示 📢

![](img/dff0270470b870d5dd62fa20484701ca_8.png)

![](img/dff0270470b870d5dd62fa20484701ca_9.png)

完成店铺信息公示是商品上架前的最后一步。如果未完成，系统会阻止你发布商品。

**操作路径**：根据后台提示找到店铺公示入口（通常在设置相关页面）。

填写店铺公示信息时，请确保所有联系方式（联系人、手机号）为你本人的真实信息，以便平台在遇到紧急问题时能联系到你。经营类型可以全选，填写完毕后点击保存即可。
