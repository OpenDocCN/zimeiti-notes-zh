# 新媒体运营教程：P18：02.3：工具篇-AI生成PPT 🛠️

## 概述
在本节课中，我们将学习第二款强大的AI生成PPT工具。这款工具在生成速度、稳定性和模板丰富度方面具有优势，能与上一节课分享的工具形成互补，帮助我们更高效地制作演示文稿。

上一节我们介绍了第一款AI生成PPT工具，本节中我们来看看另一款同样高效但侧重点不同的工具。

## 工具核心优势
这款工具主要有以下几个优势：
*   **生成速度快且稳定**：响应迅速，操作流畅。
*   **模板库庞大丰富**：提供海量模板样式，便于选择和调整细节。
*   **内容依赖详细大纲**：要获得最佳生成效果，需要提供结构清晰、内容详实的大纲。

## 两种生成方式
该工具提供两种核心的PPT生成方式。

![](img/34e131c2d4d7cb736d21dc00d5e02b99_1.png)

以下是两种方式的具体操作步骤：

![](img/34e131c2d4d7cb736d21dc00d5e02b99_3.png)

1.  **导入本地大纲生成**
    *   首先，使用如ChatGPT等AI工具，根据主题生成一份详细的大纲和内容。
    *   将生成的内容整理成带有清晰层级结构的文档。
    *   在PPT工具中选择“导入本地大纲”功能，上传整理好的文档。
    *   工具会快速解析文档结构，随后用户可从丰富的模板库中挑选合适的风格。
    *   点击“生成PPT”，工具便会基于大纲和所选模板自动生成幻灯片。

2.  **AI智能生成**
    *   在工具界面选择“AI智能生成”功能。
    *   直接输入PPT的主题，例如“装修行业在小红书平台的运营推广计划”。
    *   工具会自动构思并生成详细的大纲和内容。
    *   同样，在生成大纲后，挑选一个合适的模板。
    *   点击生成，工具将合成完整的文案并套用模板，形成初版PPT。

![](img/34e131c2d4d7cb736d21dc00d5e02b99_1.png)

## 编辑与优化
生成初版PPT后，可以进行灵活的编辑和优化。

以下是主要的编辑功能：

*   **大纲编辑**：在编辑界面，可以随时调整和细化每一页幻灯片的具体内容。通过回车换行等方式优化文本格式，有助于工具更好地理解内容结构。
*   **模板更换**：如果对当前页面或整套幻灯片的样式不满意，可以一键更换模板。工具提供了“模板替换”功能，响应速度很快。
*   **元素调整**：右侧编辑栏提供文字设置、形状设置、背景设置等功能，支持在线更换图片，可自定义程度较高。
*   **页面布局**：单击单张幻灯片，可以更换该页的布局模板，直到找到最合适的排版。

![](img/34e131c2d4d7cb736d21dc00d5e02b99_3.png)

## 使用技巧与总结
本节课中我们一起学习了第二款AI生成PPT工具的使用方法。

结合两款工具，可以总结出以下技巧：
*   **内容为王**：无论是“导入大纲”还是“AI生成”，提供**结构清晰、细节丰富**的原始内容（大纲）是获得高质量PPT的关键。可以先用ChatGPT等工具辅助完善内容。
*   **方式选择**：“导入本地大纲”方式**可控性更强**，适合内容已准备充分的场景；“AI智能生成”方式**更便捷快速**，适合快速起稿。
*   **善用编辑**：生成后务必利用工具的编辑功能进行**细节调整和模板优化**，使PPT更符合个人需求。

这款工具以其**快速的响应、海量的模板**与上一节课的工具形成了良好互补。工具的领取地址和方式可在视频下方的置顶评论区及简介信息中找到。
