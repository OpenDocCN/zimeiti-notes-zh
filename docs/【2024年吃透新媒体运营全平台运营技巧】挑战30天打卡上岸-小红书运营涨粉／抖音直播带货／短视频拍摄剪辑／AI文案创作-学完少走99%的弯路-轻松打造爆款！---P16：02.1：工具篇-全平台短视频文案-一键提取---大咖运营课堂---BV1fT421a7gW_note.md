# 新媒体运营全平台技巧：P16：02.1：工具篇-全平台短视频文案一键提取 📝

在本节课中，我们将学习一个极其高效的运营工具，它能帮助我们从各大短视频平台一键提取视频文案，从而极大地提升内容创作和素材整理的效率。

上一节我们介绍了文案创作的重要性，本节中我们来看看如何快速获取优质文案素材。以往，当我们看到优秀的短视频时，想要提取其文案内容，通常需要借助一些较为原始的方法。

以下是几种传统的文案提取方法：
*   使用“轻抖”等小程序。
*   先将视频下载到本地，再导入“剪映”等剪辑软件中提取字幕。

这些方法虽然可行，但存在平台限制和操作繁琐的问题。例如，抖音平台可以使用轻抖，但如果面对小红书、视频号、哔哩哔哩、快手等多个平台，就需要分别寻找下载和提取的方法，过程复杂。

因此，今天我将分享一个特别实用的在线工具。它能够跨平台工作，你只需将目标短视频的链接复制到工具网页中，即可在线一键提取出视频的文案内容。
