# 新媒体运营：P14：01.10.2：训练AI写出符合小红书风格的文案 ✍️🤖

在本节课中，我们将学习如何训练AI（如ChatGPT）来撰写符合小红书平台风格的文案。上一节我们介绍了如何让AI生成爆款标题，本节中我们来看看如何让AI理解并创作出符合平台调性的完整内容。

## 概述

撰写符合平台风格的内容，关键在于让AI理解该平台独特的语言习惯、语气和内容结构。我们将通过向AI“投喂”特定案例，教会它分析并模仿小红书的文案风格。

## 核心步骤：向AI提供学习材料

![](img/3a4a63cda67e46c57e2ac1e691b83294_1.png)

要让AI理解小红书风格，你需要提供足够多且具有代表性的爆款笔记作为学习样本。

![](img/3a4a63cda67e46c57e2ac1e691b83294_3.png)

![](img/3a4a63cda67e46c57e2ac1e691b83294_4.png)

![](img/3a4a63cda67e46c57e2ac1e691b83294_5.png)

以下是向AI下达初始指令的示例代码，用于设定学习目标：

```
接下来我会给你提供一些小红书上面的爆款笔记文案。
你需要根据这些文案，分析小红书平台的文案撰写风格、特定的平台用语、词汇、语气等。
每次接收文案后，请告诉我你已学习，并给出你对这篇文案的风格、语气、类型等相关分析。
```

![](img/3a4a63cda67e46c57e2ac1e691b83294_7.png)

**注意**：这个学习过程不是一次性的。在同一个对话中持续向AI提供资料，它能够积累“记忆”，从而更好地理解你的要求。

![](img/3a4a63cda67e46c57e2ac1e691b83294_9.png)

## 选择学习材料的要点

![](img/3a4a63cda67e46c57e2ac1e691b83294_10.png)

![](img/3a4a63cda67e46c57e2ac1e691b83294_12.png)

![](img/3a4a63cda67e46c57e2ac1e691b83294_14.png)

选择提供给AI的文案时，需要注意几个核心要素，以确保AI能学到精髓。

![](img/3a4a63cda67e46c57e2ac1e691b83294_16.png)

![](img/3a4a63cda67e46c57e2ac1e691b83294_18.png)

以下是选择学习材料时需要关注的几个要点：

1.  **人称与视角**：观察文案是以第一人称分享，还是以第二、第三人称叙述。小红书多以“我”的亲身经历展开。
2.  **平台特色用语**：注意收集如“宝子”、“姐妹”、“安利”、“踩雷”、“种草”等平台常用词汇。
3.  **语气与情绪**：分析文案是活泼少女风、亲切分享型，还是吐槽避坑风格。
4.  **内容结构**：避免只提供通用的流水账（如第一天、第二天行程）。应选择那些有强烈个人观点、情感表达或独特结构的文案。

![](img/3a4a63cda67e46c57e2ac1e691b83294_20.png)

**关键**：你提供的材料质量，直接决定了AI学习的效果。应寻找多样化的、能体现平台特色的文案，而非千篇一律的通用内容。

## 实践演练：生成小红书风格文案

在向AI提供一定量的学习材料后，即可指令它进行创作。

以下是一个生成小红书风格旅游攻略的指令示例：

```
请结合你的学习，帮我写一篇具有爆款潜质的张家界5日游攻略小红书笔记。
要求涵盖丰富的小红书风格的emoji表情，同时配上十个符合小红书旅游赛道风格的爆款标题。
```

**结果分析**：初次生成的内容可能不尽如人意。例如，AI可能只罗列景点名称，而缺乏深度分析和实用建议（如路线规划、避坑指南、推荐理由）。这说明提供的学习材料可能不足或不够典型。

**解决方案**：可以分步引导AI。例如，先让它列出张家界所有知名景点，再针对每个景点，指令它结合网络评价总结出特色与避坑点，最后整合成一篇详实、有个人观点的攻略。

## 关于标题生成的补充说明

如果需要AI生成高质量标题，最好在新的对话中，结合上一节课所教的**固定标题指令模板**和**爆款关键词库**来操作。这样能避免当前对话中其他学习材料的干扰，使标题更符合爆款公式。

**公式**：`爆款标题 = 固定指令模板 + 垂直赛道关键词库 + 核心主题`
