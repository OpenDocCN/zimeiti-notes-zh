# 剪映手机版教学：P16：核心剪辑功能--特效功能 🎬

在本节课中，我们将要学习剪映App中一个非常有趣且强大的功能——特效。特效功能借鉴了短视频平台中流行的道具玩法，通过在视频中添加各种动态效果，可以让你的视频变得更加生动、有趣，并极大地提升视频的观感和氛围。

上一节我们介绍了其他核心剪辑功能，本节中我们来看看如何为视频“锦上添花”，使用特效功能。

![](img/485ddf6f5f3ae94fcaeb99912ecae7e0_1.png)

## 特效功能概述 ✨

在剪映的特效功能中，内置了海量的特效素材库。通过在视频的特定时间点植入合适的特效元素，可以大幅提升视频的整体画面感觉和氛围。这一功能在制作变装类、反转类或美妆类视频时尤为常用。掌握特效的使用，对于提升视频的最终效果有非常大的帮助。

## 特效应用实例解析

以下是本节课将使用的一个热门视频案例，它巧妙地运用了多种特效。

![](img/485ddf6f5f3ae94fcaeb99912ecae7e0_1.png)

![](img/485ddf6f5f3ae94fcaeb99912ecae7e0_3.png)

这个案例是之前非常流行的“变装甩头发”视频。它虽然简短，但通过特效的运用，给人耳目一新的感觉。我们接下来将分解这个视频，看看它具体使用了哪些特效。

![](img/485ddf6f5f3ae94fcaeb99912ecae7e0_5.png)

1.  **开场部分**：使用了一个“模糊变清晰”的特效，让视频从模糊状态逐渐变得清晰，营造开场悬念。
    ![](img/485ddf6f5f3ae94fcaeb99912ecae7e0_8.png)

![](img/485ddf6f5f3ae94fcaeb99912ecae7e0_6.png)

2.  **舞蹈部分**：在人物跳舞时，背景添加了紫色的“妖气”类梦幻特效，增强了动感和氛围。
    ![](img/485ddf6f5f3ae94fcaeb99912ecae7e0_10.png)

![](img/485ddf6f5f3ae94fcaeb99912ecae7e0_8.png)

3.  **结尾甩头发部分**：使用了一个“抖动”或“横纹故障”类的动感特效，强化了动作的冲击力和视觉节奏感。
    ![](img/485ddf6f5f3ae94fcaeb99912ecae7e0_12.png)

## 特效添加实操步骤 🛠️

![](img/485ddf6f5f3ae94fcaeb99912ecae7e0_10.png)

了解了特效的效果后，我们进入剪映App，看看如何一步步为视频添加这些特效。

![](img/485ddf6f5f3ae94fcaeb99912ecae7e0_12.png)

以下是添加特效的具体操作流程：

1.  **导入素材**：打开剪映，点击“开始创作”，选择你的视频素材导入项目。
    ![](img/485ddf6f5f3ae94fcaeb99912ecae7e0_3.png)

2.  **进入特效菜单**：在底部工具栏中找到并点击 **`特效`** 按钮。
    ![](img/485ddf6f5f3ae94fcaeb99912ecae7e0_5.png)

![](img/485ddf6f5f3ae94fcaeb99912ecae7e0_14.png)

3.  **选择并添加第一个特效（变清晰）**：
    *   在特效列表中，找到 **`基础`** 分类。
    *   选择 **`变清晰`** 特效。
    *   点击后，该特效就会添加到视频轨道上方。你可以拖动特效轨道两端的白框，调整其作用的时长和位置，使其与视频开场部分对齐。
    ![](img/485ddf6f5f3ae94fcaeb99912ecae7e0_14.png)

4.  **添加第二个特效（梦幻背景）**：
    *   点击 **`新增特效`** 按钮。
    *   在列表中找到 **`梦幻`** 分类，选择类似 **`妖气`** 的特效。
    *   将其添加到视频中，并调整时长，覆盖人物跳舞的片段。
    ![](img/485ddf6f5f3ae94fcaeb99912ecae7e0_16.png)

![](img/485ddf6f5f3ae94fcaeb99912ecae7e0_16.png)

5.  **添加第三个特效（动感抖动）**：
    *   再次点击 **`新增特效`**。
    *   进入 **`动感`** 分类，选择 **`横纹故障`** 或 **`抖动`** 特效。
    *   将其添加到视频结尾甩头发的部分，并调整好时长。
    ![](img/485ddf6f6f3ae94fcaeb99912ecae7e0_18.png)

6.  **预览与调整**：添加完成后，播放预览整体效果。你还可以为视频配上合适的背景音乐，使视听效果更加完整。

## 特效使用技巧与建议 💡

剪映的特效功能支持**叠加使用**，这意味着你可以在同一段视频上添加多个不同的特效，创造出更复杂的视觉效果。

以下是几个常用且效果出众的特效分类，建议在创作中多加尝试：

*   **基础类**：`开幕`、`闭幕`、`变清晰`、`变模糊`、`电影感`，常用于视频转场或风格化处理。
*   **动感类**：`抖动`、`放大`、`冲击波`，适合用于强调节奏点或制造视觉冲击。
*   **梦幻类**：`星云`、`花瓣`、`光斑`，能为视频增添浪漫或唯美的氛围。
*   **复古类**：`录像带`、`噪点`、`方格`，可以轻松营造怀旧或复古的胶片感。
