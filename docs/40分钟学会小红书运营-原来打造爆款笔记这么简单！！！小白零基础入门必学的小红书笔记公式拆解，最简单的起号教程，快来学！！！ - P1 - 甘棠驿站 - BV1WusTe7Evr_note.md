# 小红书运营：P1：爆款账号与笔记拆解教程 🔍

![](img/037171d46474682e832d5295e9bef81c_0.png)

在本节课中，我们将学习小红书账号与笔记的底层运营逻辑。通过拆解爆款属性规则、账号分析步骤和笔记分析步骤，你将理解如何从零开始打造有潜力的小红书账号。

## 爆款属性的基础规则 📊

上一节我们介绍了课程的整体结构，本节中我们来看看小红书平台的基础流量规则。了解这些规则是分析一切爆款账号和笔记的前提。

### 账号基础流量层级

小红书账号分为八个流量层级。每个新账号发布笔记时，都会获得一个基础的曝光量。

*   **基础曝光量**：新账号发布的首篇笔记，通常有 **800-1200** 的基础曝光量，取中间值可视为 **1000**。
*   **层级划分**：层级根据单篇笔记的“小眼睛”（浏览量）数量划分：
    *   第一层级：0-200
    *   第二层级：200-500
    *   第三层级：500-2000
    *   第四层级：2000-2万
    *   第五层级：2万-10万
    *   第六层级及以上：10万以上（通常需要官方合作或付费推广）

### 曝光增长机制（CES评分系统）

笔记发布后，系统会根据互动数据（CES评分）决定是否给予更多曝光。核心公式可以理解为：

**额外曝光量 = 基础曝光量 × 增长系数**

增长系数由以下互动率决定：

1.  **点击率（小眼睛数量）**：这是最核心的数据。当点击率超过 **11%** 时，笔记会进入流量池，获得额外曝光（约20%增长）。点击率越高（如13%、15%），获得的曝光加成比例也越高。
2.  **互动率（点赞、收藏、评论、转发、关注）**：这些互动行为会转化为“评分”，影响笔记在同层级内容中的**排名**。排名越高，获得的曝光展示就越多。

### 笔记效果的时间效应

笔记的流量推荐有有效期，这与能否突破层级紧密相关。

*   **初始有效期**：新笔记通常有 **3天** 的最佳流量期。流量分布大致为：第一天80%，第二天15%，第三天5%。
*   **层级突破与有效期延长**：如果在有效期内，笔记数据达到下一层级的标准（如从第一层突破到第二层），那么该笔记的推荐有效期会**延长**，并立即获得新层级对应的基础曝光量。
*   **自然流量上限**：对于普通账号，不借助付费推广，单篇笔记的自然流量上限通常在**第五层级**（即小眼睛数10万封顶）。计算公式为：
    `最大曝光量 ≈ 小眼睛数量 × 11（基础倍数）`

## 影响权重的隐藏属性 🔑

理解了基础层级和CES评分后，我们还需要关注几个不直接显示但至关重要的“隐藏属性”。

以下是四个关键的隐藏属性点：

*   **点赞率**：超过11%是突破基础层级的起点。不同类目要求不同，但**11%** 是一个基础参考值。笔记点赞量破千后，往往更容易获得系统推荐。
*   **互动率（评论）**：评论区的互动质量会影响曝光。当评论互动率超过 **5%** 时，会获得额外10%-15%的曝光加成。需注意，评论率高于30%可能被判定为异常数据。
*   **账号回复率**：这是隐藏的关键属性。当笔记评论数较少（如低于100条）时，账号**主动回复的比例应高于20%**。评论数越多，回复比例可以适当降低，但保持一定互动能提升权重。
*   **完播率（平均停留时长）**：用户观看笔记的时长。平均停留时长数据优于 **15%** 的同类型笔记时，会获得约5%-7%的额外曝光加成。这关联着用户的“收藏”行为。

**重要提示**：每一篇笔记的流量计算都是**独立**且从零开始的。即使账号已有爆款笔记，新笔记仍需依靠自身数据“闯关”。这保证了新账号也有打造爆款的机会。

## 爆款账号分解步骤 📱

掌握了核心规则，我们就可以开始实战分析。本节将手把手教你如何拆解一个成功的账号。

我们将从以下几个维度分析目标账号：

*   **个人信息**：包括头像、名称、背景图、简介、地址。这些元素共同构成了账号的“门面”，体现了专业度和定位。
*   **社交数据**：关注数、粉丝数、获赞与收藏数。关注列表往往透露了账号的“学习对象”或行业定位。
*   **内容数据**：笔记总数量、笔记的点赞/收藏比。高且均衡的赞藏比通常意味着内容粘性高。

**案例分析**：假设我们分析一个女装账号。
1.  **头像与背景**：使用专业Logo，背景是服装陈列架，显示其非个人小作坊，定位清晰。
2.  **名称与简介**：名称即品牌名，简介中注明“实体店”并提示“私信无法回复”，明确其**引流至线下或其他平台**的目的。
3.  **社交数据**：关注88人（且设置为不可见，推测是同行），粉丝1万。赞藏比为1.2万:7000，相对均衡，说明内容受喜爱程度和用户粘性不错。
4.  **内容数量**：发布70篇笔记，保持了一定的更新频率。

通过以上分析，我们可以判断这是一个**定位清晰、以线下引流为目的、内容质量较为稳定**的运营型账号。

## 爆款笔记分解步骤 📈

最后，我们来学习如何具体分析单篇爆款笔记，找到其成功的数据密码。

你需要跟踪目标笔记在一定周期内（如3-15天）的核心数据变化，并记录如下：

![](img/037171d46474682e832d5295e9bef81c_10.png)

`笔记热度 = f(点击率， 点赞率， 收藏率， 评论率， 时间)`

**操作步骤**：
1.  选定一篇近期爆款笔记。
2.  连续多日记录其**点赞、收藏、评论、估算曝光量（≈小眼睛数×11）**。
3.  计算每日的**点赞率、收藏率、评论率**。
4.  观察数据拐点：通常在发布后的**第1天**数据飙升，如果突破当前层级，则第2、3天会继续获得大量曝光。数据增长幅度（如每日是否保持5%-8%以上的增长）决定了笔记的“寿命”。

通过这种数据化的复盘，你可以清晰地看到：
*   笔记是在哪一天、因为哪个数据达标（如点击率突破11%）而进入下一流量池的。
*   互动率（赞、评、藏）如何影响其热度的持续。
*   从而反推出自己笔记的优化方向。

## 总结 🎯

![](img/037171d46474682e832d5295e9bef81c_12.png)

本节课中我们一起学习了小红书运营的核心基础：
1.  **规则是地基**：理解了 **流量层级**、**CES评分系统** 和 **时间效应**，就明白了流量从哪里来、如何增长。
2.  **数据是指南**：关注 **点击率、点赞率、互动率、完播率** 这些可见与隐藏的属性，用数据指导内容优化。
3.  **拆解是方法**：通过系统性地分析**爆款账号的定位**与**爆款笔记的数据轨迹**，将抽象的成功经验转化为可执行的运营动作。

掌握这些，你就拥有了分析小红书平台、诊断自身账号、规划内容方向的初步能力。记住，每一篇笔记都是新的开始，持续产出符合平台规则和用户喜好的内容，是成长的关键。
