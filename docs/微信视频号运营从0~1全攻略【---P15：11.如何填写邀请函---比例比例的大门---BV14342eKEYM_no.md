# 微信视频号运营从0到1全攻略：P15：11.如何填写邀请函 📝

在本节课中，我们将学习如何正确填写视频号创作者计划的邀请函。这是参与官方活动、获取流量扶持的关键一步，我们将一步步拆解每个填写项，确保信息准确无误。

![](img/14d52810337e27a7cc68c2d351cbf258_1.png)

## 概述
填写邀请函是加入视频号创作者计划、获取官方资源支持的必要流程。本节将详细讲解如何找到并填写账号昵称、粉丝量、邀请人、实名信息及手机号等核心信息。

![](img/14d52810337e27a7cc68c2d351cbf258_3.png)

## 查找账号信息
上一节我们介绍了如何注册视频号，本节中我们来看看如何找到自己的账号昵称和粉丝数量，这是填写邀请函的基础。

![](img/14d52810337e27a7cc68c2d351cbf258_4.png)

以下是查找账号信息的详细步骤：

1.  **打开微信**
    首先，在您的手机上找到并打开微信应用。

![](img/14d52810337e27a7cc68c2d351cbf258_6.png)

![](img/14d52810337e27a7cc68c2d351cbf258_7.png)

2.  **进入“发现”页**
    在微信底部导航栏，点击“发现”选项卡。

3.  **找到“视频号”入口**
    在“发现”页面中，向上滑动，找到并点击“视频号”入口（图标为黄色蝴蝶形状）。

    ![](img/14d52810337e27a7cc68c2d351cbf258_3.png)
    ![](img/14d52810337e27a7cc68c2d351cbf258_4.png)

4.  **进入个人主页**
    进入视频号页面后，点击右上角的个人头像图标，进入个人主页。

5.  **查看昵称与粉丝数**
    在个人主页中，向下滑动，找到“我的视频号”部分。这里会清晰显示您的**账号昵称**和**粉丝数量**。例如：`天道酬勤，39917个粉丝`。

    ![](img/14d52810337e27a7cc68c2d351cbf258_6.png)
    ![](img/14d52810337e7cc68c2d351cbf258_7.png)

## 填写邀请函各项信息
找到账号信息后，我们就可以开始填写邀请函了。邀请函通常包含以下几个必填项，请务必确保信息真实准确。

![](img/14d52810337e27a7cc68c2d351cbf258_9.png)

以下是需要填写的核心信息项：

![](img/14d52810337e27a7cc68c2d351cbf258_10.png)

![](img/14d52810337e27a7cc68c2d351cbf258_11.png)

*   **账号昵称与粉丝量**：填写上一步查看到的昵称和粉丝数。例如：`天道酬勤` 和 `39917`。
*   **活动邀请人**：根据您加入计划的渠道，填写对应的邀请人姓名。例如，通过本教程渠道可填写：`张良` 或 `张良老师`。
*   **实名信息**：此处必须填写您微信支付绑定的**真实姓名**。例如：`周建国`。
*   **结算手机号**：填写您本人常用的、能接收验证码的手机号码。格式为连续数字，例如：`13812345678`。

## 提交与确认
所有信息填写完毕后，最后一步是提交并确认。

1.  检查所有填写项无误后，点击页面底部的**蓝色“提交”按钮**。
2.  系统会弹出确认框，提示“确定要提交吗？”，请点击**“确认”**。

    ![](img/14d52810337e27a7cc68c2d351cbf258_9.png)
    ![](img/14d52810337e27a7cc68c2d351cbf258_10.png)
    ![](img/14d52810337e27a7cc68c2d351cbf258_11.png)

![](img/14d52810337e27a7cc68c2d351cbf258_13.png)

![](img/14d52810337e27a7cc68c2d351cbf258_14.png)

3.  提交成功后，页面会显示“**已成功加入创作者分成计划**”等类似提示，即表示操作完成。
