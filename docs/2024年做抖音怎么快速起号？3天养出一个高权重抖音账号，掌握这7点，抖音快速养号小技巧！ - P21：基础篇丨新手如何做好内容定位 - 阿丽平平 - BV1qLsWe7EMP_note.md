# 抖音运营入门：P21：新手如何做好内容定位 📝

在本节课中，我们将学习一个高效的内容选题公式，帮助新手创作者解决“不知道发什么”的难题，从而更精准地定位内容，吸引目标粉丝。

![](img/0191d6671ccf141f1f275190e8b03ded_1.png)

你现在做抖音时流的泪，都是当初确定选题时脑子里进的水。是否为了想选题想到头秃，不知道明天该发布什么内容？今天教你一套公式，一分钟帮你解决这个苦恼。

会找选题的人，能有更多时间用于视频制作。每天发布三条视频，一个月就是九十条。依靠堆量都能形成竞争力，根本不需要投入大量资金推广。而且，擅长选题的人能比你更精准地找到用户的痛点，也更容易吸引精准粉丝驻足观看，从而快速为账号打上标签。

![](img/0191d6671ccf141f1f275190e8b03ded_3.png)

那么，有什么办法可以解决选题困难的问题呢？

![](img/0191d6671ccf141f1f275190e8b03ded_5.png)

当然有。这是一套百万粉丝博主常用的选题技巧。为了方便理解，我将其总结成了一个公式：**内容定位 + 用户痛点 + 借势热点**。

用好这个公式，你的内容一定不会跑偏。例如，你想和我一样做一个知识博主，就需要去挖掘一些行业关键词，例如：播放量、粉丝量、定位、选题等。这些都是关键词，不包含任何主观描述。

然后，在这个基础上寻找用户的痛点，例如：便捷、高效、技巧、教学、内容输出等。最后，去蹭近期的热点，将它们进行任意组合即可。

接下来，我们把这个公式套用在其他行业里再试试。以美妆行业为例，内容定位可以是：美白、除皱、口红。用户痛点则是：平价好用、明星同款、避坑扫雷。最后，借势热点进行排列组合，例如：“迪丽热巴同款美白面膜推荐”、“七夕千万别送女朋友口红”等。

![](img/0191d6671ccf141f1f275190e8b03ded_7.png)

以上已经举了两个例子。至于举一反三，就交给你了。你可以根据自己行业的特性，将你的思考过程分享出来。

![](img/0191d6671ccf141f1f275190e8b03ded_9.png)

本节课中，我们一起学习了解决抖音内容选题难题的核心公式：**内容定位 + 用户痛点 + 借势热点**。掌握这个公式，可以帮助你系统化地生成内容创意，更精准地触达目标用户，从而高效地为账号打上标签，实现快速起号。
