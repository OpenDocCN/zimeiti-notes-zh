# 亚马逊跨境电商入门实操课：P12：如何注册店铺：资料准备 📝

![](img/0fd9fa5f9fd1ca5b6bd98d28e93d12f9_0.png)

在本节课中，我们将学习注册亚马逊店铺前需要准备的所有核心资料。了解并备齐这些资料是成功开店的第一步。

上一节我们介绍了如何选择亚马逊站点，本节中我们来看看注册店铺具体需要准备哪些材料。

## 资料清单总览

注册亚马逊店铺需要准备六项核心资料。以下是每一项资料的详细说明与注意事项。

### 1. 营业执照彩色扫描件

第一份需要准备的资料是**营业执照的彩色扫描件**（即清晰的营业执照照片）。这份执照可以是企业性质，也可以是个体工商户性质，在当地工商局即可免费办理。

*   **若无执照的解决方案**：可以借用家人或朋友的营业执照（例如开超市、服装店的执照）。如果无法借用，可以寻找代办机构办理，或联系推荐您学习的助理老师协助办理，费用通常在几百元至一千元左右。
*   **执照要求**：
    *   **注册资本与年限**：无硬性要求，无论是1元还是1000万，新办或旧执照均可。
    *   **经营范围**：需带有销售性质即可（如零售、批发）。**特别注意**：**切勿**在经营范围中添加“进出口贸易”字样，否则需额外办理进出口手续，属于画蛇添足。
    *   **经营范围与销售产品**：执照的经营范围与您计划在亚马逊销售的产品**无需一致**。例如，使用超市的执照注册的店铺，完全可以销售服装。

### 2. 法人身份证正反面照片

第二份资料是**营业执照上法人的身份证正反面照片**。

**核心关联性**：**营业执照与法人身份证必须是同一个人**。而后续其他资料可以与这两份资料的所有人不同。

### 3. 付款信用卡

第三份需要准备的资料是用于支付亚马逊平台月租的**付款信用卡**。

*   **唯一作用**：支付每月39.99美元的店铺月租。
*   **卡片类型**：必须是**全币种信用卡**，卡面需带有 **`Visa`** 或 **`Mastercard`** 标识。国内各大银行（如工商、建设、招商、中信等）均可办理。
*   **若无信用卡的解决方案**：可以借用他人的信用卡，或花费约100元购买一张虚拟信用卡，每月按时存入足额月租即可。实际上，店铺产生营业额后，亚马逊会优先从店铺余额扣款，信用卡作用有限。

### 4. 海外收款账户

![](img/0fd9fa5f9fd1ca5b6bd98d28e93d12f9_2.png)

![](img/0fd9fa5f9fd1ca5b6bd98d28e93d12f9_3.png)

第四份资料是**海外收款账户**，其功能类似于国际版的支付宝。

![](img/0fd9fa5f9fd1ca5b6bd98d28e93d12f9_4.png)

*   **核心作用**：用于接收并提现您在亚马逊店铺的销售所得。就像淘宝收入通过支付宝提现，亚马逊收入则通过此账户提现至您的国内银行卡。
*   **获取方式**：可通过服务商提供的链接免费注册，流程与注册支付宝类似，只需绑定本人身份证和银行卡，几分钟即可完成。

### 5. 电话号码

第五份资料是一个**中国大陆的手机号码**，用于接收验证码和官方通知。

### 6. 电子邮箱

第六份资料是一个**电子邮箱地址**，用于注册账号和接收重要邮件。推荐使用**163邮箱**，126邮箱、QQ邮箱等也可使用。

本节课中我们一起学习了注册亚马逊店铺必须准备的六项核心资料：营业执照、法人身份证、付款信用卡、海外收款账户、手机号和邮箱。请务必确保营业执照与法人身份证对应同一人，并注意避免在执照经营范围中添加“进出口贸易”。准备好这些材料后，您就可以着手开始正式的店铺注册流程了。
