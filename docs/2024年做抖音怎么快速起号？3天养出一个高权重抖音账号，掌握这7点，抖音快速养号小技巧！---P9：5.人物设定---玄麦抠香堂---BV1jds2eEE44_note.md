# 抖音运营指南：P9：人物设定 🎭

![](img/2ff7027dfe5d75cf12cf5d9884559e81_0.png)

在本节课中，我们将要学习抖音账号运营中一个至关重要的环节——**人物设定**（简称“人设”）。一个清晰、真实且富有吸引力的人设，是账号获取流量、建立信任并最终实现变现的核心基础。

![](img/2ff7027dfe5d75cf12cf5d9884559e81_2.png)

上一节我们介绍了账号定位，本节中我们来看看如何具体地构建账号的人格化形象。

![](img/2ff7027dfe5d75cf12cf5d9884559e81_4.png)

## 什么是人设？

![](img/2ff7027dfe5d75cf12cf5d9884559e81_6.png)

在抖音中策划人设，就像在游戏中创建角色一样，需要提前对人物角色进行设定。例如，角色的称呼、职业、拥有的技能点等。

但与游戏角色不同的是，我们在抖音中设定的人物角色是**人生角色**，因此需要更具**真实性**。

![](img/2ff7027dfe5d75cf12cf5d9884559e81_8.png)

## 人设的重要性

![](img/2ff7027dfe5d75cf12cf5d9884559e81_10.png)

无论是短视频还是直播，一个成功的IP能够解决粉丝信任的问题。它能够起到裂变传播的作用，同时也具备很强的变现能力。

![](img/2ff7027dfe5d75cf12cf5d9884559e81_12.png)

因此，我们运营账号，无论是创作内容还是策划选题，目的都是在获取流量的同时**打造IP**。打造IP的核心，就是在一定的场景或情境中，进行**账号人格化**的过程。

![](img/2ff7027dfe5d75cf12cf5d9884559e81_14.png)

这个过程需要注入情感、塑造故事、并贴上具有辨识度的标签。账号人格化，也就是我们通常所说的**账号人设**。

## 如何策划账号人设？

![](img/2ff7027dfe5d75cf12cf5d9884559e81_16.png)

对于账号人设的策划，我们可以从以下几个方面着手。以下是构建人设的核心要素列表：

![](img/2ff7027dfe5d75cf12cf5d9884559e81_18.png)

1.  **名字/昵称**
    一个IP首先得有一个叫法，这是IP传播的名片。要明确这个IP叫什么，就像一个品牌的品牌名或商标名一样重要。

2.  **职业/身份**
    人格化必然要具备身份。这是用户认识这个人或这个IP的联想入口。例如：老师、教授、博士、律师等。

3.  **称呼/社会关系**
    一个IP有了亲切的称呼，能快速拉近与用户的距离，起到情感共鸣的作用。例如：“XX哥”、“XX大爷”、“XX姥姥”。

4.  **形象/第一印象**
    以用户的视角来看待IP，使IP形象化，更容易占据用户心智。例如：大叔形象、萌萌哒、铁憨憨、甜美可爱、光头等。

5.  **性格/爱好/特点**
    一个具有鲜明特点的IP，除了具有很强的记忆点之外，还具有很强的真实性。过于完美的形象容易脱离大众。例如：暴躁、娘炮、爱吃、爱喝酒等。

6.  **穿着打扮/财富地位**
    这属于社会的刻板印象和大众的普遍认知。借助用户的固有思维，可以快速为IP在用户心智中定位。例如：有钱代表会赚钱，开豪车代表成功人士，打扮土气可能代表朴实。

以上内容策划好之后，就可以结合你自身的产品优势、服务优势等，将其落实到**账号搭建**中，即抖音账号的“五件套”：**头像、昵称、背景图、简介、视频封面**。

## 提炼人设的“超级符号” 🔑

除了对账号IP进行人格化的基础策划之外，还需要通过选题和内容策划，为你的人设提炼出独特的**记忆点**，即通常所说的“超级符号”。

超级符号的提炼可以从以下几个方面着手。以下是提炼超级符号的关键维度列表：

*   **画风（视觉）**：例如特定的**场景**、**道具**、**服饰**风格。
*   **听觉**：例如标志性的**BGM**、**特效音**、**配音**声线特点。
*   **文案**：能够反映优势且容易记忆的**提炼词**。例如：“简单粗暴”、“XX天花板”。
*   **表情/情绪**：例如**笑**、**惊恐**、**无奈**等标志性表情或情绪传递方式。
*   **语言特色**：例如**方言**、**口音**、**错发音**、爱用网络语、说话慢条斯理或抑扬顿挫等。
*   **动作/习惯**：例如**撩头发**、**扶眼镜**等习惯性的肢体动作。

**超级符号**提炼好之后，在你进行短视频创作时，就可以不断地在视频中重复呈现这些元素，从而强化用户记忆，形成独特的账号标签。

## 总结

本节课中我们一起学习了抖音账号**人物设定**的完整方法。我们首先理解了人设是赋予账号真实人格的过程，然后系统地学习了从**基础身份信息**到**外在形象性格**的六大策划要点，最后掌握了通过视觉、听觉、行为等多维度提炼**超级符号**，以强化人设记忆点的技巧。一个成功的人设，是连接内容、粉丝与变现的坚实桥梁。
