# 新媒体运营精品教程：P17：02.2：工具篇-AI生成PPT 🚀

在本节课中，我们将学习如何使用一款高效的AI工具来辅助生成PPT。这款工具能根据你提供的主题或现有文档，自动梳理大纲并生成详细的演示文稿内容，极大地提升内容创作效率。

上一节我们介绍了AI工具的基本应用场景，本节中我们来看看如何具体使用它来制作PPT。

## 工具核心功能

这款工具主要提供两种生成模式：

1.  **主题生成模式**：输入一个主题，AI将结合生成式技术（如ChatGPT）自动创作全文并生成PPT。
2.  **文档生成模式**：上传已有的文档（如TXT、PDF），AI会自动总结内容大纲，并基于此大纲生成详细的PPT。

以下是两种模式的核心逻辑描述：

*   **主题生成**：`PPT = AI_Generate(用户输入的主题)`
*   **文档生成**：`大纲 = AI_Summarize(上传的文档)` -> `PPT = AI_Generate(大纲)`
