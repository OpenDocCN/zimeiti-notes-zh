## 开店 🛒

上一节我们介绍了课程的整体框架，本节中我们来看看如何开设小红书店铺。

开店流程本身并不复杂。以下是具体步骤：

1.  打开小红书应用。
2.  点击左上角的菜单图标。
3.  选择“创作中心”。
4.  点击“更多服务”选项。
5.  选择“店铺”，点击“立即入住”。
6.  按照官方要求提交资料完成入住。

有人将小红书开店包装成付费项目出售，这十分奇特。下面介绍小红书店铺的两种主要形式：

*   **个人店**：分为两种。
    *   **个人店**：仅需身份证和1000元保证金即可开通。
    *   **个体工商户店**：需要身份证、营业执照和1000元保证金。
*   **企业店**：分为普通企业店、专卖店和旗舰店。均需要营业执照，并需缴纳600元年费。

![](img/d690b98f40be9aba4b5641689c0ffdce_7.png)

对于初学者，建议直接选择**个人店**。实测表明，平台并未对企业店有流量倾斜。唯一的区别是，个人店只能开一家，而一份营业执照可开三家店。如果一个店都做不好，开再多店也无用。

小红书电商不适合“店群”模式。这是因为小红书的电商自然流量占比很低，铺再多的店和货也用处不大。因此，任何鼓吹“小红书店群”的说法都不可信。小红书要做的是**账号矩阵**，而非店铺矩阵，这一点将在流量获取部分详细说明。

开店成本主要包括以下几项：

*   **保证金**：1000元（不做可退）。
*   **平台扣点**：营业额的5%。
*   **支付渠道费**：每单约0.7%。
*   **流动资金**：初期可采用无货源模式，准备2000-3000元完全足够。

## 选品 📦

成功开店后，下一步就是决定卖什么。选品其实很简单，核心逻辑是：**利用市场数据辅助选品，而非凭空想象**。我们要借助大卖家的选品能力来弥补自身不足。

以下是三个推荐的选品渠道：

1.  **某东、某宝**：寻找符合小红书平台调性的爆款产品。例如，有电商朋友将其淘宝爆款搬到小红书，第一篇笔记就出单，第二篇笔记单量破百。
2.  **某音精选联盟**：查看精选联盟排行榜，了解现阶段热卖商品。将符合小红书用户调性的产品直接搬运测试。
3.  **某多多销量排行榜**：选择适合小红书的类目排行榜，从中挑选爆品搬运到小红书。

整个选品过程只需判断一点：这些排行榜上的爆品，是否有可能匹配小红书用户群体。如果匹配，就搬过来测试。这相当于借助别人的能力为自己选品，是最快捷、最稳妥的方式，成功率远高于自己盲目测试。

## 流量获取 🚀

![](img/d690b98f40be9aba4b5641689c0ffdce_9.png)

产品上架后，就进入最重要的环节——获取流量。小红书是种草平台，店铺九成以上的流量都来自发布的笔记。这就是为什么之前强调不要做店群，而要做账号矩阵：尽可能多地发布笔记来获取流量。

流量通路是：用户通过笔记对产品产生兴趣，然后进入店铺完成购买。堆砌一堆店铺的效果，远不如多发几篇优质笔记。

因此，**如何在小红书上发布笔记，是电商成功的关键**。直接发布硬广效果很差，因为小红书用户认知较高，硬广无法获得信任，更谈不上转化。

发布笔记时，必须从**素人（用户）角度**切入，做种草内容，而非官方视角。这才是有效的方法。

那么，这种种草笔记怎么做呢？其实很简单：**去各大平台的商品评论里找素材**。

![](img/d690b98f40be9aba4b5641689c0ffdce_11.png)

具体操作如下：

![](img/d690b98f40be9aba4b5641689c0ffdce_13.png)

*   **素材来源**：某音、某宝、某东的商品买家秀。寻找好看的图片或视频素材直接使用。
*   **文案来源**：基于买家评价进行修改。这些评价是用户的真实使用体验，天然适合用作种草笔记，且取之不尽。

当你单个账号的标准化运营流程（SOP）跑通后，就可以开始进行**矩阵引流**：通过批量注册小红书账号、批量发布笔记，为店铺引流，放大收益。
