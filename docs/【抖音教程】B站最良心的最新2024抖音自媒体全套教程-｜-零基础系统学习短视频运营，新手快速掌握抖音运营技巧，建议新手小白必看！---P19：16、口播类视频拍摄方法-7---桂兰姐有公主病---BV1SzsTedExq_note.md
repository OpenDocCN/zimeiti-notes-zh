# 短视频运营教程：P19：口播类视频拍摄方法-7 🎬

在本节课中，我们将学习口播类视频的核心拍摄方法。我们将通过一个具体案例，详细拆解其构图、内容定位与拍摄技巧，帮助你掌握制作高质量口播视频的关键要素。

上一节我们介绍了视频拍摄的通用注意事项，本节中我们来看看如何将这些要点具体应用到口播类视频的拍摄中。

口播类视频的拍摄方法具有通用性。无论是金融、装修、养生还是情感口播，其拍摄方式都遵循相同的逻辑。同样，记录日常生活、拍摄段子或美食视频，其拍摄方法也基本一致。因此，掌握以下两种类型的拍摄方法，几乎可以覆盖所有短视频的拍摄场景。

首先，我们分析第一个类型：口播视频。这里以一位学员“种小草”的案例进行讲解。

![](img/5c102bc7b75683d6ff8ddd7e7c236734_1.png)

她的账号名为“种小草”，内容定位是情感口播。作为一名40岁带三个孩子的宝妈，她的一条视频播放量超过8000万，并借此涨粉30万。这条视频成功的关键在于精准吸引了目标群体。例如，如果你的内容瞄准20多岁的学生群体，并持续制作他们感兴趣的内容，那么涨粉速度就会很快，且粉丝非常精准。

接下来，我们具体拆解这条视频的拍摄细节。

**关于构图比例**
抖音平台采用 **9:16** 的竖屏格式。关键在于视频内容内部是选择横屏还是竖屏构图。案例中的视频采用了 **4:3** 的横屏构图。需要强调的是，无论选择哪种构图，都应保持统一。如果你决定用横屏，那么所有视频都应采用横屏；如果选择竖屏，则全部使用竖屏。保持一致性是重要的细节。

**关于人物主体**
人物应居于画面中央，约占画面的 **70%** 左右。避免将脸部怼满屏幕拍摄，这会影响观感。应在人物的左、右、上方适当留白，使画面看起来清晰、舒适。

**关于出镜形象**
如果你对自己的外貌不够自信，是否还能做口播？答案是肯定的。抖音的成功并非完全依赖颜值。口播的核心在于传递内容和知识。只要出镜形象清晰、体面即可。可以利用美颜滤镜进行适当修饰，但注意不要过度夸张。

现在，我们一起观看这条涨粉30万的视频片段。观看时请思考：这条视频主要想吸引哪个特定群体？

![](img/5c102bc7b75683d6ff8ddd7e7c236734_3.png)

![](img/5c102bc7b75683d6ff8ddd7e7c236734_3.png)

视频文案示例：
> “跟婆婆在一起住是什么样的感受？我跟我婆婆在一起住了16年了……我最大的感受就是这十多年以来，我娘家人很少在我家住上几天……这对于我内心来说其实是一个很大的缺失。”

有同学可能认为是关于“婆媳关系”。但这具体指向哪个群体呢？实际上，男性对婆媳关系的感触通常不如女性深刻。而未结婚的年轻女性也难以真正产生共鸣。因此，这条视频最能打动的是**已婚女性群体**，她们对婆媳关系有切身体会，容易与视频内容产生共鸣。我们后续可以通过数据分析来验证这一点。

从画面中可以看到，主讲人使用了美颜效果。如果你对自己的外貌不自信，可以适当开启美颜功能，让自己看起来更精神、体面即可。其构图也保持了 **4:3** 的横屏比例。

有同学可能会问：“老师，口播视频必须在户外拍吗？可以在家拍吗？” 答案是：可以在任何地方拍摄。关键在于背景要整洁、美观，避免杂乱难看的背景。

![](img/5c102bc7b75683d6ff8ddd7e7c236734_5.png)

另一个常见问题是：“做口播需要背下所有台词吗？我记不住怎么办？” 解决方法在剪辑环节。你可以借助提词器工具，实现 **边看边读边录**，这大大降低了拍摄难度。

![](img/5c102bc7b75683d6ff8ddd7e7c236734_5.png)

**关于收音清晰度**
口播视频中，声音清晰至关重要。为此，推荐你购买一个**手机收声麦克风**。这种设备价格便宜，只需十几元，在淘宝、京东等平台都能买到，能显著提升音频质量。
