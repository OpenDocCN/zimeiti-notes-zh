## ❓ 人设强度判断
现在，考验大家的时候到了。请根据上述文案判断：这是一个**强人设**视频，还是**弱人设**视频？

答案是：**弱人设**。

判断依据如下：
1.  内容与讲述者本人**无关**。视频虽然讨论了“穷人”和“富人”，但并未涉及博主自身的经历、身份或观点。
2.  形式与内容**分离**。视频画面是治愈系的手工皂制作过程，而音频是关于阶层跨越的观点论述，两者没有直接关联。

## ✨ 视频火爆的原因：“双向吸引”模型
那么，为什么这样一个“弱人设”视频能够火爆？

关键在于它运用了 **“双向吸引”** 策略。

**公式：视频吸引力 = 画面吸引力 + 内容吸引力**

这个视频同时满足了两种不同需求的用户：
*   **A类用户**：被**画面**吸引。觉得手工皂制作过程解压、治愈，能吸引他们停留观看。
*   **B类用户**：被**内容**吸引。认同或思考视频中关于“原生家庭”、“朋友圈层”、“风险意识”的观点，产生共鸣，从而愿意听完整个音频。

两类用户被不同的元素吸引，但共同贡献了**完播率**和**互动数据**，从而推高了视频的整体流量。

## 💡 “双向吸引”策略的应用
理解了核心原理后，我们可以将其应用到自己的视频创作中。以下是具体的应用思路：

如果你觉得自己的主画面吸引力不足，可以尝试为其搭配一个能吸引另一类人群的“背景音”或文案。

*   **宝妈账号**：拍摄宝宝日常或做饭画面，可能受众较窄。
    *   **应用策略**：可以为画面配上有趣的育儿故事、家庭搞笑段子或情感共鸣文案。
*   **实体店/工厂账号**：单纯拍摄产品、车间，内容枯燥。
    *   **应用策略**：可以配上创业故事、行业干货、从业经历分享或个人人生感悟。

![](img/3a605302ff20aa45c270ca94fc2263a1_2.png)

**核心代码逻辑（比喻）：**
```python
if 主画面吸引力弱:
    搭配强吸引力的音频/文案 # 如故事、干货、共鸣观点
else:
    强化主画面优势
```
通过这种“**画面与声音各司其职**”的方式，你就能覆盖更广的受众，提升视频的完播率。

## 📝 本节总结
本节课我们一起分析了一个弱人设却高流量的案例。我们学习了：
1.  **判断人设强弱**的关键：内容是否与博主自身强相关。
2.  **“双向吸引”模型**：通过“画面”和“内容”分别吸引不同用户，是提升视频数据的有效技巧。
3.  **策略应用**：当你觉得视频主体吸引力单一时，可以尝试为其注入故事性、知识性或共鸣性的音频/文案，实现“双向吸引”。

很多视频数据不佳，并非内容全无价值，而仅仅是缺少了一些包装和吸引用户的技巧。抖音运营本身就是一个需要不断学习和测试的行业。
