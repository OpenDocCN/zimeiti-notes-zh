# 抖音运营教程：P16：如何衡量选题

在本节课中，我们将学习如何判断一个抖音选题是否优秀。我们将通过一个具体的例子，引入五个核心的衡量标准，帮助你从众多选题中筛选出最具潜力的那一个。

上一节我们介绍了如何寻找和生成选题，本节中我们来看看如何判断这些选题的质量。

我们如何衡量一个选题是不是好选题？这里给出十个选题作为例子，请大家思考哪个选题给你的印象最深刻。

以下是十个示例选题：
*   简单家庭减肥操
*   订婚这些流程你知道吗
*   玩手机躺着也能减肥
*   12种女生干练穿搭方法
*   六个小办法，甩掉高原红
*   厨房收纳七件神器
*   新手必备的化妆全流程
*   适合婚礼的50首歌曲
*   国庆旅游好物清单
*   摩洛哥旅行攻略

对于大多数用户而言，“玩手机躺着也能减肥”这个选题往往印象最深刻。这是为什么呢？因为用户普遍倾向于选择“低成本”的方案。这其实就是衡量好选题的第一个核心标准。

## 衡量选题的五个标准

基于用户的选择逻辑，我们可以总结出五个衡量选题好坏的标准。以下是这五个关键维度：

1.  **低成本**：指用户达成目标所需付出的代价（如时间、精力、金钱）很低。公式可表示为：**用户获得感 / 用户付出成本 > 1**。例如，“三个方法变白”就比“一百个方法变白”更具吸引力。
2.  **痛点**：指精准击中用户最迫切、最强烈的需求或烦恼。例如，用户已经认定某款口红好看，但觉得价格太贵，那么“优惠券”或“折扣”信息就直接击中了其“价格”痛点。
3.  **熟悉**：选题基于用户熟悉的场景或共同记忆，容易引发共鸣。例如，“你有没有在电影院里碰到过这种人？”、“放假回家妈妈常说的几句话”。
4.  **精准**：选题内容明确指向特定人群的特定需求。
5.  **高频**：选题涉及的需求或场景在用户生活中出现频率很高。

在实际操作中，我们最常用到的是 **低成本**、**痛点** 和 **熟悉** 这三个标准。

## 如何应用这些标准

了解标准后，我们来看看如何具体应用。当你产生一个选题想法时，可以将其代入这五个维度进行检验。

基本上，一个选题如果能符合其中的两到三点，就可以被认为是一个合格的、有潜力的好选题。我们的工作流程通常是：先寻找选题方向，然后通过“九宫格”等方法扩展出具体选题，最后再用这些标准进行衡量和筛选，决定最终拍摄哪一个。

当然，并非每个选题都能完美符合所有标准，筛选和删减是必然的过程。如果有同学觉得自己的所有选题都不够好，我们还可以根据文案脚本进行进一步的优化和调整。

本节课中我们一起学习了衡量抖音选题质量的五个核心标准：**低成本**、**痛点**、**熟悉**、**精准**和**高频**。重点掌握前三个标准，并学会将你的选题代入这些维度进行检验，就能有效地筛选出优质选题，为制作爆款视频打下坚实基础。
