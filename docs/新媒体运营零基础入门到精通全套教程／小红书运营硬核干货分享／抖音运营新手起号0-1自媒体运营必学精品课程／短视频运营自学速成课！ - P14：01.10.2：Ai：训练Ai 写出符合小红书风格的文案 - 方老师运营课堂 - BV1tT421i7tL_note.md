# AI小红书运营教程：P14：01.10.2：训练AI写出符合平台风格的文案 ✍️🤖

## 概述
在本节课中，我们将学习如何训练AI（如ChatGPT）来撰写符合小红书平台风格的文案。上一节我们介绍了如何让AI生成爆款标题，本节我们将重点探讨如何让AI理解并创作出符合平台调性和爆款结构的内容正文。

## 核心思路：让AI学习平台风格
大部分AI能够生成内容，但关键在于如何让它写出符合特定平台风格的内容。要实现这一点，我们需要让AI学习该平台的文案风格、常用词汇和语气。

以下是训练AI的核心步骤：

![](img/d926db3067c3ae20c2ab144daffe026f_15.png)

![](img/d926db3067c3ae20c2ab144daffe026f_17.png)

![](img/d926db3067c3ae20c2ab144daffe026f_19.png)

1.  **提供学习指令**：首先，需要明确告知AI你的目标。例如，你可以输入：“接下来我会给你提供一些小红书上的爆款笔记文案，你需要根据这些文案分析小红书的文案撰写风格、特定的平台用语、词汇、语气等。”
2.  **投喂范例材料**：向AI提供多篇具有代表性的小红书爆款文案。这些材料应能体现平台的独特风格，例如第一人称的亲切分享、少女感的活泼用语（如“宝子”）、特定的平台文化词汇等。
3.  **要求反馈与分析**：在每次提供文案后，要求AI确认已理解，并给出对该文案风格、语气、类型等方面的分析。这能确保AI确实在学习，并为你提供正向反馈。
4.  **基于学习成果创作**：当AI学习了足够多的范例后，即可指令它结合所学知识，针对特定主题创作一篇新的、具有平台风格的文案。

```
接下来我会给你提供一些小红书上面的爆款笔记文案，你需要根据这些文案分析小红书平台的文案撰写风格、特定的平台用语、词汇、语气等。每次接收文案后，请告诉我你已理解，并给出你对这篇文案的风格、语气、类型等相关分析。

```

## 效果评估与优化
AI生成的初稿可能不尽如人意。例如，它可能只列出了景点名称，而缺乏深度分析、路线规划或避坑建议，显得空洞。

![](img/d926db3067c3ae20c2ab144daffe026f_79.png)

![](img/d926db3067c3ae20c2ab144daffe026f_81.png)

![](img/d926db3067c3ae20c2ab144daffe026f_83.png)

这通常由以下原因导致：
1.  **投喂的材料质量或代表性不足**：如果提供的范例本身深度不够，AI的学习效果就会受限。
2.  **AI模型的能力限制**：免费的GPT-3.5模型在复杂任务上的表现通常不如付费的GPT-4.0。
3.  **指令可以更精确**：对于旅游攻略类内容，可以分步指令AI。例如，先让它列出张家界所有重要景点及特点，再针对每一天的行程规划详细路线、亮点和注意事项。

![](img/d926db3067c3ae20c2ab144daffe026f_85.png)

![](img/d926db3067c3ae20c2ab144daffe026f_87.png)

![](img/d926db3067c3ae20c2ab144daffe026f_89.png)

![](img/d926db3067c3ae20c2ab144daffe026f_91.png)

**优化建议**：
*   **分步引导**：对于复杂内容，不要期望AI一步到位。可以指令它先搭建框架，再丰富细节。
*   **结合专项指令**：在创作时，可以再次结合上节课的“爆款标题指令库”，让生成的标题更精准。
*   **开启新对话**：当需要AI执行特定、独立的指令（如专门生成标题）时，建议开启一个新的对话窗口，以避免之前对话上下文的干扰。
*   **持续迭代**：AI训练是一个持续的过程。根据生成结果，你可以继续投喂更优质的范例，或调整你的指令，直到获得满意输出。

## 总结
本节课我们一起学习了如何训练AI写出符合小红书风格的文案。核心在于通过**提供明确指令**和**投喂高质量范例**，让AI学习平台的独特行文风格、用语和语气。关键在于，我们要让AI成为理解我们领域要求和平台调性的“专家”，而不仅仅是泛泛的知识库。通过持续的“指令-反馈-优化”循环，我们可以让AI成为创作符合平台标准内容的得力助手。
