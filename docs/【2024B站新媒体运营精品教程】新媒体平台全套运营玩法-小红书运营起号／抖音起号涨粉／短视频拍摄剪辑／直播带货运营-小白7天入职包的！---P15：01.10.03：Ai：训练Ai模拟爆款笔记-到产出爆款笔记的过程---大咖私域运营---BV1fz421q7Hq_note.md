## 分析爆款笔记的核心要点

![](img/2876e58456680d24874ecf164b9b745a_11.png)

要让AI有效学习，我们需要引导它分析爆款内容的几个关键维度。以下是需要AI重点关注的几个方面：

![](img/2876e58456680d24874ecf164b9b745a_13.png)

![](img/2876e58456680d24874ecf164b9b745a_15.png)

![](img/2876e58456680d24874ecf164b9b745a_17.png)

1.  **爆款因素分析**：分析内容为何能成为爆款，例如是否引发了情感共鸣、提供了实用价值或具有独特性。
2.  **内容结构拆解**：拆解文案的叙事结构。
    *   **开篇**：用什么方式（如提问、痛点描述、名人引言）吸引用户继续阅读。
    *   **中间**：提供了哪些具体内容来满足用户需求、解决痛点，以提升用户停留时长。
    *   **结尾**：如何设置“钩子”（如鼓励互动、提供联系方式）引导用户私信或关注，形成互动。
3.  **关键词策略分析**：分析笔记中是否刻意植入了特定关键词。
    *   如果有，关键词是什么？
    *   关键词的出现频率和分布情况如何？
    *   是如何结合中文分词法进行设计的？

## 实战演练：引导AI分析爆款笔记

![](img/2876e58456680d24874ecf164b9b745a_25.png)

**第一步：向AI下达清晰的指令**
我们需要给AI一个明确的分析框架。指令模板如下：
```
我给你提供一些小红书平台上关于[你的领域，如：考研]的爆款笔记文案，你需要分析：
1. 文案的爆款因素。
2. 文案的整体结构（开篇如何吸引用户，中间如何戳中痛点，结尾如何引导互动）。
3. 分析笔记是否设计了关键词。如果有，请指出关键词、出现频率和铺设方式；如果没有，请说明。
```
**第二步：提供爆款文案素材**
从目标平台（如小红书）上寻找点赞数据高的爆款笔记，将文案内容复制给AI。建议提供3-5篇不同风格的笔记，让AI学习更全面。

![](img/2876e58456680d24874ecf164b9b745a_40.png)

![](img/2876e58456680d24874ecf164b9b745a_42.png)

![](img/2876e58456680d24874ecf164b9b745a_44.png)

![](img/2876e58456680d24874ecf164b9b745a_46.png)

**第三步：获取并整合AI的分析结果**
AI会对每篇笔记给出分析。之后，我们需要让它对所有分析进行总结和去重，提炼出该领域爆款笔记的共性因素。
可以使用的指令是：
```
请将上面分析过的几篇爆款笔记的爆款因素总结一下，整合成一个清晰的列表。
```
这样，我们就得到了一份该领域的“爆款元素清单”，例如：情感共鸣、逆袭故事、明确的时间规划、具体的资源建议、正能量鼓励等。

## 从分析到产出：指导AI生成爆款文案

在AI学习了爆款模式后，我们就可以指导它进行创作。

![](img/2876e58456680d24874ecf164b9b745a_48.png)

**第一步：生成内容大纲**
首先，让AI根据总结的爆款因素，规划一篇新笔记的大纲。指令如下：
```
请结合你前面分析的爆款笔记因素，为我规划一篇关于[你的领域]的小红书笔记大纲。大纲需要细分到每个部分的具体要点。
```
AI会生成一个包含开篇、主体、结尾等部分的结构化大纲。

**第二步：根据大纲撰写完整文案**
在大纲确认无误后，指令AI生成完整文案，并再次强调爆款要素和格式要求。
```
请按照上面的大纲，撰写完整的笔记文案。要求：
1. 结构严格遵循爆款笔记的要素（如开篇吸引、痛点切入、结尾引导互动）。
2. 融入一个简单的“逆袭故事”来引发读者共鸣。
3. 使用适量的emoji表情。
4. 尝试以自然的方式融入关键词，例如：“考研规划”、“政治备考”。
```
**第三步：优化与调整**
生成的初稿可能不够完美。我们可以根据需求，要求AI对特定部分进行修改、扩写或精简，例如：“将开篇的故事写得更详细一些”或“在资源推荐部分加入更具体的书单”。

## 进阶技巧：创建专属AI助手

为了获得更稳定、更符合个人风格的结果，可以利用高级AI工具（如ChatGPT的GPTs功能）创建专属助手。
*   **上传学习资料**：将整理好的平台风格指南、爆款案例、常用表情包合集等文件上传给AI助手，让它进行深度学习。
*   **设定固定指令**：在助手配置中预设好分析框架和文案风格要求，避免每次对话都需要重复指令，提高效率。

## 课程总结

![](img/2876e58456680d24874ecf164b9b745a_54.png)

本节课我们一起学习了利用AI辅助内容创作的全流程：
1.  **训练阶段**：通过提供爆款笔记和清晰指令，引导AI分析出内容的爆款因素、结构特点和关键词策略。
2.  **产出阶段**：基于AI的分析总结，先让它生成符合爆款逻辑的内容大纲，再据此撰写完整文案，并可反复优化。
3.  **效率提升**：通过创建专属AI助手，固化工作流程，实现更高效、更稳定的内容产出。
