# 抖音短视频运营教程：P40：14、学员分享-5 💬

在本节课中，我们将通过一位学员的真实分享，了解普通人如何通过系统学习和实践，在抖音上成功起号并获得收入。本节内容将包含学员的成长经历、关键学习点以及老师对后续教学环节的预告。

我们首先来看一位名叫“海椒”的学员案例。他拥有122万粉丝。大家可以先听一下他的普通话和语音面貌，并猜猜他一个月的收入大概是多少。

![](img/19dba8e9831df8350ea2e383ed85a868_1.png)

以下是学员梁海娇（海椒）的分享原文：

“Hello，微课的教育老师们，你们好，我是你们的学员，我叫梁海娇。我是去年8月21号，通过小飞老师的介绍，后来去听了阿果老师的公开课。但是我没有听阿国老师的公开课之前，我有去了解别的教育的老师的公开课。我觉得他们老其他老师讲的课程都没有阿狗老师讲的比较动心。听到阿果老师的公开课以后，知道抖音的前程和逆势是非常好的。后来我就抓住了这个机会，交了学费。

9月份我正式上课，一边学习一边发布作品。我非常感谢阿果老师的指导。前期我不懂得如何定位，他很有耐心地给我定位，在后期变现方面，也很有耐心地给我指导和解答。

我的解答老师是加菲老师，他非常有耐心地解答我的任何一个问题。特别是前期，我对这方面一点都不懂。我的文化只有小学六年级，拍视频、剪辑视频都是零基础。

后面通过老师们的课程，基础班是八一老师讲的课，他讲得非常好，非常详细，任何一个点都会讲到。其他学员也说八一老师的课讲得非常好。后面我也听过晓峰老师的课程。通过万国老师的指导、八一老师的课程、加菲老师的解答，还有平时和小飞老师谈心，我的账号做起来了。

目前粉丝给我带来了一份经济收入。对于一个全职宝妈来说，这份收入已经很满足了。感谢大家，也感谢老师的指导。新的学员们，如果你们看到我的视频，你们也要努力，多去操作，坚持更新和发布作品。微课教育的老师们都很好，很负责任，很有耐心。希望你们也一样能把账号做起来。”

感谢大家，拜拜。

看完海椒的视频，主要目的有两个：第一，让大家了解他的基础条件；第二，让大家对他的收入有一个直观感受。稍后我们会在拍摄环节，直接通过官方平台数据来揭秘他的月收入。抖音是一个看重方法和执行的行业，与年龄、学历关系不大。

接下来，我们看看课程中关于“拍段子”这个方向的教学。如果你想做对口型、搞笑剧情、沙雕段子或顾客趣闻这类内容，可以在公屏上扣“1”。

![](img/19dba8e9831df8350ea2e383ed85a868_9.png)

关于拍段子，这里有一个真实学员案例。他是一名在工地辛苦工作的“搬砖”工人。学习之前，他自己拍的搞笑视频播放量只有两三千。

![](img/19dba8e9831df8350ea2e383ed85a868_9.png)

![](img/19dba8e9831df8350ea2e383ed85a868_11.png)

经过系统学习后，他给老师的反馈是：“播放量快要破500万了，评论涨粉还在涨，我不知道该怎么回了。” 希望大家未来也能体验到这种播放量暴涨的心情。

拍段子本身并不难，但需要你具备一定的表现力，不能过于腼腆害羞。核心目的是让观众觉得有趣、好笑。对于不知道拍什么的同学，完全不用担心。

![](img/19dba8e9831df8350ea2e383ed85a868_11.png)

以下是课程提供的支持：
*   **素材提供**：老师会提供段子素材。
*   **脚本提供**：老师会提供拍摄用的详细脚本。
