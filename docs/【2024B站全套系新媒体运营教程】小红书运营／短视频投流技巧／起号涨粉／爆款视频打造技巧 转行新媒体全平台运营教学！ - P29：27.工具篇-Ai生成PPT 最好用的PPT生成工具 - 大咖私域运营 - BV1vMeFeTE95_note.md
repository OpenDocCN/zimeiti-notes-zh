# 新媒体运营教程：P29：27.工具篇-AI生成PPT 🚀

## 概述
在本节课中，我们将学习一款非常实用的AI生成PPT工具。这款工具能够根据你提供的主题或现有文档，自动梳理内容大纲，并基于大纲生成详细的PPT内容。我们将通过实际操作演示，帮助你快速掌握这个提升工作效率的利器。

## 工具核心功能与模式
上一节我们概述了工具的基本概念，本节中我们来看看它的具体工作模式。这款AI工具主要提供两种生成模式，以满足不同场景下的需求。

![](img/73271a4717e8a5d6f33146d1df68fab3_1.png)

以下是两种核心工作模式：
1.  **主题生成模式**：输入一个主题，工具会结合类似ChatGPT的生成式AI能力，自动创作PPT全文。
2.  **文档生成模式**：上传一篇已有的文档（如TXT、PDF），AI助手会自动总结文档大纲，并基于此大纲生成PPT内容。

![](img/73271a4717e8a5d6f33146d1df68fab3_3.png)

## 实战演示：基于文档生成PPT
了解了工具的模式后，我们通过一个实际案例来演示其工作流程。本次演示将使用一篇关于新媒体运营的既有文章来生成PPT。

![](img/73271a4717e8a5d6f33146d1df68fab3_5.png)

1.  **准备与上传文档**：首先，准备一篇需要转换为PPT的文档。在工具界面点击“继续”，然后上传文件。
    ![](img/73271a4717e8a5d6f33146d1df68fab3_1.png)

![](img/73271a4717e8a5d6f33146d1df68fab3_7.png)

2.  **AI解读与生成大纲**：工具会解读上传的文档，并在下方列出其分析出的内容大纲。例如，针对新媒体运营文章，可能生成“新媒体运营的本质”、“平台运营的角度”等章节。
    ![](img/73271a4717e8a5d6f33146d1df68fab3_3.png)

![](img/73271a4717e8a5d6f33146d1df68fab3_9.png)

3.  **开始制作PPT**：大纲确认后，点击“开始制作”，工具将进入PPT生成页面。生成速度通常很快，具体取决于网络环境和平台状态。
    ![](img/73271a4717e8a5d6f33146d1df68fab3_5.png)

![](img/73271a4717e8a5d6f33146d1df68fab3_11.png)

4.  **内容生成与预览**：稍等片刻，工具会基于对文档的理解和大纲，自动生成PPT的详细内容。生成完毕后即可预览。

## 工具的优势与编辑调整
生成初步PPT后，我们来看看这个工具的核心优势以及如何进行后续调整。该工具最突出的优点是能基于大纲节点自动填充详细内容，而不仅仅是格式化现有文本。

以下是工具的主要优势和编辑功能：

*   **智能内容填充**：工具能分析你的内容，并将其归类到不同的大纲节点下，自动生成对应的详细阐述。这比仅仅将已有全文排版成幻灯片格式更有价值。
*   **模板更换**：生成后，可以通过右侧的“更换模板”功能快速切换PPT的视觉风格。虽然当前模板库选择有限，但切换速度非常快。
    ![](img/73271a4717e8a5d6f33146d1df68fab3_15.png)
*   **内容编辑**：在工具右侧面板，可以对每一页的文字、形状等进行直接修改和替换。
*   **本地化调整**：你可以将生成的PPT下载到本地，使用PowerPoint或Keynote等软件进行更深入的个性化调整。

![](img/73271a4717e8a5d6f33146d1df68fab3_15.png)
