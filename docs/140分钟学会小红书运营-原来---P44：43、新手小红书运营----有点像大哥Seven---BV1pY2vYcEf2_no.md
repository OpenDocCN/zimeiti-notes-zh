# 小红书运营：P44：43、小红书店铺规划（2） 📈

![](img/9d5b77aa34b74f1b5ada98ef80d41b0a_1.png)

在本节课中，我们将要学习小红书店铺运营的中期与后期规划。我们将拆解中期规划中的核心排名机制，并了解后期发展的两个关键方向。

![](img/9d5b77aa34b74f1b5ada98ef80d41b0a_3.png)

上一节我们介绍了店铺运营的前期规划，本节中我们来看看店铺进入稳定期后的中期规划应如何操作。

![](img/9d5b77aa34b74f1b5ada98ef80d41b0a_5.png)

中期规划的重点内容非常多，其核心在于提升店铺在各种排名系统中的位置，以获得更多平台内的自然流量。如果前期的基础数据没有做好，中期规划的推进将会比较困难。我们先来了解一下中期规划包含哪些主要内容。

![](img/9d5b77aa34b74f1b5ada98ef80d41b0a_7.png)

中期规划主要涉及以下几类排名和考量因素：
*   **综合排名分类**
*   **价格排名**
*   **销量排名**
*   **关键词排名**
*   **个性化搜索**
*   **动态PK与静态PK**

![](img/9d5b77aa34b74f1b5ada98ef80d41b0a_9.png)

这些内容都与店铺的整体表现息息相关。由于内容体系庞大，本节将为大家进行概括性讲解。

## 综合排名分类 🏆

综合排名是平台默认的商品排序方式，它根据店铺的综合因素对商品进行排名。这个排名考核的维度非常广泛。

以下是影响综合排名的16项关键因素（当前为15项）：
1.  商品人气
2.  商品销量
3.  店铺信誉
4.  商品价格
5.  新店与新品扶持
6.  商家层级优先级
7.  商品下架时间
8.  标题优化
9.  转化率
10. 收藏率
11. 回头客比例
12. 好评率
13. 交易数量

**公式：综合排名 = f(人气, 销量, 信誉, 价格, 扶持政策, ...)**

![](img/9d5b77aa34b74f1b5ada98ef80d41b0a_11.png)

在这16项（实为15项）排名因素中表现越优秀，店铺的整体排名就越高。排名越高，意味着你能从平台那20%的确定性流量池中抢到更多曝光的机会。如果做不好，店铺的整体数据增长就会比较困难。

![](img/9d5b77aa34b74f1b5ada98ef80d41b0a_13.png)

## 价格排名与销量排名 💰

接下来，我们看看另外两个重要的排序维度：价格排名和销量排名。

![](img/9d5b77aa34b74f1b5ada98ef80d41b0a_15.png)

### 价格排名

![](img/9d5b77aa34b74f1b5ada98ef80d41b0a_17.png)

价格排名是根据商品的价格进行排序的。通常，运费不计算在内。对于价格相同的商品，其排名会受到**宝贝关键词匹配度**和**商品类目**的影响。

![](img/9d5b77aa34b74f1b5ada98ef80d41b0a_19.png)

**代码逻辑示意：**
```
if 商品A.价格 == 商品B.价格:
    排名 = 比较(关键词匹配度， 类目相关性)
else:
    排名 = 比较(价格)
```

平台会通过以下几个点来最终确定价格排名系统：
*   **价格竞争力**：你的定价是否有优势。
*   **价格真实性**：是否存在虚标价格再打折的行为。
*   **价格稳定性**：价格是否频繁剧烈波动。
*   **价格与销量的平衡**：是否做到“物有所值”。

如果这几个点把握不好，商品的排名就可能不理想。

### 销量排名

销量排名依据商品的销售数量进行排序。买家经常使用这个排序方式，因为它能在一定程度上反映商品的人气。

需要注意的是，销量高但排名靠后的商品，可能是由于**类目不精准**导致的。影响销量排名的因素还包括：
*   **销量趋势**：销量是持续增长还是下滑。
*   **类目相关性**：商品是否被正确归类。
*   **转化率**：流量转化为购买的比例。
*   **商品评价**：用户反馈的好坏。

## 其他核心机制 🔑

![](img/9d5b77aa34b74f1b5ada98ef80d41b0a_21.png)

除了上述排名，中期规划还需理解关键词排名、个性化搜索以及PK机制。

![](img/9d5b77aa34b74f1b5ada98ef80d41b0a_23.png)

### 关键词排名

![](img/9d5b77aa34b74f1b5ada98ef80d41b0a_25.png)

关键词排名非常细致。**长尾词**和**主关键词**的排名是分开计算的。例如，搜索“T恤”、“男士T恤”、“女士T恤”是三个不同的排名赛道。

**核心逻辑**：平台为每个关键词下的同类商品建立排名列表。你的商品在该关键词下的排名位置，决定了你能获得多少来自该关键词搜索的曝光。

### 个性化搜索

个性化搜索是平台根据**用户属性**（如浏览历史、喜好）和**店铺/商品属性**进行匹配推荐的过程。个性化搜索做得越好，吸引到的用户就越精准，后续的转化数据也会越好。

### 动态PK与静态PK

关于动态PK与静态PK，本节暂不深入展开。这两项机制通常在店铺运营达到较高层级（如第四、第五能量层级）时效果更为显著。在前期可以操作，但对整体数据的提升作用有限，且后期数据较难把控，大家可根据自身情况判断。

由于中期规划内容极为庞杂，每一个点都需要深入研究和把控，后续会考虑开设专题课程进行详细讲解。

![](img/9d5b77aa34b74f1b5ada98ef80d41b0a_27.png)

## 后期发展规划 🚀

完成了中期规划的各项优化后，便进入后期发展阶段。后期规划主要围绕两个决策节点展开。

后期发展的核心思路如下：

**节点一：加大力度投入**
如果你认为店铺仍有提升空间，并希望从中层用户向大用户发展，可以选择加大投入。这需要从以下五个方面进行：
1.  **数据分析**：深入分析各项运营数据。
2.  **店铺分析**：诊断店铺整体健康状况。
3.  **数据掌控**：建立数据监控与预警体系。
4.  **店铺优化**：持续优化商品、页面与服务。
5.  **瓶颈突破**：尝试付费推广等方式，冲击更高层级。

**节点二：开设新店**
如果你觉得当前店铺或产品已达到投入上限，不想再冒更大风险，可以考虑开设新店。
*   **吸取经验**：总结第一个店铺的运营计划和思路。
*   **复制优化**：经营类似商品，并按照已验证的模式操作，通常第二个店铺的效果会更好。
*   **再次决策**：在新店达到一定规模后，再重新评估是否要加大投入进行推广。

做好基础规划后，对类似商品进行小范围的模式调整即可。后期规划是独立且系统的工程，同样适合用专题课程来详细拆解。
