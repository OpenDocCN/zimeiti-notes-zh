## 开店流程与成本

上一节我们介绍了项目的整体框架，本节中我们来看看如何开设小红书店铺。

开店流程本身并不复杂。首先，打开小红书应用，点击左上角的菜单图标，选择“创作中心”。然后点击“更多服务”选项，选择“店铺”，最后点击“立即入住”并按官方要求提交资料即可。有人将小红书开店包装成项目出售，这很奇特。

![](img/be2d5a44225ffeb24812d3331f81aecb_3.png)

小红书店铺主要有两种形式：**个人店**和**企业店**。

*   **个人店**：分为仅需身份证和1000元保证金的个人店，以及需要额外营业执照的个体工商户店。
*   **企业店**：包括普通企业店、专卖店和旗舰店，都需要营业执照并缴纳600元年费。

对于初学者，直接选择**个人店**即可。实测表明，平台并未对企业店有流量倾斜。唯一的区别是，个人店只能开一家，而一份营业执照可以开三家店。但若一个店都未做好，开再多店也无用。

![](img/be2d5a44225ffeb24812d3331f81aecb_5.png)

小红书电商不适合以“店群”形式操作，因为其电商自然流量占比非常低。铺再多的店和货也用处不大。小红书要做的是**账号矩阵**，而非店铺矩阵，这点在后续流量部分会详细说明。

以下是开店涉及的主要成本：

1.  **保证金**：1000元，不做可退。
2.  **平台扣点**：营业额的5%。
3.  **支付渠道费**：每单约0.7%。
4.  **流动资金**：初期可采用无货源模式，准备2000-3000元完全足够。

## 选品策略与渠道

开完店后，下一步就是决定卖什么。选品其实很简单，其他平台的爆款，大概率在小红书也能成为爆款。

目前推荐通过以下三个渠道进行选品：

1.  **某东/某宝**：寻找符合小红书平台调性的爆款产品。有电商朋友将其在某宝的爆款搬到小红书，第一篇笔记就出单，第二篇笔记单量破百。
2.  **某音精选联盟**：查看某音精选联盟的排行榜，了解现阶段哪些商品卖得好。将符合小红书用户调性的产品直接搬运测试。
3.  **某多多销量排行榜**：选择适合小红书的类目排行榜，从中挑选爆品搬运到小红书。

整个选品的逻辑是**利用市场来帮助我们选品**，而非自己空想。我们要借助大卖家的选品能力来弥补自身不足。只需判断排行榜上的爆品是否可能与小红书用户匹配，如果匹配就搬过来测试。这相当于借助别人的能力为自己选品，是最快且最稳妥的方式，成功率远高于盲目测试。

## 流量获取与内容创作

产品上架后，就进入了最重要的环节：如何获取流量。小红书是一个**重内容**的平台，店铺九成以上的流量都来自发布的笔记。这就是为什么之前强调不要铺店群，而要做**账号矩阵**，尽可能多地发布笔记来获取流量。

流量通路是：用户通过笔记对产品产生兴趣，然后进入店铺完成购买。铺一堆店铺的效果，远不如多发几篇优质笔记。因此，**如何在小红书上发布笔记是电商成功的关键**。

小红书平台直接发布硬广效果很差。其用户群体认知较高，直接发广告无法获得信任，更谈不上转化。发笔记时一定要从**素人（用户）的角度**切入，站在用户立场做种草内容，而非官方视角，这才是真正有效的。

这种种草笔记的制作方法其实很简单：去各大平台（如某音、某宝、某东）的商品评论区和买家秀中寻找好看素材，直接复制使用。文案则在买家评价的基础上进行修改。这些买家评价本身就是用户的使用体验，天然适合用作种草笔记，且素材取之不尽。

当单个账号的标准运营流程（SOP）跑通后，就可以开始**矩阵引流**了。通过批量运营小红书账号、批量发布笔记，为店铺引流，放大收益。
