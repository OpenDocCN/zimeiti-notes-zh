## 二、卖课类无人直播搭建指南 📚

卖课类直播对素材原创性要求较高，目的是推广知识付费课程或小程序。其核心是准备高质量的原创视频素材与话术。

以下是搭建卖课类无人直播的具体步骤：

1.  **准备原创素材**
    *   需要准备**至少2小时**的原创视频素材。
    *   素材来源可以是自己录制并剪辑，例如录制“找不同”等小程序游戏过程。
    *   剪辑的核心目的是“把别人的内容变成自己的”，以通过平台原创审核。

![](img/1a9b21912ccbdd246b64f516b506a25f_15.png)

2.  **处理音频**
    *   使用手机单独录制2小时的讲解音频（话术）。
    *   将视频素材与音频素材导入剪映等软件，合成为一个完整的直播素材文件。
    *   公式表示为：`最终素材 = 原创视频 + 独立录制音频`

3.  **搭建直播循环**
    *   准备**8天**的不同的直播素材（每天2小时）。
    *   播放顺序为：Day1 -> Day2 -> ... -> Day8 -> Day1。
    *   这样就形成了一个8天一循环的直播素材库，无需每天制作新内容。

4.  **应对平台审查节点**
    *   平台会在特定时间点（如开播后6小时、24小时）进行人工或算法加强审查。
    *   策略：在这些审查节点**之前**，更换直播素材或短暂下播后再换素材开播，以规避检测。

## 三、关键软件与防检测技术 🛡️

无论哪种玩法，使用合适的软件并加载防检测技术是保证直播稳定的关键。本节我们来看看具体工具和一项重要技术。

### 1. 常用软件方案
*   **OBS + VR软件**：成本较低，一套方案约5-15元/年，通过虚拟摄像头实现。
*   **AB电脑方案**：使用两台电脑，通过NDI技术传输信号，稳定性更高，但成本和操作复杂度也更高。
    *   `B电脑（推流端）` -> `网络/服务器` -> `A电脑（接收&播放端）` -> `直播伴侣`
*   两种方案的底层逻辑均基于OBS及其开源插件整合而成。

### 2. 核心防检测技术：应用LUT滤镜
这是在OBS中实现视频“驱虫”和抗机审的核心技巧。

![](img/1a9b21912ccbdd246b64f516b506a25f_19.png)

*   **原理**：LUT（查找表）滤镜能直接改变视频底层的色彩和像素信息，与素材深度融合，从而改变视频的“指纹”，让平台难以识别为重复录播内容。
*   **操作**：在OBS的“视频源”上右键 -> 滤镜 -> 添加“应用LUT”。
*   **代码/公式示意**：`输出画面 = LUT_Transform(原始视频帧)`
*   **来源**：此技术最初源于游戏主播为提升画面质感而使用，后被发现在信号中断时仍能稳定播放，从而应用于无人直播防检测。

## 四、卖货类无人直播搭建指南 🛒

卖货类直播侧重于话术引导和转化，对音频处理的要求高于视频。本节我们来看看如何搭建一个稳定的卖货直播间。

以下是卖货类直播的搭建流程：

1.  **素材准备**
    *   **视频**：使用绿幕素材或真实直播场景录制片段。
    *   **音频**：这是关键。必须录制**其他平台**（如快手）主播的直播话术，**切勿使用抖音平台**的音频，以免被快速溯源。
    *   建议录制**6段**话术，每段约2小时，总时长12小时。

2.  **OBS场景搭建（以VR软件为例）**
    在OBS中按以下顺序添加来源（从底到顶）：
    1.  **底层**：VR虚拟摄像头加载的真人视频素材。
    2.  **中层**：添加图片、贴片等装饰性元素。
    3.  **上层**：添加一个真实的物理摄像头（即使不用），并启用绿幕抠图，使其透明。此举旨在让预览画面更复杂，干扰检测。
    4.  **关键步骤**：在OBS中新增一个“媒体源”，放入一个其他直播间的**实时转播流**。将此源置于顶层。
    *   **原理**：平台主要检测“预览画面”，而最终观众看到的“直播画面”仅是底层VR素材。顶层转播流用于干扰检测，但不影响实际播出内容。

![](img/1a9b21912ccbdd246b64f516b506a25f_23.png)

3.  **音频播放与风控**
    *   使用多个播放器（如暴风影音、QQ音乐、网易云音乐）同时播放不同的话术音频文件。
    *   将主要话术音频音量调大，其他背景音频音量调小，形成“**三音频叠加**”效果。
    *   **开播初期**：前45分钟主播真人出镜讲解，度过风控期。
    *   **中期**：45分钟后，主播离开，将OBS中的绿幕虚拟人像拉到顶层，开始无人直播。
    *   **后期**：每间隔几小时，切换不同播放器的主次音频音量，改变音频“指纹”。

## 五、总结与建议 ✅

![](img/1a9b21912ccbdd246b64f516b506a25f_33.png)

本节课中我们一起学习了无人直播的三种主要玩法及其搭建细节：

1.  **卖课直播**：核心是**原创长视频素材**与**8天循环**策略，重在内容本身。
2.  **通用技术**：**OBS软件**是基础，**应用LUT滤镜**是关键的防检测手段，能有效改变视频底层信息。
3.  **卖货直播**：核心是**音频处理**，采用“**外平台音源**”+“**多播放器叠加**”+“**预览画面干扰**”的组合策略来规避审查。

![](img/1a9b21912ccbdd246b64f516b506a25f_35.png)

**给初学者的建议**：从成本较低的“OBS + VR + LUT”方案开始尝试，优先处理好音频来源和素材的原创性/驱虫处理。细节决定成败，每一个步骤都需严格按照要求操作。理论结合实践，大胆测试，才能在无人直播领域找到适合自己的稳定玩法。
