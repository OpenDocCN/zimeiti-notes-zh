# 抖音运营实战：P14：7. 视频剪辑实操（下） ✂️

在本节课中，我们将继续学习视频剪辑的核心操作，重点掌握如何为多片段视频添加转场、调整比例以及添加字幕。这些步骤对于提升视频的流畅度、观感和完播率至关重要。

上一节我们介绍了视频素材的导入与初步剪辑，本节中我们来看看如何将这些片段流畅地组合成一个完整的作品。

## 添加视频转场

![](img/06f6e6eec636f8d3559b89450a8090ad_3.png)

我们的视频通常由多个片段组成，每个片段时长可能在10秒左右。这些片段之间需要有自然的衔接，这就需要使用“转场”。

![](img/06f6e6eec636f8d3559b89450a8090ad_5.png)

**为什么必须添加转场？**
转场的作用是明确告知用户，当前片段内容已结束，即将切换到下一个不同的片段。如果不添加转场，观众在切换到新片段时可能会感到突兀，产生“视频内容怎么突然变了”的疑惑。这种疑惑可能导致观众退出视频查看详情，从而**严重降低视频的完播率**。而完播率是影响视频推荐量和收益的关键指标之一。

![](img/06f6e6eec636f8d3559b89450a8090ad_7.png)

以下是添加转场的具体步骤：

![](img/06f6e6eec636f8d3559b89450a8090ad_9.png)

1.  在视频编辑轨道上，找到并点击右侧的 **“+”** 号按钮。
2.  在弹出的菜单中，选择 **“素材库”**。
3.  在素材库的 **“热门”** 分类中，可以找到大量现成的转场素材，例如各种动态效果或搞笑的间隔画面。
4.  选择转场时，请注意时长。避免选择过长的转场（如16秒），观众通常没有耐心看完。**最合适的转场时长应保持在3到5秒之间**，建议选择3秒左右的素材。
5.  点击选中合适的转场素材，即可将其添加到视频轨道中。

![](img/06f6e6eec636f8d3559b89450a8090ad_11.png)

![](img/06f6e6eec636f8d3559b89450a8090ad_13.png)

## 统一视频比例

添加的转场素材，必须与主视频的比例保持一致，以确保画面协调。我们的目标比例是 **16:9**（横屏）。

![](img/06f6e6eec636f8d3559b89450a8090ad_15.png)

以下是统一比例的步骤：

![](img/06f6e6eec636f8d3559b89450a8090ad_17.png)

1.  选中已添加的转场片段。
2.  在编辑菜单中找到并点击 **“画布模糊”** 或类似的比例调整选项。
3.  选择第一个清晰的模糊背景样式，系统会自动将转场画面适配为16:9的横屏比例。
4.  最后，点击右下角的 **“√”** 确认修改。

至此，视频的多片段拼接与比例调整就完成了。记住，寻找素材和剪辑时，务必严格遵循这些步骤和比例要求。

![](img/06f6e6eec636f8d3559b89450a8090ad_19.png)

## 添加视频字幕（文案）

![](img/06f6e6eec636f8d3559b89450a8090ad_21.png)

字幕能极大提升视频的观看体验和信息传达效率。添加字幕的步骤如下：

1.  点击手机剪辑界面左下角的 **“文字”** 选项。
2.  选择 **“新建文本”**，然后点击右侧的 **“识别字幕”** 功能。
3.  点击 **“开始匹配”**，系统会自动识别视频或音频中的人声，并生成对应的字幕文本。
4.  字幕生成后，可以进行微调。拖动字幕文本框右下角，可以放大字幕。
5.  将放大后的字幕文本框，拖动到视频画面最下方的位置，这样观看最舒适。
6.  调整好第一句字幕后，后续字幕通常会同步应用此样式，无需重复调整。

## 注意事项总结

![](img/06f6e6eec636f8d3559b89450a8090ad_23.png)

在本节课中，我们一起学习了视频剪辑后半部分的核心操作：

![](img/06f6e6eec636f8d3559b89450a8090ad_25.png)

1.  **添加转场**：用于衔接不同片段，提升流畅度，保障完播率。转场时长宜短不宜长，**3-5秒**为佳。
2.  **统一比例**：确保所有视频片段与转场素材均为 **16:9** 横屏比例，使用“画布模糊”功能快速调整。
3.  **添加字幕**：利用“识别字幕”功能自动生成，调整大小并置于画面底部，以优化观看体验。

![](img/06f6e6eec636f8d3559b89450a8090ad_27.png)

请记住，剪辑时不要使用过长的原始素材，以免影响平台审核。整个流程并不复杂，关键在于严格、细致地执行每一个步骤。按照本教程的方法操作，你就能制作出符合平台要求、观感流畅的短视频作品。
