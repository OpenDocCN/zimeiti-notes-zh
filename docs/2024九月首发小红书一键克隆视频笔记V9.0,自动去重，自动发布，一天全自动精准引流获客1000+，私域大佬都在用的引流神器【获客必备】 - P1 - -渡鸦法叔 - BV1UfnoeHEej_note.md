# 小红书克隆系统V9.0使用教程：1：软件概述与准备工作

在本节课中，我们将学习小红书克隆系统V9.0的完整使用方法。这是一个集笔记克隆、自动发布、内容采集于一体的自动化工具，旨在帮助用户高效运营小红书账号。课程将从软件注册登录讲起，逐步介绍各项核心功能。

## 软件注册与登录

首先，你需要注册一个软件账号。请务必记住你的账号和密码，因为后续需要通过账号密码和验证码进行登录。

以下是登录后的软件主界面。使用本软件前，必须遵守相关规章制度，否则将无法继续使用。



![](img/e5019d9e1dd1528305188fa3e98c2430_1.png)

## 使用规范与注意事项

规范使用软件至关重要。发布笔记时，请注意以下两点：
*   不要频繁发布违规内容。
*   不要搬运侵犯他人肖像权（如露脸）的素材。如果因搬运被追究责任，后果需自行承担。

新版软件采用了卡密激活会员的机制。登录后，如需无限使用软件，需在指定位置输入卡密激活会员。

**会员激活代码示例：**
```
1. 在“卡密”输入框填入你的卡密。
2. 点击“开通会员”按钮。
3. 在“会员信息”处查看到期时间。
```

渡鸦科技社群的会员可免费领取一年会员卡密。非会员用户每日可使用积分操作，但积分有限，仅能进行少量克隆或采集。具体积分消耗规则请查看官方声明。

![](img/e5019d9e1dd1528305188fa3e98c2430_1.png)

**重要提醒：** 每人仅限领取一个卡密，仅供自用。严禁多领或对外倒卖，违者将被取消会员资格并回收卡密。

软件从4.0版本迭代至9.0，不断优化了Bug并提升了响应速度。界面也变得更加简洁，例如移除了填写CK的步骤。

# 小红书克隆系统V9.0使用教程：3：比特浏览器配置与IP管理

上一节我们提到了新版取消了内置的账号和IP管理，本节将详细介绍如何通过比特浏览器来完成这些关键配置。

![](img/e5019d9e1dd1528305188fa3e98c2430_10.png)

## 配置浏览器窗口与IP

所有账号和IP管理现在都在比特浏览器中进行。以下是配置步骤：

1.  在比特浏览器中，点击“新建窗口”。
2.  自定义窗口名称。
3.  在代理设置中，选择“Socks5”代理方式。
4.  填入你的代理IP、端口、账号和密码。
5.  点击“检测”测试代理IP是否可用。



![](img/e5019d9e1dd1528305188fa3e1dd1528305188fa3e98c2430_3.png)

## 指纹随机化

比特浏览器可以修改浏览器指纹，使其更“干净”，降低被平台检测的风险。配置时，可以勾选“每次启动浏览器时随机指纹”，或在启动前手动点击“随机指纹”。

完成以上配置后，点击“确定”保存。



![](img/e5019d9e1dd1528305188fa3e98c2430_5.png)

## 登录小红书账号

![](img/e5019d9e1dd1528305188fa3e98c2430_12.png)

打开配置好的浏览器窗口，**不要直接搜索“小红书”**，这会导致页面卡住。正确的方法是：
1.  从已登录的浏览器中复制小红书创作者中心的网址。
2.  在比特浏览器地址栏直接粘贴该网址并访问。
3.  在打开的页面中扫码或接收验证码登录账号。



![](img/e5019d9e1dd1528305188fa3e98c2430_14.png)

![](img/e5019d9e1dd1528305188fa3e98c2430_7.png)



![](img/e5019d9e1dd1528305188fa3e98c2430_16.png)

![](img/e5019d9e1dd1528305188fa3e98c2430_8.png)



![](img/e5019d9e1dd1528305188fa3e98c2430_10.png)

## 配置优势

此配置方式确保了登录IP和发布IP完全一致，每个账号都在独立的浏览器环境中运行。这样配置的账号通常更稳定，可以维持数周不掉线。IP资源推荐使用可靠的供应商。

# 小红书克隆系统V9.0使用教程：4：克隆与发布功能详解

完成环境配置后，本节我们将深入讲解克隆系统的核心功能：克隆图片、克隆视频以及自动发布笔记。

## 克隆与发布图片笔记

回到克隆软件界面，选择你刚配置好的浏览器窗口（例如4号窗口），点击“发布”。

**注意：** 发布时请勿随意调整日志窗口的大小，保持默认大小即可，否则可能导致报错。发布成功后，你可以在“笔记管理”中查看已发布的笔记。



![](img/e5019d9e1dd1528305188fa3e98c2430_12.png)

## 克隆视频笔记（新增功能）

![](img/e5019d9e1dd1528305188fa3e98c2430_18.png)

![](img/e5019d9e1dd1528305188fa3e98c2430_19.png)

这是V9.0版本重磅新增的功能。操作步骤如下：
1.  复制你想要克隆的视频笔记链接。
2.  将链接粘贴到软件中，点击“一键获取”。
3.  软件会快速下载并驱重处理视频。视频文件保存在本地指定路径，你可以在软件下方查看具体路径。



![](img/e5019d9e1dd1528305188fa3e98c2430_21.png)

![](img/e5019d9e1dd1528305188fa3e98c2430_14.png)



![](img/e5019d9e1dd1528305188fa3e98c2430_23.png)

![](img/e5019d9e1dd1528305188fa3e98c2430_16.png)

![](img/e5019d9e1dd1528305188fa3e98c2430_25.png)

4.  对获取的标题和文案进行去重修改，并添加合适的标签。
5.  选择浏览器窗口，点击“开始发布”。

![](img/e5019d9e1dd1528305188fa3e98c2430_27.png)

**重要提醒：** 为避免侵权和降低账号权重，请尽量避免搬运露脸的人设类视频。可以选择搬运混剪类视频，或对克隆的视频进行二次创作后再发布。

![](img/e5019d9e1dd1528305188fa3e98c2430_29.png)

## 自动发布笔记功能

除了克隆，软件还支持从本地库自动发布笔记。
1.  在“文案库”和“图片库”中提前准备好标签、文案和图片，可以设置多个选项。
2.  发布时，选择“随机选取”标签、文案和图片。
3.  选择已登录账号的浏览器窗口，点击“开始发布”，软件便会自动组合内容并发布。

**注意：** 文案内容不宜过长，电脑端发布建议不超过1000字。目前多个账号发布的内容可能相同，后续版本会更新为每个账号随机发布不同内容。

在“RP自动化采集”板块，功能已全面升级。你可以采集指定关键词下的笔记，甚至可以筛选特定时间内的评论。

**采集设置示例：**
```
关键词：创业
采集时间范围：24小时内
排序方式：最新
```
采集到的都是公开数据，请仅用于内部调研与降本增效，切勿用于违规用途。
