# 剪映教程：P18：剪映核心剪辑功能——文本功能 📝

在本节课中，我们将要学习剪映的核心功能之一：文本功能。一个完整的视频不仅包含画面和背景音乐，文本或字幕也是传递信息、突出主题的关键元素。通过添加文本，可以更准确、更快速地让观众理解视频内容。

接下来，我们将详细介绍如何在剪映中为视频添加文本，并掌握文本功能的各种使用方法和技巧。

![](img/7aeb78e26573e6f61dec1554ae7500d4_1.png)

## 文本功能概述

在剪映中，我们可以自由地为视频添加字幕。添加的字幕可以单独设置样式，包括字体、字号、颜色、间距以及动画效果。

此外，剪映还提供了两个非常高效的功能：**字幕自动识别**和**歌词识别**。这两个功能可以基于视频中的音频，一键快速生成字幕，极大地降低了操作难度和时间成本。

最后，文本功能模块中还包含**贴纸**的添加选项。

关于文本功能的基本介绍就到这里。接下来，我们将进入实际操作环节。

## 功能实操演练

文本功能位于一级菜单栏。点击底部菜单栏的 **`文字`** 按钮，即可进入文本功能界面。在这里，你会看到四个主要选项：**新建文本**、**识别字幕**、**识别歌词**和**贴纸**。

以下是每个功能的详细操作步骤。

### 1. 新建文本（手动添加）

新建文本功能允许你手动输入和创建字幕。

首先，点击 **`文字`** -> **`新建文本`**，在输入框中键入你想添加的文字，例如“剪映视频剪辑教程”。输入完成后，画面上会出现对应的文字，时间轴上也会新增一个文本轨道。

接下来，我们可以对这段文字的样式进行个性化设置。点击底部的 **`样式`** 选项，可以进行以下调整：
*   **字体**：更改文字的字体，例如选择“新青年体”。
*   **样式**：调整文字的**描边**、**标签**、**阴影**等效果。你可以更改描边的颜色和粗细。
*   **花字**：直接应用抖音官方提供的各种特色艺术字。
*   **气泡**：为文字添加对话气泡样式，常用于采访或Vlog视频。
*   **动画**：为文字添加入场、出场或循环动画，例如“螺旋上升”，并可以设置动画的持续时间。

![](img/7aeb78e26573e6f61dec1554ae7500d4_3.png)

设置完成后，点击对勾保存。预览视频，即可看到添加了动画和样式的字幕效果。

### 2. 识别字幕（语音转文字）

如果你的视频包含人声对话或旁白，可以使用“识别字幕”功能自动生成字幕。

![](img/7aeb78e26573e6f61dec1554ae7500d4_5.png)

![](img/7aeb78e26573e6f61dec1554ae7500d4_6.png)

首先，确保视频轨道上已添加包含人声的音频。然后，点击 **`文字`** -> **`识别字幕`**。

在弹出界面中，点击 **`开始识别`**。剪映会自动分析音频，并将识别出的文字以字幕形式添加到视频中。识别完成后，你同样可以点击这些字幕轨道，对它们的样式进行二次编辑。

### 3. 识别歌词（音乐转字幕）

![](img/7aeb78e26573e6f61dec1554ae7500d4_8.png)

当视频中添加了带有歌词的歌曲时，可以使用“识别歌词”功能快速生成歌词字幕。

![](img/7aeb78e26573e6f61dec1554ae7500d4_10.png)

![](img/7aeb78e26573e6f61dec1554ae7500d4_12.png)

首先，为视频添加一首歌曲。然后，点击 **`文字`** -> **`识别歌词`**。

![](img/7aeb78e26573e6f61dec1554ae7500d4_14.png)

点击 **`开始识别`**，剪映会自动识别音乐中的歌词，并生成对应的字幕轨道。生成后，你可以选中任意一句歌词，像编辑普通文本一样，调整其字体、位置、颜色，并添加动画效果，使其与视频意境更匹配。

### 4. 贴纸

文本功能区域还提供了“贴纸”选项，里面包含丰富的静态和动态贴纸元素，可以进一步装饰和丰富视频画面。

## 总结

![](img/7aeb78e26573e6f61dec1554ae7500d4_16.png)

![](img/7aeb78e26573e6f61dec1554ae7500d4_17.png)

本节课我们一起学习了剪映的文本功能。我们了解到，文本功能可以快速为视频添加字幕，并能对字幕的颜色、字体、字号、动画等进行个性化调整。

![](img/7aeb78e26573e6f61dec1554ae7500d4_19.png)

我们重点掌握了三种添加字幕的方法：
1.  **手动新建文本**，适用于自定义标题或说明。
2.  **识别字幕**，适用于为旁白或对话自动生成字幕。
3.  **识别歌词**，适用于为歌曲快速添加歌词字幕。

通过有效地添加和美化字幕，可以显著提升视频信息的传达效率和整体内容质量。
