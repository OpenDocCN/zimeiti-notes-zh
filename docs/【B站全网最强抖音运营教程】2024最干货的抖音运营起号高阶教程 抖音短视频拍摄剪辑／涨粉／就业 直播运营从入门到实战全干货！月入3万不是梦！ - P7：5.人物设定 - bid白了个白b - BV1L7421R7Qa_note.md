# 抖音运营高阶教程：P7：人物设定 🎭

在本节课中，我们将要学习抖音账号运营中一个至关重要的环节——**人物设定**。一个清晰、真实且富有吸引力的人设，是账号获取流量、建立信任并最终实现变现的基石。

![](img/7da8c59cc8ae942af517d6986a0d5d8e_2.png)

## 人设策划的核心概念

![](img/7da8c59cc8ae942af517d6986a0d5d8e_4.png)

![](img/7da8c59cc8ae942af517d6986a0d5d8e_6.png)

上一段我们了解了人设的重要性，本节中我们来看看人设策划的具体内涵。

![](img/7da8c59cc8ae942af517d6986a0d5d8e_8.png)

在抖音中策划人设，类似于在游戏中创建角色，需要预先设定角色的各项属性，例如称呼、职业与技能。然而，与游戏角色不同的是，抖音中设定的是**人生角色**，因此必须更具真实感。

![](img/7da8c59cc8ae942af517d6986a0d5d8e_10.png)

无论是短视频还是直播，一个成功的IP能够解决粉丝的信任问题，起到裂变传播的作用，并具备强大的变现能力。因此，我们运营账号，无论是创作内容还是策划选题，其核心目的都是在获取流量的同时**打造IP**。

![](img/7da8c59cc8ae942af517d6986a0d5d8e_12.png)

打造IP的核心，是在特定场景或情境中，完成**账号人格化**的过程。这个过程需要注入情感、塑造故事并贴上具有辨识度的标签。

![](img/7da8c59cc8ae942af517d6986a0d5d8e_14.png)

## 账号人格化：人设策划的六个维度

![](img/7da8c59cc8ae942af517d6986a0d5d8e_16.png)

账号人格化，即我们通常所说的“人设”。对于人设的策划，可以从以下六个具体维度着手。

![](img/7da8c59cc8ae942af517d6986a0d5d8e_18.png)

以下是策划账号人设时需要关注的六个核心方面：

1.  **名字/姓氏/昵称**
    一个IP首先需要一个称呼，这是IP传播的名片。必须明确IP叫什么，其重要性等同于一个品牌的名称或商标。
2.  **职业/职务/职称**
    人格化必然需要身份。这是用户认识这个人或IP的联想入口，例如：老师、教授、博士、律师。
3.  **称呼/社会关系**
    一个IP有了亲切的称呼，能快速拉近与用户的距离，起到情感共鸣的作用。例如：XX哥、XX大爷、XX姥姥。
4.  **形象/第一印象**
    从用户视角看待IP，使IP形象化，更容易占据用户心智。例如：大叔形象、萌萌哒、铁憨憨、甜美可爱、光头。
5.  **性格/爱好/特点**
    一个具有鲜明特点的IP，除了记忆点强，也更具真实性。过于完美的形象容易脱离大众。例如：暴躁、娘炮、爱吃、爱喝酒。
6.  **穿着打扮/财富地位**
    这属于社会刻板印象和大众普遍认知。借助用户的固有思维，可以快速为IP在用户心智中定位。例如：有钱代表会赚钱，开豪车代表成功人士，打扮土气代表老师。

以上内容策划完成后，即可结合自身的产品优势、服务优势等，将其落实到**抖音账号的五件套**（头像、昵称、背景图、简介、视频封面）的搭建中。

## 提炼人设记忆点：超级符号

除了对账号IP进行人格化策划，还需要通过选题与内容策划，为人设提炼出深刻的记忆点，即通常所说的**超级符号**。

超级符号的提炼可以从以下几个感官维度进行：

以下是提炼超级符号的几个关键方向：

*   **画风（视觉）**：如独特的场景、道具、服饰风格。
*   **听觉**：如标志性的BGM、特效音、配音或独特的声线。
*   **文案**：能够反映优势且易于记忆的提炼词，例如：“简单粗暴”、“XX天花板”。
*   **表情与情绪**：如标志性的笑、惊恐、无奈等表情包式情绪传递。
*   **语言特色**：如方言、口音、错误发音、网络用语、慢条斯理或抑扬顿挫的讲话方式。
*   **动作与习惯**：习惯性的肢体动作，例如：撩头发、扶眼镜、特定的手势。

超级符号提炼好后，在进行短视频创作时，就应持续、一致地在视频中呈现这些元素，不断强化用户记忆。

## 课程总结

本节课中，我们一起学习了抖音账号运营中**人物设定**的完整方法论。我们首先明确了人设即**账号人格化**的核心目标，然后从六个维度详细拆解了如何策划一个立体、真实的人设，最后介绍了通过提炼**超级符号**来强化人设记忆点的具体方法。一个成功的人设，是连接内容、粉丝与商业价值的桥梁，务必精心策划与打磨。
