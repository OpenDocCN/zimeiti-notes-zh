# AI短视频创作：P2：抄作业教程 🚀

在本节课中，我们将学习一种通过“抄作业”方式，利用AI工具高效创作温情故事类短视频的方法。该方法操作简单，适合新手，旨在帮助账号快速涨粉并实现中视频计划变现。

![](img/24353b93b4f859a3dce0a0be934f504d_0.png)

## 概述
本教程将分步讲解如何寻找爆款选题、提取并处理文案、利用工具自动生成视频，最终通过中视频计划发布获利。核心在于借鉴已验证成功的爆款元素，降低创作难度。

---

![](img/24353b93b4f859a3dce0a0be934f504d_2.png)

上一节我们概述了本教程的目标，本节中我们来看看具体的操作步骤。

### 第一步：寻找爆款选题
首先，需要在抖音平台内确定视频创作方向。温情故事类内容容易引发观众情绪共鸣，具有较高的爆款潜力。

以下是具体操作步骤：
1.  打开抖音，在搜索框内输入关键词“**温情故事**”。
2.  进入搜索结果页后，点击顶部的“**视频**”板块。
3.  点击“筛选”按钮，选择“**最多点赞**”进行排序。

完成以上步骤后，你将看到一系列点赞量极高的爆款温情故事视频，这些就是可供参考的选题库。

---

![](img/24353b93b4f859a3dce0a0be934f504d_4.png)

在找到了优质的参考视频后，下一步是获取其核心内容。

![](img/24353b93b4f859a3dce0a0be934f504d_6.png)

### 第二步：提取并处理视频文案
这一步的目的是获取爆款视频的文案脚本，但切记不能直接照搬，需进行二次创作以避免违规。

以下是具体操作步骤：
1.  在选定的爆款视频播放页面，点击右下角“**分享**”按钮。
2.  在弹出的菜单中，找到并点击“**复制链接**”选项。
3.  将复制的链接粘贴到专业的“**文案提取工具**”中，获取视频的原始文案。
4.  **关键处理**：通读提取出的文案，理解其故事脉络，然后完全**用自己的语言和表达方式重新讲述**一遍这个故事。

经过以上处理，你就得到了一份既保留爆款基因又属于原创的文案脚本。

---

文案准备就绪后，我们将进入视频的自动化制作环节。

### 第三步：使用工具自动生成视频
利用剪映APP的智能功能，可以快速将文案转化为一个完整的视频，无需复杂的剪辑技巧。

以下是具体操作步骤：
1.  打开剪映APP，点击“**一键成片**”功能。
2.  将第二步中处理好的原创文案粘贴或导入到指定区域。
3.  剪映会根据文案内容**自动匹配画面素材**并**生成语音配音**。
4.  对自动生成的视频进行预览，确认无误后即可导出。

至此，一个完整的短视频作品就制作完成了。

---

视频制作完成后，正确的发布渠道是获得收益的关键。

### 第四步：通过中视频计划发布
为了获得播放收益，必须通过“中视频伙伴计划”的入口进行发布。

以下是具体操作步骤：
1.  在抖音App顶部搜索“**中视频计划**”。
2.  找到并进入“**中视频伙伴计划**”的官方入口。
3.  通过该入口发布第三步制作好的视频。
4.  发布时，务必勾选“**同步至西瓜视频和今日头条**”选项。

在此三个平台上的有效播放量，都将为你产生收益。
