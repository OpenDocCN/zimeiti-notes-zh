# 影视推广平台：P13：8. 影视推广平台详解 🎬

在本节课中，我们将要学习如何通过“影视推广”平台，在抖音、快手等熟悉的平台上发布视频并获得收益。我们将详细讲解其运作模式、收益结算方式以及具体操作步骤。

---

上一节我们介绍了视频在多平台发布的基础，本节中我们来看看如何在抖音、快手等平台通过特定任务获得收益。

很多同学问到，如果想把视频发布到抖音或快手平台，该如何结算收益。需要明确的是，我们制作的视频发布到除抖音、快手外的三十多个自媒体平台，可以依靠视频播放量直接结算收益。然而，抖音和快手这两个平台情况特殊。

就像之前提到的，视频发布到抖音和快手，最主要的作用是增长粉丝。这里要强调一下，是否有同学只想专注于做抖音和快手这两个自己熟悉的平台？是否希望在运营这两个平台时，不仅能涨粉，还能依靠播放量结算收益？如果有的同学，这正是本节要解决的问题。

刚才讲到的所有视频，可以发布到以下四个平台：
*   抖音
*   快手
*   小红书
*   微信视频号

这四个平台大家都很熟悉，是日常会使用的平台。小红书可能更多被女性或年轻95后用户使用。微信视频号则是微信内置的小程序，每个人都可以发布内容。

在这几个平台发布视频后，我们通过“影视推广”平台接取任务，并通过完成任务来结算奖金。可用的素材非常多，包括爱奇艺、搜狐等平台提供的任务，这些任务对应着具体的影视剧名称。

那么，影视推广具体是什么意思？影视推广是指，每一部电视剧、电影或综艺节目都需要进行宣传。现在很多广告商或制片方，由于各种原因（如过去的疫情限制），无法完全依赖线下宣传，便会利用影视推广平台。我们制作视频帮助他们推广内容，就可以依靠视频的播放量来结算奖金。

以下是进行影视推广任务的基本流程：
1.  **报名与提交**：每个视频制作完成后，需要提交一个“任务回填链接”。我们需要在链接中填写自己的微信号、账号昵称，或抖音、快手的ID。
2.  **明确任务信息**：需要清楚注明所做的是哪一部电视剧或电影。
3.  **提供作品链接**：粘贴发布作品的平台链接。
4.  **记录发布时间**：填写视频发布的具体日期。
5.  **上传播放量证明**：将视频的播放量截图上传上去。

完成回填链接的提交后，平台方就会为我们结算收益。影视推广的收益通常比单纯依靠平台播放量获得的收益更高。

现在来看一下具体的收益计算方式。短剧推广是目前非常主流的方向，很多传媒公司和网红会自制短剧，他们需要推广，便会发布任务让我们制作视频并发布到指定平台。

**收益根据播放量要求计算，并且对发布账号的粉丝数没有要求。** 结算时，通常取一部剧推广周期内，播放量最高的三个作品，将其播放量相加的总和来计算奖金。

以下是各平台大致的收益单价参考：
*   **抖音**：每1万播放量，奖金约为 **50元**。
    *   *对比：之前提到的头条号平台，每万次播放收益约为34元。抖音的收益单价明显更高，因为其日活跃用户接近10亿，流量巨大。*
*   **微信视频号**：每1万播放量，奖金约为 **10元**。
    *   *单价较低，因为微信用户基数大，人人都可能观看视频号，流量相对“廉价”。*
*   **小红书**：每1000播放量，奖金约为 **30元**。
    *   *这是单价最高的平台，换算成万次播放收益非常可观。*

此外，还有一些特殊的结算方式，例如搜狐短剧的推广。虽然搜狐视频现在的知名度不如以往，但它拥有很多视频版权，也会发布推广任务。

对于抖音和快手双平台发布的任务，有时会按照**点赞数**来结算收益，同样没有粉丝要求。例如：
*   单条作品获得5000个赞，奖金可能是100元。
*   抖音作品获得1万个赞，奖金可能是200元。
*   快手作品获得1万个赞，奖金可能是180元（有时单价也很高）。

本节课中我们一起学习了影视推广平台的运作机制。关键点在于：通过接取特定影视剧的推广任务，在抖音、快手、小红书、微信视频号等平台发布视频，并根据**播放量**或**点赞数**来结算奖金，此模式对账号粉丝数没有门槛要求，为初学者提供了明确的变现路径。记住核心流程：**接任务 -> 制作发布 -> 回填数据 -> 结算收益**。
