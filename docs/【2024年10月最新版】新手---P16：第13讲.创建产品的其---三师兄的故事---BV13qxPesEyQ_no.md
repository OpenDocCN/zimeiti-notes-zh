## 🔄 跨站点同步商品

上一节我们介绍了在单一站点创建产品的基础方法，本节中我们来看看如何将产品同步到其他亚马逊站点，以扩大销售范围并共享产品评价。

跨站点同步商品是指将一个站点（如美国站）已上架销售的产品，同步到另一个目标站点（如加拿大站或墨西哥站）。同步后，产品信息、库存（部分计划下）以及**评价**可以共享，这能为新站点链接快速积累信誉。

以下是实现跨站点同步的三种主要方法：

1.  **亚马逊物流远程配送计划**：适用于将美国站产品同步至加拿大站和墨西哥站。
2.  **通过模板上传**：适用于将产品同步至欧洲、日本等与北美不共享库存的站点。
3.  **开通泛欧计划**：适用于在欧洲境内多个国家站点间同步商品与库存。

接下来，我们将逐一详细讲解这三种方法。

### 方法一：亚马逊物流远程配送计划

此方法适用于从美国站同步商品至加拿大站和墨西哥站。开通后，亚马逊会使用美国库存直接向加拿大和墨西哥的买家进行配送。

**操作步骤如下：**

1.  登录美国站卖家后台。
2.  依次点击 **库存 > 亚马逊物流远程配送**。
3.  首次进入需阅读并同意服务条款，授权亚马逊将你的ASIN配送到加拿大和墨西哥。
4.  同意条款后，你需要手动为商品在目标站点添加报价。找到需要同步的商品，点击“编辑”。
5.  在编辑页面的“报价”部分最下方，找到加拿大和墨西哥的报价栏，填写价格等信息并保存。

完成上述步骤后，商品将在目标站点自动创建，并共享美国站的评价。

### 方法二：通过模板上传同步

若想将产品同步到欧洲、日本等站点，需要使用库存文件模板进行上传。核心在于在新站点的模板中填写原站点（如美国站）的 **ASIN** 和 **SKU**，以便系统识别为同一商品。

**关键操作点如下：**

*   在模板的 `product_id` 栏中填入原站点的 **ASIN**。
*   在 `sku` 栏中填入原站点的 **SKU**。
*   其他必填信息（如品牌、标题等）需根据目标站点要求填写。
*   在 `update_delete` 栏选择 **Update**。
*   如果原商品有变体，需在模板中正确填写变体关系。

**公式/代码表示核心概念：**
```
新站点模板中的 product_id = 原站点ASIN
新站点模板中的 sku = 原站点SKU
update_delete = Update
```

### 方法三：开通泛欧计划

泛欧计划允许卖家将库存放置在欧洲6国（如德国、法国、意大利等）中的任一国家，亚马逊会自动将订单配送至其他欧洲国家。这能简化欧洲多站点的物流管理。

**开通方式：**
1.  在卖家后台顶部搜索“泛欧计划”。
2.  进入相关页面，根据指引完成注册。
3.  通常需要已注册目标国家的VAT税号。

## 🧩 虚拟捆绑商品

了解了如何跨站点同步产品后，我们来看看如何在单一站点内通过“虚拟捆绑”来提升销售额和客单价。

虚拟捆绑商品允许你将2-3个互补或同类的独立ASIN组合在一起，创建一个新的销售链接。例如，将键盘和键盘垫捆绑销售。买家购买该捆绑链接后，亚马逊会分别从库存中拣货并一同配送。

![](img/e44ec6929956b63e97a87e34dc83cb00_5.png)

![](img/e44ec6929956b63e97a87e34dc83cb00_6.png)

**创建虚拟捆绑商品的前提：**
通常需要完成**品牌备案**。

**创建路径与要点：**
1.  在后台进入 **品牌 > 虚拟捆绑商品**。
2.  点击“创建新的虚拟捆绑商品”。
3.  选择要捆绑的2-3个ASIN。
4.  为捆绑商品设置独立的标题、主图（可展示组合效果）、SKU、价格、描述和卖点。

虚拟捆绑是一种灵活的促销方式，无需创建新的物理包装，即可推广关联产品。

## ⚠️ 危险品审核与类目审核

在创建或同步产品过程中，有时会遇到审核。本节我们将学习如何处理常见的危险品审核和类目审核。

### 危险品审核处理

如果你的产品被误判为危险品（如含电池、液体等），可按以下步骤申请豁免：

1.  在后台搜索“危险品审核”。
2.  找到需要审核的商品，点击“提供危险品文档”。
3.  根据商品情况选择上传 **SDS文件** 或 **豁免表**。对于普通误判，通常选择“不含有化学品的商品豁免表”。
4.  下载豁免表模板，如实填写商品信息（如ASIN、名称），在所有危险属性选项中选择“No”。
5.  上传填写好的豁免表，提交审核。通常3-7个工作日内会出结果。

### 类目审核检测与应对

某些类目（如美妆、玩具、母婴）需要额外资质（如FDA、CPC认证）才能销售。上架前，可通过以下方法检测目标类目是否需要审核：

**方法一：跟卖检测**
1.  找到目标竞品的ASIN。
2.  在后台“添加商品”处输入该ASIN，尝试跟卖。
3.  如果页面提示“需要获得批准”或要求提交认证文件，则说明该类目或品牌需要审核。

**方法二：自建Listing检测**
1.  参考竞品信息，手动创建一条测试Listing。
2.  使用竞品的类目，上传简易图片和标题。
3.  观察1-2天。如果Listing可售，则可能无需审核；如果被下架或要求提供证明，则触发审核。

**触发审核后的应对策略：**
*   如果确实是误判（如普通睫毛被误归为化妆品），可以在提交审核材料时，上传产品实物图并在备注中说明情况，请求人工重新审查。
*   如果确实需要认证，则需联系供应商获取相应的证书、检测报告或发票。

## 📝 本节课总结

本节课中我们一起学习了创建亚马逊产品的几种进阶方法：
1.  **跨站点同步商品**：利用远程配送计划、模板上传和泛欧计划，将产品快速拓展至其他站点，并共享评价。
2.  **创建虚拟捆绑商品**：通过品牌功能将互补产品捆绑销售，提高客单价。
3.  **处理审核**：掌握了危险品审核的豁免申请流程，以及检测和应对类目审核的实用技巧。
