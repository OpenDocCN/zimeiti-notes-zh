# 视频号运营教程：P19：18.活动介绍

## 概述
在本节课中，我们将详细介绍一个由视频号官方发起的商家扶持活动。我们将了解该活动的背景、核心负责人、提供的具体资源以及参与步骤。本次活动旨在为视频号带货商家提供实质性的帮助。

## 活动背景与负责人
为促进2024年中国电商行业的新发展，并为视频化带货打下技术基础，视频号官方特推出商家联盟活动。本次活动由短视频教研部负责人张良老师全程对接，并以公平公正为核心理念展开。

![](img/f7d8b0beda677d0521128e8ac98ced38_1.png)

张老师作为本次活动的全国总负责人，拥有独家对接权限。这意味着，只有通过张老师才能对接本次活动资源。

## 活动核心资源介绍
上一节我们介绍了活动背景，本节中我们来看看本次活动提供的具体资源。这些资源旨在解决商家在运营中遇到的实际问题。

### 1. 官方视频素材支持
许多商家面临素材拍摄困难的问题。本次活动官方已为参与者准备好了高质量的视频素材。

以下是素材支持的具体内容：
*   每个参与账号将获得**30条**成品视频素材。
*   商家只需每天发布一条即可。

### 3. 流量扶持与保底收益
除了素材，流量和启动资金也是商家的关键需求。本次活动在此方面提供了强力支持。

以下是流量与收益扶持政策：
*   **流量扶持**：每个账号可获得**90万**流量扶持。
*   **保底收益**：发布一条视频即可获得**300元**保底收益。
*   **结算周期**：收益为日结模式，今天发布，明天到账。
    ![](img/f7d8b0beda677d0521128e8ac98ced38_14.png)
    ![](img/f7d8b0beda677d0521128e8ac98ced38_15.png)

根据以上政策，若发布全部30条素材，总保底收益计算如下：
`300元/条 * 30条 = 9000元`

![](img/f7d8b0beda677d0521128e8ac98ced38_14.png)

![](img/f7d8b0beda677d0521128e8ac98ced38_15.png)
