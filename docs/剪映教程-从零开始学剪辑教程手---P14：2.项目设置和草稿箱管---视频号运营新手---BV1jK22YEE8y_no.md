# 剪映教程：P14：项目设置与草稿箱管理 📁

在本节课中，我们将学习如何对剪辑项目进行基础设置，以及如何有效管理你的草稿箱。这些操作虽然简单，但却是高效使用剪映、确保视频质量的基础。

## 项目设置 ⚙️

上一节我们了解了剪映的界面，本节中我们来看看如何为你的视频项目进行初始设置。正确的项目设置是保证视频清晰度和专业度的第一步。

![](img/d79e844ce8ea05ee3c802ab69d1463a0_1.png)

进入项目设置页面的方法是：点击右上角的**设置**图标。

![](img/d79e844ce8ea05ee3c802ab69d1463a0_1.png)

在设置页面中，主要有以下几个关键选项需要配置：

以下是项目设置的核心选项列表：
1.  **视频分辨率**：剪映提供两种分辨率选项，`1080P` 和 `720P`。目前国内主流视频平台均支持 `1080P` 分辨率，它能提供更清晰的画质。因此，建议始终将视频分辨率设置为 **`1080P`**。
2.  **自动添加片尾**：此选项控制是否为视频自动添加剪映的官方片尾。片尾样式如图所示。为了视频的简洁和专业，**推荐关闭此选项**。
3.  **“剪同款”水印**：在使用“剪同款”模板时，此选项控制是否添加水印。由于水印不支持自定义，且可能影响视频观感，**同样推荐关闭此选项**。

完成以上设置后，点击左上角的**叉号**退出，项目属性便设置完毕。

## 草稿箱管理 📂

完成项目设置后，我们再来看看如何管理你的工作成果——草稿箱。良好的草稿管理习惯能帮助你随时回溯和修改作品。

剪映的草稿箱主要分为两类：
*   **剪辑草稿**：来源于点击 **`开始创作`** 后，手动一步步剪辑完成的视频项目。
*   **模板草稿**：来源于在 **`剪同款`** 功能中，套用模板生成的视频项目。

以下是管理草稿箱的具体步骤：
1.  在草稿箱页面，点击右上角的 **`管理`** 按钮。
2.  此时可以勾选需要处理的草稿。
3.  选择底部的 **`删除`** 按钮，即可移除选中的草稿。

**重要提示**：虽然删除草稿可以节省手机存储空间，但已发布的视频若后续需要修改，就必须依赖草稿箱中的原始文件。因此，如果手机存储空间充足，**建议保留这些草稿文件**，以备不时之需。
