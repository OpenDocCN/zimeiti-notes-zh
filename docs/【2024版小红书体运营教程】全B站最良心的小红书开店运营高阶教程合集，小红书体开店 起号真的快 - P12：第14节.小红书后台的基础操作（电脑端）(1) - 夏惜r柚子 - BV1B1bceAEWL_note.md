# 小红书开店运营教程：P12：第14节. 小红书后台的基础操作（电脑端）(1) 🖥️

在本节课中，我们将学习如何通过电脑端操作小红书商家后台。我们将重点介绍几个核心的后台设置，包括保证金、支付方式、运费保以及地址管理。掌握这些基础操作是顺利开店和后续运营的前提。

![](img/bdb1e008d1be3956b3ba50cff2b40aec_1.png)

## 一、 保证金 💰

![](img/bdb1e008d1be3956b3ba50cff2b40aec_3.png)

上一节我们介绍了开店的基本流程，本节中我们来看看店铺运营中一个重要的资金环节——保证金。

保证金是平台为确保商家履约和保障消费者权益而收取的押金。根据所选经营类目的不同，保证金的金额也会有所差异。

**核心操作路径**：登录小红书商家后台（千帆系统），在左侧导航栏找到 **`资金`** -> **`店铺保证金管理`**。

以下是关于保证金的几个要点：

![](img/bdb1e008d1be3956b3ba50cff2b40aec_5.png)

*   **金额说明**：个人店铺的保证金通常在1000元左右。具体金额需根据您选择的商品类目确定，可以在后台的保证金管理页面查询。
*   **缴纳与退还**：保证金在店铺关闭且无未完结纠纷的情况下可以申请退还。平台会在商家违规或未履行对消费者的承诺（如未及时退款）时扣罚保证金。
*   **前期建议**：开店前期可以不立即缴纳保证金。但若想开通微信支付功能或计划长期经营，建议及时缴纳。
*   **店铺类型差异**：部分特殊类目（如二手奢侈品、珠宝玉石、家用电器等）不支持个人店入驻，需升级为个体店或企业店，其保证金金额也相应更高。

## 二、 支付方式开通 💳

![](img/bdb1e008d1be3956b3ba50cff2b40aec_7.png)

了解了保证金后，下一步就是开通收款渠道，让顾客可以顺利付款。

![](img/bdb1e008d1be3956b3ba50cff2b40aec_9.png)

支付方式的开通直接影响买家的购物体验。小红书支持支付宝和微信支付两种主流方式。

![](img/bdb1e008d1be3956b3ba50cff2b40aec_11.png)

**核心操作路径**：在商家后台，进入 **`资金`** -> **`账户资金`** 页面，即可找到支付宝和微信支付的开通入口。

以下是开通支付方式的注意事项：

![](img/bdb1e008d1be3956b3ba50cff2b40aec_13.png)

*   **开通条件**：**缴纳店铺保证金是开通微信支付的前提条件**。未缴纳保证金则无法开通微信收款。
*   **开通流程**：
    *   支付宝支付：通常可以即时开通并绑定。
    *   微信支付：需在缴纳保证金后的第二天进行设置和绑定。
*   **结算账户**：提现金额将结算到开店时注册的法人身份对应的支付宝或微信账户中。个体店或企业店需绑定对公账户或营业执照上的法人账户。

## 三、 运费保设置 🛡️

![](img/bdb1e008d1be3956b3ba50cff2b40aec_15.png)

开通支付方式后，我们可以设置一些提升店铺竞争力的服务，比如运费保。

运费保是一项平台服务，旨在降低消费者的下单顾虑，从而可能提升订单转化率。

![](img/bdb1e008d1be3956b3ba50cff2b40aec_17.png)

**运费保的作用是**：当消费者发生退货退款并成功后，系统会自动补贴运费，卖家无需承担此部分费用。

**核心操作路径**：在商家后台，进入 **`售后`** -> **`运费保`** 页面，点击开通全店运费保服务。

建议开通此服务，因为它能有效打消买家对运费的顾虑，提升购买意愿，同时降低因运费问题产生的交易纠纷。

## 四、 地址与物流设置 📦

最后，我们来设置发货和退货地址，这是完成订单履约的关键一步。

![](img/bdb1e008d1be3956b3ba50cff2b40aec_19.png)

![](img/bdb1e008d1be3956b3ba50cff2b40aec_21.png)

地址设置关系到发货效率和退货流程，对于无货源（一件代发）模式的商家尤为重要。

![](img/bdb1e008d1be3956b3ba50cff2b40aec_23.png)

**核心操作路径**：在商家后台，进入 **`订单`** -> **`物流工具`** -> **`地址库管理`**，可以新建和编辑发货/退货地址。

![](img/bdb1e008d1be3956b3ba50cff2b40aec_25.png)

以下是设置地址的实用技巧：

*   **退货地址策略**：对于无货源代发商家，退货地址可以填写一个偏远地区（如新疆），并在地址后备注“**退货前请联系客服**”。这样做的目的是引导消费者在退货前先沟通，方便您协调上游供应商处理。
*   **物流模板**：在同一页面，您还可以设置运费模板，根据地区设置不同的邮费政策。
*   **与手机端同步**：电脑端的地址管理与手机端设置是同步的，您可以选择在任一端口进行操作。
