# 抖音运营：P19：口播类视频拍摄方法

## 概述
在本节课中，我们将学习口播类视频的具体拍摄方法。我们将通过一个成功案例，详细拆解拍摄时需要注意的构图、内容定位、设备使用等核心要点，帮助你掌握制作高质量口播视频的技巧。

---

## 拍摄的核心注意事项
拍摄环节需要注意以下几点：**构图比例**、**主题突出**、**事物清晰**、**光线色彩**和**内容连贯**。这些原则适用于所有类型的视频拍摄。

![](img/18cf25dfd3dacb5cefe550b12d474e62_1.png)

为了便于理解，我们将以两种最常见的视频类型为例进行讲解：**口播类视频**和**日常生活类视频**。这两种类型几乎涵盖了所有的拍摄方法。本节我们将重点分析口播类视频的拍摄。

---

## 口播视频案例解析
首先，我们来看一个口播类型的成功案例。这位学员的账号名为“种小草”，她是一名40岁、带着三个孩子的宝妈，专注于情感口播领域。她的一条视频播放量超过8000万，并凭借该视频涨粉30万。

为什么她的一条视频能涨粉30万，而我们涨粉却如此困难？关键在于她吸引了**针对性极强的目标群体**。她的视频内容精准地触动了特定人群的共鸣。

### 1. 构图与画幅
抖音平台采用**9:16**的竖屏比例。在拍摄时，你需要决定视频主体是横屏还是竖屏。案例中的视频采用了**4:3**的横屏构图。

**核心要点**：保持画幅统一。如果你选择横屏，那么所有视频都应采用横屏；如果选择竖屏，则全部使用竖屏。风格统一是专业度的体现。

在构图上，人物主体应占据画面中央约**70%** 的面积。避免怼脸拍摄，需要在人物的左、右、上方适当留白，使画面看起来清晰、舒适。

### 2. 人物与内容定位
你可能会问：如果自己相貌普通，能否做口播？答案是肯定的。抖音的成功并非完全依赖颜值。对于口播视频而言，核心是**传递的内容和价值**。只要做到形象清晰、体面即可。

![](img/18cf25dfd3dacb5cefe550b12d474e62_3.png)

如果对自己的外貌不自信，可以适当开启美颜滤镜。在这个时代，合理使用美颜工具能让每个人看起来都更精神。请注意，效果不宜过于夸张。

### 3. 目标群体分析
现在，我们通过案例视频的具体内容，来分析她吸引的是哪个群体。

> **视频文案示例**：
> “跟婆婆在一起住是什么样的感受？我跟我婆婆在一起住了16年了，平时磕磕碰碰有点小不愉快，那都是正常的……我最大的感受就是这十多年以来，我娘家人很少在我家住上几天。”

这段内容主题是“婆媳关系”。那么，谁会对这个话题最有感触？显然是**已婚女性群体**。未婚或年轻的女性很难对此产生深刻共鸣。因此，这条视频精准地吸引了已婚、对家庭关系有切身感受的女性用户。

### 4. 拍摄场景与准备
拍摄口播视频对场景要求灵活。你可以在户外拍，也可以在家拍。关键点是**背景要整洁美观**，避免杂乱无章的画面影响观感。

对于担心记不住台词的同学，这里有一个实用技巧：你不需要背诵。在剪辑环节，可以使用提词器工具，实现**边看边录**，这能大大降低拍摄难度。

### 5. 收音设备建议
口播视频的另一关键是**声音清晰**。为了保证音频质量，建议你购买一个手机收声麦克风。这种设备价格便宜（约十几元），在淘宝、京东等平台都能买到，能显著提升视频的专业度。

## 总结
本节课我们一起学习了口播类视频的拍摄方法。我们总结出以下几个核心步骤：
1.  **确定统一画幅**（如4:3横屏）并保持构图合理。
2.  **明确内容定位**，吸引精准的目标用户群体。
3.  **注重个人形象**，做到清晰体面，可合理使用美颜。
4.  **选择整洁的拍摄背景**。
5.  **利用提词器**解决记台词难题。
6.  **使用收声麦克风**保证音频清晰。

掌握这些方法，你就能更专业、更高效地制作出优质的口播视频。下节课，我们将继续讲解日常生活类视频的拍摄技巧。
