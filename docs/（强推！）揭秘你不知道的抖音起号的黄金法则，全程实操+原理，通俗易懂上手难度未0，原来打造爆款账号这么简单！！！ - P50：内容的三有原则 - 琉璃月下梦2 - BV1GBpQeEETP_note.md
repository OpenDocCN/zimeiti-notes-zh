## 总结：内容的三有原则 🎯

通过以上分析，我们可以总结出一个全网通用的内容创作核心原则，即“三有原则”。

![](img/4c0ba8fdd41295515f4c10d0c138a4c7_19.png)

![](img/4c0ba8fdd41295515f4c10d0c138a4c7_21.png)

**三有原则 = 有用处 + 有兴趣 + 有共鸣**

![](img/4c0ba8fdd41295515f4c10d0c138a4c7_23.png)

目前抖音上受欢迎的视频内容，基本都符合这三点中的一个或多个。以下是常见内容类型与“三有原则”的对应关系：

*   **有用处类**：教程、推荐、干货分享等。
*   **有兴趣类**：高颜值、萌宠、炫技、情景剧、美好事物展示等。
*   **有共鸣类**：励志故事、情感鸡汤（包括反鸡汤、毒鸡汤）、引发共同感慨的生活洞察等。

理解这个原则后，我们在构思内容时，就可以有意识地从这三个方向进行延伸和创作。

---

看到这么多内容类型，你可能会感到困惑：人设、教程、推荐等九大类，我到底该做哪一类？哪一类才适合我的商业定位和内容定位？

先别急于决定。在深入选择具体类型前，我们还需要理解另一个关键概念。让我们先来看一个案例视频。

![](img/4c0ba8fdd41295515f4c10d0c138a4c7_19.png)

![](img/4c0ba8fdd41295515f4c10d0c138a4c7_21.png)

（视频为一个演唱视频，制作精良）
这个视频本身质量不错，画面和演唱都很好，点赞量也尚可。但当我们查看其账号主页时，会发现一个问题：获赞5.8万，粉丝却只有5173。

**计算公式：粉丝转化率 ≈ 总获赞数 / 粉丝数**
这个账号的比值明显偏高，意味着很多用户点赞后并没有选择关注。

为什么会出现“有赞无粉”的情况？可能你会想到“内容不垂直”。垂直，意味着账号持续专注于同一领域（如只做唱歌、只做美食）。对于大多数普通创作者而言，垂直是必要的，因为用户通常因内容价值而关注，而非创作者本人。

但这个案例账号的封面显示他一直在做唱歌内容，似乎是垂直的。那么问题出在哪里？

主要有两方面原因：
1.  **主题/IP辨识度问题**。
2.  **账号整体搭建问题**。

我们将在后续章节中详细拆解这两个问题。
