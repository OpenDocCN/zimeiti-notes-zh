# 抖音运营教程：P18：16、口播类视频拍摄方法-7

## 概述
在本节课中，我们将学习口播类视频的具体拍摄方法。我们将通过一个成功案例，详细拆解其构图、内容定位、拍摄技巧等核心要素，帮助你掌握制作高质量口播视频的关键。

---

## 拍摄的核心注意事项
进入拍摄环节，需要重点关注以下几点：**构图比例**、**主题突出**、**事物清晰**、**光线色彩**和**内容连贯**。这些要点看似抽象，接下来我们将通过具体案例进行讲解。

![](img/46b90f8e5f734de843c32b28175b7e85_1.png)

视频拍摄主要分为两大类：**口播类型**和**日常生活记录类型**。本节课，我们将聚焦于口播类型的拍摄方法。口播视频的拍摄逻辑是通用的，无论你从事金融、装修、养生还是情感领域，其拍摄方法都一致。

## 案例解析：情感口播账号“种小草”
首先，我们分析一个学员的成功案例，账号名为“种小草”。这是一个情感口播账号，运营者是一位40岁、带着三个孩子的宝妈。她的一条视频播放量超过8000万，并凭借该视频增长了30万粉丝。

为什么她的视频能如此成功？核心在于**精准吸引目标群体**。她的视频内容能让人一眼看出其想要吸引的受众。例如，如果你的目标是20多岁的学生群体，那么你拍摄的所有内容都应围绕学生感兴趣的话题，这样涨粉才会又快又精准。

### 1. 构图与画幅比例
抖音平台采用**9:16**的竖屏格式。关键在于视频内容本身是选择横屏还是竖屏。以“种小草”为例，她使用的是**4:3的横屏构图**。

**重要提示**：制作视频时，画幅比例需要保持统一。如果你选择横屏，那么所有视频都应采用横屏；如果选择竖屏，则全部使用竖屏。保持一致性是需要注意的细节。

在构图上，人物主体应占据画面中央约**70%** 的面积。避免将脸部怼得过近拍摄，那样会影响观感。确保画面左、右、上方留有适当空白，这样会使画面看起来更清晰、舒适。

### 2. 出镜形象与设备
如果你认为自己外貌不够出众，是否还能做口播？答案是肯定的。抖音的成功并非完全依赖颜值。口播的核心在于**内容传递**和**知识分享**，只要让观众觉得你形象清晰、得体即可。

对于外貌不自信的创作者，可以开启美颜滤镜。在这个时代，合理使用美颜功能能有效提升形象。请放心使用，只要不过度夸张即可。

![](img/46b90f8e5f734de843c32b28175b7e85_3.png)

此外，为了保证声音清晰，建议购买一个**手机收声麦克风**。这类设备价格便宜，只需十几元，在淘宝、京东等平台均可购得。

### 3. 内容与台词
对于口播视频，另一个常见问题是：台词需要背诵吗？如果记忆力不好，背不下来怎么办？

解决方案在剪辑环节。你可以借助提词器工具，实现**边看边读边录制**，这大大降低了拍摄难度。我们后续在剪辑课程中会推荐相关工具。

### 4. 目标群体定位分析
现在，我们来分析“种小草”那条涨粉30万的视频，具体看看她吸引了哪个群体。

> **视频文案示例**：
> “跟婆婆在一起住是什么样的感受？我跟我婆婆在一起住了16年了，平时磕磕碰碰有点小不愉快，那都是正常的……我最大的感受就是这十多年以来，我娘家人很少在我家住上几天，这对于我内心来说其实是一个很大的缺失。”

这段内容围绕“婆媳关系”展开。那么，谁会对这个话题最有感触？显然是**已婚女性群体**。未婚或年轻的女性对此难以产生深刻共鸣，而男性群体的感受也不如女性直接。因此，这个账号成功吸引了**已婚女性**这一精准受众，从而引发强烈共鸣，实现数据爆发。

## 总结
本节课我们一起学习了口播类视频的拍摄方法。我们通过“种小草”的案例，重点剖析了：
1.  **构图**：统一使用**4:3横屏**或**9:16竖屏**，人物居中占比70%，画面留白。
2.  **形象**：内容重于颜值，得体即可，可借助美颜滤镜和**收声麦克风**提升质量。
3.  **台词**：无需背诵，可借助工具边读边录。
4.  **定位**：内容是**精准吸引目标群体**的关键，如案例中针对“已婚女性”谈论“婆媳关系”。

掌握这些要点，你就能迈出制作优质口播视频的第一步。下一节，我们将继续讲解日常生活类视频的拍摄技巧。
