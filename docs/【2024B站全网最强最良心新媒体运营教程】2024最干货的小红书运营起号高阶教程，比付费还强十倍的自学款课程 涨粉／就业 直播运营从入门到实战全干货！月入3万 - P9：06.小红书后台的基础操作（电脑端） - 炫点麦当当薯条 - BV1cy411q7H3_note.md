# 小红书运营：P9：06.小红书后台的基础操作（电脑端） 💻

在本节课中，我们将学习如何通过电脑端操作小红书商家后台，涵盖保证金、支付方式、运费保和地址设置等核心功能。这些是店铺运营的基础，掌握它们能帮助你更顺畅地管理店铺。

![](img/1c212d92f827f864d39a340a31e84fb8_1.png)

上一节我们介绍了手机端的操作，本节中我们来看看电脑端后台的具体设置。

![](img/1c212d92f827f864d39a340a31e84fb8_3.png)

## 保证金 💰

保证金是开店时需要缴纳的一笔押金，用于保障消费者权益。前期可以暂不缴纳，但若想长期经营或开通更多支付方式，则建议缴纳。

以下是关于保证金的几个要点：

![](img/1c212d92f827f864d39a340a31e84fb8_5.png)

*   **金额**：个人店铺的保证金通常在**1000元**左右，具体金额根据所选经营类目而定。企业店铺的保证金则在2000元至数万元不等。
*   **缴纳与退还**：保证金在店铺关闭后可以申请退还。但若店铺存在违规行为（如未履行退款承诺），平台会从中扣款。
*   **必要性**：缴纳保证金是开通微信支付功能的必要条件。许多用户习惯使用微信支付，因此缴纳保证金有助于提升店铺的支付便利性和转化率。

**操作路径**：登录小红书千帆后台 -> 点击左侧“资金” -> 选择“店铺保证金管理” -> 选择对应类目并充值。

## 支付方式开通 💳

开通多种支付方式可以方便不同支付习惯的顾客下单，提升购买体验。

![](img/1c212d92f827f864d39a340a31e84fb8_13.png)

以下是两种主要支付方式的开通说明：

*   **支付宝支付**：提交资料后通常可以即时开通。
*   **微信支付**：需要先缴纳店铺保证金，审核通过后次日方可开通。提现金额将进入店铺注册时使用的法人账户。

![](img/1c212d92f827f864d39a340a31e84fb8_15.png)

**操作路径**：登录小红书千帆后台 -> 点击左侧“资金” -> 在“支付设置”中分别开通支付宝与微信支付。

![](img/1c212d92f827f864d39a340a31e84fb8_17.png)

![](img/1c212d92f827f864d39a340a31e84fb8_15.png)

## 运费保设置 🛡️

运费保是一项平台服务，开通后，若消费者符合条件退货，系统会自动补贴运费。这能有效打消顾客对运费的顾虑，提升订单转化率，并降低售后纠纷。

**核心作用**：`提升转化率，降低纠纷率`。

![](img/1c212d92f827f864d39a340a31e84fb8_19.png)

**操作路径**：登录小红书千帆后台 -> 点击左侧“售后” -> 找到“运费保”服务并点击开通。

![](img/1c212d92f827f864d39a340a31e84fb8_21.png)

![](img/1c212d92f827f864d39a340a31e84fb8_19.png)

![](img/1c212d92f827f864d39a340a31e84fb8_23.png)

## 地址设置 📍

![](img/1c212d92f827f864d39a340a31e84fb8_25.png)

正确设置发货和退货地址对于订单处理和售后至关重要。对于采用“一件代发”模式的无货源商家，需要特别注意。

![](img/1c212d92f827f864d39a340a31e84fb8_27.png)

以下是地址设置的关键点：

![](img/1c212d92f827f864d39a340a31e84fb8_29.png)

*   **退货地址**：建议填写供应商地址（如一些偏远地区仓库），并在地址后附加“**联系客服**”字样。这样既能引导顾客先联系你，又能将退货直接导向货源方。
*   **操作路径**：登录小红书千帆后台 -> 点击左侧“订单”下的“物流工具” -> 点击“新建地址”进行设置。

![](img/1c212d92f827f864d39a340a31e84fb8_27.png)

![](img/1c212d92f827f864d39a340a31e84fb8_31.png)

本节课中我们一起学习了小红书电脑端后台的四个基础操作：保证金的查看与缴纳、支付方式的开通、运费保服务的设置以及发货/退货地址的配置。这些设置是店铺正常运转的基石，请务必根据自身情况妥善完成。熟练掌握后台操作，能为你的店铺运营打下坚实基础。
