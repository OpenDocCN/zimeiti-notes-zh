## 人设定位：我是谁？🤔

![](img/268ad3b3ac7325ce8d7ce461a09350b0_35.png)

上一节我们介绍了账号定位的重要性，本节中我们首先来看看人设定位。人设定位的目的是让用户清楚地知道，提供内容的人是谁，有什么样的背景和经历。

**人设定位的核心目标有两个：**
1.  **建立专业性**：让用户相信你是在特定领域长期深耕的专家，你提供的内容具有参考价值。
2.  **增强记忆点**：让用户能记住你，在众多账号中脱颖而出。

*   **基础形象**：即“账号五件套”，包括头像、名称、简介、背景图和封面。这些元素需要统一风格，清晰传达你的领域。
*   **内容展现**：在视频或图文内容中，通过标准的开场动作、固定的口头禅、统一的视频节奏或背景音乐，形成独特的个人风格。

关于人设，有一个关键问题需要思考：是否所有类型的账号都必须打造强人设？例如，影视解说类账号，用户记住的往往是内容本身（如“XX说电影”这个栏目），而非背后的具体人物。因此，人设的强弱需根据赛道特性来决定。

## 内容定位：我该发什么？📝

![](img/268ad3b3ac7325ce8d7ce461a09350b0_185.png)

有了人设和商业方向，接下来就需要确定内容定位，即具体发布什么类型的内容来实现目标。内容定位需要解决三个核心问题：

![](img/268ad3b3ac7325ce8d7ce461a09350b0_197.png)

![](img/268ad3b3ac7325ce8d7ce461a09350b0_199.png)

![](img/268ad3b3ac7325ce8d7ce461a09350b0_201.png)

![](img/268ad3b3ac7325ce8d7ce461a09350b0_203.png)

![](img/268ad3b3ac7325ce8d7ce461a09350b0_205.png)

![](img/268ad3b3ac7325ce8d7ce461a09350b0_207.png)

1.  **内容主题**：我单篇内容想表达什么？是分享知识、记录生活，还是吐槽观点？主题必须清晰。
2.  **内容吸引力**：我的内容如何吸引用户？在同质化竞争中，你的内容差异化和亮点在哪里？
3.  **平台推流**：我的内容如何获得平台更多的推荐？这需要研究当前平台内，你所在赛道下，哪类细分话题是用户最关心、流量最大的。

![](img/268ad3b3ac7325ce8d7ce461a09350b0_209.png)

![](img/268ad3b3ac7325ce8d7ce461a09350b0_211.png)

![](img/268ad3b3ac7325ce8d7ce461a09350b0_213.png)

![](img/268ad3b3ac7325ce8d7ce461a09350b0_215.png)

![](img/268ad3b3ac7325ce8d7ce461a09350b0_217.png)

![](img/268ad3b3ac7325ce8d7ce461a09350b0_219.png)

**代码示例（内容定位逻辑）：**
```python
if 内容主题清晰 and (内容有差异化亮点 or 切中平台热门细分话题):
    获得初始推荐流量的概率更高
else:
    可能陷入同质化竞争，起步较难
```

**方案A：育儿经验分享型**
*   **人设定位**：一位拥有3岁宝宝的妈妈，分享真实的生娃、养娃、带娃经历。
*   **商业定位**：推荐纸尿裤、奶瓶、童装、食品等宝宝用品。
*   **内容定位**：解决宝妈们的具体痛点，如“孩子不爱吃奶粉怎么办？”、“宝宝睡眠不规律如何调整？”。

**方案B：产后恢复型**
*   **人设定位**：一位产后体重暴涨30斤，但成功恢复身材甚至有所提升的宝妈。
*   **商业定位**：推荐产后恢复产品，如塑身衣、护肤品、身材管理计划等。
*   **内容定位**：分享“产后修复标准动作”、“妊娠纹科学处理”、“应对产后情绪焦虑”等知识。

![](img/268ad3b3ac7325ce8d7ce461a09350b0_281.png)

![](img/268ad3b3ac7325ce8d7ce461a09350b0_289.png)

![](img/268ad3b3ac7325ce8d7ce461a09350b0_291.png)

这两个方案虽然同属母婴赛道，但人设、商业和内容的侧重点完全不同。更重要的是，方案B的账号在主打“产后恢复”的同时，也可以适当结合方案A的“育儿知识”，因为吸引的都是宝妈群体。这体现了定位的灵活性：在垂直深耕核心领域的同时，可以横向拓展关联内容，丰富账号维度，覆盖更广泛的用户需求。

         -> (可选) 在垂直领域基础上，横向拓展关联内容

本节课中，我们一起学习了新媒体账号定位的三个核心组成部分：**人设定位**、**商业定位**和**内容定位**。账号定位就像航海时的指南针，为你的所有运营动作指明方向。一个清晰、精准的定位，能帮助你建立专业形象，吸引目标用户，并规划出可行的变现路径。在开始创作内容之前，务必花时间思考并确定好这三大定位，这是账号能否做起来的第一步，也是最重要的一步。
