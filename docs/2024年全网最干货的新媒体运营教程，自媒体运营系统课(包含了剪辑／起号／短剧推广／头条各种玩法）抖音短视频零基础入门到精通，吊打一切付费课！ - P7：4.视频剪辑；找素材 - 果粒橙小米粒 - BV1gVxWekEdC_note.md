# 新媒体运营系统课：P7：4.视频剪辑：找素材 🎬

![](img/26595e6baea2a88ace6fb505cfb07fb8_0.png)

在本节课中，我们将要学习视频剪辑类内容创作中至关重要的一步：如何合法、高效地寻找素材。我们将明确哪些素材可以安全使用，并掌握从多个渠道获取素材的具体方法。

---

![](img/26595e6baea2a88ace6fb505cfb07fb8_2.png)

上一节我们介绍了视频剪辑类内容的广泛性，本节中我们来看看如何为这类内容寻找合适的素材。

![](img/26595e6baea2a88ace6fb505cfb07fb8_4.png)

首先必须强调，我们制作视频**不能进行搬运**。搬运视频会导致两个严重后果：**1. 审核不通过**；**2. 审核通过后，因重复度过高被限流**。这会使账号无法成长。

因此，我们遇到的第一个核心问题就是：如何为视频剪辑类内容寻找素材？

以下是寻找素材的三个主要渠道：

**1. 所有视频平台**
第一个渠道是各大视频与自媒体平台。我曾提供过一份包含30多个自媒体平台的列表，这就是我们寻找素材的第一个宝库。不要只局限于一个平台（例如只从抖音找），每个平台的用户偏好和内容风格都有差异，广泛搜索能获得更丰富的素材。

**2. 影视及各类垂直网站**
第二个渠道是专门的网站。
*   **影视网站**：例如“豆瓣电影”。其优势在于，当你搜索一部影视剧（如《狂飙》）时，如果详情页标注了“**爱奇艺独家播放**”等信息，则说明该内容有明确版权，不能随意剪辑使用。反之，没有此类标注的内容，版权风险相对较低。此外，我还整理了近50个国内外影视资源网站。
*   **其他垂直类网站**：如果你不想做影视剪辑，而对动漫、综艺、搞笑、旅游、体育、美食等内容感兴趣，我也整理了超过60个相关网站，涵盖图片、音效、素材等，能满足各类创作需求。

![](img/26595e6baea2a88ace6fb505cfb07fb8_6.png)

**3. 影视推广平台**
第三个渠道是影视推广平台。这是版权方（电影、电视剧出品方）为了宣传曝光，将正版内容授权给自媒体创作者进行二次创作和推广的平台。其核心优势有两点：
*   **版权无忧**：平台内的所有影视内容均**有版权授权**，创作者可放心使用。
*   **有播放收益**：在抖音、快手等通常不提供播放量收益的平台，通过参与此类推广任务并提交任务链接，创作者可以**依据播放量结算奖金**，实现流量变现。

![](img/26595e6baea2a88ace6fb505cfb07fb8_8.png)

![](img/26595e6baea2a88ace6fb505cfb07fb8_10.png)

很多同学可能会有疑问：版权方为什么不自己发广告，而要让我们来做？

![](img/26595e6baea2a88ace6fb505cfb07fb8_12.png)

原因在于成本与效果。在传统渠道（如城市中心商圈）投放广告，费用动辄十万甚至百万起步，且现代用户对广告普遍抵触，曝光效果有限。而通过自媒体创作者生产内容，能在用户聚集的视频平台获得更自然、更高频的曝光，成本效益比更高。

![](img/26595e6baea2a88ace6fb505cfb07fb8_14.png)

上一部分我们明确了找素材的渠道，接下来我们看看找到素材后，具体该如何正确使用。

我们以在“西瓜视频”搜索电视剧《狂飙》为例，演示正确流程。搜索后，你会得到大量相关视频内容。

**关键问题：能否直接下载整个视频使用？**
答案是不能。直接下载使用就等同于搬运，会触发平台审核机制。

正确的素材使用方法是：
1.  **片段化提取**：从大量视频中，寻找符合你视频主题的片段。
2.  **控制时长**：每个提取的片段**时长不应超过10秒**。超过这个时长，平台算法极易判定为重复内容。
3.  **控制数量**：一个成片大约使用**6个这样的片段**为宜。
4.  **融入创作**：虽然素材非原创，但在筛选、组合片段时，必须加入自己的思路和想法，形成新的叙事或观点。这才算得上是经过创作的二次创作视频。

最后，关于实操工具的选择，你可以根据习惯使用手机或电脑进行剪辑。

本节课中我们一起学习了视频剪辑的素材寻找之道。我们明确了搬运的危害，掌握了从**视频平台、垂直网站、影视推广平台**三大渠道获取素材的方法，并学会了如何通过**片段化提取（每个片段<10秒）和融入个人思路**来合法、安全地使用素材，将找到的素材转化为属于自己的原创作品。记住，思路和加工过程才是你作品的核心价值。
