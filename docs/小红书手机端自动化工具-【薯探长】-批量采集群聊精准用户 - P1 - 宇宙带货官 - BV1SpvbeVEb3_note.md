“薯探长”是一款专为小红书手机端设计的自动化工具，其主要功能是批量采集小红书群聊中的精准用户信息。这对于希望进行精准营销、社群运营或用户分析的用户来说，是一个高效的辅助工具。

## 准备工作

![](img/f8422710e851e0579f9bc089a94c824a_1.png)

为了顺利运行“薯探长”工具，你需要确保满足以下条件。以下是具体的要求列表：

1.  **一部安卓手机**：工具目前主要支持安卓系统的手机端小红书应用。
2.  **开启手机的开发者选项与USB调试模式**：这是让电脑能够控制手机的必要步骤。
    *   通常可以在“设置”->“关于手机”->连续点击“版本号”7次来开启开发者选项。
    *   然后在“开发者选项”中开启“USB调试”。
3.  **在电脑上安装必要的驱动和自动化框架**：例如ADB（Android Debug Bridge）工具。
4.  **获取“薯探长”工具脚本或程序**：你需要拥有该工具的安装包或脚本文件。
