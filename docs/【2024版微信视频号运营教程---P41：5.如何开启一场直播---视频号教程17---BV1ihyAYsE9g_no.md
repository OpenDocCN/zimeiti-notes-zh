# 微信视频号运营教程：P41：5.如何开启一场直播 🎥

在本节课中，我们将学习如何从零开始，在微信视频号中发起并完成一场直播。我们将涵盖直播前的准备工作、直播中的关键操作以及直播后的收尾步骤。

## 直播前的准备工作

![](img/e3620f4ef265b63058779731405bc8fb_1.png)

上一节我们介绍了视频发布与预热，本节中我们来看看直播前需要完成哪些具体准备。

![](img/e3620f4ef265b63058779731405bc8fb_2.png)

![](img/e3620f4ef265b63058779731405bc8fb_3.png)

![](img/e3620f4ef265b63058779731405bc8fb_4.png)

确定好要推广的商品并发布相关视频后，需要等待视频获得一定的初始流量。当视频播放量达到约1000左右时，即可准备开播。

以下是开播前必须完成的准备工作列表：
*   **准备直播话术**：提前设计好直播时要讲的内容。
*   **准备备用视频**：为防止开播初期无人进入直播间，需提前准备几个视频并保存到草稿箱，以便在直播时补发，吸引流量。
*   **创建直播预告**：如果当前不方便立即直播，但视频流量已起，务必先创建一个直播预告。预告中需明确未来的开播时间。这样，感兴趣的观众可以提前预约，等你开播时他们会第一时间收到提醒。

![](img/e3620f4ef265b63058779731405bc8fb_6.png)

## 发起直播与基础设置

准备工作就绪后，就可以正式发起直播了。点击“发起直播”按钮，进入直播设置界面。

![](img/e3620f4ef265b63058779731405bc8fb_8.png)

以下是直播初始设置的关键步骤：
*   **设置直播主题**：主题一般可写为“视频同款热卖中”。你也可以参考其他成功账号的写法。
*   **添加商品**：在“商品”选项中，提前添加你计划售卖的商品。可以添加单个主打商品。
*   **设置画面效果**：在“画面”设置中，可以使用美颜、滤镜、人物特效等功能。
*   **添加直播贴图**：可以使用上节课提到的“微微商水印相机”等工具制作简单的介绍贴图。具体做法可参考对标账号。
*   **设置直播模式**：如果是教学或测试，可将直播模式改为“部分可见”，避免无关用户进入影响流程。对于卖货直播，通常选择公开模式。
*   **标记位置**：卖货直播需要标记地理位置，可随意选择一个位置。

![](img/e3620f4ef265b63058779731405bc8fb_10.png)

![](img/e3620f4ef265b63058779731405bc8fb_12.png)

完成以上设置后，点击“开始”按钮即可开播。

![](img/e3620f4ef265b63058779731405bc8fb_14.png)

## 直播中的核心操作

![](img/e3620f4ef265b63058779731405bc8fb_16.png)

直播开始后，你的主要任务是引导观众并促进转化。以下是直播过程中需要关注的核心操作。

![](img/e3620f4ef265b63058779731405bc8fb_18.png)

![](img/e3620f4ef265b63058779731405bc8fb_20.png)

开始直播后，你需要立即点开购物车，展示商品。这样观众才能在直播间看到商品弹窗，这对于提升转化率至关重要。

以下是直播中可进行的互动与氛围营造操作：
*   **专注话术讲解**：无需过度关注复杂功能，专注于你准备好的产品讲解话术。
*   **管理直播贴图**：开播后仍然可以添加或调整直播贴图。
*   **添加背景音乐**：可以播放一些背景音乐来活跃直播间气氛。
*   **利用评论上墙**：如果某位观众的评论特别正面或有价值，可以长按该评论，选择“点上墙”。这样，新进入直播间的用户就能看到这条精选评论。
*   **注意直播规范**：直播时务必避免使用违规词汇。
*   **分享直播间**：你可以将直播间链接分享给其他人，吸引更多观众。

## 结束直播与生成回放

![](img/e3620f4ef265b63058779731405bc8fb_22.png)

当直播完成后，需要正确结束流程并考虑留存资料。

直播结束后，点击屏幕右上角的“叉号”即可结束直播。

以下是结束直播时的可选操作：
*   **生成直播回放**：结束直播时，系统会提示是否生成直播回放。你可以选择生成，以便后续使用或复盘。

![](img/e3620f4ef265b63058779731405bc8fb_24.png)

本节课中我们一起学习了微信视频号直播的完整流程，从开播前的预告与内容准备，到直播中的商品展示、互动技巧与氛围营造，最后到直播的规范结束与回放留存。掌握这些步骤，你就能独立开启一场基础的视频号直播了。
