以下是商品质量分的主要考核维度：
*   **标题与关键词**：是否覆盖精准且有搜索量的词。
*   **主图与详情**：图片是否清晰、美观、有吸引力（影响点击率）。
*   **属性与价格**：填写是否完整、准确；价格是否合理。
*   **款式与受众**：产品所属的细分市场体量大小、受众群体是否广泛。一个热门款式的天生权重会比小众款式更高。
*   **原创性**：新品且内容原创，会获得更多的初始流量推荐。

以下是商品数据权重的核心考核点：
*   **销量与成交额**：单量（`订单数`）和销售额（`成交额`）是最直接的指标。
*   **转化率**：`转化率 = 订单数 / 访客数`。转化率越高，说明流量利用得越好。
*   **UV价值**：`UV价值 = 成交额 / 访客数`。这是衡量每个访客能带来多少价值的核心指标。UV价值越高，权重越高。
*   **客单价与增长率**：在同类产品中，客单价更高、数据增长趋势更好的商品，会获得更多青睐。
