# 抖音运营教程：P21：扶持加权与降权限流

在本节课中，我们将学习抖音平台的内容审核规则，了解哪些内容会获得平台的流量扶持，哪些内容则可能导致账号被降权限流。掌握这些规则是账号健康运营、快速起号的基础。

上一节我们介绍了账号权重的分类，本节中我们来看看具体的内容发布规范。

## 🚀 扶持加权的内容特征

![](img/7d360cfcf17e42ff7ccf6601c881071f_3.png)

平台会优先推荐符合以下特征的原生内容：

以下是获得流量扶持的视频需要满足的具体条件：

![](img/7d360cfcf17e42ff7ccf6601c881071f_5.png)

*   **视频格式**：必须为**竖版（9:16）** 的移动端原生视频。
*   **画面质量**：画面平稳，作品清晰度高。
*   **声音质量**：无过多杂音或背景噪音。

这些要点是抖音推荐机制的基础，符合标准的内容更容易进入流量池。

## ⚠️ 会导致降权限流的内容

反之，发布以下类型的内容将面临被限流甚至处罚的风险。

以下是明确会被平台限制传播的内容类别：

1.  **违规信息**：内容涉及吸烟、暴力、炫富（过度展示金钱）等。
2.  **外部引流**：视频中出现或口播提及**微信、微博、淘宝**等外部平台联系方式。长时间展示商品Logo、手机号、二维码（通常二维码出现**不要超过3秒**，但审核严格时瞬间出现也可能被驳回）。
3.  **硬性广告**：进行明显的营销广告、推销行为，意图直接引流卖货。
4.  **无口播的好物推荐**：这是需要特别注意的一点。如果视频是**无口播的拆箱或物品展示**（例如：单纯展示玩具、纸巾等产品），并试图挂小黄车带货，极易导致商品链接被下架。

针对第4点“无口播的好物推荐”，这里展开说明：

有学员曾发布展示玩具的视频，仅配背景音乐，未做任何口头介绍，结果小黄车被下架。平台判定其为单纯的广告展示。

**正确的做法是**：即使不做复杂剧情，也应加入简单的口播介绍。例如：
```text
“我儿子玩这个玩具，专注地玩了四个小时都没停。”
“这款纸巾沾水后依然坚韧，可以反复使用。”
```
通过口播赋予视频分享和体验的属性，而不仅仅是商品展示，这样更容易通过审核。

其他如直接口播广告、念联系方式等硬广行为，在抖音生态中是行不通的。
