## 付费起号的优势与平台选择

上一节我们介绍了课程目标，本节中我们来看看为何选择付费起号以及为何首选千川平台。

目前，付费起号是最高效的起号方法。过去的低价起号方式会引入大量低价意向人群，导致后续向正价商品转化时非常困难。

![](img/6204970611dc7275550a48b0ef85d704_5.png)

因此，下面分享一个干货：如何使用千川进行正价起号。此方法能快速获取精准流量。即使是新手，只要遵循步骤操作，大约七天就能成功起号。

![](img/6204970611dc7275550a48b0ef85d704_7.png)

以下是平台选择的要点：

![](img/6204970611dc7275550a48b0ef85d704_9.png)

*   **选择千川，而非小店随心推**：小店随心推难以带动自然流量，且容易消耗预算。此外，从去年下半年开始，其流量中西瓜、头条等泛流量占比较大，导致流量精准度不足。

## 千川投放的核心设置

![](img/6204970611dc7275550a48b0ef85d704_15.png)

了解了平台选择后，我们进入千川后台，学习核心的投放设置。

![](img/6204970611dc7275550a48b0ef85d704_17.png)

![](img/6204970611dc7275550a48b0ef85d704_19.png)

以下是三个关键的投放设置步骤：

1.  **投放方式选择“控成本投放”**：前期只需使用“控成本投放”即可。即使后期月消耗达到百万级别，控成本投放依然适用。无需担心新手前期跑量问题。
2.  **优化目标选择“成交”或“支付ROI”**：前期以积累成交数据为目标，选择 **`优化目标：成交`**。当流量稳定后，可以转为 **`优化目标：支付ROI`** 来稳定投产。
3.  **根据起号方式决定计划类型**：计划类型取决于你选择的起号模式。主要有两种模式：
    *   **微付费起号（带动自然流）**：投直播间画面（即“直投”）。
    *   **纯付费起号**：投视频素材。

## 方法一：直投微付费起号（带动自然流）

![](img/6204970611dc7275550a48b0ef85d704_27.png)

本节我们将详细讲解第一种方法：通过直投进行微付费起号，从而带动自然流量。这种方法适合新号及新手操作。

![](img/6204970611dc7275550a48b0ef85d704_29.png)

首先，新号适合从简单的计划开始。以下是搭建“徕卡计划”的具体步骤：

![](img/6204970611dc7275550a48b0ef85d704_31.png)

*   **行为兴趣设置**：点击“视频互动”、“直播互动”和“商品互动”。**注意**：不要点击“关注”行为，因为关注人群可能不够精准。
*   **时间范围**：选择“近15天”。
*   **达人定向**：不要选择宽泛的“达人分类”。为了给直播间打上精准标签，需要手动添加对标同行。寻找约30个同行账号，将他们全部添加到达人定向列表中。

![](img/6204970611dc7275550a48b0ef85d704_33.png)

计划搭建完成后，起号阶段需要配合数据运营。新号通常在1-7天内会迎来自然流量，因此需要做数据递增。

![](img/6204970611dc7275550a48b0ef85d704_35.png)

**数据递增示例**：
```
第一天：成交10单
第二天：成交12单
第三天：成交15单
...
```
每天的交易数据应比前一天有所增长，这样系统更容易识别并推荐你的直播间。

最后需要明确，前期投放通常会有一定亏损。因为新号没有标签和模型，不可能一开始就盈利。这是建立账户模型过程中的正常成本，需要能够接受。

## 方法二：素材纯付费起号

![](img/6204970611dc7275550a48b0ef85d704_41.png)

上一节我们介绍了直投起号，本节中我们来看看第二种以素材为核心的纯付费起号方法。

![](img/6204970611dc7275550a48b0ef85d704_43.png)

纯付费起号的核心是培养主力计划。当你的视频素材经过测试表现良好后，可以按以下步骤操作：

![](img/6204970611dc7275550a48b0ef85d704_45.png)

![](img/6204970611dc7275550a48b0ef85d704_47.png)

1.  **搭建行为兴趣计划**：选择“行为兴趣”定向，在“电商互动行为”中，选择“30天”时间范围。这里采用 **`大行为，小兴趣`** 的关键词策略（若不了解此策略，可查阅相关教程视频）。
2.  **积累基础标签**：为新素材投放，先积累20到50单以上的成交，以打下基础的成交标签。
3.  **切换托管计划**：达成50单后，将原计划关闭。然后复制该计划，使用“托管”投放方式继续投放，这样更容易提升ROI。

## 课程总结与要点回顾

![](img/6204970611dc7275550a48b0ef85d704_53.png)

本节课中，我们一起学习了2024年千川正价起号的完整方法。

![](img/6204970611dc7275550a48b0ef85d704_55.png)

我们首先明确了付费起号的高效性，并对比了千川与小店随心推的差异。接着，我们掌握了千川投放的三个核心设置：控成本投放、优化目标选择以及根据起号模式决定计划类型。

![](img/6204970611dc7275550a48b0ef85d704_57.png)

课程重点讲解了两种实操方法：
*   **直投微付费起号**：通过搭建徕卡计划、做数据递增来带动自然流量，周期约7天，前期需接受合理亏损。
*   **素材纯付费起号**：通过“大行为小兴趣”定向为素材积累基础成交标签后，转投托管计划以提升ROI，周期较短。

![](img/6204970611dc7275550a48b0ef85d704_59.png)

总的来说，投素材起号速度更快（约两三天），而直投起号则旨在用较少付费撬动自然流量（约七天）。投入预算越大，起号速度通常越快。
