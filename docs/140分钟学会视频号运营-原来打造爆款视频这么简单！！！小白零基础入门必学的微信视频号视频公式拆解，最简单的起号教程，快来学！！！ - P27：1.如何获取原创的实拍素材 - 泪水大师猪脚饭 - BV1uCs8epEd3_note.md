# 视频号运营：P27：如何获取原创的实拍素材 📸

在本节课中，我们将学习如何获取原创的实拍素材。这些素材可以用于视频搬运和混剪，以增加视频的原创度，降低被平台判定违规的风险。我们将介绍两种简单实用的获取方法。

## 概述

视频号运营中，除了搬运和混剪，添加原创素材是提升视频原创性的有效方法。自己购买样品实拍成本较高，因此我们需要寻找其他途径获取免费或低成本的原创素材。本节课将详细讲解两种获取方式。

## 原创素材的重要性

![](img/64e9470243093f856dd483e95098fe98_1.png)

上一节我们提到了视频搬运的常见性。然而，完全依赖搬运可能导致内容重复，增加违规风险。因此，在混剪视频时加入原创素材至关重要。原创素材能显著降低平台判罚违规的几率。

![](img/64e9470243093f856dd483e95098fe98_3.png)

## 获取原创素材的两种方法

以下是两种获取原创实拍素材的核心方法。

![](img/64e9470243093f856dd483e95098fe98_5.png)

![](img/64e9470243093f856dd483e95098fe98_6.png)

### 方法一：使用“产选原创素材”小程序

第一种方法是利用微信小程序“产选原创素材”。该工具与“配音神器”、“青抖”等类似，是一个专门提供素材的小程序。

操作步骤如下：
1.  在微信中搜索并打开“产选原创素材”小程序。
2.  注册并登录账号。
3.  绑定抖音账号（部分功能可能需要）。
4.  在小程序内浏览各类目（如食品饮料、日用百货、美妆护肤、母婴用品等）的视频素材。
5.  选择需要的视频，直接点击下载即可。

**优点**：视频清晰度高，拍摄专业。
**缺点**：素材库相对有限。

### 方法二：从电商平台买家秀获取

第二种方法是从淘宝或拼多多等电商平台的商品评论区获取买家秀视频。这些由真实用户拍摄的视频是优质的原创素材来源。

![](img/64e9470243093f856dd483e95098fe98_8.png)

#### 从淘宝获取素材

在淘宝获取素材通常需要录屏，因为手机端直接下载较困难。

![](img/64e9470243093f856dd483e95098fe98_10.png)

操作步骤如下：
1.  在淘宝搜索你的目标商品（例如，根据视频号选品中心的爆款榜选择商品）。
2.  进入商品详情页，找到“评价”区域。
3.  筛选带“视频”的评价。
4.  点开视频，点击屏幕进入“清屏模式”。
5.  **确保打开视频声音**，然后使用手机自带的录屏功能进行录制。

#### 从拼多多获取素材

从拼多多获取素材更为方便，可以通过分享链接后下载。

**方式A：通过微信与录屏（备用）**
1.  在拼多多找到目标商品的带视频评价。
2.  点击视频下方的“分享”按钮。
3.  将链接发送给微信好友或文件传输助手。
4.  在微信中打开链接，进入视频清屏模式后进行录屏。

**方式B：通过QQ浏览器直接下载（推荐）**
1.  复制拼多多商品评价中视频的分享链接。
2.  打开**QQ浏览器**（其他浏览器可能不支持），将链接粘贴到地址栏并访问。
3.  在浏览器中打开视频，点击右下角的“**...**”更多选项。
4.  选择“下载”即可将视频保存到手机相册。

**权限设置提示**：如果使用QQ浏览器下载后，视频无法保存到相册或在剪辑软件中找不到，请检查手机应用权限设置。路径通常为：`手机设置 -> 应用管理 -> QQ浏览器 -> 权限管理`，确保其拥有存储空间访问权限。

![](img/64e9470243093f856dd483e95098fe98_12.png)

## 总结

本节课我们一起学习了两种获取原创实拍素材的方法：
1.  通过“产选原创素材”微信小程序下载高清专业素材。
2.  从淘宝、拼多多等电商平台的买家秀中，通过**录屏**或**QQ浏览器下载**的方式获取真实用户拍摄的视频。

获取这些素材后，便可以在后续的混剪创作中融入它们，有效提升视频的原创度。下节课，我们将学习如何发布一条带货视频，并演示如何将这些原创素材剪辑到搬运的视频中。
