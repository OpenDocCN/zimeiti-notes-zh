在本节课中，我们将学习小红书平台的核心运作机制——笔记的审核流程与流量分发规则。理解这套机制是做好小红书运营的基础，能帮助你有效规避违规风险，并提升笔记的曝光机会。

## 笔记发布后的审核流程

![](img/a71300738badbb111f0263d16f674317_6.png)

上一节我们介绍了小红书平台的基本概念，本节中我们来看看笔记发布后，平台是如何进行审核与分发的。整个流程主要包含三个核心阶段。

### 第一阶段：机器审核

笔记上传后，首先进入机器审核系统。该系统会根据预设的基本规则对内容进行初步筛查。

以下是机器审核的主要检查项：

*   **敏感内容检测**：识别笔记中的敏感词汇、敏感图片等违规信息。
*   **基础质量评估**：检测笔记的字数、广告词出现频率、图片清晰度以及图片中文字与物体的可识别性。

如果笔记存在明确违规内容，可能会被直接处理。如果内容存在疑似违规或机器难以判断的情况，则会被标记并转入下一阶段。

### 第二阶段：去重检测

通过机器初步审核后，笔记会进入去重检测阶段。系统会将你的笔记与平台数据库中的现有内容进行比对，以判断内容的原创性。

以下是去重检测的核心比对维度：

![](img/a71300738badbb111f0263d16f674317_8.png)

*   **图片相似度**：检查图片是否为直接复制或高度模仿他人。
*   **文案重复率**：分析文案内容、段落结构在平台内的重复比例。

![](img/a71300738badbb111f0263d16f674317_10.png)

![](img/a71300738badbb111f0263d16f674317_12.png)

如果笔记被判定为抄袭（如重复率接近100%），将会被处理。通过对图片（如裁剪、加滤镜）和文案（如调整段落、扩充内容）进行创造性修改，可以有效降低相似度，通过此环节。

### 第三阶段：人工审核

人工审核主要针对机器审核标记出的“疑似违规”内容进行二次复核。由于内容量巨大，并非所有笔记都会经历此阶段。

以下是人工审核的重点关注领域：

*   **政治相关**：涉及国家政策、领导人等的内容。
*   **色情擦边**：带有性暗示或低俗倾向的内容。
*   **法律严禁**：其他违反法律法规的平台红线内容。

目前，平台算法已较为完善，触发明确红线的内容大多由机器直接处理。若创作者认为自己的笔记被误判，可以通过申诉渠道反馈，但新账号应尽量避免违规，以免对账号权重产生隐形影响。

## 账号权重与流量分发

了解了审核流程后，我们来看看影响笔记最终曝光的关键因素——账号权重。系统会根据账号的综合表现赋予一个隐形的“权重分”，它直接影响笔记的初始推荐量。

我们可以用以下模型来理解账号权重的变化：

![](img/a71300738badbb111f0263d16f674317_14.png)

```
新号权重 ≈ 0
持续优质更新 → 权重提升 (例如 0 → 5)
权重提升 → 笔记初始推荐流量池扩大
```

*   **新号阶段（低权重）**：新注册账号权重很低，笔记初始推荐量有限（可能仅几十个曝光）。此阶段应专注于“养号”，即保持内容垂直、定期更新、坚决不违规，通过良好的互动数据（点赞、收藏、完播率）逐步积累权重。
*   **成熟账号（高权重）**：当账号持续发布优质垂直内容后，权重会提升。系统会在相应领域（如美妆、穿搭）内给予更高的初始流量推荐，笔记更容易被推送给感兴趣的用户。

需要强调的是，“权重分”是便于理解的概念，平台并无公开数据。其核心逻辑是：**账号的历史表现越好，新内容获得的初始测试流量就越大**。

## 内容匹配与千人千面推荐

通过审核且获得初始流量后，笔记便进入推荐系统。小红书采用“千人千面”的智能分发模式。

![](img/a71300738badbb111f0263d16f674317_16.png)

**核心机制**：系统会分析笔记内容特征（如关键词、图片标签），同时分析用户行为标签（如浏览喜好、互动历史），然后将两者进行匹配，将笔记推荐给最可能感兴趣的用户。

![](img/a71300738badbb111f0263d16f674317_18.png)

例如，一篇关于“油皮护肤”的笔记，会被优先推荐给历史行为中关注护肤、搜索过控油产品的用户。
