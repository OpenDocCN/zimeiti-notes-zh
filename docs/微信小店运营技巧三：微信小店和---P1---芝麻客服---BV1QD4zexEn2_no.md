# 微信小店运营技巧三：P1：微信小店与视频号小店的区别及主页管理 🛍️

在本节课中，我们将学习微信小店与视频号小店的核心区别，并重点掌握升级为微信小店后新增的“主页管理”功能。这对于优化店铺展示、提升用户体验至关重要。

## 概述

各位视频号商家请注意，在9月25日之前，必须将视频号小店升级为微信小店，否则新增商品的上架将受到影响。如果您已购买芝麻视频号助手，请及时联系客服协助完成升级。

升级完成后，微信小店将新增“主页管理”功能。该功能允许您对店铺主页进行深度定制，包括商品排序、分类、设置背景图以及配置精选展示位。

## 微信小店主页管理功能详解

上一节我们明确了升级的必要性，本节中我们来看看“主页管理”功能具体包含哪些操作。以下是该功能的四个核心设置项：

### 1. 商品排序
您可以通过修改商品前的数字序号来调整其在主页的展示顺序。
**操作公式**：`修改商品序号 -> 调整展示排序`
**注意**：商品序号不能超过商品总数。

### 2. 商品分类
此功能允许您为商品设置分类，最多支持两级分类。
**操作流程**：
1.  新增分类（默认状态为隐藏）。
2.  选择“展示”后，分类将实时对用户可见。

![](img/b0557a08f845d3bf4538bcf901469a61_1.png)

### 3. 设置主页背景图
您可以上传一张图片作为店铺主页的顶部背景。
**规格要求**：图片将按 **16:9** 的比例展示在主页顶部。
**生效条件**：图片上传后需经平台审核，审核通过后生效。

### 4. 设置精选展示位
这是向用户推荐重点商品或内容的关键区域。以下是配置展示位的具体规则与选项：

*   **展示位数量**：
    *   小图模式：至少需配置 **3** 个展示位。
    *   大图模式：至少需配置 **1** 个展示位。
    *   两种模式最多均允许配置 **5** 个展示位。
*   **可配置的内容类型**：可以链接到当前店铺的商品、公众号文章或视频号内容。
    *   **重要提示**：所链接的公众号或视频号，不需要与当前视频号小店绑定关联。
*   **自定义封面**：您可以重新上传或修改展示内容的封面图。
