## 平台角色与内容形式

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_18.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_20.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_22.png)

首先，我们来了解小红书平台上的基本角色和内容形式。

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_24.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_26.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_28.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_30.png)

在小红书平台上，个人用户通常被称为“小红薯”。企业用户同样可以入驻，但本节我们主要讨论个人用户。这些用户通过分享内容来互动，而小红书将用户发布的这些内容称为“笔记”。

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_32.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_34.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_36.png)

“笔记”这个名称的由来，源于小红书希望鼓励用户持续创作与商品、购物体验相关的分享。每当用户购买到心仪商品或拥有美好体验时，记录并分享这一过程，就如同写下一篇生活笔记。

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_38.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_40.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_42.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_44.png)

## 个人账号的导流路径

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_46.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_48.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_50.png)

上一节我们介绍了平台的基本概念，本节中我们来看看个人账号如何实现流量引导。

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_52.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_54.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_56.png)

个人小红薯可以针对某个商家的具体产品、某个单品或一系列商品进行分享。例如，购买了一件上衣后，觉得它显白、显瘦、材质舒适、款式百搭，就可以将这种体验写成笔记分享到小红书。

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_58.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_60.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_62.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_64.png)

当其他用户看到这篇笔记后，可能会对商品产生兴趣，进而询问购买渠道或直接搜索商品。这个过程就将笔记的“阅读流量”引导至了具体的商品或商家店铺，最终可能促成销售。

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_66.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_68.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_70.png)

此外，如果个人账号的笔记内容质量很高、非常优秀，也可能会被相关领域的商家主动关注。商家可能会寻求合作，例如进行产品推广。这就为个人创作者提供了另一种变现途径，即通过优质内容吸引商业合作。

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_72.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_74.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_76.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_78.png)

以下是个人账号导流的核心路径总结：
*   **路径一（主动带货）**：用户分享好物笔记 → 吸引读者兴趣 → 读者搜索商品/店铺 → 产生购买。
*   **路径二（被动合作）**：用户持续产出优质垂直内容 → 吸引品牌方关注 → 获得商业合作机会 → 实现内容变现。

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_80.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_82.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_84.png)

## 企业账号的导流路径

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_86.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_88.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_90.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_92.png)

除了个人账号，企业账号是小红书平台上另一种重要角色。

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_94.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_96.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_98.png)

企业类账号通常用于推介自家商品，即向潜在消费者“安利”产品。企业通过平台发布笔记，向目标用户展示产品的优点、价值与利好信息。

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_100.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_102.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_104.png)

当目标消费者看到这些内容后，可能会对产品产生兴趣或购买欲望。企业从而能够将这些潜在客户引导至自己的官方店铺或商城，最终实现销售转化。

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_106.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_108.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_110.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_112.png)

企业账号的导流逻辑可以概括为以下公式：
**企业发布产品价值笔记 → 吸引目标消费者 → 引导至企业店铺/商城 → 实现销售转化**

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_114.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_116.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_118.png)

## 平台的核心价值与总结

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_120.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_122.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_124.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_126.png)

当然，也存在部分用户将小红书纯粹作为记录生活或经营自媒体的平台。但从商业运营的专业角度看，我们应聚焦并发挥小红书最根本的价值。

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_128.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_130.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_132.png)

这个核心价值在于：**无论是个人还是企业账号，都通过创作优质内容来获取流量，并将这些流量进行合理转化，最终实现变现或营收。** 其核心逻辑可以简化为：
`内容创作 → 获取流量 → 流量转化 → 实现变现`

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_134.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_136.png)

![](img/ae247c5a0b6e8b3559dd89d85e51e8c4_138.png)

本节课中我们一起学习了小红书的导流营销逻辑。我们了解到，平台通过“笔记”连接用户与商品，个人账号可以通过分享带货或吸引合作，企业账号则直接进行产品营销。所有行为的最终目的，都是将内容产生的关注度（流量）转化为实际的商业收益。理解这一逻辑，是做好小红书运营的第一步。
