# 新媒体运营：P16：02.1：工具篇-全平台短视频文案一键提取 📝

在本节课中，我们将学习一个非常实用的运营工具，它能帮助我们从抖音、小红书、哔哩哔哩等多个平台的短视频中，一键提取出完整的文案内容。掌握这个工具，将极大提升你的内容收集与创作效率。

---

上一节我们介绍了文案提取的需求，本节中我们来看看如何通过一个在线工具实现跨平台的一键提取。

![](img/d3493d056c8c8472b7e4255011661053_1.png)

这个工具的核心功能是：**用户只需将目标短视频的链接粘贴到指定输入框，工具即可在线自动识别并提取出视频中的全部文案**。它支持包括抖音、快手、哔哩哔哩、小红书、视频号在内的多个主流平台。

![](img/d3493d056c8c8472b7e4255011661053_3.png)

![](img/d3493d056c8c8472b7e4255011661053_5.png)

相较于传统的提取方法（如使用“轻抖”小程序，或将视频下载后导入“剪映”进行识别），此工具的优势在于：
*   **全平台支持**：无需为不同平台寻找不同工具。
*   **在线操作**：无需下载视频文件，节省时间和存储空间。
*   **高效便捷**：复制链接、点击提取，两步即可获得文案。

![](img/d3493d056c8c8472b7e4255011661053_7.png)

![](img/d3493d056c8c8472b7e4255011661053_9.png)

以下是使用该工具的具体操作步骤：

1.  **打开工具网页**：访问该在线工具的网页界面。
2.  **复制视频链接**：在目标短视频平台（如抖音），找到你想要提取文案的视频，复制其分享链接。
3.  **粘贴并提取**：将复制的链接粘贴到工具页面的输入框中，点击“提取文案”按钮。
4.  **获取与处理文案**：稍等片刻，工具会显示提取出的纯文本文案。你可以直接“一键复制”，也可以使用“一键格式化”功能，让文案按照字幕段落的形式规整排列。

![](img/d3493d056c8c8472b7e4255011661053_11.png)

此外，该工具也支持**上传本地视频文件**来提取文案，扩展了其使用场景。

![](img/d3493d056c8c8472b7e4255011661053_13.png)

为了让大家更直观地理解，我们进行两个平台的实际演示。

**首先演示提取抖音视频文案。**
我们找到一个情感语录类短视频，复制其链接并粘贴到工具中。点击提取后，很快便得到了准确的文案内容：“听完你就释怀了，村上春树说，我动了离开你的念头，不是因为你不好，也不是因为不爱了……”。格式化后，文案会分段清晰呈现。

**接下来演示提取哔哩哔哩视频文案。**
我们同样复制一个B站视频的链接（链接格式通常为 `https://www.bilibili.com/video/BV...`），粘贴到工具中并提取。结果显示，对于知识分享类视频“新手做直播电商，怎么才能打造出直播爆品……”，工具也能准确识别并提取出核心文案，准确性很高。

这个工具在运营工作中的价值在于：我们可以快速提取优质视频的文案，在此基础上进行修改、调整和二次创作，例如用于自己的口播视频脚本，这适用于众多行业领域。它集成了多个单一平台工具的功能，非常强大。
