# 跨境电商副业指南：P8：苹果手机抹机实操演示 📱

在本节课中，我们将学习如何对苹果手机进行“抹机”操作。这是跨境电商运营中准备设备的关键一步，目的是将手机恢复至出厂设置，并配置为目标市场（如美国）的语言、时区和隐私设置，为后续工作做好准备。

## 实操演示：以iPhone 6S为例

首先，我们来演示苹果手机的抹机流程。苹果手机型号众多，如iPhone 7、8、11、12、13等。本次演示以iPhone 6S为例。苹果手机系统相同，因此抹机流程大致相似。如有不懂之处，可截图或录屏咨询解答老师。

![](img/018abaec141f1d1ba582694c6b288ce2_1.png)

![](img/018abaec141f1d1ba582694c6b288ce2_2.png)

接下来是苹果手机抹机的实操教程。演示所用手机为iPhone 6S，是一部新手机，未插卡，仅连接了Wi-Fi。

以下是抹机的具体步骤：

1.  **进入设置**：点击手机主屏幕的“设置”图标。
2.  **找到还原选项**：在设置菜单中向下滑动，找到并点击“通用”。
3.  **选择抹除**：在通用菜单中，找到“还原”选项并点击。
4.  **确认抹除**：选择“抹掉所有内容和设置”，然后点击“抹掉iPhone”进行确认。

完成此步骤后，手机会自动重启，进入恢复界面。此过程通常需要1到2分钟，具体时间因手机而异，请耐心等待。

重启完成后，手机会进入初始设置界面。此时，请按以下步骤操作：

1.  **激活并选择语言**：按下Home键（屏幕下方圆形按钮），在语言选择界面，选择目标市场语言。例如，做美区或英区选择“English”；做台湾市场可选择“繁体中文”。
2.  **选择地区**：进入地区选择页面，滑动到最下方，选择“United States”（美国）。
3.  **连接网络**：点击继续后，选择并连接一个可用的Wi-Fi网络（例如用个人手机开热点或连接家庭Wi-Fi）。连接成功后，系统需要约一分钟进行设置。
4.  **跳过Apple ID**：在Apple ID登录页面，选择“忘记密码或没有Apple ID？”，然后点击“稍后在设置中设置”。后续步骤中若再次出现Apple ID相关提示，均选择“不使用”或“稍后设置”。
5.  **同意条款**：阅读并同意苹果的条款与条件。
6.  **关闭定位与诊断**：在接下来的设置中，关于定位服务和诊断数据，均选择“禁用”或“不发送”。
7.  **关闭Siri**：在Siri设置页面，选择“稍后设置”。
8.  **完成外观设置**：在屏幕外观（如深色模式）等可选设置中，使用默认选项或根据喜好选择，一路点击继续即可。

完成上述步骤后，手机将进入主界面，但抹机设置尚未全部完成。

## 关键系统设置

手机进入主界面后，还需进行几项关键的系统配置。

以下是需要检查或设置的选项：

1.  **网络权限**：如果系统弹出“允许应用使用数据”的提示，务必选择“允许使用WLAN与蜂窝数据”。
2.  **设置时区**：进入“设置” > “通用” > “日期与时间”。关闭“自动设置”，并在时区中选择目标城市，例如美国洛杉矶（Los Angeles）。
3.  **关闭定位服务**：进入“设置” > “隐私与安全性” > “定位服务”，确保定位服务已关闭。
4.  **关闭广告跟踪**：在“隐私与安全性”设置中，找到“跟踪”选项（可能在定位服务下方），关闭“允许App请求跟踪”。此选项并非所有手机版本都有，如有则关闭。

完成以上所有设置后，抹机操作才算全部完成。此时，手机已恢复纯净状态，并配置为适合目标市场的环境。

如果发现主屏幕上的系统应用图标显示为等待下载状态（云朵图标），只需点击该图标，在弹出的权限请求中选择“允许”，应用便会开始下载安装。

本节课中我们一起学习了苹果手机的完整抹机流程。从恢复出厂设置、选择语言地区，到配置网络、时区和隐私权限，每一步都是为跨境电商运营打造一个干净、合规的设备环境。请务必按照步骤仔细操作。
