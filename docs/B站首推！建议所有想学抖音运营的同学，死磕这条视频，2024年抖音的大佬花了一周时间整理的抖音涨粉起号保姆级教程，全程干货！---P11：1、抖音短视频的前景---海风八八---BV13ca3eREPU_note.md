# 抖音运营入门：P11：1、抖音短视频的前景

![](img/b22ca8faaad8980225bd726ae7ac0e33_1.png)

在本节课中，我们将要学习抖音短视频平台的发展前景，分析其市场潜力与用户生态，并探讨普通人入局的机会。

![](img/b22ca8faaad8980225bd726ae7ac0e33_3.png)

了解抖音短视频的前景，需要先通过一张图片来认识。

![](img/b22ca8faaad8980225bd726ae7ac0e33_1.png)

这张图片来自抖音创作者大会的数据报告。报告显示，有超过2200万人在抖音的合计收入超过了417亿元。抖音希望将这个数字翻一番，让创作者的总收入达到800亿元。

![](img/b22ca8faaad8980225bd726ae7ac0e33_3.png)

这代表了巨大的流量扶持。同时，这里引入第一个知识点：抖音上有两类人。第一类人叫做“玩家”，他们单纯地玩抖音，没想过从中赚钱。另一类人，可以称作“创作者”，他们希望通过制作抖音内容来获取收益。区分这两类人很重要，后续还会提到。

那么，为什么我们要选择抖音短视频呢？市面上有快手、火山等众多短视频平台，为何非得做抖音？这就像开实体店选址，一定会找一个人流量大、位置好的地方。做短视频同样要选择一个流量大的平台。如果平台自身流量不足，就无法分给创作者，也就难以赚到钱。

抖音的日活跃用户超过8亿，月活跃用户更是达到了40亿人次。这是一个极其庞大的流量池。

流量大，竞争压力会不会很大？答案是不会。抖音上的创作者大约只有5000万。创作者并非特指网红达人，任何发布内容的人都是创作者。例如，邻居老太太用手机拍下自己做的饭传到抖音上，她也算创作者。因此，90%的竞争压力都来自于身边的普通人。

接着，做抖音短视频需要购买电脑或摄像机吗？不需要。只需要一部智能手机和一个手机号，就可以注册抖音账号。抖音倡导全民通过短视频分享和记录生活，门槛很低。

那么，当前有什么样的风口值得我们去做呢？首先，抖音计划扶持1000名优质的艺术创作者，帮助他们涨粉超过100万。有的同学可能会说：“老师，我不会创作。” 这不用担心。如果什么都会，就不需要来听课了。正是因为不会，才需要学习。

市场庞大，竞争相对温和，行业也容易产出内容。现在是大数据时代。大数据时代的特点是，平台会根据你的喜好推荐内容。你喜欢看帅哥，刷到的就是有腹肌的帅哥；你喜欢看美女，刷到的就是长腿美女。你喜欢看什么，抖音就推荐什么。

有的同学是宝妈，不知道拍什么，可以拍宝宝或自己做的饭。有的同学开实体店，想引流，可以拍自己的产品。有的同学是普通上班族，想分享生活，也可以拍。月活跃用户40亿的庞大群体中，只要你的内容不触犯法律和道德底线，不是低俗内容，总有一部分人会喜欢你。

一定会有人喜欢你的。那为什么你只有几十个粉丝呢？是因为喜欢你的人还没有刷到过你的作品。所以，对于新手来说，心情迫切可以理解，但不要把涨粉作为第一个目标。我们的第一个目标是先把流量和播放量做起来。播放量涨起来之后，粉丝自然就来了。

本节课中我们一起学习了抖音平台的巨大流量前景、创作者与玩家的区别、入局的低门槛优势，以及在大数据时代下内容创作的普适性。核心在于理解**先做流量，后得粉丝**的逻辑，并开始思考自己可以创作的方向。
