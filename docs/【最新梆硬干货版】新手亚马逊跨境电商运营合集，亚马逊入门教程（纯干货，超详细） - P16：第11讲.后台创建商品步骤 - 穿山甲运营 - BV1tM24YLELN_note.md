# 亚马逊跨境电商运营：P16：后台创建商品步骤

在本节课中，我们将学习如何在亚马逊卖家后台手动创建商品。我们将详细介绍创建过程中的必填项、常见问题以及一些实用技巧，帮助你顺利完成商品上架。

---

## 一、 进入创建页面与基础信息填写

首先，我们需要进入卖家后台的“添加新商品”页面。为了操作方便，建议先将后台语言切换为中文。

**步骤：**
1.  在后台找到并点击“添加新商品”。
2.  选择“我要添加未在亚马逊上销售的新商品”。

接下来，我们开始填写商品的基础信息。

**以下是创建商品时需要填写的关键字段：**

*   **商品编码：** 通常选择 **UPC**、EAN 或 ISBN。本教程以 **UPC** 为例，你需要准备有效的UPC码。
*   **商品名称：** 格式一般为 **品牌名 + 产品核心名称**。这是吸引买家点击的关键。
*   **品牌：** 如果商品暂无注册品牌，可以勾选“无品牌”，或填写通用品牌名如 **NC**、**NP**。
*   **制造商：** 这里有一个实用技巧。如果你计划注册品牌但证书尚未下发，可以在此处填写你**计划注册的商标名称**。前提是该名称已通过服务商核名，确认可用。这为日后品牌备案后修改信息提供了便利。

> **注意：** 关于品牌修改的难度已随时间增加。早期可自行或通过客服轻松修改，但现在即使联系客服并提交使用证据，过程也可能非常繁琐。

*   **其他属性：** 根据商品实际情况填写。例如，“商品是否有保质期”通常选“否”；“商品数量”指包装内含数量，常填“1”；“批次数目/类型”按重量等单位填写；不相关的属性（如“是否具有热敏性”）可直接忽略或选“否”。

## 二、 处理商品变体

![](img/fe8ebb3afec2b6a363a72d6b6cd75bdf_1.png)

上一节我们填写了基础信息，本节中我们来看看如何处理具有多属性（如颜色、尺寸）的商品，即创建变体。

![](img/fe8ebb3afec2b6a363a72d6b6cd75bdf_3.png)

在“重要信息”部分找到“是否有变体”选项。如果商品有变体（例如，同款T恤有不同的颜色和尺寸），请选择“是”。

![](img/fe8ebb3afec2b6a363a72d6b6cd75bdf_5.png)

![](img/fe8ebb3afec2b6a363a72d6b6cd75bdf_6.png)

**以下是创建变体的具体步骤：**

![](img/fe8ebb3afec2b6a363a72d6b6cd75bdf_8.png)

![](img/fe8ebb3afec2b6a363a72d6b6cd75bdf_9.png)

![](img/fe8ebb3afec2b6a363a72d6b6cd75bdf_10.png)

1.  **选择变体主题：** 根据商品属性选择，如“颜色”、“尺寸”或“颜色+尺寸”等。可以参考竞争对手的商品页面是如何设置变体的。
2.  **添加变体值：** 在出现的表格中，为每个属性添加具体值。例如，颜色添加“Black”、“Gray”，尺寸添加“XL”、“XXL”。
3.  **填写子体信息：** 系统会为每个属性组合生成一个子商品（子体）。你需要为每个子体填写独立信息：
    *   **卖家SKU：** 建议使用清晰的命名规则，例如 **`ProductName_Color_Size`**（如 `Lashes_Black_XL`）。这便于库存管理。
    *   **商品编码：** 每个子体都需要一个独立的UPC码。
    *   **价格与数量：** 为每个子体设置单独的价格和库存数量。
4.  **父体信息：** 父商品（父体）的SKU可以更通用，如 `Lashes_Set1`，用于代表整个变体系列。

![](img/fe8ebb3afec2b6a363a72d6b6cd75bdf_11.png)

> **提示：** 如果变体较多，表格可能较长，注意向右滚动填写所有必填字段，确保信息完整。

## 三、 完成报价、配送与图片上传

创建好变体结构后，我们需要继续完善商品的其他关键信息。

**首先，设置报价和配送：**
*   **价格：** 填写你计划销售的商品价格。
*   **数量：** 填写初始库存数量。
*   **配送渠道：** 可以选择“卖家自行配送”或“亚马逊配送（FBA）”。一个常见的操作习惯是**先选择“卖家自行配送”**，待商品信息完全审核通过后，再在库存管理中转换为“亚马逊配送”。这可以避免一些初始设置上的冲突。

**接着，上传商品图片：**
*   通常需要上传至少1张主图和8张辅图，共9张图片。
*   **变体图片技巧：** 对于有变体的商品，除了在每个子体中上传图片外，系统会默认使用父体的主图作为变体选择器（如颜色小图）的展示图。为了保持变体选择器图片风格统一美观，建议在**父体的“图片”选项卡中，最后一张上传你希望用作统一展示的图片**。

## 四、 解决常见错误与商品状态管理

![](img/fe8ebb3afec2b6a363a72d6b6cd75bdf_14.png)

![](img/fe8ebb3afec2b6a363a72d6b6cd75bdf_15.png)

在填写过程中，你可能会遇到系统报错。常见的错误及解决方法如下：

*   **UPC报错：** 有时系统提示“使用的UPC与尝试提供的产品不匹配”，但这**可能不是UPC本身的问题**，而是品牌字段填写导致的。尝试将品牌从“通用”更改为 **`NP`** 或 **`NC`**，往往可以解决此问题。
*   **保存并完成：** 填写所有信息后，点击“保存并完成”。商品可能进入“正在处理”状态。

![](img/fe8ebb3afec2b6a363a72d6b6cd75bdf_17.png)

商品创建后，你需要关注其状态，并在“管理库存”页面进行处理：

1.  **补全草稿：** 如果商品因信息缺失（如缺少长描述、图片）而上传失败，会在“目录”>“补全草稿”或“管理库存”中显示为“草稿”。点击“编辑”补全信息。
2.  **修复禁止显示：** 商品可能显示“禁止显示”（不可售）。点击该提示查看原因，通常是“图片缺失”或“描述问题”。根据提示在“修复商品”页面补充内容（如上传图片、填写长描述），即可恢复在售。
3.  **优化商品信息：** 在“商品信息控制面板”中，亚马逊会提供优化建议（如补充更多产品属性：材质、保养说明等）。**尽可能补全这些建议信息**，有助于提升商品曝光和转化率。

![](img/fe8ebb3afec2b6a363a72d6b6cd75bdf_19.png)

## 五、 总结与检查

![](img/fe8ebb3afec2b6a363a72d6b6cd75bdf_24.png)

本节课中，我们一起学习了在亚马逊后台手动创建商品的完整流程。

**核心步骤总结如下：**
1.  **填写基础信息**，包括UPC、标题、品牌（技巧：制造商处可填预注册品牌名）。
2.  **设置商品变体**，选择主题并详细填写每个子体的SKU、价格和库存。
3.  **完善报价、配送**（可先选自配送）和**图片上传**（注意变体图片的统一性）。
4.  **处理发布后的状态**，解决报错、补全草稿、修复禁止显示问题，并利用控制面板优化信息。

![](img/fe8ebb3afec2b6a363a72d6b6cd75bdf_26.png)

最后，创建完成后，请务必前往亚马逊前台搜索你的商品ASIN或标题，确认商品页面是否已正常、完整地显示。确保主图、标题、价格、描述等所有信息准确无误，这才意味着商品已成功上架。
