# 服装行业流量爆发营：045：形象改造视频的角色饱满度提升

![](img/c8b1ee0a38896896720e7ca5e1dfd9f8_0.png)

![](img/c8b1ee0a38896896720e7ca5e1dfd9f8_1.png)

在本节课中，我们将学习如何提升形象改造视频的吸引力。核心在于增加视频中角色的饱满度，通过引入更多角色来丰富故事内容，使其更具趣味性和看点。

上一节我们介绍了形象改造视频的多个维度，本节中我们来看看如何通过增加角色来让故事更饱满。

许多服装行业从业者拍摄的形象改造视频，通常只有两个角色：改造者和被改造者。有时改造者甚至不露脸。这样的故事线较为单薄。我们可以分析一个近期很火的账号“垫底辣孩”，它专注于国际超模风格改造。该账号的视频中，常有一些看似无关紧要的角色，但它们极大地增加了视频的趣味性。

以下是“垫底辣孩”视频中的一个例子：

> 如何成为一个品牌代言人？今天要拍的是LANCOME。

在这段素材开头，画面后方有一位老爷爷。这位老爷爷是其视频中的固定角色，会不定期出现在不同场景中。同样，在“门书”的视频中，也有一位表情厌世、沉默寡言的搞笑助理角色。这些角色的灵感可能来源于周星驰电影中的经典形象。增加角色的丰富性能让整个故事更加饱满。

我们再看“垫底辣孩”的另一张截图，视频中经常出现一个小妹妹的角色。这个角色也参与了丰富视频内容，通过多维角色的互动来提升饱满度。

此外，回顾上一节课模仿的“罗姐”案例，她说因为儿子感到丢人而难过。我们当时也可以在旁边增加店员角色，例如店员可以说：“气死了，这小孩，我下次要教训他。”或者增加曾佳接到丈夫电话的桥段：“说你又哪去了，怎么还不回家做饭？”这些都能丰富角色。再到后来邀请儿子到现场，都是为了在一个视频中呈现多个不同角色，从而使整个故事更加饱满。

> （视频对话内容，涉及多个角色互动，如“叫爸爸”、“我是杨幂来了”、“男生就是欠收拾啊”等，并配有背景音乐。）

在第二段素材中，这是一个走进男生宿舍的改造视频。因此，里面包含了大量角色：有的男生瘫在床上，有的在吃泡面，有的在洗头。你会发现，每个角色虽然只短暂出现，却让整个视频的丰满度大幅提升。

所以，在我们进行形象改造视频创作时，可以设法让角色更加丰富。有些角色可以实际出场，有些甚至可以像打电话那样只出现声音，不一定需要真人露面。当一个视频中出现**3~4个角色**时，这个故事就会非常吸引人。
