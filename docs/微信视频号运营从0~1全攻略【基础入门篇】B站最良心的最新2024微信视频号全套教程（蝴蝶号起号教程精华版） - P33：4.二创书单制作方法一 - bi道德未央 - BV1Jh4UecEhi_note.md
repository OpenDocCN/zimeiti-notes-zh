# 微信视频号运营从0~1全攻略：P33：4.二创书单制作方法一 📚

在本节课中，我们将学习如何为书单类视频进行二次创作。这是一种比直接混剪原创度更高的方法，通过替换和重组现有视频素材，制作出全新的内容。

---

## 课程概述

剪辑是制作各类视频的必备技能。对于新手而言，初期的不熟悉感是正常的，通过练习可以快速掌握。本节课程将重点讲解书单类视频的二次创作方法，这是一种有效且易于出单的内容形式。

## 书单类视频的逻辑与优势

上一节我们介绍了混剪的基础方法，本节中我们来看看如何通过二次创作提升视频的原创度。

书单类账号的运营逻辑，是通过一段影视剧片段、故事或论述讲明一个道理，最后推荐一本书籍，从而引发观众的购买欲望。这种内容形式比直接展示产品功能的“硬广”更自然，转化效果也更好。

## 二次创作实操步骤

下面，我们将一步步演示如何进行书单视频的二次创作。

![](img/0c024ca23b7f194aa66db0c17f1bfceb_7.png)

![](img/0c024ca23b7f194aa66db0c17f1bfceb_9.png)

### 第一步：寻找对标素材

![](img/0c024ca23b7f194aa66db0c17f1bfceb_11.png)

![](img/0c024ca23b7f194aa66db0c17f1bfceb_13.png)

1.  打开抖音，搜索目标书籍，例如“女孩要懂得保护自己”。
2.  点击“商品”标签，按“销量”排序。
3.  进入商品页，找到“达人秀”板块。这里聚集了大量销售该商品的视频，我们可以从中寻找合适的素材进行借鉴和二次创作。

![](img/0c024ca23b7f194aa66db0c17f1bfceb_15.png)

### 第二步：提取核心元素

找到点赞量较高的对标视频后，我们需要提取两个核心元素：
1.  **视频素材**：复制视频链接，使用去水印工具（如“配音神器”）保存原视频到相册。
2.  **文案脚本**：使用文案提取工具，将视频的解说文案提取出来。

![](img/0c024ca23b7f194aa66db0c17f1bfceb_17.png)

### 第三步：生成新配音与处理原视频

1.  **生成新配音**：将提取的文案导入配音软件（如剪映的配音功能或第三方工具），选择合适的声音生成新的音频文件。生成前注意检查文案是否有错别字，并可以调整语速、添加停顿。
    *   **代码/操作示例**：在配音软件中，`选择音色 -> 输入/粘贴文案 -> 调整参数（语速、停顿）-> 合成配音 -> 导出音频`。
2.  **处理原视频**：将去水印后的原视频导入剪映。首先关闭视频原声，然后使用蒙版功能遮盖掉原有的字幕。
    *   **公式/操作逻辑**：`添加矩形蒙版 -> 调整覆盖字幕区域 -> 选择“反转” -> 设置背景为“画布模糊”`。

### 第四步：替换与重组视频素材

![](img/0c024ca23b7f194aa66db0c17f1bfceb_19.png)

这是二次创作的核心环节，即用新的素材替换原视频中的片段。

![](img/0c024ca23b7f194aa66db0c17f1bfceb_21.png)

![](img/0c024ca23b7f194aa66db0c17f1bfceb_23.png)

以下是替换素材的两种方法：

*   **方法一：使用“替换”功能**
    *   在时间轴上分割出想要替换的片段。
    *   点击“替换”，从剪映素材库或手机相册中选择新素材。此方法快捷，但可能留有原片段的痕迹。
*   **方法二：删除后添加新素材（推荐）**
    1.  删除原视频中需要替换的片段。
    2.  在对应位置导入从抖音或素材库找到的新视频素材。
    3.  调整新素材的时长，使其与删除的片段大致相同，并关闭其原声。
    4.  确保片段衔接自然，必要时进行分割或微调。

**寻找新素材的途径**：
*   **剪映素材库**：搜索关键词（如“老师讲课”、“学生学习”）。
*   **抖音直接搜索**：搜索更具体的关键词（如“女讲师培训”），找到合适片段后保存使用。

### 第五步：合成与导出

![](img/0c024ca23b7f194aa66db0c17f1bfceb_25.png)

![](img/0c024ca23b7f194aa66db0c17f1bfceb_27.png)

![](img/0c024ca23b7f194aa66db0c17f1bfceb_29.png)

1.  **添加新配音**：将第三步生成的新配音音频导入剪映，与替换重组后的视频画面进行对齐。可以分割、移动音频块以实现精准匹配。
2.  **添加背景音乐**：选择一首风格匹配的纯音乐作为背景音，音量不宜过大。
3.  **生成新字幕**：使用“文本”->“识别字幕”功能，为新的配音生成字幕。调整字幕样式，并确保其覆盖掉之前用蒙版遮盖的区域。
4.  **最终检查与导出**：播放整个视频，检查音画同步、字幕准确性。最后，将帧率设置为60帧进行导出。

![](img/0c024ca23b7f194aa66db0c17f1bfceb_31.png)
