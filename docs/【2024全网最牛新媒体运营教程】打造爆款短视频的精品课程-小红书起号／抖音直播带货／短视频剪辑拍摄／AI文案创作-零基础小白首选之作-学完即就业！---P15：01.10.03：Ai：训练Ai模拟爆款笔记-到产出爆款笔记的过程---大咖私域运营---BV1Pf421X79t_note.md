## 概述
在本节课中，我们将学习如何训练AI（以ChatGPT为例）分析爆款笔记，并最终生成具备爆款潜质的新笔记。我们将从分析爆款内容的结构、关键词和情感共鸣点入手，逐步引导AI学习并产出符合平台调性的优质文案。

本节中，我们来看看如何通过分析具体的爆款内容，让AI学习其成功要素，并最终生成新的爆款笔记。核心在于分析以下几个点：

![](img/5bb07f5efff13b83cc33c55b78615b06_19.png)

![](img/5bb07f5efff13b83cc33c55b78615b06_21.png)

![](img/5bb07f5efff13b83cc33c55b78615b06_23.png)

以下是需要AI重点分析的几个方面：
1.  **爆款因素**：分析内容为何能成为爆款，例如是否引发了情感共鸣、提供了实用价值或讲述了逆袭故事。
2.  **内容结构**：
    *   **开篇**：用什么方式（如提问、制造焦虑、引用名人）吸引用户继续阅读。
    *   **中间**：如何通过内容戳中用户痛点，提供解决方案，提升用户停留时长。
    *   **结尾**：如何设置“钩子”（如鼓励互动、引导私信、提供福利）促使用户产生进一步行为。
3.  **关键词策略**：分析笔记中是否刻意植入了关键词，以及关键词的出现频率和分布方式。

### 第一步：向AI提供指令与样本
我们开启一个新的ChatGPT（3.5版本）对话，并输入以下指令：
```
你是一名小红书内容分析专家。我将提供几篇小红书平台上关于考研的爆款笔记文案，请你分析：
1. 爆款因素：这些文案为何能成为爆款？
2. 整体结构：开篇用什么吸引用户？中间用什么内容戳中用户痛点？结尾用什么方式引导用户互动（如私信、关注）？
3. 关键词分析：分析每篇笔记是否结合中文分词法设计了关键词。如果有，请指出关键词是什么、出现频率如何；如果没有，请说明。
```
随后，我们将找到的爆款笔记文案复制粘贴给AI。

### 第二步：分析AI的反馈
AI对第一篇笔记的分析反馈如下：
*   **爆款因素**：情感共鸣（开篇制造焦虑）、逆袭故事、提供实用备考方法、使用表情符号增加亲和力、结尾传递正能量。
*   **结构分析**：准确指出了开篇、中段、结尾各自的作用。
*   **关键词分析**：认为该笔记没有明显设计的关键词，多为常用词汇。

我们继续提供第二篇、第三篇笔记文案给AI分析，让它积累更多样本。过程中发现，ChatGPT 3.5有时会“忘记”之前的指令或分析过的内容，需要适时提醒它回顾总结。

### 第三步：整合爆款要素并生成大纲
在AI分析了多篇样本后，我们要求它整合所有分析过的爆款因素，去重后总结成一个清晰的列表。
指令如下：
```
请将你前面分析过的几篇考研爆款笔记的爆款因素总结整合一下，形成一个全面的爆款要素列表。
```
AI总结的要素通常包括：情感共鸣、逆袭故事、明确的备考计划、针对性建议、具体的时间节点、资源技巧分享、正能量鼓励等。

接着，我们指令AI根据这些爆款要素，规划一篇新笔记的大纲。
指令如下：
```
现在，请结合你总结的爆款笔记因素，为我规划一篇关于考研的小红书笔记大纲。大纲需要结构清晰，每个主要部分下要列出细分的要点。
```
AI生成的大纲示例：
*   **1. 引人入胜的开头**：用问题或故事吸引注意，简要介绍主题。
*   **2. 专业选择分析**：分析考研专业趋势。
*   **3. 详细备考规划**（核心部分）：
    *   政治备考计划
    *   英语备考计划
    *   数学备考计划
    *   专业课备考计划
*   **4. 时间管理与心态调整**：提供时间管理技巧和心态鼓励。
*   **5. 互动引导结尾**：鼓励读者提问、互动。

### 第四步：根据大纲生成完整文案
我们认可大纲后，便指令AI根据大纲和爆款要素生成完整文案。
指令如下：
```
请根据你上面提供的大纲，撰写一篇完整的小红书考研笔记文案。要求：
1. 结构严格遵循爆款笔记的要素和结构。
2. 开头需要包含一个简单的“逆袭小故事”来引发读者共鸣。
3. 文中需要包含一定量的emoji表情。
4. 需要尝试结合中文分词法，融入“考研”等相关关键词。
```
AI生成的文案具备了爆款的许多特征：有共鸣式的开头、清晰的干货结构、积极鼓励的结尾，并嵌入了表情符号。虽然对于专业运营人员来说可能还需优化，但对于初学者或一般运营需求而言，这已是一篇合格且具有潜力的笔记草稿。

## 进阶技巧与工具提示
*   **与AI对话的措辞**：指令的措辞会影响产出。例如，“包含丰富的emoji”和“包含一定的emoji”会导致AI使用表情的数量不同。
*   **使用更强大的模型**：ChatGPT 4.0在理解复杂指令、保持上下文连贯性和内容创意上通常表现更佳。
*   **利用定制化AI助手（如GPTs）**：可以创建自定义的GPTs，预先上传你总结的爆款结构文档、专属表情包列表或优秀文案范例，打造一个更懂你领域和风格的私人AI助手，避免每次重复输入核心指令。

## 总结
本节课我们一起学习了从分析到产出的完整流程。首先，我们教会AI如何拆解爆款笔记的**爆款因素**、**内容结构**和**关键词策略**。然后，通过让AI**整合学习**这些要素，并据此**规划大纲**，最终**生成**了一篇融合了多项爆款特征的考研领域笔记。掌握这个方法，你就能让AI成为你内容创作中的高效助手，持续产出更符合平台调性的潜力内容。
