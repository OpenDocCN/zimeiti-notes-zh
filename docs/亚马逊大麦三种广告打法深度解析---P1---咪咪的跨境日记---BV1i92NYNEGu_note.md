# 亚马逊广告策略：P1：三种核心广告打法深度解析 🎯

在本节课中，我们将要学习一位亚马逊卖家如何通过调整广告策略，实现从亏损到盈利的转变。我们将深入解析其采用的三种核心广告打法：漏斗打法、否定打法以及农村包围城市打法。每种打法都将从理论基础、核心逻辑到具体操作步骤进行详细阐述，旨在为初学者提供清晰、实用的指导。

## 概述

一位亚马逊卖家通过关键性地调整广告策略，在一年内实现了从亏损300个（单位）到盈利300个（单位）的转变。其核心策略主要包含三种不同的广告调整方法。接下来，我们将逐一拆解这些策略。

## 漏斗打法 🔄

上一节我们概述了三种核心打法，本节中我们来看看第一种：漏斗打法。此打法主要包含两个层面：客户成交的漏斗模型与广告投放的漏斗模型。

### 客户成交漏斗

客户成交漏斗描述了消费者从认知到购买的完整过程，通常包含四层。以下是其核心影响因素与相应的布局策略：

1.  **认知层**：影响用户首次发现产品的因素。策略是广泛覆盖，提高品牌或产品的曝光度。
2.  **兴趣层**：影响用户产生进一步了解欲望的因素。策略是通过精准的内容和关键词吸引目标用户。
3.  **决策层**：影响用户在多个选项中做出选择的关键因素。策略是突出产品优势、评价和差异化卖点。
4.  **行动层**：最终促使客户完成购买的影响因素。策略是优化购买流程，减少摩擦，并提供强有力的行动号召。

### 广告漏斗打法

广告漏斗打法则专注于通过广告工具引导用户完成这一过程，其布局包含五个关键步骤。以下是具体步骤：

1.  **广泛曝光**：使用广泛匹配类型或自动广告活动，在漏斗顶部获取大量流量。
2.  **精准筛选**：分析广告报告，筛选出表现优秀的关键词和用户搜索词。
3.  **重点投放**：将筛选出的优质关键词放入词组匹配或精准匹配广告活动中，进行重点投放。
4.  **持续优化**：根据广告表现数据（如ACOS、转化率），持续调整出价和预算。
5.  **否定无效流量**：定期将不相关或表现差的搜索词添加为否定关键词，提升广告效率。

## 否定打法 🚫

在掌握了漏斗打法的基本框架后，我们来看看第二种策略：否定打法。这种方法的核心在于通过排除无效流量来提升广告的整体表现和投资回报率。

![](img/e452ed5f689233c5f2e9d0afdd70bf1b_2.png)

本节将介绍不同的否定应用场景、具体操作方法和实战案例。

*   **场景一：否定不相关流量**
    *   **操作**：在自动广告或广泛匹配广告活动中，将与你产品完全不相关的用户搜索词添加为“否定精准匹配”。
    *   **案例**：你销售的是“iPhone 13 手机壳”，但广告却显示了“iPhone 13 充电器”的搜索词，此时应将“充电器”相关词否定。

![](img/e452ed5f689233c5f2e9d0afdd70bf1b_3.png)

*   **场景二：否定表现差的词**
    *   **操作**：对于长期有点击但无转化或转化成本过高的关键词，将其在相应的广告活动中添加为否定关键词。
    *   **案例**：关键词“cheap phone case”带来了大量点击，但订单为零，且拉高了ACOS，应考虑否定该词。

*   **场景三：防御性否定**
    *   **操作**：在品牌广告或定位到自身产品的广告活动中，否定自己的品牌词或ASIN，以避免自己与自己竞争，浪费广告费。
    *   **公式**：`否定关键词 = {自身品牌名， 自身主要ASIN}`

## 农村包围城市打法 🌾➡️🏙️

最后，我们来探讨第三种策略：农村包围城市打法。这种策略类似于“用小词带动大词”的思路，即先攻克竞争较小、成本较低的长尾关键词市场，积累权重和销量，再逐步进攻核心大词市场。

![](img/e452ed5f689233c5f2e9d0afdd70bf1b_5.png)

以下是该打法从理论到实践的六个应用步骤：

1.  **目标确定**：明确你的核心“城市”（即竞争激烈、流量大的主关键词）和“农村”（即竞争较小、更精准的长尾关键词）。
2.  **农村开拓**：创建广告活动，专门针对这些长尾“农村”关键词进行投放。出价可以相对积极，以快速获取市场份额。
3.  **数据积累**：通过“农村”广告积累稳定的销量、好评和关键词自然排名权重。
4.  **权重传导**：产品在长尾词下的排名和销量提升，会间接提升其在相关大词（核心“城市”）下的权重。
5.  **试探城市**：当产品在“农村”市场稳固后，开始尝试对核心大词进行较低出价的广告投放，测试转化效果。
6.  **全面进攻**：如果核心大词开始产生良好转化，则逐步提高其广告预算和出价，正式发起对“城市”的进攻。

## 总结

本节课中，我们一起学习了三种高效且经过实战验证的亚马逊广告打法。**漏斗打法**帮助我们系统化地布局广告，引导客户完成购买旅程；**否定打法**教会我们如何精炼流量，提升广告效率；**农村包围城市打法**则提供了一种从易到难、积小胜为大胜的竞争策略。掌握这三种策略的组合应用，将能有效优化你的广告结构，助力实现销售目标。
