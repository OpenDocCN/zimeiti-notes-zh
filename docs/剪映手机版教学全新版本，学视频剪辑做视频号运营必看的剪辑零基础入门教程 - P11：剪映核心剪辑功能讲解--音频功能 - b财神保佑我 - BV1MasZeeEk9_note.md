在本节课中，我们将要学习剪映APP的核心剪辑功能之一——音频功能。一条完整的短视频不仅包括画面，还需要合适的背景音乐（BGM）或音效来提升氛围和吸引力。本节将详细介绍如何为视频添加和处理音频。

上一节我们介绍了分割功能，本节中我们来看看音频功能。一条吸引人的爆款视频，离不开一个热门的BGM。在剪映中，我们可以通过音频功能为视频添加背景音乐或音效，还可以通过提取音乐和抖音收藏的方式直接选用热门音频，最后也能为视频录制旁白。这些操作能显著提升视频的听感质量和整体氛围。

![](img/a0f0f8622efaa2445afd26776fd0fbb7_13.png)

音频功能位于剪映的一级菜单。添加视频素材后，在底部菜单栏点击 **音频**，即可进入操作界面。该界面包含以下五个主要功能：添加音乐、音效、提取音乐、抖音收藏和录音。

接下来，我们将通过实操逐一讲解这些功能。



首先，导入一段示例视频素材。预览原素材，发现它只有环境音（如风声），缺乏明确的主题氛围。我们可以通过添加音频来强化视频想传达的思想。

以下是添加音频的第一种方法。

![](img/a0f0f8622efaa2445afd26776fd0fbb7_15.png)

![](img/a0f0f8622efaa2445afd26776fd0fbb7_17.png)

1.  **添加音乐**
    选中视频轨道，点击轨道下方的 **添加音频** 按钮，或直接点击底部菜单的 **音频**。然后选择 **添加音乐**，系统将跳转至剪映的音乐素材库。此曲库与抖音热门音乐互通，并按类别（如卡点、流行、旅行、美食等）和热门推荐进行了划分。
    例如，选择 **夏天的风** 这首音乐，点击 **使用** 即可将其添加到时间轴。
    添加后，需要调整音频位置并对齐视频开头。接着，关闭视频原声以避免干扰。预览时若发现音频长度超过视频，可使用 **分割** 功能裁切掉多余部分。
    为了让音乐过渡更自然，可以对音频进行淡化处理。选中音频轨道，点击 **淡化**，分别设置 **淡入时长** 和 **淡出时长**（例如各1-2秒）。
    此外，还可以调整音频的 **音量**。由于平台压缩可能导致音量变小，建议适当调高。**踩点** 功能用于卡点视频制作，**变速** 功能可调整音频播放速度（范围0.2倍至4倍）。

2.  **音效**
    音效常用于Vlog、采访等视频中，能增加趣味性或突出特定动作。在音频界面选择 **音效**，即可进入丰富的音效素材库，包含游戏、魔法、美食、动物、环境等多种类别。选择合适的音效点击使用即可。



![](img/a0f0f8622efaa2445afd26776fd0fbb7_29.png)

![](img/a0f0f8622efaa2445afd26776fd0fbb7_30.png)

![](img/a0f0f8622efaa2445afd26776fd0fbb7_31.png)

![](img/a0f0f8622efaa2445afd26776fd0fbb7_32.png)

3.  **提取音乐**
    此功能可将手机相册中某个视频的音频单独提取出来，用于当前剪辑。操作时，选择 **提取音乐**，从相册选取一个带音频的视频文件，系统便会将其音频提取并添加到时间轴上。



![](img/a0f0f8622efaa2445afd26776fd0fbb7_34.png)

![](img/a0f0f8622efaa2445afd26776fd0fbb7_36.png)

4.  **抖音收藏**
    若想在抖音上使用热门音乐拍同款，可先在抖音APP中收藏该音乐。在剪映的音频界面选择 **抖音收藏**，即可直接使用已收藏的音乐进行剪辑。



![](img/a0f0f8622efaa2445afd26776fd0fbb7_38.png)

![](img/a0f0f8622efaa2445afd26776fd0fbb7_40.png)

5.  **录音**
    此功能可为视频录制后期旁白或解说。在音频界面点击 **录音**，按住红色按钮即可开始录制。录制完成后，音频会出现在时间轴上，可对其音量等进行调整。

![](img/a0f0f8622efaa2445afd26776fd0fbb7_42.png)
