## 从销售思维到用户思维

![](img/a2604063870f039a3d950210f096b634_32.png)

![](img/a2604063870f039a3d950210f096b634_34.png)

![](img/a2604063870f039a3d950210f096b634_36.png)

单纯以销售思维制作内容，往往会直接强调产品优势或促销信息。例如，一个医美账号可能会说：“我是医美机构，我能让你变美，我们的医生很专业。” 或者卖书的账号会说：“这本书原价59元，今天七夕特价49元。”

![](img/a2604063870f039a3d950210f096b634_38.png)

![](img/a2604063870f039a3d950210f096b634_40.png)

![](img/a2604063870f039a3d950210f096b634_42.png)

这样的内容在**淘宝**这类电商平台是可行的，因为用户打开淘宝时，本身就带有明确的购物需求和消费预期。公式可以表示为：

![](img/a2604063870f039a3d950210f096b634_44.png)

![](img/a2604063870f039a3d950210f096b634_46.png)

![](img/a2604063870f039a3d950210f096b634_48.png)

![](img/a2604063870f039a3d950210f096b634_50.png)

**电商平台内容逻辑：用户意图（购物） → 接受促销信息 → 产生购买**

![](img/a2604063870f039a3d950210f096b634_52.png)

![](img/a2604063870f039a3d950210f096b634_54.png)

![](img/a2604063870f039a3d950210f096b634_56.png)

然而，在**抖音**这类娱乐平台，这套逻辑行不通。用户打开抖音的核心目的是娱乐和放松。如果此时强行推送广告，会引起用户的反感。平台逻辑的差异是根本原因：

![](img/a2604063870f039a3d950210f096b634_58.png)

![](img/a2604063870f039a3d950210f096b634_60.png)

![](img/a2604063870f039a3d950210f096b634_62.png)

**娱乐平台内容逻辑：用户意图（娱乐） → 抵触硬广 → 划走视频**

![](img/a2604063870f039a3d950210f096b634_64.png)

![](img/a2604063870f039a3d950210f096b634_66.png)

![](img/a2604063870f039a3d950210f096b634_68.png)

![](img/a2604063870f039a3d950210f096b634_70.png)

因此，我们必须从“我想卖什么”的销售思维，转向“用户想看什么”的用户思维。

![](img/a2604063870f039a3d950210f096b634_72.png)

![](img/a2604063870f039a3d950210f096b634_74.png)

![](img/a2604063870f039a3d950210f096b634_76.png)

## 用户思维的核心：提供价值与利益

![](img/a2604063870f039a3d950210f096b634_78.png)

![](img/a2604063870f039a3d950210f096b634_80.png)

![](img/a2604063870f039a3d950210f096b634_82.png)

互联网思维有很多种，但最核心、第一个需要掌握的就是**用户思维**。其核心是永远思考用户喜欢什么、需要什么。

![](img/a2604063870f039a3d950210f096b634_84.png)

![](img/a2604063870f039a3d950210f096b634_86.png)

![](img/a2604063870f039a3d950210f096b634_88.png)

我们通过一个具体的例子来理解。假设你学习了专业的灯光知识（如测光、底光、蝴蝶光、轮廓光），并想在抖音上分享，最终目标是知识付费。

![](img/a2604063870f039a3d950210f096b634_90.png)

![](img/a2604063870f039a3d950210f096b634_92.png)

![](img/a2604063870f039a3d950210f096b634_94.png)

以下是制作这个教学视频时，不同思维的体现：

![](img/a2604063870f039a3d950210f096b634_96.png)

![](img/a2604063870f039a3d950210f096b634_98.png)

![](img/a2604063870f039a3d950210f096b634_100.png)

![](img/a2604063870f039a3d950210f096b634_102.png)

### 1. 知识罗列式（非用户思维）
如果你只是随机地、平铺直叙地讲解这几个知识点：
- 测光怎么打，能让视频更清晰。
- 底光怎么打，能让画面更恐怖。
- 蝴蝶光怎么打，能让人更美丽。

![](img/a2604063870f039a3d950210f096b634_104.png)

![](img/a2604063870f039a3d950210f096b634_106.png)

![](img/a2604063870f039a3d950210f096b634_108.png)

同时，标题可能是：“卷心菜教你三种打光方式”。这种内容只是知识的堆砌，没有考虑用户的观看动机。

![](img/a2604063870f039a3d950210f096b634_110.png)

![](img/a2604063870f039a3d950210f096b634_112.png)

![](img/a2604063870f039a3d950210f096b634_114.png)

### 2. 用户思维式
用户思维要求我们审视每一个知识点，思考：“用户看完这个，能获得什么具体的利益或感受？”

![](img/a2604063870f039a3d950210f096b634_116.png)

![](img/a2604063870f039a3d950210f096b634_118.png)

![](img/a2604063870f039a3d950210f096b634_120.png)

![](img/a2604063870f039a3d950210f096b634_122.png)

- **蝴蝶光**：效果是“让人更漂亮”，这直接提供了“变美”的**利益**，对用户有强吸引力。
- **底光**：效果是“让画面更恐怖”，这只是一种**效果**展示，对大多数用户的实用价值较低。
- **测光**：效果是“让视频更清晰”，这提供了“提升视频质量”的**实用价值**。

![](img/a2604063870f039a3d950210f096b634_124.png)

![](img/a2604063870f039a3d950210f096b634_126.png)

![](img/a2604063870f039a3d950210f096b634_128.png)

基于以上分析，用户思维下的视频制作会：

![](img/a2604063870f039a3d950210f096b634_130.png)

![](img/a2604063870f039a3d950210f096b634_132.png)

![](img/a2604063870f039a3d950210f096b634_134.png)

**调整内容顺序**：将最具吸引力、能提供直接利益的内容放在视频最前面。例如，首先展示“蝴蝶光如何让人变漂亮”，抓住用户注意力。

![](img/a2604063870f039a3d950210f096b634_136.png)

![](img/a2604063870f039a3d950210f096b634_138.png)

![](img/a2604063870f039a3d950210f096b634_140.png)

![](img/a2604063870f039a3d950210f096b634_142.png)

**优化视频标题**：标题不应只是概括内容，而要突出用户能获得的利益。对比以下两个标题：
- 普通标题：`人人必会的三个打光神技`
- 用户思维标题：`能把姑娘变漂亮的打光方式，我只教一次`

![](img/a2604063870f039a3d950210f096b634_144.png)

![](img/a2604063870f039a3d950210f096b634_146.png)

![](img/a2604063870f039a3d950210f096b634_148.png)

用户思维标题直接点明了“变漂亮”这一核心利益，更能吸引目标用户点击。

![](img/a2604063870f039a3d950210f096b634_150.png)

![](img/a2604063870f039a3d950210f096b634_152.png)

## 实践要点：贯穿始终的思考

![](img/a2604063870f039a3d950210f096b634_156.png)

![](img/a2604063870f039a3d950210f096b634_158.png)

![](img/a2604063870f039a3d950210f096b634_160.png)

用户思维不仅体现在标题和开头，它需要贯穿内容创作的每一秒。你需要不断自问：
- 用户看到这里会有什么反应？
- 这一秒的内容对他有用吗？有趣吗？
- 他是否有继续看下去的动机？

![](img/a2604063870f039a3d950210f096b634_162.png)

![](img/a2604063870f039a3d950210f096b634_164.png)

![](img/a2604063870f039a3d950210f096b634_166.png)

![](img/a2604063870f039a3d950210f096b634_168.png)

即使是7-8秒的短视频，也需要对每一帧进行这样的思考与设计，确保全程都能抓住用户的兴趣。

本节课中我们一起学习了抖音内容创作的黄金法则——用户思维。我们认识到抖音作为娱乐平台与淘宝等电商平台的根本区别，从而必须将思维从“我要卖什么”转变为“用户想看什么”。通过分析具体案例，我们掌握了如何通过调整内容顺序、优化标题来突出用户利益，并将这种思考方式贯穿于视频创作的每一秒。记住，在抖音上，提供价值、满足用户娱乐或学习需求的内容，远比硬性推销更容易成功。
