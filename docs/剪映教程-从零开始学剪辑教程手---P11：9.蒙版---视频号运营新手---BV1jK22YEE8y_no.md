# 剪映教程：P11：9.蒙版与关键帧 🎬

在本节课中，我们将要学习剪映中两个核心且强大的功能：**蒙版**和**关键帧**。我们将首先认识蒙版的基本概念和界面操作，然后学习如何结合关键帧制作出流畅的无缝转场效果。这两个功能是提升视频剪辑水平的关键。

## 认识蒙版 🎭

蒙版功能允许我们对视频画面进行局部显示或隐藏，从而创造出丰富的视觉效果。简单来说，蒙版就像一个“形状切割器”，可以决定视频的哪一部分被显示出来。

以下是蒙版的基本操作步骤：

1.  **导入素材**：首先，在剪映中导入至少两段视频素材。
2.  **找到蒙版工具**：选中需要添加蒙版的视频片段，在下方菜单栏中找到并点击“**蒙版**”按钮。
3.  **选择蒙版形状**：系统提供了多种预设形状，包括：
    *   **线性**：创建一条分界线，一侧显示，一侧隐藏。
    *   **镜面**：创建两条平行线，中间部分显示，两侧隐藏。
    *   **圆形**：创建一个圆形显示区域。
    *   **矩形**：创建一个矩形显示区域。
    *   **爱心**/**星形**：创建爱心或星形显示区域。
4.  **调整蒙版**：选择形状后，可以通过以下方式调整：
    *   **拖动边缘**：调整蒙版的大小和范围。
    *   **旋转**：用双指旋转蒙版，改变其角度。
    *   **羽化**：拖动边缘的**双箭头图标**，可以柔化蒙版边缘，实现自然的过渡效果。

**核心概念**：蒙版作用于哪个图层，哪个图层的画面就会被蒙版形状所限定。例如，使用圆形蒙版，则只有圆形区域内的画面会显示，其余部分会被遮盖。

上一节我们介绍了蒙版的基本操作，本节中我们来看看如何将视频素材放入画中画轨道来配合蒙版使用。

有时，直接选中素材可能无法使用“切画中画”功能。此时，可以按照以下步骤操作：

1.  点击底部工具栏的“**画中画**”选项。
2.  选择“**新增画中画**”。
3.  从相册中再次导入需要的视频素材。
4.  新增的画中画图层会自动缩小，可以手动将其调整至合适大小。

另一种更快捷的方法是：在主轨道上，直接将需要作为画中画的视频片段**向左拖动**，直到出现“画中画”轨道标识后松开，即可快速切入画中画。

## 结合关键帧制作无缝转场 🔄

关键帧是记录画面属性（如位置、大小、蒙版形状等）在时间轴上变化的工具。通过设置起始和结束两个关键帧，软件会自动生成中间的变化过程，从而形成动画。

下面，我们学习如何用蒙版结合关键帧制作一个简单的无缝转场效果。

1.  **准备素材**：将两段视频上下叠加在时间轴上（一段在主轨道，一段在画中画轨道）。理想情况下，素材本身带有一些运镜或遮挡效果，转场会更自然。
2.  **添加起始蒙版与关键帧**：
    *   选中画中画图层，在视频**开始位置**添加一个**线性蒙版**。
    *   旋转并调整蒙版角度，并适当羽化边缘。
    *   在视频开始处，点击添加**关键帧**图标（菱形按钮）。这个关键帧记录了蒙版初始状态。
3.  **设置结束蒙版与关键帧**：
    *   将时间线移动到视频的**结束位置**。
    *   点击“蒙版”按钮，在蒙版编辑界面中，将线性蒙版的**分界线完全拖出画面之外**。
    *   此时，软件会自动在时间线当前位置生成一个新的关键帧，记录蒙版的最终状态。
4.  **预览效果**：播放视频，你会看到画中画图层随着蒙版分界线的移动逐渐显现或消失，与主轨道视频形成无缝衔接的转场。

**核心概念**：关键帧动画的原理是 `起始状态（关键帧1） -> 自动补间 -> 结束状态（关键帧2）`。我们只需定义好开头和结尾，软件会自动计算中间的过渡动画。

## 关键帧的其他应用 ✨

除了控制蒙版，关键帧还可以独立用于创建各种动画效果。例如，让一个静态图片或视频移动起来：

1.  在视频的起始位置，点击添加**关键帧**。
2.  将时间线向后移动一段距离。
3.  在预览窗口中，直接**拖动视频画面**到另一个位置。
4.  此时，软件会自动在当前时间点生成第二个关键帧。
5.  播放预览，视频就会从第一个关键帧的位置平滑移动到第二个关键帧的位置。

你甚至可以在中间添加更多关键帧，让运动路径变得更加复杂和有趣。

## 课程总结 📝

本节课中我们一起学习了：
1.  **蒙版**：它是一个形状工具，用于控制视频画面的显示区域，可以实现局部显示、画面叠加等效果。
2.  **关键帧**：它是制作动画的基础，通过记录画面属性在不同时间点的状态，让软件自动生成平滑的过渡动画。
3.  **无缝转场**：结合蒙版和关键帧，我们可以手动制作出比内置转场更灵活、更匹配视频内容的无缝转场效果。
