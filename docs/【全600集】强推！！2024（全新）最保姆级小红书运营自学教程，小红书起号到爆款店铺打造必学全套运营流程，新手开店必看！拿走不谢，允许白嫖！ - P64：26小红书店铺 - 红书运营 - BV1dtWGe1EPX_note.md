## 商品准备 🛍️

商品准备是开店的第一步，核心在于确定店铺的产品结构并掌握正确的上架节奏。

### 产品基础数量要求

店铺的商品总数应控制在7至15件之间。具体的产品结构搭配如下：

*   **主产品**：店铺的核心爆款，销量最高。
*   **副产品**：辅助销售的产品，能产生少量订单即可。
*   **补量产品**：用于丰富店铺品类，初期无需产生实际销量。

以下是不同店铺模式下的产品搭配方案：

*   **多店模式**：每个店铺准备7件产品。搭配方案为：1件主产品 + 2件副产品 + 4件补量产品。
*   **单店模式**：店铺最多可上架15件产品，但需分阶段操作。
    1.  **初期**：先上架7件产品，搭配为：1件主产品 + 2件副产品 + 4件补量产品。
    2.  **推广第二款爆品时**：调整为：2件主产品 + 1件副产品 + 其余为补量产品。可通过下架一件补量产品，再上新主产品来实现。
    3.  **推广第三款爆品时**：结构变为：3件主产品 + 其余全为补量产品。
    4.  **后续**：当店铺数据稳定后，所有产品均可转为正价销售品。

**核心原则**：店铺总商品数 ≤ 15件。这是为了维持健康的店铺动销率，便于运营操作。

### 商品发布技巧

正确的上架顺序和时间能帮助产品获得更好的系统初始权重。

以下是商品发布的具体步骤和注意事项：

1.  **首日**：上架1-2件**副产品**。目的仅是让小红书系统识别并收录你的店铺和商品，建立基础排名。**不要**为这些副产品发布推广笔记。
2.  **次日及以后**：开始上架**主产品**。每天上架的新商品不要超过3件。
    *   **快速节奏示例**：第一天（2副+1补）、第二天（1主+2副）、第三天（1副）→ 共7件产品上架完毕。
    *   **推荐节奏**：每天上架1-2件产品，拉长上架周期，效果更稳定。
3.  **上架时间**：
    *   **副产品**：可在任意时间段上架。
    *   **主产品**：必须在**流量高峰期**上架。平台高峰期通常为中午11:00-14:00和晚上17:00-20:00。
        *   选择其中一个高峰期作为你店铺固定的主产品上架时段。
        *   操作流程：在高峰期上架主产品 → 等待30-60分钟让系统充分收录 → 再开始进行推广宣传。

**重要提示**：如果跳过“先上副产品”的步骤，直接上架主产品并立即推广，系统可能因未充分收录店铺信息而无法给予有效权重，导致前期推广效果不佳。
