# 剪映电脑版自学教程：P14：12. 抖音核心剪辑功能—色度抠图 🎬

在本节课中，我们将要学习剪映电脑版中一个非常强大的核心剪辑功能——**色度抠图**。这个功能可以让我们轻松地将绿幕背景的视频素材抠出，并合成到其他视频中，从而制作出各种特效视频。

色度抠图是剪映近期新增的一项功能，它极大地方便了用户在手机端进行视频合成。过去，这类操作通常需要在专业的软件如Premiere或After Effects中完成。现在，通过剪映的色度抠图，我们可以快速实现绿幕抠像，将特效素材无缝融入自己的视频项目。

## 色度抠图原理与应用

上一节我们介绍了色度抠图的基本概念，本节中我们来看看它的具体应用场景。色度抠图最典型的应用就是处理**绿幕素材**。通过移除视频中特定的颜色（通常是绿色或蓝色），我们可以将前景物体分离出来，叠加到新的背景上。

以下是色度抠图的核心操作步骤：

1.  **导入素材**：首先，需要准备两段视频。一段是作为背景的主视频，另一段是带有纯色背景（如绿色）的特效素材。
2.  **使用画中画**：将绿幕素材以**画中画**的形式叠加到主视频轨道上方。
3.  **应用色度抠图**：选中画中画素材，在编辑面板中找到**色度抠图**功能。
4.  **取色与调整**：使用取色器吸取绿幕的颜色，并通过调整**强度**和**阴影**参数，使抠图效果干净、自然。

## 实战演示：制作飞机飞过天空的特效

![](img/938fbc4abb4c925c01c4270ad7229116_1.png)

了解了基本原理后，我们通过一个实例来具体操作。我们的目标是让一架飞机飞过一段天空视频。

首先，我们导入两段素材：一段是天空背景视频，另一段是绿幕背景的飞机视频。



接着，我们将飞机视频通过**画中画**功能添加到天空视频上方。未抠图前，飞机视频会带着绿色背景显示。



![](img/938fbc4abb4c925c01c4270ad7229116_3.png)

现在，我们开始使用色度抠图功能移除绿色背景。

![](img/938fbc4abb4c925c01c4270ad7229116_5.png)

1.  选中画中画轨道上的飞机素材。
2.  在编辑面板中找到**色度抠图**工具（通常位于蒙版工具旁边）。
3.  使用**取色器**（小手图标）点击画面中的绿色背景。
4.  缓慢向右拖动**强度**滑块，观察绿色背景逐渐消失。调整到背景完全被移除，同时飞机边缘清晰为止。
5.  可以适当增加**阴影**值，让合成后的飞机边缘更柔和，与背景融合得更自然。



![](img/938fbc4abb4c925c01c4270ad7229116_7.png)

抠图完成后，我们可能还需要对素材进行一些优化处理：

*   **裁剪水印**：如果素材带有水印，可以使用**编辑**中的**裁剪**功能将其移除。
    

![](img/938fbc4abb4c925c01c4270ad7229116_9.png)

![](img/938fbc4abb4c925c01c4270ad7229116_11.png)

*   **调整位置与时长**：根据背景视频的构图，调整飞机的位置、大小。同时，裁剪掉素材多余的开头和结尾部分，使飞机的出现和消失时机更符合逻辑。
    

![](img/938fbc4abb4c925c01c4270ad7229116_12.png)

*   **添加动画**：为了让飞机入场更自然，可以为其添加一个**渐显**动画。



经过以上步骤调整后，我们便完成了一个简单的特效合成。最终效果是飞机从天空背景中飞过，实现了逼真的合成效果。
