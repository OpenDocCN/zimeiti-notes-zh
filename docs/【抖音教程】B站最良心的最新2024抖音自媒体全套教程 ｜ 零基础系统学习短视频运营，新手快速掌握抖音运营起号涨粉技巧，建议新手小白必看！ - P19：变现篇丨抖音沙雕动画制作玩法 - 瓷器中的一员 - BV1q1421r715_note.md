# 抖音自媒体教程：P19：变现篇丨抖音沙雕动画制作玩法 🎬

在本节课中，我们将学习一种被称为“懒人做法”的抖音沙雕动画制作方法。这种方法无需复杂技术或原创配音，仅需一部手机即可完成，旨在帮助新手快速掌握制作自带流量动画作品的技巧。

## 概述：沙雕动画的“懒人做法”

这种沙雕动画的制作方法只有少数人知道。它不需要你掌握任何技术，也不需要你编写段子或配音，就能让作品自带流量。虽然它看起来复杂，似乎不是一个人能完成的，但其实一部手机就能制作。

![](img/0f05f204d4e3ffd2323171aa3a090899_1.png)

## 第一步：软件设置与素材选择

上一节我们介绍了沙雕动画的便捷性，本节中我们来看看具体的制作起点。首先，你需要打开指定的制作软件。

![](img/0f05f204d4e3ffd2323171aa3a090899_3.png)

在制作上，你只需要打开这个软件，并选择合适的视频比例。对于抖音，选择九比十六的比例；对于西瓜视频，选择十六比九的比例。分辨率选择1080P会使视频更清晰。

![](img/0f05f204d4e3ffd2323171aa3a090899_5.png)

帧率设置保持默认不动即可。在场景选择板块中，有四个分类，例如你可以选择一个“古代”场景。

向右滑动界面，你就能看到该分类下的全部场景。角色选择的操作方式也是一样的，分类中有许多角色模型，包括沙雕风格、武侠风格、斗罗大陆风格和奥特曼风格等，非常丰富。

![](img/0f05f204d4e3ffd2323171aa3a090899_7.png)

## 第二步：动画制作与编辑

当我们把场景和角色都选择好后，就可以开始制作动画了。它的操作方式与手机剪映软件相似。

以下是动画制作的核心步骤：
1.  **添加动作**：点击角色，选择添加动作。
2.  **预览动作**：点击播放，查看人物动作效果。
3.  **调整内容**：如果不喜欢当前动作，可以更换。你也可以再添加其他角色，用来丰富情节。
4.  **合成动画**：最后，导入音频文件，软件即可合成出完整的动画视频。

这个过程是不是很简单呢？

## 第三步：配音的“懒人玩法”

动画制作完成后，配音是赋予沙雕动画灵魂的关键。下面教你一个无需自己编段子也能获得流量和收益的懒人方法。

![](img/0f05f204d4e3ffd2323171aa3a090899_9.png)

这个方法就是“拍同款”。一些聪明的博主已经开始使用这个方法。具体操作是，你去寻找一些搞笑博主的视频，直接使用他的原声来为你的沙雕动画配音。

这样做有三个主要好处：

1.  **原创度高**：作品原创度极高，不会被平台限制播放和收益。
2.  **蹭取流量**：你可以蹭到大博主的流量。有些用户喜欢到原声作品合集里，观看其他人的模仿视频，这为你带来了额外的曝光。
3.  **节省精力**：省去了自己编段子和配音的精力，提高了制作效率。

有了节省下来的时间，你就能多运营一个账号，从而获得双份的流量与收益。所以，有时你缺少的不是资源，而是发现资源的眼光。
