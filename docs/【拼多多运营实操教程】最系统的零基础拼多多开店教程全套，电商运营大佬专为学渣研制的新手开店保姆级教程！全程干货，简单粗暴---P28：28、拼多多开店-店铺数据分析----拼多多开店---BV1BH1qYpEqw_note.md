### 1. 流量数据分析

![](img/a0d9e1b77ebd961f23fefc24ce1bd258_24.png)

流量数据直接反映了店铺的曝光和访客情况。当发现店铺访客数下降时，我们需要按步骤排查原因。

![](img/a0d9e1b77ebd961f23fefc24ce1bd258_26.png)

![](img/a0d9e1b77ebd961f23fefc24ce1bd258_27.png)

以下是分析流量下降的步骤：
1.  **确认市场大盘**：首先观察整个行业的大盘趋势。如果市场平稳或上升，而你的店铺数据下降，则属于异常情况，需要重点排查。
2.  **检查竞争环境**：查看是否有同款或相似款商品通过报名活动或竞价方式，抢占了你的流量资源位。如果是，需及时调整自身活动策略。
3.  **分析核心指标**：若排除外部竞争，则需深入分析店铺内部数据。首要关注**支付转化率**。可以查看近期（如过去4-7天）的转化率数据。
    *   如果转化率稳定且良好，流量不应异常下降。
    *   如果转化率波动大且**低于行业均值**，系统可能会判定你的商品流量承接能力弱，从而减少曝光，导致访客下降。转化率公式可表示为：
        **支付转化率 = (支付买家数 / 访客数) × 100%**
4.  **检查其他关联指标**：如果转化率正常，则需进一步检查**客单价**、**成交额**、**UV价值**等数据。任何一个指标表现薄弱，都可能影响整体流量。可以通过运营手段（如设置优惠券、满减）或引导客户行为来优化这些数据。

### 2. 交易数据分析

交易数据能帮助我们理解店铺的成交质量和客户结构。

![](img/a0d9e1b77ebd961f23fefc24ce1bd258_31.png)

![](img/a0d9e1b77ebd961f23fefc24ce1bd258_32.png)

![](img/a0d9e1b77ebd961f23fefc24ce1bd258_33.png)

在交易数据中，除了成交额外，还需特别关注以下两点：
*   **老客户占比**：老客户复购率高，说明产品和服务获得了认可。如果老客户成交占比高，应重点优化拉新数据；如果占比低，则需加强客户维护。
*   **店铺关注用户数**：这个数据反映了用户的粘性和精准度。
    *   如果关注数持续健康增长，说明进店用户精度高。
    *   如果店铺访客下降的同时，关注数也在减少或增长缓慢，则可能是**引流不精准**导致转化下降，进而引发流量下滑。

![](img/a0d9e1b77ebd961f23fefc24ce1bd258_35.png)

![](img/a0d9e1b77ebd961f23fefc24ce1bd258_36.png)

### 3. 商品数据分析

商品数据可以帮助我们定位到具体出问题的商品。

在“商品数据”中，“商品概况”提供了整体视图，而“商品明细”则更为关键。在商品明细中，可以查看每个商品的具体数据，并通过“查看详情”分析其在不同时间周期的数据曲线。

![](img/a0d9e1b77ebd961f23fefc24ce1bd258_43.png)

![](img/a0d9e1b77ebd961f23fefc24ce1bd258_44.png)

![](img/a0d9e1b77ebd961f23fefc24ce1bd258_45.png)

![](img/a0d9e1b77ebd961f23fefc24ce1bd258_46.png)

以下是分析单品问题的思路：
1.  **定位问题商品**：当店铺整体数据下滑时，通过商品明细找出是哪个或哪些商品的数据在下降。
2.  **分析商品指标**：针对问题商品，重点分析其**转化率**、**成交额**、**流量来源**及**流量流失情况**。
3.  **制定优化策略**：
    *   如果发现流失严重到竞品店铺，需警惕竞争。
    *   如果只是自身转化率或客单价低，则可以通过**优化主图详情页**、**设置营销活动（如优惠券、满减、秒杀）**、**调整价格策略**等方式来提升。
