# 抖音运营教程：P26：人物设定

## 概述
在本节课中，我们将要学习抖音账号运营中一个至关重要的环节：**人物设定**。我们将探讨如何像设计游戏角色一样，为你的抖音账号策划一个真实、立体且富有吸引力的人格化形象，并了解如何通过“超级符号”强化人设，从而提升账号的辨识度、信任度与变现能力。

## 人设策划的核心概念

上一节我们介绍了账号定位的基础，本节中我们来看看如何将定位具体化为一个鲜活的人物。

![](img/76739ed330cefabd877b0fe75baa171d_4.png)

在抖音中策划人设，类似于在游戏中创建角色。你需要预先设定角色的**称呼、职业、技能**等属性。然而，与游戏角色不同，抖音中的人物设定是**人生角色**，因此必须更具真实感。

无论是短视频还是直播，一个成功的IP（人格化形象）能够解决粉丝的信任问题，起到裂变传播的作用，并具备强大的变现能力。因此，我们运营账号——无论是创作内容还是策划选题——其核心目的都是在获取流量的同时**打造IP**。

![](img/76739ed330cefabd877b0fe75baa171d_6.png)

打造IP的核心，是在特定的场景或情境中，完成**账号人格化**的过程。这个过程需要注入情感、塑造故事，并贴上具有辨识度的标签。

![](img/76739ed330cefabd877b0fe75baa171d_8.png)

**账号人格化 = 注入情感 + 塑造故事 + 贴上标签**

这个“人格化”的结果，就是我们通常所说的**账号人设**。

![](img/76739ed330cefabd877b0fe75baa171d_10.png)

## 人设策划的六大维度

![](img/76739ed330cefabd877b0fe75baa171d_12.png)

对于账号人设的策划，我们可以从以下六个具体方面着手。以下是每个维度的详细说明：

![](img/76739ed330cefabd877b0fe75baa171d_14.png)

1.  **名字/姓氏/昵称**
    一个IP首先需要一个称呼，这是IP传播的名片。必须明确这个IP叫什么，其重要性如同一个品牌的商标名。

2.  **职业/职务/职称**
    人格化必然要具备身份。职业是用户认识这个人或IP的联想入口，例如：老师、教授、博士、律师。

![](img/76739ed330cefabd877b0fe75baa171d_16.png)

3.  **称呼/称谓/社会关系**
    一个恰当的称呼能快速拉近与用户的距离，产生移情效果。例如：XX哥、XX大爷、XX姥姥。

![](img/76739ed330cefabd877b0fe75baa171d_18.png)

4.  **形象/第一印象**
    从用户视角塑造IP形象，使其更容易占据用户心智。例如：大叔形象、萌萌哒、铁憨憨、甜美可爱、光头。

5.  **性格/爱好/特点**
    一个具有鲜明特点的IP，不仅记忆点强，而且更真实。过于完美的形象反而容易脱离大众。例如：暴躁、娘炮、爱吃、爱喝酒。

6.  **穿着打扮/财富地位**
    借助社会普遍认知和刻板印象，可以快速在用户心智中为IP定位。例如：有钱代表会赚钱，开豪车代表成功人士，打扮土气可能代表老实人。

## 从人设到账号搭建

将以上六个维度的内容策划好后，下一步就是结合你自身的产品或服务优势，将其落实到**抖音账号的五件套**中：
*   头像
*   昵称
*   背景图
*   简介

## 提炼人设的“超级符号”

除了对账号进行人格化策划，还需要通过持续的选题与内容策划，为你的人设提炼出独特的**记忆点**，即“超级符号”。

超级符号的提炼可以从以下几个方向入手：

1.  **画风（视觉）**
    *   **场景与道具**：固定的拍摄环境或标志性物品。
    *   **服饰**：具有个人特色的着装风格。

2.  **听觉**
    *   **BGM/特效音**：专属的背景音乐或音效。
    *   **配音与声线**：独特的嗓音或说话方式。

3.  **文案**
    *   能够反映核心优势且易于记忆的提炼词或口号。例如：“XX，简单粗暴”。

4.  **表情与情绪**
    *   标志性的情绪传递方式，如夸张的笑、惊恐、无奈等表情包式演绎。

5.  **语言特色**
    *   **方言/口音**：独特的发音。
    *   **语速语调**：如慢条斯理、抑扬顿挫。

6.  **动作与习惯**
    *   习惯性的肢体动作，例如：撩头发、扶眼镜、标志性手势。

**代码示例：超级符号检查清单**
```plaintext
超级符号清单：
- [ ] 视觉符号（场景/道具/服饰）: _________
- [ ] 听觉符号（BGM/声线）: _________
- [ ] 文案符号（口号/金句）: _________
- [ ] 表情符号（标志性情绪）: _________
- [ ] 语言符号（口音/语调）: _________
- [ ] 动作符号（习惯性动作）: _________
```

当超级符号提炼完成后，在你进行短视频创作时，就应不断地在视频中重复呈现这些元素，强化用户的记忆。

## 总结
本节课中，我们一起学习了抖音账号**人物设定**的完整流程。我们首先明确了人设的本质是**账号人格化**，然后从**名字、职业、称呼、形象、性格、穿着**六个维度拆解了人设的策划方法。接着，我们探讨了如何将人设落地到账号基础设置中，并最终通过提炼视觉、听觉、文案、表情、语言、动作等多方面的**超级符号**，为人设注入强大的记忆点和传播力。记住，一个成功的人设是连接你与粉丝，并实现账号长期价值的关键。
