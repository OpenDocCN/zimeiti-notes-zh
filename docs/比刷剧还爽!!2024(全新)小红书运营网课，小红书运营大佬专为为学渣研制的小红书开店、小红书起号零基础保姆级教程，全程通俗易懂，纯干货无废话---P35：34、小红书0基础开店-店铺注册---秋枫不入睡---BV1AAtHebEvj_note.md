# 小红书运营：P35：34、小红书0基础开店-店铺注册 📝

![](img/6e134b407aa01faf0a7600a4e0dabaad_1.png)

在本节课中，我们将要学习如何从零开始完成小红书店铺的注册。这是开启小红书电商之旅的第一步，我们将详细拆解每一步操作，确保你能顺利完成。

上一节我们介绍了开店前的准备工作，本节中我们来看看如何具体操作，完成店铺的注册流程。

## 注册前的准备

店铺注册有一些需要我们特别注意的选项。因为其注册入口在后台系统中隐藏得比较深，不像其他平台那样直接打开就能找到。

以下是注册前需要确认的事项：
*   使用个人手机即可操作。如果有备用或工作手机，使用它们进行操作也可以。
*   在手机上**下载小红书App**。

## 账号登录与入口寻找

首先，我们需要用手机号登录小红书账号。登录后，操作步骤如下：

1.  点击App右下角的**个人头像**，进入个人中心。
2.  在个人中心页面的**右上角**（或左上角），点击**三条横杠**的菜单图标。
3.  在弹出的菜单中，选择 **“创作中心”**。
4.  进入创作中心后，点击 **“更多服务”**。
5.  在更多服务的页面中，大约在**第三排**的位置，找到并点击 **“开通店铺”** 的方框。该页面通常有多个功能方框，我们只需关注“开通店铺”即可。

## 开始认证与填写信息

点击“开通店铺”后，系统会显示一个引导界面。我们直接点击界面上的 **“立即认证”** 按钮即可开始注册流程。

接下来，按照系统的正常步骤操作，提交事先准备好的材料（如身份证正反面照片），并选择店铺类型。

**关于店铺类型的选择**：系统通常会提供个人店、企业店、旗舰店等类型。对于新手，强烈建议选择 **“个人店”**。

> 因为个人店本来就是让我们上手实验操作的。做任何项目，我们都需要一个熟悉的过程。熟悉了流程和模式后，再进行更深入的操作。前期开个人店，先把所有流程摸清，可以避免因初期投入过大、挫折感强而失去兴趣。

选择“个人店”后，进入填写店铺信息的环节。这些信息后期可以修改，但建议尽量填写准确，避免后续不必要的更改。

## 完成注册与等待审核

店铺信息填写完成后，提交并完成认证。至此，小红书店铺的注册申请就提交成功了。

注册申请提交后，平台需要时间进行审核。审核反馈时间通常为**3天**，但实际结果可能在**5分钟到1天左右**内返回。与其他电商平台相比，小红书的系统反应可能稍慢一些，这是正常现象。

## 重要补充：下载客服工作台

本节课最后一个关键点是：店铺注册成功后，**必须在手机里额外下载“小红书客服工作台”App**。

这个工作台专门用于处理店铺的**售后与客户沟通**。当店铺产生订单后，与买家的聊天沟通无法在普通的小红书App内完成，必须通过这个独立的客服工作台进行操作。因此，务必提前下载好。
