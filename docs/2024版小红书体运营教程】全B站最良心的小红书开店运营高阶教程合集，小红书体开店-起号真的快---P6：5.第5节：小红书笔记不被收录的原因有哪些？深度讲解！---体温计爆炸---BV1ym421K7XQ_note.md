# 小红书运营教程：P6：小红书笔记不被收录的原因深度解析 🔍

在本节课中，我们将深入探讨小红书笔记发布后未被平台收录的常见原因。理解这些原因对于确保您的内容能够正常曝光和传播至关重要。

上一节我们介绍了笔记发布的基本流程，本节中我们来看看哪些因素会导致笔记无法被系统收录。很多人发布了笔记，但平台并未收录，这通常由以下五点原因导致。

以下是五点核心原因的具体分析：

1.  **存在违规推广行为**
    推广相关商品时，笔记内容可能包含了硬性广告信息，例如直接出现品牌名称等。

2.  **包含平台敏感词**
    笔记或账号内容中如果存在违规敏感词，即使笔记暂时被收录，后续系统检测到后也会将其剔除。

3.  **内容本身违规**
    此点涵盖图片和文字两部分的违规情况。可能直接导致不被收录，也可能在收录后因被举报而限流。
    *   **图片方面**：建议不要带有其他平台的水印或二维码，此类引流方式已不适用于小红书。
    *   **文字方面**：必须避免出现违规词、敏感词、不文明词汇或诱导性内容。

    此外，需要特别强调，必须严格遵守所有平台的通用规则，**绝对不要**涉及政治敏感、破坏民族团结、颠覆国家政权等原则性内容。我们从事网络项目应通过合法营销获取合理收入，切勿触碰法律法规明令禁止的词汇及其谐音。

4.  **发布频率异常**
    发送文章频率过高、过快，可能导致账号行为异常。系统会判定为机器人滥发内容，从而不收录笔记。

5.  **发布设备被拉黑**
    平台能够通过读取设备的硬件参数（如串号、IMEI码、手机号等）来识别设备。如果某台设备曾有过违规行为，即使卸载重装APP或更换账号，平台仍能识别并限制该设备发布的内容。

本节课中我们一起学习了导致小红书笔记不被收录的五种主要原因：违规推广、敏感词、内容违规、发布频率异常以及设备被拉黑。发布笔记后若未被收录，可对照以上五点进行排查和调整。
