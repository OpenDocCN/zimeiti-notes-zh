# 小红书运营教程：P51：06.小红书后台的基础操作（电脑端） 💻

在本节课中，我们将学习如何通过电脑端操作小红书商家后台。我们将重点介绍几个关键的后台设置，包括保证金管理、支付方式开通、运费保设置以及地址管理。掌握这些基础操作是顺利开展店铺运营的第一步。

![](img/efae7edc36addb13fc4f903889a17b13_1.png)

![](img/efae7edc36addb13fc4f903889a17b13_3.png)

上一节我们介绍了手机端的相关操作，本节中我们来看看电脑端后台如何设置。

## 一、保证金管理

保证金是开店时需要向平台缴纳的一笔信用担保资金。前期可以暂不缴纳，但若想开通更多支付方式或长期经营，建议缴纳。

![](img/efae7edc36addb13fc4f903889a17b13_5.png)

保证金的金额根据店铺类型和经营类目有所不同。
*   **个人店**：保证金通常在 **1000元** 左右。
*   **个体店**：大部分类目也在 **1000元** 左右，但可以经营一些个人店无法开通的类目（如二手奢侈品、珠宝玉石等）。
*   **企业店**：保证金在 **2000元至数万元** 不等。

![](img/efae7edc36addb13fc4f903889a17b13_7.png)

**保证金特点**：
1.  **可退还**：店铺关闭后，在无违规扣款的情况下，保证金可以全额退还。
2.  **可能扣款**：若发生损害消费者权益的行为（如承诺退款但未执行），平台会从保证金中扣款赔付。
3.  **影响功能**：未缴纳保证金将无法开通微信支付功能。

![](img/efae7edc36addb13fc4f903889a17b13_9.png)

![](img/efae7edc36addb13fc4f903889a17b13_11.png)

**操作路径**：登录小红书千帆后台 -> 点击左侧菜单栏的 **“资金”** -> 选择 **“店铺保证金管理”**。
在此页面，你可以查询应缴金额、进行充值或申请退款。

## 二、支付方式开通

![](img/efae7edc36addb13fc4f903889a17b13_13.png)

开通多元化的支付方式可以方便更多消费者下单，提升转化率。主要支持**支付宝**和**微信支付**两种。

以下是开通步骤：
1.  在后台首页，点击进入 **“资金”** 模块。
2.  找到 **“支付设置”** 或类似选项，里面会列出支付宝和微信支付的开通入口。
3.  **开通支付宝**：通常可以即时完成注册和绑定。
4.  **开通微信支付**：需要先缴纳保证金，审核通过后（通常第二天）方可设置。提现金额将进入注册小店时使用的法人账户。

![](img/efae7edc36addb13fc4f903889a17b13_15.png)

![](img/efae7edc36addb13fc4f903889a17b13_17.png)

**注意**：开通企业店需要绑定营业执照。

## 三、运费保设置

运费保是一项平台服务，旨在解决消费者退货时的运费顾虑，从而提升订单转化率，降低交易纠纷。

**运费保的作用**：当消费者成功退货退款后，系统会自动补贴运费，无需卖家承担。

![](img/efae7edc36addb13fc4f903889a17b13_19.png)

![](img/efae7edc36addb13fc4f903889a17b13_21.png)

**操作路径**：后台 -> **“售后”** 模块 -> 找到 **“运费保”** 或 **“全店运费保服务”**。
建议点击开通此服务，这对提升店铺竞争力有积极作用。

![](img/efae7edc36addb13fc4f903889a17b13_23.png)

![](img/efae7edc36addb13fc4f903889a17b13_25.png)

## 四、地址设置

正确设置发货和退货地址对于订单处理至关重要，尤其对于一件代发（无货源）模式的商家。

![](img/efae7edc36addb13fc4f903889a17b13_27.png)

![](img/efae7edc36addb13fc4f903889a17b13_29.png)

**设置要点**：
1.  **发货地址**：根据你的货源情况填写。如果是一件代发，通常填写供应商所在地或一个统一的物流中转地址。
2.  **退货地址**：这是一个重要技巧。建议填写一个非真实的偏远地区地址（例如新疆），并在地址后附加 **“退货前请联系客服”** 等字样。这样可以在消费者发起退货时，为你争取与供应商或消费者沟通的时间，避免自动同意退款造成损失。
3.  **自动同意退款**：在售后设置中，可以开启“七天无理由退货”等自动同意退款规则，提升消费者体验。

**操作路径**：后台 -> **“订单”** 或 **“物流工具”** -> 选择 **“地址管理”** 或 **“新建地址”**。

![](img/efae7edc36addb13fc4f903889a17b13_31.png)

本节课中我们一起学习了小红书电脑端后台的四个基础操作：保证金的查询与缴纳、支付方式的开通、运费保服务的启用以及地址的策略性设置。这些是店铺搭建的基石，正确配置能为后续的顺利运营铺平道路。请根据你的实际经营模式，逐一完成这些设置。
