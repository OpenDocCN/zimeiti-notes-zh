# 抖音短视频运营教程：P3：小说推文项目详解 📚

在本节课中，我们将要学习一种无需实拍、无需原创文案的抖音变现玩法——小说推文项目。我们将详细拆解其原理、操作步骤与收益模式，让初学者也能快速上手。

## 概述

小说推文是一种通过发布特定视频，引导用户搜索并阅读小说，从而获取平台佣金的项目。其核心优势在于操作简单，无需复杂拍摄与文案创作。

## 项目原理与收益模式

上一节我们介绍了项目的基本概念，本节中我们来看看它的具体运作方式和如何赚钱。

这种视频的变现并非仅依靠中视频计划的播放量收益。视频文案会朗读某部小说的精彩段落，只要有人通过该视频去搜索小说的完整版，那么推荐者就能获得佣金。

其核心模式是 **CPA（按行动付费）**。用户通过作者提供的特定关键词在小说平台完成搜索并成为用户，即算一次有效推荐。每成功推荐一位读者，佣金大约为 **10元**。

视频评论区通常有一条作者置顶的评论，引导用户去搜索。其中包含的关键词，就是作者与平台约定的追踪代码。

## 视频制作全流程

了解了原理后，我们进入实操环节。制作一个小说推文视频主要分为以下几个步骤。

![](img/517b433152ecb5d82f30bb0183c745ff_1.png)

以下是制作视频的具体步骤列表：

1.  **获取文案**：无需自己创作。直接从小说中复制一段约1000字、能引发好奇心的经典段落。
2.  **生成配音**：将复制好的文案粘贴到AI配音工具中，选择一个合适的主播声音，导出音频文件。
3.  **准备素材**：寻找多个解压类视频素材（如史莱姆、切肥皂、洗地毯等）。注意不要只使用一个，需对多个素材进行重新组合剪辑。
4.  **合成视频**：使用剪映等工具，导入配音音频和解压视频素材。将解压视频的原声调小或关闭，确保配音清晰。最后导出成品视频。

整个视频的生产速度很快，基本可以在 **10分钟** 内完成一个。

## 项目规模与收益测算

制作单个视频只是开始，规模化操作才能放大收益。本节我们来算一笔账，看看这个项目的潜力。

根据了解，小说推文项目在抖音上大约每 **1万次播放** 能产生 **2到5单** 转化。每单佣金约为 **10元**。

我们可以做一个测算：
*   假设操作 **10个** 抖音号。
*   每个号每天发布 **3条** 视频。
*   每条视频仅获得抖音基础的 **500** 播放量。
*   那么每日总播放量为：`10 * 3 * 500 = 15,000`。
*   按最低转化率计算，每日预计可出 **5单** 以上，日收益约 **50元+**。

实践中，许多人可以操作二三十个账号。更重要的是，每发布 **100条** 作品，很可能出现一条爆款视频。一旦爆款产生，其带来的播放量和佣金将非常可观，单条视频收益数千元是可能实现的。
