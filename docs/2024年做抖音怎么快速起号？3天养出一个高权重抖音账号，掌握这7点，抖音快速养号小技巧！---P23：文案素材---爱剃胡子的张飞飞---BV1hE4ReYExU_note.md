# 抖音快速起号教程：P23：文案素材 📝

在本节课中，我们将学习如何为抖音视频寻找和获取文案素材。这是将创意转化为具体视频内容的关键一步，掌握高效获取素材的方法，能帮助你快速制作原创内容。

上一节我们介绍了视频创意的来源，本节中我们来看看如何为已确定的创意寻找合适的文案。

## 文案素材来源

确定了视频创意后，就需要搜索相应的文案。直接搬运他人视频素材并进行处理是一种方式。但如果想基于特定创意搜索文案，并以另一种形式制作视频，则需要寻找文字素材。

以下是推荐的一个实用工具。

## 从文案到视频

![](img/7167140061a3257b2c498edb2e80f565_39.png)

找到文案后，下一步就是制作视频。接下来介绍一种制作视频的形式。

![](img/7167140061a3257b2c498edb2e80f565_37.png)

关于视频创意，之前提到的5118热词搜索以及本节课的淘金阁文案搜索，都是有用的工具。大家需要确保自己已经掌握其使用方法。

![](img/7167140061a3257b2c498edb2e80f565_39.png)

课后可以自行搜索练习，尝试找到能用的素材。

接下来就进入视频制作环节。视频制作可以采用真人拍摄，也可以采用音乐影视混剪。音乐影视混剪也有相应的资源网站，或者采用录屏方式。

之前分享过的台词混剪网站、以及如“mv cat”这类可以下载电影的网站，都可以用来获取影视素材进行混剪创作。

![](img/7167140061a3257b2c498edb2e80f565_41.png)

![](img/7167140061a3257b2c498edb2e80f565_41.png)

本节课中我们一起学习了如何利用“淘金阁”这类工具，从海量互联网文章中高效获取文案素材，并明确了将文章转化为视频内容与直接搬运视频的区别。这是进行内容原创和快速生产的重要基础技能。
