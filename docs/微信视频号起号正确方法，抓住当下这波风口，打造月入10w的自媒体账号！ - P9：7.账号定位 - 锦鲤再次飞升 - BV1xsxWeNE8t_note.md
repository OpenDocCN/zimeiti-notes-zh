# 微信视频号运营教程：P9：7.账号定位

## 概述
在本节课中，我们将学习微信视频号账号运营中的一个关键步骤：账号定位。我们将重点解决两个初学者常见的问题：如何快速获取初始粉丝，以及如何解决原创视频素材的难题。课程内容简单直白，旨在帮助初学者快速上手。

---

## 解决初始粉丝与素材问题

上一节我们介绍了账号定位的重要性，本节中我们来看看如何解决起步阶段的两个核心难题。

![](img/2328d9b971cc7a0fa418a39aac08b5a5_1.png)

首先，需要将账号之前发布的作品隐藏。这是账号重新定位时的必要操作。

![](img/2328d9b971cc7a0fa418a39aac08b5a5_3.png)

其次，关于内容发布频率，素材无需着急准备。坚持一天发布一条视频即可。

我们的首要目标是解决获取100个粉丝的问题，然后再集中解决素材问题。请按顺序逐个解决这些问题。

## 如何领取素材

了解福利内容后，以下是领取素材的具体步骤和要求。

![](img/2328d9b971cc7a0fa418a39aac08b5a5_9.png)

领取素材需要满足三个条件：
1.  在公屏上打出“想”字。
2.  在课堂上积极互动。
3.  完整听完课程（后台会统计听课时长与互动次数）。

满足条件后，请按以下流程操作：
1.  找到向你发送听课链接的助理老师。
2.  向助理老师说明是“张老师”让你来领取素材的。提及我的名号会有所帮助。

请注意沟通方式。助理老师与大家并无私交，礼貌沟通并提及我的要求，是顺利领取素材的关键。
