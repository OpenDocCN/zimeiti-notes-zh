## 剪辑是必备技能

上一节我们介绍了混剪的基础概念，本节中我们来看看如何通过二次创作提升视频的原创度。剪辑是视频制作的必备技能。新手刚开始可能觉得困难，根源在于剪辑零基础。自己动手剪辑十几个视频，熟悉每个功能的作用后，再看课程就会觉得非常简单。

## 书单视频的优势

对于做纯视频带货的学员，建议尝试制作书单视频。书单类型是常青树，只要认真操作，基本都能出单。除了按照之前的混剪课程操作，我们还可以进行二次创作。

二次创作，就是将各种视频素材打乱顺序重新组合。这种方式比直接使用一条视频进行混剪的原创度更高。

## 二次创作实操步骤

本节将具体演示如何进行二次创作。我们将以书籍《女孩要懂得保护自己》为例。

### 第一步：寻找并分析对标视频

![](img/f6ea0d64b1a8689dcb2e7eb064d56307_17.png)

1.  打开抖音，搜索目标书籍，例如“女孩要懂得保护自己”。
2.  点击“商品”，按“销量”排序，找到销量高的商品。
3.  进入商品页，点击“达人秀”，这里聚集了大量售卖此书的视频。
4.  选择一个点赞量较高的视频进行分析。观察会发现，这类视频通常由网络素材拼接而成，通过一段文案引出书籍。

### 第二步：提取原始素材与文案

1.  复制视频链接，使用去水印工具（如“配音神器”）保存视频到相册。
2.  使用文案提取工具，将视频的配音文案提取出来。

![](img/f6ea0d64b1a8689dcb2e7eb064d56307_19.png)

### 第三步：生成新的配音

![](img/f6ea0d64b1a8689dcb2e7eb064d56307_21.png)

![](img/f6ea0d64b1a8689dcb2e7eb064d56307_23.png)

1.  检查提取的文案是否有错别字，并进行修正。
2.  使用配音工具（如剪映的“文本朗读”功能），选择合适的声音生成新的配音。
    *   **代码示例（剪映文本朗读思路）**：
        ```python
        # 伪代码：在剪辑软件中选择文本并生成语音
        选中文案文本 -> 点击“文本朗读” -> 选择“女生情感”音色 -> 生成配音
        ```
3.  如果觉得某段语速过快，可以添加停顿，然后重新合成。
4.  将生成的新配音导出保存。

### 第四步：在剪辑软件中替换素材

1.  将去水印后的原视频导入剪映。
2.  **遮盖原字幕**：使用矩形蒙版和模糊背景功能，将原视频的字幕遮盖掉。
3.  **关闭视频原声**。
4.  **替换视频片段**：这是二创的核心。
    *   **方法A（替换）**：将原视频分割，选中要替换的片段，点击“替换”，从素材库或本地导入新素材。
    *   **方法B（删除并新增）**：删除原片段，在对应位置添加自己找到的新素材。确保新素材时长与原片段大致相同。
5.  **寻找新素材的途径**：
    *   在剪映“素材库”中搜索关键词（如“老师讲课”、“学生学习”）。
    *   直接在抖音搜索相关场景关键词（如“女生台上培训”），保存合适的无水印片段使用。
6.  替换素材后，务必**关闭新素材的原声**。

### 第五步：组合成片

![](img/f6ea0d64b1a8689dcb2e7eb064d56307_25.png)

1.  将所有替换好的视频片段按顺序排列。
2.  导入第三步生成的新配音音频。
3.  根据配音调整视频片段的长度和位置，确保音画同步。
4.  添加合适的背景音乐。
5.  通过“识别字幕”功能，为新的配音生成字幕，并调整样式和位置。
6.  检查并遮盖可能残留的旧字幕痕迹。
7.  将帧率调到60，导出视频。

本节课我们一起学习了书单视频的二次创作方法。核心在于**替换原始视频的素材、生成新的配音和字幕**，从而组合成一个原创度更高的新视频。关键操作包括：分析对标视频、提取文案、生成新音频、在剪辑软件中替换素材片段并确保关闭所有原声。这种方法比简单混剪更具原创性，重在实操练习。
