# 新媒体运营课程：P12：01.10.0：ChatGPT 4.0 的几种使用渠道 🚀

## 概述
在本节课中，我们将学习如何获取和使用当前最强大的 AI 对话模型——ChatGPT 4.0。对于新媒体运营人员而言，掌握 AI 工具可以极大地提升工作效率，例如撰写脚本、制作 PPT 和生成报告。本节课将介绍多种体验和使用 ChatGPT 4.0 的渠道，包括免费和付费方式。

## 微软 Edge 浏览器 Copilot 🌐
上一节我们介绍了 AI 对运营工作的重要性，本节中我们来看看第一个推荐渠道：微软 Edge 浏览器中的 Copilot 功能。它是微软基于 ChatGPT 4.0 模型打造的 AI 助手，集成在 Windows 系统中，智能程度与 4.0 模型相近。

![](img/ed66e59c364bbe1334a8233e4326184f_1.png)

![](img/ed66e59c364bbe1334a8233e4326184f_3.png)

使用 Copilot 有两个前提条件：
1.  登录微软账号（可使用 Outlook 邮箱注册）。
2.  需要使用“梯子”（VPN）才能访问该服务。

Copilot 提供了三种对话模式：
*   **更有创造力**：AI 的回答更具开放性和创意。
*   **平衡**：介于创造力和精确性之间，适合一般用途。
*   **更精确**：回答更为严谨和准确，适合专业内容需求。

例如，我们可以输入以下指令来生成脚本：
```
请帮我写一个关于装修的短视频脚本，时长控制在3分钟左右。
```

**优势**：
*   集成在浏览器侧边栏，可基于当前打开的网页内容进行分析。
*   能执行一些简单的系统命令（如切换暗黑模式、打开记事本）。

**缺点**：
*   回答精准度略低于官方 ChatGPT 4.0。
*   连续对话能力受限，无法在同一窗口无限次提问。
*   记忆功能有所缺失，难以基于历史对话优化后续回答。

如果你的 Edge 浏览器没有 Copilot 图标，可以尝试下载 **Dev（开发）频道** 的最新测试版。你可以在 Edge 官网找到下载链接，或通过浏览器“设置”>“关于”检查并更新至最新版本。

## 第三方平台与共享账号 🔗
除了官方集成工具，我们还可以通过第三方平台或共享账号来使用 ChatGPT。以下是几个可用的渠道：

### 1. 灵感 (AI 聚合平台)
这是一个集成了多种 AI 模型的平台，包括 GPT-3.5、GPT-4.0 以及国内主流模型。

**特点**：
*   GPT-4.0 有一定免费使用额度。
*   无需“梯子”即可访问。
*   响应速度可能较慢。

例如，使用 GPT-4.0 模型生成内容：
```
写一篇云南丽江的旅游攻略。
```

### 2. Wee (第三方接口网站)
这个网站提供 GPT-3.5 和 GPT-4.0 模型，速度较快。

**特点**：
*   初始赠送 1万 token（仅限 GPT-3.5 使用）。
*   需要“梯子”访问。
*   具备语音回答功能。

### 3. ChatGPT 账号共享平台
这是一个共享 ChatGPT 账号的网站，账号状态由颜色标识：
*   **绿色**：空闲，可用。
*   **橙色/红色**：正在使用或使用频繁。
*   **金色**：付费的 GPT-4.0 (Plus) 账号。

![](img/ed66e59c364bbe1334a8233e4326184f_5.png)

![](img/ed66e59c364bbe1334a8233e4326184f_7.png)

**使用方法**：
1.  访问共享平台网站。
2.  寻找绿色或金色的可用账号。
3.  点击并使用。注意，付费账号有使用次数限制（例如每3小时约30条消息）。

![](img/ed66e59c364bbe1334a8233e4326184f_9.png)

![](img/ed66e59c364bbe1334a8233e4326184f_11.png)

**注意**：所有通过此方法访问的渠道**都需要“梯子”**。

![](img/ed66e59c364bbe1334a8233e4326184f_13.png)

![](img/ed66e59c364bbe1334a8233e4326184f_15.png)

## 国内可用的 AI 大模型 🇨🇳
如果无法使用“梯子”，我们可以考虑国内公司开发的 AI 大模型。它们无需特殊网络即可访问，虽然能力与 ChatGPT 4.0 有差距，但对于基础运营工作仍有帮助。

以下是几个主流选择：

*   **百度文心一言**：百度推出的 AI 模型，有免费版和付费的 4.0 版本。适合生成基础文案和获取灵感。
    ```
    请帮我写一个关于装修的短视频脚本文案，镜头丰富，语言吸引人。
    ```

*   **字节跳动豆包 AI**：字节跳动推出的 AI 助手。回答结构清晰，能提供景别、画面、台词等详细脚本要素。

![](img/ed66e59c364bbe1334a8233e4326184f_17.png)

*   **其他选择**：阿里的**通义千问**、**天工大模型**等也都是可用的国内 AI 工具。

![](img/ed66e59c364bbe1334a8233e4326184f_19.png)

我们可以根据自身需求，选择一款用起来最顺手的国内 AI 模型来辅助工作。

## 官方 ChatGPT 4.0 体验 🎯
最后，我们对比一下官方 ChatGPT 4.0 的生成效果。使用相同的指令：
```
请帮我写一份关于装修的短视频脚本，文案要求镜头丰富，语言吸引人。
```
官方 4.0 模型生成的脚本在场景设计、叙事节奏和语言感染力上通常更胜一筹，更能体现其强大的内容创作能力。作为运营人员，我们的原则是：**什么工具能高效提升工作，就用什么工具**。

## 总结
本节课我们一起学习了体验和使用 ChatGPT 4.0 的多种渠道：
1.  通过 **微软 Edge Copilot** 间接使用（需梯子）。
2.  利用 **第三方平台**（如“灵感”、“Wee”）或 **共享账号**（需梯子）。
3.  使用 **国内 AI 大模型**（如文心一言、豆包）作为替代方案（无需梯子）。

![](img/ed66e59c364bbe1334a8233e4326184f_21.png)

掌握这些渠道，能帮助你在不同条件下都能利用 AI 为新媒体运营工作赋能。后续课程将继续深入探讨 AI 与运营结合的具体技巧和方法。
