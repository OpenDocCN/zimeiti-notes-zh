# 剪映手机版教学：P44：动漫分屏效果进阶实战

![](img/3e90d40d0b961e81d695da6e57081f07_1.png)

在本节课中，我们将通过一个具体的实战案例，深入学习并加强动漫分屏效果的制作技巧。我们将以《海贼王》角色索隆的视频为例，演示如何将多张图片与视频结合，制作出富有节奏感和视觉冲击力的分屏动画。

上一节我们介绍了基础的分屏操作，本节中我们来看看如何结合音乐节奏，制作更复杂的动态效果。

## 准备素材与裁剪图片

首先，我们需要导入视频素材并进行初步剪辑。

1.  点击“开始创作”，导入准备好的动漫视频片段。
2.  在时间轴上选取需要的片段，将不需要的后半部分裁剪并删除。

接下来，我们需要添加并处理角色图片。

以下是添加与裁剪图片的步骤：
*   点击“添加”，选择“照片”，导入索隆的4张图片。
*   选中第一张图片，点击下方工具栏的“编辑”，选择“裁剪”。
*   在裁剪比例中选择 **5:8**，然后移动调整框，确保人物位于画面合适位置，点击确认。
*   重复以上操作，将其余3张图片也裁剪为相同的 **5:8** 比例，并调整人物居中。

## 排列素材与添加动画

图片准备完毕后，我们需要将它们与视频进行合成。

1.  将4张裁剪好的图片全部“切换为画中画”。
2.  将所有画中画图片拖动到时间轴最前端。
3.  为了制造黑色背景效果，可以灵活地将其中一张图片“切到主轨道”上，然后将其余图片在主轨道和画中画轨道上依次排列整齐。

现在，为图片添加动态入场效果。

以下是添加动画的步骤：
*   选中第一个画中画图片，点击“动画”，选择“入场动画”。
*   为了制造激烈效果，我们选择“向右下甩入”，并将动画时长设置为 **0.5秒**。
*   依次为剩下的3张图片添加同样的“向右下甩入”动画，时长均为 **0.5秒**。

## 对齐音乐节奏

为了使动画与音乐完美配合，我们需要精细调整动画的出场时间。

![](img/3e90d40d0b961e81d695da6e57081f07_3.png)

1.  放大时间轴，便于精细操作。
2.  按照从左到右的顺序，将每个图片的动画开始时间依次延迟约 **10帧**（即大约0.3秒），制造依次飞入的动感效果。
3.  将原视频素材的音量调整为 **0**，进行静音处理。
4.  点击“音频”，选择“提取音乐”，从原视频中提取背景音乐，并拖动到时间轴最前端。
5.  根据音乐的节奏点（如强烈的节拍或人声进入点），调整画面出现的时机，确保视听同步。
6.  裁剪掉音乐多余的部分，并为音乐结尾添加“淡出”效果，让结束更自然。

本节课中我们一起学习了动漫分屏效果的进阶制作方法。关键步骤包括：将图片统一裁剪为 **5:8** 比例，使用“向右下甩入”动画增强动感，以及最重要的——根据音乐节奏精确调整每个元素的出现时间。掌握这些技巧，你就能制作出更具专业感和冲击力的视频作品。
